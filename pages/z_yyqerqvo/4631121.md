### CECC计划提名伊力哈木为2019年诺贝尔和平奖得主
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-10-27T07:28:07+08:00">
   2018年10月27日 07:28
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <p>
  美国国会及行政当局中国委员会(CECC) 周五在推文中重申，将计划提名维吾尔族学者伊力哈木·土赫提(Ilham Tohti）为2019年诺贝尔和平奖获得者。
  <br/>
  <br/>
  CECC在推文中说，维吾尔族学者伊力哈木周四刚刚度过49岁生日，他仍然以“分裂国家罪”在监狱中服刑。 CECC说，伊力哈木推动维族人和汉族人的对话，并批评目前新疆的高压政策，他应当立即被释放。
  <br/>
  <br/>
  推文中还强调，CECC主席、共和党参议员克里斯·史密斯和马克·鲁比奥计划提名伊力哈木为2019年诺贝尔和平奖获得者，并希望全球各国议员也采取同样举措。
  <br/>
  <br/>
  伊力哈木1969年出生于中国新疆维吾尔自治区阿图什县，大学本科在东北师范大学就学。后来到北京中央民族大学经济学院读研究生。随后留校任教，成为民族大学国际结算专业的副教授。
  <br/>
  <br/>
  多年来，伊力哈木主要研究新疆地区的收入差距及失业问题。近年来，伊力哈木时常对中国政府的新疆政策和少数民族政策提出批评。他认为，新疆问题的症结主要不在于维吾尔族人，而在于北京政府对维吾尔族的政治和经济歧视从而使维吾尔族人在自己的土地上成为二等公民，北京政府不正面解决歧视问题而一味地“反恐”只是使新疆问题恶化。他在2014年以”分裂国家罪”被判终身监禁。
 </p>
</div>
