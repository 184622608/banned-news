### 打压基督教行动蔓延 江苏教会屡遭冲击被关闭
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-09-09T01:14:42+08:00">
   2018年9月9日 01:14
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <span class="dateline">
  华盛顿 —
 </span>
 <p>
  在中国各地加紧打压基督教家庭教会之际，近期多次被骚扰的江苏省徐州市道恩教会9月7日遭到当局几个部门的联合冲击。
  <br/>
  <br/>
  据总部设在美国的宗教权益组织“对华援助协会”消息，江苏徐州道恩教会9月7日上午再次遭到当地宗教局、派出所等十多人的冲击，指教会没有向有关部门登记，在非宗教场所从事宗教活动，阻止信徒聚会。
  <br/>
  <br/>
  此前两周，道恩教会5个聚会点中的3个被当局下令关闭。公安向房东施压，不准租房给教会，逼迫信徒加入官方认可的“三自”教会，而传道人因拒绝向当局登记被罚款。
  <br/>
  <br/>
  消息表示，继浙江、河南等地大规模强拆包括官方认可教会在内的教堂的十字架、取缔官方不认可的家庭教会后，安徽、江苏等其他省份也开始仿效，取缔教会，拆十字架，只是还没有大规模展开。
  <br/>
  <br/>
 </p>
