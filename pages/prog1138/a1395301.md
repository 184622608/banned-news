### 孟宏伟事件疑团重重 妻子报案法国或透露玄机
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】
 <b>
  新闻周刊
 </b>
 （649）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 离奇失踪事件震惊海内外。孟宏伟曾频繁颁发〝红通令〞来抓捕中国逃亡海外人士，没想到自己也在中国被失踪。有法媒分析，孟宏伟的案情可能极为敏感或严重，北京当局才不顾在国际上丢丑的风险，也要抓人。
 <br/>
 <br/>
 法国警方透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_格蕾丝．孟.html" target="_blank">
  格蕾丝．孟
 </a>
 于当地时间10月4日晚，向法国警方报案。称孟宏伟自9月29号回中国后失踪。
 <br/>
 <br/>
 法国警方4日晚确认〝孟宏伟不是在法国境内失踪的〞、〝全案不是单纯的刑事案件〞。于是警方对此立案，并由里昂警署展开调查。
 <br/>
 <br/>
 7号格蕾丝在法国里昂举行记者会，但她背对摄影机，不愿露面。
 <br/>
 <br/>
 旅居澳洲的前台湾总统府国策顾问邱垂亮：〝我们外面得到的一个印象是说，中国人不管全世界哪里，是没有什么人权、安全的，他要抓就抓。现在唯一能解决这问题是国际刑警组织在法国。法国跟中国之间怎么样进行讨论，处理这个问题？〞
 <br/>
 <br/>
 《德国之声》对此报导说：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 失踪：中国开国际玩笑？〞
 <br/>
 <br/>
 美国华府人权律师叶宁：〝恐怕发生了一些非常紧急的事情，才会用这种硬着陆的这么粗暴、这么不光彩、不体面的方式来做这样的事情。这会在国际社会造成笑话，整个就是一个丑闻。〞
 <br/>
 <br/>
 在格蕾丝发言后不到一小时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共中纪委.html" target="_blank">
  中共中纪委
 </a>
 国家监委网站出现一则短消息：公安部副部长孟宏伟涉嫌违法，目前正接受国家监委监察调查。
 <br/>
 <br/>
 同日国际刑警组织的推特（Twitter）上有一则声明称：〝收到国际刑警组织主席孟宏伟先生的辞呈，立即生效。〞
 <br/>
 <br/>
 10月8号凌晨中共公安部长赵克志主持会议披露，孟宏伟涉及收受贿赂，并提到要坚决彻底肃清周永康流毒影响。
 <br/>
 <br/>
 原大陆法学博士张杰：〝（孟宏伟）他的影响力和他掌握的机密可能会远远超过王立军，（中共）可能采取诱骗的方式让他回国。后续他的妻子还有没有核弹级的爆料，或者其他东西，我们都不知道，可能还需要拭目以待。〞
 <br/>
 <br/>
 此前港媒《苹果日报》分析说，今次孟宏伟被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘密抓捕.html" target="_blank">
  秘密抓捕
 </a>
 ，可能涉嫌中共政治忌讳诸如北戴河政变等案件。
 <br/>
 <br/>
 法国《世界报》评论说，作为重大跨国治安组织的主席却凭空〝被消失〞，孟宏伟必定涉及非常敏感严重问题，才会让北京宁可冒着对国际出丑也要逮人的政治风险。
 <br/>
 <br/>
 孟宏伟2016年起担任国际刑警组织主席，任期到2020年止。2017年底，他被免去中共国家海洋局副局长和海警局局长职务，今年4月他又被免去公安部党委委员职务。
 <br/>
 <br/>
 原大陆法学博士张杰：〝如果说国际刑警组织的主席都在中国没有安全，那么中国（中共）跟一个黑社会又有什么区别呢？〞
 <br/>
 <br/>
 夏小强：〝在中共的体制下，从中共的党总书记、到中共各级的高官、企业家甚至普通民众，没有一个人是安全的。〞
 <br/>
 <br/>
 <br/>
