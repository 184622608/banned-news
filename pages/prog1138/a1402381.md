### 周晓辉：孟晚舟被拘与张首晟自杀并非巧合
------------------------

<div class="wysiwyg">
 12月6日当天推出的两条重磅消息无疑吸引了不少人的眼球，一个是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 副董事长、首席财务官、华为创始人任正非之女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 1日在加拿大温哥华转机时被拘捕，而美国当局正提出将其引渡到美国；一个是美国顶尖的华裔科学家、斯坦福大学物理系终身教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张首晟.html" target="_blank">
  张首晟
 </a>
 ，也在1日跳楼自杀身亡，终年55岁，其家人称是因为抑郁症所致。
 <br/>
 <br/>
 两则同时发生在12月1日，却同时在6日曝出的消息，网上是众说纷纭，各种小道消息也是满天飞，而二者是否仅仅是巧合，是否存在关联，恐怕不是表面看来那样简单。
 <br/>
 <br/>
 首先可以肯定的是，加拿大政府应美国的要求，拘捕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 ，是相当不寻常的，美国一定是持有一定的证据才会动手抓人。从目前各方透露的消息看，美国抓捕孟晚舟应与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 违反美国对伊朗的出口禁令有关。早在今年4月，就有美国媒体报导了美国司法部正在对其调查。孟晚舟曾是香港天通科技（Skycom Tech）的董事会成员，该公司曾试图向伊朗最大的行动电话商出售禁运的Hewitt Packard（HP）计算机设备，至少有13件销售提案，同时被列为〝机密〞，但销售提案后来遭到惠普禁止。
 <br/>
 <br/>
 提案虽然被禁止，但未必代表华为不会另辟蹊径，而这个证据极有可能被美国当局掌握。究竟是什么证据呢？有网友在知乎上爆料援引〝可靠内部消息〞称，孟晚舟被捕原因是华为可能与中东敏感地区（注：指伊朗）有贸易往来，其一位为美国政府工作的女同学，拿到了一份有孟签名的合同。如果果如其言，这样的证据倒是可以解释为何美国政府敢于做如此不同寻常之事。
 <br/>
 <br/>
 至于华为和孟晚舟为何不顾美国禁令，与伊朗继续保持生意往来，应与其并非是一家简简单单的公司，而是具有官方、军方背景，身负政治任务有关。今年4月在美国发布的美中经济安全审查委员会（USCC）的报告中，就指中共企业制造的信息技术产品可能经过篡改，以便进行监听或干预美国政府运作。该报告指出，中共政府可能支持一些企业进行间谍活动，以提高中企竞争力并促进政府利益。报告并列出包括华为、中兴通讯、联想、北京华胜天成技术、浪潮集团等多家中国ICT企业。
 <br/>
 <br/>
 几个月前中兴因向伊朗出口违禁产品被制裁后，根据网络曝光的中兴内部绝密文件，中兴公司的产品并不只卖给伊朗一个禁运国，还包括苏丹、朝鲜、叙利亚、古巴。而且，中兴还在想尽办法规避美国的检查，并且在文件中提到了〝公司的风险〞。明知风险还要前行的背后是因为有中共这只推手。
 <br/>
 <br/>
 作为喜欢与伊朗、朝鲜、苏丹、叙利亚这些邪恶政权为伍的中共，明里暗里向这些国家提供了不少帮助，允许、支持中兴和华为等向上述国家出口产品，就是为邪恶输送力量的重要组成部分，其目的是使这些国家分担中共所面临的来自美国等西方国家的压力，使这些国家有能力通过制造事端牵制美国等的精力。
 <br/>
 <br/>
 是以，背后有着中共鬼影、一直背负其指令的中兴、华为，也就无法理直气壮地、坦然面对美国的调查了，对于孟晚舟缘何被拘也是心知肚明的。任正非、孟晚舟纵有让人称道的经商才能，但他们与中共的天然联系，也是华为5G如今被美、澳、新、英等拒绝，孟晚舟陷入困境的根本原因。孟晚舟如何解困，恐怕不是华为所能解决得了的，而北京当局和华为为此将付出怎样的代价，是否如中兴一般被重罚并不能排除在外。
 <br/>
 <br/>
 再说说
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张首晟.html" target="_blank">
  张首晟
 </a>
 。尽管家人公开称其自杀是源于抑郁症，但无论是从其灿烂无比的学术生涯，还是风生水起的公司经营，都无法让人看出其抑郁在哪里。网上亦有一些曾在近两年见过张首晟的网友表示，完全看不出其有抑郁的症状，反而觉得其很阳光。
 <br/>
 <br/>
 还有一位抑郁症康复患者留言称，90%以上的抑郁症都不会在一个月内快速自杀，病情有一个自然发展的过程。通常情况下三到五个月是自杀的密集时间，再过一段时间自杀率通常会急速下降。考虑到一个月以前还能自如交流，病情极大的概率上不可能发展得如此迅速，几乎必然存在外部的因素。
 <br/>
 <br/>
 也许，这外部的因素不太可能是其投资失败，倒极有可能与华为背后的中共鬼影相联系。转折点应在2013年。
 <br/>
 <br/>
 2013年，中关村发展集团前来斯坦福大学参观，张首晟作为校方代表与其进行了深入的交流，并且很快达成合作意向，决定成立
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丹华资本.html" target="_blank">
  丹华资本
 </a>
 。这是一家风险投资基金，基金管理人为丹晟创业投资管理（北京）有限公司，法人代表为他曾经的学生兼得力助手谷安佳。该资本专注于投资最具颠覆性和影响力的科技成果和商业创新，关注的投资领域涵盖人工智能、虚拟/增强现实、大数据、区块链、企业级应用等具有颠覆性的新兴技术，而这些领域也是〝中国制造2025〞所关注的。
 <br/>
 <br/>
 也因为这样的合作，11月20日，美国发布的更新版301调查报告中指，中共利用其支持的风险投资公司对美国新创进行少数股权的小额投资，就可以简单获得与美国最新技术相关资讯。报告中就点名了张首晟创办的丹华资本。
 <br/>
 <br/>
 路透社也曾报导，丹华资本受到北京市政府资助的国有企业中关村发展集团支持，在一些最敏感的技术领域持有美国公司股份。其投资包括数据管理和安全公司Cohesity，该公司的客户包括美国能源部和美国空军。无人机初创公司Flirtey于5月被美国交通部门选中，帮助交通部将无人机安全地集成到美国空域，Flirtey也是丹华资本的投资对象。
 <br/>
 <br/>
 值得注意的是，据网上爆料，丹晟创业投资管理（北京）有限公司与华为技术有限公司也存在着隐秘的关联。具体来说，就是华为投资控股有限公司不仅投资华为技术公司，还投资国创开元股权投资基金，该基金又投资北京红杉信远股权投资中心，该中心又投资北京拱顶石科技有限公司，而该公司的监事许维娅亦是丹晟创投公司的监事。
 <br/>
 <br/>
 此外，丹晟创投投资丹晟投资基金中心，投资该中心的还有一家北京昌平中小微企业双创发展基金有限公司，该公司投资北京智行者科技有限公司，同时投资北京智行者公司的另外两家是广州信德创业营股权投资合伙企业、珠海广发信德科技文化产业股权投资基金，而这两家的投资者徐文伟是华为技术有限公司的董事。
 <br/>
 <br/>
 虽然有些复杂，但丹华资本与华为存在企业投资关联是不争的事实。2017年4月网上还有一个帖子称，华为公司董事余承东专程开车到深圳五洲宾馆，将正在那里开会的张首晟接走，〝一起吃饭谈重大事情〞。究竟谈了什么大事，不得而知，大胆的推测是商谈丹华公司帮助华为购买芯片和其它高端设备，投资美国高科技获取前沿技术等。对此内幕，美国政府应该掌握一二。
 <br/>
 <br/>
 不仅与华为存在关联，张首晟还是2009年被中共吸纳的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_〝千人计划〞.html" target="_blank">
  〝千人计划〞
 </a>
 成员，而这些成员，业已成为美国FBI重点调查对象，这不能不给其相当大的压力。一份非机密FBI备忘录中说，中共的〝千人计划〞，通过经济间谍活动和盗窃知识产权，对美国企业和大学构成严重威胁。或许直到此时，曾表示〝要把前沿的研究方向带回中国〞的张首晟才意识到，自己把美国的一些高端技术带回中国很可能涉及犯罪。他的内心无法再保持安宁，因为他无法面对身败名裂的那一天的到来。
 <br/>
 <br/>
 网上有消息指，在张首晟自杀那一天，有美国FBI到其实验室约谈，而这或许是压倒其内心的最后一棵稻草。而此前一天，他曾和孟晚舟一同参加了一个晚宴。
 <br/>
 <br/>
 华为、孟晚舟和张首晟走到今天，如果说前者作为中共利益集团的一员，还是心甘情愿为中共效力，那么后者则是过于天真，完全混淆了中共和中国的区别，被诱骗踏上了中共的贼船。但不管是主动效力还是被诱骗，历史的诸多教训已经在告诉世人，谁与中共同行，下场都是可悲的。
 <br/>
 <br/>
