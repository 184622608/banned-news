### 中共食药监局司长王立丰落马
------------------------

<div class="wysiwyg">
 【新唐人2018年08月24日讯】中共官网周四通报，中国食品药品监管（食药监）总局药品化妆品注册管理司司长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立丰.html" target="_blank">
  王立丰
 </a>
 ，因〝涉嫌严重违纪违法〞，正在接受审查。通报没有说明王立丰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 的原因，不过，外界认为，可能与长生疫苗案有关。
 <br/>
 <br/>
 据报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立丰.html" target="_blank">
  王立丰
 </a>
 所在的国家食药监总局药品化妆品注册管理司，管辖诸如优化药品、化妆品注册和行政许可管理流程，以及疫苗监管质量管理体系评估、药品行政保护等相关工作。
 <br/>
 <br/>
 而药品化妆品注册司是副局长吴浈的分管部门，王立丰和吴浈共事多年。
 <br/>
 <br/>
 8月16日，吴浈因涉长生生物问题疫苗案被立案审。
 <br/>
 <br/>
 吴浈任国家食品药品监督管理局副局长期间，长期分管药品注册、监管、审核等工作，手握重权。
 <br/>
 <br/>
 据报，吴浈近年来两次被实名举报，涉及延申生物疫苗案等问题，被指控滥用职权、以权谋私等。
</div>
