### 吉林检察院副检察长吴长智落马
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】中共官媒周二通报，吉林省检察院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副检察长.html" target="_blank">
  副检察长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴长智.html" target="_blank">
  吴长智
 </a>
 ，因涉嫌〝严重违纪违法〞被调查。据报，吴长智是中共江派高官杜青林的外甥，涉嫌干扰央视前主持人芮成钢案，以及香港富商、央视主播刘芳菲的丈夫刘希泳命案。
 <br/>
 <br/>
 公开资料显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴长智.html" target="_blank">
  吴长智
 </a>
 一直在吉林省司法系统任职。曾任吉林省检察院党组成员、反贪局局长。2016年11月，任吉林省检察院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副检察长.html" target="_blank">
  副检察长
 </a>
 。
 <br/>
 <br/>
 据报，吴长智是中央统战部前部长、江派大员杜青林的外甥。
 <br/>
 <br/>
 吴长智任吉林省反贪局局长期间，涉嫌包庇央视前主持人芮成钢，致使其案件罪名大大缩水。
 <br/>
 <br/>
 2017年2月，芮成钢被吉林省蛟河市法院一审判处有期徒刑六年。
 <br/>
 <br/>
 吴长智还被指涉嫌央视女主播刘芳菲的丈夫、香港富商刘希泳非正常死亡案，而刘希泳案涉及多名前中共政治局常委。
 <br/>
 <br/>
 另据《明慧网》报导，吴长智在任吉林桦甸市委副书记、政法委书记期间，被列为迫害法轮功的相关责任人。
</div>
