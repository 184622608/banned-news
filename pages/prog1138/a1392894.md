### 湖南衡阳前书记李亿龙 二审判18年
------------------------

<div class="wysiwyg">
 【新唐人2018年09月25日讯】中共官媒周二报导，前湖南衡阳市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李亿龙.html" target="_blank">
  李亿龙
 </a>
 ，一审法院宣判后上诉，近日二审维持原判18年。
 <br/>
 <br/>
 法院指控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李亿龙.html" target="_blank">
  李亿龙
 </a>
 犯有受贿、贪污、滥用职权和巨额财产来源不明罪，今年1月15日，一审
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_判刑18年.html" target="_blank">
  判刑18年
 </a>
 。李亿龙被指受贿926万余元、贪污142万余元，另有4950万余元财产不能说明来源。
 <br/>
 <br/>
 据报，有40多名官员涉李亿龙案，其中不少人已被判刑。
 <br/>
 <br/>
 1998年到2016年，李亿龙曾任湖南怀化市委书记、衡阳市委书记、湖南省农村工作领导小组副组长等职。
 <br/>
 <br/>
 李亿龙作风霸道，衡阳市民称其为〝霸道书记〞、〝四光书记〞，即〝土地卖光，项目上光，财政资金花光，看中的女人玩光〞。
 <br/>
 <br/>
 另据《明慧网》报导，李亿龙为了升官发财，积极参与迫害法轮功学员，多次被点名，并列入恶人榜。
</div>
