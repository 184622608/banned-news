### 淫僧释学诚被踢出政协常委 两大要职被免
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】因诱奸女尼，中共原佛教协会会长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_释学诚.html" target="_blank">
  释学诚
 </a>
 ，近日再被免去中共政协常务委员和民族和宗教委员会副主任职务，至此他几乎被免所有重要职务。港媒此前曝光释学诚〝成魔之路〞，包括违戒收众多女弟子，专找高学历
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_处女.html" target="_blank">
  处女
 </a>
 下手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵.html" target="_blank">
  性侵
 </a>
 后直接抛弃等。
 <br/>
 <br/>
 11月29日，中共政协第十三届全国委员会常务委员会第四次会议在北京闭幕。中共原佛教协会会长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_释学诚.html" target="_blank">
  释学诚
 </a>
 再被免去民族和宗教委员会副主任职务，同时其第十三届全国政协常委、委员资格也被终止。
 <br/>
 <br/>
 释学诚是中共培养的最高级别政治和尚，拥有一大堆头衔。
 <br/>
 <br/>
 8月初，两位出身清华大学的工程博士，前龙泉寺都监释贤佳（刘新佳）、释贤启（杜启新），联手撰写了长达95页的检举文件。文件中举报释学诚利用密宗的〝男女双修〞等说法迷惑、诱奸修学不深的女弟子，给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出家女弟子.html" target="_blank">
  出家女弟子
 </a>
 发送言词不堪入目简讯。
 <br/>
 <br/>
 另外在释学诚主持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京龙泉寺.html" target="_blank">
  北京龙泉寺
 </a>
 期间，未经许可的5处房屋违章建筑总面积达约1.37万平方米，估计使用人民币数亿元，资金动向不明。
 <br/>
 <br/>
 世界各大媒体8月初广泛报导了相关新闻，美国CNN更是在头条位置，斥责释学诚是〝披着袈裟的恶魔〞。
 <br/>
 <br/>
 迫于舆论压力，中共不得不做出回应。自8月中旬起，释学诚先后被免去了中共佛教协会会长、常务理事、理事、福建省佛教协会会长、常务理事、理事、北京市海淀区龙泉寺住持（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_方丈.html" target="_blank">
  方丈
 </a>
 ）、陕西省佛教协会名誉会长、陕西法门寺佛学院院长和陕西省扶风法门寺住持等职务。
 <br/>
 <br/>
 有佛教界知情人士透露，释学诚8月中旬已经被带返故乡福建福州，在一个小院〝闭门思过〞禁止出国。
 <br/>
 <br/>
 但在10月下旬，苹果日报披露，释学诚虽然已经辞职下台，但他在自己的根据地——福州、莆田仍然地位相当稳固，操控多间庙宇。报导中披露了释学诚〝成魔之路〞的更多细节。
 <br/>
 <br/>
 报导说，有举报信形容释学诚根本是〝淫僧成魔〞，被精神控制的女弟子，甚至甘愿带其他人上师父的床，来换取出国深造的机会。报导还详述了释学诚是如何一步步培养出不达标的女弟子。
 <br/>
 <br/>
 2013年，释学诚先后把7名在北京龙泉寺的专职女义工，调配到他的家乡福建仙游极乐寺，让她们成为〝冒牌比丘尼〞。这明显是违背戒律的，但女尼却认为〝那是师父对自己的无限慈悲〞。
 <br/>
 <br/>
 2014年约有30名女弟子受戒，成为释学诚收下的第一批女弟子；2015年共有47人，成为释学诚收下的第二批女弟子；2016年有78人；2017年有100人，〝龙泉寺〞的女尼队伍日益壮大。
 <br/>
 <br/>
 举报信中指控，这些快速出家的女尼，约有一半被陆续派到国内外近20个分院，出家女尼因此成了释学诚〝弘法〞的主力。举报信中提到6名受性骚扰的女尼，都在国外弘法，都是释学诚专门挑选出来的。
 <br/>
 <br/>
 举报人称，释学诚不仅多次发不雅简讯骚扰女弟子，还强行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵.html" target="_blank">
  性侵
 </a>
 高学历女尼。根据受害人指控，释学诚喜好
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_处女.html" target="_blank">
  处女
 </a>
 ，诱奸前要再三确认对方是处女才下手，完后直接抛弃，导致一名女尼自杀，至少有3位被外派的女尼精神失常。
 <br/>
 <br/>
 举报人释贤启，8月曾向美国CNN描述，他是如何开始调查释学诚的。他说，当时一名〝恐惧而紧张的〞尼姑告诉他，释学诚向她发送露骨简讯。在获得这些简讯之后，举报人释贤启和释贤佳，跟警方联系，并向技术专家谘询。专家排除了释学诚帐号被骇的可能性。
 <br/>
 <br/>
 释贤启说，许多虔诚的女居士想要入寺，释学诚总是同意，他说：〝我不知道我是在将她们送入虎口。〞释贤启表示，〝释学诚的博学形象在我脑海里轰然倒塌〞，〝留下的只是巨大的恐惧。他是个穿着僧袍的邪魔。〞
 <br/>
 <br/>
 上述举报在舆论发酵后，迫于舆论压力，释学诚终遭调查。据悉，在调查期间，释学诚一度移居北京广济寺，有专人盯守，统战部纪检组人员找他核查，经过几天拉锯，他终于承认部分举报。
 <br/>
 <br/>
 释学诚于1982年在福建莆田广化寺落发为僧。年轻时被〝政治和尚〞、时任中共佛教协会会长的赵朴初提携，23岁就成为最年轻的名寺方丈，之后其职位一路攀升。
 <br/>
 <br/>
 中共地下党员出身的赵朴初听命于中共，祸乱佛教界，他提拔的少林方丈释永信也被称作〝少林CEO〞，被指祸乱佛法，多年前也曾被举报涉嫌淫乱，并有私生子。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
