### 3年轻女生掌控西安千亿国企 知情人曝内情
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】针对陕西西安〝千亿资产
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国企.html" target="_blank">
  国企
 </a>
 由80后任董事长，95后任董事〞事件，5日凌晨，西安官方通报，高新区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财政局.html" target="_blank">
  财政局
 </a>
 局长被免职。但有媒体发现，新任的年轻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高管.html" target="_blank">
  高管
 </a>
 不只3人而是10人。有知情人透露了其中的内情。
 <br/>
 <br/>
 西安高新区官方5日凌晨，就80后、90后女生任职千亿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国企.html" target="_blank">
  国企
 </a>
 董事会一事发布声明称，已免去西安高新区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财政局.html" target="_blank">
  财政局
 </a>
 局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王进杰.html" target="_blank">
  王进杰
 </a>
 职务。 声明还称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安高新控股.html" target="_blank">
  西安高新控股
 </a>
 有限公司属于政策性企业，截至今年6月底，企业总资产人民币1270亿元净资产403亿。
 <br/>
 <br/>
 对于舆论关注的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李甜.html" target="_blank">
  李甜
 </a>
 等3名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高管.html" target="_blank">
  高管
 </a>
 ，通报称，她们并没有特殊的家庭背景。目前，李甜月工资4,351元人民币；朱玥月工资3,600元人民币；赵雪莹月工资4,200元人民币，在任职前后薪资无变化。
 <br/>
 <br/>
 事件缘于西安高新控股9月份发布人事调整公告，1984年8月出生的李甜，出任公司的法人代表、董事长兼总经理。而新任的二名董事：赵雪莹1993年生、2016年毕业于西安建筑科技大学、曾任职银行；朱玥1995年生，2017年毕业于陕西师范大学。
 <br/>
 <br/>
 相关消息在11月2日被陆媒挖出，李甜等3名年轻高管，掌控总资产超过千亿人民币的国企，引起舆论质疑。
 <br/>
 <br/>
 11月5日，上游新闻报导说，西安高新控股有限公司，目前任职的董事会和监事会不只是3名高管，而是10名高管，而且全部是〝80后90后〞，最早任职时间为2017年11月5日，最晚任职时间为2018年9月4日，被舆论关注的三名当事人
 <br/>
 <br/>
 相关记者获得的《西安高新控股有限公司2018年度第十一期超短期融资券募集说明书》，披露了西安高新控股的董事会、监事会等10名高级管理人员简介。
 <br/>
 <br/>
 10名新任职的年轻高管任职日期分为两个批次：2017年11月5日，2018年9月4日。其中，董事会共有董事7名，监事会设有监事3名。
 <br/>
 <br/>
 据知情人透露，西安高新区控股有限公司，变更董事会背景是，中共审计署在审计西安时，提出〝西安所有平台公司的负责人，均由财政局或其他政府官员担任，不符合政企分离要求〞。
 <br/>
 <br/>
 而该公司董事长原为西安高新区财政局局长，因这家公司与财政局基本是〝一套人马，两块牌子〞，因此局长无法担任，由于〝干部青黄不接〞，所以就安排了若干年轻人在台前代为履职。
 <br/>
 <br/>
 知情人说，实际这些新任职高管没有任何控制力和履职能力，说穿了就是〝影子〞，西安高新控股的实际控制人是高新区财政局。
 <br/>
 <br/>
 另有消息说，这背后原因是因这些员工的工位距离财务总监非常近，便于签字。而这些员工均是基层人员，符合政企分离的要求。因而西安高新区控股有限公司在变更董事会时，便选择李甜、赵雪莹和朱玥等人。
 <br/>
 <br/>
 但网友们对迟来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停职调查.html" target="_blank">
  停职调查
 </a>
 及官方的回应仍感不满，同时还质疑上述知情人的说法，认为〝毕业就成董事，搞世袭是中国的现状，普遍的现状〞。
 <br/>
 <br/>
 公开资料显示，西安高新控股公司于2003年10月成立，注册资本11.3亿元，由高新区财政局负责管理。截至今年6月底，高新控股公司总资产1270亿元，净资产403亿元。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉)
