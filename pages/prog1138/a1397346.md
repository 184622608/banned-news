### 中共高官曝北京内部气氛：不是你死就是我亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】经济困局加上贸易战高压令中共政权风雨飘摇。执政危机同时引爆中共内部积聚的矛盾，令权力斗争再度激化。日前哈佛学者论及中共改革时，引述中共高官透露出中南海内斗的残酷气氛。
 <br/>
 <br/>
 哈佛大学肯尼迪政府学院（简称肯尼迪学院）的威廉．奥弗霍尔特（William Overholt）博士向BBC透露，一个高层中共官员访问哈佛时告诉他，〝北京的气氛是：不是你死，就是我亡〞。
 <br/>
 <br/>
 BBC在10月29日的报导中说，奥弗霍尔特博士日前在香港中文大学出席研讨会时，向BBC记者透露了上述内容。奥弗霍尔特在1990年代初曾成功预言中共的〝经济崛起〞。
 <br/>
 <br/>
 奥弗霍尔特透露〝北京气氛〞时，正在向BBC记者论述中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济改革.html" target="_blank">
  经济改革
 </a>
 。
 <br/>
 <br/>
 他说，中共经济改革困局背后，是同样复杂的政治局势。因为经济改革会使各个利益集团受损，他们因此起来阻碍改革。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 通过反腐运动打击利益集团，巩固权力，同时强推改革。
 <br/>
 <br/>
 但是，反腐的副作用是使得经济改革的实施者——中共官员们变得畏首畏脚。由于害怕被政敌指控腐败，官员只好选择〝不作为〞，〝像课堂上不愿回答问题的学生一样缩在最后一排〞。
 <br/>
 <br/>
 那位中共高层官员告诉奥弗霍尔特，〝这个（改革）过程的斗争是非常激烈的〞，北京内部已陷入你死我活的利益争夺和权力斗争。
 <br/>
 <br/>
 奥弗霍尔特没有透露那位中共高层官员的身份。但据公开资料，哈佛大学肯尼迪学院一度是中共高官的培训基地，有中共〝海外党校〞之称。习近平的经济智囊、中共副总理刘鹤也曾在该学院获得MPA（公共管理硕士）学位。今年以来，刘鹤因为中美贸易谈判多次赴美。
 <br/>
 <br/>
 刘鹤日前在党媒专访中论及经济改革，也提到改革困境和官员的〝不作为〞、〝不敢担当〞，与奥弗霍尔特所谈内容大同小异。
 <br/>
 <br/>
 奥弗霍尔特还在BBC专访中指出，北京当局通过不断集权控制各个利益集团，然后强推改革，等于将中国置于一个政治压力的沸腾水壶上，可能一时管用，但不会永远有效。
 <br/>
 <br/>
 他说，目前无法对中国未来经济走势进行预测，唯一可以肯定的是：〝巨变即将发生〞。
 <br/>
 <br/>
 事实上，中国目前的局势似乎也在印证奥弗霍尔特透露的内幕。
 <br/>
 <br/>
 习近平上台后，通过反腐扫荡一度掌控政局二十年的江泽民派系，同时削平各个政治山头。为了解决经济危局，习当局开始强推经济改革，用铁腕手段清扫中共权贵掌控的各领域利益地盘。不过，经济改革举步维艰，权力和利益之争却让中共继续分化，不只是江派贪官反习，包括红二代内部也开始分裂。
 <br/>
 <br/>
 其中，随着邓小平家族的女婿吴小晖入狱，和邓朴方〝生死之交〞的樊立勤在北大贴大字报批评习近平，邓习两家的矛盾已经公开化。日前又传出邓朴方演讲稿，力挺遭习当局压制的邓小平理论，不点名地提醒北京当局〝认清自己的份量〞，再次引起外界关注。
 <br/>
 <br/>
 中美贸易战爆发以来，中共内部批评习当局过早抛弃〝韬光养晦〞的声音开始抬头。因此传出的逼习近平下台的所谓〝政变〞流言不断，显示中共内部又开始激烈的权斗。
 <br/>
 <br/>
 日前，习近平〝南巡〞无讲话，定调经济政策的十九届四中全会一再延期。自由亚洲引述分析称，这说明中共高层就如何应对贸易战仍然没有〝统一认识〞，其背后原因可能是北京的权力斗争已陷于胶着。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
