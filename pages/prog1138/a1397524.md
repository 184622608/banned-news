### 中共司法部前高官卢恩光被判12年
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】周二，中共前司法部政治部主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢恩光.html" target="_blank">
  卢恩光
 </a>
 被判有期徒刑12年，被控行贿人民币两千多万元。据报，卢恩光在年龄、入党材料、工作经历、学历和家庭情况等方面造假，被称为〝五假〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副部级干部.html" target="_blank">
  副部级干部
 </a>
 。
 <br/>
 <br/>
 判决称，1992年到2016年间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢恩光.html" target="_blank">
  卢恩光
 </a>
 为谋取教师身分、荣誉称号、职务提拔、工作调动等，向国家工作人员行贿共计人民币2074万元。
 <br/>
 <br/>
 卢恩光曾任《华夏日报》社长、中国残疾人福利基金会副理事长兼秘书长等职。 2009年，任司法部政治部副主任兼人事警务局局长。 2015年11月任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共司法部.html" target="_blank">
  中共司法部
 </a>
 政治部主任。
 <br/>
 <br/>
 2016年12月卢恩光落马，是中共十八大以来司法系统落马的第一个部级官员。
 <br/>
 <br/>
 卢恩光用全面造假方式，从私营企业主变为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副部级干部.html" target="_blank">
  副部级干部
 </a>
 ，从高中毕业生变成博士。
 <br/>
 <br/>
 中共官媒消息，中纪委前书记王岐山曾点名卢恩光的造假细节：〝1990年入党志愿书中，竟写了学习邓小平1992年南巡讲话的体会〞。
</div>
