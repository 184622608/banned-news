### 房峰辉张阳旧事 揭〝房一枪〞〝张麻袋〞的由来
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】日前，中共党媒通报，前军委委员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张阳.html" target="_blank">
  张阳
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房峰辉.html" target="_blank">
  房峰辉
 </a>
 被开除中共党籍，并褫夺上将军衔，有关两人的丑闻也被曝光。据说，张被取外号〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张下跪.html" target="_blank">
  张下跪
 </a>
 〞、〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张麻袋.html" target="_blank">
  张麻袋
 </a>
 〞，房则被取外号称〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房一枪.html" target="_blank">
  房一枪
 </a>
 〞。
 <br/>
 <br/>
 10月16日，中共官媒《新华社》先后通报对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房峰辉.html" target="_blank">
  房峰辉
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张阳.html" target="_blank">
  张阳
 </a>
 的审查结果。张房两人同时被开除中共党籍，并被取消上将军衔。
 <br/>
 <br/>
 原中共联合参谋部参谋长房峰辉，涉嫌严重违法犯罪案移送审查起诉。原中共政治工作部主任张阳已在2017年11月23日〝自缢死亡〞，通报痛斥其以自杀方式企图逃避惩处，宣布追缴其涉案财物。
 <br/>
 <br/>
 值得关注的是，两人都被指控〝搞两面派做两面人〞，〝对党不忠诚不老实〞和〝政治上蜕变、经济上贪婪〞三宗相同的罪名。
 <br/>
 <br/>
 分析认为，张房双双遭当局严辞痛批，除显示其背后牵连甚深之外，官方也是在表明一个态度，即张阳之死并不能遏阻对案件的调查。
 <br/>
 <br/>
 此前就有猜测说，张阳可能是希望通过自杀保护他想保护的人。
 <br/>
 <br/>
 去年张阳自杀后，海外党媒引述知情人士透露，涉及张阳案的有40多人，目前均被调查。其中，张阳仕途旧地原广州军区至少有32人被带走，军委机关也有8人被查。
 <br/>
 <br/>
 消息人士还称，涉及张阳案的深度和广度，不亚于中共前军委副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 案、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐才厚.html" target="_blank">
  徐才厚
 </a>
 案。
 <br/>
 <br/>
 张阳在徐才厚主管的军队政工系统任职多年，被视为徐的嫡系。而房峰辉是郭伯雄的老乡兼旧部，据悉也是郭的〝头马〞，早年为了上位，房直呼郭为〝姐夫〞。
 <br/>
 <br/>
 去年11月底，有海外中文媒体披露，张阳在广州军区任职时，与多名商人关系密切，私下收受财物，其别墅装修花费300多万元人民币，全部是由他人支付。
 <br/>
 <br/>
 因涉买官卖官，张阳在原广州军区有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张麻袋.html" target="_blank">
  张麻袋
 </a>
 〞的绰号。广州军区有人披露，张阳的贪腐问题很严重，想跟他买官，得装一麻袋钱才能见面。
 <br/>
 <br/>
 港媒《明报》刊文称，徐郭落马后，房张两人的前景已趋黯淡。当年徐才厚被查后，张阳为检讨自己过往的关系，竟向习近平下跪求饶。
 <br/>
 <br/>
 而房峰辉则在郭伯雄被查消息初起时，私下表示，〝谁要敢动老首长（郭伯雄），我一枪毙了他〞，态度嚣张。
 <br/>
 <br/>
 文章说，虽然后来房张在军中清除郭徐余毒的运动中都高调表态，但无论是态度倨傲的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房一枪.html" target="_blank">
  房一枪
 </a>
 〞，还是情状可怜的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张下跪.html" target="_blank">
  张下跪
 </a>
 〞都难逃被清算的命运。
 <br/>
 <br/>
 徐才厚、郭伯雄是中共前党魁江泽民的军中亲信，郭伯雄于2015年4月落马，2016年被判处无期徒刑；徐才厚2014年3月落马，2015年3月因患癌症死亡。
 <br/>
 <br/>
 郭徐两只军虎落马后，彻底肃清〝郭徐余毒〞成为习近平军队整改的重中之重。十八大以来，至少有160个将军因贪腐被查处，其中包括田修思、王建平等数位上将。
 <br/>
 <br/>
 据中共党媒报导，过去5年，至少有13,000个军官被调查。有美媒评论称，这个数字比中共建军85年来，内战外战加文革倒下的将军总和还要多许多倍。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：范铭）
