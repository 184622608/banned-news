### 港媒：中共招募31名高中生 研发〝杀手机械人〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】日前，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京理工大学.html" target="_blank">
  北京理工大学
 </a>
 从5,000余名候选人中，招募31名〝最聪明〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高中生.html" target="_blank">
  高中生
 </a>
 ，培训他们成为研发人工智慧（AI）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武器.html" target="_blank">
  武器
 </a>
 杀手的最年轻AI武器科学家。这一消息引发国际媒体关注。
 <br/>
 <br/>
 中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京理工大学.html" target="_blank">
  北京理工大学
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_BIT.html" target="_blank">
  BIT
 </a>
 ）网站日前公布消息称，该校〝智能
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武器.html" target="_blank">
  武器
 </a>
 系统实验班〞10月28日举行了开班仪式，共招收27名男生和4名女生，这些新生是从5,000余名候选人中挑选出来的。
 <br/>
 <br/>
 香港《南华早报》以&lt;中国最聪明的学生被招录开发人工智慧（AI）〝杀手机器人〞&gt;为题报导说，北京理工大学是该国最大的武器研究机构之一，新计划的启动证明了它对军用AI技术发展的重视。
 <br/>
 <br/>
 根据中共国防部网站的信息，北京理工大学隶属于国防科学技术工业委员会，是中共建政以来重点建设的高校之一，也是首批进入〝211工程〞和〝985工程〞建设行列的高校之一。
 <br/>
 <br/>
 北京理工大学的前身是创办于1940年的延安自然科学院，是中共创办的第一所理工科大学。也是国防科学技术研究的重要基地之一，及高级武器研究机构之一。
 <br/>
 <br/>
 报导说，中共与美国和其他国家，正在竞相开发致命的人工智慧（AI）应用，从具有自学晶元的核潜艇，到可以爬入人体血管的微观机器人。
 <br/>
 <br/>
 AI武器，被外界称为〝杀手机器人〞（killer robots），通常是指致命的自主武器，可以在没人类参与下，选择和销毁人类目标。因背后涉及道德问题备受争议。
 <br/>
 <br/>
 英国《每日邮报》称，一批〝最聪明的爱国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高中生.html" target="_blank">
  高中生
 </a>
 〞被中共招募，研究制造人工智慧兵器。这31名高中生都未满18岁。
 <br/>
 <br/>
 一位不愿透露姓名的北京理工大学教授说，〝这些孩子都特别聪明，但光聪明是不够的，其他品质也是我们需要的，比如创造性思维，战斗意志，面对挑战时的坚持。开发新武器的热情也是必须的，他们必须是爱国者。〞
 <br/>
 <br/>
 根据项目手册，每位学生将分配两名具有学术和国防背景的导师，未来学生将被分配专业领域的兵器实验室，在专家指导下磨炼自己的技能。
 <br/>
 <br/>
 在完成4年课程后，研究所希望这些学生继续读博士课程，最终成为中共AI武器工程的下一代领军人物。
 <br/>
 <br/>
 《商业内幕》网站称，中共目前在各个领域发展人工智慧，包括前景可达数万亿美元的汽车自动驾驶和面部自动识别。
 <br/>
 <br/>
 7月份，中共发布了AI发展计划，未来几年内国内人工智慧产业产值推升到1,500亿美元，2030年将中国建设为〝AI创新中心〞。
 <br/>
 <br/>
 报导认为，中共很大程度上保持了AI武器技术发展的不透明性，报导引述预测，中国〝可能会利用AI增强战斗力，比如无人自主系统〞。
 <br/>
 <br/>
 加州大学伯克利分校智能系统中心主任拉塞尔（Stuart Russell）将中共的BIT大学项目描述为〝一个非常糟糕的想法〞。
 <br/>
 <br/>
 拉塞尔说〝永远都不应该允许机器来决定杀人。这样的武器很快成为大规模杀伤性武器。而且，它们也增加了战争的可能性。〞 〝追求致命性自主武器的开发，将会严重危害到国际、国家、地方及个人的安全。〞
 <br/>
 <br/>
 他补充说，希望所有这些中国学生在开始课程之前观看〝杀手机器人〞（Slaughterbots）影片。
 <br/>
 <br/>
 该影片描绘了一个令人不安的未来，成群的低成本无人机可以藉助AI技术，如面部识别来屠杀人类。杀手机器人不到3秒的时间就可完成刺杀任务。
 <br/>
 <br/>
 中共是联合国中支持禁止致命自主武器系统的26个国家之一。中共官媒9月曾发文称，中共呼吁制定一项新的常规武器公约，〝禁止使用完全自主的武器系统〞。
 <br/>
 <br/>
 如今，中共通过北京理工大学招募高中生研究AI武器的举动，令国际专家学者对中共的真正立场表示怀疑。
 <br/>
 <br/>
 麻省理工的特马克（Max Tegmark）教授说，中共是支持联合国禁止致命的自主武器系统的第一个大国，如果北京理工大学的这批高中生研发这种武器，那将是一个耻辱。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
