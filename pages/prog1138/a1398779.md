### 知情者：赖小民案三年审不完 大牌女星等数百人卷入
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】中共国家监委对中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融集团.html" target="_blank">
  华融集团
 </a>
 原董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 案进行了近7个月的调查后，近日正式将其移送检察机关，赖小民随即被正式逮捕。有陆媒援引两名接近该案的知情者透露，该案涉及数以百计的人员，没有三年的时间难以办完。
 <br/>
 <br/>
 中共国家监委终结对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 案件的调查并将其移送检察机关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_审查起诉.html" target="_blank">
  审查起诉
 </a>
 后，中国大陆媒体《经济观察报》刊发报导，引述知情人士透露了该案的一些敏感细节。
 <br/>
 <br/>
 报导称，两位接近赖小民案情的人士透露说，赖小民一案牵涉的资金超过16亿元人民币，而自从赖小民今年4月中旬被调查以来，他本人交代和调查人员〝挖掘梳理〞出了数以百计的〝关系人〞。
 <br/>
 <br/>
 其中一位知情人士表示，以赖小民案为导火索，中国金融系统的反腐现在才是开端。他说：〝仅仅他（赖小民）交待的这些涉案事情和涉案人，以现有的力量，没有三年时间，难以办完。〞
 <br/>
 <br/>
 知情人士并透露说，赖小民搞〝权色交易〞的对象，既包括欲求职于华融公司的普通女性，亦包括〝大牌女明星〞，有的女性还故意将自己〝大尺度的写真照〞附在简历中，递交给赖小民，〝彼此心照不宣〞 。
 <br/>
 <br/>
 资料显示：赖小民1962年生于江西赣州瑞金，从江西财经学院的国民经济计划专业毕业后，就进入中国人民银行工作，历任计划资金司中央资金处副处长、处长；计划资金司银行二处处长；信贷管理司副司长；监管二司副司长等职。
 <br/>
 <br/>
 2003年，中国银监会成立，赖小民即调入银监会工作，历任银行监管二部副局级干部；北京银监局筹备组组长，北京银监局局长；银监会办公厅主任。
 <br/>
 <br/>
 2009年，赖小民出任华融公司党委副书记、总裁；2012年，升任该公司的党委书记、董事长。
 <br/>
 <br/>
 今年4月17日，中央纪委国家监委对外发布了赖小民涉嫌严重违纪违法正接受调查的消息。
 <br/>
 <br/>
 10月15日，中共中央纪委国家监委宣布了对赖小民开除党籍、开除公职的处分决定。当天，中纪委官网发文评论称，〝从中央纪委通报看，赖小民的问题十分严重，一些细节更是触目惊心。〞
 <br/>
 <br/>
 此后，中国银保监会、中国人民银行等先后召开了赖小民案的警示教育大会。会议措辞严厉，称〝赖小民案件性质极其恶劣、教训极为惨痛〞，赖小民给〝国有金融资产造成重大损失，危害国家金融安全和金融稳定，影响极其恶劣，触目惊心、教训深刻。〞
 <br/>
 <br/>
 11月7日，最高检察院通报称，国家监察委员会赖小民案件的调查终结，移送检察机关审查起诉。
 <br/>
 <br/>
 经最高检察院指定管辖，由天津市人民检察院第二分院审查起诉。天津市检察院第二分院随即对赖小民作出逮捕决定。
 <br/>
 <br/>
 赖小民案是中央纪委和国家监察委合署办公后，联合正式办理的第一个金融大案。
 <br/>
 <br/>
 （记者黎明报导/责任编辑：李泉）
