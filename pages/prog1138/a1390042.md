### 河南平顶山前公安局长段玉良落马
------------------------

<div class="wysiwyg">
 【新唐人2018年09月03日讯】中共官媒周一通报，河南省平顶山市政协主席、党组书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_段玉良.html" target="_blank">
  段玉良
 </a>
 ，涉嫌〝严重违纪违法〞，目前正接受调查。
 <br/>
 <br/>
 官方通报称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_段玉良.html" target="_blank">
  段玉良
 </a>
 〝投案自首〞，但没有提到他被查的具体原因。
 <br/>
 <br/>
 2000年1月起，段玉良曾任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_河南平顶山.html" target="_blank">
  河南平顶山
 </a>
 检察院检察长、市委常委、政法委书记、公安局长、党委书记、纪委书记等职；2012年2月起，任平顶山市政协主席、党组书记。
 <br/>
 <br/>
 另据《明慧网》报导，段玉良任职期间，积极追随江泽民集团迫害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 ，被多次曝光。因对当地迫害法轮功学员负有责任，被〝追查迫害法轮功国际组织〞追查。
</div>
