### 孟妻聘两公司〝寻找孟宏伟〞 北京称她拒绝与夫通话
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 在中国被查后，其妻近日再发声明说，她已雇用两家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_律师事务所.html" target="_blank">
  律师事务所
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_寻找.html" target="_blank">
  寻找
 </a>
 她丈夫下落。而中共外交部回应称，孟妻拒绝与孟宏伟通电话。有学者认为，即使请20家公司去介入该案件，其结果也有可能只是零。建议孟妻抛出手中可以和中共谈判的筹码。
 <br/>
 <br/>
 中共公安部副部长、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 前主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 在中国已经〝失踪〞一个多月了。路透社报导称，孟妻发出了一份电邮声明，透露自丈夫失踪后，来自世界各地的人，询问如何提供帮助，她目前已经选择两家海外的法律机构介入案件，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_寻找.html" target="_blank">
  寻找
 </a>
 丈夫的下落。
 <br/>
 <br/>
 孟妻委托的两家公司，分别来自位于巴黎的Marsigny Avocats事务所和伦敦的Lindeborg法律顾问公司。
 <br/>
 <br/>
 旅美媒体人陈小平对此不看好，他说：即使是20家公司去介入孟宏伟这个案件，其结果也有可能只是零。但是觉得孟妻这样的选择可能会有其他特别的理由。
 <br/>
 <br/>
 法学学者钱跃君对自由亚洲表示，孟宏伟案毫无疑问是政治案件，这两家法律公司除了可以向中共要说法之外，还可以敦促法国政府、国际刑警组织介入该案，透过法律寻求帮助。
 <br/>
 <br/>
 钱跃君说：不论孟妻手中是否有传说中可以谈判的筹码，但她目前的国际行动是对案件的走向有帮助的。她如果做这件事情，可以公布出来谈判的筹码，而且要切切实实拿出东西来。如果中共是无赖，也要说清楚，原来孟宏伟是被非法绑架的是权斗的悲剧。
 <br/>
 <br/>
 就孟妻雇用了两家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_律师事务所.html" target="_blank">
  律师事务所
 </a>
 寻找她丈夫的下落。中共外交部10月30日在记者会上指责孟宏伟的妻子，一方面宣称自己不知道孟宏伟下落，另一方面又拒绝与之电话联系，其具体原因中方并不了解。
 <br/>
 <br/>
 陆慷宣称，可以随时安排孟宏伟的妻子和孟宏伟通话联系，但孟妻自己拒绝这样做。
 <br/>
 <br/>
 不过，陆慷的上述表态，没有显示在中共外交部官方网站的文字记录中。
 <br/>
 <br/>
 孟宏伟9月29日从法国前往中国后神秘消失，孟妻10月5日向法国警方报案后，立即引发国际舆论界的哗然。
 <br/>
 <br/>
 在国际舆论的压力之下，中纪委10月7日深夜通报，孟宏伟涉嫌违法，目前正接受中共国家监委监察调查。
 <br/>
 <br/>
 目前仍不清楚孟宏伟被拘押在什么地方，也不清楚他有没有被允许聘请法律顾问。
 <br/>
 <br/>
 中国民主运动海外联席会议主席魏京生对美国之音说，孟宏伟应该卷入了突发事件，而且事件非常严重，才导致北京不得不紧急处理。孟宏伟的动向应该危及习近平。
 <br/>
 <br/>
 有港媒说，孟宏伟涉嫌参与北戴河的未遂政变案，因为案件重大而紧急，北京当局才不顾影响而紧急抓人。
 <br/>
 <br/>
 对于孟妻求助西方社会声援保护，很多网民认为孟宏伟被抓是咎由自取、作恶多端的报应，做了中共10多年的酷吏刀把子，他曾让无数中国民众无故失踪，现在终于轮到他自己了。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：戴明）
