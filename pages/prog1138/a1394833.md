### 孟宏伟妻身份神秘 受访CNN时电话响不停
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】日前，前中共公安部副部长兼国际刑警主席孟宏伟回国后被抓，孟妻在法国报警。网传孟妻在法国享受奢侈生活，但身份不为人知。9日，孟妻接受CNN采访时不断有电话骚扰。有分析指，北京处理孟百密一疏，漏掉了孟妻。
 <br/>
 <br/>
 10月9日，孟宏伟的妻子格蕾丝．孟（Grace Meng）在法国里昂接受了美国有线电视新闻网CNN的专访。
 <br/>
 <br/>
 她说，丈夫失踪后，她收到了来自陌生人的威胁电话，称已有两个专门小组正过来找寻她，并以她和孩子的安全相威胁。
 <br/>
 <br/>
 CNN报导称，在接受采访时，她把脸藏起来，她的手机响了三次。
 <br/>
 <br/>
 孟妻自称，这是中共领馆人员一直在不停地打电话，但她拒绝与他们单独会晤，除非有媒体在现场。
 <br/>
 <br/>
 有推特消息指，格蕾丝．孟持香港护照，无业，两年来在里昂豪华街区租住一幢豪宅。但是，她并无中共〝驻外人员家属〞身份，在法国的奢侈生活并非由国际刑警组织或中共公安部出资。
 <br/>

 <br/>
 也就是说，格蕾丝．孟虽然自称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟妻子.html" target="_blank">
  孟宏伟妻子
 </a>
 ，但此前外界并不知道她的存在。
 <br/>
 <br/>
 从她10月7日记者会上的背影来看，其年龄要比65岁的孟宏伟年轻很多。而且，她和孟宏伟的儿子才刚刚7岁。
 <br/>
 <br/>
 有分析指，孟宏伟对自己的处境早有预感，回国前应该安排了安全预案。格蕾丝．孟的手机收到他发回的尖刀图案，随后选择在法国报警，被指其安全预案的一部分。
 <br/>
 <br/>
 亦有分析认为，北京本打算低调处理孟宏伟，但百密一疏，没能有效控制格蕾丝．孟，或令她成为一个〝女版王立军〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟落马.html" target="_blank">
  孟宏伟落马
 </a>
 后，中共公安部会议称，孟是〝一意孤行、咎由自取〞，并表示〝绝不允许跟组织谈条件、谈‘价钱’〞，或暗示孟宏伟落马前一直在和北京当局进行对抗和博弈。
 <br/>
 <br/>
 因此有舆论指，北京不顾国际影响拿下孟宏伟，或因事态紧急。也有猜测称，格蕾丝．孟或掌握与北京谈判的某些〝筹码〞。
 <br/>
 <br/>
 格蕾丝．孟在接受外媒采访时不断美化其丈夫，称孟宏伟〝毕生追求法治〞、〝没有能力贪腐〞等。但该说法受到大量舆论质疑，有报导指，孟宏伟当年作为周永康直接下属，是中共暴力维稳体系的骨干中坚人员之一，镇压法轮功、维权人士甚至活摘器官应该都有份参与。在孟宏伟手中失踪的良心犯，恐难以计数，〝毕生追求法治〞之说，不过是极具讽刺的粉饰而已。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
