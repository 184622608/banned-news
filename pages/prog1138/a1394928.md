### 孟宏伟为何被查？日媒揭习近平真实用意
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被调查后，孟的中共公安部副部长一职已由习近平的旧部邓卫平接任。10日，日媒分析，北京不顾及国际影响，突然抓捕孟宏伟这一大动作背后，透露出习近平希望彻底〝执掌〞公安部的真实意图。
 <br/>
 <br/>
 10月10日，日媒《每日新闻》报导说，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 兼国际刑警组织（ICPO）主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 〝失踪〞7天后，突然于10月7日被证实遭中共调查，北京的这一大动作背后，透露出了中国国家主席习近平彻底〝执掌〞公安部的考量。
 <br/>
 <br/>
 孟宏伟曾任中共公安部副部长长达14年，期间担任部长的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 ，是江派利益集团的核心人物，〝政法系〞被称为周的〝独立王国〞。
 <br/>
 <br/>
 但在2012年，习近平上台后，掀起反腐风暴，公安部高官纷纷落马，周永康也被判处无期徒刑。但与周有牵连的孟宏伟却能全身而退，甚至还于2016年被中共推上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 的要职。因此被认为并没有受到周永康事件的影响。
 <br/>
 <br/>
 而此次，中共突然大动干戈扣押孟宏伟，透露出中共不顾及国际法和颜面，坚决逮捕孟宏伟的决心。
 <br/>
 <br/>
 日媒认为，中共公安部掌握众多机密，其中包括涉及中共高层的相关秘密。消息人士猜测，孟宏伟拥有中共重要人物违法的证据。可以说，公安部的特殊地位，是孟宏伟突然被查的一个重要背景。
 <br/>
 <br/>
 中共公安部过去20年来，一直被江派利益集团把持。从1998年3月开始，公安部部长先后由江派大员贾春旺、周永康、孟建柱、郭声琨担任。2017年，中共十九大上，郭声琨升任政法委书记后，其空缺被中共河北省委书记赵克志接替。
 <br/>
 <br/>
 赵克志上任后，多次要求公安部继续清洗周永康的〝余毒〞。
 <br/>
 <br/>
 中共十八大至今，北京当局已拿下了包括前政法委书记周永康、时任公安部副部长李东生、公安部前常务副部长杨焕宁、公安部政治部主任夏崇源，重庆市公安局长何挺等多名副部级以上官员。
 <br/>
 <br/>
 同时还罢免了公安部副部长陈智敏，调离了傅政华、黄明等多名副部长，他们多是周永康的心腹。
 <br/>
 <br/>
 此次孟宏伟的中共公安部副部长一职，9日已由习近平的旧部邓卫平接任。至此，在公安部高层中，已有6人属于习近平阵营的人马。包括公安部部长赵克志，公安部常务副部长王小洪，副部长孟庆丰、许甘露，及中纪委驻公安部纪检组组长邓卫平等。
 <br/>
 <br/>
 外界预计，随着孟宏伟的落马，中共公安部的人马将被再次掀起一阵清洗高潮，直到习阵营全面接管公安部高层为止。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
