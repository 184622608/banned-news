### 【微视频】国际刑警主席被中共双规
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】下面请看赵培为我们带来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微视频.html" target="_blank">
  微视频
 </a>
 。
 <br/>
 <br/>
 赵培：2018年10月5号媒体传出消息，国际刑警组织的主席、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 公安部副部长孟宏伟的妻子在里昂报警说孟宏伟回国之后失踪。法国警察办事真认真，确认了孟宏伟不是在法国境内失踪之后断定是在大陆失踪的。
 <br/>
 <br/>
 10月7号，孟的妻子召开记者会详细说明孟宏伟的失踪情况。孟宏伟9月下旬前往大陆，25号他发给妻子的第一份简讯说，〝等我的电话〞。随后的第二个信息是一把刀子。自此之后，孟宏伟没有任何消息，然后孟的妻子报警说，孟宏伟失踪了。
 <br/>
 <br/>
 在孟宏伟妻子的新闻发布会不到1个小时的时间里，中共中央纪委国家监委网站发布消息说，〝公安部副部长孟宏伟涉嫌违法，目前正接受国家监委监察调查〞，这个意思就是被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双规.html" target="_blank">
  双规
 </a>
 了。国际刑警组织也收到孟宏伟的辞职书，并表示了〝立即生效〞。这份辞职书的真假不重要，但是中共的意思很明显了，这是要国际刑警组织不要插手孟宏伟的事情。
 <br/>
 <br/>
 在这段时间里，更多关于孟宏伟的背景被挖掘出来了。香港媒体说，孟宏伟自2004年开始长期任公安部副部长，并且认为与当时的政法委书记周永康来往密切。2015年周永康被判刑之后，孟宏伟也开始被削权。2017年12月被免去中共国家海洋局副局长、中共海警局局长的兼职。2018年4月又被免去公安部党委委员一职。
 <br/>
 <br/>
 孟宏伟在周永康手下也参与了迫害法轮功学员的人权犯罪行为，被明慧网列入迫害法轮功学员的涉案责任人之一。2016年11月，孟宏伟当选国际刑警组织主席，任期四年。他也是国际刑警组织自成立以来的一位来自中共的主席。这么一个人权罪犯成了刑警的主席，这本身就是一件讽刺的事件，现在还被双规了，国际刑警组织的脸还往哪搁呀，一个中共贪腐罪犯代表的组织。这也是给所有国际组织，这里包括联合国、世界贸易组织、国际货币基金组织提个醒，与中共合作就是自我毁灭。这个事也说明，川普退出某些国际组织的行为是正确的。
 <br/>
 <br/>
 在这种背景下，孟宏伟被双规就不是什么惊奇的事情。孟宏伟妻子报警失踪甚至是举行记者会的举动就是摆明了要把事情闹大，目的就是救孟宏伟。如果国际上的压力足够大，中共又不能及时让孟宏伟认罪，丈夫就保住了。这也能解释孟宏伟发简讯的含义了，如果一切正常，就是打电话，如果被抓就发给刀子，他的妻子接到刀子的短息立马报警把事情闹大。
 <br/>
 <br/>
 孟宏伟的妻子说自己的行为是，〝为祖国人民，为所有的妻子的丈夫和孩子的父亲不再消失〞。我要说的是，欢迎来到地球，地球上的中共不是中国人的祖国，这里有道德的人比如说法轮功学员就是会非自然消失，他们或许被关、被杀甚至是被活摘器官；这里有正义感的人比如高智晟等维权律师也一直在失踪状态，他们的妻子也在奔走呼吁。这一切的罪魁祸首就是中共，而帮凶就是你的丈夫和周永康这些人。中国人一直忙着救孩子、丈夫、妻子和亲人，如果有空，请把您知道的中共恶行公布与众，这才能帮助正义。
</div>
