### 孟宏伟另一重要身份被解除 中共官媒罕见通报
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被查事件一度引发国际舆论哗然，中共官方对于孟宏伟案则一直讳莫如深。近日，中共官媒罕见通报，孟宏伟又一重要身份被解除。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 在经历〝被失踪〞、〝被辞职〞、被调查受贿后，至今音讯全无。11月30日，中共官方罕见通报，孟宏伟又一重要身份被解除。
 <br/>
 <br/>
 当天，中共全国政协举行第十一次主席会议，取消了孟宏伟的政协委员资格。但是中共当局没有正式公开宣布这一决定。
 <br/>
 <br/>
 12月1日，中共官媒在关于政协会议的报导中，提到会议〝审议通过撤销孟宏伟第十三届全国政协委员资格的决定〞。这是中共当局在宣布孟宏伟被调查，辞去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 职务之后，官媒首次提及有关孟宏伟的消息。
 <br/>
 <br/>
 孟宏伟自9月29日从法国前往中国大陆神秘消失后，孟妻于10月5日向法国警方报案，引发国际舆论界的哗然。在国际舆论的压力之下，2天后（10月7日），中共中纪委深夜通报，孟宏伟涉嫌违法，正接受调查。
 <br/>
 <br/>
 10月8日，中共外交部新闻发言人陆慷在例行记者会上表示：〝孟宏伟收受贿赂、涉嫌违法，本人已书面提交辞去国际刑警组织主席的辞呈。中国（共）公安部已正式通报国际刑警组织总秘书处。〞
 <br/>
 <br/>
 孟宏伟事件浮出水面后，在中共官方告示里，仍保留孟宏伟公安部副部长官职的称呼。直到11月7日，中共官网显示，该职务已被人替换。
 <br/>
 <br/>
 中国人力资源和社会保障部官网11月7日发布通告称，中共国务院任免国家工作人员。其中写道：〝任命林锐为中国（共）公安部副部长。〞这意味着孟宏伟已被免职。孟宏伟目前被拘押何处，外界仍无从知晓。
 <br/>
 <br/>
 孟宏伟离奇失踪被调查事件，曾一度引发国际舆论哗然，外界猜测，北京当局不顾及国际影响，突然抓捕孟宏伟这一大动作背后，应有重大隐情。
 <br/>
 <br/>
 据悉，孟宏伟曾任中共公安部副部长长达14年，期间担任部长的周永康，是江派利益集团的核心人物，〝政法系〞被称为周的〝独立王国〞。外界据此推测，孟宏伟很可能是作为周永康〝余毒〞被清洗。
 <br/>
 <br/>
 有海外媒体分析，作为浸淫中共公安系统多年的高官，孟宏伟的落马或将牵动中共公安系统，甚至整个政法系统的又一轮反腐风暴。孟宏伟被捕的消息公布后，中共公安部连夜召开会议通报事件，并宣布查处孟的同党。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
