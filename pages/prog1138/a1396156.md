### 赖小民香港情妇一次受贿上亿元
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】华融董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 落马后，被曝在香港拥有情妇和两私生子。中共官媒20日披露，赖在香港确实有情妇，而且一次就敢收受上亿元人民币贿赂。
 <br/>
 <br/>
 北京时间10月20日，中共官媒《中国经济周刊》在官方微信公众号上披露，中国华融资产管理股份有限公司原党委书记、董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 的贪腐细节，称其为〝金融贪腐第一案〞。
 <br/>
 <br/>
 现年56岁的赖小民，长期在中共央行工作，曾任央行银行监管二司副司长、银监会办公厅主任，2009年起担任华融总裁、董事长。华融作为中国最大的资产管理公司，其实际控制人为中共财政部。
 <br/>
 <br/>
 文章引用业内人士的话称，华融出事是操控和失控交织的产物，赖小民的胆大妄为超出常人想像。指赖一心想在中共央行或原中共银监会再谋高职，华融为其提供了利益输送的平台。由此可见，赖小民案或牵涉金融高层。
 <br/>
 <br/>
 此前已有海外中文媒体披露，赖小民与已落马的重庆前市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙政才.html" target="_blank">
  孙政才
 </a>
 关系密切，孙政才2012年调任重庆后，赖小民多次拜会孙政才和时任重庆市长黄奇帆。
 <br/>
 <br/>
 赖小民主掌的华融在重庆投资金额达40亿元，并在重庆成立华融渝富股权投资基金管理公司。赖小民落马后，他与孙政才、黄奇帆的合照再次被媒体翻出。
 <br/>
 <br/>
 10月15日，中纪委宣布对赖小民开除党籍、开除公职的处分决定，通报中指赖小民〝擅权妄为、腐化堕落、道德败坏、生活奢靡，甘于被‘围猎’，严重违反党的纪律〞。
 <br/>
 <br/>
 通报还用了三个〝特别〞，即〝群众反映特别强烈、腐败问题特别严重、性质特别恶劣。〞
 <br/>
 <br/>
 赖小民落马后，被媒体曝光其〝3个100〞，即100多套房、100多个关系人、100多个情妇。有关部门已查明，赖小民通过华融置业等子公司，在广东省珠海市开发了某地产项目，一共120套房，其中100套房全部通过假摇号，分给了他的前妻及诸多情妇。
 <br/>
 <br/>
 《中国经济周刊》透露，赖小民在香港确实有情妇，而且一次就敢收受上亿元人民币贿赂。
 <br/>
 <br/>
 据陆媒财新网此前报导，赖小民被调查期间，其在北京的多处房产中共搜出2.7亿元人民币现金，重达3吨。
 <br/>
 <br/>
 16日，财新网又披露，赖小民母亲账上还有3亿元人民币存款，是华融内外人士逢年过节以各种方式〝孝敬〞的结果。
 <br/>
 <br/>
 今年4月17日早上，赖小民到中共银保监会开会时，当场被官方宣布接受调查，随后中纪委通报了他落马的消息。
 <br/>
 <br/>
 据21世纪经济报导引述不同渠道的消息说，赖小民落马的原因很可能是涉嫌与多家民企存在利益输送。赖小民与落马的安邦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴小晖.html" target="_blank">
  吴小晖
 </a>
 、明天系掌门人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肖建华.html" target="_blank">
  肖建华
 </a>
 、华信董事会主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叶简明.html" target="_blank">
  叶简明
 </a>
 等多名江派〝白手套〞都有生意往来，他们都持有华融的股份。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
