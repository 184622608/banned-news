### 中纪委部署大动作 贸战迫使北京向金融界开刀
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 书记赵乐际日前出席会议，宣布中纪委下一步将向中管
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融企业.html" target="_blank">
  金融企业
 </a>
 派驻纪检监察组。有学者分析，由贸易战引发的中共经济问题，激化了中共内部分裂，面对经济下滑的棘手问题，金融界不可避免成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_权力斗争.html" target="_blank">
  权力斗争
 </a>
 的主战场。
 <br/>
 <br/>
 北京时间11月2日，赵乐际在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 国家监委派驻机构改革动员部署会上说，开展向中管
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融企业.html" target="_blank">
  金融企业
 </a>
 派驻纪检监察组的各项工作，将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中管企业.html" target="_blank">
  中管企业
 </a>
 、党委书记和校长列入中央管理的高校纪检监察体制改革。
 <br/>
 <br/>
 对于〝向中管金融企业派驻纪检监察组〞，熟悉中共政情的人士称，这可是个重大调整，按照赵乐际的说法，中共十几家中管金融企业将要被安上〝探头〞。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 令中共经济面临危局的背景下，中纪委对金融界〝磨刀霍霍〞，引发外界关注。而在中纪委部署
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反腐.html" target="_blank">
  反腐
 </a>
 工作会议之前，10月31日，习近平亲自主持召开中共中央政治局会议，分析研究当前经济形势，部署当前经济工作。中共官方罕见承认〝经济下行压力增大〞。
 <br/>
 <br/>
 外界认为，这显示中共经济问题已严重到无法掩盖。金融界或不可避免成为中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_权力斗争.html" target="_blank">
  权力斗争
 </a>
 的主战场。
 <br/>
 <br/>
 自中共十八大以来，金融反腐一直保持高压状态，金融领域有多名高管落马。包括原中共保监会原主席项俊波、中国保险集团股份有限公司原总裁王银成、中共证监会原副主席姚刚、中共证监会原主席助理张育军、中国出口信用保险公司原副总经理戴春宁、农行原副行长杨琨、以及原银监会党委委员、主席助理杨家才。
 <br/>
 <br/>
 今年4月，原中国华融董事长赖小民落马，成为中共史上第一大金融腐败案。陆媒报导称，赖小民落马或许只是中共金融反腐的开始。早在今年1月，中纪委全会已释放反腐新动向，首次将金融信贷作为〝重点领域和关键环节的腐败问题〞，意味着金融领域的反腐将向纵深发展。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
</div>
