### 刘强东已不是习近平〝自己人〞？接连缺席重要活动
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】11月7日，第五届世界互联网大会在浙江乌镇开幕。性侵丑闻缠身的电商巨头京东集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 再次缺席。梳理发现，在过去60多天里，刘几乎缺席所有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重磅会议.html" target="_blank">
  重磅会议
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重要场合.html" target="_blank">
  重要场合
 </a>
 。特别是在11月初，刘已被踢出习近平的〝自己人〞圈子。
 <br/>
 <br/>
 11月7日，中国第五届乌镇世界互联网大会开幕。中国互联网巨头阿里巴巴创始人马云、腾讯集团马化腾、小米集团雷军等，多位中国互联网巨头大佬现身乌镇。但独缺性侵丑闻缠身的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 。
 <br/>
 <br/>
 舆论纷纷猜测，刘强东遭北京〝抛弃〞了，在过去60多天里，刘强东除了在案发初期现身京东集团〝正常开展工作〞，以及10月中旬亮相英国王室婚礼外，几乎缺席所有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重要场合.html" target="_blank">
  重要场合
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重磅会议.html" target="_blank">
  重磅会议
 </a>
 。
 <br/>
 <br/>
 9月17日，2018世界人工智慧大会在上海开幕，马云、马化腾、李彦宏、雷军等众网际网路大佬云集，京东方面称，刘强东将缺席本次大会，但未说明具体原因。
 <br/>
 <br/>
 9月18日至20日，天津2018夏季达沃斯论坛上，依然没有刘强东的身影，而在年初的2018冬季达沃斯论坛，刘强东携妻子章泽天在达沃斯开专场会议、秀英文、宴宾客。
 <br/>
 <br/>
 10月24日，中共官方发布《改革开放40年百名杰出民营企业家名单》，鼓励民企老板〝不忘创业初心〞。名单中也没有刘强东。
 <br/>
 <br/>
 11月1日，习近平主持召开民营企业座谈会，安抚所有民营企业和民营企业家，并称他们都是〝自己人〞。而刘强东仍缺席此次重要活动。由此可见，刘强东已被踢出习近平的〝自己人〞圈子。
 <br/>
 <br/>
 而这一切，源于刘强东在美国涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 被捕丑闻。
 <br/>
 <br/>
 据美国警方公布的报告，刘强东涉嫌强奸发生在8月31日凌晨1点左右，当晚被捕入狱，16个小时后获释。警方对刘强东一案的描述是〝一级性犯罪：强奸既遂〞。
 <br/>
 <br/>
 美国警方9月21日已初步完成调查案件移交检方。美国检方表示，这是一个〝棘手的案子〞案件，仍在审查中。有华裔刑案律师称，刘强东遭起诉的可能性超过90%，涉及重新逮捕等问题。
 <br/>
 <br/>
 11月2日，美媒披露刘强东案的具体细节，报导称刘强东在性侵受害者时，遭到多次反抗，最后仍得逞。刘还告诉21岁的受害女大学生，可以像中共头号女间谍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓文迪.html" target="_blank">
  邓文迪
 </a>
 一样。
 <br/>
 <br/>
 随着刘强东性侵丑闻发酵，京东集团的股价也急转直下，持股机构数量也暴跌。2018年第二季度的持股机构达585家，而第三季度仅有100家。
 <br/>
 <br/>
 今年45岁的刘强东不仅是中国商界亿万富豪，同时也是中共全国政协委员，并兼任中国民间商会副会长。作为中共的红顶商人，在美国涉及强奸犯罪指控，无疑也是中共一大丑闻。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
