### 滕彪：欲盖弥彰的暴行
------------------------

<div class="wysiwyg">
 中共在东突厥斯坦（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 ）的大规模侵犯人权被媒体和人权组织披露后，当局在相当长的时间里装聋作哑、百般抵赖，今年八月在回应联合国种族歧视问题小组的质询时，仍然矢口否认集中营的存在。
 <br/>
 <br/>
 但无数的事实和证据在迅速积累，国际社会越来越认识到了事态的极端严重性。BBC、华尔街日报、纽约时报等做了大量的调查、报道；人权观察、大赦国际、维吾尔人权项目、中国人权捍卫者（CHRD）等诸多人权机构发布了详细的研究报告；不少集中营的受害者讲述了他们的亲身经历，失踪者和被关押者的家属向外界讲述他们亲人的遭遇，他们都冒着遭受进一步迫害的危险；很多人通过自媒体账号发布图片、视频和文字信息，尤其值得一提的是在加拿大的中国留学生Shawn Zhang，根据卫星图片发现了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 集中营的迅速扩张，并不断在推特上发布。 华盛顿大学的Darren Byler教授发表长文揭示了在新疆的〝人类改造工程〞的另一个方面：在超过一百万维吾尔人被送进集中营的同时，大约一百万汉人以〝亲戚〞身份、以脱贫或传授〝文明生活〞等名义，带着监控任务进驻维吾尔人家庭，时间从几周到一年以上不等。
 <br/>
 <br/>
 这些信息不可辩驳地指向一个事实：中共正在新疆实施大规模的种族迫害和文化屠杀。这个事实得到了美国、欧盟、其他很多西方国家以及联合国的认可。
 <br/>
 <br/>
 中共无法抵赖，转而进行掩饰和狡辩。最重要的一个借口，是声称那些只是职业教育培训。但效果却是欲盖弥彰。
 <br/>
 <br/>
 美国记者Megha Rajagopalan（李香梅）因为发表多篇有关中共当局严密监控维吾尔人的报导，而被拒绝签证；留学生Shawn Zhang在中国的家人受到警察威胁；美国公民鲁珊・阿巴斯(Rushan Abbas)在华盛顿一家智库发表演讲之后，她的生病的姐姐和64岁的阿姨在中国西北的家中失踪。维吾尔人和穆斯林少数民族受到警告不得与外界联系，发布信息的人被关押甚至判刑，很多生活在海外的维吾尔人与新疆家人的联系被彻底切断。一切海外媒体受到阻拦，关键道路全被封锁 - 如果没有发生见不得人的巨大暴行，为什么要如此费力掩盖？
 <br/>
 <br/>
 10月16日中共喉舌央视的〝焦点访谈〞作了一期节目，极力美化所谓〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育中心.html" target="_blank">
  再教育中心
 </a>
 〞。如果把它与纳粹美化集中营的宣传片放在一起，可以发现很多相似之处。极权主义的宣传手腕、美学趣味和语言风格都如出一辙。
 <br/>
 <br/>
 〝学员们〞穿着像极了囚服的统一服装，学习汉语课本，学习《去极端化条例》，听干部宣讲〝党的政策〞；精心挑选的受访者说着一些感恩戴德的话，唱着红歌、跳着强烈的共产风格的舞蹈，室内的摄像头 - 这一切都透露着与〝职业培训〞毫无关系的诡异气息。
 <br/>
 <br/>
 在另一张被广泛转载的官方照片里，近千人穿着囚服整齐地端坐在铁丝网围着的地面上。这是什么性质的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_职业培训中心.html" target="_blank">
  职业培训中心
 </a>
 〞？
 <br/>
 <br/>
 法新社查阅了1500份新疆政府公开文件，分析出新疆至少有181座再教育营；和田负责管理营区的部门，在其采购的清单上包括大量警棍、电击棒、手铐、胡椒喷雾 - 这是怎样的〝职业培训中心〞？
 <br/>
 <br/>
 细心的人们从公开发布的建筑工程投标、招工启事也能发现与集中营有关的重要线索。
 <br/>
 <br/>
 《环球时报》的主编胡锡进在中共屏蔽的推特上，频频发布洗地文章。10月17日，他的一条英文推特值得注意，他写道：〝我不认为新疆的‘职业培训中心’的‘被培训者’是自愿去的，但他们在那里得到了真诚的教育和训练，以便将来重返正常生活。这是让新疆稳定的成本最低的方式。〞一贯为中共邪恶暴行辩护的胡主编这回也犯了愚蠢的错误：在中共至今仍完全否认强迫关押的情况下，这种说法等于承认了所谓的培训全属强迫。胡锡进还说，他知道人数有多少，远远不到100万，但他拒绝透露具体数字。
 <br/>
 <br/>
 罪行累累，欲盖弥彰。〝墨写的谎话，绝掩不住血写的事实〞。中共不会不知道：一个谎言需要一千个谎言来掩饰。我们接下来还会看到中共的各种理论和借口、各种谎言和表演，来淡化、美化、正当化他们的反人类暴行。
 <br/>
 <br/>
 ──转自《自由亚洲电台》本文只代表作者的观点和陈述。
 <br/>
 <br/>
