### 上海进博会 胡春华再添一新头衔
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】中共首届〝国际进口博览会〞正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海.html" target="_blank">
  上海
 </a>
 召开，陪同
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 会见境外企业家代表的中共国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡春华.html" target="_blank">
  胡春华
 </a>
 ，又添〝中俄总理定期会晤委员会中方主席〞新头衔。自中共两会担任副总理后，胡春华已接任多个兼职，显示其角色吃重。
 <br/>
 <br/>
 党媒报导，首届〝国际进口博览会〞（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 ）5日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海.html" target="_blank">
  上海
 </a>
 开幕，中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，会见了参加进博会的境外企业家代表。中共国家副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡春华.html" target="_blank">
  胡春华
 </a>
 、中共外长王毅、公安部长赵克志、中共办公厅主任丁薛祥、上海市委书记李强、国家发改委主任何立峰等人参与了会见。
 <br/>
 <br/>
 报导称，11月6日，中共国务院副总理、中俄总理定期会晤委员会中方主席胡春华，与俄罗斯副总理、委员会俄方主席阿基莫夫（Maxim Akimov）共同主持中俄总理定期会晤委员会第22次会议。
 <br/>
 <br/>
 上述消息显示，胡春华又添一〝中俄总理定期会晤委员会中方主席〞新头衔。该头衔此前由时任副总理、现任中共政协主席汪洋担任。
 <br/>
 <br/>
 胡春华在今年两会上担任中共副总理后，已接任多个兼职，包括中共国务院扶贫开发领导小组组长等，前任汪洋的大部分工作，农林、扶贫、水利等事务也已接管。但商贸中最重要的中美全面经济对话，由在副总理中排名靠后的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 负责。
 <br/>
 <br/>
 在此之前，世界经济论坛东盟会议9月11日至13日在越南河内召开，胡春华出席了此次峰会开幕式并致辞。由于参与这次会议的多是东盟各国的最高领导人，香港分析家刘斯路对中央社分析说，胡春华以副总理身份出席最高领导人与会的论坛，似乎有点不寻常。
 <br/>
 <br/>
 他认为，从胡春华参加如此高级别的论坛看，其工作份量和重任正在增加，但未来胡春华的动向如何，还要看他与习近平之间的互动。
 <br/>
 <br/>
 中共十九大前夕，江派选定的中共第六代接班人、前重庆市委书记孙政才落马后，在中共十九大上习近平没有按惯例安排接班人选，同样有第六代接班人之称的胡春华，也意外没有进入中共政治局常委。但与孙政才的命运截然不同，属胡锦涛阵营的胡春华，在仕途上的过渡有惊无险。
 <br/>
 <br/>
 分析认为，胡春华进京任副总理，是循了汪洋的旧路，从广东省委书记调任副总理，分管范围也与当年汪洋高度近似。虽然胡春华〝接班失败〞，但与汪洋相比并不亏，胡春华55岁进京，比当年汪洋年轻3岁，有年龄上的优势。
 <br/>
 <br/>
 从胡春华的分工没有缩水来看，习近平当初处理孙政才，除了孙出工不出力、怀有二心外，对胡春华的联动作用，应该就只限于把他挡在政治局常委会之外，所以胡春华如今才会获得大致完整的副总理权柄。
 <br/>
 <br/>
 而商贸中最重要的中美全面经济对话由刘鹤负责，分析认为，是因这块骨头太难啃，但对中南海来讲又太重要。所以必须找个经贸方面的强手去接招，但今年以来中美贸易经过4轮谈论仍无果，近几个月来，贸易战已经令北京处于严峻的内忧外患困境。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
