### 河南鹤壁副公安局长张建明落马
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】中共河南纪监委网站日前通报，河南省鹤壁市公安局副局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张建明.html" target="_blank">
  张建明
 </a>
 ，因〝涉嫌严重违纪违法〞，目前正接受调查。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张建明.html" target="_blank">
  张建明
 </a>
 2004年起，任鹤壁市公安局刑侦支队副支队长、队长等职；2009年12月起，任鹤壁市公安局副局长、党委委员，期间还兼任淇县公安局党委书记、局长，正县级侦查员，调研员。
 <br/>
 <br/>
 官方没有公布张建明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 的具体原因。讽刺的是，张建明曾被授予河南省〝优秀人民警察〞等称号。
 <br/>
 <br/>
 另据《明慧网》报导，张建明在鹤壁市公安局、淇县公安局任职期间，对参与迫害多名当地法轮功学员负有责任。
 <br/>
 <br/>
 海外〝追查迫害法轮功国际组织〞也将其列入参与迫害责任人名单，进行追查。
</div>
