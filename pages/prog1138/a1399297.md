### 美媒万字长文揭刘强东性侵案内幕：白天上课晚上笙歌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东CEO.html" target="_blank">
  京东CEO
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 在美国涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女留学生.html" target="_blank">
  女留学生
 </a>
 一案，持续发酵。有美媒13日披露刘强东一案的更多细节。称而刘强东是个早上在校上课，晚上去参加派对的富商。
 <br/>
 <br/>
 美国《纽约时报》当地时间11月13日发万字称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东CEO.html" target="_blank">
  京东CEO
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 所参加的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_明尼苏达大学.html" target="_blank">
  明尼苏达大学
 </a>
 博士项目，是学校的赚钱机器，学生都是商界巨头。刘强东是明尼苏达大学卡森管理学院学生，到明尼亚波利斯，是为了履行美中企管博士计划的在美居留规定。
 <br/>
 <br/>
 但负责管理该课程的卡尔森管理学院拒绝评论刘强东性侵案一事。
 <br/>
 <br/>
 文章披露刘强东所参加的博士项目，是该学院专门为亚洲高管设计，从去年开始参与该项目的亚洲高管，为该学院，带来超过1,000万美元学费收入，每个学生的学费超过8.5万美元。涉事女生是该项目的志愿者。
 <br/>
 <br/>
 明尼苏达大学一向颇受中国留学生青睐。目前学校大约有3,000多名，分别来自中国大陆、香港和台湾地区的留学生。学校里有十几个社团是面向中国留学生的，包括戏剧协会、音乐俱乐部和基督教团契。
 <br/>
 <br/>
 和其他学校的类似课程一样，明尼苏达大学的全球工商管理博士课程，也适合像刘强东这样，希望以高级学位给自己镀金的有钱人。截至今年10月初，约有236名学生参与了该课程。
 <br/>
 <br/>
 课程与北京的清华大学合作，主要在中国授课。但课程的第二年，学生将前往明尼苏达学习一周。对于刘强东参加的这个班级，学校为他们安排了讲座和娱乐活动，包括自助晚宴与橄榄球比赛。
 <br/>
 <br/>
 刘强东和卡尔森课程的其他高管，今年夏天，还参加了家族财富管理和全球品牌拓展等主题课程，并参观了3M和General Mills等当地公司，晚上是娱乐活动，包括夜游密西西比河和明尼托卡湖。
 <br/>
 <br/>
 网站DBA Compass的创始人托马斯・格拉夫（Thomas Graf）说，这样的课程项目于大学而言更像是一笔交易。〝我们为您提供给正规的头衔，您给钱，我们给您任务。〞
 <br/>
 <br/>
 长文披露，明尼苏达大学有超过3,000名中国学生，校园内遍布珍珠奶茶店、火锅店和中国超市，到处都能听到中文；而且明尼苏达机场的指示牌，第二语言是中文。
 <br/>
 <br/>
 刘强东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 案发生于今年8月底，但他被捕仅16个小时之后被释放，并返回中国。校方也没有公开讲述过事件。学生Ellen Schneider说，如果不是她在学校专栏提及刘强东的案件，都没有人会知悉事件，她为校方没有作出适当的立场十分失望。
 <br/>
 <br/>
 根据餐厅当晚员工的描述，以及涉事女生的微信内容显示，8月30日晚，包括刘强东和涉事女生在内的一行人，在一家名为Origami的日本餐厅用餐。
 <br/>
 <br/>
 他们从附近的酒类商店Lake Wine &amp; Spirits购买了数十瓶红酒。两名店员证实收据为真，收据显示，他们一共购买了近数千美元的红酒。
 <br/>
 <br/>
 第二天，涉事女生给朋友发微信称，自己在晚宴后遭到刘强东的强奸。她当时〝喝得很醉〞，俩人坐在豪华轿车后座时，她没能阻止刘强东对她动手动脚。
 <br/>
 <br/>
 涉事女生也告诉警方，刘强东在她自己的公寓强奸了她。她曾多次恳求对方〝不要这样做〞。
 <br/>
 <br/>
 美国明尼阿波利斯警方9月初发布的消息，刘强东在8月31日是以〝强奸〞的重罪遭到逮捕。根据当地法律，若嫌犯被正式定罪，将面临12年至30年的监禁。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
