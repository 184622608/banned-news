### 14名福彩官员贪污千亿？中纪委：具体不便公开
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】中国福利彩票发行管理中心前主任、副主任先后落马后，网路流传该中心已落马的14名局处级官员，共贪污1,360亿人民币福彩资金。13日，中纪委纪检组回应相关传闻说是谣言，具体数据不便公开。
 <br/>
 <br/>
 陆媒11月13日报导说，中纪委驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民政部.html" target="_blank">
  民政部
 </a>
 纪检组，回应有关福彩管理机构14名官员贪污1,360亿元（人民币，下同）传闻称，网传的消息均为谣言，不过，这些贪官到底贪污了多少，官方称〝具体数据尚不便公开〞。
 <br/>
 <br/>
 中纪委监委网站11月9日发布，中国福利彩票发行管理中心原主任鲍学全、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王素英.html" target="_blank">
  王素英
 </a>
 ，原副主任王云戈、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冯立志.html" target="_blank">
  冯立志
 </a>
 等4名原负责人的忏悔视频。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福彩中心.html" target="_blank">
  福彩中心
 </a>
 前副主任冯立志10月31日被调查；福彩中心前主任王素英9月12日落马。至此，自十八大以来，该中心已有14名局处级官员因涉违纪违法问题而被查，4名民政部部级官员被问责。
 <br/>
 <br/>
 由于事关民众利益，福利彩票系统的腐败案受到极大的关注。随后中国网路流传出，〝14名局处级官员贪污1,360亿福彩资金〞的传闻。
 <br/>
 <br/>
 福彩中心是中共民政部直属事业单位，负责全国的福利彩票发行和组织销售工作。中彩在线公司是福利彩票重要票种之一〝中福在线〞的独家运营商。福彩中心是中彩在线公司的最大股东。
 <br/>
 <br/>
 尽管中纪委否认14名官员贪污千亿元，但官方此前披露的数字显示，涉及的金额巨大。
 <br/>
 <br/>
 《经济参考报》3年曾曝光，中彩在线公司已由名义上的国有控股企业，悄然转变为高管掌控的个人〝财富帝国〞，该公司总经理贺文利用职权隐瞒监管部门向其〝关联方〞输送利益，涉及金额达数十亿元。
 <br/>
 <br/>
 2002年至2014年，中福在线的总销售额超过1,300亿元。贺文通过中彩在线公司获得〝管理费〞27亿元，而代表中共国家的〝福彩中心〞却只拿到18亿元。福彩中心主任鲍学全与贺文之间有巨额经济往来。
 <br/>
 <br/>
 另外，涉及民生、福利的中共民政部前民政部部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李立国.html" target="_blank">
  李立国
 </a>
 、副部长窦玉沛，被指涉嫌卷入逾2,000亿元的贪腐大案。
 <br/>
 <br/>
 李立国自2005年7月至2016年7月通过挪用公款、侵吞彩票资金，送给11名情妇逾11亿2,470余万元。而且把15套在北京、天津、珠海的住宅送给其中9名情妇。
 <br/>
 <br/>
 2017年，经相关部门查证、审计出25个省（区、直辖市）福利彩票涉嫌做假、挪用、侵吞资金，数目达1,700余亿元。由此推算，在中国30多个省份中，民政部在福利彩票一项中的贪腐金额极有可能超过2,000亿元。
 <br/>
 <br/>
 利用彩票主管地位而〝吃〞彩票的情况屡见不鲜。包括上述文章所提到的鲍学全、王云戈等在内，据不完全统计发现，近年来被调查的彩票系统厅级、处级官员，已有30多人。
 <br/>
 <br/>
 《经济参考报》记者王文志3年前要一篇调查报导称，中彩在线独家运营〝中福在线〞已沦为〝私人领地〞，负责即开型彩票〝中福在线〞的中彩在线公司，名义上是一家由民政部福彩中心控制的国企，实际上从成立之初就被一个叫贺文的自然人控制。
 <br/>
 <br/>
 观察人士分析认为，时任高层某个当权派决定吃下这块肥肉，遂由贺文作为代理人负责具体操办。贺文只是一个提线木偶，横财是幕后权贵操纵者的。中共民政部也是江派二号人物曾庆红的基本地盘、势力范围之一。
 <br/>
 <br/>
 如今当局要撕开年销量上千亿元的福彩黑幕，被 认为主要也是挖背后的人。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
