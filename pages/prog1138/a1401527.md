### 军队整肃重磅消息  传3名上将同日被抓
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】中共军队正在进行新一轮整肃，继日前传出120名中共将领正在接受反腐调查使军中已出现极大内乱之后，近日外媒再曝出重磅消息：3名军中上将同日被抓。但有关消息仍未获官方证实。
 <br/>
 <br/>
 11月30日，包括联合新闻网等多家海外媒体报导称，11月29日同一天被抓的3名中共军中上将，都已退出现役，这3名上将是前海军司令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴胜利.html" target="_blank">
  吴胜利
 </a>
 、前南京军区司令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英挺.html" target="_blank">
  蔡英挺
 </a>
 、前中央军委装备发展部政委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王洪尧.html" target="_blank">
  王洪尧
 </a>
 。
 <br/>
 <br/>
 报导说，三人中，吴胜利多次传出被抓，但事后都以亮相辟谣，原因是，吴胜利是浙江省太子党，曾在海军福建基地担任过司令，所以不是有人说情，就是上面念旧情，次次刀举起又放下。
 <br/>
 <br/>
 三人中的蔡英挺，2016年初已被免去南京军区司令要职，调任军事科学院院长閒差，是当时军改中唯一转任閒职的大区司令，也是军方唯一未到龄就被要求退休的上将，传说被连降十级。
 <br/>
 <br/>
 之所以被贬，当时的说法是，蔡英挺是前中共军委副主席张万年的秘书，而且他与前军委副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐才厚.html" target="_blank">
  徐才厚
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 两人过从甚密，他女儿嫁了法国人。
 <br/>
 <br/>
 三人中的王洪尧，是山东农村兵老行伍出身，辗转多个集团军任职，因搭上徐才厚而步步高升，最后升到总装备部政委，军改又平安坐直通车进军委装备发展部政委。直到有人举报才被迫脱军装，到中共人大常委会任环资委副主任委员。
 <br/>
 <br/>
 对于上述三人同日被抓的消息，目前还没得到中共官方证实。
 <br/>
 <br/>
 自央视8月晚间播出习近平在中共军委党建会议上，强调从严治理军队腐败，反腐不会〝变风转向〞之后，8月下旬起，就陆续传出有中共将领落马。
 <br/>
 <br/>
 其中包括原总参情报部部长、东部战区副司令兼参谋长杨晖中将被传落马，以及西部战区副司令刘小午被传已被拘查。还有南部战区政委魏亮和原广州军区司令徐粉林被降级退役等。
 <br/>
 <br/>
 而上述四人均与已落马的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军老虎.html" target="_blank">
  军老虎
 </a>
 郭伯雄、徐才厚、张阳、房峰辉等人有关联，被认为是清除他们在军中余毒的行动之一。
 <br/>
 <br/>
 10月16日，中共通报了中共前军委委员张阳和房峰辉的审查结果，两人均被开除党籍、军籍和取消上将军衔，房峰辉被移送起诉；已自杀死亡的张阳被追缴赃款。
 <br/>
 <br/>
 10月19日，有美媒称，随着房峰辉移送审判，中共十八大以来，已有7名上将被查，其中有两个军委副主席、总参谋长、 总政治部主任、空军政委、武警司令等。
 <br/>
 <br/>
 报导引述消息人士透露，习近平已经开始进行第二轮军队清洗了，下一步首当其冲的是东海和北部战区的领导层，虽然未必是战区司令，但级别一定不低。而官方宣布将房峰辉移送起诉，对军队震动很大，中共将领们都不敢肯定自己是否能平安。
 <br/>
 <br/>
 10月24日法广消息称，由于中共军队内部120名将领正在接受反腐调查，军中已出现极大内乱。消息还说，在军中〝政治学习会〞上，针对此事新兵反覆提出许多疑问，而他们的政委们都无法解答。
 <br/>
 <br/>
 上述有关消息至今仍未获中共官方证实。不过自习近平又一次发出反腐不会〝变风转向〞信号之时，有分认为这已经意味着军队新一轮的大整肃拉开帷幕。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明）
