### 揭郭伯雄儿子被抓内幕 总政派人专机押送
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】中共中央军委前副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 之子、浙江省军区副政委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭正钢.html" target="_blank">
  郭正钢
 </a>
 ，于3年前被中共军方宣布立案侦查。有港媒曾称，郭正钢是被中共军方用军机从杭州押至北京。
 <br/>
 <br/>
 据港媒报导，2015年两会前夕，中共军方的总政保卫部派出专职军官，从杭州用专机把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭正钢.html" target="_blank">
  郭正钢
 </a>
 押解到北京。同年2月，中共军事检察机关对郭正钢立案侦查。70后的郭正钢晋升少将军衔仅46天即〝落马〞一度引舆论瞩目。
 <br/>
 <br/>
 郭正钢落马当晚，党媒曾在微信公号调侃道，〝郭正钢火了〞，〝在被查前，那可是‘坐着火箭往上飞’〞，〝今天郭正钢被查，绝对算得上是‘天上掉下个郭正钢’。〞
 <br/>
 <br/>
 资料显示，郭正钢出生于1970年1月，陕西礼泉人，19岁入伍，在职大学学历，曾任浙江舟山市委常委、警备区政委，浙江省双拥领导小组办公室主任、军区政治部副主任，2010年晋升大校军衔。2013年4月，郭正钢升任浙江省军区党委常委、政治部主任，跻身副军职将领。
 <br/>
 <br/>
 2015年2月，郭正钢因〝涉嫌违法犯罪〞被立案侦查。陆媒称，郭正钢的再婚妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴芳芳.html" target="_blank">
  吴芳芳
 </a>
 也同时被抓。而两人涉案，与吴芳芳操盘的一件涉及浙江省军区的土地案有关。
 <br/>
 <br/>
 报导称，在这个拖而未决、风波不断的项目中，吴芳芳曾获得浙江省军区下属的一块土地的租赁权，再以承诺回报的方式获大笔集资款进行开发。后因未能对付款人履约，形成巨大债务纠纷，长年遭债权人追索，浙江省军区驻地成抗议场所。
 <br/>
 <br/>
 报导披露，郭吴同时被抓，所涉事项不止于上述土地案，还有其他更严重问题在调查中。
 <br/>
 <br/>
 郭正钢落马前，自2014年起，海外中文媒体频繁出现大量
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 及其子女的贪腐黑幕。同年12月下旬，大陆微信圈还一度热传一封〝举报信〞，披露了郭伯雄贪人民币千亿军费，总参有秘密洗钱渠道等问题。消息称郭伯雄贪的军费〝粗略估算应不少于千亿元〞。
 <br/>
 <br/>
 举报信中揭露，郭正钢曾经在酒醉后，谩骂习近平掀起的军中反腐，对酒肉朋友称：〝有人想鼓捣我们家，白日做梦，全军干部一半以上是我家提拔的，都在要位上干着。〞
 <br/>
 <br/>
 郭伯雄贪赃的巨额来源，除了买官卖官卖地的〝收益〞之外，还有海量的全军采购武器装备弹药的惊人回扣，〝巨额军费早已落入郭伯雄家族囊中〞。
 <br/>
 <br/>
 公开信还披露，总参一个部的中南办事处有一条秘密免税进出口渠道。郭伯雄等人就通过这个渠道，赚了数以千亿元的黑钱。这个渠道不仅隐藏着巨额款项的大案要案，而且涉及〝国家机密〞，从中可以查找到事关国家安全方面的重大漏洞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐才厚.html" target="_blank">
  徐才厚
 </a>
 和郭伯雄都是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 在军中的代理人，两人一起替江把持军权10多年，架空了时任军委主席胡锦涛。郭、徐还阻击习近平进军委。在一个有数人在场的军方半公开场合，徐才厚曾对郭伯雄说：〝让他（指习近平）干5年就滚蛋！〞
 <br/>
 <br/>
 徐才厚于2014年3月落马，2015年3月艾滋病发死亡；郭伯雄于2015年落马，2016年7月被判终身监禁。据港媒披露，郭伯雄落马后，在秦城监狱服刑期间，不但自曝接受徐才厚转交贿款的详情，还咬出江泽民，把军队变质问题归咎于江泽民对军队的领导。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
