### 岐山书记落马 陕西官场持续震荡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】自秦岭违建别墅案曝光以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陕西.html" target="_blank">
  陕西
 </a>
 官场持续震荡。据陕西省纪委网站消息，12月7日，陕西省纪委通报，陕西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_岐山县委书记.html" target="_blank">
  岐山县委书记
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_何宏年.html" target="_blank">
  何宏年
 </a>
 涉嫌严重违纪违法，目前正接受纪律审查和监察调查。
 <br/>
 <br/>
 公开资料显示，今年49岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_何宏年.html" target="_blank">
  何宏年
 </a>
 一直在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陕西.html" target="_blank">
  陕西
 </a>
 省任职。曾先后担任过共青团宝鸡市委维权办副主任、主任、农村部部长、青年部部长，共青团宝鸡市委副书记，中共凤翔县委常委、组织部长、常务副县长，岐山县委副书记、县长，蔡家坡经济技术开发区管委会主任等职务。
 <br/>
 <br/>
 2016年5月至落马前，何宏年任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_岐山县委书记.html" target="_blank">
  岐山县委书记
 </a>
 、蔡家坡经济技术开发区党工委书记。
 <br/>
 <br/>
 目前为止，当局尚未公布何宏年因何罪名被查。不知他是否与秦岭违建别墅案有关。
 <br/>
 <br/>
 此前，因秦岭违建别墅案，陕西官场已经历一轮大地震，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上官吉庆.html" target="_blank">
  上官吉庆
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_钱引安.html" target="_blank">
  钱引安
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_程群力.html" target="_blank">
  程群力
 </a>
 、卫旭峰、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魏民洲.html" target="_blank">
  魏民洲
 </a>
 等一批高官均已受到惩处。
 <br/>
 <br/>
