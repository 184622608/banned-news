### 陈思敏：中纪委大动作与最大骗贷案的背后
------------------------

<div class="wysiwyg">
 11月2日，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 书记赵乐际出席了一个部署会。会上，赵乐际透露了中纪委下一步的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反腐.html" target="_blank">
  反腐
 </a>
 大动作，向中管金融机构——涉五大银行等15家金融业直接派驻纪检监察组。
 <br/>
 <br/>
 这次金融
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反腐.html" target="_blank">
  反腐
 </a>
 部署，正值习李二人发话关注民企融资难的问题，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 层面此一大动作也不难理解，反腐数据说明，金融机构从国有大行到省市地方银行，对企业放贷的审批是不看手续而看关系。
 <br/>
 <br/>
 刚好有个现成判案可以举例说明，11月2日，媒体聚焦报导，堪称目前规模最大的银行骗贷案——420亿中美天元集团骗贷案终于迎来了一审宣判。
 <br/>
 <br/>
 民营企业普遍融资难，而中美天元集团这家广西民企，不仅从银行骗贷420亿，而且牵涉轰动全国的杀人未遂案——2014年广西柳州银行董事长李耀清当街被砍案。
 <br/>
 <br/>
 这起凶杀与骗贷数百亿的主犯是中美天元集团掌门人吴东，而吴东背后的高官是他的岳父陆兵——广西自治区前政法委书记和政府主席，以及广西〝610〞办公室的重要负责人，也是广西指挥镇压法轮功的重要责任人。
 <br/>
 <br/>
 中美天元集团吴东案显示，陆兵虽然在2008年退居中共人大，至2013年完全退休，但一直力助女婿的事业从广西跨到了北京，跨出了国界，并在他出事后插手干预广西官场和公检法系统。
 <br/>
 <br/>
 公开报导显示，2010年3月9日，中美天元集团投资担保与石油化工公司在北京举行成立仪式，参加者除了广西的老领导，还有北京市相关部门领导，四大股份制银行等金融界代表，剪彩仪式由央视〝名嘴〞主持。中美天元的石油板块经营专案项目涉上中下游，分公司分布于北京、辽宁、湖北、广西、广东及香港和东南亚等地区。除此，当年中美天元集团下属石油公司已与沙特石油公司签署了一份百万吨炼油设备投资合作协定。
 <br/>
 <br/>
 2013年10月，广西柳州银行董事长一职由李耀清接任。履新半年期间，李耀清有眼不识吴东，挡了吴东的财路。2014年5月，李耀清当街遭到围殴砍杀。之后，从12名凶犯被缉拿、被审讯，到2016年一审判决前这段期间，据北京律师李庄微博透露：本地的被害方代理律师纷纷受到威胁，副市长兼公安局长突然被调离，主管刑侦副局长接令停止侦办此案，市长（肖文荪）投河自杀，连环大爆炸（邮包炸弹）烽火连天，……巨大压力下，当地有司不得不将故意杀人变成〝故意伤害〞，贷款诈骗改为〝骗取贷款〞。李庄微博说，这些，既匪夷所思，又触目惊心。
 <br/>
 <br/>
 有报导称，广西中美天元集团说是民企，更像是涉黑团伙，但哪个涉黑团伙有此实力？虽然对集团掌门人、凶案骗贷案主谋吴东的〝黑伞〞，媒体究没有指名道姓，只是点到〝背后的高官身影〞，但吴东和陆兵的关系，官场坊间皆知。而陆兵即便没有正式落马，但女儿女婿家破，自己也身败名裂。
 <br/>
 <br/>
 所以广西民企吴东案这起被指规模最大的银行骗贷案，除了表面上金额420亿元，还有更深层的启示，那就是江泽民这场迫害导致的各种腐败及恶果：610系统、政法系统、地方官场、金融系统、国有资产、地方黑势力、雇凶杀人、谋财害命、杀人灭口、制造连环爆炸恐吓等等，令人触目惊心。
 <br/>
 <br/>
 ──转自《大纪元》本文只代表作者的观点和陈述。
 <br/>
