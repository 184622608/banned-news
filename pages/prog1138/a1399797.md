### 侨报董事长被杀牵出更深内幕 背景直通中南海
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】日前，位于美国加州洛杉矶阿罕布拉市的中文媒体《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侨报.html" target="_blank">
  侨报
 </a>
 》大楼内发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，《侨报》董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢一宁.html" target="_blank">
  谢一宁
 </a>
 被一名员工枪杀身亡。据报导，《侨报》实际是中共国务院侨办的，直通中南海，是直接服务于中共的海外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大外宣.html" target="_blank">
  大外宣
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_统战机构.html" target="_blank">
  统战机构
 </a>
 ，介入海外侨团和留学生事务。谢一宁之死牵出其中更多内幕。
 <br/>
 <br/>
 美西时间11月17日凌晨，《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侨报.html" target="_blank">
  侨报
 </a>
 》在其网站上发讣告，确认该报董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢一宁.html" target="_blank">
  谢一宁
 </a>
 于16日上午在办公室被枪杀。
 <br/>
 <br/>
 据美媒报导，11月16日，位于美国洛杉矶阿罕布拉市（Alhambra）的《侨报》大楼内发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，谢一宁身中数枪当场身亡。《侨报》员工、56岁的嫌犯陈忠启( Zhong Qi Chen)，当天下午已被警方逮捕。
 <br/>
 <br/>
 阿罕布拉警局的卡尔（Steven Carr）在新闻发布会上介绍，警方在接到911报警后，立即赶到案发现场。当时房间内有4个人，除了其中一名中年男性中多枪死亡外，其他员工未曾受伤。在场4人都被带走警局接受讯问，在随后的调查中，警方确认陈忠奇是开枪之人，并将其逮捕。
 <br/>
 <br/>
 目前警方确认，犯罪嫌疑人陈忠奇为死者同事，犯罪动机尚不明确，此案被定为一起职场暴力事件。
 <br/>
 <br/>
 据报，枪击案是因资方提高业绩要求所致。命案发生当天，《侨报》高层主管曾与公司业务部门召开年度业务会议。
 <br/>
 <br/>
 在会上，《侨报》高管向业务员提出，明年的业务额需在今年全年的基础提高10%。多名业务员与主管进行了激烈的讨论。一名男性业务员在会后留在现场与主管继续讨论。
 <br/>
 <br/>
 《世界日报》报导，曾经与谢一宁熟识的一名友人说，谢一宁曾担任《中新社》驻华盛顿记者。任期到后，在洛杉矶开办《侨报》，主管《侨报》业务。
 <br/>
 <br/>
 《中新社》全称为《中国新闻社》，是中共中央级喉舌媒体，中共统战部直属事业单位。中共自己曾承认全球80%的华文媒体接受《中新社》的供稿。
 <br/>
 <br/>
 据悉，谢一宁1960年出生，原籍潮州。1982年新闻系毕业后进入中新社任记者，1987年外派担任《中新社》驻美国白宫记者。其正式头衔为《中新社》驻美国特派员。1991年，谢担任美国《侨报》首席记者，1992年起任该报美西版总裁，后任董事长至今。
 <br/>
 <br/>
 《自由亚洲》电台17日报导说，由于谢一宁和《侨报》背后都有中共官方元素，该命案迅速在华人圈引发震荡。
 <br/>
 <br/>
 报导说，谢一宁的朋友陆先生透露，谢一宁是《侨报》董事长，他的前妻是该报总经理；而他现任妻子是位于洛杉矶的凤凰卫视主播郭晶晶。
 <br/>
 <br/>
 凤凰卫视是中共的〝第二个央视〞。美国凤凰台是香港凤凰卫视的美国分台。香港凤凰卫视是中共在香港设立，中共前党魁江泽民之子江绵恒是大股东。
 <br/>
 <br/>
 美国独立非盈利机构詹姆斯通基金会（Jamestown Foundation），早在2001年曾披露，中共做出了巨大努力介入北美的中文媒体，美国的中文报纸：《世界日报》、《星岛日报》、《明报》和《侨报》等都受中共直接或间接的控制。
 <br/>
 <br/>
 而《侨报》则是直接受中共控制，是中共国务院侨办的，其资金和主要编辑、管理人员全部来自北京。该报的特点是大量及时地报导来自中共的消息，代表着中共官方的声音和观点。
 <br/>
 <br/>
 如《侨报》自江泽民1999年下令镇压法轮功以来，一直帮助中共在海外打压法轮功。据不完全统计，1999年至2002年5月，《侨报》共刊登300多篇针对法轮功的反面文章，几乎平均3天一篇，观点与中共官媒同出一辙。
 <br/>
 <br/>
 曾和《侨报》接触过的原体制内人士李先生说，《侨报》属中共以非官方名义在海外设立的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大外宣.html" target="_blank">
  大外宣
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_统战机构.html" target="_blank">
  统战机构
 </a>
 ，直接介入海外侨团和留学生事务，全方位为中共服务。
 <br/>
 <br/>
 李先生说，目前美国正在加大对中共海外渗透机构的反制力度，这肯定会给中共海外机构造成很大的压力，从而导致其内部出现问题。
 <br/>
 <br/>
 公开资料显示，《侨报》在美国属于全国性的媒体集团，拥有日报、周报、中文网站、英文网站，并在全美15个华人聚居的城市设有发行点的。其总部所在地阿罕布拉市位于南加州洛杉矶以东14英里 ，是一个华人聚集的城市。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿）
