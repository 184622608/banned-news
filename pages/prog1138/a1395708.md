### 江泽民出事了？知情人曝江家紧急召集
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】屡传病危的中共前党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 ，似乎情况真的不妙了。16日，有上海消息人士透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江家人.html" target="_blank">
  江家人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_紧急集合.html" target="_blank">
  紧急集合
 </a>
 ，可能是因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民病危.html" target="_blank">
  江泽民病危
 </a>
 或更糟。
 <br/>
 <br/>
 希望之声17日引述上海消息人士透露，江家16日有异常情况，纷纷〝集合〞，可能
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 病危或更糟。
 <br/>
 <br/>
 之前的10月9日，也有海外爆料者披露，江泽民的身体健康不容乐观。10月3日，有美媒也披露，江泽民身体据称不断反覆，似乎并不乐观。
 <br/>
 <br/>
 再往前推，9月30日，美媒爆料，中共国务院在北京大会堂举行十一招待会，没有中共元老的镜头，江泽民、李鹏、朱鎔基、胡锦涛、温家宝等人〝集体缺席〞。
 <br/>
 <br/>
 中南海的十一招待会，已经成为观察中共政坛动向的窗口。外界分析，如果元老集体缺席的消息属实，可能存在几种原因。其中包括元老们的身体出现状况。
 <br/>
 <br/>
 中南海知情者爆料说：江泽民和李鹏年岁已高，且患有老年人常见的疾病，健康频频亮起〝红灯〞，甚至遭遇多次病危险情。但由于中共高官的健康状况极为保密，各类消息真假难辨，中共官方也未予证实。
 <br/>
 <br/>
 有分析说，江泽民已经92岁，按说无论出现何种情况都是正常，但外界之所以如此关注他的生死，与他身上背负太多血债且是习近平头号政敌有关。江一人的生死牵扯到整个江家乃至包括曾庆红、刘云山等人在内的庞大江氏派系的存亡。
 <br/>
 <br/>
 去年十九大召开前，就有美媒引述知情人爆料说，习近平特别为出席十九大开幕式的江泽民准备了一套〝死亡预案〞，一旦江泽民突然在会场咽气，当局该如何应对。
 <br/>
 <br/>
 当时的直播镜头中也可以看到，江泽民出席十九大时，几乎是被架进会场落座。
 <br/>
 <br/>
 事实上，近年来，有关江〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_死去活来.html" target="_blank">
  死去活来
 </a>
 〞的消息不断传出。不过，截至目前为止，只有过一次新闻正式报导的，以及唯一一次有过官方正式回应。
 <br/>
 <br/>
 有过正式报导是2011年7月6日，香港亚视新闻独家报导江泽民病逝。同日《山东新闻网》也以大字标题报导江泽民逝世消息，只是随即被撤。
 <br/>
 <br/>
 后来，海外传媒引述消息称，亚视误报死讯之前，江泽民在上海生过一场大病，这次生病和吃有关。据称这次吃的甲鱼体内有寄生虫，结果吃出了事。
 <br/>
 <br/>
 有过官方正式回应的是2014年9月9日，中共外交部发言人华春莹，被媒体问到江泽民死讯传闻是否属实时，她表示：〝无法证实〞。
 <br/>
 <br/>
 江泽民2014年的死讯，源于日本《东京新闻》报导，报导称江泽民在8月上旬因旧疾恶化，病情相当严重而住院急救。
 <br/>
 <br/>
 报导称，江泽民此次是〝精神受到相当大的打击〞。江一直反对调查周永康，但在习当局7月29日立案查处周之后，江泽民的身体不支紧急入院。
 <br/>
 <br/>
 去年江泽民已是多次传出病危消息。而每次传出江的死讯时，网上一面倒的的欢呼庆祝，并藉此把江泽民的汉奸身世、出卖国土、镇压六四学生，迫害法轮功、垂帘听政、党政军大老虎总后台、江家父子贪腐第一等等罪行抖露一番。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
