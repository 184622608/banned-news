### 习近平召见民企老板 刘强东王健林再度缺席
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 〞左倾政策成为热门话题，引发舆论强烈反弹后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 一个多月内4次抛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_救心丸.html" target="_blank">
  救心丸
 </a>
 ，近日习更是召见
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民企老板.html" target="_blank">
  民企老板
 </a>
 开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_座谈会.html" target="_blank">
  座谈会
 </a>
 进行安抚，然而备受关注的京东集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 ，与万达集团董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王健林.html" target="_blank">
  王健林
 </a>
 两人缺席会议。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 11月1日主持召开民营企业
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_座谈会.html" target="_blank">
  座谈会
 </a>
 ，接见10名企业家代表。习在座谈会上强调：所有民营企业和民营企业家完全可以吃下定心丸、安心谋发展！民营企业和民营企业家是自己人。
 <br/>
 <br/>
 习近平还开出〝期票〞，下一步支持民营企业发展壮大，包括减轻企业税费负担，解决民营企业融资难融资贵问题，保护企业家人身和财产安全等。
 <br/>
 <br/>
 这是习近平这1月多来，第4次发表言论安抚民营企业。
 <br/>
 <br/>
 据新华社报导，当天的在座谈会上，有10位企业家代表先后发言。中共经济界消息人士曹山石在推特曝光与会人员的座席名次图。备受关注的京东集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 ，与万达集团董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王健林.html" target="_blank">
  王健林
 </a>
 两人都缺席此次座谈会。
 <br/>
 <br/>
 在此之前，中共官方发布《改革开放40年百名杰出民营企业家名单》，同样被视为，为民企再打一剂〝强心针〞。中共官媒称，推荐这个名单的目的是要在所谓的改革开放40年之际，鼓励
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民企老板.html" target="_blank">
  民企老板
 </a>
 〝不忘创业初心〞。但名单中也没有王健林和刘强东。
 <br/>
 <br/>
 王健林不仅是中国商界亿万富豪，还是中共第十七次人大代表、第十一届政协委员及第十届全国工商联副主席。此外，王健林还担任中共民间商会副会长、中共企业联合会副会长、中共企业家协会副会长、中共商业联合会副会长和中国慈善联合会副会长。
 <br/>
 <br/>
 2015年，《胡润全球华人富豪榜》发布，王健林首次超越李嘉诚成为全球华人首富。但在2017年王健林运程逆转，因涉嫌在海外违规并购而招致北京重罚。如今王健林已由首富变为首负。
 <br/>
 <br/>
 而刘强东此前在美国涉嫌强奸被捕丑闻震惊舆论。据美国警方9月4日公布的报告，刘强东涉嫌强奸发生在8月31日凌晨1点左右，当晚被捕入狱，16个小时后获释。
 <br/>
 <br/>
 警方对刘强东一案的描述是〝一级性犯罪：强奸既遂〞（CSCR-1,CSC-RAPE-COMPLETE）。根据明尼苏达州的法律，一级强暴罪只要最后被定罪，刑期至少12年起。
 <br/>
 <br/>
 如今刘强东强奸案仍在调查中，有华裔刑案律师认为，刘强东遭起诉的可能性超过90%，案件一旦开始会涉及重新逮捕等问题。
 <br/>
 <br/>
 刘强东不仅是中国商界亿万富豪，同时还有中共全国政协委员身份，并兼任中国民间商会副会长。作为中共的红顶商人，却在美国遭遇强奸犯罪指控，无疑是中共一大丑闻。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：赵云）
