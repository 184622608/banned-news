### 陆记者揭秘：王立军叛逃美领馆前 打了一个重要电话
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】薄熙来心腹、重庆原公安局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 出逃成都美领馆事件是国际性爆炸新闻，也是中共高层撕裂的分水岭。《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南方周末.html" target="_blank">
  南方周末
 </a>
 》的记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_褚朝新.html" target="_blank">
  褚朝新
 </a>
 曾披露，在王立军出逃前，他与王联系的专用电话有一个重庆号码的未接电话，但他回过去居然语音提示是空号。有朋友笑称，他可能错过了一个重要的电话。
 <br/>
 <br/>
 <h4>
  未接听的重要电话
 </h4>
 <br/>
 2013年4月25日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_褚朝新.html" target="_blank">
  褚朝新
 </a>
 发表题为〝我与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 的三次接触（四）〞文章说，历经四个月，最后一篇终于与大家见面了。因一些特殊的原因，很多事情暂时只能说到目前的程度。
 <br/>
 <br/>
 文章说，褚朝新和王立军曾通过一个专用电话联络。2012年2月2日，王立军被罕见人事调动。
 <br/>
 <br/>
 2月3日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南方周末.html" target="_blank">
  南方周末
 </a>
 总部和几个主要的记者站开例会。褚朝新看没有王立军相关的新闻，于是致电王立军，王说，〝是正常的工作调整〞。
 <br/>
 <br/>
 褚朝新对王立军说，〝外界传闻很多，有人说组织部宣布调整你的分工的时候，你把枪都掏出来拍到桌子上了，跟薄熙来公开破裂了。〞王仍说〝正常工作调整〞。
 <br/>
 <br/>
 2月5日下午4点多，褚朝新突然发现与王立军联络的专用手机上，有一个显示重庆号码的未接电话。相隔不过一两个小时，他打过去对方居然语音提示是空号。
 <br/>
 <br/>
 十几个小时候后，2月6日，爆出王立军出逃成都美领馆的国际性爆炸新闻。后来，有朋友开玩笑说，褚朝新可能错过了一个重要的电话。
 <br/>

   <br/>
   <font size="-1">
    《南方周末》的记者褚朝新曾披露，在王立军出逃前，他与王立军联系的专用电话有一个重庆未接电话。（Feng Li/Getty Images）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 2月11日，重庆一个区公安分局的局长告诉褚朝新，王立军的手机开机了。他当即试着拨了一下，果然开机了，但无人接听。
 <br/>
 <br/>
 <h4>
  神秘短讯曝光
 </h4>
 <br/>
 2月15日凌晨5点，与王立军联系的那个手机上收到了一条简讯：英国人尼尔伍德在渝被害，立军破案剑指GKL，被休假，奇入美领馆，被消失。薄、谷要取信党、政、国、民、世界和十八大。（有图，就不发了，将来或许有别的用途。）
 <br/>
 <br/>
 这条简讯透露了英国商人海伍德被杀案成为王立军和薄熙来、谷开来翻脸的导火索。当时，褚朝新看了这条简讯，并不相信谷开来杀人。这条简讯就被閒置在手机里。
 <br/>
 <br/>
 直到2012年3月24日晚上，网友杨海鹏发微博说，薄瓜瓜的英国保姆死在重庆了，褚朝新一下子想起了那条简讯，于是在微博上公布了简讯大概内容。随后，他因披露尼尔伍德被杀的消息被警方调查，相关微博也被删除。
 <br/>
 <br/>
 4月14日，网友刘沙沙引用褚朝新微博信息说，〝不否认，是我3月首发尼尔伍德的名字。昨有无证公人因此试图找我并骚扰我的朋友，以无暂住证为由要带其去派出所。经过斡旋并微博直播，公人道歉。事实证明，尼尔伍德不是谣言。若他们需要把说过事实的人关进去，我愿意做这个人，成就一段历史，届时朋友们也不必搭救，记住并转播这段即可。〞
 <br/>
 <br/>
 据报导曾帮助褚朝新发消息的《21世纪经济报导》记者王思璟也受到了公安的调查。
 <br/>
 <br/>
 薄熙来、王立军事件爆发后，中共进一步收紧操控媒体并向外界释放各类消息。有分析指出，海伍德死因能在网路上泄漏，很可能是倒薄一方的需要，同时保薄一方也在极力〝辟谣〞封锁屏蔽信息，这是权斗双方在利用媒体搏击。

 <h4>
  王立军的保命材料内幕
 </h4>
 <br/>
 王立军
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叛逃美领馆.html" target="_blank">
  叛逃美领馆
 </a>
 时携带保命材料，随即成为国际及中国社会关注的焦点。在2012年2月份习近平访问美国的时候，美国媒体《华盛顿自由灯塔》发表报导称，王立军告诉美领馆官员，薄熙来和周永康密谋联手想搞掉习近平再夺权。
 <br/>
 <br/>
 5月份，美国国务院发表的人权报告中首次明确提到中共强制摘取法轮功学员器官一事。
 <br/>
 <br/>
 大纪元独家获悉王立军交给美国政府的材料内容包括下面六个方面：
 <br/>
 <br/>
 一、薄熙来及其家属的贪腐证据。（目前人们仅从海伍德案中已经得知，薄熙来和其夫人谷开来，转移到国外的资金就高达60亿美金。）
 <br/>
 <br/>
 二、薄熙来到任重庆后收买军队高层的证据。（黄奇帆在倒戈后，揭发出来的薄熙来罪行之一就是，薄声称至少可以调动两个军。国防部长梁光烈、原成都军区政委张海阳，是薄熙来在军中的支持者。）
 <br/>
 <br/>
 三、薄熙来指示处死文强等重庆高官，及下令逮捕李庄的证据。
 <br/>
 <br/>
 四、薄熙来、周永康联手搞掉习近平密谋夺权的计划与证据。薄熙来把计划透露给王立军，并告诉王整个计划是得到〝老头子〞的支持的。（老头子是指江泽民。薄周要夺习近平的权，正是江泽民和曾庆红一手安排的：江泽民原定的18大继承人是薄，而不是习。习只是江为了让薄有时间爬上来的过渡人物。）
 <br/>
 <br/>
 五、薄熙来入政治局常委掌握实权后，计划制造民意把重庆模式推向全国，以胡温派系、民间资本家、异议人士、宗教异议者为清洗对象，开展一次文革式的政治运动，〝不惜牺牲50万人，也要确保红色江山不变天〞。（这就是王立军事件发生后，薄熙来为什么要冒天下之大不韪包围美国驻成都领馆，甚至向黄奇帆下达〝不惜一切代价让王立军消声〞的命令的根本原因。乌有之乡的人早在2011年就公开提出恢复反汉奸法，企图把清洗对象归为汉奸而加以屠杀。）
 <br/>
 <br/>
 六、薄熙来指示及参与活体摘取法轮功学员器官的相关证据（录音、密件等）及政法系统下达的对法轮功及异议人士的镇压文件。（这是中共网路封锁及过滤的最高级别的关键词、亲共媒体最忌讳的敏感内容。）
 <br/>
 <br/>
 <h4>
  中共隐瞒王立军重罪
 </h4>
 <br/>
 2012年9月，四川成都中级法院对王立军徇私枉法、叛逃、滥用职权、受贿案宣判，共计有期徒刑15年。
 <br/>
 <br/>
 不过，海外人权组织追查迫害法轮功国际组织的负责人汪志远指出，中共对王立军的判决，完全没有涉及他所参与的活摘法轮功学员器官的罪行。根据追查迫害法轮功国际组织的调查结果显示，一位曾与王立军合作的移植医生、原锦州解放军205医院泌尿外科主任陈荣山，多次说漏嘴，承认摘取法轮功学员器官，做器官移植手术。
 <br/>
 <br/>
 中国问题专家横河也指，当局避重就轻，对王立军犯下的重罪并未提及。〝第一个就是王立军当时在锦州开了一个现场心理研究中心，做了几千起器官移植，涉嫌跟活摘法轮功学员器官有直接关系；另一个是在重庆打黑的时候涉嫌酷刑逼供、甚至酷刑逼供致死；另外一个就是有证人出来说，他建了一个女子特警队，涉嫌强奸女警官。〞
 <br/>
