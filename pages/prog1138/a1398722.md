### 搞权色交易 河北前副秘书长吴立芳被双开
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】周五，河北省政府〝前副秘书长〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴立芳.html" target="_blank">
  吴立芳
 </a>
 等两名官员，因涉嫌受贿被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双开.html" target="_blank">
  双开
 </a>
 ，案件移送检察院。在此之前，河北省已有7名主要官员落马。
 <br/>
 <br/>
 通报称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴立芳.html" target="_blank">
  吴立芳
 </a>
 严重〝违纪违法〞；收受消费卡，违规出入私人会所；不如实报告个人有关事项；收受礼品、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_搞权色交易.html" target="_blank">
  搞权色交易
 </a>
 等，落马后还〝对抗审查〞。
 <br/>
 <br/>
 吴立芳2000年1月起，先后在河北省三河市、廊坊市、张家口市和保定市担任市长、市委书记等职。2016年11月，任河北省政府副秘书长、省政府机关党组成员。今年5月20日落马。
 <br/>
 <br/>
 中共十八大后，河北官场持续被清洗。今年7月，河北省政协前副主席艾文礼因〝涉嫌严重违纪违法〞投案自首，正在接受审查和调查。
 <br/>
 <br/>
 目前，由江派亲信掌控的中共河北官场已经有七只〝大老虎〞落马，他们是：杨崇勇、梁滨、张越、景春华、周本顺、张杰辉和艾文礼。
</div>
