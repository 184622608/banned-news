### 薄熙来一度逼宫胡锦涛  温家宝与周永康火拚
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】2012年2月6日，中共前重庆市公安局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 夜逃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美领馆.html" target="_blank">
  美领馆
 </a>
 ，掀开了中共高层生死搏击的序幕。围绕着事件引发的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 倒台，以及薄倒台前逼宫胡锦涛，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 曾因力保薄熙来与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_温家宝.html" target="_blank">
  温家宝
 </a>
 在政治局会议上激烈争吵，使得中南海分裂大曝光。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
   周永康
  </a>
  力保
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
   薄熙来
  </a>
 </h4>
 <br/>
 2012年2月6日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 携带机密材料闯入美国驻成都领事馆。
 <br/>
 <br/>
 据港媒报导，同月16日中共政治局常委开会讨论如何应对〝危机〞，尤其是如何处理薄熙来的问题。当时在京的常委有8人。其中胡锦涛、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_温家宝.html" target="_blank">
  温家宝
 </a>
 、李克强和贺国强4人主张依法处理薄熙来，但另外4人吴邦国、贾庆林、李长春和周永康反对。
 <br/>
 <br/>
 当时，习近平正在美国访问，接到了中办的电话，征求其意见。习近平表态说，支持胡锦涛等人的意见：追究薄熙来的责任。习在〝倒薄〞中投下了关键一票。
 <br/>
 <br/>
 王立军出事后，备受〝共同谋反〞困扰的周永康在同年〝两会〞前力挺重庆，并在〝两会〞时为薄熙来〝站台〞，同时不断向薄熙来告知中央的动向。
 <br/>
 <br/>
 报导称，这期间周永康的秘书曾经集合〝毛左〞们秘密开会，部署了一套反攻计划。薄熙来也因为有周永康的撑腰，在〝两会〞上对胡锦涛进行逼宫。
 <br/>
 <br/>
 2012年3月14日两会闭幕日，时任中共总理的温家宝，在新闻发布会上措辞严厉，对重庆事件首次作出公开回应，批重庆模式〝开历史倒车〞。
 <br/>
 <br/>
 两会结束的第一天，3月15日薄熙来的重庆市委书记职务被免。
 <br/>
 <br/>
 据报导，温家宝还力主进一步追究薄熙来违法的事实，及其由此而造成的王立军出逃和〝唱红打黑〞等一系列问题，将薄熙来撤职查办。
 <br/>
 <br/>
 与此同时，周永康使出混身解数力保薄熙来。消息说，周永康是唯一反对撤去薄熙来职务的政治局常委，他甚至向高层建议，将薄熙来调任西藏。他认为只有薄熙来才可以用所谓的〝铁血手段〞解决西藏的混乱局面。
 <br/>
 <br/>
 4月10日，薄熙来的政治局委员职务被撤。
 <br/>
 <br/>
 <h4>
  温家宝与周永康在政治局争吵
 </h4>
 <br/>
 据报，在同年5月初的一次政治局扩大会议中，参加方面有政治局常委、委员和地方大员、最高级别军头以及已经退休的一些中共元老如曾庆红等。在众人面前，温家宝与周永康撕破脸皮，展开公开决战。
 <br/>
 <br/>
 温家宝继续就薄熙来的事件质问周永康，并要求调查周永康。但是周永康拿出海外的温家宝负面传言，要求对温家宝的夫人同时也进行调查。曾庆红也表示支持。
 <br/>
 <br/>
 温家宝则罕见抛出重话称：可以对我温家宝和家人进行调查，〝如果我本人及家人有任何敛财行为，我马上辞职！〞
 <br/>
 <br/>
 另有消息称，温家宝在另一场合讲话说：〝前六七年，其实更早时候，镇压法轮功给中国带来的可怕后果就已经看到了，我们经过调查发现江泽民使用令人震惊的国家财力去镇压一个手无寸铁的民间团体，非常荒谬，一直到现在，这个问题中央都没有去面对、去解决。〞
 <br/>
 <br/>
 温家宝还说：〝不施麻药，摘活人器官，还拿去赚钱，这是人干的事情吗？这种事情发生多年了，我们要退休了，还没解决……〞，〝现在出来王立军这件事，全世界都知道了，借处置薄熙来把法轮功的问题解决了，应该是水到渠成。〞
 <br/>
 <br/>
 温家宝的话被认为他希望藉助公审薄熙来的机会解决法轮功的问题，这引起江泽民集团的极大恐慌，周永康决定孤注一掷，通过特务向外媒释放关于温家宝等中共高层的家族机密，以期搅乱局势。
 <br/>
 <br/>
 <h4>
  周永康向外媒喂料 力图搅乱局势
 </h4>
 <br/>
 2012年10月有关温家宝〝家族贪腐事件〞迅速成为海内外各大媒体的焦点。来自北京的消息称，直接指挥向外媒喂料的是周永康的亲信、中共〝610办公室〞主任兼公安部副部长李东生。
 <br/>
 <br/>
 消息说，李东生策划了包括对国际媒体释放攻击温家宝、习近平家族的材料活动，甚至承诺提供给海外一些网站和出版社巨额资金，出版攻击习近平阵营要员的书籍。
 <br/>
 <br/>
 消息来源还说，2013年12月20日，李东生被〝双规〞调查后，很快交代了这一问题。这也是促使习近平决心打〝周老虎〞的最重要原因。
 <br/>
 <br/>
 2014年1月21日，一个名为〝国际调查记者同盟〞（ICIJ）的机构公布一份含有250万份文件的资料，其中包括中共最高层及其亲友海外秘密资金情况。
 <br/>
 <br/>
 该报告称，包括习近平、温家宝、李鹏、胡锦涛及邓小平在内，至少5名现任与前任中共中央政治局常委的亲属，在英属维京群岛和科克群岛等离岸金融中心持有离岸公司。
 <br/>
 <br/>
 当时国际舆论普遍认为这份调查报告被选择性的喂料，而报告中，缺了中国第一大贪江泽民及曾庆红、周永康等巨贪家族。
 <br/>
 <br/>
 当时就有评论认为，这很可能是周永康和曾庆红等中共巨贪家族所为。
 <br/>
 <br/>
