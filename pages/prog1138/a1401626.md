### 党内分裂加剧 习近平安保层级急升
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】近期，中国国家主席习近平频频外访，中方安排的超级严密的安保措施成为了舆论关注的热点。不断升级安保措施的背后，显示中共政局的暗潮汹涌。面对愈演愈烈的贸易战，中共内部高度分化，国内经济濒临崩溃，社会动荡日渐加剧，致使习进退两难。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平外访.html" target="_blank">
   习近平外访
  </a>
  超严密安保成舆论关注热点
 </h4>
 <br/>
 近期，习近平频频外访十分忙碌。11月中旬出席亚太经合（APEC）峰会后回国没几天，又于11月27日离开北京，乘专机离开北京赴西班牙首都马德里，开始了他为期9天的欧洲拉美之行。
 <br/>
 <br/>
 11月29日，习近平抵达了布宜诺斯艾利斯，出席即将举行的二十国集团领导人第十三次峰会（G20峰会），并对阿根廷共和国进行国事访问。
 <br/>
 <br/>
 海外媒体注意到，习近平出访时的安保措施安排得异常严密。当中方为了配合习近平出席亚太经合（APEC）峰会，专门从中国空运两辆防弹专车到巴布亚新几内亚后，〝防暗杀〞立即成为海外媒体热炒的话题。
 <br/>
 <br/>
 据《路透社》报导，习近平11月15日抵达巴布亚新几内亚后，他的两辆防弹专车也由中方空运至当地。外界观察到，习近平的专车均由中国产红旗L5加入防弹结构改装而成，配备4公升引擎，外貌类似劳斯莱斯。
 <br/>
 <br/>
 按照国际惯例，国家首脑出访他国，都是由东道主国家负责来访政要的安保，包括提供专车。而这次中方不但自己空运防弹专车来使用，就连习这次出访巴布亚新几内亚的行程，也仅提前三天对外公布，而不是按以往的惯例那样提前至少一周。舆论认为这样的安排与安保考虑有关。
 <br/>
 <br/>
 一时间，海外媒体开始扎堆热炒习近平出访时超严密的安保措施。
 <br/>
 <br/>
 今年7、8月份中共北戴河会议期间，港媒《南华早报》曾爆料称，今年习近平在北戴河享有特殊的最高安保，其中一个细节是，只有习近平火车专列才可以停在北戴河火车站的一号月台，而且沿途每隔200米都设有岗哨警卫。
 <br/>
 <br/>
 今年4月，习近平在湖北武汉会见到访的印度总理莫迪时，湖北当地不但对相关地段实施戒严，还部署了数百狙击手，并出动上万便衣警察进驻街头。网传的视频显示，习近平离开武汉后，当上万名便衣撤离时，简直就像是举行了一次大游行。
 <br/>
 <br/>
 2017年〝七一〞前夕，习近平到访香港时，香港警方也动用了近10,000警力来执行安保任务。
 <br/>
 <br/>
 据港媒的报导，2017年6月29日习近平抵港后，乘坐的专车是分别在车门、车顶、车尾等处配备了防弹装甲的宝马七系防弹车。这款专车能承受炸弹的爆炸和自动步枪近距离的扫射。该专车的内空调系统还有特殊的安全配备，当车辆遭受毒气攻击时，会自动关闭车窗、锁车门，同时关闭外部空气通道。
 <br/>
 <br/>
 当时香港警方也每天派出近一万人候命，包括飞虎队、机动部队、反恐特勤队等全部上阵。为防止来自空中的袭击，还在维多利亚港一带设立了临时禁航区。此外，香港警方还在习近平下榻的酒店附近的大楼屋顶上派驻了多名伏击手。
 <br/>
 <br/>
 有港媒曾评论称，对习近平的安全构成最大威胁的，可能是来自中共的权斗。
 <br/>
 <br/>
 <h4>
  中共内部高度分裂 近平进退两难
 </h4>
 <br/>
 〝中国人权网〞今年8月中旬曾发表一篇特约评论文章称，去年年初习近平还〝踌躇满志、春风得意〞，但仅仅几个月后就陷入了〝内外交困、进退失据〞的窘迫困境之中。军师王沪宁和鼓吹手胡鞍钢等人出的〝馊点子〞和〝歪敲鼓点〞固然对出现这种困局负有责任，习近平自己大权独揽也是重要的原因。
 <br/>
 <br/>
 《为什么会造成崇祯模式？》一文则表示，习近平知道〝很多人在搅局，甚至很多人在背叛〞，因为对付美国的对策，前脚刚研究出来，后脚对方就完全知道了。因此习认为〝有人在出卖国家〞。
 <br/>
 <br/>
 该文分析称，从今年3月份贸易战开打以来，北京当局的各种对策〝朝令夕改〞，没有一个完全的统一的策略，这是中共高层内部高度分化的结果。
 <br/>
 <br/>
 文章写道，〝对贸易战，高层对立的两面各不相让，各有对策，结果弄到现在，已经不可收拾了，不但贸易战一败涂地，而且更为严峻的是国内经济正濒临崩溃。〞
 <br/>
 <br/>
 文章称，习近平在党内〝很孤独〞，他一个人想努力，但大伙都不配合，他总觉得如果给他二十年的时间，他可以〝还大家一个强盛的中国〞。但中共内部的高度分化，让他没多久就陷入了进退两难的境地，〝这真是活脱脱一个大明末日的崇祯〞。
 <br/>
 <br/>
 事实上，当今的中国经济不断下行，各种社会矛盾日益加剧，贸易战也步步升级，可以说是〝内外不宁〞。
 <br/>
 <br/>
 今年以来，北京当局现行的一系列内政外交政策的正确性遭到了广泛的质疑。中共高层内部意见分歧与角力的一些迹象已显现出来。在个人崇拜降温、〝梁家河大学问〞退场、〝中国制造2025〞不提的同时，还传出了中共党内元老批评、甚至逼迫习近平交权的传闻。
 <br/>
 <br/>
 与此同时，中国社会各阶层的群体抗争事件又开始升温。其中影响力比较大的包括：全国范围的卡车司机大罢工、退伍军人维权上访、深圳佳士科技公司工人为组织工会而发起维权行动、P2P网贷〝爆雷〞引发各地〝金融难民〞的上访潮等等事件。
 <br/>
 <br/>
 有分析认为，在这种内外交困的背景下，如果中美贸易战持续升级，甚至出现〝直接和全方位的对抗〞，北京当局还能否继续维持现状就很难预期了。
 <br/>
 <br/>
 今年8月下旬，网络上曾热传一篇题为《历代王朝灭亡前十大征兆》的博文。文中罗列的十大灭亡征兆为：一、官民对立，社会动荡。二、思想混乱，人心浮动。三、加强社会各方控制。四、乱加税收，横征暴敛。五、乱发纸币，货币贬值。六、上层醉生梦死，下层苦不堪言。七、体制僵化，不思进变。八、中下层失去上升空间。九、军队腐败，士气低迷。十、粉饰太平，歌功颂德。
 <br/>
 <br/>
 有中国网民翻墙到海外中文网站上留言评论称，上述博文列出的十大灭亡征兆，中共已经几乎全部都具备了。中共控制国民的维稳体系虽然不断升级加强，但社会中下层的反抗与动荡也越来越加剧，中共这个腐朽的政权早已千疮百孔，习近平就是想保也保不住了。
 <br/>
 <br/>
 （记者唐迪综合报导/责任编辑：明轩）
