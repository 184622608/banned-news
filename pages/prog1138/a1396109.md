### 中方掌握孟宏伟家书 要求孟妻单独见面
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】国际刑警组织前主席、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被中共秘密抓捕后，他的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_格蕾丝・孟.html" target="_blank">
  格蕾丝・孟
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Grace Meng.html" target="_blank">
  Grace Meng
 </a>
 ）一直在海外呼吁国际社会，关注她丈夫的下落和现状。日前，孟妻对外媒透露，中共官方曾要求与她单独见面，并称他们手上握有孟的家书。
 <br/>
 <br/>
 据《美国之音》19日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_格蕾丝・孟.html" target="_blank">
  格蕾丝・孟
 </a>
 日前透露称，孟宏伟回国后被秘密抓捕的事曝光后，中共外交官曾秘密联络她，说掌握着孟写给她的信件。
 <br/>
 <br/>
 但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Grace Meng.html" target="_blank">
  Grace Meng
 </a>
 回应称，自己不会在没有律师和记者在场的情况下与中共外交官见面。她要求中共外交官将孟的信件转交法国警方。
 <br/>
 <br/>
 今年9月底，孟宏伟回到中国大陆后突然与家人失去联系，住在里昂的Grace Meng便向法国警方报警，并召开记者会呼吁国际社会关注孟的下落。其后，北京当局被迫向外界宣布孟宏伟涉嫌受贿违法，正在接受调查。
 <br/>
 <br/>
 Grace Meng之后多次接受外媒采访。她声称自己向法国警方报警后，曾接到过恐吓电话，被威胁说在比利时有〝两个小组〞一直在盯着她。最近她还表示，自己现在不能确定孟宏伟是否还活着。
 <br/>
 <br/>
 为了确保居住在里昂的Grace Meng一家的生命安全，法国警方对她和她的孩子采取了保护措施。
 <br/>
 <br/>
 Grace Meng在接受英国广播公司采访时，还用英文抨击中共政府〝残忍、肮脏和愚蠢〞，〝什么都做得出来，没有底线，对所有的中国人都一样〞。
 <br/>
 <br/>
 出于保护其安全的考虑，英国广播公司在播出对英格蕾丝・孟的专访报导时，特意对她的面部画面进行了技术处理。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：李泉）
</div>
