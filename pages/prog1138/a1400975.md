### 江泽民名字再上花圈 网传已脑死亡多时
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】近日，中共前党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 老上级、原电子工业部部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张挺.html" target="_blank">
  张挺
 </a>
 病亡。江泽民的名字再次出现在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_送花圈.html" target="_blank">
  送花圈
 </a>
 〞名单中。近几个月以来，江泽民快要不行了的说法屡屡出现。有港媒称，江7月底已经
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脑死亡.html" target="_blank">
  脑死亡
 </a>
 。但消息仍没得到官方证实。
 <br/>
 <br/>
 中国电子商会网站25日报导，前中共电子工业部部长，党组书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张挺.html" target="_blank">
  张挺
 </a>
 因病医治无效，于15日在北京去世，享年96岁。
 <br/>
 <br/>
 张挺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遗体.html" target="_blank">
  遗体
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_告别仪式.html" target="_blank">
  告别仪式
 </a>
 23日在八宝山公墓举行，报导称，中共现任和前任高层习近平、李克强、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 、胡锦涛等送上花圈。
 <br/>
 <br/>
 江泽民是张挺的老下属和接任者，作为江当年发迹地的老上司去世，江泽民只是出现在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_送花圈.html" target="_blank">
  送花圈
 </a>
 〞的名单中，引人关注。
 <br/>
 <br/>
 10月18日，中共前领导人胡锦涛与现任高层习近平、王岐山等，现身在中共维族高官、人大前副委员长司马义・艾买提遗体送别仪式上。港媒称，江泽民也是只〝送花圈〞，没有现身。
 <br/>
 <br/>
 11月2日，中共前全国人大副委员长、光大集团前名誉董事长王光英遗体在北京八宝山公墓火化。习近平、胡锦涛等人现身默哀，同样未见江泽民的身影。
 <br/>
 <br/>
 在近期多次公开场合，江泽民〝只见名字不见人〞的迹象，引起外界猜想。自2011年被香港亚视报导〝死讯〞后，江泽民已多次传出各类死亡的消息。近几个月以来，江泽民快要不行了的说法屡屡出现。
 <br/>
 <br/>
 香港《前哨》11月号刊发署名文章披露，7月底的北戴河会议前夕，中南海上演一逼宫戏功败垂成，原因有二，一是军方没表态，二是江泽民在北戴河会议前突然进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脑死亡.html" target="_blank">
  脑死亡
 </a>
 状态。不过，有关江泽民脑死亡的消息，目前没有官方信息证实。
 <br/>
 <br/>
 据美媒引述上海消息人士透露，江家10月16日有异常情况，纷纷〝集合〞，可能江泽民病危或更糟。10月3日，也有消息披露，江泽民身体据称不断反覆，似乎并不乐观。
 <br/>
 <br/>
 9月30日，中共国务院在北京大会堂举行十一招待会。有美媒爆料，此次招待会上，江泽民、李鹏、朱鎔基等人〝集体缺席〞。由于十一招待会早已成为观察中共政坛动向的窗口。有分析认为，元老集体缺席，不排除掩盖某元老病危的消息。
 <br/>
 <br/>
 最近一段时间以来，胡锦涛、温家宝、胡启立、朱鎔基、李瑞环、吴邦国等退休常委，接连以不同方式密集〝露面〞，一向爱出风头的江泽民至今仍销声匿迹，生死不明。
 <br/>
 <br/>
 分析认为，按理说江泽民已经92岁了，无论出现何种情况都是正常，但外界之所以如此关注他的生死，主要与江身上背负太多血债且是习近平的头号政敌有关。江一人的生死，将牵扯到整个江家乃至包括曾庆红、刘云山等人在内的庞大江氏派系的存亡。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
