### 金融系统持续地震 央企恒天董事长被查
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】中共金融系统持续大地震，继中国华融董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 落马后，另一家大型央企
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恒天集团.html" target="_blank">
  恒天集团
 </a>
 董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张杰.html" target="_blank">
  张杰
 </a>
 ，日前被带走调查。据知情人士称，中纪委近日收到实名举报信，张杰涉嫌巨额利益输送。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张杰.html" target="_blank">
   张杰
  </a>
  被实名举报
 </h4>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恒天集团.html" target="_blank">
  恒天集团
 </a>
 全称〝中国恒天集团有限公司〞，成立于1998年，是一家以纺织装备制造为核心主营业务的大型央企，资产规模达900亿。
 <br/>
 <br/>
 据称，张杰于10月12日被监管机构带走。14日，张杰单位及家属被通知其接受调查。
 <br/>
 <br/>
 公开信息显示，张杰一直在纺织工业部门任职，曾担任中共纺织机械（集团）有限公司副董事长，中国恒天集团公司副总经理、党委委员，中国华源集团公司总裁，中国恒天集团有限公司董事长。
 <br/>
 <br/>
 中国财经自媒体〝叩叩财讯〞15日刊文称，张杰落马或与恒天集团旗下的恒天财富一笔巨额融资有关。恒天财富是一家理财平台，大股东是中国巨型民企中植集团，资产超10,000亿元人民币。
 <br/>
 <br/>
 今年3月13日，中植集团老大解植锟突然〝响应国家号召〞，交出近万亿的〝钱袋子〞中融信托。中融信托是中国最大的信托巨头之一，管理资产8584.72亿，去年中融信托营收65.3亿，位列行业第二；28亿利润位列行业第四。
 <br/>
 <br/>
 同日，A股上市公司〝经纬纺机〞发布公告称，通过股份及现金结合的方式，收购中植集团所持中融信托32.9%股权。交易完成后，公司将持有中融信托约70.4%股权。恒天集团为〝经纬纺机〞的实际控制人，也就是说，恒天集团成为中融信托真正的老大。
 <br/>
 <br/>
 〝叩叩财讯〞引述知情人士称，中纪委近期收到实名举报信，举报张杰通过恒天财富，为一家企业提供非法巨额融资，涉嫌利益输送。据称，恒天财富实际控制人为一王姓女子，她在接受中纪委调查时承认，自己是张杰的情人。
 <br/>
 <br/>
 <h4>
  巡视组发现问题
 </h4>
 <br/>
 早在张杰被查前，2016年11月10日至2017年1月10日，中共巡视组对恒天集团进行巡视时，发现诸多问题。
 <br/>
 <br/>
 2017年9月，巡视组发布对恒天集团的巡视整改要求。要求恒天集团坚决把党对国有企业的领导落到实处，要求恒天集团认真贯彻落实问责条例，对执行当局路线方针政策不力，违规开展融资性贸易等问题，加大问责力度。
 <br/>
 <br/>
 此后，中纪委收到多份有关张杰的实名举报。张杰除涉嫌利益输送外，还被举报好打高尔夫球，向组织隐瞒个人情况和财产。
 <br/>
 <br/>
 <h4>
  恒天、国机重组
 </h4>
 <br/>
 在此之前，据官媒报导，2017年7月，恒天、国机两大央企进行重组，〝恒天系〞双线整合成市场化改革试点。
 <br/>
 <br/>
 恒天集团将被正式并入中国机械工业集团公司（简称国机集团）麾下，成为其全资子企业，如此之后，恒天集团将不再作为国资委直接监管企业。
 <br/>
 <br/>
 报导称，除了在有关业务领域合作外，在机制创新方面，中国恒天或将作为国机集团的市场化改革试点，在公司治理结构、股权结构等方面进行改革。
 <br/>
 <br/>
 目前，官方尚未通报张杰被查到具体原因。今年4月17日晚间，闪电落马的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 ，长年主持的华融是财政部控股的四大金融巨无霸之一，关于他落马的真实原因，当局至今也未公布。不过，有分析认为，毫无疑问的是，中共金融界已开始引发大地震，未来恐还有高管落马。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：赵云)
