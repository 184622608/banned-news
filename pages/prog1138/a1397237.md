### 【禁闻】浪子回头造福故里 含冤入狱众人营救
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓成联.html" target="_blank">
  邓成联
 </a>
 今年3月被非法绑架，在看守所遭到迫害。他曾经捐款30万为老家修路，如今家乡父老纷纷签名，呼吁无罪释放这个好人。
 <br/>
 <br/>
 在湖北省东部的蕲春县邓信村，流传着一个现代〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_浪子回头.html" target="_blank">
  浪子回头
 </a>
 〞的故事，主人公的名字叫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓成联.html" target="_blank">
  邓成联
 </a>
 。
 <br/>
 <br/>
 知情者于先生：〝他以前高中没读完就在外面打工，后来在上海开了一家自己的公司，生意做得还不错的。但后来呢，因为随着社会风气的败坏，他也就随波逐流，沾染了很多的恶习。年纪轻轻就有神经衰弱，严重的胃病。还有，把一个好端端的家庭给拆散了，使得他的儿子没有一个完整的家庭。〞
 <br/>
 <br/>
 直到有一天，邓成联开始修炼法轮功——这是1999年前在中国大陆广受欢迎的功法，要求修炼者按照〝真、善、忍〞做好人。不到一年，邓成联就戒掉了所有的陋习。他变得和善，事事为他人着想。久治不愈的各种疾病也不翼而飞。
 <br/>
 <br/>
 于先生：〝很多人目睹了他的变化以后都感到奇怪，觉得有些不可思议，很难以相信。后来时间长了，他变得越来越好，大家也就真的相信，有浪子回头的事迹。〞
 <br/>
 <br/>
 邓成联长年住在上海，但除了身边的人，他也一直在考虑，怎样才能帮助到家乡的父老乡亲们。2010年度，邓成联共捐款三十多万元，为家乡修了几条平整宽阔的水泥路，把往日偏僻闭塞的小山村与周边热闹的镇、大的集市连接起来了。每次回老家，邓成联都会挤出时间去看望村里的孤寡老人、困难户，并慷慨解囊，解决他们的燃眉之急。
 <br/>
 <br/>
 然而中共从1999年开始镇压法轮功，强迫修炼者放弃信仰，这一政策一直持续至今。邓成联认为自己信仰〝真、善、忍〞没有错，并坚持向周围的人讲清真相。
 <br/>
 <br/>
 在今年的3月23号，上海市闵行分局和虹桥派出所警察没有出示任何手续，将邓成联从家中非法绑架，关到闵行区看守所。据知情者丁连浩透露，公安声称在邓成联家中抄到三部手机，里面有在上百人的微信群，内容有法轮功真相。他认为邓成联就是因为传播事实真相被中共囚禁。
 <br/>
 <br/>
 据法轮大法明慧网通讯员从上海发来的消息，邓成联在闵行看守所时，曾24小时被迫戴着手铐、脚镣，长达十多天。邓成联几次绝食反迫害，看守所曾经强制对他导尿，把他生殖器插红肿。又将他绑在所谓的〝死人床〞上插鼻饲，强行灌食。造成邓成联鼻子红肿出血，因为一直被绑〝死人床〞，邓成联的后背也溃烂了。
 <br/>
 <br/>
 邓成联代理律师曾武表示，他们已经对看守所这种行为提出控告。
 <br/>
 <br/>
 邓成联代理律师曾武：〝他现在一直绝食，那看守所他认为对他有迫害的行为。强制灌食，插那个鼻饲管。然后灌食的时候把他五花大绑，绑在那个床上。所以他要控告，而且他认为自己是无罪的。〞
 <br/>
 <br/>
 曾武律师表示，站在法律角度，邓成联是无罪的。
 <br/>
 <br/>
 曾武：〝从我们这个法律角度，第一点肯定他这个从无罪角度（辩护）我们是有充分理由的，不管是国内法律还是国际的法律规定。第二看守所的迫害行为那肯定是违法的。有明确的看守所条例，还包括一些人权公约都可以作为依据的。〞
 <br/>
 <br/>
 为了营救邓成联，近期，他的亲友发起徴签。家乡的父老乡亲纷纷签名，呼吁释放这个好人。
 <br/>
 <br/>
 于先生：〝他家乡的父老都是很踊跃的去签名。还了解到一个情况，村委当初没有钱的时候，邓成联好几次给村委捐款，数额几千不等。所以他们请村委签字时，他们书记第一个带头签字，并说这样好的人不帮，还帮谁呀！副书记也亲自签名了。而且民兵连长都签字了。〞
 <br/>
 <br/>
 要求立即释放邓成联的呼声，不仅来自他国内的亲友和乡亲。海外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 也在华盛顿举行大型的反迫害集会上呼吁释放邓成联等法轮功学员，引发国际社会关注。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/陈建铭
