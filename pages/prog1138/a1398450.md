### 【微视频】自贡银行挤兑背后是地方债的冰山
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】下面请看赵培为我们带来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微视频.html" target="_blank">
  微视频
 </a>
 。
 <br/>
 <br/>
 赵培：2018年11月2号，大陆网站发布消息说，四川
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自贡银行.html" target="_blank">
  自贡银行
 </a>
 被几家股东用〝壳公司〞非法套取了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贷款.html" target="_blank">
  贷款
 </a>
 400亿元，这些贷款已经严重逾期、欠息，形成巨额亏损，还不上了，银行要倒闭了。这条消息被转载到自贡的论坛上，星期五，当地大批储户涌到了自贡银行的十多间分行提款，引发挤兑。
 <br/>
 <br/>
 中共的应对策略三步，首先是把发帖的31岁男子叶某给安上〝造谣罪〞给抓起来了；其次是中共银行监委会自贡分局和自贡银行发布澄清声明，媒体出来辟谣；第三就是调集资金应急了。咱们就想看看这个事情的后续发展，中共是怎么调集资金。中共有可能把自贡银行的储蓄点给关上一段时间来缓冲，中共的这次做法将会成为以后的银行挤兑行为的范本。
 <br/>
 <br/>
 咱们说一下自贡银行的背景，它的前身是自贡市商业银行，2017年改名为自贡银行，号称地方股份制商业银行。这次网友说的三家掏空银行的股东是上市公司华西能源、四川友华、浙江嘉丰房地产。中共的媒体出来辟谣说这三家都不是股东。但是网友挖出的证据显示这三家都是自贡银行的股东。比如据中财网2013年5月2日发布的上市公司[公告]《华西能源：关于签订增资自贡市商业银行合作协议的公告》，称：〝本次增资完成后，公司成为自贡商业银行第一大股东，所持股份不超过增资后的自贡商业银行总股本的20%〞。中共是以谣言辟谣，百姓岂能上当，挤兑就不可避免了。
 <br/>
 <br/>
 除了民间的三大股东之外，自贡银行的政府背景也很深。自贡银行网站说它按照中共自贡市委、市政府〝3年500亿、5年1000亿〞的发展战略要求发展。它的业务方面介绍说，〝与区县政府建立了广泛的合作协议，在工业强市、商务流通、土地征用补偿、旧城拆迁、国企改革、工业园区建设、城市改造、医保系统建设等重大项目中给予地方政府鼎力支持〞。从这些内容可以看出，自贡银行很多资金贷给了地方政府。
 <br/>
 <br/>
 自贡银行挤兑背后是中共上市企业、地方政府债务的巨大冰山。全国的地方政府债务明面上是40万亿，但是隐性债务也有40万亿到60万亿，这就是债务危机。地方政府债务运作就像这张图一样，政府通过城投平台从银行借贷，然后投资基础建设项目。自贡银行就是那个倒霉银行之一，而它的股东里面就有一个房地产公司。这背后的套路是地方政府债务危机，城投平台无法偿还债务，自贡银行肯定会有危机。这就能解释为什么三家股东选择掏空银行跑路。
 <br/>
 <br/>
 对比来说，中共银行、地方政府这种运作手法与P2P平台跑路事件差不多，本质上是套取百姓投资或者存款。不同的是，中共运用政府力量掩盖债务危机要比P2P平台强悍，但是中共也准备了破产跑路，2018年9月13号，中共发了文件《关于加强国有企业资产负债约束的指导意见》，里面规定〝依法依规实施国有企业破产〞，也就是允许城投平台破产，受此连累的银行会更多，这是比P2P严重百倍的金融灾难。
