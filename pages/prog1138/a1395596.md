### 被指涉反习政变 中共军头房峰辉张阳党籍被削
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】中共官方周二发布通告，宣布开除前中共中央军委委员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房峰辉.html" target="_blank">
  房峰辉
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张阳.html" target="_blank">
  张阳
 </a>
 的党籍，同时房峰辉被正式移送审查起诉。此前，房、张二人已经因涉嫌行贿、受贿而被取消了军衔和军籍。
 <br/>
 <br/>
 北京时间10月16日中共喉舌媒体新华社发布公告称，中共中央决定，对原中共中央军委联合参谋部参谋长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房峰辉.html" target="_blank">
  房峰辉
 </a>
 和原中共中央军委政治工作部主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张阳.html" target="_blank">
  张阳
 </a>
 给予
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开除党籍.html" target="_blank">
  开除党籍
 </a>
 处分，并追缴其涉案财物。理由是房、张二人涉嫌〝行贿、受贿、巨额财产来源不明犯罪〞，而且〝情节极为严重，数额特别巨大，影响极其恶劣〞 。
 <br/>
 <br/>
 中共官方同时宣布，军事检察机关对房峰辉案的侦查终结，移送审查起诉。
 <br/>
 <br/>
 中共官方在宣布上述决定的同时，还在公告中批判房张二人系〝双面人〞，指其在生活、经济、政治各方面都〝品行低劣〞。更特别斥责已经上吊自杀的张阳〝最终以自杀方式企图逃避党纪国法的惩处〞，指其〝严重损害〞、〝严重玷污〞了中共党和军队的形象云云。
 <br/>
 <br/>
 资料显示：房峰辉和张阳同为中共前中央军委委员，均拥有上将军衔。2017年下半年两人先后被当局秘密调查。同年11月，张阳抢先在家中上吊自杀身亡；其后，北京当局于2018年1月正式向外界宣布了房峰辉涉嫌行贿、受贿被移送军事检察机关的消息，并废除了房、张二人的上将军衔。
 <br/>
 <br/>
 海外盛传，房峰辉和张阳分别是已经落马的中共中央前军委副主席郭伯雄和徐才厚的亲信。2017年中共十九大前，他们二人密谋仿效1976年抓捕〝四人帮〞的手段发动政变，控制习近平并逼迫其下台。不料政变计划被洩露，二人随即遭到北京当局秘密调查。张阳探知风声后先行自杀，房峰辉则被当局秘密抓捕。
 <br/>
 <br/>
 在中共官方正式对外宣布房、张二人涉贪被查的消息后，中共喉舌媒体和中共军报在发文表态〝拥护中央决定〞时，均强调要〝全面彻底肃清〞郭伯雄和徐才厚的〝流毒〞。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：秦锋）
