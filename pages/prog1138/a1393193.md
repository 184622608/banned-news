### 受贿两亿多 〝大老虎〞杨崇勇一审判无期
------------------------

<div class="wysiwyg">
 【新唐人2018年09月27日讯】中共官方周四通报，河北省人大常委会前副主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨崇勇.html" target="_blank">
  杨崇勇
 </a>
 ，因受贿罪，一审被判
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无期徒刑.html" target="_blank">
  无期徒刑
 </a>
 。杨崇勇被控受贿人民币超过2亿元。
 <br/>
 <br/>
 判决称，2008年到2016年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨崇勇.html" target="_blank">
  杨崇勇
 </a>
 担任中共河北省委常委、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_河北省副省长.html" target="_blank">
  河北省副省长
 </a>
 期间，利用职务便利，为有关单位和个人在企业项目建设、用地指标办理、公司设立审批等事项上提供帮助，索取、收受他人财物，共计折合人民币2.06亿余元，构成受贿罪。
 <br/>
 <br/>
 法院称，杨崇勇有主动提供侦破其它重大案件重要线索等表现，故从轻处罚，判处
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无期徒刑.html" target="_blank">
  无期徒刑
 </a>
 。
 <br/>
 <br/>
 另据《明慧网》报导，杨崇勇在河北任职期间，积极追随江泽民集团迫害法轮功，被《明慧网》多次点名，并列为参与迫害责任人。
</div>
