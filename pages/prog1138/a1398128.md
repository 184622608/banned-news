### 湖北武汉中级法院院长王晨被双开
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】中共纪委监委官网周一通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖北武汉.html" target="_blank">
  湖北武汉
 </a>
 市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中级法院.html" target="_blank">
  中级法院
 </a>
 前院长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王晨.html" target="_blank">
  王晨
 </a>
 ，因涉嫌受贿罪及巨额财产来源不明罪，被双开。
 <br/>
 <br/>
 通报称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王晨.html" target="_blank">
  王晨
 </a>
 对抗审查；不如实报告个人有关事项；收受影响执行公务的礼金，违规购买非上市公司股份并从中获利；索取或非法收受诉讼当事人财物，数额巨大，涉嫌受贿罪；其家庭财产、支出明显超过其收入，差额巨大，本人不能说明来源，涉嫌巨额财产来源不明罪。
 <br/>
 <br/>
 王晨曾任汉江
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中级法院.html" target="_blank">
  中级法院
 </a>
 党组书记、院长；黄石市中级法院党组书记、院长；湖北省高级法院党组成员、副院长、审判委员会委员；武汉市中级法院副院长、代理院长、院长等职。
 <br/>
 <br/>
 另据《明慧网》报导，王晨追随江泽民集团迫害法轮功学员，据不完全统计，王晨任武汉市中级法院院长7年多，武汉地区法院对法轮功学员非法判刑82人。
 <br/>
</div>
