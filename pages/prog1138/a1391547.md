### 广东前环保厅长李清被判15年
------------------------

<div class="wysiwyg">
 【新唐人2018年09月14日讯】中共官媒周五消息，中共广东省环保厅前厅长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李清.html" target="_blank">
  李清
 </a>
 受贿案，终审被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_判刑15年.html" target="_blank">
  判刑15年
 </a>
 ，李清被指受贿人民币两千多万元。
 <br/>
 <br/>
 判决称，2002到2015年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李清.html" target="_blank">
  李清
 </a>
 任广东潮州市长、广东环保局局长、广东环保厅厅长等职期间，在环保事项审批、专项资金补贴、项目招投标、特许经营、人事安排等方面为他人谋取利益，收受他人财物，共计人民币2024万多元、港币52万元、美元6万元等。
 <br/>
 <br/>
 1998年4月到2003年4月，李清曾任广东潮州市委副书记、潮州市长。
 <br/>
 <br/>
 另据《明慧网》报导，广东潮州市三任市长李清、汤锡坤、卢淳杰在任期间，追随中共江泽民集团迫害法轮功，目前都被逮捕起诉。
 <br/>
 <br/>
 据《明慧网》不完全统计，他们在任期间，截至2017年6月，潮州市至少有2名法轮功学员被迫害致死；至少20人遭非法判刑；31人遭非法劳教；遭非法关押、拘留、送洗脑班迫害的至少有32人。
</div>
