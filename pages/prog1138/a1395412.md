### 天津首虎转移资产细节曝光 宝石金条交侄子托管
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】天津首虎、前公安局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武长顺.html" target="_blank">
  武长顺
 </a>
 犯贪腐罪被判死缓。近日，中共官方曝光武长顺转移巨额资产的细节，武在落马前一天，曾将大量宝石金条等巨额财物，交由其侄子藏匿。武长顺除巨贪外私生活也相当糜烂，有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_9个私生子.html" target="_blank">
  9个私生子
 </a>
 ，但经鉴定只有3个是亲生的。
 <br/>
 <br/>
 2017年5月27日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武长顺.html" target="_blank">
  武长顺
 </a>
 被判处死缓，终身监禁，不得减刑、假释。他是继前云南省委书记白恩培后，第二个被判处死缓，终身监禁的〝老虎〞。
 <br/>
 <br/>
 近日，中共裁判文书网披露，武长顺落马前转移巨额财产。武在被抓前一天，将大量财物交予其侄子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武玉峰.html" target="_blank">
  武玉峰
 </a>
 藏匿及保管。
 <br/>
 <br/>
 今年2月，武玉峰犯掩饰、隐瞒犯罪所得罪，被判处有期徒刑5年半，并处罚金人民币20万元。
 <br/>
 <br/>
 据武玉峰交代，2014年7月19日晚，武长顺让他把一个红色大木箱，搬到一个地方的三楼，还给了他一个袋子，内有一个红色木盒子和一个装满宝石的小袋，让他保存好。
 <br/>
 <br/>
 他还透露，2012年至2014年，武长顺陆续在其处保存了2,000多万现金、存单、银行卡、房本，被调查前10天左右，武长顺还交给他28根金条。
 <br/>
 <br/>
 据判决书显示，2014年7月20日，武玉峰在得知武长顺被中央纪委立案调查后，便将武长顺交给他保管的1,874万元人民币、3.25万美元、9.3万欧元、宝石、金条、书画、银行卡等财物分别窝藏、转移至其朋友处藏匿。
 <br/>
 <br/>
 公开资料显示，武长顺在天津公安系统工作44年，担任11年天津市公安局副局长兼公安交管局局长、11年天津市公安局局长，他因残酷迫害天津的法轮功学员而受到周永康的赏识，也因此受到海外追查国际的追查。
 <br/>
 <br/>
 武长顺在天津〝红白两道〞可谓根深树茂，关系众多，号称〝武爷〞。习近平曾说：〝（天津）有个武爷，天津的停车场都成他们家的了，无法无天……十八大后还这么疯狂，前所未闻。〞
 <br/>
 <br/>
 据称，武长顺2014年7月20日被调查时，官方一共拉走了12辆卡车的赃物，当中包括了逾亿元人民币的现金。
 <br/>
 <br/>
 武长顺落马后，有天津网民披露消息，称巨贪武长顺私生活糜烂，除包养警花外，还与一名女艺人和数名女大学生非法同居，先后生育了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_9个私生子.html" target="_blank">
  9个私生子
 </a>
 。武被调查后，〝经组织鉴定，9个私生子中只有3个是他亲生的。〞
 <br/>
 <br/>
 还有报导称，1992年初，天津南开公安分局在整治娱乐场所时，武长顺因与一名女性从事不正当活动，被抓了个正着。时任天津市副市长、公安局局长宋平顺亲自去分局，将他〝救〞出来。这件事当时在公安局内都传遍了。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
