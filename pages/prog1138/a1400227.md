### 国际刑警组织换主席 韩国人接替孟宏伟
------------------------

 【新唐人北京时间2018年11月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警.html" target="_blank">
  国际刑警
 </a>
 组织前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_主席.html" target="_blank">
  主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被捕一个多月后，11月21日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 藉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正阳.html" target="_blank">
  金正阳
 </a>
 当选国际刑警组织主席，接替孟宏伟。美国方面表示，坚决支持韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_候选人.html" target="_blank">
  候选人
 </a>
 当选。
 <br/>
 <br/>
 当地时间11月21日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警.html" target="_blank">
  国际刑警
 </a>
 组织在阿联酋迪拜举行年度大会，来自190多个国家的1000多名代表与会，最终选举
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 藉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正阳.html" target="_blank">
  金正阳
 </a>
 （Kim Jong Yang，音译），为新任国际刑警组织
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_主席.html" target="_blank">
  主席
 </a>
 。
 <br/>
 <br/>
 美国国务卿蓬佩奥（Mike Pompeo）表示，美国坚决支持韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_候选人.html" target="_blank">
  候选人
 </a>
 金正阳出任国际刑警组织新任主席。
 <br/>
 <br/>
 国际刑警组织前主席、中共公安部前副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，于9月29日从法国前往中国后神秘消失，孟妻10月5日向法国警方报案后，立即引发国际舆论界的哗然。
 <br/>
 <br/>
 在国际舆论的压力之下，中纪委10月7日深夜通报，孟宏伟涉嫌违法，正接受调查。同一天，国际刑警组织接到孟宏伟的辞职信。
 <br/>
 <br/>
 孟宏伟辞职后，国际刑警组织为确保组织日常运作不受影响，需要尽快选出新主席。于是来自韩国的国际刑警组织高级副主席金正阳，与俄罗斯内务部资深官员普罗科普丘克（Alexander Prokopchuk），开始角逐国际刑警组织主席一职。
 <br/>
 <br/>
 美国联邦参议员夏亨（Jeanne Shaheen）此前发表声明，呼吁国际刑警组织大会的会员，拒绝支持普罗科普丘克出任主席。他们担心俄国可能会滥用主席职务，锁定政治异己。认为选举普罗科普丘克为新任主席，如同〝引狼入室〞。
 <br/>
 <br/>
 国际刑警组织成立于1914年，总部设在法国里昂，现有192个成员国，是唯一的全球性警务合作组织，也是仅次于联合国的第二大国际组织。
 <br/>
 <br/>
 国际刑警组织主席由全体大会选举产生，任期4年，4年选举一次，不得连选。作为该组织最高负责人，负有确保该组织各项活动遵循会议的各项决议、指导和监督秘书长工作等职责。
 <br/>
 <br/>
 64岁的孟宏伟，2016年11月当选国际刑警组织主席，是首个当选国际刑警组织主席的中国人，任期至2020年。
 <br/>
 <br/>
 孟宏伟上任之初，就有人权组织担心，致力于破坏人权的中共，或会利用组织作为通缉海外异见人士的平台。
 <br/>
 <br/>
 因此，孟宏伟当时出任主席一职，被外界认为是中共利用贿赂渗透了国际刑警组织，以及中共与法国幕后交易的结果，同时也意味着后者已经堕落为中共的外围组织，从而在国际社会已经丧失了公信力，正在逐渐淡出人们的视野。
 <br/>
 <br/>
 如今韩国候选人金正阳出任国际刑警组织新任主席，外界期望该组织能真正的履行职责使命，提升其公信力，让国际对该组织重拾信心。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
