### 郑晓松坠亡后 中共十九届中委面临大换血
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办主任.html" target="_blank">
  中联办主任
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠楼身亡.html" target="_blank">
  坠楼身亡
 </a>
 ，成为首名离世的中共十九届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中央委员.html" target="_blank">
  中央委员
 </a>
 。按中共惯例，将有候补中委递补。与此同时，由于多名现任省委书记临近退休，预料中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十九届中委.html" target="_blank">
  十九届中委
 </a>
 将进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大幅调职.html" target="_blank">
  大幅调职
 </a>
 。
 <br/>
 <br/>
 公开报导显示，中共十九届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中央委员.html" target="_blank">
  中央委员
 </a>
 会现有委员204人。他们的主要身份有：中共国家领导人，中央直属机构主要负责人，国务院组成部门正部级主要负责人，中共军队正大军区级主要领导，各省、自治区、直辖市党委书记及省长（自治区主席、市长）；各团体主要负责人等。
 <br/>
 <br/>
 中央委员有权提名中共政治局的成员。
 <br/>
 <br/>
 香港《明报》10月29日报导，明年内达到65岁正省部级退休年龄的高官包括：云南省委书记陈豪、贵州省委书记孙志刚、山西省委书记骆惠宁、辽宁省委书记陈求发、民政部长黄树贤、农业农村部长韩长赋及新疆政府主席雪克莱提等一批官员。
 <br/>
 <br/>
 除了少数民族官员可以破例，其他人如果未能更上层楼，都要卸任行政职务。
 <br/>
 <br/>
 在204名中委之外，还有172名候补委员，按照中共惯例，当中央委员因故出缺时，候补委员可以依得票数的高低来递补中央委员。
 <br/>
 <br/>
 由于多名现任省委书记即将临近退休年龄大限，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十九届中委.html" target="_blank">
  十九届中委
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大幅调职.html" target="_blank">
  大幅调职
 </a>
 ，预料将从明年开始揭开序幕。
 <br/>
 <br/>
 10月20日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠楼身亡.html" target="_blank">
  坠楼身亡
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办主任.html" target="_blank">
  中联办主任
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 ，是首名出事的第十九届中央委员。
 <br/>
 <br/>
 在郑晓松之前，还有一名〝出事〞的十九届中央委员。8月16日，因毒疫苗案，时任国家市场监督管理总局党组书记、副局长毕井泉引咎辞职，他是首位被处理的十九届中央委员。
 <br/>
 <br/>
 不过，毕井泉并非因贪腐问题被处理，也没有死于非命，目前仍是中央委员。因此，郑晓松才算是第一个〝倒下〞的。
 <br/>
 <br/>
 中央委员会是中共的顶层建筑，其成员是否稳定往往是观察中共政权是否稳固的风向标。此前的十八届中央委员会，在习近平反腐风暴下大批中委落马被查。
 <br/>
 <br/>
 根据中共官方数据，十八大以来，有43名中央委员和中央候补委员、9名中央纪委委员被立案审查。
 <br/>
 <br/>
 时政评员郑中原说，中共十八届中央委员会面目全非，除了说明这些人当年在选拔的时候就有问题，证明相当一部分人是各派势力妥协的产物。
 <br/>
 <br/>
 比如，在中共十八大前，江泽民、曾庆红、周永康、令计划、郭伯雄、徐才厚等人，直接或间接操纵十八大人事安排，一大批买官之徒得以青云直上。
 <br/>
 <br/>
 习近平曾对十九大中委和候补中委候选人亲自把关审核。但观察发现，十九大上位的中央委员和中央候补委员中，仍有不少本身与周永康、令计划等〝野心家〞们关系密切，且腐败丑闻缠身，仍是边腐边升，其中包括张春贤、郭声琨、黄明、韩正、刘奇葆、李鸿忠、吉炳轩、傅政华等等。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：祝馨睿）
