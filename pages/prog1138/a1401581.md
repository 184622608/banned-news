### 内蒙古官场震荡持续 第五巡视组原组长被双开
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】今日，中共官方通报称，内蒙古自治区中共党委第五巡视组原组长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨静波.html" target="_blank">
  杨静波
 </a>
 被双开。
 <br/>
 <br/>
 通报称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨静波.html" target="_blank">
  杨静波
 </a>
 违反政治纪律，对抗组织审查；在干部选拔任用过程中收受他人财物；违规收受礼金；利用职务之便为他人谋取利益并收受财物，涉嫌受贿罪；家庭财产和支出明显超过其合法收入，且数额特别巨大，不能说明来源，涉嫌巨额财产来源不明罪。
 <br/>
 <br/>
 通报还称，杨静波已被中共开除党籍、开除公职处分；收缴其违纪违法所得；其涉嫌犯罪问题被移送检察部门审查起诉。
 <br/>
 <br/>
 杨静波，蒙古族人，1959年4月出生，曾任通辽市委副书记、组织部部长等职，2017年6月开始任内蒙古自治区党委第五巡视组组长。
 <br/>
 <br/>
 早前，今年9月8日，内蒙古纪委监委网站发布消息称，内蒙古自治区党委第五巡视组组长杨静波涉嫌严重违纪违法，目前正在接受纪律审查和监察调查。
 <br/>
 <br/>
 陆媒曾报导，今年7月，杨静波的老同事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_许亚林.html" target="_blank">
  许亚林
 </a>
 因受贿罪、贪污罪、挪用公款等罪，一审被判无期徒刑。许亚林从2007年开始先后任通辽市经济技术开发区管委会书记、通辽市委常委、通辽经济技术开发区党工委书记，通辽市副市长等职，与杨静波有着多年交集，后于2015年11月落马。
 <br/>
 <br/>
 近来，内蒙古官场持续震荡，中共官媒11月30日报导称，内蒙古自治区农牧业厅原副厅长、党组成员，兽医局原局长许燕辉（副厅级）因涉嫌受贿罪、私分国有资产罪、挪用公款罪，被内蒙古自治区检察院锡林郭勒盟分院逮捕。
 <br/>
 <br/>
 11月21日，中共中纪委监委官网发布消息称，内蒙古乌兰察布市人大常委会原副主任、察右中旗原旗委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵向红.html" target="_blank">
  赵向红
 </a>
 被调查。
 <br/>
 <br/>
 11月6日，中共内蒙古纪委监委官方微信公众号发布消息称，内蒙古自治区发改委原副主任、能源局原局长赵文亮被〝双开〞，即开除党籍、开除公职。
 <br/>
 <br/>
 10月30日，内蒙古自治区公安厅原副厅长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟建伟.html" target="_blank">
  孟建伟
 </a>
 落马。而在此之前，多名内蒙古官员被查，而其中尤以公安系统官员最为多。其中包括呼和浩特市公安局副局长杜宝君被查。
 <br/>
 <br/>
 杜宝君曾与省部级官员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵黎平.html" target="_blank">
  赵黎平
 </a>
 是上下级关系，跟冤案〝呼格吉勒图案〞专案组组长冯志明更是同事关系。
 <br/>
 <br/>
 （记者欧阳静报导//责任编辑：李泉）
