### 内蒙古原政法委书记邢云被查
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】中共纪委监委网站周四通报，内蒙古人大常委会原副主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邢云.html" target="_blank">
  邢云
 </a>
 ，因〝涉嫌严重违纪违法〞，正在接受调查。邢云是中共十八大后，内蒙古第六名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 的省部级官员。
 <br/>
 <br/>
 通报没有说明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邢云.html" target="_blank">
  邢云
 </a>
 被查的具体原因。
 <br/>
 <br/>
 邢云长期在内蒙古任职，曾任内蒙古自治区伊克昭盟盟委书记，鄂尔多斯市委书记；2001年12月到2006年12月，任内蒙古党委常委、包头市委书记；2006年12月到2012年2月，任内蒙古政法委书记。2016年1月退休。
 <br/>
 <br/>
 内蒙古是江派前常委刘云山发迹的地方，刘云山在内蒙古有庞大关系网。中共十八大以来，内蒙古官场被持续清洗，除了6名省部级官员，还有50多名厅官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 。
 <br/>
 <br/>
 另外，邢云在任期间，积极参与迫害法轮功学员，被〝追查迫害法轮功国际组织〞列入追查名单。
</div>
