### 北京反恐新增两大高层 习罕见未出席指示
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】日前，由中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 成立的中共〝国家反恐怖工作领导小组〞扩大成员，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交部.html" target="_blank">
  外交部
 </a>
 副
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_部长.html" target="_blank">
  部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乐玉成.html" target="_blank">
  乐玉成
 </a>
 和国家安全部部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈文清.html" target="_blank">
  陈文清
 </a>
 跻身副组长行列。但陆媒没有提及习近平是否做出指示。
 <br/>
 <br/>
 陆媒10月12日报导，中共国家反恐工作领导小组领导层出现人事调整，根据公安部网站消息，9月29日，国家反恐怖工作领导小组在京召开电视电话会议，透露出〝一正五副〞的领导编制，纳入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交部.html" target="_blank">
  外交部
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国安.html" target="_blank">
  国安
 </a>
 部的高层。
 <br/>
 <br/>
 中共国家反恐怖工作领导小组副组长、国务院副秘书长孟扬主持会议；中共国务委员、国家反恐怖工作领导小组组长、公安部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_部长.html" target="_blank">
  部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵克志.html" target="_blank">
  赵克志
 </a>
 出席并讲话。
 <br/>
 <br/>
 中共国家反恐怖工作领导小组副组长——国家安全部党委书记及部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈文清.html" target="_blank">
  陈文清
 </a>
 、外交部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乐玉成.html" target="_blank">
  乐玉成
 </a>
 、军委联合参谋部参谋长助理陈光军、公安部反恐专员刘跃进等人，出席了会议。
 <br/>
 <br/>
 据报，中共〝国家反恐怖工作领导小组〞是国务院的议事协调机构，成立2013年，前身是2001年成立的〝国家反恐怖工作协调小组〞，已落马的中共前政法委书记周永康，与现任政法委书记孟建柱都曾担任小组组长。
 <br/>
 <br/>
 2013年11月，中共高层出于对稳固中共政权的需要，又成立国家安全委员会，主任是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 。
 <br/>
 <br/>
 2015年11月，官方公布的消息显示，习近平亲信、时任武警司令王宁，成为中共〝国家反恐怖工作领导小组〞的副组长，时任中共国务委员、公安部部长郭声琨，成为这个小组的组长。
 <br/>
 <br/>
 根据中共公安部官网今年5月份消息，该小组的副组长包括：孟扬、陈光军、中共武警部队司令员王宁，及中共公安部党委委员兼副部长李伟。其中，李伟在8月初卸任中共公安部党委委员兼副部长职务。
 <br/>
 <br/>
 从今次中共〝国家反恐怖工作领导小组〞最新组成来看，武警已经撤出该小组的领导层，转而由中共国安部与中共外交部两个副部级官员进驻。
 <br/>
 <br/>
 近年来，江泽民、曾庆红等江派人马多次利用制造重大恶性社会公共安全事件搅局，评论人士分析，在眼下中美贸易战令北京处于外患内忧的危局时刻，中共国家反恐怖工作领导小组增编，目的可能是防止江氏集团利用暴力袭击事件来混水摸鱼，制造混乱。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
</div>
