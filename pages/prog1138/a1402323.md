### 华为对〝捞人〞悲观   内部发文梁华代理孟晚舟
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司副董事长兼CFO孟晚舟在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 被逮捕，7日举行听证会，决定保释或引渡回
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 。华为似乎对〝捞人〞悲观。一份华为内部文件称，现任董事长梁华将暂时代行孟晚舟的CFO职务。
 <br/>
 <br/>
 香港01报导说，12月7日，一份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 董事会常委会的通知文件，称现任董事长梁华将会暂时代行CFO的职务。
 <br/>
 <br/>
 公开资料显示，现年54岁的梁华拥有博士学位，自95年加入华为后，历任供应链总裁、流程与IT管理部总裁、全球技术服务部总裁、审计委员会主任等。
 <br/>
 <br/>
 今年3月22日，在华为董事会换届选举中，梁华出任新董事长，同时担任公司持股员工理事会理事长。
 <br/>
 <br/>
 华为内部迅速发布文件由梁华代理孟晚舟代行CFO的职务，外界认为华为对孟晚舟能否保释悲观。
 <br/>
 <br/>
 12月6日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 媒体称，孟晚舟是在加拿大转机时，被加国政府代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 政府暂时扣留。
 <br/>
 <br/>
 美国指控孟晚舟涉嫌违反对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 的贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 ，正在寻求引渡她回美国受审，7日孟晚舟一案举行听证会，决定保释或引渡回美国。
 <br/>
 <br/>
 美国财政部外国资产控制办公室（OFAC）负责执行对伊朗的贸易制裁，视华为违反制裁令，对国家安全及外交关系严重威胁。一旦被裁定孟晚舟违反制裁令罪成，最高可被罚款2000万美元及入狱30年。
 <br/>
 <br/>
 据悉，引渡孟晚舟到美国受审，是源于1971年12月美国与加拿大在华府签订的《美加国际引渡条约》，条约规定美加两国可将其境内被对方指控或判定犯罪的人，应对方请求，移交该国审判或处罚。
 <br/>
 <br/>
 孟晚舟是华为创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的女儿，华为的接班人。据悉，孟晚舟和任正非均持有香港身份证。
 <br/>
 <br/>
 华为官网显示，孟晚舟毕业于华中理工大学，硕士学历，于1993年加入华为，历任公司国际会计部总监、华为香港公司首席财务官、账务管理部总裁，现任副董事长，是华为的接班人。
 <br/>
 <br/>
 华为公司于1987年在中国深圳成立，自2012年以来一直是全球最大的电信设备制造商。但华为从未上市，管理架构相当神秘，被指长期帮助中共监视中国民众。
 <br/>
 <br/>
 任正非曾是中共军队团职干部，刚卸任的董事长孙亚芳加入华为之前，曾长期在中共国安部任职。华为因有中共政府或军方背景，被认为其会在中共当局强压下，在其世界各地销售的硬体中植入监听装置或干扰通讯等。
 <br/>
 <br/>
 随着华为在国际上的不断扩张，美国等西方国家忧虑日深，许多国家与地区逐渐封杀华为产品。
 <br/>
 <br/>
 早在2012年美国众议院调查委员会曾指，华为可能涉及间谍活动，威胁美国国家安全。目前，美国除了禁售华为手机外，美国总统川普8月又签署法案，禁止美国政府人员使用华为产品。
 <br/>
 <br/>
 中美贸易战持续升温后，中美冲突扩散至经济、政治、军事、文化等各个领域。美政府先是制裁中兴通讯，并将目标对准中共的科技产品，外界皆预期华为会会是下一个目标。
 <br/>
 <br/>
 此次加国因应美国要求，以涉嫌违反美国对伊朗的出口禁令逮捕孟晚舟，外界关注是否是美方对华为开的第一枪。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
