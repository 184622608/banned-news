### 揭秘：〝六四元凶〞联名举报江泽民 江吓得一身冷汗
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】在〝六四〞时担任中共北京市长的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈希同.html" target="_blank">
  陈希同
 </a>
 被以贪污罪入狱，陈希同案是原中共党魁江泽民一手操办的。有观察家指，陈因反对江泽民，成为中共权力斗争的牺牲品。陈希同出事前曾联合省级官员，给邓小平写联名信
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_举报江泽民.html" target="_blank">
  举报江泽民
 </a>
 ，江吓得一身冷汗，于是要致陈希同于死地。
 <br/>
 <br/>
 原北京市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈希同.html" target="_blank">
  陈希同
 </a>
 曾做报告认定1989年的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四学潮.html" target="_blank">
  六四学潮
 </a>
 〞是〝暴乱〞、当局镇压是〝平暴〞，陈因此被认为是〝六四元凶〞。对此，陈解释：〝中央让我做报告，我不能不做，（对这份报告）我一个字也没有参加讨论，一个标点符号也没有改，但是我承担责任。〞
 <br/>
 <br/>
 1995年，陈希同在声势鼎盛之际，因涉及原北京副市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王宝森.html" target="_blank">
  王宝森
 </a>
 自杀案被以经济犯罪罪名判刑16年。但观察家普遍认为，陈希同下台是因为他与当时中共党魁江泽民内斗失败。
 <br/>
 <br/>
 2012年5月底，陈希同在香港出版名为《陈希同亲述众口铄金难铄真》的回忆录，由前中共总书记赵紫阳智囊姚监复执笔，赵紫阳秘书鲍彤之子鲍朴出版。鲍朴说，这是罕见的、由被指是〝六四〞事件幕后黑手的人，第一手谈及当年详情。
 <br/>
 <br/>
 在书中，陈形容他的贪腐案是文革以来最大冤案，暗批前中共总书记江泽民是整他的罪魁祸首，并试图撇清〝六四〞责任，称自己只是一名傀儡。
 <br/>
 <br/>
 <h4>
  陈希同指江泽民是大骗子
 </h4>
 <br/>
 陈希同被指控自1991年7月至1994年11月，在对外交往中接受贵重礼物22件，共计价值人民币55.5万余元。
 <br/>
 <br/>
 陈希同在书中披露，把从外国政要收到的礼物转给北京艺术剧院（BPAT），实际上是来自江泽民的指令。
 <br/>
 <br/>
 他说：〝有一天，江泽民视察了北京艺术剧院，发现他们资金短缺。他让人给我带信，让我帮助北京人艺。〞
 <br/>
 <br/>
 于是，陈希同去了香港，从富豪那里寻求捐助。他当时希望筹集一亿元为剧院成立发展基金，并决定把外国政要赠送的所有礼物捐赠给该基金。
 <br/>
 <br/>
 陈希同表示：〝根据规定，我们或者把礼物捐给慈善机构，或者上交政府。我所做的完全合法，一切都妥善地做了记录，并有收据。〞他认为，自己只是把这些礼品借到办公室估价，不构成犯罪。
 <br/>
 <br/>
 当年陈希同案是原中共党魁江泽民一手操办，陈此前引述曾任北京市委书记李锡铭的话说，江泽民是一个大政治骗子，投机分子。
 <br/>
 <br/>
 陈希同说：〝在权力斗争中，任何事情均会发生。对手会用一切卑鄙手段去夺取权力。〞
 <br/>
 <br/>
 有观察家认为，陈希同因反对当时的中共国家主席江泽民，成为中共权力斗争的牺牲品。
 <br/>
 <br/>
 <h4>
  陈希同
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_举报江泽民.html" target="_blank">
   举报江泽民
  </a>
 </h4>
 <br/>
 据《江泽民其人》一书记述，当年，江泽民踏着〝六四〞学生鲜血上台后，最怕军队不跟他这个没摸过枪的军委主席走。当年邓小平的健康每况愈下，江在十四大（1996年）从军队核心赶走杨家兄弟，并大力培植亲信。然后，江泽民开始盘算把矛头指向北京市这个重要政治滩头阵地。
 <br/>
 <br/>
 据悉，陈希同与邓小平关系非常好。江泽民被提到北京，陈心里不平衡，也看不起江。在1995年初，陈希同曾联合省级干部，给邓小平写联名信举报江泽民。
 <br/>
 <br/>
 但是邓把信交给薄一波，想让薄一波看一看他推荐的江泽民是个什么货色。而薄一波却把江叫来，据悉，江一身冷汗、战栗不已，最后薄、江达成交易，薄帮助搞倒陈希同，从周围下手，而江给薄熙来及亲信加官晋爵。
 <br/>
 <br/>
 薄熙来这些年能在仕途上飞黄腾达，就是凭着父亲薄一波手里有陈希同向邓小平告江的那张状纸。内容只有四个人知道：陈希同、邓小平、薄一波和江泽民。江泽民于是要致陈希同于死地。
 <br/>
 <br/>
 随后江泽民与薄一波合谋，先从北京的副市长王宝森下手，对付陈希同。
 <br/>
 <br/>
 1995年4月4日早上，王宝森死在北京近郊怀柔县一个叫崎峰茶的山上，当时王宝森被官方报导为饮弹自杀，死时衣着完整，右手紧握一把手枪，食指扣在扳机上，枪口指向左胸。
 <br/>
 <br/>
 大陆坊间认为王宝森是被自杀。中共国安内部消息曾透露，王宝森死时，身边有江泽民派的国安特工在场。而当时，在现场警察用探雷器只找到了子弹头，该子弹壳已经被踩入土里，而王死的地方人迹罕至。
 <br/>
 <br/>
 王宝森死后，江泽民一手操办了陈希同案。1995年4月，陈希同因为王宝森自杀案引咎辞职，1998年7月被北京高级法院以贪污罪、玩忽职守罪判处有期徒刑16年。
 <br/>
 <br/>
