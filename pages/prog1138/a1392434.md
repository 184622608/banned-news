### 中共能源局局长努尔‧白克力落马
------------------------

<div class="wysiwyg">
 【新唐人2018年09月21日讯】中共官媒周五通报，中共发改委党组成员、副主任，国家能源局党组书记、局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_努尔‧白克力.html" target="_blank">
  努尔‧白克力
 </a>
 ，因涉嫌〝严重违纪违法〞，目前正接受调查。他是继刘铁男之后，第二个被查的国家能源局局长。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_努尔‧白克力.html" target="_blank">
  努尔‧白克力
 </a>
 在新疆任职31年，1998年2月起，先后任乌鲁木齐市委副书记、市长，新疆维吾尔自治区党委常委、政法委副书记、党委副书记、政府主席等职。2014年12月起，任国家发改委党组成员、副主任，国家能源局党组书记、局长。
 <br/>
 <br/>
 据报，新疆曾是江派周永康的地盘，受其亲信王乐泉、张春贤长期把持，努尔‧白克力就是王乐泉提拔的。
 <br/>
 <br/>
 中共发改委被认为是贪腐窝点。已
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 的前发改委副主任兼国家能源局局长刘铁男，被指是江泽民的〝财务管家〞，长期为其输送利益。
 <br/>
 <br/>
 努尔‧白克力落马，是与新疆官场还是与能源系统有关，目前还不清楚。
 <br/>
 <br/>
 另据《明慧网》报导，努尔‧白克力积极追随江泽民集团迫害法轮功，被列为迫害责任人。
 <br/>
 <br/>
 海外〝追查迫害法轮功国际组织〞也对其相关责任进行追查。
</div>
