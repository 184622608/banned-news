### 习近平礼服成话题 传真正内涵被江泽民封杀
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 近期访问西班牙，身着穿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中山装.html" target="_blank">
  中山装
 </a>
 引起舆论关注，但外界发现习的这身
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中式礼服.html" target="_blank">
  中式礼服
 </a>
 ，不是真正意义上的中山装。真正的中山装另有内涵，在电视剧《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_走向共和.html" target="_blank">
  走向共和
 </a>
 》中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙中山.html" target="_blank">
  孙中山
 </a>
 对此有解读，但相关内容由于太过敏感，遭江泽民封杀。
 <br/>
 <br/>
 近日，中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 一行抵达西班牙进行国事访问，在西班牙举行的欢迎晚宴上，包括中共外长王毅在内的所有中方官员，都身着中山服出席活动。引发舆论关注。
 <br/>
 <br/>
 但外界发现，习近平穿的这款中式服装，采用3个暗兜，上身只有左胸兜，无兜盖，佩饰帕巾，前襟纽扣隐藏，侧边绣花。此款中式服装，并非真正意义上的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中山装.html" target="_blank">
  中山装
 </a>
 。中国人所称的〝中山装〞，始自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙中山.html" target="_blank">
  孙中山
 </a>
 。孙中山被誉为〝中国首位开创自由民主之路的先驱〞，在台湾则被尊为〝国父〞。
 <br/>
 <br/>
 中山装外套领子为八字形翻领、前襟有单排五颗扣子和有袋盖的4个口袋，另外袖口还有3颗纽扣。在中共建政初期，中山装是中共第一代党魁毛泽东等领导人的着装风格。因此，这种服装也被西方称为〝毛服装〞，也成了〝极左〞和〝守旧〞的象征。
 <br/>
 <br/>
 中共10年文革浩劫结束后，中共领导人在公开活动中，改变了清一色中山装的风格，开始穿夹克和西装。而西装成了〝西化〞和〝开放〞的象征。
 <br/>
 <br/>
 英媒BBC称，习近平上台后提倡民族复兴〝中国梦〞，近期习访问西班牙的这身
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中式礼服.html" target="_blank">
  中式礼服
 </a>
 ，是一百年来中国人在隆重的正式场合从中山装转向西装，再向中式服装的回归。
 <br/>
 <br/>
 而真正的中山装另有内涵，孙中山在《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_走向共和.html" target="_blank">
  走向共和
 </a>
 》中，对此有解读，但相关内容由于太过敏感，2003年已经遭到江泽民封杀。
 <br/>
 <br/>
 <h4>
  孙中山演讲中介绍中山装
 </h4>
 <br/>
 电视剧《走向共和》是中国一部近代史电视连续剧，共60集，剧情背景为自清朝慈禧太后归政光绪皇帝，至民初袁世凯称帝失败，约20年间的中国历史。叙事围绕六位主角发展：慈禧太后、李鸿章、光绪皇帝、康有为、袁世凯和孙中山。
 <br/>
 <br/>
 在最后一集里，孙中山在演讲中，提到〝中山装〞是民国六年张勋复辟期间设计出的服装。
 <br/>
 <br/>
 他说：我今天穿的这身衣服有点古怪是吧，连裁缝都说是很奇怪的。但是我要说这是，这是为了完善共和。这就是共和的衣服。这边，我设计了三颗扣子，共和的理念，就是平等、自由、博爱。这边也有三颗扣子──民族、民权、民生。
 <br/>
 <br/>
 那宪法呢？我说的不是三权宪法。我发明了个新词，叫五权宪法。（孙中山用手拍着中山装上的四个口袋）这里装的是立法权，这儿装的是行政权，这儿装的是司法权，这三权你们都很熟悉，叫间接民权。
 <br/>
 <br/>
 我情有独钟的是直接民权。要让普通的民众都有直接参政、议政的权力！一个是考试权，我们中国古代就有考试的传统，后来把科举废除了，当然，这对后来大兴新学有好处，可当官就不再考试了，这不好，所以我们要把考试权还给民众。
 <br/>
 <br/>
 今后，凡行政用人，一定要经过考试，不管是谁！还有一个是弹劾权。没地儿装了，（孙中山解开上衣纽扣，指着内侧左边的口袋）装在这儿，弹劾权！为什么要把弹劾权藏在里面呢？因为它是民众的杀手鐧，它说不定什么时候就突然杀出来，弹劾你。所以你要战战兢兢的当官，老老实实的为民做事。
 <br/>
 <br/>
 不过，在2003年中共央视首播《走向共和》引起轰动后，很快遭到禁播。
 <br/>
 <br/>
 之所以禁播，消息称，时任中共中央军委主席江泽民，认为剧情有影射他垂帘听政之嫌而大为不满，写了条子给中共研究室主任王沪宁，加上一些学者意见后呈报胡锦涛决定禁播。
 <br/>
 <br/>
 中共前党魁江泽民2002年恋权留任中共军委主席一职，2004年〝全退〞后仍旧退而不休〝老人干政〞，被民间号称为慈禧太后。
 <br/>
 <br/>
 不过遭禁播的原因，还有另外两种说法，一是电视剧情描述洋务运动时以古讽今，清政府只改革经济不改革政治，造成大面积腐败，颇似中共政局；二是在描述辛亥革命时倡导民主、自由、平等、博爱的观点，刺痛了中共的神经。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
