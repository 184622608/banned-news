### 中共贵州前副省长王晓光被〝双开〞
------------------------

<div class="wysiwyg">
 【新唐人2018年09月20日讯】中共官媒周四通报，贵州前副省长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王晓光.html" target="_blank">
  王晓光
 </a>
 被〝双开〞，并立案审查。同一天，湖北省黄冈前公安局局长汪治怀被逮捕。
 <br/>
 <br/>
 通报称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王晓光.html" target="_blank">
  王晓光
 </a>
 封官许愿，对抗审查；大搞特权，违规公款豪华装修住所，大吃大喝；违规为他人职务提拔、企业承揽工程提供帮助，收受好处费；违规从事营利活动并获取巨额利益；与多名女性搞钱色、权色交易；干预司法活动；生活奢靡等。
 <br/>
 <br/>
 王晓光4月1日落马，是中共十八大以来，贵州省第三名落马的〝老虎〞。
 <br/>
 <br/>
 另据《明慧网》报导，王晓光在担任遵义市委副书记、市长期间，对当地迫害法轮功学员负有责任。
 <br/>
 <br/>
 周四通报被捕的前湖北省黄冈市政府前副市长、政法委副书记，公安局长汪治怀，也因对该地区法轮功学员被迫害负有责任，多次被《明慧网》列为参与迫害责任人。
</div>
