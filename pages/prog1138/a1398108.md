### 西媒：习近平进博会〝老调重弹〞 投资者非常警惕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 的背景下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 11月5日在上海首届〝国际进口博览会〞的演讲，引发国际舆论关注。不过，多家外媒报导称，习的演讲又是〝老调重弹〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空洞的承诺.html" target="_blank">
  空洞的承诺
 </a>
 。投资者们对此非常警惕，他们不敢轻易在中国进行投资。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 的演讲中承诺，将进一步削减进口关税，并开放中国国内市场。不过，外界普遍认为，这些都是〝老调重弹〞。关于〝削减进口关税〞的计划，早在9月份中共官方就已经宣布过。
 <br/>
 <br/>
 对于习近平的讲话，香港加拿大皇家银行资本市场的亚洲外汇策略负责人特瑞恩（Sue Trinh）认为，习近平只是重复了很多已经在计划中的政策。而对于投资者关注的，显示北京会认真对待开放经济的具体措施，习近平并未提及。
 <br/>
 <br/>
 彭博社也报导称，习近平并未在演讲中释放很快将出台突破性措施的信号，以应对中国股市下跌、人民币贬值的不利局面。
 <br/>
 <br/>
 此外，习近平的讲话虽然做出了多项承诺，但缺乏具体措施与改革的明确时间表。外界认为，这很可能是北京又一次地〝放空炮〞。
 <br/>
 <br/>
 上海欧盟商会主席安德烈（Carlo Diego D’Andrea）对此感到失望，称他们不能在改革只是〝有望到来〞的基础上，就让在中国的欧洲公司首席执行官（CEO）们开展商贸活动。
 <br/>
 <br/>
 今年以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 不断升级，导致中国经济迅速下滑。美国《华尔街日报》报导称，习近平承诺将进一步扩大对外开放，并在上交所设科创板试点注册制，加大资本市场的改革力度。这表明北京希望通过加大改革开放等措施，来抵抗经济下滑压力，缓解中美贸易摩擦。
 <br/>
 <br/>
 不过，英媒BBC称，习近平并没有回应美国关切的，有关中共对国有企业实行补贴的问题。事实上，对于解决中美贸易争端问题，北京至今没有拿出实质性具体措施。
 <br/>
 <br/>
 对于习近平的讲话，市场做出了反应。截至11月5日上午收盘，香港恒生指数下跌2.65%，中国上证综指和深证成指均下跌逾1%。英媒BBC分析认为，现在的投资者对于〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空洞的承诺.html" target="_blank">
  空洞的承诺
 </a>
 〞表现得非常警惕。
 <br/>
 <br/>
 在此之前，11月1日，习近平同美国总统川普（特朗普）进行了长时间通话，双方重点讨论了贸易问题。习近平说〝我愿同总统先生在出席阿根廷二十国集团领导人峰会（G20）期间再次会晤，就中美关系及其他重大问题深入交换意见。〞
 <br/>
 <br/>
 川习通话结束后美股应声上扬，中国股市也难得一见的反弹。这显示，中美领导人的良好沟通，对于改善中美贸易争端问题，释放了利好信号。
 <br/>
 <br/>
 中美首脑通话后，中共媒体开始努力〝展望贸易和解〞。但美国《华盛顿时报》11月2日报导，川普总统在前一天接受专访时说，尽管中美两国在贸易问题上有所进展，但他并不认为双方会在本月底的G20首脑峰会期间宣布达成协议。这无异于给中共的宣传泼了一盆冷水。
 <br/>
 <br/>
 另外，在中共官媒对于川习通话的报导中，北京表示，〝中方已就中美关系多次阐明原则立场〞。这似乎也传递了中方在关键问题上不会让步的信息。而习近平此次在进博会的演讲中，并未公布突破性的举措，似乎也佐证了这一点。
 <br/>
 <br/>
 不过，川普对于解决中美贸易纠纷表现得很有信心。他在专访时表示，最终美国肯定会跟北京达成一个非常合理的协议。〝我可以告诉你，他们希望跟我们达成协议。〞
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：文慧)
