### 【新闻看点】外交丑闻迭爆 中共蠢到无朋友
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】【
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新闻看点.html" target="_blank">
  新闻看点
 </a>
 】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交丑闻.html" target="_blank">
  外交丑闻
 </a>
 迭爆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 蠢到无朋友：一年前在北京召开的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 大会上，这个组织的负责人、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 表示，中国将在全球执法中发挥越来越大的作用。北京则对与会的1000名各国代表说，中共〝遵守国际规则〞，中国是世界上最安全的国家之一。
 <br/>
 <br/>
 滑稽的是，一年后的10月8日（昨天），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 公安部在凌晨召开的党委会议上，通报了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 〝收受贿赂、涉嫌违法〞。而就在前一天，中共监察委官网发布消息称，孟宏伟正在接受调查。大约两个小时后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 在推特发布消息，称收到了孟宏伟的辞呈、立即生效。
 <br/>
 <br/>
 国际刑警组织掌门人突然在中国接受调查，对国际刑警组织来说，算是一桩丑闻。但对中共来说，本末倒置的荒唐处理则是很愚蠢的作法。《苹果日报》评论员李平撰文指出，中共现在是四面树敌，丑闻迭爆，已经蠢到没有了朋友，用中国古人的话讲就是〝众叛亲离〞。
