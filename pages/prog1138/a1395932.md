### 【禁闻】两名军老虎被开除党籍 震摄谁
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】10月16号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共官方.html" target="_blank">
  中共官方
 </a>
 公布了对中共军委前参谋长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房峰辉.html" target="_blank">
  房峰辉
 </a>
 ，和前军委政治部主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张阳.html" target="_blank">
  张阳
 </a>
 的处理通报。两人都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被开除党.html" target="_blank">
  被开除党
 </a>
 籍，房峰辉被移交司法起诉，而此前已经自杀的张阳也被追缴涉案财物。
 <br/>
 <br/>
 中共党媒《新华社》10月16号报导，中央军委决定开除联合参谋部原参谋长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房峰辉.html" target="_blank">
  房峰辉
 </a>
 ，和政治工作部原主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张阳.html" target="_blank">
  张阳
 </a>
 两人的党籍。两人此前已被取消上将军衔。
 <br/>
 <br/>
 《新华社》说，两人〝严重违反党的政治纪律和政治规矩〞，涉嫌腐败，〝情节极为严重，影响极其恶劣〞。
 <br/>
 <br/>
 在两人的罪名中，有三项是一致的，如〝搞两面派做两面人〞、〝不忠诚不老实〞和〝政治上蜕变、经济上贪婪〞等。
 <br/>
 <br/>
 旅美时事评论员夏小强：〝习近平当局在现在这个时候开除房峰辉和张阳的党籍，在官方媒体中的措词十分的严厉，这主要是房峰辉和张阳他们都是江派集团主要的军中人马。房峰辉是前军委副主席郭伯雄的老乡、旧部。张阳是徐才厚的嫡系。〞
 <br/>
 <br/>
 房峰辉和张阳都在去年8月突然从公众视野中消失，他们的岗位悄悄换上别人。去年11月23号张阳被发现在北京的家中自缢身亡。中共军网随即证实，当时中央军委正在调查核实他卷入郭伯雄、徐才厚等案的线索。并抨击他〝用自杀手段逃避惩处，行径极其恶劣〞。
 <br/>
 <br/>
 张阳死后，当局对房峰辉的调查仍在持续。直到今年1月，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共官方.html" target="_blank">
  中共官方
 </a>
 才正式宣布房峰辉因涉嫌行贿、受贿受到调查。党媒《央视》说，房峰辉和张阳是老搭档，两人都受郭伯雄、徐才厚流毒侵蚀。
 <br/>
 <br/>
 这次官媒公开对两人的处分，尤其引发关注的是，已经自杀的张阳仍被追缴涉案财物。
 <br/>
 <br/>
 夏小强：〝张阳已经自杀身死，但是却被追缴财物，这种做法在此前十分的少见。这显示出中共当局对他其实是等于赶尽杀绝。其实也是向张阳和房峰辉身后的江派集团发出的一个警告和震慑信号。〞
 <br/>
 <br/>
 旅美时事评论员郑浩昌认为，这可能跟中共四中全会准备开锣有关系。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝这个跟十九大前的做法很类似，习近平是拿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军老虎.html" target="_blank">
  军老虎
 </a>
 在会前祭旗。这个说明什么呢？习近平现在不仅有贸易战这个外患，还有内忧，他担心四中全会可能有对头就当前贸易战的这个情况对他发难，所以事先采用了震慑的措施。〞
 <br/>
 <br/>
 按中共惯例，历届中央委员会在任期内都会召开七次全体会议。其中，四中全会一般在10月召开，根据当前相对重要的问题而确定主题。
 <br/>
 <br/>
 目前中美贸易战日益升温。《韩联社》10月15号报导说，如果四中全会举行的话，应该会从大方向上决定中国的经济政策，很有可能讨论应对中美贸易摩擦的计画。
 <br/>
 <br/>
 旅美时事评论员夏小强表示，由于习近平的反腐打虎并没有拿下〝腐败总教练〞江泽民，以及江派二号人物曾庆红，因此留下隐患。
 <br/>
 <br/>
 夏小强：〝这就使得江派势力一直在发动反扑。特别是2018年中美贸易战之后，对中共控制的经济造成了巨大的冲击，中共政权遭到了巨大的危机。在这个背景下，中共内部以江派为首的反习势力正在借这个机会发动反击。〞
 <br/>
 <br/>
 夏小强分析，从整体上来讲，中美贸易战只是世界围剿中共政权的一个导火索，未来中共政权将在全方位受到以美国为首的国际力量的打击。这种态势将会加剧中共高层的内斗和分化，也给中共当局发出了明确的警示信号：在中共体制内，已经无法解决中国社会的危机和困境，中共将呈现加速解体的趋势。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/周天
