### 孟宏伟被撤销全国政协委员资格
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】中共官媒周五报导，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部.html" target="_blank">
  公安部
 </a>
 前副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，被撤销全国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政协委员.html" target="_blank">
  政协委员
 </a>
 资格。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 之前是中共第十三届全国政协外事委员会委员、国际刑警组织主席。
 <br/>
 <br/>
 10月5日，孟宏伟的妻子在法国报案，称孟宏伟回国后失踪，引起国际关注。
 <br/>
 <br/>
 10月7日，中共官方宣布孟宏伟因〝涉嫌违法〞，正在接受调查。
 <br/>
 <br/>
 孟宏伟长期在中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部.html" target="_blank">
  公安部
 </a>
 任职，2004年4月起，任中共公安部副部长；2016年11月当选国际刑警组织主席时，曾引发国际人权组织的批评和谴责。
 <br/>
 <br/>
 1999年以来，公安部成为中共江泽民集团利用来迫害法轮功的工具，孟宏伟作为公安部副部长，被《明慧网》列为迫害法轮功学员的涉案责任人之一，也被《追查迫害法轮功国际组织》追查。
</div>
