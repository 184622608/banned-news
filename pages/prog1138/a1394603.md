### 中共调查孟宏伟 国际刑警任命代理主席
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】上月底失踪的前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警.html" target="_blank">
  国际刑警
 </a>
 组织主席孟宏伟，已确认在中国被逮捕。周一，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部.html" target="_blank">
  公安部
 </a>
 通告，公安部副部长孟宏伟因收受贿赂、涉嫌违法问题严重而接受调查。而孟宏伟也已于周日辞去国际刑警组织主席一职。
 <br/>
 <br/>
 孟宏伟曾是周永康部下，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部.html" target="_blank">
  公安部
 </a>
 宣称〝要彻底肃清周永康流毒影响〞。孟宏伟9月29日从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警.html" target="_blank">
  国际刑警
 </a>
 组织的里昂总部返回中国之后便再无任何消息，她的妻子于上周向法国警方报案，外界才获知此事。
 <br/>
 <br/>
 目前，国际刑警组织任命其执行委员会高级副主席、韩国人金正阳为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代理主席.html" target="_blank">
  代理主席
 </a>
 ，并将于11月选出新主席。
</div>
