### 李鹏之女曝曾遇高层压力 靠打坐滤清思路
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】中共前总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李鹏.html" target="_blank">
  李鹏
 </a>
 之女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李小琳.html" target="_blank">
  李小琳
 </a>
 ，近日以陶行知国际教育基金管理委员会主席的新头衔现身。因身份特殊，她的一举一动都备受关注。她曾自称，遭遇高层压力时，〝把自己关在屋子里，打坐〞滤清思路，找出解决方案。
 <br/>
 <br/>
 陆媒11月11日报导，由陶行知教育基金会、行知丝路研究院等单位举办的〝行知丝路研究院成立、陶行知国际教育基金成立暨国学智慧与大健康善行峰会〞在河北廊坊举行。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李小琳.html" target="_blank">
  李小琳
 </a>
 首次以〝陶行知国际教育基金管理委员会主席、行知丝路研究院校长〞的身份亮相。
 <br/>
 <br/>
 公开资料显示，陶行知国际教育基金的主要业务范围为国际商学培训、交流合作办学等。
 <br/>
 <br/>
 因身份的特殊性，李小琳的一举一动都备受外界关注。
 <br/>
 <br/>
 今年5月，中共官方称批准了李小琳退休申请，未到龄退休的李小琳由此离任大唐集团。
 <br/>
 <br/>
 今年57岁的李小琳曾长期在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国电力.html" target="_blank">
  中国电力
 </a>
 系统任职，2003年，担任新成立的中国电力投资集团的副总经理，同时担任该公司旗下的中电国际的总经理。2008年1月，李小琳升任中国电力董事长。
 <br/>
 <br/>
 2013年的中共两会上，李小琳讲述自己的人生历程，自己也是经过努力工作实践，大学毕业之后，从工程师开始做起，又是从技术员，一步一个脚印地努力走过来的。当然也有很委屈的时候，也有压力非常大的时候。
 <br/>
 <br/>
 她讲述在中国电力时〝最困难的时候〞：上级领导不理解、煤价飙升、行业经营环境恶劣。她自称遇到压力时，处理方式是〝把自己关在屋子里，打坐〞。〝要先把自己‘静’下来，不想思绪，不想工作，静下来后，滤清思路，列出问题，找出解决方案，再和相关人员谈，知晓他们的建议，做到成竹在胸。对于这个‘秘诀’，她并不是一开始就会这样处理，是‘工作中逼出来的’〞。
 <br/>
 <br/>
 2014年李小琳在出席中电国际年中工作会议时强调，要〝懂大势、不出事、知历史〞。在2015年博鳌论坛上，她被问及对央企反腐的看法时称，〝我们大力支持反腐！〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李鹏.html" target="_blank">
  李鹏
 </a>
 家族过去陷贪腐丑闻，对于李小琳平安退休，当时有分析说，这可能与李鹏家族力挺习近平反腐〝打虎〞运动，被习放过有关。
 <br/>
 <br/>
 李鹏的儿子李小鹏，于2008年6月由商入仕，出任塌方式腐败的山西省省长。他多次发言力挺习近平的反腐〝打虎〞，并在山西省积极配合习近平，拿下了多名令计划、周永康的心腹。
 <br/>
 <br/>
 消息称，自习近平上台后，李鹏家族在不同时间和场合发声支持习近平阵营，而李鹏则要求其子李小鹏退出政坛，并叮嘱女儿李小琳尽量低调再低调。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
</div>
