### 央视〝后宫〞丑闻  曝周永康偏爱女主播玩过〝奖励〞下属
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】前中共政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 的心腹
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李东生.html" target="_blank">
  李东生
 </a>
 落马后，有关央视成中南海〝后宫〞的丑闻曝光天下。而周更因玩弄女性成癖有〝百鸡王〞之称。据披露，周玩过包括包括央视女主播、女星等约400人，玩过后〝奖励〞给下属，后者心知肚明却欣然接受。
 <br/>
 <br/>
 <h4>
  〝百鸡王〞
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
   周永康
  </a>
  早年就玩弄、强奸妇女
 </h4>
 <br/>
 早在2012年重庆事件爆发后，有关周永康的贪腐淫乱、杀妻谋权、活摘贩卖尸体等罪恶，就不断曝光天下，尤其是其淫乱丑闻，更是被大量曝光。
 <br/>
 <br/>
 周永康在任中共四川省委书记期间，亦多次传出强奸妇女，还包括宾馆服务员。但他受到石油王康世恩的袒护，因此一直安然无事。
 <br/>
 <br/>
 2012年3月，中共前重庆市委书记薄熙来落马后，媒体披露，周永康长期接受薄熙来提供的女人，其中28名已经确认，包括歌手、女演员以及大学女生。
 <br/>
 <br/>
 周永康仅在北京就有6处〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行宫.html" target="_blank">
  行宫
 </a>
 〞可以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_淫乐.html" target="_blank">
  淫乐
 </a>
 。同时，周永康还被传与薄熙来、中央军委副主席徐才厚等高官共享情妇汤灿。
 <br/>
 <br/>
 2013年底，前〝6．10〞办公室主任兼中共公安部党委副书记、副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李东生.html" target="_blank">
  李东生
 </a>
 落马后，有关周永康的性丑闻再度被大量曝光。
 <br/>
 <br/>
 <h4>
  李东生为周永康
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉皮条.html" target="_blank">
   拉皮条
  </a>
 </h4>
 <br/>
 据港媒披露，李东生为周永康拉皮条，是周永康与现任妻子贾晓烨相识的媒人。
 <br/>
 <br/>
 消息称，李东生在央视担任副台长期间，不但自己玩弄央视女记者、女主播，也不断开发央视美女资源，向中央领导输送。曹建明在担任最高检察长期间，娶了央视最当红主播之一王小丫。
 <br/>
 <br/>
 而周永康也娶了王小丫的央视同事贾晓烨。这两人都是李东生向中央领导
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性贿赂.html" target="_blank">
  性贿赂
 </a>
 的一部分。
 <br/>
 <br/>
 周永康与比其小28岁的贾晓烨结婚导致周跟其两个儿子差点反目成仇。贾晓烨生于1970年，而周的长子周滨（斌）据称是1972年出生的，只比贾小两岁。
 <br/>
 <br/>
 据海外中文媒体报导，央视的美女记者或美女主持人，总是利用央视提供的难得机会，去巴结中共高层。
 <br/>
 <br/>
 每逢获得采访机会，这些美女记者和美女主持人都会想方设法跟这些中共领导人套上关系，很多情况下都是〝一拍即合〞，这就是为什么中央电视台变成中央领导人〝后宫〞的原因。
 <br/>
 <br/>
 当时贾在央视经济频道当临时记者，抓住一个机会，被央视安排采访周永康，两人一拍即合。而坊间的另一说法是，到了央视经济频道后，贾晓烨经常与摄影师一起采访周而双方熟识，其后贾晓烨采访周时就不再带摄影师同行。
 <br/>
 <br/>
 据说，央视仅几名与贾关系不错的大腕女主播知道周贾两人的亲密关系，但都秘而不宣。
 <br/>
 <br/>
 李东生是中南海的三大皮条客之一。知情人士透露，三大皮条客常把女星、女主持人、美女记者等输送给中央高层。其中，李东生则利用权力网罗美女，淫乱程度外界难以想像。
 <br/>
 <br/>
 <h4>
  美女睡过赠下属
 </h4>
 <br/>
 2014年港媒报导，早在周永康认识贾晓烨之前，就跟央视一名知名美女主持人发生过关系，这种关系持续了一段时间后，周就把该美女主持人〝送〞给了一位跟自己很熟的老部下，后来这名央视美女主持就嫁给了周的老部下。
 <br/>
 <br/>
 报导引述一知情者说法称，这名美女主持人的现任丈夫，就是政法委系统一个重要高官。这名高官因跟周永康关系密切，至今，这名跟周有过不正当关系的央视美女主持人仍经常出现在各种重要会议上。而这段野史在央视早就不是秘密了。
 <br/>
 <br/>
 知情人称：〝这是周永康的一贯的生活作风，只要他看上眼的美女，他就会想方设法把她弄到手，睡了一段时间后就再换一个美女。〞
 <br/>
 <br/>
 除了把自己睡过的美女送给老部下，周永康还把数名被他玩过的央视美女和明星们〝奖励〞给亲信们，这些亲信有的是政界要员，有的是商界名人。
 <br/>
 <br/>
 而这些娶了周永康玩弄过的女人的亲信，早就心知肚明，但为了保持与周的密切关系，都心甘情愿地戴着周永康送给他们的这顶〝绿帽子〞。
 <br/>
 <br/>
 <h4>
  周永康睡过400美女
 </h4>
 <br/>
 香港《文汇报》记者也曾披露，周至少睡过400位女性，其中不乏一线女星、女演员及女大学生等各行业美女！
 <br/>
 <br/>
 周永康还有个癖好，就是喜欢与同僚或下属交换女人，除了与薄熙来共用情妇，还与薄熙来的妻子谷开来有一腿。为拉拢下属，周永康还会将其玩过的女性，转赠下属。
 <br/>
 <br/>
 在周永康被软禁的几天前，周还独自驱车至北京百盛商场地下停车场，与央视国际频道女主播叶迎春玩车震，全程遭调查人员跟拍，周竟全然不知晓。
 <br/>
 <br/>
 有网路媒体2014年曾刊登一则署名〝央视前主播〞的匿名投书，要求媒体报导央视女主播与官员的〝绯闻〞时手下留情，称其实〝那根本不是绯闻〞，而是央视女主播长期就是党政高官洩欲与发洩的工具，一开始是中宣部系统，后来扩大到政法、广电和组织部门的高官。
 <br/>
 <br/>
 投书中控诉，〝这已经不能算是‘潜规则’了，而成了‘明确的规定’，谁想上位，先上领导的床，然后还得不停上大领导的床。〞她说，除了央视如此，地方电视台也好不到哪里，省委省政府的高级官员，几乎都以拥有或者淫辱过一两次电视台的女主播为荣。
 <br/>
 <br/>
 投书文末提到，她自己早已离开央视，因为她的夫婿是宣传部门的领导，他知道内幕，因此逼她离职。
 <br/>
