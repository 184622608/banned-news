### 巨额受贿 中共财政部前副部长张少春被诉
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】中共官媒周五通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共财政部.html" target="_blank">
  中共财政部
 </a>
 前党组副书记、副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张少春.html" target="_blank">
  张少春
 </a>
 ，因受贿，二期数额特别巨大，被起诉。
 <br/>
 <br/>
 通报指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张少春.html" target="_blank">
  张少春
 </a>
 利用职务上的便利，为他人谋取利益，非法收受他人财物，数额特别巨大，应当以〝受贿罪〞追究其刑事责任。
 <br/>
 <br/>
 据报，张少春在国内外拥有130多套房产、148名情妇，被查出的资产总值为600多亿元人民币，境外存款300多亿。其儿子、媳妇、侄子等名下有70多幢房产。
 <br/>
 <br/>
 分析认为，张少春的落马对财政部，乃至整个财政系统触动很大。张少春在财政部任职长达30多年，任副部长时，分管税政司、预算司和国防司。
 <br/>
 <br/>
 此外，张少春2007年起还担任孔子学院理事会副主席，对孔子学院予以财政支持。而中共在海外开办孔子学院和孔子课堂，被认为是对国际社会进行中共意识形态渗透和配合中共搞统战。
</div>
