### 孟宏伟被抓水多深？美媒：习江斗又一个回合
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】国际刑警组织原主席、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被抓后，公安部紧急宣布成立工作组，查处孟的同党。分析认为，孟案背后的水很深，北京不顾顔面的抓人，表明中共决策层两股力量正在较劲。有美媒分析，孟案是习江斗的又一个回合。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 妻子葛瑞丝‧孟（Grace Meng）10月初向法国警方报案其丈夫失踪，并于7日召开记者会，向世界公布其丈夫回到中国境内与家人失联。立即引发全世界关注。
 <br/>
 <br/>
 中共官方7日深夜通报：孟宏伟涉嫌违法，目前正接受中共监委监察调查。
 <br/>
 <br/>
 中共公安部官网8日即于头条位置刊文称，公安部部长赵克志,当日凌晨紧急召开公安部党委会议，就孟宏伟被调查一事进行通报，指控孟受贿,并强调〝彻底肃清
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 流毒影响〞，同时宣布成立工作组，查处孟的同党。
 <br/>
 <br/>
 周永康是中共前党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 利益集团的核心人物，其落马前掌控的〝政法系〞，被称为周的〝独立王国〞。孟宏伟曾任中共公安部副部长长达14年，期间担任部长的是周永康。
 <br/>
 <br/>
 纽约时报引述署名评论分析，周永康是江泽民的门生，而江被认为是习近平的劲敌。孟宏伟被抓后不久，习近平的一名旧部已经接替了孟的职务，所以，这件事可以被简单地看作中国共产党最高层正在进行的、习近平与江泽民之间权力斗争的又一个回合。
 <br/>
 <br/>
 评论认为，考虑到中美关系的恶化，北京看到中共官员未来在世界主要组织担任领导人的机会不多了，拿下孟宏伟不会有很高的国际代价，同时还可能会给反对者一个下马威。
 <br/>
 <br/>
 香港苹果日报此前分析说，在遭到美国为首的世界性政治、经济、军事围剿之下，北京仍不顾一切进行党内大清洗，表明中共高层当前的矛盾已激化到了不可想像地步。
 <br/>
 <br/>
 自由亚洲引述北京匿名知情人士的话说，北京不顾国际社会压力，对国际刑警组织主席下手，表明中共决策层两股力量正在较劲。
 <br/>
 <br/>
 中共公安部过去20年来，一直被江派人马把持，历任部长分别为江派的要员贾春旺、周永康、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟建柱.html" target="_blank">
  孟建柱
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭声琨.html" target="_blank">
  郭声琨
 </a>
 。
 <br/>
 <br/>
 江泽民派系不仅通过中共政法系统打压民众迫害人权人士，而且将〝刀把子〞用于中共内部派系斗争。
 <br/>
 <br/>
 周永康落马后，习近平一直未能彻底清理中共政法系统。目前中共政法委〝一把手〞郭声琨，以及委员周强、傅政华、孟建柱、孟宏伟等人，都是周永康的亲信，此前赵克志宣布查处孟的同党。接下来，整个政法系统或将大震荡，背后很可能涉及中南海两个派系之间的生死博弈。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：赵云）
