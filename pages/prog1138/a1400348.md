### 中共常委倾巢出动 四中全会年内泡汤？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】中美贸易战下中国经济持续下行。近期中共政治局常委和委员四处奔走，忙于〝经济
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_调研.html" target="_blank">
  调研
 </a>
 〞。尤其几名常委最近更是倾巢出动，频繁外访或国内调研。加之年底适逢中共改革开放40周年，外界分析，年底举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 可能要泡汤了。
 <br/>
 <br/>
 台湾中央社20日称，最近一段时间以来，中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 、国务院总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 、国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 ，及多名其他中共政治局常委，频繁在中国各地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_调研.html" target="_blank">
  调研
 </a>
 。
 <br/>
 <br/>
 其中习近平11月15至21日访问太平洋三国，出席亚太经合组织（APEC）峰会。9、10月分别到东北及广东调研，针对中美贸易冲突，两度提出中国必须〝自力更生〞。
 <br/>
 <br/>
 李克强11月12日访问新加坡。14日出席第21次东盟领导人峰会。10月11日至19日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出访.html" target="_blank">
  出访
 </a>
 欧亚三国。6月时到湖南，主题是产业向中西部转移等议题，7月前往西藏考察。
 <br/>
 <br/>
 中共全国政协主席汪洋8月赴西藏调研，提出促进扶贫工作；中共中央政治局常委、国务院副总理韩正5月时也前往广东，11月在北京调研。
 <br/>
 <br/>
 被称为中共〝第8常委〞的王岐山，9月前往江苏、上海，要求推动科技创新和对外合作，10月22日王岐山开启了中东四国之行，11月5日至7日访问新加坡，出席了首届中美创新经济论坛；11月中旬则到湖北，提出推进经济社会全面可持续发展等议题。
 <br/>
 <br/>
 中共常委们出访及国内考察多是集中经济议题，结合接下来中共即将召开十九届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 ，外界普遍预期，本届四中全会将处理中共当前的经济政策。
 <br/>
 <br/>
 翻查过去半年的中共官媒公开报导，中共政治局已分别于6、7、9、10月下旬举行会议。中央社分析认为，中共委员会举行全体会议之前，文件起草至少需要半年。政治局常委在这段时间频繁外出调研，且召开多次政治局常委会议及政治局会议，再加上年底举行改革开放40周年活动，因此四中全会今年举行机会不大。
 <br/>
 <br/>
 按照中共此前惯例，中共每届中央全会召开的当年，在7、8月间公布大致开幕时间，然后在正式召开前夕的政治局会议上，再次敲定准确日期和细节。今年因加开了一次确定修宪的二中全会，外界一度猜测十九届四中全会将沿用上届三中全会的旧例，最迟在10月召开。
 <br/>
 <br/>
 因上述惯例也被打破，分析认为，本届全会11月下旬仍未能召开的话，可能要循例于明年9月召开。
 <br/>
 <br/>
 自中共改革开放以来，十九届四中全会是开会时间最晚的一次，除了1989年开会时间定于6月之外，其他几届大都是在9月中下旬召开，只有2012年前相对定在10月下旬召开。
 <br/>
 <br/>
 至于四中全会一再推迟的原因，有报导称，北京政界有两个传言，一是因为有政变风险，传武警有一批人策划〝起事〞；另外一个传言是，中美贸易战使得中国经济急剧恶化，习近平希望在四中全会上，能拿出一个有说服力的计划。
 <br/>
 <br/>
 但面对积重难返的国内经济问题，及中美贸易战的压力，特别是在中共体制的束缚之下，北京很难再推出有效的应对之策。因中共权力斗争陷入胶着状态，当前北京高层无力掌控局势，因此四中全会一推再推。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
</div>
