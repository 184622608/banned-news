### 广州司法局副局长王文生被调查
------------------------

<div class="wysiwyg">
 【新唐人2018年09月05日讯】中共官媒9月3日消息，广州市司法局党委委员、副局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王文生.html" target="_blank">
  王文生
 </a>
 ，因涉嫌〝严重违纪违法〞，目前正接受调查。
 <br/>
 <br/>
 通报没有提及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王文生.html" target="_blank">
  王文生
 </a>
 被查的具体原因。
 <br/>
 <br/>
 王文生长期在广州任职。2006年11月起，王文生先后任广州市白云区副区长、区政法委副书记、区委常委、政法委书记；2015年1月起，任广州市司法局党委委员、副局长。
 <br/>
 <br/>
 另据报导，王文生任广州白云区政法委书记期间，因积极追随中共江泽民集团迫害法轮功，被〝追查迫害法轮功国际组织〞列入追查名单。
</div>
