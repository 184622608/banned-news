### 赖小民权色交易细节曝光 女性求职附艳照
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融.html" target="_blank">
  华融
 </a>
 资产管理股份有限公司前党委书记、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_董事长.html" target="_blank">
  董事长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逮捕.html" target="_blank">
  逮捕
 </a>
 后，陆媒披露有关他搞权色交易的更多细节，包括求职于华融公司的女性，甚至大牌女明星，必须把写真照附在简历中，递交给赖小民。
 <br/>
 <br/>
 11月9日，《经济观察报》引述消息人士透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 涉案超过16亿元，这还未包括他实际拥有的房产及其他贵重资产。
 <br/>
 <br/>
 消息人士还表示，赖小民自今年4月被调查以来，其交待和调查人员〝挖掘梳理〞出了数以百计的〝关系人〞，〝以赖小民案为导火索，金融系统的反腐现在才是开端。仅仅他交待的这些涉案事情和涉案人，以现有的力量，没有三年时间，难以办完。〞
 <br/>
 <br/>
 10月15日，中纪委宣布对赖小民双开，通报中提及赖小民搞政治投机，为个人职务升迁拉关系，搞个人宣传，捞取政治资本，与多名女性搞权色交易等罪名。
 <br/>
 <br/>
 接近赖小民案调查的消息人士称，与赖小民进行〝权色交易〞的女性，既包括打算求职于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融.html" target="_blank">
  华融
 </a>
 公司的普通女性，亦包括有相当知名度、在奥斯卡获奖电影中演出重要角色的〝大牌女明星〞。
 <br/>
 <br/>
 而〝女性求职者会将自己大尺度的写真照附在简历中，递交给赖小民，彼此心照不宣〞。
 <br/>
 <br/>
 之前财新网披露的赖小民案细节更令人咋舌，〝100多套房、100多个关系人、100多位情人，其中几十位来自华融内部，此外甚至也不乏家喻户晓的女明星。〞
 <br/>
 <br/>
 现年56岁的赖小民，4月被中央纪委国家监委立案调查。10月中纪委发出通告称，决定开除其党籍公职。天津市检察院第二分院11月7日对赖小民作出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逮捕.html" target="_blank">
  逮捕
 </a>
 决定。
 <br/>
 <br/>
 当时，中纪委在通报用词空前严厉，称赖小民〝擅权妄为、腐化堕落、道德败坏、生活奢靡，甘于被‘围猎’，严重违反党的纪律〞。 还用了三个〝特别〞，即〝群众反映特彆强烈、腐败问题特别严重、性质特别恶劣。〞
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：赵云）
