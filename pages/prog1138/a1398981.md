### 胡习联手两大动作  埋徐才厚郭伯雄落马伏笔
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】中共军队总后勤部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷俊山.html" target="_blank">
  谷俊山
 </a>
 落马后，被曝〝供出了几乎所有人〞，包括前中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军委副主席.html" target="_blank">
  军委副主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐才厚.html" target="_blank">
  徐才厚
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 。而谷当时被查是由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 拍板。在18大前，胡锦涛与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 已完成军中布局，并罕见提前公布军委副主席名单，这两大动作被指提前埋下了徐才厚、郭伯雄落马的伏笔。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐才厚.html" target="_blank">
  徐才厚
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 同为江泽民一手提拔，被称为是江泽民的〝军中最爱〞。据称，2004年江泽民延期留任军委主席两年届满，不得不让出主席位子时，军委扩大会上江家军发难。
 <br/>
 <br/>
 徐才厚以所谓〝军机大事必要的连续性〞为由，要求军委内常设江泽民办公室，并发明创造了〝军委首长〞这一军内〝太上皇〞的称呼。
 <br/>
 <br/>
 于是江泽民在八一建军楼的办公室，一直运行到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 祼退前的2012年夏、中共18大召开前夕。
 <br/>
 <br/>
 江泽民卸任后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军委副主席.html" target="_blank">
  军委副主席
 </a>
 郭伯雄、徐才厚，替江把持军务实际是〝监军〞。胡锦涛很长一段时间都没有军中实权，徐才厚等人动不动就要〝军委首长〞批示。
 <br/>
 <br/>
 2008年汶川地震时，时任国务院总理的温家宝气得几次摔电话，也无力调动直升机和救援部队及时赶到灾区救人。因为江一直不下令让军队参与地震救灾。
 <br/>
 <br/>
 2012年2月，前重庆市副市长王立军，逃亡美领馆事件发生后，胡锦涛和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 开始了一系列联手反击江派的动作。先后采取对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷俊山.html" target="_blank">
  谷俊山
 </a>
 进行调查和收拢军权两大动作，为摆平徐才厚、郭伯雄扫清了路。
 <br/>
 <br/>
 <h4>
  胡锦涛拍板查谷俊山
 </h4>
 <br/>
 中共军队总后勤部副部长谷俊山案曝光于2012年初，直到2014年3月31日谷才被提起公诉，历时两年多。原因是谷俊山案一度受到中共军方高层的抵制，最后落马是由胡锦涛命令中共纪委介入调查，谷俊山被停职，调查才有所进展。
 <br/>
 <br/>
 据称，当时〝军委首长〞江泽民、副主席徐才厚、郭伯雄，还有国防部长梁光烈，都是谷俊山的后台。习近平上台后，该案调查范围扩大。
 <br/>
 <br/>
 港媒消息称，中共总后高层第一次向时任中央军委主席胡锦涛汇报情况，讲了两个多小时，向胡锦涛建议把谷俊山调离总后，胡不同意，认为这样的人，调到什么地方都是祸害，胡下决心惩处谷俊山，将其〝绳之以法〞。
 <br/>
 <br/>
 <h4>
  胡习军队布局 提前公布军委副主席名单
 </h4>
 <br/>
 2011年末的军委扩大会议上，总后勤部政委刘源，在胡锦涛与习近平的授意下，直接〝逼宫〞梁光烈和前中央军委副主席徐才厚、郭伯雄。
 <br/>
 <br/>
 刘源直指三人在位多年，〝对于军中严重腐败，更有不可推卸的责任！〞
 <br/>
 <br/>
 2012年中共18大之前，习近平的亲信张又侠任总装备部部长；习当年的军中老友赵克石成为总后勤部部长。
 <br/>
 <br/>
 同年11月1日至4日，中共17届七中全会决定，增补两名中央军事委员会副主席，而原来的军委副主席习近平、郭伯雄和徐才厚并没有退下，军内形成罕见的5名副主席并存的局面。此次增补的两名军委副主席内定将会接徐才厚和郭伯雄的班，名单罕见在18大之前就已经公布。
 <br/>
 <br/>
 2013年，习近平又让亲信钟绍军出任中央军委办公厅副主任，授予大校军衔，习因此较快的掌控军队。同年中共两会前夕，习近平下令彻查以谷俊山案为代表的军队腐败案。
 <br/>
 <br/>
 据报，谷俊山在调查期间〝供出了几乎所有人〞，特别是前中共军委副主席徐才厚。
 <br/>
 <br/>
 2014年3月31日，谷俊山被提起公诉。同年3月15日，徐才厚被调查，2015年3月患癌死亡；郭伯雄于2015年落马，2016年7月被判终身监禁。
 <br/>
 <br/>
 港媒披露，郭伯雄于2015年落马后，在监狱服刑期间，不但自曝接受徐才厚转交贿款的详情，还咬出江泽民，把军队变质问题归咎于江泽民对军队的领导。
 <br/>
 <br/>
 徐才厚和郭伯雄两人不但替江把持军权10多年，架空了胡锦涛。两人还曾阻击习近平进军委。在一个有数人在场的军方半公开场合，徐才厚曾对郭伯雄说：〝让他（指习近平）干5年就滚蛋！〞
 <br/>
 <br/>
 而徐才厚统管总后勤部和总政治部期间，各地军队医院发生大量活摘法轮功学员器官的行为，这种骇人听闻的活摘器官黑幕，至今仍被中共掩盖。
 <br/>
 <br/>
