### 改革开放40年展 王沪宁抬习避谈邓小平
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】中共庆祝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 40周年大型展览13日在北京举行开幕式，中共政治局常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王沪宁.html" target="_blank">
  王沪宁
 </a>
 致辞时说，十八大以来，以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核心.html" target="_blank">
  核心
 </a>
 的中央，为改革开放翻开了新的历史篇章。但只字未提推动改革开放的中共前领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 。
 <br/>
 <br/>
 11月13日，党媒报导说，〝庆祝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 40周年〞的大型展览，当天在北京天安门广场旁的中国国家博物馆举行开幕式，中共中央政治局常委、中央书记处书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王沪宁.html" target="_blank">
  王沪宁
 </a>
 出席并致辞。
 <br/>
 <br/>
 王沪宁在讲话中说，自中共十八大以来，以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核心.html" target="_blank">
  核心
 </a>
 的中央推进全面深化改革，为中共的改革开放翻开了新的篇章。
 <br/>
 <br/>
 王沪宁还说，举办大型展览，就是要展示40年来，中国人民在中共的领导下，开辟的所谓中国特色社会主义道路，进入新时代，要坚持以习近平思想为指导，增强〝四个意识〞，坚定〝四个自信〞，把改革开放不断推向深入。
 <br/>
 <br/>
 展览中展示了40年来，特别是2012年中共18大以来，中国人民的生产生活发生的变迁。
 <br/>
 <br/>
 但根据中共党媒报导内容，王沪宁只字未提推动中共40年来改革开放的前领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 。
 <br/>
 <br/>
 习近平10月下旬，南下视察被称为改革前沿的广东期间，整个行程中，同样只字未提到邓小平；而人民日报10月30日，针对改革开放发表评论文章，也罕见的只字未提邓小平。
 <br/>
 <br/>
 而在习近平〝南巡〞期间，邓小平长子邓朴方，盛赞邓理论的一篇演讲稿，在中国网路上热传。
 <br/>
 <br/>
 邓朴方在演讲中，除了热捧邓小平〝改革开放的伟大革命〞外，还宣称〝我们要坚持长期处于社会主义初级阶段这个基本判断〞，并不点名地提醒，〝我们一定要有这种实事求是的态度，保持清醒的头脑，知道自己的份量〞。
 <br/>
 <br/>
 有传媒分析说，在外界质疑习近平是否还继续坚持邓小平提出的改革开放，以及是否重新评价邓的改革开放〝功绩〞之际，邓朴方的讲话，显示中共内部对坚持〝改革开放〞立场的分歧。
 <br/>
 <br/>
 对于中共所谓的改革开放，观察人士普遍认为，在共产党领导下，任何改革都徒有虚名，不过是政治和权力运作的需要，不属于真正意义的改革。中共体制决定了任何改革都不可能成功。中共改革40年后的中国现状已经证实了这一点。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明)
 <br/>
