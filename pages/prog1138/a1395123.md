### 湖北黄冈前公安局长汪治怀被〝双开〞
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】中共官媒周四通报，湖北省黄冈市前副市长、兼公安局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汪治怀.html" target="_blank">
  汪治怀
 </a>
 被〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双开.html" target="_blank">
  双开
 </a>
 〞。通报称，汪治怀为涉及黄赌毒的黑恶势力充当〝保护伞〞，是湖北第一案。
 <br/>
 <br/>
 通报称，2012年3月起，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汪治怀.html" target="_blank">
  汪治怀
 </a>
 任黄冈市政府副市长、市委政法委副书记、市公安局党委书记、局长等职。
 <br/>
 <br/>
 汪治怀在任职期间，为涉毒、涉赌、涉黄等犯罪嫌疑人提供保护；还利用职务便利，为他人谋取不正当利益并收受财物，数额巨大；串供堵口，对抗审查；并与一名女性发生并长期保持不正当性关系。
 <br/>
 <br/>
 据《明慧网》报导，黄冈是湖北省迫害法轮功最严重的地区之一 。汪治怀任职黄冈期间，积极追随江泽民迫害法轮功，对当地多名法轮功学员遭绑架等迫害负有责任。
</div>
