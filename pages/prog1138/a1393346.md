### 受贿7千万 证监会前副主席姚刚判18年
------------------------

<div class="wysiwyg">
 【新唐人2018年09月28日讯】中共官媒周五通报，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_证监会前副主席.html" target="_blank">
  证监会前副主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚刚.html" target="_blank">
  姚刚
 </a>
 因受贿罪、内幕交易罪，被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_判刑18年.html" target="_blank">
  判刑18年
 </a>
 ，并处罚金人民币1100万元。据报，姚刚曾参与做空大陆股市，并涉北大方正高管腐败案以及令计划案。
 <br/>
 <br/>
 法庭称，2006年到2015年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚刚.html" target="_blank">
  姚刚
 </a>
 担任中共证监会主席助理、副主席等职务期间，为相关单位在并购重组、股份转让过程中股票停复牌、避免被行政处罚等事项上提供帮助，收贿人民币6961万余元。
 <br/>
 <br/>
 2015年11月13日，姚刚被调查，是证监会落马的首个副部级官员，也被称为股民最痛恨的部级〝老虎〞。
 <br/>
 <br/>
 姚刚在证监会任职期间，分管发行监管部，掌控证监会内最有权势的中国A股市场IPO发审大权，长达13年。
 <br/>
 <br/>
 2015年大陆股灾，北京当局动用数千亿人民币〝救市〞。期间，姚刚被指涉嫌与境外合谋，做空大陆股市。
 <br/>
 <br/>
 据报，这次股灾是针对习近平当局的〝经济政变〞。
 <br/>
 <br/>
 此外，姚刚被曝是〝西山会〞成员，涉及北大方正高管腐败案和令计划案。
</div>
