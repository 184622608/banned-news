### 孟宏伟与郭声琨关系密切 刀子图或不仅是示警
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】北京突然拿下公安部副部长兼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警主席.html" target="_blank">
  国际刑警主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，多被认为是清理〝周永康余毒〞。但港媒指，孟和中共政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭声琨.html" target="_blank">
  郭声琨
 </a>
 也是校友。公安部声明显示孟被抓前和北京当局有过博弈，有分析指其失联前发回的尖刀图或是行动暗号。
 <br/>
 <br/>
 香港《明报》10月9日文章指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 2004年升任公安部副部长，时任公安部长是前政治局常委周永康。但是，孟宏伟2013年成为海警局首任局长，跻身正部级，当时的公安部长则是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭声琨.html" target="_blank">
  郭声琨
 </a>
 。
 <br/>
 <br/>
 文章指，孟宏伟参加工作后，在湖南的中南大学拿到工科硕士学位，与郭声琨成了校友，郭也是1990年代在该校取得管理学硕士学位。
 <br/>
 <br/>
 文章分析，中共官方通报暗示，孟疑似涉及周永康等政法系统旧案，是否会牵连他近年的上司孟建柱、郭声琨，仍需观察。
 <br/>
 <br/>
 郭声琨被指是江派大佬曾庆红的亲戚，和周永康等前政法高层，均属江家人马。有时评人士分析指，周永康已落马多年，孟宏伟令北京不顾国际脸面对他下手，或另有后台指使其与最高当局对抗。
 <br/>
 <br/>
 孟宏伟落马后，公安部召开会议宣称，孟被查是因为其〝一意孤行、咎由自取〞，当局拿下孟〝非常及时〞，并称〝绝不允许跟组织讲条件、谈‘价钱’〞。这似乎暗示孟宏伟落马前，已经在和北京当局长期对抗和博弈。
 <br/>
 <br/>
 自由亚洲电台10月8日引述匿名人士分析，通过孟妻7日在记者会上的发言，基本可以判断北京已与孟妻取得联系，并抛出谈判筹码。但孟宏伟久居官场，在回国被抓应该有所警觉，可能早就制定了保命预案，或已将一些重料交予妻子保管。
 <br/>
 <br/>
 海外社交媒体也有人发帖引述匿名〝资深国安〞分析，9月25日孟失联前给其妻手机发送的尖刀图案，可能不只是示警，而是代表着事先约定的一种行动方案，可能意指〝短兵相接〞或〝鱼死网破〞。
 <br/>
 <br/>
 根据公开资料，孟宏伟在去年12月就被免去中共国家海洋局副局长、中共海警局局长职务，今年4月其公安部党委委员的职务也被拿掉，只保留了公安部副部长的兼职。这显示其前途早已不妙。
 <br/>
 <br/>
 10月4日，孟宏伟妻子在国际刑警组织总部所在地——法国里昂报警，称孟回国后失踪。10月7日，孟妻在当地召开记者会指，孟宏伟在9月25日给她发来短信〝等我电〞，4分钟后又发回一把尖刀的照片，随后失联。
 <br/>
 <br/>
 在孟妻发言不到一小时，中纪委发布了孟宏伟落马的通报。
 <br/>
 <br/>
 《南德意志报》10月8日文章称，孟宏伟此次被控，指向他曾经保护北京早前肃清的政治力量周永康。
 <br/>
 <br/>
 法国巴黎政治学院教授白夏向自由亚洲表示，北京两年前向国际社会不断游说和施压，将孟宏伟安置到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警主席.html" target="_blank">
  国际刑警主席
 </a>
 的位置上，此番不顾国际影响抓捕他，想来不会是贪腐那么简单。但北京亦会因此付出国际政治成本。
 <br/>
 <br/>
 旅居法国的评论人王龙蒙表示，政法系统当年对法轮功、维权人士的迫害，孟宏伟都有份参与，害人的人最终也被人害，说明这个体制下没有一个人是安全的。孟宏伟事件引发国际关注，其积极意义在于让国际上越来越看清中共政权是秩序规则的破坏者。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
</div>
