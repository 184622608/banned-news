### 习近平曾为一事怒拍桌子 现场官员吓破胆
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】早年湖南省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_衡阳.html" target="_blank">
  衡阳
 </a>
 市发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贿选案.html" target="_blank">
  贿选案
 </a>
 ，党媒曾刊文提到，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 曾对此案
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拍桌子.html" target="_blank">
  拍桌子
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发火.html" target="_blank">
  发火
 </a>
 ，连发六个〝到哪儿去了？〞令当时涉案官员几乎吓破了胆。
 <br/>
 <br/>
 党媒在〝领导人拍案时刻〞一文曾提到，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_衡阳.html" target="_blank">
  衡阳
 </a>
 破坏选举案曾令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拍桌子.html" target="_blank">
  拍桌子
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发火.html" target="_blank">
  发火
 </a>
 ，并连发六问。
 <br/>
 <br/>
 衡阳
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贿选案.html" target="_blank">
  贿选案
 </a>
 于2013年12月29日立案调查时，官方通告，涉案金额达1.1亿元，涉及56名省人大代表、518名市人大代表、68名代表大会工作人员。
 <br/>
 <br/>
 媒体称，这是1949年以来涉案金额最大、涉及党政官员和人大代表最多的一起选举弊案。
 <br/>
 <br/>
 据衡阳一位官场人士回忆，2013年1月2日，即湖南省人大代表选举当天，他在衡阳一家茶楼饮茶，邻座一位老板模样的人在大声打电话，〝谁谁谁给4000，谁谁谁给5000……〞
 <br/>
 <br/>
 当时，习近平对该案三次公开点名，拍桌子，六连问。
 <br/>
 <br/>
 关于这三次点名，据陆媒报导，2014年中纪委三次全会上，2016年中纪委六次全会上，2016年两会期间。
 <br/>
 <br/>
 关于拍桌子，〝领导人拍案时刻〞一文提到，习近平也有拍桌子发火的时候，衡阳破坏选举案就曾令他拍桌子。
 <br/>
 <br/>
 关于六连问，媒体报导，在2014年中纪委会议上，习近平连发六个〝到哪儿去了？〞一，市委和市政府到哪儿去了？二，当地的纪委？三，当地人大领导班子？四，领导干部到哪儿去了？五，国法观念到哪儿去了？六，良知到哪儿去了？等等。
 <br/>
 <br/>
 湖南官方政务信息显示，2012年4月18日上午，湖南省县乡两级人大换届选举工作学习班在长沙开班。时任省委书记的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周强.html" target="_blank">
  周强
 </a>
 在学习班上强调，〝依法组织投票选举，确保县乡两级人大换届选举圆满成功，迎接十八大胜利召开〞。
 <br/>
 <br/>
 但是在习李十八大上台后不久，2013年1月起，湖南邵阳、衡阳两地连续曝出贿选丑闻。
 <br/>
 <br/>
 媒体调查报导曾指，贿选在湖南官场从来就不是新闻，就以习近平三次点名、拍桌子、六连问的衡阳贿选案来说，是〝老百姓都知道的事情〞，只是时任湖南一把手周强〝到哪儿去了？〞
 <br/>
 <br/>
 周强于2006年出任湖南省代省长。2007年2月其正式任省长。2010年4月，周强接任中共湖南省委书记。同年9月，兼任湖南省人大常委会主任。2013年3月，其任最高法院院长。
 <br/>
 <br/>
 周强在湖南任职期间，发生了多件引起争议和轰动的大事件，包括李旺阳离奇死亡案、被逼卖淫幼女母亲唐慧喊冤被劳教的唐慧案、湘西非法集资系列案及主犯曾成杰被枪决案等。
 <br/>
 <br/>
 中共〝十八大〞前，周强曾是进入政治局的热门人选。但最终落选，有说法称，周强未能在〝十八大〞进入政治局，与令计划、周永康有关。
 <br/>
 <br/>
 此外，周强在任共青团中央第一书记和最高法院院长期间，都曾因参与污衊和迫害法轮功被海外〝追查迫害法轮功国际组织〞列入追查名单。
 <br/>
 <br/>
 （记者李蕓报导/责任编辑：赵云）
