### 百位情妇百套房 赖小民被称〝3个100〞贪官
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融.html" target="_blank">
  华融
 </a>
 资产管理股份有限公司前董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 15日被立案审查。陆媒披露，赖小民拥有〝100多套房、100多个关系人、100多位情人〞，情人中不乏家喻户晓的女明星。此外，赖小民家藏3吨现金，仅母亲存款就有3亿人民币。
 <br/>
 <br/>
 财新网日前报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 案有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_3个100.html" target="_blank">
  3个100
 </a>
 〞：100多套房、100多个关系人、100多位情人。其中几十位情人来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融.html" target="_blank">
  华融
 </a>
 内部，也不乏家喻户晓的女明星。
 <br/>
 <br/>
 陆媒盘点赖小民的朋友圈，特别提到了台湾女星吴佩慈的男友纪晓波，以及大陆女星赵薇的丈夫黄有龙。
 <br/>
 <br/>
 纪晓波是天行国际的控制人。2015年3月，华融系旗下华融金控正式借壳天行国际，之后天行国际连年亏损，令人匪夷所思。财新报导，纪晓波和华融还有其他不少合作。
 <br/>
 <br/>
 纪晓波在娱乐圈的绯闻女友众多，比如刘颖、杨恭如、林心如、熊黛林、黄奕等。
 <br/>
 <br/>
 去年4月，黄有龙把旗下顺龙控股的股份都抵押给了华融。当时黄有龙赵薇的空壳收购案已经被立案调查。赖小民冒着风险支持黄，被指关系非同一般。
 <br/>
 <br/>
 此外，从2015年到2017年，华融至少给王永红的中弘股份发放51亿贷款和借款。而王永红猎艳于演艺圈早已不是新闻。
 <br/>
 <br/>
 今年4月，赖小民落马。陆媒报导，办案人员在赖小民的几处房产里，搜出的外币现金以人民币计算，共计2.7亿元，相当于约3吨重，放在一起超过3个立方米。
 <br/>
 <br/>
 此外，赖小民母亲账上被指有存款3亿多人民币，都是赖小民的〝朋友〞和下属们〝孝敬〞的。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
