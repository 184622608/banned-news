### 李克强变身推销员：来中国多点投资
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】近日，中共国务院总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 在新加坡发表演讲时，三度用英语对本地企业说：〝我只有一个请求，来中国多点投资〞。网友因此再送他一个称号——最棒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推销.html" target="_blank">
  推销
 </a>
 员。
 <br/>
 <br/>
 据陆媒报导，11月13日上午，正在新加坡进行访问的中共总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 ，在〝新加坡讲座〞上发表了一场主旨演讲，演讲中，李克强〝连续三次用英文招商〞。
 <br/>
 <br/>
 据报，李克强每次出访都会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推销.html" target="_blank">
  推销
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国货.html" target="_blank">
  中国货
 </a>
 ，而此次访问新加坡期间，他没有公开推销中国货，而是三度用英语对本地企业说：〝我只有一个请求，More investments!（来中国多点投资！）〞
 <br/>
 <br/>
 他还承诺：〝如果你们在中国的投资，遇到营商环境方面的不公正的待遇，你们可以向我们今天在座的部长们提出来，甚至直接向我投诉……We are waiting for your more investment in China！（我们等着你们在中国更多投资）〞。
 <br/>
 <br/>
 不少网友调侃，李克强是〝最棒的推销员〞。不过，由于中共治下不可能有公正可言，对于相关承诺，不少人仍持质疑态度。
 <br/>
 <br/>
 事实上，李克强早被国人誉为〝首席推销员〞、〝中国最强推销员〞、〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超级推销员.html" target="_blank">
  超级推销员
 </a>
 〞。每到一地李克强都要推销〝中国货〞。
 <br/>
 <br/>
 早在2013年12月，党媒就曾报导说，李克强卖高铁，变身为〝超级推销员〞。报导还说，李克强就任中共国务院总理之后，已经多次扮演〝超级推销员〞的角色。
 <br/>
 <br/>
 2015年11月，党媒也曾以题为〝中国高铁推销员李克强〞报导说，11月25日，李克强与中东欧16国领导人，共同登上从苏州开往上海的高铁列车。而李克强走到哪里，〝超级推销〞的旋风就刮到哪里，他向外方推荐中国高铁几乎已成了其出访的必做功课。
 <br/>
 <br/>
 事实上，不管是中共的高铁，还是投资环境，都不被国际认可，中共总理亲自出来搞推销似乎作为不大，而中共多年以来，一直搞金钱外交，不顾及国民疾苦大肆向第三国撒钱，一直被国际舆论所诟病。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
