### 孟宏伟辞职信曝光：两行铅字无签名
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】前中共公安部副部长兼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 9月回国被抓后，国际刑警随后宣布接受孟宏伟辞职。日前外媒揭露，国际刑警接到的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职信.html" target="_blank">
  辞职信
 </a>
 只是打印的两行文字，没有孟的签名，但国际刑警迫于中共压力，依然选择接受。
 <br/>
 <br/>
 法国世界报11月9日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 10月7日接到中方转交的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职信.html" target="_blank">
  辞职信
 </a>
 ，只有打印出来的两行字：〝由于涉嫌违法，我自愿辞去国际刑警组织的主席职务。〞
 <br/>
 <br/>
 信的下方虽然注有孟宏伟的名字，但没有他的亲笔签名。
 <br/>
 <br/>
 报导说，无法证实这封辞职信是出自孟宏伟的手，确定他是否自愿辞职就更难。不过，国际刑警组织还是宣布接受孟宏伟的辞职。
 <br/>
 <br/>
 报导还披露了该报获得的一些内部电子邮件，其内容显示，北京在扣押孟宏伟后，又将对国际刑警组织的经费支持当作筹码，施压该组织选择了〝不让北京生气〞的立场。
 <br/>
 <br/>
 其中，中共公安部在10月8日发给国际刑警组织秘书长的电子邮件中强调，在孟宏伟事件后，北京将继续履行〝支持国际刑警组织〞的承诺，并邀请秘书长访问中国。
 <br/>
 <br/>
 报导指，自2010年以来，中共给国际刑警组织的经费几乎翻了一倍。孟宏伟事件暴露了国际刑警组织的财务独立性问题。
 <br/>
 <br/>
 国际刑警组织秘书长8日刚刚宣布，该组织〝无权〞追查孟宏伟的下落，〝必须接受〞孟宏伟辞职。
 <br/>
 <br/>
 近年以来，中共一直在极力渗透操控国际组织，其称霸世界的野心越来越明显。除了国际刑警组织，联合国很多部门都与中共关系〝密切〞。
 <br/>
 <br/>
 日前，联合国人权理事会对中共进行了第三轮国别审议，多国代表聚焦新疆〝再教育营〞、西藏人权以及香港新闻自由等问题。
 <br/>
 <br/>
 但审议前夕，联合国官方公布的〝团体摘要报告（修订稿）〞删去了香港众志、西藏人权民主中心、无代表国家和人民组织、维吾尔人权项目、南蒙古人权信息中心和世界维吾尔代表大会等组织的报告摘要。外界认为，这是中共施压的结果。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：云涛）
</div>
