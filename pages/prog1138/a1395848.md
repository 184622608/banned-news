### 谁提拔了孟宏伟？分析：中南海有只看不见的手
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被调查后，公安部连夜召开会议通报事件，再次强调〝彻底肃清
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 流毒影响〞。评论分析认为，这说明当初中南海那只把孟推上来的看不见的手还〝活着〞，如果〝刀把子〞的隐患已除，也用不着反覆强调要清除周永康〝余毒〞了。
 <br/>
 <br/>
 截止10月18日，国际刑警组织前主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 落马已11天，但有关孟的话题热度未减。香港《南华早报》分析说，孟宏伟被拿下之后，基本上是习近平5年前掌权之后，逐步整顿拥有200万成员的公安系统的最后一步棋。
 <br/>
 <br/>
 报导引述分析说，公安部的部长以及副部长们的组成和排名，反映了中共领导人传统上，对这支负责政权安全的单位的政治重要性。孟出事前是公安部老资格副部长。
 <br/>
 <br/>
 中共公安部过去20年来，一直被江派人马把持，历任部长分别为江派的要员贾春旺、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 、孟建柱、郭声琨。孟建柱、郭声琨则被指是江派二号人物曾庆红的嫡系。
 <br/>
 <br/>
 习近平上台后，对中共公安、政法系统进行了大清洗。包括中共前公安部部长、前政法委书记周永康及其亲信李东生、张越、杨焕宁、夏崇源等被查。
 <br/>
 <br/>
 周在任时的那批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 ，有些加入其他党或政府部门，有些则被提早退休或降职，也有人被查。
 <br/>
 <br/>
 但中共前公安部部长郭声琨，则一路干到任满方才退下岗位，并且晋身中共政治局委员，出掌中共政法委。
 <br/>
 <br/>
 孟宏伟是公安部2013年那个班底的，只是少数一直留在公安部的副部长，孟还负责整顿和扩充中国的海岸防卫队，同时调升至副部长，2016年年底，孟更成为了中共用重金买下的国际职务——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 。
 <br/>
 <br/>
 但在今年9月底，孟宏伟从法国返回中国后〝失踪〞，10月7日深夜，中纪委官网通报，孟宏伟〝收受贿赂、涉嫌违法〞正接受调查。
 <br/>
 <br/>
 孟落马后，其公安部副部长一职，10月9日已由习近平旧部邓卫平接任。
 <br/>
 <br/>
 至此，公安部12名党委委员，除两名业务专家外，其余全部被换掉。包括公安部部长赵克志，常务副部长王小洪，副部长邓卫平、许甘露，孟庆丰等，都是习近平的亲信。
 <br/>
 <br/>
 这是否意味着中南海对公安部的清洗结束了呢？分析认为，习整肃公安部远未到最后一步。
 <br/>
 <br/>
 阿波罗网特约评论人王笃然分析，十九大前，习近平刚刚把军权抓到手，江系长期掌控包括公安部在内的政法系统，习还未来得及大规模清除政法系统的江系势力，抓捕孟宏伟还远谈不上是最后一步。
 <br/>
 <br/>
 时评人崔士方也持同样看法。他分析说，周永康把持公安系统足足10年，冰冻三尺非一日之寒，无法指望一夜间就消融。更关键的是，当初那只把孟宏伟推上来的看不见的手还〝活着〞呢。如果〝刀把子〞的隐患已除，就不用公安部长赵克志反覆讲要清除周永康〝余毒〞。
 <br/>
 <br/>
 目前公安部高官共11人，除了部长赵克志，常务副部长王小洪，副部长邓卫平、许甘露，孟庆丰是习近平的人，还有6人都不是习的人，如副部长孙力军就是孟建柱一手提拔的。
 <br/>
 <br/>
 目前，孟宏伟被抓事件余波未了，赵克志在公安部通报孟宏伟事件时称，〝坚决彻底肃清周永康流毒影响。〞并宣布成立工作组，查处与孟宏伟共同收受贿赂的官员。
 <br/>
 <br/>
 外界预计，随着孟宏伟的落马，中共公安部以至于整个政法系，将再次掀起一阵大清洗高潮，直到习阵营全面接管公安部高层为止。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
