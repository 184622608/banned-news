### 港媒：陈良宇狱中反目 曾因一事大骂江泽民
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】据港媒此前揭露，前上海市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈良宇.html" target="_blank">
  陈良宇
 </a>
 原是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 的〝死党〞，但陈入狱后与江反目。他在狱中惊悉儿子被抓后，狂怒大骂江泽民，警告说：〝如果我儿子保不住命，你两个儿子一个也别想活！〞
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈良宇.html" target="_blank">
   陈良宇
  </a>
  握
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江绵恒.html" target="_blank">
   江绵恒
  </a>
  罪证
 </h4>
 <br/>
 2007年9月，陈良宇案发后，陈的儿子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈维力.html" target="_blank">
  陈维力
 </a>
 在其父下属的帮助下潜逃出境。2008年3月，陈良宇入狱服刑。
 <br/>
 <br/>
 据港媒披露，同年7月8日，中共警方利用陈维力女友为饵，诱捕了由美赴马来西亚的陈维力，并引渡回国，按当时官媒所列罪名，陈维力足以被判处死刑。
 <br/>
 <br/>
 陈良宇在狱中惊悉儿子被抓后，狂怒大骂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 ，并警告说：〝如果我儿子保不住命，你两个儿子一个也别想活！〞
 <br/>
 <br/>
 报导称，陈维力被抓后，北京当局以最快的速度对其审判。2008年8月3日，陈维力案一审判决为：陈维力因在其父受贿案中扮演从犯角色，判处有期徒刑三年，缓刑四年。
 <br/>
 <br/>
 外界有传言指，陈维力被轻判是陈良宇在狱中的话起了作用，因为其手中握有江泽民儿子江绵恒，足以被枪毙十回的贪污犯罪证据。
 <br/>
 <br/>
 坊间称江泽民家族是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共第一贪.html" target="_blank">
  中共第一贪
 </a>
 〞，江绵恒凭藉江泽民的权势，在上海〝闷声大发财〞。1994年，江绵恒用数百万人民币〝贷款〞买下上海市经委价值上亿元的上海联合投资公司，开始他的〝电信王国〞生涯。到2001年，上联和上联控股的公司已有十余家。
 <br/>
 <br/>
 上海商界人士称，江绵恒的董事头衔多得数不清，上海若干重要经济领域他都染指。甚至上海过江隧道、上海地铁的董事会，他也有份。有位商人坐上海航空公司的班机，无意中发现空中杂志上刊登的上航董事会举行会议的照片，其中一人即是江绵恒，但上航正式股东名单则从未向社会公布过。
 <br/>
 <br/>
 上海商界人士私下说，江绵恒既是中国电信大王，也是上海滩的大哥大。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
   胡锦涛
  </a>
  拿下陈良宇
 </h4>
 <br/>
 在此之前，陈良宇与江泽民的关系密不可分。中共〝十六大〞后，江有意将陈作为隔代接班人选来培养，以接替胡锦涛。
 <br/>
 <br/>
 陈良宇因此仗着有江泽民作靠山，为人极其嚣张，经常公开叫板胡、温，不将两人放在眼里。当时，江泽民、黄菊与陈良宇被称为上海帮的〝铁三角〞。
 <br/>
 <br/>
 江一直处心积虑要置胡于死地，胡锦涛执政十年，曾三次险遭江泽民暗杀。2006年发生黄海暗杀事件后，愤怒不已的胡锦涛决定进行反击，拿下陈良宇。
 <br/>
 <br/>
 2006年9月24日，陈良宇乘专机到北京出席中共政治局全体会议时，一下飞机就突然被抓捕。
 <br/>
 <br/>
 在此前一天，中央政治局常委会开了一个长达9小时的会议，对陈良宇问题进行表决。当时的九名政治局常委表决结果：六票（吴邦国、曾庆红投向反对陈良宇的一方）赞成，三票（贾庆林、黄菊、李长春）弃权。
 <br/>
 <br/>
 同时，胡温高层将陈良宇的情况向在上海的江泽民作了汇报，江泽民见大势已去，无奈之下，只好选择抛弃陈良宇。
 <br/>
 <br/>
 2008年，陈良宇被指涉及上海社保基金巨额挪用案，及利用职务之便谋取不当利益，被判刑18年。
 <br/>
 <br/>
 陈良宇落马后，他的〝死党〞老上级江泽民，再次展示其暗黑一面，狠批陈良宇为〝害群之马，罪有应得〞。
 <br/>
 <br/>
 据中共官媒《内参》9月25日报导：上海市民得悉陈良宇下台，有人走上街头庆贺，喝洋酒叫〝爽〞。上海市委机关50条电话线的铃声此起彼伏，来电都纷纷表示支持、拥护中央的决定。
 <br/>
 <br/>
