### 河北公安厅原副厅长陈庆恩落马
------------------------

<div class="wysiwyg">
 【新唐人2018年09月10日讯】中共河北省纪委官网消息，河北省公安厅原党委委员、副厅长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈庆恩.html" target="_blank">
  陈庆恩
 </a>
 ，因〝涉嫌严重违纪违法〞，目前正接受调查。陈庆恩成为河北省委前政法委原书记张越之后，第二个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 的公安系统高官。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈庆恩.html" target="_blank">
  陈庆恩
 </a>
 在河北公安任职31年，是前河北省公安厅厅长、党委书记张越的〝旧部〞，两人共事两年多。张越今年7月被判刑15年。
 <br/>
 <br/>
 1982年1月起，陈庆恩任职河北邯郸市公安局，曾任副局长、刑警支队支队长，后转任廊坊市公安局长、党委副书记、书记等职；2006年9月起，任秦皇岛市公安局长、党委书记、政法委书记、市政协副主席；2011年8月，任河北省公安厅党委委员、副厅长，兼任秦皇岛市公安局党委书记、局长。
 <br/>
 <br/>
 据报，陈庆恩在廊坊、秦皇岛及省公安厅任职期间，积极追随江泽民集团迫害法轮功，是当地迫害法轮功学员的主要责任人，被〝追查迫害法轮功国际组织〞多次通告。
 <br/>
 <br/>
 《明慧网》也多次点名，陈庆恩对当地迫害法轮功学员负有责任。
</div>
