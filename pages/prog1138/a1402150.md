### 华为孟晚舟被捕时机微妙 与川习会同时上演
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的女儿、副董事长兼CFO孟晚舟，日前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 被捕，并拟引渡到美国受审。事件引发全球轰动。值得关注的是，孟晚舟被捕的时机非常微妙，与在阿根廷举行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞的时间点惊人巧合。这被认为是美国对贸易战〝缓刑〞90天的同时，对中共的制裁并未放松。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 《环球邮报》（the Globe and Mail）引述加国司法部的声明报导，加国已逮捕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 首席财务官孟晚舟，她因涉嫌违反美国对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 的贸易制裁，而将被引渡到美国。
 <br/>
 <br/>
 司法部声明说：〝孟晚舟于12月1日在温哥华被捕。美国要求将她引渡，拟于12月7日举行保释听证会。〞
 <br/>
 <br/>
 路透社此前报导说，纽约联邦检察官至少从2017年开始，就一直在调查华为，检察官就原产于美国的产品运往伊朗和其它国家、违反美国出口和制裁法令的指控进行调查。
 <br/>
 <br/>
 孟晚舟是华为创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 与第一任妻子孟军所生的长女，现任公司CFO、副董事长，主导在全球建立了五个共享中心，担任华为系旗下共11间公司的董事。
 <br/>
 <br/>
 华为1987年在深圳成立，有中共军方背景，2012年开始成为全球最大的电讯设备制造商。多年以来西方情报机构一直质疑华为设备的安全性，认为它受制于中共政府，存在间谍和安全隐患。
 <br/>
 <br/>
 美国此前已经采取了一系列行动，阻止或减少华为和中兴通讯公司进入美国市场，因为这两家公司被指控，利用其技术在美国进行间谍活动。
 <br/>
 <br/>
 值得注意的是，孟晚舟被捕当晚，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞正在阿根廷上演，中美元首就贸易问题展开磋商。双方同意休战90天，给北京3个月的缓冲谈判时间来解决贸易平衡、知识产权等问题。
 <br/>
 <br/>
 根据白宫在会晤后发布的声明，习近平承诺减少中美间的贸易不平衡，并将从美国采购大量农产品等产品。
 <br/>
 <br/>
 川普则同意2019年1月1日暂时对2,000亿美元中国进口商品的关税维持在10%。如果在90天内双方不能达成协议，对中国商品的关税提高到25%。
 <br/>
 <br/>
 这意味着虽然中美达成停火90天协议，但未来仍祸福难料，因此，在川习会之外，美国抓捕违反美国出口和制裁法令的中国科技巨头华为高管，以震慑中共，迫使北京做出改变，以解决两国产生分歧的根本性问题。
 <br/>
 <br/>
 而孟晚舟被捕一事，恐怕对正在进行全球〝攻城略地〞的华为来说也是一个〝晴天霹雳〞，今后华为的业务开展之路恐被堵死。而且美国政府正在努力劝说德国、意大利和日本等盟国官员及电信高管，围堵华为、弃用华为。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
</div>
