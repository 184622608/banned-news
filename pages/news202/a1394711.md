 华尔街日报
 <br/>
 在美国对中国产品加征关税后，一些中国出口商通过更换海关商品编码来逃避美国的关税。尽管关于更换编码的数据很少，但从美国海关对中国原产地可疑出口分类的裁定数量激增可以看出，更换编码的情况在增加。
 <br/>
 <br/>
 台湾中央社
 <br/>
 东汉科学家张衡发明可以检测地震的〝候风地动仪〞。1951年，大陆考古学家王振铎把它复制出来，但其内部结构缺乏合理性，龙口中的铜丸没法吐下来，据说唐山大地震时该模型也毫无动静。对这个复制版的质疑越来越多。因为无法成功复制。2017年秋天的大陆统编本初中历史教材中，删除了张衡发明地震仪的内容。
 <br/>
 <br/>
 联合新闻网
 <br/>
 今年8月辽宁省沈阳市爆发中国大陆首宗非洲猪瘟后，疫情至今仍未受控。官方通报，辽宁省营口市、鞍山市接连两天爆发疫情。目前中国已有8个省份确诊非洲猪瘟疫情。
