### 南太平洋新喀里多尼亚岛7.6强震 深度仅10公里
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】太平洋海啸警报中心（PTWC）指出，南太平洋
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新喀里多尼亚岛.html" target="_blank">
  新喀里多尼亚岛
 </a>
 东部外海今天发生规模7.6强震，当地已观测到海啸。当局呼吁民众前往高地避难。
 <br/>
 <br/>
 路透社报导，这起海底
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 深度仅10公里，震央在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新喀里多尼亚岛.html" target="_blank">
  新喀里多尼亚岛
 </a>
 （New Caledonia）东侧的罗雅提群岛（Loyalty Islands）东南东方155公里。
 <br/>
 <br/>
 太平洋海啸警报中心指出：〝新喀里多尼亚岛和万那杜（Vanuatu瓦努阿图）部分沿岸地区可能会出现比浪高1公尺到3公尺的海啸。〞
 <br/>
 <br/>
 当地官员表示：〝我们已发对新喀里多尼亚岛沿岸发部警报，目前尚未有任何伤亡。〞
 <br/>
 <br/>
 一名在当地工作的法商发言人表示，地震感觉十分强烈，但没有持续很久。
 <br/>
 <br/>
 新喀里多尼亚是法国的海外属地，位于南太平洋，由新喀里多尼亚岛、洛亚蒂群岛和无人居住的切斯特菲尔德群岛组成。
 <br/>
 <br/>
 (责任编辑：卢勇信)
