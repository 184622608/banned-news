### 【禁闻】中共渗透丹麦 警察再曝内幕
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】
 <b>
  上千人
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日内瓦游行.html" target="_blank">
   日内瓦游行
  </a>
  抗议中共侵犯
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权.html" target="_blank">
   人权
  </a>
 </b>
 <br/>
 <br/>
 在联合国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权.html" target="_blank">
  人权
 </a>
 理事会11月6号审议中国人权情况的当天，上千名来自世界各地的新疆、西藏、蒙古等被中共迫害的少数民族人士，在日内瓦举行游行集会，抗议中共暴行。
 <br/>
 <br/>
 他们手持写有〝停止酷刑〞、〝停止种族屠杀〞、〝释放所有政治犯〞等内容的横幅，沿路高喊〝西藏要自由〞、〝新疆要自由〞、〝蒙古要自由〞等口号，抗议中共侵犯人权、欺压他们的族人。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共渗透.html" target="_blank">
   中共渗透
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丹麦.html" target="_blank">
   丹麦
  </a>
  警察再曝内幕
 </b>
 <br/>
 <br/>
 继丹麦司法部判决哥本哈根警方违反《宪法》对抗议民众犯下严重错误，警方宣布赔偿8名当事法轮功学员之后，丹麦又有警察向公众发声，披露中共渗透丹麦的内幕。
 <br/>
 <br/>
 据丹麦广播电视台DR的P1频道8月份报导，他们新近得到一份操作指令文件，内容是关于哥本哈根警方如何对待2004年中共人大委员长吴邦国访问丹麦期间的抗议者的。
 <br/>
 <br/>
 几位当时参与工作的警察披露，根据指示，在中共官员访问期间，抗议者应被尽量遮挡、不被看到，因为来访者不喜欢抗议者。
 <br/>
 <br/>
 一名在行动中负责无线服务的警察表示，这更多的是可见度问题而不是安全问题。
 <br/>
 <br/>
 他说，他当时注意到，中共驻丹麦大使馆发出了法轮功学员将连续两天举行抗议活动的信息，之后，警方就被要求成立民事拘捕小组，还要求组员穿着制服，跟随抗议活动。
 <br/>
 <br/>
 报导说，这份最新曝光的警察指令文件，是中共插手丹麦民主社会内政的又一证据。
 <br/>
 <br/>
 <b>
  14家国际人权组织 吁中共释放
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
   黄琦
  </a>
 </b>
 <br/>
 <br/>
 国际特赦、记者无国界、自由之家等14家国际人权组织11月5号发表联合声明，要求中共立即释放〝六四天网〞创办人黄琦。
 <br/>
 <br/>
 声明警告说，黄琦在狱中健康严重恶化，如果不接受治疗，恐怕有生命危险。
 <br/>
 <br/>
 声明还说，黄琦被关押纯粹是因为和平行使自己的言论自由权，中共应立即无条件释放他，并〝同时终止拒绝及时治疗良心犯、形同虐待的政策。〞
 <br/>
 <br/>
 编辑/周玉林
