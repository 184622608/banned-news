### 印尼天灾不断 山洪暴发冲走12名学童酿22人身亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 近来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天灾.html" target="_blank">
  天灾
 </a>
 频传，日前苏拉威西岛强震海啸酿成2010人死亡5000多人失踪后，12日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北苏门答腊.html" target="_blank">
  北苏门答腊
 </a>
 省(Sumatra)
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曼代灵纳塔尔.html" target="_blank">
  曼代灵纳塔尔
 </a>
 地区又遭暴雨袭击，洪水至少冲走12名儿童，造成11死1失踪。另国家灾害应变总署证实这起山洪暴发共有22人死亡。
 <br/>
 <br/>
 据《路透》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曼代灵纳塔尔.html" target="_blank">
  曼代灵纳塔尔
 </a>
 地区（Mandailing Natal）警察局长表示，洪水于12日下午4时15分洪水袭击了木阿拉沙拉迪村(Muara Saladi)一所学校，当时有29名学生正在校内上课，除了被冲走的12人之外，已确认其余学生的安全。
 <br/>
 <br/>
 另当地有12栋民宅被洪水冲走，其中有9栋严重受损。
 <br/>
 <br/>
 随后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 国家灾害应变总署(BNPB)发言人苏托波(Sutopo Purwo Nugroho)表示，连日豪雨在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北苏门答腊.html" target="_blank">
  北苏门答腊
 </a>
 省(North Sumatra)证实最少造成17人死亡，在西苏门答腊省(West Sumatra)有5人死亡。
 <br/>
 <br/>
 苏托波表示，国家灾害应变总署已经前往灾区，但因多处地区发生土石流，阻挡他们前进。
 <br/>
 <br/>
 印尼位处热带地区，又多岛屿，常发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天灾.html" target="_blank">
  天灾
 </a>
 和豪大雨，洪水和土石流很常见。不过最近天灾不断，苏拉威西岛(Sulawesi)西海岸在9月28日遭规模7.5强震和海啸袭击，造成近2千人死亡，5千人下落不明。爪哇和峇里岛海岸在11日发生规模6.0的浅层地震，造成至少3人丧生。
 <br/>
 <br/>
 在今年夏天，峇里岛附近的龙目岛(Lombok)遭遇一系列夺命地震，总共造成555人死亡，将近1,500人受伤，还有数十万人无家可归。
 <br/>
