### 波兰地方选举 检测执政党民意支持
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波兰.html" target="_blank">
  波兰
 </a>
 星期日（10月21日）举行地区选举，选举结果对疑欧派的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_执政党.html" target="_blank">
  执政党
 </a>
 、〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法律与公正党.html" target="_blank">
  法律与公正党
 </a>
 〞（PiS，Law and Justice）的民意支持度将是一个检测。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民意调查.html" target="_blank">
  民意调查
 </a>
 显示，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法律与公正党.html" target="_blank">
  法律与公正党
 </a>
 〞在省议会的选举中可能赢得33-37%的选票。立场居中的反对党联盟（Civic Platform and Nowoczesna）预计将获得24%的支持率。选举后的各党结盟将影响
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_执政党.html" target="_blank">
  执政党
 </a>
 控制议会的数量。
 <br/>
 <br/>
 立场居中的华沙市长候选人特拉斯科沃斯基（Rafal Trzaskowski）看来周日会赢得41-42%的选票，并有望在11月4日的第二轮投票中击败来自执政党的对手。华沙是自由主义的堡垒，也是反对政治干预司法的抗议地点，赢得华沙对任何政党都是最大的奖赏。
 <br/>
 <br/>
 华沙居民安娜（ANNA SZYMANIAK）：〝目前的政治局势（选举）是对执政党所做的变化表达反对意见的好机会。〞
 <br/>
 <br/>
 目前虽然执政党在国家级的政治中占据主导地位，但只控制了少数地方的市政厅。执政党如能成功获得席位，将更好的获得地方竞选资金，对2019年的国会大选产生影响。
 <br/>
 <br/>
 新唐人记者徐哲、天羽综合报导
