### 美国拟制裁新疆书记   15国驻华大使联署
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】15个西方国家的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻华大使.html" target="_blank">
  驻华大使
 </a>
 ，日前罕见地联署致信新疆党委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈全国.html" target="_blank">
  陈全国
 </a>
 ，要求他讲清楚侵犯新疆维吾尔人权利的情况。同时美国也正在考虑对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈全国.html" target="_blank">
  陈全国
 </a>
 和其他涉嫌践踏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆人权.html" target="_blank">
  新疆人权
 </a>
 的官员和公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_实施.html" target="_blank">
  实施
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 。
 <br/>
 <br/>
 11月15日，路透社独家报导说，以加拿大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻华大使.html" target="_blank">
  驻华大使
 </a>
 为首的14个西方国家，以及欧洲联盟的驻华大使，罕见地联署致函新疆党委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈全国.html" target="_blank">
  陈全国
 </a>
 。
 <br/>
 <br/>
 信函表示，联合国有关新疆的调查报告中，揭示出的中国少数民族的遭遇,使他们极为担忧。为了更好地了解新疆的状况，他们请求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈全国.html" target="_blank">
  陈全国
 </a>
 与他们尽早会面，解释中共在新疆大规模拘押维吾尔人侵犯人权等问题。
 <br/>
 <br/>
 这封信函将被抄送到中共外交部、公安部和中共负责的国际事务部。一位外交消息人士表示，这封信正在各国传递，希望更多国家大使联名。
 <br/>
 <br/>
 目前已经在这封信函上签字的大使来自加拿大、英国、法国、瑞士、欧盟、德国、荷兰、澳大利亚、爱尔兰、瑞典、比利时、挪威、爱沙尼亚、芬兰和丹麦。
 <br/>
 <br/>
 报导说，这是多个国家的外交官针对中国人权问题的一个不同寻常的广泛协调举动，显示了中共在新疆的人权打压行为，正在引发来自国际社会越来越大的反弹。
 <br/>
 <br/>
 联合国人权小组8月表示，根据许多可信的报导，100万甚至更多的维吾尔人被关押在类似于〝秘密集中营〞的地方。
 <br/>
 <br/>
 领导这次行动的加拿大外交部在一个声明中表示，他们高度关注中共在新疆大规模拘押和广泛监视维吾尔人做法，加拿大外交部官员在联合国，也直接向中共官员提出了维吾尔人处境的问题。
 <br/>
 <br/>
 美国大使没有签名，但一位美国外交官表示，美国目前已积极致力于倡导维吾尔人，和其它新疆穆斯林少数民族的权益。
 <br/>
 <br/>
 美国驻华大使馆在一份声明中表示，美国自2017年4月，得知中共在多个再教育营里拘压80万以上、甚至可能近200万维吾尔人的信息以来，就一直对这个问题警觉。
 <br/>
 <br/>
 声明说，美国将继续呼吁中共结束类似的适得其反政策，并立即释放所有被任意拘押的人。美国政府，将致力于对那些犯有侵犯人权罪行的人进行问责，包括针对那些新疆官员进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 。
 <br/>
 <br/>
 11月14日，美国国会及行政当局中国委员会的一位共同主席，共和党籍参议员卢比奥 (Marco Rubio)和民主党籍参议员梅南德兹 (Robert Menendez)， 以及其他15位两党参议员，联署向参议院提交了类似的《维吾尔人权政策法案》。
 <br/>
 <br/>
 该法案促请美国的情报部门，列出在新疆建造和营运〝再教育基地〞的中国公司名单，促请美国政府正视新疆的再教育营，及中共侵害维族的人权问题。
 <br/>
 <br/>
 当天，美国国会及行政当局中国委员会(CECC)共同主席史密斯众议员，在美国众议院呈交了由多名两党议员，联署的一项有关新疆少数民族人权的议案。
 <br/>
 <br/>
 该议案谴责中共强行拘留约100万维吾尔人，呼吁中共立即停止对维吾尔人任意拘押、施予酷刑以及逼迫他们放弃宗教信仰的做法，并表示要对相关中共官员进行制裁。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
