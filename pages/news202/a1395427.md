### 英国侨界庆祝双十国庆餐会隆重举行
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】10月6日晚，台湾旅英侨学工商各界在伦敦市中心千禧酒店联合举办了庆祝107年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双十国庆.html" target="_blank">
  双十国庆
 </a>
 餐会。近300名旅英各界
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侨胞.html" target="_blank">
  侨胞
 </a>
 和嘉宾出席了活动。请看来自英国伦敦的报导。
 <br/>
 <br/>
 驻英代表处代表林永乐，副代表程云翔，侨务秘书张希贤，旅英各界
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侨胞.html" target="_blank">
  侨胞
 </a>
 和侨领代表，还有来自伦敦的地方市议员等众多嘉宾出席了当天的餐会。
 <br/>
 <br/>
 庆祝餐会活动由旅英声乐家黄德馨领唱国歌揭开了序幕。林永乐带领侨领们切开国庆蛋糕，共同祝愿中华民国国泰民安国运昌隆。今年的活动是历年来报名最踊跃，参加人数最多的一次。主办方也精心安排了文艺表演和幸运抽奖。
 <br/>
 <br/>
 林永乐感谢侨界的热情参与，他并谈到庆祝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双十国庆.html" target="_blank">
  双十国庆
 </a>
 的意义。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻英台北代表.html" target="_blank">
  驻英台北代表
 </a>
 处代表 林永乐：〝我觉得双十国庆对我们住在台湾的人来讲，我们能够很高兴，我们的国家是有自由，有民主，尊重人权，尊重法制，我们是一个开发自由的社会，我们觉得能够把这样的讯息让大家都知道。〞
 <br/>
 <br/>
 活动筹委会主任委员蓝国瑜在接受采访时提到，今年餐会的推广工作做的好，大家对台湾认识和支持也越来越多。
 <br/>
 <br/>
 餐会活动中有伦敦华韵合唱团带来《茉莉花》等传统国语歌谣，台语歌曲，另外还有长笛演奏，和知名小提琴家王怀悌也为现场宾客带来了精彩表演。
 <br/>
 <br/>
 整场晚会持续了三个多小时，活动在最后抽出由中华航空和长荣航空提供的伦敦到台北的往返机票的奖项中，气氛达到了高潮， 也令侨胞们感受到英国这10多年来最热闹温馨的国庆庆祝餐会。
 <br/>
 <br/>
 新唐人记者鲍天宇、霄龙英国伦敦报导
</div>
