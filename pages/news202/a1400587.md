### WTO改革分歧大 美倡透明化 中共要特殊待遇
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】如何改革
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界贸易组织.html" target="_blank">
  世界贸易组织
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 ，成为中美争议的焦点。目前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革方案.html" target="_blank">
  改革方案
 </a>
 上，美中仍有很大分歧。美国倡导透明化，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 则继续要求享受特殊待遇。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）（09/25）：〝一些国家违反了世贸组织所有原则，却被世贸组织接纳。中国扭曲市场的行为，以及他们做交易的方式是不能被容忍的。〞
 <br/>
 <br/>
 美国总统川普强力要求改革
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界贸易组织.html" target="_blank">
  世界贸易组织
 </a>
 ，并表示不改革就退出。
 <br/>
 <br/>
 11月12号，美日欧等国向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 提交了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革方案.html" target="_blank">
  改革方案
 </a>
 ，内容包括要求会员国增加透明度，并对违规国家进行处罚。
 <br/>
 <br/>
 举例来说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 长期对钢铁等产业实施过度补贴，按WTO现行制度，有义务进行汇报。但中共自2006年起，几乎没有向WTO提交过有关补贴的报告。而WTO目前没有任何相应的处罚制度。
 <br/>
 <br/>
 美日欧在改革方案中，要求剥夺违规国家的发言权，并采取其它限制措施作为惩罚。
 <br/>
 <br/>
 俞伟雄：〝中国加入WTO十几年来，WTO一直没有办法叫它做一个改善，这本身是它的一个败笔，它应该想办法怎么样来监督它的会员国，如果会员国不遵守这些规定的话，要采取制裁。〞
 <br/>
 <br/>
 同时，川普政府认为，WTO允许会员〝自行认定〞自己的开发程度，也是不合理规定。
 <br/>
 <br/>
 目前中共按照人均GDP等数据，自我认定为〝发展中国家〞，可以履行较低水平的义务，获取发达国家最大程度的市场开放，指定高于发达国家的关税等。
 <br/>
 <br/>
 23号，中共商务部公布了它的WTO改革主张，没有回应美国关切的透明度改革等问题，而是强调，不允许剥夺中方以〝发展中国家〞身分，继续享有特殊和差别待遇。
 <br/>
 <br/>
 唐靖远：〝有一个重要问题被忽略了，中国现在是‘国进民退’‘党富民穷’，中共通过高税负低福利，把财富从老百姓口袋转到党库里。在对外扩张争夺市场的时候，它讲‘国富’，用举国体制补贴一些战略产业，来跟市场经济的私企进行不公平竞争。但要履行国际义务时，它就说我‘民穷’。这对市场经济国家来说是非常不公平的。〞
 <br/>
 <br/>
 在本月底的G20峰会上，有可能讨论WTO改革问题。但目前美中改革方案仍有很大分歧。欧盟贸易专员日前表示，如果中方在WTO改革谈判中不让步，美国确实可能放弃WTO体系。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
