### 难阻大篷车难民 川普：边境紧急状态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】周日，来自洪都拉斯的数千
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 部分已经进入墨西哥
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境.html" target="_blank">
  边境
 </a>
 ，继续向美国边境前进。同时，约有2000难民认为成功进入美国希望渺茫，决定返回家园。周一早上，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 连发推文 ，批评洪都拉斯、危地马拉和墨西哥阻止难民不利，宣布南部边境进入国家紧急状态。
 <br/>
 <br/>
 周日，约有5000
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 ，浩浩荡荡走在墨西哥
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境.html" target="_blank">
  边境
 </a>
 城市街道上，继续向美国南部边境前进。
 <br/>
 <br/>
 很多难民为躲避洪都拉斯的黑帮暴力和贫穷，前往美国寻求庇护。
 <br/>
 <br/>
 洪都拉斯残疾难民：〝洪都拉斯政府不会帮助任何人。即使是残疾人，也不会得到任何帮助。看看这里的人都是没有工作的，所以人们都在离开祖国。〞
 <br/>
 <br/>
 墨西哥边境警方试图劝说难民前往难民庇护所，但很多难民怕被遣返，不敢上车。
 <br/>
 <br/>
 在边境，墨西哥警方曾动用催泪弹，驱散想要强行过境的难民，让他们办理难民申请手续。但很多难民不愿等待，涉水过河，穿越墨西哥边境。也有难民选择返回家园。
 <br/>
 <br/>
 周日，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 感谢墨西哥做出的努力。
 <br/>
 <br/>
 美国总统 川普：〝我代表美国人民非常感谢墨西哥所做的，他们正在加紧在做。〞
 <br/>
 〝他们（墨西哥边境警察）有些人受重伤，至少有六人，非常感谢墨西哥在边境上做出的努力，非常感谢。〞
 <br/>
 <br/>
 周一，川普推文说，〝危地马拉、洪都拉斯和萨尔瓦多没能有效阻止大篷车难民离开本国，非法前往美国。现在我们将切断或大幅减少给这些国家的巨大常规外援。〞
 <br/>
 <br/>
 川普还宣布，南部边境进入国家紧急状态。
 <br/>
 <br/>
 推文说：〝很不幸，看来墨西哥警察和军队阻挡不住大篷车向美国南部边境推进。罪犯和不知底细的中东人混在其中。我已警告边境巡逻队和美军，进入国家紧急状态。必须改变（移民）法！〞
 <br/>
 <br/>
 川普再次批评民主党开放边境的移民政策，期待与民主党参众两院少数党领袖舒默和佩洛西举行会谈，商讨解决大篷车难民问题。
 <br/>
 <br/>
 新唐人记者安心、宇微综合报导
</div>
