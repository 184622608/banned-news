### 不因卡舒吉变化 川普维护美国沙特关系
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特记者.html" target="_blank">
  沙特记者
 </a>
 卡舒吉在沙特领馆遇害，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统周三表态，不想因此事影响和沙特政府的关系，因为那会让其它国家乘虚而入。
 <br/>
 <br/>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特记者.html" target="_blank">
  沙特记者
 </a>
 卡舒吉在沙特领事馆遇害，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 否定中情局得出了明确结论，此前中情局有消息说，沙特王储与此事有关。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统 川普：〝美国中央情报局对此进行了研究，他们对此进行了很多研究。他们没有任何明确的结论。〞
 <br/>
 <br/>
 外界呼吁制裁沙特，但川普反对因此损害美国与沙特政府的关系，因为那会让别的国家乘虚而入。
 <br/>
 <br/>
 美国总统 川普：〝如果我说我们不要你们（沙特）的生意，如果我说我们要中断（合作），他们将得到来自俄罗斯和中共的军事设备和其它物资。〞
 <br/>
 <br/>
 川普指出，如果中断沙特与合作，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 也会上涨。
 <br/>
 <br/>
 美国总统 川普：〝我维持着油价低下来，他们（沙特）帮助了我维持油价低。〞
 <br/>
 <br/>
 川普周三发推文指出，油价从过去的每桶82美元降到现在的54美元，这要感谢沙特。他并希望油价更低。
 <br/>
 <br/>
 周二，国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 （Mike Pompeo）表示支持川普总统的立场，强调总统是在维护美国的安全利益和持续繁荣。并指出美国上周因此制裁了17名沙特个人，已做出了强力回应。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝美国将继续与沙特保持（良好）关系，他们是美国的重要伙伴。〞
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
