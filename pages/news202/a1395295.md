### 川普：布朗森牧师被释放没有交换条件
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】周五下午，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 就土耳其释放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布朗森.html" target="_blank">
  布朗森
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牧师.html" target="_blank">
  牧师
 </a>
 ，以及沙特记者失踪事件，对媒体发表看法。川普强调，布朗森牧师获释不存在任何交换条件。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周五离开白宫，赶往俄亥俄州集会前，短暂接受记者采访。
 <br/>
 <br/>
 有关土耳其释放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布朗森.html" target="_blank">
  布朗森
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牧师.html" target="_blank">
  牧师
 </a>
 ，川普强调，不存在任何交易。
 <br/>
 <br/>
 记者：〝你会解除对土耳其的制裁吗？〞
 <br/>
 <br/>
 美国总统川普：〝（布朗森被释放）和任何事都没有关系，不存在任何交易，没有任何交易。但我们很高兴他能回来，状态良好地回来。我听説他状态很好。〞
 <br/>
 <br/>
 周六，川普发推文，再次强调布朗森牧师获释是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孤立事件.html" target="_blank">
  孤立事件
 </a>
 。川普还表达了对土耳其的谢意，认为这有助于改善两国之间的关系。
 <br/>
 <br/>
 在另一则推文中，川普透露，布朗森牧师周六下午将造访白宫。
 <br/>
 <br/>
 推文说：〝安德鲁・布朗森牧师从土耳其获释，将于下午2点半在椭圆办公室和我会面。能看到他本人真是太好了。他是一位了不起的基督徒，经历了这次苦难。我感谢埃尔多安总统的帮助！〞
 <br/>
 <br/>
 在周五的采访中，记者问到沙特记者卡舒吉在土耳其失踪一事。
 <br/>
 <br/>
 川普表示，这是一起非常糟糕的事件，但目前还没有定论，他会就此与沙特国王通话。
 <br/>
 <br/>
 59岁的沙特著名记者卡舒吉，10月2号前往沙特驻土耳其领事馆后失踪。有土耳其报纸声称，当局的调查显示卡舒吉已经在领事馆内遇害，并曾遭受酷刑折磨。
 <br/>
 <br/>
 卡舒吉曾做过沙特王室的顾问，但经常批评王室，特别是沙特王储穆罕默德・本・萨勒曼。
 <br/>
 <br/>
 卡舒吉失踪事件曝光后，引发国际社会关注。
 <br/>
 <br/>
 新唐人记者林宇、田瑞综合报导
</div>
