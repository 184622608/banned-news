### 目睹儿子抢劫 墨大妈拖鞋怒打不肖子后报警
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】日前，在墨西哥城一辆公交车上发生一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_持枪抢劫.html" target="_blank">
  持枪抢劫
 </a>
 事件。劫匪的老母亲恰好也在公交车上，当她认出自己的儿子后，立即脱下拖鞋追打这个不肖子，制服儿子后还报了警。相关消息在社交网站上引起热议，人们纷纷留言称赞这位母亲大义灭亲。
 <br/>
 <br/>
 据西班牙《国家报》11月5日报导，当天，现年35岁的拉蒙．M试图在开往墨西哥城的一辆公交车上抢劫乘客的钱财。他上车后掏出手枪威胁乘客交出财物，但他没有注意到自己的母亲当时就坐在这两公交车的后排。
 <br/>
 <br/>
 正当全车的乘客都为遭遇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_持枪抢劫.html" target="_blank">
  持枪抢劫
 </a>
 的匪徒而惶恐之际，拉蒙的老母亲认出劫匪竟然是自己的儿子。于是愤怒的母亲当即脱下自己的一只拖鞋，当着所有乘客的面手持拖鞋开始追打那个不争气的儿子。
 <br/>
 <center>
  <br/>

   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-7/p9114181a917322342-ss.jpg"/>
   <br/>
   <font size="-1">
    墨西哥城一辆公交车上发生一起持枪抢劫事件，劫匪的老母亲恰好也在公交车上，当她认出自己的儿子后，立即脱下拖鞋追打这个不肖子并制服了他。（图片来源：La Nueva Radio YA）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 这位老母亲还招呼其他乘客一起制服了自己的儿子。随后，包括母亲在内的多名乘客都打电话报了警。
 <br/>
 <br/>
 警察随后赶往现场逮捕了劫匪拉蒙，而拉蒙的母亲和其他乘客也跟随警察去做了笔录。
 <br/>
 <br/>
 事后，这位老母亲在接受媒体采访时表示，自己的家庭是基督教家庭，自己时常教导儿子要追随上帝，没想到他会做出这种作奸犯科的行为，这令她感到十分痛心，也是她无法容许的。
 <br/>
 <br/>
 警方则对媒体表示，劫匪拉蒙是在犯罪现场被抓，并有多名目击者指认，他将被送往墨西哥北部的一座监狱关押，等待法庭的审判。
 <br/>
 <br/>
 据报导，墨西哥城的交通工具上发生抢劫案件的概率，在整个墨西哥全国的排名是前5位。
 <br/>
 <br/>
 事件发生时，有乘客用手机拍下了当时颇有戏剧性的场面，并把图片和现场情况在网络社交媒体上公开，引起众多网友的围观和点赞，网友们几乎一边倒地支持那位母亲大义灭亲的行为。
 <br/>
 <br/>
 （记者竺颖报导/责任编辑：东野）
