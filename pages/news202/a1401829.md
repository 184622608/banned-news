### 【禁闻】G20首次同意改革WTO 气候议题有分歧
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】二十国集团
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 12月1号发布联合声明。声明首度承认需要改革世界贸易组织
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 ，但删去了〝保护主义〞的措辞。至于气候变化议题，虽然各国同意继续推动环保，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 与其它19国存在明显分歧。
 <br/>
 <br/>
 为期两天的阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 结束，并发表联合声明。
 <br/>
 <br/>
 其中，改革
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 成为重点。
 <br/>
 <br/>
 联合声明指出，G20全体领导人支持多边贸易，但承认目前的全球贸易系统存在缺陷，需要改革WTO，但没有说明细节。
 <br/>
 <br/>
 这是G20首脑首次承认WTO无法达成它的设立目标，需要改革。
 <br/>
 <br/>
 旅美政经分析人士秦鹏：〝我认为这是一个好事情，这也反映了几乎所有国家的共识，那就是这些年WTO被中共给严重扭曲和破坏了，所以必须改革，否则
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 肯定会退出，或者另外组局，而其它很多发达国家也会跟进。〞
 <br/>
 <br/>
 旅美政经分析人士秦鹏表示，拟议中的改革，应该会特别完善对于争端处理的机制，这是过去WTO最薄弱的环节之一。
 <br/>
 <br/>
 白宫首席经济顾问哈塞特（Kevin Hassett）11月接受媒体专访时曾暗示，有可能把中共驱逐出WTO。
 <br/>
 <br/>
 美国的中国问题专家毛思迪（Steven W.Mosher），11月29号发表于《英文大纪元》的评论提到，针对中共的不公平贸易行为，美国正在和日本、欧盟起草WTO改革规则。
 <br/>
 <br/>
 不过秦鹏认为，WTO改革不会一帆风顺。
 <br/>
 <br/>
 秦鹏：〝中共这十几年就是靠钻WTO的空子来发家，所以它也肯定不甘心让新的规则把自己的手脚束缚起来。它会依然自说自话，‘一个WTO各自表述’。不过，目前其它主要经济体也是做了两手准备，一方面在协力共同推动改革，另一方面通过双边自由贸易协议，以及新的区域多边合作，在努力减少中共对国际贸易规则的破坏。〞
 <br/>
 <br/>
 美国《北京之春》杂志荣誉主编胡平表示，G20成员国对WTO需要改革虽然有了基本共识，但要如何改法，中间分歧还很多。
 <br/>
 <br/>
 美国《北京之春》荣誉主编胡平：〝（WTO）它到底还有多大用处，怎么样才能够发挥它别人所期待的功能，这个都还是没有解决的问题。〞
 <br/>
 <br/>
 另外，在美国反对下，联合声明中没有提及〝对抗贸易保护主义〞，这与过往的G20声明有明显差别。
 <br/>
 <br/>
 秦鹏表示，〝保护主义〞一词，以前常被其它国家，尤其是中共用来攻击美国总统川普。但通过一年多的贸易谈判，美国和大部分国家基本达成了一致。
 <br/>
 <br/>
 秦鹏：〝包括欧盟、日本、加拿大、墨西哥等，基本上已经或者正在谈判自由贸易协定。而且这里边（新北美自贸协定）有针对中共的‘毒丸条款’。对于中共来讲，现在实际上是有求于美国，这次，所以它肯定也不会坚持再提（保护主义）。〞
 <br/>
 <br/>
 就在G20峰会前夕，金砖五国领袖举行非正式会谈，还发表声明反对单边及保护主义，促请所有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世贸.html" target="_blank">
  世贸
 </a>
 成员反对与世贸不一致的措施。
 <br/>
 <br/>
 秦鹏说，这是中共为了面子和对内宣传所捣鼓出来的东西。而这些说辞，怎么看都像在说中共自己。
 <br/>
 <br/>
 气候议题方面，19个成员国在声明中同意实施〝不可逆转的〞巴黎气候协定（Paris Agreement）。不过，美国重申退出巴黎协定的决定，强调美国对经济发展和能源获取和安全的承诺，使用各种能源和技术，同时也保护环境。
 <br/>
 <br/>
 秦鹏：〝我们知道美国的环境保护是很好的。巴黎
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_气候协议.html" target="_blank">
  气候协议
 </a>
 最大的受益国，中共这边来讲，其它这些国家被中共欺骗了，它通过碳交易等方式从其它国家拿钱。中共环境保护问题不是（它宣称的）技术原因，也不是发展中国家原因。〞
 <br/>
 <br/>
 根据巴黎气候协定，到2025年之前，发达国家每年要向发展中国家提供约1000亿美元援助金。而在一百多个签署国中，美国每年就要承担约750亿美元。川普认为这不公平，给美国带来沉重的经济负担。同时，在这一协议下，中国还可以增加碳排放13年。美国则每年大约要减少16亿吨的碳排放。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/陈建铭
