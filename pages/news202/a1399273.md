### 高盛前银行家涉〝一马〞 马国首相：我们被骗了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】华尔街金融巨擘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高盛.html" target="_blank">
  高盛
 </a>
 集团2名前银行家因涉嫌卷入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马来西亚.html" target="_blank">
  马来西亚
 </a>
 〝一马发展公司（1MDB）〞贪污案，本月初遭起诉。对此，马国首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马哈蒂尔.html" target="_blank">
  马哈蒂尔
 </a>
 周二（13日）表示，高盛集团〝欺骗〞了我们。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马哈蒂尔.html" target="_blank">
  马哈蒂尔
 </a>
 （Mahathir Mohamad）周二接受CNBC电视台访问时，被问到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高盛.html" target="_blank">
  高盛
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大马.html" target="_blank">
  大马
 </a>
 的作为时说，证据显示高盛做了错事，高盛集团的银行家在处理一马公司的交易时，欺诈了大马。
 <br/>
 <br/>
 对于高盛内部负责监督防范风险作为的法律遵循制度，马哈蒂尔说：〝运作状况不是很好。〞但他没有透露详情。
 <br/>
 <br/>
 美国司法部日前拘捕前高盛集团银行家黄宗华（Ng Chong Hwa），并起诉另一位前高盛高层、东南亚地区主管莱斯纳（Tim Leissner），2人均被指控图谋受贿和洗钱，后者已认罪，且同意偿还4,370万美元的非法所得。高盛表示正在配合调查，可能面临天价罚款，该公司亚洲投资银行前联席主管韦拉（Andrea Vella）已被停职。
 <br/>
 <br/>
 美方也宣布起诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马来西亚.html" target="_blank">
  马来西亚
 </a>
 华人刘特佐（Low Taek Jho），指控刘特佐在〝一马发展公司〞和高盛（Goldman Sachs）及其他公司的多起交易中，起到中间人作用。
 <br/>
 <br/>
 36岁的刘特佐在马来西亚前首相纳吉布在任时与其子阿齐兹关系密切，被控参与挪用一马主权基金45亿美元的公款并协助洗钱；纳吉布败选后，马来西亚当局试图追捕刘特佐到案，但始终下落不明，有消息指他已整容、改名，并逃往中国大陆藏匿。
 <br/>
 <br/>
 检方还表示，高盛为〝一马发展公司〞工作时，收取了约6亿美元费用。
 <br/>
 <br/>
 纳吉布于2009年创立〝一马发展公司〞，美国司法部称，该公司挪用资金约45亿美元。目前至少6个国家在调查此事。
 <br/>
 <br/>
 马国财政部长林冠英12日表示，马国将要求高盛退回从一马公司的交易中赚取的所有款项。负面消息重击高盛股价，当天单日重挫7.46%，创2011年以来单日最大跌幅。
 <br/>
 <br/>
 马哈蒂尔接班人安华13日也称，马国必须采取更强硬的手段，向高盛索回款项，以挽回马国的形象。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁)
