### 蓬佩奥访朝鲜韩国 第二次川金会有望　
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】访问北京之前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 周日在平壤与金正恩举行会谈。蓬佩奥表示，双方就具体安排达成一致后，国际核查人员将获准进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 的核导弹试验场，同时，双方十分接近就美朝第二次峰会达成协议。
 <br/>
 <br/>
 周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 在平壤与金正恩举行会谈。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 官方电视台报导，金正恩表示，与蓬佩奥的对话〝有成效，很愉快〞，他很满意。双方立场获得充分理解，并交换了意见，并同意就第二次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川金会.html" target="_blank">
  川金会
 </a>
 进行工作谈判。
 <br/>
 <br/>
 蓬佩奥表示，金正恩准备允许国际核查人员前往朝鲜一个导弹引擎测试设施，以及丰溪里核试验设施。这是初期实施无核化的主要关键点之一。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 政府周一表示，金正恩与蓬佩奥会晤时间长达5个半小时，显示金正恩对此次访问的重视和诚意。
 <br/>
 <br/>
 蓬佩奥离开朝鲜后，前往韩国向文在寅报告这次朝鲜之行的进展。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝我们进行了良好、有成效的对话。正如川普总统所说，还有很多步骤要做，今天我们走了一步，向前迈出一步。我认为这对我们所有人来说，都是个好结果。〞
 <br/>
 <br/>
 韩国总统文在寅祝愿第二次川金会早日实现，朝鲜无核化指日可待。
 <br/>
 <br/>
 韩国总统文在寅：〝我深切希望川金会尽快召开，在无核化和和平进程方面，取得不可逆转的决定性进展。〞
 <br/>
 <br/>
 川普总统周日发推文说，〝蓬佩奥与金正恩在平壤会谈愉快。新加坡峰会协议取得进展！我期待不久的将来，再次见到金正恩〞。
 <br/>
 <br/>
 新唐人记者李梅、张琪综合报导
</div>
