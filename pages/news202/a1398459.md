### 第二轮美中外交安全对话 周五将开启
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】美国和中国将于周五，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华盛顿.html" target="_blank">
  华盛顿
 </a>
 举行第二轮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中外交安全对话.html" target="_blank">
  美中外交安全对话
 </a>
 。分析人士认为，这表明美中双方都有意图改善，连续几个月的紧张经贸和军事关系。
 <br/>
 <br/>
 周一，美国国务院表示，周五美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 与国防部长马蒂斯，将和中共政治局委员、中央外事工作委员会办公室主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨洁篪.html" target="_blank">
  杨洁篪
 </a>
 ，以及国防部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魏凤和.html" target="_blank">
  魏凤和
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华盛顿.html" target="_blank">
  华盛顿
 </a>
 会面。
 <br/>
 <br/>
 美国前驻华大使芮效俭接受媒体采访时说，这意味着两国都有意图改善双边关系，不让经贸冲突影响两国整体关系发展。另外，朝鲜核问题将是双方目前最重要议题。
 <br/>
 <br/>
 美中双方原定10月举行第二轮外交安全对话，但因美中贸易战升级，双方经济和军事紧张关系日益紧张，10月初，副总统彭斯严词指责中共当局盗窃美国敏感技术、实施外交和军事扩张，甚至试图干预美国中期选举。
 <br/>
 <br/>
 不过，两国最近出现缓和。11月2日，川普与习近平通电话，川普表示可能就贸易议题与中国达成好的协议。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
</div>
