### CNN记者暂获通行证 川普警告再不规矩就踢出去
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】CNN一记者被吊销
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 记者证后，将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）告上了法庭。16日，美国联邦法官裁决白宫暂时归还该记者的通行证。对此，川普警告，如果该记者〝再次行为不端，我会把他踢出去〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 16日接受了福克斯新闻（Fox News）主持人克里斯．华莱士（Chris Wallace）的专访时，当被问及联邦法官的裁决时，川普表示，〝这很好，这不是什么大问题〞。
 <br/>
 <br/>
 川普说，〝联邦法官的裁定是要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 制定记者的行为准则，我们正在草拟这项规则，快要完成了。〞〝如果他（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN记者.html" target="_blank">
  CNN记者
 </a>
 奥卡斯塔）再有不良行为，我们会把他踢出去，或者我们会停止新闻发布会。〞
 <br/>
 <br/>
 〝我会说谢谢大家，谢谢你们的到来，我要走人了。其他记者也不会对搞事的人太友好的。〞川普说。
 <br/>
 <br/>
 华莱士问行为准则的内容，川普回答：〝我们会有礼仪规则，让记者知道不能进行冗长的提问。〞
 <br/>
 <br/>
 〝我们正在制定，我们会有规矩的，你不能老是问问题。我们有很多记者在那间屋子里，他们没有提问的机会，全是因为这个人（奥卡斯塔）站起来为自己和CNN大声发问、陈述意见。〞川普补充说。
 <br/>
 <br/>
 白宫新闻发言人莎拉．桑德斯（Sarah Sanders）16日表示，〝作为对法庭的回应，我们将暂时恢复该记者的通行证。我们还将进一步制定规则和程序，确保今后的新闻发布会公平有序。白宫里一定要有礼仪。〞
 <br/>
 <br/>
 她同时说，法庭明确表示，宪法第一修正案并没有赋予个人进入白宫的绝对权利。
 <br/>
 <br/>
 美国中期选举结束后，川普于11月7日在白宫召开记者会。奥卡斯塔在提问中，试图与川普争辩中美洲大篷车非法移民的问题，并抓住麦克风不放，不愿意交给站在他旁边的一名白宫助手。会后，白宫撤销了他的驻白宫记者证。
 <br/>
 <br/>
 《福克斯》新闻评论说，奥卡斯塔的做法很不专业，并挤占了在场其他记者的宝贵提问时间。
 <br/>
 <br/>
 奥莱利（Bill O‘Reilly）在新闻节目〝No Spin News〞中，也特别就此发表评论说，〝美国媒体被腐蚀了。那些掌管新闻企业的老板们，对员工下达指令，告诉他们‘你要这样报导这条新闻’。我从业45年，从未经历过这种事，但如今总在发生。〞
 <br/>
 <br/>
 奥莱利说：〝CNN的目的是要抹黑川普。每个人都知道这一点。《纽约时报》、NBC、MSNBC和《华盛顿邮报》也抱有同样的目的。〞
 <br/>
 <br/>
 奥莱利在节目最后向奥卡斯塔提出了忠告。〝奥卡斯塔，没有人在乎你的想法和观点。如果你想阐明观点，可以去脱口秀上讲。〝你（在白宫新闻会上）浪费了美国总统和人民的时间。观众希望看到的是一问和一答。你的做法对每一个人都是彻头彻尾的不尊重。〞
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿）
</div>
