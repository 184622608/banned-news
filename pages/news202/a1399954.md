### 大篷车难民非法入境 墨西哥居民抗议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】周日，墨西哥边境城市蒂华纳，数百当地居民，和少数圣地亚哥居民举行示威游行，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 从中美洲涌入当地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 。目前，这些难民还在寻求进入美国，申请庇护。
 <br/>
 <br/>
 墨西哥当局预测，上周一，约6,000名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 涌入蒂华纳和另一座边境城市墨西卡利。
 <br/>
 <br/>
 周日，数百名蒂华纳市居民走上街头，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 难民非法进入墨西哥。他们的示威横幅上写着：〝移民可以，非法移民不行〞，〝不要入侵者〞。
 <br/>
 <br/>
 蒂华纳居民 DAVID SOLANO：〝第一，问题不在于出现移民，而在于他们（进入墨西哥）的方式。第二，恰帕斯州的移民官放他们进来，是他们惹的麻烦，但他们（移民官）平静地坐在恰帕斯州。这些移民官在哪？〞
 <br/>
 <br/>
 数千名大篷车难民涌入，导致蒂华纳收容所饱和，部分难民睡在街头。当局预测，难民将增加到一万人。
 <br/>
 <br/>
 蒂华纳居民RAFAEL DE JESUS ARMA：〝（大篷车难民）攻击代表这个国家的当局。撞倒墨西哥政府设置的路障。这不能接受。〞
 <br/>
 <br/>
 大部分难民称，他们不愿呆在墨西哥，将前往美国寻求庇护。
 <br/>
 <br/>
 洪都拉斯大篷车难民 KARINA ROSALES：〝他们有权抗议，因为他们不喜欢我们在这。尽管如此，他们（墨西哥人）是好人。他们和我们一样。但是我们只是路过，我们不会待在这。〞
 <br/>
 <br/>
 在美墨边境，墨西哥和美国同时加强防御。周六以来，蒂华纳通往圣地亚哥的车道上，筑起了大型钢铁路障。圣地亚哥一侧的边境拉上了带刺的铁丝网。
 <br/>
 <br/>
 美国总统川普已经向边境调遣了6,000名士兵，并暂停
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法入境.html" target="_blank">
  非法入境
 </a>
 者申请庇护的资格。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
</div>
