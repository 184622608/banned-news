### 11国新版TPP 日本宣布协定年底生效
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】周三，澳大利亚宣布完成对《全面与进步跨太平洋伙伴关系协定》，简称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CPTPP.html" target="_blank">
  CPTPP
 </a>
 的核准程序。同一天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 经济大臣茂木敏充表示，有11个国家参加的这项协定，12月30日起正式生效。
 <br/>
 <br/>
 周三，澳大利亚成为第六个完成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CPTPP.html" target="_blank">
  CPTPP
 </a>
 协议审核程序的成员国，之前加拿大、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 和墨西哥等国已完成核定程序。
 <br/>
 <br/>
 新西兰贸易和出口发展部长帕克表示，CPTPP是代替世贸组织、最重要的多边贸易方案。
 <br/>
 <br/>
 新西兰贸易和出口发展部长帕克：〝世贸组织目前已失效。〞
 <br/>
 <br/>
 新西兰贸易和出口发展部长帕克：〝目前（替代世贸组织）最重要的就是CPTPP，成员国代表全球13%国内生产总值。〞
 <br/>
 <br/>
 CPTPP参与国希望能消除
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易壁垒.html" target="_blank">
  贸易壁垒
 </a>
 ，推动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自由贸易.html" target="_blank">
  自由贸易
 </a>
 。
 <br/>
 <br/>
 周三，日本经济大臣茂木敏充表示，协议将于年底生效。
 <br/>
 <br/>
 日本经济大臣茂木敏充：〝11个国家参加的TPP协定，12月30日生效。〞
 <br/>
 <br/>
 《全面与进步跨太平洋伙伴关系协定》的前身是《跨太平洋伙伴关系协定》，简称TPP。
 <br/>
 <br/>
 去年美国宣布退出TPP后，今年1月，日本带领11个成员国，签署了CPTPP。随后，川普下令经济幕僚考虑美国重新加入TPP，同时也和日本进行双边贸易谈判。
 <br/>
 <br/>
 日本经济大臣茂木敏充认为，美日贸易谈判不会影响美国重返TPP，只会起积极作用。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
