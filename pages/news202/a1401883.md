### 美财长：中方态度转变 承诺1.2万亿额外贸易
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】美国财政部长姆努钦周一（3日）表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_承诺.html" target="_blank">
  承诺
 </a>
 额外购买价值超过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_1.2万亿.html" target="_blank">
  1.2万亿
 </a>
 美元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国商品.html" target="_blank">
  美国商品
 </a>
 ，以此作为美中元首日前达成协议的一部分，并将立即采取措施予以执行。
 <br/>
 <br/>
 据路透社（Reuters）报导，美国财政部长姆努钦（Steven Mnuchin）周一告诉美国全国广播公司财经新闻频道（CNBC），在中国国家主席习近平于布宜诺斯艾利斯（Buenos Aires）会晤美国总统川普后，中方官员的论调发生明显转变，因为习近平已明确
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_承诺.html" target="_blank">
  承诺
 </a>
 将对美国公司开放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 市场。
 <br/>
 <br/>
 姆努钦说，中共表示将额外购买超过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_1.2万亿.html" target="_blank">
  1.2万亿
 </a>
 美元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国商品.html" target="_blank">
  美国商品
 </a>
 ，不过细节仍需谈判。
 <br/>
 <br/>
 姆努钦表示，这是中共首次向美方承诺将真正达成协议。
 <br/>
 <br/>
 美国国家经济委员会主任库德洛（Larry Kudlow）也表示，他、姆努钦及美国贸易代表莱特希泽（Robert Lighthizer）与中国副总理在阿根廷共进午餐，对方表示北京将立即就新的承诺采取行动。
 <br/>
 <br/>
 库德洛承认，美方清楚中共以往在兑现承诺方面做得不太好。不过他想说的是，习近平并未卷入其中。
 <br/>
 <br/>
 库德洛表示，中方不能再拖延，他们所用的词汇是——〝立即〞。他在白宫告诉记者们，华盛顿希望看到北京迅速做出结构性改变，包括知识产权和技术转让等问题。他向CNBC表示，美国人将首次获得中国大陆公司的大部分所有权。
 <br/>
 <br/>
 美中双方已同意立即展开谈判，如不能在未来90天内达成协议，美国将把价值2千亿美元的中国产品的关税税率从10%提高到25%。
 <br/>
 <br/>
 （记者李佳欣编译报导/责任编辑：程以仁）
</div>
