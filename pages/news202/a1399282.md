### 日沿岸〝幽灵船〞或破纪录 设备简陋疑朝鲜鱼船
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】日本海岸每年都会数十艘破旧船隻随着海流漂向，今年漂至日本沿岸的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幽灵船.html" target="_blank">
  幽灵船
 </a>
 〞已经多达95艘，有可能超过2017年的104艘纪录。其中单是今年11月份就有33艘之多，且部分船上还有已经成白骨状的遗体。
 <br/>
 <br/>
 综合媒体报导，日本沿岸每年都漂来大量的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幽灵船.html" target="_blank">
  幽灵船
 </a>
 〞，自2014年的65艘、2015年的45艘、2016年的66艘，以及2017年的104艘；今年截至目前为止已经飘来95艘的空船。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海上保安厅.html" target="_blank">
  海上保安厅
 </a>
 表示，大部分的〝幽灵船〞都是木制船，通常没有任何船员，其中有一部份甚至只有尸体或白骨，以及小锅、杓子、渔具等物品，并在船上发现韩文的标志，怀疑这些船隻都来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 ，因船体结构不够强，且设备简陋，无力返回朝鲜，只能在茫茫大海上漂流。
 <br/>
 <br/>
 外界也猜测，幽灵船数量激增，可能是因为朝鲜派出更多渔船出海捕鱼，让国民能有更多蛋白质来源。
