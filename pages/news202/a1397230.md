### 专家赞彭斯演说 称中共早对西方搞冷战
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】本月初，美国副总统彭斯就美中关系发表重要讲话，在美国学界和政策圈内引起持续回响，他们都对这篇讲话给予高度评价。
 <br/>
 <br/>
 美国副总统彭斯在华府智库〝哈德逊研究所〞发表的美中关系讲话，全面揭露
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对美国多方面的威胁和恶意影响，被认为是美国对华政策的重要战略转变，也是一篇讨伐中共的檄文。
 <br/>
 <br/>
 加利福尼亚大学河滨分校教授林培瑞：〝他的这个讲话，是我一辈子听到一个美国高级的官员，关于中国的最好的一个讲话。很多不但是真话，而且是以前没有说过的真话。我们学者说过，很多记者也说过，可是高级官员说这些很明白的、批评北京政府的话，这是第一次。所以是很重要的讲话。〞
 <br/>
 <br/>
 前副总统切尼副国家安全顾问叶望辉：〝我认为，这是美国官员很多年以来，对美中关系发表的最全面演讲。我从未看到任何美国高层官员演讲中，谈到中共对于美国内政带来的挑战。这是展开对话的第一步。〞
 <br/>
 <br/>
 曾协助副总统切尼处理亚洲关系的叶望辉，精通两岸事务。他认为，副总统彭斯的演说，是对美中关系目前状况最正确的描绘，也是对中共最有效的回应。
 <br/>
 <br/>
 前副总统切尼副国家安全顾问叶望辉：〝中共展开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冷战.html" target="_blank">
  冷战
 </a>
 式的反美活动，  自有中华人民共和国以来就一直在进行。对我而言，这有点像美国政府内，在恐怖主义问题上的对话。恐怖主义份子早就对我们宣战了，在我们决定好如何回应前。由中共统治下的中华人民共和国，已先对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西方.html" target="_blank">
  西方
 </a>
 文明宣布冷战，问题只是西方文明是否决定要做出更有效的回应。〞
 <br/>
 <br/>
 叶望辉认为，中期选举后，美国政府与国会将在美中关系上采取具体行动。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
