### 美韩战时指挥权将移交 驻韩美军仍留守
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】美国与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 计划明年将战时作战指挥权移交韩国，先期工作已经展开。不过，在朝鲜半岛最终、完全和可核实地去核化之后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 仍将继续驻守朝鲜半岛，帮助韩国抵御外来的安全威胁。
 <br/>
 <br/>
 美国国防部长马蒂斯（James Mattis）和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 防长郑景斗（Jeong Kyeong-doo）当地时间周三（10月31日）在五角大楼会晤，双方就美韩联合部队在战时作战指挥权移交后将遵循的原则，达成协议并签署〝联合防卫指南〞。
 <br/>
 <br/>
 《美国之音》报导，根据五角大楼公布的英文韩文双语版的协议，在作战指挥权移交后，韩国将任命一位将军级官员担任美韩联军司令部司令，美国将任命一位将军级军官担任副司令。韩国国防部将在联合防御中担任领导角色，美国国防部将填补韩国军事能力上的欠缺，确保韩国的安全。
 <br/>
 <br/>
 韩国《朝鲜日报》报导，美韩于2014年的谘商会议达成〝有条件归还战时作战指挥权〞协议，如果韩国军方具备主导联合防卫和应对朝鲜核弹的战力，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 将归还战时作战指挥权。
 <br/>
 <br/>
 美韩两国判断移交指挥权是否成熟的3阶段中，明年将展开第一阶段确认基本运作能力，如果能依序进行，预计2021年即可完成验证。
 <br/>
 <br/>
 韩国军方人士称，最快在2022年5月前，美方将依据协议移交战时作战指挥权。
 <br/>
 <br/>
 联合声明还约定韩美开展太空及网络防务产业合作，并最终决定搁置原定于12月举行的〝警戒王牌〞"（Vigilant ACE）联合空中演习以免影响无核化对话。但为保持协防态势不放松，美国和韩国将通过单独演习弥补空白，双方将于12月1日之前商定日后的演习方向。
 <br/>
 <br/>
 美国国防部将这一协议称为美韩同盟关系的一个里程碑。美国目前有2万8千军人驻扎在韩国。
 <br/>
 <br/>
 美国总统川普（特朗普）今年6月在新加坡会见朝鲜领导人金正恩之后曾表示，他希望看到美军在未来某个适当时候撤出朝鲜半岛。
 <br/>
 <br/>
 防务专家们此前曾表示，韩国希望美军继续留守朝鲜半岛与中共不断在地区实施军事扩张有关。
 <br/>
 <br/>
 （记者李红报导/责任编辑：文慧）
