### 孟宏伟妻子真名疑曝光 网传北京还有豪宅
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】中共公安部副部长兼国际刑警组织主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 回国后被抓，他的妻子在法国的公开活动也引发关注。不过孟妻刻意对外隐瞒身份和真容，显得颇为神秘。但是，已有多名疑似知情者在网上曝光其真实身份，指其生活奢侈。
 <br/>
 <br/>
 10月10日，网上流传一则视频显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Grace Meng.html" target="_blank">
  Grace Meng
 </a>
 的真名叫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高歌.html" target="_blank">
  高歌
 </a>
 ，通过打网球与孟宏伟结识，当时高歌还未离婚。
 <br/>
 <br/>
 视频还爆料，高歌在北京拥有千万的房产。
 <br/>
 <br/>
 其实，早在10月7日，就有网友匿名在海外网站留言指，Grace Meng真名叫高歌，比孟宏伟小19岁，给孟生了一对双胞胎。高歌用香港身份擅自到达法国，以每年60万人民币的价格租了一套别墅，孩子在那里上国际学校，日常消费粗算最少也要每年200万元人民币。
 <br/>
 <br/>
 该网友指，高歌是孟宏伟贪腐的同案犯。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-12/p9075961a838147054.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-12/p9075961a838147054-ss.jpg"/>
   <br/>
   <font size="-1">
    网友在海外网站留言，曝光孟宏伟妻子身份资料。（网页截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 也有推特消息称，Grace Meng持香港护照，无业，两年来在法国里昂最豪华别致的街区之一租住一幢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_豪宅.html" target="_blank">
  豪宅
 </a>
 ，月租5万多人民币。由于并无〝驻外人员家属〞身份，她在法国的奢侈生活并非由国际刑警组织或中共公安部出资。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-10/p9073891a641019835.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-10/p9073891a641019835-ss.jpg"/>
   <br/>
   <font size="-1">
    推特上有关孟宏伟妻子的身份资料。（网页截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 上述多个消息的内容大致相符。不过，目前并未得到官方消息证实。
 <br/>
 <br/>
 日前，Grace Meng在法国两次接受外媒采访，声称自己和孟宏伟〝无辜〞，指责北京对孟进行〝政治迫害〞。因此有猜测指，上述〝内幕〞消息，或是北京内部有意透出，以反驳Grace Meng的说法。
 <br/>
 <br/>
 也有分析指，孟宏伟身为正部级官员，其家属的身份资料，中组部和情报部门应该一清二楚，Grace Meng受访时刻意对自己的身份外貌保密，或意在塑造〝弱者〞及〝无安全感〞形象，博取舆论主动与大众同情。
 <br/>
 <br/>
 10月4日，Grace Meng向法国警方报警，指孟宏伟回中国后失踪。7日，Grace Meng接受CNN采访，曝光孟宏伟9月25日失联前用手机给她发回的一个尖刀图案。
 <br/>
 <br/>
 外界普遍分析，该图案可能是示警，更可能是一个行动代码。孟应该对自己的处境早有警觉，事先安排了安全预案。
 <br/>
 <br/>
 Grace Meng对外媒表示，北京已经和她有过联络，并针对她派出两个行动小组。
 <br/>
 <br/>
 外界猜测，从孟宏伟9月25日发回尖刀图案，到Grace Meng 10月4日报警，这中间一周里，Grace Meng也应该没有坐以待毙，而是早就有所行动。
 <br/>
 <br/>
 孟宏伟落马后，中共公安部会议称，孟是〝一意孤行、咎由自取〞，并表示〝绝不允许跟组织谈条件、谈‘价钱’〞，或暗示孟宏伟落马前一直在和北京当局进行对抗和博弈。
 <br/>
 <br/>
 有分析称，孟宏伟作为公安系统高官，手中或掌握中共高层海外资产的情况。
 <br/>
 <br/>
 外界观察人士注意到，有港媒日前开始曝光中共政治局常委家族在香港的豪宅和资产，时间与孟妻扬言追求〝真相〞有巧合之处，但目前尚无证据显示二者之间有任何因果联系。
 <br/>
 <br/>
 孟宏伟被指周永康余党。外界认为，中共政法系统镇压法轮功、维权人士甚至活摘器官，孟宏伟应该都有份参与。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
