### 中美洲难民潮 川普警告关闭美国边境
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】来自洪都拉斯和危地马拉等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲.html" target="_blank">
  中美洲
 </a>
 国家的大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民潮.html" target="_blank">
  难民潮
 </a>
 ，继续涌向美国南部边境。周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统向这些国家发出警告，要求他们阻止难民潮跨越边境，否则将关闭边境，停止援助。
 <br/>
 <br/>
 为躲避纷乱和贫穷，大批洪都拉斯民众拖家带口，穿越丛林，取道危地马拉和墨西哥，希望逃到美国寻求庇护。
 <br/>
 <br/>
 洪都拉斯移民Carlos Castellano:〝这是必须的，因为政府引起的经济危机，所以我们要移民。〞
 <br/>
 <br/>
 危地马拉边境警察拦截了大批没有合法文件的难民，为安全起见，他们提供大篷车，让难民乘坐通过边境。
 <br/>
 <br/>
 上周六开始，涌入危地马拉的洪都拉斯难民持续增加。
 <br/>
 <br/>
 危地马拉总统莫拉莱斯周三表示，将与洪都拉斯合作，帮助想回来的难民返回家园。
 <br/>
 <br/>
 危地马拉总统 Jimmy Morales:〝我们和洪都拉斯总统赫尔南德斯一直保持通话，以协商出合适的、多样而且更人道的返回方式。这是针对那些想回来的民众，我们知道有些人想回来。我们会提供资源帮助。〞
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统连发推文，批评民主党开放边境的移民政策，再次引来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民潮.html" target="_blank">
  难民潮
 </a>
 。
 <br/>
 <br/>
 推文还警告说：〝除了停止对这些国家的所有援助，那似乎没能控制难民数量，我必须用最强烈的措辞，要求墨西哥阻止这次难民潮的猛烈攻势，如果墨西哥做不到，我将动用美军，并且关闭南部边境！〞
 <br/>
 <br/>
 〝对我们南部边境的攻击，包括犯罪隐患和毒品的涌入，作为总统，这比贸易，比美墨加贸易协定更重要。〞
 <br/>
 <br/>
 川普批评民主党人，要为脆弱的移民法律承担责任，并呼吁共和党参选人在中期选举中，可以用移民问题来争取选民。
 <br/>
 <br/>
 新唐人记者安心、宇微综合报导
</div>
