### 脱欧草案获英内阁支持 过下议院是难关
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】周三，英国脱欧谈判迈出一大步，英国内阁通过了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 。但草案尚要得到英国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下议院.html" target="_blank">
  下议院
 </a>
 和欧盟27国首肯。
 <br/>
 <br/>
 5小时内阁紧急会议后，周三傍晚，首相梅走出唐宁街10号大门，宣布内阁通过了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 。语气中没有庆祝的意味，却有一丝沉重。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国首相.html" target="_blank">
  英国首相
 </a>
 特雷莎・梅：这是决定性的一步，使我们能够在未来的日子里继续前进并达成协议。(我相信)这个决定符合整个英国的最佳利益。
 <br/>
 <br/>
 首相梅称，这500页脱欧草案的背后是数千小时的工作量和英欧之间进行的多次会谈。
 <br/>
 <br/>
 英国首相 特雷莎・梅：我们面临着艰难的抉择，尤其是有关北爱尔兰保障措施的问题。
 <br/>
 <br/>
 周四，首相梅将向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下议院.html" target="_blank">
  下议院
 </a>
 解释内阁的决定。
 <br/>
 <br/>
 预计12月中下议院将对草案投票，这将艰难的一战。目前，工党、保守党中强硬脱欧派和民主统一党都已表示将投反对票，超过了半数。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
