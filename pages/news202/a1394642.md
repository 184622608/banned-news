### 两货船在地中海相撞 导致燃油外洩
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】两艘大型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货船.html" target="_blank">
  货船
 </a>
 周日（7日）在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科西嘉岛.html" target="_blank">
  科西嘉岛
 </a>
 附近海域相撞后导致大规模
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_漏油.html" target="_blank">
  漏油
 </a>
 。法国和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 当局周一（8日）已派船尝试控制长达20公里的油污，预计行动需耗时数天。法国当局已对事件展开调查。
 <br/>
 <br/>
 美联社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 海事局周一发表声明称，从法国海军发布的照片中可见，肇事的突尼斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货船.html" target="_blank">
  货船
 </a>
 船头直接撞向塞浦路斯货船的舷侧，导致大量燃油外洩。
 <br/>
 <br/>
 法国海事局发言人周一表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_漏油.html" target="_blank">
  漏油
 </a>
 已经造成了20公里长、数百米宽的污染面积。地点从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科西嘉岛.html" target="_blank">
  科西嘉岛
 </a>
 向西北方向，接近法国和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 的海岸。所幸事故中没有人受伤。
 <br/>
 <br/>
 发言人表示，正在严密观察当地的天气变化。法国、意大利和摩纳哥正联合处理污染事故。。
 <br/>
 <br/>
 科西嘉警方表示，有关部门正在调查事发原因。
