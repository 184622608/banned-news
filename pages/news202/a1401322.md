### 三米水晶巨星 纽约洛克斐勒圣诞树华丽点灯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】在纽约市，点亮洛克菲勒广场的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞树.html" target="_blank">
  圣诞树
 </a>
 ，就意味着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞佳节.html" target="_blank">
  圣诞佳节
 </a>
 即将来临。每年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_点灯仪式.html" target="_blank">
  点灯仪式
 </a>
 已经成为重要传统，下面随我们的记者一起来看今年的盛况。
 <br/>
 3、2、1
 <br/>
 <br/>
 在我身后的这颗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞树.html" target="_blank">
  圣诞树
 </a>
 ，就是今晚的大明星，成千上万的民众在寒风中等待，就是为了见证它被点亮的那一刻。
 <br/>
 <br/>
 纽约民众Lindsey Lindsaymonarca：〝非常漂亮，很大，我很爱今年的星星。〞
 <br/>
 <br/>
 纽约民众Zayeli Zempoaltecato：〝我们在这里能很清楚的看到圣诞树，无法形容的感受。〞
 <br/>
 <br/>
 今年的圣诞树，是一颗75岁的挪威云杉，来自纽约上州的沃基尔镇（Wallkill）。72英尺高，直径45英尺，重约12吨。
 <br/>
 <br/>
 树上装饰着超过五万盏的彩灯，最顶端的星星是由300万颗施华洛斯奇水晶组成，重达900磅，相当耀眼。
 <br/>
 <br/>
 洛克菲勒圣诞树星星设计师Daniel Libeskind：〝星星象征和平，不分信仰、种族、国家，星星能和每个人产生连结，不分任何界线，我认为这就是星星的美妙之处，这就是启发，我们在世界的每一处都能看到星星的光芒，将人类连结在一起。〞
 <br/>
 <br/>
 还没有进入12月，民众就已经开始唱着圣诞歌曲，欢庆圣诞，可见圣诞节日的感染力。
 <br/>
 <br/>
 歌手Billy Porter：〝圣诞节代表爱，也代表家人，也代表能停歇片刻，跟爱的人相聚，记起我们在这地球上的目的。〞
 <br/>
 <br/>
 纽约民众Nancy Gutierrez：〝能感觉到人们的团结，在这个重要的时刻。〞
 <br/>
 <br/>
 这是纽约市洛克斐勒中心的第86颗圣诞树，它将在接下来的一个多月，陪伴纽约民众
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欢度节日.html" target="_blank">
  欢度节日
 </a>
 。
 <br/>
 <br/>
 新唐人记者王愉贺、奥利弗纽约报导
</div>
