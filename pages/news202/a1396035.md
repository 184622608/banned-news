### 中美洲难民潮持续 蓬佩奥访墨西哥
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】连日来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 大篷车载着数千难民，不断涌向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_危地马拉.html" target="_blank">
  危地马拉
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 ，试图从美国南部边界入境。美国总统川普周四警告，必要时会关闭边境。周五，国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 到访墨西哥和危地马拉，协商解决
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民危机.html" target="_blank">
  难民危机
 </a>
 。
 <br/>
 <br/>
 周三晚间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 警方封锁边境，阻止难民通过，大批难民滞留在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_危地马拉.html" target="_blank">
  危地马拉
 </a>
 和洪都拉斯边境。
 <br/>
 <br/>
 也有很多难民没有停下脚步，他们绕过封锁线，拖家带口穿过丛林，跋山涉水，继续向西走。
 <br/>
 <br/>
 为躲避国内黑帮暴力、毒品泛滥和贫穷，数千洪都拉斯难民想要穿越危地马拉和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 ，到美国寻求庇护。
 <br/>
 <br/>
 洪都拉斯难民 Alfredo Dominguez：〝在我们国家真的没有人能活下去。大部分人都是为了安全。这是事实。〞
 <br/>
 <br/>
 成功越过边界的难民，有的选择先在危地马拉的庇护所暂时停留，补充食物和水，然后继续前行。
 <br/>
 <br/>
 洪都拉斯难民 Karen Aviles：〝我出来是因为我女儿染上毒瘾，为了救她。她流落街头。我留下六个小孩在家，带出来两个，年龄最大的两个。年纪小的可以等等。〞
 <br/>
 <br/>
 周五，美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 抵达墨西哥，将会晤墨西哥总统涅托和外交大臣卡索，商讨如何处理洪都拉斯难民问题。之后，蓬佩奥还将前往危地马拉。
 <br/>
 <br/>
 川普总统周四表示，如果中美洲国家不能有效阻止大篷车难民，除了停止给这些国家的援助，必要时还将动用军队，并关闭南部边境。
 <br/>
 <br/>
 <br/>
 新唐人记者安心、张琪综合报导
</div>
