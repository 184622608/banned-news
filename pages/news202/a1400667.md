### APEC公报姗姗来迟  最终版本仍存分歧
------------------------

<div class="wysiwyg">
 【新唐人2018年11月24日讯】周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 峰会，终于发布了迟来的最终声明。公报中罕见提到，有少数国家，对部分内容，持有不同看法，再次印证，美中双方在贸易立场上的分歧。
 <br/>
 <br/>
 已经结束五天的2018年度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 峰会，周五发布了一份〝主席声明〞。不同于以往的〝领袖声明〞，这份〝主席声明〞一开始就阐述了各成员国之间存在的分歧。
 <br/>
 <br/>
 声明的开头说，〝这份主席声明反映了主席国对所有APEC成员国主流观点的评估〞，〝大部分经济体同意以下内容，但有少数可能对第9，16和17小节存在不同或额外的看法〞。
 <br/>
 <br/>
 分析认为，类似内容从未在近年的APEC公报中出现，等于承认所有成员国无法在某些议题上达成一致。
 <br/>
 <br/>
 公报在存有争议的第9小节中说，〝我们进一步敦促各经济体，以自由、公平、开放的姿态促进区域贸易往来〞。
 <br/>
 <br/>
 同样存有争议的第16、17小节中，公报呼吁改善WTO的运作机制。而公平贸易和WTO机制，正是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易.html" target="_blank">
  美中贸易
 </a>
 争端的核心议题。
 <br/>
 <br/>
 日本《日经新闻》此前披露，最初公报草案中包含〝对抗一切形式的保护主义〞的内容，但北京对此提出不同的意见。
 <br/>
 <br/>
 《华尔街日报》则引用一位美国官员的话说，其他所有成员国都同意上述内容，只有北京反对。
 <br/>
 <br/>
 峰会结束后，更爆出中共官员硬闯主办国巴布亚新几内亚外交部长办公室的事件，令全球感到震惊。《华盛顿邮报》在报导中，将此行为称为〝发脾气〞外交。
 <br/>
 <br/>
 APEC峰会的一系列风波，让全球对于月底G20峰会上的川习会更加关注。新加坡国立大学南亚研究学院资深研究员帕利特认为，这次会晤取得的最好结果，恐怕也只限于在一定程度上缓和美中目前的贸易争端。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
