### 法媒：中共在美遇阻  转向联合国悄悄渗透扩权
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】华盛顿日前揭露并指责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 通过一系列的〝影响力活动〞，试图干预即将举行的美国中期选举。法国媒体称，在美遇阻之际，中共还在向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 悄悄开辟路径以渗透扩权。
 <br/>
 <br/>
 10月8日，《法广》发表评论文章称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 的发言通常并不声张，却常常以援引联合国宪章作为借口来迷惑人：诸如国家主权以及不干涉他国原则等等。但在联合国派遣维和部队，竞选联合国秘书长职务等重大事件上，中共的存在感越来越强。
 <br/>
 <br/>
 据悉，目前北京派出的维和部队蓝盔兵共有2500人，部署在黎巴嫩、马里、刚果金或者南苏丹。
 <br/>
 <br/>
 文章引述一名联合国外交官指出，来自北京的维和兵份量大增，尤其在自愿向联合国出资方面，中共的份额也在大幅增加。相反，川普政府则减少了对联合国的经费贡献。
 <br/>
 <br/>
 这一外交官还告诉《法新社》，中共通过大量资助联合国调查、联合国特别任务来〝购买影响力〞，并且把自己中意的候选人推举到联合国各个重要岗位，〝这是获得信息的源泉，同时也是发挥影响力的中途站。〞
 <br/>
 <br/>
 另一名不愿公开姓名的联合国外交官也称, 中共不缺席任何在联合国次一级的权力机构，比如15个安理会理事国的专题讨论，而且其正在寻求取代荷兰成为联合国涉及阿富汗文件的主要起草国，〝中国（共）正在攫取联合国大权〞。
 <br/>
 <br/>
 此外，欧盟的外交官们还注意到，中共与俄罗斯之间在投票问题上一直保持相互支持的关系。如今，北京在联合国力量上升的同时，正在缓缓削弱着这种关系。当俄罗斯动用否决权时，北京有时以弃权应对。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：祝馨睿）
 <br/>
</div>
