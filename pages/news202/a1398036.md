### 意大利连续坏天气 西西里河流决堤多人死
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雨.html" target="_blank">
  暴风雨
 </a>
 肆虐超过一周的意大利，大部分地区几乎瘫痪。周六，意大利南部岛屿西西里岛，河水泛滥造成至少9人死亡。
 <br/>
 <br/>
 意大利国家广播电台说，由于遭受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雨.html" target="_blank">
  暴风雨
 </a>
 袭击，西西里北部巴勒莫（Palermo）附近河水漫过河堤，淹没一户人家，当时两个家庭的人聚在一起共度周末，除一人在洪水中紧紧抓住一棵树幸免遇难，其他9人不幸死亡。
 <br/>
 <br/>
 周日，西西里岛附近发生的另一起事故中，洪水冲走了一辆汽车，造成两人失踪，其中包括前往医院的一名医生。
 <br/>
 <br/>
 早些时候，意大利北部的暴风雨造成至少15人死亡，阿尔卑斯山谷(Alpine Valleys)附近数百万棵树被连根拔起，几个村庄几天内没有电力，交通隔断。威尼托大区区长札亚周六在山区视察了灾情(Luca Zaia)。
 <br/>
 <br/>
 一轮强降雨自10月27日开始席卷意大利。29日，全境出现大范围暴风雨，包括威尼托、利古里亚、罗马和托斯卡纳等6个大区发布红色预警，大量学校被迫停课。
 <br/>
 <br/>
 新唐人记者严枫、天羽综合报导
