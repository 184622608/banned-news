### 疑当朝鲜间谍 法参议院官员被控叛国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】曾足迹遍布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜半岛.html" target="_blank">
  朝鲜半岛
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国参议院.html" target="_blank">
  法国参议院
 </a>
 高阶公务员奎内迪，25日被控叛国及为朝鲜从事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍.html" target="_blank">
  间谍
 </a>
 活动，目前被拘留在法国国内情报总局（DGSI）位于巴黎郊区的总部。
 <br/>
 <br/>
 奎内迪（Benoit Quennedey）自2007年起担任〝法韩友好协会〞（Franco-Korean Friendship Association）会长，这是1960年代由同情社会主义分子与共产党的记者所成立，推广与平壤更紧密的关系，并支持韩朝鲜统一。
 <br/>
 <br/>
 调查奎内迪工作3月便已展开。司法部门消息人士表示，奎内迪是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国参议院.html" target="_blank">
  法国参议院
 </a>
 高阶行政人员，身兼法韩友好协会会长，在参议院任职于建筑、传统及林园部门。但奎内迪被控叛国及为朝鲜从事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍.html" target="_blank">
  间谍
 </a>
 活动，于25日晚间遭拘。
 <br/>
 <br/>
 法国参议院也证实，他的办公室日前已遭警方搜索，他被控〝向外国势力传递消息叛国〞遭限制出境，参议院的工作也已被禁。
 <br/>
 <br/>
 据帮奎内迪出过书的出版商〝德尔加〞（Delga）的网站，奎内迪的足迹遍布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜半岛.html" target="_blank">
  朝鲜半岛
 </a>
 。在上传至YouTube的影片中说：〝我自2005年起造访那里7次，在朝鲜，你会注意到，路边没有垃圾。〞
 <br/>
 <br/>
 朝鲜领导人金政恩与美国总统川普，今年6月在新加坡举行峰会，双边局势大微缓和。而8月时在YouTube的分享影片显示，奎内迪表示乐见紧张局势缓解。
 <br/>
 <br/>
 一名奎内迪以前的同学表示，奎内迪称赞朝鲜是世外桃园已近20年。奎内迪形容贫困、孤立的朝鲜是〝发展的模范〞，并称赞朝鲜公民的免费教育及健康照护制度。
 <br/>
 <br/>
 奎内迪曾于法国的巴黎政治大学（Sciences Po）就学，也曾就读于培养高级公务员及政治领袖的国家行政学院（ENA）。
 <br/>
</div>
