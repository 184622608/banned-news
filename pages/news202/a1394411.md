### 展威慑力大规模演习 美军多架战机纷抵乌克兰、英国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】美国为展现对欧洲地区安全承诺，以及对当前俄罗斯潜在威胁起到威慑力，6日，多架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-15C战机.html" target="_blank">
  F-15C战机
 </a>
 首次飞抵乌克兰，准备参加当地史上最大型的空军演习；同时也派出多架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-22隐形战机.html" target="_blank">
  F-22隐形战机
 </a>
 飞抵英国短期部署。
 <br/>
 <br/>
 9月底，俄国军方才在克里米亚半岛进行多项军事演习，升高紧张情势；北约为此首度在乌克兰举行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_干净天空2018.html" target="_blank">
  干净天空2018
 </a>
 〞（Clear Sky 2018）联合演习，展现北约维护欧俄边界安全的坚定立场。
 <br/>
 <br/>
 〝干净天空2018〞空军演习将于8日至19日举行，旨在加强区内国家保护领空、空中机动作战、空中伤患撤离、网络防御的能力。
 <br/>
 <br/>
 参演国家包括美、英、比利时、丹麦、爱沙尼亚、荷兰、波兰、罗马尼亚、乌克兰等共9个国，将出动包括F-15战机、KC-135加油机以及MQ-9无人机在内的航空器，展现维护边界的决心。
 <br/>

 另一方面，作为军方〝猛禽调整部署〞行动的一部分，美军多架F-22隐形战机结束在中东的部署后，于5日飞抵英国莱肯希思空军基地，期间将与北约成员国及伙伴的空军共同训练及执行任务。
 <br/>
 <br/>
 军方表示，是次部署行动展示美国对欧洲地区安全的承诺，以及对当前与潜在威胁的威慑力。
