### 濒临与俄开战 乌总统向德国北约寻求军事协助
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】继扣押乌克兰3艘船舰后，俄罗斯又封锁了乌克兰位于亚速海的2个海港，令乌克兰与俄罗斯全面开战的可能性上升。乌国总统指控俄当局企图重建沙皇时期的〝俄罗斯帝国〞，正觊觎乌克兰的领土，他向德国总理和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北约.html" target="_blank">
  北约
 </a>
 盟国喊话，寻求军事支持。
 <br/>
 <br/>
 为应对俄罗斯近日在亚速海的步步紧逼，乌克兰总统波洛申科（Petro Poroshenko）29日通过德国《图片报》向德国总理默克尔（Angela Merkel）喊话，希望她能够与乌克兰的其他盟友一起，再次出手支持乌克兰对抗俄罗斯。
 <br/>
 <br/>
 与此同时，波洛申科还向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北约.html" target="_blank">
  北约
 </a>
 盟国发出呼吁，希望各盟国能够尽快向亚速海派遣军舰，维护该水域的航行安全并支持乌克兰。
 <br/>
 <br/>
 波洛申科表示，俄罗斯总统普廷（Vladimir Putin）想重建沙皇时期的〝俄罗斯帝国〞，因此现在觊觎着乌克兰的领土。他呼吁西方国家能加大对俄罗斯的制裁，以避免乌克兰被迫与俄罗斯开战。
 <br/>
 <br/>
 据台湾《自由时报》的报导，默克尔11月26日已分别与波罗申科及普京通过电话，但目前德国并未决定是否要对俄罗斯采取加大制裁的行动。北约秘书长史托腾伯格（Jens Stoltenberg）则重申：北约与乌克兰的现有合作框架中，并不包括军事支持。
 <br/>
 <br/>
 不过，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 现任外交和安全政策高级代表茉格里尼（Federica Mogherini）已经与28个成员国发表了一个联合声明，要求俄罗斯当局必须确保任何船隻都能自由通过克赤海峡，并释放被俘的乌克兰舰艇与官兵。
 <br/>
 <br/>
 今年11月25日，乌克兰海军的两艘炮艇和一艘拖船在穿过连接黑海和亚速海的刻赤海峡（Kerch Strait）时，遭俄罗斯海军拦截。俄罗斯船舰向乌克兰船隻开火，导致约6名船员受伤，随后俄军扣查了乌克兰的这3艘船隻以及20多名船员。俄罗斯方面的理由是乌克兰的船舰闯入了俄的〝领海〞，但俄方的这一说辞并未得到国际社会的认可。
 <br/>
 <br/>
 联合国安全理事会26日针对俄乌之间的这场冲突，召开了紧急会议进行讨论。
 <br/>
 <br/>
 美国驻联合国代表黑莉（Nikki Haley）在这次会议上明确指出，俄罗斯扣押乌克兰船隻是〝严重的违法行动〞，将阻碍美俄之间发展正常关系的努力。
 <br/>
 <br/>
 〝阻碍乌克兰合法穿越刻赤海峡，是违反国际法的。这是一种傲慢的行为，国际社会必须谴责并不能接受。〞黑莉说：〝（为了）维护国际和平与安全，俄罗斯必须立即停止其非法行为，尊重所有国家的自由航行权利。〞
 <br/>
 <br/>
 同一天（11月26日）乌克兰国会通过表决，决定在该国的10个地区，从28日起实施戒严30天的戒严令，军队全面备战。
 <br/>
 <br/>
 欧盟和北约也相继就当前俄乌的冲突而发表声明，敦促俄乌双方克制，同时要求俄罗斯恢复刻赤海峡的航行自由。
 <br/>
 <br/>
 11月28日，乌克兰总统波罗申科（Petro Poroshenko）在当地电视台一档节目亮相时表示，乌克兰正面临与俄罗斯全面战争的威胁，希望任何人不要认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄乌冲突.html" target="_blank">
  俄乌冲突
 </a>
 只是小事一桩。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：东野）
