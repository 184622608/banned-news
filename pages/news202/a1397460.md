### 恶劣天气袭欧 意大利水坝溃堤冲击港口游艇
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狂风暴雨.html" target="_blank">
  狂风暴雨
 </a>
 影响欧洲部分地区，意大利、法国、西班牙纷纷传出灾情。意大利热那亚一座大坝因水压过高溃堤，目击者描述当时宛如海啸来袭一样，目前已知至少11人死亡；法国与西班牙则因暴风雪，部份地区旅客受困、电力中断。
 <br/>
 <br/>
 意大利热那亚（Genoa）附近的拉巴罗（Rapallo），一座大坝港因水压过高溃堤，惊愕不已的港口工作人员向媒体说，当时情景〝简直如海啸来袭一样〞，导致停靠在港口豪华游艇强烈碰撞，船身几乎全毁。
 <br/>
 <br/>
 猛烈风暴刮起每小时180公里强风造成6人丧生，加上昨日证实的5死，总死亡人数来到11。另继去年遭60年一遇的旱灾后，意大利北部本周受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狂风暴雨.html" target="_blank">
  狂风暴雨
 </a>
 侵袭，河水泛滥。
 <br/>
 <br/>
 有〝水都〞之称的旅游胜地威尼斯（Venice）29日变成一片泽国，全市有3/4面积被淹没，水位暴涨至160厘米，创下10年来最高。当局暂停市内水上巴士服务，并疏散圣马可广场（San Marco Square）游客。
 <br/>
 <br/>
 威尼斯所在的威尼托大区（Veneto）省长扎亚认为，威尼斯水灾的严重程度可媲美1966年的佛罗伦斯水灾。
 <br/>
 <br/>
 <center>

   </font>

 </center>
 <br/>
 而在意大利与瑞士交界处，斯蒂维奥山口（Stelvio Pass）降下大雪，造成约170名游客及饭店工作人员受困。
 <br/>
 <br/>
 法国中央高原地区（Massif Central）则发生暴风雪，山路积雪过多，造成1000多名驾驶受困。东部城市里昂（Lyon）有火车因积雪动弹不得，400名旅客夜宿车厢。东部和中部地区约有19万5000户人家无电可用。
 <br/>
 <br/>
 西班牙周末一场大雪导致北部的阿斯图里亚斯（Asturias）若干区域与外界失联，100多名军人正协助电路维修人员前往灾区。
 <br/>
 <br/>
 在克罗埃西亚境内亚得里亚海(Adriatic Sea)沿岸受到强风豪雨袭击，街道瞬成河流，居民电力遭断，海陆交通均受阻。
