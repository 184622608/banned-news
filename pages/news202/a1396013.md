### 中国人俄罗斯遇绑匪 竟是自己同胞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】近日，一位中国妇女向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警方.html" target="_blank">
  警方
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_报案.html" target="_blank">
  报案
 </a>
 ，称自己19岁的儿子遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绑架.html" target="_blank">
  绑架
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绑匪.html" target="_blank">
  绑匪
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_勒索.html" target="_blank">
  勒索
 </a>
 1000万卢布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赎金.html" target="_blank">
  赎金
 </a>
 。警方破案后说，绑匪的身份令人惊讶，竟然是在当地经商的中国同胞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 鄂木斯克内务部10月18日宣布，破获一起外国公民遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绑架.html" target="_blank">
  绑架
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_勒索.html" target="_blank">
  勒索
 </a>
 案。
 <br/>
 <br/>
 一名中国母亲近日向当地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警方.html" target="_blank">
  警方
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_报案.html" target="_blank">
  报案
 </a>
 称，有不明身份的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绑匪.html" target="_blank">
  绑匪
 </a>
 带走了她19岁的儿子，绑匪给她打电话，向她勒索巨额
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赎金.html" target="_blank">
  赎金
 </a>
 。
 <br/>
 <br/>
 这位中国母亲用俄语描述说：〝当时绑匪打电话过来，要求我给他们1000万卢布（约106万人民币）。我对他说，这是夜里，我到哪儿去找这么多钱。最后，我从女儿那里借到了一些钱，凑足了赎金。〞
 <br/>
 <br/>
 于是这位母亲与当地警方取得联系，在俄警的安排下，同绑匪进行了赎金交接，换回了儿子。很快，俄警破获了这起绑架案。
 <br/>
 <br/>
 破案后，俄警说，绑匪身份令人感到惊讶。原来这个绑架案的策划者，竟然是同样在俄罗斯经商的另一名中国同胞。
 <br/>
 <br/>
 不过，俄警至今也未对外透露，这起被绑架的中国公民和报案的中国妇女的姓名等个人信息。仅称，遭到绑架的这位年轻人的母亲，是在俄罗斯经商的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
  中国人
 </a>
 。
 <br/>
 <br/>
 俄警还以此案正在进一步调查为由，拒绝透露已被抓的这名中国绑匪的姓名等个人信息，只是称，这名嫌疑人在被拘捕过程中，试图逃跑和反抗，脸部有擦伤和淤伤。
 <br/>
 <br/>
 目前该案仍在调查审理中。
 <br/>
 <br/>
 近年来，中国人在俄罗斯绑架自己同胞的案件频发。早在2005年初，莫斯科警方就曾向在俄罗斯的中国人发出通告，一定要小心警惕你们自己的同胞，因为他们更清楚华商在俄罗斯的生活及经济情况。
 <br/>
 <br/>
 根据俄罗斯警方掌握数据，绝大多数在莫斯科发生的中国人被害案件，均是由中国人策划雇凶或直接参与干的。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：赵云）
