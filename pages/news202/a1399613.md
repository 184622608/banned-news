### 种族灭绝罪成 红色高棉两头目终身监禁
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】周五，联合国柬埔寨法院特别法庭判决，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红色高棉.html" target="_blank">
  红色高棉
 </a>
 〞，又称柬埔寨共产党的两名前头目，犯有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_种族灭绝罪.html" target="_blank">
  种族灭绝罪
 </a>
 ，判处
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_终身监禁.html" target="_blank">
  终身监禁
 </a>
 。这是官方首次裁决柬埔寨共产党犯有种族灭绝罪 。
 <br/>
 <br/>
 周五，柬埔寨法院特别法庭宣布，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红色高棉.html" target="_blank">
  红色高棉
 </a>
 〞、也就是柬埔寨共产党的〝二号人物〞、92岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农谢.html" target="_blank">
  农谢
 </a>
 ，和前国家主席、87岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乔森潘.html" target="_blank">
  乔森潘
 </a>
 ，对柬埔寨穆斯林和越南裔民众，犯有危害人类罪和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_种族灭绝罪.html" target="_blank">
  种族灭绝罪
 </a>
 。
 <br/>
 <br/>
 联合国、柬埔寨官员和民众，都对法院的判决表示欢迎。
 <br/>
 <br/>
 联合国法律事务副秘书长 MIGUEL DE SERPA SOARES：〝这个法院再一次证明，有能力按国际标准，起诉并审理最复杂案件。〞
 <br/>
 <br/>
 农谢和乔森潘在2014年因危害人类罪，被判
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_终身监禁.html" target="_blank">
  终身监禁
 </a>
 ，这次再被判处终身监禁。
 <br/>
 <br/>
 遭受〝红色高棉〞迫害的一些幸存者参加了法院判决，他们表示，两名被告罪有应得。
 <br/>
 <br/>
 〝红色高棉〞幸存者 PRAK DOEUN：〝我认同判决结果，他们应判终身监禁。〞
 <br/>
 <br/>
 〝红色高棉〞幸存者 SOK SABON：〝他们罪有应得，因为他们犯下大罪。我的家人都被杀了，我是（红色高棉）统治下，（家里）活下来的唯一孩子。〞
 <br/>
 <br/>
 1975年到1979年，柬埔寨共产党统治期间，得到中共支持，暴力〝清洗〞城市，展开全国性的大屠杀。柬埔寨约有200万人被迫害致死，相当于全国人口的四分之一。
 <br/>
 <br/>
 新唐人记者  毕心慈  宇微综合报导
</div>
