### 大篷车移民开枪袭警 墨西哥警察逮捕2人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 当局日前逮捕了两名洪都拉斯人，并指控他们向护送大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 的墨西哥联邦警察
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_射击.html" target="_blank">
  射击
 </a>
 。这是继上周末洪都拉斯移民与墨西哥边境人员发生冲突后，出现的又一攻击事件。
 <br/>
 <br/>
 据布莱巴特新闻（Breitbart News）10月30日报导，事件发生于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 恰帕斯州（Chiapas）伊格纳西奥．萨拉戈萨（Ignacio Zaragoza）市附近，当时，一些墨西哥警察正护送来自洪都拉斯的大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 穿过墨西哥的南部边境州。
 <br/>
 <br/>
 报导援引墨西哥内政部的消息称，22岁的男子杰森（Jerson）和17岁的男孩卡洛斯（Carlos）向护送大篷车的警察队伍开枪
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_射击.html" target="_blank">
  射击
 </a>
 。
 <br/>
 <br/>
 联邦警察趁两名枪手的手枪卡住之机逮捕了他们，期间无人受伤。警察收缴了一把格洛克（Glock）手枪及9发子弹。
 <br/>
 <br/>
 事件发生前，墨西哥刚刚向危地马拉境内的大篷车移民发出警告。这些移民正在准备燃烧弹并购得枪支、用以对抗墨西哥警察并突破边界。
 <br/>
 <br/>
 上周末，大篷车移民已与墨西哥当局爆发数次冲突，他们向墨西哥边防部队投掷石块，并使用各种临时武器对付阻拦他们入境的人员。
 <br/>
 <br/>
 目前已进入墨西哥境内的洪都拉斯移民约有7000左右，而且据墨西哥主流日报〝El Universal〞上周报导，正向美国进发的移民已高达14,000人，预计多数人不久就将抵达美国南部边界。
 <br/>
 <br/>
 更为严重的是，上周末，第二批试图前往美国的大篷车移民在墨西哥和危地马拉边界遭墨西哥防暴警察拦截后，双方爆发冲突，一些难民强行突破边境大门，导致1名移民死亡，6名警察受伤。
 <br/>
 <br/>
 鉴于当前情况，美国国防部10月29日宣布，将派遣大约5200名美军把守美墨边境。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在推特上宣布，洪都拉斯移民的行径是在〝入侵〞美国，并警告他们，〝我们的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军队.html" target="_blank">
  军队
 </a>
 正等着你们〞。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
