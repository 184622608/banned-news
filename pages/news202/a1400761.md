### 经商待客〝有温度〞！品味日本阿美横街
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】在东京的闹市区上野有一条商店街，它的历史浓缩了日本在二战结束后，市场从物资匮乏到一片欣欣向荣的发展过程，而如今在国际化不断推进的过程中，它又有展现出活力四射、充满人情味的特点，吸引着来自世界各地的观光客。这条街就是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿美横街.html" target="_blank">
  阿美横街
 </a>
 。
 <br/>
 <br/>
 记者：出了上野车站中央口，就能看到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿美横街.html" target="_blank">
  阿美横街
 </a>
 的大看版，从这里一直走下去就可以到山手线的御徒町车站，虽然只有短短500米的商店街，但却集中了涵盖衣食住行各方面的商品，令人目不暇接。
 <br/>
 <br/>
 进入21世纪以来这条街平均一天要接待10万人以上，每逢节假日，平均每天有50万人来访。从早到晚，游人熙熙攘攘，络绎不绝。
 <br/>
 <br/>
 商铺老板国立真康：〝在平日这里外国游客非常多，东南亚来的游客会到7成，其中中国游客大约占到5成。一方面是价格便宜，再有就是有的国家关税比较高，所以在日本买礼物的人很多。〞
 <br/>
 <br/>
 从最早的集市开始起步，阿美横街不断地吸收多种店铺，发展到现在拥有400多家店铺的商业街，阿美横街是东京都内最大的商店街之一。
 <br/>
 优惠的价格和丰富的种类使这里人气旺盛，多年来一直生意兴隆。
 <br/>
 <br/>
 泰国游客：〝在礼品店我买到了礼物，送给我女儿。〞
 <br/>
 <br/>
 印度尼西亚游客：〝商品价格很便宜，比起新宿涉谷这里便宜很多。〞
 <br/>
 <br/>
 中国上海游客：〝比较亲民的价格，不是很贵的，但东西质量好，东西是有所值。〞
 <br/>
 <br/>
 近年来随着电商行业的急速发展，很多传统的商店街渐渐衰落了，零售和批发的商业环境越来越严峻。阿美横街的经营者们仍以充满人情味、保持有温度的商店街为发展目标，注重和客户面对面交流与接触。
 <br/>
 <br/>
 新唐人日本记者站报导
</div>
