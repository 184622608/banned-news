### 牛顿预言曝光：2060年世界重置成〝神的王国〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牛顿.html" target="_blank">
  牛顿
 </a>
 是英国著名物理学家，以阐述三大运动定律而闻名于世。牛顿等很多杰出的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科学家.html" target="_blank">
  科学家
 </a>
 都论述过神主宰着这个世界。牛顿生前曾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_预言.html" target="_blank">
  预言
 </a>
 ，世界将在2060年〝重置〞，地球将再次成为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神的王国.html" target="_blank">
  神的王国
 </a>
 〞。
 <br/>
 <br/>
 艾萨克．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牛顿.html" target="_blank">
  牛顿
 </a>
 爵士（Sir Isaac Newton）经典名言为〝如果我看得比较远，那是因为我站在巨人的肩膀上〞。
 <br/>
 <br/>
 近日，《纽约邮报》报导说，奥地利天文学家弗洛里安．弗赖斯特（Florian Freistetter）整理牛顿生前的资料时，发现了牛顿不只阐述万有引力与3大运动定律，甚至还用"Jehovah Sanctus Unus"的别名撰写文稿。
 <br/>
 <br/>
 文稿中，牛顿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_预言.html" target="_blank">
  预言
 </a>
 世界将在2060年〝重置〞，地球将再次成为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神的王国.html" target="_blank">
  神的王国
 </a>
 〞。
 <br/>
 <br/>
 弗赖斯特说，牛顿生前花费很多时间研究宗教文本，并尝试为过去的事件建立年表，以便将它们分门别类，同时牛顿确信未来的事件，已经被上帝所决定。
 <br/>
 <br/>
 牛顿从《圣经》中撷取了一些〝预言〞的时间点，对牛顿而言，2060年将是个崭新的开始，可能伴随着战争和天灾，但最终还是一个新神圣时代的开端。
 <br/>
 <br/>
 牛顿是十七至十八世纪英国的物理学家、数学家、天文学家，也是近代
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科学家.html" target="_blank">
  科学家
 </a>
 中最杰出的代表人物，他是世界上最伟大的科学家之一。
 <br/>
 <br/>
 他从研究自然的奥秘开始，但最终进入了宗教殿堂，向世人证明了高级生命所创造的宇宙是何等奇妙和伟大。他18岁进康桥大学学习，27岁即出任康桥大学教授，发明微积分与积分法两项定理。他在天文、地理、数学、神学等方面都取得了辉煌成就。
 <br/>
 <br/>
 牛顿毕生的主要精力用于对精神世界的探索，视科学为业余之事。他在谈到自己的科学成就时说，他不过是在〝追随神的思想〞，〝照神的思想去思想而已〞。
 <br/>
 <br/>
 牛顿临终前，面对仰慕他智慧和称颂他伟大科学成就的人，却谦虚地说：〝我的工作和神的伟大创造相比，我只是一个在海边拾取小石和贝壳的小孩子。真理浩瀚如海洋，远非我们所能尽窥。〞
 <br/>
 <br/>
 弗赖斯特说，牛顿生前广泛阅读基督宗教的《圣经》，因此他以《圣经》为基础撰写许多著作，对于2060年的预言，牛顿写下〝它可能晚点结束，但我看没有理由太快结束（It may end later, but I see no reason for its ending sooner.）〞。
 <br/>
 <br/>
 除牛顿之外，爱因斯坦、巴斯德、伽利略等很多杰出的科学家都论述了神主宰着世界。
 <br/>
 <br/>
 爱因斯坦说：〝人类已知的是有限的一个圆，未知的是圆外的世界，是无限的。〞人的大脑70%以上被神封闭，思维结构和智慧有限。
 <br/>
 <br/>
 天文学家哥白尼说：〝当人类看见上帝管理下宇宙的庄严秩序时，必定会感受到一种力量，催迫自己趋向规律的生活，去履行各种道德，并可以从万物中认出造物主上帝确是真善之源。〞
 <br/>
 <br/>
 联合国曾调查了300位最著名的科学家信神者占92%，他们认为世界是神的杰作。
 <br/>
 <br/>
 牛顿、爱因斯坦、伏特、欧姆、安培、爱迪生、伦琴……原子能专家普莱特、康普敦、费米等都走入对神的信仰这条路。
 <br/>
 <br/>
 相关神学理论认为，人在迷中，神安排了人类社会历史的发展。神赋予先知们如姜子牙、诸葛亮、李淳风、袁天罡（《推背图》）、刘伯温（《烧饼歌》）等开启功能观察天象，预知未来社会的发展。
 <br/>
 <br/>
 古人相信神，神会通过思维感测（意识）点化人的灵魂（信息）创造奇迹，黄帝的人体经络，著名医学家扁鹊、华佗、孙思邈、李时珍等用功能观察另外空间人体得病的根本原因，并且手到病除。
 <br/>
 <br/>
 在古代神州的人们都信神，佛光普照，精神意识起着主导作用，张衡的地动仪等〝四大发明〞，还有众多奇迹都是神的安排和点化……
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
