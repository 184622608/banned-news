### 蓬佩奥东亚行  接近促成第二次川金会
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】
 <b>
  新闻周刊
 </b>
 （649）10月6号到8号，美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 连访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 ，并与除了习近平之外的三国元首会面。其中，蓬佩奥与金正恩再次会面引人关注，美国总统川普已经确认，美朝双方正在筹备举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第二次川金会.html" target="_blank">
  第二次川金会
 </a>
 。
 <br/>
 <br/>
 美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 ，8号结束了东亚四国之行。他在社交媒体贴出一组图片，显示本次访问取得的成果。
 <br/>
 <br/>
 蓬佩奥此行的重点是在平壤与金正恩会面。不过他在此之前首先造访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 ，与首相安倍晋三会晤。
 <br/>
 <br/>
 日本首相 安倍晋三：〝我感谢蓬佩奥国务卿在访问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 之前先访问日本。〞
 <br/>
 <br/>
 离开东京后，蓬佩奥10月7号在平壤与金正恩会晤，双方进行了长达5个半小时的会谈。
 <br/>
 <br/>
 在出行之前，蓬佩奥已经明确表示，不论与朝鲜谈话结果如何，美国目前不会放松对朝鲜的经济制裁。
 <br/>
 <br/>
 美国国务卿 Michael Pompeo：〝经济制裁将继续。这一核心条件，给予我们实现无核化的能力，并没有改变。〞
 <br/>
 <br/>
 最终，美朝双方都对会谈结果表示满意。据朝鲜官方电视台报导，金正恩表示，与蓬佩奥的对话〝有成效，很愉快〞。双方立场获得充分理解，并交换了意见。
 <br/>
 <br/>
 美国总统川普也在周日发推文说，〝蓬佩奥与金正恩在平壤会谈愉快。新加坡峰会协议取得进展！我期待不久的将来，再次见到金正恩〞。
 <br/>
 <br/>
 蓬佩奥则表示，与金正恩的谈判取得实质进展。金正恩准备允许国际核查人员前往朝鲜一个导弹引擎测试设施，以及丰溪里核试验设施。这是初期实施无核化的主要关键点之一。
 <br/>
 <br/>
 同时，两国元首也都亲口确认，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第二次川金会.html" target="_blank">
  第二次川金会
 </a>
 正在筹备当中。
 <br/>
 <br/>
 美国总统 川普：〝他们正在安排会议，我们将宣布（同金正恩的会谈）。〞
 <br/>
 <br/>
 川普周二对记者表示，如果第二次与金正恩会晤，地点可能不会在新加坡，也不排除在美国或朝鲜会晤的可能，而会晤的时间至少会等到中期选举之后。
 <br/>
 <br/>
 蓬佩奥离开朝鲜后，前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 向文在寅报告这次朝鲜之行的进展。文在寅表达了对第二次川金会的祝愿。
 <br/>
 <br/>
 韩国总统文在寅：〝我深切希望川金会尽快召开，在无核化和和平进程方面，取得不可逆转的决定性进展。〞
 <br/>
 <br/>
 蓬佩奥10月8号来到东亚之行的最后一站北京，与中共政治局委员杨洁篪和外交部长王毅会晤。
 <br/>
 <br/>
 谈到美中之间近来的紧张关系，王毅将所有矛盾归咎美国，蓬佩奥则直言两国存在根本分歧。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 外长 王毅：〝我们要求美方停止错误做法。〞
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝你提到的这些议题我们有根本的分歧，我们密切关注中国的行动。〞
 <br/>
 <br/>
 美中战略安全对话原订十月在北京举行，中共藉口关税等冲突临时取消，蓬佩奥对此表示遗憾。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝我很遗憾，我们两国之间的战略对话，你们全都选择不进行。这原本对我们来说是个重要机会，可以讨论更广泛的谈判条件、重要议题，为两国人民提供机会。〞
 <br/>
 <br/>
 此次东亚之行，习近平是蓬佩奥唯一没有会晤的元首。分析认为，北京在中期选举前，不会和美方进行实质性的谈判。
 <br/>
 <br/>
 美国财长姆努钦日前在国际货币基金组织年会上透露，美方尚未决定川普是否将在11月底的G20峰会上会晤习近平。
 <br/>
 <br/>
 撰稿：林宇 剪辑：明玉
