### 约旦死海暴洪 学生团遭冲走 至少18死失踪人数不明
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_约旦.html" target="_blank">
  约旦
 </a>
 一所学校师生及家长25日赴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_死海.html" target="_blank">
  死海
 </a>
 附近景点郊游，但当地降倾盆大雨，引发暴洪，师生们试图逃命，但洪水来得太快太大，造成至少18人丧生，而很多罹难者都是不到14岁的孩子。而因洪灾横贯多处渡假区，当局至今难于估算受灾或罹难者总数。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_约旦.html" target="_blank">
  约旦
 </a>
 观光胜地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_死海.html" target="_blank">
  死海
 </a>
 一带，25日遭遇夏日旱季以来第一场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雨.html" target="_blank">
  暴风雨
 </a>
 ，还伴随10级阵风，不仅在约旦北部造成多处洪水灾情，死海沿岸的马恩温泉区更突然山洪爆发。一组校外教学团的中学生〝整团被冲走〞。
 <br/>
 <br/>
 路透社报导，一名获救者的父亲阿布尤塞夫（Abu Yousef）表示，学校巴士载着37名学童和7位老师去度假区郊游。在一狭窄溪流附近，他们遇上暴雨，瞬间就淹没了整个区域。
 <br/>
 <br/>
 他说，学生们逃命冲向校车，但车门关着。而老师试图救学生，但洪水来得太快太大，救不了。孩子们被大水冲往死海。
 <br/>
 <br/>
 从视频中也可看见，在桥梁中间围观者将一名成人吊挂下桥墩，救起一名孩童。
 <br/>
 <br/>
 <b>
  相关视频：
 </b>
 <center>
 </center>
 <br/>
 警政首长法瑞德（Farid al Sharaa）对国营电视台表示，已出动警队直升机和数百名士兵搜寻，有34人获救，但其中有些人伤势严重。
 <br/>
 <br/>
 救难人员说，很多罹难者都是不到14岁的孩子，一些家长随同到这个著名景点野餐，也受了伤或失去性命。但救难人员未提供具体伤亡数字。
 <br/>
 <br/>
 因洪灾横贯多处渡假区，目前救出的受灾或罹难者总数，早已超过中学团的44人。目前约旦政府出动2,000军队全力搜救，以色列国防军也派出了救灾特种直升机部队越境救援；但直到26日清晨，约旦政府仍未确定失踪人数，仅知道至少18名师生死亡。
 <br/>
 <br/>
 国王阿布杜拉二世（Abdullah Ⅱ）已取消访问巴林（Bahrain）行程，密切关注救援行动。这是约旦近年来数一数二严重的灾难。
