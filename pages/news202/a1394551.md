### 朝无核化会谈获大进展 金正恩允视察团入朝验证
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 结束第四次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访朝.html" target="_blank">
  访朝
 </a>
 后表示，与朝鲜的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无核化.html" target="_blank">
  无核化
 </a>
 会谈取得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重大进展.html" target="_blank">
  重大进展
 </a>
 ，朝鲜已经答应让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际核视察团.html" target="_blank">
  国际核视察团
 </a>
 入朝进行核验证。
 <br/>
 <br/>
 美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 （Mike Pompeo）在周一（8日）访问中国前的记者会上表示，朝鲜领导人金正恩已经允许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际核视察团.html" target="_blank">
  国际核视察团
 </a>
 访问朝鲜，随着礼宾、运输等后勤工作达成共识之后，视察团将访问丰溪里核试验场和另一个导弹发动机试验场。
 <br/>
 <br/>
 蓬佩奥说，〝金正恩说他们已经准备好了〞，〝我们取得了重大的进展，以后还会持续获得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重大进展.html" target="_blank">
  重大进展
 </a>
 ，取得比历届政府更多的进展。〞
 <br/>
 <br/>
 而在此之前，朝鲜一直拒绝国际核视察团
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访朝.html" target="_blank">
  访朝
 </a>
 进行核验证。
 <br/>
 <br/>
 在记者会上，蓬佩奥还透露，美朝双方就第二次川金峰会达成协议。目前，两位领导人都认为可以往下一步前进了，〝尽管现在还没到那个阶段，但将来会的。〞
 <br/>
 <br/>
 金正恩上月在一封信函中，曾提议同美国总统川普（特朗普）举行第二次峰会。蓬佩奥表示，接下来美朝双方将会有更频繁的高级别工作会议商讨。
 <br/>
 <br/>
 上周末，蓬佩奥接连访问了日本、韩国、朝鲜三国领导人，周一抵达北京，会晤中共外交部长王毅和中央政治局委员杨洁篪。
 <br/>
 <br/>
 在朝鲜与金正恩会面后，蓬佩奥特别透过推特表示感谢。他说：〝在平壤与金正恩领袖进行的会晤很顺利。我们在新加坡峰会达成的协议基础上，继续取得了进展。谢谢（朝鲜）对我及我团队的款待。〞
