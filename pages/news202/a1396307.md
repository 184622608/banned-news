### 二次〝川金会〞或推迟 韩媒：金正恩年内难访首尔
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】由于美朝高峰会可能推迟到2019年初举行，韩国媒体认为，朝鲜领导人金正恩年内回访首尔的计划也将可能无法顺利成行。
 <br/>
 <br/>
 美国总统川普（特朗普）与金正恩于6月在新加坡举行会谈后，日前美朝双方已就第二次〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川金会.html" target="_blank">
  川金会
 </a>
 〞进行磋商。一名美国资深官员19日告诉路透社记者，第二次〝川金会〞可能在2019年年初举行。
 <br/>
 <br/>
 韩国〝东亚日报〞22日在头版头条报导，青瓦台一直预测第2次〝川金会〞将于11月底至12月初举行。
 <br/>
 <br/>
 报导说，青瓦台认为，9月在平壤举行的韩朝峰会达成协议，金正恩回访首尔的时间是12月，也就是在〝川金会〞后。韩国总统文在寅当时在韩朝峰会联合记者会上表示，金正恩决定在近期访问首尔。他也解释如果没有特别的情况，近期指的就是在今年内。
 <br/>
 <br/>
 青瓦台绘制的蓝图是朝鲜与美国在第二次〝川金会〞时，就终战宣言等非核化相应措施谈判后，金正恩于12月访问首尔并发表〝首尔宣言〞，内容包括就缓和韩朝军事紧张局面和经济合作等达成协议。
 <br/>
 <br/>
 报导分析，因为第二次〝川金会〞的推迟，终战宣言和文在寅的非核化路线图也会受到影响，金正恩年内回访首尔的计划也可能无法顺利成行。
 <br/>
 <br/>
 不过，青瓦台目前仍然坚持，即使川金会推迟也要按原计划推进金正恩回访首尔，也就是说以文在寅访问平壤为起点，透过发展韩朝关系来促进非核化的〝朝鲜半岛主人论〞已正式启动，因此将利用金正恩回访首尔来推动非核化协议。
 <br/>
 <br/>
 不过，如果成为非核化分水岭的第二次川金会推迟到明年，包括金正恩回访首尔在内，文在寅的非核化路线图也势必要进行部分修改。
 <br/>
 <br/>
 报导说，韩国外交消息人士称，只有在第二次川金会达成协议，才能找到头绪缓和针对韩朝合作的制裁，如果朝鲜与美国继续较劲，金正恩回访首尔的意义就会减半。
 <br/>
 <br/>
 美国国务卿蓬佩奥在10月初曾第四次出访朝鲜，并与金正恩就无核化议题进行会谈。
 <br/>
 <br/>
 尽管金正恩承诺说，只要美国和朝鲜准备就绪，朝鲜将允许国际核检人员进入该国的丰溪里核试验场进行检验，但金正恩却拒绝提供核设施清单给蓬佩奥，除非美方签署韩战终战宣言。
 <br/>
 <br/>
 日本《读卖新闻》引述消息称，当蓬佩奥要求金正恩提供至少部分核设施清单时，据称金正恩回复说，在美国和朝鲜缺乏互信的情况下，就算美国拿到这份清单，也不会相信清单内容。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
