### 俄乌爆海上军事冲突 北约将举行紧急会议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】乌克兰海军与俄罗斯海军日前在与黑海相连的亚速海爆发小规模军事冲突，有大约6人受伤，乌方的3艘舰艇被俄方扣留。为此，乌克兰宣布全国进入战时戒严状态。周一，北大西洋公约组织（NATO）决定与乌克兰召开紧急会议，商讨如何解决目前乌、俄海军对峙的状况。
 <br/>
 <br/>
 当地时间11月26日上午，北约秘书长斯托尔滕贝格（Jens Stoltenberg）与乌克兰总统波洛申科（Petro Poroshenko）通过电话后宣布，北约将与乌克兰在比利时布鲁塞尔召开紧急会议，讨论如何应对当前发生的俄罗斯与乌克兰海军对峙的问题。
 <br/>
 <br/>
 与此同时，联合国安理会也为此事召开了紧急会议。
 <br/>
 <br/>
 据公开的资讯，乌克兰的3艘船隻25日试图穿过连接黑海和亚速海的克赤海峡时，遭俄罗斯海军以擅闯领海为由，动用武力阻挡乌克兰船隻，双方对峙数小时后，乌克兰的3艘船舰全部被俄罗斯方面扣留。
 <br/>
 <br/>
 乌克兰国防部随后发表声明称，在这场对峙中，乌克兰有2艘小型炮舰和1艘拖船受到俄罗斯特种部队船隻的攻击。
 <br/>
 <br/>
 俄罗斯联邦安全局（FSB）26日则证实俄海军扣留了乌克兰的3艘军舰，并承认在逼迫这3艘船停止的过程中，俄军使用了武器。
 <br/>
 <br/>
 乌克兰与俄罗斯的黑海冲突事件发生后，乌克兰总统波罗申科提议在国内实施为期60天的战时戒严状态，并得到了乌克兰国家安全和国防委员会（NSDC）的支持。目前这项动议将进入最高拉达（议会），以获得最终批准。
 <br/>
 <br/>
 波罗申科对外发表声明时表示，乌克兰实施戒严令并不意味着基辅方面将采取军事进攻行动，而且基辅的戒严决定也不会侵犯乌克兰公民的权利和自由，乌克兰只会采取防御性行动保卫领土和公民的安全。目前基辅已经要求北约和欧盟〝协调行动〞，以确保乌克兰得到保护。
 <br/>
 <br/>
 波罗申科表示，他周一将就亚速海局势和基辅计划在该国实施战时状态一事，与北约秘书长斯托尔滕贝格（Jens Stoltenberg）举行会谈。同时，他还计划当天与乌克兰盟国的领导人举行会谈。
 <br/>
 <br/>
 此前，乌克兰宪法法院11月22日刚做出裁决，确认在该国的宪法中写入〝乌克兰加入北约和欧盟〞的相关条款合法。
 <br/>
 <br/>
 据了解，在这次乌、俄黑海冲突发生前，乌克兰国家边防局曾于11月15日对外证实，乌方曾在克里米亚北部的亚速海水域扣留了15艘非法进入克里米亚港口的〝外国船舶〞，这其中也包括不少来自俄罗斯的船隻。
 <br/>
 <br/>
 针对最近乌克兰与俄罗斯之间发生的冲突与对峙，北约已表示〝密切关注亚速海和刻赤海峡的情况〞，欧盟则呼吁俄罗斯〝克制〞并采取降低冲突的行动。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
