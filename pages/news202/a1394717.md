### 时事拼盘：乌克兰弹药库爆炸 金正恩邀教皇访朝
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】下面请看一组新闻简讯
 <br/>
 <br/>
 周二凌晨，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 国防部弹药库发生火灾，引发连环爆炸，直到白天仍可听到爆炸声。附近约1万2千人被疏散，当地交通和能源供应中断，暂无人员伤亡。乌克兰国防部表示，多处发生爆炸，可能是人为破坏。国家安全部将进行调查。
 <br/>
 <br/>
 韩国总统府周二表示，朝鲜领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 希望邀请教皇方济各访问平壤,声称为促进朝鲜半岛的和平。朝鲜与梵蒂冈没有建交。韩国总统文在寅将在下周三和周四访问梵蒂冈时向教皇转达邀请。
 <br/>
 <br/>
 周二，澳大利亚政府提出新计划，将强制新移民去大城市以外居住最长5年。这只涉及没有和一定地域绑缚在一起的签证。雇主担保签证和家庭团聚签证不受影响。
 <br/>
 <br/>
 澳大利亚城市基础设施和人口部长塔吉：〝墨尔本，悉尼和昆士兰州东南部的大城市减少一些压力。〞
 <br/>
 <br/>
 周二，德国救援人员教印尼地震和海啸幸存者如何生产饮用水。一些人喝了不干净的水后发生腹泻。官方最新数字显示，9月28日重创苏拉威西岛西海岸的7.5级地震和随之而来的海啸，已造成2,010人死亡，预计遇难人数还将上升。
</div>
