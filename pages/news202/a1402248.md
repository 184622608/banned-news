### 伦敦博物馆妙用美食 搭建奇彩姜饼城市
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】圣诞节将至，各式各样的姜饼屋也纷纷亮相。英国伦敦的维多利亚与艾伯特博物馆独出心裁，用姜饼搭建了几座丰富多彩的城市。一起去看一看吧。
 <br/>
 <br/>
 城市规划听起来好像很深奥，不过英国的建筑师发挥创意，用糖和香料构建了多个可以食用的未来城市，告诉您城市规划可以有趣又美妙哦。
 <br/>
 <br/>
 这些姜饼城市真可谓应有尽有。各式各样的房屋建筑，道路交通，还有娱乐设施等等，真是让人目不暇接。
 <br/>
 <br/>
 其中超过60栋建筑都是通过烤箱制作出来的。
 <br/>
 <br/>
 这里有一座86cm高的塔楼，恐怕是这个城市中的最高建筑了吧。
 <br/>
 <br/>
 这个缆车是由甘草制造而成的。而人行道，自行车道则完全糖做的。
 <br/>
 <br/>
 还有云霄飞车呢。
 <br/>
 <br/>
 虽然设计和制作这些城市非常不容易，但是建筑师们还是乐在其中。并以此向公众介绍城市规划及建筑行业知识。
 <br/>
 <br/>
 APT建筑师罗伯特：〝如果你发现自己在姜饼(创造)中找到乐趣，然后将这种感觉转移到一些更实际些的材料上。实际上，它们不是差的太远。做设计是很有趣的。〞
 <br/>
 <br/>
 今年展出的姜饼城市，还有一个特别之处，其中一个展馆里的建筑是由机器人制作出来了。听起来是不是很令人惊讶。这可是世界首例哦。
 <br/>
 <br/>
 怎么样，您是否也在考虑今年自己动手用姜饼做点特别的呢？
 <br/>
 <br/>
 新唐人记者周琪综合报导
</div>
