### 俄媒体人：普京私下一直称中共为主要威胁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】贸易战压力，在一些领域拉近了中俄之间的关系。但双方持续数十年的利益冲突依然难解。俄资深媒体人日前透露，十几年来，俄罗斯总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 都把中共视为主要威胁。
 <br/>
 <br/>
 《美国之音》10月17日报导，虽然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 在公开场合总是称赞俄中关系，但俄罗斯著名媒体人维尼吉克托夫透露，普京其实一直保持着对中共的戒心。
 <br/>
 <br/>
 维尼吉克托夫是俄罗斯主要媒体〝莫斯科回声〞广播电台的主编。他最近在与前德国联邦情报局局长汗宁的一次访谈中说，这些年来，他经常问普京，俄罗斯的主要威胁是谁。
 <br/>
 <br/>
 在2000年代初期时，普京的回答是：伊斯兰恐怖主义、中共扩张和西方对俄罗斯的遏制。10多年后的今天，普京的回答是：西方、伊斯兰恐怖主义和中共扩张。
 <br/>
 <br/>
 维尼吉克托夫称，在普京的心目中，这三者依然是俄罗斯主要威胁，只是排名次序有变化。
 <br/>
 <br/>
 三者排名中，西方从第三位移到了第一位。这似乎与美国川普（特朗普）政府升级美俄对抗有关。而川普对中俄同时采取行动，也是促使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中俄关系.html" target="_blank">
  中俄关系
 </a>
 有所靠近的主要原因。
 <br/>
 <br/>
 不过，中俄数十年交恶，双方戒心积累已久，并在越南、蒙古等问题上存在激烈利益冲突。中共近年来加速对外扩张，在中东和非洲等地区排挤俄罗斯原有势力，也引起俄罗斯警惕和抵制。
 <br/>
 <br/>
 此外，《美国之音》还引述分析指，普京的主要幕僚和亲信，包括国家安全委员会秘书长帕特鲁舍夫，前国防部长伊万诺夫，联邦安全局局长博尔特尼科夫，对外情报局局长纳雷什等人，几乎都是普京的同龄人。
 <br/>
 <br/>
 这些人年轻时都经历过中苏交恶和珍宝岛冲突，并在克格勃环境熏陶中成长，而中共长期以来都是苏联和克格勃的主要针对目标。
 <br/>
 <br/>
 在上述访谈节目中，汗宁对维尼吉克托夫表示，中共打算2025年称霸世界，对全世界包括俄罗斯都构成威胁。在这一点上，俄罗斯与西方有共同利益，应该携手应对中共在一些领域的挑战。
 <br/>
 <br/>
 而港媒分析，中美俄之间任何两方都很难结盟，三者之间只能是〝三角关系〞。
 <br/>
 <br/>
 俄罗斯〝顶级战争〞军事观察网站10月16日发表评论文章，题为《中国是超级大国吗？有什么证据？》。文章指，中国只是被中共塑造成了一个〝世界工厂〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 很快将所谓的〝厉害了我的国〞打回原形。
 <br/>
 <br/>
 文章还说，美国与西方各国之间的关系是友好关系，而俄中相互接近是在与西方对抗的背景下，被外部压力〝推〞着勉强站在一起，并不是真的友好。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
