### 港媒：任正非全家持有加国护照 只有任是例外
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 集团董事长任正非之女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 ，因涉嫌欺诈日前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 被捕。中共外交部等多个部门正在展开〝营救〞孟晚舟的活动，不过有媒体披露孟晚舟持有加拿大护照，北京并无权介入。
 <br/>
 <br/>
 在12月6日的中共外交部记者会上，有记者提问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 入境
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 时是否持中国护照，发言人当时回答称〝无法在此讨论关于此案的细节〞，让在场记者哗然。第二天的外交部例行记者会上，又有记者作出关于孟晚舟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国籍.html" target="_blank">
  国籍
 </a>
 的询问，发言人声称根据中共的《国籍法》，孟晚舟是中国公民。
 <br/>
 <br/>
 中国驻加大使馆前日也发表声明称孟是中国公民，而且无违反美加法律的行为，但该说法稍后已被官媒删去。
 <br/>
 <br/>
 另据香港《苹果日报》引述消息人士称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创办人任正非全家只有任自己没持加拿大护照，而其他成员包括孟晚舟在内，全部都拥有加拿大护照。
 <br/>
 <br/>
 据悉，加拿大容许双重国籍，但中国不承认双重国籍。中共的《国籍法》规定如果中国公民加入或取得外国国籍，其中国国籍即自动丧失。
 <br/>
 <br/>
 报导称，如果孟晚舟持有加拿大护照的话，中共大使馆和外交部对〝放人〞的呼吁就十分的尴尬。
 <br/>
 <br/>
 孟晚舟的法庭保释听证会在12月7日举行。据媒体最新报导，孟的律师马丁（David Martin）在作保释辩护时提到，孟晚舟目前有两本护照，其中香港护照在被捕当天已被没收，另一本中国护照也已在日前寄到加拿大。
 <br/>
 <br/>
 为证明孟晚舟没有潜逃的动机，马丁还说孟和先生在2009年在温哥华置产，她的儿子在2009年到2012年期间也曾在加拿大留学。
 <br/>
 <br/>
 而加国的检方则提到，孟晚舟在2014年到2016年期间经常进出美国，但是自从今年4月获悉美国政府对华为展开调查后，则避免前往美国。孟最后一次进出美国是2017年3月。
 <br/>
 <br/>
 不过上述报导并没有提及孟是否有加拿大护照。
 <br/>
 <br/>
 孟晚舟是华为的首席财务官，也是其董事会副主席及创始人任正非的长女。华为公司被外界认为与中共政权的关系密切，近几年引起其它国家情报界的关注，并担忧中共政权是否通过华为设备进行间谍活动。
 <br/>
 <br/>
 记者陈远辉报导
</div>
