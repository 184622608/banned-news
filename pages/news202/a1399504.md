### 中兴助委国监控国民 美议员痛斥中共〝输出威权主义〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】有美国媒体爆料称，中国电讯企业
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴.html" target="_blank">
  中兴
 </a>
 公司协助
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 政府建立了具有监控功能的新身份证——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_祖国卡.html" target="_blank">
  祖国卡
 </a>
 。有美国参议员认为，这证明中共正在向世界输出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威权主义.html" target="_blank">
  威权主义
 </a>
 〞。此前不久，也曾有国际人权非政府组织发布报告称，中共正向全球输出网络审查和监控模式，推助了〝数码威权主义〞的崛起，导致全球网络自由度持续下降。
 <br/>
 <br/>
 路透社14日发表的一篇调查报导称，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴.html" target="_blank">
  中兴
 </a>
 通讯正协助
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 政府建立一个〝通过身份证监控公民行为〞的系统，在委国推出一种名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_祖国卡.html" target="_blank">
  祖国卡
 </a>
 〞的新型智能身份证的项目里，中兴通讯扮演了〝核心角色〞。
 <br/>
 <br/>
 据报导，委内瑞拉前任总统查韦斯在2008年派代表团访问深圳的中兴总部时，就已经了解到中共政府如何利用智能卡来开发一套追踪监控国民行为的系统，以及使用庞大的数据库中收集的讯息来实施政府对公民的个人财产状况乃至病例等方方面面的监控。
 <br/>
 <br/>
 现在，委国真的推出了新型智能身份证〝祖国卡〞以及某种数据社保卡来加强对国民的监控。委政府正不断强化把这种智能卡与国民依赖的食品补贴、健康和其他社会计划捆绑一起。而据已经使用〝祖国卡〞的人透露，管理员在发卡前会询问有关收入、政治活动和社交媒体资料等问题。
 <br/>
 <br/>
 报导表示，委内瑞拉的〝祖国卡〞令一些公民和人权组织感到担忧，他们认为该卡可能是马杜罗政府监督民众和为其支持者分配稀缺资源的工具。
 <br/>
 <br/>
 报道称今年5月马杜罗击败反对党获连任之前，对持有〝祖国卡〞的一些选民的投票作出引导和操控。
 <br/>
 <br/>
 路透社的报导还特别援引美国参议员鲁比奥的回复表示，中共正在出口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威权主义.html" target="_blank">
  威权主义
 </a>
 ，马杜罗政权愈来愈依赖中兴通讯，恰恰是中国国企对美国国家安全利益构成威胁的最新例证。
 <br/>
 <br/>
 针对上述消息，中国互联网专家郝培强在接受《自由亚洲电台》采访时表示：事件的核心不在于技术输出，而在于一些独裁或独裁边缘的政府，正纷纷效仿中国政府管控公民的〝丰富经验〞。
 <br/>
 <br/>
 台湾国立大学政治学教授李酉潭则指出：中南美洲的诸国中，除古巴是明确的专制政体外，类似委内瑞拉这样的〝半民主自由〞的国家出现政治倒退，与中共输出其专制的集权模式有关，而华为、中兴等企业在这当中则扮演了〝不光彩的角色〞。
 <br/>
 <br/>
 事实上，早在今年11月1日，国际非政府组织〝自由之家〞就曾发表报告称，中共正向全球输出网络审查和监控模式，而〝数码威权主义〞的迅速崛起，导致全球的网络自由度，连续第八年出现下降。
 <br/>
 <br/>
 〝自由之家〞在这份报告中表示，该组织研究了全球65个国家的网络自由状况后，发现其中36个国家的代表参加过中共举办的培训和研讨会，内容则与强化对新媒体和信息的管控有关。
 <br/>
 <br/>
 报告称，中企还向一些人权记录较差的政府提供电信硬件和面部识别技术以及数据分析工具，帮助这些政府加强对国民的控制，〝数码威权主义〞随之蔓延。
 <br/>
 <br/>
 当时，安徽省检察院前检察官沈良庆在接受新唐人电视台记者采访时分析称：自由民主国家越多，中共的极权专制越受到挑战，因此中共才输出自己的价值观，试图让恐怖蔓延世界。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
