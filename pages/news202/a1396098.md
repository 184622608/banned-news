### 英男埃及猝死 返国验尸心肾不翼而飞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国男子.html" target="_blank">
  英国男子
 </a>
 汉弗莱斯上个月和家人到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃及.html" target="_blank">
  埃及
 </a>
 旅游时不幸病发身亡，埃及医院以警方调查为由强制验尸，检验报告称遗体未见异常。家属将遗体运回英国后再度验尸却发现其肾脏和心脏不翼而飞。
 <br/>
 <br/>
 62岁的汉弗莱斯（David Humphries）与家人在9月前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃及.html" target="_blank">
  埃及
 </a>
 旅游，因胸口疼痛，被送往当地医院诊治。医师开了一些抗生素药物，就让他回家了。但汉弗莱斯服药几天后，病情并未见好转，9月18日和外孙玩耍时，一头栽进游泳池中。家人紧急将其送医救治，但已仍回天乏术。
 <br/>
 <br/>
 他的女儿古道尔（Anita Goodall）怀疑，是医院治疗不当才害父亲魂断异乡。因此拒绝在医院进行验尸。但医院以警方调查为由，强行将他的遗体进行解剖检验。归还遗体时医院还强调〝状态不适合观看〞。
 <br/>
 <br/>
 解剖报告显示，汉弗莱斯的遗体未见异常。家属对此结果存有怀疑，10月1日回国后，古道尔决定把父亲的遗体交给英国验尸官再次检查，结果被告知遗体中少了肾脏和心脏。
 <br/>
 <br/>
 这让家属非常震惊，她们表示，埃及人相信，如果没有心脏人就不能安息，无法理解他们为什么会这样对待我们的家人。家属也猜测，心脏很有可能是汉弗莱斯死亡的原因，但现在永远可能都不会知道真相了。
 <br/>
