### 【禁闻】遭电视认罪 英男要求吊销央视营业执照
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】
 <h4>
  遭
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电视认罪.html" target="_blank">
   电视认罪
  </a>
  英男要求吊销
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
   央视
  </a>
  营业执照
 </h4>
 <br/>
 英国一名男子11月23号向英国监管机构投诉，要求吊销中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
  央视
 </a>
 及其国际分支机构环球电视网（CGTN）在英国的营业执照。
 <br/>
 <br/>
 据〝美联社〞报导，这名叫彼得・汉弗莱的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国公民.html" target="_blank">
  英国公民
 </a>
 和中国籍妻子经营着一家调查公司，他们早前被英国制药公司葛兰素史克聘用，在中国调查该公司贿赂行为时，被中共指控非法获取中国公民个人信息而入狱两年。随后，两人被迫在央视上公开承认有罪。
 <br/>
 <br/>
 据报导，中共官方的环球电视网在英国拥有免费和付费频道，并正在扩大欧洲业务，新的制作中心今年年底将在伦敦启用。
 <br/>
 <br/>
 英国通讯监管机构Ofcom表示，他们正在优先评估汉弗莱的投诉，如果发现中共央视有任何破坏规则的行为，将会采取〝必要的执法行动〞。
 <br/>
 <br/>
 在2012年，Ofcom曾根据类似的投诉，吊销了一家伊朗官方电视台的营业执照。
 <br/>
 <br/>
 <br/>
 <h4>
  一带一路惹祸？中共驻巴领馆遇袭
 </h4>
 <br/>
 中共驻巴基斯坦卡拉奇（Karachi）领事馆11月23号遭遇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恐怖袭击.html" target="_blank">
  恐怖袭击
 </a>
 ，至少有2名巴国警察死亡，一名警卫受伤，3名实施袭击的枪手也被击毙，中方没有人员伤亡。
 <br/>
 <br/>
 多家媒体引述现场消息说，枪手持手榴弹和其他武器，企图闯入领事馆，门外的保安人员开枪制止。之后疑犯在领事馆附近开枪，并投出一枚手榴弹。
 <br/>
 <br/>
 事发后，巴基斯坦争取独立的武装组织俾路支解放军发言人，在接受路透社采访时声称对本次攻击事件负责。这名发言人表示，他们派人闯入中共领事馆，是因为中共在剥削他们的资源。
 <br/>
 <br/>
 中共近年来在〝一带一路〞战略下，投资几十亿美元推进〝中巴经济走廊〞项目。据报导，这个项目对巴基斯坦而言，是巨大的债务负担。
 <br/>
 <br/>
