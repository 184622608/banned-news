### 加国大麻合法化在即 专家：危险性很高
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】
 <b>
  新闻周刊
 </b>
 （649）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻合法化.html" target="_blank">
  大麻合法化
 </a>
 即将到来。但是医学专家认为，大麻天然伴随着严重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健康风险.html" target="_blank">
  健康风险
 </a>
 ，敦促民众要做充分了解。
 <br/>
 <br/>
 纳米研究（Nanos Research）民调显示，几乎一半（48％）的安省人不知道娱乐和药用大麻之间的差异。即使如此，70%以上的人担心经常食用大麻的人影响心理健康，并引发并发症；60%的人担心经常使用大麻会上瘾。
 <br/>
 <br/>
 那使用休闲大麻到底有哪些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健康风险.html" target="_blank">
  健康风险
 </a>
 呢？
 <br/>
 <br/>
 1.使用大麻很危险
 <br/>
 <br/>
 安省医学协会发表了一份主题为〝围绕休闲大麻的健康风险〞的研究报告，认为大麻天然伴随着严重的健康风险，特别是经常使用的人，和小小年纪就开始吸大麻的。
 <br/>
 <br/>
 健康危害包括上瘾，欲罢不能；一系列呼吸和循环系统疾病，诱发家族遗传病，导致抑郁和焦躁等精神疾病，以及影响人的智力和认知能力。
 <br/>
 <br/>
 安省医学协会当选主席甘地医生（Dr.Sohail Gandhi）：〝如果在年轻时使用大麻，青少年，特别是25岁以下的人群，患病率上升，如抑郁症、精神分裂症、焦虑症、精神病，也就是开始出现幻觉，使用大麻的危险性相当高。〞
 <br/>
 <br/>
 有些民众认为，大麻能使人精神愉悦和放松，对治疗抑郁和焦虑症有效。甘地医生说：〝对已经和焦虑症、抑郁症挣扎的人们，不幸的是有时大麻会使病情恶化。〞
 <br/>
 <br/>
 2.年轻人上瘾比例超15%
 <br/>
 <br/>
 甘地医生说，目前研究证明的大麻治疗作用，主要用于癌症及化疗带来的疼痛，并不能推广到所有人群。很多人觉得，大麻是一种植物，不会让人上瘾，甘地医生可不这么认为。
 <br/>
 <br/>
 安省医学协会当选主席甘地医生（Dr.Sohail Gandhi）：〝大麻上瘾的风险是11人中有1人，大概9%，这个比例会增高，如果青少年时开始使用大麻的话，上瘾的比例是15%-16%。后果是严重的抑郁症，给自己造成伤害，自杀念头的风险增加，精神病的风险增加，看到不存在的东西，害怕，甚至伤害别人。〞
 <br/>
 <br/>
 3.能使大脑变迟钝
 <br/>
 <br/>
 年轻人要注意了！大麻能使大脑迟钝，在竞争激烈的社会，这也许会使你和美好前途擦肩而过。
 <br/>
 <br/>
 安省医学协会当选主席Dr.Sohail Gandhi医生：〝因为大麻是一种镇静剂，会放缓使使你反映灵敏的大脑活动，所以你想问题不会象平时那样快。〞
 <br/>
 <br/>
 4.其它危害
 <br/>
 <br/>
 吸大麻带来肺癌的风险也比吸菸高。因为人们吸大麻时会吸入更深，让大麻烟在身体里存留的时间更长些。
 <br/>
 经常使用大麻的青少年，还可能出现因肚子痛而不停呕吐的症状。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻合法化.html" target="_blank">
  大麻合法化
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 虽然即将成为事实，安省医学协会认为，能合法购买大麻的年龄应该推迟到25岁，如果联邦自由党政府想打击大麻地下市场的话，完全可以效仿葡萄牙的做法，持有少量大麻等毒品不会被刑事起诉，不必采取大麻全面合法化的做法。
 <br/>
 <br/>
 联邦反对党——保守党一直认为大麻合法化太过匆忙，时机不成熟。明年将是加拿大联邦选举，届时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻立法.html" target="_blank">
  大麻立法
 </a>
 将发生何种变化还不得而知。
 <br/>
 <br/>
 <br/>
 新唐人记者刘海英多伦多报导
