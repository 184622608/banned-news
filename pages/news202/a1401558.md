### G20场边会晤 川普文在寅重申2次〝川金会〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】白宫表示，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 今(30日)在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布宜诺斯艾利斯.html" target="_blank">
  布宜诺斯艾利斯
 </a>
 举行的20国集团（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 ）峰会场边会晤韩国总统文在寅时重申，有意和朝鲜领导人金正恩举行第2次峰会，并继续强力执行现有制裁，以确保朝鲜了解非核化是唯一的路。
 <br/>
 <br/>
 白宫发言人桑德斯（Sarah Sanders）说，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统论及他有意举行美国与朝鲜第2次高峰会。〞
 <br/>
 <br/>
 桑德斯表示，川普和文在寅〝重申他们就下一步骤密切合作的承诺〞。
 <br/>
 <br/>
 桑德斯表示，他们一致认为有必要〝继续强力执行现有制裁，以确保朝鲜了解非核化是唯一的路〞。
 <br/>
 <br/>
 韩国青瓦台发言人尹永灿（Yoon Young-chan）也表示，川普〝请韩国和美国密切合作，以利下次峰会成为朝鲜半岛非核化进程的另一座历史里程碑〞。
 <br/>
 <br/>
 据了解，11月上旬，美国国务卿蓬佩奥（Mike Pompeo）原本预计会见朝鲜高官，但朝鲜坚持美国需在减轻制裁情况下，华府无预警宣布会议延期。
 <br/>
 <br/>
 (责任编辑：程以仁)
