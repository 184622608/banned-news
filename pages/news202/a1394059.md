### 印尼地震海啸1424死  海啸警报没有反应
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 爆发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震海啸.html" target="_blank">
  地震海啸
 </a>
 第六天，搜救工作持续进行。周四，官方更新死亡人数为1424人，恐怕还会增加。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 官方公布最新数据，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震海啸.html" target="_blank">
  地震海啸
 </a>
 死亡1424人，多数人死于山崩和海啸。官员表示，死亡人数恐怕还会增加。
 <br/>
 <br/>
 在重灾区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_帕卢.html" target="_blank">
  帕卢
 </a>
 ，罗亚罗亚酒店的搜救工作仍在持续进行。据报，还有50人埋在瓦砾堆中，随着时间的流逝，生还的机会越来越渺茫。
 <br/>
 <br/>
 从直升机俯视灾区，尤其是沿岸地带，大片房屋被海啸和泥石流冲走，留下一片光土和废墟。
 <br/>
 <br/>
 撤离到避难中心的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灾民.html" target="_blank">
  灾民
 </a>
 ，回想起地震那一刻，仍心有余悸。
 <br/>
 <br/>
 当地居民ROOSMAWATI：〝第一次地震发生时，我立即倒地，我的孩子撞到头，然后发生第二次地震，更强，我跟孩子一起逃跑，我听到人们喊叫水在上升。〞
 <br/>
 <br/>
 地震发生后，灾区电力中断，通讯受阻，极度缺乏食物和生活物资，打劫商家情况一度十分猖狂，灾区濒临混乱。
 <br/>
 <br/>
 印尼当局呼吁国际协助救灾，之后主要电信业恢复运营，情况有所改善。
 <br/>
 <br/>
 民众呼吁政府关照灾民，将灾害降到最低。
 <br/>
 <br/>
 当地居民MUHAMMAD ASYKUR：〝我们必须要更有信心，多祈祷。我们希望政府未来更加关照灾民。〞
 <br/>
 <br/>
 这场灾难造成超过7万人无家可归，还有至少150人被埋在瓦砾堆中。
 <br/>
 <br/>
 强震带来的海啸，使伤亡更加严重。帕卢的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海啸警报.html" target="_blank">
  海啸警报
 </a>
 系统因疏于维护，事发时没有反应，天灾加上人祸，导致许多人来不及逃生。
 <br/>
 <br/>
 新唐人记者李梅、严曦综合报导
 <br/>
</div>
