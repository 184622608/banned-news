### 中共黑客加紧网攻 窃美公司知识产权
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】周二（10月9日），美国一家领先的网络安全公司报告，在过去一年中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑客.html" target="_blank">
  黑客
 </a>
 攻击，加紧窃取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国公司.html" target="_blank">
  美国公司
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 。
 <br/>
 <br/>
 据美国全国广播公司新闻网（NBC News）10月9日报导，美国网络安全公司〝CrowdStrike〞的联合创办人阿帕洛维奇（Dmitri Alperovitch）在周二发布的年中报告中说，〝大标题实际是——‘中国回来了’〞。
 <br/>
 <br/>
 报告称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在奥巴马政府末期曾暂停对美国网攻，但在过去的一年中，又再次加紧窃取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国公司.html" target="_blank">
  美国公司
 </a>
 的商业机密。
 <br/>
 <br/>
 阿帕洛维奇表示，据CrowdStrike公司观察，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑客.html" target="_blank">
  黑客
 </a>
 袭击的领域主要包括生物技术、国防、采矿、制药、专业服务和运输公司等方面。
 <br/>
 <br/>
 阿帕洛维奇指出，〝他们又回来大规模窃取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 了〞。
 <br/>
 <br/>
 CrowdStrike、其它私营公司和美国情报机构都报告说，在2015年9月之后，由于美中双方达成协议，可以看到出于纯粹经济目的中共网络间谍活动在下降。而美国间谍机构则表示，他们不会出于经济目的而从事间谍活动。
 <br/>
 <br/>
 但美国的企业和政府专家都表示，在过去一年中，由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 针对中共的不公平贸易采取了越来越强硬的立场，中共政府于是又藉助黑客加紧窃取美国公司的知识产权了。
 <br/>
 <br/>
 阿帕洛维奇是最早的网络安全研究人员之一，他公开宣称中共进行网络盗窃，并表示，据他的公司观察，来自中共国家安全部（Ministry of State Security）的网络攻击正在上升，而且远比中共军方的手法更熟练、更精通。
 <br/>
 <br/>
 2012年，时任美国国家安全局局长的亚历山大（Keith Alexander ）将军描述，中共及其它对立国家窃取它国知识产权是〝历史上最大的财富转移〞，因为中共已经用偷来的配方、方案和技术建立起整套的工业体系，而这些是西方公司花费了数年时间和数十亿美元才开发出来的。
 <br/>
 <br/>
 然而，中共通常否认，它参与了由国家资助的知识产权网络盗窃。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
