### 对川普让步 德国拟开放进口美国天然气
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】欧洲最大的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_能源市场.html" target="_blank">
  能源市场
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 将开放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进口.html" target="_blank">
  进口
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国天然气.html" target="_blank">
  美国天然气
 </a>
 ，从而减少对俄罗斯天然气的依赖。这被德国媒体视为美国总统川普的胜利，并成为德国能源政策的转折点。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 总理默克尔本月初告诉国会议员，德国政府将为建造海运码头提供支持，用来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进口.html" target="_blank">
  进口
 </a>
 美国液化天然气。工程预算高达5.76亿美元。
 <br/>
 <br/>
 默克尔强调这是〝战略性决策〞，而且〝不管这些海运码头在头几年是否盈利〞，都要进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国天然气.html" target="_blank">
  美国天然气
 </a>
 。
 <br/>
 <br/>
 外界认为，这意味着默克尔向美国做出重大让步，降低了俄罗斯在欧洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_能源市场.html" target="_blank">
  能源市场
 </a>
 的影响力。
 <br/>
 <br/>
 德国是欧洲的最大能源市场。去年德国从俄罗斯进口的天然气最多，占德国进口天然气总额的45%，比前年增加了4%。
 <br/>
 <br/>
 德国一直在抵制美国天然气，因为美国天然气需要液化后再海运，而俄罗斯天然气是管道直接输送，比美国便宜很多。
 <br/>
 <br/>
 川普上任后，游说欧洲各国采购美国天然气，减少或者放弃购买俄国天然气，作为与美国改善贸易关系的一部分。
 <br/>
 <br/>
 今年7月，川普批评德国花数十亿美元，从俄罗斯大规模购买石油和天然气，让德国高度仰赖俄罗斯，成为其〝俘虏〞。
 <br/>
 <br/>
 川普还说，北约投入大量军费保护德国不受俄罗斯侵犯，德国却向俄罗斯输送资金。川普批评默克尔无视盟国利益。
 <br/>
 <br/>
 德国决定在进口天然气这件事情上对美国让步，得到的好处是，可以改善和美国的贸易关系。另外，德国正在修建的北溪管道引起的争议也可以平息了。这条管线建成后会把大量俄罗斯的天然气输送到德国。美国和一些欧洲国家都反对这个项目。
 <br/>
 <br/>
 新唐人电视综合报导
</div>
