### 土耳其法院宣判 美国牧师当庭释放
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】周五（10月12日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国牧师.html" target="_blank">
  美国牧师
 </a>
 安德鲁．布伦森（Andrew Brunson）在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 出庭并被判决获释。他因被指控参与土耳其政变而遭监禁两年之久，并在近期引发了美土两国间严重的外交危机。
 <br/>
 <br/>
 周五，布伦森在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 伊兹密尔（Izmir）出庭，法官判处他已服刑期，并立即
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_释放.html" target="_blank">
  释放
 </a>
 。
 <br/>
 <br/>
 这一判决为打破目前的美土外交僵局提供了条件。
 <br/>
 <br/>
 现年50岁的布伦森是北卡罗来纳州的福音派基督教牧师。土耳其政府指控他在2016年的军事政变中从事间谍活动并勾结恐怖组织、企图推翻总统埃尔多安（Recep Tayyip Erdogan）。布伦森否认上述指控，但一直遭土耳其政府关押。直到今年7月，在美国的强大压力下，土耳其才因〝健康问题〞将他放出监狱，但仍把他软禁在家。
 <br/>
 <br/>
 周五，土耳其法院依旧给出的定罪，正式判处布伦森有期徒刑3年零1个月。但由于他已被拘禁2年多，所以不要求他继续在狱中服刑。
 <br/>
 <br/>
 这是布伦森第4次出庭接受听证，但他仍然坚称，他是无辜的。
 <br/>
 <br/>
 布伦森出庭时身穿黑西装、白衬衣，并佩戴一条红色领带，显得十分庄重。他的妻子则在旁听席观看他为自己辩护。
 <br/>
 <br/>
 法官传2名证人出庭作证。一个名叫卡尔坎（Levent Kalkan）的证人以前曾指控布伦森援助了恐怖组织，但周五出庭的两名新证人，一人表示不能肯定卡尔坎所陈述的内容，另一人则表示她甚至都不认识布伦森。
 <br/>
 <br/>
 由于土耳其当局此前拒不释放布伦森牧师，川普总统8月10日宣布制裁土耳其，对从土国进口的钢铝成倍加征关税，铝提高到20％，钢铁提高到50％，并明确指出美土关系当前非常紧张。
 <br/>
 <br/>
 美国南卡罗来纳州联邦参议员格雷厄姆（Lindsey Graham）8月12日表示，如果土耳其不释放布伦森，两国关系没有恢复正常的可能。
 <br/>
 <br/>
 格雷厄姆指出，对布朗森的指控完全都是骗人的，他与土耳其政变没有任何关系。
 <br/>
 <br/>
 但格雷厄姆也表示，美土关系可以修复，因为土耳其毕竟是美国的北约盟友、且设有一个美军空军基地。但前提是，土耳其必须让布朗森返回家园。
 <br/>
 <br/>
 布伦森案判决当天，川普总统发表多篇推文表示关注。判决结束后，他很快通报了这一消息。
