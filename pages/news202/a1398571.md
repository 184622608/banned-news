### 意大利帮助巴基斯坦基督徒 出走欧洲
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 政府周四表示，巴基斯坦女性基督徒毕比被无罪释放后，意大利将帮助她和家人离开巴基斯坦，到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧洲.html" target="_blank">
  欧洲
 </a>
 居住。
 <br/>
 <br/>
 毕比八年前因被指控渎神罪，被判死刑。上周，巴基斯坦最高法院宣判她无罪，引发穆斯林极端教派大规模上街抗议，要求公开处决毕比。
 <br/>
 <br/>
 当局后来达成协议，称将阻止毕比离境。
 <br/>
 <br/>
 周四，官员表示，毕比现在可以去任何地方。
 <br/>
 <br/>
 巴基斯坦外交办公室发言人MOHAMMAD FAISAL：〝她现在是自由人，法庭令已经宣布，她一旦做出决定，就可以去她想去的地方。〞
 <br/>
 <br/>
 毕比今年53岁，是5个孩子的母亲。
 <br/>
 <br/>
 在巴基斯坦人，只有2%的人口是基督徒，毕比的案件引发基督教和人权组织的强烈关注。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 政府周四表示，将帮助毕比和家人离开巴基斯坦，到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧洲.html" target="_blank">
  欧洲
 </a>
 居住。
 <br/>
 <br/>
 意大利天主教机构发言人MARTA PETROSILLO：〝目前我们得知，她只能跟她先生说话，还没有见到女儿们，但她和家人终于将齐聚一堂，他们正在等待离开巴基斯坦，但今天早上他们还在巴基斯坦。〞
 <br/>
 <br/>
 毕比的律师已逃离巴基斯坦，在荷兰寻求庇护，他证实毕比已经出狱。
 <br/>
 <br/>
 新唐人记者李梅、天羽综合报导
</div>
