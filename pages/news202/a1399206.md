### 英国脱欧草案达成 能否过关前景不乐观
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】周二，英国和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 达成了一项
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 ，英国首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特雷莎．梅.html" target="_blank">
  特雷莎．梅
 </a>
 将于周三召开内阁紧急会议，审议草案内容，如果通过，将提交欧盟投票。内阁成员对方案看法不一，无论是脱欧派还是留欧派，都有人因为反对首相的脱欧方案而辞职。
 <br/>
 <br/>
 英欧于周二在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 总部布鲁塞尔达成了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 。该草案还需经过英国内阁以及欧盟成员国的同意。最后再返回到英国下议院投票表决。
 <br/>
 <br/>
 英国内阁和议会在脱欧问题上仍存在很大分歧，无论脱欧派还是留欧派，都对首相的方案不满。前脱欧事务大臣戴维斯和前外交大臣约翰逊属于脱欧派，于今年7月辞职。
 <br/>
 <br/>
 约翰逊的弟弟、交通次长约翰逊属于留欧派，上周五辞职。他表示，还有几位内阁成员在考虑辞职。
 <br/>
 <br/>
 前外交部长约翰逊认为，这个草案不会被英国下议院接受和英国人民接受。这代表了一大部分脱欧派人士的看法。
 <br/>
 <br/>
 英国前外交大臣 约翰逊：〝在（首相的）方案下，我们将留在关税同盟，将留在大部分单一市场内。这个(英国)国会将对管辖这个国家的法律没发言权。〞
 <br/>
 <br/>
 反对党工党的脱欧事务发言人斯塔莫尔和约翰逊持相反意见,但同样反对脱欧草案。
 <br/>
 <br/>
 工党脱欧事务发言人 斯塔莫尔：〝我们需永久性地与欧盟建立关税同盟,我们需要一个强大的单一市场关系。〞
 <br/>
 <br/>
 连线本台欧洲记者 王羽音：〝目前首相梅可以说正在进行一场赌博,大部分内阁和议会成员都不希望无协议脱欧,因为这将给英国经济带来巨大打击。所以首相梅总是强调,要么接受我的脱欧方案,要么无协议脱欧。而对留欧派提出的第三种可能--二次脱欧公投,她是坚决反对。目前看来,这三种可能性都存在,其中二次公投的呼声越来越高。〞
 <br/>
 <br/>
 新唐人电视综合报导
</div>
