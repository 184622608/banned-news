### 四级飓风威拉 周二下午强势登陆墨西哥
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 威拉在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 太平洋海岸外短暂增强为5级风暴之后，转弱为4级风暴，预计当地时间下午登陆墨西哥。美国国家飓风中心预报，威拉将带来强风和洪水，威胁生命安全。
 <br/>
 <br/>
 网络视频显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 威拉到来前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 西部城市耶勒帕降下暴雨，引发山洪。
 <br/>
 <br/>
 周二， 美国国家海洋和大气管理局预告，飓风威拉短暂增强为5级风暴之后，转弱为4级风暴，周二下午登陆墨西哥，可能再减弱到3级。
 <br/>
 <br/>
 美国国家飓风中心警告，虽然威拉减弱，但仍可能给墨西哥中西部和西南部地区带来15到30厘米的降雨，有些地方高达18英寸，山区可能发生洪水泛滥和山体滑坡，灾害可能危及生命。
 <br/>
 <br/>
 墨西哥当局宣布，纳亚里特和锡那罗亚州的19个城市进入极度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_紧急状态.html" target="_blank">
  紧急状态
 </a>
 。
 <br/>
 <br/>
 当局警告民众离开高风险地区，必要时撤离家园，疏散到临时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_避难所.html" target="_blank">
  避难所
 </a>
 。
 <br/>
 <br/>
 居民们忙着填充沙袋，保护商店橱窗，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_储备食物.html" target="_blank">
  储备食物
 </a>
 与必需品，准备抵御飓风。
 <br/>
 <br/>
 商店老板ABEL RAMIREZ：〝我们为风暴潮做足了准备。(记者问：人们害怕吗？) 当然。他们因为害怕而准备。〞
 <br/>
 <br/>
 当地官员表示，有7,000至8,000人从低洼地区撤离，其中大部分位于锡那罗亚州。
 <br/>
 <br/>
 新唐人记者李梅、宇微综合报导
</div>
