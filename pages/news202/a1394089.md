### 俄罗斯骇客报复性攻击 美欧以法律制裁
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】美国司法部周四对七名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 特工提出起诉，罪名是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_骇客.html" target="_blank">
  骇客
 </a>
 攻击国际反兴奋剂机构。同一天，荷兰也指责俄方发动的一系列网络攻击。
 <br/>
 <br/>
 美国司法部助理部长德梅斯:〝今天起诉了一些在（荷兰）海牙被捕的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 特工，以及他们在莫斯科的同事。〞
 <br/>
 <br/>
 美国司法部助理部长德梅斯周四表示，起诉的这7个人，涉嫌对国际反兴奋剂机构进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_骇客.html" target="_blank">
  骇客
 </a>
 攻击，因为该机构谴责俄罗斯政府支持运动员使用兴奋剂，并支持禁止俄罗斯运动员参加国际体育比赛。
 <br/>
 <br/>
 美国司法部助理部长德梅斯:[被告认为，他们可以使用匿名的手段，肆无忌惮地，在自己的国家和其他主权国家的领土上，破坏国际机构，掩盖其政府的错误行为。他们错了。〞
 <br/>
 <br/>
 7人中有3人，也在今年7月被〝通俄门〞特别检察官穆勒起诉，罪名是入侵美国民主党的电邮帐户，并泄漏内容。
 <br/>
 <br/>
 同一天，荷兰安全部门称，曾在4月份粉碎俄国间谍对〝禁止化学武器组织〞的网络攻击，原因是该组织曾确认，今年3月在英国发生的神经毒剂袭击案中，案犯使用的是俄罗斯研发的〝诺维乔克〞毒剂。
 <br/>
 <br/>
 欧盟理事会主席图斯克:  〝代表欧盟强烈谴责这次(网络)攻击。〞
 <br/>
 <br/>
 北约秘书长 斯托尔滕贝格: 〝必须停止其鲁莽的行为模式。〞
 <br/>
 <br/>
 荷兰当局表示，他们因此驱逐了四名俄罗斯特工。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
