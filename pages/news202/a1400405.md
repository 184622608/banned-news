### 英国欧盟达成一致 落实未来关系框架
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国.html" target="_blank">
  英国
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 宣布，就未来双方关系框架达成宣言草案。欧盟各国领导人将于周日，决定是否最后通过该宣言草案，以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 协议草案。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国.html" target="_blank">
  英国
 </a>
 首相特雷莎・梅表示，英国和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 就双方未来关系框架，原则上达成宣言草案。
 <br/>
 <br/>
 英国首相特雷莎・梅：〝我们达成的协议，是在英国和欧盟之间达成的。欧盟其他27国领导人，将在周日欧盟理事会特别会议前，审查这份协议。〞
 <br/>
 <br/>
 特雷莎・梅表示，周日的协议能够解决直布罗陀未来地位问题，该议题之前一直悬而未决。
 <br/>
 <br/>
 英国首相特雷莎・梅：〝我相信，周日我们能为整个英国大家庭达成协议，包括直布罗陀。〞
 <br/>
 <br/>
 同一天，特雷莎・梅在首相府和奥地利总理库尔茨举行会谈，奥地利是欧盟轮值主席国。
 <br/>
 <br/>
 周四，欧盟委员会发言人 席纳斯表示，欧委会主席容克致信欧盟理事会主席图斯克，宣布欧盟和英国就未来关系框架达成宣言草案。
 <br/>
 <br/>
 欧盟委员会发言人席纳斯：〝（信中）附有政治宣言草案，列出欧盟和英国未来关系框架，谈判代表（就此）达成一致。〞
 <br/>
 <br/>
 据报道，根据草案，双方关系〝将尊重欧盟单一市场和关税同盟，以及英国内部市场的完整性，并承认英国在这种经济关系之外，制定的独立贸易政策。〞
 <br/>
 <br/>
 欧盟和英国同意〝发展活泼、广泛和平衡的经济合作关系〞。同时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 后的过渡期将延长〝一到两年〞。
 <br/>
 <br/>
 周四，德国总理默克尔表示，将尽可能达成协议。
 <br/>
 <br/>
 德国总理默克尔：〝我会尽一切可能达成脱欧协议。无序脱欧可能是最糟糕情况。〞
 <br/>
 <br/>
 北爱尔兰民主统一党领袖福斯特，周三表示，该党10名国会议员仍将支持特雷莎・梅政府，但不同意首相梅的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧方案.html" target="_blank">
  脱欧方案
 </a>
 。
 <br/>
 <br/>
 欧盟和英国就未来关系达成草案后，英镑兑美元短线大涨，短线突破1.29关口。
 <br/>
 <br/>
 新唐人记者毕心慈、明玉综合报导
</div>
