### 英法追随川普围堵中共 英剑指华为5G
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】英国政府部门近期正在审查该国的电信供应链的安全性，中国5G供应商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司或将受到英政府的严厉审查。与此同时，法国财政部也指出，中国不透明的央行货币互换交易可能掩盖潜在的经济问题。有舆论称，英法等国正追随川普的对华举措
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_围堵中共.html" target="_blank">
  围堵中共
 </a>
 。
 <br/>
 <br/>
 《华尔街日报》日前引述电讯业高管透露，英国政府正在评估电信设备市场的组成方式，恐将加大对中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司的审查力度。
 <br/>
 <br/>
 据这篇报导披露，英国政府上个月向国内多家电信公司致信称，政府将对英国是否过于依赖单一硬件提供商做出评估。信中并提醒英国的电信业界，在布建5G服务时应慎重选择供应商。
 <br/>
 <br/>
 英国政府在信中提醒业界，为了确保国家关键基础设施保持韧性和安全，政府在今年7月启动了对电信基础设施的审查，审查结果可能导致现行规则的变化，电信公司的5G供应链可能会受到影响，因此呼吁业者在执行任何采购决定前考虑审查的问题。
 <br/>
 <br/>
 报导称，这封信虽未点名任何公司，但收到此信的电信行业高层认为，英政府此举显然针对华为，表明英国政府将加大对中国华为的审查力度。
 <br/>
 <br/>
 有分析人士指出，上述决定反映了英国政府对华为不断扩张的担忧，可能导致华为遭受更严格的审查，甚至被禁止在英国推出5G。
 <br/>
 <br/>
 事实上，自从2005年赢得与英国电信的合同以来，华为已经成为英国电信供应链的核心部分，英国所有主要的无线和互联网提供商都使用华为的硬件或服务，英国的商用互联网流量采用的特殊设备也主要依靠华为来提供。出于国家安全的考量，英国政府近期已开始加大力度审查中国制造的设备。
 <br/>
 <br/>
 今年4月，英国政府曾警告不要使用中兴通讯的设备和服务，担心中国政府可能迫使中兴通讯渗透或破坏通信；7月，英国政府专家小组宣布，通过审查发现华为设备存在工程缺陷，给使用该公司产品的运营商带来了新风险。
 <br/>
 <br/>
 此前，美国以国家安全为由，禁止华为参与某些政府合同，该公司的设备在美国的销售受到限制；由于担心华为可能为中共政府的间谍活动提供便利，澳大利亚政府也决定禁止华为和中兴通讯参与其新的电讯网络建设项目；加拿大最近也已表示，不排除禁止华为为其5G网络提供电信设备；日本政府则表示，正在考虑对华为采取限制举措。
 <br/>
 <br/>
 不仅是华为和中兴的电讯设备的安全性受到质疑和拒绝，法国财政部本周二（6日）也在一份研究报告中，对中国不透明的央行货币互换交易提出质疑。
 <br/>
 <br/>
 据路透社报导，法国财政部的这份研究报告指出，中共政府正试图通过不透明的央行货币互换交易来提高其全球金融影响力，而中共的这些交易安排还可能利用短期流动性，掩盖各国潜在的经济问题，致使这些问题随时间的推移而一步步恶化。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：戴明）
