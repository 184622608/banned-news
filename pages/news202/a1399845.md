### 阿根廷发现失踪一年潜艇 失事前曾爆炸
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】一年前在大西洋失踪的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 海军潜艇〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣胡安号.html" target="_blank">
  圣胡安号
 </a>
 〞，已在周五发现残骸。阿根廷政府周六承认，目前没有足够的技术打捞在大西洋深处发现的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潜艇残骸.html" target="_blank">
  潜艇残骸
 </a>
 。
 <br/>
 <br/>
 国防部长阿瓜德（Oscar Aguad）在新闻发布会上说，海军缺乏技术打捞〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣胡安号.html" target="_blank">
  圣胡安号
 </a>
 〞，这艘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潜艇残骸.html" target="_blank">
  潜艇残骸
 </a>
 被发现在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 巴塔哥尼亚瓦尔德斯半岛（Valdes Peninsula）附近海域800米深处。他表示，官员仍需要确定下一步措施。
 <br/>
 <br/>
 国防部长阿瓜德：〝我们没有办法打捞沉船，我们甚至不能下到那么深的水域。〞
 <br/>
 <br/>
 周六上午早些时候，阿根廷海军表示，受阿根廷政府委托寻找
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪潜艇.html" target="_blank">
  失踪潜艇
 </a>
 的美国海洋勘探公司在东部瓦尔德斯半岛（Valdes Peninsula）用遥控潜水器确认找到了潜艇遗骸。
 <br/>
 <br/>
 2017年11月15日，圣胡安号在距离阿根廷沿岸450公里的海域，正准备返回马德普拉塔市（Mar del Plata）基地时，突然失去音讯，失事潜艇于1983年德国制造，2008至2014年间军方曾花120万美元更换引擎和电力装置。有调查人员日前指潜艇有很多缺陷。
 <br/>
 <br/>
 阿根廷军方表示，在潜艇失事之前，〝圣胡安号〞艇长曾报告潜艇换气装置进水，导致其中一组电力供应系统短路，但艇长后来说问题得到控制，但在〝圣胡安号〞最后一次通讯的附近检测到爆炸。海军表示爆炸可能是由于船长报告的短路问题引发的〝氢浓度〞过大引起的。
 <br/>
 <br/>
 事后阿根廷政府连同17个国家的舰艇及飞机，全面展开搜索，但都一无所获，经过整整一年的时间，才终于寻获潜艇位置。
 <br/>
 <br/>
 新唐人记者严枫、田瑞综合报导
</div>
