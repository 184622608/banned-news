### 法国西南部降暴雨 村庄遭暴洪侵袭至少13死
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】欧洲近来气候异常，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 官员表示，法国西南部今天降下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴雨.html" target="_blank">
  暴雨
 </a>
 ，河水变成汹涌的激流，造成至少13人死亡。
 <br/>
 <br/>
 中世纪要塞城市卡尔卡松（Carcassonne）周遭的数座城镇和村庄遭暴洪侵袭，造成许多汽车翻覆、道路毁坏和民宅倒塌。
 <br/>
 <br/>
 卡尔卡松北边的村庄维拉尔多内（Villardonnel）一座修道院因河水暴涨被淹没，一名年迈的修女遭河水冲走丧生。在此同时，地方当局表示，维勒盖扬克村（Villegailhenc）是重灾区，一夜间至少4人死亡。
 <br/>
 <br/>
 总理菲力普（Edouard Philippe）赶往灾区之际，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 内政部表示，共13人丧生。这场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴雨.html" target="_blank">
  暴雨
 </a>
 在短短数小时内，降下相当于3个月的降雨量。
 <br/>
 <br/>
 维勒盖扬克居民塞居哈（Helene Segura）表示：〝房子里到处都是水，所有东西都泡在水里。〞维勒盖扬克至少有一座小桥崩塌。
 <br/>
 <br/>
 当局派出直升机和600名消防员协助救援行动，尤其奥德河（Aude River）泛滥平原的救灾行动。根据法国环境部监测洪水官网（Vigicrues），奥德河暴涨到100年来最高水位。
