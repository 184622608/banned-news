### 美国务卿蓬佩奥访朝 与金正恩会晤两小时
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】在结束了对日本的访问之后，美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 周日先后对朝鲜和韩国进行访问。在平壤，蓬佩奥与朝鲜领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 举行了两小时会谈，蓬佩奥表示双方继续朝着朝鲜实行无核化方向迈进。
 <br/>
 <br/>
 在此次访问途中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 希望在这次平壤之行中能为美朝第二次峰会的时间和地点作出安排，但承认他这次有可能无法敲定可公开宣布的峰会安排细节。
 <br/>
 <br/>
 这是蓬佩奥自3月以来的第四次访问朝鲜，此行最初计画于8月底开始，但在最后一刻被美国总统川普取消，川普指责去核化缺乏进展。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 6月在新加坡与川普进行史无前例的会谈时承诺，努力实现无核化，但平壤的行动达不到华盛顿要放弃其核武器的步骤，包括申报所有核设施和导弹设施清单。
 <br/>
 <br/>
 蓬佩奥近日表示，对朝鲜的国际制裁必须保持原样，直到其放弃核武器。
 <br/>
 <br/>
 一位美国代表团的官员表示，对朝鲜的这次访问〝比上一次更好〞，但谈判将是长期的。
 <br/>
 <br/>
 在结束平壤之行后，蓬佩奥于周日晚些时候抵达韩国、会见韩国总统文在寅，他还将与韩国外交部长康京和举行会谈。周一，蓬佩奥将前往此次东北亚之旅的最后一站、北京。
 <br/>
 <br/>
 新唐人记者严枫、天羽综合报导
</div>
