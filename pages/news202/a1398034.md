### 法属新喀里多尼亚举行独立公投
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南太平洋.html" target="_blank">
  南太平洋
 </a>
 法国所属群岛新喀里多尼亚（New Caledonia）星期日（11月4日）举行公民表决，决定是否成为世界上一个最新的主权国家。稍早根据百分之70的计票结果，超过百分之59的选民拒绝独立、选择继续留在法国。
 <br/>
 <br/>
 这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_独立公投.html" target="_blank">
  独立公投
 </a>
 是新喀里多尼亚长达30年非殖民化进程的结果。这也是自1977年位于非洲东部非洲之角（Horn of Africa）的吉布提投票独立以来，在法国领地上举行的第一次独立公投。高度自治的新喀里多尼亚群岛人口总数约为27万人，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南太平洋.html" target="_blank">
  南太平洋
 </a>
 最大的经济体之一。支持独立的土著卡纳克人（Kanaks）和忠于法国的殖民定居者后裔之间长期处于紧张关系。
 <br/>
 <br/>
 投票者们在投票站排成长队。他们被问〝你想让新喀里多尼亚获得完全主权并独立吗？〞
 <br/>
 <br/>
 法国曾是一个跨越加勒比海、撒哈拉以南非洲和太平洋的殖民大国。新喀里多尼亚如果宣布独立，不仅有损法国的骄傲，也会削弱法国在印度-太平洋地区的势力。针对这次公投结果，预期法国总统马克龙将在巴黎时间午后1点发表全国讲话。
 <br/>
 <br/>
 新唐人记者徐哲、雨薇综合报导
</div>
