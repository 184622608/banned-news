### 暴风雨席卷意大利酿17死 数千公顷森林遭破坏
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超强风暴.html" target="_blank">
  超强风暴
 </a>
 席卷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 北部山区，截至3日已造成17人死亡，数千公顷的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_森林.html" target="_blank">
  森林
 </a>
 遭破坏，许多几百年老树被连根拔起，满目疮痍。
 <br/>
 <br/>
 最高时速达190公里的风暴，席卷了位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 北部的多洛米蒂山脉(Dolomiti)，包括特伦蒂诺-上阿迪杰大区(Trentino-Alto Adige )和威尼托大区(Veneto)，受创最严重地。
 <br/>
 <br/>
 意大利民防保护部（Civil Protection Agency）3日表示，一名德国观光客2日在萨丁尼亚岛（Sardinia）遭雷击丧命，数天前遭雷击的另一人在医院不治。使原本死亡人数从15人上升至17人。
 <br/>
 <br/>
 意大利农牧协会（Coldiretti）发表声明说，暴风摧毁达数千公顷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_森林.html" target="_blank">
  森林
 </a>
 、约1400万株树木，许多几百年树龄的松树也被连根拔起，许多受害人是遭强风吹倒的树木压死。
 <br/>
 <br/>
 山体滑坡也毁坏了部分公路和村庄，导致许多村庄陷孤立及道路交通受阻。
 <br/>
 <br/>

 <br/>
 威尼托大区区长扎亚表示，该区仍有16万人无电可用。位于该区的水都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威尼斯.html" target="_blank">
  威尼斯
 </a>
 （Venice）的许多广场和人行道，则遭到过去10年来最严重洪水淹没。
 <br/>
 <br/>
 威尼托地区首长柴亚（Luca Zaia）表示，威尼托地区灾损高达至少10亿欧元。
