### 德国黑森州选举失利 默克尔将淡出政坛
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默克尔.html" target="_blank">
  默克尔
 </a>
 所在的基督民主同盟党再次经历地方选举失败，周一，默克尔宣布不再参选党主席，这预示着〝默克尔时代〞将在本次任期后终结。
 <br/>
 <br/>
 两周前的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 巴伐利亚州选举和周日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑森州选举.html" target="_blank">
  黑森州选举
 </a>
 ，成为终结〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默克尔.html" target="_blank">
  默克尔
 </a>
 时代〞的最后两根稻草。
 <br/>
 <br/>
 在这两个州，默克尔的执政联盟惨败，得票率都下跌超过10%。和默克尔联合执政的社会民主党经历了同样的败绩。
 <br/>
 <br/>
 反移民难民的右派政党德国选择党人气窜升，进入州议会。
 <br/>
 <br/>
 外界普遍认为，默克尔的难民政策拖累两个执政党大失民心。周一，德国总理默克尔宣布，不再参加12月份的基民盟党主席选举；2021年总理任期到期后，不再寻求连任，也不在政界担任任何职务。
 <br/>
 <br/>
 德国总理默克尔：〝通过这一决定，我努力做出贡献，使德国政府能够最终集中力量在治理好（国家）上。〞
 <br/>
 <br/>
 德国民众也认为，现在是换人的时候了。
 <br/>
 <br/>
 柏林游客 Christian Edler：〝我认为现在会有新人加入（政党）董事会，这将为政界带来新鲜空气。〞
 <br/>
 <br/>
 连线本台欧洲记者王羽音：〝2015年默克尔宣布‘接收难民无上限’，这让她大失民心。2017年德国大选的时候,默克尔所在政党选票流失严重，之后的组阁又花了半年时间。几个月前，默克尔的传统盟友基督教社会同盟党又因为难民问题，挑战默克尔。外界评论德国政府四分五裂，民众不满情绪不断升级。最近两次州选举，正是这种不满情绪的爆发。默克尔做了18年基民盟党主席，13年德国总理，目前看来，默克尔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_淡出政坛.html" target="_blank">
  淡出政坛
 </a>
 ，似乎是解决问题的唯一办法了。〞
 <br/>
 <br/>
 新唐人电视 综合报导
