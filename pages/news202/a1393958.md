### 时事拼盘：两德统一28周年 三学者获诺贝尔化学奖
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】下面请看一组新闻简讯
 <br/>
 <br/>
 周三是两德统一28周年纪念日。盛大庆祝活动在柏林举行。布兰登堡门被用历史照片包覆，重现1989年11月9日柏林墙倒塌。不过，原东德地区经济发展迟缓，2015年难民潮爆发后，当地极右翼势力抬头，仍然不同程度分裂着德国社会。
 <br/>
 <br/>
 德国总理默克尔：〝两德统一还没有结束。〞
 <br/>
 <br/>
 2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诺贝尔.html" target="_blank">
  诺贝尔
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_化学奖.html" target="_blank">
  化学奖
 </a>
 于周三在瑞典揭晓。美国科学家阿诺德、史密斯和英国科学家温特爵士共同获奖。他们在研究酵素演化以及胜肽、抗体方面做出卓越贡献。阿诺德获得二分之一奖金，史密斯与温特各获得四分之一。
 <br/>
 <br/>
 马来西亚反贪机构于周三逮捕了前第一夫人罗斯玛，周四开庭提审,罪名是洗钱和其他非法活动。罗斯玛是前首相纳吉布的妻子，罗斯玛被捕当天，纳吉布也被警方传召，以协助调查〝一马公司〞的案件。
 <br/>
 <br/>
 〝失踪〞了123天后,大陆知名影星范冰冰在微博上发表了《致歉信》，承认在影片《大轰炸》和其他一些合同中利用〝阴阳合同〞逃税。在这条微博发出的几个小时前，中国官方通讯社援引税务部门称，范冰冰连缴费带罚款,共需上缴人民币8.8亿元，以免除刑责。
