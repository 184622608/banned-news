### 伦敦五十万人游行反脱欧  要求二次公投
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】周六，数十万支持英国留在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 的民众走上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦.html" target="_blank">
  伦敦
 </a>
 街头，希望政府举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二次公投.html" target="_blank">
  二次公投
 </a>
 。组织者称这是迄今规模最大的一次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反脱欧游行.html" target="_blank">
  反脱欧游行
 </a>
 。
 <br/>
 <br/>
 游行民众高举
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 旗帜，打着有节奏的鼓点，从海德公园出发，途径唐宁街，前往国会大厦。组织者称这次游行约有50万人参加，是至今规模最大的。
 <br/>
 <br/>
 游行者反对英国脱离欧盟，希望政府就脱欧问题举行第
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二次公投.html" target="_blank">
  二次公投
 </a>
 。
 <br/>
 <br/>
 游行者 Sarah Bennett：〝我们必须留在单一市场，必须留在关税同盟。如果不这样，英国的经济将遭受灾难性的打击。〞
 <br/>
 <br/>
 年轻抗议者希望在第二次公投时能投下一票，支持英国留在欧盟。
 <br/>
 <br/>
 17岁抗议者 Patrick Robinson：〝我非常希望能举行二次公投，我的第一选择当然是留在欧盟。所以，一张人民的选票，是我绝对想要的。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦.html" target="_blank">
  伦敦
 </a>
 市长塞迪克・汗（Sadiq Khan）也加入了游行队伍，他对现政府领导的脱欧谈判感到失望，希望用公投这种更英国的方式，决定英国的未来。
 <br/>
 <br/>
 伦敦市长 Sadiq Khan：〝现政府只能达成不好的脱欧协议，或者更糟，根本达不成协议。这些选项跟我们期待的差之千里，政府没有权力赌上我们的未来。〞
 <br/>
 <br/>
 〝有些人认为公投是不民主、不爱国的，我认为事实上恰恰相反。没有比相信英国人民的判断力，更民主，更英国的了。〞
 <br/>
 <br/>
 英国首相特蕾莎・梅没有举行二次公投的意愿，反对党工党上个月曾表态，对二次公投持开放态度，必要时可以采用。
 <br/>
 <br/>
 而脱欧的支持者则认为，二次公投会导致宪法危机。
 <br/>
 <br/>
 新唐人电视安心综合报导
</div>
