### 肯尼亚巴士翻覆车顶掀翻 42人当场身亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肯尼亚.html" target="_blank">
  肯尼亚
 </a>
 (
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肯亚.html" target="_blank">
  肯亚
 </a>
 )西部10日上午发生一起重大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴士翻覆.html" target="_blank">
  巴士翻覆
 </a>
 事故意外，当场有42人丧命，死亡可能还会再攀升。目前未有进一步事故原因细节。
 <br/>
 <br/>
 警方表示，出事巴士共搭载52名乘客，从首都奈洛比（Nairobi）开往西部城镇卡卡梅贾（Kakamega）。
 <br/>
 <br/>
 裂谷省（Rift Valley）交警艾罗姆（Zero Arome）对媒体表示：〝我接获通报，超过40人当场死亡，但因巴士车顶掀翻，死亡人数恐会攀升。〞
 <br/>
 <br/>
 从社交媒体发出的照片可见，巴士侧翻在一片草地上，巴士车顶扭曲掉落一旁，乘客物品掉落一地，现场一片凌乱。
 <br/>
 <br/>
 另一名在事故现场的交警对媒体表示：〝事故造成42人丧命，我们仍在现场（处理）。〞
