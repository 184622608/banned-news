### 访问平壤前  美国务卿先访日会晤安倍
------------------------

<div class="wysiwyg">
 【新唐人2018年10月06日讯】周六，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国务卿蓬佩奥.html" target="_blank">
  美国国务卿蓬佩奥
 </a>
 ，在东京会晤日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍.html" target="_blank">
  安倍
 </a>
 晋三，两人针对朝鲜和贸易等议题，交换了意见。蓬佩奥随后飞往平壤，展开今年的第四次访朝之旅。
 <br/>
 <br/>
 日本东京是蓬佩奥本次东亚之行的第一站，而下一站朝鲜则是蓬佩奥此行的重点。
 <br/>
 <br/>
 日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍.html" target="_blank">
  安倍
 </a>
 晋三：〝我感谢蓬佩奥国务卿在访问朝鲜之前先访问日本。〞
 <br/>
 <br/>
 蓬佩奥则表示，将保持与日方沟通，分享朝鲜谈判的进度，并向平壤提出遭绑架日本公民的问题。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝我们将就如何推进（无核化）达成完全协调、统一的意见。若朝鲜无核化要成功，这是必要的。〞
 <br/>
 <br/>
 蓬佩奥表示，将和平壤确认美朝双方达成完全共识，范围包括导弹试射项目，以及化学和生物武器项目的处理等。
 <br/>
 <br/>
 蓬佩奥早些时候还对媒体表示，此次访朝的目的是确保〝双方了解真正要取得的目标〞，同时希望针对川普第二次会晤金正恩的大致日期和地点展开讨论。
 <br/>
 <br/>
 在与安倍的会晤中，蓬佩奥也谈到日本和美国的贸易谈判，已经处于敲定协议的阶段。
 <br/>
 <br/>
 上月底，安倍在纽约与美国总统川普会晤，宣布开启双边贸易谈判。美方的主要要求是减少每年690亿美元的贸易逆差，日方则不希望看到美方对日本汽车加征关税。
 <br/>
 <br/>
 蓬佩奥当天也会晤了日本外相河野太郎。随后他会前往平壤，之后继续访问首尔和北京。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
