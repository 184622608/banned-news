### 川普马克龙会晤  就北约军费投入达共识
------------------------

<div class="wysiwyg">
 【新唐人2018年11月10日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 ，周五抵达法国，参加纪念一战〝停战100周年〞的系列活动。川普和法国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 ，在周六举行会晤。双方就〝欧洲应该增加对北约的军费投入〞一事，达成共识。
 <br/>
 <br/>
 川普与夫人梅拉尼亚周五晚间抵达巴黎，开启了法国之旅。
 <br/>
 <br/>
 川普此行的主要目的是和其他国家元首一起，参加纪念
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一战停战.html" target="_blank">
  一战停战
 </a>
 100周年的活动。此外，川普也计划和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 举行双边会晤。
 <br/>
 <br/>
 周六，川普与马克龙在巴黎爱丽舍宫会晤，双方握手并多次向媒体致意。
 <br/>
 <br/>
 马克龙本周早些时候曾发表言论，认为欧洲应该增强军队实力，以抵御中国、俄罗斯乃至美国。川普在前往法国的当天发推文，批评马克龙的说法对作为盟友的美国很不尊重，再次提出欧洲各国目前没有公平分担
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北约军费.html" target="_blank">
  北约军费
 </a>
 的问题。
 <br/>
 <br/>
 不过，在周六的会晤中，双方都表露出友好的态度，并强调美国和法国同盟具有悠久的历史，两国人民之间有深厚的友谊。
 <br/>
 <br/>
 美国总统 川普：〝我们希望看到一个强大的欧洲，这对我们很重要。我们会用最好、最有效的方法，这是我们都想看到的。〞
 <br/>
 <br/>
 而针对川普提出欧洲应该增加军费支出的问题，马克龙表示赞同欧洲应该在北约中承担更大的责任。
 <br/>
 <br/>
 法国总统 马克龙：〝我相信我的增强欧洲防御提案与此相符，因为这意味着欧洲在北约有更大的担当和实力，我认为这很公平，也很重要。〞
 <br/>
 <br/>
 川普与马克龙还预计讨论贸易、伊朗、以及美国退出《中导条约》冲突等议题，并与两位第一夫人共进午餐。
 <br/>
 <br/>
 川普将在周六和周日分别拜访巴黎周边的两座公墓，参加纪念一战活动，并发表正式演説。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
