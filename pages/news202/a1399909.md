### 冒死冲过边境 脱北士兵：年轻人对金正恩无效忠之心
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 士兵吴青成去年在枪林弹雨中冲过边境，向韩国投诚，受到全球关注。他日前首度接受媒体专访透露，父亲是朝鲜将军，而朝鲜年轻一代对于领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 已没有效忠之心。
 <br/>
 <br/>
 日本《产经新闻》报导，今年25岁的吴青成(Oh Chong-song，译音)，其实是一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 将领的儿子。
 <br/>
 <br/>
 报导说，吴青成虽然出身上层阶级家庭，但对于朝鲜政治、领袖都很冷漠，已没有忠诚之心。
 <br/>
 <br/>
 吴青成说，自己对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 的统治〝无感〞，也不在乎朋友对他的看法。他说，〝我这个世代大概有80%都抱持冷漠心态。〞
 <br/>
 <br/>
 他说，因为大家都认为世袭的政权不会改变，对政府无感或无忠诚度是很自然的，更别提政府〝连喂饱人民的能力都没有〞。
 <br/>
 <br/>
 对于朝鲜半岛近期情势的大幅改变，吴青成说，他在朝鲜服役时，〝真的觉得和美国的战争一触即发〞，感受到的情势非常紧张。
 <br/>
 <br/>
 此前，有韩国媒体报导，吴青成在朝鲜因谋杀遭追缉，但他否认。他说，投诚前因为和朋友间发生问题，他喝酒后要回岗位途中，他冲过岗哨，因为担心遭处决，所以干脆继续狂奔，冲过边界。
 <br/>
 <br/>
 对于投诚他并不后悔，也能理解他冲过边界时，同袍必须对他开枪，他说，〝如果他们不开枪，一家会受到很严重的惩罚。如果我是他们，也会这么做〞。
 <br/>
