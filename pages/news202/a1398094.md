### 埃及宗教武装喋血案 警撃毙19名极端分子
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃及.html" target="_blank">
  埃及
 </a>
 恐怖分子在明亚省以西沙漠中与安全部队交火时，有19名被打死。内政部4日发声明表示，这些恐怖分子与上周五（2日）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开罗.html" target="_blank">
  开罗
 </a>
 3辆载有基督徒的巴士遇袭有关。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃及.html" target="_blank">
  埃及
 </a>
 内政部声明中说，极端分子袭击警方，警方还击。其后调查发现，死亡的极端分子与发动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开罗.html" target="_blank">
  开罗
 </a>
 巴士袭击的组织有关，亦涉嫌在国内发动多次袭击。
 <br/>
 <br/>
 2日，一辆从埃及南部索哈杰省开往明亚省(Minya)一座
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科普特基督教堂.html" target="_blank">
  科普特基督教堂
 </a>
 （Coptic Christianity）的巴士遭武装分子袭击，造成至少7名科普特人死亡、12人受伤。当局接报后，派出大批军警前往该区，封锁通往出事现场的路段，并截查来往车辆。
 <br/>
 <br/>
 极端组织〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊斯兰国.html" target="_blank">
  伊斯兰国
 </a>
 〞（IS）的西奈半岛分支此前已承认责任。
 <br/>
 <br/>
 <b>
  相关视频：
 </b>
 <center>
 </center>
 <br/>
 科普特人信奉基督教，仅占埃及总人口的10%，近年屡成袭击目标。去年5月，一群蒙面枪手向开往同一家基督教堂的巴士开枪，造成28人死亡，22人受伤。同年12月，一名IS枪手对开罗南部郊区的一座教堂发起袭击，造成9人死亡。
 <br/>
 <br/>
 今年10月，埃及军事法庭已针对教堂发起自杀式袭击的17名极端分子判处死刑。此外，埃及军队于2018年2月在西奈半岛对IS进行了一次攻撃，据军方估计，该行动已造成450多名IS分子死亡。
 <br/>
