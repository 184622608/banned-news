### 减收女考生 日本不仅东京医大另有其他大学
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东京医科大学.html" target="_blank">
  东京医科大学
 </a>
 日前爆出在入学考试时操作给分，藉以减收女生争议，日本文部科学省(文科省)针对约30所设置医学系且录取率男高于女的大学进行访查，发现另有其他几所大学，针对性别及年龄设有差别待遇。
 <br/>
 <br/>
 东京医大日前被媒体报导为高官小孩加分以获政府补助款，接着又爆出从2011年来持续对女考生及重考多次的男考生，在给分上有明显的差别待遇，意图
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_减收女考生.html" target="_blank">
  减收女考生
 </a>
 。
 <br/>
 <br/>
 日本放送协会（NHK）报导，为此，文科省针对日本设有医学系的81所公私立大学进行调查，发现有8成的大学医学系录取率都是男考生高于女考生。
 <br/>
 <br/>
 文部科学大臣柴山昌彦12日在内阁会议后的记者会上说，他们针对女考生录取率低于男考生的大学中的约30所进行访问调查，得知其中有几所大学确实有针对性别及年龄设置差别待遇；情势发展至今确实是个问题。
 <br/>
 <br/>
 不过，柴山未具体表明这几所大学的名字。
 <br/>
 <br/>
