### 美专家：结构性改变是关键  否则90天后贸战重燃
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】在阿根廷结束的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞结束后，白宫发布声明称，中方在四大方面做出了承诺，美方暂停
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加征关税.html" target="_blank">
  加征关税
 </a>
 ，给习近平90天缓冲谈判时间。对此，有美国专家说，如果双方不能在涉及美中关系的结构性问题上达成一致，贸易战还会重燃。
 <br/>
 <br/>
 美国总统川普（特朗普）与中国国家主席习近平于12月1日，在阿根廷布宜诺斯艾利斯举行晚宴后，白宫发表声明称，双方都表示会谈〝高度成功〞。
 <br/>
 <br/>
 声明表示，川普总统同意，在2019年1月1日，不会将2,000亿中国商品10%惩罚性关税提高到25%，而中共同意增加采购美方农业、能源、工业和其它产品，以减少两国之间的贸易不平衡。声明说，中共已同意立即从美国购买农产品。
 <br/>
 <br/>
 另外，中方同意将芬太尼指定为管制物质，这意味着向美国出售芬太尼的人，将受到中国法律规定的最高惩罚。
 <br/>
 <br/>
 声明还说，美中双方同意立即就结构性改变问题展开为期90天的谈判，这些问题包括强制技术转让、知识产权保护、非关税壁垒、网路入侵和网路盗窃。如果双方无法在这个期限内达成一致，美方将继续把关税税率提高至25%。
 <br/>
 <br/>
 美中双方还一致认为，在朝鲜问题上取得了很大的进展，川普和习近平一起，与金正恩共同努力，以完成朝鲜半岛无核化。
 <br/>
 <br/>
 川普总统在声明中说：〝这是一场精彩的、富有成果的会谈，对美国和中国具有无限可能性。与习主席合作，是我的荣幸。〞
 <br/>
 <br/>
 不过，有美国专家对美中贸易战的前景不乐观。美国智库战略与国际研究中心资深研究员葛来仪（Bonnie Glaser）对美国之音分析说：此次中美的贸易协议是一个占位符（placeholder），本可以在数月前达成。
 <br/>
 <br/>
 葛来仪说，除非美国和中国的谈判人员就美中经济关系中的结构性的问题达成协议，否则〝 两国可能会在90天后重返贸易战〞而这些结构性的问题也是最初促使川普总统实施关税措施的起因。
 <br/>
 <br/>
 华盛顿智库威尔逊中心基辛格美中关系研究所主任戴博(Robert Daly)认为，特习会达成的协议让贸易战暂时停火，给出谈判时间，但是，中共做出川普政府要求的所有结构性改变的可能性仍然很低，在90天的谈判期内要维持积极的氛围也不容易。
 <br/>
 <br/>
 分析人士认为，尽管美国没有直接提出中共需要在体制上，或是在政治结构上做出改变，但这种经济上的要求，实际上直指中共体制，但中共的底线就是维持统治，影响到其政权和统治的事情，它绝不会干。因此双方最终要达成协议很难。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
