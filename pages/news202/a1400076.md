### 未领到APEC执勤津贴 巴新军警闯国会砸家俱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚太经济合作组织.html" target="_blank">
  亚太经济合作组织
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 ）领袖峰会日前才在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴布亚新几内亚.html" target="_blank">
  巴布亚新几内亚
 </a>
 (巴新)落幕，20日传出，巴新军警因没拿到APEC执勤津贴，闯进国会砸烂窗户和家俱事件。
 <br/>
 <br/>
 法新社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴布亚新几内亚.html" target="_blank">
  巴布亚新几内亚
 </a>
 这个贫穷的太平洋岛国，上周在首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摩斯比港.html" target="_blank">
  摩斯比港
 </a>
 （Port Moresby）举办为期2天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 领袖峰会，为了迎接来访的各国领袖，添购了40辆玛莎拉蒂（Maserati）跑车，来接送出席峰会的贵宾。
 <br/>
 <br/>
 峰会结束后，今(20)日却有巴新军警闯进国会抗议，要求发放他们在APEC执勤应得的350基那（约103美元）津贴，他们大肆破坏国会窗户、盆栽、家俱洩愤。
