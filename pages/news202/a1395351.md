### 尼泊尔雪崩 韩国登山团队至少七人遇难
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼泊尔.html" target="_blank">
  尼泊尔
 </a>
 警方星期六（10月13日）报告，一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国登山队.html" target="_blank">
  韩国登山队
 </a>
 和他们的尼泊尔向导，在尼泊尔西部喜马拉雅山的一处营地，遭遇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雪.html" target="_blank">
  暴风雪
 </a>
 ，至少7人丧生。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼泊尔.html" target="_blank">
  尼泊尔
 </a>
 西部谷尔加山峰（Mount Gurja）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雪.html" target="_blank">
  暴风雪
 </a>
 引发了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雪崩.html" target="_blank">
  雪崩
 </a>
 ，共有9人失踪。根据警方报告，村民们在山峰底部发现了7具尸体。这个地区很难由步行到达。
 <br/>
 <br/>
 这是尼泊尔两年来发生的最惨痛的登山事故。一名救援人员在尼泊尔首都加德满都接受采访时说，雪崩摧毁了登山探险者的营地。
 <br/>
 <br/>
 谷尔加山峰距离加德满都约130英里。救援人员表示，一旦条件改善，将尝试再次抵达探险者营地。
 <br/>
 <br/>
 尼泊尔一侧的喜马拉雅山脉拥有8座世界上的最高峰。目前正是秋季的登山高峰季节；而外国登山者和徒步探险者是这个国家的主要外汇来源。
 <br/>
 <br/>
 新唐人记者徐哲、雨薇综合报导
