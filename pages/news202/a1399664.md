### 杀1/5人口 红色高棉两头目 被判种族灭绝罪
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】周五，针对20世纪最血腥的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红色高棉.html" target="_blank">
  红色高棉
 </a>
 大屠杀，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国柬埔寨特别法庭.html" target="_blank">
  联合国柬埔寨特别法庭
 </a>
 裁定，柬埔寨共产党的2名高官犯下〝种族灭绝〞等多项罪刑，被判处无期徒刑。
 <br/>
 <br/>
 周五，联合国柬埔寨法院特别法庭裁定，由柬埔寨共产党统治的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红色高棉.html" target="_blank">
  红色高棉
 </a>
 政权，前2号人物
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农谢.html" target="_blank">
  农谢
 </a>
 ；和前5号人物
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乔森潘.html" target="_blank">
  乔森潘
 </a>
 ，在1975到1979年统治柬埔寨期间，对境内的穆斯林和越南人犯下〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_种族灭绝罪.html" target="_blank">
  种族灭绝罪
 </a>
 〞，判处终身监禁。
 <br/>
 <br/>
 〝红色高棉〞幸存者 SOK SABON：〝他们罪有应得，因为他们犯下大罪。我的家人都被杀了，我是（红色高棉）统治下，（家里）活下来的唯一孩子。〞
 <br/>
 <br/>
 柬埔寨共产党1975年夺权后，奉行〝马列主义和毛泽东思想〞，宣称要实现〝纯粹共产主义〞。掌权的短短44个月内，强制清空城市，驱赶平民下乡强制劳动，导致大量人口因饥饿、过度劳累死亡。柬共还施行党内清洗、宗教迫害、劳改营等政策，共造成柬埔寨约170万人死亡，约占总人口的1/5。
 <br/>
 <br/>
 2014年，农谢和乔森潘，已因强行驱赶柬埔寨人到农村强制劳动，被判犯下〝危害人类罪〞，判处无期徒刑。
 <br/>
 <br/>
 人权组织〝公民力量〞创办人 杨建利博士：〝我觉得这种审判，正义虽然来得晚，但是是非常必要的。因为以共产主义理念建起来的红色恐怖政权，在世界还在延续。像中国、北韩、古巴、越南等等。柬埔寨的审判告诉大家，任何统治者的暴行最后都会得到历史的审判。〞
 <br/>
 <br/>
 一些评论人士认为，这一审判，是柬埔寨〝转型正义〞的里程碑。
 <br/>
 <br/>
 但华府人权律师叶宁认为，这起审判耗时12年、耗资近3亿美金，审判结果不足以警示及吓阻其他现行人权罪犯；他认为，国际社会要想真正推动公平正义，也不应只满足于审判历史旧案。
 <br/>
 <br/>
 华府人权律师 叶宁：〝对正在进行当中的灭种罪，正在进行当中的酷刑罪，正在进行当中的反人类罪，（联合国）基本上处于一种休克状态，冬眠状态。〞
 <br/>
 <br/>
 杨建利：〝我们很期待国际社会，能够挑选共产党统治中国历史上的一些暴行，国际法庭对这样的暴行进行审判。这是我们目前倡导的。〞
 <br/>
 <br/>
 柬埔寨共产党的罪行，也曾获得中共大力支持。联合国特别法庭检调人员克雷格・艾奇逊，2014年撰文披露，红色高棉档案显示，中共为红色高棉提供了规模惊人的援助，包括军事物资、后勤和人员等。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
