### 金正恩想坐韩国高铁 青瓦台建议：12月18至20日回访
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】外界一直关注朝鲜领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 何时回访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首尔.html" target="_blank">
  首尔
 </a>
 。据韩国总统府青瓦台消息人士表示，已向朝鲜建议〝金正恩于12月18至20日回访首尔〞，目前正等待朝鲜当局的回复。韩国相关部门也展开其参访行程的沙盘推演，而金正恩曾表示想乘坐韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高铁KTX.html" target="_blank">
  高铁KTX
 </a>
 ，但因维安问题，韩国正在考虑。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文在寅.html" target="_blank">
  文在寅
 </a>
 周一前往新西兰访问，期间他在专机内表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 仍有可能在2018年内回访韩国，期待金正恩早日做出决定。
 <br/>
 <br/>
 韩国〝东亚日报〞5日在头版头条以〝青瓦台向朝鲜建议－邀请金正恩18~20日回访〞为标题，报导青瓦台欲敦促金正恩于年内回访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首尔.html" target="_blank">
  首尔
 </a>
 的消息。
 <br/>
 <br/>
 报导引述青瓦台消息人士表示，若要实现金正恩年内回访，朝鲜最迟得于本周内作出决定。
 <br/>
 <br/>
 消息人士表示，韩国近期透过台面下管道向朝鲜作出前述建议，具体来说，由于12月17日是朝鲜前国防委员长金正日逝世7周年，因而建议金正恩从18日至20日之间对首尔进行回访。
 <br/>
 <br/>
 青瓦台和国家情报院等有关部会已研讨就金正恩回访的具体事宜展开沙盘推演，包括金正恩回访首尔的警护和住宿等问题。
 <br/>
 <br/>
 相关部门同时也考虑安排金正恩参访济州岛汉拏山白鹿潭、首都圈大企业工厂，其中也将考虑优先安排参访曾于9月随同文在寅总统访问平壤的韩国三星电子等。
 <br/>
 <br/>
 青瓦台和执政党共同民主党还打算安排金正恩访问国会。
 <br/>
 <br/>
 消息人士说，由于金正恩对经济开发颇表关心，韩国还考虑安排参观铁路相关日程。此前，金正恩曾表示想乘坐韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高铁KTX.html" target="_blank">
  高铁KTX
 </a>
 。
 <br/>
 <br/>
 金正恩回访韩国，朝鲜最关注的部分是警卫和安全问题。一名青瓦台官员表示，金正恩的个人警卫队就有超过100人，估计访韩时难以秘密行动，另一方面，韩国保守团体也有可能趁金正恩访韩期间示威，朝鲜方面对此感到担心。
 <br/>
</div>
