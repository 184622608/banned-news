### 12月3日全球看中国
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】下面是今天的全球看中国。
 <br/>
 <br/>
 1. 德国之声
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 策略政策研究所名为《采花酿蜜》的报告指出，中共从2006年开始，将2500多名隐瞒军人身份的学者送到美、英、加、澳、德五国的顶尖大学深造，并通过设立海外党支部来确保他们会如期归国，将所学用于中共的军事科技。
 <br/>
 <br/>
 2. 自由亚洲电台
 <br/>
 <br/>
 美国参议院外交委员会最近通过了《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西藏旅行对等法.html" target="_blank">
  西藏旅行对等法
 </a>
 》，要求中共政府允许美国记者、外交官和游客不受限制地去西藏报道或旅行。法案由民主党众议员麦高文，总统签字后便可生效。
 <br/>
 <br/>
 3. 大纪元时报
 <br/>
 <br/>
 中共与巴拿马建交后，打算在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴拿马运河.html" target="_blank">
  巴拿马运河
 </a>
 入口处的阿马多半岛设立大使馆。但因美国施压，巴方取消了在半岛上原定给中共的用地。此外，中共在巴拿马市使领馆区的建馆计划也被华府反对。
 <br/>
 <br/>
 4. 明慧网
 <br/>
 <br/>
 中共借扫黑绑架修炼〝真、善、忍〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 ，包括八旬老人。仅十一月，就至少有31人被非法判刑，48人被非法庭审，分布全国十八个省、自治区、直辖市。法庭非法罚金和警察抢劫勒索超过十八万元。
</div>
