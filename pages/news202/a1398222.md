### 大篷车移民筋疲力竭 自尊被玩弄情绪不稳
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】大批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲移民.html" target="_blank">
  中美洲移民
 </a>
 即将抵达美国南部边界，为了防止移民非法越境，美军已沿边界拉起铁丝网。而这些中美洲移民大部分已经筋疲力竭，前方是军队等待，自尊又被主办单位玩弄，让他们渐失耐心且情绪不稳。
 <br/>
 <br/>
 目前已有4批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 大规模进入墨西哥，截至周日，总计已超过12,000人。美国国土安全部上周发表声明称，向美国进发的大篷车移民来自至少20个国家，其中270余人有犯罪前科。
 <br/>
 <br/>
 美国总统川普（特朗普）上周表示将派1.5万名美军驻守美墨边境，并且宣布调整庇护政策，要求向美国政府申请庇护的移民必须在边境口岸合法地亲自递交申请。新规不再允许移民入境后提交申请即可离开，然后借等待听证的时间而在美国立足。
 <br/>
 <br/>
 这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲移民.html" target="_blank">
  中美洲移民
 </a>
 面临两难，移民跋涉千里，双脚肿胀且脚底磨出水泡，不少人筋疲力竭，而前方是军队等待，自尊又被主办单位玩弄，让他们渐失耐心且情绪不稳。
 <br/>
 <br/>
 据悉，移民3日在墨西哥维拉克鲁兹州(Veracruz)的3处地点休息。移民组织者原本向他们保证会有巴士可搭，但一觉醒来发现期待落空，许多人的情绪已近崩溃。
 <br/>
 <br/>
 跟着夫婿及两个年少孩子远征的36岁赛拉．卡布瑞拉(Saira Cabrera)说，人们很生气也很困惑。
 <br/>
 <br/>
 20岁的裴瑞兹(Gerardo Perez)则说，〝他们玩弄我们的自尊，昨晚说会有巴士接驳时，大家都很高兴，没想到一早醒来却希望落空。〞
 <br/>
 <br/>
 墨国维拉克鲁兹州州长尤尼斯(Miguel Angel Yunes)称，墨西哥市的水源系统正在整修，700万人周末无水可用，把大批移民送到那里是不对的。因此拒绝提供交通输运。
 <br/>
 <br/>
 墨国内政部表示，上月19日入境的首批移民中，约有3000人向当局申请难民，另有数百人返家。
 <br/>
 <br/>
 川普周日（11月4日）表示，一些中美洲国家领导人可能盗用了外援，他誓言大幅削减给这些国家的援助，以应对日益膨胀的移民大潮。
