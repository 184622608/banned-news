### 遏中共扩张 东盟决定明年与美在南海联合军演
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】东南亚国家联盟（东盟）10国本周五在新加坡举行了第12届国防部长年会。这次会议决定，东盟各国明年将与美国在南海进行军事演习，以压制中共在南海的势力扩张。
 <br/>
 <br/>
 当地时间10月19日，新加坡、印尼、越南、老挝、汶莱、泰国、缅甸、菲律宾、柬埔寨、马来西亚等10国，在新加坡召开国防部长年会（ADMM）并签署了联合声明。
 <br/>
 <br/>
 这份声明承诺，东盟各国政府将致力于促进地区的和平与稳定，并在反恐、空中安全、海上安全和网络安全等方面加强相互的合作，与会各国并决定2019年将与美国一起在南海举行联合军事演习。不过，南海联合军演的确切地点与时间还有待讨论。
 <br/>
 <br/>
 台媒报导称，东盟各国原拟在本月与中共军方在广东省湛江外海进行军演，但部份成员国因为与中方在南海主权问题上有争议反对与中共军方举行联合军演。这次更通过国防部长年会作出新的决议，敲定明年与美国举行联合军演以遏制中共在南海的势力扩张。
 <br/>
 <br/>
 东盟轮值主席国新加坡的国防部长黄永宏日前还对外表示，若美国能与东南亚国协联合军演，可增进美方在此区域的角色份量。
 <br/>
 <br/>
 据了解，东盟各国还将于10月20日继续举行国防部长扩大会议，除已经与会的上述成员国外，澳大利亚、新西兰、印度、日本、韩国、俄罗斯、美国、中国等8国的国防部长也将参与这次扩大会议。
 <br/>
 <br/>
 资料显示，东南亚国家联盟简称东盟，是集合东南亚区域国家的一个政府性国际组织。东盟最初成立于冷战时期，其成立的初期主要任务之一就是防止区域内共产主义势力扩张。东盟的成员国之间的合作都侧重于军事安全与政治中立。冷战结束后，东南亚各国政经情势发生了一些变化，东盟也接纳了越南加入。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
