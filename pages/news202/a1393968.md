### 央视记者英国闹场 中共反要英国道歉
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 官媒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视记者.html" target="_blank">
  央视记者
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孔琳琳.html" target="_blank">
  孔琳琳
 </a>
 ，在英国保守党年会上闹场，并殴打会务人员。但央视和中共外交部门反而要求英方道歉。英国保守党人权委员会副主席批评，这是荒谬至极的霸凌行为。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视记者.html" target="_blank">
  央视记者
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孔琳琳.html" target="_blank">
  孔琳琳
 </a>
 ，在英国保守党年会上闹场，并袭击会务人员。
 <br/>
 <br/>
 还原事件，9月30日，保守党人权委员会和〝香港监察〞，在英国伯明罕举办论坛，讨论香港法治现状。
 <br/>
 <br/>
 事发时，保守党人权委员会副主席罗杰斯正在致辞。他说，他亲中国，希望中国和中国人民成功，但他批评中国当前的政府以及它对待人民的方式。他呼吁保障一国两制在香港的实行。
 <br/>
 <br/>
 这时，央视记者孔琳琳起身高叫，宣称罗杰斯〝撒谎〞〝反华〞。
 <br/>
 <br/>
 由于她在非提问环节喊叫，干扰会议进程，被请离会场。当会务人员接近时，孔琳琳拒绝离开，并突然出手打人。
 <br/>
 <br/>
 原人民日报记者程凯认为，这些行动早已超出记者权利范围。
 <br/>
 <br/>
 前《人民日报》驻深圳首席记者 程凯：中国严格的讲，是没有〝记者〞这个行业的，只有共产党的宣传喉舌的工作人员。就算她是个记者，她也离开了记者的职责。如果按照记者的标准要求，采访必须是中立的是客观的，你不是要参加人家的辩论发言，她本身是不该在会上发言的。
 <br/>
 <br/>
 唐浩：在西方社会有个很重要的概念，叫做〝沟通理性〞与〝公共领域〞。但是目前这些案例，当事人都不是正常的理性发言与沟通，而是霸道的破坏公共规矩、扰乱社会秩序，甚至伤害他人，不管自己在不在理，都要用尽手段压倒别人，达成自己〝一言堂〞的目的。这一点正好跟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的言行如出一辙。
 <br/>
 <br/>
 据英国BBC报导，孔琳琳涉嫌普通袭击罪被捕。目前获准保释，但调查还在进行。
 <br/>
 <br/>
 而央视宣称，孔琳琳被无指控释放，并称她是〝正常履职〞。中共驻英使馆称，孔琳琳被剥夺言论自由，要求会议组织方道歉。
 <br/>
 <br/>
 夏小强：中共如今在中美贸易战的重压之下，处于内忧外患，政权遭遇危机，这个时候他们会采取所有方式，来挑动中国民众的民族情绪，转嫁危机。第二，由于中共现在在意识形态等各方面全面左转，各个部门和个人都抱着宁左勿右的态度，央视（和外交部）力挺央视记者，也是中共整体左转之下，表现出来的一种行动。
 <br/>
 <br/>
 程凯：自从美中贸易战以后延伸到意识形态领域，中共的大外宣已经处于守势了。所以这时候，很多负责这方面的官员，更要表现一种强硬的态度，以对国内进行效忠。
 <br/>
 <br/>
 有中国网民批评，中共的外交政策是〝对流氓国家撒钱，对文明国家撒泼〞，在英国以言论自由的名义闹场，在国内却强迫追寻自由的人上央视认罪。
 <br/>
 <br/>
 罗杰斯认为，中共表现出〝越来越高的侵略与霸凌，甚至超越边境〞。他呼吁英国与中共外交交涉，如果中共官方人员坚持这种做法，呼吁未来限制中共官方人员参与保守党活动。
 <br/>
 <br/>
 程凯：中共野心勃勃地试图把它的意识形态强加给全世界。我想未来，随着全世界都对中共这种对外的意识形态侵袭的行动进行反击，它将面临被全世界驱赶的局面。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
