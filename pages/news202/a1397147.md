### 埃博拉疫情扩散 刚果抵制措施已164人死
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】最近两个月，非洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刚果民主共和国.html" target="_blank">
  刚果民主共和国
 </a>
 （RDC）东部的埃博拉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 持续恶化。由于当地社区抵制预防措施，这波疫情在10月中严重扩散，至今已造成164人死亡。
 <br/>
 <br/>
 《法新社》报导，刚果当局10月中旬曾表示，刚果东部靠近乌干达边界的地区、北基伍省（North Kivu）小镇贝尼（Beni）爆发了第二波埃博拉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 （Ebola virus）。先前疫情集中在距离贝尼20公里的曼吉纳（Mangina）。
 <br/>
 <br/>
 但刚果卫生官员上周五（26日）表示，东部地区爆发的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃博拉疫情.html" target="_blank">
  埃博拉疫情
 </a>
 （Ebola virus）已经导致164人死亡。
 <br/>
 <br/>
 自从疫情8月初爆发以来，刚果埃博拉疫区已经记录了257个病例，其中有222例确诊，35例则被认为很可能。世界卫生组织（WHO）10月17日表示，虽然死亡病例达到139例，但仍不足以将之定为全球健康紧急状况。
 <br/>
 <br/>
 报导称，爆发第二波疫情的一个主要原因，是当地社区对防治措施有抵制。上周四（25日），贝尼就大约有1000名学生上街游行，发起对抗埃博拉运动。
 <br/>
 <br/>
 刚果卫生部说，自从8月8日开始疫苗接种以来，现在已经有2万2千人接种了疫苗，其中贝尼地区就有1万1328人。
 <br/>
 <br/>
 欧洲疾病预防控制中心（ECDC）评估：虽然截至10月26日在邻国没有确认病例，但在与卢旺达和乌干达有重要跨境人口流动的地区爆发疫情的事实仍令人担忧。此外，实地应对措施的实施仍然具有挑战性，因为疫情爆发区长期处于人道主义危机，近来暴力事件与武装袭击事件激增，造成安全局势很不稳定。
 <br/>
 <br/>
 ECDC称，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刚果民主共和国.html" target="_blank">
  刚果民主共和国
 </a>
 受埃博拉病毒病影响的地区居住或旅行的欧盟、欧洲经济区公民，只要遵守建议的预防措施，接触该疾病的可能性很低。
 <br/>
 <br/>
 资料显示：埃博拉病毒是丝状病毒科的其中一种病毒，可导致感染者出现恶心、呕吐、腹泻、肤色改变、全身酸痛、体内出血、体外出血、发烧等症状，而罹患埃博拉出血热的人大多会因为中风、心肌梗塞、低血容量性休克或器官衰竭而死亡。
 <br/>
 <br/>
 埃博拉是人畜共通病毒，主要的感染途径是透过患者体液传染，如血液、汗、呕吐物、排泄物、尿液、唾液或精液等，目前并无飞沫感染的证据。尽管世界卫生组织苦心研究，至今仍没有辨认出任何有能力在爆发时存活的动物宿主。
 <br/>
 <br/>
 由于这种病毒1976年首次爆发的部落就在非洲刚果境内的埃博拉河附近，因此被称为埃博拉病毒，该病毒最近4次爆发的所在地也在埃博拉地区。
 <br/>
 <br/>
 因埃博拉的高致命力，加上目前世界上还没有任何疫苗被证实有效，埃博拉被列为生物性危害第四级病毒，也同时被视为是生物恐怖主义的工具之一。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
