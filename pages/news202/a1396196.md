### 沙特上演谍战片 身亡记者背景复杂
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】
 <b>
  新闻周刊
 </b>
 （650）在沙特驻土耳其领馆神秘失踪的记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 ，目前已证实
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_身亡.html" target="_blank">
  身亡
 </a>
 。此事件备受外界关注。美国总统川普表示，当前的情况很复杂。
 <br/>
 <br/>
 为办理跟前妻离婚以及再婚需要的手续。9月26日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 第一次进入沙特驻土耳其伊斯坦布尔领事馆。
 <br/>
 <br/>
 10月2日，再次进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特领馆.html" target="_blank">
  沙特领馆
 </a>
 的卡舒吉却没有出来，他的土耳其籍未婚妻稍后向土耳其官方报案。
 <br/>
 <br/>
 卡舒吉失踪事件逐渐引发国际关注。沙特官方因坚持否认知情，受到西方主流媒体多方质疑。
 <br/>
 <br/>
 15日，美国总统川普和沙特国王通电话，16日，国务卿蓬佩奥前往沙特，会见了沙特国王和王储萨勒曼，重申美国对卡舒吉失踪的担忧。
 <br/>
 <br/>
 稍后，蓬佩奥又与土耳其总统及外长会面，沟通相关事态。
 <br/>
 <br/>
 当天，土耳其当局首次进入沙特领馆进行9小时搜查。
 <br/>
 <br/>
 19日晚，白宫发言人桑德斯在推特上发表声明，表示认可沙特官方的调查取得进展，将密切关注进一步的调查结果，并支持以公开、透明、合法的程序找出真相。
 <br/>
 <br/>
 20日，沙特阿拉伯电视台宣读了官方声明，称卡舒吉已
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_身亡.html" target="_blank">
  身亡
 </a>
 ，死因是在沙特领馆内与人搏斗。
 <br/>
 <br/>
 声明中说，沙特当局把此事当作最高级别的大事处理，已经就此事逮捕了18人，另有2名高级官员被解职。
 <br/>
 <br/>
 此前，有消息传出卡舒吉曾遭酷刑折磨并遭到肢解，不过沙特的声明中并未对此做出解释。
 <br/>
 <br/>
 美国总统川普直言，当前的情况很复杂。
 <br/>
 <br/>
 美国总统 川普: 〝他们（沙特）是美国在中东一个很好的盟友，我们需要他们来牵制伊朗，所以没有一个最简单的解决方案，目前的情况也很不简单。〞
 <br/>
 <br/>
 有报导指，卡舒吉的身世背景相当复杂。他不仅曾是沙特阿拉伯的知名记者，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特王室.html" target="_blank">
  沙特王室
 </a>
 有着广泛的人脉，还一度为政府高官担任顾问。不过他后来成为异见人士，批评王储
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_萨尔曼.html" target="_blank">
  萨尔曼
 </a>
 和沙特政府。2017年出走美国，生前任职《华盛顿邮报》专栏记者。
 <br/>
 <br/>
 最为外界关注的是，卡舒吉曾是基地恐怖组织头目本・拉登的密友，对本・拉登抱有极大好感，并发表过对其的一系列采访报导。
 <br/>
 <br/>
 很多沙特媒体指出，卡舒吉与沙特情报部门有着错综复杂的关系。他当时受情报机构委托，试图劝服本・拉登与沙特王室修好。外界据此推测，卡舒吉手中很可能握有9・11恐怖袭击事件的一些内幕。
 <br/>
 <br/>
 <br/>
 撰稿：梁东 剪辑：杨月
