### 英脱欧谈判触礁 爱尔兰边界问题成焦点
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国.html" target="_blank">
  英国
 </a>
 正式脱欧还有不到6个月时间，英国脱欧事务大臣蓝韬文与欧盟首席谈判代表巴尼耶，周日（14日）于比利时布鲁塞尔会面，但谈判尚未达成协议。双方最大的障碍可能是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱尔兰边界.html" target="_blank">
  爱尔兰边界
 </a>
 问题。
 <br/>
 <br/>
 双方决定在周三（17日）欧盟峰会召开前搁置谈判。有官员表示，欧盟方面的时间表也因而有所调整。
 <br/>
 <br/>
 蓝韬文（Dominic Raab）表示，谈判取得进展，但在制定担保方案上仍有部分未解决的问题。
 <br/>
 <br/>
 巴尼耶（Michel Barnier）则表示，尽管〝极度努力〞，但双方在部分重要议题上仍未有共识。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国.html" target="_blank">
  英国
 </a>
 脱欧后，如何免除欧盟成员国爱尔兰与英国北爱尔兰（Northern Ireland）之间的边境检查，是双方最大的分歧之处。
 <br/>
 <br/>
 自1998年英爱政府签订了和平协议后，北爱尔兰才逐步走向和平的进程。如今若北爱尔兰伴随着英国脱欧而离开欧盟，则其退出关税同盟和欧洲单一市场的同时，必然需要在与爱尔兰共和国的边境设置〝硬边界〞。这显然是爱尔兰共和国和北爱尔兰都无法接受的。
 <br/>
 <br/>
 所以，脱欧协议要求双方采取具体行动维护协议顺利实施，避免发生 〝硬边界〞问题。但是一些条款还涉及英国和欧盟贸易谈判的内容，所以只能放在英国正式脱欧后来完成，包括关税和贸易协议的制定等。
 <br/>
 <br/>
 脱欧条约中的〝后备〞计划将规定怎样避免边界摩擦的问题，直到英国脱欧之后找到了替代的解决方案。
 <br/>
 <br/>
 欧盟方面表示，它需要在岛上设立一个〝单一监管区域〞，并通过爱尔兰海的海关进行监管。但英国首相特蕾莎．梅之前表示，这对英国这样的主权国家来说是〝不可接受的〞。
 <br/>
 <br/>
 欧盟峰会举行前夕，欧盟其余27个成员国大使周日也都另外出席会议。巴尼耶将于周三晚宴上向其余27个成员国的领袖简报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧谈判.html" target="_blank">
  脱欧谈判
 </a>
 进展。
 <br/>
 <br/>
 距离正式脱欧的日期越来越近，英国不但看不到万众一心迎接挑战的迹象，反而是内讧不断升级。前外长约翰逊近日又向特蕾莎．梅开火，称她的脱欧方案将给英国带来危险，首相对此则表示愤怒。英国和欧盟的关系也被外交大臣亨特火上浇油，他将欧盟比作苏联，招致众怒。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
