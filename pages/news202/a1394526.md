### 法轮功首次瑞士苏黎世游行 民众支持
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】上周五（10月5日），来自欧洲的部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员首次在瑞士最大的城市苏黎世，举行了盛大的集会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_游行.html" target="_blank">
  游行
 </a>
 ，呼吁人们共同制止中共对法轮功学员的迫害。在工作日得到这样的游行许可在苏黎世是非常罕见的，民众在对法轮功学员表达支持的同时也鼓励中国人要从共产主义的悲剧中突破出来。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_游行.html" target="_blank">
  游行
 </a>
 的队伍从苏黎世海尔维广场（Helvetiaplatz）出发，穿过市中心，沿着利马特河畔，一直走到市政府公园。欧洲天国乐团嘹亮的乐声让路人们停下了脚步。
 <br/>
 <br/>
 活动主办方发言人 Manu Huwyler ：〝通过这次大游行我们想告诉全世界以及苏黎世的人们，在中国究竟发生了什么，想告诉他们这场持续了十九年之久的对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 的迫害。〞
 <br/>
 <br/>
 活动策划人 Erich Bachmann: 〝对法轮功的迫害没有任何法律依据。即使在中国境内，依然有成千上万勇敢的人提交了对江泽民的诉讼，我们呼吁国际社会做出回应。我想这是结束这场迫害的关键。〞
 <br/>
 <br/>
 周五是工作日，瑞士人都知道，工作日得到这样的游行许可在苏黎世是非常罕见的。
 <br/>
 <br/>
 专员 Shala Nexhmije：〝我觉得这个活动很棒，因为人们都会说，啊哈，原来是这么回事。〞
 <br/>
 <br/>
 中学教师 Elsa Höhn：〝我希望所有遭受迫害或牵扯进去的人，能够足够坚强地挺过来。我想对中国人说，保持开放的态度，保持清醒，睁开眼睛，竖起耳朵，看看发生了什么事情！别只是听中共的一面之词。
 <br/>
 <br/>
 比利时大学生 Lauren Bouden：〝中国人不应该害怕拥有自己的想法。重要的是思考人生中什么是重要的。〞
 <br/>
 <br/>
 来自波兰的Dejcz女士对共产主义有亲身体会。
 <br/>
 <br/>
 护士 Dejcz Megi：〝我觉得这是一场悲剧。共产主义就是一个悲剧。它在整个世界，在欧洲，当时在波兰所做的，都是悲剧。保持坚强！继续抗争。波兰也从共产主义之中突破出来了。也许你们也能做到。〞
 <br/>
 <br/>
 平和的集会和游行引起了人们对法轮功的关注。沿途路人们拿出手机记录这一特殊的场景，有的仔细阅读拿到的传单，想更多的了解法轮功。
 <br/>
 <br/>

  瑞士苏黎世
 </a>
 报导
</div>
