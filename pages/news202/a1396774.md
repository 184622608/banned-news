### 乌克兰真善忍美展  游客赞叹感佩
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】周二，〝真善忍国际美展〞在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 第二大城市敖德萨开展，吸引传统绘画艺术爱好者前来观赏。目前，〝真善忍国际美展〞已经在全球80个国家举办了1,000多场展览。
 <br/>
 <br/>
 真善忍国际美展的作品，大多由中国艺术家创作。画作展示了内心精神境界，也揭露了侵害人权的悲剧。
 <br/>
 <br/>
 参展作品都是风格独特的现实主义作品，画面清晰，主题深刻，展示人类对最真理的追求，以及善恶的较量。
 <br/>
 <br/>
 美展主办方 Svetlana Vetryak：〝艺术家们希望复兴道德，恢复传统，使人感受到生活的和谐，寻求善良，忍受生活中的考验。〞
 <br/>
 <br/>
 这次参展的15位艺术家都是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 修炼者，画展以法轮大法的主要修炼原则〝真、善、忍〞命名。作品有些来自画家的亲身经历，有些揭露了中国的现状。
 <br/>
 <br/>
 艺术史学家Irina Timokhova：〝（画作）的结构，面前的天堂和天空表现得很完美，突然，出现了可怕的黑暗，比如盖世太保，集中营，噩梦和地下室。完全不同的画面，然后画面中光明再现。〞
 <br/>
 <br/>
 画展吸引大量参观者，对画作赞不绝口。
 <br/>
 <br/>
 画展游客Tatiana Pelikhovskaya：〝有了一些新想法。我也想修炼。我意识到需要多行善事，才能由内而外展现美好。〞
 <br/>
 <br/>
 画展游客Anatoly Tsehovich：〝画作真实感人，有立体感，走在展厅，无论站在哪儿，他们都在看着你。我喜欢这些画。〞
 <br/>
 <br/>
 一些作品揭示了1999年，中共前党魁江泽民镇压，使法轮功修炼者的孩子失去亲人，成为孤儿；有些作品展示了修炼者在残酷迫害中，对信仰的坚定。
 <br/>
 <br/>
 画展主办方Svetlana Vetryak：〝画作中表现的都是光明和希望，还有修炼人的慈悲。对他（被迫害的法轮功学员）来说，身体折磨并不可怕，信仰使他重生，不会屈服于迫害。〞
 <br/>
 <br/>
 敖德萨的〝真善忍国际美展〞将持续到10月31日。
 <br/>
 <br/>
 新唐人记者安娜、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 敖德萨报导
</div>
