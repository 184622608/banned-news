### 应对周末抗议 法国部署九万警力 店铺关门
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】为了应对周末的抗议活动，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 政府在全国部署近九万
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警力.html" target="_blank">
  警力
 </a>
 ，并启用装甲车。巴黎大部分地区，包括埃菲尔铁塔在内的很多景点也将关闭。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 总理爱德华・菲利普宣布，周六，全国将部署8万9千名安全部队，巴黎部署8千人。
 <br/>
 <br/>
 法国总理爱德华・菲利普：〝我们会使用国家宪兵队十几辆装甲车，一些场合会用到。〞
 <br/>
 这是法国政府2005年以来，第一次在城市使用装甲车。防止上周六焚烧汽车和抢劫商店的暴行重演。
 <br/>
 <br/>
 周五，法国内政部部长卡斯塔纳表示，〝黄背心〞活动偏离了初心，暴徒混入其中。政府会尽量保护反对暴力的抗议者。
 <br/>
 <br/>
 法国内政部部长卡斯塔纳：〝会使用装甲车。这些不是军队，不是坦克。是专门的国家警车，特别是用来清理路障。〞
 <br/>
 <br/>
 为应对周六抗议，巴黎大部分地区将会关闭，包括埃菲尔铁塔和卢浮宫在内的数十个景点，以及香榭丽舍大道上的奢侈品商店等。
 <br/>
 <br/>
 香榭丽舍大道报亭店主JOSE RUSSO：〝是的，周六我会关门，这是肯定的。〞
 <br/>
 <br/>
 上周六小型零售商的收入减少了20%至40%，酒店预定率下降了15%至25%。
 <br/>
 <br/>
 周五，数百名法国高中生进行抗议，在巴黎共和广场（Place de la Republique）焚烧垃圾箱，但没有其他暴力行为。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
