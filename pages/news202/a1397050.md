### 第31届东京电影节开幕 红毯群星璀璨
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】10月25日在东京六本木举办了第31届东京国际电影节的开幕式，从来自109个国家和地区的1829部作品中选出了16部作品将角逐本届电影节最佳影片
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金麒麟奖.html" target="_blank">
  金麒麟奖
 </a>
 。
 <br/>
 <br/>
 记者现在位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东京电影节.html" target="_blank">
  东京电影节
 </a>
 的会场六本木。开幕式的红毯秀即将拉开这场光影盛宴的帷幕，下面就让我们通过星光大道的红毯秀一睹明星的风采。
 <br/>
 <br/>
 首先登上红毯的是担任本次电影节大使同时也是演员的松冈茉优。她对于获邀担任大使感到荣幸同时希望更多人能够爱上电影。
 <br/>
 <br/>
 松冈茉优: 〝集合了国际间丰富的元素，汇集了各种各样的电影，我希望这些能成为大家喜欢电影的契机〞
 <br/>
 <br/>
 日本影星杉野希妃、柳英里纱、广濑阿丽斯、福士苍汰、役所广司也陆续踏上红毯。中国影星王丽坤，海清，徐峥也纷纷亮相红毯。
 <br/>
 <br/>
 徐峥: 〝我觉得我们不会去想这个电影能够给你带来什么样的名啊利啊这些，其实我觉得创作还是要本着一个创作的初心就是你想表达什么，当你表达了以后你就希望更多的跟观众交流。〞
 <br/>
 <br/>
 入围了本届电影节〝亚洲未来〞单元的中国影片《武林孤儿》讲述了在一所武术学校接受功夫训练的少年和他们的老师之间的一段不同寻常的经历。
 <br/>
 <br/>
 《武林孤儿》的女主角刘芷含：〝特别的紧张而且特别的意外，没有想到黄璜导演作为一个新人导演，我们所有的这些演员都是新人，然后一下通过很短时间的拍摄就进入了东京电影节，虽然没有进入主竞赛单元，但是让大家看到了中国内地的学院派的电影是一个什么样的水准。〞
 <br/>
 <br/>
 进入主竞赛单元的中国电影《诗人》剧组也在红毯上亮相。
 <br/>
 <br/>
 女主演宋佳：〝我是觉得本身入围就是一个很大的荣誉了，就像你刚才说的我都不知道是1800多部，所以我觉得我们又能够代表中国内地，我觉得这是一个无上的荣誉。〞
 <br/>
 <br/>
 《诗人》讲述了在矿场工作的业余诗人李五（朱亚文 饰）和妻子陈蕙（宋佳 饰）为帮助丈夫实现诗人梦想，而谱写出一部曲折凄美的爱情故事。
 <br/>
 <br/>
 前SMAP成员稻垣吾郎携他的作品《半世界》进入竞赛单元，他也相当期待能获得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金麒麟奖.html" target="_blank">
  金麒麟奖
 </a>
 。
 <br/>
 <br/>
 稻垣吾郎: 〝当然参展了就希望能得奖，同时也希望藉此机会让更多的人知道这部电影，使更多人能去电影院去看这部电影。〞
 <br/>
 <br/>
 东京国际电影节是亚洲最大的国际A类电影节。本届电影节金麒麟奖将于11月2号晚间揭晓。
 <br/>
 <br/>
 新唐人记者文萱日本东京报导
</div>
