### 马其顿通过修宪案  国名改为〝北马其顿〞
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】东南欧国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马其顿.html" target="_blank">
  马其顿
 </a>
 ，周五通过了议会修宪投票，即将正式更改
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国名.html" target="_blank">
  国名
 </a>
 ，为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北马其顿.html" target="_blank">
  北马其顿
 </a>
 〞，为加入北约和欧盟铺平道路。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马其顿.html" target="_blank">
  马其顿
 </a>
 议会周五举行修宪投票，120名议员中的80名投下赞成票，正好达到三分之二的修宪门槛。
 <br/>
 <br/>
 马其顿议会议长 贾菲里：〝我宣布，议会决定启动修改宪法的投票，获得80票赞成，达到了决议通过的法定人数。〞
 <br/>
 <br/>
 这次修宪投票，是马其顿修改
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国名.html" target="_blank">
  国名
 </a>
 程序的最后一次投票。最迟从明年一月开始，马其顿共和国就将更名为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北马其顿.html" target="_blank">
  北马其顿
 </a>
 共和国。
 <br/>
 <br/>
 马其顿总理 扎埃夫：〝今天对我国是历史性的一天，我们写下了历史。〞
 <br/>
 <br/>
 此前，马其顿与南方邻国希腊，一直存在国名的争议。由于希腊北部也有马其顿省，希腊认为古代马其顿隶属于希腊文化，因此阻拦马其顿以当前的国名加入北约和欧盟。
 <br/>
 <br/>
 今年6月，马其顿和希腊达成协议，同意更名为〝北马其顿〞，而希腊则不再阻止〝北马其顿〞加入北约和欧盟。
 <br/>
 <br/>
 此次修宪，为马其顿日后加入北约和欧盟铺平了道路。
 <br/>
 <br/>
 新唐人记者林宇综合报导
