### 今年最惨重攻击 阿富汗自杀炸弹逾50死80人伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 官员周二（20日）表示，在首都喀布尔众多高阶神职人员出席的一个宗教活动中，发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀炸弹.html" target="_blank">
  自杀炸弹
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_攻击.html" target="_blank">
  攻击
 </a>
 。已知逾50人丧生，80多人受伤。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 卫生部发言人马杰罗（Wahid Majroh）表示，这次事件锁定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_攻击.html" target="_blank">
  攻击
 </a>
 出席纪念活动的宗教领袖。
 <br/>
 <br/>
 喀布尔警方发言人穆贾希德（Basir Mujahid）表示，一名男性
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀炸弹.html" target="_blank">
  自杀炸弹
 </a>
 客趁活动正在进行之际，在大厅内引爆了身上的炸弹。
 <br/>
 <br/>
 当时活动大厅内聚集的人众数以百计，另有活动场地的工作人员说，自杀炸弹客引爆炸弹时，大厅内约有1000人。除了已知有超过50人丧生外，至少83人受伤，其中有24人命危。
