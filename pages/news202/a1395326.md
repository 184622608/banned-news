### 诡异暴风雪夺人命 尼泊尔5韩人4向导罹难(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼泊尔.html" target="_blank">
  尼泊尔
 </a>
 12日发生近年来最致命登山事故之一，5名韩国人和4名尼泊尔向导在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_格加山.html" target="_blank">
  格加山
 </a>
 遭猛烈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雪.html" target="_blank">
  暴风雪
 </a>
 袭击，9人全数罹难。另有一名与登山队分开行动的丁姓韩籍登山者因路途中身体不适先行下山，没想到竟成为唯一幸存者。专家称，这支登山队经验深厚，为何会发生这种事，实在让人难以理解。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼泊尔.html" target="_blank">
  尼泊尔
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_格加山.html" target="_blank">
  格加山
 </a>
 （Mount Gurja）海拔7913公尺，事发在海拔3500公尺处的营地。搜救团队13日最初通报，仅在这支登山队营地附近发现4名韩国人和4名尼泊尔向导的遗体，未发现另一名韩籍登山客，后来证实暴风来袭时他人在营地，同样罹难。
 <br/>
 <br/>
 据了解，9人的尸体在营地附近被发现，其中，包含另一名与登山队分开行动的丁姓韩国籍登山者。丁男原本与向导以及另一名崔姓男子一同往营地移动，但崔男在路途中因身体不适先行下山，成为此团的唯一幸存者。
 <br/>
 <br/>
 《韩联社》报导，丁男是韩国山岳会理事并未随同登山队一同上山，而是另外与崔姓顾问押后，9日在抵达营地前，崔男出现不适症状，因此放弃登山，先行折返，并相约于11日在古尔加卡尼村庄重新会合。
 <br/>
 <br/>
 但过了11日预定的午餐时间后，丁姓理事与登山队队员却迟迟未出现，并失去联系，崔男紧急与远在韩国的山岳会代表联系，并判断情况不乐观，因此两人决定委托另一名尼泊尔籍向导与5名当地人上山查看。
 <br/>
 <br/>
 一行人到达营地后，发现营地物品都被吹得四散，9个人被大雪覆盖已罹难。尼泊尔警方在13日上午投入的救难直升机进行搜索任务，但却因风雪过大，无法降落，只好折返。
 <br/>
 <br/>
 目前韩国外交部正与尼泊尔政府积极配合，希望能尽速将9人的遗体送下山。
 <br/>
 <br/>
 据了解，率领这支韩国登山队现年51岁的的金昌浩（Kim Chang-ho），曾于2013年创下未佩戴氧气装备最快登上全球14大高峰的纪录。
 <br/>
 <br/>
 对于，这场诡异暴风夺走这么多条人命令经验丰富的登山专家费解，协助搜救任务的美国全球救援（Global Rescue）成员李查兹（Dan Richards）表示，这场诡异
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雪.html" target="_blank">
  暴风雪
 </a>
 发生海拔相对低，而这支登山队〝经验深厚，怎么还会发生这种事，实在让人难以理解〞。
 <br/>
 <br/>
 登山经验丰富的德国登山客比尔陵（Billi Bierling）也说：〝当时他们位置并不高，发生这种事实在罕见。〞
 <br/>
 <br/>
 这场反常风暴是2015年强震引发圣母峰山崩、夺走基地营18条人命以来，尼泊尔登山业最致命事故。
