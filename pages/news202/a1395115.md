### 土耳其拘押美国牧师今受审 面临35年徒刑或当庭释放
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】引发美国与土耳其外交危机的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牧师布朗森.html" target="_blank">
  牧师布朗森
 </a>
 ，今(12)日受审，在要求将他释放并准许返家的施压升高之际，外界认为土耳其可能放人。
 <br/>
 <br/>
 综合媒体报导，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牧师布朗森.html" target="_blank">
  牧师布朗森
 </a>
 （Andrew Brunson）遭土耳其当局指控参与恐怖行动，在土耳其已被拘押2年，自7月底以来改遭居家软禁，但仍禁止出境。
 <br/>
 <br/>
 布朗森遭拘押不只造成土美之间近来最严重的外交争端之一，还造成土耳其货币里拉崩盘,，暴露出土耳其经济的脆弱。
 <br/>
 <br/>
 土耳其司法当局此前不断驳回释放布朗森的请求，但此次布朗森若被判有罪，他将因被控协助恐怖团体及从事间谍活动，面临达35年有期徒刑。此举将引发来自华府和金融市场的反弹，将对土耳其带来重大伤害。
 <br/>
 <br/>
 对此，观察家认为有愈来愈多迹象显示，布朗森可能在今天获释。
 <br/>
