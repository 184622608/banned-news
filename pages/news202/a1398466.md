### 时事拼盘：一战停战纪念  巴基斯坦女基督徒被无罪释放
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】下面请看一组新闻简讯
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一战.html" target="_blank">
  一战
 </a>
 停战纪念100周年之际，法国总统马克龙表示，将向一战法军总司令贝当元帅致敬，引发犹太团体和反对党的批评。贝当在二战期间向德国投降，至今在法国仍被视为叛国者，战后被判死刑，但因为一战时战功显赫，改为终身监禁。
 <br/>
 <br/>
 意大利政府周三表示，巴基斯坦女性基督徒毕比被无罪释放后，意大利将帮助她和家人离开巴基斯坦，到欧洲居住。毕比八年前因被指控渎神罪，被判死刑。上周巴基斯坦最高法院宣判她无罪，引发穆斯林极端教派大规模抗议，要求公开处决毕比。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度.html" target="_blank">
  印度
 </a>
 北部城市阿约提亚于印度节日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_排灯节.html" target="_blank">
  排灯节
 </a>
 前夕，在萨拉育河岸点燃了30万1152盏粘土油灯，历时五分钟，打破了吉尼斯世界纪录。之前的纪录是印度哈里亚纳邦在2016年9月23日创下的，当时有15万零9盏油灯同时点燃。
