### 巴黎纪念一战结束一世纪 德俄美法等国领导参加
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】周日，法国巴黎继续举行纪念
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第一次世界大战.html" target="_blank">
  第一次世界大战
 </a>
 停战100周年的活动。法国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 ，以及美、德、俄等大约70多位世界各国的首脑和领导人，出席了在凯旋门前的庄严纪念仪式。
 <br/>
 <br/>
 星期日，巴黎阴雨绵绵。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 和夫人布里吉特在爱丽舍宫外，欢迎来自全世界的70多位国家首脑和领导人，其中包括德国总理默克尔、俄罗斯总统普京和美国总统川普。
 <br/>
 <br/>
 马克龙与默克尔还于周六一起参加了在法国北部城市贡比涅（Compiegne）举行的纪念活动，并参观了1918年11月11日签署
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停战协议.html" target="_blank">
  停战协议
 </a>
 的火车车厢。默克尔是78年来第一个参观这个地点的德国总理。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第一次世界大战.html" target="_blank">
  第一次世界大战
 </a>
 从1914年爆发，至1918年11月11日上午11时在法国东北部签署的停战协议生效时结束。这场延烧全欧洲以致全世界的战争中仅阵亡士兵就达1000万人。
 <br/>
 <br/>
 周日随后在凯旋门举行的纪念仪式上，马克龙发表了二十分钟的演讲。
 <br/>
 <br/>
 马克龙说：〝让我们再一次庄重承诺和平高于一切。（让我们在座的所有政治领导者们）再次确认在自己人民面前的真正的巨大的责任，将和平传承给我们的子孙。〞
 <br/>
 <br/>
 隆重的纪念仪式虽然保安严密，但吸引了大批观众。
 <br/>
 <br/>
 法国观众LOEIZA GORBES：〝因为是100周年纪念。我很重视对战争以及围绕其一切的记忆，我想既然我有机会来，那为什么不来呢。〞
 <br/>
 <br/>
 马克龙随后将主持首届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎和平论坛.html" target="_blank">
  巴黎和平论坛
 </a>
 ，目的是寻求促进多边安全和治理方法，以便最终避免导致第一次世界大战爆发的错误。
 <br/>
 <br/>
 新唐人记者徐哲、天羽综合报导
 <br/>
</div>
