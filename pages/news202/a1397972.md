### 大篷车难民涉水入境墨西哥 美军抵边境设防
------------------------

<div class="wysiwyg">
 【新唐人2018年11月03日讯】中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 继续向美国南部边境前进。周五，又有部分来自萨尔瓦多和危地马拉的难民，从水陆两方，越过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 边境。同时，美军在南部边境部署军队，总数将超过万名士兵，保护边境安全。
 <br/>
 <br/>
 来自萨尔瓦多的难民，周五涉水越过危地马拉与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 的边境河，进入墨西哥。他们说，要去美国。
 <br/>
 <br/>
 萨尔瓦多难民 William Roque:〝我们骗过墨西哥（政府），涉水渡河过来了。〞〝我们要去美国。〞
 <br/>
 <br/>
 另有一少部分萨尔瓦多和危地马拉难民，周五抵达危地马拉与墨西哥的边境桥。
 <br/>
 <br/>
 危地马拉驻墨西哥领事对等候在边境护栏外的难民表示，要守秩序，墨西哥会放每个人进入。
 <br/>
 <br/>
 危地马拉驻墨西哥领事 Mauricio Ituarte:〝我们希望大家不要有任何骚动，对危地马拉和墨西哥政府都保持泠静和尊敬。大家排成一队，每个人都会进来。〞
 <br/>
 <br/>
 同时，人数众多的洪都拉斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 已抵达墨西哥东南部的韦拉克鲁斯州，并继续向美国南部边境前进。
 <br/>
 <br/>
 美国国土安全部周四发表声明，目前在墨西哥境内的大篷车难民，来自20多个国家，其中有黑帮背景，犯罪历史的人数，约有270多人。
 <br/>
 <br/>
 川普总统已决定向美墨边境派遣15,000名美军。周四开始，美军运输机将士兵和各种重型军事设备陆续运抵边境。
 <br/>
 <br/>
 周五，美军士兵开始在德州和墨西哥边境桥周边安置带刺铁丝网。
 <br/>
 <br/>
 川普周五在回答记者提问时表示，下周将发布行政令，全面应对当前的大篷车难民危机。
 <br/>
 <br/>
 川普再次警告，难民如果硬闯边境，像在墨西哥那样投掷石块，他已经告诉军队，将视为对美军开火。
 <br/>
 <br/>
 但美军不会对难民开枪，但是扔石块的人，要付出代价。
 <br/>
 <br/>
 川普总统：〝不，他们（军人们）不会开枪。 但我不希望这些非法入境者扔石头。〞
 <br/>
 <br/>
 美国总统 川普：〝他们对墨西哥军人的所作所为是可耻的。他们用石块打他们，有些人受伤很重，而且他们扔石头打边境人员的脸 。（如果）他们对我们的人这样做，他们将被逮捕。（闪白）他们将被逮捕关押很长时间。〞
 <br/>
 <br/>
 新唐人记者安心综合报导
</div>
