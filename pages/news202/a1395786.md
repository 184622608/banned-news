### 另类视角看贸战   瑞媒：愿贸易战天长地久
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】瑞士媒体《金融经济报》（Finanz und Wirtschaft）周二（16日）发表评论文章指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易.html" target="_blank">
  中美贸易
 </a>
 战会让双方经济都受损，但贸易链的改变却会使其他国家受益。因此，全球其他国家有理由期盼中美贸易争端〝天长地久〞。
 <br/>
 <br/>
 由欧洲政策研究中心主任格罗斯（Daniel Gros）撰写的评论文章《川普贸易战的赢家》中称，中美相争的大幕已经拉开，这会引发许多重大且不可预测的地缘政治后果。文章认为，这对于世界其他国家而言，并不一定是坏消息。
 <br/>
 <br/>
 文章进一步分析称，随着中美互相向对方加征关税，在美国市场上，欧洲企业相对就能获得对中国企业的竞争优势；而在中国市场上，欧洲企业也会获得对美国企业的竞争优势。
 <br/>
 <br/>
 文章说，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易.html" target="_blank">
  中美贸易
 </a>
 的很大一部分有可能被转移到欧洲、日本及其他亚洲国家。欧洲的获益将尤其显著，因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 将继续是美国和中国的最大外贸伙伴。〞
 <br/>
 <br/>
 文章套用中国的一句谚语〝鹬蚌相争，渔人得利〞分析认为，除了中美两国之外，世界所有经济体可能都会是这场贸易冲突的获益者，因此，〝全球其他国家有足够的理由期盼中美贸易争端天长地久〞。
 <br/>
 <br/>
 <h4>
  中美贸易冲突日趋升级
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
   新冷战
  </a>
  来临？
 </h4>
 <br/>
 中美对抗日益升级。自美国副总统彭斯（Mike Pence）本月初发表的中美关系演讲后，外界认为〝新冷战〞时代来临。美国学者也担忧中美冲突存在升级的隐患。
 <br/>
 <br/>
 战略与国际研究中心中国项目主任葛来仪（Bonnie Glaser）表示，〝我不想用冷战的标签，但中美关系的确充满争端，双方存在的问题这些年越来越多。这些问题一直无法得到成功解决。〞
 <br/>
 <br/>
 中共一直期望中期选举后，中美贸易紧张关系能有所缓和，但华府专家普遍认为，川普（特朗普）启动关税措施与国内政治状况无关，而是一项长期战略，也是两党共识。葛来仪说：〝川普政府要的是互惠公平的关系，服务于美国利益。遏制中国不是川普政府的目标，但我了解为何中共这么想。〞
 <br/>
 <br/>
 白宫首席经济顾问库德洛（Larry Kudlow）上周日（14日）也表示，中美是否已经开启〝新冷战〞，这要留给历史学家去评判，但他一直参与和中方的贸易谈判并不令人满意。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
