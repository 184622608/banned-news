### 中共驻巴基斯坦卡拉奇领事馆遭恐袭
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】周五上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴基斯坦.html" target="_blank">
  巴基斯坦
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡拉奇领事馆.html" target="_blank">
  卡拉奇领事馆
 </a>
 发生恐怖袭击事件，两名巴基斯坦警卫殉职，三名行凶枪手被击毙。
 <br/>
 <br/>
 当地时间周五上午9点15分左右，三名恐怖分子携带炸药，试图进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴基斯坦.html" target="_blank">
  巴基斯坦
 </a>
 南部城市卡拉奇的中领馆，被保安阻止。一名恐怖分子引爆炸药，恐怖分子与警卫交火，两名当地警卫殉职，三名恐怖分子被击毙。
 <br/>
 <br/>
 巴基斯坦〝别动队〞上校 MOHAMMAD FAISAL：〝今天早上（当地时间）9点15分左右，三名恐怖分子试图攻击克利夫顿第四街区的中领馆。〞
 <br/>
 <br/>
 周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 外交部发言人也证实了卡拉奇领馆恐怖袭击事件。
 <br/>
 <br/>
 报道称，两名申请签证的平民遇害。领事馆内21名中国员工没有受伤，之后被转移到更安全地带。
 <br/>
 <br/>
 当地警方随后收缴了恐怖分子的武器，并在附近进行搜查。
 <br/>
 <br/>
 巴基斯坦〝别动队〞上校MOHAMMAD FAISAL：〝我们收缴了恐怖分子的爆炸物和武器，搜查行动已经完成。〞
 <br/>
 <br/>
 巴基斯坦武装组织〝俾路支解放军〞宣称对袭击事件负责，该组织声称反对中共对巴基斯坦的资源掠夺。
 <br/>
 <br/>
 2015年，巴基斯坦前政府与中共签署了51项合作协议，多项涉及〝一带一路〞的〝中巴经济走廊〞项目。
 <br/>
 <br/>
 北京向巴基斯坦提供数百亿美元贷款，没有给巴基斯坦带来经济利益，反而让其背上钜额债务。
 <br/>
 <br/>
 周五，巴基斯坦总理伊姆兰・汗称，袭击并不〝意外〞。
 <br/>
 <br/>
 伊姆兰・汗11月初刚访问中国，和中共签订了协议。他说，协议签署过程保密，不能公之于众，但还是有很多人知道了。
 <br/>
 <br/>
 巴基斯坦总理伊姆兰・汗：〝但是很多人人都知道了，中国人也知道，我们和中国的这种协议，从来没有和其他国家签署过。〞
 <br/>
 <br/>
 这次袭击是在巴基斯坦发生的，对中领馆最强烈的攻击。
 <br/>
 <br/>
 新唐人记者毕心慈、张琪综合报导
</div>
