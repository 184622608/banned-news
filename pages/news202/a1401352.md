### 【禁闻】称对等制裁美官员 中共大使被批缺常识
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】中共在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 关押的百万维吾尔和其他少数民族民众，遭到国际社会谴责，美国政府表示，考虑制裁相关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共官员.html" target="_blank">
  中共官员
 </a>
 和中国公司。中共驻美大使崔天凯日前声称，如果美国实施制裁，中共将对美国进行〝同等〞报复。有评论指出，崔天凯缺乏基本常识。
 <br/>
 <br/>
 中共党媒《中国日报》报导说，11月27号，中共驻美国大使崔天凯前往〝路透社〞华盛顿分社，与该社20多名记者座谈并接受采访，就中美元首会晤、中美关系、中美经贸、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 等问题回答提问。
 <br/>
 <br/>
 〝路透社〞的一篇英文报导专门报导了《中国日报》没有提及的内容。
 <br/>
 <br/>
 报导透露，崔天凯声称，如果美国就新疆高官侵犯人权实施制裁，中共将对美国〝按比例〞进行对等报复。
 <br/>
 <br/>
 崔天凯说，美国是用导弹和无人机杀死恐怖份子，而中共只是对恐怖份子进行〝再教育〞，把他们变成正常人。
 <br/>
 <br/>
 美国维吾尔协会主席 伊利夏提：〝中国指控维吾尔人的所谓的恐怖行动全部都是中国的单面之词。具体IS和维吾尔人，IS是一个极端恐怖组织，有各种轻重武器，他们的所作所为全世界有目共睹，而这些被抓捕的维吾尔人手无寸铁的无辜百姓，崔天凯不懂得怎么去比较，他连参照物都没找好，所以崔天凯缺常识。〞
 <br/>
 <br/>
 美国维吾尔协会主席伊利夏提说，其实中共才是最大的恐怖组织，中共通过各种运动杀死几千万手无寸铁的百姓，诸如挖人心肝的湖南道县吃人运动就是最残忍的恐怖行为。
 <br/>
 <br/>
 崔天凯把中共在新疆建所谓的〝再教育营〞关押民众与美国在伊拉克和叙利亚打击〝伊斯兰国〞恐怖份子的军事行动相提并论。被网友嘲讽为：把红匪描绘成菩萨，把平民说成是恐怖份子。
 <br/>
 <br/>
 伊利夏提：〝中国的再教育是把整个民族当成敌人，然后把他们关起来，年龄最大的80岁到90岁，年龄最小的10来岁儿童，美国从来不对无辜的贫民进行抓捕或者轰炸，美国对IS的轰炸用导弹或者飞机是双方交战，也就是说对方也有武器。〞
 <br/>
 <br/>
 位于美国首都华盛顿的研究机构〝2049年工程研究院〞的新疆问题研究员德鲁-琼斯（Drew Jones），在接受〝自由亚洲电台〞采访时表示，中共驻美大使把中共当局将一百多万无辜的维吾尔人关押在拘禁营的做法，与美国打击真实恐怖份子的行动等同起来，实在是太不恰当了。
 <br/>
 <br/>
 此前，联合国人权小组披露，中共将100万以上的维吾尔人关押在〝再教育营〞进行酷刑虐待，很多人被折磨致死。事件引起国际社会的关注和谴责。
 <br/>
 <br/>
 9月份，美国政府表示，考虑制裁参与迫害新疆少数民族的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共官员.html" target="_blank">
  中共官员
 </a>
 和制造监控设备的中国公司。
 <br/>
 <br/>
 路透社的报导说，崔天凯声称要〝按比例〞对等报复的言论，是中共官员对美国制裁威胁迄今为止最强硬的一次反应。
 <br/>
 <br/>
 据报导，美国的制裁行动将基于美国人权法案---〝全球马格尼茨基法〞。该法案允许美国政府冻结世界任何国家违反人权官员在美国的财产，并禁止他们到美国旅行，以及禁止美国公司与这些官员做生意。
 <br/>
 <br/>
 旅美时事评论员夏小强：〝他这种说法很可笑的，美国制裁中共高官就是可以在美国冻结他们的财产，很显然美国的官员没有在中国有财产，它怎么能用同等的方式对待美国呢。〞
 <br/>
 <br/>
 旅美中国问题观察人士张健说，这次崔天凯的表态，等于终于公开承认了国际社会对其指控的集中营的存在。
 <br/>
 <br/>
 旅美中国问题观察人士张健：〝搞这个非人道的方式就足以证明中共要搞的到底是什么，他今天能从肉体上和精神上去消灭新疆这些少数民族，那么在未来它也会将这种方式扩散到其他的国家。〞
 <br/>
 <br/>
 旅美时事评论员夏小强表示，大家会拭目以待，看看中共到底会怎样以同等的方式制裁美国。
 <br/>
 <br/>
 采访、编辑/刘惠 后制/钟元
