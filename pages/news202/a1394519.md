### 印尼天灾死亡增至1944人 5岁童失踪一周奇迹生还
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】印尼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏拉威西岛.html" target="_blank">
  苏拉威西岛
 </a>
 上月28日发生规模7.5
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强震.html" target="_blank">
  强震
 </a>
 ，引发6公尺高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海啸.html" target="_blank">
  海啸
 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土壤液化.html" target="_blank">
  土壤液化
 </a>
 ，造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_帕卢.html" target="_blank">
  帕卢
 </a>
 等地灾情严重，当局8日公布最新死亡人数，已增至1944人同时仍有约5000人下落不明。一名5岁童的母亲以为儿子被海啸卷走，但5日竟传来好消息，奇迹生还。
 <br/>
 <br/>
 当时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强震.html" target="_blank">
  强震
 </a>
 伴随
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海啸.html" target="_blank">
  海啸
 </a>
 接踵而至的时候，5岁男童尤马迪尔（Jumadil）与祖母在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_帕卢.html" target="_blank">
  帕卢
 </a>
 （Palu）一个沿海公园，混乱中，祖母找不到尤马迪尔，只好先逃生，过程中还受了伤。
 <br/>
 <br/>
 据悉，尤马迪尔与祖母走散后不久，即被警方救走。但家人遍寻不着尤马迪尔，担心不已。直到尤马迪尔的叔叔在脸书(Facebook)登寻人启示，在避难中心的人认出尤马迪尔，26岁的母亲拉马蒂亚（Susi Rahmatia）才得以在5日与儿子团聚。
 <br/>
 <br/>
 天灾令无数人家破人亡，据政府负责人的报告，因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土壤液化.html" target="_blank">
  土壤液化
 </a>
 导致地区地层下沉，原本坚实的地面变成松散不稳，泥流一瞬间将数千间房屋吞没，帕卢的2个重灾区〝贝托波（Petobo）和巴拉罗亚（Balaroa）大约有5000人尚未被发现〞，而这两个地区几乎已从卫星图中消失。
 <br/>
 <br/>
 印尼灾难管理局发言人苏托波（Sutopo Purwo Nugroho）7日表示，在泥泞的土堆中寻找尸体，让搜救工作变得更加困难，从11日开始，救援行动将会变动，缩减人员搜索和停止重型机具的使用，未搜寻到的人将视为失踪或死亡。
