### 美事先知情 华为风暴涉及银行欺诈？
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】来看到华为副董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被捕的最新发展。今天(周五07Dec)，有外媒报导，华为员工曝光消息，孟晚舟在华为内部会议时，曾要求员工视情况违反法律。另外，华为副董事长孟晚舟在12月1号遭到逮捕，与川习会同一天，时间点引起广泛怀疑。美国国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波顿.html" target="_blank">
  波顿
 </a>
 表示，自己事先知情。
 <br/>
 <br/>
 CNBC电视台记者 Eunice Yoon：〝事件发生的时间点广泛受到怀疑。加拿大官员表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在12月1日被逮捕，正是同一天，川普与习近平共进晚餐，着手解决的他们的贸易协议。〞
 <br/>
 <br/>
 美国国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波顿.html" target="_blank">
  波顿
 </a>
 （John Bolton）今天接受美国全国公共电台（NPR）访问时表示，他〝事先知情〞，但不清楚川普总统是否也知道。
 <br/>
 <br/>
 CNBC电视台记者 Eunice Yoon：〝她是华为的代表，她最近成为公司的副董事长，所以她也是公司的财务长，同时也是副董事长。许多人私下讨论，认为她会成为创办人，也就是她父亲的继承人。〞
 <br/>
 <br/>
 《南华早报》报导，孟晚舟和华为创办人任正非，10月29号在一场内部会议中向员工表示，可以视情况违规。孟晚舟提到，国外法令分为〝红线〞与〝黄线〞。〝红线〞是〝没有讨价还价空间、必须严格遵守〞；黄线则是〝实际执行层面有困难，公司可以建立藐视法令所需负担的‘沉没成本’（sunk costs）。〞任正非更表示，〝我们不要因为美国攻击我们就绑手绑脚，如果我们因此绑手绑脚的，就没办法继续生产了，那么干嘛还要遵守法规？〞
 <br/>
 <br/>
 路透社报导，最近美方还调查，华为透过汇丰控股向伊朗进行非法交易。但也有消息指出，汇丰没有受到调查。消息出来后，6日，汇丰股价下挫3.619%。对此，汇丰发言人拒绝做出评论。
 <br/>
 <br/>
