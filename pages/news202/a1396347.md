### 卡舒吉被杀事件 川普:不排除制裁沙特
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】沙特知名记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被杀事件.html" target="_blank">
  被杀事件
 </a>
 ，沙特外交部长已承认是误杀。卡舒吉是美国居民，也是受到西方国家关注的沙特异议人士，美国总统川普表示，不排除对沙特实施制裁。西方多国高官与西门子等大企业也已取消参加本周二在沙特举行的大型投资会议。
 <br/>
 <br/>
 沙特知名记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 进入沙特使馆被杀，美国总统川普周日表示，不排除对沙特实施制裁。
 <br/>
 <br/>
 美国总统川普：〝但是还有其他事可以做，包括制裁。〞
 <br/>
 <br/>
 川普重申，对沙特实施武器禁运对美国不利。
 <br/>
 <br/>
 美国总统川普：〝有4千5百亿美元，我认为有超过1百万个工作，我觉得我们若取消这样的订单，将没有好处，将重伤我们远远大于伤害他们。〞
 <br/>
 <br/>
 沙特是世界上最大的石油出口国，有传闻称，沙特可能会向西方客户实施石油禁令。
 <br/>
 <br/>
 不过，沙特能源大臣法利赫周一表示，沙特不想实施类似1973年那样的石油禁运，不会将石油与政治混为一谈。
 <br/>
 <br/>
 1973年石油危机，石油输出国组织宣布石油禁运，暂停出口，油价爆涨近4倍。
 <br/>
 <br/>
 投资专家表示，舒吉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被杀事件.html" target="_blank">
  被杀事件
 </a>
 重创沙特声誉，一些大企业已取消参加周二在利雅得举行的〝未来投资倡议〞会议。
 <br/>
 <br/>
 FIDANTE PARTNERS投资长 克莱门特：〝这对沙特的声誉是巨大损失。我们已经看到，所有大企业领导人和政治领导人退出未来的投资会议。〞
 <br/>
 <br/>
 美国财政部长姆努钦，以及英、法、荷兰等国高级官员已宣布取消出席。
 <br/>
 <br/>
 摩根大通、福特汽车、Uber、西门子等大型企业，也决定不出席会议。
 <br/>
 <br/>
 新唐人记者李梅、张琪综合报导
</div>
