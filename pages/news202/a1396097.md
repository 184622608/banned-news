### 美国〝毒丸〞条款发威？英中自贸谈判搁浅
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】英国专家日前分析，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 等原因，导致中英两国自由贸易协定的谈判搁浅。此前美国成功与墨西哥、加拿大达成贸易协议后，外媒普遍认为中共政府正在遭到美国及其盟友的孤立。尤其美国政府在新协议中推出的〝毒丸〞条款，对中方突围孤立的企图是有力的打击。
 <br/>
 <br/>
 俄罗斯卫星通讯社19日报导称，英国牛津分析公司（Oxford Analytica）的分析师凡妮莎・罗西（Vanessa Rossi）日前表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 和英国脱欧进程的缓慢，导致中共政府与英国政府之间的自由贸易协定谈判陷入停滞。
 <br/>
 <br/>
 罗西表示，脱欧可能是一个漫长的进程，可能需要数年的时间。在此之前，英国政府不知何时能与其他国家进行自由贸易谈判。同时，美国政府的贸易政策也给英、中之间的自由贸易协定谈判带来阻力。
 <br/>
 <br/>
 她说：〝贸易战导致美国与一系列伙伴的关系发生波动，现在贸易谈判需要更多的时间，这可能给英国的利益带来打击。〞
 <br/>
 <br/>
 今年夏天，英国外交大臣亨特访问中国期间曾宣布，英国希望在2019年3月29日脱欧后与中方进行自贸协定谈判。不过，此后相关的议题便没有了下文。
 <br/>
 <br/>
 有美国媒体报导称，在中美贸易战的背景下，川普（特朗普）政府正积极寻求与欧盟、日本和英国分别达成新的贸易协议，同时阻止中共与这些国家（包括加拿大、墨西哥）等达成双边自贸协议。
 <br/>
 <br/>
 此前，在美国分别与墨西哥、加拿大达成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新北美自由贸易协定.html" target="_blank">
  新北美自由贸易协定
 </a>
 后，国际舆论普遍判断，正与美国打贸易战的中共在国际贸易关系中将更加孤立。
 <br/>
 <br/>
 10月10日中共国务委员兼外交部长王毅与加拿大外长弗里兰通电话时，试图敦促加拿大恢复已经停滞的中加自由贸易谈判。
 <br/>
 <br/>
 不过，此前渥太华已经在与美国达成的新协议中，同意了一项特殊的条款，即：如果美国、墨西哥和加拿大贸易协定(USMCA)中的任何一个国家与一个〝非市场经济国家〞签署贸易协定，另外两个国家可以在六个月内自由退出，并达成双边贸易协定。
 <br/>
 <br/>
 海外舆论一致认为，这一条款主要是针对中共政府制定的，这一条款实际上让美国获得了对渥太华和北京之间任何可能贸易协定的否决权。美国商务部长罗斯则把这个特别条款戏称为〝毒丸〞条款。
 <br/>
 <br/>
 加拿大政府与中共政府曾于2016年同意开始自贸协定试探性协商，但双方的谈判并未取得任何进展。加拿大全球事务部曾列出了2017年举行的四次面对面试探性会议，但是2018年一次也没有举行。在新北美自由贸易协定达成后，外界更认为，加中之间的贸易谈判已难以为继。
 <br/>
 <br/>
 美国商务部长罗斯10月5日接受路透社采访时曾坦言，在新签署的美墨加贸易协定中加入的〝毒丸〞(poison pill)条款，可能会在未来美国与其它国家和地区达成的贸易协定中被复制。他表示，有了新北美自由贸易协定中加入这一条款的先例，在其他贸易协定中加入此条款将变得更加容易。他说：〝人们会明白，这是达成协议的先决条件之一。〞
 <br/>
 <br/>
 当时，罗斯的上述谈话引起了国际舆论的广泛关注，有媒体将其解读为〝川普布大棋盘联手全球封堵中国贸易〞。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：李泉）
