### 天主教徒巴基斯坦免死 极端穆斯林抗议
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴基斯坦.html" target="_blank">
  巴基斯坦
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院.html" target="_blank">
  最高法院
 </a>
 周三推翻了对一名女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天主教徒.html" target="_blank">
  天主教徒
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_死刑判决.html" target="_blank">
  死刑判决
 </a>
 ，她曾因亵渎罪而被下级法院处以死刑。最高法院的这一决定引发大规模抗议活动。
 <br/>
 <br/>
 毕比是四个孩子的母亲，从2010年开始被关在死囚牢里。她是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天主教徒.html" target="_blank">
  天主教徒
 </a>
 ，也是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴基斯坦.html" target="_blank">
  巴基斯坦
 </a>
 第一名因为〝亵渎法〞而被判死刑的妇女。
 <br/>
 <br/>
 她2009年被邻居指控，不是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆斯林.html" target="_blank">
  穆斯林
 </a>
 的她用了别人的杯子喝水，被阻止时还说了贬低伊斯兰教的话。但毕比始终否认指控。
 <br/>
 <br/>
 在巴基斯坦的保守穆斯林信仰中，亵渎罪可引发暴力冲突，甚至面临被私刑处死。
 <br/>
 <br/>
 名为〝拉巴伊克运动〞(Tehreek-e-Labaik，TLP)的激进伊斯兰政党周三封锁了主要城市的道路，以示对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院.html" target="_blank">
  最高法院
 </a>
 的抗议，甚至呼吁推翻政府和军队。
 <br/>
 <br/>
 巴基斯坦总理伊姆兰．汗（Imran Khan）同一天发表电视讲话，对极端伊斯兰份子提出警告。
 <br/>
 <br/>
 巴基斯坦总理 伊姆兰．汗：【我们不会允许发生任何破坏行为。我们不会允许阻碍交通。我呼吁你们，不要把国家推到被迫采取行动的地步。】
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
