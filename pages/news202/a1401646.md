### G20川习会  美中贸易争端迎来拐点?
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】
 <b>
  新闻周刊
 </b>
 （656）2018年度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会，周五和周六两天在阿根廷首都布宜诺斯艾利斯举行。而比起峰会本身，周六晚上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的会晤更加引人关注。此次会晤能否缓和美中之间在贸易上乃至其他层面的矛盾？请看下面的综合报导。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，决定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会结束当晚举行晚宴会谈。这次会晤被一些媒体称为〝世纪谈判〞，似乎连G20峰会本身，都成为两人谈判的背景板。
 <br/>
 <br/>
 对于这次谈判，外界有不同的预期。但普遍的共识是，川普政府不太可能在关税问题上软化态度，谈判的压力在中共一方。
 <br/>
 <br/>
 中共驻美大使 崔天凯：〝我们想要达成解决协议，我们非常想看到（紧张气氛）得到降温。〞
 <br/>
 <br/>
 《华尔街日报》11月28号报导，中共官员表示，当务之急是说服美方暂停在明年1月1号将2,000亿美元中国进口商品的关税税率从10%上调到25%的计画。中共还希望美国能放弃其它针对中国产品的关税行动，并取消对中国钢铝产品征收的惩罚性关税。
 <br/>
 <br/>
 对此，川普则对《华尔街日报》表示，他预计会持续推进这个提高关税税率的计画。川普还说他〝极不可能〞接受北京的要求，推迟提高关税。
 <br/>
 <br/>
 前往G20峰会临行前，川普再对记者表示，更想达成协议的是北京。
 <br/>
 <br/>
 美国总统 川普：〝我们非常接近与中国达成某些实际措施，不过我不一定想那么做。通过关税和税收，成百上千亿美元进入美国。所以我还没想好。但我可以告诉你我认为中国想要达成协议。我持开放态度，不过坦白说，我觉得目前的局面不错。〞
 <br/>
 <br/>
 北京是否会做出实质性让步呢？数家媒体披露，中方已经提出，如果美国暂缓施加进一步的关税，北京将同意放宽采购美国农产品和能源产品的限制；并愿意就〝结构性问题〞和美国谈判。例如〝中国制造2025计划〞，以及对大型国企的控制和补贴等。此前中方一直拒绝讨论这些问题。
 <br/>
 <br/>
 不过，有专家质疑，从这些条件看，中共并未改变〝以拖待变〞的态度。
 <br/>
 <br/>
 俞伟雄：〝其实从川普上台后，美方已经跟中方在谈所有这些问题，已经谈了2年了。中国以〝暂缓实施〞来做接下来谈判的前提，我现在不清楚还可以谈什么？〞
 <br/>
 <br/>
 在G20峰会的第一天，美国、墨西哥和加拿大三国元首，共同举行了《美墨加贸易协定》的签字仪式。这份协议中明确规定，签约国不能和非市场经济国家达成贸易协议。
 <br/>
 <br/>
 作为签约国之一的美国，始终否认中共的市场经济地位。受《美墨加贸易协定》的制约，这意味着美国事实上已经不能同中共达成贸易协议。
 <br/>
 <br/>
 对此，川普本周也已经明确表态，谈判的底线是北京要开放市场。
 <br/>
 <br/>
 美国总统 川普：〝这是底线，中国必须公平对待我们，以前它们从来没有这样做。他们必须公平对待美国。〞
 <br/>
 <br/>
 白宫经济顾问库德洛表示，达成零关税、零补贴、零非关税壁垒的公平自由贸易是川普真正想要的。然而中共到目前为止并没有遵守任何相关国际规则。
 <br/>
 <br/>
 有白宫官员表示，要冷静看待
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 可能达成的成果。
 <br/>
 <br/>
 白宫国家安全顾问 博尔顿：〝不期待这次会晤会达成具体协议，但对于双方代表会有一个指导性纲领出炉。〞
 <br/>
 <br/>
 美国参议院外委会议员马基则直言，中共在经济、军事、人权等多方面对他国实施干预，挑战美国战略，因此不能仅仅从关税战的角度看待美中争端。
 <br/>
 <br/>
 美国参议院外委会议员 Edward Markey：〝必须保证，川普政府不会与中共达成某种协议，只是技术上缓和关税争端，而不解决重要的战略结构性问题。〞
 <br/>
 <br/>
 据最新消息，G20峰会达成的公报内容中，有意避开〝保护主义〞、〝不公贸易〞等敏感字眼。而在中共主导的金砖五国首脑声明中，包括了敦促WTO打击〝贸易保护主义〞和〝单边主义〞的字眼，语气仍然接近中共一直以来立场，暗含对美国的指责意味。美中在外交层面的角力，仍在继续。
 <br/>
 <br/>
 目前，美国朝野许多人士已经清楚认识到，中共挑战美国维护世界秩序的野心，也已将中共明确视为战略对手。专家认为，美中之间全方位竞争的〝冷战〞格局正在形成，一次川习会将很难成为改变这一局势的拐点。
 <br/>
 <br/>
 撰稿：林宇
