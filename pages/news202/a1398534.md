### 蓬佩奥与金英哲会谈推迟 韩外长：是应朝鲜要求
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 原定8日在纽约会见朝鲜2号人物
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金英哲.html" target="_blank">
  金英哲
 </a>
 ，讨论两国领袖第2次峰会，但华府周三（7日）突然宣布会议延期。韩国外交部长康京和周四（8日）表示，是朝鲜（北韩）要求延后高阶会谈。
 <br/>
 <br/>
 据韩联社（Yonhap）报导，康京和（Kang Kyung-wha）周四告诉议员，美国告诉韩方，他们收到朝鲜的讯息要延后会面。
 <br/>
 <br/>
 美国国务院并未进一步解释
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 （Mike Pompeo）与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金英哲.html" target="_blank">
  金英哲
 </a>
 （Kim Yong Chol）的会面推迟原因。
 <br/>
 <br/>
 美国总统川普(特朗普）周三表示，原定于周四在纽约举行的会议将重新安排，但他仍希望在明年初与朝鲜领导人金正恩举行第二次峰会。他也都没有解释为什么会谈被推迟。
 <br/>
 <br/>
 不过，川普也强调美国〝不急于〞展开谈判。对于朝鲜一再要求放松制裁，川普表示，他也希望对朝鲜放宽制裁，但平壤必须先有所行动。
 <br/>
 <br/>
 川普在白宫的记者会中说，〝我们并不急于求成，也不急于做任何事情。〞
 <br/>
 <br/>
 川普说，〝在我当选总统以前，他们已经处理了70多年。在核领域也已有25年了。那是很长一段时间。〞
 <br/>
 <br/>
 他说，〝我在这4～5个月里取得了比之前70年来更多的进步，没有其他人可以做到我已经完成的工作。但我会说这很简单，我们不急，制裁正在进行，无论什么时候都是。〞
 <br/>
 <br/>
 川普今年6月在新加坡与金正恩举行峰会，双方签署了一项概况声明，要求朝鲜半岛实现去核化。但在此后的几个月中，朝鲜终止核武器计划的进展有所放缓。
 <br/>
 <br/>
 平壤当局一直要求美国放宽针对朝鲜武器计划而实施的制裁，而华府坚称应先等朝鲜非核化有了切实行动后再谈放宽制裁。
 <br/>
 <br/>
 （记者李红报导/责任编辑：文慧）
