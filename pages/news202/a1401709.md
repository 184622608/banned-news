### 【禁闻】美暂停加征关税90天 重启与否看中共
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】举世瞩目的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞已经结束，中方作出让步后，中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 暂停升级，美国答应不在1月1号提高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，不过如果90天内不能完成谈判，关税仍会升级。专家认为，能否阻止贸易战升级，关键在于北京当局能否兑现自己的承诺。
 <br/>
 <br/>
 12月1号，在阿根廷举行的二十国集团（G20）峰会之后，美国总统川普和中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，在布宜诺斯艾利斯举行了约两个半小时的晚宴会晤。
 <br/>
 <br/>
 随后，白宫发表声明，说这是一次〝高度成功的会谈〞。美国宣布，从明年1月1号起，不会对价值2,000亿美元的中国商品
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，从目前的10％提高到25％。
 <br/>
 <br/>
 中国同意从美国购买数量非常大的农业、能源、工业和其它产品，以减少两国之间的贸易不平衡。其中，美国农产品，中国已经同意立即购买。
 <br/>
 <br/>
 另外，习近平和川普都同意立即开始有关中国〝结构性改变〞的谈判。这些结构性改变涉及强制技术转让、知识产权保护、非关税壁垒、网路入侵和网路盗窃、服务业和农业。
 <br/>
 <br/>
 此外，对于此前导致美国数千人死亡、川普认为是中共对美国发动毒品战争的芬太尼，习近平同意把它列为〝管控物质〞。
 <br/>
 <br/>
 习近平还表示，先前未获批准的高通-恩智浦协议，如果再次呈交于他，他会用〝开放的态度〞来批准。
 <br/>
 <br/>
 声明还提到，川普将和习近平、金正恩一起，努力实现朝鲜半岛无核化。
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝总的来说中方作出了全面的妥协和让步，虽然还没有100%的达到美方的要求，但是根本的立场有了大幅度的让步和妥协。中方之前不承认的、不愿意做的很多事情，可能现在承认了，愿意做了。〞
 <br/>
 <br/>
 哥伦比亚大学政治学博士李天笑说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 会在这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 中停止升级，在他的预料之中。
 <br/>
 <br/>
 哥伦比亚大学政治学博士李天笑：〝川普打这个贸易战不是说要几棍子把中国的经济打垮，他是要通过贸易战这个缺口给习近平施加压力，然后让习近平改制变局，把不公平的贸易及强迫技术转让这些东西全部纠正过来，然后进一步引申战果，把它扩展到法律经济制度的改革，最后的结果就是中共制度的崩溃。〞
 <br/>
 <br/>
 不过，白宫的声明中说，双方同意在未来90天内努力完成谈判，如果在这个期限内未能达成协议，10%的关税将会被提高至25%。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田说，只要中共的体制存在，就不可能实现非关税壁垒、网路入侵、知识产权保护等一系列改变，谈判很难在90天内完成。因而谢田担心，中共还会像以前一样玩花招，以赢得90天的时间。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝在这90天的时间里，它以前2500亿的10%的关税虽然没有取消，但是更大幅度的25%关税的惩罚现在可以避免，但是川普应该很快会发现中共的承诺是靠不住的。〞
 <br/>
 <br/>
 这次谈判的美方代表包括最为鹰派的贸易政策顾问纳瓦罗，贸易代表莱特希泽等人，在这次晚宴上，川普还是一如既往的说，他和习近平的关系很友好。
 <br/>
 <br/>
 李天笑认为，川普一直对习近平很友好，是因为中美贸易巨大逆差主要是之前的在位者造成的。
 <br/>
 <br/>
 李天笑说，如果当今北京当局有足够的智慧，就不应该为中共这种掠夺性的贸易策略所造成的恶果背黑锅，或幻想摇摇欲坠的中共能保住自己的权利，而死保这个西方视为洪水猛兽的中共政权，而是应该配合川普，在如期内作出结构性改革，避免贸易战升级给民族和自己带来灾难。
 <br/>
 <br/>
 采访编辑/刘惠
