### 新版美墨加贸易协定  预定G20峰会签署
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】9月底达成的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨加贸易协定.html" target="_blank">
  美墨加贸易协定
 </a>
 ，被视为川普政府在贸易上的一项重要胜利。三方预定在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间签署该协定。目前美国国会内还有一些反对声音，但墨西哥驻美大使表示，协定依然有很大机会得到三国立法机构的顺利批准。
 <br/>
 <br/>
 墨西哥驻美大使日前确认，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间，美墨加三国将在阿根廷首都布宜诺斯艾利斯正式签署
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨加贸易协定.html" target="_blank">
  美墨加贸易协定
 </a>
 ，但尚未确定是由三国元首还是贸易代表进行签字。
 <br/>
 <br/>
 同时，加拿大与墨西哥，正和美国加紧谈判，钢铝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 永久豁免问题，并希望在月底前达成协议。
 <br/>
 <br/>
 墨西哥驻美大使 Gerónimo Gutiérrez：〝预计到签署协定时，将有一个解决方案，或是一条非常清晰的轨道，提供各方足够的确定性，即解决方案即将到来。〞
 <br/>
 <br/>
 美墨加贸易协定签署后，还需得到三国立法机构批准。美国中期选举，民主党赢得众议院多数。未来他们能否支持新协议成为关键。目前，部份民主党众议员，对于墨西哥能否真正落实协定中的劳工标准提升和环保条款提出置疑，也有部份共和党众议员，反对协定中保护同性恋劳工的新规定。
 <br/>
 <br/>
 墨西哥驻美大使 Gerónimo Gutiérrez：〝协定有很大机会，能够被三国立法机构都通过。就美国而言，协定在共和党与民主党关注的问题上，有很适当的平衡。〞
 <br/>
 <br/>
 前美国驻墨西哥大使 Earl Anthony Wayne：〝协议将为北美大陆大量的经济活动提供稳定性，所以有很好的理由来通过这一协定。〞
 <br/>
 <br/>
 边境大篷车难民问题也在困扰美墨两国。预计未来两周，将有一万名大篷车成员抵达美边境城市蒂华纳（Tijuana）。美国国土安全部官员称，大篷车成员中，藏有500多名罪犯。
 <br/>
 <br/>
 墨西哥驻美大使 Gerónimo Gutiérrez：〝我认为这对墨西哥和中美洲国家及美国是一个警钟，我们要如何更好地彼此合作，来更好地应对移民现象。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
