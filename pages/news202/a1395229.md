### 德城际火车行驶间起火 车厢全毁510乘客迅速撤离
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 一列由科隆开往慕尼黑的城际高速
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火车.html" target="_blank">
  火车
 </a>
 ，周五（12日）早上途经莱因兰-普法尔茨州小镇时，最后一节车厢突然起火，500多人迅速撤离，所幸仅有5人受轻伤。
 <br/>
 <br/>
 周五早上约6时35分，该列车行经莱因兰-普法尔茨州迪尔多夫（Dierdorf）时，最后一节车厢突然起火。列车员随即协助车上共510名乘客疏散，之后当局将这些旅客由巴士送到邻近城市蒙塔鲍尔，事件导致5人受伤。
 <br/>
 <br/>
 当局派出250人前往现场救火，涉事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火车.html" target="_blank">
  火车
 </a>
 车厢事后被全部烧毁。事件导致相关列车停止90分钟，之后火车改道另一条路线行驶。涉事路段的火车轨及附近一条公路被暂时关闭。
 <br/>
 <br/>
 当局正调查事故原因，估计可能是机件故障引起火灾。
