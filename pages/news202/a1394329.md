### 菲缉毒执法人员与警察遭伏击 6死2伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾.html" target="_blank">
  菲律宾
 </a>
 缉毒署执法人员与警察周五（5日）下午在南部小镇遭到伏击，造成6死2伤，歹徒身分及动机仍在调查中。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾.html" target="_blank">
  菲律宾
 </a>
 警方周五称，下午1时左右，5名缉毒署执法人员和3名警察在塔戈洛安镇（Tagoloan）出席反毒研讨会后，乘车返回马拉韦市（Marawi）途中，在卡派镇（Kapai）遭到武装分子伏击。5名缉毒署执法人员和1名警察不幸身亡，另2名警察则受伤送医急救中。
 <br/>
 <br/>
 卡派镇位于民答那峨岛，距马尼拉约1220公里。
 <br/>
 <br/>
 由于9月19日，民答那峨穆斯林自治区（ARMM）缉毒署执法人员在伊里甘市（Iligan）执行了一场扫毒行动，逮捕多名省议员、市议员以及他们的亲人。警方正在调查此次袭击事件是否与扫毒行动有关。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
