### 遭控逾20条罪名 纳吉布妻子正式被拘捕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月03日讯】马来西亚前总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳吉布.html" target="_blank">
  纳吉布
 </a>
 （Najib Razak，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳吉.html" target="_blank">
  纳吉
 </a>
 ）卸任后，因《反洗黑钱及反恐融资法令》、《反贪污法令》面临25项控罪，于9月19日被捕。今(3)日，他的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗斯玛.html" target="_blank">
  罗斯玛
 </a>
 也遭反贪部门以洗黑钱罪名正式拘捕。
 <br/>
 <br/>
 综合媒体报导，66岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗斯玛.html" target="_blank">
  罗斯玛
 </a>
 （Rosmah Mansor）在9月26日曾就国家基金〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一马发展公司.html" target="_blank">
  一马发展公司
 </a>
 〞（1MDB）资金涉嫌被挪用案件接受反贪部门长达13小时问话。当时被传召时，她仍手持名牌手袋由车队护送到达。
 <br/>
 <br/>
 3日，罗斯玛被控超过20条罪名，其中以洗钱占大部分。被捕后，被押往马来西亚反贪腐委员会（MACC）的总部。
 <br/>
 <br/>
 据消息人士表示，罗斯玛将于明(4)日被正式落案起诉。
 <br/>
 <br/>

 <br/>
 而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳吉.html" target="_blank">
  纳吉
 </a>
 布于9月19日被捕并遭起诉，内容包括他将1MDB的6.81亿美元汇入个人帐户，在21项控罪中，有9项指控他洗钱，5项指控他使用不合法财物，7项指控他将不法之财转移至别处。随后再出庭应讯前，又被加控4项贪污罪，涉及约5.55亿美元的1MDB资金。
 <br/>
 <br/>
 但在法庭宣读指控后，纳吉布否认全部控罪。
 <br/>
 <br/>
 纳吉布被当局调查时，当局曾在他的豪宅内搜出总值高达11亿令吉（约2.753亿美元）的大量财物。物品中包括价值1.16亿令吉（约2900万美元）不同货币的现钞、1.2万件珠宝及567个Hermès、Prada及Chanel手袋。
