### 欧洲边境〝阿凡达〞警察 过海关更快捷
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】欧洲计画试用新科技打造〝阿凡达〞边境警察，让旅客
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_过海关.html" target="_blank">
  过海关
 </a>
 的程序更安全快捷。周六，这项技术在英国的科技展中现身。
 <br/>
 <br/>
 在不久的将来，如果你通过欧洲国家的边境，可能会遇到这位彬彬有礼的〝阿凡达〞边境警察。他会询问旅行者程序性的问题。
 <br/>
 <br/>
 〝阿凡达〞边境警察 ：〝你曾经被申根区驱逐过吗？〞
 <br/>
 旅客：〝从来没有〞
 <br/>
 <br/>
 〝阿凡达〞（Avatar）的含义其实是电脑使用者的图片影像。这套程序利用语音识别功能，与旅客对话，旅客可以点按屏幕上显示的选项，并确认，简单五个步骤就可完成。
 <br/>
 <br/>
 这个项目并不是为了替代真人边境警察，而是旅客在旅行前，提前上网注册通关的一部分。
 <br/>
 <br/>
 这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_〝阿凡达〞警察.html" target="_blank">
  〝阿凡达〞警察
 </a>
 非常智能，能够辨别出一个人是否在回答他的问题时撒谎。
 <br/>
 <br/>
 曼彻斯特都市大学电脑智能专家 Keeley Crockett:〝每个旅客都要经过自动测谎程序的分析，看这个人在回答问题时有没有说谎。如何做到呢？就是运用非语言行为反应，非常非常细微的动作。不是看这个人在微笑或是皱眉，而是看非常细小的动作，比如眼睛向左看，还是向右看。通过分析这些数据得出‘危险分数’，再根据这个分数来判定这个人是否在回答问题时说谎。〞
 <br/>
 <br/>
 利用电脑智能技术进行安检已成为趋势，比如芬兰赫尔辛基机场（Helsinki airport）用脸部识别技术，杜拜的国际机场用眼睛识别技术，来控制边境安全。
 <br/>
 <br/>
 欧盟国家之间大多是软边境，近年来，由于中东局势动荡，大批难民非法通过欧盟边境，使不少欧盟成员国开始加强边境安全措施。匈牙利就在边境架起铁丝网，抵挡索马里难民非法进入。
 <br/>
 <br/>
 〝阿凡达〞边境警察将从11月开始，首先在匈牙利与希腊和拉脱维亚接壤的两个边境试行。
 <br/>
 <br/>
