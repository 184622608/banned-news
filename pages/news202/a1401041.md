### 〝大篷车〞滞留边境 墨西哥人反感升级
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】几千名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民仍然在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 边境城市蒂华纳等待入境美国，暴力事件时有发生而且不断升级，这让当地墨西哥人越发不满。
 <br/>
 <br/>
 7000多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民聚集在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 的蒂华纳市和墨西卡里，还有800人在路上。
 <br/>
 <br/>
 不仅川普政府拒绝这些来自中美洲的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 入境，越来越多的墨西哥人也对他们产生排斥。
 <br/>
 <br/>
 墨西哥最大报纸之一埃尔环球报周日公布的民调显示，十分之七的墨西哥人对大篷车移民的到来持负面态度。
 <br/>
 <br/>
 52％的人认为，如果没有有效文件，就不应该允许这些人进入墨西哥。55％的人认为应该对大篷车移民采取更严格的措施。
 <br/>
 <br/>
 在这一地区的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 两边生活着600万人口，过境经济规模达到2200亿美元。另外还有成千上万的人每天过境上班和上学。保持边境开放对他们来说非常重要。
 <br/>
 <br/>
 周日，因为移民暴力冲击美国边境，美国不得不关闭边境5个小时。
 <br/>
 <br/>
 而几千名移民常驻此地，也给当地医疗系统带来压力。
 <br/>
 <br/>
 墨西哥蒂华纳医学院院长 桑切斯：每天，都会有(移民)来寻求治疗龋齿，胃痛，动脉高血压。
 <br/>
 <br/>
 白宫顾问康威周二表示，川普正在为修建美墨边境墙而向国会寻求资金，并最终会让墨西哥为建墙费买单。
 <br/>
 <br/>
 美国总统顾问 康威：五十亿美元是一笔巨额首付，但最近发生的事表明，我们需要保证边境安全。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
