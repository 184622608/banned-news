### USCC报告 建议国会抵制中共影响力扩张
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中经济安全审查委员会.html" target="_blank">
  美中经济安全审查委员会
 </a>
 周三发表年度报告。针对中共政策对美国构成的威胁，委员会进行了全面评估，并向国会提出十项建议。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中经济安全审查委员会.html" target="_blank">
  美中经济安全审查委员会
 </a>
 主席 Rubin Cleveland：〝中共日益加强的对内专制、和对外强势做法，挑战美国领导地位和国际正常秩序，迫使我们进行严格评估。〞
 <br/>
 <br/>
 美中经济安全审查委员会发表的年度报告指出，中共为维护其权力，系统性地给予中国企业尤其是国有企业不公平优势。
 <br/>
 <br/>
 对于中共未履行其世贸组织义务，损害他国预期利益，报告建议国会应要求美国贸易代表办公室（USTR）与盟国协调一致，在世界贸易组织内，对中共进行提告。
 <br/>
 <br/>
 美中经济安全审查委员会委员 Michael R. Wessel：〝现在的问题是，能否通过和盟友的协调，与中共谈判达成协议，解决这些问题。如果不行，我们就要采取相应措施。〞
 <br/>
 <br/>
 报告还谈到，贸易战影响下，中共加倍鼓励增长的做法，令其债务负担2017年冲破38万亿美元；同时，中国公司在美国股市融资超过一万亿美元，却不不遵守美国的审计和披露法。
 <br/>
 <br/>
 美中经济安全审查委员会主席 Rubin Cleveland：〝我们的国家在面临险境，由于中共保护其权力和政权利益，对于经济稳定及我们所深入联结的全球金融系统，构成威胁。〞
 <br/>
 <br/>
 委员会建议美国创建一个基金，协助印太等地区国家的基础设施建设，以抵制中共通过〝一带一路〞，输出其独裁统治模式。
 <br/>
 <br/>
 委员会建议，应依法起诉在美国境内〝威胁、胁迫或以其它方式恐吓美国居民〞的中共分支机构，并再次点名中共官媒《中国日报》。
 <br/>
 <br/>
 美中经济安全审查委员会副主席 白嘉玲：〝我们建议国会，指示国家反情报和安全中心，提供中共在美国的影响和宣传活动年度报告。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
