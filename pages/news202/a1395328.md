### 国会补选赢得惊人胜利 安华重返政坛
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马来西亚.html" target="_blank">
  马来西亚
 </a>
 前副首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安华.html" target="_blank">
  安华
 </a>
 13日以压倒性票数赢得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国会.html" target="_blank">
  国会
 </a>
 议员补选，这项胜选意味安华正式重返政坛，令其更接近登上首相宝座。现任首相马哈蒂尔先前曾公开宣布，他将会在担任大马首相2年后下台，让位给安华。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安华.html" target="_blank">
  安华
 </a>
 （Anwar Ibrahim）此前与现任首相高龄93岁的马哈蒂尔（Mahathir Mohamad）交恶长达20年。2015年安华被控鸡奸遭判刑入狱而无法参选，但两人尽释前嫌重新结盟。
 <br/>
 <br/>
 今年5月，马哈蒂尔领导〝希望联盟〞（Alliance of Hope）在大选中赢得惊人胜利，推翻前门徒纳吉布（Najib Razak）并终结执政联盟〝国民阵线〞（Barisan Nasional）长达61年的统治。
 <br/>
 <br/>
 马哈蒂尔其后为安华争取到王室特赦。安华13日以大幅领先超过2万3500票，压倒性赢得狄克森港（Port Dickson）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国会.html" target="_blank">
  国会
 </a>
 席次，重返国会。
 <br/>
 <br/>
 安华过去20年来一直是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马来西亚.html" target="_blank">
  马来西亚
 </a>
 改革运动的代表性人物。他在胜选集会上告诉数百名支持者：〝改革不会于一个月、4个月或一年内结束。改革意味清理和解放媒体，肃清贪腐，让合乎道德行为成为政治文化。〞
 <br/>
 <br/>
 马哈蒂尔先前曾宣布，他将会在担任大马首相2年后让位给安华。
