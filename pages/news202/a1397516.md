### 大麻阴霾笼罩万圣节 讨糖当心含大麻
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万圣节.html" target="_blank">
  万圣节
 </a>
 又称〝万鬼节〞，是西方世界的传统节日。万圣节当夜的主要活动之一是小朋友装扮成各种鬼怪，逐门逐户敲门讨糖吃。然而，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻合法化.html" target="_blank">
  大麻合法化
 </a>
 已经笼罩全
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 的阴霾下，大麻糖、大麻巧克力等大麻食用品泛滥成灾，今年的万圣节让家长们备感担忧。
 <br/>
 <br/>
 今天是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 娱乐性
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻合法化.html" target="_blank">
  大麻合法化
 </a>
 后的第一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万圣节.html" target="_blank">
  万圣节
 </a>
 ，家长们担忧孩子外出讨要来的糖果堆里混杂着〝裹着糖衣的毒品〞。近日，温哥华警察局提醒家长检查孩童讨到的糖果，以免他们误食了大麻糖。
 <br/>
 <br/>
 亲子情协会主席Karen Chan：〝（万圣节）不要让孩子出外讨糖，很危险啊。〞
 <br/>
 <br/>
 尽管售卖的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻糖果.html" target="_blank">
  大麻糖果
 </a>
 、食品等包装纸上标有CBD,THC等印记，但包装拆下后的零食本身很难被识别出来。
 <br/>
 <br/>
 亲子情协会主席Karen Chan建议家长带孩子们参加教会等机构举办的万圣节活动，同时对孩子加强教育。
 <br/>
 <br/>
 亲子情协会主席Karen Chan：〝不单是万圣节讨糖，如果学校有糖果，父母应该告诉孩子，（在）学校不要吃，同学给你的也不要吃，让孩子拿回家，让爸妈家长看过才吃。〞
 <br/>
 <br/>
 此外，根据联邦《刑事法》，向18岁以下的未成年人提供或贩售大麻者，最高将面临14年监禁。
 <br/>
 <br/>
 新唐人记者舒璨加拿大温哥华报导
</div>
