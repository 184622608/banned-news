### 蓬佩奥：中共滥用国际组织 美将建立世界新秩序
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 星期二在一次演讲中，严词批评中共与俄罗斯、伊朗等政府长期滥用国际组织为其自身牟取利益。他表示，为了改变这种现状，美国政府将推动对这些国际组织的改革，并建立起一个新的世界秩序。
 <br/>
 <br/>
 据《美国之音》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 （Mike Pompeo）是12月4日在布鲁塞尔举行的美国德国马歇尔基金会上发表演讲时作出了上述表态。
 <br/>
 <br/>
 在这次题为《改革基于规则的国际秩序》的演讲中，蓬佩奥明确表态说，美国政府将不会继续容忍中共以及伊朗和俄罗斯〝玩世不恭地滥用国际组织〞，也不会继续放任他们肆意违反国际条约和多边协议的做法。
 <br/>
 <br/>
 他表示，联合国、欧盟、国际货币基金组织、世界银行等形成二战后国际秩序基础的机构急需改革，而华府将采取行动来推动对这些国际机构的改革，并建立起一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界新秩序.html" target="_blank">
  世界新秩序
 </a>
 。
 <br/>
 <br/>
 蓬佩奥还在演讲中敦促美国的盟国，希望各个盟国政府能够与川普总统一起，对世贸组织、联合国、国际货币基金组织等机构进行诚实地评估。
 <br/>
 <br/>
 〝不良的行为者正利用我们领导力的缺乏，为他们自己谋利。〞蓬佩奥说：〝这是美国一味退守的毒果。川普总统现在决心要扭转这种局面。〞
 <br/>
 <br/>
 在这次有关美国外交政策的演讲中，蓬佩奥表示，川普总统在执政的这两年里，一直致力于在美国最优秀的民主传统中，团结〝各个伟大的国家〞去共同建立一个〝新的自由秩序〞，并致力于防止战争，让各个国家都走向更辉煌的繁荣。
 <br/>
 <br/>
 蓬佩奥强调，〝在川普总统的引领下，我们不会放弃国际上的领导地位，也不会放弃我们在国际体系中的朋友。〞
 <br/>
 <br/>
 他说：〝我们的行动是在维持、保护和促进由主权国家组成的一个开放、正义、透明和自由的世界。这个工程将需要各国真正的，不是假装的，恢复自由秩序。这将需要一个坚定自信的美国和不仅来自己国家的领导，而是全世界民主国家的领导。〞
 <br/>
 <br/>
 （记者何雅婷报导/责任编辑：东野）
</div>
