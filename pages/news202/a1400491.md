### 涉诈骗中国人现金 日警逮捕台湾女子
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】日本警方怀疑台湾籍女性陈玉英是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诈骗集团.html" target="_blank">
  诈骗集团
 </a>
 成员，涉诈骗中国籍女性约1500万日圆（约13万美元），已将其逮捕。该诈骗集团在日本可能已骗取中国籍女性约1.2亿日圆（约106万美元）。
 <br/>
 <br/>
 综合日媒23日报导，57岁的陈玉英出身于台湾，今年8月，她涉嫌伪装成中国警察，博取中国籍女性信任后，骗取约1500万日圆现金，被日本警视厅组织犯罪对策二课以诈欺嫌疑逮捕。陈玉英承认收受现金，但她否认涉及诈骗。
 <br/>
 <br/>
 日本警方怀疑陈玉英可能是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诈骗集团.html" target="_blank">
  诈骗集团
 </a>
 成员，这个诈骗集团主要以中国籍女性为下手对象。
 <br/>
 <br/>
 日本警方表示，7～9月间，诈骗集团用相同手法骗取中国人现金的案例就有10多件，金额高达1.2亿日圆，警方正调查这几起案件的关联。
