### 美司法部：中方间谍盗取商用飞机数据
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】美国司法部周二发表声明，指控中共情报人员与黑客和私人公司内部人员合谋，侵入公司电脑系统，窃取商用客机
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_涡轮发动机.html" target="_blank">
  涡轮发动机
 </a>
 数据等商业信息。这是美国司法部今年9月以来，第三次对江苏省国安厅情报人员提出指控。
 <br/>
 <br/>
 美国司法部指控，从2010年1月到2015年5月，中共江苏国安厅情报人员和同谋侵入多家美国公司的电脑系统，窃取商用航空技术数据。
 <br/>
 <br/>
 被指控的有：江苏国安厅情报人员查荣、柴孟，以及他们雇用的黑客张樟桂、刘春亮、高洪昆、庄晓伟和马志奇等。
 <br/>
 <br/>
 起诉罪名是盗窃一种
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_涡轮发动机.html" target="_blank">
  涡轮发动机
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核心技术.html" target="_blank">
  核心技术
 </a>
 。这种技术由一家法国公司和一家美国公司合作开发，法国公司在江苏苏州市设有办事处。
 <br/>
 <br/>
 美国司法部表示，江苏公安厅指示两名法国公司员工侵入公司电脑窃取数据。他们还侵入了制造该涡扇发动机部件的其它几家美国航空航天公司。
 <br/>
 <br/>
 这是自今年9月以来第三起涉及江苏国安厅情报人员的商业间谍案件。
 <br/>
 <br/>
 新唐人记者王羽音、张琪综合报导
</div>
