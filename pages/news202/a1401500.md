 周五，失踪的马来西亚航空
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_MH370.html" target="_blank">
  MH370
 </a>
 班机上的乘客和机组人员的家属向马来西亚官员递交了一些飞机碎片，他们认为这是失踪飞机的一部分。这些碎片是马达加斯加渔民发现的。MH370于2014年3月8日在从吉隆坡飞往北京的途中消失，据推测在印度洋南部坠毁。但尚无确切结论。
 <br/>
 <br/>
 周四，80多只鲸鱼在新西兰查塔姆群岛搁浅，之后30多条鲸鱼得以重新在水中浮起，回到大海。51条鲸鱼因为搁浅而死亡。其中一条因为状况不佳而被安乐死。他们的尸体被埋葬。
 <br/>
 <br/>

  洛克希德・马丁公司
 </a>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国防部.html" target="_blank">

 </a>
