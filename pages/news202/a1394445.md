### 巴西总统大选投票 〝巴西版川普〞引关注
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 选民纷纷涌向投票站，为支持的国家总统候选人投票，并同时要选出27位州长、多名国会议员等，这场选举被认为是巴西历史上最重要的选举。
 <br/>
 <br/>
 今年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统大选.html" target="_blank">
  总统大选
 </a>
 被认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 自军事独裁统治结束后30多年来，最具分裂性的选举之一。有〝巴西版川普〞之称的右翼社会自由党参选人、现任国会议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波索纳洛.html" target="_blank">
  波索纳洛
 </a>
 （Jair Bolsonaro）与工党的左翼候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈达德.html" target="_blank">
  哈达德
 </a>
 （Fernando Haddad）之间的竞争最为激烈。
 <br/>
 <br/>
 波索纳洛认为工党犹如委内瑞拉一样政治腐败，是由前总统卢拉（“Lula”da Silva）领导的经济上无能、道德又败坏的政治集团。
 <br/>
 <br/>
 波索纳洛反对同性恋，誓言要打击犯罪和腐败，支持拥枪权。由于他对犯罪的强硬立场，毫无掩饰的言论，以及清廉的联邦议员职业生涯，赢得很多选民支持。
 <br/>
 <br/>
 在9月6号的总统竞选集会上，波索纳洛被人持刀刺伤，被紧急送医。9月29号，才得以出院。然而，这次遇刺风波，反而推动了波索纳洛的支持率。
 <br/>
 <br/>
 根据最新民调，波索纳洛获得32%的支持率，哈达德落后10个百分点，支持率为22%。
 <br/>
 <br/>
 另外三位候选人，分别是戈麦斯（Ciro Gomes），阿尔克明（Geraldo Alckmin）和席尔瓦（Marina Silva）, 他们自称是明智的中间派，但支持率远不及波索纳洛和哈达德。
 <br/>
 <br/>
 新唐人电视安心综合报导
</div>
