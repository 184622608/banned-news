### 核灾32年后 切尔诺贝利再建太阳能电厂
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】10月5日，乌克兰在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_切尔诺贝利.html" target="_blank">
  切尔诺贝利
 </a>
 为一座太阳能发电厂揭幕，距离它100米的地方，恰恰是三十年前引发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核灾.html" target="_blank">
  核灾
 </a>
 难的切尔诺贝利核电站。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_切尔诺贝利.html" target="_blank">
  切尔诺贝利
 </a>
 开发太阳能公司官员 Vhen Variagin：〝这不是简单一座太阳能发电厂，它是唯一的发电厂，距离1986年发生爆炸的第四核反应炉仅100米。〞
 <br/>
 <br/>
 这座1兆瓦的太阳能发电厂由乌克兰和德国企业联合开发，耗资约120万美元。这里的3,800个面板可以为2,000套公寓提供能源。
 <br/>
 <br/>
 切尔诺贝利核电站负责人表示很高兴看到废弃核电站所在地还能出产太阳能电力。
 <br/>
 <br/>
 该地区已是受污染地区，基本上不适合居住。游客需要有导游带辐射指示计陪同。
 <br/>
 <br/>
 1986年4月26日，乌克兰北部切尔诺贝利核电站第四号反应炉发生震惊世界的爆裂事故，其爆炸和核辐射等，先后造成约9万人伤亡，另有30多万居民撤离该地区。
 <br/>
 <br/>
 新唐人电视台李子玉综合报导
</div>
