### 跟进美国 博尔索纳罗提出迁使馆到耶城
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】被称为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西版川普.html" target="_blank">
  巴西版川普
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔索纳罗.html" target="_blank">
  博尔索纳罗
 </a>
 ，将于2019年1月上任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西总统.html" target="_blank">
  巴西总统
 </a>
 ，而现在他的施政议程已备受瞩目。
 <br/>
 <br/>
 巴西新当选总统希望把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大使馆.html" target="_blank">
  大使馆
 </a>
 从特拉维夫迁往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_耶路撒冷.html" target="_blank">
  耶路撒冷
 </a>
 。
 <br/>
 <br/>
 巴西新当选总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔索纳罗.html" target="_blank">
  博尔索纳罗
 </a>
 ：〝我认为迁移以色列大使馆没有任何问题。〞
 <br/>
 <br/>
 巴西新总统还提出任命反贪腐法官莫罗为司法暨公共安全部部长。
 <br/>
 <br/>
 2014年起，莫罗担任了巴西大规模反贪腐调查〝洗车行动〞的一审法官，他铁面无私，不惧各方压力将巴西前总统鲁拉
 <br/>
 <br/>
 巴西联邦法官 莫罗：〝我明白这个国家需要打击腐败和犯罪。如果大家能众志成城一起努力，我们是可以做到的。〞
 <br/>
 <br/>
 新总统对此表示欣赏。
 <br/>
 <br/>
 巴西新当选总统 博尔索纳罗：〝这让我感到高兴，因为始终困扰巴西人的事情之一就是腐败问题。现在，莫罗将在巴西全力打击腐败。〞
 <br/>
 <br/>
 新唐人记者李子玉综合报导
