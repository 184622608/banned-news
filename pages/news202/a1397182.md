### 印尼客机坠毁 189人全部遇难
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】印尼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_廉价航空.html" target="_blank">
  廉价航空
 </a>
 公司〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮子航空.html" target="_blank">
  狮子航空
 </a>
 〞（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Lion Air.html" target="_blank">
  Lion Air
 </a>
 ）的一架载有189人的客机，周一早上6点20分从雅加达起飞后不久坠落大海。据印尼官方证实，此次空难无人生还。
 <br/>
 <br/>
 印尼官方表示，一架隶属于〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮子航空.html" target="_blank">
  狮子航空
 </a>
 〞公司的航班编号为JT610的波音737客机，周一早上6时20分从雅加达的苏卡诺哈达（Soekarno-Hatta）国际机场起飞13分钟后就失去联系。追踪记录显示，飞机在起飞时向南环绕，然后向北飞行，之后就突然在离岸不远的爪哇岛海域坠毁。飞机共搭载了189人，其中包括1名儿童、2名婴儿和7名机组成员。
 <br/>
 <br/>
 印尼当局表示，正在飞机坠毁的海域进行大规模的搜救行动。
 <br/>
 <br/>
 印尼国家救灾局负责人MUHAMMAD SYAUGI：〝我们是在早上6点50分接到报告，说与狮航的JT610航班失去了联系。〞
 <br/>
 <br/>
 印尼国家救灾局负责人MUHAMMAD SYAUGI：〝我相信在军方和警察的支持下，救援工作应该会非常地迅速。根据报告，包括乘客和机组成员，（失事客机）总共有189人。〞
 <br/>
 <br/>
 当局表示，必须要等到找到飞机黑匣子后，才能确认飞机坠毁的原因。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
