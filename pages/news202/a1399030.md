### 纪念〝一战百年〞法国行结束 川普回美
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】周一，美国总统川普与夫人返回美国，结束为期3天的法国纪念一战100周年活动。川普发布推文，重申此行要点。
 <br/>
 <br/>
 美国总统川普携第一夫人梅拉尼娅11月9日至11日出访法国巴黎，与法国总统马克龙以及其他国家领导人一道，参加纪念第一次世界大战停战100周年系列活动。
 <br/>
 <br/>
 周一，美国总统川普与夫人搭乘专机返回美国。
 <br/>
 <br/>
 川普连续发表三则推文，重申此行要点。
 <br/>
 <br/>
 他说，〝在法国与各国领袖的会晤中收获很多，讨论美国在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事.html" target="_blank">
  军事
 </a>
 和贸易领域必须得到公平对待，从来不是件容易事。我们为其他国家支付很大部分的军事保护。〞〝还在与这些国家的贸易中损失了数千亿美元。我告诉他们，这种情况不可能继续下去。〞
 <br/>
 <br/>
 法国之行，川普总统与加拿大总理特鲁多讨论了钢铁关税问题，特鲁多周一证实了此事。
 <br/>
 <br/>
 加拿大总理特鲁多：〝钢铁和铝的关税问题仍然受到加拿大人关注，就像许多美国公民和公司也很关心。我希望我们能在阿根廷G20峰会前，解决这个问题。〞
 <br/>
 <br/>
 新唐人记者李梅综合报导
</div>
