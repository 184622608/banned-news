### 孟晚舟被捕最新内幕  华为涉嫌三宗罪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】中国通讯巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 副董事长孟晚舟，遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 警方逮捕5天后事件才被披露出来，美方已经确认事前知情。知情人士透露，孟晚舟被捕，涉三宗罪。有美媒称，其中涉嫌利用全球银行体系逃避美国对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 的制裁，是美国调查的一部分。
 <br/>
 <br/>
 路透社7日引述知情人士透露，中国电信巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 首席财务官孟晚舟被捕，是美国对该公司涉嫌利用全球银行系统逃避美国对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 制裁调查的一部分。
 <br/>
 <br/>
 至少从2016年以来，美国一直在调查华为技术有限公司，是否违反了美国对伊朗的制裁令，以及最近该公司通过滙丰控股(HSBA.L)进行涉及伊朗的非法交易。
 <br/>
 <br/>
 报导说，2012年，滙丰银行(HSBC)支付了19.2亿美元，并与位于布鲁克林的美国联邦检察官办公室，就违反美国制裁和洗钱法达成了一项暂缓起诉协议。滙丰发言人拒绝置评。
 <br/>
 <br/>
 孟晚舟是华为创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的女儿。现任公司CFO、副董事长，担任华为系旗下共11间公司的董事。华为除了被曝涉嫌利用全球银行系统逃避美国对伊朗制裁调查外，还被指控是中共间谍机构、威胁美国国家安全等。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 司法部发言人表示，应美国当局要求，孟晚舟于12月1日被捕，法庭定于周五举行听证会。
 <br/>
 <br/>
 孟晚舟被捕消息传出后，美国共和党参议员克鲁兹、鲁比奥和Ben Sasse对逮捕行动表示赞成，他们说，华为对美国国家安全构成威胁。
 <br/>
 <br/>
 鲁比奥6日发表声明说，华为与中共政府有直接联系，长期以来对美国国家安全构成严重威胁。
 <br/>
 <br/>
 克鲁兹在推特说：〝华为是一个伪装成一家电信公司的共产党间谍机构。〞
 <br/>
 <br/>
 联邦民主党参议员沃纳在推特上说，华为与中兴一样违反了美国的制裁法律。对美国的国家安全构成威胁。他也呼吁加拿大重新考虑把华为纳入其5G基础设施任何层面的做法。
 <br/>
 <br/>
 美国国家安全顾问博尔顿（John Bolton）受访时透露，他事先就从美国司法部收到消息，孟晚舟将被逮捕。不过，他不确定川普总统是否事先知情，因为当天川普和习近平在阿根廷举行晚宴会。
 <br/>
 <br/>
 对于孟晚舟面对美国司法制度的审判，博尔顿表示，中国科技巨头华为与伊朗的商业往来，及窃取知识产权等问题，早已引起美国政府注意。
 <br/>
 <br/>
 华为是全球最大的电讯设备制造商。多年以来，西方情报机构一直质疑华为设备的安全性，认为它受制于中共政府，存在间谍和安全隐患。
 <br/>
 <br/>
 美国此前已经采取了一系列行动，阻止或减少华为和中兴通讯公司进入美国市场。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：戴明）
