### 第二波大篷车难民向墨西哥进发 疑有资助
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】在第一批中美洲〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 〞继续向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 行进之时，危地马拉的第二批难民一千多人，周二，开始向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 边境进发。美国副总统彭斯称，大篷车难民有委内瑞拉资助。
 <br/>
 <br/>
 这些人大部分是洪都拉斯人，从危地马拉东部的奇基穆拉（Chiquimula, Guatemala）徒步前行。有当地媒体报导人数可能超过两千人。
 <br/>
 <br/>
 未来几天，可能还会有其他
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 从中美洲地区出发。这些人受到媒体鼓励，认为往北前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 是更安全的路线。
 <br/>
 <br/>
 洪都拉斯当局称，在难民路线上已经至少有两名男子身亡。一人从卡车上掉下来，另一人在危地马拉试图爬上卡车时死亡。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 副总统彭斯周二在白宫记者会上提到，大篷车难民有委内瑞拉资助。
 <br/>
 <br/>
 美国副总统 彭斯：〝我和洪都拉斯总统赫尔南德斯通过话，他告诉我通过墨西哥边境、正朝着（美国）南部边境前进的大篷车难民，是由左翼团体组织、受委内瑞拉资助的。〞
 <br/>
 <br/>
 彭斯说，美国执法部门每天都要阻止恐怖份子进入美国，难保没有这些人混入难民中。
 <br/>
 <br/>
 美国副总统 彭斯：〝美国每天都要介入和阻止10多名恐怖份子或是有嫌疑的恐怖份子进入我们国家，所以很难确保不断增长的大篷车难民中没有中东人。〞
 <br/>
 <br/>
 周三一早，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 连发推文，以欧洲为例，指出严守边境的重要。
 <br/>
 <br/>
 推文说：〝那些想要为非法移民争取权益的人，好好看看过去五年欧洲的情况吧。一片混乱！他们只希望能重新做出决定。〞
 <br/>
 <br/>
 另一则推文说：〝我们是伟大的主权国家。我们有强大的边境，永远不允许非法进入美国。〞
 <br/>
 <br/>
 川普特意放上奥巴马在担任伊利诺州参议员时的一段讲话。
 <br/>
 <br/>
 美国前总统 奥巴马：〝那些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法入境.html" target="_blank">
  非法入境
 </a>
 者，这样做是不尊重法律规定，不顾那些守法的人。我们不能允许人们涌入美国，没有被发现，没有移民证件，没有调查（背景）。〞
 <br/>
 <br/>
 川普说：〝我百分百赞同奥巴马总统所言〞。
 <br/>
 <br/>
 新唐人记者安心、张琪综合报导
</div>
