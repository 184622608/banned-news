### 大炮故障？朝鲜炮门没关对准韩国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】韩朝今年9月签定《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_919军事协议.html" target="_blank">
  919军事协议
 </a>
 》，11月1日起停止双方敌对行动。但今（5）日，韩国国防部公共关系室在记者上表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 有一门位于黄海军事线〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北方界线.html" target="_blank">
  北方界线
 </a>
 （NLL）〞的大炮疑似故障，炮门没有关且正对着韩国，目前已通知朝鲜官方，要求将炮门关上。
 <br/>
 <br/>
 《韩联社》报导，韩朝今年9月进行高峰会，签定了《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_919军事协议.html" target="_blank">
  919军事协议
 </a>
 》，本月1日起停止双方在地面、海上、空中的一切敌对行动。将两国边境的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 半岛西部海域化为〝缓冲水域〞，朝鲜也将封闭部署于朝鲜半岛西部的海岸火炮。邻近韩国的黄海道内陆地区的所有火炮也不得向缓冲水域内射击。
 <br/>
 <br/>
 韩国国防部公共关系室5日在例行记者会上表示，朝鲜在黄海一带的大炮也全部关上，但却有一门大炮尚未关闭，依然对着韩国。目前国防部已对朝鲜送出公文通知，要求关闭炮门，而朝鲜的接洽窗口则表示将会报告上级，并采取必要措施。
 <br/>
 <br/>
 据韩国军方消息人士指出，在朝鲜窗口联络上级后发现，疑是炮门故障才会发生此状况。
 <br/>
 <br/>
 (责任编辑：程以仁)
