### 脱欧协议迟迟难达成 金融机构纷纷留后路
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】脱欧问题又一次让金融界坐上过山车。先是《泰晤士报》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国.html" target="_blank">
  英国
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 达成临时协议，允许英国在脱欧后继续拥有欧盟金融市场准入许可，但欧盟和英国政府随后都否认了这个説法。
 <br/>
 <br/>
 金融业是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国.html" target="_blank">
  英国
 </a>
 经济支柱之一。2017年，金融业产出占英国GDP的6.5%。其中一半来自伦敦。
 <br/>
 <br/>
 作为全球货币交易中心，流入和流出伦敦的资金比世界上任何一个城市都多。
 <br/>
 <br/>
 很多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 外国家的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融机构.html" target="_blank">
  金融机构
 </a>
 把英国当作进入欧盟的跳板，将欧洲总部设在伦敦。
 <br/>
 <br/>
 因为担心英国没有达成协议就脱欧，美国银行、瑞士信贷集团、花旗银行、摩根士丹利、高盛，汇丰等大型金融机构都已计划将欧洲总部或大批员工转移到其他欧盟国家。
 <br/>
 <br/>
 德意志银行也将把数千亿资产转回德国。
 <br/>
 <br/>
 所以，当周四媒体报导说，只要英国在金融监管方面和欧盟〝同步〞，英国在脱欧后就将继续拥有欧盟金融市场准入时，金融业大大松了一口气。英镑兑美元上涨0.5个百分点。
 <br/>
 <br/>
 但随后，欧盟首席脱欧谈判代表巴尼耶在推文中表示，这个消息是〝误导〞。强调只有欧盟才能决定授予哪个国家欧盟市场准入。但他也表示，欧盟已准备好就金融监管和英国进行密切对话。
 <br/>
 <br/>
 英国首相梅则对金融时报表示，这个消息是〝猜测〞。她的一名助手表示:〝进行建设性谈判和宣布协议已经达成之间存在着很大差异。〞
 <br/>
 <br/>
 在脱欧谈判过程中，经常传出消息说有根本性进展了,但结果不一定是这样。英国脱欧大臣拉布曾经表示在10月欧盟峰会上能取得进展，并且在几个月内达成协议。欧盟委员会主席容克也说，11月底之前能达成协议。但是10月峰会上不但没取得进展，欧盟甚至还取消了专门为脱欧召开的11月紧急峰会，说等有进展了再开会。各种各样的消息很多，不少企业也已经找到了对策，比如花旗银行就明确表示，不打算等待脱欧谈判的结果了，无论怎样，他们都继续扩展英国以外的业务。
 <br/>
 <br/>
 新唐人电视综合报导
