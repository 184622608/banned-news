### 美日澳联手反制 中共争夺巴新航母港口失败
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】北京领导人日前出席亚太经济合作峰会（APEC），对主办国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴布亚新几内亚.html" target="_blank">
  巴布亚新几内亚
 </a>
 （巴新）再次大手笔撒币，但不敌美日澳联手。美方在峰会上宣布，美澳将参与重建巴新一处可停航母的港口海军基地。而此前曾传中方将获得该港口开发权。
 <br/>
 <br/>
 11月17日，美国副总统彭斯在APEC峰会上辛辣讥讽中共的〝一带一路〞是条〝不归路〞。彭斯随后宣布，美国将加入澳大利亚和巴新，在巴新的马努斯岛（Manus Island）上重建联合海军基地。彭斯表示，美国将和这些太平洋岛国合作，保护其主权和海上权利。
 <br/>
 <br/>
 马努斯岛在第二次世界大战期间，曾是美国的一个主要海军基地，在华盛顿的太平洋战略中发挥了关键作用。
 <br/>
 <br/>
 彭斯宣布该消息之时，外界正在担心中共可能率先开发马努斯岛的深水港。8月曾有消息指，中共可能取得马努斯岛一个港口的开发合约。而岛上的深水港可停泊航空母舰，以及数百艘军舰。
 <br/>
 <br/>
 澳媒引述分析指，马努斯基地将把美军驻地进一步向太平洋南部推进，这使得中共在太平洋地区的战略野心严重受挫。
 <br/>
 <br/>
 值得注意的是，也是在17日，美日澳发表联合声明，宣布将推动在印太地区的基础建设开发计划，其战略意图明显针对中共的〝一带一路〞。本月稍早时候，澳大利亚宣布将投资太平洋22亿美元，彭斯和日本首相安倍晋三则宣布向印太地区投资700亿美元。
 <br/>
 <br/>
 而中共也在加大对太平洋岛国的投资，其中仅对巴新的贷款就达约5.9亿美元。这次APEC峰会期间，中方再次承诺提供40亿美元融资，协助巴新兴建全国道路网络。
 <br/>
 <br/>
 不过，巴新在APEC峰会上的表现令中共受挫。除了敲定和美澳合作开发军事基地外，巴新外长还强硬拒绝接见中共官员，以〝商讨〞峰会联合公报的措辞。
 <br/>
 <br/>
 有外媒评论，这些太平洋岛国对待中共投资的态度是拿走〝糖果〞，退回〝炮弹〞。
 <br/>
 <br/>
 这些岛国拒绝倒向中共，除了因为美国及其盟友的强大经济实力外，对中共〝输出中国模式〞的担忧也是一个主要原因。
 <br/>
 <br/>
 中共的〝一带一路〞，除了输出腐败之外，其制造的债务危机往往成为侵蚀沿线国家主权的工具，因此〝一带一路〞被指为中共的殖民政策。
 <br/>
 <br/>
 而美日澳联合声明称，对印太国家的经济援助，将遵守公开、透明及财政永续性等国际标准和原则，不会给相关国家带来负担。彭斯更明确，美方的支持将使这些国家真正受益，华盛顿不会推出带有约束力的〝带〞，也不输出单方向的〝路〞。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
