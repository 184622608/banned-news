### 匹兹堡犹太教堂枪击案 嫌犯被控29项重罪
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】新闻开始，先来关注周六发生在匹兹堡市的犹太教堂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，共造成11人死亡，6人受伤。周日，当局公布了死者名单。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌犯.html" target="_blank">
  嫌犯
 </a>
 被指控29项重罪。
 <br/>
 <br/>
 宾州西部地区联邦检察官 Scott Brady：〝在犹太教堂的致命攻击中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌犯.html" target="_blank">
  嫌犯
 </a>
 发出种族灭绝性言论，以及想杀害犹太人的言论。在和警方对持之后，嫌犯终究投降，目前正被拘押。〞
 <br/>
 <br/>
 46岁的嫌犯罗伯特・鲍尔斯，使用3个格洛克17手枪和一支AR-15自动步枪，攻击美国宾州匹兹堡的犹太教堂。
 <br/>
 <br/>
 该嫌犯已被控告29项联邦重罪，其中11项为使用枪枝武器造成伤害，7项为对警方动用武力，还有11项为因受害人的宗教信仰而对其谋杀。
 <br/>
 <br/>
 宾州西部地区联邦检察官 Scott Brady：〝在敬神的场所发起攻击，尤其令人发指。那是神圣的地方，是慈悲且祥和的场所，是人们聚集在一起，表达最珍视的信仰的地方。〞
 <br/>
 <br/>
 事件发生在犹太教的安息日，当时犹太教堂〝生命之树〞正在进行一场新生儿命名仪式。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 造成11人死亡，6人受伤，伤者包括4名警察。
 <br/>
 <br/>
 当局周日也公布了死者名单，其中最年轻的54岁，最年长的97岁，包括一对兄弟和一对夫妻。
 <br/>
 <br/>
 宾州西部地区联邦检察官 Scott Brady：〝今天我们站在一起，拒绝仇恨和暴力，尊重邻居，包容不同信仰，尊重彼此的差异。〞
 <br/>
 <br/>
 〝生命之树〞犹太教堂已经有超过150年历史，教堂所在的匹兹堡松鼠丘地区（Squirrel Hill）从1920年就有众多犹太人聚居。
 <br/>
 <br/>
 全美的犹太教堂和犹太中心在此枪击事件后，都加强部署警力巡逻。
 <br/>
 <br/>
 新唐人记者王愉贺综合报导
