### 182米世界最高雕像揭幕 印度军机洒花庆贺
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】全世界最高的独立英雄
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_帕特尔.html" target="_blank">
  帕特尔
 </a>
 雕像，周三（31日）在印度揭幕。总理莫迪主持了揭幕式，落成典礼上施放烟火，还有军机凌空洒下大量花瓣雨庆贺。
 <br/>
 <br/>
 莫迪（Narendra Modi）周三出席雕像揭幕式时表示，印度历史会记得这一天，这是他个人最喜欢的计划，该雕像更是〝我国工程与技术高超实力的象征。〞
 <br/>
 <br/>
 当他站在雕像前方行礼时，军机飞越雕像上空，并洒下玫瑰花瓣祝贺。落成典礼上还施放了烟火、安排舞蹈炒热气氛，场面相当盛大。
 <br/>
 <br/>
 雕像建在莫迪的家乡古茶拉底省（Gujarat）西部的1座河中岛。建这座〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_团结雕像.html" target="_blank">
  团结雕像
 </a>
 〞（Statue of Unity）是为了向独立英雄
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_帕特尔.html" target="_blank">
  帕特尔
 </a>
 （Sardar Patel）致敬，帕特尔出生于1875年10月31日，在印度独立运动中发挥主要作用。他是印度独立后的首任内政部长，有〝铁人〞绰号。
 <br/>
 <br/>
 雕像高达182米，约61层楼高，是美国自由女神像（93米）的近2倍，共用了约1850吨的青铜。该雕像耗时近4年完成，建造过程中预算不断增加，据监工委员会估计，耗资超过4.1亿美元。
 <br/>
 <br/>
 莫迪的民族主义政府将雕像作为主要建设，以迎战明年国家大选。当局31日起开放游客预约上雕像景观台参观，门票为350卢比（约4.76美元），盼能藉此带动当地观光。
 <br/>
 <br/>
 但虽说雕像有吸引旅游促进经济考量，不过，当局因为明年大选造势以及雕像工程迁移大量居民和征收土地引发当地民众抗议，甚至有22座村庄的首长连署呼吁莫迪不要参加揭幕典礼。
 <br/>
 <br/>
 据法新社报导，揭幕式当天警方警戒如临大敌。古茶拉底省官员担心社区团体可能组织抗议活动，要求为建造雕像征收的土地，支付赔偿金。
 <br/>
 <br/>
 讷尔默达区（Narmada）警察总监楚达苏马（Abhay Chudasama）表示：〝超过5000名警力已分派到雕像方圆10公里的不同驻点。〞
 <br/>
 <br/>
 楚达苏马说，〝随着当地社区及领袖发起抗议活动，已加强巡逻，无人机及直升机也将持续监看整个区域。〞
 <br/>
 <br/>
 但当地消息人士表示，数名活动人士已秘密行动。报导称，莫迪及古茶拉底省首长鲁帕玛（Vijay Rupan）的海报不是被拆除，就是脸部遭涂黑。
 <br/>
 <br/>
 报导称，当地议员兼社区团体领导人瓦沙瓦（Chotu Vasava ）表示：〝我不是反对帕特尔，但如果此地居民必须因此受苦并从家园搬离，那这雕像有什么用？〞
 <br/>
