### 贸战阴云 华南逾7成美企拟缓投资或搬迁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 至今无缓和迹象。最新消息称，如下月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会上的〝川习会〞没有成果，美国将对剩余所有中国货征税。受此影响，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华南.html" target="_blank">
  华南
 </a>
 逾7成美国公司考虑推迟在当地的进一步投资，并将部分或全部的制造活动迁移至其他国家。
 <br/>
 <br/>
 根据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华南.html" target="_blank">
  华南
 </a>
 美国商会近日公布的一项商业调查显示，美国企业认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 将给他们带来的损失，要比其他任何国家的企业还大。超过70%的美国公司考虑延后或取消在中国进一步投资，并拟将部分或全部的制造活动迁移至其他国家。这些美国企业的中国同业，有半数也做出延后投资或迁移的考量。
 <br/>
 <br/>
 其中，64%的公司称，他们正考虑将生产线移至中国之外。调查发现，中美贸易战正在导致供应链以及产业集群发生转移，多数转往东南亚。
 <br/>
 <br/>
 华南美国商会会长Harley Seyedin表示，这些公司的客户也在观望，订单正在减少，有些客户或者根本就不下单，〝他们很有可能会等到形势更为确定之后才下单。〞
 <br/>
 <br/>
 近50%的受访企业称，中国的非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 壁垒上升，如中共的官僚监管增加及海关清关速度放慢。分析师说，这些是中共对美国公司的报复措施。
 <br/>
 <br/>
 据悉，位于广州的华南美国商会有2,000多个会员。9月21日至10月10日期间，即美国对价值2,000亿美元中国商品加征关税后不久，该商会向219家会员企业进行了这项问卷调查，其中三分一从事制造业。
 <br/>
 <br/>
 11月份在阿根廷的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会期间，中美两国首脑的会谈被各界看作是缓解贸易战的机会。彭博社的报导称，美国政府官员正着手准备，一旦G20〝川习会〞不能带来任何进展，美国准备在12月初宣布对剩余大约价值2,570亿美元的中国商品开征关税。外界估计，如果这种情况发生，恐怕将有更多的美国在华企业撤离中国。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：戴明）
