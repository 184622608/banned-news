### 沙特记者失踪案 美国务卿：再给沙特几天
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】美国国务卿蓬佩奥周三晚从中东返回。周四与川普总统会面后召开记者会，就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特记者失踪案.html" target="_blank">
  沙特记者失踪案
 </a>
 表态：再给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 几天时间调查再做决定。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝我今天早上告诉川普总统，我们应该再给他们（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 ）几天时间来完成这件事。〞
 <br/>
 <br/>
 蓬佩奥表示，他也会见了土耳其总统。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝我们确信，在这两个国家的努力下，会出现一个完整的画面，呈现实际发生的情况。 〞
 <br/>
 <br/>
 财政部长姆钦在推特上表示，在周四早上与川普和国务卿会面后，他不会按最初计划参加沙特的未来投资计划峰会。许多媒体和商界领袖在记者卡舒吉失踪后，退出了会议。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
