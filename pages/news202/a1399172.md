### 巴黎恐袭三周年 巴黎伦敦市长出席纪念
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】周二，法国总理、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎市长.html" target="_blank">
  巴黎市长
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦市长.html" target="_blank">
  伦敦市长
 </a>
 出席了在巴黎举行的，巴黎恐怖袭击案
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三周年纪念仪式.html" target="_blank">
  三周年纪念仪式
 </a>
 。
 <br/>
 <br/>
 〝巴塔克兰剧场〞门前的纪念碑上刻着90位遇难者名字，法国总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲利普.html" target="_blank">
  菲利普
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎市长.html" target="_blank">
  巴黎市长
 </a>
 伊达尔戈献上花圈，全场默哀一分钟。
 <br/>
 <br/>
 2015年11月13日，巴黎发生了震动世界的一连串恐怖袭击案件，导致130人遇害。
 <br/>
 <br/>
 恐怖分子在法兰西体育场、〝巴塔克兰剧场〞以及六家酒吧和咖啡厅，发动了恐怖袭击，极端组织〝伊斯兰国〞随后宣布对恐袭负责。
 <br/>
 <br/>
 当晚，当时的法国总统奥朗德宣布，全国进入紧急戒备状态，这一状态一直持续了两年。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
