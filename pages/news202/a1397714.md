### 俄罗斯制裁乌克兰 乌总统：停止恫吓
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 当局周四宣布，将对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 的政商精英实施全面
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融制裁.html" target="_blank">
  金融制裁
 </a>
 。乌克兰总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波罗申科.html" target="_blank">
  波罗申科
 </a>
 表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莫斯科.html" target="_blank">
  莫斯科
 </a>
 应该从乌克兰领土撤出，而不是恫吓他国。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 总理梅德韦杰夫周四签署法令，针对322名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 人和68家企业进行制裁，制裁内容包括冻结其在俄罗斯拥有的资产。
 <br/>
 <br/>
 制裁对像有乌克兰总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波罗申科.html" target="_blank">
  波罗申科
 </a>
 的长子阿列克谢、前总理季莫申科、乌克兰国家安全局局长赫里察克、内政部长阿瓦科夫以及多名政府高官。
 <br/>
 <br/>
 另外，俄罗斯还将禁止从乌克兰进口部分商品。
 <br/>
 <br/>
 梅德韦杰夫表示，此举是针对基辅对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莫斯科.html" target="_blank">
  莫斯科
 </a>
 实施制裁，而制定的反制裁措施。
 <br/>
 <br/>
 波罗申科和正在当地访问的德国总理默克尔，在记者会上做出回应。
 <br/>
 <br/>
 乌克兰总统 波罗申科：〝俄罗斯只需做一件非常简单的事情，从乌克兰撤走，撤出部队和武器，恢复乌克兰的领土完整。 这将带来和平，也不需要实施任何制裁。〞
 <br/>
 <br/>
 默克尔则表示，由于莫斯科未能完全执行《明斯克协议》，德国将支持欧盟延长对俄罗斯的制裁。
 <br/>
 <br/>
 德国总理 默克尔：〝《明斯克协议》未能有效执行，我们只看到微乎其微的进展，时不时还会出现倒退。就像今天发生的情况，德国支持（欧盟）在12月延长对俄制裁。〞
 <br/>
 <br/>
 新唐人记者曹菁综合报导
</div>
