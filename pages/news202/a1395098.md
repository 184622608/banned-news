### 俄罗斯客车货车相撞 至少12人死
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 楚瓦什共和国发生严重交通意外，已知至少有12人丧生，10人受伤。
 <br/>
 <br/>
 事发地点在M7联邦公路楚瓦什共和国路段，一辆承载19名乘客的客车，与一辆货车迎头相撞，导致客车严重损毁。
 <br/>
 <br/>
 两架车上共承载约22人，已知至少有12人死亡，10人受伤，其中3人伤势严重。目前事故原因正在正调查中。
 <br/>
 <br/>
 新唐人记者陈依心、宇薇综合报导
