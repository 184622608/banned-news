### 蓬佩奥北京会王毅 双方发言尖锐分歧
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 第三次亚洲行，最后一站抵达北京。周一，蓬佩奥与中国外交部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 、国务委员杨洁篪会谈。谈到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 ，王毅将所有矛盾归咎美国，蓬佩奥则直言两国存在根本分歧。
 <br/>
 <br/>
 美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 周一抵达北京，与中共外交部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 和国务委员杨洁篪举行会谈。
 <br/>
 <br/>
 杨洁篪称，中美作为两大国，合作才会双赢，双方必须做出正确的抉择。
 <br/>
 <br/>
 蓬佩奥表示，期待双方进行坦承对话。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝我期待讨论影响我们两国和两国人民的广泛议题。我期待今天与你进行良好、坦诚、建设性的对话。〞
 <br/>
 <br/>
 蓬佩奥与中共外交部长王毅会面时，双方发言异常尖锐。
 <br/>
 <br/>
 王毅将近期
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 ，包括贸易摩擦、台湾主权问题等矛盾归咎于美国。
 <br/>
 <br/>
 蓬佩奥直言，两国之间存在根本分歧。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝我很遗憾，我们两国之间的战略对话，你们全都选择不进行。这原本对我们来说是个重要机会，可以讨论更广泛的谈判条件、重要议题，为两国人民提供机会。〞
 <br/>
 <br/>
 美中战略安全对话原订十月在北京举行，中共藉口关税等冲突临时取消，却反过来说是美方取消的。
 <br/>
 <br/>
 9月30日晚间，美国国防官员表示，中方改变计划，取消了今年的中美战略对话，因为中共国务委员兼国防部长魏凤和不能参加会议。
 <br/>
 <br/>
 新唐人记者唐楣、田瑞综合报导
 <br/>
