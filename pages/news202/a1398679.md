### 首尔出租公寓惊传火警 酿7死11伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首尔.html" target="_blank">
  首尔
 </a>
 一栋3层楼出租
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公寓.html" target="_blank">
  公寓
 </a>
 9日清晨发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大火.html" target="_blank">
  大火
 </a>
 ，由于起火地点通道狭小、宛如迷宫，至少造成7名租客死亡，11人受伤。
 <br/>
 <br/>
 据韩联社（Yonhap News Agency）报导，死伤者多是40至60岁的临时工或街头小贩，住在没有洒水灭火装置破旧建筑内。
 <br/>
 <br/>
 住在对街的61岁目击者说：〝我听到尖叫声后走出去，然后看到这栋建筑笼罩在大量的烟雾和火焰中。〞
 <br/>
 <br/>
 报导称，起火地点是名为考试院（goshiwon）的建筑，这类廉价出租
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公寓.html" target="_blank">
  公寓
 </a>
 的走廊一般非常狭窄，容易发生火患。过去5年来，当地已发生超过250起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大火.html" target="_blank">
  大火
 </a>
 。
