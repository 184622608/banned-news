### 自驾游事故飙升 日东北给外国司机发辨识标签
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】出国旅游租车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自驾.html" target="_blank">
  自驾
 </a>
 游览，方便快捷，但因道路不熟也造成当地车祸事故增加。对此，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 东北6县地方政府决定，给外国访客的车上加一个〝外国朋友驾车中〞的磁铁标签，以增加辨识度，希望能减少意外发生。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 朝日新闻报导，近年来，来自外国的访客持续增加，尤其以枫叶转红的10月最多。
 <br/>
 <br/>
 东北观光推动机构表示，造访东北的外国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观光客.html" target="_blank">
  观光客
 </a>
 大多是重游客，这些习惯旅行的游客，因东北交通不是很方便，经常利用租车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自驾.html" target="_blank">
  自驾
 </a>
 方式旅游。许多访客一下机场就租车，也引发增加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交通事故.html" target="_blank">
  交通事故
 </a>
 的忧虑。
 <br/>
 <br/>
 该机构称，外国司机2017年在东北地区租车次数达6935次，比2016年成长267%，甚至连外国访客的住宿次数都成长146%。
 <br/>
 <br/>
 国土交通省表示，选择租车自驾的人数，从2011年的17万9000人，成长到2015年的70万5000人，成长幅度近4倍。
 <br/>
 <br/>
 如果以日本整体租车业来看，每年的交通事故数约6000件左右，且有稍微减少的倾向。但外国人自驾的交通事故件数则持续增加。
 <br/>
 <br/>
 根据公益财团法人交通事故综合分析中心所做的分析，外国司机成为车祸时过失责任比率较高的〝第一当事者〞件数，从2012年的19件，成长到2017年的121件，成长幅度约6倍。
 <br/>
 <br/>
 为因应这种情况，全国租车协会一月制定提升访日外国人租车服务行动方案时，就提出发给外国司机一张可供其他司机识别的贴纸等方法，作为安全对策。
 <br/>
 <br/>
 近日，由东北6县地方政府及观光相关业者所组成的东北观光推动机构，针对外国人租车时发给磁铁标签等方式，让其他日本驾驶能清楚辨识，盼减少意外事故。
 <br/>
 <br/>
 目前，东北6县各租车公司都已拿到磁铁标签，设计图是一个爱心形状，大小为长宽15厘米的正方形，上面用日文写着〝外国朋友驾车中〞，下方还有一个不死鸟的图案，象征东北地区走过311大地震。
 <br/>
 <br/>
 东北观光推动机构成员呼吁日本司机，看到这张标签时，请保持与新手司机一样的安全距离。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
