### 【禁闻】APEC闭幕 中共官员强闯东道国外交部
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】在巴布亚新几内亚召开的亚太经合组织
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 11月18号闭幕，但未能就领袖宣言达成一致。
 <br/>
 <br/>
 巴布亚新几内亚总理奥尼尔在闭幕记者会上表示，因与会领袖未能在宣言上就世界贸易组织改革等问题达成共识，以及美中两国对声明文本表述有很大分歧，峰会领袖宣言流产。
 <br/>
 <br/>
 〝路透社〞引述一名参与峰会谈判的外交官的话说，中共外长王毅反对美国要求在共同宣言草稿中，加入反对〝不公平贸易行为〞、改革世贸组织的段落。
 <br/>
 <br/>
 这是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 史上首次未能按照惯例发表共同宣言。奥尼尔表示，他将以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东道国.html" target="_blank">
  东道国
 </a>
 身份代表全体成员，发表〝主席声明〞代替。
 <br/>
 <br/>
 澳洲媒体及〝法新社〞引述消息来源报导说，出席峰会的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共官员.html" target="_blank">
  中共官员
 </a>
 因为领袖宣言的措辞希望见东道国外长，被拒绝后，一度强行闯入巴新国外长办公室，最后在保安人员到场介入后才离开。有舆论形容北京这是〝恃强霸凌〞。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金马奖.html" target="_blank">
   金马奖
  </a>
  颁奖风波 陆艺人齐站队遭讥讽
 </b>
 <br/>
 <br/>
 第55届金马奖颁奖典礼11月17号晚在台北举行，获得最佳记录片奖的导演傅榆，在获奖感言中称:〝希望有一天我们的国家可以被当成真正独立的个体来看待，这是我生为台湾人最大的愿望。〞这番话,  被指称是台独言论，挑动了中共的敏感神经，大陆多个直播颁奖典礼的网络频道立即中断直播。
 <br/>
 <br/>
 中共喉舌媒体《人民日报》官方微博也迅速发帖，以一张包括有台湾的中国地图，配文指〝一点都不能少〞，引导
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆艺人.html" target="_blank">
  大陆艺人
 </a>
 站队。短短几个小时, 不仅一线当红艺人一起转发，就连深陷逃税丑闻的范冰冰也参与其中。
 <br/>
 <br/>
 而在颁奖现场的大陆艺人，除了导演娄烨外，其他人还一齐缺席了颁奖典礼结束后的惜别餐会。
 <br/>
 <br/>
 大陆演员的表现，成为这场颁奖典礼中比获奖作品还要热门的话题。
 <br/>
 <br/>
 有网友说：〝今晚的金马奖，大陆演职人员演绎了什么叫‘没有沉默的自由’。〞
 <br/>
 <br/>
 也有网友讽刺说：〝饭都不吃还怎么两岸一家亲。〞
 <br/>
 <br/>
 编辑/周玉林
