### 乌克兰十省戒严 谨防俄罗斯陆上威胁
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 将从周三上午9点开始，在边界十省实施
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戒严.html" target="_blank">
  戒严
 </a>
 令，因应
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 的侵扰。乌克兰总统波罗申科警告，俄罗斯对乌克兰构成的陆上威胁极为严重。
 <br/>
 <br/>
 别尔江斯克市位于亚速海北岸，周二开始准备实施
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戒严.html" target="_blank">
  戒严
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 四年前并吞克里米亚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 当时没有宣布戒严。俄罗斯今年在俄罗斯和克里米亚之间建筑的桥梁开通，使紧张局势升级。
 <br/>
 <br/>
 现在十个边界省份宣布戒严，乌克兰总统波罗申科警告，俄罗斯从陆上入侵的威胁非常严重。
 <br/>
 <br/>
 民众感受到事态危及，尽管戒严令将为生活带来一些不便，仍愿意服从。
 <br/>
 <br/>
 别尔江斯克市居民Alla Yarmak：〝既然我们真的在危险之地，在边境，有可能存在直接的军事冲突，戒严可能会解决一些问题。〞
 <br/>
 <br/>
 也有民众对实施戒严令感到无奈。
 <br/>
 <br/>
 别尔江斯克市居民Sergey Tsiganenko：〝戒严令有好多条款。我不同意其中的说法。但是一个平民如何反对呢？〞
 <br/>
 <br/>
 乌克兰总统波罗申科承诺，除非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯入侵.html" target="_blank">
  俄罗斯入侵
 </a>
 ，公民的权利和自由将不会受到限制。
 <br/>
 <br/>
 别尔江斯克市议会发言人Egorova Iryna :〝乌克兰总统表示，只有在俄罗斯入侵的情况下，才真正实施戒严令。在此之前，公民的权利和自由不会受到限制。〞
 <br/>
 <br/>
 俄罗斯批评波罗申科实施戒严令是为了拖延明年三月大选，但波罗申科将戒严令从60天减半到30天，并承诺不会延迟选举，也不会影响乌克兰民主体制。
 <br/>
 <br/>
 新唐人记者李梅综合报导
</div>
