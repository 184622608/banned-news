### 西班牙歌剧女高音卡芭叶去世 享年85岁
------------------------

<div class="wysiwyg">
 【新唐人2018年10月06日讯】西班牙著名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_歌剧女高音.html" target="_blank">
  歌剧女高音
 </a>
 蒙特赛拉特・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡芭叶.html" target="_blank">
  卡芭叶
 </a>
 ，周五晚去世。1965年，卡芭叶在纽约卡内基音乐厅的出色表演，让她闻名全球，她的美声唱法和对罗西尼、贝利尼等名家作品的深入刻划，受到好评。
 <br/>
 <br/>
 当地时间周五晚，蒙特赛拉特・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡芭叶.html" target="_blank">
  卡芭叶
 </a>
 在巴塞罗那圣十字圣保罗医院去世，享年85岁。
 <br/>
 <br/>
 西班牙国王菲利普六世发推文，称赞卡芭叶是歌剧女王、世界文化传奇，最璀璨的明星。
 <br/>
 <br/>
 西班牙首相佩德罗・桑切斯在推文中说，这消息令人悲伤。
 <br/>
 <br/>
 艺术界同仁也称赞卡芭叶的艺术成就和贡献。
 <br/>
 <br/>
 利塞奥剧院艺术总监 Christina Scheppelmann：〝蒙特赛拉特・卡芭叶是位非常重要的歌唱家。她的歌声遍布全球，为歌剧事业做出贡献，为古典音乐做出贡献。〞
 <br/>
 <br/>
 卡芭叶出生于巴塞罗那，7岁就能唱巴赫的颂赞曲，9岁进入利西奥音乐学院，一生中担纲歌剧角色90多个，演出4千多场。
 <br/>
 <br/>
 1965年，在纽约卡内基音乐厅的表演，让卡芭叶走向事业的成功。演出结束，观众起立鼓掌长达25分钟。
 <br/>
 <br/>
 1987年，卡芭叶与皇后合唱团已故主唱默丘里合作演唱〝巴塞罗那〞，之后这首歌成为1992年巴塞罗那奥运会主题曲之一。
 <br/>
 <br/>
 民众 Terence Westwood：〝她很出色，非常棒。我喜欢她歌唱的方式，她和皇后组合中的那个明星一起合作的...她真的很棒。〞
 <br/>
 <br/>
 民众 Lola Rivelles：〝她深受大家喜爱。现在她走了，真的很难过。〞
 <br/>
 <br/>
 周日将举行卡芭叶追思会，周一举行葬礼。
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
