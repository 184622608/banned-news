### 英国安全部长：使用生化武器的恐攻正在逼近
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】英国安全部长本周一在伦敦发表讲话时发出警告说，英国遭受化学或生物武器
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恐怖攻击.html" target="_blank">
  恐怖攻击
 </a>
 的可能性正在逐渐升高。
 <br/>
 <br/>
 据台湾《中央社》报导，英国安全部长本・华莱士（Ben Wallace）10月9日在伦敦一场安全论坛上发言时说：〝在我发表谈话之际，恐怖分子持续探索在我们的街道杀害我们的新方法，化学和生物武器正逐渐逼近。他们已发展出且致力于更强大的武器。我们必须为可能到来的这一天做好准备。〞
 <br/>
 <br/>
 他并表示，自己所了解到的阴谋显示，恐怖分子和敌人们的野心〝现在只受限于他们的想像力〞。
 <br/>
 <br/>
 有舆论认为，华莱士的这番说辞应该是意有所指，具体涉及的内容可能是多方面多线索，而最有可包括在其中的一个事件，就是今年3月4日发生在英国的前俄罗斯双面间谍遭毒杀的案件。
 <br/>
 <br/>
 今年3月4日，前俄罗斯上校谢尔盖・斯克里帕尔和其女儿尤利娅被发现倒在英格兰威尔特郡索尔兹伯里一处购物中心的长椅上不省人事，经送医诊断发现两人因接触了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神经毒剂.html" target="_blank">
  神经毒剂
 </a>
 而中毒，二人一度病情危殆，生命岌岌可危。经治多方抢救疗后，两人侥幸度过一劫，先后苏醒并康复出院。
 <br/>
 <br/>
 这起毒杀案件发生后，英国政府曾要求联合国安理会召开紧急会议，英国的代表作这次紧急会议上发言时，指责莫斯科在英国〝和平的〞城市中使用了国际社会明令禁止使用的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生化武器.html" target="_blank">
  生化武器
 </a>
 ，违反了国际禁止化学武器公约。
 <br/>
 <br/>
 为此，英国政府宣布驱逐了23名俄罗斯外交官，并指这些外交官都是俄罗斯〝未公开身份的情报官员〞。
 <br/>
 <br/>
 经过长达半年的调查后，英国检察官9月5日向俄罗斯籍人士亚历山大・彼得罗夫和鲁斯兰・波什罗夫（Alexander Petrov和Russian Boshirov）发出逮捕令，指控他们就是试图谋杀斯克里帕尔父女二人的凶手，并指这两名凶犯是俄国军事情报官员。
 <br/>
 <br/>
 英国首相特蕾莎・梅的发言人随即对媒体表示，英国政府将在联合国安理会会议上就此案的指控提出证据。
 <br/>
 <br/>
 当时，英国安全部长本・华莱士曾在接受媒体采访时表示，俄罗斯总统普京对前俄罗斯双重间谍遭遇的神经毒剂袭击负有〝最终〞责任。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
 <br/>
