### 美制裁〝暴政三巨头〞:贫困暴力是共产主义真正结果
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】周四（11月1日），白宫国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 （John Bolton）宣布，美国将对古巴、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 和尼加拉瓜实施新一轮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 ，并且指出，这一暴力政治的〝三架马车〞在川普政府的制裁下最终将会倒台。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 在佛州迈阿密（Miami）的自由塔（Freedom Tower）发表演讲，痛批古巴、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 和尼加拉瓜是暴力政治的〝三架马车〞，并宣布将对该〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴政三巨头.html" target="_blank">
  暴政三巨头
 </a>
 〞（Troika of Tyranny）实施新一轮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 。
 <br/>
 <br/>
 博尔顿特别强调，将再次制裁委内瑞拉的石油工业，并采取行政措施防止美国人与古巴军政府做生意。古巴军政府控制着大部分古巴经济，包括旅游业。
 <br/>
 <br/>
 他承诺，美国将 采取具体措施，多方位制裁这些国家。
 <br/>
 <br/>
 博尔顿表示，他要向这三个暴政国家传递川普总统明确释放的信息——美国不会奖励酷刑犯、杀人犯和施虐者。
 <br/>
 <br/>
 博尔顿指出，这个恐怖三角在三国境内延伸，是造成巨大人类痛苦的原因、巨大地区不稳定的动力、以及西半球共产主义肮脏摇篮的起源。这三个国家反对民主，造成了极端的贫困和暴力，这是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_社会主义.html" target="_blank">
  社会主义
 </a>
 和共产主义的真正结果〞。
 <br/>
 <br/>
 博尔顿在演讲中重申川普总统的立场，宣称委内瑞拉所出现的人道主义危机——目前医疗系统和家庭缺乏几乎所有必需的医疗用品，饥饿问题也日益严重，而这一切苦难之所以正在发生，是因为〝成功实行了社会主义〞。
 <br/>
 <br/>
 博尔顿表示，当今的美国政府不会再安抚这些邻近国家的独裁政权，不会再奖赏那些人权恶棍，〝我们将支持邻国的独立和自由〞。
 <br/>
 <br/>
 博尔顿还送给这三个独裁政府另一绰号——〝社会主义三条走狗〞（the Three Stooges of Socialism）。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
