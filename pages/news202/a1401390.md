### 狮航坠毁前曾出事照飞 人机拉锯26次
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】印尼狮子航空一架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波音客机.html" target="_blank">
  波音客机
 </a>
 上月坠毁，机上189人全部罹难。印尼政府周三（11月28日）发表初步调查报告称，出事客机在起飞前已经存在技术问题，飞机当天照飞。坠毁前，人机拉锯26次。
 <br/>
 <br/>
 印尼国家交通安全委员会航空事故调查部门主任努尔察约．乌托莫28日称，调查人员从飞行数据记录器中获得的数据显示，在狮航
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_JT610.html" target="_blank">
  JT610
 </a>
 航班飞机失事前，系统错误地认为机翼失去了升力，并启动了安全措施让机鼻持续自动往下调整。机长当时多次尝试调整控制系统让机鼻重新抬起，人机拉锯26次，但最终他失去了对飞机的控制。
 <br/>
 <br/>
 波音前工程师莱玛指过程有如〝致命的拉锯战〞。
 <br/>
 <br/>
 乌托莫说，失事客机在此前的航班中已经被发现存在多个技术问题，但狮航在这些问题没有得到完全解决的情况下继续将飞机投入运营。
 <br/>
 <br/>
 据介绍，失事客机在前一天从印尼巴厘岛飞往雅加达的途中也发生过类似问题，所幸，当时机长关闭了飞机的自动调整系统，采用手动操作控制了飞机，坚持飞行并平安抵达。随后，机长向狮航上报了该问题，狮航在进行维修后，第二天让客机继续执飞。
 <br/>
 <br/>
 印尼国家交通安全委员会检查人员还发现失事飞机存在多个问题，包括机长和副机长操作仪器上显示的飞行信息和安全警告不一致。
 <br/>
 <br/>
 报告中，印尼国家交通安全委员会还建议狮航改进安全文化，要求狮航飞行员熟读飞机使用说明并遵循飞行操作流程。
 <br/>
 <br/>
 不过，印尼国家交通安全委员会表示，该报告只是初步调查结果，完整的事故调查报告可能在一年后才能公布。
 <br/>
 <br/>
 当地时间10月29日6时20分，失事客机从雅加达起飞，前往邦加勿里洞省首府槟港，但起飞仅13分钟就与地面失去了联系，随后确认飞机已坠海。机上共有181名乘客和8名机组人员全部遇难。
 <br/>
 <br/>
 搜救人员11月1日成功打捞起失事客机的首个黑匣子——飞行数据记录器，目前仍在努力寻找另外一个黑匣子，也就是驾驶舱话音记录器。
 <br/>
 <br/>
 狮子航空公司是印尼最大的航空公司之一，是印尼私营航空公司，以执飞廉价航班为主。
 <br/>
 <br/>
 (记者李红报导/责任编辑：戴明）
