### 风暴席卷北意大利  15人死 千顷森林被毁
------------------------

<div class="wysiwyg">
 【新唐人2018年11月03日讯】本周，超强
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_风暴.html" target="_blank">
  风暴
 </a>
 席卷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 北部山区，已造成至少15人死亡，数千公顷的森林遭到破坏。
 <br/>
 <br/>
 最高时速达190公里的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_风暴.html" target="_blank">
  风暴
 </a>
 ，席卷了位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 北部的多洛米蒂山脉，包括特伦蒂诺-上阿迪杰大区和威尼托大区。
 <br/>
 <br/>
 许多几百年树龄的松树被连根拔起，遭到破坏的森林面积多达数千公顷。山体滑坡也毁坏了部分公路和村庄。
 <br/>
 <br/>
 威尼托大区区长扎亚表示，该区仍有16万人无电可用。
 <br/>
 <br/>
 一周以来，一波坏天气袭击了意大利全国，著名旅游城市威尼斯也遭洪水淹没。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
