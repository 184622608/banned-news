### 10月10日时事拼盘
------------------------

 【新唐人2018年10月10日讯】下面请看一组新闻简讯
 <br/>
 <br/>
 周三，一辆公共汽车在肯尼亚首都内罗毕和基苏姆市之间的路上翻车，50人死亡。事发时，汽车从一个坡度很大的斜坡下来，失控偏离道路，并在路边的田野上翻滚了好几次，车顶脱落了下来，行李散落一地。
 <br/>
 <br/>
 周二，川普在爱荷华州举办助选集会，支持该州及内布拉斯加州的共和党候选人。他再次批评民主党试图阻止卡瓦诺的最高法院大法官提名。并呼吁选民在11月6日中期选举出来为共和党投票。
 <br/>
 <br/>
 美国总统川普：〝在爱荷华还能提前投票，快去投票吧，用选票阻截（左派惹麻烦）。〞
 <br/>
 <br/>
 德国官员周三表示，当地警方逮捕了一名保加利亚男子，他与该国电视台女记者马里诺娃遭性侵并被杀害有关。马里诺娃的尸体于上周六在一个公园被发现。9月30日她在节目中表示，将对欧盟资金进行腐败调查。有人怀疑，她被杀害是因为当局封口。但该国官员对此否定，案件调查也尚未证实这一点。
 <br/>
 <br/>
 澳大利亚外长和国防部长周三在悉尼会见日本官员。双方计划加强在地区安全方面的合作，以及和美国、印度的联合军演，并逐步实现朝鲜半岛无核化。日本还表示，中共在南海和东海的活动具有攻击性，似乎试图通过强制手段来改变现状。
