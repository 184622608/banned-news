### 【禁闻】美官员暗示 可能将中共逐出WTO
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】
 <b>
  美报告：
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
   中共
  </a>
  未改
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不公平贸易.html" target="_blank">
   不公平贸易
  </a>
  行为
 </b>
 <br/>
 <br/>
 美国贸易代表办公室11月20号发布一份报告，更新了对中国的〝301条款〞调查的信息，指出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在盗窃知识产权和强制技术转让方面没有改变。
 <br/>
 <br/>
 美国贸易代表莱特希泽说，最新调查显示，中共并没有从根本上改变其〝不公平、不合理和扭曲市场的做法〞。这些做法是2018年3月发起〝301条款〞调查的主题。
 <br/>
 <br/>
 美国贸易代表办公室表示，中共没有对最初的301调查报告做出建设性回应，也没有采取任何实质性行动来解决美国关注的问题。这明确表示，中共不打算改变其政策。
 <br/>
 <br/>
 最新报告说，中共正在继续支持以网络攻击的方式窃取美国知识产权的行为，同时也在继续进行技术许可方面的歧视性限制。
 <br/>
 <br/>
 <b>
  河北
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
   法轮功学员
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遭迫害致死.html" target="_blank">
   遭迫害致死
  </a>
  内脏被取走
 </b>
 <br/>
 <br/>
 中共对法轮功持续19年的迫害还在加剧，河北省秦皇岛市最近又有一名法轮功学员被迫害致死。
 <br/>
 <br/>
 据〝法轮大法明慧网〞11月20号报导，这名法轮功学员名叫马桂兰，60多岁。今年7月被秦皇岛警方非法抓捕，关押到市看守所。9月17号，突然传出她被迫害致死的消息，但具体是哪一天被迫害死的还不知道。据说，某天上午6点多钟，马桂兰说自己身体有点不舒服，8点多被抬出监舍，送到秦皇岛公安医院不久，就离世了。
 <br/>
 <br/>
 报导引述内部消息说，马桂兰去世后，河北省来的不明身份的人，把她的肚子剖开，内脏取走，说是化验。再后来的消息就不得而知。
 <br/>
 <br/>
 报导说，在马桂兰去世的那一个月内，秦皇岛看守所还有另外两名被关押的非法轮功学员离奇死亡。
 <br/>
 <br/>
 <b>
  美官员暗示 可能将中共逐出
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
   WTO
  </a>
 </b>
 <br/>
 <br/>
 中共加入世界贸易组织（WTO）后，违反规则，不履行承诺，引发国际社会不满。
 <br/>
 <br/>
 11月21号，英国广播公司BBC中文网报导说，美国总统川普的经济顾问委员会主席凯文．哈塞特（Kevin Hassett）表示，中国作为WTO的一员〝行为不端〞，可能可以研究将它逐出。
 <br/>
 <br/>
 哈塞特说，〝我们从未真正料到一个国家在加入世贸组织后，会像中国那样行事。对世贸组织来说，一个成员国能够如此行为不端，这种情形还比较新。〞
 <br/>
 <br/>
 哈塞特提出，或许可以通过双边谈判，或者改革WTO，甚至将中国从WTO中剔除来解决问题。
 <br/>
 <br/>
 编辑/周玉林
