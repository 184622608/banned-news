### 巴西大选：左右翼候选人将进入决选
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】周日，巴西总统大选举行了第一轮投票。根据官方的最新结果，有〝巴西版川普〞之称的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_右翼.html" target="_blank">
  右翼
 </a>
 社会自由党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_候选人.html" target="_blank">
  候选人
 </a>
 雅伊尔・博尔索纳罗和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_左翼.html" target="_blank">
  左翼
 </a>
 阵营的劳工党候选人费尔南多・阿达将在10月28日进行第二轮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_决选.html" target="_blank">
  决选
 </a>
 。
 <br/>
 <br/>
 巴西总统大选第一轮投票的结果周日(10月7日)出炉。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_右翼.html" target="_blank">
  右翼
 </a>
 的社会自由党总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_候选人.html" target="_blank">
  候选人
 </a>
 、有〝巴西版川普〞之称的现任国会议员雅伊尔・博尔索纳罗(Jair Bolsonaro)的得票率为46.3%，几乎赢得了将近一半的选票。他的主要竞争对手，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_左翼.html" target="_blank">
  左翼
 </a>
 阵营的劳工党总统候选人、前圣保罗市市长费尔南多・阿达(Fernando Haddad)的得票率为29%。由于在第一轮投票中没有人得票过半，这样右翼候选人博尔索纳罗和左翼候选人阿达两人将在10月28日进行第二轮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_决选.html" target="_blank">
  决选
 </a>
 。
 <br/>
 <br/>
 右翼社会自由党在第一轮投票结果出炉后举行了记者会，该党总统候选人博尔索纳罗由于〝健康原因〞没有出席，而是由该党主席代为主持。
 <br/>
 <br/>
 右翼社会自由党主席 GUSTAVO BEBIANNO：〝我们赢得了五千万张选票。只要再多赢一点，就可以取得第一轮选举的完全胜利。〞
 <br/>
 <br/>
 有〝巴西版川普〞之称的右翼候选人博尔索纳罗在总统大选第一轮的胜利显示了巴西民众希望变革的心声。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
