### 一次大战百周年 3000件私人收藏品再现
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月11日讯】〝第
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一次世界大战.html" target="_blank">
  一次世界大战
 </a>
 〞即将届满100周年，生在承平时代的我们，或许很难想像这场延烧全球的战火是什么情景。不过有对夫妻，以一已之力搜集了一次世界大战超过3,000件的物品，要在100周年展出。走入他们家就犹如时光倒流，被带进了一次世界大战的前线。
 <br/>
 <br/>
 一踏进朱迪切（Giudice，意大利文〝法官〞）夫妇尚塔尔（Chantal）和赛吉（Serge）的家中，访客就像时光倒流、被带进了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一次世界大战.html" target="_blank">
  一次世界大战
 </a>
 的前线一样。
 <br/>
 <br/>
 墙上海报以法文写着大大的〝部署令（"GENERAL ORDER OF DEPLOYMENT"）〞这是当时的征兵命令，而架上则是满满各式一次世界大战的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纪念品.html" target="_blank">
  纪念品
 </a>
 。
 <br/>
 <br/>
 来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 西南方城市巴尔马（Balma）的朱迪切夫妇已经搜集一次世界大战纪念品超过20年，这是起源于现年56岁的尚塔尔源自孩提时期的热情。
 <br/>
 <br/>
 <b>
  相关视频：
 </b>
 <center>
 </center>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_收藏.html" target="_blank">
  收藏
 </a>
 家 尚塔尔（Chantal）：〝我那时大约5、6岁，我哥已经在搜集（一次世界大战纪念品）。因此，替代了玩娃娃，我（从小就）一直被制服和军用品所包围。那个热情占领了我，而且从那时起就从来不曾离开我。〞
 <br/>
 <br/>
 这对夫妇自称他们的收藏超过了3,000件，有一些塑像和其它的物件是由在战壕中找到的东西拼凑而成的。
 <br/>
 <br/>
 收藏家 尚塔尔（Chantal）：〝每一物件都代表着一名一次世界大战的步兵，所以我们有大约3,000件，意谓着3,000名步兵在这家里。这些是非常令我感动的纪念品，因为这些代表着他们的苦难。而用这些来自大战的物件，它们成功的创造了精彩的物品，是真实的艺术品。〞
 <br/>
 <br/>
 现年66岁的赛吉朱迪切（Serge Giudice）说，假模特儿穿着当年一次世界大战时军人和护士曾经穿着的衣股，错落分散在房子的周边各处，也是一种纪念那个时代暴力的方式。
 <br/>
 <br/>
 法国也正准备在11月11日纪念第一次世界大战100周年，朱迪切夫妇打算筹组一项展览会，好对大众分享他们的收藏。
 <br/>
 <br/>
 (新唐人亚太电视 Peter Wang 综合报导)
