### 韩媒：朝鲜边谈判 边生产核武关键材料
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】韩国媒体引述〝38 North〞消息报导，朝鲜
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄海北道平山.html" target="_blank">
  黄海北道平山
 </a>
 的铀矿山和设施近期一直处于营运状态。即是，朝鲜因非核化而进行韩朝、美朝谈判的过程中，朝鲜最大的铀矿山及提炼和精矿设施一直处于营运状态。另有消息称，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丹东.html" target="_blank">
  丹东
 </a>
 的一些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜餐厅.html" target="_blank">
  朝鲜餐厅
 </a>
 ，以餐厅兼酒店的形态为幌子，从事技术买卖、煤炭贸易、网路作战等的公司。
 <br/>
 <br/>
 韩国〝朝鲜日报〞报导指出，38 North对从2016年开始到今年5月拍摄的卫星照片进行分析结果显示，相关设施附近堆积的铀矿山废弃物的量明显增加。
 <br/>
 <br/>
 另外，对比2016年10月和今年5月工厂西南侧人工湖的卫星照片发现，污染严重的部分增多，推测是铀提炼过程中产生的副产品。
 <br/>
 <br/>
 38 North暗示，废弃物增加表示平山地区一直在进行铀开采、提炼和精矿（浓缩）活动。
 <br/>
 <br/>
 报导中说，此前美国与韩国对朝鲜未来的非核化协议要求朝鲜不要购买天然铀，不过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高浓缩铀.html" target="_blank">
  高浓缩铀
 </a>
 是朝鲜
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核武.html" target="_blank">
  核武
 </a>
 器计划的关键材料。
 <br/>
 <br/>
 今年8月，国际原子能总署（IAEA）在年度报告中称，朝鲜在过去的一年间，曾启动过宁边5㎿黑铅减速炉（用于核武器的钸生产设施）和再处理工厂设备。
 <br/>
 <br/>
 38 North建议称：〝在今后的非核化谈判中，有必要增加使朝鲜无法获得天然铀的条款，防止朝鲜生产高浓缩铀，应用于核武器开发。〞
 <br/>
 <br/>

 <br/>
 美国自由亚洲电台（RFA）引述美国华盛顿的安全研究机构C4ADS（Center for Advanced Defense Studies）透露：〝中国境内的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜餐厅.html" target="_blank">
  朝鲜餐厅
 </a>
 或与朝鲜政权的核扩散活动存在关联〞。
 <br/>
 <br/>
 本月1日，该研究所的分析官杰森‧奥特本出席在美国华盛顿举行的朝鲜海外派遣劳动者相关人权活动时称：〝中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丹东.html" target="_blank">
  丹东
 </a>
 的柳京饭店等中国境内的一些朝鲜餐厅的地址与酒店的位址相同，还存在许多其他企业登记在同一个地址的情况，相关企业实为以餐厅兼酒店的形态为幌子，在综合经营的结构下，从事技术买卖、煤炭贸易、网路作战等的公司。〞
