### 土耳其释放牧师布朗森 美土关系仍复杂
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】观众朋友大家好，欢迎收看新唐人全球新闻。周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 法院决定释放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国牧师.html" target="_blank">
  美国牧师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布朗森.html" target="_blank">
  布朗森
 </a>
 ，当地晚间，布朗森已乘机离开土耳其返回美国。
 <br/>
 <br/>
 美国福音派牧师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布朗森.html" target="_blank">
  布朗森
 </a>
 及妻子当地时间周五晚从伊兹密尔（阿德南．曼德列斯）机场离开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 返回美国，但具体线路未知。预计副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 将携夫人迎接布朗森一家。
 <br/>
 <br/>
 土耳其亚利加市法院在周五的庭审中判决布朗森〝参与恐怖行动〞罪成，他被认为与2016年针对土耳其总统埃尔多安的政变有关。不过，法庭决定立即释放布朗森，原因是布朗森已被埃尔多安政府扣押两年多。
 <br/>
 <br/>
 总统川普周五连发多个推文，说在解决布朗森牧师的事上下了很多功夫，并为他祈祷。他说：〝牧师布朗森被释放了。会很快回家！〞
 <br/>
 <br/>
 开庭前，副总统彭斯和国务卿蓬佩奥都再次施压土耳其，促其放人。
 <br/>
 <br/>
 副总统 彭斯（2018.10.11）：〝总统态度明确、美国政府态度非常清晰，我们将坚持立场直到布朗森牧师自由并返回美国。〞
 <br/>
 <br/>
 美国国务卿 蓬佩奥（2018.10.10）：〝这是土耳其可以做的一件正确的好事、是件人道主义的事。我希望不久布朗森牧师和他的妻子可以回到美国。〞
 <br/>
 <br/>
 此前美国多次要求土耳其放人无果，8月川普加倍征收对土耳其的钢铝关税，让〝里拉〞再创新低；自年初以来累积重贬40%。通胀严重、外资出走，让依赖外资的土耳其经济持续恶化。本周，由于外界普遍对布朗森获释持乐观态度，土耳其〝里拉〞回升2.78%。
 <br/>
 <br/>
 牧师布朗森成为美土外交危机的焦点，但两国间的争端不止于此，让彼此从北约盟友变为亦敌亦友、甚至更糟。埃尔多安不断强化反美言论，他指责美国拒绝引渡其指控策划了2016军事政变的伊斯兰教士葛兰（Fethullah Gulen）；对美国支持叙利亚库尔德民兵发出军事打击威胁；抵制美国对伊朗制裁、继续购买伊朗油气产品；同时向俄罗斯购买武器。
 <br/>
 <br/>
 未来美国是否会解除对土耳其制裁还有待观察。
 <br/>
 <br/>
 新唐人记者李兰综合报导
