### 英国欧盟仍谈不拢 脱欧过渡期或延长
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 峰会于周三和周四在比利时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布鲁塞尔.html" target="_blank">
  布鲁塞尔
 </a>
 召开，焦点之一是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国.html" target="_blank">
  英国
 </a>
 脱欧问题，但没有取得实质性进展。欧盟认为，12月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟峰会.html" target="_blank">
  欧盟峰会
 </a>
 或许是最后的机会。
 <br/>
 <br/>
 2019年3月29日是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国.html" target="_blank">
  英国
 </a>
 正式脱欧的日子，但几百个合约条款还不能开始谈判，时间明显不够了。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 脱欧事务首席谈判代表 巴尼耶：〝所以我们需要时间，我们需要更多的时间。〞
 <br/>
 <br/>
 欧盟认为，脱欧协议一再拖延的原因是，英国首相特蕾莎・梅无法摆平国内的反对声音。
 <br/>
 <br/>
 〝现在仍然存在北爱尔兰〝保障条款〞的问题……我相信协议是可以达成的，现在就是达成的时候了。〞
 <br/>
 <br/>
 梅首相在国内找到〝政治平衡点〞并不容易，因为执政党在下议院中的重要盟友〝北爱尔兰民主统一党〞表示，英国脱欧后，北爱尔兰还必须保留在欧盟关税同盟中，以避免北爱尔兰和爱尔兰共和国之间重新建立边境检查，因为那样会分裂英国。
 <br/>
 <br/>
 欧盟的态度是：要么北爱尔兰留在欧盟关税同盟中，要么英国整个留下。
 <br/>
 <br/>
 英国的强硬脱欧派则针锋相对，认为英国连同北爱尔兰地区，必须尽快离开欧盟关税同盟。
 <br/>
 <br/>
 欧盟内部很多小国家在观望，看德国法国等大国是否愿意冒险与英国较量，来保护小国爱尔兰的利益。可以说，这也是对欧盟成员国价值的考验。
 <br/>
 <br/>
 目前，欧盟提出把英国脱欧过渡时间延长到2021年底，给英国更多时间理顺内部问题。
 <br/>
 <br/>
 新唐人记者王羽音综合报道
</div>
