### 疑乘客携瓦斯桶上车 津国巴士爆炸酿42死27伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_津巴布韦.html" target="_blank">
  津巴布韦
 </a>
 15日晚间传出巴士爆炸意外，一辆开往南非的巴士，疑乘客携带
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瓦斯桶.html" target="_blank">
  瓦斯桶
 </a>
 上车，在首都哈拉雷（Harare）南方550公里处爆炸，造成车上42名乘客不幸丧命，另有27人受到烧伤。
 <br/>
 <br/>
 从现场照片显示，整辆公车烧到只剩支架。警方发言人查拉巴（Charity Charamba）表示，〝到目前为止，我们已经知道超过42人死亡。〞另证实有 27人受到严重烧伤。
 <br/>
 <br/>
 当局推断有乘客携带
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瓦斯桶.html" target="_blank">
  瓦斯桶
 </a>
 上车酿祸。
 <br/>
 <br/>
 <b>
  相关视频：
 </b>
 <center>
 </center>
 <br/>
 上星期，在连接首都哈拉雷(Harare)及东部城镇鲁萨配(Rusape)的公路上，发生了两辆巴士对撞的意外，造成47人死亡，70人受伤，为该国史上第2伤亡惨重公车车祸。
 <br/>
 <br/>
 据了解，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_津巴布韦.html" target="_blank">
  津巴布韦
 </a>
 (Zimbabwe)因多年来被忽视并缺乏经费，造成当地道路布满坑洞，驾驶又未遵守规则。民众盼事件发生后，当局应正视交通问题和道路安全，避免再传憾事。
