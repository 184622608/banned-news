### 肯尼亚〝一带一路〞爆贪腐案 3中国人被起诉
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】中共近年来推行的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞战略计划，给沿线国家带来的债务过快增长以及腐败加剧等恶劣后果越来越明显。日前，有3名中国籍公民因涉嫌在肯尼亚的〝一带一路〞项目中偷钱而被肯尼亚检方起诉，为中共的输出腐败再添一桩案例。
 <br/>
 <br/>
 综合外媒报导，肯尼亚检察部门日前起诉的三名中国籍公民，分别是中国路桥工程有限责任公司的两名李姓安全经理Li Gen和Li Xiaowu，以及1名姓孙翻译人员Sun Xin。他们被指控涉嫌在肯尼亚的火车售票系统动手脚偷钱，以及向调查人员行贿。
 <br/>
 <br/>
 检方指控上述3名中国人与另外4名肯尼亚人勾结，透过控制订票系统，为一些买了门票的乘客私自进行退票，并把退票所得资金转移到私人帐户中。
 <br/>
 <br/>
 据指，这伙人采用这种手法，平均每天偷走的金额高达1万美元，相当于去年每日收入的三分之一，但他们具体的作案时间持续了多久暂时还不确定。
 <br/>
 <br/>
 此外，3名中国人还涉嫌在东窗事发后用5千美元贿赂调查人员，试图影响正在进行的刑事调查。此案将与本周五（30日）举行保释聆讯，目前3人均被扣押，但 3人都否认控罪。
 <br/>
 <br/>
 肯尼亚检察部门在社交媒体公布相关消息时，还公开了这3名中国籍嫌疑人的照片。
 <br/>
 <br/>
 外界注意到，这起案件涉及的铁路项目，恰恰是中国在肯尼亚推行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 重点项目。肯尼亚政府曾标榜这个铁路项目是该国独立以来的最大项目。这项工程的成本耗资高达32亿美元，而这个项目自从建成以来一直面临着入不敷出的困境，其运行第一年就亏损了1亿美元。
 <br/>
 <br/>
 海外舆论界认为， 〝一带一路〞实质上是中共在全球实行〝新殖民主义〞的一个〝吸血〞工程。而事实证明，凡是受到中共〝一带一路〞关爱的国家，几乎政治经济情况都不好，不是政治腐败严重，就是债台高筑，甚至有的国家因此而被迫让自己的经济、战略资源和港口被中共控制。
 <br/>
 <br/>
 有许多学者专家都分析称，中共在中亚、东南亚、非洲等地区的〝一带一路〞项目中，不仅在输出产能，也在输出腐败。
 <br/>
 <br/>
 其中一个比较典型的例子是〝一带一路〞在马来西亚的基建投资，被指非法转入了该国前总理纳吉布成立的〝一马基金〞帐户。
 <br/>
 <br/>
 北京时政评论人章立凡曾经在接受西方媒体采访时表示，〝一带一路〞项目是携带〝特洛伊木马病毒〞的项目，在输出中国式腐败的同时，有可能让更多目标国在巨额债务中受到中共越来越多的箝制。不过，现在一些国家已经开始警醒，这个项目最终的命运将会是一个巨大的〝烂尾工程〞。
 <br/>
 <br/>
 （记者竺颖综合报导）
