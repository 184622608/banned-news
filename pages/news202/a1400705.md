### 中美洲移民压境美墨边境 拉美各国也担忧
------------------------

<div class="wysiwyg">
 【新唐人2018年11月24日讯】不断涌向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲移民.html" target="_blank">
  中美洲移民
 </a>
 ，已经让美国国土安全部部长称作〝边境危机〞；他们也给沿途的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉美.html" target="_blank">
  拉美
 </a>
 各国制造困局。十几个拉美国家正在商讨对策并向美国等求助。同时，这些已经聚集在美国边境的移民表示，他们不会离开。
 <br/>
 <br/>
 过去一周，有6千至1万中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 的墨西哥城市蒂华纳，让边境安全威胁不断上升。美国国土安全部长尼尔森称其为〝边境安全危机〞；总统川普表示在考虑关闭边境。
 <br/>
 <br/>
 这些来自中美的移民也给沿途国家带去麻烦，其中委内瑞拉难民最为突出。周四和周五，十多个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉美.html" target="_blank">
  拉美
 </a>
 国家在厄瓜多尔的基多（Quito）会晤，商讨如何解决委内瑞拉难民问题。厄瓜多尔官员表示，安置这些难民需要至少5.5亿美元。30日该国计画与美国、加拿大、欧盟、日本等国就解决该问题进行协商。
 <br/>
 <br/>
 厄瓜多尔人力资源部副部长 Santiago Chavez：〝委内瑞拉政府必须提供更多帮助，为其公民承担更多责任。〞
 <br/>
 <br/>
 厄瓜多尔估计已有60万委内瑞拉人从哥伦比亚进入该国。哥伦比亚已经收容了100万委国人，而且每天仍有至少3千人涌入。联合国数据显示，2015年以来超过300万委国人因为国内的恶性通货膨胀、严重的粮食短缺等原因逃离家园。
 <br/>
 <br/>
 这些人中的一部分在大篷车上。
 <br/>
 <br/>
 川普总统周六转贴前几日推文，强调美国加强了边境防护。不过，有聚集在墨西哥边境城市的移民表示，即使川普派遣军队驻守边境，他们也不会离开。
 <br/>
 <br/>
 大篷车成员 Carlos Varela：〝我们不想离开，你不能强迫我们。〞
 <br/>
 <br/>
 美国副总统彭斯此前向媒体披露，他获得消息，移民大篷车获得了左翼团体资助、其中包括委内瑞拉政府的资助。这些团体的目标是〝废除边境〞。
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
