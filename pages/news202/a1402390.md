### 【禁闻】欧盟官员：华为对产业和安全构成风险
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】
 <b>
  11月份 486名
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
   法轮功学员
  </a>
  被非法抓捕
 </b>
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对法轮功的迫害依然严酷。据〝法轮大法明慧网〞报导，在刚刚过去的11月份，中共又非法抓捕了486名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 ，其中年龄最大的90岁。
 <br/>
 <br/>
 非法抓捕遍及24个省、自治区、直辖市的73个城市，其中最迫害严重的省份是黑龙江，非法抓捕186人，非法抓捕人数最多的城市则是哈尔滨，共99人。
 <br/>
 <br/>
 报导说，除非法抓捕外，中共还在11月份对数百名法轮功学员进行了骚扰、抄家、非法判刑、强制洗脑等迫害。
 <br/>
 <br/>
 另外，这一个月内，中共警察勒索法轮功学员的现金达20多万元人民币。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟官员.html" target="_blank">
   欧盟官员
  </a>
  ：
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
   华为
  </a>
  对产业和安全构成风险
 </b>
 <br/>
 <br/>
 因首席财务官在加拿大被抓，中国科技公司华为这两天成为全球舆论的焦点。
 <br/>
 <br/>
 12月7号，欧盟技术专员也表示，欧盟应该担心华为及其它中国科技公司，因为它们对欧盟的产业和安全构成风险。
 <br/>
 <br/>
 据〝大纪元新闻网〞报导，在布鲁塞尔当天举行的新闻发布会上，欧盟委员会副主席安西普（Andrus Ansip）表示，他担心中国科技公司被要求与中共情报机构合作，利用〝强制后门〞访问加密数据。
 <br/>
 <br/>
 <b>
  150多名学者联署 谴责中共骚扰
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新西兰教授.html" target="_blank">
   新西兰教授
  </a>
 </b>
 <br/>
 <br/>
 150多名来自世界各国的中国问题专家，12月6号联名致信新西兰政府，要求它为坎特伯雷大学教授布莱迪（Ann-Marie Brady）提供人身保护。布莱迪也是一名研究中国问题的专家，最近一年持续受到中共骚扰。
 <br/>
 <br/>
 香港《南华早报》报导说，布莱迪教授去年9月发表一篇题为《魔法武器》(Magic Weapons)的文章，揭露中共如何在国外扩展政治影响力。从那时起，她就一直在遭受来自中共当局的骚扰和恐吓。她在基督城的家和办公室今年2月遭入室盗窃，她的汽车也在上个月被人故意损毁。
 <br/>
 <br/>
 公开信说：〝我们这些研究中国问题的学者们，对近来我们的同仁安-玛丽．布莱迪教授遭受的一系列恐吓行动感到震惊和警觉。这些恐吓行动显然是为了对她研究当代中国的写作进行报复，因为布莱迪教授对中共政府在海外进行宣传和渗透的机制进行了调查。〞
 <br/>
 <br/>
 编辑/周玉林
