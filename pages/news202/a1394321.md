### 印尼强震海啸已夺1558人命 近20万人需人道援助
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 苏拉威西岛上周发生毁灭性地震和海啸后，已知至少逾1558人遇难，仍有超过1000人下落不明。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 周五（5日）表示，正在筹集5050万美元〝即刻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_救援.html" target="_blank">
  救援
 </a>
 〞金，以帮助印尼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灾民.html" target="_blank">
  灾民
 </a>
 。
 <br/>
 <br/>
 中央社报导，苏拉威西岛（Sulawesi）9月28日遭遇威力强大的地震和海啸侵袭，帕卢（Palu，巴路）许多房屋因此倒塌，海啸掀翻车辆，死亡人数目前已升至1558人，另外仍有超过1000人失踪。
 <br/>
 <br/>
 人们愈来愈担心，有大量民众被埋在巴拉罗阿（Balaroa）一处大型政府住宅社区。强震使巴拉罗阿的地面变的残破不堪。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 国家搜救总署发言人拉提夫（Yusuf Latif）说：〝我们估计有超过1000栋房屋遭到掩埋，所以可能有1000多人下落不明。但我们无法确定，因为部分民众有可能已经自行逃出来。〞
 <br/>
 <br/>
 此前，官员预估约100人下落不明。印尼当局先前暂定周五是搜寻压在房屋下幸存者的最后期限，因为将幸存者从瓦砾堆中救出的机会已经微乎其微。
 <br/>
 <br/>
 因余震不断，道路损坏严重，经过数天延误后，国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_救援.html" target="_blank">
  救援
 </a>
 团体终于陆续抵达灾区。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 （UN）表示，将近20万人需要人道援助。UN正在筹集5050万美元〝即刻救援〞金，以帮助印尼强震及海啸
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灾民.html" target="_blank">
  灾民
 </a>
 。
 <br/>
 <br/>
 根据声明，联合国计划未来3个月对19万1000人提供援助。
 <br/>
 <br/>
 联合国估计，这次强震海啸导致印尼6万5000户民宅出现灾损，包括1万户民宅〝受海啸冲击全毁〞，以及1万5000户民宅因强震严重受损。
