### 阿富汗国会大选日暴力频传 近170人死伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 周六（20日）举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国会大选.html" target="_blank">
  国会大选
 </a>
 ，混乱状况不断。当天，一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀炸弹客.html" target="_blank">
  自杀炸弹客
 </a>
 在首都喀布尔一处投票所引爆炸弹，造成至少15人死亡，超过25人受伤。全国攻击事件死伤人数累计已近170人。
 <br/>
 <br/>
 中央社引述路透社报导，这起自杀炸弹攻击发生在喀布尔（Kabul）北部。一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 高级维安官员表示，炸弹客试图进入投票所时引爆，造成10名民众和5名警察遇难，超过25人受伤。这似乎是当天多起攻击事件中最严重的一起。
 <br/>
 <br/>
 阿富汗卫生部代理发言人吉尔（Mohibullah Zeer）表示，周六大多数死伤落在喀布尔，当地发生数起爆炸案，至少已酿成4死78伤。
 <br/>
 <br/>
 意大利非政府组织〝紧急〞（Emergency）表示，已有37人被送往他们在喀布尔的外科医院，其中一名儿童到院前已死亡。
 <br/>
 <br/>
 阿富汗一名警方发言人表示，周六投票过程中，北部昆都兹市（Kunduz）遭到3枚火箭攻击，并未传出伤亡。
 <br/>
 <br/>
 阿富汗西部赫拉特省（Herat）一个投票所附近也被一枚火箭击中，警方发言人说，有一名儿童受伤，另在其他地方也发生数起爆炸案。
 <br/>
 <br/>
 阿富汗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国会大选.html" target="_blank">
  国会大选
 </a>
 已延宕超过3年，这次有接近900万人登记投票。
 <br/>
 <br/>
 战乱蹂躏的阿富汗尽管存在军事攻击威胁，周六，全国各地仍有选民在投票所外苦候数小时，等待投票所开门。
 <br/>
 <br/>
 但由于准备工作一团乱，导致选民登记名册遗失或不完整，首度采用的选民生物辨识装置也状况频传，投票所延后开放的情况相当严重。许多投票所因技术问题和选务人员不足，根本没有开放。
 <br/>
 <br/>
 阿富汗独立选举委员会（IEC）表示，由于雇来负责选务的教师未能准时到场，大部分投票所都延迟开放。IEC已承诺将投票时间延长4小时。
 <br/>
 <br/>
 IEC还告诉记者，当地时间下午1时（格林威治时间8时30分）之前未能开放的投票所，会在周日再度开放投票。
 <br/>
 <br/>
 42岁的穆斯塔法（Mustafa）在喀布尔一所清真寺外受访时表示：〝我们等了一个小时，他们都还没开放投票。队伍越来越长，他们必须赶快登记我们的选票，我们害怕可能被炸弹或爆炸击中。〞
 <br/>
 <br/>
 在投票所外等了4个小时的佛路格（Tabish Forugh）发推文称，过去的选举〝连只是稍微类似的…混乱情况〞都没见过。
 <br/>
 <br/>
 选前数个月内，已有数以百计的人遭到杀害或伤害。南部坎达哈省（Kandahar）有权有势的警察首长18日遭暗杀丧命，令人对安全部队保护投票所的能力更缺乏信心。攻击事件后，坎达哈省的投票已宣布延后一周。
 <br/>
 <br/>
 塔利班（Taliban）在选前几天屡次发出警告，呼吁候选人退选，并警告阿富汗选民，〝为保护自身生命安全〞，应抵制投票。超过2500名角逐阿富汗众议院席位的候选人中，已有至少10人遭到杀害。
 <br/>
 <br/>
 目前爆炸案无人出面宣称犯案，但民兵组织塔利班（Taliban）稍早称，已在饱受战火蹂躏的阿富汗〝假选举〞中执行超过300起攻击。
