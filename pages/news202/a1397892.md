### 日机师酒驾闯关 酒精浓度超标10倍在英被捕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本航空.html" target="_blank">
  日本航空
 </a>
 一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副机长.html" target="_blank">
  副机长
 </a>
 近日在英国执勤前喝了2瓶葡萄酒和逾1.8公升啤酒，企图闯关，但在飞机起飞前，英国警方对其进行验血后发现，其血液中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒精浓度.html" target="_blank">
  酒精浓度
 </a>
 是合法上限的近10倍，当场被警方逮捕。
 <br/>
 <br/>
 该航班原定英国10月29日从伦敦出发飞往羽田。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本航空.html" target="_blank">
  日本航空
 </a>
 （Japan Airlines）高层在日本告诉记者，42岁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副机长.html" target="_blank">
  副机长
 </a>
 实川克敏执勤前通过了内部呼气测试，但在搭乘巴士前往伦敦希斯罗机场（Heathrow Airport）停机坪时，一身酒气引起巴士司机的怀疑并报告相关单位。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦警方.html" target="_blank">
  伦敦警方
 </a>
 发言人表示，在飞机预定起飞前50分钟对实川克敏进行了验血，验出每100毫升血中的酒精含量为189毫克，是机师酒精含量的容许上限20毫克的近10倍。
 <br/>
 <br/>
 《朝日新闻》报导，实川克敏当时被捕后获释，10月31日再次被捕，11月1日出庭时，他坦承犯案，预计将在本月29日接受宣判。依照英国法律，最重可处2年以下有期徒刑或罚金，也可能两者兼行。
 <br/>
 <br/>
 报导称，实川克敏在飞行前20小时曾在饭店的酒吧与自己的房间内，各喝了一瓶红酒与玫瑰红酒、3瓶330毫升的瓶装啤酒以及2罐440毫升的罐装啤酒。在值勤前的2小时，实川克敏与另外两名机长一起接受了酒测，但是不知何故都没有验出酒精值。
 <br/>
 <br/>
 日航主管告诉记者：〝我们确信，那（内部呼气测试）做得并不准确。〞
 <br/>
 <br/>
 日航高级董事总经理Toshinori Susumu在1日的新闻发布会上为此事件道歉，他说，〝发现了严重违反法律法规的行为，我向所有人道歉。〞
 <br/>
 <br/>
 根据日本法令，正副机长在值勤前8小时禁止饮酒，但对酒测则没有标准值。而日本航空内部规定值勤前12小时禁止喝酒，或是在酒测中不得超过每公升0.1毫克，违反其中任何一项都不能执勤。
 <br/>
 <br/>
 日本航空从去年起已经将国内机场的酒测器更换成高性能的新型款式，想要从中造假几乎不可能。但在国外的分部还是使用旧版酒测器。
 <br/>
 <br/>
 事发后，日航也决定将在本月中全面更换海外据点的酒测器，同时将修正公司内部的规定，要求机师值勤前24小时禁酒。
 <br/>
 <br/>
 该事件发生的前一天，日本另一家航空公司也因一名机师喝醉挂病号、导致多个航班延误而致歉。
