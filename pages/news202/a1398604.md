### 华北阴霾影响首尔PM2.5飙升 韩环团欲告中共
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】近日，受中国华北地区阴霾扩散的影响，韩国首尔地区的PM2.5浓度急剧飙升，一度高达93μg/m3。韩国忧虑中共或因贸易战而放宽对环境污染的管控，韩国环保团体则打算针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空气污染.html" target="_blank">
  空气污染
 </a>
 严重的问题对韩国政府和中共提起控告。
 <br/>
 <br/>
 据台湾《自由时报》报导，由于受到华北地区逐渐扩散的阴霾影响，首尔地区的PM2.5浓度从本月3日起持续飙升。至6日，首尔地区的PM2.5浓度已高达93μg/m3。对此，韩国政府有关机构发布了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空气污染.html" target="_blank">
  空气污染
 </a>
 警报。
 <br/>
 <br/>
 当天，有256位韩国民众进入青瓦台官网请愿专区请愿，希望韩国政府能够透过外交手段敦促中共共同解决空气污染的问题。
 <br/>
 <br/>
 报导称，韩国政府现在很担忧中共会因为贸易战而放宽空污管制政策，全力加速生产以弥补经济损失，从而导致空气污染的问题更加严重。
 <br/>
 <br/>
 韩国气象中心已预测，2018年入冬后发生的霾害恐将对韩国产生更可怕的后果。
 <br/>
 <br/>
 根据《自由亚洲电台》报导，中韩2国从6月起共创〝环境合作中心〞，希望藉由探讨、交流双方的气象资料，来抑制霾害对韩国的影响，韩方环境财团律师池贤英认为，中国坦言其国内对空污影响甚钜，却迟迟不与韩国进行更密切的环境改善合作；韩方希望中国提供全国大气资料，但4个月来，中方都说还在整理数据。
 <br/>
 <br/>
 报导还表示，韩国环保团体过去就曾经以集体提告的方式，向中共提出过民事损害赔偿诉讼，近期更打算再次对中、韩两国政府提起诉讼。
 <br/>
 <br/>
 据了解，2017年4月到5月期间，韩国环境财团代表崔律和社会各界名流90多人就空气污染起诉韩国政府和中共，索赔每人300万韩元，合计共2.73亿韩元，相当于167万元人民币。这是韩国人首次因空气污染问题集体起诉中共。
 <br/>
 <br/>
 当时环保团体提出的诉状表示，中国作为国际社会的成员，有义务将污染物质管理在允许的范围之内。但是很显然，中国（中共）并没有确切地进行管理，以至于污染物质越过国境，导致韩国的空气污染。
 <br/>
 <br/>
 诉状同时也指责韩国政府未能采取有效措施改善空气污染的问题，并据此起诉中、韩两国政府在治理空气污染方面〝怠慢〞和〝懒政〞，并向两国政府索赔。
 <br/>
 <br/>
 当时，韩国环境部发布的调研结果显示，韩国境内的空气污染约有76%来自〝境外〞，包括中国、蒙古和朝鲜等国。
 <br/>
 <br/>
 有专家指出，要打赢以其他国家政府为被告的诉讼，首先需要对相关污染的状况和原因进行量化统计，以世界卫生组织的标准进行数据化，对于造成污染的工业区等进行共同调研，根据统计数据起诉才有胜诉的可能性。
 <br/>
 <br/>
 另有舆论认为，胜诉的可能性虽然不大，但是诉讼的行动应该会起到敦促中方改善空气污染的作用。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
