### 前俄特工遭毒杀案 第二名俄嫌遭曝光
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】毒害俄罗斯前特工斯克里帕尔父女的第二名嫌疑人身份被接露，是俄国军事情报局的军医。不过，嫌犯和俄罗斯当局否认涉案。
 <br/>
 <br/>
 调查团体〝冒险者〞周一公布报告，确认毒害俄罗斯前特工斯克里帕尔父女的第二名嫌疑人，是为俄罗斯军事情报局工作的军医
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_米什金.html" target="_blank">
  米什金
 </a>
 ，现年39岁。
 <br/>
 <br/>
 米什金和另一名男子契皮加，因涉嫌今年三月毒害俄罗斯前特工斯克里帕尔父女，谋杀未遂，被英国警方通缉。
 <br/>
 <br/>
 契皮加曾获得俄罗斯最高军事勋章。
 <br/>
 <br/>
 〝冒险者〞的报告说，米什金完成学业后，被俄罗斯情报局招募，他在莫斯科的住址用的是俄罗斯军事情报局总部的地址。
 <br/>
 <br/>
 两名嫌犯上个月出现在俄罗斯国家电视台，宣称无辜。俄罗斯也否认参与中毒事件。
 <br/>
 <br/>
 这场中毒事件引发国际社会关注，美国、英国以及欧洲20多个国家下令驱逐俄罗斯外交官，冷战以来前所未见。
 <br/>
 <br/>
 前俄罗斯特工斯克里帕尔父女，今年3月遭俄国军用毒剂诺维乔克攻击，经救治后出院。英国调查后认为，是俄罗斯在背后操纵。
 <br/>
 <br/>
 6月底，一对住在附近的中年男女触摸到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诺维乔克毒剂.html" target="_blank">
  诺维乔克毒剂
 </a>
 ，女子不治身亡。
 <br/>
 <br/>
 英国认为，两名嫌犯使用假名进入英国。面部识别专家匹配相隔15年的两张照片，使用模拟年龄增长技术，还原了嫌犯的真实身份。
 <br/>
 <br/>
 新唐人记者李梅、明玉综合报导
 <br/>
