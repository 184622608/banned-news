### 默克尔卸任党主席 基民盟选接班人
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国总理.html" target="_blank">
  德国总理
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默克尔.html" target="_blank">
  默克尔
 </a>
 正式卸任基民盟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党主席.html" target="_blank">
  党主席
 </a>
 。虽然默克尔到2021年才卸任总理，但德国政坛新篇章已经开启。
 <br/>
 <br/>
 18年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党主席.html" target="_blank">
  党主席
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默克尔.html" target="_blank">
  默克尔
 </a>
 已经成了德国基民盟的代名词。13年的总理，默克尔在欧盟经历重重危机之时，主导欧洲政治，建立各国共识。
 <br/>
 <br/>
 周五，一代政治强人默克尔迈出淡出政治舞台的第一步。德国进入后默克尔时代。
 <br/>
 <br/>
 在汉堡的基民盟大会上，她的简短卸任讲演感动人心。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国总理.html" target="_blank">
  德国总理
 </a>
 默克尔：〝在这一刻，我内心充满的唯一感觉，一个压倒一切的感觉是：感谢。非常高兴担任党主席，这是一种荣幸。谢谢。〞
 <br/>
 <br/>
 默克尔走下台时，与会的1千多名代表全体起立鼓掌近10分钟。
 <br/>
 <br/>
 还有人打出了〝感谢老板领导基民盟党18年〞的标语。
 <br/>
 <br/>
 多名党主席候选人中，呼声最高的是默克尔看好的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_接班人.html" target="_blank">
  接班人
 </a>
 卡伦鲍尔和默克尔的昔日对手麦兹。
 <br/>
 <br/>
 卡伦鲍尔被外界视为〝小默克尔〞，很可能延续默克尔的难民政策。分析人士认为，她仍然会继续默克尔的社会政策偏左、经济政策偏右的路线。
 <br/>
 <br/>
 麦兹10月底重返政坛，他对默克尔向非法移民和难民打开大门的政策不满，有意把基民盟带回更保守的立场，并承诺当选后会强化美德关系。
 <br/>
 <br/>
 两人都主张加强欧盟团结。
 <br/>
 <br/>
 虽然默克尔到2021年总理任期才满，但因为不再担任党主席，因此不能像以前一样主掌德国政坛。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
