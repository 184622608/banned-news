### 落实军事协议 韩朝各拆10哨所建筑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】根据〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_9．19军事协议.html" target="_blank">
  9．19军事协议
 </a>
 〞，韩朝各自撤除在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非军事区.html" target="_blank">
  非军事区
 </a>
 （DMZ）内的10个试点哨所，双方之前已撤出哨所的兵力与武器，周一（12日）开始拆除工作，12月将互相检验对方的拆除情况。
 <br/>
 <br/>
 韩联社报导，陆军参谋长（相当于陆军司令）金勇佑周一主持落实〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_9．19军事协议.html" target="_blank">
  9．19军事协议
 </a>
 〞的现场讨论会，并前往江原道铁原
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非军事区.html" target="_blank">
  非军事区
 </a>
 视察哨所的拆除情况。
 <br/>
 <br/>
 根据〝9．19军事协议〞，韩朝军方从11月1日起试点撤除部分哨所，这是双方为消除军事紧张和防止武装冲突而采取的实质性且具重要象征意义的措施。之前双方已将试点哨所的全部兵力和武器撤出，周一起开始拆除哨所。
 <br/>
 <br/>
 韩国陆军人士表示，军方雇用专业民营拆迁公司，使用挖掘机（怪手）等设备，以拆除韩国方防护墙、内部设施、左右两侧防护墙和朝鲜方防护墙这4个程序进行铲除作业。之后，韩国将在当地开展山林复原工作。
 <br/>
 <br/>
 报导说，韩国政府官员透露，韩国军方原本计划用爆破方式拆除，但考虑环境和安全等因素后决定用挖土机。之后，韩方将在相应地区开展山林复原工作。而朝鲜军方则使用爆破方式拆除试点哨所及其附属设施。
 <br/>
 <br/>
 韩朝之前计划在11月底前各撤除11个试点哨所，但后来商定各自保存一个哨所。韩国选的是1953年签订〝停战协定〞后最早建成的、东海岸距离朝鲜哨所最近的一处，朝鲜则选择金正恩曾访问的、位于中间地带的一处哨所。
 <br/>
 <br/>
 韩国总统文在寅与朝鲜领导人金正恩，于4月27日与5月26日两度在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_板门店.html" target="_blank">
  板门店
 </a>
 举行高峰会。接着，两人又于9月18日至20日在平壤举行第三次高峰会，19日两人发布〝平壤共同宣言〞。
 <br/>
 <br/>
 当天，韩国国防部长官宋永武、朝鲜人民武力相（相当于国防部部长）努光铁还签署了《9．19军事协议》，作为《9月平壤共同宣言》的附属协议。
 <br/>
 <br/>
 韩朝11月1日凌晨0时起依据《9‧19军事协议书》，全面中止一切陆地、海上、空中的敌对行为，并调整军事演习计划和方式。同时，美国和韩国在华盛顿举行第50次美韩安全谘商会议（SCM），达成共识并签署〝联合防卫指南〞，韩国将阶段性收回〝战时指挥权〞，不过美军仍然会继续驻守
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜半岛.html" target="_blank">
  朝鲜半岛
 </a>
 。
 <br/>
 <br/>
 不过，据日本《共同社》报导，日方消息人士称，美国政府已告知日本政府，若朝鲜无核化没有进展，最快将于明年春天重启美韩大规模联合军演。演习可能是〝关键决心〞，或是〝鹞鹰〞。消息人士称，〝很难说朝鲜正积极致力于无核化，美国总统川普（特朗普）的忍耐是有限的〞。
 <br/>
 <br/>
 韩国国防部之前表示，计划在12月1日前决定是否重启军演。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
