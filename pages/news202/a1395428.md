### 韩破最大冰毒案逮捕6嫌 疑台日黑道涉入(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首尔.html" target="_blank">
  首尔
 </a>
 警察厅广域搜查队周一(15日)宣布破获当地历来最大宗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冰毒.html" target="_blank">
  冰毒
 </a>
 案，市值超过3700亿韩圜（约3.478亿美元），共逮捕6名国际贩毒组织成员，包括日本及台湾的黑帮分子。
 <br/>
 <br/>
 据《韩联社》引述警方公布的消息指出，被捕的毒贩分别是3名台湾人、2名日本人及1名韩国人。其中一名25岁台湾疑犯于7月6日由泰国曼谷出发，将将112公斤的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冰毒.html" target="_blank">
  冰毒
 </a>
 密封藏在螺丝制造机中，成功透过船运从釜山港入境。
 <br/>
 <br/>
 其后，他与另一名台湾同党将机器运至
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首尔.html" target="_blank">
  首尔
 </a>
 附近的华城市仓库并取出毒品，藏在首尔一间公寓内。
 <br/>
 <br/>
 在7月至8月间，这名25岁嫌犯共进行3次交易，将22公斤毒品流入市面，赚取11亿韩圜（约103万美元）。8月底，他察觉警方已展开调查，企图返回台湾，但在仁川国际机场遭警方逮捕，剩下的90公斤毒品也被没收。
 <br/>
 <br/>
 警方表示，现正追缉另外4名主谋者，已要求泰国警方追查船运纪录。
 <br/>
 <br/>
 警方也推测，此案应有台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竹联帮.html" target="_blank">
  竹联帮
 </a>
 成员，以及日本3大黑道组织成员涉入。
 <br/>
