### 前俄谍毒杀案 第2嫌被曝是俄军情局军医
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前双面间谍.html" target="_blank">
  前双面间谍
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯克里帕尔.html" target="_blank">
  斯克里帕尔
 </a>
 父女遭人以苏联研发的化学武器〝诺维乔克〞毒害一案，调查团体〝冒险者〞周一（8日）确认涉案的第2名嫌疑人，是俄国军事情报局（GRU）的军医。
 <br/>
 <br/>
 总部设在英国的〝冒险者〞（Bellingcat）周一公布报告说：〝我们如今确认‘彼卓夫’（Alexander Petrov）真名叫米什金（Alexander Yevgenyevich Mishkin），是一名训练有素、受雇于俄国军事情报局的军医。〞
 <br/>
 <br/>
 报告中说，米什金1979年7月出生在俄国北部阿汉格斯克（Arkhangelsk）地区，长大后在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 其中一间专门训练军方医疗人员的学府毕业，期间受训成为俄罗斯海军一名军方医生。
 <br/>
 <br/>
 不过，他在就学期间已被俄罗斯总参谋部情报总局看中，成为情报人员，并在2010年移居莫斯科后取得秘密身分，包括化名为彼卓夫的身分证和护照。米什金据说曾多次造访乌克兰，还将俄国军事情报局总部列为住家地址。
 <br/>
 <br/>
 另外，冒险者在上月底公布的另一份调查报告中称，在案发现场的另一名疑犯波希洛夫（Ruslan Boshirov），是于2014年授予俄罗斯最高荣誉勋章〝俄罗斯英雄〞的39岁上校契皮加（Anatoliy Chepiga）。该组织还公开了契皮加2003年的护照，照片与波希洛夫容貌极其相似。
 <br/>
 <br/>
 报导说，契皮加1979年出生于俄罗斯远东地区的尼科拉耶夫卡（Nikolayevka）村庄，毕业于当地一所有名的军校，是一名〝功勋彪炳〞的俄国情报局上校，以波希洛夫的假身分在英国从事秘密任务。
 <br/>
 <br/>
 冒险者说，该组织透过多个公开来源及〝熟识他的人的证词〞来搜集资讯。
 <br/>
 <br/>
 英国维安单位9月初指名波希洛夫和彼卓夫涉嫌在今年3月企图以苏联研发的化学武器〝诺维乔克〞（Novichok），在英格兰索尔兹伯里（Salisbury）杀害俄国前双面谍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯克里帕尔.html" target="_blank">
  斯克里帕尔
 </a>
 （Sergei Skripal）和他女儿尤莉雅（Yulia）。
 <br/>
 <br/>
 维安单位也说，两人上述名字可能是化名。
 <br/>
 <br/>
 英国首相特蕾莎‧梅（Theresa May）此前曾表示，毒杀案两嫌是俄罗斯军事情报局人员，依〝高层〞指示行事。
 <br/>
 <br/>
 两嫌被指认身分后，曾以化名现身俄国电视媒体，坚称他们只是前往英格兰城市索尔兹伯里（Salisbury）旅游的观光客。波希洛夫还否认为俄罗斯军事情报局（GRU）工作。
 <br/>
 <br/>
 英国政府指控俄罗斯当局企图杀害斯克里帕尔，报复他与欧洲情报单位合作。分析人员将检视冒险者的最新报告，针对俄国情报单位被控涉入其中寻找更多线索。
