### 斯里兰卡总统突换总理  恐酿宪政危机
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】周五，南亚岛国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯里兰卡.html" target="_blank">
  斯里兰卡
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统.html" target="_blank">
  总统
 </a>
 西里塞纳，突然宣布，撤换现任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总理.html" target="_blank">
  总理
 </a>
 维克勒马辛哈。不过，维克勒马辛哈拒绝卸任，观察人士担心，此事可能引发，斯里兰卡的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宪政危机.html" target="_blank">
  宪政危机
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯里兰卡.html" target="_blank">
  斯里兰卡
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统.html" target="_blank">
  总统
 </a>
 西里塞纳，周五任命前总统拉贾帕克萨为新任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总理.html" target="_blank">
  总理
 </a>
 。当地电视台播出了拉贾帕克萨宣誓的画面。
 <br/>
 <br/>
 拉贾帕克萨曾于2005年至2015年间担任斯里兰卡总统。拉贾帕克萨持亲北京立场，他将斯里兰卡带入中共的一带一路项目，导致国家背上沉重外债。在2015年的总统大选，拉贾帕克萨因败选，未能连任。
 <br/>
 <br/>
 消息公布后，拉贾帕克萨的支持者走上街头庆祝。但被撤换的总理维克勒马辛哈随即发表声明，拒绝卸任。
 <br/>
 <br/>
 遭解职的斯里兰卡总理  维克勒马辛哈：〝我的总理职位来自宪法第42条，并符合宪法第19修正案。我仍得到（议会）多数支持，因此我会继续担任总理，并履行总理职责。〞
 <br/>
 <br/>
 根据2015年通过的宪法第19号修正案，总统只有在现任总理失去资格时才可以将其撤换。观察人士担心，本次事件将引发斯里兰卡的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宪政危机.html" target="_blank">
  宪政危机
 </a>
 。
 <br/>
 <br/>
 有斯里兰卡民众接受采访时也表示，总统不应该绕过议会任命新总理。此举让斯里兰卡在全世界面前蒙羞。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
