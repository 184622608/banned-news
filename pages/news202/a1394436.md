### 里斯本附近突发山火 七百多消防员紧急扑救
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】距离葡萄牙首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里斯本.html" target="_blank">
  里斯本
 </a>
 附近的度假区周六突发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山火.html" target="_blank">
  山火
 </a>
 ，迫使数百人连夜撤离，700多名消防队员在6架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灭火飞机.html" target="_blank">
  灭火飞机
 </a>
 的支援下加紧扑救。
 <br/>
 <br/>
 野火是在星期六晚些时候发生的，10月份异常炎热的天气，加上强风，导致火势迅速蔓延。官方报导有18人受轻伤，多数是消防人员。
 <br/>
 <br/>
 当局连夜疏散了在当地露营的300人，以及附近村庄的近50位居民。目前仍有两个火势强劲的地点令人担忧。
 <br/>
 <br/>
 葡萄牙去年遭受了历史上最严重的野火，造成114人丧生。为确保悲剧不再重演，政府增聘了数百名消防人员，并加强控制、保证土地拥有者清除可引起火灾的诱因，事先疏散危险地区的居民。
 <br/>
 <br/>
 新唐人记者徐哲、天羽综合报导。
</div>
