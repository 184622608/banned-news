### 【禁闻】中共间谍渗透 法国反击钟声响起
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 媒体日前起底
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 大规模渗透活动。这是继
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 、新西兰、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 、德国、英国等国对中共间谍行为提出警报后，又一个西方国家加入这一行列。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 《费加罗报》日前刊登独家专题系列报导指出，至少有4,000名法国公务机关的主管和员工、或战略企业合作伙伴，以及意见领袖圈子里的红人，曾在《领英》（LinkedIn）或〝Viadeo〞等商务社交网站上，被中共情报人员搭上。
 <br/>
 <br/>
 这些中共情报人员使用的是假身份。
 <br/>
 <br/>
 报导说，法国对外安全总局（DGSE）和国内情报总局（DGSI）搜集了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 的充分资讯。报导并举出真实案例，而这些案例只是中共在法国运作间谍行动中的冰山一角。
 <br/>
 <br/>
 报导称，相关行动由中共国安部领导，该部门有近20万名员工。相对于法国对外安全总局和国内情报总局的人数，分别只有约6,000人和4,000人。
 <br/>
 <br/>
 根据报导，这些中共间谍的网络形象通常是年轻、有趣、会挑逗、有才华，且毕业于中国名校，他们背后的公司，有些在社交网站有上千人关注。欧洲情报部门资料显示，中共特务机构在《领英》网站上，至少管理着500个类似的假用户。
 <br/>
 <br/>
 报导引述消息表示，所有与战略影响力和敏感领域相关的单位都可能被盯上，像医疗卫生、资讯、核能、奈米科技和电讯等单位。法国情报单位判断，有数百个目标已进入〝受到相当牵连程度〞的阶段。
 <br/>
 <br/>
 时事评论员夏小强：〝中共的体制本身决定了它没有自主开发高科技的创新能力，所以它用间谍手段全面渗透，来盗窃西方国家的知识产权，进而威胁和影响到西方国家的政治和经济的安全。〞
 <br/>
 <br/>
 《费加罗报》另一篇系列报导提到，中共间谍除了在网络上〝钓人〞，还有偷窃、复制器材等手段。
 <br/>
 <br/>
 报导说，法国当局认为：这种大规模间谍操作〝对国家主权造成重大威胁，并置法国经济财产于险境〞，而〝反击的钟声已经响起〞。
 <br/>
 <br/>
 德国、英国、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 都曾公开发出警告说，中共间谍机构通过社交网络收集情报。
 <br/>
 <br/>
 在此之前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 、新西兰、美国、德国的多家媒体，也都详细揭露过中共间谍渗透的问题。
 <br/>
 <br/>
 夏小强：〝美国此前引渡在比利时的中共特工，已经发出明确信号，中共在海外的间谍随时都有可能被美国引渡。〞
 <br/>
 <br/>
 中共面对各国的指责，多年来一直辩称〝缺乏证据〞，或者提出抗议。
 <br/>
 <br/>
 不过，大陆法学博士张杰观察，西方国家对中共间谍的防范和打击是必然趋势。他表示，以前西方对中共采取绥靖政策，希望中共能有所改变，如今愿望落空，整个西方世界看待中共的视角都变了，认为中共这个红色巨人非常恐怖，正在打破世界秩序，所以开始从多方面进行审视。
 <br/>
 <br/>
 大陆法学博士张杰：〝你看我们的高铁、飞机，其实很多技术都来自于西方，并且是通过不正当的方式窃取的。所以使（西方国家）它们开始觉醒。目前这个红色帝国会对整个人类造成危害。现在反击虽然晚了一点，但是亡羊补牢吧！〞
 <br/>
 <br/>
 张杰表示，西方世界这种觉醒和反击，代表着一种新的战略格局正在形成。
 <br/>
 <br/>
 张杰：〝现在（西方国家）国家安全高于利益，同时中国（中共）不遵守国际贸易规则，在这种情况下（西方国家）开始联手反击、制裁。中国（中共）目前它的渗透、商业间谍活动已经是有一定规模了。但是由于它这种红色帝国的理念、价值观都是反人类的，所以它不可能成功的，也不可能得到中国人民的支持。〞
 <br/>
 <br/>
 今年5月，法国逮捕了两名涉嫌向中共出卖情报的法国对外安全总局前雇员。
 <br/>
 <br/>
 《法国国际广播电台》报导，法国情报部门近年来十分关注中共情报机构在欧洲，尤其在法国的部署，只是做得不动声色。
 <br/>
 <br/>
 为什么？
 <br/>
 <br/>
 因为〝不要打草惊蛇〞！
 <br/>
 <br/>
 采访/常春 编辑/王子琦 后制/陈建铭
