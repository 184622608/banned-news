### 不给糖就捣蛋  东京晴空塔万圣节亲子游行
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】10月的最后一天是西方的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万圣节.html" target="_blank">
  万圣节
 </a>
 ，也是一年一度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亲子同乐.html" target="_blank">
  亲子同乐
 </a>
 的好机会。在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东京.html" target="_blank">
  东京
 </a>
 ，一场不给糖就捣蛋的变装游行即将上演，随记者去看看。
 <br/>
 <br/>
 记者张卉：我现在位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东京.html" target="_blank">
  东京
 </a>
 墨田区，世界第一高塔东京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晴空塔.html" target="_blank">
  晴空塔
 </a>
 ，这里已经被南瓜灯笼和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万圣节.html" target="_blank">
  万圣节
 </a>
 装饰占领，在家长带领下的变装完毕的小朋友们更是跃跃欲试，准备开启今天的神奇之旅了。
 <br/>
 <br/>
 小魔女，超级玛丽，阿拉蕾，小黄人，天真可爱的小朋友们摇身一变，成了不给糖就捣蛋的淘气小精灵。更有一家大小齐齐变装，一大早就聚集在晴空塔下，等待一年一度的万圣节亲子游行开始。
 <br/>
 <br/>
 小朋友家长：每年都和朋友来参加这个活动，每年都很开心，所以今年也来了。
 <br/>
 <br/>
 上午11点，由家长和小朋友们组成的游行队伍出发，横穿全长120米的天空町商店街。期间35家店铺，无论是店面的布置还是糖果的储备，都已经做足准备，以满足孩子们一个精彩甜蜜的万圣节之旅。
 <br/>
 <br/>
 满载来的小朋友们聚集到天空露天广场，在这里还有天空塔妹妹和她的小伙伴们等待着他们，一起跳起快乐的魔法舞蹈。
 <br/>
 <br/>
 东武集团商业运营本部 内田 麻纪惠：晴空塔每个季节都有不同的活动，秋天的庆典就是万圣节游行。希望大家可以享受这个活动的同时，知道一年都可以在这里开心度过。
 <br/>
 <br/>
 除了这些庆祝活动之外，各种路边
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艺人表演.html" target="_blank">
  艺人表演
 </a>
 等，也将气氛推向高潮。成为了东京旅行新看点。
 <br/>
 <br/>
 新唐人日本记者站报导
</div>
