### 相继与美日、俄中会晤 莫迪G20推文表差别
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度总理莫迪.html" target="_blank">
  印度总理莫迪
 </a>
 在阿根廷出席20国集团（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 ）峰会相当活跃，30日深夜先出席印度、美国、日本首次三边领袖峰会，促进印太海上合作，随后又出席印度、俄罗斯和中国领袖峰会。但在看待这两场场边的重量级领袖峰会，从莫迪在推特的推文内容中仍可发现差别。
 <br/>
 <br/>
 中央社报导，莫迪说，与美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 、日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 举行的三边峰会为JAI（日本、美国和印度的英文开头字母）。JAI在印地语（Hindi）意为〝胜利〞，言外之意非常明显。
 <br/>
 <br/>
 在推文中，莫迪表示，JAI三边峰会象征3个友好国家聚在一起，今天别具历史意义，因为这是JAI峰会伟大的开始。
 <br/>
 <br/>
 他在推文中指出，与川普、安倍举行富有成果的会谈，这场会谈目的在促进印度太平洋地区的连通、海上合作和稳定。
 <br/>
 <br/>
 印度外交部次长顾凯杰（Vijay Gokhale）在JAI峰会结束后向印度媒体表示，莫迪与川普、安倍都同意自由、开放、包容及以规则为基础的秩序，对印太地区的和平与繁荣是不可或缺的。
 <br/>
 <br/>
 印度〝国际亚洲新闻社〞（ANI）报导，顾凯杰说，莫迪等3位领袖也同意东协在实现印太地区和平与繁荣中，扮演核心角色。
 <br/>
 <br/>
 中国持续在印太地区扩张政治、经济与军事影响力后，川普政府提出〝印度太平洋〞新战略观，且希望透过与印度、日本和澳洲的四边联盟遏制中国的扩张，而印度则在美国这项战略中扮演重要角色。
 <br/>
 <br/>

 <br/>
 莫迪虽多次强调印太地区必须拥有包容性，且不针对任何特定国家，但面对中共的威胁与在南亚及印度洋部署的包围网，印度仍持续强化与美、日和越南等东南亚国家的安全合作，都有对抗中国在南亚与印度洋地区的扩张的意味。
 <br/>
 <br/>
 不过，面临2019年大选寻求连任的莫迪政府，在中印部队洞朗对峙事件及美中贸易战后也调整政策，4月与中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在武汉举行中印非正式峰会，同意维持边界的和平，习近平也希望拉拢印度共同对抗美国兴起的贸易保护主义。
 <br/>
 <br/>
 因此，在结束与川普、安倍的JAI三边峰会后约5个小时，莫迪又在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 场边与俄罗斯总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 （Vladimir Putin）、中国国家主席习近平举行第2次印、俄、中三边峰会，这次会议距离首次印、俄、中三边峰会相隔12年。
 <br/>
 <br/>
 莫迪仅在推特上短短地提到，他和普京、习近平讨论广泛议题，进一步巩固印、俄、中三国邦谊，促进世界和平。
 <br/>
 <br/>
 此外，印度外交部发言人库玛（Raveesh Kumar）在推特指出，莫迪在G20第2次会议中提出针对潜逃经济罪犯的9点打击计画，希望各国建立共识打击经济罪犯。
