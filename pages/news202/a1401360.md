### 博尔顿前往巴西 邀准总统博索纳罗访美
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】周四，美国国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 会见了该国候任总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博索纳罗.html" target="_blank">
  博索纳罗
 </a>
 ，并邀请他访问美国。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 的车队停在了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博索纳罗.html" target="_blank">
  博索纳罗
 </a>
 的家门外，登门会见了这位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 候任总统还有他的候任内阁成员。
 <br/>
 <br/>
 博索纳罗被视为右翼保守派人士，被称为巴西版川普。他曾公开表示，非常喜欢美国川普总统。他曾表示，将扭转巴西的外交政策，不再支持世界上的独裁政权，并不再贬损美国、意大利、以色列等民主国家。
 <br/>
 <br/>
 他们在会谈中表示，美国巴西两国建立更紧密的关系非常重要。博尔顿并邀请博索纳罗访问美国。
 <br/>
 <br/>
 结束在巴西里约热内卢的闪电式访问后，博尔顿启程前往阿根廷，与川普总统共同出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
