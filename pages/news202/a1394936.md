### 印尼爪哇岛6.0强震 观光客被震醒至少3死
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】印尼旅游胜地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爪哇岛.html" target="_blank">
  爪哇岛
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴厘岛.html" target="_blank">
  巴厘岛
 </a>
 海岸周四（11日）凌晨发生规模6.0浅层地震，观光客从睡梦中被震醒，已知至少3人丧生。
 <br/>
 <br/>
 地震发生在当地时间周四凌晨1时44分，震央位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爪哇岛.html" target="_blank">
  爪哇岛
 </a>
 松伯伦尼亚（Sumberanyar）北北东方约39公里处，深度仅9公里，属极浅层地震。地震导致东爪哇省的苏门内普（Sumenep）地区发生房屋倒塌，至少3人死亡。
 <br/>
 <br/>
 印尼国家灾害应变总署发言人苏托波（Sutopo Purwo Nugroho）表示，地震今天清晨发生，许多人在睡梦中，来不及逃出。地震造成损害范围不大。
 <br/>
 <br/>
 度假胜地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴厘岛.html" target="_blank">
  巴厘岛
 </a>
 的登巴萨（Denpasar），民众感到强烈地震，仓惶逃出建筑物。
 <br/>
 <br/>
 邻近的观光胜地巴厘岛也感受到强烈震度。岛上居民苏迪阿尼（Ni Komang Sudiani）说：〝地震很大，我马上就醒来带着孩子逃出家门。我的邻居也都在逃命。〞
 <br/>
 <br/>
 妇人德维（Davy）表示，这次地震真的很强，而且持续很久。她当时立即逃到巴厘岛一家饭店的停车场避难，这里距离国际货币基金（IMF）和世界银行（World Bank）本周举行年会的地方只有数公里。
 <br/>
 <br/>
 印尼官方表示，此次地震没有引发海啸警报。
 <br/>
