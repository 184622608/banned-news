### 韩国KTX列车惊传脱轨 车头横跨轨道酿14伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】一列载有198名乘客的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国高速铁道.html" target="_blank">
  韩国高速铁道
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_KTX.html" target="_blank">
  KTX
 </a>
 ）列车，今(8日)上午仅离站行驶7分钟后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 ，造成至少14人受伤，所幸全部没有生命危险。相关单位正调查出事原因。
 <br/>
 <br/>
 负责营运
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_KTX.html" target="_blank">
  KTX
 </a>
 的韩国铁路集团（Korea Railroad Corp）表示，出事列车于早上7时30分(当地时间)，由江原道的江陵市开出，预定在9时30分抵达首尔。
 <br/>
 <br/>
 这辆编号806号列车共有10节车厢，以时速103公里，在珍富站附近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 ，其中除了10号车厢外，其余都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱轨.html" target="_blank">
  脱轨
 </a>
 ，有4节是横卧在路轨上，其中有2节横跨在另一个轨道。
 <br/>
 <br/>
 坐在第8号车厢的乘客回忆说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高铁.html" target="_blank">
  高铁
 </a>
 从江陵站出发7分钟后，突然传出一声巨响，车厢剧烈摇摆后向左倾斜。有乘客说，列车不像在铁轨上行驶，而是硬生生在地面上弹跳。
 <br/>
 <br/>
 有乘客说，列车出轨前曾滑行50公尺远，期间听到类似车轮摩擦碎石的声音。也有乘客说，列车出轨时不少人从座位上跌了下来，惨叫声连连。
 <br/>
 <br/>
 另有乘客说，出事后看见一名老妇头部受伤流血，他们在车厢外等了50分钟才有救援人员到场。
 <br/>
 <br/>
 铁路公司安排乘客搭乘巴士至珍富站，再转搭另一班列车。而由首尔往珍富站的列车保持正常行驶。
 <br/>
 <br/>
 目前出事路段暂停使用，政府的运输及交通部门已派人至现场调查。
 <br/>
