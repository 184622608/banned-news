### 墨尔本中心爆恐袭1死2伤  犯人遭击毙
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】当地时间9号下午，澳大利亚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨尔本.html" target="_blank">
  墨尔本
 </a>
 ，发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恐怖袭击事件.html" target="_blank">
  恐怖袭击事件
 </a>
 。嫌犯纵火烧车，并用刀刺死1人，刺伤2人。当地警方迅速赶到，将嫌犯击毙。
 <br/>
 <br/>
 事件发生在下午4点30分左右。警方接到报警，位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨尔本.html" target="_blank">
  墨尔本
 </a>
 中心商业区的伯克街，有一辆汽车起火燃烧。
 <br/>
 <br/>
 警方赶到现场，发现汽车上载满了野餐用的瓦斯罐，所幸没有发生爆炸。此时，警方与犯人相遇，犯人用刀具攻击警察。
 <br/>
 <br/>
 目击者表示，犯人此前已经用刀攻击民众。警察随后击中犯人胸部，犯人在送往医院后不治身亡。
 <br/>
 <br/>
 警方表示，事件造成民众一死两伤。与犯人对峙的警察基本平安。
 <br/>
 <br/>
 维多利亚州警司 大卫・科雷顿：〝有三人遇刺，不幸的是，一人当场死亡，另外两名受害者目前正在医院救治。〞
 <br/>
 <br/>
 事件发生后，澳大利亚总理莫里森迅速在推特发表声明，谴责这一事件，赞扬警察的勇敢，并表示已经将事件作为恐怖袭击处理。
 <br/>
 <br/>
 目前警方尚未公布犯人以及受害者的身份，但表示犯人有家属位列情报部门的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恐袭.html" target="_blank">
  恐袭
 </a>
 观察名单。
 <br/>
 <br/>
 维多利亚州总警长 格拉汉姆・阿什顿：〝有关犯人，警方知道这个人，主要原因是他有一些亲属受到我们的特别关注。〞
 <br/>
 <br/>
 目前，事发附近街区已经被封锁，预计将至少持续到周六上午。同一地区2017年1月发生了汽车撞人袭击，造成6人死亡。警方表示，目前没有发现两次袭击之间有关联。
 <br/>
 <br/>
 新唐人记者林宇、张琪综合报导
</div>
