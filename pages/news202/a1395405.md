### 暗杀川普 揭秘:美特勤局挫败ISIS行动
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统对恐怖主义实行强硬政策，伊斯兰国（I
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_SIS.html" target="_blank">
  SIS
 </a>
 ）恐怖分子在川普上任一年后曾策划过一次暗杀行动，但被美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特勤局.html" target="_blank">
  特勤局
 </a>
 挫败。美国电视网络〝国家地理频道〞周日（10月14日）揭开了这一秘密。
 <br/>
 <br/>
 周日，美国有线数字和卫星电视网络〝国家地理频道〞（National Geographic Channel, NGC）发表了一段视频，揭秘了2017年11月发生的一次未遂的暗杀
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 行动。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特勤局.html" target="_blank">
  特勤局
 </a>
 （U.S. Secret Service）总统保卫科（Presidential Protective Division）负责人奥纳托（Anthony Ornato）曾一再告诉手下特工，他相信，当2017年11月川普赴菲律宾马尼拉（Manila）参加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东盟.html" target="_blank">
  东盟
 </a>
 （ASEAN）成立50周年活动及第31届峰会时，将会面临暗杀威胁。
 <br/>
 <br/>
 在川普抵达马尼拉前几天，特工吉布森（Gibson）表示，有可靠信息显示，东盟会议期间将有暗杀发生。当时他说，马尼拉的危险级别正在上升。
 <br/>
 <br/>
 川普参加东盟峰会之前，I
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_SIS.html" target="_blank">
  SIS
 </a>
 通过视频发出了一系列威胁——配有多发子弹的川普照片、敦促武装分子等待并在菲律宾伏击川普的信息。
 <br/>
 <br/>
 在川普乘〝空军一号〞专机抵达马尼拉之前，一名其它机构的特工告诉吉布森，他发现了一条推文，内文写道，〝将与川普同时出现在马尼拉……我会带一个队友〞，并附带一张刺杀美国前总统肯尼迪的凶手奥斯瓦尔德（Lee Harvey Oswald）的脸部照片。
 <br/>
 <br/>
 在此人的社交媒体Instagram账号上，特工们还发现，这名男性嫌疑人手中挥舞着一本关于如何暗杀的书的副本，书名是《刺客的最佳历史》（The Definitive History of the Assassin）。
 <br/>
 <br/>
 特工们后来依据IP地址追踪，发现该男子实际上就在马尼拉市中心，距川普下榻的酒店只有数公里远。而他在社交媒体上的帖文透露，他正要向该酒店的方向行进。
 <br/>
 <br/>
 这一情况出现在〝空军一号〞抵达马尼拉之前20分钟的时候，当时，特勤局仍未找出嫌疑人的具体位置。负责川普本次出行的特工拉根（Chad Ragan）要求特勤局给出答案：〝我需要更新（消息），现在！〞
 <br/>
 <br/>
 不过，几分钟后，特工便找到了嫌疑人所在位置——就在距离川普所居酒店仅一英里远的一个公园里，而他正跟另一个碰头。在菲律宾国家警方的帮助下，美国特勤局特工横扫了该公园，并逮捕了嫌疑人。
 <br/>
