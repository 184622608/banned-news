### 〝再别康桥〞美景被淹没 英拟限制陆客流量
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】在徐志摩〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再别康桥.html" target="_blank">
  再别康桥
 </a>
 〞号召下，越来越多到英国旅游的中国人(陆客)涌入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_剑桥.html" target="_blank">
  剑桥
 </a>
 ，抱着康河边的老柳，甚至坐到树杈上拍照留念。由于陆客实在太多，剑桥官方旅游机构正在考虑限制中国观光客数量。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_剑桥.html" target="_blank">
  剑桥
 </a>
 是英国历史最悠久的大学城，有31所学院，也有图书馆和博物馆，位于伦敦以北约80公里处，依着康河发展。但如今〝夏虫也为我沉默，沉默是今晚的康桥〞，徐志摩笔下的康桥（现称剑桥）已不再沉默。
 <br/>
 <br/>
 《BBC中文网》报导，据统计，2015～2017年间，前往剑桥游玩并过夜的陆客就有1.7万人，而2016年到剑桥一日游的陆客高达26.7万人，在英国仅次于伦敦、爱丁堡、曼彻斯特。
 <br/>
 <br/>
 报导称，剑桥前市长、市议员约翰・西普金（John Hipkin）表示，剑桥一日游的游客里陆客占比相当高，他也直言中国旅行社为了赚钱〝组团过大〞。西普金担忧，剑桥恐被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国游客.html" target="_blank">
  中国游客
 </a>
 淹没，未来将〝自毁〞。
 <br/>
 <br/>
 来英国旅游的中国人，多数都会到剑桥抱着康河边的老柳，甚至坐到树杈上拍照留念，尽管此柳可能已经非彼柳。而在那里的中国游客数量和游客素质，像在世界其他地方一样，也经常成为热门话题。
 <br/>
 <br/>
 剑桥官方旅游管理服务机构〝访问剑桥〞的负责人艾玛・索尔顿（Emma Thornton）称，在剑桥古镇狭窄的街道上，经常出现超过50人的中国旅行团，这也造成当地交通堵塞。该机构正在考虑推出限制陆客数量的措施，包括规定旅行团人数上限、协调旅行团之间的游玩路线避免蜂拥一处。
 <br/>
 <br/>
 报导还提及，除了剑桥，基德林顿（Kidlington）这个名不经传的小镇，自2016年夏天起也有大量陆客涌入。那里没有什么著名的观光景点，更没有什么特别之处，但这些陆客似乎对村里的一砖一瓦、一草一木都兴致盎然，一直拍照，甚至连门外的垃圾桶都不放过。
 <br/>
 <br/>
 报导称，这也让当地人百思不得其解，解铃还需系铃人，北京市华远国际旅游有限公司导游孙建锋道出其中的秘密，原来将游客带到此地是避免那些不愿花钱去附近布伦海姆宫（Blenheim Palace）参观的人发现旅行社卖的门票高出当地门票。
 <br/>
 <br/>
 前往布伦海姆宫的集合地点原本是在布伦海姆宫附近，但一些聪明的游客发现，在布伦海姆宫现场买票，只需约25美元，而向旅游团买则要68美元，于是陆客们悄悄自己步行前往。
 <br/>
 <br/>
 因此，导游将那些不参观布伦海姆宫的人就留在8公里外的基德灵顿，从那里无法步行前往布伦海姆宫。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
