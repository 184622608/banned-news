### 华为CFO被捕或引渡美国 中方要求放人
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】中国华为公司副董事长兼首席财务官（CFO）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 （Wanzhou Meng），因涉嫌违反美国对伊朗的贸易制裁，上周六在加拿大温哥华机场被逮捕，可能被引渡到美国。中国方面周四要求立即释放孟晚舟。
 <br/>
 <br/>
 加拿大司法部证实，12月1日应美国要求在温哥华国际机场逮捕了华为副董事长兼首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 。
 <br/>
 <br/>
 孟晚舟从母姓，父亲是华为公司创始人任正非（Ren Zhengfei）。川习会在阿根廷登场的当晚孟晚舟在加拿大被捕，涉嫌违反美国对伊朗的出口禁令，可能被引渡到美国。若罪名成立，将对华为的经营带来打击。
 <br/>
 <br/>
 专家指出，这起事件还可能影响美中谈判，两国紧张关系正在迅速升级。
 <br/>
 <br/>
 中共外交部发言人耿爽周四称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中方.html" target="_blank">
  中方
 </a>
 已要求美国立即释放被拘押的人员。
 <br/>
 <br/>
 华为周三则发布声明表示，相信〝加拿大和美国的法律体系，最终将得出公正的结论。〞
 <br/>
 <br/>
 加拿大司法部表示，由于孟晚舟申请〝报导禁止令〞，不能对外公开，目前无法提供更进一步的细节，孟晚舟的保释聆讯将在周五进行。
 <br/>
 <br/>
 美国媒体今年4月曾报导，美国司法部对华为是否违反美国贸易禁令，向伊朗违法出口，进行调查。
 <br/>
 <br/>
 美国今年采取行动，减少华为和中兴通讯公司进入美国市场。
 <br/>
 <br/>
 澳大利亚、新西兰、德国和英国也相继质疑华为受制于中共，存在间谍和安全隐患，采取抵制行动。
 <br/>
 <br/>
 华为官网信息显示，孟晚舟毕业于华中理工大学，拥有硕士学位，1993年加入华为，现任副董事长兼首席财务官。
 <br/>
 <br/>
 华为从未上市，管理架构神秘，被指长期帮助中共监视中国民众。
 <br/>
 <br/>
 新唐人记者李梅、张琪综合报导
</div>
