### 蓬佩奥东亚四国行 朝鲜去核再进一步
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】10月8日，美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 结束对中国的访问后返回美国，为此次东亚4国行画上句点。
 <br/>
 <br/>
 周一美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 在社交媒体Instagram贴出组图，宣布6-8日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东亚四国.html" target="_blank">
  东亚四国
 </a>
 之行取得成果。
 <br/>
 <br/>
 8日，蓬佩奥行程最后一站北京，与中共政治局委员杨洁篪和外交部长王毅会晤。
 <br/>
 <br/>
 中国外长 王毅：〝我们要求美方停止错误做法。〞
 <br/>
 <br/>
 王毅把美中之间的摩擦归咎于美方。蓬佩奥直言不讳的予以驳斥。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝你提到的这些议题我们有根本的分歧，我们密切关注中国的行动。〞
 <br/>
 <br/>
 对于中方取消月底两国国防部长安全对话等一系列行动，蓬佩奥表示遗憾。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝我感到遗憾，对于中美两国的战略性对话，中方选择不进行。〞
 <br/>
 <br/>
 此次中国行，蓬佩奥没有与中国国家主席习近平会面。
 <br/>
 <br/>
 蓬佩奥出访东亚前，副总统彭斯发表美中关系主题演讲，外界普遍认为是历史性发言，美国将中共定位为全面竞争对手，并宣布具体措施来重塑以尊重互利为基础的双边关系。
 <br/>
 <br/>
 在抵达北京前，蓬佩奥向韩国总统文在寅介绍了与金正恩的会晤情况。
 <br/>
 <br/>
 7日，蓬佩奥与金正恩会谈达5个多小时，此次会晤，蓬佩奥再次提出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 必要的去核行动、对应的检验步骤及美方相应措施。金正恩准备允许国际核查人员进入朝鲜监督去核（一个导弹引擎测试设施和丰溪里核试验设施）。
 <br/>
 <br/>
 朝鲜官方媒体报导，金正恩表示与蓬佩奥的对话〝有成效，很愉快〞。
 <br/>
 <br/>
 美国总统川普发推文称赞国务卿朝鲜之行的收获，并表示〝期待不久再次与金主席会晤。〞
 <br/>
 <br/>
 韩国总统 文在寅：〝蓬佩奥国务卿的朝鲜之行为未来不久的第二次川金会创造了条件。〞
 <br/>
 <br/>
 除了第二次川金会，文在寅还提到，预计很快金正恩会出访俄罗斯，而且习近平可能访问朝鲜。他表示，朝鲜半岛的新秩序正在建立之中。
 <br/>
 <br/>
 而在此次亚洲行的首站日本，蓬佩奥与日本首相安倍晋三及外相河野太郎分别进行会晤，就访朝议题提前沟通。
 <br/>
 <br/>
 对于经济制裁朝鲜，蓬佩奥在出访前已经明确，是实现无核化的核心条件，不会改变。
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
