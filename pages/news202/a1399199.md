 周一，包括《纽约时报》（The New York Times）和福克斯新闻（Fox News）在内的很多媒体都报导，朝鲜在山中隐藏了多个小型的秘密导弹基地。该消息显然是在质疑川普对朝政策的实际效果。
 <br/>
 <br/>
 该消息来自华府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_智库.html" target="_blank">
  智库
 </a>
 〝战略与国际研究中心〞（Center for Strategic and International Studies, CSIS）。该智库根据最新发布的卫星图像分析，朝鲜至少隐藏着16个小型导弹基地，而此前朝鲜官员并未承认它们的存在。
 <br/>
 <br/>
 CSIS的报告称，在距离韩朝非军事区仅50英里远的黄海北道黄州郡的〝沙坎墨〞（Sakkanmol）基地，装备有总部大楼、军营、维修站和地下隧道入口，这些地下隧道可以隐藏导弹和运输卡车。
 <br/>
 <br/>
 该报告还称，根据政府、国防部和情报机构官员以及〝脱北者〞所提供的消息，可以确定共有大约15至20个由朝鲜军方所控制的导弹基地。
 <br/>
 <br/>
 上周三，针对朝鲜威胁如美国不减轻制裁就将继续发展核武器，川普表示，〝不急于〞解决朝鲜问题，因为对朝制裁仍在持续，而朝鲜则已停止发射导弹和火箭，朝鲜所扣押的美国人质也已返回家园。
 <br/>
 <br/>
 不过川普同时表示，他会在明年早些时候会晤金正恩。
 <br/>
