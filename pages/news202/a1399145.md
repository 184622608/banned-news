### 迫叛军谈判 也门亲政府军发动猛攻酿150人丧命(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_也门.html" target="_blank">
  也门
 </a>
 (
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叶门.html" target="_blank">
  叶门
 </a>
 )爆发军事冲突，有沙特阿拉伯为首联军当后盾的也门政府消息人员说，他们发动一场向港口城市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_荷台达.html" target="_blank">
  荷台达
 </a>
 码头推进的大规模攻击，遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青年运动.html" target="_blank">
  青年运动
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叛军.html" target="_blank">
  叛军
 </a>
 顽抗击退。据医务人员，24小时内至少有150人丧命。
 <br/>
 <br/>
 沙特为首联盟发言人在沙国首都利雅德（Riyadh）向媒体表示，为了迫使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叛军.html" target="_blank">
  叛军
 </a>
 走上谈判桌，〝行动仍持续进行〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_荷台达.html" target="_blank">
  荷台达
 </a>
 （Hodeida）居民说，荷台达周遭地区的冲突昨晚有所停歇。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青年运动.html" target="_blank">
  青年运动
 </a>
 （Houthis）自2014年掌控战略地位重要的荷台达码头迄今，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_也门.html" target="_blank">
  也门
 </a>
 粮食进口和援助品80%都以荷台达为入口点，因此对于1400万面临饥荒危机的也门民众来说，荷台达码头是重要生命线。
 <br/>
 <br/>
 联合国警告，港口只要有一丁点破坏，都会引发饥荒。
 <br/>
 <br/>
 也门亲政府军效忠者则极力将青年运动叛军赶出荷台达。新一波的攻撃发生后，据统计，荷台达省各家医院24小时内共通报111名叛军丧生，32名亲政府战士身亡。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-13/p9124002a760872793.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-13/p9124002a760872793-ss.jpg"/>
   <br/>
   <font size="-1">
    2018年11月10日，也门亲政府军在荷台达东郊持续向码头推进。（STRINGER/AFP/Getty）
   </font>
  </a>
 </center>
 <br/>
 为此，英国外交大臣韩特（Jeremy Hunt）赶赴波斯湾地区，访问沙特及阿拉伯联合大公国首都并会晤两国领导人，试图推动各国展开停火谈判。
 <br/>
 <br/>
 韩特表示，在沙国证实他们愿意把50名受伤的青年运动士兵撤离到阿曼（Oman）接受治疗后，结束也门近4年内战的谈判前景〝更为真实了〞。
 <br/>
 <br/>
 韩特告诉路透社：〝这是个迫切的人道主义局势，所以我要让每个人都明白我们面临局势的紧迫性。〞
 <br/>
 <br/>
 上月底，美国国防部长马蒂斯（Jim Mattis）与国务卿蓬佩奥（Mike Pompeo）罕见的先后公开呼吁，沙特阿拉伯联军与也门〝青年运动〞停火，并于未来一个月就持续的也门内战坐上谈判桌。
 <br/>
 <br/>
 但青年运动持续利用也门首都沙那国际机场和毗邻的空军基地发动攻击，导致联合国主导的和平会谈无法顺利举行。
