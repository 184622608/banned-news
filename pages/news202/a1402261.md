### 孟晚舟若罪成将获刑最高30年 2千万美元罚款
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创始人任正非之女、华为副董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在加拿大被捕，面临到美国受审的命运。港媒东网12月6日报导，根据美国法律，个人被裁定违反
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁令.html" target="_blank">
  制裁令
 </a>
 罪成，面临的最高处罚是入狱30年，罚款2000万美元。
 <br/>
 <br/>
 报导还指，美国财政部外国资产控制办公室（OFAC）负责执行对伊朗的贸易制裁，将违反制裁视为对美国国家安全及外交关系的严重威胁。
 <br/>
 <br/>
 当日早些时候，加拿大公布了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 12月1日晚间被捕的消息。加拿大是应美方要求抓捕孟晚舟，罪名是违反美国对伊朗的制裁禁令。根据美方要求，孟晚舟或将被引渡到美国受审。
 <br/>
 <br/>
 据今年4月美媒消息，美国商务部宣布对中兴的禁售令的同时，美国司法部也就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 与伊朗违规交易展开刑事调查。
 <br/>
 <br/>
 孟晚舟被抓后，中共官方指责加拿大和美国〝侵犯人权〞。
 <br/>
 <br/>
 但港媒指，华为在美国有业务，必须接受当地法律监管。而美方制裁伊朗的法律规定，禁止非美国公民在未获授权的情况下，向伊朗再输出商品、科技或服务。
 <br/>
 <br/>
 另外，根据美加签订的司法双边协定，加拿大也有权暂时扣留违反美国法律的第三国公民，并引渡当事人至美国。
 <br/>
 <br/>
 此外，还有媒体爆料，孟晚舟具有双重国籍，同时持有中国护照和加拿大护照，因此也在加拿大法律管辖范围内。
 <br/>
 <br/>
 路透社称，孟晚舟被捕的消息，加剧了中美两国紧张形势进一步加剧的担心，惴惴不安的情绪笼罩金融市场。
 <br/>
 <br/>
 彭博社报导，这是美中贸易战的一次重大升级，将加剧人们对美中新冷战的担忧。川普（特朗普）政府坚持要求中共停止向战略部门提供政府支持，而这是中共〝中国制造2025〞政策的一部分。
 <br/>
 <br/>
 美国前中央情报局中国分析师韦德宁（Dennis Wilder）认为，中方可能会认为美方此举意在增加90天谈判期间的压力。
 <br/>
 <br/>
 但彭博社援引分析师表示，此案更有可能与贸易谈判分开进行，只是川普加强打击中共经济间谍活动和违反制裁禁令的一部分。
 <br/>
 <br/>
 韦德宁称，〝这是在传递新游戏的信号〞，美方试图阻止中共的间谍活动，并向北京展示这会产生〝真正的后果〞。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
</div>
