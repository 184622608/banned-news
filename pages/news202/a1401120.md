### 威胁国家安全 新西兰政府禁止华为设备
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新西兰电信.html" target="_blank">
  新西兰电信
 </a>
 公司Spark新西兰(Spark NZ)今(28日)表示，政府通讯安全局以对国家安全构成重大风险为由，拒绝公司在计划架设的行动网路中，使用中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 技术有限公司5G设备的提案。并称，这些决定不会影响在2020年7月1日前推出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5G网路.html" target="_blank">
  5G网路
 </a>
 的计画。
 <br/>
 <br/>
 Spark NZ先前通告新西兰(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽西兰.html" target="_blank">
  纽西兰
 </a>
 )政府通信安全局(GCSB)，称打算在其计划中的5G无线接入网路中使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 5G设备，该网络涉及与手机信号发射塔基础设施相关的技术。
 <br/>
 <br/>
 GCSB认定，Spark NZ（Spark New Zealand）提议在其计划中的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5G网路.html" target="_blank">
  5G网路
 </a>
 中使用华为5G设备，将升高国家安全风险。
 <br/>
 <br/>
 Spark NZ今日声明中表示，政府通讯安全局做出的这项决定，代表Spark无法实施或执行相关提案，在计画架设的5G网路中使用华为设备。
 <br/>
 <br/>
 华为尚未立即回应置评请求。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-28/p9147011a489880920.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-28/p9147011a489880920-ss.jpg"/>
   <br/>
   <font size="-1">
    Spark NZ提议在其计划中的5G网路中使用华为5G设备，将升高国家安全风险。(Phil Walter/Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 Spark NZ方面表示，公司相信这一裁定不会影响Spark 5G网路于2020年7月1日正式投入营运的计画。
 <br/>
 <br/>
 据悉，最近西方国家日益担忧中国企业在亚太地区不断增强的影响力。
 <br/>
 <br/>
 在此之前，澳洲就以〝对国家安全构成重大风险〞为由禁用华为5G设备。另外，澳洲8月扩大国家安全法规，排除据信与外国政府有挂钩的电信设备供应商。
 <br/>
 <br/>
 〝华尔街日报〞上周报导，美国政府试图说服盟国的无线与网路供应商，避用中国华为技术有限公司的电信设备。
 <br/>
 <br/>
