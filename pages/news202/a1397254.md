### 印尼狮航新机海上坠毁 机上恐无人生还
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】观众朋友大家好，欢迎收看新唐人全球新闻。周一，印尼发生了有史以来第二大空难，一架波音飞机起飞后不久坠入海中。印尼官方确认，机上189人全部遇难。
 <br/>
 <br/>
 一只鞋漂浮在海面上，这是周一坠落大海的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼狮航.html" target="_blank">
  印尼狮航
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_JT610.html" target="_blank">
  JT610
 </a>
 班机上旅客的遗物。
 <br/>
 <br/>
 当地时间周一早上，飞机从雅加达起飞13分钟后，就与塔台失联，不久后在海岸九英里外坠毁。目前失事原因还在调查中。
 <br/>
 <br/>
 狮航表示，失事客机是今年新产出的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波音737.html" target="_blank">
  波音737
 </a>
 飞机，8月才开始运营。客机失事前状况良好。
 <br/>
 <br/>
 印尼总统佐科威)当天慰问了旅客家属。
 <br/>
 <br/>
 印尼总统佐科威：〝我已下令他们连续工作24小时，（夜间）使用灯光加速搜救。〞
 <br/>
 <br/>
 不少飞机残骸漂浮在海面上，机身沉入水下98到115英尺处。印尼搜救人员表示，机上189人恐怕无一生还。
 <br/>
 <br/>
 印尼国家搜救机构运营总监Bambang Suryo Aji：〝我的预测是无人存活，因为我们只能找到不完整的身体碎块。〞
 <br/>
 <br/>
 印尼是全球增长最快的航空市场之一，这次坠机是该国第二大空难。1997年，一架空中巴士A300型客机坠落在印尼山区，机上234人全部遇难。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
