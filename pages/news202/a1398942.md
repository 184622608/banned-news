### 箱根诸侯巡游复兴 300年古风再现
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】日本在继承和传承自己本民族文化上，做出了众多努力。著名的日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_箱根温泉乡.html" target="_blank">
  箱根温泉乡
 </a>
 ，就复兴了一种叫做诸侯巡游的庆典，承传文化的同时，也促进了箱根温泉的观光业。
 <br/>
 我现在位于秋高气爽的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_箱根温泉乡.html" target="_blank">
  箱根温泉乡
 </a>
 ，在我身后一场传统诸侯巡游正在盛大上演，游行再现了日本江户时代诸侯进京述职的景象，带领游客穿越时空，体味不一样的东瀛风情。
 <br/>
 <br/>
 威武的火枪队鸣枪开道，伴随打头人〝回避…行礼…〞的叫声，家臣们根据各自的职责，手持长枪、弓箭、火枪等武器徐徐前行，公主和仕女队伍端庄温婉，倩影引人注目。再现了江户时代（1603—1867年）诸侯队伍列队进京的情景。
 <br/>
 <br/>
 诸侯巡游主催方井岛诚行：〝大名行列（诸侯巡游）是江户时代〝参勤交代〞制度，要求诸侯轮流到中央述职，规定诸侯每隔一年必须往返领地和江户（现东京）之间一次。现在的城镇概念也是在漫长的历史过程中形成的，我们将其呈现，期望更多人可以了解日本传统文化。〞
 <br/>
 <br/>
 被演员们热情感染，沈醉在传统文化的气氛中的国内外游客，更是开心不已。
 <br/>
 <br/>
 意大利游客：〝从东京到京都的路上，听说有这样棒的巡游，就带大家来了。〞
 <br/>
 <br/>
 日本游客：〝日本的文化，长年坚持是一件很辛苦的事，希望大家坚持传承下去。〞
 <br/>
 <br/>
 著名的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_箱根诸侯巡游.html" target="_blank">
  箱根诸侯巡游
 </a>
 肩负传承大名行列的特有文化。成为每年不可错过的经典行程。
 <br/>
 <br/>
 新唐人日本记者站报导
