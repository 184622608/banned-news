  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍.html" target="_blank">
   安倍
  </a>
  港媒：离间美日同盟
 </b>
 <br/>
 <br/>
 日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍.html" target="_blank">
  安倍
 </a>
 晋三10月25号抵达北京，展开为期三天的访华行程。这是七年来日本首相首次访华。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 方面对安倍晋三一行高规格接待，安排了三军仪仗队迎接。中共国家主席习近平、总理李克强、全国人大委员长栗战书都将与安倍会晤。
 <br/>
 <br/>
 中日两国因钓鱼岛主权纠纷，近年来关系紧张。中共党媒一向反日，这次却一反常态，纷纷盛赞安倍，高唱中日友好。
 <br/>
 <br/>
 香港《苹果日报》分析说，中共盛情接待安倍的目的，是想拉拢日方，离间美日同盟关系。
 <br/>
 <br/>
 据报导，安倍此行将探讨包括确定习近平明年访日、朝鲜无核化和中美贸易战、知识产权保护、两国互换货币机制、〝海空联络机制〞、军队定期交流等一系列问题。
 <br/>
 <br/>
 海外舆论普遍预期，在两国过去十多年积怨颇深、中美贸易战如火如荼的背景下，安倍此行恐怕很难有突破性进展。
 <br/>
 <br/>
 <b>
  大闹
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国保守党年会.html" target="_blank">
   英国保守党年会
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视女记者.html" target="_blank">
   央视女记者
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被起诉.html" target="_blank">
   被起诉
  </a>
 </b>
 <br/>
 <br/>
 英国伯明翰警方10月24号向香港自由新闻证实，上个月大闹英国保守党年会的中共央视驻英国女记者孔琳琳，已经被正式起诉，她将于11月7号在伯明翰法院出庭受审。
 <br/>
 <br/>
 伯明翰警方表示，48岁的孔琳琳9月30号在伯明翰国际会议中心举行的保守党会议上，涉嫌动手打人。如果被判有罪，她将面临最高六个月的有期徒刑或罚款，或者两者兼施。
 <br/>
 <br/>
 英国保守党今年9月底举行年会，30号与非政府组织〝香港监察〞合办香港问题研讨会。孔琳琳在会议快结束时突然起身大骂与会者〝反华〞、〝汉奸〞，在被要求离场时，又高声叫喊，并扇了会场一位香港义工两巴掌。她随后被警方拘捕，不久获释。
 <br/>
 <br/>
 中共驻英国大使馆事后发表声明，要求会议主办方道歉。
 <br/>
 <br/>
 央视则声称孔琳琳是〝无指控释放〞。
 <br/>
 <br/>
