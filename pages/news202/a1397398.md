### 【禁闻】美下令限制对中国芯片制造商出口
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】1. 美下令限制对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国芯片.html" target="_blank">
  中国芯片
 </a>
 制造商出口
 <br/>
 <br/>
 美国政府日前下令限制本土企业向一家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国芯片.html" target="_blank">
  中国芯片
 </a>
 制造商出售技术和产品。这家被列入商务部出口管制实体名单的中国企业是福建晋华集成电路公司。
 <br/>
 <br/>
 美国商务部10月29号发表的声明说，这家中国制造商新增的存储芯片生产能力，威胁到美国军用系统芯片供应商的生存能力，对美国国家安全利益构成重大风险。而该公司生产能力的扩大，很可能得益于〝源自美国的科技〞。
 <br/>
 <br/>
 BBC中文网引述报导指称，这家由中共政府支持的企业，窃取了美国芯片制造商美光科技（Micron Technology）的知识产权。
 <br/>
 <br/>
 这项出口禁令30号开始生效，今后美国企业如果要向晋华出口零件、软件和科技产品,  必须申请特定许可证。
 <br/>
 <br/>
 早前，美国商务部也曾对中国企业〝中兴通讯〞下达出口禁令，一度令中兴濒临破产。
 <br/>
 <br/>
 2.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国电信.html" target="_blank">
  中国电信
 </a>
 利用北美节点
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_窃取网络通信.html" target="_blank">
  窃取网络通信
 </a>
 <br/>
 <br/>
 来自美国海军战争学院和以色列特拉维夫大学的两名学者，日前发布一项合作研究报告，指称中共国有企业〝中国电信〞，多年来一直利用其在北美的互联网节点设施，窃取西方国家的网络通信内容。这份报告刊登在最新一期的专业期刊《网络军事事务》上。
 <br/>
 <br/>
 报告说，从2000年开始，中国电信就一直其利用部署在西方国家的网络接入点（PoP）设施劫持、窃取互联网通信。他们利用边界网关协议（BGP）允许接入点宣布路由错误的规定，将正常的网络通信〝绕道〞传输到中国。
 <br/>
 <br/>
 报告列举2016年加拿大通往韩国政府网站的网络路由为例说，最短路由应该是加拿大-美国-韩国，而被劫持的路由则通过位于加拿大多伦多的中国电信节点，先后转发到中国电信加利福尼亚节点、中国大陆节点，最终迂回到韩国。而这种绕道传输的方式持续了大约6个月，说明这不可能是临时性网络故障、或者拥堵引起的路由改变。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战效应.html" target="_blank">
  美中贸易战效应
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台商.html" target="_blank">
  台商
 </a>
 回流
 <br/>
 <br/>
 受美中贸易战影响，在中国大陆投资的台湾高科技公司开始返乡。
 <br/>
 <br/>
 台湾科技部长陈良基10月29号在立法院表示，两家精密机械厂和一家信息科技公司已经进驻台湾3个不同的科学园区。
 <br/>
 <br/>
 陈良基说，除了这3家之外，还有另外20家台商正在规划回流，他们进驻科学园区所需的土地，也已经准备就绪，可以立即使用。
