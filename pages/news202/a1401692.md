### 特拉维夫用羊群除草 保护生态多样性
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特拉维夫.html" target="_blank">
  特拉维夫
 </a>
 ，公园里邀请来羊群，帮助解决杂草问题，生态学家表示，这个方法既解决了公园绿地的杂草难题、同时保护了生态的多样性。
 <br/>
 <br/>
 乍一看，放牧的羊群让这个草地小丘看起来像是苏格兰连绵起伏的丘陵，但这里却是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特拉维夫.html" target="_blank">
  特拉维夫
 </a>
 、熙熙攘攘的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 沿海大都市。
 <br/>
 <br/>
 这片绿色山地是约书亚公园（Ganei Yehoshua）的一部分。而格尔（Asaf Gal）是特拉维夫唯一的牧羊人。每周一次，每次三个小时，带着他的两只牧羊犬到这里来，领着一群羊在草地上吃草。
 <br/>
 <br/>
 借助于这些毛茸茸的生物、他们可靠的牧羊人和他的狗的帮助，生态学家正在努力保护这个公园的生态多样性。
 <br/>
 <br/>
 该试点项目是由公园的生态学家沙勒姆（Liav Shalem）发起，他在纽约和巴黎看到了类似的举措，幷决定也在特拉维夫引用。
 <br/>
 <br/>
 沙勒姆说，绵羊吃掉了增长迅速的杂草，但不会吃其他植物，例如海葵和各种各样的花。
 <br/>
 <br/>
 生态学家沙勒姆：〝我们希望看到那些花在春天绽放，牧羊人帮助我们实现这一点，即使在都市也能做到。〞
 <br/>
 <br/>
 放牧绵羊是公园的理想解决方案，因为除草机无法到达公园的某些区域。又因为公园离住宅区很近，所以也不能使用除草剂。
 <br/>
 <br/>
 沙勒姆认为让社区参与这个过程很重要，幷鼓励学校参与这个项目。如果成功，他们甚至可能将试点计划扩大到特拉维夫的其它地区，或许可以为这个城市丛林带来更多的自然景观。
 <br/>
 <br/>
 新唐人记者严枫、雨薇综合报导
