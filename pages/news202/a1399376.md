### 彭斯亚洲行释冷战警告：中共必须改变
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】观众朋友大家好，欢迎您收看全球新闻。日前，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 接受采访时表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统期待与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 达成协议，但北京要做出根本改变，否则美国将使用〝冷战〞战略应对。
 <br/>
 <br/>
 美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 周三在新加坡会晤了部分出席东南亚国家联盟峰会（ASEAN）的国家首脑，包括印度总理莫迪。
 <br/>
 <br/>
 美国副总统 彭斯：〝总理先生，总统（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ）和我感谢您对自由开放的印太区域合作的大力支持。〞
 <br/>
 <br/>
 前一天在从日本前往新加坡的途中，彭斯接受《华盛顿邮报》采访时明确表示，如果中共要避免与美国及其盟友进行全面冷战，就必须有根本和实质的改变。
 <br/>
 <br/>
 国际评估与战略中心军事专家 费学礼：〝与中共的冷战冲突，清晰显示出，美国将积极寻求摧毁中国共产党，也会积极寻求帮助中国人认识到，之前的那个（国民）政府与他们今天不得不忍受的中共独裁政府之间的不同。〞
 <br/>
 <br/>
 时政评论人士 横河：〝这里有两层意思，第一美国准备的最坏打算就是冷战；谈不成的责任在中共、美国不要冷战、但不怕冷战。〞
 <br/>
 <br/>
 彭斯表示川普期待G20与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 能达成协议，但如果北京不拿出实质性方案回应美国在不公平贸易、知识产权盗窃、中国市场准入及公共海域主权等要求，美国将对更多中国产品加征关税。
 <br/>
 <br/>
 不过专家对G20达成协议并不乐观；即使到时中共拿出某种程度的妥协方案，具体实施需要多久仍是问题。
 <br/>
 <br/>
 国际评估与战略中心军事专家 费学礼：〝我认为中共并没有要妥协的意思。美国必须用更多努力提升实力来支持盟友，并带领盟友一道对抗中共，直到中共意识到唯有妥协才是解决之道。〞
 <br/>
 <br/>
 时政评论人士 横河：〝有一些对中共统治至关重要的，中共永远不会真正让步的，这一点必须要看到。〞
 <br/>
 <br/>
 彭斯此次亚州行的一项主要内容，是推广美国的印太战略、来抵制中共〝一带一路〞计画。
 <br/>
 <br/>
 美国副总统 彭斯（2018.11.13）：〝美国寻求合作而非控制。威权主义和侵略在印太区域没有市场。〞
 <br/>
 <br/>
 17日，彭斯将在巴布亚新几内亚参加亚太经合组织峰会，并发表重要演讲。习近平也会出席。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
