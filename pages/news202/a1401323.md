### 川习会将不发声明不见记者 川普：关税战会很长
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 开打4个多月后，美国总统川普（特朗普）与中国国家主席习近平将于12月1日在阿根廷G20f峰会落幕后共进晚餐并展开双边会谈。不过有消息称，这次美中首脑会谈无论是否取得突破，都不会举行联合声明或联合记者会。川普则在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 开幕前夕发推文称，美中之间的关税战要走的路还很长。
 <br/>
 <br/>
 阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 即将举行。据主办方公布的日程，与会各国领导人的峰会将于当地时间11月30日上午10时30分，在布宜诺斯艾利斯的科斯塔萨尔格罗（Costa Salguero）会展中心拉开帷幕。当天的工作议程结束后，各国领导人将携夫人共同出席在科隆大剧院举办的文化活动。
 <br/>
 <br/>
 12月1日上午，领导人的闭门会议继续进行，预计于当地时间下午2时45分结束会议后，现任阿根廷总统毛里西奥．马克里将举行新闻发布会，通报领导人峰会的成果。其他国家的领导人也将举行各自的新闻发布会。
 <br/>
 <br/>
 据此前美中两国政要对外放出的消息来看，川普与习近平将于12月1日下午G20峰会结束后举行双边会谈，而会谈的形式将是〝会晤加晚宴〞。
 <br/>
 <br/>
 香港《南华早报》29日报导称，有多名熟悉安排的消息人士透露，这次川普与习近平在布宜诺斯艾利斯的〝晚宴会晤〞，不论双方是否在谈判中取得了突破，都不会有联合声明或者联合记者会。
 <br/>
 <br/>
 其中有一名消息人士进一步解释说，不发表联合声明并不意味着华盛顿和北京缺乏诚意，或者双方就认定了不会有突破。而是为了给中美各自保留更多的余地〝去按照自己认为合适的方式，处理会晤的结果〞。
 <br/>
 <br/>
 该消息人士还说，华盛顿将用自己的版本去阐述这次会晤，而北京也有可能会发表一份关于这次会晤的简短声明。
 <br/>
 <br/>
 报导称，一名美国官员也证实，这次美中首脑会晤，没有举行联合记者会或发表联合声明的计划，将由中美领导人来单独阐述关于这次会晤的印象。
 <br/>
 <br/>
 值得注意的是，美国总统川普与美东时间29日凌晨4点32分在推特上发出一条推文称，因为对中国（商品）征税，几十亿美元流入了美国政府，与中国的关税战还有〝一段漫长的路要走〞。不想支付关税的话，就来美国建厂。让我们的国家变得比以往任何时候都富裕！
 <br/>
 <br/>
 此前，川普于本月26日接受《华尔街日报》访问时也曾明确表态，现在就想让美国政府延后或暂停提高对中国商品的关税是〝几乎不可能〞的。也就是说，美国政府将会按原计划从2019年1月1日起，将对2000亿美元中国产品加征的关税从10%上调到25%。
 <br/>
 <br/>
 美国国家经济顾问库德洛（Larry Kudlow）也于27日在白宫吹风会上继续向北京施压。他说：〝我非常失望，我知道总统也极其失望，因此周六的晚餐会是一个展开新一页、缔造突破的机会。〞
 <br/>
 <br/>
 库德洛强调，华府与北京在知识产权盗窃、迫使美国公司转让技术等美方不满的问题上，依然存有分歧。他放话说：〝习主席要加紧行动，给我们提供一些新的想法。〞
 <br/>
 <br/>
 对此，英国媒体BBC中文网评论称，川普和库德洛都在G20峰会前作出强硬表态，为即将到来的川习会谈〝泼冷水〞，令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 继续展示出〝鏖战的趋势〞。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
