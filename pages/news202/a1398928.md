### 大江户文化巡礼 传统艺人再现江户风情
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东京.html" target="_blank">
  东京
 </a>
 古代叫做江户，主要是庶民文化。现在也保留着江户风貌的神乐坂，每年都在此时举行文化祭让人们体验江户时代的娱乐。
 <br/>
 <br/>
 黄昏时刻、漫步在神楽坂、倾听民谣，大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江户文化.html" target="_blank">
  江户文化
 </a>
 巡礼，彷佛带我们走入了江户时代。
 <br/>
 <br/>
 〝庵呗〞民谣是由日本古乐器三味线、笛子、鼓伴奏，吟唱的表演形式伴随日本独特的调子，让人回味不绝。
 <br/>
 <br/>
 城端曳山会馆馆长山下茂树：〝庵呗是江户时代从端呗演变而来，是全日本33个被录入世界无形文化财的文化祭之一。〞
 <br/>
 <br/>
 大江户巡礼，每年11月中旬举办，邀请各界
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统艺人.html" target="_blank">
  传统艺人
 </a>
 再现江户风情。
 <br/>
 <br/>
 新内流艺人鹤贺伊势吉：〝新内流也叫新内富士，是江户时代发展成熟，三味线音乐的一种，以江户为中心，源自净琉璃的弹唱艺能。〞
 <br/>
 <br/>
 法国落语演员尻流复写二：〝作为法国人，在法国以及法语圈用法语演出〝落语〞，希望大家可以喜爱。〞
 <br/>
 <br/>
 此次活动也吸引了众多外国游客。
 <br/>
 <br/>
 外国游客：〝我很喜欢看传统的文化和听音乐，很美。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东京.html" target="_blank">
  东京
 </a>
 都历史文化财团齐藤宽人：〝以充满风情的神乐坂为舞台，希望让大家能体验到传统艺能。〞
 <br/>
 <br/>
 在保留着江户时代格局的神楽坂，江户文化继续传承着。
 <br/>
 <br/>
 新唐人日本记者站报导
</div>
