### 联合国调停 新一轮也门和谈在瑞典举行
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_也门.html" target="_blank">
  也门
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内战.html" target="_blank">
  内战
 </a>
 持续三年多，造成数万人死亡，几百万人处于饥荒边缘。敌对双方在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 斡旋下终于在瑞典重启和谈，但外界对此并不抱很大希望。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_也门.html" target="_blank">
  也门
 </a>
 政府军和叛军胡塞武装在2016年终止对话两年多之后，周四重新坐到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谈判.html" target="_blank">
  谈判
 </a>
 桌前。
 <br/>
 <br/>
 新一轮和谈由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 也门问题特使格里菲思主持，在瑞典首都斯德哥尔摩以北几十公里的一座城堡内举行，预计和谈将持续约一周。
 <br/>
 <br/>
 胡塞代表团一行6人周二飞抵瑞典首都斯德哥尔摩，也门哈迪政府外长亚马尼周三也率领12人代表团从也门出发。
 <br/>
 <br/>
 卡塔尔半岛电视台引述知情人士消息说，联合国会推出一系列建立互信的措施，例如：反叛军胡塞武装控制的荷台达市停火，以沙特和阿联酋为首的支持也门政府的联军也停止对也门各地的空袭；另一边，胡塞武装要停止发射各式火箭以及对沙特、阿联酋发动无人机攻击等等。
 <br/>
 <br/>
 由于双方严重缺乏互信，外界对此次谈判并不抱太高期望，认为双方能如期现身，就已是一大突破。谈判焦点或落在荷台达市停火，因为这里是也门大部分粮食及救援物资的入口。
 <br/>
 <br/>
 联合国也门问题特使 格里菲思：在人道主义方面，联合国正试图做的事情是，荷台达市问题是一个特殊的人道主义问题，它是让也门人民活下去的通道。
 <br/>
 <br/>
 世界卫生组织（WHO）指出，目前已有近1万人在也门
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内战.html" target="_blank">
  内战
 </a>
 中丧生，但人权组织认为实际数字可能是现有的5倍。另据挪威难民理事会称，也门有800万人处于饥荒边缘。这一数字还在快速上升。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
