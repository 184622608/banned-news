### 不满默克尔难民政策 巴伐利亚选举爆冷
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】周日,在德国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴伐利亚.html" target="_blank">
  巴伐利亚
 </a>
 选举中，反难民移民的德国选择党首次进入巴伐利亚州议会，成为本次选举最大赢家。选举结果动摇了德国三党联合执政的根基。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴伐利亚.html" target="_blank">
  巴伐利亚
 </a>
 是德国第二大联邦州。该州执政党〝基社盟〞和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默克尔.html" target="_blank">
  默克尔
 </a>
 的〝基民盟〞是姐妹党。两党组成了联盟党，并与另一个〝社民党〞组成三党联合政府。
 <br/>
 <br/>
 周日选举中，三党中的两党遭遇失败。
 <br/>
 <br/>
 基社盟只得到37.2%的选票，比2013年选举减少10%，不得不和其他政党联合执政。而60年多来，基社盟在巴伐利亚几乎一直是单独执政。
 <br/>
 <br/>
 〝社民党〞得票率遭腰斩，只有9.7％。
 <br/>
 <br/>
 两党选举失败给了饱受批评的联合政府又一记重击。
 <br/>
 <br/>
 相应地，反移民难民的德国选择党首次在该州参选就斩获10.2%的选票，引发〝政坛地震〞，再次彰显民众对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民政策.html" target="_blank">
  难民政策
 </a>
 的不满。
 <br/>
 <br/>
 同时，绿党票数翻倍，以17.5%跃居第二大党。
 <br/>
 <br/>
 默克尔周一表示，她愿意从选举结果中吸取教训。
 <br/>
 <br/>
 德国总理默克尔：〝作为大联合政府的总理，我必须更加努力保证重新赢得（民众）信任。〞
 <br/>
 <br/>
 而〝基社盟〞虽遭遇失败，但还会以该州第一大党的身分组阁。该党党魁兼德国内政部长泽霍费尔并在周一表示，还会支持默克尔联合政府。
 <br/>
 <br/>
 基社盟领袖泽霍费尔：〝我们会尽一切努力，确保联合政府能够继续稳定开展工作。〞
 <br/>
 <br/>
 连线本台欧洲记者王羽音：〝巴伐利亚是难民和非法移民进入德国的门户。因为默克尔打开大门,所以一共有100多万难民涌入德国，主要是穆斯林。而巴伐利亚是德国基督教传统最浓厚的州,民众对政府的难民政策非常不满。前几个月,基社盟和基民盟之间因为难民问题而矛盾激化,政府闹分裂也让选票流失。默克尔这次已经和党内重要人达成共识,她不会在党内遭到弹劾。但两周后的黑森州大选,如果出现类似结果,默克尔可能面临下台危机。〞
 <br/>
 <br/>
 新唐人电视综合报导
</div>
