------------------------

 玛格丽特二世亲自设计服装和舞台布景，并出席首秀。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丹麦女王.html" target="_blank">
  丹麦女王
 </a>
 玛格丽特二世的绘画作品也在哥本哈根的博物馆展出，从1980年代就与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蒂沃利芭蕾舞团.html" target="_blank">
  蒂沃利芭蕾舞团
 </a>
 合作、设计舞台布景和服装。
 <br/>
 <br/>
 多年来女王已经成为一名经验丰富的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_设计师.html" target="_blank">
  设计师
 </a>
 ，她与蒂沃利芭蕾舞团合作了七个剧目。而最​​著名的是备受好评的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_《胡桃夹子》.html" target="_blank">
  《胡桃夹子》
 </a>
 ，涉及的服装超过100件。2012年演出后，女王对服装和舞台布景做了修改。今年再度上演。
 <br/>
 <br/>
 蒂沃利芭蕾舞团的演员来自世界各地，他们感到和女王相处完全没有压力。
