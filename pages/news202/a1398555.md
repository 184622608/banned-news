### 印尼狮航坠机　波音对〝737 MAX客机〞发警告
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】根据印尼媒体报导，狮航客机失事的初步调查显示，新式737MAX型客机，飞行控制系统读数错误，可导致飞机以大角度俯冲坠落。生产这款飞机的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波音公司.html" target="_blank">
  波音公司
 </a>
 ，已向使用这种型号客机的航空公司发出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安全警告.html" target="_blank">
  安全警告
 </a>
 ，并提醒飞行员需以现行程序处理问题。
 <br/>
 <br/>
 客机在判断是否处于失速状况的重要依据，是监测飞行角度的〝攻角〞感应器，〝攻角〞是机翼方向跟空气流向的夹角，攸关机翼的升力。
 <br/>
 <br/>
 印尼国家运输安全委员会日前指出，狮航客机出事时机头突然向下，坠海一刻的速度甚至达每小时965公里。
 <br/>
 <br/>
 狮航客机失事前最后4次飞行时，空速表都发生问题。但目前还无法判定空速表问题是否与〝攻角〞有关。专家认为，训练飞行员处理问题或许更加重要。
 <br/>
 <br/>
 目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波音公司.html" target="_blank">
  波音公司
 </a>
 已在全球售出219架波音737MAX，其中美国西南航空拥有23架、加拿大航空拥有18架、美国航空15架、印尼狮子航空13架。
 <br/>
 <br/>
 新唐人记者陈依心、天羽综合报导
</div>
