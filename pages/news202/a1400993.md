### 俄罗斯曝光被捕士兵 乌克兰议会批准戒严
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 冲撞扣押三艘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 海军船隻与23名水手事件升级，引发国际关注，要求双方克制，敦促俄罗斯停止挑衅。周二，俄罗斯在电视上曝光
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰士.html" target="_blank">
  乌克兰士
 </a>
 兵，遭到乌克兰抗议。
 <br/>
 <br/>
 BBC报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 在电视上曝光
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 士兵，强迫他们说一些反对自己国家的话，遭到乌克兰抗议。
 <br/>
 <br/>
 乌克兰海军司令说，这些士兵是在威逼下被迫说谎。西方国家谴责俄罗斯使用武力，要求双方克制。
 <br/>
 <br/>
 德国外长海科．马斯：〝总理默克尔与俄罗斯总统普京和乌克兰总统波罗申科再次通话，要求缓和局势。我们也不会放弃多年来的努力。〞
 <br/>
 <br/>
 奥地利外长克奈斯尔表示，欧盟可能对俄罗斯采取进一步制裁，欧盟将在下个月10号开会，结果如何将取决于事件进展。
 <br/>
 <br/>
 奥地利外长克奈斯尔：〝我们非常关注，因为这是一场攻击，这个争端有可能会演变得更危险。〞
 <br/>
 <br/>
 美国总统川普周一表示，他不想见到俄罗斯与乌克兰危机升级。
 <br/>
 <br/>
 美国总统川普：〝我们不喜欢发生的这些事，无论哪一方面，我们都不喜欢。希望他们妥善处理好。我知道欧洲也不愿看到现在的局面，他们也在努力，我们一起在努力。〞
 <br/>
 <br/>
 乌克兰议会经过5个小时的辩论，通过总统波罗申科的提案，在十个省份实施30天
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戒严.html" target="_blank">
  戒严
 </a>
 ，应对俄罗斯的动作。
 <br/>
 <br/>
 这十个省份都是与俄罗斯、白俄罗斯和摩尔多瓦接壤的地区。
 <br/>
 <br/>
 俄罗斯周日扣押了三艘乌克兰海军船隻，并强行使用武器命令船隻停止，造成船员受伤，23人被捕。
 <br/>
 <br/>
 乌克兰要求俄罗斯立即释放船隻与船员，但俄罗斯置之不理。周二
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克里米亚.html" target="_blank">
  克里米亚
 </a>
 法庭判决扣押这些船员两个月。俄罗斯在2014年并吞了乌克兰领土克里米亚。
 <br/>
 <br/>
 <br/>
 新唐人记者李梅、宇微综合报导
