### 狮航班机坠海第二天 未发现生还者与黑匣子
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】印尼狮航客机坠海事故发生第二天，搜索工作持续进行，尚未发现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生还者.html" target="_blank">
  生还者
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑匣子.html" target="_blank">
  黑匣子
 </a>
 的下落。失事客机乘客家属们心情沉重，祈祷奇迹发生。
 <br/>
 <br/>
 周二，搜索失事狮航客机行动进入第二天。搜索范围扩大到10海里，搜救人员找到更多坠毁班机的残骸碎片，以及皮夹护照等乘客遗物，带回岸边。
 <br/>
 <br/>
 印尼总统佐科・维多多 周二前往雅加达北部丹戎不碌港口视察，从海上带回来的飞机残骸碎片，以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乘客遗骸.html" target="_blank">
  乘客遗骸
 </a>
 、遗物暂时放置在这里。
 <br/>
 <br/>
 狮子航空公司声明表示，24个放置人体遗骸的尸体袋，被送往医院进行鍳定，预计还会有更多。
 <br/>
 <br/>
 家属们心情沉重，祈祷奇迹发生。
 <br/>
 <br/>
 乘客家属HARIS KISWANTO：〝我们仍在祈祷奇迹发生，祈祷我哥哥健在，也许滞留在某地。〞
 <br/>
 <br/>
 乘客家属SUGIHARTO FONG：〝我们希望其他单位，例如警察，可以协助搜寻，赶快找到他，我们会持续祈祷。〞
 <br/>
 <br/>
 目前，搜索重点是找到驾驶舱录音机和飞行数据记录器，也就是两个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑匣子.html" target="_blank">
  黑匣子
 </a>
 ，以帮助确定空难原因。
 <br/>
 <br/>
 狮航JT610班机，在起飞13分钟后坠入海中，机长曾回报技术问题，呼救要求返回机场，不料班机随后迅速坠落大海。
 <br/>
 <br/>
 截至目前，全机189人，没有发现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生还者.html" target="_blank">
  生还者
 </a>
 。
 <br/>
 <br/>
 新唐人记者李梅、明玉综合报导
