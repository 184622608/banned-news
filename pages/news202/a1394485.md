### 蓬佩奥访平壤之际 朝副外长抵莫斯科与中俄会谈
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 展开亚洲巡访期间，7日第4次访问朝鲜与其领导人金正恩商讨无核化及为二次川金会铺路之际，俄罗斯媒体报导，朝鲜副外长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔善姬.html" target="_blank">
  崔善姬
 </a>
 6日已抵达莫斯科，将与中共及俄罗斯代表就朝鲜半岛议题召开三方会谈。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔善姬.html" target="_blank">
  崔善姬
 </a>
 （Choe Son Hui）此前曾作为美朝核谈判的朝方代表，上周四(4日)访问北京，与中共外交部副部长孔铉佑对话。据朝鲜官媒确认，双方表示会积极寻求方法，加强中朝关系。
 <br/>
 <br/>
 随后，崔善姬6日抵达莫斯科。据俄罗斯驻朝鲜大使馆职员透露，崔善姬8日将与俄罗斯官员会面，隔天再与中俄官员作三方会谈。
 <br/>
 <br/>
 有分析指，崔善姬将争取中俄对平壤减免制裁的支持，同时反映朝鲜在无核化谈判上，会寻求有传统邦交的国家合作。也有分析认为，崔善姬访华很可能是在与美国进行核谈判之前与中方交换意见。
 <br/>
 <br/>
 韩联社引述北京一名消息人士报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 访朝前，崔善姬访华可能是为进行事先协调，也可能是向中共介绍联合国大会期间美朝磋商情况，通过崔善姬今后的移动路线可以更加明确其访华意图。
 <br/>
 <br/>
 据了解，美国国务卿蓬佩奥（Mike Pompeo）7日访问朝鲜时，由美国国务院对朝政策特别代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_史蒂夫．毕根.html" target="_blank">
  史蒂夫．毕根
 </a>
 （Stephen Biegun）陪同。据信崔善姬是毕根的谈判对手。
