### 涉一马弊案 大马对华裔富商刘特佐等5人发逮捕令
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】涉及马来西亚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一马.html" target="_blank">
  一马
 </a>
 发展公司（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_1MDB.html" target="_blank">
  1MDB
 </a>
 ）弊案的金融逃犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘特佐.html" target="_blank">
  刘特佐
 </a>
 和另4名被告，警方今(5日)再提出新的刑事指控，并对外发出逮捕令，如果刘特佐等人的行踪在其他国家被发现，政府将展开引渡程序。
 <br/>
 <br/>
 除了华裔富商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘特佐.html" target="_blank">
  刘特佐
 </a>
 （Low Teak Jho），另4名涉案者分别是刘特佐生意伙伴陈金龙（Eric Tan Kim Loong，音译）、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_1MDB.html" target="_blank">
  1MDB
 </a>
 前法律顾问卢爱璇（Jasmine Loo Ai Swan，音译）、前业务发展董事唐敬志（Casey Tang Keng Chee，音译）、前财务执行董事倪崇兴（Terence Geh Choh Heng，音译）。
 <br/>
 <br/>
 路透社报导，马来西亚全国总警长弗兹（Mohamad Fuzi Harun）发表声明表示，刘特佐与这4人，因导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一马.html" target="_blank">
  一马
 </a>
 发展公司损失高达11亿700万美元，4日指控他们涉及13项洗黑钱与背信。
 <br/>
 <br/>
 警方表示，5人目前均已逃出大马，已对他们发出逮捕令。〝一旦发现他们出现在任何其他国家，警方和检察总署将采取行动，要求把他们引渡回大马，上庭面控。〞
 <br/>
 <br/>
 现年37岁的刘特佐因涉及在2009年至2011年间将10亿300万美元转入瑞士银行而再面临5项洗黑钱罪名；刘特佐和生意伙伴陈金龙也在新加坡一个银行账户收取1亿2600万美元而再面对另两项洗黑钱控状。刘特佐今年8月已面临8项
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗钱.html" target="_blank">
  洗钱
 </a>
 罪名。
