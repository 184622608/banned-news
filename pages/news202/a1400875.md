### 美驻联合国大使：俄扣乌船舰系侵犯主权
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】联合国周一召开紧急会议讨论俄罗斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扣留乌克兰船舰.html" target="_blank">
  扣留乌克兰船舰
 </a>
 一事。美国驻联合国大使妮基．黑莉（Nikki Haley）在会议上直言俄此举是侵犯乌克兰领土主权的行为，破坏了美俄之间建立正常关系的努力。
 <br/>
 <br/>
 据台湾《中央社》报导，联合国安理会26日针对俄罗斯日前在亚速海（Azov Sea）扣留乌克兰船艇事件召开紧急会议。美国驻联合国大使妮基．黑莉在会议上发言时表示，俄罗斯动用武力强行扣留乌克兰船隻是〝对乌克兰领土主权的无耻侵犯〞，她还抨击俄罗斯此举再次〝鲁莽升高〞了该地区的紧张局势。
 <br/>
 <br/>
 妮基．黑莉在会中还指出：虽然美国对于与俄罗斯建立正常关系一事深表欢迎，但俄罗斯当局做出的类似这起扣船事件的〝不法行动〞，将使得美俄建立正常关系的目标无法实现。
 <br/>
 <br/>
 黑莉没有提及美方会否对俄罗斯采取进一步的制裁措施，而是呼吁俄罗斯当局用行动给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克里米亚问题.html" target="_blank">
  克里米亚问题
 </a>
 所引发的紧张情势降温。
 <br/>
 <br/>

 </center>
 <br/>
 <br/>
 本月25日，俄罗斯海军在亚速海（Azov Sea）扣留乌克兰船艇时提出的理由是，乌克兰船隻经过的克里米亚（Crimea）沿海水域是俄罗斯的领海。
 <br/>
 <br/>
 事实上，克里米亚原本属于乌克兰，虽然俄罗斯2014年出兵并吞了克里米亚地区，但这一行动一直被国际社会视为侵略，并未获得国际社会的承认。
 <br/>
 <br/>
