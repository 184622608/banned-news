### 乌克兰要求俄罗斯立即释放船隻与船员
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 紧张局势升级。周一，乌克兰议会将决定是否宣布戒严。俄罗斯周日扣押了三艘乌克兰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海军舰艇.html" target="_blank">
  海军舰艇
 </a>
 ，造成6名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_船员.html" target="_blank">
  船员
 </a>
 受伤，乌克兰要求俄罗斯立即释放船隻与船员。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 总统波罗申科周一紧急召开国家安全和国防委员会会议，要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 立即释放3艘海军船隻和23名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_船员.html" target="_blank">
  船员
 </a>
 ，作为缓和两国危机和区域紧张的第一步。
 <br/>
 <br/>
 乌克兰总统波罗申科：〝我要求俄罗斯联邦领导层，立即释放乌克兰军人，（俄罗斯）违反国际法，残酷拘留乌克兰军人，他们生死未卜。〞
 <br/>
 <br/>
 波罗申科要求俄罗斯即刻做出回应。同时，乌克兰议会周一下午将决定是否宣布戒严。
 <br/>
 <br/>
 俄罗斯周日扣押了三艘乌克兰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海军舰艇.html" target="_blank">
  海军舰艇
 </a>
 ，并强行使用武器命令船隻停止，造成6名船员受伤。
 <br/>
 <br/>
 乌克兰表示，俄罗斯冲撞其中一艘船是侵略行为，而俄罗斯称这些船隻非法进入其水域，是在挑衅。
 <br/>
 <br/>
 俄罗斯外交部长拉夫罗夫：〝在昨天的事件中，这是明确的挑衅。〞
 <br/>
 <br/>
 事发时，俄罗斯紧急向该海域派遣两架战斗机和两架直升机。
 <br/>
 <br/>
 这三艘船在克里米亚海岸航行被扣押，俄罗斯2014年并吞了克里米亚。根据2003年莫斯科和基辅政府签订的条约，这个区域是共享领海，两国可以自由航行。
 <br/>
 <br/>
 周一，欧洲理事会主席图斯克推文说：〝我谴责俄罗斯使用武力，俄罗斯当局必须交回乌克兰船员和船隻，克制进一步挑衅。欧洲将团结一致支持乌克兰。〞
 <br/>
 <br/>
 欧盟执委会发言人也表示，事态发展不可接受。
 <br/>
 <br/>
 欧盟执委会发言人柯契姜琪克：〝这些事态发展是不可接受的，我们要求俄罗斯能立即释放船隻及其船员，并确保对军人提供医疗援助。〞
 <br/>
 <br/>
 美国驻联合国大使黑利发推文说，周一上午，安理会将就此事件召开紧急会议。
 <br/>
 <br/>
 同时，在俄罗斯驻基辅大使馆外，大约150人抗议俄罗斯向乌克兰海军舰艇开火并打伤6名水手。
 <br/>
 <br/>
 新唐人记者李梅、田瑞综合报导
</div>
