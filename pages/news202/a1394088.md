 大纪元时报
 <br/>
 周三，美国参院通过《2018年善用投资促进发展法》。该法案将海外私人投资公司与美国国际发展署两个部门合并为〝美国国际发展金融公司〞，并计划向全球贫困地区提供600亿美元资助。法案发起人表示，这是对中共一带一路的反击。
 <br/>
 <br/>
 台湾中央社
 <br/>
 本周，美国〝CrowdStrike〞和〝火眼〞两家网路安全公司的专家揭露，随美中贸易战升温，中共骇客活动激增，包括与中共当局有关的骇客集团〝云端跳跃〞。美国国土安全部发出技术警告，指云端跳跃从事网路间谍活动并窃取智慧财产。
 <br/>
 <br/>
 台湾青年日报
 <br/>
 中共当局的计画生育政策，催生了庞大鉴定胎儿性别的黑市。除了B超，一种〝寄血验子〞的鉴定方式也流行起来，导致非法堕胎。不少人喜欢男胎，使女婴减少，进而造成中国社会性别失衡。
