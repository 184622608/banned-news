### 日本旅游胜地箱根 展出中土艺术珍品
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】中土文化，对周边的邻国产生了深远的影响。在日本知名的旅游地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_箱根.html" target="_blank">
  箱根
 </a>
 ，一家美术馆就展出了难得一见的珍贵中国传统艺术雕塑。
 <br/>
 <br/>
 成立于1988年，位于日本富士山
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_箱根.html" target="_blank">
  箱根
 </a>
 国立公园的成川美术馆，所藏日本画展品，堪称日本第一。
 <br/>
 <br/>
 邻近芦之湖畔，得天独厚的地理位置，使其可以将巍峨的富士山与碧水波光的芦之湖收入同一画框，堪称箱根绝景。
 <br/>
 <br/>
 美术馆展示来自中国的翡翠、象牙等艺术作品。
 <br/>
 <br/>
 翡翠雕塑〝吉祥家园〞采用了透雕技法，用整块翡翠雕琢出锦鸡成双、喜鹊成对、丝瓜吐青、牡丹透翠等各种惟妙惟肖的造型。
 <br/>
 <br/>
 这件〝华夏文明〞则将上古至今的历史长河凝缩至一颗象牙上，描绘出开天辟地、大禹治水、秦统天下、丝绸之路、成吉思汗等中华历史画卷。细致逼真，令人百看不厌。
 <br/>
 <br/>
 成川美术馆经理小原胜好：我们学过中国的历史。这个象牙上雕刻得成吉思汗，秦始皇等等，这些东西我们确实都学过。大家都很赞赏。因为是邻国嘛，很多人都非常感兴趣。
 <br/>
 <br/>
 鸡血石雕刻〝万龙戏珠〞，以及玉石雕刻多层塔型玉薰炉等展品，使得中土艺术之光，在东瀛熠熠生辉。
 <br/>
 <br/>
 新唐人日本记者站报导
</div>
