### 川习通话 美中贸易摩擦或有缓解希望
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】观众朋友大家好，欢迎收看新唐人全球新闻。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易摩擦.html" target="_blank">
  美中贸易摩擦
 </a>
 持续，日前，中共当局承认
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 下行压力加大，在此背景下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 周四的通话受到各方关注。白宫首席经济顾问认为，美中贸易关系将有改善的可能。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周四在推文中说：我与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 主席进行了一段很长也很好的通话。我们讨论了许多话题，贸易占很大比重。这些讨论对我们将在阿根廷G-20峰会的会面有很大好处。关于朝鲜的讨论也很好。
 <br/>
 <br/>
 白宫首席经济顾问 库德洛：〝我不想说太多，但（美中贸易关系）有可能会有些解冻。〞
 <br/>
 <br/>
 美国白宫首席经济顾问库德洛周四表示，如果中国按照国际惯例行事，美国加征的关税可以去除。他还说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 下行压力大，经济发展正在放缓。
 <br/>
 <br/>
 白宫首席经济顾问 库德洛：〝我认为，他们（中国）经济正在疲软。我们（美国）的经济景气。〞
 <br/>
 <br/>
 周一，川普周接受福克斯新闻采访时说，美中将可能达成一项贸易协议，但若无法达成，美国已准备好第三轮对所有剩余的进口自中国的商品加征关税，价值2670亿美元。
 <br/>
 <br/>
 川普称，在美中贸易战，中国受到的打击和压力更大，因为中国对美国的出口要大于从美国的进口。目前中国经济放缓，而美国经济景气。
 <br/>
 <br/>
 新唐人记者王羽音综合报道
