### 【禁闻】美拟限制对华监控科技出口
------------------------

 【新唐人2018年10月03日讯】
 <b>

   法轮功
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港集会游行.html" target="_blank">
   香港集会游行
  </a>
  各界声援
 </b>
 <br/>
 <br/>
 10月1号，香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员举行集会游行，呼吁制止中共迫害，法办迫害元凶，同时声援中国人退出中共党、团、队组织。
 <br/>
 <br/>
 当天的集会，香港前立法会议员梁国雄、支联会副主席蔡耀昌、前立法局议员冯智活、香港前区议员徐百弟等政界名人到场支持，并在会上发言声援法轮功反迫害，谴责中共暴政，呼吁各界共同努力，捍卫中国的人权自由。
 <br/>
 <br/>
 集会结束后，近千人的游行队伍，经九龙区繁华闹市，历时三个小时，行进到终点尖沙咀码头。
 <br/>
 游行队伍由天国乐团、〝除恶扬善  天下太平〞、〝法轮大法  和平之路〞、〝天灭中共  三退是福〞等方阵组成。吸引了众多民众观看，不少大陆游客当场向三退义工表明要退出中共党、团、队。
 <br/>
 <br/>
 <b>
  美拟限制对华
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监控科技.html" target="_blank">
   监控科技
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出口.html" target="_blank">
   出口
  </a>
 </b>
 <br/>
 <br/>
 随着中共在新疆建〝再教育营〞拘押维吾尔人的消息被西方媒体广泛报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 政府正在商讨,  准备增加对华技术的出口限制，并对相关的中共官员、企业进行制裁。
 <br/>
 <br/>
 美国商务部长罗斯在写给国会的一封信中表示，将对中国的个人和企业进行评估，可能在未来几周内宣布出口政策的改变，严格限制那些可能被中共政府用于监视技术的出口。
 <br/>
 <br/>
 美国参议员鲁比奥和史密斯上个月致信罗斯，要求美国政府针对中共迫害少数民族采取制裁手段。
 <br/>
 <br/>
 制裁措施一旦实现，将是川普政府首次因人权问题制裁中共。
 <br/>
 <br/>
