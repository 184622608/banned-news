 周五，美国纪念
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珍珠港事件.html" target="_blank">
  珍珠港事件
 </a>
 77周年。1941年12月7日，日本袭击了位于夏威夷的珍珠港军事基地，夺走了2390名美国人的生命。随后美国向日本宣战。截至2016年，只有不到200名当时在夏威夷美军基地的军人在世。总统川普当天也发推文，公布了他与第一夫人敬献花圈的照片。
 <br/>
 <br/>
 一股横跨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中南部.html" target="_blank">
  美国中南部
 </a>
 并将抵达东岸的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冬季风暴.html" target="_blank">
  冬季风暴
 </a>
 ，周三在加州成形。国家气象局表示，风暴在周五加剧，未来几天将有十几个州有降雪，超过20个州将遭受强风和降水，给数百万人带来不便。
 <br/>
 <br/>
