### 动容！骑士安静牵车 哀悼狮航罹难者
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】印尼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮航.html" target="_blank">
  狮航
 </a>
 JT 610班机失事，造成机上189人全数丧生，经过多日搜寻乘客遗体及比对身分，首位确认身分的罹难者是24岁的公务员黛薇。为了向她的家人致哀，经过她住家附近的摩托车骑士纷纷下车牵行车辆，安静地经过黛薇老家。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮航.html" target="_blank">
  狮航
 </a>
 （Lion Air）这架波音737 MAX 8型客机10月29日上午6时20分从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雅加达.html" target="_blank">
  雅加达
 </a>
 飞往邦加槟港（Pangkal Pinang），途中机师曾要求调头返回原机场，但不久便坠入爪哇海（Java Sea）。机上乘客恐已全部罹难。
 <br/>
 <br/>
 经过多日搜寻乘客遗体及比对身分，印尼警方近日公布首名罹难者为24岁的女性乘客黛薇（Jannatun Shintya Dewi），她在东爪哇苏达约县（Sidoarjo）任职印尼能源部的公务员。
 <br/>
 <br/>
 印尼新闻网站雅加达椰子网（Coconuts Jakarta）报导，黛薇老家前的街上布满亲友及民众致哀的花牌，当地民众除了致赠花牌表示哀悼之外，许多摩托车骑士经过黛薇老家时，纷纷下车，用牵行的方式，安静地经过黛薇老家，贴心的举动也感动许多人。
 <br/>
 <br/>
 <br/>
 <br/>
 拍下影片上传至网路社群的网友巴哈那（Bahana）在影片中打上字幕表示，他询问友人为何黛薇老家一带的摩托车骑士都要这么做，答案是：〝这是一种致哀，也是对屋主（黛薇家人）的遭遇表示感同身受。〞
 <br/>
 <br/>
 巴哈那对这样的景象表示震惊，因为那是发生在早晨、人们正赶着去上班的时候。当时整条街上没有摩托车引擎刺耳的声音，而是出奇安静。
 <br/>
 <br/>
 他说，这样的景象让他为印尼感到骄傲，印尼对他人表示同情及同理的高尚文化或许在某些地方已经式微，但在其他地方，这样的价值仍坚定维持下去。
 <br/>
 <br/>
 当地居民苏鲁（Suruh）表示，这是当地盛行多年、在世代间流传的传统。在村庄里，民众经过丧家时，必须下车（摩托车），用牵行的方式经过丧家。
 <br/>
 <br/>
