### 贸易战川习本周首度交手 生产线拟撤出中国
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】首先来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二十国集团峰会.html" target="_blank">
  二十国集团峰会
 </a>
 周五即将登场，美国总统川普与中国领导人习近平12月1日将在阿根廷首都布宜诺斯艾利斯共进晚餐。这是中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 以来，双方首度会面，两国都非常关注，一些在中国的生产线已经做出部分或全面撤出的打算。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 以来，美国的进口商与中国的出口商由于关税成本增加，必须寻找替代方案。
 <br/>
 <br/>
 中国制造商表示，一些美国的进口商舍弃自中国进口货品，改从其他国家进口。位于南京的石英实体面材工厂LEADA SOLID SURFACE正面临客户流失的困境。
 <br/>
 <br/>
 LEADA SOLID SURFACE总经理KIN YOU：〝我们的客户说，‘抱歉，我们得想办法购买其他国家的石英石，如墨西哥、菲律宾或欧洲。’〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二十国集团峰会.html" target="_blank">
  二十国集团峰会
 </a>
 本周五召开，川普与习近平将在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 之后首度会面，令外界高度关注。
 <br/>
 <br/>
 制造商表示，贸易战开打至今影响甚钜，很难扭转局面，一些供应商已做出将部分或全部生产线撤出中国的打算。
 <br/>
 <br/>
 Tortuga背包执行长普罗塔：〝很多中国工厂都在考虑，在东南亚其他地方开设另一家工厂，也许将所有业务转移到那里，或是将成品送到那里，这样他们就可以算是在那里制造的。〞
 <br/>
 <br/>
 普罗塔说，关税成本不断增加，到明年会增加到42.6%，公司考虑到越南设厂。
 <br/>
 <br/>
 Tortuga背包执行长普罗塔：〝我们正在寻找一些替代方案，最有可能是越南。〞
 <br/>
 <br/>
 美中贸易战开打以来，北京一直未能就知识产权和不公平贸易行为等问题，提出解决办法。
 <br/>
 <br/>
 美国随时准备对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国商品.html" target="_blank">
  中国商品
 </a>
 征收更多关税。除了对所有剩余的中国进口产品价值约2670亿美元征收关税，并计画从明年1月1日起，将价值2000亿美元的中国进口商品关税税率从10％增加到25％。
 <br/>
 <br/>
 新唐人记者李梅综合报导
</div>
