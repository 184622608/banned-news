### 川普政府叫停非法入境难民庇护申请
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】周四，美国司法部和国土安全部也宣布，禁止〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法入境.html" target="_blank">
  非法入境
 </a>
 者〞申请庇护。不过，规定没有追溯力，已入境的非法移民不受影响。
 <br/>
 <br/>
 美国司法部代理部长惠特克和国土安全部部长尼尔森，周四共同发布声明。
 <br/>
 <br/>
 声明说：〝今天，我们用国会授予我们的权力，禁止违反总统暂停入境或其他限制令的外国人寻求政治庇护。〞
 <br/>
 <br/>
 声明引述美国《移民和国籍法》相关条款说：〝如果总统发现任何外国人或任何类别的外国人进入美国，或不利于美国利益，可以在他认为必要的期间，宣布暂停所这些人作为移民或非移民入境，或对外国人入境施加他认为适当的限制。〞
 <br/>
 <br/>
 声明说：〝我们的庇护系统不堪重负，无法律依据的庇护申请太多了，这些申请让国家资源负担沉重，使美国政府无法迅速处理那些真正应获得庇护资格人士的申请。〞
 <br/>
 <br/>
 庇护是美国政府对外国人因种族、宗教、国籍、特定社会群体或政治见解，逃避本国迫害而提供的救助。
 <br/>
 <br/>
 据报，截至2018年1月21日，美国移民局已经积压了31.1万庇护案件，移民局局长西斯纳用〝灾难〞一词，形容案件积压程度。
 <br/>
 <br/>
 外界认为，美国政府的庇护新规，主要针对来自洪都拉斯等中美洲国家大篷车难民。目前，大篷车难民人数已经从高峰期的7千人降到4千人。这是今年第二波难民潮，第一波发生在春季。
 <br/>
 <br/>
 新唐人记者王羽音、张琪综合报导
