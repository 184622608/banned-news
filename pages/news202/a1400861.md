### 【禁闻】中共驻巴领馆遭袭 一带一路遇新抵制
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 驻巴基斯坦领事馆日前遭到武装袭击。外界认为，巴国境内反对中巴经济走廊的情绪已经升级到了一个新的层面。
 <br/>
 <br/>
 11月23号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 驻巴基斯坦卡拉奇领事馆遭到武装袭击，造成2名巴方警卫人员和2名平民死亡。3名袭击者也被击毙。
 <br/>
 <br/>
 事发不久后，巴基斯坦〝俾路支解放军〞（Baluch Liberation Army）宣称对此次袭击事件负责。
 <br/>
 <br/>
 〝俾路支解放军〞是俾路支省的一个武装分离组织，他们反对中共在俾路支省的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 投资项目。
 <br/>
 <br/>
 俾路支省位于阿富汗和伊朗边境，拥有丰富的矿产和天然气资源。作为中共一带一路旗舰项目的中巴经济走廊，主要工程瓜达尔港口项目正是在俾路支省。
 <br/>
 <br/>
 〝俾路支解放军〞发言人贾德．俾路支(Jiand Baloch)告诉〝路透社〞,  他们袭击中共驻卡拉奇领事馆，是因为中共正在占有他们的资源。
 <br/>
 <br/>
 中华经济研究院特约研究员吴惠林指出，中共的掠夺不仅表现在攫取他国的资源上，连人工都从中国带去，而一旦被投资国还不起中共的债务，就会被吃掉。
 <br/>
 <br/>
 中华经济研究院特约研究员吴惠林：〝所以在我看来就是要吃掉人家，整个这样下去的话，结果都被它吃掉了，悄悄的占领全世界，这样下去的话其实对大家都不好。〞
 <br/>
 <br/>
 此次袭击事件并非俾解针对中国人发动的第一起袭击，而俾解也不是巴基斯坦唯一一支分离组织武装。早在中巴经济走廊项目开启之前，就至少有7名中国援巴工程人员死于俾路支解放军之手。
 <br/>
 <br/>
 2013年，中巴经济走廊项目开启后，该项目在俾路支省的一些工程也成为俾路支解放军袭击的目标。到2016年9月，至少44名在中巴经济走廊项目中工作的巴基斯坦工人因恐怖袭击丧生。
 <br/>
 <br/>
 2016年5月，发生在卡拉奇的一起爆炸事件炸伤一名中国工人。
 <br/>
 <br/>
 一年后，一颗炸弹在中国车队经过卡西姆港之后被引爆，〝信德革命军〞（Sindhudesh Revolutionary Army）随后宣称对袭击事件负责，并警告中国人，不要参与任何违反信德人利益的项目。
 <br/>
 <br/>
 去年5月，2名中国青年在俾路支省首府被绑架后遇害。
 <br/>
 <br/>
 今年8月，一名俾解自杀式炸弹袭击者，袭击了一辆载有中国采矿工人的公共汽车，造成5人受伤。
 <br/>
 <br/>
 引爆自杀炸弹的是俾解领导人亚斯拉姆．俾路支的儿子瑞罕（Rehan Baloch）。他在袭击前准备的视频声明中说：〝在中共的帮助下，数千名俾路支人遭到巴基斯坦政府屠杀，村庄遭到洗劫和烧毁。我要以此行动，让中国及其人民意识到，如果没有俾路支人民的同意，任何试图干预俾路支问题的人都会面对俾路支民族的愤怒。〞
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝这些例子说明中共在向外扩张的时候，第一个它没有顾及到对当地权益的伤害带来的后果，第二个为了攫取它的利益，它不顾老百姓的安全，把他们送到这样一些极其危险的、各种矛盾极其复杂的高危地区去。〞
 <br/>
 <br/>
 伦敦皇家联合国防和安全研究所的国际安全研究主任潘图奇说，这次对中领馆的攻击反映出中国日益成为俾解关注的焦点，类似的问题在一带一路沿线的其它国家也会出现。
 <br/>
 <br/>
 唐靖远：〝其实一带一路的本质，它是要以经济为纽带形成一个以中共为核心的政经体系，来和以美国为首的自由贸易体系相对抗，甚至于幻想能取而代之的，所以它带来的结果当然只能是坏的，遭受到沿线民众的不同程度的反抗，甚至是这种暴力的反对，也就不奇怪了。〞
 <br/>
 <br/>
 旅美时事评论员唐靖远说，由中共的邪恶本质决定了一带一路必然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_输出腐败.html" target="_blank">
  输出腐败
 </a>
 ，导致当地民众反对，同时中共的暴力特征也必然会遭遇暴力抵制。
 <br/>
 <br/>
 采访编辑/刘惠 后制/王明宇
