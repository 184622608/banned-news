### 拒绝中共要求 马来西亚释放11名维族人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】泰国去年11月发生25名非法入境的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维吾尔族人.html" target="_blank">
  维吾尔族人
 </a>
 集体越狱事件，部分人逃到马来西亚后再被捕。中共此前一直要求将这些维族人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遣返.html" target="_blank">
  遣返
 </a>
 ，但近日，马来西亚的检察官在倾听维族人的呼吁后，基于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人道.html" target="_blank">
  人道
 </a>
 的理由释放了他们，并让他们前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 。
 <br/>
 <br/>
 综合媒体报导，25名非法入境泰国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维吾尔族人.html" target="_blank">
  维吾尔族人
 </a>
 ，遭逮捕后，去年11月凿破拘留中心墙壁，以棉被结索成功逃走，部分人逃到马来西亚后又被捕。
 <br/>
 <br/>
 中共获知后，施压马来西亚政府要求将人送返北京。
 <br/>
 <br/>
 据了解，马来西亚当局此前曾把维族难民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遣返.html" target="_blank">
  遣返
 </a>
 中国，今年2月，在亲中前首相纳吉布（Najib Razak）政府期间，曾考虑将这11名维族人引渡到中国，但改朝换代后，当局对维族人的政策已有所改变。
 <br/>
 <br/>
 这些维族人的辩护律师以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人道.html" target="_blank">
  人道
 </a>
 主义理由提出上诉，马来西亚检察部门最后同意放弃起诉这11名维族人，并将他们释放。这些人已经在本周二(9日)乘飞机抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 。
 <br/>
 <br/>
 报导说，中共近年来在新疆严酷迫害维族人，导致数百、甚至数千人通过东南亚辗转逃到土耳其避难。
