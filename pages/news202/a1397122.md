### 破纪录的俄国冰激凌 生产在〝云端〞
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】不同于一般的工厂，建在偏远的郊区，在俄罗斯，一家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冰激凌.html" target="_blank">
  冰激凌
 </a>
 工厂，建在云端，成为了海拔最高的冰淇凌工厂。一起去看看。
 <br/>
 <br/>
 在俄罗斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莫斯科.html" target="_blank">
  莫斯科
 </a>
 的这家冰淇凌工厂，打破世界纪录，建在离地237米的高处。位于莫斯科
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摩天大楼.html" target="_blank">
  摩天大楼
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观景台.html" target="_blank">
  观景台
 </a>
 ，获得〝制造
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冰激凌.html" target="_blank">
  冰激凌
 </a>
 最高地点〞的世界纪录。
 <br/>
 <br/>
 俄罗斯纪录主编 Stanislav Konenko：〝这在世界任何地方都找不到，哪里都有冰淇凌，但亮点是，在这么高的地方生产，现在还没有竞争对手。但总是这样，有的人没有想到这一点。〞
 <br/>
 <br/>
 这家冰淇凌工厂今年5月开张，每天生产大量的冰淇凌，每小时就有超过500个甜筒冰淇凌。蘸上美味的比利时巧克力酱，可以直接从传送带送入客人手中。
 <br/>
 <br/>
 冰淇凌工厂员工 Julia Tkachuk：〝和店里出售的那种不同，这种更柔和，是甜食爱好者的最佳选择。不停的吃，也不会冻嘴。另一个好处是，华夫饼外壳不会变软，放到生产线上，只要3秒就能吃到。〞
 <br/>
 <br/>
 工厂所在的楼层是89楼，从这里眺望，能一览莫斯科美景。而所有来观景台的游人，都能免费享用到冰淇凌，数量不限。
 <br/>
 <br/>
 游客：〝非常美妙的味道，可能因为增加了高度。〞
 <br/>
 游客：〝有些特别，和在地面的感觉不同，在这里，感觉在飞翔。〞
 <br/>
 游客：〝冰淇凌就像云彩一样，和别的相比非常好吃。〞
 <br/>
 <br/>
 除了满足观光的游客，大部分冰淇凌，经过精心包装，会送到商店销售。
 <br/>
 <br/>
 新唐人电视台 俄罗斯记者站报导
</div>
