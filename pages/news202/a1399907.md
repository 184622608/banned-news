### 日俄北方领土四岛争议 俄：不可能归还
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二战结束.html" target="_blank">
  二战结束
 </a>
 以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南千岛群岛.html" target="_blank">
  南千岛群岛
 </a>
 （日称北方领土）四岛主权归属至今未绝，两国也未签署和平条约。俄罗斯总统普京9月提议，两国今年在〝未设任何前提〞下签署和平协议，但东京当局说，应先解决领土争议。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 媒体塔斯社（TASS）报导，克里姆林宫发言人培斯科夫（Dmitry Peskov）对俄罗斯电视台说：〝你说这意味自动归还某些领土？绝不可能。〞
 <br/>
 <br/>
 日俄间的四岛争议可追溯至第二次世界大战末期，俄罗斯发动千岛群岛登陆行动，占领千岛群岛，战后声称拥有千岛群岛主权。
 <br/>
 <br/>
 战后，东京当局宣称部分有争议的岛屿并非千岛群岛的一部分，应归还
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 ，包括齿舞岛(Habomai Islands)、色丹岛(Shikotan)、国后岛(Kunashir)、择捉岛（Iturup)。而日俄自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二战结束.html" target="_blank">
  二战结束
 </a>
 以来，因四岛归属争议，一直未能签署正式结束战争状态的和平条约。
 <br/>
 <br/>
 普京（Vladimir Putin）9月提议，两国今年在〝未设任何前提〞下签署和平协议。但东京当局仍然认为，两国应先解决领土争议。
 <br/>
 <br/>
 日相安倍晋三11月16日在澳大利亚港口城市达尔文强调，他与普京此前在新加坡就北方四岛问题达成的共识与日本政府长期以来的立场并不矛盾。
 <br/>
 <br/>
 分析人士表示，安倍此说法的立场实际已有变化。分析认为，日方可能寄希望于先拿回齿舞、色丹两岛，再与俄方就国后、择捉两岛进行谈判。
 <br/>
 <br/>
 但普京15日在新加坡表示，《苏日共同宣言》只写了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏联.html" target="_blank">
  苏联
 </a>
 准备把齿舞、色丹两岛交给日本，并未提及以什么条件交付两岛以及交付后的主权归属问题。
 <br/>
 <br/>
 日媒据此认为，普京这一表态显示，即使是宣言写明交付的两岛，日本想要拿回主权并不容易。此外，在俄方看来，国后、择捉两岛不在谈判范围之内。
 <br/>
 <br/>
 共同社认为，安倍急于在剩余任期内解决日俄间的〝悬案〞，但如果急于求成，可能招致国内反弹，有关谈判进程能否如日本政府所愿恐怕是未知数。
 <br/>
 <br/>
 而在俄罗斯国内，即使只归还齿舞、色丹两岛，反对声音也很强烈。同时，俄方也担忧，一旦两岛交付日本，日美可能基于《日美安保条约》在岛上设立美军基地。
 <br/>
 <br/>
 日方为打消俄方疑虑，曾通过多个渠道向俄承诺不会让美军驻扎在归还岛屿上，但这又可能招致美方不满，使安倍政府在未来谈判中两头为难。
 <br/>
 <br/>
 日本外务省发言人上周表示，安倍将于明年初前往俄罗斯参与会谈。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南千岛群岛.html" target="_blank">
   南千岛群岛
  </a>
  （Yuzhnyy Kurilskye ostrova）
 </h4>
 <br/>
 据维基百科，南千岛群岛位于太平洋西北部的千岛群岛向南延伸部分，总面积达5,038.33km²，包括国后岛（俄称库纳希尔岛）、择捉岛（俄称伊图鲁普岛）、齿舞群岛（俄称哈伯迈群岛）及色丹岛（俄称希科坦岛）。其东北边为得抚岛，西南边是北海道岛，西边是库页岛（日本称桦太）；原住民为阿伊努人。
 <br/>
 <br/>
 目前归俄罗斯萨哈林州管辖，但日本对此地有主权声索，并称其为北方领土、北方地域或北方四岛，行政上划归北海道根室振兴局。
 <br/>
 <br/>
 南千岛群岛在二战结束前为日本领土，但在八月风暴行动的军事进攻后被苏联占领，成为苏联及其继承国俄罗斯的领土之一。俄方认为依照《旧金山和约》，日本已放弃包括南千岛群岛在内的整个千岛群岛之主权，日方则认为该和约所定义的千岛群岛不包括南千岛群岛，因此认定南千岛群岛仍为其领土、并一直向俄方要求归还主权，两方因此产生主权争议。
 <br/>
 <br/>
