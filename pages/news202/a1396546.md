### 印尼地震海啸 马国前副相：因同性恋等活动招〝天谴〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】马来西亚（大马）前副首相、反对党领袖阿迈德扎希周二（23日）在国会表示，印尼帕卢地震和海啸，可能是因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_LGBT.html" target="_blank">
  LGBT
 </a>
 （指
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_同性恋.html" target="_blank">
  同性恋
 </a>
 、双性恋、变性人）活动而招致〝天谴〞。他希望大马及那些反对LGBT行为的人获得上苍的宽恕。
 <br/>
 <br/>
 马来西亚《中国报》报导，阿迈德扎希（Ahmad Zahid Hamidi）周二表示，大家都知道印尼中苏拉威西省（Sulawesi）帕卢（Palu，巴路）最近发生了地震和海啸，整个地区被破坏。他认为，这是上苍的惩罚，因为当地有逾1000个居民涉及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_LGBT.html" target="_blank">
  LGBT
 </a>
 活动。同时他也担心，马国可能因LGBT活动而受到上苍的惩罚。
 <br/>
 <br/>
 阿迈德扎希询问政府官员，该国宗教局遏制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_同性恋.html" target="_blank">
  同性恋
 </a>
 、双性恋和变性人的计划是否成功，他说，〝我们需要确保大马及那些反对LGBT行为的人获得上苍的宽恕。〞
 <br/>
 <br/>
 阿迈德扎希说，根据国际同性恋、双性恋与跨性别协会在今年4月公布的一项报告显示，共有74个国家表明拒绝LGBT。
 <br/>
 <br/>
 主管宗教事务的首相署部长慕扎希称，大马是全球74个拒绝同LGBT的国家之一。自2011年以来，共1450名来自国内各州的同性恋、双性恋及变性人自愿参与改造治疗。
 <br/>
 <br/>
 他解释，完成改造治疗的人中，有些外形装扮已与自身性别相符合，也有一些有所差异，但他们已停止犯下与宗教教义不符的罪行，且有遏制犯罪的自制力。
 <br/>
 <br/>
 苏拉威西岛9月28日遭遇威力强大的地震和海啸侵袭，帕卢许多房屋因此倒塌，海啸掀翻车辆。根据官方公布的数据，地震及地震引发的海啸共造成1,234人死亡，799人受伤，另有99人失踪，近6万人被迫转移。
 <br/>
 <br/>
 （记者李红报导）
