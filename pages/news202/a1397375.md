### 台国防副部长：美台国防工业合作进展大
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】本月28日至30日，2018年度〝美台国防工业会议〞在美国马里兰州首府举行。今年会议规模达到举办18年来的第3大。与会的中华民国国防部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张冠群.html" target="_blank">
  张冠群
 </a>
 表示，今年也是三年来双方合作取得进展最大的一次。
 <br/>
 <br/>
 在〝美台国防工业会议〞上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张冠群.html" target="_blank">
  张冠群
 </a>
 表示，未来台湾推动国防自主，将采用自主系统开发与军售双轨管道并行，同时希望加强台美相关产业交流。
 <br/>
 <br/>
 中华民国国防部军备副部长张冠群：〝我的感觉是，今年整个大家的共识都凝聚了。然后实质上面的一些讨论也比较具体。所以基本上我觉得这三年的进展，今年我感觉是最大的一次。〞
 <br/>
 <br/>
 国家安全会议副秘书长陈文政：〝我们国防自主目前按步就班，已经略有进度。那未来是怎么样让我们的国防自主，能够带领着我们的国防产业走向国际化。〞
 <br/>
 <br/>
 张冠群认为，台湾的国防自主概念已获美方肯定，而台湾的国防产业无论在技术、人才、还是价格方面，都具有竞争优势，也是与其它国家合作的基础。
 <br/>
 <br/>
 中华民国国防部军备副部长张冠群：〝我们的电子业，我们的通讯业，甚至我们精密加工业，我想在全世界上面都是属于领先群的。这都是我们的优势。〞
 <br/>
 <br/>
 立法委员罗致政表示，目前川普政府对中共态度已很清楚，印太战略提出后，美方对台湾角色的期待也很大。这种氛围下，推动台美之间在国防、军事上的更多合作，较为乐观。
 <br/>
 <br/>
 立法院外交及国防委员会召集委员罗致政：〝至少从短期内看到，在军售的项目上就开始有些进展。所以我们认为这是一个战略上的转变，台湾也不断地强调，我们可以在这个战略的转变里面，扮演非常重要的角色。〞
 <br/>
 <br/>
 主办会议的美台商业协会会长韩儒伯预期，年底前美国会有另一波对台军售。他认为，摆脱包裹式军售做法，将有助于避免美国其他行政部门及中共对军售的干预。
 <br/>
 <br/>
 美国会众院外交委员会议员Gerry Connolly：〝美国继续对台军售和提供训练，是非常适当的。美国的对台政策，绝对不应让北京来主导或改变。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
