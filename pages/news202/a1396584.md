### 希腊6.8浅层强震 7国长达800公里一起摇晃
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】美国地质调查局（USGS）表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希腊.html" target="_blank">
  希腊
 </a>
 旅游胜地桑特岛（Zante）附近的爱奥尼亚海（Ionian Sea）发生规模6.8强震，震央深度仅16公里，包括邻近的阿尔巴尼亚等7个国家，长达800公里都感受到明显的摇晃，目前尚未有灾情传出。
 <br/>
 <br/>
 这起强震发生在格林威治时间24日晚间10时55分（当地时间凌晨零时45分），震央位于桑特岛南部Lithakia西南35.9公里处，皮尔戈斯市（Pyrgos）东北80公里处，距离雅典西南方约282公里。
 <br/>
 <br/>
 欧洲独立科学组织EMSC在推特上表示，这次是国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 ，该地震影响至少8个国家，除了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希腊.html" target="_blank">
  希腊
 </a>
 自己以外，还包括马尔他、利比亚、意大利、马其顿、阿尔巴尼亚、波士尼亚波斯尼亚保加利亚与土耳其，长度达到800公里。
 <br/>
 <br/>
 USGS估计，高达1930万人可感受到此波地震，其中5.4万人可能感受到〝强烈〞至〝非常强烈〞的地震，可能令该地区建物造成部分灾损，而影响地区应该相对局部。
 <br/>
 <br/>
 欧洲地中海地震中心（EMSC）回报发现海平面出现约15-20公分变化，警告民众勿到现场观察，当地海平面变化可能大上许多。
