### 中共驻巴基斯坦领事馆遇袭  有伤亡
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】首先来关注中国驻巴基斯坦领事馆遭到攻击的事件。当地时间周五(11月23日) 清晨，中国驻巴基斯坦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡拉奇.html" target="_blank">
  卡拉奇
 </a>
 市领事馆，遭人投掷手榴弹和开枪攻击。目前已知至少有2名巴基斯坦警察死亡，有1名保安警卫受重伤，有3名行凶的枪手被击毙。
 <br/>
 <br/>
 根据当地媒体报导，周五清晨时分，位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡拉奇.html" target="_blank">
  卡拉奇
 </a>
 的中国领事馆外突然传出连续枪响，在确定遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武装攻击.html" target="_blank">
  武装攻击
 </a>
 后，领事馆的保安警卫也立刻开枪回击，巴基斯坦国也派出军警前往支援。
 <br/>
 <br/>
 有目击者在推特发文说，事发时有3、4名男子试图闯入领事馆，并向在馆外站岗的警卫开枪，引爆手榴弹。
 <br/>
 <br/>
 当地警方发言人向媒体记者表示，中国领事馆很可能就是枪手的攻击目标。
 <br/>
 <br/>
 当地医院表示，至少2名警察在枪战中死亡，有一名保安警卫受伤。
 <br/>
 <br/>
 新唐人记者陈依心、尚敬综合报导
