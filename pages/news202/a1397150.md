### 巴西总统决选揭晓 股市蓄势大涨
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统.html" target="_blank">
  总统
 </a>
 大选即将结束计票，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_极右派候选人.html" target="_blank">
  极右派候选人
 </a>
 波索纳洛确定当选第38任新总统。这将让巴西国策向右急转，投资人寄望他上台后会推动市场改革，激励
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 继续上涨。分析师预测未来一年巴西股市会大涨至历史新高。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 周日（28日）举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统.html" target="_blank">
  总统
 </a>
 选举第二轮投票，根据目前选举法院开出96.3%选票结果，自由社会党籍候选人波索纳洛（Jair Bolsonaro）获得55.6%的有效票，超过左派哈达德（Fernando Haddad）的44.5%。波索纳洛确认当选，将在明年元旦就任总统。
 <br/>
 <br/>
 波索纳洛上台将让巴西国策向右急转，转向民营化国有企业并开放在雨林区采矿。有分析师选前就看好伞兵上尉出身的波索纳洛会当选，纷纷预测未来一年巴西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 会大涨至历史新高。
 <br/>
 <br/>
 乐观者看好波索纳洛与他的经济顾问古埃德斯主导的经济政策，后者是出身芝加哥大学的经济学者，并曾创立私募基金，与民间企业关系深厚。
 <br/>
 <br/>
 瑞银策略师预计，巴西圣保罗证交所指数（Ibovespa）未来两个月可能看涨近40%，其他如花旗在内券商和投资机构也一面倒看好巴西股市或加码巴西股票。
 <br/>
 <br/>
 据9月中旬的民调显示波索纳洛支持率一路窜升后，Ibovespa以美元计算已大涨29%，在彭博追踪的全球90个股价指数中表现最佳。巴西里尔汇率也回升13.5%。
 <br/>
 <br/>
 摩根士丹利发布报告称，波索纳洛有政治资源和意愿来推行财政重整和结构改革，这对短到长期经济成长潜力都有正面效益。
 <br/>
 <br/>
 不过，波索纳洛还必须落实包括年金改革在内的财政重整，才可能节制预算，这是先决条件。Rio Bravo投资公司首席经济师布契尼认为：〝如今最大的风险就在于执行。〞
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
