### 习近平访马德里 西班牙：不计划签一带一路
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 周二（27日）抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 进行3天以商业为主轴的国是访问，希望强化与欧洲的关系。但西国政府高层表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马德里.html" target="_blank">
  马德里
 </a>
 当局不计划签〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞倡议。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 国王菲利佩六世（King Felipe VI）和王后雷蒂西亚（Queen Letizia），周二晚在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马德里.html" target="_blank">
  马德里
 </a>
 （Madrid）市郊的札苏拉宫（Zarzuela palace）与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 进行了会晤。周三，习近平将与西班牙总理桑杰士（Pedro Sanchez）展开会谈。
 <br/>
 <br/>
 这是习近平前往阿根廷出席20国集团（G20）峰会途中的一站，也是中国国家主席时隔13年再次访问西班牙，时值两国庆祝建交45年。这3天的访问没有安排记者会。
 <br/>
 <br/>
 西班牙政府表示，双方将签署至少18项商业及合作协议，包括扩大进口在亚洲市场需求量大的西班牙高档伊比利生火腿等商品。
 <br/>
 <br/>
 中国是西班牙在欧洲联盟以外最大贸易伙伴。根据西班牙外交部数据，去年西班牙对中国出口增加28.3%，但与中国贸易赤字仍增加了2.3%，达到194亿欧元（约221亿美元）。
 <br/>
 <br/>
 不过，西班牙高阶政府官员周二在习近平专机抵达前表示，西班牙不会签署加入寻求加强欧亚互联互通的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞倡议。
 <br/>
 <br/>
 一名拒绝洩露身分的西班牙总理办公室官员也表示：〝我们将不签署这项倡议。〞
 <br/>
 <br/>
 在谈到具体原因时，这名官员表示：〝欧亚已有连结倡议，我们认为欧洲国家应该在现行框架内作业。〞
 <br/>
 <br/>
 他所说的现行框架是欧洲联盟（EU）9月20日提出发展亚洲基础设施的外交政策计划。虽然欧盟公开否认是为回应北京〝一带一路〞，但也透露与中方竞争、并试图影响〝北京做法〞的意图。新战略透过在亚洲地区开发基础建设的类似计划，加强欧亚之间的联系。
 <br/>
 <br/>
 中共在2013年公布这项价值高达数十亿美元的倡议，目标是通过港口、铁路、公路和工业园区网络连接这两大洲。
 <br/>
 <br/>
 一带一路倡议范围涵盖65个国家，估计代表全球60%人口和1/3经济产出。但这项计划也使各界担忧中共扩大全球影响力的野心，欧盟和一些欧洲国家担忧，中共正通过〝16+1〞地区机制拉拢中东欧国家，试图分裂欧盟。一些国家纷纷觉醒，谴责和抵制中共的一带一路计划。
 <br/>
 <br/>
 该项目在非洲多国也引发了当地民众的不满。在9月3日至4日的中非合作论坛期间，中共宣布向非洲提供600亿美元的贷款和援助，但外界发现，中共此举并未引发非洲人民的好感。肯尼亚警方9月5日突袭中共官媒环球电视网的非洲总部，民间也出现大量仇视中共的言论。
 <br/>
 <br/>
 那么，非洲人为何没有感受到中共的〝深情厚谊〞，反而如此愤怒呢？一些观察人士批评，中共在非洲的投资及借贷是制造〝债务陷阱〞，非洲小国难以偿还沉重的债务。
 <br/>
 <br/>
 例如，中共〝一带一路〞重点合作国吉布提债务2014年约占GDP一半，到2016年升至85%。引人注意的是，吉布提在2016年底所有外债的82%都是对中共的债务。
 <br/>
 <br/>
 肯尼亚与中共合作建造的铁路项目，被当地人批评债务问题。肯尼亚交通部公开数据，指这个铁路项目营运首年为1亿美元亏损。据估计指出，中共目前持有肯尼亚70%的债务。而另一非洲小国津巴布韦最近不得不请求中共减免债务，以便继续从北京贷款。
 <br/>
 <br/>
 受〝一带一路〞影响而深陷债务危机的斯里兰卡，去年12月，因无力偿还债务，不得不将其最重要的战略港口汉班托塔港以及港口周围1.5万英亩的土地交给中共，租期长达99年。斯里兰卡政府的批评者谴责此举有损国家主权。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
