### 10月17日国际新闻简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国第一夫人.html" target="_blank">
  美国第一夫人
 </a>
 乘坐的专机，因为机舱内冒烟被迫降落。下面请看一组新闻简讯。
 <br/>
 <br/>
 周三，第一夫人梅拉妮亚的飞机起飞10分钟后，在机舱内发现烟雾，被迫降落在华府的军事基地。所有人员都安然无事。随后，梅拉妮亚转机飞往杰斐逊大学医院，推广她发起的儿童〝做到最好〞系列活动。她感谢医疗界建立美国新生儿节制症（NAS）跟进系统，关注儿童药物滥用。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 宣布，退出有144年历史的万国邮政联盟（UPU）。因中国在这一机构中列为开发中国家，享有国际运输优惠。美国本土制造的正品通过美国邮政的运输价格，比中国寄来的仿冒品邮费要贵。美邮政管理局（USPS）在从中国运到美国的每个小包裹上，要损失一美元。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周三召开内阁会议，要求部长将明年的开支预算削减5％，摆脱浪费，称为〝五美分计划〞。川普早就想精减政府开支，这个〝五美分计划〞将持续到明年。
 <br/>
 <br/>
 10月17日，总部设在法国里昂的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 警察组织宣布，93个国家的警察突击搜查出55吨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒品.html" target="_blank">
  毒品
 </a>
 ，包括可卡因，海洛因以及数百万粒合成毒品。逮捕了1,300人。在东南亚收缴了1800万粒毒品。
