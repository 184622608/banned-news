### 日ODA转投印度联手堵中共 党媒仍唱〝中日共荣〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍.html" target="_blank">
  安倍
 </a>
 晋三结束访华的次日，中共党媒还在高歌〝中日共存共荣〞，安倍就在私人别墅与来访的印度总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莫迪.html" target="_blank">
  莫迪
 </a>
 商讨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_围堵中共.html" target="_blank">
  围堵中共
 </a>
 ，并确定向印度追加3千亿日元的政府开发援助（ODA）。而安倍访华时刚刚叫停持续近40年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对华ODA.html" target="_blank">
  对华ODA
 </a>
 。
 <br/>
 <br/>
 10月29日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍.html" target="_blank">
  安倍
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莫迪.html" target="_blank">
  莫迪
 </a>
 在东京举行了正式会谈。28日，安倍在自己位于山梨县的私人别墅招待莫迪，这是安倍首次在这里招待外国元首，日媒称之为〝罕见优待〞。而27日，安倍刚刚从北京返回日本。
 <br/>
 <br/>
 日媒报导，会谈一开始，安倍就表示〝日印关系是世界上发展潜力最大的双边关系〞。双方将在国安、数据、基础建设、防灾等领域加强合作。其中，日印推进两国安保领域合作的一个重要背景，是中共在海上影响力的不断扩大。
 <br/>
 <br/>
 德媒指，日印领导人此次会谈达成的共识是，确保〝一个自由开放的印度太平洋〞，两国将密切合作，联手应对中共在南中国海的扩张。
 <br/>
 <br/>
 接待完莫迪后，安倍下月还将赴澳大利亚访问，商讨日本自卫队和澳军的军事合作。
 <br/>
 <br/>
 此外，日印双方还将推进在基础设施建设方面的合作，日本将向采用日本技术的印度高铁计划追加3千亿日元（约合184亿元人民币）的ODA贷款。而上个年度，日本政府向印方提供的贷款只有1千亿日元。
 <br/>
 <br/>
 在中国大陆媒体的一些报导中，日本对印度的3千亿日元援助被〝缩水〞成3亿日元，未知只是一时笔误还是刻意为之。
 <br/>
 <br/>
 就在安倍访华前的23日，日本政府突然宣布停止1979年开始的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对华ODA.html" target="_blank">
  对华ODA
 </a>
 。日本ODA是针对发展中国家的援助项目，其中对华ODA近40年的援助总额约3万亿日元。
 <br/>
 <br/>
 有分析指，日方以中方〝经济现状不再需要〞为由叫停对华ODA，要求和中共〝平等合作〞，似乎在配合美国政府对中共以〝发展中国家〞名义在世贸组织骗取好处的谴责。
 <br/>
 <br/>
 时评人士陈破空指出，日方此举等于是宣称日中关系本质上已是竞争关系。
 <br/>
 <br/>
 他还分析说，安倍回日本第二天就热情迎接莫迪，这是一个有心的安排，往浅了说是要在中印两个亚洲大国间寻求平衡，往深了说是宣示：日本、印度、美国和澳大利亚的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印太联盟.html" target="_blank">
  印太联盟
 </a>
 〞绝不会因为中共的拉拢而改变。
 <br/>
 <br/>
 亦有外媒分析，安倍正进一步推进在大国之间的〝平衡外交〞，在稳固〝印太战略安全纽带〞的同时，寻求把自己定位为在该地区与中共抗衡的一股力量。
 <br/>
 <br/>
 10月25日至27日，安倍访问北京，获得高规格接待。中共官方彻底颠覆往日的反日立场，高调宣传〝中日友好〞，在中国社会引起很大争议。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：）
