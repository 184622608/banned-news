### 菲国苏比克湾惊传枪响 拍卖商人遭当众射杀
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】菲律宾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏比克湾.html" target="_blank">
  苏比克湾
 </a>
 经济特区一家高档饭店昨(28日)晚发生枪击命案，一名二手设备拍卖商人在饭店大门口遭人当众射杀，震惊全国。
 <br/>
 <br/>
 席丁（Dominic Sytin），创立并经营规模极大的工业及运输设备拍卖公司，创业的成功故事还上过总部设于澳洲的CEO杂志。
 <br/>
 <br/>
 警方报告中说，席丁昨晚约7时49分走出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏比克湾.html" target="_blank">
  苏比克湾
 </a>
 高档饭店〝灯塔〞（Lighthouse）时，突遭一名骑机车的杀手靠近开枪，踫！踫！身中4弹身亡，杀手随后逃逸无踪。
 <br/>
 <br/>
 警方正在调查凶手身分及杀人动机。菲国二手卡车及工业设备绝大部分是从苏比克湾进口，经济利益庞大，外界推测枪击命案可能与商业纠纷有关。
 <br/>
 <br/>
 苏比克湾位于马尼拉以北约136公里，是菲国面积最大的经济特区，依山傍海，几条联外道路都设有关卡，连交通罚单未缴都出不了特区，长年以来治安良好，这起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 令菲国民众大感震惊。
 <br/>
 <br/>
