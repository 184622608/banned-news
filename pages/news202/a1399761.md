### 未受贸易战影响  美中举行联合救灾演习
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 逐步升温，不过，两国士兵，周六共同完成了，灾难救援演习，展示出美中，在其他领域的合作。
 <br/>
 <br/>
 周六，美中联合灾难救援演习在南京结束。演习中，两国士兵共同合作，模拟从地震摧毁的建筑中搜救幸存者等场景。
 <br/>
 <br/>
 目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 处于紧张阶段。中共国防部长魏凤和、中共中央外事办主任杨洁篪上周造访华府，与美国国务卿蓬佩奥以及国防部长马蒂斯会晤，重启美中外交安全对话。
 <br/>
 <br/>
 外界分析，这一对话很难说明美中关系将出现重大缓和。但周六的灾难救援演习表明，两国军队在部分领域仍存在合作关系。
 <br/>
 <br/>
 这是双方第十四次举行这一联合演习，去年的演习在美国进行。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
