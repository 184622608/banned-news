### 国际刑警组织俄国候选人 美议员炮轰：狐狸看管鸡舍
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】美国参议员等批评人士近日呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警.html" target="_blank">
  国际刑警
 </a>
 组织，拒绝俄罗斯官员普罗科普丘克出任新主席。他们形容，选择普罗科普丘克出任新主席，就像是〝让一只狐狸掌管鸡舍〞
 <br/>
 <br/>
 来自世界各国代表周三（21日）将于迪拜举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警.html" target="_blank">
  国际刑警
 </a>
 组织大会（Interpol General Assembly），选出总部位于法国的国际刑警组织下任主席。
 <br/>
 <br/>
 除韩国前警官、现任国际刑警组织代理主席金正阳有望胜出外，俄国内政部官员、国际刑警组织副主席普罗科普丘克（Alexander Prokopchuk）也可能有机会出任该职位。
 <br/>
 <br/>
 日益高涨的批评声浪，呼吁国际刑警组织拒绝让普罗科普丘克接掌国际刑警组织，包括美国联邦参议员夏亨（Jeanne Shaheen）、韦克尔（Roger Wicker）、库恩斯（Chris Coons）与卢比奥（Marco Rubio）19日都发表声明，呼吁国际刑警组织大会的会员，拒绝支持普罗科普丘克角逐主席。
 <br/>
 <br/>
 他们担忧莫斯科可能会滥用职权，对付俄国政治上的反对人士。因为之前俄国就经常滥用国际刑警组织的权利清算旧帐，打压政治反对派、异议人士和记者。他们并形容选择普罗科普丘克出任新主席，就像是〝让一只狐狸掌管鸡舍〞。
 <br/>
 <br/>
 孟宏伟今年9月底由法国前往中国后失踪，引发国际关注后，中共当局才宣布孟宏伟涉嫌违法正接受调查，并通知国际刑警组织孟宏伟请辞主席一职。
 <br/>
 <br/>
 10月7日，国际刑警组织发布公告，已收到孟宏伟的辞呈，由韩国的国际刑警组织高级副主席金正阳，暂时担任该组织代理主席一职。
 <br/>
 <br/>
 64岁的孟宏伟，2016年11月当选国际刑警组织主席，是首个当选国际刑警组织主席的中国人，任期至2020年。上任之初，就有人权组织担心，致力于破坏人权的中共，或会利用该组织作为通缉海外异见人士的平台。他当选主席，被外界认为是中共利用贿赂渗透了国际刑警组织，以及中共与法国幕后交易的结果。
 <br/>
 <br/>
 国际刑警组织成立于1914年，总部设在法国里昂，现有192个成员国，是唯一的全球性警务合作组织，也是仅次于联合国的第二大国际组织。
 <br/>
 <br/>
 国际刑警组织主席由全体大会选举产生，任期4年，4年选举一次，不得连选。作为该组织最高负责人，负有确保该组织各项活动遵循会议的各项决议、指导和监督秘书长工作等职责。
 <br/>
 <br/>
 不过，国际刑警组织之前称，负责国际刑警组织机构日常工作的是秘书长而非主席。目前是德国籍的史托克（Jurgen Stock）担任此职。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
