### 美客机准备起飞 男子竟开舱门引起混乱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月28日讯】美国廉航〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边疆航空.html" target="_blank">
  边疆航空
 </a>
 〞（Frontier Airlines）22日一架客机在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 准备起飞时，一名乘客突然在机上呕吐，急于离开机舱，竟然打开已经在跑道上滑行的机舱门，差点被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吸入引擎.html" target="_blank">
  吸入引擎
 </a>
 中，幸好机上乘客及空服员及时上前阻止悲剧发生，客机也被迫煞停。
 <br/>
 <br/>
 事发在上周一(22日)下午2时10分，这架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边疆航空.html" target="_blank">
  边疆航空
 </a>
 编号F9 87的客机，准备从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎昆.html" target="_blank">
  坎昆
 </a>
 （Cancun）飞往密苏里州圣路易（St. Louis）。
 <br/>
 <br/>
 坐在美籍乘客格雷（Tommy Gray）前排的卡兰兹耶夫（Andy Karandzieff）说：〝坐在我们后方的男子开始呕吐，之后开始拍打驾驶舱的门。〞空服员劝阻他并要带他回座位，但格雷想离开机舱，突然跑向逃生舱门，并把它打开，导致客机自动放出充气滑梯。
 <br/>
 <br/>
 机组人员和其他乘客见状，她几个人上前制伏格雷，避免他跳出机外。
 <br/>
 <br/>
 乘客英厄姆（Kathleen Ingham）拍下混乱过程说：〝如果不是英勇的众人，及时将他按住，他一定会被卷入引擎。〞
 <br/>
 <br/>
 <b>
  相关视频：
 </b>

 </center>
 <br/>
 航警后来到场将引起机舱大混乱的格雷带走，并将他送医治疗。客机最后返回闸口，航空公司安排乘客转乘其他客机，或安排他们入住饭店一晚。
 <br/>
 <br/>
 格雷的母亲事后表示，儿子是与爸爸到坎昆度假，但前一晚在饭店内遇袭，导致头部受创。并称，相信儿子是因为被人在他的饮料〝下药〞，才会行为异常。
 <br/>
