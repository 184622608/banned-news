### 德国地方选举姐妹党失利 默克尔联合政府遭重击
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】德国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴伐利亚邦.html" target="_blank">
  巴伐利亚邦
 </a>
 周日(14日)举行议会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选举.html" target="_blank">
  选举
 </a>
 ，长年在当地握有过半多数的基督教社会党（CSU
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基社党.html" target="_blank">
  基社党
 </a>
 ）出现空前失利，社会民主党（SPD 社民党）得票率也惨遭腰斩至9.6%，反应选民强烈不满，造成对总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默克尔.html" target="_blank">
  默克尔
 </a>
 的联合政府重大压力。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴伐利亚邦.html" target="_blank">
  巴伐利亚邦
 </a>
 自1960年代以来，向来是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基社党.html" target="_blank">
  基社党
 </a>
 一党独大，也是总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默克尔.html" target="_blank">
  默克尔
 </a>
 （Angela Merkel）的保守派姐妹党。但这次基社党得票率仅37%，比4年前足足少了10%，沦为需与他党共组执政联盟的窘境，外界认为，有可能找上保守派地方政党〝自由选民〞（Free Voters）党。
 <br/>
 <br/>
 另一默克尔联合政府的盟友社民党在巴伐利亚的得票率也惨遭腰斩至9.6%，拱手将当地第2大党地位让给绿党（Greens）。
 <br/>
 <br/>

     选举
    </a>

   </font>
  </a>
 </center>
 <br/>
 而这次巴伐利亚选举的大赢家是极右翼反移民政党〝德国另类选择党〞（AfD），获11%的选票并首度在邦议会里拥有席次。AfD于德国16个邦议会里再下一城。德国全国共有16个邦。
 <br/>
 <br/>
 彭博报导，现在各方都将基社党在巴伐利亚的重挫归咎于中央政府内的分崩离析。而今夏一连串边缘政策危机后，默克尔的联合政府就已岌岌可危。
 <br/>
 <br/>
 而联合政府风暴核心者，正是身兼内政部长的基社党党魁赛贺佛（Horst Seehofer），他严辞批评默克尔的移民政策，想把基社党向右调整。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/10-15/p9080841a809760477.jpg" target="_blank">

    德国身兼内政部长的基社党党魁赛贺佛(前)批评默克尔的移民政策，想把基社党向右调整。( ODD ANDERSEN/AFP/Getty Images)
   </font>
  </a>
 </center>
 <br/>
 接下来，默克尔的挑战将是10月28日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑森邦.html" target="_blank">
  黑森邦
 </a>
 （Hesse）选举，她所属的基督教民主党（CDU
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基民党.html" target="_blank">
  基民党
 </a>
 ）自1999年以来一直是黑森邦执政党，但民调显示也不乐观，在当地得票率预计会掉至1960年代以来新低。
 <br/>
 <br/>
 基民党将于12月在汉堡举行大会改选党魁，若基民党在黑森邦失利，默克尔的威信无疑会进一步削弱。
 <br/>
 <br/>
 社民党秘书长克林拜耳（Lars Klingbeil）为此呼吁，这是给历经数月紧张与内讧的默克尔政府该改调整的一大警讯，〝柏林应拿出另一番执政风格〞。令外界分析，社民党恐因得票率持续滑落而退出联合执政。
