### 克里米亚校园枪击案18死 凶嫌自尽
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】观众朋友大家好，欢迎收看新唐人全球新闻。周三，俄罗斯占领的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克里米亚.html" target="_blank">
  克里米亚
 </a>
 地区发生一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_校园枪击案.html" target="_blank">
  校园枪击案
 </a>
 ，造成至少18人死亡，数十人受伤。凶嫌自杀身亡。
 <br/>
 <br/>
 俄罗斯官员表示，18岁的学生罗斯利亚科夫周三中午持枪闯入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克里米亚.html" target="_blank">
  克里米亚
 </a>
 地区克赤市一所技术学院。先在学校餐厅引爆装满金属碎片的爆炸物，随后持枪沿大楼走廊随意扫射。导致至少18人死亡，数十人受伤。
 <br/>
 <br/>
 俄罗斯调查委员会代表 Svetlana Petrenko：〝从犯罪现场来看，调查人员初步认定，这名年轻人在大学里向别人开枪之后自杀。〞
 <br/>
 <br/>
 学生和老师形容案发现场一片混乱。校园里首先传出多起爆炸声，然后是一阵枪击。惊慌失措的学生试图逃离建筑物。
 <br/>
 <br/>
 克赤技术学院有大约850名学生。伤亡的很多是十几岁的学生。
 <br/>
 <br/>
 罗斯利亚科夫，是该校四年级学生。监视器拍到他持枪进入学校，并在校内开枪的画面。
 <br/>
 <br/>
 调查小组曾经表示此案是恐怖袭击，可能有共犯。后来改为〝大规模谋杀〞。
 <br/>
 <br/>
 犯案动机目前尚不清楚，俄罗斯政府已着手调查。
 <br/>
 <br/>
 克里米亚将为遇难者哀悼三天，俄罗斯社会保险基金称，会为遇难者家庭每家提供100万卢布的抚恤金。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
