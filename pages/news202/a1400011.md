### 燃油飙涨 法国黄背心抗议拟24日到巴黎
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】为抗议燃油不断飙涨，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 各地自上周末开始发起示威抗议活动。周一，抗议活动进入第三天，参与者封锁了3个油库的入口，并阻断几个主要道路通行。抗议者还计划24日到巴黎再举行大规模的示威活动。
 <br/>
 <br/>
 周一清晨，高速公路和环岛仍有数十个路障，数量比17日的2000多个少了很多。
 <br/>
 <br/>
 中央社报导，一名记者表示，约200辆卡车停在西部城市勒恩（Rennes）一条通往油库的主要道路上，部分抗议者在勒恩露营过夜。其他人继续在超市停车格露营。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 内政部国务秘书努涅斯（Laurent Nunez）告诉法国数位电视频道CNews：〝这场活动并没有什么独特之处…显然没有像上周六的大。〞他说，警方将继续介入，确保主要道路不会受阻。
 <br/>
 <br/>
 不过，抗议者已计划再举行大规模的示威活动。右翼政党〝法国站起来〞（Stand Up France）一名成员呼吁示威者24日前往巴黎，〝封锁〞这座城市。这条贴文在脸书（Facebook）已吸引16万5000人次点阅。
 <br/>
 <br/>
 另一个脸书专页也呼吁，24日在巴黎举行大型集会活动，〝因为那里是政府所在！！！〞，贴文同样获得广大民众关注。
 <br/>
 <br/>
 法国的柴油价格在过去一年来已经上涨大约23%，汽油价格也上涨了17.6％。
 <br/>
 <br/>
 17日一早，法国全境按照预期开始了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄.html" target="_blank">
  黄
 </a>
 背心〞行动，示威者们都穿上黄背心，意图在交通要道、大转盘、高速公路出口及停车场等地组成人肉路障，来抗议攀升的油价。
 <br/>
 <br/>
 根据警方数据，当天法国境内大约上演了2000场示威活动，但其中只有100多场是提前去警署备案过，属于合法的。
 <br/>
 <br/>
 巴黎警察局表示，17日夜间，普瓦西地区和吉尔市的3个雷达被人纵火烧毁。政府出动了防暴警察来驱散示威人群，有地区爆发警民冲突，警察向示威人群发射催泪弹，并筑起路障，但司机们没有退让的意思。仅仅在北加莱和阿尔萨斯这两个地区，就有600多个城市进行示威。
 <br/>
 <br/>
 抗议者们称，政府藉着环保、绿能、反污染等理由，不断加征燃料税，而明年一月再次宣布的调涨，成为导致抗议爆发的最后一根稻草。
 <br/>
 <br/>
 〝黄背心〞抗议活动已吸引近30万人参与，其中一人不幸丧生，超过400人受伤。约160人被捕。
 <br/>
 <br/>
 总统马克龙执政18个月以来，为了重振经济，放松对劳动法的管制，并对负债累累的国有铁路运营商进行改革。而增加燃料税、烟草税以及社会福利税等措施，令不少法国人倍感压力，全国性大规模罢工也时有发生。
 <br/>
 <br/>
 法媒报导，目前，马克龙的民意支持率已经跌落至25%。
 <br/>
