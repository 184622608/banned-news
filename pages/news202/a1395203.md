### 军方人员个资遭骇 美国防部展开调查
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】美国军方当地时间周五（12日）表示，美国国防部正在调查一起骇客攻入部分国防部人员的旅行纪录，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_个资.html" target="_blank">
  个资
 </a>
 及信用卡
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_资料外流.html" target="_blank">
  资料外流
 </a>
 事件。
 <br/>
 <br/>
 路透社报导，美国国防部发言人布齐诺（Joseph Buccino）中校表示，骇客攻入一个商业供应商，导致部分可供辨识个人身分的资料外洩。
 <br/>
 <br/>
 美国军方高层10月4日得知该事件。布齐诺称，这次被骇事件只有一小部分的人受到影响，国防部已对此展开调查。
 <br/>
 <br/>
 但美联社报导引述一名熟悉内情的美国官员消息称，这次的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_骇客攻击.html" target="_blank">
  骇客攻击
 </a>
 事件可能会有3万名员工受到影响，而且随着调查的进行，这项数据可能会持续攀升。估计攻击事件可能发生在几个月前，但直到近日才被发现。
 <br/>
 <br/>
 不过，这位没有透露姓名的官员也表示，没有机密资料受到波及。
 <br/>
 <br/>
 网路安全日益受到美国军方和情报官员的关注。
 <br/>
 <br/>
 西方国家上周口径一致谴责俄罗斯发动全球性骇客攻击，锁定的目标有反禁药的体育机构、核能电厂公司和化学武器监督组织。
 <br/>
 <br/>
 美国司法部10月4日对7名俄罗斯特工提出起诉，罪名是骇客攻击国际反兴奋剂机构。同一天，荷兰也指责俄方发动的一系列网络攻击。
 <br/>
 <br/>
 美国司法部助理部长德梅斯4日称:〝今天起诉了一些在（荷兰）海牙被捕的俄罗斯特工，以及他们在莫斯科的同事。〞
 <br/>
 <br/>
 德梅斯表示，起诉的这7个人，涉嫌对国际反兴奋剂机构进行骇客攻击，因为该机构谴责俄罗斯政府支持运动员使用兴奋剂，并支持禁止俄罗斯运动员参加国际体育比赛。
 <br/>
 <br/>
 另外，一个自称〝入侵真相〞的神秘团体，近期也引起了西方媒体的特别关注。该团体专门揭露中共网军黑底或曝光中共暗中支持的骇客行动。
 <br/>
 <br/>
 〝入侵真相〞去年曾经揭发广东博御信息技术公司（Guangdong Bo Yu Information Technology Co）两名员工吴英卓（Wu Yingzhou，译音）和董浩（Dong Hao，译音）隶属于与中共有关联的APT3。半年后，美国当局起诉了这两个人，指控他们涉及参与了APT3入侵穆迪分析公司（Moody’s Analytics）和德国西门子公司（Siemens AG）盗取情报的犯罪行动。
 <br/>
 <br/>
 根据科技新闻网站Motherboard的报导，〝入侵真相〞今年8月曾经在推特上贴文宣告，他们的宗旨是让中国（中共）操控的骇客在从事非法网络活动前，能够三思而后行。帖文称，〝我们正致力于厘清，到底哪些是单纯的‘骇客公司’，而哪些是掩护中国（中共）政权并在日后作为代罪羔羊而招募的骇客。〞
 <br/>
 <br/>
 近年来，针对美国政府、企业和个人的网络攻击给美国造成重大经济损失，并影响到民众的生活。川普总统（特朗普）9月底签署了其上任后首个国家网络安全战略。此战略授予美国多个部门更大的权限，主动出击清除中俄等境外势力的网络威胁。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
