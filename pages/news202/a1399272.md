### 北海道初雪 旭川迟到28天 滑雪场开不了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】日本今年秋季天气偏暖，位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北海道.html" target="_blank">
  北海道
 </a>
 最北端的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_稚内.html" target="_blank">
  稚内
 </a>
 13日晚间气温骤降，终于降下今年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_初雪.html" target="_blank">
  初雪
 </a>
 ，比去年延迟了26天，创下1990年以来最晚初雪纪录，另其他8个主要城市至今还没有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下雪.html" target="_blank">
  下雪
 </a>
 ，是北海道132年以来降雪最迟。
 <br/>
 <br/>
 入秋以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北海道.html" target="_blank">
  北海道
 </a>
 一带一直没有冷空气南下，日本气象专业网站〝Weather News〞（WNI），北海道14日终于降下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_初雪.html" target="_blank">
  初雪
 </a>
 ，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_稚内.html" target="_blank">
  稚内
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旭川.html" target="_blank">
  旭川
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网走.html" target="_blank">
  网走
 </a>
 等地都关测到降雪。
 <br/>
 <br/>
 位于北海道最北端的稚内，这场初雪比去年晚了26天，比往年平均延迟23天；旭川比去年晚了28天，比往年平均晚了22天，创下史上第2晚纪录；网走则比起去年晚了27日，也往年平均晚了14日。
 <br/>
 <br/>
 据悉，北海道有记录以来，最晚的初雪，是发生在1886年；当时函馆与札幌在11月18日，才降下第一场雪。
 <br/>
 <br/>
 原本订在11月16日开始营业〝札幌国际滑雪场〞，因迟迟未下初雪，没有积雪，不少旅客因此取消行程。
 <br/>
 <br/>
 日本气象厅表示，下周可能会更冷，降雪会越来越多，札幌等地也将降雪。并提醒民众，由于初雪刚降时，特别容易发生擦撞、打滑等事故，前往北海道旅游，要做好冬季用轮胎的准备。
 <br/>
 <br/>
 气象厅也表示，由于今年冬天，太平洋可能发生圣婴现象，预料今年日本冬天，整体会偏暖。
 <br/>
 <br/>
 (责任编辑：卢勇信)
