### 博尔顿会见〝巴西川普〞 讨论地区安全
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】美国国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 周四前往巴西，和当选总统波索纳洛在他的家中会面，讨论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地区安全.html" target="_blank">
  地区安全
 </a>
 问题。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 和波索纳洛在巴西首府里约热内卢南部，波索纳洛的家中举行了一小时会谈。
 <br/>
 <br/>
 博尔顿在会谈后的推文中表示，讨论卓有成效。
 <br/>
 <br/>
 他说：〝和巴西总统波索纳洛及其国家安全团队，进行了广泛而富有成效的会谈。我向波索纳洛转达川普总统的邀请，希望他访问美国。我们期待与巴西建立活跃的伙伴关系。〞
 <br/>
 <br/>
 参加会谈的还有三位巴西候任的官员，包括国家安全顾问、国防部长和外交部长。其中外交部长埃内斯托．阿劳若反对全球化，认为气候变化理论是马克思主义的入侵思想。
 <br/>
 <br/>
 周三，波索纳洛表示，将向巴西政府施加压力，放弃主办明年的联合国气候会议。他在竞选期间承诺，巴西将退出《巴黎协定》，并有意进一步发展亚马逊雨林。
 <br/>
 <br/>
 博尔顿周二在华府表示，波索纳洛当选是巴西的〝历史机遇〞，美国将和巴西在安全、经济和其他问题上合作。周四的会谈，为川普总统和波索纳洛的合作作准备。
 <br/>
 <br/>
 博尔顿预期将寻求波索纳洛的支持，向委内瑞拉政府施压。
 <br/>
 <br/>
 波索纳洛今年十月在巴西大选中获胜，被誉为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西川普.html" target="_blank">
  巴西川普
 </a>
 〞。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
</div>
