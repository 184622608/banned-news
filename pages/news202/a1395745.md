### 继乌拉圭之后 加拿大成第二个休闲大麻合法国
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 成为继
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌拉圭.html" target="_blank">
  乌拉圭
 </a>
 之后，第二个在零售店合法销售和购买休闲用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻.html" target="_blank">
  大麻
 </a>
 的国家。此举引起不少民众对青少年健康的担忧。
 <br/>
 <br/>
 周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 最东部的纽芬兰省和拉布拉多省，最先开始在零售店合法买卖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻.html" target="_blank">
  大麻
 </a>
 。
 <br/>
 <br/>
 加拿大是第一个将休闲用大麻合法化的西方国家，但在多伦多和温哥华等大城市，这一法规并没有引起共鸣，没有零售店将出售大麻。
 <br/>
 <br/>
 另外，加拿大13个省将各自制定相关规定，一些省允许私营商店经营，一些省只允许政府所有的商店销售。
 <br/>
 <br/>
 休闲性大麻合法化，是加拿大总理特鲁多所在的自由党的竞选承诺之一，他们认为，这样可以更好地保护年轻人，同时阻止犯罪分子通过黑市牟利。
 <br/>
 <br/>
 但民众针对大麻合法化争论激烈，尤其担忧危害青少年健康。
 <br/>
 <br/>
 新唐人记者王羽音、明玉综合报导
</div>
