### 亚太峰会再交火？传彭斯将发檄文猛批中共
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）的10月对华政策演讲，被指拉下中美新冷战铁幕，有美国官员透露消息指，即将举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚太经合峰会.html" target="_blank">
  亚太经合峰会
 </a>
 ，彭斯将再次发布演讲，预计将当着出席峰会的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的面，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抨击中共.html" target="_blank">
  抨击中共
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 11月11日启程前往亚洲，进行为期一个星期的访问，他将在11月17日出席在巴布亚新几内亚首都、莫尔兹比港举行的亚太经合组织（APEC）峰会，并就美国的〝印度-太平洋〞战略发表重要演讲。
 <br/>
 <br/>
 日前，在彭斯此次亚太之行的电话吹风会上，一名美国高级官员对美国之音透露，彭斯在APEC峰会上的这篇演讲，将〝深入和全面〞地阐述美国对地区安全构架以及战略前景的构想。
 <br/>
 <br/>
 这名官员说，预计彭斯在讲话中将再次强烈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抨击中共.html" target="_blank">
  抨击中共
 </a>
 。而这与彭斯10月初的演讲〝有关联〞。当时，彭斯在演说中全面抨击中共，被外界称作是向中共下达的檄文。
 <br/>
 <br/>
 《澳大利亚金融评论》11月11日也报导称，彭斯此行的重点是在亚太经合组织（APEC）首席执行官论坛上发表的演讲。
 <br/>
 <br/>
 中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 也将出席论坛并发表讲话。在其之后，彭斯也将就美国〝印度-太平洋〞战略发表重要演讲，并当着习近平的面，继续抨击中共的〝威权主义和侵略〞。
 <br/>
 <br/>
 上述美国官员还透露，彭斯将宣布美国政府出资600亿美元，对印太地区各国提供经济援助，以抗衡中共的〝一带一路〞，加强美国在该地区的存在。
 <br/>
 <br/>
 在此之前，彭斯9日在《华盛顿邮报》发表文章称，〝我们（指美国和印太地区国家）将抵抗任何威胁我们利益和价值观的人。美国寻求的是合作，而不是控制。〞
 <br/>
 <br/>
 中共推进的〝一带一路〞，被外界认为是利用高额放贷，使参与的贫穷国家陷入债务陷阱，从而被中共掌控。
 <br/>
 <br/>
 据中共外交部和中共官媒11月12日发布的消息，习近平将于11月15日至21日访问巴布亚新几内亚、文莱、菲律宾。期间，习近平还将在17日至18日出席APEC峰会。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：文慧)
</div>
