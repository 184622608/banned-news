### 巴西空战演习 14国百架军机显实力
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 空军周三表示，包括美国在内的14个国家空军，本周将参加在巴西西北部基地举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空战演习.html" target="_blank">
  空战演习
 </a>
 。
 <br/>
 <br/>
 大约100架军事飞机将参与代号为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南方十字.html" target="_blank">
  南方十字
 </a>
 〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空战演习.html" target="_blank">
  空战演习
 </a>
 ，包括来自德克萨斯州美国空军国民警卫队的六架F-16，以及来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 、加拿大、智利、法国、秘鲁和乌拉圭等国家的飞机。
 <br/>
 <br/>
 〝南方十字〞演习控制部门 指挥官FRANCISCO BENTO：〝参与演习的是国际空军，有些国家贡献飞机，有些是观察者。这是对常规和非常规战争所进行的合作演习。〞
 <br/>
 <br/>
 战机演练的模拟情节，是为世界各地潜在的重大事件做准备，包括维护和平与稳定局势的行动，以及在自然灾害发生后提供人道援助等各种应急行动。
 <br/>
 <br/>
 巴西空军准将LUIZ GUILHERME SILVEIRA DE MEDEIROS：〝我们更多的是进行非常规战争的训练。巴西与参与国合作中特别使用这方面的经验。我们已经参与过和平行动....〞
 <br/>
 <br/>
 今年着重强调非常规战争的模式，演习包括从飞机上跳伞、从直升机上滑绳下降等技巧训练。
 <br/>
 <br/>
 新唐人记者陈依心、宇薇综合报导
</div>
