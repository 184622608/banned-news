### 澳洲警方逮捕3男  制止大规模阴谋恐袭
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳大利亚.html" target="_blank">
  澳大利亚
 </a>
 警方周二表示，成功破获一起阴谋
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恐怖袭击.html" target="_blank">
  恐怖袭击
 </a>
 ，并拘捕3名男子，指控他们企图在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨尔本.html" target="_blank">
  墨尔本
 </a>
 发动大规模的攻击。
 <br/>
 <br/>
 澳洲维省警察总长 阿什顿：〝我们有足够的证据采取行动，制止这起恐袭，所以今天我们执行搜捕令，并将三人拘押。〞
 <br/>
 <br/>
 根据警方消息，这三名男子嫌犯年龄分别是21岁，26岁和30岁，是土耳其裔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳大利亚.html" target="_blank">
  澳大利亚
 </a>
 公民，他们的护照在今年初因安保原因已被取消。他们在最近几天，试图采购0.22口径的半自动步枪，并企图在人群密集的地方，实施大规模的恐袭行为。
 <br/>
 <br/>
 澳洲维省警察总长 阿什顿：〝从所获悉的证据得知，嫌犯肯定在寻找人群密集的地方，有很多人聚集的地方，因为他们试图尽可能的杀害更多的人。〞
 <br/>
 <br/>
 警方目前还没掌握预谋恐袭的确切地点，但有充分迹象表明，当时恐袭已近在眼前。
 <br/>
 <br/>
 目前三名嫌犯已被正式指控企图发动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恐怖袭击.html" target="_blank">
  恐怖袭击
 </a>
 ，如果罪名成立，将被判终身监禁。
 <br/>
 <br/>
 新唐人记者陈依心、天羽综合报导
</div>
