### 呼应川普？日本拟停止对华援助要求平等
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 政府消息人士周二（23日）透露，日本将正式宣布停止日本对华〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政府开发援助.html" target="_blank">
  政府开发援助
 </a>
 〞。首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 将于周四（10月25日）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访华.html" target="_blank">
  访华
 </a>
 时告知中方。这项计划实施至今已约40年。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 外相河野太郎在周二举行的记者会上正式宣布，日本对华
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政府开发援助.html" target="_blank">
  政府开发援助
 </a>
 （Official Development Assistance, ODA）计划将以本年度的新项目为终点，画上句号。
 <br/>
 <br/>
 他还强调：〝考虑到中国的经济水平，（这一计划）恐怕已不需要。（未来）将在对第三国的开发援助和人道援助方面，与中国合作。〞
 <br/>
 <br/>
 日本放送协会（NHK）报导称，今后，日方将在平等立场上，讨论新的合作方法，建立〝开发合作对话〞，以支援
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发展中国家.html" target="_blank">
  发展中国家
 </a>
 等方式谋求合作。
 <br/>
 <br/>
 日本内阁官房长官菅义伟周二证实，日本政府将于25日开始的安倍首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访华.html" target="_blank">
  访华
 </a>
 时，与中方协商今后的合作。
 <br/>
 <br/>
 日本政府开发援助始于1979年，是日本政府对某些发展中国家提供的经济援助。其中，对华ODA是长期以来争议较大的议题。
 <br/>
 <br/>
 迄今为止，日本共以有偿资金贷款、无偿资金合作，以及技术合作的方式，向中共提供了合计3兆日元（合约近2,000亿人民币）的帮助。
 <br/>
 <br/>
 <h4>
  中共长期藉〝发展中国家〞名义捞取利益
 </h4>
 <br/>
 美国总统川普（特朗普）曾多次指中共藉〝发展中国家〞地位，从中捞取私利。
 <br/>
 <br/>
 上月底，川普在美国北达科他州法戈市（Fargo, North Dakota）的一场募款集会中，提到希望终止对中共和印度等所谓〝发展中国家〞的经济补贴。
 <br/>
 <br/>
 川普说，〝一些国家被认为是发展中的经济体，因为他们还不够成熟，所以我们给他们支付补贴。〞而中国、印度等一些正在增长的国家也自称为发展中国家，〝在这一分类下，他们获得了补贴〞。
 <br/>
 <br/>
 〝我们必须付钱给他们，这整件事太疯狂了，我们要阻止这种不合理的事。〞川普表示。
 <br/>
 <br/>
 川普尤其指责世贸组织（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 ）是最糟糕的，它使中共用不公平贸易行为，从美国掠夺财富。〝我们不能让中国（中共）每年从美国拿走5000亿美元，并重建他们自己。〞这也是中美贸易冲突不断升级的原因之一。
 <br/>
 <br/>
 美国驻世贸大使在内的经贸官员指出，中国的市场规模显示，中国已不再是发展中国家；更谴责中共操控经济。
 <br/>
 <br/>
 香港城市大学前政治学教授郑宇硕表示，中国现在已经是国际经济体的第二位，从美国的角度来说，中国很难说自己是发展中国家。而且，美国已经把中共定义为战略竞争对手。
 <br/>
 <br/>
 而日本此次终止对华政府开发援助，也被认为与川普的策略不谋而合。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：）
