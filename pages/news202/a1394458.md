### 搜出危险品 肯尼亚逮捕五中国公民
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】10月5日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肯尼亚.html" target="_blank">
  肯尼亚
 </a>
 移民局在首都内罗毕，以威胁国家安全为由逮捕了五名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国公民.html" target="_blank">
  中国公民
 </a>
 。警方称，在嫌疑人的住所找到了迷彩服，金属探测器，催泪瓦斯等物品。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肯尼亚.html" target="_blank">
  肯尼亚
 </a>
 移民局局长亚历山大・穆特希（Alexander Muteshi）表示，从嫌疑人身上搜到了对肯尼亚国家安全构成威胁的物品。
 <br/>
 <br/>
 穆特希说，这五名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国公民.html" target="_blank">
  中国公民
 </a>
 声称计划在肯尼亚开设一家安保公司，但初步调查显示，他们没有获得当地的工作许可，都持旅游签证。
 <br/>
 <br/>
 近年来，中国与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲.html" target="_blank">
  非洲
 </a>
 国家间的商贸往来频繁，中共也加大对非洲〝大撒钱〞行动。
 <br/>
 <br/>
 9月5日，肯尼亚移民局警察在内罗毕曾突袭中共央视旗下的〝中国环球电视网非洲分台〞，带走七名中国籍的编辑、记者，并带走了在同一大楼的〝中非发展基金〞等中资机构和企业的6名人员。
 <br/>
 <br/>
 新唐人电视综合报导
</div>
