### 遏止海洋污染 欧洲议会高票通过限塑法案
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】为遏制塑料制品对海洋污染日益严重的势头，欧洲议会今天（24日）以压倒性多数投票通过限塑令，欧洲联盟所有成员国将禁止使用塑胶餐具等一系列一次性塑胶制品。
 <br/>
 <br/>
 据英国广播公司（BBC）报导，欧洲议会在对这个限塑法案进行表决时，有571票赞同，仅有53票反对。
 <br/>
 <br/>
 据报导，被这项法案列入禁止使用清单的物品包括塑胶餐具、餐盘、吸管、搅拌棒、气球杆、棉花棒等日常用品。该法案同时呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 的食品与饮料业者减少使用塑胶杯、塑料瓶等一次性塑胶容器。
 <br/>
 <br/>
 上述塑料用品之所以被列入禁用清单，主要因为这些类别的塑料产品是最常见污染海洋的塑胶制品，而且已经有了其他材质的替代品，例如纸吸管和硬纸容器。
 <br/>
 <br/>
 法案并要求，对于其他〝尚未有替代品〞的塑胶制品，例如汉堡盒和三明治包装，
 <br/>
 欧盟的每个成员国必须在2025年前减少25%。
 <br/>
 <br/>
 BBC的报导引述一名欧洲议会议员称，如果现在还不采取行动，到了2050年，〝海洋里的塑胶制品恐怕会比鱼还多〞。
 <br/>
 <br/>
 负责提案的欧洲议会议员黎斯（Frederique Ries）则表示，〝对我们的海洋、环境和未来世代，这是场胜利〞。
 <br/>
 <br/>
 限塑法案在欧盟各国正式生效前，仍须通过若干程序，不过从这次在欧洲议会上获高票通过来看，预计以后的程序应该都可以通过。欧盟已对外表态称，希望限塑令可以于2021年前在所有成员国生效。

 <br/>
 <br/>
 今年的〝世界海洋日〞这天（6月8日），联合国新闻网曾发表主题为《保护海洋 向塑料污染宣战》的专题报导。
 <br/>
 <br/>
 该报导指出，海洋覆盖地球面积近四分之三，占地球全部水资源的97％，超过30亿人的生计依赖于海洋和沿海的多种生物，海洋渔业直接或间接雇用了2亿多人。同时，海洋还吸收了约30％人类活动产生的二氧化碳，缓冲着全球暖化的影响。
 <br/>
 <br/>
 然而联合国的数据显示，每年估计有至少800万吨的塑料制品进入到海洋中，相当于平均每秒钟就有一卡车的塑料垃圾被倒入海中，对海洋生物、渔业、旅游业造成严重影响。
 <br/>
 每年还有100万只海鸟和10万只海洋哺乳动物因塑料污染而丧生。
 <br/>
 <br/>
 更严重的是，塑料会被分解成微型碎片，然后被浮游生物所摄入，并最终影响到食物链和人类餐盘中的食物构成。
 <br/>
 <br/>
 联合国秘书长古特雷斯在〝海洋日〞当天发表致辞称，塑料废物阻塞水道，危害以捕鱼和旅游业为生的社区，致使海龟、鸟类、鲸鱼和海豚死亡，并且流到地球上的最偏远地区，充斥人类的整个食物链。人类如不做出改变，塑料废物很快就会在重量上超过海洋中的所有鱼类。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：东野）
