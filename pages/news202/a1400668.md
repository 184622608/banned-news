### 当代的典雅 意大利顶级厨具在台设第二家门市
------------------------

<div class="wysiwyg">
 【新唐人2018年11月24日讯】擅长打造个人特色的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 顶级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_厨具.html" target="_blank">
  厨具
 </a>
 BOFFI，极罕见的在台湾开设第二家门市，该公司总裁Roberto Gavazzi也出席了第二家门市的开幕，才华洋溢的西班牙设计师Patricia Urquiola 为BOFFI 打造的Salinas系列就放在门口，俐落活泼又能多元组合，代表着与台北，迥然不同的特色。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 顶级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_厨具.html" target="_blank">
  厨具
 </a>
 BOFFI在台湾的第二家门市就位于台中市的富豪区七期，开幕当天贵宾云集，总裁Roberto Gavazzi也到场一一介绍团队。
 <br/>
 <br/>
 BOFFI总裁Roberto Gavazzi：〝我非常谢谢在座的各位，还有我们非常棒的团队。〞
 <br/>
 <br/>
 BOFFI在1934年成立，从手工家俱涉足顶级厨具，是目前意大利少数坚持以半手工完成产品的公司。BOFFI曾经与知名的设计师、建筑师合作，包括无印良品设计总监深泽直人（Naoto Fukasawa）、被Newsweek评为〝最受世界尊敬的100位日本人〞的佐藤大（Sato Oki），还有在2016年过世的知名建筑师札哈・哈蒂（Zaha Hadid）。
 <br/>
 <br/>
 Roberto Gavazzi 表示，对合作设计师的要求不多，但必须符合BOFFI的设计理念。
 <br/>
 <br/>
 例如这组放在台中门市的主打产品由西班牙Patricia Urquiola所设计，模组化生产的管状金属可改变为不同样式，不过，BOFFI最擅长处理的就是重达数百公斤的石材与上漆技术的处理。
 <br/>
 <br/>
 BOFFI成立超过84年，在实木、石材等材质的运用下了不少功夫，结合生态永续的概念，受到不少名人喜爱。
 <br/>
 <br/>
 新唐人记者谢心欣综合报导
