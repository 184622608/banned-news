### 【禁闻】不打自招 中共使馆抗议美墨加协议挨讽
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】中共驻加拿大大使馆在10月5号发声明，谴责美国、墨西哥、加拿大三国新签订的贸易协定中的条款，不过和此前中共驻瑞典、驻英国使领馆相比，这次驻加使馆的发声，在国内几乎没有引发关注。
 <br/>
 <br/>
 上周，美国、加拿大和墨西哥宣布达成取代北美自由贸易协定的美墨加新贸易协定（The United States-Mexico-Canada Agreement，USMCA），备受瞩目。
 <br/>
 <br/>
 除了三国的互惠条例，同样引发关注的是，第32章第10条中规定：三方中任何一方如果与〝非市场化经济体〞签署自贸协定，要提前三个月通知其他各方。而其他各方有权在提前六个月通知的条件下，终止适用美墨加新贸易协定，并且用双边协定取代。
 <br/>
 <br/>
 这一条款被认为明显指向中共。10月5号，中共驻加拿大大使馆在官网刊登了发言人杨云东对此的评论。杨云东说，他们〝反对在WTO框架外杜撰‘市场经济国家’和‘非市场经济国家’等概念〞......谴责〝有关国家公然干涉别国主权的霸权行径〞。
 <br/>
 <br/>
 这是近期第三次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共使馆.html" target="_blank">
  中共使馆
 </a>
 在全球关注的事件中发声，然而在中国大陆几乎没有引发关注。而此前中国游客在瑞典〝碰瓷〞，以及央视女记者在英国打人事件中，中共使馆发出类似的谴责，都在国内被高调报导。
 <br/>
 <br/>
 推特上对此嘲笑声一片。有人说，中共〝对号入座，它承认自己是非市场经济国家〞；〝说好的市场经济呢，不打自招啊〞。也有人问〝问题是三国协议，关WTO 什么事，为什么人家自定的协议要符合WTO ?〞，还有人说中共是〝干涉美、加、墨的内政〞。
 <br/>
 <br/>
 时事评论员田园博士表示，美国对〝市场经济国家〞和〝非市场经济国家〞有严格定义。
 <br/>
 <br/>
 时事评论员田园博士：〝什么是非市场经济国家呢？有这么几方面，第一个它就是完全不遵守国际贸易规则，也不遵守基本的经济规则；中央政府有极大的权力干预私营经济，和该国经济的方方面面。那么很显然这个定义一看大家都知道，中共政府对经济的干预那在世界上估计是出了名的。〞
 <br/>
 <br/>
 田园博士举例，中共在2008年、09年国际经济危机时，推出四万亿元的刺激计划，引发了地方债务危机、产能过剩、楼市泡沫等一系列〝后遗症〞。中国的广义货币总量已经超过美欧日的总和，目前中共还在把这种债务模式推向世界，放贷给一些信贷评级很低的国家。
 <br/>
 <br/>
 田园博士：〝在这种情况下呢，当然美国要把中共认定为‘非市场经济国家’。川普总统的理念非常清楚，我不再跟你搞这种所谓的多边协定，我要搞双边协定，这样的话我们大家有很多的自由度。〞
 <br/>
 <br/>
 北京时政观察人士华颇：〝川普不希望中国的商品经由墨西哥和加拿大输入到美国，或者说在墨西哥和加拿大成立公司，来进入到美国。〞
 <br/>
 <br/>
 美墨加协定被认为是美国总统川普在经贸领域重塑〝朋友圈〞的范例，今后美欧、美日如果签订自贸协定，很可能也会加入对非市场经济国家的限制性条款。
 <br/>
 <br/>
 田园博士：〝就是说如果你以后日本和欧盟想跟中共谈自贸协定的话，你也要三思而后行了。这一点我个人认为是川普总统在这个双边协定，推动这个理念的过程中，其实是使出的最有效的一招。〞
 <br/>
 <br/>
 华颇：〝而且川普前几天还发表了一个演说，还有前两天彭斯演说，无异于就像二战结束以后邱吉尔发布的铁幕演讲一样。〞
 <br/>
 <br/>
 川普政府近期再度显示了扭转不平衡贸易的决心，川普在西维吉尼亚州的一次集会上说：〝要么中国开放市场、实行公平贸易，要么我们不跟他们做生意。就这么简单。〞而副总统彭斯在哈德逊研究所的演讲，更被认为是对中共最严厉的警告。
 <br/>
 <br/>
 采访/陈汉 编辑/尚燕 后制/李沛灵
