### 救坠湖少年 马国6消防员被卷入暗流溺毙  生前画面曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】马来西亚一名少年3日与朋友到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_废矿湖.html" target="_blank">
  废矿湖
 </a>
 垂钓，竟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失足坠湖.html" target="_blank">
  失足坠湖
 </a>
 ，当局获报后，当天动员前往救援，其中6名消防员于夜晚时分潜入水中搜救，不料6人都被卷入漩涡中，虽然在场的其他消防员欲将他们拉上岸，无奈湖水力道太大而无法如愿，6人全数罹难。如今，完整的救援影片曝光，令人不胜唏嘘。
 <br/>
 <br/>
 从影片中可见，6名消防员换上蛙人装备在水流湍急的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_废矿湖.html" target="_blank">
  废矿湖
 </a>
 闸门前，慢慢前进，没多久就遇上强大暗流，其中一人不幸被卷入后，其他5人因相互绑着绳子，无法挣脱。
 <br/>
 <br/>
 岸上人员在探照灯的照射下，面对眼前紧急状况，努力的要将6人拉出水面，由于暗流漩涡湍急，花费约30分钟才将人拉出，但他们都早已失去意识，经当地医院紧急抢救仍回天乏术。
 <br/>
 <br/>
 据当地官员表示，〝水闸门出水处虽然水深不深，但因水流湍急，有如冲力强大的瀑布〞，疑是其中一人不小心滑倒，把其他5人一同强拉下水，最终酿成悲剧。
 <br/>
 <br/>
 6名殉职消防员最年轻成员仅24岁，而最年长的人才34岁。他们的遗体在4日早上被送到莎阿南区消拯局，灵柩上覆盖国旗，以表扬亡者对这次事件的牺牲。
 <br/>
