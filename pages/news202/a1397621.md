### 北海道小岛凭空消失 日本领海恐缩小
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】日本北海道北端有一个名为〝鼻北〞的小岛被纪录在地图上，但最近有人前往勘查时却发现小岛不见了。北海道小樽第一管区海上保安总部称，〝有可能是被海浪或水流侵蚀而消失〞，近期将展开调查。
 <br/>
 <br/>
 综合日媒报导，这座小岛名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鼻北小岛.html" target="_blank">
  鼻北小岛
 </a>
 〞，在地图上被标记在距离北海道猿拂村约500米的外海上。曾调查日本各地岛屿，47岁的清水浩史为出版《秘岛图监》下一集，今年9月前往视察鼻北小岛，却没找到。于是向北海道小樽第一管区海上保安总部报告。
 <br/>
 <br/>
 海上保安厅计划着手调查，航海情报课课长辅佐藤井智雄表示，可能是因为风雪等因素，使得小岛受侵蚀而消失。
 <br/>
 <br/>
 负责周边海域开发的木原智彦表示，现在从陆地上看已看不见该岛，在附近海域航行的渔民从渔船上看也说看不见该岛了。一些年纪大的渔民称〝自昭和50年代(上世纪70年代左右)，这周边就有小岛存在的。当时（出海)都要绕路。〞可是现在小岛已经消失了。现在当地渔民在该海域进行海上作业时，也只能依据经纬度及航行经验绕行以避免触礁。
 <br/>
 <br/>
 根据日本当局消息，岛屿测量工作是在1987年实施，记录显示该处有一座高于海平面约1.4米的岛屿。2014年被国家命名〝鼻北小岛〞，是日本158座无人岛之一。这些数据都被明确记录在日本海图及国土地理院的地图中。不料在短短4年内便有可能失去领海判定。
 <br/>
 <br/>
 根据联合国海洋法相关条约，岛屿定义为即使潮涨也会露出水面的陆地区域。领海的范围是以离岛的米数为基点而确定的。如果〝鼻北小岛〞真的消失了，也就意味着日本的领海缩小了，对渔业恐会带来影响。
 <br/>
 <br/>
 小岛的消失见证了沧海桑田，世事多变难以预料。不过，有忧就有喜，日本的小笠原群岛的西之岛，由于火山活动相当活跃，岛屿面积不断扩大，让日本领海范围增加了4平方公里。
