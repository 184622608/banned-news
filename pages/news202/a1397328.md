### 突尼斯首都自杀式炸弹袭击 20伤
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】当地时间周一下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_突尼斯首都.html" target="_blank">
  突尼斯首都
 </a>
 突尼斯市中心发生一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀式炸弹袭击.html" target="_blank">
  自杀式炸弹袭击
 </a>
 ，造成至少20人受伤。
 <br/>
 <br/>
 突尼斯内政部表示，一名30岁的女子在突尼斯市中心主干道的警察巡逻队附近引爆身上炸弹，袭击者本人当场死亡。共有15名警察和5名民众在袭击中受伤，其中包括2名儿童。
 <br/>
 <br/>
 目击者KAMEL BRIKI：〝我听到了爆炸声。一名女子引爆了自己，造成警察受伤，我当时正好在附近的街道，爆炸声非常的大。（爆炸地点）非常接近市政剧院。〞
 <br/>
 <br/>
 突尼斯总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃塞卜西.html" target="_blank">
  埃塞卜西
 </a>
 当天强烈谴责这起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恐怖袭击事件.html" target="_blank">
  恐怖袭击事件
 </a>
 。
 <br/>
 <br/>
 新唐人记者李博文综合报导
