### 声乐大赛周四开锣 上届获奖选手赞水准高
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】新唐人主办的第七届〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全世界华人声乐大赛.html" target="_blank">
  全世界华人声乐大赛
 </a>
 〞将于周四起一连三天，在纽约举行。上届获奖选手、香港女高音刘倪彤称赞大赛水准高，是提升技能、认识朋友的好平台。
 <br/>
 <br/>
 身为英国赫尔大学音乐硕士的刘倪彤，上届成功入围
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_声乐大赛.html" target="_blank">
  声乐大赛
 </a>
 复赛，并荣获最具潜质大奖。她称赞大赛水准很高，有很多世界顶尖歌唱家参赛，是拓宽视野、交流学习的好机会。
 <br/>
 <br/>
 香港女歌唱家刘倪彤：〝比赛你会觉得大家都是竞争对手，但是最后找到很多朋友很友善，而且有些是前辈级的，他们也很愿意去教授，很愿意分享，其实你这样可以唱得更好，真的可以学到很多，而且觉得自己的框框太小了。〞
 <br/>
 <br/>
 新唐人首次将中文歌曲引入世界级大赛，并要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人选手.html" target="_blank">
  华人选手
 </a>
 在决赛中都要演唱一首中文歌曲，她对此也深表认同。
 <br/>
 <br/>
 香港女歌唱家刘倪彤：〝因为始终是我们的母语，我们会更加容易理解歌词，以及如何演绎，我相信能令观众（感受）的层次更加深入。〞
 <br/>
 <br/>
 刘倪彤目前致力于培养年轻歌剧人才，她认为，要成为好的歌唱家，不断学习是非常重要的，希望选手们都能从比赛中获益良多，将美声唱法推向更多的华人社区。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
 <br/>
</div>
