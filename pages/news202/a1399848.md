### 亚太经合会议落幕 史上首次未能达成公报
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】在巴布亚新几内亚首都莫士比港举行的，2018年亚太经合组织峰会（APEC）周日落幕。因美中贸易和投资方面的巨大分歧，导致会议无法达成一致 ，出现史上首次领袖峰会闭幕，未能发布首脑共同宣言。
 <br/>
 <br/>
 周日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 闭幕式上，会议主持人巴布新几内亚总理奥尼尔（O'Neill）表示，这次会议讨论了亚太地区的重大问题和面临的挑战。
 <br/>
 <br/>
 巴布新几内亚总理奥尼尔（O'Neill）：〝今天我们进行了富有成效的讨论，并就影响该地区的重大问题和继续面临的挑战，交换了非常坦率的意见。〞
 <br/>
 <br/>
 美国和中国之间在太平洋地区的竞争成为这次APEC会议焦点。会议上，西方盟国一致反对中国的〝一带一路〞计划。当天稍早，美国、日本、澳大利亚、新西兰，还与巴布亚新几内亚签署了一项17亿美元的援助项目，为巴布亚新几内亚的广泛地区提供电力和互联网，被外界看作是美国及盟国对太平洋地区展示重大战略影响的步骤。
 <br/>
 <br/>
 巴布亚新几内亚有800万人口，其中五分之四居住在城市地区以外。这个项目计划到2030年，帮助巴布新几内亚拥有电力的人口数从13%增加到70%。
 <br/>
 <br/>
 美国副总统彭斯：〝去年，川普总统提出了一个自由开放的印度-太平洋地区的愿景，美国对世界这个地区的承诺从未如此强烈。我们相信今天公告，以及美国在这份联合公告的这部分，证明了美国和我们的企业在这个地区的投资是前所未有的。〞
 <br/>
 <br/>
 新唐人记者徐哲、田瑞综合报导
</div>
