### 〝大篷车难民〞 身陷两国边境桥
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】周六，数千来自洪都拉斯的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 〞滞留在危地马拉与墨西哥边境之间的桥上，等待进入墨西哥境内。数百名警察在墨西哥一侧把守。来看报导。
 <br/>
 <br/>
 周五发生了三千难民冲过危地马拉边境围栏的事件。他们冲向苏恰特河上的桥梁，但是遇到墨西哥防暴警察的人墙阻拦。
 <br/>
 <br/>
 50多人仍冲过警察防线，但是警方使用胡椒喷雾驱退其他人。目前数千人滞留在两国之间的桥梁上。
 <br/>
 <br/>
 警方和移民机构人员分批放人办理难民登记。 登记之后的人可以去庇护所过夜。
 <br/>
 <br/>
 难民 Selvin Flores：〝有点冷，谢天谢地不算太冷。也没下雨，大家没有淋湿。还担心小孩被雨淋，因为大家也没遮拦。还好我们挤在一起，把苫布系在栏杆上为孩子们遮挡一下。〞
 <br/>
 <br/>
 洪都拉斯难民是为了躲避国内的黑帮暴力、毒品泛滥和贫困，离开家园。希望穿越危地马拉和墨西哥，到美国寻求庇护。
 <br/>
 <br/>
 难民 Selvin Flores：〝我们自己也坐成一排，防止有人像上次那样破坏秩序，引起麻烦。〞
 <br/>
 <br/>
 墨西哥当局允许少部分持有护照和签证的人立刻入境。申请难民可以留在墨西哥，但是如果非法越境被抓住将面临关押和遣返。
 <br/>
 <br/>
 新唐人电视综合报导
