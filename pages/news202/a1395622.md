### 刚果埃博拉疫情扩散风险增高 专家被迫撤出疫区
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】最近两个月，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲刚果.html" target="_blank">
  非洲刚果
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃博拉疫情.html" target="_blank">
  埃博拉疫情
 </a>
 持续恶化，已出现的211个病例中有135人已经死亡。该地区近期暴力事件与武装袭击事件激增，导致在疫区进行防疫工作的国际专家被迫撤离，世界卫生组织随即把埃博拉区域传播的风险从〝高〞提升至〝非常高〞。
 <br/>
 <br/>
 据CNN报导，世卫组织发言人贾撒列维克（Tarik Jasarevic）日前透露，由于暴力事件与武装袭击的激增，在过去的几周里，美国疾病管制与预防中心驻扎在疫区防止病毒扩散的专家遭受严重干扰，防疫工作已经难以为继，几天后专家们将不得不撤离该区域。
 <br/>
 <br/>
 有关机构预估，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃博拉疫情.html" target="_blank">
  埃博拉疫情
 </a>
 可能从刚果传播到乌干达、卢旺达、南苏丹和布隆迪，但扩散到非洲以外的风险比较低。不过，世卫组织仍决定于17日在日内瓦举行紧急委员会会议，以确认非洲的埃博拉疫情是否构成国际突发公共卫生事件。
 <br/>
 <br/>
 资料显示：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃博拉病毒.html" target="_blank">
  埃博拉病毒
 </a>
 （Ebola virus）是丝状病毒科的其中一种病毒，可导致感染者出现恶心、呕吐、腹泻、肤色改变、全身痠痛、体内出血、体外出血、发烧等症状，而罹患埃博拉出血热的人大多会因为中风、心肌梗塞、低血容量性休克或器官衰竭而死亡。
 <br/>
 <br/>
 埃博拉是人畜共通病毒，主要的感染途径是透过患者体液传染，如血液、汗、呕吐物、排泄物、尿液、唾液或精液等，目前并无飞沫感染的证据。尽管世界卫生组织苦心研究，至今仍没有辨认出任何有能力在爆发时存活的动物宿主。
 <br/>
 <br/>
 因为这种此病毒1976年首次爆发的部落就在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲刚果.html" target="_blank">
  非洲刚果
 </a>
 境内的埃博拉河附近，因此被称为埃博拉病毒，该病毒最近四次爆发的所在地也在埃博拉地区。
 <br/>
 <br/>
 因为埃博拉的高致命力，加上目前世界上还没有任何疫苗被证实有效，埃博拉被列为生物性危害第四级病毒，也同时被视为是生物恐怖主义的工具之一。
 <br/>
 <br/>
 （记者李明综合报导/责任编辑：东野）
