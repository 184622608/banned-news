### 疑天灯酿祸 韩储油槽大火烧掉266万公升汽油
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】韩国高阳市一家国营石油公司的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_储油槽.html" target="_blank">
  储油槽
 </a>
 ，7日上午11时突然发生火灾，烈焰、浓烟直窜天际，所幸储油槽位于远离住宅的偏远地区，没有人伤亡。火势直至17小时才扑灭，但已烧掉可装满250辆油罐车266万公升
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汽油.html" target="_blank">
  汽油
 </a>
 。经连日来的调查后，这场意外竟是一名斯里兰卡籍工人施放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天灯.html" target="_blank">
  天灯
 </a>
 酿祸。
 <br/>
 <br/>
 据韩国媒体报导，事发当时，当局派出680名消防员灌救，并出动224台灭火载具，包括消防直升机。并警告附近居民留在家中，关闭所有门窗。
 <br/>
 <br/>
 当局表示，引发大火的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_储油槽.html" target="_blank">
  储油槽
 </a>
 高8.5公尺、半径为28.4公尺，灌救17小时，直到8日凌晨3时58分才扑灭火势。共燃烧了266万公升
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汽油.html" target="_blank">
  汽油
 </a>
 ，油槽内大约还剩下440万公升。
 <br/>
 <br/>
 <b>
  相关视频：
 </b>
 <center>
 </center>
 <br/>
 经当局近日展开联合调查后，这场意外竟是一名在储油槽附近工地工作的27岁斯里兰卡籍人士点燃了1盏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天灯.html" target="_blank">
  天灯
 </a>
 升空，天灯随后降落在储油槽旁的草皮上，疑引发火势，而火焰从储油槽的通风系统窜入，才引起这起意外。
 <br/>
 <br/>
 目前警方已逮捕这名涉有重嫌的斯里兰卡工人。
 <br/>
