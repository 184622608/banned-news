### 美大使批联合国：选更多人权恶棍进人权理事会
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】周五（10月12日），即将卸任的美国驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 大使妮基．黑利（Nikki Haley）抨击刚刚结束的联合国选举结果，指出联合国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权理事会.html" target="_blank">
  人权理事会
 </a>
 正选入更多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侵犯人权.html" target="_blank">
  侵犯人权
 </a>
 的成员，而美国退出该组织显然是正确的选择。
 <br/>
 <br/>
 周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 大会新选举了18名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权理事会.html" target="_blank">
  人权理事会
 </a>
 成员，而这些人员基本上事先已得到承认，因为没有其它国家挑战他们入选的合理性。
 <br/>
 <br/>
 这18名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新成员.html" target="_blank">
  新成员
 </a>
 中，有6名来自被总部设于瑞士的人权组织〝联合国观察〞（UN Watch）列为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侵犯人权.html" target="_blank">
  侵犯人权
 </a>
 的国家——索马里，厄立特里亚，喀麦隆，孟加拉，菲律宾，巴林。
 <br/>
 <br/>
 这些国家将与其它一些人权记录不佳的国家——古巴、中共、卡塔尔和沙特阿拉伯等，一起成为联合国人权理事会成员。
 <br/>
 <br/>
 尽管阿根廷、捷克和丹麦等一些人权记录较好的国家也被选入，但黑利认为，上述人权记录不佳的国家加入人权理事会，正说明美国今年早些时候退出该组织是正确的。
 <br/>
 <br/>
 黑利指出，联合国大会居然再次没有争议地挑选人权记录不佳的国家进入人权理事会，这种缺乏标准的行动在继续破坏该组织，并再次说明为什么美国选择退出是正确的。
 <br/>
 <br/>
 黑利表示，美国将继续支持可以使人权理事会变得可信的改革。更重要的是，美国将继续担当世界人权领袖，而 不管人权理事会由哪些可疑的成员构成、以及做出怎样糟糕的决定。
 <br/>
 <br/>
 黑利将于明年1月辞职。周二，川普总统在白宫椭圆形办公室与黑利一起宣布了这一消息。但截至目前，白宫还没决定谁将接替黑利的角色。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
