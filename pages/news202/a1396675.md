### 10月24日国际新闻简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】下面请看一组新闻简讯。
 <br/>
 <br/>
 10月24日，川普总统签署了国会两党罕见共同支持的法案，抵制毒品。使过度依赖阿片类药物的人获得更广泛的治疗。
 <br/>
 <br/>
 法案扩大了在医疗保险中获得治疗的途径；禁止邮寄毒品、比海洛因毒性更强的芬太尼; 并提供联邦资金解决危机。
 <br/>
 <br/>
 2017年美国有72,000人因毒品过量致死。
 <br/>
 <br/>
 为清理海洋污染，欧洲委员会通过提案，禁止一次性塑料制品。计划到2021年禁止10个已有替代材料的塑料制品。
 <br/>
 <br/>
 目前每年2500万吨塑料废品，仅回收四分之一。欧盟要求到2025年做到回收90％的一次性塑料制品，但是一些成员国担心难以实施。
 <br/>
 <br/>
 10月23日，盎格鲁-保加利亚探险队考古学家称在黑海发现了最古老的完整沉船，有2400多年历史。
 <br/>
 <br/>
 英国南安普顿大学使用远程深水摄像机记录了沉船的图像，称它为〝希腊贸易船〞。这个发现将帮助考古界认识古代造船与航海技术。
</div>
