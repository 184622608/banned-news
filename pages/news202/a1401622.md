### 川普与默克尔会面 承诺解决贸易问题
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】周六，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和德国总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默克尔.html" target="_blank">
  默克尔
 </a>
 举行会谈，除了双边贸易，还讨论了世贸组织、乌克兰和叙利亚等问题。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 表示，两国军事关系非常好，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易问题.html" target="_blank">
  贸易问题
 </a>
 未来会解决。
 <br/>
 <br/>
 美国总统川普：〝我们存在巨大的贸易不平衡，但我们会解决这个问题。我认为，随着时间推移，两国的贸易会越来越好。〞
 <br/>
 <br/>
 德国总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默克尔.html" target="_blank">
  默克尔
 </a>
 表示，双方将讨论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公平贸易.html" target="_blank">
  公平贸易
 </a>
 和世贸组织（WTO）的问题，以及乌克兰、叙利亚和俄罗斯违反《中导条约》（INF Treaty）等问题。
 <br/>
 <br/>
 德国总理默克尔：〝我非常高兴，我们将进行双边会议。〞
 <br/>
 <br/>
 川普称，当晚将和习近平参加小组会议，主要讨论贸易问题。
 <br/>
 <br/>
 美国总统川普：〝我们将讨论贸易，可能还有其他事情，但主要是贸易。这是一个非常重要的会议。〞
 <br/>
 <br/>
 参加周六晚的会议之后，川普就会返回华府。
 <br/>
 <br/>
 美国总统川普：〝今晚我们会进行会谈，然后我将返回华盛顿，我们可以在这周或者之后某个时候再会谈。〞
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
</div>
