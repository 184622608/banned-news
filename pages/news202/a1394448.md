### 德国葡萄大丰收 20年来最佳酿酒季
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】经过炎热、干燥的夏季，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_葡萄.html" target="_blank">
  葡萄
 </a>
 大丰收，有望成为20年来最佳酿酒时节，喜欢喝德国葡萄酒的人士可有福了。一起来看看。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 去年冬季多雨，今年夏季多为高温少雨的天气，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_葡萄.html" target="_blank">
  葡萄
 </a>
 的生长提供了极佳条件。
 <br/>
 <br/>
 今年葡萄获得大丰收，有望酿制出10.7亿升葡萄酒，比过去10年增长了23%。
 <br/>
 <br/>
 德国葡萄酒学院发言人Ernst Buescher:〝今年的葡萄品质是过去二十年最好的。产量上来讲1999年也大丰收，但是品质不如今年。今夏少雨结的葡萄非常好，给白葡萄酒带来特殊香气，酿出酒体饱满的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红葡萄酒.html" target="_blank">
  红葡萄酒
 </a>
 。〞
 <br/>
 <br/>
 德国有13个葡萄产地，今年产量都大幅提升，从5%到19%不等，在莱恩中部地区的葡萄产量甚至增长了64%。
 <br/>
 <br/>
 莱茵黑森、法尔兹等地都以出产的葡萄酒闻名，喜欢喝德国葡萄酒的人士今年可以窖藏不少好酒了。
 <br/>
 <br/>
 新唐人电视安心综合报导
</div>
