### 世界七大奇景〝佩特拉古城〞被淹 近4千游客惊逃
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】中东国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_约旦.html" target="_blank">
  约旦
 </a>
 连日来豪雨不断，造成多处地区发生水患与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土石流.html" target="_blank">
  土石流
 </a>
 ，9日还传出，世界七大奇景之一的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佩特拉古城.html" target="_blank">
  佩特拉古城
 </a>
 在土石流的夹击下，变成一片汪洋，吓得上千名民众纷纷逃离现场，游客直呼：〝以为快死了！〞。期间传出3700名旅客一度受困。最新消息指出，目前游客都陆续撤出。
 <br/>
 <br/>
 综合媒体报导，从网友上传的影片中可以看到，大量泥沙与洪水冲进峡谷间的通道，现场水流湍急，游客只能踩在一旁的岩石块上，沿着峡谷旁惊险的慢慢往外爬逃离，若一个不小心有可能就会失足被冲走。
 <br/>
 <br/>
 据了解，事发当时现场不断打雷下雨，随后短短几分钟内，大量洪水就这样暴冲进古城内的所有低洼地区。而当时约有3700多名游客在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佩特拉古城.html" target="_blank">
  佩特拉古城
 </a>
 中旅游，场面混乱。
 <br/>

 一名泰国旅客表示，洪水来袭时，她跑到地势较高的地区逃难，所幸与她同行的父母最后也平安无事的离开古城。
 <br/>
 <br/>
 她说，当地人与业者纷纷用驴子把旅客载出古城，并要求所有人马上撤离。
 <br/>
 <br/>
 一名日本女游客就说，水流的速度非常快又深，还听说有行李箱被冲走，直呼：〝真的好恐怖！〞
 <br/>
 <br/>
 一名有15年经验的美国健走老手克里斯(Chris Mandrick)说，这是他这辈子看过最大的洪水。他说，当时不断打雷、下大雨，在短短5分钟内，洪水就淹进佩特拉古城的入口，当地人也直呼，这种水势真的是从来没看过。
 <br/>
 <br/>
 有旅客痛批佩特拉古城的营运单位认为，如果考虑到下雨的状况，就不应该开放游客进场。
