### 韩议员：金正恩掌权6年购买奢侈品逾40亿美元
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】日前，有韩国国会议员披露，对一些相关政府资料分析发现，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 自上台以来的6年间，朝鲜当局共花费约40亿美元购买汽车、化妆品、贵金属等奢侈品；仅去年从中国进口的奢侈品就花费了6.4亿美元，严重违反了联合国制裁决议。
 <br/>
 <br/>
 据韩国《中央日报》报导，当地时间10月21日，韩国国会议员尹相炫（Yoon Sang-hyun）披露，通过对韩国政府公开的对朝鲜禁运奢侈品名单与中国海关的贸易统计资料进行分析可发现，从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 2012上台到2017年，朝鲜官方共花费40.0429亿美元购买奢侈品。
 <br/>
 <br/>
 据称，朝鲜购买的奢侈品中，电子设备所占的份额最大，花费共计20.1198亿美元；其次是购买汽车，共花费13.9025亿美元；然后是酒类产品，花费了1.6545亿美元。
 <br/>
 <br/>
 此外，朝鲜当局还购买了大量的化妆品、香水、手表、皮草制品、乐器、贵重金属等其他奢侈品。
 <br/>
 <br/>
 拥有自由韩国党籍的尹相炫在一份声明中指出，这些奢侈品，除了被金正恩一家直接消费外，还被金正恩用来当作奖励下属的礼物，送给那些支持他的独裁统治的朝鲜精英家庭，而汽车和手表是金正恩〝礼物治国〞最常使用的物品。
 <br/>
 <br/>
 尹相炫并披露，仅去年从中国进口的奢侈品的金额就达到6.4亿美元，占朝鲜从中国进口额的17.8%。
 <br/>
 <br/>
 声明中指出，朝鲜政府如果把去年购买奢侈品的外汇，用来从国际市场购买大米，可买到165多万吨，〝是朝鲜短缺粮食80.2万吨的2倍〞。
 <br/>
 <br/>
 尹相炫直言，〝朝鲜的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_粮食短缺.html" target="_blank">
  粮食短缺
 </a>
 ，不是因为国际社会的援助不够，而是金正恩根本没有想让居民吃饱饭。〞
 <br/>
 <br/>
 过去中共官方多次对外界宣称，该政府拥护并遵守了联合国对朝鲜作出的制裁决议。如果尹相炫上述有关朝鲜从中国进口大量奢侈品的说法得到证实的话，意味着中共一直在欺骗国际社会。到目前为止，中共官方未对尹相炫的相关指控作出任何回应。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
