### 印尼: 狮航坠毁前航程曾出技术故障
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】再来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼空难.html" target="_blank">
  印尼空难
 </a>
 搜寻与失事调查进展。印尼当局继续调查狮子航空公司客机失事原因，有初步资料显示，失事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮航客机.html" target="_blank">
  狮航客机
 </a>
 JT610在坠毁前的一个航程，曾经出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_技术故障.html" target="_blank">
  技术故障
 </a>
 。
 <br/>
 <br/>
 根据当地媒体转述峇里机场负责人的说法，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮航客机.html" target="_blank">
  狮航客机
 </a>
 JT610在出事前，曾接载旅客由峇里飞往雅加达，飞行员在客机起飞后几分钟，通报出现速度和高度的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_技术故障.html" target="_blank">
  技术故障
 </a>
 ，但随后飞行员又通报故障已解决，客机飞行正常，客机在星期天晚上安全降落。
 <br/>
 <br/>
 但这架客机却在第二天早上由雅加达再次起飞后，不久就坠毁在爪哇海，全机189人遇难。
 <br/>
 <br/>
 随着失事客机的一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑匣子.html" target="_blank">
  黑匣子
 </a>
 被找到，印尼当局表示，希望尽快解开失事之谜。
 <br/>
 <br/>
 印尼国家运输安全委员会主席 索尔贾安托．贾约诺：〝寻获这个黑匣子，可让我们找出飞机失事之谜。〞
 <br/>
 <br/>
 遇难者的家属也表示，希望了解客机失事的原因。
 <br/>
 <br/>
 遇难者家属 Johanes Sayersz：〝找到黑匣子让(遇难者)家人感到宽慰，因为我们能够知道到底发生什么了。〞
 <br/>
 <br/>
 目前，狮航不做任何回应，表示空难原因正在调查中。
 <br/>
 <br/>
 新唐人记者陈依心、尚敬综合报导
</div>
