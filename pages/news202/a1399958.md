### 脱欧草案 获英国企业和欧盟部长支持
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】周一，英国首相特雷莎・梅在英国工业联合会上讲话，重申〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 〞。联合会总裁和主席表示支持。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟部长.html" target="_blank">
  欧盟部长
 </a>
 也普遍同意〝脱欧草案〞。
 <br/>
 <br/>
 周一，在英国工业联合会的年会上，特雷莎・梅表示，相信英国会达成协议。
 <br/>
 <br/>
 英国首相特雷莎・梅：〝我有信心我们可以在（欧洲）理事会达成协议。〞
 <br/>
 <br/>
 特雷莎・梅表示，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 〞同意和欧盟建立综合自由贸易区，零关税、所有商品领域不收取费用，这将有利于英国商业发展。
 <br/>
 <br/>
 英国首相特雷莎・梅：〝世界快速发展，但地理位置不是。欧洲永远是我们最接近的商品市场，确保我们拥有自由流动的边界至关重要。〞
 <br/>
 <br/>
 英国工业联合会总裁在年会上发言，支持〝脱欧草案〞。
 <br/>
 <br/>
 英国工业联合会总裁CAROLYN FAIRBAIRN：〝当其他国家正在形成激烈竞争的未来，我们在威斯敏斯特的神经中心，似乎生活在自己狭隘的世界里。他们似乎正在玩高风险的游戏，结果可能是意外的‘无协议’。〞
 <br/>
 <br/>
 英国媒体报道称，工业联合会主席约翰・艾伦也支持〝脱欧草案〞。
 <br/>
 <br/>
 周一，欧盟首席英国脱欧谈判代表巴尼耶表示，27位欧盟国家部长同意〝脱欧草案〞。
 <br/>
 <br/>
 欧盟首席英国脱欧谈判代表巴尼耶：〝我很高兴部长们今天支持整个计划，特别是成员国支持‘脱欧草案’。〞
 <br/>
 <br/>
 荷兰外交部长STEF BLOK：〝现在有一个概念性的‘脱欧协议’，我们认为它令人满意，是可行的。〞
 <br/>
 <br/>
 双方争论焦点是过渡期问题。
 <br/>
 <br/>
 欧盟首席英国脱欧谈判代表巴尼耶：〝欧盟方面我们仍需决定，欧盟同意延长过渡期的内部程序。〞
 <br/>
 <br/>
 特雷莎・梅希望过渡期在2022年英国大选时截止。巴尼耶表示，必须明确过渡期限。
 <br/>
 <br/>
 欧盟首席英国脱欧谈判代表巴尼耶：〝我们的想法是，过渡期的可能延期，需要和英国就延期的财政贡献达成协议，（延期）不能无限期，必须确定。〞
 <br/>
 <br/>
 11月25日，欧盟将举行峰会，讨论〝脱欧草案〞。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
