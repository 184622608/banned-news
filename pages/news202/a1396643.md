### 飓风威拉扫过墨西哥 周三减弱进内陆
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三级飓风.html" target="_blank">
  三级飓风
 </a>
 〝威拉〞周二在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 西北部登陆，风速达到每小时193公里，引发洪水。当地居民数千人被疏散。
 <br/>
 <br/>
 周二晚间，飓风〝威拉〞在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 度假小镇马萨兰附近的海滩登陆，掀起的巨浪最高达到8到10米。
 <br/>
 <br/>
 据美国国家飓风中心消息，飓风〝威拉〞登陆时虽然减弱到三级，但是近年来从太平洋进入墨西哥的最强飓风之一，带来狂风暴雨，电线吹断，树木刮倒。
 <br/>
 <br/>
 当地部分航班被取消，机场里几乎看不到旅客。
 <br/>
 <br/>
 大雨使得河流水位上升，引发洪水。
 <br/>
 <br/>
 当地学校停课。
 <br/>
 <br/>
 数千人被疏散，其中也包括游客。
 <br/>
 <br/>
 美国游客卡斯特罗每年都来这里，这次意外遇到飓风。
 <br/>
 <br/>
 美国游客 卡斯特罗：〝当我们办理旅馆入住手续时，他们说它(飓风)看起来像是三到五级。我们说〝哇！〞，我们大吃一惊。〞
 <br/>
 <br/>
 周三早上，飓风继续向墨西哥内陆移动，风势减缓。
 <br/>
 <br/>
 新唐人记者王羽音、天羽综合报导
</div>
