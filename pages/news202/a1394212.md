### 国际刑警组织主席孟宏伟 去中国后失踪
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】法国警方周五宣布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，9月底前往中国后下落不明，法国警方将对此事件展开调查。
 <br/>
 <br/>
 法国媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 的妻子周五报案，说孟宏伟9月29日前往中国后，与家人失去联络。
 <br/>
 <br/>
 国际刑警组织总部设在法国里昂，孟宏伟夫妇居住在当地。
 <br/>
 <br/>
 法国警方已就孟宏伟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 案展开调查。国际刑警组织尚未对此作出回应。
 <br/>
 <br/>
 孟宏伟现年64岁，任中共公安部副部长，2016年11月当选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 ，任期四年，是首个担任这一职位的中国人。有人权组织担心，北京可能利用该组织追捕外逃的异见人士。
 <br/>
 <br/>
 去年，孟宏伟仕途开始下滑。12月，孟宏伟被免去中共国家海洋局副局长和中共海警局局长职务。今年春天，他再次被免去公安部党委委员职务。
 <br/>
 <br/>
 孟宏伟长期在公安部任职，2004年4月升任中共公安部副部长、党委委员，曾经是周永康的副手之一。
 <br/>
 <br/>
 新唐人记者王羽音、宇微综合报导
 <br/>
</div>
