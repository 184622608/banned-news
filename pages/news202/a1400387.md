### 美媒曝中共APEC霸道行径 彭斯演讲被断直播信号
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】本届亚太经济合作峰会（APEC)，中共在美国反击下铩羽而归。外媒曾报导中共试图操控APEC的行为，包括禁止外媒采访，强闯主办国外交部。日前美媒披露中共还曾施压其他国家，并被怀疑在美国副总统演讲时切断网络信号。
 <br/>
 <br/>
 《华盛顿邮报》11月20日文章指，在这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 上，中共官员彷佛在自己的地盘般来来去去，并企图使用各种霸凌手段获得他们想要的东西。
 <br/>
 <br/>
 华顿专栏作家罗金（Josh Rogin）在文章中称，一名出席峰会的美国官员告诉他，〝撒泼外交〞似乎已经成了中共跟国际社会打交道的惯例。
 <br/>
 <br/>
 文章透露，在会议结束前的一个星期内，中共代表团一再使用挑衅、霸凌、偏执和奇怪的伎俩，以展现其〝支配地位〞，并施压主办国巴布亚新几内亚和其他所有国家满足中方的要求，强闯主办国外交部长办公室只是其中一例。
 <br/>
 <br/>
 在中方阻挠下，本届APEC峰会破天荒地首次没有发表联合公报。而当峰会失败的消息宣布时，中共代表团竟然当众鼓掌庆祝，令其他国家的外交官十分反感。
 <br/>
 <br/>
 文章还披露，11月17日，习近平和美国副总统彭斯（Mike Pence）先后在邮轮上发表演说。但当彭斯严厉抨击中共的演讲开始5分钟后，在岸边国际媒体中心聆听演讲的记者团的网络突然被切断，直到演讲进入尾声时才恢复，导致在中心的记者没有听到演讲的大部分内容。
 <br/>
 <br/>
 罗金在文章称，官员告诉他，不能确定是中共动了手脚，但会展开调查。
 <br/>
 <br/>
 不过，如果网络中断是人为因素的话，没有谁比中共更具备作案动机和条件。中共长期对峰会主办国巴布亚新几内亚进行经济援助，更是此次APEC峰会的主要赞助方。而中方提供的赞助中，很可能也包括技术支持和网络设备。
 <br/>
 <br/>
 中共在国内的新闻管制已经轻车熟路，正在借金钱外交将舆论控制输出到海外。
 <br/>
 <br/>
 据此前媒体报导，中共在APEC峰会期间举行了太平洋国家领导人峰会，但只允许中共官媒采访，所有外媒记者，包括主办国记者都禁止入场。中方还建议外国媒体事后〝引用新华社通稿〞。
 <br/>
 <br/>
 另外，北京领导人在巴布亚新几内亚莫尔兹比港口一条公路开通仪式上演讲时，也只允许中共中央电视台录音。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
