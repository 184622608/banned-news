### 迎假日季  柏林植物园点亮灯饰 灿烂夺目
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】今年已渐渐到了尾声，欧美国家也迎来了假日季。在德国柏林的植物园，提早点亮了漂亮的圣诞灯饰，吸引游客，感受浓浓的节日气氛，一起去看一下。
 <br/>
 <br/>
 超过1000名游客周四夜晚来到植物园，赶在第一天体验
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞花园.html" target="_blank">
  圣诞花园
 </a>
 的气氛。
 <br/>
 <br/>
 活动的组织者狄克曼表示，11月中旬是点亮圣诞灯饰的最佳季节。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柏林植物园.html" target="_blank">
  柏林植物园
 </a>
 圣诞灯饰负责人 狄克曼：〝两星期前，我们为打造圣诞花园的柏林、德累斯顿、斯图加特举行记者会。温度当时是摄氏23、24度，我有点担心。冬天到底会不会来？所以今天能穿戴围巾和冬天夹克，我真的很开心。〞
 <br/>
 <br/>
 今年的柏林圣诞花园，共有超过30个创意主题，使用了150万个灯泡。多彩的灯光结合柔和的音乐，营造出浪漫的圣诞气氛，令人流连忘返。
 <br/>
 <br/>
 柏林居民  Sonja Drachholz：〝此时此地真的有圣诞的感觉，虽然现在还很暖和，秋季和夏季多年来一直都比较晚，不过这里很有圣诞味，在圣诞节前感受到迷人的气氛。〞
 <br/>
 <br/>
 去年，柏林植物园在圣诞季接待了超过14万5千名游客。今年的圣诞灯饰展将一直持续到明年1月6号。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
