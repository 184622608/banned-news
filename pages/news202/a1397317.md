### 狮航客机坠毁189人罹难 印尼官员塞车逃过一劫
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼廉航.html" target="_blank">
  印尼廉航
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮子航空.html" target="_blank">
  狮子航空
 </a>
 周一（29日）起飞后仅13分钟，就在首都雅加达东北的爪哇海上空失事，机上189人无人生还。不过，一名财政部官员因交通堵塞未赶上这趟坠毁航班而幸运逃过一劫。
 <br/>
 <br/>
 <h4>
  交通问题救他一命
 </h4>
 <br/>
 法新社报导，印尼财政部官员苏堤万（Sony Setiawan）与同事每周都搭乘狮航（Lion Air）这架JT610班机。但出事当天，因交通堵塞，他抵达机场时间太晚而没赶上航班，结果却幸运躲过一劫。
 <br/>
 <br/>
 苏堤万告诉法新社，〝我经常乘坐JT610 ，我的朋友和我总是乘坐这架飞机。〞他说，出事当天不知道为什么收费公路上的交通非常糟糕。之前他经常在凌晨3点就可以抵达雅加达，但当天早上6点20分他才到达机场，因此错过了航班。
 <br/>
 <br/>
 苏堤万最终乘坐另一架飞往邦加槟港（Pangkal Pinang）的航班，当他安全降落时才得知飞机失事，他的朋友们不幸遇难。他立即打电话给他的家人报平安，虽然感恩自己逃过一劫，但苏堤万对朋友们的遇难感到非常伤心。他说，〝我第一次听到（这个消息时），我哭了。〞他说，〝我知道我的朋友们在那趟（JT610）班机上。〞
 <br/>
 <br/>
 印尼财政部长穆利亚尼（Sri Mulyani）在雅加达参加一个部门活动后，开始寻求有关20多名财政部工作人员的信息。
 <br/>
 <br/>
 <h4>
  飞机坠海深约30米
 </h4>
 <br/>
 这架载有189人的印尼狮航JT610航班周一坠落大海。据飞机当天飞行状况图显示：JT610航班的高度、速度和爬升率都存在明显异常。在短短的13分钟内，飞机虽然经历多次爬升和下降，颠簸异常，但高度反常地维持在了5,000英尺（约1524米）左右。根据前几日正常条件下的飞行记录，在飞抵失联点附近时飞机的飞行高度实际上应超过10,000英尺（约3048米）。
 <br/>
 <br/>
 飞机坠入海中深约98至115英尺（30米至35米）。印尼国家搜救局副局长Nugroho Budi Wiryanto表示，包括士兵、警察和当地渔民在内的约300人参与搜索。
 <br/>
 <br/>
 救援人员将从海上找到的相关物品，从氧气瓶到个人物品，如钱包、手机、现金和背包等，摆放在雅加达港口的防水油布上。
 <br/>
 <br/>
 交通部长Budi Karya Sumadi表示，已将9个尸袋送往医院进行身分鉴定，但官员说这些尸袋内仅有部分人体，而不是完整的遗体。路透社目击者看到有更多尸袋送抵港口。
 <br/>
 <br/>
 当地时间周一晚，搜救机构运营主管Bambang Suryo表示，潜水员已停止搜索，但声纳船隻将继续寻找机身，据信许多受害者被困大海。
 <br/>
 <br/>
 印尼官方表示，飞行员在起飞后不久要求折返雅加达，但获准后，地勤人员随即与飞机失去联系。
 <br/>
 <br/>
 搜救机构负责人Muhmmad Syaugi在新闻发布会上称，没有收到来自该飞机应急发射器发出的任何遇险信号。
 <br/>
 <br/>
 气象部门的资料显示，事发时的天气状况良好，不但视野清晰，而且没有强气流及强风，初步排除天气原因导致客机失事。有航空专家认为事件十分神秘，希望早日寻回黑盒解开谜团。初步调查显示，客机的油箱曾漏油并爆炸。
 <br/>
 <br/>
 <h4>
  机上至少有23名财政部官员
 </h4>
 <br/>
 印尼方面表示，初步信息显示，机上共载有189人，其中乘客181人（男性124人，女性54人，及3名儿童），空乘8人（含2名飞行员）。机上至少有23名政府官员，4名国家锡矿PT Timah的员工，以及Timah子公司的3名员工。一名狮航官员说，一名意大利乘客和一名印度飞行员也在遇难飞机上。
 <br/>
 <br/>
 坠毁狮航JT610航班是一架几乎全新的波音737 MAX 8客机，当时正前往Bangka-Belitung锡矿区首府邦加槟港。国家搜索和救援机构发言人Yusuf Latief说，可能没有幸存者。
 <br/>
 <br/>
 狮航方面发表声明说，失事客机今年8月才开始为狮航服役，客机机长和副驾驶累计飞行时间达到11,000个小时。
 <br/>
 <br/>
 据悉，失事客机是由中国民生投资股份有限公司于今年8月13日借予狮航，两日后注册及取得适航证明，飞行时数只有800小时。
 <br/>
 <br/>
 客机制造商波音公司对空难深感悲痛，准备为事故提供技术协助。受空难影响，澳洲政府呼吁官员及承办商，暂时不要安排乘坐狮航客机，会在空难调查完成后再检讨指示。
 <br/>
 <br/>
 过去20年，印尼航空公司几乎每年都会发生一起空难事故，共造成至少532人死亡。2014年12月，当时印尼亚航一架空客A320在从泗水飞往新加坡时坠入爪哇海中，机上162人全部遇难。
 <br/>
 <br/>
 印尼总统维多多表示，已下令国家交通安全委员会彻查事件，搜救人员会尽力寻找遇难者，希望国民一起继续祈祷。他也向死难者家属致哀，希望他们节哀。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
