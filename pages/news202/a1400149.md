### 右派纪念西班牙佛朗哥 遗体将重新安置
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】11月20日是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 前国家元首和首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛朗哥.html" target="_blank">
  佛朗哥
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逝世周年.html" target="_blank">
  逝世周年
 </a>
 ，他的支持者周二在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马德里.html" target="_blank">
  马德里
 </a>
 附近的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_烈士谷.html" target="_blank">
  烈士谷
 </a>
 陵墓举行纪念活动。目前，西班牙政府正计划将佛朗哥遗体迁出烈士谷。
 <br/>
 <br/>
 周二，右派人士在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_烈士谷.html" target="_blank">
  烈士谷
 </a>
 陵墓教堂为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛朗哥.html" target="_blank">
  佛朗哥
 </a>
 举行弥撒。〝为了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 运动〞领导者古迪雷斯表示，他们会继续努力阻止将佛朗哥遗体迁出陵园。
 <br/>
 <br/>
 〝为了西班牙运动〞领导者古迪雷斯：〝我们一直会记住他并尊重他，因为这是我们的权利。〞
 <br/>
 <br/>
 目前，西班牙新掌权的社会主义政府已于9月得到议会同意，将把佛朗哥的遗体从烈士谷迁出，并将那里改名为〝法西斯主义受害者纪念馆〞，安葬3万多名西班牙内战阵亡的军人。
 <br/>
 <br/>
 佛朗哥家人表示，希望把他的遗体迁到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马德里.html" target="_blank">
  马德里
 </a>
 的阿尔穆德纳罗马天主教大教堂。今年10月，该教堂旁曾爆发反对佛朗哥的数千人抗议。
 <br/>
 <br/>
 43年前去世的佛朗哥是个备受争议的人物。
 <br/>
 <br/>
 很多右派人士认为他铁腕打击共产主义或社会主义者，力图恢复一个传统的西班牙，并为西班牙二战后的经济快速发展奠定了基础。
 <br/>
 <br/>
 但不少左派人士则批评他实行独裁统治，称他是法西斯，并对他杀死大量反对者或共产主义者恨之入骨。
 <br/>
 <br/>
 佛朗哥二战时执行亲德国和意大利的政策，但保持中立。二战后只和美国保持盟友关系。他还恢复了西班牙王室，扶持了一名奉行民主的国王。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
