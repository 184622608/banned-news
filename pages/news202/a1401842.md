### 墨西哥总统欲加强执法 多方合作解决移民潮
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】12月1日，墨西哥新总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥夫拉多尔.html" target="_blank">
  奥夫拉多尔
 </a>
 就职，一上台他就开始解决美墨双方都关心的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 问题。周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）对美墨关系的未来发表了积极愿景。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥总统.html" target="_blank">
  墨西哥总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥夫拉多尔.html" target="_blank">
  奥夫拉多尔
 </a>
 （Andres Manuel Lopez Obrador）周六（12月1日）在宪法广场（索卡洛，Zocalo）就职的同时，上千名大篷车成员搬进了距离美墨边境更远的新避难所。
 <br/>
 <br/>
 新唐人特约记者Kimberly Hayek：〝蒂华纳新的移民帐篷距离美墨边境大约10英里，警察告诉我们这个避难所大约有1,500人，还将有超过3千人从墨西卡利过来，那里距离我们所在的蒂华纳东部有80英里。〞
 <br/>
 <br/>
 蒂华纳市长透露，目前已经有超过7千大篷车成员聚集在该市。
 <br/>
 <br/>
 墨西哥蒂华纳市长Juan Manuel Gastélum：〝根据我们的法律，应该是联邦政府负责解决移民问题，这是为什么我们（没有行动）；而且蒂华纳也从未经历过目前的状况。〞
 <br/>
 <br/>
 由于美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 不断施压，墨西哥已采取措施将部分大篷车成员先安置在该国等待，但在周末一些人强行越过边境进入美国。一个总部位于加州的共产主义组织BAMN11月29日在社交媒体发布信息，鼓动大篷车成员在12月1日墨西哥总统就职当天冲击边境。不过一些人并不认同。
 <br/>
 <br/>
 大篷车成员、洪都拉斯人Juan Medrano：〝不、不，我真不觉得可行，这让我有生命危险。事实上，他们在用生命在赌。我不会这么做。〞
 <br/>
 <br/>
 为解决
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 潮，奥夫拉多尔领导的墨西哥新政府已与萨尔瓦多、危地马拉和洪都拉斯三国政府达成协议，阻止中美洲人继续前往墨西哥。周一，他宣布正与美国和加拿大协商一个三方合作项目，共同出资解决中美洲及墨西哥移民问题。
 <br/>
 <br/>
 奥夫拉多尔誓言解决该国长期面临的贫困及暴力问题，计画在墨西哥北部边境打造低税收经济特区，把墨西哥劳动力留在国内。同时，推出一项加强国民警卫队的战略方案，第一阶段将调集约6万军队及警力打击犯罪，其中包括禁毒。
 <br/>
 <br/>
 去年墨西哥有2万5千人被杀害，其中7-10月4个月时间就有1万人。
 <br/>
 <br/>
 川普在今天的推文中说，无论怎样，不允许非法移民进入我们国家。还有，停止贩毒！我们（与墨西哥总统）未来将有很好的合作！
 <br/>
 <br/>
 新唐人记者李兰、特约记者Kimberly Hayek纽约、蒂华纳连线报导
</div>
