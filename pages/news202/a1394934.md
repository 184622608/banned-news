### 拒做同性恋蛋糕 英最高法院：非歧视
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北爱尔兰.html" target="_blank">
  北爱尔兰
 </a>
 由基督宗教家庭经营的亚设烘焙公司，之前因拒绝制作有〝支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_同性婚姻.html" target="_blank">
  同性婚姻
 </a>
 〞装饰字句蛋糕被判有罪。经上诉后，当地时间周三（10日），英国最高法院推翻原判决，称此举并非歧视。
 <br/>
 <br/>
 中央社报导，最高法院表示，亚设公司（Ashers Baking Company）〝反对的是蛋糕上的装饰字句，而非传达讯息者的任何个人特质，或与其相关的任何人。〞
 <br/>
 <br/>
 最高法院首席大法官赫尔（Brenda Hale）周三表示，今天的裁决不应被解读为纵容歧视，〝因为一个人的宗教、性别、残疾、性取向、宗教或信仰而拒绝为其服务，这令人深感羞耻，也是对人类尊严的侮辱。不过本案并未发生上述情况。〞
 <br/>
 <br/>
 亚设公司的名称取自圣经旧约中人物的名字，在英国各地雇用80名员工。该公司2014接获
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_同性恋.html" target="_blank">
  同性恋
 </a>
 社运人士盖瑞斯‧李（Gareth Lee）的订单，但因有〝支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_同性婚姻.html" target="_blank">
  同性婚姻
 </a>
 〞装饰字句而拒绝为其制作蛋糕。当年正值欧洲和西方愈来愈能接受同性伴侣结婚。
 <br/>
 <br/>
 自荷兰2001年首开先例后，目前已有15国允许同性婚姻，包括英国在内。不过由英国统治的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北爱尔兰.html" target="_blank">
  北爱尔兰
 </a>
 并非同性婚姻法的适用地区。
 <br/>
 <br/>
 盖瑞斯‧李的委任律师艾伦（Robin Allen）表示，本案受到英国各地媒体的极大关注，让他的委托人处境很尴尬。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
