### 挑南瓜过万圣节 英国家庭享受亲子之乐
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】10月的最后一天就是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万圣节.html" target="_blank">
  万圣节
 </a>
 了，到南瓜田里挑选一个大南瓜，先做节日装饰，之后还可以一饱口福。一起去英国肯特郡去感受一下。
 <br/>
 <br/>
 滚南瓜,头顶南瓜,抱南瓜,孩子们想出各种各样的方式,把自己挑选的南瓜放进小车里,之后带回家。
 <br/>
 <br/>
 10月底
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万圣节.html" target="_blank">
  万圣节
 </a>
 就要来了,南瓜是必不可少的装饰。
 <br/>
 <br/>
 三年来，英国肯特郡农民查理的22英亩南瓜田吸引了很多家庭来挑选自己喜爱的南瓜。
 <br/>
 <br/>
 这里的南瓜和葫芦有70种,大小不同,形状各异。最大的有三四十公斤。
 <br/>
 <br/>
 这些瓜可以放上几个月不坏。万圣节的时候当装饰,之后又可以享口福,无论煲汤还是烤着吃,味道都很赞。
 <br/>
 <br/>
 查理的南瓜田十月份开始对外开放。对于他来说,让人们来瓜田自己选南瓜还有不同的意义。
 <br/>
 <br/>
 南瓜田主查理：〝不仅是因为我喜欢南瓜，而是摘南瓜能让一家人享受乡村，在田野里呼吸新鲜空气。〞
 <br/>
 <br/>
 瓜田里到处都回荡着孩子们的欢笑声，艾米也带着小女儿来到这里摘瓜。
 <br/>
 <br/>
 摘南瓜者艾米：〝一家人一起来真的很好。这些南瓜都很好看。所以，很高兴用它们装饰房子。〞
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
