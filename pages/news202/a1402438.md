### 孟晚舟求保释细节曝光  愿戴电子脚镣 上交护照
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 副董事长兼首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 ，在当地时间12月7日在卑诗省最高法院出席保释
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_听证.html" target="_blank">
  听证
 </a>
 会。其代表律师请求法官给予保释时说，孟有健康问题，不宜长期被拘，此外她还愿意在保释期间戴上电子脚镣，并上交所有护照。法庭没有做出裁决，听证将在下周继续进行。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被指涉及故意隐瞒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 与旗下香港公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Skycom.html" target="_blank">
  Skycom
 </a>
 的关系，而Skycom有向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 出售设备，违反美国制裁伊朗的法例。她12月1日于加拿大被捕，并在7日出席保释
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_听证.html" target="_blank">
  听证
 </a>
 会。
 <br/>
 <br/>
 孟晚舟的代表律师马丁（David Martin）表示，加拿大可相信她不会逃走，因此举会令其父任正非因此受辱以及尴尬。此外她也〝爱中国，不会让中国蒙羞〞。此外，孟晚舟的丈夫及女儿都在温哥华，故此她不会逃走。
 <br/>
 <br/>
 马丁还说，孟有高血压和睡眠窒息症等健康问题，又指其愿意放弃仅有的两本护照，包括中国护照，和在被捕时遭扣押的香港特区护照。此外，孟愿意在保释期间戴上电子脚镣接受监控，并以2个总值1400万加元的在加物业作抵押。
 <br/>
 <br/>
 不过，检控官认为，孟晚舟虽在温哥华有两间房屋，但只用作度假，每年夏天仅逗留数周，在与不在加拿大也没有太多有意义的连系。
 <br/>
 <br/>
 检方还指出，孟晚舟在美国的每一项控罪最高刑期是30年，这令她有潜逃回中国的充分动机。而且孟拥有大量资金来源，再多的保释金对她而言都会显得微不足道。此外，孟晚舟的儿子在美国波士顿上学，自〝Skycom〞2017年4月遭美国调查以来，孟就再没入境美国，可见她是一直有意逃避执法部门的刑事调查。
 <br/>
 <br/>
 由于法官未对孟晚舟的保释申请作出裁决，听证将在12月10日继续进行。在此之前，孟继续处于拘押状态。
 <br/>
 <br/>
 （记者蓝铭浩报导/责任编辑：戴明）
 <br/>
