### 乌克兰大军火库爆炸 橘光笼罩夜空 万人疏散陆空停摆
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 北部一座大型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军火库.html" target="_blank">
  军火库
 </a>
 9日当地凌晨3点30分左右发生大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸.html" target="_blank">
  爆炸
 </a>
 ，橘色亮光笼罩黑暗夜空。当地上万人紧急疏散。爆炸发生后，方圆20公里范围内的空域关闭，铁路和公路交通停驶。目前爆炸规模以及受影响的地区范围仍有待厘清。
 <br/>
 <br/>
 综合媒体报导，事发地点位在基辅以东约176公里处的切尔尼戈夫（Chernigov）地区第6军械库。
 <br/>
 <br/>
 从网路社交媒体可见网友拍下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸.html" target="_blank">
  爆炸
 </a>
 震撼场面，橘色亮光笼罩夜空，轰轰巨响不绝于耳，距离现场50公里的地方都可听到爆炸声。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军火库.html" target="_blank">
  军火库
 </a>
 爆炸后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 军方表示已对事发区域采取了〝限制措施〞。并对事发区域进行陆空交通封锁。
 <br/>
 <br/>
 乌克兰国家紧急情况部立即成立紧急应变中心，并表示，已有大约1万人被疏散。相关部门也派遣220名应急人员和60名技术人员到现场，武装部队负责人也动身前往关切。
 <br/>
