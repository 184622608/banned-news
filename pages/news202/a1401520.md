### G20首日川普会5国领袖 美日印三方战略会谈
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 首日，除出席全体成员会议外，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 还与四国领袖进行了会晤，其中美、日、印三方会谈强调了彼此之间的战略伙伴关系。之后，川普与韩国总统文在寅也进行了双边会谈。
 <br/>
 <br/>
 30日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 总统马克里在布宜诺斯艾利斯正式迎接18国首脑与欧盟代表，就全球经济合作及挑战展开磋商。
 <br/>
 <br/>
 他首先与老朋友美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 进行了会晤。川普与马克里父亲曾是生意伙伴，两人都在沃顿商学院深造。双方的会谈以贸易和军购为主。马克里感谢美国及川普政府给予的巨大帮助，特别是在促进阿根廷经济发展上。
 <br/>
 <br/>
 阿根廷总统 马克里：〝阿根廷的主要投资者是美国公司，我们希望他们继续投资阿根廷，支持我们的人才、帮助我们增加好工作。〞
 <br/>
 <br/>
 去年美国促成国际货币基金组织对阿根廷的贷款协议以助其渡过货币贬值难关。
 <br/>
 <br/>
 结束与马克里的会谈，川普与日本首相安倍晋三就贸易及朝鲜等问题交换意见，特别在美日贸易间的赤字问题上正在合作解决。
 <br/>
 <br/>
 美国总统 川普：〝我们的关系相当好，也许是历史上双方关系最紧密时期。他们正在与我一道解决赤字问题，希望能尽快平衡赤字。〞
 <br/>
 <br/>
 之后印度总理莫迪加入，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美日印.html" target="_blank">
  美日印
 </a>
 度三方进行了短暂会谈，莫迪强调了与美国、日本的战略伙伴关系，表示三国有共同的民主价值理念。他说日本、美国、印度三国名字的首字母（Japan, America and India-JAI）在印度语中代表〝成功〞的意思。
 <br/>
 <br/>
 美国总统 川普：〝我们称之为三边（关系），三方关系非常好、非常强。我认为我们和印度、日本的关系都是前所未有的强而有力。〞
 <br/>
 <br/>
 日本首相安倍则希望三方在印太区域有更密切的合作。
 <br/>
 <br/>
 由于俄罗斯与乌克兰的紧张局势，川普取消了与普京的会晤，于是与澳大利亚总理莫里森进行了首次会面。
 <br/>
 <br/>
 美国总统 川普：〝我们刚刚彼此认识，不错。我认为我们将会有很好的关系。〞
 <br/>
 <br/>
 澳大利亚总理 莫里森：〝不仅因为战略合作，在经济及人民之间都有（好的）关系。〞
 <br/>
 <br/>
 莫里森在抵御中共在经济、技术等领域的渗透上属于鹰派力量，促成禁止华为、中兴参与澳洲5G网络建设议程。美国国内目前在对待中共的态度上、特别是贸易战以来，鹰派立场也成为主导。
 <br/>
 <br/>
 19国加欧盟的G20首脑会晤，代表了全球2/3人口和85%的经济总量。此次会晤东道主马克里表示，希望在尊重各国历史、文化的前提下，从差异中促进对话。
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
