### 【禁闻】英公民吁政府撤销央视英国执照
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国公民.html" target="_blank">
  英国公民
 </a>
 韩飞龙（Peter Humphrey）因曾在中国被强迫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电视认罪.html" target="_blank">
  电视认罪
 </a>
 ，  日前向英国电信监管机构（ofcom）递交投诉书，要求撤销中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
  央视
 </a>
 和其国际部门〝中国环球电视网〞在英国的营业执照。
 <br/>
 <br/>
 据〝美国之音〞报导，韩飞龙和他的华裔妻子卢英曾在2013年作为私人侦探，受雇调查英国制药公司葛兰素史克（GSK)在中国涉嫌贿赂的事件。
 <br/>
 <br/>
 他们在向葛兰素史克提交了调查报告后不久，就被中共以非法获取中国公民个人信息为由，判刑两年。随后，韩飞龙夫妇在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
  央视
 </a>
 上公开认罪。
 <br/>
 <br/>
 11月23号，在伦敦召开的记者会上，韩飞龙表示，他和妻子公开认罪是在被胁迫下进行的。第一次录像前，他被迫服用镇静剂，中共官方要求他照本宣读他们写在大字报上的话；第二次是以接受央视女记者采访的形式进行的，他说，〝央视女记者访问我像是在质询，请问记者是这样的吗？〞
 <br/>
 <br/>
 韩飞龙夫妇的认罪镜头，后来通过央视的国际部门中国环球电视网在英国播出。
 <br/>
 <br/>
 韩飞龙认为，央视的做法违反了英国法律，23号当天，他向英国电信监管机构ofcom递交了投诉书，要求撤销央视和中国环球电视网在英国的营业执照。
 <br/>
 <br/>
 美国华府人权律师叶宁表示，外国人在中国开私家侦探是经过中共允许的，但由于调查外企和中共官员触碰到了中共的底线，因此才遭到打压。
 <br/>
 <br/>
 美国华府人权律师叶宁：〝现在他（韩飞龙）提出取消央视这个执照，这样的一种作法我觉得，他也是为了英国的国家利益，在进行维护正义的行动。〞
 <br/>
 <br/>
 曾遭中共当局关押的瑞典人权活动人士达林（Peter Dahlin）在记者会上表示，国际人权组织〝保护卫士〞（Safeguard Defenders）记录了央视播出的100多起逼供。其中至少有8人的被迫认罪过程在英国播出过。
 <br/>
 <br/>
 旅美人权活动家、《北京之春》顾问刘青：〝中共的央视，它实际上是中共进行社会迫害言论压制，0436它是其中的一部分。逼迫他们抓的但很多的人到电视台自我栽赃，自我认罪，只有这样0512进行认罪表现之后呢，它们才给他一点生路。〞
 <br/>
 <br/>
 旅美人权活动家、《北京之春》顾问刘青表示，央视逼迫认罪，实际上就是在展览共产党自己的罪恶，因为共产党的统治最需要的就是社会的恐怖。
 <br/>
 <br/>
 〝保护卫士〞在今年6月份的一份长达八十页的报告中指出，央视不仅为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电视认罪.html" target="_blank">
  电视认罪
 </a>
 提供平台，同时也是中共警方制作、剪辑和播出这些电视认罪的积极合作者。
 <br/>
 <br/>
 报导说，央视目前正在扩大它在欧洲的业务，新的制作中心预定将在今年底明年初在伦敦开张。
 <br/>
 <br/>
 英国电信监管机构ofcom表示，他们会优先处理韩飞龙的投诉，如果发现央视有任何违法行为，就会采取行动。
 <br/>
 <br/>
 刘青：〝西方来说它是一个民主和法制的社会，再有西方的考虑会更全面更多一些。只有非常过硬的证据和非常过硬是事实，才可能得到真正的严肃的处理和解决。如果在这些方面有所缺失，或者英国屈从于利益的需要，这个事情都不会成为很大的问题。〞
 <br/>
 <br/>
 今年9月，美国正式要求中共媒体〝新华社〞和央视国际部〝中国环球电视网〞登记为〝外国代理人〞。这一举措被外界称为是美国向中共〝大外宣〞打响的第一枪。
 <br/>
 <br/>
 在这种大环境下，英国如何选择，备受关注。
 <br/>
 <br/>
 采访编辑/常春/周天
