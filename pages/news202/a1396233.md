### 起飞3小时玻璃现裂痕 冰岛航空急降加国军用机场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】一架原定于19日晚间从美国佛州奥兰多（Orlando），飞往冰岛首都雷克雅维克（Reykjavik）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冰岛航空.html" target="_blank">
  冰岛航空
 </a>
 客机，飞了3个小时，机长发现驾驶舱的玻璃竟出现裂痕，于是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_紧急降落.html" target="_blank">
  紧急降落
 </a>
 在加拿大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魁北克.html" target="_blank">
  魁北克
 </a>
 军用机场。
 <br/>
 <br/>
 综合媒体报导，这架加上机组员共155人的波音757-200客机，飞到加拿大上空时，机长发现驾驶舱的玻璃有裂痕，于是宣布将尽快降落，于晚间10点20分，平安降落于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魁北克.html" target="_blank">
  魁北克
 </a>
 市以北约200公里的巴戈特维尔（Bagotville）军用机场。
 <br/>
 <br/>
 据一位乘客霍夫（Harrison Hove）在推特上发文称，航空公司的员工估计该裂痕大约有20公分长。
 <br/>
 <br/>
 由于巴戈特维尔机场是军用、国内线机场，没有入境柜台，因此加拿大边境服务局的人员得上机为乘客办理入境手续。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冰岛航空.html" target="_blank">
  冰岛航空
 </a>
 在官方网站表示，编号FI 688客机因〝技术性问题〞，转飞巴戈特维尔，机上的乘客及机组人员全数都被安排到饭店休息，并同时安排另一架客机，将他们接往目地。
 <br/>
