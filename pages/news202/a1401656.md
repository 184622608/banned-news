### 彭斯与伊万卡率团观礼前夕 美驻墨领馆遭攻击
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 和总统川普的女儿兼顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊万卡.html" target="_blank">
  伊万卡
 </a>
 ，1日(周六)率代表团出席墨西哥新总统奥夫拉多尔就职典礼。不料，稍早传出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国驻墨西哥领事馆.html" target="_blank">
  美国驻墨西哥领事馆
 </a>
 遭爆裂物攻击，所幸未造成人员身亡，但因攻击时间敏感，调查工作已由联邦单位接手。
 <br/>
 <br/>
 有关当局1日表示，事发在11月30日晚间，墨西哥第2大城瓜达拉哈拉（Guadalajara），美国领事馆遭爆裂物攻击，爆炸将一面外墙炸出一个40公分大洞，但无人受伤。
 <br/>
 <br/>
 检方消息来源说，监视录影机有捕捉到有一个人朝大楼丢掷爆裂物后逃逸。尽管大楼有大批警卫，嫌疑人仍逃过追捕。
 <br/>
 <br/>
 瓜达拉哈拉所在的墨西哥西部省份哈里斯科（Jalisco）检察官办公室在推特（Twitter）上说：〝调查工作已由联邦单位接手，他们会在适当时间公布最新发展相关资讯。〞
 <br/>
 <br/>
 1日早上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）和总统川普的女儿兼顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊万卡.html" target="_blank">
  伊万卡
 </a>
 （Ivanka Trump）将飞抵墨西哥市，率美国代表团出席墨国新总统奥夫拉多尔（Andres Manuel Lopez Obrador）就职典礼。
 <br/>
 <br/>
 奥夫拉多尔是墨西哥70多年来，首位左翼总统，媒体称，奥夫拉多尔就职，将成为墨西哥的转折点。
