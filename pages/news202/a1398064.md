### 朝鲜扬言不取消制裁将恢复核武 蓬佩奥：清楚谈判对手
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】朝鲜近日扬言，如果美国不取消对它的经济制裁，它可能会扩大它的核武器库。美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 周日（11月4日）称，对这个威胁并不担忧。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 美东时间周日接受《福克斯新闻频道》采访时说，他下周将在纽约与朝鲜第二号人物金英哲会晤，恢复去核化问题的谈判。但他没有透露具体时间地点。
 <br/>
 <br/>
 蓬佩奥称这次谈判是个〝好机会〞，可以争取确定朝鲜可能终止核武计划的时间和方式。
 <br/>
 <br/>
 美国总统川普（特朗普）今年6月在新加坡与朝鲜领导人金正恩举行峰会，双方签署了一项概况声明，要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜半岛.html" target="_blank">
  朝鲜半岛
 </a>
 实现去核化。但在此后的几个月中，朝鲜终止核武器计划的进展有所放缓。
 <br/>
 <br/>
 蓬佩奥之后4次访问朝鲜，上周五（11月2日）他在《福克斯电视台》上说，在他与金正恩的会谈中，金正恩明确表示愿意去核。
 <br/>
 <br/>
 但蓬佩奥也表示，对朝鲜去核的核查至关重要。他说：〝我们必须确保做到这一点。我们不能仅仅听任何人说的话。还有大量工作，但是我相信，我们将维持经济压力，直到金委员长履行了他今年6月在新加坡对川普总统做出的承诺。
 <br/>
 <br/>
 对此，朝鲜外交部在当天发布声明扬言，如果美国不解除经济制裁，朝鲜将考虑恢复其原来的政策，同时发展核武器和经济。
 <br/>
 <br/>
 不过，朝鲜并未威胁取消目前正在与美国进行的谈判，而是在声明中说改善关系与制裁相互违背。
 <br/>
 <br/>
 这之前朝鲜外交部美国研究所所长权正根也在朝鲜官媒《朝中社》发表评论，批评美国不顾朝鲜半岛局势变迁，坚持对其制裁，并暗示将重启经济与核武的并进路线。
 <br/>
 <br/>
 蓬佩奥周日回应说，〝我对此（威胁）并不担忧〞，〝我们的重点十分明确。我们清楚我们的谈判对手，清楚我们的谈判立场。川普总统十分清晰地阐述了他的立场〞。
 <br/>
 <br/>
 蓬佩奥说，〝在我们实现最终目标（终止朝鲜的核计划）前不会取消经济制裁〞。美国的目标是在川普第一个任期（2021年1月）期满之前能够签署最终的协议。
 <br/>
 <br/>
 蓬佩奥说，尽管谈判进展缓慢，美国与朝鲜的关系依然存在有利的迹象。
 <br/>
 <br/>
 他说：〝我们没有再看到导弹试验〞，〝没有出现过核试验。我们还拿到了交还的（韩战）美军遗骸。这些都是良好的步骤〞。
 <br/>
 <br/>
 据悉，上个月韩国政府考虑取消部份针对朝鲜的制裁，川普则表示：〝没有我们的同意他们不会这么做。〞首尔当局随即收回提议。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
