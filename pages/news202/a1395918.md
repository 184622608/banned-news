### 〝中国客每小时都在吃〞 国际邮轮应对有绝招
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】近年来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国游客.html" target="_blank">
  中国游客
 </a>
 屡屡在海外爆出一些国人尴尬蒙羞的举动——随地吐痰乃至随地大小便，公共场合争抢座位大声喧哗，或在飞机上任性争吵乃至大打出手，以至于让中国媒体无可奈何地把这些现象归结为中国游客的〝海外综合症〞。不过，也有一些中国游客普遍具有的习性，让那些与中国游客打交道的外国公司一边抱怨却又一边暗喜。
 <br/>
 <br/>
 曾有英国媒体报导称，不少
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际邮轮.html" target="_blank">
  国际邮轮
 </a>
 抱怨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国游客.html" target="_blank">
  中国游客
 </a>
 太能吃，为此他们甚至不得不拆掉邮轮上一些设施，按照中国游客的需求进行邮轮改建。
 <br/>
 <br/>
 据英国《金融时报》报导，地中海邮轮公司总裁对该媒体抱怨称，中国游客乘船旅行时普遍都太爱吃东西，他们一日三餐之间没有明显的间隔，几乎〝每小时都在吃〞，〝他们根本停不下来〞。这位国际邮轮公司的总裁甚至哀叹道：并质疑这样吃下去，〝我们如何能承受？〞
 <br/>
 <br/>
 该报导称，尽管多间国外邮轮公司都在抱怨中国游客〝根本停不下来〞的餐饮习惯，但这些中国游客却给邮轮带来了丰厚可观的利润。
 <br/>
 <br/>
 报道称，有统计显示，平均每位中国客在邮轮上的消费可达200美元，大部分都是用于购买吃食或其他物品，轮船公司的管理层纷纷预估，未来这一数字还将不断增加。
 <br/>
 <br/>
 因此，国际邮轮的管理者们一边抱怨中国游客的饮食习惯难伺候，一边也绞尽脑汁迎合中国游客的喜好和需求。不少邮轮特别在船上提供夜宵并供应大多数中国游客都喜欢的火锅，部分邮轮公司更拆掉部分中国客不感兴趣的酒吧和SPA等设施，在空间有限的邮轮上改建成餐厅和购物中心，以及一些娱乐设施。
 <br/>
 <br/>
 据报导，公主邮轮的旅游产品中就包括中式火锅、海上日出太极等中国特色的服务；而在以天津为母港的黄金公主号游轮上，曾特意聘请了来自中国北方的主厨，推出一系列迎合中国北方游客口味的餐食。
 <br/>
 <br/>
 （记者欣然报导/责任编辑：秦锋）
