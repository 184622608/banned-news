### 东窗事发翻版中兴？贸战关键时刻美抓捕华为CFO
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】观众朋友大家好，欢迎收看新唐人全球新闻。加拿大媒体周三披露，中国科技大厂华为的首席财务长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 近日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大被捕.html" target="_blank">
  加拿大被捕
 </a>
 ，可能被引渡到美国，引发各方强烈关注。带您了解事件的详细动态。
 <br/>
 <br/>
 12月1日，加拿大当局应美国要求，在温哥华拘留了华为公司首席财务长、华为创始人任正非的长女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 。美国政府正寻求引渡她。由于孟晚舟申请了〝禁止报导令〞，加拿大无法公布更多细节。
 <br/>
 <br/>
 最早披露消息的加拿大《环球邮报》报导，孟晚舟涉嫌违反美国对伊朗的出口管制。
 <br/>
 <br/>
 2010年，由于伊朗违规发展核项目，威胁美国国安，美国宣布，禁止对伊朗出口包含美国军民两用零件的设备，违者可能失去美国出口许可。
 <br/>
 <br/>
 但美国商务部工业安全局，后来获得大陆〝中兴通讯〞2011年制定的两份机密文件，显示中兴公司利用多家影子公司做掩护，继续向伊朗等国家出口美国禁运物资。文件提到，另一家代号为F7的中国公司也在这么做。文件中对F7的描述，如试图收购美国三叶公司，和华为完全一致。
 <br/>
 <br/>
 2016年3月，美国商务部公布了这两份密件，并宣布制裁中兴。10位美国议员认为F7就是华为，要求也展开调查。
 <br/>
 <br/>
 2016年6月，美国商务部对华为发出行政传唤，要求其配合调查，是否违反美国的出口管制。
 <br/>
 <br/>
 2017年，纽约联邦检察官加入调查。
 <br/>
 <br/>
 2018年4月，美国司法部对华为展开刑事调查。
 <br/>
 <br/>
 叶宁：只要是在美国出口管制名单下面的技术，产品，规定了不准直销或转口到特定国家的话，当事人只要事实触犯了这个法律，就是被逮捕被起诉被判刑坐牢的对象。
 <br/>
 <br/>
 此前中兴受到的是美国行政处罚，而华为受到刑事调查，意味其潜在案情更为严重。如果最终罪名成立，不仅华为公司将面临更严厉的制裁；涉案人员和主管也可能面临刑事指控。
 <br/>
 <br/>
 谢田：美国政府肯定是非常谨慎，它一定是有证据了，并且把证据提供给加拿大警方了，加拿大警方才会应美国的要求来拘捕。
 <br/>
 <br/>
 华为公司是全球最大的电讯设备制造商，也是中国5G技术的领头羊。同时，华为也是中共金盾工程等监控系统，最重要的硬件提供商。
 <br/>
 <br/>
 而华为高度依赖美国的技术和零组件，其50家核心供应商中，至少18家是美国公司。未来如果受到美国出口禁令，影响巨大。
 <br/>
 <br/>
 谢田：华为可能成为中兴第二，肯定会受到美国政府的制裁，如果再加上芯片、关键组件禁运，可能让华为一蹶不振。华为的全球销售，市场份额，都会产生巨大的影响。
 <br/>
 <br/>
 对华为的调查历经2年多，最终在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 关键时刻东窗事发。中共外交部要求美加立刻放人。外界也关注，华为案的走向。
 <br/>
 <br/>
 谢田：美国对华为的处理，可能跟中兴一样，也可能不一样。因为华为比中兴，对西方商业运作的冲击更大。我想这对中国一个教训就是，你可以跟美国人做生意，跟世界做生意，但是你要诚诚实实的做。
 <br/>
 <br/>
 孟晚舟的保释听证会将于本周五举行。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
