### 加拿大蒙特利尔圣诞游行 天国乐团获赞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大蒙特利尔.html" target="_blank">
  加拿大蒙特利尔
 </a>
 周六迎来了一年一度的迎新年圣诞大游行，数万观众驻足观看。法轮大法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天国乐团.html" target="_blank">
  天国乐团
 </a>
 受邀参加大游行，获得各界人士的赞誉。看本台记者发自加拿大蒙特利尔的报导。
 <br/>
 <br/>
 第68届蒙特利尔圣诞大游行，周六中午在市中心圣凯瑟琳大街登场。
 <br/>
 <br/>
 五彩缤纷的圣诞花车，体现了法裔古典文化的特色。游行团体竞相呈现最精彩的装扮及表演，卡通人物，神话经典，锣鼓声鸣，舞姿翩翩，迎接2019新年的到来。
 <br/>
 <br/>
 由众多华人组成的法轮大法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天国乐团.html" target="_blank">
  天国乐团
 </a>
 ，成为本届游行中阵容最大的团体。嘹亮的鼓乐，靓丽的服饰，整齐的步伐，为传统古典的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞游行.html" target="_blank">
  圣诞游行
 </a>
 注入了一缕清新的气息。
 <br/>
 <br/>
 据法轮大法天国乐团消息，大约有150名来自美国纽约、渥太华及蒙特利尔的天国乐团成员组成了本届游行方阵。
 <br/>
 <br/>
 游行组委会主任Catherine Rivard：〝天国乐团的表演非常棒！乐团阵容大，演奏技艺高，我们非常高兴他们能来参加游行。〞
 <br/>
 <br/>
 天国乐团为现场观众演奏了圣诞歌、铃儿响叮当、法轮大法好、法轮圣王和法正乾坤等曲目。
 <br/>
 <br/>
 新闻工作者Tim Sargeant：〝天国乐团非常杰出！我很喜欢他们的服饰。很壮观的乐团，非常棒！〞
 <br/>
 <br/>
 电脑科技教师Melina Chuan：〝天国乐团很专业啊，而且人数也很多啊。〞
 <br/>
 <br/>
 华裔观众Jacklincy：〝天国乐团非常好，非常棒！〞
 <br/>
 <br/>
 据悉，近年来天国乐团多次受邀参加蒙特利尔圣诞大游行，并屡获各界的赞誉。
 <br/>
 <br/>
 新唐人记者睿明、安妮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大蒙特利尔.html" target="_blank">
  加拿大蒙特利尔
 </a>
 报导
