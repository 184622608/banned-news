### 强风雨肆虐意大利 〝水城〞威尼斯也遭殃
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】周二，大雨和强风
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_橙色警报.html" target="_blank">
  橙色警报
 </a>
 连续第三天袭击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 ，使这个南欧国家几乎陷入瘫痪，死亡人数升至9人。威尼斯老城75%被水淹。
 <br/>
 <br/>
 周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雨.html" target="_blank">
  暴风雨
 </a>
 继续在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 肆虐，周一风力时速超过100公里，这在意大利非常罕见。位于意大利中部的阿布鲁佐地区和拉齐奥地区发布了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_橙色警报.html" target="_blank">
  橙色警报
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首都罗马.html" target="_blank">
  首都罗马
 </a>
 也在警报区内。
 <br/>
 <br/>
 意大利国家消防局称，在过去两天，消防局在全国接到七千多起与暴风雨有关的警报，其中还有一名消防员被树砸死。
 <br/>
 <br/>
 意大利北部的旅游港口城市拉帕洛，强风和海浪摧毁了码头，数十艘大型游艇被损坏。树被吹倒，一些停在路边的车辆被倒下的树或被风吹落的金属板砸坏。救援人员使用切割机，试图将压在车上的金属板切成小块，让汽车重见天日。著名旅游城市水城威尼斯变成了水淹威尼斯，城区75%被淹，最高水位超过5英尺，是有史以来第四高水平。最热门的景点之一圣马克广场也变成一片汪洋。
 <br/>
 <br/>
 不过，在膝盖深的水中，很多威尼斯的店铺照常开门，店主人穿着雨靴忙里忙外，游人则在水中逛街购物，休息喝咖啡。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
