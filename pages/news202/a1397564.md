### 俄罗斯安全局遭自杀炸弹袭击  一死三伤
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】10月31日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 一名17岁青年在俄国联邦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安全局.html" target="_blank">
  安全局
 </a>
 大厅里引爆自制炸弹自杀，造成三名警员受伤。
 <br/>
 <br/>
 一名17岁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 青年在该国一处地方联邦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安全局.html" target="_blank">
  安全局
 </a>
 大厅引爆自制炸弹，当场死亡，另有三名警员受伤。
 <br/>
 <br/>
 有居民听到爆炸巨响并感受到冲击波。
 <br/>
 <br/>
 目击者 Maria Vavilina：〝明显（看到）有肢体被炸飞了，然后救护车赶到，另一名男子被抬出去。〞
 <br/>
 <br/>
 调查人员表示，初步将事件判定为恐袭，调查将包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌犯.html" target="_blank">
  嫌犯
 </a>
 的亲朋好友，以查明犯案动机。
 <br/>
 <br/>
 嫌犯在犯案前在社交媒体发布了一段警告，但尚未证实。不过这段警告显示，嫌犯自称是共产主义者。
 <br/>
 <br/>
 两周前的10月18日，俄罗斯占据的克里米亚一所技术学院，发生重大枪击案，19人死亡、50人受伤，嫌犯也是一名18岁的年轻人。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
