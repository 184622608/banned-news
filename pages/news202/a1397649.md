### 印尼：失事狮航班客机〝黑匣子〞已找到
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】再来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼空难.html" target="_blank">
  印尼空难
 </a>
 搜救的进展情况。周四，印尼媒体报导，印尼搜寻团队已找到失事狮航客机JT610的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑匣子.html" target="_blank">
  黑匣子
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑匣子.html" target="_blank">
  黑匣子
 </a>
 就是飞行数据记录仪，能够监控飞机的电子和机械系统，同时记录飞机上的各种声音，包括飞行员的交谈和机上异常的声音等情况。黑匣子通常是解开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞机失事.html" target="_blank">
  飞机失事
 </a>
 之谜的第一步资料。
 <br/>
 <br/>
 这架编号为JT610的波音客机，9月29号周一上午，从印尼雅加达起飞后13分钟就坠毁。机上189人全部遇难。随后，搜救团队展开大规模搜寻。这次寻获黑匣子，是解开狮航客机JT610失事原因的关键步骤。
 <br/>
 <br/>
 狮航客机JT610的失事事件，也是印尼有史以来第二大空难。
 <br/>
 <br/>
 新唐人记者陈依心、宇薇综合报导
</div>
