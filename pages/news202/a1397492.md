### 泰警查获70公斤海洛因 逮3名台湾人2泰国人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】泰国警方昨(30)日晚间在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清迈.html" target="_blank">
  清迈
 </a>
 查获一辆货车上载运200块一级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒品.html" target="_blank">
  毒品
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海洛因.html" target="_blank">
  海洛因
 </a>
 ，共重70公斤，逮捕3名台湾人，或将面临最重死刑处分。
 <br/>
 <br/>
 据泰国警方提供的护照资料显示，3名台湾人分别是包姓（37岁)、张姓（37岁）和吴姓男子（22岁），他们和2名泰国男子涉嫌将200块的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海洛因.html" target="_blank">
  海洛因
 </a>
 砖分装在4个行李箱内，而遭到警方逮捕。
 <br/>
 <br/>
 其中台藉吴姓男子有贩毒的前科、张姓男子也有违反枪炮弹药等前科纪录。
 <br/>
 <br/>
 据泰国法令，这5名运送海洛因的嫌疑人，可能面临死刑或是无期徒刑的处分。
 <br/>
 <br/>
 事发后，泰国司法部肃毒委员会（ONCB）和缉毒局（NSB）已于当晚通报中华民国驻泰国经济文化办事处。驻泰处法务部调查局秘书今天赶往协助调查。
 <br/>
