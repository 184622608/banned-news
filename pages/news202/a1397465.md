### 颱风玉兔袭菲律宾 已知8死近40人恐遭活埋
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】颱风玉兔近日袭击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾.html" target="_blank">
  菲律宾
 </a>
 北部的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吕宋岛.html" target="_blank">
  吕宋岛
 </a>
 ，除造成基础设施损坏之外，还引发多起土石流，恐有近40人被活埋。目前已证实至少有8人死亡，包括4个孩童。
 <br/>
 <br/>
 颱风玉兔（Yutu）于29日清晨挟瞬间风速高达每小时230公里的强风从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吕宋岛.html" target="_blank">
  吕宋岛
 </a>
 （Luzon）东岸登陆，横越吕宋岛北部地区。玉兔于30日晚间进入南海，预料会于周三（31日）下午离开菲国，转向前进中国东南沿海省分。强度也会持续减弱，但外围东南风将与东北风发生〝共伴效应〞。
 <br/>
 <br/>
 上个月吕宋岛才遭颱风山竹（Mangkhut）横扫，这次玉兔再次来袭，强风不但吹走建筑物屋顶，更引发多起致命土石流。
 <br/>
 <br/>
 地区救灾主管卡兰丹（Ruben Carandang）表示，在科迪勒拉（Cordillera）山区纳托宁市（Natonin）一座公共建筑被突然来袭的土石流〝完全掩埋〞，据信当时有多达31名值班人员及村民在内躲避风雨。
 <br/>
 <br/>
 菲媒引述地方政府的报导说，救难人员已挖出3具尸体，但因暴风雨依然强劲加上水位上涨，搜救行动已经暂时停止。
 <br/>
 <br/>
 主管当局已证实，土石流在纳托宁市以南的巴拿威镇（Banaue）夺走一名父亲和3个年幼孩子的性命。另外，在邻省的另一起土石流造成一名5岁女童丧生。搜救人员已经寻获他们的尸体，另有多人失踪或受伤。
 <br/>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山竹颱风.html" target="_blank">
  山竹颱风
 </a>
 侵袭，至少有127人因土石流、溺水、触电、被断树或电线杆压中等因素死亡，另外还有100多人下落不明，毁损的基础设施还没能重建，当地民众也还未能摆脱农作物破坏造成的影响，相隔一个月再次遭颱风破坏。
 <br/>
 <br/>
 另外，台湾中央气象局预报，周三（10月31日）受东北季风及中颱玉兔外围环流双重影响，北部及东半部有阵雨，降雨时间长并有局部大雨或豪雨。气象专家吴德荣表示，从31日起3天防颱风共伴效应，北台湾将是又溼又凉的天气。
