### 从广州飞曼谷 泰国航班夜降惊险冲出跑道
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】搭载115人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泰国航空.html" target="_blank">
  泰国航空
 </a>
 TG 679班机，8日晚间10点48分在降落曼谷国际机场时，疑因天候因素、打滑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冲出跑道.html" target="_blank">
  冲出跑道
 </a>
 ，滑到一旁的草坪上，所幸机上无人受伤。
 <br/>
 <br/>
 曼谷机场表示，机上97名乘客及18名机组人员安全获救后，机场已在第一时间提供旅客住宿地点。
 <br/>
 <br/>
 这架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泰国航空.html" target="_blank">
  泰国航空
 </a>
 （Thai Airways）波音747-400飞机，8日晚间从中国广州飞往曼谷机场。
 <br/>
 <br/>
 泰国航空公司将与相关机构合作调查事故原因。
 <br/>
 <br/>
