### 日本与湄公河领袖会议 促进区域安全
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】周二(10月9日)，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湄公河.html" target="_blank">
  湄公河
 </a>
 流域国家在东京举行第10届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_领袖会议.html" target="_blank">
  领袖会议
 </a>
 ，并就彼此之间的合作关系，以及促进区域和平与安全考量，一致通过了〝2018年东京战略〞文件。
 <br/>
 <br/>
 周二上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湄公河.html" target="_blank">
  湄公河
 </a>
 流域国家在东京都迎宾馆举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_领袖会议.html" target="_blank">
  领袖会议
 </a>
 。成员包括日本、柬埔寨、老挝、缅甸、泰国及越南等国家领导人。
 <br/>
 <br/>
 日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 表示，湄公河地区拥有繁荣的巨大潜力，为了共同实现更美好的未来，各国领袖一致通过这份〝2018年东京战略〞文件。
 <br/>
 <br/>
 日本首相 安倍晋三：〝要增进日本与湄公河国家的关系，加强合作，实现繁荣的未来。〞
 <br/>
 <br/>
 这次领袖会议主要探讨的议题包括，打造以人为中心的社会、实现绿色湄公河地区等原则，并将进行高品质的基础设施修建等计划。
 <br/>
 <br/>
 缅甸领导人 昂山素季：〝大家一致同意，日本和湄公河国家之间的合作已有成功的历史，我们有信心继续合作，不仅是为了繁荣，也是为了区域和平与安全。〞
 <br/>
 <br/>
 会议也一致要求朝鲜采取具体行动迈向无核化。促进区域和平与安全、南海议题、以及海上和空中航行自由等事宜，是这次日本和湄公河国家领袖会议重点探讨的议题。
 <br/>
 <br/>
 新唐人记者陈依心、明玉综合报导
</div>
