### 〝极危〞飓风威拉直扑墨西哥 民众囤食物忙防范(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】挟强风豪雨的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 威拉（Willa）22日(当地时间)逼近太平洋沿岸，朝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 直扑而来。气象局预测，〝极其危险〞威拉料为墨西哥部分地区带来150毫米至450毫米雨量。对预期将带来威胁生命的暴风雨，当地民众急囤食物防灾，商店架上的面包已被一扫而空。
 <br/>
 <br/>
 路透社报导，美国国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 中心（National Hurricane Center）表示，威拉稍早增强为5级飓风后，如今已减弱为4级，目前正朝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 度假小镇马萨兰 (Mazatlan）附近扑去，预计明(23日)下午或晚上抵达。
 <br/>
 <br/>
 威拉预计是近几年来自太平洋扑向墨西哥的最强大飓风之一。
 <br/>
 <br/>
 目前最新讯息，威拉近中心最大风速为每小时250公里，虽略为减弱，但美国国家飓风中心指出，威拉抵达墨西哥海岸时，预计仍会是威力强大危险飓风。并警告：〝豪雨将引发危及生命的暴洪及山崩。〞
 <br/>
 <br/>
 <center>

 <br/>
 〝极其危险〞的飓风威拉料为墨西哥西纳罗亚（Sinaloa）、纳耶瑞（Nayarit）及哈里斯科（Jalisco）这3州的部分地区带来150毫米至300毫米的雨量，若干地区的雨量将高达450毫米。
 <br/>
 <br/>
 美国国家飓风中心表示，纳耶瑞州数个旅游度假胜地，及哈里斯科州瓦拉塔港（Puerto Vallarta）的海滩度假村均靠近威拉的移动路径，这些地点预计将出现〝危及生命的暴潮、强风及豪雨〞。
 <br/>
 <br/>
 目前，马萨兰市郊的加油站稍早已出现排队车龙，附近的便利店也可见购物人潮。
 <br/>
 <br/>
 加油站服务员帕尔多（Zulema Pardo）说，过去几个小时以来，民众鱼贯排队囤积基本用品，购买大量的水及汽油，架上面包也被一扫而空。她说：〝人们都吓坏了。〞
 <br/>
 <br/>
 而预计威拉来袭，当局已疏散太平洋沿岸数以千计的民众，当地居民则忙以木板装钉窗户，学校也宣布停课。
