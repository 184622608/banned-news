### 向美示好？一美国公民将被朝鲜驱逐出境
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】朝鲜中央通讯社周五报导称，一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法入境.html" target="_blank">
  非法入境
 </a>
 朝鲜的美国人即将被朝鲜当局
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驱逐出境.html" target="_blank">
  驱逐出境
 </a>
 。
 <br/>
 <br/>
 据朝鲜官方称，这位名叫布鲁斯．拜伦．洛伦斯的美国公民，在今年10月16日通过朝中边境
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法入境.html" target="_blank">
  非法入境
 </a>
 朝鲜后，被朝鲜有关机构抓获。在调查过程中，洛伦斯承认自己是非法入境，而且受美国中情局的〝控制〞,现在朝鲜当局决定将他
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驱逐出境.html" target="_blank">
  驱逐出境
 </a>
 。不过，《朝中社》的报导并未公布这名男子被驱逐的确切日期。
 <br/>
 <br/>
 此前，路透社曾报导称，一位同名的美国男子于2017年11月被发现在朝鲜与韩国的强化边界附近徘徊，随即被韩国方面驱逐出韩国，但当时并未证实该男子有朝鲜的身份。
 <br/>
 <br/>
 外界注意到，这次朝鲜当局很快就主动宣布将这名〝非法入境〞的美国人驱逐出境，与朝鲜当局过去动辄长时间监禁在朝美国公民的作为明显有了改变。
 <br/>
 <br/>
 美联社在相关报导中分析称，现在朝鲜当局快速释放被抓的美国人，〝表明朝鲜仍希望保持与美国对话的气氛，尽管核外交停滞不前。〞
 <br/>
 <br/>
 2016年美国大学生Otto Warmbier到朝鲜旅行，结果在离开朝鲜前被朝方逮捕，朝鲜当局指控他试图从酒店房间偷取政治宣传海报，并因此对他判处了15年苦役。后来经过美国川普政府的多方努力，朝鲜方面同意将Otto Warmbier释放，但当他被送回美国时，他已经处于植物人状态，并在回国后几天死亡。朝鲜当局否认曾对Otto Warmbier使用酷刑，但这起事件激起了美国人的强烈愤慨，金正恩当局也因此受到国际社会的强烈谴责。
 <br/>
 <br/>
 今年5月，在金正恩与美国总统川普会面谈判朝鲜弃核问题前，为了向美国示好，朝鲜当局曾释放三名美国人，Kim Dong Chul，Kim Hak Song和Tony Kim，这三人当初在朝鲜被拘捕时，都被指控从事了间谍活动。
 <br/>
 <br/>
 今年以来，朝鲜当局虽然在遭受国际社会的严厉经济制裁后对外允诺可以考虑放弃核武器的研发，但至今在弃核方面并没有实质性的进展。
 <br/>
 <br/>
 就在本周五，朝鲜官媒刚刚报导了金这个你恩视察〝新战术武器的试验设施〞的消息。这篇报导称，新的战术武器的试验成功，展示了该国〝快速发展的防卫能力〞，将成为保卫朝鲜的〝铜墙铁壁〞。
 <br/>
 <br/>
 据路透社的报导，朝鲜官媒配发了一张金正恩在海滩上被一些朝鲜官员簇拥的照片，画面上并没有显示任何武器，但有国际武器专家辨认出，照片里那些官员中有一位是朝鲜军方炮兵部队的高级将领。
 <br/>
 <br/>
 舆论认为，在美国和朝鲜的去核谈判停滞不前的当下，金正恩当局公开宣传该国正在开发新武器，这对美朝谈判的政治气氛造成了消极的影响。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：明轩）
