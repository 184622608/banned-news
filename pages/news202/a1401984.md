### 60天内俄国不履约 美国就退出中导条约
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】12月4日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北约峰会.html" target="_blank">
  北约峰会
 </a>
 有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 的问题成为重点。除了近期扣押
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰船隻.html" target="_blank">
  乌克兰船隻
 </a>
 和船员的事件外，针对俄罗斯违反《中程导弹条约》，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 也提出明确警告。
 <br/>
 <br/>
 《中程导弹条约》是里根总统与戈尔巴乔夫于1987年签署，双方同意禁止两国部署所有射程在500至5500公里之间的巡航导弹和弹道导弹。
 <br/>
 <br/>
 但从 2014 年起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 违反条约，研发SSC-8导弹和9M729导弹，可飞越数千英里。俄罗斯虽然否认，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务卿表示已发现证据。
 <br/>
 <br/>
 〝SSC-8导弹的所有测试都来自卡普斯京亚尔的固定和移动发射器。它的射程范围直接威胁欧洲。〞
 <br/>
 <br/>
 川普总统10月20日表示，因为俄罗斯多次违约，美国将退出《中程导弹条约》。周二
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北约峰会.html" target="_blank">
  北约峰会
 </a>
 上，蓬佩奥正式宣布美国决定。
 <br/>
 <br/>
 〝（美国）将在60天后暂停履行条约，除非俄罗斯恢复完全和可证实符合规范。〞
 <br/>
 <br/>
 北约秘书长（Jens Stoltenberg）指出要做好应对俄罗斯违约的准备。
 <br/>
 <br/>
 同时，蓬佩奥表示，美国总统川普正在促进世界各国建立一个新的自由秩序，以防止战争并实现所有人的更大繁荣。同时批评中共、俄罗斯与伊朗违反多个国际公约。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝我们正在凝聚世界高尚国家，建立一个新的自由秩序，以防止战争并实现所有人的更大繁荣。〞
 <br/>
 <br/>
 此外，峰会期间，北约国家也敦促俄罗斯释放被扣押的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰船隻.html" target="_blank">
  乌克兰船隻
 </a>
 和船员。
 <br/>
 <br/>
 〝我们也希望被捕获的乌克兰公民立即获释〞。
 <br/>
 <br/>
 〝我们不会接受任何形式，攻击乌克兰领土完整〞。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
