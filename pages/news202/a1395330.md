### 176年来首个热带风暴 侵袭葡萄牙
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】早在9月22日生成的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热带风暴.html" target="_blank">
  热带风暴
 </a>
 莱斯利（Leslie）一直在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大西洋.html" target="_blank">
  大西洋
 </a>
 徘徊摇摆不定，路径像蝴蝶结般的摇摆，终于在10月13日登陆葡萄牙，强风豪雨将首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里斯本.html" target="_blank">
  里斯本
 </a>
 街上树木吹倒，预计14日侵袭西班牙。据气象纪录，莱斯利可能成为176年来袭击葡萄牙的最强风暴。
 <br/>
 <br/>
 《法新社》报导，风速可达每小时120公里的莱斯利13日登陆葡萄牙，已从1级飓风减弱成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热带风暴.html" target="_blank">
  热带风暴
 </a>
 ，但当局对部分地区发布红色警戒，要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里斯本.html" target="_blank">
  里斯本
 </a>
 (Lisboa）南部海滩的冲浪者返回到岸边，并警告住在海边的民众应留在室内，渔民也要回到港口。
 <br/>
 <br/>
 TAP葡萄牙航空也取消了7班往返里斯本的航班。
 <br/>
 <br/>
 葡萄牙民防局局长科斯塔（Luis Belo Costa）表示，莱斯利的强度虽减弱，但速度却加快，因此会比预计中更早抵达里斯本以北的地区。
 <br/>
 <br/>
 有气象学家预计，莱斯利将于14日侵袭西班牙。西班牙民防部警告，南部与东部预计面临非常强烈的风暴，北部与东部则可能会有大雨。
 <br/>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大西洋.html" target="_blank">
  大西洋
 </a>
 生成，因没有引导气流，所以一直在海面徘徊，路径甚至像〝蝴蝶结〞，寿命达22天。而据气象纪录显示，伊比利半岛鲜少有热带气旋登陆，脸书粉丝团〝台湾颱风论坛｜天气特急〞指出，莱斯利将成为自1842年来，侵袭葡萄牙最强，也是第一个登陆的热带气旋。
