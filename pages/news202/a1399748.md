### 不满油价调涨 法国群众抗议酿1死227人伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】法国数以万计民众17日走上街头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃料税上涨.html" target="_blank">
  燃料税上涨
 </a>
 ，群众阻挡街道示威，不幸造成撞人意外，导致一名老妇人丧命。另全国约有227人受伤，其中7人重伤，包括一名警察。
 <br/>
 <br/>
 事发在东部萨瓦省（Savoie），原因是一名母亲要带女儿去看医生，所驾驶汽车被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 人群包围，在惊慌下，汽车突然加速冲向人群，造成一名63岁妇人丧生。肇事母亲事后被警方带回讯问，还惊魂未定。
 <br/>
 <br/>
 法国内政部表示，这场法国大规模示威活动，全国各地估约近28万3000人参与活动，大约2000人聚集交通路口，有的则堵在主要公路和全国要道上。目前统计有227人受伤，其中7人重伤，当天共逮捕了117人，其中73人遭拘禁。

 <br/>
 <br/>
 此次事件有超过70%的法国民众表示对抗议行动的支持，明显对马克龙施政下，人民购买力无法提升的积怨，愤怒如野火蔓延全国。
