### 在野党主席被控收贿被捕 秘鲁法院下令放人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘鲁.html" target="_blank">
  秘鲁
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_在野党主席.html" target="_blank">
  在野党主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_藤森惠子.html" target="_blank">
  藤森惠子
 </a>
 因被控收受巴西建设公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Odebrecht.html" target="_blank">
  Odebrecht
 </a>
 的不法献金，已被羁押一周。秘鲁法官周三（17日）宣布，上诉法院下令将其释放。
 <br/>
 <br/>
 法官萨瓦奈（Octavio Sahuanay）表示，法院裁定立即释放本案拘留的嫌疑人，包括前日裔总统藤森谦也（Alberto Fujimori）的女儿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_藤森惠子.html" target="_blank">
  藤森惠子
 </a>
 （Keiko Fujimori）。
 <br/>
 <br/>
 43岁的藤森惠子是国会最大党〝人民力量党〞（Popular Force）主席。她因在2011年竞选总统时疑似收受来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Odebrecht.html" target="_blank">
  Odebrecht
 </a>
 的选举资金但实为收贿，于本月10日被逮捕。一同被捕的还有包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘鲁.html" target="_blank">
  秘鲁
 </a>
 前日裔部长吉山（Jaime Yoshiyama）在内的19人。
 <br/>
 <br/>
 Odebrecht是拉丁美洲各国政治丑闻的核心。藤森惠子先前并未直接遭受调查，检方注意力集中在她2011年竞选阵营两名高阶幕僚身上。
 <br/>
 <br/>
 藤森惠子于2005年成为该国政治派系领袖，被视为藤森政治王朝的接班人。她在2016年的总统选举中败给现任总统库琴斯基（Pedro Kuczynski）。
 <br/>
 <br/>
 藤森惠子的父亲藤森谦也出生于秘鲁首都利马，曾于1990～2000年担任该国总统，拥有秘鲁和日本的双重国籍。2000年其贪污丑闻被揭发后，在同年举办的总统大选后回到日本定居。
 <br/>
 <br/>
 2009年4月7日，秘鲁特别法庭以参与绑架杀人案与〝践踏人权罪〞判处藤森25年监禁。但由于健康恶化，在狱中服刑12年后于2017年12月获现任总统特赦。秘鲁法院10月3日下令再次将其逮捕。
