### G20公报首次同意改革WTO 白宫称川普胜利
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】在阿根廷首都布宜诺斯艾利斯的二十国集团（G20）峰会结束后发布了领袖公报，首次未提〝反对贸易保护主义〞的内容，并提出要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革.html" target="_blank">
  改革
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 。白宫表示，这是美国总统川普(特朗普)的胜利。而参与人员透露谈判激烈的程度前所未有。
 <br/>
 <br/>
 12月1日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 发布领袖公报，针对各界瞩目的贸易议题，G20成员警告日渐增长的地缘政治威胁，以及重申多边贸易的利益。公报首次删去保护主义的措辞。
 <br/>
 <br/>
 公报称：〝多边贸易体系没有达到目标，还有改进的空间〞，各国首脑同意于2019年6月在日本大阪举行的峰会上继续讨论世贸组织（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 ）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革.html" target="_blank">
  改革
 </a>
 。
 <br/>
 <br/>
 公报还表示，欢迎〝强劲的全球经济增长，同时体认各国间的（经济增长）同步程度越来越低〞。
 <br/>
 <br/>
 成员国说，他们注意到当前的贸易问题，一致认为财政政策应该在需要时重建缓冲，且其运用应以灵活及对增长有利的方式为之，同时要确保公共债务的可持续发展。
 <br/>
 <br/>
 参与G20领袖声明谈判工作的一位欧盟官员告诉《华尔街日报》，由于川普与习近平定在12月1日举行晚宴，为了避免破坏美中贸易可能达成休战的可能性，成员国协商人员多盼能达成共识，竭尽所能地完成声明。
 <br/>
 <br/>
 欧洲谈判代表说，谈判非常激烈，G20成员经过一周的谈判，11月30日终于突破分歧完成公报草案。即使在此前的一天，各方都对能否达成共识不抱希望。
 <br/>
 <br/>
 参与谈判的欧盟官员透露，为了取得美国的认可，其他成员让步，同意美方要求，取消了原本在G20峰会领袖公报中的这段措辞：〝保护主义和贸易争端对经济不利，全球机构是必要且有用的〞。另一方面，中方则要求删去〝不公平贸易行为〞的文字。
 <br/>
 <br/>
 妥协的结果是，文本内容承认贸易和投资是〝增长、生产力、创新、创造就业和发展的重要引擎。〞
 <br/>
 <br/>
 对于WTO的功能，领袖公报写道：〝多边贸易体系没有达到目标，还有改进的空间〞，〝我们支持WTO的必要改革，以改善其职能，我们将在下一次峰会上审查相关进展。〞各国首脑同意于2019年6月在日本大阪举行下一次峰会。
 <br/>
 <br/>
 一名美国高级官员告诉路透社，这是G20首脑首次认可WTO目前无法达成它的设立目标，并且需要改革。报导说，这也是G20公报首次未提到贸易保护主义的内容，公报内容广泛满足了川普〝美国优先〞的政策。
 <br/>
 <br/>
 一位白宫高级官员说，他十分赞赏该公报内容，特别是呼吁WTO改革及改变全球贸易体系的文字，对于公报提及川普退出巴黎气候协议决定的措辞，也受到了美国官员的欢迎。
 <br/>
 <br/>
 法国总统马克龙告诉媒体记者，规范全球贸易及处理贸易争端的WTO需要与时俱进。
 <br/>
 <br/>
 《美国之音》引述一名白宫高级官员表示，G20公报的另一个巨大成就是，中共首次在国际场合承诺提供透明的基础建设财务支援，以确保不会造成发展中国家的债务负担。
 <br/>
 <br/>
 此外，对于争议较大的气候变化议题，美国成功地让公报单列一个段落，说明其退出巴黎气候协议。公报第21段，美国重申其决定退出这个协议，并确认其对经济增长及能源取得与安全的强烈承诺，在保护环境的同时，运用所有能源资源及技术。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿）
