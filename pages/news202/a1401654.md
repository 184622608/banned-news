### G20公报首次同意改革WTO 白宫称川普胜利
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】在阿根廷首都布宜诺斯艾利斯的二十国集团（G20）峰会结束后，发布领袖公报，首次未提〝反对贸易保护主义〞的内容，并提出要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革.html" target="_blank">
  改革
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 。白宫表示，这是美国总统川普(特朗普)的胜利。
 <br/>
 <br/>
 12月1日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 发布领袖公报，针对各界瞩目的贸易议题，G20成员警告日渐增长的地缘政治威胁，以及重申多边贸易的利益。公报首次删去保护主义的措辞。
 <br/>
 <br/>
 公报称：〝多边贸易体系没有达到目标，还有改进的空间〞，各国首脑同意于2019年6月在日本大阪举行的峰会上继续讨论世贸组织（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 ）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革.html" target="_blank">
  改革
 </a>
 。
 <br/>
 <br/>
 公报还表示，欢迎〝强劲的全球经济增长，同时体认各国间的（经济增长）同步程度越来越低〞。
 <br/>
 <br/>
 成员国说，他们注意到当前的贸易问题，一致认为财政政策应该在需要时重建缓冲，且其运用应以灵活及对增长有利的方式为之，同时要确保公共债务的可持续发展。
 <br/>
 <br/>
 G20成员经过一周的谈判，11月30日终于突破分歧完成公报草案。不久前的亚太经合组织（APEC）峰会，因中美分歧当时未能发表声明。
 <br/>
 <br/>
 根据参与G20谈判的欧盟官员透露，美国将〝反对贸易保护主义〞的字眼从最终声明中拿；而中共删除〝不公平贸易行为〞的这一句话。中共曾在APEC峰会上反对这句话，担忧这是专门针对中共。
 <br/>
 <br/>
 《路透社》当天早些时候说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 前，G20成员国领袖公布的最终声明中，支持对世界贸易组织（WTO）进行改革。而经过各国谈判代表11月30日彻夜谈判而拟定的公报中，仅提到了〝目前的贸易问题〞，并未提供任何详情。
 <br/>
 <br/>
 最终声明称，〝我们承认多边贸易体系做出的贡献，该体系目前未能达到其目的，具有改进的空间，我们因此支持对WTO进行必要的改革，以改善其功能。我们将在下次峰会时评估进展。〞
 <br/>
 <br/>
 《彭博社》称，G20公报首次未提到贸易保护主义的内容，公报内容广泛满足了川普〝美国优先〞的政策。
 <br/>
 <br/>
 一位白宫高级官员说，他十分赞赏该公报内容，特别是呼吁WTO改革及改变全球贸易体系的文字，对于公报提及川普退出巴黎气候协议决定的措辞，也受到了美国官员的欢迎。
 <br/>
 <br/>
 法国总统马克龙告诉媒体记者，规范全球贸易及处理贸易争端的WTO需要与时俱进。
 <br/>
 <br/>
 《美国之音》引述一名白宫高级官员表示，G20公报的另一个巨大成就是，中共首次在国际场合承诺提供透明的基础建设财务支援，以确保不会造成发展中国家的债务负担。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
