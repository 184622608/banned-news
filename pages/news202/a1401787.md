### 打扮时髦像玄彬 脱北士兵吴青成：有自由，挺好
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】朝鲜〝脱北〞士兵
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴青成.html" target="_blank">
  吴青成
 </a>
 染了咖啡色头发，在韩国展开新生活，他接受韩国媒体访问时说，他现在的生活〝有自由，挺好〞，但〝体会到在韩国赚钱很难〞，他希望努力工作，多交税并多做义工。有人说他像韩国演员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_玄彬.html" target="_blank">
  玄彬
 </a>
 ，但他说〝但一点都不像啊〞
 <br/>
 <br/>
 《朝鲜日报》报导，今年25岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴青成.html" target="_blank">
  吴青成
 </a>
 ，去年11月经板门店共同警备区冒死奔向韩国，今年6月起住在首尔，外表打扮与普通韩国青年无异。
 <br/>
 <br/>
 吴青成说，有人说他像韩国演员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_玄彬.html" target="_blank">
  玄彬
 </a>
 ，〝我上网找了照片，但一点都不像啊〞。吴青成因老家在开城邻近非军事区，因此他说话并无明显朝鲜口音。
 <br/>
 <br/>
 报导说，吴青成承认早前接受日本《产经新闻》的专访，并收取100万韩圜（约960美元）的报酬，但扣除去日本的机票及住宿费后，已所剩无几。
 <br/>
 <br/>
 谈及《产经新闻》专访时，吴青成说翻译误解他的意思，他不是指韩国军队不像军队，而是说韩国兵役期较朝鲜短，因此较为容易，《产经新闻》为此已向他道歉。
 <br/>
 <br/>
 他强调他流着与韩国人一样的血，更会铭记韩军士兵在非军事区舍身相救的一幕。
 <br/>
 <br/>
 谈及对韩朝峰会的看法，吴青成称对政治没兴趣。由于双亲都还在朝鲜，吴青成希望统一。
 <br/>
 <br/>
 他说，因腰椎有病，不能从事体力劳动，而换了工作，现在企业工作。还形容韩国的生活〝有自由，挺好，想去哪里就去哪里，想做什么就做什么〞。但也〝体会到在韩国赚钱不容易〞。
 <br/>
 <br/>
 他说，〝还未确定在韩国的目标，很难选择，正在为此烦恼，（今后）我要努力工作，多交税，还打算为病人做义工。〞
