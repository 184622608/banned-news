### 【禁闻】台湾爱国同心会成员涉嫌贿选 被限出境
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾九合一选举.html" target="_blank">
  台湾九合一选举
 </a>
 临近，亲共组织〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱国同心会.html" target="_blank">
  爱国同心会
 </a>
 〞推出的候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张秀叶.html" target="_blank">
  张秀叶
 </a>
 ，以及该会会长周庆峻，由于涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贿选.html" target="_blank">
  贿选
 </a>
 ，最近被台北地检署拘查。目前两人已被限制居住，并且禁止出境、出海。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾九合一选举.html" target="_blank">
  台湾九合一选举
 </a>
 即将于11月24号举行。台北市爆出首起疑似市议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贿选.html" target="_blank">
  贿选
 </a>
 案。〝中国民主进步党〞推出的台北市万华区议员候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张秀叶.html" target="_blank">
  张秀叶
 </a>
 ，涉嫌在10月初以餐会方式行贿选民，餐会资金不排除来自大陆。
 <br/>
 <br/>
 所谓的〝中国民进党〞由宣扬共产党的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱国同心会.html" target="_blank">
  爱国同心会
 </a>
 〞等组织成立，〝爱国同心会〞会长周庆峻是党主席，〝统促党〞白狼张安乐任荣誉总监。
 <br/>
 <br/>
 台北地检署11月12号进行了搜查、约谈。并在13号以涉嫌违反《选罢法》，把张秀叶和周庆峻列为被告，限制居住及禁止出境、出海。
 <br/>
 <br/>
 检方接获举报说，今年10月1号，〝爱国同心会〞在台北中山区一家日本餐厅举办餐会，宣称是庆祝中共建政69周年，免费招待民众用餐。设席50多桌，同时张秀叶在会场向有投票权的选民拉票。检方调阅餐会帐单，发现每桌成本至少7千元，如果以1桌10人来计算，每人可获得7百元的餐会利益，超过30元的贿选标准，涉嫌违反《选罢法》。
 <br/>
 <br/>
 另外张秀叶涉嫌在今年8月间，招待〝爱国同心会〞20多名干部，前往中国贵州旅游约一周，每人旅费1万多元新台币，被招待的干部大多是大陆配偶，有投票资格，食宿由中共支付。
 <br/>
 <br/>
 台湾《联合新闻网》报导，周庆峻常去大陆拜访中共特定人士，返台携带大批人民币入境，透过地下汇兑换成台币，再将钱交给张秀叶运用。
 <br/>
 <br/>
 台湾大学国家发展研究所法学博士曾建元：〝以张秀叶来讲，她现在是中华民国的国民。那如果她是接受境外的政治献金，或者是跟中华民国敌对的政权的资金来参与台湾的选举的话，对其他的候选人是一个不公平的竞争。所有的国家其实都是禁止国外的政府资金，对国内选举的任何一个候选人或政党资助。〞
 <br/>
 <br/>
 此案也是首宗被曝光的中共资金介入选举的案件。检方在张秀叶处所查扣一本帐册，里面有资金流向的记录。检方将以此调查张、周二人是否接受中共金援、触犯《政治献金法》。
 <br/>
 <br/>
 而国际媒体《半岛电视台》曾在9月发布卧底调查报导，揭露〝爱国同心会〞明知违法，仍接受中共金援。
 <br/>
 <br/>
 〝爱同会〞总干事张秀叶/其他成员VS半岛卧底 ：〝拿大陆的钱，谁要是（被）抓到，都是要坐牢的。（台商支持就没问题吧？）对，就譬如会长（周庆峻）现在是台商。大陆（中共）就是基本上知道你哪几个企业，是支持统一的，它尽量会，就是不让你亏损。〞
 <br/>
 <br/>
 〝爱同会〞总干事张秀叶VS爱同会成员：〝（所以以后你参选，这些人在帮忙发动，对不对，我是有这样的安排啦。）举牌也不错啊，一天八百吧。（一天九百块，他们八百我九百，因为我叫人给他。）〞
 <br/>
 <br/>
 张秀叶曾是大陆国营事业员工，1992年嫁到台湾，不久与丈夫离婚。她定居台湾后加入了〝爱国同心会〞并成为总干事，多次带人到台北〝101大楼〞前宣扬中共，还攻击、谩骂法轮功学员。一度因攻击前来维持秩序的警察，被警方以现行犯逮捕。
 <br/>
 <br/>
 今年中共强力介入台湾选举，引发警惕。台湾国安局在11月初表示，中共影响台湾选举的手法至少有十几种，从资金影响台湾赌盘，到招待宗教团体、渔会、农会人员到大陆。
 <br/>
 <br/>
 台湾调查局10月底则透露，正调查33宗中共利用金钱介入选举的案件。美国国会的〝美中经济与安全审查委员会〞提出报告指出：中共正在加强对台湾政治活动，试图妨碍台湾民主发展，利用社群媒体等，散播不实讯息。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/李沛灵
