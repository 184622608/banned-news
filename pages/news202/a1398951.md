### 波兰庆独立百周年 视共产崩溃为大成就
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波兰.html" target="_blank">
  波兰
 </a>
 总统安杰伊・杜达，在首都华沙主持仪式，纪念波兰独立一百周年。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波兰.html" target="_blank">
  波兰
 </a>
 首都华沙的文森堤・维托斯（Wincenty Witos）纪念碑前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波兰总统.html" target="_blank">
  波兰总统
 </a>
 杜达与其他政府官员共同出席了纪念波兰独立一百周年的纪念仪式，他们现场共同演唱了波兰爱国歌曲〝誓言〞。
 <br/>
 <br/>
 在士兵向二战前三次担任波兰总理的文森堤・维托斯敬献花圈，随后，总统杜达在纪念碑前单膝跪地，以缅怀先驱。
 <br/>
 <br/>
 18世纪后期开始，波兰经历了由俄罗斯，普鲁士和奥匈帝国分割和统治的时期，长达123年，最终于1918年重夺主权，获得独立。
 <br/>
 <br/>
 今年一份民调显示，波兰人把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产主义.html" target="_blank">
  共产主义
 </a>
 在波兰崩溃视为这100年来最为成功的事件。
 <br/>
 <br/>
 美国总统川普曾于2017年7月（5-6日）访问波兰，得到了杜达右翼政府极受尊重的礼遇。他在周日发推文说：波兰是伟大的国家，祝贺它独立100周年。
 <br/>
 <br/>
 此外，阿联酋的杜拜和韩国首尔也分别以波兰旗帜颜色的灯光，对波兰庆祝独立表示祝贺。
 <br/>
 <br/>
 新唐人电视综合报导
</div>
