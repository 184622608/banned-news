### 美国医学会年会  反强摘决议案受关注
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】最近，在美国最大的医生组织--
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国医学会.html" target="_blank">
  美国医学会
 </a>
 (AMA)举办的年会上，一项旨在防止强摘器官的决议案被提出，并得到与会医学界专家的支持。
 <br/>
 <br/>
 日前，在美国华府举行的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_医学会年会.html" target="_blank">
  医学会年会
 </a>
 上，哥伦比亚特区医学会代表提出一项决议案，呼吁防止让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_器官移植.html" target="_blank">
  器官移植
 </a>
 手术成为〝冷酷的种族灭绝中有利可图的工具〞。
 <br/>
 <br/>
 决议案提出三项具体要求，包括来美接受移植手术培训的外科医生，必须同意遵守
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国医学会.html" target="_blank">
  美国医学会
 </a>
 的道德准则；必须对中国的器官移植进行独立，透明调查；美国政府必须立法，将不符合美国医疗透明和道德准则的国家列入黑名单。
 <br/>
 <br/>
 哥伦比亚特区医学会代表团主席 Joseph Gutierrez：〝由于越来越多的关于违反道德的报告，我们非常担忧，并希望所有人都注意到这一状况。〞
 <br/>
 <br/>
 医生反对强制摘取器官组织执行主任 Torsten Trey：〝我们的研究和对目击者的采访显示，这些器官基本上是从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员身体上强摘的。中共蓄意铲除法轮功，对他们进行残酷的种族灭绝和强摘器官。〞
 <br/>
 <br/>
 医学专家表示，中国一直是世界器官移植数量最少国家之一。1999年中共开始迫害法轮功，中国快速上升为全球最主要的器官移植国，大量外国病患到中国进行器官移植旅游。
 <br/>
 <br/>
 医生反对强制摘取器官组织副主任 Weldon Gilcrease：〝我们需要各个组织给出政策，也需要在州和联邦政府立法，表示我们不会培训中国医生进行器官移植，然后回到中国，参与这一反人类罪行。〞
 <br/>
 <br/>
 专家认为，决议案有助于让人们了解正在中国发生的暴行，同时也希望全球医学界共同采取行动。
 <br/>
 <br/>
 美国医学协会理事会前主席 Raymond Scalettar  ：〝我们的要求是，让世界医学协会采取更大行动，也许可以成立一个机构，要求中共政府必须遵守准则，并对所发生的一切提供数据。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
