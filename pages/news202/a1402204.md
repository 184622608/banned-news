### 《马三家来信》加国会首映 揭中共暴行
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】周三（12月5日）晚间，入围
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥斯卡最佳纪录片.html" target="_blank">
  奥斯卡最佳纪录片
 </a>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_求救信.html" target="_blank">
  求救信
 </a>
 》首次在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大国会.html" target="_blank">
  加拿大国会
 </a>
 播放，多名前来观看的国会议员被影片中法轮功学员在中共酷刑迫害下仍然坚守信仰的精神感动落泪。来看报导。
 <br/>
 <br/>
 参议员Marilou Mcphredran：〝像在坐很多人一样，我落泪了。〞
 <br/>
 <br/>
 纪录片揭露了中国马三家劳教所用超出人所能承受的酷刑折磨法轮功学员，和马三家劳工产品背后的血泪。
 <br/>
 <br/>
 国会议员Harold Albrecht：〝今晚看到的这个故事非常令人不安，因为劳教所里的人权受到了侵犯。〞
 <br/>
 <br/>
 前公共安全部长Steven Blaney：〝纪录片揭露了中国人权状况，要想改变这一状况所面临的挑战以及主人公的巨大勇气。〞
 <br/>
 <br/>
 这部纪录片在中国秘密拍摄而成，影片还根据主人公孙毅本人手绘的场景，用动画的方式呈现了劳教所夜以继日的酷刑折磨来逼他放弃信仰。但他始终坚守信仰。
 <br/>
 <br/>
 国会议员Harold Albrecht：〝电影的主人公有这种坚持和信念。非常值得称赞。我认为他教给我们所有人很多东西。〞
 <br/>
 <br/>
 前公共安全部长Steven Blaney：〝纪录片非常励志，给人们上了一堂关于勇气，内心充满希望和警戒的课。〞
 <br/>
 <br/>
 国会议员Cheryl Hardcastle：〝今晚我会沉默，思考这部电影，这样的电影对我们来说是非常深刻和有益的。这让我们学会不能放弃。〞
 <br/>
 <br/>
 国会议员Harold Albrecht：〝我们经常认为我们面临着难以解决的问题，而他当然有一个似乎无法逾越的问题，但他做的很好，按照他所信仰的而活着，坚定不移，坚持不懈。〞
 <br/>
 <br/>
 这部电影引发议员们更加关注因炼法轮功而被中共关押在北京第一看守所的的加拿大公民孙茜和其他法轮功学员的命运。
 <br/>
 <br/>
 参议员Marilou Mcphredran：〝对加拿大公民孙茜和法轮功学员的迫害显然违反了联合国人权宣言，和禁止酷刑的公约。〞
 <br/>
 <br/>
 <br/>
 新唐人记者周荔敏、舒璨加拿大渥太华报导
</div>
