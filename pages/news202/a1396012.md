### 罕见攻击酿3死13伤 阿富汗警察首长罹难 美指挥官幸存
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】阿富汗南部大城市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎达哈.html" target="_blank">
  坎达哈
 </a>
 政府特区素来戒备森严，18日竟在一场高层的安全军事会议上，爆发枪击事件，在场的包括当地警察首长等3人丧命，另有13人受伤，美军指挥官米勒则躲过一劫。而行凶者是省长的保镳，塔利班已宣称犯案。
 <br/>
 <br/>
 据美联社一名摄影师表示，事发在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎达哈.html" target="_blank">
  坎达哈
 </a>
 市（Kandahar 坎大哈）政府特区，18日举行高层的安全军事会议结束，大家拍摄团体照后要离开特区。当时突然爆发枪战，大家赶紧逃走，美方的与会者坐上直升机离开，但当美军成员试图接近直升机时，还曾与阿富汗警方发生小冲突。
 <br/>
 <br/>
 行凶的枪手是一名行政首长的保镳，而死者包括在当地以强硬作风闻名的警察首长拉兹克（Abdul Raziq）。美国与北大西洋公约组织（NATO）部队指挥官米勒（Scott Miller）也在场，但安然躲过一劫。
 <br/>
 <br/>

 <br/>
 报导中说，事后塔利班宣称犯案，并称他们的目标就是拉兹克以及米勒。不过，一起出席会议的美国陆军上校布特勒（David Butler）则称，当时枪手的目标是拉兹克，虽然米勒也在附近，但没卷入枪战中。
 <br/>
 <br/>
 另2名遇害的死者分别是坎达哈情报机构首长默哈敏（Abdul Mohmin）及一名阿富汗记者，而开枪的枪手则当场被击毙。
 <br/>
 <br/>
 伤者一共有13人，包括3名美国人，稍早一度被报导死亡的坎达哈省长伟萨（Zalmay Wesa）则受到枪伤，接受治疗后恢复稳定中。
 <br/>
 <br/>
 米勒事后在推特上发文表示，〝今天我失去了很棒的朋友拉兹克将军，我们一起共事好多年了，阿富汗更是失去了一名爱国者，我向阿富汗的人民致哀〞。
 <br/>
 <br/>
 华府智库威尔逊中心（Wilson Center）专家柯格曼（Michael Kugelman）表示，美国与北约驻阿富汗部队指挥官遭到攻击是前所未见的事，加以拉兹克遇刺丧命，〝将对阿国安定造成重大打击，也将不利反叛乱行动〞。
 <br/>
 <br/>
 坎达哈地区此前由塔利班占领，拉兹克先前也遇过好几次被暗杀的危机，去年曾在一场攻击中，造成5名阿拉伯联合大公国的外交官员丧命，当时拉兹克躲过一劫。
