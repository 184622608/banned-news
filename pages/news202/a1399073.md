### 朝鲜秘密导弹基地曝光 大多深藏山谷
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】再来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜问题.html" target="_blank">
  朝鲜问题
 </a>
 。一份最新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卫星图像.html" target="_blank">
  卫星图像
 </a>
 显示，虽然朝鲜承诺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无核化.html" target="_blank">
  无核化
 </a>
 ，但在各地仍藏有不少
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘密导弹基地.html" target="_blank">
  秘密导弹基地
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国智库.html" target="_blank">
  美国智库
 </a>
 战略与国际研究中心在一份报告中表示，已经确认了朝鲜15到20个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘密导弹基地.html" target="_blank">
  秘密导弹基地
 </a>
 的所在地。
 <br/>
 <br/>
 报告称，这些基地可以制造不同型号的导弹，包括可以袭击美国本土的洲际弹道导弹。虽然这些基地不是发射台，但在紧急情况下仍可以发射导弹。
 <br/>
 <br/>
 报告中说:〝（这些）弹道导弹作战基地规模较小，分散在朝鲜各地，除少数例外，大多都位于狭窄的山谷中。〞
 <br/>
 <br/>
 报告引用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卫星图像.html" target="_blank">
  卫星图像
 </a>
 ，指出距离非军事区仅约50英里的一处导弹基地的运作似乎〝相当活跃且维护得相当好〞，有总部大楼、营房、维修站和可能用来隐藏导弹和运输卡车的地下隧道入口。
 <br/>
 <br/>
 该报告综合了美国政府官员，国防和情报部门以及朝鲜〝脱北者〞提供的信息。朝鲜否认了指控。
 <br/>
 <br/>
 美国总统川普与朝鲜领导人金正恩今年6月在新加坡举行首次峰会，金正恩虽然承诺努力实现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无核化.html" target="_blank">
  无核化
 </a>
 ，但到目前为止，朝鲜所做的仍很有限。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
