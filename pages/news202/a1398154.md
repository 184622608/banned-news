### 反制中共 专家吁台传真实资讯到大陆
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】台湾九合一选举即将在11月24日举行。中共通过虚假信息、破坏台湾民主也成为广泛关注话题。华府专家认为，台湾利用民主自由优势，努力将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_真实资讯.html" target="_blank">
  真实资讯
 </a>
 广传大陆，才是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反制中共.html" target="_blank">
  反制中共
 </a>
 的根本之道。
 <br/>
 <br/>
 台湾行政院3日表示，中共网军正在影响台湾选举，并已达到〝大军过境〞的可怕程度。国安单位也证实，中共已有〝巨魔工厂〞，专门培养自媒体，在各社群媒体建立帐号，对台湾展开〝认知空间战〞。对此，美台专家认为，台湾已成中共〝锐实力〞试验场，以虚假信息破坏台湾民主。
 <br/>
 <br/>
 中华文化总会副会长江春男：〝过去，我们用不断的选举来巩固民主。而现在，中共则用选举来削弱我们对西方民主的信念。〞
 <br/>
 <br/>
 《华盛顿自由灯塔》资深编辑 Bill Gertz：〝台湾是一把指向中共心脏的民主匕首。北京的统治者是这样看。他们投下巨资在所谓统战上，试图颠覆这个民主前哨。〞
 <br/>
 <br/>
 目前，中共军队网军编制约三十万人，〝五毛党〞则超过二百万。去年台湾国安局每月遭到的网路攻击多达十万起。
 <br/>
 <br/>
 针对中共透过网络散播假新闻与不实消息，控制舆论，美国在台协会台北办事处处长郦英杰（Brent Christensen）表示，希望将来台美能找到共同合作方式，联手对抗虚假资讯。
 <br/>
 <br/>
 中华文化总会副会长江春男：〝有些网路的攻击，那个可以看出来是有计划的、有组织的。那可以追到它的来源，那个都是跟情报有关系的，跟国防军事有关系，那一般人做不到了。要追查他们。〞
 <br/>
 <br/>
 华盛顿自由灯塔资深编辑 Bill Gertz：〝我认为(对台湾)更大的威胁是非军事的恐吓、威胁，信息操控的威胁。〞
 <br/>
 <br/>
 美国副总统彭斯不久前演讲，称赞台湾拥抱民主制度，为所有华人展现出一条更好道路。专家也建议台湾应利用其民主自由优势，努力将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_真实资讯.html" target="_blank">
  真实资讯
 </a>
 广传大陆，这才是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反制中共.html" target="_blank">
  反制中共
 </a>
 的根本之道。
 <br/>
 <br/>
 华盛顿自由灯塔资深编辑 Bill Gertz：〝台湾处于很有利位置，可成为中国内部民主改革的灯塔。这将需要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾领导人.html" target="_blank">
  台湾领导人
 </a>
 展现出更强大的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治意愿.html" target="_blank">
  政治意愿
 </a>
 。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
