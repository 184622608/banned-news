### 彭斯亚洲行  力推印太战略  直指〝一带一路〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】
 <b>
  新闻周刊
 </b>
 （654）美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 ，本周代表川普接连访问日本，出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东盟峰会.html" target="_blank">
  东盟峰会
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东亚峰会.html" target="_blank">
  东亚峰会
 </a>
 以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 。在一系列活动上，彭斯力推
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印太战略.html" target="_blank">
  印太战略
 </a>
 ，承诺保障地区安全、尊重各国主权，并当着习近平的面不点名的批评〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞项目。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 和夫人12号晚间抵达东京横田机场，开启了一周的亚洲之行。
 <br/>
 <br/>
 13日上午，彭斯罕见地以副总统身份，与日本首相安倍晋三举行联合记者会。安倍首相表达了对美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印太战略.html" target="_blank">
  印太战略
 </a>
 的支持，而这正是彭斯一周行程的主题。
 <br/>
 <br/>
 日本首相安倍晋三：〝实现印太地区经济繁荣、自由与开放，并建立在公平贸易原则上。〞
 <br/>
 <br/>
 周三，彭斯在新加坡出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东盟峰会.html" target="_blank">
  东盟峰会
 </a>
 ，并与印度总理莫迪会晤。印太战略这一关键词再次出现。
 <br/>
 <br/>
 美国副总统彭斯：〝总理先生，总统（川普）和我感谢您对自由开放的印太区域合作的大力支持。〞
 <br/>
 <br/>
 分析认为，美国推行印太战略的主要目的之一，在于抑制中共凭藉〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞在该地区进行势力扩张。此前在从日本前往新加坡的途中，彭斯接受《华盛顿邮报》采访时表示，如果中共要避免与美国及其盟友进行全面冷战，就必须有根本和实质的改变。
 <br/>
 <br/>
 时政评论人士横河：〝这里有两层意思，第一美国准备的最坏打算就是冷战；谈不成的责任在中共、美国不要冷战、但不怕冷战。〞
 <br/>
 <br/>
 周四，彭斯在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东亚峰会.html" target="_blank">
  东亚峰会
 </a>
 上发表讲话，承诺保障地区安全，同时宣布美国将提供600亿美元的发展融资，主要用于印太基础设施项目。
 <br/>
 <br/>
 美国副总统彭斯：〝美国对印太地区的承诺坚定而持久。我们致力于坚守海洋和天空的自由，肩并肩地与你们一起维护航行自由。美国决心保障你们国家的主权安全，包括在陆地、海洋和数位化世界。〞
 <br/>
 <br/>
 彭斯强调，帝国主义和侵略行为，在印太地区没有立足之地。言语中针对中共的意味十分明显。美国国会外交事务委员会亚太小组主席约霍则直言，融资计划的目标正是〝一带一路〞。
 <br/>
 <br/>
 美国会众院外委会亚太小组主席Ted Yoho：〝我们的目标是建设经济，并提供各国一带一路之外的强有力的不同选择。别忘了，我们的总体目标是从援助转向贸易，如能这样，这些国家及其人民将富裕得多，但条件是他们的政府要尊重基本人权。〞
 <br/>
 <br/>
 峰会期间，与会的李克强主动与彭斯交谈。不过分析认为，川普政府的态度已经非常明确，美中关系的走向将取决于中共的行为，而不是言语。
 <br/>
 <br/>
 美国会众院外委会亚太小组主席Ted Yoho：〝我们希望能有有效的会谈。不是有效的会谈，而是有效的行动。〞
 <br/>
 <br/>
 周六，彭斯与习近平同时出席巴布亚新几内亚举行的APEC年度峰会。两人分别发表主题演讲，内容上互相针对、擦出火花。
 <br/>
 <br/>
 习近平首先发表演说，其中提到〝贸易战不会有真正的赢家〞，同时批评贸易保护主义和单边主义，指责美国的意味不言而喻。
 <br/>
 <br/>
 不过，之后上台演说的彭斯，强硬地回应，除非中共改变不良的行为模式，否则美国不会改变当前的政策方向。
 <br/>
 <br/>
 美国副总统彭斯：〝我们已经针对和中国的（贸易）不平衡采取了决定性的行动。我们对2,500亿中国商品加征关税，还可以把这一数字加倍，但我们期望看到更好的结果。除非中国（共）首先改变其行为，否则美国的方针不会改变。〞
 <br/>
 <br/>
 习近平在演说中宣称，〝一带一路〞项目不是〝陷阱〞，没有地缘政治目的。彭斯则不点名地对此予以反驳。
 <br/>
 <br/>
 美国副总统彭斯：〝不要因为外国债务丧失主权，保护你的利益，保护你的独立，像美国那样，要始终把自己的国家放在第一位。〞
 <br/>
 <br/>
 川普将在月末的G20峰会上与习近平举行会晤。对于美中关系的走向，著名中国问题专家章家敦认为，中共缺少信用与诚意，在关键问题上，中共恐怕很难做出令美国满意的让步。
 <br/>
 <br/>
 撰稿：林宇 剪辑：明玉
