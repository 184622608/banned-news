### 中美洲移民压境 墨西哥：提供援助 含医疗及工作
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】洪都拉斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 难民潮，日夜兼程继续北上，人数已经过万。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 当局周五（26日）宣布，只要留在南部两个州，墨西哥将提供医疗照护、小孩教育与临时工作机会。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 总统潘尼亚尼托（Enrique Pena Nieto）周五公布〝宾至如归〞（make yourself at home）计划，他宣布同意留在奇亚帕斯州（Chiapas）与瓦哈卡州（Oaxaca）的移民，可以〝获得医疗照护，甚至可以送孩子上学〞。
 <br/>
 <br/>
 他说，想要接受他提议的移民，必须向国家移民局（National Migration Institute）递交申请文件。
 <br/>
 <br/>
 潘尼亚尼托说：〝这项计划只适用于遵守墨西哥法律的人，对在墨西哥拥有难民身分者而言，这是让他们迈向永久解决办法的第一步。〞
 <br/>
 <br/>

 <br/>
 这是墨西哥为了解决移民队伍涌向美国问题的第一个广泛且详尽计划。
 <br/>
 <br/>
 这些中美洲移民主要来自洪都拉斯，一些移民计划申请政治庇护，理由是他们在洪都拉斯受到帮派威胁，或是受到尼加拉瓜政府对反对派的攻击威胁。但随着这个群体变得越来越大，人群也变得越来越多样化。绝大多数
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 成员拒绝在墨西哥申请避难，担心这一过程可能会导致他们被拘留或被驱逐出境。
 <br/>
 <br/>
 据报导，大篷车最初形成时规模约200人，进入墨西哥边境时增加到4,000人，穿越墨西哥时增加到约7,000人。目前，这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 日夜兼程继续北上，人数已经过万。该队伍中包括许多曾经被美国驱逐出境的人。先前从美国被驱逐的人明确表示，他们打算非法越境，避开巡逻人员进入美国。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）10月22日发推文表示，美国将开始对那些未能阻止大量移民涌入墨西哥和美国边境的国家切断外援。川普指责危地马拉、洪都拉斯和萨尔瓦多没有〝能够阻止人们离开自己的国家，并非法前往美国〞。
 <br/>
 <br/>
 这3个国家在2017年从美国获得大约5亿美元的资金。〝我们现在将开始切断或随后减少例行给予他们的大规模外援。〞川普发推文说。
 <br/>
 <br/>
 26日，川普再发推文说：〝美国在非法移民上每年花费数十亿美元。这不会再继续了。民主党必须投票支持更有力（但是公平）的法律。如果不行，我们将不得不采取更强硬的措施。〞
 <br/>
 <br/>
 据报，川普为阻止大篷车难民继续北上，考虑出台新的行政令，全面阻止任何人进入美国南部边境，包括难民申请者。
 <br/>
 <br/>
 知情人向《福克斯新闻》透露，美国政府考虑派遣800名军人到边境。周末公布增兵细节。下周二（10月30日），也就是中期选举前一周完成调动。
