### 美智库：朝鲜仍有13个导弹操作基地在运作
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】本周一，美国一个智库发布报告称，研究最新商业卫星图像发现，朝鲜境内至少存在20处未对外宣布的秘密导弹操作基地，其中更至少有13个仍在运作而且近期还进行过小规模设施改进。
 <br/>
 <br/>
 据《美国之音》报导，总部设在华盛顿的〝战略及国际研究中心〞12日发布的报告宣布了上述情况。报告还披露，那些被辨认出的基地分散在偏远的山区，可以储备不同射程的弹道导弹，其中包括据信最远程可以达到美国本土的导弹。
 <br/>
 <br/>
 报告并指出，朝鲜从未承认过这些导弹操作基地，但准确地披露核武和导弹能力是达成去核协议的重要部分，而上述情况突显美国政府想要以和平的方式让平壤放弃核武和导弹项目，所面临的挑战不小。
 <br/>
 <br/>
 朝鲜领导人金正恩在今年稍早时候曾先后与文在寅进行过三次会晤，也曾在今年6月12日与美国总统川普（特朗普）在新加坡会谈，并承诺将朝着〝朝鲜半岛完全无核化〞的方向努力。
 <br/>
 <br/>
 但此后朝鲜在如何实现〝无核化〞承诺的问题上一直没有给出实质性的行动计划，也未采取进一步实质性的弃核行动，却一直力求韩美宣布韩战正式结束并放松对朝鲜的经济制裁，遭到美国政府的拒绝。华府要求平壤当局先拿出具体的无核化行动，然后才会讨论朝鲜方面的上述要求。
 <br/>
 <br/>
 今年9月19日，朝鲜当局从对韩国政府承诺，愿意在外国专家在场的情况下〝永久性〞拆除其主要导弹设施，并称如果美国采取对等行动，朝鲜愿意关闭主要核设施。
 <br/>
 <br/>
 言犹在耳，现在朝鲜被美国智库踢爆仍然在暗中运作并改进十几个可以提高发射常规和核弹头能力的导弹基地，令美国舆论界提出质疑：金正恩当局恐怕一直在从事一个〝大骗局〞，其先前对美韩以及国际社会作出的弃核承诺很可能只是其释放的烟幕弹，目的不过是欺骗国际社会放松警惕和制裁，为其继续暗中发展核武器赢得喘息之机，以图日后〝东山再起〞。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
