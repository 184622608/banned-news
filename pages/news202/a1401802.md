### 气候大会波兰开幕 世银提供2千亿融资
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】联合国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_气候大会.html" target="_blank">
  气候大会
 </a>
 （COP24）12月2日至14日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波兰.html" target="_blank">
  波兰
 </a>
 登场，近200国代表将讨论落实《巴黎协定规则手册》。世界银行周一宣布，将提供2000亿美元作为气候融资。
 <br/>
 <br/>
 三年前《巴黎气候协定 》承诺将全球平均气温升幅限制在低于摄氏2度，而且最好控制在摄氏1.5度以下。
 <br/>
 <br/>
 195个会员国代表接下来两周将拟定落实《巴黎协定规则手册》，规范各国政府如何记录与报告温室气体排放量，以及如何努力减少温室气体排放。
 <br/>
 <br/>
 虽然美国已退出《巴黎气候协定》，国务院仍派遣代表前往参与会议。
 <br/>
 <br/>
 世界银行执行长乔治艾娃（Kristalina Georgieva）周一宣布，将提供2000亿美元作为气候融资。
 <br/>
 <br/>
 世界银行执行长乔治艾娃：〝我们宣布了，将提供2000亿美元作为气候融资，其中1000亿美元来自世界银行集团（IBRD / IDA）的公共部门。〞
 <br/>
 <br/>
 煤炭成为本届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波兰.html" target="_blank">
  波兰
 </a>
 气候会议主要关注的焦点之一。主办国波兰在开幕时呼吁，化石燃料工业转型过程中，要兼顾劳工权益。
 <br/>
 <br/>
 不过，环保人士质疑，这可能是变相支持化石燃料工业继续营运，认为煤矿公司赞助这次波兰气候会议，动机并不单纯。
 <br/>
 <br/>
 最新研究报告显示，全球温室气体排放量回升。这次会议考验各国是否有决心为达成《巴黎气候协定》目标，采取具体措施。
 <br/>
 <br/>
 世界银行执行长乔治艾娃：〝谈判时，将孩子或孙子的照片放在你面前，它将提醒你的职责。〞
 <br/>
 <br/>
 新唐人记者李梅、田瑞综合报导
</div>
