### 疑喷胡椒喷雾 意国夜总会人踩人酿6死120伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 东北部海滨城市安科纳一所
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_夜总会.html" target="_blank">
  夜总会
 </a>
 安排饶舌歌手演出，吸引约千人进场，8日凌晨疑有人在人群中喷洒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡椒喷雾.html" target="_blank">
  胡椒喷雾
 </a>
 ，造成恐慌，发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人踩人.html" target="_blank">
  人踩人
 </a>
 惨剧，至少6死逾120人伤，死者大多为青少年，另包括一名陪同女儿进场的母亲。
 <br/>
 <br/>
 当地媒体报导，出事的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_夜总会.html" target="_blank">
  夜总会
 </a>
 名为〝蓝灯笼〞（Lanterna Azzurra），位于安科纳（Ancona）附近的科里纳尔多镇（Corinaldo）。
 <br/>
 <br/>
 英国天空电视台报导，场内有人喷洒有毒物质，导致大批顾客争相逃走。当地消防部门在Twitter推文：〝意外肇因或为（有人）喷洒令人感觉刺痛的物质。〞
 <br/>
 <br/>
 一名幸存少年说，事发于当地时间凌晨1时，他与友人在夜总会内跳舞，等待表演者上台时，突闻到一阵刺鼻的气味，随即跟随人群跑向其中一个紧急出口，但出口却被锁上，在场保安要求他们走回会场。
 <br/>
 <br/>
 现场混乱不堪，很多伤者被辗压受伤，部分人更出现骨折。事发后，多辆救护车、消防车及警车到场，即时对伤者简易包扎及送医治理。
