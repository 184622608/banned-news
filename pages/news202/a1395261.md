### 暴风雪又遇山体滑坡 韩团登喜马拉雅山疑8死1失踪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】5名韩国人和4名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼泊尔.html" target="_blank">
  尼泊尔
 </a>
 导游组成的队伍，近日在尼泊尔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_喜马拉雅山脉.html" target="_blank">
  喜马拉雅山脉
 </a>
 葛迦山（Mount Gurja）营地，疑似遭暴风雪袭击。周六（13日）救援直升机已从空中看到8具遗体，另1人失踪。
 <br/>
 <br/>
 该团队10月7日出发，疑似因遇到暴风雪待在大本营内，等待天气好转后继续前行。登山组织者在与他们失联24小时后报警。据目击者称，暴风雪以及随即发生的山体滑坡将大本营埋没。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼泊尔.html" target="_blank">
  尼泊尔
 </a>
 警方发言人ailesh Thapa周六告诉法新社，他们认为登山队遭遇了暴风雪，因为附近的树木都被折断、帐篷被吹到散架，甚至连遗体也散落四处。虽然救援直升机周六抵达营地上空，从空中看到8具遗体，另1人的下落不明，但因为条件恶劣直升机无法停留。
 <br/>
 <br/>
 警方救援队目前正在步行前往现场，而直升机则有望在周日（14日）天气状况有所好转后再飞往营地。葛迦山山峰海拔7193米，营地位于3500米，离最近的村庄徒步至少需要一天。
 <br/>
 <br/>
 据悉，韩国知名登山家金昌浩（Kim, Chang-Ho），也在这次意外中丧命。他是第一个无氧登顶14座海拔8000米及以上山峰的韩国登山家。他还打破了当时由一位波兰登山者保持的耗时纪录。
