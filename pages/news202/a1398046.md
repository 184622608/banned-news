### 亚洲最大规模的街头表演 日本大道艺世界杯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】从11月1日开始为期四天的大道艺世界杯在日本静冈举行。来自23个国家的102组团队聚集静冈，给观众带来精湛技艺。
 <br/>
 <br/>
 记者现在位于日本静冈市的骏府城公园，作为静冈的一大盛典，大道艺世界杯每年都吸引上百万人前来观看，就是为了能一睹世界级别的街头艺术表演。
 <br/>
 <br/>
 大道艺的舞台就在骏府台公园和静冈的街道上。集合了魔术，杂技，高空舞蹈，口技等技艺，来自世界各地的街头艺人们各个身怀绝技，大显身手。
 <br/>
 <br/>
 五颜六色的篮球从俩个到三个，随着音乐的节奏在艺人的手里好像被施了魔法，可以在手指上自由转动。
 <br/>
 <br/>
 谁说跳舞一定要在平地上，两根绳索一样可以变换各种花样，做出各种高难动作，来自俄罗斯的艺人用他们惊人的臂力凌空而舞。
 <br/>
 <br/>
 作为日本本土的艺人村田奈央身着和服，随着日本传统音乐在翩翩起舞中融合着魔术的元素给人耳目一新的感觉。
 <br/>
 <br/>
 不管是铁质的巨大铁环还是点燃的火棒都好似被两位来自台湾的舞者注入了生命力，他们可以游刃有余的操控。火舞大环的完美契合，同时给观众带来感动和热情。
 <br/>
 <br/>
 台湾艺人蔡宏毅、杨世豪：〝我们的中国团名叫做星火燎圆，圆是圆形的圆，那个圆存在我们心中，也存在于每个观众心中。我们希望观众看完我们的表演，可以更勇敢的面对自己的梦想，自己想做的事，追随心中的圆走下去。〞
 <br/>
 <br/>
 去年大道艺世界杯的冠军得主张海轮在今年依然是全场的焦点。不管是传统技艺变脸，还是在六张椅子的高度上自由平衡的倒立，都完美呈现了中国传统杂技的精髓。
 <br/>
 <br/>
 观众：〝做这样危险的动作真的不要紧吗，大家担心的都忘了鼓掌了。像这样的表演我还没有看过，我希望这样的中国人能来日本，能够使中日更友好，那就太好了〞
 <br/>
 <br/>
 大道艺世界杯不仅促进了静冈的经济还增进了国际间的文化交流。
 <br/>
 <br/>
 大道艺世界杯実行委员长杉山元：〝（这样的活动使）很多客人来到静冈，当然包括海外的客人，这样就使我们增加了很多交流的机会〞
 <br/>
 <br/>
 大道艺世界杯从1992年开始，每年都会举办，今年预估会有150万人前来观看。
 <br/>
 <br/>
 新唐人记者文萱日本静冈报导
