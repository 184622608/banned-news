### 〝关税强人〞川普促中共行动 贸易里根时代来临？
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】G20川习会已过去4天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 官媒首次承认美国对中方暂缓升级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 90天的时间限制。同时，在保护知识产权方面，中共连续有新行动。有专家认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 将可能带来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球贸易.html" target="_blank">
  全球贸易
 </a>
 的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根时代.html" target="_blank">
  里根时代
 </a>
 〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在今日推文中说，中国给出很强的讯号让他相信，习近平会兑现他在G20会晤中的承诺。
 <br/>
 <br/>
 继周一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 国家发改委等30多部委签署并发通知，将惩戒知识产权失信者后，中共总理李克强周三主持会议通过《专利法修正案（草案）》，加大打击侵犯知识产权。同一天，官媒并首次承认美方90天的暂缓升级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 期限。
 <br/>
 <br/>
 川普虽然还未就双方此次会晤做出正式说明，但从2日至今的推文中，可以看出他不断强调要达成〝真正的协议〞，否则就没有协议；没有协议，接踵而来的就是关税。
 <br/>
 <br/>
 美国财长姆努钦重申，90天内达成的协议一定要是可实现的、有具体时间表和细节的方案；川普总统真正关注的是基础性、结构性的问题。
 <br/>
 <br/>
 美国财长 姆努钦：〝让我说明一下这个问题，因为一直好像有人迷惑，我认为总统川普始终关注的是结构性的问题。他一直在说的也是两个中国（的概念），如果我们可以纠正这些问题、中国能开放市场，贸易赤字问题会自己（由市场）解决。我们也有协议如果中共不兑现承诺会被惩罚。〞
 <br/>
 <br/>
 专家认为，川普对中共的关税战，是其整体策略的一环，来源于其理念。通过关税最大化施压，促使习近平改变中共体制，进而重塑世界格局。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院讲席教授 谢田：〝要分析川普的行为必须从他的理念和动机入手。我们知道他是一个强调回归传统、在全球铲除社会主义、共产主义邪恶势力的这样一个回归传统、回归保守的美国总统，对中共的策略是他在全球铲除共产主义的最重要一环，他使用经济贸易的手段、外交手段改变中国的经济结构从而改变中国的政治结构。〞
 <br/>
 <br/>
 前北大经济学教授、旅美学者 夏业良：〝川普年轻的时候、30多岁的时候就展示了他对国际事务的看法，一方面他很羡慕和敬佩里根总统，另一方面他对当时以苏联为首的共产主义阵营表示出一种强烈对抗的决心。贸易是其中一个重要手段，其实他是要重塑未来世界格局。所以这次非常关键，对美国没有什么损失，就是向后延长90天而已，但是90天后如果没有见效，那（对中共）惩罚的措施和力度可能比现在还要强。以前川普在竞选的时候曾经说到可能对中共课以40%的高关税，所以后面的牌还有很多。〞
 <br/>
 <br/>
 川普在3日推文中提到，除了贸易，他与习近平还能在朝鲜无核化等更多议题上发挥积极影响；也希望未来与习近平、普京在军备竞赛上能有所协议。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
