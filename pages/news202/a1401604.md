### 抗议涨燃油税 法国消防员加入〝黄背心〞运动
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】周六，法国南部300名消防员，也加入〝黄背心〞运动，抗议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃油税.html" target="_blank">
  燃油税
 </a>
 上涨。
 <br/>
 <br/>
 法国东南部昂蒂布镇的300名消防员，周六在高速公路的一处收费站举行抗议活动。他们身穿象征着抗议运动的〝黄背心〞，手举信号弹，并用路障拦住部分道路，以引起过路者的关注。
 <br/>
 <br/>
 有抗议者表示，他们是为了表达自己的公民诉求。
 <br/>
 <br/>
 职业消防员国家自治联合会主席格瑞蒂：〝我不会说是联合会参加了‘黄背心’运动，这只是身为一名公民的行为。〞
 <br/>
 <br/>
 职业消防员国家自治联合会主席格瑞蒂：〝各省不在大城市居住的法国人，总体上受困于缺少谋生方法，来支撑整个家庭所需的开销。〞
 <br/>
 <br/>
 法国总统马克龙执政以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃油税.html" target="_blank">
  燃油税
 </a>
 不断上涨。马克龙此前以推广绿色能源为由，再次计划增加燃油税，引发了席卷全国的〝黄背心〞抗议运动。不过，马克龙表达强硬立场，拒绝让步。
 <br/>
 <br/>
 据警察工会估计，目前在法国大约仍有31,000名抗议者，并设置了582处的路障。
 <br/>
 <br/>
 新唐人记者林宇综合报导
