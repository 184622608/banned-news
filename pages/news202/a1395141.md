### 德日联手〝五眼联盟〞 7国围堵中共海外情报扩张
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】日前，路透社披露，由美国、澳洲、新西兰、加拿大、英国创立的情报共享网络〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五眼联盟.html" target="_blank">
  五眼联盟
 </a>
 〞（five eyes）正在加强与德国、日本的合作，为了更有效
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遏制中共.html" target="_blank">
  遏制中共
 </a>
 在全球的势力扩张，今年以来，成员国们持续稳定地在交换有关中共海外活动的情报。
 <br/>
 <br/>
 路透社的这篇报导称，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五眼联盟.html" target="_blank">
  五眼联盟
 </a>
 〞当前增强与伙伴国的合作，主要是为应付中共在全球内的政治经济战略扩张与布局。
 <br/>
 <br/>
 报导引述一位美国官员称，〝我们的盟友，包括志同道合的伙伴，经常谈到如何应对中国（中共）自信的国际战略，而且正在凝聚共识。〞
 <br/>
 <br/>
 然后，这位官员又透露称，现在可能已开始讨论〝更深层的合作方案〞。不过该官员没有透露相关合作方案的具体内容。
 <br/>
 <br/>
 截至目前，德国政府和日本政府都没有就〝五眼联盟〞的最新近况作出评论。倒是中共外交部发言人陆慷于本月12日话有所指地称，希望其他国家能够对中国（中共政府）持〝包容〞的心态，〝不要搞小团体〞。
 <br/>
 <br/>
 最近两年，面对中共势力在海外大肆扩张，越来越多的国家对此提高了警惕。
 <br/>
 <br/>
 就在今年5月下旬，由于新西兰执政工党的某高层政要被曝接受了与中共关系密切的华裔议员杨健的重大政治献金，更有消息指国家党的前任总理英格利希经常向杨健通报情况，导致美国一些政要开始忧虑新西兰可能正步澳洲的后尘，面临中共势力的入侵与干预。以至于中情局前分析师马蒂斯等人一度提出建议，指考虑到事态可能已经涉及中共势力入侵〝政治核心〞的问题，希望〝五眼联盟〞 讨论是否应该继续让新西兰留在这个共享情报联盟内。
 <br/>
 <br/>
 不过，对于上述顾虑，新西兰现任和前任总理都断然否认该国政府存在任何受到中共影响的问题，甚至连该国的反对党的领袖，也否认国家党议员杨健从前总理处分享信息的说法，并表示国家党遵守政治捐赠法，并没有看到任何不正当影响的迹象。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
