### 韩朝边界传枪响 韩士兵厕所中弹身亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩朝边界.html" target="_blank">
  韩朝边界
 </a>
 16日传出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪响.html" target="_blank">
  枪响
 </a>
 ，一名韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_士兵中弹.html" target="_blank">
  士兵中弹
 </a>
 ，送医途中不治。韩国军方正调查该名士兵死因，当局也释出事件似乎跟朝鲜无关。
 <br/>
 <br/>
 韩国军方表示，死者是21岁姓金（Kim）的一等兵，今年8月才被调到该单位，负责感热观察设备。
 <br/>
 <br/>
 16日傍晚5时，金姓士兵在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩朝边界.html" target="_blank">
  韩朝边界
 </a>
 的东部岗哨厕所内被发现中弹，送医途中不治。
 <br/>
 <br/>
 韩国官员表示，事件发生时，该名士兵正在站夜哨。一名资深军官表示：〝我听到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪响.html" target="_blank">
  枪响
 </a>
 后立刻跑到厕所，发现他中弹倒卧在地。〞
 <br/>
 <br/>
 军官表示，当局正在调查该名士兵死因。另案发至今，并未发现朝鲜士兵有异常行动，此起事件似乎与朝鲜无关。
 <br/>
 <br/>
