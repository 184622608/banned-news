### 多国民权代表聚巴黎 促推进中国民主
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】10月8日，法国多个人权团体，共同举办了为期三天的有关维护
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人权.html" target="_blank">
  中国人权
 </a>
 及推进民主价值观的研讨会，世界多国的人权代表出席了此次会议。
 <br/>
 <br/>
 金秋，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎.html" target="_blank">
  巴黎
 </a>
 位于13区的会议厅里，来自澳大利亚、新西兰、加拿大、香港及欧洲20国等的人权代表及法国法兰西中国团结等多个人权团体，还有支持人权的国际反死刑反酷刑等多个组织出席了这场题为：支持大陆与台湾自由民主化法兰西大会。
 <br/>
 <br/>
 民主中国阵线副主席张建： 我们为什么召开大会呢，主要是考虑到海外民主团体还有信仰团体大家在一起能够抱柴取暖，同时，为中国现在的民主和自由和法制这个现状，大家今后通力合作，推动中国的信仰自由、言论自由这一切。
 <br/>
 <br/>
 作家及加拿大民主中国阵线副主席盛雪：中共自己它确确实实是一步步走向绝路，当然了整个社会各个阶层的觉醒以及开始越来越多的人开始进行实质性的抗暴行动，也起了很大的作用。中共迫害法轮功后，其实法轮功学员的反抗是特别值得大家敬仰的，而这个过程当中，是付出了特别多，包括人体活摘的事件，在这个现代文明史上可以说是空前绝后的这样一个暴行。
 <br/>
 <br/>
 盛雪表示中共是当今人类文明史上的最大敌人。
 <br/>
 <br/>
 <br/>
 澳大利亚悉尼民主中国阵线总部监事：张小刚：现在我们看到很多人、包括信仰的，拆迁的、包括做生意的、包括这些炒股票的、在某一个时刻，都会被中国共产党认为是它们的威胁，他要用权力去镇压，只要共产党存在一天没有任何人是安全的。
 <br/>
 <br/>
 澳州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国民主.html" target="_blank">
  中国民主
 </a>
 阵线论坛主席秦晋：法轮功的强势出现，一定会让中共的高层坐卧不安。秦晋表示，法轮功真善忍的力量值得大家敬佩，人们看到了天象下的希望与未来。
 <br/>
 <br/>
 澳州中国民主阵线论坛主席秦晋：这天时的变化已经到来了，对于中国共产党来说它的好日子到头了，整个西方国际社会都会对中共的态度发生根本性的逆转，在这种国际国内的双重压力下，中共很容易引起崩溃。这也是我们在可见的将来，可以看的到的。
 <br/>
 <br/>
 新唐人记者鑫研、思远法国巴黎报导
 <br/>
