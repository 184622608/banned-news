### 经济学人：遏制中共成美国整体共识 两党在比谁更强硬
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】英国《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济学人.html" target="_blank">
  经济学人
 </a>
 》杂志最新一期以《中美对抗》为封面故事，指出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统川普（特朗普）对中共的态度是正确的做法。同时美国两党已经就遏制中共形成共识，并在互相比较看谁对中共更加强硬。
 <br/>
 <br/>
 据《法广》引述《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济学人.html" target="_blank">
  经济学人
 </a>
 》的这篇文章，在过去的四分之一的世纪里，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 对中共的态度建基于一个〝聚合〞的信念，政治和经济的融合不但可助中国致富，并使其更开放、多元化以及民主。美国一直以为只要给予中共一个正确的诱因，中国最终会成为一个负责任的世界秩序维护者。
 <br/>
 <br/>
 文章说，〝聚合〞已经死去。川普政府指控中共介入美国的政治和文化，偷窃智慧财产，从事不公平贸易，以及企图不只在亚洲称霸，还窥伺支配全球的野心，是一个邪恶和破坏规矩的角色。
 <br/>
 <br/>
 早前美国副总统彭斯曾警告中共正从事〝全政府动员〞的攻击。文章称，这并不只是彭斯和川普的想法，民主党和共和党已经在相互比较看谁对中共更强硬。这是自从上世纪40年代以来，美国商界、外交官员、军方的态度，首次如此快速地达成共识，认定美国正面对一个意识形态和战略性的敌人。
 <br/>
 <br/>
 文章指出，川普政府针对中共做了三点有效措施。首先是强大美国，一切以国家安全为重。比如，加强审批企业的转手或合并；早前将一名涉嫌从事间谍活动的中共国安人员从比利时引渡回美国加以审判；以及增加国防预算，增加援外经费，对付中共在海外的撒钱行为等。
 <br/>
 <br/>
 其次，川普成功提醒美国需要对中共行为的期望有所调整。文章说，中共利用金钱影响小国的外交政策。如今，西方的政党、智库和大学部门，需要更公开他们的捐款来源。同时，国企也得到政府的津贴和保护，在海外收购多用途的科技或扭曲国际市场秩序。
 <br/>
 <br/>
 第三点是川普政府在贸易问题上丝毫不含蓄的出招和强硬的姿态，已经使得中共领导人方寸大乱，而他们还以为美国不会贸然搞砸这个局面。
 <br/>
 <br/>
 文章同时指出，川普对待中共的态度，是完全正确的做法。但到了目前这个阶段，美国需要一个完整的战略，而非只是战术。这个战略包括推行美国自由民主的价值观，因中共比任何西方民主都要烂。同时，在贸易方面与欧盟和日本联手，迫使中共改变，在防卫问题上维持与老朋友如日本和澳洲的关系，并结交如印度和越南等新朋友，因为盟友是美国对中共最好的防卫力量。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：祝馨睿）
 <br/>
