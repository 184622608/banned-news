### 油价物价学费都要涨 法国抗议活动加剧
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 民众的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄背心.html" target="_blank">
  黄背心
 </a>
 运动〞进入第三周，抗议的对象从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃油税.html" target="_blank">
  燃油税
 </a>
 扩展到了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高物价.html" target="_blank">
  高物价
 </a>
 ，而且巴黎的游行演变成了的最严重骚乱。周一，中学生和救护车司机又发起了新一轮抗议。
 <br/>
 <br/>
 两周的抗议活动中，已有四人死亡。新近去世的是一位80岁女士。上周六马赛发生暴力示威，警察动用催泪弹。这位女士准备关窗户时，被飞来的催泪弹击中面部，不治身亡。
 <br/>
 <br/>
 周一抗议继续。大约50人封锁了马赛附近一家油库的入口道路，全国加油站已几乎无油可加。
 <br/>
 <br/>
 抗议者要求政府取消明年1月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃油税.html" target="_blank">
  燃油税
 </a>
 上涨计划，并且呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 下台。
 <br/>
 <br/>
 抗议人士还取消了原定的周二和总理菲利普的会晤。
 <br/>
 <br/>
 抗议者两周来在巴黎最繁华的街区造成严重破坏，焚烧了数十辆汽车，抢劫了多家精品店，毁坏了住宅和咖啡馆，这是自1968年以来巴黎发生的最严重的骚乱。
 <br/>
 <br/>
 周一，马克龙召开紧急部长会议商量对策。据报道，会上没有讨论是否实施国家紧急状态，但也没排除该可能。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 财长勒梅尔和商界代表见面，评估周末暴力抗议带来的损失。他说，一些零售商和餐馆的销售额下降了20-50%。
 <br/>
 <br/>
 勒梅尔表示，政府需要采取措施保证民众得到更多利益
 <br/>
 <br/>
 法国财政部长勒梅尔：〝因此我们必须加快减税计划。而为此，我们必须加快削减公共开支。〞
 <br/>
 <br/>
 巴黎救护车司机周一也举行了抗议，从巴黎协和广场到法国国民议会的道路被数十辆救护车封锁。抗议人士认为，马克龙的一系列社会保险和医疗改革将让救护车公司陷入生存危机。
 <br/>
 <br/>
 全法大约100所高中的学生也在周一上街抗议教育和考试改革。
 <br/>
 <br/>
 而截至周日中午，有25万人签名抗议法国公立大学将非欧盟学生的注册费提高十几倍。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
