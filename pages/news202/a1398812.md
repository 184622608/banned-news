### 牙买加客机迫降冲出跑道 机翼引擎断裂6伤(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】法属地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圭亚那.html" target="_blank">
  圭亚那
 </a>
 （也译盖亚那）交通部长派特森9日表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牙买加.html" target="_blank">
  牙买加
 </a>
 航空公司一架载有126人的班机，在首都乔治城机场紧急降落，造成6人受伤。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牙买加.html" target="_blank">
  牙买加
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞天航空.html" target="_blank">
  飞天航空
 </a>
 （Fly Jamaica Airways）这架编号OJ256波音757-200型客机，当时搭载118名乘客，和8名机组人员，其中82名乘客是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 人。9日凌晨2时10分，预定从乔治城机场（Georgetown）飞往加拿大多伦多。
 <br/>
 <br/>
 但起飞后约20分钟，飞机液压系统出现问题，在空中盘旋约45分钟后折返乔治城机场紧急降落，惊险的滑离跑道，机身冒出白烟，机翼断裂，机场人员紧急派出消防员洒水降温。
 <br/>
 <br/>
 派特森（David Patterson）表示，有6名伤者已送至机场附近医院治疗，他们没有生命危险，状况稳定，未传出有人骨折。
 <br/>
 <br/>
 从视频中可以看到，飞机的左机翼直接断掉，引擎断开严重损坏，场面十分惊人。
