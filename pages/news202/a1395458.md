### 记者失踪案 土耳其沙特联合彻查
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 称，将联合搜查沙特大使馆，调查沙特知名记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉失踪案.html" target="_blank">
  卡舒吉失踪案
 </a>
 。卡舒吉是美国居民，美国总统川普表示，已经和沙特国王通话。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 外交人士表示，土耳其和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 周一下午联合行动，进入沙特阿拉伯领事馆搜查，调查沙特记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉失踪案.html" target="_blank">
  卡舒吉失踪案
 </a>
 。
 <br/>
 <br/>
 卡舒吉是沙特知名记者，曾是沙特王室顾问，但经常批评王储。本月2号，他进入沙特驻伊斯坦布尔领事馆后失踪。土耳其认为他被谋杀后，尸体被移走，但沙特则否认这些指控。
 <br/>
 <br/>
 据报，土耳其情报部门称，当局取得一段录音，证明卡舒吉进入领事馆后已经被杀害。
 <br/>
 <br/>
 卡舒吉是美国居民，其失踪事件引起国际社会关注。
 <br/>
 <br/>
 周一， 美国总统川普离开白宫去佛州前，对媒体表示，他刚和沙滩国王通话月20分钟，国王否认了解卡舒吉失踪事件，但正在与土耳其合作调查。
 <br/>
 <br/>
 美国总统川普：〝我问他，他坚决否认。〞
 <br/>
 <br/>
 美国总统川普：〝国王告诉我，土耳其和沙特正在联手合作，密切合作，彻底调查事件。〞
 <br/>
 <br/>
 川普说，他已经告诉国务卿蓬佩奥，如有必要，将立即赶往沙特。目前，有必要就去土耳其。
 <br/>
 <br/>
 川普表示，全世界都在关注这件事，沙特和土耳其能够达成协议，合作调查事件，是件好事。
 <br/>
 <br/>
 新唐人记者李梅、李静明综合报导
</div>
