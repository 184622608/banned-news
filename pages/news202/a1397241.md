### 时事拼盘：红袜世界大赛夺冠 英超俱乐部老板坠机身亡
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】下面请看一组新闻简讯
 <br/>
 <br/>
 1. 周日，美国职棒大联盟年度总冠军〝世界大赛〞第5战中，波士顿红袜队打出4支全垒打，5比1击败洛杉矶道奇队。红袜队以总战绩4胜1败，第九次夺得世界大赛冠军。周日一战，双方打到第18局，历时7小时20分，创下世界大赛局数最多、时间最久的纪录。
 <br/>
 <br/>
 2. 万圣节给孩子们发糖果，美国总统川普和夫人梅拉尼娅也不例外。周日，几百名来自军人家庭的儿童来到白宫，接受了美国第一家庭发给他们的糖果。有的孩子还和川普夫妇合影。白宫也被装饰得充满节日气氛。
 <br/>
 <br/>
 3. 欧洲大部分国家周日将时钟回拨一小时，进入冬令时。不过，这可能是欧盟国家最后一次回拨钟表，因为欧洲委员会上个月宣布一项废除夏令时的计划。周一，欧盟各国交通部长在布鲁塞尔举行非正式会议，首次对该计划进行辩论。
 <br/>
 <br/>
 4. 周一，英超莱切斯特城俱乐部球员和球迷来到王权体育馆悼念俱乐部老板维猜。周六，维猜因私人直升机在体育场外坠毁而遇难。他是泰国华裔，身价49亿美元，是泰国第五大富豪。
</div>
