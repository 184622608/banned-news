### 古巴再传声波攻击 加拿大外交人员患怪病返国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】本周四，加拿大外交部对外证实，该国驻古巴工作人员中，有人遭受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_声波攻击.html" target="_blank">
  声波攻击
 </a>
 出现神秘症状，已获准返国。近年来，美国和加拿大驻古巴外交人员频频遭到疑似
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_声波武器.html" target="_blank">
  声波武器
 </a>
 攻击，导致部分人员出现脑部、中枢神经系统损伤。去年在中国也有美国外交人员遭到类似袭击，令声波攻击疑案更趋复杂。
 <br/>
 <br/>
 加拿大一名高级官员29日在记者会上证实，该国一名驻古巴的外交人员今夏出现神秘症状，医疗团队对其进行检测后，研判是受高科技
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_声波攻击.html" target="_blank">
  声波攻击
 </a>
 ，目前该人员已获外交部同意返国。
 <br/>
 <br/>
 加拿大官员没有披露这名外交人员具体的病情，但他披露，为了安全起见，加国政府从今年4月起，就不允许家属陪同外交人员前往古巴，目前驻古巴的加国外交人员仅剩12人。
 <br/>
 <br/>
 此前，因遭受疑似
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_声波武器.html" target="_blank">
  声波武器
 </a>
 攻击而出现离奇症状的加国国民已有12人，加上最近这一例，受害人员增至13人，其中包括8名外交人员及5名家属。
 <br/>
 <br/>
 事实上，最近两年美国驻古巴的外交人员及家属也有26人先后遭受声波攻击而出现类似于脑震荡和轻度创伤性脑损伤的症状。
 <br/>
 <br/>
 据美国国务院的报告，这些遭受声波攻击的人员都说自己听到奇怪的声音后，出现了头痛、恶心、听力下降、认知问题和其它健康问题。这个事件曾一度导致美国与古巴的关系恶化，并促使美国驱逐古巴外交官。
 <br/>
 <br/>
 然而事情到了今年变得更为复杂，因为在中国大陆竟然也出现了美国外交人员疑似遭受〝声波攻击〞的事件，令本就因为贸易冲突而十分紧张的中美关系雪上加霜。
 <br/>
 <br/>
 今年5月23日，美国国务院对身在中国的美国公民发出警告，要他们小心会造成轻度创伤性脑部损伤的〝声波攻击〞(Sonic Attack)，因为当局发现派驻在广州市的驻外人员出现了类似的症状。
 <br/>
 <br/>
 根据美国驻广州市外交人员当时的报告，从 2017年年末到 2018年4月，出事的这么美国外交人员听到了微弱、模糊和异常的声音，身体也感到了异常的压力。该人员随即被送回美国接受医疗评估和检查，最后得出的结果是他受到了轻微创伤性脑损伤。
 <br/>
 <br/>
 今年6月8日，美国国务院又发布了针对全中国范围内的〝健康警报〞。这份声明表示，美国国务院已收到医学确认，那些驻华工作人员出现的奇怪症状，与美国驻哈瓦那大使馆工作人员报告的症状一致，都是出现了〝头晕、头痛、耳鸣、疲劳、认知问题、视力问题、耳部和听力损失，以及睡眠困难。〞为此，美国政府提醒所有在中国大陆境内的美国公民，一旦出现类似异常现象应立即联系就医检查，并告知美国大使馆。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑:东野）
