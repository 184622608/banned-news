### 应对难民潮 美防长批准南部边境增兵
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】美国国防部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蒂斯.html" target="_blank">
  马蒂斯
 </a>
 批准了国土安全部的要求，将派遣更多军人前往南部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境.html" target="_blank">
  边境
 </a>
 ，阻止大批非法移民从墨西哥进入美国。
 <br/>
 <br/>
 知情人向《福克斯新闻》透露，美国政府考虑派遣800名军人到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境.html" target="_blank">
  边境
 </a>
 。周末公布增兵细节。下周二，也就是中期选举前一周完成调动。
 <br/>
 <br/>
 美国总统川普早些时候曾表示，他将〝派出军队〞来解决他所说的边境上的国家紧急状态。
 <br/>
 <br/>
 成千上万的中美洲移民正在从南向北穿越墨西哥，距离美墨边境大约一千英里，希望最终进入美国。
 <br/>
 <br/>
 一位官员表示，大部分增援人员都是现役军人，但因为美国《地方安保法》（Posse Comitatus Act）禁止军人在国内和警察一样执法，所以增派人员不包括作战部队，而是为边境巡逻队提供后勤支援，其中包括工程师，医务人员和飞行人员。
 <br/>
 <br/>
 目前，已经有大约2000名国民警卫队成员在边境协助执法。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
