### 入境美国希望渺茫  大蓬车移民改寻〝墨西哥梦〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】眼见入境美国的希望渺茫，数千滞留美国与墨西哥边界的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 ，只好逐渐打消〝美国梦〞，换成寻求〝墨西哥梦〞。目前在墨国政府设立的工作计划下，当中2250人在墨北边境登记找工作、申请签证及社会保险福利。
 <br/>
 <br/>
 墨西哥这项工作计划目的是劝他们留下来，而非加紧脚步前往美国，虽然那里工资虽较高，但美国视这些不明身分的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 为非法入侵者。
 <br/>
 <br/>
 不过实现墨西哥梦也是漫长的进程，据估计这波
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大蓬车.html" target="_blank">
  大蓬车
 </a>
 移民约有6000位，其中有2250人在边境城市蒂华纳（Tijuana）登记找工作、申请签证及社会保险福利。蒂华纳位于加州圣地牙哥对面。
 <br/>
 <br/>
 但目前仅有15位移民确实受雇，26岁的洪都拉斯青年格瓦拉（Denny Guevara）就是其中一人。格瓦拉在抵达蒂华纳后，一周内成功取得工作及居留文件。他一家超市担任理货员，他说，〝看来这就是我的命运。〞
 <br/>
 <br/>

 <br/>
 他说：〝我想要的是更好的生活，逃离我国的暴力，一个找到工作的机会。〞而他的梦想，是赚够钱寄给留在洪都拉斯的妻子和3个孩子。
 <br/>
 <br/>
 23岁的萨尔瓦多妈妈罗培兹（Yansy Lopez）也是移民之一。她赶往面试前表示：〝他们愿意让我每天装箱12小时，从周一到周六，每周给我2000披索（约97美元）。〞她在家乡时也为美国服装品牌做过同样工作。
 <br/>
 <br/>
 罗培兹说，在萨国暴力、恶名昭彰的帮派成员虐杀她的继子后辞职。她当时薪水也差不多，还有奖金。她说：〝我是最快的，我装的箱子比大家多。〞她微笑着说：〝如果他们这里给我工作，哎呀，总之我先前有经验。〞
 <br/>
 <br/>
 也在超市找到工作的27岁洪都拉斯汽车技工贾西亚（Edwin Garcia）说，他从没有过这么体面的工作。他表示，当初离开洪都拉斯是因为穷到连帮女儿买牛奶都买不起。
 <br/>
 <br/>
 他说，新工作〝就像一场梦，我试着接受这一切…这是美好的感觉，好似我融入（墨西哥）社会〞。
