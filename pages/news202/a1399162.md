### 以巴冲突再起 加沙数百枚火箭狂轰以色列
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】来自巴勒斯坦组织哈玛斯集团数百枚火箭，12日自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加沙.html" target="_blank">
  加沙
 </a>
 (加萨)走廊炮击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 ，以军也发动空袭回击。而这次冲突，据悉是自2014年加沙战争以来，巴勒斯坦最猛烈的炮击。
 <br/>
 <br/>
 12日傍晚，巴勒斯坦激进组织
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈马斯.html" target="_blank">
  哈马斯
 </a>
 （Hamas），向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 发射数百枚火箭，空袭警报从黑夜响到白天。以军也袭击70多处激进分子据点，以示回应。
 <br/>
 <br/>
 点点火光宛如流星划过天空，以色列〝铁穹〞防空系统拦截飞弹。
 <br/>
 <br/>
 以军声称击中哈马斯军情总部，以及旗下艾格撒电视台（Al-Aqsa TV），爆炸产生的烈焰，照亮整个夜空。目前未传人员伤亡，工作人员据信在以军空袭前的鸣枪示警后撤离大楼。
 <br/>
 <br/>
 而飞弹从天而降，撃中以色列一辆路过的巴士，后半部炸到粉碎，冒出巨大火光。以色列军方说，巴士被反战车飞弹击中，造成数人受伤，有一名军人重伤。
 <br/>
 <br/>
 <center>

   <br/>
   <font size="-1">

   </font>
  </a>
 </center>
 <br/>
 以军表示，飞弹防御系统已拦截数十枚来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加沙.html" target="_blank">
  加沙
 </a>
 的火箭，其他大多落在空旷地区，不过有些击中房屋和其他民用建筑。医护人员通报约有20名以色列人受伤。
 <br/>
 <br/>
 另外，加沙卫生部表示，以色列空袭造成巴勒斯坦人3死9伤。
 <br/>
 <br/>
 这次冲突，据传导火线是11日深夜，以色列特种部队深入加沙走廊（Gaza Strip），歼灭7名哈马斯军官，但是同样传出有人员阵亡。哈玛斯（Hamas）指控以军发动斩首行动，誓言报复。
 <br/>
 <br/>
 据悉，以色列9日才同意卡达政府为哈马斯送入1500万美元现金，协助调停，但不到48小时，双方无预警再次开火，恐让终结数月动荡的努力受阻。
 <br/>
 <br/>
 联合国秘书长古特瑞斯（Antonio Guterres）发言人说，古特瑞斯12日敦促各方展现最大程度克制。
 <br/>
 <br/>
 声明说，联合国中东和平进程特使穆拉狄诺夫（Nickolay Mladenov）持续与埃及和有关各方密切合作以恢复平静。
