### 两伊边境6.3强震 伊朗逾600人伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊拉克.html" target="_blank">
  伊拉克
 </a>
 边界附近，25日深夜发生规模6.3
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 造成600多人受伤，所幸多为轻伤，还没传出人员死亡。
 <br/>
 <br/>
 这起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 至少7个省分可感受到摇晃，其中以西部克尔曼沙省（Kermanshah）震度最大。地震导致部分地区出现山崩，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 当局表示救难团队已进入所有城镇与乡间。
 <br/>
 <br/>
 伊朗国家电视台引述官员说法：〝受伤人数已达646人…多数人由于伤势轻微，并未住院。〞
 <br/>
 <br/>
 电视播出萨尔波勒扎哈卜（Sarpol-e Zahab）镇房屋损毁画面，今早还测得2起规模分别为5.2及4.6的余震。
 <br/>
 <br/>
 由于当地去年在强震后，仍有许多人无家可归。对余震的担忧迫使人们冒着寒夜露宿街头。
 <br/>
 <br/>
 克尔曼沙省去年曾遭到7.3强震，造成600多人死亡及数千人受伤，是伊朗十几年来死伤最惨重的地震。
 <br/>
 <br/>
 伊朗坐落主要断层带，容易发生频繁地震。这次强震连远在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊拉克.html" target="_blank">
  伊拉克
 </a>
 首都巴格达、库德族自治区的阿比尔（Erbil）甚至科威特等地都可感受到，但皆未传出灾情。
