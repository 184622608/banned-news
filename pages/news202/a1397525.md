### 推翻死刑 巴基斯坦高院：天主教徒无罪
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】巴基斯坦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天主教徒.html" target="_blank">
  天主教徒
 </a>
 毕比因〝一杯水〞被指控侮辱先知罪，2010年被判
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_死刑.html" target="_blank">
  死刑
 </a>
 。这起案件曾引发国际广泛关注。周二，巴基斯坦最高法院推翻有罪裁决，宣布无罪释放毕比。
 <br/>
 <br/>
 周三，巴基斯坦最高法院推翻对毕比的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_死刑.html" target="_blank">
  死刑
 </a>
 裁决。
 <br/>
 <br/>
 毕比的辩护律师表示，毕比已被无罪释放，他们还将依法寻求补偿。
 <br/>
 <br/>
 毕比的辩护律师 GHULAM MUSTAFA CHAUDHRY：〝如果有合法的补偿措施，我们一定会申请。〞
 <br/>
 <br/>
 2009年，她在田里工作时被派去取水，一起工作的穆斯林妇女说她不是穆斯林，不能碰水碗，并指控她。2010年，毕比成为巴基斯坦第一个因〝侮辱先知罪〞被判死刑的女性，关押在死囚牢。
 <br/>
 <br/>
 毕比的遭遇引发国际社会广泛关注。
 <br/>
 <br/>
 天主教前教宗本笃十六世曾呼吁释放毕比。2015年，教宗方济各会见了毕比的家人。
 <br/>
 <br/>
 美国保守派团体——美国法律与司法中心为其筹款呼吁。
 <br/>
 <br/>
 脸书也设立了营救毕比的网页。
 <br/>
 <br/>
 在巴基斯坦，〝侮辱先知罪〞即使未经证实也可能引发暴力冲突，甚至被私刑处死。两名政客因帮助毕比被暗杀。
 <br/>
 <br/>
 周二，美国宗教自由委员会发表声明，对巴基斯坦最高法院释放毕比表示欢迎。
 <br/>
 <br/>
 不过，据该委员会2018年报告，巴基斯坦还有大约40人因亵渎罪面临死刑或无期徒刑。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
