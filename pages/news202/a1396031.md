### 泰国古城墙涂鸦 两人被捕 面临10年监禁
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】周四，两名外国游客在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泰国清迈.html" target="_blank">
  泰国清迈
 </a>
 著名旅游景点的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_古城墙.html" target="_blank">
  古城墙
 </a>
 上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_涂鸦.html" target="_blank">
  涂鸦
 </a>
 ，涉嫌破坏考古遗迹，被泰国警方逮捕。如果罪名成立，两人将面临
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_10年监禁.html" target="_blank">
  10年监禁
 </a>
 和3万多美元罚款。
 <br/>
 <br/>
 监控录像显示，10月18号凌晨4点，两名青年来到清迈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塔佩门.html" target="_blank">
  塔佩门
 </a>
 附近的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_古城墙.html" target="_blank">
  古城墙
 </a>
 外，开始
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_涂鸦.html" target="_blank">
  涂鸦
 </a>
 。随后，还站在远处欣赏。
 <br/>
 <br/>
 当时还有另外两名外国游客，站在旁边观看，但没有参与涂鸦。
 <br/>
 <br/>
 周四，泰国警方在清迈一家旅馆，逮捕了在古城墙涂鸦的两名青年。他们是23岁的英国游客Furlong Lee和加拿大游客Brittney Schneider。
 <br/>
 <br/>
 警方将他们带回到现场指认。不过，涂鸦已经被清除，可以看到清除后墙砖颜色不同。
 <br/>
 <br/>
 警方表示，他们破坏的是泰国历史古迹，将面临指控。
 <br/>
 <br/>
 泰国警官 TEERASAK  SRIPRASERT：〝这是一处历史遗迹。他们将被控破坏考古遗迹，面临10年以下监禁，一百万泰铢（约30,656美元）以下罚款。〞
 <br/>
 <br/>
 两名青年承认涂鸦事实，表示当时是喝醉了。
 <br/>
 <br/>
 泰国警官 COLONEL TEERASAK  SRIPRASERT：〝他们承认指控，说是因为激动，当时还有点喝醉了。〞
 <br/>
 <br/>
 清迈古城1296年建成，红砖砌成的城墙高两米，非常气派。目前，只留下塔佩门一处古迹，每年吸引大量游客。
 <br/>
 <br/>
 新唐人记者毕心慈、张琪综合报导
</div>
