### 西澳巨牛像〝吸铁〞 因体型太大躲过屠宰
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 西澳省一只高1.94米、重1400公斤巨牛因身形高大粗壮，站在牛群中间完全是〝鹤立鸡群〞，因无法跻进出口工厂机器，又卖不出去，幸运的躲过屠宰的命运留在农场，保得余生安稳。
 <br/>
 <br/>
 这只号称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 最大家牛是荷兰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乳牛.html" target="_blank">
  乳牛
 </a>
 历克斯（Knickers）现年7岁，因其身材粗壮，其他牛都会跟随牠的步伐。
 <br/>
 <br/>
 农场主人皮尔逊（Geoff Pearson）表示，〝历克斯的体型太大了，连那些出口工厂的铁链都完全锁不到牠。我们农场的牛只售出率很高，但尼克斯非常幸运地可以留下来〞。
 <br/>
 <br/>
 皮尔逊说，由于牠〝太大只〞，〝每次牠站起来行走，均会有数百只牛跟随，我们就知道历克斯正在移动，能简单的掌握牛群去向。〞
 <br/>
 <br/>
 据健力士世界纪录大全，世上最大的牛类身重2267公斤，而最高的家牛则有2米高，因此历克斯堪称是全澳洲最大只的牛。
 <br/>
 <br/>
 至于为何历克斯会长得如此巨大，兽医兼乳牛专家马瑟索尔（Rupert Mothersole）说，公牛如果自小被阉割但仍可好好生活的话，牠的〝生长板〞（growth plates）会保持开放，这样也有可能长得如历克斯一样巨大。
 <br/>
 <br/>
 生长板又称骺板（Epiphyseal plate），位于骨头的上下两端，可持续不断增生软骨组织，是决定身高的关键。受遗传、营养和身体健康的影响，生长板到某一阶段就会闭合而停止分裂，届时即使给予再多的生长激素都无法再增高。
 <br/>
 <br/>
