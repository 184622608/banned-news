### 【禁闻】欧亚多国助阵美国 遏止中共扩张
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】日前结束的欧亚首脑峰会发表声明，明确提出反对贸易保护，提高透明度、创造公平竞争环境。外界认为，这是直指中共，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 助阵。
 <br/>
 <br/>
 10月19号，欧盟28个成员国和亚洲21个国家，在比利时首都布鲁塞尔举行的峰会上承诺，将支持市场原则和基础设施项目的国际准则，反对保护主义。
 <br/>
 <br/>
 《华尔街日报》评论说，这是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 对抗中国经济影响力的行动，得到了众多欧洲和亚洲国家的助阵。这些自由市场国家联手对抗中共的国家资本主义，从根本上赞同必须遏制中共影响力的观点。
 <br/>
 <br/>
 而亚欧会议声明所说的〝提高透明度、创造公平竞争环境和创新融资机制〞，是暗指中共的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞，间接为反对中共海外投资项目的政策背书，因为这些项目以牺牲邻国利益为代价，来增强中共的国际影响力。
 <br/>
 <br/>
 文章称，这一切表明，欧洲和亚洲对抗中共投资做法的行动，遵循了美国的倡议，也就是在关税措施以外加大对中共的经济压力。
 <br/>
 <br/>
 旅美时政评论员蓝述：〝因为中共经济的崛起和中共这种独裁之下的经济，它把国内对付中国老百姓的这种方式通过贸易手段，出口到全世界，这是引起周边国家不安的一个因素，中共的这种对外扩张，以及这种不公平的贸易，这是全世界都看得很清楚的。〞
 <br/>
 <br/>
 不过，中共却反过来宣传，声称亚欧声明的内容指的是美国。中共官媒《中国新闻网》，批评美国推行单边贸易政策，是保护主义等等。
 <br/>
 <br/>
 蓝述：〝其实这种说法不对，美国仍然是一个尊重自由贸易的国家，川普总统需要的是公平贸易，而不是贸易保护。中国的汽车卖到美国来只打2.5%的税，美国的汽车卖到中国去要打25%的税，这就是一个不公平。〞
 <br/>
 <br/>
 旅美时政评论员蓝述指出，其实亚欧峰会强调的遵守WTO规则，履行其相关义务，致力于确保贸易自由、开放和公平，打击保护主义等等，都是在直指中共。而美国作为自由世界的领头羊，目前执行的单边贸易政策，正是在纠正中共破坏了的世界次序。
 <br/>
 <br/>
 中共国务院总理李克强，在亚欧会议上发言时也説，必须坚持自由贸易等基本原则，坚决反对一切形式的保护主义。
 <br/>
 <br/>
 不过，欧洲官员表示，他们注意到了中共的言辞和行动之间的差距。
 <br/>
 <br/>
 中华经济研究院特约研究员吴惠林向〝新唐人〞表示，其实中共才是自由贸易的真正破坏者。
 <br/>
 <br/>
 中华经济研究院特约研究员吴惠林：〝WTO本来就是多边的，让很多国家坐在一起，就是要把关税去掉，变成真正的自由贸易，中共加入以后，表面上签订一些协定，答应什么时候开放一种什么样的东西，它其实是把一些东西破坏掉，所以自从中共加入以后，它是破坏自由贸易。〞
 <br/>
 <br/>
 惠林认为，中共的做法会把中国人民乃至世界人民带入深重的苦难之中。
 <br/>
 <br/>
 一名欧盟高级官员称，一些国家意识到了中共慷慨提供的资金有时可能是有毒的，希望越来越多的国家加入进来，迫使中共改变其行事方式。
 <br/>
 <br/>
 面对当前中共〝一带一路〞给亚洲国家留下的一个个债务陷阱，欧盟承诺，为那些同意采用西方招标和财务模式的亚洲国家提供资金。这是继最近美国倡议建立一个公正、透明的替代方案，应对中共〝一带一路〞债务陷阱后，又一个针对中共海外投资的重大动作。
 <br/>
 <br/>
 采访编辑/刘惠 后制/周天
