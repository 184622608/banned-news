### 粮食缺口跨红线 中共贸战〝子弹〞能打多远？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 在9月24日加码后，美国遏制中共的力度陡然上升，而两国都声称应付贸易战的〝子弹还很多〞。有美国媒体分析，中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_粮食.html" target="_blank">
  粮食
 </a>
 缺口早已跨越红线，难以真正硬气的和资源丰富的美国长期抗争。
 <br/>
 <br/>
 上个月底，中国国家主席习近平视察〝中国粮仓〞东北，强调中国实体经济、制造业和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_粮食.html" target="_blank">
  粮食
 </a>
 等要自力更生，都要靠自己。当时就有评论指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 使中共在国际社会陷入空前孤立，中国正面临潜在的粮食不足危机，这也正是习近平视察东北粮仓的用意——〝备荒〞。
 <br/>
 <br/>
 资料显示，中国的耕地面积从1991年开始便逐步减少，年均减少433万亩。2016年下降至1.3亿公顷，人均耕地面积更是减少到1.4亩，只有美国的1/7、俄罗斯的1/8。
 <br/>
 <br/>
 另外，中共推动的城镇化进程，不仅夺去了粮食生产的土地和水资源，还抢走了最重要的资源──农业人口。2004—2015年，中国外出农民工数量呈直线上升，10年来增长48%，导致务农人口剧减。
 <br/>
 <br/>
 《世界日报》的报导称，由于工业化厂房和住屋需求，中国有大量的良田被建成房屋，以拉高地方GDP增长率。如第一大产粮省河南，每年就有平均20万亩农地变成建地。中国可耕地已不到15亿亩，其中三分之一的耕地还有酸雨污染。而养活14亿国人的防线是耕地最少须维持18亿亩。
 <br/>
 <br/>
 有专家曾预测，2020年中国粮食需求约7亿吨，但自产只有5.54亿吨，近2亿吨须靠进口，使中国粮食安全面临威胁，更难言短时间内〝靠自己〞。
 <br/>
 <br/>
 2013年迄今，中国每年进口粮食都逾2200万吨，粮食自给率已跌至86%。报导说，这一比率已经跨越警戒线，但中共官方对此却予以否认。专家的观点认为，粮食自给率短缺10%，社会可能不安；如有30%不足，就可能引发动乱。
 <br/>
 <br/>
 报导称，中共声称贸易战〝子弹还很多〞，即使其如对波音飞机、苹果手机等下手，使美国遭受重大损失，但美国如对中国粮食、能源等进口勒住咽喉，中共会〝致命〞。军备再先进、武器再强大，粮食不足就难以和外国抗衡，或打经年累月的持久战，这是国家安全问题。
 <br/>
 <br/>
 报导认为，如加上能源、主要矿产大部分靠进口，对外技术高度依存，〝厉害了，我的国〞想和美国叫板〝谁的子弹多〞，只能是鼓舞士气的做法，〝如真的放到现实中检验，人民的肚子也掌握在外人手中，如何真正硬气和资源丰富的超强美国抗争？〞
 <br/>
 <br/>
 报导最后指出，中共施政只重视GDP数字漂亮，如今粮食安全却已隐隐亮起红灯，粮食风险积重难返，已无法靠转基因作物等弥补。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：戴明）
