### 日本爱知5.1地震未有影响 海地5.9地震疑建筑物倒塌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱知.html" target="_blank">
  爱知
 </a>
 县东部7日上午发生规模5.1
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 ，深度约40公里，所幸没有引发海啸威胁，也未对陆上交通及核电厂等造成任何影响。稍早，位于加勒比海岛国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海地.html" target="_blank">
  海地
 </a>
 也发生规模5.9地震，深度仅11.7公里，传有墙壁倒塌，就连邻近的多明尼加也有部分地区感受到地震威力。
 <br/>
 <br/>
 放送协会（NHK）报导，这起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 发生在日本时间上午10时14分，震央位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱知.html" target="_blank">
  爱知
 </a>
 县东部，长野县根羽村等地，被观测到震度4有感地震。目前尚未接获相关灾情通报。
 <br/>
 <br/>
 日本道路交通资讯中心表示，包括中央自动车道及东名高速道路等高速道路，并未受到这起地震影响。
 <br/>
 <br/>
 中部电力公司表示，位于静冈县御前崎市的滨冈核电厂因停止运转，未出现任何异常；周边设置的辐射线监测仪器，也未发现数值有任何变化。
 <br/>
 <br/>
 另外，位于加勒比海岛国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海地.html" target="_blank">
  海地
 </a>
 当地时间7日零时11分（北京时间7日早上8时11分），发生规模5.9地震，震央位于和平港西北西方外海，深度仅11.7公里，连邻近的多明尼加也有部分地区感受到地震摇晃威力。
 <br/>
 <br/>
 从民众发到推特上的照片，可见到建筑物墙壁倒塌，疑似有人被压在墙壁砖头下面。当局尚未传出伤亡与损害等灾情报告。
