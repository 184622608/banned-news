### 海地5.9浅层地震 建物倒塌已知11死逾百人伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】位于加勒比海岛国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海地.html" target="_blank">
  海地
 </a>
 7日凌晨(当地时间)发生规模5.9
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 ，深度仅11.7公里，有多栋建筑物墙壁倒塌，目前已知11人死亡，超过百人受伤。另外日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱知.html" target="_blank">
  爱知
 </a>
 县东部上午也发生规模5.1地震，所幸未引发海啸威胁。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海地.html" target="_blank">
  海地
 </a>
 7日零时11分（北京时间7日早上8时11分）发生规模5.9
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 ，震央位于和平港西北西方外海，深度仅11.7公里，当地约有25万人居住。首都太子港及邻近的多明尼加的部分地区都感受到地震威力。
 <br/>
 <br/>
 从民众发到推特上的照片，可见到建筑物墙壁倒塌，疑似有人被压在墙壁砖头下面。
 <br/>
 <br/>
 政府发言人稍后表示，地震暂时造成11人死亡，多座建筑物受损。路透社则报导，海地西北地区警察首长希莱尔（Jackson Hilaire）表示，和平港至少7人死亡、超过100人受伤。
 <br/>
 <br/>
 另在和平港以南的城镇格罗斯莫恩（Gros-Morne）行政首长泰德（Jean Renel Tide）表示，有4人丧生，包括一名男童遭倒塌建筑压死。
 <br/>
 <br/>
 海地曾在2010年曾发生大地震，造成达20万人死亡，这次地震唤起人们可怕的回忆。海地总统莫伊兹透过社交网站呼吁居民保持冷静，称各地政府会为灾民提供协助。民防部门指有部分伤者送往医院治疗。
 <br/>

 日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱知.html" target="_blank">
  爱知
 </a>
 县东部上午也发生规模5.1地震，深度约40公里，所幸没有引发海啸威胁，也未对陆上交通及核电厂等造成任何影响。
 <br/>
 <br/>
 放送协会（NHK）报导，这起地震发生在日本时间上午10时14分，震央位于爱知县东部，长野县根羽村等地，被观测到震度4有感地震。目前尚未接获相关灾情通报。
 <br/>
 <br/>
 日本道路交通资讯中心表示，包括中央自动车道及东名高速道路等高速道路，并未受到这起地震影响。
 <br/>
 <br/>
 中部电力公司表示，位于静冈县御前崎市的滨冈核电厂因停止运转，未出现任何异常；周边设置的辐射线监测仪器，也未发现数值有任何变化。
