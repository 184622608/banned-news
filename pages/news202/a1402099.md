### 俄与沙特计划石油减产 川普促欧佩克保持供应维护低价
------------------------

 【新唐人北京时间2018年12月06日讯】周三（12月5日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统敦促
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_石油.html" target="_blank">
  石油
 </a>
 输出国组织（OPEC，也称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧佩克.html" target="_blank">
  欧佩克
 </a>
 ）继续增加石油供应以保持国际油价低廉，反对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 阿拉伯正考虑减产的计划。
 <br/>
 <br/>
 周三上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在推特上呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧佩克.html" target="_blank">
  欧佩克
 </a>
 不要减产，以使当前的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_石油.html" target="_blank">
  石油
 </a>
 价格得以维持。
 <br/>
 <br/>

 <br/>
 据《华盛顿观察家报》（The Washington Examiner）12月5日报导，川普之所以敦促欧佩克保持石油供应，是因为欧佩克和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 正计划在周四及周五的维也纳会议上讨论是否削减石油产量。
 <br/>
 <br/>
 俄罗斯总统普京（Vladimir Putin）上周末表示，他与欧佩克最大的石油生产国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 阿拉伯已达成协议，计划在今年为满足川普的要求而暂时提高石油产量后，将于明年再次续签关于削减石油产量的协议，以便提高石油价格。
 <br/>
 <br/>
 近几周来，由于沙特和俄罗斯开始提高石油产量，因而国际石油价格已经下降。
 <br/>
 <br/>
 沙特和俄罗斯的增产行动部分是为了回应川普的呼吁。川普担心石油价格在美国中期选举前上涨，也担心石油问题会破坏美国对伊朗的重新制裁。
 <br/>
 <br/>
 在近期提高石油产量之前，欧佩克和俄罗斯已经执行一项减产协议长达18个月之久。
 <br/>
 <br/>
 目前石油市场处于供过于求的局面，因为沙特和俄罗斯为抵消伊朗减产带来的损失而反应过重，而实际情况并不如所担心的那么严重。
 <br/>
 <br/>
 川普上个月称赞沙特为降低油价做出了贡献，但同时鼓励沙特继续增产，这也是他不断重复提出的要求。
 <br/>
 <br/>
 （记者李佳欣编译报导/责任编辑：东野）
