### 博尔顿严厉警告：海军再遇挑衅将按交战规则处置
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】上周，白宫国家安全顾问博尔顿（John Bolton）直截了当地警告
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 ，美军不会忽视中共军舰在国际水域逼近美国军舰的挑衅性行动。
 <br/>
 <br/>
 博尔顿是川普政府中的鹰派，主张强硬的对华政策。除了警告
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 军方的军事冒险外，他也再次批评了中共干扰美国中期选举并窃取美国技术的行为。
 <br/>
 <br/>
 关于中共军舰在中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海.html" target="_blank">
  南海
 </a>
 挑衅美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海军.html" target="_blank">
  海军
 </a>
 〝迪凯特号〞驱逐舰（USS Decatur）的事件，博尔顿表示，这种挑衅行为是不可接受的 。
 <br/>
 <br/>
 9月30日，〝迪凯特号〞正在南海的斯普拉特利群岛（Spratlys，中方称南沙群岛——译注）附近海域航行时，中共军舰〝兰州舰〞逼至〝迪凯特号〞前方45码（约41米）处，美舰采取紧急措施，才得以避免碰撞。
 <br/>
 <br/>
 博尔顿警告中共不要再冒险，因为按照美国海军的交战规定，指挥官有权保护自己的战舰。
 <br/>
 <br/>
 博尔顿是在10月11日接受美国电台脱口秀主持人休伊特（Hugh Hewitt）的采访时做出上述表述的。
 <br/>
 <br/>
 博尔顿明确表示，美方不会容忍中共威胁美国军人。中共方面需要明白，美国决心让国际海上通道畅通无阻。
 <br/>
 <br/>
 中共一直在中国南海的有争议岛屿上修建前哨军事基地。自2013年以来，中共在该地区所开发的面积已超过3200英亩（约合12.9平方公里），并开始在其中一些岛屿上部署先进的导弹。
 <br/>
 <br/>
 中共声称，整个南海海域的主权都归它所有，而该水域每年的航运贸易额估计达5万亿美元。
 <br/>
 <br/>
 美国一贯主张南海地区属国际水域，联合国常设仲裁法院（PCA）也驳回了中共对该地区的主权要求。
 <br/>
 <br/>
 博尔顿表示，美国不承认中共主张的合法性。
 <br/>
 <br/>
 他进一步指出，中共军舰几乎撞上美舰，这是中共危险行为的一个例证。现在，已经有更多盟友加入反对中共的行动中去，除了菲律宾和日本等邻近国家外，英国、澳大利亚等国也同美国一样，继续穿越南海水域在公海航行，而且，这些国家都准备在这方面做得更多。
 <br/>
 <br/>
 博尔顿还表示，这些盟国可能还会开采南海的海底能源，或与中共合作，或者根本不跟它合作。
 <br/>
 <br/>
 他指出，中共日前的挑衅行为是奥巴马政府对以往的美中对抗忽视或错误处理的结果，而川普总统却在以一种让中共感到困惑的方式回应中共的威胁和对抗。中共以前从未见过哪位美国对其如此强硬。
 <br/>
 <br/>
 博尔顿还批评了中共盗取美国知识产权以及干扰美国中期选举的行为。
 <br/>
 <br/>
 博尔顿评论说，他认为中共的行为需要在整个一系列问题上进行调整，包括贸易领域、国际事务，以及军事和政治领域。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
