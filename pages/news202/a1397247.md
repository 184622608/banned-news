### 中美洲人非法越境不止 美派5200兵守边
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】近日，大量
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲难民.html" target="_blank">
  中美洲难民
 </a>
 非法越境
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 试图前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 。而周日，又有一批中美洲难民试图越过墨西哥南部边境，遭到墨西哥当局阻拦。
 <br/>
 <br/>
 据报导，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 特昆乌曼和危地马拉的接壤地段，约千人的一批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲难民.html" target="_blank">
  中美洲难民
 </a>
 突破了边境门，同防卫警察发生冲突，他们向工作人员和联邦警察投掷石块，鞭炮，玻璃瓶。导致六名警察受伤，一名洪都拉斯难民死亡。
 <br/>
 <br/>
 墨西哥内部部长Alfonso Navarrete：〝墨西哥政府抵制发生在危地马拉边界的暴力事件。我们重申，进入墨西哥的唯一途径是遵守移民法。〞
 <br/>
 <br/>
 而据最新消息，这批难民中已有几百人渡河进入墨西哥。
 <br/>
 <br/>
 此前，已有一批中美洲移民入境墨西哥，有前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 的可能。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统推文指出，许多帮派成员和一些非常坏的人混入大篷车前往美国南部边境。他声明，如果不通过法律程序，就不许进入美国。
 <br/>
 <br/>
 美国国土安全部誓言阻止这批大篷车移民非法进入美国，此前，已有约2,000名国民警卫队前往边境。
 <br/>
 <br/>
 而五角大楼也确认，本周增派5200名士兵到美墨边境进行防卫。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
