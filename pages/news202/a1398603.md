### 欧盟最大党推出欧盟执委会主席候选人
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】周四，德国总理默克尔的盟友
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韦伯.html" target="_blank">
  韦伯
 </a>
 ，赢得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧洲人民党.html" target="_blank">
  欧洲人民党
 </a>
 的党内初选，明年将竞选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 执委会主席。
 <br/>
 <br/>
 周四在芬兰赫尔辛基举办的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧洲人民党.html" target="_blank">
  欧洲人民党
 </a>
 代表大会上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韦伯.html" target="_blank">
  韦伯
 </a>
 赢得了79%的选票，击败前芬兰总理斯塔布。
 <br/>
 <br/>
 EEP党竞选主任 Dara Murphy：〝欧洲人民党的2019年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 执委会主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_候选人.html" target="_blank">
  候选人
 </a>
 获得了492票，他是韦伯。〞
 <br/>
 <br/>
 明年5月，韦伯将代表该党竞选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟执委会主席.html" target="_blank">
  欧盟执委会主席
 </a>
 。
 <br/>
 <br/>
 欧洲人民党是欧盟最大的跨国中间偏右政党，联合了基督教民主党和保守党。
 <br/>
 <br/>
 选举后，韦伯的竞选对手斯塔布上台向他表示祝贺，两人联手向台下致意。
 <br/>
 <br/>
 在胜选讲话中韦伯表示，这不是他个人的胜利，而是作为欧盟领导力量的欧洲人民党的胜利。
 <br/>
 <br/>
 EEP党欧盟执委会主席候选人 韦伯：〝虽然我们刚进行了选战，但我们首先感受到的是欧洲人民党内部的团结，因为我们对这个大陆的未来有一个清晰的看法。〞
 <br/>
 <br/>
 韦伯来自德国巴伐利亚，属于和默克尔联合执政的基督教社会联盟党，他支持默克尔的难民政策，同时主张打击非法入境和走私，保卫欧盟外部边境。
 <br/>
 <br/>
 如果明年5月韦伯胜选，他将成为欧盟最具有影响力的人之一。
 <br/>
 <br/>
 新唐人记者王羽音、张琪综合报导
</div>
