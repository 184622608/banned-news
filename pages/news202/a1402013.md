### 港媒：白宫声明暗藏两大玄机 中美谈判风浪在后面
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 落幕后，美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 暂缓进行。双方宣布的会谈成果说法不一，引发外界对这场博弈中谁输谁赢的讨论。有港媒认为，白宫的声明虽然是一番溢美之词，但是却暗藏两大玄机，有强烈施压中共的意味，双方在后续谈判中势必波涛汹涌。
 <br/>
 <br/>
 据此前媒体报导，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞并没有出现剑拔弩张的气氛，晚宴还在掌声中结束。川普与习近平的会谈后，随即登上空军一号专机回国。飞机起飞后没多久，白宫就这次元首会晤发表声明。
 <br/>
 <br/>
 香港《经济日报》分析认为，白宫的声明初看没有半点火药味，其中更形容这是一次〝高度成功的会谈〞，又对习近平作出一番溢美之词，称〝习主席做出了不起的人道主义姿态，同意把芬太尼列为管控物质〞。不过，声明的细节中却暗藏两大玄机，有强烈施压中共的意味：
 <br/>
 <br/>
 一是川普和习近平同意立即开始有关结构性改变的谈判。这些结构性改变涉及强制技术转让、知识产权保护、非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 壁垒、网路入侵和网路盗窃、服务业和农业。
 <br/>
 <br/>
 二是双方同意，努力在未来90天内完成谈判。如果双方在这个期限内未能达成协议，美国对中国加征关税的税率将从10%提高至25%。
 <br/>
 <br/>
 分析指，中共一直将〝经济结构改革〞视为不能动摇的底线，其中包括技术转让，乃至产业补贴、特别是对高新科技的扶助政策。而川普明言要透过谈判，改变中共的这些行为，实际上美国是提高要价，要北京作出更多让步才能实质停止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 。
 <br/>
 <br/>
 另外，白宫在声明中明确提出了3个月的〝最后通谍〞。美国财政部长努钦（Steven Mnuchin）对CNBC表示，北京必须信守诺言，做出实实在在的进展，因为川普政府不会再忍受任何的拖延。
 <br/>
 <br/>
 外界注意到，中共官方在自己的通报中，只字未提结构改革和3个月的谈判期限。报导认为，这有可能意味着北京并不认同美国政府设定的谈判议程和时限，这也预示着未来的中美谈判势必波涛汹涌、难以平静。
 <br/>
 <br/>
 报导最后称，就算中共愿意接受美方的议程，也很难想像两国能在3个月内谈妥结构性改革。如果最后以谈崩收场，两国的关系将滑向更坏的境地。
 <br/>
 <br/>
 英国《金融时报》也认为，由于习近平几乎不可能同意或实施对中国经济的如此广泛的改革，到2019年2月底，贸易战几乎肯定会以更加凶猛的方式重新开打。但川普展现强势和理性来暂停贸易战，给中共设定严苛的条件，实际上除了将推动贸易战全面爆发的势头稍稍收敛之外，并没有做出任何让步。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：戴明）
 <br/>
