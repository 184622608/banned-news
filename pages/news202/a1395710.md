### 摩洛哥火车冲出轨撞桥墩 至少6死86伤　
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】北非国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摩洛哥.html" target="_blank">
  摩洛哥
 </a>
 一辆客运列车周二（16日）行经首都拉巴特附近城镇时意外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 撞到桥墩上，车厢扭曲变形，已知至少6死86伤。
 <br/>
 <br/>
 法国《巴黎人报》（Le Parisien）报导，事发于当地时间周二中午，一辆客运列车在拉巴特（Rabat）以北20公里的城镇布格纳代勒（Sidi Bouknadel）镇意外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 撞到桥墩上。
 <br/>
 <br/>
 从现场照片中可见，现场一片狼藉，铁道上有多节列车出轨，部分车厢撞击桥墩后翻覆。当地官员告诉美联社，有部分乘客受困于扭曲变形的车厢内，救难人员正在抢救中。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摩洛哥.html" target="_blank">
  摩洛哥
 </a>
 国营铁路公司（ONCF）表示，目前已经派人到现场，事发原因仍在调查中。
