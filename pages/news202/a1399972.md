### 美日印澳联手制衡中共  维持军事优势
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 周日(11月18日)结束亚太之旅。此行，美国、日本、印度与澳大利亚通过四边安全对话，讨论了强化安全领域合作，联手制衡
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 。专家也指出，美军对中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事.html" target="_blank">
  军事
 </a>
 优势，至少能够维持20年。
 <br/>
 <br/>
 美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 18日晚返抵华府，结束为期一周的亚太之行。本次访问中，他先后与日本、印度和澳大利亚领导人举行会谈。此外，在东盟和东亚峰会期间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美日印澳.html" target="_blank">
  美日印澳
 </a>
 四国官员还在场边举行了四边安全对话，讨论如何制衡
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在区域内的影响力。
 <br/>
 <br/>
 哈德森研究所美国海上力量中心副主任 Bryan McGrath：〝这四个国家有共同利益，要保持开放的海上通道，让印度洋与南海、东海的交通通畅。因为这些国家具有共同利益，他们的合作很自然，很高兴他们也正在这样做。〞
 <br/>
 <br/>
 亚太经合会议上，彭斯宣布，美国将与澳大利亚和巴布亚新几内亚合作，重新启动具有战略意义的朗布伦海军基地扩建，以抑制中共扩大区域战略影响力的野心。
 <br/>
 <br/>
 专家认为，相对于中共，美国及其盟友在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事.html" target="_blank">
  军事
 </a>
 实力上依然拥有强大优势。
 <br/>
 <br/>
 日本海上自卫队前舰队司令官香田洋二：〝我依然对美国的军事能力有很强信心，在日本和澳大利亚盟友的支持下。我认为美国军力在过去任何时候都胜于中共，至少在未来20年中还将如此。〞
 <br/>
 <br/>
 在敏感的南海问题上，彭斯表示，南中国还不属于任何国家。美国将继续在国际法许可范围内和国家利益要求下，在任何地方进行飞航。
 <br/>
 <br/>
 日本海上自卫队前舰队司令官香田洋二：〝黄岩岛的人工岛礁建设，由于美军的存在，目前已经停止。当然我们还要做很多努力，来保持美国的区域存在。在此条件下，才能保持现状。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
