### 【禁闻】数百学者联署 谴责中共新疆政策

  </a>
 </b>
 <br/>
 <br/>
 美国政府11月26号在哥伦比亚特区联邦地区法院提起诉讼，要求没收三家被控为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 银行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗钱.html" target="_blank">
  洗钱
 </a>
 的公司的3百多万美元。
 <br/>
 <br/>
 哥伦比亚特区联邦检察官11月27号发表声明说，这三家公司是一家同意与美国政府合作的新加坡公司、和总部在中国的蜡品公司Apex Choice Ltd、以及总部在中国的木业公司Yuanye Wood Ltd。
 <br/>
 <br/>
 诉状指称，这三家公司为被制裁的朝鲜银行洗钱，朝鲜银行则利用这些被洗过的资金为朝鲜当局购买物资，并非法进入美国金融市场。
 <br/>
 <br/>
 <b>
  数百学者联署谴责中共
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
   新疆
  </a>
  政策
 </b>
 <br/>
 <br/>
 中共在新疆建立大规模再教育营关押上百万无辜民众，事件继续国际社会受到关注。11月26号，来自美、英、法和日本等国的学者，在美国华盛顿国家新闻俱乐部举行新闻发布会，谴责中共的新疆政策。
 <br/>
 <br/>
 一份有278名学者签名的声明也在会上发布。
 <br/>
 <br/>
 学者们向各国政府和相关机构提出七点建议，呼吁他们发布官方声明，要求中共当局立即废止〝教育转化〞监禁系统，释放所有被拘禁者，并对相关中共官员和科技公司实行经济制裁等。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709.html" target="_blank">
   709
  </a>
  家属
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
   李文足
  </a>
  获
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瑞典人权奖.html" target="_blank">
   瑞典人权奖
  </a>
 </b>
 <br/>
 <br/>
 〝709案〞被抓律师王全璋的妻子李文足，日前获得2018年瑞典〝爱德尔斯塔姆人权奖〞（The Edelstam Prize），评审委员会表示，奖项是为了表扬李文足捍卫人权的贡献和勇气。
 <br/>
 <br/>
 由于李文足遭中共限制出境，不能出席11月27号在瑞典斯德哥尔摩举行的颁奖仪式，她的奖项由流亡美国的人权活动人士陈光诚的妻子袁伟静代领。
 <br/>
 <br/>
 评委会主席卡罗琳．爱德尔斯塔姆表示，中国至今仍是一个一党专制的极权国家，中共针对基本人权的言论、结社、集会等进行系统性打压，甚至实施强制失踪。李文足和其他709案的妻子们，不但致力于营救她们的丈夫，并且利用网络声援、探监、上访检察机关和法院等非传统方式，反击中共当局的镇压，展现出了非凡的勇气。
 <br/>
 <br/>
 李文足的丈夫王全璋，曾为法轮功等弱势群体辩护，他在2015年〝709大抓捕事件〞中被当局带走，一直关押至今。
 <br/>
 <br/>
