### 云南白水河古冰川大量消融 气候影响堪忧
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】每年有数百万人到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_云南.html" target="_blank">
  云南
 </a>
 省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_玉龙雪山.html" target="_blank">
  玉龙雪山
 </a>
 的中国白水一号冰川欣赏冰川之美，但近年来那里冰川消融的速度令人担心。
 <br/>
 <br/>
 号称世界屋脊的青藏高原又被称为南北极之外的〝第三极〞。第三极冰川的季节性融化影响着亚洲十大河流包括长江、黄河、湄公河和恒河，对中国、越南到阿富汗的亿众人口来说至关重要。
 <br/>
 <br/>
 而第三极冰川中的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白水河.html" target="_blank">
  白水河
 </a>
 冰川是世界上融化最快的冰川之一。
 <br/>
 <br/>
 多年来，科学家们一直在观察白水1号冰川的部分地段：中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_云南.html" target="_blank">
  云南
 </a>
 省的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_玉龙雪山.html" target="_blank">
  玉龙雪山
 </a>
 冰川。在过去十年中冰川每年消退约27米多。那里的植物，如雪莲的根部已经从泥土中暴露出来。
 <br/>
 <br/>
 冰川学家、玉龙雪山冰川与环境观测研究站主任王世锦对冰川消融的速度表示担忧。
 <br/>
 <br/>
 冰川学家王世锦：〝每年冰川后退的幅度大约在20米左右，冰面下降一到两米。〞
 <br/>
 <br/>
 这些照片有助于说明玉龙雪山的消融。
 <br/>
 <br/>
 根据2018年〝地球物理研究杂志〞的报告，自1982年以来，冰川已经减少了60％的质量并缩小了250米。
 <br/>
 <br/>
 根据玉龙雪山公园官员的说法，去年有260万游客参观了这座山。
 <br/>
 <br/>
 科学家在2015年发现，在中国调查的冰川中82％已经消融。
 <br/>
 <br/>
 他们警告说，〝冰川融化对水资源的影响正逐渐变得越来越严重〞。
 <br/>
 <br/>
 为了保护冰川，当局已将游客数量限制在每天10,000人，并禁止在冰上徒步旅行。
 <br/>
 <br/>
 新唐人记者严枫、天羽综合报导
