### 美对华政策大调整 彭斯将全方位批评中共
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）将于当地时间4日发表白宫的〝中国（共）政策〞。根据彭斯办公室公开的演说摘录，彭斯将从政治、外交、经贸及军事多方面批评中共的全球政策，并对北京示警。
 <br/>
 <br/>
 预计美东时间上午10月4日11点多，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 将在华盛顿的智库哈德逊研究所，发表白宫对北京的政策。
 <br/>
 <br/>
 据演说摘录显示，在经济层面，彭斯将聚焦中美贸易战及中共的〝一带一路〞战略计划。他将不点名提到一间美国主要企业，近期被中共威胁，要其公开反对川普（特朗普）的贸易政策，否则拒绝向该企业发出商业许可证。
 <br/>
 <br/>
 彭斯还批评中共以〝债务外交〞扩大全球影响力，〝如今这个国家正向亚洲、非洲、欧洲及拉丁美洲的国家，借出数千亿美元的基建资金。然而贷款条件并不透明，且收益多归北京。〞
 <br/>
 <br/>
 此外，彭斯还将特别提到中共向委内瑞拉借出50亿美元，以〝延长贪腐及无能的马杜罗政权寿命〞。
 <br/>
 <br/>
 在军事层面，彭斯将对美国狄卡特号（USS Decatur）驱逐舰9月30日在南海自由航行时，被中共一艘旅洋级驱逐舰挑衅式靠近，两舰之间发生〝不安全互动〞差点相撞事件，发表演说。
 <br/>
 <br/>
 彭斯会指出：〝尽管面对这种鲁莽的骚扰，美国海军将持续在国际法允许及符合国家利益要求的各处飞行、航行及操演。我们不会被吓倒，更不会退缩。〞
 <br/>
 <br/>
 在政治层面，彭斯指美国情报部门已确定州及地方的政府与官员，成为中共的目标，中共企图在政策上导致联邦与地方出现分裂。
 <br/>
 <br/>
 此外，彭斯将指控中共利用挑衅的方式，干预美国国内政策及政治，更直言〝中国（共）想要另一位美国总统〞。
 <br/>
 <br/>
 稍早前，川普在主持联合国安理会会议时，指责中共试图干预今年11月举行的美国中期选举。并指美国已掌握大量证据。
 <br/>
 <br/>
 同时，彭斯还会指控中共迫使3个拉丁美洲国家与台湾断交，声称这危及台海局势稳定：〝我们承认一中政策的同时，台湾拥抱民主证明所有中国人有更好的道路。〞
 <br/>
 <br/>
 此前，针对彭斯将发表的对中共政策的演说，川普政府资深官员举办了背景简报会。美国之音报导，资深官员说，中共正在从政治、经济、军事以及宣传方面，往外扩张实力。
 <br/>
 <br/>
 资深官员还指出，中共压制美国大学的学术自由。川普总统曾表示，所有的中国留学生都是间谍。美国国会也曾经针对孔子学院对美国大学学术自由的影响，举办了听证会。
 <br/>
 <br/>
 一直以来，几家在美国的中共官方媒体多年来都假装是新闻媒体，但现在已经规定他们必须登记为外国政府代理人。
 <br/>
 <br/>
 此外，中共还以签证来威胁美国学者跟记者。许多美国媒体记者会因为他们的报导，被拒发签证，或拒绝更新他们的签证。
 <br/>
 <br/>
 资深官员指出，就连好莱坞也对中共〝磕头〞，因为当他们在拍摄要行销到全世界的电影时，也必须考虑到中共的审查制度。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉）
