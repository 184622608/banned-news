### 孟晚舟受审或控欺诈 面临最高30年刑期
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】观众朋友大家好，欢迎收看新唐人全球新闻。12月1号
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被捕.html" target="_blank">
  被捕
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 财务长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 ，周五在加拿大温哥华的卑诗省最高法院，列席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保释.html" target="_blank">
  保释
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_聆讯.html" target="_blank">
  聆讯
 </a>
 。据在场记者透露，她可能被指控欺诈罪，面临最高30年刑期。
 <br/>
 <br/>
 现年47岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 是中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司创办人任正非的女儿，也是该公司的财务长。
 <br/>
 <br/>
 有出席孟晚舟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保释.html" target="_blank">
  保释
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_聆讯.html" target="_blank">
  聆讯
 </a>
 的记者透露，孟晚舟可能会因欺诈罪被判最多30年刑期，并需缴纳2000万美元罚金。
 <br/>
 <br/>
 不过，加拿大司法部目前拒绝给予孟晚舟保释，理由是她有逃离加拿大的动机。但孟的律师表示，孟晚舟不会违反法庭命令而让她的父亲感到难堪。
 <br/>
 <br/>
 截至发稿，法院还没有做出相关裁决。不过，法官解除了限制媒体就法庭上出示的证据或文件进行报导的禁令。
 <br/>
 <br/>
 加拿大是在美国的要求下逮捕孟晚舟。美国指控孟晚舟利用一家华为子公司SkyCom，在伊朗做生意，误导美国金融机构，违反美国对伊朗的制裁。白宫经济顾问库德洛周五在CNBC表示，此前美国已给华为多次警告。
 <br/>
 <br/>
 孟晚舟还可能被引渡到美国，引渡过程可能要数周或数月。
 <br/>
 <br/>
 连线温哥华新唐人记者孟雪之：〝周五上午10点，孟晚舟身着绿色上衣，在等候开庭前，一直与代表律师交谈，神情略显轻松。法庭内，旁听聆讯的人潮拥挤，警方派出警力维持现场秩序。为了满足民众需要，法院内走廊上六台电视机也对庭审进行实时转播。聆讯期间，多家媒体一直在外面等待消息。〞
 <br/>
 <br/>
 新唐人记者孟雪之、李子玉连线报导
