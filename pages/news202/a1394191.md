### 奥委会确定2026冬奥申办城市
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥委会.html" target="_blank">
  奥委会
 </a>
 执委会周四表示，已经确定加拿大的卡尔加里、瑞典首都斯德哥尔摩、意大利的米兰/科尔蒂纳丹佩佐成为申办2026年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冬奥会.html" target="_blank">
  冬奥会
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_候选城市.html" target="_blank">
  候选城市
 </a>
 。
 <br/>
 <br/>
 周四，国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥委会.html" target="_blank">
  奥委会
 </a>
 副主席小萨马兰奇在阿根廷布宜诺斯艾利斯表示，国际奥委会执委会是在听取了2026
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冬奥会.html" target="_blank">
  冬奥会
 </a>
 工作小组的汇报后做出上述决定的。加拿大的卡尔加里、瑞典的斯德哥尔摩、意大利的米兰/科尔蒂纳丹佩佐这三个城市的申办条件较好，已经具备了举办一届精彩冬奥会的基础。而另一个申办城市、土耳其的埃尔祖鲁姆由于基础设施比较薄弱，因而未能获得候选资格。
 <br/>
 <br/>
 小萨马兰奇尤其对意大利的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_申办方.html" target="_blank">
  申办方
 </a>
 案赞赏有加。
 <br/>
 <br/>
 国际奥委会副主席小萨马兰奇：〝（意大利）提交了一个非常好的方案，符合我们提出的2020新理念。〞
 <br/>
 <br/>
 最初有7个国家都有意申办2026冬奥会，但是日本，瑞士，奥地利都因为举办冬奥会的成本过于高昂而先后退出。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
