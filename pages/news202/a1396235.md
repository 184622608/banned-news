### 沙漠国家淹大水 卡达1天降下整年雨量
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】中东国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡达.html" target="_blank">
  卡达
 </a>
 首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜哈.html" target="_blank">
  杜哈
 </a>
 20日降下暴雨，一天的雨量竟将一整年份的雨量给下完了，造成大范围的淹水，房子、车子与商店都泡在水中，学校因此停课，航班被打乱。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡达.html" target="_blank">
  卡达
 </a>
 为热带
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙漠.html" target="_blank">
  沙漠
 </a>
 性气候，但20日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜哈.html" target="_blank">
  杜哈
 </a>
 （Doha）郊区阿布哈穆尔（Abu Hamour）降下的84毫米雨量，竟破了该区10月份平均。
 <br/>
 <br/>
 《半岛电视台》（Al-Jazeera）的气象学家施特夫．高堤耶（Steff Gaulter）在推特上表示，从英国民航局资料显示，杜哈的平均年雨量为77毫米。
 <br/>
 <br/>
 由于大雨造成卡达航空只好改变数个航班降落机场，但卡达与沙特阿拉伯、阿拉伯联合大公国处于断交状态，无法使用这两国的领空，因此只能从科威特与伊朗绕远路，并对外公布，未来一阵子班机起降可能会误点。
 <br/>
 <br/>
 不过，淹水虽然造成不便，但有些卡达居民从未见过淹水，倒是感到很新奇，对各种灾情一笑置之。

 卡达位于阿拉伯湾西海岸的中部，由沙特阿拉伯向北延伸的一个半岛，周围有几个岛屿，大部分地区为覆盖沙土的荒漠。从南到北全长160公里，自东向西宽80公里，包括诸岛在内总面积11532.5平方公里。三面临海，在西北部与巴林隔海相望，相距仅不到30公里。
