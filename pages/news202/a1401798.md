### 卡塔尔宣布退出OPEC 国际油价应声上涨
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】石油输出国家组织（OPEC）将在本周召开会议之际，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡塔尔.html" target="_blank">
  卡塔尔
 </a>
 今天（12月3日）突然宣布，明年1月1日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退出OPEC.html" target="_blank">
  退出OPEC
 </a>
 。消息传出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际油价.html" target="_blank">
  国际油价
 </a>
 应声上涨。
 <br/>
 <br/>
 OPEC成员国及其盟国，包括俄罗斯，预定本周四（6日）在维也纳举行会议，商讨减产问题，以求稳定10月以来已下滑近30%的原油价格。不料在会议召开前三天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡塔尔.html" target="_blank">
  卡塔尔
 </a>
 突然宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退出OPEC.html" target="_blank">
  退出OPEC
 </a>
 。
 <br/>
 <br/>
 受此消息影响，布伦特原油期货周一上涨4.12%，报61.91美元；而西德州中级原油（WTI）期货快速拉升4.71%，报53.33美元。
 <br/>
 <br/>
 卡塔尔能源部长卡阿比（Saad al-Kaabi）在多哈表示，此举是考量到〝技术性和战略性〞改变，不是出于政治动机。
 <br/>
 <br/>
 2017年6月5日以来，OPEC主要成员国沙特以及其它三个阿拉伯国家宣布与卡塔尔断交，并实施贸易禁运，指责卡塔尔支持恐怖主义及其地区竞争对手伊朗。遭卡塔尔否认。
 <br/>
 <br/>
 COMMERZBANK全球金融经济学者Peter Dixon:〝卡塔尔受到海湾邻国的强大压力，却说退出OPEC不是政治原因，我觉得就是政治原因。〞
 <br/>
 <br/>
 卡塔尔在OPEC已有57年，是14个成员国中最小的。日产量仅为每天60万桶，而沙特日产量为1100万桶，占OPEC总产量的三分之一，也是世界最大出口国。
 <br/>
 <br/>
 卡塔尔表示今后将把重点放在天然气生产上。卡塔尔是全球液化天然气（LNG）第一大出口国，年产量7700万吨。
 <br/>
 <br/>
 新唐人电视李净明综合报导
