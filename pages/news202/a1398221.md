### 伦敦治安堪忧 72小时3起谋杀 可疑包裹现街道
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】英国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦.html" target="_blank">
  伦敦
 </a>
 今(5)日在国会大厦附近发现一起可疑包裹，警方封闭该处调查，不久后重新开放，警方尚未对此进一步说明。另外，本月初伦敦南部在短短72小时内，先后有3名男子被人谋杀，令民众更忧虑首都的治安问题。
 <br/>
 <br/>
 综合媒体报导，英国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦.html" target="_blank">
  伦敦
 </a>
 警方周一稍早表示，因一起〝仍在发展中的事件〞，警方已将国会大厦附近的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国会广场.html" target="_blank">
  国会广场
 </a>
 （Parliament Square）的街道及地铁西敏站的一些出入口关闭。
 <br/>
 <br/>
 警方消息人士透露，一个可疑包裹正在调查中，关闭区域可能会在不久后重新开放，但未透露进一步详情。
 <br/>
 <br/>
 随后，警方在推特贴文说：〝西敏市白厅大道（Whitehall Westminster）发生的事件，现已落幕，此一区域的街道正在重新开放中。〞
 <br/>
 <br/>
 另外，伦敦的治安问题令民众愈来愈忧虑，上周四(1日) 晚，一名15岁少年在伦敦南部被人用刀刺死；一天后，另一名17岁少年在地铁站外也被人用刀当街刺死。4日中午，又有一名22岁男子被人用刀刺伤，经救护员到场抢救，但不治身亡。
 <br/>
 <br/>
 警方事后逮捕两名分别19岁及21岁疑犯。短短72小时内，先后有3名男子被人谋杀，令公众愈来愈忧虑首都的治安问题。
 <br/>
 <br/>
 伦敦市长简世德批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谋杀案.html" target="_blank">
  谋杀案
 </a>
 猖獗，与政府削减警队预算有关。
 <br/>
 <br/>
 (责任编辑：夏明义)
