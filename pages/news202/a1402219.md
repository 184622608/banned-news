### 美军战机日本海域相撞 2人获救 5人失踪
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】当地时间周四，距离日本海岸320公里上空，美国一架战机和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空中加油.html" target="_blank">
  空中加油
 </a>
 机，在例行训练时相撞。两人获救，5人失踪。搜救行动仍在进行，事故原因不明。
 <br/>
 <br/>
 周四凌晨两点，两架美国海军陆战队飞机，F/A18〝大黄蜂〞战机和KC-130加油机，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本海域.html" target="_blank">
  日本海域
 </a>
 上空例行训练时发生相撞。
 <br/>
 <br/>
 美国驻日本大使威廉・哈格蒂表示，两架飞机练习
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空中加油.html" target="_blank">
  空中加油
 </a>
 时发生意外。
 <br/>
 <br/>
 美国驻日本大使威廉・哈格蒂：〝今天凌晨，日本海域的空中加油练习中，一架装有燃料的KC-130加油机和一架F-18战机相撞。〞
 <br/>
 <br/>
 事故发生时，两架飞机共有7名海军陆战队员。之后两人获救，分别在日本南部岩国空军基地（Marine Corps Air Station Iwakuni）和日本军舰接受治疗。搜救行动仍在进行。
 <br/>
 <br/>
 美国驻日本大使威廉・哈格蒂感谢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本自卫队.html" target="_blank">
  日本自卫队
 </a>
 （Self Defence Force）的及时搜救。
 <br/>
 <br/>
 美国驻日本大使威廉・哈格蒂：〝我要感谢日本自卫队，立即加入我们的行动，对有关人员进行搜救。〞
 <br/>
 <br/>
 威廉・哈格蒂表示，美军冒着生命危险在保护日本和该地区。
 <br/>
 <br/>
 美国驻日本大使威廉・哈格蒂：〝他们每天冒着生命危险，保护日本和这个地区，有时会付出极大代价。〞
 <br/>
 <br/>
 最近四年，美国军事航空事故发生率飙升了近40%，至少133名军人遇难。这促使美国国会举行听证会解决问题。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
</div>
