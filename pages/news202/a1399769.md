### 中共嚣张驱逐东道主媒体  巴国媒体愤怒:这是耳光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】中国国家主席习近平日前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴布亚新几内亚.html" target="_blank">
  巴布亚新几内亚
 </a>
 （巴新）与8个太平洋岛国领导人举行会晤时，中共官员竟阻止来自其他国家的媒体以及巴新本国的媒体进入会场，令当地媒体大呼屈辱。与此同时，美国宣布将与澳洲联手在巴新极富战略价值的马努斯岛建造海军基地，此举据信是为了防范中共在该地区的军事扩张。
 <br/>
 <br/>
 正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴布亚新几内亚.html" target="_blank">
  巴布亚新几内亚
 </a>
 进行国事访问的习近平，于16日出席了在当地举行的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国-太平洋岛国领袖峰会.html" target="_blank">
  中国-太平洋岛国领袖峰会
 </a>
 〞。这个由中共发起的峰会邀请了南太平洋8个与中国有邦交关系的国家首脑与会，被认为是北京推动其一带一路倡议，扩大其对太平洋地区影响力的一个重要举措。
 <br/>
 <br/>
 不过，据外媒报导，中共官员在这次会议的现场表现得十分嚣张，肆意阻挡其他国家的媒体参加会议，就连巴国当地媒体也被拒绝入场，引起外媒与巴新当地媒体的强烈反感。有巴新本国的媒体形容，被驱逐的感受就像被人〝打了一记耳光〞一样屈辱。
 <br/>
 <br/>
 综合外媒报导，那些被驱赶的国际媒体与巴新本国媒体，都是通过向巴国政府申请，并获得了特别通行证的。然而，就在这些媒体的记者和摄影师到达举行会议的饭店后，却遭中国代表团官员驱赶阻挡，只有中共中央电视台被允许进入。
 <br/>
 <br/>
 事后，巴新本国的媒体〝国民报〞（The National）愤怒地表示，在自己的国家进行采访却被他国官员禁止，已经伤害到了巴新的新闻自由原则。
 <br/>
 <br/>
 澳洲广播公司（ABC）则报导称，部分巴新政府官员对中国代表团成员驱逐媒体行为相当不满。
 <br/>
 <br/>
 与此同时，前往巴布亚新几内亚出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 的美国副总统彭斯与澳洲总理莫里森，则乘此次会晤的机会敲定了在巴新极富战略价值的马努斯岛（Manus Island）合作打造海军基地的计划。
 <br/>
 <br/>
 舆论认为，中共近来正致力于加强自身在太平洋的影响力与军力，若让中共在马努斯岛取得先机，恐给关岛的美军基地带来威胁。
 <br/>
 <br/>
 今年6月间，巴新总理欧尼尔（Peter O'Neill）曾对外透露，中共政府可能会资助巴新的港口开发。这个消息让作为邻国的澳洲政府忧心忡忡，因为中方投资在巴新建设港口，意味着今后作为战略要地的马奴斯岛将可能经常性地开放让中共的军舰停泊。
 <br/>
 <br/>
 为此，澳洲政府立即采取了反制中共扩张的行动。虽然今年8月间澳洲领导班子发生了改变，但抗衡中共的提案还是迅速定了案。
 <br/>
 <br/>
 到本月稍早时候，澳洲政府及对外宣布将出钱资助马奴斯岛港口的发展，而预计这次在巴新首都莫士比港（PortMoresby）举行的APEC峰会上，澳洲将与巴新正式签署相关协议。当此时机，美国政府决定与澳洲政府合作，共同帮助巴新在马努斯岛打造大型海军基地，被认为是为了强化抗衡中共在太平洋地区的势力扩张。
 <br/>
 <br/>
 （记者黎明综合报导）
