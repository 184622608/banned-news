### 中美APEC冷战内幕 习近平彭斯回避对方演讲
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 和美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 ，在刚刚闭幕的亚太经济合作峰会（APEC）上针锋相对的演讲，一时成舆论焦点。日前港媒披露中美代表团在峰会上冷战的更多细节：双方始终保持距离，刻意回避对方领袖的演讲。
 <br/>
 <br/>
 据《南华早报》11月19日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 17日先后在APEC发表讲话，互相激烈地指责。但双方都回避了对方的演讲。
 <br/>
 <br/>
 在习近平发表演讲时，彭斯等人并没有到场。而在习近平演讲结束，彭斯上台前，中方代表团也集体离开了会议厅。
 <br/>
 <br/>
 报导还说，整个峰会期间，除了习近平和彭斯在公开场合曾有短暂的交流外，中美代表团始终保持距离，双方都以团队的形式进出，没有机会和对方人员自由交谈。
 <br/>
 <br/>
 在演讲中，习近平指责美方施行〝贸易保护主义〞，宣称〝无论冷战、热战还是贸易战都没有赢家〞。而彭斯则回击中方的贸易壁垒才是真正的保护主义，讽刺〝一带一路〞是条〝不归路〞，并警告在中方做出改变前，美方绝不会改变方向。
 <br/>
 <br/>
 有分析指，在G20的中美首脑峰会前，美方继续对中方施压，而中方也在为G20峰会做出强硬姿态。经过APEC的激烈交锋，外界对中美在G20峰会〝和解〞进一步降低了期望值。
 <br/>
 <br/>
 在此之前，中方媒体一直宣传中美关系〝有望缓和〞。但据路透社19日报导，来自中共官员的消息指，经过几轮的谈判，中方已对G20能否出现转机持〝谨慎态度〞。
 <br/>
 <br/>
 此外，因为中方的反对，此次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 没有发布联合公报。多家外媒消息指，中方坚决反对公报中〝反对任何形式的贸易保护主义〞这一措辞，认为这是美方刻意针对中方的言论。
 <br/>
 <br/>
 而讽刺的是，中方一直抨击美国的〝贸易保护主义〞。外界评论认为，中方对联合公报的激烈反应等于变相承认，他们知道自己才是真正的〝贸易保护主义〞。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
