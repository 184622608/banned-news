### 外交争端现转圜？沙国邀卡塔尔参加波湾峰会
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】自去年遭中东一些国家孤立的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡塔尔.html" target="_blank">
  卡塔尔
 </a>
 (
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡达.html" target="_blank">
  卡达
 </a>
 )，日前接到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特阿拉伯.html" target="_blank">
  沙特阿拉伯
 </a>
 邀请其国王塔米姆出席9日举行的波斯湾合作理事会(GCC)高峰会。由于这是卡塔尔陷入最严重的外交危机后，获得的邀请函显得意义重大。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡塔尔.html" target="_blank">
  卡塔尔
 </a>
 新闻社（Qatar News Agency）在推特表示，卡塔尔已收到〝来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特阿拉伯.html" target="_blank">
  沙特阿拉伯
 </a>
 国王的邀请函〞，参加9日在利雅德(Riyadh)举行的波斯湾合作理事会（Gulf Cooperation Council‧GCC）高峰会。
 <br/>
 <br/>
 不过，推文中未提到塔米姆（Sheikh Tamim bin Hamad Al-Thani）是否将应邀前往利雅德开会。
 <br/>
 <br/>
 据悉，去年由科威特主办GCC峰会时，沙国、阿联与巴林都是派部长或副总理出马，而非由国家元首出席。
 <br/>
 <br/>
 科威特官员上个月表示，本月的利雅德峰会，GCC全部6个成员国预期都将派员参加。
 <br/>
 <br/>
 GCC是由巴林、科威特、阿曼、卡塔尔、沙特阿拉伯与阿拉伯联合大公国共同组成。自2017年6月以来，沙国、阿联与巴林，还有非属GCC的埃及，指控卡塔尔当局资助〝恐怖主义〞、导致区域不稳定，还与沙国最大对手伊朗关系亲近，切断与卡塔尔的关系，并对卡塔尔实施陆海空封锁。
 <br/>
 <br/>
 尽管卡塔尔否认指控，但外交争议拖延一年半，且几乎没有迹象显示问题获得解决。在此期间，科威特一直在居中调解。
 <br/>
 <br/>
 此前，包括沙国与阿联先前曾多次表示，解决与卡塔尔的争端并非他们优先要务。但在沙国记者哈绍吉(Jamal Khashoggi)10月遭谋杀后，利雅德当局遭到国际舆论强烈批评，饱受争议的沙国王储萨尔曼亲王(Mohammed bin Salman)上个月便罕见释出和解评论，称赞卡塔尔的经济。
 <br/>
 <br/>
 本周稍早，卡塔尔宣布将退出石油输出国家组织（OPEC）。外界解读，这项举动是卡塔尔故意怠慢沙国这个OPEC主导国家。
 <br/>
 <br/>
