### 中共驻菲大使与可疑毒枭密切往来 杜特尔特：在他家过夜
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾.html" target="_blank">
  菲律宾
 </a>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜特尔特.html" target="_blank">
  杜特尔特
 </a>
 周四（4日）发表演说时爆料，纳卯市一名可疑华人毒枭与中共驻菲律宾大使赵鉴华有往来，去年中共国务院总理李克强访菲时，此人也有陪同。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜特尔特.html" target="_blank">
  杜特尔特
 </a>
 （Rodrigo Duterte）周四在马拉坎南宫向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾.html" target="_blank">
  菲律宾
 </a>
 军校校友会发表演说时称，这个麦可．杨（Michael Yang），有人说他是毒枭，中共大使赵鉴华甚至在他家里睡觉。李克强来访时，他也是陪同人员之一。
 <br/>
 <br/>
 李克强去年11月曾到马尼拉出席第31届东南亚国家协会（ASEAN）领袖会议暨相关会议。
 <br/>
 <br/>
 他补充说，他常和赵大使在一起，他骗不了赵大使，因为赵是军人出身。
 <br/>
 <br/>
 杜特尔特说，大约16年前，麦可．杨来到菲律宾，现已在纳卯市买下整片街区，经营多家百货公司。他说，〝但我不知道他在玩什么，他可能是个吸毒者，也可能是个毒贩。〞
 <br/>
 <br/>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒品.html" target="_blank">
  毒品
 </a>
 深恶痛绝的杜特尔特再次警告：〝不要搞毒品，因为我会把你活埋。〞他也向在场军校校友保证，不会包庇违法的中国人。
 <br/>
 <br/>
 马拉坎南宫周四深夜向媒体传送了杜特尔特的演说文字纪录。
 <br/>
 <br/>
 一名要求匿名的侨领告诉中央社记者，杜特尔特口中的麦可．杨是一名在纳卯市经商的中国新移民。
 <br/>
 <br/>
 杜特尔特近来批抨中国在南海造岛并阻挠通行自由，周四又批中国外交官与可疑毒枭有关系。
 <br/>
 <br/>
 而另一方面，菲美军方高层上月底在菲律宾三军总部会面，同意增加军事交流及往来，从今年的261项增至明年的281项，增幅将近8%，包括大规模双边军事演习在内。
 <br/>
 <br/>
 有观察人士将此视为可能是杜特尔特微调菲中关系的前兆。
 <br/>
 <br/>
 另外，杜特尔特周四在马尼拉演说时还透露，他3周前接受内视镜和结肠镜检查后，医师要他本周再次接受检查，检查是要诊断消化道和结肠是否有任何异常。
 <br/>
 <br/>
 73岁的杜特尔特表示：〝我不知道自己如今身体状况如何，要等候结果出来。如果是癌症，我会告诉你们。〞他又说：〝如果已到了第3期，我不会再接受治疗，不会延长在这个位子受苦的时间。〞
 <br/>
 <br/>
 这是他首度似乎证实了自己健康有问题。
 <br/>
