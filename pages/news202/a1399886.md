### 俄媒罕见连轰中共：友好只是装样子
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】尽管中俄两国领导人一直在努力推动两国关系的发展，但日媒发现，俄媒近期接连提及〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共威胁论.html" target="_blank">
  中共威胁论
 </a>
 〞，甚至认为中共对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 的友好只是〝装样子〞。
 <br/>
 <br/>
 日本媒体近日刊文称，近年来，中共与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 在维护朝鲜、对抗美国及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济.html" target="_blank">
  经济
 </a>
 合作等方面处于〝统一战线〞。但不少
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 媒体相继报导认为，〝俄罗斯与中国之间的友好关系只是装样子〞。
 <br/>
 <br/>
 俄媒认为，中共对俄的经济贡献过少，反而是在侵蚀俄的权益，因此产生对中共的不满与警戒。
 <br/>
 <br/>
 据俄罗斯《商报》10月份报导，中共银行近期暂停与俄罗斯方面的往来，俄方接连出现被拒绝开户的情况，一些被排除在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际制裁.html" target="_blank">
  国际制裁
 </a>
 范围外的俄罗斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_企业.html" target="_blank">
  企业
 </a>
 及个人也不能幸免。
 <br/>
 <br/>
 报导称，中共没有仔细辨明企业是否是制裁对象，就封锁俄罗斯全部企业的相关交易。这一问题在6月的中俄首脑会谈上被谈及，中方当时承诺将〝改正〞，但什么也没做。
 <br/>
 <br/>
 俄罗斯《新报》也称，中共虽然表现得像俄罗斯的朋友，但其眼睛只盯着自身的利益。中国经济增长放缓，中共为了转移民众不满，维护自身统治的所谓正当性，极有可能对外采取攻击性的外交政策，如占领俄罗斯西伯利亚以远东地区等。
 <br/>
 <br/>
 俄罗斯《观点报》提及日前在珠海举行的航空展中，有俄罗斯联合航空公司和中国商飞公司合力研发的CR929远程宽体客机的全尺寸模型。
 <br/>
 <br/>
 俄罗斯试验设计局〝Atom〞发表声明称，珠海航展展出的飞机其实是俄国的产品。中共在没有通知俄方的情况下，窃取了他们的设计。
 <br/>
 <br/>
 该局总经理兼总工程师库津（Maxim Kuzin）表示，他们不会采取法律措施，因为这没有用。他认为这是中共政府的行为，并对这种厚颜无耻的行为感到震惊。
 <br/>
 <br/>
 除此之外，还有俄罗斯专家表示，如果中美之间开战，俄罗斯一定要站在华盛顿一方，因为北京将给俄带来更大的战略威胁。
 <br/>
 <br/>
 据俄罗斯〝列格努姆〞网路通讯社（REGNUM），9日引述俄罗斯莫斯科人文大学国际关系和外交学教研室主任普拉托什金（Nikolay Platoshkin）的话说，如果中美对抗，俄罗斯应当站在美国一方。
 <br/>
 <br/>
 普拉托什金说，自1945年以来，中共已两次攻击苏俄，这是前所未有的情况。在20世纪70年代，苏联军队的70%不是位于欧洲，而是位于中俄边境。
 <br/>
 <br/>
 普拉托什金说，鉴于中美之间的矛盾越来越多，对于俄罗斯来说，最重要的就是尽可能回避冲突。中共试图使俄罗斯〝闭着眼睛出牌〞也是不现实的，俄罗斯不会卷入别国的冲突。
 <br/>
 <br/>
 尽管中俄领导人分别称，两国〝友谊一直在加强〞，但美国ABC电视台引用专家分析说，实际上中俄自己都不相信双方会结成军事联盟。
 <br/>
 <br/>
 包括俄罗斯军事专家亚历山大‧格尔茨（Aleksander Goltz）在内的许多分析人士也说，在俄罗斯与西方交恶、与美国对抗升级之际，中共并不是俄罗斯的盟友，更不会为俄罗斯而战，而俄军也一直把中共看成潜在威胁。
 <br/>
 <br/>
 俄罗斯与中共9月份举行大型联合军演时，美国国防部长马蒂斯也曾表示，由于长远利益不同，中共和俄罗斯走不到一起，因为每个国家通常会根据自己的利益来行动，〝而我看不到俄罗斯与中国的长远利益会使他们结成盟友。〞
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：祝馨睿）
