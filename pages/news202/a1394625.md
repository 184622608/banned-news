### 扯！鼻塞难呼吸 俄男拿长刀插头求空气（视频/慎入）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】近日，有人发现俄罗斯1名男子契诃夫的头上插着一把刀，立即报警求助。契诃夫告诉警方，是因为自己
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鼻塞.html" target="_blank">
  鼻塞
 </a>
 很难呼吸，才会拿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长刀插头.html" target="_blank">
  长刀插头
 </a>
 求空气。所幸，20公分的刀子竟没有伤到他的脑部，惊呆医生，直呼他能活着真是奇迹。
 <br/>
 <br/>
 事发于俄国顿内次克市的罗斯托夫（Poctob）地区，近日有路过的司机发现契诃夫（Yury Zhokhov）头上插着一把刀，立刻报警求助。警方到场后询问是谁将刀子插入其脑部时，他自称，是因为自己
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鼻塞.html" target="_blank">
  鼻塞
 </a>
 很难呼吸，所以才拿起长刀插入脑部，希望让空气进入体内。警方听到都被震惊了，直劝他不要动，坐好，并立即将他送医急救。
 <br/>
 <br/>
 契诃夫被送往顿内次克的市中心医院（Central City Hospital）后，医护人员检查发现他的脑部被近20公分的刀插入，位置正好在左右脑正中央，所以他才能幸运存活。
 <br/>
 <br/>
 医生称，他送医时仍有意识，还会开玩笑，让在场的人都觉得不可思议。医生直呼，〝他能活着就是个奇迹。〞所幸，经过开脑手术后，契诃夫已经逐渐恢复了健康。
 <br/>
 <br/>
 警方称，契诃夫是当地一间工厂的技术员，经了解，他曾在精神病院接受过治疗，不排除他攻击自己与精神障碍疾病有关。
