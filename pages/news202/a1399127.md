### 迎〝换血〞 墨克尔姊妹党党魁决定请辞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】德国总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨克尔.html" target="_blank">
  墨克尔
 </a>
 执政联盟的成员巴伐利亚邦基督教社会党（CSU,
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基社党.html" target="_blank">
  基社党
 </a>
 ）党魁赛贺佛12日表示，他将辞去党魁职务，时间将于本周宣布。
 <br/>
 <br/>
 69岁的赛贺佛（Horst Seehofer）为德国现任内政部长，担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基社党.html" target="_blank">
  基社党
 </a>
 党魁10年，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨克尔.html" target="_blank">
  墨克尔
 </a>
 的盟友也是批评者。赛贺佛在德国东部城市包曾（Bautzen）告诉媒体，辞去党魁不影响他在联邦政府的职务。
 <br/>
 <br/>
 中央社报导，德国执政联盟上台后，因移民问题和情报首长的仕途两度濒临瓦解。德国选民在最近两次的地方选举中，用选票惩罚保守派和社民党。
 <br/>
 <br/>
 而赛贺佛宣布请辞的时机就在墨克尔（Angela Merkel）宣布第四任期届满后不再寻求连任，并将辞去基督教民主党（CDU）党魁职务。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基民党.html" target="_blank">
  基民党
 </a>
 是基社党的联邦姊妹党。
 <br/>
 <br/>
 赛贺佛表示，10月巴伐利亚邦选举基社党遇挫，并非促成他请辞的原因。他表示：〝我之所以做这个决定是希望2019年是基社党换血的一年。〞
 <br/>
 <br/>
 此外，德国社会民主党青壮派领袖屈内特（Kevin Kuehnert）今天表示，墨克尔和社民党组成的执政联盟撑不过明年。
 <br/>
 <br/>
 屈内特表示，社民党将密切观察墨克尔12月辞去基民党党魁后情势。
 <br/>
 <br/>
 屈内特告诉德国广播电台（Deutschlandfunk）：〝明年将有很多地方选举，政治发展将带来重大冲击。我难以想像我们可以撑过这一年。我们必须利用我们的时间来准备因应提前选举‘那一天’。〞
 <br/>
 <br/>
 (责任编辑：卢勇信)
