### 【禁闻】《魔鬼在统治着我们的世界》系列报导(10)：法律篇
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】下面继续来看系列报导《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魔鬼在统治着我们的世界.html" target="_blank">
  魔鬼在统治着我们的世界
 </a>
 》,今天要播出的是第十章：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法律篇.html" target="_blank">
  法律篇
 </a>
 。本章从多个角度剖析了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产主义.html" target="_blank">
  共产主义
 </a>
 对人间法律的践踏和严重变异。
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魔鬼在统治着我们的世界.html" target="_blank">
  魔鬼在统治着我们的世界
 </a>
 》第十章：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法律篇.html" target="_blank">
  法律篇
 </a>
 ，论述了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产党.html" target="_blank">
  共产党
 </a>
 作为反神的邪教，将法律变成暴政工具，把邪恶合法化，同时又不择手段向全世界渗透，使西方法律被严重变异。
 <br/>
 <br/>
 文章开篇就阐明法律与信仰的关系。作者写道，法律是维护社会公平正义、惩恶扬善的决定性因素。自古以来，人间法律都应该秉承神意、天道，是正义与神圣的。
 <br/>
 <br/>
 比如，人类历史上第一部成文法典，是古巴比伦的《汉谟拉比法典》。当时，太阳神将法典授予汉谟拉比，让他依此来治理巴比伦。对信神者来说，神所主持的公平正义原则、善恶标准是永恒不变的。人敬仰神，依照神意制定法律，这样法律的道德基石才稳固，不会随着时间而变异。
 <br/>
 <br/>
 但共产党以仇恨为基础，仇视神、反对传统道德。因此，共产党的法律没有公平正义可言，只是共产党的暴政工具。马克思本身就是一名撒旦教徒，他声称要将世界摧毁成一片废墟。
 <br/>
 <br/>
 美国华府人权律师叶宁：〝卡尔．马克思的一生，就是一个魔鬼教徒来糟蹋人类的一生。它从目的到手段、到它的理论叙事方式、话语形态，通通反映的是魔鬼的思维体系和话语体系。〞
 <br/>
 <br/>
 文章谈到，一方面，共产政权可以完全无视法律、以〝革命〞的名义〞实行疯狂的国家恐怖主义；另一方面，共产党又随意制定、修改法律，来蒙骗、打压民众，美化与维护统治。共产党自己则永远凌驾于法律之上。
 <br/>
 <br/>
 对此，美国华府人权律师叶宁举例说，中共在宪法中，明确地把共产党定义为高高在上的领导阶级，而普通民众是社会底层，也就是当今所谓的〝低端人口〞。
 <br/>
 <br/>
 叶宁：〝世界上有这样的宪法吗？这也叫宪法？！流氓无产阶级专政它是作为一个政体、一个国体来规定的，写进了宪法大纲。这样的一部宪法就是一部赤裸裸的海盗法规、海盗手册。〞
 <br/>
 <br/>
 作者论述，在共产国家，法律已经成了暴政工具。在西方自由国家，法律也被严重变异。共产党在全球推广进化论、无神论，切断人与神的联系，颠覆法律的道德基础。它在教育、媒体、法律等西方各领域寻找代理人，竭力影响与控制法律的制定与实施。
 <br/>
 <br/>
 这其中，共产邪灵经常利用的就是所谓的自由主义、进步主义。但实际上，〝自由〞的基础是普世价值。背离普世价值、片面放大所谓〝自由〞，恰恰是魔鬼引诱人堕落的手段。
 <br/>
 <br/>
 福建宁德中级法院经济庭原审判长李建峰：〝这些魔鬼打着左派的旗号、给人们充分自由的旗号，否定神的嘱托，实际上就否定了每个人在神面前的平等，否定了普世价值，否定了自然法。他们强调个人自由至上，纵容毒品、同性恋，纵容
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产主义.html" target="_blank">
  共产主义
 </a>
 思潮等等，侵占这个世界。〞
 <br/>
 <br/>
 文章写道，近几十年来，美国法律的变异非常严重。这包括：移除〝十诫〞、加税、堕胎合法化、承认同性恋婚姻、淫秽色情出版合法化、毒品合法化、限制执法、为犯罪分子开绿灯、严格限制家长管教孩子等等。
 <br/>
 <br/>
 李建峰：〝逐步地把一些魔鬼撒旦的意志合法化、法律化，从而改变了这个社会的基本结构，使人们、善良人们的灵魂被魔鬼所剥夺。那么这些事件正在周围发生，我们必须要引起充分的重视，不能让魔鬼的计谋得逞。〞
 <br/>
 <br/>
 文章强调，这些匪夷所思、反传统的法律放纵邪恶、打击善良，把社会引向歧途，加速了人类道德的沉沦，使人类走向毁灭的边缘。因此，坚守道德底线，把神的诫命作为衡量善恶的标准，尽快摆脱共产邪灵的控制，也就成为人类当前刻不容缓的严峻任务。
 <br/>
 <br/>
 采访/陈汉 编辑/李谦 后制/柏妮
