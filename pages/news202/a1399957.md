### 涉嫌少报薪资50亿 日产董事长戈恩被逮捕
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本汽车大厂.html" target="_blank">
  日本汽车大厂
 </a>
 日产汽车董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戈恩.html" target="_blank">
  戈恩
 </a>
 涉嫌少报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薪资.html" target="_blank">
  薪资
 </a>
 ，遭到逮捕。日产汽车将解除戈恩的职务。
 <br/>
 <br/>
 经过数个月调查，东京地方检察厅特搜部周一逮捕了日产汽车董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戈恩.html" target="_blank">
  戈恩
 </a>
 ，涉嫌合谋的董事凯利也被逮捕。
 <br/>
 <br/>
 检方表示，戈恩没有据实申报有价证券报告书，少报了大约50亿日圆，相当于444万美元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薪资.html" target="_blank">
  薪资
 </a>
 。
 <br/>
 <br/>
 日产汽车证实，戈恩长年少报薪资及股票价值，并涉嫌公款私用等多项严重违规行为，董事会将在本周四开除戈恩的职务。
 <br/>
 <br/>
 这起事件，将对全球汽车销量排名第二的日产
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雷诺.html" target="_blank">
  雷诺
 </a>
 三菱汽车联盟造成严重打击。
 <br/>
 <br/>
 法国总统马克龙表示，现在评论还太早，但法国作为雷诺汽车的股东，将对联盟的稳定保持高度警觉。
 <br/>
 <br/>
 法国总统马克龙：〝(法国)作为股东国，将对（雷诺）集团的稳定保持高度警觉。作为股东，我们将全力支持。〞
 <br/>
 <br/>
 戈恩还同时担任三菱汽车董事长。三菱汽车周一表示，目前还不清楚如何处理戈恩的董事长职务。
 <br/>
 <br/>
 戈恩生于巴西，曾在法国求学。日产汽车陷入经营危机时，由雷诺公司派到日产，2001年6月起任首席执行官，推动日产的重建，是雷诺、日产和三菱企业联盟的第一推手。
 <br/>
 <br/>
 新唐人记者李梅、张琪综合报导
