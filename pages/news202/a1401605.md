### 分歧大于共识？  G20公报〝难产〞引发关注
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会，周五开幕。此次峰会，正值美中贸易摩擦，俄罗斯和乌克兰发生冲突，等敏感事件。已经有媒体披露，峰会公报的协商过程，非常〝艰难〞，引发外界关注。
 <br/>
 <br/>
 以经济为主题的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会，周五举行开幕仪式，各国领导人集体合影。不过，德国总理默克尔因为专机出现故障，未能及时赶到会场。
 <br/>
 <br/>
 阿根廷总统马克里，作为东道主向各位与会领袖致辞。马克里强调，希望各国通过对话取得共识。
 <br/>
 <br/>
 阿根廷总统马克里：〝我希望通过一天半的努力，我们能够为未来十年达成共识奠定基础。这对在场的每个人都是一项重大任务。我们都在见证社会正在发生的深刻变化。〞
 <br/>
 <br/>
 不过，路透社引用德国官员的话披露，和以前相比，这次峰会公报的协商过程异常艰难。
 <br/>
 <br/>
 在贸易、移民、气候变化等存在争议的重要议题上，与会各国直到峰会开幕前一天，仍存在重要分歧。
 <br/>
 <br/>
 两周前的APEC峰会，史无前例的未能达成领袖公报。《日经新闻》披露，北京方面对〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保护主义.html" target="_blank">
  保护主义
 </a>
 〞一词的定义，与其他国家发生分歧。
 <br/>
 <br/>
 此事引发外界关注，本次G20峰会公报能否及时发布，以及公报的内容和用词。有欧洲官员透露，与会各国已经同意在公报中加入改革WTO的声明。而在气候议题上，川普政府预计将坚持立场，继续反对《巴黎气候协议》。
 <br/>
 <br/>
 此外，俄罗斯一周前扣押乌克兰船隻引发的风波，导致川普取消了与普京在峰会的会晤。七国集团外长也发布联合声明，要求俄罗斯释放乌克兰船员。
 <br/>
 <br/>
 而峰会之外，全球的焦点仍聚集在周六晚间的川习会上。美国贸易代表莱特希泽已经表示，美中之间是否会达成协议，完全取决于两位领导人。
 <br/>
 <br/>
 新唐人记者林宇综合报导
