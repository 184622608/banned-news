### 效仿美国 澳拟将驻以大使馆迁至耶路撒冷
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳大利亚.html" target="_blank">
  澳大利亚
 </a>
 当局周二宣布，计划将该国驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 大使馆，由特拉维夫迁往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_耶路撒冷.html" target="_blank">
  耶路撒冷
 </a>
 。这是继美国5月迁馆后，第一个考虑跟进的西方国家。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳大利亚.html" target="_blank">
  澳大利亚
 </a>
 总理 莫里森：〝我首先要强调的是，当局对中东地区‘以巴两国方案’的立场不变。但与此同时，我们对这一建议（承认
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_耶路撒冷.html" target="_blank">
  耶路撒冷
 </a>
 是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 首都）持开放态度，作为一种可能的方案。〞
 <br/>
 <br/>
 莫里森总理表示，承认耶路撒冷的首都地位，并将大使馆迁到圣城的提案，他以开放心态看待，此举将改变澳洲数十年来的外交策略。
 <br/>
 <br/>
 当局同时计划加强和以色列的安全合作，并重新评估伊朗核协议，以确保协议符合国家利益和现行政策。
 <br/>
 <br/>
 以色列总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内塔尼亚胡.html" target="_blank">
  内塔尼亚胡
 </a>
 当天在推特发文，对莫里森表示感谢，并承诺将继续巩固两国关系。
 <br/>
 <br/>
 新唐人记者曹菁综合报导
</div>
