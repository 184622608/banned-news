### 英超俱乐部老板坠亡 华人血统背景惊人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】英媒日前报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英超.html" target="_blank">
  英超
 </a>
 莱斯特城（Leicester City）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俱乐部.html" target="_blank">
  俱乐部
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_主席.html" target="_blank">
  主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维猜.html" target="_blank">
  维猜
 </a>
 （Vichai Srivaddhanaprabha）被证实在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_直升机.html" target="_blank">
  直升机
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠毁.html" target="_blank">
  坠毁
 </a>
 事故中身亡，机上5人全部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇难.html" target="_blank">
  遇难
 </a>
 。
 <br/>
 <br/>
 10月29日，英媒BBC报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英超.html" target="_blank">
  英超
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俱乐部.html" target="_blank">
  俱乐部
 </a>
 莱斯特城证实，俱乐部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_主席.html" target="_blank">
  主席
 </a>
 老板
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维猜.html" target="_blank">
  维猜
 </a>
 ・斯里瓦塔那布拉帕的一架私人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_直升机.html" target="_blank">
  直升机
 </a>
 在俱乐部体育馆外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠毁.html" target="_blank">
  坠毁
 </a>
 。
 <br/>
 <br/>
 事故发生于英国时间10月27日晚20:30左右，机上人员共有5名，包括维猜与他的两名工作人员、直升机飞行员及一名乘客。事发前一小时，莱斯特城刚刚在主场王权球场结束与西汉姆联队的比赛。
 <br/>
 <br/>
 目击者称，飞机是在起飞后几秒钟内失去控制撞向地面，现场燃起大火。
 <br/>
 <br/>
 随后，英国紧急服务人员赶到现场救火并搜查残骸，出事区域被莱斯特郡警方封锁。当时机上5人全部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇难.html" target="_blank">
  遇难
 </a>
 。
 <br/>
 <br/>
 10月28日，大批英国莱斯特城球迷在King Power体育场外悼念飞机失事遇难者。
 <br/>
 <br/>
 10月29日，泰国报纸刊登英超莱斯特城俱乐部主席维猜去世消息。
 <br/>
 <br/>
 事发时身在现场的BBC莱斯特电台记者伊恩・斯特林格（Ian Stringer）表示，〝他是位亿万富翁，一名成功的财富人士，〞〝但他同时又是那么谦虚和亲切友好。〞
 <br/>
 <br/>
 现年60岁的维猜是泰国亿万富翁，生前为免税财团王权国际集团（King Power International Group）主席，该集团价值约360亿美元。
 <br/>
 <br/>
 2010年，维猜以3900万英镑价格收购莱斯特城，为其付清欠款，并见证球队在4年后重返英超。2015年至2016年赛季，莱斯特城以5000比1的夺冠赔率出发，一举将英超冠军头衔收入囊中。
 <br/>
 <br/>
 维猜与泰国王室关系友好，频繁资助王室的慈善项目，其姓氏〝西瓦德诺巴帕〞是由泰国国王御赐，意为〝光荣进步之光〞。
 <br/>
 <br/>
 《福布斯》杂志评选他为泰国第五大富豪，拥有价值38亿英镑的商业帝国。但维猜低调著称，极少接受媒体采访。
 <br/>
 <br/>
 值得一提的是，维猜拥有华裔背景，祖籍为中国福建省漳州市诏安县，泰国有名的诏安籍实业家。
 <br/>
 <br/>
 （记者馨综合报导/责任编辑：李泉）
