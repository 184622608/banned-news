### 法媒：孟宏伟问题极为敏感严重 北京丢丑也要抓人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 离奇失踪事件震惊海内外。据称孟已被中共方面控制。有法媒分析，孟宏伟的案情可能极为敏感或严重，北京当局才不顾在国际上丢丑的风险，也要抓人。
 <br/>
 <br/>
 法国电视台《Europe 1》5日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 9月29日告别妻小，只身从国际刑警组织的里昂总部返回中国。但一从法国离境，孟宏伟就与同僚及妻儿失联，至今仍不知生死、不知所踪。
 <br/>
 <br/>
 法国警方透露，孟宏伟失踪后，孟妻于当地时间10月4日傍晚，向法国警方报案。但法国检警至今仅能确认〝孟宏伟不是在法国境内失踪的〞、〝全案不是单纯的刑事案件〞。
 <br/>
 <br/>
 国际刑警组织表示，已关注到有关报导。国际刑警组织已正式向中共询问孟宏伟下落，并要求给予正式答覆。
 <br/>
 <br/>
 10月5日，港媒《南华早报》爆料，孟宏伟抵达中国后，立即被中纪委〝带走〞，目前正接受调查。据悉，他涉及经济问题及其他贪腐问题，其中包括孟违规在香港购置房产。
 <br/>
 <br/>
 但也有分析认为，中共官方因孟贪腐违规购房而大动干戈，对其扣押惊动世界，逻辑上不成立。
 <br/>
 <br/>
 法国《世界报》评论说，虽然孟宏伟并不掌管国际刑警组织的运作实权，但作为重大跨国治安组织的主席却凭空〝被消失〞，难堪的窘况很难不让国际社会对中共投以质疑的眼神。
 <br/>
 <br/>
 评论认为，除了难堪之外，此案也让中共内部的争斗与贪腐问题被搬到国际舞台检视。因此孟宏伟所涉及的问题，必定极为敏感或严重，才会让北京宁可冒着对国际出丑也要逮人的政治风险。
 <br/>
 <br/>
 港媒《苹果日报》也分析说，孟宏伟可能涉嫌中共政治忌讳，涉嫌北戴河政变案。因为案件重大而紧急，中共才不顾国际影响而紧急对孟宏伟问话。
 <br/>
 <br/>
 因为北京的沉默，孟宏伟失踪案至今仍没有得到中共官方消息证实或澄清。
 <br/>
 <br/>
 今年64岁的孟宏伟，出生在黑龙江省哈尔滨市，曾在中共安全机关担任过多项职务，2016年当选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 后，依然保留中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 的职位。
 <br/>
 <br/>
 孟宏伟在公安部一路高升期间，公安部长正好是权倾一时的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 。当周在2013年落马并判以无期徒刑后，与周有牵连的孟却能全身而退，甚至还被中共政权推上国际刑警组织主席的要职。
 <br/>
 <br/>
 然而孟宏伟上任一年多后，中共政局剧变。2017年底，孟宏伟被免除中共国家海洋局副局长、中共海警局局长的兼任职务，2018年4月，又被免去已担任14年的公安部党委委员一职。
 <br/>
 <br/>
 中共十九大上，赵克志接任中共公安部部长一职，多次要求公安部继续清洗周永康的〝余毒〞。因此，有分析认为，孟宏伟作为周的〝余毒〞被拿下也不令人意外。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：祝馨睿）
</div>
