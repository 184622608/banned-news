### 美媒：国会报告中俄朝伊或动用电磁脉冲攻击美国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】周三有美国媒体爆料称，美国国会最近发布的一份报告指称，中、俄、朝、伊等拥有核武的政权都在积极发展
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电磁脉冲.html" target="_blank">
  电磁脉冲
 </a>
 的作战能力，在未来与美发生极端冲突时，有可能使用电磁脉冲（EMP）来攻击美国。
 <br/>
 <br/>
 美国媒体《华盛顿自由灯塔》（Washington Free Beacon）网24日的一篇报导披露，美国国会最近发布的一份报告称，在过去的10年间，包括朝鲜在内的一些拥有核武的国家（政权）一直在积极发展
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电磁脉冲.html" target="_blank">
  电磁脉冲
 </a>
 的作战能力，对美国的国家安全构成了威胁。
 <br/>
 <br/>
 这份报告表示，中国（中共）和俄罗斯还拟定了电磁脉冲的作战计划，而中、俄和伊朗三国的政府甚至还考虑使用高空核爆的手段来阻止美军搜集情报以及导航等作战能力。
 <br/>
 <br/>
 报告呼吁美国国防部应开发专属的电磁脉冲武器，并建议美国的核武应包含有限度的核爆电磁脉冲攻击，以便将来能够抵御、威震那些敌对的国家。
 <br/>
 <br/>
 据公开的资讯，电磁脉冲武器主要分为核爆电磁脉冲武器与非核电磁脉冲武器两种。其中〝核爆电磁脉冲〞，指的是当核弹在一定的高度爆发时，伽玛辐射与空气中的氧、氮原子相撞击而产生的强大的电磁脉冲。
 <br/>
 <br/>
 电磁脉冲虽不致命，但可以在大范围内长时间地破坏地面的电网，而受到〝电磁脉冲〞干扰的地区将因断电而陷入黑暗之中。更重要的是，电网长期断电会导致一些重要的基础设施无法正常运作。被电磁脉冲打击的电器件经受从外沿上的暂时电子破坏直到近中心的过压摧毁。
 <br/>
 <br/>
 例如：电源传输网络与通讯网络也极易被电磁脉冲所摧毁，而且任何含半导体的电子设备，包括机载平台的系统在受到电磁脉冲的冲击后，都会由于瞬变高压而损坏或烧毁。
 <br/>
 <br/>
 电磁脉冲对电、电子设备的破坏效应是不可预见的，这些电磁脉冲打击力量可用来对付依靠复杂电子设备的敌方平台与设施，特别是敌方的指挥、控制与通讯系统和空防系统。
 <br/>
 <br/>
 不过，核爆产生的电磁脉冲效应的主要麻烦是，它会破坏有效范围的所有电子设备。因为脉冲以无方向性的方式传送并覆盖行星区，因此它在对敌军造成影响的同时也会破坏友军的设备，甚至危及自身。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
