### 开火扣军舰俄罗斯动作大 乌克兰拟戒严
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 扣押三艘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 海军舰艇，并导致6名船员受伤。引发乌克兰及其他西方国家的谴责。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波罗申科.html" target="_blank">
  波罗申科
 </a>
 周一宣布，他已签署行政令，将于本月28日起全国戒严30天，以防止俄方潜在的入侵行为。但不会限制公民权利，也不会影响到明年3月的总统大选。这一戒严令尚待国会批准。
 <br/>
 <br/>
 乌克兰总统Petro Poroshenko：〝在这种情况下，戒严不会和竞选活动开始后的任何一天活动重合。〞
 <br/>
 <br/>
 周日晚些时候，一些乌克兰人聚集在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 驻基辅大使馆外抗议，并有人投掷烟雾弹。
 <br/>
 <br/>
 乌克兰当局并呼吁其西方盟国，考虑进一步制裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莫斯科.html" target="_blank">
  莫斯科
 </a>
 。
 <br/>
 <br/>
 美国对此事表达〝深切关注〞，并认为2014年俄罗斯占领克里米亚后，其行为没有收敛。
 <br/>
 <br/>
 美国驻联合国大使黑利：〝过去4年，国际社会联合发出了强烈、清晰、广泛的呼声，以谴责俄罗斯入侵乌克兰。周日的所作所为，是俄罗斯侵犯乌克兰领土行为的一部分。〞
 <br/>
 <br/>
 同一天，北约秘书长斯托尔滕贝格（Jens Stoltenberg）也发声支援，敦促俄罗斯释放乌克兰海军舰艇和水手。
 <br/>
 <br/>
 北约秘书长Jens Stoltenberg：〝所有（北约）盟国都完全支持乌克兰的领土完整和主权。因此，我们呼吁俄罗斯确保无阻碍进入乌克兰港口，并允许乌克兰人在亚速海和刻赤海峡的航行自由。〞
 <br/>
 <br/>
 对于外界回应，俄方则指责乌克兰与西方密谋策划危机。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
