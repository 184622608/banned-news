### 时事备忘录：高管被捕 业务四处碰壁 华为是何背景
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被捕引起各方的关注，一起来了解背后的来龙去脉。
 <br/>
 <br/>
 目前，外界掌握的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被捕的原因是，涉嫌利用全球银行体系和伊朗进行交易，违反了美国对伊朗的制裁。孟晚舟是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的女儿，有传闻说，孟晚舟可能就要在12月接替任正非主掌华为，她的被捕因此更受关注。
 <br/>
 <br/>
 而任正非曾在中共解放军工程部门任职，从解放军上校军衔退役，1978年加入中共。
 <br/>
 <br/>
 华为自称是民营企业，和中共政府没关系。但批评者质疑，作为极权政府，中共是否会允许一家已经打入西方国家基础设施的技术公司保持独立？
 <br/>
 <br/>
 华为现在是全球最大的手机和网络零部件供应商，美联社报导指，华为早已为中共军队和安全部门从事间谍活动。
 <br/>
 <br/>
 2008年以来，美国政府和智库进行的多分调查显示，华为从中共官方得到钜额资助，并为中共提供情报服务。中共政府和军方都是华为的重要客户，也是其研发合作的伙伴，长期无偿向华为提供关键技术。
 <br/>
 <br/>
 因此，西方国家非常担心华为带去的国家安全隐患。
 <br/>
 <br/>
 美国、澳大利亚、新西兰和英国已经宣布不准在本国5G网络建设中使用华为产品。德国政府内部也对华为有疑虑。
 <br/>
 <br/>
 12月7日，欧盟技术专员安西普表达了对包括华为在内的中国制造商的担心。
 <br/>
 <br/>
 同一天，日本也传出可能禁止本国政府向华为和中兴通讯采购产品。
 <br/>
 <br/>
 加拿大则正在对华为的产品进行安全审查。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
