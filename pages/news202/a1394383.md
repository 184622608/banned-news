### 梅拉尼亚造访埃及 谈与川普沟通方式
------------------------

<div class="wysiwyg">
 【新唐人2018年10月06日讯】周六，美国第一夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼亚.html" target="_blank">
  梅拉尼亚
 </a>
 访问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃及.html" target="_blank">
  埃及
 </a>
 ，并以此结束首次非洲之行。梅拉尼亚接受记者提问时，谈到大法官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 ，以及她和川普之间的沟通等话题。
 <br/>
 <br/>
 周六，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼亚.html" target="_blank">
  梅拉尼亚
 </a>
 造访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃及.html" target="_blank">
  埃及
 </a>
 ，这也是她首次非洲之行的最后一站。
 <br/>
 <br/>
 埃及总统塞西和夫人会晤梅拉尼亚，对她的到来表示欢迎。
 <br/>
 <br/>
 随后，埃及古文物部长阿纳尼和旅游部长马沙特，陪同梅拉尼亚参观了埃及著名的大金字塔和狮身人面像。
 <br/>
 <br/>
 梅拉尼亚在狮身人面像前回答了记者的提问，谈到大法官提名人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 以及控诉遭性侵的福特女士。
 <br/>
 <br/>
 美国第一夫人 梅拉尼亚：〝如果是説最高法院大法官提名人卡瓦诺，我认为他非常有资格进入最高法院。我也很高兴福特博士的声音被公众听到，卡瓦诺法官的意见也被公众听到。〞
 <br/>
 <br/>
 对于是否信任福特的指控，梅拉尼亚没有正面回答，但表示她反对任何形式的暴力和欺凌。
 <br/>
 <br/>
 美国第一夫人 梅拉尼亚：〝我认为所有受害者，我们都要去帮助。不管他们受了什么样的欺凌。〞
 <br/>
 <br/>
 记者也问梅拉尼亚，是否会影响川普的看法和决定。梅拉尼亚说，她会如实地向丈夫表达自己的想法。
 <br/>
 <br/>
 美国第一夫人 梅拉尼亚：〝我告诉他。我会如实表达我的看法和建议，有时他会听，有时他不会。但我有自己的声音和观点，对我来説能表达我的感受很重要。〞
 <br/>
 <br/>
 梅拉尼亚这次非洲之行，也表达了对美国国际开发署的支持。目前，该机构正在和埃及合作，保护地标景观。
 <br/>
 <br/>
 当天，梅拉尼亚将返回美国，她用〝非凡、难忘〞来形容这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲之旅.html" target="_blank">
  非洲之旅
 </a>
 。
 <br/>
 <br/>
 新唐人记者林宇、张琪综合报导
</div>
