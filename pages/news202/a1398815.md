### 墨尔本恐怖袭击酿2死2伤 嫌犯来自索马利亚(视频)
------------------------

<div class="wysiwyg">
 【新唐人2018年11月10日讯】澳大利亚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨尔本.html" target="_blank">
  墨尔本
 </a>
 9日下午发生恐怖袭击事件。嫌犯疑开四轮传动车冲撞商家起火燃烧，并用刀刺死1人，刺伤2人。当地警方迅速赶到，将嫌犯击毙。伊斯兰圣战组织不久声称对此袭击承担责任。
 <br/>
 <br/>
 袭击事件发生在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨尔本.html" target="_blank">
  墨尔本
 </a>
 市中心的商业区的布尔克街（Bourke Street）。社交网络上流传的视频显示，一辆汽车起火燃烧，一名持刀男子袭击路人，一名受害者当场死亡，另外两人受伤。
 <br/>
 <br/>
 两名警察赶到现场，在一名路人的协助下，警察中有一人开枪，袭击者不久后倒地。
 <br/>
 <br/>
 据澳洲《世纪报》（The Age）报导，受害者包括当地一家知名餐厅的共同老板马拉斯皮纳（Sisto Malaspina），遭歹徒攻击身中多刀，不幸罹难。
 <br/>
 <br/>
 许多客人前往店门口致意，多年的合作伙伴潘格拉乔（Nino Pangrazio）伤心地说，〝我觉得糟透了。我们从1974年开始合作，在那之前的10年就已经是朋友了。〞

 <br/>
 当地警方负责人透露，当局至少3年前就认识袭击者，是一名在当地居住了数十年之久的索马里亚的希尔阿里（Hassan Khalid Shire Ali），于1980年代幼年时和家人逃到澳洲。
 <br/>
 <br/>
 警方起初称，希尔阿里与极端组织没有联系，排除恐袭。但不久调查发现，希尔阿里驾驶载有瓦斯钢瓶的四轮传动车冲入市中心，冲撞商家起火燃烧，才将此事件被定性为恐怖主义。
 <br/>
 <br/>
 稍早，武装警力突击墨尔本西郊和东北郊两处和希尔阿里家人及同伙有关的地方，但据信不存在威胁。
 <br/>
 <br/>
 据了解，当局此前担心希尔阿里试图赴叙利亚加入恐怖组织伊斯兰国（IS），他的澳洲护照已在2015年遭撤销。而他的兄弟早已遭控试图取得枪枝并杀害纽约跨年民众，将在明年依恐攻相关控罪受审。
 <br/>
 <br/>
 澳洲联邦反恐警官麦卡尼（Ian Mccartney）表示，即便目前伊斯兰国在伊拉克和叙利亚失去地盘，但这次袭击是〝一记警钟〞，而〝希尔阿里如何以及何时从激进观点演变成昨天发动攻击，将成为调查重点。〞
 <br/>
 <br/>
 伊斯兰国事后透过宣传部门说凶手是〝伊斯兰国战士，他执行这项任务〞，但没有提供相关证据。
 <br/>
 <br/>
 据了解，事发现场的街区2017年12月也曾发生袭击者驾车冲撞行人事件，有6人遇害。目前驾车袭击事件的肇事者仍受审判。
