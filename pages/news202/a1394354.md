### 〝康妮〞袭韩 大量航班取消  周六晚逼近日本
------------------------

<div class="wysiwyg">
 【新唐人2018年10月06日讯】周六，强度降为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热带风暴.html" target="_blank">
  热带风暴
 </a>
 的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康妮.html" target="_blank">
  康妮
 </a>
 〞扫过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 ，给韩国南部，带来大量降雨，并继续向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 逼近。
 <br/>
 <br/>
 当地时间周六上午8点左右，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热带风暴.html" target="_blank">
  热带风暴
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康妮.html" target="_blank">
  康妮
 </a>
 〞在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 庆尚南道沿海地区登陆，局部风速高达每小时115公里。
 <br/>
 <br/>
 受此影响，釜山地区遭遇大量降雨，部分地区出现洪水。当局表示，已有276个航班被迫取消。
 <br/>
 <br/>
 在游客聚集的济州岛，〝康妮〞的中心已经开始远离，但仍然能感受到强风和大浪。
 <br/>
 <br/>
 周六中午时分，〝康妮〞中心离开朝鲜半岛，进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 海海域，向北海道移动。
 <br/>
 <br/>
 日本气象厅发布信息，〝康妮〞将在周六夜间逼近本州岛的北陆地区，其强度将进一步下降，在周日凌晨预计降为温带气旋。
 <br/>
 <br/>
 不过，日本气象厅警告，局部地区仍可能出现每小时90公里的强风，在北海道，周六至周日的24小时降雨量预计达到120毫米。
 <br/>
 <br/>
 九州岛也受到〝康妮〞影响，部分地区交通出现延误，日本中西部部分航班出现了晚点或取消的情况。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
