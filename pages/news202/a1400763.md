### 圣诞节将至 伦敦名店橱窗秀点亮节日
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】度过了感恩节，马上就要迎来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞节.html" target="_blank">
  圣诞节
 </a>
 了，在英国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦.html" target="_blank">
  伦敦
 </a>
 的街头和著名商店的橱窗，也已经装饰一新，充满温馨的节日氛围。
 <br/>
 <br/>
 你有没有见过像这样的跳舞女王？
 <br/>
 <br/>
 在摄政街（Regent Street）的上空，闪烁着蓝色和金色翅膀的天使们正在保护着游客。
 <br/>
 <br/>
 玩具店哈姆利门前，一些本应正在忙着帮圣诞老人准备派送玩具的真人扮演的精灵正在跳舞，与顾客一起开派对。
 <br/>
 <br/>
 橱窗里所有的圣诞老人都在尽显他们的魅力。
 <br/>
 <br/>
 不过鉴于对英国脱离欧盟和经济的担忧，英国人在这个节日里，消费上可能会比以往收紧腰包。而对于业主们而言，无论街道怎样繁华，吸引顾客进到商店内才是所期盼的。
 <br/>
 <br/>
 新唐人记者徐哲、天羽综合报导
</div>
