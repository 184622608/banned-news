------------------------

 【新唐人2018年10月08日讯】1. 批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港府打压.html" target="_blank">
  港府打压
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新闻自由.html" target="_blank">
  新闻自由
 </a>
 万人连署要说法
 <br/>
 <br/>
 香港政府日前拒绝延长英国《金融时报》记者马凯的工作签证，被认为是效仿中共，打压新闻言论自由,  在香港引发不满。
 <br/>
 <br/>
 香港记者协会、香港外国记者会、国际记者联会、无国界记者等新闻组织发起联署，要求港府解释拒签原因，已经获得一万五千人签名。
 <br/>
 <br/>
 10月8号，发起联署的这些团体前往港府总部递交联署信，并举行抗议活动,  要求港府给出合理说法。但港府以〝不评论个别事件〞为由拒绝。
 <br/>
 <br/>
 马凯是香港外国记者会副主席，他被拒延签证，被认为与今年8月邀请主张港独的民族党人陈浩天演讲有关。
 <br/>
 <br/>
 2. 不忘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产主义恐怖.html" target="_blank">
  共产主义恐怖
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波兰.html" target="_blank">
  波兰
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悼念受害者.html" target="_blank">
  悼念受害者
 </a>
 <br/>
 <br/>
 东欧和前苏联地区的一些国家, 最近几年越来越强调不忘共产主义恐怖历史。
 <br/>
 <br/>
 10月4号,  波兰总统府举行仪式，悼念共产主义恐怖受害者，并向一批被共产党政权安全机构和秘密警察杀害人士的家属颁发了身份鉴定证书。这些被杀害人士被确认为是效忠波兰政府军队的士兵，其他人则来自反抗苏共和波兰共产党政权的〝自由和独立〞等地下组织。
 <br/>
 <br/>
 波兰总统杜达在仪式上说，这些波兰英雄们曾为自由、主权和独立而战斗。
 <br/>
 <br/>
 波兰的邻国立陶宛也在10月6号以国葬的方式隆重下葬了反抗共产党政权的游击队领袖-瓦纳加斯。
 <br/>
 <br/>
 立陶宛总统率领几乎全部内阁高官,  还有几位前任总统，以及立陶宛天主教红衣主教出席了葬礼。
 <br/>
 <br/>
 瓦纳加斯领导的游击队反对斯大林在当地推行集体农庄政策。他被认为是立陶宛的民族英雄。下面请您收看时事禁闻。
