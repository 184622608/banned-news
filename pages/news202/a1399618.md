### 英环境大臣不辞职 奥地利吁通过草案
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】周五，英国环境大臣戈夫表示不会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 ，但他拒绝转任脱欧事务大臣。英国内阁两次辞职潮后，戈夫是目前内阁中唯一具有深厚政治实力的脱欧派人物。他的留任让首相特雷莎・梅松了一口气。
 <br/>
 <br/>
 周五，英国环境大臣戈夫表示，对首相充满信心，并说他将和其他内阁成员一起，为达成好的脱欧协议而努力。但没表态是否支持脱欧
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_草案.html" target="_blank">
  草案
 </a>
 。
 <br/>
 <br/>
 英国环境大臣戈夫：〝 我绝对（对首相有信心）。我认为，我们专注于在未来获得正确的（脱欧）协议绝对是至关重要的。〞
 <br/>
 <br/>
 英国媒体报导，特雷莎・梅曾询问戈夫是否愿意转任脱欧事务大臣，被戈夫拒绝。外界猜测，脱欧事务部是否会被取消。
 <br/>
 <br/>
 周五首相府宣布，周末将确定新脱欧事务大臣人选。
 <br/>
 <br/>
 同时，不断有保守党议员提交不信任信件。如果提交人数达到48人，下议院将对特雷莎・梅进行不信任投票。
 <br/>
 <br/>
 欧盟轮值主席国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥地利.html" target="_blank">
  奥地利
 </a>
 总理库尔茨呼吁英国下议院支持脱欧草案。他认为，目前的脱欧草案可让英国有序脱欧，并和欧盟保持有序的共存关系。
 <br/>
 <br/>
 奥地利总理库尔茨：〝 这对双方来说都是一个好协议，没有哪方被欺骗，此协议阻止了硬脱欧。我真的希望英国下议院能支持此协议（草案）。〞
 <br/>
 <br/>
 欧盟大国法国希望英国成为警示其他国家的范例，以避免其他国家也加入脱欧行列。
 <br/>
 <br/>
 法国财政部长 Bruno Le Maire：〝英国脱欧显示了，离开欧洲单一市场的经济代价是非常高昂的。〞
 <br/>
 <br/>
 周五，英镑回升近半个百分点。前一天，因为内阁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 潮大跌将近两个百分点。
 <br/>
 <br/>
 11月25日，欧盟27国首脑将讨论脱欧草案。
 <br/>
 <br/>
 新唐人记者王羽音、尚敬综合报导
