### 小米被控侵犯专利 美企要求下禁令
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月03日讯】日前，中国手机制造商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小米.html" target="_blank">
  小米
 </a>
 被爆出在美国被控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侵犯专利.html" target="_blank">
  侵犯专利
 </a>
 ，导致小米股价暴跌。这也是继中兴通讯遭到美国禁令制裁后，又一家中国通讯公司被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国企业.html" target="_blank">
  美国企业
 </a>
 要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下禁令.html" target="_blank">
  下禁令
 </a>
 ，引起大陆业界关注。
 <br/>
 <br/>
 美国技术公司Dareltech日前在纽约南区联邦法院控告
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小米.html" target="_blank">
  小米
 </a>
 故意侵犯其4项专利权利。这4项专利，主要运用在小米生产销售的自拍杆三脚架产品。
 <br/>
 <br/>
 据悉，这并非小米产品第1次被指控侵权。2015年，荷兰皇家KPN公司；今年1月、5月中国通讯公司酷派；6月底专利权人袁弓夷，都分别向法院提诉告诉，指控小米产品侵犯其专利权。
 <br/>
 <br/>
 据Dareltech公司的诉状显示，今年8月27日，该公司曾以信件通知小米其侵权行为，小米的回复也证实存在侵权行为；在9月12日的信中，小米承认了Dareltech公司的信件，但却表示〝小米不需要取得Dareltech的专利授权〞。
 <br/>
 <br/>
 Dareltech公司认为小米蓄意且有意识地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侵犯专利.html" target="_blank">
  侵犯专利
 </a>
 ，向法院提出9项要求，包括向小米发出永久禁令，以及赔偿相关专利费用的3倍金额。
 <br/>
 <br/>
 小米被指控侵犯专利的消息令其股票暴跌。10月2日，小米集团在港股股价，盘中跌至14.8港元，创下上市以来新低。小米自7月上市后，股价已从最高价的22.20港元，下跌超过33%。
 <br/>
 <br/>
 这也是继中兴通讯遭到美国禁令制裁后，又一家中国通讯公司被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国企业.html" target="_blank">
  美国企业
 </a>
 要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下禁令.html" target="_blank">
  下禁令
 </a>
 ，引起大陆业界关注。
 <br/>
 <br/>
 今年4月，中国通讯制造商龙头企业中兴通讯，因违反美国相关制裁法令，被美国商务部下达禁售令，禁止7年内购买美国芯片、软件等产品。中兴通讯随即陷入停摆状态，濒临破产。
 <br/>
 <br/>
 随后中共高层出面求情，中兴在缴纳14亿美元天价罚金，更换管理层并接受美国监督后，才获得免除禁令。此事件对中国通讯业产生震慑效应。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
</div>
