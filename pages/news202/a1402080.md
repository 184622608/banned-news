### 欧洲南美联手打击黑手党  90人被捕
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧洲.html" target="_blank">
  欧洲
 </a>
 各国警方联手行动，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南美.html" target="_blank">
  南美
 </a>
 警方跨国合作，逮捕了90名意大利
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑手党.html" target="_blank">
  黑手党
 </a>
 成员。
 <br/>
 <br/>
 这次行动，意大利，荷兰，德国和比利时警方联手出击。有90人被捕。其中意大利41人，德国21人，比利时14人，荷兰5人。
 <br/>
 <br/>
 嫌犯来自臭名昭着的意大利
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑手党.html" target="_blank">
  黑手党
 </a>
 〝光荣会〞。〝光荣会〞的老巢位于意大利南部卡拉布里亚大区的洛克里市。那里是全球犯罪集团的据点。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧洲.html" target="_blank">
  欧洲
 </a>
 司法合作组织称，这些嫌犯涉嫌贩运可卡因，洗钱，贿赂和暴力。
 <br/>
 <br/>
 周二，意大利警方宣布，逮捕了西西里岛黑手党新头目米内欧和其他45名嫌犯，罪名是诈骗钱财、非法持有枪支、纵火和组织犯罪等。
 <br/>
 <br/>
 西西里岛黑手党为自己取名为〝我们的事业〞，它和〝光荣会〞，还有那不勒斯地区的卡莫拉黑手党是意大利最主要的三个犯罪集团。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
