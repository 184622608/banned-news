### 伦敦举办双十国庆酒会嘉宾云集
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】驻英国台北代表处于10月9日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦.html" target="_blank">
  伦敦
 </a>
 的国家自由俱乐部(National Liberal Club)举行了庆祝中华民国107年国庆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒会.html" target="_blank">
  酒会
 </a>
 。台英国会小组共同主席罗根勋爵（Lord Rogan），下议院1922委员会主席布雷迪爵士（Sir Graham Brady），圣文森驻英高级专员路易斯（Cenio E.Lewis）等各国驻英使节，上下议院国会议员，政学侨等各界人士大约450人应邀出席，共襄盛举。
 <br/>
 <br/>
 驻英台北代表处代表林永乐在致辞中表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 这两年来在外交政策方面持续加强与英国等理念相近国家的伙伴关系，并透过新南向政策，强化与亚太国家关系，以确保区域和平，稳定与共同繁荣。
 <br/>
 <br/>
 其他担任致辞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嘉宾.html" target="_blank">
  嘉宾
 </a>
 纷纷盛赞中华民国经贸与民主的发展成就，以及对国际发展的贡献，并呼吁英国政府应更积极支持台湾参与国际组织，期盼英国脱欧后可与台湾发展更全面的关系。
 <br/>
 <br/>
 今年以来，台湾的外交空间连续受到中共的多方打压，在英国社会，尤其是政界有何影响，对此林永乐表示：〝我觉得我们今年的宾客比去年还要多，而且很多是国会的朋友，还有很多地方的朋友，包括智慧城市，当然还有很多国会的议员，包括上议院，下议院，让我们觉得有更多英国的朋友在支持我们。我们也希望台湾和英国的合作关系能够越来越好，能够对双方的经济更有帮助。〞
 <br/>
 <br/>
 前来参加活动的客人也纷纷谈到他们的感受。
 <br/>
 <br/>
 旅英台湾侨胞：〝很开心，有很多认识的朋友，来参加国庆，台湾被重视觉得很开心。〞
 <br/>
 <br/>
 活动期间还安排播放中华民国国情影片，并发送其他相关的文宣品，以增进宾客对中华民国政府政策的了解。
 <br/>
 <br/>
 新唐人记者鲍天宇、霄龙英国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦.html" target="_blank">
  伦敦
 </a>
 报导
</div>
