### 比利时维修战机突启动机炮 误射F-16爆炸起火全毁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比利时.html" target="_blank">
  比利时
 </a>
 空军11日发生工程师维修战机时，战机的机炮竟意外启动，并发射炮弹，击中一架载满燃油正准备执行任务的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-16战机.html" target="_blank">
  F-16战机
 </a>
 ，起火爆炸，还波及停在旁边的另一架F-16战机受损。两名技工事后出现听力损伤。
 <br/>
 <br/>
 事发于上周四下午约2时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比利时.html" target="_blank">
  比利时
 </a>
 佛洛雷恩斯（Florennes）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空军基地.html" target="_blank">
  空军基地
 </a>
 ，两名维修工程师正在机库维修一架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-16战机.html" target="_blank">
  F-16战机
 </a>
 。不知为何战机的机炮突然启动，发射的炮弹击中停泊在附近一架F-16战机起火爆炸，波及旁边另一架F-16战机。
 <br/>
 <br/>
 由于当时F-16战机载有高达1万公升的燃油，正准备升空训练，大火引起爆炸，并传出数次爆炸声，黑烟滚滚窜升，基地人员如临大敌，消防员不久后扑灭火势，但中弹的F-16战机已经面目全非。
 <br/>
 <br/>
 2名维修工程师则被炸伤，所幸送医后无重大外伤，但听力受损。
 <br/>
 <br/>
 <b>
  相关视频：
 </b>
 <center>
 </center>
 <br/>
 <br/>
 军方证实1架战机被烧毁，另1架则是轻微损伤。比利时国防贸易工会表示，事件并非蓄意或涉及恐袭，据信是人为错误或技术故障。当局正调查事件原因。
 <br/>
 <br/>
 佛洛雷恩斯空军基地位于法国南部半岛，以前是北约组织的核心基地，目前是比利时空军第2战术联队基地。据悉，比利时目前拥有2个中队与60多架F-16战斗机，目前有48架被分配到北约执勤。
