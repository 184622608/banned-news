### 俄罗斯乌克兰再起冲突 国际社会关注
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】
 <b>
  新闻周刊
 </b>
 （656）11月25日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 扣押了三艘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 海军舰艇，导致6名船员受伤，全部24名船员目前仍被关押在俄罗斯境内。俄罗斯的举动引来乌克兰和其他西方国家的谴责。美国总统川普为此取消了与俄罗斯总统普京原定在G20会议期间的会面。
 <br/>
 <br/>
 周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 禁止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 16至60岁的男性入境。
 <br/>
 <br/>
 根据乌克兰与俄罗斯签订的双边条约，两国均有权使用亚速海。
 <br/>
 <br/>
 但2014年的克里米亚危机，让原属于乌克兰的克里米亚半岛被纳入俄罗斯管辖范围，两国在亚速海的争端开始升级。这一次的冲突的起源就是，乌克兰军舰试图通过连接克里米亚半岛与俄罗斯的刻赤海峡。
 <br/>
 <br/>
 11月25日，乌克兰海军的两艘军舰和一艘拖船从黑海港口敖德萨启程，准备前往位于亚速海、也是乌克兰境内的马里乌波尔港。刻赤海峡是此次行程的必经之路。
 <br/>
 <br/>
 在刻赤海峡，俄罗斯海军声称，乌克兰舰队是非法入境，经历长时间对峙后，就在乌克兰舰队企图返回敖德萨港时，拖船遭到俄罗斯边防船的冲撞。
 <br/>
 <br/>
 乌克兰方面还表示，俄罗斯向他们的军舰开火，造成6名海军士兵受伤。
 <br/>
 <br/>
 稍后，俄罗斯封锁了刻赤海峡上唯一通往亚速海的水路通道，阻止乌克兰船隻通过。并扣押了乌克兰三艘军舰与全部24名船员，引发国际关注。
 <br/>
 <br/>
 俄罗斯在电视上曝光这些乌克兰士兵，显示他们在讲反对自己国家的话，遭到乌克兰抗议。乌克兰海军司令说，这些士兵是在威逼下被迫说谎。
 <br/>
 <br/>
 11月27日，克里米亚法庭判决，对扣押的24名乌克兰船员，拘留期延长60天。俄总统普京并在事件后首次发声，指责是乌克兰〝策划〞了此次事件，是政治家为提高支持率。
 <br/>
 <br/>
 这一事件再次受到其他西方国家的关注。
 <br/>
 <br/>
 奥地利外长克奈斯尔：〝我们非常关注，因为这是一场攻击，这个争端有可能会演变得更危险。〞
 <br/>
 <br/>
 美国的乌克兰特使沃尔克（Kurt Volker）表示，希望事态不要再升级，并质疑，2014年俄罗斯占领克里米亚后，欧洲没有严格落实对俄制裁。
 <br/>
 <br/>
 北约秘书长斯托尔滕贝格（Jens Stoltenberg）也发声支援，敦促俄罗斯释放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰海军舰艇.html" target="_blank">
  乌克兰海军舰艇
 </a>
 和水手。
 <br/>
 <br/>
 北约秘书长 斯托尔滕贝格: 〝所有（北约）盟国都完全支持乌克兰的领土完整和主权。因此，我们呼吁俄罗斯确保无阻碍进入乌克兰港口，并允许乌克兰人在亚速海和刻赤海峡的航行自由。〞
 <br/>
 <br/>
 乌克兰总统波罗申科11月26日宣布，28日上午9点开始，全国戒严30天，以防止俄方潜在的入侵行为。但不会限制公民权利，也不会影响到明年3月的总统大选。
 <br/>
 <br/>
 提案已获得议会通过，在与俄罗斯、白俄罗斯和摩尔多瓦接壤的十个省份内实行。
 <br/>
 <br/>
 实施戒严后，乌克兰总统波罗申科亲自视察边防，呼吁严防俄罗斯对乌克兰的陆上威胁。
 <br/>
 <br/>
 乌克兰总统 波罗申科：〝我们必须确保有效的人员培训。 在需要时，击败敌人
 <br/>
 <br/>
 11月29日，乌克兰军队在亚速海沿岸进行了战术演习。
 <br/>
 <br/>
 同一天，在前往阿根廷参加G20峰会的途中，美国总统川普在推特上发文，表示由于俄罗斯尚未释放此前被抓的乌克兰船员，他决定取消与俄罗斯总统普京原定在G20会议期间的会面。
 <br/>
 <br/>
 30日，乌克兰再发禁令，禁止能参与战争的16岁到60岁的俄罗斯男子进入乌克兰，惟有出于人道主义原因，才允许例外，比如俄罗斯人到乌克兰参加亲戚的葬礼。
 <br/>
 <br/>
 12月初，欧盟成员开会，欧盟是否会对俄罗斯采取进一步制裁，将取决于事件进展。
 <br/>
 <br/>
