### 韩媒：金正恩因一事恼羞成怒 朝鲜官员急赴中国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】因朝鲜在去核化进程中的拖延，美国近日明确表示，将继续对朝施加史无前例的制裁。朝鲜想利用〝无核化〞欺诈美国的伎俩失效，经济仍陷入困境，这令朝鲜领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 十分恼火，急令朝鲜官员赴中国大陆求援。
 <br/>
 <br/>
 美国中期大选结束后，川普政府（特朗普）宣布将持续对朝鲜施加严厉制裁。朝鲜对此反应激烈。11月7日，原定的美朝高级别会谈被无限期延期后，双方就无核化和制裁问题发生意见冲突。
 <br/>
 <br/>
 当日，美国总统川普在谈及朝鲜问题时称：〝无需操之过急〞，暗示将对朝进行〝长期战〞。
 <br/>
 <br/>
 川普还明确表示，在韩半岛实现完全无核化之前，将要求所有的印度、太平洋国家维持对朝制裁施压，并强调了美国〝先无核化，后缓解制裁〞的基调。
 <br/>
 <br/>
 在9日召开的中美〝2+2外交安全对话〞中，美国国务卿迈克．蓬佩奥向中方再次强调履行联合国的对朝制裁，以实现朝鲜的FFVD（最终、全面且可查证的无核化）。
 <br/>
 <br/>
 12日，韩国《东亚日报》报导称，多位对朝消息人士透露，朝鲜想将无核化作为踏板，却没有取得什么成果，朝鲜内部为此很慌张，很焦躁。
 <br/>
 <br/>
 美国坚持制裁方针，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 对朝鲜民众作出的保证就无法实现，金对此十分恼火。为了挽回金的面子，朝鲜官员们急赴中国大陆，接触中国沈阳的投资者们，吸引他们对朝投资。
 <br/>
 <br/>
 消息人士还透露，朝鲜官员请求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共援助朝鲜.html" target="_blank">
  中共援助朝鲜
 </a>
 物品，也在偷偷的进行。
 <br/>
 <br/>
 美国不主张暂停制裁以开展人道主义援助，美国常驻联合国代表黑利表示：〝过去对朝鲜的人道主义援助没有流向朝鲜居民，而是到了掌权者和政权手里。〞
 <br/>
 <br/>
 她警告说：〝如果人道主义援助不能发挥应有的作用，到不了我们希望的居民手里，我们不会做任何事情。〞
 <br/>
 <br/>
 对于美国与朝鲜之间出现的对峙局面。黑利11月8日在联合国总部对记者表示，〝到目前为止，我们给了很多‘胡萝卜’。我们将不会放下大棒。〞
 <br/>
 <br/>
 她强调，美国与朝鲜进行对话，并中断了韩美联合军事演习，拿出了很多〝胡萝卜〞，但朝鲜没有做出足以解除制裁的事情。
 <br/>
 <br/>
 〝朝鲜核武器和导弹的威胁仍然存在，朝鲜不允许核察人员入境对核武器和弹道导弹设施进行核查。（在这种情况下）我们有必要维持目前的路线。〞
 <br/>
 <br/>
 黑利说，朝鲜仅仅中断核武器和弹道导弹的试验是不够的，如果不允许对相关设施进行核查，美国就不会解除制裁。
 <br/>
 <br/>
 事实上，对于朝鲜弃核问题，外界一直持怀疑态度。韩媒《朝鲜日报》12日报导称，韩国首尔大学原子核工学系的教授许均烈（音译）日前接受采访时表示，朝鲜废除核武器实际上是不可能的。
 <br/>
 <br/>
 首先，现在无核化是不可能实现的，因为朝鲜的核武器太多了，朝鲜现在拥有50至60个核弹头，销毁了10%左右，剩下的大概不会销毁。
 <br/>
 <br/>
 其次，朝鲜拥有优质的铀，朝鲜宣称自己拥有2,000吨铀，在韩国看来朝鲜的铀浓缩设施大概在4,000个左右，美国则认为在10,000个。这意味着，即便朝鲜现在把50个核弹头全销毁，只用3周的时间就可以再造出来新的。
 <br/>
 <br/>
 最后，朝鲜有10,000名从事核开发的人员。这10,000名拥有铀浓缩与再处理经验的人员让无核化难以实现。
 <br/>
 <br/>
 9日，美国副总统迈克・彭斯启程对亚太四国进行巡访之前表示：〝美国明确表示将持续对朝鲜施加史无前例的外交和经济压力。〞〝我们（美国）的决定将迫使那个国家（朝鲜）坐到谈判桌上。〞
 <br/>
 <br/>
 而朝鲜官媒则声称，〝如此的话，就不需要对话了〞，〝朝鲜将恢复经济、核武力并进的路线〞。
 <br/>
 <br/>
 韩媒《东亚日报》报导，随着美朝无核化谈判进展不顺，金正恩访问俄罗斯的日程预计将从本月下旬推迟到明年。有分析称，在美朝关系冷却的情况下，朝鲜和俄罗斯都对举行会谈感到负担。还有人指出，在对朝鲜制裁尚未解除的情况下，经济合作等两国间的议题协调可能遇到了困难。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：范铭）
