### 欧盟首次拒绝成员国预算案 意股债市动荡
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 拒绝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 提交的2019年预算案，这是欧盟首次拒绝成员国的预算案，引发意大利股市、债市再度动荡。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 国家中赤字占国民生产总值比例第二高的国家，上届政府曾向欧盟承诺，年度赤字预算将保持在GDP的0.8%的低位。
 <br/>
 <br/>
 但新上任的政府将明年预算赤字占 GDP 比重上调到 2.4%。
 <br/>
 <br/>
 周二，欧洲委员会副主席杜姆布罗夫斯基斯正式宣布欧盟拒绝了意大利的预算案。
 <br/>
 <br/>
 欧洲委员会副主席杜姆布罗夫斯基斯：〝今天，欧洲委员会第一次要求一个欧元区国家修改其预算草案。我们通过了一项决定，给意大利最多三周时间修订2019年预算草案。〞
 <br/>
 <br/>
 杜姆布罗夫斯基斯还表示，如果意大利在三个星期内不修改预算案，那么欧盟将对意大利进行处罚。
 <br/>
 <br/>
 周一，意大利在发给欧洲委员会的信里表明，知道预算案违反了欧盟规定，但是仍会坚持自己的做法。
 <br/>
 <br/>
 意大利人对自己政府的做法看法不一。
 <br/>
 <br/>
 罗马人德安格利斯赞成欧盟的财政紧缩政策。
 <br/>
 <br/>
 罗马居民德安格利斯：〝意大利是欧洲的一部分，所以得遵循一定的规则，这些规则一直在拯救我们。〞
 <br/>
 <br/>
 商人卡鲁齐则表示支持意大利政府的做法。
 <br/>
 <br/>
 意大利商人卡鲁齐：〝很明显，他们（意大利政府）将继续这个做法。我们需要看看我们有多强大，来保住这种立场。〞
 <br/>
 <br/>
 预算案被拒的消息传出后，意大利国债价值下降，收益率上扬；意大利股票FTSE MIB当天下跌 0.8%。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
