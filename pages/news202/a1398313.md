 香港经济日报
 <br/>
 美国务院周一宣布，国务卿蓬佩奥（Mike Pompeo）与国防部长马蒂斯（Jim Mattis）将于11月9日在华府主持第二轮美中外交安全对话，中方代表为中央外事办主任杨洁篪和国防部长魏凤和等。川普与习近平也在上周通话，同意月底在G20峰会会面。

 美联社
 <br/>
 11月6日，继人脸识别系统之后，中共当局开始在北京和上海街道安装新的监视〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_步态识别.html" target="_blank">
  步态识别
 </a>
 〞（gait recognition）系统，号称可以在50米内透过人们走路的姿势辨识出其身分，即便他们没有露出脸部也是一样。英、美、日研究该技术超过十年，但从未用此技术监视人民。
 <br/>
 <br/>
 大纪元时报
 <br/>
 为期六天的中国国际进口博览会11月5日在上海开幕。上海市公安局如临大敌，大街小巷到处有警察，游客必须通过两次安检。长宁区延安高架路沿线居民楼房窗户被贴上封条，居民被警告不要将头随意探出窗外，以防被狙击手击毙。
 <br/>
 <br/>
 苹果日报
 <br/>
 11月6日，《人民日报》旗下的杂志《新闻战线》总编辑胡欣，被称在报社跳楼自杀，终年66岁。微博有人转发其去世消息，但遭当局瞬间删除。据转述，员工在报社撞见胡欣，见她面色憔悴，没想到她会想不开，死因不明。胡欣北大哲学系毕业，1990年入《人民日报》，2008年底调任《新闻战线》杂志总编。
