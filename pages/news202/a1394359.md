### 印尼强震逾1600人遇难 外国驰援涌入
------------------------

<div class="wysiwyg">
 【新唐人2018年10月06日讯】印尼苏拉威西岛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 过去了一周，死亡数字已经攀升到1649人。一个法国救援专家组，周六在帕卢市市郊开始了挖掘工作，希望找到埋在废墟下的遇难者。
 <br/>
 <br/>
 印尼国家减灾管理局周六表示，已经确认有1649人在苏拉威西岛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 和海啸丧生，其中大部分死于海啸。
 <br/>
 <br/>
 目前，救援人员仍在地震废墟中寻找幸存者，随着不断发现遗体，死亡数字预计将上升。
 <br/>
 <br/>
 周六，一个法国专家救援队伍赶赴受灾最严重的帕卢市郊，在一处废墟开始搜救工作。
 <br/>
 <br/>
 由于地震引发的泥石流和土地塌陷，导致至少1,700户房屋被掩埋，数百人失踪。
 <br/>
 <br/>
 救援队伍负责人 Arnaud Allibert：〝我想在这里我们还能找到遗体，很多遗体仍然埋在土里，由于大量泥沙吞噬了房屋，这是非常复杂的作业。〞
 <br/>
 <br/>
 救援队伍负责人表示，计划移走所有的瓦砾，并在土壤中搜寻遗体，整个作业可能需要四到五个月才能完成。
 <br/>
 <br/>
 印尼副外长法希尔表示，已经收到25个国家和4个外国组织的捐助物资。印尼当局正在致力于将这些物资送到灾区。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
