### 【禁闻】孟晚舟被抓 中美贸易战协议是否受影响
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】中国华为公司副董事长兼首席财务官（CFO）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在加拿大被捕的消息迅速传播，引发极大关注。评论认为，这应该不会影响
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 结果，但是显示出美国要求中共改变作法的决心。
 <br/>
 <br/>
 华为首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被捕的消息在12月5号传出，引发极大关注。她12月1号在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_温哥华.html" target="_blank">
  温哥华
 </a>
 过境时被扣留，当天正值中国国家主席习近平和美国总统川普在阿根廷会晤，双方达成了〝90天缓冲期〞的协议。
 <br/>
 <br/>
 由于投资者担心此事会使美中关系恶化，贸易关系再度紧张，全球股市受到重挫。
 <br/>
 <br/>
 《路透社》12月7号引述两位美国官员的话说，川普对逮捕孟晚舟的计画并不知情。这一表态显然是不想让此事影响美中的重要贸易协商。
 <br/>
 <br/>
 而中共商务部新闻发言人在5号也打破沈默，声称将尽快落实已经和美国达成共识的具体事项。6号中共商务部再度表示〝有信心在90天内与美国达成协议〞。川普7号发推特说〝我同意！〞。
 <br/>
 <br/>
 旅美时事评论员蓝述认为，孟晚舟的案件应该不会影响
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 结果。
 <br/>
 <br/>
 旅美时事评论员蓝述：〝孟晚舟即使被引渡了，将来怎么处理，我想都不会影响中美之间在川习会达成暂时性的妥协吧。对华为的调查也不是从今天才开始的。北京方面面对美国的要求，它需要做的决定远比一个华为公司要大得多得多。如果中方答应这个要求的话，它将意味中国经济模式的一个整体上的变化。〞
 <br/>
 <br/>
 不过评论人士认为，孟晚舟被抓的确释放出强烈信号。
 <br/>
 <br/>
 旅美时事评论员夏小强：〝和之前的中兴不同。美国对中兴的制裁停留在行政制裁方面，那么对华为的这种制裁已经上升到了一个刑事指控。所以说，美国向中共发出一个强烈的信号：要中共停止这种间谍活动，停止这种盗窃知识产权的活动，并且停止支持伊朗、朝鲜这些流氓国家。〞
 <br/>
 <br/>
 《美国之音》12月7号说，孟晚舟被抓后，白宫高级官员仍旧在美中贸易谈判上持强硬立场。白宫贸易与经济顾问纳瓦罗说，川普政府正在要求中国做出根本性的改变。
 <br/>
 <br/>
 同时，孟晚舟被抓原因也被进一步深挖。
 <br/>
 <br/>
 《华尔街日报》7号援引知情人士的消息说，一位负责评估〝汇丰控股有限公司〞近几年反洗钱和制裁控制举措的监督人，向纽约东区联邦检察官举报，华为的帐户存在可疑交易。美国引渡孟晚舟，让她在东区联邦法院出庭。
 <br/>
 <br/>
 《路透社》2013年曾报导，孟晚舟2007年担任由华为母公司控股的一家管理企业的公司秘书。这家管理企业持有香港注册公司Skycom Tech的全数股份。2008年2月到2009年4月，孟晚舟又在Skycom的董事会任职，她与Skycom在过去10年间有过大量财务及其它方面的关联。而Skycom在伊朗有业务，他们在伊朗的华人员工佩戴华为工牌或有华为的名片。Skycom曾试图向伊朗最大的手机运营商出售遭禁运的惠普电脑设备。
 <br/>
 <br/>
 美国商务部，财政部和司法部都在对华为进行调查，这很可能是受中兴的牵连。在此之前，美国调查中兴通讯违反对伊朗的禁售令时，发现中兴在内部机密文件中，记录了代号为〝F7〞的公司如何逃避管制，向禁运国出口美国敏感外构件。
 <br/>
 <br/>
 由于华为(HuaWei)的英文缩写H.W.，类似于英文单词丈夫和妻子（Husband与Wife）的缩写，大陆业界惯用〝夫妻〞的谐音〝F7〞作为华为的代号。
 <br/>
 <br/>
 中兴的文件还说，〝F7〞在2010年收购美国公司3Leaf的计画因影响美国国家安全被美国政府否定。而当年被华盛顿反对收购3Leaf的公司正是华为。
 <br/>
 <br/>
 美国已经警告盟友，使用华为设备可能带来潜在安全风险。最新消息是，日本也将在下周修订关于政府采购的内部规则，实质上禁止本国政府向中国华为和中兴通讯采购电信产品。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/陈建铭
