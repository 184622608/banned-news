### 【禁闻】德国城市展示法轮功反迫害大型海报
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】
 <b>
  德国城市展示
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
   法轮功
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反迫害.html" target="_blank">
   反迫害
  </a>
  大型
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海报.html" target="_blank">
   海报
  </a>
 </b>
 <br/>
 <br/>
 为庆祝〝人权之路〞建成25周年，德国中部城市纽伦堡从10月22号起在市区展示25个关于人权的大型广告牌，一幅
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反迫害.html" target="_blank">
  反迫害
 </a>
 的大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海报.html" target="_blank">
  海报
 </a>
 也被立在繁忙的车道边。
 <br/>
 <br/>
 海报主体是一张大照片，展示多名法轮功修炼者在北京天安门广场请愿，打出写着中文〝真、善、忍〞的横幅；一名中共警察，手上还拿着从其他修炼者手中夺下的横幅。
 <br/>
 <br/>
 大照片上用德语写着：〝19年来，法轮功修炼者在中国没有人权〞。
 <br/>
 <br/>
 海报的下方还有三张小照片：一张是一名法轮功学员被两名便衣按在地上，其中一名便衣用鞋踩在法轮功学员的脸上；一张是一名女法轮功学员被两名穿制服的警察抓住，双手被警察反扣在背后，嘴被另一名警察捂住。第三张是一位母亲抱着孩子，手上拿着一家三口的照片。当地法轮功学员表示，小孩的父亲因修炼法轮功被中共迫害致死。
 <br/>
 <br/>
 海报主图和小照片之间的德文说：〝佛家气功法门——法轮功因其原则〝真、善、忍〞而遭受迫害。〞
 <br/>
 <br/>
 据〝大纪元新闻网〞报导，这些海报将在纽伦堡展示10天。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康奈尔大学.html" target="_blank">
   康奈尔大学
  </a>
  停止与
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人民大学.html" target="_blank">
   中国人民大学
  </a>
  合作
 </b>
 <br/>
 <br/>
 美国常春藤学府康奈尔大学日前宣布中断与中国大陆的中国人民大学合作，原因是不满该大学压制言论和学术自由，惩罚支持深圳劳工权利的学生。
 <br/>
 <br/>
 英国《金融时报》报导说，康奈尔暂停的项目包括两个学术交流和一个研究项目。这是近年来国外大学与中国大学之间中断学术合作的首例。
 <br/>
 <br/>
 康奈尔大学工业及劳工关系学院国际项目主任弗莱德曼表示，中国人民大学禁止学生谈论劳工问题，严重侵犯学术自由，令双方的合作研究无法进行。
 <br/>
 <br/>
 他表示，外国大学应当重新评估与中国大学的合作情况。
 <br/>
 <br/>
 深圳佳士科技的部分工人今年7月下旬发起要求组建工会的抗争行动，人民大学的一些学生参与声援，其中哲学院本科生杨舒涵因此遭到当局威胁，返校后被学校要求休学一年。另一位本科生张子涵也因同样原因被老师当着全体同学的面批评。同时，该大学还威胁取消一个有关工人权益的学生社团的登记注册。
 <br/>
 <br/>
 编辑/周玉林
