### 贸战停火 谈判启动  刘鹤访美面临棘手难题
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】美国总统川普和中国国家主席习近平在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会期间的会晤，促使中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 暂时停火90天。期间双方将继续谈判，如若不能就知识产权、科技转移等议题取得进展，华府将恢复加征
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 至25%的决定。有消息披露，中共副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 将率团前往美国，重开谈判。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞在12月1日落幕。中共外长王毅在记者会上说，这次会晤达成重要共识，两国将停止互相加征新的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，中方愿意扩大从美国进口，逐步缓解贸易不平衡的问题。
 <br/>
 <br/>
 白宫方面则发表声明说，川普和习近平同意迅速开始关于强迫技术转让、知识产权保护、非关税壁垒、网路入侵和偷窃、服务业和农业方面结构性改变的谈判。双方同意努力在未来90天里完成这项工作。如果在90天后双方仍无法达成协议，关税比例将从10%提升至25%。
 <br/>
 <br/>
 香港《经济日报》分析，对于〝川习会〞的成果，中共强调双方将停止加征新关税，而白宫声明则披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 只是暂时停火，反映中美仅为未来的谈判确定框架，仍无法全面化解彼此贸易分歧。
 <br/>
 <br/>
 荣鼎谘询（Rhodium Group）公司的中国问题专家康索那利（Nicholas Consonery）认为，川习会后的一个关键观察点是〝中方愿意走多远，以及川普政府（的对华立场）将坚持到多大程度。〞
 <br/>
 <br/>
 康索那利说，〝我们现在要观察中国（中共）政府，是否会认真地推进结构性经济改革，不过依过去的历史来看，我们不会被带到非常乐观的结局。〞
 <br/>
 <br/>
 日本《产经新闻》的报导则说，美国对中共的贸易制裁举措，目的不仅仅是要缩小对华贸易逆差，更注重遏制中共主导的制造技术提高、打击中共猖獗的知识产权侵犯、以及无孔不入的黑客攻击。而这些，几乎是中共发展的核心手段，不会轻易放弃。
 <br/>
 <br/>
 〝川习会〞前，英国《金融时报》曾引述三位知情人士透露，如果达成停战协议，北京首席贸易谈判代表、中共副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 暂定于12月12日至15日带领30名中方官员前往华府，与美国官方进行贸易谈判。
 <br/>
 <br/>
 此前11月27日习近平离开北京出访时的团队名单中，并没有刘鹤。当时，他正在德国出席第八届中欧论坛汉堡峰会。不过在此次的〝川习会〞中，刘鹤却意外现身。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：祝馨睿）
 <br/>
