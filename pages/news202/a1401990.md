### 蓬佩奥回击欧洲盟国：批评川普完全错误
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】周二（12月4日），美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 （Mike Pompeo）驳斥了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧洲盟国.html" target="_blank">
  欧洲盟国
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_批评.html" target="_blank">
  批评
 </a>
 。这些国家声称，川普的〝美国优先〞立场正危及全球舞台上久已建立的国际组织。
 <br/>
 <br/>
 周二，正在比利时首都布鲁塞尔出席北约（NATO）会议的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 在会前表示，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统的领导下，美国并没有放弃国际领导地位和国际体系中的朋友，而实际上恰恰相反，破坏国际秩序的正是那些川普的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_批评.html" target="_blank">
  批评
 </a>
 者。
 <br/>
 <br/>
 蓬佩奥指出，来自伊朗和中共的批评者指责美国是造成世界体系崩溃的原因，声称美国正以〝单边〞行为取代〝多边〞行动，听起来似乎每种多边行动都是可取的。
 <br/>
 <br/>
 蓬佩奥表示，甚至美国的欧洲盟友有时也说，美国的行事不符合自由世界的利益，但这种说法是完全
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_错误.html" target="_blank">
  错误
 </a>
 的。
 <br/>
 <br/>
 蓬佩奥认为，川普上任近2年了，这个世界应该知道，川普政府的使命就是〝重申我们改革自由国际秩序的主权〞，而不是打破国际秩序。
 <br/>
 <br/>
 〝我们希望我们的朋友帮助我们并同样行使他们的主权〞，蓬佩奥表示，〝我们渴望让国际秩序服务于我们的公民〞。
 <br/>
 <br/>
 欧洲当前非常关注川普提出的分摊北约经费问题。蓬佩奥指出，欧盟是美国最大的贸易伙伴，欧洲的繁荣也使美国从中受益。
 <br/>
 <br/>
 蓬佩奥在北约会议上做了20分钟的发言，他在演讲中表示，每个国家都必须诚实地承认其对本国公民的责任，并问问当前的国际秩序是否在尽力服务于其人民的利益。
 <br/>
 <br/>
 蓬佩奥指出，〝如果不能，我们就必须要问问我们怎样才能纠正它〞，而这就是川普正在做的事情——他正让美国回归其在世界上的传统领导角色。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
</div>
