### 脱欧草案引内阁辞职潮 英首相再陷窘局
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国首相.html" target="_blank">
  英国首相
 </a>
 特雷莎・梅周三在5小时的马拉松式
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内阁.html" target="_blank">
  内阁
 </a>
 会议后宣布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 得到内阁支持。话音未落，两名大臣提出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 ，一些其它级别的政府官员也加入了辞职潮。首相梅面对本党议员的弹劾，有可能相位不保。
 <br/>
 <br/>
 英国脱欧事务大臣 拉布：我觉得（首相）与欧盟商定的协议有两个致命缺陷。
 <br/>
 <br/>
 拉布4个月前出任脱欧事务大臣，其前任戴维斯也是因抗议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 。
 <br/>
 <br/>
 拉布认为，目前的脱欧草案会威胁到英国主权完整，而且将英国锁定在关税同盟中，不知何时脱身。
 <br/>
 <br/>
 随后，就业与养老事务大臣麦克维，北爱尔兰事务国务大臣等也相继加入辞职潮。
 <br/>
 <br/>
 这导致英镑周四重贬近2%。
 <br/>
 <br/>
 首相梅则回应，可以理解辞职的官员，但她坚持立场。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国首相.html" target="_blank">
  英国首相
 </a>
 特雷莎・梅：但是我非常确信，我所提出的脱欧方案，对我们的国家和全体人民来说，都是正确的。
 <br/>
 <br/>
 周四，首相向下议院解释脱欧草案时，保守党议员里斯-莫格指出首相之前承诺的，和草案内容相悖。他表示将提交对首相的不信任信件。
 <br/>
 <br/>
 保守党议员 里斯-莫格：监于我尊敬的朋友（首相）所说的，和我尊敬的朋友（首相）所做的不一致，我难道不应向Altrincham &amp; Sale West选区议员布雷迪提交(不信任)信件吗？
 <br/>
 <br/>
 首相梅是保守党成员，如果自家阵营中有48位议员提交不信任信件，下议院就可发起对首相的不信任案投票。如果半数议员支持不信任案，首相梅将下台。
 <br/>
 <br/>
 现在离英国脱欧只有四个多月了，保守党内如果弹劾首相，现在是最后时机，否则新首相没有时间和欧盟达成新的协议，就会面临无协议脱欧的风险。还有另外一个可能，就是留欧派呼吁的二次脱欧公投，现在越来越多的政要和民众呼吁。如果到了完全没有希望达成协议的时候，也许英国政府就会松口进行二次公投了。
 <br/>
 <br/>
 新唐人电视综合报导
