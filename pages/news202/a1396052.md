### 炸沉中国非法渔船 印尼女部长炮轰中共〝跨国犯罪〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】印度尼西亚海事暨渔业部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏西.html" target="_blank">
  苏西
 </a>
 ，日前在接受媒体采访时，指中共是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_跨国犯罪.html" target="_blank">
  跨国犯罪
 </a>
 〞，并称来自中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法渔船.html" target="_blank">
  非法渔船
 </a>
 经常出现在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 海域。
 <br/>
 <br/>
 周四（18日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏西.html" target="_blank">
  苏西
 </a>
 （Susi Pudjiastuti）在雅加达举行新闻发布会，并向中共发出了强硬信息。她在接受香港《南华早报》采访时表示，〝他们（中国渔船）正在做的不是捕鱼，而是跨国组织犯罪。〞〝中共拥有世界上最大的捕捞船队。〞
 <br/>
 <br/>
 她说：〝在非法、无管制和未经报告的捕捞问题上，我们曾与中共发生过多次分歧，但他们仍然不同意将这些行为归类为‘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_跨国犯罪.html" target="_blank">
  跨国犯罪
 </a>
 ’。〞
 <br/>
 <br/>
 同时，她补充说：〝如果没有国际合作，我们将无法与之（中共）抗争。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 渔业部官员玛斯玛斯（Mas Achmad Santosa）也表示，渔业跨国组织犯罪破坏环境，也破坏人权，是对海洋资源的最大威胁之一。
 <br/>
 <br/>
 苏西于2014年加入印尼内阁，任职海事暨渔业部长。她上任后，对非法闯入该国海域的境外渔船采取强硬做法，并下令将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法渔船.html" target="_blank">
  非法渔船
 </a>
 直接炸掉。此举虽受到各界抨击，但苏西仍受印尼民众喜爱，且民调始终居高不下。
 <br/>
 <br/>
 自苏西上任至今，印尼已
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸沉.html" target="_blank">
  炸沉
 </a>
 448艘涉嫌非法捕捞的外国渔船，其中包括中国渔船。苏西表示，炸渔船的目的在于对试图在印尼海域
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法捕鱼.html" target="_blank">
  非法捕鱼
 </a>
 和非法开发其它资源的外国渔船，发出警告和威吓。
 <br/>
 <br/>
 今年53岁的苏西小时候却家境贫寒。为了帮助家里改善经济，苏西不得不在高中时辍学，开始专心从事渔获贸易仲介；虽然后来没有继续在校园深造，不过，她从未停止在职场上的学习。
 <br/>
 <br/>
 她最早的工作是在西爪哇滨海小镇潘甘达朗（Pangandaran）的渔市场卖鱼。
 <br/>
 <br/>
 18岁时，苏西卖了一些珠宝，赚到75万印尼盾，并开始投资渔获仲介的生意。她于1996年成立了自己的渔获加工厂，也建立了自身的龙虾产品品牌。
 <br/>
 <br/>
 靠卖鱼起家的苏西经过多年奋斗，后来摇身一变成为印尼渔业产品出口商以及苏西航空公司（Susi Air）总裁。2012年，其航空公司已有32架各式飞机，并雇用180名机师。
 <br/>
 <br/>
 从卖鱼起家到成为航空公司总裁，再到加入内阁，掌理印尼重返海洋大国政策，苏西奋斗的传奇人生为人称颂。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：秦锋）
</div>
