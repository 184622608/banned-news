### 多种传染病泛滥 数千大篷车移民在墨治疗
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】根据来自美墨边境的报导，目前滞留在边境的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 中，有数千人正在接受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传染病.html" target="_blank">
  传染病
 </a>
 治疗，所患疾病包括普通的呼吸道传染、肺结核甚至艾滋病。不良的卫生条件还有可能导致更多疫情爆发。
 <br/>
 <br/>
 据美国保守派网站〝每日电讯〞（The Daily Caller）11月29日报导，一直在报导
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 情况的福克斯新闻（Fox News）记者詹金斯（Griff Jenkins）从边境发来消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 蒂华纳市（Tijuana）的一名卫生部门发言人确认，他们已经给2,000余名移民实施了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传染病.html" target="_blank">
  传染病
 </a>
 治疗。
 <br/>
 <br/>
 詹金斯报导说，蒂华纳市已经为2,267名中美洲移民实施了治疗，其中60%为呼吸道传染，3人被确认患有肺结核，4例艾滋病，101例头蝨感染，4例水痘，而且爆发肝炎的隐患很大，因为6,000来名移民挤在原本只能容纳1,000人的空间里，卫生条件很差。
 <br/>
 <br/>
 受收费情况影响，一些大篷车移民开始转而攻击移民组织者。一名寻求庇护的洪都拉斯移民表示，他感觉自己从一开始就被组织者骗了，移民们都快崩溃了。
 <br/>
 <br/>
 将近一个月前，美国深夜新闻讽刺类电视节目《每日秀》（The Daily Show）主持人特雷弗．诺亚（Trevor Noah）曾嘲笑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ，因为川普声称大篷车中有很多〝强壮而危险的人〞，而且还充斥着疾病。
 <br/>
