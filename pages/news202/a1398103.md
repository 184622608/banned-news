### 解决人力短缺 澳〝打工度假〞签证肯务农可留3年
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲农业.html" target="_blank">
  澳洲农业
 </a>
 界曾抱怨收成季人力短缺严重，尤其在农业省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_昆士兰.html" target="_blank">
  昆士兰
 </a>
 ，因而引发政策改变。澳洲总理莫里森5日宣布，政府将延长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_打工度假.html" target="_blank">
  打工度假
 </a>
 签证期限到第3年，协助纾解澳洲农场人力短缺问题。
 <br/>
 <br/>
 综合媒体报导，原本一年期的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_打工度假.html" target="_blank">
  打工度假
 </a>
 〞签证，允许旅行者在澳洲多停留一年，条件是必须到偏远的北领地（Northern Territory）打工。
 <br/>
 <br/>
 自明年7月起，这些旅行者的签证还能延长到第3年，只要他们有6个月时间在人力短缺特别严重的农业区打工。
 <br/>
 <br/>
 此前为了减少移民，澳洲保守派政府自2017年以来持续缩限临时工作签证。但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲农业.html" target="_blank">
  澳洲农业
 </a>
 界曾抱怨收成季人力短缺严重，尤其在农业省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_昆士兰.html" target="_blank">
  昆士兰
 </a>
 。
 <br/>
 <br/>
 2017至2018年间，澳洲共核发超过20万份打工度假签证。在45国有资格申请这类签证的国民当中，英国、德国和法国申请者占最多。
 <br/>
 <br/>
 澳洲总理莫里森（Scott Morrison）今天访问东部昆士兰省（Queensland）的农业社区时，宣布这项政策改变。
 <br/>
 <br/>
 莫里森的联合政府明年5月必须面临大选考验，昆士兰省是他们的重要战场。
 <br/>
 <br/>
 不过，据新南威尔士大学及悉尼科技大学于上周公布的联合调查结果，在〝打工度假〞的外劳中，有三分之一人的时薪不足12澳元（约8.7美元），仅为最低工资的一半。其中又以蔬果采集业的情况最低，有一成半外劳时薪仅5澳元（约3.6美元）。
 <br/>
 <br/>
