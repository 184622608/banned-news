### 俄情报头子科罗博夫〝长期重病〞过世 享寿62
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 国防部消息，俄罗斯军事情报局长科罗博夫（Igor Korobov）21日因〝长期重病〞过世，享寿62岁。
 <br/>
 <br/>
 俄国防部未就科罗博夫的死讯提供进一步细节。只称，他是个〝了不起的人，忠诚的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 子民，也是热情的爱国者〞。
 <br/>
 <br/>
 科罗博夫于2016年担任反恐执行局主席，并曾指挥俄罗斯军事情报局(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_GRU.html" target="_blank">
  GRU
 </a>
 )打击叙利亚境内的恐怖主义部队。2017年，他被授予国家最高荣誉〝俄罗斯联邦英雄〞称号。
 <br/>
 <br/>
 此前，英国指控俄国特工企图在索尔兹伯里市（Salisbury），以神经毒剂诺维乔克（Novichok）毒害前俄罗斯间谍斯克里帕尔（Sergei Skripal）和他的女儿尤莉雅（Yulia Skripal）。
 <br/>
 <br/>
 西方国家据此将一连串鲁莽攻击归咎于GRU。荷兰政府确信GRU企图骇入禁止化学武器组织（OPCW）总部。美国情治单位则表示，GRU幕后操弄美国2016年总统大选。但俄罗斯全盘否认上述指控。
 <br/>
 <br/>
 科罗博夫和其他几名GRU官员因〝代表俄国政府从事破坏网路安全活动〞，同遭美国制裁。
 <br/>
 <br/>
