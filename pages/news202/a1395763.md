### 川普孤立中共再出重拳 美宣布与欧日英谈新贸易协议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】美国贸易代表莱特希泽周二通知国会，美国政府将分别向英国、欧盟、日本寻求签订贸易协议，以重新平衡全球贸易。舆论认为，美国与英欧日一旦达成新的贸易协议，中共政府将更加被孤立，必将面临更为严峻的挑战。
 <br/>
 <br/>
 10月16日，莱特希泽（Robert Lighthizer）对外界透露上述消息时表示，这个决定是在美国总统川普（特朗普）的指示下作出的，美国政府希望通过与日本、欧盟和英国谈判贸易协定来扩大贸易和投资。莱特希泽说： 〝我们将致力于为美国工人、农民、牧场主和企业及时完成谈判，并达成实质性结果。〞
 <br/>
 <br/>
 莱特希泽进一步表示，与日本和欧盟的贸易谈判，计划于2019年1月开始分别进行，这次美国政府将致力于改变过去美国与日本及欧盟贸易的长期不平衡状态，解决美国出口商饱受的关税及其他非关税壁垒的困扰。2019年英国脱欧后，华府也将与其进行贸易谈判，期望达致〝自由、公平、互惠〞的新的贸易协议。
 <br/>
 <br/>
 此前有消息称，美国副总统彭斯与日本副首相麻生太郎将于今年11月中旬在日本举行会谈，为莱特希泽和日本经济振兴部长茂木敏充之间开展进一步贸易谈判铺平道路。
 <br/>
 <br/>
 根据美国《贸易促进授权法》（TPA）的规定，美国贸易代表办公室（USTR）计划与其他国家政府进行谈判后，需要提前90天通知美国国会，并与国会保持磋商，以确保USTR在参考国会意见下展开谈判的立场。此外，USTR还将在联邦公报中发布通知，征求公众就贸易谈判方向、重点和内容提出建议。
 <br/>
 <br/>
 有舆论认为，如果美国政府的上述计划顺利达成，将会增加美国与欧洲及日本、英国之间的进出口交易，而中共政府面临的挑战将会变得更严峻。
 <br/>
 <br/>
 事实上，早在今年10月4日白宫首席经济顾问库德洛就曾对外表示，美国正在与其它主要经济体联合，组成贸易联盟，以应对中共不公平贸易行为，例如倾销、网络盗窃和强制技术转让等。
 <br/>
 <br/>
 美国驻欧盟大使桑德兰（Gordon Sondland）在比利时布鲁塞尔告诉记者，一旦美国与欧盟达成适当的贸易协议，就可以形成联合同盟来对抗中共。
 <br/>
 <br/>
 前不久，美国政府先后与墨西哥以及加拿大政府达成新的贸易协议后，美国商务部长罗斯曾对外界透露说，在刚刚签署的美墨加贸易协定（USMCA）中有一个〝毒丸条款〞，是一项〝堵住贸易协定中漏洞的举措〞，可以有效防止中国商品借道加墨流向美国，削弱美对中的贸易壁垒。
 <br/>
 <br/>
 按照这个条款的规定，这个协议中的任何一成员国如果与〝非市场经济国家〞达成自由贸易协议，则其它成员国可以在六个月后退出并建立其自己的双边贸易协定。
 <br/>
 <br/>
 由于中国一直未获得国际社会承认其〝市场经济国家〞地位，这意味着如果加拿大和墨西哥想和中共政府达成自贸协定，就要承担与美国自贸协定破裂的风险。
 <br/>
 <br/>
 有舆论认为，上述〝毒丸〞条款将成为美国在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 中祭出的重型武器，有利于在全球贸易格局中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孤立中共.html" target="_blank">
  孤立中共
 </a>
 政府。
 <br/>
 <br/>
 路透社曾在相关报导中分析称，〝毒丸〞条款实际上赋予了华盛顿对加拿大和墨西哥与其他国家签订的自贸协定的否决权，以确保这些自贸伙伴国是市场经济体，而且国家没有在经济中占据主导地位，后者是川普政府对中国发动关税战的核心诉求。
 <br/>
 <br/>
 香港中文大学助理教授胡荣向BBC中文表示，与特朗普政府正在进行的贸易谈判还有日本和欧盟，针对中国的〝毒丸〞条款很有可能也会摆上谈判桌。不排除为了保持对美国市场的进入和维持贸易关系，日欧也会和美国达成类似协议，这样自然会对中国在全球贸易和供应链中的地位造成削弱。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
