### 三菱解除戈恩董事长职务
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】日产汽车公司前董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戈恩.html" target="_blank">
  戈恩
 </a>
 ，因涉嫌少报薪资等行为遭日本警方逮捕，日产随后解除了他的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_董事长职务.html" target="_blank">
  董事长职务
 </a>
 。周一，日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三菱.html" target="_blank">
  三菱
 </a>
 汽车董事会也宣布，解除戈恩兼任的董事长职务。
 <br/>
 <br/>
 周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三菱.html" target="_blank">
  三菱
 </a>
 汽车总裁益子修宣布，董事会决定解除
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戈恩.html" target="_blank">
  戈恩
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_董事长职务.html" target="_blank">
  董事长职务
 </a>
 。
 <br/>
 <br/>
 三菱汽车总裁益子修：〝董事会一致通过提议，解除戈恩的公司董事长职务。〞
 <br/>
 <br/>
 戈恩被控多年少报薪资4400万美元，日前被日产汽车解职。日产表示，戈恩还可能涉嫌不实陈述公司投资支出，将公司资产挪为己用。
 <br/>
 <br/>
 戈恩涉嫌违反日本金融商品交易法，已被警方逮捕。
 <br/>
 <br/>
 据悉，戈恩的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雷诺汽车.html" target="_blank">
  雷诺汽车
 </a>
 行政总裁职务，暂时由他人代理。
 <br/>
 <br/>
 戈恩打造了雷诺-日产-三菱汽车联盟，他被解职后，外界担心，是否影响三家公司未来合作关系。
 <br/>
 <br/>
 新唐人记者易文、张琪综合报导
</div>
