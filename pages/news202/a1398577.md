### 提问粗鲁 CNN记者通行证被白宫吊销
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 结束后，川普总统在周三召开记者会。美国有线电视新闻网（CNN）驻白宫记者阿科斯达，穷追不舍的提问，拒绝放下发问的话筒，并于前来收走话筒的女职员发生肢体冲突。当晚，白宫宣布取消了阿科斯达的采访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通行证.html" target="_blank">
  通行证
 </a>
 ，来看详情。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN记者.html" target="_blank">
  CNN记者
 </a>
 阿科斯就大篷车难民问题质问川普，两人都坚持己见。
 <br/>
 <br/>
 CNN记者阿科斯达：〝你为什么把它描绘成入侵者.....？〞
 <br/>
 <br/>
 美国总统川普：〝因为我认为这是入侵，我们观点不同。〞
 <br/>
 <br/>
 CNN记者  阿科斯达：〝你是否认为妖魔化移民.....？〞
 <br/>
 <br/>
 美国总统川普：〝不，不，我希望他们来，但他们要合法地进来，你知道他们必须按照程序进来。我要走程序，我要让人们进来。我们需要人。等等，你知道我们为什么需要人吗？因为我们有数百公司搬进来。我们需要人。〞
 <br/>
 <br/>
 CNN记者阿科斯达：〝但是他们还在千里之外呢。〞
 <br/>
 <br/>
 美国总统川普：〝坦率地说，你应该让我来治理国家，你管CNN，如果你做得好，你们的评级会好得多。〞
 <br/>
 <br/>
 随后，阿科斯不顾川普要求其坐下，紧握麦克风，继续提问。
 <br/>
 <br/>
 美国总统川普：〝有你这种员工，CNN应该惭愧。你是个粗鲁有糟糕的人，CNN不该雇傭你。〞
 <br/>
 <br/>
 当晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫发言人.html" target="_blank">
  白宫发言人
 </a>
 桑德斯发表声明宣布，白宫取消阿科斯达的记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通行证.html" target="_blank">
  通行证
 </a>
 。目前还不清楚白宫是否核发其他通行证给他。
 <br/>
 <br/>
 新唐人记者易文、宇微综合报导
</div>
