### 员工罹癌生怪病10年争议 三星半导体道歉并赔偿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三星电子.html" target="_blank">
  三星电子
 </a>
 （Samsung Electronics）旗下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_LCD工厂.html" target="_blank">
  LCD工厂
 </a>
 因忽视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_员工健康.html" target="_blank">
  员工健康
 </a>
 防护，导致多人罹患癌症或罕见疾病，经过10年的缠讼与抗争，三星公司23日终于正式公开道歉。
 <br/>
 <br/>
 综合媒体报导，劳工权益团体表示，约有240人在三星工厂服务期间因为工作关系罹患各种疾病，包括16种癌症、罕见疾病等，其中80人不幸死亡。另外还有员工饱受流产之苦，或是产下有先天疾病的孩子。
 <br/>
 <br/>
 在2007年痛失爱女黄于宓的黄相基（Hwang Sang-gi），带领半导体工人组织、对抗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三星电子.html" target="_blank">
  三星电子
 </a>
 。据悉，黄于宓是2003年进入三星半导体工作，4年后却因白血病过世，年仅22岁，黄尚吉替女儿打官司，获得韩国最高法院判定是职灾，但同一起官司的其他3名原告却败诉。
 <br/>
 <br/>
 黄相基和其他家属在过去10年间没有放弃，他们长期在三星电子总部大楼外面举牌抗议，辛苦的争取，终于在今年7月得到回报。
 <br/>
 <br/>
 双方在调停委员会仲裁下，直至本月稍早，三星才和受害者达成协议，愿意赔偿每人最高达1.5亿韩元（约14.2万美元）。
 <br/>
 <br/>
 三星电子并在今天公开致歉，并承诺会履行公布的所有赔偿细项。三星董事长兼执行长金奇南（Kim Ki-nam，音译）发声明表示：〝我们对那些罹患疾病的员工和家属表达诚挚歉意。我们未能对半导体和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_LCD工厂.html" target="_blank">
  LCD工厂
 </a>
 员工作出完善的健康防护。〞
 <br/>
 <br/>
 受害人家属联盟负责人黄相基说：〝三星的道歉对受害者家属来说根本不够，但我们愿意接受。〞黄相基的22岁女儿已因罹患白血病不幸死亡。
