### 印度高速列车冲撞看烟火人群 至少61死72伤(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度.html" target="_blank">
  印度
 </a>
 北部一列高速行驶的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火车.html" target="_blank">
  火车
 </a>
 周五（19日）撞击聚集在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿姆利则.html" target="_blank">
  阿姆利则
 </a>
 市观看节庆烟火秀的人群，至少造成61人死亡，72人受伤。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度.html" target="_blank">
  印度
 </a>
 时报报导，印度官员说，周五，人群聚集在旁遮普邦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿姆利则.html" target="_blank">
  阿姆利则
 </a>
 （Amritsar）郊外的铁轨上观看烟火，庆祝印度教的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十胜节.html" target="_blank">
  十胜节
 </a>
 〞（Dussehra）。
 <br/>
 <br/>
 事发后不久，警方称，死亡人数已上升到61人，另有72人受伤者被送入多所医院，其中有数人重伤。
 <br/>
 <br/>
 警方在现场告诉记者，施放烟火时声响很大，可能因此民众没有听见
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火车.html" target="_blank">
  火车
 </a>
 疾驶而来的声音。但一些目击者表示，列车在高速通过宗教庆祝场地之际并没有鸣笛。
 <br/>
 <br/>
 当地政府官员说，〝十胜节〞庆祝活动每年都在阿姆利则郊外举行，他们已通知铁路部门要让列车低速行驶。但从视频中可看见，黑夜里烟火如火柱般往上直窜，冷不防的一列火车呼啸而过，民众根本来不及反映。
 <br/>
 <br/>
 官员说，事发后，铁路运输终止了几个小时，阿姆利则开往孟买金庙，以及阿姆利则开往德拉敦的火车都被困在阿姆利则附近。
 <br/>
 <br/>
 印度总理莫迪称这起事故是〝令人心碎的悲剧〞，并要求官员立即救助受伤民众。
 <br/>
 <br/>
 火车是印度主要旅行交通工具，铁路网遍布各地。根据2012年政府统计报告，印度每年有将近1万5000人死于铁路
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交通事故.html" target="_blank">
  交通事故
 </a>
 。不过，虽然列车事故发生率相对频繁，但周五发生的是近年来最惨重的事故之一。
 <br/>
 <br/>
 旁遮普省（Punjab）省长辛赫（Amarinder Singh）已下令调查这起事故发生的原因。
