### 中联部长口出狂言 要求日本政府〝领导〞媒体
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】中国媒体是中共的宣传工具，而在自由社会，媒体则是监督政府的独立社会力量。日前，中共中联部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宋涛.html" target="_blank">
  宋涛
 </a>
 却公开要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本政府.html" target="_blank">
  日本政府
 </a>
 做日本媒体的〝领导者〞，引导〝中日友好的方向〞。台媒中央社称，宋涛〝语出惊人〞。
 <br/>
 <br/>
 10月10日，日中执政党交流机制会议在日本北海道开幕。该会议属于中国共产党与日本执政自民、公明两党的定期对话。
 <br/>
 <br/>
 中央社报导，中共中央对外联络部（中联部）部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宋涛.html" target="_blank">
  宋涛
 </a>
 在会议上发表演讲称，中日友好是〝正确的战略和方向〞，并希望日本能够加入中共〝一带一路〞倡议。
 <br/>
 <br/>
 宋涛强调，为了中日关系朝友好方向发展，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本政府.html" target="_blank">
  日本政府
 </a>
 〝对日本媒体的管制非常有必要〞，应成为媒体的〝领导者〞，〝引导舆论形成〞，让媒体〝客观理性地〞报导中国的资讯。
 <br/>
 <br/>
 〝笔杆子〞一直是中共的统治工具之一。数十年来，中共都在操控媒体欺骗民众，为官方内外政策造势宣传。宋涛如此自信地向日本自民、公明两党〝介绍经验〞，显示他作为中共党务官员，在这方面颇有心得。
 <br/>
 <br/>
 不过，如果要把中共这套特有的手段〝推销〞给民主社会，估计会遇到难以逾越的障碍。
 <br/>
 <br/>
 中央社称宋涛〝语出惊人〞，说明在自由社会，人们对本国政府〝领导〞媒体的做法闻所未闻、想都未曾想。
 <br/>
 <br/>
 在自由社会，虽然也有政客用权势和利益左右媒体的现象，但是在民主框架下，社会媒体是独立于政府的力量，监督和批评政府是媒体的责任。
 <br/>
 <br/>
 即便是已在中共虎口下的香港，虽然北京通过威吓和收买掌控了多数媒体的舆论导向，但是依然无权完全控制这些媒体的所有报导，而且仍有一些港媒坚持对北京提出严厉批评。
 <br/>
 <br/>
 在美国，除了《美国之音》等少数几个用于对外宣传的官方网站之外，政府不被允许运营面向国内的媒体。
 <br/>
 <br/>
 目前，多数美国媒体受左派思潮影响，都对川普（特朗普）政府持反对态度，甚至时常发出一些刻意〝制造〞的新闻攻击川普，被川普多次斥责为假新闻。但即便如此，川普仍未动用权力关闭任何一家媒体。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
