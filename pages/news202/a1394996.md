 德国之声
 <br/>
 10月9日，中国新疆人大常委员会公布修改后的〝新疆维吾尔自治区去极端化条例〞，将外界关注的〝再教育营〞称为〝职业技能教育培训中心〞，将其合法化。但美国国会周三发表2018年度中国人权报告，呼吁白宫向中共施压立即关闭〝再教育营〞。
 <br/>
 <br/>
 大纪元时报
 <br/>
 10月8号，英国广播公司BBC播出调查报导，名为〝该相信谁？中国的器官移植〞。BBC采访了曾参与活摘器官的中国医生，并曝光，中国的法轮功学员不但遭受中共酷刑迫害，还被定期强迫抽血、体检等，被怀疑与活摘器官有关。
 <br/>
 <br/>
 美国媒体VOX
 <br/>
 亚利桑那州女子瓦莱斯在沃尔玛买的钱包里发现一张纸条，上面说中国广西英山监狱强迫囚犯做奴工，监狱条件非常恶劣。VOX记者沃伦特意前往英山监狱实地调查，当地居民和狱监家属都证实此事，而且沃伦还发现百度搜索有监狱招揽业务的广告。
