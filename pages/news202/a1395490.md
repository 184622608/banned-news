### 【时事拼盘】法国西南部遭世纪山洪 韩登山客尼泊尔遇难
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】下面请看一组新闻简讯。
 <br/>
 <br/>
 意大利将2019年赤字目标提高到国民生产总值的2.4％，引发金融市场不安，并招致欧盟批评，意大利当局已将是否批准预算的决定推迟到周二。欧盟指出，意大利公债已达到GDP的131％，他们不希望意大利成为第二个希腊，或者更严重。
 <br/>
 <br/>
 法国西南部遭遇一个世纪以来最严重的山洪灾害，洪水淹没了村庄和农田。至少有13人丧生，一人失踪。数千人从该地区撤离。官方表示，一夜之间倾泻的洪水高达7个月降雨量。
 <br/>
 <br/>
 日前，澳大利亚昆士兰一名护理人员遭到一只袋鼠袭击，伤到了头部、上身和四肢。他的妻子也被袋鼠攻击，导致肺部塌陷，肋骨断裂割伤。他们的儿子最终帮忙击退袋鼠，受了轻伤。当地救护人员表示，袋鼠攻击力强劲，但袭击人相当罕见。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼泊尔.html" target="_blank">
  尼泊尔
 </a>
 西部喜马拉雅山峰遭遇风暴袭击，一队韩国登山者和他们向导的营地被摧毁，九人失踪。警方已发现七具尸体。这是两年来尼泊尔最严重的攀岩事故。
