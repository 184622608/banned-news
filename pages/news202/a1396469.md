### 不明原因 法国3省现无臂新生婴儿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 近10多年来不断有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无臂婴儿.html" target="_blank">
  无臂婴儿
 </a>
 出生，这个问题在今年9月被媒体广泛报导后，才再次受到舆论关注。目前，官方对初步调查结果表示不太满意，称会重新研究问题。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无臂婴儿.html" target="_blank">
  无臂婴儿
 </a>
 的出生出现在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 的3个省，分别为安省(Ain)在2009和2014年出现7宗、西部罗亚尔-大西洋省（Loire Atlantique）在2007和2008年出现3宗、西北部的布列塔尼地区(Bretagne)在2011和2013年有4宗。
 <br/>
 <br/>
 天生无臂婴儿问题事件由设于东部罗纳阿尔卑斯区的畸形婴儿登记部门，在2014年报告中提及，至今年9月才在媒体广泛报导下引起舆论关注。
 <br/>
 <br/>
 法国卫生部长布赞（Agnès Buzyn）21日上电视节目上表示，之前调查没有找到问题原因，官方并不满意和不能忍受，宣布会重新研究。并强调，会继续支持有关部门在这方面的研究工作，目前不会关闭任何部门。
 <br/>
 <br/>
