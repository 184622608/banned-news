### 模仿热播剧海报 川普幽默宣布制裁伊朗
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】周五（11月2日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推特.html" target="_blank">
  推特
 </a>
 上贴出一张具有电影
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海报.html" target="_blank">
  海报
 </a>
 风格的图片，以极特别的方式宣布制裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 ，引发了网络关注。
 <br/>
 <br/>
 周五，美国政府宣布全面制裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推特.html" target="_blank">
  推特
 </a>
 上公布了这一消息，但风格极具特色。
 <br/>
 <br/>

 <br/>
 在该推文中，川普没有使用任何自己的语言，但一切意味尽在不言中。他模仿美国奇幻电视热播剧《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_权力游戏.html" target="_blank">
  权力游戏
 </a>
 》（Game of Thrones）中的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海报.html" target="_blank">
  海报
 </a>
 形象，告诉世界：〝制裁要来了。〞（Sanctions are coming.）
 <br/>
 <br/>
 白宫周五宣布，恢复2015年伊朗核协议签署前对伊朗的全部制裁，包括能源、造船、航运和银行业。制裁措施从下周一（11月5日）开始执行。
 <br/>
 <br/>
 今年5月，川普宣布退出伊朗核协议并恢复制裁伊朗，以惩罚并制止其对恐怖主义的支持。
 <br/>
 <br/>
 最新一轮制裁针对的是伊朗经济的核心领域，白宫称，这是迄今对伊朗所实施的最严厉的制裁。
 <br/>
 <br/>
 美国国务卿蓬佩奥（Mike Pompeo）周五表示，新的制裁将使此前的制裁加速取得成功效果，美国的最大化施压已导致伊朗货币里亚尔急剧下跌，伊朗总统鲁哈尼（Hassan Rouhani）的内阁陷入混乱，伊朗人民甚至正在更大声地反对腐败、虚伪的伊朗政权。
 <br/>
 <br/>
 有趣的是，在美国官方公布这一消息后，川普又采用〝模因〞（meme）在推特上幽默地强调了一下。
 <br/>
 <br/>
 目前尚不知是谁创作了川普这一电影风格的海报形象。
 <br/>
 <br/>
 不过，HBO电视网对川普的举动表示不满，暗示川普在〝盗用〞他们的〝商标〞。
 <br/>
 <br/>
 这幅海报的设计别具匠心，字体和语句都故意模仿《权力游戏》中的标志性口号——〝冬天要来了。〞（Winter is coming.）
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
