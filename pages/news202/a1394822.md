### 涉刺杀总统 委国被捕议员坠楼身亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反对党.html" target="_blank">
  反对党
 </a>
 议员阿尔班被控参与8月4日无人机刺杀总统马杜罗失败事件，他是8日在审前羁押期间
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠楼身亡.html" target="_blank">
  坠楼身亡
 </a>
 。数百民众今天在他的丧礼高呼要求正义，美国点名马杜洛政府涉入阿尔班之死。
 <br/>
 <br/>
 中央社报导，52岁的阿尔班（Fernando Alban）为卡拉卡斯市议员，被控参与8月4日无人机刺杀
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 总统马杜罗（Nicolas Maduro）失败事件，他是8日在审前羁押期间身亡。
 <br/>
 <br/>
 当局说，阿尔班当时要求到洗手间，而从国家情治机构10楼窗口跳下自杀。但阿尔班的支持者说他是遭到谋杀。
 <br/>
 <br/>
 阿尔班葬在首都卡拉卡斯东部一座墓园，送葬队伍从委内瑞拉中央大学（The Central University of Venezuela）出发，走了好几公里远。
 <br/>
 <br/>
 一名年轻人举着标语牌，上面写着：〝马杜罗独裁政权置人于死地〞，另一面牌则写着〝正义〞。
 <br/>
 <br/>
 美国和国际社会同为阿尔班之死抱屈，指责马杜罗应为阿尔班丧命负责。
 <br/>
 <br/>
 白宫在声明中说：〝美国谴责马杜罗政府涉入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反对党.html" target="_blank">
  反对党
 </a>
 议员阿尔班之死。〞联合国与欧洲联盟也呼吁调查阿尔班命案。
 <br/>
