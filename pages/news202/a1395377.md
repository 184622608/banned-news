### 谷歌内部会议记绿曝光 〝中国项目〞再引关注
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】近期，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌.html" target="_blank">
  谷歌
 </a>
 被爆为进入中国市场，在设计符合中共审查标准的搜索引擎，引发全球各界的广泛批评。近日，美国媒体又曝光了一份谷歌内部会议纪录，洩露了谷歌这个项目更多的信息。
 <br/>
 <br/>
 美国新闻网站《拦截》（The Intercept）10月9号公布了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌.html" target="_blank">
  谷歌
 </a>
 搜索引擎主管戈麦斯（Ben Gomes），在7月18号员工会议上的一段谈话纪录，内容是关于代号为〝蜻蜓〞的项目。
 <br/>
 <br/>
 这个秘密项目，是谷歌的所谓〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国项目.html" target="_blank">
  中国项目
 </a>
 〞，为中国开发一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_审查搜索引擎.html" target="_blank">
  审查搜索引擎
 </a>
 ，将会把〝人权〞、〝学生抗议〞和〝诺贝尔奖〞等词汇列入屏蔽的黑名单。
 <br/>
 <br/>
 戈麦斯在讲话中表示，这个项目对公司很重要，他们的根本使命是服务全球用户，这过程会给公司和员工都带来很多附加好处。
 <br/>
 <br/>
 他还提到中美贸易战等因素使公司放缓了与中共官员的谈判。谷歌要求中共批准谷歌在中国市场启动他们开发的审查搜索引擎。
 <br/>
 <br/>
 戈麦斯之前曾表示，审查搜索引擎有望在6个月到9个月内发布，也可能更快。但他这次在谈话中也承认，这个项目的未来存在很大的不确定性，是一个〝艰难的旅程〞。他不会过多的注重项目发布的时间表。
 <br/>
 <br/>
 新唐人电视安心综合报导
</div>
