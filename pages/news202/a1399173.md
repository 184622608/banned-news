### 以色列还击巴勒斯坦 加沙地带再起硝烟
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴勒斯坦.html" target="_blank">
  巴勒斯坦
 </a>
 武装分子对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 发动攻击，导致双方至少4人死亡。
 <br/>
 <br/>
 周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴勒斯坦.html" target="_blank">
  巴勒斯坦
 </a>
 极端组织〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈马斯.html" target="_blank">
  哈马斯
 </a>
 〞，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 境内一辆公交车突然发动导弹袭击，车上一名军人严重受伤。之后〝哈马斯〞向边境发射了300多枚火箭弹和迫击炮弹。
 <br/>
 <br/>
 以色列军方表示，周一哈马斯向以色列南部发射了400多枚导弹。
 <br/>
 <br/>
 以色列国防部发言人 EMMANUEL NACHSHON：〝过去24小时，400多枚迫击炮弹和火箭攻击了以色列城镇，造成致命后果。〝哈马斯〞故意攻击我们的平民，这一点我们绝不能，也不会接受。〞
 <br/>
 <br/>
 周二，以色列摧毁了〝哈马斯〞情报机构和电视台，以色列空袭前发出警告，电视台办公人员得以提前撤离。
 <br/>
 <br/>
 以色列国防部发言人 EMMANUEL NACHSHON：〝以色列国防军将以强大、致命的方式回击，确保让〝哈马斯〞知道利害，让他们明白必须立即停止袭击。〞
 <br/>
 <br/>
 周二，巴勒斯坦继续向以色列发动攻击。据媒体报道，此次交火导致至少三名巴勒斯坦军人和一名以色列平民遇害。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
