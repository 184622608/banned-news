### 贸易战到新冷战？美中有冲突加剧隐忧
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】观众朋友大家好，欢迎收看新唐人全球新闻。美国副总统彭斯本月初发表的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 演讲，引发各界讨论，美中关系是否已从单纯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，转向全面
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 了呢？
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国学者.html" target="_blank">
  美国学者
 </a>
 也担忧，美中冲突升级的风险，在日益增加。
 <br/>
 <br/>
 《华尔街日报》日前发表文章，美中正逐渐迈向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 。对此，白宫首席经济顾问库德洛（Larry Kudlow）周日表示，冷战一词留给历史学家去评判，但他一直参与和中方的贸易谈判并不令人满意。
 <br/>
 <br/>
 战略与国际研究中心中国项目主任 葛来仪：〝我不想用冷战的标签，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 的确充满争端。双方存在的问题这些年越来越多。这些问题一直无法得到成功解决。这也令人感到一种溢出效应。〞
 <br/>
 <br/>
 专家认为，美国国内对中国加入世贸后，美国利益受损，已有广泛共识。同时，美中之间在贸易、安全领域的诸多问题累积下来，已影响到其它各领域。美中军事对抗风险日益突显。
 <br/>
 <br/>
 战略与国际研究中心中国项目主任 葛来仪：〝不只是南海、还有东海和台湾。这些问题总是存在更加凸显的风险。〞
 <br/>
 <br/>
 9月底，美军驱逐舰在南沙群岛附近执行〝航行自由〞任务时，遭中方舰艇危险逼近，几乎造成碰撞。而媒体披露，美国海军科研船汤玛斯号目前正停靠高雄港。在美中关系敏感时刻，其象征意义引人关注。
 <br/>
 <br/>
 战略与国际研究中心副总裁 Michael Green：〝我猜想，这件事被公开，因为很多五角大楼与国会官员，感到美国应显示对台湾的更高支持度，以平衡或抵制来自北京日益加大的压力。〞
 <br/>
 <br/>
 中共一直期望中期选举后，美中贸易紧张关系能有所缓和，但华府专家普遍认为，川普启动关税措施与国内政治状况无关，而是一项长期战略，也是两党共识。
 <br/>
 <br/>
 战略与国际研究中心中国项目主任 葛来仪：〝川普政府要的是互惠公平的关系，服务于美国利益。遏制中国不是川普政府的目标，但我了解为何中共这么想。〞
 <br/>
 <br/>
 川普和习近平有可能会在11月阿根廷举行的G20会议上举行会谈。专家认为，即使双方在贸易问题上有所缓解，其它问题上的争端也可能继续加剧。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
