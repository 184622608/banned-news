### 中东行动指挥官在巴林身亡 美海军：暂排除他杀
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】美国海军表示，甫于5月展开包含
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第五舰队.html" target="_blank">
  第五舰队
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中东行动.html" target="_blank">
  中东行动
 </a>
 指挥官斯蒂尔尼12月1日在巴林寓所身亡，暂无他杀嫌疑。目前海军犯罪调查局（NCIS）及巴林内政部正合作展开调查。
 <br/>
 <br/>
 美国海军军令部长李察逊（John Richardson）在声明中表示：〝这对斯蒂尔尼（Scott Stearney）的家人、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第五舰队.html" target="_blank">
  第五舰队
 </a>
 与整个海军是件震惊的消息，史提尔尼是战功彪炳的海军战士。〞
 <br/>
 <br/>
 他说：〝目前没有他杀嫌疑。〞
 <br/>
 <br/>
 第五舰队副指挥官海军少将施里森（Paul Schlise）在斯蒂尔尼过世后，暂代遗缺。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海军中将.html" target="_blank">
  海军中将
 </a>
 斯蒂尔尼5月展开他在包含第五舰队（Fifth Fleet）的美国中央海军司令部（US Naval Forces Central Command）及海上联合部队（Combined Maritime Forces）指挥官职务。
 <br/>
 <br/>
 巴林王国（简称巴林），是一个位于西亚、邻近波斯湾西岸的岛国，首都麦纳玛。巴林岛为巴林最大的岛屿，共有55公里长，18公里宽。沙特阿拉伯位于巴林西部，并可经由法赫德国王大桥连接；伊朗则位在巴林北方200公里处；卡达半岛位于巴林湾东南侧。气候属热带沙漠气候。2010年，巴林总人口为1,234,571人，其中包括666,172名外籍人口。
