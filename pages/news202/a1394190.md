### 韩国前总统李明博一审被判15年
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】韩国首尔中央地方法院周五（10月5日）对前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李明博.html" target="_blank">
  李明博
 </a>
 涉嫌贪污受贿案作出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一审.html" target="_blank">
  一审
 </a>
 判决，判处77岁的李明博15年有期徒刑，处罚金130亿韩元。
 <br/>
 <br/>
 首尔中央地方法院法官 CHUNG KYE-SUN：〝法庭判处被告（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李明博.html" target="_blank">
  李明博
 </a>
 ）15年监禁和130亿韩元罚款。 如果被告不付罚款，将被关押在劳教所里。〞
 <br/>
 <br/>
 法官判决表示，李明博16项指控中多项罪名成立，包括贪污240亿韩元、收受三星公司的贿赂、造成国有财产流失等。
 <br/>
 <br/>
 法院当天对宣判过程进行了电视直播，李明博由于健康问题和对电视直播不满而没有出席庭审。
 <br/>
 <br/>
 从2008年到2013年出任过韩国总统的李明博是韩国宪政史上第四名被逮捕的前总统。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
