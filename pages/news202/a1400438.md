### 中共扩张5G市场 专家吁美国调整政策
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】近期，美中在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5G.html" target="_blank">
  5G
 </a>
 产业上竞争激烈，不仅关系到未来经济发展，也将重新划分全球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络.html" target="_blank">
  网络
 </a>
 版图。专家呼吁美国调整相关政策，阻止中共的全球扩张。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5G.html" target="_blank">
  5G
 </a>
 时代已经来临！美国、欧盟、韩国、日本及中国都计划在2019年下半年启动5G
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络.html" target="_blank">
  网络
 </a>
 的商用部署，到2020年实现商用。
 <br/>
 <br/>
 5G是指第五代
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移动通信.html" target="_blank">
  移动通信
 </a>
 系统，其传输速度可达4G网络的100倍，适应物联网、无人驾驶汽车、商用无人机等新技术应用。预计到2035年，全球5G经济产出将达12.3万亿美元。
 <br/>
 <br/>
 哈德森研究所高级研究员 Arthur Herman：〝谁能在这一领域处于领先和主动地位，将在全球通讯技术中长期扮演主导角色，直到21世纪末。〞
 <br/>
 <br/>
 目前，美中之间在5G领域竞争激烈，都希望成为新一代全球通讯技术主宰者。北京把发展5G作为2025计划重点之一，提供大量政府补贴，扶持移动运营商和设备制造商，发展相关技术及基础设施。而美国等西方国家的5G发展，基本靠市场与私人企业推动。
 <br/>
 <br/>
 Rivada Networks公司首席执行官 Declan Ganley：〝我认为有一个精心策划的政府补贴计划，目的是把爱立信、诺基亚、三星踢出市场，让人们在安全等方面偷工减料，获得对基本安全领域，即网络领域的控制。〞
 <br/>
 <br/>
 据德勤公司报告，2015年以来，中国在建设5G上的支出超出美国240亿美元。同时，中共扶植本国企业，迅速占据全球5G网络设备市场。目前已有61个国家同意使用或正在测试中国的硬体设备，其中华为是最主要供应商。中共这种倾举国之力的竞争模式令业界深感担懮。
 <br/>
 <br/>
 诺基亚美洲地区政策与公共事务主管 Brian Hendricks：〝不只是一个网络营运商来到拉美国家，进行推销，而是整个银行系统，和外交介入，是多部门、跨部门的方式。〞
 <br/>
 <br/>
 未来，网络安全将是国家安全的最前沿。专家认为，依赖中共支持的企业，掌控各国5G网络基础建设，将带来巨大安全隐患。
 <br/>
 <br/>
 Rivada Networks公司首席执行官 Declan Ganley：〝在5G经济中，每一平方英里内将有超过一百万的设备被联通。承载这些信息的网络将有潜在的被窃密风险，安全受到损害，这些都是我们需要非常严重关切的。〞
 <br/>
 <br/>
 5G基础设施投资巨大，合同一旦锁定很难更改。出于国家安全考量，今年8月，澳洲政府宣布，禁止华为和中兴参与该国5G网络建设；印度和韩国也都已宣布拒绝华为投资5G项目；英、德两国也正在考虑这一问题。
 <br/>
 <br/>
 明年2月底，各国政府、监管机构和业界领袖，将出席在西班牙巴塞罗纳举行，年度世界移动通信大会。届时，各国2019年的5G相关发展计划就将确定。专家认为，目前正是阻止中共通过不公平竞争占据全球市场的关键时刻。美国必须快速调整产业政策，保证企业有能力投入5G网络建设。
 <br/>
 <br/>
 Rivada Networks公司首席执行官 Declan Ganley：〝当美国这样做时，世界其它国家都会跟随。这对所有经济体都有利。美国要首先来做。现在到明年二月间，还有足够时间来宣布(政策改变)。所以还是可以做到，但这是最后时刻。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
