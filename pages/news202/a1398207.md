### H7N9禽流感病毒 日本首次证实可经飞沫传播
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】医学界早前认为，禽流感
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_病毒.html" target="_blank">
  病毒
 </a>
 不会在哺乳类动物之间以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞沫传播.html" target="_blank">
  飞沫传播
 </a>
 。但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 有关研究近日首次证实，H7N9病毒可透过飞沫传播。研究人员担心H7N9可能引发新的全球流感大流行。
 <br/>
 <br/>
 <h4>
  研究：H7N9
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_病毒.html" target="_blank">
   病毒
  </a>
  可以
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞沫传播.html" target="_blank">
   飞沫传播
  </a>
 </h4>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 放送协会（NHK）报导，东京大学的研究人员近日将两只雪貂分开在两个相距7厘米的笼子中饲养，其中一只已感染病毒，而另一只则是健康的。4天后，研究人员证实健康的那只雪貂也染病，而在两只染病动物的80厘米范围内，都可以找到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_H7N9禽流感.html" target="_blank">
  H7N9禽流感
 </a>
 病毒。由于雪貂患流感疾病与人类非常近似，所以常成为实验对象。
 <br/>
 <br/>
 报导称，科学家早前相信受禽流感感染的哺乳类动物，飞沫中并没有病毒。人类感染禽流感，主要是通过接触染病的禽鸟或其粪便，或接触受污染的环境等，但如今证实病毒可透过飞沫传播，可能会增加病毒人传人的能力。
 <br/>
 <br/>
 <h4>
  专家忧引发全球流行
 </h4>
 <br/>
 研究人员认为，当禽流感病毒出现基因变异后，将可以变成由人传人的新型流感。负责研究的教授河冈义裕担忧，H7N9禽流感病毒或会导致全球流行。他也呼吁人们提高警戒并开展进一步研究。
 <br/>
 <br/>
 日本政府为防范疫潮爆发，决定将在国内储备的H5N1型禽流感疫苗，在两年内逐步转为针对H7N9型的新疫苗。
 <br/>
 <br/>
 <h4>
  98%病例来自内地
 </h4>
 <br/>
 据香港卫生防护中心《禽流感疫情周报》数据显示，由2013年3月首度发现人类感染H7N9个案，到今年10月27日，全球共报告1,567宗病例。其中有1,536宗，也就是逾98%病例来自中国大陆，死亡率高达3～4成。其中最多的是浙江省，310宗、第二名则是广东省，259宗病例。香港发现21宗个案，全部由内地传入。
 <br/>
 <br/>
 香港大学感染及传染病中心总监何栢良表示，香港在1997年已发现H5N1禽流感病毒可经禽鸟传人，当时医学界已有共识，病毒可经飞沫传人。
 <br/>
 <br/>
 何栢良认为，虽然禽流感病毒有机会经飞沫传播，但从现阶段看，不论由禽鸟传人或由人传人的风险较低。不过他也强调，在中国大陆确诊H7N9感染人类的个案中，有数十个病毒样本属高致病性，是人或禽鸟造成的严重感染。
 <br/>
 <br/>
 上周三（10月31日），中共国家卫生健康委员会通报，广西一名44岁男子因感染H5N6死亡。通报称，患者在10月18日发病，3天后入院治疗，并于10月27日死亡。
 <br/>
 <br/>
 据悉，死者生前否认在发病前曾接触过活家禽。
 <br/>
 <br/>
 人类感染H7N9型禽流感的死亡率约为3～4成，比H5N1的6成虽然低，但之前研究显示H7N9在禽类中传播时不会使禽类出现症状，而H5N1则会导致感染禽类死亡，所以追踪H7N9在禽类中的传播更加困难。而自从5年前首次发现H7N9可感染人类后，感染者数目也比过去20年感染H5N1的多一倍。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
