### 北海道札幌今晨终下雪了 平128年来纪录
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北海道.html" target="_blank">
  北海道
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_札幌.html" target="_blank">
  札幌
 </a>
 市20日下起今年第一场瑞雪，平了128年前观测史上札幌最晚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下雪.html" target="_blank">
  下雪
 </a>
 的纪录，比往年晚23天，比去年晚28天。
 <br/>
 <br/>
 日本放送协会（NHK）报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_札幌.html" target="_blank">
  札幌
 </a>
 管区气象台表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北海道.html" target="_blank">
  北海道
 </a>
 上空1500公尺附近有寒气流入，气温低于足以让平地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下雪.html" target="_blank">
  下雪
 </a>
 的约零下6度。
 <br/>
 <br/>
 受此影响，札幌市19日凌晨0时50分降下雨夹雪（霙），气象台宣布札幌市观测到今年第一场瑞雪。
 <br/>
 <br/>
 气象台表示，本周可望出现冬季型气压分布，让冷空气卷入北海道，北海道可望断断续续出现下雪。
 <br/>
 <br/>
 北海道于本月14日旭川市、网走市、稚内市观测到今年第一场瑞雪。本月19日，带广市观测到今年第一场瑞雪。
 <br/>
 <br/>
 另气象厅也观测到，中太平洋与东太平洋赤道位置的海域水温较以往高出0.9度，证实今年确定已经发生圣婴现象，这也是2016春天以来，睽违2年半之久。此外，气象厅也认为，此次圣婴应该会持续到2019年的春天。
