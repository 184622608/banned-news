 持续调查。到周一，土耳其传唤了20名沙特领馆人员。土耳其总统埃尔多安周一召开部长级会议，周二将公布调查报告。
 <br/>
 <br/>

  土耳其
 </a>
 法庭持续调查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特记者.html" target="_blank">
  沙特记者
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被害案.html" target="_blank">
  被害案
 </a>
 。截至周一，已已经传唤了包括司机在内的20名沙特领馆人员，检察官将要求领馆内全部45名员工做出陈述。
 <br/>
 <br/>
 土耳其周一召开部长级会议，埃尔多安总统称，将在周二公布调查报告。
 <br/>
 <br/>
 沙特外长阿德尔・朱拜尔周日接受美国福克斯新闻采访时承认，卡舒吉被谋杀。
 <br/>
 <br/>
 沙特外长阿德尔・朱拜尔：〝有人在他们的职权范围之外干了这件事。这显然是个巨大的错误。错上加错的是，他们还试图对此事加以掩盖。这对任何政府都是无法接受的。不幸的是，这些事情都发生了。〞
 <br/>
 <br/>
 朱拜尔还强调，沙特王储萨勒曼并不知情，整个事件是〝流氓行动〞。
 <br/>
 <br/>
 沙特媒体周一报导，沙特国王和王储致电卡舒吉的儿子，表达慰问。
 <br/>
 <br/>
 周一，土耳其媒体公开最新监控录像，显示10月2号中午，卡舒吉和未婚妻从住所前去领馆。目前，卡舒吉的未婚妻受到警方保护。
 <br/>
 <br/>
 新唐人记者李梅、天羽综合报导
