### 巴西右转 反共亲美 拉美〝粉色浪潮〞退潮
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 总统大选，右翼候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波索纳洛.html" target="_blank">
  波索纳洛
 </a>
 击败了左翼对手，将在明年就任总统。他的当选，意味着拉美〝粉红浪潮〞退潮，并可能改写巴西〝反美国亲中共〞的外交政策。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 总统大选，右翼候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波索纳洛.html" target="_blank">
  波索纳洛
 </a>
 ，以近56%的得票率，领先左派对手哈达德11个百分点。终结了左翼〝工人党〞连续4届当选的历史。
 <br/>
 <br/>
 路透社高级记者 Gabriel Stargardter：人们厌倦了持续上升的暴力、腐败，和打击犯罪不力的现象。
 <br/>
 <br/>
 曾长期掌权的巴西工人党，主张实现〝民主社会主义〞，其高福利、高税收政策、加上产业升级缓慢，透支了经济活力，从2015年开始，巴西陷入经济衰退。
 <br/>
 <br/>
 同时，工人党先后2任总统因贪腐丑闻被判刑和弹劾。警方过去4年间进行了2056起反腐调查，涉案损失370亿美元。
 <br/>
 <br/>
 路透社高级记者 Gabriel Stargardter：工人党被指责，带来可怕的经济衰退，和一系列社会问题。
 <br/>
 <br/>
 波索纳洛承诺，减少政府干预经济，精简政府，降低税收，并强调传统家庭和宗教价值。他在胜选致词中说：〝我们不能再被社会主义、共产主义、民粹主义和左派极端主义迷惑⋯我们必将改变巴西的命运。〞
 <br/>
 <br/>
 波索纳洛：我们将捍卫宪法、民主和自由。这不是一个党或一个人的誓言，这是对神的誓言。
 <br/>
 <br/>
 波索纳洛曾公开表示，他是美国总统川普的崇拜者。他将扭转巴西的外交政策，〝不再为凶残的独裁政权喝采，不再贬损美国、意大利、以色列等民主政体。〞
 <br/>
 <br/>
 同时，波索纳洛批评，过去十几年，中共大举投资巴西能源部门和基础建设，收购巴西发电部门等战略项目，是〝掠夺者〞行为。他曾在2016年说，他希望美国取代中共，成为巴西最大的贸易伙伴。
 <br/>
 <br/>
 陈破空：南美洲最大的人口国跟美国携手，将极大的改变拉丁美洲的政治版图，对挤出中共对拉丁美洲的渗透也具有重大的意义。而且在美中对抗性的较量中，可以说美国又增加了一个盟友。
 <br/>
 <br/>
 过去近20年来，拉美政坛一度由左翼控制，被称为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_粉色浪潮.html" target="_blank">
  粉色浪潮
 </a>
 〞。但过去3年，阿根廷、智利、巴西等重要国家已先后〝右转〞，被认为〝粉色浪潮〞进入历史低点。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
