### 大篷车难民继续北上 川普：黑帮或混入其中
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】再来关心中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 潮。周二，进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 的难民继续向北缓慢推进。墨西哥总统表示，违反规定的难民不能待在墨西哥，或进入美国。
 <br/>
 <br/>
 周一，大小车队，包括步行的，约七千到一万洪都拉斯难民，继续从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 南部北上，距离美国边境，田纳西的麦卡伦还有1800公里。
 <br/>
 <br/>
 难民称，他们主要为了躲避国内的暴力和贫困。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 ：〝洪都拉斯三、四十岁以上的妇女，找不到工作。〞
 <br/>
 <br/>
 墨西哥希望在难民进入边境前，驱散车队。当局在边境设立机构，办理难民登记，登记后的难民可留在庇护所，等待审批，审批需几周时间。
 <br/>
 <br/>
 墨西哥总统表示，违反规定的难民不能待在墨西哥，或进入美国。
 <br/>
 <br/>
 墨西哥总统 涅托：〝到达你们的目的地将非常困难，无论是进入美国，还是留在墨西哥。只有拥有难民保护和经过法律程序，你们才能进入我们国家，最终获得这种权利。〞
 <br/>
 <br/>
 周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 批评洪都拉斯和危地马拉阻止难民不利，表示将减少对洪都拉斯、危地马拉和萨尔瓦多的援助。
 <br/>
 <br/>
 美国总统川普：〝我们给了他们数百万美元，他们什么也没为我们做。〞
 <br/>
 <br/>
 川普批评民主党开放边境的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民政策.html" target="_blank">
  移民政策
 </a>
 ，担忧
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑帮.html" target="_blank">
  黑帮
 </a>
 和恐怖分子会混入难民中，威胁美国安全。
 <br/>
 <br/>
 美国总统川普：〝到（大篷车难民）中间去，你会发现（里面）混有MS-13黑帮分子，混有中东人，你会发现这一切。我们不允许他们待在我们的国家。我们要安全，我们要安全。〞
 <br/>
 <br/>
 川普周一宣布，南部边境进入国家紧急状态。
 <br/>
 <br/>
 新唐人记者毕心慈、张琪综合报导
</div>
