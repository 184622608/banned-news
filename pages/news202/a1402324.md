### 日本政府重大决定：封杀华为
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】〝封杀
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 〞的模式从美国开启后，逐渐向世界各个发达国家扩散。继澳洲、纽西兰、英国等国家后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 政府也作出重大决定，加入〝封杀华为〞的行列。
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 《读卖新闻》7日报导，模式由于担心情报泄漏以及遭到网路攻击，日本将修改采购规定，禁止本国政府向中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 和中兴通讯采购电信产品。
 <br/>
 <br/>
 报导引一名政府消息人士称，日本政府预计最快将于12月10日（下周一）修改内部采购规定。为避免激怒北京，日本不大可能会点出华为的名字，拟采用〝考虑到安全保障上的观点〞等措辞，但华为与中兴将是禁止对象。
 <br/>
 <br/>
 美国政府2018年8月通过的《国防授权法》，称华为和中兴与中共情报机构有关，禁止美国政府机构使用其产品。
 <br/>
 <br/>
 而澳大利亚和纽西兰也已经禁止华为参与被称为〝5G〞的高速大容量通信网建设，理由是构成安全风险。几天前，英国电信集团也表示，将把华为设备从其核心3G、4G网路中移除，也不会在5G网路的核心部分使用华为的设备。
 <br/>
 <br/>
 另外，华为创始人任正非的女儿，公司副董事长兼首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 ，则因涉嫌违反了美国对伊朗的贸易制裁，近日被加拿大警方逮捕，美国要求将其引渡。
 <br/>
 <br/>
 虽然华为公司一直坚称其设备是安全的，公司为员工共有，是独立于政府运作的公司。但外界一直质疑，华为有中共军方的背景，受制于中共政府。其创始人任正非在开办华为公司之前就是现役军人。有消息说，华为公司长期从中国军方获得关键技术，并与军方有多个合作项目。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：李泉）
</div>
