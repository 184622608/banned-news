### 京都大学105年〝吉田寮〞要拆 学生拒搬迁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】现存最老的日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京都大学.html" target="_blank">
  京都大学
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉田寮.html" target="_blank">
  吉田寮
 </a>
 〞宿舍是一栋撑了105年，摇摇欲倒的木造建筑。当局因其抗震度严重不足，要求约200名住宿生于今年9月底前全部搬出。但持反对立场的住宿学生近日联合发起不搬离运动，拒绝搬迁。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京都大学.html" target="_blank">
  京都大学
 </a>
 于1897年创设，几乎同时设立了学生宿舍。宿舍于1913年拆除重建，原先是第三高等中学，后来成为京都帝国大学的学生宿舍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉田寮.html" target="_blank">
  吉田寮
 </a>
 。木造2层楼的吉田寮，分北、中、南3栋，包括2015年完工的西寮新栋在内，现今约有200人住宿。
 <br/>
 <br/>
 大学当局于去年12月以建物严重老朽化及抗震度不足为由，提出〝勿再招募新住宿生〞、〝今年9月底前全员搬出〞等要求。但现居学生们认为花费几亿元日币去盖新宿舍，还不如拿来全面翻新吉田寮。
 <br/>
 <br/>
 10月，京大校方再次宣布吉田寮为需要拆除的危楼，希望住宿学生搬迁到取代宿舍。
 <br/>
 <br/>
 而持反对立场的住宿学生近日联合发起不搬离运动，要守住拥有悠长〝学生自治文化〞的老宿舍。他们希望透过对话，说服校方撤回这些要求。
 <br/>
 <br/>
 据悉，所谓学生自治宿舍，名义上是从管理、清洁、收费全都由学生自发组织负责经营的宿舍。但实际上是，只要在〝不会出事〞的底线之上，住宿学生可以做任何事。包括燃起营火一群人弹吉他唱歌，在交谊厅娱乐，在大厅地板摊着数人大的画纸作画等等。
 <br/>
 <br/>
 甚至还有外国旅人因其住宿费便宜，闻名而来，但吉田寮的残破外观、庭院养满山羊、兔子、鸡等动物，走廊上堆放的霉臭杂物以及那些烧焦后无人清理的平底锅，都让人深切感受到〝能住在这里的人肯定适应力很强〞。
 <br/>
 <br/>
 据摄影家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野村干太.html" target="_blank">
  野村干太
 </a>
 介绍，吉田寮的起居室，基本上是6到10个榻榻米大的共住房，相当于每人拥有3个半榻榻米大空间，另外还有麻将间、漫画间、游戏间和钢琴间等娱乐空间。
 <br/>
 <br/>
 这里的住宿费包含水电费在内每个月约2500日圆（约22美元），持有京都大学学籍的学生都有资格入住。住宿学生从大学生到研究生、留学生、旁听生都有。性别、年龄和国籍更是形形色色。包括已故小说家梶井基次郎、诺贝尔物理学奖得主赤崎勇等极富个性的文化人和学者，都曾在此栖身过。
