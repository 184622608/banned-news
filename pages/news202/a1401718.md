### 阿根廷G20峰会落幕 回顾梳理六大看点
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】12月1日，为期两天的阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 落幕，本次各国领袖会聚一堂，其中有六件大事深受外界关注，下面我们来一起来回顾。
 <br/>
 <br/>
 G20由世界上19个重要工业国及欧盟组成，这20个国家或区域代表了全球2/3人口和85%的经济总量。
 <br/>
 <br/>
 1.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合公报.html" target="_blank">
  联合公报
 </a>
 <br/>
 <br/>
 但此次峰会，正值美中贸易摩擦，俄罗斯和乌克兰发生冲突等敏感事件，有媒体披露，峰会公报的协商过程，非常〝艰难〞，引发外界关注。
 <br/>
 <br/>
 不过，经过连夜工作，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 公报内容12月1日最终定稿。由于此前美中立场分歧导致APEC峰会公报流产，所以本次G20峰会的公报格外受关注。
 <br/>
 <br/>
 公报中説：〝目前的WTO系统未能完成其目标，有改进的空间。因此我们支持对WTO进行必要的改革，改善运作机制。我们将在下次G20峰会上审核改革的进度。〞
 <br/>
 <br/>
 美国已多次呼吁WTO改革，以应对北京的诸多违规行为。
 <br/>
 <br/>
 同时，为了避免公报流产，公报中避开了〝保护主义〞、〝不公贸易〞等敏感字眼。
 <br/>
 <br/>
 2. 美中贸易
 <br/>
 <br/>
 备受瞩目的川习会于12月1日晚举行，双方达成了90天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 短暂停火协议。中方作出了较大的让步，但是在90天后，如果中方不能满足美国的改革要求，美国将继续加大对中方的关税施压。同时，会上，川普重申他与习近平的关系〝非常特别〞。
 <br/>
 <br/>
 3. 美墨加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易协议.html" target="_blank">
  贸易协议
 </a>
 <br/>
 <br/>
 11月30日，美国总统川普和加拿大总理特鲁多，和当时即将卸任的墨西哥总统尼托（Enrique Pena Nieto），共同签署了《美墨加贸易协定》。该协议将取代以往的《北美贸易协议》。但签署后，还需各自国家的立法机构批准，才能正式生效。
 <br/>
 <br/>
 川普表示，新协议将首先支持劳工阶层，支持高薪水的制造业就业，促进美国包括农业、制造业和服务业在内的出口机会。
 <br/>
 <br/>
 4.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特王储.html" target="_blank">
  沙特王储
 </a>
 面临压力
 <br/>
 <br/>
 除了最重要的贸易议题外，一些外围事件也影响着峰会的进程。
 <br/>
 <br/>
 例如：10月初，沙特记者卡舒吉，在沙特驻土耳其伊斯坦布尔的领事馆内遇害，有舆论质疑沙特王储萨尔曼涉嫌此案，沙特王室否认萨尔曼对此事先知情。
 <br/>
 <br/>
 萨尔曼代表沙特出席本次G20峰会，也受到了这方面的压力。
 <br/>
 <br/>
 英国首相特蕾莎・梅11月30日会见萨尔曼，当面敦促他设法阻止类似案件发生。
 <br/>
 <br/>
 当然，也有领导人对萨尔曼释出善意，如俄罗斯总统普京，就被拍到与萨尔曼击掌。
 <br/>
 <br/>
 5. 俄罗斯乌克兰冲突
 <br/>
 <br/>
 同时，因为俄乌之间爆发新的冲突，俄罗斯扣押乌克兰船隻和水手，川普总统称，因此取消与普京在G20峰会上的会谈，普京则对此表示遗憾。
 <br/>
 <br/>
 德国总理默克尔12月1日与普京举行会谈，呼吁俄方释放乌克兰船隻和水手，双方并同意今后就此议题进行更进一步的谈判。
 <br/>
 <br/>
 6. 气候议题
 <br/>
 <br/>
 在最后的公报中，除美国外的19个国家或地区，再次申了他们对《巴黎气候协议》的坚持。而美国早在2017年6月，已宣布退出这一协议。川普经常强调，气候变暖是〝骗局〞，而且巴黎协议对美国不公平。
 <br/>
 <br/>
 而美国总统川普在峰会结束后，已于周日凌晨抵达美国。
 <br/>
 <br/>
 新唐人电视综合报导
