### 澳洲墨尔本市区砍人事件 有伤亡
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】当地时间周五(11月9日)下午约4点20分，澳大利亚墨尔本发生火烧车和砍人的攻击事件。目前已知有1人死亡，2人受伤。
 <br/>
 <br/>
 根据当地媒体报导，在墨尔本街头的事发现场，先传出连续爆炸声，一辆汽车起火。一名男子下车朝人群挥刀，已知造成1人死，2人受伤。
 <br/>
 <br/>
 警方赶到现场，与肇事男子对峙一阵，最后警方开枪击伤肇事男子，才将其制服。
 <br/>
 <br/>
 现场救护人员表示，受伤者中，其中1名颈部受伤，伤势严重，另1人则是头部受伤，都已送医救治，目前状况不明。
 <br/>
 <br/>
 现场已被警方封锁，民众被要求待在室内，附近的店家也暂停营业。
 <br/>
 <br/>
 警方表示，目前仍无法确认男子是否蓄意攻击人群，或是否与恐袭有关。事件正在调查中。
 <br/>
 <br/>
 新唐人记者陈依心、尚敬综合报导
