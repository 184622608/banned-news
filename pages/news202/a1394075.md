### 欧洲法轮大法交流会 35国学员相聚布拉格
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】9月29日，一年一度的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧洲.html" target="_blank">
  欧洲
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮大法.html" target="_blank">
  法轮大法
 </a>
 修炼心得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交流会.html" target="_blank">
  交流会
 </a>
 在捷克首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布拉格.html" target="_blank">
  布拉格
 </a>
 召开。来自欧洲35个国家的部分法轮功学员参加了这一盛会。
 <br/>
 <br/>
 1500多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧洲.html" target="_blank">
  欧洲
 </a>
 法轮功学员汇聚捷克首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布拉格.html" target="_blank">
  布拉格
 </a>
 ，参加了2018年欧洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮大法.html" target="_blank">
  法轮大法
 </a>
 修炼心得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交流会.html" target="_blank">
  交流会
 </a>
 。 来自13个国家的20名学员分享了他们在工作和生活中如何通过修炼做好人，遇事向内找自己的不足，抓紧时间讲真相救人的心得体会。法轮功创始人李洪志先生为大会发来贺词。
 <br/>
 <br/>
 第一位发言的是来自土耳其的Strnad。她从2000年就开始修炼，如今生活在布拉格。她讲述了在怀孕及照顾小宝宝的过程中，如何提高自身承受力，尽量为别人着想，同时做好母亲和讲真相的工作。
 <br/>
 <br/>
 捷克法轮功学员 Nurgül Strnad：法轮功教会了我，让我的心胸变得越来越宽广。我真的很高兴。我也能看到别人（通过修炼）发生的变化。
 <br/>
 <br/>
 来自希腊的Tagari女士是一名政治官员。通过她的努力，去年希腊议会举办了关于发生在中国的活摘器官的听证会。在向政治家讲真相过程中，Tagari发现自己有自满的心，陶醉于好的名声，于是在实践中有意识地放下这个执着。
 <br/>
 <br/>
 来自拉脱维亚的25岁小伙子雷蒙兹4年前在书店发现了《转法轮》一书，从此加入了修炼的行列，在日常生活中他用真善忍的标准要求自己，主动帮助母亲做家务，不和周围的人生气。他开始学习中文，如今能用中文部分阅读法轮功书籍，还用毛笔字抄写 《转法轮》。
 <br/>
 <br/>
 希腊法轮功学员 Zafizis Georgopoulos：心得交流真是太棒了，给了我很大帮助。我明白了自己在修炼中不能懈怠。如果不在这样的环境中，回到日常生活中，就会松懈，觉得还有很多时间。在这里我看到其它学员，他们不松懈，一直在精进修炼。
 <br/>
 <br/>
 新唐人记者伊仁、金卓捷克布拉格报导
