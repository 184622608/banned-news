### 美国〝极限施压〞伊朗  中共或寻求规避制裁
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 的新一轮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济制裁.html" target="_blank">
  经济制裁
 </a>
 周一已全面生效。中国虽获得临时豁免，可继续进口伊朗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_原油.html" target="_blank">
  原油
 </a>
 ，但专家认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 将会以各种手段与伊朗暗中进行其它交易。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 第二轮制裁本周一正式生效。这些制裁直接针对伊朗的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_原油.html" target="_blank">
  原油
 </a>
 、银行和航运等核心经济部门，涉及近千个公司、实体和个人，以及飞机和船隻。
 <br/>
 <br/>
 前白宫国安会国际经济事务主任 David Mortlock：〝参与伊朗某些交易的外国公司将面临制裁威胁。这是根本性的改变。〞
 <br/>
 <br/>
 在美国第二轮制裁之前，波音、通用电气、西门子等众多跨国企业已暂停或退出在伊朗的巨额投资项目。伊朗政权正面临前所未有困境，国内失业增加，非法交易上升，投资与贸易骤降，腐败加剧。医药进口也从欧美西方国家转向中印两国，但药品品质大大降低。
 <br/>
 <br/>
 美国大西洋理事会中东问题研究员 Holly Dagres：〝有药剂师告诉我，她看到的最糟状况是，从中国进口的针管上有凝固的血迹。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 是进口伊朗原油的最大买家。不过，这次华府将中国列入八个豁免国家和地区，可暂时继续进口伊朗原油。
 <br/>
 <br/>
 华府智库欧亚集团研究战略主任 Meredith Sumpter：〝我们预计，中共要么遵守美国的制裁规定，要么会用如中国的中小银行，来协助进行部份(非法)交易。〞
 <br/>
 <br/>
 专家认为，中共要想规避美国制裁，可能会避免大型国有银行系统和运输公司与伊朗进行直接交易。但根据美国法律，制裁的临时豁免待遇期限仅为180天。
 <br/>
 <br/>
 华府智库欧亚集团研究战略主任 Meredith Sumpter：〝重要的不是过去已发生的，而是明年3月会发生什么？那将是下次豁免的期限日。华盛顿要如何敦促北京进一步减少从伊朗进口石油？北京又会如何反应？〞
 <br/>
 <br/>
 美国国家安全顾问博尔顿日前已表示，除已经宣布的石油和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济制裁.html" target="_blank">
  经济制裁
 </a>
 外，美国还将对伊朗实施更多制裁。专家认为，美国最终目的是通过〝极限施压〞，迫使伊朗改变行为，或者实现政权更替。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
