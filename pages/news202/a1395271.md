### 高科技3D建模型  助千年古城战后重建
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】古代的两河流域文明，包括如今的伊拉克和叙利亚，留下了不少珍贵的世界文化遗产，然而在连年的战火摧残下，许多遗迹，遭到毁坏。法国技术人员，结合无人机拍照和人工智能等技术，希望帮助重建，这些宝贵的文化遗产。
 <br/>
 <br/>
 从叙利亚的帕尔米拉、阿勒颇到伊拉克的摩苏尔，许多中东的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千年古城.html" target="_blank">
  千年古城
 </a>
 ，在战火中遭到严重破坏，满目疮痍。
 <br/>
 <br/>
 为了重建这些宝贵的遗址，一组法国科技人员，开发技术，模拟城市重建后的模样。
 <br/>
 <br/>
 这项技术首先使用无人机拍摄数十万张照片，然后输入超级计算机进行3D建模，再使用人工智能算法在一定程度上进行模拟重建。
 <br/>
 <br/>
 项目创始人 Yves Ubelmann：〝我们正在生成的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_3D模型.html" target="_blank">
  3D模型
 </a>
 能帮助创建城市地图，总结土地归属，进行城市分析，从而帮助建筑师决定如何展开重建工作。〞
 <br/>
 <br/>
 这个名为ICONEM的项目，另一个目的，是希望记录下这些文化古城的样貌，给后人留下宝贵的历史财富。
 <br/>
 <br/>
 安德鲁・梅隆基金会 执行副总裁  Mariet Westermann：〝我真的非常感动。一开始看到如此大规模的破坏。即使曾经去过那里的人，现在也无法去那儿（看到同样的景观）了。聚焦于此非常重要。〞
 <br/>
 <br/>
 这些科技人员，已经和联合国教科文组织合作，规划摩苏尔的重建工作。此前，摩苏尔遭到恐怖组织〝伊斯兰国〞占领达三年之久，惨遭战火蹂躏。据估计，重建工作至少需要花费20亿美元。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
