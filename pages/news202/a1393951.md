### 地震海啸已致1407死 印尼火山又爆发
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼苏拉威西岛.html" target="_blank">
  印尼苏拉威西岛
 </a>
 上祸不单行，上星期发生7.5级的强震，并引发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海啸.html" target="_blank">
  海啸
 </a>
 ，导致1400多人死亡。本周三又有一座
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火山.html" target="_blank">
  火山
 </a>
 喷发，目前还没有伤亡报告。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼苏拉威西岛.html" target="_blank">
  印尼苏拉威西岛
 </a>
 发生强震和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海啸.html" target="_blank">
  海啸
 </a>
 五天以来，搜救人员仍在奋力救灾。截至周三发稿，死亡人数已上升到1407。
 <br/>
 <br/>
 强震夷平地面上的一切，放眼望去，到处是扭曲的屋顶，绵延不断的瓦砾堆。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 时，土壤在震动下变得像液体一样松散。据当地救灾部门称，贝托波有744所房屋被液化了的土壤完全吞噬，数百人被埋在土下，极可能已经遇难。
 <br/>
 <br/>
 此外，苏拉威西岛的索普坦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火山.html" target="_blank">
  火山
 </a>
 当地时间周三上午爆发，火山灰喷至4公里高空。
 <br/>
 <br/>
 当局已经通知火山半径4公里范围内的居民随时待命，等待疏散命令。所幸目前没有人员伤亡。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
