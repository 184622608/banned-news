### 川普纪念一战阵亡美军 法国之行结束
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】周日，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 到访了，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 纪念
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一战.html" target="_blank">
  一战
 </a>
 活动的最后一站，巴黎外郊叙雷讷的美军公墓，以纪念在第一次世界大战中阵亡的美军士兵。随后，他与夫人乘坐空军一号返回美国。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：今天，我们延续神圣的义务，来缅怀在这片土地上长眠的战争英雄。
 <br/>
 <br/>
 座落在巴黎外郊叙雷讷的美军公墓，埋葬着约1500名美军士兵。川普与负责维护该遗址的美国战争纪念碑委员会（ABMC）秘书威廉・麦茨将军等人共同出席了纪念仪式。
 <br/>
 <br/>
 美国总统 川普：经历了雨雪、冰雹、泥泞、毒气、子弹和迫击炮，他们坚守阵地，并向前推进直到胜利，一场伟大的胜利。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一战.html" target="_blank">
  一战
 </a>
 中，史上首次出现在欧洲战场的美军，于1918年初在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 的贝洛森林与发起春季攻势的德军展开激战，这个公墓埋葬的便是那场战斗中丧生的美军士兵，其中900多人并没有找到遗体。
 <br/>
 <br/>
 1914至1918年，是德国、奥匈帝国等组成的同盟国与英法美、中华民国等组成的协约国之间的战争。在付出多于1600万人的生命后，协约国最终获胜。
 <br/>
 <br/>
 川普说，一战中有超过26,000美国人丧生，不止于95,000人负伤。
 <br/>
 <br/>
 美国总统 川普：一场代价昂贵但巨大的胜利，我们无从得知战士们是否再次见到亲人。
 <br/>
 <br/>
 当天早些时候，川普夫妇并在巴黎出席了由法国总统马克龙主持的一战停战一百周年纪念仪式。
 <br/>
 <br/>
 一同出席活动的还有德国总理默克尔、俄罗斯总统普京等70多位世界各国首脑。在凯旋门前，马克龙发表了二十分钟的演讲。
 <br/>
 <br/>
 马克龙说：〝让我们再一次庄重承诺和平高于一切。（让我们在座的所有政治领导者们）再次确认在自己人民面前的真正的巨大的责任，将和平传承给我们的子孙。〞
 <br/>
 <br/>
 周日在法国的活动结束后，川普总统一行便启程返回美国。
 <br/>
 <br/>
 在美国，11月11日的一战
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纪念活动.html" target="_blank">
  纪念活动
 </a>
 通常被称为〝老兵节〞，美国官方也举办了各种活动纪念一战结束100周年。
 <br/>
 <br/>
 新唐人电视综合报导
</div>
