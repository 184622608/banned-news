### 喀麦隆学校惊传集体掳人 82师生下落不明(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】西非国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_喀麦隆.html" target="_blank">
  喀麦隆
 </a>
 西北省首府巴门达（Bamenda）一间学校，5日惊传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集体掳人.html" target="_blank">
  集体掳人
 </a>
 事件。一间寄宿学校有82名师生，遭分离主义份子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绑架.html" target="_blank">
  绑架
 </a>
 ，目前下落不明。外界认为，值此总统就职典礼前夕发动攻击，明显挑衅政府。
 <br/>
 <br/>
 综合媒体报导，事发在5日早上，有约82人遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绑架.html" target="_blank">
  绑架
 </a>
 ，其中除了校长、一名老师和校车司机，其余包括79名年龄介乎11至17岁的学生。
 <br/>
 <br/>
 从视频中可见，学生疑在清晨起床前被集体绑架，现场床褥衣物散乱，有荷枪实弹警察在学校四周高度戒备。军方一名消息人士对路透社表示：〝人质被带到树林里。〞
 <br/>
 <br/>
 绑匪放话，只有达成建立新国家的目标后才会放人。当地官员表示，已要求军队尽快救回人质。
 <br/>
 <br/>

 <br/>
 <br/>
 独立份子在占领区内实施宵禁，所有公立学校被迫关闭，但还是有民营学校冒险收学生。
 <br/>

 而这起

 </a>
 事件选在总统毕亚（Paul Biya）就职典礼的前一天发动攻击，挑衅政府意味浓厚，而
