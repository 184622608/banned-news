### G20川习峰会 美中就贸易战停火达成协议
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】首先来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会。美国总统川普和中国国家主席习近平，星期六晚，就中美双方对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停火.html" target="_blank">
  停火
 </a>
 达成有条件协议：包括从明年1月1日起，90天内美国将维持对2000亿中国进口商品加征10%关税，而不提高至25%，在此期间中方将进行一系列的改革，否则仍将出口面临关税上涨。
 <br/>
 <br/>
 根据协议，1月1日起美国不对中国产品加征更多关税；而中国则同意购买大量的美国产品，包括立即购买美国农产品。白宫声明中还表示，两位领导人还同意立即就中共强制技术转让、知识产权保护、非关税壁垒、网络入侵及网络盗窃、服务和农业等方面的〝结构性改变〞展开谈判。
 <br/>
 <br/>
 双方同意将在未来90天内完成协议内容；否则，美方将对进口于中国2000亿商品的关税提高至25%。
 <br/>
 <br/>
 在川习会后的新闻发布会上，中共外交部长王毅表示，会谈的重点是〝取消所有美国关税和中国报复性关税〞，但王毅没有谈及〝截止日期〞和中方让步的具体内容。
 <br/>
 <br/>
 王毅表示：〝中国愿意根据国内市场的需求和人民的需求增加进口，包括来自美国的可销售产品，以逐步缓解双向贸易的不平衡。〞
 <br/>
 <br/>
 王毅还表示：〝双方同意互相开放市场，随着中国推进新一轮改革，美国的合理担忧可以逐步解决。〞
 <br/>
 <br/>
 此外，在川普总统的敦促下，中国同意将芬太尼列为〝管控物质〞，这意味着向美国出售芬太尼的人将依法受到中国的重罚。
 <br/>
 <br/>
 川普在声明中表示，“这次会谈很棒很有成效，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中.html" target="_blank">
  美中
 </a>
 两国提供了无限的机会”。
 <br/>
 <br/>
 新唐人记者徐哲、雨薇综合报导
</div>
