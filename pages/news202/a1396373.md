### 10月22日简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】1.美菲发表联合声明 加强双边贸易与合作
 <br/>
 <br/>
 美国贸易代表莱特希泽和菲律宾外交部长周一发表联合声明，将加强双边贸易活动，在若干领域展开合作，使两国农民、进出口商以及消费者从中受益。
 <br/>
 <br/>
 外界认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 退出跨太平洋伙伴关系协定后，可能会和菲律宾达成东南亚地区的第一个双边协议。2017年，美国对菲律宾货物出口总额为85亿美元。
 <br/>
 <br/>
 2. 美将推出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新非洲战略.html" target="_blank">
  新非洲战略
 </a>
 创造就业 带去繁荣
 <br/>
 <br/>
 美国国务院非洲事务助理国务卿蒂伯・纳吉向媒体表示，川普政府的新非洲战略侧重在为非洲创造良好投资环境，给当地人带来更多工作机会。上周五，美国国务卿蓬佩奥称中共在非洲进行〝掠夺性行为〞，中共在外国投资时带有附加条件，在被投资国家无法还债时便施加束缚。
