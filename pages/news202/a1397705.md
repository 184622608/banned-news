### 印尼坠机捞出首个黑匣子 搜救工作持续
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】周四，印尼狮航航班坠海失事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_搜救工作.html" target="_blank">
  搜救工作
 </a>
 进入第四天，失事飞机首个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑匣子.html" target="_blank">
  黑匣子
 </a>
 被打捞出水，调查人员希望借此能找出飞机坠毁的原因。
 <br/>
 <br/>
 周四上午，印尼海军陆战队潜水员打捞出失事飞机
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑匣子.html" target="_blank">
  黑匣子
 </a>
 中的飞行数据记录仪(FDR)，并全力搜寻另一个黑匣子，驾驶舱语音记录仪。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇难者.html" target="_blank">
  遇难者
 </a>
 家属 Johanes Sayersz：找到黑匣子让(遇难者)家人感到宽慰，因为我们能够知道到底发生什么了。
 <br/>
 <br/>
 周一，狮航JT610航班起飞后13分钟，与地面失去了联系，专家分析飞机当时高速坠入海中。坠毁前飞机并没有发出求救信号，但飞行员曾在起飞后要求返回机场。
 <br/>
 <br/>
 截至目前，打捞人员只打捞上一小部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞机残骸.html" target="_blank">
  飞机残骸
 </a>
 和乘客的个人物品。寻获的56袋尸体残骸已移交给印尼警察总部医院进行身份鉴定。
 <br/>
 <br/>
 一架水下无人机探测到一个物体，据推测可能是飞机主体的一部分。如果是，那么大多数遇难者的尸体都会被发现。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
