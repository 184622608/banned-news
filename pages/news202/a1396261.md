### 成千难民进入墨西哥 川普重申保护边境
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】再来关注中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 潮情况。周日，部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 难民滞留在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_危地马拉.html" target="_blank">
  危地马拉
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 边境，另一部分已经进入墨西哥。危地马拉政府出动警力，协助约2000名资源回乡的难民回到洪都拉斯。
 <br/>
 <br/>
 周日黎明，成千的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 ，进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 后，继续向北跋涉。
 <br/>
 <br/>
 他们周六，游泳或用皮筏，拖家带口，越过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_危地马拉.html" target="_blank">
  危地马拉
 </a>
 和墨西哥边境。
 <br/>
 <br/>
 洪都拉斯难民 Maykol：〝我们冒着巨大风险过河，冒着生命危险，因为在洪都拉斯没有美好的生活，没未来，我们将付出努力继续前进。〞
 <br/>
 <br/>
 墨西哥官员称，周六有640人跨河入境。
 <br/>
 <br/>
 被困在墨西哥与危地马拉边境的难民，生活状况堪忧，有的居住在临时搭建的帐篷中，有的只能露天席地而坐，还有体力不支的人昏倒在地。
 <br/>
 <br/>
 洪都拉斯难民 Amabilia Piril：〝从昨天开始孩子们脱水，营养不良，我们经历了很多，可怜我们和孩子吧，我们已无法承受了。〞
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 已经警告墨西哥、危地马拉以及洪都拉斯，阻止难民进入美国，否则将停止援助，甚至关闭美墨边境。
 <br/>
 <br/>
 危地马拉和墨西哥加紧阻止难民，一些难民决定返回家园。
 <br/>
 <br/>
 洪都拉斯难民 Javier Portillo：〝因为我们没办法如愿，完成美国梦，无法达成，所以选择回洪都拉斯。〞
 <br/>
 <br/>
 危地马拉总统 Jimmy Morales：〝我们仍需要和洪都拉斯确定，约有2,000人已安全返回洪都拉斯。〞
 <br/>
 <br/>
 不过，危地马拉总统莫拉莱斯表示，约有5000-5400人，寻求进入墨西哥边境。
 <br/>
 <br/>
 川普总统周五在亚利桑那州集会上，谈到大篷车难民潮，批评民主党的移民政策。
 <br/>
 <br/>
 美国总统 川普：〝我们在这里説话的时候，民主党正在公开邀请数百万的非法移民，让他们来破坏我们的法律、进犯我们的边境，并击溃我们的国家。〞
 <br/>
 <br/>
 川普再次强调，不管对手如何阻拦，也要修建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境墻.html" target="_blank">
  边境墻
 </a>
 。
 <br/>
 <br/>
 新唐人记者宇亭综合报导
</div>
