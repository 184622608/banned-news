### 乌克兰戒严加强边防 俄延长拘押24船员
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】再来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 扣押
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰军舰.html" target="_blank">
  乌克兰军舰
 </a>
 及船员的事件。周三，乌克兰开始实施戒严，以防潜在的边境袭击。美方则希望事态不要升级。
 <br/>
 <br/>
 周三实施戒严后，乌克兰总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波罗申科.html" target="_blank">
  波罗申科
 </a>
 （Petro Poroshenko）视察边防。
 <br/>
 <br/>
 乌克兰总统 波罗申科：〝我们必须确保有效的人员培训。在需要时，击败敌人。〞
 <br/>
 <br/>
 目前，乌俄双方的边境控管都有加强，但平民穿行边界还没有遇到问题。
 <br/>
 <br/>
 同一天，美国的乌克兰特使沃尔克（Kurt Volker）表示，希望事态不要再升级，并要求俄方释放乌克兰船隻人员。
 <br/>
 <br/>
 同时，沃尔克质疑，2014年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 占领克里米亚后，欧洲没有严格落实对俄制裁。
 <br/>
 <br/>
 美国乌克兰特使 沃尔克：〝我的理解是在克里米亚做生意的公司，包括俄罗斯公司，不能在欧盟做生意。我不知道这个标准是否得到了尽可能的严格维护。〞
 <br/>
 <br/>
 此外，沃尔克（Kurt Volker）提到，美国前奥巴马政府没有给乌克兰可致命的防御设备，但现在情况改变。
 <br/>
 <br/>
 美国乌克兰特使 沃尔克：〝所以本届美国政府所做的就是取消禁令，可以提供可致命防御设备。〞
 <br/>
 <br/>
 周三，俄罗斯一个法庭裁决，对扣押的全部24名乌克兰船员，拘留期延长2个月。俄总统普京并在事件后首次发声，指责是乌克兰〝策划〞了此次事件，是政治家为提高支持率。
 <br/>
 <br/>
 不过此事件后，美国总统川普周二表示考虑取消G20与普京的会晤，但普京周三提出，希望与川普会面。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
