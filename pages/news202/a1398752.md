### 聪明得令人心疼 柬埔寨男童操8国语言兜售商品（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】近日，一段拍摄于柬埔寨旅游景区的视频片段引起众网民的极大兴趣。在短短几分钟的影片中，一个小男孩流利地使用粤语、中国普通话、英语、韩语、泰语、法语、马来语、菲律宾语向游客兜售他篮子中的纪念品，让一众中国网友佩服得几乎要五体投地。
 <br/>
 <br/>
 网络上曝光的视频片段，据称拍摄于柬埔寨暹粒市著名的观光景区吴哥窟。
 <br/>
 <br/>
 视频画面上，一个年约7岁的小男孩身穿红色T恤衫，挎着一个装满了旅游纪念品的蓝色篮子，紧跟着游客一边走着，一边向一名女游客推销他的商品。
 <br/>
 <br/>
 当女游客对小男孩会说中国话表示赞赏时，小男孩满不在乎地说，〝我会讲广东话、普通话、英语、泰语、韩语和法语〞。
 <br/>
 <br/>
 女游客听到他会法语，立即用法语对他说〝Je fais coucou（你好）〞，不料这孩子一听此话，立刻开始使用法语兜售他的商品。
 <br/>
 <br/>
 接着女游客试着对孩子讲粤语，小男孩立刻就开始用粤语来介绍他手中的商品售价。然后女游客故意又改用日语向他问话，孩子就又改用日语流利地夸赞自己的商品有多好，并热切地用中国话请求对方〝好买一个啦，1块美金啊。〞
 <br/>

 <br/>
 在女游客的要求下，男孩又改用泰语继续推销他手中的纪念品。为了促销，他还唱了一首中国歌曲《我们不一样》，音准和吐字都相当准确，终于打动了女游客，花了两美元购买了5个纪念品。
 <br/>
 <br/>
 最后，小男孩还另外秀了一下马来语和菲律宾语。细心的网友数了数，发现加上柬埔寨国语高棉语，小男孩一共会说8个国家的语言。
 <br/>
 <br/>
 看过视频的网友们纷纷为小男孩惊人的语言天赋点赞，并留言表达对这个孩子的喜爱与折服。
 <br/>
 <br/>
 有网友大呼：〝不可思议。〞有网友惊叹：〝真的很厉害，真的要跪了！〞有网友笑称：〝简直是快速翻译机。〞
 <br/>
 <br/>
 还有网友留言称：〝这个遇到，直接给他一块美金。〞有网友感叹：〝这要好好栽培肯定不得了。〞
 <br/>
 <br/>
 更有网友赞叹道，〝聪明得令人心疼。〞还有网友留言问：〝可以领养吗？〞
