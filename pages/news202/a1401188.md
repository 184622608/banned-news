### 11月28日全球看中国

 【新唐人2018年11月28日讯】下面是今天的全球看中国
 <br/>
 <br/>
 1. 德国之声
 <br/>
 超过200名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌员工.html" target="_blank">

 <br/>
 <br/>
 2. 香港01
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新西兰.html" target="_blank">
  新西兰
 </a>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 5G设备的提议。此外，澳大利亚、加拿大、英国等，都相继对使用华为设备提出禁令或警告。
 <br/>
 <br/>
 3. 大纪元时报
 <br/>
 日前，辽宁省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_610办公室.html" target="_blank">
  610办公室
 </a>
 一份下令升级打压法轮功等团体的文件曝光，美国国际宗教自由委员会官员表示，文件内容〝令人震惊〞，〝要让中共知道美国正在看着，美国对此关注〞，中共对法轮功的迫害必须停止。
 <br/>
 <br/>
 4. 自由亚洲电台
 <br/>
 北京维权人士倪玉兰今年八月被房地产公司拉入〝黑名单〞，女儿几经周折为她找到住处，这是她五年来第二十一次搬家。倪玉兰曾是一名执业律师，因自家房屋遭强拆而走上维权之路，并多次发声支持709被捕律师。
