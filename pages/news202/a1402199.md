### 华为CFO孟晚舟被捕 陆港台股市大跌
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】最新消息，加拿大政府应美国要求，于12月1日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_温哥华国际机场.html" target="_blank">
  温哥华国际机场
 </a>
 ，逮捕了中国华为创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的女儿、华为首席财务官孟晚舟。消息传出后，大陆香港和台湾的股市大跌。
 <br/>
 <br/>
 孟晚舟涉嫌违反美国对伊朗的制裁，可能被引渡到美国，于12月7日出席法庭聆讯。
 <br/>
 <br/>
 华为副董事长孟晚舟在加拿大被捕的消息传出后，今天(周四) 大陆香港和台湾股市大跌。
 <br/>
 <br/>
 香港恒生指数 HSI 挫跌2.1%, 大陆沪综指数 SSEC 跌0.9%。
 <br/>
 <br/>
 台股开盘也摔200点，包括台积电、大立光、联发科、联亚、联钧、全新与台耀等，相关概念股重摔。另外，大立光昨天公布11月营收，来到7个月新低，股价今天跳空大跌220元开盘，股王失足，也是拖累苹概股今天走弱的原因。
 <br/>
 <br/>
 日经指数N225也下跌1.7%。
 <br/>
 <br/>
 新唐人记者陈依心、天羽综合报导
</div>
