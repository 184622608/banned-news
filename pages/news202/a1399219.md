### 彭斯访日后前往东盟会 推自由开放印太
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】周二，美国副总统彭斯与日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 ，就朝鲜、贸易及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印太地区.html" target="_blank">
  印太地区
 </a>
 合作等议题，进行探讨。随后，彭斯启程前往新加坡参加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东盟峰会.html" target="_blank">
  东盟峰会
 </a>
 。
 <br/>
 <br/>
 彭斯与安倍单独会谈后，举行联合记者会。彭斯重申，在朝鲜彻底实现去核化之前，美国针对平壤的施压与制裁，不会有丝毫松懈。
 <br/>
 <br/>
 美国副总统 彭斯：〝川普总统相信他与金正恩的关系良好。自今年6月在新加坡举行历史性峰会后，我们在协议方面取得良好进展，但仍有很多工作要做。在朝鲜最终实现可充分验证的无核化之前，我们对其制裁力度不变。〞
 <br/>
 <br/>
 彭斯同时也向日本施压，要求其削减对美国的贸易顺差。安倍则回应称，将依照日美9月联合声明，以互利的方式，进一步扩大双边贸易和投资。
 <br/>
 <br/>
 另外，彭斯在记者会上强调，美国致力于维护一个〝自由开放的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印太地区.html" target="_blank">
  印太地区
 </a>
 〞，不会容忍任何国家在当地奉行威权主义，以及侵略他国主权的行为。
 <br/>
 <br/>
 美国副总统 彭斯：〝美国寻求合作而不是控制。我们希望印度太平洋的每个国家，可以自由地选择发展道路，追求本国利益。我们不会容忍在印太地区的威权和侵略行为。〞
 <br/>
 <br/>
 彭斯的讲话虽然没有直接点名批评中共，但暗示意味明显。有日媒指，彭斯抗衡中共专制的态度坚定而明确。
 <br/>
 <br/>
 在结束日本的访问之后，彭斯搭机抵达新加坡，代表川普总统出席第三十三届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东盟峰会.html" target="_blank">
  东盟峰会
 </a>
 。
 <br/>
 <br/>
 除东盟十国之外，印度、俄罗斯、中国和日本也将出席本届峰会。其中，李克强和彭斯是否会在会议间隙举行单独会谈，是外界关注的焦点之一。
 <br/>
 <br/>
 新唐人记者曹菁综合报导
</div>
