### 川普警告 英脱欧或影响双边贸易
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】美国总统川普周一表示，英国的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 〞，可能损害未来美英贸易关系。但英国首相发言人强调，脱欧后英国可以和其他国家自由贸易。
 <br/>
 <br/>
 川普总统表示，欧盟刚签署的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 〞对欧盟来说很好，但可能会影响美英贸易。
 <br/>
 <br/>
 美国总统川普：〝我认为我们要认真审视，（脱欧后）英国是否允许贸易。现在看这项协议，他们可能不能和我们有贸易往来，这不是件好事。〞
 <br/>
 <br/>
 周日，欧盟27国领导人签署了〝脱欧草案〞。英国首相特雷莎・梅需要在12月11日前，获得多数议员支持，才能保证〝脱欧草案〞在议会顺利通过。
 <br/>
 <br/>
 美国总统川普：〝我想英国不愿伤害美英贸易，英国首相不想这样，希望她能对此做些什么。〞
 <br/>
 <br/>
 川普总统没有阐明，他说是的哪一份贸易协议。
 <br/>
 <br/>
 美国总统川普：〝就目前协议而言，她可能不会、他们可能不能和美国进行贸易。我认为他们不想这么做。那将对（美英贸易）协议产生很大的负面影响。〞
 <br/>
 <br/>
 英国首相特雷莎・梅的发言人周一（11月26日）表示，〝脱欧草案〞允许英国和包括美国在内的其它国家签署独立的贸易协议，他还表示，英美双方代表就未来贸易协议的准备工作已经召开了五次会议。
 <br/>
 <br/>
 周二（11月27日），英镑兑美元和欧元应声下跌，跌到近两周来最低点。
 <br/>
 <br/>
 特蕾莎・梅周二（11月27日）开始在英国巡回，希望获得更多对〝脱欧草案〞的支持。她首战将到达北爱尔兰和威尔士。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
 <br/>
