### 出生六天萌化 德国公布北极熊宝宝画面
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柏林动物园.html" target="_blank">
  柏林动物园
 </a>
 最近公布了出生六天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北极熊宝宝.html" target="_blank">
  北极熊宝宝
 </a>
 画面，偎在妈妈柔软的皮毛里，模样让人萌化。
 <br/>
 <br/>
 毛绒绒，像一只豚鼠大小，出生六天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北极熊宝宝.html" target="_blank">
  北极熊宝宝
 </a>
 ，在妈妈怀里蠕动着。
 <br/>
 <br/>
 柏林第二大动物园公布了12月1日新添的小北极熊的红外线影像，但是它只有50%的存活机会。所以要隔离保护至少六周。
 <br/>
 <br/>
 北极熊母子被安置在安静的环境，类似野生北极熊出生后的环境：待在雪下面很深的熊窝里，一连几个月只吃母熊的奶水。
 <br/>
 <br/>
 近年来，动物园出生的多只北极熊都夭折了。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柏林动物园.html" target="_blank">
  柏林动物园
 </a>
 最瞩目的北极熊努特，人工一手养大的，但是2011年四岁的时候染病去世。
 <br/>
 <br/>
 目前新出生的熊宝宝状态良好，熊妈妈今年三月到九月体重从230公斤增加到390公斤，给他它准备了充足的养料。如果顺利，春天就会与公众见面啦。
 <br/>
 <br/>
 新唐人电视综合报导
</div>
