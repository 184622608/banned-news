### 没人信中共 王岐山新加坡演讲现尴尬一幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】在经济陷入危局之际，北京再次宣称〝开放市场〞，愿意和美国磋商贸易。但北京失信几十年后，国际社会很难再相信其空头承诺。美学者透露，6日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新加坡.html" target="_blank">
  新加坡
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_创新经济论坛.html" target="_blank">
  创新经济论坛
 </a>
 的与会者，普遍不愿对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 的讲话表达信任。
 <br/>
 <br/>
 11月6日，中共国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新加坡.html" target="_blank">
  新加坡
 </a>
 圣淘沙嘉佩乐酒店举行的首届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_创新经济论坛.html" target="_blank">
  创新经济论坛
 </a>
 上发表演讲，再次承诺〝坚持扩大开放〞，强调〝中美互利共赢〞，并表示愿意和美方就经贸问题展开磋商，推动达成一个〝双方都能接受的方案〞。
 <br/>
 <br/>
 不过，美国智库战略与国际研究中心中国研究项目主任甘思德（Scott Kennedy）在推特（Twitter）上透露，王岐山演讲之后，欧亚集团（Eurasia Group）创始人布雷默（Ian Bremmer）曾问在场的与会者，如果认为王的讲话展示了北京进一步自由化和做出妥协的意愿，请举手。
 <br/>
 <br/>
 结果，几乎没有人举手。
 <br/>
 <br/>
 《美国之音》文章指，北京在高喊〝改革开放〞几十年后，再度保证〝开放市场〞，与世界共享经济发展的成果，这种〝口号〞已经不能打动西方了。
 <br/>
 <br/>
 事实上，北京加入世贸组织之后的十几年间，拒不履行开放中国市场的承诺，反而肆意盗窃外国技术产权，输出市场扭曲。但是面对中共的经济实力和层出不穷的流氓经贸手段，多数国家和企业都敢怒不敢言。
 <br/>
 <br/>
 今年以来，世界第一大经济体——美国强势打响中美贸易战，很快得到国际社会的群起响应。公开表达对北京的真实态度，已经没有那么大压力。
 <br/>
 <br/>
 王岐山在新加坡的遭遇，和目前上海进口博览会面临的状况相似。
 <br/>
 <br/>
 这场由北京最高当局强力推动一年有余的〝盛会〞，最终只邀请到十几个国家的首脑与会。用《纽约时报》的话说，来的大部分都是欠了北京很多钱的国家。很多网友称之为〝丐帮大会〞。
 <br/>
 <br/>
 不仅如此，〝欠了北京很多钱〞的肯尼亚总统（Uhuru Kenyatta）在5日早上的进博会开幕式上，还痛批中共贸易不公。
 <br/>
 <br/>
 他说，中肯贸易在过去10年里翻了8翻，〝但是，贸易严重地向中国（中共）有利的一面倾斜。〞
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
