### 法国救护车堵塞协和广场 税改危机加剧
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国救护车.html" target="_blank">
  法国救护车
 </a>
 司机和大学生，也加入连日的抗议，反对马克龙推行的医疗改革和燃料税。法国总理会见最大反对党领袖，寻求解决危机的方法。法国财长表示，政府必须加速减税。
 <br/>
 <br/>
 周一清晨，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国救护车.html" target="_blank">
  法国救护车
 </a>
 司机进行第三次抗议，反对马克龙推行的医疗改革，指责这将损害小救护车公司的利益。
 <br/>
 <br/>
 数十辆救护车拉响警报，将协和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广场.html" target="_blank">
  广场
 </a>
 通往国民议会的道路堵得水泄不通。抗议者表示，80%的救护车公司面临倒闭危险。
 <br/>
 <br/>
 救护车司机 RACHID SOUKHMANI：〝一些救护车公司损失了30%至70%的收益，影响巨大。〞
 <br/>
 <br/>
 周一，巴黎郊区、Aubervilliers地区的高中生也走上街头，抗议政府改革。路上汽车被焚烧，店铺门窗被砸，货品散落一地。
 <br/>
 <br/>
 法国总统马克龙急于解决全国的大规模抗议 ，周一，他任命的总理爱德华・菲利普会见法国反对派、最大极右派政党领袖瓦奎兹 。
 <br/>
 <br/>
 瓦奎兹会谈后表示，政府没能理解和解决国民的愤怒，唯一办法是取消燃料增税。
 <br/>
 <br/>
 法国财长布鲁诺・勒梅尔周一表示，要解决当前危机，政府必须加速减税。
 <br/>
 <br/>
 法国财政部长布鲁诺・勒梅尔：〝因此结果是，我们必须加快减税计划。〞
 <br/>
 <br/>
 勒梅尔还表示， 为了减税，政府必须加速削减公共开支。
 <br/>
 <br/>
 法国财政部长布鲁诺・勒梅尔：〝减少公共开支，就可以减少税收。〞
 <br/>
 <br/>
 法国连日来的大规模抗议，导致酒店预订下降，零售业受损，旅游和运输行业市场活动下降。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
</div>
