 城市伊斯坦布尔，与土耳其总统埃尔多安就

  叙利亚
 </a>
 危机举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四方峰会.html" target="_blank">
  四方峰会
 </a>
 。
 <br/>
 <br/>
 这次峰会前一天，叙利亚政府军对西北部的伊德利卜（Idlib）省进行了空袭，导致7人死亡。这是俄罗斯和土耳其同意在这个地区建立非军事区以来，单日死亡人数最多的一天。
 <br/>
 <br/>
 土耳其长久以来支持目标推翻阿萨德政权的反叛力量；一个月前，土耳其与阿萨德的主要外国盟友俄罗斯达成协议，在伊德利卜建立非军事区，由两国军队在此巡逻。目前伊德利卜和临近地区是反叙利亚政府武装控制的最后一个据点，同时也是300万平民的家园。
 <br/>
 <br/>
 而在今年一月俄罗斯举行的叙利亚和平会议上，与会国同意叙利亚需要一部新的宪法，新宪法有150人的委员会完成，其中三分之一由阿萨德政府任命、三分之一由反对派组织任命、另三分之一有联合国任命。
 <br/>
 <br/>
 叙利亚政府对伊德利卜的空袭，突显俄罗斯与土耳其非军事区协议的脆弱。
 <br/>
 <br/>
 叙利亚危机自2011年爆发以来，因各方寻求的〝国家〞利益不同，很难找到合作点。法国、德国和美国均希望各方能够求同存异，为走出危机找到政治解决途径。
 <br/>
 <br/>
 新唐人记者徐哲、天羽综合报导
