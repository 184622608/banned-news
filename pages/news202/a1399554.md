### 日旅巴司机突晕倒蛇行 乘客控方向盘救下全车人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】日本一辆载33名参加〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神秘之旅.html" target="_blank">
  神秘之旅
 </a>
 〞的旅客
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观光巴士.html" target="_blank">
  观光巴士
 </a>
 ，周四（15日）由爱知县丰桥市出发，行经三重县时一条
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_隧道.html" target="_blank">
  隧道
 </a>
 时，司机突然失去意识还
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_口吐白沫.html" target="_blank">
  口吐白沫
 </a>
 ，令巴士在隧道内S形，惊险万分。幸好坐在前排两名乘客合力控制方向盘，最后巴士慢慢擦撞墙壁停了下来，全车乘客幸运的逃过一劫。
 <br/>
 <br/>
 事发时，巴士开进全长2,610公尺的高丸山
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_隧道.html" target="_blank">
  隧道
 </a>
 后，不久巴士就失控的S形前行。
 <br/>
 <br/>
 坐在前排的两名乘客上前一看，发现46岁司机全身抽蓄、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_口吐白沫.html" target="_blank">
  口吐白沫
 </a>
 ，立即将司机的脚从油门上移开，抓紧巴士方向盘，闪避数辆汽车后，慢慢擦撞左方墙壁，让巴士停了下来。
 <br/>
 <br/>
 警方获报后到场将司机送院，并将旅游巴士移离隧道，恢复交通顺畅。巴士司机则已恢复意识没有生命危险。
 <br/>
 <br/>
 据了解，这趟〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神秘之旅.html" target="_blank">
  神秘之旅
 </a>
 〞参加者仅知道目的地为和歌山县串本町，其他行程细节均不了解。事后，一对70多岁的夫妇受到惊吓，决定取消旅行；但其他乘客则乘搭另一旅巴继续行程。
