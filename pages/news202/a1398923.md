### 澳洲水果藏针引恐慌 警逮捕一名50岁女子
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月11日讯】澳洲多地此前发生过数十宗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_水果藏针.html" target="_blank">
  水果藏针
 </a>
 事件，引起全国恐慌。经相关执法单位展开全国调查后，于今（11日）拘捕一名50岁涉案女子。目前警方尚未公布后续详情，包括疑犯身份、动机及面临的控罪。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_水果藏针.html" target="_blank">
  水果藏针
 </a>
 事件于9月发生，主要涉及超市出售的盒装
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_草莓.html" target="_blank">
  草莓
 </a>
 (士多啤梨)，一名男子因进食藏针的草莓而入院。随后陆续有人将超市购买的草莓插入刺针的图片及视频放到社交网站。
 <br/>
 <br/>
 而〝草莓藏针〞事态也逐渐发酵，9月17日开始有人在香蕉中也发现有藏针；18日，新南威尔斯州又有澳洲民众发现在超市购买的苹果中也被藏了针。
 <br/>
 <br/>
 事件导致澳洲多地的超级市场回收水果及下架，果农更倾倒大批草莓。为此昆士兰政府悬赏10万澳元（约7.2万美元），并提高有关罪行的刑罚，由入狱10年增至15年。呼吁民众提供线索抓捕嫌犯。
 <br/>
 <br/>
 昆士兰警方日前与多个政府、执法及情报部门合作，展开全国缉凶行动。经复杂及广泛的调查后锁定疑犯，被捕女子将于周一（12日）在布里斯班出庭。
