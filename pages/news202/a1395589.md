### 记者失踪案调查持续 蓬佩奥访沙特
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】首先来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 失踪案的最新进展。周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 警方结束领馆搜查。调查结果可能对沙特不利，沙特将承认审讯卡舒吉，导致其意外死亡，是误杀。美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 周一赶往沙特，会见了沙特国王、王储和沙特外交部长。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 警方调查人员周一晚间进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 阿拉伯驻伊斯坦布尔领事馆，进行了9个小时的搜查，周二一早离开。
 <br/>
 <br/>
 接下来，土耳其调查人员将扩大搜索范围，计划周二搜查沙特领事的住所。
 <br/>
 <br/>
 土耳其电视台曾播出录像，一辆大型轿车在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 进入领事馆两小时后离开领事馆，随后停在领事的住所。
 <br/>
 <br/>
 沙特记者卡舒吉，经常批评沙特阿拉伯时政，去年流亡美国。他在10月2日进入沙特阿拉伯驻伊斯坦布尔领事馆后失踪。
 <br/>
 <br/>
 事件引发国际关注。
 <br/>
 <br/>
 周一，美国总统川普和沙特国王通话，随后，国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 前往沙特。
 <br/>
 <br/>
 周二上午，蓬佩奥在利雅得与沙特国王会面，重申美国对卡舒吉失踪的担忧。
 <br/>
 <br/>
 蓬佩奥还会见了沙特王储萨勒曼。卡舒吉曾撰文批评王储，引起王储的不满。
 <br/>
 <br/>
 蓬佩奥周三将前往伊斯坦布尔实地了解案情。
 <br/>
 <br/>
 土耳其与沙特的联合调查仍在进行中。
 <br/>
 <br/>
 土耳其总统埃尔多安周二表示，警方在领事馆发现一些表面被重新涂过的痕迹。
 <br/>
 <br/>
 土耳其总统埃尔多安：〝调查正在研究许多事情，例如有毒物质和那些被重新涂漆去除的部分。这些将有助于得出结论。〞
 <br/>
 <br/>
 土耳其官员称，他们有视频和音频等证据，证明卡舒吉被利雅得派出的15人袭击小组审讯和谋杀。
 <br/>
 <br/>
 沙特阿拉伯国王否认知情，但外传沙特正在准备一份报告，承认卡舒吉在审讯中意外死亡。
 <br/>
 <br/>
 卡舒吉的家人发表声明，要求国际社会对此案进行调查，建立一个独立、公正的国际委员会，彻查案件。
 <br/>
 <br/>
 新唐人记者李梅、李静明综合报导
</div>
