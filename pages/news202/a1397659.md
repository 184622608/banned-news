### 韩朝停止海陆空敌对行为 保留驻韩美军
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜半岛.html" target="_blank">
  朝鲜半岛
 </a>
 从11月1日起，全面停止陆海空一切敌对行动。与此同时，美国与韩国在五角大楼达成协议，确认
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_半岛去核化.html" target="_blank">
  半岛去核化
 </a>
 后，美军将继续留守韩国。
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜半岛.html" target="_blank">
  朝鲜半岛
 </a>
 和平进程又推进一步。
 <br/>
 <br/>
 韩国国防部表示，朝韩从11月1日零时起，全面停止陆海空一切敌对行动。包括禁止在军事分界线实施军事演习，划设
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁飞区.html" target="_blank">
  禁飞区
 </a>
 ，停止在东部和西部海域缓冲区进行火炮射击。
 <br/>
 <br/>
 周四，韩国总统文在寅向国会说明这一进展。
 <br/>
 <br/>
 韩国总统文在寅：〝朝鲜南北方通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事协议.html" target="_blank">
  军事协议
 </a>
 ，完全消除了军事冲突的危险。〞
 <br/>
 <br/>
 此举是落实韩朝峰会期间签署的《9・19军事协议》，双方承诺通过和平方式解决所有军事冲突，在任何情况下都不动用武力。协议得到联合国军司令部的支持。
 <br/>
 <br/>
 美国国防部长马蒂斯和韩国防长郑景斗，周三在五角大楼达成协议，确认
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_半岛去核化.html" target="_blank">
  半岛去核化
 </a>
 后美军将继续留守，帮助韩国抵御外来威胁。
 <br/>
 <br/>
 双方确定，美韩联合部队在战时作战指挥权移交后，由韩国将军级官员担任美韩联军司令部司令，美国将军级军官则担任副司令。
 <br/>
 <br/>
 马蒂斯强调，目前并不是完全叫停所有合作与军事演习，暂停大型军事演习是为支援外交谈判。
 <br/>
 <br/>
 新唐人记者李梅、宇微综合报导
</div>
