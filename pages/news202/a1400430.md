### 见证美国诞生 漫谈〝感恩节〞来历及风俗
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国人.html" target="_blank">
  美国人
 </a>
 来说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 是一个意义深远的节日。这个节日有着什么样的起源和习俗呢？随记者来了解一下。
 <br/>
 <br/>
 1620年，从英国出发的五月花号船满载着失去土地的农民，生活艰苦的工人以及受到宗教迫害的清教徒到达美洲。那一年的冬天，不少人因为饥寒，染病身亡。在印第安人的帮助下，这些新移民学会了狩猎、捕鱼，他们播种下的南瓜，玉米和小麦在来年的秋天获得了大丰收。在欢庆丰收的日子，欧陆的新移民们烤好火鸡，玉米跟南瓜馅饼，邀请帮助他们的本地印第安人一同庆祝，感谢上帝的恩赐。从那以后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国人.html" target="_blank">
  美国人
 </a>
 将每年11月的第四个星期四定为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 ,为感谢上帝赐予每一年的丰收祝福。
 <br/>
 <br/>
 感恩节这天，举国上下热闹非凡，人们去教堂祈祷，举行盛大的化妆游行，其中最著名的就是从1924年开始的梅西百货感恩节游行。在这一天，许多社区还会将不易腐烂的食物打包后分发给穷人。分别了一年的亲人们也会从天南海北回家团聚，品尝以火鸡为主的感恩节美食。
 <br/>
 <br/>
 火鸡是感恩节的传统主菜，人们通常会把火鸡肚子里塞上各种香料拌好的面包碎块，然后整只烤出，由男主人将火鸡切成薄片，再由女主人分送给大家。传统的火鸡配菜有土豆，蔓越莓果酱，甜玉米和各种蔬菜。除此之外，南瓜馅饼也是必不可少的甜品之一。
 <br/>
 <br/>
 宴会过后，人们会做些传统的游戏，比如将火鸡胸部的叉骨取出，两人各执一端，默念心中的愿望，直到折断，拿到较长一端骨头的人愿望就会实现。
 <br/>
 <br/>
 1989年的感恩节，当时的美国总统老布什曾收到一只火鸡作为贺礼，但他并不打算将牠宰杀，还公开在白宫玫瑰园幽默地说：〝它现在已经得到总统的赦免，可以附近的农场度过余生。〞从那以后，每一位美国总统都延续在感恩节前〝赦免〞火鸡。今年川普总统赦免的两只火鸡，一只叫豌豆仔，一只叫萝卜丝，它们会在弗吉尼亚理工学院得到饲养，安度余生。
 <br/>
 <br/>
 感恩节的次日便是〝黑色星期五〞，为了购买到物美价廉的商品，许多人家还会在团聚后全家出动，昼夜排队，这也逐渐成为一种新的感恩节习俗。
</div>
