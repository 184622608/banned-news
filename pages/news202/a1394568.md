### 二战纳粹空拍照藏线索 意外发现苏联〝万人冢〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】近日，俄罗斯历史学家和考古学家从一名纳粹轰炸机飞行员拍下的照片中透露的线索，找出了莫斯科郊外的又一个苏联
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万人冢.html" target="_blank">
  万人冢
 </a>
 的确切地点，那里埋有数千被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏联秘密警察.html" target="_blank">
  苏联秘密警察
 </a>
 NKVD枪杀的人。在研究考察中，学者们发现NKVD为了掩盖罪证特意在一些坟冢上种植了树木。
 <br/>
 <br/>
 据台湾中央社报导，最近俄罗斯历史学家与考古学家透过一名纳粹轰炸机飞行员的空拍照片，找出了又一个莫斯科〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万人冢.html" target="_blank">
  万人冢
 </a>
 〞的确切地点，这里埋有数千名遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏联秘密警察.html" target="_blank">
  苏联秘密警察
 </a>
 杀害的人。
 <br/>
 <br/>
 苏联即将解体之际，苏联情报机构国家安全委员会（KGB）曾对外开放档案，外界因此得以知道这莫斯科西南部科穆纳尔卡地区（Kommunarka）存在不少〝万人冢〞，那是苏联的秘密警察NKVD处决各种政治犯的刑场所在地。
 <br/>
 <br/>
 这次被曝光的〝万人冢〞则是独裁者史达林掌权时期秘密警察NKVD于1930年代的的莫斯科3个处决场之一。
 <br/>
 <br/>
 苏联时期，这些〝万人冢〞的地点是未公开的国家机密。但苏联解体后一名纳粹战机飞行员在1942年飞越莫斯科拍下的科穆纳尔卡空拍照，意外为史学家和考古学家调查苏联秘密警察当年所犯罪恶提供了关键的线索。
 <br/>
 <br/>
 这张1942年的空拍照片显示出该地区当时的树木高度，学者在比对研究后发现，有些树木是被种在新坟墓上的，这是NKVD常见的掩盖恶行手法。
 <br/>
 <br/>
 历史学家认为，在1937至1941年期间，至少有6609人遭枪杀后丢入科穆纳尔卡的万人冢。
 <br/>
 <br/>
 领导这项调查的古拉格历史博物馆（Gulag History Museum）馆长罗曼诺夫（Roman Romanov）对抗媒体表示，学者们下一步讲的目标是找出哪些墓埋有多少受害人的尸体，以及他们的身份。
 <br/>
 <br/>
 罗曼诺夫说：〝较小墓地可能埋有30人，其他可能达100人以上。我们想知道谁被埋葬在哪里。〞
 <br/>
 <br/>
 据公开的资讯，苏联作为共产极权的代表政权，出于政治斗争的需要曾经屠杀了大批无辜的民众。仅1937年至1938年大清洗期间，秘密警察机构〝苏联内务人民委员会〞在乌克兰文尼察地区就秘密处决了9000至11000人。
 <br/>
 <br/>
 1943年德国占领乌克兰期间，意外发现了文尼察的万人冢。之后国际卡廷委员会首次对该遗址进行调查，同时又在卡廷森林发现了一起类似的波兰战俘大屠杀现场。
 <br/>
 <br/>
 1943年5月，调查人员在三个不同的地点开始发掘：西部的果园，中央公墓和人民公园，在这三个地点共发现了91个万人冢，总共挖掘出来94320 具尸体，而人民公园的挖掘工作并没有结束，据估计还有更多的尸体被埋在那里。
 <br/>
 <br/>
 德国委员会于1943年7月29日完成了报告，报告确定几乎所有的受害者都是在1937年后期-1938年期间被杀害的。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
