### 失事客机曾四次速度故障  印尼：继续搜寻
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】再来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮航客机.html" target="_blank">
  狮航客机
 </a>
 空难调查进展。印尼国家运输安全部门表示，根据寻获的一个飞行记录器数据显示，失事的狮航班机的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_速度记录仪.html" target="_blank">
  速度记录仪
 </a>
 ，在最后四次飞行都有故障发生。
 <br/>
 <br/>
 印尼促请波音公司和美国政府采取行动，检查与出事客机相同机型的速度仪，防止同类事件在其它地方发生。
 <br/>
 <br/>
 印尼国家运输安全委员会 主席SOERJANTO TJAHYONO：〝印尼运输安全委员会将与美国携手，详细分析最后四次飞行所记录的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_速度故障.html" target="_blank">
  速度故障
 </a>
 。〞
 <br/>
 <br/>
 这次空难造成189人死亡，部份遇难者家属在当局召开的记者会上情绪激动，指责狮航妄顾安全。
 <br/>
 <br/>
 印尼政府承诺，继续全力搜寻遇难者的遗体，调查空难的原因，给家属一个交代。
 <br/>
 <br/>
 印尼国家搜救机构负责人MUHAMMAD SYAUGI：〝我承诺继续搜寻，不会放弃。〞
 <br/>
 <br/>
 目前，印尼当局还在搜寻沉没在海底的另一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑匣子.html" target="_blank">
  黑匣子
 </a>
 ：驾驶舱录音机。
 <br/>
 <br/>
 航空安全专家表示，现在还不能最后确定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮航客机.html" target="_blank">
  狮航客机
 </a>
 坠毁的原因。
 <br/>
 <br/>
 新唐人记者陈依心、明玉综合报导
</div>
