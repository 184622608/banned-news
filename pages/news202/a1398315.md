### 时事拼盘：94岁前纳粹看守受审 前IS占区现202个死人坑
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】下面请看一组新闻简讯
 <br/>
 <br/>
 周二，德国明斯特地方法院青少年法庭对一名前纳粹集中营看守进行开庭审判，罪名是协助谋杀罪。现年94岁的嫌犯曾是波兰施图特霍夫纳粹集中营看守。因为当时他未满21岁，所以由青少年法庭审理此案。1942年到1945年之间，该集中营中有6万5千人死亡。
 <br/>
 <br/>
 联合国周二表示，曾被伊斯兰国占领的伊拉克北部和西部地区发现了202个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_死人坑.html" target="_blank">
  死人坑
 </a>
 ，共埋葬了6千到1万2千具伊拉克平民、军人和警察的尸体。联合国此前曾表示，伊斯兰国在这一地区共杀害了3万3千名平民。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_喀麦隆总统.html" target="_blank">
  喀麦隆总统
 </a>
 毕亚周二举行就职典礼，其统治进入第36个年头。周一早上，该国英语区一所学校的79个孩子和老师以及司机被绑架。政府称，分裂主义分子应对此事负责，但被对方否认。
 <br/>
 <br/>
 欧洲委员会提出，欧盟成员国将对谷歌和脸书等企业的数字营业额征收3%的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_数字税.html" target="_blank">
  数字税
 </a>
 。法国等大国力推数字税，但爱尔兰、卢森堡等小国担心此举可能导致它们的税收减少。丹麦则担心会引起美国报复。
