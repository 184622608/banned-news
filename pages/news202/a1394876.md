### BBC罕见报导中共活摘器官 揭法轮功被迫害真相
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】周一（10月8日），英国广播公司世界新闻频道（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_BBC.html" target="_blank">
  BBC
 </a>
 World News）报导了关于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活摘.html" target="_blank">
  活摘
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员器官的调查，同时采访了两名了解相关真相的嘉宾。这是西方大媒体罕见公开报导中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_器官移植.html" target="_blank">
  器官移植
 </a>
 真相最近的一个例子。
 <br/>
 <br/>
 该访谈节目围绕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 强摘良心犯器官展开，并重点报导了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员在中国大陆所遭受的迫害。
 <br/>
 <br/>
 节目首先以一些调查录像揭开了中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_器官移植.html" target="_blank">
  器官移植
 </a>
 的黑幕。视频显示，调查者询问一名在中国天津某医院做器官移植的病人：她多长时间等到了器官？这名躺在医院病房里的病人回答——两个月。
 <br/>
 <br/>
 报导揭露说，但中共政府否认
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活摘.html" target="_blank">
  活摘
 </a>
 器官，〝假装不了解情况〞。
 <br/>
 <br/>
 调查人员发现，在中国大陆获取器官是很容易的事情。采访视频展示，接受器官移植的病人及其家属表示，他们所获得的器官不知来自哪里，医院只花了2个小时就把新鲜的器官送到了。
 <br/>
 <br/>
 在国际调查的压力下，中共近年承认曾使用死刑犯器官用于移植。但中共政府宣布，从2015年1月1日起，全面停止使用死刑罪犯器官作为供体来源。因此理论上说，自此之后，公民自愿去世后捐献器官已成为中国大陆器官移植唯一获取供体的渠道。
 <br/>
 <br/>
 然而，中共无法自圆其说的重点之一就在于，志愿者死后捐献器官的时间如何及时匹配接受移植手术者需要器官的时间。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_BBC.html" target="_blank">
  BBC
 </a>
 指出，法轮功学员实际上是中共器官移植的主要供体。在中国的监狱和劳教所，被关押的法轮功学员几乎都被迫接受过不明原因的体检，甚至多次。视频中，几名法轮功学员向调查人员讲述了他们被检查身体的情况。
 <br/>
 <br/>
 BBC在视频中展示，原中共卫生部副部长黄洁夫近期在参加一个国际会议时遭到记者追访，但他拒不承认中共活摘法轮功学员器官，并称去年中共只从事了15,000例器官移植手术，而并非所指控的每年100,000例，同时声称相关调查是在〝胡说八道〞。
 <br/>
 <br/>
 但是，一名记者质问黄洁夫：〝为什么我打了一个电话，一家中国医院很快就向我提供了肝脏？这怎么可能？〞
 <br/>
 <br/>
 黄洁夫不正面回答记者问题，只是回避说，〝我不想回答这个问题，有人有政治议程⋯⋯〞
 <br/>
 <br/>
 BBC主持人接着采访了调查中共活摘器官并着有《屠杀》（The Slaughter: Mass Killings, Organ Harvesting, and China’s Secret Solution to Its Dissident Problem）一书的美国作家葛特曼（Ethan Gutmann）和流亡英国的前新疆维吾尔族外科医生安华．托蒂（Enver Tohti）。
 <br/>
 <br/>
 托蒂首先讲述了1995年他在中国大陆时所经历的一次事件。
 <br/>
 <br/>
 托蒂描述，他1995年在新疆担任外科医生期间，曾受命从事过一起活摘死刑犯器官的手术。这段经历后来被收入2016年所拍摄的纪录片《难以置信》（Hard to Believe）中，并一度在世界范围内引起轰动。
 <br/>
 <br/>
 当主持人问到托蒂对这段经历的感受时，托蒂表示，他在中共的洗脑教育下长大，当时坚信〝祖国〞所告诉他的一切都是正确的。中共告诉人们，被判刑的都是坏人。在那种观念的支配下，当他从事那个活摘器官的手术时，他认为自己只是在履行职责、而不是在犯罪。
 <br/>
 <br/>
 托蒂之后补充说，他出国后才真正意识到他曾经施行的活摘死刑犯手术是在犯罪。他非常内疚，所以他要把过去的经历说出来。
 <br/>
 <br/>
 针对被监禁的法轮功学员被莫名地验血，他表示，这种行为非常可疑，没有合理的解释；而在他看来，合乎逻辑的解释就是，中共在建设它的器官移植数据库。
 <br/>
 <br/>
 葛特曼则表示，据他和其他人权人士的调查，中共每年所从事的器官移植手术数量巨大。中共宣称器官来自志愿捐献者，但其数量根本无法弥合实际的器官移植数据。
 <br/>
 <br/>
