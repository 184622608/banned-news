### 【禁闻】美中关系重建 宗教自由或是前提
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】美国总统川普（特朗普）执政后，美国在宣布退出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国人权理事会.html" target="_blank">
  联合国人权理事会
 </a>
 的同时，特别关注中国人权，日前有人权组织负责人透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 也在重建之中，一切协议都会把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教自由.html" target="_blank">
  宗教自由
 </a>
 作为前提条件。
 <br/>
 <br/>
 在日前一个主题为〝捍卫中国人权的最后尝试〞的会议上，华盛顿国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教自由.html" target="_blank">
  宗教自由
 </a>
 圆桌会议共同主席兼协调员Greg Mitchell透露，他们上周和美国国务院有关部门，讨论了有关中国人权和宗教自由问题，并达成了一致意见。
 <br/>
 <br/>
 会后，Greg Mitchell向新唐人透露，在美国与中共的各种谈判中，美方要求中共必须改善宗教自由，并将此视为前提，否则美国不会签字。
 <br/>
 <br/>
 Greg Mitchell：〝现任政府的一份中国声明也同样显示，他们在双边关系中注入宗教自由，因为他们就像所有这些关于贸易经济发展国家安全的双边讨论一样，现在基本上都都在重新谈判。〞
 <br/>
 <br/>
 此前，美国政府的一系列动作，也佐证了他的说法。
 <br/>
 <br/>
 10月4号，美国副总统彭斯在华盛顿哈德逊研究所发表演讲时，抨击中共剥夺中国人的人权和宗教信仰自由。
 <br/>
 <br/>
 8月28号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国参议员.html" target="_blank">
  美国参议员
 </a>
 马克卢比奥（R-FL）和美国众议员克里斯史密斯（R-NJ），分别向美国国务卿和财政部发出信函，指出中国存在人权危机，敦促他们〝迅速采取行动〞，根据〝全球马格尼茨基人权问责法〞制裁中共政府官员和实体，信中还有一份中共官员名单。
 <br/>
 <br/>
 外界注意到，美国现任政府对人权的重视确实恢复到了几十年前的高度。这两年的美国国会的人权报告中一再强调，中共正在加重对民众的监管和控制。
 <br/>
 <br/>
 6月份，美国宣布退出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国人权理事会.html" target="_blank">
  联合国人权理事会
 </a>
 时强调，因为联合国人权理事会成了一个〝虚伪和自私〞的机构，是一个〝具有政治偏见的污水池〞，美国想对这个机构进行改革，但没有一个国家有勇气随着美国大胆改革人权理事会。
 <br/>
 <br/>
 国际宗教自由核心小组顾问Miranda Robinson：〝我明确地看到这届政府，我们的总统，我们的副总统正在关注，国务院正在关注，正如你刚才所听到的那样，很多国会议员都在关注，所以我相信事情会发生变化，我只想对中国人民说，我们正在观察，我们正在倾听，我们非常关心，我们正在尽我们所能。〞
 <br/>
 <br/>
 Greg Mitchell建议美国政府能够在中美双边关系方面的机构中，成立一个和宗教相关的政策工作组，专门讨论中国宗教自由问题。
 <br/>
 <br/>
 Greg Mitchell：〝这将有助于美国继续向中国官员展示为什么在中国推进宗教自由符合中国自身利益，他们可以享受现行的美国奖学金制度，所有的研究和证据表明，宗教自由与经济发展、和平与繁荣、稳定安全之间存在着联系。这就让他们看看，不仅仅是因为这是正确的事情，我们就要他们去做，而是让他们自己看到，只有做到了这一点，才更有可能实现自己的目标。〞
 <br/>
 <br/>
 这次名为〝捍卫中国人权的最后尝试〞的会议强调，尽管美国持续通过外交，公众关注，制裁以及把中国列为特别关注国（CPC），但是中共政府还是在继续剥夺其公民的思想和宗教自由的基本权利，同时违反了1951年〝联合国难民公约〞，遣返了朝鲜难民。
 <br/>
 <br/>
 该会议强调，美国必须调动全政府的力量，来制止中共迫害人权。
 <br/>
 <br/>
 采访编辑/刘惠 后制/周天
