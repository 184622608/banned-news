### 大惊失色！小女孩从妈妈手中滑落轨 火车急驶而过...
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度.html" target="_blank">
  印度
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北方邦.html" target="_blank">
  北方邦
 </a>
 （Uttar Pradesh）马图拉（Mathura）火车站，一名妇人匆忙下火车时，不慎将1岁女儿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_掉落火车轨道.html" target="_blank">
  掉落火车轨道
 </a>
 中。就在错愕之际，此时火车已启动开始出站，月台上的民众大惊失色，火车离站后，奇迹的发现小女孩仍安然无恙。
 <br/>
 <br/>
 综合媒体报导，小女孩的的父亲索努（Sonu）向当地媒体忆述，他带妻子和两名孩子上了火车，却在列车开始离站时发现钱包不见了，急忙叫家人赶忙下车。
 <br/>
 <br/>
 索努说，妻子慌忙之余手一滑，怀中的1岁女儿从车子和月台的隙缝掉下去，卡在路轨中间。
 <br/>
 <br/>
 从视频中可见，火车启动离站后，月台上2名男子急忙跳下路轨，女婴一动不动的卡在路轨和月台中间，不仅奇迹生还，更是毫发未伤，只是吓得嚎啕大哭。男子急忙将小女孩抱起，送回妈妈的怀中。妇人抱过爱女，喜极而泣。
