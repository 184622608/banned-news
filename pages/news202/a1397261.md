### 狮航客机坠毁细节曝光 航空专家：十分神秘
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】导致189人罹难的印尼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮子航空.html" target="_blank">
  狮子航空
 </a>
 客机坠机事件震惊了世界。有关这次空难的更多细节被媒体曝光，显示该飞机在坠毁前，曾以异乎寻常的速度大幅度上升后又急速下降，而当时的天气状况十分良好。有航空专家称这起事件〝十分神秘〞，只有等待找回失事飞机的黑盒才有望解开谜团。
 <br/>
 <br/>
 当地时间10月29日早上6时20分，印尼廉航
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮子航空.html" target="_blank">
  狮子航空
 </a>
 （Lion Air）一架波音737 MAX 8客机从雅加达的苏卡诺哈达（Soekarno-Hatta）国际机场起飞仅13分钟后就失去联系。其后这架启用仅两个月的客机被证实坠毁在雅加达东北的爪哇海的海床上，机上189人全部罹难，其中包括1名幼童及2名婴儿。
 <br/>
 <br/>
 综合媒体消息，经过初步调查后，有消息指出事客机的油箱疑似曾发生漏油并爆炸，但具体坠毁的原因尚无确凿的结果。
 <br/>
 <br/>
 据航空纪录网站的讯息，狮子航空这架出事的客机起飞4分钟后，曾以明显不正常的速度爬升至5400呎高度，在4900至5400呎短暂徘徊后，客机又以惊人的速度下降至4500呎，然后又突然再攀升至5400呎，之后便从雷达上消失。
 <br/>
 <br/>
 据报导，一般情况下，客机的正常爬升速度是每分钟1800呎，而正常下降速度是每分钟2500呎，但失事客机在出事前的异常爬升率，每分钟高达2800呎，下降的速度更是每分钟高达7688呎。
 <br/>
 <br/>
 诡异的是，据气象部门发布的消息，客机失事时的天气状况十分良好，并没有出现任何强气流或强风，飞机驾驶的视野十分清晰，因此可以排除天气原因导致客机失事。
 <br/>
 <br/>
 有消息指，驾驶该客机的机师在起飞后不久，曾向地面控制塔要求折返。但其要求获准后，地面指挥塔却随即与客机失去了联络。
 <br/>
 <br/>
 有航空专家称，已曝光的客机失事前的反常表现令人感到匪夷所思，事件显得〝十分神秘〞，希望早日寻回黑盒解开谜团。
 <br/>
 <br/>
 迄今为止，俗称黑匣子的飞行纪录仪仍未被寻获。
 <br/>
 <br/>
 印尼总统佐科．维多多已透过媒体向死难者家属致哀，并呼吁他们能保持冷静。他表示，自己已下令国家交通安全委员会彻查这起空难事件，搜救人员会尽力寻找遇难者。
 <br/>
 <br/>
 客机的制造商波音公司也对空难表达了悲痛，并宣布将为这起事故提供技术协助。
 <br/>
 <br/>
 据香港东网报导，周一失事的这架客机是由中国民生投资股份有限公司于今年8月13日借给狮航的，两日后就注册并取得了适航证明，飞行时数只有800小时。
 <br/>
 <br/>
 澳洲政府则公开呼吁公众暂时不要安排乘坐狮航客机，称待这起空难的调查完成后再检讨指示。
