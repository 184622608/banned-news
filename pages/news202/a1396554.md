### 法媒聚焦中共间谍渗透 吹响反击号角
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】中共利用各色人等的专业或业余的间谍大规模渗透西方国家，已引起海外多国政府的高度警惕。日前，有法媒引述法国情报机构的资料，披露了中共如何利用海外社交网站相关操作手法，一步步接近并牵制和利用法国各领域的领袖或红人。
 <br/>
 <br/>
 法国《费加罗报》（Le Figaro）23日的一篇报导披露，法国对外安全总局（DGSE）和国内情报总局（DGSI）长期调查后提出预警，中共近年来以刺探秘密为目的，安排了不少间谍特工，大规模地透过领英（LinkedIn）或Viadeo等商务社交网站，去物色法国各个领域圈的领袖或红人，甚至渗透到法国各个权力中心，企图得到资料。
 <br/>
 <br/>
 情报单位指出，近年中国情报人员在职业人脉、商务相关社群网站上〝有规模地运作〞，意图渗透到行政部门、权力圈和法国大企业内部。目前已发现，至少有4000名法国的官员、科学家、高级管理部门主管等深具影响力的人物，以及战略企业合作伙伴及意见领袖圈子里的红人等，曾经被使用假名字的中共情报人员搭上。
 <br/>

 </center>
 <br/>
 据报导披露，那些活跃于网络社交平台上的中共特工们，大多拥有年轻的形象、富有才华、毕业于名校、社群关注度高。一旦法国的目标人物被这些中共情报人员搭上线后，下一步就会被对方提供的各种诱人的工作条件引诱上鈎。
 <br/>
 <br/>
 比如，以招待形式邀请目标人物前往中国参加某个研讨会或工作会议，然后他们可能当场被中共国家安全部官员〝雇用〞。而目标人物的照片、金融转帐纪录、甚至是与中国间谍单位合作的证据或已洩露一些敏感资讯就会成为把柄，被中共情报人员日后用来牵制目标对象为己所用。
 <br/>
 <br/>
 一个典型的例子是：在法国驻亚洲某地的大使馆工作的国际志愿者夏勒（Charles），曾经在3个月内接收到好几家中国的招聘公司或智库向他发出的邀约。这些邀约信对夏勒一番称赞后，通常会提议以300美元到500美元价格请他撰写有关法中关系的粗略报告，试图把夏勒推入一个错综复杂的情境里。夏勒考上公务员后察觉到危险，便在自己完全被牵制之前停止配合中共特工的要求。
 <br/>
 <br/>
 报导表示，这些真实案例只是中共在法国运作间谍行动的冰山一角，中共的目标是大规模掠夺法国国家内部和经济财产的敏感资料。
 <br/>
 <br/>
 这篇报导还引用欧洲情报部门资料透露，中共的特务机构在LinkedIn网站至少管理着500个类似假用户。他们的目标通常是极具潜力的法国年轻主管，或专攻国防、外交的政治经济领域专家等。
 <br/>
 <br/>
 报导称，几乎所有与战略影响力和敏感领域相关的单位都可能被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 盯上。甚至有些被中共间谍吸收的法国人还被唆使去参加行政考试，以便渗透到法国各部门和情报机构。其中有些人可以及时警觉到这是个骗局，但据法国情报单位的调查，至少有〝数百个目标已进入受到相当牵连的阶段〞。
 <br/>
 <br/>
 法国政府认为，中共这种大规模间谍操作对法国的国家主权已〝造成重大威胁〞，同时也置法国经济财产于险境。报导称，〝反击的钟声已经响起〞。
 <br/>
 <br/>
 （记者黎明报导/责任编辑：戴明）
