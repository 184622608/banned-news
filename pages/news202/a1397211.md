### 德国选举执政党失利 默克尔辞党主席
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】周日，德国黑森州选举公布投票结果，德国两大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_执政党.html" target="_blank">
  执政党
 </a>
 失利。周一德国总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默克尔.html" target="_blank">
  默克尔
 </a>
 宣布，不再连任执政的基民党主席。
 <br/>
 <br/>
 根据开票结果，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默克尔.html" target="_blank">
  默克尔
 </a>
 所在的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基民盟.html" target="_blank">
  基民盟
 </a>
 保住了第一大党地位，但得票率从5年前的38.3%降到27.2%。
 <br/>
 <br/>
 黑森州基民党总理Volker Bouffier：〝今晚情绪复杂，我们遭受了巨大损失。〞
 <br/>
 <br/>
 另一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_执政党.html" target="_blank">
  执政党
 </a>
 社民党得票率从5年前的30.7%降到19.6%，这是1946年以来，该党在黑森州最差的选举结果。
 <br/>
 <br/>
 黑森州社民党领袖 THORSTEN SCHAEFER-GUMBEL：〝对黑森州的社民党来说，这是个艰难的夜晚，我们没有得到想要的结果。〞
 <br/>
 <br/>
 两党组成的全国执政联盟，面临解散危机。
 <br/>
 <br/>
 绿党和选项党是此次黑森州选举大赢家。绿党获得19.6%选票，与社民党不相上下。
 <br/>
 <br/>
 右翼选项党获得12.1%的选票，成为黑森州第四大党，在德国所有16个州议会中占据了席位。
 <br/>
 <br/>
 周一，德国总理默克尔宣布，不再连任基民盟主席，也不再连任总理。
 <br/>
 <br/>
 德国总理默克尔：〝首先，十二月在汉堡举行的基民盟党代表大会上，我不再作为主席候选人。第二，第四个德国总理任期是我最后一个任期。〞
 <br/>
 <br/>
 她还说，2021年大选时，她不会再竞选任何政治职务。
 <br/>
 <br/>
 现年64岁的默克尔，2000年起一直连任基民党主席，是该党连任时间最长的主席。
 <br/>
 <br/>
 消息传出后，欧元下跌0.4%，跌至当天最低点。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
</div>
