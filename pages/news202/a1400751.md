### 欧盟同意英国脱欧条款 敦促英国人支持首相梅
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 领导人在比利时首都布鲁塞尔会晤，正式同意
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 协议，并敦促英国人支持英首相特蕾莎・梅政府的这套
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧方案.html" target="_blank">
  脱欧方案
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 27国领导人在布鲁塞尔举行的特别峰会上用不到半个小时，批准了长达600页的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 协议、以及26页的概述明年3月29日英国离欧后双方未来自由贸易关系的宣言。一天前，英国在协议中有关直布罗陀的问题上妥协；而协议需要经过英国议会批准才能生效。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国首相.html" target="_blank">
  英国首相
 </a>
 梅先期抵达布鲁塞尔，周日欧盟批准协议后她再次与欧盟领袖会谈，并在新闻发布会上表示这份协议是唯一可能英国脱欧协议。
 <br/>
 <br/>
 欧盟首席谈判代表巴尼耶Michel Barnier则表示这套方案是未来密切关系的基础。
 <br/>
 <br/>
 欧盟首席谈判代表巴尼耶Michel Barnier（0.18-0.24）：〝现在是每个人承担责任的时候了，每个人。〞
 <br/>
 <br/>
 不过，欧盟领导人对首相特蕾莎・梅在国内面临的对协议的批评声音并不乐观。荷兰首相吕特Mark Rutte赞扬了首相梅对复杂谈判的处理能力，但同时对首相梅反对者警告说，达成的协议已是最好的方案。
 <br/>
 <br/>
 立陶宛总统格里包斯凯特Dalia Grybauskaite则对表示担忧，称退出程序尚未完成，并列举出如果英国议会投票否决了这份协议，包括选举出新的议会或呼吁重新谈判，可能会出现的四种结果。
 <br/>
 <br/>
 立陶宛总统格里包斯凯特Dalia Grybauskaite（1.17-1.50）：〝一切都可能发生，至少有四种情况可能发生，而取决于英国方面决定选择哪条路。〞
 <br/>
 <br/>
 格里包斯凯特：〝这可能是民众再行投票，可能是重新选举，也可能是要求重新谈判，我计算过至少四种可能的情形。〞
 <br/>
 <br/>
 目前，英国首相特蕾莎・梅面临着如何获得分歧严重的英国议会的支持。这也是欧盟面临的最大问题，特丽莎-梅的少数党政府是否能驾驭执行这份协议。欧盟委员会主席容克Jean-Claude Juncker表示相信英国政府将成功获得英国议会的支持。
 <br/>
 <br/>
 特蕾莎・梅在致全囯的一份公开信中请求民众支持脱欧协议：〝不论你是否投票脱离或者留在（欧盟），这是一份符合我们国家利益的协议，一份适合于我们整个国家和所有民众的协议。〞
 <br/>
 <br/>
 新唐人记者徐哲、天羽综合报导
 <br/>
