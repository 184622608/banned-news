### 防敲诈 韩国计程车将引进扫码支付 可用支付宝付款
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 是中国游客出国游的热门城市。为方便游客出行，防止外国游客被〝敲竹槓〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首尔.html" target="_blank">
  首尔
 </a>
 计程车将于明年1月起引进动态扫码支付功能，其中包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支付宝.html" target="_blank">
  支付宝
 </a>
 ，让中国人出行更方便。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首尔.html" target="_blank">
  首尔
 </a>
 市政府表示，目前中国游客占整体访韩游客的3成以上，但大多中国游客不熟悉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 的信用卡支付系统，因而需要携带现金。
 <br/>
 <br/>
 资料显示，2017年韩国观光公社接到的外国游客投诉，有关计程车的投诉内容占13.3%，位居第二；购物位居第一，占28.0%。有关计程车的投诉内容，〝敲竹槓〞行为占有最大比重，达46.5%。以计程车利用不方便为由，投诉的游客39.5%是中国人。
 <br/>
 <br/>
 另外，以中国为例，支付方式以二维条码为主的移动支付方式比重从2014年的4%，到2017年大增为63%。
 <br/>
 <br/>
 韩国政府决定，在韩国国内将首次制定、可适用于多种支付系统的〝计程车扫码支付〞标准，并与便捷支付系统开发商，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支付宝.html" target="_blank">
  支付宝
 </a>
 服务商合作，引进扫码支付，从2019年1月起安装于所有首尔计程车上。
 <br/>
 <br/>
 首尔市政府表示，当局尤其关注支付系统对外国游客的便利程度。韩国国内的线上支付多使用印有收费资讯的静态条码（Static QR code），但首尔计程车将搭载动态条码（Dynamic QR code）支付系统。游客利用动态条码系统，按路程和时间计算的车费可即时产生二维码，能更有效提高游客使用的便利性，还可防止被不法司机〝敲竹槓〞。
 <br/>
 <br/>
 （记者李红报导/责任编辑：文慧）
