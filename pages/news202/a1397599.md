### 俄媒：川普退出《中导条约》并非针对俄罗斯
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】日前，美国总统川普宣布，美国即将退出在冷战时期与苏联签署的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中导条约.html" target="_blank">
  中导条约
 </a>
 》，理由是莫斯科多年来没有遵守该条约。有俄罗斯媒体分析认为，此次华盛顿实际针对的其实不是俄罗斯，而是不受该条约约束的中共。
 <br/>
 <br/>
 川普在宣布华盛顿将退出美俄之间的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中导条约.html" target="_blank">
  中导条约
 </a>
 》之后，还提出建议希望北京能够加入，共同签署美俄中三国之间新的《中导条约》，但遭到中共的拒绝。
 <br/>
 <br/>
 《自由亚洲电台》引述俄罗斯媒体报导说，在俄美受到《中导条约》限制的近几十年来，中共却在中短程
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_导弹.html" target="_blank">
  导弹
 </a>
 领域全力研发，并已经处于领先地位。北京所拥有中短程导弹的性能和数量已经使其成为中共在亚太地区战略、战术部署的主力。
 <br/>
 <br/>
 报导称，美国退出或修改《中导条约》背后的推动力，很可能不是对俄罗斯的违规行为表达不满，而是亚太地区军事力量平衡的变化。美国面对拥有如此强大的中短程导弹的中共，自己却被绑住手脚，无法研发制造这些导弹，使美方处于相当不利地位。
 <br/>
 <br/>
 俄罗斯专家对于北京拒绝加入新的《中导条约》表示，中共只希望俄罗斯和美国能够继续遵守《中导条约》，自己却不愿在中短程导弹问题上受到任何的约束，因为其生产的绝大多数导弹都在《中导条约》的中短程导弹约束范围内。因此，中共拒绝签署《新中导条约》的做法并不令人感到惊讶，其批评俄罗斯和美国退出《中导条约》的宣传，只是希望尽力约束俄美。
 <br/>
 <br/>
 俄罗斯科学院远东研究所主任研究员、军事专家瓦西里．卡申说：〝此前有西方研究机构根据一些测算得出结果说，中共拥有2000多枚已经部署的中短程弹道导弹和射程在4000公里内的陆基中程巡航导弹。但就像苏联解体后，美国发现自己测算苏联核弹头的数量比实际数量少一倍那样，北京拥有的导弹数量可能远远大于这个数字。〞
 <br/>
 <br/>
 还有西方的媒体也认为，川普退出《中导条约》主因是中共不受该条约约束，可自由发展中程导弹。而美国受条约限制不能发展新型武器，无法应对中共在西太平洋地区的军事扩张，又不能坐视其生产这类武器，选择终结并脱离《中导条约》，不再任由俄罗斯违反协议，也可能向北京发出强有力的信号。
 <br/>
 <br/>
 《中导条约》是美国前总统里根和前苏联总书记戈尔巴乔夫，在1987年签订。该条约禁止两国拥有、生产或试射所有射程在500公里至5500公里之间的陆基巡航导弹和弹道导弹，以保障美国本土以及在欧洲和远东地区盟友安全。在苏联解体后，俄罗斯继承这项条约，双方根据条约总共销毁约2692枚中程导弹。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：戴明）
