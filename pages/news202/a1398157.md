### 川普释善意 对伊制裁暂时豁免中国大陆
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】周一，美国政府公布了对伊朗第二批制裁措施的具体细节，同时公布了一份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_豁免名单.html" target="_blank">
  豁免名单
 </a>
 ，暂时允许八个国家和地区在美国启动对伊朗新的制裁后的一段时期内可以继续进口伊朗石油，其中就包括中国大陆和台湾。
 <br/>
 <br/>
 11月5日，美国政府正式宣布，美对伊朗的新一轮经济制裁从美东时间11月4日午夜全面生效。
 <br/>
 <br/>
 根据美国国务卿蓬佩奥（Mike Pompeo）与财政部长姆努钦（Steven Terner Mnuchin）在发布会上公布的制裁措施细节，美国政府将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁伊朗.html" target="_blank">
  制裁伊朗
 </a>
 逾700名个人和实体、飞机和船隻；制裁50家伊朗银行及其子公司，其中包括伊朗国家银行；制裁伊朗航空，包括其拥有的67架飞机；制裁伊朗能源和运输领域的逾200名个人和船隻。
 <br/>
 <br/>
 这些制裁是川普（特朗普）政府〝极限压力〞政策的一部分，目的是迫使伊朗放弃其核武项目，并停止支持美国在中东的极端武装组织。
 <br/>
 <br/>
 姆努钦直言，伊朗政权将面临越来越多的〝金融孤立〞，直到他们彻底改变当前那些危险的做法为止。
 <br/>
 <br/>
 蓬佩奥此前也曾对媒体称：〝伊朗要么改变做法，要么看着自己的经济崩盘，美国将惩罚那些与伊朗做生意的国家。〞
 <br/>
 <br/>
 美国政府11月5日同时还公布了对伊朗制裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_豁免名单.html" target="_blank">
  豁免名单
 </a>
 ，暂时允许八个国家和地区继续进口伊朗石油，被列入豁免名单的国家或地区包括：印度、意大利、希腊、日本、韩国、土耳其、中国大陆和台湾。不过，根据美国法律，制裁临时豁免待遇期限为180天。
 <br/>
 <br/>
 川普政府表示，他们希望在未来几个月内将德黑兰的出口削减至零，并要惩罚任何违反其原油禁运的实体或个人。
 <br/>
 <br/>
 稍早前（8月7日）美国总统川普曾在推特（Twitter）上发文称：〝任何人，同伊朗做生意就不要想再和美国做生意。〞
 <br/>
 <br/>
 而北京当局曾宣布，中国11月暂停订购伊朗原油，当时外界就推测北京方面正在等待美方的豁免名单。
 <br/>
 <br/>
 海外舆论认为，美国政府这次将中国大陆和台湾列入制裁伊朗的豁免名单，是为美国总统川普将于11月底在G20峰会上与中国国家主席习近平举行双边会谈营造良好气氛而有意向北京当局释放的善意。
 <br/>
 <br/>
 11月1日，川普曾在社交媒体推特（Twitter）上发帖称：〝刚和中国的习近平主席通了很长时间的电话，聊得很好，我们谈了很多话题，重点在贸易上。〞他并表示，有关在阿根廷举行的G20峰会期间举行各种会晤的讨论，也进展得很顺利。
 <br/>
 <br/>
 据公开的资讯，2003年伊朗宣布成功提炼出〝铀〞; 2006年初，伊朗恢复了对核燃料的研究。同一年，国际社会开始与伊朗展开核谈判，经过漫长的谈判过程后，最终伊朗于2015年与美、英、法、德、俄罗斯和中国共同签署了〝伊朗核协议〞。当时美国正处于奥巴马执政时期。
 <br/>
 <br/>
 根据这份协议，伊朗仍可在民用领域开展核计划，但不得再继续研发核武器，而且必须在15年内必须将铀浓缩浓度限制在不超过3.67％的范围内。作为交换条件，国际社会解除对伊朗的经济制裁，伊朗在海外的数百亿美元资产得以解冻。
 <br/>
 <br/>
 不过，在此后伊朗并没有严格遵守这份协议。在国际社会解除经济制裁后，伊朗的弹道导弹项目并没有因此受到制约，同时该政权在也门和叙利亚冲突中所扮演的角色也未改变。
 <br/>
 <br/>
 以色列近期公布的证据证明，伊朗在与国际社会签署上述协议后，仍然一直在秘密研发核武器。
 <br/>
 <br/>
 美国川普政府认为，〝伊朗核协议〞的内容〝烂透了〞，解除经济制裁，等于为伊朗保证了研究经费，而这份协议并没能有效防止伊朗暗中继续进行核武研发，实际上成为了伊朗发展核武器的保护伞。
 <br/>
 <br/>
 今年5月，美国政府决定退出原先的伊朗核协议，重启对伊朗的全面制裁。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
