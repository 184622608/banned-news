### 时事拼盘：法国放弃燃油增税 欧洲警察扫荡光荣会
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】法国马克龙政府决定暂缓增加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃油税.html" target="_blank">
  燃油税
 </a>
 六个月，但黄背心抗议者认为当前措施还不足以减轻税负，继续抗议。周三，法国政府宣布，彻底放弃在2019年预算案中增加燃油税。过去两周，黄背心示威出现暴力骚乱，极右翼和极左派人都有加入。
 <br/>
 <br/>
 周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 总统普京表示，如美国退出《中程导弹条约》并制造武器，俄罗斯将研发被禁止的导弹。但美国指出，2014年起俄罗斯已经违约，研发可打击数千英里外目标的导弹，威胁欧美。
 <br/>
 <br/>
 周二，意大利警方逮捕西西里
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑手党.html" target="_blank">
  黑手党
 </a>
 新头目后。周三，欧洲各国警察又联手突袭了恶名昭彰的黑手党组织〝光荣会〞，其涉嫌跨国走私毒品、帮派结盟、洗钱及土地诈欺等罪名。其中，意大利逮捕41人，德国逮捕21人，比利时14人。
 <br/>
 <br/>
 周三，近日，接连爆出中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 硬闯或偷越美墨边境。周二，又有一对大篷车母女因非法越境被逮捕。目前，美国总统川普和他在共和党中的支持者，正积极争取在新的政府预算中，加入修建边境墙的拨款。
</div>
