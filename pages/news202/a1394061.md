### 纳吉布妻子罗斯玛受审 被控17项罪
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马来西亚.html" target="_blank">
  马来西亚
 </a>
 前总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳吉布.html" target="_blank">
  纳吉布
 </a>
 的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗斯玛.html" target="_blank">
  罗斯玛
 </a>
 在吉隆坡法庭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受审.html" target="_blank">
  受审
 </a>
 ，她被指控洗钱等17项罪名。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马来西亚.html" target="_blank">
  马来西亚
 </a>
 反贪污委员会周三第三次传唤前总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳吉布.html" target="_blank">
  纳吉布
 </a>
 的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗斯玛.html" target="_blank">
  罗斯玛
 </a>
 ，之后将她监禁。周四上午，罗斯玛在吉隆坡法庭应讯。
 <br/>
 <br/>
 罗斯玛被控洗钱等17项罪名，但她全部否认。
 <br/>
 <br/>
 法官最后让罗斯玛以200万令吉，相当于50万美元交保候传，并命令其交出国际护照。
 <br/>
 <br/>
 罗斯玛是马来西亚史上首位被指控的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前总理夫人.html" target="_blank">
  前总理夫人
 </a>
 。如果罪名成立，每项罪都面临5到15年徒刑。法院8号将再次开庭审理此案。
 <br/>
 <br/>
 周四，前总理纳吉布因涉及马来西亚政府全资拥有的〝一马公司〞案，也被法院传唤出庭。他面临30多项指控，包括洗钱和滥用权力等。
 <br/>
 <br/>
 反贪污委员会怀疑，〝一马公司〞遗失的数十亿资金流入了纳吉布家族的腰包。
 <br/>
 <br/>
 目前，美国等至少六个国家都在调查〝一马公司〞案。
 <br/>
 <br/>
 新唐人记者 李梅 严曦综合报导
</div>
