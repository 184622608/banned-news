### 沙特知名异议记者在土耳其失踪 疑被杀
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 阿拉伯著名记者兼专栏作家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 周二前往沙特驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 领事馆办理结婚文件，其后神秘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 。土耳其当局相信，卡舒吉在领事馆内被绑架、谋杀；沙特方面则表示绑架指控毫无根据。
 <br/>
 <br/>
 一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 官员表示〝土耳其警方的初步评估是，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 先生在伊斯坦布尔的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 阿拉伯领事馆遇害。〞并认为〝谋杀是有预谋的，尸体随后被移出领事馆。〞
 <br/>
 <br/>
 不过，这名官员没有透露警方凭什么根据作出这样的推断，也没有出示任何实质证据。
 <br/>
 <br/>
 沙特方面则表示指控毫无根据。沙特总领事奥泰比周六表示，沙特政府和相关部门正协助土耳其寻找卡舒吉的下落。
 <br/>
 <br/>
 59岁的卡舒吉毕业于美国印第安纳州立大学，一向大胆公开批评沙特阿拉伯政府的政策，一直担心自己会遭受报复，一年前逃离了沙特。
 <br/>
 <br/>
 上周二，他进入沙特驻伊斯坦布尔领事馆内，领取他即将举行婚礼所需的文件。但其后离奇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 ；他的土耳其裔未婚妻说，她当天一直在领事馆门外等候，未见他出来。沙特官员则坚称，卡舒吉当天办理完手续后，已经离开领事馆。
 <br/>
 <br/>
 周六早些时候，土耳其官员表示，检察官已经开始调查卡舒吉的失踪事件。
 <br/>
 <br/>
 美国国务院对事件表示关注，一名官员表示目前无法确认有关报导、正在寻求更多信息。
 <br/>
 <br/>
 新唐人记者严枫、雨薇综合报导
</div>
