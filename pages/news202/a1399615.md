### 金正恩视察新武器设施 遣返一美国人
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】再来关注朝鲜半岛局势。朝鲜官媒周五称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 视察了新战术武器的试验设施，这是去年以来朝鲜首次公开宣布金正恩视察武器设施。同一天，朝鲜驱逐一名扣押的美国公民。
 <br/>
 <br/>
 朝鲜中央通讯社报道称，周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 视察了新战术武器测试场。上次视察是去年11月29日。
 <br/>
 <br/>
 官媒发出唯一照片，显示金正恩站在沙滩上，军方官员陪同，没有看到武器。
 <br/>
 <br/>
 金正恩称，其父金正日生前对这项武器开发非常感兴趣，并亲自参与指导。
 <br/>
 <br/>
 周五，韩国统一部回应，对朝鲜的新武器还需了解更多信息。
 <br/>
 <br/>
 韩国统一部副发言人 LEE EUGENE：〝关于武器类型，我们需要了解更多，因为他们没有特别提及武器详情，也没有在金正恩视察报道中出现。〞
 <br/>
 <br/>
 美国国务院对朝鲜的举动作出回应，表示对金正恩能履行承诺充满信心。
 <br/>
 <br/>
 周四，白宫就彭斯和文在寅〝东盟峰会〞期间会面发表声明，重申美韩联盟是保证朝鲜半岛和平与安全的关键。
 <br/>
 <br/>
 分析人士表示，朝鲜这次低调视察更可能是安抚军心，而不是破坏外交谈判。也有分析认为，这也可能是对美韩联合军演的回应。
 <br/>
 <br/>
 另外，朝鲜官媒周五称，朝鲜将遣返去年10月入境的美国公民布鲁斯・劳伦斯，称其自称受美国中情局指使。
 <br/>
 <br/>
 去年11月，韩国发现一位相同名字的男子在韩朝边境徘徊，随后将其遣返。
 <br/>
 <br/>
 美国驻韩大使馆发言人表示，无法确认韩朝两国遣返的是否是同一个人。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
</div>
