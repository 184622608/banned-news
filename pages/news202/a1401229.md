### 俄军压境 乌克兰总统：不是开玩笑 战争随时爆发
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 近日与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 于黑海附近的克赤海峡爆发军事冲突，引起国际社会关注。乌克兰总统波罗申科稍早表示，俄罗斯军队已将驻守边境的坦克数量增加了3倍，不是开玩笑的，双方随时有爆发全面战争的风险。
 <br/>
 <br/>
 周三（28日），波罗申科（Petro Poroshenko）视察边防时称：〝我们必须确保有效的人员培训。在需要时，击败敌人。〞
 <br/>
 <br/>
 波罗申科周二（27日）在接受国家电视台访问时表示：〝我不希望任何人认为这是开玩笑。这个国家如今面临与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 爆发全面战争的威胁。〞
 <br/>
 <br/>
 波罗申科还展示了分别于9月和10月取得的，距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 边境18公里处的一个俄罗斯军事基地的卫星图片。
 <br/>
 <br/>
 波罗申科说，俄罗斯军方在两国边境地区部署的兵力大幅增加，该基地的坦克车数量也增至3倍。而在克赤海峡（Kerch Strait）事件之后，这些坦克车并没有撤离该基地。
 <br/>
 <br/>
 他认为，在该事件发生之后，乌克兰军队应该被赋予抵抗的能力，以防俄罗斯进行全面的地面入侵行动。
 <br/>
 <br/>
 克赤海峡事件发生于上周日（25日），俄军向3艘乌克兰军舰开火，并扣押了乌克兰军舰。事件中，多名乌克兰士兵受伤，全部24名海军官兵目前均被俄罗斯方面扣押。其中12人被拘留在克里米亚半岛。周三，俄罗斯一个法庭裁决，对扣押的24名乌克兰船员的拘留期延长2个月。
 <br/>
 <br/>
 该事件不仅引爆两国之间的外交危机，也引发国际社会对俄罗斯的谴责，并商讨对该国进行制裁。
 <br/>
 <br/>
 俄罗斯声称，乌克兰船艇未能对俄方的停船要求做出回应，此举违反了国际法。而乌克兰则在隔天宣布实施戒严30天。
 <br/>
 <br/>
 俄罗斯批评波罗申科实施戒严令是为了拖延明年3月大选，但波罗申科将戒严令从60天减半到30天，并承诺不会延迟选举，也不会影响乌克兰民主体制。
 <br/>
 <br/>
 俄乌两国之间的冲突由来已久。自俄罗斯在2014年并吞克里米亚以来，两国冲突已造成超过1万人丧生。俄罗斯今年在俄罗斯和克里米亚之间建筑的桥梁开通，使紧张局势升级。25日首次爆发重大海上冲突，外界担心紧张局势会进一步升温。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）接受《华盛顿邮报》采访时表示，他不喜欢这种好斗行为，也决不赞同这类好斗行为。他说，他可能不会在20国集团（G20）峰会上与俄罗斯总统普京进行会谈。他正在等待其安全团队有关乌克兰局势的详细报告，并以此作为根据作出最终决定。但普京周三提出，希望与川普会面。
 <br/>
 <br/>
 川普稍早表示，〝我们不喜欢发生的这些事，无论哪一方面，我们都不喜欢。希望他们妥善处理好。我知道欧洲也不愿看到现在的局面，他们也在努力，我们一起在努力。〞
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
