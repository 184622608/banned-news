### 美副总统出访亚太  APEC可能再批中共
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 12日抵达日本，展开为期一周的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚太之行.html" target="_blank">
  亚太之行
 </a>
 。预计他将与日方主要讨论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝核.html" target="_blank">
  朝核
 </a>
 及区域安全议题。另外，彭斯还将在亚太经合组织峰会上发表重要演讲。
 <br/>
 <br/>
 当地时间12日晚7点，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 和夫人抵达日本东京横田机场。13日上午，彭斯将与日本首相安倍晋三举行会谈及联合记者会。
 <br/>
 <br/>
 战略与国际研究中心高级副总裁 Matthew Goodman：〝副总统彭斯和首相安倍的讨论，包括朝鲜、中国、区域安全与海上安全问题。〞
 <br/>
 <br/>
 彭斯周日在阿拉斯加停留期间曾表示，此次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚太之行.html" target="_blank">
  亚太之行
 </a>
 ，旨在加强美国和该地区盟国的关系，并重申美国坚持朝鲜半岛无核化的决心。
 <br/>
 <br/>
 对于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝核.html" target="_blank">
  朝核
 </a>
 局势，专家分析，日前朝鲜突然取消与国务卿蓬佩奥的重要谈判，显示出双方在无核化问题上可能触及到实质性阶段。
 <br/>
 <br/>
 对于最早在明年1月启动的日美贸易谈判，各方都在关注。但专家认为，这次双方可能不会进行太多讨论。
 <br/>
 <br/>
 战略与国际研究中心高级副总裁 Matthew Goodman：〝此行另一个亮点是，副总统要求并得到了与首相安倍的联合记者会。这很不同寻常，因为礼仪上不太对等。〞
 <br/>
 <br/>
 本此亚太之行，彭斯还将代表川普总统出席美国-东盟峰会和东亚峰会，并于17日参加在巴布亚新几内亚召开的亚太经合组织峰会。在那里，他将就美国的印太战略发表重要演讲。官员透露，他将深入、全面阐述美国对区域安全构架及战略前景的构想，并对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在亚太地区一系列行为提出强烈批评。
 <br/>
 <br/>
 中国问题专家 专栏作家 章家敦：〝我们看到的是，美国内部对于中共已有一个新的共识。每个人都在担忧中共走得太远了，已对美国和国际社会构成威胁。因此，必须要采取新的政策。所以，美国目前的态度已有很大不同，在其它地区也是如此。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
