### 英国议会审议〝脱欧草案〞 首相连受打击
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】周三，英国〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 〞审议程序进入第二天。首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特雷莎．梅.html" target="_blank">
  特雷莎．梅
 </a>
 连受打击，不得不在周三公布〝脱欧法律意见〞。如果下周议会否决此方案,脱欧走向决定权将交到议会手中。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国首相.html" target="_blank">
  英国首相
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特雷莎．梅.html" target="_blank">
  特雷莎．梅
 </a>
 ：〝作为政治家，我们有责任兑现英国民众在2016年全民公投中取得的结果。〞
 <br/>
 <br/>
 两年前英国脱欧公投中，一半以上的人选择脱欧。但没人想到，这条路有多么曲折。
 <br/>
 <br/>
 周二，英国下议院启动为期五天的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 〞审议程序。首相梅两次遭遇失败。
 <br/>
 <br/>
 首先，下议院通过了对政府〝藐视议会〞的指控。
 <br/>
 <br/>
 导致首相府周三很不情愿地向议会公布了〝脱欧法律意见书〞全文。
 <br/>
 <br/>
 该〝意见书〞警告，旨在保证爱尔兰和北爱尔兰之间没有硬边界的〝爱尔兰保障条款〞有可能使英国在未来陷入和欧盟的长期谈判拉锯战。
 <br/>
 <br/>
 这让首相梅口中的〝好脱欧草案〞再次受到打击。
 <br/>
 <br/>
 下议院周二通过的另一项提案称，如果目前的脱欧方案被下议院否决，那么下议院有权提出修正案。目前脱欧方案通过的希望渺茫，这项动议几乎是把〝脱欧方向〞决定权交到了议会手里。
 <br/>
 <br/>
 这让支持软脱欧的议员和留欧派都大受鼓舞。
 <br/>
 <br/>
 连线本台欧洲记者 王羽音：〝两次投票反映了首相梅在议会里的支持力量不足。第一次投票时她的执政盟友倒戈，第二次，她所在党派有20多位议员倒戈。〞
 <br/>
 <br/>
 〝不过她现在又多了一个办法去说服脱欧派议员来支持她。之前她是用‘无协议脱欧’的风险来说服各方投票支持她的方案。现在呢，随着留欧派的声势越来越大，首相梅也开始对脱欧派议员说：如果不接受她的方案，那么也许要面临留在欧盟的风险。但是外界普遍认为，这些办法还是无法让她在下周二的议会投票中得到足够的支持。〞
 <br/>
 <br/>
 新唐人电视综合报导
