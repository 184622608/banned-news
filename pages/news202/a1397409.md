### 巴西当选总统誓惩不实媒体 建廉洁政府
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】巴西当选总统博尔索纳罗将于明年1月上台，他誓言上台后建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_廉洁政府.html" target="_blank">
  廉洁政府
 </a>
 ，并打击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不实媒体.html" target="_blank">
  不实媒体
 </a>
 。
 <br/>
 <br/>
 被称为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西川普.html" target="_blank">
  巴西川普
 </a>
 〞的博尔索纳罗表示，在竞选期间，几乎所有关于他的假新闻都来自《圣保罗页报》。他上台后，将消减给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不实媒体.html" target="_blank">
  不实媒体
 </a>
 的政府资助，避免人民被误导以及制造国家分裂。
 <br/>
 <br/>
 巴西新总统 博尔索纳罗：〝受够了谎言，受够了假新闻。我们现在正处于一个新时代，我想为每个巴西人服务，不仅仅是那些支持我的人。  如果巴西无法摆脱伦理、道德和经济危机，我们将在未来承受恶果。我们的目标是共同创建一个伟大的国家，避免分裂。〞
 <br/>
 <br/>
 对于组建新内阁的人选，博尔索纳罗透露，他希望联邦法官莫罗出任司法部长。
 <br/>
 <br/>
 莫罗是巴西家喻户晓的〝反贪英雄〞，他在2014年发起的大规模反腐调查〝洗车行动〞，揭开了政商界高层的贪腐丑闻，多名高官被判入狱或下台，包括前总统卢拉。
 <br/>
 <br/>
 现年63岁的博尔索纳罗，将于明年1月1日宣誓就职。他称，将尽快推动施政议程，组建一个正直廉洁的政府。
 <br/>
 <br/>
 新唐人记者曹菁综合报导
</div>
