### 亚速海争端升级 俄罗斯向乌克兰军舰开火
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 封锁了刻赤海峡上唯一通往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚速海.html" target="_blank">
  亚速海
 </a>
 的水路通道，阻止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 船隻通过，然而，事件进一步升级，乌克兰声称俄罗斯已向他们的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军舰.html" target="_blank">
  军舰
 </a>
 开火，造成海军士兵受伤。
 <br/>
 <br/>
 周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 海军三艘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军舰.html" target="_blank">
  军舰
 </a>
 从黑海港口奥德萨（Odessa）出发，准备前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚速海.html" target="_blank">
  亚速海
 </a>
 的马里乌波尔（Mariupol），在必经的刻赤海峡航行时，其中一艘拖船遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 边防船的冲撞，造成发动机，以及船体多处损坏。
 <br/>
 <br/>
 在撞船事件后，俄罗斯将巨型货轮停泊在乌克兰连结俄罗斯的克里米亚大桥（Crimean Bridge）的桥下，也是唯一通往亚速海的水道，以阻止乌克兰军舰通过。
 <br/>
 <br/>
 乌克兰海军司令官 Igor Voronchenko ：〝已经证实，俄罗斯没有提供从刻赤海峡通往亚速海的自由船隻通道。〞
 <br/>
 <br/>
 随后，乌克兰声称俄罗斯追逐乌克兰海军的两艘炮舰以及遭冲撞的拖船，并开火，造成两名海军士兵受伤。而俄罗斯并未对此作出任何回应。
 <br/>
 <br/>
 根据两国签订的双边条约，两国均有权使用亚速海。但自从俄罗斯在2014年吞并克里米亚之后，两国在亚速海的争端开始升级。
 <br/>
 <br/>
 由于克里米亚大桥由俄罗斯兴建，为俄罗斯控制船隻进出亚速海提供了便利条件。
 <br/>
 <br/>
