### 中小企业扩展市场有妙招—日本物产展
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中小企业.html" target="_blank">
  中小企业
 </a>
 可谓是日本经济的中流砥柱，根据日本经济产业省关于中小企业的调查，全日本大型企业与中小企业的数量分别为1.1万与380万。然而众多中小企业虽然拥有惊人的技术与手艺，但知名度低很少被民众了解。为解决这一难题，每年11月，中小企业联合在京举办〝日本全国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_物产展.html" target="_blank">
  物产展
 </a>
 〞，让民众边吃边喝高高兴兴地认识全国各地方知名产品。
 <br/>
 <br/>
 记者：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 全国有47个都道府县，根据各地环境的不同小吃也各有特色。一年一度的日本全国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_物产展.html" target="_blank">
  物产展
 </a>
 中，来自全国各地的350多家企业齐聚一堂，是一个一次性网罗各地知名小吃的好机会。
 <br/>
 <br/>
 日本全国物产展是全日本最大规模的物产展，由名为全国商工会联合的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中小企业.html" target="_blank">
  中小企业
 </a>
 团体于每年11月中下旬在东京池袋举办。来自北海道至冲绳地区的350多家企业参展，展出特产超过3000件，3天时间就可以吸引15万人前来。
 <br/>
 <br/>
 全国商工会联合市场开拓支援课 长谷川 大佑：〝日本全国各地都有很多中小企业，本次物产展的目的就是帮助他们市场拓展。〞
 <br/>
 <br/>
 各展区内的展位按地名依次分开，截止至采访当日，票选出的人气最高日本点心是指在熊本县内栗子口味的点心。
 <br/>
 <br/>
 福田屋有限公司市场本部长新村茂夫：〝作为板栗的名产地，这是使用熊本县产的和栗制成的烤点心。这个点心只有熊本县内贩卖。〞
 <br/>
 <br/>
 顾客可以轻松品尝到日本列岛的各地特产，各地知名小吃饮食角更是人气鼎盛。不仅很多家庭成员集体行动，还吸引了很多在日读书的外国留学生前来。
 <br/>
 <br/>
 外国留学生：〝我想尝试一下这个鸡翅，我很喜欢烧烤的东西和那个沾酱。〞
 <br/>
 <br/>
 企业也能在欢声笑语中推广自己产品、提高知名度，这真可谓是顾客与企业双赢的一大秒招。
 <br/>
 <br/>
 新唐人日本记者站采访报导
</div>
