### 中共驻巴国领馆遇恐袭 2名警察殉职
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】当地时间11月23日，中共驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴基斯坦.html" target="_blank">
  巴基斯坦
 </a>
 领馆遭遇武装恐袭，两名当地警员在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交火.html" target="_blank">
  交火
 </a>
 过程中丧生。事发后，一支来自该国最贫困省份的组织宣布对袭击负责，并表示袭击目标是中共领事馆。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴基斯坦.html" target="_blank">
  巴基斯坦
 </a>
 黎明新闻电视台报导，当地时间11月23日9时30分左右，中共驻卡拉奇领事馆附近发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交火.html" target="_blank">
  交火
 </a>
 。
 <br/>
 <br/>
 俄罗斯卫星网报导称，3到4名不明身份人员试图闯入中共驻卡拉奇领馆，向馆外驻守的安保人员开枪，随后安保人员进行了还击。
 <br/>
 <br/>
 巴基斯坦当地媒体称，3到4人袭击了中共驻卡拉奇市的领事馆，向警卫开枪，他们用手榴弹炸毁了领事馆的大门。事发后，一支庞大的安全部队已抵达现场，总领馆被大批军警重重包围。
 <br/>
 <br/>
 巴基斯坦《论坛快报》援引信德省警方称，嫌疑人试图进入卡拉奇领事馆。该地还发生多次爆炸，两名警察确认死亡。当地电视台播出现场冒烟片段，有目击者称看见爆炸。
 <br/>
 <br/>
 防御承包商和分析师Ikram Sehgal说，两名警察牺牲后，武装分子朝领事馆大门行进。Ikram Sehgal的安保人员立即关闭大门，来领事馆办理业务的平民被领进馆内，中国职员被带到安全地带。
 <br/>
 <br/>
 巴基斯坦警方证实，位于港口城市卡拉奇的中共总领事馆遭遇不明身份枪手袭击，两名当地警员在交火过程中丧生。
 <br/>
 <br/>
 巴基斯坦外交部长Shah Mehmood Qureshi说，两名恐怖分子已经被击毙，他们正在与中共总领事馆取得联系。他还说，巴基斯坦将和中共一道击败欲阻止巴基斯坦恐怖分子的发展的势力。
 <br/>
 <br/>
 巴基斯坦三军新闻局（ISPR）称，3名恐怖分子被击毙，所有中国公民都安全，局势已在控制之下。
 <br/>
 <br/>
 事发后，一个声称与〝俾路支省解放军〞（Baluch Liberation Army）武装组织，有关联的社交网路帐户宣布对袭击负责，并表示袭击目标是中共领事馆。
 <br/>
 <br/>
 俾路支省是巴基斯坦最贫困的省份。俾路支解放军是俾路支省的一个分离主义组织，该地区地域辽阔，暴力事件频发。
 <br/>
 <br/>
 英媒BBC驻伊斯兰堡记者凯尔马尼（Secunder Kermani）称，〝俾路支省解放军〞并非伊斯兰极端武装，而是争取俾路支省从巴基斯坦独立的民族主义武装组织。这支武装已在区内活动多时。
 <br/>
 <br/>
 目前尚无法证实该组织的说法。袭击发生的区域，位于阿拉伯海附近，包括一个大型商场。报导称，几英里外都能听到交火的声音，社交媒体上的目击者发布了烟雾升入空中的照片。该地区的学校进入一级防范状态。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
