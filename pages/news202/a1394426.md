### 美国务卿蓬佩奥4会金正恩 为2次川金会铺路
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 5日展开亚洲行访问东北亚各国，6日先抵日本，会见首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 ，讨论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 核武、飞弹和日人绑架问题。7日搭专机再飞抵平壤，与朝鲜领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 会面，并说讨论相关废核问题，也为二次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川金会.html" target="_blank">
  川金会
 </a>
 铺路。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 （Mike Pompeo）抵东京后与安倍会晤，安倍感谢蓬佩奥会晤
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 前，先到访东京。而就朝鲜无核化议题，两人都同意，美日在朝鲜半岛问题上，应该紧密合作。
 <br/>
 <br/>
 蓬佩奥对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 说：〝对于如何继续下去，我们会在充分协调后达成统一见解。若朝鲜非核化要成功，就须如此。〞并称，〝我们也会提出被绑架者的问题。〞
 <br/>
 <br/>
 金正恩虽就朝鲜半岛非核化做出承诺，但日本仍视朝鲜为〝严重威胁〞，且正积极推动可在大气层外拦截飞弹弹头的陆基神盾（AegisAshore）作战系统，以强化防御弹道飞弹的能力。

 结束日本行程后，蓬佩奥搭乘专机飞抵平壤，与朝鲜领导人金正恩会面。蓬佩奥在推特贴出自己在东京搭乘专机前站在舱门口挥手的照片，写着〝下一站#平壤，准备会晤金主席和继续履行我们的工作，以落实总统和金主席在#新加坡高峰会做出的承诺〞。
 <br/>
 <br/>
 随后再于北京时间下午4时30分左右上传与金正恩会晤照片。并留言称，我访问平壤并与金主席会面，我们将以新加坡美朝首脑会谈达成的协议为基础，继续推动有关事宜取得进展。

 据了解，川普与金正恩于6月会面后，有关〝弃核〞议题迟迟未有进展，蓬佩奥此次是半年来第4度平壤行，此前的行程都分别出现变数。
 <br/>
 <br/>
 蓬佩奥近期仍坚持国际对朝鲜制裁须予维持，直到其放弃核武，他3日称，尽管俄罗斯和中共〝起意可开始思索降低制裁力道的适当时机〞，但上周联合国大会仍一致支持对平壤制裁。
 <br/>
 <br/>
 结束访朝行程后，当天启程飞往韩国，与韩国总统文在寅进行会晤。据韩联社报导，蓬佩奥于当天下午抵达韩国京畿道乌山美军基地。蓬佩奥将拜会韩国总统文在寅，之后同韩国外交部长官康京和举行晚餐会，介绍访朝结果。
 <br/>
 <br/>
 蓬佩奥将于8日从韩国启程前往中国，向中方介绍访朝结果。
