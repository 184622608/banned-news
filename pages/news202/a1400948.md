### 日拟花逾兆日圆增购百架F-35 改装准航母搭载战机
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】日本为因应邻国强化军备及顾及美国总统川普要求扩大购买美国防卫装备的想法，正计划再向美国采购100架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-35A.html" target="_blank">
  F-35A
 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-35B.html" target="_blank">
  F-35B
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_隐形战机.html" target="_blank">
  隐形战机
 </a>
 ，以取代部分老旧的F-15战机。
 <br/>
 <br/>
 日本经济新闻报导，目前日本拥有约200架F-15，其中一半不能升级，它们将被新战机取代。
 <br/>
 <br/>
 欲购买的F-35战机定位在最新锐的第5代战机，有日本购买的A型机，作为现役F-4战机的后继机；及可短场起飞、垂直降落的B型机，为此自卫队将改装旗下的〝准航母〞出云号及加贺号直升机护卫舰。
 <br/>
 <br/>
 报导说，原本的计划，日本将采购42架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-35A.html" target="_blank">
  F-35A
 </a>
 ，而打算增购的战机则包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-35B.html" target="_blank">
  F-35B
 </a>
 ，以每架逾100亿日圆(约8850万美元)价格，追加购机预算可能突破一兆日圆。预计2024年度前依序接机。
 <br/>
 <br/>
 日本政府将于下月公布新版防卫计划大纲，到时正式宣布增购战机的计划。
 <br/>
 <br/>

   </font>
  </a>
 </center>
 <br/>
 据悉，增购F-35战机的背景，跟川普一再要求增加购买美国防卫装备有关。增加购买高价战机，可避开川普要求降低对日贸易逆差的压力，也可为明年初可能正式谈判商品贸易协定（TAG）增加有利的想法。
 <br/>
 <br/>
 另外，在日本周边的中共与俄罗斯，都在导入最新世代的战机，增购新机也是为了因应邻国强化军备。
