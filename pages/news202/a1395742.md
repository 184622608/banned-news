### 万锦市众多候选人：我们不要大麻店
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】加拿大大麻合法化，对于大麻的销售，由各省立法，根据安省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻销售立法.html" target="_blank">
  大麻销售立法
 </a>
 ，市级政府可以选择禁止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻零售店.html" target="_blank">
  大麻零售店
 </a>
 ，也可以修改市级法律来限制在公众场所等吸食大麻，我们来看看安省华裔人口最集中的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万锦市.html" target="_blank">
  万锦市
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_候选人.html" target="_blank">
  候选人
 </a>
 们怎么说。
 <br/>
 <br/>
 前区议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_候选人.html" target="_blank">
  候选人
 </a>
 Nirmala Armstrong：〝作为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万锦市.html" target="_blank">
  万锦市
 </a>
 ，我们很自豪，我们已经表达了我们的立场，投票否定（零售大麻店），我认为，我们不要它（大麻店）在这里〞
 <br/>
 <br/>
 今年联邦政府公布大麻合法化时间表后，安省的华裔人口众多的万锦市（Markham）、列治文山市（Richmond Hill）都投票反对在该市开设大麻店，离多伦多20公里远的国王镇（king City）议会也在去年9月投票禁止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻零售店.html" target="_blank">
  大麻零售店
 </a>
 。
 <br/>
 <br/>
 今年安省新当选的保守党政府，修改了前政府的大麻销售法案，所有市级政府都有一次性的机会，在明年1月22日前决定，是否允许开设大麻零售商店，也可以修改市府法律来限制吸食大麻的地方。这也意味着所有市级政府在选举后，要再次做出决定。
 <br/>
 <br/>
 万锦市第五选区候选人律师Scott Au：〝如果人们想买大麻的话，他们可以去士嘉堡、Pickering、旺市购买，在万锦市我想声明，想对我们的居民说，我们不支持卖大麻，特别是对我们的孩子们，不能在万锦市到处吸大麻。〞
 <br/>
 <br/>
 万锦市第五选区候选人Tommy Chan：〝我在大麻上的立场是一个大大的‘不’。〞
 <br/>
 <br/>
 万锦市市长候选人Jawed Syed:〝我在这里养育了我的两个孩子，我非常反对这种事情（大麻），我不希望我的孩子上瘾。〞
 <br/>
 <br/>
 万锦市第7选区前议员候选人Khalid Usman：〝我在政纲中清清楚楚的写着，我不支持大麻，我在万锦市议会做的一件事就是，反对大麻，我们不要大麻店。〞
 <br/>
 <br/>
 市府政府固然可以禁止大麻商店，但居民可以开车到任何地方去买，必然给执法带来挑战。第五选区候选人黄家富Fred kWong认为，在即成的事实面前，市级政府能做的有限。
 <br/>
 <br/>
 第五选区候选人黄家富Fred kWong：〝政策和现实，很多人的思想还是反对这个政策，政策完全不在我们的掌握之中，如果你不喜欢这个政策，下次选举，把他们选下去。〞
 <br/>
 <br/>
 大麻合法化是联邦自由党在2015年大选时为吸引年轻选民的承诺之一，总理特鲁多曾公开表示，自己年轻时就吸大麻。
 <br/>
 <br/>
 第五选区候选人黄家富Fred kWong：〝立法者（国会议员）们不知道这对国家的未来会有怎样的后果，一直推动，没有真正谘询受影响的人们。每个人请给自己写封信，即使你19岁，你也有目标和理想，写下来，也写下你使用大麻的情况，2、3个月后看看，这只是个经验，还是变成在消耗着你的生命，因为直到目前还没有相关研究，你使用大麻后人生开始一路下滑。〞
 <br/>
 <br/>
 新唐人记者刘海英多伦多报导
</div>
