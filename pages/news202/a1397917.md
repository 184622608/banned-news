### 韩媒：美韩军演取消或延迟 协防战力恐下降
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】随着半岛局势升温，美韩联合军演不是取消就是推迟。对此，韩媒周六（11月1日）发表社论称，韩美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_协防战力.html" target="_blank">
  协防战力
 </a>
 势将因此急趋下降，情势堪忧。
 <br/>
 <br/>
 中央社引述韩国〝东亚日报〞周六以〝推迟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩美军演.html" target="_blank">
  韩美军演
 </a>
 长期化，令人担忧联合防卫能力下降〞为标题发表社论称，美韩联合军事演习的空白期正在迈向长期化，当一连串的韩美联合军演因非核化谈判而延迟或停摆，韩美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_协防战力.html" target="_blank">
  协防战力
 </a>
 势将急趋下降，情势堪忧。
 <br/>
 <br/>
 社论称，10月31日举行的韩美年例安全谘商会议决定，原定于12月进行的美韩联合空中演习〝警戒王牌〞延期举行，军方相关人士预测，明年3至4月的〝关键决断〞、〝秃鹰〞军演也很有可能推迟举行。
 <br/>
 <br/>
 特别是，如果预定于下周举行的朝鲜美国高阶会谈、年底朝鲜国务委员长金正恩回访首尔、明年初第2次朝鲜美国高峰会谈等〝接力对话〞相继举行，可能会营造出把延迟演习视为理所当然的氛围。
 <br/>
 <br/>
 果真如此，今年4至5月缩减规模实施的〝关键决断〞和〝秃鹰〞后，美韩联合军演将出现一年多的空白。
 <br/>
 <br/>
 社论称，美韩联合演习大体分为上半年的〝关键决断〞、〝秃鹰〞和下半年的〝乙支自由卫士〞（Ulchi Freedom Guardian）两部分。此外，还有中队、大队级海军陆战队联合训练等小规模训练，但，熟悉和检验有事时根据美军增援联合作战计划程序在韩半岛的展开程序，是以〝关键决断〞、〝鹞鹰〞、〝乙支自由卫士〞为核心的。
 <br/>
 <br/>
 但在美国总统川普（特朗普）与朝鲜领导人金正恩6月12日在新加坡举行高峰会后，华府表示将与韩国暂停几项军演，包括夏季〝乙支自由卫士〞联合军演。
 <br/>
 <br/>
 社论称，对朝鲜来说，中断美韩联合军事演习是军事上最大的夙愿。美韩军事当局表示，将研究另外准备演习以弥补演习空白和检验战时作战指挥权移交条件，但朝鲜有可能就此也会说三道四。
 <br/>
 <br/>
 专文称，韩美军演并不是为了攻击朝鲜。韩美两国军官的任职每隔1至2年就会更换一次，如果演习中断一年以上，韩美两国将处于连一次配合都没做过的状态。演习空白的长期化会与战时作战权转换相结合，将导致韩美联合防卫能力的实际下降。
 <br/>
 <br/>
 被内定为驻美韩军司令的罗伯特‧艾布伦斯，9月在美国参议院听证会上忧虑地表示：〝很明显，因为推迟演习，美韩联军的准备态势已明显削弱。〞美韩联合军演是最强有力的对朝鲜遏制政策，也是让朝鲜走上对话舞台的压迫手段，长期空白断不可行。
 <br/>
 <br/>
 韩国防长郑景斗（Jeong Kyeong-doo）提到在五角大厦与美国国防部长马蒂斯（Jim Mattis）举行的〝第50次韩美安全谘商会议〞时表示，美韩正在检讨〝一些主要的、大型的演习〞未来该如何做。
 <br/>
 <br/>
 郑景斗并说：〝在12月1日以前，我们将对明年可能举行的任何重大演习作最后决定。〞
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
