### 美中下一轮贸易战？ 港商加速撤离大陆
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 持续延烧，外媒报导指出，倘若下月底〝川习会〞未能达成协议，美国将对全部剩下的中国商品
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加征关税.html" target="_blank">
  加征关税
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港商.html" target="_blank">
  港商
 </a>
 代表表示，贸易战一旦升级，将有更多港商加速撤离大陆。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 不断升级，以大陆为主要生产基地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港商.html" target="_blank">
  港商
 </a>
 忧心忡忡。躲过两次加税的港商表示，如果美国全面征税的话，对钟表业打击很大。
 <br/>
 <br/>
 香港表厂商会会长林伟雄：〝我们下半年接单的情况有放缓的迹象。如果假设美国全面加25%的关税，对5000亿美元的所有产品，基本上钟表无一避免。这方面我觉得这对钟表行业打击必然是很大的，也是很严峻的。〞
 <br/>
 <br/>
 香港贸发局统计，贸易战下，香港出口前景较2008年海啸更差，第四季度或现负增长，更多港商加速撤离大陆。
 <br/>
 <br/>
 香港贸发局研究总监关家明：〝有迹象显示越来越多的港商考虑，甚至开始有行动，怎样快点去找地方去搬。〞
 <br/>
 <br/>
 香港表厂商会会长林伟雄：〝我们11月8到10日出团去泰国，看一些工厂，尤其是一些配件厂，我们商会一出团，已经即刻爆团了。〞
 <br/>
 <br/>
 受到贸易战影响，九月香港整体出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_口货值.html" target="_blank">
  口货值
 </a>
 同比上升4.5%，远低于市场预期增加8.6%；进口同比上升4.8%，远低于预期增加10.5%。有港商预计，明年美国一旦全面开征关税，广东省或有一半港商将倒闭。
 <br/>
 <br/>
 新唐人记者梁珍、郭威利、宋碧龙香港报导
</div>
