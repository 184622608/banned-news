### 〝川习会〞前中共态度软化 学者：贸易休战仍不乐观
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 开打后的首场〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞将在月底举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会登场。外界留意到，经过近一年的经贸摩擦后，北京的态度已从最初的〝以牙还牙〞，转向〝希望双方能相向而行〞。不过有学者认为，监于美国提出的要求将改变中共的工业政策，要达到休战的结果不容乐观。
 <br/>
 <br/>
 在近日举行的中共外交部媒体吹风会上，中共商务部副部长王受文就中美元首在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会期间可能举行的会谈表示，希望双方〝能够在相互尊重，在平衡诚信互利的基础上相向而行，最后找出问题的解决方法〞。
 <br/>
 <br/>
 《香港经济日报》称，综观今年中美贸易摩擦演变，虽然北京的公开表态始终是〝不愿打、不怕打〞、〝希望回到谈判桌上〞，但调门已出现显著变化，从最初〝不惜一切代价〞、〝打硬仗〞、〝以牙还牙〞的强硬，逐渐转向川习通话时的表态〝两国经济团队要加强接触，就双方关切问题展开磋商，推动中美经贸问题达成一个双方都能接受的方案〞。即便是面对美国副总统彭斯在APEC会议上的批评，北京的表态仍显克制，避免与之正面冲突，影响〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞气氛。
 <br/>
 <br/>
 报导引述中国人民大学国际关系学院教授时殷弘表示，中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 存在两个层面，而且长期不会改变：一层是贸易方面的高关税征收博弈，另一层是高科技封锁。北京希望在关税层面的贸易战上能有一个较长时间的休战期，以争取一个压力较小的外部环境，加速经济优化等方面的调整。
 <br/>
 <br/>
 早前外界流传一份北京向美国提出的让步清单，并将其细分为142个项目，归类为三大类议题：愿意进一步谈判、已经在处理，以及无法接受。而川普（特朗普）对此表示，清单缺少几个重要项目，他无法接受。
 <br/>
 <br/>
 时殷弘认为，北京作出的让步主要集中在巨量进口美国商品和大幅放宽对美市场准入，但美国核心关切的包括中共停止盗窃美国知识产权、停止向企业提供特惠和补贴、停止强迫在华美企转让技术等具体内容，而且不断重申〝要的不是承诺而是执行落实〞。
 <br/>
 <br/>
 时殷弘表示，从目前看，达成休战的基本协议必须包含北京做出重大基本让步，而美国的要求将会改变中共的工业政策，〝这对北京来说是‘城下之盟’，显然双方立场仍相差较远。〞
 <br/>
 <br/>
 美国方面则对中美贸易问题发出了不同声音。美国贸易代表办公室在20日再度批评中共未改变〝不公平〞的贸易做法，指责中共支持的知识产权和技术盗窃活动还在继续。而川普本人则是〝唱红脸〞，称他已经为〝川习会〞作好充分准备，并强调与习近平的〝关系很棒〞。
 <br/>
 <br/>
 不过，中共外交部发布的消息只提到习近平将在11月27日至12月5日出访西班牙、阿根廷、巴拿马和葡萄牙，并在期间出席G20峰会，未有透露〝川习会〞的具体日期。加上中美两国官僚层面未就贸易问题展开正式谈判，给这场举世瞩目的会面蒙上未知，有舆论一度担忧〝川习会〞可能存在变数。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：祝馨睿）
