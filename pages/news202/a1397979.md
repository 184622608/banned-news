### 假日季来临 伦敦市场节日礼物新概念
------------------------

<div class="wysiwyg">
 【新唐人2018年11月03日讯】假日季节到了，很多人在筹划买什么礼物、怎么装饰家里，商家也在想方设法开发新产品和新包装。一起看看今年有什么新花样。
 <br/>
 <br/>
 和往年一样，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞节.html" target="_blank">
  圣诞节
 </a>
 少不了蛋糕。今年伦敦的节日市场上，将出现更多美味、又有益健康的蛋糕。
 <br/>
 <br/>
 蛋糕店业主Valerie Hufton：〝我们虽然有一些非素食的，但多数还是素食蛋糕。我们开发了自己的皇家糖衣，里面不含鸡蛋，还有杏仁蛋白软糖，我们还要找到素食酒精。〞
 <br/>
 <br/>
 在包装上，店主准备了更多经济实惠的纸盒、大篮子，顾客可以反覆使用。
 <br/>
 <br/>
 艾米莉从事包装袋设计20多年，她说，自然、手工制作的礼品装饰，是今年的流行趋势。
 <br/>
 <br/>
 华裔业主陈先生，拥有一家企业，生产各种购物和礼品袋。
 <br/>
 <br/>
 华裔购物袋业主Benedict Chan：〝特别是在圣诞节，有很多人用塑料袋，你在街上能看到，还有包装材料等等，很大的浪费。我们有一系列棉质礼品包装、礼品袋，可以多次使用。〞
 <br/>
 <br/>
 布斯先生是知名巧克力品牌洛可可的高管，他说他们的产品不用化学试剂杀菌，不仅有利于工人健康，也有益顾客。
 <br/>
 <br/>
 今年的圣诞节产品博览会将持续到4号。
 <br/>
 <br/>
 新唐人记者易文张琪综合报导
</div>
