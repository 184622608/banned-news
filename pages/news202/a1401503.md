### 向俄示威？乌克兰亚速海军演 禁俄成年男子入境
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】 近日，俄罗斯与乌克兰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚速海冲突.html" target="_blank">
  亚速海冲突
 </a>
 引发的紧张局势持续发酵。在俄罗斯封锁了亚速海两个港口并威胁要在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克里米亚.html" target="_blank">
  克里米亚
 </a>
 部署地对空防空飞弹系统后，乌克兰军方在亚速海附近陆地上进行了军演。同时乌克兰官方宣布，戒严期间禁止俄罗斯16至60岁男子入境。
 <br/>
 <br/>
 日前，乌克兰国防部对外发放了乌军在亚速海（Sea of Azov）附近军演的片段，并称演习是模拟遭俄罗斯入侵的作战应对。
 <br/>
 <br/>
 视频中画面显示，乌克兰军方进行的是实弹演练，除了出动坦克等陆上部队，也有派出直升机参加军演，还发射了导弹及防空炮。舆论认为这是针对俄罗斯近日扣押乌克兰船舰等一系列挑衅行为做出的回应。
 <br/>
 <br/>
 与此同时，乌克兰官方颁布禁令，宣布在该国戒严期间，禁止俄罗斯16至60岁男子入境，唯进行〝人道主义活动〞者不在此限。
 <br/>
 <br/>
 目前接酿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克里米亚.html" target="_blank">
  克里米亚
 </a>
 的边境已被乌克兰军方封锁，非乌克兰国民不许进出，意味外国人要经陆路进出克里米亚半鸟，只能依靠连接俄罗斯南部的克里米亚大桥。
 <br/>
 <br/>
 乌克兰的这些示威性质的举动，都是针对近日在亚速海附近的克赤海峡与俄罗斯爆发的武装冲突而采取的行动。
 <br/>
 <br/>
 本月25日，俄罗斯海军突然在亚速海附近的刻赤海峡（Kerch Strait）向经过那里的
 <br/>
 3艘乌克兰船舰开火，俄方出动战机及直升机向乌方船隻发射了火箭炮，并扣留了这3艘船隻与所有人员。
 <br/>
 <br/>
 次日，乌克兰总统波洛申科即宣布戒严30天，并警告俄罗斯，乌俄之间的局势可能演变为〝全面战争〞。作为反击，俄罗斯则宣布将在克里米亚部署S400地对空防空飞弹系统。
 <br/>
 <br/>
 乌克兰建设部长Volodymyr Omelyan29日表示，位于亚速海的两个港口，别尔江斯克（Berdyansk ）与马里乌波尔（Mariupol），已遭到俄罗斯封锁，禁止任何船隻进出，已有35艘船在活动时遭到限制，只被允许开往俄罗斯港口。
 <br/>
 <br/>
 Omelyan称，〝俄罗斯的目标很简单，藉由封锁乌克兰在亚速海的港口，逼我们放弃自己的领土，这块依照国际法规定属于我们的领土。〞
 <br/>
 <br/>
 据公开的资讯，亚速海东为俄罗斯、北接乌克兰、西临克里米亚，行经的船隻主要载运谷物和钢铁。
 <br/>
 <br/>
 针对乌俄之间近日发生的冲突与对峙，联合国秘书长古特雷斯（Antonio Guterres）呼吁，国际社会合力促使各方进行有意义对话，协助缓解紧张局势，避免事态恶化。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
</div>
