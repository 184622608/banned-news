### 斯里兰卡陷政治危机 国会爆全武行10多人受伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯里兰卡.html" target="_blank">
  斯里兰卡
 </a>
 国会14日通过对新总理拉贾帕克萨的不信任案，15日议长正式宣布时，遭拉贾帕克萨所属政党的国会议员团团围住，最后爆发推挤冲突，过程中有人拿东西乱砸，造成10多人受伤，议会被迫中止。
 <br/>
 <br/>
 总统席瑞塞纳（Maithripala Sirisena）10 月 26 日撤换总理威克瑞米辛赫（Ranil Wickremesinghe），转而任命前强人总统拉贾帕克萨（Mahinda Rajapakse）出任新总理。
 <br/>
 <br/>
 不久，席瑞塞纳宣布解散国会，提早于明年1月举行大选。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯里兰卡.html" target="_blank">
  斯里兰卡
 </a>
 最高法院13日裁定，此举违宪。
 <br/>
 <br/>
 议长贾雅苏瑞亚（Karu Jayasuriya）15日宣布，拉贾帕克萨和他的内阁遭不信任投票否决，斯里兰卡现在没有政府。
 <br/>
 <br/>
 没想到，拉贾帕克萨所属政党的国会议员将议长团团包围，顿时两派人马爆发推挤冲突、打成一团，过程中还有人拿东西砸，不少人从阶梯上摔下来，还有议员拿水桶朝议长席位丢，造成10多人受伤。在吵闹声中，议长宣布休会。

 拉贾帕克萨则以议员身分发言，〝我们225位议员找不出方法解决这个冲突，那就把责任交给我国1千5百万位选民吧。〞呼吁国会举行大选，结束这场政治危机。
 <br/>
 <br/>
 不过，城市和西部开发部长长拉纳瓦卡：〝我们呼吁总统接受国会多数决。〞
 <br/>
 <br/>
 威克瑞米辛赫任内实施经济改革，受到西方国家欢迎，遭总统革职后，拒绝搬离官邸。贾帕克萨虽然未获信任案，但不代表威克瑞米辛赫能重新回锅，因为总统握有指派下一任总理的权力。
 <br/>
 <br/>
 总统目前持续和议长以及其他党派的领袖碰面，希望能透过协商，解决这场政治危机。
