### 声乐大赛获奖选手感言 唱美声身心灵受益
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】由本台主办的第七届全世界华人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美声唱法.html" target="_blank">
  美声唱法
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_声乐大赛.html" target="_blank">
  声乐大赛
 </a>
 11月10号在纽约圆满落幕。获奖选手分享了他们学唱美声，身心灵受益的美妙体验，并感谢大赛给予他们演唱
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中文歌曲.html" target="_blank">
  中文歌曲
 </a>
 的难得机会。
 <br/>
 <br/>
 大赛铜奖得主 蔡维恕：〝美声需要特别开阔的身心灵，全部都要打开。你内心的喜怒哀乐或是恐惧也好，全部都要表现出来，所以你要真真实实的面对你自己。〞
 <br/>
 <br/>
 大赛铜奖得主 洪筱欢：〝因为我们的乐器就在我们自己的身体里面，所以有一个健康的心灵，维持健康的生活方式，其实是很重要的。〞
 <br/>
 <br/>
 大赛铜奖得主 蔡孟融：〝古典乐和美声法，这样的曲调会建筑你的理性和感性可以均衡。〞
 <br/>
 <br/>
 大赛银奖得主 王佩颖：〝因为我们学声乐的基本功就练气，气息的控制。不是只有唱歌，对身体也非常有帮助，就是要作息规律。〞
 <br/>
 <br/>
 来自美国麻州的Simeng Wu捧得大赛女子组金奖，她在准备中文参赛曲目《红豆词》时，用心揣摩歌词的深层涵义，感受到中华文化的博大精深。
 <br/>
 <br/>
 大赛金奖得主 Simeng Wu：〝美声来唱中文真的是很有韵味，每个元音和辅音都加在一块儿，声音圆润，字非常清晰，然后韵也很到位，还有音高、音准还有音律，都要做到到位的话，真的是很美很美，真的感到中国的文化很深厚。〞
 <br/>
 <br/>
 选手们还表示，在国际舞台上，用美声唱
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中文歌曲.html" target="_blank">
  中文歌曲
 </a>
 ，是大赛难能可贵的地方。
 <br/>
 <br/>
 大赛银奖选手 王佩颖：〝我唱西方歌剧的感觉，和我唱中文歌曲完全不一样。我觉得这个比赛的意义就是，不管是哪一个国界的人，听到中文歌，心声会唤醒。〞
 <br/>
 <br/>
 大赛铜奖得主 洪筱欢：〝因为我有的是西方音乐方面的教育和文化上的薰陶。我怎样把那个东西运用到自己的文化，然后用自己的语言表达出来，去平衡他，两者互相Interact，所以是一个很不错的学习经验。〞
 <br/>
 <br/>
 大赛铜奖得主 蔡维恕：〝中文歌曲的表现上面，能够把诗词字句等东西，能够分享给观众，让大家了解到我们音乐的美，我们文字的美。这对我们华人的文化、中文的艺术，在做一个发扬和推广。所以，我觉得这个比赛非常非常的难能可贵。〞
 <br/>
 <br/>
 新唐人记者安心、奥利弗纽约报导
</div>
