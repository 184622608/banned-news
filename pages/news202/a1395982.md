### 韩朝扫雷工作20日结束 共同警备区将对外开放
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】根据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 9月19日签订军事协议，双方将在10月25日之前撤出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共同警备区.html" target="_blank">
  共同警备区
 </a>
 的兵力和武器。韩国消息人士周五（19日）称，韩朝为解除板门店共同警备区（JSA）武装而实施的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扫雷工作.html" target="_blank">
  扫雷工作
 </a>
 20日结束。
 <br/>
 <br/>
 韩联社报导，韩朝从本月1日开始在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共同警备区.html" target="_blank">
  共同警备区
 </a>
 实施
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扫雷工作.html" target="_blank">
  扫雷工作
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 政府一位消息人士19日透露，韩方扫雷作业已经完成，朝方也已进入收尾阶段，相关工作将于20日结束。朝方共发现5枚地雷并已爆破，韩方没有找出地雷。
 <br/>
 <br/>
 韩朝于9月19签订的军事协议规定，双方需在完成扫雷工作后5天之内撤出所有兵力和武器。朝方将撤除5个哨所，韩方将撤除4个哨所，共同警备区内的冲锋枪、AK-47步枪、手枪等武器也会全部撤出。
 <br/>
 <br/>
 解除武装后，共同警备区将对外开放，双方将各派35人在共同警备区内执勤，这些人不会携带枪械，同时将佩戴黄底蓝色的〝板门店民事警察〞袖章。韩朝民众以及外国游客等可于上午9点到下午5点自由出入参观。
 <br/>
 <br/>
 另一方面，韩朝和联合国军司令部还将拆卸或调整共同警备区内的监视设备，并将围绕解除武装后共同管理机构的构成、任务、运营方式等事宜举行3方协商机制的第二次会议。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
</div>
