### 印尼狮航客机坠海凶多吉少 机上189乘客中有23名官员
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】一架从印尼首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雅加达.html" target="_blank">
  雅加达
 </a>
 起飞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮子航空.html" target="_blank">
  狮子航空
 </a>
 （Lion Air）内陆客机，周一(29日)上午6时20分起飞后13分钟失联。搜救单位稍后透过简讯表示，〝当局已经证实飞机
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠毁.html" target="_blank">
  坠毁
 </a>
 〞，机上搭载189人恐全部罹难，据最新消息，机上还包括至少23名印尼政府官员。
 <br/>
 <br/>
 综合媒体报导，狮航这架飞机原本预定今天上午6时20分从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雅加达.html" target="_blank">
  雅加达
 </a>
 的苏卡诺哈达国际机场（Soekarno-Hatta Airport）飞往邦加岛的邦加槟港机场（Pangkal Pinang），但飞机起飞13分钟后，于6时33分失联。
 <br/>
 <br/>
 据飞航追踪网站FlightRadar24，失事客机编号为JT610，是一架波音737MAX8，起飞后一度升空至5450英呎，接着在不到一分钟的时间内便急速坠落。
 <br/>
 <br/>
 印尼救援部门发言人稍后表示，客机起飞后约13分钟与航空管制塔失去联络，证实已
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠毁.html" target="_blank">
  坠毁
 </a>
 海中。
 <br/>
 <br/>
 失事客机当时载有188人，包括178名成年乘客、1名小孩、2名婴儿及7名机组人员。但彭博新闻报导，印尼国家运输安全委员会修正最新机上人数为189人，比稍早多出1人，分别包括2名孩童和1名婴儿及8名机组员，目前全部生死未卜。
 <br/>
 <br/>
 据了解，客机从苏加诺哈达国际机场出发，预定飞往航程只有一小时多的苏门答腊东方邦加岛最大城镇邦加槟港，但客机于西爪哇省加拉璜(Karawang)一带失联。
 <br/>
 <br/>
 目前救援人员赶往距离雅加达国际机场外约24公里处的海域上搜救，不久传出搜救人员已经在海面上发现救生衣及手机等乘客个人用品。
 <br/>
 <br/>

 <br/>
 印尼廉价航空百家争鸣，机票价格屡创新低，低票价让人忧心飞航安全。而狮航是印尼最大的民营航空公司，也是东南亚著名的廉航，但安全纪录不佳，2002年至2015年间共发生6起事故。
 <br/>
 <br/>
 其中2013年，由万隆飞峇里岛的波音737客机在降落时入海，机体断成两截，机上108人中45人受伤。2018年5月，由泰国清迈飞往曼谷的班机飞行时引擎突然爆炸，飞机返航安全着陆。
 <br/>
 <br/>
 欧盟基于安全考虑，于2007年禁止包括狮航在内的数十家航空公司飞往欧洲，狮航至今仍在欧盟禁飞名单上。
