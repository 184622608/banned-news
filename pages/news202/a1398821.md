### 专家曝中美若开战 俄罗斯站边华盛顿
------------------------

<div class="wysiwyg">
 【新唐人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 时间2018年11月10日讯】尽管中俄两国领导人一直在努力推动两国关系的发展。但有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 专家表示，如果中美之间开战，俄罗斯一定要站在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华盛顿.html" target="_blank">
  华盛顿
 </a>
 一方，因为北京将给俄带来更大的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战略威胁.html" target="_blank">
  战略威胁
 </a>
 。
 <br/>
 <br/>
 11月9日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 列格努姆通讯社，引述俄罗斯莫斯科人文大学国际关系和外交学教研室主任普拉托什金（Nikolay Platoshkin）的话说，如果中美对抗，俄罗斯应当站在美国一方，因为自1945年以后，中共曾两次攻击苏联，这是前所未有的情况，因此在20世纪70年代，苏联军队的70%不是位于欧洲，而是位于中国边境。
 <br/>
 <br/>
 普拉托什金说，鉴于中美之间的矛盾越来越多，对于俄罗斯来说，最重要的就是尽可能回避冲突。
 <br/>
 <br/>
 根据此前〝全俄社会舆论研究中心〞开展的一项民意调查，在中美关系陷入紧张的局势之下，53%的俄罗斯人表示，选择中立；20%的俄罗斯人表示倾向于中方，但不会参与冲突；另有4%的人表示公开支持中共；6%的人公开支持美方。
 <br/>
 <br/>
 普拉托什金分析说，尽管俄罗斯对中方的好感仍在，但是中共试图使俄罗斯〝闭着眼睛出牌〞也是不现实的，俄罗斯不会卷入别国的冲突。
 <br/>
 <br/>
 9月份，俄罗斯与中共举行大型联合军演时，美国国防部长马蒂斯也曾说，由于长远利益不同，中共和俄罗斯走不到一起。因为每个国家通常会根据自己的利益来行动，〝而我看不到俄罗斯与中国（中共）的长远利益会使他们结成盟友。〞
 <br/>
 <br/>
 尽管中俄领导人分别称，两国〝友谊一直在加强〞，但ABC新闻引用专家的分析说，实际上中共或俄罗斯自己都不相信这两个国家会结成军事联盟。
 <br/>
 <br/>
 美国之音表示，包括俄罗斯军事专家亚历山大‧格尔茨（Aleksander Goltz）在内的许多分析人士也说，在俄罗斯与西方交恶、与美国对抗升级之际，中共并不是俄罗斯的盟友，更不会为俄罗斯而战。而俄军也一直把中共看成潜在威胁。
 <br/>
 <br/>
 英媒BBC评论文章，分析美中俄三国的关系认为，在中美全面对抗，贸易战持续升温的情况下，拉拢同受美国打压的俄罗斯，共同抗美，成为中共在外交和地缘战略的不二选择。
 <br/>
 <br/>
 但问题是对中俄双方而言，尽管有历史问题，以及存在对各自战略目的的猜疑，但两国关系的基调是稳定的，好也好不到哪儿，坏也坏不到哪儿，不会出现大起大落。
 <br/>
 <br/>
 但如果其中一方和美国关系好，或者两国都和美国好，双方关系就不可能过于密切，联盟更谈不上；但如果两国同受美国施压，尤其是美国将两国作为战略对手，那么，中俄很可能结成联盟。
 <br/>
 <br/>
 不过，评论也说，要判断中俄抗美联盟可走多远不好确定。因为这取决于美国，随着中美紧张局势加深，未来美国重新恢复和俄的正常关系，也不能完全排除。中俄欲建立的抗美联盟就会瓦解。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：赵云）
