### 这9个加拿大圣诞节趣事  你或许不知道
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】世界各地庆祝圣诞的方式各有特色。但是加拿大和圣诞节的关系可以说是非常密切，这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_趣事.html" target="_blank">
  趣事
 </a>
 你很有可能不知道哦！
 <br/>
 <br/>
 1.圣诞老人是加国公民
 <br/>
 2010年，加拿大政府正式宣布，圣诞老人成为加拿大公民。
 <br/>
 <br/>
 2.红鼻子驯鹿鲁道夫是加国人
 <br/>
 家喻户晓的圣诞卡通片《红鼻子驯鹿鲁道夫》，除雪人外，配音演员都是加拿大人。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞小精灵.html" target="_blank">
  圣诞小精灵
 </a>
 多是加国人
 <br/>
 自1982年始，圣诞老人邮局每年聘请大量加拿大小精灵，帮圣诞老人回复20多万封来信。
 <br/>
 <br/>
 4.这些地方天天过圣诞节
 <br/>
 西北地区的驯鹿站，新省的圣诞岛，诺亚，萨省的雪橇湖，安省的Holly镇，火鸡角，和曼省的雪花镇。
 <br/>
 <br/>
 5.浓郁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞树.html" target="_blank">
  圣诞树
 </a>
 情节
 <br/>
 加拿大每年砍将近550万棵圣诞树，平均每7个人就砍一棵。
 <br/>
 <br/>
 6.爱看芭蕾舞《胡桃夹子》
 <br/>
 1995至2011年1月圣诞期间，加拿大国家芭蕾舞团出演《胡桃夹子》，跳破了5548双舞鞋。
 <br/>
 <br/>
 7.爱送圣诞礼物
 <br/>
 Ipsos Reid 民调发现，每5名加拿大人中就有4个人有圣诞节送礼计划，人均花费近1182元。
 <br/>
 <br/>
 8.圣诞大餐少不了火鸡
 <br/>
 加拿大火鸡鸡农协会报告显示，2012年共吃掉880万只火鸡。
 <br/>
 <br/>
 9.政府大楼必有圣诞彩灯
 <br/>
 自1985年至今，各地政府大楼都会点亮圣诞彩灯，寓意全国13个省团结一心。
 <br/>
 <br/>
 看完这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_趣事.html" target="_blank">
  趣事
 </a>
 ，你是不是更加期待圣诞节的到来呢？
 <br/>
 <br/>
 新唐人记者心宇多伦多报导
</div>
