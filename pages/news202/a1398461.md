### 一战终战百年 伦敦塔旁点亮一万火炬
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】本周日是第一次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界大战.html" target="_blank">
  世界大战
 </a>
 终战一百年纪念日，法国和德国总统上周日齐聚德法边界城市斯特拉斯堡，参加纪念音乐会；英国也在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦塔.html" target="_blank">
  伦敦塔
 </a>
 点亮一万把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火炬.html" target="_blank">
  火炬
 </a>
 ，纪念在战争中死去的人们。
 <br/>
 <br/>
 号角声中，一万支
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火炬.html" target="_blank">
  火炬
 </a>
 点亮了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦塔.html" target="_blank">
  伦敦塔
 </a>
 旁已经干涸的护城河，纪念第一次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界大战.html" target="_blank">
  世界大战
 </a>
 结束一百周年。
 <br/>
 <br/>
 100年前的11月11日，德国法国签署了停战协议。
 <br/>
 <br/>
 从上周日到这周日，每天下午5点，志愿者们要用将近一个小时的时间把火炬点燃，之后一直燃烧4个小时。志愿者中很多人都有家人参加过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一战.html" target="_blank">
  一战
 </a>
 。
 <br/>
 <br/>
 在一战期间，1600多名英国人去欧洲大陆参战之前，在这条干枯的护城河里宣誓。
 <br/>
 <br/>
 陆军卫兵都铎的祖父曾参加过两次世界大战，他也来参加点燃火炬的仪式。
 <br/>
 <br/>
 英国陆军卫兵 都铎：〝回到这里是一件非常让人震撼的事，100年过去了。所有那些(在这里)签约去参战的人，我在同一地点向他们表达敬意。〞
 <br/>
 <br/>
 这周日，美国总统川普、俄罗斯总统普京等70多位各国领导人，将飞往法国巴黎，参加一战停战百年纪念仪式。
 <br/>
 <br/>
 一战5年的战火造成约1700万人丧生，2000多万人受伤。战火延烧到全球，欧洲是主战场。
 <br/>
 <br/>
 新唐人记者王羽音、张琪综合报导
</div>
