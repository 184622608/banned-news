### 博尔顿会普京 美退出中导条约计划不变
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】正在莫斯科访问的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 ，周二与俄罗斯总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 会谈。俄罗斯当局稍后宣布，美俄领袖计划11月在巴黎会面。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 上周宣布，由于俄罗斯多年来一直违反《中程导弹条约》，美方准备退出。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 周二表示，美国的举措很难称之为友好，不过莫斯科愿意与华府保持对话。他提议，如果能和川普总统直接对话，将对美俄关系大有改善。
 <br/>
 <br/>
 俄罗斯总统 普京：〝在我看来，尽管我们各自有不同的观点，但我们必须且需要找到接触点。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 回应，美方持同样观点。他表示，川普总统期待11月11日在巴黎与普京会面。当天是第一次世界大战结束100周年纪念日，法国将举行大型庆典。
 <br/>
 <br/>
 美国国家安全顾问 博尔顿：〝我相信川普总统期待与您在巴黎会面。尽管我们之间由于国家利益不同而存在差异，但在可能互相合作的领域内，两国的合作仍很重要。〞
 <br/>
 <br/>
 结束与普京的会谈后，博尔顿随后在记者会上宣布，美国计划退出《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中导条约.html" target="_blank">
  中导条约
 </a>
 》的决心不变，与普京的会面不会影响白宫的决策。
 <br/>
 <br/>
 新唐人记者曹菁综合报导
