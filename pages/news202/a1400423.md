### G20公报草案避提5字 中美幕后角力硝烟再起
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】月底的G20峰会期间，中美首脑会晤或将上演。中美在会晤前的角力仍在继续。英媒报导指，本届G20峰会的联合公报草案删除或弱化了多处对美国的批评，〝反保护主义〞措辞并未出现。
 <br/>
 <br/>
 英国《金融时报》日前报导了该报取得的峰会公报草案。报导称，在11月30日峰会开始前，公报的内容仍在谈判之中，并可能出现变化。
 <br/>
 <br/>
 草案中，呼吁各国〝持续开放市场，确保公平竞争环境〞，并〝认识到多边贸易体系的重要性〞，但是与往年相比，少了〝反对保护主义〞的措辞。
 <br/>
 <br/>
 报导指，自十年前论坛成立以来，〝反对保护主义〞一直是G20峰会的主要表述之一。但在去年G20与今年G7峰会上，因为被指责施行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易保护主义.html" target="_blank">
  贸易保护主义
 </a>
 〞，美国总统川普和各国领袖爆发了冲突。
 <br/>
 <br/>
 另外，虽然这次G20草案提到多边贸易体系的〝重要性〞，但与去年G20公报形容全球贸易体系扮演〝关键角色〞相比，今年的措辞明显降调。
 <br/>
 <br/>
 中美贸易战爆发后，北京一直将自己扮演成多边贸易体系的〝维护者〞，并抨击美国为〝保护主义〞。
 <br/>
 <br/>
 草案并要求，各国贸易部长要提出确保世界贸易组织(WTO)能继续发挥作用的提案，在明年峰会上提交报告。
 <br/>
 <br/>
 川普已放话，若WTO不改革，美国将退出WTO。而中共因为不遵守WTO规则，已成为众矢之的，美国提出的WTO改革被指明显针对北京。日前美、日、欧已联合提出WTO改革方案，其中增加多项针对中共的惩罚条款。
 <br/>
 <br/>
 另外，G20公报草案在气候行动方面的表述也比以前温和，内容仅简略提及2015年巴黎气候协定，并未呼吁在气候方面做更多承诺。而去年公报则抨击美国宣布退出巴黎气候协定，宣称气候协定〝不可逆转〞。
 <br/>
 <br/>
 此前，中共也对美国宣布退出气候协定提出批评，党媒还宣传中方可以替代美国，成为气候协定的〝领导者〞。
 <br/>
 <br/>
 近年来，中共无视国际规则的各种举动，已引起国际社会普遍不满。作为西方世界的龙头，美国反击中共的决心越来越坚定，这也越来越促使国际社会逐渐远离北京。
 <br/>
 <br/>
 《华尔街日报》报导，上周的APEC峰会上，与会的20多个国家都同意将反对〝贸易保护主义〞和〝所有不公平贸易行为〞写入联合公报，唯独中方强烈反对，最后导致联合公报难产。
 <br/>
 <br/>
 在APEC峰会上，中方继续指责美方是〝保护主义〞，但美方反击称，中共加入WTO以来，一直拒绝对外开放市场，并设置各种贸易壁垒，这才是真正的〝保护主义〞。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
