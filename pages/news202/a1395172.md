### 时事拼盘：华盛顿大主教辞职 澳洲遭遇罕见风暴冰雹
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】下面请看一组时事简讯
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_教宗方济各.html" target="_blank">
  教宗方济各
 </a>
 周五接受了美国华盛顿教区大主教乌尔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 。乌尔被指在1988至2006年担任匹兹堡教区主教期间，包庇性侵儿童的神父。今年8月一项调查报告显示，宾州6个教区300多名神父在过去70年间性侵了约1000名儿童。乌尔否认包庇指控，教宗则说他是代罪羔羊。
 <br/>
 <br/>
 强风暴雨周四袭击了澳大利亚昆士兰州东南部，风暴强度达到4级，带来冰雹，雷雨天和龙卷风等极端天气，造成大面积损失。有至少五人受伤，包括一名婴儿和一名小男孩。
 <br/>
 <br/>
 美国太空总署署长布莱登斯汀周五表示，他对美俄太空合作关系充满信心，并期望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国宇航员.html" target="_blank">
  美国宇航员
 </a>
 再次乘坐俄制的联盟号火箭升空。但本周发生的火箭发射失败也表明，开发其他进入太空的方式很重要。自美国2011年退出航天飞机计划以来，美国一直依靠俄罗斯火箭将宇航员运送到空间站。
 <br/>
 <br/>
 周四，非洲乌干达东部山区，暴雨引发山体滑坡，巨石滑落河中，引起河水泛滥，洪水将房屋摧毁，牲畜被冲走。31人遇难。
