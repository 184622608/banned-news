### 乌克兰事件 普京G20压力罩顶  向川普〝简短〞解释
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】俄罗斯扣留
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰船艇.html" target="_blank">
  乌克兰船艇
 </a>
 事件后，俄国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 可谓是如乌云罩顶般前来出席阿根廷20国集团高峰会。对于自己引起的纷争，普京1日称，只要目前乌克兰当权者持续执政，战争会持续下去。并说，他已向美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 简短说明有关乌克兰危机事件。
 <br/>
 <br/>
 综合媒体报导，俄国25日于克里米亚(Crimea)旁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚速海.html" target="_blank">
  亚速海
 </a>
 (Azov Sea)，攻击与扣押3艘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰船艇.html" target="_blank">
  乌克兰船艇
 </a>
 及24名船员，乌克兰随后紧急下达30日的〝戒严〞，并禁止16到60岁的俄国男子进入乌国领土。两国关系再陷危机，恐爆发武装冲突。
 <br/>
 <br/>
 俄国海军扣留乌克兰船艇事件，引发欧洲各国愤怒。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 （Vladimir Putin），原本计划趁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会期间举行会谈，但峰会前夕，川普推文表示，因为不认同25日俄国在克里米亚外海，扣押乌克兰舰艇，要取消和普京的会谈。
 <br/>
 <br/>
 11月30日，普京在20国峰会晚宴上，两人短暂会面时，捉住机会向川普解释莫斯科的立场。他说，〝我们站着谈一会儿，我回答他有关黑海一事的询问。〞
 <br/>
 <br/>
 另外，俄罗斯总统府发言人佩斯科夫（Dmitry Peskov）则说，〝双普〞在G20场边进行简短寒暄，但没有机会多聊几句。
 <br/>
 <br/>
 12月1日，峰会尾声，普京再强势说，只要目前乌克兰当权者持续执政，〝战争会持续下去。〞还说：〝当前的乌克兰当权者无意解决冲突，尤其是透过和平手段。〞
 <br/>
 <br/>
 不过，乌克兰总统波洛申科（Petro Poroshenko）则表示，事件发生后，他就指示下属安排与普京通电话，要说〝我们已经准备好协商，以缓和紧张情势升高…但可惜的是，普京至今并未回复我直接通话的要求〞。
 <br/>
 <br/>
 亲俄分离主义分子与乌国政府军4年前在乌东爆发冲突以来，估计夺走超过1万条性命，其中1/3为平民。
 <br/>
 <br/>
 这场战争考验俄国与西方强权的关系，西方各国归咎整起冲突由普京而起，主因俄军2014年兼并克里米亚半岛，替亲俄分离主义者夺取乌东各城市奠定基础。
 <br/>
 <br/>
 (责任编辑：卢勇信)
</div>
