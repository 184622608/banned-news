### 彭斯演讲痛批中共 美重整对华战略关系
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】美国副总统彭斯日前针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 的讲话，全面剖析了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在各领域对美国构成的威胁与挑战。再来聼聼美国智库学者的评论。
 <br/>
 <br/>
 美国副总统 Mike Pence：〝这位总统不会退缩；美国人民不会动摇。〞
 <br/>
 <br/>
 副总统彭斯发表演讲，正值美中贸易关系紧张之际。他详细列举了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 各种不公平贸易手段，包括动用安全机构，策划对美国技术的全面窃取。专家认为，演讲显示出美方将在贸易战中坚持到底的决心。
 <br/>
 <br/>
 美国传统基金会经济学家 Steve Moore：〝川普（特朗普）政府在将要到来的贸易谈判中，对中共会很强硬。他们想达成协议，但要按照美国的条件，改变中共一些贸易滥用行为，保护知识产权。副总统相当于列出了条款，这就是我们要求的。〞
 <br/>
 <br/>
 美国副总统 Mike Pence：〝北京动用整个政府机构，通过政治、经济和军事及宣传手段推动其影响力，并在美国牟利。〞
 <br/>
 <br/>
 彭斯在演讲中，多方面揭露中共对美国构成的威胁，包括对美国社会各领域的渗透与分化。他特别指出，中共还企图影响中期选举、颠覆川普政府。
 <br/>
 <br/>
 美国传统基金会外交政策专家 Michael Gonzalez：〝中共的所作所为令人不可思议，非常有侵略性。川普政府对此问题非常严肃。我想现在已不必留情了，看到的是坦白得多的表达。这是整个战略关系的重启。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯演讲.html" target="_blank">
  彭斯演讲
 </a>
 中18次提到〝中共〞或〝共产党〞，显然有意将〝中国〞与〝中共〞区分开来。这也意味着美国把中共视为真正威胁和反击目标。
 <br/>
 <br/>
 美国传统基金会经济学家 Steve Moore：〝共产主义在中国是彻底的失败。我们希望看到中国保证向自由的方向发展，自由市场，而不是国家控制和政治打压。坦白说，过去四五年，我们认为中国是向错误方向发展。这就是副总统要阐明的。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
