### 危机浮现 〝一带一路〞半月内四处遇挫
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】近期，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 计划在多个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家.html" target="_blank">
  国家
 </a>
 接连遇挫。专家指出，一带一路，输出中共的政治经济模式，其弊端正在世界范围显现。中共一带一路计划，接连爆出丑闻。
 <br/>
 <br/>
 11月26日，在全球积极推销〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞的香港官员何志平，因涉及跨国贿赂案，在纽约受审。诸多法庭文件，或将曝光
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 如何透过海外贿赂，吸引他国加入一带一路。
 <br/>
 <br/>
 11月25号，〝一带一路〞在非洲肯尼亚的标准铁路项目，曝出贪腐贿赂丑闻。项目承建方〝中国路桥公司〞的3名中国员工，入侵订票系统，转走大量资金，并试图贿赂调查人员。
 <br/>
 <br/>
 11月23日，〝一带一路〞的重要枢纽巴基斯坦，一个分离组织，因不满中共剥夺巴基斯坦的天然资源，武装袭击了当地中领馆。
 <br/>
 <br/>
 11月17日，马尔代夫新总统就任，他批评前政府向中共举债疯狂投资基础建设，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家.html" target="_blank">
  国家
 </a>
 债台高筑。该国财长也批评，〝一带一路〞众多项目〝价格虚高〞〝所有交易秘密进行〞。
 <br/>
 <br/>
 唐靖远：从经济角度上讲，一带一路是中共在国内那套〝高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_负债.html" target="_blank">
  负债
 </a>
 推动基础建设〞的翻版。中国的经济体量大，债务问题是累积了一二十年达到爆棚程度，而不少一带一路沿线国家经济体量小，所以短短几年就迅速掉入债务陷阱。
 <br/>
 <br/>
 李天笑：（一带一路）输出的不仅是过剩产品，中共政治经济模式中的弊病，必然的，也输送到其他国家。
 <br/>
 <br/>
 〝全球发展中心〞报告指出，参与〝一带一路〞项目的68个国家中，已有23个债台高筑，还有8个有债台高筑的风险。
 <br/>
 <br/>
 唐靖远：然后它们就得选择拿矿产资源、战略企业、甚至领土使用权来跟中共抵债，必然引起这些国家国内反弹，也让自由社会警惕中共扩张的野心。
 <br/>
 <br/>
 数据显示，中国2013年以来在〝一带一路〞沿线国家，宣布投资的1674个基础设施中，已有234个遭遇阻碍。
 <br/>
 <br/>
 唐靖远：一带一路一开始就是一个大赌局。它那种高负债、低效率的基建，在民主国家是很难通过的，而在威权和独裁国家容易过关。它的路线图上很多都是这种国家。这些国家的政权更迭，或者安全动荡，都可能让一带一路烂尾。
 <br/>
 <br/>
 另外，一带一路沿线较为自由开放的国家，也有民众质疑中共的开发方式具有殖民色彩，而发起反对活动。
 <br/>
 <br/>
 李天笑：前景确实是不看好的。因为这些国家都看到会对他们的经济带来不可承受之重。而且现在美国和西方国家都提出可以用西方公平交易、私营企业（主导）的方式进行援助以后，更有信誉，对一带一路冲击很大。
 <br/>
 <br/>
 彭斯：今天，让我告诉（印太）地区以及全世界所有国家：不要接受有可能损害你们的主权的外国债务。要保护你们的利益。要维护你们的独立。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
