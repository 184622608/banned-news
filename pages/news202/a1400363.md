### 日航超卖机位 罕见取消航班 乘客被迫留宿机场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本航空.html" target="_blank">
  日本航空
 </a>
 公司（JAL）昨(21日)一架从羽田机场飞往福冈机场的末班客机，因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超卖机位.html" target="_blank">
  超卖机位
 </a>
 ，在与乘客协调时花了不少时间，导致班机被取消，约400名乘客受影响，找不到住宿处的部分旅客被迫留宿羽田机场。
 <br/>
 <br/>
 日本放送协会（NHK）报导，日航这班客机原订在当地时间昨天晚间7时45分，从羽田机场飞往福冈机场，是当天飞往福冈的最后一个航班。。
 <br/>
 <br/>
 但因原本座位数375个，日航接受约400人订位，也就是所谓的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超卖机位.html" target="_blank">
  超卖机位
 </a>
 （overbooking，接受比座位数还多的订位）。日航紧急协调部分旅客转乘其他航班，但因花了不少时间，导致航班无法在福冈机场关场的晚间10时前抵达，因此被迫取消。
 <br/>
 <br/>
 约400位乘客无法如期前往福冈，部分旅客因找不到住宿处，只好在临时开放的羽田机场航厦内过夜。
 <br/>
 <br/>
 日航表示，将安排部分旅客搭乘上午飞往福冈的临时航班，并为造成旅客这么大的不便深深致歉。
 <br/>
 <br/>
 航空界相关人士说，因花时间协调旅客，导致航班被迫取消，可说相当罕见。
 <br/>
 <br/>
