### 俄唯一航母大修 出厂时船坞沉没 4伤1失踪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】俄罗斯航空母舰库兹涅佐夫号从上月起在北部的第82船舶修理厂大修。本周二，该航母完成维修准备出坞时，浮动船坞突发事故沉没，造成至少4人受伤、1人失踪，尚未来得及出坞的航母又被倒塌的起重机砸坏，不得不移送至另一船厂继续维修。
 <br/>
 <br/>
 俄罗斯船坞沉没事故发生于当地时间10月30日凌晨3点30分左右。当时刚刚经过大修的库兹涅佐夫号航母正准备出坞，电力供应突然中断，PD-50浮船坞的储水池随即被海水填满，整个浮船坞沉没到水下。与此同时，浮船坞两侧的两台塔式起重机也轰然倒塌，一台跌入海中，一台砸落在〝库兹涅佐夫〞号航母的甲板上。
 <br/>
 <br/>
 事后，该船舶修理厂的对外联络部负责人叶夫根尼．格拉德舍夫告诉媒体，浮船坞完全沉没后，库兹涅佐夫号航母被拖船拖离现场，送到附近的第35船舶修理厂接受新的维修。
 <br/>
 <br/>
 俄罗斯政府则发布公告称，这次的事故不影响库兹涅佐夫号的维修使用，倒塌的起重机并未对航母的关键设备造成损坏，现航母已被拖到另一海军船厂继续维修。
 <br/>
 <br/>
 这次事故至少导致三人受伤，一人失踪，包括潜水员在内的救援人员仍在对失踪人员进行搜救。
 <br/>
 <br/>
 据《自由亚洲电台》报导， 发生事故沉没的浮船坞是1980年从瑞典订购的，是世界上最大的浮船坞之一，主要用于修理俄罗斯的〝库兹涅佐夫〞号航母和〝彼得大帝〞号重型核动力巡洋舰等超大型舰艇。
 <br/>
 <br/>
 报导称，由于资金紧张，俄罗斯无力设计建造新的航母，这艘老旧的〝库兹涅佐夫〞号航母是俄罗斯唯一的航母。因为资金、舰载机、防卫等诸多问题，这艘航母很少出海执行任务，长期停靠在军港。
 <br/>
 <br/>
 近期，〝库兹涅佐夫〞号航母被普京派往叙利亚出征，结果它一路上黑烟滚滚，期间还发生了舰载机坠毁的事故，扫了普京的脸面，回航后被拖往船厂大修。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：东野）
