### 狂风暴雨袭欧 意大利老城几乎被淹 至少11死
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】狂风
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴雨.html" target="_blank">
  暴雨
 </a>
 影响
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧洲.html" target="_blank">
  欧洲
 </a>
 部分地区，意大利、法国、西班牙、奥地利等国都遭遇恶劣天气侵袭。意大利
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威尼斯.html" target="_blank">
  威尼斯
 </a>
 老城70%被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪水.html" target="_blank">
  洪水
 </a>
 淹没。已知至少有11人死亡。热那亚一座大坝因水压过高决堤，现场人员描述宛如海啸来袭。
 <br/>
 <br/>
 意大利的大部分地区连续几天经历恶劣天气，使这个国家从北到南陷入瘫痪。许多地区的学校和幼儿园仍然关闭，包括威尼托、利古里亚、罗马和托斯卡纳的部分地区。
 <br/>
 <br/>
 意大利〝水都〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威尼斯.html" target="_blank">
  威尼斯
 </a>
 （Venice）遭受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪水.html" target="_blank">
  洪水
 </a>
 淹没，周一（29日）洪水已经涨至156厘米，是10年来最高水位，70%的老城区都淹在水下。周二，情况有所改善，但当局预计水位仍为110厘米。猛烈风暴刮起每小时180公里强风造成6人丧生，加上周一证实的5人遇难，总死亡人数来到11人。
 <br/>
 <br/>
 威尼斯市长布鲁尼亚罗（Luigi Brugnaro）在圣马克广场视察，背后就是汹涌的海水。广场附近的居民已被疏散。布鲁尼亚罗呼吁所有居民留在家中。

 <br/>
 南蒂罗尔的布伦纳高速公路部分路段遭到崩落的山泥堵塞，仅有一条车道可通行。
 <br/>
 <br/>
 利古里亚西北部地区所有港口全部关闭，周一晚上到达的船隻必须在海外等待风暴结束。里维埃拉海岸掀起8～10米高的大浪。利古里亚有22,000户家庭断电。热那亚的Cristoforo Colombo国际机场也于周二早上关闭，取消所有航班。
 <br/>
 <br/>
 在热那亚（Genoa）东南约30公里的沿海城市拉帕洛（Rapallo），一座大坝因水压过高溃堤，风暴卷起游艇摔到岸上，有数十艘游艇被摧毁。港口工作人员惊愕不已，直呼当时情景〝像海啸来袭〞。
 <br/>
 <br/>
 在热那亚以西的萨沃纳港，汽车码头被淹。两个仓库因短路发生火灾。数百辆原定出口到中东的奢侈品牌玛莎拉蒂汽车已经被摧毁。
 <br/>
 <br/>
 在特伦托省的迪马罗，洪水冲破了河岸，200人被带到安全地带。
 <br/>
 <br/>
 在南部，船隻停止通行，在那不勒斯和伊斯基亚岛之间的交通停止。在撒丁岛的阿尔盖罗，冰雹有乒乓球一样大。周二的天气应该有所改善，但是气象服务警告说西北地区有新风暴来袭。
 <br/>
 <br/>
 而在意大利与瑞士交界处，斯蒂维奥山口（Stelvio Pass）降下大雪，造成约170名游客及饭店工作人员受困。
 <br/>
 <br/>
 在著名的意大利科尔蒂纳丹佩佐滑雪胜地，数十人不得不疏散。在利古里亚同样发出最高警报。民防人员预报说风暴潮可能达到7米。在旅游区五渔村（Cinque Terre）的蒙特罗索（Monterosso）镇，居民不得不离开低层公寓。
 <br/>
 <br/>
 在罗马，树木像火柴一样随风飞舞，天线从屋顶飞出，汽车被压扁，街道堵塞。罗马斗兽场、帕拉蒂尼山和帝国议事广场全部关闭。在罗马南部的弗罗西诺内省，周一有两人丧生，倒下的树木击中了他们的汽车。

 <h4>
  法国、奥地利等国也遭遇恶劣天气
 </h4>
 <br/>
 法国中央高原地区（Massif Central）发生暴风雪，山路积雪过多使交通瘫痪，造成1000多名司机受困。上卢瓦尔省和卢瓦尔省，800多辆汽车堵在路上动弹不得。司机不得不步行离开。东部城市里昂（Lyon）有火车因积雪而停止运行，400名旅客夜宿车厢。东部和中部地区约有19万5000户人家无电可用。

 由于冬季提早到来，当局此前已发布第二高警报。周二早上法国全国范围内发布冰雪等恶劣天气警告。在法国南部，当局警告有大雨和洪水。
 <br/>
 <br/>
 在奥地利，周二在克林西亚和东蒂罗尔部分地区因洪水及进一步降雨和风暴威胁，学校关闭。
 <br/>
 <br/>
 克罗地亚也受到恶劣天气影响，境内亚得里亚海沿岸受到强风豪雨袭击，街道瞬成河流，居民电力遭断，海陆交通均受阻。北部港口城市里耶卡周围的高速公路暂时关闭。由于强风暴，杜布罗夫尼克和里耶卡之间的渡轮也被取消。周一上午整个克罗地亚沿海地区发布了最高级别警戒。
 <br/>
 <br/>
 而在西班牙，周末一场大雪导致北部的阿斯图里亚斯（Asturias）若干区域与外界失联，100多名军人正协助电路维修人员前往灾区。
