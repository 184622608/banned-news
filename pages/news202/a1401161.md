### 日本计划扩充军备 军机商看好合作前景
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】周三，东京举办
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_航空航天展.html" target="_blank">
  航空航天展
 </a>
 ，多家国际知名军工企业参展。日前有消息传出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 正在考虑增购新一代F-35战机，让许多企业看好与日本的合作前景。
 <br/>
 <br/>
 2018年度东京国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_航空航天展.html" target="_blank">
  航空航天展
 </a>
 览会，周三到周五在东京国际会展中心举行。
 <br/>
 <br/>
 多家国际知名军工企业，包括军用飞机制造商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洛克希德・马丁.html" target="_blank">
  洛克希德・马丁
 </a>
 、航空航天设备公司英国宇航系统等参加了这次展出。
 <br/>
 <br/>
 其中，洛克希德・马丁公司研发的新一代F-35系列战机，分为A、B、C三个版本。其中F-35A为正常起飞，而F-35B战机可以低速起飞以及垂直起落。
 <br/>
 <br/>
 目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 已经订购了42架F-35A战机。《日经新闻》报导，日本政府正在考虑追加订购数十架F-35A和F-35B战机。每架F-35战机的价格大约在8800万美元左右。
 <br/>
 <br/>
 英国宇航系统参展代表也看好未来与日本合作的前景。
 <br/>
 <br/>
 英国BAE System军机业务部主任 Andy Latham：〝日本正在寻求更多的合作伙伴。我认为英国也是潜在的合作者之一，新型军机研发项目也是互惠合作的一个机会。〞
 <br/>
 <br/>
 日本将在12月发布新的国防目标以及军事购买计划。在中共对外扩张的压力下，外界普遍预计，日本会进一步增加国防支出。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
