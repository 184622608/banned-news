### 北约挪威大军演 俄TU-142战斗机低飞搅局
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】俄罗斯一架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_图波列夫-142.html" target="_blank">
  图波列夫-142
 </a>
 战斗机，近日闯入北大西洋公约组织（NATO）最大规模军事演习搅局。
 <br/>
 <br/>
 这架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_图波列夫-142.html" target="_blank">
  图波列夫-142
 </a>
 （Tupolev
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_TU-142.html" target="_blank">
  TU-142
 </a>
 ）发出嗡嗡声响低空飞行，从停靠在挪威外海的美军
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_惠特尼山号指挥舰.html" target="_blank">
  惠特尼山号指挥舰
 </a>
 （USS Mount Whitney）准备在甲板上拍团体照的舰艇官兵头上呼啸而过。
 <br/>
 <br/>
 一名美军士兵说：〝这是架长程海上巡逻侦察机。〞尽管他看过许多关于图波列夫-142的照片，但这可以说是他第一次活生生看到这架战机在天空翱翔。
 <br/>
 <br/>
 这架TU-142苏联时期战斗机的出现，或代表俄罗斯对于北大西洋公约组织（NATO‧
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北约.html" target="_blank">
  北约
 </a>
 ）的〝2018
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三叉戟联合军演.html" target="_blank">
  三叉戟联合军演
 </a>
 〞（Trident Juncture 18）感到不满，首次反制行动。
 <br/>
 <br/>
 俄国认为这场长达2周的演习是反俄罗斯的武力展示，警告俄方不会默不作声，并于上周宣布将在此区域试射飞弹的计划。
 <br/>
 <br/>
 据挪威大部分民用机场的营运公司Avinor，俄国发出飞航公告（NOTAM）表示，11月1日到3日会在挪威海试射飞弹。
 <br/>
 <br/>
 北约秘书长史托腾柏格（Jens Stoltenberg）10月31日表示，任何飞弹试射〝将不会改变我们演习的计画〞。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-4/p9111052a70333817.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-4/p9111052a70333817-ss.jpg"/>
   <br/>
   <font size="-1">
    2018年11月2，在挪威外海，俄罗斯一架图波列夫-142战斗机低空飞行。(JONATHAN NACKSTRAND/AFP/Getty Images)
   </font>
  </a>
 </center>
 <br/>
 北约组织于10月25日至11月7日期间，在挪威举行冷战结束以来最大规模军事演习〝2018三叉戟联合军演〞。军演意在训练北约成员国迅速动员，协防遭攻击的盟友。俄罗斯及白俄罗斯各有两名军事观察员，受邀观察这次军演。
 <br/>
 <br/>
 成员国将出动约5万名兵力、1万辆军车、250架军机以及60艘舰艇，包括一艘美国航空母舰。
 <br/>
 <br/>
 北约这次的演习地点，距离挪威与俄罗斯位于北极地区198公里长边界有一段距离，但莫斯科对北约此举表示愤怒，令其宛如芒刺在背，扬言报复。
 <br/>
 <br/>
 不过，挪威中将雅可布森（Rune Jakobsen）表示：〝核心演习区距俄罗斯边界1000公里，空中行动也将与边界保持最长500公里距离。〞，俄罗斯人应该没有任何惧怕的理由。
