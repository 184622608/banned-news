### 赴G20峰会 默克尔专机遇故障迫降
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 周五开幕，各国首脑纷纷在周四陆续抵达阿根廷首都布宜诺斯艾利斯，然而德国总理默克尔的专机却出了点状况，请看报导。
 <br/>
 <br/>
 默克尔前往阿根廷参加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 的专机，在从柏林起飞大约1个小时后，由于出现技术故障，当地时间晚上9点被迫紧急降落在德国西部城市科隆。德国总理府发言人表示，默克尔及其随行人员将换乘另外一架飞机继续前往阿根廷，不过要到周五的晚上才能抵达，将无法出席周五早上的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开幕式.html" target="_blank">
  开幕式
 </a>
 。
 <br/>
 <br/>
 周四晚间，参加G20峰会的美国总统川普，法国总统马克龙，英国首相特里莎・梅，中国国家主席习近平等其他各国首脑纷纷抵达阿根廷首都布宜诺斯艾利斯。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
