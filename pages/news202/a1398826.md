### 提前2年解散国会选 斯里兰卡面临政治瘫痪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯里兰卡.html" target="_blank">
  斯里兰卡
 </a>
 总统席瑞塞纳眼见指定的总理人选拉贾帕克萨在国会难取得多数支持后，9日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_解散国会.html" target="_blank">
  解散国会
 </a>
 并宣布将大选提前至1月，未来两个月将面临政治瘫痪而陷入危机，美国及欧洲联盟（EU）已表达关切，并要求斯国尊重民主。
 <br/>
 <br/>
 席瑞塞纳（Maithripala Sirisena）日前撤换前总理威克瑞米辛赫（Ranil Wickremesinghe），但威克瑞米辛赫拒绝下台，这两周来，他与曾任总理的拉贾帕克萨（Mahinda Rajapakse）一再进行权斗。
 <br/>
 <br/>
 席瑞塞纳眼见贾帕克萨取得国会多数支持的胜算机率偏低，便签署法令解散225席的国会，并计划在明年1月5日举行国会选举，比表定时间提前将近两年，遭反对阵营谴责此举已然违法。
 <br/>
 <br/>
 威克瑞米辛赫所属的联合国家党（United National Party）发推文说：〝总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_解散国会.html" target="_blank">
  解散国会
 </a>
 为非法且违宪。我们将持续抗争，确保本国民主地位至高无上。〞
 <br/>
 <br/>
 联合国家党表示：〝他（席瑞塞纳）剥夺了人民的权利及我们所享的民主。〞
 <br/>
 <br/>
 席瑞塞纳此举也引发国际社会关注，美国国务院透过推特声明：〝美国对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯里兰卡.html" target="_blank">
  斯里兰卡
 </a>
 解散国会，进一步加深政治危机的消息深表关切。〞
 <br/>
 <br/>
 〝身为斯里兰卡坚定的合作伙伴，我们认为必须尊重民主制度及程序，才得以确保稳定及繁荣。〞
 <br/>
 <br/>
 联合国及欧盟也加强施压席瑞塞纳，要求让国会表决决定应由拉贾帕克萨或威克瑞米辛赫组阁。
 <br/>
 <br/>
 欧盟表示，当前危机已经伤害斯里兰卡的国际声誉，并与挪威及瑞士发布联合声明，呼吁斯里兰卡国会开议，立即进行投票。
 <br/>
 <br/>
 声明说：〝任何进一步的延宕，都可能损害斯里兰卡的国际声誉并让投资者裹足不前。〞
 <br/>
