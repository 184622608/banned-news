### 〝手推车侠〞大战〝孤狼〞恐攻  澳洲街友获钜额捐款
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】澳大利亚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨尔本.html" target="_blank">
  墨尔本
 </a>
 9日下午发生〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孤狼.html" target="_blank">
  孤狼
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恐怖攻击.html" target="_blank">
  恐怖攻击
 </a>
 事件，造成2死2伤。警方在擒伏歹徒过程中，一名见义勇为的街友当时试图以购物推车拦阻歹徒，被封为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手推车侠.html" target="_blank">
  手推车侠
 </a>
 〞（trolleyman），其英勇事迹传出后，外界今天涌入大量捐款。
 <br/>
 <br/>
 袭击事件发生在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨尔本.html" target="_blank">
  墨尔本
 </a>
 市中心的商业区的布尔克街（Bourke Street），歹徒希尔阿里（Hassan Khalif Shire Ali）开四轮传动车冲撞商家起火燃烧，并用刀刺死1人，刺伤2人。
 <br/>
 <br/>
 警方赶到现场擒伏歹徒时，歹徒不断拿刀攻击，期间46岁街友罗杰斯（Michael Rogers）不断将购物推车推向歹徒。最后，歹徒遭警方开枪击中，送医不治。
 <br/>
 <br/>
 <b>
  相关视频：
 </b>
 <center>
 </center>
 <br/>
 这段过程被网友拍下上传网路后，不少网友称罗杰斯是〝英雄〞，群众还在募资平台GoFundMe上协助他的募款活动，今(12日)已突破10万8000澳元（约8万美元），是原本目标4万5000澳元（约3.3万美元）的两倍多，而且还在持续增加中。
 <br/>
 <br/>
 有网友留言说：〝我是因为他的英勇而捐款。澳洲感谢你！〞
 <br/>
 <br/>
 罗杰斯事后接受澳洲媒体访问时说，他当时的行为是〝临时起意〞，〝我当时吓坏了，我也担心我自己。〞但〝我就是想帮忙，做我人生中第一次对的事。〞
 <br/>
 <br/>
 他说，〝我看到旁边有手推车，所以就赶紧推过来撞向他。我重复那个动作…若干次，但就是没有撞倒他。〞
 <br/>
 <br/>
 罗杰斯坚称他〝不是英雄，但我觉得自己可能救了一些人的性命，也许啦〞。
 <br/>
 <br/>
 据了解，30岁的希尔阿里是在索马利亚出生的澳洲人，当局至少3年前就知道他，警方此前担心希尔阿里试图赴叙利亚加入恐怖组织伊斯兰国（IS），于2015年撤销他的澳洲护照。
 <br/>
 <br/>
 警方表示，希尔阿里犯下这起案件，是受到伊斯兰国（IS）启发，但与这个圣战组织没有直接关系，似乎是独自犯案。
 <br/>
