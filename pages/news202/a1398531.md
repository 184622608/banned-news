### 央视女记者打人遭提堂 有新视频证据展示
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视女记者.html" target="_blank">
  央视女记者
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孔琳琳.html" target="_blank">
  孔琳琳
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国保守党年会.html" target="_blank">
  英国保守党年会
 </a>
 上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_掌掴.html" target="_blank">
  掌掴
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港义工.html" target="_blank">
  香港义工
 </a>
 ，被英国当局控告后，于当地时间11月8日提堂，但孔琳琳未有出席聆讯。控辩双方均称，由于刚收到一条新的闭路电视片段，需要更多时间研究。
 <br/>
 <br/>
 英媒11月8日报导，英格兰西米德兰兹郡（West Midlands）警方证实，来自伦敦国王十字区、现年48岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孔琳琳.html" target="_blank">
  孔琳琳
 </a>
 ，被控普通袭击罪，案件于11月8日在伯明翰裁判法院提堂，但被告孔琳琳未有现身。
 <br/>
 <br/>
 报导说，控辩双方均称，由于刚收到一条新的闭路电视片段，需要更多时间研究，因此申请押后，法官同意押后申请，但提醒辩方律师，必须与孔琳琳沟通，下次开庭她应该出席聆讯。
 <br/>
 <br/>
 法官最终决定将案件延至11月21日再审。
 <br/>
 <br/>
 孔琳琳打人事件发生于9月30日，当日，英国保守党在人权组织〝Hong Kong Watch〞的安排下，筹办一个关于〝香港自由、法治和自治受到侵蚀〞的讲座，会议由英国国会议员布鲁斯（Fiona Bruce）女士主持。
 <br/>
 <br/>
 会议接近尾声时，英国保守党人权委员会副主席罗杰斯（Benedict Rogers）致辞说：〝我是亲中国的，不是反中国的。我希望中国和中国人民成功。我批评中国当前的政府以及它对待人民的方式……。〞
 <br/>
 <br/>
 罗杰斯的话音刚落，持大会记者证的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视女记者.html" target="_blank">
  央视女记者
 </a>
 孔琳琳从听众席上站起来，对罗杰斯尖叫：〝你撒谎！你反华！你想分裂中国！你连中国人都不是。其他人都是汉奸！〞
 <br/>
 <br/>
 主持人布鲁斯女士要求孔琳琳离开会场。24岁义工Enoch Lieu试图让孔琳琳离开会场时，被她打了两个耳光。
 <br/>
 <br/>
 旁边一位英国与会者制止其继续打人时，孔琳琳大喊：〝你没有权利！英国没有民主！我是这的记者。我有抗议的权利！〞孔琳琳还反咬一口说，〝他（Enoch Lieu）欺负我！〞
 <br/>
 <br/>
 现场录像显示，当时一名华人工作人员在劝离过程中，手接触到了孔琳琳的风衣，她喊道：〝放开我〞，并打了两下工作人员的手臂。随后，孔琳琳离开会场后，被警方逮捕。
 <br/>
 <br/>
 10月1日，英国警方以涉嫌普通袭击罪逮捕了孔琳琳。但在中共驻伦敦大使馆介入后被释放。
 <br/>
 <br/>
 中共驻英国使馆发言人还反咬一口，说央视记者受人身侵犯，并且呼吁英国保守党不要再〝干涉内政〞，还要求活动的主办者应向孔琳琳道歉。
 <br/>
 <br/>
 伯明翰警方在10月25日证实，孔琳琳被正式起诉普通袭击罪。
 <br/>
 <br/>
 旅居英国的作家张朴对自由亚洲表示，英国警方的检控，令中共驻英大使馆和央视丢脸。孔琳琳式的专制通行规则，在西方的法治面前碰壁，孔琳琳也将因此付出代价。
 <br/>
 <br/>
 张朴说，孔琳琳成了共产党的网红，中共派出来的记者都是带着任务的，打人在英国是犯法的事情，这事就看法庭怎么认证了，有可能拘留几天，也有可能把她驱逐出境。她也让整个中共政府、大使馆彻底丢脸了，这个文明的世界鄙视它们。
 <br/>
 <br/>
 在孔琳琳打人事件发生后，有网友起底孔琳琳，是中信集团原董事长孔丹的女儿，但未获证实。孔琳琳被正式指控后，中共驻英使馆和央视暂未有进一步反应。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
