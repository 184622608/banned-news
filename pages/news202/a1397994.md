### 港媒：美选举结果或使北京一大期待落空
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】11月底，美国总统川普（特朗普）和中国国家主席习近平将在阿根廷G20峰会上会晤。有港媒认为，川习会能否达成协议，要看美国中期选举结果。而北京当局的应对策略也会随着共和党的胜负发生变化。
 <br/>
 <br/>
 日前，川普和习近平的电话沟通，表面上显得气氛融洽，对外界释放出积极的讯号。尤其是川普特别表示，将会与北京达成一个很好的协议。
 <br/>
 <br/>
 不过，香港《星岛日报》则认为，虽然中美领导人在G20上的会晤结果，或受美国中期选举结果的影响，可实际上川普和中共打贸易战在两党拥有着很高共识。文章以为这次川习会能否达成协议，要看美国中期选举结果。
 <br/>
 <br/>
 此前有分析认为，北京期待着美国中期选举的结果，如果川普的共和党落败，他的政策可能发生变化。
 <br/>
 <br/>
 有观点认为，北京这一期待会有落空的危险，因为美国不论民主党还是共和党，目前在对华政策上是有极大的共识的。同时两党对于贸易战也有着相当高的共识。即使民主党掌握了众议院，也很难说会朝着对北京有利的方向发展。
 <br/>
 <br/>
 据海外媒体报导，在川普与习近平通话的当天，白宫经济顾问库德洛对外表示，川普和习近平通电话，代表美中关系正在〝融冰〞，〝川习会〞有望打破两国间的诸多僵局。如果双方能达成一个满意的协议，美方也可能会撤回部分关税。
 <br/>
 <br/>
 但库德洛同时也警告，如果北京方面未能就美国认为非常重要的一系列问题提出解决办法，美国政府的贸易战攻势将〝更猛烈〞。
 <br/>
 <br/>
 美国中期选举将于11月6日进行投票。今年，参议院将有35席议员改选，众议院435名议员全数改选，另外还有36个州进行州长改选。就联邦政府一层来看，选举结果将决定国会两院将由哪个政党控制，即共和党保住多数党优势，还是民主党会掀起〝蓝波〞。
 <br/>
 <br/>
 （记者欧阳静报导/责任编辑：祝馨睿）
