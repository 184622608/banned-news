### 人民币亚洲表现最差 中国恐被列入〝汇率操控国〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】美国财政部将在下周宣布半年一次的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率.html" target="_blank">
  汇率
 </a>
 报告〞。有消息传出，白宫方面密切关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 的贬值问题，正在考虑把中国正式列入〝汇率操控国〞。
 <br/>
 <br/>
 随着美中贸易紧张程度的升温，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 兑美元在过去6个月来贬值约9%，〝十一〞长假后的第一个交易日，在岸和离岸汇价双双失守6.93关口。人民币成为亚洲表现最差的货币之一。中共央行在10月7日宣布的年内第四次降准，更被业界人士认为会加大人民币贬值的压力。
 <br/>
 <br/>
 据〝彭博社〞引述美国财政部知情人士透露，美国总统川普对人民币不断贬值感到担忧，正对美国财长穆努钦（Steven Mnuchin）施压，要求其在下周公布的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率.html" target="_blank">
  汇率
 </a>
 报告中，将中国列为〝汇率操纵国〞。
 <br/>
 <br/>
 如若这一消息得到落实，将意味着美国将首次以白纸黑字的在官方文件中，给予中国〝汇率操纵国〞的标签。
 <br/>
 <br/>
 若被列为汇率操纵国，美国政府将先与这些国家或经济体协商、谈判；若未改善，可能拒绝让该经济体获取美国海外私人投资公司融资、禁止参与美国政府采购或贸易制裁、呼吁国际货币基金（IMF）加强监督等手段。
 <br/>
 <br/>
 有分析认为，汇率操纵国面对的最大的惩罚是惩罚性
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 。川普在9日也重申，如果中共对美国最近的关税措施进行报复，他将对额外价值2670亿美元的中国进口商品加征关税。
 <br/>
 <br/>
 每年4月及10月，美国财政部会向国会提交〝国际经济和汇率政策报告〞，内容包括审视其他主要经济的汇率政策有否对美国不公。
 <br/>
 <br/>
 在去年美国财政部汇率操控国名单上，中国并未上榜。今年4月中国仍列在观察名单，当时报告称，美国强烈关切北京在矫正双边贸易失衡方面缺乏进展，要求中共建立更公平互惠的场域。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：戴明）
