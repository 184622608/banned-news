### 杀201名农民 危地马拉前士兵被判5160年
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】中美洲国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_危地马拉.html" target="_blank">
  危地马拉
 </a>
 法院周三（21日）以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_屠杀罪.html" target="_blank">
  屠杀罪
 </a>
 判处一名前士兵5160年徒刑，他在该国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内战.html" target="_blank">
  内战
 </a>
 时屠杀了201名农民。另外还有几名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凯毕尔.html" target="_blank">
  凯毕尔
 </a>
 成员被定罪，每人被判处超过6000年徒刑。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_危地马拉.html" target="_blank">
  危地马拉
 </a>
 法院周三宣布，罗培兹（Santos Lopez）是171起杀人案的〝罪魁祸首〞，每起案件判处他30年徒刑，总计5130年。另外，他还因与一名幸存孩童命案有关，额外加刑30年。不过，这些刑期只是象征性质，因为危地马拉执法上限为50年。
 <br/>
 <br/>
 根据调查，罗培兹隶属一支巡逻队，这支巡逻队1982年12月在与墨西哥接壤的杜斯艾雷斯村（Dos Erres）进行了大屠杀，201人被杀。这些士兵当时试图夺回被游击队偷走的约20把步枪，导致19名士兵在稍早一场埋伏行动身亡。
 <br/>
 <br/>
 此事件是因为危地马拉当时的军事独裁者埃弗拉因・里奥斯・蒙特（Efraín Ríos Montt）要求政府军执行焦土政策而造成。他因种族灭绝罪被起诉并于去年4月去世。
 <br/>
 <br/>
 杜斯艾雷斯村的故事被写入2017年纪录片〝寻找奥斯卡〞（Finding Oscar），讲述寻找一名男孩的故事，这名男孩后来活了下来，由一名士兵养大。该片由美国名导史蒂芬史匹柏（Steven Spielberg）担任监制。
 <br/>
 <br/>
 根据联合国（UN）调查数据显示，危地马拉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内战.html" target="_blank">
  内战
 </a>
 期间，约20万人丧生或下落不明。内战于1996年结束。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
</div>
