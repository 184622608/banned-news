### 韩男从伊拉克返国 现类似MERS症状后死亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 仁川市当局周二（6日）表示，一名男子在出现〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中东呼吸症候群冠状病毒.html" target="_blank">
  中东呼吸症候群冠状病毒
 </a>
 感染症〞（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_MERS.html" target="_blank">
  MERS
 </a>
 -CoV）症状后死亡。该男子在伊拉克出差4个月后于5日返回韩国。
 <br/>
 <br/>
 《韩联社》英文网报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 仁川广域市发言人Park Sung-ho透过电话告诉记者，这名60岁男子在中东出差4个月后，于当地时间11月5日返韩。但他突然出现不适症状，6日早在首尔以西40公里处仁川一家医院去世。
 <br/>
 <br/>
 Park Sung-ho说，〝开始时他突然发高烧，之后就失去了知觉，这是一种与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_MERS.html" target="_blank">
  MERS
 </a>
 有关的典型症状。周二一早，他被送医治疗后很快死亡。〞
 <br/>
 <br/>
 所幸，这名男子后来对MERS检验结果呈现阴性。Park Sung-ho表示，男子还患有糖尿病和脑血管疾病，可能是因慢性疾病而身亡。
 <br/>
 <br/>
 但报导称，男子的死亡值得注意。2015年5月，MERS首次袭击韩国，曾导致187人感染，其中38人死亡，近1.7万人接受隔离，社会一度陷入恐慌混乱，旅游业遭受了重创。同时，在中国也出现过MERS，中国广东曾因一名韩国患者的来访而出现了中国首例MERS病例，后经隔离治疗一个月后该患者痊愈出院。
 <br/>
 <br/>
 今年9月8日，韩国在时隔3年之后首次确认一个感染MERS病毒的病例。一名61岁男子到中东国家科威特出差返国后，确诊罹患MERS，经过治疗，该男子恢复健康。与这名病人有过接触的至少20人被隔离，约440人接受监察。
 <br/>
 <br/>
 MERS病毒于2012年9月首次在沙特阿拉伯被发现，与2003年震惊世界的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非典型肺炎.html" target="_blank">
  非典型肺炎
 </a>
 〞（SARS）同属冠状病毒，临床表现颇相似，一度被称为〝类SARS〞。相比SARS病毒，MERS病毒的传染性较弱，但致死率高达46%。
 <br/>
 <br/>
 为更有效控制疫情，韩国政府宣布把疾病防控主管部门疾病管理本部升级为副部级单位。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
