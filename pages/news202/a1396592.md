### 罗马地铁电扶梯崩塌 乘客堆叠酿逾20人伤(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】意大利
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗马.html" target="_blank">
  罗马
 </a>
 市中心一个地铁站23日发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电扶梯崩塌.html" target="_blank">
  电扶梯崩塌
 </a>
 事件，导致20多人受伤，其中1人伤势较重，而伤者大多是俄罗斯足球迷。消防人员说，事发部分起因于嚣闹的足球迷。目前这个地铁站已关闭。
 <br/>
 <br/>
 事发于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗马.html" target="_blank">
  罗马
 </a>
 特米尼（Termini）火车站附近的地铁站，从YOUTUBE一段视频画面可见，共和地铁站（Repubblica）下行电扶梯上站着满满的约数十人，突然电扶梯急剧加速，许多人发出尖叫，之后一群人就堆叠在运行中的电扶梯底部。
 <br/>
 <br/>
 意大利有媒体引述目击者报导，足球迷看似喝醉了，在电扶梯上又跳又舞，之后电扶梯就崩了。但球迷们否认这类说法。
 <br/>
 <br/>
 罗马市长拉吉（Virginia Raggi）对媒体记者表示，〝在任何情形下，我们要实地了解状况，并提供伤者及其家人必要协助。〞
 <br/>
 <br/>
 相关单位已展开多项调查。
