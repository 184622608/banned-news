### 〝黄背心〞演变成暴乱 凯旋门古迹遭破坏
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】法国〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄背心.html" target="_blank">
  黄背心
 </a>
 〞运动进入第三周，示威运动进一步升级成了反政府的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴乱.html" target="_blank">
  暴乱
 </a>
 。周日，刚刚从阿根廷G20峰会返回的法国总统马克龙，视察了遭暴乱者洗劫破坏的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凯旋门.html" target="_blank">
  凯旋门
 </a>
 及附近地区，之后与内阁成员召开了紧急会议，商讨是否要宣布进入紧急状态。
 <br/>
 <br/>
 星期六（12月1日）在巴黎市中心，一群穿黑衣
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄背心.html" target="_blank">
  黄背心
 </a>
 的蒙面暴徒冲入市中心的繁华地带，焚烧数十辆汽车和街边建筑物，掠夺商店，砸破窗户，与警方发生冲突，要求马克龙下台。
 <br/>
 <br/>
 同一天，巴黎地标，法国最受尊敬的纪念碑——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凯旋门.html" target="_blank">
  凯旋门
 </a>
 的内部遭暴力洗劫，在凯旋门博物馆入口处展出的玛丽安雕像，是法国共和国的象征，被暴徒粉碎。
 <br/>
 <br/>
 而凯旋门外部遭到涂鸦破坏，写满〝马克龙下台〞，〝黄背心将胜利〞、〝让革命之火燃烧〞等口号。这场持续到深夜的暴力抗议活动震惊各界，有超过400人被捕，100多人受伤。
 <br/>
 <br/>
 这是法国首都1968年以来发生的最严重骚乱，成为法国总统马克龙上任18个月来，面临的最严峻挑战。
 <br/>
 <br/>
 周日，从阿根廷G20峰会回来后，马克龙首先赶到凯旋门，审查巴黎市中心被破坏的情况，并考虑宣布紧急状态和部署士兵等措施。
 <br/>
 <br/>
 随后，召集内阁紧急会议，商讨对策。
 <br/>
 <br/>
 〝黄背心〞抗议活动在11月17日爆发，并通过社交媒体迅速传播，抗议者在法国各地阻断道路，抗议政府于明年元旦起调涨燃油税。马克龙为履行〝巴黎气候协议〞，今年已将柴油税每公升调涨了6.2％。抗议者认为马克龙忽略中下阶层人民的生活状况，以环保之名忽视人民真正的需求。
 <br/>
 <br/>
 尽管民众不满情绪高涨，目前法国政府没有任何让步迹象。法国政府将暴动归咎与极右翼和极左翼的激进份子所为。
 <br/>
 <br/>
 这也让人联想到1871年出现的〝巴黎公社〞，同样是由社会底层暴民和流氓领导的武装叛乱，其领导成员包括社会主义者、共产主义者等各种激进分子。最终引发大规模杀戮和破坏。毁掉了巴黎及城中的大量文物、古迹和艺术品，对法国文化造成了重大破坏。
 <br/>
 <br/>
 新唐人记者李欣综合报导
