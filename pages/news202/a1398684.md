### 疑炸弹扔汽车 墨尔本男子再持刀斩人3伤(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】澳洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨尔本.html" target="_blank">
  墨尔本
 </a>
 市中心今（9日）下午发生街头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_持刀斩人.html" target="_blank">
  持刀斩人
 </a>
 案事件，一名男子放火焚烧汽车后，持刀袭击多人，之后男子被赶到的警察开枪制伏，情况危急。事件共造成至少3人受伤，其中1人伤势严重疑已身亡。
 <br/>
 <br/>
 事发在地点在伯克商业街（Bourke Street）。有目击者称当时听到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸.html" target="_blank">
  爆炸
 </a>
 声，又有人称看到有男子疑似把炸弹扔向汽车，另有民众担心会进一步爆炸而逃离该区域。
 <br/>
 <br/>
 从民众拍摄的疑似现场片段中可见，一辆汽车起火燃烧，一名身穿黑衣的男子，持刀袭击2名警员，有民众推购物车撞向男子，然后警员开枪制伏该名男子。
 <br/>
 <br/>
 警方随后封锁事后现场附近，并要求民众待在室内，商家也暂停营业。
 <br/>
 <br/>
 救护人员表示，目前现场有3名伤患，其中1人颈部受伤，伤势严重，另一人则是头部受伤，目前状况不明。另从照片中有发现，现场有看到一具尸体被白布覆盖。
 <br/>
 <br/>
 目前尚无法确认男子是否蓄意攻击人群，死伤状况尚不明朗。
