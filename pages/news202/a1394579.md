### 蓬佩奥铺路二次川金会 或聚焦终战宣言
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 出访东亚4国，为第二次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川金会.html" target="_blank">
  川金会
 </a>
 铺路。外界同时关注，如果川普与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 二次会晤，双方是否会签署终战宣言及其影响。对此，联合国军副总司令艾尔(Wayne D. Eyre)中将表示，终战宣言不同于和平协议，不应影响联合国军地位。
 <br/>
 <br/>
 周日(10月7日)，在平壤与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 举行会谈后，国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 说，金正恩准备允许国际核查人员进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 核试验及导弹试验设施，同时双方十分接近就第二次美朝峰会达成协议。有专家认为，这可能意味着川普政府同意与朝鲜签署终战宣言，以换取其在无核化上采取重大行动。
 <br/>
 <br/>
 在朝鲜看来，签署终战宣言，意味着其政权可以获得安全保障。但联合国军副总司令艾尔(Wayne D. Eyre)中将对此表示担忧。
 <br/>
 <br/>
 联合国军副总司令 Wayne Eyre中将：〝即使没有法律问题，在情绪上，人们也会开始质疑联合国司令部继续存在的必要性。很快他们也会质疑美国在朝鲜半岛的驻军。〞
 <br/>
 <br/>
 因韩战而建立的联合国军由17国组成，目前主要任务是维持半岛稳定，及监督落实联合国对朝制裁。艾尔中将表示，联合国军最终会被取消，但必须在适当时刻。
 <br/>
 <br/>
 联合国军副总司令 Wayne Eyre中将：〝终战宣言与和平协议完全不同，需要继续有一个维和机制，以正式降低冲突事件的风险。〞
 <br/>
 <br/>
 韩国总统文在寅在第三次韩朝峰会后表示，终战宣言不会影响联合国军地位。在朝鲜实现全面无核化前，继续保留联合国军。
 <br/>
 <br/>
 不久前退休的美国国务院亚太代助卿董云裳认为，川普政府目前抓住机会，不断试探朝鲜是否已有实质性改变，但必须小心朝鲜在各国间制造分歧。
 <br/>
 <br/>
 前美国国务院亚太代助卿 董云裳：〝朝鲜最擅长的就是分裂国际社会。他们实力很弱，但他们非常精于此道。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
