### 朝鲜士兵再闯军事分界线 投奔韩国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】在韩朝关系缓和之际，韩国媒体1日引述联合参谋本部报导，朝鲜再次发生士兵〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱北.html" target="_blank">
  脱北
 </a>
 〞事件。一名朝鲜士兵上午7时56分许(当地时间)，越过江原道东部战线
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事分界线.html" target="_blank">
  军事分界线
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_MDL.html" target="_blank">
  MDL
 </a>
 ）投奔韩国，过程中没有发生枪战等冲突。
 <br/>
 <br/>
 《韩联社》报导，韩军用监视装备观察到一名朝鲜士兵越过江原道东部战线
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事分界线.html" target="_blank">
  军事分界线
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_MDL.html" target="_blank">
  MDL
 </a>
 ）向南移动，并根据程序采取安全措施，确保其人身安全。
 <br/>
 <br/>
 韩国联合参谋本部还透露，事发区域前方的朝鲜军队没有特殊动向，而该名〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱北.html" target="_blank">
  脱北
 </a>
 〞士兵南下过程中，没有发生枪战等冲突。
 <br/>
 <br/>
 报导说，接下来，相关单位将对该朝鲜士兵南下过程进行详细调查。
 <br/>
 <br/>
 报导称，这是朝韩在拆除非武装地带（DMZ）最前方监视哨所（GP）后，出现的首例朝鲜军人叛逃事例。
 <br/>

 <br/>
 <br/>
 近期因韩朝关系趋于缓和，韩国国防部表示已拆除20个边防哨站，也逐步拆除双方边境的地雷，日后将合作在该处寻找韩战时阵亡的士兵遗体。
