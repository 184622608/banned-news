### 议会通过脱欧草案 首相：仍准备无协议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】周四，英国议会对内阁通过的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧.html" target="_blank">
  脱欧
 </a>
 协议，展开激烈争论。脱欧大臣周三辞职，加剧了首相特雷莎・梅执政的不稳定。特雷莎・梅也表示，英国将继续为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无协议脱欧.html" target="_blank">
  无协议脱欧
 </a>
 做准备。
 <br/>
 <br/>
 周四，特雷莎・梅对议会表示，只有三种选择：支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧.html" target="_blank">
  脱欧
 </a>
 协议；
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无协议脱欧.html" target="_blank">
  无协议脱欧
 </a>
 ；或放弃脱欧。她表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧协议.html" target="_blank">
  脱欧协议
 </a>
 将〝保持英国的完整性〞。
 <br/>
 <br/>
 英国首相特雷莎・梅：〝选择明确。我们可以选择无协议离开，（或者）冒险完全放弃脱欧；或者选择团结，支持可协商的最好协议。〞
 <br/>
 <br/>
 遭工党反驳后，特雷莎・梅松口表示，英国仍需为无协议脱欧作准备。
 <br/>
 <br/>
 英国首相特雷莎・梅：〝我们一直为无协议作准备，也将继续为无协议作准备。〞
 <br/>
 <br/>
 议会中，反对派工党领袖科尔宾指责特雷莎・梅，通过失败的脱欧协议，导致政府混乱。
 <br/>
 <br/>
 英国反对派工党领袖科尔宾：〝脱欧协议，和远离中心的政治宣言，代表破坏性的巨大失败。〞
 <br/>
 <br/>
 保守党高级议员里斯莫格 表示，将提交对特雷莎・梅不信任函。
 <br/>
 <br/>
 周四，英国脱欧大臣拉布和养老金大臣麦克维相继辞职。
 <br/>
 <br/>
 拉布表示，脱欧协议存在两大致命缺陷：一是威胁到英国的完整性，二是英国将丧失对退出机制的发言权。
 <br/>
 <br/>
 拉布辞职后，英镑应声下跌，创下17个月来最大跌幅，英国10年期国债收益率也大幅下跌。
 <br/>
 <br/>
 周四，欧洲议会主席安东尼奥・塔亚尼表示，英国的协议解决了脱欧三大难题：公民权、爱尔兰边境和金融协议。
 <br/>
 <br/>
 欧洲议会主席安东尼奥・塔亚尼：〝对欧洲议会来说，我们对（协议中）的三个关键问题表示满意。〞
 <br/>
 <br/>
 周四，德国总理默克尔表示，欢迎英国的脱欧协议草案，她表示最坏的情况是不能达成协议。
 <br/>
 <br/>
 德国总理默克尔：〝我很高兴经过长时间谈判，这不容易，终于提出了（脱欧）提案。〞
 <br/>
 <br/>
 欧盟理事会主席图斯克周四发推文表示，如果没有特别情况，欧盟将在11月25日，和英国正式确定脱欧协议。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
