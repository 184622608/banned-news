 周三晚，美国南卡州发生枪击案，1名警察殉职，6名警察受伤。经调查，嫌犯是越战退伍军人霍普金斯,现年74岁,他身有残疾，近年面临着多项指控。
 <br/>
 <br/>
 一个多月前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际空间站.html" target="_blank">
  国际空间站
 </a>
 附近的联盟号飞船上发现的小洞，至今成因仍是谜。周四,美国航天局表示, 11月将进行太空行走，调查原因。有专家猜测，小洞是太空垃圾撞击导致，或是制造飞船时的缺陷，还可能有人蓄意破坏，甚至有猜测是外星人所为。
 <br/>
 <br/>
 俄罗斯总统普京和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度.html" target="_blank">
  印度
 </a>
 总理莫迪的年度首脑会谈周四拉开帷幕。预计两国将签署价值超过50亿美元的导弹购买协议。此前,美国警告印度不要从俄罗斯购买S-400地对空导弹，这将招致美国制裁。美国是印度最大的武器供应商之一，过去十年中卖给印度价值150亿美元的武器。
 <br/>
 <br/>
 在以色列访问的德国总理默克尔周四表达了对伊朗核问题的立场。
 <br/>
 <br/>
 德国总理默克尔：〝我们非常确信并强烈赞同以色列的立场，即必须采取一切措施防止伊朗拥有核武器。〞
 <br/>
 <br/>
 她并提到，伊朗在叙利亚和黎巴嫩的军队是对以色列的威胁。
