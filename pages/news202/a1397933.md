### 搜寻狮航失事客机 1名潜水人员疑因减压症不幸过世
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】印尼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮航.html" target="_blank">
  狮航
 </a>
 （Lion Air）失事客机造成机上189人全数丧生，2日再传出一名参与搜寻的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潜水人员.html" target="_blank">
  潜水人员
 </a>
 疑因减压症不幸过世。
 <br/>
 <br/>
 综合媒体报导，印尼官员3日表示，参与搜寻罹难者遗体和飞机残骸任务，48岁的国家搜救总署志工
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潜水人员.html" target="_blank">
  潜水人员
 </a>
 安托（Syachrul Anto）昨天过世。
 <br/>
 <br/>
 安托曾在9月印尼帕卢（Palu）发生强震海啸灾情时，加入当地的搜救行动，近4年前曾参与亚洲航空（Air Asia）空难的搜救作业。
 <br/>
 <br/>
 这架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮航.html" target="_blank">
  狮航
 </a>
 客机10月29日从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雅加达.html" target="_blank">
  雅加达
 </a>
 飞往邦加槟港（Pangkal Pinang），但在起飞数分钟后坠入爪哇海（Java Sea）。
 <br/>
 <br/>
 初步资料显示，这架JT610客机在坠毁前的一个航程，曾经出现技术故障。目前已经寻获〝飞航资料记录器〞（Flight Data Reocorder），至于第2具黑盒子〝驾驶舱语音记录器〞（Cockpit Voice Recorder）目前还在搜寻中，一旦找到或许可以解答这架加入航线服务才不久的波音737 MAX 8失事原因。
 <br/>
 <br/>
