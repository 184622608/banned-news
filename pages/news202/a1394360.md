### 盘点三大假新闻 日媒警告：中共砸钱攻台
------------------------

<div class="wysiwyg">
 【新唐人2018年10月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假新闻.html" target="_blank">
  假新闻
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_攻击台湾.html" target="_blank">
  攻击台湾
 </a>
 ，现在，日本媒体也关注到这个事件。读卖新闻报导指出，中共成立对台小组，砸钱制造假新闻，报导也盘点了，过去的三次假新闻。
 <br/>
 <br/>
 斗大的标题写着，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假新闻.html" target="_blank">
  假新闻
 </a>
 撼动台湾。在日本关西机场淹水8千旅客被困事件满月之际，日媒各大报纷纷推出报导。
 <br/>
 <br/>
 读卖新闻报导写道，在台湾11月选举前，总统蔡英文已经开始拿出打击假新闻的对策。而取缔假新闻的开端，就是台湾驻大阪办事处处长自杀一事。在关西机场关闭期间，网路上流传，中国的大阪总领事馆安排15辆大巴士进入关西机场，优先救出中国旅客。
 <br/>
 <br/>
 读卖新闻指出，为了动摇蔡政府，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 中央宣传部与国家网路情报办公室，组成〝对台工作小组〞，编造让争取网路新闻点阅数的台湾媒体上钩的内容，方法有以身边生活问题来散播假新闻。以及如果能成功让台湾媒体转载的话，则有奖金。
 <br/>
 <br/>
 中华民国总统 蔡英文：〝我们社会各层面中，充斥越来越多精心捏造的假新闻，意图诋毁、破坏我们的民主生活方式。这些挑战不仅冲击台湾，更影响、破坏所有亚太地区理念相近国家。〞
 <br/>
 <br/>
 除了关西机场事件，报导也整理出近年来造成话题的台湾假新闻。包括今年6月，网路疯传一张一卡车香蕉被倾倒在地上的照片，媒体报导后造成香蕉价格受损，后来才证实这是2007年的照片。
 <br/>
 <br/>
 还有一个，是8月底蔡英文总统搭装甲车查勘水灾时，下令士兵可随时射击，被指把灾民当敌人。结果真相是参与救灾的士兵或车辆都没武装，因为灾区水患还未退，所以蔡英文才搭装甲车勘灾。
 <br/>
 <br/>
 新唐人亚太电视综合报导
</div>
