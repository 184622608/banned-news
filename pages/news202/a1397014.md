### 行李搬运工罢工 比利时最大机场200航班取消
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比利时.html" target="_blank">
  比利时
 </a>
 官员周五（26日）表示，因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布鲁塞尔.html" target="_blank">
  布鲁塞尔
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_机场.html" target="_blank">
  机场
 </a>
 行李
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_搬运工.html" target="_blank">
  搬运工
 </a>
 在假期
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罢工.html" target="_blank">
  罢工
 </a>
 ，约200多个航班被迫取消，影响数千名旅客出行。工会代表和飞行伙伴周五谈判未达成任何协议，预计周六(27日）上午10点继续。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布鲁塞尔.html" target="_blank">
  布鲁塞尔
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_机场.html" target="_blank">
  机场
 </a>
 线上讯息显示；〝因行李运输地勤公司飞行伙伴（Aviapartner）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_搬运工.html" target="_blank">
  搬运工
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罢工.html" target="_blank">
  罢工
 </a>
 ，飞行伙伴直到10月28日早上6时，不处理航班行李。〞
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比利时.html" target="_blank">
  比利时
 </a>
 法语广播电视台（RTBF）报导，搬运工是因要求提高人员配置而罢工，约有200个进出机场的航班被取消。工会代表和飞行伙伴周五紧急开会协商，但在未达成任何协议下结束，预计谈判周六上午10点继续。
 <br/>
 <br/>
 罢工在邻国荷兰学校假期结束时开始，恰好撞上比利时将展开的诸圣节（All Saints Day）假期。
 <br/>
 <br/>
 机场发言人皮耶哈德（Nathalie Pierard）周五表示，在比利时公司飞行伙伴罢工后，约300名旅客昨夜滞留布鲁塞尔札范登国际机场（Zaventem airport），预计周六晚上将有更多人停留。
 <br/>
 <br/>
 皮耶哈德说，受影响最多的是瑞恩航空公司（Ryanair）的航班，〝周五是秋季最忙的日子之一，今天应该有4.7万旅客准备起飞。〞
 <br/>
 <br/>
 布鲁塞尔机场网站表示，30家航空公司受到打击，包括英国航空公司（British Airways）、易捷航空公司（EasyJet）及摩洛哥皇家航空公司（Air Maroc）。
 <br/>
 <br/>
 部分航班转移至布鲁塞尔另一个国际机场沙勒罗瓦（Charleroi），或法国北部里尔（Lille）机场，建议旅客联络航空公司。
 <br/>
 <br/>
 布鲁塞尔机场是比利时最大机场，去年有近2500万旅客运输量，而南部的第2大机场沙勒罗瓦机场则是700万人次。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
