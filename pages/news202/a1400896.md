### 壮观殊胜 全球法轮功学员23年排字展正信
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】11月24日，来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 及世界各地的5400位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 ，在台北排出巨幅英文版《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转法轮.html" target="_blank">
  转法轮
 </a>
 》图像。而这种
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_排字.html" target="_blank">
  排字
 </a>
 活动，已有23年历史了。
 <br/>
 <br/>
 11月24号，在台北自由广场，一本巨大的英文版《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转法轮.html" target="_blank">
  转法轮
 </a>
 》跃然眼前。蓝底金字的书本，还有42道光芒，是由约5400位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 排列而成。
 <br/>
 <br/>
 〝《转法轮》这本书在全世界流行很广，已经将近有40种不同语言的版本。那中文版和英文版是流行最广的，我们这次排的是英文版。〞
 <br/>
 <br/>
 〝很震撼。〞
 <br/>
 <br/>
 〝我感觉非常震撼，非常好。〞
 <br/>
 <br/>
 法轮功学员：〝师父谢谢您！师父谢谢您！〞
 <br/>
 <br/>
 目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 有数十万人修炼法轮功。从2000年开始，他们每年都会举办大型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_排字.html" target="_blank">
  排字
 </a>
 活动。
 <br/>
 <br/>
 2009年，排出立体图形——中文版《转法轮》。
 <br/>
 <br/>
 2010年，排出立体莲花图形，和〝真善忍〞字样。
 <br/>
 <br/>
 2012年，排出法轮功创始人李洪志先生肖像。同一年还在总统府前，排出法轮图形和〝法正天地〞字样。
 <br/>
 <br/>
 2013年，排出立体莲花和〝法轮大法〞字样。
 <br/>
 <br/>
 2014年2次排字，一次排出法轮功修炼者双手合十〝谢师恩〞；一次排出女佛手持《转法轮》和〝佛光普照、礼义圆明〞大字。
 <br/>
 <br/>
 2015年，排出法轮图形和〝法轮常转〞字样。
 <br/>
 <br/>
 2016年，排出法轮图形和〝真善忍〞字样。
 <br/>
 <br/>
 2017年，排出法轮功修炼者立掌打坐，以及〝助师正法，法正乾坤〞大字。
 <br/>
 <br/>
 今年排出英文版《转法轮》，法轮功学员表示，在展现壮观图像的同时，也向社会传递正面能量。
 <br/>
 <br/>
 〝现场很祥和，很舒服。〞
 <br/>
 <br/>
 〝《转法轮》这本书，是对人身心都有益，他解决了很多人困惑的问题，他可以指导你修炼，明白更高深的道理。〞
 <br/>
 <br/>
 法轮功学员排字，起源自中国大陆。
 <br/>
 <br/>
 1996年，中国贵州法轮功学员排出法轮图形，从97年到99年，大陆法轮功学员先后排出〝真善忍〞、〝法轮图形〞及〝法轮大法〞字样。
 <br/>
 <br/>
 1999年中共党魁江泽民迫害法轮功后，大陆法轮功学员失去自由修炼环境，海外法轮功学员则延续了大型排字活动。通过壮观的图形文字，传递法轮功学员坚守信仰、和平反迫害的讯息，并要求立即停止迫害。
 <br/>
 <br/>
 我们要向不了解真相的公众说明，中共对法轮功的迫害仍在发生。
 <br/>
 <br/>
 作为美国公民，我有责任来这里告诉我的政府，法轮大法是好的。〞
 <br/>
 <br/>
 如今，法轮功已传播到全球100多个国家地区，全球法轮功学员排字的风景线，也折射出中国大陆与自由社会的鲜明对比。
 <br/>
 <br/>
 新唐人记者林澜综合报导
</div>
