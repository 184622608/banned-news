### 对中共承诺已麻木 美欧商会代表：需看到实际行动
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】北京日前在上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进口博览会.html" target="_blank">
  进口博览会
 </a>
 上承诺将继续开放市场。但欧美商会指，对中共的承诺已经麻木，没有具体行动的口号只令他们感到失望，已有欧洲企业呼吁欧盟也启动针对中共的贸易战。
 <br/>
 <br/>
 在11月5日的进博会开幕式上，北京最高领导人表示，将进一步降低关税，放宽市场准入，推进服务业开放，加强对知识产权的保护，并在教育、医疗等领域放宽外资股比限制等。
 <br/>
 <br/>
 美联社当日报导，在上海的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国商会.html" target="_blank">
  美国商会
 </a>
 主席杰瑞特在电邮中评论称，除非中方能相应展现有实质意义和可衡量的改变，否则对美企以及很多他国公司来说得不到什么成果。
 <br/>
 <br/>
 在中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟商会.html" target="_blank">
  欧盟商会
 </a>
 副主席、上海分会主席代开乐说，〝我们今天一直等待
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 主席向世界宣布未来几天将开始的改革，但是我们想听到的改革，（如）落实改革的完整步骤，以及明确的时间表却都没有提出来。〞
 <br/>
 <br/>
 代开乐表示，进博会虽然可能对许多国家减少对中方的贸易逆差有所帮助，但并不会解决中共内部改革不足的问题，川普（特朗普）政府带来的贸易压力可能让欧洲国家暂缓进一步对中国市场的投资。
 <br/>
 <br/>
 欧盟商会还在当晚发表声明称，对于一个被中方称作是里程碑的活动，欧洲商界原本是抱着更高的期待，而这种一再的重申承诺，令欧洲商界愈发感觉麻木。
 <br/>
 <br/>
 该商会还说，中方应该把重点放在向外资开放新的领域，而不是在圈定的领域中继续〝试点〞改革。欧盟企业已不再对中国自贸区心存幻想，这种〝胆小的改革方法〞与中国的经济影响力无法相衬。
 <br/>
 <br/>
 另据《纽约时报》，中国欧盟商会秘书长唐亚东表示，一些欧洲企业在过去几个月开始支持一种更强硬的立场，希望欧盟象美国一样限制中方对欧洲的出口，除非中共向欧洲企业提供同等的市场准入条件。
 <br/>
 <br/>
 他说，〝这是一个根本性的变化。〞
 <br/>
 <br/>
 《美国之音》报导说，尽管中共在过去40年里一直声称坚持开放和进步，但中国仍然是最奉行贸易保护主义的经济体。在经济合作与发展组织有关市场开放程度的排名中，中国在62个国家中排第59位。
 <br/>
 <br/>
 外界一直认为，中共只是在要求外国市场对中方开放，而中共自身一直实施高关税壁垒政策，阻碍外国产品进入中国市场。同时，中共对经济的行政干预也备受诟病。
 <br/>
 <br/>
 长期以来，北京一直在宣扬对外开放的承诺，但是在中国经营的外国企业在市场准入、知识产权、法律法规等诸多方面依然面临困境。很多外国公司谴责中共监管机构试图把他们挤出有前景的产业。外国公司在中国还面临强制转移技术的压力。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
