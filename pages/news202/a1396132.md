### 捍卫航行自由 法国明年派遣航空母舰赴印度洋
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】时值中国在具争议海域独断日增之际，法国国防部周五(19日)表示将于明年派遣
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_航空母舰.html" target="_blank">
  航空母舰
 </a>
 前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度洋.html" target="_blank">
  印度洋
 </a>
 ，捍卫海上航行自由。
 <br/>
 <br/>
 法国国防部长帕利（Florence Parly）接受〝普罗旺斯报〞（La Provence）访问时表示，目前在土伦（Toulon）港进行维修更新的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戴高乐号.html" target="_blank">
  戴高乐号
 </a>
 〞（Charles de Gaulle R91）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_航空母舰.html" target="_blank">
  航空母舰
 </a>
 ，应准备获派于明年初在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度洋.html" target="_blank">
  印度洋
 </a>
 上航行。
 <br/>
 <br/>
 帕利表示，法国〝永远站在第一线，捍卫国际海域上自由航行不可剥夺的权利〞。
 <br/>
 <br/>
 今年5月，法国西北风级两栖突击舰〝迪克斯木德号〞（Dixmude）在南海航行，同年8月法国军机也在此区域上空飞行。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/10-20/p9088901a422227386.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-20/p9088901a422227386-ss.jpg"/>
   <br/>

    图为，2017年2月8日，停靠土伦海军基地维修的戴高乐航空母舰。(ANNE-CHRISTINE POUJOULAT/AFP/Getty Images)
   </font>
  </a>
 </center>
 <br/>
 据了解，35,000吨级的戴高乐号航空母舰于2001年5月18日正式成军，是法国史上拥有的第十艘航空母舰，除了是法国目前正在操作中的唯一一艘航空母舰外，也是有史以来第一艘不属于美国海军的核子动力航空母舰。
 <br/>
 <br/>
 据维基百科表示，戴高乐号航空母舰与美国所拥有的数艘核子动力航舰一样，都是采用斜向飞行甲板，而不采用欧制航空母舰常见的滑跳式甲板设计。但其吨位只有美国的同类舰隻一半，因此戴高乐号只配备了两具舰首弹射器（美军航舰通常为四具），而舰载机的上限也只有一半，约为40架上下，主要包括海基版本的疾风式战斗机与超级军旗攻击机两款法制战机，以及美制的E-2C鹰眼式空中预警机。
 <br/>
 <br/>
 戴高乐号配备有非常先进的电子设备与法国最新锐的阿斯特15型防空飞弹（Aster 15）与萨德哈尔（SADRAL）轻型短程防空飞弹系统，使得整体的攻击能力远超过去法国拥有过的几艘航舰，在由法英合作的次世代航空母舰还没登场之前，戴高乐号可能还得稳居欧洲战力最强的航空母舰好一段时间。
