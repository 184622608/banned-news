### 澳洲水果藏针案 亚裔女职员恐面临10年监禁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 多地此前发生过数十宗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_水果藏针.html" target="_blank">
  水果藏针
 </a>
 事件，引起全国恐慌。经相关执法单位展开全国调查后，于今（11日）拘捕一名为水果农场工作的亚裔女职员。她12日在布里斯班出庭，被控污染食物罪，可面临10年监禁。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_水果藏针.html" target="_blank">
  水果藏针
 </a>
 事件于9月发生，主要涉及超市出售的盒装
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_草莓.html" target="_blank">
  草莓
 </a>
 (士多啤梨)，一名男子因进食藏针的草莓而入院。随后陆续有人将超市购买的草莓插入刺针的图片及视频放到社交网站。
 <br/>
 <br/>
 〝草莓藏针〞事态也逐渐发酵，9月17日开始有人在香蕉中也发现有藏针；18日，新南威尔斯州又有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 民众发现在超市购买的苹果中也被藏了针。
 <br/>
 <br/>
 警方经过2个月调查后，于首支在水果内被发现的针上，从DNA检验出50岁的郑美吾（My Ut Trinh，音译）涉案，遂采取逮捕行动。她被控7项污染食物罪，最高刑罚为3年监禁，但当局指控其犯罪情节特别严重，可能被加刑至10年。
 <br/>
 <br/>

  </a>
 </center>
 <br/>
 警方消息透露，郑美吾于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_昆士兰.html" target="_blank">
  昆士兰
 </a>
 省卡布丘（Caboolture）管理一间劳工中介公司，为昆士兰草莓农场安排员工。郑女对雇主心生不满，表明〝想他们垮台、关门大吉〞，于是犯案。
 <br/>
 <br/>
 昆士兰省侦缉督察长瓦克尔（Jon Wacker）表示：〝这是一起重大且史无前例的警方调查行动，过程相当繁复。〞其举动触发多人模仿犯罪，终引发全国性的〝食物恐怖主义〞风波。
 <br/>
 <br/>
 而事件导致澳洲多地的超级市场回收水果及下架，果农更倾倒大批草莓。
 <br/>
 <br/>

 昆士兰草莓种植者协会（Queensland Strawberry Growers Association）发言人罗林（Jennifer Rowling）乐见警方起诉嫌犯的消息，还说这场危机〝严重冲击〞昆士兰省的草莓农。
 <br/>
 她告诉澳洲广播公司（ABC）：〝不过，令人不安的是起诉罪名仅与6或7盒盒装草莓有关，证实大多数事件都是模仿或误传。〞
