  盗取专利技术
 </a>
 的事件，这次的受害者，是韩国大厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三星.html" target="_blank">
  三星
 </a>
 。韩国媒体报导，三星的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曲面萤幕.html" target="_blank">
  曲面萤幕
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专利技术.html" target="_blank">

 </a>
 被设备商卖给中国竞争对手。目前，韩国检方已经起诉涉案的11人。三星预估，这将让三星在三年内，营收损失超过58亿美元。

 中国再传盗取专利！这次的受害者，是韩国大厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三星.html" target="_blank">
  三星
 </a>
 。
 <br/>
 <br/>
 韩国KBS News：〝‘边缘显示’技术，是一种环绕手机角落，并可以触控的技术，也是三星在六年内，投入1500亿韩元的先进技术。〞
 <br/>
 <br/>
 韩国媒体报导，与三星合作30年的设备商Toptec，在中国厂商的利诱下，将三星的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专利技术.html" target="_blank">
  专利技术
 </a>
 ，卖给中国。
 <br/>
 <br/>
 韩国KBS News：〝三星委托生产‘边缘显示’设施的承包商，将设备带到中国。通过这种方式，从5月开始之后三个月，将设施和设计图交给了中国，获得了155亿韩元。〞
 <br/>
 <br/>
 《日经亚洲评论》报导，据传，非法取得技术的中国厂商，是三星主要对手华为的面板供应商京东方，与中国第二大面板厂华星光电。
 <br/>
 <br/>
 韩国KBS News：〝由于设备和技术遭到洩露，三星预估，三年内三星将遭受6.5万亿韩元的销售损失。〞
 <br/>
 <br/>
 目前，韩国检方起诉了11名嫌犯，其中包括Toptec的总裁兼董事总经理，以及Toptec附属公司的副总裁。
