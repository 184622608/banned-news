### 加拿大最大炼油厂爆炸引大火 冲击波憾动附近民宅
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】加拿大最大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炼油厂.html" target="_blank">
  炼油厂
 </a>
 8日发生爆炸引发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大火.html" target="_blank">
  大火
 </a>
 ，冲出30米高火舌，住在炼油厂附近的住宅都能感受到明显摇晃。火势在数小时后才控制，已知4人受伤。
 <br/>
 <br/>
 综合媒体报导，尔文
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炼油厂.html" target="_blank">
  炼油厂
 </a>
 （Irving Oil refinery）位在新布朗斯威克省（New Brunswick）圣约翰（St. John's）距离市中心仅约5公里，是加拿大最大炼油厂，每日产能达30万桶，过半数产品出口美国。
 <br/>
 <br/>
 加拿大广播电台（Radio Canada）引述炼油厂附近居民毕比（Rob Beebe）说，爆炸发生时，他整个房子都在摇晃。
 <br/>
 <br/>
 居民达尔泽尔（Gordon Dalzell）形容∶〝我们的房子、窗户都在摇动，能直接感受到爆炸的冲击。〞
 <br/>
 <br/>
 上传社群网站的照片可见到事发现场滚滚浓烟不断的往上窜升伴随凶猛烈火。炼油厂员工形容，当时的爆炸冲击波足以将他吹倒在地，他与同事立即离开。
 <br/>
 <br/>
 事发后，当局封锁炼油厂附近部分道路，并建议附近居民留在家中。当地医院发出橙色警报，预备接收伤者。而爆炸和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大火.html" target="_blank">
  大火
 </a>
 造成4人受伤。
 <br/>
 <br/>
 拥有这家炼油厂的公司推文表示，当地发生〝重大事件〞，并说：〝我们现阶段积极评估情况，有消息会再进行通知。〞
 <br/>
 <br/>
 稍后厂方表示，意外事件是因柴油处理机器故障所造成，对未酿成严重后果表示〝感恩〞。
 <br/>
 <br/>
 当局正调查爆炸大火的起因。
 <br/>
 <br/>
 (责任编辑：程以仁)
