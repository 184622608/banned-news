### 批评阿萨德政权 叙利亚2名记者遭枪杀
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叙利亚.html" target="_blank">
  叙利亚
 </a>
 广播电台和国际观察组织周五（23日）称，2名一直批评叙国政权和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基地组织.html" target="_blank">
  基地组织
 </a>
 的关键性人物，于叛军在西北部最后重要基地被不明枪手开枪打死。
 <br/>
 <br/>
 Fresh FM电台在脸书页面贴文称，公民记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法瑞斯.html" target="_blank">
  法瑞斯
 </a>
 （Raed Fares）和吉内德（Hamod Jnaid）在伊德利布省（Idlib）的〝卡夫纳贝尔（Kafr Nabel）镇被一辆箱型车内的不明攻击者开枪打死〞。
 <br/>
 <br/>
 BBC报导，法瑞斯是Radio Fresh的创始人。Radio Fresh是一个播放新闻消息的频道，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叙利亚.html" target="_blank">
  叙利亚
 </a>
 反对派控制区播放新闻，一直是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基地组织.html" target="_blank">
  基地组织
 </a>
 的暗杀目标。
 <br/>
 <br/>
 4年前，当
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊斯兰.html" target="_blank">
  伊斯兰
 </a>
 国（IS）占据伊德利卜省时，该站的办公室就曾遭到武装分子的袭击。2016年，法瑞斯被叙利亚前基地组织附属机构沙姆解放组织（Hayat Tahrir al-Sham）拘留。
 <br/>
 <br/>
 现年46岁的法雷斯早些时候告诉《华盛顿邮报》，说他至少在一次暗杀行动中幸存下来。他曾多次被基地组织的叙利亚分支机构绑架。法雷斯说，〝但我们没有放弃。我们仍在广播我们对叙利亚革命的独立报导，反恐和倡导宽容。〞
 <br/>
 <br/>
 法瑞斯也是叙利亚总统阿萨德的评论家，他10月5日在脸书上发表的最后一篇帖文中描述了Kafr Nabl的抗议活动。
 <br/>
 <br/>
 他在贴文中写到，〝Kafr Nabl的人民在Huriyah‘自由’广场高呼：人民希望政权垮台。我们在2011年就开始这样做了，我们还在继续。〞
 <br/>
 <br/>

 设于英国的叙利亚人权了望台组织（Syrian Observatory for Human Rights）表示，这2名男子遭攻击后伤重不治。
 <br/>
 <br/>
 Radio Fresh的脸书页面于11月23日也张贴了法瑞斯及吉内德被枪杀的照片。
 <br/>
 <br/>
 目前没有任何团体出面承认犯行。
 <br/>
 <br/>
 由叙国前基地组织分支激进分子领导的沙姆解放组织掌控伊德利布省大部分地区和周遭区域，其余地区则由亲土耳其的叛军把持。伊斯兰国集团（IS）也在伊德利布省占据一方。
