### 悉尼1小时降近1个月大雨 海陆空交通大乱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】受到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_低压系统.html" target="_blank">
  低压系统
 </a>
 影响，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 气象局周三(28日)，发布强风、暴雨警告，影响范围包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悉尼.html" target="_blank">
  悉尼
 </a>
 和新州沿海地区。据悉，悉尼(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雪梨.html" target="_blank">
  雪梨
 </a>
 )一早就下起滂沱大雨，海陆空交通大乱，悉尼机场多班客机延误或取消、铁路因淹水停驶、市面成泽国，约9000户断电；气象部门表示雨势经已减弱，但全日继续有雨。
 <br/>
 <br/>
 暴雨1小时内的降雨量超过73毫米，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悉尼.html" target="_blank">
  悉尼
 </a>
 (Sydney)以往11月的平均总雨量，仅为84毫米。新州紧急救援局稍早向住在低洼地区的居民发出防洪警告，要求人们用沙袋封住门口，清理屋顶沟槽和排水沟，并做好疏散的准备。
 <br/>
 <br/>
 今早上班尖峰时间，令上班族苦不堪言，雨水顺着楼梯往地铁灌，西装笔挺的上班族被迫拉高裤裆、拿起皮鞋，狼狈地涉水而行；民宅被水淹没，屋主无奈地摇头。
 <br/>
 <br/>
 稍早前，悉尼机场只有一条跑道正常运作，现已恢复正常，但仍有50班航班要取消；铁路及海上渡轮服务一度暂停，但逐渐恢复运作。
 <br/>
 <br/>

 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 气象局稍早表示，因受到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_低压系统.html" target="_blank">
  低压系统
 </a>
 影响，从周三起，破坏性强风和大暴雨将袭击悉尼和新州沿海地区。悉尼局部地区的降雨量恐达100毫米，造成洪水。提醒民众需做好应对恶劣天气的准备。
 <br/>
 <br/>
 气象局预报员派恩（Anita Pyne）表示，悉尼和伊拉瓦拉地区的降雨量可能最大，但南海岸（South Coast）北部、中央海岸（Central Coast）和蓝山地区预计也受到严重影响。
 <br/>
 <br/>
 暴风雨从早上开始，大部分地区的降雨将在6-12个小时内结束。
 <br/>
 <br/>
 派恩表示，预计悉尼和伊拉瓦拉的降雨量可达100毫米，局部地区降雨量可能超过200毫米。
 <br/>
 <br/>
 新州紧急救援服务局（NSW SES）警告说，内皮恩（Nepean）河上游和下游、霍克斯伯里河（ Hawkesbury River）、科洛河（Colo River）、 乔治斯河（Georges River）和沃罗诺拉河（Woronora River）可能会暴发轻度至中度洪水。
 <br/>
 <br/>
 随着降雨减弱，周三晚的强风可能会达到破坏性的程度，气象局预报员杜马（Olenka Duma）说，新州部分沿海地区可能遭受破坏。
 <br/>
 <br/>
 预计低压系统周三离开沿海地区，周四前将向北移动。
