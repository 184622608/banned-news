### 追查美记者失踪案 川普派蓬佩奥赴中东并致电沙特国王
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】周一（10月15日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统表示他已致电
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特国王.html" target="_blank">
  沙特国王
 </a>
 ，而对方否认知晓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特记者.html" target="_blank">
  沙特记者
 </a>
 卡舒吉（Jamal Khashoggi）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 原因。为进一步追查事件，川普宣布他将派国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 （Mike Pompeo）立即赴沙特与国王面谈。
 <br/>
 <br/>
 周一上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在推文中表示，为了彻查受雇于美国《华盛顿邮报》（The Washington Post）的沙特阿拉伯记者卡舒吉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 一案，他将派美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 专门赴沙特进行会谈。
 <br/>
 <br/>
 〝刚刚与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特国王.html" target="_blank">
  沙特国王
 </a>
 谈过，他否认知道任何可能已经发生于‘我们沙特公民’身上的事情。他说他们正与土耳其政府密切合作寻求答案。我即刻派我们的国务卿与国王会晤！〞
 <br/>
 <br/>

 <br/>
 周一上午，川普在赴佛州慰问灾区前向驻白宫的记者们表示，他已要求蓬佩奥立即登机前往沙特。如果需要的话，还将奔赴土耳其等其它地方，以敦促彻底调查事件真相。
 <br/>
 <br/>
 川普告诉记者，蓬佩奥在一小时内就会启程。
 <br/>
 <br/>
 川普接着表示，在他看来，这很像是流氓杀手干的，不过，〝谁知道呢？〞
 <br/>
 <br/>
 川普指出，沙特国王在电话中断然否认该事件是沙特政府干的，但〝我们（美国）将努力把事件弄个水落石出〞。
 <br/>
 <br/>
 卡舒吉是《华盛顿邮报》专栏作家，长期报导沙特和中东事务，是沙特王储的批评者，于去年逃离沙特。
 <br/>
 <br/>
 10月2日，卡舒吉因结婚一事前往沙特驻土耳其伊斯坦布尔领事馆领取文件，但之后离奇消失。
 <br/>
 <br/>
 据土耳其媒体报导，卡舒吉已在领事馆遇害，尸体已转移。土耳其《每日晨报》（Sabah）揭露，在卡舒吉失踪当日，一个由15名沙特人组成的〝暗杀小组〞分别乘坐两架私人飞机抵达伊斯坦布尔国际机场，然后前往土耳其领事馆。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
