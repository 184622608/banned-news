### 联合国令放宽对伊朗部分制裁 美不认同
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】10月3日，继
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 最高法院宣布，基于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 签署的友好条约，美国应解除对伊朗部份产品的制裁限制。随后，美国国务卿蓬佩奥宣布，美国废止早已过期的与伊朗签署的条约。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务卿蓬佩奥：〝我宣布，美国废止1955年与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 签署的友好条约。这早在39年前（美伊两国断交）就该过期。〞
 <br/>
 <br/>
 美国总统川普5月宣布退出伊核协议后，8月重新向伊朗实施第一轮制裁，第二轮制裁将于11月实施。
 <br/>
 <br/>
 伊朗向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 法院提告，指川普政府所实施的制裁违反了所谓的〝友好条约〞。
 <br/>
 <br/>
 因此
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国法院.html" target="_blank">
  联合国法院
 </a>
 周三宣布，基于1955年美国同伊朗的条约，美国应解除对伊朗部份物品的进口禁令。
 <br/>
 <br/>
 国际法院法官Abdulqawi Yusuf：〝法院认为，美国根据其根据1955年条约所承担的义务，必须解除任何因2018年5月8日宣布的措施而产生的障碍，向伊朗境内出口人道主义物品所需货物。〞
 <br/>
 <br/>
 美国已表示对联合国法院不予理会。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝伊朗正在滥用国际法院达到政治宣传目的。〞
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
