### 欧亚会议开幕 法轮功学员场外请愿
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧亚会.html" target="_blank">
   欧亚会
  </a>
  议开幕
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
   法轮功学员
  </a>
  场外
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_请愿.html" target="_blank">
   请愿
  </a>
 </h4>
 <br/>
 欧亚经济论坛10月18号在比利时首都布鲁塞尔开幕，比利时和荷兰的部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 ，当天下午在会场外和平
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_请愿.html" target="_blank">
  请愿
 </a>
 ，向包括中共总理李克强在内的各国首脑呼吁停止迫害法轮功。
 <br/>
 <br/>
 他们打出〝法轮大法好〞、〝真善忍好〞、〝法办江泽民〞、〝停止迫害法轮功〞等横幅，并在现场演示法轮功功法。
 <br/>
 <br/>
 比利时的法轮功负责人尼科（Nico）表示，他们举办这个请愿活动，目的是让各国首脑了解仍在在中国持续的对法轮功的迫害。同时希望向欧洲领袖们发出一个强烈信号，那就是，在同中共谈论经济和贸易的同时，要知道自己正在与一个严重破坏传统文化和价值观的专制政权在谈话。
 <br/>
 <br/>
 <h4>
  中共驻澳门最高官坠楼死亡 引议论
 </h4>
 <br/>
 正当外界盛传中国国家主席习近平即将二次〝南巡〞时，中共国务院港澳办10月21号突然在网页宣布，澳门中联办主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 ，因患抑郁症20号
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_堕楼身亡.html" target="_blank">
  堕楼身亡
 </a>
 。
 <br/>
 <br/>
 澳门中联办网站显示，郑晓松19号还在中联办办公楼会见了民联智库成员。
 <br/>
 <br/>
 香港和澳门政商界都对郑晓松的突然自杀感到惊愕，认为极不寻常。
 <br/>
 <br/>
 中国大陆网友则对中共官员的频繁坠楼表示疑惑。
 <br/>
 <br/>
 有人问：〝为啥每次中共官员或者中共国企业家的死因不是跳楼坠亡...就是车祸...〞
 <br/>
 据不完全统计, 今年1到9月, 已经发生7起中共官员坠楼事件。
 <br/>
 <br/>
 <h4>
  7被囚者获评〝中国杰出民主人士〞
 </h4>
 <br/>
 七位被中共关押在监狱的中国民主党人，今年被美国旧金山〝中国民主教育基金会〞评选为年度〝中国杰出民主人士〞。颁奖典礼10月20号在旧金山中国城举行，该基金会会长方政在仪式上宣布秦永敏、刘贤斌、陈西、陈树庆、陈卫、吕耿松、谢长发为获奖者。并称他们为〝中国民主党狱中群英〞。 他说, 7人总共坐牢已经超过109年。
