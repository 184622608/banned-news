### 大韩航空会长赵亮镐 涉贪污诈欺八项罪名
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大韩航空.html" target="_blank">
  大韩航空
 </a>
 会长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵亮镐.html" target="_blank">
  赵亮镐
 </a>
 涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贪污.html" target="_blank">
  贪污
 </a>
 欺诈，遭检方起诉八项罪名。这是大韩航空公司赵亮镐夫人及女儿，因涉嫌殴打谩骂员工遭逮捕调查后，曝出的又一起丑闻。
 <br/>
 <br/>
 周一，首尔检方以涉嫌挪用公款、渎职、欺诈以及违反《药事法》等8项罪名，起诉韩进集团兼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大韩航空.html" target="_blank">
  大韩航空
 </a>
 公司会长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵亮镐.html" target="_blank">
  赵亮镐
 </a>
 。
 <br/>
 <br/>
 检方指控，赵亮镐造成韩航超过1700万美元的亏损，在2013年到2018年5月之间，私设贸易公司提供〝中介服务〞，以不公平方式得标，项目从飞行设备到免税商品都有。
 <br/>
 <br/>
 自从韩国总统文在寅上台后，誓言要打击腐败，韩国大企业集团人人自危，包括乐天集团高管和三星负责人都曾因涉及贪腐被判刑。
 <br/>
 <br/>
 同一天，今年3月因为涉嫌向两个人泼水，被警方限制出境的赵亮镐35岁次女赵显旼，已经被撤销所有指控。
 <br/>
 <br/>
 赵亮镐的长女、时任副会长的赵显娥在2014年闹出〝坚果门〞事件。她当时要求一架大韩航空公司的飞机返回纽约，只因她不喜欢空服员在头等舱的服务方式。2015年，赵显娥因违反航空安全法等罪名，被判入狱一年。
 <br/>
 <br/>
 今年五月，赵亮镐的夫人李明姬因涉嫌殴打谩骂员工，接受首尔地方警署调查，警方确认有10多名受害者。
 <br/>
 <br/>
 多年来，赵亮镐家庭的一系列丑闻引发员工对管理层的抗议，要求他们下台。
 <br/>
 <br/>
 新唐人记者李梅、张琪综合报导
