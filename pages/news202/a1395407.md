### 记者失踪案延烧 西方企业退出沙特投资会议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】受雇于《华邮》的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特阿拉伯.html" target="_blank">
  沙特阿拉伯
 </a>
 记者卡舒吉失踪继续燃烧，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）13日称，可能对沙国进行制裁。随后，从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技大亨.html" target="_blank">
  科技大亨
 </a>
 到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_媒体巨擘.html" target="_blank">
  媒体巨擘
 </a>
 ，西方国家多家企业纷纷退出下周在利雅德举行的未来投资计划会议。
 <br/>
 <br/>
 外资对沙国经济多元化与创造就业的计划至关重要，但因卡舒吉（Jamal Khashoggi）失踪案持续发酵，媒体机构和愈来愈多企业主管接连宣布，退出23日起在利雅德举行的未来投资计划（Future Investment Initiative）会议。
 <br/>
 <br/>
 路透社报导，摩根大通集团（JPMorgan Chase &amp; Co.）与福特汽车公司（Ford Motor Co.）周日（14日）先后表示，取消出席未来投资计划会议。两家公司没有解释取消原因，对于是否与卡舒吉失踪案有关，公司也没有置评。
 <br/>
 <br/>
 英国商业大亨布兰森（Richard Branson）和美国共乘服务巨擘优步（Uber）执行长柯霍斯洛夏西（Dara Khosrowshahi）等也已经宣布不会出席会议。柯霍斯洛夏西的退出可能对沙特影响重大，因为沙特主权财富基金公共投资基金（PIF）已对Uber叫车服务应用程式（App）投资了35亿美元。
 <br/>
 <br/>
 苏格兰男星杰瑞德巴特勒（Gerard Butler）原计划要出席新片〝潜舰猎杀令〞（Hunter Killer）在沙特的首映活动，临行前也已取消访问。他周六告诉美媒，〝我们在原定要启程前一天，听到卡舒吉失踪消息…但真的要出发时，觉得这不是明智之举。〞
 <br/>
 <br/>
 美国财政部长梅努钦（Steven Mnuchin）仍计划出席会议，但白宫国家经济会议主席柯德洛（Larry Kudlow）在美国广播公司（ABC）节目〝本周〞（This Week）上说可能计划有变数。
 <br/>
 <br/>
 为〝华盛顿邮报〞（WP）撰写专栏的卡舒吉向来直言批评沙特政策，他10月2日进入沙特驻土耳其伊斯坦布尔领事馆后人间蒸发。土耳其认为他遭到杀害分尸，沙特则否认相关指控。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 13日表示，假如调查发现沙特当局参与了卡舒吉的失踪事件，美国〝会非常沮丧和愤怒〞，将会对其进行〝严厉的惩罚〞，但不会停止对沙特军售。
 <br/>
 <br/>
 另外，欧洲3大经济体德国、法国及英国周日也在联合声明中说，3国〝极为严肃〞的看待此案，同时也鼓励沙特和土耳其联手调查找出真相，希望〝沙国当局能提供完整详细的回应〞。声明中说，〝我们已将这项讯息直接传达沙特有关当局。〞
 <br/>
 <br/>
 记者失踪效应，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特阿拉伯.html" target="_blank">
  沙特阿拉伯
 </a>
 证交所全股指数（TASI）周日早盘一度大跌7.02%至7001.94点，抹去今年来所有涨幅，且该指数186档成分股几乎全数下挫，仅3档个股幸免。创下2015年8月以来最大单日跌幅。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
