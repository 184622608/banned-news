### 彭斯会晤昂山素季 关注罗兴亚人遭遇
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】周三，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东盟峰会.html" target="_blank">
  东盟峰会
 </a>
 期间，会见了缅甸领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_昂山素季.html" target="_blank">
  昂山素季
 </a>
 。彭斯表示，缅甸对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗兴亚人.html" target="_blank">
  罗兴亚人
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迫害.html" target="_blank">
  迫害
 </a>
 毫无藉口，但昂山素季拒绝批评缅甸政府。
 <br/>
 <br/>
 在与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_昂山素季.html" target="_blank">
  昂山素季
 </a>
 会谈之前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 表示，缅甸政府的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迫害.html" target="_blank">
  迫害
 </a>
 导致70万
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗兴亚人.html" target="_blank">
  罗兴亚人
 </a>
 逃亡，对此没有藉口。
 <br/>
 <br/>
 美国副总统彭斯：〝（缅甸）军方和义务警员的暴力和迫害，导致70万罗兴亚人逃到孟加拉国，对此没有藉口。〞
 <br/>
 <br/>
 彭斯表示，希望听到昂山素季在追究导致数十万人流离失所暴行的负责人方面，取得进展。
 <br/>
 <br/>
 彭斯还表示，美国重视媒体自由和独立。去年，两名路透社记者在缅甸被逮捕和判刑。
 <br/>
 <br/>
 美国副总统彭斯：〝在美国我们相信我们的民主制度和理想，包括自由和独立的媒体。去年秋天（缅甸）逮捕和监禁了两名记者，使数百万美国人深感不安。〞
 <br/>
 <br/>
 彭斯表示，希望美国在缅甸支持的民主改革，以及美国对罗兴亚人的人道主义支持，最终能促成和平、成功的民主社会，为缅甸民众带来新的开始。
 <br/>
 <br/>
 昂山素季表示，双方通过讨论和交换意见，有助于互相了解。但她没有批评缅甸政府迫害万罗兴亚人。
 <br/>
 <br/>
 新唐人记者毕心慈、宇微综合报导
</div>
