### 〝医疗机械手臂〞大暴走 戳破大动脉害死病患
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】被誉为全球最先进的医疗设备〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_达文西.html" target="_blank">
  达文西
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_机械手臂.html" target="_blank">
  机械手臂
 </a>
 〞，可以进行细微的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手术.html" target="_blank">
  手术
 </a>
 作业。许多医院引进机械手臂协助手术进行，但3年前有机械手臂发生问题影响手术，当时的执刀医生内尔近日在听证会中还原当时的情况。
 <br/>
 <br/>
 综合英国媒体报导，该事故发生于2015年2月，在英国纽卡斯尔（Newcastle）的弗里曼医院（Freeman Hospital），当地政府从本月5日到9日对此事件举行了为期5天的听证会。
 <br/>
 <br/>
 内尔（Sukumaran Nair）表示，当时英国各大医院都抢着使用这项新技术，自己所在的医院也购入了该设备。而且对医生进行了培训。
 <br/>
 <br/>
 不过，当他奉命使用该
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_机械手臂.html" target="_blank">
  机械手臂
 </a>
 为一位患有心脏病的病患动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手术.html" target="_blank">
  手术
 </a>
 前，内尔因有事错过了两堂有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_达文西.html" target="_blank">
  达文西
 </a>
 （DaVinci）机械手臂使用的课程。
 <br/>
 <br/>
 内尔说，当初那场手术他就像是被〝赶鸭子上架〞，〝一切都准备不足，我就像还不会走路，就被要求要跑步一样。〞
 <br/>
 <br/>
 内尔描述当天使用机械手臂动手术的种种怪象，包括机械手臂发出刺耳尖锐的声音，导
 <br/>
 致他和助理医师Thasee Pillay无法正确沟通手术流程。他发现病患的心脏缝线并没有按照〝有组织的方式〞放置。
 <br/>
 <br/>
 接着内尔看到机械手臂随意移动、触碰干扰Thasee Pillay的手。
 <br/>
 <br/>
 更为严重的是机械手臂竟把病患的大动脉戳破，病患的血喷溅到机械手臂的镜头上，导致内尔无法继续操作，只好改回人工方式，修复大动脉后继续完成手术。
 <br/>
 <br/>
 但不幸的是，病人在手术后不到一周内就逝世。内尔也承认如果用人工方法，病患可能不会死亡。
 <br/>
 <br/>
 内尔在听证会中还指控，当天原本有两名机械手臂专家负责监看手术过程，但当机械手臂出现问题、场面失控时，内尔却找不到2名〝专家〞的身影，让他很无助。
 <br/>
 <br/>
 对此，2名专家在听证会中辩称是因〝不想打扰医师工作〞才会离开现场。
 <br/>
 <br/>
 内尔也对发生此事故表示遗憾，他表示，事发后他就再也没有使用过机械手臂来为病人开刀。
 <br/>
 <br/>
 据悉，这次手术是英国第一次进行此类手术，当时也没有告诉69岁的病患Stephen Pettitt，他是第一个使用机械手臂做心脏手术的人。
 <br/>
