### 孟晚舟保释听证进行中 检方要求不予保释
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】最新消息，华为CFO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保释听证会.html" target="_blank">
  保释听证会
 </a>
 还在进行，加拿大检方指不予
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保释.html" target="_blank">
  保释
 </a>
 。
 <br/>
 <br/>
 截至美东时间7日15时，加拿大法院对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保释.html" target="_blank">
  保释
 </a>
 听证仍在进行中。但加拿大检方表示孟晚舟有潜逃可能，要求不能被保释。
 <br/>
 <br/>
 检方表示，美国早在今年8月就已对孟晚舟下达逮捕令，请求加方予以合作。但加拿大相关部门直到11月才批准执行。刚好孟晚舟12月1日入境转机，立即实施逮捕。
 <br/>
 <br/>
 加国检方指，孟晚舟应该还押，等候引渡美国，因为她有可能弃保潜逃。
 <br/>
 <br/>
 检方表示，孟晚舟家族的巨额财富使她有潜逃风险。
 <br/>
 <br/>
 但孟晚舟的律师辩称，有钱不是拒保理由。该律师还企图使用中国文化中的〝面子〞概念为孟争取保释机会，称孟不会违反法院命令，因为这会令她的父亲和〝整个中国（中共）〞难堪。该律师还建议，可以把电子监视当成保释条件。
 <br/>
 <br/>
 孟晚舟是华为创办人任正非的女儿，于12月1日在温哥华转机时被逮捕，报导指是她由香港转机去墨西哥。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
