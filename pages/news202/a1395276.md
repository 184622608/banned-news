### 亚太法轮功学员首尔大游行 平和壮观
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】13日，来自台湾、香港、日本、新加坡等地的一千五百多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员，在韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首尔.html" target="_blank">
  首尔
 </a>
 举行了盛大的游行活动。平和而壮观的游行队伍，让韩国民众驻留，并对法轮功有进一步的了解。
 <br/>
 <br/>
 刚刚来到韩国的亚洲各国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员聚集在一起，不一会儿做好了游行的准备。
 <br/>
 <br/>
 浩浩荡荡的天国乐团在前面开道，由〝洪法、讲真相、退党〞三个方阵组成的游行队伍从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首尔.html" target="_blank">
  首尔
 </a>
 最热闹的崇礼门广场出发。
 <br/>
 <br/>
 这些方阵在讲述法轮大法洪传26年来的历程，以及在中国的遭遇。
 <br/>
 <br/>
 游行队伍中展示法轮功功法的花车和仙女队、穿民族服装的队伍，吸引了不少路人的视线。
 <br/>
 <br/>
 韩国法轮功学员：〝感谢师尊为我们指点了我们人生的真谛〞
 <br/>
 <br/>
 印尼法轮功学员：〝全世界各个民族都欢迎法轮大法，世界需要真善忍〞
 <br/>
 <br/>
 游行队伍途经距离青瓦台不远的光华门广场，很多首尔民众第一次看到如此壮观的法轮功游行惊叹不已。
 <br/>
 <br/>
 他们纷纷用手机拍照，了解法轮功真相，有的向队伍挥手致意。
 <br/>
 <br/>
 一些市民虽然对法轮功不是很了解，但都表示这样的游行令人欢喜并有平和的感觉。
 <br/>
 <br/>
 韩国市民：〝游行队伍非常与众不同，平和，感觉非常好〞
 <br/>
 <br/>
 韩国市民：〝彼此不区分谁是韩国人谁是外国人，大家就像一个民族一样，这一点非常触动我的心灵。〞
 <br/>
 <br/>
 参加游行的法轮功香港学员表示，能在韩国参加游行感到特别有意义。
 <br/>
 <br/>
 香港法轮功学员：〝我在韩国展示法轮大法的美好，非常高兴〞
 <br/>
 <br/>
 一位韩国学员再次向法轮大法创始人李洪志老师表达感激之情。
 <br/>
 <br/>
 韩国法轮功学员：〝非常感谢，修炼后我的心情平和，让我身体健康，非常感谢引导我修炼的师父〞
 <br/>
 <br/>
 新唐人记者杨云韩国首尔报导
</div>
