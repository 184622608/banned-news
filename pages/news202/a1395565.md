### 追随川普 澳考虑搬迁驻以大使馆到耶城
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】继美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）今年5月将美国驻以色列大使馆搬迁到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_耶路撒冷.html" target="_blank">
  耶路撒冷
 </a>
 后，澳洲总理莫里森周二（16日）也表示，政府正在考虑把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲驻以大使馆.html" target="_blank">
  澳洲驻以大使馆
 </a>
 迁往耶路撒冷，承认其首都地位。
 <br/>
 <br/>
 莫里森（Scott Morrison）周二召开记者会时表示，澳洲正式承认
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_耶路撒冷.html" target="_blank">
  耶路撒冷
 </a>
 为以色列首都，并将大使馆迁到耶路撒冷的提案，是个值得考虑的问题，是〝合理〞且〝具说服力〞的。
 <br/>
 <br/>
 他说：〝我们致力于以巴两国取得好的方案，但坦白说，两国方案进行并不顺利，没有太多进展，而我们不能老是做同样的事，却期待不同的结果产生。〞
 <br/>
 <br/>
 莫里森的决定可谓是打破澳洲过去几代政府数十年来的政策。此举可能会在世界范围内引起重大回响，因为这是继美国之后，第一个考虑将大使馆迁往耶路撒冷的主要国家。
 <br/>
 <br/>
 因本月20日，悉尼犹太票仓温特沃思（Wentworth）选区即将举行关键性议员补选，莫里森所属自由党的候选人过去担任澳驻以大使，但民调落后。莫里森在选前倒数几天作这项令人意外的宣布，被认为可能是想拉拢犹太人票仓。
 <br/>
 <br/>
 他告诉悉尼晨锋报，温特沃思选区补选候选人、前澳洲驻以色列大使沙马（Dave Sharma）在迁移大使馆的问题上对他产生了很大影响。
 <br/>
 <br/>
 沙马一直支持澳洲效仿美国，将大使馆迁移到耶路撒冷。他今年5月在费尔法克斯媒体发表的一篇评论文章里称，西耶路撒冷已经无法避免的成为了以色列的一部分，将大使馆迁移到西耶路撒冷是承认这一现实。
 <br/>
 <br/>
 莫里森也表示，大使馆设在哪里和何时搬迁需要〝较长期的考虑〞。他说，〝在未来的几个月，我将要和内阁同事进一步商讨这个（迁移大使馆的）问题。〞
 <br/>
 <br/>
 莫里森还透露，会考虑迁移大使馆，在时间上是受到了联合国即将对巴勒斯坦是否应该担任77国集团（G77）主席国进行投票的影响。澳洲反对巴勒斯坦担任77国集团主席国。
 <br/>
 <br/>
 周二，以色列总理内塔尼亚胡发推文感谢莫里森，他说，〝今天我和莫里森进行了通话，他告诉我，他正在考虑正式承认耶路撒冷是以色列的首都，并将澳洲驻以色列大使馆迁移到耶路撒冷。对此，我非常感谢。我们将继续巩固以色列和澳洲之间的纽带。〞
 <br/>
 <br/>
 长期以来，承认耶路撒冷为以色列首都一直是一个极具争议的话题。以色列在1948年独立后指定耶路撒冷为〝永恒的、不可分割的〞首都，而巴勒斯坦人将耶路撒冷的东部看成是自己的未来首都。
 <br/>
 <br/>
 今年5月，美国驻以色列大使馆正式从特拉维夫迁往耶路撒冷，兑现了美国20多年前的承诺。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在迁移大使馆时强调，耶路撒冷是以色列政府所在地，是以色列最高法庭、国会大厦等重要建筑所在地，承认耶路撒冷的首都地位只是承认现实。
 <br/>
 <br/>
 另外，除了迁移大使馆之外，莫里森周二还宣布，将通过两国大使馆建立国防联系，计划加强和以色列之间的安全合作。
 <br/>
 <br/>
 与此同时，莫里森还表示，澳洲政府将考虑是否放弃支持伊朗核协议。
 <br/>
 <br/>
 伊朗近年来着手开发核武器，想利用核威慑力增加谈判筹码、攫取更多利益。国际社会与伊朗展开核谈判之后，美、英、法、德、俄、中及欧盟于2015年与伊朗签署〝伊朗核协议〞，该协议旨在通过减缓经济制裁来换取伊朗约束核计划。
 <br/>
 <br/>
 今年5月，川普宣布退出伊朗核协议。他说，该协议不但没能阻止伊朗发展核武器，也没能规范伊朗资助恐怖主义的行为。他认为，伊朗只是受困于近年来疲敝的国民经济而选择〝暗渡陈仓〞，在通过核协议获得海量资金及合作的同时暗地里继续研发核武器。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
