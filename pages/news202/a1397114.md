### 巴西大选决选开始 波索纳洛有望当选总统
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】巴西周日举行总统决选，根据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民调.html" target="_blank">
  民调
 </a>
 公司周六公布的结果，显示极右派
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统候选人.html" target="_blank">
  总统候选人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波索纳洛.html" target="_blank">
  波索纳洛
 </a>
 ，持续以两位数领先左派对手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈达德.html" target="_blank">
  哈达德
 </a>
 。
 <br/>
 <br/>
 巴西总统选举，因为第一轮投票没有候选人赢得过半数选票，10月28日举行第二轮投票进行决选。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民调.html" target="_blank">
  民调
 </a>
 机构Ibope本周稍早的结果显示，号称〝巴西版川普〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波索纳洛.html" target="_blank">
  波索纳洛
 </a>
 （Jair Bolsonaro）支持度为54%，主要竞争对手左翼候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈达德.html" target="_blank">
  哈达德
 </a>
 （Fernando Haddad）支持率为46%；而周六晚些时候公布的其他民调（Datafolha）显示，波索纳洛将赢得55%选票，哈达德为45%。
 <br/>
 <br/>
 63岁的波索纳洛对犯罪立场强硬，矢言通过给予警察更大的自主权来打击巴西武装罪犯。巴西如今充斥贪腐、暴力犯罪和经济不安，波索纳洛的政见引发广大共鸣。一个月前，波索纳洛遭持刀民众刺伤腹部，住院数周出院后、人气更旺。
 <br/>
 <br/>
 周六，波索纳洛通过社交媒体进行最后冲刺。他在推文中说：〝我们将透过捍卫法律、遵循宪法为巴西带来改变，这个国家属于我们所有人。〞〝有不同意见、种族及倾向的巴西。〞
 <br/>
 <br/>
 他的决选对手、55岁的哈达德出身巴西劳工党（PT），誓言让巴西经济繁荣并打击贫穷；哈达德也坚信自己能逆转颓势、拿下胜利。
 <br/>
 <br/>
 投资者们对对波索纳洛支持率领先感到振奋，波索纳洛选择经济学家和投资银行家古埃德斯（Paulo Guedes）作为自己的经济顾问，如果他当选，古埃德斯可能是财政部长人选。分析师纷纷预测未来一年巴西股市会大涨至历史新高。
 <br/>
 <br/>
 新唐人记者严枫、雨薇综合报导
</div>
