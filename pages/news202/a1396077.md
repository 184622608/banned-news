### 10月18日国际新闻简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴塞罗纳.html" target="_blank">
  巴塞罗纳
 </a>
 著名的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大教堂.html" target="_blank">
  大教堂
 </a>
 ，已经修建了130年。每年吸引2千万游客。如今获得市议会许可，可以继续开工。来看一组欧洲简讯。
 <br/>
 <br/>
 这座教堂自从1882年开始动工，高耸的170公尺尖塔使它成为全世界最高的教堂。由高第（Antoni Gaudi）设计，包括18座塔楼，雕有耶稣、圣母玛利亚以及12位使徒，4位福音传教士。
 <br/>
 <br/>
 教堂东面的彩色玻璃以蓝绿色为基调，西面以红黄色为主。
 <br/>
 <br/>
 根据协议，教会将投资3600万欧元，部分资金用于建设教堂通道、扩大入口，预计2026年完工。
 <br/>
 <br/>
 法国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿尔卑斯山.html" target="_blank">
  阿尔卑斯山
 </a>
 有一座山村，今年四月因山体滑坡堵住了唯一的山路。不仅物资需要直升机运输，连汽车也需要从空中飞到公路上。当地人凑齐空运费用3000欧元的三分之二，其余一千由当地政府支付。
 <br/>
 <br/>
 居民表示他们能携带的任何东西都背在肩上。没法背的，只能等待直升机。冬天还没有取暖用的油，虽得到空运援助，但费用昂贵。
 <br/>
 <br/>
 10月末是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万圣节.html" target="_blank">
  万圣节
 </a>
 。最近几天，在希腊东北部的维斯托尼达（Vistonida）湖边，提前出现了恐怖景象。
 <br/>
 <br/>
 延伸几百米，树木和灌木被包裹在灰色蛛网中。专家说，当地的高温和湿度导致昆虫繁殖增加，吸引了蜘蛛。蜘蛛网将很多植物树木包裹。另一个希腊西部城镇也发生了相似现象。
</div>
