### 极端势力裹挟黄背心运动 名雕像被砸损毁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】法国抗议政府调涨燃料税的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄背心运动.html" target="_blank">
  黄背心运动
 </a>
 〞，已从最初的大规模聚集抗议行动演变为烧砸抢掠的暴力骚乱。不仅许多汽车被焚烧，商店被抢劫，而且连
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凯旋门.html" target="_blank">
  凯旋门
 </a>
 知名的玛丽安娜（Marianne）雕像的一只眼睛也被砸毁。
 <br/>
 <br/>
 法国从11月底开始发生大规模上街示威活动。抗议的人群在街头聚集数天后，演变为打砸抢烧的骚乱。
 <br/>
 <br/>
 当地时间12月1日，一群穿黑衣黄背心的蒙面暴徒冲入巴黎市中心的繁华地带，打砸或焚烧了数十辆汽车和街边建筑物。他们还砸破商店窗户，洗劫商店中的物品，甚至攻击警察。连法国的标志性建筑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凯旋门.html" target="_blank">
  凯旋门
 </a>
 亦遭纵火和打砸破坏，而法国共和国的象征——凯旋门博物馆入口处展出的玛丽安雕像一只眼睛亦遭暴徒砸烂。
 <br/>
 <br/>
 截至目前，至少有逾百人在骚乱中受伤，警方则逮捕了至少412人。法国当局已宣布进入紧急状态。
 <br/>

   </font>

   </font>
  </a>
  <br/>
 </center>
 <br/>
 刚参加完G20高峰会返回法国的马克宏12月2日召开了紧急会议，并任命总理菲利佩（Edouard Philippe）与示威代表及政党领袖进行沟通对话，以寻求解决分歧方案。
 <br/>
 <br/>
 马克宏还对公众发表了讲话，他呼吁抗议人士冷静，同时也表态法国政府不会接受抗议人士的暴力举动，会将采取了犯罪行为的暴徒绳之以法。
 <br/>
 <br/>
 法国内政部长卡斯塔纳则谴责那些试图〝制造混乱〞的人试图摧毁巴黎。他表示，这些〝暴徒〞具有高度的组织性并已渗透整场运动。
 <br/>
 <br/>
 法国内政部国务秘书努涅斯也表示，暴力行动其实都并非来自真正的〝黄背心〞人士，而是掺和进来的其他一些别有用心的人所为。
 <br/>
 <br/>
 警方则对外证实，抗议者中有极左派和极右派人士，他们很有经验，懂得如何与警方对抗。比如，他们知道如何烧车和躲避警方的催泪瓦斯等等。
 <br/>
 <br/>
 据联合报报导，在骚乱现场有极左派人士高喊反对资本主义的口号，并在商店墙壁上写下〝所有警察都是混蛋〞（All Cops are Bastards）的英文首字缩写。
 <br/>
 <br/>
 沟通会议将于3日开始进行，参加的人除了党内领导人外，还有巴黎市长及一些愿意和政府对话的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄背心运动.html" target="_blank">
  黄背心运动
 </a>
 抗议人士。
 <br/>
 <br/>
