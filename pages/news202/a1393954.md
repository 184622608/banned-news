### 10月3日全球看中国
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】下面是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球看中国.html" target="_blank">
  全球看中国
 </a>
 <br/>
 <br/>
 大纪元时报
 <br/>
 美国商务部周二宣布，对进口自中国的锻钢配件作出反倾销和反补贴的最终裁定，认定中国生产的锻钢配件以低于公平价值8%至142%的价格在美国销售。并认定中共正在向其锻钢配件生产商提供可诉讼的补贴，因而相关配件在美国最终倾销率为13.41%。
 <br/>
 <br/>
 纽约时报
 <br/>
 中共政府于9月28日向记者发送指令，提出六个需〝管控〞的经济选题。分别是：经济数据不及预期，地方政府债务风险成隐患，中美经贸摩擦影响逐渐显现，国内消费者信心指数下降，滞胀预期升温，借社会热点事件渲染民众生活艰难。
 <br/>
 <br/>
 美国之音
 <br/>
 五角大楼受白宫指令，正在调查美军在多大程度上依赖外国的军事物资。特别是如果中国切断相关供应或在零部件中安插破坏装置，美军将面临多大风险。一份即将出炉的报告会罗列各项风险并提出应对建议。
</div>
