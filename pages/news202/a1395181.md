### 美日欧联手改革WTO  美大使：无中间道路
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】目前，美、日、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 正在联手推动对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世贸组织.html" target="_blank">
  世贸组织
 </a>
 的全面改革，以因应中共贸易模式带来的挑战。周五，美国驻世贸大使谢伊在华府研讨会中更表示，在世贸组织改革问题上没有中间道路。
 <br/>
 <br/>
 周五，美国驻世贸大使谢伊（Dennis Shea）谈到，美国与日本、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 联手推动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世贸组织.html" target="_blank">
  世贸组织
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 ）改革的最新进展。
 <br/>
 <br/>
 美国驻世贸大使 Dennis Shea：〝美国、欧盟、日本正在技术官员层级，讨论这些新规则的制定，包括长期存在的产业补贴、国有企业，还有潜在的强制技术转移的规则。〞
 <br/>
 <br/>
 川普总统曾多次谈到，世贸组织如不改革，美国就将退出。贸易代表莱特希泽（Robert Lighthizer）也曾表示，世贸组织现有规则，无法应对中共统治下、非市场经济体的中国。今年7月，美国还向世贸组织大会提交报告，标题是〝中国贸易方式破坏经济模式〞。
 <br/>
 <br/>
 谢伊希望，今年底，美、日、欧盟能就世贸改革具体方案完成内部程序，明年初，再联络其它成员国，争取更多支持。
 <br/>
 <br/>
 美国驻世贸大使 Dennis Shea：〝这些国家想走中间路线，但实际上，他们必须选择一边。〞
 <br/>
 <br/>
 目前，负责世贸组织争端解决的上诉机构，只有三名法官，到2019年底，如果美国仍不同意选举新法官，上诉机构就将瘫痪。
 <br/>
 <br/>
 美国驻世贸大使 Dennis Shea：〝美国已提出自己的观点，观点也很清楚。我们不是只提出15个月，而是已经讲了15年。我们的建议是，上诉机构必须遵守各方在1994年达成的规定。〞
 <br/>
 <br/>
 11月底，川普将与习近平在20国峰会上会面。专家认为，这可能有助于缓解美中之间贸易紧张局势，但要真正解决问题，必须从根本上改变中国经济结构。
 <br/>
 <br/>
 战略与国际研究中心国贸项目主席 William Reisch：〝我们对中共在双边贸易上的一些要求，也同样是在世贸组织规则改革中要达成的。一切都会在日内瓦的世贸上汇合在一起。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
