### 美国安顾问：一特质决定川普G20不会为习而退让
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月28日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 近期陷入胶着状态，外界开始关注11月即将将在阿根廷举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上，川普（特朗普）与习近平的双边会谈，能否为这场贸易冲突找到新的转机。不过，美国一名前国安顾问日前在接受美国媒体采访时表示，川普非常善于与对手建立个人关系并热情以待，但这并不会影响他在美国贸易问题上保持强硬态度，他是一个公私分明的人。
 <br/>
 <br/>
 据《美国之音》报导，美国前国安顾问叶望辉（Stephen Yates）10月26日在华盛顿出席全球台湾研究中心的一场活动后接受了媒体的采访。
 <br/>
 <br/>
 当被问及美国总统川普与习近平主席的会面是否有助于解决美中贸易争端的问题时，他表示：川普总统非常擅长与对手建立个人关系，甚至可以相当热情，就如之前他他在新加坡与朝鲜领导人金正恩的互动，以及在海湖庄园与习近平会晤时所表现的那样。但是，即使在个人关系很好的情况下，川普在他希望看到有所改善的领域上会表现得〝非常直率〞，因此相信他在贸易问题的立场上应该还是会非常强硬，不会有所退让。
 <br/>
 <br/>
 叶望辉并分析称，估计川普与习近平在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 中的双边会谈〝首要重点是经济议题〞，不过，届时川普也可能会想到诸如南中国海的争端、自由航行，以及2艘美国战舰通过台湾海峡等问题，因为美国政府内部对这些议题都很担忧。
 <br/>
 <br/>
 叶望辉还特意提醒外界，除了关注川普与习近平的会谈之外，大家还应该关注美国副总统彭斯下个月即将在亚太经济合作会议APEC上与中方的互动，因为那将是美中之间〝另一个回合的对话〞。
 <br/>
 <br/>
 对于副总统彭斯最近发表的那个专门讨论美中关系的演说，叶望辉表示，他认为彭斯这演说中对美中关系在2018年的现在到底处于何种状态，作出了〝最正确的描绘〞。
 <br/>
 <br/>
 叶望辉说，当然，当今的美国政府所采取的对中政策与尼克松时代或卡特时代都有根本上的不同，但学识渊博、有经验的学者专家应该都会〝更专注于如何应对今日面对的挑战〞，而不是把心思放在如何〝守住过去那些不完整的文字〞上面。
 <br/>
 <br/>
 当被问及〝当今美中关系的状态是否代表一个新冷战的开始〞时，叶望辉说：自从中共建立政权以来，它就已经对美国展开冷战式的活动。这种情形就像美国政府在恐怖主义问题上与美国人民的对话一样，〝在我们决定该如何反应前，他们就已经先宣战了。〞他说：〝现在的问题只是在于，西方文明是否决定要做更有效的回应。〞
 <br/>
 <br/>
 事实上今年以来，川普曾多次公开表态称，他与习近平是好朋友，但中美之间这种不公平的贸易关系必须改变，这是两个问题，不能混为一谈。
 <br/>
 <br/>
 今年6月下旬，美国政府正式公布了对500亿美元中国进口商品加征关税的清单，并警告中方如果采取反制行动，将考虑再对高达2000亿美元的中国产品加征关税。其后不久又出台新的措施，禁止中资持有一定股权的企业投资美国科技公司，并收紧对美国技术出口的管控，禁止关键技术转移到中国。
 <br/>
 <br/>
 当时，就曾有亲北京的香港媒体发文评论称，当初川普当选美国总统后，北京以为像川普这样的〝商人总统〞，只要给他点甜头和面子，再与他和他的家人发展一点私人关系实施一些〝怀柔政策〞，就可让他在处理中美关系时对中方〝手软〞。但北京方面显然低估了川普奉行〝美国优先〞政策的决心和维护美国国家利益的意志。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
