### G20月底在阿根廷举行 美墨加30日签自贸协议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 经过紧张谈判，于9月30日宣布达成新的美加墨新贸易协议（USMCA），取代20多年前签署的北美自贸协定。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 经济部长伊尔德方索．瓜哈尔多周四（11月9日）表示，新贸易协议将于11月30日签署。
 <br/>
 <br/>
 据路透社报导，瓜哈尔多（Ildefonso Guajardo ）周四在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 城举行的一个活动上宣布，美墨加新贸易协议将于11月30日在阿根廷首都布宜诺斯艾利斯（Buenos Aires）签署。
 <br/>
 <br/>
 瓜哈尔多表示，目前尚未确定3国领导人是否会参加该协定的签署仪式。
 <br/>
 <br/>
 11月底至12月初，20国峰会（G20）也将在阿根廷举行。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统川普（特朗普）表示，他将会在和G20峰会期间与中国国家主席习近平一起用餐，并与他讨论贸易问题。
 <br/>
 <br/>
 <h4>
  新版
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨加协议.html" target="_blank">
   美墨加协议
  </a>
  有孤立中共意味
 </h4>
 <br/>
 《大纪元》报导，经过一年多的谈判，美国和墨西哥于今年8月首先达成了双边贸易协定，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 因与美国在几个关键问题上存在争议，未加入美墨协议。
 <br/>
 <br/>
 9月30日午夜，美国和加拿大在最后时刻达成一致，与墨西哥一起达成新版北美自贸协议，新协议更名为〝美国-墨西哥-加拿大协议〞。新协议在生效前，还还需要3国的国会议员批准。
 <br/>
 <br/>
 该协议被外界指有孤立中共的意味。协议规定，如果协议中的任何成员国与〝非市场经济国家〞达成自贸协定，其它成员国有权退出新协定。美国知名智库〝大西洋理事会〞（Atlantic Council）首席执行官肯普（Fred Kempe）认为，这里的〝非市场经济国家〞实则就是中共治下的中国。
 <br/>
 <br/>
 中共自2016年底以来一直大力要求欧美发达国家承认其市场经济地位，但由于这些国家极力反对中共的倾销政策，而公开反驳与拒绝承认其市场地位资格。
 <br/>
 <br/>
 外界认为，理论上讲，川普政府最新宣布的美墨加自贸协定将代替之前的北美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自由贸易.html" target="_blank">
  自由贸易
 </a>
 协议（NAFTA），但这次新增的第32.10条款使任何一个国家与中共缔结类似贸易协定时，三思而行，同时也可以阻止中国商品借道邻国进入美国。
 <br/>
 <br/>
 白宫贸易顾问纳瓦罗（Peter Navarro）10月2日在接受全国公共广播电台采访时也证实了这一点。他说，〝一小部分〞协议是针对中国（中共）的。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
