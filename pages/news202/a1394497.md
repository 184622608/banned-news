### 游曼谷遇枪战 AK47扫射酿至少2死5伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】泰国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曼谷.html" target="_blank">
  曼谷
 </a>
 披耶泰县一家知名商场附近周日（7日）晚间传出两派人马起争执，爆发械斗，并掏出大型枪械驳火，结果流弹击中在旁等车的旅客，造成至少2死5伤。死者据报是一名老挝(寮国)游客和一名印度游客。
 <br/>
 <br/>
 综合媒体报导，事发于周日晚间8时30分左右，两帮约20名青年在著名的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_水门商场.html" target="_blank">
  水门商场
 </a>
 （Watergate Pavillion Mall）附近一间桌球俱乐部起争执，随后，从停车场旁的巷弄肉搏演变成枪击。
 <br/>
 <br/>
 目击者忆述，两帮团伙其中3名男子，手持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_AK47步枪.html" target="_blank">
  AK47步枪
 </a>
 突然开火，结果流弹击中在旁等车的游客。
 <br/>
 <br/>
 警方据报赶到现场，两帮团伙即做〝鸟兽散〞逃跑，警方在现场检获大批9毫米和7.62毫米口径的子弹壳，包括AK47的弹壳。
 <br/>
 <br/>
 报导中说，死伤者几乎都是无辜的游客或当地人，其中刚好有一个印度旅行团刚在商场内用餐完毕，逛完街，到停车场等待搭乘旅游巴回到酒店，不料竟遇上枪战。
 <br/>
 <br/>

 <br/>
 死者包括一名28岁老挝游客托内胶（Keovongsa Thonekeo）和45岁印度人德拉杰（Gakhrejr Dheeraj）送院后伤重不治，另伤者包括2名泰国人、2名印度人和1名老挝人。
 <br/>
 <br/>
 目前暂未有嫌犯被捕。警方周一（8日）调阅出事现场监视器，正追查涉案人下落及涉案枪械详情。
