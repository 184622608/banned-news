### 【禁闻】马尔代夫或退出与中共的自贸协议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞计划在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南亚.html" target="_blank">
  南亚
 </a>
 的发展，可能面临变数。印度洋岛国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马尔代夫.html" target="_blank">
  马尔代夫
 </a>
 新政府声言，前政府与中共签署的自由贸易协议（FTA），是一个不能让马尔代夫受益的单向协议，马尔代夫要退出这一协议。分析认为，马尔代夫的亲中政策也可能面临结束。
 <br/>
 <br/>
 路透社11月19号刊登独家报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马尔代夫.html" target="_blank">
  马尔代夫
 </a>
 民主党领袖纳希德（Mohamed Nasheed）指出，马尔代夫新政府将退出与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的自由贸易协议（FTA），因为马、中两国达成这种协议是个错误。
 <br/>
 <br/>
 纳希德是马尔代夫新任总统索利赫（Ibrahim Mohamed Solih）的顾问，他也是马尔代夫前总统。他表示，跟中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自贸协议.html" target="_blank">
  自贸协议
 </a>
 虽然已经获得议会批准，但议会不会获得法律变更，让这项零关税协议生效。
 <br/>
 <br/>
 时事评论员田园博士表示，马尔代夫新政府首先是从经济上考虑。它看到了中共用高债务模式来推动发展〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞计划。
 <br/>
 <br/>
 时事评论员田园博士：〝中共一带一路虽然给东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南亚.html" target="_blank">
  南亚
 </a>
 一些国家巨额投资，但最终的结果是使这些国家债务危机爆发，像巴基斯坦，像斯里兰卡，最后斯里兰卡不得不采取几乎相当于是割地赔款的方式，来偿还中共的债务。〞
 <br/>
 <br/>
 纳希德还形容，跟中共的自贸协议是〝一个单向条约〞，〝中国不从我们这里购买任何东西〞。
 <br/>
 <br/>
 田园：〝因为马尔代夫是一个1200个珊瑚岛组成的国家，它国内产出的东西，可以说是微乎其微。它跟中共签了这个零关税协议之后，只能是被中共的廉价产品，把国内仅剩一点点的民族工业全部摧毁。所以这对马尔代夫来说，是一个灭顶之灾。在几年之内这个国家的经济命脉可能将会被中共完全掌控。〞
 <br/>
 <br/>
 田园表示，从地缘政治上看，这可能是马尔代夫倒向印太战略的一步。预料马尔代夫会加强自己在印太联盟战略里的地位，最终可能会成为印太联盟的盟友。
 <br/>
 <br/>
 田园：〝一直以为马尔代夫是处于印度的势力范围之内，可是马尔代夫即将离任的这一届政府，却倒向了中共，造成了印度对这个非常愤怒。那么在新政府上台之后，审时度势，他看到了印度在美国川普总统所倡导的印太新战略里面，是一个非常重要的组成部分。中国对马尔代夫（来讲）是八竿子打不着的地方，可是印太联盟就在家门口，而且还有美国和澳大利亚这样的后盾。〞
 <br/>
 <br/>
 位于印度洋的马尔代夫，在国际远洋运输中具有特殊的战略地位，是中印两国近年争夺影响力的对象。
 <br/>
 <br/>
 马尔代夫是最早响应中共〝一带一路〞倡议的国家之一，它在2014年12月加入。
 <br/>
 <br/>
 前任总统亚明（Abdulla Yameen）在去年12月访华时，与中共签署了马尔代夫第一个对外双边自贸协议。当月底，议会在一个小时之内就匆忙通过了这份极具争议的自贸协议。
 <br/>
 <br/>
 今年9月的总统大选，来自马尔代夫民主党的索利赫意外击败前总统亚明。
 <br/>
 <br/>
 和亚明的亲中立场有所不同，索利赫的立场明显亲印度和西方。这场选举也被认为是中、印之间的一场角力。
 <br/>
 <br/>
 因此，外界预计索利赫上台后，马尔代夫的亲中政策将会结束。
 <br/>
 <br/>
 北京方面希望马尔代夫不要退出自贸协议。中共外交部周二回应，相信马尔代夫政府会做出所谓的正确选择。
 <br/>
 <br/>
 但能否如北京所愿，外界并不看好。
 <br/>
 <br/>
 在刚刚结束的亚太经合组织（APEC）峰会上，美国副总统彭斯警告小国不要被中共诱惑，他还将〝一带一路〞称作〝压缩带〞和〝单行路〞。
 <br/>
 <br/>
 北京时政观察人士华颇：〝中国（中共）以前一直和（马尔代夫）前总统关系非常不错。0419印度可能也有大笔资金的投入。所以马尔代夫这次和中共翻脸，我估计有美、印联手抵消中国（中共）扩张的手段。〞
 <br/>
 <br/>
 索利赫上周六在首都马里市发表就职演说时指出，由于前任政府遗留下来的巨额中国债务，目前国库被洗劫一空。他表示，马尔代夫欠中国15亿美元巨额贷款。这相当于马尔代夫国民生产总值的四分之一。
 <br/>
 <br/>
 印度总理莫迪当天也出席了索利赫的就职仪式，并承诺向马尔代夫伸出援手。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/周天
