### 克里米亚校园枪击案 民众悼念遇难者
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】乌克兰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克里米亚.html" target="_blank">
  克里米亚
 </a>
 克赤市一所技术学院，周三发生重大枪击案，目前已有19人死亡、50人受伤，嫌犯自杀。周四，当地民众前往案发地点，悼念遇难者。
 <br/>
 <br/>
 周四，克赤市居民手捧鲜花，来到案发地点。点燃蜡烛，悼念遇害者。
 <br/>
 <br/>
 克赤市居民：〝保安干什么去了？大量（保安）人员在哪？人们会问：为什么出这种事？孩子们近距离被枪杀？〞
 <br/>
 <br/>
 警方公布，嫌犯名叫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗斯利亚科夫.html" target="_blank">
  罗斯利亚科夫
 </a>
 ，18岁，是克赤市技术学院四年级学生。
 <br/>
 <br/>
 周三，他持枪在校园内扫射。警方随后在校园内发现了他的尸体，可能是自杀。校园餐厅还发现一枚炸弹，随后被安全解除。
 <br/>
 <br/>
 监控录像显示，10月13日，嫌犯购买了150发猎枪子弹。
 <br/>
 <br/>
 乌克兰总统波罗申科周三表示，枪击案是一起〝悲剧〞。
 <br/>
 <br/>
 乌克兰总统波罗申科：〝对我们来说（这件事）为什么重要？因为这些是乌克兰公民。当然，当乌克兰公民遇害，不管发生在哪里，都是悲剧。〞
 <br/>
 <br/>
 目前还不清楚嫌犯作案动机，但已确认是单独作案，定性为〝大规模谋杀〞。俄国政府已着手调查。
 <br/>
 <br/>
 新唐人记者毕心慈、严曦综合报导
</div>
