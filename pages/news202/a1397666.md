### 土耳其检察官：卡舒吉被勒死后肢解
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 首席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_检察官.html" target="_blank">
  检察官
 </a>
 周三发表声明，沙特记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 进入沙特领事馆后被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_勒死.html" target="_blank">
  勒死
 </a>
 ，尸体随后被〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肢解.html" target="_blank">
  肢解
 </a>
 〞。卡舒吉未婚妻表示，卡舒吉在进入领馆前就担心会被抓回沙特。卡舒吉尸体仍没有下落。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 司法部部长哈米特・居尔再次呼吁沙特合作调查。
 <br/>
 <br/>
 沙特最高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_检察官.html" target="_blank">
  检察官
 </a>
 周二结束在伊斯坦布尔的调查，期间未能说明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 遗体的下落，以及是谁下令杀人。
 <br/>
 <br/>
 土耳其当局发表书面声明表示，会议没有取得具体进展，呼吁沙特与土耳其密切合作，揭露事件真相。
 <br/>
 <br/>
 卡舒吉的未婚妻哈蒂丝周二接受采访时表示，卡舒吉进入领馆前曾担心被抓回沙特。
 <br/>
 <br/>
 卡舒吉未婚妻哈蒂丝：〝是的，他担心会被抓捕送回沙特阿拉伯。〞
 <br/>
 <br/>
 哈蒂丝还说，她一直希望卡舒吉还活着，直到沙特承认他已被杀害。
 <br/>
 <br/>
 卡舒吉未婚妻哈蒂丝：〝我想至始至终都相信他还活着。直到沙特承认他们杀了他，我才认清发生了这场可怕悲剧。〞
 <br/>
 <br/>
 美国总统川普不断关注调查进展，记者向川普提问是否觉得受到沙特背叛。
 <br/>
 <br/>
 美国总统川普：〝没有背叛我，不。我只希望一切顺利。我们有很多事实。我们有很多我们一直关注的事情。他们没有背叛我。我的意思是，也许他们背叛了自己。我们必须再看看情况如何。〞
 <br/>
 <br/>
 新唐人记者李梅、宇微综合报导
</div>
