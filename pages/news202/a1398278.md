### 94岁纳粹集中营守卫 出庭受审
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】周二，德国94岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳粹.html" target="_blank">
  纳粹
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集中营.html" target="_blank">
  集中营
 </a>
 嫌犯，在明斯特地方法院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出庭受审.html" target="_blank">
  出庭受审
 </a>
 。他被控犯有100项协助谋杀罪。由于被告犯罪时不满21岁，因此在青少年法庭进行庭审。
 <br/>
 <br/>
 94岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳粹.html" target="_blank">
  纳粹
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集中营.html" target="_blank">
  集中营
 </a>
 嫌犯坐着轮椅进入法庭，根据德国法律，还不能公开他的姓名和面貌。
 <br/>
 <br/>
 美国、加拿大、以色列和波兰等国的12名受害者家属，也赶来参加庭审。
 <br/>
 <br/>
 共同原告班・柯恩是大屠杀幸存者茱蒂・梅索的孙子。茱蒂・梅索的母亲在施图特霍夫集中营毒气室被杀害。
 <br/>
 <br/>
 大屠杀幸存者后代班・柯恩：〝她能够见证，尽管只是从远方目睹一部分过程，也是一种了结。让德国人倾听她的故事，对她来说非常有力。〞
 <br/>
 <br/>
 1942到1944年，嫌犯在纳粹的施图特霍夫集中营担任看守，期间约有6万5千人被杀害。嫌犯被控犯有100项协助谋杀罪。
 <br/>
 <br/>
 高级检察官ANDREAS BRENDEL：〝嫌犯被控是数百起谋杀案的同谋，因为他在施图特霍夫集中营担任看守时，人们被毒气杀死、被从脑后枪击死亡、挨饿冻死。〞
 <br/>
 <br/>
 由于被告健康状况不佳，庭审每天最多持续两小时。
 <br/>
 <br/>
 共同原告律师ONUR U. OEZATA:〝对他们（原告）来说，这不是一位94岁的老人要坐牢和受苦的问题，而是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伸张正义.html" target="_blank">
  伸张正义
 </a>
 ，即使只是在很小的方面。〞
 <br/>
 <br/>
 办案人员在审查施图特霍夫集中营文件时，发现了嫌犯，他也承认曾在该集中营任职。
 <br/>
 <br/>
 新唐人记者李梅、天羽综合报导
</div>
