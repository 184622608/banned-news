### 【禁闻】美将立国际新秩序 抗中俄滥用国际组织
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】为了阻止伊朗、俄罗斯和中共不遵守国际协定而获利，并且避免战争和促进繁荣，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 将建立自由民主的新世界秩序。美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 在布鲁塞尔发表演说，敦促美国的盟国一起对世界贸易组织、联合国、国际货币基金组织等机构进行诚实的评估。
 <br/>
 <br/>
 12月4号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 （Mike Pompeo）在布鲁塞尔发表外交政策演说中表示，遵循我们伟大民主的优良传统，我们正在号召各国建立新的自由秩序，以避免战争和促进繁荣。
 <br/>
 <br/>
 蓬佩奥说，〝为了维护、捍卫和促进世界的开放、公正、透明和自由。这项计画必须切实，而不是假装恢复各国间的自由秩序。这需要坚定自信的美国，也需要全球民主国家领导阶层共同努力。〞
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝我们的使命是重申我们的主权，改革自由国际秩序，希望我们的朋友予以协助，也发挥他们主权的影响力。希望这个国际秩序服务于我们的公民，而不是控制他们。〞
 <br/>
 <br/>
 蓬佩奥这篇〝改革基于规则的国际秩序〞讲话，表示川普政府不再愿意接受中共、伊朗和俄罗斯违反多项国际条约和多边协议的做法，美国将采取行动，改革二战后形成国际秩序基础的那些机构。例如联合国、欧洲联盟（EU）、非洲联盟（African Union）、国际货币基金和世界银行等组织，都急需改革创新。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝我们退出巴黎协定，因为其中的条款对美国不公平。现有的条款是从美国那里抽钱，补贴中国这样污染严重的国家。〞
 <br/>
 <br/>
 蓬佩奥还指责中共利用世界贸易组织的规则，不公平的发展自身的经济利益。美国正在敦促世界银行（World Bank）和国际货币基金（IMF）减少对中共等国的资助，因为这些国家已经能进入金融市场筹集资本。
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝如果说一个经济体它已经膨胀到了世界第二的规模，但是这个经济体它却从来不遵守最基本的正常的贸易规则。就是说，这个经济体是一个扭曲的经济体，是一个畸形的经济体，它就必然会对全世界的这个经济体系带来严重的冲击。〞
 <br/>
 <br/>
 旅美时事评论员唐靖远表示，中共利用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际组织.html" target="_blank">
  国际组织
 </a>
 满足自己的政治目地，达到对外扩张和输出影响力，已经持续很长时间了。
 <br/>
 <br/>
 唐靖远：〝利用政治上的这种威胁，加上经济上的收买或者是通过种种就是不法的手段来拉帮结伙，在整个联合国里面就是捆绑一部分，尤其是大量的都是这种第三世界的一些这个小国，然后来取得在票数上面的一些优势。〞
 <br/>
 <br/>
 评论认为，彭佩奥这次的讲话内容，已经对中共释放非常清楚的信号，也是非常严厉的警告。
 <br/>
 <br/>
 唐靖远：〝我们看到彭佩奥的讲话，他其实和美国副总统彭斯在这个哈得逊研究所的讲话，以及在APEC会议上面的这个公开发表的演讲等等，这个核心的脉络，这个核心的内容，其实是一致的。〞
 <br/>
 <br/>
 旅美时事评论员郑浩昌也表示，彭佩奥的演讲和稍早之前美国副总统彭斯的演讲，可以看作是姐妹篇，他们传达的都是川普政府决心改变整个已经腐烂了的世界格局。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝彭斯演讲针对的是中共这个蛇头，彭佩奥针对的是整条蛇，这条蛇就不仅仅是几个国家，还包括了其他几个病泱泱的国际组织，所以无疑是一场正邪大战。〞
 <br/>
 <br/>
 彭斯11月17号在亚太经济合作会议（APEC）企业领袖高峰会发表演说，批评中共的〝一带一路〞不仅损害那些与中国合作项目的国家主权，更带来当地的腐败与高额债务。彭斯强调，中共如果不改变，美国绝对不会松手。
 <br/>
 <br/>
 有评论指出，中共这样的专制政权，最后只能得到一个结果，那就是越来越孤立，越来越被排斥在国际社会之外。
 <br/>
 <br/>
 采访/常春 编辑/黄亿美
