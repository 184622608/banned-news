### 获释牧师布伦森受专访 谈及土耳其遭遇
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】曾在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 传教23年的美籍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牧师布伦森.html" target="_blank">
  牧师布伦森
 </a>
 ，被控犯有恐怖主义和间谍罪关押约两年，上周五被释放，周二，这位牧师面对媒体吐露心声。
 <br/>
 <br/>
 美籍牧师Andrew Brunson：〝我们去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 的目的是告诉人们耶稣基督。我们非常公开地这样做。我们没有参与任何政治活动。〞
 <br/>
 <br/>
 2016年，他遭到逮捕被粗暴对待，被关在20人的拥挤牢房中。
 <br/>
 <br/>
 美籍牧师Andrew Brunson：〝我被关在最多八人的牢房内，但那里装了20多个人。它非常拥挤，非常孤单。〞
 <br/>
 <br/>
 他被指称和非法恐怖主义库尔德工人党（PKK）及2016年未遂的政变有关联。
 <br/>
 <br/>
 美籍牧师Andrew Brunson：〝例如，说我支持库尔德工人党，这是一个库尔德恐怖主义活动。这样当然，很多土耳其人（对我）变得更愤恨。〞
 <br/>
 <br/>
 美国政府坚称布伦森无罪。布伦森牧师一案在美土关系的危机中成为一大聚焦点。川普总统今年8月对土耳其罚以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，部分原因是因为土耳其不放弃起诉布伦森。在美国加征关税后，土耳其货币大跌。上周五，土耳其法院释放了布伦森。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
