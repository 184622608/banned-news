### 被灭口？普京心腹坠机亡 飞行员遗体有弹孔
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】俄罗斯副检察总长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡拉佩特杨.html" target="_blank">
  卡拉佩特杨
 </a>
 近日搭乘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_直升机.html" target="_blank">
  直升机
 </a>
 去打猎时，飞机坠毁，机上4人全数身亡。有知名记者爆料，直升机飞行员身上有弹孔，加上卡拉佩特杨生前调查多起外国暗杀案，外界怀疑他或被杀人灭口。但俄当局否认机上有人中弹。
 <br/>
 <br/>
 58岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡拉佩特杨.html" target="_blank">
  卡拉佩特杨
 </a>
 （Saak Karapetyan）是前国会议员，同时是总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 的得力助手，他上周三（3日）与侄儿乘搭的AS-350
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_直升机.html" target="_blank">
  直升机
 </a>
 去打猎，结果飞机在科斯特罗马（Kostroma）坠毁，卡拉佩特杨与侄儿、直升机飞行员，及另一名乘客全部丧生。
 <br/>
 <br/>
 空难发生后，俄国官方声称，是直升机〝超载〞导致飞机撞树坠机，但俄国知名记者多伦科（Sergey Dorenko）在《莫斯科之声》电台（Govorit Moskva）上爆料：〝飞行员身上有两个5.45毫米口径的子弹孔…直升机的螺旋桨也有子弹痕。〞批评当局隐瞒实际原因，称54岁的直升机飞行员经验丰富，他是在坠机前被枪杀。
 <br/>
 <br/>
 此外，坠机后也有匿名人士在网上发表评论称，卡拉佩特杨的死〝将会解决俄罗斯当局内部几件恼人的事件。〞
 <br/>
 <br/>
 不过，俄国调查委员会否认了该说法，声称所有死难者的身上都没有子弹孔。
 <br/>
 <br/>
 卡拉佩特杨生前曾主导调查多起国际瞩目的案件，包括：俄罗斯前双面间谍斯克里帕尔（Sergei Skripal）与女儿尤莉亚（Yulia Skripal）今年3月在伦敦身中神经毒剂诺维乔克（Novichok）案；2013年普京政敌、俄富商比瑞索夫斯基（Boris Berezovsky）在伦敦自家的上锁浴室中上吊身亡；2006年向英国投诚的俄罗斯前特务李维宁科（Alexander Litvinenko）遭人以放射性元素钋210毒死等案。
 <br/>
 <br/>
 俄国有评论称，卡拉佩特杨负责调查这三起案件，目的是想掩饰莫斯科当局参与的证据。另外还有人声称，他可能是向英国，美国和瑞士秘密机构洩密的管道。
