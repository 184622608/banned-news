### 联手应对中共  美日启动双边贸易谈判
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】周二(10月16日)，就在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 通知国会，有意与日本展开双边贸易协定谈判之际，美国国务院官员表示，谈判也将包括第三国市场扭曲行为，并希望安倍首相访中时，也向中方提出这些美日共同关切议题。
 <br/>
 <br/>
 美日领导人9月联大期间，同意展开双边贸易协定谈判，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 已在本周二通知国会。
 <br/>
 <br/>
 美国国务院日本科科长张茱莉：〝不只包括货物协定，确实还包括服务，并将包括贸易和投资问题。也包括第三国贸易行为，即我们所谈到的各种市场扭曲行为。〞
 <br/>
 <br/>
 战略与国际研究中心副总裁 Michael Green：〝日本方面称这是贸易和货物协定，即TAG，但他们不提服务和其它议题。另外，谈判的范围和程度还是有些模糊。〞
 <br/>
 <br/>
 上月联大期间，美日欧盟曾发表联合声明，一致反对不公正贸易行为，明显剑指
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 。本月初，美墨加贸易协定达成，其中纳入限制和非市场经济体交易的条款。专家认为，美国会敦促日本在未来贸易协定中加入这一所谓〝毒丸条款〞，与美国联手，向中共施压。
 <br/>
 <br/>
 日本首相安倍晋三下周将出访中国。对于日本是否将在美中贸易关系紧张之际扮演调停角色，张茱莉表示，并未听到这一说法。
 <br/>
 <br/>
 美国国务院日本科科长张茱莉：〝我们是双边性地处理我们在贸易和投资上的忧虑。不过当然，美国和日本有很多共享价值，无论是在WTO，还是APEC。〞
 <br/>
 <br/>
 张茱莉强调，针对知识产权窃取、强制技术转让，国有企业及政府补贴等不公平贸易行为，美日两国有共同立场。
 <br/>
 <br/>
 美国国务院日本科科长张茱莉：〝我们共享这些在繁荣经济体中仰赖的经济支柱。如果安倍首相与中共领导人讨论这些，我们当然欢迎。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
