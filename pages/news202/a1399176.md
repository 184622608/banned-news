### 东盟峰会新加坡开幕 美国俄罗斯都出席
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】第33届东南亚国家联盟峰会，周二在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新加坡.html" target="_blank">
  新加坡
 </a>
 正式登场，东盟十国、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 、中国等多国领导人出席，新加坡出动了几千名保安人员。
 <br/>
 <br/>
 第33届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东盟峰会.html" target="_blank">
  东盟峰会
 </a>
 周二开幕，轮值主席国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新加坡.html" target="_blank">
  新加坡
 </a>
 呼吁东盟团结合作，推进区内整合。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 由副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 代表川普总统出席会议，外界关注彭斯是否会就印太战略发表演说，他是否将与中共总理李克强会面也受到关注。
 <br/>
 <br/>
 东盟国防部长会议上个月宣布，明年将与美国举行联合海事演习。
 <br/>
 <br/>
 美国强调南海航行自由攸关国家利益，会例行性在南海执行自由航行任务。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 总统普京，则罕见首次访问新加坡并参加东盟峰会，外界分析，这是俄罗斯想加强与该区域国家的关系。
 <br/>
 <br/>
 新唐人记者李梅综合报导
</div>
