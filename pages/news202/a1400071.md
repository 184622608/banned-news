### 韩国统一部：朝鲜只有完全弃核才能发展经济
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】韩国统一部长官赵明均周二（20日）在博鼇亚洲论坛上表示，朝鲜（北韩）所期望的经济发展离不开国际社会的帮助。如果朝鲜
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非核化.html" target="_blank">
  非核化
 </a>
 没有进展，就无法达成其经济发展目标。
 <br/>
 <br/>
 韩联社报导，赵明均（Cho Myung-Gyun）周二在首尔召开的〝博鳌论坛2018首尔会议〞中表示，朝鲜领导人金正恩正在视察鼓励全国各地开展经济建设，朝鲜官员也一直在各种场合强调发展经济和提高人民生活水准。
 <br/>
 <br/>
 赵明均说，建立
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜半岛.html" target="_blank">
  朝鲜半岛
 </a>
 和平机制将为全力发展经济的朝鲜开启新的大门。韩国政府一直积极向朝鲜说明这一点。
 <br/>
 <br/>
 赵明均还提及，完全
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非核化.html" target="_blank">
  非核化
 </a>
 是韩国政府的坚定目标，政府一直尊重并遵守国际制裁框架，在今后的非核化过程和韩朝经济合作实践中，也将继续与国际社会保持密切合作。
 <br/>
 <br/>
 另外，韩国外交部19日表示，外交部半岛和平交涉本部长李度勋当天上午启程赴美，将与美国国务院对朝政策特别代表斯蒂芬・比根举行朝核六方会谈韩美团长会议。
 <br/>
 <br/>
 双方将对进来朝鲜半岛局势进行分析，交流对朝谈判战略。双方还会就韩美在实现半岛无核化、构建半岛和平机制等方面的合作交换意见。同时，双方将就可能在本月底举行的朝美高级别会谈推进情况进行沟通。美朝高级别会谈原定于本月8日举行，但被推延。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
