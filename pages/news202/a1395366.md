### 千余洪都拉斯人涌向边境 美国再临移民危机
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】日前又有上千名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 人奔向美墨边界企图
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 ，这是数日前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 副总统彭斯（Mike Pence）敦促拉美几国领导人对大规模移民采取强硬立场后，出现的又一股非法移民浪潮。
 <br/>
 <br/>
 据路透社10月13日报道，周六，至少有1,300人离开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 北部的圣佩德罗苏拉（San Pedro Sula）开始了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 行军〞（March of the Migrant），其中包括许多年幼的孩子。
 <br/>
 <br/>
 35岁的范妮．巴拉奥纳（Fanny Barahona）说，她相信他们会抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 。她抱怨在洪都拉斯找不到工作，而且还要在自己或孩子说不定哪天被杀的恐惧中生活。
 <br/>
 <br/>
 巴拉奥纳是一名失业教师，他带着9岁的儿子和仅仅2岁的女儿同行。
 <br/>
 <br/>
 25岁的索利斯（Javier Solis）也表示，他在洪都拉斯已失业一年，而且甚至在街上走路都有可能遭到杀害，所以，他不惜冒险进入美国。在此前的尝试中，他曾抵达墨西哥，但后来遭到遣返。
 <br/>
 <br/>
 目前，超过64％的洪都拉斯家庭都生活在贫困之中，圣佩德罗苏拉还是世界上谋杀率最高的地区之一。
 <br/>
 <br/>
 据移民大军的组织者富恩特斯（Bartolo Fuentes）说，大约有1300人参加了所谓的〝移民行军〞，他们计划从圣佩德罗苏拉穿越危地马拉，然后再进入墨西哥。参与者将申请难民身份以留在墨西哥，或申请签证进入美国。
 <br/>
 <br/>
 今年4月下旬，一度有大约600名〝大篷车移民〞涌向美墨边境的墨西哥小城蒂华纳（Tijuana）。虽然鉴于川普和墨西哥当局的压力，这群移民的人数已从高峰时的1500人逐渐递减，但仍有一些人成功跨越边界并向美国申请庇护。
 <br/>
 <br/>
 川普政府随后采取了严厉措施，以经济惩罚的方式，促使有关国家控制自己的公民大规模外逃。
 <br/>
 <br/>
 川普总统5月23日宣布，美国将扣除一大笔给这些非法移民来源国的援助。
 <br/>
 <br/>
 上周四，在华盛顿举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲.html" target="_blank">
  中美洲
 </a>
 繁荣与安全会议（Prosperity and Security in Central America）上，美国副总统彭斯会见了洪都拉斯、危地马拉和萨尔瓦多等国的总统，并告诉他们，如果他们能在解决移民、腐败和帮派暴力问题等方面做出更多，美国愿意帮助其发展经济并向其投资。
 <br/>
 <br/>
 彭斯说，来自洪都拉斯和危地马拉的非法移民已分别增长了61％和75％。
 <br/>
 <br/>
 去年，川普政府建议向中美洲国家提供4.6亿美元的援助，比国会2016年批给前总统奥巴马（Barack Obama）的数额减少了30％。
 <br/>
