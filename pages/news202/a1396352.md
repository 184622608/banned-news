### 白宫温和派转强硬 知情人:川习会仅为重温私人关系
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】美国总统川普（特朗普）与中国国家主席习近平有望在G20峰会上进行双方会谈，不过，外界对这一次的川习会能否缓和中美贸易冲突并不乐观。外界注意到，被外界视为白宫〝鸽派〞官员的财长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姆钦.html" target="_blank">
  姆钦
 </a>
 （Steven Mnuchin）日前发出的言论也开始变强硬，有港媒发文分析称，就算川习会能够实现，恐怕也难以解开〝中美死结〞。
 <br/>
 <br/>
 美国财政部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姆钦.html" target="_blank">
  姆钦
 </a>
 是美国政府内力主通过与中方谈判来解决中美贸易冲突的官员，被外界视为白宫内阁中少有的几名〝鸽派〞官员之一。
 <br/>
 <br/>
 姆钦最近出访中东在耶路撒冷接受媒体访问。路透社记者现场提问：中国今年第3季度的GDP数据下降，会否忧虑中国经济下滑可能蔓延至其他新兴市场、并造成美国金融市场不稳定？
 <br/>
 <br/>
 〝我不担忧它将破坏美国市场的稳定〞，〝总体而言，目前我没有看到（中国经济增长放缓）蔓延的风险〞。
 <br/>
 <br/>
 姆钦当时回答称：〝我不担心它（中国经济下滑）会破坏我们市场的稳定。〞他接着进一步表示，〝从大面上来看，目前我并未看到蔓延的风险。〞
 <br/>
 <br/>
 中共国家统计局上周五（19日）对外公布的数据显示，中国经济今年第三季度GDP增长放缓到6.5%，这是自从2009年全球金融危机以来最慢的季度GDP增长。同时，中国大陆近半年来股市大幅抛售、人民币急剧贬值，外界普遍认为，这些状况都与中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 息息相关。
 <br/>
 <br/>
 如今，一贯主张与中方谈判化解贸易战风险的姆钦的公开言论明确转为强硬，引发了外界的关注与解读。
 <br/>
 <br/>
 立场比较亲中共的《香港经济日报》日前发文分析称：有舆论认为美中贸易战倘伤及中国经济，美国难独善其身，这或许促使美方在贸易战〝收手〞，但姆钦在以耶路撒冷直言〝不怕中国经济下滑波及美国〞。此外，姆钦还曾透露美方可能改变判定一国滙率操纵的标准，美方这样做的矛头也被认为是直指中国（中共政府）。姆钦这一系列的表态或意味着，在贸易冲突上美方〝没有让步空间〞，美国政府将持续〝出手〞，直到中方作出让美方满意的承诺。
 <br/>
 <br/>
 这篇文章还援引观察人士表示：美国政要近日对外展示的强硬姿态，一方面可能是美方想在这次G20峰会上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会晤.html" target="_blank">
  川习会晤
 </a>
 前〝提高自己叫价〞；但另一方面也反映出，美中之间的贸易分歧可能仍然南辕北辙，对于川习会面能否缓减中美贸易冲突〝实在不能乐观〞。
 <br/>
 <br/>
 文章又引述观点称：中美之间的对抗正从经贸逐渐蔓延到地缘、军事、乃至意识形态领域，因此两方要在短期内就达成贸易和解〝恐怕难上添难〞。
 <br/>
 <br/>
 事实上，日前美国新闻网站Axios也在一篇报导中引述知情人士透露称，川普总统无意放宽对中国商品的关税，相反，他希望中共能够从关税中感受到更大的压力，还需要更多的时间，关税战给中方带来的苦果才能逐渐显露出来。该消息人士表示，川普认为〝关税持续时间越长，他就会拥有更多的（谈判）筹码。 〞而川普的团队对总统与习近平下个月在G20峰会期间的会晤也并不抱太大期望，他们认为G20峰会是国家元首会议，而不是贸易会议。
 <br/>
 <br/>
 知情人士还告诉Axios称：〝川普考虑将此次会议作为与习主席的私人关系重温，而非详细地讨论（贸易）〞。
 <br/>
 <br/>
 针对中美贸易战的问题，美国财政部长姆钦（Steven Mnuchin）的团队近日也曾对外放话称，中共没有必要把购买美国产品作为一枚〝关键棋子〞，因为美国政府关注的不只是销售产品，而是中方必须解决知识产权盗窃和市场准入等结构性问题。
 <br/>
 <br/>
 对此，时政评论员唐靖远表示：中美之间的冲突从实质上来看是体制冲突，只要中共的体制不做实质性的改变，川习会面也谈不拢。中共现在的策略几乎就是抱定了死守城池的态度，继续用拖延战术来应对美方的要求。
 <br/>
 <br/>
 （记者唐迪综合报导/责任编辑：明轩）
