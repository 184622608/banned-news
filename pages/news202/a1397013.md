### 斯里兰卡陷宪政危机 前强人总统突任新总理
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯里兰卡.html" target="_blank">
  斯里兰卡
 </a>
 总统席瑞塞纳26日突然任命前强人总统拉贾帕克萨出任新总理，现任总理威克瑞米辛赫则遭解职。据了解，在席瑞塞纳撤换总理前，他与威克瑞米辛赫在经济政策和日常政府治理上出现严重分歧，而这起政治任命案让斯里兰卡陷入前所未有的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宪政危机.html" target="_blank">
  宪政危机
 </a>
 。
 <br/>
 <br/>
 依据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯里兰卡.html" target="_blank">
  斯里兰卡
 </a>
 宪法，总统有权指定拥有国会多数的人士担任总理。但在2015年通过的第19号修正案补充规定，现任总理只有在失去国会议员资格（诸如不符合选举人或候选人资格、遭到不信任投票或自己请辞）时，才可加以撤换。
 <br/>
 <br/>
 尽管反对派人士以根据宪法抗议席瑞塞纳（Maithripala Sirisena）没有权力这么做，但总统办公室仍宣布解除现任总理威克瑞米辛赫（Ranil Wickremesinghe）的人事决定。
 <br/>
 <br/>
 威克瑞米辛赫仍坚称是合法总理，他在全国直播的记者会向记者表示：〝我以斯里兰卡总理身分向你们致辞。我仍然是总理，我会继续担任总理。〞
 <br/>
 <br/>
 威克瑞米辛赫表示，他要求国会议长贾雅苏瑞亚（Karu Jayasuriya）召集国会议员开会。
 <br/>
 <br/>
 <center>

   </font>
  </a>
 </center>
 <br/>
 贾雅苏瑞亚则回应道，他正在征询法律意见，假如他承认拉贾帕克萨为新总理，将于明天宣布这项消息。
 <br/>
 <br/>
 与此同时，拉贾帕克萨（Mahinda Rajapakse）则在总统办公室宣誓就职总理，并呼吁威克瑞米辛赫下台说，联合国家党（United National Party）党员必须〝尊重民主、尊重国家、尊重法律〞。
 <br/>
 <br/>
 而斯里兰卡全国和平会议(National Peace Council of Sri Lanka)执行董事佩瑞拉(Jehan Perera)指出，斯里兰卡现在已经陷入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宪政危机.html" target="_blank">
  宪政危机
 </a>
 。要以民主的方法化解这次的危机，总统应该立即召集国会开议，重新选举一位总理。
 <br/>
 <br/>
 据了解，斯里兰卡政府原本由席瑞塞纳所属统一人民自由联盟（United People's Freedom Alliance）及威克瑞米辛赫所属联合国家党的联盟共同执政，但统一人民自由联盟26日稍早退出联合政府。
 <br/>
 <br/>
 外界分析，拉贾帕克萨以总理之姿重新掌权，意味着斯里兰卡将走回以暴力对付政治异议者和记者的老路。

 <br/>
 <h4>
  庞大债务难逃中共掌心
 </h4>
 <br/>
 据纽约时报此前报导，中共藉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 计划全球投资，令斯里兰卡身陷债务危机，不得不将〝不通往任何地方的港口〞赫班托达港（Hambantota Port）租借给中共。
 <br/>
 <br/>
 报导说，赫班托达地区人口稀少，是贾帕克萨的故乡，藉着大举开发赫班托达，不少来自中国的贷款流进拉贾帕克萨及其竞选团队的帐户，同时承诺让中国公司承包多数工程，付给高额的工程款与工人薪资。
 <br/>
 <br/>
 报导中说，2015年总统大选，至少有760万美元从中国港湾的渣打银行帐户汇往拉贾帕克萨竞选活动的分支机构，但腐败的内政仍让拉贾帕克萨下台。
 <br/>
 <br/>
 席瑞塞纳上任后虽然急于让斯里兰卡重新靠拢印度、日本和西方，但庞大的债务让斯里兰卡无法脱离中共的手掌心。
 <br/>
 <br/>
 在拉贾帕克萨离任前，该国的债务增加3倍达到448亿美元。仅2015年就有46.8亿美元的贷款到期。
 <br/>
 <br/>
 斯里兰卡最后与中共签约，等同放弃赫班托达港，但周边国家担忧，最终协议禁止外国将该港口用作军事目的，除非获得当地政府的许可。而这个条款之所以存在，是因为中共海军潜艇已经可以在斯里兰卡停靠。但还欠中共一大笔钱的斯里兰卡，几乎没有真正的控制权。
