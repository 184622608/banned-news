### 抗议国际法院判决 美终止1955年美伊条约
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】 周三（10月3日），美国正式宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_终止.html" target="_blank">
  终止
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_1955年.html" target="_blank">
  1955年
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美伊条约.html" target="_blank">
  美伊条约
 </a>
 ，以抗议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际法院.html" target="_blank">
  国际法院
 </a>
 （The International Court of Justice, ICJ）要求美国放宽对伊朗制裁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_判决.html" target="_blank">
  判决
 </a>
 。
 <br/>
 <br/>
 周三，美国国务卿蓬佩奥（Mike Pompeo）宣布，美国退出63年前签署的美伊双边友好条约。此前不久，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际法院.html" target="_blank">
  国际法院
 </a>
 刚刚裁决，美国政府必须撤销某些因核问题而实施的对伊制裁。
 <br/>
 <br/>
 蓬佩奥在国务院亲自发布了这一消息，声明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_1955年.html" target="_blank">
  1955年
 </a>
 签署的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美伊条约.html" target="_blank">
  美伊条约
 </a>
 已过期39年，并指责伊朗企图滥用海牙国际法院的裁决来削弱美国的制裁政策。
 <br/>
 <br/>
 伊朗援引1955年协议向国际法院控告，川普政府在今春退出2015年的伊朗核协议后，限制伊朗的〝人道主义贸易〞，并宣称美国的行为违反了国际法。
 <br/>
 <br/>
 周三，国际法院初步裁决，华盛顿必须〝通过自己的选择〞来消除对伊朗重施制裁所引发的〝障碍〞，包括限制向伊朗出口药品和医疗器械、食物和农产品、以及确保民航安全所必需的零件和设备。
 <br/>
 <br/>
 美国此前曾声明，国际法院不应质疑美国制裁伊朗，因为事件涉及美国国家安全问题。国际法院院长、索马里的优素福（Abdulqawi Ahmed Yusuf）表示，法院将继续审理此案，美国可以质疑国际法院的管辖权。
 <br/>
 <br/>
 蓬佩奥表示，他对国际法院没有认识到自己并无相应权限感到〝失望〞，但他认为，法院的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_判决.html" target="_blank">
  判决
 </a>
 实际上是〝伊朗的失败〞。
 <br/>
 <br/>
 蓬佩奥还指出，国际法院拒绝按照伊朗的要求帮其更彻底地摆脱美国的制裁，并表示，美国的制裁政策已考虑到人道主义的例外情况，而伊朗政权却把国际援助用于支持国外的军事冒险而不是用于本国民生。
 <br/>
 <br/>
 蓬佩奥随后宣布，美国将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_终止.html" target="_blank">
  终止
 </a>
 1955年的美伊友好条约，解除了伊朗诉讼美国的所谓法律基础。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
</div>
