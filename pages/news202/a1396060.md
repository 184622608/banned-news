### 沙国称将军听错命令错杀《华邮》记者
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】美国《华盛顿邮报》特约记者卡舒吉本月初在土耳其惨遭肢解虐杀，事件引起全球舆论哗然，有迹象显示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙乌地阿拉伯王储.html" target="_blank">
  沙乌地阿拉伯王储
 </a>
 与卡舒吉遇害有重大牵连。日前，沙国当局放出消息称，王储本来是指示情报将军将卡舒吉带回沙国审问，但将军却错听成要将卡舒吉〝干掉〞。外界怀疑，这个新的说法是王储为转嫁罪责而炮制的谎言。
 <br/>
 <br/>
 《日本每日新闻》可日前报导从，沙国当局曾一力否认该国王储与卡舒吉（Jamal Khashoggi）遭虐杀事件有关联，但随着更多证据被各国媒体挖掘曝光，沙国的说法也发生了变化，其最新的说法是，王储本来指示情报将军阿西里（Ahmed al-Assiri）将卡舒吉带回沙国审问，没想到阿西里〝搞错〞了命令，听成王储要将卡舒吉〝干掉〞。
 <br/>
 <br/>
 不过，对于上述消息，其他外媒大多不予采信。舆论认为，这是沙国当局蓄意对外释放的假消息，目的是把王储主导虐杀卡舒吉的罪责，推诿转嫁给一个〝做了超出职位权限事情〞的将军。也就是说，阿西里成为了沙国当局的替罪羊。
 <br/>
 <br/>
 现年59岁的卡舒吉是《华盛顿邮报》的特约记者，同时他也是沙乌地阿拉伯的公民。今年10月2日卡舒吉在进入沙国伊斯坦堡领事馆后便音讯全无。其后，调查人员在伊斯坦堡领事馆内发现了已遭卡舒吉肢解的遗体。
 <br/>
 <br/>
 卡舒吉近两年屡屡发文批评沙乌地阿拉伯〝独裁〞，外界有舆论认为，这可能是他招致杀身之祸的原因。不过，也曾有媒体披露卡舒吉是穆斯林兄弟会的支持者，指他曾与本．拉登关系密切。
 <br/>
 <br/>
 随着各国媒体披露出越来越多的证据，外界高度怀疑沙国王储穆罕默德是谋杀卡舒吉行动的幕后指使者。而对于来自国际社会的质疑与批评的声浪，沙国政府开始采取了〝全盘否认〞的态度，后来才不得不转变为〝承认部分事实〞。
 <br/>
 <br/>
 据公开的资讯，随着调查的深入，一个由15名成员组成的暗杀特务团队逐步被挖掘出来，这些成员的身份则大多数是沙国的军方人员或情报官员。
 <br/>
 <br/>
 其中一名被媒体曝光的嫌犯阿波史丹尼，是沙国皇家空军的一名中尉，该人员被指涉嫌参与杀害卡舒吉后不久，就在沙国首都的一起可疑车祸中丧生，让外界怀疑沙国当局已展开灭口行动。
 <br/>
 <br/>
 美国《华盛顿邮报》近日更起底了前述15名嫌犯中的12人的真实身分，其中有一个名为穆特瑞伯（Maher Abdulaziz Mutreb）的嫌疑人被证实是沙国外交官。穆特瑞伯此前时常陪同沙国王储外访。
 <br/>
 <br/>
 土国媒体《晨报》日前公布的一批照片显示，穆特瑞伯在卡舒吉失踪当日进入了沙国驻土耳其领事馆。此外还有一系列照片显示，穆特瑞伯曾在案发前后进出与案件相关的场所。这些照片被视为卡舒吉案与沙国王储有关的有力证据。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
