### 英国脱欧关键期 首相梅地位可能遭挑战
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国首相.html" target="_blank">
  英国首相
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特雷莎．梅.html" target="_blank">
  特雷莎．梅
 </a>
 ，在宣布政府内阁批准
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 后，内阁重臣相继辞职，英国首相的地位可能遭到保守党内议员们的挑战。她在周日接受〝天空〞新闻采访时表示，未来7天对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 来说非常关键。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国首相.html" target="_blank">
  英国首相
 </a>
 梅：〝我认为（脱欧）草案符合国家利益，未来7天会非常重要，事关国家未来，人们的工作，他们的生计，子孙的未来。〞
 <br/>
 <br/>
 梅谈到她将继续专注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 谈判。
 <br/>
 <br/>
 如果保守党要更换领导人，须通过该党下议院议员进行〝不信任投票〞。而触发这个程序，必须由至少15%的保守党下议院议员提出书面动议。如果首相在投票中赢得信任，可继续担任首相，而且在一年之内不可以再有新的不信任投票。否则，首相就必须辞职。
 <br/>
 <br/>
 按照目前下议院保守党议员席次计算，至少要有48名保守党议员向保守党议会党团主席致函，表达〝不再信任党领导人〞。
 <br/>
 <br/>
 英国前脱欧部长多米尼克.拉布（Dominic Raab）周日对BBC表示，梅首相的脱欧协议存在致命缺陷，但她仍然可以改变方向。拉布认为英国与欧盟未来的关系缺乏明确性。
 <br/>
 <br/>
 另一方面，英国反对派工党领袖杰里米-科尔宾周日在“天空”新闻采访中说，工党不会支持首相在即将举行的议会投票中支持英国脱欧协议，
 <br/>
 <br/>
 工党领袖杰里米．科尔宾：〝我们将投票反对协议，因为不符合我们的期望，不符合英国利益，因此政府要返回欧盟，尽快再次谈判。〞
 <br/>
 <br/>
 工党部分成员要求就该协议进行全民公决。对此，科尔宾表示举行第二次英国退欧公投是未来的选择，而不是现在。
 <br/>
 <br/>
 新唐人记者严枫、天羽综合报导
