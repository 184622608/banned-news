### 中共〝掠夺性行为〞  智库：拉美警惕不足
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】日前G20峰会在阿根廷举行，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对拉美的投资与影响力渗透也成为话题之一。习近平在会后接连访问阿根廷与巴拿马，蓬佩奥则曾呼吁区域内国家，警惕中共经贸行为背后的政治野心。
 <br/>
 <br/>
 G20峰会后，习近平首先访问东道主阿根廷。双方签署了未来五年共同行动计划，并达成超过30项协议。2日晚，习近平抵达巴拿马。这也是去年巴国与台湾断交、转向北京后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 领导人首次访问。双方讨论了关于自由贸易协议，及加强〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞合作。
 <br/>
 <br/>
 面对中共不断扩大在拉丁美洲的影响力，美国国务卿蓬佩奥曾呼吁区域内国家，警惕中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_掠夺性行为.html" target="_blank">
  掠夺性行为
 </a>
 〞。G20期间受访时，他再度强调，与美洲国家保持密切关系非常重要，并对中共进入这些国家的方式感到担心。
 <br/>
 <br/>
 美国国务卿 Mike Pompeo：〝中共的到来，总是表面是来支持那个国家的，但其实不都是带着对该国人民的良好意愿而来，大多数情况还是为了中共。〞
 <br/>
 <br/>
 前哥斯特黎加贸易部长 Anabel Gonzalez：〝拉丁美洲国家了解这些风险非常重要，从中共及世界其它地区经验得到教训。〞
 <br/>
 <br/>
 拉美咨询委员会开放与发展中心研究员 Juan Pablo Cardenal：〝那些政治精英们并不真正了解，中国作为一个国家的复杂性。他们只关注商机，而忽视了风险。〞
 <br/>
 <br/>
 中国目前已成为拉丁美洲第二大贸易伙伴和第三大投资来源国。九个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉美国家.html" target="_blank">
  拉美国家
 </a>
 已被中共拉入〝一带一路〞计划。2005年到2017年，中共通过两大国有银行，对拉美地区累计贷款超过1500亿美元，最大贷款对象就是委内瑞拉、其次是巴西、阿根廷及厄瓜多尔。其中，对委内瑞拉专制政权的贷款饱受国际批评；而巴西新当选总统波索纳洛一直称中共为〝掠夺者〞，企图主宰巴西重要经济部门。
 <br/>
 <br/>
 拉美咨询委员会开放与发展中心研究员 Juan Pablo Cardenal：〝不透明和无监督，这就是中共制度的一部份。这与民主国家的软实力很不同。至少他们(民主国家)是完全开放，以竞争为基础的。〞
 <br/>
 <br/>
 当前，在美中两套完全不同价值体系竞争之下，拉美国家正面对重要抉择。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
