### 也门一半人口处饥荒 UN特使将至 叛军停火释善意
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】联合国特使葛瑞菲斯预计近日前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_也门.html" target="_blank">
  也门
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叛军.html" target="_blank">
  叛军
 </a>
 所掌控的首都沙那，敲定即将在瑞典举行的和平谈判。为此，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青年运动.html" target="_blank">
  青年运动
 </a>
 〞（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡塞武装.html" target="_blank">
  胡塞武装
 </a>
 组织）高层官员已要求旗下官兵，停止发射火箭或使用攻击无人机。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青年运动.html" target="_blank">
  青年运动
 </a>
 〞高级革命委员会（Higher Revolutionary Committee）主席胡提（Mohammed Ali al-Huthi）在推特贴文表示，他们希望〝所有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_也门.html" target="_blank">
  也门
 </a>
 官方团体〞停火。
 <br/>
 <br/>
 胡提表示，目的在于〝支持（联合国）特使的努力，并证明我们的善意〞。
 <br/>
 <br/>
 联合国特使葛瑞菲斯将启程赴也门，盼促成和平谈判以结束内战。也门
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叛军.html" target="_blank">
  叛军
 </a>
 〝青年运动〞（Huthi）高层官员已要求旗下官兵，停止发射火箭或使用攻击无人机。
 <br/>
 <br/>
 联合国特使葛瑞菲斯（Martin Griffiths）16日表示，他预计在本周内前往沙那（Sanaa），以尽快敲定即将在瑞典举行的和平谈判。
 <br/>
 <br/>
 葛瑞菲斯表示，沙特阿拉伯所支持的也门政府及叛军双方，都重新表达致力政治解决的承诺，双方都〝坚定保证〞会出席和平谈判。但和谈日期尚未确定。
 <br/>
 <br/>
 〝青年运动〞2014年占领沙那及沿海城市荷台达（Hodeida），促使沙特阿拉伯领导的联军隔年出手介入，帮助也门政府军。
 <br/>
 <br/>

  </a>
  <br/>
 </center>
 <br/>
 上月底，美国国防部长马蒂斯（Jim Mattis）与国务卿蓬佩奥（Mike Pompeo）先后公开呼吁，沙特联军与叛军停火，并于未来一个月就持续的也门内战坐上谈判桌。
 <br/>
 <br/>
 但青年运动仍利用也门首都沙那国际机场和毗邻的空军基地发动攻击，导致联合国主导的和平会谈无法顺利举行。
 <br/>
 <br/>
 世界卫生组织（WHO）指出，从沙国领导的联军介入以来，有近1万人在内战中丧生，但人权组织认为死亡人数可能是这个数字的5倍。另据挪威难民理事会（Norwegian Refugee Council）称，目前已有800万人处于饥荒边缘。
 <br/>
 <br/>
 也门内战已陷入严重的人道危机。联合国官员表示，除非战斗停止，否则这一数字将很快上升至1200万，而这一数字约占也门人口的一半。
 <br/>
