### 【禁闻】红色高棉两头领被判种族灭绝罪
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红色高棉.html" target="_blank">
   红色高棉
  </a>
  两头领被判
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_种族灭绝罪.html" target="_blank">
   种族灭绝罪
  </a>
 </b>
 <br/>
 <br/>
 联合国资助的柬埔寨特别法庭，11月16号在金边首次裁定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红色高棉.html" target="_blank">
  红色高棉
 </a>
 ，也就是柬埔寨共产党的两名前头领犯下了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_种族灭绝罪.html" target="_blank">
  种族灭绝罪
 </a>
 〞，判处他们终身监禁。
 <br/>
 <br/>
 其中一人是红色高棉政权领袖波尔布特的副手，现龄92岁的农谢，他被控消灭穆斯林占族人和越南族人。据说农谢有华人血统，中文名为刘平坤，在红色高棉主管意识形态，是杀人〝总设计师〞。
 <br/>
 <br/>
 另一人是87岁的乔森潘，他在红色高棉政权中先后做过总理和国家主席，他被指控对越南裔民众实施种族灭绝。
 <br/>
 <br/>
 法官宣读对农谢和乔森潘的判决时，很多受害者和家属在法庭旁听。除了种族灭绝，农谢和乔森潘还被判犯有一长串罪行，其中包括强迫婚姻、强奸和宗教迫害等。
 <br/>
 <br/>
 柬埔寨共产党总书记波尔布特以毛泽东为导师，按照中共的指导方针实行大屠杀，从1975年至1979年统治柬埔寨期间，杀害了大约200万人，占当时柬埔寨人口的四分之一。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
   苹果
  </a>
  手机供应商
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裁员.html" target="_blank">
   裁员
  </a>
  八千 引发工潮
 </b>
 <br/>
 <br/>
 全球最大手机屏幕供应商、港资惠州伯恩光学公司，日前宣布裁员8,000人，引发大规模抗议。
 <br/>
 据报导，被裁的工人因年底找不到工作，再加上对赔偿条件不满，从11月9号开始连续几天包围工厂示威，当局派出大批防暴警察驻守工厂，驱散抗议者。到15号，劳资双方初步达成协议，工人散去。
 <br/>
 <br/>
 伯恩光学公司是苹果和三星手机最大的屏幕供应商，由于受美中贸易战的影响，最近苹果大幅削减手机定单，使伯恩光学陷入无订单可做的困境。
 <br/>
 <br/>
 <b>
  甘肃农业专家遭迫害离世
 </b>
 <br/>
 <br/>
 甘肃省庆阳市宁县高级农艺师郭振邦，因修炼法轮功，十几年来一直被中共人员骚扰、关押、恐吓。今年3月起，警察和社区人员又多次上门对他进行威胁、施压，逼迫他放弃信仰。导致他在压力中，卧床不起，10月底不幸含冤离世，终年76岁。
 <br/>
 <br/>
 据〝法轮大法明慧网〞报导，郭振邦曾任县科委主任、人大代表、政协委员等职，是享受中共国务院特殊津贴的农业专家，1998年因身体疾病，开始修炼法轮功，短短时间，就恢复了健康。但1999年之后，屡遭中共迫害，健康再度受损，导致离世。
 <br/>
 <br/>
