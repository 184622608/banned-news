### 怀旧美食品尝童年的味道——完州野生美食节
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】远离城市的喧嚣，卸下一切烦恼忧愁，在周末好时光里来一次回归
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_童年.html" target="_blank">
  童年
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美食.html" target="_blank">
  美食
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旅行.html" target="_blank">
  旅行
 </a>
 ，是个不错的选择，完州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野生美食节.html" target="_blank">
  野生美食节
 </a>
 以传统方法体验美食的健康庆典，吸引了大批游客观光参加，请看来自韩国的报导。
 <br/>
 <br/>
 为期三天的完州野生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美食.html" target="_blank">
  美食
 </a>
 节在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国完州.html" target="_blank">
  韩国完州
 </a>
 举行，庆典期间虽然下起雨来，但是仍未挡住前来参访庆典的游客。
 <br/>
 <br/>
 得天独厚的地理条件，孕育出品种多样且优质的农作物和野生产品。在完州土生土长的柿子非常有名气，当地人用天然发酵的方式制做柿子醋，不仅美味，而且营养丰富。
 <br/>
 <br/>
 高山农业协会职员 金成培：〝这个地区有水源地，因此水很干净，用干净的水浇灌，生长的环境空气清新，在这样的条件下生长的柿子，再经过三年的发酵做出的醋就是我们的醋。〞
 <br/>
 <br/>
 在田间嬉戏最能让大人们回想起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_童年.html" target="_blank">
  童年
 </a>
 ，也是孩子们最喜欢的地方，在田里抓蚂蚱，采野菜，和野生昆虫近距离接触，享受大自然带给人们的乐趣。
 <br/>
 <br/>
 完州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野生美食节.html" target="_blank">
  野生美食节
 </a>
 参访游客：〝有意思，以前也来过，但是现在比以前的活动更多了。〞
 <br/>
 <br/>
 河里徒手抓鱼活动，让人们玩的不亦乐乎，将抓来的鱼用最原始的方式进行烹饪，既好玩又能品尝美味。
 <br/>
 <br/>
 完州野生美食节丰富多彩的活动和美食都是由当地居民亲自准备的，这些更是让庆典充满乡村质朴的气息。
 <br/>
 <br/>
 完州郡郡守 朴成日：〝完州郡推行为消费者提供最安全饮食的政策，这样的政策得到世界的认可，曾获得米兰诺城市美食政策奖，希望大家喜欢我们柿子醋，同时也喜欢我们的农作物，希望大家参访完州野生美食庆典。〞
 <br/>
 <br/>
 新唐人记者金燕韩国完州报导
</div>
