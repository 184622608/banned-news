### 外媒笑谈习安会：安倍亲访止于〝出借猫熊〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 结束为期三天的访华行程已回国。有美媒评论说，安倍与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 会面时，为双方关系改善做了最大的演出，包括双方企业全力配合达成52项合作备忘录（MOU）等，只是〝空中画大饼〞口惠而实不惠。唯一实质的承诺大概就止于〝出借猫熊〞。
 <br/>
 <br/>
 日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 27日已经结束访华，乘坐专机返回日本。安倍访华的第一天（25日），曾会见了中共总理李克强。10月26日，安倍与中共国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在北京钓鱼台国宾馆举行会谈（习安会）。
 <br/>
 <br/>
 当晚安倍在推特上透露，他与习近平的会谈中，双方确认了〝化竞争为协调〞、〝成为合作伙伴而非威胁〞、〝推进自由公平的贸易〞等三个新原则。
 <br/>
 <br/>
 中共官媒也一致唱好中日新关系，声称这次双方签定的MOU达52项，包括金融、保险、IT、基建、物流及医疗保健等范围。
 <br/>
 <br/>
 台湾《自由时报》28日刊发评论说，这是安倍时隔7年正式访华，为日中关系改善做了最大的演出。即使双方企业全力配合达成52项合作备忘录（MOU），但包括钓鱼台争端、中国擅自开发东海油田、日本人被以间谍罪拘捕，以及日本食品管制进口等悬案，均未能得中方进一步承诺，唯一实质的承诺仅在于〝出借猫熊〞。
 <br/>
 <br/>
 评论说，日本首相上次访中是在2011年的野田佳彦。换言之，安倍在2012年底上台后从未正式访中，习近平上台后也从未访日，两国的〝梁子〞结得很深。
 <br/>
 <br/>
 除了历史旧帐外，中共试图扩大在东海及南海的影响力，长期派遣公务船巡航钓鱼台海域、在南海建设军事基地，日本因极力牵制成了中共进出海洋的最大绊脚石。
 <br/>
 <br/>
 评论认为，这些影响日中关系的变数，并不会因为安倍访华和李克强或习近平握手就能解除。促使中共积极拉拢日本的主因是美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 加遽。
 <br/>
 <br/>
 而安倍马上抓住这个机会做顺水人情。综观这次他访华带去的大礼，包括重启两国〝货币互换协议〞，金额较5年前扩大约10倍达3兆日圆，另外就是两国合作进军第三国市场。
 <br/>
 <br/>
 日本进军海外基建过去曾多次在中国的削价竞争下败北，对安倍来说，与中国合作进军第三国市场具有经贸利益的考量。
 <br/>
 <br/>
 值得注意的是，这次双方签定的MOU达52项有近五分之一与金融有关，日本野村控股、大和证券两大证券公司与中共官股共同成立规模超过1000亿元的〝日中产业协力基金〞，以金援两国企业未来相互投资或进军亚洲第三国时所需。
 <br/>
 <br/>
 虽然5大证券、银行业宣示未来将向其他金融机构继续募资，以确保2000亿元的投资能力，但和日本政府向来以ODA或输出信用援助海外基建相比，规模和等级相去甚远。
 <br/>
 <br/>
 评论说，安倍这次访中演出卖力，但带回来的回礼看来不算丰硕，日本最期待的中方解除日本食品进口管制，以及外相河野太郎要求中方撤除先前部署于该海域的浮标，均没有得到具体承诺。而中共长期派公务船巡逻钓鱼台的问题更是无解，
 <br/>
 <br/>
 虽然这是日中关系改善的第一步，但毕竟这是中国在美中摩擦下的策略性举措，影响日中关系的诸多变数仍然存在，第一步跨出之后的下一步才是考验。
 <br/>
 <br/>
 日本前外交官出身的评论家河东哲夫，在安倍访中前已警告，日中两国企业体质不同，中方的海外基建投资有太多不良记录，两国合作进军第三国恐怕只是〝空中画大饼〞口惠而实不惠。
 <br/>
 <br/>
 日本《时事通讯社》说，在中美贸易战下，中共急需世界第三大经济体〝日本的技术与资金〞。两国虽然努力营造明年两国领导人互访的气氛，但双方几乎避开在钓鱼台、东海及历史等有争议的问题，中日的新时代只是〝同床异梦〞。
 <br/>
 <br/>
 日本《产经新闻》则形容中日两国的关系只是〝沙上城堡〞，日本不可能背弃西方盟友，替中共在国际上说项，故两国不可能建立〝真正友谊〞。
 <br/>
 <br/>
 时事评论员刘锐绍对港媒说，中共拉拢日本，是企图〝借日本对抗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 〞，中共的如意算盘不可能实现。
 <br/>
 <br/>
 过去数月，中共曾一度想拉拢欧洲对抗美国，但遭到欧盟的拒绝，而美欧还宣布将共同努力实现〝零关税、零补贴〞的贸易制度。欧盟主席容克更是向川普承诺，欧盟将与美国联手，共同应对中共的不公正贸易行为。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：祝馨睿）
