### 顶级八音盒赏鉴会 精妙工艺秀奢华
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】10月23日，来自瑞士的世界顶级机械
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_八音盒.html" target="_blank">
  八音盒
 </a>
 御爵（Reuge），在加拿大温哥华的香格里拉酒店首次举办私人邀请赏鉴会，品牌CEO库帕卡特先生展示了多款限量版八音盒，与嘉宾分享精湛工艺下的美妙音律。
 <br/>
 <br/>
 清澈透亮的旋律发自一台名叫Anastasia的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_八音盒.html" target="_blank">
  八音盒
 </a>
 ，全球限量仅有12台，独特的转轮式机芯彰显瑞士工匠巧夺天工的技艺。
 <br/>
 <br/>
 御爵CEO Kupper Kurt: 〝你可以看到（八音盒里）有6个圆筒运转，每一个圆筒上有4首不同的旋律，因而它是一个非常复杂的工艺，这是世界上任何其他人都无法做到的。〞
 <br/>
 <br/>
 这是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安徒生童话.html" target="_blank">
  安徒生童话
 </a>
 中的夜莺吗？镀金的鸟笼，极致的细节，奢华感十足，吸引不少嘉宾观赏拍照。
 <br/>
 <br/>
 御爵CEO Kupper Kurt: 〝我们家会唱歌的音乐鸟也很特别，身上有真的鸟毛，独一无二，你按下按钮，音乐鸟会跳出来唱歌，如果你观察的话，唱歌的时候鸟的嘴会张开的，所以是很精准的。〞
 <br/>
 <br/>
 镀金的麦穗在八音盒的妙音下摇曳，意境宁静致远，这款由御爵与悉尼奥格斯堡大学联合打造的slightly windy也在赏鉴会上相当亮眼。
 <br/>
 <br/>
 秉承瑞士精密机械制造的深厚底蕴以及一丝不苟的工匠精神，Reuge御爵八音盒那清脆灵动的乐音，艺术品般的精美外观以及卓越的品质打动了无数皇室名流和收藏家。
 <br/>
 <br/>
 御爵CEO Kupper Kurt: 〝在如今快节奏的时代，今天流行的东西明天就过时了，这些都是转瞬即逝的东西，所以是音乐盒是这样一个你会永远珍藏的礼物。〞
 <br/>
 <br/>
 在电子设备横行的当下，人们在iPod和iPhone上轻滑指尖便可以欣赏到成千上万的音乐作品。可纯手工制作的御爵（Reuge）八音盒，在不使用电力的情况下，让成千上百个精密机械零件和谐运转。御爵成为超越时代的艺术品，为人们代代珍藏。
 <br/>
 <br/>
 新唐人记者舒璨加拿大温哥华报导
</div>
