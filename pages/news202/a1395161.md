### 西班牙国庆关注统一 与加泰渊怨未解
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】10月12日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 国王菲利浦六世在首都马德里阅兵，庆祝西班牙的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国庆.html" target="_blank">
  国庆
 </a>
 日。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 国王菲利浦检阅军用飞机、伞兵和军队仪仗队，庆祝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国庆.html" target="_blank">
  国庆
 </a>
 。该国国庆是为纪念代表西班牙王室的航海家哥伦布，于1492年抵达美洲，因为这彰显了西班牙的国威和成就。当天，成千上万人走上街头挥舞着国旗，高呼西班牙
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_统一.html" target="_blank">
  统一
 </a>
 。
 <br/>
 <br/>
 过去一年，西班牙经历了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加泰.html" target="_blank">
  加泰
 </a>
 罗尼亚的独立风波。
 <br/>
 <br/>
 去年10月27日，加泰罗尼亚议会宣布独立，成立共和国。10月30日，西班牙政府实施宪法第155条，强行接管加泰罗尼亚当地政府，禁止公投并宣布独立违宪。
 <br/>
 <br/>
 加泰罗尼亚企业不看好独立，因为如此将不属于欧盟，超过3000家企业包括当地第一大银行都搬离该地区。
 <br/>
 <br/>
 一年后，西班牙与当地的分歧仍在。
 <br/>
 <br/>
 国庆日当天，西班牙首相桑切斯与国王共同出席庆祝仪式。桑切斯反对加泰罗尼亚独立，但愿意继续就分歧展开对话。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
