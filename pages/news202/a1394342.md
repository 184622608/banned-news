### 筑地市场营运最后一天 道别人潮挤满场内
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】号称〝日本厨房〞的筑地市场旧址营运今天是最后一天，就像过往的每个清晨，市场内人声鼎沸，而鱼贩们赶在黎明前参加最后一次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲔鱼.html" target="_blank">
  鲔鱼
 </a>
 拍卖，并依依不舍地向有83年历史的筑地市场道别。
 <br/>
 <br/>
 综合媒体报导，筑地市场此前因设备老旧，一直是当局关注议题，但因新址〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丰洲市场.html" target="_blank">
  丰洲市场
 </a>
 〞被检测出土地有污染问题，搬迁一再推迟。如今将结束营运，83年历史将永远成为回忆，并于11日于丰洲市场重新开幕。
 <br/>
 <br/>
 丰洲市场是筑地的1.7倍大，场内加强冷冻冷藏及真空包装等加工设施，鲜度与卫生管理大幅提升。
 <br/>
 <br/>
 许多民众到场与筑地市场道别，而最后的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲔鱼.html" target="_blank">
  鲔鱼
 </a>
 则拍出了438万日元（约38,632美元）高价。
 <br/>
 <br/>
 知名的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉野家.html" target="_blank">
  吉野家
 </a>
 〝筑地一号店〞也将在当地时间13时关门，门口可见到饕客大排长龙。
 <br/>

 <br/>
 尽管许多人承认筑地市场建物老旧，无法支持庞大的经营量，但筑地市场数十年来一直是世界美食之都东京的心脏，有〝日本的厨房〞之称。
 <br/>
 <br/>
 筑地市场业者代表吉田猛社长致词时表示，对把筑地市场推向世界的大家表示由衷感激。
 <br/>
 <br/>
 而对于市场常客来说，心中也是五味杂陈，68岁退休拍卖员石井久夫（Hisao Ishii）说：〝今天是个伤心的道别日，筑地试着赶上时间，但它渐渐变老了。我今天是来跟筑地说谢谢还有再见的。〞
 <br/>
 <br/>
 鱼类批发商吉田猛（Takeshi Yoshida）则表示，筑地市场已经〝青史留名〞，现在是〝交棒〞的时候了。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_筑地巿场.html" target="_blank">
  筑地巿场
 </a>
 原是一片东京湾上的填海地，1923年关东大地震后，政府在筑地明石町成立临时巿场，于1935年划为正式的批发市场。占地23公顷，是全球最大型的鱼市场，每日有超过1万8000辆大小车辆及接近4万2000人进出、每日卖出鱼货超过1600吨、日销售额达1400万美元。
 <br/>
 <br/>
 如今搬迁后，筑地市场将改建成东京奥运的物流交通枢纽，场外市场的商店街仍然会保留，也不会改变名字。
