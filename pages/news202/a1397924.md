### 惊险！日2架战机空中碰撞 机身受损安全降落
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】日本航空
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自卫队.html" target="_blank">
  自卫队
 </a>
 2日2架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-2战机.html" target="_blank">
  F-2战机
 </a>
 训练期间发生碰撞，分别导致机身的垂直尾翼、导弹发射装置及燃料箱受损，所幸当时并未搭载导弹。两架战机事后安全返回基地，两名飞行员无受伤。
 <br/>
 <br/>
 事发于周五下午近4时(当地时间)，2架驻守福冈县筑城基地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-2战机.html" target="_blank">
  F-2战机
 </a>
 结束战斗训练，在靠近互相目测机身是否有物体跌落时，一架战机的垂直尾翼与另一架战机左机翼下的导弹发射装置碰撞。
 <br/>
 <br/>
 随即在机师确认没有漏油及可以继续飞行后，于半小时后顺利返抵基地。
 <br/>
 <br/>
 筑城基地由于战机的垂直尾翼受损，出现导致重大事故的危险，已经展开调查意外原因。
