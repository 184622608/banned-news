 <br/>
 今年3月，国会通过2019财年政府综合预算法案，其中包含7160亿美元的巨额军费。不过该项拨款不仅用于装备武器，还包含五角大楼的其它所有开销。
 <br/>
 <br/>
 这项巨额拨款为川普政府保障国防安全提供了前提。今年6月，川普命令国防部 立即建立一支〝太空部队〞（space force），该部队将被列为第六支美国武装部队，与美国空军同级但各自独立。
 <br/>
 <br/>
 川普表示，他期望通过重返月球和登上火星来复兴美国逐渐衰退的太空计划。
 <br/>
 <br/>
 川普的出发点在于，针对中俄在太空领域的发展，通过恢复美国在太空项目的领先地位来更好地保证国家安全。
 <br/>
 <br/>
 8月初，副总统彭斯（Mike Pence）在五角大楼宣布了美国的太空计划，将川普建立太空部队的命令继续推进一步。
 <br/>
 <br/>
 彭斯表示，太空部队预计于2020年建成，这将为美国武装力量书写新的历史篇章，旨在威慑和挫败新一代威胁美国安全的势力。
 <br/>
 <br/>
 彭斯强调了加强太空建设的必要性。他指出，中共和俄罗斯一直在从事高水平的太空活动，且能够让自己的卫星接近美国，这对美国的太空系统构成了前所未有的新的威胁。
 <br/>
 <br/>
 彭斯透露，中俄都在重金研发高超音速的弹道导弹，以便在很低的高度就能达到5英里/每秒的飞行速度，以至于可能逃脱美国导弹防御雷达的监测。对美国来说，这是非常危险的。
 <br/>
 <br/>
 彭斯指出，事实上，中共前一周声称，他们刚刚成功测试了一种高超音速飞行器。
 <br/>
 <br/>
 但川普总统在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 后信心满满地表示，他相信在将来某一天，习近平和普京都会愿意坐下来就停止军备竞赛进行协商，从而减少各自的经济消耗，显然，这种信心也是川普在G20取得巨大外交成功的另一方面的表现。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
