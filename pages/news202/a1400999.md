### 法国人抗议能源负担 马克龙不肯让步
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】过去两周，法国境内爆发上千场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_示威游行.html" target="_blank">
  示威游行
 </a>
 ，抗议政府上调
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃油税.html" target="_blank">
  燃油税
 </a>
 。法国总统马克龙周二发表讲话，他表示了解民众的不满，但不会改变对能源政策的态度。
 <br/>
 <br/>
 周六，法国巴黎和其他城市数万名民众，身穿黄马甲，游行抗议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃油税.html" target="_blank">
  燃油税
 </a>
 上涨。游行过程中，抗议者向警方扔石块，在路上纵火，警方以催泪弹和水枪回应。示威活动造成2人死亡，数百人受伤。
 <br/>
 <br/>
 周二，法国总统马克龙针对能源政策发表了一小时演讲，表示不会将暴徒和传达信息的民众混在一起。
 <br/>
 <br/>
 法国总统马克龙：〝我想说，我不会将这些不可接受的举动，和他们的抗议混为一谈。我不会将暴徒和想要传递信息的民众混为一谈。
 <br/>
 〞
 <br/>
 <br/>
 抗议者在高速公路上设置路障，其诉求从要求降低燃油税，发展到要求马克龙下台。马克龙则表示不会向制造混乱的人屈服。
 <br/>
 <br/>
 法国总统马克龙：〝我理解这些市民，但我不会向制造混乱和破坏的人屈服。〞
 <br/>
 <br/>
 马克龙上台一年多，柴油价格上涨23%。大部分抗议者表示不反对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_能源改革.html" target="_blank">
  能源改革
 </a>
 ，但反对以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绿色改革.html" target="_blank">
  绿色改革
 </a>
 为名，让中下层民众承担改革成本。
 <br/>
 <br/>
 有报道称，2015年〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎气候协定.html" target="_blank">
  巴黎气候协定
 </a>
 〞签署后，提升燃油税成为法国政府内部新的〝政治正确〞。
 <br/>
 <br/>
 今年6月，美国总统川普宣布美国将单方面退出〝巴黎气候协定〞。
 <br/>
 <br/>
 新唐人记者毕心慈、宇微综合报导
</div>
