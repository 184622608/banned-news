### 法国政府涨税喊停 左派政策累积问题多
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】过去几个周末，巴黎的抗议活动出现了砸商店，烧汽车等暴乱现象。周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国政府.html" target="_blank">
  法国政府
 </a>
 松口表示，暂停增加燃油税计划，并将和民众进行协商。
 <br/>
 <br/>
 周二，法国总理菲利普在和部分抗议者代表谈话后表示，暂停增加燃油税计划六个月，这期间，政府将讨论如何为工薪阶层提供帮助。
 <br/>
 <br/>
 这是法国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 上台18个月以来第一次收回改革措施。
 <br/>
 <br/>
 法国国会议员格瑟林认为，自从11月7日抗议爆发以来，政府忽视了很多事情。导致现在的措施不足以解决问题。
 <br/>
 <br/>
 法国国会议员 格瑟林：我觉得，三周前本可以听取和接受的，现在看上去都太晚了，而且不够。
 <br/>
 <br/>
 过去两个多星期里，抗议人群的目标很快从高油价扩展到了总统马克龙的其他改革措施。所以当政府承诺让步后，身穿黄马甲的抗议者仍然没有停止。
 <br/>
 <br/>
 抗议者 Loic Hennequin：我不认为（政府暂停增加燃油税）就足够了。(我们)有太多要求，来自养老金领取者的，来自年轻人的和来自工人的要求。
 <br/>
 <br/>
 马克龙一年半之前上台时，法国失业率高达将近10%，而且多年来偏左的经济政策导致公共开支太大，经济竞争力差。上台后，马克龙消减国营企业的福利、修订劳动法、让企业更容易辞退员工，改革职业训练课程等等，都是满足长远发展需求的，但在短期会损害一些民众的利益。在尝到甜头之前，改革换来的只有示威和骚乱，马克龙支持率从六成降到现在的不到三成。有评论说，改革的步伐太快了。也有评论说，现在马克龙只能期待法国今年冬天特别冷，让示威者无法在街上久留。
 <br/>
 <br/>
 新唐人电视综合报导
