### 经济压榨结仇  一带一路被指是巴中领馆遇袭罪魁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】中共驻巴基斯坦喀拉蚩的领事馆遭3名自杀式袭击者的武装攻击后，巴基斯坦民族主义分离势力〝俾路支解放军〞宣布对这起事件负责，并声称动机是反对中共军事势力在巴基斯坦领土上扩张。有分析指，巴基斯坦人是不满中共通过推行〝一带一路〞来掠夺该国资源而发起了这场攻击。
 <br/>
 <br/>
 当地时间11月23日上午，巴基斯坦民族主义分离势力俾路支解放军（Balochistan Liberation Army，BLA）派出3名袭击者攻击了中共驻巴基斯坦喀拉蚩领事馆，导致2名当地警员和2名当地平民死亡，3名袭击者则全部被当场击毙。领事馆内21名中方职员则都安全地躲过一劫。
 <br/>
 <br/>
 袭击事件发生后，BLA武装组织发布声明称，这次袭击行动是该组织派人实施的，他们的目的很明确，〝我们不会容忍任何中国军队在俾路支的土地上扩张〞。
 <br/>
 <br/>
 巴基斯坦政府随后则对外仅宣称，〝巴基斯坦与中国的友谊不会受到阻挠，中、巴两国已经联合起来加强维安，防止类似事件破坏两国的合作。〞
 <br/>
 <br/>

 <br/>
 据公开的资讯，巴基斯坦俾路支省地区拥有相当多的天然气、煤炭、铜矿与金矿储量，也是中共推行的〝中巴经济走廊计划〞（CPEC）中的核心。尽管巴基斯坦政府积极在该地区搞开发，但俾路支省仍然是巴国最贫困的地区。当地民众则长期不满中共势力在该地区的扩展，他们指控中共说的是帮助该国发展经济，但实际上是剥削他们的劳动来发展中国的企业，还趁机掠夺了该地区的矿产等资源。
 <br/>
 <br/>
 针对这次的领馆袭击事件，《自由亚洲电台》援引学者观点表示，中共近年来不断以〝一带一路〞名义压榨一些弱势国家，以致当地民众以激进的暴力手段来宣洩不满。
 <br/>
 <br/>
 据了解，早在今年8月中旬，巴基斯坦就已经发生过一起专门针对中国矿产企业的自杀式汽车袭击。
 <br/>
 <br/>
 8月11日，中国〝一带一路〞重点专案的中冶集团巴基斯坦山达克（Saindak）铜金矿的一辆大巴，装载着18名中方人员赶去机场准备前往卡拉奇的途中，突然遭到自杀式汽车炸弹的袭击。
 <br/>
 <br/>
 这起事件造成3名中方工程师和3名巴方雇员受伤，而自爆身亡的袭击者，也是俾路支武装组织的成员。
 <br/>
 <br/>
 巴基斯坦警方事后对外界表示，当地的安全局势已经恶化，他们曾通报外交部，并在领事网上发布安全预警，要求中方的企业和人员半年内不要去往当地。
 <br/>
 <br/>
 当时，从事中国海外援助专案多年的人士何先生在接受《自由亚洲电台》的采访时表示，中共政府层面的外援专案，几十年来一直都存在，只不过〝一带一路〞主要采取了贷款和专案运作的方式进行。
 <br/>
 <br/>
 他认为，无论是政治还是经济上，中共的专案都非常失败，因为实际上真正通过这些专案受益的人，只是当地的政府官员和参与投资或建设项目的中国国企的官员，而那些项目并没能对当地产生多少经济效益。
 <br/>
 <br/>
 据称，当下有数万名中国工人，在巴基斯坦服务于总值达570亿美元的中巴经济走廊（CPEC）建设，但近来频繁传出有烂尾的可能，频频发生的恐怖袭击则已成为中方〝一带一路〞西进的最大麻烦。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
