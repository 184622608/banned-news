### 美拟退出万国邮联  指责中共〝占便宜〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】美国白宫10月17日宣布，即日起启动退出万国邮政联盟（Universal Postal Union）的程序。川普（特朗普）政府指责，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万国邮联.html" target="_blank">
  万国邮联
 </a>
 现行费率制度，让中共和其他国家受益，对美国不公平。外界认为，中国电商将受到直接影响，中国寄美包裹价格会上涨。
 <br/>
 <br/>
 白宫的声明称，川普政府已收到美国国务院就8月23日总统签署邮递〝终端费率〞备忘录的报告回复，川普同意国务院的建议——基于9月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万国邮联.html" target="_blank">
  万国邮联
 </a>
 的新邮资协议未取得令美国满意的进展，美国将采用自行申报的终端费率，且不迟于2020年1月1日实施。
 <br/>
 <br/>
 据《美国之音》报导，白宫两名匿名高级官员在当天的记者会上称，万国邮联当前的〝终端费〞政策对美国不公平，全球多国都在国际邮件计费问题上〝占美国的便宜〞。
 <br/>
 <br/>
 在现行的国际邮政体系下，从中国寄送小包裹到美国，比从美国寄送到中国，运费更便宜。而在邮递到美国的国际包裹中，来自中国的占大多数。
 <br/>
 <br/>
 白宫高级官员表示，美国公司支付1美元国内
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邮费.html" target="_blank">
  邮费
 </a>
 ，中国承运商只需支付20美分，这种不公平的国际邮政费率制度相当于是给中共的补贴，也导致大量假冒商品和芬太尼涌入美国。如果能够纠正邮费费率的不公平，也能在一定程度上令假货和芬太尼问题得到好转。
 <br/>
 <br/>
 长期以来，主要商业团体也一直在敦促美国总统川普采取行动纠正邮政税率制度，称这使得他们对中国等竞争对手处于明显劣势。
 <br/>
 <br/>
 今年4月，美国全国制造商协会（National Association of Manufacturers）主席兼CEO蒂蒙斯（Jay Timmons），就曾致函财政部长姆努钦，称万国邮联体系让〝美国自己的邮政补贴中国进口的这种几乎是令人难以置信的做法〞。
 <br/>
 <br/>
 该商协会说，在去年的财政年度中，美国邮政因此损失大约1.7亿美元，〝由于中国蓬勃发展的对美电子商务销售，这个损失还在以每年大约40%的速度增长〞。
 <br/>
 <br/>
 蒂蒙斯表示，〝川普总统专注消除中国（共）从美国邮政局获得的反美制造商补贴，他应该获得巨大赞誉。〞
 <br/>
 <br/>
 根据万国邮联的规定，退出程序为期一年。白宫声明表示，在退出程序期间，美国国务院将通过双边或多边谈判来解决总统备忘录中讨论的问题。
 <br/>
 <br/>
 万国联盟总干事海珊称，将会寻求与美国的代表就此事进一步磋商。美国方面也说，〝如果谈判成功，川普政府会准备撤销‘退出通知’、并留在万国邮联。〞
 <br/>
 <br/>
 《纽约时报》的报导说，白宫宣布退出万国邮联是川普政府制衡中共、惩罚其不公平贸易政策的协调行动的一部分，此举将导致中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 形势更加升级。
 <br/>
 <br/>
 外界也预测，美国退出万国邮联后，中国运往美国的小包裹运费成本都将上调，那些通过阿里巴巴或亚马逊出售商品的中国电商将受到直接影响。
 <br/>
 <br/>
 万国邮政联盟成立于1874年，总部位于瑞士伯尔尼，目前有192个成员国。该组织为成员国设置邮费、协调各国邮政政策。根据邮联目前的条款，中国和一些其他发展中国家通过美国邮政向美国市场运输商品时，可以享受较低的邮费。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：范铭）
