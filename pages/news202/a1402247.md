### 北京雪上加霜 欧盟拟审查海外投资剑指中共
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】本周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 多数国家达成协议，决定建立一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外国投资审查体系.html" target="_blank">
  外国投资审查体系
 </a>
 ，以保护欧洲战略性的技术和基础设施。舆论认为，这项计划的主要目标指向中共在海外的扩张。
 <br/>
 <br/>
 据路透社12月5日的报导，据欧洲理事会和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 轮值主席国奥地利的发言人披露，欧盟多数国家同意建立一个广泛的体系，以协调对外国投资的审查，只有意大利和英国持保留意见。
 <br/>
 <br/>
 该报导透露，这项计划主要是因应来自中国的投资急增的情况而出台，欧盟执委会将对港口、能源、网络等战略技术和基础设施方面的外国投资进行调查。
 <br/>
 <br/>
 报导表示，欧洲议会和来自欧盟28个成员国的谈判代表上月就保护关键行业达成了一项交易，但希腊、塞浦路斯、卢森堡马耳他和葡萄牙等国对这项提案表示反对，欧洲议会打算在明年2月或3月对该提案进行投票。
 <br/>
 <br/>
 据公开的资讯，欧盟28国的谈判代表经过长达数月的协商，于今年11月20日达成了一项协议，同意建立对外国投资的审查机制。这项协议旨在保护欧洲战略性的技术和基础设施，例如港口和能源体系。
 <br/>
 <br/>
 根据这项计划，欧盟执委会将调查关键领域的外国投资，并提出建议。这些建议将解释重要基建系统是否可能受到破坏，以及多年来花巨资研究的新技术是否可能落入外国人手中。
 <br/>
 <br/>
 该协议虽未提及中国，但其支持者们在磋商中提到国有企业对欧洲的投资问题以及窃取技术的现象，明显剑指中共。
 <br/>
 <br/>
 而此前，欧洲议会曾在今年5月，通过了一项旨在加强审查外国投资的草案。
 <br/>
 <br/>
 路透社报导称，从去年开始，欧盟国家已经成为中共投资的第一大目标。法国、德国及意大利等欧盟国家政府，对在欧洲市场越来越活跃的外国投资（尤其是中共的投资）感到担忧，特别是在高科技领域。这些国家要求欧盟立法，对外国投资进行限制。
 <br/>
 <br/>
 欧洲议会议员普劳斯特（Frank Proust）曾说：〝在这个全球化时代，是时候显示欧洲不那么天真了。我们不反对外国投资，但是反对不正常的投资。〞
 <br/>
 <br/>
 除了欧盟，最近几个月，澳大利亚、日本和加拿大等国家出于对国家安全的考虑，都加入了反对中共投资扩张的行列。受到这些发达国家的抵制后，中共在海外投资骤降。
 <br/>
 <br/>
 根据联合国贸易和发展会议汇编的数据，今年中共在全球的对外直接投资，自2002年以来首次下降至1,246亿美元，而2016年中共对外投资曾高达1,961.5亿美元。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
