### 人工智能竞争 道德问题突显美中差异
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】最近，中共招募
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高中生.html" target="_blank">
  高中生
 </a>
 接受培训，研发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人工智能.html" target="_blank">
  人工智能
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武器杀手.html" target="_blank">
  武器杀手
 </a>
 ，引发国际争议与担忧。伴随美中之间在该领域竞争加剧，相关的伦理道德立场，双方也突显出差异。
 <br/>
 <br/>
 多年来，美国一直在全球引领
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人工智能.html" target="_blank">
  人工智能
 </a>
 开发与应用，政府业有持续研发投入。但近两年，中共已将人工智能作为经济计划的重要部分，投入巨资、扶植大型企业，招揽专家，试图成为应用领域领导者。美中竞争日趋激烈。
 <br/>
 <br/>
 白宫总统技术政策副助理 Michael Kratsios：〝在总统去年提交国会的预算中，我们特别把人工智能作为研发优先项目。〞
 <br/>
 <br/>
 国防高等研究计划署防卫科学室主任 Valerie Browning：〝(我们)下一个人工智能发展计划，承诺未来五年投入20亿美元，提升人工智能技术。〞
 <br/>
 <br/>
 技术发展也带来道德争议。今年4月，谷歌有4000员工联署抗议，要求公司停止开发有助强化美军无人监视机的人工智能技术(Maven项目)，认为涉及伦理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_道德问题.html" target="_blank">
  道德问题
 </a>
 。谷歌随后决定不与五角大楼就此项目续约，并公布一系列使用人工智能的伦理指导原则。
 <br/>
 <br/>
 美国联邦众议员 人工智能核心小组成员 Ro Khanna：〝我们是民主社会，不象中国。中国基本是政府控制公司，国家所有。他们不在乎公民权利。他们攫取人们的数据，运用技术为所欲为。我们可不想成为中国那样。我们有一个自己的系统，自由的企业，公司有自主权，可以决定他们想做什么。〞
 <br/>
 <br/>
 专家也提出，谷歌这样的跨国公司，在面对专制政权时，往往存在双重标准。比如为重返中国市场，目前谷歌正在为中共建立一个审查版搜索引擎。
 <br/>
 <br/>
 国防分析研究所专家 David Sparrow：〝我感到震惊，谷歌员工对Maven项目的反感度，远超过他们对中共大规模监控状况的反感。〞
 <br/>
 <br/>
 最近，中共招募
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高中生.html" target="_blank">
  高中生
 </a>
 接受培训，研发人工智能
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武器杀手.html" target="_blank">
  武器杀手
 </a>
 ，引发国际争议与担忧。
 <br/>
 <br/>
 对于致命性的自主武器，美国军方则有自己的使用原则。
 <br/>
 <br/>
 国防高等研究计划署防卫科学室主任 Valerie Browning：〝需要指出，按照指导原则，自动化机器和人工智能，不能在无人介入的情况下使用，在参与致命性的行动时。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
