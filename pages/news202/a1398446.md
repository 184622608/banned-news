### 菲总统鼓励警员杀死贩毒上司：给奖金免费游香港
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】本周二，菲律宾现任总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜特尔特.html" target="_blank">
  杜特尔特
 </a>
 (Rodrigo Duterte)在一个打击毒品讲座上发表讲话时称，任何员警杀死贩毒的上司，不但可获得奖金，还可免费到香港旅游。此语再次成为世界媒体报导的热点话题。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜特尔特.html" target="_blank">
  杜特尔特
 </a>
 自从上台以来，一直致力于在菲律宾打击贩毒吸毒，不但对毒贩采取了基本无差别的〝杀无赦〞铁腕政策，还常常发表一些惊人之语，招致西方人权组织的质疑甚至谴责。
 <br/>
 <br/>
 当地时间11月6日，杜特尔特在马拉坎南宫举行的打击毒品及军事化讲座上发表讲话时，提及了涉嫌贩毒的警长拉培兹(Santiago Rapiz) 周一在缉毒行动中被自己部下射杀的事件。
 <br/>
 <br/>
 据ABS-CBN新闻网报导，杜特尔特在讲座上说：〝昨晚一名警长被杀了，一名忍者，我要嘉奖杀死涉及贩毒的警方高层的任何一位员警，我会给你奖金、还招待你去香港旅游。〞
 <br/>
 <br/>
 此前，杜特尔特曾宣布：杀死参与贩毒的警长，可获300万披索的奖金。
 <br/>
 <br/>
 据了解，菲律宾高阶员警拉培兹涉嫌回收扫毒行动中没收的毒品早已被菲警方暗中盯上，11月5日在民答那峨岛（Mindanao）的扫毒行动中，他的部下警员试图逮捕他时，拉培兹进行了反抗，结果被部下当场射杀。
 <br/>
 <br/>
 有媒体报导称，拉培兹之前曾跟2016年自杀身亡的毒枭奥迪塔(Melvin Odicta)合作贩毒，是杜特尔特要整肃的对象之一。而杜特尔特把这些涉及贩毒的警方高层统称为〝忍者〞。
 <br/>
 <br/>
 据报导，杜特尔特在这次座谈会上再次警告贩毒者说：〝如果让我抓到你(贩毒者)，不好意思，结果就是这样。我真的会杀死你，所以我说‘不要破坏我的国家，因为我会杀死你；不要摧毁我们的年幼子女，因为我会杀死你。’〞
 <br/>
 <br/>
 杜特尔特表示，毒品交易是涉及菲律宾国家安全的问题，而不少警员竟然愚蠢地卷入其中，今后还将要求军方协助警方缉毒。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
