### 大篷车移民搭车北上 〝笼车〞也派上用场
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】周日，一大批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲难民.html" target="_blank">
  中美洲难民
 </a>
 在墨西哥的瓜纳华托州搭乘交通工具，意图前往美国圣地亚哥边境，但尚有1600英里的距离。
 <br/>
 <br/>
 这些难民藉助各种交通工具北上，甚至包括运输动物的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_笼车.html" target="_blank">
  笼车
 </a>
 〞。
 <br/>
 <br/>
 墨西哥克雷塔罗州政府之前表示，5771名难民周日离开三处庇护所，启程前往以北的瓜纳华托州。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 主要来自洪都拉斯、危地马拉、萨尔瓦多和尼加拉瓜，其中一些难民表示，他们去美国是为了远离当地的贫穷、黑帮暴力和政府动荡。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统10月31日发推文表示，已派遣军队守卫边境，不允许移民非法入境。而且川普指出，这些大篷车移民中，〝包括一些非常坏的暴徒和黑帮分子〞。
 <br/>
 <br/>
 上周五，川普还发布公告，在接下来约3个月内，暂停给非法越过美墨边境的外国人提供政治庇护申请，除非美国墨西哥间达成相关协议。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
</div>
