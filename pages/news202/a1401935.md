### 抗议不断 法国总理暂停调涨燃油税
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国总理.html" target="_blank">
  法国总理
 </a>
 爱德华・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲利普.html" target="_blank">
  菲利普
 </a>
 ，周二宣布，将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃油税.html" target="_blank">
  燃油税
 </a>
 等三种税收的上涨计划，暂停六个月。这是马克龙政府上台18个月以来，首次巨大转变。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国总理.html" target="_blank">
  法国总理
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲利普.html" target="_blank">
  菲利普
 </a>
 听取各方意见后，周二决定暂停增加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃油税.html" target="_blank">
  燃油税
 </a>
 六个月。
 <br/>
 <br/>
 法国总理爱德华・菲利普：〝我宣布暂停这些调涨税收的措施六个月。〞
 <br/>
 <br/>
 三周以来，成千上万的法国人，在路口、收费口和街道，表达对燃油涨价的愤怒，甚至演变为暴力破坏行动。
 <br/>
 <br/>
 菲利普说，任何人都不能〝视而不见〞。暂停增加燃油税的决定立即生效。政府会利用这段时间，讨论解决方案，帮助依赖汽车上班和购物的工薪阶层。
 <br/>
 <br/>
 法国总理爱德华・菲利普：〝要给我们时间，对数周来人们提出的主要问题，进行切实讨论。〞
 <br/>
 <br/>
 菲利普也指出，民众不能期望，在获得更好公共服务的同时，降低税收。双方必须作出妥协。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
