### 3千大篷车移民到达美墨边境 当地居民示威抗议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】总数大约5500人的中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移.html" target="_blank">
  大篷车移
 </a>
 民跋涉逾一个多月后，近日正陆续抵达美墨边境城市蒂华纳。日前有数百当地居民集会示威，抗议不断到来的非法移民把他们的生活搞得一团糟。与此同时，为防范出现大批移民强行冲过边界进入美国的极端现象发生，美军已在边界拉开数哩长带有倒鈎铁丝的围篱，并关闭了边界入关口的车道。
 <br/>
 <br/>
 近几日，越来越多来自中美洲的大篷车难民横越墨西哥抵达蒂华纳地区(Tijuana)，大批移民在当地许多露天广场扎营驻守，等待排队向美国政府申请移民庇护。
 <br/>
 <br/>
 就在近几天，蒂华纳涌入近3000名移民后，该地区气氛日益紧张。美国联邦政府估计，该地区移民人数很快将暴增至1万人。
 <br/>
 <br/>
 本月18日，大约有500名当地居民在蒂华纳富裕地区集会示威，抗议不断涌入该地区的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移.html" target="_blank">
  大篷车移
 </a>
 民把他们的生活搞得一团糟。
 <br/>
 <br/>
 在抗议现场，集会示威的当地居民不断高唱墨西哥国歌或高呼一些反移民的口号，期间甚至有人对露天驻扎的移民丢石头。
 <br/>
 <center>
  <br/>

   </font>
  </a>
  <br/>
 </center>
 <br/>
 目前已经有数千移民在蒂华纳海边接近美墨边界的友谊公园扎营。大批来自萨尔瓦多、危地马拉或洪都拉斯的移民驻扎在这里，导致当地的暴力事件日渐猖獗。
 <br/>
 <br/>
 墨西哥内政部长Alfonso Navarrete日前通过媒体呼吁墨西哥民众不要把大篷车移民〝视为罪犯〞。
 <br/>
 <br/>
 有大篷车移民在接受媒体采访时表示，他们历经千辛万苦才到达美国边境，而当地居民的示威抗议令他们感到惊惧不安。
 <br/>
 <br/>
 据公开的资讯，美国联邦边关审查人员在蒂华纳进入加州圣地牙哥的入关口，每天只处理100件庇护申请案，而在装满移民的大篷车队抵达前，专门用于登记申请者姓名的本子上，就已登记了超过3000个姓名，而且预计还将有数千移民陆续到达。这意味着，有大批来自中美洲的非法移民，将在边境城市蒂华纳长期滞留等待。
 <br/>
 <center>
  <br/>

   <br/>
   <font size="-1">
    近期，将于总数大约5500人的中美洲大篷车移民经过长途跋涉抵达美墨边境。(GUILLERMO ARIAS/AFP/Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 美国政府曾经表示，美国的整个庇护体系已经饱和，因为在过去五年来，移民申请案暴增20倍，目前仍有70万件积案等待处理。
 <br/>
 <br/>
 为因应大规模大篷车队移民的到来可能带来的潜在威胁，美国政府已明显加强边界安全，关闭了入关口的车道，树立水泥路障，上方布满带刺铁丝网。一旦发生大批移民企图强行通过边界的事件，这些铁丝网能迅速移动，阻挡通路。
 <br/>
 <br/>
 目前，来自肯塔基州诺克斯堡的第19工兵营约100名美军已经在美墨边界拉开数哩长带有倒鈎铁丝的围篱，他们还在德州边境小镇拉雷多（Laredo）河岸布置了齐肩高的折式铁丝网络障。

  <br/>
 </center>
 <br/>
 负责架设铁丝网的科普尼可中尉在接受美国媒体的采访时表示：一些渡河而来的移民会用西语诅咒驻守在那里挡住他们去路的美军，甚至向他们丢掷瓶子，但美国军人们对这些移民的回应就正面得多。
 <br/>
