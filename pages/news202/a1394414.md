### 日75万人观赏土浦烟火赛 烟火坠地爆炸10人伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_茨城县.html" target="_blank">
  茨城县
 </a>
 土浦市6日举行〝土浦全国花火（烟火）竞技大会〞期间，发生烟火升空未点燃竟在坠地时爆炸，造成包括儿童在内的至少10人受伤。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土浦全国花火竞技大会.html" target="_blank">
  土浦全国花火竞技大会
 </a>
 是日本三大烟花火大会之一，今年已办至第87届。本届共有56队烟火队伍参赛，估计约有75万人到场观赛。
 <br/>
 <br/>
 意外事件发生在当地傍晚约6时30分，烟火升空时没有被引燃，结果掉回地面时发生裂爆。导致至少10人受伤，伤者年龄届乎9岁至66岁，大多烧伤及擦伤，已送院接受治疗。
 <br/>
 <br/>
 后来由于事发现场出现强风，大会为霆保观众安全，宣布中止活动。
