### 日本丰洲市场首次开放 民众：不管排多久都想吃到
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】有〝日本的厨房〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东京.html" target="_blank">
  东京
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_筑地市场.html" target="_blank">
  筑地市场
 </a>
 关闭后，数百商户于11日迁移到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丰洲市场.html" target="_blank">
  丰洲市场
 </a>
 ，场内的食店和商店于13日首次开放给一般民众参观，市场内设有通道供民众参观拍卖实况，餐饮店与商店也正式营业，民众一早就在市场外大排长龙，准备尝鲜和参观。
 <br/>
 <br/>
 日本放送协会（NHK）报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丰洲市场.html" target="_blank">
  丰洲市场
 </a>
 于11日开场，目前规定除了周日、日本国定假日及休市日，其他时间从清晨5时到下午5时，都开放一般民众进场参观。场内餐饮店也正式启用，吸引大批人潮前往。
 <br/>
 <br/>
 一般参观不需费用，也不用预约，之后可观看生鲔鱼拍卖时，则是从清晨5时30分左右开始。
 <br/>
 <br/>
 而为为维护水产品卫生，造访的民众需走参观者专用通道，隔着透明玻璃观看中盘商与买家互动的卖场；参观民众也可以站上专供拍照的市场专用小型搬运车，拍摄纪念照。
 <br/>
 <br/>
 今天上午10时(当地时间)，丰洲市场正式开放一般民众进场，许多观光客及带着家人的民众，一同造访丰洲新市场。
 <br/>
 <br/>
 丰洲市场当天有约100家商店营业，包括从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_筑地市场.html" target="_blank">
  筑地市场
 </a>
 迁至的吉野家牛肉饭总店、寿司店、小丸子店等，不少外国游客前来大快朵颐。
 <br/>
 <br/>
 一名来自香港的28岁陈姓游客指，丰洲市场是一栋栋的建筑物，与吵吵闹闹的筑地很不同，但在批发市场内吃到的寿司新鲜度没有差别，并称，不管排多久时间都想吃到。
 <br/>
 <br/>
 住在丰洲当地的20岁男学生说，今天就是来看看丰洲市场，如果能因为丰洲市场吸引更多人到丰洲，让这里更热闹就好了。
 <br/>
 <br/>

 <br/>
 据了解，丰洲市场比筑地大了1.7倍，依照类别分为三大区块，封闭型设计，温度卫生都好管理，但因为新市场面积太大，有业者迷路误闯禁区，也有人因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞车.html" target="_blank">
  塞车
 </a>
 而苦恼 。
 <br/>
 <br/>
 有货车排了2小时左右仍无法进入停车场，运输业者形容〝状况糟糕〞。日本民众 ：〝平常差不多45分钟可以到，但今天花了一个半小时才到，真的很塞。〞
 <br/>
 <br/>
 外界担心，新市场的交通便利度不足以应付流量。
