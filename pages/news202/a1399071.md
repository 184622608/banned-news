### 【禁闻】美老兵节游行 警察为法轮功队伍喝采
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】
 <b>
  依法为
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
   法轮功
  </a>
  做无罪辩护 律师
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遭打压.html" target="_blank">
   遭打压
  </a>
 </b>
 <br/>
 <br/>
 大陆律师胡林政最近遭湖南当局处罚，被停止执业6个月。理由是他去年在云南做了一场无罪辩护。
 <br/>
 <br/>
 长沙市司法局11月5号网站刊登的处罚决定书说，2017年12月1号，胡林政作为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员张某的律师，在云南省红河州开远市法院的法庭上辩护说，〝没有证据证明法轮功是x教，张某信仰法轮功是宪法36条赋予的宗教自由，限制法轮功信仰是违法行为。〞
 <br/>
 <br/>
 长沙市司法局指称他的这些言论，否定了中共认定法轮功是x教组织的性质，扰乱了法庭秩序。
 <br/>
 <br/>
 但是多名大陆律师表示，在现行的中共法律文件中，没有一个明确提出法轮功是x教。而宪法第三十六条规定：〝中华人民共和国公民有宗教信仰自由。任何国家机关、社会团体和个人不得强制公民信仰宗教或者不信仰宗教，不得歧视信仰宗教的公民和不信仰宗教的公民。〞
 <br/>
 <br/>
 大陆知名维权律师谢燕益表示，胡林政律师没有违法，他是严格按照宪法和法律在依法执业。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美老兵节游行.html" target="_blank">
   美老兵节游行
  </a>
  警察为法轮功队伍喝采
 </b>
 <br/>
 <br/>
 11月11号，美国旧金山法轮功学员应邀参加当地的第九十八届老兵节游行（Veterans'Day Parade），他们的队伍不仅受到沿途观众的欢迎，连维持秩序的交通警察都喝采鼓掌。
 <br/>
 <br/>
 据〝法轮大法明慧网〞报导，游行在旧金山最著名的旅游景点渔人码头举行，观众有来自世界各地的游人，也有当地居民。几乎所有看到游行的人们都表示，法轮功天国乐团的队伍让他们印象深刻。
 <br/>
 <br/>
 一名交通警察也陶醉在天国乐团演奏的乐曲中，不停地为法轮功的队伍鼓掌。他说：〝我喜欢这支队伍（天国乐团），非常棒。我希望中国人能够享有自由。〞
 <br/>
 <br/>
 游行主办者也表示：〝法轮功的队伍能来参加游行，我感到荣幸。〞
 <br/>
 <br/>
 <b>
  大陆知名异见人士
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭泉.html" target="_blank">
   郭泉
  </a>
  刑满出狱
 </b>
 <br/>
 <br/>
 大陆知名异见人士、南京师范大学前副教授郭泉，11月12号刑满获释，但家属被禁止去监狱迎接他。
 <br/>
 <br/>
 郭泉曾参与军转干部维权，2007年因发表公开信要求中国进行宪政改革，宣布组建〝中国新民党〞，主张在中国实行多党制和军队国家化等，遭当局重点打压，被学校撤销副教授职务，后被以涉嫌〝颠覆国家政权罪〞拘捕，判刑10年。
 <br/>
 <br/>
 最近几天，一些网友在海外社交媒体推特上发文，对他表示敬意，并欢迎他回家。
 <br/>
 <br/>
 编辑/周玉林
