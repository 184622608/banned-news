### 巴西里约土石流 民众睡梦中被埋10死多人失踪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月11日讯】巴西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里约热内卢.html" target="_blank">
  里约热内卢
 </a>
 都会地区10日凌晨发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土石流.html" target="_blank">
  土石流
 </a>
 ，多栋民宅遭毁损，居民在睡梦中被埋或受困，已知至少10人死亡，11人受伤送医，另有多人下落不明。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里约热内卢.html" target="_blank">
  里约热内卢
 </a>
 尼特洛伊（Niteroi）地区连日下雨，近日因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土质松动.html" target="_blank">
  土质松动
 </a>
 ，一块大石从好望山丘（Morro da Boa Esperanca）顶滚落，压毁6栋民宅。
 <br/>
 <br/>
 当地居民协会表示，基于安全考量，其中几栋民宅，之前已经由里约民防厅下令查封，但屋主们拒绝离开，不料，10日凌晨再发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土石流.html" target="_blank">
  土石流
 </a>
 。
 <br/>
 <br/>
 尼特洛伊市长尼佛斯（Rodrigo Neves）表示，这次遭土石流浸袭的地区虽为山坡地，但不属高地质灾害风险区，并非因边坡滑动造成坍方，而是岩石破裂分离下坠造成的悲剧。
 <br/>
 <br/>
 里约民防厅指出，警方已救出11名伤者送往附近卫生所就医，但还有多人下落不明。
 <br/>
 <br/>
 民防厅长朱尼尔（Roberto Costa Junior）表示，清除瓦砾、援救灾民和寻找生还者的工作将持续进行。许多民众也自发的加入清除瓦砾工作，组成人龙一桶一桶将瓦砾运出。
 <br/>
 <br/>
 2010年4月，尼特洛伊被称为〝垃圾山〞的本巴山丘（Morro do Bumba）亦发生严重土石流，当时约50栋民房瞬间被烂泥砂石掩埋，造成200多人丧生。
 <br/>
 <br/>
 尼特洛伊市政府指出，自2013年以来已斥资巴西币逾1500万元，在市内50处地点进行边坡崩塌防治工程。
