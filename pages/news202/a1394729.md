### 中国大妈墨尔本扮红卫兵跳文革舞 网友:邪灵附体
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】近日，有网友在网络上曝光了一段视频，显示一群
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国大妈.html" target="_blank">
  中国大妈
 </a>
 在澳洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨尔本.html" target="_blank">
  墨尔本
 </a>
 装扮成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文革.html" target="_blank">
  文革
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红卫兵.html" target="_blank">
  红卫兵
 </a>
 〞的模样，在公众场合嘴里念念有词地大跳文革舞。这群大妈的这种丑态被中国网友评论为〝邪灵附体〞、〝群魔乱舞〞。
 <br/>
 <br/>
 据中国网友在推特上发出的推文，今年9月30日，一群
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国大妈.html" target="_blank">
  中国大妈
 </a>
 为了庆贺中共10月1日的建政纪念日而聚集到澳洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨尔本.html" target="_blank">
  墨尔本
 </a>
 ，她们一个个身穿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文革.html" target="_blank">
  文革
 </a>
 时期
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红卫兵.html" target="_blank">
  红卫兵
 </a>
 的衣着，在公众场所跟随着现场音乐而一边手舞足蹈，一边嘴里还囔念著文革红歌的歌词。有在旁边围观的人拍摄下了这个奇怪的场面。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-10/p9072751a968394561.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-10/p9072751a968394561-ss.jpg"/>
   <br/>
   <font size="-1">
    中国大妈9月30日聚集澳洲墨尔本，装扮成中共文革时期的红卫兵，一边手舞足蹈跳文革舞，一边嘴里还念念有词的读歌词，被中国网友评论为〝邪灵附体〞、〝群魔乱舞〞。（图片来源：推特）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 有网友在推特发帖指出，澳洲墨尔本近些年遭到中共极左势力的严重入侵，那里总有一些极左群体动辄就要出来闹一闹。2017年9月9日这些人曾经预谋举办大规模〝纪念毛泽东活动〞，但遭澳洲政府阻止。不料今年十一前又跑出来跳文革舞，〝令当地民众相当傻眼〞。
 <br/>
 <br/>
 许多华人网友看完影片后表示很反感或十分不屑，纷纷评论称〝群魔乱舞〞、〝令人作呕〞、〝给文革招魂〞。
 <br/>
 <br/>
 有网友留言称，〝这些人是马屁精，没灵魂而已。〞
 <br/>
 <br/>
 也有网友进一步指出，〝共产党是邪教，是邪灵附体。〞
 <br/>
 <br/>
 还有网友感叹说：〝文革余孽还没有除净，让这些垃圾又出来现眼了。〞
 <br/>
 <br/>
 （记者李明报导/责任编辑：秦锋）
