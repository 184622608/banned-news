### 意大利高赤字预算送审 欧盟评判陷两难
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 政府周一通过2019年预算案，财政赤字升至GDP的2.4%，和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 政策背道而驰。这不仅让欧盟陷入两难境地，也让市场担心，意大利债务是否会再次引发欧债危机。
 <br/>
 <br/>
 周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 将今后三年的财政预算案提交到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 ，内容包括增加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福利.html" target="_blank">
  福利
 </a>
 ，部分减税，并降低退休年龄，财政赤字提高到GDP的2.4%。
 <br/>
 <br/>
 这虽然低于欧盟赤字上限3%，但高于意大利今年的1.8%。此举也是今年新上台的意大利政府兑现选战承诺。
 <br/>
 <br/>
 意大利总理孔特周二表示,〝财政紧缩的道路不能再走下去〞。他对扩张性预算〝感到自豪〞，并希望欧盟〝不带偏见〞地和意大利对话。
 <br/>
 <br/>
 但是欧盟一直要求负债累累的成员国降低赤字。意大利2017年总债务额高达近2.7万亿美元，相当于GDP的132%，超过欧盟规定上限60%的两倍多。
 <br/>
 <br/>
 9月底预算案刚出台时，欧盟委员会就严厉警告意大利，必须控制公共支出。但意大利并没有修正赤字。
 <br/>
 <br/>
 欧盟委员现在有两周时间审查预算案，预计可能会打回修改。
 <br/>
 <br/>
 对该预算，意大利民众褒贬不一。
 <br/>
 <br/>
 罗马居民 Domenico Ciucci：〝说到底，我认为（欧盟）在攻击意大利，没有仔细分析（意大利的）预算案。〞
 <br/>
 <br/>
 罗马居民 Laura Cossettini：〝我们有一个幼稚的业余政府。〞
 <br/>
 <br/>
 连线本台欧洲记者王羽音：〝欧盟如果拒绝预算案，意大利以后会对欧盟采取更不合作的态度。意大利执政两党上台前是反欧盟的，上台以后才有所缓和。但是同意大利的预算案就更糟糕了。意大利债务有可能飙升。2010年希腊的债务引发了欧债危机，当时希腊债务达到3200多亿欧元。意大利的经济规模是希腊的10倍，债务总额是当时希腊的7倍多。如果意大利出现债务违约，经济崩溃，会给世界经济造成严重后果。〞
 <br/>
 <br/>
 新唐人电视综合报导
</div>
