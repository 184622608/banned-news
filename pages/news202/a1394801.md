### 中共〝航母杀手〞自身难保  俄专家揭内幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东风21D导弹.html" target="_blank">
  东风21D导弹
 </a>
 一直被中共吹嘘为攻击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军航母.html" target="_blank">
  美军航母
 </a>
 的杀手。但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 媒体最近刊文披露，中共从俄罗斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_购买.html" target="_blank">
  购买
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏-35战机.html" target="_blank">
  苏-35战机
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_S-400防空导弹.html" target="_blank">
  S-400防空导弹
 </a>
 ，是用来保护东风21D导弹的，而该导弹有一个短板，很难应对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巡航导弹.html" target="_blank">
  巡航导弹
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无人飞机.html" target="_blank">
  无人飞机
 </a>
 的攻击。
 <br/>
 <br/>
 据称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东风21D导弹.html" target="_blank">
  东风21D导弹
 </a>
 是中共专门为应对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军航母.html" target="_blank">
  美军航母
 </a>
 威胁而研制，射程约为3,000千米（1,900英里）。已于2009年服役。截至到2014年10月，东风21D型已经在中国东南和东北地区部署了两个旅。
 <br/>
 <br/>
 东风21D的特点，中共官媒形容，〝反应速度快、慑控范围广、突防能力强、寻跟目标准、打击精度高〞。官媒称之为中共攻击美军航母的杀手鐧。
 <br/>
 <br/>
 然而，美国之音10月9日引述
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 主要大报〝消息报〞最近发表的一篇文章说，中共在远程防空导弹和重型歼击机领域，仍然落后于俄罗斯和美国。
 <br/>
 <br/>
 这篇由军事历史学家波尔坚科夫所撰写的文章说，南中国海和台湾海峡，成为未来中美两国爆发潜在军事冲突的地区。为了应对美国海军航母威胁，中共成功开发出了世界上第一款主要用来攻击美国航母和大型水面舰艇的反舰弹道导弹东风21D。
 <br/>
 <br/>
 这种可移动式的反舰弹道导弹，目前成为中共岸防力量的主力。
 <br/>
 <br/>
 但东风21D有一个短板，很难应对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巡航导弹.html" target="_blank">
  巡航导弹
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无人飞机.html" target="_blank">
  无人飞机
 </a>
 的攻击。而美军从1991年起就在积累对付移动式导弹发射系统的经验。
 <br/>
 <br/>
 文章说，中共在沿海地区成功组建了由远程预警雷达系统，重型远程歼击机和远程防空导弹系统所构成的防空防御体系，但中共在军工科技方面进展不顺利，这成为中共从俄罗斯采购
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_S-400防空导弹.html" target="_blank">
  S-400防空导弹
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏-35战机.html" target="_blank">
  苏-35战机
 </a>
 的主要原因。
 <br/>
 <br/>
 S-400防空导弹和苏-35战机是最近几年中俄之间最大的两笔军火交易，同时也深受外界关注。
 <br/>
 <br/>
 文章披露，中共采购上述两种武器，被用来保护中国主要用来攻击美国航母的东风21D反舰弹道导弹。
 <br/>
 <br/>
 S-400防空导弹的最大优势，能很快部署和发射。既可拦截飞机，也能拦截弹道导弹，S-400系统中，最新式导弹的最远射程能达到400公里，能拦截60公里远的弹道导弹目标。能轻而易举的为准备发射的东风21D反舰导弹提供有效保护。
 <br/>
 <br/>
 俄罗斯从2018年起向中共交货S-400防空导弹；中共此前从俄罗斯采购的24架苏-35战机中，目前已经有14架到货，全部被部署在离台湾和南中国海不远的广东湛江地区。
 <br/>
 <br/>
 中共军队的苏-35战机伴随轰-6K等战机，今年春季环绕台湾岛飞行，以及在南中国海地区巡逻后，俄罗斯一些主要军事媒体立刻转发大量图片对此报导分析。
 <br/>
 <br/>
 美国因中共从俄罗斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_购买.html" target="_blank">
  购买
 </a>
 先进武器，9月20日公布对中共军方实施制裁。
 <br/>
 <br/>
 美国国务院称，中共于2017到2018年购买了10架俄罗斯的苏-35战机和S-400地对空导弹的相关设备。这批军购违反了美国2017年通过的《以制裁反击美国敌人法案》。这是美国第一次利用这部制裁法律惩罚俄罗斯军事装备的外国买家。
 <br/>
 <br/>
 俄媒称，为了避开美国制裁，中俄双方以后也可能会使用两国本国货币结算。
 <br/>
 <br/>
 俄罗斯军事评论人士德罗兹坚科说，中共仍然无法摆脱在尖端武器领域对俄罗斯的依赖。两国的军事技术合作将持续下去。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
