### 抗衡中共 澳洲斥资15亿建太平洋基金
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 总理莫里森宣布，将为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太平洋国家.html" target="_blank">
  太平洋国家
 </a>
 的基础建设，提供20亿澳元，相当于15亿美元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资.html" target="_blank">
  投资
 </a>
 ，以遏制中共在太平洋地区的影响。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 总理莫里森表示，20亿澳元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资.html" target="_blank">
  投资
 </a>
 将用于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太平洋国家.html" target="_blank">
  太平洋国家
 </a>
 和东帝汶的基础设施建设。
 <br/>
 <br/>
 澳大利亚总理莫里森：〝我们不仅是自愿结成的伙伴，还是‘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太平洋家庭.html" target="_blank">
  太平洋家庭
 </a>
 ’成员，从而联系在一起。〞
 <br/>
 <br/>
 莫里森还说，澳大利亚不能把自己在西南太平洋地区的优势，看作理所应当。
 <br/>
 <br/>
 澳大利亚总理莫里森：〝我相信，‘太平洋家庭’的关系应该展开新篇章了。〞
 <br/>
 <br/>
 太平洋岛国众多，拥有大量海洋资源。中共2011年以来，向太平洋国家提供了13亿美元优惠贷款和援助，成为仅次于澳大利亚的第二大援助国。
 <br/>
 <br/>
 西方国家担心，北京输出的大量债务，会迫使太平洋小国在主权和政治上妥协。
 <br/>
 <br/>
 澳大利亚总理莫里森：〝我们需确保，我们的太平洋伙伴关系，随着时间推移，变得更加强大。〞
 <br/>
 <br/>
 今年以来，澳洲情报与安全机构对中共在太平洋地区过于活跃，表示担忧。澳洲还承诺，将协助巴布亚新几内亚建立新海军基地，遏制中共在该国开发工程。
 <br/>
 <br/>
 新唐人记者毕心慈、明玉综合报导
</div>
