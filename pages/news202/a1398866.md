### 纳瓦罗：川普主导谈判 华尔街勿干涉
------------------------

<div class="wysiwyg">
 【新唐人2018年11月10日讯】周五，白宫贸易高级顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 在华府智库〝战略与国际研究中心〞发表演说，警告
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华尔街.html" target="_blank">
  华尔街
 </a>
 ，不要干涉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府的贸易政策、不要做中共在美国的外国代理人。
 <br/>
 <br/>
 白宫贸易高级顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 警告
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华尔街.html" target="_blank">
  华尔街
 </a>
 ，不要干涉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 施政。
 <br/>
 <br/>
 白宫贸易高级顾问 纳瓦罗：〝华尔街，远离这些谈判。我敦促这些未注册的外国代理人不要干涉这些事情。在任何时候达成的任何交易，都将是川普要的条款，而不是华尔街的条款。〞
 <br/>
 <br/>
 纳瓦罗批评华尔街高管，在川普和习近平G20峰会前，对川普政府施压、期望其缓和对华贸易政策并达成协议。
 <br/>
 <br/>
 白宫贸易高级顾问 纳瓦罗：〝全球的百万富翁是中国政府实施影响力的一部分，在阿根廷G20峰会前全面施压白宫，未注册外国代理人的使命，他们就是未注册的外国代理人，就是施压这届总统以达成某些协议。〞
 <br/>
 <br/>
 美国前财长、原高盛集团主席保尔森本周在新加坡公开发言，指责川普正在美国与中国之间竖起新的〝经济铁幕〞、破坏全球化。
 <br/>
 <br/>
 纳瓦罗劝告其不要干涉美国贸易谈判代表的工作。
 <br/>
 <br/>
 白宫贸易高级顾问 纳瓦罗：〝我敦促这些未注册的外国代理人不要干涉这些事情。我们只有一个贸易谈判代表，那就是莱特希泽。你们能跟川普和莱特希泽站在一起这很好，但谈判不是出自于这两人时就不对了。〞
 <br/>
 <br/>
 纳瓦罗说，川普总统具有勇气和智慧，为全球被不公平贸易侵害的国家发声，不需要华尔街的干预。
 <br/>
 <br/>
 白宫贸易高级顾问 纳瓦罗：〝想想看华尔街银行家和对冲基金经理人所运用的美中〝穿梭外交〞策略。美国总统不需要这种穿梭外交政策。当这些未获（中共）支付任何报酬的外国代理人，从事这种所谓的外交活动时，他们的所作所为都只是在削弱川普总统及其谈判地位，完全没有好处。〞
 <br/>
 <br/>
 纳瓦罗表示，川普希望中国经济能够进行更深层次的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性变革.html" target="_blank">
  结构性变革
 </a>
 〞，开发市场，与美国公平贸易。
 <br/>
 <br/>
 新唐人记者李兰、王凯迪连线报导
</div>
