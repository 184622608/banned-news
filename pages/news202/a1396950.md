### 北约冷战后最大军演 31国参加 俄不满
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北约.html" target="_blank">
  北约
 </a>
 10月25日展开为期两周冷战后最大规模军事演习。所有29个成员国及两个伙伴国家芬兰和瑞典都派员参加。这次军演被外界视为是北约针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 的抗衡。
 <br/>
 <br/>
 65艘舰船、250架战机、1万辆军车和5万人参加了名为〝三叉戟2018〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北约军演.html" target="_blank">
  北约军演
 </a>
 ，其中也包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 航空母舰杜鲁门号战斗群。
 <br/>
 <br/>
 军演为期两周，地点在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_挪威.html" target="_blank">
  挪威
 </a>
 及其附近海域。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北约.html" target="_blank">
  北约
 </a>
 海军副司令Herve Blejean:〝三叉戟演习汇集了大量舰船，内部的作战协调，各兵种配合，信息管理，这些对海军单独行动的船隻来讲都是新内容。〞
 <br/>
 <br/>
 北约说，演习是预防性的行动，不是挑衅。并向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 发出了观察邀请。但俄罗斯仍然表示不满。
 <br/>
 <br/>
 外界认为，这次北约军演是为了和俄罗斯今年9月的大规模军演抗衡。俄罗斯出动了将近30万人、3万6千部车辆和80艘战舰，还邀请了中国参加。
 <br/>
 <br/>
 2014年俄罗斯吞并了属于乌克兰的克里米亚地区以后，一些北约国家担心，俄罗斯也会蚕食他们的领土。
 <br/>
 <br/>
 作为回应，过去四年里，北约加强了东欧和波罗的海地区的军力。
 <br/>
 <br/>
 2017年美军在波兰设立了欧洲新总部。
 <br/>
 <br/>
 和俄罗斯接壤的挪威不仅购买了美国的F-35隐形战机，而且还说服美国把驻扎在挪威的美国海军陆战队人数增加一倍以上。
 <br/>
 <br/>
 连线本台欧洲记者王羽音：〝美国是北约的主导力量。周五，也就是军演第二天，美国正式邀请俄罗斯总统普京访问美国。同时美国也表示，正在考虑是否对俄罗斯进行制裁，因为俄罗斯在英国给前间谍下毒。〞〝一方面展示实力，一方面进行制裁，另一方面则是展开对话，这是川普经常使用的办法。就看这次对俄罗斯是否能起到作用了。〞
 <br/>
 <br/>
 新唐人电视综合报导
</div>
