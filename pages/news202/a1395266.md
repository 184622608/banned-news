### 法国皇后神隐珍宝 珍藏200年重现苏富比
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】200多年前的稀世珍宝，居然在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏富比.html" target="_blank">
  苏富比
 </a>
 拍卖场上重见天日。美国纽约12日在苏富比拍卖场，展出了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_皇后.html" target="_blank">
  皇后
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_玛丽安东妮.html" target="_blank">
  玛丽安东妮
 </a>
 的七件珍藏首饰，将在下个月正式拍卖。苏富比估计拍卖价格可望超过300万美金，一起来欣赏百年珍宝风采。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_皇后.html" target="_blank">
  皇后
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_玛丽安东妮.html" target="_blank">
  玛丽安东妮
 </a>
 （Marie Antoinette）珍藏的七件首饰，沈寂两百多年后再现世界舞台，包括49克拉的天然珍珠钻石吊坠，这颗罕见的巨型天然珍珠长26公厘、宽18公厘，单单这一件首饰就价值至少100万美金。此外还有119颗天然珍珠项炼，和一对巨型珍珠耳环等等。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏富比.html" target="_blank">
  苏富比
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠宝.html" target="_blank">
  珠宝
 </a>
 拍卖副总裁 艾佛瑞特（Frank Everett）见过的稀世珍宝多不胜数，最令他无法置信的是，这样的稀世珍宝居然被珍藏两百多年，一直没有过手他人。
 <br/>
 <br/>
 苏富比珠宝拍卖副总裁 艾佛瑞特：〝这项收藏从她（玛丽安东妮）开始就一直在他们家族手里。所以有纪录显示她打包这些珠宝，在她入狱的前夕。她最后自由的一夜照顾了这些珠宝并将它们送走，首先我认为是比利时然后奥地利，她的故乡。她的大女儿是唯一幸存下来的孩子，最终还是收回了它们，从此以后一直在她们家族里。这太令人震惊了。所以我们不只有玛丽安东妮的出处，它也来自这个家庭。几个世纪以来它一直没有易手，我仍然难以置信。〞
 <br/>
 <br/>
 根据珠宝本身的价值，苏富比估价在150到300万美元之间。但是考量到超过两百年的皇室历史价值，苏富比珠宝拍卖副总裁艾佛瑞特认为可能会出现更高的价格。
 <br/>
 <br/>
 苏富比珠宝拍卖副总裁 艾佛瑞特：〝我们依照珠宝的价格预估，我们就可以知道从哪里开始。我想玛丽安东妮依然有相当的吸引力，至今已被她珍藏超过两百年。然后我想我们将可以看到一些强势的价格。〞
 <br/>
 <br/>
 玛丽安东妮是法国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_路易十六.html" target="_blank">
  路易十六
 </a>
 的皇后，在38岁的芳龄因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国大革命.html" target="_blank">
  法国大革命
 </a>
 和国王双双被送上断头台。艾佛瑞特表示，考量到玛丽安东妮的历史地位，以及她对近代服装流行美学的影响，这次拍卖可望能吸引到珠宝收藏家以外的买主。
