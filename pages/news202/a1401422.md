### 中企利诱洩三星技术 韩9人被检方起诉
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】中国企业以重金为诱饵窃取外国商业机密又出一实例。韩国检方周四（29日）对9名涉嫌向中国公司洩露
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三星.html" target="_blank">
  三星
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_OLED曲面屏技术.html" target="_blank">
  OLED曲面屏技术
 </a>
 的工作人员提起诉讼，另有2人因人在中国，暂时未将其起诉。
 <br/>
 <br/>
 路透社报导，韩国检方周四称，9名韩国人因涉嫌将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三星.html" target="_blank">
  三星
 </a>
 电子(005930.KS)的柔性显示(flexible display)技术泄露给中国一家公司，近日被检方起诉。
 <br/>
 <br/>
 水原地区检察官办公室指控，供应商Toptec Co Ltd(108230.KQ)的执行长B某以及8名员工在今年稍早出售有关三星有机发光二极体(OLED)显示面板的信息给中国一家公司。
 <br/>
 <br/>
 检方透露，B某经营的公司与三星有着30多年的合作关系，一度是生产三星手机面板等自动化设备的中坚企业，但是后来利润开始下滑。
 <br/>
 <br/>
 诉状中称，他们成立了一家独立的空壳公司，在获取三星的使用设备和屏幕图纸后，将其洩露给这家空壳公司，再转售给中共公司，共谋利155亿韩元（约1,380万美元）。期间他们还使用假名、虚假的联系方式和个人邮箱以掩人耳目。
 <br/>
 <br/>
 据悉，三星的这项
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_OLED曲面屏技术.html" target="_blank">
  OLED曲面屏技术
 </a>
 在三星最新款的galaxy note9等高端手机上使用，是受韩国工业技术保护法保护的国家核心技术。三星为开发该技术，组建了一个38人的技术团队，投资额达1,500亿韩元（约1.34亿美元），费时6年才研究成功。
 <br/>
 <br/>
 韩国检方有关人士告诉韩联社，因B某等人犯下的罪行，让中国公司省下数年的时间及大笔金钱，轻而易举就获得与三星电子产品一样质量水平的生产线。但Toptec公司否认有不当行为。
 <br/>
 <br/>
 另外，将三星技术盗走的中国公司两名职员虽然也涉嫌参与技术洩露，但因人在中国，很难调查，所以未将其起诉。
 <br/>
 <br/>
 <h4>
  中共企业偷窃韩企技术手段繁多
 </h4>
 <br/>
 大纪元报导，以三星为代表的韩国企业拳头产业的先进技术，屡遭中共洗劫，中共采取的手段也是无所不用其极。
 <br/>
 <br/>
 除了以重金作为诱饵，收买韩企工作人员帮助窃取技术，近几年，中共公司还不断以高薪等作为诱饵招揽韩国技术人员，以获取核心技术，导致韩国半导体、面板等领域的技术人才大批跳槽到中共公司。
 <br/>
 <br/>
 韩媒《每日经济》今年6月引述韩国电子业界的一相关人士的话报导，预计目前中共的京东方中有一百多名韩国技术人才，半数来自三星电子，半数来自LG电子及SK海力士。
 <br/>
 <br/>
 据报导，中企夺取韩国产业技术的手段还包括，派遣隶属中方的间谍直接介入；中企职员伪装成在其韩国竞争企业的合作企业就业，之后接近核心技术后窃取；利用国际展示会，接近在国际展示会上发布的韩国尖端技术，利诱展示的负责人以及试图对其网攻等。
 <br/>
 <br/>
 <h4>
  美国全面打击
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共商业间谍.html" target="_blank">
   中共商业间谍
  </a>
 </h4>
 <br/>
 面对中共对它国的产业核心技术巧取豪夺，目前世界各国纷纷产生警惕，美国在寻求公正交易的贸易战下，又开始重拳打击中共商业间谍。
 <br/>
 <br/>
 近来，除了起诉中共情报机构官员、间谍及黑客，美国政府还起诉了涉嫌盗窃美国公司商业机密的中国福建晋华公司，并对其下达〝禁售令〞，令该公司处于全面瘫痪状态，此举被外界认为对其它中企震慑力相当大。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
