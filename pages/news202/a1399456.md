### 俄罗斯堪察加半岛附近海域6级地震
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】据美国地质调查局地震信息网消息，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 远东地区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_堪察加半岛.html" target="_blank">
  堪察加半岛
 </a>
 附近海域，11月15日发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_6级地震.html" target="_blank">
  6级地震
 </a>
 。
 <br/>
 <br/>
 地震发生在当天清晨，震中位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_堪察加半岛.html" target="_blank">
  堪察加半岛
 </a>
 东部海域，震源深度７６.２公里。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 堪察加边疆区部分地区有明显震感。
 <br/>
 <br/>
 俄罗斯堪察加边疆区紧急情况总局表示，这次地震没有造成人员伤亡或建筑物损毁，也没有引发海啸。
 <br/>
 <br/>
 新唐人记者陈依心、天羽综合报导
