### 印尼锁定失事客机坐标  已找到49袋遗骸
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】周三，印尼军方表示，找到疑似失事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_JT610航班.html" target="_blank">
  JT610航班
 </a>
 的部分机身。目前，搜救人员已经找到49袋遗骸，并交给警方进行身份鉴定。
 <br/>
 <br/>
 一位印尼军方官员周三表示，搜救人员在水下32米处，发现了一个长约22米的物体，疑似为周一失事的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮航客机.html" target="_blank">
  狮航客机
 </a>
 机身。
 <br/>
 <br/>
 印尼军方长官  Hadi Tjahjanto：〝今早我从搜救机构得知，他们相信已经找到 （失事的）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_JT610航班.html" target="_blank">
  JT610航班
 </a>
 客机的部分机身，但还有待进一步确认。〞
 <br/>
 <br/>
 印尼海军表示，已经锁定了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_机身坐标.html" target="_blank">
  机身坐标
 </a>
 ，将在目标海域进行搜寻。
 <br/>
 <br/>
 目前，搜救工作仍在继续进行。印尼当局表示，已经找到了49袋遗骸。由于机身在冲击海面时解体，许多尸体残缺不全。
 <br/>
 <br/>
 印尼官方目前正在鉴定遇难者身份。
 <br/>
 <br/>
 印尼国家搜救署公共准备部门负责人 Didi Hamzar：〝我们找到了49袋遗骸，已经把它们交给印尼警方的灾难遇难者鉴别单位。〞
 <br/>
 <br/>
 同一天，失事航班乘客的家属抵达雅加达北部的一処港口，从海上找到的物品中认领亲人的财物。
 <br/>
 <br/>
 印尼官方已经在一天前表示，乘客生还的希望十分渺茫。不过，一些乘客家属仍然盼望着奇迹的发生。
 <br/>
 <br/>
 失事客机家属 FINANDA AIRUBABA：〝我们希望会有奇迹发生。我确信我的父亲还活着，他一直是个很坚强的人。〞
 <br/>
 <br/>
 目前，已有至少151名乘客家属提供了DNA样本。印尼官方至今没有公布任何一名遇难者的身份。
 <br/>
 <br/>
 <br/>
 新唐人记者林宇、天羽综合报导
 <br/>
