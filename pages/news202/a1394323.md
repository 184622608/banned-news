### 3天4国行程紧凑 蓬佩奥展开充满变数亚洲行
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 5日展开亚洲行，将出访日本、韩国、朝鲜后，预计8日访问北京。稍早，已对外公布会晤名单，但唯独中国没有说明。有分析表示，由于近期中美贸易纠纷有越演越烈的趋势，再加上美国副总统彭斯才强烈批评中共，蓬佩奥访华的行程可能会出现变数。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 （Mike Pmopeo）周五出发展开3天的亚洲行。6日先抵达日本，与首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 和外相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_河野太郎.html" target="_blank">
  河野太郎
 </a>
 会面。7日(周日)转到朝鲜访问，与朝鲜领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 会面，同日再到访韩国，分别与总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文在寅.html" target="_blank">
  文在寅
 </a>
 和外长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康京和.html" target="_blank">
  康京和
 </a>
 会面。
 <br/>
 <br/>
 8日(周一)，蓬佩奥到中国访问，目前尚不清楚蓬佩奥到北京会与何人见面，若依循过往，他应该会与中共国务委员兼外交部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 及中央政治局委员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨洁篪.html" target="_blank">
  杨洁篪
 </a>
 会面。
 <br/>
 <br/>
 中方官媒稍早引述中共外交部发言人表示，两国官员会就双方〝共同关注的议题交换意见〞，但同样没有透露蓬佩奥预定会跟哪些中共官员见面。
 <br/>
 <br/>
 稍早，蓬佩奥谈及此行目的时说，这趟最主要目的在与朝鲜就非核化议题取得更深入的进展。但他不愿说明具体的谈判内容，仅表示会确保双方能了解对方真正想要实现的目标。
 <br/>
 <br/>
 蓬佩奥还说，此行也将触及美国总统川普与金正恩的二次峰会，希望能敲定川金二会的时间与地点。但他坦言并不容易，因为峰会细节十分复杂，停留的时间可能也不足以取得成果。
 <br/>
 <br/>
 这是蓬佩奥半年多来第4度前往平壤，而此前的行程分别都出现变数，由于朝鲜外交部长李勇浩日前才公开批评美国没有回应朝鲜的善意，令蓬佩奥这次的访问再次蒙上阴影。
 <br/>
 <br/>
