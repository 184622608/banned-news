### 马达加斯加举行大选
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲.html" target="_blank">
  非洲
 </a>
 国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马达加斯加.html" target="_blank">
  马达加斯加
 </a>
 举行了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统大选.html" target="_blank">
  总统大选
 </a>
 。请看报道。
 <br/>
 <br/>
 11月7日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马达加斯加.html" target="_blank">
  马达加斯加
 </a>
 的一千万合资格选民涌入全国各地的投票站，开始投票。
 <br/>
 <br/>
 凌晨6点，选民们就已经在投票站外排队等候。
 <br/>
 <br/>
 选票上印有36位总统候选人及其所属党派的照片，方便选民确认后投票。
 <br/>
 <br/>
 选民们表示，很高兴能够通过民主选举的方式选出自己认可的总统，也因此对国家的前景表示乐观。
 <br/>
 <br/>
 选民 LAURENT RAKOTOARIVONY：〝我很感恩我们能够通过选举的方式，选出我们认可的国家领导人。〞
 <br/>
 <br/>
 选民 HUGUES MARCELIN：〝我一大早就赶来投票，履行我的公民职责。我希望选举能给我们国家带来大的改变。〞
 <br/>
 <br/>
 根据马达加斯加的选举法规定，如果在周三的总统选举中没有人赢得超过一半的选票，将在12月19日举行第二轮决选。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
