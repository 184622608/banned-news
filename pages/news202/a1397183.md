### 巴西总统大选 〝巴西版川普〞波索纳洛获胜
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】周日（10月28日），巴西举行总统大选投票。有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西版川普.html" target="_blank">
  巴西版川普
 </a>
 〞之称的右翼候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波索纳洛.html" target="_blank">
  波索纳洛
 </a>
 （Jair Bolsonaro）胜出，当选新一任总统，这也意味着巴西这个世界第四大民主国家将向右翼政策方向转变。
 <br/>
 <br/>
 10月28日，巴西举行总统大选第二轮也是最后一轮的投票，选前民调一直保持领先的右翼候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波索纳洛.html" target="_blank">
  波索纳洛
 </a>
 ，毫无悬念地击败左翼工人党的哈达德（Fernando Haddad），当选新一任巴西总统。
 <br/>
 <br/>
 巴西高等选举法院在当晚确认了波索纳洛的胜选。
 <br/>
 <br/>
 巴西高等选举法院主席ROSA WEBER：〝波索纳洛赢得55.54%的选票，哈达德赢得44.46%的选票。根据这个计票结果，波索纳洛当选巴西联邦共和国总统。〞
 <br/>
 <br/>
 波索纳洛的支持者们涌上街头，欢庆胜利，对波索纳洛的当选寄以厚望。
 <br/>
 <br/>
 巴西民众 PAULA MACHADO：〝执行公义，保障民众的安全，健康和教育，其他的都会好起来。〞
 <br/>
 <br/>
 63岁的波索纳洛是前陆军上尉，来自保守的社会自由党。他公开表示自己是美国总统川普的崇拜者，他在堕胎、种族、移民、同性恋和枪支法等方面的观点与言论，为他赢得了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西版川普.html" target="_blank">
  巴西版川普
 </a>
 〞的称号。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
