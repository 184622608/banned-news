### 阿根廷班机遇强烈乱流 如机枪扫过混乱酿15伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷航空.html" target="_blank">
  阿根廷航空
 </a>
 18日一架班机由美国飞往阿根廷，在巴西上空竟遇到强烈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乱流.html" target="_blank">
  乱流
 </a>
 ，令客机偏离原有的飞行高度，上下严重摇摆，令机舱犹如被机关枪扫过一样混乱，造成机上15人受伤，所幸客机最终安全飞抵目的地。
 <br/>
 <br/>
 综合媒体报导，这架A330客机，编号AR1303的航班，载有192名乘客。当时由佛罗里达州迈阿密飞往布宜诺斯艾利斯。
 <br/>
 <br/>
 客机飞经巴西布朗科河上空时遇到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乱流.html" target="_blank">
  乱流
 </a>
 ，客机上上下下严重摇摆，氧气罩弹出、飞机餐、行衣、厨房乱七八糟混乱，还有机舱零件掉落。
 <br/>
 <br/>
 最终客机安全飞抵目的地后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷航空.html" target="_blank">
  阿根廷航空
 </a>
 将机上7名受伤乘客现场治疗，另有8名伤者送院接受进一步检查，伤者主要是撞伤及擦伤。
