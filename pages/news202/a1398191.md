### 反击朝鲜要挟？美韩海军陆战队重启联合军演
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】11月，美国和韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海军陆战队.html" target="_blank">
  海军陆战队
 </a>
 将重启联合军演，届时将有近500名美韩海军士兵参加为期两周的多兵种联合演习。三日前，平壤当局公然要挟美国取消对朝鲜的经济制裁，否则将重启发展核武器。舆论认为，美韩重启联合军演是华盛顿对金正恩要挟的回应。
 <br/>
 <br/>
 据《韩联社》报导，当地时间11月5日，韩国国防部宣布重启联合军演的消息时表示，此次演习是韩国海军交换项目的一部分，将在韩国南部庆尚北道浦项地区展开。
 <br/>
 <br/>
 韩国国防部一名发言人表示，将有500人和两栖装甲突击车参加本次军演，美军方面则将出动美国驻冲绳
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海军陆战队.html" target="_blank">
  海军陆战队
 </a>
 的第三远征部队参与军演，主要是营级的多个兵种参加。
 <br/>
 <br/>
 报导引述韩国海军陆战队一名负责人表示，今年11月实施的海军陆战队联演，是根据韩美商定的年度计划进行的，该演习属于保持联合防卫态势的年度例行战术大队训练。
 <br/>
 <br/>
 过去，平壤一直把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美韩联合军演.html" target="_blank">
  美韩联合军演
 </a>
 视为专门针对朝鲜的〝严重挑衅〞。今年6月12日美朝首脑在新加坡会谈后，为了推进朝鲜半岛无核化谈判，鼓励金正恩弃核，美韩曾宣布暂停联合军演。
 <br/>
 <br/>
 然而〝川金会谈〞后，朝鲜迟迟不采取实质性弃核举措，美朝谈判进展缓慢，一再陷入僵局。
 <br/>
 <br/>
 今年8月28日，美国国防部长马蒂斯对媒体放话称：〝我们先前以中止多项最大军事演习的措施来释放善意〞，但朝鲜当局的表现让人失望。美韩现在〝不再有暂停（军演）的打算了〞。同时，马蒂斯还表示，是否重启军演以及何时再度军演，〝一切将视谈判进度走着瞧〞。
 <br/>
 <br/>
 近期，朝鲜当局极力敦促韩国先变相放松对朝鲜的经济制裁，遭到美国政府明确、坚决的反对。
 <br/>
 <br/>
 11月2日，朝鲜外务省官员突然对外放话称，如果美国不取消对朝鲜的经济制裁，朝鲜将重新开始发展核武器。
 <br/>
 <br/>
 对此，美韩海军陆战队宣布重启联合军演，被外界视为对平壤当局的强硬回击。
 <br/>
 <br/>
 美国中期选举之后，11月6日，美国国务卿蓬佩奥将再次前往平壤，与朝鲜方面协商下一次美朝峰会的准备工作。美韩重启小型联合军演是否会影响这一行程，尚有待观察。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：范铭）
