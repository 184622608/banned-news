### APEC峰会  彭斯习近平演说擦出火花
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】2018年度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 峰会，在周六开幕。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 ，各自代表中美发表演说。其中，彭斯当着习近平的面，再次在贸易、南中国海等问题，表达了美方的强硬立场，成为外界关注的焦点。
 <br/>
 <br/>
 周六，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 峰会在巴布亚新几内亚首都莫尔兹比港开幕。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 首先发表演说，其中提到〝贸易战不会有真正的赢家〞，同时批评贸易保护主义和单边主义，针对美国的意味不言而喻。
 <br/>
 <br/>
 不过，之后上台演说的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 ，强硬地回应，除非中共改变不良的行为模式，否则美国不会改变当前的政策方向。
 <br/>
 <br/>
 美国副总统彭斯：〝我们已经针对和中国的（贸易）不平衡采取了决定性的行动。我们对2,500亿中国商品加征关税，还可以把这一数字加倍，但我们期望看到更好的结果。除非中国（共）首先改变其行为，否则美国的方针不会改变。〞
 <br/>
 <br/>
 两年来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府始终要求北京停止盗窃知识产权、放宽市场准入并减少对企业的补贴，以减少美中之间的巨额贸易逆差。
 <br/>
 <br/>
 不过，北京以报复性关税回应美方的要求。彭斯表示，川普政府绝不会妥协。
 <br/>
 <br/>
 美国副总统彭斯：〝我们对习近平主席和中国非常尊重，但如总统所说，中国占美国便宜已经太多年，这样的日子已经结束了。〞
 <br/>
 <br/>
 对于习近平在演说中称〝一带一路〞项目不是〝陷阱〞，没有地缘政治目的，彭斯也在演说中不点名的予以反驳。
 <br/>
 <br/>
 美国副总统彭斯：〝不要因为外国债务丧失主权，保护你的利益，保护你的独立，像美国那样，要始终把自己的国家放在第一位。〞
 <br/>
 <br/>
 彭斯也表达了美国在南中国海问题上的立场，强调美国将继续在这一海域自由航行。当天，彭斯会晤澳大利亚总理莫里森，宣布美国将加入澳大利亚，在巴布亚新几内亚的马努斯岛建立海军基地。
 <br/>
 <br/>
 美国总统川普没有出席此次峰会，不过，他计划与习近平在月底的G20峰会上会面。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
