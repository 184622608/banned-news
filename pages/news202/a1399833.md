### 希腊纪念学运45周年 变成火烧雅典8人被捕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】为了纪念
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_学运45周年.html" target="_blank">
  学运45周年
 </a>
 ，17日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希腊.html" target="_blank">
  希腊
 </a>
 首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雅典.html" target="_blank">
  雅典
 </a>
 上万民众聚集街头，但原本平和的游行，到了夜晚后，变成警民冲突，雅典街头火光、烟硝味四散，场面混乱，最后以暴力收场，其中有8人被捕。
 <br/>
 <br/>
 1973年11月14日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希腊.html" target="_blank">
  希腊
 </a>
 学生为了抗议军政府独裁，上万民众走上街头示威，有人还到美国大使馆前抗议，指责华府当时暗挺军方。3天后，军方竟出动坦克冲进校门，造成惨重死伤，其中至少24名学生死亡。
 <br/>
 <br/>
 如今军政府早已垮台，但希腊近年爆发债务危机，政府缩紧经济，老百姓苦不堪言，17日，趁着纪念
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_学运45周年.html" target="_blank">
  学运45周年
 </a>
 ，学生再次走上街头，游行以美国驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雅典.html" target="_blank">
  雅典
 </a>
 大使馆为重点。
 <br/>
 <br/>

   </font>
  </a>
 </center>
 <br/>
 路透社报导，超过1万名示威者在雅典市中心参与和平游行，他们举着写有〝抵制〞（Resistance）字样的横幅，高喊与希腊8年债务危机相关口号。
 <br/>
 <br/>
 当地学生说：〝做为学生今天一起纪念这个学生起义的日子，我们人民站起来反对独裁统治，我们至今仍活在同个剥削我们的制度中。〞
 <br/>
 <br/>
 游行结束后，部分示威者则跑到当年学运地点——雅典理工学校（Athens Polytechnic）所在地闹事，让原本平和的游行，入夜后突然转调。
 <br/>
 <br/>
 群众用椅子当起路霸、拒绝离场，从屋顶向警方投掷石块和爆裂物、雅典街头顿时火光四起。
 <br/>
 <br/>
 最后，当局出动5000名警力，施放催泪气及发射水炮，驱散示威人群。
