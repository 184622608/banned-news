### 日澳防长和外长高级会议 对朝鲜施压
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】周三，日本、澳大利亚国防和外长高级别会议在澳大利亚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悉尼.html" target="_blank">
  悉尼
 </a>
 举行。双方表示加强
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美日澳同盟.html" target="_blank">
  美日澳同盟
 </a>
 ，强化安保合作。
 <br/>
 <br/>
 澳大利亚外长佩恩、国防部长派恩会见了日本外务大臣河野太郎和防卫大臣岩屋毅。
 <br/>
 <br/>
 两国表示要继续对朝鲜施压，推动实现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜半岛无核化.html" target="_blank">
  朝鲜半岛无核化
 </a>
 。
 <br/>
 <br/>
 澳大利亚外交部长 佩恩：〝对于要求朝鲜放弃核武器和弹道导弹计划，我们持续施加压力。〞
 <br/>
 <br/>
 对于地区安全，日本希望同澳大利亚、美国和印度稳步实施军事演习。
 <br/>
 <br/>
 日本防卫大臣岩屋毅：〝（关于共同军演），如果可能的话，我们希望印度也加入，四国进行联合军演。我们会尽最大努力，实现这个目标。〞
 <br/>
 <br/>
 澳大利亚表示愿意参加联合军演。
 <br/>
 <br/>
 澳大利亚国防部长 派恩：〝如果我们受到邀请进行军事演习，我们很高兴参加。〞
 <br/>
 <br/>
 对于美中贸易战，两国表示，希望美中两国按现行规则解决贸易问题。
 <br/>
 <br/>
 此外，两国还讨论了中国南海局势。
 <br/>
 <br/>
 新唐人记者毕心慈、张琪综合报导
</div>
