### 美阿联手空袭 塔利班指挥官及29名成员被击毙
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 官员和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塔利班.html" target="_blank">
  塔利班
 </a>
 成员周日（12月2日）称，阿富汗和美国特种部队在阿国南部省份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赫尔曼德.html" target="_blank">
  赫尔曼德
 </a>
 联手作战，塔利班高阶指挥官阿卜杜勒・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马南.html" target="_blank">
  马南
 </a>
 被炸死，但此消息还未得到美军证实。
 <br/>
 <br/>
 路透社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赫尔曼德.html" target="_blank">
  赫尔曼德
 </a>
 省（Helmand）省长雅新汗（Mohammad Yasin Khan）周日表示，阿卜杜勒・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马南.html" target="_blank">
  马南
 </a>
 （Abdul Manan）是负责在赫尔曼德省指挥
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塔利班.html" target="_blank">
  塔利班
 </a>
 叛乱分子的官员，上周六（12月1日），他在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 和美国对南部赫尔曼德省的联合空袭行动中被击毙。
 <br/>
 <br/>
 空袭发动时，马南正在纳瓦扎德区（Nawzad）与地方指挥官及战士开会，与他一同被击毙的还有29名塔利班武装分子。
 <br/>
 <br/>
 阿富汗政府官员说，马南是阿富汗南部塔利班组织的重要头目，他的死将对阿富汗安全局势起到积极影响。
 <br/>
 <br/>

 </center>
 <br/>
 赫尔曼德省和邻近坎达哈省（Kandahar）的塔利班成员证实了他的死讯，但美国军方还没有证实此消息，美军表示，正在对此进行确认。
 <br/>
 <br/>
 阿富汗官员将击毙马南视为一大胜利。自国际作战任务大多于2014年结束以来，塔利班对赫尔曼德的控制范围逐渐增加。
 <br/>
 <br/>
