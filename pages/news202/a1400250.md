### 韩国金钟阳当选国际刑警组织新主席
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】国际刑警组织周三宣布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金钟阳.html" target="_blank">
  金钟阳
 </a>
 当选新一任主席，任期两年。金钟阳目前担任代理主席。之前引起欧美担忧的俄罗斯热门候选人落选。
 <br/>
 <br/>
 国际刑警组织194个成员国，参加了日前在迪拜召开的大会，选出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金钟阳.html" target="_blank">
  金钟阳
 </a>
 担任主席职位。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 为象征意义，任期四年。日常工作由来自德国的秘书长于尔根・施托克主持。
 <br/>
 <br/>
 国际刑警组织副主席之一，来自俄罗斯的罗科普丘克曾一度领先，被认为是主席职位的热门人选。
 <br/>
 <br/>
 但欧美国家普遍反对，指责俄罗斯利用刑警组织迫害反对派和异议人士。批评人士还指出，罗科普丘克曾任俄罗斯内政部少将，直接听命于普京。
 <br/>
 <br/>
 克里姆林宫批评人士、金融家比尔・布劳德表示：〝如果俄罗斯掌管国际刑警组织，西方社会就需寻找替代方案。〞
 <br/>
 <br/>
 本周，美国四名参议员公开敦促川普总统，反对罗科普丘克担任主席。
 <br/>
 <br/>
 国际刑警组织原主席孟宏伟，今年9月回国后失踪，之后中共官方宣称他因〝涉嫌违法〞被调查。孟宏伟长期在中共公安部任职，2004年4月起，任中共公安部副部长；2016年11月当选国际刑警组织主席时，曾引发国际人权组织的批评和谴责。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
 <br/>
