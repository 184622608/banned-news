### G20场外抗议 反对经济全球化
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】周四，阿根廷举办的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会前夕，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 者走上首都街头，反对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济全球化.html" target="_blank">
  经济全球化
 </a>
 。阿根廷则部署了2万名警力待命。川普表示取消和普京的会谈。
 <br/>
 <br/>
 11月30日至12月1日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会在阿根廷首都举行，当地左翼团体邀请世界各地社会活动者，参加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 ，要求政府关心国内需求。
 <br/>
 <br/>
 G20峰会抗议者DIANA：〝首先，政府需照顾本国需求，而不是外面发生的事。我们真的不需要G20，G20峰会对我们没有任何帮助。〞
 <br/>
 <br/>
 周四的抗议没有出现暴力事件，主要是年轻人、老人和一些表演团体。
 <br/>
 <br/>
 G20峯会抗议者NORA DE CORTINAS,：〝我们不想要新自由主义经济，那是带来饥饿，给日常生活带来破坏的经济，正如这个法西斯政府所做的那样。〞
 <br/>
 <br/>
 川普此行预期将与中国领导人习近平、日本首相安倍晋三等首脑会谈。
 <br/>
 <br/>
 去年德国汉堡召开的G20峰会遭遇暴力抗议，今年阿根廷政府和抗议团体都表示，不容忍暴力。
 <br/>
 <br/>
 活动团体准备周末举行反对国际货币基金组织的抗议活动。国际货币基金组织同意提供560亿美元，帮助阿根廷应对经济危机，阿根廷政府则承诺采取严格的经济紧缩政策，引发国内不满。
 <br/>
 <br/>
 新唐人记者毕心慈、明玉综合报导
</div>
