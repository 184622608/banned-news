### 历史性裁决 前赤柬2领袖种族灭绝罪成
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】联合国柬埔寨法院特别法庭于16日作出历史性的判决，首次裁定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红色高棉.html" target="_blank">
  红色高棉
 </a>
 （Khmer Kraham又译
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赤柬.html" target="_blank">
  赤柬
 </a>
 ）两名前领袖犯下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_种族灭绝罪.html" target="_blank">
  种族灭绝罪
 </a>
 行，被判终身监禁。这次是首次有前赤柬领袖被裁定这项罪名成立。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赤柬.html" target="_blank">
  赤柬
 </a>
 (红色柬埔寨、柬埔寨共产党)这2名前领袖，分别是87岁的乔森潘(Khieu Samphan)及92岁的农谢，他们当天被小巴及救护车护送到法庭。在2014年，他们被法庭裁定犯下反人道罪行，当时被判处终身监禁。
 <br/>
 <br/>
 法官今天表示，农谢行使最后决策权，因此要为所有罪行负上责任，包括针对回教占族人的种族灭绝；乔森潘则被指对国内越南裔民众种族灭绝。
 <br/>
 <br/>
 据称，农谢有华人血统，中文姓名为〝刘平坤〞，主管意识形态，是杀人总设计师。乔森潘则先后做过总理及国家主席，虽然，名义上是国家之首，但实质上受〝狂人〞波尔布特操控，在赤柬政权中属第五号人物。
 <br/>
 <br/>
 两人是赤柬最后在世的前领袖，除了首次被裁定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_种族灭绝罪.html" target="_blank">
  种族灭绝罪
 </a>
 罪成外，还面临其他反人道罪行，包括谋杀、灭绝、奴役、虐待，以及因政治、种族、宗教理由迫害他人。
 <br/>
