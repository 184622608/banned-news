### 联合国人权理事会审议 多国批中共侵犯人权
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权理事会.html" target="_blank">
  人权理事会
 </a>
 在瑞士日内瓦举行会议，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人权.html" target="_blank">
  中国人权
 </a>
 纪录进行第三次审议。西方多国批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 持续侵犯人权，并敦促停止对法轮功等宗教信仰的迫害。
 <br/>
 <br/>
 11月6日到9日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权理事会.html" target="_blank">
  人权理事会
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人权.html" target="_blank">
  中国人权
 </a>
 现状，进行第三次〝普遍定期审查（UPR）〞。
 <br/>
 <br/>
 在会前，多个国家提交了对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的质询清单，涉及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 西藏问题、宗教自由、网络监控、709大抓捕、香港自由等众多领域。
 <br/>
 <br/>
 6号，欧美澳13国代表当面敦促中共，关闭新疆的〝再教育营〞。
 <br/>
 <br/>
 〝我们对中共政府镇压新疆的维吾尔族，哈萨克族和其他穆斯林的行为，感到震惊。我们建议中国废除一切形式的任意拘留，包括新疆的拘留营，并立即释放拘留营里的数十万人，或许是数百万人。〞
 <br/>
 <br/>
 〝他们因种族或宗教被任意拘留，而且未经适当的程序。〞
 <br/>
 <br/>
 西方多国也对中共侵犯民众宗教信仰权利表达关注。美国在质询清单中说：中共〝关闭和强拆教堂，关押法轮功学员，中方有何措施停止打压宗教自由？〞德国质询中共：〝对于外界指控中国监狱出现器官摘取，如何回应？〞加拿大代表则当面向中共提出建议。
 <br/>
 <br/>
 〝结束对宗教和信仰团体的起诉和迫害。包括对穆斯林、基督徒、藏传佛教徒和法轮功。〞
 <br/>
 <br/>
 中共自2009年来，三次接受普遍定期审查；不过多个国际人权组织指出，中国人权状况并未改善，反而持续恶化。于此同时，中共却借助联合国人权理事会的匿名投票机制，多次当选理事会成员国。
 <br/>
 <br/>
 在把中共迫害法轮功这些客观事实清查前，在它没有履行联合国《世界人权宣言》之前，它在践踏自己人权义务的同时，又成为联合国人权委员会的成员，这对联合国本身，是最大的讽刺，国际社会不应该让这样的笑话发生。
 <br/>
 <br/>
 美国多次推动人权理事会改革无效，今年6月宣布退出，这次仅以观察员身份出席。但美国驻联合国大使黑利表示，美国将继续支持改革联合国人权理事会。
 <br/>
 <br/>
 〝中共能够逃脱大规模侵犯人权的责任，是因为其他国家让它们逃脱，因为它们和中共有经济利益等关系。现在是改变的时候了。〞
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
