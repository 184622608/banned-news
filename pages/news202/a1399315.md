### 英欧达成脱欧草案 英内阁审议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】当地时间周三下午，英国首相梅召开内阁紧急会议，讨论和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 达成的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧.html" target="_blank">
  脱欧
 </a>
 草案。首相梅希望能让分裂的内阁统一意见，但即使如此，草案仍需要得到欧盟27国和英国下议院的首肯，才能正式通过。目前看来困难重重。
 <br/>
 <br/>
 英国和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 终于达成了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧.html" target="_blank">
  脱欧
 </a>
 草案，但这只是艰难历程的第一步。
 <br/>
 <br/>
 英国首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特雷莎・梅.html" target="_blank">
  特雷莎・梅
 </a>
 ：〝我们的脱欧谈判现在已经进入尾声了。〞
 <br/>
 <br/>
 下一步就要看欧盟各国，以及英国的政治家如何表态了。
 <br/>
 <br/>
 上周五，又一名内阁成员因为反对首相的脱欧方案而辞职，不过外界普遍认为，周三的内阁会议上很有可能达成共识。
 <br/>
 <br/>
 11月底的欧盟成员国投票过关机率也不小。
 <br/>
 <br/>
 最有可能遭遇滑铁卢的是英国下议院投票。
 <br/>
 <br/>
 英欧双方最大分歧点是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱尔兰边界.html" target="_blank">
  爱尔兰边界
 </a>
 问题，草案提议，英国脱欧后仍留在关税同盟中一段时间，以避免和欧盟成员国爱尔兰之间重设边界。
 <br/>
 <br/>
 草案确认了英国向欧盟支付390亿英镑的〝分手费〞，还保证了居住在英国的300万欧盟公民和居住在欧洲的100万英国人的现有权利。
 <br/>
 <br/>
 无论是脱欧还是留欧阵营，都对草案有所批评。
 <br/>
 <br/>
 保守党议员伯恩认为，草案违背了全民公投的结果，就是英国将彻底脱离欧盟。
 <br/>
 <br/>
 英国保守党议员 伯恩：〝今天，你将失去很多保守党议员和全国几百万选民的支持。〞
 <br/>
 <br/>
 首相梅的执政伙伴北爱尔兰民主统一党的10名国会议员都属于脱欧派。
 <br/>
 <br/>
 国会议员威尔逊周二发推文，将这份草案形容为对英国的羞辱。
 <br/>
 <br/>
 推文说：〝在脱欧公投期间推出的所有大话和好话未来几周还将被推出，试图让我们陷入困境。我们很清楚，我们不会投票支持这种羞辱！〞
 <br/>
 <br/>
 反对党领袖科尔宾是留欧派，也反对这份草案。
 <br/>
 <br/>
 英国反对党领袖 科尔宾：〝这个（草案）没有提供一个支持各行业就业的经济解决方案。〞
 <br/>
 <br/>
 各种不定因素导致金融市场反应不一。周三早盘，英镑飙升，但很快又回落。
 <br/>
 <br/>
 到2019年3月29日，英国正式脱欧还有五个月时间。这段时间里还会发生什么，还有待观察。
 <br/>
 <br/>
 新唐人记者王羽音、田瑞综合报导
</div>
