### 沙特：记者在领馆与人搏斗遇害  川普回应
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】当地时间周六，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 阿拉伯发布官方声明，首次承认沙特记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 ，在沙特
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻土耳其领事馆.html" target="_blank">
  驻土耳其领事馆
 </a>
 内死亡，声称他死于〝搏斗〞。对此，美国总统川普做出了回应。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 阿拉伯电视台周六一早宣读了官方声明。声明中宣称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 在沙特
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻土耳其领事馆.html" target="_blank">
  驻土耳其领事馆
 </a>
 内死亡，死因是与人搏斗。
 <br/>
 <br/>
 沙特电视台新闻主播：〝检察官对公民卡舒吉失踪事件展开了初步调查，调查显示，他和一些人在沙特驻伊斯坦布尔领事馆会面并交谈，双方发生了争执并搏斗，导致卡舒吉死亡。〞
 <br/>
 <br/>
 声明中说，沙特官方已经就此事逮捕了18人。声明中还说，沙特当局把此事当作最高级别的大事处理。
 <br/>
 <br/>
 沙特电视台新闻主播：〝沙特采取了下述行动调查事件真相，我们已经在10月6日派出国家安全团队前往土耳其调查，并与土耳其安全部门合作。〞
 <br/>
 <br/>
 白宫发言人桑德斯，周五晚发表白宫声明，表示认可沙特官方的调查取得进展，将持续密切关注进一步的调查结果，并支持以公开、透明、合法的程序为死者伸张正义。
 <br/>
 <br/>
 美国总统川普也针对此事发表看法。
 <br/>
 <br/>
 美国总统川普：〝沙特阿拉伯是很好的盟友，但（卡舒吉）所发生的事让人无法接受。〞
 <br/>
 <br/>
 美国总统川普：〝他们逮捕了很多人，这是一个好的开端。〞
 <br/>
 <br/>
 川普表示，如果美国对沙特采取制裁行动，将与国会合作进行。但他不希望以对沙特的军售合同作为代价，届时将影响几十万人的就业。
 <br/>
 <br/>
 另一方面，卡舒吉的未婚妻简吉兹，发推文质疑沙特方面的声明。她质问沙特政府，卡舒吉的尸体在哪？
 <br/>
 <br/>
 此前，有消息传出卡舒吉曾遭酷刑折磨并遭到肢解，不过沙特的声明中并未对此做出解释。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
