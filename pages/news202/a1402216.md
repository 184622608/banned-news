### 加拿大情报局：外国间谍活动侵蚀关键产业
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】12月4日，加拿大安全情报局局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维格诺.html" target="_blank">
  维格诺
 </a>
 （David Vigneault）发表演讲，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外国间谍.html" target="_blank">
  外国间谍
 </a>
 行为对加拿大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关键经济领域.html" target="_blank">
  关键经济领域
 </a>
 的侵蚀形成趋势，他警告外国间谍瞄准的都是有研发新型技术潜力的公司和大学。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大情报局.html" target="_blank">
  加拿大情报局
 </a>
 局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维格诺.html" target="_blank">
  维格诺
 </a>
 （David Vigneault）特地选商界人士为主的经济俱乐部发表演讲，警告
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外国间谍.html" target="_blank">
  外国间谍
 </a>
 行为正在瞄准加拿大的技术关键产业。演讲的时间相当耐人寻味，正是美国总统川普和中国达成约定，开始谈判中共强制技术转让和知识产权盗窃的几天之后。就在几个月前，加拿大政府以国家安全理由拒绝了中共国有企业中国交建（中国交通建设股份有限公司）收购加拿大建筑公司Aecon。目前加拿大政府正在考虑是否禁止中国电信公司华为参与加拿大的5G通讯建设。
 <br/>
 <br/>
 加拿大情报局局长David Vigneault：〝加拿大情报局看到由国家支持的间谍活动在加拿大关键领域形成趋势，这些领域关系到加拿大建立和保持一个繁荣的以知识为基础的经济。我说的这些领域是人工智能技术、量子技术，5G，生物制药和清洁技术等领域，换句话说，这些领域是加拿大未来经济增长的基础。〞
 <br/>
 <br/>
 维格诺说，虽然外国间谍行为通常使用加密技术和加密货币如比特币等来隐藏身份和意图，但还是有迹可循，主要有3种盗窃技术的方式。比如，加拿大一家机器人技术公司扩展到海外，在该国设立办公室并雇佣当地人员，开始和当地政府打交道。
 <br/>
 <br/>
 加拿大情报局局长David Vigneault：〝这时，这家公司就变得很脆弱了，新的办公室或实验室会成为外国情报机构的目标，或者有内部人员开始做手脚。眨眼之间，该公司的技术被访问，其产品被采用逆向工程，外国竞争对手开始以更便宜的价格在全球销售相同的技术，不久之后，这家加拿大公司破产了。〞
 <br/>
 <br/>
 第二种方式是投资加拿大公司，或直接购买加拿大公司，所有的技术自然流向外国。第3种方式则是通过网路直接盗窃。
 <br/>
 <br/>
 维格诺警告，外国政府支持的各种技术盗窃可能威胁到加拿大的国防和主权利益，甚至利用加拿大技术制造武器，来对付加拿大和盟国。
 <br/>
 <br/>
 加拿大情报局局长David Vigneault：〝有敌意的国家通常针对那些积极参与新兴技术的公司或大学–因为这种潜在的革命性发现可以带来巨大的利润。〞
 <br/>
 <br/>
 尽管这次情报局长的演讲中没有直接提华为的名字，情报局前局长Ward Elcock不久前公开表示，很难想象由中共政府控制的华为不会其技术中设陷阱和后门。目前由美国、澳洲、新西兰、英国、加拿大组成的国际情报5眼联盟中，美国、澳洲和新西兰都禁止华为参与5G网络建设。加拿大则正在评估，总理杜鲁多表示将〝根据事实、证据及加拿大人的最佳利益做出决定。〞
 <br/>
 <br/>
 新唐人记者刘海英多伦多报导
