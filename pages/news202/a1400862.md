### 惩戒硬闯美墨边境 墨驱逐500大篷车移民
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 政府周日（11月25日）宣布，将大约500名硬闯美国边界的中美洲移民立即
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驱逐.html" target="_blank">
  驱逐
 </a>
 出境。美国国土安全部也声明，不会容忍此类侵犯美国主权的行为发生。
 <br/>
 <br/>
 周日晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 内政部声明，墨西哥政府在下加利福尼亚州（Baja California）地方当局的帮助下，逮捕了大约500名擅闯美墨边界的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民。由于这些移民企图采用暴力和非法手段跨越边界，墨西哥政府决定将他们
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驱逐.html" target="_blank">
  驱逐
 </a>
 出境。
 <br/>
 <br/>
 社交媒体上流传的视频和照片显示，周日，数百名中美洲移民跨过墨西哥蒂华纳（Tijuana）一条河上的人行天桥后，径直冲向美国圣地亚哥（San Diego）的边界。
 <br/>
 <br/>

 <br/>
 这一事件使连接蒂华纳和圣地亚哥的边境口岸告急，美国和墨西哥随后都关闭了各自一方的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入境口岸.html" target="_blank">
  入境口岸
 </a>
 。
 <br/>
 <br/>
 一些移民试图强行突破边境围栏，美国边境人员被迫使用催泪瓦斯应对这种局面。
 <br/>
 <br/>
 不过，美国海关和边境保护局（U.S. Customs and Border Protection, CBP）周日晚宣布，圣地亚哥的入境口岸已经重新对汽车及行人开放，并确认边防巡逻人员已扣押多名非法入境移民。
 <br/>
 <br/>
 CBP还确认，一些美国边防人员被移民投掷的石块砸中，而作为应对措施，CBP部署了用以驱散人群的装置，包括胡椒球发射系统和装催泪瓦斯的小罐子。
 <br/>
 <br/>
 国土安全部长尼尔森（Kirstjen Nielsen）声明，国土安全部不会容忍此类无法无天的行为，而且，为了边境安全和公共安全，将会毫不犹豫地关闭入境口岸。
 <br/>
 <br/>
 尼尔森表示，除此之外，国土安全部还将在法律允许的最大范围内起诉那些毁坏美国财产、危及边境运营或侵犯美国主权的不法之徒。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
