### 时事拼盘：美俄宇航员安全脱险 川习会有望再次进行
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】下面请看一组新闻简讯
 <br/>
 <br/>
 周四，哈萨克斯坦发射升空的俄罗斯飞船〝联盟号〞出现故障，美国宇航员海格和俄罗斯宇航员奥夫金宁被迫紧急降落。医生表示,目前两人身体状况良好。事故后,俄罗斯暂停所有载人航天飞行。
 <br/>
 <br/>
 周四，印度东部沿海地区遭热带气旋〝提特里〞袭击，最大阵风时速约103英里，带来暴雨，并引发洪水。至少5人死亡，数十万人无电可用。当地政府已疏散海岸地区近30万居民。
 <br/>
 <br/>
 美国联邦调查局周三称，纽约男子罗森菲尔德计划于11月6日中期选举日,在华府国家广场进行自杀式炸弹袭击。理由竟是为推动一种新的政治体制，随机选择官员，而不是选举。罗森菲尔德称,他网购炸药,在自家地下室制造了一枚200磅的炸弹。如罪名成立，他将面临20年监禁。
 <br/>
 <br/>
 美国白宫首席经济顾问库德洛周四表示,白宫正在为川普和习近平的会晤做准备。但目前还不能确定,在11月30日和12月1日的阿根廷20国峰会上,两人是否可以会面。
