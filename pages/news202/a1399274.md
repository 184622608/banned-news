### 天价纪录！珍稀珠宝〝粉红遗产〞5000万美元成交
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】珍稀珠宝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_19克拉粉钻.html" target="_blank">
  19克拉粉钻
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_粉红遗产.html" target="_blank">
  粉红遗产
 </a>
 〞（Pink Legacy）13日在日内瓦拍卖敲槌，成交价5037.5万瑞郎（5000万美元）创新纪录。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佳士得.html" target="_blank">
  佳士得
 </a>
 国际珠宝部门主管卡达奇亚（Rahul Kadakia）形容，这颗〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_粉红遗产.html" target="_blank">
  粉红遗产
 </a>
 〞是〝全球最棒的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_钻石.html" target="_blank">
  钻石
 </a>
 之一〞。
 <br/>
 <br/>
 〝粉红遗产〞由Swatch集团旗下公司海瑞温斯顿（Harry Winston）买下，立刻将钻石更名为〝温斯顿粉红遗产〞（The Winston Pink Legacy）。
 <br/>
 <br/>
 佳士得欧洲地区负责人库利尔（Francois Curiel）表示：〝粉红遗产拍卖创下5000万美元极高价，相当每克拉260万美元，这是粉钻1克拉价格的世界纪录。〞此价包括了所有费用与佣金。
 <br/>
 <br/>
 他说：〝这颗赞石对我而言，就是钻石界的达文西（Leonardo Da Vinci），再也找不到更好的了。〞
 <br/>
 <br/>
 〝粉红传奇〞原为全球最大钻石珠宝商戴比尔斯（De Beers）经营者奥本海默（Oppenheimer）家族所有，拍卖前预估可拍得3000万到5000万美元之间。佳士得未透露卖家身分。
