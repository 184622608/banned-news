### 散播争议性言论 脸书删82个伊朗假帐号
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脸书.html" target="_blank">
  脸书
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Facebook.html" target="_blank">
  Facebook
 </a>
 ）近日发声明称，已移除82个疑似来自伊朗的假帐号。这些假帐号试图利用移民、种族关系、反对美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_敏感话题.html" target="_blank">
  敏感话题
 </a>
 散播争议性言论，以左右美、英政治。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脸书.html" target="_blank">
  脸书
 </a>
 于当地时间26日表示，已经删除82个佯装成美国和英国公民的假帐号、专页和社群，他们并非针对某一个政党或候选人，但会购买广告、对特定争议性议题，包括移民和种族关系等可能引起激烈反应的议题相关讯息留下煽动性言论，企图制造对立、冲突，以影响美英政治。
 <br/>
 <br/>
 这些账号在脸书分享的贴文中包括针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 、英国首相特蕾莎．梅（Theresa May）及美国最高法院新任大法官卡瓦诺（Brett Kavanaugh）的相关争议发表煽动性评论。
 <br/>
 <br/>
 负责脸书网路安全政策的葛莱彻（Nathaniel Gleicher）表示，这82个疑似来自伊朗的网页、群组和帐号，违反参与〝不实〞行为的政策。
 <br/>
 <br/>
 葛莱彻说，这些帐号与今年稍早撤销的帐号重叠，并与伊朗官方媒体有连结，但尚未确定罪魁祸首身分。他在电话简报中说：〝通常很难知道这种活动的幕后黑手是谁。〞
 <br/>
 <br/>
 不过，葛莱卓也说，虽然目前无法确定用户的真实身分，但〝战情室〞（war room）发现他们的来源几乎都和伊朗有关。但因证据不足，无法证明这些账号是否和伊朗政府有关联。
 <br/>
 <br/>
 脸书的〝战情室〞总部设于美国加州，有2万名工作人员在那里专门揪出散布假消息的假帐号。
 <br/>
 <br/>
 根据脸书说法，这些帐号和网页可追溯到几年前，但多数线上活动出现在过去一年，包括期间花费100美元2次购买线上广告，首次购买广告是在2016年6月，第2次在今年年初。
 <br/>
 <br/>
 这次脸书肃清的账户共删除30个网页、33个帐号、3个脸书群组以及16个Instagram帐号，是打击外国势力透过假消息和脸书操弄，影响美国及其他地区大选的行动一环。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
