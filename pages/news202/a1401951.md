### 【禁闻】高通婉拒中国重审并购恩智浦案
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】习近平在日前的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 上表示，先前没有通过中国核准的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高通.html" target="_blank">
  高通
 </a>
 收购
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恩智浦.html" target="_blank">
  恩智浦
 </a>
 〞一案，他将以开放态度再次审核。不过，高通回应，不会再次向中共当局提交并购案申请。
 <br/>
 <br/>
 美国晶片大厂〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高通.html" target="_blank">
  高通
 </a>
 〞公司发言人3号表示，很高兴得知总统川普和中国国家主席习近平关注高通此前收购
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恩智浦.html" target="_blank">
  恩智浦
 </a>
 的提案，但是计画中的并购案已经过了交易期限。他说：〝高通认为这件事已经结束，现在全力专注于继续执行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5G网络.html" target="_blank">
  5G网络
 </a>
 。〞
 <br/>
 <br/>
 高通和恩智浦是全球半导体大公司，在某一些国家有很大的市场份额。为了担心合并后形成市场垄断，有损消费者权益，因此合并交易必须得到这些国家监管机构的批准。
 <br/>
 <br/>
 高通的收购计画自2006年10月宣布以来，已经在欧盟和韩国等8个司法辖区获得批准，只有中共以反垄断为由拒绝放行。
 <br/>
 <br/>
 恩智浦首席执行官克莱默（Rick Clemmer）早在9月已经表示，高通并购恩智浦案已经〝过去了〞，恩智浦公司也进行了调整，为了安抚投资者，还宣布了新的季度股息。
 <br/>
 <br/>
 据了解，如果这宗收购案成功通行，将帮助高通从智慧型手机市场向汽车和物联网等新市场扩张。
 <br/>
 <br/>
 台湾企业家与社会运动人士高为邦：〝时间已经错过了，而且高通也付了一笔20亿美元的分手费，人家（恩智浦）已经把公司重新改组了，而且经过半年的时间。所以，高通要并购恩智浦，也未必人家同意，在这种情况之下，高通只能说，这个已经成过去式了，目前没有这个考量。〞
 <br/>
 <br/>
 这宗价值440亿美元的收购被标榜为全球最大的半导体收购案。今年的7月25号，是高通、恩智浦双方商量交易的最后截止日。在交易过期前，高通和恩智浦等待中共监管机构的批准已经等了近两年。
 <br/>
 <br/>
 7月24号，中共商务部发言人表示，初步认定高通方案难以解决相关市场竞争问题。25号，高通宣布放弃对恩智浦的收购，并向恩智浦支付了20亿美元的分手费。
 <br/>
 <br/>
 高通还提出300亿美元的库藏股计画，以平息股东的不满。过去12个月，高通已花费超过200亿美元买回股票。
 <br/>
 <br/>
 高通被视为美中贸易争端的受害者之一。
 <br/>
 <br/>
 高为邦：〝欧盟跟韩国都批准了之后，中国就剩下最后一个了，所以它就拖时间，故意的，那当然这个是因为它对美国的不满。中美贸易的战争，它把这个当作筹码，在里边运用，结果就是不准嘛。〞
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田也表示，是中共的刻意阻挠，让高通错失了良机。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝当时的价钱是当时有效，现在它即使要重新提出，可能就是另外一个价钱，可能还涉及到其他9个国家的反垄断调查，所以它可能不会那么轻易的再提出来，实际上是中共的刻意阻挠让公司错失良机了。〞
 <br/>
 <br/>
 谢田表示，中共从来不在乎中国消费者的权益是否受到伤害。
 <br/>
 <br/>
 谢田：〝中共根本不在乎垄断和商业运作，中共在乎唯一的事情就是它自己的利益和生存，它那个时候做出这个决定，完全是为难美国，报复美国，当时完全是把商业作为一个政治棋子。〞
 <br/>
 <br/>
 谢田表示，这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 ，美国提到高通这个案子，也是因为高通是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 中，中共枪口下的牺牲品。
 <br/>
 <br/>
 谢田：〝政治的干预，国家的干预，就是说不能市场运作，也不尊重法律，这正好是贸易战中川普要求中共停止的一项内容，就是说，停止中共对经济的控制，所以作为一个最新的例子，很自然就被提出来了。〞
 <br/>
 <br/>
 台湾企业家高为邦则表示，这次川习会，川普释出善意，把加税方案延了90天，所以中方表示，如果高通收购案再度提出，会持开放态度。高为邦认为，即使高通再次向中共提案，也不一定能通过。
 <br/>
 <br/>
 采访/陈汉 编辑/陈洁 后制/李沛灵
