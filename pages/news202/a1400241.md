### 密谋数十亿美元洗钱案 委内瑞拉前官员认罪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 前总统查维斯政府中的一名官员，20日在佛罗里达州联邦法院，承认2017年犯下数十亿美元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗钱案.html" target="_blank">
  洗钱案
 </a>
 ，法官预计27日宣判。
 <br/>
 <br/>
 54岁的委内瑞亚公民安德拉德（Alejandro Andrade Cedeno）2007年到2010年担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 国家财政办公室（Venezuelan National Treasury Office）主席，也领导委内瑞拉国家经济发展银行（National Economic Development Bank of Venezuela）
 <br/>
 <br/>
 美国司法部表示，安德拉德住在佛罗里达州惠灵顿村（Wellington），他承认在2017年12月22日犯下罪行。
 <br/>
 <br/>
 司法部透露，另一名委内瑞拉公民希梅内斯（Gabriel Jimenez Aray），是多明尼加现已解散的佩拉维亚银行（Banco Peravia）所有人，他3月20日对密谋洗钱认罪。
 <br/>
 <br/>
 根据司法部，安德拉德与希梅内斯承认的罪行，是贪腐计划的一部分，涉案人士也包括委内瑞拉民营电视台全球电视（Globovision）所有人高林（Raul Gorrin Belisario）。
 <br/>
 <br/>
 50岁的高林住在迈阿密，2017年8月16日在佛罗里达州南区遭控违反海外反腐败行为法（Foreign Corrupt Practices Act）与洗钱。
 <br/>
 <br/>
 根据起诉书，高林拿出数百万计美元贿赂两名委内瑞拉高官，包括安德拉德，获得以有利于委国政府的汇率从事外汇交易的权利。
