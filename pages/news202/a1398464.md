### 【禁闻】媒体：中期选举之后 中美关系依旧
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】
 <b>
  媒体：
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
   中期选举
  </a>
  之后 美对华政策不会变
 </b>
 <br/>
 <br/>
 备受瞩目的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 11月6号落下帷幕。美国国会由原先的共和党掌控参众两院，变成共和党掌控参议院、民主党掌控众议院。不过，外界普遍认为，这一变化不会影响目前美国的对华政策。
 <br/>
 <br/>
 白宫发言人桑德斯也表示，无论哪个党派取得参、众议院控制权，都不会改变川普的施政。
 <br/>
 <br/>
 在海外社交媒体推特上，一些中文网友也翻出民主党众议院领袖南希．佩洛西早前呼吁惩罚中共的言论，并评论说，在打击中共的问题上，美国的两党目标一致。
 <br/>
 <br/>
 〝德国之声〞则直接报导说：〝中期选举之后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系.html" target="_blank">
  中美关系
 </a>
 依旧是寒冬〞。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美朝对话.html" target="_blank">
   美朝对话
  </a>
  推迟 网友：受美中关系影响
 </b>
 <br/>
 <br/>
 美朝对话再次出现波折。美国务卿蓬佩奥和朝鲜高级官员金英哲原定11月7号在纽约举行的会面已经被推迟。
 <br/>
 <br/>
 美国国务院6号晚间宣布这个决定时，没有给出具体原因，也没有给出新的时间表。
 <br/>
 <br/>
 在此之前，美国也曾在8月底取消蓬佩奥赴朝鲜的安排。
 <br/>
 <br/>
 今年6月美朝领导人川普和金正恩在新加坡举行历史性的峰会后，外界一直批评朝鲜未作出不可逆转的去核行动。而朝鲜方面的态度也几度反覆。
 <br/>
 <br/>
 有中国网友总结说：〝川普对中共一软，金三就硬，川普对中共一硬，金三就软。总是如此。〞
 <br/>
 几天前，川普与习近平刚刚通了电话，并在事后表示这次通话〝时间长、非常好〞。
 <br/>
 <br/>
 <b>
  英警告电信公司审查
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5G.html" target="_blank">
   5G
  </a>
  供应商 剑指
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
   华为
  </a>
 </b>
 <br/>
 <br/>
 英国政府最近警告国内电信公司，在建设5G网络时要仔细审查供应商。业内人士说，这将导致中国电信设备制造商华为在英国受到更严厉的审查。
 <br/>
 <br/>
 据《金融时报》11月7号报导，英国政府上个月致信多家国内电信公司，说将审查英国是否过于依赖单一硬件供应商。信中并未点名具体公司，但收到信的企业高管都表示，此举是针对华为。
 <br/>
 分析人士指出，英国政府的做法反映了英国对华为不断扩张的担忧，而这可能导致华为在英国遭受更严格的审查，甚至被禁止在英国推出5G。
 <br/>
 <br/>
 编辑/周玉林
