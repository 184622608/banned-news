### 在刘强东晚宴后强奸女模  华富商判囚4年
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】中国电商巨头京东集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 ，此前美国涉嫌性侵案件尚未完结。而3年前在其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳大利亚.html" target="_blank">
  澳大利亚
 </a>
 的一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_私人晚宴.html" target="_blank">
  私人晚宴
 </a>
 后，中国籍商人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐龙伟.html" target="_blank">
  徐龙伟
 </a>
 （Longwei Xu，音译）涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女模特.html" target="_blank">
  女模特
 </a>
 一案近日宣判，徐被判入狱4年。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳大利亚.html" target="_blank">
  澳大利亚
 </a>
 华商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐龙伟.html" target="_blank">
  徐龙伟
 </a>
 ，3年前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_私人晚宴.html" target="_blank">
  私人晚宴
 </a>
 后，涉嫌性侵当年只有20岁的一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女模特.html" target="_blank">
  女模特
 </a>
 儿，他早前已被裁定猥亵、袭击、在明知女方不同意的情况下，仍与其发生性行为等7项罪名成立。
 <br/>
 <br/>
 10月12日，徐龙伟被悉尼唐宁中心法院判处入狱4年，2年半内不得保释。
 <br/>
 <br/>
 由于案发于刘强东在澳洲举行的晚宴之后，此案件备受关注。
 <br/>
 <br/>
 据澳大利亚媒体报导，徐龙伟是刘强东的〝朋友〞，但是刘强东曾委托律师发表声明，声称与当事双方都不熟悉。在案件调查过程中，刘强东的代表律师，曾经两次申请法庭禁令，要求为刘强东的身份保密，但被法官拒绝。
 <br/>
 <br/>
 在10月12日的庭审宣判上，法官在判决中称：徐龙伟在事发的酒店内，曾向受害人提出可以付钱，并说出〝你不可能拒绝我〞等话。
 <br/>
 <br/>
 法官形容徐龙伟是一个傲慢的人，认为自己的魅力会让受害人屈服，〝表现出一种权力优越感，丝毫不考虑受害者的情况下做他自己想做的事〞。
 <br/>
 <br/>
 被告律师称，徐龙威与前女友有一个刚出生不久的孩子，要靠徐龙伟的经济资助度日，因此提请法官考虑量刑对徐龙伟家的影响。
 <br/>
 <br/>
 法官强调，〝侵犯他人身体是严重的犯罪，在受害人身上发现多处伤痕。〞徐最终被判入狱​​4年的最高刑期。
 <br/>
 <br/>
 上述事件发生于2015年12月10日，受害人作为泳装模特，参加了一个在游艇上举行的化妆派对，期间结识认了一个名叫可可（Coco）的女子，该女子邀请她参加了那次私人晚宴。
 <br/>
 <br/>
 据称，可可一再邀请受害人与从中国来的非常富有的〝某些朋友会面〞。
 <br/>
 <br/>
 陪审团获悉，受害人多次拒绝了邀请，但最后接受了可可的邀请，去参加她弟弟12月26日的〝生日宴会〞。
 <br/>
 <br/>
 受害人在私人晚宴上被灌醉后，被告徐龙伟借口送她回家将她带至香格里拉酒店。
 <br/>
 <br/>
 受害人说，入房后徐龙伟将她的手机扔到一边，用力将她的衣服脱下，又用手指插入她的下体，一直将她压住使其无法动弹，期间，徐又声称会买一艘船给她，但她拒绝。徐龙伟当时掴了她一耳光。
 <br/>
 <br/>
 当晚23时，受害人趁徐龙伟睡着时逃离，致电男友到酒店接她，随后，到警局报案。警员搜查徐的酒店房间，在其手上发现受害人DNA，遂将他拘捕并提出起诉。
 <br/>
 <br/>
