### 东亚峰会  美提出600亿美元发展融资
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】周四，美国副总统彭斯出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东亚峰会.html" target="_blank">
  东亚峰会
 </a>
 ，并发表讲话。除了对于区域安全的承诺，他还表示，美国将提供600亿美元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发展融资.html" target="_blank">
  发展融资
 </a>
 ，并以印太基础设施项目作为首要任务。
 <br/>
 <br/>
 周四(11月15日)，美国副总统彭斯出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东亚峰会.html" target="_blank">
  东亚峰会
 </a>
 ，并发表讲话。除了对于区域安全的承诺，他还表示，美国将提供600亿美元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发展融资.html" target="_blank">
  发展融资
 </a>
 ，并以印太基础设施项目作为首要任务。
 <br/>
 <br/>
 美国副总统彭斯在东亚峰会上表示，美国在该地区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资.html" target="_blank">
  投资
 </a>
 已超过1.4万亿美元，并且坚持最高质量标准及尊重各国主权。为促进新投资，美国还将提供600亿美元的发展融资，而印太基础设施项目将是首要任务。
 <br/>
 <br/>
 〝传统基金会〞亚洲项目主任Walter Lohman：〝美国在东南亚的投资远超中国，日本远超中国，欧盟则是最大投资者。当我们谈论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中竞争.html" target="_blank">
  美中竞争
 </a>
 时，似乎是彼此实力相当，但并非如此。〞
 <br/>
 <br/>
 美国国会上个月通过国际发展投资法案，决定合并现有部门，设立一个规模更大的美国国际发展金融公司，为其提供600亿美元资金，协助全球贫困地区基础设施与经济发展。
 <br/>
 <br/>
 美国会众院外委会亚太小组主席Ted Yoho：〝我们的目标是建设经济，并提供各国一带一路之外的强有力的不同选择。别忘了，我们的总体目标是从援助转向贸易，如能这样，这些国家及其人民将富裕得多，但条件是他们的政府要尊重基本人权。〞
 <br/>
 <br/>
 目前美中竞争也是区域国家关切的焦点。本月底G20峰会上，川普将与习近平举行会谈。彭斯日前受访时曾表示，中共必须在贸易等多项问题上做出重大让步，否则美国已准备好对中共在各方面加强施压。
 <br/>
 <br/>
 美国会众院外委会亚太小组主席Ted Yoho：〝我们希望能有有效的会谈。不是有效的会谈，而是有效的行动。〞
 <br/>
 <br/>
 不过专家认为，中共恐怕很难在关键问题上做出令美国满意的让步。
 <br/>
 <br/>
 中国问题专家章家敦：〝但即使双方达成协议，中共也会违背的。因为中共根本没有准备要诚实地与其他国家进行交易。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
