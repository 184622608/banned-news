### 德国起诉2华裔工程师 盗窃商业机密或领刑4年
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】德国化工企业朗盛集团（Lanxess AG）一名华裔工程师和他的助理被德国检方提起刑事指控，理由是他们涉嫌窃取朗盛化工的商业机密，并试图在中国仿制朗盛的化学反应设备。舆论认为，该案显示德国打击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共工业间谍.html" target="_blank">
  中共工业间谍
 </a>
 活动的力度正在加强。
 <br/>
 <br/>
 据路透社15日报导，德国科隆市的一名检察官透露，总部位于科隆的化工企业朗盛集团大约两年前向警方报案，指该企业的一名48岁华裔德籍工程师涉嫌盗窃该公司的商业机密。经过调查，检方今年6月对这名华裔工程师提起了刑事指控。
 <br/>
 <br/>
 德国检方称，当初朗盛集团在发现该企业的一些与化学反应设备相关的数据失窃后，就解雇了这位华裔工程师。
 <br/>
 <br/>
 此外，一名40岁的华裔德国籍助理也因此案而一同被起诉。因为检方发现，涉事的这位华裔工程师曾使用中文通过电子邮件与那名华裔助理沟通，试图在中国境内复制朗盛化学反应堆生产化学制品。
 <br/>
 <br/>
 检方表示，如果起诉的罪名成立，这两人将面临长达4年的监禁。此案还涉及第3名共犯参与其中，不过目前该名共犯下落不明。
 <br/>
 <br/>
 报导表示，本次案例与以往类似的案子有所不同。过去曾发生过的几起商业机密窃取案都发生在中国办厂的德国制造企业里，盗窃机密的人员大多数在中国当地雇傭的人员，但朗盛集团的这起商业机密盗窃案却发生在德国国内，这在过去是很少见的。
 <br/>
 <br/>
 目前，德国检方尚未对外公开此案嫌犯的身分。
 <br/>
 <br/>
 德国情报机构BfV在7月发布的年报中警告企业称，中国很想成为高科技产品出口国，有可能通过窃取知识产权来实现，报告并称很难分辨国家间谍活动和工业间谍活动。
 <br/>
 <br/>
 德国情报机构〝联邦宪法保卫局（BfV）〞在今年7月发布的年度报告中曾指出，中国（中共政府）很想打造一个〝高科技产品的出口国〞，但他们有可能会透过窃取智慧财产来实现这个目标。而这些间谍现在很难辨别是〝国家间谍〞还是普通的〝工业间谍〞。
 <br/>
 <br/>
 就在不久前，美国司法部刚刚于11月1日发布消息称，美国联邦大陪审团对中国福建晋华集成电路有限公司和台湾联电电子股份有限公司以及三名台湾人提出了刑事起诉，指控这两家公司及同谋者从事经济间谍活动，盗窃了美国美光公司的价值高达87.5亿美元的技术资料。
 <br/>
 <br/>
 舆论认为，上述案例表明，德国正跟随美国川普政府的步伐，加大了保护其本国企业智慧财产的力度。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：秦锋）
