### 情报官警告：中共秘密网军 正拦截破译全球通讯
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】印度情报机构官员警告，中共编号61398的秘密网军部队，在中共情报官员协助下，从事网络间谍活动，定位情报搜集、追踪网络资讯来源，且拦截和破译全球数字通讯，攻击其他国家政府和工业组织。
 <br/>
 <br/>
 11月8日，印度Zee News报导，一名不愿具名的印度情报官员说，他们发现，中共负责网络战的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_61398部队.html" target="_blank">
  61398部队
 </a>
 旗下的骇客团体在中共情报官员的协助下，从事网络间谍活动，透过对电脑系统长期入侵的〝高级长期威胁〞（APT）系统，攻击他国政府和工业组织。
 <br/>
 <br/>
 他披露，61398部队透过网络定位情报搜集、追踪网络资讯来源，且拦截和破译全球数字通讯，同时使用木马恶意软件透过网络窃取资料。
 <br/>
 <br/>
 印度国防设施也可能遭受了61398部队的网络攻击。
 <br/>
 <br/>
 该情报官员说，最近印度首艘自制核动力潜舰〝歼敌号〞（INS Arihant）完成第一次〝威慑巡航〞，这艘潜舰可携带在水下发射核子弹头的飞弹，同时也从美国取得可侦测海上和水下敌方舰艇的P8I海上巡逻机，他认为，这些都可能成为中共骇客的目标。
 <br/>
 <br/>
 7月份，就有海外中文媒体称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共军方.html" target="_blank">
  中共军方
 </a>
 在上海和北京特别部署的61398网络部队，6月底已完成史上最大规模的〝扩编〞，除了继续加大各类网络和社交平台的监控外，对全球特定目标进行分散式阻断服务攻击（DDoS），将是日常〝训练〞的主要任务。
 <br/>
 <br/>
 报导引述中共军方高层知情者的话称，此次扩编方案，早已报请中南海最高层核准，整个行动历时长达半年，主要包括从各大军事院校和科研机构，以及战区选拔各类专业人才。再对相关人员的整个家族和好友圈，进行周密细致和深入的背景审查，经层层把关后，分别抽调上海和北京。
 <br/>
 <br/>
 一名技术主管透露，网络部队可以综合运用DDoS攻击的多种方式，而上海和北京并无明确分工，但可以协调作战，并已进行数以万计的远程测试。
 <br/>
 <br/>
 <h4>
  揭秘中共61398部队隶属的
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总参三部.html" target="_blank">
   总参三部
  </a>
 </h4>
 <br/>
 中共的61398部队也叫总参三部二局，2013年美国网络安全公司Mandiant的报告分析，这个单位主要针对美国。
 <br/>
 <br/>
 中共61398部队，其上级总参三部是一个非常神秘的部门。61398部队的司令总部隐藏于上海一幢12层大厦内。而总参三部的总部设在北京。
 <br/>
 <br/>
 在上海、青岛、珠海、哈尔滨、成都等地驻有机构，职能是负责〝信号情报〞，工作估计包括监听电子通讯、分析卫星情报、破解密码等，编制估计有数万人。
 <br/>
 <br/>
 外国专家说，总参三部的运作单位遍布于全中国，它从专业院校招募，估计拥有10万多名黑客，语言学家，分析师和官员，组成10多个军事情报局。它的多个子部门根据地域和任务的不同而分担不同责任。
 <br/>
 <br/>
 根据《华尔街日报》2013年获得对中共总参三部一些秘密运作的了解。这是一个全球监视组织，是中共军事战略的核心，其任务是监视和分析全球大量通讯，包括大使馆电文，企业电子邮件和刑事犯罪网络。
 <br/>
 <br/>
 位于维吉尼亚的智库2049项目的执行主任斯托克斯（Mark Stokes）提供了未公开发表的对总参三部的分析。
 <br/>
 <br/>
 在上海北部总参三部一个基地，一块农田树立着几十根细细的无线电发射塔。
 <br/>
 <br/>
 在北京市外，总参三部基地躲在山区一个秘密小镇，隐藏在看上去普普通通的住宅楼后——而从附近的山上可以看得到其中的70多栋建筑物和足球场。它被认为主要用于监控欧洲的运作。
 <br/>
 <br/>
 报导说，总参三部作为共产党红军的地下单位，建立于上世纪30年代。它截获他国的电报和无线电信息，帮助毛泽东的军队在1949年获得政权。
 <br/>
 <br/>
 近年随着北京现代化的高科技国防武器库发展，总参三部越来越让各国政府和全球公司不安。
 <br/>
 <br/>
 从北京附近的山头，中国版的NSA（美国国安局）在监视俄罗斯和追踪导弹。它的军事专家在一个被称为中国夏威夷的岛屿上分析互联网电话，它从隐藏在居民楼后面的一个秘密小镇上窃听欧洲。
 <br/>
 <br/>
 前美国官员说，这个组织还围绕着上海建立了一些专门用于观察美国的设施，设施之一位于连接中国和美国的主要越洋通信电缆附近。
 <br/>
 <br/>
 监控活动在2013年5月份密集增多，当时美国司法部起诉了中共总参三部的5名军官，指控他们10年来再三盗窃美国商业机密。虽然中共拒绝承认指控，但是美国司法部证据确凿，以网络间谍和黑客的罪名起诉了这5名军官。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明)
