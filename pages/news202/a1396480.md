### 临床专家：大麻成瘾风险大无药可治
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻合法化.html" target="_blank">
  大麻合法化
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 已经成为事实，在加拿大这个移民社会引起相当大的反响，也让医学界相当关注，加拿大成瘾及精神健康中心（The Centre for
 <br/>
 Addiction and Mental Health简称CAMH)专门研究成瘾临床治疗的科学家Bernard Le Foll医生说，30%吸
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大麻.html" target="_blank">
  大麻
 </a>
 的人会出现不同程度的成瘾症状。
 <br/>
 <br/>
 成瘾及精神健康中心成瘾临床治疗科学家Dr.Bernard Le Foll：〝与大麻使用有关的主要症状是成瘾的风险，就是用量失控，估计约30％的使用者会在人生的某个阶段失去控制，也就是说他们会使用更多的大麻，无法控制大麻店使用量。估计9％会发展成更严重的成瘾形式。〞
 <br/>
 <br/>
 开始吸大麻的年龄越小，越容易出现成瘾问题，如果有个人或家族成瘾史，风险更高。
 <br/>
 <br/>
 刚开始的症状是，情绪、食欲发生变化，睡眠不规律，对大麻的渴望上升，经常性的想着大麻，和家人、朋友相处的时间变少，和他人的关系变得冷漠，工作表现下降。
 <br/>
 <br/>
 Dr.Bernard Le Foll：〝你经常吸食大麻，当你停止使用，你会出现大麻停断症状：需要吸食更多才能达到同样效果，控制使用量也很困难，通常在停用大麻后的一天或两天出现戒断症状，停用四五天后达到高峰。〞
 <br/>
 <br/>
 严重的成瘾症状会演变成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_精神分裂症.html" target="_blank">
  精神分裂症
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抑郁症.html" target="_blank">
  抑郁症
 </a>
 、焦虑、妄想症等。
 <br/>
 <br/>
 如何戒掉大麻，目前还没有有效药物，主要靠本人有决心和动力去戒，精神和心理谘询作为辅助。
 <br/>
 <br/>
 Dr.Bernard Le Foll：〝目前我们没有有效的药物来帮助人们戒掉大麻，不像戒菸，你可以使用尼古丁贴。〞
 <br/>
 <br/>
 〝他们必须有动力去戒掉大麻，而且他们得明白想如何度过人生，如何保持这种改变，改变自己的行为，有时人们可以自己做到，有时不那么容易，需要来自专业医务人员的帮助。〞
 <br/>
 <br/>
 戒掉大麻的心理治疗过程需要3-4个月的时间，每个星期参加小组辅导，辅助以家庭医生的支持。在安省治疗大麻毒瘾一般能由安省医疗保险系统支付。
 <br/>
 <br/>
 Le Foll医生认为，公众对大麻危害性所知甚少，让人们了解吸菸的危害就花了30-40年的时间，对大麻危害的公众教育即使加大力度，也需要很长时间。
 <br/>
 <br/>
 新唐人记者刘海英多伦多报导
