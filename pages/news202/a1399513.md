### 彭斯誓言力保印太自由 反对中共扩张
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】周四，出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东盟峰会.html" target="_blank">
  东盟峰会
 </a>
 的美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 表示，美国对印太地区有着坚定和持久的承诺。彭斯当天还与普京以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 有短暂交流。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 在峰会上阐述美国的印太战略和愿景，并重申美国对该地区的承诺。他说，东盟是美国〝不可或缺也无可取代〞的战略伙伴。
 <br/>
 <br/>
 美国副总统 彭斯：〝美国对印太地区的承诺坚定而持久。我们致力于坚守海洋和天空的自由，肩并肩地与你们一起维护航行自由。美国决心保障你们国家的主权安全，包括在陆地、海洋和数位化世界。〞
 <br/>
 <br/>
 彭斯强调，美国的印太战略不会排挤任何国家，但要求各国尊重邻国主权，以及区域秩序。帝国主义和侵略行为，在印太地区没有立足之地。
 <br/>
 <br/>
 彭斯的讲话被认为是针对中共近年来，在印太地区的扩张。
 <br/>
 <br/>
 峰会期间，彭斯还与韩国总统文在寅举行了双边会晤。
 <br/>
 <br/>
 彭斯透露，川普总统计划2019年与金正恩第二次会面，但美国不会在朝鲜无核化立场上作出任何让步，也不会重蹈覆辙。
 <br/>
 <br/>
 文在寅承诺美韩保持密切合作，最终实现朝鲜半岛无核化。
 <br/>
 <br/>
 尽管美中代表在本次峰会没有正式会谈的计划，不过在会议间隙，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 主动与彭斯交谈，谈话内容目前还不清楚。
 <br/>
 <br/>
 中共在东盟会上力推〝区域全面经济伙伴关系〞协议，但该协议的谈判被推迟到明年，当局的努力遭冷遇。
 <br/>
 <br/>
 为期三天的峰会周四落幕，新加坡总理李显龙在闭幕礼上，交棒给下任主席国泰国总理巴育。
 <br/>
 <br/>
 新唐人记者曹菁综合报导
</div>
