### 川普披露底线中共或难接受 库德洛:G20会晤将摊牌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会将在月底于阿根廷举行，外界预计美中两国元首将就贸易问题直接摊牌。白宫国家经济委员会主任库德洛披露，美国总统川普已经开出了要求中共接纳的主要条件，要求美中达成的协议必须符合美国最佳利益，否则该协议不可能达成。
 <br/>
 <br/>
 20日，川普的首席经济顾问、白宫国家经济委员会主任库德洛在接受FOX新闻采访时表示，川普对于与北京达成贸易协议仍持乐观态度，但必须得符合美国的利益，否则协议不可能达成。川习两人在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会的会面，将是美中贸易的关键时刻。
 <br/>
 <br/>
 库德洛说，协议必须包括〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 保护、必须包括所有权改变、停止强迫技术转让，零
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，必须撤销非关税壁垒，必须有可执行性，必须有严格时间表，必须超过我们现在所看到的东西。〞
 <br/>
 <br/>
 《自由亚洲电台》报导认为，美方要求中共撤销非关税壁垒，即是要求美国的金融机构如银行，保险公司，甚至是谷歌等原本受到限制的企业，将来均可无条件进入中国，而中共在这些领域恐难作出让步。
 <br/>
 <br/>
 另外，川普在白宫接受记者提问说，〝如果我们没达成协议，我还有另外价值2500亿美元的关税。而且相信我，我会对他们实施。〞
 <br/>
 <br/>
 外界认为，川普提到的2500亿美元，应指对价值2670亿美元的大陆输美商品加征关税，他此前已多次警告中共，华盛顿不排除采取更多关税措施。
 <br/>
 <br/>
 另一方面，美国贸易代表办公室根据〝301条款〞，对中共的知识产权和技术转让政策进行调查后，美贸易代表莱特希泽在20日透过声明表示，最新调查显示中共没有采取任何实际行动，从根本上改变其不公平、不合理，且扭曲市场的举措。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：李泉）
