### 美国朝鲜问题特使：朝鲜去核有信心
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】周一，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜问题.html" target="_blank">
  朝鲜问题
 </a>
 特使史蒂芬・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比根.html" target="_blank">
  比根
 </a>
 ，在首尔会见韩国外交部半岛和平交涉部部长李度勋，以及韩国外长康京和，讨论平壤与华盛顿会谈的进展。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比根.html" target="_blank">
  比根
 </a>
 表示，要正式结束朝鲜战争，首先必须解除朝鲜核武器。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国特使.html" target="_blank">
  美国特使
 </a>
 比根：〝我们有共同目标，结束朝鲜半岛70年来的战争和敌对行动。达成这一目标的首要条件，是实现最终的，可完全核实的朝鲜无核化。我绝对相信这可以实现。〞
 <br/>
 <br/>
 最近几个月，韩国和朝鲜关系取得进展，引发美国对一系列问题的关注，尤其是联合国对朝鲜核计划和导弹计划的制裁问题。
 <br/>
 <br/>
 美国特使比根表示，华盛顿和首尔正在努力，确保两国在对待
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜问题.html" target="_blank">
  朝鲜问题
 </a>
 上保持一致。
 <br/>
 <br/>
 韩国外交部半岛和平交涉部部长李度勋：〝无核化进程处于关键时刻，我们需要尽可能多举行会议，确保我们两个盟国间没有任何隔阂。〞
 <br/>
 <br/>
 比根希望尽快和朝鲜进行谈判。
 <br/>
 <br/>
 美国特使比根：〝我们非常期待尽快开始和朝鲜，进行工作层面的谈判。〞
 <br/>
 <br/>
 这个月早些时候，比根曾提出和朝鲜外交部副部长崔善姬举行会面。
 <br/>
 <br/>
 新唐人记者毕心慈、宇微综合报导
</div>
