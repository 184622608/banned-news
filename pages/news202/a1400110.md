### 难民故意破坏 美国短暂关闭边境严防
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 短暂关闭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境通道.html" target="_blank">
  美墨边境通道
 </a>
 ，加强安全措施。一名官员透露，大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 以男性为主，藏有500多罪犯，却利用女性和儿童博取同情。
 <br/>
 <br/>
 周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 关闭了墨西哥进入美国的边境通道数小时，部署铁丝网和混泥土路障。同时关闭了加州圣地亚哥、圣思多罗口岸两条人行道中的一条通道，防止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 大规模闯关。
 <br/>
 <br/>
 周一，美国国土安全部部长尼尔森发推特表示，一些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 故意在边境入境口岸造成破坏。
 <br/>
 <br/>
 推文说：〝进入美国有合法和非法（两种）方式。我们部署了额外兵力，来保护我们的边界。〞
 <br/>
 <br/>
 约六千名中美洲难民，已抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥边境.html" target="_blank">
  墨西哥边境
 </a>
 城市蒂华纳和墨西加利，相信人数将增加到一万人。数百名难民待在蒂华纳收容所内，等待进入美国申请庇护。
 <br/>
 <br/>
 也有些人改变想法，希望留在墨西哥找工作或寻求庇护。
 <br/>
 <br/>
 中美洲难民：〝我会去抗争，如果不能进入美国，我会在墨西哥申请庇护，在这找工作。〞
 <br/>
 <br/>
 周日，蒂华纳数百居民反对难民的游行，引起收容所内难民的担忧。
 <br/>
 <br/>
 24岁的洛佩兹和丈夫，带着两个年幼的女儿，为逃避黑帮暴力，从洪都拉斯来到墨西哥。
 <br/>
 <br/>
 中美洲难民Jessica Lopez：〝我们听说墨西哥的一些人反对我们，那些不想要我们的人，准备将我们从这赶走，我真的感到害怕。〞
 <br/>
 <br/>
 周二，美国加州北区地方法院联邦法官蒂卡尔，发布临时禁制令，阻止川普总统取消非法入境难民申请庇护资格。
 <br/>
 <br/>
 新唐人记者毕心慈、明玉综合报导
</div>
