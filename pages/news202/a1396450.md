### 川普欲退出《中导条约》 俄表态愿与美方合作挽救条约
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】美国国家安全顾问博尔顿周一（22日）先后与俄罗斯联邦安全局长派楚希夫和俄罗斯外交部长拉夫罗夫会晤。随后，俄国安全会议发表声明称，俄方愿就冷战时期签定的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_《中导条约》.html" target="_blank">
  《中导条约》
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_INF.html" target="_blank">
  INF
 </a>
 ）与美方进行合作。
 <br/>
 <br/>
 博尔顿（John Bolton）周一与拉夫罗夫（Sergei Lavrov）讨论这项条约的存废，也与俄国安全会议主管官员派楚希夫（Nikolai Patrushev）会谈了近5个小时。
 <br/>
 <br/>
 在派楚希夫和博尔顿会谈后，俄国安全会议发表声明说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中程核飞弹条约.html" target="_blank">
  中程核飞弹条约
 </a>
 崩解，将对〝国际禁止扩散和武器管制的整个法律体系形成严重打击〞。
 <br/>
 <br/>
 声明中强调，派楚希夫与博尔顿的会谈〝具有建设性而且务实〞，俄方愿与美国合作以挽救这项条约。
 <br/>
 <br/>
 两人的会谈还商讨了新战略武器裁减条约（New START）可能延长5年的相关问题，该条约预定在2021年到期。此次访俄，博尔顿会就核武条约、叙利亚、伊朗、朝鲜等问题与俄方沟通。
 <br/>
 <br/>
 博尔顿周二（23日）将与俄国总统普京商讨美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）的计划，也就是放弃这项存在30年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_《中导条约》.html" target="_blank">
  《中导条约》
 </a>
 。
 <br/>
 <br/>
 10月20日，川普在内华达州参加选举造势活动后，登上空军一号前告诉记者：〝俄罗斯违反协议，他们多年来一直违反协议。……我不知道为什么奥巴马总统不谈判或退出。我们不会让他们违反核协议，他们出去发展武器，而我们不被允许。〞
 <br/>
 <br/>
 川普告诉记者：〝我们是那些留在协议中的人，我们尊重协议，不幸的是，俄罗斯没有履行协议。所以我们将终止协议，我们将退出。〞
 <br/>
 <br/>
 川普所指的协议，是已故美国总统里根和苏联领导人戈尔巴乔夫于1987年签署的《中程核飞弹条约》，简称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_INF.html" target="_blank">
  INF
 </a>
 ，双方依约要各自销毁一定数量的中程飞弹，并禁止持有、制造或试射射程介于500至5500公里的中程和中短程常规核导弹，目的是缓解冷战时期的紧张气氛。
 <br/>
 <br/>
 美国政府指控俄罗斯在过去4年发展和部署新型巡航导弹，威胁欧洲国家以及与西方结盟的苏联国家，已违反条约。
 <br/>
 <br/>
 其实签订该条约时，代表美国签字的里根总统当时更倾向于太空防御核导弹计划。川普政府也正在力促国会尽快通过组建美国太空军的计划。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
