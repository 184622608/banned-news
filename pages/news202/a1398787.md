### 汽车闯入英相和比国总理车队 2警摔车受伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国首相.html" target="_blank">
  英国首相
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特丽莎梅.html" target="_blank">
  特丽莎梅
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比利时总理.html" target="_blank">
  比利时总理
 </a>
 米歇尔的车队，9日，为纪念
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第一次世界大战.html" target="_blank">
  第一次世界大战
 </a>
 结束100周年，预备一座英国公墓献花，不料遭一名汽车驾驶干扰，造成两名比利时摩托车警察受伤。当局其后指事件为交通意外，并不涉及恐袭。
 <br/>
 <br/>
 综合媒体报导，肇事驾驶起先在比利时西部蒙斯（Mons）附近的高速公路上对高速行进中的车队超车，但还没有完全超越就试图开进车队使用的车道，导致在车队中段执行警戒任务的两名摩托车警察失控翻车，差点撞上米歇尔的座车。
 <br/>
 <br/>
 随后，两人车队继续原定行程前往法国参加后续纪念活动，一同为一战中首位和最后一位牺牲的英国士兵，献上白玫瑰和罂粟花制成的花圈。
 <br/>
 <br/>
 米歇尔（Charles Michel）的发言人事后告诉路透社，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特丽莎梅.html" target="_blank">
  特丽莎梅
 </a>
 （Theresa May）的座车行驶在车队前头，没有目击到这起事件。并形容驾驶〝漫不经心〞，但不可能是一起攻击。
 <br/>
 <br/>
 不过，当局至今仍未公布涉事司机的身份和动机，亦未知意外原因。
 <br/>
 <br/>
 (责任编辑：卢勇信)
