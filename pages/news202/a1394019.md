### 入境新西兰要注意 拒交手机密码将被重罚
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新西兰.html" target="_blank">
  新西兰
 </a>
 的旅客要注意了，10月1日开始实施的〝2018报关与征税法〞规定，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境.html" target="_blank">
  边境
 </a>
 检查官有权要求入境者告知
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手机密码.html" target="_blank">
  手机密码
 </a>
 ，拒绝查看者最高可罚5000新西兰币（约3200美元）。此举也引发人权团体抗议。
 <br/>
 <br/>
 据2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新西兰.html" target="_blank">
  新西兰
 </a>
 《海关法》的最新规定，海关官员现在拥有了搜查〝信息储存设备的全部搜索权力〞，该权利包括〝要求设备拥有者提供访问设备所必要的信息〞。
 <br/>
 <br/>
 也就是说，当地入境检查官员有权要求旅客解锁其手边任何一个电子装置，并〝初步搜索〞该机资料，但不会查看云端数据与网页纪录。如果入境者不提供密码会被重罚5000新西兰币。
 <br/>
 <br/>
 除了检查设备信息之外，海关还有权力复制设备中的数据，并且还可以没收以进行进一步搜索。这个新规，是全球各地入境检查当局对不愿提供
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手机密码.html" target="_blank">
  手机密码
 </a>
 者开罚的首例。
 <br/>
 <br/>
 对此，〝新西兰公民自由理事会〞（NZCCL）批评新规〝严重侵犯个人隐私〞。
 <br/>
 <br/>
 但新西兰海关发言人表示，修法是有必要的，因为现在〝多数被禁资料与文件都以电子方式储存〞。而且也不是所有的设备都会被检查，自2017年开始抽检电子装置以来，至今1400万名旅客中，只有537台设备被查。
 <br/>
 <br/>
 这些潜在犯罪行为包括持有令人反感的资料、儿童色情内容，或与毒品、金融犯罪有关的信息。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
</div>
