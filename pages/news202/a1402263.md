### 〝黄背心〞诉求升级 法国政府称周六恐有大暴动
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】法国〝黄背心〞运动经过近20天的发酵，近日已演变为暴力骚乱事件。虽然法国政府已宣布取消调涨燃油税的政策，但〝黄背心〞的诉求已扩大至多个方面，抗议行动将继续下去。法国政府称事态危急，本周六恐会出现大暴动。
 <br/>
 <br/>
 法国抗议燃料税调涨的〝黄背心〞运动从上个月17日启动以来，抗议的人群情绪越来越亢奋，逐渐演变为夹杂着打砸抢烧等暴力行为的骚乱。法国政府为平息众怒，日前已宣布取消调涨燃油税，但群情高昂的〝黄背心〞们并不打算就此罢休。
 <br/>
 <br/>
 〝黄背心〞成员玛莉琳（Maryline）6日接受RMC广播电台访问时称，取消燃油税虽然是好消息，但他们已不满足于此，因为人们认为政府还会用其他办法来弥补取消燃油税的损失。她说：〝我们已经习惯政府右手给个东西，又用左手加倍收回〞。
 <br/>
 <br/>
 玛莉琳表示，不只有燃油税，政府必须考虑抗议者的其他要求，要做更多让步，因为不高兴的人很多，不会罢休。
 <br/>
 <br/>
 据法广报导，〝黄背心〞成员日前已宣称，本周六（12月8日）将会继续抗议，他们认为法国政府应做更多让步，包括提高退休金、提高最低工资、撤销其他税目、恢复课征富人税等等。

 不过，法国总统府12月6日对媒体表示，根据基层打探的消息，事态十分危险，8日在巴黎恐会有一场数千人的〝大暴动〞，他们只想用暴力进行大破坏、甚至杀人。
 <br/>
 <br/>
 法国政府称，因为有太多人滥用暴力，导致现在事态变得很危险，政府呼吁民众本周六尽量待在家中，以确保自身的安全。
 <br/>
 <br/>
 据了解，法国政府原本计划在明年1月1日调涨燃料税，不料因此引发全国性的〝黄背心〞示威运动，之后随着公众情绪日渐失控，事态演变为严重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_都市暴动.html" target="_blank">
  都市暴动
 </a>
 ，在全国范围内已造成3死263伤。
