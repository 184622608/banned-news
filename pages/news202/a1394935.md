### 孟宏伟因贪腐被抓？妻四字披露真实原因
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 在中国被查的消息传出后，孟妻召开新闻发布会，并高调接受多家西方媒体采访。她在接受CNN采访时，用四字披露孟宏伟被逮捕的真实原因：〝政治原因〞。
 <br/>
 <br/>
 北京当局7日深夜宣布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 涉嫌贪污受贿落马，次日凌晨，公安部长赵克志紧急召开会议，宣布成立工作组，查处孟宏伟的同党。
 <br/>
 <br/>
 该消息传出前的几小时，孟妻Grace Meng在法国里昂召开记者招待会上说，孟宏伟失踪了，她要追求所谓的〝真相、历史责任〞等。
 <br/>
 <br/>
 孟宏伟被查的消息公布后，Grace Meng先后接受美联社（AP）、美国有线电视新闻网（CNN）等媒体的专访，高调替孟发声。
 <br/>
 <br/>
 她说，孟宏伟失踪后，她接到陌生男子的威胁电话。对方说，他们有〝两组人马〞已经过来，目标就是她。
 <br/>
 <br/>
 Grace Meng还替孟宏伟开脱，说北京当局调查孟宏伟只是借口。她在接受CNN采访时说，她认为孟宏伟被中共逮捕的真实原因，是因为〝政治原因〞。
 <br/>
 <br/>
 Grace Meng还透露，她和孟宏伟有两个7岁的双胞胎儿子，她现在和媒体联系，是因为担心一家人的安全，需要得到帮助等。
 <br/>
 <br/>
 目前，法国警方已经对Grace Meng母子进行了安全保护，但Grace Meng不愿意公开自己的相貌，外界亦不知晓她的中文姓名。
 <br/>
 <br/>
 而孟宏伟被抓，外界普遍认为，其做为中共前政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 的〝余毒〞被清洗。
 <br/>
 <br/>
 孟宏伟长期在中共公安部任职，周永康2003年3月出任公安部部长后，孟宏伟2004年4月升任副部长、党委委员，2013年周落马，他的多名公安部旧部也陆续被查。
 <br/>
 <br/>
 2017年底，孟宏伟突然被免去中共海洋局副局长、海警局局长职务；今年4月，他又被免去公安部党委委员一职。同时，孟多次缺席公安部高层会议。中共官场早已传出其仕途〝很不妙〞的消息。
 <br/>
 <br/>
 日本《每日新闻》10日称，孟宏伟被捕的背后原因，很可能是因为中共想要完全控制，掌握着中共秘密的公安部门。
 <br/>
 <br/>
 文章称，中共公安和司法部门统称〝政法系〞，〝政法系〞曾因周永康的权势被称为〝独立王国〞，而公安部持有中共机密情报是周的力量之源，孟宏伟也拥有中共重大人物的违法证据。
 <br/>
 <br/>
 而孟妻高调接受媒体采访，并寻求帮助之举，非常罕见，表明她手里或握有令中共恐惧的核弹。
 <br/>
 <br/>
 此前，中共前政治局委员薄熙来，以及中共前中办主任令计划等人被调查后，薄之子薄瓜瓜、令计划之弟令完成都在海外，但他们都保持〝低调〞，尤其是令完成，被指手握〝核弹性秘密材料〞，但他也没高调发声。
 <br/>
 <br/>
 原北大教授夏业良对美国之音说，孟妻在法国开新闻发布会，使用的都是高调词汇。看得出来，孟被抓前已有充分准备，孟妻打算藉助国际平台给中共警示，否则就曝光手中的相关材料。
 <br/>
 <br/>
 夏业良说，希望本次能够看到孟妻爆出猛料，不要像令完成一样。他还说，王立军逃往美领馆，以及孟妻在法国开发布会，都是求助于西方自由世界。不过，王立军没有充分准备退路，孟宏伟则是事先策划、步步为营。
 <br/>
 <br/>
 对于孟妻求助西方社会的问题，很多网民认为孟宏伟被抓是咎由自取、作恶多端的报应。做了10多年中共的酷吏刀把子，他曾经让很多人被失踪，现在轮到他自己了。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
