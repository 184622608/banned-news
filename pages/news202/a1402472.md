### 港媒：孟晚舟还有第8本护照 美国没掌握
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月09日讯】在华为CFO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 12月7日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保释听证会.html" target="_blank">
  保释听证会
 </a>
 上，加拿大检方以潜逃风险为由，没有批准她的保释申请。其中一项风险是，美方证据指她同时拥有7本护照，辩方无法讲清楚。而港媒随后揭露，孟至少还有一本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国护照.html" target="_blank">
  中国护照
 </a>
 ，美方尚未掌握。
 <br/>
 <br/>
 中国华为创始人任正非之女、华为首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 12月1日在加拿大被捕。12月7日举行了长达一天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保释听证会.html" target="_blank">
  保释听证会
 </a>
 ，加拿大检方拒绝保释，控辩双方展开激烈辩论，最终没有做出裁决，将于10日再次开庭。
 <br/>
 <br/>
 听证会上，检方提供美国司法部文件指证，孟晚舟在过去11年间，获发至少7本不同国家或地区的护照，其中包括4本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国护照.html" target="_blank">
  中国护照
 </a>
 （号码开首为G）和3本香港特区护照（号码开首为K）。
 <br/>
 <br/>
 孟晚舟辩方律师马丁（David Martin）辩称，这些多为过期护照，她现在只有两本护照有效：一本中国护照和一本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港护照.html" target="_blank">
  香港护照
 </a>
 。香港护照在她被抓时被加拿大官方收走，她的中国护照正从中国寄来，也准备交出。
 <br/>
 <br/>
 不过，根据公开资料，号码首字母为G的中国护照是普通因私护照，而中国普通护照的有效性一般为10年，香港护照有效期一般也是10年。
 <br/>
 <br/>
 据此判断，孟晚舟在过去11年获得4本中国护照，3本香港护照，那么她申请这些护照时不大可能是使用同一身份（除非她多次丢失护照再补办，而这种可能性很小）。
 <br/>
 <br/>
 如果孟晚舟是使用多重身份获得这些护照的话，即使其中某一身份的护照有效期满，她也很容易使用这个身份再次申请换发新护照。也就是说，辩方声称她现在只有两个有效护照的可信度极低。
 <br/>
 <br/>
 另外，据香港《明报》8日报导，通过翻查香港公司注册处纪录发现，孟晚舟于2004年曾在华为技术投资有限公司的周年申报表上，申报持有首字母为P的中国护照。
 <br/>
 <br/>
 号码首字母为P的护照，一般是中国普通因公护照。虽然孟晚舟2004年申报的那本护照目前早已过期，但有效期满后同样很容易申请换发同样类型（首字母为P）的中国护照。
 <br/>
 <br/>
 也就是说，《明报》消息显示，孟晚舟很可能至少还有一本美方尚未掌握的中国护照。
 <br/>
 <br/>
 《明报》还引述专家分析称，根据中共法例，每人只容许持有一个户籍，如该人持有香港永久性居民身份，就被视为放弃中国大陆户籍。而根据香港法例，若持有香港特区护照，也不允许同时持有中国护照。
 <br/>
 <br/>
 不过，在中共治下的特权社会里，政商界人物同时持有多国护照的情况非常普遍。
 <br/>
 <br/>
 据报，在7日的听证会上，孟晚舟的律师提出巨额保释金和多达12项严厉的条件，以争取孟晚舟获得保释，但依然无法打消加拿大检方的顾虑。
 <br/>
 <br/>
 华为被指同时具有中共军方和中共情报机构的背景。孟晚舟拥有多本护照，也被指符合中共情报人员的特征。
 <br/>
 <br/>
 根据听证会透露的信息，美国司法部指控孟晚舟当年亲自在伊朗建立〝过桥〞公司，欺骗国际银行机构，和伊朗从事非法交易，而且美方已掌握确凿证据。根据美国法律，如果孟晚舟被引渡到美国受审，将面临最高30年刑期。
 <br/>
 <br/>
 在7日听证会结束后，海外中文网络上开始热传〝如果孟晚舟逃回中国会被驱逐〞的文章，被怀疑在为孟晚舟获得保释机会制造舆论。
 <br/>
 <br/>
 目前美国官方宣布，美中已经达成共识，孟晚舟事件不会影响美中贸易谈判。观察人士指，这显示北京为向美方〝求和〞，不会为孟晚舟向美方提出交涉，孟要脱身只能另想对策。
 <br/>
 <br/>
 中共外交部8日召见了加拿大驻华大使，要求立即释放孟晚舟，〝否则必将造成严重后果〞。《美国之音》指，这是中共到目前为止对孟晚舟事件做出的〝最为强烈〞的反应。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
