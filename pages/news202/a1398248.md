### 12月31日起中国籍入境加拿大需提供指纹和照片
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 移民部宣布，从12月31起亚太和美洲国家的公民在入境加拿大前需要收集
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生物识别信息.html" target="_blank">
  生物识别信息
 </a>
 ，也就是要提供
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_指纹.html" target="_blank">
  指纹
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_照片.html" target="_blank">
  照片
 </a>
 。届时中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆公民.html" target="_blank">
  大陆公民
 </a>
 ，无论到加拿大旅行或学习、工作及申请移民都需要提供指纹和照片。不过，持有台湾护照和香港护照的人到加拿大旅行时不需要提供。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 移民和公民部宣布，正在扩大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生物识别信息.html" target="_blank">
  生物识别信息
 </a>
 收集计划。从2018年12月31日起，来自亚洲，亚太和美洲国家的国民在申请旅行签证、学习许可和工作许可，以及申请永久居民时需要提供
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_指纹.html" target="_blank">
  指纹
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_照片.html" target="_blank">
  照片
 </a>
 。
 <br/>
 <br/>
 移民部说，收集指纹和照片的目的是，阻止那些对加拿大的安全构成风险的人及罪犯进入加拿大，同时能简化和加快有合法身份的人们入境。
 <br/>
 <br/>
 中国公民目前还不需要提供指纹和照片，到了12月31日，申请到加拿大的旅行签证，申请学习、工作许可，及申请移民时需要提供这两种生物识别信息。
 <br/>
 <br/>
 香港和台湾，这两个地区到加拿大旅行不需要提供指纹和照片，只有在申请学习、工作许可或者申请永久居民身份的情况下才需要提供。
 <br/>
 <br/>
 中国的另一个特别行政区澳门，和中国大陆一样，无论旅行，还是学习、工作及申请移民加拿大都需要提供指纹和照片。
 <br/>
 <br/>
 加拿大的邻居，美国公民只有在申请移民到加拿大时才需要提供生物识别信息。
 <br/>
 <br/>
 收集生物识别信息由签证中心进行，加拿大允许外国人在任何国家提供，比如中国人在旅行美国时，想进入加拿大的话，可以在美国的任何加拿大签证中心提供指纹和照片。
 <br/>
 <br/>
 收费是，单人85加元，一个家庭的最高额为170加元。
 <br/>
 <br/>
 新唐人记者新宇报导
</div>
