### 北京妥胁内幕曝光 美首获中方关键承诺
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】中美元首在上周六的会晤，使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 暂时〝停火〞90天，不过外界对于双方达成的协议内容了解甚少。近日有外媒披露，美国首次获得了中共在几个重要议题上做出的让步承诺。
 <br/>
 <br/>
 在刚结束的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 〝川习会〞上，川普同意暂缓明年1月起对2000亿美元中国商品
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 从10%提高至25%的计划，北京则承诺立即购买更多美国产品，同时两国将就强制技术转让、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 保护、非关税壁垒、网路入侵和网路盗窃、服务业和农业等结构性问题，在未来90日内展开谈判。一旦谈判不成功，美国表明会继续提升关税。
 <br/>
 <br/>
 《美国之音》引述美国财政部长努钦（Steven Mnuchin）表示，美国对同北京达成贸易协议的前景表示乐观，因为中共在几个重要议题上做出让步的承诺，这是中共首次就结构问题和非结构问题拿出一个很长、具体的回应。
 <br/>
 <br/>
 努钦对CNBC说，〝这是我们第一次得到他们做出的承诺，这将是一个真正的协议〞。他表示，中共在谈判桌上还做出了另外1.2万亿美元的承诺，但有关细节需要双方磋商。这不仅是买美国货，这事关向美国公司开放市场，保护美国技术，这是非常重要的结构问题。
 <br/>
 <br/>
 不过他又强调，北京必须信守诺言，做出实实在在的进展，因为川普政府不会再忍受任何的拖延。
 <br/>
 <br/>
 值得注意的是，2017年4月6日，美中两国元首曾在美国佛罗里达州海湖庄园举行会晤，过后双方宣布开展为期百天的贸易谈判计划。但最后这一〝百日谈判〞并未阻止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 开打。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：李泉）
