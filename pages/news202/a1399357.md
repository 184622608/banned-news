### 【禁闻】涉活摘器官 陆专家被拒参加国际会议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】
 <b>
  涉
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活摘器官.html" target="_blank">
   活摘器官
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陆专家.html" target="_blank">
   陆专家
  </a>
  被拒参加
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际会议.html" target="_blank">
   国际会议
  </a>
 </b>
 <br/>
 <br/>
 大陆器官移植专家董家鸿，因被怀疑参与非法摘取人体器官，日前被以色列一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际会议.html" target="_blank">
  国际会议
 </a>
 取消参会邀请。
 <br/>
 <br/>
 董家鸿是北京清华大学长庚医院执行院长、中共军队301医院肝胆外科的原主任，他原本被第7届〝中以高科技投资峰会〞邀请为中方嘉宾。这个峰会11月18号将在以色列海法市开幕，主题涵盖生命科学、数字健康、机器人技术等。
 <br/>
 <br/>
 11月14号，〝自由亚洲电台〞报导说，国际组织〝终结中国滥用器官移植国际委员会〞10天前向该峰会的主办方〝海法经济组织〞发出联署抗议，要求主办方拒绝董家鸿参会。
 <br/>
 <br/>
 抗议随即得到主办方积极回应，目前大会已经确认，拒绝邀请董家鸿参会。
 <br/>
 <br/>
 <b>
  疑收中资
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_干预选举.html" target="_blank">
   干预选举
  </a>
  台爱同会成员被拘
 </b>
 <br/>
 <br/>
 将于11月24号举行的台湾九合一选举已经进入倒计时阶段，台北地检署11月12号拘捕了参选台北市区议员的中共外围组织〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱国同心会.html" target="_blank">
  爱国同心会
 </a>
 〞的秘书长张秀叶，还有会长周庆峻，指称他们疑似利用中共资金安排饮宴和旅游，贿赂选民，涉嫌违反《选罢法》。检方13号已经对他们实施限制居住和禁止出境。
 <br/>
 <br/>
 〝爱国同心会〞是中共专职打压法轮功的外围特务组织，成员多次涉及暴力殴打事件。
 <br/>
 <br/>
 检调单位日前接报说，张秀叶背后的团体，今年10月初，在台北市中山区一家餐厅举办餐会，筵开50多席、每席金额至少5千元新台币（约1,267港元），宣称庆祝中共建政69周年。张秀叶在舞台上大挥五星旗，并在会场拜票。
 <br/>
 <br/>
 台湾《联合新闻网》则报导说，张秀叶涉嫌在今年8月间，招待爱国同心会20多名干部，前往中国贵州旅游，每人旅费1万多元新台币，旅游时间约一周，食宿由中共支付。
 <br/>
 <br/>
 编辑/周玉林
