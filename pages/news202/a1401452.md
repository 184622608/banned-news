### 法官穿囚衣 印尼肃贪会突击地院押送人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼肃贪.html" target="_blank">
  印尼肃贪
 </a>
 委员会官员28日突击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雅加达.html" target="_blank">
  雅加达
 </a>
 南区地方法院，逮捕涉及在民事案件
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贪污.html" target="_blank">
  贪污
 </a>
 的2名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 在内的5名嫌犯。而法官穿上囚衣被押送的画面成为各大媒体焦点。
 <br/>
 <br/>
 印尼〝第一新闻〞（Berita Satu）报导，5名嫌犯分别是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雅加达.html" target="_blank">
  雅加达
 </a>
 南区地方法院的2名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 伊斯瓦尤（Iswahyu Widodo）和伊尔万（Irwan），法院的书记拉马丹（Muhammad Ramadhan）、律师阿里夫（Arif Fitrawan）和民间企业员工马丁（Martin P. Silitonga）。
 <br/>
 <br/>
 肃贪委员会在声明中表示，伊斯瓦尤和伊尔万涉及在民事官司中收受贿款。
 <br/>
 <br/>
 行贿的马丁和阿里夫供称，两名法官开口要20亿印尼盾（约13.9万美元），用来交换法院对一桩企业并购纠纷案件放水。
 <br/>
 <br/>
 经双方讨价还价后，行贿者最后决定给两名法官4万7000元新加坡币（约3.4万美元）和1.5亿印尼盾(1.2万美元)。肃贪委员会突击法院的行动中，现场查扣4万7000元新加坡币。
 <br/>
 <br/>

 扫除贪腐是印尼总统佐科威上任以来甚为看重的课题。印尼内政部长查约（Tjahjo Kumolo）日前表示，共有2357名政府官员已被证明涉及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贪污.html" target="_blank">
  贪污
 </a>
 案件，最迟将在年底革职。另有27名涉贪民意代表及高官褫夺公权终身，让这些曾在政坛意气风发的达官显将无法在印尼政坛卷土重来。
