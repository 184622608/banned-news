### 〝去美元化〞 俄买人民币大亏22亿美元
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 2017年起逐渐提高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 外汇存底比率推行〝去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美元.html" target="_blank">
  美元
 </a>
 化〞，如今占比达到5％。美中贸易战开打至今人民币暴跌9.4％，让俄罗斯惨赔22亿美元。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 央行发布的公告数据，统计至今年3月31日止，官方在1年内把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美元.html" target="_blank">
  美元
 </a>
 外汇存底占比降低1.6个百分点，占整体外汇比率的43.7％；欧元比则降低3.5个百分点，占22.2％；英镑比率则降低0.3个百分点，来到7.9％；
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 占比则为5％，约为230亿美元。
 <br/>
 <br/>
 俄罗斯《独立报》在10月2日发表文章，题目为《去美元化成本高昂》。文章引述官方数据分析说，莫斯科和北京积极鼓动去美元化可能面临巨大威胁，因为俄罗斯央行购入人民币后已出现大幅亏损，人民币贬值9.4%，给莫斯科带来了22亿美元的损失。
 <br/>
 <br/>
 自由亚洲报导，随着美中贸易战的不断升级以及美国对俄罗斯加大制裁，莫斯科与北京曾多次表示要加强合作，应对来自华盛顿的压力。其中包括加强金融领域的合作，加快去美元化进程。
 <br/>
 <br/>
 俄罗斯外贸银行行长安德烈．科斯京周二（10月2日）告诉媒体，他所提出的俄罗斯经济去美元化建议得到普京总统总体上的支持。不过，科斯京强调说，这并不意味着完全放弃使用美元。
 <br/>
 <br/>
 科斯京今年7月向普京提出的建议中，包括俄罗斯应在进出口业务结算中加快向使用欧元、人民币或卢布等币种结算过渡；将俄罗斯大型企业的注册地转移到俄法律管辖区内；通过俄罗斯存管机构发行欧洲债券；对证券市场所有参与者按统一从业规则实施许可证制度等等。
 <br/>
 <br/>
 同时，在中国官方媒体的报导中也可以看到诸如 〝全世界都在去美元化！人民币将取代美元〞等宣传。
 <br/>
 <br/>
 对此，有俄罗斯媒体和专家评论称，无论是北京还是莫斯科，目前都不具备挑战美元的实力。
 <br/>
 <br/>
 俄罗斯联邦经济发展部长马克西姆．奥列什金（Maksim Oreshkin）对此表示，〝去美元化〞窒碍难行，因为这需要懂得针对其他国家的的货币进行操作，懂得对流通性很差的货币进行运作，懂得这个国家的宏观经济等等，向这些方面的改变还需要数年的时间，现在办不到。
 <br/>
 <br/>
 报导称，此外，俄官媒关于去美元化的宣传已经在社会上造成了一定的恐慌。对此俄罗斯中央银行行长10月3日发表对外声明称，〝当局没有将美元储蓄帐户强制兑换为卢布帐户的计划，也无强制现金兑换的计划，甚至没有任何这样的想法。〞
 <br/>
 <br/>
 对此，有评论分析认为，因中俄之间的贸易额所占全球经贸总量的比例很小，而且中俄在同美国打贸易战时底气都不很足，这些因素都显示出，双方虽然拥有很强烈的愿望，但北京和莫斯科目前都不具备挑战美元的实力。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
