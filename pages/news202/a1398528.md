### 抵制一带一路 澳洲将援助太平洋岛国20亿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】近年来，中共靠着优惠贷款、捐助
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太平洋岛国.html" target="_blank">
  太平洋岛国
 </a>
 等方式，成为仅次于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 第2大的出资国。据报导，澳洲为对抗中共的影响力，预计将创立1个20亿澳币（约14.57亿美元）的基金，为太平洋岛国建设基础设施。
 <br/>
 <br/>
 据《路透》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 总理莫里森（Scott Morrison）周四（8日）赴昆士兰省演说时将宣布，计划创立1个20亿澳元（约14.57亿美元）的基金，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太平洋岛国.html" target="_blank">
  太平洋岛国
 </a>
 建设基础设施。这项基金将可大幅提高澳洲对太平洋国家及东帝汶发展基础设施的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_援助.html" target="_blank">
  援助
 </a>
 。
 <br/>
 <br/>
 澳洲将主要以成立新基础设施基金的方式，重新聚焦太平洋地区。这些援助经费将用于电信、能源、运输及水利等必要基础建设，料能进一步拓展援助资金的用途。
 <br/>
 <br/>
 澳洲与中共一直在争夺太平洋岛国的影响力，这些国家虽人口稀少，却掌控丰富的海洋资源。
 <br/>
 <br/>
 自2011年起，太平洋岛国有越来越多的建设计划获得中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞资金，中共提供给这些国家13亿美元的优惠贷款，成为太平洋地区仅次于澳洲的金主国。但西方国家忧心这几个小国恐怕将因积欠北京款项不堪负荷，最终受制于中共。
 <br/>
 <br/>
 澳大利亚希望这些小国能保持独立，并认为中共金援不够透明，而且附带不利条款。比如斯里兰卡就因为无力偿还债务，去年让中共以99年的租约，取得一座大型深水港的控制权。
 <br/>
 <br/>
 中共是澳洲最大的贸易伙伴，澳洲去年底指控中共干涉本国内政，使得两国关系紧张。
 <br/>
 澳洲外交部长潘恩（Marise Payne）周四将访问北京并与中共外长会晤，这是两国关系恶化后，坎培拉当局首度派高阶官员访中。
 <br/>
 <br/>
 澳洲今年已矢言研拟数项针对太平洋地区的基础设施计划，但因中共动作频频，坎培拉不得不挪用援助预算，为该地区的相关计划提供资金。
 <br/>
 <br/>
 澳洲政府已不断采取行动阻挡中共，包括因担心原承包商华为有安全上的疑虑，今年5月宣布，花费2亿澳币(约1.4亿美元）建立巴布亚新几内亚和索罗门群岛之间的海底电缆。此外，本月稍早澳洲当局也表示会帮助巴布亚新几内亚开发海军基地，封杀中共参与巴布亚新几内亚开发港口的机会。
 <br/>
 <br/>
 （记者李红报导）
</div>
