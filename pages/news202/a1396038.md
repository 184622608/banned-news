### 马来西亚前副总理被捕 控45项罪
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马来西亚.html" target="_blank">
  马来西亚
 </a>
 反腐败委员会日前逮捕了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前副总理.html" target="_blank">
  前副总理
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扎希德.html" target="_blank">
  扎希德
 </a>
 。周五，检方以涉嫌贪污和洗钱等45项罪名，对扎希德提出指控。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扎希德.html" target="_blank">
  扎希德
 </a>
 ((Ahmad Zahid Hamidi))被控8项滥用职权罪，10项背信罪和27项洗钱罪，共计涉案金额约1亿1400万令吉，约合2800万美元。
 <br/>
 <br/>
 不过，扎希德否认所有指控。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马来西亚.html" target="_blank">
  马来西亚
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前副总理.html" target="_blank">
  前副总理
 </a>
 扎希德((Ahmad Zahid Hamidi))：〝我尊重法治。〞
 <br/>
 <br/>
 扎希德现年65岁，2015年出任马来西亚副总理。 今年5月，扎希德当选〝马来民族统一机构〞((United Malays National Organisation))((也简称〝巫统〞))主席。今年大选前， 该机构主宰马来西亚政治长达60年。
 <br/>
 <br/>
 扎希德也是新总理马哈蒂尔((Mahathir Mohamad))上台后，被指控贪污的又一位高官。
 <br/>
 <br/>
 之前，马来西亚前总理纳吉布((NAJIB RAZAK))，因涉嫌〝一马发展公司〞((1MDB))数十亿资金流失案，被控洗黑钱和贪污等32项罪名。纳吉布的妻子罗斯玛((Rosmah Mansor))被控洗钱等17项罪名。但两人都不认罪，目前被取保候审。
 <br/>
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
 <br/>
