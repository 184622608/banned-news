### 没意识到民怨 法政府补救措施难挽〝黄背心〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】法国抗议调涨燃料税〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄背心运动.html" target="_blank">
  黄背心运动
 </a>
 〞愈演愈烈，这个在社群媒体发起的运动没有广泛认可的领导人，而政府承诺采取一连串补救行动，被认为来得晚又小，也让人怀疑是否可以浇熄民众怒火，一再考验上任18个月的总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 。
 <br/>
 <br/>
 以下中央社有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄背心运动.html" target="_blank">
  黄背心运动
 </a>
 的重点整理：
 <br/>
 <br/>
 <h4>
  ●谁在抗议？
 </h4>
 <br/>
 法国10月底爆发黄背心运动（Mouvement des gilets jaunes，Yellow vest movement），抗议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 （Emmanuel Macron）政府计划调高燃料税，作为支持洁净能源倡议的一环。
 <br/>
 <br/>
 然而，抗议调涨燃料税的黄背心运动如雪球般愈滚愈大，逐渐演变成更广泛的反政府运动，而示威人士大多是生活在小镇和乡村地区的人民。
 <br/>
 <br/>
 许多人批评前投资银行家马克龙是傲慢的〝富人总统〞，忽视全国各地为生活挣扎的普罗大众。
 <br/>
 <br/>
 穿上法国汽车驾驶必备的萤光黄背心，抗议人士11月17日首度在全国各地封堵道路和加油站抗议。
 <br/>
 <br/>
 从那时起，许多人持续发起示威、瘫痪交通，还导致在繁忙的岁末假期来临前，出现燃料短缺的情况。
 <br/>
 <br/>
 主办单位号召群众在每周六发起示威，首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎.html" target="_blank">
  巴黎
 </a>
 的抗议现场上周末沦为警民冲突战场，导致200多辆汽车被焚毁，412人被捕。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/12-8/p9162712a40399443.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/12-8/p9162712a40399443-ss.jpg"/>
   <br/>
   <font size="-1">
    2018年12月8日，法国防暴警察施放催泪瓦斯，与抗议者在巴黎香榭丽舍大道上发生冲突。（ERIC FEFERBERG/AFP/Getty Images）
   </font>
  </a>
 </center>
 <br/>
 <h4>
  ●示威群众要什么？
 </h4>
 <br/>
 不同示威人士有不同目标。这个在社群媒体扎根的草根运动没有广泛认可的领导人。
 <br/>
 <br/>
 有些抗议人士聚焦降低燃料税和其他经济负担，表示许多家庭正为马克龙推动改革和振兴法国经济付出代价，尤其是低收入家庭。
 <br/>
 <br/>
 其他人则针对马克龙，说他必须下台，对马克龙自去年上任后不久便为富人减税的决定怒不可遏。立即增加最低工资和养老金福利也是集会口号。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/12-8/p9162713a887405604.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/12-8/p9162713a887405604-ss.jpg"/>
   <br/>
   <font size="-1">
    2018年12月8日，法国防暴警察施放催泪瓦斯，与抗议者在巴黎香榭丽舍大道上发生冲突。（ERIC FEFERBERG/AFP/Getty Images）
   </font>
  </a>
 </center>
 <br/>
 <h4>
  ●他们为何构成挑战？
 </h4>
 <br/>
 法国当局坦承没有意识到民怨有多深，也宣布暂停原订在明年1月1日的上调燃料税计画，电力和天然气价格冻涨，严格的车辆排放管制也延后上路。
 <br/>
 <br/>
 不过，示威群众和马克龙的政敌认为，这些举措来得太晚，规模也太小。
 <br/>
 <br/>
 法国总理菲力普（Edouard Philippe）提出发放额外津贴给低薪劳工的想法。
 <br/>
 <br/>
 但谈判并未取得任何成果，尤其是声称领导这项运动的示威人士在很大程度上拒绝受邀谈判，还有些抗议人士则是受到其他〝黄背心〞威胁。
 <br/>
 <br/>
 令当局更加头痛的是，尽管巴黎上周陷入一片混乱，黄背心运动仍广受民众支持。
 <br/>
 <br/>
 民调显示，70%至80%的受访者支持这些坚守立场的示威群众，而马克龙的支持率也掉到了23%的新低纪录。
 <br/>
 <br/>
 黄背心运动先驱穆罗（Jacline Mouraud）说：〝这项运动揭开法国数以百万计民众的生活是什么模样。〞
 <br/>
 <br/>
 穆罗今天接受法国第一大报〝费加洛报〞（Le Figaro）访问时说：〝承认每个月底捉襟见肘，已不再是丢脸的事了。〞
 <br/>
 <br/>
