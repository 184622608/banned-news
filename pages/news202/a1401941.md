### 英议会开始辩论 欧盟律师称可放弃脱欧
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】周二，英国下议院就首相特雷莎・梅和欧盟达成的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧协议.html" target="_blank">
  脱欧协议
 </a>
 〞，展开五天讨论，并于下周二进行历史性投票。欧盟法院的顾问则表示，英国可以自行撤回
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧.html" target="_blank">
  脱欧
 </a>
 决定。下周二英国议会将投票决定，是否通过〝脱欧协议〞。
 <br/>
 <br/>
 如果投票通过，英国将于3月29日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱离欧盟.html" target="_blank">
  脱离欧盟
 </a>
 。如果投票失败，可能导致两种结果：一是特雷莎・梅要求二次投票，但会增加〝无协议〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧.html" target="_blank">
  脱欧
 </a>
 的可能性，梅首相也可能因此下台。二是英国可能进行脱欧二次公投。
 <br/>
 <br/>
 周一，英国六个反对党的议员提出联合申请，表示如果政府不告知关于〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧协议.html" target="_blank">
  脱欧协议
 </a>
 〞的全部法律建议，就要求启动程序，指控政府藐视议会。
 <br/>
 <br/>
 首相所在保守党的政治盟友、北爱尔兰民主统一党（DUP）表示，将投票反对〝脱欧协议〞，反对派工党要求首相下台。
 <br/>
 <br/>
 周二，英格兰银行（BOE）行长马克・卡尼告诉英国议会的财政委员会，如果脱欧，最糟糕的情况下，经济遭受的打击比全球经济危机时更严重。
 <br/>
 <br/>
 英格兰银行（BOE）行长马克・卡尼：〝这些是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 背景下的小概率事件。〞
 <br/>
 <br/>
 周二，欧洲法院（ EUROPEAN COURT OF JUSTICE）的高级法律顾问表示，英国有权单方面撤回脱欧通知。
 <br/>
 <br/>
 欧洲法院高级法律顾问 MANUEL CAMPOS SANCHEZ-BORDONA：〝欧盟第50条约规定，在脱欧协议正式结束前，允许单方面撤回脱欧决定。〞
 <br/>
 <br/>
 虽然法律顾问的意见不具有法律约束力，但法院倾向于在最后裁决时，遵循其意见。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
