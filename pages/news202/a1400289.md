### 【禁闻】《华邮》曝光中共〝发脾气外交〞细节
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】刚刚结束的亚太经合组织（APEC）峰会没能发表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合宣言.html" target="_blank">
  联合宣言
 </a>
 ，这是20年来的首次。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共外交.html" target="_blank">
  中共外交
 </a>
 部将此事归咎于美国。但美国《华盛顿邮报》11月20号发表文章，曝光了中共在APEC的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发脾气外交.html" target="_blank">
  发脾气外交
 </a>
 〞丑态。
 <br/>
 <br/>
 《华盛顿邮报》11月20号发表专栏作家Josh Rogi的文章〝中国在APEC的‘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发脾气外交.html" target="_blank">
  发脾气外交
 </a>
 ’〞，曝光了本次APEC会议期间，〝中共官方代表团做出一系列咄咄逼人、欺凌、偏执和怪异的花招，企图主导会议，逼迫东道国和其他人屈服于它〞。
 <br/>
 <br/>
 文章说，中共最先对外进行的恐吓行为，是禁止国际媒体和当地媒体采访习近平与太平洋岛国领导人的会谈。虽然在会谈前，记者们已经获得了巴布亚新几内亚政府的采访许可证。但中共官员禁止他们进入大楼，只允许中共官媒报导。引发媒体众怒。
 <br/>
 <br/>
 随后，也就是上周六，美国副总统彭斯在游轮上发表讲话，刚开始五分钟，媒体中心的互联网突然出现问题，大多数记者无法及时听到、报导彭斯讲话。但彭斯演讲完后，媒体中心的互联网神秘的又恢复正常。美国官员表示，虽然他们无法确定中共是否应对此负责，但他们正在调查发生的事情。
 <br/>
 <br/>
 文章说，接下来，事情变得更加疯狂。当APEC成员国就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合宣言.html" target="_blank">
  联合宣言
 </a>
 进行闭门谈判时，中共官员对他们表示不满，要求与巴布亚新几内亚外交部长举行会谈。被巴国外长拒绝后，中共代表团一度强行闯入外长的办公室。这件事震惊了其他国家的外交官员。
 <br/>
 <br/>
 谈判持续到上周日，美国官员透露，除中共以外，20个国家所有代表最终同意联合宣言的内容。而中共代表团主要反对一句话：〝我们同意打击包括所有不公平贸易行为在内的保护主义。〞他们认为，这句话是针对中共，因此在正式会议中大吼大叫，又在有限的会议期间发表长篇独白。
 <br/>
 <br/>
 文章说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 20年来首次陷入这样的混乱，当时间耗尽，峰会正式宣言难产时，让他国外交官反感的是，中共代表团爆发出热烈的掌声。
 <br/>
 <br/>
 旅美时事评论员夏小强：〝中共把这种蛮横傲慢的外交方式正在向全世界推广，其实让世界都感到非常的可笑。那么中共这样违反正常的外交利益和规则，不仅仅无法使问题得到解决，其实也会失去和任何一个国家正常交往的平台。〞
 <br/>
 <br/>
 旅美时事评论员夏小强表示，从APEC峰会期间中共代表团的举止可以知道，中共向来不尊重、不遵守国际政治和贸易的规则。
 <br/>
 <br/>
 不过，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共外交.html" target="_blank">
  中共外交
 </a>
 部11月19号不点名的批评华盛顿政府，将APEC峰会没能按惯例发表宣言的结果，归咎于美国。
 <br/>
 <br/>
 夏小强：〝这是中共的一贯的作风，就是它往往是倒打一耙，贼喊捉贼。这也是中美贸易战的一个缩影。因为中共多年来不遵守世界贸易规则、盗窃美国的知识产权，那么当美国要求中共改变的时候，中共连最基本的共识都不承认。〞
 <br/>
 <br/>
 Rogi的文章说，从中共政府错误的〝悲喜剧〞中可得出三个结论。首先，中共官员的行为越来越厚颜无耻。其次，中共大部分的行为偏执、过度敏感，这清楚地表明中共政府感受到美国和盟国的压力。第三，中共的行为导致其他国家和它渐行渐远。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝中国传统文化是讲礼的，但是中共夺权之后，却是讲斗。经历了几十年的反美宣传之后，那些官员早已把‘与所谓的境外反华势力作斗争’看作是当然的正义举动。其实这样搞，强调外交礼节的西方国家怎么受得了啊，这实际是给G20的美中领导人会面硬生生的埋了个地雷。〞
 <br/>
 <br/>
 白宫官员表示，〝白宫将东盟和APEC看作是为G20做铺垫〞，这种局面对即将到来的川习会并不是一个好兆头。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/陈建铭
