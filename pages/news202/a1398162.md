### 第二轮对伊朗制裁生效 中日等临时豁免
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】11月5日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 对伊朗的第二轮制裁正式生效，对伊朗的核心收入来源实施禁令，但有8个国家得到临时豁免。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 宣布对伊朗的新一轮制裁，这些制裁直接针对伊朗的原油、银行和航运部门，这些部门与伊朗支持恐怖主义、大规模杀伤性武器扩散，及侵犯人权行为等有关。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝我们的目标是遏止伊朗政权的收入，这些收入被他们用于资助整个中东乃至全世界的暴力和破坏活动。〞
 <br/>
 <br/>
 蓬佩奥说，超过20个国家已将进口伊朗的原油归零，自5月来，伊朗已损失超过25亿美元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_石油.html" target="_blank">
  石油
 </a>
 收入。不过美国暂时豁免了八个对伊朗原油依赖比较大的国家，给他们更多时间实现执行制裁。而其他国家若不执行，将被美国惩罚。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝美方将给予这些国家豁免，中国、印度、意大利、希腊、日本、韩国，台湾和土耳其。〞
 <br/>
 <br/>
 伊朗总统鲁哈尼回应说，将继续贩卖石油，称美国的制裁违反法律与联合国的决议。
 <br/>
 <br/>
 本次美国恢复制裁的力度更大，制裁名单上增加了超过700多个伊朗实体及个人。
 <br/>
 <br/>
 以色列首相 内塔尼亚胡：〝增加的制裁很大束缚了伊朗恐怖政权。〞
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
