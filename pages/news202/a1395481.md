### 韩朝推半岛局势缓解 法国愿出力促弃核
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】10月15日，正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 进行国事访问的韩国总统文在寅与法国总统马克龙举行首脑会谈，双方重点讨论了朝鲜半岛无核化及构建和平机制等事宜。
 <br/>
 <br/>
 周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 总统马克龙在巴黎爱丽舍宫接见到访的韩国总统文在寅。双方随后举行新闻发布会。
 <br/>
 <br/>
 马克龙表示，法国作为一个核大国，虽然没有直接参与美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩朝.html" target="_blank">
  韩朝
 </a>
 三方的无核化谈判，但已准备好为朝鲜无核化提供帮助。
 <br/>
 <br/>
 法国总统：〝作为一个拥有核武的国家，法国愿意在技术层面上为这一进程作出贡献。我们在核领域拥有切实的技术和科学专业知识，对无核化肯定是有帮助的。〞
 <br/>
 <br/>
 文在寅本人相信，朝鲜当局有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弃核.html" target="_blank">
  弃核
 </a>
 的真实意愿。
 <br/>
 <br/>
 韩国总统文在寅：〝朝鲜方面同意销毁核设施，他们应该相信，他们做出了正确的选择。〞
 <br/>
 <br/>
 同一天，韩朝在板门店非军事区韩方一侧再次举行高层会谈，推动执行《平壤宣言》。双方同意，可能在今年11月底或12月初，共同出席一项韩朝铁路的破土动工仪式。他们还同意，可能在10月底讨论共同申办2032年奥运会的计划。
 <br/>
 <br/>
 新唐人记者蔡美音综合报导
</div>
