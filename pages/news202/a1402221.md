### 欧盟对科技巨头加强立法 阻遏不公平竞争
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟议会.html" target="_blank">
  欧盟议会
 </a>
 委员会投票通过，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技巨头.html" target="_blank">
  科技巨头
 </a>
 采取更强硬立场，打击不公平商业行为。周四法国财长表示，若欧盟不能达成一致，法国将从明年征收
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_数字税.html" target="_blank">
  数字税
 </a>
 。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟议会.html" target="_blank">
  欧盟议会
 </a>
 一个委员会投票通过，加强立法迫使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技巨头.html" target="_blank">
  科技巨头
 </a>
 在子公司设立〝信息隔离墙〞，使用数据前获得商家许可，各国当局将被赋予更多权利，追究违规者。
 <br/>
 <br/>
 今年四月，欧盟委员会起草了规则提案，防止应用程序商店、搜索引擎、电子商务网站和酒店预定网站的不公平行为，确保科技公司和传统企业公平竞争。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不公平交易.html" target="_blank">
  不公平交易
 </a>
 行为包括：对公司不利的可追溯合同，使公司难以和在线平台结束协议的条款。
 <br/>
 <br/>
 丹麦左派议员克里斯托尔・沙尔德莫斯是此次立法的主要推动者。
 <br/>
 <br/>
 欧盟竞争委员会专员玛格丽特・维斯塔格正在调查，亚马逊是否使用商家数据制造假货。
 <br/>
 <br/>
 这项名为〝平台对企业〞（P2B）的提案需获得欧盟议会和成员国的同意，才能成为法律。
 <br/>
 <br/>
 周四，法国财政部长布鲁诺・勒梅尔表示，如果欧盟不能就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_数字税.html" target="_blank">
  数字税
 </a>
 达成协议，法国将从明年起对科技巨头进行征税。
 <br/>
 <br/>
 法国财政部长布鲁诺・勒梅尔：〝我们会从2019年向科技巨头征税，在全法征收数字税，如果届时欧洲国家没能提出税收方案的话。〞
 <br/>
 <br/>
 布鲁诺・勒梅尔表示，科技巨头在法国赚取大量利益，却比中小企业少交14%的税收。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
