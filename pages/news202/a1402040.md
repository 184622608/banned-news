### 中朝贸易走私 丹东随处可见朝鲜螃蟹
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 截至去年共进行了6次核试验，遭联合国安理会进行多次制裁。这些制裁限制朝鲜对外出口，也限制一些商品进口。但近日外媒记者亲自到中国边境城市丹东探访，发现有大量从朝鲜流入的螃蟹和海鲜。
 <br/>
 <br/>
 自2006年开始到去年9月，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 共进行了6次核武试验，引发国际社会强烈批评，也因此受到了联合国安理会的多次制裁。从2006年的1718号制裁决议，到2017年的2375号决议，这些制裁不但限制朝鲜的煤炭、铁矿产和海鲜等出口，非民生用品也不可以运进朝鲜。
 <br/>
 <br/>
 外界认为，朝鲜能在安理会制裁的12年期间仍能生存、持续发展核武器和导弹，还能推动经济发展，应该与中朝贸易走私有直接的关系。
 <br/>
 <br/>
 为了解中朝贸易走私情况有多严重，自由亚洲电台记者上周前往中国丹东进行实地采访。
 <br/>
 <br/>
 丹东与朝鲜的新义州隔岸相望，中朝贸易大约有7成都经过这两个城市交易。据称，除了不受制裁的民生用品和其他贸易之外，这里同时也是贸易走私的主要渠道。甚至还有消息称，朝鲜的稀土就是通过丹东运往中国。不过，自由亚洲称，记者没有采访到相关的走私内容。
 <br/>
 <br/>
 但自由亚洲记者在丹东市场看到了很多螃蟹，而商家没有任何掩饰的称，这些螃蟹都是从朝鲜进来的。
 <br/>
 <br/>
 报导称，根据安理会2017年的2371号决议，禁止联合国成员国进口朝鲜海鲜。韩国国家安保战略研究院的林秀浩对此表示，这些海鲜应该是走私的水货。据韩方了解，中国和朝鲜船隻在公海接触，之后双方换船，将海鲜运到中国。
 <br/>
 <br/>
 据介绍，换船原本是指双方渔船在公海上互相交换商品，但之后在丹东发展出另一模式。中方会将多艘空船交给朝鲜，之后约好时间在公海上接过装满海鲜的船隻。过程中，中方会支付人民币和大米等粮食，换取满船的海鲜。
 <br/>
 <br/>
 自由亚洲电台记者在丹东还看到的另一现象，返回朝鲜的货车全部都会将油箱装满，再通过友谊桥返回。当然，这并不违反安理会决议。
 <br/>
 <br/>
 记者还注意到丹东的工厂将布料运往朝鲜，进行成衣加工，以及在丹东工作的朝鲜工人。这明显的违反了2375号决议。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
