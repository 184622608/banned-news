### 震惊日本社会 宫崎县宁静小镇凶杀案酿7死
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 九州宫崎县的一处民宅，26日发现6具遗体，其中包括一名女童。警方傍晚又在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_命案.html" target="_blank">
  命案
 </a>
 现场外的一座桥下再发现第7具遗体。由于尸体有外伤，警方已按凶杀案调查。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 放送协会（NHK）报导，宫崎县警方26日上午接获民众报案，称无法连系上在高千穗町押方的亲戚饭干保生一家人。警方到场后发现在屋外有一名女性死者，屋内则有3名男性死者、1名成年女性死者和1名女童尸体。
 <br/>
 <br/>
 警方说，案发后由于找不到屋主的次子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_饭干昌大.html" target="_blank">
  饭干昌大
 </a>
 ，认为他可能会知道内情，所以持续搜寻其下落，结果当天傍晚在离案发现场2公里外的一处停车场，发现饭干昌大的车子，并在停车场附近桥下约100米处发现疑似饭干昌大的遗体。
 <br/>
 <br/>
 警方称，遗体外伤像是刀刃所留下，但目前并未在案发现场附近找到类似凶器。
 <br/>
 <br/>
 目前，警方已查明7具遗体身分，包括屋主72岁的饭干保生、66岁妻子饭干实穗子、饭干夫妇次子饭干昌大、41岁妻子饭干美纪子、次子夫妇21岁长子饭干拓海、7岁长女饭干唯，及饭干昌大44岁友人松冈史晃。
 <br/>
 <br/>
 相关人士称，饭干昌大跟妻女3人参加了公司25日结束的2天1夜的员工旅游，过程中3人购买纪念品等，看起来感情非常融洽。
 <br/>
 <br/>
 警方认为，3人结束员工旅游返家后的25日到26日上午间，应该就是6人遇害的期间，正在进一步调查详细案情。
 <br/>
 <br/>
 饭干昌大友人松冈史晃也不幸遇害，他的妻子告诉警方，家属突然听到松冈史晃遭遇不幸都感到极度悲伤与震惊。
 <br/>
 <br/>
 日本产经新闻报导，这起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_命案.html" target="_blank">
  命案
 </a>
 让一向平和宁静的小镇，弥漫着相当森严的气氛，也让住在命案现场附近的民众难掩不安情绪。
 <br/>
 <br/>
 一名62岁男性回想，25日时还跟过去一样生活的邻居，没想到竟然会变成一起6死命案，〝真的很震惊〞。
 <br/>
 <br/>
 饭干唯就读于高千穗町立押方小学2年级，副校长谷正胜称，她是一位开朗活泼的小女孩，〝从没忘记写回家作业，是非常认真的学生〞。
 <br/>
