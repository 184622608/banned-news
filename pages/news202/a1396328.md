### 土耳其：周二公布沙特记者遇害调查
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 总统埃尔多安周日表示，他将在周二（10月23日）公布土耳其方面针对沙特记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 遇害案的调查结果。同一天，沙特外长阿德尔・朱拜尔接受美国福克斯新闻采访时説，卡舒吉的被谋杀是一个〝巨大的错误〞。
 <br/>
 <br/>
 周日（10月21日），埃尔多安在伊斯坦布尔的一个集会上发表演说时表示，他会在周二(10月23日)宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 方面调查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 遇害案的详情，伸张正义。
 <br/>
 <br/>
 土耳其总统埃尔多安：〝为了伸张正义，整个事件将被彻底的揭开。是彻底完全的，而不是分步骤的。为什么那15个（沙特）人要到这里来，为什么有18个（沙特）人被捕。我将会详细的解释这些事情。〞
 <br/>
 <br/>
 同一天，沙特外长阿德尔・朱拜尔（Adel al_Jubeir）在接受美国福克斯新闻（Fox News）采访时承认，卡舒吉是被谋杀的，而且这是一个〝巨大的错误〞。
 <br/>
 <br/>
 沙特外长阿德尔・朱拜尔：〝有人在他们的职权范围之外干了这件事。这显然是个巨大的错误。错上加错的是，他们还试图对此事加以掩盖。这对任何政府都是无法接受的。不幸的是，这些事情都发生了。〞
 <br/>
 <br/>
 不过沙特外长朱拜尔强调，沙特王储萨勒曼王子对此案并不知情，整个事件是一次〝流氓行动〞。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
