### 〝黄背心〞抗议后 巴黎清理市容 马克龙谴责暴力
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国巴黎.html" target="_blank">
  法国巴黎
 </a>
 的清洁人员在地标
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凯旋门前.html" target="_blank">
  凯旋门前
 </a>
 ，清理前一天〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄背心.html" target="_blank">
  黄背心
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力抗议.html" target="_blank">
  暴力抗议
 </a>
 留下的垃圾。法国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 针对这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力事件.html" target="_blank">
  暴力事件
 </a>
 ，发表推文严辞谴责。
 <br/>
 <br/>
 清洁人员周日动用重型机器清除抗议后留下的满地垃圾。
 <br/>
 <br/>
 游客索菲：〝你总是看到有人摧毁一切，这埋没了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄背心.html" target="_blank">
  黄背心
 </a>
 想传达的信息，很遗憾。看到香榭丽舍大街变成这样，我感到非常难过。〞
 <br/>
 <br/>
 周六，数千人聚集在巴黎抗议燃油税上涨、表达对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 总统的愤怒。警方指出，由于有极端人士介入，让抗议转趋激进。示威者向警察投掷石块；还在巴黎市中心的几条街道中间烧毁大型胶合板和其它材料，放出大量烟雾。警察通过发射催泪瓦斯和水炮作出回应。
 <br/>
 <br/>
 在一篇措辞强硬的推文中，马克龙谴责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力事件.html" target="_blank">
  暴力事件
 </a>
 。
 <br/>
 <br/>
 马克龙推文：“感谢我们执法部门的勇气和专业；那些袭击他们的人可耻。那些暴力侵害其他公民和记者的人可耻。那些试图恐吓当选官员的人可耻。共和国不容这种暴力存在。”
 <br/>
 <br/>
 抗议活动中至少有20人受伤，其中一人伤势严重。数十人被捕。
 <br/>
 <br/>
 新唐人记者严枫、田瑞综合报导
