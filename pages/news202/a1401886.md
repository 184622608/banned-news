### 全世界最好用护照 阿联拔头筹 中国列第58
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】全世界最好用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_护照.html" target="_blank">
  护照
 </a>
 在那一国？据加拿大国际金融顾问公司阿尔顿资本（Arton Capital）编制的〝护照指数排名〞，答案是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿拉伯联合大公国.html" target="_blank">
  阿拉伯联合大公国
 </a>
 (，目前不必事先申请签证就能前往167个国家。中国大陆名列第58、中华民国为第28。
 <br/>
 <br/>
 <b>
  <a href="https://www.passportindex.org/byRank.php">
   加拿大国际金融顾问公司阿尔顿资本
  </a>
 </b>
 编制的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_护照.html" target="_blank">
  护照
 </a>
 指数排名〞（Passport Index）对象为联合国193个会员国加上非会员国的6个国家与地区。
 <br/>
 <br/>
 〝每日邮报〞报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿拉伯联合大公国.html" target="_blank">
  阿拉伯联合大公国
 </a>
 去年甚至连前10名都没有挤进，今年拔得头筹。杜拜领导人穆罕穆德亲王（Sheikh Mohammed bin Rashid Al Maktoum）在推特发文祝贺表示：〝恭喜阿拉伯联合大公国及人民，也感谢外交部及国际合作团队。〞
 <br/>
 <br/>
 阿拉伯联合大公国（简称阿联‎），是由阿布达比、夏尔迦、杜拜、阿吉曼、富吉拉、欧姆古温、哈伊马角七个酋长国组成的联邦制君主国。
 <br/>
 <br/>
 德国和新加坡并列第2名，不必事先申请签证就能前往166个国家；美国、丹麦、瑞典、芬兰、卢森堡、法国、意大利、西班牙、挪威、荷兰、韩国护照屈居第3，免（落地）签国家有165个。
 <br/>
 <br/>
 另外，英国、比利时、奥地利、日本、希腊、葡萄牙、瑞士、爱尔兰、加拿大等第4，免（落地）签国家164个。
 <br/>
 <br/>
 据统计，香港第13、中国大陆第58；中华民国护照免（落地）签国家有131个，名列第28，但据中华民国外交部统计，自帛琉12月1日起给予台湾入境免签证待遇后，已经有170个国家及地区给予中华民国护照免签证或落地签证待遇。
 <br/>
 <br/>
 另外，伦敦顾问公司Henley &amp; Partners日前也根据各国护照免签国家数量，对200个国家与地区的护照好用度进行排名，日本位居全球之冠，台湾排在第26名。
 <br/>
 <br/>
