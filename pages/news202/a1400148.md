 周一，芝加哥发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 。凶嫌与一家医院紧急病房的女医生发生争执后，枪杀了她，随后又枪杀了一名药剂师和一个警察，凶嫌也当场毙命，但还不清楚是自杀还是被警察击毙。遇害女医生是一名38岁的基督徒；遇害警察28岁，是三个孩子的父亲。
 <br/>
 <br/>
 周一，危地马拉富埃戈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火山.html" target="_blank">
  火山
 </a>
 爆发，周围约4千人撤离，超过2千人被安置到避难所。这座火山喷出热灰、熔岩和天然气流，它是中美洲最活跃的火山之一，高3,763米，乃今年第三次喷发。
 <br/>
 <br/>
 周二，澳大利亚墨尔本警方逮捕了三名青年男子，他们涉嫌策划在墨尔本发动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恐怖袭击.html" target="_blank">
  恐怖袭击
 </a>
 ，并为此寻找半自动枪械。澳洲警方及安全和反恐机构共同参与了这次行动。而在不到两周前，墨尔本刚发生过一起〝独狼式〞恐袭。
 <br/>
 <br/>
 经历6个多月旅程，美国宇航局InSight
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太空船.html" target="_blank">
  太空船
 </a>
 即将于11月26日结束3亿英里旅程，在火星表面着陆。届时NASA将进行现场直播。今年5月，InSight从加州范登堡（Vandenberg）空军基地发射，这是2012年以来，NASA太空船首次尝试火星着陆，也将首次尝试探测火星地下。
