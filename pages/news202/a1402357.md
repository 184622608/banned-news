### CNN 办公楼传炸弹警报 虚惊一场
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】周四半夜，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国有线电视新闻网.html" target="_blank">
  美国有线电视新闻网
 </a>
 CNN在纽约的办公室接到炸弹威胁，所有工作人员被警方疏散一个多小时。警方搜索后未找到任何爆炸物，证实是虚惊一场。
 <br/>
 <br/>
 周四晚上10点35分，纽约警方接到报警，声称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_时代华纳中心大楼.html" target="_blank">
  时代华纳中心大楼
 </a>
 ，CNN纽约办公室藏有五枚炸弹。凌晨12点之前，警方疏散了大楼内的人员，逐层搜查。
 <br/>
 <br/>
 警方还封锁了曼哈顿中城西58街，第8大道和第9大道之间的路段，禁止车辆和行人通过。
 <br/>
 <br/>
 CNN报道称，警方没有发现任何可疑物品。
 <br/>
 <br/>
 警方疏散人员时，CNN主播唐・莱蒙的直播节目被迫中断，临时播出提前录制的其他节目。CNN之后将电视画面切换到大楼外，唐・莱蒙继续播报。
 <br/>
 <br/>
 莱蒙在推特说：〝直播时被疏散。炸弹威胁。改播预录节目。纽约警察局正在调查。敬请关注。〞
 <br/>
 <br/>
 今年十月纽约的系列邮包炸弹事件，同一栋大楼收到寄给前中央情报局长约翰・布伦南的包裹，发现含有爆炸装置。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
