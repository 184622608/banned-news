### 保护资讯安全 美欧日拟建〝数据贸易圈〞
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】日本经济新闻报导，日本希望与美国、欧盟制定跨境数据流通规则，保护个人和企业在人工智能等领域的资讯安全。
 <br/>
 <br/>
 日本的目标是，禁止数据流向不能充分保护个人资讯、不能保障网路安全的国家，数据跨境转移必须取得本人同意，还要提高透明度。
 <br/>
 <br/>
 美日欧三方贸易代表将对此进行磋商，在明年6月的G20峰会之前公布协议。
 <br/>
 <br/>
 西方社会担心，中共通过百度、阿里巴巴、腾讯的三大企业获取大量个资，若采用品质和数量俱佳的大数据，人工智能的竞争力将呈现飞跃性的提升。而且中共已开始向东南亚、中东和非洲推销其模式，〝中国模式的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_数据贸易圈.html" target="_blank">
  数据贸易圈
 </a>
 〞可能会扩大。
 <br/>
 <br/>
 目前美日欧采用在一定程度上匿名化的数据，因此在人工智能的竞争力上有可能落后。如果建立透明度高、兼顾人权的〝贸易圈〞，就能在区域内共享大量数据，增强竞争力。
 <br/>
 <br/>
 这项协议将由日本的个人讯息保护委员会、美国联邦贸易委员会（FTC）和欧盟委员会司法总署确认具体内容，由各国推进立法。
 <br/>
 <br/>
 新唐人电视综合报导
</div>
