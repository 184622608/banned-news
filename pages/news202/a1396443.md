### 剑指一带一路？川普将公布非洲新战略
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】美国川普（特朗普）政府即将公布最新的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲.html" target="_blank">
  非洲
 </a>
 战略，侧重在非洲大陆的投资和创造就业机会。外界解读，美国的这一动作是抗衡中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞的策略。
 <br/>
 <br/>
 据《美国之音》报导，美国国务院负责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲.html" target="_blank">
  非洲
 </a>
 事务的助理国务卿纳吉（Tibor Nagy）日前说，美国历届政府的非洲战略优先考虑结束当地的战争、停止人道灾难、以及稳定非洲局势。如果还有余力，才会试图推动经济发展。
 <br/>
 <br/>
 纳吉透露，川普政府的新非洲战略，将侧重在非洲大陆创造良好的投资环境和创造就业机会，吸引外资进入非洲，并强调美国将比中共更能为非洲带来繁荣和工作机会。
 <br/>
 <br/>
 此前，中共曾宣布向非洲提供600亿美元的援助和投资计划，并在今年9月的中非论坛上再承诺追加600亿美元。
 <br/>
 <br/>
 纳吉指出，中共的大型投资项目通常不会为非洲创造大量工作机会。而美国的投资往往会给当地创造很多工作机会，因此非洲人民很希望美国企业进入非洲。
 <br/>
 <br/>
 近期，美国国会和行政当局联手制定了《国际发展投资法案》，初期也刚好拨出600亿美元，帮助非洲和全球其他贫困地区兴建基础设施和发展经济。
 <br/>
 <br/>
 上周五，美国国务卿蓬佩奥呼吁拉美国家应警惕中共的〝掠夺性行为〞，〝中国（中共）提供大笔资金，但是其附加条件让那些国家的人民在两年、五年或是十年之后陷入很可怕的境地。〞
 <br/>
 <br/>
 欧亚国家也在同一天作出表态。 欧盟28个成员国、挪威、瑞士和21个亚洲国家，承诺遵守基础设施项目的市场原则和国际标准，希望跟华盛顿一道遏制中共影响力。
 <br/>
 <br/>
 美国副总统彭斯则在10月初表示，中共向贫困国家提供的巨额基础设施贷款不透明，带来的利益压倒性地流向北京。
 <br/>
 <br/>
 7月底，美国还发布了〝印太经济愿景〞，祭出1.13亿美元扩大投资亚太地区，主要聚焦新科技、能源、基础建设三大领域。蓬佩奥表示，1.13亿美元只是头期款，印太经济愿景是川普印太策略的主轴，美国、日本、澳洲的三边投资协定可望强化此一提案。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：范铭）
