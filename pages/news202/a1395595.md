### 微软公司联合创始人保罗・艾伦病逝
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】http://www.vulcan.com/
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微软公司.html" target="_blank">
  微软公司
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合创始人.html" target="_blank">
  联合创始人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保罗・艾伦.html" target="_blank">
  保罗・艾伦
 </a>
 ，因患非霍奇金氏淋巴瘤并发症，周一在西雅图
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_病逝.html" target="_blank">
  病逝
 </a>
 ，终年65岁。
 <br/>
 <br/>
 本月初，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保罗・艾伦.html" target="_blank">
  保罗・艾伦
 </a>
 在推特表示，再次接受非霍奇金氏淋巴瘤治疗。他九年前曾治愈，没想到旧病复发，15号因并发症不幸
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_病逝.html" target="_blank">
  病逝
 </a>
 。
 <br/>
 <br/>
 保罗・艾伦的推特公布了他的遗照和遗言。
 <br/>
 <br/>
 他说:〝只要我们一起努力，拥有迫切感和决心，就没有我们成就不了的事。〞
 <br/>
 <br/>
 40多年前，保罗・艾伦预见到电脑将成为人类生活中不可或缺的一部分。1975年，他说服高中好友比尔・盖茨，从哈佛大学辍学，两人联合创立了著名的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微软公司.html" target="_blank">
  微软公司
 </a>
 。
 <br/>
 <br/>
 得知艾伦去世的消息，比尔・盖茨表示：〝我最亲密的老友艾伦去世令人心碎。他应该拥有更长的生命，他对科技和慈善事业的贡献将世代流传。〞
 <br/>
 <br/>
 艾伦和盖茨1983年各奔前程，但是艾伦继续持有微软股票，成为亿万富翁，名列世界第四大富豪。
 <br/>
 <br/>
 微软执行长Satya Nadella在声明中说: 〝他做事不声不响又持之以恒，创造了神奇的产品、经验和制度，以此改变了世界。〞
 <br/>
 <br/>
 保罗‧艾伦还致力于体育和慈善事业，拥有NBA 的波特兰拓荒者队、美式足球 NFL 西雅图海鹰队、西雅图海湾者足球俱乐部。
 <br/>
 <br/>
 新唐人记者李梅、李静明综合报导
</div>
