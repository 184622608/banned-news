### 遭土耳其拘押两年 美国牧师获释
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 拘押两年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国牧师.html" target="_blank">
  美国牧师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布朗森.html" target="_blank">
  布朗森
 </a>
 ，周五被土耳其法庭宣布释放，可以离开土耳其，返回美国。不过，他仍然被判参与恐怖行动的罪名成立。这起案件曾引发美国与土耳其的外交危机。美国总统川普随后发推文宣布，布朗森牧师刚刚获释，将很快回到美国。
 <br/>
 <br/>
 美国福音派牧师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布朗森.html" target="_blank">
  布朗森
 </a>
 被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 当局指控参与恐怖行动和政变，被拘押两年，7月底改为居家软禁，禁止出国。
 <br/>
 <br/>
 周五，土耳其亚利加市法院判决布朗森〝参与恐怖行动〞罪名成立，入狱3年零一个半月。
 <br/>
 <br/>
 但因为他已经被拘禁两年，并且表现良好，所以法庭决定立即释放。这意味着他可以离开土耳其，返回美国。
 <br/>
 <br/>
 消息传出，美国总统川普第一时间连发推文说：〝对布朗森牧师（的事）做了非常艰苦的工作！〞
 <br/>
 <br/>
 〝我为布朗森牧师祈祷，我们希望他能尽快安全回家！〞
 <br/>
 <br/>
 此前，美国多次要求释放布朗森，土耳其都表示拒绝。两国外交关系一度紧张。8月，美国以经济制裁向土耳其施压,，其货币里拉崩盘。
 <br/>
 <br/>
 判决前一天，由于预计布朗森将获释，土耳其里拉上涨3％。判决后，汇率稳定。
 <br/>
 <br/>
 新唐人记者王羽音李静明综合报导
