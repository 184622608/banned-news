### 美技术出口管制将更严 影响中国科技企业
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国商务部.html" target="_blank">
  美国商务部
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工业安全局.html" target="_blank">
  工业安全局
 </a>
 周一出台了一份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_技术出口管制.html" target="_blank">
  技术出口管制
 </a>
 先期通知，并向公众征询意见。外界认为，中国将是受到这项条例影响最大的国家。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国商务部.html" target="_blank">
  美国商务部
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工业安全局.html" target="_blank">
  工业安全局
 </a>
 在这份先期通知中，列出了14个政府考虑进行管制的类别，包括人工智能、芯片、量子计算、机器人、面印和声纹技术等。
 <br/>
 <br/>
 这些领域涉及国家安全和高技术，被认为是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前沿科技.html" target="_blank">
  前沿科技
 </a>
 的关键领域。
 <br/>
 <br/>
 商务部工业安全局从即日起向公众征询意见，截止到12月19日。
 <br/>
 <br/>
 美国商务部和其它审查机构评估意见后，将更新出口管制清单，可能成为美国历年来最严格的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_技术出口管制.html" target="_blank">
  技术出口管制
 </a>
 条例。
 <br/>
 <br/>
 这份先期通知中没有直接提到中国，但外界认为，中国将是受影响最大的国家。很多中国高科技公司赖以生存的芯片将面临新的封锁。
 <br/>
 <br/>
 其中，华为芯片技术被曝一半以上来自美国；海康威视的人工智能监控设备依靠从美国的英伟达和英特尔进口的芯片。他们的产品被认为是中共当局侵犯人权的工具。
 <br/>
 <br/>
 新唐人记者李梅、张琪综合报导
</div>
