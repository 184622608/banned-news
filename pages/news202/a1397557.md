### 大篷车移民大规模逼近 美民众带枪抵边境堵截
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】来自中美洲的大约7500名移民乘坐〝大篷车〞，正穿过墨西哥领土向美国进发。为此，美国总统已下令5200名军人前往边境执行堵截任务。与此同时，一批美国民众自发携带枪支向美墨边境集结，他们准备与军人一起阻止〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 〞进入美国。
 <br/>
 <br/>
 据法广报导，目前大约有2000美国〝志愿者〞正自发向美国南部边境集结。这些志愿者都是普通民众，他们大多自行携带枪支并穿着防弹背心赶往美墨边境，准备与边境巡逻队和国民警卫队一道严防死守，阻止正不断涌向美国边境的中美洲〝移民大篷车队〞进入美国。
 <br/>
 <br/>
 一位〝民兵〞告诉媒体，他们先期抵达边境，以便为后续即将到来的志愿者作安排。德克萨斯州民兵组织的主席麦高利（Shannon McGauley）在接受媒体采访时也表示，他已在德克萨斯州边境处的3个地点部署了民兵成员，预计未来几天陆续还有人抵达那里。
 <br/>
 <br/>
 近日，大量来自洪都拉斯等中美洲国家的移民乘坐着各式各样的〝大篷车〞进入墨西哥境内，然后络绎不绝地涌向美墨边境。
 <br/>

   </font>
  </a>
  <br/>
 </center>
 <br/>
 为了减轻边境的巨大压力，墨西哥总统涅托于10月26日向已经进入墨西哥的〝移民大篷车队〞开出了〝优厚条件〞—— 如果这些外来移民愿意留在墨西哥南部的两个州，墨西哥政府将为他们提供住所、医疗、教育和工作。
 <br/>
 <br/>
 然而，大多数移民拒绝了墨西哥政府的挽留，他们执意要前往美国。
 <br/>
 <br/>
 〝谢谢！〞一些坐着大篷车进入墨西哥的移民高声喊道，〝但是我们还是要向北走，我们唯一的目的地是美国。〞
 <br/>
 <br/>
 据墨西哥政府统计，在已大约7500名移民中，仅有大约1700名同意留在墨西哥。
 <br/>
 <br/>
 对于那些不愿留下的移民，墨西哥当局允许他们徒步穿越墨西哥的领土，但是拒绝向他们提供食物、住所或洗浴设施，只将这些服务提供给同意留在墨西哥的移民。
 <br/>

 对于这场突如其来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 浪潮的来袭，美国总统川普（特朗普）正在作紧急应对。
 <br/>
 <br/>
 川普曾在社交媒体推特上发文将此事称作〝国家紧急事件〞，他并透过推文向那些正试图非法入境美国的大篷车移民喊话，〝请你们回去，你们无法获准入境。这是对我们国家的入侵，我们的军队正等着你们。〞
 <br/>
 <br/>
 川普还签署了行政令，以便有关机构能够迅速颁布针对性的法规，以阻止这些来自中美洲国家的非法移民进入美国领土。
 <br/>
