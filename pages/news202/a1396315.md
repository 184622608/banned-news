### 加拿大卑诗省1小时3起规模6.5以上强震
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】加拿大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卑诗省.html" target="_blank">
  卑诗省
 </a>
 （British Columbia）22日1小时内连发3起规模6以上强震，其中最强为规模6.8
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国地质调查所.html" target="_blank">
  美国地质调查所
 </a>
 （USGS）表示，第一起强震发生在周一(22日)北京时间下午1时40分左右，震央位于哈迪港（Port Hardy）218公里处，规模6.6，震源深度仅11公里。
 <br/>
 <br/>
 约40分钟后，又几乎是同地点又发生规模6.8
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 ，深度为21公里。再2分钟后，又有规模6.5地震，深度10公里，3起地震均属于极浅层地震。哈迪港位于温哥华岛东北方。
 <br/>
 <br/>
 目前尚未传出任何人员伤亡和灾情，也未立即发布海啸警报。
 <br/>
 <br/>
 (责任编辑：夏明义)
