### 金正恩视察新武器测试 美国务院发话提醒
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】美国国务院周四晚间就朝鲜弃核问题发表一份声明，表示对实现美国总统川普与朝鲜首脑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 今年6月在新加坡达成的推动朝鲜弃核的协议有信心。舆论认为，这是对金正恩视察〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新战术武器.html" target="_blank">
  新战术武器
 </a>
 〞所引发的外界担忧作出的回应。
 <br/>
 <br/>
 美东时间11月15日晚间，美国国务院发布一份声明称，〝在新加坡峰会上，川普总统和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 主席针对彻底实现无核化以及为朝鲜创造光明前景等议题达成多项协议。我们正在和朝鲜讨论如何实现这些承诺。川普总统很清楚的表明，若金正恩能彻底去核化，朝鲜就会有光明未来。〞
 <br/>
 <br/>
 这份声明再次强调，美国政府有信心能实现川普与金正恩先前达成的有关朝鲜无核化的协议。
 <br/>
 <br/>
 不过，这份声明并未针对金正恩视察〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新战术武器.html" target="_blank">
  新战术武器
 </a>
 〞的消息发表评论。
 <br/>
 <br/>
 朝鲜官媒《朝中社》日前报导称，朝鲜新的战术武器已成功开发，并在国防学院测试中心完成了测试，金正恩亲临现场视察了这些新的战术武器。
 <br/>
 <br/>
 这篇报导引发外界担忧，怀疑金正恩今年6月12日与川普在新加坡达成的有关朝鲜将无核化的协议会否发生变化。
 <br/>
 <br/>
 因此，有与论认为，美国国务院这次发表声明称〝有信心实现川普与金正恩的协议〞，便是间接对外界的前述担忧作出了回应。
 <br/>
 <br/>
 （记者欣然综合报导/责任编辑：东野）
</div>
