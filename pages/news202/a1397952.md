### 美日西太平洋联合军演  巩固印太战略
------------------------

<div class="wysiwyg">
 【新唐人2018年11月03日讯】本周，美国和日本在西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太平洋.html" target="_blank">
  太平洋
 </a>
 海域举行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_利剑.html" target="_blank">
  利剑
 </a>
 〞联合军演。双方将领表示，军演意在巩固美日同盟的，印太地区战略。而加拿大，今年也首次参加了演习。
 <br/>
 <br/>
 本次〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_利剑.html" target="_blank">
  利剑
 </a>
 〞联合军演，一共动员了57,000名水手、海军陆战队员和飞机驾驶员，人数比前年增加了近25%。
 <br/>
 <br/>
 其中，日方共派出47,000人参加演习，相当于动员日本军队人数的五分之一。
 <br/>
 <br/>
 美方则派出〝尼米兹〞级航母雷纳德・里根号。在为期10天的演习，内容涵盖空中实战、两漆登陆作战以及弹道导弹拦截等项目。
 <br/>
 <br/>
 双方表示，演习将巩固美日同盟在印太地区的军事实力，展示双方维护该地区开放和自由的决心。
 <br/>
 <br/>
 <br/>
 美国海军少将 卡尔・托马斯：〝‘利剑’这样的演习正是我们需要做的，在这一海域增强（军事）实力。〞
 <br/>
 <br/>
 日本海军少将 Hiroshi Egawa：〝基于理解美日同盟，对维护整个印太地区稳定至关重要，我们希望有力地提高和美国海军共同（作战）的能力。〞
 <br/>
 <br/>
 今年，加拿大首次派出一艘补给船参与〝利剑〞演习。同时，英国、法国、澳大利亚和韩国也都派出观察员对演习进行考察。
 <br/>
 <br/>
 分析认为，由于中共在南中国海域的扩军行为，美日以外的西方各国也在寻求增加印太地区的军事能力。
 <br/>
 <br/>
 〝利剑〞演习从本周一开始，将在下周四结束。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
