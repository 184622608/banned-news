### 英议会再议脱欧 欧盟为无协议做准备
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】周一，英国首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特雷莎．梅.html" target="_blank">
  特雷莎．梅
 </a>
 敦促
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 不要让爱尔兰边境〝保障条款〞僵局打乱
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧谈判.html" target="_blank">
  脱欧谈判
 </a>
 ，她仍相信可达成协议，但欧盟部长也在为不能达成协议做准备。
 <br/>
 <br/>
 周二英国首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特雷莎．梅.html" target="_blank">
  特雷莎．梅
 </a>
 和英国内阁讨论脱欧协议，她希望尽量不使用〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保障协议.html" target="_blank">
  保障协议
 </a>
 〞。
 <br/>
 <br/>
 英国首相 特雷莎．梅：〝我一直相信，我们所做的就是努力确保《保障协议》不会到位，实际上我们说的不是2021年12月（结束），而是2021年1月1日。〞
 <br/>
 <br/>
 特雷莎．梅所属保守党内的疑欧派担心，〝保障协议〞会使英国无限期留在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 关税同盟，北爱尔兰民主统一党表示，不会接受任何将北爱尔兰和英国分开的协议。
 <br/>
 <br/>
 英国首相特雷莎．梅：〝我仍然确信，没有协议比达成坏协议好。〞
 <br/>
 <br/>
 英国国际发展部部长表示，部长们支持特雷莎．梅的脱欧协议。
 <br/>
 <br/>
 英国国际发展部部长 PENNY MORDAUNT：〝我们进入谈判最后阶段，我们都支持首相达成最好结果。〞
 <br/>
 <br/>
 出席欧盟会议的各国部长希望能达成协议。
 <br/>
 <br/>
 德国负责欧盟事务部长 罗斯：〝（特雷莎．梅应当）负起责任，并有建设性。〞
 <br/>
 <br/>
 同时，他们也在为不能达成协议做准备。
 <br/>
 <br/>
 法国欧盟事务部长 卢瓦索：〝但我们得为所有可能的情况准备，包括不能达成协议。〞
 <br/>
 <br/>
 比利时外交部长 雷恩代尔：〝我希望我们不会达不成协议，但我们不得不为此准备。〞
 <br/>
 <br/>
 周三，特雷莎．梅将向欧盟各国首脑陈述对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧谈判.html" target="_blank">
  脱欧谈判
 </a>
 的看法。
 <br/>
 <br/>
 新唐人记者毕心慈、明玉综合报导
</div>
