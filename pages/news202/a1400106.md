### 朝鲜炸毁边境哨所 美军遗体运送回国
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 在韩朝边境非军事区，炸毁10处哨所，履行韩朝上个月签署的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_《9・19军事协议》.html" target="_blank">
  《9・19军事协议》
 </a>
 中的部分内容。与此同时，韩国联合国军司令部举行韩战美军士兵遗骸运送回国仪式。
 <br/>
 <br/>
 韩国国防部提供的视频显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 周二在韩朝边境非军事区，炸毁了10处哨所。
 <br/>
 <br/>
 根据韩朝上月签署的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_《9・19军事协议》.html" target="_blank">
  《9・19军事协议》
 </a>
 ，韩朝商定年底前撤除各自的11个哨所，并在月底前撤离驻扎在那里的设备和人员。
 <br/>
 <br/>
 不同于朝鲜，韩国以挖掘机拆除，以维护自然环境以及人员安全。
 <br/>
 <br/>
 与此同时，韩国联合国军司令部举行仪式，将一名美军士兵的遗骸运送回国。
 <br/>
 <br/>
 联合国指挥官艾布拉姆斯将军表示，这名士兵在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜战争.html" target="_blank">
  朝鲜战争
 </a>
 中丧生，当时隶属第32步兵团。
 <br/>
 <br/>
 韩国国防部说，去年10月，在朝韩边境附近的阳谷县挖掘出这名士兵的遗体。
 <br/>
 <br/>
 联合国指挥官ROBERT B. ABRAMS：〝感谢你。愿神保佑这位牺牲的战士，看护他一路走好，以安慰那些等待勇士归来的人们。〞
 <br/>
 <br/>
 1950年6月25日，朝鲜共产党军队发动突袭，跨越划分南北韩的北纬38度线，朝鲜战争爆发。以美国为首的联合国部队，与中共和苏联支持的朝鲜作战，历时三年。
 <br/>
 <br/>
 联合国指挥官 ROBERT B. ABRAMS：〝让我们永远不要忘记他们。让我们永远留出时间和空间来纪念他们。让我们永远不要放松，直到他们都回家。〞
 <br/>
 <br/>
 美国军方统计，有36,500多名美军在朝鲜战争中丧生，至今仍有7,700名美国军人下落不明。美国一直努力寻找阵亡将士的遗体，将他们运送回国安葬。
 <br/>
 <br/>
 新唐人记者李梅、天羽综合报导
</div>
