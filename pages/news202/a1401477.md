### 美墨加正式签署协议 北美新贸易市场即将形成
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】周五（11月30日），美加墨三国首脑利用共赴阿根廷参加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会之机，在布宜诺斯艾利斯（Buenos Aires）正式签署了新的三边贸易协定。
 <br/>
 <br/>
 周五，借用二十国集团（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 ）峰会这一平台，美国总统川普、加拿大总统特鲁多（Justin Trudeau）和墨西哥总统涅托（Enrique Peña Nieto）在阿根廷首都会晤并正式签署了新的贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_协议.html" target="_blank">
  协议
 </a>
 。该协议取代了1994年签订的争议巨大的《北美自由贸易协定》（NAFTA），重新调整了美国与邻国间以往不公平的贸易关系，是川普政府在经济和政治上取得的双重的重大胜利。
 <br/>
 <br/>
 川普在签字仪式开始时表示：〝这是场战斗，而战斗有时会产生很棒的友谊。〞
 <br/>
 <br/>
 川普指出，新签署的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨加.html" target="_blank">
  美墨加
 </a>
 协定》（USMCA）能使三国全都受益，〝这可能是有史以来最大的贸易协议。〞
 <br/>
 <br/>
 特鲁多也表示，整个谈判过程给经济造成了严重的不确定性，如果还没达成协议，这种不确定性只会越来越糟、越来越具破坏性。新协议的签订解除了这些风险。
 <br/>
 <br/>
 《美墨加协定》（USMCA）是10月初达成的。9月30日夜，美加经过数轮艰苦的谈判，最终就关键性问题达成妥协，为挽救北美自由贸易市场铺平了道路。
 <br/>
 <br/>
 根据美加达成的协议，今后加拿大将向美国开放乳业市场，而美国则暂停向加拿大生产的汽车加征关税。
 <br/>
 <br/>
 加拿大与美国协商成功后，加入此前已达成的美墨协议，将协议由双边变为三 边，维护了整个北美的自由贸易市场。
 <br/>
 <br/>
 美国并没在新协议中应允永久免除钢铝税，加拿大和墨西哥对此表示不满。特鲁多说，希望能够进一步加强合作。
 <br/>
 <br/>
 另外，新协议能否执行还要取决于美国国会批准与否。民主党在中期选举中赢得众议院后底气大增，他们质疑称，新协议没有保护美国工人的利益，因为墨西哥工资低，在竞争中可能会削弱美国工人的优势。
 <br/>
 <br/>
 新协议也需获得加拿大和墨西哥立法机构的批准。但三国政府达成协议，已为北美公平自由贸易市场的形成完成了最关键的一步。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
