### 大篷车移民渡河入墨 川普发警告
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 政府阻挡成批的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 入境，从周六（10月20日）开始，大约2000名滞留在墨西哥与危地马拉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境桥.html" target="_blank">
  边境桥
 </a>
 的移民开始强行渡河越境，并誓言继续向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 挺进。
 <br/>
 <br/>
 据《华盛顿时报》（The Washington Times）21日报导，周六，一部分决意入境的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 在危地马拉边境城市希达哥（Ciudad Hidalgo）的一个公园里聚集，并在商议后表示，他们已放弃合法进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 的尝试，因为申请庇护的程序太慢了。
 <br/>
 <br/>
 这部分〝难民〞举手表决，最终决定继续向北挺进，并敦促滞留在跨越苏恰特河（Suchiate River）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境桥.html" target="_blank">
  边境桥
 </a>
 上的其他移民加入他们的行列。
 <br/>
 <br/>
 周六，数千
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 〝大篷车难民〞冲向危地马拉与墨西哥之间的边境桥妄图入境，墨西哥当局不得不出动大批防暴警察把守边境，并严格控制人数、分批放人办理难民登记手续。
 <br/>
 <br/>
 但是，很多移民迫不及待地想要穿越边界。
 <br/>
 <br/>
 来自洪都拉斯丹利（Danli）的移民杜阿尔特（Erasmo Duarte）对自己的意图直言不讳：〝我们准备到达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 。〞
 <br/>
 <br/>
 尽管川普总统几天前刚刚警告，将在美国中期选举前把大篷车移民和边境安全纳入竞选议题，并派军队驻守美墨边境拦截非法移民，但仍有众多洪都拉斯移民不肯放弃尝试。
 <br/>
 <br/>
 墨西哥政府拒绝移民大规模入境，只接受小部分人的庇护申请，并对部分人发放为期45天的入境许可。
 <br/>
 <br/>
 墨西哥内政部说，他们已收到640份来自边境的难民申请。
 <br/>
 <br/>
 很多移民已不耐烦，并回避通过边境大门入境，公然在数百墨西哥防暴警察的注视下踏上竹筏或是游泳、涉水穿越界河。一些人向当地人支付了相当于1.25美元的费用，从而得以被送过泥泞水域。
 <br/>
 <br/>
 这些人抵达墨西哥沿岸后，没有遭到警察拘留。
 <br/>
 <br/>
 洪都拉斯总统埃尔南德斯（Hernandez）和危地马拉总统莫拉莱斯（Jimmy Morales）在危地马拉进行紧急会晤后表示，一周以来，估计已有5400洪都拉斯移民进入危地马拉，但其中大约2000人目前已自愿返回。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑:东野）
 <br/>
