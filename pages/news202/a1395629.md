### 土耳其扩大搜查 沙特记者失踪悬念未解
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】下面继续关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 记者卡舒吉失踪案的进展。周二一早，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 警方对沙特领事馆搜查九小时后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扩大搜查.html" target="_blank">
  扩大搜查
 </a>
 范围，包括了沙特总领事奥泰比)的住所，以进一步取证调查。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 媒体报导说，周二（10月16日），就在土耳其警方准备对奥泰比的住所进行搜查前的几小时，奥泰比搭乘商业航班，离开土耳其返回利雅得。
 <br/>
 <br/>
 卡舒吉是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 知名记者，曾是沙特王室顾问，经常批评王储。本月2号，他进入沙特驻伊斯坦布尔领事馆后失踪。土耳其认为他在领事馆内遭到谋杀，尸体被转移。
 <br/>
 <br/>
 土耳其电视台曾播出录像，一辆大轿车在卡舒吉进入领事馆两小时后离开领事馆，随后停在领事的住所。
 <br/>
 <br/>
 卡舒吉失踪事件引发国际关注。周一，美国总统川普和沙特国王通电话，周二，国务卿蓬佩奥前往沙特，会见了沙特国王和王储萨勒曼，重申美国对卡舒吉失踪的担忧。
 <br/>
 <br/>
 沙特王室否认知情，但据CNN和《纽约时报》报导，沙特王室准备承认审讯卡舒吉意外致死，但消息尚未得到证实。
 <br/>
 <br/>
 新唐人记者蔡美音综合报导
