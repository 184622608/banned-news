### 大篷车移民闯入境口岸 爆发小规模暴力事件
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】周日（11月25日），几百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 突破墨西哥防暴警察的封锁，向美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入境口岸.html" target="_blank">
  入境口岸
 </a>
 挺进。个别移民日前已开始用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力.html" target="_blank">
  暴力
 </a>
 对付美国边境人员，墨西哥边境城市上周也宣布面临人道主义危机。
 <br/>
 <br/>
 周日，几百名滞留边境的中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 推开墨西哥防暴警察，企图强行进入美国。他们的行为是有组织地进行的。
 <br/>
 <br/>
 上周一（11月19日），美国边境官员关闭了加州境内的圣伊西德罗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入境口岸.html" target="_blank">
  入境口岸
 </a>
 （San Ysidro Port of Entry），并加紧安装障碍物以应对数千名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民的到来。该口岸是美国最繁忙的入境口岸，也是从墨西哥蒂华纳（Tijuana）进入加州圣地亚哥（San Diego）的交通要塞。
 <br/>
 <br/>
 美国官员称，为应对移民，美方周日在边境口岸增加了人员部署，其中包括空军和海军陆战队员。
 <br/>
 <br/>
 周六（11月24日），美国边境人员首次报告了大篷车移民的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力.html" target="_blank">
  暴力
 </a>
 行为。一名31岁的洪都拉斯男子先点燃一棵树，然后又向美国边境人员及直升飞机投掷石块。
 <br/>
 <br/>
 移民们手持手绘的美国国旗和洪都拉斯国旗，高呼〝我们不是犯罪分子！我们是国际工人！〞，施压美国边境人员让他们进入美国。
 <br/>
 <br/>
 目前滞留在蒂华纳的中美洲移民已有5,000多人，他们希望向美国申请庇护并进入美国。美国边境官员正在处理他们的申请，但每天只能处理100多份，远远无法满足移民大军的要求。
 <br/>
 <br/>
 一些墨西哥警察携带防暴盾牌站在墨西哥海关一侧，但大篷车移民推开他们，很容易就突破了警方的封锁，并向美国海关挺进。
 <br/>
 <br/>
 上周五，蒂华纳市市长佳斯特姆（Juan Manuel Gastelum）宣布，向数量巨大的大篷车移民提供食宿，这已使蒂华纳市陷入人道主义危机。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
