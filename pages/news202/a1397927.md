### 逼跪掌掴下属 韩研发公司老板下手狠
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】韩国一家机械人研发公司老板涉嫌欺凌员工，除
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_掌掴.html" target="_blank">
  掌掴
 </a>
 下属及逼迫员工下跪外，还被揭发强迫员工用弓箭和武士刀杀鸡，过程被拍下后报警。警方2日将这名老板拘捕，并封锁其住宅、办公室等10个地点展开搜查。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩泰未来科技.html" target="_blank">
  韩泰未来科技
 </a>
 （Hankook Mirae Technology）董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梁振浩.html" target="_blank">
  梁振浩
 </a>
 被调查，媒体爆出体罚员工片段。从视频中可见，梁振浩双手插腰，对跪在地上的员工破口大骂。员工爬起身后他继续骂：〝你没有眼睛吗？你以为你是谁啊？〞随即右手大力
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_掌掴.html" target="_blank">
  掌掴
 </a>
 员工。被打的员工脸上立刻出现红手印，想后退但梁振浩仍步步进逼：〝好笑吗？看着我！你这混帐！〞再举起左掌掴员工。之后更不断迫员工下跪道歉。
 <br/>
 <br/>
 画面中除了有一名员工(像是主管)在梁振浩旁请他息怒外，所有员工都不敢看。
 <br/>
 <br/>
 10月30日，调查媒体再发布另一支片段，显示梁振浩在公司的工作坊活动中要求下属射杀活鸡。但下属没射准，梁振浩先对下属不满：〝你是在演戏吗？〞最后亲手一箭射中在地上的活鸡，活鸡也立刻发出悲鸣。
 <br/>
 <br/>
 另一支片段则显示梁振浩下令要员工挥武士刀〝凌空击杀〞活鸡，由一名员工负责抛活鸡，让另一名持刀员工击杀；他还亲自指导，要员工由上往下挥武士刀。最后活鸡应声倒地，羽毛四散。
 <br/>
 <br/>
