### 孟晚舟欺诈罪保释听证暂无结论 下周继续听证
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】当地时间12月7日下午，加拿大法院对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 首席财务官、副董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 的案件进行保释
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_听证.html" target="_blank">
  听证
 </a>
 。法官未对孟晚舟的保释申请作出裁决，听证将在12月10日继续进行。在此之前，孟继续处于拘押状态。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_听证.html" target="_blank">
  听证
 </a>
 会在加拿大温哥华卑诗省最高法院举行，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 身着绿色上衣和长裤出庭。经过长达5个小时的听证后，法官并未作出是否交保的决定，但表示会在周末考虑各方提出的保释条件，而下次开庭订于10日上午10点。
 <br/>
 <br/>
 加拿大检察官在法庭上表示，孟晚舟涉嫌欺诈罪，美国司法部在今年8月向她发出逮捕令。
 <br/>
 <br/>
 检方称，从2009年到2014年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 涉嫌利用其在香港的非官方子公司〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Skycom.html" target="_blank">
  Skycom
 </a>
 〞在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 开展业务，违反了美欧对伊朗的制裁。孟晚舟刻意隐瞒〝Skycom〞和华为的关系，误导美国金融机构与〝Skycom〞进行生意往来，从事与对伊制裁相抵触的业务。
 <br/>
 <br/>
 路透社曾在2012年披露，华为的合作伙伴〝Skycom〞向伊朗一家手机通讯公司出售了至少130亿欧元的惠普电脑设备，此举违反了对伊朗的制裁。
 <br/>
 <br/>
 〝Skycom〞的记录显示，孟晚舟在2008年2月至2009年4月期间担任公司董事。
 <br/>
 <br/>
 检方进一步指出，自〝Skycom〞2017年4月遭美国调查以来，孟晚舟就再没有去过美国。这意味着，孟晚舟意识到调查后，刻意避免前往美国。而在此前孟晚舟曾频繁赴美。
 <br/>
 <br/>
 检方表示，如果孟晚舟在美国被定罪，她每一项控罪的最高刑期是30年，这令她有潜逃回中国的风险。由于孟拥有大量资金来源，再多的保释金对她而言都会显得微不足道。
 <br/>
 <br/>
 检方还解释，保释的授予是基于〝信任〞，孟晚舟具有〝广泛的不诚实模式〞（extensive pattern of dishonesty），因此没有资格保释。此外，检察官还特别提醒法官，中国与加拿大及美国之间都没有引渡条约。
 <br/>
 <br/>
 孟晚舟的律师则说，孟有高血压和睡眠窒息症等健康问题，又指其愿意放弃仅有的两本护照，包括中国护照，和在被捕时遭扣押的香港特区护照。同时，法庭可以凭孟晚舟的个人尊严，和她对父亲，华为创办人任正非的尊重，不会违反法庭的命令，孟还愿意接受电子监视作为保释条件。
 <br/>
 <br/>
 由于此次的听证会未能达成结果，下一次听证会将于12月10日继续进行。
 <br/>
 <br/>
 此外，决定孟晚舟是否会被引渡到美国的〝引渡听证会〞将与保释听证会分开举行。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：戴明）
 <br/>
</div>
