### 金正恩生日非国定假日 原因或难说出口
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】朝鲜（北韩）国营机构发行的2019年版的日历中，与之前一样，并没把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜领导人.html" target="_blank">
  朝鲜领导人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生日.html" target="_blank">
  生日
 </a>
 列为法定假期。有分析认为，此举不单是因金正恩没有傲人政绩，还可能有说不出口的原因，那就是没钱。
 <br/>
 <br/>
 韩联社周二（4日）报导，朝鲜今年的月历由朝鲜外文出版社制作，在2019年日历中，朝鲜已故领导人金日成的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生日.html" target="_blank">
  生日
 </a>
 （4月15日）与金正日的生日（2月16日），都是被标记为红色，为国定假日，唯独
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 （Kim Jong-un）1月8日的生日是用与平常日子一样的黑色标记，为非假日。
 <br/>
 <br/>
 报导称，金正恩自2012年执政以来，朝鲜一直都没在官媒公开提及或庆祝金正恩的生日。金正恩的生日也从未列为公众假日。若照日历的讯息解读，明年朝鲜官方也不会替金正恩举行大型的庆生典礼。
 <br/>
 <br/>
 报导引述分析称，金正恩有可能认为自己在政治上尚未有〝傲人成绩〞，同时他可能是觉得自己还年轻，需向其父和爷爷表达敬意。
 <br/>
 <br/>
 不过，也有分析认为，这背后可能还有更实际的、说不出口的经济原因。因为如果把金正恩的生日列为国定假日，当局必须在假日的当天向国民分发大量食物，例如油、肉等，以朝鲜目前的经济状况来看，这是一个不小的负担。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
</div>
