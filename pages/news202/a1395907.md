### 19死逾40伤  克里米亚校园枪案凶手老练如特种兵
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】克里米亚本周三发生的校园枪击事件已造成19人死亡，其中年纪最小的仅15岁，另有逾40人在这场袭击中受伤。俄媒称，18岁的主嫌行凶行动十分老练，如经验老到的特种部队成员。
 <br/>
 <br/>
 当地时间10月17日下午，现年18岁的男生罗斯里亚科夫（Vladislav Roslyakov）手持枪械闯入克里米亚港市（Crimea）一所技术学院并大开杀戒，随后举枪自尽，令国际社会十分震惊。截至目前，俄罗斯调查人员仍在调查凶嫌犯案的动机。
 <br/>
 <br/>
 据俄国《工商日报》（Kommersant）报导，枪手Vladislav Roslyakov来自一个相当贫穷的家庭，其父亲是身障人士，母亲是一所医院里的杂役。
 <br/>
 <br/>
 报导称，凶嫌17日下午持枪闯入技术学院后，在校园中〝逐一进入各个教室，宛如经验老到的特种部队成员，先丢土制手榴弹，再开枪扫射。〞
 <br/>
 <br/>
 俄罗斯调查委员会则对外表示，监视摄影机拍摄到了罗斯里亚科夫持枪进入学校以及在校园内疯狂滥射的画面。
 <br/>
 <br/>
 事后，学生和教职员回忆称，事件发生时，首先传出爆炸，接着又发生多起爆炸，然后是一阵枪击，校园内顿时一片混乱，惊慌失措的学生试图逃离建筑物。。
 <br/>
 <br/>
 这所学院的校长格瑞班尼柯娃（Olga Grebennikova）在接受克里米亚媒体采访时说：〝到处都是尸体，到处都是孩子的尸体，这是真正的恐怖行径！〞
 <br/>
 <br/>
 乌克兰克里米亚半岛2014年被俄罗斯兼并，为此国际社会对俄罗斯提出了严厉谴责，不少西方国家还对俄罗斯采取了制裁措施，但该半岛一直没有因俄的兼并而爆发重大暴力事件。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
