### 日本樱花竟在秋季开放 专家：可能是天气异常惹的祸
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 天气预报公司周三（17日）表示，近日从北海道至九州，各地都传出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_樱花.html" target="_blank">
  樱花
 </a>
 开放的消息。在日本，樱花绽放向来代表春季来临，为什么会在入秋时开花？对此，专家分析，这可能是今年天气异常所致。
 <br/>
 <br/>
 据天气预报公司〝Weather news〞称，该公司在10月9日收到民众投书吉野
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_樱花.html" target="_blank">
  樱花
 </a>
 开的讯息，民众还提供了在九州宫崎县延冈市拍摄的照片。照片中可见，在几乎已经没有树叶的树枝上出现几朵绽放的樱花。
 <br/>
 <br/>
 截至本月14日该公司共收到354起樱花开花报告，发现范围遍及九州至北海道，特别是今年受颱风影响严重的关东南部、近畿、东海地区。远离海洋的内陆地区也有开花报告，日媒也在几个赏樱名胜发现花踪。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 花之会的树医和田博幸推测，异常天气可能是导致樱花突然开花的原因。他说，盛开于春天的樱花，通常在7、8月形成花芽，但因那时气温很高所以不会开花。此时花芽根部连着的叶子会发出一种植物性荷尔蒙离层素(abscisic acid)，抑制花开。
 <br/>
 <br/>
 而今年因颱风频繁来袭，带来强风和盐害，导致树叶大量脱落，无法合成足够的可抑制花芽生长的物质，再加上颱风后出现异常温暖的天气，可能是出现罕见〝秋樱〞的原因。
 <br/>
 <br/>
 和田博幸说，之前也曾发生过类似情况，但他不记得有过这种大规模开花的情况。对于是否会影响明年的赏樱季，和田博幸表示，现在绽放的樱花明春可能不会开花，但因为异常开花情况只是一小部份，应该不会影响明年的赏樱季。
 <br/>
 <br/>
 不过，和田博幸担心的是，颱风导致樱花树树枝折断或从根部坍塌，如果没有及时适当修剪，樱花树可能会从断枝的地方开始腐烂，这可能会影响未来几年樱花的花开荣景。
 <br/>
 <br/>
 另外，红叶前的银杏黄叶也受到颱风带来的盐害影响出现异常情况，不少银杏的叶子掉落、变成褐色或枯萎，不像以前会转变成黄色，形成亮丽风景线。
