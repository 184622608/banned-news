### 欧盟认同脱欧草案 〝直布罗陀〞问题待决
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国首相.html" target="_blank">
  英国首相
 </a>
 特雷莎・梅周四表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 委员会已经同意了英国提出的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 。周日27个欧盟成员国将进行表决，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 已表示将投反对票。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国首相.html" target="_blank">
  英国首相
 </a>
 特雷莎・梅：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 和英国已经就(脱欧)协议文本达成一致。
 <br/>
 <br/>
 虽然英欧在原则上达成了一致，但直布罗陀问题可能将使情况变得复杂。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 首相桑切斯周三再次表示，如果脱欧协议中不能保证西班牙对直布罗陀未来地位拥有否决权，那么西班牙将在周日投下反对票。
 <br/>
 <br/>
 300年前西班牙将直布罗陀割让给英国，但西班牙一直声称拥有直布罗陀主权，可是直布罗陀人自己并不想回到西班牙。西班牙认为，现在直布罗陀跟着英国脱欧，这不是英国和欧盟之间的问题，还要加上西班牙。但英国把西班牙排除在外。现在就要看周日怎么摊牌了。
 <br/>
 <br/>
 另外，12月英国下议院的投票也不乐观。首相梅的执政伙伴北爱尔兰民主统一党(DUP)党魁福斯特周三表示，该党10名议员不支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧草案.html" target="_blank">
  脱欧草案
 </a>
 。
 <br/>
 <br/>
 北爱民主统一党党魁 福斯特：这个草案很糟糕。首相应想到她在下议院得不到足够支持，她应利用这个机会回过头来，去为英国人达成一个更好的方案。
 <br/>
 <br/>
 再加上反对党工党议员，执政党保守党中的强硬脱欧派议员，和部分刚刚加入反对阵营的留欧派议员，目前看来，脱欧草案无法赢得下议院多数。
 <br/>
 <br/>
 首相梅并不是完全没有办法了，如果草案在下议院被否决，那么英国可能面临无协议脱欧，这是绝大部分议员都想避免的。而且到时候二次公投的呼声将空前高涨，〝留在欧盟〞可能成为选择之一，这是脱欧派绝对不愿看到的。那么在下议院投票时，有些议员可能会权衡利弊，转而支持首相的方案。
 <br/>
 <br/>
 新唐人电视综合报导
