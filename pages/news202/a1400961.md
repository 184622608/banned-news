### 脆弱娇嫩〝番红花〞 希腊青年返乡种〝红金〞翻身
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_番红花.html" target="_blank">
  番红花
 </a>
 ，被视为和松露、鹅肝、鱼子酱世界三大美食并列的顶级食材，素有〝香料女王〞之称，是许多知名料理不可或缺的香料。它还具有疗效，甚至可当麻醉药与染料。而如今它也成为振作
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希腊.html" target="_blank">
  希腊
 </a>
 经济的良方。
 <br/>
 <br/>
 旱地上，美丽的紫色花朵在清晨陆续绽放，不仅为大地带来生机，更为疲弱的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希腊.html" target="_blank">
  希腊
 </a>
 经济带来转机。这些花朵有着与本身颜色不同的响亮名字，它们是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_番红花.html" target="_blank">
  番红花
 </a>
 (crocus flower)，也称做藏红花，在希腊
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济危机.html" target="_blank">
  经济危机
 </a>
 期间，成了许多人的经济支柱。被当地人称之为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红金.html" target="_blank">
  红金
 </a>
 〞的番红花食用香料(saffron)，就是取自于这娇嫩花朵中，花蕊上细微的红色柱头。
 <br/>
 <br/>
 希腊年轻人这几年因为高失业率，不少人选择回到偏乡小镇种番红花，带动番红花产业蓬勃发展。拥有双学位的土木工程师历西斯・基罗 (Zisis Kyroy)，在炎热的太阳下，走在田间采集番红花。他曾经在失业中挣扎，最后选择放弃都市生活回到他希腊北部的家乡科罗克斯 (Krokos)种植番红花。生产番红花香料给了他稳定的收入。
 <br/>
 <br/>
 番红花农 基罗：〝回归故乡是一个选择，我并不后悔而且我也会告诉其他正在努力找工作的年轻人，一起参与种植番红花。〞
 <br/>
 <br/>
 番红花是世界上最昂贵的食用香料，因为它既费时又费工。它非常脆弱娇嫩，花朵枯萎的很快，最好在清晨以人工采摘完毕。一切都必须手工完成，从采摘到清洗，分拣到烘干等。而农民需要采摘15万花朵，才能挑拣出一公斤珍贵的番红花。
 <br/>
 <br/>
 因为许多年轻人的加入，当地农场合作社的成员人数翻了一番。从2000年以来，当地的番红花产量从一开始仅供内销的30公斤增加到目前的4公吨。它的出口正在蓬勃发展，约有70％销到欧美和日本等国。
 <br/>
 <br/>
 科罗克斯当地生产番红花已有三个世纪的历史。对于世代在那种植番红花的家庭来说，很乐见年轻人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_返乡.html" target="_blank">
  返乡
 </a>
 务农，他们希望能将这传统作业世世代代传承下去。
