### 墨西哥将迎新总统 曾承诺会尊重美国
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】本周六，在今年7月赢得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 大选的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥夫拉多尔.html" target="_blank">
  奥夫拉多尔
 </a>
 将正式上任，成为新一任墨西哥总统。
 <br/>
 <br/>
 现年55岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥夫拉多尔.html" target="_blank">
  奥夫拉多尔
 </a>
 被认为是一名左翼民粹主义及民族主义者，曾三度参选总统，最终在今年7月以53%的得票率高票当选。
 <br/>
 <br/>
 奥夫拉多尔在大选中承诺将遏制腐败，实现变革，包括停止耗资巨大的新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 城国际机场的建造，主张通过公投来终止墨西哥石油公司的垄断，提高对学生及老人的补贴等。
 <br/>
 <br/>
 墨西哥候任总统 奥夫拉多尔：〝我在这里不是当装饰品，我拥有墨西哥人民的授权，墨西哥人希望结束腐败及有罪不罚的现状。〞
 <br/>
 <br/>
 尽管此前并无外交经验，但奥夫拉多尔在候任期间展现出了较好的外交能力，预计将引领墨西哥回归传统外交政策。
 <br/>
 <br/>
 在对待重要邻国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 方面，奥夫拉多尔承诺将会尊重美国及川普总统，得到川普的赞赏。
 <br/>
 <br/>
 由于川普目前正在阿根廷出席G20峰会，美国将会派出副总统彭斯、白宫顾问伊万卡．川普前往墨西哥，出席奥夫拉多尔的就职典礼。
 <br/>
 <br/>
 新唐人记者乔安综合报导
</div>
