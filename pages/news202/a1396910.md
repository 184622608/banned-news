### 韩朝军事将领会议 板门店将成无武器区
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】韩国和朝鲜周四解除了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_板门店.html" target="_blank">
  板门店
 </a>
 共同警备区的武装。周五，两国军事将领在〝非军事区〞举行会谈，商讨实施最新达成的军事协议。
 <br/>
 <br/>
 周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩朝军事会议.html" target="_blank">
  韩朝军事会议
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_板门店.html" target="_blank">
  板门店
 </a>
 朝鲜一侧的统一阁举行。韩国防部对朝政策官员、陆军少将金度均，和朝鲜军方首席代表、陆军中将安益山等人会面，商讨实施最新的军事协议，还可能讨论恢复联合军事委员会等问题。
 <br/>
 <br/>
 韩国防部对朝政策官 金度均：〝根据9.19军事协定，最终看到板门店共同警备区实现非军事化，非常高兴。〞
 <br/>
 <br/>
 上个月，最新的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩朝峰会.html" target="_blank">
  韩朝峰会
 </a>
 达成军事协议，停止所有韩朝边境的敌对行为，包括停止军事演习、设立禁飞区、非军事区内撤离武器和排雷等。
 <br/>
 <br/>
 上周，两国排除了共同警备区内的地雷。
 <br/>
 <br/>
 周四，韩国国防部表示，在非军事区排雷期间，挖掘出两名韩国士兵遗骸，非军事区内可能还有一万多名士兵遗骸。
 <br/>
 <br/>
 明年4月起，两国将联合展开遗骸发掘工作。
 <br/>
 <br/>
 据报，周六，板门店在1953年签署停战协议六十多年来，将首次没有任何武器。一旦两国国防官员，以及联合国专家评估后，将只保留35名不携带武器的卫兵，在共同警备区巡逻。届时，游客可以在该区域自由行走。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
