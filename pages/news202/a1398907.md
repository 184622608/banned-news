### 日本野猪撞翻西装男猛攻12秒 双脚缝13针(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月11日讯】日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福冈.html" target="_blank">
  福冈
 </a>
 街头日前发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野猪.html" target="_blank">
  野猪
 </a>
 攻击人事件，一名穿着西装笔挺的男子走在街上，冷不防被一头野猪撞倒，野猪还以利牙攻击男子后逃逸，男子双脚遭咬受到撕裂伤，缝了13针。
 <br/>
 <br/>
 事发地点就在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福冈.html" target="_blank">
  福冈
 </a>
 JR〝今宿〞站前，上月26日，一名穿着西装的上班族正缓缓走在街上，突然有一只身长约1公尺的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野猪.html" target="_blank">
  野猪
 </a>
 快速从左侧窜出，当场将他撞翻倒地，男子还来不及爬起来又被连续顶撞好几下，急忙用公事包阻挡，加上经过的驾驶狂按喇叭，试图将野猪吓跑，大约12秒，野猪才扬长而去。
 <br/>
 <br/>
 这位48岁的上班族双脚受到撕裂伤，送医后缝了13针。
 <br/>
 <br/>
 据了解，今宿站附近有一座山，这只野猪应该是因为交配期到山下来寻找对象，但在轨道附近被电车吓到，慌忙逃离时正好遇见西装男才会展开攻击，事后这只野猪在附近沿海地区被捕获后，交由当地的狩猎俱乐部处理。
