### 冲突升级 俄封锁2港口 乌挖掘战壕备战
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 近日发生严重军事冲突，俄罗斯除了在克里米亚半岛部署防空飞弹系统外，还封锁两处乌克兰港口；乌克兰总统波罗申科则宣布戒严30天，并称双方随时有爆发全面战争的风险，而驻守在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚速海.html" target="_blank">
  亚速海
 </a>
 沿岸的乌克兰军队士兵，已开始挖掘战壕备战。
 <br/>
 <br/>
 据《路透》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 建设部长奥梅良(Volodymyr Omelyan)29日(周四)在脸书上发文，位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚速海.html" target="_blank">
  亚速海
 </a>
 的两个港口，别尔江斯克（Berdyansk）与马里乌波尔（Mariupol），已遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 封锁，并禁止任何船隻进出。已有35艘船隻在活动时，只被允许开往俄罗斯港口。
 <br/>
 <br/>
 亚速海东为俄罗斯、北接乌克兰、西临克里米亚，行经的船隻主要载运谷物和钢铁。
 <br/>
 <br/>
 奥梅良表示，〝俄罗斯藉由封锁乌克兰在亚速海的港口，逼我们放弃自己的领土，这块依照国际法规定属于我们的领土。〞
 <br/>
 <br/>
 他说，还有18艘船隻等待进入亚速海(Azov Sea)，其中4艘要到别尔江斯克、14艘到马里乌波尔。
 <br/>
 <br/>
 英国《每日快报》报导，为随时应对与俄罗斯的潜在冲突，驻守在距离亚速海沿岸只有800米村落的乌克兰军队士兵，自戒严日开始，即在亚速海沿岸挖掘战壕。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-29/p9149191a297925227.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-29/p9149191a297925227-ss.jpg"/>

   </font>
  </a>
 </center>
 <br/>
 有士兵透露，部队正在打包紧急口粮和弹药，〝如果有什么事发生，我们可以对抗敌人的攻击，并向各个方向移动，而不是只停留在一个地方。〞
 <br/>
 <br/>
 当地时间11月25日，三艘乌克兰海军军舰穿越俄罗斯边境，并向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克赤海峡.html" target="_blank">
  克赤海峡
 </a>
 (Kerch Strait)上航行，期间，俄罗斯船隻与乌方沟通无果，向乌克兰军舰开火，并扣留军舰和船员。
 <br/>
 <br/>
 乌克兰总统波洛申科于隔日宣布戒严30天，并称俄罗斯军队已将驻守边境的坦克数量增加了3倍，不是开玩笑的，双方随时有爆发〝全面战争〞的风险。
