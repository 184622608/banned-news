### 美就退出中导条约发最后通牒 俄专家指罪在中共
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 后第三天，美国务卿宣布，如果俄罗斯60天内不拆除违约开发的导弹，美国将退出《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中导条约.html" target="_blank">
  中导条约
 </a>
 》。俄方一面否认违约，一面指责中共大举开发中程导弹才是祸首。中共已是美国最大的对手，外界认为美方此举的真正目标是中共。
 <br/>
 <br/>
 12月4日，在比利时布鲁塞尔参加北约外长会议的美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 对外宣布，如果俄方在60天后仍没有〝完全、可验证地〞履行《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中导条约.html" target="_blank">
  中导条约
 </a>
 》，那么美方将暂停履行自己的条约义务，并启动正式退出条约的程序。
 <br/>
 <br/>
 蓬佩奥指责，俄罗斯研发和部署SSC-8型陆基巡航导弹（俄方代号为9M729型陆基巡航导弹），这违反了《中导条约》，并严重损害了美国及其盟邦的国家安全。
 <br/>
 <br/>
 他还补充说，在这60天内，美国不会生产、试验或部署《中导条约》所禁止的导弹。
 <br/>
 <br/>
 《中导条约》是美苏冷战期间两国签署的协议，内容是双方承诺销毁和停止开发中程导弹。
 <br/>
 <br/>
 面对美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最后通牒.html" target="_blank">
  最后通牒
 </a>
 ，莫斯科方面的反应有两个：一个是否认违约，另一个是谴责中共。
 <br/>
 <br/>
 俄罗斯外交部女发言人扎哈罗娃当天立即否认美方指责，声称〝俄罗斯一直在严格遵守协议〞。但她并没有解释关于俄罗斯9M729型陆基巡航导弹的问题。
 <br/>
 <br/>
 俄罗斯联邦委员会国际事务委员会第一副主席扎巴罗夫则威胁称，如果美国退出《中导条约》并在欧洲部署导弹，俄方将作出同等回应，在境内欧洲部分扩大同类导弹的数量。
 <br/>
 <br/>
 另外，俄罗斯媒体和专家都表示，《中导条约》可能作废的责任主要在中共，认为这是中共全面研发各式中短程导弹的结果。
 <br/>
 <br/>
 俄罗斯著名时事评论人士米哈利尔．塔拉图塔表示，美俄之间完全可以谈妥《中导条约》的技术细节，但主要问题在中共，因为中共武器库中90-95%的导弹都是中短程导弹。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）在10月20日首次宣布，美国可能将退出《中导条约》。当时外界分析，在美中走向新冷战的背景下，川普剑指《中导条约》，真正目标是中共。当时白宫也明确了与中共、俄罗斯一起谈判《中导条约》的意图。
 <br/>
 <br/>
 数十年来，因为不受《中导条约》的制约，中共大举发展中程导弹，中国目前已是全世界最大的中程导弹拥有国。这令恪守《中导条约》的美国在这一军事领域明显处于劣势。
 <br/>
 <br/>
 值得注意的是，在蓬佩奥对俄罗斯发出最后通牒的前一天（3日）一早，川普连发四篇推文，谈论1日晚的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 。其中第四篇推文提到，希望在不久的将来，和北京、莫斯科的领导人一起讨论，如何阻止失控的军备竞赛。
 <br/>
 <br/>
 蓬佩奥第二天就对俄罗斯喊话，不得不让人怀疑此举与北京有关。
 <br/>
 <br/>
 在川习会上，美中敲定了暂缓贸易战升级的90天谈判期，但这并没有影响美国在贸易之外的其它领域与中共继续对抗。
 <br/>
 <br/>
 今年8月，川普政府通过的新国防法案中，已经明确将美中关系定性为〝长期战略竞争〞，并将其列为美国的根本战略。
 <br/>
 <br/>
 蓬佩奥12月4日在布鲁塞尔出席马歇尔基金会的演讲时还表示，川普政府正致力于建立基于规则的国际新秩序，以遏制中共、伊朗和俄罗斯等一贯靠违反国际协定获利的〝坏人〞。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
