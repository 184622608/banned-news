### 意大利禁未经许可的难民飞机降落
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 内政部长周日宣布，禁止未经许可搭载
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞机.html" target="_blank">
  飞机
 </a>
 在其机场降落。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 内政部长马泰奥．萨尔维尼（Matteo Salvini）在周日的推文中说，〝在柏林或布鲁塞尔的任何人，如果想用未经许可的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞机.html" target="_blank">
  飞机
 </a>
 把数十名移民送到意大利，他们必须知道，没有任何一个机场会让他们降落。我们关闭所有机场，就像我们关闭所有港口一样。〞
 <br/>
 <br/>
 意大利的劳工部长迪马尤（Luigi Di Maio）也在脸书上回应，他不知道是谁允许租用飞机搭载移民到意大利，因为至今没有签订任何协议。
 <br/>
 <br/>
 意大利和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 正在协商，德国把已在意大利申请庇护的移民送回意大利的程序。目前，两国还未签署任何协议。
 <br/>
 <br/>
 德国内政部发言人则表示，在未来几天内不会有搭载移民的飞机在意大利转机。
 <br/>
 <br/>
 新唐人电视安心综合报导
</div>
