### 大马巫统主席遭控45项罪名 51万美元交保候审
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马来西亚.html" target="_blank">
  马来西亚
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巫统.html" target="_blank">
  巫统
 </a>
 主席、前副首相拿督斯里阿莫札希近期频繁遭反贪污委员会传唤，18日在反贪污委员会进行侦讯遭到逮捕，19日被控涉嫌滥用职权、背信、洗钱等45项罪名，最重恐将面临1.1亿令吉(2811万美元)的罚款。
 <br/>
 <br/>
 综合媒体报导，现年65岁的阿莫札希(Ahmad Zahid Hamidi)曾在纳吉政府期间担任副首相，5月在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巫统.html" target="_blank">
  巫统
 </a>
 大选失利后，出任巫统党代理主席并在此后的巫统选举中当选主席。
 <br/>
 <br/>
 阿莫札希被控违反2009年反贪污法令与2011年反洗钱和与反恐融资法令，并涉嫌于2014年至 2017年间挪用基金会公款，此外也涉及滥用职权、背信等共计45条罪名，包括27条洗钱罪名、8项受贿罪，以及10条背信罪。
 <br/>
 <br/>
 19日上午，阿莫札希前往法庭接受开庭，前首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳吉布.html" target="_blank">
  纳吉布
 </a>
 (Najib Razak)也来到法庭支持，但由于罪状实在太冗长，换了4个口译员才将控状宣读完成。纳吉布则在开庭后半个小时就中途离席。而阿莫札希则全盘否认指控。
 <br/>
 <br/>
 法官最后允许阿莫札希以200万令吉（约51万美元）保外候审，包括在今天需缴付100万令吉，其余则须在一周内缴清。
 <br/>
 <br/>
 马来民族统一机构(United Malays National Organization，巫统) 自1957年大马独立以来一直统治着大马，直到今年5月才在大选中吞败。
 <br/>
 <br/>
 纳吉布此前被控侵吞主权财富基金〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一马发展公司.html" target="_blank">
  一马发展公司
 </a>
 〞(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_1MDB.html" target="_blank">
  1MDB
 </a>
 )款项，遭到逮捕与起诉，成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马来西亚.html" target="_blank">
  马来西亚
 </a>
 第一个卸任后被指控的首相，纳吉太太罗斯玛（Rosmah Mansor）在这个月初也被指控17项洗黑钱罪名。
 <br/>
 <br/>
 (责任编辑：卢勇信)
