### 难民推挤栏杆涌入 川普拟边境驻军
------------------------

 【新唐人2018年10月19日讯】观众朋友大家好，欢迎收看新唐人全球新闻。上周末爆发了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民潮.html" target="_blank">
  难民潮
 </a>
 。如今有三千多〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 〞，冲过危地马拉和墨西哥的边境检查站。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周五表示，如果墨西哥不能阻止难民，美国将派军队驻扎边境。
 <br/>
 <br/>
 在危地马拉和墨西哥边境检查站，三千多〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 〞涌向围栏，推搡冲撞。一些在人潮簇拥下来到金属围栏附近，开始攀爬。在人潮拥挤下，围栏终于被撞开，大批难民走过边境。两辆军用吉普车就停在附近，危地马拉的防暴警察，在一旁观看，没有行动。
 <br/>
 <br/>
 难民多为年轻人或中年人，背着便携背包。虽然背井离乡，但面容轻松。有的微笑向摄影机招手。
 <br/>
 <br/>
 据信，大篷车难民目前进入危地马拉和墨西哥之间的无人地带。
 <br/>
 <br/>
 为躲避国内的黑帮暴力、毒品泛滥和贫困，数千
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 难民想要穿越危地马拉和墨西哥，到美国寻求庇护。
 <br/>
 <br/>
 美国国务卿蓬佩奥周五与墨西哥总统会面，此前他已经与巴拿马总统和外长讨论相关议题。美国要求墨西哥阻止难民前往美墨边境。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统已经明确了我们今天面临的最大问题，在我们谈判贸易协定的时候。我们正在面对一个危机时刻，创纪录的难民数字。〞
 <br/>
 <br/>
 墨西哥已经要求联合国难民署协助处理难民身分的申请。
 <br/>
 <br/>
 总统川普周五对墨西哥的努力表示感谢。
 <br/>
 <br/>
 美国总统川普：〝我要感谢墨西哥，希望他们继续努力。如果不能奏效，我们将动用军队。不是国民警卫队，而是军队。让军队驻扎在边境，不会让他们进来。他们最好掉头回去。不会让他们进入美国。〞
 <br/>
 <br/>
 新唐人记者李子玉综合报导
