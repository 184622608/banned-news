### 危国火峰火山喷发 4000居民惊吓急撤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_危地马拉.html" target="_blank">
  危地马拉
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火峰火山.html" target="_blank">
  火峰火山
 </a>
 （Fuego）今年6月的喷发造成近200人死亡与235人失踪。19日再喷出大量灰烬和岩浆，当局20日宣布进入红色警戒状态，并撤离近4000位居民。
 <br/>
 <br/>
 距离首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_危地马拉.html" target="_blank">
  危地马拉
 </a>
 市约35公里的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火峰火山.html" target="_blank">
  火峰火山
 </a>
 ，海拔3763公尺，今年已第5次喷发。
 <br/>
 <br/>
 今天撤离家园的许多民众表示，他们担心6月的致命火山喷发会再次重演。
 <br/>
 <br/>
 罗迪欧村（El Rodeo）民众贾西亚（Miriam Garcia）说道：〝我们感到害怕，这就是我们撤离的原因。〞
 <br/>
 <br/>
 同样来自罗迪欧村的华瑞兹（Oscar Juarez）表示：〝你必须尽快离开，因为当那些东西（火山物质）靠近时，你就没有时间离开，即便你是用跑的，因为它来得非常快。〞
 <br/>
 <br/>
 火峰火山最近一次喷发从10月12日延续至13日，当时有62人被迫撤离家园，高速公路也因而关闭。
 <br/>
 <br/>
 此外，危地马拉境内巴卡雅火山（Pacaya）和圣地亚吉托火山（Santiaguito）的活动最近几个月来有所增加，但尚未进入爆发阶段。
