### 大篷车难民 墨西哥部长：人道主义危机
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】首先来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 情况。周三，部分进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 的洪都拉斯难民，由于天气炎热，加上长途跋涉，健康堪忧。墨西哥内政部长表示，这是一场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人道主义危机.html" target="_blank">
  人道主义危机
 </a>
 。
 <br/>
 <br/>
 周四，第一批3500名难民已抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 的瓦哈卡州，第二批部分难民滞留在墨西哥塔帕丘拉。一名非盈利组织成员表示，许多难民出现中暑等健康问题。
 <br/>
 <br/>
 非盈利组织成员 JAIME MONTEJO：〝由于饮食不当，造成各种疾病，中暑、疲劳、脚痛，孩子们头痛，还有肠胃疾病。〞
 <br/>
 <br/>
 难民们求助医护人员，缓解病痛。
 <br/>
 <br/>
 周三，墨西哥内政部长纳瓦雷特表示，有6000多难民进入墨西哥，其中2200人申请庇护，700人返回故土。
 <br/>
 <br/>
 他还说，墨西哥正在经历前所未有的局面，不是媒体报道的大篷车，而是成批难民迁移，已发展成〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人道主义危机.html" target="_blank">
  人道主义危机
 </a>
 〞。
 <br/>
 <br/>
 墨西哥内政部长纳瓦雷特：〝这是大批移民迁移，（墨西哥）政府是这样看的，也是这样应对的。墨西哥很少发生现在这样的移民危机。我们在谈人道主义危机。〞
 <br/>
 <br/>
 美国总统川普周三重申，不允许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 入境美国。
 <br/>
 <br/>
 美国总统川普：〝至于说大篷车（难民），我们的军队已经出发。我们有大约5800士兵。我们将增加到一万到一万五千人，另外还有边境巡逻队、移民和海关执法局和其他边境人员。〞
 <br/>
 <br/>
 周三川普在推特上发布视频，批评民主党开放边境政策很危险。视频中的墨西哥裔非法移民布拉卡蒙特斯，2014年杀害两名警察，被送上法庭，但他毫无悔意，甚至扬言要杀更多人。
 <br/>
 <br/>
 新唐人记者毕心慈、张琪综合报导
</div>
