### 川普拟退出中导条约  遏制中共军力再加码
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】日前有美国媒体援引美国政府官员透露，川普（特朗普）政府准备退出30年前与苏联政府签订的《中程导弹条约》（简称《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中导条约.html" target="_blank">
  中导条约
 </a>
 》），不仅是因为俄罗斯近年来屡屡违反这个条约，而且还有部分原因是为了反制中共在太平洋拥有的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中程弹道导弹.html" target="_blank">
  中程弹道导弹
 </a>
 打击力量〞。
 <br/>
 <br/>
 《纽约时报》10月20日的一篇报导援引未具名的美国官员和外交官披露，美国总统川普准备废除30年前（1987年）美国政府与苏联政府签署的《中程导弹条约》（Intermediate-range Nuclear Forces Treaty）。估计白宫国家安全顾问博尔顿（John Bolton）下周出访莫斯科时，将会向俄罗斯领导人普京通报美国政府的这个计划。
 <br/>
 <br/>
 资料显示，《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中导条约.html" target="_blank">
  中导条约
 </a>
 》是上个世纪冷战时期，美国总统罗纳德・威尔逊・雷根与苏联领导人戈尔巴乔夫于1987年签署的一份公约，双方同意按约定各自销毁一定数量的中程导弹。该条约并禁止两国部署所有射程在500至5,500公里之间的陆基巡航导弹和弹道导弹。这一举措是冷战高峰时期的降温之举，曾经在很大程度上消除了冷战末期笼罩在欧洲大陆上空的核阴霾。
 <br/>
 <br/>
 不过，大约从2014年起，俄罗斯当局开始违反这个条约的限制，部署了被条约禁止的核武器来威胁欧洲国家以及与西方结盟的前苏联国家，美国政府为此多次对俄罗斯提出警告。
 <br/>
 <br/>
 报导引述熟知内情的美国官员称，川普政府想要退出《中导条约》的原因不仅是针对俄罗斯违反了条约，还因为这项条约实际上也限制了美国部署新武器来反制中国（中共）不受制约的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中程弹道导弹.html" target="_blank">
  中程弹道导弹
 </a>
 打击力量〞。
 <br/>
 <br/>
 上述美国政府官员表示，中导协议限制了美国在全球任何地方部署中程打击武器，尤其是西太平洋。与此同时，因为中国不是《中导条约》的签约国，中共就可以不受约束地发展可携带核武的中程飞弹，以至于现在的中共国已经成为了〝全球最大的中程弹道导弹打击力量拥有国〞。所以，美国军方现在已经开始针对这种局面做未雨绸缪的准备。
 <br/>
 <br/>
 据报导，针对中共国已部署的中程核武，美国正在发展可用来反制的核武。在当前的过渡期内，美国军方正致力于改良现有的武器，包括可能先把非核武的战斧飞弹部署到亚洲，地点可能是日本或关岛。
 <br/>
 <br/>
 （记者黎明报导/责任编辑：明轩）
