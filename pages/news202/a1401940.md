### 蓬佩奥北约峰会前演讲:川普在建立新秩序
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】美国国务卿蓬佩奥周二表示，美国总统川普正在召集世界各国建立一个新的自由秩序，以防止战争并实现所有人的更大繁荣。同时批评中国、俄国与伊朗违反多个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际公约.html" target="_blank">
  国际公约
 </a>
 。
 <br/>
 <br/>
 周二在参加北约峰会前，蓬佩奥在布鲁塞尔向外交官演讲。他表示，川普总统并不是要破坏北约等机构，也不是放弃美国的领导，恰恰相反，川普是要建立一个新的自由秩序。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝我们正在凝聚世界高尚国家，建立一个新的自由秩序，以防止战争并实现所有人的更大繁荣。〞
 <br/>
 <br/>
 蓬佩奥重申，一些国家说美国的行为不符合世界利益，这是完全错误的。美国希望为民众服务，而不是控制他们。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝我们的使命是重申我们的主权，改革
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自由国际秩序.html" target="_blank">
  自由国际秩序
 </a>
 ，希望我们的朋友予以协助，也发挥他们主权的影响力。希望这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际秩序.html" target="_blank">
  国际秩序
 </a>
 服务于我们的公民，而不是控制他们。〞
 <br/>
 <br/>
 在气候大会召开之际，蓬佩奥解释美国退出巴黎气候协定是因为协定缺乏公平的条款。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝我们退出巴黎协定，因为其中的条款对美国不公平。现有的条款是从美国那里抽钱，补贴中国这样污染严重的国家。〞
 <br/>
 <br/>
 他同时批评伊朗破坏区域稳定，美国退出伊朗协议是为了保护美国和盟友的安全。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝我们退出伊朗协议，因为伊朗的暴力和破坏稳定，破坏了协议的精神，使美国人民和我们盟友处于危险之中。〞
 <br/>
 <br/>
 蓬佩奥呼吁北约各国支付他们的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公平份额.html" target="_blank">
  公平份额
 </a>
 ，加强这一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事联盟.html" target="_blank">
  军事联盟
 </a>
 。
 <br/>
 <br/>
 新唐人记者李梅综合报导
</div>
