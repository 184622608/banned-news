### 疑自拍打晕机师 英富豪一家5口坠机身亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】英国富豪
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金巴斯集团.html" target="_blank">
  金巴斯集团
 </a>
 （Compass）执行长考辛斯（Richard Cousins）去年12月31日一家5口旅游澳洲，在悉尼搭乘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_水上飞机.html" target="_blank">
  水上飞机
 </a>
 时，竟发生坠机意外，造成包括机师在内共6人全数身亡。经最新调查报告，造成飞机失事原因，疑是机上一名乘客在自拍时，意外打昏飞行员，导致飞机失控坠落。
 <br/>
 <br/>
 综合媒体报导，当时这架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_水上飞机.html" target="_blank">
  水上飞机
 </a>
 于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新南威尔斯州.html" target="_blank">
  新南威尔斯州
 </a>
 （New South Wales）郊区考恩（Cowan）霍克斯伯里河（Hawkesbury River）上空飞行，一名目击者表示，飞机坠机当下与他距离约500公尺，他亲眼看见飞机的紧急转弯右旋，而后冲进河水里。
 <br/>
 <br/>
 但最新报告中指出，飞机的状况都良好，飞行也安全，初步判定失事原因应该是人为疏失。疑似是前座乘客在拍摄风景照片与自拍时，不小心撞到44岁机师摩根（Gareth Morgan），甚至打到他的头部造成晕眩，才会导致飞机失事。
 <br/>
 <br/>
 澳洲交通安全局（Australian Transport Safety Bureau）也表示，机师摩根死因几乎是溺死和头部撞伤，而完整调查报告仍在进行中。
 <br/>
 <br/>
 这架出事水上飞机型号为〝DHC-2 Beaver〞，出自于知名飞机公司〝Sydney Seaplanes〞，是雪梨最受欢迎的旅游航班之一，许多政商名流喜爱租机旅游。
