### 北海道5.3地震暂未酿灾 专家提醒后续强震
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北海道.html" target="_blank">
  北海道
 </a>
 今(5)日再度发生里氏规模5.3
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 ，震央位在北海道胆振地方中东部，从震央所在位置等来研判，专家推估，应该是9月6日7级地震的余震，提醒未来还有可能强震发生。
 <br/>
 <br/>
 日本气象厅表示，上午的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 震源深度31公里，各地观测到最大震度五弱，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北海道.html" target="_blank">
  北海道
 </a>
 厚真町、北海道武川町、北海道平取町。
 <br/>
 <br/>
 消防单位表示，这起地震造成住在札幌市东区的一名30多岁女性被倒下的架子压伤，所幸只是轻伤。
 <br/>
 <br/>
 而上次强震的重灾区厚真町附近，这次观测到出现震度5弱有感地震，经从直升机上空观察，截至上午10时止，并没有出现新的土石崩落情况。
 <br/>
 <br/>
 目前各主要电厂、陆空交通都正常，郊区也未出现土石崩落。受地震影响受地震影响的北海道新干线，很快就恢复行驶，其他地方电车线部分则照常行驶。
 <br/>
 <br/>
 东京大学地震研究所教授古村孝志说，从震央所在位置等来研判，今天地震应该是9月6日北海道震度7地震的余震。
 <br/>
 <br/>
 他说，虽然距上次强震已经过约一个月，余震次数持续减少，但未来还是有可能发生强烈摇晃的有感地震，大家还是需要多加留意。
 <br/>
 <br/>
