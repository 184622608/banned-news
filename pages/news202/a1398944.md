### 至少五千大篷车移民继续北上 欲入美国
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】川普政府边境政策更加严格，但来自中美洲的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 周日仍继续向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 边境移动，尽管他们难以确定能否进入美国。
 <br/>
 <br/>
 周日（11月11日），来自中美洲的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 在墨西哥中部克雷塔罗州，争相登上一辆卡车，他们想继续前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 寻求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治庇护.html" target="_blank">
  政治庇护
 </a>
 。尽管川普政府于上周五发公告，在接下来约三个月中，暂停给非法越过美墨边境的外国人提供政治庇护申请，除非美国墨西哥间达成相关协议。
 <br/>
 <br/>
 克雷塔罗州政府统计，从周五到周六，有6,531名中美洲移民穿越该州，并有5,771人周日早晨从当地为他们准备的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_避难.html" target="_blank">
  避难
 </a>
 所启程，墨西哥当地官员并为他们准备了车辆，协助他们北上前往美墨边境。
 <br/>
 <br/>
 川普曾在演说中提到，这些移民中可能混入了一些坏人。为了保护国家安全和维护边境法治，川普当局早已派兵到南部边境协助国土安全部守卫。而这些继续前往美国的中美洲人能否入境，前景一片茫然。
 <br/>
 <br/>
 新唐人电视 综合报导
</div>
