### G20峰会 中共施压阿根廷 阻挠法轮功学员请愿
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】各国首脑在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 之际，再次浮现中共长臂渗透事件。中共施压阿根廷，迫使当地警方阻拦法轮功学员和平请愿。阿根廷法轮大法学会表示，将采取法律行动应对。
 <br/>
 <br/>
 2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 ，11月30号在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 首都布宜诺斯艾利斯开幕。
 <br/>
 <br/>
 29号，数名阿根廷法轮功学员前往习近平下榻的喜来登酒店附近和平请愿。他们手举横幅，要求〝停止迫害法轮功〞、〝法办江泽民〞。
 <br/>
 <br/>
 当天下午，中共驻阿根廷使馆和当地的亲共侨团组织了大批华人，统一乘车，穿红衣，持红旗，来到酒店附近。
 <br/>
 <br/>
 据现场目击者透露，一名阿根廷便衣警察和一名中共大使馆官员交谈后，几名警察随即来到法轮功学员面前，没有说明任何理由，就强行取下法轮功学员的横幅。
 <br/>
 <br/>
 阿根廷法轮功学员表示，他们请愿的出发点，是呼吁立即停止在中国大陆对法轮功的残酷迫害，请愿活动没有任何违法行为。
 <br/>
 <br/>
 阿根廷法轮大法学会发言人 傅丽维：〝阿根廷的法律是，每个人都可以和平地示威、表达、表态，只要是平和的。那是完全合法的。（警察）突然就跑过来，没有讲一句话，就来抢（横幅）。〞
 <br/>
 <br/>
 法轮功学员试图向警方解释；但警察随即上前抢夺横幅，一名警察并用警棍击打法轮功学员的手部，用警棍扼住法轮功学员的喉部，并将他猛烈的向后拉。
 <br/>
 <br/>
 当地警察局长也在现场指挥。他指着法轮功学员的背包，告诉警察〝里面有炸弹〞，警察随即搜查了法轮功学员的背包。
 <br/>
 <br/>
 现场一名阿根廷女警察告诉一名法轮功学员，她知道法轮功学员在做好事，但她爱莫能助，因为命令来自上级。
 <br/>
 <br/>
 现场的阿根廷法轮功学员 Meilin Klemann：〝（女警察）说，我记得你们，上次你们举办过一次很棒的游行，你们很好，你们是好人。她对我说，我很抱歉现在发生的事，但我无能为力。上级命令了，所以我们得这么做。但我知道，我们这么做是不对的，我很抱歉。〞
 <br/>
 <br/>
 现场的阿根廷法轮功学员  Jeremiah Arauz：〝一名女警察对我说，警察的程序是错误的，他们得到的命令是，只允许打红旗的人在那里。她说，警察的做法是错误的，程序上也是错的。〞
 <br/>
 <br/>
 资深中国问题专家 横河：〝在正常情况下，没有中共的压力的话，那么阿根廷警方和其他国家的警方一样，是尊重法轮功学员举办活动或发出声音的情况的。那是民主国家正常的。而在中共的压制下，这些国家会表现出不正常的状态来，像这次在阿根廷的情况就是这样。很明确的，这是来自中共的压力。 〞
 <br/>
 <br/>
 时事评论员 唐靖远：〝这是中共渗透西方，干预一个民主国家的民主生活的鲜活例子。中共利用所谓经济合作及外交要求，在西方世界扩散中共的意识形态，和极权暴政思维，已经对自由社会的人权、信仰自由、言论自由等根本价值观构成了威胁。〞
 <br/>
 <br/>
 当天，警方拘捕了9名阿根廷法轮功学员，并于第二天下午全部释放。
 <br/>
 <br/>
 阿根廷法轮大法学会发布声明说，中共驻阿根廷大使馆收买阿根廷警方，策划了这起暴力拘捕行动。声明重申法轮功的和平请愿是合法的，将采取法律行动，起诉本次事件的肇事者。
 <br/>
 <br/>
 新唐人电视综合报导
