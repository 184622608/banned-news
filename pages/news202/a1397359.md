### 封存两世纪 法国玛丽皇后珍珠现身台北
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】引领18世纪时尚潮流的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 末代皇后－－玛丽安东尼的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠宝.html" target="_blank">
  珠宝
 </a>
 ，将在下个月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏富比拍卖会.html" target="_blank">
  苏富比拍卖会
 </a>
 上，重见天日。这些华丽的首饰，经过动荡的法国大革命时代，保存完整，经过两世纪后首度面世，更显得珍贵。
 <br/>
 <br/>
 日内瓦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠宝.html" target="_blank">
  珠宝
 </a>
 专家 Andres White Correal：〝这是第一次，这套珠宝面世拍卖，两百五十年来首次。每套珠宝都有历史，每一样珠宝背后都有历史，最常见的是情人之间的爱情，特殊日子的纪念。这一套超乎寻常，特别是关系到，一位史上最为著名皇后的命运。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 末代皇后玛丽，因为政治联姻从奥地利嫁到法国，法国大革命时期，她有许多珠宝首饰，被送回奥地利母家保存至今。1793年她被送上断头台处死。
 <br/>
 <br/>
 不过，虽然上好的天然珍珠首饰，被认为更受市场欢迎，但珠宝专家个人，更喜欢与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_玛丽皇后.html" target="_blank">
  玛丽皇后
 </a>
 有更深关连的珠宝。
 <br/>
 <br/>
 日内瓦珠宝专家 Andres White Correal：〝最重要的，是这个珍珠，因为它的尺寸非常罕见，而且它非常华丽。不过，对我个人来说，是这些其它的珠宝，比如戒指，上面有她名字的字母，并且里面有编织过的她的发丝。〞
 <br/>
 <br/>
 除了拍卖玛丽皇后留下的珠宝，另外也有法国王室、奥地利皇室后代的珍藏。而拍卖前的巡展，民众除了皇室珠宝，还能欣赏到知名珠宝制造商打造的首饰，与来自日内瓦以及纽约的各色钻石。
 <br/>
 <br/>
 新唐人亚太电视高健伦、黄彦菱台湾台北报导
</div>
