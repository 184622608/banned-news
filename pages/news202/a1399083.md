### 川普法国行 短暂会普京 互动良好
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统结束了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 的三日行程返回美国，再次强调美国在贸易和北约军费上需要被公平对待。与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 的短暂会晤也被外界关注。下面回顾一下川普的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎.html" target="_blank">
  巴黎
 </a>
 之行。
 <br/>
 <br/>
 周日晚，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 携第一夫人梅兰妮亚返回美国。
 <br/>
 <br/>
 11月9日至11日，川普赴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 、与全球70多国领袖一道参加纪念第一次世界大战一百周年系列活动，俄罗斯总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 也参与其中。
 <br/>
 <br/>
 周日，川普和普京在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎.html" target="_blank">
  巴黎
 </a>
 凯旋门的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一战.html" target="_blank">
  一战
 </a>
 停战百年纪念仪式上握手寒暄，普京还向川普竖起大拇指。
 <br/>
 <br/>
 普京之后对媒体表示，与川普进行了简短且良好的对话，期望月底在阿根廷出席G20峰会期间，与川普进行全面磋商，包括讨论《中导条约》问题。此前有消息，届时，川普也会与习近平会晤。
 <br/>
 <br/>
 时事评论员唐靖远：〝普京向川普竖大拇指以及川普眨眼，他们的友好互动其实并不奇怪。因为从战略关系上看，双方在遏制中共全面扩张这方面，还有重大的契合点。川普退出中导条约，真正原因是美俄都被条约捆住了手脚，而中共却藉机大力发展中程导弹，不但威胁到美国战略利益，其射程更覆盖了俄罗斯全境，普京对此其实心知肚明，他们的表态，更像是在唱双簧而非对抗。从个人特点上看，川普和普京都主张国家优先政策，都与共产主义思潮影响下的全球化格格不入。〞
 <br/>
 <br/>
 马克龙主持了在凯旋门前一战停战百年的纪念活动。
 <br/>
 <br/>
 法国总统马克龙（2018.11.11）：〝让我们再一次庄重承诺和平高于一切。
 <br/>
 <br/>
 此次法国行，川普与马克龙的互动较此前马克龙访美时有所不同。
 <br/>
 <br/>
 在川普9日晚抵达巴黎前3天，马克龙对法国媒体表示，欧洲应组建一支联合部队，防范俄国、中国及美国，德国表达赞同。此言论引发川普强烈不满，称〝非常侮辱〞，他说美国资助北约的太多了，欧洲应该承担责任。
 <br/>
 <br/>
 在10日与马克龙的会晤中，虽然后者释出缓和态度，不过川普平静回应。
 <br/>
 <br/>
 法国总统马克龙（2018.11.10）：〝我相信我的增强欧洲防御提案与此相符。我认为这很公平，也很重要。〞
 <br/>
 <br/>
 双方的会谈，除了就北约军费投入达成共识，还就美国退出《中导条约》、贸易、伊朗等问题交换了意见。
 <br/>
 <br/>
 时事评论员唐靖远：〝对法国和德国而言，美俄关系的改善，可能会冲击到欧盟在国际体系中的地位，因为美俄修好，马克龙想建立欧洲军队就没了足够的理由，同时，俄国一旦重返G7集团，法国或德国在欧洲事务主导权可能面临大幅削弱。〞
 <br/>
 <br/>
 一战百年纪念活动中，川普原计划参访两座美军公墓，不过由于天气原因，他取消了10日在艾思纳－马内（Aisne-Marne American Cemetery）美军公墓的行程。而是于11日前往巴黎外郊的叙雷纳美军公墓（Suresnes American Cemetery）悼念，那里埋葬着约2,491名美军士兵、包括失踪的军人。
 <br/>
 <br/>
 美国总统川普（2018.11.11）：〝他们坚守阵地，并向前推进直到胜利，一场伟大的胜利。〞
 <br/>
 <br/>
 1914至1918年，德、奥、匈等组成的同盟国与英、法、美、中华民国等组成的协约国发生战争，约6500万人参战，1600多万人丧生。美国首次现身欧洲战场，有116,000士兵牺牲。最终，协约国获胜。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
