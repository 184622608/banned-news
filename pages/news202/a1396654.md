### 日媒曝安倍叫停对华援助内幕 中宣部弄巧成拙
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】23日，日本宣布将正式停止对华〝政府开发援助〞（ODA）。而日媒消息，在确定日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三访华.html" target="_blank">
  安倍晋三访华
 </a>
 行程后，中共宣传系统就下令积极报导
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本对华ODA.html" target="_blank">
  日本对华ODA
 </a>
 。以此判断，日本突然宣布叫停对华ODA，应在中共开始宣传之后。
 <br/>
 <br/>
 10月23日，日本外相河野太郎在记者会上正式宣布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本对华ODA.html" target="_blank">
  日本对华ODA
 </a>
 计划将以本年度的新项目为终点，首相安倍25日访华时，将正式告知中方这一决定。
 <br/>
 <br/>
 而据日本《产经新闻》24日报导，有来自中共宣传机构的消息指，在确认安倍25日开始访华后，中共政府即刻向党媒下达指令，要求积极报导日本对华ODA。
 <br/>
 <br/>
 报导指，中共此前一直很少提及日本对华ODA，引起日本国内不满。今年10月23日，是《中日友好条约》40周年，中共政府计划展现〝中日友好氛围〞。
 <br/>
 <br/>
 据此前中共官方报导，早在10月12日，中共外交部就对外宣布，应中共国务院总理李克强邀请，安倍晋三将于10月25日至27日对中国进行正式访问。
 <br/>
 <br/>
 虽然中方有意向日本展现某种〝友好气氛〞，但此次安倍访华的气氛，估计不会那么〝友好〞。
 <br/>
 <br/>
 据此前消息，中日双方本来商定安倍在10月23日，即《中日友好条约》40周年纪念日当天抵达北京，其象征意义明显。
 <br/>
 <br/>
 不过，9月下旬，当日本和欧盟、美国发表部长联合声明、谴责中共贸易不公之后，北京随即以〝不方便〞为由，要求安倍将访华行程向后推延一两天。
 <br/>
 <br/>
 而就在23日当天，日本突然宣布将停止对华ODA，未知是否对中方外交〝失礼〞行为的回敬。
 <br/>
 <br/>
 据日媒报导，日本外相在宣布停止对华ODA时强调，〝考虑到中国的经济水平〞，恐怕已不再需要该项援助，未来日方将在〝平等立场〞上和中方讨论新的合作方法。
 <br/>
 <br/>
 此前，美国总统川普（特朗普）曾多次谴责中共掌控世界第二大经济体，还在假借〝发展中国家〞地位，在世贸组织中捞取好处。日本此次宣布停止援助中共，要求中日〝平等〞，被指有配合美国的意向。
 <br/>
 <br/>
 日本ODA项目始于1979年，是日本政府对某些发展中国家提供的经济援助。其中，对华ODA是长期以来争议较大的议题。
 <br/>
 <br/>
 日本对华ODA被指给中国的变相战争赔款，大多采用低息或无息贷款的形式，至今援助总额超过3万亿日元（约合3千亿人民币）。日本对华ODA在邓小平实施〝改革开放〞初期充当了启动资金的角色，对中共发展经济一度起到很大作用。但是中共为了煽动仇日情绪的政治需要，很少公开提及这个项目。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
