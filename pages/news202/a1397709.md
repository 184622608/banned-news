### 美制裁古巴似〝独行侠〞 UN再扮扯腿角色
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】11月1日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国大会.html" target="_blank">
  联合国大会
 </a>
 以压倒性多数第27次通过决议，呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 终止对古巴的经济禁运。但是否解除对古巴五十多年的禁运，最终要取决于美国国会。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国大会.html" target="_blank">
  联合国大会
 </a>
 就古巴要求终止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 单方面经济制裁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_决议案.html" target="_blank">
  决议案
 </a>
 进行投票，189个国家赞成，美国和以色列反对，乌克兰和摩尔多瓦弃权。
 <br/>
 <br/>
 投票前，美国驻联合国大使黑利告诉与会各成员国，坚持正确的理念。
 <br/>
 <br/>
 美国驻联合国大使黑利：〝但每年最令人遗憾的并不是美国可能独自反对这个决议，最令人遗憾的是浪费每个人的时间。又一次，各国认为他们可以做些对美国不利的事。但是你们这么做时，并没有伤害到美国，你们实际上伤害了古巴人民，你们是在告诉古巴政权，他们对待自己人民的方式是可以接受的。〞
 <br/>
 <br/>
 1992年以来，联合国大会每年都表决通过要求美国结束对古巴经济封锁的决议。美国此前一向投票反对，但在2016年奥巴马任期内首次弃权。
 <br/>
 <br/>
 今年，美国提出了对这一决议的八项修正案，其中包括给予古巴人民集会言论自由等权利，但未获通过。
 <br/>
 <br/>
 新唐人记者蔡美音综合报导
