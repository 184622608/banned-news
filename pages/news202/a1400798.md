### 欧盟通过英国脱欧协议 荷兰总理：所有人都是输家
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】欧洲联盟成员国领袖25日正式通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 协议，英国将成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 28个成员国中首个退出的国家。欧盟执委会主席荣科形容，这将是一场〝悲剧〞。荷兰总理吕特说，所有人都是输家。
 <br/>
 <br/>
 中央社报导，历时18个月的波折谈判，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 27个成员国25日在布鲁塞尔举行高峰会，正式通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 协议及欧英未来关系政治宣言。因事前已经进行了多次协调沟通，因此当天的峰会大约进行了40分钟就通过了协议。
 <br/>
 <br/>
 协议通过让双方进入分手倒数计时，多位领袖出席时发表感言，荣科（Jean-Claude Juncker）说今天是〝令人感伤的一天〞，又说英国脱欧是一场〝悲剧〞，对双方都有影响。
 <br/>
 <br/>
 吕特（Mark Rutte）则说欧洲在英国脱欧事件上没有赢家，所有人都是输家，但协议达到平衡结果，对于继续一起前进的欧盟27个成员国而言是可以接受的。
 <br/>
 <br/>
 而法国总统马克龙（Emmanuel Macron）称，欧盟成员国在重要时刻展现团结，与英国达成的是一项好协议，他希望未来数月内可以与英国进一步界定未来关系，英国未来将继续具有重要角色。
 <br/>
 <br/>
 协议接下来还需经过英国国会及欧洲议会批准，之后英国才能在2019年3月29日脱欧。不过，因欧英有多项歧见未达共识，协议脱欧过渡期到2020年，若再谈不拢也可能延长至2022年。
 <br/>
 <br/>
 根据英国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧协议.html" target="_blank">
  脱欧协议
 </a>
 ，英国到2020年底前与欧盟维持一段过渡期，期间英国虽不再拥有欧盟事务的投票权与影响力，但其他部分仍暂时维持不变，包括欧洲公民在英国居住与工作权益、英国仍受欧盟预算规范等的监督、英国仍会继续参与欧盟关税同盟与单一市场。
 <br/>
 <br/>
 由于这份协议可能面临英国国会激烈反对，欧盟当天称通过的协议是最好的版本，暗示不会再有让步或重新谈判，呼吁英国民众支持脱欧协议。
 <br/>
 <br/>
 由于这份协议在英国掀起巨大反对声浪，英国首相特蕾莎．梅目前最大的难关不在布鲁塞尔而在伦敦，如何说服英国国会支持成为各界瞩目焦点。她周日在记者会上表示，今天与欧盟达成的脱欧协议就是唯一可行方案，未来几周内英国将展开重大辩论，她也呼吁国会能够批准。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁)
