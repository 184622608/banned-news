### 英媒称美暂缓对华征税 美方回应：计划不变
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 在即，美中拟重启磋商，双方互动频繁。日前，英媒称，有消息人士透露，美方将暂缓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对华征税.html" target="_blank">
  对华征税
 </a>
 。但美国贸易代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莱特希泽.html" target="_blank">
  莱特希泽
 </a>
 随即予以否认，并表示课税计划不变。另有消息称，北京已向美国提出让步清单，不过内容似乎仍是〝老调重谈〞，预计川习会晤不会使贸易争端有太大突破。
 <br/>
 <br/>
 路透社11月15日报导称，美国贸易代表办公室一位发言人在一份声明中表示：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莱特希泽.html" target="_blank">
  莱特希泽
 </a>
 代表并没有对行业高管说过，未来的301条款关税行动暂停〞。
 <br/>
 <br/>
 在此之前，英国《金融时报》引述消息人士透露，美国贸易代表莱特希泽告诉企业高管，由于美中双方准备重启磋商，对2670亿中国商品征收新一轮关税的行动暂停。
 <br/>
 <br/>
 不过，莱特希泽于周四否认了这则报导所言。上述发言人表示〝这项关税计划...没有任何改变。任何与之相反的报导均是不正确的。〞
 <br/>
 <br/>
 迄今，美国已向2,500亿美元中国输美商品加征关税，川普（特朗普）总统曾表示，若北京不能按照美国的要求进行公平贸易，美国将对另外2,670亿美元中国商品征税。
 <br/>
 <br/>
 近日，美中贸易争端出现新的情况，川普和习近平将在11月末的阿根廷G20峰会期间，举行会晤，就美中贸易关系重新定调。在此之前，美中双方将重启磋商，中共国务院副总理刘鹤近日将赴美进行新一轮贸易谈判。
 <br/>
 <br/>
 与此同时，中方为此次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 谈做了准备工作，已就美方的要求提交一份书面回应清单。路透社14日报导称，这是贸易战开打以来，中方首次这么做。但报导未透露清单更多细节。
 <br/>
 <br/>
 彭博社15日援引3名消息人士称，中方提交的清单仍是老调重谈，只不过是在重申之前的承诺，如提高外资持股比例等等。但核心的贸易手段、政策如〝中国制造2025〞等议题，并没有包含在内。
 <br/>
 <br/>
 其中的2名消息人士说，中方的承诺并没有达到川普政府要求〝重大结构改革〞的标准，仅是中方自以为〝有在进行建设性谈话〞的象征。
 <br/>
 <br/>
 15日，川普政府一位高级官员对路透社说，中方对美国贸易改革要求的书面回应，不太可能导致川习会谈取得突破进展。预计美中双方的谈判之路还很漫长。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
