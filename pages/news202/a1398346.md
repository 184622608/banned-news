### 巴国总理访华 电视台字幕误把〝北京〞写〝乞讨〞
------------------------

<div class="wysiwyg">
 【新唐人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 时间2018年11月07日讯】巴基斯坦国家电视台近日在报导该国总理伊姆兰‧汗（Imran Khan）访华的新闻时，打出的字幕中把〝北京〞误写为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乞讨.html" target="_blank">
  乞讨
 </a>
 〞，引起舆论轰动。为此，该媒体日前向中方道歉，并将电视台的3名职员停职。
 <br/>
 <br/>
 因近日播出的伊姆兰．汗访华报导中出现字幕错字的事故，巴基斯坦国家电视台（Pakistan Television Corporation, PTV）日前向中国进行了公开道歉。
 <br/>
 <br/>
 据了解，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_字幕事故.html" target="_blank">
  字幕事故
 </a>
 发生于11月4日。当时新就任巴基斯坦任总理的伊姆兰‧汗正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 访问，为解决巴国国内的经济危机向北京当局寻求数十亿元的经济援助。
 <br/>
 <br/>
 巴基斯坦电视台（PTV）在进行直播报导时，打出的背景字幕中错把〝北京〞（Beijing）写成了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乞讨.html" target="_blank">
  乞讨
 </a>
 〞（Begging），为时20秒，随即被中国网民发现。网友将电视画面截图后在社交媒体上公开，直指画面〝是有史以来最滑稽讽刺的事情〞。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-7/p9114841a162090265.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-7/p9114841a162090265-ss.jpg"/>
   <br/>
   <font size="-1">
    巴基斯坦国家电视台近日在报导该国总理伊姆兰‧汗（Imran Khan）访华的新闻时，打出的字幕中把〝北京〞误写为〝乞讨〞，引起舆论轰动。（电视画面截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 这个错误主要是因为〝北京〞与〝乞讨〞的英文单词的拼写和发音都十分相似所致，原本只是普通的文字排版错误，但因为这次伊姆兰‧汗访华的主要目的恰恰是寻求资金援助的，难免令人容易联想到〝乞讨〞之意，因而引发舆论的轰动效应。
 <br/>
 <br/>
 有网友在推特上调侃称，〝不需要道歉，巴基斯坦总理其实就是去讨钱的。〞网友Rajan Sharma‏也留言称：〝展现了巴基斯坦的现状。〞
 <br/>
 <br/>
 为此，巴国电视台已于11月5日在Twitter的官方账号上发帖，对这次的字幕排版失误表示遗憾，并宣布已将3名对该事件负有责任的工作人员停职。
 <br/>
 <br/>
 据公开的资讯，巴基斯坦近年陷入财政危机，上个月该国政府已获得沙特阿拉伯提供52亿欧元的资金援助，但仍然不足以解决这场经济危机，于是向北京当局和国际货币基金组织（IMF）寻求紧急援助。
 <br/>
 <br/>
 不过，据英国BBC的报导，伊姆兰．汗此次访华不仅仅是为寻求资金解决债务危机，与中方讨论如何促进中巴贸易〝更加平衡〞 也是伊姆兰．汗此访的重要诉求。其中，中巴经济走廊（CPEC）和巴基斯坦债务危机是外界关注的焦点。
 <br/>
 <br/>
 伊姆兰．汗在竞选时，对巴本届政府和中国的经济合作持批判态度，并称那些与中国企业合作的项目涉嫌腐败，包括投资高达600亿美元的CPEC框架下的一些合作项目也存在问题。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：戴明）
