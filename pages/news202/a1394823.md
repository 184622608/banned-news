### 暴力黑帮〝山口组〞遭美制裁  日警再强行搜索总部
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】美国财政部2日宣布对日本最大规模
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑帮.html" target="_blank">
  黑帮
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山口组.html" target="_blank">
  山口组
 </a>
 〞旗下4名成员及企业进行制裁后，其次级团体〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中岛组.html" target="_blank">
  中岛组
 </a>
 〞干部在9日因涉嫌诈欺，遭日本警视厅派出京都、和歌山、高知等地警力搜索。
 <br/>
 <br/>
 据日媒《朝日新闻》报导，今年2月疑似有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山口组.html" target="_blank">
  山口组
 </a>
 次级团体干部伪装成银行职员，以退还医疗税等藉口，打电话给年长的老先生，诈骗150万日圆（约13,290美元）。
 <br/>
 <br/>
 经警视厅一番调查后，警方派出京都、和歌山、高知等地警力，前往位于神户市的总部进行搜索，并于10月3日逮捕山口组次级团体〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中岛组.html" target="_blank">
  中岛组
 </a>
 〞的藤井幸治及其他7名成员，9日通缉共犯嫌疑通缉29岁的伊藤欲弥。
 <br/>

 报导说，衽2017年10月至2018年5月这段期间，嫌犯藤井等人即涉嫌至少85起案件，金额更是高达1亿日圆（约88.6万美元），从20多名涉案人员口供中得知，诈骗金额多流向中岛组干部。
 <br/>
 <br/>
 警方表示，日本诈骗案之所以与日俱增，主要是受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑帮.html" target="_blank">
  黑帮
 </a>
 在幕后撑腰，提供庞大资金来源影响，但他们组织严谨，要找到主谋者不容易，为突破现况，日本警察厅开始加强对黑帮、准黑帮及外籍人士成立的犯罪组织的取缔。
 <br/>
 <br/>
 据了解，美国财政部2日首次对山口组相关企业实施管控，宣布对山口组旗下4名成员，及山辉不动产、东洋信用时业进行经济制裁。除冻结他们在美国内部拥有的资产外，包括跨足贩卖美术品、高尔夫练球场营运等领域，同时禁止其与个人、企业接触。
 <br/>
 <br/>
 美国财政部辖下专门负责打击洗钱及金融犯罪部门的副部长曼德尔科（Sigal Mandelker）表示，〝美国下一步会瞄准和本次制裁目标的合作对象，以破坏犯罪组织在全球的金融体系。〞清查看似合法的公司，背后是否有非法运送武器等情形。
 <br/>
