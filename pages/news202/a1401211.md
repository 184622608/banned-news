### 时事拼盘：张家口大爆炸22死 狮航空难初步调查有结果
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】下面请看一组新闻简讯
 <br/>
 <br/>
 周三，中国河北省张家口市盛华化工厂附近发生爆炸，已知22人死亡，另有至少22人受伤，大约50辆汽车起火。化工厂已关闭，该厂一名员工表示，爆炸不是发生在化工厂内。
 <br/>
 <br/>
 造成189人罹难的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼空难.html" target="_blank">
  印尼空难
 </a>
 ，初步调查结果周三出炉。客机失事原因是，防失速自动系统出现差错，在正常状况下不断压低飞机的机鼻，驾驶则拚命想将机鼻拉起来，但最后客机还是坠海。该飞机在前一天也出现相同问题，当时驾驶员关掉自动系统，安全降落。
 <br/>
 <br/>
 被命名为〝戴安娜〞的风暴周三袭击了英国和爱尔兰，风力每小时70英里，伴随有大雨。英国气象局对爱尔兰海附近区域发布黄色警报，一直到周三午夜才解除。部分地区还发布了洪水警告。
 <br/>
 <br/>
 身高可以救命。澳大利亚一头名为〝短裤〞的巨型牛身高6.4英尺，体重3086磅。因为身材过于魁梧，〝短裤〞无法进入牛肉加工设施，它因此而逃过屠刀，得以回到农场度过余生。〝短裤〞被认为是澳大利亚最大的牛。
