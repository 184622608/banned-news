### 模拟战争爆发 美军罕见40战机赴英同时演习
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】欧洲地缘政治局势日渐紧张，美国近期继派出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战机.html" target="_blank">
  战机
 </a>
 参加在乌克兰的多国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演习.html" target="_blank">
  演习
 </a>
 后，9日起也罕见的在英国举行大型内部演练空战及对地攻击技术。指挥官表示，演习是模拟战争爆发时的作战能力。
 <br/>
 <br/>
 综合媒体报导，这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演习.html" target="_blank">
  演习
 </a>
 属于〝异机种空战训练〞（DACT），包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空军.html" target="_blank">
  空军
 </a>
 及海军派出40架不同型号的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战机.html" target="_blank">
  战机
 </a>
 ，在北海上空模拟空战，从而增强空军及海军战机机师的作战能力。
 <br/>
 <br/>
 演习为期两周，由驻守英国莱肯希思空军基地的第48战机联队主办，参加的战机分别为美军现役的第4代及第5代战机，包括驻守当地的空军F-15C与F-15E战机，以及刚完成在中东部署、5日才飞抵英国短暂停留的6架F-22隐形战机，另有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜鲁门号航空母舰.html" target="_blank">
  杜鲁门号航空母舰
 </a>
 （USS Harry S. Truman CVN-75）8架F/A-18E/F战机。
 <br/>
 <br/>
 另外，美军出动空中加油机支援参演的战机，北约亦派出一架空中预警机负责指挥及控制事宜。
 <br/>
 <br/>
 一名参加演习的F-22飞行官表示，这次演习是相辅相成。他说：〝每款战机在某方面都有其优胜的地方，因此演习有助我们了解每款战机的优点及缺点。〞
 <br/>
 <br/>
 隶属第48战机联队的第492战机中队指挥官伍滕（William Wooten）中校形容让军方多款战机在英国或欧洲同时参演，实属非常罕有情况。
 <br/>
 <br/>
 伍滕表示：〝我们进行大型演习以训练
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空对空.html" target="_blank">
  空对空
 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空对地.html" target="_blank">
  空对地
 </a>
 技术，模拟我们参战。〞
 <br/>
 <br/>

 <br/>
 有分析认为这次演习针对俄罗斯的意味极浓。据了解，北约将于10月25日至11月7日在挪威及附近地区举行〝三叉戟结合18〞演习，共有31个国家派出4.5万名士兵、150架军机、60艘军舰及1万多辆军车，包括美军杜鲁门号航母战斗群。这是自前苏联解体后最大规模的军演。
 <br/>
 <br/>
 对此，俄罗斯总统普京于11日突然下令举行战略核力量演习，出动战略核潜艇及战略轰炸机参与，并发射可携带核弹头的潜射弹道导弹及巡航导弹。
