### 撤销特赦 秘鲁法院下令逮捕前总统藤森
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘鲁.html" target="_blank">
  秘鲁
 </a>
 法院周三（3日）撤销对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前总统.html" target="_blank">
  前总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_藤森谦也.html" target="_blank">
  藤森谦也
 </a>
 的特赦，并下令立即将其逮捕。藤森去年12月才获秘鲁现任总统库琴斯基以健康情况转差为由特赦。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘鲁.html" target="_blank">
  秘鲁
 </a>
 司法部周三在发表声明称，最高法院已〝下令逮捕并拘留
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前总统.html" target="_blank">
  前总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_藤森谦也.html" target="_blank">
  藤森谦也
 </a>
 （Alberto Fujimori），因此他可能会再度入狱服刑。〞
 <br/>
 <br/>
 路透社报导，80岁的藤森的律师培瑞斯（Miguel Perez）随即提出上诉，寻求暂缓执行逮捕令。培瑞斯称，藤森有心脏病，如果再次将他送回牢中的话，可能会有生命危险。
 <br/>
 <br/>
 藤森的女儿藤森惠子（Keiko Fujimori）周三称，她认为法院这项决定不人道、不公平。
 <br/>
 <br/>
 藤森惠子是秘鲁主要反对党人民力量党（Popular Force）领袖，被视为藤森政治王朝的接班人。
 <br/>
 <br/>
 藤森是日本裔秘鲁农业学家，曾经在1990年到2000年期间担任秘鲁总统。2009年4月7日，秘鲁特别法庭以参与绑架杀人案与〝践踏人权罪〞判处藤森25年监禁。
 <br/>
 <br/>
 但由于健康恶化，在狱中服刑12年后于2017年12月获现任总统特赦。
 <br/>
 <br/>
 藤森在秘鲁颇具争议，反对者认为他是一个腐败又践踏人权的独裁者，支持者则认为在他执政期间，促进了经济发展，严厉打击和瓦解了〝光辉道路〞等左派恐怖组织。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
</div>
