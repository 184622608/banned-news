### 热带气旋席卷印度东岸 酿33死近18万人急撤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】致命
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热带气旋.html" target="_blank">
  热带气旋
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加亚.html" target="_blank">
  加亚
 </a>
 （Gaja）16日席卷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度.html" target="_blank">
  印度
 </a>
 东岸，将数以千计树木连根拔起并摧毁房屋，造成33死，十余万人被迫逃离家园暂住避难中心。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加亚.html" target="_blank">
  加亚
 </a>
 挟带时速120公里的强风于16日登陆后，急速抵达坦米尔那都省（Tamil Nadu），所经之处灾情惨重，数千棵树遭连根拔起，饲养的家禽牲畜也难幸免。
 <br/>
 <br/>
 灾难管理局（Disaster Management Authority）官员表示：〝因气旋袭击，目前有20位男性、11位女性以及2名孩童死亡。〞另有17万7500人暂居351个营地。多数死亡是由洪患、房屋倒塌及触电导致。

 随着整体灾情逐渐明朗，数百名紧急救难人员前往修复道路及电缆。美国海军一架直升机及两艘舰艇也加入坦省救灾工作。
 <br/>
 <br/>
 坦米尔那都省省长帕兰尼司瓦米（Edappadi Palaniswami）宣布，罹难者家属将可获得100万卢比（约13800美元）抚慰金。
 <br/>
 <br/>
 印度气象部门表示，加亚向西进入克勒拉省（Kerala）后，继续向阿拉伯海前进。
 <br/>
 <br/>
 印度东岸10月曾遭热带气旋蒂特利（Titli）侵袭，在奥里萨省（Odisha），造成至少2人死亡。去年，热带气旋奥克希(Cyclone Ockhi)侵袭坦米尔那都和克勒拉两省，造成将近250人死亡。
