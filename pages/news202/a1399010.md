### 挤铁路陆桥看表演 印尼群众推挤坠地3死20伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】印尼第二大城
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泗水.html" target="_blank">
  泗水
 </a>
 9日举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_建国英雄日.html" target="_blank">
  建国英雄日
 </a>
 的户外表演活动，吸引当地民众到场围观。但有民众爬上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_铁路陆桥.html" target="_blank">
  铁路陆桥
 </a>
 上居高看表演，一列火车放慢速度行经该路段，由于太多人挤在狭窄的空间，导致多人摔落地面，造成3人死亡，逾20人受伤。泗水警局还在调查这起案件。
 <br/>
 <br/>
 雅加达椰子网（Coconuts Jakarta）报导，意外发生在上周五晚间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泗水.html" target="_blank">
  泗水
 </a>
 （Surabaya）当地庆祝年度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_建国英雄日.html" target="_blank">
  建国英雄日
 </a>
 ，在历史上的泗水战场一带举行户外表演活动。
 <br/>
 <br/>
 表演现场挤满人潮，许多民众还爬上附近的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_铁路陆桥.html" target="_blank">
  铁路陆桥
 </a>
 ，希望有更好的位置观看表演。但悲剧发生了，从视频中可看到，一列火车以缓慢速度行经该路段，但陆桥上挤满民众，在民众纷纷走避及推挤下，许多人从6公尺高的陆桥跌落，造成3人死亡，至少20人受伤。
 <br/>
 <br/>
 死者分别是9岁的儿童和两名16岁及17岁的青少年。
 <br/>
 <br/>
 <b>
  相关视频：
 </b>
 <center>
 </center>
 <br/>
 事件发生后，泗水市长丽斯玛（Tri Rismaharini）表示，因这场表演未获得泗水市政府批准，死者不会获得赔偿金，对于受伤的民众，市政府会补助医疗费用。
 <br/>
 <br/>
 不过，表演活动主办单位则称，他们事先已获得泗水警局的表演批准。最后，东爪哇省政府同意提供所有伤亡者补偿金。
 <br/>
 <br/>
 印尼国营的铁路公司（PT Kereta Api）泗水分部发言人卡图特（Gatut Sutiyatmoko）表示，外界可能会责怪火车驾驶，为什么不将火车停下来。
 <br/>
 <br/>
 他说，火车经过陆桥的正常时速是每小时30公里，意外发生前，火车驾驶试图减速至时速20公里，且法律也禁止任何人闯入铁路。
 <br/>
 <br/>
