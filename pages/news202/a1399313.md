### 东盟峰会第二天 RCEP协议谈判推迟
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东盟峰会.html" target="_blank">
  东盟峰会
 </a>
 期间，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 会见印度总理穆迪，强调两国在亚太事务中的作用。东盟峰会的另一重要议题，2012年开始谈判的〝区域全面经济伙伴关系协定〞，被推迟到明年。
 <br/>
 <br/>
 周三，新加坡总理李显龙表示，涵盖16个国家的〝区域全面经济伙伴协定〞谈判，延至到明年完成。
 <br/>
 <br/>
 这项协定由东盟十国发起，中国、日本、印度、澳大利亚、韩国和新西兰也参加谈判，经济规模约占全球GDP的三分之一，涵盖全球45%人口。
 <br/>
 <br/>
 正如外界预料，谈判各方分歧严重。协议得到北京推动，但印度担心，对中国企业开放印度市场，会影响其国内企业生存。
 <br/>
 <br/>
 周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东盟峰会.html" target="_blank">
  东盟峰会
 </a>
 期间，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 对印度总理穆迪表示，感谢印度推动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜无核化.html" target="_blank">
  朝鲜无核化
 </a>
 进程。
 <br/>
 <br/>
 美国副总统彭斯：〝我们感谢印度，坚定支持美国对朝鲜施压。我们期待寻求更多方式合作，（川普）总统将与金正恩主席举行另一次峰会，希望最终为朝鲜半岛带来和平。〞
 <br/>
 <br/>
 周三，日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 表示，日本将推动全球自由公平贸易。他强调，年底生效的〝跨太平洋伙伴关系协定〞，是推动自由贸易进程的重要成就。
 <br/>
 <br/>
 日本首相安倍晋三：〝我们期待达成TPP协定的扩展。〞
 <br/>
 <br/>
 韩国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文在寅.html" target="_blank">
  文在寅
 </a>
 周三正式邀请东盟各国领导人明年访问首尔，纪念韩国和东盟建交30周年。
 <br/>
 <br/>
 韩国总统文在寅：〝我想邀请东盟国家领导人访问首尔，参加2019年东盟—韩国纪念峰会。〞
 <br/>
 <br/>
 东盟峰会期间，文在寅还和俄罗斯总统普京讨论了朝鲜半岛和平进程议题。
 <br/>
 <br/>
 新唐人记者毕心慈、明玉综合报导
</div>
