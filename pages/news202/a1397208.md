### 印尼狮航客机坠海 全机189人生还渺茫
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】印度尼西亚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狮子航空.html" target="_blank">
  狮子航空
 </a>
 客机周一升空后不久坠入海中，机上载有189人。搜救人员在失事海域发现了飞机的残骸碎片以及乘客尸体，目前尚未发现幸存者。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼狮航.html" target="_blank">
  印尼狮航
 </a>
 JT610班机，上午6点20分从雅加达起飞，预计1小时后抵达邦加勿里洞省首府槟港。
 <br/>
 <br/>
 班机起飞13分钟后失联，官方证实，飞机在离海岸约15公里处落海坠毁。
 <br/>
 <br/>
 空难发生后，超过300名搜救人员、军警和志愿者，在坠机现场紧急搜救。
 <br/>
 <br/>
 搜救人员在失事海域找到了飞机残骸碎片、乘客尸体，还有乘客的个人物件，但至今没有发现生还者。
 <br/>
 <br/>
 搭上这架班机的乘客家属都在焦急等待亲人的下落，许多人流着眼泪。
 <br/>
 <br/>
 乘客友人HAERUDIN：〝他是一个好人，很关照这地区的环境。〞
 <br/>
 <br/>
 这可能是印尼1997年以来死伤第二惨重的空难。
 <br/>
 <br/>
 印尼狮航首席执行官西拉伊特表示，失事班机前一段航程出现过技术问题，但已经按照程序立即排除。
 <br/>
 <br/>
 狮航首席执行官西拉伊特：〝确实报告有技术问题，但是我们有按照飞机制造厂的程序解除。〞
 <br/>
 <br/>
 失事客机属于波音737MAX机型，今年8月投入运营。
 <br/>
 <br/>
 波音公司对受害者和家属表达同情与哀悼，并表示随时准备为事故调查提供技术援助。
 <br/>
 <br/>
 新唐人记者李梅、张琪综合报导
