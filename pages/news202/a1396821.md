### 比利时购美F35战机 对北约展示忠诚
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比利时.html" target="_blank">
  比利时
 </a>
 周四表示，将以45.5亿美元的价格购买美国F-35战斗机。比利时放弃购买欧洲颱风战斗机，一个是出于价格和质量考虑，另外也想向美国领导下的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北约.html" target="_blank">
  北约
 </a>
 展现忠诚。
 <br/>
 <br/>
 F-35是世界上最先进的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_隐形战机.html" target="_blank">
  隐形战机
 </a>
 之一。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比利时.html" target="_blank">
  比利时
 </a>
 这次购买的34架，首批交付时间是2023年。此外，还买了无人机、护卫舰、扫雷舰和装甲车。
 <br/>
 <br/>
 此前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北约.html" target="_blank">
  北约
 </a>
 盟国中购买F-35战机的有英国、荷兰、意大利和土耳其。比利时是世界上第13个选择F-35的国家。
 <br/>
 <br/>
 比利时国防部长范德普特说：〝在我们的七个评估标准中，美国战机都是最好的。〞
 <br/>
 <br/>
 比利时政府表示，到2030年，购买F-35的费用、飞行员培训、机库维修等总费用比预算少6.8亿美元。比购买欧洲战机优惠。
 <br/>
 <br/>
 连线本台欧洲记者 王羽音：〝去年北约的军费开支，美国承担了百分之70多。北约秘书长今年还多次表示，在美国总统川普不断要求和领导下，去年北约各成员国军费开支总额增加了400亿美元。但是比利时的军费开支很低，只有本国GDP的0.9％，离北约要求的2%差得很远，离川普要求的3％差得更远。〞
 <br/>
 <br/>
 比利时国防部长范德普特周四在推特上说：〝正在大力投入军事防御。〞
 <br/>
 <br/>
 很多欧洲重要机构，包括欧盟在内，设在比利时首都布鲁塞尔。比利时的这个决定可能会成为其他一些欧洲国家的风向标。
 <br/>
 <br/>
 新唐人电视综合报导
</div>
