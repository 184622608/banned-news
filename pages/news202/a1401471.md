### 乌克兰禁俄男子入境 加强边境演习
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】再来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 危机。俄罗斯扣押乌克兰三艘军舰之后，乌克兰宣布，在戒严期间禁止俄罗斯16岁至60岁男性
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入境.html" target="_blank">
  入境
 </a>
 ，并加强边境军事演习。两国之间紧张形势升级。
 <br/>
 <br/>
 周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌克兰.html" target="_blank">
  乌克兰
 </a>
 宣布，禁止能参与战争的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 男子进入乌克兰，年龄段为16岁到60岁。只有出于人道主义原因，才允许有例外，比如俄罗斯人到乌克兰参加亲戚的葬礼。
 <br/>
 <br/>
 在宣布这一政策时，乌克兰总统波罗申科回顾了2014年俄罗斯吞并克里米亚，以及俄罗斯对乌克兰东部分离份子的支持。他说，阻止俄罗斯全面入侵非常重要。
 <br/>
 <br/>
 有关官员表示，乌克兰也可能对已经
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入境.html" target="_blank">
  入境
 </a>
 的俄罗斯公民施加额外的限制。
 <br/>
 <br/>
 上周末，俄罗斯在克里米亚扣押了三艘乌克兰海军舰艇。隔天，乌克兰宣布实施戒严30天。
 <br/>
 <br/>
 乌克兰总统波洛申科周二警告说，俄罗斯军队在边境地区集结，两国面临爆发全面战争的危险。
 <br/>
 <br/>
 周四，乌克兰军队在亚速海沿岸进行了战术演习。
 <br/>
 <br/>
 俄罗斯扣押乌克兰军舰一事引起国际社会对俄罗斯的谴责。
 <br/>
 <br/>
 一位俄罗斯国会议员对媒体表示，俄国目前没有计划禁止乌克兰男子入境。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
