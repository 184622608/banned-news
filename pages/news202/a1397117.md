### 欧盟国家周日进入冬令时 明年夏时制可能废除
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】法国及欧洲其它大部分国家，从本周日起开始实行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冬令时.html" target="_blank">
  冬令时
 </a>
 ，为期5个月，不过，今年的冬令时也可能是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 国家的最后一次，因为欧盟正在讨论取消夏令时计划。
 <br/>
 <br/>
 和其他
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 居民一样，巴黎人在当地时间10月28日凌晨3点正式结束夏令时，时钟将被回拨1个小时至2点。欧洲各地航班、火车和轮船等交通工具的时刻表也随之变动。
 <br/>
 <br/>
 法国自1976年开始实行夏令时和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冬令时.html" target="_blank">
  冬令时
 </a>
 ，其初衷是更多地利用阳光，相应减少照明用电以节约能源。
 <br/>
 <br/>
 尽管多睡了一小时，一些人说他们已经习惯了冬季，改变时间对他们的日常生活影响很小。
 <br/>
 <br/>
 法国人格拉德说：〝当我们年纪大了，无论钟表显示是什么时间，我们总是同一时间气喘，时间改变对我来说绝对没有影响。〞
 <br/>
 <br/>
 今年的冬季时间变化可能是欧洲的最后一次，因为欧盟委员会上个月宣布了一项废除每年两次变化时间的计划。一些欧盟议员也认为这套节省日光的制度太过时，而且时间的变动将影响人们的身心健康。数百万欧洲人在调查中说，从三月份开始提前一小时，然后在十月再次把时钟拨后，这是一种痛苦。
 <br/>
 <br/>
 在欧盟之外，少数几个欧洲国家已经停止了夏令时和冬令时转换，其中包括俄罗斯、土耳其、白俄罗斯和冰岛。
 <br/>
 <br/>
 欧盟委员会希望每个欧盟国家在2019年4月之前决定他们是否保留冬令时和夏令时。成员国在周一（10月29日）的交通部长在布鲁塞尔举行非正式会议时，第一次有机会讨论这个问题。
 <br/>
 <br/>
 新唐人记者严枫、雨薇综合报导
