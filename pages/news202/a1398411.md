### 川崎市民节庆 共同经营广场
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】位于日本东京湾的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川崎市.html" target="_blank">
  川崎市
 </a>
 ，不仅是京滨工业地带的重镇和交通枢纽，也是保持与市民交流传统的地方。每年11月初的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_市民节.html" target="_blank">
  市民节
 </a>
 ，都会给整座城市带来勃勃生机。
 <br/>
 <br/>
 在这个天高气爽的金秋时节，神奈川县的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川崎市.html" target="_blank">
  川崎市
 </a>
 迎来了第41届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_市民节.html" target="_blank">
  市民节
 </a>
 。在这次市民节上，来自日本全国各地的土特产汇集一堂，川崎市的各社团也带来了精彩的表演节目。
 <br/>
 <br/>
 今年的市民节主题是：〝大家一起来经营广场〞。在市民广场上，除了当地的特产之外，还有来自茶叶之乡──静冈县牧之原出产的新鲜绿茶，北海道的海产品和荞麦面，岩手县花卷市的脆饼点心等等，种类繁多，琳琅满目。
 <br/>
 <br/>
 川崎市经济劳动局产业振兴部观光宣传推进课青井满：〝去年市民节来现场的观光人数有60万人，希望今年来的更多。欢迎大家一起来高高兴兴地参与！〞
 <br/>
 <br/>
 来自川崎市的各社会团体给大家带来了多种表演节目，今年还增加了现场气功体验活动，川崎市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 爱好者协会应邀做了功法演示，伴随着舒缓的音乐，台上与台下一起互动，其乐融融。
 <br/>
 <br/>
 川崎市法轮功爱好者协会小椋：〝为了让更多的日本人了解法轮功，让更多人体验到气功的美妙之处，并使有缘人能够修炼法轮功，我们参加了这次活动。〞
 <br/>
 <br/>
 川崎市法轮功爱好者协会勅使河原：〝随着音乐的响起，自然而然的就聚集了很多观众，表演的同修又都是那么自然，真的很高兴。〞
 <br/>
 <br/>
 除了在市民广场上的活动以外，还有大型的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_游行庆典.html" target="_blank">
  游行庆典
 </a>
 及义卖。
 <br/>
 <br/>
 新唐人日本记者站报导
</div>
