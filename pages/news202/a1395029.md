### 彭斯告诫中美洲诸国：与中共做生意要小心
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 日前在出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲安全与繁荣论坛.html" target="_blank">
  中美洲安全与繁荣论坛
 </a>
 时，告诫中美洲诸国的领袖和外长们，与中共政府建立商业关系时要小心谨慎。
 <br/>
 <br/>
 据香港东网报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 周四（11日）在华盛顿出席了一个讨论中美洲安全与繁荣的论坛。在与萨尔瓦多、危地马拉和洪都拉斯领袖以及墨西哥外长会面时，彭斯谈及中共政府正越来越多的参与中美洲地区事务的现状。他说：〝当你们和中国（中共政府）等其他国家建立商业关系时，我促请你们注意并要求这些关系中的透明度，并好好照顾你们和我们的长远利益。〞
 <br/>
 <br/>
 萨尔瓦多、危地马拉和洪都拉斯被称为中美洲北方三角区。彭斯在于这些国家的领袖谈话时强调，应该加强美国与这些国家之间的关系。他说：〝今日连中国（中共政府）也希望在（中美洲的）北方三角区扩大影响力，我们相信要解决这些问题，是要加强美国与北方三角区以及北半球所有国家的联系，尤其是加强我们和这些国家之间的经贸关系。〞
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-12/p9076521a582473704.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-12/p9076521a582473704-ss.jpg"/>
   <br/>
   <font size="-1">
    美国副总统迈克・彭斯2018年10月11日在华盛顿特区举行的中美洲繁荣与安全会议上发表讲话。(Justin Sullivan/Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 针对彭斯的上述说辞，有舆论认为，美国已经成功与墨西哥、加拿大以及欧盟等达成了一些协议，一个共同反制中共不公平贸易的国际联盟正在形成。彭斯日前对中美洲诸国领袖们的这番提醒与告诫，也可视为美国政府扩大这个〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗中联盟.html" target="_blank">
  抗中联盟
 </a>
 〞外延的一个方面。
 <br/>
 <br/>
 事实上，白宫国家经济会议主席库德洛（Larry Kudlow）今年10月4日曾在华府经济俱乐部举行的一场活动中谈论经贸议题，他再度批评中共政府窃取美国智慧财产权、强迫技术转移、对美产品征收高额关税等作为，并质问说： 〝我们为什么要接受呢？〞
 <br/>
 <br/>
 他还坦言，美国近期与墨西哥、加拿大达成了新版贸易协定，也在与日本、欧盟对话；他认为，美国与这些国家正在迈向形成一个愿意对抗中共不公平贸易行为的〝贸易联盟〞。
 <br/>
 <br/>
 库德洛更进一步指出，如果中国大陆最终能够降低贸易关税并开放市场，美国可以大量出口，这样做中国的普通消费者与业者都会受益。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：东野）
