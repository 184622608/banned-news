### 韩朝边境对峙缓和 朝鲜人权状况仍堪忧
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】11月1日，韩国和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 签署的部分军事协议开始生效，而与此同时，国际人权组织披露了朝鲜官员性侵女性的严重问题。
 <br/>
 <br/>
 周四，上个月韩国与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 签署的禁飞区和禁止在重要边界附近进行军事演习的协议开始生效。该协议希望双边停止一切敌对行为，并移除非军事区内的所有警卫哨所和地雷。但美方指出，禁令可能削弱防御能力。
 <br/>
 <br/>
 此前，美国要求朝鲜充分公开核与导弹设备，一位韩国议员称，韩方已发觉朝鲜准备开放核试验和导弹试验场供国际检查。美国国务卿并计划下周访问朝鲜。
 <br/>
 <br/>
 但朝鲜媒体周四报导，金正恩在考察一处度假村工地时又发牢骚，指责所谓国际〝敌对势力〞及其制裁，强调朝鲜发展经济要自食其力。
 <br/>
 <br/>
 而在同一天，朝鲜官员性侵女性的问题也受到关注。
 <br/>
 <br/>
 在搜集了超过60名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜女性.html" target="_blank">
  朝鲜女性
 </a>
 的反馈后，人权观察指出，其中一些女性甚至不懂，强制性行为不该成为生活的一部分。
 <br/>
 <br/>
 人权观察执行总监 Kenneth Roth：很多朝鲜人告诉人权观察，当（朝鲜）官员或有权的人选上了一个女人，她别无选择必须服从。她必须做任何对方所要求的，不管是不是跟性、钱或其它官员想做的事。
 <br/>
 <br/>
 人权观察认为，国际社会在改善与朝关系时，却对其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权状况.html" target="_blank">
  人权状况
 </a>
 有所忽略。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
