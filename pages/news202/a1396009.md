### 沙特遇害记者背景复杂 曾是本拉登密友
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特记者.html" target="_blank">
  沙特记者
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 日前在沙特驻土耳其领事馆〝失踪〞事件，引发国际舆论关注。有报导指，卡舒吉的身世背景相当复杂，他曾是基地恐怖组织头目本拉登的密友。据信，卡舒吉手中握有9．11
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恐怖袭击.html" target="_blank">
  恐怖袭击
 </a>
 事件的一些内幕。
 <br/>
 <br/>
 沙特阿拉伯记者，现年59岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 于10月2日进入沙特领馆后神秘失踪。16日，土耳其警方对沙特驻土耳其领事馆进行搜查。随后，有土耳其官员向外界透露，卡舒吉两周前在沙特领馆被杀，尸体被肢解。
 <br/>
 <br/>
 卡舒吉的官方身份是一名〝记者〞，他在推特（Twitter）上拥有将近200万粉丝，是世界知名的政治评论家之一。不过，翻查卡舒吉的个人信息，从其家族背景来看，他并非一名普通的〝记者〞。
 <br/>
 <br/>
 卡舒吉来自土耳其卡舒吉（kaşıkçı）家族，其祖父穆罕默德．卡舒吉（Muhammad Khashoggi）是现代沙特开国国王阿卜杜拉．阿齐兹．伊本．沙特（Abdullah Bin Abdul-Aziz）的私人医生。他的叔叔阿德南（Adnan Khashoggi）是世界闻名的亿万富翁军火商。
 <br/>
 <br/>
 卡舒吉在沙特王室有着广泛的人脉，《纽约时报》称，〝过去三十年，只要是和沙特有关的人士他好像都认识。〞
 <br/>
 <br/>
 最为外界关注的是，卡舒吉还是基地恐怖组织头目本．拉登（Osama bin Laden）的密友，两人的关系非同一般。他长年和本．拉登保持接触，对其抱有极大兴趣和好感，并发表过对本．拉登的一系列采访报导。
 <br/>
 <br/>
 2011年，本拉登死后，卡舒吉曾在推特上写道：〝我瘫倒而泣，我的心碎了。〞
 <br/>
 <br/>
 本．拉登发起美国911
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恐怖袭击.html" target="_blank">
  恐怖袭击
 </a>
 事件。2001年9月11日早晨（美国东部时间），基地恐怖组织成员劫持美国4架民航客机，其中2架撞向纽约世界贸易中心双塔，1架撞向了五角大楼，另1架因机舱内乘客与劫机歹徒搏斗，最终在宾夕法尼亚州坠毁。此次恐怖袭击事件导致近3000人死亡，震惊世界。
 <br/>
 <br/>
 911恐怖袭击事件发生后，美国政府对本．拉登与基地组织进行大追捕。美国当地时间2011年5月2日凌晨，美国政府宣布，本．拉登当晚在巴基斯坦西北边境，阿伯塔巴德的住宅内被击毙。
 <br/>
 <br/>
 另外值得一提的是，很多沙特媒体都曾指出，卡舒吉与沙特情报部门有着错综复杂的关系。他当时受沙特情报机构委托，试图劝服本．拉登与沙特王室修好。
 <br/>
 <br/>
 据沙特媒体描述，沙特亲王费萨尔（Turki Al Faisal）1977年到2001年担任沙特情报部长的24年间，卡舒吉一直在为费萨尔工作。
 <br/>
 <br/>
 费萨尔于2005年担任沙特驻美大使后，卡舒吉作为费萨尔的官方助手也一同前往美国。在9．11事件发生10天前，2001年9月1日，费萨尔从情报局辞职。
 <br/>
 <br/>
 外界据此推测，卡舒吉很可能是唯一掌握沙特王室与基地组织〝在9．11袭击中相互勾结证据的非王室成员〞。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
