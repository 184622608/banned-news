### 法国政府加税致油价涨 28万人上街抗议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】在法国，因燃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 格上涨而引发的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄背心.html" target="_blank">
  黄背心
 </a>
 〞示威活动持续到周日，已造成400多人受伤，截至记者发稿，抗议活动仍未平息。
 <br/>
 <br/>
 法国的柴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 格在过去一年来已经上涨大约23%，汽油价格也上涨了17.6％。
 <br/>
 <br/>
 法国多地区因此于近日爆发抗议，示威者们身穿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄背心.html" target="_blank">
  黄背心
 </a>
 ，高喊〝马克龙下台〞，以抗议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国政府.html" target="_blank">
  法国政府
 </a>
 加征油税导致油价上涨。有地区爆发警民冲突，警员向示威人群发射催泪弹，并筑起路障。
 <br/>
 <br/>
 抗议活动已造成1人死亡、400多人受伤，其中14人受重伤。在法国各地超过2000个地方皆举行了抗议活动，有157人被拘押。
 <br/>
 <br/>
 抗议者们指出，政府藉着环保、绿能、反污染等理由，不断加征燃料税，而明年一月再次宣布的调涨，成为导致抗议爆发的最后一根稻草。
 <br/>
 <br/>
 法媒报导，目前，总统马克龙的民意支持率已经跌落至25%。
 <br/>
 <br/>
 新唐人电视王愉贺综合报导
