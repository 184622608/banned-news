### 中驻巴使馆遇袭 再显〝一带一路〞负面影响
------------------------

 【新唐人2018年11月24日讯】周五，中共在

  巴基斯坦
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡拉奇.html" target="_blank">

 <br/>
 <br/>
 当地时间周五上午9：15，3名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_袭击.html" target="_blank">
  袭击
 </a>
 者试图进入位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡拉奇.html" target="_blank">
  卡拉奇
 </a>
 的中国驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴基斯坦.html" target="_blank">
  巴基斯坦
 </a>
 使领馆，被当地警察阻止，至少5人死亡、包括2名警察和3名袭击者。
 <br/>
 <br/>
 巴基斯坦总理伊姆兰・汗：〝中国领事馆没有人员伤亡。我们的安保人员迅速抵达现场制服了袭击者。〞
 <br/>
 <br/>
 巴基斯坦〝俾路支解放武装〞组织通过社交媒体声称对袭击负责，但巴政府还未就袭击者身份发布声明。
 <br/>
 <br/>
 〝俾路支解放武装〞组织基于巴基斯坦俾路支省，主张俾路支独立。他们认为中共对巴的投资是在〝侵略〞巴基斯坦。今年8月就有3名参与〝

  一带一路
