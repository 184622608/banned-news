### 韩国第4位前总统被判刑 李明博获刑15年罚130亿韩元
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】韩国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李明博.html" target="_blank">
  李明博
 </a>
 遭控受贿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贪污.html" target="_blank">
  贪污
 </a>
 共16项罪名，法院今(5)日针对此案做一审判决，李明博被判处15年徒刑，罚锾130亿韩元（约1235万美元），全案仍可上诉。李明博成为韩国史上因贪腐被逮并接受审判的第4位前总统。
 <br/>
 <br/>
 据《韩联社》报导，现年76岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李明博.html" target="_blank">
  李明博
 </a>
 (Lee Myung-bak)于2008年至2013年担任韩国总统期间，涉嫌收受贿赂、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贪污.html" target="_blank">
  贪污
 </a>
 、渎职、逃漏税、滥权、背信及违反选罢法。今年3月22日被关押，4月初被首尔中央地检署以16项罪名正式起诉，检方上月求刑20年。
 <br/>
 <br/>
 法院今(5)日对其进行宣判，过程均透过电视直播，不过李明博以健康问题和不满直播而缺席庭审。
 <br/>
 <br/>
 李明博16项指控中有多项罪名成立，包括涉贿约111亿韩元（约1054.5万美元）、来自国家情报院的7亿韩元（约66.5万美元）、三星公司垫付的585万美元诉讼费，韩企Daebo集团贿赂的5亿韩元（约47.5万美元）。
 <br/>
 <br/>
 李明博成为韩国宪政史上第四名被提请批捕的前总统。此前遭批捕的前总统包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢泰愚.html" target="_blank">
  卢泰愚
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全斗焕.html" target="_blank">
  全斗焕
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朴槿惠.html" target="_blank">
  朴槿惠
 </a>
 。
 <br/>
 <br/>
 1941年出生的李明博，出身贫苦家庭，通过自身努力考入名校，曾于现代集团工作近30年，由一名普通职员努力成为企业高管。随后，历任国会议员、首尔市长等职，2008年至2013年出任韩国总统，被韩国人奉为〝工薪族的神话〞。
 <br/>
 <br/>
