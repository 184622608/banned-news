### 伊朗奇特卷入贸易战 疯狂囤积美大豆引围观
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】中美贸易战近期出现了奇怪模式，中国进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 量剧减，美国另一个对手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 却开始大举进口美国大豆。目前无法确定伊朗如何处理这些本不需要的大豆。但此前专家分析，中国市场缺口巨大，或从第三国转运美国大豆。
 <br/>
 <br/>
 据《华尔街日报》11月9日报导说，中国已不再是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 的最大进口国，但美国至少还有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 这个大客户。
 <br/>
 <br/>
 报导说，中美贸易冲突激化之后，中国大幅削减美国大豆购买量，今年9月仅进口了2,400万美元的美国大豆，而去年9月的进口量是11亿美元。
 <br/>
 <br/>
 不过，随着美国重启对伊朗的制裁，原本对大豆没有多少需求的伊朗，突然开始趁美国大豆价格走低而大举囤货。
 <br/>
 <br/>
 报导引述美国农业部的数据指，伊朗今年上半年根本没有购买美国大豆，但在8月突然进口了价值1.4亿美元的美国大豆，成为当月美国大豆的最大外国买家。9月，伊朗又购买了4,400万美元的美国大豆。
 <br/>
 <br/>
 法媒报导说，伊朗此举很让人惊讶。伊朗自身不能消费这么多大豆，是否准备出手，以及是否转运中国，引起外界关注。
 <br/>
 <br/>
 值得注意的是，和许多美国口中的〝流氓国家〞一样，伊朗始终和中共保持密切的合作关系。美国此前威胁11月5日后开始制裁进口伊朗石油的第三国公司，但中国两大国企依然坚持进口伊朗石油到〝最后一刻〞。
 <br/>
 <br/>
 中共开始对美国大豆加征关税后，始终宣称可以通过增购巴西等国的大豆，填补美国大豆的市场份额。
 <br/>
 <br/>
 不过，目前已进入南半球大豆供应的淡季。专家普遍认为，中方将不得不继续进口美国大豆，以弥补大豆需求的巨大缺口。而通过第三国转运，就是一个既能规避关税、又不失面子的操作手法。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
