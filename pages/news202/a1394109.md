### 美媒惊曝 美国主板遭中共植入恶意芯片
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】美国彭博商业周刊，4号发表长篇调查报告，曝光
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共军方.html" target="_blank">
  中共军方
 </a>
 透过中国工厂，在为美国代工的主板中植入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恶意芯片.html" target="_blank">
  恶意芯片
 </a>
 。目前已知近30家公司受到波及，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果公司.html" target="_blank">
  苹果公司
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 。
 <br/>
 <br/>
 彭博商周：〝这是一种极度危险的攻击。〞
 <br/>
 <br/>
 彭博商业周刊报导，2015年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 为并购视频处理公司Elemental，而对其进行安全评估，意外发现Elemental服务器的主板上，比原始设计多出一块微芯片。只有一粒米大小，具有内存、连网和处理能力。
 <br/>
 <br/>
 亚马逊向美国政府报告后，引起情报界的震动，随即展开长达3年的绝密调查。最终确认，Elemental的服务器，由美国超微（Supermicro）公司组装；而超微公司的主板，几乎全部在中国制造。
 <br/>
 <br/>
 美国调查人员监听发现，中共政府透过中间人，联系了超微在中国的4家制造分包商，贿赂工厂经理，并威胁不配合就关厂。迫使工厂同意，把政府提供的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恶意芯片.html" target="_blank">
  恶意芯片
 </a>
 植入超微主板。
 <br/>
 <br/>
 这些芯片，由中共军队一个专门负责硬件攻击的部门，设计制造。体积极小，外观又伪装成普通组件，因此很难被发现。
 <br/>
 <br/>
 这是美国目前已知最严重的，从供应链源头直接窜改硬件的事件。
 <br/>
 <br/>
 张健：〝硬体攻击可以摧毁掉所有网络指挥系统，也可以监控、窜改指令。我认为，硬体上的监控，是我们所能想到所有的威胁手段，最严重的一种，也是危害最深的一种。民间个人是无法做到这个事情的，一般也不会敢于做这个事情。〞
 <br/>
 <br/>
 目前，已发现近30家公司受到恶意芯片的影响，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果公司.html" target="_blank">
  苹果公司
 </a>
 、亚马逊、一家大型银行和美国政府承包商。
 <br/>
 <br/>
 苹果和亚马逊都否认受到影响。但苹果和亚马逊的资深员工告诉彭博商周，公司已经或准备低调撤换含恶意芯片的服务器。
 <br/>
 <br/>
 超微也否认自己植入恶意芯片。一名美国情报官员说，超微是全球最大的主板供应商，在100多个国家有900多个客户，攻击超微将影响全世界。
 <br/>
 <br/>
 张健：〝硬体攻击查起来非常难，也就像亚马逊这样一个商业和科技巨擘可以做到这样的侦查手段，换作其他公司，你想一想美国有多少高科技公司可能面临这样的问题。〞
 <br/>
 <br/>
 彭博商周说，中共的目标，是长期获取高价值的企业机密，和敏感的政府网络资讯。
 <br/>
 <br/>
 全球75%的手机和90%的个人电脑，是在中国生产的。彭博商周报导说，这次发现的重大安全隐患，可能促使企业将生产链撤离中国。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
