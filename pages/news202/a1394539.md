### 诺贝尔经济学奖揭晓 美国学者获奖
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】周一，2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诺贝尔经济学奖.html" target="_blank">
  诺贝尔经济学奖
 </a>
 揭晓。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国学者.html" target="_blank">
  美国学者
 </a>
 威廉・诺德豪斯和保罗・罗默，因为将气候变化和技术创新结合到经济分析领域，获得殊荣。
 <br/>
 <br/>
 诺贝尔评委认为，两位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国学者.html" target="_blank">
  美国学者
 </a>
 的研究，对全球经济可持续增长作出贡献。
 <br/>
 <br/>
 诺贝尔评委 GORAN K HANSSON：〝（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诺贝尔经济学奖.html" target="_blank">
  诺贝尔经济学奖
 </a>
 ）一半授予威廉・诺德豪斯，表彰他将气候变化结合到长期经济分析领域的成就。另一半授予保罗・罗默，表彰他将技术创新结合到长期宏观经济分析的贡献。〞
 <br/>
 <br/>
 保罗・罗默是纽约大学史登商学院教授，曾担任世界银行首席经济学家。他的研究阐述了经济力量如何使企业产生新想法和创新意愿，为所谓的〝内生增长理论〞提供了基础。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诺贝尔奖.html" target="_blank">
  诺贝尔奖
 </a>
 官方推特发文说：〝保罗・罗默的研究表明，思想积累如何维持长期经济增长。〞
 <br/>
 <br/>
 罗默接受电话采访，说他没有想到会获奖。
 <br/>
 <br/>
 诺贝尔经济学奖获得者保罗・罗默：〝今天早上有两个电话，我都没接，我以为是诈骗电话，我没想到会获奖。〞
 <br/>
 <br/>
 威廉・诺德豪斯是耶鲁大学经济学教授，美国经济学家。他创建了可描述经济与气候相互作用的定量模型。
 <br/>
 <br/>
 诺贝尔经济学奖评奖委员会认为，这两项研究有助于人们了解经济增长及其附带问题。
 <br/>
 <br/>
 诺贝尔经济学奖评委会主席 PER STROMBERG：〝一个是如何不断有新想法和新发明，这对长期繁荣很重要，第二个是如何解决经济发展对气候的负面影响。〞
 <br/>
 <br/>
 今年是诺贝尔经济学奖设立50周年。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
</div>
