### 不满马克龙教育改革 法国学生持续抗议
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】周二，法国的巴黎、波尔多、图卢兹等多个城市的大学生和高中生，持续
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_教育改革.html" target="_blank">
  教育改革
 </a>
 ，其中包括设立新的高校选拔制度，以及增加注册费。
 <br/>
 <br/>
 学生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 到周二加剧，全国范围爆发各种抗议事件，示威者点燃建筑物，和警方发生暴力冲突。
 <br/>
 <br/>
 视频显示，抗议者焚烧垃圾桶，警方向抗议者使用催泪弹。
 <br/>
 <br/>
 图卢兹附近、布拉尼亚克市的一所高中，部分建筑被点燃。
 <br/>
 <br/>
 早前法国政府宣布，从2018学年开始，新增大学入学选拔制度。
 <br/>
 <br/>
 上个月，法国总理爱德华・菲利普宣布，明年9月起，非欧盟国籍的学生，公立大学注册费从几百欧元上涨到数千欧元。
 <br/>
 <br/>
 周三，学生的抗议仍在继续。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
