### 加拿大公开承认 收到卡舒吉〝遇害录音〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】周一，正在法国参加〝巴黎和平论坛〞的加拿大总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特鲁多.html" target="_blank">
  特鲁多
 </a>
 表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大情报机构.html" target="_blank">
  加拿大情报机构
 </a>
 已经听过沙特记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇害录音.html" target="_blank">
  遇害录音
 </a>
 〞。
 <br/>
 <br/>
 记者：〝你说，加拿大（情报）机构听过这些录音？〞
 <br/>
 加拿大总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特鲁多.html" target="_blank">
  特鲁多
 </a>
 ：〝是的〞
 <br/>
 记者：〝他们（情报机构）听了吗？〞
 <br/>
 加拿大总理特鲁多：〝是的〞
 <br/>
 记者：〝你听了吗？〞
 <br/>
 加拿大总理特鲁多：〝没有。〞
 <br/>
 <br/>
 沙特记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 生前一直批评沙特王储萨勒曼，土耳其认为，他上个月在沙特驻土耳其领馆内遇害。沙特当局承认卡舒吉遇害是有预谋的，但他的遗体还没找到。
 <br/>
 <br/>
 加拿大总理特鲁多：〝在调查卡舒吉遇害这个问题上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大情报机构.html" target="_blank">
  加拿大情报机构
 </a>
 和土耳其情报机构，一直非常密切合作。加拿大充分了解，土耳其分享的内容简报。〞
 <br/>
 <br/>
 上周末，土耳其总统埃尔多安表示，已将卡舒吉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇害录音.html" target="_blank">
  遇害录音
 </a>
 转交给了美国、法国、德国和英国政府。加拿大是首个公开承认收到录音的西方国家。
 <br/>
 <br/>
 特鲁多表示，将与盟友讨论针对沙特的下一步行动。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
</div>
