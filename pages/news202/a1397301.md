### 彭博社：川习会若不理想 美将对所有中国商品征税
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 目前仍然处于僵持状态。最新消息称美国总统川普（特朗普）和中国国家主席习近平，将在下个月举行的二十国峰会（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 ）期间会面。有美国媒体透露，若
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 结果不理想，美国将宣布对尚未征收
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 的中国货品全部征收关税。
 <br/>
 <br/>
 11月30日和12月1日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会将在阿根廷的布宜诺斯艾利斯举行。白宫首席经济顾问库德洛（Larry Kudlow）日前确认，川普将在G20峰会上与习近平会面，不过并没有透露过多细节。
 <br/>
 <br/>
 《彭博社》10月29日的报导引述3名消息人士称，如果川习贸易会谈失败，美国政府将在12月初公布针对中国新产品的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 清单。清单公布后，将经过60天的公众意见征询期，关税实施时间可能在2月初中国新年假期前后。
 <br/>
 <br/>
 据悉，这张征收关税清单将包括之前尚未征收关税的中国货品，总值约为2570亿美元。
 <br/>
 <br/>
 其中2名消息人士还表示，美国官员显然已经做好准备，一旦中美领袖峰会无法就贸易争端取得进展，就会实施有关举措。不过他们也强调，华府尚未做最后决定。
 <br/>
 <br/>
 报导还说，〝中国制造2025〞是目前美国最在意的事情，也是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 协商中的关键。美国指控中共强制转移技术、窃取他国科技。
 <br/>
 <br/>
 另外也有消息来源显示，川普的一些贸易顾问并不希望与中共讨论贸易课题，除非北京能证明已认真看待美国的谈判要求。《华尔街日报》称，美国已要求中共就强制技术转移及其他中美经贸议题，交出具体的解决及让步方案，否则美方拒绝重新展开贸易谈判。
 <br/>
 <br/>
 自从7月6日中美贸易战正式开打之后，中美高层会晤与谈判即告暂停。此前中共副总理刘鹤带队访问华盛顿以及美国财长、商务部长领衔访问北京景象不再。仅仅8月23日，应美方邀请，中共商务部副部长王受文短暂访美，而因其会晤层级较低，中美没有达成任何共识。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：范铭）
