### 时事拼盘：加州野火已致59死 涉卡舒吉案5人或判极刑
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】下面请看一组新闻简讯
 <br/>
 <br/>
 加州大火已经肆虐一周，周四死亡人数上升到59人，约130人失踪，9000多户住宅被烧毁，5.2万人被疏散。周四气温下降给灭火提供了便利条件，消防人员已成功减缓了大火蔓延到人口稠密地区的势头。
 <br/>
 <br/>
 周四，已有1500多名中美洲移民乘坐公共汽车抵达墨西哥边境的蒂华纳市，等待时机进入美国。预计未来几天还将有数千人到达。
 <br/>
 <br/>
 洪都拉斯移民 Josue Vazquez:〝我们现在看看他们怎么办，我们就站在(美国的)家门口。〞
 <br/>
 <br/>
 中共商务部周四证实，11月初中美领导人通话后，双方已恢复经贸领域对话。之前因为中共干预美国选举等问题，双方对话停止了三个月。如果北京不答应解决美国提出的保护知识产权等议题，美国将对最后一批价值2670亿美元的中国进口商品征收关税。
 <br/>
 <br/>
 沙特检方周四表示，将对11名涉嫌杀害记者卡舒吉的嫌犯中的5人提出死刑判决。这5名嫌犯于10月2日在沙特驻伊斯坦布尔领事馆内将卡舒吉毒杀，并肢解尸体，运出领馆。土耳其不满沙特检方的处理，认为这些嫌犯或另有后台。目前涉案最高官员是沙特前情报局副局长阿西里。
</div>
