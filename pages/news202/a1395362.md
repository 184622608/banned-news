### 日本最大针织品商店街 乡土与传统聚人气
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 东京日暮里的纤维街，早在大正与昭和时代起便以经营生活不可或缺的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_针织品.html" target="_blank">
  针织品
 </a>
 而闻名，近年来更成为海内外游客新的人气地。这里已不光是传统意义上的观光地，浓厚的乡土气息与日本制造已经成为海外了解日本的重要元素。
 <br/>
 <br/>
 记者：东京的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日暮里纤维街.html" target="_blank">
  日暮里纤维街
 </a>
 〞是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 最大规模的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_针织品.html" target="_blank">
  针织品
 </a>
 商铺云集地，这里从布匹布料到成衣、针织品小物一应俱全，不仅是当地居民生活不可或缺的一部分，还受到来日日本各地以及海外游客的喜爱。
 <br/>
 <br/>
 〝日暮里纤维街〞位于东京东日暮里，是日本规模最大的布匹布料批发零售地。近年来，这里经营的日本制造的纤维布料，已是大量游客相继追逐的人气商品。
 <br/>
 <br/>
 日本客人：〝时隔十几年再次来到这里，变得干净整洁了，很容易就能找到自己想要的布料。〞
 <br/>
 <br/>
 纤维街上聚集了80家以上专门做纤维针织品生意的商家。这家商店已有35年的历史，所经营的商品声名远播海外，每天接待的客人中很多是外国游客。
 <br/>
 <br/>
 布料商店店员：〝这里汇集和日本全国各地的客人，日本的客人大多数都是为自己买布料，现在来自国外的客人也很多，多数都是买给亲人朋友的礼物，平日的客人有大约四成是来自国外的客人。〞
 <br/>
 <br/>
 澳大利亚游客：〝这家商店不仅美观，而且有很多种布料，材质很舒服，色彩鲜艳，可以供我这种游客的选项非常丰富。我和在澳大利亚的朋友们分享，并叫他们下次来日本一定要来这里。〞
 <br/>
 <br/>
 日本优质的服务态度和日本制造的信誉，拓宽了日本观光的多元化。
 <br/>
 <br/>
 新唐人日本记者站报导
</div>
