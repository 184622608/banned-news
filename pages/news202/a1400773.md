### 叙利亚再爆毒气攻击 上百人呼吸失常
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叙利亚.html" target="_blank">
  叙利亚
 </a>
 再度传出毒气袭击事件，北部城市阿勒颇约107人出现窒息症状。叙利亚政府及俄罗斯宣称毒气事件为反政府军所为，反政府军则指控其为诬陷。
 <br/>
 <br/>
 周六晚上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叙利亚.html" target="_blank">
  叙利亚
 </a>
 北部城市阿勒颇（Aleppo）爆出3个地区遭受含氯气的毒气袭击，造成107人呼吸困难。传出画面显示多名伤者被急救送医，并戴着氧气面罩。
 <br/>
 <br/>
 伤者出现呼吸困难、眼睛发炎红肿、发抖和晕厥现象。
 <br/>
 <br/>
 这是自从两年前叙利亚政府收复阿勒颇市之后，最严重的伤亡事件。
 <br/>
 <br/>
 叙利亚政府和俄罗斯皆宣称此次攻击为反政府军所为，反政府军则指控其为诬陷，表示他们并没有能力制造
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_化学武器.html" target="_blank">
  化学武器
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国总统.html" target="_blank">
  法国总统
 </a>
 马克龙周日在记者会上，谴责使用化学武器，但表示不能确定是反政府军作为。
 <br/>
 <br/>
 法国总统 Emmanuel Macron：〝我们没有清楚且足够的资讯来作出评论，尤其这个资讯是来自俄罗斯。〞
 <br/>
 <br/>
 他希望知情人士能将相关证据交至荷兰海牙的〝禁止化学武器组织〞（OPCW）总部，予以调查。
 <br/>
 <br/>
 新唐人记者王愉贺综合报导
</div>
