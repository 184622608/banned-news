### 韩法院判日本公司赔偿战时劳工 日方不满
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】韩国法院周四判决，日本三菱重工需要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赔偿.html" target="_blank">
  赔偿
 </a>
 二战期间的韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_劳工.html" target="_blank">
  劳工
 </a>
 。日本表示不接受判决结果。韩国指日本反应过激。
 <br/>
 <br/>
 周四，韩国最高法院判决日本三菱重工，需
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赔偿.html" target="_blank">
  赔偿
 </a>
 二战期间强制征用的10名韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_劳工.html" target="_blank">
  劳工
 </a>
 。其中一位（Kim Seong-ju）已年届九十。
 <br/>
 <br/>
 日本内阁官房长官菅义伟表示〝完全不接受判决结果〞，并召唤韩国大使。
 <br/>
 <br/>
 日本内阁官房长官菅义伟：〝这一判决令人深感遗憾，完全不能接受。〞
 <br/>
 <br/>
 菅义伟表示，韩国需确保
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本公司.html" target="_blank">
  日本公司
 </a>
 的公平经济活动，否则日本可能向国际法庭提起诉讼。
 <br/>
 <br/>
 日本内阁官房长官菅义伟：〝向国际法庭提起诉讼，保护日本公司公平的经济活动。〞
 <br/>
 <br/>
 首尔回应，尊重法院判决，敦促日本克制过激反应。
 <br/>
 <br/>
 韩国外交部发言人卢奎德：〝韩国政府尊重最高法院，赔偿强征劳工受害者的决定，包括今天的判决。〞
 <br/>
 <br/>
 10月30日，韩国法院还裁定日本钢铁公司、新日铁住金赔偿四名韩国劳工，四人中只有一位94岁原告)还在人世。
 <br/>
 <br/>
 外界担忧，日韩如果因此关系恶化，可能影响双方对朝鲜的制约。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
</div>
