### 以巴达成停火 以防长不满 宣布辞职
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】周一和周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 与巴勒斯坦在加沙地带爆发了2014年以来最激烈的冲突，周三，双方停火。但以色列
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国防部长.html" target="_blank">
  国防部长
 </a>
 因为不满停火而宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 。
 <br/>
 <br/>
 在日前冲突中，巴勒斯坦哈马斯曾一天内向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 发射火箭弹和迫击炮弹共约460枚，以色列予以有力还击。
 <br/>
 <br/>
 周三，以巴双方在加沙边界停火，但以色列
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国防部长.html" target="_blank">
  国防部长
 </a>
 利伯曼（Avigdor Lieberman）宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 。他对停火不满，认为不应给对方喘息，要立足长远解决问题。
 <br/>
 <br/>
 以色列国防部长 利伯曼：〝我们首先必须解决南方边境问题，南方的问题优先于其他一切，而且我们（停火）示弱，显然也会反映到我们的前线去。〞
 <br/>
 <br/>
 而以色列总理（Benjamin Netanyahu）指出他在根据具体的时间和状况，保护以色列人民安全。
 <br/>
 <br/>
 以色列总理 内塔尼亚胡：〝我知道以色列安全的更大远景，但我不能公告大众。我们的敌人（哈马斯）乞求停火，他们完全知道这其中原因。但我不能透露我们未来的计划。〞
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
