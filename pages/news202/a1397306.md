### 视大陆为掠食者 巴西新总统忧华经济扩张
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】巴西大选结束，社会自由党候选人、63岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波索纳罗.html" target="_blank">
  波索纳罗
 </a>
 以超过55%的选票赢得了总统选举。中国大陆是巴西的最大贸易伙伴，但波索纳罗视大陆为谋求主导巴西经济关键领域的掠食者。多家媒体纷纷推断，北京的愁云又将加重了。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波索纳罗.html" target="_blank">
  波索纳罗
 </a>
 (Jair Bolsonaro)也像美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）一样，一再批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 ，并提出了〝巴西优先〞、〝让巴西再次伟大〞的口号。所以巴西选民把他称为〝热带川普〞、〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西川普.html" target="_blank">
  巴西川普
 </a>
 〞。他在批评中共近年大举投资巴西时称，〝中国（中共）并不是在巴西采购，而是正在买下整个巴西〞。波索纳罗对华立场强硬，并主张阻止大陆购买巴西的能源及基础设施。
 <br/>
 <br/>
 波索纳罗认为巴西不应让大陆购买土地或控制关键产业，他反对巴西把部分国营企业私有化，忧虑大陆买家得标会扩大在巴西的影响力。
 <br/>
 <br/>
 <h4>
  波索纳罗称中共为〝掠夺者〞
 </h4>
 <br/>
 《大纪元》报导，波索纳罗对中共的批评有理有据，焦点之一是2016年洛阳栾川钼业公司（CMOC）在巴西购买了总值17亿美元的铌矿（niobium）。铌矿是强化钢材硬度的一种稀土，在汽车、建筑、喷气发动机等产品制造方面有着广泛的应用。在全球的铌矿供应当中，巴西占据了85%的市场份额。
 <br/>
 <br/>
 而中共却以相对低廉的价格，直接买走大量的铌矿。波索纳罗认为，中共在操控购买巴西的关键矿物，这些资源不能〝价廉物美〞的落入中共手中，而应该获得更大的收益。
 <br/>
 <br/>
 此外还有长江三峡集团，2016年用14.8亿美元控制了巴西两座大坝；中共国家电网公司去年用49亿美元取得巴西经营发电、配电和售电服务的〝CPFL〞能源公司及其子公司的多数股份；中共海航机场集团入主巴西繁忙的里约热内卢国际机场等等。
 <br/>
 <br/>
 正因为这些，这位〝巴西川普〞把中共称为〝掠夺者〞，对中共的〝投资〞非常警惕，誓言将阻止中共大规模收购巴西的能源资产和基础设施。也正因为他对中共的敌视态度，更大可能的赢得了选民支持。
 <br/>
 <br/>
 外界认为，波索纳罗胜选，对中共来说是个噩耗，中巴贸易关系可能将陷入瘫痪。中共曾计划提升与巴西的经贸合作，应对美中贸易战。中共期望至少可以增加进口巴西大豆和猪肉，弥补美国粮食减少出口中国的缺口。
 <br/>
 <br/>
 中美贸易战开打后，巴西出口至大陆的大豆总值更急增22%，80%的巴西大豆都是运往大陆。
 <br/>
 <br/>
 但中共的〝如意算盘〞可能打不响了，〝巴西川普〞可能会让中共的这个计划受阻。观察人士告诉香港《经济日报》，如果无法理顺中巴关系，甚至可能会爆发〝中国与巴西版〞的贸易冲突。
 <br/>
 <br/>
 <h4>
  波索纳罗称自己是〝川普的崇拜者〞
 </h4>
 <br/>
 中共其实也早就看透这一点，所以在竞选阶段就想尽办法拉拢波索纳罗。《路透社》报导，9月初，中共外交专门会见了他的高级经济顾问盖德斯（Paulo Guedes），邀请他去中国〝旅行〞。两名中共官员在上周还到波索纳罗竞选经理、国会议员Onyx Lorenzoni办公室游说。
 <br/>
 <br/>
 不过，中共的〝预防针〞能否奏效很难说，因为川普在票选结果公布后，第一时间就给〝巴西川普〞打了电话，两人已经开始有了互动。
 <br/>
 <br/>
 波索纳罗担任过7届国会参议员，曾说自己是〝川普的崇拜者〞。他也曾在2016年受访时表示：〝要恢复美国为巴西第一贸易伙伴。〞他同样主张严刑峻法打击犯罪、肃清贪腐，反对同性恋等。分析人士认为，在未来交往中，相信美巴关系将进一步紧密。
 <br/>
 <br/>
 时事评论员高天韵表示，巴西有强大的国力、文化、经济和军事实力。做为南美洲第一大国，巴西也是G20和金砖五国的成员国，对世界经济的影响很大。巴西还时常协调中南美洲其它国家的事务，调停南美洲发生的战争等等。所以美国前总统尼克松曾说过〝巴西走向哪里，拉美就走向哪里〞。
 <br/>
 <br/>
 高天韵称，在拉美和南美，古巴、委内瑞拉、秘鲁等多国都深受共产主义和社会主义的危害，左翼势力至今仍不可小觑。巴西作为拉美最大的经济体，它的政体走向对区域的影响力不可忽视。如今〝极右翼〞的〝巴西川普〞将要执政，南美大陆会不会随之发生变局呢？我们拭目以待。
 <br/>
 <br/>
 南美国家也出了一个〝川普〞目前可能是让中共最烦恼的事。不过，如果细心观察，最近让中共烦心的事不只这一件。打开各家网站，一连串各种中共不高兴的消息：印度总理莫迪访问日本深化两国防务合作、美国康奈尔大学全球首例中断了与中国人民大学之间的学术交流和合作研究项目、美国重量级参议员司法委员会主席查克・格拉斯利（Chuck Grassley）呼吁继续对中共保持强硬立场等等。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
