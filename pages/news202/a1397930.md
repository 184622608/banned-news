### 塔利班教父家中遇刺身亡 支持者上街骚乱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塔利班.html" target="_blank">
  塔利班
 </a>
 之父〞之称的穆斯林教士哈克，2日（周五）在巴基斯拉瓦尔品第（Rawalpindi）的寓所内遭人刺杀，当场死亡，凶手的身份及动机未明。事发后，大批激进穆斯林教徒上街示威骚动，场面混乱。目前尚未有组织承认责任。
 <br/>
 <br/>
 半岛电视台引述哈克（Maulana Samiul Haq）儿子透露，哈克当时身体不适卧床休息，当事司机兼保镳刚好外出15分钟，返回后发现他满身鲜血，已奄奄一息。
 <br/>
 <br/>
 路透社则引述其哈克的侄儿说，哈克先是被利刀刺伤，再开枪打死。哈克所属政党领袖证实他遭谋杀。
 <br/>
 <br/>
 警方事后将哈克遗体送往医院作进一部化验。期间有上百名支持者，情绪激动的在当地及首都伊斯兰堡骚乱，除放火焚烧高速公路收费站，也有支持者破坏商店和车辆。哈克家人呼吁支持者保持克制。
 <br/>
 <br/>

 <br/>
 82岁的哈克是一名富争议人物。他在邻近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 边境开设学院，其中一名学生为创办极端组织
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塔利班.html" target="_blank">
  塔利班
 </a>
 的主脑奥马尔（Mullah Mohammad Omar）。哈克于1985至1997年间，曾担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴基斯坦.html" target="_blank">
  巴基斯坦
 </a>
 参议院议员，在今年7月的选举中与执政党结盟，新上任的巴国总理伊姆兰汗本来希望与他合作进行多项改革，包括伊斯兰学校的教育。
 <br/>
 <br/>
 近日有阿富汗教士呼吁哈克运用对塔利班的影响力，协助阿富汗结束17年来的内战冲突。
