### G20发布领袖公报  同意改革WTO
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 已经达成一份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公报.html" target="_blank">
  公报
 </a>
 ，其中认同需要对世贸组织（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 ）实施改革，但避开了〝保护主义〞、〝不公贸易〞等敏感字眼。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 各国经过周五挑灯夜战，完成了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公报.html" target="_blank">
  公报
 </a>
 内容的最终定稿。
 <br/>
 <br/>
 由于此前美中立场分歧导致APEC峰会公报流产，此次G20峰会的公报内容格外受到外界关注。
 <br/>
 <br/>
 公报承认需要对现有世贸组织进行改革。公报中説，〝目前的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 系统未能完成其目标，有改进的空间。因此我们支持对WTO进行必要的改革，改善运作机制。我们将在下次G20峰会上审核改革的进度。〞
 <br/>
 <br/>
 川普政府已经多次表达了对世贸组织的不满，尤其是北京近二十年来未能兑现入世承诺，至今没有对外开放国内市场。欧盟、日本等多个G20成员也支持这一立场。
 <br/>
 <br/>
 在最具争议性的贸易议题上，G20公报仅以〝现存贸易问题〞的字眼一带而过。
 <br/>
 <br/>
 报导称，这是吸取了APEC峰会的教训，有意避开〝保护主义〞、〝不公贸易〞等敏感字眼。
 <br/>
 <br/>
 另一方面，金砖五国首脑周五在G20峰会上举行了非正式会晤。五国随后发表声明，呼吁促进更加开放的国际贸易，敦促WTO执行打击〝贸易保护主义〞和〝单边主义〞的职责。从内容看，其语气更接近于中共方面的立场。
 <br/>
 <br/>
 对于G20峰会结束当晚的川习会，外界有不同的预期。不过普遍的共识是，美中贸易争端的前景，最终还是决定在川普和习近平两人手中。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
