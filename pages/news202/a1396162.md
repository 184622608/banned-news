### 打好与欧盟关系 脸书聘任英国前副首相任副总
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脸书.html" target="_blank">
  脸书
 </a>
 聘请
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国前副首相.html" target="_blank">
  英国前副首相
 </a>
 克莱格，担任国际事务与通讯传播的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副总.html" target="_blank">
  副总
 </a>
 裁。克莱格有20年的政治经验，外界认为，脸书招募来自政治界的主管，是想打好与欧盟的关系。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国前副首相.html" target="_blank">
  英国前副首相
 </a>
 克莱格（Nick Clegg）：〝新闻才刚报出来，关于我在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脸书.html" target="_blank">
  脸书
 </a>
 的新工作。这显而易见地很令人兴奋，特别是我过去20年在欧洲议会、英国国会的政治生涯，这是一场大冒险对我去最大的自由论坛。〞
 <br/>
 <br/>
 据BBC报导，克莱格在脸书的正式职称将是〝国际事务与通讯传播
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副总.html" target="_blank">
  副总
 </a>
 裁〞。51岁的克莱格，从西元1999年当选欧洲议会议员，开始了20年的政治生涯。克莱格在西元2005年当选英国国会议员，曾任英国自由民主党党魁、英国副首相，直到去年落选国会议员才暂时沈寂。
 <br/>
 <br/>
 消息指出，脸书创办人扎克柏格（Mark Zuckerberg）和脸书营运长桑德柏格（Sheryl Sandberg）亲自出马招聘克莱格，是看中他曾经参与欧盟执行委员会的贸易谈判工作。
 <br/>
 <br/>
 今年三月，脸书遭骇8700万脸书用户资料，扎克柏格被迫道歉，并承诺增加大笔投资加强个资保护，欧盟也在今年五月颁布最严个资法《通用数据保护条例》（GDPR），严惩外洩个资的企业或机构。
 <br/>
 <br/>
 脸书执行长 扎克柏格：〝我们没有负起我们应该负起的责任，这是一个错误。对此我很抱歉。〞
 <br/>
 <br/>
 今年九月，脸书又因为2900万个资遭骇，根据欧盟的《通用数据保护条例》，对企业组织罚款最高可达年营收的4%，对脸书来说相当于14亿欧元（约新台币500亿元）。
 <br/>
 <br/>
 脸书深陷个资外洩风暴，数名高层离开公司，而克莱格将接替6月离职的施拉格（Elliot Schrage）。面对资安危机和欧盟的钜额罚款，就看克莱格要如何将他的政治智慧，运用在与欧盟的谈判桌上。
 <br/>
 <br/>
 新唐人亚太电视江子扬编译
