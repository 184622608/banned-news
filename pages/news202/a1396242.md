### 日本古代贵族的观月祭
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】中国有十五赏月的传统，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 有十三观月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_风俗.html" target="_blank">
  风俗
 </a>
 。一千二百多年前的日本平安时代，曾是贵族间流传的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统文化.html" target="_blank">
  传统文化
 </a>
 ，如今在日本国学院大学得到了承传。
 <br/>
 <br/>
 这里是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 国学院大学，今晚在这里举行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观月祭.html" target="_blank">
  观月祭
 </a>
 〞。十三之夜，少男少女，身着古装，奏着鼓乐，载歌载舞，对月亮表示秋收后的感激之情。
 <br/>
 <br/>
 观月祭是日本传统的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_风俗.html" target="_blank">
  风俗
 </a>
 ，在中国皇历九月十三日前夜，人们举行祭拜向月亮表达感激之情。现在，日本只有国学院大学的神道文化学部会每年举办这项活动。
 <br/>
 <br/>
 国学院大学学生武田雅人：〝观月祭是古代就有的观月活动，现在的日本这种古代的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统文化.html" target="_blank">
  传统文化
 </a>
 在逐渐消失。因此，希望大家能再次感受，重新思考，古时候日本人的所思所想。〞
 <br/>
 <br/>
 首先是由笙，数种笛子与鼓所奏，表达秋意的曲子；之后由4名舞姬身着以“十二单”为模型制作的华服，手持扇子，铃铛为祈求和平而舞。还有手持金戈，身着戎装，英姿飒爽的少年翩翩起舞。整个表演肃穆，古朴，彷佛把我们带回了日本的平安时代。
 <br/>
 <br/>
 新唐人日本记者站报导
</div>
