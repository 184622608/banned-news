### 中美洲难民潮 五角大楼：加派军队守卫边境
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】中美洲大篷车难民继续向美国挺进，与墨西哥警察爆发冲突，墨西哥内政部长重申反对暴力，并表示进入墨西哥的唯一途径是遵守移民法。美国国防部加派数百名士兵，守卫西南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境.html" target="_blank">
  边境
 </a>
 。
 <br/>
 <br/>
 大篷车难民继续从墨西哥南部往北走，向美国边界靠近。
 <br/>
 <br/>
 墨西哥当局表示，这些难民如果愿意留在墨西哥南部两州，将为他们提供医疗、子女教育和临时工作。但被难民拒绝，他们要坚持朝美国行进，一些人已出现身体不适。
 <br/>
 <br/>
 洪都拉斯难民 ANGELA ALEJANDRA HERNANDEZ：〝我们决定继续前进，在往前一些，我们有一点喉咙痛、发烧。〞
 <br/>
 <br/>
 同时，第二批欲前往美国的难民，被墨西哥防暴警察拦截在危地马拉和墨西哥
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境.html" target="_blank">
  边境
 </a>
 。
 <br/>
 <br/>
 这批大多数来自洪都拉斯的难民，人数一度超过1千人，处境令人担忧。
 <br/>
 <br/>
 在特昆乌曼镇，一些难民突破边境大门，与墨西哥警察爆发冲突，造成1名移民死亡，6名警察受伤。
 <br/>
 <br/>
 墨西哥内政部长重申反对暴力，进入墨西哥的唯一途径是遵守移民法。
 <br/>
 <br/>
 墨西哥内政部长ALFONSO NAVARRETE：〝墨西哥政府反对在危地马拉边界发生的暴力事件。重申进入墨西哥的唯一途径是遵守移民法。〞
 <br/>
 <br/>
 美国国防部周五表示，除了已有2100名国民警卫队成员被派遣到南部边境，保卫美国国土之外，还将向南部边境加派数百名士兵，以应对美墨边境难民危机。
 <br/>
 <br/>
 新唐人记者李梅、田瑞综合报导
</div>
