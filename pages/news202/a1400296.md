### 戈恩将延长拘押10天 雷诺任命临时主席
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日产汽车公司.html" target="_blank">
  日产汽车公司
 </a>
 董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戈恩.html" target="_blank">
  戈恩
 </a>
 因涉嫌动用公司资产和低报个人薪水被捕，周三，检察官决定延长拘押戈恩10天，此事的负面影响不止于日产一家公司。
 <br/>
 <br/>
 日产公司（NISSAN）内部调查发现其主席兼执行长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戈恩.html" target="_blank">
  戈恩
 </a>
 （Carlos Ghosn）涉嫌财务不法行为，包括个人使用公司资金和低报个人薪水。
 <br/>
 <br/>
 检察官称，戈恩在超过五年间低报薪水，大约是实际8800万美元的一半。检察官正在仔细权衡是否也对尼桑公司进行起诉。
 <br/>
 <br/>
 此外，戈恩同时是日本三菱汽车公司(Mitsubishi Motors)主席，法国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雷诺.html" target="_blank">
  雷诺
 </a>
 公司(Renault)的主席兼执行长。
 <br/>
 <br/>
 周二，雷诺公司任命拉斐特（Philippe Lagayette）担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_临时主席.html" target="_blank">
  临时主席
 </a>
 ，以顶替戈恩职位，而博洛尔（Thierry Bollore）将担任公司运营长。
 <br/>
 <br/>
 但雷诺董事会还没有让戈恩离职，正等待有关调查的更多进展。据报，检察官已决定将戈恩的拘押期延长10天。
 <br/>
 <br/>
 而戈恩身居要职，他的被捕也对尼桑、三菱、雷诺三家公司的联盟影响巨大。
 <br/>
 <br/>
 周三，日本政府发言人呼吁该联盟要保持稳定。
 <br/>
 <br/>
 目前戈恩还没有发表个人声明。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
