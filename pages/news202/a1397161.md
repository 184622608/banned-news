### 平安时代贵族女性打扮 绚子公主脱皇籍嫁平民
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_皇室高圆宫.html" target="_blank">
  皇室高圆宫
 </a>
 三女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绚子.html" target="_blank">
  绚子
 </a>
 今(29)日以平安时代贵族女性的打扮出嫁，继二姊千家典子婚后脱离皇籍，绚子下午正式向户政单位提出结婚登记书后，也将脱离皇籍。在绚子离开皇室后，皇室成员剩下日皇明仁及另外17位皇族。
 <br/>
 <br/>
 放送协会（NHK）报导，29日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绚子.html" target="_blank">
  绚子
 </a>
 与在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 邮船公司工作现年32岁的守谷慧举行结婚典礼，绚子上午10时30分过后抵达明治神宫，和已在鸟居前迎接她的守谷一同走向会场明治神宫神乐殿。
 <br/>
 <br/>
 婚礼在日本时间上午11时30分举行，宾客包括绚子母亲久子、姊姊承子、守谷的父亲等双方亲人及友人逾40多人，看到新人抵达后，大家都表达祝福并拍手；绚子和守谷则在进入会场前，不断鞠躬回应大家祝福。
 <br/>
 <br/>
 绚子在婚礼上穿着平安时代的服装，发型造型也采用平安时代贵族女性的〝大垂发〞。绚子跟守谷在交杯酒后交换婚戒，并宣读结婚誓词。
 <br/>
 <br/>
 绚子在加拿大留学时期的女性友人，今天也到结婚仪式会场观礼。这位女性友人说，看到绚子这么美丽，让她内心充满喜悦，〝绚子个性让人感觉温暖且富包容力，希望绚子跟守谷一定要建立一个属于自己的幸福家庭〞。
 <br/>
 <br/>
 同样到场观礼的守谷友人后藤康之说，守谷今天笑容相当爽朗愉快，守谷平常就是个好人，希望两人能成为一对最棒的夫妻。
 <br/>
 <br/>
 到高中都跟绚子就读同所学校的近藤祥子说，绚子虽然看起来有些紧张，但服装非常适合；她向绚子祝贺新婚快乐，绚子回她说，看到她能来参加婚礼很开心。近藤也祝福绚子跟守谷建立一个最棒的家庭。
 <br/>
 <br/>
 结婚仪式结束后，绚子将在一般参拜客的注视下，与守谷参拜祭奠日皇明治等的神殿。
 <br/>

 <h4>
  拥新户籍 离开皇室
 </h4>
 <br/>
 两人下午将赴新居所在地的东京都港区区公所，正式提出结婚登记书，在拥有新的户籍后，绚子就正式离开皇室，变成〝守谷绚子〞。
 <br/>
 <br/>
 绚子1990年9月15日在东京都出生，现年28岁，父亲是高圆宫宪仁亲王，母亲是高圆宫妃久子。高圆宫宪仁亲王已于2002年过世。
 <br/>
 <br/>
 绚子二姊典子2014年嫁给岛根县出云大社神职人员，脱离皇籍，婚后名为千家典子；大姊承子未婚。
 <br/>

 <br/>
 <h4>
  交往5个月答应求婚
 </h4>
 <br/>
 绚子是在2017年12月经母亲久子介绍，跟当时决定就任东京一个非营利组织（NPO）理事的守谷慧相识。这个NPO主要工作是协助世界上需要帮助的儿童。
 <br/>
 <br/>
 随着绚子跟守谷持续深入交往，在今年4月时，绚子答应了守谷的求婚。
 <br/>
 <br/>
 接下来，两人7月共同出席记者会宣布喜讯，绚子当时说，希望建立一个笑声不断且开朗的家庭。
 <br/>
 <br/>
 8月举行下聘的〝纳采之仪〞结束后，绚子上星期也完成〝朝见之仪〞，前往皇居向日皇夫妇告别。
 <br/>
 <br/>
 在婚前这段时间，绚子仍继续原本的公务，包括9月到青森县视察海洋少年团的训练，及10月前往福井县访视，观看国民体育大会的竞赛项目。
 <br/>
 <br/>
 婚前的平日晚上跟周末时，绚子跟守谷有时会一同到餐厅用餐，有时前往婚后新居查看，并挑选结婚戒指与家具。
 <br/>
 <br/>
 绚子婚后将离开皇室，跟守谷在东京都港区的华厦中开始两人的生活。
 <br/>
 <br/>
 宫内厅表示，目前两人暂无蜜月旅行的计画。
 <br/>
 <br/>
 绚子现在的工作是大学的研究员，未来是否会有变化，将跟大学方面商量后再做决定；至于从久子承继的2团体名誉总裁职务，也将分别跟2团体进行协调。
 <br/>
