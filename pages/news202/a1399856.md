### 多伦多榕树下训练中心为残疾人办筹款晚宴
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】服务于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_残疾人.html" target="_blank">
  残疾人
 </a>
 士的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_多伦多.html" target="_blank">
  多伦多
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_榕树下训练中心.html" target="_blank">
  榕树下训练中心
 </a>
 将在十一月二十四日晚六点在钓鱼台国宴，举办筹款活动
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_榕树下训练中心.html" target="_blank">
  榕树下训练中心
 </a>
 在过去的十六年中，为大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_多伦多.html" target="_blank">
  多伦多
 </a>
 地区的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_残疾人.html" target="_blank">
  残疾人
 </a>
 士提供训练课程，帮助他们在家中或外出的环境中，具备日常生活技能。
 <br/>
 <br/>
 2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_筹款晚宴.html" target="_blank">
  筹款晚宴
 </a>
 的主题为〝一生眷顾，无言的爱〞。
 <br/>
 <br/>
 榕树下执行董事David Yung:〝我们做为慈善机构，给有发展性障碍的人群提供服务〞
 <br/>
 <br/>
 今年晚宴的目地就是筹集资金作为中心运作及未来装修费用，以便提供更好，更完善的设施。
 <br/>
 <br/>
 榕树下执行董事David Yung:〝每一笔善款，我们都会用在帮助我们的学生身上〞
 <br/>
 <br/>
 今年的筹款晚宴，还有歌唱，舞蹈，乐器和魔术等多项表演。有关筹款晚宴的详情，请致电905-947-0036进行查询。
 <br/>
 <br/>
 新唐人记者多伦多报导
</div>
