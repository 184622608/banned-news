### 回应俄乌冲突 川普取消与普京G20会晤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周四（11月29日）宣布，鉴于俄罗斯尚未释放所扣押的乌克兰船隻及船员，他决定取消与俄罗斯总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 （Vladimir Putin）在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会期间的会晤。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在推特上宣布了自己因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄乌冲突.html" target="_blank">
  俄乌冲突
 </a>
 而取消〝双普会〞的决定。
 <br/>
 <br/>
 〝基于俄罗斯还没把船隻和水手送回乌克兰的事实，我已决定，最好所有相关方都取消之前所安排的我与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 总统在阿根廷的会议。这种情况一解决，我就会盼望（与普京）再次举行峰会。〞
 <br/>
 <br/>

 <br/>
 周二，川普在接受《华盛顿邮报》（The Washington Post）采访时表示，在俄罗斯向乌克兰海军舰艇开火并抓捕船员后，他可能不会在布宜诺斯艾利斯（Buenos Aires）会晤普京。
 <br/>
 <br/>
 川普告诉《华邮》，他当晚就会收到关于该事件的全面报告，然后决定是否会晤普京。他明确表示，〝我不喜欢这种侵略〞。
 <br/>
 <br/>
 川普是在〝空军一号〞专机上宣布取消与普京的会晤的，当时他正在赴二十国集团（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 ）峰会的路上。期间，白宫发言人桑德斯（Sarah Sanders）宣布，川普还将取消与土耳其和韩国领导人之间的正式会议，代之以在峰会上与各国领导人进行非正式会谈。
 <br/>
 <br/>
 11月25日，两艘乌克兰海军舰艇和一艘拖船在试图穿越连接黑海和亚速海的刻赤海峡（Kerch Strait）时，遭到俄罗斯海军武力拦截。俄舰声称这三艘船隻擅闯其领海，藉此向乌克兰船隻开火，造成六名船员受伤，三艘船隻以及二十多名船员遭到扣押，使得克里米亚（Crimea）危机再度升级。
 <br/>
 <br/>
 美俄关系自2014年俄罗斯吞并乌克兰克里米亚后一直比较紧张，虽然川普表示愿意在一定的前提下与俄罗斯改善关系，但近日发生的俄乌冲突可能会导致两国关系再度进入低点。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
