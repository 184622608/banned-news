### 安习会侧重经济 日企签备忘录非关一带一路
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 周六结束
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访问中国.html" target="_blank">
  访问中国
 </a>
 行程。这次中日两国在第三国经济合作上，签署了多项备忘录，这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安习会.html" target="_blank">
  安习会
 </a>
 也被认为是中日关系重新定调。不过，日本并不认为这次的合作是在帮助中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 计划。
 <br/>
 <br/>
 日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 ，今天结束了三天的中国访问行程，返回日本。时隔7年再度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访问中国.html" target="_blank">
  访问中国
 </a>
 ，外界认为，这是中日两国关系正式重回正轨，不过主要侧重在经济层面。
 <br/>
 <br/>
 日本首相 安倍晋三：〝这次与会的中日两国企业，以及相关机构，共签署了超过50项合作备忘录。〞
 <br/>
 <br/>
 26日的 〝中日第三方市场合作论坛〞，日本知名企业的高层联袂出席，参与人数达500人。
 <br/>
 <br/>
 朝日电视台：〝这次中日合作，中国有想要加速
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 构想的意图。但日本认为这次合作，并不是要协助中共的一带一路，双方认知有所差异。〞
 <br/>
 <br/>
 日本媒体报导，在美中贸易战背景下，日企确实难以尽全力与中国合作。而虽然此行安倍获得了一些成果，但在敏感的安全保障等议题上，要如何拿出解决问题之道，仍有待观察。
 <br/>
 <br/>
 朝日电视台记者 前田辅：〝在美中贸易战中面对川普政权，中共向美国成功地炫耀，日本站在自己这一边。但钓鱼台等议题仍悬而未解。〞
 <br/>
 <br/>
 有分析指出，这次习安会的最大意义是为中日关系重新定调，不过美日军事同盟是日本的外交基石，日本在安全战略问题上必然跟随美国。安倍作为〝印太战略〞的倡导者，也不会放弃在军事上与美国一起围堵中共。
 <br/>
 <br/>
 新唐人亚太电视综合报导
