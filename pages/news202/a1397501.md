### 印尼：找到失事客机部分机身
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 军方周三表示，周一坠毁的狮航客机
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_部分机身.html" target="_blank">
  部分机身
 </a>
 可能已经被找到。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 武装部队长官哈迪（Hadi Tjahjanto）周三表示，搜救人员目前在水下32米处发现了一个长约22米的物体，疑似为周一(10月29日)失事的狮航客机的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_部分机身.html" target="_blank">
  部分机身
 </a>
 。
 <br/>
 <br/>
 印尼军方长官哈迪：〝今早我从搜救机构得知，他们相信已经找到 （失事的）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_JT610航班.html" target="_blank">
  JT610航班
 </a>
 客机的部分机身，目前还有待进一步的确认。〞
 <br/>
 <br/>
 印尼军方长官哈迪：〝海军舰艇已经锁定了疑似坠落客机机身的坐标，我现在将前往那里。〞
 <br/>
 <br/>
 对于正在焦急等待，期盼奇迹发生的遇难者家属们来说，找到坠毁客机机身的〝好消息〞给他们带来了新的希望。
 <br/>
 <br/>
 失事客机家属 FINANDA AIRUBABA：〝我们希望会有奇迹发生。我确信我的父亲还活着，他一直是个很坚强的人〞
 <br/>
 <br/>
 救援人员在飞机失事地点总共发现了24具遇难者遗体，已有151名遇难者家属前来提供了DNA样本以及医疗和牙科记录。
 <br/>
 <br/>
 印尼狮航的波音737客机周一坠毁在印尼爪哇岛北部海域，飞机上共载有189人。搜救工作目前还在进行当中。
 <br/>
 <br/>
 新唐人记者李博文综合报导
