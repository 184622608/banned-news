### 北海道暴雪影响交通 观光客留意航班
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】受到低气压通过影响，日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北海道.html" target="_blank">
  北海道
 </a>
 连日降雪，预估到明(9日)将降下更多雪，气象单位呼吁民众留意大雪、暴风雪可能对交通带来的影响，观光客这几天往返北海道应留意最新航班资讯。
 <br/>
 <br/>
 日本放送协会（NHK）报导，日本气象新闻公司（weather news）报导，受到低气压通过影响，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北海道.html" target="_blank">
  北海道
 </a>
 道东的大树町，今天出现日本这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冬天.html" target="_blank">
  冬天
 </a>
 首次摄氏零下20度的低温，来到零下22度；除了大树町，北海道还有4个地方被观测到低于零下20度。
 <br/>
 <br/>
 札幌管区气象台表示，从6日到今(8日)清晨的主要地区降雪量，以北海道东部的厚岸町太田地区42公分最多，根室市厚床地区40公分居次，连札幌市区都有18公分的雪量。
 <br/>
 <br/>
 预估到明天持续受到冬季型气压分布影响，北海道将降下更多的雪。位于日本海侧的地区，到明天清晨6时止的24小时降雪量甚至可能多达50公分；下周降雪情况预估仍将持续。
 <br/>
 <br/>
 札幌管区气象台呼吁民众特别留意大雪、暴雪及雪堆对交通产生的影响。
 <br/>
 <br/>
 据北海道新千岁机场官网航班资讯，昨(7日)部分航班因天候状况取消或延迟，今天大多数航班则维持正常起降，这2天搭机往返北海道的旅客，应多留意最新航班资讯。
