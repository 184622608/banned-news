### 【禁闻】纪念共产主义受害者日 川普发表声明
------------------------

 【新唐人2018年11月08日讯】
 <b>
  放弃〝
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
   中国制造2025
  </a>
  〞？学者：不可能
 </b>
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 11月7号在记者会上说，中共已经放弃了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞计划。他说：〝因为我发现这个计划很放肆。我对他们说这很冒犯，因为这意味着到2025年他们会在经济上称霸全球。我说这不会发生。〞
 <br/>
 <br/>
 对于中共放弃〝2025计划〞的说法，中国大陆多名学者都表示，他们在国内未曾听说过，估计是中共对外改变了说法。
 <br/>
 <br/>
 〝自由亚洲电台〞引述广州中山大学经济系教授林江的话说：中共绝对不会放弃这个计划的，因为放弃了，不仅对经济有很大影响，而且对政治也有严重影响。
 <br/>
 <br/>
 网上也网友评论说，这是中共的所谓〝中国梦〞，他们不可能放弃的，又是个骗人的把戏。
 <br/>
 <br/>
 <b>
  纪念
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产主义受害者日.html" target="_blank">
   共产主义受害者日
  </a>
  川普发表
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_声明.html" target="_blank">
   声明
  </a>
 </b>
 <br/>
 <br/>
 11月7号是美国〝全国纪念共产主义受害者日〞，白宫当天发表川普总统的声明，纪念〝一亿多名被共产主义极权政权杀害和迫害的人〞。
 <br/>
 <br/>
 川普在声明中说，自从1917年俄罗斯布尔什维克革命以来，我们目睹了共产主义暴政意识形态所造成的后果——痛苦、压迫和死亡。共产主义把与生俱来的人权置于所谓的全体幸福之下，其结果是消灭宗教自由、私有财产、言论自由和经常发生的灭绝生命。
 <br/>
 <br/>
 他说，这些恐怖包括在〝大饥荒〞中蓄意让乌克兰人挨饿，在〝大恐怖〞中清洗俄罗斯人，在〝杀人场〞谋杀柬埔寨人，还有枪杀试图投奔自由的柏林人。这些（暴行）和其它很多暴行的受害者都默默地见证了这样一个无可否认的事实，那就是，共产主义以及对共产主义的追求，对人类精神和人类繁荣将永远具有破坏性。
 <br/>
 <br/>
 1917年11月7号，俄罗斯爆发〝十月革命〞，建立了世界上第一个共产党政权。去年11月7号，在〝十月革命〞爆发一百周年当天，川普政府宣布这一天为〝全国纪念共产主义受害者日〞。今年是白宫第二次就〝全国纪念共产主义受害者日〞发表总统声明。
 <br/>
 <br/>
 编辑/周玉林
