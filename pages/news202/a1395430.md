### 韩朝举行第五次高级别会谈
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】周一(10月15日)，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 在板门店韩方一侧的〝和平之家〞举行了第五次高级别会谈。
 <br/>
 <br/>
 此次高级别会谈的韩方首席代表为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 统一部长官赵明均，朝方首席代表为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 和统会会长李善权。
 <br/>
 <br/>
 韩朝双方在会谈中商定本月底举行体育会谈，商讨共同申办2032年奥运会。双方还讨论了开放离散家属常设会面场所和视频团聚，启动铁路、公路连接项目，尽快举办将军级军事会谈，平壤艺术团赴韩演出等一系列话题。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
