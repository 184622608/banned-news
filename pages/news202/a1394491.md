### 遣送移民变人球 德意互踢难解困局
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】有媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 计划将一些从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 上岸进入欧洲联盟、寻求庇护未获批准的非洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 ，以包机方式送回意大利。对此，意大利内政部长萨尔维尼7日声称要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关闭机场.html" target="_blank">
  关闭机场
 </a>
 。
 <br/>
 <br/>
 德新社（DPA）7日报导，柏林政府有意开始以包机方式，将部分庇护申请未获批准的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 送回到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 。报导说，第一班包机预定8日出发，另一班安排在10月17日。这些移民主要来自尼日利亚，先前是经意大利进入欧洲联盟国家。
 <br/>
 <br/>
 萨尔维尼（Matteo Salvini）对此作出回应，周日他在推文中说，〝在柏林或布鲁塞尔的任何人，如果想用未经许可的飞机把数十名移民送到意大利，他们必须知道，没有任何一个机场会让他们降落。〞
 <br/>
 <br/>
 萨尔维尼还提到意大利今年夏天针对禁止移民救难船进入意国海港的决定，说：〝我们将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关闭机场.html" target="_blank">
  关闭机场
 </a>
 ，如同关闭港口一样。〞
 <br/>
 <br/>
 而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 内政部发言人对此回应称，为了确保遣返的顺利实施，当局通常不会公布其具体时间和目的国。但该发言人亦明确表示，在未来几天内不会有搭载移民的飞机在意大利转机。
 <br/>
 <br/>
 意大利的劳工部长迪马尤（Luigi Di Maio）也在脸书上称，他不知道是谁允许租用飞机搭载移民到意大利，因为至今没有签订任何协议。
 <br/>
 <br/>
 意大利和德国正在协商，德国把已在意大利申请过庇护的移民送回意大利的程序。
 <br/>
 <br/>
 9月13日，德国内政部长泽霍费尔宣布，该国已与意大利就签署难民返还协议达成一致。当中规定，德国今后凡在同奥地利边境截获的难民中发现已在意大利申请避难者(也就是〝二次移民〞)，将有权当即将其遣返意大利。
 <br/>
 <br/>
 但与此同时，德国每遣送一人回意大利，便有义务从意大利接收一名从海上救起的难民。该协议内容与德国此前与西班牙和希腊所达成的相关协议类似。
 <br/>
 <br/>
 德国政府视与意大利、希腊和西班牙3国达成协议为消除其内部分歧、解决难民危机的重要一步。但截至7日，萨尔维尼还没有在协议上签字。
 <br/>
 <br/>
 泽霍费尔7日告诉德国《星期日世界报》，德国与意大利的协议此前已完成谈判，但萨尔维尼又要求德国支持意大利在欧盟难民政策问题上的立场，否则就不签该协议。泽霍费尔呼吁德意两国总理能出面解决困局。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
