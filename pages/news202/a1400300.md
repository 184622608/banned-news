### 川习会前 美报告指〝中共仍未改变做法〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 前夕，美国贸易代表办公室周二更新了〝301条款〞调查报告，指出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 至今没有改变
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不公平贸易.html" target="_blank">
  不公平贸易
 </a>
 的做法。
 <br/>
 <br/>
 美国贸易代表办公室（USTR）周二更新了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_301调查.html" target="_blank">
  301调查
 </a>
 报告。报告分6个章节。
 <br/>
 <br/>
 第一节回顾了美中贸易争端的源头，源于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 延续多年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不公平贸易.html" target="_blank">
  不公平贸易
 </a>
 作法。但过去几个月美中多次会谈，中方一直宣称不会改变。
 <br/>
 <br/>
 第二节到第五节，分析了中共仍在进行的不公平贸易做法。包括：1、网络窃取知识产权和商业机密；2、继续限制外商投资，以此施压外企转让技术给中国实体；3、利用不公平的行政许可审批制度，迫使外企转让技术；4、有计画的收购美国公司，以获得知识产权和尖端技术。
 <br/>
 <br/>
 报告第六节，USTR发表结论：中方没有从根本上改变其不公平、不合理以及扭曲市场的行为，相反最近几个月有变本加厉的迹象。
 <br/>
 <br/>
 谢田：〝不排除USTR是帮助川普政府在阿根廷会谈前，给中共以更明确的警告，就是我们清楚的知道你们没有实质上的改变，并且这个是不被我们认可的。〞
 <br/>
 <br/>
 中共一直否认，其主导了知识产权盗窃和强制技术转让行为。
 <br/>
 <br/>
 上周，北京向华府提交的142项贸易谈判清单，也被川普拒收，指其缺少了4到5个关键事项。
 <br/>
 <br/>
 白宫首席经济顾问库德洛说，北京必须拿出更好方案，才有可能与美国达成协议。
 <br/>
 <br/>
 库德洛(11/20)：〝协议内容必须包括知识产权保护，取消外企所有权限制，停止强迫技术转让，零关税，撤销非关税壁垒。必须有可执行性，必须有严格时间表。〞
 <br/>
 <br/>
 川普20号重申，如果无法和中方达成协议，将对更多中国商品加征关税。
 <br/>
 <br/>
 同时专家预计，不论G20谈判结果如何，美国都将采取措施、限制中共继续获得美国技术。19号，美国商务部出台了至今最严格的技术出口管制框架方案，考虑对人工智能等14项新兴技术，进行出口管制。
 <br/>
 <br/>
 谢田：〝我认为美国已经认识到了，中共不可能放弃（现有做法）。但是实际上美国不管你放弃不放弃，我的行动都在向前推进。〞
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
