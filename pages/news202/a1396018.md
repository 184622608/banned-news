### 阿富汗三高官遇刺身亡 美司令逃生
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】周四，阿富汗官员与驻阿富汗北约高官在坎大哈省举行例会后，突然遭到枪手的袭击，造成三名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗高官.html" target="_blank">
  阿富汗高官
 </a>
 死亡，而北约驻阿富汗指挥官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯科特．米勒.html" target="_blank">
  斯科特．米勒
 </a>
 上将因防弹衣而死里
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃生.html" target="_blank">
  逃生
 </a>
 。
 <br/>
 <br/>
 阿富汗官员表示，北约高官和阿富汗坎大哈省当地官员周四在坎大哈省省长官邸举行的这次会议，主要是确保本周六（10月20日）即将举行的大选的安全性。当与会者结束会议，走在省长官邸院子里时，一名负责会议安全的保镖突然开枪袭击与会人员，造成阿富汗最高安全指挥官阿卜杜勒．拉齐克、坎大哈省情报局局长阿卜杜勒．莫赫姆、坎大哈省省长扎尔迈．韦萨等三名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗高官.html" target="_blank">
  阿富汗高官
 </a>
 死亡。另有包括3名美国人在内的多人受伤。而北约驻阿富汗指挥官、美军上将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯科特．米勒.html" target="_blank">
  斯科特．米勒
 </a>
 因为防弹衣而死里
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃生.html" target="_blank">
  逃生
 </a>
 。
 <br/>
 <br/>
 塔利班随后宣称对袭击事件负责，并表示，袭击的首要目标正是北约驻阿富汗指挥官斯科特．米勒上将。
 <br/>
 <br/>
 新唐人记者李博文综合报导
