### 英超球队老板直升机坠毁 火球吞噬机体伤亡不明(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月28日讯】英国足球队
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_列斯特城队.html" target="_blank">
  列斯特城队
 </a>
 的泰籍老板维猜私人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_直升机.html" target="_blank">
  直升机
 </a>
 ，27日惊传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠毁.html" target="_blank">
  坠毁
 </a>
 在列斯特城队主场，现场燃起熊熊大火、浓烟密布，画面相当怵目惊心，而维猜目前生死未卜，伤亡情况也尚待进一步厘清。
 <br/>
 <br/>
 天空电视新闻网（Sky）报导说，出事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_直升机.html" target="_blank">
  直升机
 </a>
 据信载多达20人，直升机在比赛结束后约40分钟起飞，但直升机起飞几秒钟后，尾部推进器突然没有运转，发动机就停了下来，失去动力的直升机便撞向几百公尺外的停车场。
 <br/>
 <br/>
 目睹事发经过的自由工作摄影记者布朗（Ryan Brown）告诉英国广播公司5电台（BBC Radio 5 Live）：〝不夸张，发动机就这么停了，我转过头来，它发出呼呼声响，接着一阵死寂，螺旋桨开始旋转，然后有很大碰撞声。〞

 另有目击者表示，发生爆炸以后，有2名警察直接跑向失事飞机，但是火球太过灼热，让他们不得不退回去。
 <br/>
 <br/>
 也有目击者说，这架直升机在球场另一边
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠毁.html" target="_blank">
  坠毁
 </a>
 。有球员从里面跑出来，然后大批保全赶忙冲进去查看状况。
 <br/>
 <br/>
 从手机拍摄的事发现场显示，大火吞噬机体；现场摄影记者表示，直升机机舱冒出浓烟。
 <br/>
 <br/>
 目前尚不清楚泰国富豪维猜（Vichai Srivaddhanaprabha）是否在失事直升机上，路透社向维猜所属的皇权集团（King Power）的两名人员询问维猜是否有在直升机时，两人都拒绝证实。
 <br/>
 <br/>
 不过，英国广播公司（BBC）引述消息人士报导，维猜当时人在直升机里。而直升机机师身分及机上人员名单尚待进一步厘清。
 <br/>
 <br/>
 另外，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_列斯特城队.html" target="_blank">
  列斯特城队
 </a>
 前锋瓦迪（Jamie Vardy）和中后卫马奎尔（Harry Maguire）双双在推特上打出祈祷表情符号。
 <br/>
 <br/>
 维猜拥有泰国免税集团皇权集团，2010年以3900万英镑收购列斯特城队（Leicester City），隔年2月成为球队主席。该球队曾在2016年夺下英格兰足球最高等级的比赛冠军，2017年进入了欧洲冠军联赛的四强决赛。
