### 沙特记者卡舒吉被害案 埃尔多安:沙特预谋暗杀
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】周二，土耳其公布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉被害案.html" target="_blank">
  卡舒吉被害案
 </a>
 的细节。土耳其总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃尔多安.html" target="_blank">
  埃尔多安
 </a>
 认为，沙特当局预谋
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暗杀.html" target="_blank">
  暗杀
 </a>
 了卡舒吉，要求涉案的18人沙特特工小组在土耳其受审。
 <br/>
 <br/>
 周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃尔多安.html" target="_blank">
  埃尔多安
 </a>
 在议会上公布调查细节，10月2日卡舒吉被害当天，一支15人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 特工队进入领事馆。领事馆的摄像系统硬盘首先被拆除。
 <br/>
 <br/>
 埃尔多安认为，沙特预谋
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暗杀.html" target="_blank">
  暗杀
 </a>
 了卡舒吉。
 <br/>
 <br/>
 土耳其总统埃尔多安：〝到目前获得的资料和证据显示，卡舒吉在一场非常残忍的谋杀中被害。〞
 <br/>
 <br/>
 埃尔多安要求已经回到沙特的这些人，必须在土耳其受审。
 <br/>
 <br/>
 土耳其总统埃尔多安：〝责备一些情报成员，无论我们还是国际社会都不会对此满意。人类的良知要得到安慰，从下达命令的人到执行的人，这些人都有责任。〞
 <br/>
 <br/>
 埃尔多安说，该团队另外3名男子前往伊斯坦布尔和伊斯坦布尔的Belgrad森林，勘查地点。目前，土耳其正在两个地点寻找卡舒吉的遗体。
 <br/>
 <br/>
 土耳其还公布一份录像，一名男子从领馆走出去，穿着卡舒吉的衣服，带着眼镜，从领馆走出来。对比录像看到，两个人脸型、身材和头发不同。
 <br/>
 <br/>
 周二，美国财长姆奴钦会见了沙特王储。姆奴钦与英国、法国、荷兰等国官员，已经取消出席周二开幕的沙特投资会议。
 <br/>
 <br/>
 沙特阿拉伯外交部长朱贝尔周二在印度尼西亚表示，沙特将对卡舒吉被害经过进行透明、彻底的调查。
 <br/>
 <br/>
 美国总统川普周一说，他并不满意目前所听到的消息，但很快就会知道事情始末。
 <br/>
 <br/>
 美国总统川普：〝我很快就会知道，因为我现在在土耳其有一群很棒的人，在沙特阿拉伯有一群人。我们很快就会知道。我们拥有非常有才能的人，能很好的完成这些工作。他们今晚、明天回来。我很快就会知道。〞
 <br/>
 <br/>
 <br/>
 新唐人记者李、梅张琪综合报导
</div>
