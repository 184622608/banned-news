### 韩朝百姓及游客 最快今年可跨越军事分界线交流
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】韩朝军事当局和联合国盟军指挥部16日举行会议，商讨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_板门店.html" target="_blank">
  板门店
 </a>
 共同警备区（JSA）完全非武装化，让韩朝百姓及外国人游客可自由跨越
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事分界线.html" target="_blank">
  军事分界线
 </a>
 （MDL）往来于JSA内的韩朝两侧等相关细节措施。
 <br/>
 <br/>
 韩国〝朝鲜日报〞报导，3边会谈是在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_板门店.html" target="_blank">
  板门店
 </a>
 韩国辖区〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自由之家.html" target="_blank">
  自由之家
 </a>
 〞举行，并就上述内容的JSA非武装化细节措施达成了共识。
 <br/>
 <br/>
 报导说，韩朝领导人9月19日在平壤百花园迎宾馆高峰会谈后见证下，由韩朝国防部长签署的〝9.19军事协议〞，韩朝双方分别派遣35人以下的完全非武装人员执行JSA的共同警备任务。
 <br/>
 <br/>
 此外，双方居民和外国游客等，可在上午9时至下午5时自由往来于JSA内。但为了防止韩朝民间人士和外国游客彼此混合，将对韩朝的出入时间进行分离。
 <br/>
 <br/>
 韩国政府相关人士说，最快在今年内，韩朝居民和外国游客就能够跨越
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事分界线.html" target="_blank">
  军事分界线
 </a>
 ，在JSA进行交流和接触。不过，为了防止韩朝居民越境到对方管辖地区，JSA朝鲜辖区的〝72小时桥〞尽头将设韩国哨所，韩国辖区的进入道路一带将设朝鲜哨所。
 <br/>
 <br/>
 1953年7月韩战结束后设置的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_板门店.html" target="_blank">
  板门店
 </a>
 JSA，原本韩朝双方卫兵可以自由往来于军事分界线南北两方，但是1976年发生朝鲜军人在JSA内以斧头砍杀美军的事件后，韩朝双方的岗哨也分别调回军事分界线己方辖区，并禁止双方人员对话。
 <br/>
 <br/>
 (责任编辑：叶日观)
 <br/>
