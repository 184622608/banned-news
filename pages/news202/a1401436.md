### G20将成美中贸易战拐点？ 外界期望低
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 与习近平的双边会晤被视为重中之重。针对贸易争端，双方能达成怎样协议，外界普遍期望不高。美国参议院民主党议员马基（Edward Markey）日前也呼吁，美国必须要求中方解决重要的战略结构性问题。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 将与习近平举行会晤。不过，外界对于双方能够达成怎样的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易协议.html" target="_blank">
  贸易协议
 </a>
 ，冀望不高。
 <br/>
 <br/>
 战略与国际研究中心中国经济专家 甘思德：〝他们或者达成非常表面的协议，或者没有任何协议。如果记得的话，他们一开始谈到举行峰会，然后说一个重要会谈，现在降为一场晚餐。所以我们不要对于结果有太高期望。〞
 <br/>
 <br/>
 美国参议院外委会议员马基日前表示，过去美国期望藉通过〝接触政策〞，促使中国实现自由化的战略已经失败。中共已成为美国重大的〝战略挑战者〞。
 <br/>
 <br/>
 美国参议院外委会议员 Edward Markey：〝我们对于中共政府和政策方向的愿望，并没有实现。我对中共的一些政策与行为的趋势，深表担忧。〞
 <br/>
 <br/>
 马基认为，无论在经济、军事、人权及长臂干预它国方面，中共政策都令人担懮。
 <br/>
 <br/>
 美国参议院外委会议员 Edward Markey：〝必须保证，川普政府不会与中共达成某种协议，只是技术上缓和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 争端，而不解决重要的战略结构性问题。〞
 <br/>
 <br/>
 川普日前曾表示，中国必须开放市场，与美国公平竞争，双方才有可能达成协议。
 <br/>
 <br/>
 他还说，不太可能接受中方提出的推迟惩罚性关税的要求。
 <br/>
 <br/>
 专家认为，目前形势下，G20峰会不可能成为美中贸易争端的转折点。
 <br/>
 <br/>
 战略与国际研究中心中国经济专家 甘思德：〝美中都会保持对对方的巨额关税，美国会继续修订出口管制和高科技投资政策，并采取其它步骤，同时中共也会做出反应。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
