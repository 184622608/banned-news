### 马国再说〝不〞 遣送11名维族人到土国惹恼北京
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】马来西亚政府近日释放了11名非法入境大马的维吾尔族人，并以人道的理由将他们遣送至土耳其，此举引发中共的强烈不满。大马现任首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马哈蒂尔.html" target="_blank">
  马哈蒂尔
 </a>
 （Mahathir Mohamad）15日首次亲自出面就此事发声。这是大马政府继取消投资逾200亿美元的〝一带一路〞项目后，再次对北京方面的要求说〝不〞。
 <br/>
 <br/>
 本月15日，马来西亚现任首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马哈蒂尔.html" target="_blank">
  马哈蒂尔
 </a>
 在国会首次就大马政府允许11名维族人前往土耳其一事作出公开回应，他表示，这11人来到马来西亚后并没有任何不法行为，所以予以释放。
 <br/>
 <br/>
 这是11名维族人逃入马来西亚境内以来，大马官方首次针对此事件对外发表评论，也是大马政府继取消与中资企业逾200亿美元的〝一带一路〞合作案后，再一次对中共官方的要求说〝不〞。
 <br/>
 <br/>
 据路透社报导，这次得以离开吉隆坡前往土耳其的11名维族人，早在2014年间从中国新疆出逃后，曾经被泰国警方拘留。其后，这些维族人又于2017年底成功越狱，在逃入马来西亚境内后，又被大马警方逮捕。
 <br/>
 <br/>
 中共官方指控这些人与海外恐怖组织有关系，涉嫌在新疆地区发动攻击，并要求大马政府遣返这11名维族人。不过，大马政府最终接受了辩方的上诉理由，判断这些人如果被遣返中国将有性命危险，因而以人道理由撤销起诉，并允许他们离开吉隆坡前往土耳其。
 <br/>
 <br/>
 该事件令北京方面十分恼怒。中共外交部曾发表声明称，这些人都是中国国民，中方〝坚决反对将他们遣送到第三国〞。
 <br/>
 <br/>
 《美国之音》则报导称，中方近年在新疆对维族人扩大镇压，很多维族人辗转经由东南亚国家前往土耳其，但泰国等国的政府常常会依照中方的请求，将出逃的维族人遣返回中国。对此，国际人权团体曾提出强烈批评，指那些被遣返中国的维族人，将受到酷刑、甚至死刑等迫害。
 <br/>
 <br/>
 此外，据马来西亚媒体〝亚洲新闻〞报导，马哈蒂尔指定的首相继任者安华近期曾公开呼吁，认为马国政府应该就中共打压维族人的问题与中共政府进行正式会谈。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
