### 道路变成〝冰雹河〞罗马司机弃车逃(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】意大利21日(周日)出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天气异常.html" target="_blank">
  天气异常
 </a>
 现象，包括首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗马.html" target="_blank">
  罗马

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛罗伦斯.html" target="_blank">

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比萨市.html" target="_blank">

 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_米兰.html" target="_blank">
  米兰
 </a>
 都受到风暴吹袭。罗马更在当天傍晚下起冰雹，加上豪雨，导致道路变成〝冰雹河〞，多名司机被困，无奈只好弃车而逃。另外，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞浦路斯.html" target="_blank">
  塞浦路斯
 </a>
 也因到龙卷风吹袭，造成广泛破坏。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗马.html" target="_blank">
  罗马
 </a>
 (Roma)21日傍晚气温由摄氏20度骤降至12度，数小时之内最高录到3呎深降雨量，并出现冰雹。而雨水加上冰雹令汽车难以移动，造成当地交通阻塞。雨水也涌入地下铁路车站，令7个车站和1个火车站在交通尖峰时间被迫关闭。
 <br/>
 <br/>
 在同样受风暴吹袭的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛罗伦斯.html" target="_blank">
  佛罗伦斯
 </a>
 (Florence)及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比萨市.html" target="_blank">
  比萨市
 </a>
 (Pisa)，因事故频传，消防员疲于奔命。而位于意大利北部的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_米兰.html" target="_blank">
  米兰
 </a>
 (Milano)因风暴的吹袭，大树被强风吹倒；博洛尼亚(Bologna波隆那）因天气恶劣，多个航班出现延误现象。
