### 北约协防之下 科索沃决定自建军队
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】周四，位于巴尔干半岛的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科索沃.html" target="_blank">
  科索沃
 </a>
 共和国议会表决同意政府的提议，准备建立自己的正规军队。这让北约进退两难。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科索沃.html" target="_blank">
  科索沃
 </a>
 是前南斯拉夫的一部分，1999年科索沃战争后，脱离塞尔维亚的统治，2008年宣布独立。
 <br/>
 <br/>
 北约一直是科索沃的保护者，同时也承担着维护这一地区和平的任务。目前在当地有四千驻军。
 <br/>
 <br/>
 科索沃一直没有正规军。现有的武装力量武器装备少，主要用于维护治安，应对自然灾害等。
 <br/>
 <br/>
 虽然科索沃不属于北约，但这支武装力量接受北约的训练。
 <br/>
 <br/>
 北约的立场是，科索沃成立正规队，需要征得当地塞族的同意。
 <br/>
 <br/>
 周四在科索沃议会投票中，11名塞族议员用退场抵制建立军队的动议。
 <br/>
 <br/>
 塞尔维亚国防部长乌林在一份声明中说，科索沃唯一的武装部队只能是北约领导的维和部队，科索沃军队将〝威胁到塞尔维亚和塞族人〞。
 <br/>
 <br/>
 在北约斡旋下，塞尔维亚和科索沃正在就主权承认以及土地交换等问题谈判。
 <br/>
 <br/>
 另外科索沃希望加入北约，所以也需要考虑北约的立场。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
