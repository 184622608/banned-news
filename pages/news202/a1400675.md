### 安省改革社会福利 鼓励人们重返职场
------------------------

<div class="wysiwyg">
 【新唐人2018年11月24日讯】加拿大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安省.html" target="_blank">
  安省
 </a>
 ，有1百万人领取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_社会福利.html" target="_blank">
  社会福利
 </a>
 ，周四，政府推出了改革措施，提高了领取福利的门槛，为了鼓励人们工作，大幅增加了福利人士能保留的工作收入。
 <br/>
 <br/>
 社会服务厅长Lisa MacLeod：〝可悲的事实是，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安省.html" target="_blank">
  安省
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_社会福利.html" target="_blank">
  社会福利
 </a>
 系统没有帮助到想帮助的人们。它使人们陷入了依赖政府的循环中，而且往往剥夺他们的工作尊严和独立性。〞
 <br/>
 <br/>
 目前安省的社会福利有超过240个福利支付标准，800多条规则，对管理者和申请者来说都相当费解，也让福利人士很难摆脱对政府资助的依赖。
 <br/>
 <br/>
 新的福利改革将大规模减少各种行政规管，简化程序，并鼓励工作。
 <br/>
 <br/>
 社会服务厅长Lisa MacLeod:〝首先我们要给那些明显无法工作的人提供慈善的 帮助，第二，给那些想工作的人提供有意义的帮助。〞
 <br/>
 <br/>
 对于残疾人福利ODSP，残疾的定义将和联邦保持一致，也就是说会有更严格的标准。不过领取残疾福利人士每年可以有6千元的工作收入，之后工作收入的25%能保留，不会影响福利金额。
 <br/>
 <br/>
 领取普通福利ONTARIO WORK的人们，每月的头300元工作收入可以保留，之后工作收入的25%能保留，不影响福利金额。
 <br/>
 <br/>
 目前安省有1百万人领取社会福利，政府因此的支出是每年100亿，每月只有1%的福利人士重返职场，重返职场的人中80%的人一年内又再度依赖社会福利。
 <br/>
 <br/>
 保守党政府将陆续推出更多具体措施，对于有意工作的福利人士，政府网站ontario.ca/openforbusiness有帮助的资源，并在不断更新中。
 <br/>
 <br/>
 新唐人记者刘海英多伦多报导
</div>
