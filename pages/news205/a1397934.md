### 香港女星蓝洁瑛陈尸家中 传曾遭〝影坛大哥〞性侵
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】曾有〝靓绝五台山〞美誉的香港女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓝洁瑛.html" target="_blank">
  蓝洁瑛
 </a>
 ，3日被发现伏尸家中，其死讯震惊中港台。蓝洁瑛生前曾控述自己早年遭两位〝影坛大哥〞性侵。导致她被逼走上疯癫之路，生活更要靠友人接济。
 <br/>
 <br/>
 香港多家媒体报导，11月3日凌晨，香港知名影星，前TVB艺人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓝洁瑛.html" target="_blank">
  蓝洁瑛
 </a>
 ，被发现死在赤柱马坑邨良马楼寓所内，终年55岁。
 <br/>
 <br/>
 一名雷姓女子前往蓝洁瑛家，发现单位内传出异味及噪音，且无人应门，她随即报警求助。
 <br/>
 <br/>
 警员接报到场后，消防破门而入，发现事主倒卧地上，现场证实死亡。警方初步调查后认为，事件没有可疑，死因仍待调查。
 <br/>
 <br/>
 公开资料显示，蓝洁瑛于1963年出生，1983年投考香港无线电视艺员训练班，与刘青云、吴启华、陶大宇、吴君如、刘嘉玲等为同学，毕业后演出无线多部连续剧的女主角，有〝靓绝五台山〞美誉。
 <br/>
 <br/>
 蓝洁瑛曾出现过《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_义不容情.html" target="_blank">
  义不容情
 </a>
 》、《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大时代.html" target="_blank">
  大时代
 </a>
 》、《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大话西游.html" target="_blank">
  大话西游
 </a>
 》等多部影视作品。其在电影《大话西游》中饰演的〝蜘蛛精〞春三十娘一角，尤其令人印象深刻。
 <br/>
 <br/>
 蓝洁瑛在中港台盛极一时，岂料〝靓绝〞光辉只是昙花一现。
 <br/>
 <br/>
 陆媒形容蓝洁瑛：〝90年代电视剧里，人们记住她是《大时代》的罗慧玲，90年代电影里，人们记住她是《西游记第101回之月光宝盒》中的春三十娘和蜘蛛精，但一切就戛然而止至90年代。〞
 <br/>
 <br/>
 20世纪90年代，传出蓝洁瑛被强奸一事，多年来，蓝洁瑛一直精神不稳，无法工作，十分潦倒，要靠援助度日，至今传出香消玉殒，大起大跌命运令人希嘘。
 <br/>
 <br/>
 前好友杨曼莉透露，蓝洁瑛跟她说有两位圈中人曾出相助她渡难关，对她不离不弃。其中一个是刘德华，另一个是曾与她一起主持儿童节目《430太空梭》的曾华倩，二人均有出手救济她。
 <br/>
 <br/>
 但是，到了后期因蓝洁瑛不停改电话，死前数年，蓝洁瑛跟圈中人已经断绝来往。
 <br/>
 <br/>
 2013年12月，香港杂志〝Next Magazine〞发布了一个访问影片， 内容为蓝洁瑛亲述自己早年遭到两位〝影坛大哥〞性侵的经历，影片中对话人提出了两位〝影坛大哥〞的名字，但此录音被删除掉。
 <br/>
 <br/>
 影片播出后反响极大，引起不少媒体和中国网民的推测，但蓝洁瑛和知情的好友依然拒绝点名。
 <br/>
 <br/>
 今年初，有媒体收到了香港某知情人士提供的访问视频，视频中，蓝洁瑛对话中点出了性侵她的两位大佬名字，导致她多年来被逼走上疯癫之路，生活更要靠友人接济。不过其中一人已过世，另一人召开记者发布会否认此事。
 <br/>
 <br/>
 蓝洁瑛影迷会3日发文悼念写道：〝蓝洁瑛走得很安详，警方暂时还没联系上她家姐，希望能够好好的办理身后事。〞
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
