### 莫文蔚入行25年 出黑胶唱片明年香港开唱
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】为庆祝入行25周年，香港歌星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莫文蔚.html" target="_blank">
  莫文蔚
 </a>
 宣布出版首张
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑胶唱片.html" target="_blank">
  黑胶唱片
 </a>
 ，同时预告明年在香港开户外演唱会，给歌迷一个惊喜。
 <br/>
 <br/>
 不知不觉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莫文蔚.html" target="_blank">
  莫文蔚
 </a>
 已经出道25年，身材、形象却没有岁月的痕迹。精心打扮的她，出席首张
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑胶唱片.html" target="_blank">
  黑胶唱片
 </a>
 《我们在中场相遇》发布会。新唱片不仅汇聚很多经典歌，还有和新生代当红音乐人合作部份，最开心的是，圆了她出黑胶唱片的梦想。
 <br/>
 <br/>
 香港歌手 莫文蔚：〝很开心，因为我是在黑胶年代成长的孩子，但无奈我入行时是90年代初，黑胶刚刚结束了，没有人再出黑胶，已经进入CD年代，所以一直有少少遗憾。〞
 <br/>
 <br/>
 莫文蔚25周年全球巡回演唱已完成10个城市，已经13年没有在香港开演唱会，莫文蔚预告明年将回归香港，举行户外演唱会。密集开演唱会，她坦言对自己体力、精神都是一个考验。
 <br/>
 <br/>
 香港歌星 莫文蔚：〝我应该将以往做过的所有〝哇〞、所有的绝招，再次用不同的形式搬上舞台，实在太多东西搞。整个秀、两个多小时，很忙很忙，很多花式，很多事情要做。〞
 <br/>
 <br/>
 莫文蔚至今发行了超过30张专辑、7度入围且2度获得金曲奖最佳女歌手、举办超过75场个人演唱会，今年12月22日将第四次在台北小巨蛋登场，也是至今在小巨蛋开演唱会最多的香港女艺人。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
</div>
