### 贸易战90天停火未解忧 港商加速撤离大陆
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】虽然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 达成90天
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停火协议.html" target="_blank">
  停火协议
 </a>
 ，但在大陆设厂的港商对前景不乐观。有商会预料，未来近一成的港商或将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_倒闭.html" target="_blank">
  倒闭
 </a>
 或转行。也有港商透露加紧考察东南亚各国，准备将部份大陆厂搬走避税。
 <br/>
 <br/>
 在东莞设厂29年、美国订单占1/4的港商刘达邦表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 对港商冲击很大，商会有两成会员准备搬厂到东南亚，还有一些面临结业或裁员。
 <br/>
 <br/>
 香港中小型企业联合会永远荣誉主席：〝有的提早放假，有的厂已经结业，安抚员工。有些老板走佬，有些没有走，但要赔偿员工。做(IPONE)玻璃的伯恩都辞退了八千人。〞
 <br/>
 <br/>
 为求生存，从未想过会搬厂的刘达邦，近几个月也要密集式到东南亚各地考察，另辟途径避税。
 <br/>
 <br/>
 香港中小型企业联合会永远荣誉主席：〝可以考虑搬一部分过去。最快的就是在那边物色一些同类型的厂，跟他们合作，挂我们的名，在那里生产，就可以解决入口美国关税问题。〞
 <br/>
 <br/>
 国际中小企联合商会会长罗志光称，贸易战冲击下，逾50家会员已打算
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_倒闭.html" target="_blank">
  倒闭
 </a>
 或转行，占680个会员总数约8%。一半会员预料生意收缩30%至40%。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
</div>
