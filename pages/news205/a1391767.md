### 〝山竹〞创纪录破坏 港现灾难片般惨状
------------------------

<div class="wysiwyg">
 【新唐人2018年09月16日讯】挟十级飓风的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山竹.html" target="_blank">
  山竹
 </a>
 〞对香港造成严重的破坏，风速打破过往记录。在各方的要求下，教育局宣布星期一全港停课一天，教协及各政党表示欢迎。
 <br/>
 <br/>
 超强
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山竹.html" target="_blank">
  山竹
 </a>
 〞令香港海陆空交通运输瘫痪，并对沿海及市区造成严重的破坏。有市民拍到楼宇在一阵阵劲风下出现晃动，不少住宅及办公大楼的玻璃门窗被吹毁。沿海住宅区海水倒灌，水深至腰身。有网民拍摄到海水涌进，令柴湾海旁如电影〝明日之后〞的场景。香港天文台指，〝山竹〞录得每小时平均风速达155公里，打破了1962年温黛及1999年约克吹袭本港时创下的的纪录。
 <br/>
 <br/>
 另外包括教协、民主党、新民主同盟等建议当局尽早宣布周一所有学校停课，让学校先检查校舍情况，才于星期二正式复课。教育局在傍晚4时许宣布为为确保学生的安全，决定停课一天。教协及各政党欢迎教育局接纳建议。
 <br/>
 <br/>
 教协会长冯伟华：〝基于对学生的安全及明天回学校时可能会造成很大的混乱，有学校受到不同破坏，所以停课决定应该是好的决定，至于教育局能听取业界意见，我们觉得是一个好的决定。〞
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
