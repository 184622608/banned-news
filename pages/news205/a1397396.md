### 〝武侠〞金庸在香港去世 终年94岁
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】香港著名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武侠小说家.html" target="_blank">
  武侠小说家
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金庸.html" target="_blank">
  金庸
 </a>
 ，周二在香港去世，终年94岁。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武侠小说家.html" target="_blank">
  武侠小说家
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金庸.html" target="_blank">
  金庸
 </a>
 ，原名查良镛，于1924年3月10日生于浙江，其父在1950年的〝镇反〞运动中被中共枪杀。
 <br/>
 <br/>
 金庸后来移居香港。从1955年出版《书剑恩仇录》开始，创作了十多部流行的武侠小说，包括《倚天屠龙记》、《射雕英雄传》，《神雕侠侣》等。他的小说还被翻拍成电影和电视剧等。
 <br/>
 <br/>
 武侠小说家金庸：〝读者很流行，我现在也修改过去年轻的时候写的东西，不太成熟。有的时候写的匆匆忙忙，也没有查字典啦，查书了，历史的年代写错了。像《射雕英雄传》你看过，黄蓉和郭靖，读者一推算，黄蓉比郭靖大了，不对了。〞
 <br/>
 <br/>
 金庸在小说中塑造的众多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武侠高手.html" target="_blank">
  武侠高手
 </a>
 ，以及侠肝义胆、重情重义的故事，在对人管理严格的中国，被年轻人畅谈。
 <br/>
 <br/>
 10月30日下午，金庸在香港养和医院去世，终年94岁。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
