### 千五港人十一游行拒绝危城
------------------------

<div class="wysiwyg">
 【新唐人2018年10月02日讯】香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民间人权阵线.html" target="_blank">
  民间人权阵线
 </a>
 在十一当日举行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拒绝危城.html" target="_blank">
  拒绝危城
 </a>
 〞游行，约有1,500人参与，要求彻查港铁沙中线丑闻。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拒绝危城.html" target="_blank">
  拒绝危城
 </a>
 〞游行下午3时由铜锣湾东角道起步，要求政府彻查沙中线工程问题，以及反对二十三条立法。
 <br/>
 <br/>
 民阵召集人岑子杰：〝我们今日发起这个〝拒绝危城〞，我们是要守护香港，守护我们的民主自由和人权，令香港不要再继续变差下去⋯⋯我们一直看到很多市民非常关心沙中线问题，制度崩坏问题，23条问题。〞
 <br/>
 <br/>
 有市民不满中共侵害港人自由。
 <br/>
 <br/>
 香港市民方先生：〝所以10月1日不应该庆祝，这一日应该叫做国殇日，是要告诉共产党现在中国大陆这个政权令到人民非常痛苦，民不聊生，所以我们每年10月1日都要出来游行，告诉这个政权，我们是非常憎恨这个政权。〞
 <br/>
 <br/>
 队伍抵达终点政府总部，进入俗称的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公民广场集会.html" target="_blank">
  公民广场集会
 </a>
 ，民阵表示有1500人参与游行。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
