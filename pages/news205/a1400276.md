 赵培：2018年11月14号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国会.html" target="_blank">
  美国国会
 </a>
 〝美中经济与安全审查委员会〞发布了2018年中国的年度报告，建议美国商务部和有关部门审视军民两用物资出口到香港是否有足够的管制，同时也要检讨香港是否再有别于中国大陆，拥有独立的关税区地位。消息传出来之后，中共香港特首也是大惊失色，11月16号说是要与美国沟通这份报告。
 <br/>
 <br/>
 香港今天的危机纯粹是中共祸害出来的，一直想要把香港给吃到口里。2014年6月在中共常委张德江的指挥下，中共〝国新办〞发表香港白皮书想改动〝一国两制〞，引发了香港人上街的雨伞运动。今天香港特首想去美国解释〝一国两制〞还在，香港能保持自由、人权方面良好的记录。
 <br/>
 <br/>
 现在的问题是美国信不信呢？美中经济与安全审查委员会的报告里面说很多事实，媒体列举了几个事例：香港民族党被取缔，《金融时报》总编马凯工作签证被拒，1997年之后北京直接拒绝美舰访港至少四次，特别是美国制裁中共中央军委装备发展部及其主管李尚福后，中共否决了美军舰艇黄蜂号访港的要求。从这些行为来说，香港已经变得与大陆城市没有区别。
 <br/>
 <br/>
 这里涉及到一个国家认同的问题，香港与大陆是不是一个国家？这个标准就是国家的定义，什么是国家，其实就是政权和制度。美国人认为现在香港被中共给赤化了，是同一个政权和制度，所以就应该同一个关税区。香港特首要跟美国解释一国两制，那意思就是我们不是共产党，你看共产党在大陆怎么迫害百姓，我们不会这么做。
 <br/>
 <br/>
 按照中共的标准，你这个特首不爱国呀，但是中共不敢这么说，反而要配合特首的不爱国、不爱党的解释。这是为什么呢？主要有两个理由，一是中共想通过香港注册的皮包公司购买美国对中共管制的高科技产品然后走私到大陆；二是中共权贵还要利用香港的特殊地位把钱给洗白偷运到自由世界。
 <br/>
 <br/>
 香港特首的解释有用吗？很难。2000年，江泽民接受哥伦比亚广播公司采访的时候说自己不是独裁者，人家记者华莱士说，〝有句美国俗话说，如果走起来像鸭子，叫起来像鸭子，那就是一只鸭子〞。如果香港政府的做法与共产党一样了，那就是共产党，解释也没用。
 <br/>
 <br/>
 从现实来看，中共是想破坏香港的法治、赤化香港，但是在香港人的抵抗下没有完全得逞而已，比如说反二十三条，反国教等认同的政策。这同时也说明香港人不认同中共，国家认同度低。这没有什么不好，国家是政权，不认同无所谓，中共把文化与政权的认同给混为一谈，这就是骗人。传统文化才是中国人的标签不是中共政权，台湾人、大陆人、香港人政权不同但都是中国人。保住中华传统文化，中国人就能千秋万代，这才是根本。
