### 港楼股齐跌 地产大亨：中共制度需大变
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 拖累香港经济，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_楼股齐跌.html" target="_blank">
  楼股齐跌
 </a>
 。中原地产创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_施永青.html" target="_blank">
  施永青
 </a>
 表示，中国需要在党国关系等方面，出现制度上大的改变，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港楼市.html" target="_blank">
  香港楼市
 </a>
 才有机会复苏。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 拖累下，香港股市较今年高位跌两成，楼价较高位跌5%，成交量更跌一半。中原地产创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_施永青.html" target="_blank">
  施永青
 </a>
 表示，虽然楼价跌幅较股市少，但贸易战下大陆经济放缓，会进一步拖累
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港楼市.html" target="_blank">
  香港楼市
 </a>
 。
 <br/>
 <br/>
 中原地产创办人施永青：〝因为香港的服务业主要服务中国大陆，如果（大陆）经济有下行压力，香港也会受到压力，所以大家对投资前景有所担心。在这个情况下，入市意欲自然下降，我认为楼价调整不是一年半载可以完成，估计最快也要两三年。〞
 <br/>
 <br/>
 他估计楼价未来会跌两三成。何时反弹？他称中国需要在美国压力下找到新的经济增长点外，还需要从量变到质变，倒逼制度改革，才能复苏经济和楼市。
 <br/>
 <br/>
 中原地产创办人施永青：〝（中国需要）系统、制度性的改变，包括党国关系、国民关系、雇傭关系，这些都是要比较大的制度上的改革，才有机会抵挡到美国的全方位施压。〞
 <br/>
 <br/>
 仲量联行早前也表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港楼价.html" target="_blank">
  香港楼价
 </a>
 上升十年周期已完结，预计明年楼价最多跌25%。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
</div>
