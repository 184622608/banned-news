### 涉猥亵侵犯罪 香港律政司检控官坠楼身亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】港媒报导，香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_律政司.html" target="_blank">
  律政司
 </a>
 高级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_检控官.html" target="_blank">
  检控官
 </a>
 王永森疑涉非礼3名女同事，被控4项猥亵侵犯罪。今早(17日)，惊传王永森在大角嘴海帆道凯帆轩寓所
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠楼身亡.html" target="_blank">
  坠楼身亡
 </a>
 。
 <br/>
 <br/>
 王永森涉非礼案件，本月14日于东区法院出庭，王永森被控2014年至2018年期间先后对3名女同事非礼行为。
 <br/>
 <br/>
 王永森获准以现金2000元保释外出，但须要每月的第一天向旺角警署报到及不准接触控方证人。案件押后至明年1月17日再提讯。
 <br/>
 <br/>
 不料，上午9时58分传出，传出王永森于海帆道11号凯帆轩2座的住宅从高处坠下，当场死亡。
 <br/>
 <br/>
 警方获报到场调查，在现场检获遗书，正设法联络死者家人进一步了解情况。
 <br/>
 <br/>
 王永森(Vincent)任职
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_律政司.html" target="_blank">
  律政司
 </a>
 刑事检控科高级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_检控官.html" target="_blank">
  检控官
 </a>
 至今逾20年，案发时在刑事检控科的第一分科负责提供高院案件的法律意见，先后被委派在不同组别提供法律意见及出庭打官司，早年一度获得上诉庭法官司徒冕公开赞扬其准备工夫充足及表现佳。
 <br/>
 <br/>
 惊爆〝＃Me Too〞后，非礼事件惊动刑事检控专员梁卓然后，王永森被勒令休假，但他不予理会，仍继续现身办公室，直至《头条日报》独家报道事件后，王永森在家人劝说下同意休假。
 <br/>
 <br/>
 而受害者中不仅有女检控官，甚至律政司刑事检控科的普通女职员都胸部〝中招〞，一度向当时王永森的直属上司作口头投诉，但非礼事件未有停止，并愈爆愈激烈，令女检控人员及相关受害女同事感到被〝性骚扰〞，认为事态严重，决定报案调查。
 <br/>
 <br/>
