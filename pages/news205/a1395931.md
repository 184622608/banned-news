### 陈日君批中共透过梵蒂冈消灭地下教会
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梵蒂冈.html" target="_blank">
  梵蒂冈
 </a>
 日前秘密签订
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_主教任命协议.html" target="_blank">
  主教任命协议
 </a>
 ，香港荣休主教
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈日君.html" target="_blank">
  陈日君
 </a>
 枢机周四表示，对梵蒂冈的做法感到痛心，批评中共借梵蒂冈消灭大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地下教会.html" target="_blank">
  地下教会
 </a>
 。
 <br/>
 <br/>
 一直反对中梵协议的香港荣休主教
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈日君.html" target="_blank">
  陈日君
 </a>
 枢机强调大陆的地下主教才是正权主教，他直言目前中梵建交的协议令他痛心，批评对大陆的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地下教会.html" target="_blank">
  地下教会
 </a>
 很残忍。
 <br/>
 <br/>
 香港荣休主教陈日君枢机：〝在家中开弥撒，政府知道，当然知，隔离邻舍知吗？当然知，那么多人去，它无镇压那么多年了，现在说不行，现在命令你，2月1日开始不准了，只可去地上政府认可的， 所以很痛心。说得难听些，它是靠
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梵蒂冈.html" target="_blank">
  梵蒂冈
 </a>
 帮它消灭地下。（主持：梵蒂冈岂不是很大罪？）事实是这样。〞
 <br/>
 <br/>
 对于有消息指，新协议下教宗手握最终的主教任命权，但前期提名、遴选程序皆受北京操控，陈日君指此举是切割教宗的主教任命权，应该由教宗先选择主教。他又笑言，教宗不会去台湾，因为去了就不能去北京，因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 很小气。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
