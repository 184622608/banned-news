### 见识〝山竹〞威力 港童追风动弹不得喊痛
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月16日讯】挟带狂风暴雨直扑珠江口一带的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山竹.html" target="_blank">
  山竹
 </a>
 〞，令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 高挂10号飓风球示警，并改发红色暴雨警告。天文台表示，具破坏性的猛风正影响香港离岸水域。有网友上传视频，疑是年轻人想测试10号风球的威力，跑上街被强风吹得动弹不得；另有高楼的住户，拍下工地作业手臂，在空中转圈。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山竹.html" target="_blank">
  山竹
 </a>
 〞狂风暴雨正肆虐珠江口一带，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 多处强风豪雨，有些地方风力大到寸步难行。天文台在上午9时40分改发10号飓风信号，预料香港平均风速每小时118公里或以上。
 <br/>
 <br/>
 脸书帐号Kelvin Ling上传视频到群组〝Tai Po 大埔〞，可以看到一名身穿绿色衣衫的男孩被强风吹倒花圃边，在原地动弹不得，随后一名穿黑衣的男子却前往将男孩救起，但因地面湿滑直接跌倒，冲到花圃边，看到都觉得痛。
 <br/>
 <br/>
 黑衣男随后慢慢扶起男孩，两人在狂风暴雨中，慢慢走到附近凉亭暂避。
