### 何志平案开审 华信叶简明物业流拍
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】前香港高官何志平涉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贿络联合国官员案.html" target="_blank">
  贿络联合国官员案
 </a>
 ，26日在美国正式开审。其幕后金主华信集团主席叶简明，自今年3月被调查后，旗下一篮子物业沦为银主盘。星期一在香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拍卖.html" target="_blank">
  拍卖
 </a>
 ，但最终均未到价收回。
 <br/>
 <br/>
 华信集团持有的湾仔会展广场办公大楼21-23楼物业，起价分别是4.78亿、4.8亿和4.82亿港元，由仲量联行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拍卖.html" target="_blank">
  拍卖
 </a>
 。但最多只承接一口价，全部流拍，拍卖活动只用了15分钟。
 <br/>
 <br/>
 拍卖官：〝现在宣布因为没有人出价，物业将会被收回。〞
 <br/>
 <br/>
 有参与竞投人士认为价钱太高，令投资者有保留。卷入何志平涉贪案的中国华信能源主席叶简明，今年3月涉嫌经济犯罪，在大陆被调查。据称他在港拥有总值30亿的物业，全部沦为银主盘。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
 <br/>
 <br/>
