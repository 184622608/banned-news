### 马建吁港人团结 捍卫言论和创作自由
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】一度被取消在港举办讲座的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国流亡作家.html" target="_blank">
  中国流亡作家
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马建.html" target="_blank">
  马建
 </a>
 上周五成功入境港，最终顺利出席讲座，他很高兴今次风波令很多港人认识到言论、写作空间将受到损害的问题。他呼吁港人一起捍卫言论和创作自由。
 <br/>
 <br/>
 本身是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 永久居民的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马建.html" target="_blank">
  马建
 </a>
 上周六在中环大馆举行介绍新书《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国梦.html" target="_blank">
  中国梦
 </a>
 》的讲座，全场爆满，他透露新书是受了英国作家乔治．欧威尔的钜着《1984》的触动而诞生，强调中国梦的实现就必须尊重历史。
 <br/>
 <br/>
 作家马建：〝我主要强调一点‘记忆是最了不起的’，也是人最值得保存的，我们只有保存自己的记忆、自己的历史，才会有国家的历史，当这些东西一点点被删掉的时候，小说里面的马道德就彻底崩溃了。〞
 <br/>
 <br/>
 对于〝大馆〞之前曾一度以马建讲座涉及宣扬政治理念拒绝提供场地，他形容对方无知。不过他很高兴事件引起港人关注言论及写作空间受到伤害的问题。
 <br/>
 <br/>
 作家马建：〝不是有一个作家受到伤害与自己没关系，其实你们以后都要受到伤害，是一样的，他们也是一个人，有家庭的普通人，他受到伤害你们也要关心，你们受到伤害的话他也关心，这样言论的空间才能保得住。〞
 <br/>
 <br/>
 马建提醒港人，香港一国两制正逐渐变质，中共以温水煮蛙的形式令香港变成与其他中国城市一样。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
 <br/>
</div>
