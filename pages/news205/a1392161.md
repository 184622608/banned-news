### 陈方安生呼吁港人 力抗港府恶行
------------------------

<div class="wysiwyg">
 【新唐人2018年09月19日讯】面对港府及北京的威权管治，前政务司长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈方安生.html" target="_blank">
  陈方安生
 </a>
 在一论坛上严厉批评港府操控建制派在议会内以票数压倒一切，呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港人.html" target="_blank">
  港人
 </a>
 要力抗政府恶行。
 <br/>
 <br/>
 前政务司司长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈方安生.html" target="_blank">
  陈方安生
 </a>
 日前在公民实践培育基金举行的论坛上发言，她表示，以往政府会尽力游说议员支持政策，以理服人，批评现今港府以票数压倒一切，硬推不受欢迎或未经仔细审议的政策或条文，是大倒退。但她对未来仍很乐观，呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港人.html" target="_blank">
  港人
 </a>
 要抵抗政府恶行。
 <br/>
 <br/>
 前政务司司长陈方安生：〝我相信香港的核心价值不会崩溃。但我们仍旧要努力，我们每人要做好我们的本份，...坚守原则，本着良心做事，力抗政府侵犯公众利益的恶行，要求政治领袖开诚布公、透明、问责、尊重民意，这是香港人应得的。〞
 <br/>
 <br/>
 前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_立法会议员.html" target="_blank">
  立法会议员
 </a>
 吴霭仪则认为民主派要守住道德尊严，虽然北京不愿取消功能组别，但民主派仍可以一个个拿下议席，为民主运动打下强心针。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
