### 中共宪法掺入基本法 议员要求港府解释
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】中共当局一直企图将中共的宪法凌驾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基本法.html" target="_blank">
  基本法
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港府.html" target="_blank">
  港府
 </a>
 近日被揭露悄悄地在《基本法》中加入《宪法》，引起议员的担忧。
 <br/>
 <br/>
 星期三的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 立法会大会有建制派议员提出有关《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基本法.html" target="_blank">
  基本法
 </a>
 》与中国《宪法》教育的口头质询时，议会阵线朱凯廸拿出立法会内供议员使用《基本法》，指此版本突然加入中国《宪法》内容，要求当局解释港人如何遵守《宪法》第一条，〝禁止任何组织或者个人破坏社会主义制度〞的条文。
 <br/>
 <br/>
 议会阵线朱凯廸议员：〝我理解的宪法是界定政府的权力，保障人民不受政府的侵犯，不过中共拿住本宪法就不是这样用，它用来斩人用来实行专政，最近我们立法会议员桌子抽屉那本宪法都突然厚了，加入了宪法。〞
 <br/>
 <br/>
 政制及内地事务局局长聂德权承认，该合并版本是由政府提供的最新印刷本，是为了配合推广《基本法》与《宪法》的关系。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
