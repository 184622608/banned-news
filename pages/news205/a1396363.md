### 习近平抵珠海 或主持港珠澳大桥通车
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 星期二早上在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海.html" target="_blank">
  珠海
 </a>
 举行大桥开通典礼，盛传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 主礼，但官方只证实他抵达珠海，当地保安极严密，但官方一直未公布习近平是否主持大桥通车仪式。支联会主席何俊仁直言不可思议。
 <br/>
 <br/>
 去年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 访港前三日已公布访港消息，今次在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海.html" target="_blank">
  珠海
 </a>
 的开通典礼直到前一天仍未公布确定消息。不过本港多间传媒引用党媒的报导，习近平已抵达珠海巡视，相信会出席22日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 开通典礼。支联会主席何俊仁对当局迟迟不公布习的行程，直言不可思议。
 <br/>
 <br/>
 支联会主席何俊仁：〝但是一个领导人在自己国家里保安都紧张成这样，听说整支军队来了，到哪都整队御林军，都不知多少千人来保护。〞
 <br/>
 <br/>
 对于大桥通车前夕，突然传出澳门中联办主任郑晓松坠楼身亡的消息。他认为显示中共内部政治气氛极度紧张。
 <br/>
 <br/>
 支联会主席何俊仁：〝令人奇怪的是什么死因研究都没做，家人也还没说话，港澳办就说人家是忧郁症，港澳办怎么知道人家是忧郁症自杀呢？若知道他忧郁症严重就不应给其履行公职。〞
 <br/>
 <br/>
 澳门司法警察局公布郑晓松坠楼身亡的调查结果，称死因无可疑，没有任何刑事成分。但没提到郑有忧郁症的问题。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
