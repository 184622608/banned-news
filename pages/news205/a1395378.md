### 港逾万市民游行反大型填海批特首媚共
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特首.html" target="_blank">
  特首
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林郑月娥.html" target="_blank">
  林郑月娥
 </a>
 ，上周三在新一份施政报告提出〝明日大屿〞的大型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_填海.html" target="_blank">
  填海
 </a>
 计划，引来民间的强烈抗议。多个民间团体周日发起游行反对计划，批评林郑月娥此举是讨好亲共财团，现场至少有一万多人参加游行。
 <br/>
 <br/>
 游行星期日下午三点半从铜锣湾的东角道出发，前往政府总部。发起游行守护大屿联盟召集人谢世杰批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特首.html" target="_blank">
  特首
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林郑月娥.html" target="_blank">
  林郑月娥
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_填海.html" target="_blank">
  填海
 </a>
 计划会将香港过万亿的财政储备挥霍殆尽。
 <br/>
 <br/>
 守护大屿联盟召集人谢世杰：她常说工程发展将来会惠及很多市民，但大家要留意工程发展到最后，其实由工程开始到结束到建屋，我们每一个香港市民都是在津贴这些大财团。香港不是土地不足，是土地分配不均问题。〞
 <br/>
 <br/>
 许多父母专门带着子女游行。
 <br/>
 <br/>
 香港市民钟女士：觉得是特区政府用这去转移视线转移话题。这个不是担心，这个很明显是利益输送，在那个（董建华的）团结基金会的成员全都是发展商，摆明今次是政府用香港人的钱去打本给资本家做生意，所以这个绝对是利益输送问题。〞
 <br/>
 <br/>
 香港市民邓先生：〝政府很怆悴做这个决定，亦好似霸王硬上弓式的夹硬来。（带孩子游行）让他们认识我们作为人除了自己的生活之外，我们做的事都应该顾及大自然。
 <br/>
 <br/>
 游行队伍下午五时抵达政府总部并举行集会。至少超过一万人参加游行，警方则指高峰期有5800人参加游行。
</div>
