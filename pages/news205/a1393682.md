### 十一〝国殇日〞 港法轮功学员集会游行反迫害场面壮观
------------------------

<div class="wysiwyg">
 【新唐人2018年10月01日讯】10月1日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反迫害.html" target="_blank">
  反迫害
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集会游行.html" target="_blank">
  集会游行
 </a>
 ，多位政界人士赞扬法轮功的真善忍精神，谴责中共暴行。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 星期一早上11时在九龙长沙湾游乐场举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国殇日.html" target="_blank">
  国殇日
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反迫害.html" target="_blank">
  反迫害
 </a>
 集会。
 <br/>
 <br/>
 多位香港政界人到场声援。包括前立法会议员梁国雄、支联会副主席蔡耀昌、前立法局议员冯智活及前区议员徐百弟，他们谴责中共长达19年对法轮功的迫害，同时也敬佩法轮功学员以真善忍精神讲真相。
 <br/>
 <br/>
 支联会副主席蔡耀昌：〝我们今年十一继续出来表达对中共暴政的不满，我们希望我们的努力在不久的将来能得到成果。〞
 <br/>
 <br/>
 前区议员徐百弟：〝中国有句叫‘仁者无敌’，我们相信法轮功的朋友他们表现的仁者行为一定是无敌的。会将邪恶压制消失。〞
 <br/>
 <br/>
 集会结束后，约900人的游行队伍约下午2时由长沙湾游乐场出发，在天国乐团的领头下沿着弥敦道、广东道前往终点尖沙嘴码头。沿途挤满观看的大陆游客及香港市民。
 <br/>
 <br/>
 大陆游客陈先生：〝冲击我前十几年的印象，（怎么说呢）前十几年来我们接受的都是马克思主义这一套，但是这个游行至少是特别有序吧！唉！但是我也谈不上政治这种东西！但是我爱的是这片土地。〞
 <br/>
 <br/>
 香港市民陈先生：〝香港人习惯有宗教自由，但国内有它自己一套。..拍下给我的儿子做一个纪念，（为何）香港暂时还有这个自由，将来不知怎么样，给我的儿子做一个纪念。〞
 <br/>
 <br/>
 游行队伍约下午5时抵达尖沙嘴码头，结束一天的活动。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
