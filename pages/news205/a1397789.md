 香港立法会星期三起一连二天辩论由毛孟静议员提出的引用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特权法.html" target="_blank">
  特权法
 </a>
 传召保安局局长李家超及入境事务处处长曾国卫解释
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马凯.html" target="_blank">
  马凯
 </a>
 被拒工作签证续期申请的理由。有民主派议员批评此举是变相驱逐马凯。
 <br/>
 <br/>
 香港立法会议员谭文豪：〝行为等同以政治手段打压

 <br/>
 <br/>
 也有议员讽刺指，香港外国记者会（FCC）仅是办了一场论坛就踩中了中共及港府所设的〝红线〞。
 <br/>
 <br/>
 香港立法会议员尹兆坚：〝梁振英都去过FCC，他去就可以，他宣扬很多歪理很多港人都不喜欢，但他去讲就可以，人家去讲和你不同意见就不行？这是共产党逻辑，给你讲就是支持，如果不支持的话就打压你、禁止你。这就是中共的逻辑。〞
 <br/>
 <br/>
 议案最终在建制派占多数的情况下被否决。保安局局长李家超则重申，不评论个别个案，也不会交代拒绝签证的理由。
 <br/>
 <br/>
