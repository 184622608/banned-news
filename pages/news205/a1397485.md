### 港一地两检司法覆核案开审
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】高铁香港段虽已通车，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一地两检.html" target="_blank">
  一地两检
 </a>
 的实施一直备受争议。香港高等法院星期二一并处理由前立法会议员梁国雄等五人提出〝一地两检〞条例违宪，申请的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法覆核案.html" target="_blank">
  司法覆核案
 </a>
 。
 <br/>
 <br/>
 五位申请人包括理前立法会议员梁国雄、社工吕智恒、〝长洲覆核王〞郭卓坚、梁颂恒以及新民主同盟成员古俊轩，他们指高铁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一地两检.html" target="_blank">
  一地两检
 </a>
 违反基本法和一国两制。不过古俊轩在聆讯期间提出撤回申请，获法官批准。
 <br/>
 <br/>
 开庭前，数名社民连成员到高等法院声援梁国雄，他表示提出司法覆核是希望法院能够以法治精神判案。
 <br/>
 <br/>
 前立法会议员梁国雄：〝才要来到这里就政府是否违宪、有否程序公义进行司法覆核。所以我们希望香港人明白，我们没有滥用司法覆核，是因为人大常委滥用释法权，香港政府滥用扭曲的选举制度得到的公权力而造成的。〞
 <br/>
 <br/>
 广深港高铁香港段〝一地两检〞方案是在香港境内设立〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内地口岸区.html" target="_blank">
  内地口岸区
 </a>
 〞，实施大陆法律，被批评是割地方案。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
