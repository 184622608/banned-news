### 内外夹击 中资北美并购量价齐跌
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】近年在海外不断〝爆买〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中资.html" target="_blank">
  中资
 </a>
 企业受内外夹击。不仅中共当局连出规条，严堵资金外流，包括欧美等国，也纷纷收紧门槛，有调查显示，中资在美国前三季度的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资.html" target="_blank">
  投资
 </a>
 额〝量价齐跌〞。
 <br/>
 <br/>
 安永与中国并购公会联合发布报告，截至今年首3季止，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中资.html" target="_blank">
  中资
 </a>
 在美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资.html" target="_blank">
  投资
 </a>
 ，并购金额仅录84亿美元，按年大跌38%；数量按年跌25%至119宗。安永分析，中美贸易战、以及监管政策变化都是影响因素。
 <br/>
 <br/>
 安永中国海外投资业务部全球主管周昭媚：〝一直以来美国，都是投资者喜欢去的热门国家，每年都是在头三名，你现在看到很大的影响。现在政策就是信心的问题，现在不明朗的情况下，大家都不愿意，为什么我要踏出去。〞
 <br/>
 <br/>
 安永预期，随着人民币贬值，明年中资并购会更加审慎，预料并购额会进一步缩减。对于长江基建早前于澳洲收购天然气管道公司被拒，安永不评论具体案例，但强调个别敏感行业或会在并购时受到一定限制。
 <br/>
 <br/>
 新唐人记者梁珍、郭威利香港报导
</div>
