### 〝里根号〞弃五星改挂港旗 学者：帮港人出口气
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】〝川习会〞下周举行之前，日前访港的美国航空母舰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根号.html" target="_blank">
  里根号
 </a>
 ，在舱内没有悬挂中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五星旗.html" target="_blank">
  五星旗
 </a>
 ，而是挂上了香港区旗，和美国国旗并列，引起港人热议。
 <br/>
 <br/>
 川习会前，北京突然允许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根号.html" target="_blank">
  里根号
 </a>
 访港，里根号又改挂香港区旗，和美国国旗并列，引起港人热议。
 <br/>
 <br/>
 香港教育大学社会科学系客席讲师黄伟国：〝以往来（港）时挂五星红旗及美国国旗，而这一次是特区旗及美国国旗。但是这一次中联办，驻港解放军完全没有反映。其实给美国政府看在眼内，中国（共）政府是欺善怕恶，我觉得令很多香港人觉得出了口气。〞
 <br/>
 <br/>
 香港教育大学社会科学系客席讲师黄伟国：〝以往来（港）时挂五星红旗及美国国旗，而这一次是特区旗及美国国旗。但是这一次中联办，驻港解放军完全没有什么反应。其实给美国政府看在眼内，其实中国政府是欺善怕恶，我觉得令很多香港人出了口气。〞
 <br/>
 <br/>
 美国国会报告，上周批评中共近年不断蚕食香港的政治体系、法治和自由。也有分析认为，美航母藉此传递某种讯息。
 <br/>
 <br/>
 时事评论员蔡泳梅：〝打香港区旗是要提醒中国（共）政府，香港不是于大陆内地的城市，香港是一个享有高度自治的特区。〞
 <br/>
 <br/>
 翻查资料，2008年小鹰号、2011年以及2014年华盛顿号抵港时挂的是中共国旗，而去年里根号访港时则挂上印有〝76〞的舰队旗帜，既无香港区旗，也无中国国旗。
 <br/>
 <br/>
 新唐人记者梁珍、李海华香港报导
</div>
