### 大陆边防侵占土地 议员批违反基本法
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】早前传真社报道深圳边防人员涉嫌越境占用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙头角.html" target="_blank">
  沙头角
 </a>
 港方边境禁区约2万呎土地。星期三立法会答问大会上，有议员质疑政府是否不知情长达6年之久，批评深圳当局违反
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基本法.html" target="_blank">
  基本法
 </a>
 。
 <br/>
 <br/>
 公民党陈淑庄在立法会答问大会上，追问深圳边防人员越境一事，质疑政府不知情长达6年之久。特首林郑月娥回应指，日前已与深圳当局沟通，但双方对深圳河界线有不同理解，为免引起疑虑，大陆人员将暂停使用有关土地。
 <br/>
 <br/>
 陈淑庄会后批评深圳当局的做法已违反《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基本法.html" target="_blank">
  基本法
 </a>
 》。
 <br/>
 <br/>
 公民党陈淑庄：〝我们见到深圳市可以自把自为将我们的特区在国务院令限制的边界，无端端的以防洪为理由，改了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙头角.html" target="_blank">
  沙头角
 </a>
 的河道将香港的边界吃了进去，不光是违反了国务院令，也违反了《基本法》。〞
 <br/>
 <br/>
 朱凯迪引用Google Map分析，驳斥港府应是知情。
 <br/>
 <br/>
 立法会议员朱凯迪：〝因为这是严重的事，一是有越境执法的意图，二是很明显侵犯香港土地的管理范围，同时侵犯香港私人业主的私有财产，这是基本法保障的。〞
 <br/>
 <br/>
 陈淑庄警告林郑月娥不要为了讨好大陆当局一错再错，港人绝对不会接受先抢地后改令的处理方法。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
