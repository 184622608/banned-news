### 港团体抗议 促中共释放黄琦等人
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】近日有消息指，大陆知名维权网站〝六四天网〞创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 ，将被追加〝洩露国家秘密〞罪名。香港有团体到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 抗议，要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 释放黄琦等人。
 <br/>
 <br/>
 公民党与社民连等十多人，从西环西区警署游行到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 ，要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 释放释放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 。他们批评中共用同样的手法对待二人，不认罪就不开审。
 <br/>
 <br/>
 立法会议员郭家麒指，黄琦目前长期肾衰竭，中共继续关押他等同谋杀。
 <br/>
 <br/>
 公民党立法会议员郭家麒：〝只要老共不让他医其实他一定会死这是必然，更可耻的是人在你手上，在监狱仍不足够，它要你叩头认罪，要你将自己人格拿走，然后它说我拉你因为你错不是我错，连最后的彩都要拿这是很可耻，我们今日来是对强盗政权施以最大控诉。〞
 <br/>
 <br/>
 由于中联办一贯不肯接信，抗议人士最后将给中共中央的信件贴在中联办的门牌上。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
