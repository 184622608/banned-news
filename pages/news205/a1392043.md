### 港占中案11月开审 控罪不合理
------------------------

<div class="wysiwyg">
 【新唐人2018年09月18日讯】2014年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雨伞运动.html" target="_blank">
  雨伞运动
 </a>
 发起人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_占中三子.html" target="_blank">
  占中三子
 </a>
 及另6人被控公众妨扰等罪，案件排期于11月19日开审。被告称将进行抗辩。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_占中三子.html" target="_blank">
  占中三子
 </a>
 戴耀廷、陈健民、朱耀明及立法会议员陈淑庄、邵家臻等九人，星期一早上到西九龙裁判法院，就案件预审出庭，预计审讯约一个月。大部分被告不想就所谓的证据争论，但陈健民认为控罪并不合理，将影响
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 言论自由，将就此进行抗辩。他并强调9人有港人支持，无畏无惧。
 <br/>
 <br/>
 〝占中三子〞之一陈健民：〝参与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雨伞运动.html" target="_blank">
  雨伞运动
 </a>
 有100万人，现在他们不只在审我们9人，而是在审100万香港人，所以有那么多香港人支持我们，我们是完全无恐惧的。〞
 <br/>
 <br/>
 持续79天的雨伞运动，成为香港有史以来最大型的抗争运动，青年学子及市民的和平理性的诉求，得到国际社会的赞誉。本月28日是伞运四周年，民间团体将继续举行纪念活动。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
