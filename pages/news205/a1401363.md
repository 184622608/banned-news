### 港大学内建党支部 议员不满港府回避
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】早前传媒报导一批在香港城市大学深造的大陆法官，在校内成立中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党支部.html" target="_blank">
  党支部
 </a>
 。香港官员多次回避议员的相关质疑，引发不满。
 <br/>
 <br/>
 民主党立法会议员黄碧云就城市大学有攻读法学硕士或博士的大陆法官在港根据中国共产党章程的规定，成立了中共临时支部的问题。质问此举是否削弱市民对〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一国两制.html" target="_blank">
  一国两制
 </a>
 〞及〝港人治港〞的信心。
 <br/>
 <br/>
 立法会议员黄碧云：〝政府及专上院校会否采取措施，阻止中共党员在校园内进行政治活动，由任何人在港成立的中共支部，须否根据《社团条例》申请注册或豁免注册。〞
 <br/>
 <br/>
 法律界议员郭荣铿则批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港府.html" target="_blank">
  港府
 </a>
 一再回避问题。
 <br/>
 <br/>
 法律界议员郭荣铿：〝在这个议事厅中哪位是共产党员，他应该堂堂正正站出来告诉大家。〞
 <br/>
 <br/>
 政制及内地事务局副局长陈帅夫仅重申香港一国两制、港人治港、高度自治已透过基本法保障。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
 <br/>
</div>
