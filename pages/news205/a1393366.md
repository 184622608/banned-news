### 三千港人纪念伞运四周年 无惧中共争民主
------------------------

<div class="wysiwyg">
 【新唐人2018年09月28日讯】今年9月28日星期五是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港雨伞运动.html" target="_blank">
  香港雨伞运动
 </a>
 四周年，多个政党及民间团体再次相聚金钟连侬墙下，并在下午5时58分，即当年警方施放第一枚催泪弹的时间，举行默站仪式。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_占中三子.html" target="_blank">
  占中三子
 </a>
 之一陈健民强调
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港人.html" target="_blank">
  港人
 </a>
 良知不泯灭，终获胜利。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伞运四周年.html" target="_blank">
  伞运四周年
 </a>
 集会下午5时重返金钟连侬墙举行，约有32个政团参加。在台上台下一起高唱《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_撑起雨伞.html" target="_blank">
  撑起雨伞
 </a>
 》后，下午5时58分即当年警方施放第一枚催泪弹的时间，全体默站3分钟，不少人撑起代表雨伞运动的黄伞。包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_占中三子.html" target="_blank">
  占中三子
 </a>
 之一陈健民及当年的学运领袖呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港人.html" target="_blank">
  港人
 </a>
 毋忘伞运精神，强调港人良知不泯灭，终获胜利。
 <br/>
 <br/>
 占中三子之一陈健民：〝在催泪弹中我们见得到，一场运动如果可以叫起我们香港人的良知，我们坐牢也是值得的，我希望你们相信我，只要我们香港人良知不泯灭，我们这场运动一定有希望，胜利一定属于我们。〞
 <br/>
 <br/>
 香港众志秘书长黄之锋：〝改变了香港很多的沈默大多数，令他们有政治觉醒，这就是我们曾经拥有的雨伞精神。雨伞运动肯定是香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主.html" target="_blank">
  民主
 </a>
 运动不能忽略的一个传奇，希望大家能让我们更年轻的一代知道雨伞运动的历史，怎么影响香港民主运动光辉的一页。〞
 <br/>
 <br/>
 民阵副召集人区诺轩则呼吁港人踊跃参加10月1日的〝拒绝危城〞游行，抗议政府取缔民族党，剥削港人结社自由，以及多项工程丑闻。
 <br/>
 <br/>
 主办方估计有三千人参加活动，反映香港人坚持雨伞精神。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
