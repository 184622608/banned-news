### 香港12年首跟随美加息！超低息成过去式
------------------------

<div class="wysiwyg">
 【新唐人2018年09月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国联储局.html" target="_blank">
  美国联储局
 </a>
 一如预期加息四分一厘，香港银行业12年首度上调最优惠利率。香港财经官员警告，香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超低息.html" target="_blank">
  超低息
 </a>
 环境已经过去。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国联储局.html" target="_blank">
  美国联储局
 </a>
 自2015年底以来，第八次加息后，香港银行业终于步入加息周期。包括滙丰、恒生、渣打和中银先后宣布上调最优惠利率(P)八分一厘，是12年首度加息。
 <br/>
 <br/>
 金管局亦上调贴现窗基本利率25个基点，至2.5厘。香港财经官员预期，美国将持续加息，预计最终
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港元利率.html" target="_blank">
  港元利率
 </a>
 会很贴近美元利率，呼吁市民小心管理好风险。
 <br/>
 <br/>
 香港金管局总裁陈德霖：〝现在全球的经济情况和宏观的金融环境，为香港带来颇大的不稳定性。因为资产市场尤其是楼市，会否显著调整，中美贸易纠纷会否恶化和持续，美元利息正常化的速度是否较预期快这些风险，都可能为市场带来波动和不稳。〞
 <br/>
 <br/>
 香港财政司司长陈茂波：〝香港一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超低息.html" target="_blank">
  超低息
 </a>
 环境已过去，未来息口趋升机会比较高。由于息口往上升，对市场和资产价格都有影响，请市民小心管理好风险。〞
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
</div>
