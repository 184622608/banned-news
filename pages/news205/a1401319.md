------------------------

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共

 人大释法，民主派四位议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梁国雄.html" target="_blank">
  梁国雄
 </a>
 、罗冠聪、姚松炎及刘小丽被裁定宣誓无效而丧失立法会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_议员资格.html" target="_blank">
  议员资格
 </a>
 。当中梁国雄提出上诉，上诉庭星期三展开聆讯。

 <br/>
 包括同样被DQ的刘小丽、姚松炎以及多位民主派立法会议员一早到高等法院声援，提出上诉的社民连成员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梁国雄.html" target="_blank">
  梁国雄
 </a>
 批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 人大就《基本法》第104条进行释法，将合法就任的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_议员资格.html" target="_blank">
  议员资格
 </a>
 取消，其后发展到连参选资格也被褫夺，是践踏基本法的行为。
 <br/>
 <br/>
 社民连成员梁国雄：〝我希望同时有这个责任令到香港的司法机关就此问题有正式的判决。所以姚教授、小丽老师去参选和我都是为了同一目的，大家不要乱猜测，今日我们看到共产党在选举中的行为，用党国机器不用申报，用各种方法去扰乱视听，用所谓鎅票的方法。其实我最大的心愿就是赢回议席。让良心法庭裁决。〞
 <br/>
 <br/>
 代表梁国雄的资深大律师李柱铭在庭上强调，中共人大是在梁宣誓后才决定释法，释法不应有追溯力。
