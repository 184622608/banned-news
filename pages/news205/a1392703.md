### 港高铁开通 团体抗议割地两检
------------------------

<div class="wysiwyg">
 【新唐人2018年09月23日讯】造价高达884亿港元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高铁.html" target="_blank">
  高铁
 </a>
 香港段星期日正式通车，前一日由特首林郑月娥主持通车仪式。多个团体到场抗议，不满高铁工程耗费大量公款，并担忧实施
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一地两检.html" target="_blank">
  一地两检
 </a>
 会破坏法治。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高铁.html" target="_blank">
  高铁
 </a>
 香港段星期日正式营运，首班列车前往深圳北，早上七时准时由西九龙站开出。首日通车便出现混乱，包括班车延误、取票机失灵、行李安排不当等。而在前一天的通车仪式上，大力推介高铁的特首林郑月娥承认日后公干未必适合搭乘高铁。
 <br/>
 <br/>
 特首林郑月娥：〝不过恐怕我每一次的出外行程都非常紧凑，所以在长途暂时来说未必适合我公干之用。〞
 <br/>
 <br/>
 包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一地两检.html" target="_blank">
  一地两检
 </a>
 关注组和民阵等团体在场外抗议，上演街头剧讽刺林郑及中港官员，不满高铁耗费巨资，忧虑实施一地两检会破坏法治。
 <br/>
 <br/>
 工党主席郭永健：〝大家见到我们就在这个地方穿一个洞，这个地方基本法无法施行，香港人权法的保障，保障不了。〞
 <br/>
 <br/>
 也有示威人士穿上中共官兵制服讽刺高铁一地两检〝割让〞香港。
 <br/>
 <br/>
 鸠呜团成员钱宝芬：〝看到都知道一定是中国共产党，这个很深刻的就是很经典的，共产党是一个血腥暴力的政权搞文化大革命，所以我们特地选这套衣服出来，就是提醒香港人现在共产党打过来了，要消灭我们香港。〞
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
