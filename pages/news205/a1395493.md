### 刘小丽遭取消参选 斥港府跟随大陆
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】香
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港民主派参选人.html" target="_blank">
  港民主派参选人
 </a>
 刘小丽，上周五遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港府.html" target="_blank">
  港府
 </a>
 取消参选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_立法会.html" target="_blank">
  立法会
 </a>
 九龙西补选的资格，她指出：今日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 已进入谎言治港、指鹿为马的时代。
 <br/>
 <br/>
 刘小丽星期一在一个电台的节目中坦言，对港取消她
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_立法会.html" target="_blank">
  立法会
 </a>
 补选的参选资格感到荒谬和非常粗暴，强调从2016年开始多番表示不支持〝港独〞和拥护《基本法》，从来都没变过。
 <br/>
 <br/>
 刘小丽：这件事荒谬到我们可能比大陆更差，我们无经过法庭审讯，无经过申辩，无经过客观论辩已取消我的参选资格。而究竟这会否是终身。因为在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 根本无这个，只能靠粗暴人治践踏和干预去做。所以今日你问我，我觉得不是适台对象，而应该问中联办也好、林郑月娥也好。〞
 <br/>
 <br/>
 刘小丽形容，今次事件显示当局令香港进入谎言时代。
 <br/>
 <br/>
 刘小丽：我们今日的香港已进入谎言治港、指鹿为马的时代，甚至是撕破面皮、厚颜无耻说啥也行，权在我手，我可任意操弄这个社会我想做的任何事，去践踏香港人的利益。
 <br/>
 <br/>
 刘小丽透露正谘询法律意见考虑是否上诉。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
