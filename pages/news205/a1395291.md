### 韩医热吹港 清肺名医徐孝锡谈养生之道
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】呼吸道疾病是四大杀手之一，现代医学束手无策。但起源于中医的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩医.html" target="_blank">
  韩医
 </a>
 ，却有其独特的疗法。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清肺名医.html" target="_blank">
  清肺名医
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐孝锡.html" target="_blank">
  徐孝锡
 </a>
 ，在香港首次演讲，吸引数百名观众来参加。
 <br/>
 <br/>
 继三年前北美四个城市巡回演讲受到好评后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐孝锡.html" target="_blank">
  徐孝锡
 </a>
 院长再度来到台湾和香港，介绍其独特的扁康疗法，再度引起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩医.html" target="_blank">
  韩医
 </a>
 的热潮。
 <br/>
 <br/>
 一身传统韩服，韩国名医徐孝锡，用粤语和观众问好后，以〝治病先清肺〞为题，系统阐述扁康
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_养生之道.html" target="_blank">
  养生之道
 </a>
 。起源于中医的传统韩医认为，〝肺为五脏之首〞，徐孝锡发现，通过清肺，扁桃体健康后，免疫力增强，一些疑难病症也得到治疗。
 <br/>
 <br/>
 扁康韩医院院长徐孝锡：〝连最可怕的肺纤维化，2年之内不会死，还能活到10年以上，这样造成的差异是非常大的，仅仅是因为扁桃体恢复了健康。〞
 <br/>
 <br/>
 参与听讲的民众，收获良多。
 <br/>
 <br/>
 香港肺纤维化患者陈先生：〝我在网上看，只有半年命。我听到以后吓到死，我应该今天六月死了，六月没死，现在十月了。〞
 <br/>
 <br/>
 香港肺气肿患者阮先生：〝气顺了，好睡了。〞
 <br/>
 <br/>
 香港健康顾问沈小姐：〝一般的哮喘、COPD（慢性阻塞性肺病）很难医好，我们用运动，但他的效果好像快。〞
 <br/>
 <br/>
 香港市民孟先生：〝精彩，很多内容对我们健康有帮助。〞
 <br/>
 <br/>
 韩国公司代表JUSTIN：〝徐医生解释为何肺部这么重要。我本来以为我的肺干净，因为我不抽烟。但现在我知道我的肺不干净，所以我需要清洗它，以达到健康的目的。〞
 <br/>
 <br/>
 徐孝锡先生表示，他的技法传承自扁鹊药典。其独特的扁康疗法，47年至今治愈15万鼻炎、哮喘等患者。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙、余钢香港报导
</div>
