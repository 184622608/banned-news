### 港企涉伦敦金骗案 大陆受害人赴港请愿
------------------------

<div class="wysiwyg">
 【新唐人2018年09月21日讯】香港警方破获近期多宗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伦敦金骗案.html" target="_blank">
  伦敦金骗案
 </a>
 ，受害的不仅有香港人，还有大批中国和海外苦主。超过60名大陆伦敦金受骗苦主，星期五到香港礼宾府、警察总部请愿，要求政府立法规管贵金属交易。
 <br/>
 <br/>
 口号：严惩骗徒 绳之以法
 <br/>
 <br/>
 游行队伍早上由皇后像广场出发，先后游行至礼宾府、警察总部请愿递信。今次是至今最大一批请愿活动，代表逾22万宗骗案的受害人，涉骗金额高达250亿港元。当中有36间香港公司牵涉到内，透过在大陆聘请过百名中介诈骗。
 <br/>
 <br/>
 伦敦金苦主大联盟总召集人杨先生：〝因为讯息和网络太过发达，有的利用(微信)摇号，网路识人，社交平台，去骗取苦主的金钱。受骗金额个人达致几百至一千万。〞
 <br/>
 <br/>
 他们要求政府立法规管贵金属场外买卖，以保障投资者。一批苦主还到位于上环的金银业贸易场抗议，要求讨回公道，双方一度发生冲突，有一名大陆苦主报称受伤。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
 <br/>
 <br/>
