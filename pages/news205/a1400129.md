### 封公民广场被判违宪 民主派要港府道歉
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】香港前特首梁振英在2014年将被称为为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公民广场.html" target="_blank">
  公民广场
 </a>
 的政府总部东翼前地加设围栏和限制市民进入示威抗议，并因此触发雨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伞运.html" target="_blank">
  伞运
 </a>
 动。高等法院日前裁定港府的做法违宪。民主派要求港府立即拆除围栏和取消限制，并向黄之锋等伞运人士道歉。
 <br/>
 <br/>
 退休摄影记者张德荣2014年不满港府围封〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公民广场.html" target="_blank">
  公民广场
 </a>
 〞，申请司法覆核挑战港府的决定。四年后，高等法院星期一作出裁决，指港府的做法违宪。特首
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林郑月娥.html" target="_blank">
  林郑月娥
 </a>
 星期二出席行会时回应说，尊重法院裁决，但需要时间研究判词，已要求行政署检视如何改善守则。
 <br/>
 <br/>
 特首林郑月娥：〝毕竟是政府总部，很多同事及市民出入前来开会，所以如何能够尊重市民表达意见自由、权利，亦能够可以保障到政府总部有效安全、顺利运作是一个平衡工作。〞
 <br/>
 <br/>
 民主派九龙西补选候选人李卓人批评林郑月娥毫无悔意。
 <br/>
 <br/>
 补选候选人李卓人：〝大家记得当年双学三子及很多示威者入了公民广场亦受了刑责，现在他们受完苦受完刑责，现在林郑出来，输了官司只是说要看看字眼看看裁决，我恐怕她接下来仍然‘死鸡撑饭盖’继续要上诉。〞
 <br/>
 <br/>
 民主派议员呼吁港府放弃上诉，尽快拆除围栏及向重夺公民广场案入狱的双学三子道歉。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
