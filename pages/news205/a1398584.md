### 非洲猪瘟攻陷16省市 输港猪肉涨一成
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】据中共农业部星期四披露，福建莆田市、湖南省涟源市、吉林省龙井市和江西省万年县均发生一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 案例，令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 版图扩大到中国16个省、直辖市，共计53例。受猪瘟影响，大陆输
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港猪肉价.html" target="_blank">
  港猪肉价
 </a>
 钱近期涨一成。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 来势汹汹，不仅攻陷大陆16个省市，甚至网传死猪丢路边〝骇人〞录像，令人担心
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 〝比想像更严重〞。有猪肉业界表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港猪肉.html" target="_blank">
  香港猪肉
 </a>
 有95%均从大陆输入，供港猪场至今没有爆发非洲猪瘟，但肉价涨了一成。
 <br/>
 <br/>
 猪肉行总商会理事长许伟坚：〝因为国内这么多省市来说，发生猪瘟之后，猪场杀猪方之后，一段时间之后才会再饲养，变成国内需求多了，紧张了，影响到香港这两个月猪肉价钱上升10%左右。〞
 <br/>
 <br/>
 有猪肉档职员表示，生意没有受影响。市民则说，会减少去大陆。
 <br/>
 <br/>
 猪肉档职员：〝来香港给香港人的有信心保证，香港是信得过的。〞
 <br/>
 <br/>
 香港市民1：〝香港有安检的，有安检不担心。〞
 <br/>
 <br/>
 记者：〝去大陆玩吃猪肉呢？〞
 <br/>
 <br/>
 香港市民1：〝煮熟一些。〞
 <br/>
 <br/>
 香港市民2：〝香港不用担心。〞
 <br/>
 <br/>
 记者：〝回去大陆呢？〞
 <br/>
 <br/>
 香港市民2:〝现在没有回去。〞
 <br/>
 <br/>
 香港市民3：〝这一两年我都很少回去。回去后我都比较少吃，没必要时，我都少搞这些，少回去。〞
 <br/>
 <br/>
 记者：〝对他们食物不放心？〞
 <br/>
 <br/>
 香港市民3：〝都有一些。〞
 <br/>
 <br/>
 立法会议员郭家麒呼吁港府，除加强卫生健康教育外，食物安全中心一定要严格把关，避免问题猪只入港。
 <br/>
 <br/>
 新唐人记者梁珍、陈仲明香港报导
 <br/>
 <br/>
</div>
