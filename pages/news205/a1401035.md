### 涉基因编辑婴儿 中共科学家赴港演讲惹争议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】深圳
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科学家.html" target="_blank">
  科学家
 </a>
 贺建奎，日前声称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 ，令一对双胞胎对爱滋病病毒免疫，引发医学道德伦理争议。贺建奎原定星期二出席香港会议，并在次日发表演讲。不过，与会的基因专家批评实验不道德，呼吁停止做
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胚胎.html" target="_blank">
  胚胎
 </a>
 的基因编辑。
 <br/>
 <br/>
 香港一连三天举行〝人类基因组编辑国际峰会〞，峰会前一天，深圳学者贺建奎声称已编辑婴儿，引发轩然大波。与会的美国科学院院士张锋，早前以声明表示忧心，并要求停止做
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胚胎.html" target="_blank">
  胚胎
 </a>
 的基因编辑。他期望今次峰会能够就此讨论。
 <br/>
 <br/>
 美国科学院院士张峰：〝我想在新技术的应用方面，有很多考虑因素需要考虑，我不是监管方面的专家，但是我认为这样的会议可以带来很多不同的声音，不同的观点。在开始思考我们如何确保基因编辑等新技术能够最成功和负责任地开发时，这一点非常重要和有用。〞
 <br/>
 <br/>
 有港大同学也反对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 ，批评不道德。
 <br/>
 <br/>
 香港大学彭同学：〝以后这个技术发达了，那总会有有钱的人，比如说他们把自己的孩子改造了，智力更优秀，身体健康比穷人家的孩子更强，那这样子的话，社会的阶级分化就会愈来愈严重。所以我觉得这个伦理问题非常严重。〞
 <br/>
 <br/>
 贺建奎没有出席首日开幕会议，但主办方预计他会继续周三的演讲。
 <br/>
 <br/>
 中共一直在做各种违反人类伦理的实验，继换头、克隆猴之后，今次基因编辑婴儿事件，已引起上百名中国学者联署谴责。
 <br/>
 <br/>
 新唐人记者梁珍、黄晓翔香港报导
 <br/>
