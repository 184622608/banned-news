### 港珠澳大桥陆客过多 团体发起〝光复〞行动
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 通车后，连续三个周末数以万计的大陆游客，蜂拥而至临近城市〝东涌〞，给当地居民的生活带来困扰。周日，当地团体发起〝光复〞行动，要求警方取缔〝黑工〞导游，严正执法。
 <br/>
 <br/>
 伞后组织〝东涌人〞星期日下午在东涌东荟城附近发起〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_光复东涌.html" target="_blank">
  光复东涌
 </a>
 〞行动，寻找和举报黑工导游，并规劝大陆游客的不雅行为，不会针对遵守秩序的游客。
 <br/>
 <br/>
 〝光复东涌〞行动发起人王进洋：〝他们的行为包括吐口水、吐痰、将垃圾扔在地上，将奶粉完全扫得一干二净，面对这个问题我们很希望今次的光复行动可以第一，针对一些不明来历的旅行团及一些滥用香港法律漏洞及灰色地带，俗称一导游多团这些情况，第三，今次我们这个光复行动绝对不会与暴力这两个字挂勾。〞
 <br/>
 <br/>
 有东涌居民不满自大桥通车后，大量大陆游客涌入，滋扰原本宁静的生活，认为事件的源头在于政府。
 <br/>
 <br/>
 东涌居民梁小姐：〝不关她们的事，根本上是政府的政策，大陆和香港的政策非常不好，你在开这条桥之前没有想清楚吗？是不是？没有想到今天这种情况出现？〞
 <br/>
 <br/>
 亲共的立法会议员叶刘淑仪约下午3时到场视察，遭参与〝光复东涌〞的人士包围，批评她当年支持兴建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 ，出卖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港人.html" target="_blank">
  港人
 </a>
 。叶刘淑仪最后在警方的重重保护下离开现场。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
 <br/>
 <br/>
</div>
