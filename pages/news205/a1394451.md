### 港民团抗议当局拒绝英媒记者签证
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港政府.html" target="_blank">
  香港政府
 </a>
 上周五拒绝香港外国记者会第一副主席马凯续领工作
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_签证.html" target="_blank">
  签证
 </a>
 申请，多个团体到入境处抗议当局打压言论自由。
 <br/>
 <br/>
 口号：反对打压言论自由、秋后算帐、政府可耻。
 <br/>
 <br/>
 民阵及社民连等团体约20多人，到湾仔入境事务大楼外抗议港府践踏新闻言论自由，做法像中共。
 <br/>
 <br/>
 民阵召集人岑子杰：〝我们见到今日的特区政府跟大陆政府一样有妄想被害症，觉得每个人都要分裂它的国土一样，要所有的传媒人士要靠边站，要站在政府的立场不能有所偏差。
 <br/>
 <br/>
 立法会议员许智峯：〝是对着新闻自由的正面挑战，我们以为只有在一国一制的地方，只有在共产政权的地方才有这种情况，可以因为你的言论、因为你的政见，连工作都受影响，连工作的机会都不给你。〞
 <br/>
 <br/>
 社民连梁国雄认为打压言论自由是中共〝国耻〞，民阵表明本月中会到日内瓦，向联合国反映港府打压新闻自由。另外，四个新闻界团体将在周一到政府总部抗议，要求特首作出解释。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
