### 贸战及人币拖累 港周大福等金店销售降
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】受贸易战及人民币下跌等影响，香港知名的周大福、六福等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金店.html" target="_blank">
  金店
 </a>
 ，纷纷预告下半年增长放缓。
 <br/>
 <br/>
 周大福公布业绩，半年度收入达297亿港元，按年增加两成，但受人民币贬值影响录得3亿亏损，令其纯利仅增8.8%。不过，管理层称，受外围因素影响，近两个月同店销售增长有所放缓。
 <br/>
 <br/>
 周大福董事总经理黄绍基：〝10月、11月都看到有放慢的增长迹象，因为市场有很多我们不可控的消息，我们没办法控制，我们仍然维持审慎乐观的态度。〞
 <br/>
 <br/>
 周大福当天收市跌6.9%，股价曾创一个月低位。另一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金店.html" target="_blank">
  金店
 </a>
 ，六福（0590）亦指出，踏入下半年港澳及内地同店销售均下跌，预示下半年零售寒冬或会重临。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
 <br/>
 <br/>
</div>
