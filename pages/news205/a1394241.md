### 港数民团将赴联合国 抗议中共打压人民
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 人权理事会将在下月就中国的人权状况进行第三轮普遍定期审议工作，中国维权律师关注组等团体将派代表出席下周在瑞士日内瓦举行的前期会议，促联合国施压
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 改善国内人权。
 <br/>
 <br/>
 身兼支联会及中国维权律师关注组主席的何俊仁批评，近年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 打压律师不遗余力，等同向法律开战，所谓〝依法治国〞只是幌子。如王全璋被羁押超过1,100日仍未受审或释放。
 <br/>
 <br/>
 中国维权律师关注组主席何俊仁：〝大家看到这个政权对人民极度的镇压，其实是回到极权时代，不止是一个专制政府，是一个走向极权的政府，它对人民的自由打压是无底线，完全是不文明，是一个反文明的制度。我们今次去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 会用这些个别个案去显示人权状况的严峻。〞
 <br/>
 <br/>
 他指中共至今仍未批准实施《公民权利和政治权利国际公约》，违反对国际社会作出的承诺。他要求国际社会关注并施压中共，释放所有被羁押维权律师。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
