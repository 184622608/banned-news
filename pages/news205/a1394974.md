### 扁康院长徐孝锡首访香港 掀韩医热潮
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩医.html" target="_blank">
  韩医
 </a>
 热席卷台港，以清肺闻名的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扁康韩医院院长.html" target="_blank">
  扁康韩医院院长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐孝锡.html" target="_blank">
  徐孝锡
 </a>
 ，继前一天在台北讲座引起热潮后，星期四又抵达第二站香港，将举行讲座介绍其独特的扁康疗法。
 <br/>
 <br/>
 被韩国人称为扁鹊再世的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扁康韩医院院长.html" target="_blank">
  扁康韩医院院长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐孝锡.html" target="_blank">
  徐孝锡
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首度访港.html" target="_blank">
  首度访港
 </a>
 。人未到已掀起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩医.html" target="_blank">
  韩医
 </a>
 的热潮，星期六的讲座数日前已全部爆满。
 <br/>
 <br/>
 前一天才完成台北讲座，徐孝锡次日赶抵香港，在大纪元新唐人媒体集团副总裁陪同下，步出机场。首次到访香港，他对香港之行充满期待。
 <br/>
 <br/>
 韩国扁康韩医院院长 徐孝锡：〝这次来港是要将清肺的新的想法，让香港市民听完讲座，引起共鸣的话，这就是新的健康之道。我就是带着这个期盼来到香港。〞
 <br/>
 <br/>
 大纪元新唐人媒体集团副总裁 冯晋：〝越来越多的人认识到，现代真正的传统医学，才能够解决人的健康问题。
 <br/>
 <br/>
 徐孝锡被韩国版福布斯杂志称为〝韩方世界化的领头人〞，46年来曾治愈15万鼻炎、哮喘等患者，遍及世界30多个国家，被评选为患者最想见到的韩医师之一。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
