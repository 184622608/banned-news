### 港媒选前撑建制派 或违反当地选举法
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】今次九龙西补选日当天，中共喉舌《大公报》头版全版撑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_建制派.html" target="_blank">
  建制派
 </a>
 候选人陈凯欣，并免费派发。工党怀疑有人违反相关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选举法.html" target="_blank">
  选举法
 </a>
 例及申报经费之规定，到廉政公署总部报案，要求廉署进行调查。
 <br/>
 <br/>
 数名工党成员及新民主同盟立法会议员范国威星期四下午到位于北角的廉政公署总部报案。他们指《大公报》及《龙周》二份报纸，在九龙西补选日当天的头版撑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_建制派.html" target="_blank">
  建制派
 </a>
 候选人陈凯欣，质疑违反
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选举法.html" target="_blank">
  选举法
 </a>
 例。
 <br/>
 <br/>
 工党副主席麦德正：〝这样的刊物是没有做选举申报，他到底用了多少钱？没有人知。其实我们正研究其他的报章有无类似的范例，这个是犯了选举条例，所以今天我们来这里是进行严正的举报。〞
 <br/>
 <br/>
 立法会议员范国威：〝摆明是助选，摆明是要干预选举结果，也摆明候选人没有做出正式正确合理的申报，这些是涉及大量的选举经费，是违规的、不合理的，是下三滥的手段，我们不能够容忍、不能够接受。〞
 <br/>
 <br/>
 虽然选举已有结果，他们强调为了守护香港选举廉洁，若再发现有违反选举条例的证据会再到廉署举报。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
