### 港学者批建制派向北京投降 加速制度败坏
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】早前遭中共全国政协副主席、前特首梁振英入禀控告的理大助理教授钟剑华，周一表示，香港制度加速破坏，源于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_建制派.html" target="_blank">
  建制派
 </a>
 团体向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 投降。
 <br/>
 <br/>
 理大应用社会科学系助理教授和社会政策研究中心主任钟剑华指，近年来香港各方面都在收紧，以政改为例，原本以为2012年有双普选，之后由五步曲变成三步曲，结果搞出一个人大831方案，普选遥遥无期，皆因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_建制派.html" target="_blank">
  建制派
 </a>
 团体人士向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 投降。
 <br/>
 <br/>
 理大助理教授钟剑华：〝一些特别是工商界、左派机构团体、工会，以致于专业界别，他们真的不介意帮北京做打手。他们的投降主义在香港，或是完全放软手脚，北京说什么就是什么的这种态度加快香港制度的破坏。〞
 <br/>
 <br/>
 对于有论调指港人发起的占领行动激怒中共，以致北京收紧对香港的政策，他认为这是借口。
 <br/>
 <br/>
 理大助理教授钟剑华：〝有这些借口就好像比较冠冕堂皇，可以动员一些人抓住一套论述，为这些做法鸣锣开道。也可以欺骗一些基层的民众，我觉得这是它们20多年来一贯的策略。〞
 <br/>
 <br/>
 钟剑华呼吁当权者不要依仗制度，做得太过火。尤其是特首应有胸襟接受批评。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
