### 刘小丽代表民主派报名参选九龙西补选
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】香港立法会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九龙西补选.html" target="_blank">
  九龙西补选
 </a>
 提名期由10月2日展开，因中共人大释法导致宣誓无效丧失议员资格的工党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘小丽.html" target="_blank">
  刘小丽
 </a>
 代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 率先报名。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘小丽.html" target="_blank">
  刘小丽
 </a>
 在多位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 立法会议员的陪同下上午11时半到九龙城政府合署，递交约二千个市民提名，率先报名参选。
 <br/>
 <br/>
 工党刘小丽：〝香港现在面对很大的政治风暴，保皇党酝酿修改议事规则，23条杀到来，绝对破坏力比山竹过次无不及，他们拥有天文数字压迫香港人，以丰厚资源打压，究竟香港人是否这此屈服，我希望香港人能问是非分真假。〞
 <br/>
 <br/>
 至于一直被批评是分薄刘小丽票源的前民协成员、立法会前议员冯检基则在下午报名参选。多次反口不参选又参选的冯，反称被抹黑，现场有市民批评他受中共利用。
 <br/>
 <br/>
 另外，被视为建制派代表的食卫局前政治助理陈凯欣，则在亚皆老街球场宣布参选，早前在红磡展示巨型广告牌，被批评提早偷步宣传。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
