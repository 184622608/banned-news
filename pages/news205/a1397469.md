### 武侠小说家金庸去世 生前曾后悔写小说
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】10月30日晚，香港知名的武侠
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小说.html" target="_blank">
  小说
 </a>
 家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金庸.html" target="_blank">
  金庸
 </a>
 去世。陆媒披露，金庸生前曾后悔当初选择写小说，原因是〝写小说，对自己却没有什么好处。〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金庸.html" target="_blank">
  金庸
 </a>
 （原名查良镛），10月30日下午在香港位于跑马地的养和医院去世。终年94岁。
 <br/>
 <br/>
 金庸出生于1924年3月10日，浙江袁花镇，家世系浙江海宁查氏，其家为书香门第，金庸本人是查升之孙查揆的后裔。1948年移居香港。
 <br/>
 <br/>
 从1955年开始至1972年，查良镛以笔名〝金庸〞创作了十多部武侠
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小说.html" target="_blank">
  小说
 </a>
 ，代表作有《射雕英雄传》、《神雕侠侣》、《倚天屠龙记》、《天龙八部》、《笑傲江湖》、《鹿鼎记》、《雪山飞狐》、《书剑恩仇录》等。其中一些作品被翻拍成电影和电视连续剧。
 <br/>
 <br/>
 据不完全统计，金庸有超过3亿读者，但是陆媒报导说，金庸生前曾后悔当初选择写小说而不是搞学术研究。他说，〝写小说，娱乐的是别人，对自己却没有什么好处。〞
 <br/>
 <br/>
 1959年金庸创办了《明报》，1989年宣布辞去《明报》社长一职，1993年宣布辞任董事局主席，同年将《明报》集团售予香港媒体人于品海。
 <br/>
 <br/>
 1972年，《明报》宣布封笔后，慢慢地涉入政治。1973年春，金庸应中华民国政府邀请前往台湾，并与中华民国总统蒋经国会面。
 <br/>
 <br/>
 中共文化大革命结束后，金庸曾多次到访中国，并先后见了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 、赵紫阳等中共多名领导人。2009年担任中共作协第七届全国委员会名誉副主席。成为中共统战对象。
 <br/>
 <br/>
 鲜为人知的是，金庸父亲查枢卿，在中共篡政初期的〝镇反〞运动中惨遭杀害。1981年7月，时任中共最高领导人的邓小平接见金庸。期间邓小平主动提及查枢卿被处决之事，向金庸道歉。金庸点头说：〝人入黄泉不能复生，算了吧！〞
 <br/>
 <br/>
 据悉，从中共建政一直到改革开放初期，武侠小说作为一种文学门类在大陆被禁读。金庸被邓小平接见回到香港后，立即给邓小平专寄了一套《金庸小说全集》。也就是在邓小平会见金庸后不久，金庸小说在内地〝开禁〞，并很快成为畅销书。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉)
