### 港民团抗议特首风灾言论凉薄
------------------------

<div class="wysiwyg">
 【新唐人2018年09月20日讯】超强颱风〝山竹〞袭卷香港，社民连到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特首.html" target="_blank">
  特首
 </a>
 办
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 林郑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_风灾.html" target="_blank">
  风灾
 </a>
 言论冷淡，要求港府反省改善。
 <br/>
 <br/>
 约十名社民连成员星期四上午带同道具到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特首.html" target="_blank">
  特首
 </a>
 办
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 ，讽刺林郑月娥，听不见也看不见市民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_风灾.html" target="_blank">
  风灾
 </a>
 后的心声。社民连批评林郑风灾后只会向主子邀功，无视树木倒塌、道路阻塞，交通停顿，令普罗大众苦不堪言。还辩称无法理依据去宣布全港停工一天。
 <br/>
 <br/>
 社民连梁国雄：〝林郑对打工仔女浪费数小时受困路途中不单毫无关心，也没有任何行动即时解困，更对市民的怨言没有聆听之心，反说负气凉薄之言。.. 所关心者是一天停工对老板的经济损失，而不是市民最迫切灾后清理复元的工作。〞
 <br/>
 <br/>
 社民连要求林郑调查今次风灾善后处理，以及造成混乱的原因，向市民公开交待。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
