### 林和立：中美贸易战已上升到全面冷战
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】中美双方同意在明年一月一日起不会有新的贸易征税措施。香港时事评论员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林和立.html" target="_blank">
  林和立
 </a>
 在一论坛上指，中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 已升级到全面冷战，认为中美在G20的峰会上达成的协议已经不能解决两国之间的持续冲突。
 <br/>
 <br/>
 时评家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林和立.html" target="_blank">
  林和立
 </a>
 早前在一论坛上已预料中美双方在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上将达成协调，美方暂停明年一月向价值2000亿美元的中国产品增加额外关税。他指，达成协议对解决中美目前的冲突问题已经不起作用。
 <br/>
 <br/>
 时评家林和立：〝其他方面围绕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 的问题，包括南中国海，包括台湾，包括特朗普要打击中国高科技发展，亦牵涉香港，有关美国人会否继续让香港享受一个大陆以外的特别关税区，这个特别的宽容条件呢？这个亦已进入冷战的范畴。〞
 <br/>
 <br/>
 林和立又指，目前中国大陆的经济在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 中已经大受打击。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
 <br/>
</div>
