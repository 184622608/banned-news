### 谢霆锋分享港式家常菜 低调回中共限酬令
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】香港影帝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢霆锋.html" target="_blank">
  谢霆锋
 </a>
 醉心下厨，前年更获颁世界首位〝米其林之友〞，这次他出席公开活动，分享自己的家常菜。不过，平面媒体报导，谢霆锋被问到近来中共的限酬令时，态度低调不愿多谈。
 <br/>
 <br/>
 久未露面的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢霆锋.html" target="_blank">
  谢霆锋
 </a>
 ，这次回港和连锁快餐店合作，设计港式地道风味的汉堡包，被问到最爱的美食是什么？谢霆锋这么回答。
 <br/>
 <br/>
 香港明星 谢霆锋：〝全世界的人都会问，我自己的家常菜是什么？是什么大菜？我说其实不是什么大菜，而是我们在片场时，阿姐派给我的那盒饭盒。开出来通常是有很多道汗水的鱼香茄子饭，豉椒排骨，或者窝蛋免治牛肉饭。我今次也就我自己的想法，将窝窝蛋免治牛肉饭变成煎蛋免治肉酱包。〞
 <br/>
 <br/>
 谢霆锋这次更亲自创作MV《青空》，早前走遍香港各处取景，带出对香港家的思念及回忆熟悉的味道，期间在铜锣湾拍摄时更引起人潮。
 <br/>
 <br/>
 香港明星 谢霆锋：〝想起很多在香港的经历也好，我的成长也好，不知不觉都已经过了20多年，希望将整个Package，无论是你们的味道，你们的视觉、你们的听觉，都放在这个项目里面。〞
 <br/>
 <br/>
 另外，中共对明星开铡，祭出限酬令，规定艺人每季酬劳上限为1千万人民币。谢霆锋女友，艺人王菲被传因此将退回9千万人民币，对此，谢霆锋低调回应：〝不要乱说〞。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
</div>
