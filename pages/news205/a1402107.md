### 港民主派抗议特首政治审查 操控选举
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】香港立法会议员朱凯廸早前被选举主任取消村代表参选资格，民主派议员趁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特首.html" target="_blank">
  特首
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林郑月娥.html" target="_blank">
  林郑月娥
 </a>
 周三出席立法会质询大会，向她抗议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治审查.html" target="_blank">
  政治审查
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_操控选举.html" target="_blank">
  操控选举
 </a>
 ，特首答问大会首次被迫取消。
 <br/>
 <br/>
 口号：林郑政府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_操控选举.html" target="_blank">
  操控选举
 </a>
 可耻，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治审查.html" target="_blank">
  政治审查
 </a>
 可耻，思想审查可耻。
 <br/>
 <br/>
 民主派议员在会议厅门口列队拉起横幅抗议林郑政府无理取消朱凯廸议员参选乡郊代表选举的资格。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林郑月娥.html" target="_blank">
  林郑月娥
 </a>
 进入会议厅后，民主派议员继续高喊口号抗议，包括毛孟静等三位民主派议员先后被逐出会议厅，期间一度暂停会议。大约十五分钟后会议恢复，民主派继续抗议。梁君彦再作警告但无效，宣布取消会议。
 <br/>
 <br/>
 民主派强调面对中共与港府日趋严重的政治打压，会抗争到底。
 <br/>
 <br/>
 立法会议员朱凯廸：〝香港人的自由不需要向共产党磕头，不需要向林郑磕头，是港人本身就有的。我们何时变成要跪低过关才可履行我们的政治权利呢，香港人不应低头，民主派也不会低头。〞
 <br/>
 <br/>
 民主派会议召集人毛孟静：〝我们是说政治审查、思想审查可耻，这个林郑政府摆明就是要操控选举，这是绝对不能接受的。这种心态是百分之百将议会人大化。〞
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
