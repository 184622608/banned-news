### 韩企首赴港办招商会 打通国际贸易通道
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】限韩令以及中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易.html" target="_blank">
  贸易
 </a>
 战升温氛围下，越来越多的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 企业选择在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 招商，令香港和韩国之间的贸易增长相当可观。
 <br/>
 <br/>
 首次来港招商的是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全罗北道中小企业协会.html" target="_blank">
  全罗北道中小企业协会
 </a>
 ，今次协同旗下15个会员，带来数十种健康产品等，和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绿色东方.html" target="_blank">
  绿色东方
 </a>
 合作举行招商会并签订谅解备忘录。
 <br/>
 <br/>
 韩国全罗北道中小企业协会会长金正植：〝以前我们也参加过很多博览会，但很多都是流于形式，这次绿色东方起到了连接消费者和制造厂家之间的桥梁作用，我觉得这部分是最吸引人的地方。〞
 <br/>
 <br/>
 韩国全罗北道中小企业协会会员李汉揆：〝首先进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 ，然后再进入东南亚或中国市场，或是在香港取得成功后，进入国际市场，所以说这是个非常重要的通道。 〞
 <br/>
 <br/>
 活动吸引不少香港买家洽谈合作。
 <br/>
 <br/>
 买家谢小姐：〝他们找回来的商家，其实很多都很有理念。很多和我说，我不会理会成本，总而言之我希望将健康带给人。〞
 <br/>
 <br/>
 绿色东方市场部经理Germaine Tse：〝我们集团旗下有市场策划的业务，也有自己的零售店，所以刚好在这个时期，我们可以帮到海外企业很容易、一站式地开发自己的市场。〞
 <br/>
 <br/>
 香港和韩国双边
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易.html" target="_blank">
  贸易
 </a>
 去年达港币3087亿元，较前年上升了23.4%。香港同时也是中韩贸易的重要转口港，经香港的转口贸易2017年达港币2389亿元。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙、陈仲明香港报导
</div>
