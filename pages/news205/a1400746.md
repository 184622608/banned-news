### 港九龙西补选 民主派动员撑李卓人抗衡中共
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】香港星期日举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_立法会.html" target="_blank">
  立法会
 </a>
 九龙西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_补选.html" target="_blank">
  补选
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 全体总动员声援
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李卓人.html" target="_blank">
  李卓人
 </a>
 ，呼吁选民务必集中票源全投李卓人，对抗中共将香港变为大陆城市，选出能够捍卫香港一国两制，及守护香港核心价值的候选人。
 <br/>
 <br/>
 口号：〝选情严峻，争你一票〞、〝集中就赢，分散就输〞。
 <br/>
 <br/>
 代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 出战的工党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李卓人.html" target="_blank">
  李卓人
 </a>
 早上8时在九龙区石硖尾展开拉票活动，包括被政府剥夺参选资格的刘小丽、民主党主席胡志伟、公民党主席梁家杰及多位民主派
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_立法会.html" target="_blank">
  立法会
 </a>
 议员到场声援。李卓人强调今次选举关乎港人的未来。
 <br/>
 <br/>
 工党李卓人：〝二个未来的对决，两阵的对决，我李卓人是代表我们继续捍卫香港一国两制，我们的自由和法治、人权，争取我们应有的真普选的权利，五号的保皇党（陈凯欣）所代表的是只会配合中共打压，将香港变为另一个大陆城巿。〞
 <br/>
 <br/>
 民主派强调中共投入大量资源渗透社会各方面，只差一步就掌握港人命运。呼吁选民务必集中票源，投票支持李卓人。
 <br/>
 <br/>
 民主党主席胡志伟：〝3号李卓人是唯一的选择，也是民主派支持的唯一代表，如果大家不能集中票源的话，其实就是帮共产党在港的代言人。所以大家在港想安生乐命，所需要是一个能够真正维护一国两制、核心价值的候选人。〞
 <br/>
 <br/>
 九龙西有约49万选民，选民从早上七时半到晚上10时半，可到所属票站投票。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_补选.html" target="_blank">
  补选
 </a>
 其他候选人包括伍迪希、曾丽文、冯检基、陈凯欣。
 <br/>
 <br/>
 扒：今次补选是民主派关键的一票，关乎能否守住立法会分组点票的否决权。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报
