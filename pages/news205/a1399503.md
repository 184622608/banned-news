### 美拟重审视港关税地位 各界担忧促保两制
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】美中经济与安全审查委员会年度报告批评北京严重侵蚀
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 特区的自治以及言论自由，要求应重新审视香港独立关税区。来自多个界别的立法会议员担忧
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港府.html" target="_blank">
  港府
 </a>
 若不改变现状，香港未来堪虞。
 <br/>
 <br/>
 报告严厉批评北京干预
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 内部事务，违反〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一国两制.html" target="_blank">
  一国两制
 </a>
 〞、〝高度自治〞，情况令人忧虑，严重损害香港作为国际金融中心的地位，并举例如高铁香港段的〝一地两检〞决定、《国歌法》立法，以及拒绝续发《金融时报》亚洲新闻编辑马凯的签证等事件。建议美国商务部应考虑将香港和中国大陆视为同一个关税区域，限制潜在军用化的高科技输港。
 <br/>
 <br/>
 代表不同界别的立法会议员形容报告令人担忧，警告
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港府.html" target="_blank">
  港府
 </a>
 不能掉以轻心。
 <br/>
 <br/>
 资讯科技界立法会议员莫乃光：〝如果这些限制对于一些香港完全是本土的，香港资金香港公司香港投资人或本身在香港提供服务及研发的公司都受到同样的影响，对于我们现在希望不论帮助香港的经济发展创科行业等等都会有很大的影响。〞
 <br/>
 <br/>
 自由党党魁钟国斌：〝我期望他们尽快派人到华盛顿或在香港跟美国领事或驻香港的美商会尽快做一些游说工作或解释工作令这件事尽快平息，不要令它再烧下去，如果真当了香港做中国其中一个普通城市，香港可以说〝玩完〞。〞
 <br/>
 <br/>
 民主党则发表声明要求特首林郑月娥紧急评估今次的美国国会报告并提交报告给北京参考，尽力挽回香港的独特地位。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
