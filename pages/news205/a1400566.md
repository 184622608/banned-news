### 港民主派李卓人吁支持 批中共打压选情
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】香港九龙西立法会补选周日举行，民主派的唯一参选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李卓人.html" target="_blank">
  李卓人
 </a>
 周五宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选情告急.html" target="_blank">
  选情告急
 </a>
 ，呼吁选民集中票源支持他。
 <br/>
 <br/>
 口号：〝票投3号
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李卓人.html" target="_blank">
  李卓人
 </a>
 ，民主告急，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选情告急.html" target="_blank">
  选情告急
 </a>
 。〞
 <br/>
 <br/>
 李卓人星期五下午在在黄埔举行选举告急记者会，他指最近的两个民调都显示他未能取得足够的民主派支持者的选票，加上面临原本民主派人士冯检基倒戈分薄票源，令他选情严峻。
 <br/>
 <br/>
 立法会补选参选人李卓人：市民要明白，今次是民主和保皇的对决，民主和政府的对决，两阵对决集中就赢，分散就输。
 <br/>
 <br/>
 多个民主派政党代表到场支持，呼吁选民全力支持李卓人。
 <br/>
 <br/>
 香港众志秘书长黄之锋：仍然犹豫是否投票支持李卓人的年青人，多一席未必能扭转形势，但可以拖延北京政府对年轻一代的清算。
 <br/>
 <br/>
 李卓人又批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 动用庞大资源打压和抹黑他的选情。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
