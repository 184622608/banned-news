### 六新闻团体促港府 解释拒续马凯签证
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 六个新闻界组织星期一向特区政府请愿，要求解释
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外国记者会.html" target="_blank">
  外国记者会
 </a>
 副主席马凯工作签证不获续期的原因。批评此举损害香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新闻自由.html" target="_blank">
  新闻自由
 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_言论自由.html" target="_blank">
  言论自由
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 记者协会、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外国记者会.html" target="_blank">
  外国记者会
 </a>
 等六个新闻界组织的代表，到政府总部递交一万五千个签名。他们批评入境处拒绝为马凯工作签证续期，损害香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新闻自由.html" target="_blank">
  新闻自由
 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_言论自由.html" target="_blank">
  言论自由
 </a>
 ，并要求特首林郑月娥解释。
 <br/>
 <br/>
 香港外国记者会主席陈翡表示：〝我们不予置评，只是等待香港当局就此议题给予解释，这是政府的责任，我们只是在等待一个合理的解释。
 <br/>
 <br/>
 香港记协主席杨健兴：〝只是给一个平台，给不同意见的人表达，这是香港言论自由、新闻自由很重要的地方。……更加担心的是今次，起码是记协认为有政治报复、秋后算帐的问题。〞
 <br/>
 <br/>
 全体法律界选委发表声明，指特区政府不为马凯工作签证续期的决定是前所未有，对香港新闻自由有深远影响，应该得到完满的解释。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
