### 澳门中联办主任郑晓松自杀 疑点重重
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】中共国务院港澳办，周日突然在官网宣布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门中联办主任.html" target="_blank">
  澳门中联办主任
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 患有抑郁症，前一晚堕楼身亡。消息恰巧发生在外传习近平南巡，将出席下周二港珠澳大桥开通仪式前夕，令外界质疑。
 <br/>
 <br/>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门中联办主任.html" target="_blank">
  澳门中联办主任
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 因患抑郁症，堕楼身亡。香港支联会秘书长李卓人认为，因抑郁症跳楼死亡，令人联想背后到底有否其他原因，因近年来接连发生中共官员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀.html" target="_blank">
  自杀
 </a>
 或是被自杀的事件。
 <br/>
 <br/>
 香港支联会秘书长李卓人：〝根本所有共产党的官员，他们自己也是受到很多他们的压力，或者来自他过去所做的事，怕被人清算那种问题，而使他们抑郁，或甚至使他们自寻短见〞。
 <br/>
 <br/>
 由于港珠澳大桥将于星期二（23日）举行通车典礼，加上有传习近平南下主持典礼，李卓人认为事件不寻常。
 <br/>
 <br/>
 香港支联会秘书长李卓人：〝其实某程度也是对整个港珠澳大桥，亦都是一个控诉来的，控诉内容我们不知道，但是他死亡本身其实是一个控诉。〞
 <br/>
 <br/>
 郑晓松去年9月20日接替王志民任澳门中联办主任一职。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
