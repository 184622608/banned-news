### 强颱潭美原地打转 侵台机率低周五六最靠近
------------------------

<div class="wysiwyg">
 【新唐人2018年09月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强颱〝潭美〞.html" target="_blank">
  强颱〝潭美〞
 </a>
 目前缓慢往西北方移动，气象局指出，潭美今明两天呈现滞留状态，实际路径仍须观察，但从台湾东方海面北上机率较高，依目前预测，周五周六，是颱风最接近台湾的时候，北部东北部将有明显降雨。
 <br/>
 <br/>
 受到东北风影响，北部以及东半部，云量增多，有阵雨以及局部大雨发生的机率，中南部天气比较稳定，而在温度上，已经稍微有些凉意，尤其在北部东北部，中南部则是比较闷热，外界也高度关注，强颱潭美的最新动态。
 <br/>
 <br/>
 气象局简任技正伍婉华：〝目前各国模式预测，我们分析从东方海面北上机率是比较高的，但是北上过程中，距离我们多近，必须看到时候太平洋高压甚至北方潮汐牵引力量，这部分是必须观察的。〞
 <br/>
 <br/>
 25日上午8时，潭美颱风来到鹅銮鼻东南东方880公里海面，缓慢往西北移动，气象局表示，今明两天潭美的导引气流不明显，中心近似滞留，对台湾的实际影响仍需观察。
 <br/>
 <br/>
 气象局简任技正伍婉华：〝以目前预测，我们预估星期五到星期六是潭美颱风最接近我们的时候，接近过程中，在星期五星期六，北部东半部地区的降雨情况，尤其北部东北部降雨情形就会明显，有可能会有大雨或局部性豪雨。这是目前判断路径，未来随着颱风路径改变，北台湾的降雨程度也会有所调整。〞
 <br/>
 <br/>
 至于是否最快周四发布海上警报？气象局表示，由于颱风直接侵袭台湾机率降低，北上时距离台湾的远近也仍需观察，目前持保留态度，认为重点应该放在颱风外围环流，对台湾北部以及东北部的影响程度。
 <br/>
 <br/>
 新唐人亚太电视王冠霖、张媛婷台湾台北采访报导
