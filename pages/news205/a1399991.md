### 【禁闻】香港雨伞运动案开审 国际关注
------------------------

 【新唐人2018年11月19日讯】2014年

  香港
 </a>
 民众争取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_真普选.html" target="_blank">
  真普选
 </a>
 的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雨伞运动.html" target="_blank">

 </a>
 〞，震撼国际社会，时隔四年之后，9名〝雨伞运动〞的参与者，被控〝公众妨扰〞等罪名，11月19号在西九龙法院受审。
 <br/>
 <br/>
 九名被告人包括当年〝占领中环〞运动的三名发起人戴耀廷、陈健民与朱耀明，以及时任学生领袖张秀贤、钟耀华等。他们9人都在法庭上否认全部控罪。
 <br/>
 <br/>
 100多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 市民带着〝占领行动〞的象征物黄色雨伞、还有和平占中等标语，在法庭外声援。
 <br/>
 <br/>
 案件也引起国际关注，英国下议院多名议员谴责香港政府的起诉行为，一些国际人权组织也批评这次审判威胁香港市民的表达自由。
 <br/>
 <br/>
 戴耀廷则表示，香港的高度自治和法治也与他们一起同时被审判。
 <br/>
