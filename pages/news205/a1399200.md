### 中共军队违法占用土地 港人促当局交代
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】香港传真社日前报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳边防部队.html" target="_blank">
  深圳边防部队
 </a>
 涉嫌非法占用香港边境的土地长达六年，其中涉及建桥、跨境执法、倒垃圾。民主派政党质疑港府失职，督促当局向公众交代。
 <br/>
 <br/>
 据传真社报导，深圳边防人员自2012年起涉嫌越境占用香港土地，更建造桥梁方便进入香港境内，弃置垃圾。香港公民党星期二到到政府总部示威抗议，质疑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳边防部队.html" target="_blank">
  深圳边防部队
 </a>
 人员违反《基本法》对特区划界的规定，并不满政府相关部门长达6年都没有执法。
 <br/>
 <br/>
 立法会议员郭家麒：〝这些解放军代表仍可继续施施然说，是啊，可以行去缓冲区，可以拘捕非法入境者或罪犯，即原来它可在香港界内执法，这是相当严重和离谱的事，根据传媒利用一些卫星图片，这个占用土地长达6年，今次事件是解放军赤裸裸地侵犯和伤害一国两制。〞
 <br/>
 <br/>
 特首林郑月娥出席行政会议前回应有关事件，她表示有关部门正在研究此事，若有结果稍后会对外公布。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
