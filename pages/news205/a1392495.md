### 刘小丽参选西九补选 港民主派全力支持
------------------------

 </a>

  参选

 ，她强调今次是关键一席，取回此席，才能阻挡议事规则再一次被修改，抵抗不公不义的政权。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘小丽.html" target="_blank">
  刘小丽
 </a>
 在深水埗街头举行记者会正式宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参选.html" target="_blank">
  参选
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九龙西立法会.html" target="_blank">
  九龙西立法会
 </a>
 补选，她强调港人需要真正的港人治港，而非西环（中联办）治港，因此是非常关键的一席，以阻止建制派再次修改议事规则，捍卫良好制度。

 <br/>

 <br/>
 <br/>
 包括民主党、公民党及香港众志等十多位立法会议员及人士到场支持。
 <br/>
 <br/>

 <br/>
 <br/>
 刘小丽即日收集市民的提名，并将在10月2日提名期首日交提名。
