### 港媒实测：中共居住证 外洩个资风险极高
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】之前在台湾引发的高度争议的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共居住证.html" target="_blank">
  中共居住证
 </a>
 ，港媒最近实测发现，中共最近发放的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港澳台居民.html" target="_blank">
  港澳台居民
 </a>
 居住证，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外洩个资.html" target="_blank">
  外洩个资
 </a>
 风险极高，用大陆市面上可买到的读卡器，就可隔空读取；领卡的8万名港澳居民、几万名台湾人都深陷隐私危机。
 <br/>
 <br/>
 长居广东的香港人 张敏锐：〝很惊讶，都会很危险，我们刚才很清楚看到正反面，所有资料像是复印出来，还要地址、证件号码，相都可以独立拆出来，可以(被用来)做很多事情。〞
 <br/>
 <br/>
 民进党立委 林俊宪：〝它(中共)有很多你难以想像，在黑暗面的事情，像这种个资的泄漏，我觉得一点都不惊讶。中共也在世界，进行很多间谍性的行为。那这样东西如果外流出来，我想这种状况，恐怕会比我们看到的更严重。国安问题，而且是全世界共同关注的问题，台湾应该更要提高警觉才对。〞
 <br/>
 <br/>
 国民党立委 李彦秀：〝这个在香港其实还是违法的，我相信香港的相关单位，也会有一定的处理方式。在大陆或香港，官方对于这个侵害隐私的部分，要做进一步的措施、行政措施跟处理。〞
 <br/>
 <br/>
 港媒发现，有读卡器隔空六公分，仍可读取居住证资料、即时列印。目前，至少超过八万名领证港澳居民、几万名台湾居民，个人资料随时可能被曝光。
 <br/>
 <br/>
</div>
