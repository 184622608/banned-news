### 林郑与习并肩同行 韩正反随其后有何玄机？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 23日出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 揭幕仪式，他在台上仅说一句话便匆匆离开。但在习入场时罕见与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港特首.html" target="_blank">
  香港特首
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林郑月娥.html" target="_blank">
  林郑月娥
 </a>
 并肩进场，而中共港澳工作协调小组组长韩正等人反随其后，引起政界热议。
 <br/>
 <br/>
 耗资1200多亿元人民币，工程历时近9年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 ，10月23日举行通车仪式，由中共国家发改委主任何立峰负责主持，主礼嘉宾除
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 、韩正外，还有中共中央书记处书记、中央办公厅主任丁薛祥、负责经济的国务院副总理刘鹤等人。
 <br/>
 <br/>
 然而作为香港行政长官的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林郑月娥.html" target="_blank">
  林郑月娥
 </a>
 ，虽然官阶低过现场一众具政治局常委，和政治局委员头衔的官员，但当主礼嘉宾进场时，一身粉红衣着的林郑月娥，笑着与习近平并肩〝同行〞进场。
 <br/>
 <br/>
 而负责港澳事务的中共国务院副总理韩正，则与澳门特首崔世，被安排在林郑月娥与习近平之后一同进场，引起政界热议。
 <br/>
 <br/>
 虽然林郑月娥称，自己是主礼团唯一女性，中共副总理客气让她先行，但有港媒认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港特首.html" target="_blank">
  香港特首
 </a>
 的排位，如按规矩，任何情况都不可能与习近平并肩而行。
 <br/>
 <br/>
 《星岛日报》引述港府消息人士的话表示，23日大桥开幕仪式，有数百名贵宾及记者出席，在这样的场合，中共中央安排林郑月娥与习近平并排出场，明显是有政治含意：一是，为了凸显中共中央对港澳的重视；二是，显示中共中央对林郑月娥的支持。
 <br/>
 <br/>
 报导分析，中共政治的不成文〝规矩〞，领导人出场时，作为第一把手的国家主席一般都会行在前头，其他领导人与他保持一定的距离，而当天韩正等领导人在出场后，亦放慢脚步，与习保持一定距离。而林郑与习亦步亦趋与习并肩同行，被中国媒体记者认为，是因为林郑月娥不熟悉国情。
 <br/>
 <br/>
 香港苹果则引述熟悉中共政治的评论员分析，认为虽然中共中央刻意安排在镜头前厚待林郑月娥，凸显香港与内地融合，有挺港挺林郑月娥意味，但是，林郑不要飘飘然，北京为她立威，很可能要求她日后推动民意，为接受中共的23条立法、落实北京要香港融入中共发展政策提供支持。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：戴明）
</div>
