### 〝川习会〞前中共软化 美航母〝里根号〞访港
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】中美首脑下周于G20峰会见面前，美国核动力航空母舰〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根号.html" target="_blank">
  里根号
 </a>
 〞连同另外3艘军舰，星期三抵达香港访问。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根号.html" target="_blank">
  里根号
 </a>
 是美国第七舰队的主力舰，是尼米兹级核动力航空母舰中最大、最新的一艘。星期三早上九点，里根号和其他三艘军舰抵达香港，停泊于青衣对出海域，本台和其他50多名中外媒体获邀上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_航母.html" target="_blank">
  航母
 </a>
 采访。
 <br/>
 <br/>
 两天前，〝里根〞号航母在菲律宾海海域进行了双航母演习，对于今次访港，〝里根号〞司令官不肯透露何时离开，但重申是为了加强区内的安全和稳定。
 <br/>
 <br/>
 列根号司令官海军少将Karl Thomas：〝美国海军会继续在印太地区存在。海军军舰访港，是要加强印太地区安全和稳定，如同这一次，从而为整个地区包括美国带来和平与繁荣。〞
 <br/>
 <br/>
 记者：里根号共有5000名士兵，满载时可载飞机70架。
 <br/>
 <br/>
 对于即将登陆香港，船上官兵们，尤其是约30名华裔军人表现兴奋。
 <br/>
 <br/>
 广东出身的里根号下士江军：〝这里风景很漂亮，东西好吃，因为是第一次来，所以蛮开心兴奋的。〞
 <br/>
 <br/>
 九七之后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 至少拒绝过四次美舰访港，包括今年9月〝黄蜂号〞的申请。今次里根号成功访港，被视为习特会前，中共态度软化的信号。香港政府更一连五天设立禁飞区，迎接美国舰队群。不过，专家分析美方并不会因此让步，双方在贸易战上达成实质性结果的机会渺茫。
 <br/>
 <br/>
 新唐人记者梁珍、陈仲明香港报导
 <br/>
 <br/>
