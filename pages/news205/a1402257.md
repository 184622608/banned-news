### 因一带一路贿非洲政要 何志平罪名成立
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】香港民政事务局前局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_何志平.html" target="_blank">
  何志平
 </a>
 被控贿赂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲官员.html" target="_blank">
  非洲官员
 </a>
 案，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国纽约.html" target="_blank">
  美国纽约
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联邦法院.html" target="_blank">
  联邦法院
 </a>
 裁定七项罪名成立，明年三月判刑。有议员指出此事件涉及香港智库及银行，要求廉政公署介入调查。
 <br/>
 <br/>
 曾任廉署调查主任的民主党立法会议员林卓廷表示，根据控罪书及种种证据显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_何志平.html" target="_blank">
  何志平
 </a>
 贿赂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲官员.html" target="_blank">
  非洲官员
 </a>
 案，香港的角色是非常关键。因何志平以〝中华能源基金会〞副主席接触联合国官员，是中国华信集团的非公务基金，在香港注册。同时部分贿款是经香港汇丰银行转到美国。有理由认为在香港有人或公司协助，甚至教唆何志平进行贪污，而相关资金源自香港，他要求廉署介入调查。
 <br/>
 <br/>
 立法会议员林卓廷：〝所以我认为香港的廉政公署以至警方有法定的权力和职责去全面调查何志平今次案件是否有任何香港的公司或者人士涉案，当中是否有涉嫌协助教唆贿赂或洗黑钱的情况。香港作为国际金融中心，一定要非常小心避免给国际社会以为香港成为国际贪污行贿以至洗黑钱的中心。〞
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
