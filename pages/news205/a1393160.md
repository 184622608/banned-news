### 港民团纪念伞运四周年 续争真普选
------------------------

<div class="wysiwyg">
 【新唐人2018年09月27日讯】本月28日是香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雨伞运动.html" target="_blank">
  雨伞运动
 </a>
 四周年，多个团体计划在当日举行不忘伞运精神活动，强调在中共及特区政府的强权下，继续争取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主普选.html" target="_blank">
  民主普选
 </a>
 。
 <br/>
 <br/>
 〝六一七民间约章〞、新民主同盟等团体在金钟连侬墙举行记者会，宣布将在周五、即
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雨伞运动.html" target="_blank">
  雨伞运动
 </a>
 四周年举行纪念活动，主题是〝坚持雨伞精神、我们是香港人〞。
 <br/>
 <br/>
 新民主同盟立法会议员范国威：〝为了香港人的福祉、为了香港人的未来。所以每一年都在站出来，告诉港人我们没有放弃、我们要坚持下去，坚持下去才有希望。〞
 <br/>
 <br/>
 立法会议员毛孟静批评中共以文攻武吓，欺骗港人，目的就是不给港人真普选。
 <br/>
 <br/>
 立法会议员毛孟静：〝从DQ民主派议员开始，之后一路不给参选剥夺参选资格，到现在取缔政党，何时轮到其他民间团体，因此不得不站出来。雨伞精神我们争取民主是理直气壮，坚持到底。〞
 <br/>
 <br/>
 纪念活动将在周五下午四时于金钟连侬墙添美道举行，已有至少26个民间团体表示参与。并会在当年发射第一枚催泪弹的时间，下午5时58分默站三分钟。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
