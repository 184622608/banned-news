### 港民主派将访美 欲保香港特殊关税地位
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】美国国会上月发表报告，要求重新考虑是否取消
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 特殊的关税地位，引起政商界高度关注。多位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 议员将在本月中访问美国，向当地政商界人士表达维持香港作为独立关税区重要性。特首林郑月娥称不假手于人。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 立法会议员郭荣铿及涂谨申本月先后访美，游说美国保留
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 独立关税区。特首林郑月娥回应称尊重议员外访，但希望对方作出事实的陈述，不过她强调自己访美更有权威，不会〝假手于人〞介绍香港情况。
 <br/>
 <br/>
 特首林郑月娥：〝我本人代表香港，更有权威地说香港现在的情况。除了外访之外，我差不多每星期都接待很多外来访客，包括外国政府的官员、国会议员、商界领袖，这些工作我会继续做的。〞
 <br/>
 <br/>
 郭荣铿反驳林郑的说法，强调香港〝一国两制〞正在不断被削弱，外国看事实而非权威。
 <br/>
 <br/>
 立法会议员郭荣铿：〝人家一问你， 事实上你是赶马凯，事实上你是DQ议员，事实上你是DQ候选人，一而再再而三，事实见到香港的铜锣湾书店，事实见到一国两制和法治基础一直被这些事件削弱，你硬去讲你很有权威，人家信不信呢？〞
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
