### 支联会呼吁G20峰会各国领袖 施压中共
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 将于本周五在阿根廷举行，香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支联会.html" target="_blank">
  支联会
 </a>
 等团体星期二到中联办要求中共释放黄琦、王全璋及所有良心犯，呼吁参与峰会的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_各国领袖.html" target="_blank">
  各国领袖
 </a>
 ，关注大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权状况.html" target="_blank">
  人权状况
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支联会.html" target="_blank">
  支联会
 </a>
 、中国维权律师关注组及社民连等20多人，由西区警署游行至中联办抗议。身兼支联会及中国维权律师关注组主席的何俊仁表示，〝六四天网〞创办人黄琦入狱至今，健康状况很差，随时有生命危险。曾为法轮功辩护的著名维权律师王全璋失踪超过三年似人间蒸发，生死未卜，上庭遥遥无期。他表示将致函参与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_各国领袖.html" target="_blank">
  各国领袖
 </a>
 ，要求他们向中共领导人反映2人的情况。
 <br/>
 <br/>
 中国维权律师关注组主席何俊仁：〝中国政府如果仍然这样虐待、残酷对待自己的人民，这个国家是不值得尊重的。老实说将来的贸易无理由完全不看
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权状况.html" target="_blank">
  人权状况
 </a>
 ，无理由不看这个国家是否一个文明的国家，这是重要的〞。
 <br/>
 <br/>
 抗议人士将公开信、黄琦及王全璋头像贴在中联办外，也有人将示威牌扔入中联办内，然后和平散去。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
