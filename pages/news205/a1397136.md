### 港民团抗议中共打压宗教自由
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】自今年2月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 实施新修订的《宗教事务条例》后，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基督教会.html" target="_blank">
  基督教会
 </a>
 的打压和逼迫加剧。香港多个团体，周日来到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 抗议中共打压
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教自由.html" target="_blank">
  宗教自由
 </a>
 ，要求中共立即停止对教会的打压，以及释放所有被捕的教牧同工。
 <br/>
 <br/>
 基督徒关怀香港学会等约25人下午3时从西环西区警署游行到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 ，抗议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 打压
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教自由.html" target="_blank">
  宗教自由
 </a>
 。今次守护宗教自由游行发起人之一刘志雄牧师表示，中共除大规模的打压全国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基督教会.html" target="_blank">
  基督教会
 </a>
 ，即使早前中梵签署临时协议，天主教一样继续受到打压。他强调中共打压宗教自由不是孤立事件。
 <br/>
 <br/>
 刘志雄牧师：〝今日不仅是国内教会受到打压，很多的族群，例如维吾尔族，社会不同的部门例如商界，也受到高度的控制和打压，我们希望唤醒更多人去关注，因为自由是整体，宗教自由受到伤害，以后各种自由也会受到破坏。今日国内受打压，他日我们香港可能会受到更大的控制和打压，所以我们要彼此守望。〞
 <br/>
 <br/>
 抗议人士抵达中联办外进行默哀及唱诗歌。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
