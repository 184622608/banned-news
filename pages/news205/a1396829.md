### 港三跑拟向广西购海砂 环团忧利益输送
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】造价过千亿的香港国际机场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第三跑道.html" target="_blank">
  第三跑道
 </a>
 正进行填海工程，由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海砂.html" target="_blank">
  海砂
 </a>
 短缺，早前有传媒揭露，当局拟向广西采购价格更贵的海砂，被指涉嫌利益输送。
 <br/>
 <br/>
 早前〝传真社〞报导，今年6月21日广西钦州市政府发出《钦州市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海砂.html" target="_blank">
  海砂
 </a>
 供港工作方案》，计划在钦州湾外湾订出一个800公顷的采砂区，当中400公顷是供应给香港。环保触觉星期四召开记者会，指香港机管局最初在三跑填海工程中使用越南海砂，中后期转用大陆机砂，现在突然转用大陆海砂，批评买砂过程欠透明，令人担忧其中涉及利益输送。
 <br/>
 <br/>
 环保触觉召集人谭凯邦：〝过往有人提过买砂会否变成利益输送大陆呢？是客观环境造成，因为大陆比较近，大陆的批文可以透过一些利益输送解决，所以我们认为这个价格是海鲜价和黑箱作业。从机管局不用越南的砂，可能得知那边的砂价格更贵，或者拿不到批文。〞
 <br/>
 <br/>
 立法会议员范国威表示将在立法会提出质询，要求政府及机管局交代买砂进展及有否超支，以及一旦超支，费用由谁承担。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
