### 港非遗市集与民同乐 承传客家功夫
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】在香港，一连三天的第三届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非物质文化遗产.html" target="_blank">
  非物质文化遗产
 </a>
 市集，周日在尖沙咀闭幕，其中重头戏是舞麒麟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_擂台赛.html" target="_blank">
  擂台赛
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_客家功夫.html" target="_blank">
  客家功夫
 </a>
 表演，吸引不少市民观看。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港文化节.html" target="_blank">
  香港文化节
 </a>
 至今第四届，其中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非物质文化遗产.html" target="_blank">
  非物质文化遗产
 </a>
 市集市集今年首次增到三天，向市民推广
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统文化.html" target="_blank">
  传统文化
 </a>
 ，包括首设擂台表演，邀请4队海外麒麟队伍和5队本地麒麟队交流，将客家舞麒麟文化推广到国际。同时举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_客家功夫.html" target="_blank">
  客家功夫
 </a>
 表演，展示十种客家传统功夫。
 <br/>
 <br/>
 香港文化节创办人赵式庆：〝香港文化，包括传统武术文化处于一个十字路口。所以这个十字路口我们做什么，如何去推广保育，用什么新的方法去纪录、传译以及承传等等，会导致香港以后整体文化的发展。〞
 <br/>
 <br/>
 客家功夫文化研究会会长李石年：〝客家拳是速打，不会拖泥带水。练武可以修心养性、用来强身健体。练功夫的人练的越久，你根本不会和人家斗，争风吃醋、斗狠等，这些不会再有。〞
 <br/>
 <br/>
 此外，主办单位安排了富有特色的木偶戏、醒狮、舞麒麟等表演，以及设立传统小食摊位，充满了浓郁的怀旧味道。
 <br/>
 <br/>
 香港非物质文化遗产共有480个项目，当中西贡坑口客家舞麒麟、长洲太平清醮以及大澳端午龙舟游涌等，是当中最具特色的十大项目之一。
 <br/>
 <br/>
 新唐人记者梁珍、陈仲明香港报导
</div>
