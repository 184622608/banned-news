### 美国会报告批中共打压港自由 议员促交待
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 特首
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林郑月娥.html" target="_blank">
  林郑月娥
 </a>
 公布任内第二份施政报告的同一天，美国国会与行政当局中国委员会（CECC）发表中国人权年报，指香港自由继续受北京打压，并建议美国考虑订立〝香港人权及民主法〞，监察香港自治状况。多位议员关注香港若无法维持一国两制，中美贸战将波及香港。
 <br/>
 <br/>
 在星期四的立法会施政报告的答问大会上，民主党胡志伟议员追问特首
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林郑月娥.html" target="_blank">
  林郑月娥
 </a>
 ，如何看待美国考虑取消
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 政策法。
 <br/>
 <br/>
 香港立法会议员胡志伟：〝因为大家见得到在马凯事件上，无论是美国政府、欧盟、英国、美国的政府、加拿大政府都分别发了声明，史无前例提出一个很严肃的指控，就是担心香港〝一国两制〞的实施是否能够有我们的初衷，能够确保到言论自由、资讯自由，能够继续在香港自由地流通。〞
 <br/>
 <br/>
 议会阵线毛孟静议员也对美国对香港未来的定位表示担忧。
 <br/>
 <br/>
 香港立法会议员毛孟静：〝美国真的取消对香港政策法，现在未成事实，我们当然忧虑，你看现在恒生指数江河日下，真的香港经济受到很沈重的打击。〞
 <br/>
 <br/>
 香港政府批评〝中国人权年报〞偏颇失实，对此表示遗憾，重申外国议会不应以任何形式干预香港特别行政区内部事务。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
