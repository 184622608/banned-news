### 金庸被中共重点统战 晚年立场成污点
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】香港武侠小说家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金庸.html" target="_blank">
  金庸
 </a>
 日前病逝，生前他共创作十多套武侠小说。成名的金庸在香港主权移交前的谈判时期，成为中共统战、拉拢的重点对象。之后金庸政治立场转趋保守，一些事情成为他的污点。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金庸.html" target="_blank">
  金庸
 </a>
 10月30日病逝于香港养和医院，享年94岁。金庸原名查良镛，1924年3月生，浙江海宁人。1940年代后期移居香港。
 <br/>
 <br/>
 金庸写过十多部武侠小说，代表作包括：《射雕英雄传》、《神雕侠侣》、《倚天屠龙记》、《天龙八部》等，作品更屡被翻拍电影，家喻户晓。
 <br/>
 <br/>
 有评论说，金庸的小说的确著名，但内容到底对人是否有益值得商榷。小说里面没什么道德主张，宣扬的多是市侩精神及政治寓言。甚至将三山五岳的修炼人描述的肮脏不堪，败坏了人们对信仰的认知和对神佛的敬仰。
 <br/>
 <br/>
 现实中，金庸年轻时曾在左派报章工作，他承认自己对共产党曾有很大寄望，〝但到了一个转变，等到它反右、大跃进、人民公社, 我就对它很失望很失望。〞
 <br/>
 <br/>
 1957年，金庸不满《大公报》不愿发表反对中共的〝大跃进〞的文章而辞职，两年后创办了《明报》。
 <br/>
 <br/>
 他创办的〝明报〞在上世纪60年代经常公开反对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文化大革命.html" target="_blank">
  文化大革命
 </a>
 ，当时他曾被香港左派分子严厉批评，甚至收到死亡威胁。金庸因此远避新加坡。
 <br/>
 <br/>
 金庸最后一部小说是《鹿鼎记》，当时10年浩劫的文革正在进行中。《鹿鼎记》中描写的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神龙教.html" target="_blank">
  神龙教
 </a>
 ，是明末清初江湖一个邪教，目的在于一统江湖，〝称雄天下〞。
 <br/>
 <br/>
 金庸后来在接受耶鲁大学东亚研究博士生傅楠（Nick Frisch）访问时，承认了神龙教是在影射共产党，承认他的最后几部小说的确是影射〝文革〞。
 <br/>
 <br/>
 <h4>
  获邓小平青睐后政治变得保守
 </h4>
 <br/>
 邓小平重新掌权改写了金庸的政治命运。1981年7月，邓小平以中共中央副主席身份在北京大会堂单独接见了金庸，金庸成为获邓小平接见的首位香港人。
 <br/>
 <br/>
 当时，邓小平主动提到金庸父亲查枢卿被中共处决的事，并要求金庸〝团结起来向前看〞，得到金庸的认可。
 <br/>
 <br/>
 金庸在香港主权移交前的谈判时期，成为中共统战、拉拢的一个重点对象，一些事情成为他的污点。
 <br/>
 <br/>
 上世纪80年代金庸开始参政，并于1985年出任香港基本法起草委员会委员，但他提出的选举香港特首等方案遭到民主派的批评。
 <br/>
 <br/>
 香港时事评论员桑普形容，金庸具有文人精神，他希望透过仗义执言改善当局施政。
 <br/>
 <br/>
 桑普说，〝他是希望中华复兴能出现， 他看到毛泽东没有办法做到的东西，他可能从邓小平处获得一显身手的机会，绽放出〝不写小说，只做政治〞的金庸。当然引起了很大争议。〞
 <br/>
 <br/>
 1988年，金庸作为《基本法》起草委员会委员和查济民委员，提出〝政制协调方案〞，被批评者认为阻碍了香港的民主发展，出卖了香港的民主。
 <br/>
 <br/>
 1989年，中国爆发爱国学生民主运动，遭到邓小平开枪镇压。金庸与香港民主党创党主席李柱铭等人，因此辞去起草委员会委员职务。
 <br/>
 <br/>
 金庸1991年接受港媒专访时表示，〝我从来都反对共产党主义制度，……我相信在我的这一生应可看到共产党垮台。〞
 <br/>
 <br/>
 但后来，中共继续对金庸进行拉拢。他和北京的关系再度改善，1995年，金庸担任香港特别行政区筹委会委员；2009年担任中共作协第七届全国委员会名誉副主席。
 <br/>
 <br/>
 1999年3月，金庸又被浙江大学聘为教授，且兼任浙大人文学院院长。同年10月，在浙大发表演讲，批评美国的资本主义，赞扬中共的社会主义，当时香港《苹果日报》、《争鸣》杂志等，纷纷批评金庸的言论，认为他给中共唱赞歌〝昏庸〞，他是给专制者按摩的弄臣等等。
 <br/>
 <br/>
 时事评论员刘锐绍形容，金庸为人〝知所进退〞，虽然想看到香港有民主，但知道北京的界线后，也不会作出坚持，会考虑官方的接受能力。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
