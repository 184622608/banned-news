### 港议员忧中美贸易战升级 已有产品被限制
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】香港政、商界的持续高度关注早前美中经济与安全审查委员（USCC）发表年度报告，令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 升级，担忧将重创香港的经济。多位立法会议员要求港府提出应对措施，有议员更透露部份电子设计软件已被限制。
 <br/>
 <br/>
 香港立法会星期二召开工商事务委员会，多位议员担忧
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 持续升级，尤其是美中经济与安全审查委员（USCC）早前的报告，要求港府相关官员交代对事件评估和跟进行动。有议员透露已有相关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_产品.html" target="_blank">
  产品
 </a>
 受到限制。
 <br/>
 <br/>
 资讯科技界莫乃光：〝其实我都听说有电子设计软件，现在已被出产商说香港如果有学界朋友要购买已经有问题，他说放在同中国内地同一个组别之中。〞
 <br/>
 <br/>
 香港驻美国总经济贸易专员麦德伟强调，一直有向美方反映香港意见，但未有回应有关报告会否影响〝香港政策法〞。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
 <br/>
