### 全球亿万富翁增长最快是中国 但落榜也快
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】一份调查显示，全球增长亿万富翁速度最快的是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 。去年中国诞生了106名新晋亿万富豪，即每星期诞生超过两个，但同时有一半的人跌出榜外。
 <br/>
 <br/>
 瑞银联同罗兵咸永道发布2018年亿万富豪报告称，去年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 亿万富豪由318名上升至373名，接近全球总数五分之一，当中97%为白手兴家。而香港则由69人下跌至67人。
 <br/>
 <br/>
 去年中国期内诞生了106名新晋亿万富豪，主要来自科技及消费行业，亦有51人跌出名单。预计明年财富上榜名单变化也大。
 <br/>
 <br/>
 罗兵咸永道香港私人银行谘询服务合伙人沈樱淇：〝中美贸易战和股市波动，也会造成一些（财富的）震荡，造成明年财富增长受影响。〞
 <br/>
 <br/>
 福布斯发表的2018年中国400富豪榜也显示，有超过四分之三经历了财富下跌，尤其是制造业。去年上榜的93人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落榜.html" target="_blank">
  落榜
 </a>
 。排名首位的阿里巴巴董事长马云，财富比一年前缩水40亿美元。
 <br/>
 <br/>
 新唐人记者梁珍、郭威利香港报导
 <br/>
 <br/>
