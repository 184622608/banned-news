### 香港97年后首次取缔政党或开不好先例
------------------------

<div class="wysiwyg">
 【新唐人2018年09月25日讯】香港政府周一取缔民族党，当地活动人士称此举开启败坏先例，担心
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港府.html" target="_blank">
  港府
 </a>
 未来或随意取缔不喜欢的政党或组织。
 <br/>
 <br/>
 社民连、人民力量、工党等约20多名成员到湾仔警察总部抗议当局引用《社团条例》将民主党定性为非法社团，是以言入罪，剥夺港人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结社自由.html" target="_blank">
  结社自由
 </a>
 。并指，《社团条例》是1997年6月由临时立法会通过修订，60名议员全部由中共委任，担忧当局借此箝制异见声音。
 <br/>
 <br/>
 社民连前立法会议员梁国雄：〝此例一开，将来政权则可肆意禁止任何视为〝眼中钉〞的政党及组织运作，无疑是继过往将异见者排除于选举制度后，进一步扼杀存于香港异见声音的举动。〞
 <br/>
 <br/>
 人民力量副主席谭得志：〝今天你说讲港独是踩红线，明天你说揭露高铁是踩红线，后天你说揭露内地食品安全是踩红线。〞
 <br/>
 <br/>
 2010年代，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港府.html" target="_blank">
  港府
 </a>
 运用社团条例施加党禁，拖延及阻挠各新兴政党及公民社会团体注册，如香港众志等，香港众志秘书长黄之锋表明今次港府取缔行动属政治考虑，强调会严阵以待。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
