### 港学者分析中期选举 川普成大赢家
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 落幕，民主党重夺众议院控制权，共和党则保住参议院。香港智明研究所研究总监许桢分析今次选举，他认为现任总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 是最大的赢家，巩固在党内地位，成为共和党内的唯一政治明星，但民主党则未出现重心人物。
 <br/>
 <br/>
 香港中文大学未来城市研究所副主席、智明研究所研究总监许桢在港台节目上指，今次美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 结果，于执政共和党来说，是〝小输当赢〞，对民主党而言是〝小羸当输〞，而真正的大赢家是总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 。
 <br/>
 <br/>
 智明研究所研究总监许桢：〝整个共和党内第一，无人再公开大规模挑战特朗普（川普）路线，第二他成为了党内最重要亦是唯一的明星，所以我们从共和民主两党党内权力分布去看，很明显特朗普巩固了他在共和党内权势，反过来民主党仍未有任何重心人物出现。〞
 <br/>
 <br/>
 对于民主党领袖佩洛西（Nancy Pelosi）是否会如某些美国传媒所言在众议院用尽主导权，大力制约特朗普？他觉得未必。
 <br/>
 <br/>
 智明研究所研究总监许桢：〝因为不论共和党也好民主党也好，共和党内有温和派，民主党内有保守派，民主党想透过些微众议院优势然后可大力制约特朗普，我觉得不现实。〞
 <br/>
 <br/>
 他并认为川普如何运用现有的政治资源，背后有很多盘算。同时川普也未必会放松对中国的关税问题。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
