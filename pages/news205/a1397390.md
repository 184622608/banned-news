### 港老字号得龙饭店结业 食客感不舍
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】屹立香港55年的得龙大饭店，周三将正式结业，不少顾客都感到不舍。第二代接班人、老板曾国生表示，不会放弃做
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统菜.html" target="_blank">
  传统菜
 </a>
 ，未来将改为经营私房菜等。
 <br/>
 <br/>
 位于新蒲岗的得龙大饭店结业消息，自月初传出后，小店生意大涨数倍，门口大排长龙。订位电话被打爆，有的老顾客甚至要上门提前订位。
 <br/>
 <br/>
 食客李先生：〝我这样的年纪一定知道得龙。他们的烧味等各样很好吃，以前他们的手工菜更好吃，现在说不做真有点可惜。〞
 <br/>
 <br/>
 食客潘先生：〝太新派的不好吃。很多地方都没有做古式的金钱鸡，鸭脚包等。他不做是很多人的损失。〞
 <br/>
 <br/>
 得龙由现年102岁的曾义于1963年创立。98年旧机场关闭、03年SARS等，一度令小店生意一落千丈，但长子曾国生透过还原一道道
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_怀旧名菜.html" target="_blank">
  怀旧名菜
 </a>
 ，成功将其起死回生。不过，现今他身体转差，加上人手流失等，均令小店无法撑下去。
 <br/>
 <br/>
 得龙第二代老板曾国生：〝包括人和铺头、还有很多基本设备，楼宇已经老化了。这是其中一个原因。第二，营运成本很高，主要是人工很贵，你请不到好的师傅，你要很高人工才请得到好的师傅。〞
 <br/>
 <br/>
 但他强调自己不会停下来，还会继续经营
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统菜.html" target="_blank">
  传统菜
 </a>
 。
 <br/>
 <br/>
 得龙第二代老板曾国生：〝吃饭、吃中菜的次数始终最多，受欢迎。特别是这些比较少人做的菜色，别人嫌麻烦的菜色，如果你们还懂做，一定有吸引力。〞
 <br/>
 <br/>
 得龙共有十多款传统菜，包括古法太爷鸡、金钱鸡、鸭脚包、以山楂熬汁的咕咾肉，以及得奖菜式〝砵酒焗桶蠔〞等都均是招牌菜。
 <br/>
 <br/>
 新唐人记者梁珍、陈仲明香港报导
 <br/>
 <br/>
</div>
