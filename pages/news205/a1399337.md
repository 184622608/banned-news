### 港楼价黄金十年完结 仲量联行料明年跌25%
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】面对中美贸易战及股市波动等不利因素，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_仲量联行.html" target="_blank">
  仲量联行
 </a>
 看淡香
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港楼价.html" target="_blank">
  港楼价
 </a>
 ，预料明年底前下跌最多两成半。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_仲量联行.html" target="_blank">
  仲量联行
 </a>
 认为，2008年金融海啸至今，香港长达十年的楼市上升周期已经完结，过去两个月楼价已下跌，加上中美贸易战恶化，股市下跌、大陆经济放缓等不利因素，预期明年底前楼价最多跌两成半。
 <br/>
 <br/>
 仲量联行董事总经理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾焕平.html" target="_blank">
  曾焕平
 </a>
 ：〝楼价下行的风险正在逐步增加，中美贸易战对楼市及整体经济有很大的影响。这个影响不单是心里影响，实在财务都有很大影响。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中资.html" target="_blank">
  中资
 </a>
 投地大幅减少，高价投地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海航.html" target="_blank">
  海航
 </a>
 甚至要转卖土地。
 <br/>
 <br/>
 仲量联行董事总经理曾焕平：〝因为中资本身，财务上尤其是这一年都有压力在。〞
 <br/>
 <br/>
 该行建议政府可撤销部分辣招，否则不但会加剧楼价调整，更会对香港经济造成打击。
 <br/>
 <br/>
 新唐人记者梁珍香港报导
</div>
