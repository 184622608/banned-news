### 港府计划万亿填海 民主派质疑政治任务
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】香港特首林郑月娥在本年度施政报告突然提出〝明日大屿〞大型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_填海.html" target="_blank">
  填海
 </a>
 造地计划，将耗资过万亿港元，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 及民间团体批评，这是港府执行北京的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治任务.html" target="_blank">
  政治任务
 </a>
 。
 <br/>
 <br/>
 守护大屿联盟等多个民间团体联同
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 政党，星期四中午到政府总部前抗议。他们拉起一幅长达40米的〝反对东大屿都会计划〞巨型横额，抗议林郑月娥一意孤行推行明日大屿计划，是在执行北京的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治任务.html" target="_blank">
  政治任务
 </a>
 ，要求港府撤回东大屿人工岛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_填海.html" target="_blank">
  填海
 </a>
 方案。
 <br/>
 <br/>
 职工盟秘书长李卓人：〝为何要搞呢，因为沙石要从大陆购买，输送利益给中资国企大财团，用这方法将我们一代人辛苦累积的储备输送给财团、输送给北京，这是整个明日大屿的目的，不是为了造地。〞
 <br/>
 <br/>
 守护大屿联盟发言人谢世杰批评，港府将海洋变陆地，将陆地变私有产权，无限延续地产发展商的霸权。联盟会继续收集10万个巿民签名，要求特首与民间对话撤回计划。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
 <br/>
 <br/>
