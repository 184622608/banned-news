### 备战新唐人武术大赛 德国选手赴港学武
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】中华武术博大精深，随着《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叶问.html" target="_blank">
  叶问
 </a>
 》等系列电影热播，令一代宗师叶问形象深入民心，在海内外掀起一股咏春热。最近一对德国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_咏春门徒.html" target="_blank">
  咏春门徒
 </a>
 ，为备战明年新唐人全世界华人武术大赛，专程来香港拜师学武，一起去看看。
 <br/>
 <br/>
 〝咏春传正宗 华夏振雄风〞，起源于广东佛山的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_咏春拳.html" target="_blank">
  咏春拳
 </a>
 ，由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叶问.html" target="_blank">
  叶问
 </a>
 带到香港，后因徒弟李小龙而蜚声海内外。
 <br/>
 <br/>
 来自德国的SOPHIE和FRANKIE，为备战明年新唐人全世界华人武术大赛，专门来香港学习咏春两周，并拜叶问第三代弟子梁岳贤为师。
 <br/>
 <br/>
 德国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_咏春门徒.html" target="_blank">
  咏春门徒
 </a>
 SOPHIE：〝因为这是咏春发源地，所以我们想来这里学习。我们学习了小念头、黐手，这是非常重要的，对基本练习来说，这是非常好的。〞
 <br/>
 <br/>
 德国咏春门徒FRANKIE：〝我喜欢中国功夫，因为这是传统的，但在欧洲，更像打斗和拳击。〞
 <br/>
 <br/>
 叶问第三代弟子梁岳贤：〝我本人是梁岳贤，是叶问的徒孙，也是梁相系的徒弟，今天有东德两个学生跟我玩（功夫），认我做师父，我很开心。〞
 <br/>
 <br/>
 她们还到访叶问教功夫的深水埗三太子庙，以及叶问倡导成立、树有叶问雕像的咏春体育会等地，追寻一代宗师足迹。期间并和多位咏春名师交流学习取经。
 <br/>
 <br/>
 叶问爱徒黄淳梁的儿子黄匡中: 〝（1968年成立的咏春体育会）这是一个永久会址，当时也有一个特别意思。所以很多人来到这里，不只是来一个地方，其实是一个精神象征，这也是叶问师公他走之前最后一个武馆。〞
 <br/>
 <br/>
 梁相国术总会创会主席车世杰：〝保留咏春真正的传统，就像刚才（示范）那样，什么叫黐手，现在没有人这样做黐手。〞
 <br/>
 <br/>
 咏春以近身搏击，拳快而防守紧密著称。叶问在港传技20余年，至今弟子门人近 200 万，遍及全球60多个国家和地区。
 <br/>
 <br/>
 新唐人记者梁珍、陈仲明香港报导
 <br/>
</div>
