 星期五被港府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_取消参选资格.html" target="_blank">
  取消参选资格
 </a>
 。刘小丽斥责港府配合中共政治打压。
 <br/>
 <br/>
 香港政府星期五晚上正式宣布民主派支持的参选人刘小丽提名无效，理由是刘小丽支持〝民主自决〞真心拥护《基本法》和效忠香港特别行政区。
 <br/>
 <br/>
 民主派议员随即发表联合声明，谴责港府无理褫夺香港人的参选资格，用政治手段来打压异见。
 <br/>
 <br/>
 民主派召集人毛孟静：〝今次做法完全显示反映特首林郑月娥企图以恐惧整治反对的声音，对于北京和港府认为不听话的人肆意取消参选资格，务求赶尽杀绝，以此恫吓香港人，制造白色恐怖。〞
 <br/>
 <br/>
 民主派议员之后联同刘小丽一起到金钟政府总部和特首办公室抗议。刘小丽对被港府取消参选资格感到悲愤，批评港府完全扭曲了她的政见，完全是进行政治筛选。
 <br/>
 <br/>
 工党前立法会议员刘小丽：〝今日的香港，我们的人权一次又一次被人这样粗暴践踏，去到一个违反国际公约的地步，香港人如果再不起身站出来，到底我们的人权还可以退到哪里呢。〞
 <br/>
 <br/>
 刘小丽当日稍早前的上已经陪同工党副主席李卓人九龙城政府合署报名参选，一旦她被取消资格后，李卓人可以继续代表民主派与亲北京的建制派争夺席位。
 <br/>
 <br/>
 新唐人记者林秀宜报导
