### 美国忧港高科技洩露大陆 港议员提警讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】〝美中经济与安全审查委员会〞上周发表年度报告批评北京压制及干预香港人权等方面，考虑将香港和中国视为同一个关税区域，限制潜在军用化的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高科技.html" target="_blank">
  高科技
 </a>
 输港。星期日在一论坛上有讲者指
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 忧虑香港会将高科技转移大陆。
 <br/>
 <br/>
 身兼港区人大代表的商界代表、立法会议员田北辰在论坛上强调，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 若将香港与大陆列为同一关税区很要命，他认为美国最担心是从香港将技术转移到大陆，包括粤港澳大湾区发展和河套发展计划。
 <br/>
 <br/>
 立法会议员田北辰：〝他们现在担心的是看到我们越来越多经济融合，首先是大湾区、前海、最近讲河套，河套到底属于香港或内地或大家共同去管，将来签了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高科技.html" target="_blank">
  高科技
 </a>
 转移的约，去到河套又如何分？我觉得这才是真正问题，特区政府一定要认真研究，作出一个交代。不然将来水洗也不清。〞
 <br/>
 <br/>
 有学者也认为美国国会报告提及的〝军民两用科技〞是重点。
 <br/>
 <br/>
 中大全球研究社会科学学士课程讲师陈伟信：〝很简单问一个香港商人去美国买科技的时候，货物到香港究竟要进行何种清关手续，或是可以直接在西九送往内地，这个是美国一直跟踪不到的。〞
 <br/>
 <br/>
 同场的民主党议员林卓廷则呼吁港府和建制派议员在中美贸易战如火如荼之际切勿火上加油，否则自食苦果。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
