### 贺建奎港大演讲 大陆专家批评动机不纯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】因首创〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 〞而遭全球声讨的的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南方科技大学.html" target="_blank">
  南方科技大学
 </a>
 副教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 ，周三出席港大举行的国际峰会。有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆专家.html" target="_blank">
  大陆专家
 </a>
 认为〝基改婴〞事件疑点重重，呼吁当局调查。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 风波后终于现身香港大学，但较原定时间拖了一个多小时。他先声称研究结果遭〝泄漏〞而道歉，但发言遭到现场人员打断。
 <br/>
 <br/>
 他爆出还有另外一个孕妇怀有〝基改婴〞，又透露研究经费自南科大，以及自己个人。不过，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南方科技大学.html" target="_blank">
  南方科技大学
 </a>
 否认知情。
 <br/>
 <br/>
 出席会议的大陆伦理专家邱仁宗呼吁当局调查事件，又狠批贺建奎动机不纯。
 <br/>
 <br/>
 中国生命伦理学专家邱仁宗：〝会不会发生基因编辑的乱象？有人欺骗，打着基因编辑的旗帜，实质上胡搞，骗别人的钱，会不会出于这个？〞
 <br/>
 <br/>
 他又批评，中共《人民日报》不应抢报《基改婴》事件。
 <br/>
 <br/>
 中国生命伦理学专家邱仁宗：〝人民网作为政府办的网站，不能去掺乎这个事情。透过媒体发表的科学成果往往是糟糕的，往往是假或者很不科学的，根本不能站住脚的，因为这样，所以（贺建奎）他要经过媒体。〞
 <br/>
 <br/>
 中共首创〝基改婴〞的风波，被认为冲击人类道德底线，至今已有大陆200多名学者联名反对，
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
