### 颱风山竹摧毁百年老灯 港人灾后自救
------------------------

<div class="wysiwyg">
 【新唐人2018年09月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山竹.html" target="_blank">
  山竹
 </a>
 肆虐香港次日，多区破坏严重，电影取景地、有百年历史的中环煤气灯被毁。将军澳等重灾区，市民自发灾后清理，发挥互助精神。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山竹.html" target="_blank">
  山竹
 </a>
 袭港后，香港各区满目疮痍，到处都是塌树、招牌被砸烂。
 <br/>
 <br/>
 位于中环都爹利街的两盏百年煤气灯，曾是《喜剧之王》等不少电影取景热点，被列入法定古迹。如今也在风暴中被摧毁，引来大批网民留言感惋惜。有目击者见到狂风吹倒大树，压烂石阶和煤气灯。
 <br/>
 <br/>
 钻石会大厦管理员：〝昨晚六点二十五分，我站在这里。一倒下来的时候，风刮过，呼一声（作势推倒状）〞
 <br/>
 <br/>
 记者：〝这么夸张？〞
 <br/>
 <br/>
 钻石会大厦管理员：〝是，很厉害。山竹喔，你吃过山竹牛肉吗？〞
 <br/>
 <br/>
 在重灾区之一将军澳，路面严重损毁，满地是砖头、沙石等。因当天停课，有中学生在社交群组发起自发搬砖、灾后重建，得到数百人响应。
 <br/>
 <br/>
 中四学生石名皓：〝今次反映很热烈，我没有想到这么（热烈），感觉人间有爱，大家团结一致地收拾好整个将军澳。〞
 <br/>
 <br/>
 小五学生：〝3个小时了。〞
 <br/>
 <br/>
 记者：〝辛苦吗？〞
 <br/>
 <br/>
 小五学生：〝不辛苦，有些成就感，因为可以帮到（重建）。〞
 <br/>
 <br/>
 另外，灾难过后，因政府没有宣布停工，各区交通大瘫痪，令东铁线沿线挤满候车市民，过千人排队上班。不过，有司机在群组发起义载，也有公司主动宣布停工一天，被网民热赞好老板。
 <br/>
 <br/>
 新唐人记者梁珍、陈仲明香港报导
 <br/>
 <br/>
