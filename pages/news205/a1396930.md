### 港府首证实高铁可容6百大陆人工作
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】广深港高铁香港段已于9月23日正式通车，星期五在有关一地两检的会议上，港府官员在议员的追问下，首次证实内地口岸区可容纳600名大陆工作人员，但不愿透露实际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_执法人员.html" target="_blank">
  执法人员
 </a>
 的数字。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港立法会.html" target="_blank">
  香港立法会
 </a>
 星期五召开有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高铁一地两检.html" target="_blank">
  高铁一地两检
 </a>
 条例委员会会议。高铁已通车一个月，公民党议员郭家麒批评局高估高铁的使用量，询问内地口岸区大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_执法人员.html" target="_blank">
  执法人员
 </a>
 的数目，会否因时制宜根据客量调整。
 <br/>
 <br/>
 运输及房屋局局长陈帆：〝我所知道整体配备，内地口岸区足以容纳大约600名工作人员，至于实际操作人多少，这属于内地口岸区管辖范围，这数字毋须要向我们呈报。〞
 <br/>
 <br/>
 陈淑庄不满当局首次证实大陆工作人员的数目。
 <br/>
 <br/>
 公民党议员陈淑庄：〝我们做为议员，我们连有多少人都不知道，今天才听到有600的数字，就是不知怎么你挤几下，就告诉我们原来内地口岸区可以容纳600个工作人员。〞
 <br/>
 <br/>
 她并追问当局〝一地两检〞司法覆核案，政府若败诉会如何处理，运房局长陈帆称因为司法覆核正在进行当中，政府方面不想揣测。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
