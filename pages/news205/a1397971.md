### 港补选论坛民主派与建制派候选人针锋相对
------------------------

<div class="wysiwyg">
 【新唐人2018年11月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 立法会九龙西补选将在本月25日举行，星期五在一选举论坛上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 候选人李卓人批评亲共的候选人陈凯欣双方又就民生与政治问题针锋相对。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 电台星期五举办九龙西补选选举论坛，代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 出战的工党李卓人指号称是〝独立〞、没有政治联系的陈凯欣，事实上是代表亲共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_建制派.html" target="_blank">
  建制派
 </a>
 参选，批评她欺骗港人。
 <br/>
 <br/>
 工党李卓人：〝民建联帮你出电邮，经民联帮你出新闻稿，九龙社团联会就捧妳，帮妳卖广告，妳就出来说：我没有刻意说我不是建制派。妳是什么呢？又说自己没有政治联系，其实妳是山寨女孩，卖假货。〞
 <br/>
 <br/>
 陈凯欣反驳遭到抹黑，但承认得到建制派支持。
 <br/>
 <br/>
 候选人陈凯欣：〝大家各位选民要看清楚，什么叫抹黑。另一样，是很感谢自己一些民生政纲得到建制派所有朋友的支持。〞
 <br/>
 <br/>
 李卓人在总结发言时呼吁港人抗共。
 <br/>
 <br/>
 工党李卓人：〝香港现在被围攻，共产党、特首林郑、保皇党、中联办一起围攻香港，我们香港要反包围，香港人一起为下一代背水一战，香港人赢回一次，大家出来投票。〞
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
