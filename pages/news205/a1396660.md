### 李卓人参选九龙西补选 抗中共打压
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】代表香港民主派参选11月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九龙西补选.html" target="_blank">
  九龙西补选
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李卓人.html" target="_blank">
  李卓人
 </a>
 取得参选资格后，展开一连串的竞选工程，他以为〝下一代背水一战〞为口号，呼吁选民集中选票，赢回关键一席，抗衡
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李卓人.html" target="_blank">
  李卓人
 </a>
 早前举行街头造势活动后，星期一晚上七点到九龙湾国际展贸中心出席九龙西地方选区补选候选人简介会，他连同工党、社民连等团体大约30多人到场外抗议。李卓人批评特首林郑月娥配合
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 取消了原本民主派参选人刘小丽的参选资格。
 <br/>
 <br/>
 参选人李卓人：〝因为中共说今次的政治任务，就是在社会挑拨离间，要先打压刘小丽。用尽方法协助亲共保皇党取得议席，这个就是中共给林郑的政治任务。〞
 <br/>
 <br/>
 接着出席星期二商业电台举办
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九龙西补选.html" target="_blank">
  九龙西补选
 </a>
 论坛，李卓人表明反对特首林郑月娥在港珠澳大桥通车仪式上向习近平〝吹嘘〞的〝明日大屿〞大型填海计划。
 <br/>
 <br/>
 参选人李卓人：〝我们下一代要背负这个一万亿，整个人工岛计划本身，1700公顷，其实最益是谁？是一个输送利益，输送利益予红色资本，因为买砂、兴建，全部是红色资本，根本是一个间接上缴中央的方法。〞
 <br/>
 <br/>
 今次香港九龙西补选共有五位候选人，另外还有冯检基、陈凯欣、曾丽文及伍廸希四人。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
