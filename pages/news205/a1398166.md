### 港珠澳大桥通车 廉价团涌入疑涉黑工
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 上月24日通车，在刚过去的周日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_口岸出入境.html" target="_blank">
  口岸出入境
 </a>
 人次突破10万，创大桥开通以来的新高。不过大量
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_廉价团.html" target="_blank">
  廉价团
 </a>
 涌进邻近的东涌市区造成混乱，有议员疑涉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑工.html" target="_blank">
  黑工
 </a>
 要求当局跟进。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 通车后的第二周，临近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 口岸的东涌，涌进大量廉价大陆团，影响当地居民生活。有议员批评当局为了增加大桥流量，没有进行任何管制及监察，交通配套严重不足。
 <br/>
 <br/>
 香港立法会议员郭家麒：〝无论是餐厅、药房都叫苦连天，当地居民买不到东西。其实大家都发觉有很多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_廉价团.html" target="_blank">
  廉价团
 </a>
 ，连交通工具都没有准备好，就直接从珠海等地来香港，我觉得特区政府有责任，与大陆举办极廉价的旅行团的一些单位接触。〞
 <br/>
 <br/>
 有旅游界议员关注部分廉价团可能涉及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑工.html" target="_blank">
  黑工
 </a>
 问题，要求保安局跟进。
 <br/>
 <br/>
 旅游界立法会议员姚思荣：〝目前我已和保安局联络，要求对这些无牌的来香港的领队变成实际一个导游，在香港正经营接待内地旅行团的工作，其实某程度上等于是黑工。我希望保安局就有关巡查安排和发现涉嫌在香港打黑工的安排，作出检控和拘捕。〞
 <br/>
 <br/>
 东涌居民王进洋表示若东涌混乱情况无改善，未来一两周内会决定发起光复行动，但不会使用暴力。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
