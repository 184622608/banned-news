### 习南巡为何格外低调？港媒揭内幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在中美贸易战火升高之际考察广东，被认为是为了提振国内外对于中国经济和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 的信心。但令外界颇感诧异的是，习此次南下的安排极为低调，有港媒揭示背后的原因。
 <br/>
 <br/>
 据中共官媒报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 南下调研，22日考察了广东珠海横琴新区粤澳合作中医药科技产业园及格力电器厂。港珠澳大桥23日举行通车仪式，习近平在典礼上仅说了12个字：〝我宣布，港珠澳大桥正式开通〞。之后，匆匆离开现场。24日到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳.html" target="_blank">
  深圳
 </a>
 考察。
 <br/>
 <br/>
 《香港经济日报》25日报导说，1992年88岁高龄的中共前领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 ，以普通党员的身份〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南巡.html" target="_blank">
  南巡
 </a>
 〞。先后赴武昌、深圳、珠海和上海视察，沿途发表了一系列讲话，扫清了当时中共政治和经济领域的思想混乱，坚定了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 的原则和方向。
 <br/>
 <br/>
 习近平在2012年中共十八大后，当选中共总书记不久，同样南下广东，在深圳、珠海、顺德、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 、惠州等地视察。
 <br/>
 <br/>
 当时习近平表态：〝之所以到广东来，就是要到中国改革开放中得风气之先的地方，现场回顾中国改革开放的历史进程，将改革开放继续推向前进。〞
 <br/>
 <br/>
 此次习近平再次南下，正值中美贸易战令中共陷入内外交困之际。舆论期待，习近平会发表新一波改革开放纲领性的战略，用以提振国内外对于中国经济和改革开放的信心。
 <br/>
 <br/>
 然而，习近平至今仍只针对视察的个别项目发表指导意见。
 <br/>
 <br/>
 为何习近平这次南下如此低调，而调研参观的范围又很广泛呢？港媒分析其中的原因有三点：
 <br/>
 <br/>
 一是，习近平此次南下是调研，会在调研结束后作出综合总结。在中共高层有了共识前，习只宜就个别项目作出指导性意见，不会发表改革纲领性发言。
 <br/>
 <br/>
 二是，目前就外部环境而言，中美贸易战仍在僵持阶段，美国中期选举将至，中共高层需要做出谨慎观察，才可因应外部环境变化制订新一波改革开放战略。
 <br/>
 <br/>
 三是，中共四中全会即将召开，外界估计会议的主题是深化改革，落实三中全会提出的改革目标，同时对改革开放40年做一个总结。
 <br/>
 <br/>
 因此，相信习近平会在四中全会上，将阐述新一波改革开放的宣言，今次南下考察的结果，也将会在四中全会中呈现。
 <br/>
 <br/>
 香港多家媒体此前报导，中共第十八届四中全会将于11月召开。会议将涉及中美贸易战、政治、经济和人事等议题。
 <br/>
 <br/>
 港媒报导，自去年中共十九大结束后，一年来，中共政治和经济〝左倾〞明显，而且在外交上，随着美中贸易战火日渐升高，中美迈入新冷战时代。伴随着在军事上，台海形势日趋紧张。
 <br/>
 <br/>
 经济上，货币超发、地方债务居高不下、P2P等影子银行暴雷频繁、股市、汇市下挫，外资纷纷撤离，特别是〝国进民退〞民营企业离场的传闻，更使民营企业忧心忡忡。
 <br/>
 <br/>
 基于上述种种原因，预计习近平将在四中全会上，藉改革开放40周年的时机，强调改革开放，以鼓舞和稳定民心。
 <br/>
 <br/>
 但专家普遍认为，中共体制不倒，一切改革都是徒劳。过去40年所谓的改革开放，只是政治和权力运作的需要，不属于真正意义的改革。中共不倒台，所谓的改革都是白忙一场，中共体制决定了任何改革都是徒有虚名，都不可能成功。中共改革40年后的中国现状已经证实了这一点。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
