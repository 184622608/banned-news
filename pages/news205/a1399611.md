### 融东方智慧 绿色东方体验店逆市扩张
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】中美贸易战冲击中港经济，零售市场放缓，不过，优质
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健康产品.html" target="_blank">
  健康产品
 </a>
 仍受欢迎。其中，位于香港的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绿色东方.html" target="_blank">
  绿色东方
 </a>
 集团，推广具古老养生之道的系列产品，逆市扩张，受到瞩目。
 <br/>
 <br/>
 爱美之心人皆有之，透过五行经络排毒美容，又有没有试过？
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绿色东方.html" target="_blank">
  绿色东方
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_体验店.html" target="_blank">
  体验店
 </a>
 店长Jacky Lee：〝这个系列的每一个产品有五种不同的颜色在里面，用中医原理去调节你的经络，透过皮肤的吸入之后，它会将里面的毒素水份全部排出。〞
 <br/>
 <br/>
 体验日顾客徐小姐：〝今天的体验很好，今天试很多产品去搽，即场的效果很突出，所以很开心。〞
 <br/>
 <br/>
 不到两年光景，融合东方智慧的绿色东方第4间分店，在荔枝角开幕，以体验店形式向顾客推介健康理念。现在网购盛行，购物非常方便，但为何绿色东方仍坚持开实体店，原来背后有其文化内涵。
 <br/>
 <br/>
 绿色东方市场部经理Germaine Tse：〝中国五千年来有很多的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_养生文化.html" target="_blank">
  养生文化
 </a>
 ，现在因为年代不同遗失了，我们想要推崇这个（传统文化）。体验店是人与人的接触，经过我们员工的体贴服务，你会感觉在这里很舒服，好像在自己家里一样。〞
 <br/>
 <br/>
 体验日顾客戴先生：〝香港人非常忙碌，生活很紧张，我们需要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健康产品.html" target="_blank">
  健康产品
 </a>
 来平衡我们的生活。在这里我看到很多产品很不错，都经过精挑细选，对身体从头到脚都照顾的到。〞
 <br/>
 <br/>
 绿色东方相信，虽然现今经济不景，但面对现今假货、毒物流行，天然、健康、优质的环球保健品的需求仍会稳定上扬。未来他们准备以讲座、亲身体验等形式，向顾客推广古老的养生之道。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
