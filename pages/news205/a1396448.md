### 习近平迟到半小时 主持港珠澳通车只讲一句话
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 通车仪式23日揭幕，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 与中港澳一众高官云集珠海，出席揭幕仪式。港媒称，按官方安排，仪式原定于早9时开始，但习近平一行迟到半小时 。通车仪式上，习只说一句话便走下台，与其他嘉宾握手。
 <br/>
 <br/>
 据中共党媒消息，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 开通仪式23日上午9时在广东珠海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海口岸.html" target="_blank">
  珠海口岸
 </a>
 举行，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 出席仪式并宣布大桥正式开通。
 <br/>
 <br/>
 据悉，习近平在港珠澳大桥的开通典礼上并没有发表重要讲话，只说了一句话，〝我宣布，港珠澳大桥正式开通〞后便走下台，与其他嘉宾握手。
 <br/>
 <br/>
 出席嘉宾包括主管港澳事务的中共国务院副总理韩正、副总理刘鹤、中共中央办公厅主任丁薛祥、中央政策研究室副主任江金权、中共政协副主席董建华、梁振英、港澳办主任张晓明、广东省委书记李希、广东省省长马兴瑞、行政长官林郑月娥及澳门行政长官崔世安等。
 <br/>
 <br/>
 现场保安严密，出席嘉宾需先接受安检，而且不可携带手提电话。仪式现场播放多条介绍港珠澳大桥特色及建造工程的片段。
 <br/>
 <br/>
 香港《苹果》报导称，按官方安排，仪式在早晨9时开始，但中港澳一众高官一大早准时抵达会场后，现场不断播放有关大桥的影片，却苦候多时仍不见领导人出现。
 <br/>
 <br/>
 直到9时28分，开幕礼才正式开始，习近平与林郑月娥及澳门行政长官崔世安一起现身会场。
 <br/>
 <br/>
 港珠澳大桥耗资1200多亿元人民币，全长近50公里，连接香港、珠海和澳门。工程历时近9年。大桥23日举行开通仪式，24日正式通车，连接香港口岸人工岛及北大屿山公路往来市区的连接路也同步开放。
 <br/>
 <br/>
 据报，北京高层希望港珠澳大桥通车能带动广东西部地区的经济发展。但是，随着近几年外资企业陆续撤离珠江三角洲，其经济价值大不如前。
 <br/>
 <br/>
 分析认为，习近平此次出席大桥通车仪式，政治意味浓厚。除了表现北京对香港及广东经济发展的高度重视，也是为了纪念中共所谓的改革开放40周年。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：祝馨睿）
