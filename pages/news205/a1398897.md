### 市道转好？港维园年宵竞投进帐升13%
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】被视为香港经济晴雨表的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维园年宵.html" target="_blank">
  维园年宵
 </a>
 竞投，一连4天竞投。今年共吸引约1650名人次进场，总收入为1450万元，较去年上升12.8%。
 <br/>
 <br/>
 虽然距离明年黄历新年还有3个多月，但全港最大的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维园年宵.html" target="_blank">
  维园年宵
 </a>
 摊位已率先开始竞投，并首次搬到荔枝角政府合署举行。首日竞投气氛热烈。
 <br/>
 <br/>
 竞投官：〝52万第二次，52万第三次成交〞。
 <br/>
 <br/>
 经12口价，最大快餐档以52万成交，较去年贵2万。有花店连续九年投得一个湿货摊位，成交价逾7万，较底价贵6倍。
 <br/>
 <br/>
 306号档主黄先生：伤心，贵了几万元，几万元要做很多货才卖到—你当一盆花赚10元，你要卖几千盆才补回租金。
 <br/>
 <br/>
 食环署认为竞投气氛激烈，反映投资者看好今年市道，但就禁止港独等字眼出现在维园。
 <br/>
 <br/>
 食环署高级总监王德发：〝港独或者这样的活动，我们都不赞成在政府举办的场地进行，我们都会有关的问题进行检讨。〞
 <br/>
 <br/>
 维园年宵市场共有463个摊位，包括272个干货摊位、180个湿货摊位、8个主题摊位及3个快餐摊位。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
 <br/>
 <br/>
</div>
