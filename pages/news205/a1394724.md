### 港泛民动议传召特首 官员解释马凯事件
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】香港外国记者会第一副主席马凯工作签证不获续期，引起的风波持续发酵。民主派议员将提动议，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传召特首.html" target="_blank">
  传召特首
 </a>
 林郑月娥及官员到立会做出解释。
 <br/>
 <br/>
 民主派立法会议员星期二宣布将在本月24日立法会大会上提出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传召特首.html" target="_blank">
  传召特首
 </a>
 林郑月娥、保安局局长李家超以及入境处处长曾国卫的动议，到立法会就拒绝马凯签证一事解释。
 <br/>
 <br/>
 民主派会议召集人毛孟静：〝这件事是香港政府赤裸裸而粗暴地让外国人、香港人觉得它是企图以政治手段去打压不单止是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新闻自由.html" target="_blank">
  新闻自由
 </a>
 ，是用一个外国记者来祭旗，...你不可能说外国人、香港人质疑是完全不合理，总之我说是国际惯例不需要解释就可以蒙混过去，是一个非常无赖及失礼的说法。〞
 <br/>
 <br/>
 民主党许智峯也在立法会大会就事件提出休会待续议案。
 <br/>
 <br/>
 民主党许智峯：〝我认为这事件已经受国际关注，对我们香港人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基本自由.html" target="_blank">
  基本自由
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_言论自由.html" target="_blank">
  言论自由
 </a>
 及新闻自由的冲击下，我认为是必须讨论。〞
 <br/>
 <br/>
 公民党党魁杨岳桥批评，今次事件将香港国际金融城市的金字招牌打烂，呼吁特首尽快作出解释。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
