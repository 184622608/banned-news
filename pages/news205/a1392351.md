### 同股不同权新股美团上市 反应冷淡
------------------------

<div class="wysiwyg">
 【新唐人2018年09月21日讯】继小米后，第二只〝同股不同权〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新股.html" target="_blank">
  新股
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美团点评.html" target="_blank">
  美团点评
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港上市.html" target="_blank">
  香港上市
 </a>
 ，不过，招股反应较冷清，收报略升5.3%。
 <br/>
 <br/>
 大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外卖平台.html" target="_blank">
  外卖平台
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美团点评.html" target="_blank">
  美团点评
 </a>
 首日挂牌，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港交所.html" target="_blank">
  港交所
 </a>
 挤满了人。管理层大谈未来发展目标，不过，被问到公司何时转亏为盈时，就只是这样回应。
 <br/>
 <br/>
 美团点评首席财务官陈少晖：〝我们的核心主体业务改善的趋势今年在继续，我们对公司继续保持高速增长的同时，改善我们的财务表现是非常有信心的。〞
 <br/>
 <br/>
 美团早前公布招股结果，其公开发售部分获1.7万人申请，仅超额认购0.5倍，相对小米更逊色。有分析师认为，美团无盈利，定价太贵，不值得投资，又认反映市场对同股不同权仍然有忧虑。
 <br/>
 <br/>
 信诚证券联席董事张智威：〝美团认购反应不理想，上市的股价也没有见到很大的升幅，同股不同权是否受到市场欢迎，暂时看不到这个情况。而且要留意的是，外国同股不同权同时，有集体谈判权保证小投资者，但在香港暂时没有这个机制。〞
 <br/>
 <br/>
 美团过去三年均录得亏损，截至今年4月底止，美团录得经调整亏损20.19亿元人民币。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
</div>
