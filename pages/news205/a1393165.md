### 睡前滑手机为何兴奋？ 都是蓝光惹的祸
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月27日讯】在几乎〝人手一机〞的现代，长时间使用手机的〝低头族〞越来越多，而且很多人都有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_睡前滑手机.html" target="_blank">
  睡前滑手机
 </a>
 的习惯，但这往往会让人精神兴奋、导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失眠.html" target="_blank">
  失眠
 </a>
 。台大团队发现，手机放射出来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓝光.html" target="_blank">
  蓝光
 </a>
 ，是影响睡眠品质的重要原因。
 <br/>
 <br/>
 9月27日，台大医院皮肤部、眼科部及台湾大学生命科学系研究团队，在记者会上公布一项研究成果。该团队发现，眼睛里有一种
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓝光.html" target="_blank">
  蓝光
 </a>
 受体，接收蓝光后会向脑部发出讯号，活化全身交感神经，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失眠.html" target="_blank">
  失眠
 </a>
 。
 <br/>
 <br/>
 这项研究结果日前发表在美国国家科学院学报（Proceedings of the National Academy of Sciencesof USA）。
 <br/>
 <br/>
 据台媒中央社报导，参与研究的台大眼科部医师林颂然表示，手机、3C等产品萤幕会散发出波长长短不一的蓝光光线，其中以波长较长的蓝光对身体影响较大。
 <br/>
 <br/>
 该研究团队利用小鼠实验，在小鼠睡前照射8分钟的强烈蓝光，发现小鼠的交感神经在5分钟之内增强超过5倍，不仅心跳加速、血压升高、出汗、肾脏交感神经也活化，可使全身交感神经兴奋超过1小时。
 <br/>
 <br/>
 林颂然称，进一步研究发现，原来眼睛里有一种蓝光受体，叫作〝内生性感光视神经细胞〞（ipRGC），是用来调时差用的细胞，这种受体接受到蓝光后，会向脑部的视交叉上核发出讯号，活化全身交感神经，从而导致失眠。
 <br/>
 <br/>
 台大眼科部医师陈达庆也指出，这项研究显示，睡前若滑手机，会让眼睛接收过多蓝光，让身体误以为是在白天，促使交感神经兴奋，反而更睡不着、甚至失眠，不仅伤眼也影响睡眠。
 <br/>
 <br/>
 尤其夜间，在暗处瞳孔会放大、进入眼睛的蓝光也会增大5至6倍，对身体影响更大。
 <br/>
 <br/>
 陈达庆建议，要在光线足够的地方使用手机，夜间最好将萤幕调成偏红色系的颜色，这样可降低蓝光影响。
 <br/>
 <br/>
 另据报导，手机发出的蓝光会降低褪黑激素，影响睡眠品质。所谓褪黑激素，指的就是一种由大脑松果体所分泌的贺尔蒙，有调节生理时钟、帮助入睡等效果，又被称为〝睡眠贺尔蒙〞。因此若是褪黑激素降低，就会导致睡不好、失眠。
 <br/>
 <br/>
 此外，因为视网膜上的光敏感神经节细胞，对电子产品所发出的蓝光十分敏感，若是在这种光线下待太久，也会难以入睡。
 <br/>
 <br/>
 比利时学者Liese Exelmans曾研究844位受试者，并将研究结果刊登于国际期刊〝Social Science&amp;Medicine〞，该研究也指出，睡觉时使用手机会降低睡眠品质、增加失眠问题和让白天疲倦。
 <br/>
 <br/>
 (记者罗婷婷综合报导/责任编辑：李泉)
