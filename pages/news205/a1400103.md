### 港医院逃犯被揭是上市公司股东
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】上周五在香港玛丽医院羁留期间逃走后被捕的疑犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李亮.html" target="_blank">
  李亮
 </a>
 ，被暂控一项逃离合法羁押罪，星期一和两名同犯在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东区裁判法院.html" target="_blank">
  东区裁判法院
 </a>
 提堂。李亮被揭拥有18家公司，是上市公司中国环保科技主要股东，被捕后股价大跌。
 <br/>
 <br/>
 声称有爱滋病的34岁疑犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李亮.html" target="_blank">
  李亮
 </a>
 ，和两名同犯，由港警押送到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东区裁判法院.html" target="_blank">
  东区裁判法院
 </a>
 提堂，案件押后至下月17日再讯。
 <br/>
 <br/>
 出生于大陆的李亮，拥有多米尼加护照，自称是剑桥大学博士，创办太平洋生殖医学中心，生意遍布世界各地。
 <br/>
 <br/>
 2015年6月他涉嫌以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假证.html" target="_blank">
  假证
 </a>
 从大陆来港被捕，其后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弃保潜逃.html" target="_blank">
  弃保潜逃
 </a>
 ，列为被通缉人士，早前入境香港时被捕。
 <br/>
 <br/>
 上周五他逃走当日，其持股近两成的中国环保科技股价离奇大跌近一半，怀疑有人大笔抛货。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙、王伟明香港报导
</div>
