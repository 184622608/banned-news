### 高铁香港段通车  抗议者批超支违宪
------------------------

<div class="wysiwyg">
 【新唐人2018年09月22日讯】周六，广深港高铁香港段，举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通车仪式.html" target="_blank">
  通车仪式
 </a>
 。有数个团体，在场外抗议，批评高铁项目，严重超支，而当局强推〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一地两检.html" target="_blank">
  一地两检
 </a>
 〞更是违反
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港基本法.html" target="_blank">
  香港基本法
 </a>
 ，破坏香港法治。
 <br/>
 <br/>
 广深港高铁香港段周六举行开通仪式。香港特首林郑月娥、广东省省长马兴瑞，以及多位香港和内地高级官员出席了仪式。
 <br/>
 <br/>
 这段长26公里的高速铁路，连结香港和深圳、广州，并将香港纳入内地的高铁网络。
 <br/>
 <br/>
 不过，这一项目自去年来在香港引发巨大争议。尤其是位于香港核心地带的西九龙高铁站，划出了〝内地口岸区〞执行内地法律，令许多香港人担忧，法治受到威胁。
 <br/>
 <br/>
 数个关注团体，周六在西九龙站外举行抗议活动，批评高铁项目实施〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一地两检.html" target="_blank">
  一地两检
 </a>
 〞，违反
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港基本法.html" target="_blank">
  香港基本法
 </a>
 。
 <br/>
 <br/>
 泛民主派立法会议员范国威：〝（一地两检）也都是违反了香港基本法，割地两检，是在香港部分土地上，实行大陆的法规。对很多香港人来说，这破坏了一国两制和香港的法治。〞
 <br/>
 <br/>
 范国威还批评高铁项目严重超支，速度和效用也不如预期，是〝超废〞高铁。
 <br/>
 <br/>
 广深港高铁耗资近千亿港币，被当局形容为可带来显著经济效益。不过，批评人士认为，该项目是北京持续渗透和控制香港的又一明证。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
