### 脊椎是健康之本 港中医讲座倡养生之道
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】现代人越来越重视健康，如何养生、治未病成为关注议题。最近，绿色东方举办健康讲座，讲解中医经络运作机制，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脊椎.html" target="_blank">
  脊椎
 </a>
 经络与健康的知识，收到欢迎。
 <br/>
 <br/>
 趁着新店开幕，绿色东方一连两天举办
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中医讲座.html" target="_blank">
  中医讲座
 </a>
 ，邀请注册中医师黎珂讲解，为顾客提供养生保健资讯。中医说〝经络畅通 病不上身〞，黎珂介绍了中医经络的运作机制，并强调
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脊椎.html" target="_blank">
  脊椎
 </a>
 是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健康之本.html" target="_blank">
  健康之本
 </a>
 ，亦是百病之源。
 <br/>
 <br/>
 主讲人黎珂医师：〝实际上有很多疾病是由于脊椎的变形和受伤（引起的），椎间盘突出压迫了神经，对健康就会引起负面的影响。心脏病，胃病，这些都可能是由于脊椎的不正压迫了神经造成的。〞
 <br/>
 <br/>
 她说，中医的精华在于养生、治未病，提醒大家养生要注意从整体出发，并现场示范了日常保健脊椎、提升肌力、和护眼的妙方，
 <br/>
 <br/>
 主讲人黎珂医师：〝现在人养生都是说吃什么，补什么，其实人是一个循环，在自然界也是自然循环。其实很多东西不是光是吃，比如如果要进，首先药出。汗腺是不是通畅，该排汗的时候能否排。〞
 <br/>
 <br/>
 听众邬女士：〝（听完讲座）我们马上警觉要坐直。〞
 <br/>
 <br/>
 听众陈先生：〝特别要注意腰骨，否则以后腰骨会形成其他的病痛，所以我会更加注意。〞
 <br/>
 <br/>
 此外，讲座还有示范韩式古法美容术，用经络五行排毒美容，反响热烈。
 <br/>
 <br/>
 新唐人记者梁珍、赵若水、宋碧龙香港报导
 <br/>
 <br/>
</div>
