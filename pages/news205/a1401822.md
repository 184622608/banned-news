### 判朱凯迪提名无效 泛民批港府操控选举
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港立法会.html" target="_blank">
  香港立法会
 </a>
 议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱凯廸.html" target="_blank">
  朱凯廸
 </a>
 ，早前报告参加乡郊代表选举，被选举主任裁定提名无效，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 立法会议员周一发表声明，强烈谴责政府扩大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治审查.html" target="_blank">
  政治审查
 </a>
 ，操控选举。
 <br/>
 <br/>
 24名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 立法会议员星期一召开记者会发表联合声明，强烈谴责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港府.html" target="_blank">
  港府
 </a>
 进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治审查.html" target="_blank">
  政治审查
 </a>
 ，将《基本法》104条原本不包括乡郊代表选举任意划红线，将政治审查延伸至明年区议会选举，封杀〝政治不正确〞的参选人，操控选举，制造白色恐怖。
 <br/>
 <br/>
 立法会议员毛孟静：〝于此中美贸易战际，林郑政府又一政治打压举措，不单威吓本地羣体，亦会进一步加深国际社会的印象：香港自由确受严重伤害、一国两制果然渐趋消失。贸易战火若波及香港，林郑必成历史罪人。〞
 <br/>
 <br/>
 朱凯迪议员批评港府跟随北京进一步破坏香港的言论自由。
 <br/>
 <br/>
 立法会议员朱凯迪：〝其实他那条红线，要令到我们不能继续保护香港人的言论自由、结社自由及集会自由。〞
 <br/>
 <br/>
 他强调民主派未来要守住三条战线，包括舆论上勿让亲共建制派扭曲事实、议会内守住战线，以及到各区争取民心。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
 <br/>
