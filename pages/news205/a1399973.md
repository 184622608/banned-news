### 伞运占中九子受审 考验香港自治与法治
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】2014年一场〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雨伞运动.html" target="_blank">
  雨伞运动
 </a>
 〞震惊国际，港人的和平理性的表现赢得国际赞赏。不过四年后，包括〝占中三子〞戴耀廷、陈健民和朱耀明等9名雨伞运动参与者，被控煽惑他人犯公众妨扰等罪名，案件星期一正式开审，9人全不认罪，批评当局政治检控。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_占中九子.html" target="_blank">
  占中九子
 </a>
 案〞星期一早上在西九龙裁判法院正式开审，吸引大批中外传媒到场采访。九名被告包括三名〝占中〞发起人戴耀廷、陈健民和朱耀明，立法会议员陈淑庄及邵家臻，社民连副主席黄浩铭、学联前常务秘书钟耀华、学联前常委张秀贤以及民主党成员李永达。近百支持者到场打气。
 <br/>
 <br/>
 画面：公民抗命、无畏无惧、和平抗争、无怨无悔〞（拉横幅的画面）
 <br/>
 <br/>
 九名被告在庭上否认全部控罪，下午四点首日聆讯结束后发表联合声明并会见记者。
 <br/>
 <br/>
 占中发起人陈建民：〝即是此案成立的话，在香港会产生寒蝉效应，甚至可利用来清洗香港反对力量。我们见到煽惑罪，我们最不能接受的是香港市民走出来参与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雨伞运动.html" target="_blank">
  雨伞运动
 </a>
 ，是因为831决定根本违反基本法，是因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梁振英.html" target="_blank">
  梁振英
 </a>
 政府用催泪弹攻击和平的示威者，所以‘煽惑’这么多港人走出来，这个才是真正历史真相。〞
 <br/>
 <br/>
 占中发起人戴耀廷：〝其实今次诉讼，我觉得不只是我们九人面对诉讼，其实今次诉讼是香港的高度自治和法治也同时被审判。〞
 <br/>
 <br/>
 案件预计审讯二十日。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
