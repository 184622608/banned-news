### 港府政治封杀刘小丽参选引发议员抨击
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】香港民主派人士刘小丽
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参选.html" target="_blank">
  参选
 </a>
 立法会九龙西补选遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港府.html" target="_blank">
  港府
 </a>
 政治封杀持续引发关注，民主派议员批评港府再次跟随北京剥夺异见者参选的政治权利。
 <br/>
 <br/>
 多位民主派议员在星期一下午的立法会政制会议上，批评政府没有按程序公义予刘小丽答辩机会，就剥夺她
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参选.html" target="_blank">
  参选
 </a>
 立法会九龙西补选的资格，无意是跟随北京的政治封杀。
 <br/>
 <br/>
 公民党议员郭家麒：它认为你是对立面就安插一个名目，说你是港独，说你不尊重基本法，将该人政治权利终身剥削，差过大陆。大陆那些民运人士剥夺你政治权利三年五年，现在我们香港是终身的。你是否将香港沦落到这个田地。〞
 <br/>
 <br/>
 议会阵线议员毛孟静：你是极尽扭曲之能事，但这你是对不起香港人，你整个政府是愧对香港人，你现在纯綷鹦鹉学舌，重重覆覆说一堆差不多的说话，你解释不到自己。
 <br/>
 <br/>
 政制及内地事务局局长聂德权回应表示，有人不满选举主任的决定，可以向法院提出选举澄清。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报道
 <br/>
 <br/>
</div>
