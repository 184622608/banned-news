### 民主派再失议席 李卓人吁港人勿灰心
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九龙西补选.html" target="_blank">
  九龙西补选
 </a>
 尘埃落定，民主派再失一席，无法保住立法会分组点票的否决权，落败的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李卓人.html" target="_blank">
  李卓人
 </a>
 表示，民心不死，香港仍可以赢。
 <br/>
 <br/>
 香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九龙西补选.html" target="_blank">
  九龙西补选
 </a>
 星期一凌晨3时公布结果，民主派候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李卓人.html" target="_blank">
  李卓人
 </a>
 以万余票落败，输给建制派候选人陈凯欣。李卓人向支持他的选民致歉，提到竞选团队的辛劳，各民主派的助选，看到民主派的团结力量，形容是打了一场漂亮的仗。他呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港人.html" target="_blank">
  港人
 </a>
 可以失望，但绝对不能放弃。
 <br/>
 <br/>
 民主派候选人 李卓人：〝我一定不会放弃，我们相信整个民主派团队及支持者不会放弃。因为香港未来还有很多场仗要打，包括大家看到的来势汹汹的基本法23条立法，大家看到的林郑人工岛，一万亿的超级大白象工程。〞
 <br/>
 <br/>
 民主派也在当日下午举行选后记者会，承认选举失利，并就事件向支持者鞠躬道歉。
 <br/>
 <br/>
 民主派召集人毛孟静：〝我们失望并不绝望，不言放弃，香港的民主路仍然漫长，我们痛定思痛，承诺会继续奋斗，希望市民与我们一起并兼作战，为香港、为民主、为下一代，坚持到底。〞
 <br/>
 <br/>
 李卓人料未来中共会继续收紧，越来越多国家机器的资料投入香港吃掉香港，但相信民心不死，香港仍可以赢。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
 <br/>
