### 香港发生严重车祸 5死逾30伤
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 周五清晨发生严重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_车祸.html" target="_blank">
  车祸
 </a>
 ，造成5人死亡，超过30人受伤，其中10多人伤势严重。
 <br/>
 <br/>
 当地时间周五清晨5点左右，一辆接载国泰航空公司员工的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旅游大巴.html" target="_blank">
  旅游大巴
 </a>
 撞上了一辆因为故障而停在路边的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_计程车.html" target="_blank">
  计程车
 </a>
 ，造成计程车司机和4名大巴乘客死亡，另外有32人受伤，其中10多人伤势严重。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_车祸.html" target="_blank">
  车祸
 </a>
 后，计程车的车尾被撞到严重变形，旅游巴士车头的挡风玻璃整块脱落，左边车头和右边车尾受损，部份玻璃窗也破裂。
 <br/>
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
