### 【禁闻】中共驻港军队擅自出动 破坏驻军法
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】在港府没有请求协助的情况下，身穿迷彩制服的驻港中共军队，日前在多个郊野公园清理风灾破坏物。事件引发港人担忧，也有评论结合日前民族党遭取缔、马凯签证遭拒、刘小丽立法会参选资格被取消等，指出中共的大陆化政策日趋明显。
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 网页，10月13号，包括驻港中共军队在内的三个中共驻港机构和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中资企业.html" target="_blank">
  中资企业
 </a>
 一行600多人，到新界麦理浩径做义工，清理超强颱风〝山竹〞造成的路障和垃圾，历时8个小时。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 一些市民对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻港军队.html" target="_blank">
  驻港军队
 </a>
 的行动感到忧虑，香港政务司司长张建宗14号表示，驻港军队是参与中联办及中资企业协会牵头的公益义工活动，事前有通知渔护署，政府在当中没有扮演任何角色，也没有要求驻港军队协助，希望外界不要对事件过分演绎。他还说，驻军的出发点是为香港好，对此表示感激。
 <br/>
 <br/>
 香港记协主席杨健兴：〝这个可以说是第一次吧，他们自己出来，公开做一些行动，虽然跟安全，跟香港的防卫没有关系，但是印象还是不好的，因为香港人对解放军还是比较敏感，加上他们在清理工作的时候还是穿军服，这观感上就是不太好。香港政府的官员说香港人不要过份敏感，但是他们其实低估了解放军，香港人对他们的敏感性。〞
 <br/>
 <br/>
 香港《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻军法.html" target="_blank">
  驻军法
 </a>
 》第9条规定，香港驻军不干预香港的地方事务。第14条也称，根据《基本法》，港府必要时可向中央政府请求香港驻军协助，维持社会治安和救助灾害。
 <br/>
 <br/>
 香港资深大律师、公民党主席梁家杰指出，在港府没有请求协助的情况下，香港驻军擅自行动，这违反了驻军法。
 <br/>
 <br/>
 香港资深大律师、公民党主席梁家杰：〝除非香港特别行政区政府根据基本法的规定，向中央政府请求香港驻军协助维持社会治安和救助灾害，否则，他们是不能在驻军的地点以外有活动的。我们的政务司司长也在昨天早上，在一个电台的节目里头清楚的确认，特区政府没有向中央政府请求香港驻军协助救灾，这一次以当义工为名来破坏驻军法，是我最关注的一点。〞
 <br/>
 <br/>
 立法会议员谭文豪也指出，驻港军队若能以处理风灾为由随便出动，日后将难以界定驻军行动的界线。
 <br/>
 <br/>
 香港在1997年主权转移前夕，美国《财富》杂志的封面故事〝香港之死〞（The Death of Hong Kong），曾引爆香港社会及国际舆论。香港记者协会主席杨健兴表示，再次读〝香港之死〞，会感受到当年的论述正在陆陆续续的实现，中共的〝大陆化〞政策日趋明显。
 <br/>
 <br/>
 香港记者协会主席杨健兴：〝最近取消了一些民主派的人士立法会的一些资格，包括把民族党成为一个非法组织，都是一个共同点，都是要强调一国的原则，包括最近不发工作证给金融时报的编辑，就跟内地处理外国媒体的做法很近。另外，解放军出来跟社会融合，中国政府现在要香港人慢慢的适应，让人觉得中国就是大陆化或者内地化的政策，是越来越明显了。〞
 <br/>
 <br/>
 梁家杰：〝最近在香港发生的这一连串的事件，我觉得非常的痛心。香港本来可以向世界展示自由民主国家奉行多年的价值跟制度，在一个中国人的地方，是如何可以落实，而且不单只是落实，还是成功的，香港的这个经验是可以为二元对立的世界格局提供一条出路，特区政府现在好像一面倒的去跟内地中国模式站在一线，我想是很不明智的一个举动。〞
 <br/>
 <br/>
 不过，梁家杰表示，港人会坚持原有的价值和制度，香港还是有希望的。
 <br/>
 <br/>
 梁家杰：〝香港原来的价值跟制度在最近的几年，的确受到很大很大的冲击，但是我不会宣判香港已死，因为不同世代的香港人，包括一些年轻人，我们没有放弃香港，还是坚持我们原来的价值跟制度。〞
 <br/>
 <br/>
 梁家杰指出，邪不压正，只要坚定信念，不忘初衷，争取人民福祉的这一方，必定会胜利。
 <br/>
 <br/>
 采访/常春 编辑/陈洁 后制/
