### 马凯被拒入境 民主派斥港府箝制新闻言论自由
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】香港外国记者会（FCC）副主席、《金融时报》亚洲新闻编辑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马凯.html" target="_blank">
  马凯
 </a>
 星期四以游客身份来港被拒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入境.html" target="_blank">
  入境
 </a>
 。事件引起各界关注及谴责。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 星五发表联合声明，强烈的抗议和谴责特区政府，箝制新闻和言论自由，并要求特首林郑月娥亲自交代和就事件道歉。
 <br/>
 <br/>
 香港外国记者会副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马凯.html" target="_blank">
  马凯
 </a>
 继十月初不获特区政府续发工作签证后，星期四以旅客身分访港，遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入境.html" target="_blank">
  入境
 </a>
 处官员查问数小时后，最终被拒绝入境。
 <br/>
 <br/>
 24位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主派.html" target="_blank">
  民主派
 </a>
 立法会议员星期五发表联合声明，指事件反映特首林郑月娥不惜一切代价将事件升温，向国际社会表明香港政府是北京的傀儡，并摆明进一步政府箝制新闻自由和言论自由。
 <br/>
 <br/>
 民主派会议召集人毛孟静：〝不仅将香港本土异见声音噤声，也是向全世界表示，如你作为外人你的言论或行为是被香港政府权力机关认为是政治不正确，你将被禁足进入香港，这信息只会进一步打击香港作为国际城市的地位，吓窒外国投资者。〞
 <br/>
 <br/>
 有议员透露已有外国投资者押后对香港的投资。
 <br/>
 <br/>
 资讯科技界议员莫乃光：〝我真真正正知道有大型国际公司对香港大型，数以百亿的投资因为之前马凯事件被押后，不知会否做，这是香港一个很实实际际的损失，不单止是对我们一国两制的损失。〞
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
