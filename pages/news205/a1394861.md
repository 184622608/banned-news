### 港特首公布施政报告 泛民批向中共表忠
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 的局势下，香
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港特首.html" target="_blank">
  港特首
 </a>
 林郑月娥星期三发表其任内的第二份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_施政报告.html" target="_blank">
  施政报告
 </a>
 ，民主派议员批评报告向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共表忠.html" target="_blank">
  中共表忠
 </a>
 ，无视香港两制，是一个大陆化的施政报告。
 <br/>
 特首林郑月娥的第二份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_施政报告.html" target="_blank">
  施政报告
 </a>
 除了重点着墨在土地房屋等民生政策，对港人关注的政改问题，仅以94个字说明，同时又强调将继续引用《社团条例》取缔危害国家安全的行为。民主派议员批评报告政治上只见〝一国〞不见〝两制〞，同时首次加入〝国家安全〞字眼。
 <br/>
 <br/>
 香港立法会议员毛孟静：〝她的政策方向〝一带一路、大湾区〞，到结语时香港未来发展重燃希望又是〝一带一路、大湾区〞，林郑月娥这种不单止将香港融入，简直是溶化，的〝溶〞，大家看到香港随时在2047前已被她消失。〞
 <br/>
 <br/>
 林郑月娥发表施政报告时场内外皆有抗议声。当上午11时林郑步入会议厅时，民主派议员不满外国记者会第一副主席马凯工作签证不获续期，高喊抗议口号，部份议员离场，毛孟静等四人被逐出会议厅。
 <br/>
 <br/>
 公民党党魁杨岳桥：〝今天选择离席是要表达对今次马凯事件破坏一国两制的痛心，也非常国际社会会以为香港人对此事若无其事，所以我们用行动表示我们的不满。〞
 <br/>
 <br/>
 对于《金融时报》将就马凯被拒工作签证续期一事提出上诉，林郑发表施政报告后离开时没有回答。
 <br/>
 <br/>
 新唐人记者林秀宜香港报道
</div>
