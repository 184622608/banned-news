### 麒麟韦尔苦主赴港维权 项目曾被中共力炒
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】近日，又有一批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麒麟韦尔.html" target="_blank">
  麒麟韦尔
 </a>
 大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苦主.html" target="_blank">
  苦主
 </a>
 ，来港投诉堕入金融骗局，血本无归。有苦主披露，去年中共媒体曾参与炒作此计画。
 <br/>
 <br/>
 去年6月，多份中共官媒，包括大公网、环球网、中国日报网、中宣部重点联系的、中央级期刊《中华英才》纷纷报导一个名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麒麟韦尔.html" target="_blank">
  麒麟韦尔
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融平台.html" target="_blank">
  金融平台
 </a>
 大蓝筹计划〞。《中华英才》海外版总编辑还出席了麒麟韦尔的活动。
 <br/>
 <br/>
 湖南麒麟韦尔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苦主.html" target="_blank">
  苦主
 </a>
 ：〝他的《中华英才》的书给了我们，让我们觉得非常踏实，他们的宣传也让我们觉得非常认可。〞
 <br/>
 <br/>
 但今年八月，麒麟韦尔金融平台一夜间关闭，投资者血本无归。这对七十岁湖南夫妇100多万的养老金没了，被迫来港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 。
 <br/>
 <br/>
 湖南七十岁苦主梁女士：〝今年5月份脑干出血，在ICU病房抢救过来了，我前天来香港的，我还是在病中。〞
 <br/>
 <br/>
 湖南七十四岁苦主卢先生：〝他们拿到香港炒股，再说我们的钱不能直接付到香港，股市我们又不懂，只能付给上面。〞
 <br/>
 <br/>
 湖南麒麟韦尔苦主：〝爸爸妈妈的工资，弟弟妹妹的工资，他们怨死我了，说实话，我投了这个钱，全家想死的心情都有了。〞
 <br/>
 <br/>
 苦主控诉，今次上当受骗的原因，除了诱人的3-5倍回报率外，还因为有骗徒打者香港上市公司和央企的名义，以香港金融项目包装，希望港府调查，堵绝金融漏洞，协助他们追讨损失。
 <br/>
 <br/>
 新唐人记者梁珍、陈仲明香港报导
</div>
