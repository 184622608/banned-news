### 〝山竹〞刚过就上班 交通混乱市民轰港府
------------------------

<div class="wysiwyg">
 【新唐人2018年09月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山竹.html" target="_blank">
  山竹
 </a>
 〞令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 各区满目疮痍，造成交通运输大乱，教育局宣布周二所有学校再停课一天。
 <br/>
 <br/>
 港九新界各区许多大街小巷都布满被强风吹倒的大树、招牌、杂物等等，甚至有大客车被大树压毁挡在路中。由于主要干线的巴士停驶，港铁只做有限度的通车，导致数以十万计的打工人士赶上班，挤爆在港铁站，尤其以东铁情况最为严重，许多市民苦等数小时才能搭上车。
 <br/>
 <br/>
 70岁张太太：〝等了很久，空气很焗，个个都冒汗，个人很晕很辛苦，旁边有些人晕了。等了近2个小时才上到月台。〞
 <br/>
 <br/>
 对于港府为何不宣布停工，特首
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林郑月娥.html" target="_blank">
  林郑月娥
 </a>
 则呼吁大家互相忍让体恤。
 <br/>
 <br/>
 特首林郑月娥：〝我想在这样的情形下，由雇主、雇员，大家以一个互谅互让，体恤的方法做，是更加适合现在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 的情况。〞
 <br/>
 <br/>
 香港最大独立工会职工盟谴责林郑不知民间疾苦。
 <br/>
 <br/>
 职工盟秘书长李卓人：〝我们谴责林郑没有宣布全面公务员停工，尤其是非必要的公务员停工，因为公务员上班也要进入交通系统大家争上班。〞
 <br/>
 <br/>
 民主党及公民党也谴责政府无视社会停工声音，令上班市民民怨沸腾，草菅人命。
 <br/>
 <br/>
 新唐人记者林秀宜香港报导
</div>
