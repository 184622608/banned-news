### 菲国捣毁厨房式制毒工场 逮5名香港黑帮嫌犯
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月27日讯】菲律宾执法部门25日捣毁一宗涉及市值7.27亿菲律宾披索（约1400万美元）冰毒的毒品案，拘捕5名有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港黑帮.html" target="_blank">
  香港黑帮
 </a>
 背景的嫌犯。据悉，5名嫌犯与香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_和胜和前坐馆.html" target="_blank">
  和胜和前坐馆
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泰国大毒枭.html" target="_blank">
  泰国大毒枭
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海仔.html" target="_blank">
  上海仔
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭永鸿.html" target="_blank">
  郭永鸿
 </a>
 的贩毒集团有关。
 <br/>
 <br/>
 综合媒体报导，菲律宾缉毒署于24、25日在首都马尼拉，先后拘捕5名涉及制毒贩毒的香港男子，检获107公斤冰毒及大量制毒原料，估计市值7.27亿菲律宾披索。被捕疑犯其中3人有黑帮和胜和背景，另外2人有黑帮14K背景。
 <br/>
 <br/>
 缉毒署发言人26日公布称，涉案制毒工场如厨房式的制造冰毒，经法医毒物化学专家的评估及考虑现场发现的制毒原料，据信每周可制造50公斤冰毒，与当局过往捣破的制毒工场，每周只能制造10公斤冰毒或其他毒品更甚。
 <br/>
 <br/>
 发言人也表示，透过情资，嫌犯可能从菲律宾南部把毒品走私入境，特别是三宝颜半岛。而其毒品市场除了马尼拉，还有针对吕宋岛中部及卡拉巴松区的市场。
 <br/>
 <br/>
 由于此次捣毁的制毒工场位于公寓楼内，当局正调查业主是否明知故犯，未阻止毒贩制毒。
 <br/>
 <br/>
 菲律宾总统杜特尔特（Rodrigo Duterte杜特蒂）一上任即雷厉风行展开扫毒行动，但至今仍无法杜绝贩毒交易。缉毒署署长阿基诺说，有被捕的制毒犯人对他说，该国尚未恢复死刑，部分官员又容易受贿及贪污，因此毒贩会继续在菲律宾设置更多制毒工场。
 <br/>
