### 港珠澳大桥下周三通车 传习近平主礼
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】耗资过千亿港元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 下星期二（23日）在珠海举行开通仪式后，下星期三（24日）正式通车。外传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 很大可能主持开通仪式，不过港府称未收到大陆方面的通知。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 开通仪式下星期二在珠海举行，港府星期五起一连二日安排传媒及立法会议员参观香港口岸及连接路。传媒安排专车前往香港口岸人工岛及港珠澳大桥主桥。并首次安排拍摄香港口岸人工岛旅检大楼，市民到达香港口岸后，可在柜位或自动售票机，购买穿梭巴士车票，往来珠海及澳门口岸。
 <br/>
 <br/>
 之后运输及房屋局局长陈帆联同保安局、运输署等多个部门代表在旅检大楼会见传媒，简介政府就港珠澳大桥开通的准备工作。
 <br/>
 <br/>
 陈帆宣布大桥将于本月23日上午9时在珠港口岸大楼举行通车仪式，24日上午9时正式通车营运。将来珠海到香港机场只需要45分钟。
 <br/>
 <br/>
 早前不少传媒报导
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 很大可能在当日早上主持开通仪式，陈帆表示未收到大陆方面的通知。
 <br/>
 <br/>
 运房局局长陈帆：〝这个大桥的开通仪式是由谁来担当主持嘉宾呢？我们到今天仍未知道，因为开通仪式的举办方是内地，所以有关的消息需等待内地一方的公布。〞
 <br/>
 <br/>
 港珠澳大桥采三线双程行车，由于跟随大陆的行车安排，路面都变成〝右上左下〞。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
