### 香港众志疑中共施压联合国删意见书
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】联合国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权理事会.html" target="_blank">
  人权理事会
 </a>
 星期二（11月6日）审议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人权.html" target="_blank">
  中国人权
 </a>
 之际，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港众志.html" target="_blank">
  香港众志
 </a>
 在内的至少七个团体的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意见书.html" target="_blank">
  意见书
 </a>
 被删除，秘书长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄之锋.html" target="_blank">
  黄之锋
 </a>
 质疑中共施压联合国。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港众志.html" target="_blank">
  香港众志
 </a>
 秘书长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄之锋.html" target="_blank">
  黄之锋
 </a>
 傍晚6时紧急会见传媒，他指，联合国人权事务高级专员办事处在9月3日公布团体摘要报告将香港众志列入〝团体名单〞，香港众志提出的改善香港人权自由的建议，也被引述于改善措施的章节当中。但到10月份，重新上载时，包括香港众志等7个团体名字及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意见书.html" target="_blank">
  意见书
 </a>
 消失了。原因是必须尊重有关国家的主权、独立及领土完整性。他直言，明显是北京施压的缘故。
 <br/>
 <br/>
 香港众志秘书长黄之锋：〝我想今次事件代表我们所讲的中国干预或北京施压其实不限于香港、澳门、大陆或者台湾，不论巴丢草事件、远至澳洲，至联合国审议程序都可以夸张到明明份文件就交给联合国，9月上网都见到，但10月今次就整份文件被消失，而无任何解释，都反映整个联合国普遍定期审议程序是缺乏透明度。〞
 <br/>
 <br/>
 他表示，将跟人权观察、国际特赦组织等团体联络跟进，希望尽快得到人权高专办的回应。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
