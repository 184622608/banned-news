### 占中九子受审不改初衷 法院外上百人声援
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】周一，香港西九龙
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法院.html" target="_blank">
  法院
 </a>
 开庭审理占中，又称雨伞运动发起人戴耀廷、陈健民、朱耀明等九人，他们被控煽惑他人作出公众妨扰等罪。戴耀廷等九人不改要求真普选初衷，全部否认控罪。上百民众聚集在法庭外，表达对他们的支持。
 <br/>
 <br/>
 〝我要真普选〞、〝公民抗命、无畏无惧〞、〝我要真普选〞。
 <br/>
 <br/>
 周一上午在法庭外，上百名支持者呼喊占中时的口号，支持九位雨伞运动发起人。
 <br/>
 <br/>
 九位受审的发起人有：〝香港大学法学副教授戴耀廷、香港中文大学社会学副教授陈健民、基督教牧师朱耀明，以及立法会议员陈淑庄和邵家臻等。〞
 <br/>
 <br/>
 检方指控他们煽惑他人作出公众妨扰等罪，但他们全部否认。
 <br/>
 <br/>
 2014年，为了争取普选，历时79天的占领中环运动，被认为是香港有史以来最大规模的民主运动。
 <br/>
 <br/>
 路透社香港办公室主任庞弗雷特：〝此案对香港来说是一个里程碑。很多批评者认为，这确实是香港政府推动的政治迫害，试图压抑和遏止民主运动前进的势头。〞
 <br/>
 <br/>
 如果被判有罪，法庭可对个人判处最高七年监禁。
 <br/>
 <br/>
 国际特赦组织和人权观察组织在庭审前发表声明，批评这次审判威胁香港市民表达自由，要求港府撤销指控。
 <br/>
 <br/>
 同时，英国八名跨党派议员联署提案，谴责港府压抑民主，要求就此提出辩论。
 <br/>
 <br/>
 这是〝雨伞运动〞核心人物首次受审。去年八月，涉及学生领袖黄之锋、周永康、罗冠聪等人的案件已经审理结束。
 <br/>
 <br/>
 另外，〝雨伞运动〞期间有900多人被捕；公诉方律师表示，警方仍在进行相关调查。不少人周一排队进入法庭旁听，井然有序。
 <br/>
 <br/>
 新唐人记者李梅、明玉综合报导
</div>
