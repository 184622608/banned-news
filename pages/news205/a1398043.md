### 选情告急 港民主派接力巡区造势
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】离11月25日九龙西补选约三周时间，民主派一连二日接力巡区为候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李卓人.html" target="_blank">
  李卓人
 </a>
 造势拉票。李卓人强调今次补选是民主派与中共、港府及保皇派的一次公投，呼吁港人支持民主派候选人向中共还以颜色。
 <br/>
 <br/>
 香港民主党及公民党周末日（11月3日及4日）一连二天空群而出，分别在九龙城、美孚、大角咀巡区，呼吁港人全力动员票投民主派候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李卓人.html" target="_blank">
  李卓人
 </a>
 。
 <br/>
 <br/>
 民主党元老
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李柱铭.html" target="_blank">
  李柱铭
 </a>
 ：〝今次一定要出来，千万不要因为说不是那么紧要吧，一个位，这个位很重要，因为李卓人不赢，就是共产党的人赢，就那么简单。〞
 <br/>
 <br/>
 公民党议员郭家麒：〝所以我们见到现在我们的对手，即是所有民主派的对手，只得一个，就是躲在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_建制派.html" target="_blank">
  建制派
 </a>
 候选人背后的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 ，不论它用哪个头衔，用民建联也好，用工联会也好，也是同一老板。〞
 <br/>
 <br/>
 李卓人重申今次选举口号是〝为下一代背水一战〞，呼吁港人不要灰心，香港未来绝对有希望。
 <br/>
 <br/>
 候选人李卓人：〝今次的选举本身不是选我李卓人一个人，是选一个理念一个信念，就是整个民主派我们的信念，这是一个公投，一边是中国共产党及林郑月娥，他们没有制度公义。...一定要取回我们香港人的命运，我们的未来是属于香港人。〞
 <br/>
 <br/>
 今次香港九龙西候选人还包括陈凯欣、冯检基、曾丽文和伍廸希。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
</div>
