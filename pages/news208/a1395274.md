### IMF会员国承诺  不操纵货币恶意贬值
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】周六，国际货币基金组织，发布公报，表示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_会员国.html" target="_blank">
  会员国
 </a>
 承诺，不会将货币贬值，作为竞争手段，以获取本国的利益。分析认为，声明的目的，或在于缓解，中美贸易摩擦，尤其是美方，对人民币快速贬值，表达的担忧。
 <br/>
 <br/>
 在印尼巴厘岛举行的国际货币基金组织年会，周六进入尾声。美中贸易摩擦成为焦点议题。大会当天发表公报，呼吁加强贸易领域的对话。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_IMF.html" target="_blank">
  IMF
 </a>
 年会主席  南非储备银行行长  Lesetja Kganyago：〝公报认可当前需要加强对话、采取行动降低风险，增强对国际贸易的信心。〞
 <br/>
 <br/>
 本周，有消息传出川普政府正在考虑将中共列为〝货币操纵国〞。而IMF周六发表的公报中，也誓言各
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_会员国.html" target="_blank">
  会员国
 </a>
 不会恶意
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_操纵货币.html" target="_blank">
  操纵货币
 </a>
 。
 <br/>
 <br/>
 公报中说，〝我们承认自由、公平和互惠的交易与投资，是拉动经济和就业增长的关键动力〞，〝我们将遏制竞争性的货币贬值，并且不会将汇率作为实现竞争性目标的工具。〞
 <br/>
 <br/>
 自今年4月底以来，人民币对美元汇率贬值超过8%，目前已接近7比1大关。美国财长姆努钦，此前多次表达对人民币快速贬值的忧虑。川普政府担心，这是中共为了抵消美国加征关税所采取的手段。
 <br/>
 <br/>
 据路透社报导，中共央行行长易纲，周六在IMF年会上承诺，不会把人民币汇率作为解决贸易摩擦的工具。
 <br/>
 <br/>
 不过，中共至今对美方提出的盗窃知识产权等问题矢口否认。分析认为，美方会在多大程度上相信易纲的承诺，仍然是未知数。
 <br/>
 <br/>
 此外，姆努钦也透露，川普和习近平是否会在11月的G20峰会上会面，美方还没有做出最后决定。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
