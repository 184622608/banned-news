### 担忧贸易战前景 美股再跌 苹果销量不振
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】周二，由于市场对美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 前景担忧，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 暴跌。道琼斯指数下跌近800点，纳斯达克指数跌幅超过3%。
 <br/>
 <br/>
 受汇丰银行分析师下调评级的影响，苹果股票今日开盘前的交易中下跌2％。周二，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 来说是艰难的一天。
 <br/>
 <br/>
 截至收盘苹果股价跌超4％，道琼斯指数下跌3.1％，标准普尔指数下跌3.24％，纳斯达克指数暴跌3.8％。
 <br/>
 <br/>
 随着股价下跌，近期苹果公司的美国最大市值公司的地位一直受到亚马逊和微软的挑战。
 <br/>
 <br/>
 苹果的供应商之一Cirrus Logic削减了其收入预期，越来越增加了投资人对iPhoneXS和XR销售不佳的担忧。Cirrus股价也下跌。
 <br/>
 <br/>
 美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的不确定性对苹果和其他科技公司带来负面影响。如果美方提高关税，将会影响从中国进口的苹果手机和笔记本电脑的利润。
 <br/>
 <br/>
 富士康的高管告诉路透社，富士康这家苹果最大的手机装配商正在考虑在越南建厂，以避免贸易战的影响。
 <br/>
 <br/>
 苹果已经决定停止每季度披露销售手机的数量，此举被解读为iPhone销量进一步下滑的迹象。这让投资者感到不安。
 <br/>
 <br/>
 目前，苹果网站正在进行iPhone 手机以旧换新的促销活动，用户可以用旧手机来折扣购买新iphone。
 <br/>
 <br/>
 由于新手机相比之前型号的手机创新较少，更多人选择长时间地使用目前的手机而不是升级。
 <br/>
 <br/>
 新唐人记者jeremy Sandburg纽约报导
