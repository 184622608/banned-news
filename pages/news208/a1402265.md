### 美俄希望低油价 OPEC减产计划受制衡
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】周四，石油输出国组织OPEC在维也纳举行会议，15个成员国虽然暂时同意削减产量，但推迟了决定时间，外界认为这与美国和俄罗斯希望维持当前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 的愿望有关。
 <br/>
 <br/>
 记者：〝你有信心明天达成协议吗？〞
 <br/>
 <br/>
 沙特能源部长 法利赫：〝没有，我没有信心〞
 <br/>
 <br/>
 OPEC在维也纳周四的会议结束后，该组织的实际领导者沙特，对周五达成协议并不乐观。
 <br/>
 <br/>
 今年11月，原
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 格下跌22%，12月6日继续下降。面对减产，OPEC希望明年1月开始减产以提振油价。
 <br/>
 <br/>
 沙特能源部长法利赫周四表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_OPEC组织.html" target="_blank">
  OPEC组织
 </a>
 和其盟友每天减产100万桶原油就可以接受。但这个数字低于许多分析者预期的130万桶左右。
 <br/>
 <br/>
 不过，这一切要等周五与俄罗斯代表的会面后才可能决定。俄罗斯并不是OPEC成员，但其石油地位举足轻重。俄总统普京完全能接受当前的油价水平，低油价对俄罗斯紧张的经济状况也有好处。
 <br/>
 <br/>
 美国总统川普前一天也发推文敦促，希望OPEC能维持现状，称：世界不需要高油价！
 <br/>
 <br/>
 分析者认为，周四OPEC减产的决定推迟，是受到了美俄的意见影响。
 <br/>
 <br/>
 当天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国原油.html" target="_blank">
  美国原油
 </a>
 价格再下降超过3个百分点，来到每桶51.49美元。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
