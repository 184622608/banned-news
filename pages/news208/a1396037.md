### 10月18日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美国辉瑞药厂有意通过自愿离职和遣散的方式，裁减全球2%的员工，希望藉此精简公司架构，减少管理人员数量。目前辉瑞在全球的员工大约有9万人，将被裁减的人数为1千8百人。
 <br/>
 <br/>
 2.美国政府提供给特斯拉的减税优惠措施即将被缩减，特斯拉为了刺激买气，在其官网推出配备中程电池、订价4.5万美元的新版Model 3。目前特斯拉只有生产长程电池、价格自4.9万美元起跳的高价版Model 3。
 <br/>
 <br/>
 3.韩国央行表示，由于美中贸易战持续升温，以及韩国就业增长薄弱，决定维持主要利率在1.5%不变。不过因为美国联储局持续升息，可能造成外资出逃，韩国央行可能在11月或明年第一季升息。
 <br/>
 <br/>
 4.欧盟央行总裁德拉吉表示，违反欧盟设定的预算赤字上限不会带来经济繁荣，而且会令欧元区所有成员都付出高昂代价。意大利政府准备提高债务水平来兑现增加支出和减税的选举承诺。市场认为，德拉吉的发言暗指意大利政府不当举债。
 <br/>
 <br/>
 新唐人记者米西、正治综合报导
