### 9月25日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年09月26日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1. 周三，美联储结束为期两天的货币政策会议，市场普遍预期，美联储主席鲍威尔将宣布加息。今年，美联储已经两次升息。周二，美元小跌，市场聚焦美联储经济报告，以及未来的货币政策。
 <br/>
 <br/>
 2. 美国时尚品牌Michael Kors已经同意出资22亿美元，收购意大利精品品牌范哲思，继续进军高端时尚市场。Michael Kors以销售价格低于500美元的手提包起家，收购达成后，挂上范哲思牌子的提包，价格可能贵五倍。
 <br/>
 <br/>
 3. 豪华房车生产商宝马汽车宣布下调财务预测，今年汽车制造业务利润率将降到7%，无法实现8%到10%的目标。主要因为持续不断的国际贸易冲突，造成市场形势不确定，多个汽车市场遭受定价压力。
 <br/>
 <br/>
 4. 美国体育用品商耐吉公司公布上季财报，营收99.5亿美元，同比增长9.7%，得益于美国强劲的消费支出，以及海外市场的旺盛需求。其中北美市场营收增长6%，达到41亿美元，占总销售额约40%。鞋类增长5%，服装增长8%。
 <br/>
 <br/>
 新唐人记者密希、明玉综合报导
