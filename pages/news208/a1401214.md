### 美联储称经济前景稳固 道琼斯飙涨617点
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美联储.html" target="_blank">
  美联储
 </a>
 主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲍威尔.html" target="_blank">
  鲍威尔
 </a>
 在纽约表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 前景稳固，话音刚落，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 应声上涨。
 <br/>
 <br/>
 金融市场希望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美联储.html" target="_blank">
  美联储
 </a>
 将在2019年放缓或暂停加息。此前，美联储的预期是在12月份进行今年第四次加息，随后明年再加息三次。
 <br/>
 <br/>
 本月早些时候，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲍威尔.html" target="_blank">
  鲍威尔
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 状况持乐观态度，理由是GDP增长率高于3％，失业率近五十年低点的3.7％。
 <br/>
 <br/>
 在周三演讲中，鲍威尔表示，联储的货币政策没有预设的路径，仍将密切关注经济数据。鲍威尔还对全球经济增长放缓和贸易紧张等风险表示担忧。
 <br/>
 <br/>
 一般来说，较高的利率往往会减缓经济增速并让股市承压。
 <br/>
 <br/>
 美联储主席 鲍威尔：〝按历史标准衡量，利率仍然很低，而且它们仍然低于该水平的估计范围，这对经济来说是中性的。〞
 <br/>
 <br/>
 鲍威尔话音一落，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 大幅上涨，截至收盘，道琼斯指数上涨617点，标普上涨61.61，纳指上涨209点。
 <br/>
 <br/>
 分析师认为下月美联储仍可能会加息，但经济学家表示，如果2019年初股市波动加剧，消费者和企业信心继续恶化的话, 明年三次加息将变得不那么明确。
 <br/>
 <br/>
 新唐人电视Jeremy Sandberg纽约报导
</div>
