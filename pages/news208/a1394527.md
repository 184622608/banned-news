### 10月7日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】下面请看一组环球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财经简讯.html" target="_blank">
  财经简讯
 </a>
 。
 <br/>
 <br/>
 美联储公布8月份总
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信贷.html" target="_blank">
  信贷
 </a>
 增加201亿美元，创下5月以来最大增幅，高于市场预期，其中信用卡相关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贷款.html" target="_blank">
  贷款
 </a>
 增加48亿美元，汽车与学生贷款相关金额增加152亿美元，显示消费者愿意更自由地消费。
 <br/>
 <br/>
 美国国防部公布报告指出，美军在不可缺少的战略物资及技术供应上过度依赖中国，有超过280项可能漏洞，且风险正在扩大，虽然多国推行经济政策，损害美国制造业，但没有一个国家像中国成功锁定美国工业基础。
 <br/>
 <br/>
 美国汽车大厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福特汽车.html" target="_blank">
  福特汽车
 </a>
 通知员工，表示为了实施110亿美元的成本重组计画，将进行裁员，以扭转获利大幅下降的困境，改善财务状况，更加灵活的因应竞争压力，市场预估裁员人数将达2万多人。
 <br/>
 <br/>
 卡塔尔能源部长萨达表示原油市场供需保持平衡，最近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 上涨是地缘政治因素带来的影响。布兰特原油从今年早期的65美元每桶，上涨到现在的85美元。尼日利亚央行则预测，今年每桶石油将不会低于80美元。
 <br/>
 <br/>
 新唐人记者保罗、正治综合报导
