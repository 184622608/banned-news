### 9月28日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年09月28日讯】下面请看今天的财经速瞄
 <br/>
 <br/>
 意大利宣布大幅调高政府支出后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧洲股市.html" target="_blank">
  欧洲股市
 </a>
 今天应声下跌。投资人担心，这将导致意大利民粹派政府与希望该国减少负债的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 领袖发生冲突。伦敦富时100指数收盘下滑35.24点，巴黎CAC40指数跌46.92点。法兰克福DAX 30指数跌188.86点。
 <br/>
 <br/>
 9月27日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国联准会.html" target="_blank">
  美国联准会
 </a>
 宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_升息.html" target="_blank">
  升息
 </a>
 后，掀起全球升息风潮。印尼央行将基准利率从5.5%升至5.75%，是5月中旬以来第五次加息；菲律宾央行也宣布加息50个基点至4.5%。紧接其后的还有巴林、沙特、阿联酋、卡达央行相继将基准利率提高25个基点。香港金管局也将基准利率从2.25%上调至2.5%。
