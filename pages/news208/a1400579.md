### 11月23日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】下面是今天的财经速瞄
 <br/>
 <br/>
 周五是美国市场在感恩节假期后的首个交易日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 开盘后不断下跌，最终，WTI 1月原油期货收跌4.21美元，报50.42美元/桶。布伦特1月原油期货收跌3.80美元，报58.80美元/桶。目前，油价跌至2017年10月以来最低水平。
 <br/>
 <br/>
 由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华尔街.html" target="_blank">
  华尔街
 </a>
 担心持续贸易紧张及全球经济增长放缓前景，周五美股下跌。标普500指数收跌0.66%；道指收跌178.74点，报24285.95点；纳指收跌0.48%，创2011年以来最差
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑色星期五.html" target="_blank">
  黑色星期五
 </a>
 表现。
 <br/>
 <br/>
 受贸易战冲击，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_消费者.html" target="_blank">
  消费者
 </a>
 信心和汽车等产品的销售受到波及，中国经济日益恶化。香港《南华早报》称，中国第三季经济增长达10年来最低，与去年同期相比，10月份的汽车、房地产和消费性电子产品等销量均呈现下跌。
</div>
