### 苹果股价跌近5% 累及鸿海市值5年来新低
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果股价.html" target="_blank">
  苹果股价
 </a>
 大跌4.8%，累及合作伙伴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸿海.html" target="_blank">
  鸿海
 </a>
 集团，日前传出其子公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富士康.html" target="_blank">
  富士康
 </a>
 为了降低人力成本，将在一年内裁员34万。鸿海今(21日)开盘跌2.9%股价来到减资后低点68.9元，市值续跌破新台币兆元，是5年多来新低值。
 <br/>
 <br/>
 中央社报导，美股道琼指数大跌551.8点，其中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果股价.html" target="_blank">
  苹果股价
 </a>
 大跌4.8%，收在176.98美元，合作伙伴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸿海.html" target="_blank">
  鸿海
 </a>
 早盘股价疲弱，跌破70元关卡，最低来到68.9元，跌2.9%，续来到减资发行新股后低点。
 <br/>
 <br/>
 截至20日为止，外资法人已经连续12个交易日卖超鸿海累计达7万7604张，外资法人持股比重降到42.52%。
 <br/>
 <br/>
 从市值来看，若以鸿海减资后新股约138.63亿股粗估，鸿海今天早盘市值持续跌破新台币兆元大关，来到9551.6亿元。法人指出，这是2013年7月以来鸿海市值落在相对低值。
 <br/>
 <br/>
 不过第4季鸿海仍可受惠季节性旺季效应，欧系外资法人日前预估鸿海单季业绩可望季增16%到20%区间，有机会超过新台币1.6兆元，创历史单季次高，较去年同期历史新高下滑。
 <br/>
 <br/>
 华尔街日报引述知情人士报导，苹果新3款iPhone市场需求不如预期，最近几个星期苹果已大砍3款新iPhone的量产订单。其中iPhone XR需求问题尤其严重，苹果在10月下旬订单下修幅度达1/3。
 <br/>
 <br/>
 日系外资研究报告日前指出，根据查访供应链的结果，iPhone XR第4季恐砍单20%到30%，代表11月到明年1月零组件出货量，可能比原本预期减少40%到50%，将冲击供应链第4季甚至明年上半年的获利。
 <br/>
 <br/>
 陆媒〝IT之家〞20日报导，因苹果大幅削减订单，导致相应的代工厂、配件厂生意大减，而苹果的最大组装厂商鸿海集团（Foxconn Technology Group）为降低人力成本，也将于一年内裁减34万名员工。
 <br/>
 <br/>
 在此之前台湾《经济日报》报导，因明年全球经济情势不确定因素太多，鸿海集团为降低人力成本和固定费用或将裁员，对于总经理、副总裁等级高年薪的高层管理人员，也将依据贡献程度调整薪资。
 <br/>
 <br/>
 若鸿海实施上述裁员计划，预计将可节省逾500亿人民币的开支。
 <br/>
 <br/>
