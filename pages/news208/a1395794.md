### 10月17日财经速瞄
------------------------

 【新唐人2018年10月17日讯】下面请看今天的财经速描。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美联储.html" target="_blank">
  美联储
 </a>
 周三下午公布9月会议纪要，倾向于继续渐进加息，并将目标基金利率提高至2％至2.25％。会议纪要公布后，美股三大股指集体下行。10年期美国国债收益率上升至3.18％，而30年期国债收益率升至3.349％。

 <br/>
 10月16日盘后，美国四大科技股之一的

  奈飞
 </a>

 <br/>
 <br/>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特斯拉.html" target="_blank">
