### 美劳工部报告：平均工资增长九年新高
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】美国劳工部周五公布最新报告，美国10月非农就业增加25万人，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平均工资.html" target="_blank">
  平均工资
 </a>
 增长突破3%，创九年来新高。
 <br/>
 <br/>
 美国劳工部报告显示，10月非农就业人口增加25万人，高于市场预期20万人，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业率.html" target="_blank">
  失业率
 </a>
 维持在3.7%，仍然是48年来的低点。
 <br/>
 <br/>
 报告说，工作增加主要是健保、制造业、建筑业，以及交通运输和仓储。十月初飓风迈克尔登陆佛罗里达，但对全国就业数据没有明显影响。
 <br/>
 <br/>
 这次报告的亮点是薪资成长，10月劳工平均时薪增长3.1%，是2009年4月以来的最大增幅。
 <br/>
 <br/>
 美国劳动市场继续保持强劲势头，反映减税政策效应良好，为川普及共和党下周举行的期中选举增加信心。
 <br/>
 <br/>
 周五，川普发推文，祝贺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 持续呈现佳绩。
 <br/>
 <br/>
 <br/>
 新唐人记者易文、天羽综合报导
 <br/>
