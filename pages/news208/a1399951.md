 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国新国安法规.html" target="_blank">
  美国新国安法规
 </a>
 ，将扩大对外资投资美国机敏企业把关，包含生技、无线网络设备与奈米科技等相关行业，都受新法管制，未来这些企业无论外资的股权多小，都要评估是否需向海外投资委员会（CFIUS）申报。
 <br/>
 <br/>
 在亚太经合组织(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 )峰会中，欧美国家提出世界贸易组织(WTO)透明化，以及争端解决机制的改革，纳入会议结论，但是中方提出抗议，双方分歧无法达成共识，导致此次峰会创下首次未能发表共同宣言的纪录。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国大众汽车.html" target="_blank">
  德国大众汽车
 </a>
 (Volkswagen)宣布，到2023年，公司将投资440亿欧元，约500亿美元开发电动汽车、自动驾驶和新服务，并且在年底前与福特汽车，达成商务车合作协议，但是不打算与福特合并，也没有参股计划。
 <br/>
 <br/>
 国际评级机构惠誉发布最新报告指出，明年中国大陆房地产销量将下降5%到10%，地产开发商很可能面临更加艰难的市场状况，而亚太地区银行因对房地产行业的敞口相对较高，将面临更高的风险。
 <br/>
 <br/>
