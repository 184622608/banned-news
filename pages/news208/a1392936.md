 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美联储.html" target="_blank">

 </a>
 美股涨跌互现
 <br/>
 <br/>
 周二，国际油价大涨拉抬能源股，全球股市大多上扬，欧洲3大股市都收高，但投资人担心美中贸易战冲击，涨幅受限。美联储将于周三举行会议，市场静待再次升息，美股涨跌互现。标普收跌0.13％，道指跌约65点，纳指收涨0.18％。
 <br/>
 <br/>
 2. 又两位核心高阶主管宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离开脸书.html" target="_blank">
  离开脸书
 </a>
 <br/>
 <br/>
 周一晚，Instagram 两位创办人 Kevin Systrom 和 Mike Krieger宣布辞职，离开脸书。2012年 Instagram 被脸书以10亿美元并购，如今用户逾10亿人。脸书近期面临侵犯隐私权等许多问题，市值曾单日蒸发1200亿美元，创美股史上最惨纪录。今年1月的Statista调查更显示，逾2亿的美国用户里，将近17%的人删除脸书 app、9%移除脸书帐户。

 到机场试营运
