### 十二月购物季 德国消费者信心指数下降
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】12月圣诞节来临，是一年中的购物旺季，但今年12月，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国消费者.html" target="_blank">
  德国消费者
 </a>
 购物的兴趣预计比11月还要低。贸易摩擦和英国脱欧等事件，让消费者信心下跌。
 <br/>
 <br/>
 德国最大的市场调研机构GfK对2000名德国人进行了一项调查，12月消费者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信心指数.html" target="_blank">
  信心指数
 </a>
 为10.4，低于11月的10.6和市场预期的10.5。
 <br/>
 <br/>
 德国经济和国际形势紧密挂钩。全球贸易争端以及美国和欧盟之间有关进口汽车关税的拉锯战，打击了德国汽车出口。7月到9月之间，德国经济萎缩了0.2％。
 <br/>
 <br/>
 虽然如此，德国的就业率仍保持在高位，薪水上涨幅度高于通胀率，借贷成本低，这一切都在推动国内需求。但国际上的紧张局势导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国消费者.html" target="_blank">
  德国消费者
 </a>
 信心指数下降。
 <br/>
 <br/>
 德国市场调研机构GFK研究员 Rolf Buerkl：〝让一些消费者心神不定的因素有和美国之间的贸易争端，英国脱欧也是因素之一，还有目前乌克兰和俄罗斯之间的争端，再加上意大利预算引起的问题。〞
 <br/>
 <br/>
 在11月底的阿根廷20国集团峰会之前，欧美汽车关税争端再次成为话题。美国总统川普周一接受《华尔街日报》采访时又一次抨击欧盟在贸易上占美国便宜，并表示如果欧盟不跟美国达成公平协议，他12分钟内就可以下令对欧盟汽车加征关税。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
