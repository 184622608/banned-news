### 11月13日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.咖啡连锁店星巴克CEO，周二在一份备忘录中宣布，全球裁员350人，大部分在西雅图总部。主要影响西雅图支持中心的非店铺员工，不含咖啡店员工。这是星巴克9月宣布的重组计划的一部分。
 <br/>
 <br/>
 2.大众汽车表示，该公司最新的电动车平台，能够在未来几年生产5千万辆电动车，价格比特斯拉还要低。大众汽车已经和电池制造商签订价值480亿美元的电池供应合约，但是生产能力还没有到位。
 <br/>
 <br/>
 3.日本央行为了刺激物价上扬，启动资产收购计划。经过5年，持有的总资产价值已经超过国家经济产值，达到553万6千亿日圆。目前日本通胀率只有目标值2%的一半，未来还会持续推动资产收购计划。
 <br/>
 <br/>
 4.韩国10月的就业人口较去年同期增加6万4千人，达到2千709万人。失业率由9月的4.0%下降到3.9%，优于经济专家预期。其中，制造业就业人数减少4万5千人，连续7个月减少。
 <br/>
 <br/>
 <br/>
 新唐人记者米西、明玉综合报导
