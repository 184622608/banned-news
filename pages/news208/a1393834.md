### 10月2日财经速瞄
------------------------

 【新唐人2018年10月02日讯】下面是今天的财经消息
 <br/>
 <br/>
 10月第一个交易日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港股.html" target="_blank">
  港股
 </a>
 迎来开门黑。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造业.html" target="_blank">
  中国制造业
 </a>
 PMI数据放缓显示中国制造业活动萎缩。《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨加协议.html" target="_blank">
  美墨加协议
 </a>
 》达成后市场对中国经济前景更加忧虑，以及香港进入加息周期使得港股空头气氛浓郁。恒指收盘下跌2.4%，跌662.14点，日内一度跌超700点，以27126.38作收。燃气、航空等板块领跌，汽车、内银股大跌。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 周二宣布，提高该公司英美两国员工的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最低时薪.html" target="_blank">
  最低时薪
 </a>
 ，从11月1日起生效。所有美国员工的最低时薪升至15美元，这将使超过25万名亚马逊员工受益，包括兼职、临时工和季节性员工。英国的员工时薪上调至10.50英镑，将惠及3.7万名员工。
