### 亚洲股市下挫 全球增长担忧
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚洲股市.html" target="_blank">
  亚洲股市
 </a>
 周四下挫，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市.html" target="_blank">
  美国股市
 </a>
 隔夜大跌，创下自2011年与来最大单日跌幅，数千亿美元的资金流出全球市场。
 <br/>
 <br/>
 受隔夜
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市.html" target="_blank">
  美国股市
 </a>
 收盘重挫，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本股市.html" target="_blank">
  日本股市
 </a>
 日经指数周四早盘挫跌六个月最低位。日经指数早盘重挫3.5%至21,313.33。
 <br/>
 <br/>
 东证股价指数触及一年新低，跌3.0%至1,601.73点，导致市值于2017年9月以来首次降至600万亿日元以下。
 <br/>
 <br/>
 韩国股市周四盘中一度下跌3%，跌至2017年1月以来低点。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港恒生指数.html" target="_blank">
  香港恒生指数
 </a>
 HSI跌2.2%。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 也下跌，沪综指SSEC跌2.5%。
 <br/>
 <br/>
 澳大利亚股市触及一年低位。
 <br/>
 <br/>
 投资者越来越担心股市的高估值、企业获利趋势可能见顶。
 <br/>
 <br/>
 新唐人记者陈依心、宇薇综合报导
