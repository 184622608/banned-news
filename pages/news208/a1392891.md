  环球财经简讯

 <br/>
 <br/>
 1.传白宫拟命令 调查社交网络垄断
 <br/>
 <br/>
 消息指出，白宫正在起草行政命令，要求相关政府部门调查社交网络公司有没有违反袭断法，确保社交网络彼此间有竞争，并且解决立场偏颇问题，以保护民众免受社交网络垄断行为的影响。
 <br/>
 <br/>
 2.日拟与美签协议 降美农产品关税
 <br/>
 <br/>
 日本有意改变立场，考虑与美国达成双边贸易协议，并且降低美国农产品的进口关税，以换取美国不向日本进口汽车加征关税，日本希望参考与欧盟达成自由贸易协议的模式。
 <br/>
 <br/>
 3.美中难达协议 原因在中国
 <br/>
 <br/>
 在中国取消与美国贸易谈判之后，白宫国家贸易委员会主任纳瓦罗(Peter Navarro)表示，与中国建立新的贸易协议，可能会比墨西哥更加艰难，美国当前的目标是调整结构，让所有国家都参与和美国〝自由、公平、互惠〞的协议。
 <br/>
 <br/>
 4.2040年全球石油需求量增长15%
 <br/>
 <br/>
 石油输出国组织(OPEC)发表最新原油展望报告指出，全球石油需求从2017年到2040年将增加15%，每天需求量达到1.117亿桶，比去年预估增加60万桶，不过需求增长速度，预估将在2035年到2040年之间下降，每天需求可减少20万桶。
 <br/>
