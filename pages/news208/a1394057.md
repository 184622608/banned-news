### 10月3日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.国际货币基金组织表示，全球金融危机给世界经济留下持久的创伤，包括成长放缓、政府债务增加。全球超过6成经济体的产出仍低于危机前的水准。
 <br/>
 <br/>
 2.英国3月与欧盟达成初步承诺，脱欧后仍将分担欧盟负债，直到2020年底。但是在年金和财务担保费用增加带动下，欧盟负债攀升，英国的分手费恐怕高于预期的390亿英镑，大约506.6亿美元。
 <br/>
 <br/>
 3.自2015年以来，本田致力于自动驾驶汽车，但是进展缓慢。本田决定投资自驾车新创7.5亿美元，大股东是通用汽车。本田计划未来12年投资20亿美元，用于开发和制造基于Cruise软体的自动驾驶汽车。
 <br/>
 <br/>
 4.苹果公司执行长库克在接受采访时，对大众抱怨iPhone XS MAX的价格过高做出回应。库克表示iPhone XS MAX的硬件成本和线下用户体验是高价位的内部因素，同时美元汇价也推升了该款iPhone的价格。
 <br/>
 <br/>
 新唐人记者米西、正治综合报导
