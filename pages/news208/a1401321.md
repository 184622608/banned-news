### 德警方搜查德意志银行 涉洗钱3亿欧元
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】德国警方周四大举搜查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德意志银行.html" target="_blank">
  德意志银行
 </a>
 的法兰克福总部等六个办事处，怀疑这家德国最大的银行帮助客户
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗钱.html" target="_blank">
  洗钱
 </a>
 ，涉及金额超过3亿欧元。
 <br/>
 <br/>
 德国警方周四率领检察官、联邦和地方警察，以及税务调查人员等170多人，大举搜查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德意志银行.html" target="_blank">
  德意志银行
 </a>
 位于法兰克福的总部等六个办事处。
 <br/>
 <br/>
 当局怀疑德意志银行涉嫌帮助客户在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_避税天堂.html" target="_blank">
  避税天堂
 </a>
 设立离岸帐户从事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗钱.html" target="_blank">
  洗钱
 </a>
 ，涉及金额高达3亿1100万欧元。
 <br/>
 <br/>
 法兰克福检察官发言人尼森表示，仅在2016年，就有超过900名客户在英属维京群岛的德意志分行开户，转移了3亿1100万欧元。
 <br/>
 <br/>
 尼森说，这次调查重点是两名50岁和46岁的员工，以及其他潜在嫌疑人。
 <br/>
 <br/>
 德意志银行指出本案与2016年4月曝光的《巴拿马文件》（Panama Papers）有关，承诺全力配合调查。
 <br/>
 <br/>
 德意志银行全球公关主任埃根多夫：〝我们过去已向当局提供了有关《巴拿马文件》的所有信息。当然，现在会与德国法兰克福的检察官密切合作，我们也想尽快澄清事实。〞
 <br/>
 <br/>
 这个消息传出，德意志银行股价大幅下挫，一度下跌3.75％。
 <br/>
 <br/>
 德国联邦金融监管局今年9月下令德意志银行加强反洗钱措施，并指定一家会计公司进驻查帐。
 <br/>
 <br/>
 新唐人记者李梅、宇微综合报导
</div>
