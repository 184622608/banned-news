### 美国经济前景乐观 专家称股市会反弹
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】近日，美国股价崩跌引发关注。有分析认为，美联储过快升息可能是造成股票市场动荡的主因。专家对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 前景保持乐观。
 <br/>
 <br/>
 美股〝黑色星期三〞，一场崩跌，引发全球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融市场.html" target="_blank">
  金融市场
 </a>
 动荡，不过其实
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 ，目前欣欣向荣，9月失业率低到3.7%，2018年第2季，经济成长率，高达4.2%，通膨数据温和下，为何
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 如此反映？
 <br/>
 <br/>
 美国总统 川普：〝我们的利率上升速度，远远快过包括我自己在内的很多人所预期，我认为美国联准会失控了，我认为他们所做的是错的。〞
 <br/>
 <br/>
 川普槓上联准会，不是没原因。FED快速升息，引发通膨担忧，市场资金紧缩下，不只增加企业借贷成本，还降低股票市场吸引力。就怕吹熄，正要强力复苏的美国经济，成了猪队友。
 <br/>
 <br/>
 GUIDESTONE CAPITAL MANAGEMENT总裁 DAVID SPIKA：〝那就是市场不希望看到的情况，担忧联准会过于领先，提高利率速度太快，造成错误并导致市场调整。〞
 <br/>
 <br/>
 万宝投顾研究部执行长 王荣旭：〝第三季财报虽然不错，可是接下来第四季，甚至到明年上半年的一个经济状况，是不是会受到美国跟中国贸易之间的一个影响。〞
 <br/>
 <br/>
 美国减税利多效应逐步递减，尤其中国大陆经济不佳，影响美国的跨国企业获利。国际信评机构惠誉（Fitch），担忧美国企业可能暂缓资本支出计画，预估2018年美国企业资本支出成长约3%，2019年反转衰退0.8%。
 <br/>
 <br/>
 万宝投顾研究部执行长 王荣旭：〝所以我才认为就是说，这一次费城半导体跌的这么凶，而且领先跌，是不是他已经预先在景气的一个部分，开始有走下坡的一个情况，所以预先就已经开始在反映在股价上面。  〞
 <br/>
 <br/>
 分析师认为，美国科技股重跌，成了全球景气先行指标。]IMF《世界经济展望》报告，将明年全球经济成长率，从3.9%下修至3.7%，美国则预测由2.7％下修至2.5％。贸易制裁，当然美国也得付出不少成本。
 <br/>
 <br/>
 A&amp;G CAPITAL 首席投资专家 HILARY KRAMER：〝我想我们会看到市场在中期选举之前回归，绝对会的。但我们会看到更多的卖出，因为在企业方面还有个人投资者，都有令人难以置信的紧张情绪。〞
 <br/>
 <br/>
 不过外商瑞银(UBS)发布新闻稿，依旧看好美国多头市场并未终止，认为短期市场〝调节〞实属正常，断言这波〝牛市〞还要延续好一阵子。不论如何，市场波动，反映投资人进退维谷，后续端看美国是否祭出新一波振兴措施，就业数据、经济指标，都是重点。
 <br/>
 <br/>
 新唐人亚太电视 林嘉韦 沈唯同 台湾台北报导
</div>
