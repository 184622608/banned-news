 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制造业.html" target="_blank">

 </a>
 景气濒萎缩 官方PMI两年半新低
 <br/>
 <br/>
 中国官方今天公布，11月制造业PMI由前月50.2降至50，就落在代表景气荣枯的分界上，创2016年3月以来新低。
 <br/>
 <br/>
 3. 中国低价倾销 韩国掀倒闭潮
 <br/>
 <br/>
 韩国第3大生产企业〝现代重工绿色能源〞，不敌中国的低价太阳能面板，正式放弃中低价产线，有南韩业者透露，有多家太阳能业者因同样原因倒闭，目前的太阳能面板价格，比起2013年已腰斩。
 <br/>
 <br/>
 4. 台航空业攻电商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华航.html" target="_blank">
  华航
 </a>
 自创平台扩张客群
 <br/>
 <br/>
 中华航空正式进军电子商务市场，〝华航哩享购〞电商平台将在12月5日上线，用航空公司优势与各国供应商谈合作，12月促销活动还打出〝送机票〞；华航董事长何煖轩透露，目前华航已有免税品送货到府服务，是发展零售的优势，目标是藉电商平台，接触到更多新的消费客群。
