### 11月26日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.联合国国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_劳工组织.html" target="_blank">
  劳工组织
 </a>
 指出，全球工资增长速度出现放缓迹象，2017年只有1.8%，达到2009年以来的低点。实际工资从2008年金融危机中顺利恢复，但是开始出现衰退迹象，尤其在发达国家更加明显。
 <br/>
 <br/>
 2.英国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_渣打.html" target="_blank">
  渣打
 </a>
 集团正在考虑简化架构，将研究如何释放流动性，并减少不同法人实体的资金支出，藉此控制其全球庞大网络的成本。渣打预计公布一项战略评估，解决支出增加以及股价下跌近40%的担忧。
 <br/>
 <br/>
 3.英国部分资深内阁大臣和工党议员组建一个跨党派联盟，希望推动英国加入欧洲自由贸易联盟。因为梅伊的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧.html" target="_blank">
  脱欧
 </a>
 协议，在保守党内部就有强烈的反对阻力，部份大臣已经放弃协议过关的希望。
 <br/>
 <br/>
 4.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 Iphone用户指控AppStore对其中的软件抽取30%的佣金，导致软件开发商提高价格，最终将负担转嫁给消费者。但根据法律，应该只有软件开发商才有资格以垄断的罪名指控苹果。但最近的法院口头辩论暂时不利于苹果，法官更同情iPhone用户。
 <br/>
 <br/>
 新唐人记者米西、正治综合报导
