### 10月11日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.航空业近期受到飓风季、燃油价格飙高的影响。但是得力于航空旅行需求和票价提高，消弭了成本支出，达美航空第三季获利达到13.1亿美元，较去年同期增长13%，优于预期。
 <br/>
 <br/>
 2.调研机构IDC公布，全球第三季个人电脑出货量达到6千740万台，较去年同期下滑0.9%，但是优于市
 <br/>
 预期。美国市场因为企业笔记本电脑与电竞个人电脑的表现强劲，当季出货量来到1千720万台。
 <br/>
 <br/>
 3.由于经济成长趋弱，以及美国页岩油生产商产量的增加，石油输出国组织预计，明年全球对该组织的原油需求量，每天将减少90万桶，相当于利比亚今年的日均产量。不过，该组织及其盟国将继续确保市场供应充足。
 <br/>
 <br/>
 4.韩国统计厅发布，韩国9月的就业人口比去年同期增加4万5千人，达到2千705万5千人。失业率3.6%，其中15至29岁的年轻族群，失业率高达8.8%。9月失业人数为102万4千人，比去年同期增加了9万2千人。
 <br/>
 <br/>
 新唐人记者密西、阳阳综合报导
