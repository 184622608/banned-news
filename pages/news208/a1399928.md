### 移钱到美国 需遵守国税局法规
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】从国外带钱到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 、或在美国使用现金，要注意哪些事项？来看一下资深会计师的建议。
 <br/>
 <br/>
 很多人想把大陆资产分多次移转到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 ，想让来观光的朋友带现金进入美国。但会计师不建议把这样的钱存入银行。
 <br/>
 <br/>
 会计师 林清吉：〝家人送你的钱，你不能随便拿去银行存，你存现金的话可能会惹上《洗钱法》。〞
 <br/>
 <br/>
 中国大陆现在实施外汇管制，会计师也不建议透过香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇钱.html" target="_blank">
  汇钱
 </a>
 到美国。
 <br/>
 <br/>
 会计师 林清吉：〝很多人就从香港、利用香港的一些公司、个人把钱汇进来。这个我也不建议，为什么？因为一个公司怎么会把钱送给你呢？这个公司背后是谁。〞
 <br/>
 <br/>
 他表示，中国大陆和台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 现在从海外汇钱进入美国很困难。
 <br/>
 <br/>
 会计师 林清吉：〝台湾人带超过二百二十万（台币）以上，你可能要受制于赠与税，台湾是扣百分之二十。可是中国大陆人要送给你钱呢，目前中国大陆还没有赠与税，也没有遗产税，只是钱进到美国非常难。〞
 <br/>
 <br/>
 他说，在美国境内存钱需注意，现金超过一万美元可能会被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国税局.html" target="_blank">
  国税局
 </a>
 找上，严重者会吃官司。
 <br/>
 <br/>
 会计师 林清吉：〝很多人就为了避开这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_税法.html" target="_blank">
  税法
 </a>
 ，就一次存款七千、八千、六千，别忘了国税局讲一万美元不是一次，若分别加起来累计超过一万，他也会把你的资料报到美国国税局。一旦美国国税局收到后会问你：你为什么回避这一万元呢？就会认为你是故意的、计划性的回避这一万元的规则。〞
 <br/>
 <br/>
 同时他也提醒新移民：买珠宝、买车超过一万元也千万别用现金。
 <br/>
 <br/>
 会计师 林清吉：〝若你买车子超过一万元现金，对不起，现金超过一万，商家会把你的名字、社会安全号码跟地址报给国税局。这时候，你很快就会收到国税局来信，问你钱从哪里来，是未报的收入？这钱从哪里来，一定要有所解释。〞
 <br/>
 <br/>
 他建议，在美国消费最好用信用卡。
 <br/>
 <br/>
 新唐人记者杨阳洛杉矶采访报导
</div>
