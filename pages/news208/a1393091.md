### 阿根廷反紧缩大罢工 央行新行长闪辞
------------------------

 【新唐人2018年09月26日讯】近期，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货币贬值.html" target="_blank">

 </a>
 50%，处于新一场危机的边缘。而〝36小时全国大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罢工.html" target="_blank">
  罢工
 </a>
 〞和新上任不久的央行总裁闪辞，更使阿根廷经济雪上加霜。

 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 通膨高达40%，比索对美金的汇率已经重贬一半，导致国民生活成本不断攀升。
 <br/>
 <br/>
 保安公司雇员 Monica Haedo：我们是从糟糕走向更糟糕，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工资.html" target="_blank">
  工资
 </a>
 只够付账单和买食物。
 <br/>
 <br/>
 为了抗议通货膨胀飙升，以及总统马克里下令采取紧缩措施，工会于周一发起36小时全国大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罢工.html" target="_blank">
  罢工
 </a>
 。
 <br/>
 <br/>
 周二，阿根廷央行刚刚上任三个月的总裁卡布托突然辞职，令市场错愕。阿根廷比索当天早盘应声下跌6.7％。
 <br/>
 <br/>
 银行经理 Laura Ortigueira：我(对卡普托辞职)的反应是悲伤和被遗弃感。好像我们不知道方向在哪里。
 <br/>
 <br/>
 据媒体报导，卡布托主张在关键时刻动用央行储备缓解比索贬值速度，这与国际货币基金组织的意见相左。
 <br/>
 <br/>
 前经济政策官员桑勒里斯接替他的职位。
 <br/>
 <br/>
 周三，国际货币基金组织增加了对阿根廷的贷款，新协议贷款总额达570亿美元，以帮助阿根廷度过难关。
 <br/>
