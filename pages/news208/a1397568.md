### 10月31日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】下面请看今天的财经速瞄
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国劳工部.html" target="_blank">
  美国劳工部
 </a>
 周三发布数据，第三季度薪资增长3.1％，是十年来的最大增幅。总体薪酬成本上涨2.8％，创2008年以来最大涨幅，双双超出华尔街预期。当天也公布了10月美国私人部门ADP新增就业人数为22.7万人，创八个月新高，远超市场预期的18.7万人。
 <br/>
 <br/>
 随着国内外需求的减少，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造业.html" target="_blank">
  中国制造业
 </a>
 在10月份几乎没有增长，表明中国经济在美中贸易冲突的影响下进一步放缓。周三公布的官方采购经理人指数在10月跌至50.2，为2016年7月以来最低。而50是增长和收缩的分水岭。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 大臣拉布周三表示，他认为脱欧协议可以在11月21日之前达成。95%的条款都谈妥，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 也基本同意了英国提出的保障条款原则，爱尔兰与北爱的边境问题〝接近解决〞。消息一出，英镑短线上扬。
