 速瞄。

 经过周三的暴跌后，周四
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 大幅反弹。标普上涨近50点，涨幅1.86%，道指涨401点，涨幅1.63%。纳指涨超200点，上升2.95%。特斯拉大涨9.14%，奈飞涨3.66%。周四闭市后，亚马逊发布第三季度财报。每股盈利5.75美元，远超分析师预期的3.14美元。但是第三季度销售额566亿美元，低于预期的570亿美元。预计第四季度销售额665-725亿美元。
 <br/>
 <br/>
 美国医疗研究机构凯撒家庭基金周四（10月25日）发布报告，2018财年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白卡.html" target="_blank">
  白卡
 </a>
 （Medicaid，医疗补助）申请人数下降0.6%，这是2007年以来的首次下降，预计2019年申请人数会再创新低。报告指出：白卡申请放缓是因为经济强劲增长，低收入人群数量减少。

 <br/>
 2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆楼市.html" target="_blank">
  大陆楼市
 </a>
 的〝金九银十〞法则不再。多个省市的开发商降价求售，引来原有业主的不满，各地接连爆发业主抗议活动，有些楼盘的售楼处被打砸。从北京通州到合肥、长沙、杭州、天津、郑州等城市都陆续出现业主包围售楼处，要求退房的行动。不少房企面对业主诉求，最终都选择妥协。一些城市陆续出现土地拍卖大面积流拍现象。
