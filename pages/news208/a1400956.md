### G20川习谈判若挫 川普：或对苹果产品征10%关税
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】美国通用汽车，周一宣布裁减北美区员工人数，并减产销售低迷的车款。不过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 说，通用汽车应该推出一款卖得好的车，移回美国生产。而对于即将到来的G20川习会，川普表示，如果谈判没进展，已经准备好对所有中国商品加征
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，苹果手机和笔电也可能在其中。
 <br/>
 <br/>
 通用汽车宣布，为了因应房车销售疲软，将裁减北美地区近1.5万个工作、减少生产线，并淘汰部分销量不佳的车款。对此，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 26日受访时表示，通用汽车应该停止在中国制造汽车，改在美国生产。
 <br/>
 <br/>
 美国总统川普：〝美国为通用汽车做了很多工作。 你最好早点回流。 回到俄亥俄州，你最好早点回到那里。所以我们给予很多压力，我们有参议员，有很多人关切，给很多压力，通用他们说，雪佛兰Cruze车款销售不好，我说，那他们应该推一款卖得好的车子，然后回来美国生产。〞
 <br/>
 <br/>
 此外川普同一天接受华尔街日报采访时表示，美国〝极不可能〞推迟提高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，意味明年1月对2000亿美元中国商品的关税，将会提高到25%，还可能对苹果中国进口产品课征10%关税。
 <br/>
 <br/>
 彭博新闻记者 Joe Sobczyk:〝川普说可能对剩余的中国商品课税，这将会涵盖iPhone和苹果电脑，以及比目前这波关税，更广泛影响消费者的商品。川普认为10%的关税不会引起消费者太大反弹。〞
 <br/>
 <br/>
 消息一出，苹果股价盘后下跌2.11%至170.93美元。台股苹概三王大立光、台积电及鸿海早盘都出现跌幅。
