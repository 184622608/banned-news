### 11月8日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美联储在例会后指出，目前美国经济状况良好，就业增长、家庭支出强劲，整体经济活动步伐，只是企业投资的脚步已有所放缓，因此，美联储仍将维持升息的计划，继续逐步收紧资金。
 <br/>
 <br/>
 2.国际货币基金组织官员认为，意大利需要成长，但不是透过财政刺激能实现的，意大利没有这方面的财政空间，必须进行结构改革、劳动力市场改革、提高生产力的改革，如果采用扩张性的预算计画，经济反而可能会受害。
 <br/>
 <br/>
 3.中国民众受到不断高涨房价吸引，过去几年拼命囤房，据统计，目前中国全国城镇房地产存量大约有22%是空置房，超过5,000万套。专家警告，没有哪个国家有这么高的閒置率，如果出问题，房产抛售恐怕会像洪水一样滚滚而来。
 <br/>
 <br/>
 4.俄罗斯能源公司正在向西方石油买家施压，要求他们用欧元来购买石油，甚至在合同中添加相关惩罚条款，以避免收到美元。美国发动经济制裁，针对和俄罗斯吞并克里米亚有关的个人和公司，俄罗斯能源公司此举是为了降低新的制裁风险。
 <br/>
 <br/>
 新唐人记者保罗、正治综合报导
