### 最新报告：中国明年还有一个〝未爆弹〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】台湾官方的中央研究院最新公布2019年台湾经济情势总展望中，提及中国大陆目前出口动能扩张微弱，资金外流严重，再加上家庭、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_企业债务.html" target="_blank">
  企业债务
 </a>
 攀升，将是2019年一个〝未爆弹〞。
 <br/>
 <br/>
 台湾官方的中央研究院的这份2019年展望报告中预估，美国2019年上半年会来到景气的高点，下半年景气将受到减税效应消退，以及预算赤字压力变数所影响。
 <br/>
 <br/>
 由于中美贸易战持续升温，大陆经济也蒙上一层阴影。中共统计局11月30日公布，11月份官方制造业采购经理人指数（PMI）降至扩张与萎缩的分水岭50.0，是2016年7月以来新低。
 <br/>
 <br/>
 中央研究院经济所〝总体经济预测小组〞研究员周雨田说，明年1月1日起，中国出口到美国的2000亿美元商品仍将被课以10%关税，明显影响出口动能，出口优势也开始受到打击。大陆要靠出口顺差支撑经济成长动能回落。
 <br/>
 <br/>
 而人民币从年初至今跌幅加大，资金外流持续加剧，同时也影响中国外汇储备。从2014年最高点达到4万亿美元，目前停留在3万亿美元，将影响央行是否将出手干预人民币汇率未来走向。
 <br/>
 <br/>
 周雨田说还有比这更严重的，那就是中国最大的〝未爆弹〞是家庭和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_企业债务.html" target="_blank">
  企业债务
 </a>
 持续攀升。
 <br/>
 <br/>
 中国的债务问题从2017年以来一直是全球关注焦点。国际金融协会（Institute of International Finance）的报告也曾警告，中国债务总额与GDP之比已从危机前的171%飙升至2018年的299%。个人房屋贷款、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_消费贷款.html" target="_blank">
  消费贷款
 </a>
 和其他债务占比达到50%。
 <br/>
 <br/>
 根据国际清算银行指标，总债比超过100%是个〝危险〞指标，中国整体债务超过3倍，就算中国持续〝去槓杆〞效果仍不彰，这是一大不确定因素。
 <br/>
 <br/>
 德国保险公司安联（Allianz）的最新全球财富报告显示，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_居民槓杆.html" target="_blank">
  居民槓杆
 </a>
 率在2017年达到49.1%（40.6万亿/82.7万亿）的历史新高，比2008年的18%，激增173%。
 <br/>
 <br/>
 如果算上去年住房公积金贷款（4.5万亿）和P2P、现金贷等新型贷款，2017年中国居民槓杆率实际已突破54.5%。
 <br/>
 <br/>
 这反映了中国家庭整体债务增长迅速。美国居民槓杆率从20%升至50%用了40年，而中国不到10年。安联表示，没有其它国家的私人债务负担增长如此之快。
 <br/>
 <br/>
 飞速蹿升的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_家庭债务.html" target="_blank">
  家庭债务
 </a>
 正在侵蚀中国人的消费力，拖累经济增长。其中高房价是主要推手。房价飙涨导致家庭房贷激增，在家庭负债中占比越来越高。
 <br/>
 <br/>
 中共央行数据显示，去年中国人的住房贷款，在中国家庭债务中占比高达65%。同时，中国人的非房贷消费债务也在急速增长。
 <br/>
 <br/>
 《中国银行业发展报告2018》显示，2017年消费贷款（剔除个人房贷）余额为9.36万亿元，同比增长36.43%。
 <br/>
 <br/>
 这些数据反映出，中国人的债务仍在高速增长，而且住房依旧是中国家庭最沉重的负担。然而，筑起中国家庭重负的不仅仅是住房，还有教育和医疗。
 <br/>
 <br/>
 一份中国大学的研究报告警告，中国家庭部门的债务隐患，被中共当局和学界严重忽视，现已逼近美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融危机.html" target="_blank">
  金融危机
 </a>
 前的峰值水平，恐酿成重大潜在风险和严重失误。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：赵云）
