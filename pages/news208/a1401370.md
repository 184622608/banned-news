### 美国10月待售房屋销售 创四年新低
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 房屋市场持续传来降温讯号，10月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_待售房屋.html" target="_blank">
  待售房屋
 </a>
 销售指数比去年同期下降6.7%，创四年新低。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房地产.html" target="_blank">
  房地产
 </a>
 经纪人协会（NAR）今天公布的数据显示，10月份美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_待售房屋.html" target="_blank">
  待售房屋
 </a>
 销售的数据指数下跌。
 <br/>
 <br/>
 由于较高的抵押贷款利率影响了买家的购买能力，并且抑制了潜在买家的购屋热情，10月份签订的房屋买卖合同数目减少。今天的报告指出，10月美国待售
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房屋销售.html" target="_blank">
  房屋销售
 </a>
 指数比上个月下跌2.6%，至102.1
 <br/>
 <br/>
 这个基于房屋买卖合同签约数的指数比去年同期下降了6.7％。
 <br/>
 <br/>
 目前，住房市场低迷主因是抵押贷款利率上升造成的。在美联储决定缩减其资产负债规模并提高银行间贷款的短期利率后，导致房屋抵押贷款利率飙升。
 <br/>
 <br/>
 根据抵押贷款公司房地美（Freddie Mac）的数据，本周30年期抵押贷款的平均利率为4.81％，高于去年的3.90％。
 <br/>
 <br/>
 10月，地产销售签约数目下降主要体现在西部最为明显，降幅为8.9％，在中西部和南部地区也有所下降。然而，该指数在美国东北部地区反而增长了0.7％。
 <br/>
 <br/>
 待售房屋销售是房屋购买的一种衡量指标，最终完成销售大于是一两个月后，因此10月份的待售房屋指数显示，房屋销售数量可能会在12月份下降。〞
 <br/>
 <br/>
 新唐人记者Jeremy Sandberg纽约报导
