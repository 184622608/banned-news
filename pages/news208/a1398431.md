 <br/>
 <br/>
 中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 商会表示，习近平最新提出的开放市场承诺，只是不断重复先前没有履行的诺言，并没有提出充分的具体措施或者时间表，让欧洲企业对中共空谈式的承诺，已经没有兴趣。
 <br/>
 <br/>
 法国财政部表示，中共央行通过货币互换交易，提高其全球金融影响力，但由于缺乏透明度，可能导致各国潜在的经济问题被掩盖。中共当局经常和缺乏外汇来源的国家交换货币，但交换条件不够透明。
 <br/>
 <br/>
 日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丰田汽车.html" target="_blank">
  丰田汽车
 </a>
 公布上季业绩，销售额创历史新高，净利润率达到8.5％，超过其他汽车公司。虽然全球原料价格上涨，推动汽车成本上扬，但由于扩大亚洲和欧洲的销售，加上降低成本方案奏效，使获利优于预期。
 <br/>
 <br/>
 欧盟执委会经济事务执委表示，如果意大利和欧盟在预算问题上不能达成协议，执委会可能采取制裁手段。意大利之前提交欧盟的预算计划中，债务比例过高，违反欧盟财政规定，被欧盟驳回。目前，意大利公共债务约为GDP的130%。
 <br/>
 <br/>
 新唐人记者保罗、宇微综合报导
