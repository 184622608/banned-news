### 普悠玛设计有疏失 日制造商股价狂跌17%
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】继续来关心，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻车.html" target="_blank">
  翻车
 </a>
 事故，昨天晚上，日本媒体报导，负责制造普悠玛列车的公司，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本车辆.html" target="_blank">
  日本车辆
 </a>
 坦承，在列车设计上有疏失。而且卖给台湾的所有普悠玛，19辆都有相同问题。消息一出，今天早上日本车辆股价大跌17%，打入跌停。
 <br/>
 <br/>
 TBS电视台：〝有关造成18人死亡的台湾列车出轨事故，列车制造商，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本车辆.html" target="_blank">
  日本车辆
 </a>
 公司表示，在列车的安全装置设计上，有所疏失。〞
 <br/>
 <br/>
 日本媒体报导，制造
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车的日本车辆公司，坦承在车辆ATP设计上有疏失。造成关闭ATP后无法自动回传讯息给行控中心，而且卖给台湾的19组普悠玛列车都有相同问题。
 <br/>
 <br/>
 当时设计图是由负责设计的人绘制，再由其他人员进行检查，但却漏看了这个部分，而由于列车无法在日本进行测试，所以没有发现这项疏失。
 <br/>
 <br/>
 昨晚消息传出后，今天日本车辆公司早盘股价崩跌17%到跌停，创下2011年3月来最大跌幅。
 <br/>
 <br/>
 日本车辆社长 五十岚一弘 (2018.10.26) ：〝对于罹难者，我深挚的哀悼，也向在事故中受伤的民众慰问。〞
 <br/>
 <br/>
 日本车辆社长上月底决算报告会上，向普悠玛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻车.html" target="_blank">
  翻车
 </a>
 意外中的罹难者表达沉痛哀悼之意。表示愿意全力协助当局调查事故原因。
 <br/>
 <br/>
 立委黄国昌昨晚在脸书，贴出日本车辆制造公司致台铁的信函。指出交车时，ATP远端监控功能配线是处于未连接状态，可能是设计相关人员确认不足所致，但对ATP本身功能没有影响。至于配线是否正确连接，是由台铁实施。
 <br/>
 <br/>
 到底哪个环节出错，恐怕还得等详细调查结果出炉。
 <br/>
 <br/>
 (新唐人亚太电视 综合报导)
