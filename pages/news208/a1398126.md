### 11月4日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】下面请看一组环球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财经简讯.html" target="_blank">
  财经简讯
 </a>
 。
 <br/>
 <br/>
 根据统计，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_智能手表.html" target="_blank">
  智能手表
 </a>
 上一季出货量突破1,000万支大关，同比大幅增长67%，销售热度丝毫没有任何放缓的迹象，其中苹果(Apple) Apple Watch卖出约360万支，增长25%，排名第一，市占率达45%，其次是Fitbit，销售150万支，三星(Samsung) Galaxy Watch出货约110万支。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴菲特.html" target="_blank">
  巴菲特
 </a>
 (Warren Buffett)的波克夏公司(Berkshire Hathaway)上季营业利润翻倍，达68.8亿美元，高于预期，但由于目前企业价格，已达历史最高点，难觅合适投资标的，因此，将手中过多现金买回自家股票，上季执行9.28亿美元，占现金比率不到的1%。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴.html" target="_blank">
  阿里巴巴
 </a>
 公布上季财报，营收851.5亿人民币，同比增长54.5%，不如市场预期，净利200.3亿人民币，增长13%，每股盈余6.78元，公司表示，受到宏观经济影响，下修全年营收展望4%~6%，再度显示中国经济动能放缓的现实。
 <br/>
 <br/>
 中共在南亚的投资超过了1,000亿美元，想利用这些资金在海外获得政治影响力，巴基斯坦和孟加拉是最大的两个资金接收国，引发地缘政治紧张，对此印度政府已经提高警觉，将采取因应措施。
 <br/>
 <br/>
 新唐人记者密西综合报导
