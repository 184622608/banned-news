### 10月25日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美联储褐皮书指出，大部分地区工资与物价以温和到略高于温和的速度上升，全国各地都有劳工市场吃紧现象，虽然原料与运输成本上升，贸易环境具有不确定性，且欠缺合格工人，但制造业生产量仍温和增长，且某些地区零售商能够调高销售价格。
 <br/>
 <br/>
 2.微软公布上季财报，营收290.8亿美元，净利88.2亿美元，同比大幅增长34%，每股获利1.14美元，优于市场预期，因有越来越多的企业，使用云计算服务和Office 365软件，驱动云业务营收同比大增76%。
 <br/>
 <br/>
 3.法国收到欧盟来函，对法国提交的下一年度预算报告提出警告，提醒法国该预算报告和先前达成的协议不符。法国预算报告中2019年结构性债务将下降0.3%，之前协议中的减债数字是GDP的0.6%。
 <br/>
 <br/>
 4.苹果公司执行长库克出席在布鲁塞尔举办的国际资料保障及隐私专员国际会议，再次重申苹果公司会支持各地的隐私保护法律。库克还表示，有些企业将用户资料当作盈利的来源，在收集用户资料上越来越极端。
 <br/>
 <br/>
 新唐人记者保罗、正治综合报导
