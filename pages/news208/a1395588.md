### 10月15日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美国商务部公布，8月的企业销售额为1.46万亿美元，创下自1992年统计以来最高，年增率下降到7.8%，连续3个月走低。零售商销售额年增6.1%，成为4千468亿美元，也创下自1992年统计以来最高点。
 <br/>
 <br/>
 2.欧洲央行表示，央行按计划在12月份结束净资产购买。可能在明年中开始讨论利率问题，必要时可能在明年秋季做出决定。贸易战对欧元构成最大风险，英国脱欧对货币政策的影响将低于贸易战。
 <br/>
 <br/>
 3.日本已两次延缓将消费税增加到10%，首相安倍晋三表示，希望能在明年10月1号如期实施。增税所得中的半数，将回馈给日本民众，明年10月1号开始，幼儿教育都将完全免费。
 <br/>
 <br/>
 新唐人记者杰森综合报导
