### 尘埃落定 美墨加正式签署新贸易协议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】周五，在结束了与阿根廷总统马克里的会晤后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国总统.html" target="_blank">
  美国总统
 </a>
 川普和加拿大总理特鲁多，墨西哥总统亚尼托，共同举行了《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨加.html" target="_blank">
  美墨加
 </a>
 贸易协定》的签字仪式。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨加.html" target="_blank">
  美墨加
 </a>
 三方，经过一年半的激烈讨论，9月30日就数万亿美元的双边贸易达成协议。川普总统表示，新协议将永久改变贸易格局。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国总统.html" target="_blank">
  美国总统
 </a>
 川普：〝这是一份将永久改变贸易格局的现代协议。〞
 <br/>
 <br/>
 川普表示，新协议将首先支持劳工阶层，支持高薪水的制造业就业，促进美国包括农业、制造业和服务业在内的出口机会。
 <br/>
 <br/>
 美国总统川普：〝《美墨加贸易协定》（USMCA）是历史上最大、最重要、现代而平衡的贸易协议。〞
 <br/>
 <br/>
 经过谈判中的互不相让，加拿大总理特鲁多仍不松口，坚持把新协议称为〝新的北美自由贸易协议〞。
 <br/>
 <br/>
 加拿大总理特鲁多：〝这份新的北美自由贸易协定，将维持加拿大整体经济的稳定性。〞
 <br/>
 <br/>
 虽然三方正式签署《美墨加贸易协议》，但仍需各自国家的立法机构批准，才能正式生效，并代替过去的《北美自由贸易协议》。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
</div>
