### 美中贸易战暂不升级 全球股市回升
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 之后，美中双方同意推迟美国对中方提高关税90天，消息公布后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球股市.html" target="_blank">
  全球股市
 </a>
 飙升。
 <br/>
 <br/>
 新唐人记者Jeremy Sandberg：〝在美中同意把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 暂停90天后，周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市.html" target="_blank">
  美国股市
 </a>
 大幅上涨，道琼斯指数早盘一度上涨超过300点。〞
 <br/>
 <br/>
 周一，在20国集团峰会之后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球股市.html" target="_blank">
  全球股市
 </a>
 上涨近1％。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 价格飙升。
 <br/>
 <br/>
 科技公司的股票全线上涨，带动市场走高。芯片制造商AMD上涨9.3％。随着石油价格上涨超过5％，能源类股也上涨。黄金触及一个月以来的高水平，工业金属和其他商品包括棉花等也上涨。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 达成一项协议，美国在未来90天内推迟增加关税，以便解决美国提出的问题，包括知识产权保护，非关税壁垒和网络盗窃。作为回报，中国同意大量购买美国商品，包括农业，能源，工业品和其他产品，以减少美国对华贸易逆差。
 <br/>
 <br/>
 虽然暂时休战有可能在今年年底前稳定市场，但两国仍需要制定出持久有效的贸易协议。
 <br/>
 <br/>
 Jeremy Sandberg：〝白宫表示，如果在90天内没有达成协议，那么价值2000亿美元的中国商品的关税将从目前的10％被提升至25％。〞
 <br/>
 <br/>
 新唐人记者Jeremy Sandberg纽约报导
</div>
