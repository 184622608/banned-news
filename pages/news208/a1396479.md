### 10月22日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美国总统川普的首席经济顾问库德洛（Larry Kudlow）指责中共政府在解决贸易问题方面毫无作为，美方列出了一份详细的要求清单，其中涉及到技术等许多问题，但是五六个月过去了，基本上没有任何变化，中共根本就不予回应。
 <br/>
 <br/>
 2.根据调查，脸书(Facebook)旗下图片社交平台Instagram超越Snapchat，为美国青少年最常用的社交平台，有85%受访青少年每个月至少使用Instagram一次，比Snapchat的84% 高，另外，Instagram对行销人员的吸引力也在增长，因为多数青少年喜欢品牌在Instagram渠道上做联系。
 <br/>
 <br/>
 3.专家分析消费者在iPhone新机发布前后，利用Google搜索趋势的变化，结果显示，消费者对iPhone新机发表的关注度逐年降低，其中最低的就是今年的iPhone XR，专家认为，iPhone创新力度不够、市场饱和是兴趣减弱的主因。
 <br/>
 <br/>
 4.由于欧盟实施新机动车排放认证系统，德国汽车制造商全力争取认证，经销商赶在新规生效前打折清库存，使得汽车制造减少，经济陷入挣扎，不过，德国央行认为，其他工业部门相对稳健，未完成订单维持高位，预期第四季经济将出现反弹。
 <br/>
 <br/>
 新唐人记者杰森综合报导
