### 忧贸易前景 全球股市跌 白宫称勿信谣言
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】因为悼念前美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 ，周三
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市.html" target="_blank">
  美国股市
 </a>
 闭市，但前一天美国股市下跌超过3%，引发亚洲和欧洲的连锁反应，周三
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球股市.html" target="_blank">
  全球股市
 </a>
 下跌。
 <br/>
 <br/>
 今天
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球股市.html" target="_blank">
  全球股市
 </a>
 跌至一周低点，因长期
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国债.html" target="_blank">
  美国国债
 </a>
 收益率下跌和对中美贸易前景的不确定性增加了投资人对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 下行的担忧。
 <br/>
 <br/>
 由于中美对川习会达成的协议内容表述不一致，让市场产生困惑。外加周二美股大跌，和美国长期债券收益率下降，引发当天日本以外的亚洲股市下跌1.5％，泛欧600指数下跌1.16％。
 <br/>
 <br/>
 香港恒生、中国沪指、还有日经都有下跌。
 <br/>
 <br/>
 香港恒生指数下跌1.6％，中国沪指下跌0.6％。东京的日经225指数下跌了0.5％。&gt;&gt;
 <br/>
 <br/>
 而欧洲的伦敦富时和德国DAX等指数也纷纷出现跌幅。
 <br/>
 <br/>
 在欧洲，伦敦富时100指数下跌1.4％，德国DAX指数下跌0.8％。法国CAC 40指数下跌0.9。&gt;&gt;
 <br/>
 <br/>
 目前美国国债收益率曲线趋平，长期国债收益率低于短期债券收益率的现象，让投资人恐慌。过去50年，每当这种现象发生，随之而来就是美国经济衰退。
 <br/>
 <br/>
 但川普总统顾问纳瓦罗（Peter Navarro）表示，市场对某些谣言的反应是愚蠢的，呼吁投资人耐心等待未来美中贸易谈判的成果。
 <br/>
 <br/>
 此外，英镑摆脱17个月的低点，上涨0.3%，投资人乐观期待英国能最终留在欧盟。
 <br/>
 <br/>
 新唐人记者Jeremy Sandberg纽约报导
