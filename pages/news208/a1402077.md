### 12月5日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1. 美国与欧洲的650亿美元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易逆差.html" target="_blank">
  贸易逆差
 </a>
 中，约有300亿来自与德国的汽车和零组件贸易。美国商务部长罗斯(Wilbur Ross)建议，德国汽车可以在美国增加产量以减少逆差。大众汽车CEO随后表示，考虑和福特建立联盟，使用福特的工厂来生产大众汽车。
 <br/>
 <br/>
 2. 澳洲统计局公布，第三季度GDP环比增长0.3%，同比增长2.8%。增长急速下滑，远低于市场预期，创下两年以来最疲弱增幅。主要是商业建筑下滑，家庭支出的增长减缓，因此市场预期
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲央行.html" target="_blank">
  澳洲央行
 </a>
 明年升息机会不大。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通讯芯片.html" target="_blank">
  通讯芯片
 </a>
 大厂高通(Qualcomm)推出新一代智能手机处理器〝骁龙855〞(Snapdragon 855)。明年美国上市的5G智能手机，将会采用这款芯片，由于手机市场面临饱和，许多电信运营商和手机厂商视5G通讯技术为转机，期待能为市场注入一股新的动力。
 <br/>
 <br/>
 4.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国电信集团.html" target="_blank">
  英国电信集团
 </a>
 将在两年内移除公司4G网络中的来自华为的设备，而未来5G网络的设备供应商投标名单中，也排除华为。网络中的次要设备，例如天线，将部分使用华为产品。英国电信此举是为了使其通讯业务符合内部政策。
 <br/>
 <br/>
 新唐人记者保罗、明玉综合报导
