### 投资韩国—忠清北道经济自由区
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_忠清北道.html" target="_blank">
  忠清北道
 </a>
 位于韩国的正中央，这里的法规有别于其他城市。近来这里受到外国投资者的关注。请看韩国记者发来的报导。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_忠清北道.html" target="_blank">
  忠清北道
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济自由区.html" target="_blank">
  经济自由区
 </a>
 近期对我开发，并成为一个深受外国投资者关注的优秀事业开创区。 11月8日，尽管天气不佳，但80多位外国投资者为了解忠清北道经济自由区而来到韩国。
 <br/>
 <br/>
 在韩国除这里之外，还有6个经济自由区，而忠清北道位于韩国地理上的中心位置。而且还有地方机场、高速列车及七条国家高速公路贯穿，所以在物流和流通方面具有很强的竞争力。
 <br/>
 <br/>
 自由区方面已与新加坡投资企业PRESTIGEBIO制药公司结成战略合作关系，并于今年9月已正式进入忠清北道经济自由区 。
 <br/>
 <br/>
 PRESTIGEBIO制药公司代表金镇佑：〝现在是正式的开始阶段，所以就会发现很多事是通过民间和政府共同协商来进行的，这是很好的。〞
 <br/>
 <br/>
 投资者们在走访了已投资企业、公共机关以及研究所之后，显示出对这一体系的极大期待。
 <br/>
 <br/>
 加拿大多伦多上市企业 TSX venture  Chris Chung：〝韩国研发是很先进的，比美国和加拿大还要先进，潜力也非常大。〞
 <br/>
 <br/>
 韩国保健、医疗及行政等共6个大型国家机构已进入忠清北道经济自由区 ，这里有良好的工业基础设施，正在成为外国投资者们关注的焦点。
 <br/>
 <br/>
 新唐人记者赵润德韩国忠清北道报导
