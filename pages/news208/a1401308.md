### 11月28日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.爱利信预估，到2024年5G用户将增加50%，达到15亿户，覆盖全球人口40%以上。其中，北美和东北亚将领先采用5G。美国预计今年或明年开始提供5G服务，全球主要网络部署预计2020年开始。
 <br/>
 <br/>
 2.英国官方分析，一旦议会拒绝脱欧协议，如果是无序退出并且来自欧盟的劳动力供应枯竭，英国国内生产毛额将在15年内最多减少10.7%。根据这项研究，无论选择任何一种退出模式，英国都会变穷。
 <br/>
 <br/>
 3.苹果一名高管透露，iPhone XR从开始销售起，就一直是每日销售量最高的苹果手机。和另外两款价格更高的新iPhone比，iPhone XR已经拥有了多项重要功能，例如脸部识别等，对大多数顾客来说，其性价比更高。
 <br/>
 <br/>
 4.作为大众汽车电动车方面大规模投资计划的一部分，大众将在美国新建一家电动车工厂，生产的电动车将在美国销售。大众美国业务负责人表示，现在正在选择建造新工厂的地点。大众预计在2020年会推出一款电动车，售价在3-4万美元。
 <br/>
 <br/>
 <br/>
 <br/>
 新唐人记者米西、正治综合报导
