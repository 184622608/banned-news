### 11月29日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 <br/>
 1.路透社通过对房地产市场分析师的调查结果，预测明年美国房价涨势将随整体经济动能减弱而明显放缓。美国过去5年房价涨幅是通胀和资薪上涨速度的两倍以上，但在未来，利率上升，税改和房价预期放缓都会打压房地产需求。
 <br/>
 <br/>
 2.日本10月的失业率，由9月的2.3%扬升到2.4%，表现不如预期，但是依然接近1993年以来的最低水准。日本劳动力市场持续升温，薪资正缓慢增长，更高的薪资将有助于日本央行实现2%的通胀率。
 <br/>
 <br/>
 3.AT&amp;T将在明年推出三种影音串流服务，其中包括了华纳兄弟，特纳和HBO的电影和电视剧。AT&amp;T公司目前的影音串流服务客户数量正在减少，所以公司希望创立新的串流服务计划来留住客户，并和Netflix形成有效竞争局面。
 <br/>
 <br/>
 4.日本电信业巨头软银公司即将在日本上市，到时将发售价值210亿美元的股票，每股指导股价将为1500日元，这是日本史上最大的一次公司股票上市。软银提供的高派息率预计将吸引不少散户投资，经纪商甚至用电视广告来推广软银上市。
 <br/>
 <br/>
 新唐人记者米西、正治综合报导
