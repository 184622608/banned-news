### 11月14日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】下面是今天的财经速瞄
 <br/>
 <br/>
 周三，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在推特转发的一篇报导显示：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国企业.html" target="_blank">
  美国企业
 </a>
 雇员需求来到45年高点。全国独立企业联合会表示，过去43年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小企业.html" target="_blank">
  小企业
 </a>
 的乐观情绪只有3次达到这么高，并且过去一年来一直保持这样的高水平。
 <br/>
 <br/>
 周三官方数据显示，中国10月份汽车、食品等其他消费品支出增速创五个月最低水平，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房地产.html" target="_blank">
  房地产
 </a>
 销售增速创半年来以来最低水平。显示中国的消费和房地产市场都表现疲弱。经济学家称，中国国内需求正在走软，中国经济的未来面临困难。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 公司周二宣布，将投资50亿美元，在纽约市和北弗吉尼亚建立新总部，将分别拥有超过2.5万名员工。亚马逊还会在纳什维尔创建一个新运营中心，聘请逾5,000名员工。
