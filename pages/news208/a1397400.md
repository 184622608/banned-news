### 10月30日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】下面是今天的财经速瞄。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国央行.html" target="_blank">
  中国央行
 </a>
 周二将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 中间价下调197点，设在6.9574，是10年来最低水平。在岸和离岸人民币兑美元双双跌破6.97关口；最低6.9717，创逾10年来最低。人民币最近的下跌再次引发了有关进一步贬值是否会促使资本外流的猜测，资本外流会反过来加剧人民币跌势。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 公司上月短暂成为第二家市值达到万亿美元的美国企业。但自从该公司上周发布财报以来，股价一路走跌，短短两个交易日内便已蒸发将近1,200亿。周二，亚马逊股价再度下跌8.46美元。
 <br/>
 <br/>
 克雷塞财富顾问创始合伙人阿布林（Jack Ablin）表示，历史显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 通常会出现上涨。道富环球投资管理公司首席投资策略师阿荣尼（Michael Arone）说，到明年下半年，随着经济放缓、货币政策收紧、盈利增长减缓，两党可能开始推进另一个财政政策方案，并持续到2020大选年。这将帮助支持股市。
