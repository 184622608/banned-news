### 法国玛丽王后珠宝200年后面市 将拍卖
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】欧洲王室珠宝将于11月14日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瑞士日内瓦.html" target="_blank">
  瑞士日内瓦
 </a>
 亮相。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏富比.html" target="_blank">
  苏富比
 </a>
 将拍卖的100多件单品中，最大亮点是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 路易十六的妻子安托内特佩戴过的珠宝。
 <br/>
 <br/>
 1791年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 大革命期间，法国王后玛丽．安托内特将珠宝偷偷运到比利时，再转运到奥地利交给侄子：奥地利皇帝。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_玛丽王后.html" target="_blank">
  玛丽王后
 </a>
 后来和她的丈夫，国王路易十六一起被革命者杀害。波旁家族从此在法国退位。
 <br/>
 <br/>
 第二年，她的长女被释放后，前往奥地利取回了珠宝，之后这些珠宝就一直在家族中流传。
 <br/>
 <br/>
 240年后的今天。外人终于可以一睹玛丽王后珠宝的真容了。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏富比.html" target="_blank">
  苏富比
 </a>
 拍卖行珠宝高级总监 柯瑞尔：〝这些珠宝是法国历史的缩影。〞
 <br/>
 <br/>
 这批珠宝中估价最高的是名为〝玛丽王后的珍珠〞 的水滴形挂坠，上面是钻石点缀的蝴蝶结，下面是长度将近2.6厘米的天然珍珠。估价200万美元。
 <br/>
 <br/>
 苏富比拍卖行珠宝专家拉普林：〝从大小和质量来说都是非常珍贵的珠宝。加上背后的历史，来自世界上最著名的王后。〞
 <br/>
 <br/>
 有一个戒指带有一缕玛丽王后的头发，估价8千到1万美元。
 <br/>
 <br/>
 苏富比拍卖行称这批珠宝是〝有史以来最重要的皇家珠宝收藏之一〞。在拍卖之前会先在纽约展出。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
