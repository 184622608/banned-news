### 北京明年日子难熬  GDP增长或迎29年最低
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 影响，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 持续放缓。专家分析认为，北京明年的日子将更艰难，中国经济的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_增长率.html" target="_blank">
  增长率
 </a>
 2019年可能会迎来29年来新低。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 持续升温，依国际货币基金组织的分析，如果美中贸易紧张局势持续下去，在最糟糕的情况下，未来两年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_增长率.html" target="_blank">
  增长率
 </a>
 将下降1.6个百分点。
 <br/>
 <br/>
 中共国家统计局发表的数据显示，第三季度国内生产总值（GDP）增长率为6.5%，创下了2008年全球金融危机以来，中国经济最慢的季度增长率。
 <br/>
 <br/>
 根据投资银行（IB）分析专家，最近发布的经济展望结果，市场尤其关心中共当局对贸易战带来的不确定性、放缓的中国经济的应对。
 <br/>
 <br/>
 瑞士联合银行集团（UBS）称，中美贸易战日益升温，中共2019年的GDP增长率可能下降到5.5%。高盛的分析专家则指出，不仅是贸易战，房地产市场负债等国内因素，也导致中共2019年的经济增长率可能停留在6.2%。
 <br/>
 <br/>
 UBS中国经济研究负责人王涛说，2019年中国要迎接的主要阻碍是关税上调与贸易战相关的不确定性。
 <br/>
 <br/>
 持续了半年的中美贸易战火持续升高，中共的经济持续下滑，美国经济呈现强劲态势。目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府已对2,500亿中国商品课征10%到25%的惩罚性关税。
 <br/>
 <br/>
 美国副总统彭斯17日在APEC峰会上表示，中共设置的贸易壁垒史无前例，包括关税、盗用知识产权、强逼技术转让等，美国已对2500亿美元的中国产品征收关税，征税规模还可能将扩大一倍。
 <br/>
 <br/>
 10月份IMF及世界银行在印度尼西亚巴厘岛召开的年度大会上，IMF亚洲及太平洋部主任李昌镛（Changyong Rhee）表示，中美贸易战最具杀伤力的冲击，是对投资者信心的打击，这将造成金融市场陷入困境，而且可能会持续一段时间。
 <br/>
 <br/>
 〝这是我们认为逆势可能会持续更长时间的原因之一〞，李昌镛说，〝我不知道结局会是什么……我认为我们现在所得到的经验是，全球金融市场与经济的融合程度，没有人可以避免这场冲击。〞
 <br/>
 <br/>
 受贸易紧张局势拖累，投资人担心全球经济可能陷入困境，导致全球股灾，其中中国股市受创最为严重。中共央行为了救经济，今年多次下调金融机构存款准备金。
 <br/>
 <br/>
 李昌镛表示，由于中共采取刺激经济活动的措施，IMF对今年中国经济增长率的预测值，仍维持在之前预测的6.6%，对明年的预测值仅减少0.2个百分点到6.2%。
 <br/>
 <br/>
 而摩根大通首席中国经济学家朱海滨，10月份在香港举办的一次会议上表示，该行的底线是美国对所有中国进口产品征收关税，这意味着这将是〝2019年真正痛苦的经济事件〞。
 <br/>
 <br/>
 朱海滨说，随着转变为全面贸易战，影响将更大。他补充说，贸易战可能削减中共的经济增长1个百分点。
 <br/>
 <br/>
 1990年中国的GDP增长率为3%，之后一路飙升，到了2015年，GDP增长率为6.9%，时隔25年下降到7%以下。2016年的GDP增长率下降到6.7%，2017年GDP增长率反升为6.9%。
 <br/>
 <br/>
 从中共统计局公布的数字来看，2018年第三季度的经济增长已经呈现加速下滑的趋势。第一季度增长率是6.8%，第二季度是6.7%，第三季度GDP增长率为6.5%，创下9年半最低。
 <br/>
 <br/>
 有分析人士认为，即将举行的G20峰会期间，以中美两国首脑会晤为契机，中美可能达成协议。但大多观点认为中美在G20峰会期间达成协议的可能性不大。
 <br/>
 <br/>
 原因包括刚结束不久的APEC峰会上，美国副总统彭斯（Mike Pence）与习近平激烈争吵，以及中共提供142项最新清单美方仍不满等等。
 <br/>
 <br/>
 川普总统曾在美国的一次集会上，指责中共过去每年都从美国拿走5000亿美元，不仅偷走了美国的财富，还偷走了美国作为一个国家的尊严。这简直太疯狂了，是时候叫停这个了。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿）
