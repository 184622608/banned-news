 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国劳工部.html" target="_blank">

 公布最新

 </a>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍芯片.html" target="_blank">
  间谍芯片
 </a>
 的报导引发相关科技公司股价大跌。其中硅谷服务器公司Super Micro Computer周四股价一度狂泻58%，当天以下跌41%收盘，周五，继续下跌近3%。联想和中兴暴跌引领港股电子板块下跌，周五香港恒生指数继续下跌，本周累计下跌超过1215点，跌幅4.38%，创近8个月最大单周跌幅。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丰田汽车.html" target="_blank">
  丰田汽车
 </a>
 公司周五表示，将在全球召回240万辆汽车，以修复混合动力系统中一个可能造成汽车动​​力丧失的缺陷。此次召回主要影响2008年10月至2014年11月生产的部分普锐斯(Prius)系列汽车。
 <br/>
 <br/>
 最新数据显示，美国8月贸易逆差扩大32亿美元，增长了6.4％，来到532亿美元。主因是8月份美国大豆出口下跌10亿美元。今年到目前为止，美国对中国的贸易赤字为2611亿美元，其中386亿美元来自8月份。
