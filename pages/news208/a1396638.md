### 10月23日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.意大利政府曾经向欧盟承诺，明年预算赤字目标为国内GDP的0.8%，但是上个月底，却将赤字目标上调到2.4%。因此，欧盟决定拒绝意大利提交的预算案，同时给意大利三个星期的时间，修改并重新提交计划。
 <br/>
 <br/>
 2.快餐连锁店麦当劳公布上季财报，营收53.7亿美元，同比减少6.6%，获利16.4亿美元，同比减少12.8%，每股获利2.1美元。另外，全球同店销售年增4.2%，美国分店成长2.4%，连续13季出现增长，优于预期。
 <br/>
 <br/>
 3.市场消息人士透露，苹果公司计划2019年上半年，在美国推出影音订阅服务，并将向全球推广。其中，苹果自己的原创影音内容将向拥有苹果硬件设备的用户免费开发，第三方提供的内容，用户需要收费订阅。
 <br/>
 <br/>
 4.德国公司Volocopter表示，获得新加坡政府的支持，明年下半年将在新加坡试飞无人驾驶空中出租车。这种出租车靠电力驱动，可载两名乘客，目前最大飞行距离为30公里，希望能缓解城市交通拥挤问题。
 <br/>
 <br/>
 新唐人记者保罗、明玉综合报导
