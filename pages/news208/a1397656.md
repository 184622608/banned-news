### 10月31日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.万事达最新一季的利润为19亿美元，年增率33%，每股纯益1.82美元，高于预期。不过分析师认为，万事达依靠回馈金助攻的营收成长不是长远之计。目前全球的Mastercard及Maestro品牌信用卡量达到25亿张。
 <br/>
 <br/>
 2.美国能源资讯署的月报显示，美国8月原油日产量上升41万6千桶，成达到纪录高点1千134万6千桶。上升主因是德州和北达科他州的产量增加。
 <br/>
 <br/>
 3.消息人士透露，在英国脱欧后，英国金融服务企业只要英国金融监管与欧洲大体保持一致，欧盟就将允许英国公司获得欧洲市场准入。如果没有经过独立仲裁并给予30天以上的通知期，双方都不得单方面拒绝市场准入。
 <br/>
 <br/>
 4.据消息人士透露，大众汽车和福特汽车正在商讨共同开发自动驾驶汽车和电动车。两家公司都面临欧洲严格的车辆排放法案，需要相应的电动车方案。而在自驾车上进行合作，更是不少汽车公司减少研发成本和时间的措施。
 <br/>
 <br/>
 新唐人记者米西、正治综合报导
