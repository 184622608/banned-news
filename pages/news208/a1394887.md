### 【禁闻】贸易战升级 美拟列中国为汇率操纵国
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】美国媒体10月9号报导，美国下周可能正式宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率.html" target="_blank">
  汇率
 </a>
 操纵国。专家认为，这意味着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 升级为金融战。
 <br/>
 <br/>
 10月10号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 对美元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率.html" target="_blank">
  汇率
 </a>
 突破了6.92，创下了21个月的新低，再度向〝破7〞迈进。
 <br/>
 <br/>
 随着5月份美国开始加征
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 产品关税，人民币兑美元汇率不断贬值，一路从6.33贬到目前的6.92，在过去6个中，人民币兑美元贬值9%，成为亚洲表现最差的货币之一。
 <br/>
 <br/>
 据美国〝彭博社〞报导，美国财政部一位官员8号透露，川普政府正〝密切关注〞人民币汇率，非常〝非常担心〞中共故意让人民币贬值，抵销美国的关税措施。财政部正考虑在下周公布的一份报告中，正式将中国列为〝汇率操纵国〞。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教 授谢田：〝这个很有可能，因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货币操纵.html" target="_blank">
  货币操纵
 </a>
 本身是中共攫取贸易优势的一个办法，它一面限制美国的进口，一面给出口商进行补贴，一面操纵鼓励出口，这是中共过去2、30年一直的国策，也是中共控制中国经济、掠夺中国人民的一个手段。〞
 <br/>
 <br/>
 大陆金融分析师任中道透露，中共能够操控汇率在于中共想法设法攫取和控制所有的外汇。
 <br/>
 <br/>
 大陆金融分析师任中道：〝打个比方来说，它派一个央企到海外来发行债券，那这个债券获得的外汇，美元、英镑、欧元等等，拿回去就是它的外汇储备了，因为这相当于央企的一个融资的方式，但是只要进入中国，中国就把这外汇拦下，开始兑换成人民币。〞
 <br/>
 <br/>
 从今年3月份川普宣布要求全球公平贸易以来，中共一直怠慢美国贸易代表团的谈判愿望，以不断升级加征美国关税、甚至打击川普票仓等手段威胁，在川普宣布对中国2000亿美元产品加征对等关税后，中共在再无美国产品加征关税的情况下，用扩大国内出口退税产品数量、和加大退税率等违背国际公平贸易原则的手段，刺激出口。
 <br/>
 <br/>
 中共甚至采取阻止美国公司在全球的交易来进行报复。美国芯片制造商高通(Qualcomm)花了两年时间，敲定440亿美元收购一家荷兰半导体制造商的交易，随着中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的爆发，已经被中共的监管机构扼杀。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田认为，美中贸易冲突不断升级是中共不但不改邪归正，反而报复川普公平关税的后果。
 <br/>
 <br/>
 谢田：〝川普政府过去说得很清楚他们不想打贸易战，川普政府官员本身也没有用贸易战这个词，川普实际是要纠正以前的贸易错误，川普同时也说过，如果中共再报复，在2000亿基础上还是不愿意做出让步，不愿意改邪归正的话，再有任何报复手段的话，他就会毫不犹豫把剩下的2600亿压下去。〞
 <br/>
 <br/>
 按照美国《2015年贸易便捷与贸易促进法》的规定，一个国家和地区只要符合以下3大指标，就可以判定〝汇率操纵〞:1，〝200亿美元以上的巨额对美贸易顺差〞；2，〝大量经常帐户盈余占GDP3%以上〞；3，〝持续、单向的外汇干预〞。
 <br/>
 <br/>
 一旦确定中国为〝汇率操纵国〞，那么美国财政部将会与中方启动双边谈判，敦促中方说明低估本币价值的依据。一年后如果还在操纵汇率，就会受到惩罚。包括禁止参与美国政府采购、得到美国海外私人投资公司融资、要求国际货币基金（IMF）加强监督、签订任何贸易协议时，都考虑操纵行为。
 <br/>
 <br/>
 谢田认为，一旦涉及到外汇操控，就意味着中美贸易冲突升级到了货币战甚至金融战，同时也等于美国政府确定了中共是在吸取中国人民血汗，这样就对美国反倾销和反不正当贸易更加有利，对中共的打击会更大。
 <br/>
 <br/>
 采访编辑/刘惠  后制/周天
