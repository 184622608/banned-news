### 网约车公司Lyft申请上市 先Uber一步
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】周四，美国第二大网约车服务公司Lyft赶在竞争对手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Uber.html" target="_blank">
  Uber
 </a>
 之前宣布，已经启动首次公开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_募股.html" target="_blank">
  募股
 </a>
 程式，申请
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上市.html" target="_blank">
  上市
 </a>
 。
 <br/>
 <br/>
 网约车服务公司Lyft在今天发布新闻，表示它已向美国证券交易委员会申请首次公开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_募股.html" target="_blank">
  募股
 </a>
 。
 <br/>
 <br/>
 Lyft先于竞争对手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Uber.html" target="_blank">
  Uber
 </a>
 ，提前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上市.html" target="_blank">
  上市
 </a>
 ，目前震荡波动的股票市场对打车公司Lyft的上市是一个挑战。
 <br/>
 <br/>
 这家总部位于旧金山的公司秘密向美国证券交易委员会提交了一份登记申请书。
 <br/>
 <br/>
 预计在美国证券交易委员会完成相关审查程序后开始首次公开募股。
 <br/>
 <br/>
 目前Lyft尚未确定IPO的募股数量和发售价格范围。该公司今年早些时候的估值超过150亿美元。
 <br/>
 <br/>
 Lyft的竞争对手Uber计划明年进行首次公开募股。Uber致力于开发自动驾驶汽车的进程在过去一年有所减缓，因为它受到指控，从谷歌子公司偷窃了相关技术。并且它还涉及一起自驾汽车引发的致命车祸。
 <br/>
 <br/>
 Lyft和Uber两家公司一直致力于提供多元化交通工具。这两家公司都在今年早些时候收购了自行车共享公司。
 <br/>
 <br/>
 新唐人记者Jeremy Sandberg纽约报导
