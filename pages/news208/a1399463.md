### 11月14日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.在路透全球投资2019年展望峰会上，数位投资者表示，2019年美国股市表现将好于全球其它股市。即使美国经济在放缓，但其它经济体的放缓程度会更大，结果是其它股市要比美股承受更多压力。
 <br/>
 <br/>
 2.继英国和法国之后，西班牙希望从2040年起，禁止销售柴油和汽油新车。在政府的能源转型法草案当中，要求2021年前设置电动车充电站，2050年前电动车系统只能完全仰赖再生能源。
 <br/>
 <br/>
 3.美国市场原油期货在周二大跌，每桶油价创下今年最低纪录，而一个月前油价处于四年高位。市场分析师认为原来高涨的原油行情有所变化，不少远期石油期货价格超过了现价，说明原油远期有供过于求的趋势。
 <br/>
 <br/>
 4.巴菲特旗下的柏克夏海瑟威投资公司已经卖出了最后一批沃尔玛股票，结束了20多年的投资关系。网络零售商亚马逊对实体零售业形成的巨大威胁，而沃尔玛则在网络销售上大力投资来追赶亚马逊。
 <br/>
 <br/>
 新唐人记者米西、正治综合报导
