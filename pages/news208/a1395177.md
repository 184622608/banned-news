### 10月12日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】10月12日财经速瞄
 <br/>
 <br/>
 周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 结束前几日的下跌趋势，在震荡中上涨。道指如过山车般大幅动荡，早盘大涨超过400点后大幅下挫，一度走跌后又回涨，震荡幅度近500点，收盘时上涨287点。标普结束六连跌，收涨1.42%，上涨38.76点。科技股领涨股市，纳指飙升2.29%，上升167.83点。
 <br/>
 <br/>
 周五，华尔街财报季开始，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摩根大通.html" target="_blank">
  摩根大通
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花旗.html" target="_blank">
  花旗
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富国.html" target="_blank">
  富国
 </a>
 银行都公布了第三季度的财报，三家银行的净利润均大幅增长。摩根大通季度营收278亿美元，连续第三个季度超出市场预期，受税率降低推动净收益上涨24％，来到84亿美元。花旗集团三季度净利润为46.2亿美元，同比增长12%，超过市场预期。由于成功削减成本，美国第四大银行富国银行净利润为60亿美元，同比大增33%。
 <br/>
 <br/>
 中国海关数据显示，中国9月份对美国的贸易顺差飙升至创纪录的341.3亿美元，而8月份为310.5亿美元。1月至9月，中国对美国的贸易顺差为2257.9亿美元，而去年同期为约1,960.1亿美元。路透社分析，这种情况可能会使白宫在贸易上向北京加大施压。
</div>
