### OPEC达成减产协议 1月起每日减120万桶
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】为拉升
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 ，周五，正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维也纳.html" target="_blank">
  维也纳
 </a>
 举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_OPEC会议.html" target="_blank">
  OPEC会议
 </a>
 达成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_减产协议.html" target="_blank">
  减产协议
 </a>
 。自明年1月起，每日将减产原油120万桶。
 <br/>
 <br/>
 尽管美国总统川普（特朗普）一再表示，希望维持当前的低
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 来支撑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球经济.html" target="_blank">
  全球经济
 </a>
 发展，但出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_OPEC会议.html" target="_blank">
  OPEC会议
 </a>
 的各产油国最终达成协议，从明年1月开始减产，以提升油价。
 <br/>
 <br/>
 其中，以沙特为主的OPEC国家每日减产原油80万桶，但正在受到美国制裁的伊朗得以豁免。非OPEC国家每日减产40万桶，其中原本反对减产的俄罗斯改变态度，表示愿意合作，并同意每日减产约23万桶。
 <br/>
 <br/>
 受到产量增加及全球需求减弱的影响，国际油价自今年10月份以来已经下跌30%，而OPEC达成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_减产协议.html" target="_blank">
  减产协议
 </a>
 的消息传出后，国际油价立即应声上涨近5%。
 <br/>
 <br/>
 新唐人记者乔安综合报导
</div>
