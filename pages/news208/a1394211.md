### 10月4日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.世界贸易组织报告预测，人工智能，3D打印，区块链等新数字技术的发展，将起到历次工业革命一样的作用，会降低各行业运作成本。受到这些技术的推动，到2030年全球贸易可能会增长1.8%到2%。
 <br/>
 <br/>
 2.美国制裁伊朗，要求十一月四日起全面禁运伊朗石油，影响到国际原油价格，油价持续上涨。尽管美国原油库存增加，仍然难抵禁运造成的供给下降。美国总统川普再次要求沙特阿拉伯增产，但国际原油价格暂时没有缓解。北海布兰特原油本周涨破每桶八十五美元，创四年来新高。
 <br/>
 <br/>
 3.美国的消费者报告对市场上各种智能手机作了测评和排名。iPhone XS和iPhone XS MAX 排名第二和第三位，iPhone改进最大的是电池续航能力。排名第一的是三星的Galaxy Note9。
 <br/>
 <br/>
 4.由于贸易紧张局势和资本流动不稳，世界银行调低东亚及太平洋地区明年的经济增长预测至6.3%。到了2020年，世银预估该地区的经济增速为6%。
 <br/>
 <br/>
 新唐人记者米西、正治综合报导
</div>
