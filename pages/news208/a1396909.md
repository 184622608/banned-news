### 10月25日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美国官员表示，美国拒绝与中国重启贸易谈判，除非北京拿出具体方案来解决华盛顿对中方强制技术转让和其他经济问题的抱怨，预期双方僵局可能会影响两国元首在二十国集团领导人峰会期间的会晤计划。
 <br/>
 <br/>
 2.根据美国商务部数据显示，过去一年半以来，美国经济成长年率从2.2%，上升到2.9%，其中政府的支出，为经济提速作出了近一半的贡献，尤其是国防支出的大幅增加，推动了美国经济的普遍提速。
 <br/>
 <br/>
 3.日本机器人工业会统计今年7~9月的工业机器人总出货额同比下滑5.0%，降为1,861亿日元。为两年多以来首次出现减少，主要是向中国的出口额减少了近2成，背后的因素是中国汽车零部件厂商，受到贸易摩擦影响而暂缓投资。
 <br/>
 <br/>
 4.中国电信业者中兴通讯预计，今年将净亏损62到72亿元人民币，每股亏损1.48到1.72元人民币，突显出今年早些时候公司被禁止与美国企业开展业务的巨大影响，形成对比的是去年中兴通讯的利润为45.7亿元人民币。
 <br/>
 <br/>
 新唐人记者保罗、正治综合报导
