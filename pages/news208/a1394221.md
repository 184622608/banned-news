### 美国劳工部报告：失业率1969年以来新低
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】周五，美国劳工部公布最新就业数据。九月份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业率.html" target="_blank">
  失业率
 </a>
 为3.7%，是近50年来的新低点。全美新增就业岗位13.4万个，失业人数继续减少。
 <br/>
 <br/>
 劳工部报告显示，美国就业保持增长势头，九月份增加了13.4万个就业岗位。在专业领域、企业服务以及健康保健领域增长较多，运输业和仓储业也有所增长。
 <br/>
 <br/>
 九月份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业率.html" target="_blank">
  失业率
 </a>
 为3.7%，达到1969年以来的新低。失业人数比上个月减少27万，降到600万。
 <br/>
 <br/>
 九月份美国工人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平均时薪.html" target="_blank">
  平均时薪
 </a>
 环比增长0.3%，达到27.24美元。每周工作时数维持在34.5个小时。
 <br/>
 <br/>
 劳工部还将7、8两个月份的新增非农
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业人数.html" target="_blank">
  就业人数
 </a>
 上修了8.7万人。
 <br/>
 <br/>
 报告特别提到，上个月，飓风佛罗伦斯重创东海岸部分地区，是美国有史以来最严重的风暴之一，估计造成经济损失380亿到500亿美元。
 <br/>
 <br/>
 新唐人记者王羽音、宇微综合报导
</div>
