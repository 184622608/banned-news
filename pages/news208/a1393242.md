### 9月27日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年09月27日讯】下面是今天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财经速瞄.html" target="_blank">
  财经速瞄
 </a>
 <br/>
 <br/>
 周四，美国商务部公布数据显示，美国二季度实际GDP终值4.2%，为自2014年第三季度以来最快增速。原因是1.5万亿减税计划的驱动，促进了此前几乎停滞的消费者支出。此外，增速达8.7%的商业投资也提振了第二季度的经济表现。
 <br/>
 <br/>
 美国商务部数据显示，8月耐用品订单表现亮眼，上涨4.5％，为六个月以来最高水平。主因是受商用飞机需求激增的影响，8月商业飞机订单数量上涨69.1％，拉动了整体运输订单上涨13％。显示当前的企业信心处于多年高位。
 <br/>
 <br/>
 电商巨擘亚马逊表示，将在纽约市开设一家杂货商店，名称是〝Amazon 4-star〞，贩售最畅销、最受消费者喜爱的商品包括玩具、家用品、书籍、游戏等。这是亚马逊进军实体零售的最新动作，给Target百货等竞争对手带来最新挑战。
</div>
