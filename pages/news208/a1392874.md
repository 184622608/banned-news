### 中国股市下滑 美中贸易战令市场担忧
------------------------

<div class="wysiwyg">
 【新唐人2018年09月25日讯】周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚洲股市.html" target="_blank">
  亚洲股市
 </a>
 走势挣扎，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 下滑，最近一轮的美中贸易紧张局势，增添外界对全球经济增长风险的担忧。
 <br/>
 <br/>
 周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 出现下滑局面，上证指数开盘走弱，跌0.8%，中午仍维持同样跌幅。
 <br/>
 <br/>
 香港市场周二休假。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本股市.html" target="_blank">
  日本股市
 </a>
 日经指数周二早盘小幅上涨，这有益于芯片（晶片）设备生产类股上涨提振，但涨幅有限。美元兑日圆升破112日圆支撑了日本出口企业的信心。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市.html" target="_blank">
  美国股市
 </a>
 周一收盘标普500和道琼工业指数下跌，道琼工业指数 DJI 下跌约0.7%， 标准普尔500指数SPX跌0.35%。
 <br/>
 <br/>
 美联储将在今天（周二）举行为期两天的政策会议。
 <br/>
 <br/>
 新唐人记者陈依心、明玉综合报导
