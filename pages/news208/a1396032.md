### 中国GDP第三季增长 近10年新低
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】中共国家统计局周五公布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第三季度.html" target="_blank">
  第三季度
 </a>
 国内生产总值增长为6.5%，这是2009年第一季度以来的最低值。分析认为，主要原因是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 扭转不公平贸易，对非市场经济的影响。
 <br/>
 <br/>
 周五，中共国家统计局公布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第三季度.html" target="_blank">
  第三季度
 </a>
 GDP增长为6.5%，今年三个季度呈连续下降趋势。这是2009年第一季度以来，中国经济表现最差的季度。
 <br/>
 <br/>
 有市场分析师表示，中国第三季度数据令人失望，主要原因是工业生产放缓。
 <br/>
 <br/>
 WHI公司首席市场战略师 英格拉姆：〝看上去，其核心原因是工业生产放缓。最近的采购经理人指数表明，特别是(中国)小型公司表现不佳。〞
 <br/>
 <br/>
 也有专家认为，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 中，中国的损失更大。
 <br/>
 <br/>
 荷兰合作银行高级外汇策略师 福勒：〝我认为，美国的出口只相当于其GDP的10％，这意味着，虽然美国因贸易战有些损失，但不会像中国受到这么大影响。〞
 <br/>
 <br/>
 对美中贸易战的担忧，引发中国股市大幅抛售，股票全面走低，周四再次下挫。人民币兑美元汇率在贸易摩擦升级后也急剧下跌。
 <br/>
 <br/>
 第三季度，中国的汽车销售放缓，制造商减产10％以上。
 <br/>
 <br/>
 这加剧了市场对中国经济前景的担忧。有经济学家预测，明年的年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_增长率.html" target="_blank">
  增长率
 </a>
 将下降到6.3%。
 <br/>
 <br/>
 新唐人记者王羽音、宇微综合报导
</div>
