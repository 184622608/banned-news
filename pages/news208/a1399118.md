### 美股大跌600点 台股反弹气势受阻
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】受到苹果（Apple）股价重挫拖累，欧美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 科技类股12日深陷阴霾走跌，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 道琼指数更大跌600点，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台股.html" target="_blank">
  台股
 </a>
 今 (13) 日难逃崩跌命运，在权值股、中小型股全面走跌下，早盘一度大跌超过200点。随后苹概股有低接买盘支撑，盘中加权指数跌点缩至120点左右，9700点关卡失而复得。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 科技股近期剧烈震荡，再度不支倒地，道琼工业指数今晨收盘大跌602点。同样欧股也跟跌，包括伦敦 FTSE 100 指数、法兰克福 DAX 30 指数与巴黎 CAC 40 指数等三
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 同步下跌作收。
 <br/>
 <br/>
 苹果公司股价在供应商调降财测后惨跌5%，卖压蔓延其他类股，成为拖累大盘的元凶之一。
 <br/>
 <br/>
 日韩股市今天也都出现大跌，韩国股市跌幅超过2%、日本股市跌幅甚至达3%以上。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台股.html" target="_blank">
  台股
 </a>
 则是直接以下跌104点跳空开出，且在台积电、鸿海、大立光等权值股无力撑盘下，一度大跌超过200点，创11月以来新低，反弹气势受阻。
 <br/>
 <br/>
 <h4>
  苹果光失色 多档破底
 </h4>
 <br/>
 据中央社报导，苹果新手机销售表现失色，供应链营运遇压，随着苹果股价重挫5.04%，台湾苹概股今天跟着倒地，多档个股持续破底，股王大立光开盘3000元关卡也失守，台积电下跌逾2%。
 <br/>
 <br/>
 美股在苹果股价重挫，加上苹果3D感测供应商Lumentum外传出货量遭大幅降低而下修财测，股价应声重挫近33%，IQE同步惨跌近29%拖累下，道琼工业指数12日大跌602点，拖累台股重挫，苹概股首当其冲，成为重灾区。
 <br/>
 <br/>
 在利空讯息持续重击下，今天苹概股再度倒地，大立光3000元关卡失守，早盘低点一度来到2875元，再度破底，跌幅达5.7%。股价已创下2016年7月，28个月以来新低。
 <br/>
 <br/>
 玉晶光跌幅超过7%，低点来到181.5元，同样跌破前低，为2017年3月以来，约20个月来新低；可成低点来到244元，也加入破底行列，跌幅7.8%，创下22个月来新低。破前低的还有嘉联益、GIS-KY、美律、和硕、康控-KY、华通等。
 <br/>
 <br/>
 苹果刷脸商机褪色，身为3D感测供应商Lumentum外传出货量遭大幅降低而下修财测，代工厂稳懋受到重击，开盘直接跳空跌停价97.7元开出，失守百元关卡。
 <br/>
 <br/>
 法人认为，虽然苹果第3季获利亮眼，但硬体销售增速明显放缓，加上原本期待的换机潮并未出现，研判苹概股仍需要一段时间休息打底，反应利空消息，且新机开卖后，市场将回头检视先前预测数字，从往年经验分析，苹概股大多都会面临到一波修正卖压，今年也维持类似走势。
 <br/>
 <br/>
 (责任编辑：夏明义)
