### 11月20日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美国联邦存款保险公司统计，美国银行业上季营收达2,038亿美元，利润620亿美元，同比增长29.3%，创下历史新高。带动因素主要是利率上扬，利息收入同比增长7.5%，服务费和投资银行增加4%。
 <br/>
 <br/>
 2.百思买上季营收和获利，优于市场预期。面对来自在线零售的竞争，公司采取措施，允许顾客在店测试新科技，加快货运，吸引顾客。百思买再次上调全年获利展望，每股获利在5.09到5.19美元之间。
 <br/>
 <br/>
 3.欧洲议会和欧盟28国谈判代表近日达成一项影响深远的制度协议，来协调对外国投资欧洲的审查，以保护战略性技术和基础设施，尤其是来自中共政府的大规模投资。据统计，中共2016年在欧洲投资增长77%，达到350亿欧元。
 <br/>
 <br/>
 4.澳洲悉尼晨锋报发表调查报告，指中共政府今年以来增加了网络攻击，窃取澳洲的商业机密。美国、英国、澳洲和新西兰情报机构组成的一个联盟，侦测到这波网络攻击，主使者是中共国安部门。一名资深澳洲官员表示，澳洲企业和大学对来自中共当局的网络攻击防范不严。
 <br/>
 <br/>
 新唐人记者保罗、明玉综合报导
 <br/>
