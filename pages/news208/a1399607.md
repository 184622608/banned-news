### 11月15日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.国际货币基金组织表示，对2018和2019年全球经济增长预期，稳定在2017年的水平3.7%。不过全球几大主要经济体的增速都在放缓，主要因素包括美国利率走高、美元走强、金融市场波动以及全球贸易摩擦加剧。
 <br/>
 <br/>
 2.星巴克将微调部分饮品的价格，涨价举动被认为是缓解业绩压力的销售策略。星巴克最新一季的营收为63亿美元，年增率10.6%，销售增长与产品平均价格上涨有很大关联。有分析师认为星巴克涨价长远来说会影响顾客购买意愿。
 <br/>
 <br/>
 3.富国银行计划在美国裁员1千名员工。裁员对象主要是消费者贷款和支付、虚拟解决方案和创新团队成员。大约9百个被裁员职位将在房屋贷款部门，原因是房贷申请和还贷违约的客户数量都在减少。
 <br/>
 <br/>
 4.三星电子计划在未来几年投入220亿美元，投资5G网路和人工智慧，目标是在2020年以前至少取得20％的网络设备市场份额。三星表示，5G网络将开启人工智慧的潜能。目前三星是全球第5大网络设备制造商。
 <br/>
 <br/>
 新唐人记者米西、正治综合报导
