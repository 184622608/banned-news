### 10月29日环球财经简讯
------------------------

  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美9月通胀小幅减缓
 <br/>
 美国9月的消费者支出上升0.4%，符合分析师预估，连续7个月上升。个人消费支出物价指数年率由8月的2.2%下降至2%。目前消费者支出速度足以维持经济强势，2%的通胀率也是联储局希望看到的数据。
 <br/>
 <br/>
 2.高通称苹果欠款70亿美元
 <br/>
 去年4月开始，苹果和高通展开一系列诉讼。苹果以收费不公为由，与其供应商扣留大约10亿美元的专利使用费。高通上周在法庭听证会上，再度指控苹果拖欠70亿美元的专利使用费。双方的法律纠纷将耗费漫长时间才能解决。
 <br/>
 <br/>
 3. 拟2020年开征数位服务税
 <br/>
 英国政府计划开征新的数位服务税，主要锁定脸书、亚马逊与谷歌等美国科技业的避税大户。数位服务税的税率为企业在英国收入的2%，预计2020年4月生效，预估当年可带来4亿英镑的新税收。
 <br/>
 <br/>
 4.日9月失业率降至2.3%
 <br/>
 日本9月的失业率，由8月的2.4%滑落至2.3%，接近25年以来的最低水准，职位空缺再度上扬。日本的劳动力市场持续热络，薪资缓慢增长，更高的薪资将有助于日本央行实现2%的通胀率。
 <br/>
 <br/>
 <br/>
 新唐人记者杰森综合报导
