### 11月7日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美国商务部表示，对来自中国的通用铝合金板征收96.3%到176.2%的反倾销关税，这是现任美国政府启动调查以来首次裁定征收关税。2017年美国从中国进口了约9亿美元的铝合金板，美国铝业表示受到不公平冲击。
 <br/>
 <br/>
 2.Michael Kors公布上季财报，营收12.5亿美元，同比增长9.3%，净利1.37亿美元，同比大幅下降37%，每股获利1.27美元，不如市场预期，Micheal Kors一直在努力转向高端精品路线，连续并购Jimmy Choo和范思哲(Versace)，但仍无法扭转消费者的品牌印象。
 <br/>
 <br/>
 3.三星公司公开展示了可折叠手机原型机，希望凭藉可折叠手机来扭转移动部门获利大幅下降的局面。该款手机打开后，屏幕尺寸可达7.3英寸。有分析师指出，虽然该设计很亮眼，但由于手机厚度增加，消费者是否认同还是个问题。
 <br/>
 <br/>
 4.上周美国原油库存增幅超出市场预测的一倍。同时美国原油日产量达到了创纪录的1160万桶。虽然有诸多推高油价的因素，但由于需求放缓，石油市场可能在明年出现供过于求。消息人士称，俄罗斯和沙特阿拉伯已经开始讨论未来的减产。
 <br/>
 <br/>
 <br/>
 新唐人记者保罗、正治综合报导
