### 10月1日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年10月01日讯】10月1日财经速瞄
 <br/>
 <br/>
 美加就新的贸易协定达成框架协议的消息传出后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 在10月第一个交易日迎来开门红。在9月以近四年最佳季度表现收官后，周一，三大美股再度高开。标普500高开0.49%，收盘时上涨10.61。道指收盘时上涨192.90点。
 <br/>
 <br/>
 周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特斯拉.html" target="_blank">
  特斯拉
 </a>
 美股暴涨超过17%。上周六，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马斯克.html" target="_blank">
  马斯克
 </a>
 与SEC达成和解协议，马斯克个人和特斯拉公司各支付2000万美元罚金。但马斯克未来三年将不能担任董事长职务。马斯克并声称公司非常接近季度盈利。周一最新数据显示季度产车8万辆，或创纪录新高。
 <br/>
 <br/>
 国际投行高盛发表报告指出，中国8月和9月有8家民营企业出现债务违约，数量及金额同创历史新高。今年前7个月只有11家企业债务违约。今年以来的倒债总额为914亿元人民币，远高于2016年全年的539亿元。
</div>
