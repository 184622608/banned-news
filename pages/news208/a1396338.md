 <br/>
 美国要求世界贸易组织(WTO)调查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国智慧财产权.html" target="_blank">
  中国智慧财产权
 </a>
 政策，因为中共政府规定，在合约到期后，外国专利持有人不能阻止中国实体继续使用相关技术，同时，中共给予外国进口技术较差的待遇，涉嫌违反WTO规定。
 <br/>
 <br/>
 宝洁(Procter &amp; Gamble)发布第3季财报，营收166.9亿美元，创下近5年来新高，每股获利为1.12美元，优于市场预期，主要是美容部门销售成长所推动，公司同时重申有希望实现全年获利同比增长3~8%的目标。
 <br/>
 <br/>
 欧盟警告
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 政府预算草案违规后，意大利10年期公债殖利率飙高到3.75％，创下2014年初以来新高，而被市场当作投资风险先行指标的意大利与德国公债殖利率利差，也创下五年多来的新高，达到345个基点。
 <br/>
 <br/>
 美国全国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地产经纪商.html" target="_blank">
  地产经纪商
 </a>
 协会(NAR)表示，9月份成屋销售较前月下滑3.4%，换算成年率为515万户，连续第六个月下降，创下近三年来最低水位，低于市场预期，显示房价及抵押贷款利率的上升，使得潜在买家抱持观望态度。
 <br/>
 <br/>
