### 联准会暗示升息近尾声 美股4大指数应声大涨
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 (特朗普）和中国国家主席习近平将举行会谈，投资人期待能有好的结果。加上美国联准会主席鲍威尔周三（27日）暗示升息近尾声，美股4大指数应声全面大涨。
 <br/>
 <br/>
 鲍威尔（Jerome Powell）周三表示，目前利率略低于中性估值，投资人对此解读为升息步调可能停止或放缓。因此，美股4大指数全面大涨，道琼工业指数上涨617.70点、涨幅2.50%、收在25366.43点；NASDAQ指数上涨208.89点、涨幅2.95%、收在7291.59点；S&amp;P 500指数上涨61.62点、涨幅2.30%、收在2743.79点；费城半导体指数上涨27.37点、涨幅2.27%、收在1231.45点。
 <br/>
 <br/>
 在投资人观望气氛相对浓厚下，欧股3大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 收盘微幅上涨或下跌，伦敦FTSE 100指数终场下跌12.33点、跌幅0.18%、以7004.52点作收；巴黎CAC 40指数收盘几乎持平、微涨0.09点、收在4983.24点；法兰克福DAX 30指数跌10.23点、跌幅0.09%、收在11298.88点。
 <br/>
 <br/>
 分析师表示，美股大涨，尤其是苹果(Apple)股价涨幅逼近4%，可望激励台湾电子股表现。台股周四（29日）受到美股飙涨激励，近来弱势的苹概股、涨价概念股苏醒领涨，带动电子股信心大振，股王大立光大涨150新台币（4.87美元），台积电劲扬2%，相关类股雨露均沾。
 <br/>
 <br/>
 近期台股表现以中小型股较为突出，显示内资在年底作帐行情前有抢先布局的迹象，加上农历新年封关比往年早，农历年前的财报空窗期也有利营造作梦行情的空间。
 <br/>
 <br/>
 而外资近期也有偏多布局的迹象，不但在现货市场买多于卖，在台指期市场更加码多单至近期相对高点。分析师认为因内外资同步偏多，台股有机会持续反弹力道，可望挑战11月以来高点10014点。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
