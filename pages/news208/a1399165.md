### 11月12日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.倒债风险升高 奇异急售资产
 <br/>
 奇异公司受到电力事业亏损和220亿美元资产减记的影响，上一季大亏228亿美元。为了减少负债，奇异将紧急出售资产。截至第三季，奇异的债务达到1千140亿美元，达到其股东权益的3.7倍。
 <br/>
 <br/>
 2.思爱普80亿美元收购Qualtrics
 <br/>
 德国思爱普公司将以80亿美元收购消费者线上调查公司Qualtrics International。这将是思爱普自2014年以来最大宗的并购案。Qualtrics收集全球9千家企业的客户、员工、产品和品牌的反馈和数据。
 <br/>
 <br/>
 3.JDI连6季亏损 下调全年财测
 <br/>
 苹果供应炼之一的日本显示器公司公布，因为智能手机生产商推迟采用OLED萤幕，导致第二季亏损47亿日圆，连续6季亏损，该公司还下调全年财测，全年营收增幅由10%~20%降低到5%~15%。
 <br/>
 <br/>
 4.软银电信事业将上市 筹资210亿美元
 <br/>
 日本软银集团旗下电信事业，预计12月19号在东京证交所挂牌交易，预定募集2.4万亿日圆，大约210.4亿美元，将成为史上最大规模初次上市案之一。该电信事业正面临日本行动通讯同业降价的压力。
 <br/>
 <br/>
 <br/>
 新唐人记者杰森综合报导
