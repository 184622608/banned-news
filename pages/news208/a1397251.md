 【新唐人2018年10月29日讯】10月29日财经速瞄
 <br/>
 <br/>
 周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 开盘走高，但受消息影响，收盘下挫。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技股.html" target="_blank">
  科技股
 </a>
 遭重挫，纳指收跌1.63％，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_标普.html" target="_blank">
  标普
 </a>
 跌0.66％，道指跌约245点。亚马逊大跌6.33％，市值蒸发超500亿美元。奈飞跌5％，阿里巴巴跌6.64％。道指日内震荡超过700点。彭博社报导，如果在G20峰会上，川普和习近平的会谈不能缓解贸易战，美国准备在12月初宣布对所有剩余的中国进口产品征收关税。消息一出，美股急转直下。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高盛.html" target="_blank">
  高盛
 </a>
 首席美股策略师David Kostin称，基本面仍然可以支撑价值，一个月来美股被〝过度〞抛售，未来两个月，随着企业重新回购，股市将迎来反弹，标普有望回涨接近6 %。高盛预计，今年美股上市公司回购规模将约为1万亿美元，这将支持标普500达到高盛预计的目标价2850。
 <br/>
 <br/>
 几十个国家正加大力度，研究向谷歌和脸书等科技巨头开征
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_数字税.html" target="_blank">
  数字税
 </a>
 。英国财政大臣称，科技巨头在英国赚取了大量收入，却没有按照比例缴税，这十分不公平。他表示，英国将在2020年4月推出针对科技巨头数字服务税。消息一出，科技股遭遇重挫。
