### 10月11日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】下面请看今天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财经速瞄.html" target="_blank">
  财经速瞄
 </a>
 。
 <br/>
 <br/>
 由于投资者担心债券收益率过高以及利率上升的趋势，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市.html" target="_blank">
  美国股市
 </a>
 周四延续跌势，所有11个主要标准普尔500指数板块均走低，标普跌超2%，道指跌2.13%，收跌545点，纳指跌1.25%。
 <br/>
 <br/>
 周四，美国能源信息署公布了最新美国原油库存报告。报告显示，美国10月5日当周原油库存增加598.7万桶，大幅高于预期的280万桶。同时欧佩克再度下调全球需求预期，周四，WTI
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 与布伦特油价双双走低。WTI油价跌至$70.80，下跌3.25%，布伦特油价下跌3.55%到80.14美元。
 <br/>
 <br/>
 10月11日，大陆沪指、深成指、创业板三大股指齐跌，其中沪指下跌5.22%，创31个月最大单日跌幅，报2583.46点，下跌142.38点, 刷新近四年新低，深成指收跌6.07%；创业板报1261.88点，失守1300点关口，跌6.3%。
</div>
