### 人民币保7？大和明年看跌至7.6
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 中间价连跌四日，星期二大幅下调153个点子，报6.9629，在岸价一度跌破6.97，不过传有中资银行大手托市后，人民币反弹回升。外界估计人民币跌破七机会大，大和更预计人民币明年跌破7.6。
 <br/>
 <br/>
 曾准确预测2015年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 大跌的大和首席经济师赖志文表示，人民币兑美元〝破七〞是严重事情。他认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共央行.html" target="_blank">
  中共央行
 </a>
 目前处于两难状况。
 <br/>
 <br/>
 大和资本亚洲区首席经济师赖志文：〝（中共央行）他们是可以保七，但是很大可能是你会看到国内的债务危机，这就是现在正在发生的。〞
 <br/>
 <br/>
 赖志文预计，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 或持续10年，大陆老百姓会是最大输家！而中美债息扩阔，令人民币沽压有增无减，很可能爆发严重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信贷危机.html" target="_blank">
  信贷危机
 </a>
 。
 <br/>
 <br/>
 大和资本亚洲区首席经济师赖志文：〝美国以外全球的美元债是12万亿，我认为中国占1/4左右。因为现时中国的美元债规模，比起2015年及2016年至少高出30%至40%，人民币面对的沽压只会比15及16年多30%至40%，在贸易战环境令出口受压，盈余受压，进一步将人民币（贬值的）下额增加。〞
 <br/>
 <br/>
 该行预料人民币兑美元今年底将贬值至7.2，明年跌到7.6。赖志文担忧大陆债务违约或拖累香港银行，目前香港已有约500亿美元资金流出，预料走资情况将会加快。
 <br/>
 <br/>
 加拿大皇家银行认为，中共央行有七成半机会容许人民币跌破七，预期明年底人民币最低见7.5。
 <br/>
 <br/>
 新唐人记者梁珍、郭威利香港报导
 <br/>
 <br/>
