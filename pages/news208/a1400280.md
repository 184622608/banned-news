### 意大利预算案被判违规 欧盟警告或罚款
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 周三表示，意大利2019年预算案违反了欧盟财政规则，建议启动〝超额赤字程序〞，可能对意大利处以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罚款.html" target="_blank">
  罚款
 </a>
 。意大利稍微调整了以往的强硬态度，表示预算可以适当修改。市场乐观，欧元小幅上涨。
 <br/>
 <br/>
 欧委会副主席杜姆布罗夫斯基斯：〝(意大利预算)不符合(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 )债务要求。我们得出结论，监于债务问题，启动〝超额赤字程序〞是必要的。〞
 <br/>
 <br/>
 意大利债务总额超过国内生产总值的130％，为欧元区第二高的国家，远远超过欧盟规定的60%上限。
 <br/>
 <br/>
 但意大利在2019年预算案中，仍将财政赤字占国内生产总值比例提高到2.4%。
 <br/>
 <br/>
 欧委会副主席称，这项预算严重违背了欧盟财政规则。
 <br/>
 <br/>
 欧委会副主席杜姆布罗夫斯基斯：〝从意大利政府提出的这个(预算案)，我们看到，这个国家面对着不知不觉陷入不稳定的危险。〞
 <br/>
 <br/>
 目前欧盟迈出了制裁第一步。如果意大利不修正预算案，欧盟将启动〝超额赤字程序〞，这意味着欧盟将对意大利进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罚款.html" target="_blank">
  罚款
 </a>
 。这将是欧盟首次实行这一措施。
 <br/>
 <br/>
 意大利副总理萨尔维尼周三表现出让步姿态，称2.4%的赤字比例不能修正，但其他方面可以商量。
 <br/>
 <br/>
 意大利总理孔特态度则比较强硬，他表示预算案〝符合意大利和欧洲的利益〞。
 <br/>
 <br/>
 并希望在本周六的欧盟会议上说服欧委会。
 <br/>
 <br/>
 由于意大利的强硬态度有所缓解，资本市场反应较为正面。意大利10年期国债收益率下跌15个基点至3.46%；欧元兑美元小幅上涨0.3%。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
