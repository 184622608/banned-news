### 11月25日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 。
 <br/>
 <br/>
 世界贸易组织指出，20国集团自5月中至10月中，推行多达40项新贸易限制措施，比去年同期增加6倍，涉及的贸易规模大约4千810亿美元，四分之三都与征收
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 有关。如果情况持续，将损害全球经济增长。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑色星期五.html" target="_blank">
  黑色星期五
 </a>
 购物日，每人平均花费1千007美元，年增率4%。但是28%的美国人还没有还清去年感恩节的消费开支，，比2017年的14%急增1倍。不少美国家庭的债务问题，将在节日后更加恶化。
 <br/>
 <br/>
 欧盟成员国通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 协议。这份协议在英国掀起巨大反对声浪。英国首相梅伊表示，这份协议就是唯一可行方案，呼吁国会批准。欧盟也暗示不会再让步或重新谈判，呼吁英国民众支持。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空调大厂大金工业.html" target="_blank">
  空调大厂大金工业
 </a>
 计划用1千亿日圆，收购欧洲商用冰箱大厂AHT Cooling Systems。AHT是生产食品超市的冷藏/冷冻展示柜大。收购完成后，大金的冰箱事业营收有机会增加到1千亿日圆，达到目前的3倍。
 <br/>
 <br/>
 新唐人记者保罗、阳阳综合报导
