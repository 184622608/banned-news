### 日本第三季度GDP下滑
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】受到自然灾害和出口下降的影响，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第三季度.html" target="_blank">
  第三季度
 </a>
 的国内生产总值(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_GDP.html" target="_blank">
  GDP
 </a>
 )下降了1.2%。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 政府周三公布的经济数据显示，今年7月到9月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第三季度.html" target="_blank">
  第三季度
 </a>
 的国内生产总值（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_GDP.html" target="_blank">
  GDP
 </a>
 ）出现负增长，下降了1.2%，下滑幅度高于专家的预期。
 <br/>
 <br/>
 日本经济大臣茂木敏充在周三的记者会上表示，第三季度经济下滑是受到自然灾害和出口下降影响下的暂时现象，日本经济整体趋势依然看好。
 <br/>
 <br/>
 日本经济大臣茂木敏充：〝由于7至9月的自然灾害导致国内个人消费下降，再加上海外出口下挫，因此造成GDP暂时下滑。然而我们依然认为，日本经济在温和的复苏当中。〞
 <br/>
 <br/>
 茂木敏充还表示，未来的全球经济和金融市场充满了不确定性，日本需要对此保持警惕。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
