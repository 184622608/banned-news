### 10月30日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美国商务部以维护国家安全为由，宣布将中国晶片业者福建晋华列入出口管制实体清单。韩国晶片股周二应声上涨，SK海力士股价上升2.1％，三星电子上扬2.3％。
 <br/>
 <br/>
 2.三星电子受惠于数据中心芯片销售成长和产量提高，第三季获利创下历史新高，达到13.2万亿韩元，较去年同期增长17.5%。三星表示，第四季记忆体芯片价格将持稳，明年高密度记忆体的需求将成长。预估今年资本支出总额为31.8万亿韩元。
 <br/>
 <br/>
 3.日本三菱日联金融集团，同意以29亿美元收购受丑闻影响的澳洲联邦银行全球资产管理业务。该交易将让三菱日连在美国、澳洲和亚洲的资产管理规模增加1千496亿美元，整体资产管理规模将达到7千272亿美元。
 <br/>
 <br/>
 4.美国科技公司法拉第未来周二宣布，联合创始人、产品策略高级副总裁Nick Sampson离职。一年多来，公司三位联合创始人中两人离职，公司也陷入财务困境。之前，法拉第未来与其最大股东——中国恒大健康产业集团陷入法律纠纷。
 <br/>
 <br/>
 新唐人记者米西、明玉综合报导
