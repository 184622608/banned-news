 <br/>
 1.华尔街日报报导，美国政府正在试图说服美国盟国的通讯供应商避免使用华为生产的电信产品。有消息人士透露，美国官员和盟国相关的官员以及电信业高管会谈，他们认为华为的设备存在网络安全隐患。
 <br/>
 <br/>
 2.沙特能源部长法力赫表示，针对明年1月份原油需求疲弱的情况，沙特将做出相应的回应。他同时补充说，沙特的政策依然是保持市场的平衡，而且供过于求的局面不符合任何一方的利益。
 <br/>
 <br/>
 3.根据统计，第三季全球智能手机出货量为3.8亿支，同比减少5%，其中，韩国三星电子出货量蝉联第一，全球市占率为19%，中国华为排名第二，市占率为14%，美国苹果市占12%，出货量排名第三。
 <br/>
 <br/>
 4.日产汽车董事局一致通过，雷诺日产的董事长戈恩因为存在财务违法行为，将被公司革职。雷诺在看到了事件调查的关键性证据后，放弃了推迟开除戈恩的要求。日产汽车表示和雷诺的长期伙伴关系不会变。
 <br/>
 <br/>
 新唐人记者保罗、正治综合报导
