### 11月19日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美国11月建商信心指数 创近年低
 <br/>
 由于美国利率来到八年来最高水准，加上房地产价格累计上涨幅度，导致需求停滞不前，使得11月份住宅建筑商信心指数降到60，创2年多来最低水准，不如市场预期，显示房地产市场已出现降温。
 <br/>
 <br/>
 2.中国房地产开发商 狂发短债应急
 <br/>
 据统计，中国房地产开发商明年到期的人民币债务高达3,850亿元，相当于550亿美元，因此，开发商正在急于发售短期美元债券来偿还旧债，希望在明年债务到期之前，抢先在市场融资，以纾解资金压力。
 <br/>
 <br/>
 3.日企海外并购额 创历史新高
 <br/>
 据统计，日本企业前10月对外并购累计金额达到15.9万亿日元，创下历史新高，主要是企业利用日本国内低利率的优势，向升息的国家进行并购，其中，以对美国企业的并购数量与金额增加最为明显。
 <br/>
 <br/>
 4.苹果减少订单 冲击供应链
 <br/>
 由于苹果公司新款iPhone需求低于预期，加上推出更多款iPhone的决定，使苹果供应链陷入混乱，同时也加大了预测零部件和手机需求的难度。苹果大幅下调了9月份发布的所有三款iPhone的生产订单，令苹果供应商的管理人士及手机和零部件装配工人倍感沮丧。
 <br/>
 <br/>
 新唐人记者杰森综合报导
