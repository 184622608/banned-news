 周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股

 大跌。标普跌超3％，道指跌超600点，双双回吐年内全部涨幅。纳指下跌4.4％，创2011年8月来最大单日跌幅，其中奈飞暴跌9.4％，亚马逊跌5.91％，脸书跌5.41%。

 <br/>
 周二（10月23日），美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在接受《华尔街日报》专访时毫不留情地再次抨击美联储及主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲍威尔.html" target="_blank">
  鲍威尔
 </a>
 ，称他对美国经济增长构成威胁：〝每当我们做一些好事，美联储就加息。〞被问及是否后悔提名鲍威尔时，川普认为〝现在说这个还太早〞，但补充道〝也是有可能的。〞
 <br/>
 <br/>
 经济政策研究所周二公布的报告显示，中国加入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界贸易组织.html" target="_blank">
  世界贸易组织
 </a>
 导致

  美国贸易逆差
 </a>
 出现爆炸式增长，估计造成美国340万个就业岗位流失。这份报告评估了2001 - 2017年贸易逆差增长的影响。制造业首当其冲，在流失的岗位中74％为制造业。
