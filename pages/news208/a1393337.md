### 9月27日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年09月28日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美联储主席鲍尔(Jerome Powell)表示，希望人们多观察就业、薪资与通货膨胀数据，来获得货币政策的信号，而不是去猜测央行的意图或用字变化，这与过去10年美联储的做法有大幅的改变，过去的政策声明，含有企图影响市场观点。
 <br/>
 <br/>
 2.瑞银集团指出，多数发达市场金融中心的房地产存在泡沫风险或是严重高估情况，当中以香港的泡沫风险最大，德国慕尼黑、加拿大多伦多、温哥华、英国伦敦和荷兰阿姆斯特丹，也列为房地产泡沫的城市。
 <br/>
 <br/>
 3.为了规避美国对中国商品加征关税，日本松下公司(Panasonic)考虑将部分生产业务，从中国转移到墨西哥或是东南亚国家，据调查，在430多家美国公司中，约三分之一已经或是正考虑将生产基地转移到海外，东南亚是首选目的地。
 <br/>
 <br/>
 4.特斯拉(Tesla)执行长马斯克(Elon Musk)8月在推特发文，宣称获得公司下市所需资金，掀起风波，经过调查，美国证券管理委员会（SEC）控告马斯克诈欺，并请求法院禁止马斯克担任上市公司高层人员或董事，特斯拉恐陷入危机。
</div>
