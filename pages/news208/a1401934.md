### 12月3日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.国际研究机构Gartner预测，2019年全球穿戴装置出货量将达到2.25亿台，年增幅达到25.8%。目前的穿戴装置市场由苹果，Fitbit和三星等品牌主导。苹果的智慧手表支撑了整个穿戴装置市场的平均售价。
 <br/>
 <br/>
 2.欧洲央行明年将调整欧元区各国中央银行的出资占有率，这将影响到债券购买计划，以及欧元区最弱成员国可以收到的刺激规模，目前包括意大利、西班牙和希腊在内的12国，面临下调出资比例，刺激经济的力道因而减弱。
 <br/>
 <br/>
 3.意大利政府已经表示愿意与欧盟委员会妥协，可能不得不淡化或是延后一些选举承诺，总理孔特表示，将大规模删减早先的支出计划，努力降低预算赤字，以免因违反欧盟财政规定而遭到罚款。
 <br/>
 <br/>
 4.德国汽车生产商宝马公司表示，获利率维持8%-10%的目标不变，但是受到美中贸易战、大宗商品价格上涨，以及汇率波动等不利因素影响，明年预期收益将缩减至少10亿欧元，相当于11亿美元，需要寻找其他方法控制成本。
 <br/>
 <br/>
 新唐人记者保罗、正治综合报导
