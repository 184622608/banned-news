### 台湾工具机产值连14红 全年可破1400亿
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】接下来请看今天的财经100秒。
 <br/>
 <br/>
 1. 台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工具机.html" target="_blank">
  工具机
 </a>
 产值连14红　全年可破1400亿
 <br/>
 <br/>
 经济部今天发表统计指出，工具机去年产值年增率已由负转正，已连续14个月正成长。今年1到8月工具机业累计产值968亿元，虽然，近期受到中国倾销调查等影响，但在全球机械自动化潮流带动下，预期全年工具机业产值仍可超过1400亿元。
 <br/>
 <br/>
 2. 传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果新手机.html" target="_blank">
  苹果新手机
 </a>
 销量不佳 供应链遭砍单
 <br/>
 <br/>
 陆媒报导，苹果因为新款手机销量低迷，目前已对与合作的代工商富士康砍掉10%的iPhone XS、MAX 订单。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_iPhone销量.html" target="_blank">
  iPhone销量
 </a>
 不再公布 马斯克：以后没人排队
 <br/>
 <br/>
 苹果从本季开始，将不会在季度财报中提供iPhone等产品的个别销量，对此，电动车厂特斯拉执行长马斯克直言，苹果产品已经不像以前那样〝能够打动人心〞，他不认为将来消费者还有必要特地去抢购新款iPhone。
 <br/>
 <br/>
 4.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_友达群.html" target="_blank">
  友达群
 </a>
 创冲8K面板 备战东奥
 <br/>
 <br/>
 抢搭2020年东京奥运全面采用8K4K解析度转播赛事的商机，友达、群创今年都开始量产8K4K面板，预期接下来8K4K面板出货可望翻倍成长。
 <br/>
 <br/>
