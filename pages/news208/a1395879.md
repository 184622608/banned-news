  环球财经简讯
 </a>
 <br/>
 <br/>
 1.世界经济论坛发布国家竞争力排名年度报告，瑞士曾经连续9年拿下全球最具竞争力国家，如今掉到第4名，美国列首位，新加坡，德国位居第2和第3位，垫底的30名几乎都是非洲国家。
 <br/>
 <br/>
 2.市场预计苹果将在2020年推出使用自己开发的A系列处理器的苹果笔记本。苹果的A系列处理器，已经在iPhone和iPad上被广泛使用，如果苹果笔记本也使用该系列处理器，将打破Intel几乎垄断笔记本处理器的局面。
 <br/>
 <br/>
 3.脸书的创办人担任脸书的董事长及执行长两个职位。多家持有脸书股权的公共基金共同提交了一项提案，旨在推动脸书任命独立的董事长。但脸书创办人持有59.9%的股票，该提案在股东大会上被通过的可能性几乎为零。
 <br/>
 <br/>
 4.世界银行表示，全球贫穷人口比例，从1990年的67%下降到2015年的46%。但是还有近半数人口，每天的生活费不到5.5美元。因为经济成长减弱，贫穷人口的下滑速度未来不太可能持续。
 <br/>
 <br/>
 新唐人记者米西、正治综合报导
