### 11月26日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】11月26日财经速瞄
 <br/>
 <br/>
 26日是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网购星期一.html" target="_blank">
  网购星期一
 </a>
 ，不过，并不是所有商品都值得买。例如，智慧手机，最好等到12月电信商开始推动最终假期销售。冬季服装到1月买可能最划算。而首饰珠宝，等情人节档期会有更大折扣。
 <br/>
 <br/>
 由于寄希望于美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 缓和，英国脱欧以及石油价格趋稳，世界股市周一大多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上涨.html" target="_blank">
  上涨
 </a>
 。道指上涨超过354点，标普上涨41点，纳指上涨143点。
 <br/>
 <br/>
 纽约长岛大学穆督库塔斯教授表示，中共当局面临的最大问题不是贸易战，而是日益严重的债务。国际金融研究所（IIF）数据显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国债务.html" target="_blank">
  中国债务
 </a>
 占GDP比例是300%。也有专家表示，若考虑影子银行债务，中国债务占GDP比例可高达400%。
