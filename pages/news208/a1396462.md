### 东协成贸易战避风港 东南亚FDI大增
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】来关心美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 。许多企业为了规避
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，纷纷转移在中国的生产据点。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东南亚.html" target="_blank">
  东南亚
 </a>
 国家因为内需市场庞大，加上与中国邻近，成为许多厂商的首选。今年以来，流入越南、泰国以及菲律宾的外国直接投资，都大幅增加。
 <br/>
 <br/>
 为了避开美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，许多企业纷纷转往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东南亚.html" target="_blank">
  东南亚
 </a>
 投资，使得今年以来，东南亚国家的外国直接投资（FDI）大增。
 <br/>
 <br/>
 今年前九个月流入越南制造业的资金增加18%，其中包括韩国晓星集团（Hyosung）投资12亿美元的聚丙烯生产计划。今年1至7月泰国的净FDI较一年前增加53%，其中流入制造业的资金增加近五倍。同一期间，流入菲律宾制造业的净FDI也增加至8.61亿美元，大约是一年前的六倍。
 <br/>
 <br/>
 菲律宾电视台ABS-CBN (2018.8.9)：〝东南亚国协市场，可以利用这个机会投资基础建设，增加长期生产力。确实在发展中国家，投资报酬率仍高，是能轻易实现的目标。而且目前利率仍然相对低，特别是长期利率。我认为应该要把握这个机会。〞
 <br/>
 <br/>
 身为世界工厂以及与中国的地缘之便，一旦规避
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 的企业把生产基地移向东南亚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东协.html" target="_blank">
  东协
 </a>
 国家将成为厂商转移的投资热门地。根据调查，在中国有营运的430家受访美国企业中，有三分之一表示已经或正在考虑将生产地点移出中国。
 <br/>
 <br/>
 专家指出，到了2020年，东协的中产阶级人口将成长至总人口的65%。靠着广大的内需市场与相对低廉的劳动力，东南亚国家将有机会在美中贸易战的夹缝中获取一线商机。
 <br/>
 <br/>
 新唐人电视综合报导
