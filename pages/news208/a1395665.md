### 美股反弹 道琼指数大涨547点
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】强劲的美企财报与就业数据提振了市场情绪。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市.html" target="_blank">
  美国股市
 </a>
 在上周暴跌后反弹，周二（16日）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_道琼指数.html" target="_blank">
  道琼指数
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上涨.html" target="_blank">
  上涨
 </a>
 逾500点，标准普尔500指数所有11个板块同时反弹上涨，股市单日表现为今年3月以来最佳。
 <br/>
 <br/>
 《希望之声》报导，在一些大型美国公司公布优于预期的财报之后，周二
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_道琼指数.html" target="_blank">
  道琼指数
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上涨.html" target="_blank">
  上涨
 </a>
 超过547点，涨幅2.17%。因为医疗保健行业板块平均上涨超过2％，标普指数上涨2.15%，这是4月以来的最大单日涨幅，纳斯达克综合指数上涨2.89%。
 <br/>
 <br/>
 正在进行中的第三财报季被视为是左右股市动向的主要因素。高盛集团（Goldman Sachs  GS）和摩根・士丹利（Morgan Stanley  MS）等大型银行周二早间公布了强劲的财报。
 <br/>
 <br/>
 因表现高于预期，摩根・士丹利股价上涨超过4％。高盛也超出预期，其股价上涨1.6％。道琼斯成员强生和联合健康（UnitedHealth）都好于预期，联合健康股指跑赢大盘。
 <br/>
 <br/>
 个股方面，微软上涨3.16%，苹果上涨2.2%，推特上涨4.4%，脸书上涨3.43%。
 <br/>
 <br/>
 同日，美国劳工部数据显示，8月份的就业岗位增加了700多万，这也促成了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市.html" target="_blank">
  美国股市
 </a>
 周二的大幅上涨。
 <br/>
 <br/>
 投资公司Fort Pitt Capital的高级投资组合经理福雷斯特（Kim Forrest）认为，更多公司应该报告强于预期的收益。 〝我们对10月初的经济过于担忧，〞福雷斯特说， 〝我认为卖家分析师的标准很低。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 投资者并未因中美贸易战受到过多影响，CIBC私人财富管理公司首席投资官多纳必达（Dave Donabedian）表示，〝我们认为未来一年的收益率不会压低美经济水平，从而打击到股市。〞
 <br/>
 <br/>
 另一方面，由于汽车和黄金周的房地产销售大幅走低带来经济数据的不佳，周一中国A股受到新的打击，上证指数下降0.9％，深证A股指数下跌1.9％。
 <br/>
 <br/>
 《华尔街日报》对此评论说，〝由于汽车和十一假期的房地产销售大幅走低，国家经济数据不佳，中国股市的衰退来了。黄金周是在10月初为期一周的节日，通常是房地产类大宗消费的重要的时刻。〞
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
