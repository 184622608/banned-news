  环球财经简讯

 <br/>
 <br/>
 1. 在推出家庭智慧音箱后，亚马逊将进一步进入智慧家庭市场。亚马逊新的战略是将其智慧语音助手运行在家用电器上，推出了微波炉，时钟等智慧产品，此外亚马逊还准备以低价提供家电厂商智慧语音功能芯片。
 <br/>
 <br/>
 2. 据彭博社报导，阿联酋航空公司正在考虑并购阿提哈德航空，如果并购顺利进行，将会产生全球最大的航空公司，两家公司对并购案都不做肯定回答。目前全球规模最大的航空公司是美国航空
 <br/>
 <br/>
 3. 受益于农牧业产出和家庭支出，新西兰第2季经济增长较上季增长1%，达到两年来最快增幅。虽然增长率超过央行预测，但是商业信心下滑，可能抑制中期经济活动，降息依然是新西兰央行的选项之一。
 <br/>
 <br/>
 4. 美国第四大银行富国银行试图从业绩和名誉受损的销售丑闻中重振旗鼓，将在未来3年裁员5%~10%，作为重振计划之一，希望到2020年可以减支30亿美元。截至6月30日，富国银行共有26万4千5百名员工。
 <br/>
 <br/>
 新唐人记者米西、正治综合报导
