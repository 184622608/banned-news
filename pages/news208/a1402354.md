### 12月6日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美国能源资讯署(EIA)预计到11月底，美国石油每天产量将达到1,170万桶，创下历史新高，美国在依赖外国石油进口长达75年后，再次成为石油净出口国，预期美国今年石油产量，可能超过俄罗斯和沙特，成为全球最大石油生产国。
 <br/>
 <br/>
 2.由于担心情报外洩以及网路攻击,日本政府将修改内部采购规则，禁止政府相关单位向中国华为技术和中兴通讯购买电信设备，因为这些设备隐含安全性问题，可能会导致机密情报外洩，以及遭到网络攻击。
 <br/>
 <br/>
 3.日本总务省指出，10月份日本所有家庭平均消费支出为290,396日圆，同比下滑0.3%，连续第2个月陷入萎缩，不如市场预期，主要是因为今年冬季气温偏高，消费者对食品及御寒衣物支出减少所造成。
 <br/>
 <br/>
 4.亚马逊考虑将该公司的无人商店Amazon Go 推广到机场，以方便在机场的乘客购物。亚马逊已经在芝加哥，旧金山和西雅图等地开设了7家无人商店，主要面对在附近的需要快速解决午饭的上班族。
 <br/>
 <br/>
 <br/>
 <br/>
 新唐人记者保罗、正治综合报导
