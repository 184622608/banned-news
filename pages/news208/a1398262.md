### 11月5日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美联储前主席耶伦(Janet Yellen)表示，美国经济维持在接近充分就业的状态，有助舒缓美国不平等现象，以美国黑人为例，失业率在2010年触及高点之后一路下降，且随着招聘难度升高，企业也在投资培训员工，使欠缺技能的工人受益。
 <br/>
 <br/>
 2.日本政府公布9月所有家庭支出较去年同期意外下跌1.6%，远低于市场预期，主要是9月的颱风与地震等天灾，影响消费意愿，专家表示，未来在薪资改善的支撑下，家庭支出可能会温和上扬。
 <br/>
 <br/>
 3.据IDC统计，上季全球智能手机出货量降为3.55亿支，同比下滑6%，为连续4个季度下滑，其中，在中国销量更呈现连续6个季度同比下滑，IDC表示，由于今年销售疲软，预测明年将会迎来下一个智能手机换购潮。
 <br/>
 <br/>
 4.网络旅行社Booking Holdings公布上季财报，营收48亿美元，同比增长9%，营业利润为17.7亿美元，约每股获利37.78美元，上季贩售超过2亿房晚，同比大幅增长13.4%，可能是加入非酒店产物所贡献，虽然利润较低，但长久对公司有所帮助。
 <br/>
 <br/>
 新唐人记者杰森综合报导
