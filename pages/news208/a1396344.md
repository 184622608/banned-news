### 意大利高赤字比例新预算 引发欧盟担忧
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】再来关注欧洲消息。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 政府日前向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 提交2019年预算案，财政
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赤字.html" target="_blank">
  赤字
 </a>
 占国内生产总值比提高到2.4%，引发欧盟担忧。周一，股市下跌。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 承认，高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赤字.html" target="_blank">
  赤字
 </a>
 比例违反
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 财政规则，但承诺到2021年赤字比例将下降，如未能实现债务和赤字目标，将对其干预。
 <br/>
 <br/>
 周一，意大利总理孔特表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新预算.html" target="_blank">
  新预算
 </a>
 将有助于国内经济发展。
 <br/>
 <br/>
 意大利总理孔特：〝预算法案架构完善，关于（经济）增长的条例，有助于我们目前所做的工作。〞
 <br/>
 <br/>
 欧盟经济事务委员莫斯科维奇表示，因为意大利处于欧盟核心地位，不希望其在预算上发生危机。
 <br/>
 <br/>
 欧盟经济事务委员莫斯科维奇：〝欧盟委员会认为，意大利处于欧洲核心地位，欧元区的核心地位。〞
 <br/>
 <br/>
 意大利宣布新预算后，上周五欧元跌至近两周最低水平。评级机构穆迪将意大利信用评级下调至Baa3级，但分析师认为，意大利不会破产。
 <br/>
 <br/>
 巴德银行资本市场分析部主管 霍尔弗：〝意大利不会破产，因为那样欧元区就完了。〞
 <br/>
 <br/>
 周二，欧盟委员会将讨论预算案，并决定下一步计划。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
