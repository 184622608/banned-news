### 10月14日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】下面请看一组环球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财经简讯.html" target="_blank">
  财经简讯
 </a>
 。
 <br/>
 <br/>
 汽车资讯网站Edmunds.com公布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特斯拉.html" target="_blank">
  特斯拉
 </a>
 第3季在美国本土的销量达到6万4千727辆，直逼凌志、宝马与奔驰三大传统豪华车厂。随着特斯拉第4季产量持续创新高，销量可望继续攀升。
 <br/>
 <br/>
 德国联邦统计局公布，德国9月通胀率为2.3%，达到将近7年以来新高。主要原因是受到油价上涨拉动，当月油价年增率达到7.7%。德国9月食品价格增长2.8%，高于8月的2.5%。
 <br/>
 <br/>
 因为欧洲加强环保规范，日系车厂加快撤出欧洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柴油车.html" target="_blank">
  柴油车
 </a>
 市场。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_铃木汽车.html" target="_blank">
  铃木汽车
 </a>
 将在今年停止在欧洲贩售柴油车，三菱计划在英国、德国缩部分小中型柴油车款的销售。丰田、日产、本田和Subaru目前已经缩小在欧洲的柴油车销售。
 <br/>
 <br/>
 新唐人记者密西、阳阳综合报导
