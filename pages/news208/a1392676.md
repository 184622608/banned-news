### 英天空电视台收购案 康卡斯特竞价胜福克斯
------------------------

<div class="wysiwyg">
 【新唐人2018年09月23日讯】美国有限电视与媒体巨擘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康卡斯特.html" target="_blank">
  康卡斯特
 </a>
 ，周六在竞标收购英国广播公司天空集团的最后拍卖程序中，出价400亿美元，击败竞争对手、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默多克.html" target="_blank">
  默多克
 </a>
 旗下的21世纪
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福克斯.html" target="_blank">
  福克斯
 </a>
 公司。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康卡斯特.html" target="_blank">
  康卡斯特
 </a>
 （Comcast）和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福克斯.html" target="_blank">
  福克斯
 </a>
 公司的竞标从周五（9月21日）开始，经过了3个回合。
 <br/>
 <br/>
 康卡斯特周六（9月22日）的最终报价迅速赢得天空集团（Sky）独立董事的支持。他们敦促股东们接受这个报价。
 <br/>
 <br/>
 天空集团是欧洲最大的付费电视运营商，在7个国家拥有2300万用户，播放包括英超联赛和〝权力游戏〞在内的热门节目。
 <br/>
 <br/>
 康卡斯特旗下目前拥有环球影业（Universal Pictures），这项收购将使康卡斯特成为拥有大约5200万用户的全球最大付费电视运营商。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默多克.html" target="_blank">
  默多克
 </a>
 （Rupert Murdoch）目前持有天空集团39%的股份。康卡斯特的报价击碎了默多克长期以来追求收购天空集团的梦想。
 <br/>
 <br/>
 默多克8年前第一次竞标天空集团因电话窃听丑闻而失败。
 <br/>
 <br/>
 这项交易对迪斯尼集团也是一项挫败。此前迪斯尼同意收购福克斯电影和电视的大部分资产，包括其现有的天空集团股份。但如今根据康卡斯特的价格，迪斯尼则需要更多付钱。
 <br/>
 <br/>
 对于这次天空集团的被收购，股东们必须在10月11日前，做出决定。
 <br/>
 <br/>
 新唐人记者徐哲、天羽综合报导
</div>
