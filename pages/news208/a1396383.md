### 10月22日财经速瞄
------------------------

 【新唐人2018年10月22日讯】下面请看今天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财经速瞄.html" target="_blank">
  财经速瞄
 </a>
 <br/>
 <br/>
 周一，美国股市涨跌互见。金融股集体遭遇重挫，拖累标普500指数下挫11.90点，道指下跌126.93点。其中，美国银行收跌3.32%，花旗收跌3.30%，摩根士丹利收跌2.82%。与此同时，科技股表现亮眼，纳指收涨0.26%。
 <br/>
 <br/>
 智利法院将在未来一周就中国天齐锂业股份有限公司收购智利最大锂生产商SQM 24%股权的诉讼作出裁定。批评人士认为这个交易可能会催生锂行业垄断。锂是电动汽车和智能手机所用电池的关键元素。
 <br/>
 <br/>
 自今年7月中国从美国进口的汽车关税从25%提高到45%以来，特斯拉汽车在中国的销量大跌。据JL Warren Capital数据，进口到中国的特斯拉汽车数量从今年上半年的每月约1500辆下降到7月和8月的月平均260辆左右。目前，特斯拉汽车仅在美国生产。一辆特斯拉汽车在中国的售价比在美国高出约60%。
