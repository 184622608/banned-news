 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">

 再度全线大跌
 <br/>
 中国股市在周一短暂回升后，周二再次下挫，A股全线大跌，三大指数都重挫超2%，上证综指收盘报2,595.08点，跌幅2.26%；深证成指收盘报7,574.99点，跌幅2.24%。专家认为，官员喊话救市只能暂时缓解，中国经济内忧外患，投资人失去信心。

 2.

  默克尔
 </a>
 向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 让步 对美开放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天然气市场.html" target="_blank">
  天然气市场
 </a>
 <br/>
 德国总理默克尔已表示，支持向美国开放德国天然气市场的计划。此举意味着默克尔向川普做出了一个关键让步。此前，川普大力游说欧洲采购美国液化天然气，以降低俄罗斯对欧洲能源市场的影响力。德国也想藉此机会，解决长期以来与美国的贸易摩擦。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特斯拉.html" target="_blank">
  特斯拉
 </a>
 决定提前发布财报 逆市大涨13%
 <br/>
 全球电动车巨头特斯拉将原定于11月7日发布的2018年三季度财报，提前到本周三（10月24日）美股盘后公布，被市场认为是利好消息。投资机构香橼研究改变过去五年做空特斯拉的策略，发布报告指特斯拉占据电动汽车行业主导地位，没有竞争对手，因此转而做多，令特斯拉周二股价逆市大涨近13％，创10月3日以来最高。
