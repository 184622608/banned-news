### 妄称特斯拉下市惹祸 马斯克遭SEC起诉
------------------------

<div class="wysiwyg">
 【新唐人2018年09月28日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特斯拉.html" target="_blank">
  特斯拉
 </a>
 执行长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马斯克.html" target="_blank">
  马斯克
 </a>
 8月份曾表示〝正考虑将特斯拉私有化〞。如今，马斯克可能会为这一句推文付出代价。
 <br/>
 <br/>
 本周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国证券交易委员会.html" target="_blank">
  美国证券交易委员会
 </a>
 向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马斯克.html" target="_blank">
  马斯克
 </a>
 提起诉讼，指控他发布〝虚假和有误导性〞的声明，也没有向监管机构发出通知。委员会并要求法院禁止马斯克担任任何上市公司的高层及董事。
 <br/>
 <br/>
 这项诉讼是向纽约曼哈顿联邦法院提起的。
 <br/>
 <br/>
 8月7日，马斯克发表推文，说自己可能以每股420美元将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特斯拉.html" target="_blank">
  特斯拉
 </a>
 下市，进行私有化，并且〝资金已有保障〞。他的推特有超过2200万的粉丝。推文一出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特斯拉股价.html" target="_blank">
  特斯拉股价
 </a>
 飙升。
 <br/>
 <br/>
 美国证券交易委员会表示，没有任何金主承诺出钱购买特斯拉的股份，更没有提出过每股420美元的条件。
 <br/>
 <br/>
 美国证券交易委员会执法部门主任Steven Peikin：〝我们声明，马斯克给出每股420美元的价格，是假设特斯拉当时现有股价溢价20％，然后四舍五入到420美元。因为这个数字（大麻日）在大麻圈子中很重要，而且他相信他的女朋友会喜欢这个数字。〞
 <br/>
 <br/>
 如果罪名成立，马斯克就将支付罚款，及被禁止在美国任何一家上市公司担任董事，包括特斯拉。
 <br/>
 <br/>
 马斯克在特斯拉发表的声明中称，美国证券交易委员会的行动不合理。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
