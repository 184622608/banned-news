### 11月11日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 。
 <br/>
 <br/>
 面对年末销售旺季，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 撇弃前嫌，合作提振节日销售。之前，亚马逊只销售有限的苹果产品。而今年的假日销售期间，亚马逊将在全球多个国家销售最新的iPhone，iPad等产品，同时也说明亚马逊作为销售渠道，受到了大品牌的承认。
 <br/>
 <br/>
 德国马牌集团表示，第3季的营收较去年增长0.9%，达到107.878亿欧元。马牌执行长表示，由于第3季欧洲和中国的轿车、轻型商用车产量分别减少5%和6%，导致全球汽车产量出现10年以来第一次大幅下滑。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 能源大臣法力赫表示，沙特并没有解散OPEC的打算，而且认为OPEC还能起到石油领域的主导作用很久。之前，有消息人士透露，沙特政府资助的顶级智库正在研究解散OPEC对石油市场的影响。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 石油公司曾经宣布，向卡塔尔主权财富基金和大宗商品贸易商嘉能可出售19.5%的股权，来换取约115.7亿美元的融资。最近，有五名消息人士透露，俄罗斯国有外贸银行支付了60亿美元给卡塔尔主权基金来完成交易。
 <br/>
 <br/>
 新唐人记者保罗、正治综合报导
