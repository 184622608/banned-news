### 美股暴跌牵动全球 或是正常周期性回调
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】美股遭遇黑色星期三,经历几个月来最大跌幅。美股地震冲击欧亚股市,周三收盘急跌后,周四早盘欧洲股市一度跌至20个月来最低点。亚洲市场也跌至19个月低点。
 <br/>
 <br/>
 周三,
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_道琼斯指数.html" target="_blank">
  道琼斯指数
 </a>
 大跌超过800点，跌幅超3.15%；标普500指数收跌3.29%，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技股.html" target="_blank">
  科技股
 </a>
 占很高比重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳斯达克.html" target="_blank">
  纳斯达克
 </a>
 综合指数更下跌了约4%。
 <br/>
 <br/>
 美股地震波及欧洲股市,继周三收盘急跌之后,周四继续下跌。欧洲斯托克600指数早盘一度下挫1.2%, 触及20个月来最低位。英国富时100指数下跌1.6％。德国达克斯指数早盘低开1.28％。
 <br/>
 <br/>
 亚洲股市也是一片狼藉。周四,日经225指数，香港恒生指数和MSCI明晟亚太指数跌幅都在3.5%到4%之间。中国上证综指下跌5.22%，为2014年11月以来新低。深证成指下跌6.07%。
 <br/>
 <br/>
 不过，对于美股暴跌，美国总统川普周三说：这是好事，是我们已经等了很久的股市调整。此前也有专家指出：美股涨势惊人，有偶尔回调才健康。
 <br/>
 <br/>
 但是，川普总统对美联储加息政策不满。
 <br/>
 <br/>
 美联储于9月26日将基准利率调升了25个基点，利率达2%到2.25%。这导致美债收益率上升,吸引股市投资者转投入债市,造成股价下跌。
 <br/>
 <br/>
 投资人担心美联储可能因为通货膨胀而加快升息的步伐,导致股市再次受创。
 <br/>
 <br/>
 但周四商务部公布的9月消费者价格指数,让这一担忧有所缓解。美国9月价格仅上涨0.1％，低于预期的0.2%。
 <br/>
 <br/>
 市场对这一消息反应正面。道指和标普500指数周四开盘下跌幅度均低于预期,分别为0.31％和0.32％。
 <br/>
 <br/>
 此外,市场认为,国际贸易关系紧张以及对全球经济增长的担忧也是导致美股暴跌的原因。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
