### 11月28日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】下面请看一组环
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_球财经简讯.html" target="_blank">
  球财经简讯
 </a>
 <br/>
 <br/>
 1.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通用汽车.html" target="_blank">
  通用汽车
 </a>
 将在明年底之前裁减1万4千多名员工，并且关闭7家工厂，预计可以节省60亿美元成本。这是通用大规模重整的一环，未来重心将放在电动车和自驾车。
 <br/>
 <br/>
 2. 法国政府宣布，将在2022年之前关闭所有燃煤电厂，2035年之前降低核电比例到50%。法国计划改变生产电力方式，目标是在2030年之前，陆上风力发电和太阳能的产能分别提升3倍和5倍，还要发展
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海上风力发电.html" target="_blank">
  海上风力发电
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_水力发电.html" target="_blank">
  水力发电
 </a>
 。
 <br/>
 <br/>
 3. 东芝将和美国江森自控有限公司（Johnson Controls）合作生产车用锂电池。调查显示，2025年全球20%新车将采用结合锂电池和铅蓄电池的微混和动力系统。东芝希望以自家的锂电池结合江森公司的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_铅电池.html" target="_blank">
  铅电池
 </a>
 技术，抢攻市场。
 <br/>
 <br/>
 4. 韩国2018年吸引外商直接投资金额，11月27日达到230.4亿美元，创下历年新高。欧盟、美国和中国对韩国的投资出现增长。外商对制造业和服务业的投资保持增长态势。
 <br/>
 <br/>
 5.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宝马.html" target="_blank">
  宝马
 </a>
 汽车表示，考虑在美国设立第二家工厂，生产发动机和变速箱。宝马目前在美国南卡罗来纳州有一家组装厂。因为在美国销售增长，宝马考虑在本地生产发动机和变速箱。
 <br/>
 <br/>
 新唐人记者米西、明玉综合报导
</div>
