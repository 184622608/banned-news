### 9月26日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年09月27日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美联储（Fed）表示，美国今年经济成长率将达3.1%，比预期快速，而且失业率维持低点，通货膨胀接近2%的目标，预期经济还会继续温和扩张至少3年，因此调升利率一码，是今年第3度升息，符合市场预期，声明中也删除了〝宽松〞一词，显示未来政策将趋向中性。
 <br/>
 <br/>
 2.美国商务部公布8月独户新屋销售年率为62.9万户，环比增长3.5%、创下5月以来新高，同比上扬12.7%，从年增率来看，美国西部年率表现最好、年升19.1%。就月增率而言，美国东北部最旺、月升幅高达47.8%。
 <br/>
 <br/>
 3.由于阿根廷经济陷入衰退、货币不断贬值，国际货币基金组织(IMF)决定提高对阿根廷信贷额度，将从6月的500亿美元，增加到约570亿美元，IMF希望救助计画有助于恢复市场信心，同时保护大多数弱势群体。
 <br/>
 <br/>
 新唐人记者杰森综合报导
