### 11月21日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.由于意大利政府坚决拒绝修改赤字预算案，欧盟宣布正式拒绝意大利预算案，并对意大利启动过度赤字调控程序(Excessive Deficit Procedure，EDP)，意味意大利将要面临欧盟的制裁，也可能引发意大利退出欧元区的风险。
 <br/>
 <br/>
 2.高盛报告预测，未来几周内国际油价将继续维持高度波动状态。油价在最近的大跌之后，又因为美国原油库存下降和印度原油进口增加，而有所恢复。但高盛认为，现在油价上涨缺乏基本层面的推动力。
 <br/>
 <br/>
 3.路透社经济专栏文章指出，中国钢铁产量达到创纪录的高位，但中国经济放缓，加上美中贸易纠纷，导致钢铁价格大幅下跌。中国上海的螺纹钢期货价格在8月底到达高位，但在本周下跌18.8%。发文专家认为，中国钢铁将进入熊市。
 <br/>
 <br/>
 4.瑞典家俱公司宜家准备开始公司历史上最大规模的重组，被裁员人数约为7500人，消费模式变化对零售业造成的冲击，宜家也大受影响。为了应对这种变化，宜家准备在全球增开30家小型门市，并增加投资物流和网络销售。
 <br/>
 <br/>
 新唐人记者保罗、正治综合报导
