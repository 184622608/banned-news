### 亚股全线下挫 日本香港股暴跌800点
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】周四，亚洲股市全线下挫。此前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 遭遇八个月来最大单日跌幅，可能威胁到全球企业信心和投资。
 <br/>
 <br/>
 受周三
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 暴跌的恐慌，亚洲股市周四全线下挫。
 <br/>
 <br/>
 日本股市，周四一早暴跌816点，写下今年2月以来盘中最大跌幅；而香港恒生指数
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下跌.html" target="_blank">
  下跌
 </a>
 超过800点，A股低开3%，韩国指数则下跌2.64%。
 <br/>
 <br/>
 MSCI明晟亚太地区（除日本外）指数大跌3.9%，至2017年3月以来低点。
 <br/>
 <br/>
 日本内阁官房长官菅义伟(Yoshihide Suga)：我们看到良性经济周期正在稳步改善，我们不仅关注全球市场，也关注世界和日本经济，并希望做好一切准备。
 <br/>
 <br/>
 上海沪综指数SSEC也下跌4.3%，触及2014年底以来低点。沪深指数CSI300指数跌4%。
 <br/>
 <br/>
 全球市场大跳水，导致市场财富缩水数千亿美元。
 <br/>
 <br/>
 新唐人记者陈依心、李静明综合报导
</div>
