### 降税传闻激励特斯拉 陆能甩20年最惨车市？
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，暂时停火90天。美方暂缓将对中国商品关税提升到25%，中共则同意采购更多美国农产品，并且〝降低并移除〞对美国汽车的关税。消息一出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特斯拉.html" target="_blank">
  特斯拉
 </a>
 3日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股价.html" target="_blank">
  股价
 </a>
 大涨，也带动特斯拉概念股走扬。
 <br/>
 <br/>
 白宫经济顾问 库德洛:〝中国国家主席习进平，和副总理刘鹤告诉我们，中方会马上兑现承诺。〞
 <br/>
 <br/>
 白宫经济顾问库德洛，3日接受电视台访问时表示，应该会立即看到中方明显的改变，降低关税及非关税壁垒，在技术移转和智慧财产权保护方面也将取得进展。
 <br/>
 <br/>
 白宫经济顾问 库德洛:〝如果中共兑现承诺开放市场，他们得赶快开放，依据承诺。美国将大幅增加出口，无论是农业或工业领域，原因很简单，因为美国是当今世界上最具竞争力的经济体。〞
 <br/>
 <br/>
 美国总统川普3日在推特上表示，中国同意降低、移除美国进口车关税；目前的关税是40%，这对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特斯拉.html" target="_blank">
  特斯拉
 </a>
 、BMW、宾士等车厂无疑是重大利多。
 <br/>
 <br/>
 美国财政部长 梅努钦:〝当前的重点，是减少汽车关税，在90天内我们有很多要努力。〞
 <br/>
 <br/>
 受Model 3生产速度提高激励，特斯拉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股价.html" target="_blank">
  股价
 </a>
 3日收涨2.3%，台股特斯拉概念股也全数走扬。尽管将降低美国进口车关税，不过中国车市仍处在衰退期。今年10月，中国汽车经销商库存预警指数创三年来新高，今年车市降幅是20年以来最为严重。
 <br/>
 <br/>
