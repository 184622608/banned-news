### 10月10日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美国政府宣布，将对外国投资关键科技产业加强监管，允许美国外来投资审查委员会，对于在美国企业的任何投资进行审查，不局限在并购案，以保护美国的关键技术，不过需要更严格审查的产品仅占2.5%。
 <br/>
 <br/>
 2.美国30年期固定利率房贷的利率从4.96%上升到5.05%，比去年多出1%，为2011年2月以来新高。美国房贷银行协会专家认为，美国就业数据强劲推高房贷利率。上周美国房贷申请数量较前一周下滑1.7%，比去年同期少15%。
 <br/>
 <br/>
 3.存储芯片厂美光执行长表示，存储芯片市场结构已经改变，需求来自许多不同的应用，不再局限个人电脑市场，例如：数据中心、自动驾驶和互联网设备等，都需要存储芯片，由于需求来自多面向，供需失衡持续的时间，将较过去缩短。
 <br/>
 <br/>
 4.香港研究协会发布调查报告，有6成受访者对经济状况持负面立场，4成受访者预计房价将下跌。有专家表示，如果贸易战再升级，香港房价在2年内将下跌40%。香港房价曾连续8年名列世界第一。
 <br/>
 <br/>
 新唐人记者詹姆士、正治综合报导
