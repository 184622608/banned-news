### 12月5日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.Fed褐皮书指出，全美12个地区Fed多数认为经济持续适中成长，劳动市场全面紧绷，工资出现加速成长，企业也提高非工资方面的福利，此外，消费者支出保持稳定，旅游业维持速度，关税没有对制造业造成破坏。
 <br/>
 <br/>
 2.过去一段时间，中国消费市场是全球消费类公司眼中的宠儿，但是今年以来，中国市场的风险上升，跨国公司在印度市场找到拓张业务的路径，促使并购活动大幅增加，今年印度吸收的海外资金多达380亿美元，已经高于中国。
 <br/>
 <br/>
 3.由于英国首相特雷莎・梅(（Theresa May)的退欧计划出现变数，已经对英国短期经济造成影响，11月份采购经理人指数从10月份的52.2降到51.0，创下5年多以来最低，其中，服务业增长已经停止，制造业与建筑业呈现增长趋缓。
 <br/>
 <br/>
 4.近期油价下跌，美国股市投资者担心这会对企业利润造成压力，不少分析师认为油价下挫体现了全球增长疲软，而投资者们则开始思考这一轮的经济扩张何时会结束。本周，OPEC和非OPEC产油国在减产上达成一致，正在讨论的是具体减产的数量。
 <br/>
 <br/>
 <br/>
 <br/>
 新唐人记者保罗、正治综合报导
