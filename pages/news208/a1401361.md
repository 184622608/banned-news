### 11月29日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】下面是今天的财经速瞄
 <br/>
 <br/>
 据统计，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公募基金.html" target="_blank">
  公募基金
 </a>
 今年已有577只清盘。其中，规模不足5000万人民币的基金成为清盘基金的〝重灾区〞。这一数据是过去19年里246只公募基金清盘数的2倍多。将有更多基金选择清盘。
 <br/>
 <br/>
 周四，韩国最高法院裁定，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本三菱.html" target="_blank">
  日本三菱
 </a>
 公司必须赔偿28名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国人.html" target="_blank">
  韩国人
 </a>
 在第二次世界大战期间的强迫劳动。三菱须向23名原告赔偿每人8000万韩元。赔偿另外五名原告或其家属1.5亿韩元。但日方不认同裁决。
 <br/>
 <br/>
 周四，德国联邦就业局公布11月失业率4.8％，约200万人，创历史新低。数据较10月减少约18,000人，比去年同期少了182,000人。但有经济学家表示，有迹象显示德国经济会降温，因为英国即将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧.html" target="_blank">
  脱欧
 </a>
 和贸易冲突导致不确定性。
