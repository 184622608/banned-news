### 美国网购星期一 预计交易量78亿破纪录
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】今天是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网购星期一.html" target="_blank">
  网购星期一
 </a>
 ，研究机构预计，单日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交易量.html" target="_blank">
  交易量
 </a>
 将可达到78亿美元，创下史上新高，而且比去年增长了将近18%。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网购星期一.html" target="_blank">
  网购星期一
 </a>
 〞已经不再是〝黑色星期五〞的附属品。
 <br/>
 <br/>
 根据研究机构Adobe Analytics预估，今年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 网络购物星期一销售额，将写下78亿美元的历史新高，比黑色星期五的62亿美元网购销售额还要多。
 <br/>
 <br/>
 美国网络零售龙头亚马逊、沃尔玛，以及梅西百货等其他网上销售，都提供不同的线上优惠以吸引买家。
 <br/>
 <br/>
 一些购物专家表示，今年最优惠价格将会出现在〝网购星期一〞，而不是人们认为的〝黑色星期五〞，商家会在网购星期一提供比黑色星期五更好的电子产品。
 <br/>
 <br/>
 梅西百货看准了〝网购星期一〞的商机，不仅提前一天从星期天开打，还延长到星期三，变成〝网购周〞。
 <br/>
 <br/>
 新唐人记者李梅、李静明综合报导
</div>
