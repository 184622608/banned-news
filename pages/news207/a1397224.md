### 原中纪委官员致国家主席习近平的一封信
------------------------

<div class="wysiwyg">
 习近平主席：
 <br/>
 <br/>
 您好！
 <br/>
 <br/>
 在您的第一个任期内，您曾经说过：〝头上三尺有神明〞。正因为您有这一念，神助您打掉了440多只大老虎。这是以您个人的能力绝对做不到的。
 <br/>
 <br/>
 在您的第二个任期开始后，您忘记了〝头上三尺有神明〞的古训，跑到上海中共一大遗址，举着拳头〝宣誓〞，为不信神不敬神的马克思宣扬的共产主义奋斗终身。结果，一步错，步步错，一直错到今天。
 <br/>
 <br/>
 人类社会的历史，无论东方，还是西方，都是从神话开始的。到了21世纪的今天，无论东方，还是西方，人类社会又经历了许许多多震憾人心的神话故事，这些神话故事将整个人类的历史串起来，最后汇集成一句经典的话：〝我们不崇拜政府，我们崇拜神。〞
 <br/>
 <br/>
 20世纪美国著名预言家Jeane Dixon（1904～1997），曾准确预言了肯尼迪总统遇刺、艾森豪总统连任、印巴分治、赫鲁雪夫下台等重大历史事件。她早就预言：〝1999年，人类不会灭亡，拯救人类的希望在东方〞。
 <br/>
 <br/>
 1992年5月13日，是一个值得全世界纪念的伟大日子。这一天，在东方，在中国东北的长春市，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 ，又称法轮大法，开始洪传于世。此后短短7年间，法轮功因其净化身体、净化心灵的奇效，迅速传遍全中国，传到全世界，学炼法轮功的人数达到上亿人，远远超过已有76年历史的中共党员的人数，引起当时的中共独裁者江泽民的极大恐慌与忌恨。
 <br/>
 <br/>
 从1999年7月20日起，江泽民动用全部国家机器，以古今中外最邪恶的流氓手段，发动对法轮功的疯狂大迫害。如今19年过去了，法轮功不仅没有被迫害倒，相反，洪传到了全世界114个国家和地区。法轮功的经典著作《转法轮》已被翻译成40多种外文，在中国大陆以外公开出版发行，成为中华五千年文明史上被翻译成外文最多的中文经典。法轮功创始人李洪志先生，因为对人类身心健康的卓越贡献，受到超越党派、国界、种族、语言、文化背景的社会各阶层人士发自内心的敬仰与尊崇。在中国大陆以外，中共对法轮功的迫害已彻底失败，在中国大陆，离开高压和欺骗，这场迫害一天也维持不下去了！
 <br/>
 <br/>
 全世界的法轮功学员经历了无数的神迹，许许多多法轮功修炼者的修炼故事，就是一个又一个感天动地的神话。全世界法轮功真修弟子用他们的修炼实践证明：第一，法轮大法于国于民有百利而无一害；第二，法轮大法是千年不遇、万年不遇的正道大法。
 <br/>
 <br/>
 江泽民利用中共迫害法轮功19年多，一个最重要的结果是，把中共变成了全世界最腐败的党。中共的腐败已达到了人类有史以来登峰造极的地步。中共的腐败，如果用疾病来形容的话，已经是晚期癌症的晚期了，无药可治。一个苹果烂透了，作为人来说，肯定就把它扔掉了；一个政党烂透了，作为神来讲，肯定就把它销毁了。当年苏共是如此，东欧各国共产党是如此，今天的中共也是如此。〝天灭中共〞，不是一个口号，而是一幕我们都在亲历的历史大戏。
 <br/>
 <br/>
 如果不是神慈悲救人，中共早就被销毁了。但中国人是可贵的，包括那些加入中共的人。李洪志师尊在《洪吟二 济世》中写道：〝解开心锁 不信良知唤不回〞。〝不信良知唤不回〞，这是我年复一年，一次又一次给您写信的真正动力所在。我写给您的信，讲的全是客观事实，您真得好好看一看，认真想一想了。
 <br/>
 <br/>
 中国自古以来就叫〝神州〞，中国传统文化被称为〝神传文化〞，中国历史上有许许多多的神话故事，其实，很多都是真实发生过的事情。21世纪的今天，法轮大法洪传于世，创造了数不清的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_现代神话.html" target="_blank">
  现代神话
 </a>
 。现将我亲自编辑的《法轮大法创造的现代神话》附录于此，请您找人下载后给您仔细看一看。
 <br/>
 <br/>
 中国大陆有许多受冤的访民，不断上访，要求中共平反。法轮功被迫害19年多，却从没要求中共平反。为什么？法轮功是博大精深的伟大的佛法，中共有什么资格为法轮功平反？法轮功洪传于世，是来救人的，包括您和您的家人在内。
 <br/>
 <br/>
 请您静下心来，认真看一看《法轮大法创造的现代神话》，重新升起对神的信仰和敬畏，那么，您就有可能像美国总统川普那样，逢凶化吉，遇难呈祥。
 <br/>
 <br/>
 忠言逆耳，只是为您好，为普天下炎黄子孙好！
 <br/>
 <br/>
 王友群
 <br/>
 <br/>
 2018年10月25日于美国纽约曼哈顿
