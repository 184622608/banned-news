### 袁斌：美国康大中断与中国人大合作是受误导？
------------------------

<div class="wysiwyg">
 10月20日，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康奈尔大学.html" target="_blank">
  康奈尔大学
 </a>
 （简称康大）正式通知
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人民大学.html" target="_blank">
  中国人民大学
 </a>
 （简称人大）暂停双方合作计划，理由是该校缺乏学术自由。据悉，这是近年来国外大学因政治原因与中国大学中断合作的首例。
 <br/>
 <br/>
 然而，大陆《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球时报.html" target="_blank">
  环球时报
 </a>
 》事后却发表文章言之凿凿地称：〝据向人民大学了解，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康奈尔大学.html" target="_blank">
  康奈尔大学
 </a>
 对人大的具体指控子虚乌有，显然受了网上传闻的误导〞 ，〝独立的康奈尔大学跟了白宫战略的风〞 。
 <br/>
 <br/>
 那么事情的真相究竟是怎么回事呢？康大中断与人大的交流合作项目，到底是因为人大缺乏学术自由还是因为这家大学〝受了网上传闻的误导〞，〝跟了白宫战略的风〞？
 <br/>
 <br/>
 事情还得先从不久前大陆大学生声援深圳工人维权一事说起。
 <br/>
 <br/>
 今年夏天，深圳佳士科技公司的部分工人因为试图组织工会维权，遭到资方和政府的打压。全国各地一些富有正义感的大学生闻讯后纷纷赶到深圳声援支持维权工人。结果，当地警察不但拘捕了大约30名工人，还带走了40名支持工人的学生，其中就有人大大三学生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨舒涵.html" target="_blank">
  杨舒涵
 </a>
 。事后，杨舒涵公开发表声明说，因为她支持工人，人大强迫她停学一年。
 <br/>
 <br/>
 10月初，大陆微信圈中更流传人大经济学院2016级学生（大三）向俊伟发出的微信文章，曝光人大经济学院老师手中的一份黑名单。文章指，经济学院把12名关注社会、关心底层的学生列为重点〝作战对象〞，安排特定老师做思想工作并定期汇报，甚至是安排其他同学定期汇报他们的日常活动。而位列名单第一的大四学生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈可欣.html" target="_blank">
  陈可欣
 </a>
 甚至被强行绑架回家，不让她走出家门一步，一度还被逼到绝食住院。
 <br/>
 <br/>
 日前，康奈尔大学工业和劳工关系学院国际专案主任弗莱德曼在推特上披露了该校10月20日正式通知人大暂停合作的电子邮件截图。
 <br/>
 <br/>
 弗莱德曼在信中写道：〝就人大校方声明中所说‘个别学生受到批评是因为在学校组织的活动中违反了纪律’，我很抱歉这显示我们两个机构在此问题上存在的巨大认知差异。〞
 <br/>
 <br/>
 他表示，到目前为止，他没有看到人大学生有任何违反校规的行为，但看到的是人民大学采取的极端举措，包括监视、向学生家庭施压，甚至要求将学生拘禁在家以阻止他们继续为劳工问题发声。
 <br/>
 <br/>
 他在信中强调没有看到有学生做错的证据，同时，他说，他理解共产党已下令不让这些学生为支持工人的活动发声，所以这是超出人大决策范围之内的问题。
 <br/>
 <br/>
 〝但如果人民大学作为一个机构，你愿意不论任何缘由就执行这类政策，那么我们康奈尔大学会视此为一个大的违反学术自由的举措。〞弗莱德曼写道，〝我完全了解美中政治环境的差异，但遵循核心原则是我们进行校方合作不可妥协的内容。〞
 <br/>
 <br/>
 他表示，鉴于此情况，在康奈尔国际事务副教务长、工业和劳工关系学院院长以及学院国际专案委员会的支持下，宣布暂停所有康奈尔大学工业和劳工关系学院与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人民大学.html" target="_blank">
  中国人民大学
 </a>
 劳动关系学院以及商学院之间的互换学生计划。
 <br/>
 <br/>
 弗莱德曼补充说，如果人大的学术自由环境能有所改善，当然这是他非常期望的情况，届时康奈尔大学将对重启交换计划持开放态度。
 <br/>
 <br/>
 可见，一旦我们了解了整个事情的来龙去脉就不难明白，康大对人大的指责压根就不像《环球时报》声称的那样，是什么〝子虚乌有〞，而是事实确凿；而康大之所以中断与人大的合作，压根也不像《环球时报》声称的那样，是什么因为〝 受了网上传闻的误导〞，〝跟了白宫战略的风〞，而是因为人大缺乏学术自由，肆意践踏学生的基本人权。换句话说，《环球时报》的说法纯属是在混淆视听造谣惑众！
 <br/>
 <br/>
 不过，《环球时报》有一点倒是没说错——〝或许康奈尔大学将不是中断与中国大学学术交流的唯一美国大学，如果不久后有仿效者出现，我们不该感到意外。〞
 <br/>
 <br/>
 作者提供 本文只代表作者的观点和陈述。
 <br/>
 <br/>
