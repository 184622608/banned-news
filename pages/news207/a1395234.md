### 陈思敏：北京释放整顿公安系统的信号
------------------------

<div class="wysiwyg">
 近来的一些迹象显示，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安系统.html" target="_blank">
  公安系统
 </a>
 将再次面临整顿风暴，力度不亚于先前。
 <br/>
 <br/>
 而有关于此的明显信号，目前至少有如下三个：一、中共公安部副部长孟宏伟被捕。二、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 接连公开举报上海、北京两个地区的公安机关。三、中共中央政法委机关报《法制日报》发文为崔永元举报〝背书〞。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 逃税事件落定之后，崔永元继续举报，这次直指公安警方：7日举报上海公安涉嫌违法，10日举报北京朝阳分局新源里派出所不作为，11日再度质疑上海公安局经侦大队涉嫌腐败，而且直接点出了当事人的名字。
 <br/>
 <br/>
 崔永元举报北京朝阳分局新源里派出所不作为，主要是自从他揭露娱乐圈明星阴阳合同、天价报酬、逃税偷税等一系列问题后，一直遭到多方恐吓、威胁，但向朝阳分局报案后，却始终遭到〝请示上级〞的敷衍。
 <br/>
 <br/>
 崔永元两度举报上海警方，也与娱乐圈一位大金主有关，即影视大亨施建祥，他也是上海〝快鹿系〞掌门人。而上海公安原长宁经侦副支队长彭奋的儿子彭明达，既涉及施建祥出资的相关影视公司，也涉及快鹿集团（东虹桥担保百亿违约）的集资诈骗案。
 <br/>
 <br/>
 据称，彭明达已潜逃国外，摇身一变成了新西兰中文电视台总裁台长。而施建祥2014年年初前后即被举报，2016年3月7日外逃至美国，因而被中纪委列为红通犯。
 <br/>
 <br/>
 崔在7日首次公开举报上海公安，当天也是孟宏伟落马日，时间上不无巧合。而崔举报上海、北京两地公安的内容，在某种程度上也让看起来独立的事件显得有所牵连。
 <br/>
 <br/>
 此外，崔永元对上海和北京两地警方的举报，涉及公安机关不作为的恶劣事实，同时还暗指警方在违法事件中扮演从犯共犯的角色。如快鹿集团非法集资案，曾经被投资人举报至时任上海书记韩正，该案由上海公安局长宁分局经侦支队立案，但后无下文，还让关键人物施建祥逃至境外。
 <br/>
 <br/>
 上海公安近来被曝光的丑闻不只一桩，如10月1日《华夏时报》报导证监会处长李志玲的二审案情：〝李某1〞的证言显示，2013年12月，李志玲给其打电话，说她老公被上海市徐汇区公安抓了；2014年1月，李志玲发短信说，给原告500万元和解，警方就同意保释。〝李某1〞汇了500万元给李志玲，后来李志玲老公被警方释放了。
 <br/>
 <br/>
 这篇报导虽然是在披露李志玲受贿细节，但值得注意的是，法院裁判文书有凭有据，那么上海公安谁收了这500万？
 <br/>
 <br/>
 总而言之，崔永元目前对于公安警方这种强度的公开举报，是不寻常的，而且针对的是全国政治高地的上海、北京。
 <br/>
 <br/>
 崔永元背后是否有来自高层的授意，暂不得而知，但央级机关报《法制日报》已经刊文表态：〝有人捅破这层纸就该深查下去〞。接下来就看北京和上海警方如何对待崔永元的实名公开举报。
 <br/>
 <br/>
 在以〝扫黑除恶〞清洗中共公安政法系统〝黑伞〞的现阶段，仍要通过名人举报来触动一些基层和深层的问题，这也从一个侧面说明北京当局在这个系统还有一直想解决而没能解决的问题。
 <br/>
 <br/>
 ──转自《大纪元》本文只代表作者的观点和陈述。
 <br/>
 <br/>
