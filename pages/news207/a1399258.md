### 川人：政审重来，你还指望中共会开放？
------------------------

<div class="wysiwyg">
 11月9日，中共福建教育考试院公布规定，将对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高考.html" target="_blank">
  高考
 </a>
 考生进行思想政治品德考核，对考生的政治态度和道德品德作出全面审查，规定指出〝有反对宪法所确定的基本原则的言行或参加X教组织，情节严重的〞不得高考。这是继重庆之后，又一个省份出台措施按照中共的政治标准对考生进行政治审查（简称：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政审.html" target="_blank">
  政审
 </a>
 ），政审不合格者将被中共剥夺接受高等教育的权利。消息一出，引发舆论哗然，很多人担心政审重来将是中共第二次文革的开始。
 <br/>
 <br/>
 所谓的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政审.html" target="_blank">
  政审
 </a>
 〞，是中共政治审查制度的简称，它曾给中国人带来了无尽的伤痛。这种审查制度是建立在〝中共与敌人〞二元对立思维之上的产物，起初是中共为〝纯洁革命队伍〞而设计，后来中共靠着这种自我封闭的审查制度取得了成功，窃取了中国政权。所以，1949年后，中共把这种政治审查扩大至社会的每一个领域，以强化中共对每个中国人的绝对控制。如果我们深究下去，就会发现：中共的政审制度其实就是一种赤裸裸的〝敌对论〞、〝血统论〞和〝诛连论〞，它本身就是一种
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反人类.html" target="_blank">
  反人类
 </a>
 普世价值的邪恶制度，是一种极度封闭、极度不自信的表现。
 <br/>
 <br/>
 正是封闭与不自信，上世纪50至70年代，中共在中国实行了严厉的政审制度，并将其与大学录取进行挂钩。据了解，文革前，在中国报考大学素有〝先看政审结论，后看考试分数〞的做法，那些出身地主富农家庭或家庭成员有〝右派分子〞的子女，都会面临
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高考.html" target="_blank">
  高考
 </a>
 被降格或被拒绝录取的悲惨遭遇。而在文革中，中国大学实行〝推荐制〞招生，〝家庭成分〞几乎成为中国大学录取的唯一指标。直到1977年8月，邓小平主持召开科学和教育工作座谈会，决定恢复高考，并对高考实行政审的做法连说了三个〝繁琐〞，为高考政审制度松绑，此举得到了当时绝大多数中国人的支持，为邓本人赢得了各种光环。实践证明，中共搞政审那一套反人类的封闭做法是不得人心的。
 <br/>
 <br/>
 与中共〝政审重来〞形成鲜明对比的是中共对外宣传的〝更加开放〞。11月5日，中共在首届中共国际进口博览会上对外承诺，中共将降低关税，扩大市场准入，进口更多的外国商品，加强对智慧财产权保护等。中共官媒称〝中共的大门不会关闭，只会越开越大〞。可当天晚，中国欧盟商会发表声明称，中共的许多承诺只是重复过去的声明，欧洲商业界对这种承诺越来越不感兴趣。同样的，11月9日，美国白宫贸易高级顾问纳瓦罗表示，美国和许多国家签订贸易协定，都很信任他们能遵守、执行，但中共不同，和中共谈判即使有结果，也很难保证中共会执行。
 <br/>
 <br/>
 事实也的确如此，一边对外高呼自己将会〝更加开放〞，一边对内加紧推行〝政治审查〞，搞自我封闭，中共它到底想干什么？政审重来，你还指望中共会开放？其实，经常被中共欺骗的人都明白：〝好话说尽，坏事做绝〞是中共永远不变的本色。想当初，中共是靠着〝共产均贫富〞和〝人民当家作主〞的招牌起家的，90多年过去了，中共从原来的〝无产阶级〞变成了现在〝有产阶级〞，〝统治阶级〞变成了货真价实的〝土豪劣绅〞，但它们〝翻身作主〞之后，并没有兑现过它对中国人任何承诺。实践中，中共千方百计地保护着它们的利益和各种特权，却有意不保护中国人，相反广大中国人成为了中共邪党奴役的对象。
 <br/>
 <br/>
 近年来，那些替中国人鸣不平的维权律师正被中共一个个地判刑；那些替中国人坚守〝真善忍〞道德准则的好人正被中共一个个地活摘与绑架；那些靠着自己双手养活自己的中国人正被中共当成低端人口抛弃；那群仗义执言的中华儿女董瑶琼、华涌、岳昕、张盼成们正被中共一个个地消失……中共领导人曾表示〝要容得下尖锐批评〞，但事实证明在这只是空谈。现在中共党内人士批评中共决策层被视为〝妄议中央〞，有被〝双规〞的风险，而普通中国人发出批评中共的言论轻则遭删帖，重则被消失。这样一个越来越左的极端社会，谁会指望它能进一步开放？！中共加强对言论的管制与对政治的审查，这本身就是一种封闭。
 <br/>
 <br/>
 现在很多人觉得，中共又变得越来越〝左〞了。2015年，中共对外企发布管制措施，规定进入中共市场的外国公司必须与当地公司合资，而且必须分享它们的核心技术；2016年，中共在浙江一口气拆了近2000个教堂的十字架，并强化中共对全国宗教的管控，让宗教姓党，听党指挥；2017年10月，中共大规模地在外企强制设立党支部；2017年11月，中共在寒冷的冬天，连夜驱赶了一百多万的北京低端人口；2018年1月，中共开展了〝扫黑除恶〞政治运动，坚守〝真善忍〞准则的好人也被一并扫除；2018年3月，中共删掉了它宪法中的国家主席任期限制；2018年9月，中共迫于压力，将关押在新疆〝再教育营〞的数十万少数民族同胞，转往其它地区；2018年10月，中共军队医院全改名了，但活摘人体器官的罪恶仍在继续……这就是所谓〝更加开放〞的中共国，越〝开放〞越恐怖！
 <br/>
 <br/>
 与中共打了90多年交道的中国人，他们的亲身经历告诉世界，中共就是一个毁灭世界的魔鬼。当中共比你弱小、需要你説明时，它就会与你讲道德、讲法治、讲公义、讲规则、讲情怀，它会用各种手段取悦你，满足你；当中共与你实力差不多的时候，它就会采用各种卑鄙手段与你竞争，直到把你踩到脚下或消灭；当中共比你强大的时候，它就会无视道德、无视法治、无视公义、无视规则，利用各种手段控制你，控制你的思想，让你对它绝对服从，不敢有丝毫的反抗。在这过程中，中共一直依靠的法宝就是〝好话说尽，坏事做绝〞，它可以对需要〝统战〞的对象做出好听的承诺，并选择性、挤牙膏式地执行承诺，通过这种过程消耗你，拖垮你。这种套路，早被中共小弟朝鲜劳动党在去核谈判中玩得炉火纯青。
 <br/>
 <br/>
 40多年过去了，中共与美国的〝双边关系〞，正完全沿着上面的套路进行。1972年至2015年，中共靠着美国的长期扶持与最惠国待遇，一跃成为世界第二大经济体；2015年至今，具备了与美国正面竞争的中共，靠着渗透、收买政要、窃取技术等各种手段，开始与美国全面抗衡，全面竞争，中共的目的很简单，就是要干掉美国，最终按照中共的方式来治理全世界，中共称之为〝世界治理的中国方案〞。对这一点，美国总统川普的助理、执掌美国贸易和工业政策的经济学家彼得‧纳瓦罗看得特别清楚，川普也多次直言，中共的〝2025计划〞直接危险到了美国的国家安全。如果中共真能称霸全世界，那将是人类的浩劫，届时那些华尔街富豪也会被中共一一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肃整.html" target="_blank">
  肃整
 </a>
 。
 <br/>
 <br/>
 庆幸的是，现在重庆、福建均迫不及待地实施政审。这给世界上所有指望中共会开放的人泼了一瓢冷水。若中共真心想开放，它们搞那么多贸易壁垒干什么？它们搞那么市场准入许可干什么？它们搞那么多审查，甚至是政治审查干什么？真开放了，中共坚持的〝一党专政〞还能继续？中共的〝党领导一切〞还能维系？实践证明，那个〝好话说尽，坏事做绝〞的中共不就是一个自私自利，无恶不作的特殊利益集团。搞开放干什么？不就是为了缓解国际压力，进而保住自己的政权，好继续奴役中国人嘛，搞政审干什么？不也是为了保住自己的红色江山，进而称霸全世界嘛。所以，政审重来，中共的〝敌对论〞、〝血统论〞、〝诛连论〞又登场，你还指望中共会开放？！
 <br/>
