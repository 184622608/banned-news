### 谭耀南：美国期中选举 川普的逆袭
------------------------

<div class="wysiwyg">
 虽然不能全然说共和党赢得了这一次美国的期中选举，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 再一次跌破众多主流媒体的眼镜，打了场堪称成功的选战。开票还是进行中，不过显然共和党不但如预期的保住参议院多数党的地位，还逆势增加了3到5席，川普也成为二战后第一位支持度在五成以下、期中选举中执政党参议员席次却逆向增加的总统。执政党众议院席次虽如往例在期中选举中减少，失去了众院多数党的地位，但减少的席次也远低于预期。
 <br/>
 <br/>
 州长选举的部分，共和党的成绩也超过预期，在几个竞争激烈的州，包括俄亥俄州、爱荷华州、乔治亚州，以及动见观瞻的摇摆州佛罗里达州， 共和党的候选人纷纷以极些微的差距，赢得州长选举。虽然州长不涉及到美国联邦政府的执政权，但是共和党州长选得不错，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的执政也更稳固。
 <br/>
 <br/>
 笔者先前曾提到，这次美国期中选举分两种：大范围选区的选举和小范围选区的选举，前者是州长和参议员选举，后者是众议员选举。大范围选举共和党能维持局面，甚至胜出，小范围选举则在期中选举不利于联邦执政的共和党。
 <br/>
 <br/>
 参议员选举，我提过几个重点观察的选区：亚利桑那州、密苏里州、印第安那州、德州和佛罗里达州。选举结果，共和党在上述两党激烈对决的选区，全部获胜。参院选举可以说是共和党预期中最好的结果。
 <br/>
 <br/>
 几点初步的选后观察。
 <br/>
 <br/>
 第一，这次选举无疑是川普的信任／不信任投票。不只川普在选前自称他的名字〝形同在选票上〞，出口民调也显示川普是影响选民投票行为的主要因素。选举结果显示，他在选前重点辅选的各州，如佛罗里达、俄亥俄州、威斯康辛、亚利桑纳等州，共和党都在极为接近的情况下，大有斩获。
 <br/>
 <br/>
 可以说这一次川普是共和党这次选举结果超过预期的最主要原因。 选后共和党内的政治权力，将更向川普高度集中。川普迈向2020年连任之路也更稳固。
 <br/>
 <br/>
 第二，这一次的选举完全没有出现一些主流媒体原先预期的蓝色浪潮（blue wave），也就是认为多数选民不满川普，会在期中选举给川普的共和党一个教训。 相反的，共和党选出了在现有的政治结构下最好的结果。川普的支持度在四成五左右，57%的美国人认为国家往错误的方向走，但却有68%认为今年的经济情势比去年好。
 <br/>
 <br/>
 移民政策是共和党选民最关心的议题，医疗健保则是民主党选民最关心的议题，经济议题反而落到第三位。川普在选前成功炒热了共和党选民关心的议题，让他的支持者成为显性选民，也冲高了投票率，预期超过五成，创了期中选举投票率的近期新高。
 <br/>
 <br/>
 第三，美国在2018期中选举后，在政治上成为更加分裂、分歧的国家。反川普主义和川普的共和党在太多的议题上，会更加逆向而行。这点需专文另行论述。
 <br/>
 <br/>
 第四，和2016年美国总统大选一样，民调再次失准。选前民调结果小胜的民主党选举人，很多纷纷中箭落马。这点很难用所谓机构效应或隐（显）性选民这种帐面上的理由来解释。其中一个可能性，是民调机构在受访选民如敎育背景等的加权上出了差错，另外就是可能低估了投票率的参数。
 <br/>
 <br/>
 第五，虽然国际议题和外交政策向来不是期中选举的主轴，但川普在选后势必更加大力道地执行他的外交政策，包括与中国的全面竞争和对抗。希望看到川普在选后受国内议题及国会牵制而放缓对中国贸易及经济战的人，恐怕会大失所望了。
 <br/>
 <br/>
