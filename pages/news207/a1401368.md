### 文武：中共不抵抗策略中的阴谋
------------------------

<div class="wysiwyg">
 有媒体传出，中共针对当前中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 定下了不抵抗的策略。就是任由美国怎么打，中共就是不战也不降。
 <br/>
 <br/>
 如果
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 是在贸易领域的战争的话，那么对于西方人来说，要么是战，要么是降，可是不战也不降，西方人是难以理解的。
 <br/>
 <br/>
 这种不战不降的战略，东方人比较好理解，特别是中国人是很能理解的，就是当对方明摆着比自己强的时候，如果打肯定是失败，如果降又承担不起投降的后果，于是，在这种情况下，就选择一种不战不降的战略。
 <br/>
 <br/>
 这是东方人比较习惯的一种思维方式，例如，在日常生活中，当受到对方邀请某一天去他家做客的时候，西方人一般的回答是要么说去，要么说不去。可是中国人当出现为难的时候，往往不说去，也不说不去，而是说，到时再说。
 <br/>
 <br/>
 同样在双方争战的时候，除了战和降之外，中国人也有另外一种选择，那就是不战也不降。
 <br/>
 <br/>
 可是中共的不战不降，却与正常的中国人的思维是不同的，而是与流氓无赖的思维完全一致。
 <br/>
 一个流氓无赖，当与别人发生争执的时候，如果对方比自己弱，就向对下很手，甚至置人于死地；如果对方明显比自己强的时候，就会分清对方的为人，如果对方也是个流氓无赖，他就跪在地上向对方求饶，承认自己打不过，愿意投降认输；当发现对方是个君子的时候，他就把脖子伸过去，等着对方砍。
 <br/>
 <br/>
 因为流氓无赖这个时候下了个赌注，就是对方是个君子，不会真的杀了他。
 <br/>
 <br/>
 这一次中共与美国的贸易战，中共现在就是下了个赌注，就是美国是个君子，不会真的杀了中共，所以中共采取不战不降的伎俩。
 <br/>
 <br/>
 中共具体的做法就是，任由美国去要价，中共自己不出价，美国要什么价中共就给什么价，美国要进一步中共就退一步，要进二步中共就退二步。中共每退一次就提醒一次美国，要文明、要礼貌。
 <br/>
 <br/>
 这是什么意思呢？就是提醒美国，美国是个君子，是个民主自由的国家，不要置人于死地。
 <br/>
 <br/>
 因为中共非常清楚自己理亏，自己确实在占美国的便宜，自己确实在盗窃美国，现在被美国抓住了，如果打肯定是打不过，如果降自己就得完蛋。
 <br/>
 <br/>
 就是说，要打又打不赢，要降又不能降。
 <br/>
 <br/>
 在这种情况下，中共就下了这个赌注，就是赌美国是个正人君子，不会真的下狠手，会手下留情，会给中共留条生路，于是，中共就采取了今天这种不战也不降的伎俩。
 <br/>
 <br/>
 可是中共的不战不降绝不是通常意义上的不战不降，中共的不战不降只是一个阴谋，只是通过不战不降的伎俩，把美国所有的底线全暴露出来，把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统执政团队的弱点找到，一旦被中共找到了下手的地方，中共马上就会反戈一击，然后将美国执政团队陷于一种困境，然后中共再从中解套。
 <br/>
 <br/>
 可是美国已经不是原来的美国了，美国总统也不是原来的美国总统了，美国人民已经看清中共了，美国总统川普和执政团队也看清中共了，绝不会再给中共机会了，因为中共不是人，只是个魔鬼，怎么可能再给魔鬼机会呢？
 <br/>
 <br/>
 所以，这一次，川普说，〝我已经准备好了，我一生都是为了今天〞。
 <br/>
 <br/>
 那就是绝不放过中共。
 <br/>
 <br/>

 <br/>
