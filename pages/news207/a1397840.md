### 袁斌：摩的司机为何当街刺死运管所副所长？
------------------------

<div class="wysiwyg">
 10月31日，大陆湖南省怀化市发生一起血案！
 <br/>
 <br/>
 当天上午，该市洪江区道路
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_运管所.html" target="_blank">
  运管所
 </a>
 和交警等部门上路查摩的，就是一种可以搭客的摩托车的士。上午8时4分，副所长张仲凯等人查扣了罗某的摩的，双方为此发生冲突。僵持一段时间后，罗某离开，但过了一会，他又回来了，跟着查摩的人员。11时30分许，罗某突然持刀朝张仲凯背部连捅3刀，张即刻趴在地上，口吐鲜血，一动不动，当场死亡。罗某杀完人后，坐上一辆摩托车，逼着司机送他去公安局自首。
 <br/>
 <br/>
 看到这里，读者想必会问：罗某究竟为何要杀张仲凯？他俩到底有何恩怨？
 <br/>
 <br/>
 有消息称，之前罗某已被运管部门连抓了三次，每次都罚款1,000元，因而选择挺而走险，手刃运管队长。
 <br/>
 <br/>
 据怀化市市民黎先生对大纪元介绍，摩的一般就几千块钱，二手的千把块钱，成本比较低，起步门槛很低，一个月一两千块钱也就可以立足了。因此，当地有一些失地的农民工和进城务工的农民工靠开摩的糊口。跑一趟也就几块钱，一天挣不了多少钱。尽管当地不允许摩的载客，但当地的管理也是一阵风，管不过来。
 <br/>
 <br/>
 不难想见，对于一个靠开摩的为生，而且辛辛苦苦忙一天也挣不了多少钱的底层百姓来说，一连三次被查，而且每次都被罚1,000元，这可不是个小数字，很可能是他半个月乃至一个月的收入。而且更重要的是，官方禁摩的等于断了罗某的生路，其怨恨之情可想而知。现在车又再度遭扣，接下来又要被罚款，甚至连车都可能被没收，一怒之下积累在心底多日的仇恨一下就爆发了，杀人不正是这种仇恨导致的结果吗？可见，从本质上讲，罗某杀人属于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_官逼民反.html" target="_blank">
  官逼民反
 </a>
 。
 <br/>
 <br/>
 共产党头子曾说过哪里有压迫哪里就有反抗。现在这句话正在中共身上不断应验，准确地讲，应该说是哪里有中共的压迫哪里就有中国人民的反抗。近年来，杀警察的、杀城管的、杀强拆的、杀当官的等等事件不但屡见不鲜，甚至是有增无减愈演愈烈。按说中国老百姓也不是那种动辄因为一点不平就挥刀相向的烈性子，如果不是被逼到绝路，忍无可忍了，一般都会选择忍气吞声低头认命，除非是把他们真的逼的无路可退，活不下去了，才会被迫奋起反抗。杨佳杀警察是这样，夏俊峰杀城管是这样，贾敬龙杀强拆的村官是这样，这回
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摩的司机.html" target="_blank">
  摩的司机
 </a>
 罗某怒杀管理人员不也是同一个道理吗？
 <br/>
 <br/>
 其实，中共交警及运管部门肆意罚款早已招致民怨沸腾。早前，大陆媒体2013年曾报导，山西太原小店区的运管所〝车〞过拔毛，仅2012年上缴的罚没收入就达1,277万元。过往货被随意扣车、罚款，每笔金额从500元至2万元不等。今年5月，陕西富县曾发生过交管所中队长查车与司机发生冲突双双住院。交管所更纠集黑社会人员对司机进行殴打。今年6月，因不堪忍受花样繁多的过路、过桥费，以及各地交警和部门的层层盘剥及高油价，有大批卡车司机愤怒罢工，令中共惊恐不已。正如一位网友所说：〝如果在盛世不能生存，犯罪就是一条合理出路。〞摩的司机罗某当街刺死运管所副所长张仲凯不就是一个例证吗！
 <br/>
 <br/>
 可以断言，随着中共对中国民众的压迫越来越重，中国人民的反抗势必也越来越激烈，并终将彻底埋葬中共的暴政。
 <br/>
 <br/>

 <br/>
