### 习近平让步 中国经济或将出现结构性改变
------------------------

<div class="wysiwyg">
 川习会后，中共官方在12月5日首次承认〝90天〞谈判时限，以及将购买美国大豆及液化天然气。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 当天发推文表示：〝我相信习主席会认真对待他在我们很长且充满希望的历史性会议上所说的每一句话。所有议题都会被讨论！〞而在12月4日晚间，川普发推文说：〝我们要么对中国进行真正的交易，要么根本没有交易，（如果没有交易）此时我们将对中国产品征收关税。最终，我相信，无论是现在还是未来，我们都将达成协议。〞这是在中美展开涉及中国经济结构性改变谈判内容前，川普总统再次重申美方的谈判底线，也是对中共的警告。
 <br/>
 <br/>
 此次川习会的一个重要之处在于，这是习近平首次在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 中走向前台，直接主导中方谈判，那么，他做出的任何决定和承诺，都与他本人的政治信誉紧密相连，并且，他将直接为中美贸易战的结果承担责任。
 <br/>
 <br/>
 显然，中共官方想继续隐瞒川习会中方做出重大让步一事将越来越困难。其中一个原因，是美方向媒体和外界透露了川习会的诸多细节。
 <br/>
 <br/>
 白宫贸易顾问纳瓦罗在川习会后接受媒体采访时，主持人问：〝晚宴上中方是否重视解决美中贸易争端问题？〞纳瓦罗回答：〝一开始主要是习近平在谈，他是前30分钟里中方唯一在谈话的人。〞
 <br/>
 <br/>
 纳瓦罗透露，〝双方就整个结构性问题进行了讨论，包括中国强制技术转让、知识产权保护、非关税壁垒、网络入侵及网络盗窃、服务和农业等诸多方面〞。这其中大部分都是美方要求的结构性内容，也是中方过去不承认、拒绝谈判的问题。纳瓦罗说：〝习近平亲自就加大购买美国商品、保护知识产权、强制技术转让等许多问题作出了承诺。〞
 <br/>
 <br/>
 习近平在近期以来的多个外交场合中，手拿讲稿读稿，表现出心事重重的样子。但是，从川习会媒体播放的现场视频中，可以看到习近平完全改变此前状态。从此可以判断，此次谈判是习近平本人决定作出重大让步。
 <br/>
 <br/>
 今年5月3日，美国贸易代表团抵达北京，就双方贸易问题进行第一轮谈判。我当时分析预计，中、美之间的贸易谈判虽然在许多具体行业和细节上，会有激烈博弈，但在总体上，中共将会妥协和退让。但是，最终谈判破裂。
 <br/>
 <br/>
 5月15日中共率队赴美，与美方展开第二轮贸易谈判，谈判破裂。8月22日，重启谈判，谈判破裂。
 <br/>
 <br/>
 中美贸易谈判几经轮回，如今终于重回原点和正轨。
 <br/>
 <br/>
 美、中贸易摩擦的根源，从表面上看，是中共自入世以来，多年来不遵守国际贸易规则，盗窃美国科技知识产权，加上美中的贸易结构之间存的〝结构性的失衡〞，造成过去三十多年中共已经从美国抽走了3.5万亿～4.2万亿美元的财富。
 <br/>
 <br/>
 从深层来看，是中共的政治体制，决定了中国的经济体制，无法真正融入世界自由贸易体系。中共的经济改革发展的根本目的，还是为了政权服务。而维持中共政权存在，必须控制自由信息，中共党管一切。自由经济市场所需要的自由信息和公平法治环境，在中共治下，都不可能实现。
 <br/>
 <br/>
 但是，当美国中期选举的结果并没有对中共贸易战起到缓解作用之后，当2019年开始对2千亿商品增加的25%关税即将到来的时候，中国经济承受的压力加剧，资金外逃，失业激增，如果再不退让，就将危及中共政权稳定；同时，习近平承受中共内部的压力也不断加大。在多种因素交织下，习近平选择了对美国让步。
 <br/>
 <br/>
 那么，此次习近平用重大让步换取的三个月的谈判时间，绝非此前中共用惯常的拖延欺骗战术，就能够摆脱危机的。川普的推文，已经释放出了最后通牒的信息。因此，此次三个月的谈判，很有可能会与美方达成中国贸易结构性改变的协议或者部分协议。当然，按照中共的惯性常态，中共在压力和无奈之下会被动退守，但会尽量死守一些它认为的底线，比如开放互联网等。
 <br/>
 <br/>
 但是，在停止盗窃美国知识产权以及用国家资金补贴国企等问题，很可能会逐步做出改变。如果是这样，就会带来一个相关的问题。
 <br/>
 <br/>
 政治和经济紧密相连。中共如今对中国的统治，大部分维系在对中国经济的控制上。同时，中共利益集团又利用中国的经济发展，来攫取巨额财富，为了维持利益，只有继续维持中共的统治模式。在这一点上，中共既得利益集团是达成共识的。
 <br/>
 <br/>
 在这样的状况下，中国经济如果发生结构性改变的话，这意味着中共政权将会逐步离场，失去对中国经济的控制，最终使中共政权解体。停止对大型国企进行补贴，这些企业将会在国际自由经济市场中被淘汰，中共利益集团将会失去通过国企获得的巨额财富。因此，中共的既得利益集团将会拼死抵抗和破坏。
 <br/>
 <br/>
 因此，习近平未来面对的，将会是更加严峻的中共内部的政治压力。中美贸易谈判将可能造成的改变，已经箭在弦上。可以预计，中共内部利益被触及的势力将不会安稳，中共高层的激烈内斗将会加剧。高层政局的动态，也必然会体现在社会层面，从整体上来讲，中共将会加剧对中国社会和民众的控制。
 <br/>
 <br/>
 从客观上来讲，此次中美贸易谈判的结果，很大可能将会在一定程度上推动中国经济、政治、整个社会等诸多方面的变化。中国社会将迎来巨变。
 <br/>
 <br/>
