### 文武：中共无法承受贸易战升级的后果
------------------------

<div class="wysiwyg">
 半年前中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 开打的时候，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 是来势汹汹，高喊着要与美国奉陪到底，直至美国服输为止。可是转眼间，中共是180度的大转弯，美国是越战越勇，而中共是不战而降。
 <br/>
 <br/>
 这次阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 当面向美国让步，请求美国给它九十天的时间来满足美国提出的一切条件。
 <br/>
 <br/>
 中共为了争取这九十天的时间，马上就大量的花钱购买美国的商品，特别是美国的农产品。
 <br/>
 <br/>
 中共就是这样，欺软怕硬。原来国际社会普通软弱，中共来个横蛮不讲理，采取软磨硬泡的方法，一次又一次的度过了危机。这次，中共是不行了，是遇到对手了，原来的那一套不管用了，甚至用来吓唬人的作用都没有了，现在的美国不吃那一套。要打就真打，不断的升级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，中共一看，这下完了，这样下去肯定是完蛋了，于是，中共转过脸来就认输了。
 <br/>
 <br/>
 原因是中共现在根本就没有打贸易战的底气，中共现在在经济领域的根本问题，并不是国际贸易问题，而是中共的债务问题。中共的经济是个巨大的庞氏骗局，通过不断的举债，将经济泡沫越吹越大，目前已经到了再也无法吹得更大的时候了，现在是要么急速下滑，要么是泡沫破灭。
 <br/>
 <br/>
 如果中共的经济急速下滑，那么马上就面临着大量的失业和大量的企业倒闭，因为在中国无论是国营企业还是民营企业，大部分都是靠不断的举债在支撑着，一但停止举债，这些企业就会倒闭，人员就会失业。
 <br/>
 <br/>
 如果中共的经济泡沫直接破灭，就是中共的经济直接硬着陆。那么整个中国的经济就彻底崩溃，社会动荡将不可避免，中共可能直接垮台。
 <br/>
 <br/>
 面对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共经济.html" target="_blank">
  中共经济
 </a>
 的现实，中共政府当然是非常清楚的，一旦这个时候，与美国的贸易战还进一步升级，那无疑将加速中共经济的崩溃。
 <br/>
 <br/>
 中共的债务问题，中共的产能过剩问题，目前唯一能维持下去的原因，就是中共对外贸易和吸引外来投资。只要中共的对外贸易和外来投资能够维持一定的水准，那么中共的债务问题，中共的产能过剩问题，中共的经济泡沫问题，就能相应的维持一段时间。
 <br/>
 <br/>
 国际贸易和外来投资是中共目前唯一的暂时的救命稻草。美国对中共发起的贸易反击战，正好击中了中共的要害，一击而中，中共无法承接中美贸易战升级的后果，可以肯定地说，中美贸易战升级将大大加速中共经济崩溃。
 <br/>
 <br/>
 在这种情况下，为了避免中美贸易战进一步升级，中共现在是什么血本都愿意花，是什么要求都会答应，即便投降。当然，期待其真正遵守承诺是不现实的。
 <br/>
 <br/>
 尽管如此，投降对于中共来说，并不能真的能留中共一命，因为中美贸易战，只是中共的催命符，只是会促使中共加速崩溃的一个原因。
 <br/>
 <br/>
 就算中美之间不打贸易战，就算现在就停止中美之间的贸易战，其实，中共的崩溃也很快就会到来，这是无法改变的结果。
 <br/>
 <br/>

 <br/>
