### 周晓辉：川普在暗讽中共高官什么
------------------------

<div class="wysiwyg">
 几日前，美国总统川普（特朗普）在接受媒体采访时表示，他计划发布行政令终止现行的美国政策，即取消非美国公民以及无证移民在美国生育的孩子自动获取公民权。川普表示：〝美国是全世界唯一的国家，一个人来这里生了孩子，孩子就是美国公民了，拥有各种公民福利。这是荒唐可笑的，这种事情必须终止。〞他指出，美国为此每年要花费数十亿美元，这对美国不公平，而〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出生公民权.html" target="_blank">
  出生公民权
 </a>
 〞甚至催生了一个叫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_〝生育旅游〞.html" target="_blank">
  〝生育旅游〞
 </a>
 的产业。
 <br/>
 <br/>
 川普拟取消〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出生公民权.html" target="_blank">
  出生公民权
 </a>
 〞的言论随即引起了广泛的关注，反对意见认为这与美国宪法第14修正案相违背。是否与宪法相违背，非本文关注之处，暂且不说，倒是川普点出的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_〝生育旅游〞.html" target="_blank">
  〝生育旅游〞
 </a>
 产业的主要服务对象不少来自中国。
 <br/>
 <br/>
 据报，现在美国有数百间华人为中国产妇开办的合法不合法的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_〝月子中心〞.html" target="_blank">
  〝月子中心〞
 </a>
 。中国产妇通过缴纳不菲的费用，来到美国待产，原因就是孩子一出生就成为美国公民，可以享受美国的各种福利。21岁后即可申请父母来美，成为永久居民和公民。
 <br/>
 <br/>
 对于中国人钻美国人的空子，占美国的便宜，美国当局心知肚明，也加强了入境执法和对非法的〝月子中心〞的查处，但这并不能从根本上解决问题。然而，一旦〝出生公民权〞被取消，这样的〝生育旅游〞自然也会走向衰落。
 <br/>
 <br/>
 不过，显然〝生育旅游〞给美国带来的损失并非是让川普不满意的唯一原因。11月1日，川普在密苏里州为共和党候选人拉票的大会上，直指中国有最多孕妇赴美产子，并批评〝有外国独裁者一边想着与美开战，一边利用出生公民权〞，暗讽中共高官。
 <br/>
 <br/>
 川普如此说道：〝现在最多赴美产子的地方不是拉丁美洲国家，而是中国。你们可能会感到惊讶。〞〝想想吧，你是我们国家的敌人，你是想着开战的将军，你是讨厌我们、想对付我们的独裁者，但这个独裁者却让妻子在美国国土生子。恭喜你，你的子女现在是美国人了。有人觉得这合理吗？〞〝恭喜你，将军，你和你的子女一样拥有了美国国籍。这太疯狂了！〞
 <br/>
 <br/>
 也是，允许视美国为敌人的某国的独裁者、将军或者其他高官的妻子在美国生孩子，这不是在引狼入室吗？在发生危机时，他们会效忠美国吗？有着正常思维的人都不会认为这是合理的和正常的。而这样的事情却在当今的世界的的确确发生了，那就是中共的独裁者们和高官们或是让自己的后代在美国出生，或是让自己的子女在美国留学、移民，并将财产转移到美国。川普所打的比方剑指的正是中共高官们。
 <br/>
 <br/>
 公开资料显示，邓小平的孙子邓卓棣、江泽民的孙子江志成就是在美国出生，并自动获得美国国籍的，而中共高官们的后代自动获得美国国籍的当不止这两人。
 <br/>
 <br/>
 2011年新浪微博曾有一则很快被删除的消息透露：美国政府统计，中国部级以上的官员（包含已退位）的儿子辈74.5%拥有美国绿卡或公民身分，孙子辈有美国公民身分达到91%或以上。可以推断的是，这些高官孙子辈拥有美国公民身分的正是源于〝出生公民权〞，而这些人的背景，美国政府当是一清二楚，这也是川普讽刺中共高官的底气所在，也是川普认为应取消〝出生公民权〞的主因。
 <br/>
 <br/>
 另据中共中组部2011年的调查，几年来中共高干家属、高干子弟移民海外，包括香港和澳门在内的一共有108万人。这个名单上就包括2008年移民澳大利亚的、原政治局常委曾庆红的儿子曾伟，和去澳洲的原政治局常委贾庆林的两个儿子。至于去美国留学的中共高官的子女更是数不胜数，包括现在中南海的高层们的子女。
 <br/>
 <br/>
 为何中共高官的儿孙辈如此不待见中国，不待见中国教育，并竭力移民美国和其它西方国家呢？一个主要原因是身在中共体制内的他们，最清楚中共的残酷和翻云覆雨的手段，让儿孙辈移民就是为了保住自己攫取的财产，为了避免随时被失踪被暗杀，为了给自己寻找条后路。显然，最不爱中国的恰恰是中共的高官们。
 <br/>
 <br/>
 对于中共高官们的不耻行为，美国政府长时间都保持了沉默，而川普在提出废除〝出生公民权〞后直指中共高官，将真相告诉了美国人，也证实了中国坊间流传的中共高官的亲属大多是外国人的传言不虚。或许在川普看来，这些中共高官的儿孙们犹如特洛伊木马，在关键时刻会危及美国的安全，理应防患于未然。
 <br/>
 <br/>
 无疑，北京当局也意识到了高官的后代是外国人所引发的可能后果。10月，有媒体曝出一份禁止高官子女去美国留学、让已经出去的子女年内回国的内部文件业已出炉，而这或许是当局为避免中共高官们〝身在曹营心在汉〞，避免中共高官们因顾忌自己的子女、财产在美国而对中共政权〝阳奉阴违〞，甚至在川普极限施压后在内部发难的原因。
 <br/>
 <br/>
 北京当局的措施是否奏效，尚不得而知，但川普拟废除〝出生公民权〞所引发的连锁效应可能更大，至少中共高官们明白了，美国也不是个避风港，加入美国籍的儿孙们也已经被美国政府盯着呢。
 <br/>
 <br/>
