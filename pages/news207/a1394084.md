### 崔士方：如范冰冰是老Q 背后可有老K？
------------------------

<div class="wysiwyg">
 影坛一姐
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 神秘消失123天后，以连缴带罚吐出8.8亿的沉重代价重现公众视野，激起大片涟漪。
 <br/>
 <br/>
 在中共惯用的〝被失踪〞模式下，公众早知〝范爷〞这只高脚女靴迟早要落下，只是未料落靴时机被选在了公众都忙于旅游、媒体也处于半歇业状态的〝十一〞长假期间，目的显然是要尽量降低靴子坠地的冲击力。在此刻意安排下，〝范爷〞靴子落地虽响，实际冲击力却打了不少折扣。
 <br/>
 <br/>
 这只能说，北京当局虽然重手整治娱乐圈逃税，但并未打算〝赶尽杀绝〞。
 <br/>
 <br/>
 值得注意的一个细节是，〝阴阳合同〞事发之初，崔永元曝光的是一个1,000万：5,000万的〝大块头〞，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 案中，当局公布的只是一个1,000万：2,000万的〝小个子〞。换言之，这个〝大块头〞是另有其主，现被捏在当局手中，再加上崔永元说的那〝一抽屉阴阳合同〞，端看什么时候被亮出。这显然是税务局威吓娱乐圈的袖中牌，是想藉此让所有大腕乖乖知趣就范。
 <br/>
 <br/>
 娱乐明星名头虽响，却多是缺乏权贵背景的草根出身（姜文等〝大院子弟〞的力量其实也相当有限），头戴〝范爷〞高冠、气场强大的范冰冰也概莫能外。2007年约满离开华谊兄弟，直到2015年才借男友李晨的人脉重新与老东家华谊搭上线，华谊的王氏兄弟实在算不上范冰冰的靠山。而范冰冰的经纪人穆晓光虽然是上海著名的151会所老板（据传该会所已关门歇业），但其根基在台湾，很难想像穆晓光能成为足够粗的、可以长久依傍的大树。
 <br/>
 <br/>
 所以本质来讲，范冰冰们虽然肥得冒油，但她们其实大多是〝高级韭菜〞，而非〝镰刀〞。即便是个人财富和人脉都要高范冰冰一头的〝小燕子〞赵薇，虽然靠上了马云、华谊兄弟，但马云、华谊兄弟本身也是肥了就要被宰的猪，自身难保。所以说到底，长袖善舞的赵薇还是〝韭菜〞族。
 <br/>
 <br/>
 如果把范冰冰视作〝老Q〞，同样〝被失踪〞的明天系掌门人肖建华为〝老J〞，那么范冰冰背后并无〝老K〞，而肖建华背后的〝老K〞却是江派〝镰刀手〞曾庆红。
 <br/>
 <br/>
 所以，北京当局敲打〝老Q〞和〝老J〞，目的大不相同。
 <br/>
 <br/>
 敲打范冰冰，目的是杀鸡给猴看，让娱乐圈众多〝高级韭菜〞心惊胆战，乖乖到税务局等候被收割。当前银库羞涩的中共既可以借普罗大众的〝仇富心理〞，展现打击逃税的〝正义形象〞，又顺带收到一大笔娱乐圈补缴银两，何乐而不为？这是当局一石二鸟的伎俩。
 <br/>
 <br/>
 而当局敲打〝老J〞则是〝镰刀〞之间的互怼，是你死我活的权力争斗。
 <br/>
 <br/>
 范冰冰等〝高级韭菜〞虽然被收割，但当局并不想连根拔，原因是这批〝影视韭菜〞对中共政权还有其他重要功用。
 <br/>
 <br/>
 中共需要老百姓热衷于追星加粉，这样才能不过问政治，看不见或视若无睹中共的胡作非为。同时，利用电影、电视剧的〝主旋律〞对观众施加潜移默化的影响，可以培养出一批批红色战狼，对于中共的固权也是大大有利。
 <br/>
 <br/>
 所以我们才看到，在最新一轮的机构改制中，原本只是正司局级的广电总局辖下电影局被单列出来，由正部级的中宣部常务副部长兼任局长。这种超规格待遇背后隐藏的动机，可谓司马昭之心，路人皆知。
 <br/>
 <br/>
 ──转自《大纪元》本文只代表作者的观点和陈述。
 <br/>
