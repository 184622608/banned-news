### 共产党洗脑机制下的怪胎斯德哥尔摩综合症
------------------------

<div class="wysiwyg">
 1973年8月23日，瑞典首都斯德哥尔摩一家银行遭遇劫匪，其中一人是有前科的越狱者。两名劫匪绑架了四个职员作为人质。
 <br/>
 <br/>
 警察随即包围现场。人质被监禁在地下室，三男一女被绑火药。警察于28日突击成功，救出人质并擒获劫匪。
 <br/>
 <br/>
 与警匪对峙的六天中，人质已经形成一个非常奇怪的心理状况。他们不但拒绝外面的营救，而且认为营救他们的警察是要害他们，绑架他们的人却在保护他们。
 <br/>
 <br/>
 更加使人惊讶的是，几名人质获救以后不但不控诉劫匪，反而向媒体和法官赞美劫匪。感谢劫匪的不杀和善待之恩，女人质还和一个劫匪订了婚。一位人质建立了一个基金会在全球为绑架者筹款。
 <br/>
 <br/>
 在生命受到威胁的时候，强烈的求生欲望使人质认同了绑架他们的人，并且持续到威胁解除以后。这种精神病症被命名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯德哥尔摩综合症.html" target="_blank">
  斯德哥尔摩综合症
 </a>
 〞。
 <br/>
 <br/>
 产生斯德哥尔摩综合症需要有四个条件
 <br/>
 <br/>
 第一个条件是：要让人切实感觉到他的生命受到威胁，让他相信这个施暴的人随时会毫不犹豫地加害于他。
 <br/>
 <br/>
 第二个条件是：施暴的人一定对受害者施以小恩小惠，使受害者相信施暴者是他们生命的救主。
 <br/>
 <br/>
 第三个条件是：控制受害人的信息环境，使他与世隔绝得不到任何施暴者不想让他得到的信息。
 <br/>
 <br/>
 第四个条件是：让受害人感到无路可逃。
 <br/>
 <br/>
 不难发现  中共具备了制造斯德哥尔摩综合症的所有条件：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产党.html" target="_blank">
  共产党
 </a>
 对中国、越南、朝鲜民众实行了异常成功的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗脑.html" target="_blank">
  洗脑
 </a>
 ，其机制和产生斯德哥尔摩综合症的机制有着惊人的相似。
 <br/>
 <br/>
 共产党统治下的民众（人质）不但拒绝外界的营救，反而认为那是想加害他们的反动分子，把监禁他们的共产党（劫匪）认作保护他们的救世主。
 <br/>
 <br/>
 更让人惊讶的是，当那些共产党领袖过世时，民众不但不控诉他们反倒尊称他们为祖国的〝父亲〞。甚至还希望他们一直活着，一直统治着……
