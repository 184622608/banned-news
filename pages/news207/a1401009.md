### 袁斌：中共才是世上最该抵制的头号辱华者
------------------------

<div class="wysiwyg">
 大名鼎鼎的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_D&amp;G.html" target="_blank">
  D&amp;G
 </a>
 绝对不会料到，仅仅因为一个与筷子有关的广告宣传片，再加品牌设计师与网友争辩时爆出的几句粗口，竟然引发了一场闹得沸沸扬扬、鸡犬不宁的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辱华.html" target="_blank">
  辱华
 </a>
 〞风波，不但大陆爱国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_愤青.html" target="_blank">
  愤青
 </a>
 怒火冲天，一干明星模特誓言抵制，人民日报、环球时报等官媒口诛笔伐，就连天猫、京东、苏甯易购、网易考拉、唯品会、1号店、洋码头等中国知名电商平台也齐刷刷的将D&amp;G相关产品都一股脑下架了，甚至于最后D&amp;G创办人不得不出面公开道歉也没能平息这场风波。
 <br/>
 <br/>
 不用说，类似这样的风波近年来可谓屡见不鲜，人们已经见怪不怪了。中共的官媒和大陆的爱国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_愤青.html" target="_blank">
  愤青
 </a>
 ，不是今天说这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辱华.html" target="_blank">
  辱华
 </a>
 ，就是明天说那个辱华，简直都上瘾了！
 <br/>
 <br/>
 出于好奇，我用百度搜了一下〝辱华〞这个词，显示的结果居然有上百万条，其中包括日本辱华、韩国辱华、美国辱华、瑞典辱华、中国留学生与国外留学生辱华，有关全球知名媒体公司BBC、CNN、纽约时报辱华的消息更是多如牛毛。给人的总体印象是：全世界亡中国之心不死，全世界都在辱华。
 <br/>
 <br/>
 这倒让我想到了一个问题：究竟是谁在辱华？
 <br/>
 <br/>
 比如这次的风波，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_D&amp;G.html" target="_blank">
  D&amp;G
 </a>
 辱华了吗？爱国愤青和官媒当然认为辱华了，但不同意这种观点的也大有人在。比如我刚才看到的一篇文章就说：〝为了一个无聊广告而反应过激，本身已有辱斯文。严格来说，D&amp;G不是辱华，而是辱愚。〞
 <br/>
 <br/>
 撇开D&amp;G有没有辱华不谈，我以为对中国抱有偏见，或者说辱华的外国人当然不是没有，但世界上辱华辱得最厉害，也最需要加以抵制的绝非外国人，而是一贯假冒中国，以爱国主义自我标榜，动不动就以辱华为藉口煽动操控大陆无知网民闹事的中共。
 <br/>
 <br/>
 听我这么说，想必有人会感到不解：中共辱华？中共怎么辱华了？你且听我说。
 <br/>
 <br/>
 首先，一个众所周知的事实是，自近代以来，作为普世价值的民主宪政日渐成为世界各国人民追求的时代大潮，时至今日，可以说地球上绝大多数国家都已实行了民主宪政，国民的基本人权从而也得到了有效的保障。但中共却至今死抱着一党专政的专制体制不放，明目张胆的拒斥普世价值，肆无忌惮的践踏中国民众的基本人权，因此也极大的败坏了中国的国际形象。试问，这是不是辱华？
 <br/>
 <br/>
 其次，因为中共的权力不受监督和制约，今天的中国已成为世界上腐败程度最严重的国家之一，贪官人数更是雄踞全球首位。一个不争的事实是：无论是在中华民族的历史上还是在共产主义阵营中，腐败从来都不曾泛滥和疯狂到当今中国这般惊人的程度——不但无官不贪，而且大贪遍地，整个官场已经找不到一个没被蛀蚀的细胞。有线民甚至讽刺说，100个政府官员中有101个是腐败的。中共的腐败让中国的形象变的丑陋不堪，丑恶无比！试问，这是不是辱华？
 <br/>
 <br/>
 再者，中共当政后，大肆宣扬无神论，鼓吹无神、无佛、无道、无前生、无后世，无因果报应，一举斩断了中华民族道德良知的根基。改革开放后，中共又转而搞起了权贵资本主义、官僚资本主义，赤裸裸的宣扬一切向钱看，闷声大发财，这一切不可避免的造成了整个民族和国家史无前例的道德堕落与沉沦。今天的中国人信奉的是人不为己天诛地灭，什么都不怕，什么都敢干，为了满足自己的私欲，可以不择手段。虽然他们的穿着打扮比以往任何时候都光鲜，但所作所为却严重背离了做人的起码标准。毫不夸张的说，从古到今，中国还从来没有过一个时候像今天这样有这么多的垃圾人，这么多的人渣。他们在各种公开场合的所作所为，丢尽了中国的脸。而造成这种现状的罪魁祸首也是中共。试问，这是不是辱华？
 <br/>
 <br/>
 回顾历史，中国不仅是世界闻名的文明古国，而且是二战后崛起的新兴国家。而毛时代的中国，却以落后封闭而著称。今天的中国，虽然经济总量号称全球第二，动辄就以大国自居，甚至不知天高地厚的要引领人类发展的方向，但整个国家的国际形象却早糟得够呛。说到今天的中国，许多人脑海里立刻就会浮出极权、腐败和野蛮、不文明等字眼。而这一切无一不是拜中共所赐。
 <br/>
 <br/>
 一言以蔽之，不是别人，恰恰是中共让古老文明的中国在世人眼里变成了一个极权、腐败和野蛮的国家。试问，迄今为止还有谁如此败坏过中国的形象？如果中共不是这个世界上头号的辱华者谁才是？而我们最该抵制的不正是中共这个头号辱华者吗？
 <br/>
 <br/>

 <br/>
