### 文武：美国给中共发出最后通牒
------------------------

<div class="wysiwyg">
 昨天美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 在新加坡接受记者采访时说，这个月底举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会时，美国准备给中共最后一次机会。
 <br/>
 <br/>
 如果中共能够做出实质性的妥协，能够完全答应美国提出的条件，那么美国就准备给中共一条生路。
 <br/>
 <br/>
 如果贸易战可以称之为经贸领域的战争的话，那么，中美之间正在进行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 ，这一次在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会上就要有个了结。
 <br/>
 <br/>
 要么中共真的彻底投降；要么全面扩大贸易战，并且将贸易战延伸至经济、政治、军事、文化等各个领域，形成中美之间的全面对抗或者称为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 。
 <br/>
 <br/>
 表像往往容易掩盖实质，因为中共是强权体制，中共在国内封锁了所有与中共不同的声音，给人表面的印象好像是，中共无比的强大，可以调动一切资源，与美国打贸易战，只有美国认输、投降的理，不会有中共投降、认输的事。
 <br/>
 <br/>
 可是实质却是相反的，美国是越打越坚强，而中共是越打越心虚。
 <br/>
 <br/>
 因为美国是民主体制，美国政府代表美国人民，美国遵循的是普世价值，是在正义的一边；而中共是一党专制独裁的体制，中共不代表中国人民，中共只代表中共政治流氓集团，中共也不遵守普世价值，中共只相信金钱与权力，中共是站在正义的对立面。所以，贸易战真的打起来了，中共就不是个了，所以，这一次中共准备投降了。
 <br/>
 <br/>
 但是，中共就算是投降，可是又有谁会真的相信中共会真的投降呢？
 <br/>
 <br/>
 因为中共是不可信的，所以，这一次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 副总统给中共发出了最后的通牒，明确告诉中共不要玩花招，这一次给中共最后的机会是要中共真投降。如果是假投降，如果是又来玩弄美国，那么，美国就要与中共展开全面的对抗，就再也不给中共机会了，中共就只有等着最后的结果了。
 <br/>
 <br/>
 了解中共的人都知道，中共绝不会遵守什么协议，也绝对不会遵守什么承诺，中共自己制订的宪法自己都不去遵守，怎么可能去遵守一个什么中美之间达成的贸易协定呢？与美国谈判，不过只是中共的缓兵之计罢了。
 <br/>
 <br/>
 但是，美国这次不会管中共是什么计了，就只看中共的实际行动，也不会听中共什么承诺了，就看中共具体的做法。
 <br/>
 <br/>
 例如，南海的军事设施是否撤除，互联网的防火墙是否撤掉，非关税壁垒是否去掉，盗窃行为是否终止，就要这样一项一项的去对照、去检察，去落实。如果确实都做到了，那么贸易战就停止，如果没有全面按照美国的要求实际做到，那么就扩大 中美之间的对抗。
 <br/>
 <br/>
 这一次看来中共是没有招了，可能是真的要完了。
 <br/>
 <br/>
 作者提供 本文只代表作者的观点和陈述。
 <br/>
 <br/>
