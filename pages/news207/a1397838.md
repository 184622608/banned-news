### 新世纪影视带给世界的清流(2)影片《艺考》
------------------------

<div class="wysiwyg">
 近日，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新世纪影视.html" target="_blank">
  新世纪影视
 </a>
 基地试映了新拍摄的两部短片《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艺考.html" target="_blank">
  艺考
 </a>
 》和《过年》，我有幸先睹为快。
 <br/>
 <br/>
 成立于2017年的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新世纪影视.html" target="_blank">
  新世纪影视
 </a>
 基地，拍摄的首部华语片《密码》（Code of Revelation），就获得了2018年加拿大国际电影节（Canada International Film Festival）最高奖（Grand Prize Award）。
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艺考.html" target="_blank">
  艺考
 </a>
 》是继《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心愿.html" target="_blank">
  心愿
 </a>
 》之后，又一部让我感动的短片。影片的情节很简单：在中国华东艺术学院招生考场上，一个突发事件，让女考生安然失去了考试的机会，接下来发生的一切，一波三折。影片的前半段以轻喜剧展开，但是，故事后半段却两次反转，出人意料。
 <br/>
 <br/>
 影片时长20分钟，演员全部是来自美国及加拿大富有艺术才华的华裔，有些还是艺术专业的学生。故事剧本出自女主演郑雪菲和其母亲的部分经历。毕业于上海戏剧学院的郑雪菲说：〝拍摄过程中，只是怀着一颗纯净的心态，希望尽力把角色演好。但在剪片的过程中，看着看着，把自己感动得泪流满面。〞她坦承，影片故事跟自己的经历很类似，因此感同身受。
 <br/>
 <br/>
 郑雪菲有什么样的经历？郑雪菲和母亲因为修炼法轮功，曾在中国大陆遭受迫害。她的母亲，如今刚从监狱出来，仍在被中共监视居住。
 <br/>
 <br/>
 中共江泽民集团自1999年7月开始迫害法轮功，其间发生了大量的残酷的迫害，以及无数惊天动地、悲欢离合的故事，这场持续近二十年的全世界最大的迫害人权和人类信仰的运动，可以说是一个对世界和人类影响巨大的事件。但遗憾的是，无论是国际主流媒体，还是文学艺术和电影界，鲜有法轮功群体被迫害以及讲述法轮功修炼群体故事的作品和题材出现。这是世界的遗憾，也是耻辱。
 <br/>
 <br/>
 新世纪影视基地的获奖影片《密码》，就涉及中共迫害法轮功学员和活摘法轮功学员器官的话题。影片取材于一位目击中共活摘法轮功学员器官现场的中国警官提供的真实事件，故事围绕破解电脑密码展开。
 <br/>
 <br/>
 对于一个喜爱电影艺术的人来讲，看得电影越多，对于选择电影越来越挑剔。几乎大部分的好莱坞和欧洲电影，都可以归于类型片的范畴。每一部类型片，几乎都有一个严格的套路和规则，按照故事三幕剧或者布局、反应、进攻和解决四部结构来讲故事，其中包括第一情节点、第二情节点、故事转折点、关键点、中间点、场景、悬念、事件等等。
 <br/>
 <br/>
 随着科技的高速发展和电脑技术的应用，现代电影越来越可以打造更加逼真和宏大的场景，在电影中可以展现所有需要的内容场景，上天入地，几乎无所不能。
 <br/>
 <br/>
 但是，我对现代以好莱坞为代表的电影，越来越失望。其中一个主要的原因就是，无论电影有怎样高超炫目的科技支持，无论演员的表演如何专业精采，但是，主流电影释放的信息却是：这个世界并没有黑白分明的善恶正邪之分，传统是乏味和令人压抑的；道德是相对的，传统社会中被鄙视的丑恶行径，总可以在某种情境下被推到聚光灯的中心，受到同情甚至推崇。人们在享受娱乐的同时，道德相对主义悄悄植入人心，将人一步步推离传统。
 <br/>
 <br/>
 但是，在仅仅成立一年多时间的新世纪影视基地推出的几部电影中，我却看到了回归传统的趋势，让我感到惊喜。《密码》、《心愿》、《艺考》、《过年》这几部电影，风格传统，演员表演真诚、干净，一种打动人心的真诚和善意扑面而来。
 <br/>
 <br/>
 新世纪影视基地的电影，也正在改变着我多年形成的观影习惯、观念和趣向，我更加习惯传统的风格、叙事以及主题等等。
 <br/>
 <br/>
 在新世纪影视基地的官网中这样介绍，该影视基地的宗旨是传承〝真、善、忍〞核心价值，为所有秉持相同理念的影视人，提供一个广阔自由的创作平台；在世界需要回归传统，世人需要启迪心灵之时，制作出更多优秀影视作品，唤醒世人本性的归真，再现神传文化精髓，带给世界新的希望。
 <br/>
 <br/>
 新世纪影视基地正在带给世界一股清流，我们期待着这股清流带来更多的精采影片。
 <br/>
 <br/>
