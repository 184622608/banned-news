### 夏小强：厉害了我的国 厉害了女战狼记者
------------------------

<div class="wysiwyg">
 9月30日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国.html" target="_blank">
  英国
 </a>
 保守党在伯明翰国际会议中心（ICC）举行年会。英国国会议员布鲁斯（Fiona Bruce）女士主持召开了题为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 自由、法治和自治受到侵蚀〞的会议，中共央视驻伦敦女记者孔琳琳忽然站起来大闹会场。
 <br/>
 <br/>
 伯明翰所在的西米德兰兹地区警方发言人10月1日表示：〝昨天下午2点30分，警官赶到国际会议中心的11号厅，有报告说在有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 的会议上发生骚乱，一名来自伦敦的48岁女子涉嫌袭击被捕。她目前仍被警方拘留。〞
 <br/>
 <br/>
 在支持香港民主的保守党人权委员会副主席罗杰斯（Benedict Rogers）发言时，孔琳琳大喊大叫，说罗杰斯〝是个骗子〞，试图〝分裂中国〞，〝你是反华的！你都不是中国人！其余的（中国人）都是叛徒！〞
 <br/>
 <br/>
 主持人布鲁斯女士要求这位记者离开会场。现场的保守党成员Enoch Lieu在Twitter上写道，〝我走近她说，你已经提出了自己的观点，现在不再受到欢迎。〞〝这名记者继续喊叫，当我试图带她出去时，她骂着我，突然之间她打了我一个耳光。〞
 <br/>
 <br/>
 〝周围的人感到震惊，一些勇敢的观众走过来，试图带她出去。在纠缠中，她继续大喊大叫，拒绝离开，然后突然间又打了我一巴掌。〞 Lieu写道。
 <br/>
 <br/>
 在香港自由新闻网站上发布的视频片段，证实了部分事件。视频中可以看到，Lieu告诉一个愤怒的女人：〝这是保守党会议……请你走吧。〞而后孔琳琳做出了一个打耳光的动作。当其他人制止她并试图带她离开时，孔琳琳告诉Lieu：〝你没有权利，你们
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国.html" target="_blank">
  英国
 </a>
 没有民主，我在这里是一名记者……我有权利（抗议）！〞并对其他人说：〝他（Lieu）欺负我！〞
 <br/>
 <br/>
 不久之前，到瑞典旅游的三名中国游客，刚刚在斯德哥尔摩表演了〝一哭二闹三上吊〞的闹剧，让全世界都见识了中共宣传片《厉害了我的国》和《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战狼.html" target="_blank">
  战狼
 </a>
 2》的〝威力〞。如今，中央央视女记者在英国会场上演的〝全武行〞，又一次让全世界见识了〝巾帼不让须眉〞的中共〝女战狼〞的风采，〝女战狼〞不是我说的，而是中国网民给出的称号。
 <br/>
 <br/>
 英国保守党人权委员会副主席罗杰斯在会议上表示，〝我是爱中国的，我不是反华，我希望中国及其人民取得成功。我只是批评当前中共政府对待人民的方式。令人担忧的是，一名中共国家媒体代表，竟然滥用她的身分，试图恐吓我们行使合法言论自由的权利。她对会议代表进行身体攻击是不可接受的。这是否标志着中共越来越多的侵略和欺凌行为，远远超出了国界？我们敦促英国政府向中共当局正式致函，明确表示这种行为是完全不可接受的。未来，英国应该认真审查和限制中共官媒参加英国党派会议的资质。〞
 <br/>
 <br/>
 在英国这个法治国家，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视记者.html" target="_blank">
  央视记者
 </a>
 孔琳琳敢于在大众广庭之下暴力攻击他人，谁给了她这样的胆量？答案如下：在第一时间，中共驻英国使馆发言人就要求会议组织者向中国记者道歉，随后中央电视台的新闻显示，中共发出声明向英国当局提出正式抗议，事件和此前的瑞典警察驱离中国游客一样，演变成了外交事件。
 <br/>
 <br/>
 从央视发言人1日晚间表示〝中央广播电视总台向孔琳琳致以慰问〞来看，显而易见，孔琳琳不是一个人在战斗，她愤怒中喊出的话，不是她个人的看法，是喊出了中共官方的声音。她的暴力行为，也正代表了中共的无理和暴虐。
 <br/>
 <br/>
 其实，孔琳琳并不〝孤单〞，在她的身后，还有中共外交部长在国际场合上的表现。无论是联大会议上听到川普抵制社会主义时的〝耸肩〞——被网民称为〝抽风〞，还是此前在加拿大面对记者提问时的无理和傲慢，都和孔琳琳的行为同出一辙：无视世界文明和普世价值，展现中共的丑陋和蛮横，让中共在国际上出丑。
 <br/>
 <br/>
 从中国游客到代表官方的央视记者，再到中共外交部长级的高官，在国际社会上连续出丑之后，中共官方却高调介入，大肆报导，目的何在？
 <br/>
 <br/>
 中美贸易战已经全面展开，在贸易制裁的重压之下，已经危及到了中共政权安危的底线。中共正在采取一切手段，调动一切资源来转嫁政权危机。从政治和经济上全面左转，开始备战备荒，新的一轮〝上山下乡〞运动蠢蠢欲动……因此，在这样的背景下，抓住一切机会掀起民众对西方〝反华势力〞的仇恨，挑动民众的爱国和民族情绪，因此近期不断上演着〝厉害了我的国〞和〝战狼2〞的现实版。这就是在英国出现〝女战狼〞的主要原因。
 <br/>
 <br/>
