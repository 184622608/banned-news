### 林忌：90天后北京败得更惨
------------------------

<div class="wysiwyg">
 G20峰会习近平与川普（特朗普）会谈，一反最初所流传的〝协议〞消息，美中两国唯一达成的只有90日停止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 升级的措施，即美国原本定于明年1月1日起对2000亿美元的中国货品关税，由原本10%升至25%的措施，推迟了三个月，而这三个月之内中国必须就美国的要求，例如知识产权以至其它贸易市场开放问题上达至美国的要求，而中国却即时向美国购买农产品、能源以至工业制品，去减少两国的贸易逆差，即中国用了一大堆合约，不但无法达成协议，甚至连〝停火〞也做不到，只是令美国承诺三个月内的关税〝不再升级〞，这对美国而言，当然是有百利而无一害，而中国也许换来了宝贵的时间，但对大局真的有帮助吗？
 <br/>
 <br/>
 根据路透社的报导，早在11月22日，中国已向美国订下3348吨猪肉，以及明年的9384吨猪肉，这是自今年2月以来，以及自中国对美国猪肉征收62%报复关税以来，最大宗的购买；在美中两国未举行峰会前，就已经要大批买入猪肉的意义为什么？明显就是中国无法不买美国的猪肉；早前中国改从俄罗斯购入猪肉的结果，就是引入了俄国传入的〝非洲猪瘟〞，不但无法解决中国的需求，反而疾病的迅速扩散，令中国自己的猪只都大规模染病。
 <br/>
 <br/>
 结果一如黄豆，中国不向美国购买，除了令农产品价格上升，不但无法满足中国的需求，对美国造成的伤害是有限，而中国自己却率先无力维持，与〝华文传媒〞所吹嘘的相反，没有中国货的感恩节假期，美国人照样的过，而不买美国农产品，中国自己却活不下去了。
 <br/>
 <br/>
 然而对中国更大的冲击，当然来自于中国的出口。
 <br/>
 <br/>
 早几个月不断传出中国经济成长不断放缓，圣诞节订单取消等的不利消息，而这种情况继续持续下去，中国很多工厂将会捱不住倒闭；这90天〝停止升级〞的期限，对于中国厂商而言，只是聊胜于无——跨国集团或者赢取了更多的时间，把生产线搬离中国；对中国的商人与工人来说，则早前已受关税影响的货品，其关税又不会取消，同时市场预期问题不能解决的话，美国的买家只会继续回避风险，而不会重新购买中国货。
 <br/>
 <br/>
 由夏天开战时华文传媒不断声称〝中国成为大赢家〞、〝中国已成最后赢家〞，甚至幻想
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 〝给机会令中国技术进步〞，如今这些谎言大话已经有气无力，虽然五毛党在Youtube上载了大量这类节目的影片，但已沦为笑话。
 <br/>
 <br/>
 因此这90日的〝延期〞后，中国经济问题只会进一步浮现，而美国的立场只会变得更强硬；比起今年夏天时一早妥协，中国如今已经无牌可打——关税早已加了，美国国会的中期选举过了，厂商开始搬了，美国甚至把战线拉到更多的科技禁运，审查留学生，在海外市场封杀中国企业，例如华为的5G网络，以至与其它国家组成〝毒药条款〞排中的关税同盟；而中国，目前换来了〝宝贵的时间〞，但以为就此可以〝以拖待变〞，绝对是非现实非理性的想法。
 <br/>
 <br/>
 一如早前美国副总统彭斯对中国的警告，世界对中国（中共）的耐性早已用尽；对西方民主政体来说，反应慢、难求共识、害怕冷热战争——包括贸易战，都是民主政体难以开始的难关，但其难关在于第一关，即开战；反而开了战想停，比起不停更难。
 <br/>
 <br/>
 西方民主各国期望中国放弃其〝世界帝国〞与〝一带一路〞的野心，然而中国内部经济成长已经去到尽头，不扩张就无法再前进，不前进其中共独裁专政的基础——包括压抑中产与基层的基础，将不复存在；因此中国独裁的道路，已经限制了其选择，而这一场〝新冷战〞，几乎可以说是无法避免，只是视乎早打，或者迟打；是全冷战，还是半冷不热地持续下去。
 <br/>
 <br/>
