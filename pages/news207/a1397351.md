### 程晓容：巴西是镜子 新总统宣言冲击中共
------------------------

<div class="wysiwyg">
 巴西与中国，有不少相似之处：地大物博，贫富悬殊，政客贪污成风，社会动荡。
 <br/>
 <br/>
 10月28日，巴西总统换届，左派33年统治告终。当晚，庆祝人群燃放烟花，彻夜欢腾。
 <br/>
 <br/>
 新任总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波纳索罗.html" target="_blank">
  波纳索罗
 </a>
 在胜选后发表讲话，强调捍卫宪法、民主及自由。他说：〝作为自由的捍卫人，我将领导政府保卫遵纪守法的民众的权利。法律面前人人平等。我们将恪守这一原则〞。
 <br/>
 <br/>
 波纳索罗的支持者表示：〝他是个爱国者，他将照看这个国家，而非一个党派。〞他们期待波纳索罗让国家〝重回正轨，保障民众的安全、教育和健康〞。
 <br/>
 <br/>
 如今，巴西政坛向右转舵，将对南美和世界产生影响。以巴西为镜，中国应获启示，顺势而变。
 <br/>
 <br/>
 在拉美地区, 政客贪污是经年常态，巴西民众对此深恶痛绝，曾掀起大规模抗议。相关报导中的〝前‘腐’后继〞、〝制度漏洞〞等用语，都令人联想到同病的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 官场。
 <br/>
 <br/>
 据媒体2017年6月报道，巴西警方在过去４年里，调查了超过2千起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_腐败.html" target="_blank">
  腐败
 </a>
 案件，发现国家因腐败蒙受的损失达300多亿美元。2014年3月，媒体披露了〝巴西史上最大腐败案〞：巴西石油公司高管与政界人士伙同腐败，他们利用外包工程虚抬报价，涉案受贿金额达数十亿美元。
 <br/>
 <br/>
 不过，与中共贪官相比，这只是小巫见大巫。据中共官方消息，今年前9个月，共有114万官员因贪腐问题被处理，仅在广东省就有30,288人。中共官场的贪腐现象，纵横交错，遍布所有系统、领域。上至国级干部，下至科长村官，涉案金额往往特别巨大，情节特别严重；亿元贪官层出不穷，民众见怪不怪。不仅如此，还有官员竟然涉黑、涉嫌活摘器官等反人类罪，突破道德底线。
 <br/>
 <br/>
 因此，巴西民众的反腐呼声以及对新政权的期待，也是中国百姓的民意。波纳索罗的宣言，句句冲击中共。他说，新政府要将巴西建设成〝伟大、繁荣和自由的国家〞，要〝保卫遵纪守法的民众的权利〞。
 <br/>
 <br/>
 在中国，守法尽义务的公民受到何种对待？他们的权利可有保障？
 <br/>
 <br/>
 现实令人悲哀——维权律师被吊销执照，被强制失踪，被构陷入狱，他们的亲属被株连、监控、投诉无门。讲真话者因言获罪，〝六四天网〞创办人黄琦身陷冤狱、身患重病，85岁老母亲盼与儿子相见，望眼欲穿。疫苗受害者家属上访无果、反成〝维稳〞对象。遵循〝真、善、忍〞的法轮功学员仍然遭遇绑架、非法庭审和判刑，仍然面对被活摘器官的危险。
 <br/>
 <br/>
 如何改变国家的命运？巴西已见黎明，中国为何长夜漫漫？
 <br/>
 <br/>
 波纳索罗说，他的执政愿景〝不只是一个政党或一个人的宣言，这是在上帝前的誓言〞。
 <br/>
 <br/>
 把心灵交给神，自然笃定、坚决，不畏艰险，这正是差异的根源所在。波索纳罗还曾表示，上任后将〝清洗〞左翼〝赤色分子〞，所以中共试图拉拢这位右派，自然无果。
 <br/>
 <br/>
 今日，中国同样处在变局的关口：乱象遍地，民怨沸腾。向左走，还是向右？方向性选择，事关重大。百年史实明监：共产主义学说，内藏红色幽灵，与普世价值为敌，破坏传统、侵蚀道德、颠覆社会秩序，遗祸无穷。此时，若抱持马列主义、坚持〝党〞的领导，绝无出路。
 <br/>
 <br/>
 有外界分析，中共官员必须选择向党忠诚，还是向执政者忠诚。此言差矣。所有的人，都需要决断：选择正义还是邪恶，选择人类良知还是欺世谎言，选择光明未来还是灾难覆灭。
 <br/>
 <br/>
 ──转自《大纪元》本文只代表作者的观点和陈述。
 <br/>
 <br/>
