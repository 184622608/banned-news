### 陈思敏：彭斯演讲后效应强 中共难掩恐慌
------------------------

<div class="wysiwyg">
 就中共当局及一些相关部门而言，10月8日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_A股.html" target="_blank">
  A股
 </a>
 节后首个交易日的结果，可以说是意料之中的下跌，但跌幅却是意料之外。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_A股.html" target="_blank">
  A股
 </a>
 8日复市之前，7日一天之内，先是央行无预警宣布降准，紧接着财政部公告将研拟大规模减税降费措施。8日开盘时，央视新闻股市大头条特地放送银保监会新规理财产品可入市。
 <br/>
 <br/>
 中共三大财经部门如此罕见同时表态挺股市，沪指8日收盘仍然大跌3.7%，市值一天蒸发1.84万亿元。也就是说，央行这次不放水12,000亿元（释放增量资金约7,500亿元+到期未续作MLF约4,500亿元），那么沪指8日大跌5%或200点的可能性都有。
 <br/>
 <br/>
 影响8日A股市场大跌的原因：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港股市.html" target="_blank">
  香港股市
 </a>
 收黑、经济数据多是差的，中美贸易战没有缓解的迹象等等，这些都是已知的因素，市场也有基本耐压性，投资人和资金理应不会突然扩大悲观情绪和出逃，所以连机构都不否认的主要因素，其实是美国副总统彭斯4日在美国华府智库哈德逊研究的演讲。
 <br/>
 <br/>
 一个佐证是，据披露行情数据统计，8日截至收盘，外资净流出达96.04亿元，也创下年内第二高。值得注意的是，9月份外资净流入金额达175.79亿元，即10月份首个交易日就流出了9月份净流入资金的一半。外资这一日96亿多元的资金加速集中流出，都在避险。
 <br/>
 <br/>
 虽然彭斯讲话内容就股市本身关系不是很大，但是心理情绪影响非常大。这也中共三大财经部门罕见同时为A股注入强心针，其反映的也是中共对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯演讲.html" target="_blank">
  彭斯演讲
 </a>
 的恐慌。
 <br/>
 <br/>
 中共恐慌彭斯演讲，除了这堪称21世纪以来美国官方发出最强烈抨击的一场演说之外，特别是彭斯演讲中，在说到不少关键地方的时候，显然清楚区分了〝China〞和〝Communist Party〞，即中国、中共有别。
 <br/>
 <br/>
 而川普贸易战异曲同工，让中共在经济股市金融领域等一连串的隐性问题都变成显性。就中共而言，那些掩盖不住的问题都意味着人民对共产党没信心与不信任。
 <br/>
 <br/>
 美中贸易战开打以来，中共宣传一直表现出〝应战的信心和报复的决心〞，但内部正在逐渐透露出越来越多恐慌的迹象。现在彭斯的演讲火力远超贸易战，其影响力会传导股市，自然也会传导其他方面，而且影响时间不会只一时半刻。中共确实感到恐慌，所以拼命封锁狂删。
 <br/>
 <br/>
 但彭斯演讲影响力是删除不掉的，而且还会继续发酵，而他以一系列事实指控中共破坏现存国际秩序，并指出人类如何再往前推进的方向，对此舆论多有解读，没有中共就是最好的保障中国和世界。正如彭斯演讲结语：〝相信美国人民和中国人民之间的持久友谊〞、〝美国和中国将共同迎接未来〞。相信中国、中国人民。中共就让它尽快消失在未来吧。
 <br/>
 <br/>

 <br/>
