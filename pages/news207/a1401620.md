### 臧山：川习会成果抵不住全面冲突
------------------------

<div class="wysiwyg">
 在阿根廷的川普（特朗普）和习近平会晤，受到了特别的关注。我感觉，在这次双边会晤中，美、中双方不可能达成完美妥协，但也不可能彻底谈崩。最可能的结果是在这两个结果之间的某一点上达成妥协意向，然后公布联合声明。另一种可能是谈完后双方各说各话，届时便要看情况如何了。
 <br/>
 <br/>
 11月初，中共向美国提出了一份书面的142项贸易改革方案，希望可以和美国达成妥协。不过，应中共的要求，这一份方案至今未公布于众，外界只能通过美官员的只字片语来管中窥豹。不过根据美国贸易官员的说法，这142项的措施，基本都是过去两年中共官方已经承诺过的东西，而且缺乏美方要求的落实时间框架，也缺乏具体执行机制。
 <br/>
 <br/>
 感觉上，美方不会接受一个没有时间限制和没有执行机制的所谓贸易让步，因为在过去20年，中共对美方的类似承诺多如牛毛，甚至连中共在美国的坚定盟友——华尔街投资银行——也已经感到十分不耐烦。
 <br/>
 <br/>
 不过，中共在宣传上的创意却层出不穷。中共提出〝反对单边主义〞、反对〝贸易霸凌〞、反对〝贸易保护主义〞等等。霸凌容易理解，就是美国人借助自己体量巨大，欺负中国（和其它国家）。所谓单边主义，是与WTO等多边贸易机制相对而言的。中共认为，所有的规则和争端裁决，都应该在世贸组织这样的多边国际组织内解决。
 <br/>
 <br/>
 然而，中共并不真正尊重〝多边机制〞。首先中国加入世贸之后，至今17年尚未落实当年入世承诺，世贸这个〝多边〞对中共从无任何实质约束。
 <br/>
 <br/>
 其次，中共历来也都以〝中国特色〞躲避国际通用准则和普世价值，无论是经济贸易还是政治文化，中共都更为强调自我特色。如果说到贸易霸凌，中共更是最常采用的国家。
 <br/>
 <br/>
 中国周边国家和地区，几乎每一个都曾经被中共以经济贸易手段制裁过。日本、南韩、台湾、菲律宾、越南、菲律宾、新加坡和印度，都在最近几年经历过中共制裁。不同的是，中共并不需要像美国一样通过立法来进行制裁，而是通过行政措施，大多数的时候是做而不说，甚至在公开场合不承认〝制裁〞而已。
 <br/>
 <br/>
 我们从2018年初就已经分析过中、美两国的纠纷绝非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易冲突.html" target="_blank">
  贸易冲突
 </a>
 这么简单。目前，美国对中国的指责，最少提出了六大类。
 <br/>
 <br/>
 一、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不公平贸易.html" target="_blank">
  不公平贸易
 </a>
 措施，包括超额贸易顺差、高关税、非关税壁垒、国家贸易补贴等；
 <br/>
 <br/>
 二、知识产权盗窃，强迫外国企业转让核心技术，以电脑黑客方式盗窃美技术机密，针对包括中国制造2025计划；
 <br/>
 <br/>
 三、对商业机构进行黑客攻击（留意，不包括政府和军事机构）；
 <br/>
 <br/>
 四、全面社会政治渗透。11月27日美国参议院举行安全策略听证会，首次提出了中国对美国政治作战的说法，更把孔子学院列为〝政战工具〞；
 <br/>
 <br/>
 五、南中国海军事化；
 <br/>
 <br/>
 六、在全球军事布局，削弱美国军事优势，这里针对包括一带一路的中国全球布局，以及台湾安全。
 <br/>
 <br/>
 目前美、中较量的集中点在第一项，但经贸纠纷只是一个冲突的引发点而已。美、中纠纷在这个点上引发，显然对中共并不有利。这也是中共官方咬紧牙关保持低调的原因。
 <br/>
 <br/>
