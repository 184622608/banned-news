### 马云话音刚落 诡异事件发生 中国富豪的命运何去何从？
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马云.html" target="_blank">
  马云
 </a>
 话音刚落 诡异事件发生就在马云谈到〝我认为我们不应该想...控制自己，如果它对社会的未来有益... 〞时，发生了什么诡异事件？
 <br/>
 <br/>
 近期，中国互联网巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里.html" target="_blank">
  阿里
 </a>
 巴巴在浙江杭州举办〝2018杭州云栖大会〞，阿里巴巴创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马云.html" target="_blank">
  马云
 </a>
 出席。
 <br/>
 <br/>
 《日本经济新闻》报导说，才（宣布将）卸任阿里巴巴董事局主席一职的马云在大会成为外界焦点，面对现场提问〝中国（中共）政府可以容忍阿里巴巴发展到什么地步〞。
 <br/>
 <br/>
 马云说，〝网路就像电力，是没有边界的〞〝我们正进入一段新的时期，新的纪元〞。
 <br/>
 <br/>
 然而，就在马云谈到〝我认为我们不应该想...控制自己。如果它对社会的未来有益... 〞时，直播画面突然被切换成时钟图像再转为黑屏，声音也被播放的音乐所取代。
 <br/>
 <br/>
 对此，阿里巴巴集团发言人称，阿里巴巴问答环节一向只限现场投资人参与，并否认此事与自我审查有关。
 <br/>
 <br/>
 不少人认为：马云毫无征兆的突然退隐，是因与中共存有矛盾；也有分析说：马云隐退大背景，是中国经济环境遇到阵痛，国进民退的大趋势，以及国家对企业的缰绳收紧。还有人认为：马云的选择是明智之举，除了全身而退，还可避免，成为被中共宰割的〝肥羊〞。
 <br/>
 <br/>
 陆媒9月14日爆料，阿里旗下的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支付宝.html" target="_blank">
  支付宝
 </a>
 ，在马云宣布辞职当天即被中共官方的中国银联〝收编〞。
 <br/>
 <br/>
 2018年1月，中共曾以〝蚂蚁金服〞没有保护好顾客的个人资讯为由，下令全面调整相关系统。
 <br/>
 <br/>
 阿里巴巴今年确实受到了很多来自中共的管制，以至于阿里在基础、金融服务市场连连受挫。
 <br/>
 <br/>
 尽管如此，马云治下的阿里巴巴还是不断回应中南海的要求，不仅加强在〝一带一路〞，沿线国家的投资，而且还在芯元领域加大投入。
 <br/>
 <br/>
 香港中文大学经济学系副教授庄太量认为，中国的政治环境一方面给互联网公司的发展带来不确定性，但也并非全无好处。马云等新一批互联网富豪，之所以有今天，其实很大程度上都是因为中国政府在国家层面上阻止了国外的竞争者，通过防火墙让他们做成〝独市生意〞。
 <br/>
 <br/>
 从某种程度上来讲，可以说中国政府是游戏规则的制定者——政府想谁发达，谁就发达。
 <br/>
 <br/>
 面对强势的中国政府，草创的民营企业选择〝合谋〞。马云迎合中共的做法，被认为是在中共治下成功企业家的成功之道。
 <br/>
 <br/>
 2013年马云在一次演讲中说：〝不管阿里巴巴发展多快，也绝不与政府做生意〞，但是，到2014年5月15日，阿里巴巴就不只是跟政府谈恋爱了。马云和时任浙江省长李强签署了十个方面的〝战略合作〞。其中包括发展阿里云及大数据等高科技产业。
 <br/>
 <br/>
 马云并没有让高科技只停留在企业民间运用。据〝阿里云〞网站2016年10月26日报导，马云受邀为百万政法干警讲〝阿里经验〞：大数据为社会治理，构建预防机制。
 <br/>
 <br/>
 时任中央政治局委员、中央政法委书记孟建柱现场主持讲座；公安部部长郭声琨，最高人民法院院长周强、最高人民检察院检察长曹建明等，分别在主会场或分会场听取讲座；全国政法综治系统，除办案、执勤、值班之外的150多万干警，也通过视频系统在各分会场听讲。
 <br/>
 <br/>
 中共政法委如此高调肯定〝阿里经验〞无非是想借〝阿里经验〞打造数字化社会，提升管制社会的手段和效果达到严密监控全社会的目的。
