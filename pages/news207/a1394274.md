### 陈思敏：中共年内二度发行美元债券有内幕
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 财政部10月3日宣布计划本月在香港发行30亿的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美元债券.html" target="_blank">
  美元债券
 </a>
 ，而这是年内第二次发行美元计价主权债券。
 <br/>
 <br/>
 年内第一次是去年（2017）10月27日在香港发行20亿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美元债券.html" target="_blank">
  美元债券
 </a>
 ，时值
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 十九大闭幕之际，财政部官员代表在香港发行仪式上称，财政〝不差钱〞，发美元债券展现〝进入国际金融市场的决心〞、〝彰显市场影响力〞。受此影响市场也给予评估，中共在十九大之后，会进一步落实经济改革措施。
 <br/>
 <br/>
 而这次发行美元债券，除了距离前一次时间短，发行额度上调之外，同时也因当前贸易战没有缓和迹象，中国经济增速放缓，投资趋冷，市场以筹措资金解读。
 <br/>
 <br/>
 事实上，中共这次发行美元债券30亿美元并不是多大的数目，所以纵使缺钱但融资并非最主要目地，就时机而言，是要测试或者说重演上次发行〝盛况〞汇集市场信心，以应付中美贸易战升级，同时稳住下周（10月8日）A股收假开市后随之而来的股市动荡。
 <br/>
 <br/>
 沪深两市在十一期间休市，但中共财金官员可能都过得提心吊胆，因为香港股市10月2日起恢复交易，2日至5日四个交易日连四黑，恒指跌去4.38%，国指（H股、国企股）跌去4.43%，整个〝黄金周〞港股从头跌到尾。
 <br/>
 <br/>
 而向来与美股连动性强的香港股市，这次不仅与美股走势分歧，而且出人意外暴跌，除5日是有联想、中兴等H股因恶意晶片事件加深跌幅外，投资界分析，资金在〝趋吉避凶〞，资金转向基本面较佳的美股市场导致港股惨跌，也就是投资人对中美贸易战的预期，普遍支持持续走多的美国股汇指数。
 <br/>
 <br/>
 市场最难得的从来不是资金，而是信心。在贸易战阴云下，连香港股市都开始失去投资人的信心，何况本来就令人信心薄弱的A股。
 <br/>
 <br/>
 去年10月，中共财政部在香港发行20亿美元债券，官媒报导〝盛况〞称，20亿美元额度，却接到上百亿订单，而且是在没有任何评级机构信用评级的情况下发行美元债，当时从财政部门高层到国有大行高管齐称，这是充分显示了〝全球市场对中国经济充满信心〞，〝国际投资者对中国经济发展的信心〞。
 <br/>
 <br/>
 债券发售之后，据IFR报导，摩根大通的新兴市场债券指数（EMBI）会将其纳入。这意味着许多新兴市场基金为了追踪指标指数，也不得不购买，尽管这批债券没有信用评级。另据称，原本是面向境外投资者的美元债券，其实不少认购者来自国内。
 <br/>
 <br/>
 所以中共这次发行美元债券30亿元，将于10月9日在香港举行的固定收益投资者说明会，想〝重演〞去年发行〝盛况〞并不难。但是，根据彭博今年1月9日标题为〝China’s Landmark Sovereign Dollar Bond Deal Leaves Bitter Taste〞报导，中共这20亿美元主权债2017年10月发行，2017年11月、12月两个月就让投资者亏损0.73%，而在亚洲国家当中同期只有少数几支政府债券是负回报。
 <br/>
 <br/>
 务实而论，市场的信心就是中共自己败坏的，所以无论做什么都很难提振。香港股市这周交易天天收黑，美国副总统彭斯震撼演讲，下周即将收假开市的A股雪上加霜，走势如何还是要看8日到访北京的美国国务卿蓬佩奥会否带来利多的消息。
 <br/>
 <br/>
