### 文武：中共内斗已呈你死我活的态势
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内斗.html" target="_blank">
  内斗
 </a>
 是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的传统，不过过去中共的内斗还有章法可寻，现在中共内斗已经完全没有章法了，只剩下不是你死就是我活，只看谁能搞死谁。
 <br/>
 <br/>
 目前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 体制内的高官们，已经不把中共体制的这条船当成他们共同所有的了，已经没有人再去管这条船了，因为都知道这条船要沉了，只看谁能逃得命去。
 <br/>
 <br/>
 中共十九大后确立了市场的主体地位，确立了经济市场化的目标，可是半年多过去了，没有人去落实什么市场不市场，什么经济改革不改革。今年年初中共就布置打黑除恶，并用将打黑除恶作为中共当前管理社会的重要抓手在推行，可是半年多过去了，也是只见打雷不见下雨，因为没有人去执行，所以落实不下去。其实，什么老兵问题，什么政府机构重组的问题，全是只有雷声没有雨点。
 <br/>
 <br/>
 就是说现在中共内部已经没有人真的去干事了，要么是等着中共垮台，要么就是想着整死对方。
 <br/>
 <br/>
 所以现在新闻上出现最多的就是今天这个高官突然死了，明天那个高官突然被抓了，经报导出来的就一个接着一个，没有报导出来的不知道还有多少，高官死了或者被抓了有人知道，低官死了或者被抓了好多没有报导，甚至无人知晓。
 <br/>
 <br/>
 最近，最有名的几个高官事件，一个是海航的董事长王健突然在法国摔死了，一个是公安部副部长国际刑警组织主席孟宏伟，从法国回中国时突然失踪了，一个是中央委员澳门中联办主任突然从中联办大楼里死了。因为这几次事件国际影响比较大，所以中共压也压不住，就被爆光出来了。
 <br/>
 <br/>
 中共高层因为忙着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内斗.html" target="_blank">
  内斗
 </a>
 ，这样经济也就没有人管了，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 暴跌，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_楼市.html" target="_blank">
  楼市
 </a>
 下跌，消费低迷。
 <br/>
 <br/>
 就是与美国打贸易战也不见一个人出来说声，谁是贸易战的总指挥，没有一个人愿意出头，全都避而远之，正经的事全没有人去管。
 <br/>
 <br/>
 中国有近十四亿人，是世界上人口最多的国家，是世界第二大经济体，可是目前在台上执政的却是个中共邪恶的政治流氓集团，这群流氓，看到中共这条船快要沉了，于是个个都是想着怎么整死别人、保住自己，为自己找救生圈，抢夺别人的救生圈，想淹死别人、保住自己，所以才出来今天这个死了，明天那个死了。
 <br/>
 <br/>
 其实中共的这些人，不
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抛弃中共.html" target="_blank">
  抛弃中共
 </a>
 谁都得死。
 <br/>
 <br/>
 <br/>
 文武
 <br/>
 2018年10月31日星期三
</div>
