### 为什么有人读书很优秀，工作却很差劲
------------------------

<div class="wysiwyg">
 学校是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_学习.html" target="_blank">
  学习
 </a>
 的地方，而学校只是社会的一部分。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_读书.html" target="_blank">
  读书
 </a>
 (
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_学习.html" target="_blank">
  学习
 </a>
 )是为了什么？这个问题很重要！然而！这么重要的问题学校却没有给你明确的答案，说不定还把你弄糊涂了。
 <br/>
 <br/>
 读书长知识，知识沉淀了变成智慧，读书的本质是增长智慧，智慧就像内功，内功深厚了学什么功夫都快，智慧有了出来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工作.html" target="_blank">
  工作
 </a>
 得心应手，智慧有了就能识辩善与恶、是与非，智慧大了就能洞释天地之规律，兴盛永存。
 <br/>
 <br/>
 浅白的语言说，读书为了什么？读书的本质有三层楼：第一楼：追求物欲；第二楼：追求文明；第三楼：灵魂觉醒。
