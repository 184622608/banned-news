### 文武：中共在投降中有哪两条是中共做不到的
------------------------

<div class="wysiwyg">
 11月16日美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在白宫说，中共请求停止中美之间的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，请问美国撤消对中共的贸易制裁，并且拉出142条的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清单.html" target="_blank">
  清单
 </a>
 来请美国审阅。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 还说，中共有所表现，但是还不够，还有关键的几条中共还没有列出来。
 <br/>
 <br/>
 只有中共让美国满意了，美国才愿意与中共谈判，至于谈的结果如何那得看中共是不是真的改正了。
 <br/>
 <br/>
 中共尽管拉出142条
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清单.html" target="_blank">
  清单
 </a>
 来，尽管我们还不知道142条清单中的具体内容是什么，但是从美国要求中共撤销关税和非关税壁垒来看，起码有两条中共是做不到的，一条是撤销互联网的封锁，也就是撤掉中共所设的互联网防火墙；再一个就是中共撤销外汇管制，允许外汇自由进出。这两条中共无论如何都是做不到的。
 <br/>
 <br/>
 中共能够维持对中国人民的统治，中共能够欺骗中国民众，靠的就是迫害、谎言加封锁。
 <br/>
 <br/>
 中共一边迫害敢于反抗中共的民众，一边又编织谎言欺骗中国人，而中共能够维持谎言的伎俩就是封锁，中共一边在国内封锁不同的声音，一边高筑互联网防火墙，隔断国外的声音，把真阻隔在国门之外，让中共的谎言在国内畅行。
 <br/>
 <br/>
 如果中共撤销了防火墙，那么国外的谷歌、推特、脸书、动态网、自由门、无界、大纪元、新唐人、明慧网、正见网等等一大批自由言论的网络媒体，就会在中国大陆流行，中国民众就会接触到真实的声音，就会知道真相，中共所有的谎言就会被全面揭穿。那中共靠谎言统治的基础就被掀翻了，中共也就不复存在了。
 <br/>
 <br/>
 中共第二个不能改的就是撤销外汇管制，中共的经济是泡沫经济，全是债务，目前能够维持经济还能运行的唯一原因，就是外汇管制。一旦外汇管制放开了，外汇可以自由流动了，那么进入中国市场的外资就会迅速大量的撤离，因为美国在升息，而中国的经济在下滑，外资在中国市场已经无利可图了，正好乘着美元升息的机会，美元就会迅速流回美国。
 <br/>
 <br/>
 中共在经济领域最缺少的一个是粮食、一个是能源、一个是关键技术产品。而这些全都得靠外汇来购买，一旦没有外汇了，没有美元了，那中共经济就会彻底崩盘。
 <br/>
 <br/>
 所以这两条中共是绝对做不到的。但是中共做不到，又想要美国停止对中共的关税制裁，而美国停止对中共制裁的前提条件就是中共要消除这些壁垒。
 <br/>
 <br/>
 这样看来，不说停止中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，恐怕中美之间坐下来谈的可能性都很小。
 <br/>
 <br/>
 现在中共是想谈、想妥协、想投降，但是又改变不了自己的恶行。这样美国也就无法与中共谈了，所以美国说，中共还没有准备好。
 <br/>
 <br/>
 接下来中共可能还是想要行骗，采取一些阴谋的手法，骗得美国放弃对中共的制裁。
 <br/>
 <br/>
 但是，川普政府不是原来奥巴马的政府，中共想欺骗川普是欺骗不了的，想欺骗美国人民更是欺骗不了了，看来中共是无招了，只能是混一天算一天了。
 <br/>
 <br/>

 <br/>
 <br/>
