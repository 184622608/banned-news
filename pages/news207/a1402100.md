### 文武：中共这次算是认怂了
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 发生以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 开始是牙被打掉了嘴还硬，死不服输，死不
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_认怂.html" target="_blank">
  认怂
 </a>
 ，死不投降。可是这次在阿根廷G20峰会上，与美国谈判时，突然服软了，突然认怂了。
 <br/>
 <br/>
 原因是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 与美国正面打贸易战，本来就不是中共的本意，中共本来就不想、也不敢与美国正面交战，但是中共邪恶的本性决定了，必须吓唬吓唬人。
 <br/>
 <br/>
 美国指出中共不公平的贸易行为，中共的盗窃行为，本来就是铁的事实，但是在事实面前中共就是不承认，明明是个婊子，偏偏要说自己是个贞女。
 <br/>
 <br/>
 中共为了给自己挣个贞洁的名头，就硬着头皮与美国打贸易战，可是打着打着发现自己越打越原形暴露，不仅没有掩盖住自己，反而更让世人认清了中共是个婊子。
 <br/>
 <br/>
 中共一看这样不行了，成天高喊维护人类命运共同体，成天高喊全球一体化，成天叫着反对贸易保护主义，其结果是越喊越叫，越显露出中共就是为害人类命运共同体的祸害。中共的调门越高，中共的原形也就越毕露，中共的处境也就越艰难，在这种情况下，中共才彻底的知道了，骗不下去了，吓唬不住对方了，美国是来真的了，再这样下去，中共就支撑不下去了。
 <br/>
 <br/>
 于是，这次中共是真的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_认怂.html" target="_blank">
  认怂
 </a>
 了。
 <br/>
 <br/>
 中共首先请求美国给它三个月的宽限期，做为回报，中共立刻购买美国大量的能源、农产品和工业品，中共立即管控芬太尼的生产与流通。
 <br/>
 <br/>
 在这三个月的宽限期内，中共要切实认真的与美国谈判，主要解决中共有关强制技术转让，盗窃智慧知识产权，网络入侵，政府补贴等结构性的问题。
 <br/>
 <br/>
 就是说，只要美国在贸易领域能够出来的条件，中共全都答应，美国要怎么样、中共就怎么样，中共绝不再阳奉阴为了。
 <br/>
 <br/>
 中共怎么突然一下变了呢？怎么突然一下变得如此服从了呢？难道是中共变好了吗？难道是中共改恶从善了吗？难道真的是婊子从良了吗？
 <br/>
 <br/>
 其实，全不是，这是中共的本性决定的，中共的本性决定了中共是个欺软怕硬的货色。
 <br/>
 <br/>
 原来国际社会普遍软弱，中共就显得不可一世，谁都不敢得罪中共，谁都怕中共，哪个国家的政府都不敢得罪中共，结果一次又一次的被中共占便宜，一次又一次的被中共欺负。
 <br/>
 <br/>
 可是，这一次的美国政府，第一次站出来了，对中共说不，对中共强硬，必须要打败中共，必须要让中共服软，必须要迫使中共认怂，用美国副总统彭斯在APEC峰会上，给中共发出的最后通牒的话来说，中共必须在阿根廷G20峰会上表态认输，否则美国将与中共发生全面的冷战。
 <br/>
 <br/>
 美国政府上下一心的决心，终于让中共看到了，什么是正义的力量，什么是真正的力量，于是，中共就败下阵来了。
 <br/>
 <br/>
 这不，这几天中共就真的认真执行起美国的决定来了。
 <br/>
 <br/>
 就是说中共真的认怂了，原来从不说的芬太尼的事情，中共也开始说了，原来准备暂停给国营企业去产能的事情，这几天也开始有动作了，十几个部委一起发文， 要坚决让僵尸企业破产，不允许任何人去保护国营企业中的僵尸企业。中共原来宣传要与美国打到底的语气也变调了，变成要与美国和好了，要与美国世世代代的友好下去了。
 <br/>
 <br/>
 现在中共是真的认怂了，而不是中共真的变好了，中共永远都不可能变好，它要祸害美国，祸害世界，祸害人类的本性永远都不可能变。
 <br/>
 <br/>
 所以，对付中共唯一的办法就是一直打下去，直到中共彻底解体。
 <br/>
 <br/>

 <br/>
