### 文武：万科说要〝活下去〞的启示
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万科.html" target="_blank">
  万科
 </a>
 是中国大陆最大的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房地产.html" target="_blank">
  房地产
 </a>
 企业，在人们的印象中，万科有的是房子，因而就应该有的是钱，钱多还说什么活不活的。可是这一次万科喊出的口号却是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活下去.html" target="_blank">
  活下去
 </a>
 〞。
 <br/>
 <br/>
 可见人世间的事，是多么的变幻莫测，不说沧海桑田，就说前天哪个是大款，明天也许就不知道口袋里还有几个钱。
 <br/>
 <br/>
 就拿安邦保险的老板吴小晖来说吧，才看到安邦保险的生意做得多大多大的，几千个亿，几万个亿，说是吴小晖自己个人就有千亿身价，还曾经是邓小平的孙女婿，有钱有势的，可是不出几天，突然传来个消息，说吴小晖被抓了，进监狱了，坐牢了。吴小晖打造的所谓的安帮帝国被没收了，被国有化，一夜之间，从富可敌国的富豪，一下子成了个穷光蛋。
 <br/>
 <br/>
 同样的所谓的大富豪肖建华，说是他的明天系掌握了多少公司，能够控制多少万亿的资金，钱多得数不可数，每天晚上翻牌挑选陪他睡的女人，夜生活胜过昔日的皇帝，因为原来的皇帝选妃那是有严格的规矩的，今天的肖建华不用什么规矩就能选妃，他不是比原来的皇帝更自由些吗？
 <br/>
 <br/>
 中共里面的商人也好，官员也好，这种一会升天、一会下地的巨变，就算是个正常人恐怕也变得不正常了。
 <br/>
 <br/>
 其实人有多少钱又如何呢？钱不过是身外之物，生带不来，死带不去，通过自己诚实的劳动挣得够生活的费用就可以了，够吃、够穿、够生活所需，有一份安稳的工作，有一个安稳的家，有一个良好的工作、生活的环境，过着人一样的正常生活，安居乐业，平稳度日，这样社会不就太平了吗？
 <br/>
 <br/>
 可是偏偏出来了个中共，跑来抢占了我们的家园，迫使中国人过着这种人不人、鬼不鬼的生活。
 <br/>
 <br/>
 中共要么谁都不让挣钱，要无产阶级的〝道德〞，要又红又专，要艰苦朴素，要越穷越光荣。
 <br/>
 <br/>
 搞得中国人谁都没有钱，买米都得靠中共发的粮票，房子一家只有一间，男的、女的、老的、少的一家人，几代人挤在一间房子里。衣服都一个样，连颜色也没有几个，而且还不够穿，有时一条裤子穿几代人。
 <br/>
 <br/>
 到后来，中共又突然来了个所谓改革开放，一切向钱，笑贫不笑娼，只要能弄到钱，不管是贪来的、骗来的、盗来的、抢来的，只要是能把钱弄来，这个人就有本领，这个人就被人佩服，中共一下子又把中国人，把中国社会推向了另一个极端。
 <br/>
 <br/>
 于是，一些中国人在中共的驱使下，像发了疯似的一样去弄钱，去享乐，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房地产.html" target="_blank">
  房地产
 </a>
 就是一个简单的例子。
 <br/>
 <br/>
 先是想要一套房子，后来想要两套，先是想要几十平米，后来想要几百平米。没有钱的人想办法弄钱买房子，有钱的人买了一套还想来一套。
 <br/>
 <br/>
 特别是一些中共的贪官，买一套又一套，听说还有的贪官有几十套、几百套的房子。
 <br/>
 <br/>
 于是，中国的房子是越建越多，可是价格却还越来越高，为什么呢？因为有些中国人，不是说买房子去住，而是买房子去炒，就是把房地产市场当成了赌场，在里面寻求一夜发财的梦想。
 <br/>
 <br/>
 这种被中共搞乱的了社会，这种被中共搞乱了的房地产，谁在里面谁会得到善果呢？
 <br/>
 <br/>
 眼看着现在中共的房地产快要完蛋了，所以，才有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万科.html" target="_blank">
  万科
 </a>
 喊出要〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活下去.html" target="_blank">
  活下去
 </a>
 〞的口号。
 <br/>
 <br/>
 其实房子没有了，钱没有了，这也不一定是最坏的事，最坏的事是命没有了。
 <br/>
 <br/>
 当人们还有钱、还有势的时候，人们发现不了自己贪心的害处，当这些都失去了时候，有些聪明的人会总结出经验教训来，知道是因为自己的贪心才造成自己财物的损失，也许这时改掉自己的贪心，去掉自己的贪心，还能回归到人的正常生活上来。
 <br/>
 <br/>
 可是，当人的命还在的时候，人们认识不到自己恶念的害处，那等命没有了的时候，就没有总结经验教训的机会了，因为命已经没有了，再聪明、再会总结也没有用了。
 <br/>
 <br/>
 所以，保命比保财更关键、更重要。
 <br/>
 <br/>
 那什么是恶念呢?
 <br/>
 <br/>
 今天的善恶标准很简单，跟着中共跑就是恶，相信中共就是恶，认为中共是好的就是恶念，脱离中共就是善。
 <br/>
 <br/>
 三退了，命保住了，才真的叫〝活下去〞
 <br/>
 <br/>
 文武
 <br/>
 2018年10月4日星期四
