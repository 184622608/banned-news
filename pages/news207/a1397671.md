### 千百度：国保警察为何成大法弟子？
------------------------

<div class="wysiwyg">
 在大陆，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国保警察.html" target="_blank">
  国保警察
 </a>
 是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 和中共镇压法轮功的主要打手。但令人欣喜的是，通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法弟子.html" target="_blank">
  大法弟子
 </a>
 坚持不懈的努力，他们有些良知尚存的人终于明白了中共迫害法轮功的真相，有的人不仅主动保护大法弟子，甚至自己也成为了大法弟子。下面就是明慧网最近报导的，一位大法弟子如何通过讲真相，让原来迫害他的国保警察弃恶从善，最终也成为大法弟子的感人故事。
 <br/>
 <br/>
 一九九九年人权恶棍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 迫害法轮功一开始，某县公安一科（也就是原来的政保科，现在的国保大队）的警察按照中共邪党指令，对该县挂名的法轮功学员，实行分工包干负责制。
 <br/>
 <br/>
 当地有位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法弟子.html" target="_blank">
  大法弟子
 </a>
 在政府机关工作，因为她是国家干部身分，所以具体负责她的人是一科指导员。一开始，这个指导员听信了邪党的谎言宣传，来势汹汹，三天两头带一帮警察到她单位拍桌子、摔椅子，胁迫单位领导对她施压迫害，搞跟踪监控，进行政治和经济惩罚，开批斗会，责令转化，还实行连坐，并组织六一零和政法委人员围攻谩骂她，扬言要撤销她的职务和工职，并株连她的丈夫和儿女，严重妨碍了她的正常工作和生活。
 <br/>
 <br/>
 但无论他们态度怎样，这位大法弟子都不动心，不但不动心，还慈悲的向他们讲述法轮大法的美好和祛病健身的神奇，大法弟子做好人好事的实例，大法洪传全世界的情况，启发他们的善念良知。同时，她也讲宪法三十六条规定公民有信仰自由。是江魔头出于个人妒嫉，在中央六个常委都不同意的情况下，一意孤行的实施镇压，迫害上亿做好人的修炼人。
 <br/>
 <br/>
 接触时间一长，指导员感受到了大法弟子的善良。与此同时，单位领导和同事全都跟他说这位大法弟子是好人，工作竞竞业业，年年被评为优秀公务员，原来身体很差，学法轮功后身体好了，人脾气也好了。在事实面前，他的思想也发生了转变。原来来找她都是直呼其名，后来改口叫大姐了，还主动向她了解大法真相和炼功情况。她就在领导办公室炼五套功法给他看，他还跟着比划。
 <br/>
 <br/>
 再后来，她送给他一本《九评共产党》。他拿回去后，当天吃饭也看，睡觉也看，用了一天一夜把书看完了。他老婆问他看什么书，这么认真，是不是要考研究生啊。看完后，他对大法弟子说：《九评共产党》写得真好，对我震动很大。
 <br/>
 <br/>
 她不断的拿真相资料给他看，使他彻底转变了观念，觉醒了。他开始在自己职责范围内保护大法弟子。每次六一零布置迫害法轮功时，他都想办法提前告诉大法弟子；如遇到有人举报时，他就说：法轮功都是好人，都是江泽民搞的鬼，别太认真，马虎点算了。有一次，抓到一个发资料的法轮功学员，他让那个法轮功学员在办公室坐了两个小时就把人放了。还有一次，六一零让他出警抓法轮功学员，他直接抵制说：没有人，我们的事都做不完，哪有时间管那些事呢！
 <br/>
 <br/>
 不仅如此，他还主动传播真相资料，每次有新的资料、光碟和小册子大法弟子都会给他送二十多本，他就把这些资料放在单位食堂的饭桌上，说是收来的，故意让来吃饭的警察拿去看，掩饰的说看完了给我送来啊。其实没有人看完送过来的。他跟大法弟子说：现在公安内部有百分之九十的人都看过《九评》和真相资料了，都明白是怎么回事了。六一零和政法委的人来，他都把真相资料放在桌面上堆着，让他们一本接一本的看。有一任六一零主任明白真相后辞职不干了，去做生意去了。近几年，该县再没有出现迫害法轮功的档。
 <br/>
 <br/>
 二零一一年，他让大法弟子给他请了一套大法书 ，开始修炼大法，至今已有七年了。他不仅自己修炼，还积极向同事洪法。如今通过他洪法走入修炼的公安同事已有六个人，而且他们都做了三退声明。有一次，他下乡办案时，碰到一个老熟人，见那人病得很重，很痛苦的样子，他说：我有一个好法子能解除你的痛苦，你就诚心念诵：〝法轮大法好，真善忍好〞，就有奇效。下星期他再去的时候，那个熟人高兴地对他说，你走后，我就不停地念，第二天就好了，确实有奇效。同时又说：这么好的法，你怎么不早告诉我呢？
 <br/>
 <br/>
 一到星期天，他就到农村亲戚家到处走走，主要是讲真相救人。有人问他：你是公安干部，怎么也信法轮功呢？他回答说：在铁的事实面前，不得不信哪。谁信谁受益。
 <br/>
 <br/>
 <br/>
