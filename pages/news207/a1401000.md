 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">

 人作为全世界最优秀的人种！不同的宗教寺庙隔街相望，和平发展，而其它国家却为了宗教而战争！
 <br/>
 <br/>
 中国人作为全世界最优秀的人种！正教信仰被迫害，被活摘，面对这个星球最残忍的罪恶，正信群体非暴力反迫害讲真相唤醒世人，而其它国家的人的宗教领袖被侮辱了却绑上炸药上街。
 <br/>
 <br/>
 中国人作为全世界最优秀的人种！在几千年来历经无数劫难走过来了；在近代西来共产幽灵入侵，沦陷的几十年来民族精英被属魔的人赶尽杀绝，传统文化遭灭顶之灾依然一息尚存星火燃原，而其它民族遭遇劫难灰飞烟灭甚至连人种都灭亡了。
 <br/>
 <br/>
 中国人种之所以优秀，不是一朝一夕形成的，源自在5千年文明中千古轮回里的道德品质的沉淀而深嵌灵魂的修养。
 <br/>
 <br/>
 但是！ 《礼》曰：〝华夏，重信义！！ 而胡人，无！！〞
 <br/>
 <br/>
 1949年西来幽灵入侵的中国是沦陷的〝国〞，神洲沦陷为〝魔洲〞，国人道德沦丧，诚信崩溃！对自己民族辉煌的历史一无所知，更被垃圾品种编的几部电视剧丑化而改变了，这还是意义上的中国人吗？马列变种在神州横行，中国还是中国吗？
 <br/>
 <br/>
 古人云：〝三军可夺帅也，匹夫不可夺志也〞。如今沦陷的中国人连尊严都没有，连尊严被剥夺了都没意识到！甚至连提及〝尊严〞这个名词都被共产党魔鬼当成敌人，这还是意义上的中国人吗？共魔之下只有奴民。而不同的是有人自愿当奴民认贼作父，骂祖先，丑化侮辱中国人种。
 <br/>
 <br/>
 有人把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 作恶与古代王朝制相谈并论，实属无知！中国古代（包括民国），你的祖先是信天敬神的，有尊严的，有底线的，土地房子财产是自己的，民间开银号，没有城管，税赋低，自由迁徙，朝廷多以无为而治。
 <br/>
 <br/>
 古代是人的文化，人的行为；而中共是魔鬼文化，魔鬼行为。共产党进入中国一根竹杆插到底村村都有党支部，摧毁了中国几千年建立的道德体系，把礼仪之邦、文明之邦的中华民族变成垃圾之邦。
 <br/>
 <br/>
 中共≠中国，中共：500红色家族，黄皮俄心，祖国苏俄，迫害杀戮中国人。中国：这片土地和这片土地上生长的人。
 <br/>
 <br/>
 中共人≠中国人，中共人：黄皮俄心，祖国苏俄，无神（反神）论，以迫害中国人为乐。中国人：黄皮肤中国心，信天敬神，以帮助中国人为豪！
