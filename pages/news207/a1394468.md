### 夏小强：国际刑警组织主席落马说明了两件事
------------------------

<div class="wysiwyg">
 2018年10月7日晚，中共中央纪委国家监委网站宣布，公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 涉嫌违法，目前正接受国家监委监察调查。同一天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 表示，孟宏伟已辞去该组织的主席职务，即时生效。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席，在突然人间蒸发失踪多日后有了消息。一个国际专门打击犯罪的最大合作组织的掌门人，竟然被中纪委当作犯罪嫌疑人抓了起来，真是一件中共创造的现代奇观。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 在9月29日被召回北京失踪后，孟宏伟的妻子罕见地向法国警方报案，这当然是按照孟宏伟的原计划行事，只有这种方式，引起国际社会关注，增大了保住性命的可能性。
 <br/>
 <br/>
 孟宏伟作为中共前政法委书记周永康的亲信，长期在中共公安部任职。周永康2003年3月出任公安部部长后，孟宏伟2004年4月升任副部长、党委委员，成为周永康的主要助手之一。
 <br/>
 <br/>
 周永康在中共十八后落马后，其多名公安部旧部也被查，包括副部长李东生、常务副部长杨焕宁、政治部主任夏崇源；同时周的多名旧部傅政华、黄明等副部长也被调离。不仅如此，2017年底，孟宏伟突然被免去中共国家海洋局副局长、中共海警局局长的兼职；2018年4月，他又被免去已担任14年的公安部党委委员一职。
 <br/>
 <br/>
 这种种被削权和失势的迹象显示，孟宏伟早就已经处于〝高危〞状态了。所谓国际刑警组织主席的头衔和位置，也无法帮助孟宏伟解除危机。其实，即使孟宏伟当上联合国秘书长，遇到〝我是流氓我怕谁〞的中共，照样会让他人间蒸发。
 <br/>
 <br/>
 如今，孟宏伟作为周永康〝余毒〞被当局清理，虽然最后还是会以贪腐罪名被判刑，但是，未来是否会像薄熙来的前大管家徐明，因为心脏病猝死狱中，还真不好说。
 <br/>
 <br/>
 从中共当局不惜在国际上声誉受损紧急从法国把孟宏伟调回抓捕来看，当局定有难言之隐，可能也是为了避免再次出现第二个王立军。孟宏伟有可能涉入高层权斗甚至传闻甚嚣的北戴河政变，绝非一般贪腐或者选边站队的立场问题。
 <br/>
 <br/>
 孟宏伟在2016年起担任国际刑警组织主席，当时国际特赦组织等人权团体曾警告，把这项职务交给一个来自〝无法确保人权，且政治与司法权不分的国家〞的人物，会带来风险。
 <br/>
 <br/>
 以此观之，国际刑警组织可以让一个人权劣迹斑斑、从上到下官员几乎都是罪犯的中共高官来担当主席，也是一大奇观和讽刺。
 <br/>
 <br/>
 国际刑警组织主席孟宏伟在任上失踪后被中共抓捕，显示出如今中共高层内部政治斗争仍然激烈凶险。习当局上任五年，周永康落马已经四年，成百上千的江派官员被以贪腐罪名治罪落马，但由于江派总指挥江泽民和二号人物曾庆红仍然〝逍遥法外〞，造成中共高层政治形势至今仍然动荡，杀机四伏，危机重重。
 <br/>
 <br/>
 从国际刑警组织主席孟宏伟落马事件中，我们至少可以明白两件事情：第一，在中共体制下，从党的总书记、各级高官、企业家到普通民众，没有一个人是安全的；第二，所有参与过迫害中国民众的中共官员，都正在以各种方式遭到报应，没有一个人可以逃脱。
 <br/>
 <br/>

 <br/>
 <br/>
