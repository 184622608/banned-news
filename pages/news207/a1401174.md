### 千百度：炼功才几天 折磨她半生的病消失了
------------------------

<div class="wysiwyg">
 这是明慧网最近报导的一位大陆法轮功新学员的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_修炼.html" target="_blank">
  修炼
 </a>
 故事。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_修炼.html" target="_blank">
  修炼
 </a>
 前，她身体不好，患有头痛、腿疼、胆息肉、胆囊炎、胃病、失眠，还有许多妇科病，光手术就做了三次。她和丈夫挣的钱还不够买药看病，愁的丈夫老和别人说：你们没听过的病在我老婆身上都经历了。
 <br/>
 <br/>
 由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疾病.html" target="_blank">
  疾病
 </a>
 的折磨，她脸色常常发黄发灰，一脸老相，和比她大八岁的姐出去，人们都说她是姐姐。特别是到了晚上，经常睡不着觉，心里烦躁的像疯了一样，就在院子里转，真是痛苦极了！
 <br/>
 <br/>
 二零零九年，她四十一岁时，父亲过世了。因为疾病加上思念父亲，半年光景她头发就白了。二零一五年，母亲又去世了，本来身体不好的她又添了许多病。再加上家庭的矛盾挫折，使她对人生心灰意冷。
 <br/>
 <br/>
 就在她痛不欲生的时候，一位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_修炼大法.html" target="_blank">
  修炼大法
 </a>
 的同事跟她介绍了自己一家人的修炼故事，并劝她说，你就好好看看《转法轮》吧，这是你摆脱痛苦的唯一方法。
 <br/>
 <br/>
 开始的时候她觉的很玄，没当回事，后来被病魔折磨的太厉害了，妇科病伤口老不好，又添了乳腺增生，又肿又疼。这时那位同事又劝她说，不行我先教你炼功吧，一分钱不花，觉的好就学，不好就还吃你的药。
 <br/>
 <br/>
 她先学了四套功法，晚上炼完功，很快就入睡了。这是她几年来睡的最香的一夜。早晨起来奇迹发生了，她的乳腺增生好了，肿块消失了，不疼了。她高兴极了，大法太神奇了，大法师父太伟大了。
 <br/>
 <br/>
 第二天，她又学了第五套功法，腿一盘就双盘上了，而且第一次就坐了半小时。夜里睡觉感觉一股股热流在腿上涌动，到第四天她三十多年的腿疼病也完全好了，下楼的时候非常轻快。女儿看见她说：妈的腿好了，不用把扶手了。她高兴的说是啊，妈的腿不疼了，师父给妈治好了。
 <br/>
 <br/>
 折磨了自己半生的疾病，没想到师父几天就给拿掉了。亲身感受到大法的神奇，那一刻她泪如泉涌，心里直说谢谢师父，谢谢大法。
 <br/>
 <br/>
 同修接着告诉她，修大法，不仅要做动作，更重要的是学法，这回好好看看《转法轮》吧。于是，她洗净手，怀着激动的心情，恭敬的翻开《转法轮》，开始学法了。越学心里越明白，越学越觉的大法的美好，越学越感到大法的无边内涵。《转法轮》让她明白了人生的真正意义，明白了做人的目地和归宿。为此，她激动万分，泪水控制不住的往下流。这以后，除了做饭睡觉，她有时间就看大法书。
 <br/>
 <br/>
 随着法理的明白、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心性.html" target="_blank">
  心性
 </a>
 的提高，她身体的变化也越来越大，不到半年，在没吃一粒药的情况下，全身的病就完全消失了，不但原本瘦弱的身体变的壮实了，而且脸色红润，人也年轻了。
 <br/>
 <br/>
 认识的她人都说她胖了，变漂亮了。她告诉他们，这都是炼法轮功炼的。她的大哥二姐等许多亲朋好友，因为目睹大法在她身上体现的神奇，也纷纷走入了修炼，整个大家庭基本上都知道了大法的美好。
 <br/>
 <br/>
 回顾自己短暂的修炼经历，她感慨的说：〝虽然修炼才二年，在我和我的家庭中却发生了太多的变化，太多的不可思议，和太多的惊喜。我越来越认识到人生的意义和价值，越来越感受到师父的慈悲伟大，法轮大法的可贵难得。〞
 <br/>
 <br/>
 ──转自《大纪元》本文只代表作者的观点和陈述。
 <br/>
 <br/>
