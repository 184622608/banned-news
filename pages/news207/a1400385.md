### 文武：中美贸易战下一步的走向
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 已经进行近半年的时间了，目前贸易战处于僵持阶段。处于是战还是和的十字路口。
 <br/>
 <br/>
 美国的态度时是，如果中共想停止贸易战，那么就要完全接受美国给出的条件；如果中共不能完全接受美国给出的条件，那么美国就将贸易战扩大，扩大成新的冷战。
 <br/>
 <br/>
 中共的态度是，不想与美国打贸易战，同时也不想向美国投降。想采取一些修修补补的办法，只改变表面的一些做法，不真的去改变中共的体制机制的办法来求得美国停战。
 <br/>
 <br/>
 例如：美国要求中共撤销
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非关税壁垒.html" target="_blank">
  非关税壁垒
 </a>
 ，可是中共只是做个样子，列个清单出来，撤销一些无关紧要的障碍，可是真正的非关税壁垒并没有消除，例如，中共严格的外汇管制措施，中共严格的互联网管制措施，中共严格的农产品保护措施等等这些中共都不愿意完全撤销。
 <br/>
 <br/>
 再如：美国要求中共停止盗窃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_智慧财产权.html" target="_blank">
  智慧财产权
 </a>
 ，停止强制技术转让。可是中共根本就不承认它在盗窃智慧财产权，它在强制技术转让。为了应付一下美国，它只是象征性的发下档，强调一下它在保护智慧财产权。可是美国要求的是中共不光是有相关的法律，而是要使法律得到执行。可是中共自己的宪法中共都不会去真的执行，中共又怎么可能去执行一个什么保护智慧财产权的法律呢？
 <br/>
 <br/>
 又如：美国要求中共停止贸易补贴的行为，停止2025计画，停止一带一路计画。可是中共认为这只是中共的内政，美国无权干涉。但是美国认为，中共的国营企业是政府的企业，在自由市场经济的环境里，美国的私营企业无法与中共的国营企业实现公平竟争，所以中共的国营企业不能参与到对美的贸易中来。中共的2025计画与一带一路计画，都是中共主导的政府直接干预经济的计画，都是破坏自由市场规则的计画，所以，为了公平与自由的贸易，美国坚持中共必须停止类似2025等这类计画。
 <br/>
 <br/>
 其它方面还很多，中美之间的分岐是根本性的，制度性。是不可能通过谈判来解决的。
 <br/>
 <br/>
 但是，中共又请求美国停止贸易战，请求美国取消对中共征收的2,500亿美元的高额关税。
 <br/>
 <br/>
 用川普的话来说，中共非常想跟美国谈，中共非常想美国停止贸易战。
 <br/>
 <br/>
 可是美国的态度也是明确的，就是这个月底，在阿根廷G20峰会上，中共必须做出实质的妥协，就是中共必须彻底投降，必须改变中共经济领域的结构性的问题，否则美国就不与中共达成协议，不仅不与中共达成协议，而且还要增加更多的贸易制裁，不仅要增加更多的贸易制裁，而且还要与中共开展经济，政治，军事等各个领域的全面冷战。
 <br/>
 <br/>
 所以美国说，这个月底G20峰会，是中共最后的机会，过了这个时间点，美国就开始真的动手了。
 <br/>
 <br/>
 中共现在是进也不是，退也不是。进，无力与美国打贸易战，更无力与美国扩大贸易战，更何况与美国发生全面的冷战，中共更是无力应战了。
 <br/>
 <br/>
 退，中共是退无可退，因为美国的要求中共无法做到，中共不可能改变经济结构问题，因为中共的经济结构是中共政治结构的反应，中共的经济体制改变了，中共政治体制就得变，那中共就呆不住了，所以中共决不可能真的改变政治体制，也就决不可能改变经济体制，也就改不了贸易体制。
 <br/>
 <br/>
 这样看来，中美之间的贸易战是停不下来了，接下来中美之间的贸易战会扩大化，会激烈化，直到中共彻底失败为止。
 <br/>
 <br/>

 <br/>
