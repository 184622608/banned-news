### 乐道：川普又出迷踪拳
------------------------

<div class="wysiwyg">
 这几天正是许多围观民众，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在〝川习会〞上对中共示弱的表现大失所望之际，昨日却忽然传来加拿大已在美国要求下实施了对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公主
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 抓捕的消息。此消息一出即成为当下舆论最热点，令本已偃旗息鼓、暂作休兵的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 的剧情陡然逆转。
 <br/>
 <br/>
 恰似笔者前日撰文所分析，擅打
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迷踪拳.html" target="_blank">
  迷踪拳
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ，每一次收拳后都会在中共倏不及防下施展出更加难以预防和躲避的打击，而且一拳重似一拳，这种瞻之在前、忽焉在后、观之在左、倏现其右的拳法，对付流氓无赖着实有效，以至中共的喉舌一次次表现出对川普既抱有期望又怕再次被修理的复杂心态，这一心态再发展下去离精神病患者可不远了。
 <br/>
 <br/>
 抓捕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 这一拳的确是相当重，不像处罚中兴，处罚的是公司不是个人，而且无论罚中兴公司多少钱还都是全国民众掏腰包，那些中兴公司的当事人不光在经济上毫发无损，还都会因为在中兴事件中表现出为党分忧和为党受罚而上升为〝优质〞党棍，甚至加官进爵、异地升迁都是可以期待的。
 <br/>
 <br/>
 这也不像处罚李尚福，虽然是针对个人，但也仅仅是冻结资产、限制入境，并没有剥夺其人身自由和生命安全，关键是更没有祸及子女。
 <br/>
 <br/>
 而孟晚舟被抓捕的后果可太严重了。
 <br/>
 <br/>
 其一：加拿大皇家骑警作为加拿大的国家象征，可不像中共的公检法傀儡机关听党指挥，若非美国提供明确、充分的证据，证明孟晚舟符合被限制人身自由的司法条件，其根本不会对孟晚舟实施抓捕。
 <br/>
 <br/>
 其二：美国的法院系统具有绝对超然的司法独立和权威，若非已经掌握孟晚舟符合拘捕的充足证据，不会贸然签发拘捕令。
 <br/>
 <br/>
 其三：孟晚舟被抓捕后立即申请禁止报道令，显然是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 乃至中共早已对其进行系统训练以应对突发情况，更可能孟晚舟对自己触犯联邦法律早已心知肚明，只是心存侥幸于美国没有确凿证据实施抓捕而已。
 <br/>
 <br/>
 其四：据说此次抓捕孟晚舟，主要是因其直接参与华为公司违法向伊朗提供通讯设备和技术，而对华为此项违反联邦法律的活动，美国至少已经展开长达两年以上的调查，这次突然实施抓捕如此敏感人物，显然是时机已经成熟，实乃收网之举，孟晚舟万难侥幸脱身。
 <br/>
 <br/>
 其五：孟晚舟作为华为的最高财务主管，掌握华为所有方面和层级的经营活动，是掌握华为信息最全最多的人，她的到案，意味着华为已经完全处于被动处境，陷于任美国宰割的境地。
 <br/>
 <br/>
 其六：孟晚舟作为任正非着意长期栽培的接班人选，其突然失陷，打乱了华为迫于全球围堵而急于权力移交的步骤，无论今后华为是什么结局，接下来要发生的是，华为失去的不仅仅是孟晚舟，还有任正非，或着换句话说，任正非失去的不只是女儿，还有华为。
 <br/>
 <br/>
 其七：中共已经开始失去华为这个操作载体，其2025计划开始冰融雪消。
 <br/>
 <br/>
 然而，上面七条并非最严重的后果，最严重的后果是，川普的这一拳打在了中共党徒个人的心窝上。
 <br/>
 有人说，中国人很现实、很自私，但如果说有什么能让中国人充满希望、获得动力、享受快乐并甘愿为之付出全部的，一般不会是宗教信仰，也不全是事业前程，更不是什么理想抱负或房子车子，而是自己的孩子。
 <br/>
 <br/>
 据说，毛泽东是为了儿子，才最终没有放过彭德怀；卓琳是为了儿子，才以自杀来威胁江泽民；江泽民是为了儿子，让马仔杀人取肾；江绵恒为了儿子，迫富豪出让财富。中共的党棍们如此，其下的各级党徒也不例外，千辛万苦也要把资财留给子女，只要子女好，再难也欣慰，子女成了中国许多无神论者唯一的信仰和希望。
 <br/>
 <br/>
 而川普，竟然开始对这些中共既得利益者的子女下手，原因却仅仅是涉及某个公司的造假活动，这些既得利益者的子女甚至孙辈有多少在美国，更有多少在美国的盟国？他们入则华宅、出则豪车、挥金如土、骄奢淫逸，其中有几人没干过对美国隐瞒、撒谎、造假的事？有句话叫天网恢恢疏而不漏，这天网是上天的意志，正所谓人在做、天在看；但同时又说上天有好生之德，总会网开一面给那些幡然醒悟、远离奸邪的人们机会，全看人自己如何去把握了。
 <br/>
 <br/>
 当然，还有人认为孟被抓捕只是美国执法机构的自作主张，那天正好是川普在和习近平谈笑风生吃晚餐，根本不知道这回事，很可能不是川普的本意。的确，美国对华为的调查很早就启动，但川普作为一国总统，抓捕孟晚舟这么敏感的人物，又恰逢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 最重要的川习会之际，怎么可能不第一时间获知此事呢？12月1日晚川习会持续2小时30分钟左右，大致时间跨度在晚上18时至22时这个范围，那么从时间上推算，川普在孟晚舟被抓后的第一时间即获知的可能性高达85%。
 <br/>
 <br/>
 孟晚舟被抓捕，不啻似惊雷炸裂，已经炸醒许多对依附中共、幻想得到中共保护的人，却不知中共自身难保，只会拿身边触手可及的党徒来垫背和当炮灰。聪明的、想活命的要赶紧离开它。
 <br/>
 <br/>

 <br/>
