### 文武：平度老兵初战告捷
------------------------

<div class="wysiwyg">
 前几天开始的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东平度.html" target="_blank">
  山东平度
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议事件.html" target="_blank">
  抗议事件
 </a>
 步步升级，先是百人的抗议，后来发展到千人的规模，中共看到老兵的人数越来越多，于是调来大量警察开始驱散老兵。
 <br/>
 <br/>
 可是，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 不是平头百姓，原来都是经过部队训练过的，不是警察能随便驱散得了的，于是警察用喷雾器、催泪弹、水龙头，组织警察盾牌来驱散老兵。
 <br/>
 <br/>
 老兵们在忍无可忍的情况下，找来了打狗棍与警察对抗。至昨天下午5点止，双方处于势均力敌的对峙状态。
 <br/>
 <br/>
 昨天夜晚，警察乘着夜幕，向睡在露天的老兵们发起了突然的奇击，试图在夜幕的掩盖下一举将老兵们驱散。可是毕竟是有过战斗经验的老兵，在黑夜里仍然具有防卫能力，经过几个小时的搏斗，警察失败了，没有达到驱散老兵的目的。
 <br/>
 <br/>
 今天白天，老兵们继续在平度抗议，参加抗议的老兵们是越来越多，从全国各地都有来增援的老兵，尽管平度已经被中共封锁，各个进入平度的路口已经都被中共堵住，但是仍然还是有老兵们抄小路来到平度、来到平度的抗议现场。
 <br/>
 <br/>
 今天在平度抗议的老兵比前两天多了近一倍，具体人数无法统计，初略看上去有几千人了，精神状态和身体状态都很好，战斗精神很强，相比中共的警察一个个看上精疲力尽，无精打采的样子，尽管警察的人数众多，武器装备精良，但是士气却非常的低落。
 <br/>
 <br/>
 从网上视频传来的画面来看，老兵们的个个精神抖擞，志气高昂，喊着震天的口号，走着整齐的步伐，对中共为害老兵的恶行表示强烈抗议，要求中共补偿老兵们的损失。
 <br/>
 <br/>
 中共大小官员没有一个敢出头的，全都躲在后面，只是让一些可怜的警察们在前面丢丑。
 <br/>
 <br/>
 从传来的视频中，我们看到了、听到了老兵们喝斥警察们声音，老兵们手拿着打狗棍站在警察队伍的前面，严令警察们向后退，一群群的警察个个吓得向后步步退缩。
 <br/>
 <br/>
 至今天下午5止，老兵们已取得了初步的胜利，警察已经不敢向老兵们发起进攻了。
 <br/>
 <br/>
 接下来中共有可能会调来更多的警察，来驱散老兵，今天晚上中共也许又会突奇老兵，但无论如何，中共这一次想轻易的将老兵们驱散可能性已经不太大了。
 <br/>
 <br/>
 平度的老兵能否被中共驱散，关键就得看接下来的几天了，我个人估计，中共会调来大批的警察，将平度封锁然后逐个驱散老兵，老兵们只能是坚持几天算几天，多坚持一天就是多一天的胜利。
 <br/>
 <br/>
 当然，如果平度老兵这一次，如果直接拿下中共平度市政府，直接占领平度市各主要中共政府机关，直接宣布平度脱离中共，成立平度民管委员会，那形势就会大不一样了，也许这一天会很快的到来。
 <br/>
 <br/>
 文武
 <br/>
 2018年10月7日星期日
