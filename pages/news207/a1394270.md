### 文武：彭斯演讲中还有什么需要明确表达的
------------------------

<div class="wysiwyg">
 10月4日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国副总统.html" target="_blank">
  美国副总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 在美国华盛顿智库哈德逊研究所， 发表了美国政府有关中国政策的演讲，彭斯说，美国原来的政界甚至是美国人民，希望通过帮助中国经济的发展，使中共在政治、经济、文化等各个方面都能够向公平、自由、民主、法制的方向走。可是，二十多年过去了，中共不仅没有向公平、自由、民主、法制的方向走，反而向相反的方向走，中共反而更加专制、更强权。
 <br/>
 <br/>
 中共利用发展起来的经济，不仅不是帮助世界文明的进步，反而给文明的世界抹黑。
 <br/>
 <br/>
 中共利用政府干预的手段，利用美国民主制度中自由、宽松的环境，盗窃美国的知识产权，强制美国的技术转让，设置非关税壁垒。利用美国社会新闻自由的制度，利用美国政治派别之间的矛盾与分歧，中共派遣大量的特务渗透到美国社会各个阶层中，去干扰、影响、左右、引导美国社会的新闻、舆论，中共试图将美国的社会舆论拉向左转；拉到符合中共意愿的方向走。
 <br/>
 <br/>
 就是说，中共不仅自己不改变自己的邪恶行径，反而利用国际社会帮助中国经济发展的机会，将中共的邪恶行径推广到世界各地。
 <br/>
 <br/>
 在这种情况下，美国不得不发起对中共的贸易反击战，美国不得不在各个方面抵抗中共的渗透、抵制中共的入侵。
 <br/>
 <br/>
 但是，即使如此，美国仍然希望中共改邪归正，美国政府与美国人民都希望中国经济发展，都希望中国人民幸福、安康。
 <br/>
 <br/>
 现在可笑的是，直到今天，中共仍然无法理解美国政府和美国人民想要做什么，中美贸易战打到今天，美国在各个领域抵制中共也有些时了，可是中共仍然看不清楚，听不明白美国要做什么？
 <br/>
 <br/>
 中共为什么听不懂人说的话呢？因为在中共的思维体系里，在中共的话语体系里，没有实质的公平、自由、民主与法制。
 <br/>
 <br/>
 例如，中共现在在与美国谈判的过程中，美国提出来要公平、自由的贸易，中共就是死着劲也听不明白这里面究竟说的是什么。
 <br/>
 <br/>
 因为中共自己制订的宪法、法律、规章、条例、制度，甚至是大会、小会，甚至是所有的新闻、媒体、户外宣传的，墙壁广告栏上贴的、说的，全都是公正、公平、民主、自由、法制、人权，在中国到处都是这个东西，多的不能再多了，中共有的就是这个东西。
 <br/>
 <br/>
 现在美国又在跟中共说公平、自由，中共哪里能听得懂呢？
 <br/>
 <br/>
 在中共的思维体系里面，在中共话语体系里面，所有的什么公平、自由之类的东西，不过只是个口号而已，不过只是用来欺骗人用的把戏而已，没有一个中共的死党，会认为这个世界还有公平与自由，他们认为公平与自由是根本不存在的东西，只是一个被拿来玩弄的一个招牌而已。
 <br/>
 <br/>
 所以，中共现在最痛苦的不是别的，是听不懂美国现在要什么，前天中共驻美国的大使崔天凯还在说，中共愿意跟美国谈，中共不想跟美国打贸易战，中共不想跟美国把关系搞坏。但是，美国要什么呢？
 <br/>
 <br/>
 崔天凯的意思是说，中共现在愿意投降，可是你要我怎么样投降呢？你给出一个投降的条件我才好投降呀！不然，我都不知道你要什么，我怎么投降呢？
 <br/>
 <br/>
 中共现在着急的是想投降也投降不了了，这是中共目前最苦脑的一件事情。
 <br/>
 <br/>
 其实，不光中共不知道美国在说什么，美国也不知道中共在说什么。
 <br/>
 <br/>
 这又是为什么呢？
 <br/>
 <br/>
 因为中共就不是人，中共是魔鬼，魔鬼说的话，人当然也听不懂，例如，中共发的一份关于中美贸易的白皮书，让美国人怎么看也看不懂，因为三万多字的白皮书，让正常的人来看，都觉得是中共在要强，要与美国打到底。可是，白皮书字的背后的意思是中共并不想打，而是要降。中共是要投降，可是从白皮书表面字里又看不出这个意思来。
 <br/>
 <br/>
 其实，现在中美双边关系的问题很简单，就是必须说双方都能听得懂的话。
 <br/>
 <br/>
 那中共听得懂什么呢？
 <br/>
 <br/>
 中共只懂两个字〝拳头〞。
 <br/>
 <br/>
 谁的拳头大就听谁的，别的都少废话，别拿好听的话来说，好听的话中共比谁都会说。就只管比谁拳头大就完事了。
 <br/>
 <br/>
 如果美国打赢了中共，中共就完全听美国的，如果中共打赢了美国，美国就完全听中共的，这就是中共的思维体系，这就是中共的话语体系，也只有这个〝拳头〞中共才听得懂。
 <br/>
 <br/>
 所以我说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 副总统的演讲，如果能再明确一些就更好了，当然如果只是针对正常的社会来说，彭斯副总统的演讲已经非常全面、非常清楚、非常明确了。可是针对中共来说，就还需要更明确些，才能让中共也听得懂。
 <br/>
 <br/>
 那如何才能更明确呢？
 <br/>
 <br/>
 那就是明确的说出来，美国要的就是中共下台。
 <br/>
 <br/>
 中共下台了，美国再与没有中共的中国政府谈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系.html" target="_blank">
  中美关系
 </a>
 。
 <br/>
 <br/>
 这样中共就听得懂了，中共自己知道自己的拳头没有美国的大，所以中共自己就自己下台了，中共下台后，美中关系也就全面正常化了。
 <br/>
 <br/>
