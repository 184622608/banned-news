### 文武：中共开始过河拆桥
------------------------

<div class="wysiwyg">
 前几天，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政法系统.html" target="_blank">
  政法系统
 </a>
 在武汉召开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_打黑除恶.html" target="_blank">
  打黑除恶
 </a>
 专项斗争的推进会。会上特彆强调要刀刃向内，意思是斗争的主要矛头是政治系统的内部，要清理政法队伍，要在中共政法系统内部掀起内斗。
 <br/>
 <br/>
 果不出所料，最近中共的新闻媒体就开始报导清理政法队伍的战果。例如十五分钟之内一个地区就抓了七个贪官，其中五个来自政治系统的报导，武汉地区多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政法系统.html" target="_blank">
  政法系统
 </a>
 的高官被查，法检两家一把手均落马，今年至少十六名法院院长被查等等诸如此类的报导。
 <br/>
 <br/>
 这只是这几天的新闻报导，要是综合一下近几年的新闻报导，那中共政法系统的官员就不知道有多少官员被抓、被查了，查看一下明慧网，什么政法委书记、什么公安局局长、什么派出所所长，出事的就多了去了。
 <br/>
 <br/>
 当然我们这里不是给这些出事的中共政法系统的官员鸣不平，他们其实也是罪有应得，也是跟着中共坏事做绝，最后又被中共遗弃的可怜虫。
 <br/>
 <br/>
 我只是想通过这件事，给还不清醒的人提个醒，跟着中共跑的下场是多么的可怜。
 <br/>
 <br/>
 大家都知道，中共迫害中国民众，主要靠的是谁呢？就是中共政法系统的这些人，这些人把保护中共当成自己的使命，唯中共是从，不管中共叫他们干什么他们就干什么，根本不管什么法律不法律，道德不道德，良心不良心，只要上面一声令下，叫他们抓谁就抓谁，叫他们迫害谁迫害谁，叫他们监控谁就监控谁，成为了中共的铁杆。
 <br/>
 <br/>
 可是今天中共自己快不行了，中共首先想到的不是保护这些为中共卖命的人，而是首先要保护中共自己。
 <br/>
 <br/>
 今天的中共经济快要崩溃了，官场快要失灵了，上面的话没有人听，下面的事无人落实，中共原有控制社会的机制快越来越不行了，眼看中共自己就快要完蛋了，于是中共开始保护起自己来了，开始以反腐败的名义来建立中共的权威。
 <br/>
 <br/>
 凡是中共的官员是无官不腐，中共个个贪污，那为什么还要反腐败呢？其实就是为了建立中共的权威，抓一些人，判一些人，目的是为了吓唬住一些人。
 <br/>
 <br/>
 抓什么样的人才能更有效的建立中共的权威呢？那当然是抓有权力抓人的人，这样政法系统的人就首当其充了，抓人不就是靠公安，靠检察院，靠法院吗？中共这种邪恶至极的体制，自然首先想到的就是要整治政法系统，首先要抓的人就是政治系统的人。
 <br/>
 <br/>
 可是光靠抓贪官还不足以建立起来中共的权威来，今年又来了个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_打黑除恶.html" target="_blank">
  打黑除恶
 </a>
 的专项斗争，要抓黑恶势力的保护伞，能够保护黑恶势力的最主要部门也还是政法系统，抓保护伞当然首先就是要抓政法系统的人。
 <br/>
 <br/>
 当然真正的黑恶势力不是别人，就是中共自己，可是保护中共这个黑恶势力的主要部门也是政治系统，可是说是阴差阳错，现在中共的政法系统的人是里外不是人。
 <br/>
 <br/>
 中国有句古话，女怕嫁错人，男怕入错行，这些在中共政法系统里面混差事的人，恐怕就是前世没有积好德，落到这么一个职业里面来了。
 <br/>
 <br/>
 可是，毕竟还有一句话，叫事在人为。
 <br/>
 <br/>
 虽然从事的是政法系统的职业，可是如何做，自己还是能够选择的。过去说是糊涂，跟着中共跑以为是为了混碗饭吃，现在应该明白了，跟着中共不是什么饭碗不饭碗的问题，而是要命不要命的问题了。
 <br/>
 <br/>
 所以，到了这个关头，这些还在中共政法系统里混事的人，也真的应该清醒了。
 <br/>
 <br/>
 文武
 <br/>
 2018年10月23日星期二
