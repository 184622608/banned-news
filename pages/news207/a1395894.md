### 周晓辉：美国封堵下 王岐山赴以色列救火
------------------------

<div class="wysiwyg">
 据海外媒体综合
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 多家媒体的消息，中国国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 率代表团将于10月22日对以色列进行访问，并参加第四届中国和以色列创新峰会，就扩大中以商业贸易的机会和合作与以方进行探讨。
 <br/>
 <br/>
 为何北京派出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 带队？笔者觉得原因主要有三个：
 <br/>
 <br/>
 一、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 下，美国正在加强对知识产权的保护，加强对中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高科技.html" target="_blank">
  高科技
 </a>
 出口、中资在美国收购、投资高科技企业的限制，收紧中国科技人员的赴美签证等。欧盟等西方国家也紧随美国，采取了相应的措施，限制、封堵中共对高科技的盗取。
 <br/>
 <br/>
 在这样的大背景下，中共通过从美国等西方国家攫取技术、壮大自己，从而向世界扩张的野心受到了一定遏制，但深知自力更生不过是笑话的中共，并不愿放弃自己的野心，转而将目光投向同样拥有高科技、且与自身有合作协议的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 。能从以色列曲线获得美国等西方技术，应是北京最为渴望的。因此在与美国就贸易战达成协议前景渺茫的情况下，作为习亲信的王岐山此次奔赴以色列肩负着重任，可以说，是起着救火的作用。同时，也可表明，中南海迄今并无向美妥协的意愿。
 <br/>
 <br/>
 二、王岐山目前是中以创新合作联合委员会的中方负责人，以色列一方负责人是现总理内塔尼亚胡。王岐山率团出访顺理成章。
 <br/>
 <br/>
 值得注意的是，该委员会正是去年内塔尼亚胡访华与北京建立创新伙伴关系后建立起来的，其包括了中以双方二十多个部门，几乎涵盖了创新和经济发展各主要领域。根据中以制定的《创新合作三年行动计划》，双方合作将包括农业、水资源管理、信息技术、信息安全、生物技术、医疗健康、人工智能等重要领域。
 <br/>
 <br/>
 目前，中以已经开展了一些合作，如成立了联合实验室、孵化器、创新中心、在常州成立创新园等创新创业平台；双方还进行了人才交流，包括两国政府间的创新人才培养计划、创新领袖访问计划和青年科学家交流计划。2017年在以色列最大规模的国际性高科技盛会〝特拉维夫创新节〞召开的时候，四川成都市在特拉维夫举办了〝创业天府以色列特拉维夫专场〞活动，并为成都（以色列）创新中心揭牌。而成都是中共〝一带一路〞对外推广的重要城市之一。
 <br/>
 <br/>
 三、双方高科技合作频繁，以色列的高科技成为北京觊觎的对象。
 <br/>
 <br/>
 在中以双方的协议中，还包括以色列专家到中国进行技术指导，中国的培训团和培训班前往以色列进行高科技培训的项目。而且非常明显，北京对以色列高科技充满着浓厚的兴趣。
 <br/>
 <br/>
 据2015年揭牌的常州创新园驻以色列办公室主任汪洋介绍，在创新园提供的几个平台中，就包括全国首家国字号的中以创新投资基金，这是专门针对以色列高科技项目，总额为100亿元的投资基金。此外，还有人才和外国专家的服务平台；包括2000家高质量的以色列高端技术的中以技术合作平台。
 <br/>
 <br/>
 2016年3月，北京还投资在以色列成立了〝太库以色列全球创新中心〞。该中心总经理郑小星说，创新中心已加速培育以色列企业一百多家，企业总估值超过100亿人民币。以色列国家创新署亚太处处长阿维鲁弗敦也表示，近十年来，中以高科技合作的发展是前所未有的。
 <br/>
 <br/>
 因此，以王岐山的身分前往以色列，既表示了重视，也有可能达成更多的双边合作，以弥补无法从欧美等国进口高科技的缺憾。
 <br/>
 <br/>
 也正是因为这样的密切合作，以色列国防官员发出了警告：与中国进行贸易往来可能会给以色列带来安全风险。据《经济学人》10月11日报导，以色列官员关切两大方面，一是中国公司参与大规模基建项目，二是以色列向北京出售的技术。
 <br/>
 <br/>
 以官员警告称，一些基建项目或让中国得以暗中监视以色列一些最为敏感的资讯，如上海国际港务集团已在以色列港市海法港建造了新的运输设施，而此处停靠着被认为拥有核能力的潜艇舰队。不仅如此，以官员还担心售往中国的〝双重用途〞产品，比如人工智能和网络技术，这些产品可用于情报收集和监视等治安和军事用途。以色列商人称，中国急切地想要获取此类双用途产品。
 <br/>
 <br/>
 而以色列情报机构（摩萨德）前局长以法莲・哈勒维（Efraim Halevy）更表示，〝以色列当然不得不与中国做生意，但是我们没有一个能够确保以色列不向中国变卖关键经济资产和有价值的技术知识的严肃机制。〞
 <br/>
 <br/>
 在美欧等国家对中共保持高度警惕，在以色列国防官员发出警告声后，王岐山的以色列之行会达到其预期目的吗？中以高科技合作会受到限制吗？要知道，作为美国重要盟友的以色列，在诸多问题上都要加强与美国的合作，川普总统上台后，在一片反对声中，将美国驻以色列大使馆从特拉维夫迁到了耶路撒冷，这样的支持以色列不会不铭记在心。
 <br/>
 <br/>
 因此，如果美国向以色列发出警告和施压，以色列中止与北京在某些重要领域的合作，尤其是高科技领域，也是大概率事件。如早在2000年，因美国阻挠，以色列就撕毁了向中国出售〝费尔康〞预警机的合约，中以关系也因此经历了一段冷却期。谁能保证这样的事情不会再次发生呢？
 <br/>
 <br/>
