### 文武：自力更生这一次也救不了中共
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在历史上曾经有过几次搞自力更生度过了难关，但是这一次中共无论怎么样搞自力更生也在劫难逃了。
 <br/>
 <br/>
 因为过去中国人普遍被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 欺骗，相信中共说的一定是对的，抗战时期中共欺骗中国人民，说中共是抗战的，要共区的百姓自力更生养活中共，结果中国共区的百姓上了中共的当，自力更生养活了中共；中共篡政后、与前苏联发生矛盾闹分裂后，苏联撤走专家、设备与技术，这时中共欺骗中国人民说苏联是修正主义，在当时共产主义正在红火的时候，在人们普遍还在被共产主义欺骗的环境里，中国人就相信了中共，跟着中共搞自力更生，结果也帮助中共度过了难关。
 <br/>
 <br/>
 可是这一次，中共又处于崩溃的边缘了，中共又故伎重演，又打出自力更生的旗号，要中国百姓帮助中共渡过难关。
 <br/>
 <br/>
 可是这一次形势完全不同了，中共欺骗中国百姓的伎俩已经全被中国人民识破了，中共说得再好听，也没有人再会相信，中国人民经过多次的被中共欺骗后，认识到了一个绝对的真理，那就是绝对不能相信中共。
 <br/>
 <br/>
 再说，历史已经走过了过去的年代，过去中共一直闭关锁国，经过几十年、几代人后，中国人也不知道外面的世界究竟是个什么样子，因为中共一言堂的宣传，中国百姓还以为要去解放全人类，要去解放美国人民，后来开放后，中国百姓才知道，美国人民根本不需要中国人民去解放，中国人民要解放的是自己，而不是美国，不是英国，也不是台湾。
 <br/>
 <br/>
 可是中共为了生存，在四十年前，不得不改革开放，如果四十年前中共不搞改革开放，中共在那个时候就已经崩溃了，因为中共搞了改革开放，中共又多活了四十年。
 <br/>
 <br/>
 但是，中共邪恶的本性决定了中共永远不会走正路，中共只会走邪路，中共改革开放后，不按国际规则行事，利用中共邪恶的专制政权破坏国际贸易规则，盗窃知识产权，强制技术转让，在国际市场上贪得无厌，将本来好端端的国际市场搞得个乱七八糟。
 <br/>
 <br/>
 现在美国忍无可忍可了，美国不得不发起对中共的贸易反击战。世界各国也忍无可忍了，也都纷纷在国际市场上抵制中共，要求中共遵守国际规则，要求中共实现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公平贸易.html" target="_blank">
  公平贸易
 </a>
 。
 <br/>
 <br/>
 但是中共无法做到公平贸易。如果贸易公平了，如果中共不干预市场了，如果中共实现自由贸易了，如果中共把市场放开了，那中国无论是金融领域，还是能源领域，还是农业领域，还是食品领域，还是通信领域，几乎一些主要的领域、主要的行业都会被外企占领。
 <br/>
 <br/>
 中共利用暴力控制中国百姓暂时还行，中共利用暴力控制外国人，那中共就不行了，由于外资的占领，中国百姓生活的很多方面就与外企紧密相连了，中共对中国百姓的控制力就大大减弱了，这样中共的罪恶就越来越被暴露了，中共再想镇压也就力不从心了，中共深知只能靠谎言和封锁才压制中国百姓，中共根本就不敢真的开放市场，所以中共就无法答应美国真的开放中国市场的要求。
 <br/>
 <br/>
 在这种情况下，中共宁可再次闭关锁国也不愿意真的开放市场。
 <br/>
 <br/>
 但是，中国现在的经济已经是外向型的经济了，中国大部分的能源与粮食都需要进口，要进口就得有外汇，要外汇就得有出口，可是要出口就得与外国有贸易，特别是要与美国有贸易，这样一来，中共是闭关锁国也不行，是开放也不行，陷入完全两难的境地。
 <br/>
 <br/>
 再说，现在的中国百姓也不会跟着中共搞什么自力更生了，原来中国百姓是不知道外 面的情况，以为中共说的是对的，现在中国百姓已经知道外面的情况了，中共再号召中国百姓自力更生，艰苦奋斗，帮助中共渡过难关，那哪里还能找得到这样的百姓呢？
 <br/>
 <br/>
 现在的中国百姓是恨中共要死，都恨不得中共早死，人们都等着、盼着中共早死都还来不及呢？怎么可能去帮助中共自己更生呢？
 <br/>
 <br/>
 所以，接下来，必然会发生，当中国百姓生活艰难的情况下，当中国百姓觉得很难活得下去的时候，大家就全都会站出来，全都会起来去找中共要吃的、要穿的。
 <br/>
 <br/>
 中共强占了中国社会几乎所有的资源，大家没有吃的了，没有穿的了，那找谁要呢？除了找中共要还能找谁要呢？到那个时候，所有的中国百姓所有的怒火都会发到中共的头上来。
 <br/>
 <br/>
 所以说，中共已经没有任何办法解除当关的危机了，中共今天所面临的危机对中共来说是个死局，只有中共解体才能解开这个死局。
 <br/>
 <br/>
