### 周晓辉：孟宏伟背后的美亚柏科公司真面目
------------------------

<div class="wysiwyg">
 近日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 在任内被抓捕，成为了国际各大媒体关注的一大新闻，而让北京当局不顾自己脸面也要拿下孟的真正原因，外界也多有揣测，但孟宏伟在连续被免职之后不听劝告且〝一意孤行〞，说明背后的黑幕很深很复杂。
 <br/>
 <br/>
 据欧洲第一电台报导称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 与中国一家专门从事网路安全的大型公司美亚柏科（Meiya Pico）有着密切联系，并一直致力于加强
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 与美亚柏科的战略合作，但因其与国际刑警组织全球创新综合机构（IGCI）形成直接竞争，引起相关人员抵制，美亚公司迄今并未实质介入。
 <br/>
 <br/>
 不过，在2017年在北京举行的国际刑警组织第86届全体大会开幕式上，中共最高当局宣布在5年内将〝采取实际行动〞支持国际刑警组织，包括支持国际刑警组织在反恐、打击网络犯罪、打击新型有组织犯罪领域每年开展3次全球联合行动、成立公安部国际执法学院，为100个发展中国家援建升级国际刑警组织通信系统和刑事调查实验室等，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美亚柏科公司.html" target="_blank">
  美亚柏科公司
 </a>
 亦参与其中。
 <br/>
 <br/>
 或者，也可以这样推测，孟宏伟与美亚柏科公司的密切关系，并非单纯是个人行为，而是背后有着中共当局对外布局的考虑，其被拿下与其所扮演的角色出了差错也可能有关联。从其官网上看，参观考察该公司的中共高官们还不少。在孟宏伟落马后，公司网站上孟与其老板的合影，已经被撤下。
 <br/>
 <br/>
 赢得如此多中共高官青睐的公司，究竟有着怎样的真面目？其官网显示，该公司成立于1999年，总部位于厦门软体园二期，其主要客户为国内司法机关（警察）、执法部门（工商、税务、海关、金融、新闻等）、政务监管部门（证监会、银监会、保监会）、金融机构及大型企事业单位等，是全球电子数据取证行业两家上市企业之一（另一家为美国的Guidance公司），其目前在中国刑侦器材市场的占有率为45%。
 <br/>
 <br/>
 自成立以来，美亚柏科已逐步成为电子数据取证行业龙头企业、网络空间安全及大数据信息化专家，业务覆盖全国各省、市、自治区及部分〝一带一路〞沿线国家。其曾参与了上合青岛峰会、2018年全国两会、厦门金砖会晤、北京一带一路高峰论坛、杭州G20峰会、北京奥运会等十几个国家大型活动的信息安全保障，并且在多次社会重大突发事件中提供技术协助。按其说法，就是〝为推动国家网络安全和信息化工作贡献一份力量〞。
 <br/>
 <br/>
 得到中共当局支持的美亚柏科，现在拥有1家分公司、6家全资子公司、7家控股子公司、10家参股公司，业务也由政务向民用市场延伸。其发展的一个远大目标就是，帮助执法部门实现〝大数据打造无贼天下〞。
 <br/>
 <br/>
 再看其产品。在电子取证方面，美亚柏科拥有针对计算机、手机、移动硬盘、U盘、存储卡等各类存储设备进行取证处理的一系列软硬件产品。
 <br/>
 <br/>
 此前有媒体报导，今年8月美亚柏科曾向俄罗斯销售被称为〝魔方〞的智能手机和移动设备数据提取系统。据称该系统能够在几分钟之内轻松破解脸书、推特、Telegram等各种社交媒体帐号内的信息，解码Skype等各种即时通讯软体中的信息存档，也能够获取几乎所有已知智能手机和各种硬碟中的内容。
 <br/>
 <br/>
 此外，在今年5月北京举办的国际警备警用装备博览会上，美亚柏科也曾展示其生产的XDH-CF-5600手机扫描仪。该扫描仪只需几秒钟就可以破解智能手机密码，提取用户数据，令不少海外华人担心回国时，被海关查出曾发表过批评中共的言论而遭到报复。
 <br/>
 <br/>
 在大数据业务方面，美亚柏科2012年开始通过分析处理互联网大数据，并将研究成果应用到多个执法部门开展〝网络空间安全〞。2015年，其与国家信息中心、社科院等合作，承接政务大数据、经济动态分析、税务大数据、一带一路数据库建设等项目。
 <br/>
 <br/>
 美亚柏科还多次参与公安部对一带一路沿线国家执法部门的援助培训，目前已经对一带一路沿线国家培训了四十余期，接待了来自一带一路沿线国家中二十多个国家的六百多名学员。其公司老板坦言，其产品打入西方市场存在很多门槛，但一带一路沿线国家的认可和接受程度更高。
 <br/>
 <br/>
 显然，美亚柏科的产品被广泛应用于中共政府各部门，尤其在司法界，这自然包括帮助网警们的各种监控。网警们的主要工作是上网搜索所谓的对中共当局〝有害〞的信息，即那些可能会威胁中共政权和妨碍中共控制民众的一切信息。如对法轮功学员、维权人士、民运人士的监控。
 <br/>
 <br/>
 泡泡网上曾有一篇匿名的关于用于舆情监测的美亚软件的文章。文章说美亚软件的主操作菜单包括〝一键搜〞、〝实时巡查〞、〝舆情聚焦〞、〝事件聚焦〞、〝人物聚焦〞、〝热点聚焦〞等主要功能。这些功能全面涉及全网搜索关键词、实时监测网站、实时监测关键词、实时监测并分析舆情事件、实时监测相关人物言论以及热点舆情。所有监测基于关键词设置。
 <br/>
 <br/>
 比如〝人物聚焦〞，用于实时跟踪和查看互联网重点人物的言论。监测者可以通过在系统上添加该人物在不同网站上的虚拟身分，包括论坛、微博、博客等，可以根据数据收集进行专业的数据分析，甚至可以推断出虚拟身分，如是否为网络推手。〝人物聚焦〞用于监测自定义的〝人〞。
 <br/>
 <br/>
 文章指，舆情监测软件的厉害之处，是可以迅速显示某一事件的发布源头，即谁通过什么平台发布的。公安部门可根据监测信息，利用技术手段迅速锁定发布者。不仅如此，该软件还可以显示就某一个话题哪家媒体涉及得最多。如果监管部门已就该话题对媒体下达过禁令，那么〝不听话〞的媒体会迅速被识别出并被处以相应的惩罚。
 <br/>
 <br/>
 除了以上六大主要监测功能外，美亚还提供信息报送、舆情导控、舆情简报、工作平台等提高舆情工作效率的工具。
 <br/>
 <br/>
 美亚软件的主要市场在中国大陆，主要监测国内网络信息。不过，境外中文媒体如大纪元新闻、自由亚洲电台、阿波罗新闻网等，也在其监测范围内。其它未被列入监测的境外媒体，需监测人员手动添加到列表中。
 <br/>
 <br/>
 毋庸置疑的是，从其合作单位和产品性质看，美亚柏科公司业已成为了协助中共迫害良善、迫害自己同胞的帮凶，并以此攫取了巨额利润。美亚柏科发布的2018上半年业绩预告提到，此半年归属于上市股东的净利润约为3415.43万元～4391.27万元，较去年同期提升5%～35%。
 <br/>
 <br/>
 更让人担心的是，这个帮凶正在发展人工智能。就在今年3月，该公司特邀美国人工智能开源软件公司Skymind的深度学习技术专家到公司做SKIL平台相关技术交流培训。Skymind公司成立于2014年，总部位于旧金山，其学习系统SKIL为苹果、IBM、空客、美国航空航天、腾讯、阿里巴巴、法国电信、软银等大企业使用。
 <br/>
 <br/>
 在美国政府为防止中共当局掌握高精尖技术而加强其专制统治，从而对向中国输出高科技产品限制之际，助纣为虐的美亚柏科与Skymind的合作就不能不引起世人的警惕。因为其最终开发出的产品还将协助中共政府加大对国人的钳制。
 <br/>
 <br/>
 或许，美亚柏科公司秉持的是〝在商言商〞的原则，无需谈什么道德、良知，能够赚取最大利润就好——即便帮助中共当局迫害良善。但历史早已证明，无论是良心坏了的个体还是公司，早晚都要付出惨痛的代价。
 <br/>
 <br/>
