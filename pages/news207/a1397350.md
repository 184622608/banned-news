### 新世纪影视带给世界的清流(1)—影片《心愿》
------------------------

<div class="wysiwyg">
 我作为一个观影多年的影迷，这些年来，特别是好莱坞，已经很难看到一部，让我心满意足的电影了。
 <br/>
 <br/>
 好莱坞主导着世界电影界，也成为美国文化的世界性标志。1934年，在教会的协助下，美国电影界制定了《海思条例》（Hays Code，全称为Motion Picture Production Code）。条例的第一条规定：任何电影不得降低观众的道德水准。因此观众的同情绝不能导向罪犯、犯罪行为和罪恶；关于性的条例规定，必须维护婚姻和家庭的神圣性，电影不得暗示低俗的性关系是可接受的，或者是常见的。通奸有时是情节所必需的内容，但绝不能做露骨的处理，或者以正面或以吸引人的方式来呈现。
 <br/>
 <br/>
 但是，从上世纪50年代开始，越来越多的影片无视《海思条例》，对好莱坞原有的道德自律带来极大的冲击。60年代末，西方受共产主义思潮影响的反文化运动高潮到来，宣告了好莱坞传统道德秩序的解体。发展到现在，好莱坞影片中，宣扬人变兽、人兽交主题的作品也已经被好莱坞主流认可、受到赞誉，甚至获得奥斯卡奖。好莱坞编剧、知名影评人麦德维徳指出，好莱坞从业者按照他们的意愿改变社会的价值观：美化暴力、鼓吹性乱、贬低家庭等等。
 <br/>
 <br/>
 以暴力、色情、不伦恋情、颓废、阴暗等为主调的影片，占据了电影市场的主流，看完这样的影片，带给人们更多的是压抑、欲望以及绝望。
 <br/>
 <br/>
 但是，在这样的昏暗混乱、污浊窒息的电影世界中，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新世纪影视.html" target="_blank">
  新世纪影视
 </a>
 和其拍摄的影片，却带来了一股清流。
 <br/>
 <br/>
 美国新世纪影视基地的短片《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心愿.html" target="_blank">
  心愿
 </a>
 》，入围美国佛罗里达墨尔本独立电影人节（简称MIFF），2018年10月19日，影片在电影节上公演。美国墨尔本独立制片人电影节已经举行了20届，主办方表示，该电影节每年都会从大量的参赛影片中，选择少量的入围影片进行放映。电影节的活动包括走红毯、放映、颁奖等。
 <br/>
 <br/>
 《心愿》是大陆知名影视演员姜光宇加盟新世纪的首部作品，也是他与女主演郑雪菲首度合作的影片。
 <br/>
 <br/>
 姜光宇讲述了他参加电影节的情况。姜光宇说，他是第一次坐在电影院里观看自己主演的影片，感受独特。令他感受意外地是，几百个几乎是清一色的西人观众，很多在观看《心愿》时，都擦眼泪。这种体验与我相同，我在此前的文章中曾说，《心愿》这部短短的9分多钟的短片，竟然让我多次掉泪，冲击着我多年的观影经验。
 <br/>
 <br/>
 姜光宇还带给我们更多的观众反馈：
 <br/>
 <br/>
 墨尔本电影节的总裁罗伯特.曼（Robert Mann）先生说：〝我自己也是电影制作人，我很欣赏它传达善有善报的信息：只要你做好事，总会收获好的结果。影片鼓励人心向善，这是我们需要的。〞
 <br/>
 <br/>
 MIFF的创办人兼评委之一泰瑞・克洛宁（Terry Cronin）与另外两位评委鲍勃・理瑟克（Bob Lizek ）和杰夫・霍尔（Jeff Hall）都表示，《心愿》这部片子令人感动。克洛宁先生认为影片传达鼓舞人心的信息：〝显然影片传递出非常强烈的信息。一个好的消息，讲述人如何成为一个好人。像救赎一样。我认为人们真的很喜欢看。〞
 <br/>
 <br/>
 独立制片人安迪・赛比斯（Andy Sipes），曾获得不少世界电影奖项，他赞：〝《心愿》的主题非常好，给人们传达出光明的讯息：即便你曾经做过错事，你还是有机会变成一个更好的人，给人希望。〞
 <br/>
 <br/>
 MIFF创办人泰瑞・克洛宁先生的父母也观看了该影片，他们表示很喜欢这部影片，他们说：〝刚开始看觉得心里很难受，因为小伙子偷了一位残疾人的钱，但是没想到那位美丽的妹妹还给了他一本书，那么善良，而那位‘哥哥’看了那本书后，做的那么好，回来又帮妹妹治好了眼睛！〞泰瑞的父亲克洛宁博士说：〝我看了所有入围的影片，《心愿》最令人印象深刻。我还想问，那本书是什么样的书？很想读那本书。有没有英文版的？〞
 <br/>
 <br/>
 当得知那本书是《转法轮》，有多种语言包括英文版时，克洛宁博士和太太很高兴，马上就表示要找到这本书。
 <br/>
 <br/>
 另一位当地年轻人李尔（Lear）观看影片后非常激动，一走出来就问：〝告诉我，哪里可以找到这本书，我现在就想看这本书！〞
 <br/>
 <br/>
 姜光宇说，好的文艺作品其实能触动人的心灵。《心愿》虽然讲的是个很小的故事，但影片特别干净，没有很多世俗的东西，把今天社会上特别纯、特别善的东西展现出来了。这是影片获得好评的主要原因，因为今天人们特别需要这个东西。
 <br/>
 <br/>
 拍摄于1994年的美国影片《肖申克的救赎》，多年来在互联网电影数据库（IMDb）的史上最佳250部电影影迷评选中，一直排在第一名与第二名之间。一部好的影片，带给人信心、光明和希望，《肖申克的救赎》就是如此。我从这部影片中，看到了自由、希望、救赎和重生。
 <br/>
 <br/>
 我如今惊喜地发现，2017年成立的新世纪影视，正在给世界带来一股清流，这股清流，将给人们带来光明、希望、救赎和重生。
 <br/>
 <br/>

 <br/>
