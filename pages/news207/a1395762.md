### 乐道：有一种打砸叫维权
------------------------

<div class="wysiwyg">
 5月26日，在云南永仁县派出所，一醉酒男子找到民警说：〝你们以后有犯罪分子通知我，我想锻炼一下身体。〞据说事后民警联系了他的工友及家属，让他们加强管教。
 <br/>
 <br/>
 其实，这名男子可不是酒后戏言。在中共治下的各个派出所、刑警队，让地痞流氓把人当沙包练拳，打得人死去活来还不留伤痕，是再正常不过了。打人者大都是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警察.html" target="_blank">
  警察
 </a>
 的眼线，平时靠着告发、和警察关系熟在社会上坑蒙拐骗过日子。前阵子被反杀而死的〝昆山龙哥〞即属于这类，他那个见义勇为奖也是真的，那是他作为警察的眼线〝立功〞后所得。他们打完人，沾了血，就算是表了忠心，今后就靠警察〝罩着〞了，有打人上瘾的，还会给警察打电话主动要〝沙包〞打。这些人在派出所〝练完拳〞就走，被打的人惨遭毒手而不知道打他的姓氏名谁，却是明白了不听警察的话就还会被揍。被打的人，不是有关系、有门路的人，也不是会来事、会奉迎的人，往往都是朴实、耿直、无权无势的良善平民，这些人在那些恶警和地痞流氓眼中是真正的弱者。
 <br/>
 <br/>
 警察的这套〝借拳打人〞的损招来自中共〝挑动群众斗群众〞的邪性教育，其实长期以来各种以抵制和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 为名的群体行为，如抵制日货，抵制家乐福、肯德基、乐天等等，都是中共在背后指挥着包括地痞流氓在内的五毛、自干五式的人干的，这些人严格按着事先培训的〝引导人群、掌握节奏、适时煽动、防止失控〞等等要求进行，甚至连横幅的尺寸和内容都已被规定好了。警察虽然也隐藏在队伍中，他们一般却不会亲自动手以免授人以把柄。
 <br/>
 <br/>
 许多不知情的民众被蛊惑、带动下参与其中，尤其是热血青年在被刻意营造的气氛中极易失控，从而造成他人严重财产损失和人身伤亡等严重后果，那时，中共便会迅速抓捕违法犯罪行为人，既能达到煽动民众的政治目的，又能控制住民众行动不致引火烧身，既惩治了犯罪从而立威，更在被害人面前成功扮演了公平正义的化身，一举四得。那位在2012年的西安抗日大游行中，因把日系车车主李建利的脑袋砸出个大窟窿，而以故意伤害罪被判处有期徒刑十年的蔡洋，绝对不知道中共这下作的伎俩。
 <br/>
 <br/>
 其实不光在国内，中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑手.html" target="_blank">
  黑手
 </a>
 早就伸至境外，在2014年10月香港发生的大规模针对〝占中〞集会民众的暴力恐吓行为，甚至早在2008年发生在美国旧金山、纽约等大城市的诬陷和殴打法轮功学员的犯罪行为，都是中共有组织操纵地痞流氓黑社会打手实施的。既然中共把暴力都输出到了美国，那么我们不难想像在中共治下，没有它的许可，别说上街游行，就连家门都可以让你出不去。所以只要是没有被管制、驱赶和打压的民众活动，可以确定百分百是中共支持、默许甚至背后直接操控的。
 <br/>
 <br/>
 尤其在当下内外交困之际，中共对国内民众的管控越发严酷，许多人仅仅因在QQ里发张真相图片、在微信里说句关心亲人的话即被抓捕。除了平民，许多有良知的学者也遭到中共迫害，贵州大学经济学院杨绍政教授因讲出每年中共党务人员附体民众吸血达20万亿的真相，被校方和所在农公花瓶党先后开除并被警方边控；此外，中南财经政法大学翟桔红、北京师范大学教授史杰鹏、山东工商学院教授李默海、原重庆师范大学涉外商贸学院副教授谭松，都因发表真实言论遭中共打压。
 <br/>
 <br/>
 然而，在如此严酷管制的环境中，就在最近这一段时间，先后发生了上饶、杭州、合肥等地因为房子降价被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_打砸.html" target="_blank">
  打砸
 </a>
 围攻售楼处的现象，至今只见被打砸的企业妥协、赔偿，却未见一例对打砸者进行追求的案例报告。
 <br/>
 <br/>
 从前面的分析我们能得出结论，这些以维权之名进行的打砸行为至少是被中共默许和纵容的。事实上，国内领先房地产行业的万科集团，自2008年起实施的每次降价活动都会遭到购房者的打砸围攻，同时更受到来自相关楼盘所在地方党委的孤立、刁难和威胁、打压，也已经证明这些维权活动是官方认可、鼓励甚至操纵的。
 <br/>
 <br/>
 中共地方党委极度依赖房地产始于94分税制改革，中央一级把税基最优良的一块拿走，地方党委在财政举步维艰之际找到了征地、卖地的生财之路。2005年后，地方党委的房地产收入模式大规模启动，2009年开始，各地对房地产财政依赖度达60%以上，截至2017年，有的地市的依赖程度已经高达80%。
 <br/>
 <br/>
 2018年中共将国地税合并，表面看是精简环节、提高效率，本质是整个统治层面权力上收和集中过程中在财权上的反映，地方党委在财权被削减的情况下，自然不会任凭其所依赖的房地产价格下调，而是希望房地产企业能自行消化和承担相应的市场损失，由此也可推断出近期那些对降价楼盘的打砸围攻的背后是中共的纵容和指使。
 <br/>
 <br/>
 10月13日，浙江传媒学院文学院副院长赵思运因在开学典礼的讲话而被整肃，许多人表示没有看出讲话中有什么不妥，但有人指出，是因为赵思运引用的一段话点到了中共痛处，这段话是：〝把知道的真相告诉大家，是一种正义；把明白的常识告诉大家，是一种责任；把目睹的罪恶告诉大家，是一种良知；把了解的事实告诉大家，是一种道德；把听到的谎言告诉大家，是一种博爱；把亲历的苦难告诉大家，是一种告诫；把面临的风险和不幸告诉大家，是一种善念〞。
 <br/>
 <br/>
 一个人说一句劝善的真话就会被打压甚至被抓捕系狱，一群人以维权之名打砸却毫发无损甚至受到奖励。这善恶颠倒、是非难分的扭曲现象的存在，除了证明中共的邪恶和荒唐之外，是否还有什么深意呢？
 <br/>
