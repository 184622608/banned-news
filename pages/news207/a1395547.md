### 崔士方：北京打老虎出现的四大涉外棋局
------------------------

<div class="wysiwyg">
 中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 从法国回国后落网，其妻成为流落海外的〝棋子〞，这一夫妻棋局，与薄熙来、王立军上下级棋局，令计划、令完成兄弟棋局，马建、郭文贵官商棋局，构成了近年来中共高层打老虎的四大涉外特殊棋局。
 <br/>
 <br/>
 这四大棋局因为其〝涉外〞特性，都引起了海内外舆论的高度关注，但也因为各自的组合不同，而呈现出大不相同的特点和结局。
 <br/>
 <br/>
 薄、王棋局中，王立军虽然选择了鱼死网破，但他在美领馆留下了什么机密，至今成谜。只是这并无碍于王立军事件成为引发中共高层分崩的导火索。其实薄、王棋局中，还有一层薄熙来与薄瓜瓜的父子棋局，但相对于薄王闹掰掀起的12级颱风，瓜瓜的〝潜在报料〞可能引起的小风浪变得份量轻微，也根本无法挽回败局。这种情况下，薄瓜瓜自觉闭上嘴巴，就成了能让薄家损失最小化的当然选择。
 <br/>
 <br/>
 在令氏兄弟棋局中，令完成最终选择了沉默，很多观察家为此大感失望。但如果从人物的个性来看，就会理解事情如此收场也实非偶然。
 <br/>
 <br/>
 一直在兄长的荫蔽下低调发横财的令完成，与在辽宁铁岭即已以〝警界英雄〞大出风头的王立军，两人性格完全不同。张扬跋扈的王立军遇上野心勃勃的薄熙来，固然可以因为两人沆瀣一气，而联手搞出挑战北京权威的〝唱红打黑〞，但也很容易因为一时的利益冲突（一个耳光）而两雄反目，王最终闹出夜闯美领馆求庇护的惊人之举。
 <br/>
 <br/>
 性格软而静的令完成，在关键时刻，不难料想他会权衡再三，下出既有利于令计划获轻判，也方便自身在海外稳定立足的关键棋。所以，最后情况是令完成跟美方联系、交出机密的传闻流出，但令完成本人又始终不露面给予证实。这种招法既令北京在对令计划判刑时投鼠忌器，同时双方又不撕破脸皮、令完成本人可藉此筹码获美方保护。如此，就是一个并不出奇的结局。
 <br/>
 <br/>
 再看郭氏海外报料闹出如此巨大的风波，关键一点，是郭背后集结了几乎所有反习势力。这不仅仅与郭睚眥必报的个性有关，也与郭是一只被遥控、用以冲击十九大高层权力分肥的棋子有关。
 <br/>
 <br/>
 反观
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 的妻子Grace Meng，她虽然已向法国警方报案，但从过去的生活轨迹来看，这位网传一直在中共统战圈子里（民建青岛副主委）兜转、活在温柔富丽乡的女士，很难想像她可以在刀光剑影的高层政治斗争中破幕而出。
 <br/>
 <br/>
 与郭富商天天在Youtube上直播报料相比，Grace Meng至今只能以背面示人。且不论她手头是否握有孟宏伟特意留下的救命机密，单凭这份柔弱，夫家被擒、背后又没有什么特别势力支撑之下，要寄望她挑战中共森冷的专制铁幕成功，就多少有点非分之想了。
 <br/>
 <br/>
