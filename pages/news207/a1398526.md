### 夏小强：中共为〝金融政变〞翻案？
------------------------

<div class="wysiwyg">
 今年以来，在中美贸易战的巨大压力下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 一路狂跌，上证指数一度跌破了在2015年6月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股灾.html" target="_blank">
  股灾
 </a>
 时创下的低点。在这样的背景下，中共高层突然在2018年11月5日，公布了一个消息，这个消息，从政治和经济两方面，都释放出了重要的信号。
 <br/>
 <br/>
 2018年11月5日晚间，中信证券、海通证券、国信证券同时公告收到证监会的结案通知书，针对三家券商与司度（上海）贸易有限公司开展的业务涉嫌违法违规案，证监会经审理认为，三家公司的涉案违法事实不成立，决定该案结案。
 <br/>
 <br/>
 这是什么意思？这就是说，在2017年5月，中共官方证监会认定的这几家公司的违法事实和处罚，都被推翻了。
 <br/>
 <br/>
 这几家公司违法和受处罚的实质是什么？就是涉嫌在2015年造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股灾.html" target="_blank">
  股灾
 </a>
 的针对习近平当局的〝金融政变〞。
 <br/>
 <br/>
 2015年大陆爆发的大股灾，被外界认为是江泽民集团针对习近平当局的经济政变，或金融政变 。大陆经济学者、北京大学教授王建国曾经连发微博指出，股灾是中共内部的贪腐利益集团发动的〝金融政变〞，目的是搞垮中国的经济。
 <br/>
 <br/>
 港媒争鸣杂志也曾披露，在一次中共会议上，李克强怒斥洗钱活动泛滥，并指中共金融界高层、经贸界高层以及公安部门等有＂内鬼″。在会上，李克强点名批评央行行长周小川、时任公安部长郭声琨等。2015年7月初，在为期3天的股市及金融市场会议上，李克强又一次以不点名的严批周小川和中证监委主席肖钢，搞小圈子、内斗、互相扯皮，甚至影响到救市。
 <br/>
 <br/>
 随后，金融高层地震，证监会副主席姚刚和张育军落马，中信公司高管多人接受调查。当时的救市已经成了证监会与中信证券、私募基金等和外资联手做空，进行的一场瓜分财富的行动。
 <br/>
 <br/>
 从2015年股灾之后，习近平当局针对江泽民集团的这场〝金融政变〞，开始整顿金融市场，多名金融大鳄落马，其中包括安邦集团董事长兼总经理吴小晖等人。
 <br/>
 <br/>
 但是，如今，中共官方突然发出对此
 <br/>
 <br/>
 前〝金融政变〞翻案的消息。从表面上看，是中共在贸易战压力下，对资金紧张所作出的〝融资〞措施。其深层原因，是中共政权在中共十九大之后开始全面左转，习近平当局反腐止步于周永康，没有触及江泽民，造成江势力不断制造危机，高层内斗加剧。中共政权在内忧外患下，出现诡异局势。对金融政变的翻案，就是这种局势的表现。
 <br/>
 <br/>
 在经济层面，对〝金融政变〞的翻案，不再处罚做空的证券公司，并且证监会主席
 <br/>
 <br/>
 易纲还向企业〝道歉〞等等，这些信息表明，中共高层对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 的政策将会有大的转向。在楼市、股市和汇率这三个巨大泡沫中，中共很可能将会首先放弃中国股市的这个大泡沫——在股市的缓慢下行中，中共将不会再调集资金采取此前的大规模救市行动。中国股市的泡沫有多大呢？美国股市历经9年的牛市，市盈率中位数是15倍，而中国股市如今市盈率的中位数是30倍。这意味着，中国股市的大部分股票价格，在未来的几年内，可能将会遭遇腰斩。
 <br/>
 <br/>
