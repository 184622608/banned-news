### 袁斌：有感于〝红色公主〞李讷的人生际遇
------------------------

<div class="wysiwyg">
 众所周知，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毛泽东.html" target="_blank">
  毛泽东
 </a>
 有两个女儿，一个叫李敏，是毛泽东与贺子珍生的，另一个叫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李讷.html" target="_blank">
  李讷
 </a>
 ，是毛泽东和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江青.html" target="_blank">
  江青
 </a>
 生的。
 <br/>
 <br/>
 本文不说李敏，但表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李讷.html" target="_blank">
  李讷
 </a>
 。
 <br/>
 <br/>
 公开资料显示，1940年8月，李讷生于延安，1949年跟着父母搬进北京居住，在育英小学读三年级。1959年，李讷考取北京大学历史系，并于1966年毕业。
 <br/>
 <br/>
 李纳毕业后，正逢文革兴起，因为她的特殊背景，被分配到《解放军报》工作。
 <br/>
 <br/>
 1967年1月13日，李讷组织〝革命造反突击队〞，贴出《解放军报向何处去》的大字报〝揭批〞胡痴（其时任新华社代社长、全军文革小组成员）、宋琼以及杨子才等领导，左右了报社的运动方向。1月17日，林彪签署的《给解放军报社革命同志的一封信》（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毛泽东.html" target="_blank">
  毛泽东
 </a>
 批示〝同意，这样答覆好〞），肯定这一行动〝在报社内部点起了革命火焰〞。27岁那年，李讷轻而易举的就当上了《解放军报》总编领导小组组长（相当于总编辑）。同年1月，又升为中央文革小组办事组负责人。1973年，李讷参加了中共第十次全国代表大会，这在当年可是了不得的政治荣誉。1974年至1975年，李讷先后任中共北京市平谷县委书记和北京市委副书记。
 <br/>
 <br/>
 一言以蔽之，作为文革中的风云人物，李讷在那个时代可谓风光一时，而之所以如此，自然是因为她是毛泽东和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江青.html" target="_blank">
  江青
 </a>
 的女儿！
 <br/>
 <br/>
 不过，正所谓此一时彼一时。伴随着毛泽东的离世和文革的结束，昔日〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红色公主.html" target="_blank">
  红色公主
 </a>
 〞李纳的人生也从峰顶一下跌落到了谷底。据一篇文章介绍，在邓小平时代，李纳的生活很凄惨，〝其生活境遇竟然比大多数的普通老百姓还要艰难〞。
 <br/>
 <br/>
 文章披露，江青被判处无期徒刑入狱后，李讷也被赶出了中南海，并神秘的失踪了五年。恢复自由后，李讷已经落得了全身病痛。江青自杀前，李讷因为多种慢性病需要治疗和保养，前后花了数千元医疗费。但当李讷到单位报销时，大部分药费不能报销，需要自己付费。
 <br/>
 <br/>
 李讷看着看病的欠债，万般无奈，只好硬着头皮给中共中央写了一封信，询问她是否有权继承父亲毛泽东生前的财产，尤其是稿费。她表示自己不敢奢望多要，只想继承财产的一部分，希望能从她父亲过去的稿费中支取数千块钱，弥补因治病欠下的亏空。但没成想报告交到邓小平处后，邓小平冷漠的说了一句，毛泽东过去的财产，〝都是党和国家的财产，任何个人都不能随便支取〞。
 <br/>
 <br/>
 文章还披露，李讷平日走在北京街头，总是身着洗得发白的蓝上衣和一条肥大的旧军裤，一付贫穷潦倒、又落后于时代的平民老妪的形象，人们根本无法把她同文革时期担任过毛泽东联络员、《解放军报》总编辑、北京市委副书记的那个风云人物的形象联系起来。
 <br/>
 <br/>
 而毛泽东的外孙、李讷的儿子王效芝，从小就跟着母亲在北京街头寻找最便宜的〝菜叶〞。当他看到菜市场柜台下面有被人扔掉的菜叶，就赶紧捡进妈妈的提篮里。当菜市场售货员用怜惜的目光注视着这个刚刚满入学年龄的孩子时，她怎么也不敢相信这个孩子竟是毛的外孙。
 <br/>
 <br/>
 王效芝自己也回忆说，在家庭发生重大变故的年代里，他刚刚开始记事就学会了〝看人脸色〞。〝那时，我觉得自己是个老出错的孩子，因为人家老瞪我。〞
 <br/>
 <br/>
 李讷的人生再次印证了〝善有善报，恶有恶报，不是不报，时候未到〞的老话。李讷文革后的落魄实乃事出有因。她自己在文革中紧跟着毛泽东江青做了不少坏事，而毛泽东江青更是罪恶滔天，常言道父母作孽也会殃及子女，李讷的遭遇不就是个极具说服力的例子吗？
 <br/>
 <br/>

 <br/>
