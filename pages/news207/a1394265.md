### 高天韵：上天看未来 彭斯严辞批中共的震撼
------------------------

<div class="wysiwyg">
 〝人见目前，天见久远。〞10月4日上午，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 发表了重要演讲，全面阐述
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 的对华政策，严辞抨击中共在国内国外的不端行径，向中共发出信号：美国不会被吓倒，美国不会退缩。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 结合事实与情报信息，谴责中共在多个领域的不良企图和攻击态势。他表示，本届政府将继续采取果断行动，保护美国的利益和安全。彭斯说：〝在川普总统的领导下，我可以向你们保证，美国将坚持到底。〞
 <br/>
 <br/>
 彭斯的发言震撼强劲，那是因为，人民的利益是首位考量，将神的恩典置于高处。
 <br/>
 <br/>
 彭斯充分强调〝人民〞的作用。从发言开始，他即表示，美国人民有权了解情况。在谈及中共对美国政治和政策的恶意影响及干涉美国选举时，他说，〝我们将继续揭露它，不论采取何种形式。我们将与社会每一个阶层的领导人合作，保卫我们的国家利益和最宝贵的理想。美国人民将发挥决定性作用，他们正在这样做。〞
 <br/>
 <br/>
 值得注意的是，彭斯提到了不畏中共的美国记者和学者。他说：〝我们也很高兴看到更多的记者报导真相，没有恐惧或偏袒，深入挖掘中国（共）在何处干涉我们的社会以及背后原因。〞〝越来越多的学者也在大声疾呼，捍卫学术自由，越来越多的大学和智库也在鼓足勇气拒绝中国政府的快钱〞，〝我们相信他们的队伍会不断壮大。〞
 <br/>
 <br/>
 彭斯还说：〝美国人坚定地支持一位把美国放在第一位的总统。〞 〝中国（共）应该知道，美国人民及其两党民选官员的意志坚决。〞这些话都在警告中共：不要以为离间美国政府和人民的伎俩可以得逞。
 <br/>
 <br/>
 对于北京当局，彭斯释出善意，敦促切实的改变：〝我们想要与北京建立一种建设性的关系，使得双方共获繁荣和安全，而非分离。虽然北京已经偏离了这一愿景，不过中国的领导者们还是可以改变方向，回归几十年前两国关系刚开始时的改革与开放的精神。美国人民别无它求；中国人民也至少理应拥有。〞
 <br/>
 <br/>
 彭斯此番演讲的背景是：中美贸易战开打近三个月，战况激烈，相持不下。同时，中共在政治、媒体、科技产权和军事等多方面向美国加剧渗透，并在世界其它地区继续扩张。美国政界、学界及民间渐已觉醒，在川普总统的带领下展开全面反击。此时，美国副总统向中共喊话，阐明立场，提供选择之路。
 <br/>
 <br/>
 除了警告中共，彭斯的话也带给不同的人群深刻的思考：〝人见目前，天见久远。〞
 <br/>
 <br/>
 那些被中共收买、与中共交好的美国政客、企业家或文人，都必须清醒，必须脱离欲危害美国社会的中共政权。当彭斯点出〝谷歌应立即终止‘蜻蜓’应用的开发〞时，台下立即以掌声响应。
 <br/>
 <br/>
 那些受中共洗脑宣传蒙蔽，动辄以〝冷战思维〞或〝反华势力〞指摘美国的人，面对美国正、副总统的凛然正气，请停止谩骂，请严肃思考，请放眼满目疮痍的中华大地，倾听受苦受难的同胞的呼声。婴儿、幼童被毒疫苗夺走了生命；虔诚向神的信徒被从手中抢走经书；维权律师被强制失踪，他们的健康、事业、家庭都遭损毁；传播真相、维护正义者被非法关押、施以酷刑，甚至被折磨致死。真实的数据和信息被封锁，清洁的空气和水源被污染，纯净的心灵被毒害，国已不国。中国只有抛弃中共，才能重建希望。中国人只有抛弃中共，才能重获尊严和生机。
 <br/>
 <br/>
 中共执政者务必看清：这个政权背弃了人民，一直侵犯人权，破坏宗教和信仰自由，而且还在全球恶意扩张。今天，它的邪恶已被识破，以美国为首的自由社会已经开始出击、阻截、反制。中共习惯了用金钱和谎言铺路，它应当明白：并不是所有人，都会出卖灵魂。中共与神为敌，与普世价值为敌，这一场战役，胜负已明。
 <br/>
 <br/>
 美利坚的决心，鼓舞着受压迫的中国民众，再次把对抗邪恶的信念和启示传向世界。
 <br/>
 <br/>
