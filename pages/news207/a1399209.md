### 子华：做中共的拉磨驴下场最可悲
------------------------

<div class="wysiwyg">
 近年来，在中国大陆出现了一种令人深思的现象，就是越来越多的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 官员、甚至是高官，在特殊的日子，早早赶到寺院，不惜花大钱争着给神佛敬上第一炷香。有的求神佛保佑平安无事，有的求神佛保佑继续升官发财，虽然目的不同，但是最起码他们相信神佛是高于人类的。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 半个多世纪的无神论的洗脑和毒害中，其体制内怎么会有这么多的官员反而去求神佛保佑呢？中共宣扬无神论，与天斗与地斗与人斗，人定胜天，把自己说成是人民的大救星，人世间俨然没有任何生命能高于它。那么中共体制内的官员们怎么不求它保佑呢？就连平民百姓谁也不会给中共烧香、磕头，求保平安。一旦遇到危难时，人们会不由自主地呼喊：天啊（或老天爷啊，佛啊，菩萨啊），救救我吧！没有谁会喊：党啊，救救我吧！
 <br/>
 <br/>
 几十年来，人们经历了中共制造的一次次整人、害人的运动，从中使众多的人认清了它的真实面目和魔鬼行为。今天的中华大地，已被中共糟蹋得不堪入目，草原成沙漠，江河满污水，空气污染，土壤污染，无毒不有，无假不有，贪腐淫乱，物欲横流，疾病蔓延，灾祸频繁，传统被毁，人人相害，社会道德彻底崩溃，一切都在危及着人们的安全。
 <br/>
 <br/>
 世风败坏，灾难频繁，人们越来越意识到平安的重要。那么怎么才能平安呢？心怀善良，敬畏天地神佛，得到神佛保佑才能平安。心里总想着坑人害人，为了钱财权色跟着中共干坏事，危害良善，怎能平安？即使花再多的钱求神佛保佑，神佛也不会保佑坏人的。
 <br/>
 <br/>
 最近，中共在它自己的网站上公开通报了一百三十多个公安正、副局长被撤职、查办，便是鲜明的例子。这些曾经被中共捧得红极一时的人，而今却身败名裂，光环失尽。他们被通报的罪名多是违法乱纪，滥用职权，徇私枉法，贪污受贿等，有的已被判处有期徒刑几年、十几年、二十年、甚至无期；还有的被判处死缓；还有人畏罪自杀等。其中有被称为〝中共第一警察〞的王立军，〝天津第一警官〞的武长顺，〝烟台第一局座〞的聂作坤等。
 <br/>
 <br/>
 这些披着警服，分布在全国各地公安机关的正、副局级官员，名义上打着保护人民生命财产安全的幌子，实质是危害人民的强盗。他们贪污腐败，欺压百姓，横霸一方，为非作歹。他们整天喊着打击坏人，结果他们却是真正的坏人。难道他们一出生就注定要做坏人的吗？他们都是从小到大在中共邪理歪说的灌输下，听信了中共，充当了中共的鹰犬，才沦落到这步田地。还有几百个落马的中共高官，大都是以违法乱纪、贪污受贿等处置的。听党的话，为党拉磨，党不用了，就卸磨杀驴。这就是为中共做驴的下场，也是善恶报应的结果。
 <br/>
 <br/>
 可是这些人最大的恶行中共却不敢通报，就是他们助纣为虐，积极跟随中共江泽民集团
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迫害.html" target="_blank">
  迫害
 </a>
 善良的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员。法轮功是源于古老的中华传统文化的佛家修炼大法，是真正的正法大道。教导人们按照真善忍做好人，修心向善，使人的道德迅速回升。法轮功从一九九二年传出，短短七年在中国大陆就有一亿人学炼。而被国人公认没干过任何好事的江泽民出于妒嫉，一九九九年与中共互相勾结掀起了对法轮功的迫害运动，因找不到法轮功的不是，就编造出了一个个谎言栽赃陷害，甚至花钱雇人冒充法轮功学员恶搞〝天安门自焚〞嫁祸法轮功，让全国民众仇恨佛法，并从此给中华民族埋下灾祸。
 <br/>
 <br/>
 法轮功教人向善，明文规定禁止杀生和自杀，在被迫害十九年之久，没有发生过一起恶性报复事件，在社会上真正的法轮功学员中没有任何人杀人放火、偷盗抢劫、坑蒙拐骗、嫖娼卖淫，事实证明他们是遵纪守法的好公民，对他们的迫害都是违法犯罪行为。
 <br/>
 <br/>
 二零一七年，在〝敲门行动〞（中共迫害法轮功的代号）中，走在最前列的各地派出所长和指导员，恶报死亡的就有四十人，年龄最大的五十九岁，最小的三十二岁。这些年参与迫害法轮功的警察和〝六一零〞（迫害法轮功的非法机构）等部门，恶报死亡的人数，在海外明慧网报导出来的就有数万例（被树为〝英雄〞，在车祸中丧命的登封市公安局长任长霞就是其中一例），还有很多的没有报导出来。迫害佛法和佛法修炼者，最终都得在恶报中偿还。
 <br/>
 <br/>
 有不少警察对法轮功学员说过这样的话：知道你们是好人，共产党给我钱我就得干。明知是好人还要迫害，就是要做坏人。跟中共干，终会成为它的牺牲品。每次运动，中共都是先给一些人一点好处，让他们去整别人，等利用完了再卸磨杀驴，使其下场可悲可耻。
 <br/>
 <br/>
 中共是全世界共知的黑恶势力，却以所谓的〝打黑除恶〞贼喊捉贼地迫害一些正义人士、维权人士、上访人士、基督教和信仰真善忍的民众。
 <br/>
 <br/>
 二零一八年十月十日，吉林省以省、市、区联署发了个所谓〝打黑除恶〞的公告，十月十二日，长春市就突击绑架了二十多名法轮功学员。十一月九日，黑龙江省大庆市有十七位法轮功学员被绑架，同一天，哈尔滨市和周边县大约有四十位法轮功学员被绑架。有些地区，也出现了这种苗头。有的百姓说：打黑除恶是假的，抓人家手无寸铁的法轮功学员去充数。这就是中共所谓〝打黑除恶〞的真正意图，在它黑暗的统治下，要想做好人就没有你的生存环境。中共最终的险恶目的，就是以利益诱骗和暴力杀戮把好人变坏，再利用坏人迫害好人，逼迫每一个中国人去做坏人，完全放弃做人的标准，然后被神佛彻底抛弃，招致天惩大难，最终被淘汰。
 <br/>
 <br/>
 正邪的较量已见分晓，天灭中共早成定局。在海外大纪元网站声明退出曾加入过的党、团、少先队，并能真诚的尊重佛法，心中向善，人们才能得到神佛保佑，获得平安。谁保中共谁就与它一同玩完，这是无法抗拒的天意。顺天者生，逆天者亡。
 <br/>
 <br/>
