### 文武：千万不要相信中共
------------------------

<div class="wysiwyg">
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 打得正不可开交的时候，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 突然传来消息说要与美国和解，说和则两利，斗则两败。中共说要与美国协商，争取达成双方都能接受的协议，从而停止中美贸易战。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 变脸可真快，前几个月还说要与美国以牙还牙，还要与美国把贸易战打到底，直到把美国打得服输为止。这才没两个月，中共就变脸了，要与美国和解了，要与美国达成贸易协议。
 <br/>
 <br/>
 中共无论是说战还是说和，都千万不要相信中共，这个世界上如果有什么是绝对真理的话，那就是绝对不要相信中共。
 <br/>
 <br/>
 当年中共为了打击国民党，就说国民党是一党专制的独裁政权，中共要团结全国人民一起来推翻国民党的独裁政权，在中国建立起一个像美国一样的民主制度的国家。
 <br/>
 <br/>
 可是当中国人民听信了中共的话后，团结起来把国民党赶出中国大陆了，可是中共建立起来的政权却比国民党的政权要专制、独裁的不知道要超过多少陪了。
 <br/>
 <br/>
 上个世纪五十年代末，中共说要知识分子出来给中共提意见，中共说所有给中共提意见的人，要知无不言，言无不尽，言者无罪，闻者足戒。可是当一些知识分子相信了中共说的话后，当一些知识分子诚心诚意的给中共提意见后，中共马上就翻脸了，把这些敢于给中共提意见的知识分子们，一个个抓的抓、关的关、杀的杀。
 <br/>
 <br/>
 上个世纪八十年代末，发生爱国学生运动时，中共说青年学生们都是爱国的，中共绝对不会向学生开枪。可是一九八九年六月四日，中共就派出大量的军队向学生开枪，用军队镇压天安门广场上的爱国学生们。
 <br/>
 <br/>
 上个世纪九十年代末，也就是一九九年四月二十五日，法轮功学员去中南海集体上访，要求中共立即无罪释放被中共关押的法轮功学员，当时中共在接见法轮功学员代表时说，法轮功学员都是些善良的炼功群体，立即释放被关押的学员，叫法轮功员安心炼功，中共绝对不会再去干扰法轮功学员的炼功活动了。
 <br/>
 <br/>
 可是不出三个月，中共就在全国范围大肆抓捕法轮功学员，禁止法轮功学员在外面的一切炼功活动，直到今天中共还在抓捕迫害法轮功学员。
 <br/>
 <br/>
 这个世纪初，中共为了加入世界贸易组织，答应加入世贸组织后遵守世贸组织的规则，可是十五年多过去了，中共不仅没有遵守当时的承诺，反而变本加厉的破坏世贸组织的规则。
 <br/>
 <br/>
 今天，美国政府在忍无可忍的情况下，不得不向中共发起了贸易反击战，一开始中共还不以为然，以为只要骗骗美国，吓唬吓唬美国，这一关就过去了，中共该怎么占美国的便宜还是怎么占美国便宜。
 <br/>
 <br/>
 可是，这一次的美国不是原来的美国了，中共骗一次可以、骗两次可以、骗三次就不行了。美国政府和美国人民这次不会再相信中共了，这一次就得跟中共来真的，中共必须得改变政治、经济体制，中共必须得下台，只有这样，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 才能停止。
 <br/>
 <br/>
 美国只有与没有中共的中国政府签订双边贸易协议才会是真正的可执行的贸易协议。
 <br/>
 <br/>
 作者提供 本文只代表真正的观点和陈述。
 <br/>
 <br/>
