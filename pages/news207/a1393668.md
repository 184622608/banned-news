### 文武：中国的国庆日应该是哪一天
------------------------

<div class="wysiwyg">
 中共把现在中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国庆日.html" target="_blank">
  国庆日
 </a>
 定为十月一日，但是这个日子中国人民是不同意做为中国国庆日的。十月一日不仅不是中国的国庆日而且还应该是中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国殇日.html" target="_blank">
  国殇日
 </a>
 。
 <br/>
 <br/>
 从1949年10月1日起，中国就落入了中共的魔掌，中国人民从此沦为中共暴政统治下的奴隶，所以这一日应该是中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国殇日.html" target="_blank">
  国殇日
 </a>
 。
 <br/>
 <br/>
 那么中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国庆日.html" target="_blank">
  国庆日
 </a>
 应该是哪一天呢？
 <br/>
 <br/>
 如果按照本次人类文明中，黄帝统一中华民族算起，那么中华民族已经有四千多年的历史了，就是说中国已经存在近五千年了，那么中国的国庆日就应该是黄帝的诞生日。
 <br/>
 <br/>
 根据历史学家的考证，黄帝诞生日是西元前农历2711年3月3日（也有说2月2日）。按照现在比较流行的说法黄帝的生日是农历三月初三，那么中国的国庆日就应该是每年的农历三月初三。
 <br/>
 <br/>
 如果按照朝代的演算法 ，那么中国的国庆日就应该是中华民国成立之日。
 <br/>
 <br/>
 中国的本朝应该是中华民国，1911年10月10日辛亥革命成功后，清朝就随即瓦解，中华民国于1912年1月1日宣布成立，所以中国的国庆日应该是每年的1月1日，与元旦同日。
 <br/>
 <br/>
 中华民国从1912年1月1日成立至今，一直都存在，只是过去和现在都还在经历曲折，1928年国民党政府打败了北方的中国民国政府，但是国民政府并没有取消中华民国，而是继续延用中华民国的国体。
 <br/>
 <br/>
 至1949年10月1日中华民国在中国大陆沦陷，那这只能算作沦陷事件，并不能算着是中华民国消亡，而且中华民国的国体还被国民政府带到中国台湾，所以中华民国国体并没有消亡。
 <br/>
 <br/>
 目前中国人民要做的事情，就是要在中国大陆恢复中华民国，把中国大陆人民从中共的黑暗统治下解救出来，中华民国才是现在中国的正式朝代，中华民国才是现在中国合法的正式国体，所以中华民国成立日，1月1日才是中国的国庆日。
 <br/>
 <br/>
 作者提供 本文只代表作者的观点和陈述。
 <br/>
