### 文武：防火防盗防中共
------------------------

<div class="wysiwyg">
 前天美国国务卿蓬佩奥在布鲁塞尔宣布，美国将重建世界秩序，重回世界的领导地位，其目的是为了防止中国、俄罗斯和伊朗等独裁国家对世界的危害。
 <br/>
 <br/>
 从对世界的影响上来说，俄罗斯和伊朗还无法与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 相比，所以，美国真正要防的就是中共，就是现在的中国政府，也就是中共政府。
 <br/>
 <br/>
 用蓬佩奥的话来说，中共发展了经济，不仅不利于中国人民，而且也不利于世界；中共利用世界贸易组织的漏洞，窃取国际财富，却是用来对中国人民进行政治迫害与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权.html" target="_blank">
  人权
 </a>
 侵犯，用来破坏世界的普世价值。
 <br/>
 <br/>
 为了防止中共进一步对中国人民的迫害，为了防止中共对全世界的危害，所以美国必须要站出来，重新建立新的世界秩序。新的世界秩序将以规则为基础，以普世价值为原则，以实用性为考量，以能有效防止中共的危害为前提。
 <br/>
 <br/>
 中国的老百姓常讲，防火、防盗、防干部（中共的干部），现在美国也开始觉醒了，要防火、防盗、防中共。
 <br/>
 <br/>
 原来美国和国际社会对中共没有提防，普遍对中共软弱，结果被中共乘机做大，搞得现在整个世界不得安宁。
 <br/>
 <br/>
 当初美国给中共最惠国待遇时，却与中共的人权脱勾，当时的美国政府误以为，只要帮助中共的经济发展起来了，中共自然会走民主法制的道路，中共自然会改善中国人的人权。可是事实却与那时的美国政府的想法完全相反，中共有钱后，不仅不走宪政民主的道路，不仅不与世界文明接轨，不仅不改善中国人的人权，反而更加专制，更加独裁，更邪恶，反而更加迫害中国人的人权。
 <br/>
 <br/>
 这是血的教训，这个教训是深刻的，必须吸取这次血的教训，那就是千万不要相信中共，千万不要相信中共会变好，中共永远也不会变好，就是让中共挣再多的钱，中共也不会变好，而且中共只会是越有钱就越坏。
 <br/>
 <br/>
 这一次美国政府算是清醒了，在贸易领域，也不再跟中共谈什么了，直接就给中共加关税，直接拿出清单来要中共认领，一项一项的要求去落实，只有落实了才会减少关税，而不去听中共承诺什么，只看实际怎么在做。
 <br/>
 <br/>
 今天又传出来，应美国政府的要求，加拿大司法机关，已经把中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 的副董事长，就是华为创始人任正非的女儿孟晚舟给抓来了，不日将引渡到美国受审。
 <br/>
 <br/>
 原因是中共的华为公司违反了美国对伊朗的制裁禁令，利用美国的产品支助伊朗。华为尽管是高科技公司，但是华为的关键技术产品仍然是来自美国。可见美国对中共的态度是越来越强硬了。
 <br/>
 <br/>
 无独有偶，英国、澳大利亚等国也在行动，撤销中共华为的产品，抵制华为，抵制中共的渗透。
 <br/>
 <br/>
 防火、防盗、防中共很快会成为全世界的共识。
 <br/>
 <br/>

 <br/>
