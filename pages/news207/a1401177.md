### 袁斌：川普总统千万别再把中共的承诺当真
------------------------

<div class="wysiwyg">
 这周末举世瞩目的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞就要登场了。
 <br/>
 <br/>
 抢在这之前，11月25日，中共银保监会网站公告了银行业、保险业对外开放相关举措的最新进展。其中多项外资银行和保险机构市场准入申请已经获得了批准，包括德国安联（Allianz）得到了第一张外资控股的保险牌照。而在之前，中共国家发改委和商务部就已在6月28日发布了2018年负面清单，放松或取消了对汽车、飞机、造船业以及某些金融部门的外国股权上限和其它的一些限制。就在不久前，中共又向美国提交了一份让步清单，涉及142项内容。
 <br/>
 <br/>
 显然，迫于延续了五个月之久的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 带来的巨大压力，中共表面已放弃了当初声称要〝以牙还牙〞的强硬态度，急切的希望能在这次〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞上与美方达成停火协定，为此才被迫丢车保车，相继做出了一系列让步，承诺扩大美国商品的进口量，降低市场准入门槛，尽管在美方关切的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_智慧财产权.html" target="_blank">
  智慧财产权
 </a>
 问题，包括强制技术转让方面中共仍坚持一概否认的立场。
 <br/>
 <br/>
 现在的问题是，且不论在中共拒不承认盗窃智慧财产权和拒不放弃强制技术转让的前提下，仅仅承诺进口更多的美国商品，降低市场门槛，中美两国积累多年的贸易问题能否就因此得到解决，即便如此，中共的承诺能否兑现却值得怀疑。
 <br/>
 <br/>
 从历史上看，中共之前在贸易问题上做过的承诺并不少，但究竟又有几桩真正兑现了？
 <br/>
 <br/>
 〝美国繁荣联盟〞（Coalition for a Prosperous America，CPA）主席丹‧迪米可（Dan DiMicco）曾任美国Nucor钢铁公司董事长兼首席执行官，并在2016年总统竞选期间担任共和党候选人川普（特朗普）的贸易顾问，14日他在财经网MarketWatch上发表了一篇题为〝中国（中共）总是承诺终止贸易欺骗行为，但从未兑现〞的评论文章。他在这篇文章中写到，中共领导人多年来都在开空白支票，做出千篇一律的空洞承诺，如〝致力于支持多边贸易体制，促进自由贸易〞，然而，〝我们之前已听过几百遍了，但是北京从未真正地改变〞。这方面最典型的例子莫过于当年中共在加入WTO之前拍着胸脯许下的那些至今没几桩兑现的诺言，为此美国今天连肠子都悔青了！
 <br/>
 <br/>
 如果说这是过去的事，那么今天呢？
 <br/>
 <br/>
 美国贸易代表办公室（USTR）近日发布〝301条款〞调查更新报告指，尽管6月28日中共国家发改委和商务部发布的2018年负面清单，放松或取消了对汽车、飞机、造船业以及某些金融部门的外国股权上限和其它的一些限制，但中共实际上仍在继续使用〝外国投资限制〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汽车行业.html" target="_blank">
  汽车行业
 </a>
 就是其中的一个例证。
 <br/>
 <br/>
 为什么这么说呢？你看，中共2018年5月出版的〝汽车产业投资管理条例〞草案，将禁止对新的燃料汽车制造工厂进行某些投资，现有的燃料汽车制造企业要接受现有汽车生产、汽车出口和新能源汽车输出相关的新要求。该法规草案还规定，新成立的纯电动汽车（PEV）制造企业以及寻求扩大PEV生产能力的燃料汽车制造企业必须满足一系列繁重的要求。结果就是，寻求建立新的制造设施的外国汽车制造商在没有中国合资伙伴的情况下，要想满足中共这些要求将会变得极其困难。汽车行业的这些投资限制说明，外国汽车制造商继续受到中共在技术转让制度中所使用的行政审查和许可程式的约束。欧盟商会的中国汽车工作组在其2018/2019立场档中评论道：〝中共汽车行业新政策使外资企业不得不继续留在与中方竞争者一起合资的企业里。〞
 <br/>
 <br/>
 此外，中共宣布到2020年年底为止，将会退出对新能源汽车的补贴。但是中共又通过对汽车行业设定新能源汽车积分比例，从而迫使外国汽车制造商不得不与中国国内已经是大型新能源汽车生产商的企业创建合资企业，否则将面临巨额罚款，因而设立外资独立的新能源汽车难上加难。
 <br/>
 <br/>
 可见，即便是在贸易战大火熊熊的新背景下，中共仍在一如既往的玩弄文字游戏，表面回应美国要求，实际是拆除旧的贸易壁垒，又建立新的贸易壁垒。
 <br/>
 <br/>
 11月26日，川普对《华尔街日报》表示：〝美方唯一能够接受的协议是，中国（中共）必须同意对美国开放，与美国公平竞争。〞 但从中共的本性和它以往一贯的伎俩来看，在即将到来的〝川习会〞上，中共完全可能在拒不改变其重商保护主义以及停止补贴超大规模高科技企业的前提下，故伎重演做出更多的空洞承诺，开出更多的口头支票，像当年加入世贸前忽悠美国那样继续忽悠美国，以此诱使美方与其达成贸易战停火协定。如果真是这样的话，美国对中共的贸易反击战显然就白打了。
 <br/>
 <br/>
 为此，我们有必要提醒川普总统一声：千万别把中共的空头承诺当真！借用丹‧迪米可的话说，川普政府必须看到〝北京从根本上改变其贸易行为〞，即中共必须立即履行入世承诺，尤其是停止操纵人民币、不再倾销和补贴；终止其国家支持的对其它国家所进行的经济、网路与军事的入侵；停止所有智慧财产权盗窃和强制技术转让等不公贸易行为。〝这次绝对不再有任何藉口，容许北京占尽自由开放的美国及其它国家的便宜〞！
 <br/>
 <br/>
