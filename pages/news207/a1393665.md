### 高天韵：谁不想让川普赢？
------------------------

<div class="wysiwyg">
 近日，两条美国重大新闻引起关注，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普总统.html" target="_blank">
  川普总统
 </a>
 遇到的挑战也引人思索。
 <br/>
 <br/>
 川普就任后庄严宣告，〝我将让美国再次强大〞，〝我将让美国梦复活。〞近两年来，川普一步一步地兑现承诺，将美国引向再度强大、繁荣，令众多勤劳、朴实的美国人重燃希望，并且鼓舞了世界其它地区追寻正义的民众。
 <br/>
 <br/>
 那么，为什么有人不想让川普获胜？
 <br/>
 <br/>
 <h4>
  大法官提名风波
 </h4>
 <br/>
 9月28日，美国参议院司法委员会以11比10的投票结果，通过最高法院大法官提名人布雷特．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 （Brett Kavanaugh）的任命。该司法委员会的所有共和党参议员都对卡瓦诺的提名投出通过票，而所有民主党参议员则投反对票。
 <br/>
 <br/>
 9月27日，在参议院司法委员会的听证会上，布雷特．卡瓦诺否认了加州女教授克里斯汀．布拉西．福特对他的性侵行为的指控。当天，福特在听证会上声称，未遂的性侵发生在1982年，但是，她记不清日期和地点，甚至对年份也没有把握。她说，过后她未向亲友透露此事，因此，没有现场证人或旁证。
 <br/>
 <br/>
 针对这一指控，政治评论员Paul Sperry于9月25日在《纽约邮报》发表文章，指出了福特说辞的八大疑点。卡瓦诺在听证会上表示，这个所谓的指控已经成为国家的耻辱。
 <br/>
 <br/>
 卡瓦诺是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普总统.html" target="_blank">
  川普总统
 </a>
 提名的第二位最高法院大法官人选，之前被提名的尼尔．戈萨奇在去年4月7日获得了参议院的通过，被视为川普就任百日内的重大政绩。
 <br/>
 <br/>
 美国最高法院共有9名大法官，这一职位是终身制。大法官的立场涉及政治、法律、社会等诸多方面，对美国社会的走向能够产生重大影响。在移民、签证、堕胎、同性恋等具争议性的议案方面，大法官的裁决至关重要。某个州的不当立法也可能被最高法院推翻。因此，提名大法官，是总统的一项重大权力，也历来是两党的角力议题。
 <br/>
 <br/>
 卡瓦诺和戈萨奇都是保守派，因为川普总统十分重视传统价值观。去年1月23日，川普在第一个正式工作日重启全球堕胎禁令；2月22日，他推翻了奥巴马政府颁布的备受争议的〝跨性别厕所令〞；7月26日，川普在推特上宣布，禁止变性人参军。
 <br/>
 <br/>
 这一系列举措有助于挽救猛烈下滑的社会道德，与鼓吹无拘无束、放纵自我的过度自由的意识针锋相对。
 <br/>
 <br/>
 <h4>
  中共干涉美国
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
   中期选举
  </a>
 </h4>
 <br/>
 9月23日，在爱荷华州最大的报纸
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_《得梅因纪事报》.html" target="_blank">
  《得梅因纪事报》
 </a>
 里，出现了4版
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共官媒.html" target="_blank">
  中共官媒
 </a>
 《中国日报》的英语内容。这些以付费广告形式插入的文章，公开批评川普的贸易战行动，有意挑拨当地农民的情绪，旨在打击共和党的票仓。
 <br/>
 <br/>
 9月26日，川普总统在主持联合国安理会会议时指出，中共一直企图干扰美国11月中期选举，但是美国政府不会听任中共行事。川普说：〝他们（中共）不想让我或我们赢，因为我是第一位在贸易上挑战中国（中共）的总统，而且我们正赢得贸易——我们在各层面都在获胜。〞
 <br/>
 <br/>
 多年来，中共苦心经营、大力渗透海外媒体，并且向外国派驻官媒，发布欺骗性新闻，宣传中共的理念和策略，并探查所在国情报。中共这次在爱荷华州放箭，其狡猾和嚣张暴露无遗。
 <br/>
 <br/>
 中共利用美国的媒体自由，行破坏自由之事。证据确凿，中共面对指责，竟然矢口否认，再现流氓面目。
 <br/>
 <br/>
 美国政府对于此类〝外国代理人〞，理应严查并彻底清除。此外，美方完全有理由要求中共实行对等原则，允许美国媒体在中共官媒上刊登同类图文，以示公平。
 <br/>
 <br/>
 有第三方观察评论员指出，中共此举不明智、不必要，破坏了中美关系，还让人抓住了把柄。显然，这一步棋，中共认为很有必要。为什么？因为它腹背受敌，内忧外困：一方面，贸易战遥遥无期；另一方面，美国处处出击，抓捕中共间谍，排查科技和安全漏洞，阻截来自中共等国的网络进攻，制裁中共军方机构和高官，谴责中共压制宗教信仰自由、侵犯人权等等。
 <br/>
 <br/>
 美国强势反制中共，带动了欧、亚、澳等多国，使得全球抵制中共扩张与渗透的大潮渐成气候，这股正义之潮，得到了中国民众的强力呼应，在未来将汇聚更强、更大的力量。这怎能不令中共胆战心惊？
 <br/>
 <br/>
 中共作乱，必然引来美方更坚决、更大动作的回击，所以，它这是搬起石头砸自己的脚。再说，美国选民会轻易地被红色媒体所左右？人算不如天算。
 <br/>
 <br/>
 共产主义，导向灾难和恐怖；信仰自由、回归传统，导向安定和希望。在川普领导下，美国与中共对峙，实际上是在抗击邪恶、维护正义、守护人类文明成果。透过纷争看实质，所有观众，都必须用心灵投票。
 <br/>
 <br/>
