### 文武：区分开〝中国〞与〝中共〞的意义
------------------------

<div class="wysiwyg">
 这次美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 在华盛顿智库哈德逊研究所关于中国政策的演讲中，每当提到〝危害〞时就说是〝中共〞或者是中国现政府，每当提到〝友好〞时就说是〝中国〞或者是中国人民。彭斯的这种区分意义非常重大。
 <br/>
 <br/>
 因为中共长期以来，有计划、有预谋、有步逐、有意的混淆中国与中共的概念。长期以来，当人们一提到中共时就以为是中国，当人们一提到中国时就以为是中共，完全混淆了这两个根本不同的东西。
 <br/>
 <br/>
 中共这样做的目的，就是为了混淆视听，就是为了绑架中国，就是为了绑架中国人民。
 <br/>
 <br/>
 中国在这一次人类文明史中，有记载的历史就有五千年，中国特定的概念是中国的地域和在这个地域中的历史、文化与人民。而中共是从原来的欧洲传来的一个幽灵，由中国境内的一些流氓极端分子继承并在苏联的支持下建立起来的一个邪恶政权。
 <br/>
 <br/>
 中共是中国的敌人、中共是中国人民的敌人，当然中共也就是世界的敌人、也就是人类的敌人，也就是美国的敌人。
 <br/>
 <br/>
 中共知道自己是个幽灵，中共知道自己做恶多端，中共知道自己的目的就是为了毁灭中国，中共知道自己的目的就是为了毁灭全人类，所以中共要把自己装扮起来，所以中共要把自己与中国绑架到一起来。
 <br/>
 <br/>
 人类有人类共同的属性，那就是人类的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普世价值.html" target="_blank">
  普世价值
 </a>
 ，人类的普世价有普遍性特征与时代性特征。
 <br/>
 <br/>
 人类普世价值的普遍特征是真诚、善良与宽容，人类当前时代性的普世价值特征是公平、正义、科学、民主、自由、人权与法制。
 <br/>
 <br/>
 人类普遍性的普世价值特征不会随着时代的变化而变化 ，无论人类是处于什么样的时代，都得遵行真诚、善良与宽容的普遍性特征的普世价值。
 <br/>
 <br/>
 人类时代性特征的普世价值，会随着时代的变化而变化，现在时代性特征的普世价值，例如科学、民主与自由，在三千年前并不是人类的普世价值，所以只具有时代性的特征。
 <br/>
 <br/>
 但是无论是时代性特征的普世价值还是普遍性特征的普世价值，中共都无法适从，因为这是人的价值，在人类社会只有人才知道人的价值，也只有人才知道如何遵循人的价值。
 <br/>
 <br/>
 可是中共不是人，中共是个幽灵，中共是个魔鬼，中共是魔鬼集团，因为中共没有人的价值特征，没有人的属性，也就无法打扮成人。
 <br/>
 <br/>
 中共为了装扮成人的样子，为了让人也认为中共是人不是鬼，那么中共唯一的办法就是只能靠附体，就是必须附着人的身上，要让人看起来中共的样子是个人而不是鬼，所以中共就必须得绑架中国，就必须得绑架中国人民。
 <br/>
 <br/>
 中共只有把自己打扮成中国，中共的死党只有把自己打扮成中国人民中的成员，只有把自己打扮成代表全体中国人民，中共才能在中国站得住。
 <br/>
 <br/>
 中共经过精心的打扮后，果然让中国人上了当，果然让全世界上了当。全世界都分不清哪个是中共哪个是中国，以至于，中共无论做了多少坏事都以为是中国做的，其实中共做的坏事全是中共做的，与中国无关，中国也是受害者。
 <br/>
 <br/>
 这次，美国向中共发起了贸易反击战，有些人就糊涂了，以为美国是在反击中国。其实美国是在反击中共，美国是在帮助中国。
 <br/>
 <br/>
 美国要的是公平、自由的贸易，而中共要的是不公平、不自由的贸易。中共通过不公平、不自由的贸易，一方面把中国人当成奴隶生活奴工产品，为中共挣外汇养肥中共；另一方面，中共有钱后又去腐蚀全世界。
 <br/>
 <br/>
 中共只有通过不公平、不自由的贸易才能达到减肥中共的目的。如果贸易公平了、自由了，那么中国人就得是个正正常常的人，就有相应人的尊严，就有相应的劳工保护，而不是奴隶，因为奴隶产口不能销往全世界，这也就解救了在中国的中共的奴隶。
 <br/>
 <br/>
 如果贸易公平、自由了，那么信息就能自由流通，中共是如何迫害，如何欺压中国人的事实就会被揭露出来，中共的死党与中国人民也就区分开来了，这样也就知道了谁是敌人是谁是人民，这样也就解救了全体中国人。
 <br/>
 <br/>
 所以说，美国要求的公平、自由的贸易，是在打击中共，是在解救中国的办法之一。
 <br/>
 <br/>
 这样的例子还有很多。人们这么容易混淆中共与中国，不能说是谁的过错，人都有一个弱点，就是习惯成自然，不好的习惯更容易成为自然，就像我们这些常写揭露中共邪恶的文章的人，有时也容易混淆，自觉不自觉的就把中共说成中国，把中国说成中共，这就是坏毛病，因而也是必须要改的毛病。
 <br/>
 <br/>
 当人们都注意了，当人们都留心了，当人们都明白了，都知道要严格区分开中共与中国，那中共就原形毕露了，那中共真的是一刻也存在不下去了，所以区分开中共与中国的意义非常重大。
 <br/>
 <br/>
