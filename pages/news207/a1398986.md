### 陆明：写在余文生律师生日之际
------------------------

<div class="wysiwyg">
 提起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">
  余文生
 </a>
 律师，就会让人想起他耿直的性格、睿智的眼神、对案件的认真、对当事人的负责、对真理的执着、对执法犯法者的追究、对弱势群体的呵护……因为他是真正的〝法律人〞。
 <br/>
 <br/>
 我认识
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">
  余文生
 </a>
 律师是在2016年，我的亲人就因为信仰真、善、忍被非法绑架并抄家，我几乎找遍当地律师事务所，同时又几乎是四处碰壁，没有人同情我亲人的遭遇，更没有律师敢接我的案子。后来经人介绍，我找到了余文生律师，他挺身相助告诉我不要着急，他会尽力而为！信仰不构成犯罪……见他一身正气，侃侃而谈，似乎无惧邪恶！我不禁问道：您为维护我们弱势群体直面暴政，难道不怕他们打击报复吗？他的回答让我出乎意料：我怕有用吗？我是走过生死的人，2014年香港占中事件中，我在北京家里却被抓捕，硬说我是占中事件的参与者，把我关押在死囚牢里99天，被酷刑迫害致小肠疝气、无节制的咳嗽、晚上经常做噩梦……从死囚牢出来的第12天就做了手术。从那时起我才懂得人一定要为自己维权！也就是说，我是在那次占中事件被他们（中共）逼上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 之路的。既是亲历中共监狱酷刑的人还敢于为我们说话为真理而战，震惊之余更令我感佩。他接着又风趣地说：如果我以后再被抓捕了，请您一定要给我发正念啊，我随口应一声：行。不想这一语句无心的玩笑，竟成了残酷的现实。但我没有食言，每天的正念中都在清除您身边迫害您的因素，使您少遭受磨难，祈祷您早日摆脱魔爪，平安归来……
 <br/>
 <br/>
 在那次办案中，我有机会接触余律师，更多了一些对他的了解，余律师有更加博大的胸怀，不只关心他的当事人，还包括全国百姓的疾苦。2016～2017年京、津、冀等地空气污染严重，大量爆发雾霾，余律师站出来公开呼吁中共当局赶快想出有利的办法解决雾霾的问题，不然这些地区的人会大量爆发心、肺、甚至癌症等不治之症，之后又就此事又多次上书北京市长。他不顾个人安危，在两会前期，他曾经向国家领导人提出，应该修改部分宪法和法律。
 <br/>
 <br/>
 谁都知道，他提出的意见和建议都是于国于民有百利而无一害的，谁都能看出余律师是忧国忧民的难得人才。放眼全国有几人敢放下个人安危得失，敢为了百姓的利益去向国家领导人提出意见呢！敢为生民立命，为天下谋太平是他的真实写照。
 <br/>
 <br/>
 但这次的提议，等于在太岁头上动土，2018年1月19日余文生律师被北京市石景山区公安局非法抓捕，1月20日被拘留，1月27日转入徐州市指定居所监视居住，现关押在徐州市看守所，至今一直没有让家属聘请的辩护律师会见。据说今年夏天，余律师的妻子许艳接到了被关押的余律师的电话，余律师是哭着打来的，让人听了揪心，一个铮铮铁骨的硬汉，竟然哭着给亲人说话？！是因为对妻儿的思念？抑或因为对执政者的绝望？还是因为当局颠倒黑白的悲愤？
 <br/>
 <br/>
 上次在死囚牢99天被酷刑迫害致住院手术，这次的余律师又已被关押了292天，在这292天里您又遭遇了什么……我不敢想像……如果有哪一天余律师也在电视上〝被认罪〞了，我知道那一定是您被酷刑了，被威胁恐吓了，因为您也是血肉之躯啊！
 <br/>
 <br/>
 余文生律师的妻子许艳女士时刻都在担心丈夫的安危，申诉无门中依然努力坚持，多次往返于徐州北京之间营救丈夫，一个弱小的女子带着孩子艰难度日，还要承受来自当局的各种欺凌打压，请正义之士伸出援手帮帮她们，也许您的一个电话、一句问候都是对她莫大的支持和鼓励。前路遥远，也许是陡峭的山崖，也许是丛生的荆棘，但是她依然凭藉柔弱的身躯带着儿子坚定地走在寻找丈夫的路上……
 <br/>
 <br/>
 今天是2018年11月11日，也是余律师的51岁生日，狱中的余律师，虽然您暂时失去了自由，但是您赢得的是更多朋友的支持与关注！大家都在默默的为您祈祷，祈祷您好人一生平安！祈祷您的愿望早日实现：愿自由的阳光早日撒满我神舟大地！到那时会海阔凭鱼跃，天高任鸟飞！
 <br/>
 <br/>
