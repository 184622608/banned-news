### 【李天笑快评】川习会后贸易战前景如何？
------------------------

<div class="wysiwyg">
 <center>
  <br/>
  <br/>
 </center>
 <br/>
 <br/>
 【新唐人北京时间2018年11月20日讯】首先，很明显，美中都想达成协议，但标准存在巨大差异。中方拉出了一个非常详细的让步清单，包括142项内容，具体是什么，双方都没有公开说明。据媒体透露，中方基本是沿袭原来思路上的调整，比如加大采购缩小顺差；在特定行业提高外资比例等，但没有涉及美方的关键要求，如有效禁止盗窃知识产权和强迫技术转让，取消政府补贴企业开发高技术的2025计划等。所以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 说，清单漏掉了4-5个重要项目，因此他不能接受这份清单，也就是，达成实质性协议的条件还没有凑齐。因此，按现状推，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 上达成取消关税的协议很难马上实现，但如果中方做出进一步让步，有可能达成一个缓解冲突和框架性解决结构性问题的协议。也就是说，川普不解除现有关税制裁，也不立即加大制裁，双方就框架内的具体细节迅速商议敲定。这实际上是阶段性有条件的停战。也就是，停火但不熄火。从某种意义上说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 趋向于长期化和慢性化。
 <br/>
 <br/>
 这样，从表面上看，中方好像赢得了时间，可以拖下去，但如果中方没有下决心改变结构和体制，也不准备彻底清除阻碍改革的因素包括一直在搅水捣乱的江派残余，非常可能的情况是：一波未平，多波又起，情况会变得更加严重和复杂。最后还得解决，但难度会更大。原因如下：
 <br/>
 <br/>
 一个是，结构性和体制性改革在美方期待的事件内不能及时到位，川普对2500亿商品25%关税和全部出口商品的关税会接踵而来。国内经济将进一步下滑，越来越多外贸供应链将离开中国，出口萎缩会导致失业增加，并伴随社会动荡加剧；中共政权上在现有世界经济体系中更加孤立，局面比现在更糟更坏。
 <br/>
 <br/>
 二个是，前5年习近平主要在清理江派势力，现在江派残余会借贸易战带来的经济困难和社会问题兴风作浪，反攻倒算，除非习能迅速抓捕江泽民，斩草除根，彻底清算江泽民反人类罪行，平定天下，进而全面改制变局，进行中国式的明治维新，或类似蒋经国的大变革。这样局面才会迅速改观。也就是，江派会利用拖延贸易战造成的机会向习发难，但也可能逼习太甚，物极必反，习将江派残余一网打尽。
 <br/>
 <br/>
 三个是，如果阿根廷川习会达不成川普要求的根本性的互惠协议，以后达成协议的门槛对中方来说将更高。从彭斯在APEC前后的讲话看，以后要达成协议，除了要满足现在美方提出的经贸结构性改革外，还要撤除南海的导弹，停止并解决人权迫害等等。也就是，中方越拖，美方对达成协议的门槛设置越高。从另个视角看，门槛越高，习近平改革幅度也必须越大。这对中国进步有好处。习近平小让步小赢，大让步大赢，实质性让步实质性大赢，因为这对中国民众有益，没有失面子之说。中共邪恶不会变，但习有可塑和可变性。别人没看到这一点，川普看到了。所以川普高人一头。
 <br/>
 <br/>
 第四个是，如果贸易磨擦拖下去，美方实际比中方更具耐力。不管国会两党谁占多数，两党在中国的经贸和人权问题上态度是空前一致。而且即使6年以后美国总统由川普换成彭斯，对中共的强硬不会有半点减弱。更重要的是，天灭中共随时可能发生。从多方面看，美国都更具持久力。
 <br/>
 <br/>
 总而言之，目前距川习会还有两周时间。习近平仍可抓住这个天赐良机，提出重大改革措施，与川普达成实质性改革协议。历史上造成中美经贸磨擦、长期耍弄和占美国便宜的是中共体制和江泽民政权，习上台后前5年反腐打江集权，就是为纠正江的政策做准备，现在到了临门一脚。过了这个村，难有这个店。习近平仍有在关键时刻作出明智选择的机会。
 <br/>
 <br/>
