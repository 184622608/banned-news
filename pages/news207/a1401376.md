 人源于神，人来于天，归于天。
 <br/>
 <br/>
 那么！知道了答案了，人怎么才能〝归于天〞？途径是什么？在那里？人是三维空间的生命，智商105；神佛是四维空间的生命，智商超1000以上，〝归于天〞就是返回神的故乡。二维空间的毛毛虫蜕变成了三维空间的蝴蝶在天上飞，难道人还不如毛毛虫吗？
 <br/>
 <br/>
 承蒙天意(天，宇宙最高主宰，即天帝、创世主、造物主)！机缘之下揭示了万古以来终极问题。我和你们一样身在红尘中！我和你们不一样，我走在神路上！我所悟的都是得益于神佛在当代以人身下世度人在公开的著作中都揭示了天机（《转法轮》在线阅读： http://gb.falundafa.org/chigb/zfl.htm ），是人类历史上唯一一个跨人种肤色民族的修炼法门，是地球人唯一能〝归于天〞的途径。谁是佛？识别谁是魔，就不难找到佛了。
 <br/>
 <br/>
 〝荡荡天门万古开，几人归去几人来〞。万古以来的终极谜底揭开了，大戏即将起航！也是天灾人祸将临淘汰该淘汰的人，预示人间将要破谜了。
