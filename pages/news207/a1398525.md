### 祝圣武：请声援余文生律师
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">
  余文生
 </a>
 律师是我的好朋友，是我最敬佩的中国律师之一。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">
  余文生
 </a>
 因为呼吁军队国家化、政治民主化的宪法修改建议以及其它人权和政治活动，而于2018年1月19日被绑架，至今不被允许合法的律师会见。余文生案成为中国政府制造的无数的人权灾难中的又一个国际著名案例。
 <br/>
 <br/>
 余文生被绑架至今已经超过九个月，其诸多基本人权被剥夺，需要公众的关注和声援。
 <br/>
 <br/>
 胸怀正义的朋友，请以您的方式声援余文生。发表或转发一个贴文，在政府网站上留言抗议，更换微信头像，邮寄一张明信片，拍摄一张照片或一个视频，在街头举牌呼吁等等，都是声援的方式。
 <br/>
 <br/>
 11月11日是余文生的生日，希望关心他的朋友在那天拍摄照片或视频并分享到网路，万分感谢！
 <br/>
 <br/>
 您的行动，必定会汇聚在推动中国国家治理正常化的洪流中。
 <br/>
 <br/>
 明信片邮寄地址：江苏省徐州市铜山区三堡镇南1000米，徐州市看守所，余文生（收）电话：0516 – 85036389。
