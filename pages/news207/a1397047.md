### 乐道：华为、中兴之后 美国将中国电信纳入视线
------------------------

<div class="wysiwyg">
 10月25日，海外媒体以《美国会报告：中共欲控制物联网从事谍报》等为题，报导美国国会即将公布中共在过去的近十年内大举投资物联网技术，并利用其所连接的数十亿个电子设备，从事情报搜集、破坏活动，或谋取商业上的利益。
 <br/>
 <br/>
 过去十年，也正是中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司在美国开展市场经营的十年，让人不免怀疑中共的物联网谍报活动一定少不了华为的参与。好在今年年初，美国就以国家安全为由叫停了华为手机与当地运营商的合作，美国联邦通信委员会主席随后也表示：〝多年来，美国政府官员一直对通信供应链中某些外国通信设备提供商所构成的国家安全威胁表示担忧。在路由器、交换机和其他网络设备中隐藏的〝后门〞可以让敌对的外国势力注入病毒和其他恶意软件，窃取美国人的私人数据，监视美国企业，甚至更多。〞
 <br/>
 <br/>
 今年4月份，华为、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴.html" target="_blank">
  中兴
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联想.html" target="_blank">
  联想
 </a>
 三家中国企业，更遭到美国国会美中经济与安全审查委员会直接点名，该委员会称，中国政府〝可能支持某些企业进行商业间谍活动〞，以提高中企竞争力并促进政府利益。
 <br/>
 <br/>
 可以说，类似华为这些公司在美国的市场活动开始变得举步维艰，市场空间被持续压缩，而越来越多的迹象也表明，华为已经做好了放弃美国市场的准备。
 <br/>
 <br/>
 其实在美国，除了华为、中兴、联想这些设备提供商，还有一家中共的通讯服务提供商，和上述设备商相互配合，共同实施长期的渗透、窃密活动，这家公司就是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国电信.html" target="_blank">
  中国电信
 </a>
 。
 <br/>
 <br/>
 10月26日，一份由美国海军战争学院和特拉维夫大学两位学者发布的研究报告显示，称自2015年以来中共一直在〝劫持西方国家关键的骨干网络〞。
 <br/>
 <br/>
 报告介绍，中共自2015年与美国政府签署了协议，禁止军方为了经济利益攻击商业网站，对商用电脑网络发动直接袭击。然而中共并未遵守协议，而是通过〝华为〞和〝中兴〞等中国国有企业代而为之。当华为、中兴活动败露并被制裁后，中国电信就悄然受命，开始利用自身的优势截获网络资讯传回给中共。
 <br/>
 <br/>
 中国电信公司在北美地区的10个入网点（point-of-presence，PoP），其中8个在美国，2个在加拿大，研究人员说，中国电信利用这些PoP，得以无缝地劫持美国国内和通过美国的跨境流量，并且在几天、几周和几个月内将其重新导向到中国。
 <br/>
 <br/>
 中国电信海外业务遍及全球，其美洲分公司，公司总部位于美国弗吉尼亚州赫恩登市，在芝加哥、洛杉矶、纽约、圣何塞设有办公室，并于加拿大多伦多和巴西圣保罗设有分支机构。其主要业务是面向企业客户提供包括中国本地及国际数据传输线路、网络数据中心服务、网络管理、设备管理、系统集成等本地化的一站式技术解决方案。
 <br/>
 <br/>
 中国电信在2011年得到AT&amp;T公司承诺为其开放美国境内网路，特别之处是，中国电信和中国移动同在2011年申请美国境内电信运营许可，中国电信在2015年获得许可并成为成为首个在美国提供自主品牌移动通信服务的中国电信运营商，而中国移动却在2018年7月被美国商务部以国家安全为由建议拒绝发放电信运营许可。中国电信的移动通信服务目前租用T-MOBILE的线路，以包括留学生在内的华人群体为基础客户，提供一卡双号通信业务。其欧洲公司也已开展类似业务。
 <br/>
 <br/>
 然而资料显示，中国电信除公开的经营业务外，还承担海外中共高等级的涉外通讯安全保障任务，为其活动提供设备和服务方案，以躲避所在国家的监管和控制。比如，2015年10月18至23日，中共领导人对英国进行〝超级国事访问〞期间所用的移动通信服务全程由中国电信CTExcelbiz提供，包括手机卡、手机套餐以及移动通信技术支援。中国电信欧洲公司自称既满足了代表团的通信需求，同时也保障了全程的通信安全，受到了代表团成员的高度好评。
 <br/>
 <br/>
 今年2月28日，苹果公司把中国内地的iCloud服务交给中共官方企业云上贵州管理，意味着中国区的1亿3,000万部苹果手机的所有资料将完全由中共〝管控〞，6月底，云上贵州把iCloud服务的运营管理权交给了中国电信，笔者认为，中国电信在海外市场运营过程中，和设备提供商如华为等公司互相配合，已经掌握了大量海外用户包括海外华人的资料，再由其来管控iCloud资料，就能实现两块资料的整合，从而榨取更多海外用户信息的情报价值。
 <br/>
 <br/>
 此外，中国电信美洲公司还主办或资助各种文化活动，稳定和扩展与中国商业关系密切的海外商户和海外华人等核心用户，这些用户无形中成为中国电信在美国实施渗透和窃密的中间媒介。
 <br/>
 <br/>
 2000年，中共为封堵法轮功信息，开始建立防火墙；恰在同一年，中国电信开始立足美国，布局海外市场，至今已苦心经营长达18年，其间更和华为、中兴等设备提供商形成市场上的战略配合，对美国渗透的深度和广度究竟达到何种程度，造成的破坏和损失有多大，或许唯有随着时间的推移才能逐渐为世人所知。
 <br/>
