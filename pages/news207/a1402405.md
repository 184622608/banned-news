### 活摘器官新证 追查国际公布17个录音调查
------------------------

<div class="wysiwyg">
 12月2日，海外追查迫害法轮功国际组织（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_追查国际.html" target="_blank">
  追查国际
 </a>
 ）发布中共活摘法轮功学员器官现状最新调查报告，同时公布17个电话调查录音，内容显示中共仍在活摘法轮功学员器官进行移植。
 <br/>
 <br/>
 该报告调查时间从2018年10月19日到11月16日。
 <br/>
 <br/>
 调查对象主要针对中国顶级器官移植医院的院长、主任，很多是国家器官移植专业领域的项目负责人、国家级省级技术专家，以及迫害法轮功专职机构的负责人。
 <br/>
 <br/>
 这17个电话调查录音，涉及12家医院，分布在三个直辖市和八个省的主要城市。有北京、天津、上海、烟台、郑州、杭州、长沙、南京、广州、广西等地。
 <br/>
 <br/>
 报告内容显示：
 <br/>
 <br/>
 一、在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_追查国际.html" target="_blank">
  追查国际
 </a>
 名单上的医院和医生，都没有停手，仍旧在活摘法轮功学员器官。
 <br/>
 <br/>
 整体调查录音内容显示：对〝你们是还在用法轮功学员器官对吧？〞的问题，有11名被调查者，涉及9家医院，都做了肯定的回答。
 <br/>
 <br/>
 例如：武警北京总医院肝移植主任陈新国在调查员要求提供法轮功供体时，回答：〝好！好！〞
 <br/>
 <br/>
 武警北京总医院器官移植研究所副主任医师王建立，在被问到，你们现在用的器官是不是都是法轮功那个健康器官时，回答：〝对、对、对。〞
 <br/>
 <br/>
 北京朝阳医院肝移植主任郎韧，被问到供体，是否是法轮功的、那种正常健康供体时，回答：〝对对，没错没错，你说的对。〞
 <br/>
 <br/>
 此外，其他人的回答虽然含糊其辞，但却无一人矢口否认他们在用法轮功学员器官。
 <br/>
 <br/>
 例如对中国器官获取组织联盟（OPO）主席郑树森的调查，调查员问：你们现在器官还是法轮功那个健康的正常的器官，对不对？郑答：〝嘿嘿，……，器官来了我们就要化验过的。〞
 <br/>
 <br/>
 报告说，郑树森故意答非所问，但他没否认在用法轮功学员器官。
 <br/>
 <br/>
 郑树森曾是中共迫害法轮功的重要机构——浙江省反邪教协会的前任理事长。
 <br/>
 <br/>
 <h4>
  二、保证在一两周内可安排手术
 </h4>
 <br/>
 对手术等待时间，被调查的院长、主任基本都保证在一至两周内可安排。
 <br/>
 <br/>
 上海仁济医院肝移植中心主任杭化莲说：〝我争取一个礼拜之内帮你搞定。〞山东烟台毓璜顶医院肾移植主任柳东夫答：〝这个快的话，可能明天就有啊！〞
 <br/>
 <br/>
 <h4>
  三、披露移植供体来自〝脑死亡中心〞
 </h4>
 <br/>
 南京医科大学第一附属医院肝移植中心主任王学浩说，他们使用的器官，大部分是院外的，从脑死亡中心取来的。
 <br/>
 <br/>
 <h4>
  四、器官同步切取移植
 </h4>
 <br/>
 多家医院承认，他们使用的脑死亡捐献器官，是边切取器官边同步移植。报告说，热缺血时间和冷缺血时间都极短。这与公民死后自愿捐献情况不符。
 <br/>
 <br/>
 <h4>
  五、换一个肝只要二、三十万透露的信息
 </h4>
 <br/>
 做一例肝移植一般从七十万起价。但武警总院王建立主任对调查员说：〝没多少钱！二、三十万就可以了。〞
 <br/>
 <br/>
 报告说，王建立开出的〝优惠价〞，显然是在讨好〝省政法委维稳办副主任〞 。王建立对迫害法轮功的〝省政法委维稳办副主任〞头衔如此买帐，恰恰说明他们使用的器官和法轮功有关系。
 <br/>
 <br/>
 一例肝移植动辄几十万，甚至逾百万。而收多收少，可由主任一人说了算。调查中被问到供体来源时，普遍答覆：〝不知道，去问主任〞、〝有关器官来源、质量、手术时间、价格，你可以去门诊和主任面议。〞
 <br/>
 <br/>
 这些表明，移植主任在器官移植产业中掌控核心机密，握有很大权利，他们在器官移植的黑箱运作中起主导作用。
 <br/>
 <br/>
 报告说，所有这些被调查的人都是涉嫌活摘法轮功学员器官的重点医院的〝重量级〞人物。早被列入追查国际追查名单中的重点追查对象。
