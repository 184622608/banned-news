### 怎样躲避看不见的陷阱
------------------------

<div class="wysiwyg">
 二零一八年十月七日，四川达州市达川区发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_路面塌陷.html" target="_blank">
  路面塌陷
 </a>
 ，刚刚举办完婚礼仅有四天的小龙和小青夫妇，在去超市买喜糖返家的路上，不料被大坑吞噬。另一对遇难的是唐氏父子二人。截至发稿（十月九日晚九点），当地市政部门未就路面塌陷的原因作出说明。
 <br/>
 <br/>
 在十月七日播出的视频中，看着即将走到塌陷地点的人们，我本能的用手按住了屏幕上的他们，多么希望他们就此止步……然而他们还是撑着伞，大步流星地走到了突然塌陷的大坑位置。
 <br/>
 <br/>
 另一个视频中显示，两位消防人员在坑里营救时，又发生了坍塌。幸亏二人反应快，及时跳跃、爬出了正在坍塌的地点。当时坑内已不见了那四人的踪影，只有还在下陷的土。
 <br/>
 <br/>
 人们说这不是天灾，而是人祸，因为那坑里有一些管道通过。也就是说，那个路面被施工者搞成了像陷阱的装饰，随时都可吞噬生命。当人们在纷纷指责政府不作为时，已经于事无补了。而中国大陆之所以到处都存在这种情况，皆因中共灌输的〝人不为己，天诛地灭〞造成的。
 <br/>
 <br/>
 这个惨剧不禁让我想起一件事：几年前的一天，我在路上遇到一位同行的大姐，就与她边走边聊起来。她说很喜欢我的思想，我就告诉她：〝我的思想来自法轮功。现在人们都在三退保平安，希望你也能平安。〞大姐说：〝我啥也不信。〞
 <br/>
 <br/>
 听她这么说，我轻轻的挽住了她的左臂往前一指：〝大姐，如果你是我的亲姐姐，我看到你前边有个要命的坑，但你没看见，我是告诉你还是不告诉你呢？〞大姐说：〝当然得告诉我。〞我又说：〝告诉你，就会有人害我，我该咋办？〞大姐说：〝那你也得告诉我呀，你能眼瞅着亲姐姐掉坑里吗？〞
 <br/>
 <br/>
 我没吱声，只是喜悦的望着她。大姐见我这副神情，一下子恍然大悟：〝你看我真笨，我同意你说的，退出团和队组织。〞是啊，在法轮功学员心里，所有民众都是我们的亲人。中共迫害法轮功，就是不让我们去救我们的亲人。
 <br/>
 <br/>
 常有人问这样问题：知道
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功真相.html" target="_blank">
  法轮功真相
 </a>
 就能躲灾啦？是的，法轮功是佛法修炼，〝法轮大法好，真善忍好〞这九字吉言能使人得大福报。信与不信看似一念之差，却是把人放置在了生死界限的两边。〝无神论〞使人不相信那光滑的路面底下会藏有陷阱，因为肉眼看不见。
 <br/>
 <br/>
 中共害怕民众了解法轮功真相，是因为真相就像探测仪，能使人们躲开中共制造的千奇百怪的陷阱。愿人人拥有〝探测仪〞，愿人人平安。
 <br/>
 <br/>
