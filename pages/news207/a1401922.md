### 夏小强：川习会后习近平所面临的挑战
------------------------

<div class="wysiwyg">
 12月3日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 ，中美双方达成协议，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 休战三个月。几家欢乐几家愁。这个结果，未免让那些希望川普对中共贸易制裁〝一剑封喉〞的人感到失望。但是，从大国博弈和外交政治的常识角度来讲，川习会的结果，属于正常结果，并没有太多意外。
 <br/>
 <br/>
 中共和美国双方对谈判结果的不同表述，也就是中共官媒隐瞒谈判真相毫无意外，不管谈判结果如何，在中共宣传媒体的口径中，将永远是中共取得胜利或者是双赢。那么，这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 的实质结果是什么呢？
 <br/>
 <br/>
 第一，中共从此前的不承认违反世贸规则及盗窃知识产权等，到现在的同意在三个月内对这些问题展开谈判，等于默认了美方的指控，习近平做出了妥协让步姿态。
 <br/>
 <br/>
 第二，川习会暂时缓解了贸易战给中共带来的重压，中共有了三个月的喘息时间。
 <br/>
 <br/>
 第三，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 开打以来，中共最高领导人习近平首次公开走上台面，直接面对中美贸易战。这意味着，他将为中美贸易战的结果以及带来的一切后果承担责任。因此，中共最高领导人习近平，在12月3日的川习会之后，在诸多方面，面临着重大的选择。
 <br/>
 <br/>
 习近平与川普的关系
 <br/>
 川习会之后，川普总统连发推文，说与习主席有〝非常铁的私人关系〞、〝我们俩，是唯一能给我们两个大国在贸易等方面带来巨大和积极变化的人。朝鲜解决的方案，对中国和所有人来说都是一件好事！〞
 <br/>
 <br/>
 川普对习近平的评价，似乎又一次回到了之前川普与习近平之间〝惺惺相惜〞的〝化学反应〞状态。
 <br/>
 <br/>
 一个非常值得关注的地方是，川普总统这两年来，几乎一直把习近平本人与中共政权区分开来。有人认为，这是川普在外交上的习惯和策略，比如他一直也把金正恩和普京称为朋友。但是，如果站在另外的一个角度来解读的话，或许可以认为：川普与习近平谈判，川普作为美国总统，代表的是美国；习近平作为中共的最高领导人，自然代表的是中共。川普谈判的对象自然也就是中共，但是，习近平作为个体来讲，可以做出不同的选择。如今，习近平就面临着这样一个重大的选择。
 <br/>
 <br/>
 一些人对此次川习会结果感到失望，其主要原因就是对川普总统个人抱有了不符合实际的期望，认为川普要结束中共统治等等。其实，即使在历史过去之后，未来川普成为推动中共解体的英雄，但是，在目前来看，川普上任之后对中共采取的行动策略是全面遏制，主要行动集中在对美国的不公平贸易方面，其直接目的也是为美国利益，让美国再次强大，所谓有些人期待的〝解体中共〞，他从未公开表露过。至于说川普目前的策略是否在客观上起到解体中共的作用，那是另外一回事情。
 <br/>
 <br/>
 习近平背上的〝黑锅〞越来越沉
 <br/>
 在去年中共十九大之前，虽然中共高层政治博弈激烈，但是，习近平基本处于顺风顺水的状态。其上任之后反腐打虎、整顿吏治、取消劳教、提倡恢复传统文化等等措施，是顺应民意的，一度给外界带来一些期待。
 <br/>
 <br/>
 但是，在中共十九大之后，习近平当局的处境，在短时间内急转直下，如今处于内忧外患之中。其主要原因，是习近平到如今为止，没有放下他背上的〝黑锅〞，这黑锅里，装满了江泽民集团和中共历史上的罪恶，如今，这〝黑锅〞越来越沉，正在把习近平压垮。
 <br/>
 <br/>
 习近平在2012年底中共十八大之后上任执政，接过的是一个从胡温政府手中的烂摊子：江泽民集团架空胡温政府10年，中共党、政、军、地方基层，几乎全都被江派实力控制。江泽民势力控制中国20多年，几乎摧毁了中共的官场秩序、毁坏了中国的法制系统、经济上的疯狂失衡发展，造就了以江泽民家族为代表的众多中共利益集团，特别是江泽民利用国家机器，制造了国际上最大的迫害人权事件，迫害法轮功信仰团体以及活摘法轮功学员器官等等。
 <br/>
 <br/>
 其实，任何一个中共的最高领导人，包括习近平在内，在上任的第一天起，在中共最高领导人的位置上，自然也就背负起了中共的〝原罪〞，背负起了其前任江泽民留下的〝黑锅〞。但是，习近平上任之后头几年的行动，一度让外界以为，习近平可能将要丢下江泽民留下的这口〝黑锅〞：当习近平拿下江泽民的马仔周永康，本可以一鼓作气，拿下中国腐败的总教练江泽民，一举卸下江泽民压在他身上的沉重负担，本可以把造成中国社会危机的罪责，让江泽民集团来承担。但是，遗憾的是，习近平继续把江泽民的这口〝黑锅〞背负在身上，并且江泽民集团和维持中共体制本身存在的本能作恶，又不断往〝黑锅〞中放进新的罪恶，造成习近平如今的困局。
 <br/>
 <br/>
 在中美贸易战的背景下，习近平在5年反腐打虎中所触及的中共既得利益集团，以江泽民集团为代表，正在利用一切机会，调动全部资源，在日益加剧的中国社会乱局中，以维稳为名，制造新的危机和罪恶，比如加剧迫害法轮功和新疆维吾尔人等，出台各种新的迫害民众的措施，激化社会矛盾，让习近平背负罪名和〝黑锅〞。
 <br/>
 <br/>
 习近平所面临的挑战
 <br/>
 习近平当局如今面临的困局，并不是采用拖延欺骗战术与美国谈判就能够解决的。即使到三个月之后，中共侥幸再取得一段时间的拖延，也无法解决其深层危机。
 <br/>
 <br/>
 中国面临的根本问题，是中共体制本身，与中国社会、中国经济、世界普世价值以及中国民众的根本利益之间的根本矛盾和对立造成的。
 <br/>
 <br/>
 维持中共体制本身需要暴力和谎言，这一点永远无法改变。而中共体制和其信奉的马克思主义意识形态，都是反人类和反人类普世价值的，都已被全世界所唾弃。
 <br/>
 <br/>
 因此，坚持中共的统治模式、意识形态，必然无法与信奉普世价值的世界共存。在中共的体制下，其实从根本上无法找到解决中美贸易冲突的方法。
 <br/>
 <br/>
 因此，习近平所面临着大的挑战。从表面上看，中方在经济贸易方面要从结构上做出改变，但是，中国经济如果从结构上做出改变，其实，已经脱离了中共的统治模式，在中共体制下，中国经济无法做出结构性改变。
 <br/>
 <br/>
 中共的体制性腐败，使得习近平当局的反腐无法从根本上解决中共的腐败问题，最起码，连江泽民家族的腐败都无法撼动。所以，中国的金融危机无法从根本上解除，在中共体制下，崩盘只是时间问题。
 <br/>
 <br/>
 中国的当权者，如果继续死抱着马克思共产主义意识形态和制度不放，无论如何努力都是费尽心机，不仅一事无成，还会被共产主义死死捆绑，继续给中国和世界带来灾难，个人最终也会身败名裂。
 <br/>
 <br/>
 共产主义和共产主义政权在世界的灭亡，是大势所趋，天意使然。纵观中国历史上，那些敬天信神、顺天而行的君主，往往获得巨大成功，可以获得神佛护佑，从而开创千古盛世。
 <br/>
 <br/>
 中国如今的当政者，如果能够抓捕中共最邪恶的代表、魔鬼江泽民，结束对法轮功和普世价值〝真、善、忍〞的迫害，主动抛弃共产主义意识形态和政权，与普世价值接轨，回归传统，就可以摆脱危局，使中国社会平稳过渡到未来，同时成就自己的历史地位。
 <br/>
 <br/>
