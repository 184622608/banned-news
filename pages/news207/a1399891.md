### 文武：中共不可能真的开放
------------------------

<div class="wysiwyg">
 中共无论是在上海的进博会上、还是在新加坡的东盟峰会上，都在高调宣称要扩大开放。
 <br/>
 <br/>
 用中共领导人在新加坡的话来说：你们问的所有问题我全都回答了，中共就是要开放，总结一句话就是你们来中国投资吧。
 <br/>
 <br/>
 说穿了就是中共要做的不是开放，而是要别人把钱投到中国来。
 <br/>
 <br/>
 中共说要真的开放，要把国门打开，可是中共为什么还在封锁互联网呢？中共为什么还在禁止言论自由呢？开放了就应该与国际接轨，就应该以普世价值为行为准则，可是中共为什么还要与普世价值相违背呢？
 <br/>
 <br/>
 中共在求人的时候什么都可以答应，说的比唱的还好听，一旦中共渡过难关，马上就翻脸不认人了。
 <br/>
 <br/>
 中共在求中国民众帮助中共推翻国民党政府的时候，中共答应推翻国民党政权后在中国建立像美国一样的政权。结果把国民党政权推翻后，中共建立的却是比世界上最独裁邪恶的政权。中共在求中国的一些民主党派帮助中共对付国民党的时候，满口答应这些民主党派，要在中国建立一个多党执政的民主政府，结果中共篡政后，马上翻脸不认人，把这些民主党派只当做花瓶党，只是当个摆设而已。
 <br/>
 <br/>
 中共在求美国要加入世贸组织的时候，满口答应美国一定会遵守世贸组织的规则，结果中共得逞后，第二天就把世贸组织的规则抛到九霄云外去了。
 <br/>
 <br/>
 这次中共求美国解除对中共的贸易制裁，中共这次也是准备满口答应美国会开放市场，会按照美国的要求开放中国的市场。
 <br/>
 <br/>
 一旦美国真的解开了对中共的制裁，第二天中共就会换个法子来阻扰美国公平的进入中国市场。
 <br/>
 <br/>
 千言万语就一条，千万不要相信中共，如果相信母猪会上树，那也不能相信中共的承诺。
 <br/>
 <br/>
 中共这次遇到最大的危机就是外资的减少，因为中共的经济是泡沫经济，靠的就是外资支撑着，一旦外资撤离到一定的程度，中共的经济泡沫就破裂，中共的经济就完了，中共的政权也就支撑不下去了。
 <br/>
 <br/>
 中共这一次为了能引进外资，可以说什么条件都会答应美国，只要外资进来了，解决了中共当前的危机，中共马上就会翻脸，中共马上就会换个法子来对抗国际社会。
 <br/>
 <br/>
 国际社会对付中共的唯一办法就是制裁，一直制裁到中共政权倒台。然后国际社会再与没有中共的中国政府谈公平的贸易。
 <br/>
 <br/>
 ──转自《大纪元》本文只代表作者的观点和陈述。
 <br/>
 <br/>
