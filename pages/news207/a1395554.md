### 5个共产党不想让你知道的惊心动魄的事实
------------------------

<div class="wysiwyg">
 5个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产党.html" target="_blank">
  共产党
 </a>
 不想让你知道的惊心动魄的事实
 <br/>
 <br/>
 1. 共产教主——卡尔．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克思.html" target="_blank">
  马克思
 </a>
 是个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_撒旦.html" target="_blank">
  撒旦
 </a>
 教徒
 <br/>
 <br/>
 马克思在少年时本是一个基督徒；
 <br/>
 <br/>
 但进入大学的时候，他加入了由Joanna Southcott主持的撒旦教会从而变成了该魔鬼教派的成员。
 <br/>
 <br/>
 马克思并不是无神论者，撒旦教派相信神的存在只不过他们仇恨神，想超越神，爬到比神更高的位置，至少与神平等的位置，所以马克思的许多诗歌作品里都反对神而崇拜撒旦。
 <br/>
 <br/>
 诗歌是文艺的一种特殊方式，它可以最真实地表达出诗人的内心。 马克思在他的一些作品里如《绝望者的魔咒》（Invocation of One in Despair),《人之傲》（Human Pride）,〝Oulanem〞等，都充满着毁灭人类的渴望。
 <br/>
 <br/>
 甚至在《演奏者》一诗中他公开地承认：〝与撒旦签约出卖灵魂　打上撒旦印记　为撒旦代言　让全人类下地狱〞
 <br/>
 <br/>
 2. 红旗与五星的来源
 <br/>
 今天的中国人不知道为何中共把五星标志像神灵一样崇拜。人们基本上都不知道在红旗上的五星标志的含义，每个国家的国旗带着各国的特色，代表本国的形象。我们完全找不到五星标志和中华传统文化的关联，因为红旗及五星标志是从苏联带入中国的，它不代表任何哪个国家所以不可能是国旗。
 <br/>
 <br/>
 3. 党旗的来源（镰锤标志）
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产党.html" target="_blank">
  共产党
 </a>
 这样宣传:〝党旗上的镰锤标志就是代表农民和工人〞，可是它们真正的意义并不是这样
 <br/>
 <br/>
 卡尔．马克思是共济会的高级成员（Freemasonry)-一个供奉撒旦的教会，共产党的〝同志〞叫法(comrades)也是从共济会而来。〝Hidden hand〞(隐藏的手）也是共济会高级成员的称号。在党旗上的镰锤标志也是从共济会而来。
 <br/>
 <br/>
 4.马克思的共产邪说是信奉撒旦主义的现代版本
 <br/>
 信奉撒旦教及道德败坏的马克思编造所谓的共产主义理论，引导人们信奉它，从而达到他毁灭人类心灵的目的。
 <br/>
 <br/>
 实质上，马克思的共产邪说就是撒旦主义的现代版，只不过穿上了共产主义的外衣来蒙住人们的眼睛。
 <br/>
 它为了拉人下地狱而呼唤人们放弃神，出卖灵魂换来欲望的放纵。
 <br/>
 <br/>
 5.入党仪式就是撒旦教加入仪式
 <br/>
 <br/>
 撒旦教加入仪式
 <br/>
 站在五星标志前边举高右手，〝发毒誓〞愿意为组织牺牲生命，对组织忠诚一生……
 <br/>
 <br/>
 加入共产党仪式，站在五星标志前边举高右手，〝发毒誓〞愿意为党牺牲及永远不会背叛党……
 <br/>
 宣誓入团后,青年们就算是已宣誓加入撒旦教会；在党旗前边发誓后，人们就是已发誓把生命交给魔鬼。
 <br/>
 <br/>
 在〝圣经启示录〞里写道，〝残暴及凶悍〞〝它的名字是魔鬼或撒旦，迷惑全人类〞在人间就是一个嗜血成性、崇拜暴力的共产党。
 <br/>
 <br/>
 这条〝红龙〞通过〝入党、入团、入队〞的形式在人身上打上它的〝印记〞，这个〝兽印〞就是和魔鬼走到地狱的通行证。
