### 袁斌：中共为何不敢公布川习会关键成果？
------------------------

<div class="wysiwyg">
 备受关注的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞已经落幕，对比中美双方目前发布的消息，可以说既有相同的部分，也有不同的部分。
 <br/>
 <br/>
 哪些部分相同呢？
 <br/>
 <br/>
 一是双方都认为会谈的气氛是友好坦承的，而且对会谈的结果都表示满意；二是双方都说中方将开放市场，从美国进口更多的产品，以减少两国之间的贸易不平衡；三是美国政府对中国2000亿产品征收10%的加征关税原定在明年1月1日提高到25%，现在双方都称两国元首达成共识，明年1月1日对这些产品加征的关税仍维持在10%，而且不再对新的产品加征新的关税；四是双方都表示为了达成协议，取消现在仍然加征的关税，接下来中美将加紧谈判。
 <br/>
 <br/>
 那么哪些部分不同呢？
 <br/>
 <br/>
 一是美方称两国元首已同意立即开始就强迫技术转让，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_智慧财产权.html" target="_blank">
  智慧财产权
 </a>
 保护，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非关税壁垒.html" target="_blank">
  非关税壁垒
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网路入侵.html" target="_blank">
  网路入侵
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网路盗窃.html" target="_blank">
  网路盗窃
 </a>
 ，服务和农业等方面的结构变革进行谈判，且将尽力在未来90天内完成此项交易；二是如果在这段时间结束时，双方无法达成协议，10％的关税将提高到25％，也就是说明年1月1日对中国2000亿产品仍征收10%的关税仅仅是暂时的。我以为，这两点可以说是此次〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞最关键的会谈成果。但中共发布的官方消息却对上述两点只字不提，只是说〝两国经济团队加紧磋商，朝着取消所有加征关税的方向，达成互利双赢的具体协定。〞
 <br/>
 <br/>
 这下问题来了：既然是关键成果，为何中共对此闭口不谈？难道是不小心〝漏掉〞了？非也。
 <br/>
 <br/>
 是凡了解中美贸易战背景的人都知道，自从这场没有硝烟的战争打响后，中共一直声称要坚守核心利益和底线。什么是中共的核心利益和底线？就是接下来的90天里，中美将〝加紧磋商〞的强迫技术转让、智慧财产权保护、非关税壁垒、网路入侵和网路盗窃等问题。
 <br/>
 <br/>
 回顾贸易战五个月以来的过程，中共从一开始时的以牙还牙，与美国硬碰硬对抗，到在坚守核心利益和底线的前提之下有所退让，再到如今将就与核心利益和底线相关的结构性变革与美国进行谈判，其态度可谓经历了多次变化。如果这次美方公布的消息属实，那就是说中共在这次会谈中已经表示即便在核心利益上也不是不可以让步，即便事关底线也不是不可以有所突破，虽然让步和突破的程度都还是未知数，都还要谈，而之前中共的态度是别的问题可以谈，唯独核心利益不能让步，底线不能突破。
 <br/>
 <br/>
 中共的态度之所以会发生这样的变化，是因为在贸易战刚开打的时候，它对美国政府发动这场战争的意志和意图严重误判，但随着贸易战的不断升温，中共终于缓过神来，意识到问题的严重性了，这时它试图采取丢车保车的策略蒙混过关，哪知美国并没有善罢甘休，就此鸣金收兵，而是继续猛攻，这下中共实在扛不住了，这才有了〝川习会〞上同意与美国进行涉及〝结构性变革〞谈判的重大让步。相比较而言，尽管这次美国也做出了让步，但中共的让步显然更大。换句话说，贸易战打到今天，中共终于认怂了！
 <br/>
 <br/>
 说到这里，我想各位该明白中共公布的官方消息为什么闭口不谈此次〝川习会〞的关键成果了吧。试想，如果中共让民众知道，美国对2000亿中国产品加征的关税明年1月1日不提高到25%其实是有前提的，那就是在接下来中美将就强迫技术转让、智慧财产权保护、非关税壁垒、网路入侵和网路盗窃、服务和农业等方面的结构变革进行谈判，且将尽力在未来90天内完成此项交易，否则这段时间结束时，10％的关税仍将提高到25％，人们势必会问：说好的以牙还牙呢？说好的坚守核心利益和底线呢？怎么这下变卦，全都作废了？那一来中共在全中国人民面前的面子往哪搁？光伟正不是又成了笑话了吗？！
 <br/>
