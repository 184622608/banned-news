### 陈思敏：赖小民案情惊人 涉曾庆红家族
------------------------

<div class="wysiwyg">
 11月7日，中共最高检通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 案件，已由监察委调查终结，移送天津检察院审查起诉。
 <br/>
 <br/>
 中共央企
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融.html" target="_blank">
  华融
 </a>
 资产管理公司前董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 ，今年4月落马至今，不断爆出令人咋舌的案情：家藏3吨重现金、总值相当于2.7亿元人民币，还有〝3个100〞——100多套房、100多个关系人、100多个情妇，以及涉案超过16亿元人民币，这还不包括他实际拥有的房产等其他贵重资产，其牵涉关系人数以百计，现有办案力量没有3年难以办完等等。
 <br/>
 <br/>
 赖小民案线索错综复杂，不过线头已被提起。此前报导，赖小民调查案件的核心，是与华融有着铁三角关系的两家公司——宁夏天元锰业集团和中国港桥控股公司。而在天元系、中国港桥担任高层的刘廷安，是赖小民的江西同乡兼大学同学。
 <br/>
 <br/>
 工商资料显示，中国港桥并非公司原名，现名是在十九大前2017年3月变更而来的。在这之前，中国港桥叫至卓国际控股公司，2016年3月，至卓国际董事会宣布任命刘廷安为董事会主席。2016年下半年，宁夏天元成为至卓国际的股东。
 <br/>
 <br/>
 至卓国际主要生产并出口印刷线路板，旗下多个子公司，最主要的一家是至卓飞高线路板（深圳）有限公司，截至目前，该公司法定代表人的名字叫潘军。
 <br/>
 <br/>
 2012年11月，至卓国际将至卓飞高70%股权、深圳厂两块土地及其上的公司大楼，出让给两个买家，其中最大买家是花样年团旗下的深圳花样年。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花样年集团.html" target="_blank">
  花样年集团
 </a>
 派任至卓飞高的新董事长潘军，是时任（也是现任）花样年团董事会主席、执行董事及首席执行官。官方简历显示，潘军于1999年加盟花样年集团并负责集团专案的整体运作、制定集团的发展策略、监督计画项目以及集团的业务及营运管理。现为花样年集团（中国）有限公司总裁、深圳市花样年地产集团有限公司总裁及本集团多家子公司的董事，同时任花样年集团董事会主席、执行董事、首席执行官、提名委员会主席及薪酬委员会成员。潘军是花样年集团资深核心高层。
 <br/>
 <br/>
 花样年控股为中国地产开发商，于1996年由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾宝宝.html" target="_blank">
  曾宝宝
 </a>
 （亦称曾洁）成立，公司主要在深圳和成都等地开发房地产项目，公司注册地设于开曼群岛，公司总部则在广东省深圳市福田区。
 <br/>
 <br/>
 而曾宝宝是前中共中央政治局常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾庆红.html" target="_blank">
  曾庆红
 </a>
 的侄女，曾庆红被外媒称为前中共党魁江泽民的头号心腹，曾庆红家族控制着巨大财富。
 <br/>
 <br/>
 今年8月即赖小民落马后，工商资料显示，花样年在未经公告下，把至卓飞高的股权转让给一家名不见经传的公司，该公司仅有两名自然人股东，龚某和黄某，两人名字和深圳花样年一位高管、花样年一位员工的名字一模一样。至今花样年的首席执行官潘军仍被登记为至卓飞高的法定代表人。
 <br/>
 <br/>
 还有一些公开资料显示，在华融卷入至卓（中国港桥）之前，赖小民和华融已与曾氏家族建立了关系，而且存在重大交易。
 <br/>
 <br/>
 虽然至卓（中国港桥）与花样年的关联关系还没被广泛报导，但官媒报导借业内人士谈话指出，赖小民贪污腐败很早就开始，其实还涉及到另一个问题——〝谁是他的保护伞〞。
 <br/>
 <br/>
 确实如此，在金融业中，不良资产处置是最赚钱的一块，华融还处于央企垄断地位，其一把手职权和油水之大可想而知。按中共官场惯例，这样的位置不是随便能坐上，这也表示华融赖小民有必须服务的权力后台，央企掌门人不过是高层后台的金库出纳员。
 <br/>
 <br/>
 像华融这样的巨无霸央企，赖小民为自己或为后台敛财的主要方式，不是受贿索贿，而是直接转移国有财产，数额不是以亿为单位，而是以百亿为单位，这样的转帐需要分散借助很多外部公司体外循环，但至卓（中国港桥）这个线头已被抓住，如果当局有决心要查个水落石出，并非什么难事。
 <br/>
 <br/>
