### 古玉文：美两派学者共同发声 剑封中共
------------------------

<div class="wysiwyg">
 〝同心之言，其利断金。二人同心，其臭如兰。〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会前夕，美国亲中（共）反共两派学者重拾两千年前中国古老易经智慧，纷纷发表同心戮共宣言，助威川普谈判团队阿根廷之行，令中共大为焦灼不安。
 <br/>
 <br/>
 数十名学者称对中共希望〝幻灭〞
 <br/>
 <br/>
 11月30日，美国之音报道，美国斯坦福大学胡佛研究所周四公布了一份旨在反制中共渗透，题为《中国影响力与美国利益，提高建设性警惕》的学术性报告，报告揭示中共日益扩张的影响力正在危害美国的民主价值，敦促美国政府应采取〝针锋相对〞的报复性措施。
 <br/>
 <br/>
 该报告是由美国等7个国家的32位中国问题学者和专家所共同撰写，详细揭露中共对美国大学、智库、媒体、侨界、企业、科研等领域试图进行深度渗透和共产主义意识影响的活动。但令中共不愿面对的是，报告的撰写者们多为被外界认为的亲共学者。
 <br/>
 <br/>
 撰稿人之一胡福研究所资深研究员戴雅门说，〝一名研究中国（共）的美国大学教授告诉我们，他的大学只允许他秘密查看该大学跟孔子学院签订的合同，就像中央情报局（CIA）的活动，你要进入一个房间，不能记笔记，只能读。〞
 <br/>
 <br/>
 另一位撰稿人美国外交关系委员会高级研究员、亚洲项目主任易明在谈到中共对美国智库进行收买和威逼时说：〝有智库的负责人告诉该智库的美国学者，他不应该参加某个会议，因为中国（共）人要求他不到场。我觉得这令人不安。这不是大多数情况，但它确实发生了。〞这使得〝自由民主社会的抵御能力下降。〞
 <br/>
 <br/>
 报告还抨击中共实施岛屿军事化，建造集中营，进行种族清洗，犯下危害人类罪等扩张性危险行为，建言美国政府针对中共的策略应采取〝对等〞原则，打击中共。
 <br/>
 <br/>
 胡佛研究所公布这份报告，一方面佐证了前期美国亲华派官员佐利克、基辛格关于中美关系再也回不到过去的论断，另一方面正值
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 中美峰会前夕之际，报告论点直指中共痛楚，显示出美国左派学者反共意识与川普已经完全合拍。这对中共来说犹如雪上加霜。
 <br/>
 <br/>
 《致命中国》共同作者 赞川普精准打击中共
 <br/>
 《致命中国》是由白宫经济顾问纳瓦罗和南加州大学商学院客座教授、〝繁荣美国联盟〞高级经济学家GREG AUTRY（中文名〝安一鸣〞为英文大纪元时报记者曾铮所起）共同所着。
 <br/>
 <br/>
 日前，美国《外交政策》杂志发表了一篇安一鸣撰写的专业评论《川普的对华政策获胜—总统的贸易战正使得北京陷入困境》，盛赞川普清醒地研判〝中共是国际贸易规则的破坏者〞，并通过系列方案智慧地扭转了美国之前总统对中共的绥靖政策。
 <br/>
 <br/>
 安一鸣从消费学角度向外界解读了川普总统〝从战略上对具有高进口弹性的商品征收进口关税〞， 这样〝滋养的是美国经济，而不是中共〞， 因为25%的关税中的20.5%部分是由中共出口厂商承担，而不是美国消费者。而价格上涨，也只能导致供应商产业链改道，从中国大转移。
 <br/>
 <br/>
 安一鸣同时剖析了川普对中共在重关税压力下采取的隐蔽性转口贸易措施，早就做出防范，美墨加排共贸易协议、美韩双边贸易协定完美地提前完成了高难度防范动作。
 <br/>
 <br/>
 作为《致命中国》的作者之一，安一鸣非常清晰明了中共的邪恶本质。就人们对G20谈判的期待，他提醒人们放弃对中共幻想。他建议人们读读酷似美国《权利法案》的中共宪法关于人权与信仰自由部分，〝可这些都是虚幻的。就像这个政权的官方名称‘人民共和国’一样。中国（共）的法治一直是西方乐于接受的谎言。中国共产党服从的是实力和权力，它不会遵守法律。〞
 <br/>
 <br/>
 安一鸣在文末呼吁〝为了美国和中国，美国总统和他的团队应该坚持到底。削弱这个蛮横的独裁政权对于经济的影响是利于世界的，到底，也是利于中国人民的。〞
 <br/>
 <br/>
 从安一鸣的文章中，我们可以知晓，〝中共不等于中国〞这个命题已经被美国和西方社会深刻地认识到了，并正在被运用到对中共的反制与打击中。G20峰会，《致命中国》两位作者，纳瓦罗和安一鸣，一个场内，一个场外，为川普政府战胜中共再下一城，献策助威。
 <br/>
 <br/>
 北京当局或将收获战略性溃败
 <br/>
 所谓〝战略性放弃〞，是指舍弃或牺牲掉一些东西而实现具备终极战略性意义目标的行为。简而言之，就是战略实施者本身要对当下的现实状况和自身处境有个清醒理智的判断，通过有所〝不为〞而达到长远或终极利益的〝有所为〞。
 <br/>
 <br/>
 需要指出的是，〝战略性放弃〞需具备两个必要条件，一是对自身处境做出客观理性判断，二是敢于采取高效能、革命性思维方式。
 <br/>
 <br/>
 通常，验证某〝战略性放弃〞是否合理的逻辑范式就是〝IF NOT〞模式。其实，除了北京当局自身以外，地球人都知道，北京当局〝战略性放弃〞核心内容是什么，那就是放弃固守中共体制的危险思维。
 <br/>
 <br/>
 就目前北京当局处境而言，可谓内外交困。市场疲软，经济下行，国库空虚，贫富悬殊，民怨沸腾，国际围堵，内斗险恶，等等等等。这是中共上百年来作恶的结果在当今社会的体现。中共历史上欠下了八千万条国人的命，如今仍然渔民百姓、豢养权贵，图谋扩张、贪得无厌。
 <br/>
 <br/>
 如果不放弃（IF NOT）中共体制，将面临着什么？左转也好，右转也好，左右开张也罢，上述的矛盾与危局无一能够破解，即将面临的是西方社会越来越强力的反制与国内民众觉醒以及体制内反叛，将何去何从？
 <br/>
 <br/>
 G20中美峰会，外媒报导川普谈判团队集中10名鹰派人物，如不懂得战略性放弃，北京当局或将面临战略性溃败。
 <br/>
 <br/>
