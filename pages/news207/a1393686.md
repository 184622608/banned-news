### 文武：中共为何不敢真的开放市场
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 尽管在所有的公开场合都高调的宣传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开放市场.html" target="_blank">
  开放市场
 </a>
 ，但实际上中共却不敢真的开放市场。
 <br/>
 <br/>
 昨天美国总统川普明确表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 要么
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开放市场.html" target="_blank">
  开放市场
 </a>
 实行公平贸易，要么不与中共做生意了。
 <br/>
 <br/>
 众所周知，中共必须依靠美国的市场才能维持经济运行，可是中共宁可冒着关闭美国市场的危险也不敢真的放开中国的市场。
 <br/>
 <br/>
 那这是为什么呢？
 <br/>
 <br/>
 中共目前还能维持统治的一个重要原因就是谎言加封锁。中共的历史一片黑暗，中共的现在腐败透顶，中共经济快要崩溃，中共党内斗争你死我活。而中共在中国国内控制了所有的暴力机器和宣传机器，中共在国内一言堂，中共无论危机到什么程度，中共都说中国形势一遍大好，这样，就大面积的蒙骗了中国的民众，由于人天生的弱点，容易相信政府说的，容易相信媒体宣传的，人这种先入行为主的弱点，被中共利用得驾轻就熟，所以，中共深知封锁的重要，中共深知只有封锁才能挡住真相，只有封锁才能控制中国社会的表面平静，只有封锁才能维持中共暂时的统治。
 <br/>
 <br/>
 中共出于多活一天算一天的想法，所以中共绝不会解除封锁，所以中共绝不会真的开放市场。
 <br/>
 中共一旦真的打开了市场，那么互联网就得真的开放，信息就得自由流通，那么真相就会在中国大陆广泛传播，那么中共的谎言就会全部被揭穿。
 <br/>
 <br/>
 中共一旦真的开放了市场，那么金融就得开放，中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融危机.html" target="_blank">
  金融危机
 </a>
 就会暴发，中共的银行就得倒闭，中国人的钱就会都存到国外的银行里面去。
 <br/>
 <br/>
 中共一旦真的开放了市场，那么中共的能源企业就会倒闭，中国的能源市场就会被外国企业占领。
 <br/>
 <br/>
 中共一旦真的开放了市场，那么中国的农产品市场就基本上都会被外国占领，中国人就会大量的购买美国的猪肉、美国的牛肉，奶粉更是外国的好。
 <br/>
 <br/>
 中共一旦真的开放了市场，试想中共还有什么可以与国际市场竟争的呢？
 <br/>
 <br/>
 中共现在能够在国际贸易上战胜美国，战胜欧盟，靠的就是中共在国内的高压，迫使中国人民变成奴隶，生产奴工产品，再加上中共的政府干预，加上中共盗窃的行为，从而取得中共贸易上的优势。如果中共一旦真的开放了市场，实现公平贸易，那么中共所有的恶行就会失效，中共就变得一无所有。
 <br/>
 <br/>
 所以中共宁可不贸易，中共宁可闭关锁国，中共也不可能真的开放市场。
 <br/>
 <br/>
 要想真的打开中国市场，只有一个办法，那就是解体中共，中共解体了，中国的市场自然就真的打开了，中国百姓都愿意真的开放中国的市场。
 <br/>
 <br/>
 文武
 <br/>
 2018年10月1日星期一
 <br/>
 <br/>

 <br/>
