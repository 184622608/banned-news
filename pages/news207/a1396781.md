### 文武：日本也要停止对中共政府的援助了
------------------------

<div class="wysiwyg">
 1979年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 政府为了支持中国改革开放建设，同意给中国政府提供
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_援助.html" target="_blank">
  援助
 </a>
 。这三十九年来，日本政府已经为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共政府.html" target="_blank">
  中共政府
 </a>
 提供了2300多亿元政府援助。
 <br/>
 <br/>
 当年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 政府愿意给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共政府.html" target="_blank">
  中共政府
 </a>
 提供
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_援助.html" target="_blank">
  援助
 </a>
 ，是因为中共向日本向世界保证要发展市场经济，要改善中国人的生活水平，要改善中国人的人权。
 <br/>
 <br/>
 可是39年过去了，在日本、在美国、在世界各主要民主国家的援助下，中国的经济是发展了，中共政府也成为世界上最有钱的政府了。可是，最后换来的并不是中国人民的生活改善，并不是中国人的人权改善，而是中共更加强权了，而是中共更加贪腐了，中共政府有钱了并不是惠及中国人民，并不是惠及世界人民，而且更加危害中国人民，更加危害世界人民。
 <br/>
 <br/>
 如此说来，当年日本政府给中共政府提供援助就成了好心办坏事，钱花找罪受。
 <br/>
 <br/>
 中共有钱后，在国内国际排挤打压日本，仅仅只是恶搞日本的中共的抗日剧，在中国就不知道拍了多少部，把日本人描写成鬼子、描写成草包，描写成仅仅只是妇女儿童们拿着锅盖就把日本人打得落花流水的草包，在日本援助中共这几十年来，日本人成了被中共玩弄的笑柄。
 <br/>
 <br/>
 中共总是自以为聪明，把谁都当成了傻子，特别是谁对他好谁 就是傻子，美国政府帮助中共最多，结果美国政府最被中共瞧不起，中共一次又一次的玩弄美国政府，一次又一次的占美国的便宜；其次日本政府对中共好，中共刚开始搞改革开放的时候，对中共帮助最大的就是日本，这样日本也就被中共修理得也就最惨，连日本的首相想去自己的靖国神社都得看中共的眼色，可见帮助中共是个什么样的下场。
 <br/>
 <br/>
 今天时代不同，川普总统上台后，首先就是加强美国的军事，提高美军战斗力，接着就是对付中共，向中共说不。用川普的话来说，中共不要把美国人都当成傻子，美国也有聪明人，言外之意川普就是个聪明人。
 <br/>
 <br/>
 川普确实是个聪明人，知道中共只是外强中干，中共并没有什么好怕的，只管用强硬手段对付中共就是了，中共就是欺软怕硬，用任何讲道理的方式来对付中共都没有用，跟中共只能用实力说话，别的都没有用。
 <br/>
 <br/>
 于是，川普开始对中共加关税，并且明令不允许中共报复，如果中共报复就继续增加关税，而且加关税只是对付中共诸多手段的开始，还有很多对付中共的手段正在准备着用。
 <br/>
 <br/>
 美国开始修理中共了，中共就开始害怕了，中共就开始服软了，中共就开始认输了，中共纸老虎的面目就开始暴露了。
 <br/>
 <br/>
 日本政府一看，原来中共只是个纸老虎，原来中共并没有什么好害的，于是，日本政府再也不怕得罪中共了，再也不想给中共政府援助了。
 <br/>
 <br/>
 文武
 <br/>
 2018年10月24日星期三
 <br/>
 <br/>
 作者提供 本文只代表作者的观点和陈述。
 <br/>
 <br/>
