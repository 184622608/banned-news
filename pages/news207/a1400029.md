### 袁斌：大陆时评家为何被迫告别推特？
------------------------

<div class="wysiwyg">
 〝做为一个实名写作的时评人，过去十五年留下四千多篇时评，可是现在，为了自己能够苟活下去，不得不放弃艰苦的写作，不得不告别海内外千千万万读者朋友，我不知道什么时候还能在第一时间把最及时最准确的时评发出来，但愿这个时间不会太长，但愿黑夜快快过去。〞11月11日，大陆最知名的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_时评家.html" target="_blank">
  时评家
 </a>
 之一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡慎坤.html" target="_blank">
  蔡慎坤
 </a>
 在推特上如此向读者道别。
 <br/>
 <br/>
 作为蜚声大陆网路的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_时评家.html" target="_blank">
  时评家
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡慎坤.html" target="_blank">
  蔡慎坤
 </a>
 身兼公知、学者、投资专家等多重身份，曾先后在政府、媒体、企业供职，着有《谁来拯救中国股市》《股民辞典》等书。他的时评不仅以面广量大著称，而且立足普世价值，紧跟社会热点，观点鲜明，论据充实，分析透彻，一针见血，敢言人不敢言之言，犹如一把把刺破中共谎言的利剑，在揭示真相唤醒民众方面起到了他人不可替代的巨大作用，产生了广泛的社会影响。也正因为如此，中共宣传管控部门对他十分头疼，必欲除之而后快。
 <br/>
 <br/>
 不过，中共要除去的这回远不止是蔡慎坤一人。据海外媒体报导，近日，不少使用海外推特的大陆网民，被中共有关部门约谈要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关闭帐号.html" target="_blank">
  关闭帐号
 </a>
 或
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_删除推特.html" target="_blank">
  删除推特
 </a>
 内容，甚至有人因发送推文而遭到拘留、逮捕。海外政论杂志《北京之春》消息称，推特内容被中共当局删除的人士包括，学者高瑜、陈永苗、记者文涛、律师覃永沛、网路观察员古河、刘强本、维权人士何德普等人。而拒绝配合中共当局删除推特内容的人权活动人士中，重庆刘继春已被逮捕，居住福建厦门市的潘细佃已被公安局扣押。在这场清网行动中受冲击的人数，目前还在不断增多。
 <br/>
 <br/>
 对于大陆的自媒体，中共一向可以说是想封就封，想封杀多少就封杀多少，想什么时候封杀就什么时候封杀，可对于推特这样的国外社交媒体就没法这么干了。但中共并未因此就善罢甘休。凡是像蔡慎坤这种用实名在推特上注册的大陆网民，中共可以在国内对其施加各种各样的压力，迫使他们自己关闭帐户，或删除推特内容，事实上中共也正是这么干的。我们不知道蔡慎坤在被迫关闭自己的推特帐户之前遇到了来自中共的怎样的压力，中共为了在推特上封住他的嘴又施展了何种手段，但从蔡慎坤在推特上发布的〝现在，为了自己能够苟活下去，不得不放弃艰苦的写作，不得不告别海内外千千万万读者朋友〞这段告白来看，有一点可以肯定，那就是蔡慎坤受到的压力很大，而且是他从事时评写作15年来从来没有遇到过的。
 <br/>
 <br/>
 留心大陆时局的读者不难发现，对蔡慎坤们的打压不过是近期中共新一轮网路清剿的一个环节。据中共官媒报导，从10月20日，中共网信办等有关部门，对自媒体帐号开展了集中清理整治专项行动。近期已全网处置（关闭）9,800多个自媒体帐号。11月12日，网信办又约谈腾讯微信、新浪微博等自媒体平台并提出严重警告。
 <br/>
 <br/>
 如果说中共以往的网路清剿仅限于大陆，那么这回对推特的清网则意味着中共它对网路舆论的钳制已经跨出了国门。换句话说，中共不仅要管大陆网民在大陆网路的言论，连大陆网民在国外社交媒体上的发言也要管。由此可见中共对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自由言论.html" target="_blank">
  自由言论
 </a>
 的恐惧已到了前所未有的地步！即使你在海外发声，即使这些声音对大陆影响有限，也会吓得他们寝食不安，必欲除之而后快。
 <br/>
 <br/>
 试想，什么样的政权才会对自由言论恐惧到这种地步？中共前总书记赵紫阳的政治秘书鲍彤在接受大纪元采访时说的好：如果中共都不敢让老百姓讲话，那说明这个政府知道它不能代表人民。〝政府如果认为人民是不可靠的，那么这个政府自己一定是靠不住的。〞也就是说只有不代表人民，人民靠不住的政府才会对人民的声音如此恐惧！
 <br/>
 <br/>
 换一个角度看，一个政权恐惧到草木皆兵的地步，往往是它快要灭亡的前兆。正如艺术家追魂呐喊的那样：〝为什么你要封杀微信号，因为你恐惧，为什么恐惧？因为你要完了啊……！〞
 <br/>
 <br/>
