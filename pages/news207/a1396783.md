### 夏小强：美国退出《中导条约》的真相
------------------------

<div class="wysiwyg">
 10月20日，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 宣布将退出与俄罗斯签署的《中程核导弹条约》。川普说，美国要继续坚持遵守〝中程核导弹条约〞，而俄罗斯和中国（共）在追求核武器，这是〝无法接受的〞。这是此前美国宣布退出万国邮联之后的又一〝退群〞行动。
 <br/>
 <br/>
 为何退出《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中导条约.html" target="_blank">
  中导条约
 </a>
 》
 <br/>
 <br/>
 《中导条约》是美国前总统里根和前苏联总书记戈尔巴乔夫在1987年签订的，双方按照条约将销毁全部中程和中短程导弹及其设施。
 <br/>
 <br/>
 《中导条约》本身就是冷战下的一个条约和产物，其实现在已经名存实亡。因为美国情报机构早在10年前，就怀疑俄罗斯在违约研发中短程导弹，直到去年，美国才确认了俄罗斯违约导弹的名称：9M729，它是地面发射的射程为500至5500公里的中程巡航导弹。美国表示，9M729直接对美国的欧洲盟国和亲西方的前苏联加盟共和国构成威胁。因此，在俄罗斯早已违背条约的情况下，条约其实已经失效，美国如今宣布退出，理所当然。
 <br/>
 <br/>
 更为重要的一点是，当时《中导条约》的签订，导致中国成了冷战末期全球唯一能出口该类型导弹的国家，并直接导致了1988年中国与沙特阿拉伯达成一笔中程导弹国际贸易。
 <br/>
 <br/>
 1991年苏联解体后，冷战结束，中共接替苏联，成为世界头号共产大国。经过近三十年的发展，中国成为世界第二大经济体，中共也不断国际渗透，把共产主义推向世界，是对美国和西方世界以及世界和平的最大威胁。
 <br/>
 <br/>
 近十年来，中共不断增长的导弹力量一直令美国军事战略家感到担忧。 最好的例子是中共的东风-21D反舰弹道导弹（DF-21D）。这种特殊武器专门用来对付美国的航空母舰。
 <br/>
 <br/>
 但在开发了DF-21D后，中共并没有收敛，而是对DF-21D进行了改进，制成了更高级的东风-26型弹道导弹（DF-26）。
 <br/>
 <br/>
 DF-26可以从陆地发射，拥有对陆上重要目标或海上大中型舰船实施远程打击的能力。最令人担忧的是，它可以装备核弹头。军事分析人士认为其射程可以攻击到在关岛的美国空军和海军。
 <br/>
 <br/>
 美国国家安全顾问博尔顿22日对《华尔街日报》说，中国（共）、伊朗和朝鲜不受《中导条约》的约束。美国估计，如果中共是这个条约的一方，那么中共所有弹道导弹的三分之一到一半都将违反该条约。
 <br/>
 <br/>
 中共正在建造新一代地基导弹，这些导弹直接针对美国及其在亚太盟国的基地和海军舰艇。美国因为受《中导条约》的限制，而无法与之匹敌。
 <br/>
 <br/>
 这是美国宣布退出《中导条约》的主要原因。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
   川普
  </a>
  〝退群 〞 意在中共
 </h4>
 <br/>
 正在莫斯科进行访问的白宫国家安全顾问博尔顿10月23日与俄罗斯总统普京举行会晤时透露，华盛顿方面将决定退出《中导条约》。
 <br/>
 <br/>
 普京在与被称为华盛顿鹰派代表的博尔顿会谈时一直保持微笑和幽默，并不时地用笑话调节会谈气氛，显示美方提出退出《中导条约》并未使莫斯科感到惊讶和痛心。
 <br/>
 <br/>
 一些俄罗斯专家认为，美方获悉俄方违反《中导条约》已经多年，而现在才突然宣布退出的主要原因在于北京在该领域的不受约束的发展。如果北京也签署《中导条约》的话，其火箭部队95%的导弹都将违反《中导条约》。
 <br/>
 <br/>
 对于这个早已名存实亡的导弹条约，川普宣布退出，表面上是针对俄罗斯，其实更多的意图是针对中共。
 <br/>
 <br/>
 苏联解体后，共产主义阵营倒塌，冷战结束，俄罗斯虽然仍具备大国实力，但是，已经失去单独对抗美国的能力。随着中共经济上的崛起，中共大力发展军事，中共每年不断提升军事开支，盗窃美国和西方的高科技，制定2025计划，其中很多都用在军事方面，给美国和世界安全造成极大威胁。
 <br/>
 <br/>
 川普上任一年多的时间，到目前为止，与中共的关系已经发生根本性转折。此前美国副总统彭斯发出〝讨共宣言〞，中美进入了全面对抗阶段。
 <br/>
 <br/>
 川普退出《中导条约》就是在中美关系转向、中美贸易战日趋激烈的背景下，美国在军事方面对中共实施的打击和震慑策略。
 <br/>
 <br/>
 <h4>
  引发后果和对世界格局影响
 </h4>
 <br/>
 中共对于美国宣布退出《中导条约》，指责美国〝拿中国说事〞。党媒《环球时报》发表评论说，美国的宣布可能是〝地动山摇〞之举，称〝一场激烈而危险的军备竞赛即将拉开序幕。〞
 <br/>
 <br/>
 其实，对于中共来讲，中共多年来一直在加强军事投入，并且不断对台湾发出战争威胁。外界对于世界又进入军备竞赛的的担心可以理解，毕竟人们都希望和平，恐惧战争。但是，中共政权在如今中美贸易战的重压之下，中共控制下的经济举步维艰，中共已经发出〝自力更生〞、〝共克时艰〞的口号，中共在金融和经济上面临的危机，使得其保政权成为首选，中共已经没有能力与美国展开一场长期的军备竞赛。即使一旦展开竞赛，中共政权也可能最终被拖垮。
 <br/>
 <br/>
 川普宣布退出《中导条约》，对外界的宣示味道大于对俄罗斯的实质意义，也有宣示终结二战之后形成的世界格局的意味。
 <br/>
 <br/>
 二战之后，美国承担起重建欧洲和保护欧洲的任务，几乎欧洲每一个国家，都成为二战后马歇尔计划的受益者，几十年来，欧洲在贸易和军事方面，仍然几乎无偿地接受美国的帮助。而美国对欧洲提供的帮助，特别是在军事方面提供的保护，使得欧洲诸国，节省了大量的军事国防支出，而这笔钱，被欧洲这些自由国家执政的左翼政党大搞福利社会和接受难民，往社会主义的道路上狂奔。
 <br/>
 <br/>
 如今美国宣布退出《中导条约》，这其实也意味着，欧洲诸国，在面对着俄罗斯的军事威胁下，这些国家，很有可能将要自主负担军事投入，欧洲无偿接受美国援助和高福利的好日子，可能逐渐要结束了。这其实也从客观上，使得欧洲走向正常的发展轨道，扭转二战以后几十年来的包括欧洲在内的全世界的左转趋势。
 <br/>
 <br/>
 同时，以美国、欧洲、日本为轴心联盟，遏制以中共为中心的共产国家的世界新格局正在隐隐成型。
 <br/>
 <br/>
 <h4>
  美国为何不断〝退群〞
 </h4>
 <br/>
 川普上任之后，2017年1月23日，美国退出了《跨太平洋伙伴关系协定》，即TPP；2017年10月12日，退出联合国教科文组织；2018年5月8日，退出《伊朗核问题全面协议》；2018年6月19日，美国宣布退出联合国人权理事会；2018年10月17日，美国宣布退出万国邮联。
 <br/>
 <br/>
 美国这一年多来，不断退群，表面有多种理由。美国在这些国际组织承当了高昂的费用，却成为〝冤大头〞，比如2001年，美国被盟国法国纠合中共等独裁国家，逐出人权委员会。另外，美国国内债台高筑，国力衰落，中产阶级萎缩，收入下降等等，也都是美国退群的原因。
 <br/>
 <br/>
 但是，其深层还有二战之后全球化的背景，以及全球化和共产主义的关系问题。
 <br/>
 <br/>
 九评编辑部在其新书《魔鬼在统治着我们的世界》一书中，对全球化和共产主义的关系有详细论述。
 <br/>
 <br/>
 按照马克思的理论，共产主义的实现有赖于无产阶级在世界范围内采取共同行动，共产革命必然是一个〝全球性〞的运动。全球化的运动包括政治、经济和文化方面的全球化。
 <br/>
 <br/>
 以联合国为例，二战结束后成立的联合国是全世界最大的国际组织，当初是为了加强国与国之间的合作协调。作为一个超国家概念，联合国符合了共产主义消灭国家的目的，被利用来充分扩张共产主义势力。联合国从一开始就成为了苏联所主导的共产阵营利用的工具，变成了共产党大展身手、推动共产主义〝世界政府〞的舞台。
 <br/>
 <br/>
 联合国的宪章起草者是罗斯福的重要顾问、美国国务院官员艾尔杰‧希斯（Alger Hiss），他是苏联间谍。联合国很多重要机构的负责人是共产党人或共产主义的同路人。多任联合国秘书长都是社会主义者，甚至是马克思主义者。
 <br/>
 <br/>
 中共利用联合国作为宣传平台，联合国副秘书长在2018年1月瑞士世界经济论坛上称，中共提出的〝人类命运共同体〞理念顺应世界发展潮流。
 <br/>
 <br/>
 出版于1958年揭露共产主义的名作《赤裸裸的共产党人》一书列举了共产党人的45个目标，其中一条是：〝把联合国推崇为人类唯一的希望；如果其宪章重写，就要求去设立为一个具有独立军事力量的世界政府。〞
 <br/>
 <br/>
 共产党人和全球主义者利用各种议题先在各个领域建立超国家的机构，然后推动这些机构的联合，并不断鼓吹对联合国的推崇与依赖，最终建立世界政府。
 <br/>
 <br/>
 <h4>
  川普反全球化
 </h4>
 <br/>
 川普坚决反对全球化。
 <br/>
 <br/>
 10月22日，川普在德克萨斯州休斯敦举行聚会，川普在演讲时再次表明自己对〝全球主义〞的反对立场，说〝激进的民主党人想要回到过去，恢复腐败的、渴望权力的‘全球主义者’的统治。〞
 <br/>
 <br/>
 川普接着说，〝他们口中有一个词，有点过时了，叫做‘民族主义者’……你知道我是什么人吗？我是一个‘民族主义者’。好吗？我就是一个‘民族主义者’。〞
 <br/>
 <br/>
 民族主义、右翼、保守主义，其实这些都是川普所代表的理念的不同词语表达。保守主义，是在各国政治传统中自然生发出来的，正所谓〝天然的守旧思想是人们心灵的一种倾向〞，人类天然就对激进的改变抱有怀疑和不信任，天然就在熟悉的事物和环境中获得安全感。保守主义代表着人民对土地、家庭、团体、传统、国家的爱，天经地义。
 <br/>
 <br/>
 川普宣布退出《中导条约》，以及美国退出与联合国相关的一系列国际组织和对美国不利的条约，是川普就任美国总统之后，重塑世界政治格局和重建国际秩序，推动世界全面右转，回归传统，全面遏制中共等，在国际政治层面的表现。
 <br/>
 <br/>
