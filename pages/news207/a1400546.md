### 袁斌：中共在APEC上给国际社会上了堂政治课
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 不仅一贯在国内肆无忌惮胡作非为，还把这一套也搬到了国际大家庭，而且愈演愈烈。在刚刚落幕的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 峰会上，它的一系列令人震惊的粗鲁恶行就是其流氓嘴脸的一次大曝光。
 <br/>
 <br/>
 我盘点了一下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在这次峰会上的流氓行径主要有以下三大看点。
 <br/>
 <br/>
 一是阻止各国记者采访。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 峰会期间，习近平与太平洋国家领导人举办了一个论坛，有科克群岛、斐济、密克罗尼西亚、纽埃、萨摩亚、东加、瓦努阿图和巴布亚纽几内亚等国参加。中共官员以场地和安保为由，只允许中国媒体采访，而阻止已获得APEC采访许可的外国记者入场采访，并建议他们在报导时采用新华社的通稿。被阻止入场的，还包括APEC会议东道国巴布亚纽几内亚的记者。此外，习近平在巴布亚纽几内亚莫尔斯贝港口一条公路开通仪式上演讲时，中共官员也只允许中共中央电视台录音。为此，国际记者联合会已发表书面声明，对中共官员侵犯新闻自由的恶行进行了严厉谴责。
 <br/>
 <br/>
 二是强闯主办国外交部被撵。
 <br/>
 <br/>
 美国在这次峰会上谴责中共不公平的贸易行为，得到了多数与会国家的支持。为了阻止会议联合公报中写入谴责北京的措辞，中共官员霸道的〝求见〞巴布亚纽几内亚外长帕托，但帕托认为私下会见中方官员会影响其中立性，因此拒绝了他们的要求。谁知这之后，四名中共官员竟企图强行闯入他的办公室，但最终因被巴方保安逐出巴新外交部而未能得逞。帕托表示，和自己交谈过的每一位外交官都对中共官员的这种行为感到震惊。
 <br/>
 <br/>
 三是在会议中大喊大叫。
 <br/>
 <br/>
 据美国官员披露，在APEC官方会议中，中共官员认为各国针对中共而大吼大叫。美国官员说，房间里没有其他人大喊大叫。中共官员还在会议期间发表长篇独白，他们明知道时间很紧张，世界各国领导人都等着回国。一名美国官员说，当时间耗尽，峰会正式宣言难产时，驻扎在主会场附近一个房间内的中共代表团开始鼓掌。
 <br/>
 <br/>
 谈起中共在这次峰会上的种种恶行，一位美国高级官员将其称之为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发脾气外交.html" target="_blank">
  发脾气外交
 </a>
 〞。他说，〝它们走来走去，就像它们拥有这个地方，并试图通过欺凌来获得它们想要的东西。〞 《华盛顿邮报》专栏作家Josh Rogi于20日发表题为〝中国（中共）在APEC峰会的‘发脾气外交’〞一文。文章说，会议期间，中共官方代表团做出一系列咄咄逼人、欺凌、偏执和怪异的噱头行为，企图主导会议，逼迫东道国和其他人同意其要求。
 <br/>
 <br/>
 中共在国际社会耍流氓可谓由来已久，但在APEC峰会这样高级别的国际首脑会议上如此肆无忌惮的这么干恐怕之前还没有过。近年来，中共在国际社会可以说是四处插手，到处渗透，而且做派越来越嚣张。如果说中共之所以敢在国内胡作非为是因为垄断了权力，那么它之所以敢在国际社会胡作非为则无非是因为自持是全球第二经济大国，手里有钱。试想，如果中共进一步做大，实力真的超过了美国，那时它的倡狂霸道又将会发展到何等地步！
 <br/>
 <br/>
 说穿了，中共不仅是中国人民的公害，也是国际社会的公害。千所以万别指望中共强大了就会遵守国际规则，变的文明起来，那是绝对不可能的。其实，从起家到现在，中共的流氓本性什么时候变过？如果说有什么变化，那也只是表现在，越有钱，实力越大，它就越不会把国际规则当回事，耍起流氓来就会越无所顾忌。就这点而论，中共这回在APEC上的表演岂不是给国际社会上了一堂生动而深刻的政治课？
 <br/>
 <br/>
 作者提供 本文只代表作者的观点和陈述。
 <br/>
 <br/>
