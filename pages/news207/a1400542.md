### 文武：中国民营企业对川普的期待
------------------------

<div class="wysiwyg">
 不仅美国的私营企业被中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国营企业.html" target="_blank">
  国营企业
 </a>
 占便宜，中国的私营企业也一样受中共国营企业的挤压。
 <br/>
 <br/>
 一方面中国经济上游产业基本上都被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国营企业.html" target="_blank">
  国营企业
 </a>
 垄断了，另一方面中国的私营企业根本就没有与国营企业竟争的环境与条件。
 <br/>
 <br/>
 例如，在许可证方面，国营企业获得任何行业的许可证，基本上没有门坎，只用派发就行了，因为中共的国营企业，就是原来中共政府的部、委、局的直属企业，企业负责人就是原来部、委、局的负责人，管理许可证的就是原来同一个单位的人，只是在同一个系统里调来调去。今天是局长，明天可能就是企业的董事长了，今天是处长、科长，明天可能就是总经理或者经理了。这样一来中国的民营企业如何与国营企业谈许可经营问题呢？
 <br/>
 <br/>
 至于在融资方面，民营企业也无法与国营企业相比，融资需要资产，需要业绩，需要担保，国营企业是政府的，有的是资源，需要什么条件基本上就具备什么条件。可是民营企业就做不到，因为资源有限，要满足融资方面的条件是难上加难。
 <br/>
 <br/>
 所以中国的民营企业要想发展，一般都得与中共政府的官员勾兑，需要在政府里找保护伞，这样才能争取到一定的资源与市场。
 <br/>
 <br/>
 前段时间，中共为了降低债务，为了去扛杆，就从民营企业身上开刀，好事找不到民营企业头上来，坏事首先却被碰上了。
 <br/>
 <br/>
 中共本来就是泡沫经济，要说债务，最大的债务就是中共地方政府和国营企业的债务，其次才是一些民营企业的债务，可是中共去扛杆不是从地方政府下手，不是从国营企业下手，而是从民营企业先下手，这样一来，就把大量的民营企业给搞破产了，除了一些与中共联系紧密的民营企业外，其它的民营企业，不是倒闭就是面临倒闭。
 <br/>
 <br/>
 中共一看，民营企业关系到人民的日常生活，如果民营企业全都倒闭了，那人们的日常生活的正常秩序就被打乱了，那社会就会动乱了，中共这一下又怕了，又赶紧找民营企业座谈，又赶紧喊着、叫着要支持民营企业。
 <br/>
 <br/>
 可是中共的体制不变，中共社会主义的性质不变，中共国营企业的经济制度不变，那么中共又如何去支持得了民营企业呢，就算是口号喊得震天响，那不也是干喊吗？解决不了实质问题，落实不到实处。所以，中共无能怎么高喊口号，也不可能真的支持得到民营企业，因为中共的体制决定了不可支持真正的市场经济。
 <br/>
 <br/>
 这样一来，中国的民营企业就无望了，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国民营企业.html" target="_blank">
  中国民营企业
 </a>
 在中共这种体制下不可能真的健康发展。除非与中共勾兑，但是，能与中共勾兑上的企业毕竟有限，就算勾兑上了又有几个有好下场呢？
 <br/>
 <br/>
 所以，中国民营企业现在是绝望了，现在是对中共绝望了，对中共的这种体制绝望了。
 <br/>
 <br/>
 中国的民营企业现在把希望转到国际社会上了，转到了美国，转到了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 身上。希望美国能够出手帮助中国的民营企业；希望川普出手来帮助中国的民营企业。
 <br/>
 <br/>
 那如何帮呢？
 <br/>
 <br/>
 就是帮助解决中共的国营企业的问题，不允许中共的国营企业参与市场竟争，迫使中共的国营企业全部退出市场。
 <br/>
 <br/>
 只有中共的国营企业退出市场了，中国国内的市场才可能公平与公正，国际市场也才可能有公平与公正，迫使中共的国营企业解体，既是有利于中国民营企业的好事，也是有利于世界各国的好事。
 <br/>
 <br/>
 所以，中国民营企业期待着美国，期待着川普总统，帮助解决中共国营企业的问题，帮助解体中共的国营企业。
 <br/>
 <br/>

 <br/>
 <br/>
