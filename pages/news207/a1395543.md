### 夏小强：应对贸易战 中共疯狂转嫁危机
------------------------

<div class="wysiwyg">
 美国总统川普10月14日在接受哥伦比亚广播公司访谈时说，可能会对中共征收新一轮关税，他还表示，中共已无报复能力，应与美方进行协商。
 <br/>
 <br/>
 自从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 开始的三个月里，为应对中共的不公正贸易，川普政府至今已对中国商品征收了2500亿美元商品的关税，川普多次警告说，如果北京方面报复，美方会再对2670亿中国商品加征关税。
 <br/>
 <br/>
 尽管美国开始对大批中国产品加征关税，但是，中国9月对美国贸易顺差却飙升至创纪录的341亿美元。据中共海关总署12日公布的数据显示，9月份，中国对美出口增至467亿美元，进口降至126亿美元，中国对美国贸易顺差打破8月创下的310.5亿美元纪录，达341.3亿美元。中国今年前9个月对美贸易顺差达2257.9亿美元，比去年同期的1960.1亿美元增长了15%以上。
 <br/>
 <br/>
 为什么会这样？因为中共正在开动国家机器，调动全部资源，采取多种方式来应对贸易战对中共政权的冲击，加速和加大力度，把全部的损失和危机转嫁到中国社会和中国民众身上。
 <br/>
 <br/>
 10月15日起，最新一次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降准.html" target="_blank">
  降准
 </a>
 开始实施。中共央行正式下调大型商业银行、股份制商业银行、城市商业银行、非县域农村商业银行、外资银行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 存款准备金率1个百分点。这是央行今年以来的第四次降准，此前三次分别在今年的1月、4月、6月，算上这次的降准，央行今年已经累计下调法定存款准备金率2.5个百分点。
 <br/>
 <br/>
 本次央行降准100个基点，共释放资金约1.2万亿元，加上今年前三次释放的资金，8千亿、4千亿、7千亿，今年共计释放资金3.1万亿。很有可能，今年年底将会还有一次降准。
 <br/>
 <br/>
 通俗地说，降准会造成通货膨胀，也就是老百姓手中的钱不值钱了，被稀释，等于中共把贸易战带来的损失由社会和民众承担；同时，降准对房市有利，对维持高房价起到助力。
 <br/>
 <br/>
 尽管中共一再声称，中共不会将人民币汇率作为工具来应对中美贸易争端。但是，从6月贸易战开始之后，人民币汇率从6.4左右快速贬至6.9附近，10月15日，在岸人民币贬值失守6.93关口。多数金融机构认为，人民币破7只是时间问题。虽然，人民币贬值是一把双刃剑，会加速资金逃离中国，但是，中共在保政权的压力下，即使顾此失彼，也会操纵人民币贬值，来缓解短期危机。
 <br/>
 <br/>
 另外一方面，中共正在全面左转，包括政治、经济和意识形态等多方面。中共正在发起一场新的国有化的〝运动〞，最终要把全部大型私企收入囊中，来缓解〝缺钱〞的危机，最终还是维持政权稳定。
 <br/>
 <br/>
 在此前，中共媒体已经有文章，释放〝不惜关闭股市〞来〝共克时艰〞的信号。
 <br/>
 <br/>
 几十年前，为了〝国际共产主义事业的伟大胜利〞，毛泽东曾经放出〝豪言〞，牺牲中国几亿人生命不算什么。如今中共虽然不敢公开这样叫嚣，但是，牺牲几亿中国民众的生命的准备工作，可能方案早已备下。如果认真读过《九评共产党》这本书的人，会相信这不是危言耸听。
 <br/>
 <br/>
 〝维稳〞已经成为了中共当下唯一首要的任务，中共正在开动全部国家机器，调动全部资源、不惜一切代价。
 <br/>
 <br/>
 中共已经进入最后的疯狂。这就是中美贸易战下，川普、美国政府、西方社会和全世界的每一个人，面对的中共真实的状态。任何对中共的幻想，都是不现实的和危险的。
 <br/>
 <br/>

 <br/>
