### 贸战暂停火 川普安内攘外 助习再看清中共
------------------------

<div class="wysiwyg">
 全球瞩目的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞，终于落幕。
 <br/>
 <br/>
 美方同意暂不增加关税，并将与中方针对核心问题展开结构性谈判。双方若不能在未来90天内达成协议，美方将把规模2000亿美元的商品关税，从10％提高至25％。
 <br/>
 <br/>
 适如笔者日前所论，这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 将让贸易战进入〝中场休息〞，但不会因此人去灯熄。贸易战下半场，明年可能再度登场。
 <br/>
 <br/>
 综观这次川习会，美中双方均有所让步，促成贸易战炮火不再扩大，也等于是让中方获得一次〝缓刑〞契机。
 <br/>
 <br/>
 <h4>
  贸易战烧出经济危机 中共亟盼停火
 </h4>
 <br/>
 中共，在国内严峻的经济压力下，无疑是最盼贸易战停火的一方。
 <br/>
 <br/>
 自7月贸易战开打以来，中国经济受到剧烈冲击，第三季度GDP增长率仅6.5％（此为官方数据，实际数据可能更低），创10年来新低；10月外汇储备减少超过339亿美元，降至约30530亿美元，创18个月来新低，显示资金仍持续外流避险；10月制造业采购经理人指数PMI降至50.2，创27个月新低，显示制造业持续萎缩低迷。
 <br/>
 <br/>
 加上大批企业撤离中国，许多城市出现大规模裁员、失业，以及招工广告大幅减少现象；民众收入减少，民生消费经济也随之滑落；而中国房市也在9月份首度出现供过于求，预计第四季度房价出现下跌走势。
 <br/>
 <br/>
 种种利空消息，不但衍生越来越多的社会问题、金融风险与民间抗争，也动摇着中共统治的稳定。故而，中共极盼取消关税或至少暂停增加关税，缓解经贸压力。
 <br/>
 <br/>
 而对川普来说，尽管中方承诺增加采购美国农业、能源、工业等品，换取缩减美国的贸易赤字，但最核心的〝窃取知识产权、强迫技术转移、非关税贸易壁垒〞等问题仍未获得解决。
 <br/>
 <br/>
 不过，最终川普仍同意暂缓对中方增加关税，其中或有多项考量，大致可从两方面来分述：
 <br/>
 <br/>
 <h4>
  一、贸易战不加压 川普安内攘外
 </h4>
 <br/>
 这次川习会，美方并未升级炮火、增加压力，原因除了美方希望在过年前能稳定美国与全球经贸市场外，也避免通用汽车（General Motors）裁员万人事件引发更多企业跟进或〝被鼓动跟进〞。
 <br/>
 <br/>
 同时，川普此前明确预告，若最终谈判不成，将全面提高关税到25％，连苹果iPhone都不能避免。如今，美方再多给中方一个季度的谈判时间，也等于是给在华企业多一点缓冲时间，进行产业撤离或产能外移等应变工作。
 <br/>
 <br/>
 此外，中期选举后的国会两院新议员将于明年1月上任，此刻美中双方暂缓经贸交火，也可让川普政府更能专注应对崭新的国会生态与政治挑战。
 <br/>
 <br/>
 不过，促使川普愿意稍做让步、暂时停火的主因，可能还是在于习近平。
 <br/>
 <br/>
 众所周知，曾在商场阅人无数的川普，从首度川习会之后，对习近平大为赞赏，不但屡屡公开赞扬习是他的朋友，几乎没有恶言批评。同时，尽管川普对社会主义、共产主义政权屡屡大肆抨击、严厉制裁，但他却始终明确地将习近平与中共分开而谈。
 <br/>
 <br/>
 或许在川普眼中，习近平虽身为中国领导人，但却不等于中共，就如同中国人民不等于中国共产党。
 <br/>
 <br/>
 而川普从不将习与中共混为一谈，或许是发现习近平是被中共这部党国旧机器困绑住，从而失去自主性，失去带领中国脱离中共、回归传统的方向；同时，川普也是藉此提醒习，要把自己跟中共做出切割，勿与贪污腐败、崇尚假恶斗的中共政权为伍，因为他身上肩负着十多亿人民的生命与未来。
 <br/>
 <br/>
 因此，川普暂不升级炮火，除了给习近平这位〝老朋友〞更大的善意与空间来应对贸易战外，或许也是给习近平更多时间来冷静思考，要不要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_解体中共.html" target="_blank">
  解体中共
 </a>
 这部毁弃传统文化、背离普世价值的极权机器。
 <br/>
 <br/>
 毕竟，美中的核心歧见——知识产权、技术转移、开放市场等，之所以迟迟未能达成协议，实质上是来自中共变异价值观与西方社会普世价值观的冲突。而这些变异价值观，正源自中共的九大基因：〝邪、骗、煽、斗、抢、痞、间、灭、控〞。所以，中共始终采取欺骗、谎言、掠夺等手段来从事不公平的国际贸易。
 <br/>
 <br/>
 设若习近平不愿抛弃中共旧机器，美中双方终将无法达成真正自由而公平的贸易协定，贸易战仍将加大力度撼摇中共政权。
 <br/>
 <br/>
 反过来说，若习真希望美国与中国未来和平互惠、共存共荣，则势必得解体这部残害中国百姓近70年的极权机器，才能让中国社会回归传统正常社会、找回〝仁义礼智信〞等普世价值，获得川普、美国和其它国家的尊敬与信任。
 <br/>
 <br/>
 <h4>
  二、贸易战不减压 川普箝制中共 警醒习近平
 </h4>
 <br/>
 再从另一角度讲，川习会后，双方交战火力也并未解除或放松。
 <br/>
 <br/>
 毕竟，美中之间的不公平贸易依然存在，双方核心歧见依然未解，自然没有放松施压的理由；此外，川普不放松对中共的压力，可能也得以帮助习近平保持清醒。
 <br/>
 <br/>
 近期来，海内外多个消息来源指出，习近平受到身旁部分中共佞臣围绕吹捧、逢迎拍马，让习有所迷失。
 <br/>
 <br/>
 就连习的铁杆战友王岐山，日前也在新加坡突然脱稿演讲谈道，〝所以做人啊，要保持这份冷静头脑是不容易的，因为经常有人被捧晕。这是我的一个感受。〞
 <br/>
 <br/>
 此外，也许习身边有幕僚建议他此时效仿越王勾践〝卧薪尝胆〞，将美国视为吴王夫差，先忍让一时，佯装臣服，待日后时机成熟再予以反击。但殊不知，或许中共党内反对习的极左派、江派与其它利益集团，正是把习当成吴王来吹捧迷惑、诈降效忠，再自比勾践卧薪尝胆，等待时机予以反击，夺权取位，巩固一众权贵利益。习已自陷危机而不觉。
 <br/>
 <br/>
 故而，美方维持对北京的施压力度，让北京时时感受经济压力与政权危机的步步逼近，有所痛觉，才能压制中共党国机器的狂妄扩张；同时，也藉此警醒习近平别再迷失于佞臣营造的〝厉害了，我的国〞幻象，体认到〝危机了，我的国〞现况，并清醒认识到，造成中国一切危机、造成人民一切悲苦、造成朝政一切恶斗的根本源头，正来自于中国共产党本身。
 <br/>
 <br/>
 归根结底，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 的冲突，不是美国与中国的冲突，而是自由社会与中共体制的冲突，是普世价值与变异价值的冲突。只有把变异价值彻底消解了，才能彻底化解贸易战冲突；而要彻底消解变异价值，就得彻底解体中国共产党。
 <br/>
 <br/>
 或许，川普已经看清这一点，所以愿意再帮习近平争取更多时间来思考应变。接下来，就看习自己能否真正认清中共，做出理性睿智、利国利民的选择了。
 <br/>
 <br/>
