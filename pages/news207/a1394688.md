### 崔士方：孟宏伟案闹国际丑闻 中纪委或藏隐情
------------------------

<div class="wysiwyg">
 中共公安部副部长、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 从神秘失踪到被查消息落地，都在十一长假间几天内完成。联想到范冰冰逃税案，看起来，这段本该大多数官僚机构都不办公的日子，还有多个中共高层部门在暗地里忙碌，至少包括税务总局和中纪委，还有与贸易战有关的政府高层（应对彭斯的讨共檄文）。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 的落马，因为其妻儿在法国报案，变成国际丑闻，令原本平淡的一场铡老虎，闹出了不一样的风波。
 <br/>
 <br/>
 中纪委抓捕孟本人，却让其妻儿留在法国，任其将丈夫失踪的消息捅出去，难道真的是这些官老爷笨得无药可医吗？还不好这么说。
 <br/>
 <br/>
 这里的微妙在于，孟宏伟因为这份特殊的兼职，其常规办公地就变成了法国里昂，带上妻儿同往，就在情理之中。但颇有可能，是近日发生了某种特殊的变故，迫使当局要马上动手，而如果临时找个藉口把孟的妻儿先叫回北京，或者叫孟全家一起回来，孟本人会马上警觉，说不准立马就在法国〝阖家叛逃〞了。
 <br/>
 <br/>
 如果孟在外头把中共的底子都抖露出去，就成了王立军第二，中共显然无法承受这种更危险的结果。所以中纪委选择单独把孟宏伟召回，实际是无计可施的情况下使出的险招。
 <br/>
 <br/>
 既然可能是临时生变，而且还传闻同时被查的还有一名公安部的副部级高官。那这就与当前习近平对公安部的深度清洗有关。
 <br/>
 <br/>
 孟宏伟的背景，其实没有官方简历写的那么简单。孟在任职公安部部长助理之前，其履历一片空白。而据曾在公安部任职、现居海外的高光俊律师披露，孟出身公安部刑侦系统，曾官至刑侦局局长。
 <br/>
 <br/>
 这样一来，孟的背景就与同样出身刑侦系统的两任前公安部常务副部长杨焕宁（调到安监总局后落马）、傅政华（调任司法部长，曾任公安部刑侦局副局长），在重庆市副市长兼公安局局长任上落马的何挺（杨焕宁在西南政法学院的同学，曾任公安部刑侦局局长），乃至今年7月才从陕西政法委书记调任公安部副部长的杜航伟（曾任公安部刑侦局局长）都有了关联。
 <br/>
 <br/>
 这至少说明，孟宏伟可能是刑侦老虎中的一颗〝遗珠〞，同时也增加了杜航伟会被中纪委大网兜住的可能。
 <br/>
 <br/>
 习近平在对权力至关重要的〝刀把子〞上搞清洗，换上自家兵马，这早已是公开的秘密。周永康时期的公安部高层，目前被撂倒的部级高官至少有李东生、杨焕宁、孟宏伟，被调离的有黄明、傅政华。五人都是正部级，李、黄、傅3人还都曾任职臭名昭着的610办主任。
 <br/>
 <br/>
 被调到国安部任纪委书记的原公安部副部长刘彦平在与郭文贵的秘密谈话被曝光后，也从此销声匿迹。如把在河北省政法委书记任上落马的前公安部反邪局局长张越算上，虎群规模就更显〝壮观〞。
 <br/>
 <br/>
 还有，2013年（周永康退休失势后）才从中组部调到公安部任政治部主任的夏崇源，这个至少表面上不应隶属周家党的人也落马了，这就不能不令人生疑，周之后，从公安部高层掉出的这些〝渣子〞，其被中南海针对的主子是否还有其他人。郭声琨？孟建柱？还是身影更为巨大、曾任中组部长的曾庆红？
 <br/>
 <br/>
 当前，实际把控公安部大权的王小洪（常务副部长，习福建嫡系）至今仍兼任北京市公安局局长（已长达2年5个月）。这一从无先例的〝双钳戒备式〞兼职的延续，折射出孟宏伟的落马可能并非习近平对〝刀把子〞大清洗的句号，而仅是个逗号。
 <br/>
 <br/>
