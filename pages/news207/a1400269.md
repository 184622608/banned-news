### 袁斌：〝习川会〞将给中美贸易战画上句号吗？
------------------------

<div class="wysiwyg">
 今年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 国集团峰会将于11月30日在阿根廷举行，届时川普与习近平将藉此机会举行一对一的双边谈判，外界认为，这是中美两国政府避免
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 继续升级为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 的一个重要契机。那么，即将到来的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习川会.html" target="_blank">
  习川会
 </a>
 〞能否使中美达成协议，给两国间愈演愈烈的贸易战画上句号吗？
 <br/>
 <br/>
 我的看法是前景并不乐观。
 <br/>
 <br/>
 就目前的情势来看，中方已于近期用书面形式向美方递交了一份详细的清单，以回应美方对贸易改革的诉求，希望以此来终止美国的关税制裁。这表明经过前一段时间的较量，中共在美国的强大攻势面前已经扛不住，愿意做出让步了。据悉，这份清单包括142个专案，分为三类，分别是中方愿意进一步谈判采取行动的问题，已经着手解决的问题，以及被视为不会谈判的禁区问题。
 <br/>
 <br/>
 对此，川普总统上周五说，〝中国想要达成协议。他们送了一份清单，列出他们愿意做的事情，那是一份很大的清单〞。他表示，虽然他现在还不会接受清单，因为其中〝漏掉了四、五件大事〞，但相信在与中国达成的任何协议中，被遗漏的项目都将得到解决，〝我想我们可以成交。我们很快就会知道的〞。
 <br/>
 <br/>
 那么中方的清单中究竟〝漏掉〞了哪〝四、五件大事〞？川普没有明说。据路透社引述一名消息人士透露，中共的让步方案上的条款，很多都是重新列出北京以前做出的调整，例如在特定行业提高外资比例，而这些档并未涵盖改变工业政策例如〝中国制造2025〞计划上的承诺。CNN也报导称，中共开出的条件并没有包括白宫的核心要求，例如解决强迫技术转移和偷窃智慧财产权问题。
 <br/>
 <br/>
 如果说川普的上述表态唱的是红脸，那么美国副总统彭斯在刚刚结束的APEC峰会上唱的则是黑脸。在这次大会上，彭斯不仅措辞严厉地抨击中共通过对外大举贷款令其它国家陷入债务陷阱的做法，暗指北京推行的〝一带一路〞是一条有去无回的〝单行道〞，警告与会各国不要被中国〝不透明支票〞外交所诱惑，而且在提到加征关税时直言，〝我们已经针对和中国的（贸易）不平衡采取了决定性的行动。我们对2,500亿中国商品加征关税，还可以把这一数字加倍，但我们期望看到更好的结果。除非中国（共）首先改变其行为，否则美国的方针不会改变。〞〝我们对习近平主席和中国非常尊重，但如总统所说，中国占美国便宜已经太多年，这样的日子已经结束了。〞
 <br/>
 <br/>
 显而易见，尽管川普与彭斯表态的口气有所不同，一个温和，一个严厉，但他们的立场并无差异，那就是中共必须改弦易辙，不仅要大规模开放市场降低关税，而且还要停止强迫技术转移和偷窃智慧财产权，放弃〝中国制造2025〞。
 <br/>
 <br/>
 而中共呢，虽然做出了愿意让步的姿态，但目前在原则问题上却并有显出任何改弦易辙的迹象。中方的清单中之所以专门列出了不会谈判的禁区问题，用川普的话说也就是〝漏掉了四、五件大事〞，这不就是个证明吗？而且习近平在APEC峰会上的话锋也同样表明了这一点。他在发言中称，美国政府提出的〝美国优先〞政策是〝保护主义〞和〝单边主义〞。他还说，国与国之间如果走上对抗道路，无论是冷战、热战，还是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，都不会有真正的赢家等。
 <br/>
 <br/>
 综上所述，别看中共做出了让步的姿态，川普的口气也不像之前那么充满火药味了，但围绕着停止不停止盗用智慧财产权、强逼技术转让和不公平补贴，放弃不放弃中国制造2025，降低不降低巨额贸易逆差等原则问题，中美之间的对立并没有消解。也可以这么说，中共已经让步的并不是美国最想要的，美国最想要中共让步的中共并没有让步。
 <br/>
 <br/>
 现在的问题是，这种对立在接下来的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习川会.html" target="_blank">
  习川会
 </a>
 〞时会消解吗？我看不大可能。为什么这么说？原因很简单。一方面，如果中共完全按照美方的要求〝全面改变其行为作法〞（彭斯语），不但将在全世界面前脸面尽失，而且很可能给党内政敌授之以柄，甚至令它的统治全面动摇，所以这是绝对不可能的；反之，如果让美国放弃自己的核心要求，对中共大幅度妥协，美国政府在全世界面前还有何信誉可言？川普又将如何面对支持自己的选民？所以，这也不大可能。
 <br/>
 <br/>
 至于最终结果究竟如何，有待继续观察。
 <br/>
 <br/>
