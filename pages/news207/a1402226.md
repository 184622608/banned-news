### 觅真：中共是导致人权恶化的罪魁祸首
------------------------

<div class="wysiwyg">
 1948年12月10日，联合国通过了《世界人权宣言》，并从1950年起将每年的12月10日定为世界人权日，今年是第68个世界人权日。在国际人权日到来之际，中国的人权纪录仍然在持续恶化。呼吁国际社会关注法轮功人权！
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 迫害法轮功十九年来，以人权恶棍江泽民为首的中共犯罪集团从迫害法轮功初期就提出了〝名誉上搞臭、经济上截断、肉体上消灭〞、〝打死白打死，打死算自杀〞和〝不查身源，直接火化〞的灭绝政策，近亿法轮功学员被污衊、长期监控，数百万学员被绑架、关押、劳教、判刑及关洗脑班，甚至被酷刑致死或活摘器官致死，数百万家庭被迫害得支离破碎、妻离子散、家破人亡！
 <br/>
 <br/>
 中共在面临内忧外患、山穷水尽、穷途末路的时刻，对善良民众的迫害不但没有停止，仍然在肆无忌惮地进行着。据明慧网显示，二零一八年一至十一月份法轮功学员被非法判刑837人；非法庭审805场。一至十月份被绑架人数为3944名；被骚扰人数为3783人。
 <br/>
 <br/>
 十一月九日，黑龙江省哈尔滨市各区县、大庆市有119名法轮功学员被绑架，另有，十几位法轮功学员被骚扰，或绑架未遂。这次的绑架是黑龙江省公检法司部门蓄谋已久的阴谋，警察是拿著名单到法轮功学员家绑架的。
 <br/>
 <br/>
 据最近在海外曝光的一份辽宁省〝610办公室〞文件内容显示，从十月开始，中共系统性地加大力度迫害法轮功。
 <br/>
 <br/>
 在此仅举几例：据明慧网近日报导，天津法轮功学员周向阳（全国仅有的六十位造价工程师之一），二零零三年五月三十一日曾被非法判刑九年，在二零一五年三月二日又被绑架后，于二零一六年十二月又被冤判七年（周向阳的妻子李珊珊同时被冤判六年。）。周向阳在天津滨海监狱绝食抗议已近三年，现在的周向阳非常苍老、虚弱，每天由包夹人推着轮椅去监狱的医院给他灌食。
 <br/>
 <br/>
 周向阳的母亲王绍平老人在一封呼吁信中说：〝十七年里，我们一家很少平安在家。向阳被劳教两年加刑一年，九年监狱，只过了两个团圆年。二零零一年，我大儿子夫妇被迫害。大儿九年、儿媳三年。我们夫妇被迫害流离失所，四年后被迫害劳教，我一年半，向阳的父亲两年。〞
 <br/>
 <br/>
 如今七十多岁的周振才与老伴王绍平老人又被非法判刑。一家人被迫害得支离破碎、苦不堪言。
 <br/>
 <br/>
 在中国大陆，十九年来被中共摧残迫害致支离破碎、妻离子散、家破人亡的惨案数不胜数。如辽宁省凌源市建平县的法轮功学员刘殿元，今年八十岁，从一九九九年九月到现在，经历了七年冤狱、四年半的流离失所，二零一五年十一月再次被绑架，在七十九岁的高龄时被辽宁省建平县法院非法判刑十一年半，目前在冤狱中遭受着迫害，还有十年多的监狱生活等待着这个历经沧桑的老人。老人的妻子刘玉芳，今年六十一岁，是一位普通的农村家庭妇女，曾经被非法劳教三年，又被非法判刑四年。
 <br/>
 <br/>
 还有一家八口被迫害致五死一残的辽宁省清原县法轮功学员孙鸿昌一家；六口之家被迫害致仅剩一人的河北省怀来县法轮功学员陈运川一家；一家五口被迫害得仅剩下一人的山东省青州市神旺村法轮功学员王兴家一家；三位至亲被迫害致死的河北石家庄市高级工程师冯晓梅一家……
 <br/>
 <br/>
 最近明慧网在《这些老人们为什么不能安度晚年？》一篇评论中列举了马桂兰、许玉盘、张鲁元、羊衍海、张新伟、陈再山、李光清、杨七星等中国大陆各地数位老年法轮功学员被中共迫害的悲惨遭遇，进一步揭示了中共邪灵的流氓本性和邪恶本质及祸乱人间、毁灭人类的终极目的。
 <br/>
 <br/>
 中共长达近二十年对法轮功的残酷迫害，法轮功人权日益恶化的状况，引起了国内国际社会的广泛关注。据明慧网显示近二十一万法轮功学员及家人控告江泽民，二百八十八万国际社会正义人士连署举报江泽民，三亿二千多万民众抛弃中共退出中共邪恶组织，国际社会正在加速围剿共产邪恶主义的进程，中共政权面临着前所未有的压力，正在风雨飘摇中垂死挣扎，走向彻底崩溃。
 <br/>
 <br/>
 从二零零二年开始，美国国会先后通过〝188号〞、〝304号〞、〝605号〞和〝343号〞决议案，要求中共停止对法轮功学员的迫害，停止针对法轮功学员和其他良心犯的〝强摘器官〞行径。二零一六年九月十二日，在欧洲议会全体会议上，欧洲议会主席舒尔茨宣布了48号书面声明，要求欧盟采取行动制止中共强摘器官。美洲、欧洲、亚洲、澳洲等许多东西方国家都强烈谴责中共迫害法轮功的恶行，要求中共停止迫害法轮功，制止中共活摘器官的反人类罪行。二零一八年五月二十九日，美国国务院发布二零一七年度《国际宗教自由报告》，中国再次被列为〝特别关注国〞。（从中共开始迫害法轮功到二零一八年，中国一直被美国国务院列为宗教自由〝特别关注国〞。）
 <br/>
 <br/>
 人权是人类的普世价值，拥有人权，维护人权，保障人权是人类共同的责任。天赋人权神圣不可侵犯。《世界人权宣言》曾向全世界郑重宣示：世界上所有国家的男女不分种族区域，都享有各种基本权利和自由，其中包括生命、自由、人身安全、以及言论、结社、集会等自由。此后，世界上除少数国家外，都在努力践约着《世界人权宣言》。可是作为世界人权缔约国，中共却在肆无忌惮地践踏和破坏着《世界人权宣言》，成为世界上人权状况最恶化的国家之一。
