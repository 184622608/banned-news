### 文武：中共治国无能还硬撑着不下台
------------------------

<div class="wysiwyg">
 中共今天又高调推出一个什么区域联动机制，表面上看着像是声势很大，好像是个大动作，其实只是空喊的口号，根本没有任何实际作用。
 <br/>
 <br/>
 中国现在地区发展不平衡的问题，根本不是什么区域联动就能解决的问题，而是中共体制的问题，中国现在根本就不是市场经济，资源配置从根本上来说就与市场无关，而是与政府的主导有关。
 <br/>
 <br/>
 而中共政府又是集权的政府，所有的官员眼睛只往上看，只听上面的安排，这样一层层的推上去，就形成唯一的推动力就是中共的中央。
 <br/>
 <br/>
 而经济本身又是一个复杂体，特别是现代经济，更是纷繁复杂，经济本身又有其自身运行的规律，不以人的主观意愿为转移，人只能去顺应经济的规律而不能去操纵经济的规律，可是中共却非要去做这么一件蠢事，非要去人为的操控经济。
 <br/>
 <br/>
 人的认知是有限的，有限的认知不可能去人为的操控无限的经济。
 <br/>
 <br/>
 正是因为如此，人类现在所走的路基本都是市场经济的道路。其实，中共早在四十年前就已经认识到了，要走市场经济的道路，可是走到现在，却还不是真正意义上的市场经济，仍然还是中共管控下的半调子市场经济。
 <br/>
 <br/>
 问题是，谁都知道应该走市场经济的道路，而中共为什么却偏不去走呢？
 <br/>
 <br/>
 原因就是中共不愿意放弃手中的权力。
 <br/>
 <br/>
 其实从本质上来说，市场经济与权力并不是矛盾的，并不是说有市场就不能有权力，有权力就不能市场，反而这两者是相互协调，是相互促进的关系。
 <br/>
 <br/>
 可是为什么到中共这里市场与政权就如此的互相矛盾了呢？
 <br/>
 <br/>
 原因就是中共要的政权并不是通常意义上的政权，中共要的管天、管地、管人、还要管人的思想的一切权力。
 <br/>
 <br/>
 就像人天生就应该吃饭，可是吃尽人间的一切，这个人就会胀死了一样，一个政党执政了就应该有相应的权力，可是将权力无限扩大，要管控一切，那就胀死了。
 <br/>
 <br/>
 就像今天中共出台的一个什么区域联运机制，各个地区本来就有各个地区特点与经济发展的规律，不可能通行一个什么行政命令的方式来硬性要求各个不同的地区都来按着中共的指挥捧来一起联动是一样的。
 <br/>
 <br/>
 今天中共还开了个什么
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_互联网.html" target="_blank">
  互联网
 </a>
 治理的部际联系会议，全都是一个得性，声势大效果无。
 <br/>
 <br/>
 互联网本来就是有利于人们信息交流的平台，可是中共却弄个防火墙，不让中国人看到真实的信息，这样，假的信息，乱的信息，诈骗的信息，淫乱的信息就充斥网络，这个时候中共又来治理，那如何能治理得了。
 <br/>
 <br/>
 其实，中国社会所有乱象的根源，就是中共自己一手造成的，中共无论怎么去治也是治理不好的。
 <br/>
 <br/>
 反过来， 中共下台了，没有中共了，这些乱象的根源就铲除了，中国社会的乱象也就能够治理好了。
 <br/>
 <br/>
 ──转自《大纪元》本文只代表作者的观点和陈述。
 <br/>
 <br/>
