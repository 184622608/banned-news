### 夏小强：川普可用朝核模式应对中共贸易战
------------------------

<div class="wysiwyg">
 11月29日，美国总统川普在白宫登机赴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会前告诉记者，〝非常接近与中国就贸易问题取得进展〞。川普此言，让外界对此次备受关注的川习会能够达成协议，增加了几分期待。
 <br/>
 <br/>
 12月1日的川习会，可能是习近平当局最后一次与川普达成协议，结束
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 的机会。半年以来，中共对美方在贸易战中的应对，可以用一个字概括：拖。中共调动了全部资源，试图干涉影响美国的中期选举。但是，随着美国中期选举的尘埃落定，美国共和和民主两党在对中共贸易策略方面，获得共识，对中共采取强硬姿态。
 <br/>
 <br/>
 随着2019年1月1日的临近，2000亿美元商品的关税从10%加到25%，已经成为定局，而剩下的2670亿美元商品增加关税也已箭在弦上。中国企业的寒冬正在到来，失业潮涌动，中共政权的危机加剧。在这样的背景下，中共〝拖〞不下去了。
 <br/>
 <br/>
 中共与国际社会打交道的历史，就是一部流氓史和欺骗史。熟悉中共历史的人都知道，中共是谈判的高手，通过谈判化解生存危机，在历史上已经上演多次。中共通过与国民政府谈判，最终从国民党中夺取政权；中共通过与世贸谈判，骗取美国和国际社会信任，签下协议，然后拒不执行，欺骗玩弄美国和国际社会二十年。
 <br/>
 <br/>
 一旦进入谈判和达成协议阶段，基本上就进入了中共的〝谈判欺骗游戏程序〞。虽然，美方和川普已经多次宣称，中共必须提出具体行动方案，取消其不公平贸易措施、停止窃取美国知识产权、不再强制技术转让、取消扭曲市场的产业补贴政策及非关税壁垒措施，以及进一步开放市场等。但是，一旦谈判进入中共的〝游戏模式〞，川普就面临着同样的难题：面对中共签下的让美方满意的又一个漂亮的协议，又一个美丽的谎言，又一个永远也不会兑现的承诺。川普该怎么办？
 <br/>
 <br/>
 11月26日，川普在接受《华尔街日报》采访时表示，〝在美中之间还没有达成协议之前，如果美国政府就停止提高关税比例等政策，中共将会继续敲诈美国，就像他们过去那样继续任意妄为。〞
 <br/>
 <br/>
 看来，川普对中共的本性和手段非常了解。11月29日，白宫官员向CNBC证实，白宫贸易政策顾问、对华鹰派人物彼得・纳瓦罗（Peter Navarro）本周将出席川普与习近平的重要会晤。纳瓦罗参加川习晚餐会晤，可以被视为向中方发出信号，表明贸易鹰派在美国政府内部更具主导地位。
 <br/>
 <br/>
 那么，川普对中共贸易战完全可以采用如今正在实施的〝朝核战略〞：在朝鲜没有彻底销毁核设施之前，不停止制裁。那么也就是说，对中共的贸易战策略，不管中共签下任何漂亮的协议，在中共没有做出实际改变的行动之前，不仅要不要停止制裁，而且要继续加码制裁力度。
 <br/>
 <br/>
 可是，话又说回来，中共有可能做出实际改变吗？真正达到美国的要求吗？达到了美国的要求，等于让中共放弃对中国经济的操控，等于让中共退出历史舞台，这有可能吗？除非太阳从西边升起。
 <br/>
 <br/>

 <br/>
