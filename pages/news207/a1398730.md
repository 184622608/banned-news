### 陈思敏：中共地方政府的楼市调控骗局
------------------------

<div class="wysiwyg">
 在北京当局〝房住不炒〞的大背景下，楼市最高主管部门对调控政策将继续保持〝从严性和稳定性〞的话音才刚落地，就被安徽政府打脸了。
 <br/>
 <br/>
 据报导，安徽某楼盘打出5.7折的折扣，但是随后被当地有关部门叫停，而合肥市某楼盘降价6,000元╱平米，也在当地房产局调研后又从1.4万涨回到1.8万、2万元╱平米以上。原来都是地方政法出手禁止降价，还要求业者涨回原价。
 <br/>
 <br/>
 安徽楼市一天之内两地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房价.html" target="_blank">
  房价
 </a>
 变盘，包括位列全国二线城市的合肥楼盘，从降价30%到涨价20%至30%，这也说明官方版的70城（一线、新一线、二线、部份三线）房价统计真实性有多少。
 <br/>
 <br/>
 合肥楼盘一口气降6,000元╱平米，开发商会亏本吗？据房仲业者估算，房地产商在一块地上建造的房子数量，基本是售出三分之一就保证赚了。所以降价30%仍有高获利。房地产行业虽暴利，但都是高负债高价拿地，资金周转率差、流动性紧，偿债压力空前，资金多方承压，现金为王，开发商降价求现还是如意算盘。
 <br/>
 <br/>
 但是当地政府就不乐意了，因为开发商大降价，直接影响土地价格，间接导致土地流拍，影响土地财政，而那众所周知是地方政府的命根子。公开资料显示，合肥市的土地财政仅次于南京而在全国排第二。
 <br/>
 <br/>
 此前统计，土地成本基本可占到房价70%。据最新统计，至今二线城市楼面起拍价仍占到预计售价的至少50%。当前一些城市〝地冷价跌〞是某种假像，土地一级市场地方政府垄断，土地从高定价，如果流拍，供地少了房子就少房价也就拖住。如果拍出，地方政府大笔进帐，房企拿地成本高房价也跟涨。
 <br/>
 <br/>
 据报导，截止今年10月底，安徽多城房价普遍上扬，最大涨幅城市为马鞍山，涨幅为9.31%。先前媒体报导马鞍山楼市要凉，其实当地不仅房价没降，还有数家楼盘要加推，公开资料显示，最低最高售价从7.3千至1.45万╱平米不等。
 <br/>
 <br/>
 马鞍山在全国城市分级中，位列弱三线城市，这样的房价都明显偏高。还有更畸形的，在安徽安庆市的潜山县是一个贫困县，房价却超过了8,000元╱平米，还有省内其他贫困县房价暴涨背后，都在说明房价是地方财政重大来源，当地政府不会轻易松手。
 <br/>
 <br/>
 而安徽省只是一个缩影，同时告诉外界，别被一二三线楼市降温假像给骗了，还有四五六线城是，媒体报导，全国超过200个县城新建商品房销售均价超9,000元╱平方米。
 <br/>
 <br/>
 如湖南省县隆回，就在这样一个脱贫任务还十分艰巨的国家级贫困县，目前商品住宅均价已超4,500元╱平米，涨幅达三成，位居湖南省县级城市前列。更恶劣的是，政府还把户口、学籍之类的捆绑在房产上，变相强迫民众买房子。
 <br/>
 <br/>
 不仅贫困县，还有经济发达城市，如厦门市8月份房价下跌，随即在10月份被指8年来首次放宽落户资格，购房户〝优先〞入学、就业等等。
 <br/>
 <br/>
 还有地方政府和国有房企唱双簧，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万科.html" target="_blank">
  万科
 </a>
 高喊〝活下去〞，然后32亿大量收购华夏幸福的多个环京项目，再数十亿连续接盘海航位于河北、广州、武汉等地资产，万科与深圳政府的〝万村计画〞被指将推升当地房租暴涨，摧毁年轻人的城市梦，毁掉制造业和实体经济。
 <br/>
 <br/>
 在所谓严厉调控当前，拿地不落后的还有科技业。如顶着〝中国AI第一股〞的科大讯飞，被曝以AI为名圈地炒房套利，拿地盖别墅，地点还是安徽省宣城市扬子鳄国家级自然保护区。虽然科大讯飞号称语音辨识大厂，但被踢爆AI 同步口译是人类冒充，股价靠炒作AI题材，盈利依赖政府补贴。
 <br/>
 <br/>
 最新一起是小米科技董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雷军.html" target="_blank">
  雷军
 </a>
 26亿元拿地，11月5日，小米高调宣布携手央企华润置地耗资26.57亿元拿下北京昌平区的一个公建混合住宅专案地块。眼下网友们最关心的是，小米拿地不是自用而是进军房地产。
 <br/>
 <br/>
 小米董事长雷军除了被指曾5次放言绝不碰房地产，而且11月1日他就坐在习近平主持召开民企座谈会上听讲。不过4天时间，小米雷军不是宣称搞好实业而是宣布北京拿地，被指手机都没做好，又做房地产，除了搞科技不如搞房地产的钱来得快地原因外，也可能是被地方政府绑架了。
 <br/>
 <br/>
