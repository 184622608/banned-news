### 陈思敏：李克强稳定就业谈话的背后
------------------------

<div class="wysiwyg">
 10月24日，在中国工会全国代表大会上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 宣布布，要把稳定就业作为宏观政策的优先目标。
 <br/>
 <br/>
 稳定就业既是高层布署〝六稳〞之首，也是官方催促消费的重要前提。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 这次讲话，侧面可以反映出或印证了目前就业市场状况。
 <br/>
 <br/>
 备受关注的中小型企业，是传统人力需求大户，但向来缺乏官方统计。据中国人民大学和就业平台联合编撰的 CIER（中国就业市场景气）指数，以1为荣枯分界，高于1代表劳动景气好转、小于1代表恶化。报告称，500人以下的企业，CIER指数从第一季的0.85，第二季降至0.79；1万人以下的企业，CIER也从第一季的1.01，第二季降至0.97。以上数字表明，小公司在加速裁员，中型公司亦开始精简人力。
 <br/>
 <br/>
 引发热议的网络业，这几年一直是用人大户，但近来缩减人事传言满天飞，尤其是动见观瞻的互联网巨头〝BATJ〞──百度、阿里巴巴、腾讯、京东，即便有关当局出面否认停止征才，但从业人员透露，BATJ等就算未停止招人，也只雇用5年以上经验的专家，其它职缺通通外包，并且不断缩减校园征才名额。此外，中国人民大学中国就业研究所发布2018年第二季《中国就业市场景气报告》，指出科技和网路业招聘职位年减6%，而这也是首次出现负增长。
 <br/>
 <br/>
 用人大户的中小型企业、互联网企业都如此，其它行业用人需求萎缩可想而知。《南华早报》先前报导，因中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，中小型制造业、科技业、金融业都传裁员潮，缩编人数出现两位数增长。据最新报导，金融业因股市下跌，券商今年都不招人；地产更是严冬，万科集团全国只招88人。
 <br/>
 <br/>
 贸易战让就业压力加大，10月24日官方记者会上，中共发改委发言人孟玮也坦承，中国就业目前受到中美经贸摩擦等外部因素所带来的不确定性影响。
 <br/>
 <br/>
 目前，中美已经二轮互征重关税。外资机构此前分析，如果美国启动第三轮关税，对 2,670 亿美元的中国进口商品课税，将会再度引发中国中小企业裁员潮。摩根大通（小摩）报告，到时中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业率.html" target="_blank">
  失业率
 </a>
 可能暴增，估计会有300万人丢掉饭碗。
 <br/>
 <br/>
 贸易战影响就业，因此官方更加美化和粉饰失业统计，日前公布，今年7月至9月，全国城镇调查失业率分别为5.1%、5.0%和4.9%。据中共统计局历年图表，近10年来城镇失业始终维持在4.0%至4.3%区间，即便是发生金融海啸的2008与2009年，中国城镇失业率也仅微幅上升至4.3%而已。如果不是这场贸易战，想必城镇失业率继续维持4字头。
 <br/>
 <br/>
 不过仍有一个失业率纹风不动，即全国主要就业人员群体，25至59岁人口的调查失业率，继续保持在4.4%的低位。这个数字无论如何都要压低。据悉2019年中国应届毕业生将达860万人，为历年之最，也就是即便没有贸易战，就业问题都很突出，现在又碰上贸易战企业大规模减招，明年就业市场很难乐观。
 <br/>
 <br/>
 正因为就业率能反应宏观经济，所以失业率一直是中共官方的统计禁区，失业率的真实数字更是中共当局不愿面对的真相。就像一胎不够用了才开放二胎，中共稳就业是怕人民失业无法拉动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内需消费.html" target="_blank">
  内需消费
 </a>
 ，特别是因此失业大军会群起抗争危及政权。在贸易战副作用中，相较外汇储备流失、人民币走跌等，庞大的失业人口可能才是中共当局更担心的一颗特大未爆弹。
 <br/>
 <br/>

 <br/>
