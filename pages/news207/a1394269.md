### 周晓辉：北京全盘否认彭斯警告 却避谈一件事
------------------------

<div class="wysiwyg">
 当地时间10月4日，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 在华盛顿智库哈德逊研究所就中美关系发表演讲，全面阐述总统川普（特朗普）政府对华政策，内容涵盖中国（中共）的贸易、军事、人权以及干预美国政治等多个领域。这是迄今为止川普政府对华政策的最全面、最务实的阐述，也向北京传递了最为清晰的信号，那就是美国将改变以往的对华政策，将通过强化美国国防力量，全力遏制中共的扩张，并重振美国与世界的希望。
 <br/>
 <br/>
 在演讲中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 阐述了以下八个方面：
 <br/>
 <br/>
 一、美国对华政策业已转向，原因在于〝北京正在使用一种全政府的手段，利用政治、经济、军事工具以及宣传，在美国推进其影响和利益〞。彭斯表示，美国〝对未来的远景建立在过去的最佳时期，那时美中两国以公开和友善的态度互相接触〞。
 <br/>
 <br/>
 二、中国人民自由的希望仍没有实现。
 <br/>
 <br/>
 在回顾过往的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 后，彭斯指出正是中共在1949年掌权后，才使美中关系隔绝。但在美中恢复外交关系后，美中开始了经贸往来。苏联垮台后，美国带着〝中国将不可避免地成为自由国家〞的乐观，将中国纳入世界贸易组织，但美国的希望落空了。
 <br/>
 <br/>
 彭斯指出：〝曾有一度，北京慢慢地走向更大的自由以及对人权的更大尊重。然而，近年来，中国朝着控制和压迫本国人民的方向急转弯。〞具体来说，一方面，中共通过〝无以伦比的监控〞， 严重限制着中国人民的信息自由流通，并通过所谓的〝社会信用分数〞控制民众。另一方面，中国依然没有宗教自由，〝中国的基督徒、佛教徒和穆斯林正在经受新一波迫害浪潮的冲击〞。
 <br/>
 <br/>
 三、中国经济发展是以牺牲美国利益为代价，中共〝使用了与自由公平贸易不符的一系列政策，包括关税、配额、货币操纵、强制技术转移、知识产权盗窃以及工业补贴〞，导致中美贸易赤字巨大。为了回击北京，美国连续加征关税，目的就是达成公平与对等的协议。
 <br/>
 <br/>
 四、中共炫耀其军力抗衡美国，美国〝不会退缩〞。
 <br/>
 <br/>
 彭斯指出，中国的安全机构掌握了大量窃取美国科技的能力——包括最先进的军事技术。中国的军费是亚洲其他国家的总和，北京将在陆海空，乃至外太空抗衡美国军力作为首要任务。中国希望将美国挤出西太平洋，并试图阻止我们援助盟友，但是他们会失败。
 <br/>
 <br/>
 对于中共炫耀军力，彭斯除了提到北京在南海人工岛屿建造军事基地并部署先进的反舰和防空导弹外，还特意提到了近日发生在南海的中共军舰挑衅美舰的鲁莽行为，称〝我们不会被吓倒；我们不会退缩〞。
 <br/>
 <br/>
 五、中国用所谓的〝债务外交〞扩大其影响力。彭斯尤其提到了北京刚刚向委内瑞拉腐败无能的马杜罗政权提供的50亿美元贷款。
 <br/>
 <br/>
 六、中共说服三个拉丁美洲国家与台湾断交的行动正威胁到台湾海峡稳定，美国予以谴责。彭斯暗示美国将支持台湾，因为〝台湾对民主的拥抱为所有华人展示了一条更好的道路〞。
 <br/>
 <br/>
 七、北京的统治者到目前仍不愿改变，并〝正在推动一场全面而有协调的运动，以破坏总统、我们的议程和我们国家最珍贵的理想所受到的支持〞。
 <br/>
 <br/>
 彭斯列举了来自情报部门的一些消息来加以印证，如中共正在奖赏或胁迫美国的工商企业、电影制片厂、大学、智库、学者、记者、地方、州和联邦政府官员，通过媒体等干预美国中期选举。具体而言，中共为在全世界各地的宣传机构花费数以十亿计美元，利用并干涉在美中国学生活动，向美国大学、智库、学者、商界人士等施压。
 <br/>
 <br/>
 八、美国向北京发出强烈信息，即〝本届总统不会退缩，美国人民不会动摇。虽然我们希望改善与北京的关系，但我们将继续坚定地捍卫我们的安全和我们的经济〞。
 <br/>
 <br/>
 彭斯简略阐述了美国政府未来将采取怎样果断的行动，保护美国的利益、美国的就业和美国的安全，并表示：当涉及北京对美国政治和政策的恶意影响和干涉时，我们将继续揭露它，无论北京采取何种形式。
 <br/>
 <br/>
 彭斯还向北京最高层发出了呼吁：川普总统已经明确表示，我们希望与北京建立建设性关系，共同促进我们的繁荣与安全，而不是分离。尽管中国（中共）政府一直在进一步偏离这一愿景，但中国领导人仍可以改变路线，回归几十年前两国关系开始时的改革开放精神。美国人民别无所求；中国人民理应得到更多。
 <br/>
 <br/>
 彭斯演讲最后希望〝北京很快会以行动而不是言词作为回应，重新尊重美国〞。
 <br/>
 <br/>
 显然，北京并未如彭斯和美国所希望的那样〝重新尊重美国〞，中共官媒人民网10月5日迅即推出署名钟声的评论文章，称彭斯〝诬衊中国的种种奇谈怪论〞是〝并不高明的造谣术〞。
 <br/>
 <br/>
 一如既往，文章延续中共的说辞，指美方的说法〝不仅歪曲事实，还混淆是非，存在事实性、逻辑性和时代性错误〞，并否认中共内外政策发生改变，称中共已找到了〝符合自身国情的道路〞，将继续〝坚定不移〞地走下去；否认中共干预美国选举，认为中国媒体购买美国媒体版面影响舆论并无不妥，也谈不上打击特定选民；否认中共对美发动网络攻击；否认窃取知识产权；否认威胁台海稳定；否认在南海搞〝军事化〞；否认给一些发展中国家带来〝债务陷阱〞。
 <br/>
 <br/>
 有意思的是，否认了诸多事情的文章，却避谈一件事，即居然没有否认彭斯所言的〝中国人民自由的希望仍没有实现〞，没有否认中共对民众的监控，对基督徒、佛教徒和穆斯林的迫害。其心理大概不是不想否认，而是害怕民众知晓美国副总统关心中国人的各项自由。中共的底气、胆怯由此可见。
 <br/>
 <br/>
 而在抛出一系列〝否认〞后，文章称美方给中共〝罗织罪名〞是为了转移国内焦点，抬升选情和对华施压。在一番批评美国后，文章最后称〝中美作为两个大国，双方谁也搞不垮谁，谁也改变不了谁〞，〝期待着美方同中方一道相向而行，而不是相反。〞
 <br/>
 <br/>
 中共官媒的迅即反驳的胡说八道当然不值得一驳，但其却不仅表明北京当局十分恼怒，因为彭斯演说切中了北京的要害，而且亦在告诉美国：北京并不打算改弦更张，如果美国不接受北京的前提条件，北京将继续抗衡下去。只是还没有意识到美国坚定意志，没有真正认清形势，尚对美国中期选举抱有希望的北京政权，真的有底气抗衡下去吗？估计是〝不到黄河不死心〞。
 <br/>
 <br/>
