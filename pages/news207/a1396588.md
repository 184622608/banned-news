### 袁斌：移民潮戳穿〝盛世论〞的西洋镜
------------------------

<div class="wysiwyg">
 港媒近日报导，目前大陆中产阶级出于对中共局势的担忧与害怕，忙于把资金和子女转移到国外，许多人因此受骗。但不少人却表示，受骗上当损失的钱财，会打官司。无论最终能否拿回财产，都会继续寻求海外投资移民的契机。
 <br/>
 <br/>
 今年是中共改革开放40周年，大陆的官方媒体上充斥着各种溢美之词，其中最煽情肉麻的莫过于说什么改革开放取得了举世无双的成就，缔造了中华民族的又一个太平盛世。
 <br/>
 <br/>
 不过，一个社会和时代到底是不是盛世，关键不在于政府的宣传机器怎么定调，而在于这个社会和时代的实情究竟如何。就此而论，别的且不提，四十年来一波波汹涌的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民潮.html" target="_blank">
  移民潮
 </a>
 便彻底戳穿了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_盛世论.html" target="_blank">
  盛世论
 </a>
 〞的西洋镜。
 <br/>
 <br/>
 据媒体报导，2010年总共有50.8万中国人移民到了34个获〝经济合作与发展组织〞评定为已发展的国家，比2000年增长了45%。2011年美国接收了8.7万个来自中国的移民，明显比上一年的7万人为多。
 <br/>
 <br/>
 与以往以知识份子为移民主体不同的是，近年来出现的这波移民潮可以说是富人移民潮。2011年4月，招商银行发布的《2011中国私人财富报告》称：〝近60%接受调研的千万富翁已经完成投资移民或有相关考虑。而亿万富翁（可投资资产规模在1亿元人民币以上）中，约27%已经完成了投资移民。〞最新公布的《胡润财富报告》显示，超过16%的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国富豪.html" target="_blank">
  中国富豪
 </a>
 已经移民或向其它国家递交了入境档。此外，44%打算近期迅速移民，超过85%计划送孩子出国读书，1/3在海外拥有资产。报告称，虽然这100万名富翁都从中国经济中获利丰厚，但是只有28%对中国未来两年的发展有信心，比去年报告中的54%大幅下滑。
 <br/>
 <br/>
 据《南方人物周刊》记者调查，富人移民的原因首先是为了孩子的教育。〝中国孩子小时候太苦了，父母也会连带。中国的大学教育非常糟糕。这些有钱人都希望自己的孩子能够过得更好，出人头地，所以他们要出去。〞除此之外，还有食品、水和空气的安全，方便与自由，资金安全等。可见，中国的富人并不看好自己的祖国，压根就没把今天当作什么太平盛世。
 <br/>
 <br/>
 其实，因为同样原因而移民的又何止是富人呢，如今高官中没把老婆孩子弄到国外去的还剩几人？
 <br/>
 <br/>
 据香港《动向》杂志曝光，中共曾展开一项内部调查，结果发现第十七届中央委员会中，竟有187名委员有直系亲属在西方国家居住、生活、工作或已加入所在国国籍，占91%；142名候补委员有直系亲属在西方国家居住、生活、工作或已加入所在国国籍，占85%；113名中纪委委员有直系亲属在西方国家居住、生活、工作或已加入所在国国籍，占88%。
 <br/>
 <br/>
 来自美国政府的统计资料显示，中国部级以上官员（包含已退位）的儿子辈74.5%拥有美国绿卡或公民身分，孙子辈有美国公民身分的更是高达91%或以上。
 <br/>
 <br/>
 为何这么多高官都把自己的老婆孩子送往发达国家？当年因贪腐倒台的江西省原副省长胡长清道出了其中的奥秘。据大陆媒体报导，他曾对移居国外的儿子表示：〝总有一天中国会不行的〞，〝有两个国籍，将来就有余地了。〞
 <br/>
 <br/>
 试想，如果改革开放真的创造了中华民族历史上又一个辉煌盛世，有权有势的上层社会不用说肯定是这个盛世里活得最滋润的群体了，他们怎么可能会争先恐后地抛弃由他们当家的这个乐园，远赴陌生的异国他乡呢？那么多的权贵、富豪和成功人士不约而同地这么做，再明显不过地表明改革开放创造的根本就不是什么国泰民安的盛世，而是大厦将倾人心惶惶的乱世。
 <br/>
 <br/>

 <br/>
