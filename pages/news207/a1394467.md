### 周晓辉：孟宏伟传被查 另一名出事高官是谁？
------------------------

<div class="wysiwyg">
 10月6日，刷爆网络和朋友圈的除了美国副总统彭斯的演讲稿中文版外，还有国际刑警组织主席、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 失踪的消息。根据几家港媒披露，孟宏伟是在9月29日从法国里昂返回北京后，即被中纪委人员带走接收调查。据悉是他涉及经济问题及其它贪腐问题，其中包括其违规在香港购置房产。目前并不知晓他人被关在何处。
 <br/>
 <br/>
 另有海外媒体透露，10月3日晚上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 和另一名在公安部的副部级高官被中纪委和国家监委宣布对他们进行调查。不过，当局迄今还没有正式公布该消息。而另一名被调查的副部级高官究竟是谁，外界尚不知晓。
 <br/>
 <br/>
 事实上，作为周永康〝余毒〞、曾经的助手的孟宏伟早就出现了不祥的征兆。去年12月，他被免去国家海洋局副局长、海警局局长职务。今年4月，又被免去公安部党委委员职务。如今孟突然失踪，又被传出被查消息，估计关于其贪腐情况已经查清，业已到了收网的时候。
 <br/>
 <br/>
 那么，与孟宏伟一同传出被查消息的公安部高官会是谁呢？目前公安部副部级高官的排序是第一副部长王小洪、孟宏伟、邓卫平、杜航伟、孟庆丰、许甘露、刘跃进、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙力军.html" target="_blank">
  孙力军
 </a>
 、刘钊和部长助理林锐、聂福如。在上述高官中，王小洪、邓卫平、孟庆丰和调入时间并不长的许甘露、林锐、聂福如，都属于习近平的人马，因此可以排除在外。
 <br/>
 <br/>
 而2015年被提升为公安部反恐专员的刘跃进，曾受到周永康排挤，并不属于江派人马，因此也基本可以排除。此外，今年8月由公安部交管局局长升为政治部主任的刘钊，应该在提拔前也被反覆考察过，被查的可能性也不大。
 <br/>
 <br/>
 是以，可能的候选人只有杜航伟和孙力军了。曾任陕西省副省长、省委政法委副书记、省公安厅党委书记兼厅长的杜航伟是在今年7月调入公安部的。但是早在2017年，就有港媒披露，其作为落马的原公安部常务副部长杨焕宁的嫡系。
 <br/>
 <br/>
 报道指，杜航伟与杨焕宁一样，都曾就读于西南政法学院刑侦系。1984年，杜大学毕业后，和杨焕宁一样被选拔到公安部，长期在杨焕宁和落马的原重庆公安局局长何挺领导的刑侦局工作，曾任刑侦局办公室主任，2007年，升为刑侦局局长。2010年，其在杨焕宁的安排下，到陕西任职，并一路攀升。据报，早在2013年，就有人反映杜航伟接受性贿赂的问题，但其后被陕西纪委否认。而其再度被调入公安部，实权被削，不排除是〝调虎离山〞，便于调查。
 <br/>
 <br/>
 与孟宏伟、杨焕宁、何挺一样，涉嫌腐败的杜航伟也紧跟江泽民、周永康迫害法轮功的政策，在陕西主导迫害，因此其出事并不令人意外。
 <br/>
 <br/>
 至于孙力军的名字为大众所熟悉，还有赖于身在美国的大陆富豪郭文贵的曝料。孙力军现任公安部一局局长。公安部一局，又称公安部国内安全保卫局（以下简称〝国保局〞），是公安部最重要的部门，负责国内政治安全保卫工作，包括情报收集分析，事件处理等。国保局不仅在公安部是排名第一，而且在各省（自治区、直辖市）的公安厅（局）也是如此。据估计，大陆国保警察有数十万之众。
 <br/>
 <br/>
 在江泽民1999年7月镇压法轮功后，从公安部到各省公安厅、公安局分设相应的国保局、国保支队等，在中央〝610办公室〞的指挥下，具体执行对法轮功的迫害。在镇压法轮功之余，国保局等还处理一些群体事件和维稳等方面的事情。有律师指出，国保所做的一切都没有法律依据，是违法的，他们通常情况是内部秘密运作，常采取突击、秘密的、流氓的等方式。
 <br/>
 <br/>
 今年3月，中共曾发布《深化党和国家机构改革方案》，内中提及不再设立中央社会治安综合治理委员会及其办公室、中央维护稳定工作领导小组及其办公室，而中央〝610〞则划归公安部。按照常理，应划归到一局。
 <br/>
 <br/>
 孙力军最值得关注的背景是他曾做过前政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟建柱.html" target="_blank">
  孟建柱
 </a>
 的秘书，郭文贵曝料称，其曾帮助孟建柱为其母亲寻找肾源，还为其杀人灭口。中共十九大，孟建柱出局，其名字在〝中委〞名单上消失，而早在其在任期间，其贪腐丑闻就满天飞。郭文贵亦曝出其与情人的私生子在澳洲和美国拥有大量的财产和海外信托等，其与江泽民之子江绵恒为〝生死之交〞。
 <br/>
 <br/>
 也就是说，如果此次传出被查的如果是孙力军，那么其身后的孟建柱也难保不被波及。
 <br/>
 <br/>
 但不管被查的是杜航伟，还是孙力军，他们与孟宏伟一样，虽然表面上看属于周永康〝余毒〞，但实则都是报应使然。而赵克志上任后，整肃公安系统现在显然依旧是进行时。
 <br/>
 <br/>
