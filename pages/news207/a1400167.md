### 文武：彭斯反共的态度为什么如此坚定
------------------------

<div class="wysiwyg">
 继上个月美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 在华盛顿发表了讨共演说后，11月17日彭斯又在APEC峰会上继续发言，而且是当着中共领导人的面就说，中共必须改正自己的行为，如果中共不改正，那么美国将会在经济、军事、政治等各个领域向中共发起全面的对抗，用彭斯的话来说，就是，如果中共不改正，美国就与中共发起全面的新冷战。
 <br/>
 <br/>
 而且还给中共规定了时间和地点，也就是给中共实行双规，规定的时间是这个月底，规定的地点是阿根廷G20峰会上。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反共.html" target="_blank">
  反共
 </a>
 的态度为什么如此坚定呢？除了有川普总统和美国人民的坚定支持外，而且还与彭斯的为人有关。
 <br/>
 <br/>
 彭斯今年59岁，当过律师，当过三届美国国会议员，当过近两届州长。
 <br/>
 <br/>
 因为2016年川普邀请彭斯与他搭档一起竟选美国总统，这样彭斯就辞退了洲长的职务，与川普结伴参加美国总统大选。
 <br/>
 <br/>
 2016年彭斯辞职时，他银行所有的存款只有15,000美元，他三个孩子靠借贷上大学，用中国的老话来说，他是个清官。
 <br/>
 <br/>
 他当上副总统后，在介绍自己的时候，他说，我首先是个基督徒，其次是个保守主义者，再其次才是个共和党人。在他当议员期间，他每次投票都不以党派划分为原则，而是站在保守主义者的立场上投票，就是以美国传统文化为基点来投票。他主张减税，主张
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统价值.html" target="_blank">
  传统价值
 </a>
 观，主张传统婚姻，反对堕胎，反对同性婚姻。
 <br/>
 <br/>
 在私生活方面，他从不与女性单独私下里交往，只有在他妻子在场的情况他才会与女性有单独接触，在他的两性生活中，除了他的妻子，从来没有接触过第二个女人。
 <br/>
 <br/>
 正是因为彭斯坚定坚守传统价值观，所以彭斯在反共的态度上就尤其的坚定。
 <br/>
 <br/>
 彭斯说，美国一向与中国友好，美国希望中国人民得到幸福。他说，美国不应该被占便宜，同时，中国人民也不应该得到更少。
 <br/>
 <br/>
 每当提到中国的时候，彭斯都把中共与中国分开，也都把中共政府与中国人民分开。
 <br/>
 <br/>
 美国现在通过关税制裁的是中共政府而不是中国人民。美国希望实现公平、自由的贸易，针对的是中共破坏公平与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_限制自由.html" target="_blank">
  限制自由
 </a>
 的行为，美国强制中共改正其行为，也是为了中国人民与美国人民一样享受到同等待遇。
 <br/>
 <br/>
 彭斯说美国绝不会退缩，如果中共不改正其行为，美国就要全面制裁中共，直到中共改正。
 <br/>
 <br/>
 其实，中共永远也改变不了邪恶的行径，中共永远只会破坏公平、限制自由。但是美国也决不会后退，其结果必然是中共完败。
 <br/>
 <br/>
 彭斯希望中国人民与美国人民同样获得自由的日子也会不远了。
 <br/>
 <br/>
 谢谢！彭斯。
 <br/>
