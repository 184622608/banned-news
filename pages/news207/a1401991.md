### 周晓辉：〝钢铁老兵〞主导谈判 北京愁上加愁
------------------------

<div class="wysiwyg">
 在刚刚于阿根廷结束的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞上，双方达成了协议，美国同意暂时停止加征
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，北京则同意大量购买美国农业、能源、工业和其它产品。除此而外，最为关键也最引起关注的是，对于此前北京避而不谈的经济结构性问题，北京也终于给予了正面回应，同意就结构性改变涉及的强制技术转让、知识产权保护、非关税壁垒、网络入侵和网络盗窃、服务业和农业进行谈判，而如果在90天谈判期限内未能达成协议，美国对华10%的关税将会被提高至25%。
 <br/>
 <br/>
 无论北京出于何种目的，此番让步既证明了此前自恃〝厉害的国〞进而与美国〝奉陪到底〞的作法行不通，也折射了在美国重压下，几轮博弈下来，北京政权已着实有些扛不住。一方面，中国经济正在走向寒冬，不仅房地产和金融市场萎靡不振，地方债务爆发，而且实体经济萎缩，中小民企大量关门、裁员，外资企业或已将或正将或打算将供应链移出中国，随之而来的失业潮、消费降级以及各行各业信心指数下降，都使得中央政府无法继续与美国抗衡下去。在此种情况下，如果美国在明年1月1日将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 提升至25%，对中国经济而言将又是一记重击。
 <br/>
 <br/>
 另一方面，尽管北京一再向世界推行自己的主张，声称要为世界经济把脉，但西方国家并不买帐，反而在贸易问题上与川普政府站在同一战线，甚至追随川普对有中共政府背景的中国企业予以限制。从习近平在G20峰会上所谓的〝世界回应〞的报导中，大唱赞歌的没有一个西方学者、政客，就可看出北京是何等的不得人心。
 <br/>
 <br/>
 此外，峰会公告首次同意改革世贸组织（WTO），明显针对的就是不履行承诺的北京政权。不管北京领导人是否乐意承认，虽然在一些西方国家仍受到贵宾般的礼遇，但在涉及关键问题上，没有一个西方国家公开地愿意支持北京，毕竟基本的价值观并不相同。大国外交受挫也就在所难免，心理自然也会产生变化。
 <br/>
 <br/>
 然而，从此前相当糟糕的记录看，北京此次承诺能信守多少，没有人可以打包票，北京说了不做，说了少做的例子倒可找到不少。因此，对于90天内双方能否就强制技术转让、知识产权保护、非关税壁垒等达成协议，外界基本并不乐观，尤其在美国白宫任命有着〝钢铁老兵〞之称的贸易谈判代表罗伯特・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莱特希泽.html" target="_blank">
  莱特希泽
 </a>
 （Robert Lighthizer）主导谈判后。
 <br/>
 <br/>
 现年71岁的莱特希泽是对华贸易强硬派，也是位非常出色的贸易谈判代表。早年他曾为前参议员鲍勃・多尔工作，1983年，在里根政府开始参与贸易谈判工作，任贸易代表副代理，一共参与了二十多个国际协议的谈判，涉及钢铁、汽车、农产品等多个领域，其最为知名的案例是成功阻挠了日本对美国的钢铁和汽车出口大潮。也因此，在1985年莱特希泽离开里根政府的时候，白宫给予了他〝钢铁老兵〞的美誉。
 <br/>
 <br/>
 据报，素以强硬著称的莱特希泽，在与日本、加拿大等八国的钢铁贸易谈判中，有意采取了车轮战，在长达七个月的谈判进程中，整个团队只放了两天的假期。不久前，其主导的美、墨、加三国北美自由贸易协定谈判，采取的也是同样的策略，如在2017年8月开始的头两个月进行了四轮、总计22天的谈判。
 <br/>
 <br/>
 由此不难想见，在90天内想与北京就强制技术转让、知识产权保护等达成协议，其间的高强度、高密度交锋必不可少，而中方官员做好准备了吗？
 <br/>
 <br/>
 要知道，在白宫工作和在世达律所三十多年的工作经历，使莱特希泽不仅拥有超强的专业知识，而且拥有丰富的谈判经验，懂得如何掌控谈判节奏。这也难怪美国另一位对华鹰派人物、白宫贸易顾问纳瓦罗认为，莱特希泽是美国贸易代表办公室中最难应付的谈判代表。〝他将继续有根有据地谈判，降低关税、降低非关税壁垒，并结束（中共）所有阻止市场准入的结构性做法。〞这也意味着美国在谈判上会继续持强硬态度。
 <br/>
 <br/>
 反观中共内部，并没有类似莱特希泽这样的谈判人物，北京应该是愁上加愁。美国《纽约时报》今年5月4日的一篇报导曾就中方代表团成员进行了评说，称中方新组建的谈判团队有着深厚的经济和金融背景，但在贸易谈判问题上没有什么实务经验。而中国商务部负责与美贸易谈判的两名主要官员，去年都被授予了大使职务派驻欧洲。
 <br/>
 <br/>
 换言之，中方官员在国际贸易法方面的缺乏经验，很可能导致双方找到共同点的难度进一步加大。显然，这也是未来三个月双方谈判的难点之一。由此推之，双方在细节上出现严重纷争的情况应该并不少见。
 <br/>
 <br/>
 除去在双边谈判过程中隐藏的问题外，北京到底在强制技术转让、知识产权保护、非关税壁垒、网络入侵和网络盗窃、服务业和农业等问题上愿意做出多大让步，也是谈判能否成功的前提条件。因为美国需要的不仅仅是北京口头的表态，更需要的是体现诚意的具体的条款、监管乃至惩罚措施，而这大概是北京当局并不愿意面对的，因为这将意味着北京需要放弃〝2025制造计划〞，需要真正地开放市场，而这很可能危及其早已摇摇欲坠的政权。
 <br/>
 <br/>
 对于迄今为止都要坚持中共的领导、走〝中国发展道路〞的北京高层而言，在这抉择的紧要关头，走向哪一方将很可能决定其最终的命运。因为包括其自身也都清楚，谈判一旦无法达成一致，美国将再次举起关税重拳，而北京要想再有缓冲的机会，概率将十分低了，毕竟美国的耐心是有限的，而且，上天的耐心也是有限的。
 <br/>
 <br/>
