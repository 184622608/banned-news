### 陈思敏：沪指跌破〝股灾底〞对中共意味什么
------------------------

<div class="wysiwyg">
 10月11日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_A股.html" target="_blank">
  A股
 </a>
 史上新增一个关键交易日：沪深两市开盘双双跳空大跌，午后继续无量下跌，终场收盘，沪指不但击穿2,638点〝股灾底〞，而且失守2,600点关口，再创新低。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_A股.html" target="_blank">
  A股
 </a>
 今日大跌，导火索首先是受美股暴跌影响。
 <br/>
 <br/>
 至于美股本波抛售走势，市场多重解读：从最表面来看，是美股禁不起联准会（FED）持续升息威胁。进一步看，是第四季度前两周在历史上通常都偏空，本波重挫有80%跟技术性有关，外部利空仅为20%。再进一步看，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 冲高后（投资人获利了结操作）进入修正行情（学界定义是从过去52周的高点往下拉回至少10%），尤以攀高跌重的科技股为主。
 <br/>
 <br/>
 据美媒CNBC报导，如标普500指数涵盖的505档成分股中，已有66%或332家公司从52周的高点跌落10%或更多。而分析师指出，三分之二的成分股进入修正领域，对市场来说是健康的。
 <br/>
 <br/>
 此前报导，自中美贸易战以来，引发各界担心或成2008年金融风暴前奏；8月份新兴汇市货币齐贬，投资人又忧虑1997年亚洲金融风暴将恶梦重演。一旦美股崩盘风险被诱发，全球股市跟着一片惨绿。
 <br/>
 <br/>
 不过，美股已经展现乱世独强实例，财经媒体引述Datatrek Research共同创办人Nicholas Cola报告指出，1982至1996年间，标普500指数只有一年总报酬为负，这15年间每年平均总报酬达17.2%。2008年全球金融海啸至今，美股没有一年亏损，每年平均总报酬为15.5%。
 <br/>
 <br/>
 所以在全球金融规律里，虽怕美股打喷嚏，但美股涨或创新高表现，对于全球股市也会带来一个提振，因此美股跌或涨，其它股市也会跟跌或跟涨，唯有A股只跟跌，不跟涨，甚至每次下跌都比其它国家股市还跌得多。
 <br/>
 <br/>
 美国是世界第一大经济体，同时美国在全球股市也是领先地位。全球60家主要证券交易所，据Virsual Capitalist网站2017年发布的数据，规模最大的是纽约证交所，仅纽交所一家市值，就约占全球股票市场的27%，比倒数50 家的市值总和还要大。
 <br/>
 <br/>
 中国世界第二大经济体，A股却是全球表现很逊色的股市，长年敬陪末座。2008年美股道指从6,000多点大涨到今天2万6,000点，而A股沪指则从6,000多点下跌到今日的2,500多点。
 <br/>
 <br/>
 经济有周期，股市有涨跌，都很正常，但是中美股市一强一弱，一目了然。美股十年持续大涨，A股则是十年一路大跌。
 <br/>
 <br/>
 中共总是宣称〝经济稳中向好〞，股民则回敬〝领跌〞世界的A股证明这句话的〝假大空〞。这与其说是股民对中国经济股市失望，不如说是对中共的不满和厌恶。
 <br/>
 <br/>
 中共对股市亦如同任何一个领域的〝维稳〞，担心的情况其实不是数字本身，而是政权，死保的不是指数，而是政权。然而也像任何一个领域的民怨，股市投资人对共产党的抱怨是越来越大且毫不隐瞒。
 <br/>
 <br/>
 中共天天拉抬、拼命支撑2638点，岂料最担心的事情还是发生了，甚至一日双破底部。时候到了中共政权一夕解体，也是完全可以预料的事。
 <br/>
 <br/>

 <br/>
 <br/>
