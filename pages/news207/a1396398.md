### 袁斌：北大女生岳昕为何失踪？
------------------------

<div class="wysiwyg">
 10月19日，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佳士工人声援团.html" target="_blank">
  佳士工人声援团
 </a>
 〞推特贴出一封由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北大.html" target="_blank">
  北大
 </a>
 2015级本科生展振振书写的〝寻找北大女生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_岳昕.html" target="_blank">
  岳昕
 </a>
 〞的发起信，随后该文不断被转载，一场网上寻找岳昕的运动已悄悄展开。
 <br/>
 <br/>
 发起信写道：那个勇敢的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北大.html" target="_blank">
  北大
 </a>
 女孩——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_岳昕.html" target="_blank">
  岳昕
 </a>
 已经失踪近两个月了。两个月以来，我和同学们找过了学校，也找过了她的家，可是在哪里都找不到她，现在大家很担心她的安危。〞
 <br/>
 <br/>
 她才刚刚毕业不足三个月，就似乎要被彻底〝消失〞在这人间了！
 <br/>
 <br/>
 但是，我们不会放弃她！她曾为之呐喊的人们不会放弃她！我们决定发起一场#寻找北大女孩岳昕#的活动，从10月19日开始，集众人之力，广泛搜集线索，帮助岳昕同学早日平安回家！希望社会各界的热心人士能够一同参入到寻找岳昕的行动中来！
 <br/>
 <br/>
 看到这里，不了解内情的读者想必要问：岳昕究竟是谁？她为什么会失踪？警方难道不管这事吗？
 <br/>
 <br/>
 你且听我慢慢道来。
 <br/>
 <br/>
 岳昕出生在北京的一个中产家庭，中考时考上人大附中，高考时考上北京大学，写出过高考满分作文，她在大三时被公派出国留学，大四时拿到美国四所大学的offer，她是母亲的骄傲……
 <br/>
 <br/>
 但这些并不是最重要的，最重要的是岳昕还是一位有社会良知，关心社会现实，富有献身精神，敢于向不公正的社会现象发出挑战，与黑暗斗争到底的理想青年和年轻斗士！
 <br/>
 <br/>
 2018年4月5日，原北大中文系教授沈阳二十年前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵.html" target="_blank">
  性侵
 </a>
 女学生的事件曝出。为了追求真相，岳昕和同学们向学校提交了资讯公开申请书。为此，她被辅导员深夜约谈施压，被学校视为重点监视物件，被老师污蔑为台独分子。4月23日凌晨，岳昕的父母和辅导员把她从睡梦中摇醒，责令她保证不得再有任何行动。当天夜里，岳昕被带回家〝软禁〞。正当所有人都以为，这个大四女生将从此销声匿迹时，仅仅一个星期以后，她便发出了第二封公开信，不但揭露了校方种种卑劣的做法，而且公开表明了自己将继续争取资讯公开的决心。
 <br/>
 <br/>
 岳昕不仅关注反性侵，还关注工厂女工的权益，关注处于社会底层的劳动者群体。为了把北大后勤工人的遭遇完整的呈现出来，让更多人看到他们的生活，今年年初，她参与了〝北大后勤工人调研报告〞的撰写。做完调研报告后，她对一位同学说，一个报告能做到的太少了，她想走到工人中间去，为此已经拒掉了美国四所大学的offer。短短两个月以后，她就践行了自己的承诺，进入珠三角的工厂，成了一名普通工人。
 <br/>
 <br/>
 7月下旬，深圳佳士科技部分工人发起要求组建工会的抗争，引发了外界的广泛关注，全国一些工人和大学生闻讯后陆续前往深圳声援，岳昕也加入了他们的行列。在前往深圳之前，她设想过此行会给自己带来的种种风险，但并没有因此给自己留下任何退路。看见本该是国家主人的工人为维护合法权益，一次次被按倒在地，她勇敢地冲在最前面。公开演讲、递交材料、写声援书……终于，如她所说，她成为了自己想成为的人——一个为了理想信念奋不顾身的人，一个为了工农群众忘掉自己的人。
 <br/>
 <br/>
 8月24日，深圳警方对前来声援的工人和学生实施暴力清场。数十名工人和声援学生被带走，4名工人被刑拘，面临起诉。此后，各地声援大学生陆续获释，被遣返回家或回校。但作为活跃的声援者的岳昕则毫无消息，她的同学朋友多方寻找也没有结果。
 <br/>
 <br/>
 看到这里，我想是凡了解中国国情的读者就都明白了，岳昕之所以失踪，不是因为别的，就是因为她参加了声援深圳工人的维权活动，被警察非法抓走并囚禁了。
 <br/>
 <br/>
 不用说，在任何一个正常的民主国家，无论是工人要求组建工会以维护自己的权益，还是声援工人组建工会以维护自己的权益，都是受到法律保护的正常与合法的行为，政府不可能对此进行打压。但在中国就完全是另一回事了，不管是前者还是后者，在中共的眼里都是对其统治的挑战，轻则给你扣上〝寻釁滋事〞的帽子，重则说你是〝与敌对势力相勾结〞，〝煽动颠覆国家政权〞。在过去的2018年里，岳昕反性侵、调研工友、声援工人维权，其所作所为可以说屡屡触碰了中共的统治秩序，尤其是这回带头冲在声援深圳工人组建工会的第一线，更是踩踏了中共眼里的政治底线，这样的斗士当局怎么会放过呢？当然不会！非法抓捕并囚禁岳昕既是对她本人的打压，同时也是对所有敢于反抗现存秩序的人的恐吓。
 <br/>
 <br/>
 好端端的一个人，说消失就消失了，没有人知道他（她）的下落，甚至没有人知道他（她）是否还活着。这样的例子岳昕不是第一个，也绝不会是最后一个。岳昕的经历只不过再一次证明，在今天的中国，谁敢挑战社会不公正，谁敢跟黑暗叫板，中共就可能随时叫你消失。
 <br/>
 <br/>
