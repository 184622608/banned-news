### 愿更多的中国人身上 不再有中共的记号
------------------------

<div class="wysiwyg">
 一九四九年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 篡政后，实施独裁暴政，鼓吹无神论，宣扬假、恶、斗等党文化歪理邪说，与天地神佛为敌。通过大搞镇反、三反、五反、肃反、反右、大跃进和由此引发的三年大饥荒、四清、文革、〝六四〞​​镇压学生、屠杀西藏新疆民众、疯狂迫害法轮功等各种血腥的政治迫害运动和洗脑、屠杀等暴行，疯狂迫害民众。造成和平时期中国八千多万百姓非正常死亡，整个社会道德沦丧、黄赌毒黑社会制假贩假泛滥惊人、工人失业、农民失地、贪官横行、官商勾结、警匪一家、民不聊生，有冤无处申，有苦无处诉。以此来迫使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
  中国人
 </a>
 恐共、顺共、传统道德意识泯灭，用党文化来思维、生存，跟党走。
 <br/>
 <br/>
 特别是近二十年来对法轮功的残酷迫害造成无数善良的法轮功学员被迫害的伤、残、死、精神失常和失踪。几百万人被非法关押在看守所、劳教所、监狱、集中营、洗脑班中。遭受的活体摘取器官、注射不明药物、各种酷刑折磨等形式的迫害惨绝人寰，其中公然活体摘取法轮功学员的器官向全世界贩卖的数量巨大，举世震惊，全球抗议。
 <br/>
 <br/>
 为什么江泽民、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 要这样疯狂的迫害法轮功，就是因为法轮功是以气功形式传出的佛法天理。他从里到外洪扬的都是〝真、善、忍〞宇宙特性，所说所做的一切都是教人按照〝真、善、忍〞做好人、做更好的人，直至返本归真。其神奇的祛病功效和使人道德境界的快速提升，使全世界100多个国家和地区的政府及亿万民众受益。除了中共之外，全世界没有任何一个国家说法轮功不好，因为在人这一层面上，〝真、善、忍〞就是人的良知与普世价值，迫害〝真、善、忍〞就是在毁灭人类。而中共并非只是表面上的一个政治组织，它是由存在于另外空间的共产邪灵所控制，是以毁灭人类为目的的魔鬼。它的一切宣传，一切所为，一切恶行的最终目的就是为了摧毁所有人的道德良知，让人不信神，敢战天斗地无恶不作，而最终被神抛弃，不再受神的保护，而被它顺利带入地狱，彻底毁灭。
 <br/>
 <br/>
 法轮功的洪传使大面积的人群的道德得到了快速的回升，这令中共通过毁灭人的传统道德来达到毁灭人类的最终图谋成了泡影，所以它才非常恐慌，难以容忍，才公然践踏法律与人类的基本良知，以〝名誉上搞臭、经济上搞垮、肉体上消灭〞等灭绝政策和江泽民直接出面造谣诽谤，全国舆论铺天盖地的造假欺骗、诬陷中伤、在天安门制造假自焚升级迫害，用利益胁迫公务员为其作恶，株连儿女亲友高压恐吓，肆意绑架、抄家、拘押，活摘器官贩卖，随意非法拘禁、劳教、判刑，肆意酷刑折磨残害等各种令人发指的邪恶、卑鄙、下流的手段来疯狂的逼迫法轮功学员放弃信仰。以此来恐吓、阻止更多的人按照佛法做好人，提升自己的道德。同时逼迫更多的人跟着它伤天害理，道德败坏，以此来带动整个社会道德全面崩溃。犯下了反人类罪、酷刑罪、群体灭绝罪、……等罄竹难书的滔天罪行。
 <br/>
 <br/>
 有人说既然法轮功是佛法，那佛法无边，中共敢这么对待你们，敢这么迫害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
  中国人
 </a>
 ，神佛直接灭了它不就完了。是，神佛灭它很容易，而且一定灭它，叫天灭中共。但您想过吗，谁是中共啊，中共可不只是江泽民集团的那几个人。凡是入过中共党、团、少先队组织的人，在举拳头宣誓为它献身的时侯，共产邪灵就在其另外空间的身体上加上了一个共产党的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_记号.html" target="_blank">
  记号
 </a>
 ，就是那个党徽，圣经《启示录》上叫它〝兽记〞。这个记号只要不抹，终身带着。带这个记号的人在天上看都是中共的一份子，一旦天灭中共，那这些人就都在劫难逃，这得死多少人啊，在今天的中国，带这样的记号的人有多少啊，几乎所有的人。所以佛法慈悲众生，在另外空间销毁了共产邪灵，在人的层面上给了众生一个选择保留记号还是抹掉记号的机会，也就是给众生一个选择善恶的机会。这就是为什么有那么多的法轮功学员前赴后继，舍生忘死的向广大民众讲清受迫害的真相，讲述法轮大法的美好，揭穿中共编造的谎言，揭露中共的邪恶本质与残暴迫害法轮功犯下的滔天罪恶，就为了在天灭中共前，让更多的众生能认清中共的罪恶本质，身上不再有中共的记号而得救。
 <br/>
 <br/>
 抹掉记号的方法很简单，就是用真名或化名直接或委托他人翻墙上网到大纪元退党网站上声明退出中共的党、团、队组织。这样身上的中共记号就被上天抹掉了，在天上看你就不再是中共的人了，天灭中共就与你没有关系了。这不是叫你信什么，也不是让你反对谁，就为了在天灭中共时保住你可贵的生命，不去替中共陪葬。暂时没条件上网退的，可以先把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三退.html" target="_blank">
  三退
 </a>
 声明写在纸上贴出去或写在钱上花出去，等有机会能翻墙上网了再补办，替别人三退的必须征得本人的同意才有效。
 <br/>
 <br/>
 上天慈悲不计众生过往之过，给生命最大的得救机会，不管你曾经做过什么，只要你还不想在天灭中共时成为陪葬品，那就看看《九评共产党》、《共产主义的终极目的》和《魔鬼在统治着我们的世界》等三本书，你就能认清中共及共产主义的邪恶本质，三退自救救家人。
 <br/>
 <br/>
 真诚的希望能有更多的中国人能象那3亿多已三退的中华儿女一样，身上不再有中共的记号，在天灭中共时保住自己可贵的生命！
