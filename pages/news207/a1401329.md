### 袁斌：新疆〝再教育营〞是纳粹集中营的翻版
------------------------

<div class="wysiwyg">
 自从中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 当局将大约100万维吾尔人和其他穆斯林少数族裔关进〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育营.html" target="_blank">
  再教育营
 </a>
 〞的黑幕被曝光后，全世界良心人士无不感到震惊愤怒。但直到今年10月前，中共却一直否认这些营地的存在。此后，他们又竭力宣称这些营地是〝职业训练中心〞，甚至安排被关押在其中的〝学员〞对着央视镜头认错：〝谢谢党和政府及时发现了我、救了我〞。
 <br/>
 <br/>
 那么中共口中的〝职业训练中心〞究竟是怎样的场所？被关进里面的人其真实境遇又怎样？德国之声与美国之音近日的报导分别从不同的侧面为此给出了生动而确凿的答案！
 <br/>
 <br/>
 据德国之声11月28日报导，报导中国自由和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权.html" target="_blank">
  人权
 </a>
 状况的《寒冬》杂志26日公布了有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育营.html" target="_blank">
  再教育营
 </a>
 〞内部的影片，揭露中国官方所谓的〝职业训练中心〞其实与监狱无异。
 <br/>
 <br/>
 《寒冬》杂志的记者在8月偷偷潜入新疆伊宁市英也尔乡的〝再教育营〞。在那里，数栋四层楼橘色建筑物相连，铁窗加装铁网，出入口都有三台监视器分别对着左中右方。横向贯穿建筑墙上，斗大的字写着，〝衷心感谢以习近平同志为核心的党中央的亲切关怀〞。
 <br/>
 <br/>
 进到里面，设置就跟一般的监狱或拘留所没有差别。每一间房有两个铁窗，门上设有铁栏杆和电子锁。每一间房都有自己的厕所，但都设置了监视器。根据内部消息，一间最多住15人。每一层有28个这样的房间和3间教室。墙上到处写的像是〝感恩〞、〝明事理〞、〝让学国语成为一个习惯〞、〝在习近平新时代中国特色社会主义思想指引下为实现中华民族伟大复兴的中国梦不懈奋斗〞等标语。
 <br/>
 <br/>
 此外，某些楼层还有〝视频监看处置分控室〞，从大萤幕监看360度监视器在建筑物各处的影像。被关押者每天24小时的一举一动都逃不过〝天眼〞，完全没有自由和隐私。重要的出入口也都设置有铁门与铁栏杆。走廊上的对外铁窗也用铁网加强戒备。连被用来当做餐厅的空间也是铁幕重重。
 <br/>
 <br/>
 卧底记者也拍到在空旷的广场上设置了铁网，还有大量的铁网放置在地上等待加装。有一处上头写着〝应急避险场所〞的空间，根据线报，就是被拘留者户外活动范围。这也像是一般监狱会有的户外设施。
 <br/>
 <br/>
 如果说《寒冬》记者拍摄的影片只是反映了所谓〝职业训练中心〞的外观和内部构造，那么美国之音的报导则揭示了当局是怎样非法残忍的折磨被关押在里面的〝学员〞的。
 <br/>
 <br/>
 据美国之音报导，11月26日，在美国国家记者俱乐部一场关注新疆人权状况的新闻发布会上，29岁的维吾尔族女子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_米娜.html" target="_blank">
  米娜
 </a>
 （Mihrigul Tursaun ）回忆起过去三年自己在中国三次被捕，家破人亡的惨痛经历，难以抑制悲伤的情绪，几度掩面哭泣。现场的听众也眼含泪光，会场上不时有轻微的抽泣声。
 <br/>
 <br/>
 米娜无法忘记，2018年1月的一天，她被蒙上黑色头套，手脚绑上铁镣，带到一家医院。脱光衣服任人检查后，她被套上一件蓝色的囚服，上面写着她的号码——54。
 <br/>
 <br/>
 一名中国官员告诉她，这种囚衣是专门给那些可能判死刑的重犯穿的。〝54〞在汉语里就是〝我死了〞。
 <br/>
 <br/>
 〝我很害怕，我想，就是这样了，我会死在这个营地里，〞米娜通过一名翻译说。
 <br/>
 <br/>
 根据米娜的讲述，她所在的210号监牢是一个建在地下的牢房，没有窗，只在屋顶上有一个通风的小洞，四面都是摄像头。
 <br/>
 <br/>
 她进去的时候，里面大约关了40名妇女，小的17岁，年长的62岁。米娜认识其中大多数人。她们是她的邻居、老师的女儿、医生……很多受过良好的专业教育。
 <br/>
 <br/>
 每天清晨5点，她们在尖锐的铃声中起来，早饭前，她们要歌唱赞颂共产党的歌曲，重复〝习近平万岁〞、〝坦白从宽抗拒从严〞。背不下来的人不能吃饭，要挨打。
 <br/>
 <br/>
 每天都有新人进来，屋子里越来越拥挤，三个月后她离开的时候，37平米的小屋里挤了68个人。晚上睡觉时，十几个人站着，其他的人侧卧，每两小时轮换一次。
 <br/>
 <br/>
 米娜常常在暗夜里问自己，我做错了什么，为什么会在这里，我犯了什么罪，要受这般非人的待遇？难道只是因为我在国外生活过，会几国外语，他们就认为我是间谍吗？
 <br/>
 <br/>
 那三个月里，当局反复问讯米娜，她在国外和什么人接触，参与过什么组织。她体验了〝老虎凳〞钻心的疼痛，被迫服下不明药物，生理和心理都受到巨大创伤。
 <br/>
 <br/>
 〝我想我宁愿死掉，也不愿受这样的折磨，我求他们杀了我，〞米娜说。
 <br/>
 <br/>
 最让她觉得恐怖的，是目睹狱友被折磨致死。
 <br/>
 <br/>
 〝不幸的是，那三个月中，我目睹了68人中9人的死亡。如果我所在的小监牢，这个小县城里的210号监牢里，三个月中就有9人死亡，我无法想像全国会有多少人死去，〞米娜说。
 <br/>
 <br/>
 感谢德国之声和美国之音的报导，它们用确凿的事实无情的戳穿了所谓〝职业训练中心〞的狰狞面目。无论它的外观与内部构造，还是被关押在里面的人的恐怖遭遇，可以说都让我有一种如同置身于当年纳粹集中营一般的感觉。如果说纳粹集中营是犹太人的人间地狱，那么今日新疆的〝再教育营〞则是维吾尔人和其他其他穆斯林少数族裔的人间地狱。除了地理位置有别，关押的人种不同之外，两者在本质上完全是一回事。换句话说，新疆的〝再教育营〞完全称的上是纳粹集中营的翻版！
 <br/>
 <br/>
 广袤的新疆大地，如今到处都是狰狞的变相集中营。毋庸置疑，将百万新疆少数民族关进里面进行折磨，是中共继镇压六四爱国民主运动、迫害法轮功之后犯下的又一桩大规模的举世震惊的反人类罪，它不仅再一次撕下了中共依法治国的画皮，也必将加速这个恶党的灭亡！
 <br/>
 <br/>
