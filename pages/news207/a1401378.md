### 崔士方：610从〝负一层〞转入〝负二层〞
------------------------

<div class="wysiwyg">
 海外杂志《寒冬》（Bitter Winter）近日曝光的一份辽宁省某地防范办（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_610.html" target="_blank">
  610
 </a>
 办）文件，揭示了近期中共正在统一部署，打着〝扫黑除恶〞的旗号，系统性地加剧对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员的迫害。
 <br/>
 <br/>
 而幕后操控者，正是今年3月在〝机构改革〞中似已被〝除名〞的某地防范和处理邪教问题领导小组及其办公室（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_610.html" target="_blank">
  610
 </a>
 系统）。
 <br/>
 <br/>
 2018年3月，中共在〝机构改革〞方案中，提到〝不再设立中央维护稳定工作领导小组及其办公室，有关职责交由中央政法委员会承担〞，〝将中央防范和处理邪教问题领导小组及其办公室职责划归中央政法委、公安部〞。
 <br/>
 <br/>
 这给不少人留下一种错觉，方案中，中共对维稳办和610办的处理方式是一样的。其实不然。
 <br/>
 <br/>
 从行文看，维稳办是〝不再设立〞，并未提到〝不再设立〞中央防范和处理邪教问题领导小组及其办公室。结合这次曝光的文件，说明当局对610办和维稳办的处置并不相同，610办的职能仅是〝划归〞政法委、公安部，很大可能依然相对独立地存在于政法系统内部。而维稳办则是彻底不存在了。（之所以说〝很大可能〞，是因辽宁的省级〝改革方案〞2018年9月才落地，在9月前后，中共省市级610系统的存在形态仍有变数。）
 <br/>
 <br/>
 因为610系统迫害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 时种种手段之泯灭人性，就算按中共自家刑法，那些罪恶黑手即便一个个都被投入死牢，都无法清偿其滔天血债。所以610属于〝见光死〞，从成立至今，其所有的运作都只能在地下进行。在中共官媒上，外界充其量也就是能看到部分610官员的任命通告，以及数量非常有限的610官员到各地〝调研〞的消息。
 <br/>
 <br/>
 如果把1999年6月到2018年3月看作一个阶段，这时的610系统可认为是运作在〝负一层〞，至少明面上还有一位公安部副部长以中央610办主任的身分，出席中共中央政法委会议。
 <br/>
 <br/>
 期间的2003年，中共在公安部设立专门的〝反邪局〞（26局），出现一次短暂〝上浮〞，但这很快就随着第一任局长张越外调河北，26局拨由公安部1局代管，而重新〝下潜〞。
 <br/>
 <br/>
 2018年3月之后，610系统被〝划归〞政法委、公安部，这是一个很大的变化。公安部内部的中央610办主任不见了，610下沉更深，进入〝负二层〞。
 <br/>
 <br/>
 在宪政国家，政府运作的公开透明是必备条件，而在共产极权国家，到处是黑箱，通常情况下，运作环境越不能见光，其运作方式就越是邪恶。
 <br/>
 <br/>
 610系统进入〝负二层〞，很大一个原因，是外界不懈努力揭露邪恶、打破表层黑箱之下，中共被迫退守更深层的黑箱。但我们也必须明白，其邪恶行径也会由此变得更加隐蔽，此次偷偷摸摸部署大抓捕，就是一个明证。
 <br/>
 <br/>
 ──转自《大纪元》本文只代表作者的观点和陈述。
 <br/>
 <br/>
