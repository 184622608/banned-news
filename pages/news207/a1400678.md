### 妒忌心正摧毁中国人
------------------------

<div class="wysiwyg">
 〝乞丐不一定会妒忌百万富翁，但可能会妒忌收入更高的乞丐〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
  中国人
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_妒忌心.html" target="_blank">
  妒忌心
 </a>
 太强烈了！兄弟之间，姐妹之间，同事之间，人与人之间都有妒忌心在作祟，导致反目成仇家破人亡。妒忌心在中国社会无处不在！有的人杀人不是因为利益，不是因为别人拿了他什么东西，而是因为妒忌心而杀人害命！
 <br/>
 <br/>
 在全世界来看，西方人的性格外向没有妒忌心；而中国人性格内向，过去讲涵养不露内秀它有它的好处，不好的是近代在绝对化的教育下道德缺失，这妒忌心就表现得很强烈，而整个亚洲地区都受中国文化影响，都稍带点妒忌心，唯独中国人妒忌心很强烈。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产主义.html" target="_blank">
  共产主义
 </a>
 入侵中国，把中华民族精英赶尽杀绝，摧毁中国几千年建立的道德体系。〝无神论〞是撒旦(魔鬼)的作品，无神论的本质是：没有神(四维空间生命)，做坏事也不存在报应不必顾虑，跟他们去杀人放火，他们坐享其成，没有神，他们就是神；没有救世主，他们就是救世主。
 <br/>
 <br/>
 中国的大环境是个败坏放大器。中国以前好的东西变坏；坏的东西变的更坏。外国进来的东西，好的变坏；坏的变的更坏。
 <br/>
 <br/>
 中国人受影视作品的影响，有个错误的认识：认为西方欧美国家的人都崇拜潮流文化，其实不是的，欧美人恰恰相反，传统文化才是他们的主流文化，他们非常排斥潮流文化，教堂礼拜也是他们传统文化之一。艺术作品嘛！来源于生活高于生活。而现实西方国家的家庭是非常传统的，影视作品中的性开放乱七八糟的东西，在美国主流社会中根本找不着，它是少数人非主流。
 <br/>
 <br/>
 所有的潮流文化都有一个特点：妄视道德。比如那个衣服，好好的衣服它弄个破洞，也叫流潮。年轻人对传统文化一无所知也盲目跟风，其实西方主流社会的人叫它服装垃圾。
 <br/>
 <br/>
 正是因为中国社会妄视道德而使人的妒忌心发酵，人的所有欲望离开了道德都是不行的，那么！中国这个道德严重缺失的环境下，这个妒忌心表现的太强烈了，它正推毁人与人之间的感情。妒忌心在人与人之间，在同事之间，只要这妒忌心存在着，什么好事都变成坏事，有人因为妒忌心而作恶，也有动刀动枪杀人害命。
 <br/>
 <br/>
 城市的人，人面不识诚信崩溃！他们也许认为农村的人会比较淳朴。恰恰相反，妒忌心在农村表现太严重了，平时好来好去，一有机会作恶绝不放过，一有机会落井下石搬的是最大的石头！其根本原因是妒忌心作祟。
 <br/>
 <br/>
 整个社会来说，50后，60后的人最坏，70后好点，80后好，90后很好！泡在共产党麻烦传文化的时间越长的人越坏！人们只知道有马列，却不知道还有马列变种，红卫兵，爱国贼，伪民运就是马列变种之一，马列变种不是一种，而是几十种上上百种。
 <br/>
 <br/>
 每个中国人都要正视妒忌心这个问题，每个人心里都住着一只魔鬼，要时刻把它看紧，把它关住，不让它跑出来害人。
 <br/>
 <br/>

 <br/>
 <br/>
