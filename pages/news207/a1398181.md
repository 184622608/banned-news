### 文武：中共管不好经济硬要管
------------------------

<div class="wysiwyg">
 经济是有其自身运行规律的，只有遵循经济运行规律，才能保持经济健康持续的发展，过多人为的干涉经济，只会起反作用。
 <br/>
 <br/>
 中共邪恶惯了，管天、管地、管人、管经济，只要中共能伸得到手的地方中共都要管，结果是越管越糟，越管越乱。
 <br/>
 <br/>
 2008年发生世界经济危机时，中国的经济正处于产能过剩的阶段。按照经济运行的规律来讲，这时乘着经济危机的到来，正好是去过剩产能的大好时机，正是应该该破产的就破产，该关门的就关门，留下好的去掉坏的。
 <br/>
 <br/>
 可是，中共为了显示自己管的经济只增长不下滑的神话，在本来就已经产能过剩的情况下，不仅不是去产能，反而去增加产能，走了一条与经济运行规律完全相反的路。
 <br/>
 <br/>
 近十年来，中共光印的人民币就翻了好几翻，债务不断扩张，原本就过剩的产能越来越过剩，整个中共的经济就处于债务滚债务的模式下不断的扩张。
 <br/>
 <br/>
 这种靠
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_债务扩张.html" target="_blank">
  债务扩张
 </a>
 来维持经济增长的模式是不可能持续的，总有走到头的一天，而且越往后走越危险，因为越往后债务越高，崩溃时灾难就越大。
 <br/>
 <br/>
 可是，中共为了显示自己管控经济所谓伟大的本领，中共就是要反其道而行之，中共就是要走扩张债务的疯狂之路。
 <br/>
 <br/>
 中共这一疯就是十年，这种疯狂扩张债务发展经济的路子，在中共对外出口同步增长的情况下，确实是可以维持的，因为出口多，外汇就挣得多，这样购买力就强，就能维持债务的扩张，但是，一旦出口创汇不能与债务扩张同步增长，那么巨大的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济泡沫.html" target="_blank">
  经济泡沫
 </a>
 马上就会显现出来。
 <br/>
 <br/>
 这一次美国与中共发生了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，这样中共出口商品的利润就开始减少，对应中国国内经济的泡沫就开始显现，大量积压的产品就成了废品。
 <br/>
 <br/>
 同时中共为了转移债务，搞的一带一路项目，目前也受阻，债务向外转移也困难了。
 <br/>
 <br/>
 中共一看这下完了，债务扩张的路子走不下去了，现在又匆忙的去产能，去扛杆，去债务。可是去又去不下去，一去大量的工人就要失业。
 <br/>
 <br/>
 中共成天宣传跟着中共跑就有钱挣，好多跟着中共跑的人，相信中共的人，好多在中共国营企业里，如果这些国营企业基本上都倒闭了，哪里还有那么多跟着中共跑的人呢？
 <br/>
 <br/>
 于是，中共去扛杆不去国营企业，而是去民营企业的扛杆，就是中共的银行不给民营企业贷款了，因为所有的企业全是债务，不管是国营还是民营的，要谁还谁都还不起，不叫国营还，就叫民营还。
 <br/>
 <br/>
 还不了，就停贷，于是，好多民营企业就被停贷了，好多民营企业就没有钱了，好多民营企业就要倒闭了。
 <br/>
 <br/>
 现在这一下可好了，真正与人民生活息息相关的企业就是民营企业，民营企业要都倒闭了，人民的生活影响可就大了，那中共维稳的费用也就更高了。
 <br/>
 <br/>
 现在，中共是左也不行右也不行，于是昨天中共破天荒的召开了支持民营企业的大会，高规格的宣称要全力支持民营企业。
 <br/>
 <br/>
 什么叫高规格支持民营企业呢？不过就是再借债，再扩张，不可能还有第二条路。
 <br/>
 <br/>
 同时，债务扩张不能没有出口的支持，于是中共开始向美国投降，估计这一次中共是真的要向美国全面投降了。
 <br/>
 <br/>
 中共搞了半天自称能耐大，管得好经济，现在看来是要服输了，要向美国投降了。不过这是形势所逼。大家都听过农夫和蛇的故事，谁听信中共，谁就要倒楣。
 <br/>
 <br/>

 <br/>
 <br/>
