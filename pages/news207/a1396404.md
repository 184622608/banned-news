### 崔士方：郑晓松坠亡 隐情藏在一年之内
------------------------

<div class="wysiwyg">
 时年59岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 10月20日在其澳门住所坠楼身亡。因为事前几乎全无征兆，而又发生在习近平即将南下出席港珠澳大桥开通典礼前夕的敏感时间点，加之北京公布的〝抑郁症〞死因与澳门官方回应的含混驴唇对不上马嘴，引发猜测纷纷。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 的仕途，可以大体分为四节。
 <br/>
 <br/>
 首节在外交部。他官至时任外交部副部长姜恩柱的秘书。姜恩柱后来调任驻英大使，郑也追随到英国牛津大学外交官研究生班学习。姜后来升任新华通讯社香港分社社长（正部级），对这位贴身秘书的提携自在不言中。
 <br/>
 <br/>
 第二节在财政部。郑晓松在财政部国际司任职12年，其中两年任驻亚洲开发银行中国执行董事。随后，他升任财政部部长助理。这一节，他与两名日后落马的财政部高官有交集——前财政部副部长王保安（在国家统计局局长任上落马，受贿1.54亿，被判无期徒刑），前财政部副部长、党组副书记张少春（今年9月20日刚被双开，据传贪腐金额比王保安更惊人）。
 <br/>
 <br/>
 当时的王、张两名副部长都是郑晓松的〝领导〞。更为微妙的是，王保安和张少春均是秘书帮上位，两人分别是前财政部长项怀诚、刘仲藜的大秘。
 <br/>
 <br/>
 第三节在福建省。郑晓松在财政部部长助理的位置上待了一年多后，并没有依惯例晋升副部长，而是转轨到福建任副省长，这时已是十八大后的2013年7月。在这里，郑晓松与两名日后落马的福建省高官有共事，一位是时任福建省长苏树林（石油帮人马，后获刑16年），另一位是时任福建副省长徐钢（已获刑13年）。3年后，郑晓松升任福建省委常委、省委秘书长。
 <br/>
 <br/>
 第四节在中联部（中共中央对外联络部）和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 。这两个部门虽然简称很接近且都是正部级，但互不隶属，前者的权势要大于后者。郑晓松在福建省委秘书长位置上才待了2个月，就在2016年6月回京出任中联部副部长，2017年9月空降澳门中联办任一把手，晋升正部级。
 <br/>
 <br/>
 这一步升迁，郑晓松坐的可是火箭穿梭机。要知道，在中共官场，从副部级升正部级是个大门槛，平均耗时9年多，长者甚至蹉跎超过15年。而郑晓松只用了4年2个月就〝跳龙门〞成功，并在十九大后顺利跻身中共中央委员。郑晓松与前任澳门中联办主任王志民都被视为习近平的福建系人马。两人同时调职，分别主掌澳门中联办和香港中联办，很多分析认为，这是习近平清洗曾庆红港澳势力的举措。
 <br/>
 <br/>
 这说明，直到2017年9月这一刻，郑晓松都还是中南海属意的要角。那么何以腾达才不过一年，郑晓松的仕途就急遽下挫，以致要以死谢幕呢？
 <br/>
 <br/>
 从消息人士称郑晓松被中纪委约谈来看，郑是因涉某案而自行了断。
 <br/>
 <br/>
 王志民之前的澳门中联办主任是李刚。李刚是被贬任国侨办副主任后，才被中纪委甕中捉鳖的。但李刚与郑晓松并无渊源，且李刚案并不重，只是判留党察看一年。由轻案带出要自杀收场的重案，这种可能性不大。
 <br/>
 <br/>
 郑晓松是涉及〝新鲜滚烫〞的财政部张少春案？还是到澳门扫曾庆红〝余毒〞而被余毒传染？这目前还是一个谜。
 <br/>
 <br/>

 <br/>
