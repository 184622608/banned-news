### 程晓容：央视记者被英警方起诉 释何信号？
------------------------

<div class="wysiwyg">
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国警方.html" target="_blank">
  英国警方
 </a>
 消息，中共央视驻伦敦记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孔琳琳.html" target="_blank">
  孔琳琳
 </a>
 ，因为在9月30日的一次会议上与一名工作人员发生肢体冲突，被以〝普通袭击罪〞起诉。该案件定于11月7日在伯明翰裁判司署提讯，孔琳琳将出庭。如果被判有罪，她将面对最高六个月有期徒刑或罚款，或两者兼施。
 <br/>
 <br/>
 10月26日，中共外交部发言人针对此事回答了记者提问，称中国记者〝正当行使权利表达观点〞却遭阻挠、粗暴对待及被倒打一耙，〝中方对这种颠倒黑白的做法表示严重关切。〞
 <br/>
 <br/>
 是谁在颠倒黑白、倒打一耙？
 <br/>
 <br/>
 9月30日，在英国保守党主办的香港问题座谈会上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孔琳琳.html" target="_blank">
  孔琳琳
 </a>
 在座上批评正在发言的保守党人权委员会副主席罗杰斯，她因此被主持人要求离场。当会议工作人员Enoch Lieu指示其离开时，孔琳琳用手拍打他并高喊抗议。Enoch Lieu表示，孔琳曾两度掌掴他。
 <br/>
 <br/>
 该事件引发轩然大波，中共即刻高调支持孔琳琳、要求英方会议组织者道歉。有评论认为，女记者此举显然系中共官方操控，旨在煽动国内的〝爱国情绪〞、减缓贸易战的压力。
 <br/>
 <br/>
 中共胆敢〝亮剑〞，是利用西方的自由体制来破坏自由，进而渗透和扩张。面对外部的指责，中共惯常以〝反华〞和〝干涉主权内政〞挡箭，而对于它公然干涉外国内政的行径，则拚命抵赖。
 <br/>
 <br/>
 9月18日媒体消息，美国政府要求中共新华社驻美分支及中国环球电视网注册为〝外国代理人〞。9月23日，《中国日报》英文版在美国爱荷华州当地报纸插入四页广告，以图文内容指责川普总统，企图煽动美国农民的情绪、干涉美国选举。一周后，孔琳琳在伦敦闹场，这一切，令外界更加看清中共驻外官媒的工具作用和嚣张气焰。
 <br/>
 <br/>
 孔琳琳和中共肯定没有料到，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国警方.html" target="_blank">
  英国警方
 </a>
 会将其正式起诉。现在，事件进入司法程序，骂阵抗议都已无用。这是中共官媒在海外遇到的一大挫折，是西方社会拒绝中共言行暴力的一个信号。
 <br/>
 <br/>
 当中共声称支持〝中国记者〞〝维护正当合法权益〞时，不妨来看一看，在中国境内，中外记者受到了何种对待，特别是那些调查真相、追求新闻自由的媒体人，他们的权利可获保障？
 <br/>
 <br/>
 2010年3月，时任《中国经济时报》记者王克勤发表了2万字的调查报告《近百孩子不明病因致死致残——山西疫苗乱象调查》，曝光了假疫苗造成的大量惨剧和相关黑幕。事后，签发此文的《中国经济时报》社长、总编包月阳被调职。2011年7月18日，该调查部被解散，王克勤被解职。
 <br/>
 <br/>
 2016年1月7日和8日，驻甘肃武威市的《兰州晨报》记者张永生及《兰州晚报》、《西部商报》的两名记者先后失联，原来他们是被武威警方以涉嫌嫖娼、敲诈勒索罪名抓走，后张永生被执行逮捕。由于张永生曾发表过不少揭露当地社会阴暗面的文章，此事引起民愤，认为他是被警方陷害。在巨大的舆论压力下，张永生获释。有网民讽说：〝顺我者昌，逆我者嫖娼！〞
 <br/>
 <br/>
 2017年3月3日，在中共两会前夕，BBC驻华记者John Sudworth及其团队在湖南计划采访一位访民时遭遇暴力威胁，他们的相机被砸碎。后来，一些穿着制服的警察和两名外事部门官员出现。他们被迫删除了一些拍摄的画面，并且签下〝认罪书〞，才得以离开。
 <br/>
 <br/>
 2017年12月14日，韩国总统文在寅访华的第二天，中共警护人员无故野蛮阻拦随行采访的韩媒记者，并对一名李姓记者进行围殴。
 <br/>
 <br/>
 2018年1月，齐崇怀走出了监狱的大门。这位知名的〝反腐记者〞、〝揭黑记者〞，因为揭露山东地方官员的腐败，于2007年6月被从家中绑架，随后被以〝敲诈勒索罪〞判刑，服刑期间险遭灭口，后又被加刑。历经10年冤狱，他满头白发，妻离子散。
 <br/>
 <br/>
 2018年1月30日，北京驻华外国记者协会发布了2017年在华工作环境报告，其中提到，2017年，在受访的外国驻华记者中，超过40%的记者认为在华采访环境恶化，而2016年这一比例为29%。
 <br/>
 <br/>
 寥寥几例，凸显中共对新闻自由的扼杀、对人权的迫害。相比之下，人们这样评价孔琳琳的〝伦敦秀〞：〝她在英国无论如何闹都是安全的，绝对不会失联的。〞〝一个来自不民主国家、从不在自己领土示威的人，去另外一个民主国家，说自己有示威权利，这事多讽刺、多可笑。〞
 <br/>
 <br/>
 真正爱华者，理应为王克勤、齐崇怀鸣不平，应当高声抗议打压良心记者的公权力，并且感谢所有为中国人的权利挺身而出、敢于对抗中共的海内外团体和个人。
 <br/>
 <br/>
 那些为孔琳琳叫好的人，可曾以同样的激情声援为民请愿、放下个人安危、深陷牢笼的高智晟、王全璋、余文生等维权律师，可曾为疫苗受害者、上访民众、贫苦百姓大声疾呼？爱国，需要正视历史，需要反思中国大地的乱象和灾难，需要倾听深受暴政伤害的生命的呼声。唯此，方可辨清黑白，找到自由和希望的方向。
 <br/>
 <br/>
