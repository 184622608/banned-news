### 【走近川普】虔敬信神 回归传统 神迹屡现
------------------------

<div class="wysiwyg">
 11月5日晚间九点多，密苏里州开普吉拉多市（Cape Girardeau）的体育馆内，万头攒动。这是川普一天之内的第三场选举造势活动。
 <br/>
 <br/>
 当众人随着川普演讲而热情鼓舞之际，一名女性观众忽然晕厥倒地。川普注意到了。
 <br/>
 <br/>
 〝这里有没有医师？有劳一下，医师。拜托了，谢谢。〞川普在台上促请医师急救女观众，〝医师，你慢慢来，我们有的是时间。〞
 <br/>
 <br/>
 稍过片刻，川普对全场观众说，〝大家祷告一下，说祷告词，这样会（对女观众）很好。〞许多观众随即脱帽、默默低头，为晕厥的女观众祈祷。
 <br/>
 <br/>
 不久，有人唱起西方著名的赞美诗歌《奇异恩典》（Amazing Grace），川普随即伸手示意，请全场观众一起跟着为女观众唱颂。当观众们唱完一个段落，医护人员也顺利将女观众急救完成，用担架将女观众送离现场，现场掌声热烈响起。
 <br/>
 <br/>
 〝我想要感谢在场的每个人，感谢你们刚刚的表现。太棒了。〞川普说。
 <br/>
 <br/>
 这段插曲，实属意外，但却堪称是这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 过程中，最令人动容的美丽插曲，不但再次展现川普对神的信仰与虔敬，也体现了美国社会的传统价值与精神。
 <br/>

 <h4>
  自幼信神 川普就任总统 捍卫
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信仰自由.html" target="_blank">
   信仰自由
  </a>
 </h4>
 <br/>
 〝我的宗教信仰价值观，都是母亲灌注给我的。〞许多人听到川普虔诚信神，都曾深感惊讶，但川普强调，他从小就上教堂、敬爱神，〝神在我每一天的生活里。〞
 <br/>
 <br/>
 此言不虚。川普自上任以来，便积极提倡信仰、提醒人们崇敬神，他签署行政令提倡信仰自由、确认信仰自由将是川普政府的优先政策；美国国务院还举行全球首届〝推动宗教自由部长级会议〞，促进改善全球信仰自由，并公布《波托马克宣言》（Potomac Declaration），呼吁各国将宗教自由列为优先政策。
 <br/>
 <br/>
 在大大小小活动中，川普也一而再、再而三提醒美国民众：〝在美国，我们不崇信政府，我们崇信神！〞
 <br/>
 <br/>
 事实上，川普大力倡导宗教信仰、捍卫信仰自由，不仅仅是出自他个人的信仰因素，更是为了恢复美国的悠久历史传统，带领人们回到神的身旁。
 <br/>
 <br/>
 <h4>
  对神崇信：美国开国精神与传统价值
 </h4>
 <br/>
 〝我们认为下面这些真理是不言而喻的：人人生而平等，造物者赋予他们若干不可剥夺的权利，其中包括生命权、自由权和追求幸福的权利。〞1776年，美国建国之父们于《独立宣言》中，开宗明义阐述神对人类的崇高地位。
 <br/>
 <br/>
 不难发现，信仰神，是美国的立国之本；信仰神，也是美国社会最重要的传统信念与文化。
 <br/>
 <br/>
 〝在所有促成政治昌盛的秉性与习性当中，信仰与道德是必不可少的支柱。〞222年前，美国国父乔治‧华盛顿（George Washington）在告别演说中曾经如此提醒后人。
 <br/>
 <br/>
 法国思想家托克维尔（Alexis de Tocqueville）于1831年游历美国时，也深刻观察到，在美国人心中，自由与宗教信仰密不可分、彼此攸关。
 <br/>
 <br/>
 〝没有道德，便无法建立自由；没有信仰，便无法建立道德。〞他在《论美国的民主》（De la démocratie en Amérique）书中写道。换句话说，人有信仰，才能重视道德，用道德来自我约束言行，知可为与不可为，才不会侵犯他人，也才能进而在社会上建立起真正的自由。
 <br/>
 <br/>
 信仰对人类道德、对自由社会的重要性，川普了然于心，并全力守护提倡。
 <br/>
 <br/>
 〝每一天，我们都为宗教信仰人士挺身而出〞，8月底川普在白宫表示，〝因为我们知道，信仰与家庭——而非政府与官僚，才是美国生活的重心。而且我们知道，自由是来自创世主的恩赐。〞
 <br/>
 <br/>
 川普对信仰的恢复、对传统的守护，似乎收到了效果。
 <br/>
 <br/>
 根据皮尤研究中心（The Pew Research Center）2017年底完成的调查发现，目前美国民众仍有高达80％相信神的存在。而多数人信仰神的理由，主要是〝与神亲近〞、〝让孩子们有良好道德基础〞、〝让自己成为一个更好的人〞。
 <br/>
 <br/>
 <h4>
  虔诚信神 恢复传统 川普任内屡传奇迹
 </h4>
 <br/>
 你相信神迹吗？
 <br/>
 <br/>
 前美国二战名将巴顿将军（George Patton）以信神著称。1944年，他在德国作战时，曾领着25万将士虔诚祷告，暴风雪天气突然连日放晴，让巴顿一举扭转战局，成为历史佳话。著名的〝巴顿祷告词〞也流传至今。
 <br/>
 <br/>
 不管你相信与否，但神迹似乎也屡屡出现在川普身旁。
 <br/>
 <br/>
 2017年9月，超级飓风艾玛（Irma）侵袭美国佛州，预计将成为美国历史上最具破坏力的风暴。艾玛登陆前夕，佛州州长史考特（Rick Scott）在电视新闻上敦请全国民众为佛州百姓祈祷，而川普也在大卫营率领内阁成员，虔诚低头向神祈祷，留下经典画面。

 奇迹般地，飓风路线突然偏移，飓风中心避开了人口稠密的迈阿密，改由佛州西部沿海地区登陆，飓风强度也快速减弱。原本预计出现海水倒灌的地区，竟因风向改变，反而出现海水消失、海床裸露的罕见奇观。
 <br/>
 <br/>
 更令人惊讶的是，原本各界预期，美国接连遭受两个飓风侵袭，经济或将严重受创，第三季度经济增长率可能下降到1％，或甚至负成长。但稍后，财政部公布数据，经济不但正成长，而且增长率竟高达3.2％，出乎众人预料。
 <br/>
 <br/>
 事实上，川普当选美国总统的过程，亦可堪称神迹。
 <br/>
 <br/>
 当时川普面临众多主流媒体的猛烈抨击，负面报导不断，几乎所有民调都对川普不利、预测希拉里当选。但最后，却由川普这位政治素人逆转胜出，跌碎所有专家眼镜。
 <br/>
 <br/>
 不过，一名退休消防员马克‧泰勒（Mark Taylor）透露，早在2011年，神便已告诉他，将安排川普出任美国总统；神并在2015年再次向泰勒预告此事，川普也果真参选，并顺利当选。这段神迹故事，后来被出版为书籍《川普预言》（Trump Prophecies），并被翻拍为电影。
 <br/>
 <br/>
 不仅如此，川普今年以来，相继遭遇泛左派媒体的多轮攻击，攻击力道之大，前所未见；然而，川普的民调支持度却不但没有下降，反而节节上扬，彷佛愈挫愈勇，目前来到约50％的高点。如此难以解释的奇迹，堪称近代美国政治史上首见。

 倘若，川普率领的共和党最终能够胜出，或许可说是神对川普恩赐的又一项奇异恩典。
