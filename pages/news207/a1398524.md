### 唐浩:美国中期选举落幕 未来影响涵盖四个层面
------------------------

<div class="wysiwyg">
 众所瞩目的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 终于落幕，结果大致与先前各方预测相去不远，共和党继续主掌参议院，民主党则一如〝钟摆效应〞的预期，拿下众议院。
 <br/>
 <br/>
 这场选举，对两党来说，堪称各有所得，也各有所失。但对于川普政府及共和党来说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的未来影响，可能涵盖几个层面：
 <br/>
 <br/>
 <h4>
  一、巩固参议院优势 深化司法体系改革
 </h4>
 <br/>
 共和党不但守住参议院多数党地位，与民主党的席次差距也有所扩张，这一切不得不归功于川普奔走全国各地的卖力助选。
 <br/>
 <br/>
 例如德州参议员克鲁兹（Ted Cruz），受到民主党对手欧洛克（Beto O’Rourke）的节节进逼，选情一度告急。最后川普亲自飞往德州，为昔日的总统大选竞争对手卖力拉票，终使克鲁兹以2个百分点左右的得票差距，成功连任。
 <br/>
 <br/>
 尽管共和党失去了众议院，但当选的共和党众议员，多数也都获得川普的背书推荐，抑或在政见方向上跟随川普步伐，从而赢得选民青睐。
 <br/>
 <br/>
 因此，可以预见，未来川普在共和党内的号召力与影响力，将更为扩大、深化；未来共和党的内部意见也可望更为一致，对以后的政策、法案以及2020年的总统大选及国会选举，将带来正面助力。
 <br/>
 <br/>
 此外，参议院除了拥有立法权之外，最重要的是具有对联邦法官、大法官以及内阁官员的审查同意权。如今，共和党席次优势增加，将有助于川普在任命法官及内阁官员时更加顺利、更有效率。
 <br/>
 <br/>
 特别是联邦法官与大法官的任期都是终身制，因此川普提名的法官人选，将持续带领美国司法体系朝传统保守派方向回归，影响力可望长达数十年。
 <br/>
 <br/>
 <h4>
  二、众议院易主 川普内政挑战加大
 </h4>
 <br/>
 民主党取得了众议院主导权，毋庸置疑，未来川普政府的新政改革将在众议院遭遇不少挑战，诸如政府预算案、基础建设、健保改革及其它各项法案的立法，很可能都得面对民主党的强势拦截，需要更多的谈判协商。
 <br/>
 <br/>
 换句话说，在2020年大选及国会选举前，川普的内政工作或许将不易施展手脚。
 <br/>
 <br/>
 此外，川普政府与外国谈定的各项自由贸易协定，还需要经过国会两院的表决通过，才能付诸实行。届时，众议院民主党是否会对这些贸易协定放行，还是会百般刁难，这不但将影响川普兑现对选民的竞选承诺，还将影响美国与其它国家组成贸易同盟、孤立中共的经济大战略。
 <br/>
 <br/>
 再者，众议院拥有对总统及政府官员的调查权、弹劾权，所以也不难预见，众议院很可能会对川普发起一连串的调查案、弹劾案。
 <br/>
 <br/>
 尽管民主党深知，这些调查案可能查不出什么犯罪事实，弹劾案也几乎不可能通过，但其目的是要通过这些负面消息来打击川普、尽可能阻止川普在2020年连任；并藉此争取、巩固民主党阵营的基本盘，为2020年大选铺垫基础。
 <br/>
 <br/>
 <h4>
  三、共和党不再完全执政 〝四大威胁〞态度或转变？
 </h4>
 <br/>
 共和党未能守住众议院，不再〝完全执政〞，不仅影响美国内政，还可能牵动外交事务。
 <br/>
 <br/>
 特别是正与美国贸易交战的中共以及正遭美国经济制裁的朝鲜、伊朗、俄罗斯，不但紧盯选举结果来研判如何与川普政府博奕，甚至还被美方点名批评，试图干预中期选举。
 <br/>
 <br/>
 如今，选举结果出炉，中、俄、朝、伊与美方互动的态度会否有所转变？特别是选前一度表态放软、希望与美国重新谈判的中方，是否真心愿意与美国达成贸易协议？还是会再次〝变脸〞、回到强硬姿态与川普政府继续僵持？甚至会不会暗中笼络民主党，里外联手给川普制造难题？
 <br/>
 <br/>
 特别是川普7日在记者会上首度披露，中共有意放弃〝中国制造2025〞计划，美国将与中方达成贸易协议。但中方是否果真有意落实？抑或只是用来敷衍川普、换取贸易战休兵的〝口惠〞之计？接下来，将是重点观察期。
 <br/>
 <br/>
 <h4>
  四、美对外事务更积极
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
   美中贸易战
  </a>
  或再延烧
 </h4>
 <br/>
 共和党与民主党分掌国会两院，川普政府的内政改革或许将艰辛不易，但预料身为谈判高手的川普，也将设法与民主党沟通互动，为美国人民达成政策协议。
 <br/>
 <br/>
 不过，川普政府很可能会更积极在对外事务上发挥作用，特别是针对中共、俄罗斯、朝鲜及伊朗问题。
 <br/>
 <br/>
 川普深知，这〝四大威胁〞对美国长远发展与国家安全构成严重隐患，这些国家虽然遭到川普的强力压制，目前不敢轻举妄动，但他们很可能想要〝以拖待变〞，甚至试着〝帮助〞美国在野党促成川普不再连任。届时，〝四大威胁〞对美国带来的深远风险与危害，将更难以防治或根除。
 <br/>
 <br/>
 因此，川普很可能会在2020年前，加大力度、加快速度对这些国家施压，削弱或消除他们带来的核武威胁、经贸威胁、商业窃密、间谍渗透等，藉此守护美国的国家安全与人民安定。
 <br/>
 <br/>
 中期选举大势底定后，7日美国国务卿蓬佩奥宣布取消周末与朝鲜劳动党副委员长金英哲会面；川普也宣布，他与金正恩的第二次会面将推迟到2019年初，川普还强调，〝我们不急〞。
 <br/>
 <br/>
 事实上，川普正有意藉此延长经济制裁时间，对朝鲜与金正恩施压，加速促其认真落实无核化，而非一味空口说白话、趾高气昂地催促美国取消制裁。
 <br/>
 <br/>
 至于众所瞩目的美中贸易战，尽管川普抢先披露双方有意达成协议，但回顾近日北京高层的公开发言，几乎都是重复以前说过、却从未兑现的〝虚空承诺〞，也看不到中方有意对美方要求的〝三零二停一允许〞（注），做出具体改变。
 <br/>
 <br/>
 因此，倘若中共并非真心愿意改变不公平的贸易手段、放弃不道德的经济发展模式，甚或放弃〝中国制造2025〞的旗舰级计划，则川普政府势必继续强力施压中共政权，届时贸易战不但将难以达成协议，还可能继续升温延烧。美中贸易战的烽火刀光，也将持续交锋到下一年头。
 <br/>
 <br/>
 [注] 〝三零二停一允许〞为美方对中共的贸易要求，包括〝零关税、零补贴、零非关税贸易壁垒〞，〝停止窃取知识产权、停止强迫企业技术转移〞以及〝允许美国人在中国拥有自己的公司，不必与中国企业合资〞。
 <br/>
