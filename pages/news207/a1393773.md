### 文武：红卫兵办事过力治国无能
------------------------

<div class="wysiwyg">
 中共在上个世纪六十年代中至七十年代中，发动了一场史无前例的文化大革命，史称十年浩劫。
 <br/>
 <br/>
 在这十里年，正值少年时期的一代，就是被称为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红卫兵.html" target="_blank">
  红卫兵
 </a>
 的一代人，到今天这些人年龄正好是六十岁左右。这一代人因为受文革的影响，胆大妄为，天不怕、地不怕，敢拚、敢斗、敢杀。
 <br/>
 <br/>
 当年的红卫兵究竟杀了多少人，现在也无法准确的统计，但是把中国的传统文化与古典、古籍毁坏个精光，这确实是活生生的事实。
 <br/>
 <br/>
 由于当年的红卫兵，基本上都没有好好上学读书，现代的科学、文化知识他们没有学，而传统文化又被毁了，中国传统社会优良的风气他们也没有养成，这样他们就成为被耽误了的一代人。
 <br/>
 <br/>
 可是历史发展到今天，中共中高层的官员主要都是些六十左右的人，正好都是些当年的红卫兵。
 <br/>
 <br/>
 随着改革命开放的到来，随着上大学的兴起，这些人好多又都去到大学，半路出家，混了个文凭，出来后，一个个又都成了大学生，研究生。博士、教授的也都有。
 <br/>
 <br/>
 由于这些人，真才实学没有，投机取巧在行，这样就把原本还认真的知识界和学术界，也被这些人变成了弄虚作假、投机取巧的场所。
 <br/>
 <br/>
 中共为了维护其统治，发现只有知识才能救中共，因为中共发现，经过中共培养的知识界，文化界，艺术界，原来更能做假、原来更能骗人，原来中共培养出来的知识分子才是中共天然的盟友，才是中共真正执政的基础。
 <br/>
 <br/>
 如是，中共的官员就大量从知识界中、文化界中、艺术界中去选取，如是，中共的官员开始也有高学历的硕士、博士、博士后和些访问学者。
 <br/>
 <br/>
 这些人上台后，就占领了整个中国社会的上层，他们卡住了中国社会所有的要职。后来上来的一些人，文化大革命后开始上学的一些人，就跟着他们上行下效，就学会了这些弄虚作假、投机取巧的本领，于是整个中国社会就成了一个作假的社会。
 <br/>
 <br/>
 中共十八大后，特别是十九大后，中共的官场几乎都由这些当年的红卫兵与准红卫兵主宰着。
 <br/>
 中共官场中相比年青的人，因为上行下效的惯性效应，弄虚作假的本领更是技高一着。
 <br/>
 <br/>
 这些人，在顺利的时候，可以一起享乐，但是在逆境中的时候，就互相拆台，他们互相内斗起来也是一个比一个很。
 <br/>
 <br/>
 所以，在中共原来经济高速发展的情况下，他们在一起一个比一个贪，一个比一个色，你贪污千万元，我就贪污亿元，你贪污几十个亿，我就贪污几百个亿，你弄十个情妇、我就弄百个情妇，个个能贪、个个能占。
 <br/>
 <br/>
 但是真正治国呢？却是个个无能。
 <br/>
 <br/>
 这些人办事没有分寸，好大喜功，志大才疏。一上来就是大肆宣传梦想，厉害了我的国，中国大国崛起来了， 中国超过美国了，厉害了中国二零二五，厉害了中国的千人计划。
 <br/>
 <br/>
 口号喊得震天响，可是，事实上呢？
 <br/>
 <br/>
 中共是一点也不厉害，要说中共厉害，那就是欺负中国老百姓厉害，其它的中共是要什么没有什么。
 <br/>
 <br/>
 例如，这次美国发起了向中共的贸易反击战，这一下要来真的了，这些人就全傻眼了，来真的他们就不行了，用力过猛就砸到自己头上来了。
 <br/>
 <br/>
 前阵子中共高喊，要与美国以牙还牙，要与美国战到底，要将美国打怕、打服、打倒。可是不到几个月，美国没有倒下，中共倒是要爬下了。
 <br/>
 <br/>
 这不，前天中共高调发表了个〝白皮书〞，整本书就是喊冤叫屈，书的结尾，就是想让美国给他们机会，请美国不要把杀架在中共的脖子上，想让美国放他一马。这么厉害的中共，这么厉害的当年的红卫兵，这么厉害的后来的准红卫兵，现在怎么又软了呢，怎么又硬不起来了呢？
 <br/>
 <br/>
 其实是这些人不懂什么叫妥协，就连江湖的规矩，出来混是要还的规矩他们都不懂。
 <br/>
 <br/>
 他们只懂得阴谋与诡计，他们只知道弄巧使坏，所以，就算是中共前天发表的满篇叫屈的白皮书，其实也只是一本诈降书，因为这些红卫兵只知道弄虚作假，只知道使奸耍滑。
 <br/>
 <br/>
 说句老实话，中共的这些红卫兵们，主动倒台可能有的人还能得到善终，如果这样使坏到底，那结果就是彻底毁灭。
 <br/>
 <br/>
 文武
 <br/>
 2018年9月26日星期三
