### 周晓辉：丧失〝四心〞北京坐等玩完
------------------------

<div class="wysiwyg">
 11月9日，美国白宫贸易高级顾问、《致命中国》作者纳瓦罗表示，美国和许多国家签订贸易协议，都很信任他们能遵守、执行，但中共不同，和中共谈判即使有结果，也很难保证它能执行。纳瓦罗说，不相信中共，是因为中共过去从不遵守和美国的协议。如中共加入世贸组织时所做出的多项承诺，在十六年过渡期后都没有实行；中共2015年曾向奥巴马政府承诺，不会将南海人工岛礁军事化，也不会对美国企业进行黑客攻击，但也都没有履行。
 <br/>
 <br/>
 而就在此前，中国欧盟商会针对北京当局11月5日在进博会上的〝降低关税，扩大市场准入，进口更多的外国商品，加强对知识产权保护〞等承诺，也表达了不信任，称〝北京当局的许多承诺只是重复过去的声明，让欧洲商业界对这种承诺越来越不感兴趣〞。欧盟商界对北京承诺要扩大中国自由贸易区〝不再抱有幻想〞，中国的自贸区是〝与中国的经济实力和成熟度不一致的缩手缩脚的改革方式〞。
 <br/>
 <br/>
 孔子有言〝人而无信，不知其可也〞，意思是说一个人没有诚信就不能立足于世间，延伸可知，〝业无信则不兴〞，即干事业没有诚信也不能兴旺、发展。一个人如此，一个企业如此，一个党派、一个国家焉能不如此？从中共过往的所言所行看，西方国家对其两面三刀的做法早已看透，因此对其无法落地的言辞不再相信。换言之，中共已然使欧美国家对其丧失了信任，丧失了对其可以信守承诺的信心。
 <br/>
 <br/>
 事实上，不仅仅国际社会对中共丧失了信任、信心，中共自身也在丧失自信，中国各阶层对其也丧失了信心。1949年中共窃取政权后，背弃了其给予人民真正的民主、使中国成为一个真正的民主国家的承诺，接连发动了一个个残酷运动，戕害死了至少八千万中国人，而且迄今未休。在中共治下，假、恶、暴横行，真、善、忍被打压。
 <br/>
 <br/>
 2012年习近平上台后，在反腐的名义下，抓捕了大批江派高官、马仔，并清洗官场，进一步掌控军队、武警、中央及各省重要部门，将矛头指向江泽民，因此得到了不少国人的信任和支持。但随着中共十九大推出〝新思想〞等倒退之举，随着当局在中美贸易、对内经济等方面的举措失当，中国社会经济下滑，股市、汇市惨不忍睹，外资加速撤离，民企被打压、收购，大量企业倒闭，大量人员失业，社会抗争此起彼伏，北京政权为此加强了对维权律师、退伍军人等上访人士、法轮功学员、新疆的迫害，而假疫苗、P2P爆雷、非洲猪瘟疫情蔓延等恶性事件的出现，使得怨声载道，更让民众认清了中共政权，更加不信任当权者，民心也在加速流失。
 <br/>
 <br/>
 古语道：〝水能载舟，亦能覆舟。〞失去民心的中共政权还能走多远呢？难怪当局在中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 压力下呼吁〝共克时艰〞时，遭到民众无尽的嘲讽，更有中国人在美国驻中国大使馆微博下留言，希望美国总统川普拯救中国人。中国人靠外国总统拯救，打了谁的脸呢？
 <br/>
 <br/>
 面临着如此的危局，北京当权者也不再高喊〝厉害的国〞，也不再〝奉陪到底〞，其自信心似乎也降到了谷底，不得不放下身段，选择妥协。
 <br/>
 <br/>
 除了丧失了西方国家的信任，丧失了本国民众的信任和民心，丧失了自信心，北京当局还丧失了另外〝两心〞，那就是官员的〝忠心〞和来自上天的〝天心〞。
 <br/>
 <br/>
 关心中国时政的人不难发现，伴随着大量官员落马的是当局不断强调官员的忠诚，这说明中共党内博弈一直激烈，此前几次未遂政变所产生的影响至今仍未平复。是以，现当局突出要求官员的忠心，其实就是因为现在中共党内存有贰心的人不少，官员怠于政事，只是很难一一洞察。
 <br/>
 <br/>
 不过，〝忠〞不是〝愚忠〞。儒家认为，〝忠〞是一种人对他人他事应具有的品德和行为准则。一个人不仅要对自己的分内之事，对亲、师、友、君所交待的事要忠（尽心），而且待亲、待师、待友、待君都应该忠（尽心），但〝忠〞绝不是无原则的。孔子认为，如果君主能按照礼来对待臣下，那么臣下就会尽心去做君主所任命的分内之事。即君王以礼待臣子是臣子〝忠〞的前提。
 <br/>
 <br/>
 此外，古人还云：良禽择木而栖，贤臣择主而事。历代读书人的志向都是：辅佐明君，治平天下。因此，那些看透中共邪恶本质的中共官员选择不忠于中共，恰恰是在拯救自己。
 <br/>
 <br/>
 至于丧失〝天心〞，2002年贵州〝中国共产党亡〞的天然藏字石的横空出世以及迄今为止三亿多人的三退之举，就是明证。
 <br/>
 <br/>
 毫无疑问，丧失了〝信心、民心、忠心、天心〞的北京当局，在保党的基础上再怎么折腾都无济于事，只能坐等玩完那一天的到来。
 <br/>
 <br/>
