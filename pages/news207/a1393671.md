### 夏小强：中共干扰 美国中期选举结果如何
------------------------

<div class="wysiwyg">
 9月26日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在主持联合国安理会会议时，直言不讳地指责中共一直试图干预美国即将在11月举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 说，他是第一位挑战中共贸易政策的美国总统，因此中共〝不想要我或我们（共和党）赢〞。
 <br/>
 <br/>
 随后川普在推特上给出中共企图干扰美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的例证，贴出了中共官媒《中国日报》英文版在美国爱荷华州当地最大报纸《狄蒙因纪事报》（Des Moines Register）插入的四页广告。该广告公开攻击川普的贸易政策，公开称川普〝愚蠢行事〞、害惨该州的大豆种植者。
 <br/>
 <br/>
 川普能在2016年总统选举取胜，主要就是因为得到爱荷华州等农业重镇的支持，而中共选择针对大豆采取报复行动，就是为了打击川普在国内的支持度。
 <br/>
 <br/>
 中共利用着美国的言论和新闻自由，干涉和干扰美国的选举，是对美国国家利益的侵犯和践踏。那么，中共的干涉和对美国民众的恐吓威胁能起到什么作用呢？
 <br/>
 <br/>
 中共类似的丑行早有先例。
 <br/>
 <br/>
 2000年3月份台湾举行总统大选。此次大选政党竞争激烈，3个主要的总统候选人分别是民进党主席陈水扁、亲民党主席宋楚瑜以及国民党主席连战。陈、宋民意测验指数相当接近，连稍低。
 <br/>
 <br/>
 为了阻止民进党候选人当选，在选举前三天的3月15日，中共总理在全国人代会的中外记者会上称：〝不管是谁，只要搞台湾独立，就没有好下场〞、〝切莫一时冲动，以免后悔莫及〞、〝还有三天，世事难测，台湾同胞你们要警惕啊！〞。
 <br/>
 <br/>
 中共的言论经电视画面播出，瞬间被传播到台湾内外和全世界。外界的反应怎样呢？
 <br/>
 <br/>
 中华民国陆委会主委苏起表明，中华民国是主权独立国家，并批评中共没有权力对总统大选指指点点；同日美国国务卿阿布莱特在众议院答覆问题时也表明无法接受中共对台声明，而正在日本访问的美国国防部长威廉・科汉在厚木基地接受访问时也呼吁中共停止对台恫吓，并表明美国有义务对台湾提供防御性武器装备。
 <br/>
 <br/>
 民进党候选人陈水扁在屏东举行的造势晚会上表示，只有台湾人民有权利选择自己国家的领导人，中共并没有这个权利。陈水扁问到：是台湾人民在选自己的总统，还是中共领导人在替我们指名？我们是在选台湾未来的新领导人，还是在选特首？
 <br/>
 <br/>
 独立候选人宋楚瑜同一天在台北市中正纪念堂广场向支持群众表示，台湾所有的民众绝对不接受武力恐吓。
 <br/>
 <br/>
 由于中共的言论被视为对台湾选民威胁，引发部分选民反弹反而支持被视为偏向台独的陈水扁，成为影响选情的因素之一。因此，外界有评论认为，在关键时刻，中共〝帮助〞陈水扁赢得总统大选。
 <br/>
 <br/>
 如今，中共故伎重演，对美国中期选举的干扰和利用广告对美国民众发出潜在的恐吓，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共干扰.html" target="_blank">
  中共干扰
 </a>
 台湾选举和恐吓台湾民众如出一辙。
 <br/>
 <br/>
 那么，中共这种方式会起到作用吗？
 <br/>
 <br/>
 美国德雷克大学（Drake University）政治学教授斯基德莫尔（David Skidmore）表示，中共这种做法〝可能会适得其反〞，美国读者〝可不喜欢被外国政府操纵〞。
 <br/>
 <br/>
 爱荷华州大豆协会首席执行官利兹（Kirk Leeds）表示，农民或许多爱荷华人不会被中共的插页所左右，因为那是中国（中共）试图发放的政治信号。
 <br/>
 <br/>
 川普在7月曾宣布，为美国农民提供120亿美元的救助，旨在农民遭受贸易关税的冲击。爱荷华州的农民预计在第一轮联邦援助中获得约5.5亿美元。
 <br/>
 <br/>
 中共的共产主义意识形态和党文化所形成的一套行事方式，充斥谎言欺骗、蛮横无理、毫无信义等，与西方社会和正常世界的一切格格不入，被大部分西方世界的民众所排斥和厌恶。
 <br/>
 <br/>
 中共对美国民众发出恐吓，干扰美国中期选举，必然遭到美国民众的抵制和反感，对于中期选举，必然会起到中共预期相反的结果。很有可能，中共对美国中期选举的干扰，会重现此前中共干扰台湾选举的情况，中共〝帮助〞川普和共和党赢得中期选举。
 <br/>
 <br/>
 历史的事实多次证明了一个现象：中共每次费尽心机干出一件坏事，却都得到相反的结果，同时，干出的这件事，也都成了中共的丑事和恶事，被曝光于天下，让人们更加看清中共的邪恶和丑陋。
 <br/>
 <br/>
 ──转自《大纪元》本文只代表作者的观点和陈述。
 <br/>
