### 古玉文：赞比亚反殖民游行 打脸中共大撒币
------------------------

<div class="wysiwyg">
 中共借中非论坛向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲.html" target="_blank">
  非洲
 </a>
 无条件大撒币两个月后，中共非洲铁杆小兄弟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赞比亚.html" target="_blank">
  赞比亚
 </a>
 于11月5日发生反对中共新殖民的示威游行。
 <br/>
 <br/>
 中共多个官方媒体均援引中共驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赞比亚.html" target="_blank">
  赞比亚
 </a>
 使馆网11月6日发布的安全提醒消息称，〝5日上午，有人在基特韦点燃轮胎，拦截、拍打过往车辆，基特韦市两家中国人经营的商店遭洗劫，一家中国人经营的商店玻璃被砸碎。另有数家中国商店和企业被当地民众围住〞。
 <br/>
 <br/>
 中使馆发布的安全提醒说，〝发生小规模骚乱和游行主要原因是一些当地人听信‘政府把赞比亚林业公司出售给中国人’的传言，聚众闹事〞。
 <br/>
 <br/>
 又是〝少数不明真相者，听信别有用心者的谣言，聚众闹事〞，这一套欺骗了大陆中国人几十年的阶级斗争话术，现在随着大撒币撒到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲.html" target="_blank">
  非洲
 </a>
 土地上了。
 <br/>
 <br/>
 赞比亚是南部非洲第一个与中共建交的国家，也是1971年联合国会员大会议案上，助中共成为联合国会员国的联合倡议者之一。早在上个世纪，中共就帮助赞比亚修建铁路、建筑公路和大楼等。随着中共对赞的不断投资和进驻企业，中共涉足的行业和领域越来越多。
 <br/>
 <br/>
 在赞比亚近千家的中企覆盖了采矿业、制造业、建筑业、旅游业、农业和服务业等行业，2005年以来，中企参与建设了大量非洲地区基础设施项目，包括港口、公路、铁路、水坝、电信网络、发电站以及机场。
 <br/>
 <br/>
 中共的投资给赞比亚带来了沉重的债务负担。美国之音报道赞比亚政府债务的官方数字是106亿美元，约占GDP 34%。
 <br/>
 <br/>
 赞比亚财政部长表示，中共贷款占所有商业贷款的大约30%，而一些专业分析人士则估计，如果加上不同形式的项目融资，中共向赞比亚提供的贷款大约有50亿美元，占赞比亚债务的将近50%。
 <br/>
 <br/>
 民众和分析人士并担心这是个被政府掩瞒的债务数字，且这个比例有可能很快上升到60%。
 <br/>
 <br/>
 事实上，中共对非洲国家进行的新殖民政策使两国之间的劳资纠纷越来越严重，并不断引发人身伤害及伤亡冲突事件。中企管理人员和赞比亚工人之间经常互殴甚至枪击。2012年8月，赞比亚中资科蓝煤矿的工人示威抗议，要求增加工资，造成1名中国经理死亡，另外2名中国员工负重伤。
 <br/>
 <br/>
 中共带给赞比亚的不只是债务危机、工厂的低安全生产、工人的低工资，近年来随着中共经济的下滑，当地大宗商品价格大幅下降，失业率骤增，深深地打击了经济。〝报应来了，因为我们一直把鸡蛋放在同一个篮子里，〞赞比亚工会联合会(Zambia Congress of Trade Unions)主席尼蔻乐・希姆巴 (Nkole Chishimba)曾经这样说过。
 <br/>
 <br/>
 中共的到来，向当地官员输出腐败与权贵经济模式。当地富人可以开着路虎，住着别墅，雇傭人，穷人温饱都解决不了。贫富差距巨大。警察为了挣钱可以利用职权随意开罚单。
 <br/>
 <br/>
 贫穷的下一步就是面临失去主权，这就是中共新殖民主义的核心目标。中共提议将接管赞比亚的肯尼思‧卡翁达国际机场，如果赞比亚政府无法准时偿还其巨额负债的话。此外，中共还对赞比亚国家经营的国家广播公司有60%的控股权，这表明中共试图控制赞比亚的宣传话语权。
 <br/>
 <br/>
 据赞比亚国家电视台报道，此次示威与游行中，5日当天共有151名参与者被捕。赞比亚信息与广播部长、政府首席发言人多拉・西利亚表示，政府没有出售赞比亚林业公司，也未与任何投资者洽谈收购。
 <br/>
 <br/>
 中共新华网称中共对游行事件 〝高度重视，加紧与赞有关部门联系交涉〞，〝密切注意事态发展。〞赞比亚政府对事件采取的行动和表态很大程度上跟中共的运作有关。这种节奏酷似中共对本国民众的维稳手法。
 <br/>
 <br/>
 但非洲国家和人民对中共的新殖民主义越来越越不买帐了。中非论坛结束后没几天，肯尼亚国内第二大报刊就刊登了一幅讽刺中共为母猪，给一群非洲领导人哺乳的漫画。11月5日的上海进博会上，肯尼亚总统直言不讳的批判中共〝贸易严重地向中国（中共）有利的一面倾斜。〞
 <br/>
 <br/>
 有评论指，一些非洲国家对待北京的态度是〝左手拿钱右手打脸〞。
 <br/>
 <br/>

 <br/>
