### 文武：中共在贸易战中投降的原因
------------------------

<div class="wysiwyg">
 中共在半年前与美国打贸易战之初，一再叫喊着要与美国奉陪到底，要与美国打到底，绝不退让，绝不屈服。可是，也就才半年多点时间，中共就自己打自己的脸，就退让了，就屈服了，就投降了。
 <br/>
 <br/>
 究其原因，就是这本来就是一场不该打的贸易战。在半年前的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 之初，几乎是绝大多数的分析人士都认为，中美之间不可能打贸易战，中共根本就没有与美国打贸易战的资本，因为中美之间在贸易领域根本就没有对等的实力，中共肯定会妥协，肯定会退让，所以中美之间不可能发生贸易战。
 <br/>
 <br/>
 可是，我当初就却认为，中美之间一定会发生贸易战，因为中共的惯性决定了中共的行为。中共有一个违背常识的邪理，那就是光脚的不怕穿鞋的，我是流氓我怕谁，根本不管什么常识不常识，只看谁的手段很，只看谁的阴谋阴，先打上去再说。
 <br/>
 <br/>
 果然不出所料，中共是喊声连天，阴招频出，以为不出几下就把美国给吓回去了。
 <br/>
 <br/>
 可是中共这次是碰到对手了，这次的美国政府不原来的美国政府了，现在的美国总统也不是原来的总统了，干吓唬也不管用。
 <br/>
 <br/>
 随着时间的推移，中共就开始原形毕露了，中共的包子就开始露馅了，没有什么真材实料，只是些稻草。
 <br/>
 <br/>
 中共在经济领域的根本问题是产能过剩，整个中共的经济靠债务在维持，唯一能暂时维持中共债务的原因就是对外贸易和外来投资，一旦外贸出了问题，那么中共债务就无法维持，整个中共经济就会崩溃。所以中共根本就没有与美国打贸易战的实力，一打必败。
 <br/>
 <br/>
 可是明明知道要败，中共为什么还硬着头皮叫嚣了半年呢？
 <br/>
 <br/>
 原因是中共不仅受到来自外部的压力，更主要是来自内部的压力。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共内斗.html" target="_blank">
  中共内斗
 </a>
 已经到了你死我活的程度，任何一派都在等着对方出错，都在等着出手的机会，一旦找到了下手的机会，马上就置对方于死地。
 <br/>
 <br/>
 所以，中美贸易战从根本上来说，并不是因为与美国发生了什么贸易摩擦，出现了什么贸易纷争才发生的贸易战，中共内部各派别都非常清楚，美国提出的条件是合理的，中共根本就离不开美国，与美国和好还来不及呢？怎么可能会与美国发生什么贸易战呢？
 <br/>
 <br/>
 但是，中共内斗决定了，当美国提高关税时，中共内部各派别开始都没有谁敢提出妥协的方案，因为谁提出妥协的方案，谁就是晚清政府，谁就是李鸿章，谁就是投降派，谁就是背黑锅的人。
 <br/>
 <br/>
 所以，谁都不敢明显表示妥协，甚至谁不敢发言，谁都不敢站出来表态，这样就自然成了，以牙还牙了，就自然成了反击到底了。
 <br/>
 <br/>
 因为，这样的口号是政治正确，是不被对手打倒的口号。
 <br/>
 <br/>
 可是半年过去了，与美国打贸易战的后果显现了，事实摆在面前了，与美国的贸易战不能再打了，与美国的贸易战更不能升级了。
 <br/>
 <br/>
 得停止贸易战，在这种现实面前，中共所有的各个派别，就达成了共识，要妥协，要投降。
 <br/>
 <br/>
 中美贸易战从表面上看是中共与美国打的贸易战，实际上是中共内部各派在打仗。
 <br/>
 <br/>
 所以，打贸易战实际上是中共内斗的原因，现在中共向美国屈服，向美国投降，也是中共内斗的原因。
 <br/>
 <br/>
 作者提供 本文只代表作者的观点和陈述。
 <br/>
 <br/>
