 <h4>
  报导证据含糊 违反专业规范与伦理
 </h4>
 <br/>
 那么，这篇报导，是否真是左派媒体再次攻击川普政府的假新闻？
 <br/>
 <br/>
 首先，从技术层面检阅该篇报导，文中消息来源并不明确。记者只以一句〝采访近12名白宫幕僚与亲近川普的人〞带过，不仅人数不明确、人员无名无姓，没有直接明确的爆料者引语，也没有直接向川普本人求证或给其回应机会，实乃违反〝平衡报导〞、〝让当事人回应负面消息〞的专业规范与新闻伦理。
 <br/>
 <br/>
 毕竟，此项报导涉及对国家元首的人身攻击、国家安全及政府稳定，兹事体大；而《纽时》即便不能专访川普，也有许多机会可在记者会后向川普提问求证或请求回应，但他们却付之阙如。尽管报导中引述了白宫副发言人的一句发言，但也不禁令人质疑，是否媒体刻意不想让川普回应此事。
 <br/>
 <br/>
 <h4>
  川普信赖彭斯辅政 屡次委以重任
 </h4>
 <br/>
 其次，从政治层面检视川普与彭斯当选两年来的互动关系，也与报导内容颇为迥异。
 <br/>
 <br/>
 川普是初入华府政治圈的商界大亨，讲究务实、行动、效率，对于政界的游戏规则、暗潮汹涌，难免有所陌生；彭斯是政坛出身，历任众议员、印第安那州州长，恰恰与川普形成完美互补。他在任内极力辅佐川普，举止得体、言行得当，为川普适时分担不少辅助性工作角色，也强有力地为川普政策辩护，从而获得川普的高度评价与信任。
 <br/>
 <br/>
 因此，川普对彭斯多次委以重任，例如新成立的太空军（Space Force）建制工作，川普便特地交给热爱太空的彭斯负责督军。
 <br/>
 <br/>
 川普还让彭斯分担当前最重要的美中战略工作，彭斯不仅在10月4日发表历史性演说，揭露中共对美国的敌意行为；日前的亚太经合会（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 ）活动，也由彭斯前往，针对贸易战与中国问题发表重要演讲，与北京当局隔空交锋。
 <br/>
 <br/>
 此外，身为谈判高手的川普，还将彭斯纳为他的谈判伙伴，两人联手施展〝红脸、黑脸〞战略。
 <br/>
 <br/>
 例如，年初的韩国平昌冬奥，川普派遣彭斯前往出席，与金正恩妹妹金与正擦身而过、冷面无视，向各界宣示美国对朝鲜的强硬决心；稍后，川金会确定举办，再由川普出马对金正恩展开〝善意外交〞与谈判，最终达成无核化框架协议。
 <br/>
 <br/>
 日前刚落幕的APEC也是雷同。彭斯出席领袖峰会，强硬宣示美国反对中共的不公平贸易、反对中共威胁南海与印度—太平洋地区，彭斯表现政治家的严肃神情与坚毅语气，让中共倍感压力。接下来，或可预见，川普与习近平在G20峰会期间的互动，可望出现较为友善、乐观的气氛，有助双方沟通协商。
 <br/>
 <br/>
 川普与彭斯，〝一红脸、一黑脸〞的谈判战略，神似川普过去与其得力爱将、川普集团资深法律顾问乔治‧罗斯（George Ross）的角色合作关系，为川普的政治与外交谈判创造更多槓杆空间。
 <br/>
 <br/>
 更何况，川普在商界驰骋数十年，善于观察人员，十分重视员工对公司的忠诚与品德。川普与彭斯相处两年，他能不清楚彭斯的为人吗？倘若川普不能充分信任彭斯，他会将这些重任交给彭斯吗？
 <br/>
 <br/>
 再者，川普与彭斯都虔诚地信仰神，彭斯更是白宫内阁〝《圣经》学习会〞的发起人，川普自然更能信任一位真诚信神、重德律己的副总统。
 <br/>
 <br/>
 <h4>
  左派媒体离间分化川普政府 中共黑影舞动？
 </h4>
 <br/>
 而《纽约时报》的报导内容与事实相去甚远，报导证据力又薄弱，那么为何媒体还要刊登这篇必遭质疑的文章？
 <br/>
 <br/>
 或许，这篇报导有几项用意：1）尽管中期选举落幕，民主党拿下国会众议院，但左派媒体对川普的猛烈攻击仍将持续展开，甚至炮火升级；2）左派媒体有意挑拨、离间川普内阁团队，试图伤害政府内部士气与互信，减损人民对川普政府的观感与信任，藉此影响2020年总统大选选情；3）左派有意分化川普与彭斯的合作关系，藉此削弱川普政府对抗华府旧势力与〝深层政府〞（Deep State）的战力，削弱2020年大选连任的团队实力。
 <br/>
 <br/>
 别忘了，《纽约时报》曾在9月5日刊登一篇自称是川普政府高级官员的匿名投书，大肆攻击川普及政府内部。《纽约时报》罔顾新闻专业、刊登匿名黑函的行径，一度引发轩然大波，也招来〝损害国家安全〞、〝颠覆政府〞的质疑与批评。
 <br/>
 <br/>
 并且，部分美国左派媒体在过去数年来，与中共特务往来密切，甚至一度成为中共内部某派系在海外散发黑材料、攻击另一派系的政治打手。
 <br/>
 <br/>
 今年中期选举前夕，中共亦曾通过海外喉舌《中国日报》在爱荷华州刊登四个整版的〝类新闻广告〞，大力攻击川普。
 <br/>
 <br/>
 如今，左派媒体再次对川普发动攻势，试图分化、离间川普政府，是否又有中共的黑影晃动其后？中共是否仍在布局干预美国2020年总统大选？值得我们审慎关注。
 <br/>
