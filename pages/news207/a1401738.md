### 杨宁：G20会场外中共很恐惧 世界看在眼中
------------------------

<div class="wysiwyg">
 不管中共的媒体如何渲染北京领导人在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 举办的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上尽显〝大国〞外交手段，如何给世界经济、G20的未来发展方向〝把脉〞，发生在会场外、全世界都看在眼中的一幕，已然让这一切都暗淡无光。
 <br/>
 <br/>
 据大纪元报导，11月29日，数名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员在习近平下榻的喜来登酒店附近，进行了和平请愿。他们挂出两条传递心声的横幅，上边分别以中文与西班牙文写着：〝停止迫害法轮功〞、〝法办江泽民〞。一个在诸多民主国家或打着民主旗号的国家再正常不过的和平请愿活动，却被中共收买的亲共侨团组织和当地警官所阻挠。
 <br/>
 <br/>
 报导指，在法轮功学员请愿期间，中共驻布宜诺斯艾利斯大使馆和当地几个亲共侨团组织，组成了千人队伍，他们身穿红上衣，手举大红旗，将法轮功学员全部包围，并大喊大叫。随即，一名来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共大使馆.html" target="_blank">
  中共大使馆
 </a>
 的官员和一名便衣警察交谈后，几名阿根廷警察未说一句话，就开始强行取下法轮功学员所挂的横幅。
 <br/>
 <br/>
 根据阿根廷法律，允许公民用横幅、旗帜进行和平请愿，横幅、旗帜等都属私人物品，他人抢夺私人物品，属违法行为。是以法轮功学员护住横幅，但却遭到警察的暴力对待。录像显示，当地一名警察局长还在现场指挥，并命令一名警察搜查一名法轮功学员的背包，谎称背包中有炸弹。当天，警方拘捕了9名阿根廷法轮功学员，但于隔日下午全部释放。
 <br/>
 <br/>
 毋庸置疑，中共大使馆正是此次暴力事件的幕后推手，也应该是中共收买了当地警察，甚至将谎言灌输给阿根廷高级警官，从而以莫须有的罪名拘捕法轮功学员。
 <br/>
 <br/>
 中共驻外大使馆如此作为并非首次。无论是早前的江泽民、胡锦涛的出访，还是习近平上台后的对外访问，所到之处的中共大使馆、领事馆针对法轮功的抗议和请愿，全都是如临大敌，方法也如出一辙。
 <br/>
 <br/>
 比如2002年江访德期间，中共驻德使馆公使孙荣民等外交官使出浑身解数，使德国政府动用上千军警大规模戒严，并警告穿黄蓝衣服不能上街不能过马路，住旅馆的要被强行搜查撵出来，不服从的还得戴手铐，大声讲真话要被掐喉咙关禁闭，井盖被封，江坐火车要封锁全站等。民主国家的德国做出如此匪夷所思的举动自然引起了轩然大波。
 <br/>
 <br/>
 比如2012年6月，时任中共国家主席胡锦涛对丹麦进行国事访问期间，丹麦和北欧周边国家的部分法轮功学员集会向胡锦涛一行呼吁停止迫害法轮功。但在中共使馆的压力下，哥本哈根警方出动四辆警车挡住抗议人群，以遮挡胡锦涛一行的视线。哥本哈根警察还拘留了试图接近胡锦涛一行表达抗议的六位支持西藏人士。2013年6月，中共前政协主席俞正声访问丹麦，也遇到了部分法轮功学员的请愿，其中有两名法轮功学员因为身穿黄色T恤，而被丹麦警察强行带离抗议现场。
 <br/>
 <br/>
 随后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丹麦警方.html" target="_blank">
  丹麦警方
 </a>
 被提起诉讼。2017年12月，丹麦独立调查委员会做出决定，判决哥本哈根警方违反《宪法》。今年4月，丹麦警方发布新闻公告，正式承认他们在对待抗议民众的处理过程中，犯下了严重错误，警方并宣布对提起诉讼的八名当事人给予每人2万丹麦克朗的经济赔偿。
 <br/>
 <br/>
 再比如2014年习近平访问德国、法国、比利时期间，法轮功学员打出了〝法办元凶江泽民、罗干、周永康〞等横幅，同时呼吁所在国政府正视中共迫害法轮功的事实，并予以制止。而这些国家的中共大使馆一如既往地要求当地警方禁止法轮功学员请愿、抗议，并试图干扰当地剧院神韵的演出。
 <br/>
 <br/>
 且不说中共驻外使馆的长臂干涉行为让在场内大展〝大国自信〞的中共党魁在国外丢尽了脸，单从世界各地中共使领馆的一贯所为看，他们是何等的不自信。对揭露迫害真相、戳穿中共谎言，害怕、恐惧到了何种程度——连黄色、蓝色的衣服都不敢看到。相较于西方领导人面对诸多抗议时的定力，这样〝厉害的国〞、这样的不自信，世界都看在了眼中，谁会与之真心交往呢？
 <br/>
 <br/>
 ──转自《大纪元》本文只代表作者的观点和陈述。
 <br/>
