### 子华：中共摧毁道德良知　国人应退党自救
------------------------

<div class="wysiwyg">
 如今在中国，有句话几乎成了很多人的口头禅，当听到死人了，张口就说：中国这么多人，死几个人算什么。这句话折射出整个社会人性的普遍丧失、人心的极端冷酷。由于传统文化被破坏和社会变异思潮的影响，很多人只注重自身的利益和各种欲望的满足，想尽一切办法捞钱，吃喝玩乐，全然不顾他人，甚至见死不救。
 <br/>
 <br/>
 有个贩卖桔子的人拉了一车桔子，在国道上出车祸翻了车，人们蜂拥而上去抢桔子，谁也不去救人。现在的人不但见死不救，而且发展到盼着人死，乐见人死，催着人死。当今还普遍存在这样的情况，看着父母年迈了，体弱多病，不能创造财富了，做儿子、儿媳妇的就厌烦了，巴望老人马上死去。不少人见父母碍眼时就会说出：你咋不死呢？死了省得烦人了！
 <br/>
 <br/>
 今天的人，对待自己的父母都敢如此大逆不道，何况对待他人呢？
 <br/>
 <br/>
 甘肃庆阳十九岁的女孩被老师猥亵，状告无门，满腔的委屈、悲愤，绝望中准备跳楼，围观的人中有的催促〝怎么不跳？〞〝为了等你跳下来，我在楼顶晒一小时太阳了〞，〝活该吧，赶快跳啊，浪费大家时间〞……几个小时后，女孩在一片催促声中，跳了下来，了却了年轻的生命。如果人们能珍视生命，并对她多一些同情，替她说句公道话，让她感受到人与人之间的温暖和生命的珍贵，就会给予她生活下去的勇气，这个生命完全有机会活下来。
 <br/>
 <br/>
 仅仅是一些没有人性的人和那个流氓老师把这个女孩推向了绝路吗？
 <br/>
 <br/>
 几十年来，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的统治下，人们被强迫地灌输无神论、斗争哲学，并把传统文化视为毒草，让人不讲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_道德.html" target="_blank">
  道德
 </a>
 ，不信善恶有报，只讲党性不讲人性。中共的党性就是无神论学，斗争哲学，与天地人为敌，追求个性解放、性开放、金钱至上。通过中共几十年是非混淆、黑白颠倒的洗脑，人们渐渐地失去了人与人之间的尊重和关爱，甚至连最起码的怜悯之心都丢弃了，传统文化中人命关天，仁者爱人的道德理念荡然无存。传统道德讲〝无怜悯之心非人也〞，这种〝非人〞行为正是中共几十年如一日破坏传统文化和传统道德所要达到的目的，也是中共在历次整人害人的运动中所极力鼓励和吹捧的。
 <br/>
 <br/>
 在中共治下的社会里人人都有被推上绝路的可能。中共这个西来魔鬼正是逼着中国人走上绝路的罪魁祸首。
 <br/>
 <br/>
 在中共篡权的初期，就全部没收了资本家、工商界人士的财富，杀害和逼死了很多人，当时就有不少人不堪屈辱而跳楼自杀。时任上海市长的陈毅，就把人跳楼当成了乐子，曾每天询问〝今天又有多少空降兵？〞文革期间陈毅被中共整治，患绝症痛苦地走到生命尽头的时候，其下场比那些被逼死的资本家好多少呢？
 <br/>
 <br/>
 中共历来视生命为草芥，磨牙吮血，杀人如麻，它最喜欢见到的就是一批一批的人死在它面前，甚至满地淌的都是人的鲜血，才最过瘾呢，这也就是中共特别嗜好血红色的原因吧。中共毫无遮掩地叫嚣与天斗其乐无穷，与地斗其乐无穷，与人斗其乐无穷。中共其邪恶的本性充份体现于它的一次又一次的整人运动中，它以迫害民众、大肆杀戮维持政权，害死了八千万中国人，还毫无羞耻地鼓吹自己〝伟光正〞。它教唆人们只要站在党的立场上无论干什么坏事甚至杀人放火都是正确的，欺骗人们没有善恶报应，让人只重眼前利益，不计后果，一步一步地用歪理邪说摧毁人的道德、让人放弃人性，最终导致整个社会道德崩溃，逼着人死，看着人死，乐于人死。
 <br/>
 <br/>
 有这么一件事，在中原一个城市的郊区，有一家农民要盖房子，土地办的人知道后就让她家拿出一大笔钱才允许盖。在农村盖房不容易，很多人家都是积攒了多年的钱，才能盖得起房子，如果这家再拿给土地办那么多钱，就要拉账，所以不能满足土地办的要求。这家的儿子还在部队当兵，是军属，也不能得到照顾。一天准备施工时，土地办开车来了六七个人，阻止施工。女主人愤恨至极，从屋里拿出一瓶毒药，对他们说，你们要不让盖，我就喝药死。土地办的人说，喝药也不让盖。女主人见没有了商量余地，一气之下把一瓶药灌进了肚子。令人发指的是，土地办的六七个人眼睁睁地看着农妇把一瓶药喝完，谁也不去上前制止，直到农妇倒在地上，他们才开车扬长而去。
 <br/>
 <br/>
 一个鲜活的生命，就这样被逼得死在了中共公职人员的眼皮底下。传统道德讲人命关天，现在在中共的治下，人的生命还不如蒿草。
 <br/>
 <br/>
 文革是一场最惨的整人害人的浩劫，据专家们的保守估计，文革中被中共害死的达773万人，可中共党魁毛泽东还不尽兴，竟然说，文革这样的运动要七、八年来一次。一九八九年〝六四〞学生被杀害，中共头目仍不解恨地大喊，杀二十万换取二十年稳定。一九九九年中共迫害信仰佛法的民众时，其匪首江泽民狂妄地叫嚣，打死白打死，打死算自杀！
 <br/>
 <br/>
 近年来，在大陆经常看到这样的标语：人民有信仰，国家有力量，民族有希望。可转眼间基督教又被打压。中共向来是说一套做一套，反覆无常，耍尽流氓。
 <br/>
 <br/>
 中共最仇视神佛，它能真心容许人们对神佛的信仰吗？中共视人类为敌，它能干出对人类有益的事吗？如今在社会上广传的《共产主义的终极目的》一书，揭示了中共魔鬼的本来面目，以及其有预谋地一步步把社会道德推向全面崩溃，摧毁传统文化，切断人与神的联系，让人彻底背叛神佛，丧失人性，招致天降大难，从而把中华民族及人类推向毁灭。
 <br/>
 <br/>
 看看现在中国大陆在中共腐败荒淫和性开放的带动下，几乎全民堕落，物欲横流，人心魔变，为了钱财色欲，很多人无恶不作，根本不管他人死活，甚至抛弃家庭，伤害亲人，丧尽天良……人们的生存环境和生活条件遭到严重的破坏，毒食品，假货物弥漫中华大地，损害着人们的健康，危及着人们的安全。中共不亡，中国人会有好日子过吗？中共早期搞了三十年自立更生，直到国人几乎吃不上饭了，才被迫改革开放。改革开放三十多年，中共一路腐败堕落，在国际上臭名远扬，在国内不得民心，到了内外交困的地步，还要回头搞自立更生。说穿了，就是中共又想闭关锁国，关起门来共产中国人的私有财富，把人民视为砧板上的肉任意宰割。无论中共怎么变换手段，最终都是要把中国人带上绝路。
 <br/>
 <br/>
 觉醒的人已经看清，跟随中共只有死路一条。所以这些年来善良的法轮功学员根据天象的变化，得知了天灭中共的信息，不忍心看着自己的同胞去做中共的陪葬，年复一年，日复一日地劝人们退出中共的党、团、少先队组织，好避开大难，走向未来。
 <br/>
 <br/>
 法轮功是佛法修炼，教人们按照真善忍做好人，不坑不骗，做事先考虑别人，能使社会道德迅速回升，福泽人类。中共和江泽民流氓集团为迫害法轮功，制造出一个个谎言，并花钱买人冒充法轮功学员恶搞〝天安门自焚〞陷害法轮功，煽动全国民众仇恨佛法，让人们对佛法犯罪，从而招来灾祸。愿人们能明辨善恶，抛弃中共，支持正义，定能得到上天保护，进入新的纪元。
 <br/>
