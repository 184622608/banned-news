### 程晓容：郑晓松坠亡 中共官员应如何自救
------------------------

<div class="wysiwyg">
 近期，中共官场频发爆炸性新闻：中共澳门中联办主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠亡.html" target="_blank">
  坠亡
 </a>
 、国际刑警组织主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被诱捕。仅据公开报导，今年1月以来，至少有8个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共官员自杀.html" target="_blank">
  中共官员自杀
 </a>
 ，方式以跳楼居多。中共高官的诸种劣迹及生死之谜，可谓创下世界纪录。党国官员系高危职业，似乎难得〝善终〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 〝失踪〞后，其妻在法国报案并召开新闻发布会求助。但是很快，孟氏夫妇的婚姻及奢侈生活被起底，坊间议论：孟宏伟到底触碰了哪条底线？他们是否值得同情？紧接着，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 在港珠澳大桥开通前夕，离奇死亡。接二连三的案件令人惊心——中共官场为何如此混浊迷乱？
 <br/>
 <br/>
 日前，流亡澳洲的中国法学教授袁红冰到达台湾后，对外透露，孟宏伟手中握有几项中共重要机密，包括：孟宏伟曾参与谋杀十世班禅大师、参与制造台湾三一九枪击案等等，此外，郑晓松〝被跳楼〞也是因孟宏伟出事所致。
 <br/>
 <br/>
 这一份爆料透出了政治谋杀及在台湾制造动乱的黑幕，这和中共吹嘘的〝伟光正〞相差多远？孟宏伟究竟参与了多少、了解多少？他的上线、下线涉及何人？中共体制内，还有多少个〝孟宏伟〞、〝郑晓松〞？
 <br/>
 <br/>
 郑晓松、孟宏伟案，当令体制内外的人深思。那些落马受惩的昔日高官，那些自杀身亡者的家属，可曾意识到，正是那个令其富贵显赫、权钱无限的党，将他们彻底埋葬？那架红色的〝绞肉机〞，杀害了数不尽的同胞、摧毁了古老的传统文化、扭曲了无数国人的心灵，也把机器上的齿轮和螺丝一并绞毁。
 <br/>
 <br/>
 回顾历史、对照现实，中共鼓吹的阶级斗争，不仅对外搞得翻天覆地，也一直在内部残酷地实践着。几十年来，从国家主席、大元帅，到部长、市长，从军官、文官到名流、明星，无论对党〝忠〞或〝不忠〞，几乎无人能逃脱被整肃的命运。许多人惨死在狱中或病床上，有的最后连骨灰都不知去向。这些个体的悲惨结局，一方面与其曾参与、协同作恶有关，另一方面凸显红色暴政的嗜血本性。
 <br/>
 <br/>
 《九评共产党》指出，中共是反天、反地、反人类的邪灵，它的本质是：假、恶、暴。中共宣扬无神论，与普世价值为敌，纵容官员和民众行恶，从而制造出最反常、最邪恶的政治体系。凡是加入中共、为其效力者，被迫放弃良知和善念，还要认同和参与制造谎言、实施暴力，否则，就是对党不忠、背叛，就会被排挤、打压、失去升迁的机会，甚至会遭到构陷报复。
 <br/>
 <br/>
 在中共的淫威、欺骗和利诱下，数以百万、千万计的官员或违心或主动地推波助澜、助纣为虐，干出了一桩桩伤天害理之事。69年来，中共整治异己，清除精英，迫害维权人士、上访民众，迫害广大的法轮功学员和宗教信众，犯下滔天大罪。那些依上级指令、参与迫害的各级责任人，手上都沾着鲜血。因此，加入中共，就等于跳进了最肮脏的泥潭，如不退出，只能越陷越深，最终被污泥没顶。
 <br/>
 <br/>
 近几年，大批高官落马，充分应验了善恶有报，也在警示仍在作恶者：赶快收手、早日赎罪。跟着中共走，走向泯灭人性与良知的不归路。到头来，身后留恶名，官位财产皆成空。
 <br/>
 <br/>
 有网友写道：〝在中共体制内难免做过坏事，甚至大坏事。这样的人应该放下思想包袱勇敢的站在正义的一边来，让那个对你们威胁最大的邪恶势力土崩瓦解，你们才可能不是下一个郑晓松，你们就可能做一个扬眉吐气的正常人。〞
 <br/>
 <br/>
 所有中共官员，都面临选择。自杀不是出路。只有在思维与行动上反转、与中共决裂，才有生机。弃暗投明，需要巨大的勇气，去认清现实、否定过去；这勇气将会结束悲剧、开辟新生。
 <br/>
 <br/>
 ──转自《大纪元》本文只代表作者的观点和陈述。
 <br/>
