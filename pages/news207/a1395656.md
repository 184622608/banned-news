### 周晓辉：美舰停靠高雄 环时辟谣难掩心虚
------------------------

<div class="wysiwyg">
 近日，中华民国前海军新江舰长吕礼诗在其个人脸书上披露了这样一则消息：美国海军研究办公室排水量为3250吨的科学研究船〝汤姆斯号〞（Thomas G. Thompson（T-AGOR-23）），现正停靠于台湾高雄港9号码头。该舰由美国大学暨国家海洋实验室使用，并具有遥控水下载具及自主式水下载具，可测量盐度、温度、压力和化学性质等水文资料。预计在本月18日晚间11时离开高雄港，前往澳洲弗里曼特尔（Fremantle）港。
 <br/>
 <br/>
 吕礼诗其后在接受采访时表示，虽单纯就〝汤姆斯号〞停泊而言，本身意义并不算特殊，但该舰前往澳洲如需要补给，也可南下至菲律宾，但却在敏感时刻选择在高雄港停泊，〝引发了无限的想像空间〞，其象征意义值得观察。
 <br/>
 <br/>
 之所以会引发想像，吕礼诗分析指，首先，高雄港外海为中华民国国军时常演习的〝反潜第六区〞，平均水深约500公尺，最深约1千公尺左右，为反潜操演适合地点。假如传出的下月美国航舰将在东亚一带活动的消息属实，且有意自台海通过，〝汤姆斯号〞可能在现有资料基础上，进一步确认水下水文。
 <br/>
 <br/>
 其次，该地同时也是1996年台海导弹危机时，中共其中一枚东风15导弹的弹落点。1996年，中共为阻止李登辉连任，于当年在台海附近发射导弹和进行军事演习。当时两艘美国航母靠近台海，在美国的军事压力下，中共采取了〝三不原则〞，即发射导弹不会飞跃台湾本岛上空；海军空军不超越台海中线；即使举行登陆演习，也不会实际去占领台湾的岛屿等。据后来的曝料，彼时中共军队发射的导弹并没有使用实弹的弹头。
 <br/>
 <br/>
 无疑，结合近期美国的一系列强硬行动，吕礼诗认为美舰停靠高雄港的分析不无道理。美国副总统彭斯在10月4日就中美关系的演讲中，对于中共炫耀军力，彭斯除了提到北京在南海人工岛屿建造军事基地并部署先进的反舰和防空导弹外，还特意提到了日前发生在南海的中共军舰挑衅美舰的鲁莽行为，称〝我们不会被吓倒；我们不会退缩〞。
 <br/>
 <br/>
 10月11日，美国白宫国家安全顾问博尔顿在接受美国电台脱口秀主持人休伊特的采访时警告中共不要再冒险，因为按照美国海军的交战规定，指挥官有权保护自己的战舰。博尔顿明确表示，美方不会容忍中共威胁美国军人。美国不承认中共主张的合法性。中共方面需要明白，美国决心让国际海上通道畅通无阻。
 <br/>
 <br/>
 就中共军舰挑衅美舰一事，博尔顿称〝这是中共危险行为的一个例证〞。〝现在，已经有更多盟友加入反对中共的行动中去，除了菲律宾和日本等邻近国家外，英国、澳大利亚等国也同美国一样，继续穿越南海水域在公海航行，而且，这些国家都准备在这方面做得更多。〞
 <br/>
 <br/>
 美国川普政府在亮出自己的强硬态度的同时，也早已开始了行动。除了增加美国国防预算，打造强大的美军外，目前，为了遏制中共在太平洋地区扩展武力，美国空军开始将其远程打击的首要军备资产轮流派遣到太平洋，其中包括B-2轰炸机。据〝国家利益〞近日报导，太平洋空军总部空中和网络空间行动主任威廉姆斯（Stephen Williams）少将在一份声明中说：〝B-2幽灵轰炸机首次部署（在珍珠港-希卡姆联合基地），显示空军在世界任何地方投射炸弹能力的战略灵活性。〞
 <br/>
 <br/>
 B-2隐形轰炸机的特殊交变高频材料隐身涂层，可提高其隐形能力。其发动机也有助于隐形。此外，B-2轰炸机在空中不加油的情况下，航程可超过1.1万公里，并且能以高亚音速飞行。〝幽灵〞轰炸机最多可携带重达4万磅（1磅合0.4536千克）的核武器或常规武器。
 <br/>
 <br/>
 10月15日，美国国防部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蒂斯.html" target="_blank">
  马蒂斯
 </a>
 再次访问越南，巩固防务关系。他多次直言不讳地批评中共在南海的单方面改变现状的行为，指出中共在有争议的岛屿部署反舰导弹、地对空导弹等令该地区迅速军事化。
 <br/>
 <br/>
 由上推之，美国〝汤姆斯号〞停靠在高雄港就未必仅仅是偶然了，只是大家都不说，但心里都明白是怎么回事的。
 <br/>
 <br/>
 好笑的是，对于吕礼诗的曝料和分析，中共环球网于16日马上予以〝驳斥〞，题目是这样的：〝台独〞已经在欢呼了！台军吓得赶紧表态。报导称根据港务公司信息，该船在今年已在高雄港四次停留，并引用台湾国防部的说法：那是科研船，前身是美国海军，但目前由华盛顿大学使用，执行的任务纯属学术研究，与军方毫无关系。
 <br/>
 <br/>
 维基百科资料显示，〝汤姆斯号〞一直隶属于美国海军，其运营方之一是华盛顿大学。舰上有21名军官和船员，2名海洋技术人员，最多36名科学家。其到底与美国海军有否关系，其学术研究是否与军方有无关系，北京军方大概另有结论，而环球网相信台湾国防部的外交说辞不是傻到了极点，就是故意掩盖北京的不作为和虚弱。
 <br/>
 <br/>
 事实上，〝汤姆斯号〞今年连续四次停靠在高雄港，恰恰很能说明问题，说明美国早已为未来可能的冲突做准备，只不过近一个月将强硬态度变得更为直接和公开了。
 <br/>
 <br/>
 去年12月川普在公布的首份国家安全战略报告中，中俄就已被描述为对美国构成威胁的〝修正主义大国〞，批评中俄的言论甚至超过川普以往的表态。今年1月20日，在美国国防部公布的《国防战略报告》中明确提到，中国和俄罗斯这两个〝修正主义〞大国的重新崛起是美国的繁荣和安全的〝核心挑战〞。报告称：〝国家之间的战略竞争——而并非恐怖主义——是现在美国国家安全的首要关注点。〞
 <br/>
 <br/>
 报告指：〝中国是（美国的）战略竞争对手，中国使用掠夺性的经济手段来恐吓邻国，并且在南中国海对该地区的地貌进行军事化。〞〝中共正在利用其军事现代化、影响战和掠夺性经济，胁迫邻国，改变印度太平洋地区的秩序，使之有利于中共。在中共持续其经济及军事力量，通过长期的举国战略取得实力的同时，短期内它将继续推行军事现代化计划，追求在印太地区的霸权，长期则是取代美国取得全球优势。〞
 <br/>
 <br/>
 是以，为了应对这一趋势，美国必须增加国防支出，以使美军更强大、敏捷，以及维持在随时应战状态。那么，〝汤姆斯号〞的学术研究与此无关吗？虽然并不具备战斗力，但隶属于美国海军、且有着海军军官的〝汤姆斯号〞就不是美国军舰了吗？
 <br/>
 <br/>
 很讽刺的是，据台湾《联合报》2017年的报导，美国国会〝2018财政年度国防授权法〞（NDAA）要求评估美台军舰互停的可行性后，中共驻美公使李克新曾在华盛顿告诉美国国会友人：〝美国军舰抵达高雄之日，就是我解放军武力统一台湾之时。〞
 <br/>
 <br/>
 如今，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军舰.html" target="_blank">
  美军舰
 </a>
 已经在今年四次抵达了高雄，北京一直都静悄悄，李克新的铿锵之词也不见了踪影。难道将〝汤姆斯号〞不视为美军舰，就可以掩盖中共的虚张声势吗？环球网的辟谣只能再打了其主子的脸，彰显了其的心虚。
 <br/>
 <br/>
