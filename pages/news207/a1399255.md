### 陈思敏：秦岭山下超级别墅牵涉的权贵们
------------------------

<div class="wysiwyg">
 日前，中共中纪委副书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐令义.html" target="_blank">
  徐令义
 </a>
 因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦岭.html" target="_blank">
  秦岭
 </a>
 问题二度亲赴陕西，也让秦岭北麓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建别墅.html" target="_blank">
  违建别墅
 </a>
 再度成为焦点。
 <br/>
 <br/>
 今年以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦岭.html" target="_blank">
  秦岭
 </a>
 北麓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建别墅.html" target="_blank">
  违建别墅
 </a>
 成为热点话题，是因陕西官媒首度披露了习近平曾经6次批示，不过并没有具体指出时间。但据查，最早一次是2014年，在看到违建材料后，习近平当即批示拆除。几个月后，西安市政府对外宣称，秦岭北麓拆除违建提前完成，202套违建别墅被拆除或充公，并有110人受处分。
 <br/>
 <br/>
 最近两次在今年8月4月和7月，习近平对违建别墅接连作出重要批示，要求彻底查处。而这时隔四年的严厉批示，都是因为在西安市鄠邑区石井镇蔡家坡村，有一栋超级违建别墅，迟迟无人敢查处。
 <br/>
 <br/>
 这栋超级违建别墅，已被官媒正名是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈路别墅.html" target="_blank">
  陈路别墅
 </a>
 〞，惹火话题之一，是有网友感慨：〝我们家的房子，还没陈路别墅里的狗舍大〞。该别墅的狗舍面积达78平方米。
 <br/>
 <br/>
 统计局资料显示，西安房价涨幅全国最高，今年4月份，西安房价同比增长73.69%，西安购房者自嘲：〝房子一直在涨，房价涨了一倍，全款变首付，首付变车库。〞据房仲业最新统计，9月西安房价环比涨幅仍领跑全国，成交价集中在1万元╱平方米左右，有的楼盘可达1.3万元—1.4万元╱平方米之间。而〝陈路别墅〞所在鄠邑区最新房价，平均单价：1.23 万元╱平方米。
 <br/>
 <br/>
 媒体披露合同显示，〝陈路别墅〞占地面积14.11亩（9,409.05平方米），70年总金73.5万元。出租的蔡家坡三组有村民200多人，每位村民一年分了200多元。蔡家坡三组村里耕地人均不到一亩地，〝陈路别墅〞所占地全部为基本农田，越过了红线。
 <br/>
 <br/>
 今年9月底，存在了10年之久的〝陈路别墅〞走到了尽头。然而，从2005年年底动工，2008年8月建成，该别墅层层闯关，绕开国土部门，安然无恙地存在了十年。大大小小媒体，官媒非官媒都纷纷起底报导，并以西安秦岭最豪华别墅的主人身分揭秘〝陈路的父亲是什么人〞，呼之欲出的是陕西省西安市人大常委会原主任、原西安市长陈宝根。
 <br/>
 <br/>
 令人注意的是，〝陈路别墅〞动工的同时，2005年3月西安阎良国家航空园区也正式启动建设。2008年4月，西安阎良国家航空高技术产业基地三年展，迎来了中央领导即时任与前任两位政法委书记周永康、罗干出席站台，陪同人员有时任西安市市长陈宝根。履历显示，周永康与陈宝根交集不浅。陈宝根1995年至2017年的官场生涯中，唯一不在陕西的是2000年至2002年，他在四川担任广元市委副书记、市长，任期与由国土部部长转任四川省委书记的周永康完全贴合。
 <br/>
 <br/>
 此外，90年代后期，国土系统曾为落实江泽民的所谓〝保护天然林〞的指示而开始〝封山〞。而据媒体披露，西安这次清查结果显示：秦岭北麓违建时间跨度大，主要是1997年至2012年长达15年间。也就是早在江泽民所谓山林保护、还林还草而封山的年代，西安秦岭私自开山建别墅的口子同时打开。值得一提的是，林业系统实权人物是江泽民的妹妹江泽慧，她的众多头衔包括西安世园会组委会主席。
 <br/>
 <br/>
 不论如何，整治秦岭别墅由中纪委领衔，如此高规格的〝拆除〞行动，看来西安乃至
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陜西官场.html" target="_blank">
  陜西官场
 </a>
 还有比违建更腐败更严重的问题。
 <br/>
 <br/>
