### 袁斌：61天花104万，巨额住院费折射了什么？
------------------------

<div class="wysiwyg">
 〝4万块1瓶，我病了3年，吃了3年，为了买药，房子没了，家人也拖垮了，谁家还没个病人，你能保证一辈子不生病吗？我不想死，我想活着。〞
 <br/>
 <br/>
 看过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_《我不是药神》.html" target="_blank">
  《我不是药神》
 </a>
 的人都知道，这是这部前一阵火过的电影里的一句经典台词，曾引发了成千上百观众的强烈共鸣。
 <br/>
 <br/>
 近日，一张出自武汉大学中南医院的61天花104万的巨额住院收据刷爆了朋友圈，简直可以说就是这句台词的现实版！
 <br/>
 <br/>
 据大陆《楚天都市报》报导， 9月13日上午，67岁的刘先生（化名）在家突然出现呕吐等不适症状，于当天中午12时左右在老伴陪同下来到中南医院，确诊为急性心肌梗塞、右侧冠状动脉严重堵塞。
 <br/>
 <br/>
 由于出现多种并发症，刘先生接受心血管内科、心血管外科、骨科、消化内科、重症医学科等多学科治疗，还邀请省内相关专业的顶级专家进行会诊，病情趋于好转，但仍在加护病房。
 <br/>
 <br/>
 医院目前共开出了两张住院收据，其中第一张住院收据显示，住院日期为2018年9月13日至9月27日，共计14天，金额为632,439.19元；第二张住院收据显示，住院日期为2018年9月27日至11月27日，共61天，金额为1,040,139.71元。
 <br/>
 <br/>
 〝104万其实只是部分费用，刘先生从入院至今总治疗费用已超过170万元。〞中南医院ICU副主任周医生介绍，由于刘先生病情危重，治疗过程中使用了人工心肺仪等大量先进设备，刘先生使用人工心肺仪长达9天，每2小时还要进行一次血气功能、凝血功能监测，仅此费用每天就得2万元左右。
 <br/>
 <br/>
 11月27日，刘先生的女儿把104万元的住院收据发到微信朋友圈。她说，因为是普通受薪阶级，借了不少外债付医疗费，压力非常大。此外，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_医保.html" target="_blank">
  医保
 </a>
 虽可报销一部分，但是具体多少还不确定。
 <br/>
 <br/>
 当这张天价收据单出现在朋友圈的时候，所有看到的人一时间都炸了。网民议论纷纷：〝没钱的话，大病自我了断。〞〝看来普通老百姓只能回家等死了！〞
 <br/>
 <br/>
 今年是中共改革开放四十周年，大大小小的党媒又在奉旨吹嘘所谓〝中国奇迹〞〝中国崛起〞的辉煌成就。没错，如今的中国已然是全球第二大经济体，政府的行政成本雄踞世界第一，为了自己的面子动辄就是几十亿、上百亿在世界各地大撒币，权贵阶层更是挥金如土，论阔绰可以说没一个国家能与之相比。可底层百姓过的是什么日子？是〝买不起房，养不起娃，生不起病〞的苦日子。一张61天104万的天价住院帐单就是这种日子的生动缩影。
 <br/>
 <br/>
 我记得《我不是药神》里还有两句经典台词是这么说的：〝世界上最遥远的距离不是生与死，而是药就在那里，我却买不起！〞〝我卖药这么多年，发现世界上只有一种病：‘穷病’。〞我想再补充一句的是，之所以还有那么多中国百姓至今为〝穷病〞所困，〝买不起房，养不起娃，生不起病〞，不是因为别的，就是因为中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一党专政.html" target="_blank">
  一党专政
 </a>
 。中共凭藉着自己垄断的权力和资源，穷凶极恶的榨取中国人民创造的财富，使得他们至今仍生活在穷困之中。只有除去这个恶党，中国人民才会翻身，真正过上富足的日子！
 <br/>
 <br/>

 <br/>
