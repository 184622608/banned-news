### 夏小强：中共正在重演前苏联解体大戏
------------------------

<div class="wysiwyg">
 11月18日，2018年APEC领袖峰会在没有发表领袖宣言的情况下落幕，这是峰会有史以来第一次〝无宣言闭幕〞。
 <br/>
 <br/>
 中美之间的强烈对抗，导致了此次峰会的〝无宣言〞。习近平在峰会期间批美国的〝美国优先〞政策，称这是保护主义行动；美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 随后则抨击中共对外大举贷款令其它国家陷入债务陷阱，并警告会继续对中国加征关税。
 <br/>
 <br/>
 美国副总统彭斯在亚太经合组织峰会上，对中共再次表示强硬姿态。此前，彭斯在接受《华盛顿邮报》采访时说，如果中共不做出具体的重大让步，美国准备在经济、外交和政治上加强对该政权施压。彭斯还首次使用了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冷战.html" target="_blank">
  冷战
 </a>
 〞这个词，使人联想到前苏联在与美国陷入冷战后最终崩溃的结局，警告震慑中共意味浓厚。
 <br/>
 <br/>
 对于彭斯的言论，主流媒体和外界评论，更多把其视作美国与中共进行贸易谈判的策略，分析评论，大多停留在中美经济和贸易争端层面。
 <br/>
 <br/>
 其实，彭斯的对中共的公开言论，包括此前在哈德逊研究所的演讲，都是代表川普政府对中共也是对世界发出的战略宣言。其战略意义，已经远远超出贸易层面。
 <br/>
 <br/>
 川普就任美国总统之后，在外交上，对中共的战略已经与此前完全转向。在经济、外交、军事和意识形态等全方位，遏制中共的大战略正在全面展开。而中美贸易战，只是川普政府全面遏制中共，在经济层面的表现。
 <br/>
 <br/>
 中美贸易战的实质，是中共代表的共产主义意识形态和其中共邪恶政权，与美国为代表的普世价值和自由宪政法治，在21世纪，展开的终极对决。
 <br/>
 <br/>
 上个世纪九十年初，随着苏联解体，维持了几十年的冷战结束。以美国为代表的西方世界误以为世界最大的危机已经结束。但是，时间发展到今天，全世界蓦然回首才发现，中共政权正在从政治、经济、军事、文化等全方位面渗透自由世界，中共才是世界最大的威胁。
 <br/>
 <br/>
 以此观之，上个世纪苏联这个共产主义庞然大物的突然解体，或许是中共这个世界上最后一个最大的共产主义政权解体的彩排和预演。
 <br/>
 <br/>
 如果存在着一只历史之手的话，那么安排终结世界上最后一个最大共产政权的推手，就是美国。
 <br/>
 <br/>
 自从美国诞生之后，随着上个世纪第一次世界大战之后，美国超越英国成为世界最强大国家。美国担负起了维持世界秩序和塑造世界格局的职责。
 <br/>
 <br/>
 自第一次世界大战以来，一个不可否认的现象和事实是，在所有美国参与的重大战争和事件中，凡是以美国为敌的国家和对象，最后都以失败告终。
 <br/>
 <br/>
 第一次世界大战中，德意志帝国、奥斯曼帝国组成的同盟国，对阵法国、英国等组成的协约国，在1917年4月美国参战之前，协约国节节败退，面临失败，正是美国的参战扭转了战局。德国最终以军事胜利而战事失败而告终，被迫吞下《凡尔赛条约》的苦果，埋下了二战的种子。
 <br/>
 <br/>
 第二次世界大战，德国、意大利和日本组成轴心国，对阵英、法、苏等组成的同盟国，也正是美国在1941年12月正式参战，扭转了二次大战的战局，最终轴心国失败，美国的参战，重新塑造了二战之后的世界格局和制定了国际秩序，一直持续发展到今天。
 <br/>
 <br/>
 二战之后，苏联以美国为敌，经过几十年的冷战，美国最终通过〝星球大战计划〞和对苏联的全面遏制，造成苏联经济面临崩溃，苏联政权解体。
 <br/>
 <br/>
 如今，中共政权在政治、经济、外交、军事等全方位以美国为敌，与美国呈对抗状态；川普政府也开始对中共采取全方位遏制措施，新冷战格局开启。
 <br/>
 <br/>
 历史常有惊人的相似，中共政权，如今正在重蹈苏联解体的覆辙。一场中共解体的大戏，正在上演。
 <br/>
 <br/>
