### 陈思敏：A股暴涨暴跌的黑手
------------------------

<div class="wysiwyg">
 10月22日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_A股.html" target="_blank">
  A股
 </a>
 沪指一天爆涨百点。但一日行情。23日，A股三大股指全线走低，沪指重新跌破2,600点。
 <br/>
 <br/>
 沪指22日暴涨，不过又一次上演政策市，没有实质经济好转，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_A股.html" target="_blank">
  A股
 </a>
 只能靠高层喊话硬撑，并下令〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家队.html" target="_blank">
  国家队
 </a>
 〞配合进场护盘。已披露的数据显示，Ａ股第二季，随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 增温，沪指在6月跌势加剧，〝国家队〞进场砸1,160亿人民币护盘。
 <br/>
 <br/>
 而〝国家队〞每次逢低进场，拉抬沪指的同时，也成功抄底，日后再逢高抛售。所以〝国家队〞砸千亿护盘只是小手笔，又是稳赚不赔的买卖。
 <br/>
 <br/>
 据高盛数据，在今年第一季沪指平均约3,400点时，〝国家队〞卖出前期持仓Ａ股、套现约710亿人民币。
 <br/>
 <br/>
 据〝国家队〞之一、被指〝炫富〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_社保基金.html" target="_blank">
  社保基金
 </a>
 2017年度报告披露，其2017年投资收益率达到9.68％，远超2016年的1.73％，投资收益金额接近2,000亿元人民币。社保基金自2000年8月成立以来，累计投资收益额已突破1万亿元。这也显示，经济越不好，社保基金投资却赚越多，因此备受外界质疑是否有内幕消息。
 <br/>
 <br/>
 今年5月，社保基金然特别公开〝辟谣〞，之所以能把握股市的涨跌，强调绝非基于内幕消息。但衡诸过去股市大案，没有〝股神〞，只有内幕消息。除了内幕消息，社保基金还被质疑，证监会IPO不断，新股发行网下有相当比例配售给社保基金。
 <br/>
 <br/>
 此外，〝国家队〞重仓的股票不乏〝关系户〞。如目前已发布的三季报中，持仓超过10亿元的有6只个股，其中市值最高的一家上市公司，被指是一名退休地方大员的〝钱袋子〞。
 <br/>
 <br/>
 大家都知道，A股股民自称是韭菜，并称中国股市不如赌场，而其中最大、最本质的问题就是中共党管股市的问题。
 <br/>
 <br/>
 去年2月，证监会主席刘士余发表谈话强调不允许股市大涨大跌。今年5月，上交所第8次会员大会决议：坚决防止股市大起大落、暴涨暴跌。到底谁让A股大起大落、暴涨暴跌？
 <br/>
 <br/>
 今年9月初，股灾底2,638点恐失守之际，金融委办公室等几个重磅金融会议召开，四大行高层和两大交易所一把手等，齐声确立了沪指2,700点政策底，〝国家队〞配合进场，官媒喊话市场投资人〝坚持持股〞。但沪指却很快坚持不住，10月11日跌破了2,600点。
 <br/>
 <br/>
 10月19日，在每况愈下的三季度GDP公布后，财经高层集体出镜喊话，〝国家队〞配合进场净买超，是日A股暴涨4%，沪指收复2,600点，同时又套牢一波散户。
 <br/>
 <br/>
 2017年基金年报显示，5只〝国家队〞基金全部盈利，合计利润近175亿元。但任何一个股市都不会凭空创造财富，而是财富的转移。A股也一样，中共高层发话，社保、〝国家队〞打着〝救市〞名义，每一次进场低吸（散户手上股票），预备着下一次高抛（散户接盘）。〝国家队〞的获利，其实是股民的亏损。而〝国家队〞反覆进场出场，一波波散户被套牢，同时造成股市暴涨暴跌。可以说，A股指数每一点都是无数股民血本堆砌的。
 <br/>
 <br/>

 <br/>
