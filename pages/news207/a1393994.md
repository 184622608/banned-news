### 杨宁：范冰冰偷税案大事化小有说道
------------------------

<div class="wysiwyg">
 因涉及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_〝阴阳合同案〞.html" target="_blank">
  〝阴阳合同案〞
 </a>
 消失了三个多月的中国大陆明星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 ，终于有了最终的说法。10月3日，北京官媒发布消息，证实了范冰冰确实存在〝阴阳合同〞，即偷税漏税的行为，她也因此被罚款、补缴税款等总计超过8.8亿元。
 <br/>
 <br/>
 值得注意的是，官媒称，根据刑法规定，〝由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 ，属于首次被税务机关，按偷税予以行政处罚，而且此前未因逃避缴纳税款受过刑事处罚，上述定性为偷税的税款、滞纳金、罚款，在税务机关下达追缴通知后，在规定期限内缴纳的，不予追究刑事责任。〞不过，范冰冰的经纪人牟某广，则因指示公司员工隐匿并销毁会计证据，阻挠税务机关调查，涉嫌犯罪，被采取强制措施。
 <br/>
 <br/>
 无疑，官媒的消息就是在昭告天下，沸沸扬扬的范冰冰偷税案已经大事化小，以〝破财免灾〞方式落下帷幕。至于免于刑事处罚的范冰冰未来或在江湖成为过去时。
 <br/>
 <br/>
 这样的结局也并不太出人意料。大陆影视圈类似范冰冰这样的〝阴阳合同〞绝非个案，背后涉及的如洗钱等黑幕也绝非外界所能知晓，当局拿范冰冰开刀，为的就是杀一儆百，同时在内外交困、经济出现大问题的大背景下，增加国库收入。在范冰冰出事后，大陆各位明星、工作室已相继补缴税款，当局的目的就此达到。在目的达到后，范冰冰被〝轻轻放下〞，以此安抚众星，也就顺理成章了。当局的手段自然也让影视圈噤若寒蝉。
 <br/>
 <br/>
 事实上，如果比照当年轰动一时、风头远超范冰冰的大陆另一位影星刘晓庆〝偷税案〞，就可知范冰冰实在是〝幸运得很〞，至少被失踪的日子算不上长，也没有被关押进监狱。
 <br/>
 <br/>
 据海外媒体报导，1993年国税局开始注意刘晓庆，查她的帐，都到了掘地三尺的地步，但也没查出毛病。贴身跟踪她的人曾问：〝你到底得罪了哪个大人物？〞
 <br/>
 <br/>
 2002年6月至2003年8月，刘晓庆因涉嫌偷税被羁押在监狱422天，吃了不少苦头。彼时新华网称1996年刘晓庆出现了偷漏税问题。然而，93年查时没偷税，96年却敢让自己的帐出了毛病？据悉，真实的原因是刘晓庆得罪了时任中共最高党魁的江泽民，江派人马暗中派人潜入刘晓庆的公司做了手脚。最终刘晓庆被罚8000多万，其19套房产也全部被低于市场三分之一的价格抛售。江让办案人员直接告诉刘晓庆，只给她留一条命，条件还是闭嘴。
 <br/>
 <br/>
 从范冰冰同日发表的〝致歉信〞看，其在〝深感羞愧〞、配合官方演好戏的同时，也应被下达了封口令，即不会将此次被调查的内情公诸于众。
 <br/>
 <br/>
 范冰冰偷税案后会涉及什么内情呢？香港亲北京的《明报》9月24日曾披露，范冰冰确实被当局扣押，有关方面希望从其处打开缺口，意在剑指〝其他人〞。而总部在北京的多维网也称，范冰冰被指〝性贿赂领导人〞。至于这个〝领导人〞是谁语焉不详。但有分析指，范冰冰事件背后的目标是影视圈大佬董平和江泽民大总管曾庆红的弟弟曾庆淮。笔者也曾撰文分析了范冰冰与华谊兄弟、董平、曾庆淮间的关联。
 <br/>
 <br/>
 然而，几日后，范冰冰案却以罚款的形式落幕，背后的〝其他人〞和〝领导人〞都再次隐身。这意味着范冰冰案点到为止，不再涉及其背后各色人物。因为动曾庆红就意味着动江泽民，在北京高调纪念江的心腹黄菊冥诞、再次释放与江派妥协的信号后，藉由范冰冰牵出背后大人物，已不再可能。显然，范冰冰案的猝然落幕，又一次让外界看到了中南海高层在走向怎样危险的道路。#
 <br/>
 <br/>
