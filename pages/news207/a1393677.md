### 天佑：购买伊朗石油的中共官员们要小心了
------------------------

<div class="wysiwyg">
 作为中共军委装备部部长的李尚福中将因为代表中共购买俄罗斯苏-35战机及S-400地空导弹，违反了美国《反制美国敌人制裁法》，而在9月20日与中共军委装备部一起被美国政府列入制裁名单。它们今后不能透过美国的金融系统进行任何交易，美国公司不能与该部有任何商业来往，冻结李尚福及其亲戚在美国的财产，不允许李尚福入境美国。
 <br/>
 <br/>
 这类事情可能很快就会波及到你们了。你们都是业内人士，都肯定知道美国政府将在11月4 日重启制裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗石油.html" target="_blank">
  伊朗石油
 </a>
 出口，要求全世界都停止进口伊朗石油，违者将予以制裁。而中共外交部门却始终扬言不会遵守美国制裁计划，并将持续增购伊朗石油。作为伊朗原油的头号大客户的中共，进口了伊朗四分之一的石油。美国政府是绝不会容忍中共的这种行为的，否则，美国对伊朗的石油制裁就是空话。川普总统是说空话的人吗？美国国务院伊朗问题特别代表8月16日就明确的说了，在11月份之后，川普政府准备制裁所有购买伊朗石油的政府，包括中共。
 <br/>
 <br/>
 美国政府会因此制裁中共的石油公司。但你们替中共卖命，为中共在伊朗购买石油，直接破坏美国打击国际邪恶势力的努力，美国政府同样也会像制裁李尚福一样，连带制裁你们的，甚至联合其它发达国家共同制裁你们。你们中的一些人可能会成为第二个、第三个、……第N个李尚福。你们大半生努力的心血，存在海外的部分将被冻结，甚至付之东流，而因制裁造成的在海外财产的曝光，可能还会引来中共的追究，导致牢狱之灾。
 <br/>
 <br/>
 为中共卖命对抗人类正义，美国会制裁你们。身在中共衙门，不替中共卖命，中共会处理你，你这个替罪羊想不当都不行，除非你离开大陆，移民海外。所以能解救的方法不是很多，但不是没有，比如可以藉助外力保护自己。就是通过网上曝光等方式，将自己掌握的中共和伊朗的交易的内幕、计划提前曝光，使中共的罪恶提前暴露在光天化日之下，受到世界各国的谴责，而使罪恶被迫流产，你不就不用去执行这个计划了吗。退一步说既使曝光了中共的罪恶也未能阻止罪恶的发生，你还是被迫去执行了，你也不必担心被制裁，因为三尺头上有神灵。人在做神在看，你做了什么神都知道。你曝光中共的罪恶是做了正义的事，上帝会保佑你的。当然还有其它方法可以保护自己，总之就是你们现在该为自己的身家性命着想了，该为自己在海外的亲属、儿女着想了，该为自己的巨额海外资产的安危着想了。因为形势变了，今天的世界各国已在联合起来，共同铲除危害人类的中共和共产主义了。
 <br/>
 <br/>
 其实，所有的中共的党政军警的各级官员和国企的掌门人们，都应该未雨绸缪了。因为你们都是为中共祸害人类卖命冲在第一线的人。你们中的大多数人也都和李尚福一样，有亲属、子女在海外，有巨额资产在海外，都有可能因为为中共卖命而连带被国际社会制裁。所以及时利用各种方式，将中共的罪恶曝光出来，使计划破产，使罪恶受到谴责，从而使自己免于落得个因成为中共罪恶的执行者而被国际社会制裁和因被制裁曝光了自己的秘密又被中共处理的下场。同时也会因为曝光中共的罪恶，使更多的人免于中共罪恶的伤害，而受到上帝的保佑。
