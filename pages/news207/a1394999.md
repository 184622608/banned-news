### 古玉文：孟宏伟自救预案的成与败
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 作为中共高级官员落马，本不应该引外界如此沸腾，目前之所以成为沸点，并引外界持续关注与推测，归因于两个引爆因素，一是因为其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席身份，二是其年轻妻子〝女王立军〞式自救行动。
 <br/>
 <br/>
 外界分析，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 妻子向法国警方报案及后来的里昂记者招待会及接受CNN记者采访，均是孟宏伟在出事前与妻子商量的自救预案。
 <br/>
 <br/>
 孟宏伟作为中共公安部第三号人物，具备较高的政治敏感力和反侦查能力，加上2016年被中共推上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席的要职以后，应掌握国际警界情况。
 <br/>
 <br/>
 2012年王立军闯美使馆成功使自己避免被薄熙来做掉，可以说是开创了中共体制内高官在紧急状态下跑路逃命的一个国际丑闻兼新模式。而亲历了甚或参与了中共内部许多血雨腥风场面的孟宏伟，如果他敏感到自己将在不期之日出事的话，又身在国外，与家人商量好紧急预案，这是很有可能性的。
 <br/>
 <br/>
 从目前已发生的事实来看，这个自救预案起到了一定的作用，将中共丑陋的〝家事〞一夜间外扬，孟宏伟妻孩暂时安全，同时其手中握有的外界传闻为〝核武级炸弹〞使其与中共之间暂时有了讨价还价的相对筹码。
 <br/>
 <br/>
 这可以说是孟宏伟夫妻自救预案的成功之处。但是这个预案最终结果如何，是否能使孟宏伟安全着陆或减轻罪责，从中共对孟宏伟事件的通报来看，并不乐观。
 <br/>
 <br/>
 为什么呢？这要从孟宏伟本人说起。
 <br/>
 <br/>
 有海外自媒体爆料，中共监察部在公布孟宏伟履历的时候，刻意隐匿了一些环节。孟宏伟为北京大学法学专业毕业，后在中国警官大学经过警事培训，为中共改革开放后的干部培养第三梯队，是技术型体制内官僚。
 <br/>
 <br/>
 孟宏伟长期在中共公安部一局工作。中共公安部一局为政治保卫局，现在叫国内安全保卫局。主要负责监控镇压异议人士、民主人士、宗教敏感人士，对中共体制不满的社会人群等，说白了，就是通常说的国保局。实质是公安部内的党卫军。权力很大，身份隐秘，手法残酷是这类人的工作风格与特点。在中共掌管的公安体制内，他们应该是掌管中共政治秘密和政治隐情最多的部门，也是政治敏感性与嗅觉最灵敏的一批人，同时也是中共体制内的〝犯罪高手〞，秘密镇压了无数正义人士与宗教人士。
 <br/>
 <br/>
 海外自媒体爆料，孟宏伟作为中共培养的梯队干部，曾经在西藏自治区工作过。上个世界五十年代，中共为达到对十世班禅的统战目的，曾经派习仲勋做班禅的工作，习仲勋为人比较开明，属于中共体制内较为民主和开明的高级官员，与班禅很能谈得来。八十年代初，班禅因在北京不满中共搞的反对资产阶级自由化运动等各种政治运动，在习仲勋的帮助下从北京被软禁的状态回到了西藏。
 <br/>
 <br/>
 1962年，十世班禅在了解到中共对西藏污名化宣传和对藏民的打压之后，向中共提交了《关于西藏总的情况和具体情况以及西藏为主的藏族各地区的甘苦和今后希望要求的报告》，简称《七万言书》，通过列举大量事实，说明了在中共在〝平叛〞、改革中出现错误和问题，十分严重，强调：若不认真加以纠正，藏族将面临灭族灭教的严重危险。
 <br/>
 <br/>
 毛泽东相当震怒，将班禅定性为〝反动派的疯狂反扑〞，并软禁其两年。文革中班禅惨遭批斗，并被关押在秦城监狱长达十年，那时他年仅28岁。出狱后，1978年6月，经邓小平批准，十世班禅被迫结婚。是历届班禅中唯一结婚的。
 <br/>
 <br/>
 被统战后的班禅实际上心中对中共的政策一直另有看法，并在多种场合发表了批评中共的言论，并对流亡海外的达赖喇嘛表露出同情。这是中共无法容忍的。
 <br/>
 <br/>
 1989年1月28日，藏传佛教领袖、第十世班禅额尔德尼‧确吉坚赞在西藏日喀则市的新宫猝然离世，终年51岁，中共官方的说法是〝心脏病〞。 海外流亡作家北京大学的袁红冰的调查，十世班禅是被中共毒杀的。
 <br/>
 <br/>
 袁红冰表示，经过多年的查访，包括采访中共太子党的相关成员，下令将其毒杀的决策出自邓小平、李先念和薄一波。
 <br/>
 <br/>
 海外自媒体爆料，当年具体执行毒杀班禅的操刀手正是日前被中共宣布违法失踪的国际刑警组织主席，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共公安部副部长.html" target="_blank">
  中共公安部副部长
 </a>
 孟宏伟。孟宏伟从西藏回北京后受到重用，官至公安部助理。后任交通局局长，2004年任公安部副部长，参与过迫害法轮功信仰。
 <br/>
 <br/>
 孟宏伟2018年9月25日被失踪，10月初其妻向法国警察报警后，国内外推测孟宏伟作为总部设在法国的国际刑警组织主席，可能受命于中共，参与了灭口海航王健行动，为中共洗地。还有一种说法是其参与了对中共最高当局的政变。
 <br/>
 <br/>
 无论是哪种猜测，包括中共官方宣布的孟宏伟是因贪污被抓捕，孟宏伟作为中共体制刻意培养起来的，为中共体制效命的高级官员，曾为中共迫害过无数正义及宗教人士，是中共体制内的高级爪牙，其一生的行径并不是像他的那个年轻的香港籍妻子所描述的那么正义。
 <br/>
 <br/>
 其实，中共体制内真正实行自救和救人的行动楷模，不应是〝王立军〞这个中共死心塌地的刽子手，而是原中共驻澳洲大使馆一秘陈用林先生、天津国保局警察郝凤军先生和沈阳司法局局长韩广生先生等英雄。
 <br/>
 <br/>
 陈用林先生在中共驻澳洲大使馆一秘工作期间，开始受中共蒙蔽监控在澳洲的法轮功学员与异议人士，当他自己真正接触到了法轮功学员后，发现他们完全不是向中共诽谤的那样，而是一群有着自己善良的信仰与高尚为人准则的人。因羞于与中共为伍，2005年，陈用林先生带着夫人年幼的孩子叛逃中共使馆，在澳洲公开揭露中共迫害法轮功及异议人士，并向全世界揭露中共在澳洲有一千多个间谍。
 <br/>
 <br/>
 当时的情况非常的紧急，他在街头含泪演讲，揭露中共，人群中就有中共的间谍，他们一家三口随时都有被中共灭口之灾，但是，公开的揭露与追求正义他和家人得到了西方国家的支持，得到了全世界的声援，他们最终得到了澳洲政府的庇护。
 <br/>
 <br/>
 天津国保局警察郝凤军先生和沈阳司法局局长韩广生先生等也都是在认清了中共的邪恶之后冒着生命的危险依然选则了投向正义与光明。而这些英雄最终的结果也是令人鼓舞的，他们都已经成为西方自由社会中的一员。
 <br/>
 <br/>
 而相反的案例，王立军、令完成等中共体制内官员及家属，如果只是着眼于中共权斗，而利用西方社会的自由与舆论，也许能暂缓中共的迫害，但却难逃中共无处不在的威胁与消音。
 <br/>
 <br/>
 孟宏伟的妻子在接受记者招待会声称是因为不想被中共迫害其家人而采取了不正面面对镜头的做法。这很可能是她自己的一种说辞，实际情况更可能是她选择自救方案中的一个必要环节，因为她一定知道中共掌握了她的很多信息及肖像，很可能是她不想过多的刺激中共，或许是为同中共下一步的讨价还价做出预设。也有外界分析，在中共大外宣和黄金蓝计划无处不在的法国，选择媒体爆料都不是简单的事，单凭孟妻子本人是无法做到的，背后势必有力量策划与支持。
 <br/>
 <br/>
 但无论这种力量和孟妻子本人下一步将采取何种自救范式，如果孟妻真象自己表述的〝追求真相、正义和社会责任〞那样，应效仿陈用林先生，真正的站出来揭露中共，以孟宏伟家属的身份现身说法，公开孟本人所犯的罪行，并替其表示忏悔，表态改过，才能真正的摆脱中共的威胁，才能自救救人。
 <br/>
 <br/>
