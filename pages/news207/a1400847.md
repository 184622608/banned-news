### 吴惠林：美中博奕下台湾应如何抉择？
------------------------

<div class="wysiwyg">
 《美、中开战的起点》（Crouching Tiger :What China’s Militarism Means for the World）是一本〝地缘政治的推理故事书〞，是关于中华人民共和国在所谓〝和平崛起〞中，所隐含对美国、对全球和平与安全产生巨大威胁的推理故事。美、中两国是否会开战？
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 将扮演什么样的角色？ 是否会成为交易筹码、被出卖……等等，剧情扑朔迷离、引人入胜。虽然是推理故事、是预言书，但我们不但不能轻忽它、不要认为荒诞无稽危言耸听，反而要非常严肃的读它，尤其是台湾人民，要将该故事当成正经事，因为说故事者是彼得‧
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 （Peter Navarro）。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 当选美国总统后一夕爆红，他被川普任命为〝新设的〞〝白宫国家贸易委员会〞（National Trade Council）主任，是川普的〝最核心〞幕僚，直接研拟制定美国政经政策，应是川普最倚重的左右手。在此有必要说一说川普为何会那么看重纳瓦罗，由纳瓦罗的好朋友，自称〝天安门广场示威者〞的唐柏桥，二○一六年十二月十二日发表于《看中国》的〈一本改变历史的书〉这篇文章就可清楚明了。该文说出了〝川普为什么会如此欣赏纳瓦罗？〞的缘由。
 <br/>
 <br/>
 <h4>
  纳瓦罗是川普的左右手
 </h4>
 <br/>
 纳瓦罗在二○○六年出版了《中国战争即将到来》（The Coming China Wars）这本书，该书警告西方社会，中国（中共）是未来文明世界的主要敌人，而中共的强大最终将给世界带来毁灭性的灾难。纳瓦罗在书中列出了大量的数据和论据来支持该观点，具有极强的说服力。不过，纳瓦罗因为这本书，从此被美国政界和学术界视为极端反共的鹰派学者，在美国的中国问题专家大多数都逐渐转向拥抱中共时，纳瓦罗就成了美国中国问题专家中的极少数所谓的〝激进派〞，甚至一度遭到孤立。就在这个时候，一位慧眼识英雄的伯乐出现了，他就是川普。
 <br/>
 <br/>
 川普非常喜欢那本书，并且在一次接受《洛杉矶时报》采访时，郑重向读者推荐了该书。川普当时列出了二十本他读过的书，其中一本就是《中国战争即将到来》。这二十本书有一半以上是批评中共侵犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权.html" target="_blank">
  人权
 </a>
 和揭露中国黑幕的书。也就是说，川普早就对中共的胡作非为清楚明白，且深恶痛绝。所以说川普一直反覆强调要改变中国，迫使中共遵守国际文明秩序和贸易规则，并非一时心血来潮，也不是像很多华盛顿的老政客一样为了骗取选票。事实证明，川普当选总统后不但没有改变批评中共的态度，反而更加尖锐。这也可以从他打破过去三十多年来美国总统不跟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 领导人直接通话的惯例，与蔡英文总统通话并称呼她〝台湾总统〞，明显看出端倪。
 <br/>
 <br/>
 《中国战争即将到来》出版五年后的二○一一年，纳瓦罗和安一鸣（Greg Autry）合写了《致命中国》（Death by China）这本书，非常详尽的披露了中共如何侵犯人权、利用监狱犯人充当奴工大量生产有毒物品，以及大肆破坏生态环境、向世界输出有毒产品和诈骗、操纵汇率、进行疯狂的军事扩张、建设强大的军警特务、网路警察骇客、封杀言论自由、肆意扰乱国际秩序等，为世人揭示了中共统治下的中国的真实面貌，并提出了解决问题的方案。这本书出版后很快的引起美国社会的广泛关注。川普对这本书更给了极高的评价，并向他的朋友大力推荐。美国一位钢铁大王慨捐一百万美元，资助纳瓦罗将该书内容拍成纪录片。该片在全美公映后，造成了极大的轰动。川普并为该纪录片写了推荐语：〝《致命中国》说得很对。这部重要的纪录片用充分的事实、数据和洞察力，描述了我们与中国之间存在的问题。我强烈推荐大家观看。〞此后，纳瓦罗和川普就成了朋友，并经常就中国问题交换意见。二○一五年川普宣布参选美国总统后不久，就任命纳瓦罗作为他的经济顾问团成员。顾问团其他成员都是商界巨头或名人，纳瓦罗是唯一的专家学者。可想而知，川普对纳瓦罗有多倚重。
 <br/>
 <br/>
 由川普一直以来对中国（中共）的呛声都脱不开《致命中国》的内容，更可证明川普对纳瓦罗的倚重和信任，而纳瓦罗在《中国战争即将到来》和《致命中国》两本书之后，二○一五年又出版了《美、中开战的起点》这一本书，可以说是〝第三部曲〞，也是〝最终曲〞。本书描述成为全球第二大经济体的中华人民共和国，在兵强马壮、军事力量强大之后，会不会趁着美国走下坡及世人都被中共威逼利诱并屈服于中共糖衣毒药下发动战争，作了合理的推测。纳瓦罗并非关起门来天马行空的幻想，而是发挥学者本色，由理论、大量数据、文献，以及访查专家学者，并由历史的演变及其教训和启示进行探索。
 <br/>
 <br/>
 
 <h4>
  三部曲的〝最终曲〞
 </h4>
 <br/>
 全书分成六部，第一部〝谁在打坏主意？〞涵括五章评估军事冲突的可能性；第二部〝中国的威胁〞以十四章的篇幅仔细评估中国的实力；第三部〝谁来引爆导火线？〞共有十章描述十大引爆点，而台湾就被放在第一位，台湾人民特别要注意看；第四部〝战场在哪里？〞涵括五章探究战场；第五部〝妥协可以换来和平吗？〞以六章一一否定六种和平方案的可能性；第六部〝只有实力能带来和平〞，以五章评述〝唯有实力才可能有和平〞，既正确又务实，而川普〝让美国伟大〞、〝美国优先〞、〝制造业回流〞、〝大增军费〞、〝呛主流媒体〞等等动作，可说都是这一部内容的实践。
 <br/>
 <br/>
 的确，这本书提出了关键的问题：国际社会将如何面对北京政府彻底破坏全球经济最繁荣之处的和平与稳定？ 正如章家敦所言，日趋大胆的红色中国（中共），透过崛起的经济和军事实力，占领土地、违反贸易规则、加速研发核子武器、支持流氓政权、对自由社会进行网路攻击、对规范视若无睹，并且瓦解国际组织。为何北京会丢掉〝和平崛起〞的假象？一来中共领导人相信他们是本世纪的主宰，衰退的美国无力对抗；二来中共面临内部前所未有的挑战，无法仰赖经济成长，只好采取民族主义对外出击。
 <br/>
 <br/>
 当纳瓦罗这本书面世的时候，作为世界警察的美国，并未挺身迎向前所未有的挑战，不论是白宫或国会或美国企业，其领导人都不想面对中国（中共）即使在各方努力下仍然不愿融入国际社会的事实，甚至助纣为虐，为了短暂的私利，不但俯首听命于中共，还帮忙中国（中共）在军事、经济、太空等方面的壮大；本书最后一章就赤裸裸描绘这种景象。纳瓦罗于是引用〝团结则存，分裂则亡〞这个老格言告诫美国人〝击败敌人，而敌人就是我们自己〞，他也特别指出奥巴马政府〝重回亚洲〞政策其实是虚晃一招，不但激怒中共，更让亚洲盟国觉得被耍、受骗，让美国失去国际公信力。
 <br/>
 <br/>
 
 <h4>
  以实力追求和平
 </h4>
 <br/>
 这本书的最大特色是每一章都以〝问题〞开章，并列出数个答案供读者选择，让读者直接参与成为〝侦探〞，而纳瓦罗随即不急不徐的铺陈各个答案的可能性。就第二十章〝台湾：永不沉没的航空母舰〞来说，问题是〝以下哪个因素最有可能让中国和美国因为台湾引发战争？〞纳瓦罗提供了〝国族主义〞、〝地缘政治〞、〝意识形态〞和〝道德规范〞四个因素。那么，作为〝美国、中国开战的最可能导火线〞的〝台湾〞，这四个因素是不是都会造成影响？ 而影响力大小如何排列呢？ 除了这四项以外，还有别的因素吗？ 台湾的命运究竟会如何呢？ 就让我们由这本书来寻求答案吧！
 <br/>
 <br/>
 值得一再强调的是：作者纳瓦罗提出的〝只有实力能带来和平〞，的确是诤言，也是最好的方案。〝团结内部、提升国力〞既适合美国，更适合台湾呢！ 台湾同胞们，不要再争争吵吵，斗来斗去，赶紧认清现实，团结一致，分工合作认真打拚，充实实力；自己的前途自己救，把台湾的命运掌握在自己的手中吧！
 <br/>
 <br/>
 <h4>
  川普采信纳瓦罗
 </h4>
 <br/>
 很巧的是，当纳瓦罗这本书问世时，前美国财政部长亨利．鲍尔森(Henry M. Paulson)的《与中国打交道》(Dealing with China)这本书也同时出版，该书号称是鲍尔森的二十年内幕观察，被说是〝各界人士与中国经商交流必读之经典文本和权威注解〞，也可说是〝美国的主流看法〞，抱持〝必需帮助中国持续走向改革之路，要抓住机会帮助中国应对它面临的诸多问题和挑战，不要将中国视为敌人〞的立场，而这也正是纳瓦罗所抨击的〝助纣为虐〞论点。川普任用纳瓦罗，应表示认同他的看法，不过，川普主政后，虽一开始即对中共呛声，但川习会后川普赞许习近平，而在执政百日后，一连串的改口，其政策的摇摆更被称为〝松夹弯〞，尤其公开说〝中国不是汇率操纵国〞，还真让人以为川普真的善变、难预测、高度不确定。
 <br/>
 <br/>
 事实证明，川普在2018年3月8日签署了两关税公告，对钢材课关税25%，对铝征关税10%。正式对中共展开贸易战，所用的政策就是纳瓦罗在《致命中国》书中建议的，而且川普撤换多位高官，引进和他一路走来观点一致的一批人，这一批人被认为对中共霸道行为极度不满，必须强力的反制。于是同心队伍已集结完毕，前线、后勤都同心，协力对中共展开攻击。于是对中共〝以暴制暴〞、〝以恶制恶〞、〝以其人之道还治其人之身〞的〝美中贸易战〞热闹开打。到2018年11月，贸易战已扩及多层面，而地缘政治和军备武力竞赛更居首位，这也是纳瓦罗在这一本书所关切的面向。
 <br/>
 <br/>
 
 <h4>
  亲美、亲共，台湾选哪边？
 </h4>
 <br/>
 如上文所言，本书第二十章就谈到〝美中因为台湾而引发战争的可能性〞。纳瓦罗提供了〝国族主义〞、〝地缘政治〞、〝意识形态〞和〝道德规范〞四个因素。后三个因素都会让美国为〝保护台湾〞而战，他也指出，由于〝美国的经济极度仰赖中国，华府的官员也非常仰赖跨国企业的大笔政治献金，而这些企业在茁壮的中国贸易中可以取得大量的利益。因此，政治和经贸两方面局限了美国对台湾的支持。〞所以〝过去数十年来的美国总统，都让北京政府有足够的理由相信，美国愿意为了政治和经济实际上的考量牺牲台湾。〞
 <br/>
 <br/>
 这本书让川普洞悉〝过去美国的迟疑和软弱，已让快速军事化的中国（中共）变得更大胆，最终将大举入侵台湾。〞因而川普先下手为强，不但掀起贸易战，还持续讉责中共的多方渗透，而且副总统以下的重要官员大家炮口一致对付中共。用的就是纳瓦罗在书中所说的〝台湾的生存不只对美国的道德和意识型态有重大影响，蓬勃发展的民主也能促进自由和公平的贸易〞之观点，而且〝二战后美军在亚太地区的军事布局，非常仰赖美国在第一岛链周边的协约盟友，所形成的不可或缺的战略三角，台湾的防御就是第一岛链的支柱和中心。〞纳瓦罗更引用麦克阿瑟的话：〝台湾是永不沈没的航空母舰〞，言下之意，〝重要性不言可谕〞，美国也当然会尽力保护台湾不可。由川普近年一连串〝惠台政策〞，也可见一斑。
 <br/>
 <br/>
 那么，一方要并吞台湾、消灭台湾的自由民主，一方要保护台湾维持自由民主。台湾住民们，在正与邪、善与恶的两边，您要选靠哪一边呢？
 <br/>
