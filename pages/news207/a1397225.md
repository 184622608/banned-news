### 高天韵：巴西总统大选 带给中国启示与契机
------------------------

<div class="wysiwyg">
 10月28日晚，巴西总统大选揭晓：社会自由党候选人、被称为〝巴西版川普〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波索纳罗.html" target="_blank">
  波索纳罗
 </a>
 胜出、当选新一任总统。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波索纳罗.html" target="_blank">
  波索纳罗
 </a>
 （Jair Bolsonaro）在竞选中提出〝巴西优先〞的口号。他反堕胎、反同性恋，主张严厉打击犯罪，承诺减少国家对经济的干预，获得了巴西保守派民众及天主教的支持。由于其政治理念及个人风格与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普总统.html" target="_blank">
  川普总统
 </a>
 相似，且善用社群媒体与选民沟通，波索纳罗被称为〝巴西版川普〞、〝热带版川普〞。
 <br/>
 <br/>
 波索纳罗的获胜具有重大启示意义，综合媒体报导及评析，在此一议。
 <br/>
 <br/>
 第一，新任总统是人民及天意的选择。波索纳罗及其政党，原本势力弱小，其自参选后，人气迅速攀升，支持率从2016年7月的5%，升至今年7月的25%，再到第一轮选举斩获48%的选票，最后在第二轮赢得约56%的选票。这条轨迹表明了巴西广大民众对现状的不满，对传统理念的认同以及回归传统的意愿。
 <br/>
 <br/>
 10月28日当天，许多选民穿着写有〝我的政党是巴西〞的Ｔ恤衫，向外界显示：他们投出的一票是为了一个更好的巴西，而非某个政党。
 <br/>
 <br/>
 波索纳罗当选后，在社交媒体和电视上发表讲话，强调新政府的目标是将巴西建设成为一个〝伟大、繁荣和自由的国家〞。他还说，他一定会捍卫宪法、民主及自由，他并强调：〝这不只是一个政党或一个人的宣言，这是在上帝前的誓言。〞
 <br/>
 <br/>
 很有可能，正是这位前陆军中尉对神的信念，助他走上总统之位，演绎天意的安排。
 <br/>
 <br/>
 第二，波索纳罗反对中共的立场，引人注目。他批评中共为〝掠夺者〞。他认为，中共不应被允许拥有巴西土地或控制巴西关键工业。他曾表示：中共不是在巴西购买，而是想购买巴西。今年2月，波索纳罗以首位巴西总统候选人的身分访问台湾，对台湾示好。中共曾试图拉拢波索纳罗，可是未成。
 <br/>
 <br/>
 第三，此次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西大选.html" target="_blank">
  巴西大选
 </a>
 正值关键时节。国际舞台上，在川普总统的带领下，美国及西方多国正在联手反制中共，于政治、经济、军事、传媒等多个领域同步设防、回击。波索纳罗的胜出，将令反共联盟更强一筹。
 <br/>
 <br/>
 9月25日，川普总统在联合国大会上说：〝在所有社会主义或共产主义被实践的地方，它都导致了痛苦、腐败和衰弱。社会主义对权力的渴望导致了扩张、入侵和压迫。世界上所有国家都应该抵制社会主义及其给每个人带来的苦难。〞
 <br/>
 <br/>
 他当时特别提到了委内瑞拉的例子，〝我们正在目睹委内瑞拉的人类悲剧。超过200万人逃离了社会主义马杜罗政权，以及支持该政权的古巴赞助者。〞
 <br/>
 <br/>
 事实上，在拉美和南美，古巴、委内瑞拉、秘鲁等多国都深受共产主义和社会主义的危害，左翼势力至今仍不可小觑。巴西作为拉美最大的经济体，其政体走向对区域的影响力不可忽视。美国前总统理查德・尼克松曾说过：〝巴西走向哪里，拉美就走向哪里〞。
 <br/>
 <br/>
 如今，又一位〝川普〞的胜利，标志着保守主义思潮在世界范围日趋强劲，更多国家正在向传统回归。有网友说：〝我们希望世界上的川普越多越好！〞
 <br/>
 <br/>
 川普总统所代表的，是捍卫传统、坚守原则、不向邪恶妥协的决心。只有这样的意志，才能够重扬国威、重振民心。同时，美国的转向与对抗中共，令人审视正邪的分界，反思传统的意义。
 <br/>
 <br/>
 历史显示，在共产党执政的国家，传统文化遗迹都被公然砸烂，传统价值观被诋毁，文化精英被迫害。一代又一代国民被割断了与神、与祖先的连结，不知传统为何物，在无知中滑落。
 <br/>
 <br/>
 在诸多非共产党国家，社会道德与文化也受到了共产主义的多年持续的蚕食，人们以〝自由〞和〝进步〞为名，做出背离神的教诲的丑恶与肮脏的行为，导致人类社会普遍性的文化颓废、开放性乱的衰败和混乱。
 <br/>
 <br/>
 因此，当川普总统逆流而上、遵循神的指引、为传统原则而战时，他实际上就是在保护人类的生存环境与道德规范，在恢复社会的正常秩序，确保人类的健康发展与和平稳定。这一股正义的潮流，正在汇聚更多的正义力量，同时跨国界地鼓舞着所有善良人们心中的正气，在扬善除恶。
 <br/>
 <br/>
 有评论认为，巴西大选的结果令中南海不安。其实，北京当局何必忧虑？假如执政者当真以民为本，以传统为重，就理应为巴西的新政局鼓掌。更重要的是，这一事件带来新的启示和契机：美国正在再度强大，巴西也开始起步，那么中国呢？拥有五千年辉煌文明和传统文化的中国，也必须向传统回归，在回归中重建道德、重启希望。
 <br/>
 <br/>
