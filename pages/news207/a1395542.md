### 周晓辉：官媒持续反驳彭斯 任志强谔谔唱反调
------------------------

<div class="wysiwyg">
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 副总统10月4日就美中关系发表的强硬讲话，看来让北京当局相当地难受，同时也是相当地重视。近十天来，中共官媒几乎每天都要发表一至两篇否认、反驳的文章，涉及的官媒包括《人民日报》、〝新华网〞、《光明日报》、《经济日报》等，反驳的内容也是从政策批评到具体事件，从历史到用典，等等。这显然应该是中共中宣部安排的统一行动。其重复的依旧是陈词滥调，就是美国的指责是无理由的，美国人不懂历史，贸易战的始作俑者是美国，云云。
 <br/>
 <br/>
 10月12日，中共驻美大使崔天凯亦接受美媒的采访，再一次称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 演讲中所有的指控〝都是毫无根据的〞，再一次指贸易战是美国挑起的，中方并不愿意打，但如果必须打，中方〝就不得不予以反击〞。他并为中共当局做了无底气的辩解，口气也早已没有了早前的强硬。
 <br/>
 <br/>
 值得注意的是，还是如此前所指出的，所有的反驳都没有提到彭斯演说中所说的〝近年来，中国朝着控制和压迫本国人民的方向急转弯〞，没有否认中共正通过〝无与伦比的监控〞，严重限制着中国人的信息自由流通；没有否认当局正通过所谓的〝社会信用分数〞控制民众；没有否认中国依然没有宗教自由，〝中国的基督徒、佛教徒和穆斯林正在经受新一波迫害浪潮的冲击〞。估计中共当局觉得就这些问题反驳，更容易被老百姓们戳穿谎言，所以还是避谈为好。
 <br/>
 <br/>
 然而，在中共官方媒体如此自说自话的同时，大陆坊间却屡屡传出不同的忧虑声音。如学者时寒冰的〝大网正在逐渐完成合围之势〞，〝一切已悄然成定局〞；香港大学SPACE学院常务副院长刘宁荣教授辗转反侧，不知如何解决当下的困局，因为其给出的让民众参与国家发展、不能一人独尊天下、建立一个可以保护个人自由和尊严的制度、要更加开放、接受一个全球都认同的价值观等解决之法，很难被北京当局接纳。
 <br/>
 <br/>
 而最新一位发出挑战官媒的乃是大陆地产大佬
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任志强.html" target="_blank">
  任志强
 </a>
 。在10月11日召开的第十届〝观峰讲坛〞上，任志强就中美贸易问题，以长城做隐喻，发表了自己的看法，而这显然应该代表着相当一部分企业家、知识分子的看法。至于其是否代表某些离退休高官，这就见仁见智了。
 <br/>
 <br/>
 大致总结一下，任志强的说法主要是：一、对于贸易战的认知错了，战略自然也就错了。他先从古代修的长城的防御功能说到关税，认为关税也是防御性的，而不是用来进攻的，因为〝你进我的长城的时候，才收关税〞。〝如果所有的媒体都把关税认为是恶意进攻，首先站位就是错了。认识都错了，战略自然也错了。〞北京错误的战略体现在〝加大自己的长城〞，即针对美国加征报复性关税。在其看来，〝现在把门关上就不对了。〞
 <br/>
 <br/>
 二、根据上一个逻辑，任志强认为美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的加征关税是自我防御，而不是进攻，因为他对所有国家都一样。
 <br/>
 <br/>
 三、中美出现对立的原因出在中国。〝别人就能谈得通，因为别人开放；中国没有谈通，就是中国不肯把大门打开。〞〝因为你要把自己的大门关起来的，而不是别人挑起来的。别搞错了。〞
 <br/>
 <br/>
 四、不认为中美间要打贸易战，隐晦批评北京的加征关税的以牙还牙政策。任志强表示，做企业家有个最简单的东西：买卖不成仁义在。但现在买卖不成了，仁义也不在了，其潜台词就是北京前一阵的报复措施和歇斯底里导致了今天中美几乎走向全面的对抗。
 <br/>
 <br/>
 五、批评北京当局没有沿用邓小平的韬光养晦策略。
 <br/>
 <br/>
 六、中美贸易战结束的唯一前提就是北京拆掉长城。任志强说：〝不想打开大门的，都谈不成。我们的老百姓还在其中，因此而受害。最后的结果，不是中国把长城拆了吗？否则长城以外都不是中国的地方了，你要那个长城干嘛呀？〞其潜台词就是如果中国被排斥在了世界经济体系之外，北京当局只能自己玩过家家，其结果可想而知。
 <br/>
 <br/>
 在阐述完自己的观点后，任志强还说了这么一句〝能不能听懂是你们的事了〞。〝你们〞是谁？现场观众？北京高官？大家自可想像。
 <br/>
 <br/>
 无疑，任志强之言就是与官媒在唱反调，直接点出贸易战并非是美国挑起的，而是北京当局自我认知的错误，从而导致了战略上的错误。北京应对贸易战的恰当之做法是将自己的大门真正打开。
 <br/>
 <br/>
 在笔者看来，任志强乃至其他人给北京当局提出的破解困局之法，北京高层都心知肚明，也深知官媒的东西不过是骗骗中国人而已，对美国人完全起不到任何作用。不过，当开放大门、改变经济结构与保住权力问题存在矛盾时，北京高层是否要听这些〝谔谔〞之音，外界尚在观望，而11月底川普能否与习近平会晤是一个风向标，因为美国已经表态，北京不作大的让步，会晤不会举行。
 <br/>
 <br/>
 除了北京的态度令外界有待观察外，在北京收紧舆论的情况下，大胆直言的任志强会否再度被封杀同样引人关注。要知道，2016年2月，任志强因质疑中共〝官媒必须姓党〞的主张，遭到刘云山掌管下的官媒文革式的围攻，官媒刊登多篇文章对任进行批判。随后，任志强在腾讯、新浪的微博被强行关闭，北京西城区委还称将〝对任志强作出严肃处理〞。
 <br/>
 <br/>
 有意思的是，就在任志强被围攻之际，与其关系良好的王岐山领导下的中纪委监察网站刊登了署名文章《千人之诺诺，不如一士之谔谔》，为喧嚣几日的所谓〝任志强反党〞事件做了结论，即〝能否广开言路，接受建议，常常决定一个朝代的盛衰〞。文章还引述习近平在参加河北省委常委班子〝专题民主生活会〞时的一段讲话：〝小问题没人提醒，大问题无人批评，以致酿成大错，正所谓‘千人之诺诺，不如一士之谔谔’啊！〞〝谔谔〞的意思就是正言批评。众多唯唯诺诺之人，不如一名诤谏之士可贵。
 <br/>
 <br/>
 随后，《人民日报》发文、俞正声在政协会议上讲话都配合中纪委的文章，暗挺任志强。而随之习近平也在参加民建工商联组的讨论时发声，强调〝讲真话、说实情〞，表示〝如果舆论引导的结果是把大多数人推向反对派，那必然是在损害改革共识，掏空进步的根基〞。
 <br/>
 <br/>
 两年多后，任志强又做了一名与官媒唱反调之人，只是此一时非彼一时，在众多谔谔之士被封杀的当下，他又会怎样呢？
 <br/>
 <br/>
