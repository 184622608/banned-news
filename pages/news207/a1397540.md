### 程晓容：10月大事多 中共成众矢之的
------------------------

<div class="wysiwyg">
 10月新闻大事多，一个趋势清晰浮现：正义浪潮推进，中共已成众矢之的。越来越多的人识破了中共的恶意企图，并谴责其种种恶行。因此，拒绝、抛弃中共，乃顺势而行，识时务者为俊杰。
 <br/>
 <br/>
 本月最重大消息，当推美国副总统彭斯的对华政策演讲。10月4日，彭斯发表讲话，结合中美关系史，严厉抨击中共在国内国外的不端行径，向中共喊话：美国不会被吓倒，美国不会退缩。此番重磅严辞，被视为美国讨伐中共的檄文，震动全球。
 <br/>
 <br/>
 接下来，美方祭出了更多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反制中共.html" target="_blank">
  反制中共
 </a>
 的动作，信号明确：美国将继续全面反击，不会罢手。
 <br/>
 <br/>
 10月10日，美国司法部发布消息：中共情报官、江苏省国家安全厅第六局副处长徐延军已于10月9日被从比利时引渡到美国，并将因涉嫌经济间谍和商业窃密面对司法起诉。这是美国首度在第三国逮捕并引渡中共间谍。
 <br/>
 <br/>
 10月19日，美国国家情报总监办公室、国土安全部、联邦调查局和司法部发表联合声明，强调严防及抗击外国干扰美国选举，中共被列为实施干扰的外国势力之一。
 <br/>
 <br/>
 10月20日，美国康奈尔大学正式通知中国人民大学，暂停双方的两个学术交流和一个研究项目。据悉，暂停原因是人民大学处罚了几名支持深圳工人组织工会的学生。这是多年来外国大学首次出于此类理由中断与中国大学的合作。
 <br/>
 <br/>
 10月29日，美国商务部宣布，出于维护国家安全的考量，将中国福建省晋华集成电路有限公司列入出口管制实体清单，该禁令于次日生效。
 <br/>
 <br/>
 10月30日，美国司法部发布一份起诉书，起诉与黑客合作的数名中共情报人员。在5年多的时间里，这些中方情报员指导其他人员，多次侵入欧美私营公司的计算机系统，窃取商用喷气式客机中使用的涡轮风扇发动机机密。
 <br/>
 <br/>
 美国对中共立场坚决、行动果断，这无疑带动了其它国家，令更多政府也表明态度，对中共说〝不〞。目前，中共的〝一带一路〞计划在亚洲、非洲频频受挫，多国都以中止或削减项目进行抵制，因为掠夺当地资源、染指当地主权的中共模式不受欢迎。
 <br/>
 <br/>
 10月下旬，英国警方发布消息，中共央视驻伦敦记者孔琳琳被以〝普通袭击罪〞起诉，原因是她在9月30日的一次会议上与一名工作人员发生肢体冲突。该案件定于11月7日在伯明翰裁判司署提讯。如果被判有罪，孔琳琳将面对最高六个月有期徒刑或罚款，或两者兼施。
 <br/>
 <br/>
 10月28日晚，巴西总统大选结果出炉：极右翼社会自由党候选人波索纳罗获胜。波索纳罗被称作〝热带版川普〞，他拥护保守价值观，强调捍卫宪法和自由，反共立场鲜明，指中共为〝掠夺者〞。在左派执政33年后，右派人士主掌巴西，将如何影响南美和世界格局？变化令人振奋，未来令人瞩目。
 <br/>
 <br/>
 再看中国大陆。官场混乱分崩：中共公安部副部长、国际刑警组织主席孟宏伟被调查，中共澳门中联办主任郑晓松离奇坠亡，有关二者关联的传闻更显黑幕深重。各级贪官纷纷落马、被〝双开〞、受审，受贿金额巨大、犯罪情节恶劣。
 <br/>
 <br/>
 官员腐败及涉黑案情不断曝光，令中共自打耳光。即使如此，中共还在自吹自擂，强令民众对其服从、崇拜，并且封锁真相、侵犯人权，充分暴露其荒谬和残暴。
 <br/>
 <br/>
 本月，大陆民众继续维权行动。江西、广东顺德居民抗议当地建设污染项目；P2P难友北京聚集讨血汗钱；著名人权活动人士黄琦被非法关押、身患重病，其母蒲文清控诉中共警方对黄琦构陷虐待。10月16日，蒲文清向北京高院递交申诉材料，但是高院拒绝受理。
 <br/>
 <br/>
 至此，信息已明：中共就是〝掠夺者〞，它掠夺国家资源、掠夺民脂民膏，剥夺百姓的自由。它国内骗完国外偷，明抢暗窃、横行猖狂。中共还是个〝破坏者〞，破坏传统文化、破坏普世原则、破坏政治、经济秩序。
 <br/>
 <br/>
 善恶终有报。在中国及世界正义之士的持续努力下，中共的罪行及本质被逐步揭露，国际社会开始觉醒并展开行动。川普总统的勇气引领功不可没，这也是天意使然。此消彼长，正的力量在占据上风，中共大势已去。
 <br/>
 <br/>
 中共反天、反地、反人，制造灾难。这样的政权，无论打出多么动听的旗号，开出多么优惠的条件，都只是为了包藏它的祸心。它的存在，对人类社会有害无益。目前，在中共与美国等国的经济及军事纷争的背后，意识形态的根本差异才是导致冲突的本因。与中共的抗衡，实为正邪之战，是为了保护人类赖以生存的道德、原则和秩序，而非单纯的国力竞争。认清此点，就会坚定抗击中共的决心。
 <br/>
 <br/>
 对于中共官员来说，如果为了权力或私利而不放弃红色学说，便是选择了一条不归之路，前途堪忧。对于广大中国民众而言，退出中共组织、不相信中共的欺骗宣传、寻找真相、接受真相，就能为自己开辟新天地。解体中共，推倒红墙，失去的将是魔鬼的捆绑。
 <br/>
 <br/>
