### 新世纪影视带给世界的清流（3）—影片《过年》
------------------------

<div class="wysiwyg">
 近日，美国新影视基地试映了新拍摄的两部短片《艺考》和《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_过年.html" target="_blank">
  过年
 </a>
 》，有幸先睹为快。
 <br/>
 <br/>
 成立于2017年的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新世纪影视.html" target="_blank">
  新世纪影视
 </a>
 基地，拍摄的首部华语片《密码》（Code of Revelation），就获得了2018年加拿大国际电影节（Canada International Film Festival）最高奖（Grand Prize Award）。
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_过年.html" target="_blank">
  过年
 </a>
 》的题材与《艺考》一样，都涉及中共迫害法轮功的事件。男主角是50多岁的沈贵友, 大年三十的夜晚，女儿沈玉因为修炼法轮功被中共抓进看守所, 面临开庭审判。老沈只能一个人在家凄凉地度过大年夜，随着接连不断地敲门声，电影展开剧情……
 <br/>
 <br/>
 《过年》与我在前文介绍过的《艺考》不同，《艺考》开始的情节轻松诙谐，但是，结局反转却令人意外沉重，这让观影者看到，中共对法轮功信仰群体的迫害持续近20年，现仍在进行；而影片此前轻松喜剧的风格转换为沉重结局，也暗喻了中共对人们正常美好生活的践踏。
 <br/>
 <br/>
 在《过年》这部短短20分钟的电影中，为人们展示出了在中共迫害法轮功的运动中的众生像。这场迫害涉及到几乎全社会的每一个阶层和角落。在这场迫害中，每一个被波及到的人，法轮功学员、法轮功学员的亲朋好友、邻居、警察、律师等等，都在其中扮演者不同的角色，做出不同的反应。
 <br/>
 <br/>
 《过年》的结局，是出乎意料的反转，带给人们光明与希望，从中可以看到人心的觉醒和中共对法轮功这场迫害的穷途末路。
 <br/>
 <br/>
 在新世纪影视基地的官网中这样介绍，该影视基地的宗旨是传承〝真、善、忍〞核心价值，为所有秉持相同理念的影视人，提供一个广阔自由的创作平台；在世界需要回归传统，世人需要启迪心灵之时，制作出更多优秀影视作品，唤醒世人本性的归真，再现神传文化精髓，带给世界新的希望。新世纪影视基地正在带给世界一股清流，我们期待着这股清流带来更多的精采影片。
 <br/>
 <br/>
 我想，新世纪影视基地的创作人员，在中国新年即将到来之际，把《过年》这部影片作为新年的礼物，献给观众，献给了无数在中国大陆正在遭受迫害的法轮功学员及其家庭：寒冬即将过去，春天就要到来，等待在苦难中前行的法轮功学员们前方的，将是一个美好的大团圆。
 <br/>
 <br/>
 ──转自《大纪元》
 <br/>
