### 苦胆：这是上天的警示
------------------------

<div class="wysiwyg">
 时下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的一次次〝重要讲话〞，被溜须拍马者在报告中、电视里、报纸上大肆吹捧。其中的不少语句，堂而皇之地集成了〝习近平金句〞〝习近平语录〞，尤其是那个被阿谀逢迎者肉麻解读的热句〝功成不必在我〞，还上了公路广告牌。
 <br/>
 <br/>
 这句〝功成不必在我〞，自然叫人联想到大学者胡适的名言：〝成功不必在我，而功力必不唐捐。〞胡适的话，是赠给当年的大学毕业生的，全文刊登在1932年7月3日《独立评论》第7号上。以胡博士的身份、学养和人格，说这样的话，听起来十分自然、贴切；而这句话出自一个文化程度屡遭质疑、又爱报一串串书名的急功近利者之口，听着就觉得隔膜而滑稽可笑了。
 <br/>
 <br/>
 那块遭人诟病的公路宣传广告牌，是贵州省纳雍公路管理段制作的，上面横排的〝功成不必在我〞六个字相当醒目。这个牌子原本是从左到右读的，可是有网友发见，如果改成从右到左读的话，则变成了〝我在必不成功〞。无怪乎跟帖者断言：〝失败一定在他〞。
 <br/>
 <br/>
 对此，有人说这是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高级黑.html" target="_blank">
  高级黑
 </a>
 〞；有人说这是拍马屁拍过了头，弄巧成拙，物极必反；也有人说这是一种偶然的巧合……
 <br/>
 <br/>
 不过，在我看来，这正是天意，正是上天的警示，也是应和贵州〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_藏字石.html" target="_blank">
  藏字石
 </a>
 〞的预言。2002年，贵州省平塘县掌布乡发现一块神奇的巨石，断裂面上赫然显现天然形成的六个大字＂中国共产党亡″。大陆多位地质专家实地考察，做出权威鉴定结论，藏字石上的字形成于2亿7千万年前，至今未发现任何人工雕凿等加工痕迹，是世界地质奇观。当地政府藉此巨石开发旅游，但官方只宣传前面的5个字，导游也只介绍前5个字。而据新唐人电视台2017年8月17日报导，中共处于末日恐惧，贵州平塘县掌布风景区的主要景点〝藏字石〞被官方封闭，景区门票也有改动，〝中国共产党亡〞的〝亡〞字，被半遮半掩。
 <br/>
 <br/>
 笔者近日在浏览〝希望之声〞网页关于〝习语录〞的新闻时，看到一位署名〝林泉小语〞的网友在2018年10月6日的留言：
 <br/>
 <br/>
 我刚看过2005年中央十台对贵州藏字石的报道。在此报道的最后一行字中竟有〝走近平〞塘县——，此时，我猛然想到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，就是习接近平塘了，而平塘就是〝中国共产党亡〞的出处。
 <br/>
 <br/>
 是的，中共邪党，正在一步一步地走向失败，走向覆亡。
 <br/>
 <br/>

 <br/>
