### 胡欣跳楼身亡 给中共宣传部门人员的警钟
------------------------

<div class="wysiwyg">
 中共机关报《人民日报》旗下杂志《新闻战线》前总编辑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡欣.html" target="_blank">
  胡欣
 </a>
 ，2018年11月6日从该报社36号楼十九楼跃下身亡，终年66岁。有网络媒体称胡欣生前疑患抑郁症。网络也流传胡欣自杀的原因，遭网管删除。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡欣.html" target="_blank">
  胡欣
 </a>
 毕业于北京大学哲学系，硕士学历，1990年进入《人民日报》理论部工作，1998～2004年任《人民论坛》杂志社总编辑，做过理论部部务委员，2008年底任人民日报社《新闻战线》杂志社总编辑。
 <br/>
 <br/>
 中共喉舌《人民日报》及其《人民论坛》、《新闻战线》在江泽民迫害法轮功期间大量发表污衊、丑化法轮功和法轮功创始人李洪志先生、丑化法轮功修炼者的文章，并在海内外媒体上大肆转载，为江泽民迫害法轮功制造舆论，挑动蛊惑不明真相的人仇恨法轮功、迫害大法弟子。胡欣在《人民论坛》、《新闻战线》杂志社任总编辑期间，积极迎合中共恶党污衊法轮功。
 <br/>
 <br/>
 《人民日报》的一名记者邱明伟，曾经是《人民日报》麾下《人民论坛》的副主编，因他曾帮助过法轮功学员，为躲避政治迫害，前些时到了海外避难，他揭示出一个事实：记者、编辑常常被迫造假。因为，用一个造假记者的话说：〝不造假就没有饭吃〞，记者也只能是〝党叫干啥就干啥〞的文字工具。他在香港的一个记者会上公开宣布以真名退出中共。邱明伟说，在过去的研究中认识到法轮功主张〝真、善、忍〞，遇到矛盾向内找，是一个没有暴力、一个平和的修炼团体，〝这有什么不妥？〞他强调每个人都应该可以享有言论及宗教信仰的自由，法轮功学员在维护自己权利的时候，实际上也是在维护每一个人的权利不受侵犯。
 <br/>
 <br/>
 中国有一首民谣：〝我是党的一条狗，守在党的大门口，党让咬谁就咬谁，让咬几口咬几口。〞这首民谣非常精准地概括了中共宣传媒体的流氓本质。那些丧失良知的新闻人真的把自己当成了中共的一条狗了。因造假有功、欺骗愚弄百姓有力，中共给了胡欣最高的荣誉：《新闻战线》曾连续三年在中国新闻奖评选中，获奖总数居中国新闻专业刊物之首。
 <br/>
 <br/>
 从古至今，诽谤佛法、迫害修炼人罪恶极大，必遭天谴！谁也逃脱不了。而且人间的报应只是为了警醒世人，地狱的报应那才是偿还恶业的过程，还会殃及子子孙孙。最可悲的还是那些操纵和参与诽谤抹黑法轮功的小人了。
 <br/>
 <br/>
 明慧网发表的《诽谤法轮功 百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宣传部门.html" target="_blank">
  宣传部门
 </a>
 官员遭恶报》一文揭示了一百多个利用媒体污衊法轮功而遭恶报的实例。其中自杀式的报应有十几起：
 <br/>
 <br/>
 1. 中共理论刊物《求是》杂志副主编朱铁志在单位地下车库自缢死亡；
 <br/>
 2. 《人民日报》副刊主编徐怀谦跳楼自杀；
 <br/>
 3. 西安灞桥区宣传部长张浩然在西安市水晶岛酒店割腕自杀；
 <br/>
 4. 新华社安徽分社副社长、总编辑宋斌在办公室自缢死亡；
 <br/>
 5. 深圳《晶报》原编委、广告部总经理张敬武自杀；
 <br/>
 6. 杭州《都市快报》副总编徐行忧郁症自杀；
 <br/>
 7. 北京《体坛周报》女编辑李颖跳河自杀；
 <br/>
 8. 广东佛山三水区委宣传部长郑运安坠楼死亡；
 <br/>
 9. 黑龙江大庆市红岗区区委宣传部庞宝来迫害法轮功，跳楼自杀。
 <br/>
 …… ……
 <br/>
 <br/>
 那些还在用网络、媒体污衊法轮功的人，赶快清醒吧，天灭中共是必然，千万不要做中共的〝替死鬼〞，千万别为中共〝陪葬〞。
 <br/>
