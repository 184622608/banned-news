### 什么是幸福
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幸福.html" target="_blank">
  幸福
 </a>
 就是在自己的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福报.html" target="_blank">
  福报
 </a>
 之下过日子。
 <br/>
 <br/>
 什么意思呢？你有1百万，你在农村过日子，你会觉得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幸福.html" target="_blank">
  幸福
 </a>
 ！你有1千万你在城市过日子，你会觉得幸福！你有1亿，你在帝都过日子，你会觉得幸福！
 <br/>
 <br/>
 不然的话！你有1百万，你在城市过日子，你入不敷出，连生存都成问题了，根本谈不上幸福！
 <br/>
 生存讲的是温饱；生活才讲幸福的，没有温饱之忧得以实现欲望尚有余盈，就是幸福！
 <br/>
 <br/>
 幸福，还有更深层次的理解，就是幸庆没有迷失，礼拜神佛得庇护得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福报.html" target="_blank">
  福报
 </a>
 ，这才是幸福的根本。幸福是在道德的基础上追求的欲望，道德是圣人的教化、神佛的教化！没有道德的约束盲目追求欲望是毁灭！ 。
 <br/>
 <br/>
 人礼拜神佛得神佛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赐福.html" target="_blank">
  赐福
 </a>
 ！人拜魔鬼受魔鬼控制！
 <br/>
 <br/>
</div>
