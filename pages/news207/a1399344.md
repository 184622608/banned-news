### 颜丹：中国古装剧越拍越烂
------------------------

<div class="wysiwyg">
 近日，大陆某网对中国1976年至今所拍摄的891部古装剧进行了统计、分析。其结果显示，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国古装剧.html" target="_blank">
  中国古装剧
 </a>
 的质量确实越来越差，90年代之前还偶有平均分上8的时候，90年代也稳定在7分到8分之间〞；但〝进入21世纪后，分数便开始逐渐走低，2010年之后更是开始跌破6分的及格线〞。
 <br/>
 <br/>
 实际上，大陆所拍摄的古装剧甚至有22部在9分以上，但〝大多数都集中在2010年以前〞。而〝评分最低的古装剧，基本都集中在近几年〞。〝4分及以下〞的44部古装剧，全部出自2010年之后，且不少作品是从〝网络文学IP〞改编而来。
 <br/>
 <br/>
 看来2010年似乎是国产古装剧急转直下、越拍越烂的一个重要分水岭。以平均得分最低的〝神话奇幻剧〞为例，尽管在〝2010年后内地的256部古装剧集中，多达81部〞；〝2018年内地26部古装剧里，一半都是〞；且类型多样，比如穿越剧、衍生戏剧、有的甚至从游戏改编而来，但却并不影响这个时期的神话奇幻剧由于〝内容并没有太多新意〞，〝又臭又长〞而被评为〝烂片〞。
 <br/>
 <br/>
 与之形成鲜明对比的，是2010年以前拍摄的神话奇幻剧。由经典文学改编的《西游记》、由民间传说改编的《济公活佛》（90版）甚至分别获得了9.5分和9.3分的好评。既然都叫〝神话奇幻剧〞，为何拍摄于2010年之前和之后会呈现出如此天壤之别？
 <br/>
 <br/>
 从《西游记》和《济公活佛》被多次翻拍后，收视率每况愈下、甚至被嘘、被骂就足以让人看出，活在当下、却无法摆脱历史影响的现代观众仍拥有着这样的共识，那就是真实的历史不容戏说，传奇的人物不容恶搞。
 <br/>
 <br/>
 唐僧西天取经也好，济公展现神通也罢，不仅有史料记载，并非凭空杜撰；更在于其诠释了华夏民族早已有之的修炼文化以及出世之道。〝儒、释、道〞既是宝贵的文化遗产、精神财富，也同属于中国古老的科学体系。〝道家讲‘真’、佛家讲‘善’〞；在漫长的五千年文明中，修佛、修道的，又何止唐僧、济公？
 <br/>
 <br/>
 而中国人了解甚少，只是因为这些体现着传统价值、充满着神秘色彩的历史片段，在毛发动的〝处心积虑的灭绝中华传统文化〞的政治运动中是受到否定和批判的。这也正是大陆古装剧只能在1976年之后才开始逐渐出现在电视荧幕上的关键原因。
 <br/>
 <br/>
 然而，有意思的是，很多中国人却并没有因为中共对传统的打压、破坏、摧毁而对古老文明敬而远之。恰恰相反，他们骨子里对传统文化的热爱、对历史真实的渴求，一旦被唤醒、激发，就会表露无遗。
 <br/>
 <br/>
 《共产主义的终极目的》一书对此有细致的描述，〝文革以后，……当世人听到《杨家将》、《岳飞传》、《三国》、《水浒》等名著广播时，却是一时间街道冷清，家家聚精会神，生怕落下一集，连警察都不用执勤。正是几千年神传文化的积淀，重新唤起了世人心中的正念和久远的记忆〞。
 <br/>
 <br/>
 正是由于中华神传文化包含着〝天、道、神、佛、命、缘、仁、义、礼、智、信、廉、耻、忠、孝、节〞等内涵，因此中国〝许多人可能一生都不识字，但是对传统戏剧和评书却耳熟能详，这些文化形式都是民间百姓获得传统价值观的重要途经（见《九评共产党》）〞。随着电视的普及，电视剧这种文艺形式也自然而然的成为了一个新途径。
 <br/>
 <br/>
 早期的古装剧若能或多或少的反映出这样的内涵，获得高收视率和好评是不足为奇的。即便像《西游记》这样的〝神话〞以及与济公这样的修炼人有关的传奇故事，只要能最终揭示出〝儒、释、道〞 的价值与内涵，就会让人喜闻乐见。这也正是由导演帮受众打开了久远的记忆、二者在精神上产生了共鸣所致。
 <br/>
 <br/>
 遗憾的是，这样的好景并不长。视传统为敌的中共对神传文化以及所有中国人的精神打压，并未因政治运动的结束而停止。从最近几年〝宫斗戏霸屏〞就不难看出，〝中共对传统文化一直采取偷梁换柱的办法，把从古到今，人在背离传统文化后产生的宫廷斗争、权谋诡计、独裁专制等等‘发扬光大’，创造出一套它们的善恶标准、思维方式和话语系统，并让人认为这种‘党文化’才是传统文化的继承（见《九评共产党》）〞。
 <br/>
 <br/>
 《共产主义的终极目的》也具体指出，〝这些宫斗剧以‘古装’面目出现，表现的是古代的故事和场景，但用的都是现代人的观念思维；穿着古装演古人，其实质却是现代戏，恢复起门面却毁掉了其内涵〞；〝让人不假思索地以为党文化的斗争哲学就是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国传统文化.html" target="_blank">
  中国传统文化
 </a>
 了〞，而〝这就是中共用暴力摧残神传文化，再以党文化灌输几十年之后收到的效果〞。
 <br/>
 <br/>
 尽管有关部门统计得出，这种宫斗戏的戏迷不少，但却并不代表，中国人对真正的传统文化有所了解。恰恰相反，被这种充斥着党文化的厚黑哲学洗脑的中国人，将永远会在你争我夺、尔虞我诈的无良社会中自食恶果。可见，中共广电部门之所以在1995年之后撤掉了〝禁止私人制作电视剧〞的条款，其真实用意决不在〝还民自由〞，而是想要用此方式进一步毒害中国人。
 <br/>
 <br/>
 信神、重德、向善……，这些根植于中国人内心深处的本质特性，是信奉〝无神论〞、崇尚〝假、恶、暴〞的中共欲除之而后快的。将大量的穿着古装的烂片摆在中国人面前，中共的目的其实只有一个，那就是斩断他们对神的正信、摧毁其心中的良善。把人改造成非人，以至毁灭人类，这才是中共最险恶的用心。
 <br/>
 <br/>
