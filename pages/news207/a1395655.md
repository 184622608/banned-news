### 崔士方：孟宏伟入笼 公安部高层彻底〝洗脸〞
------------------------

<div class="wysiwyg">
 中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部.html" target="_blank">
  公安部
 </a>
 因为正部级副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 的突然倒台而被彻底〝洗脸〞，若与十八大前的格局相比，如今的公安部高层已〝面目全非〞。
 <br/>
 <br/>
 如果说中共政法委是〝刀把子〞，那么
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部.html" target="_blank">
  公安部
 </a>
 就是这个刀把子的刀刃上最尖锐的部分。中共当权者历来都把掌控〝刀把子〞的刀刃和枪杆子，看作是政权真正在手的〝放心〞标记。
 <br/>
 <br/>
 公安部虽然权炳极重，但文革后，除了部长、常务副部长，其他副部长都是副部级，正部级副职属于〝稀缺资源〞。直到1999年江泽民发动了对法轮功的残酷镇压后，江对镇压〝不力〞不满，在公安部增加了一名正部级的副部长出任中央610办主任，这时的正部级副部长才首次增加到两人。
 <br/>
 <br/>
 中共十八大后的首个两会（2013年3月），中共政府高层更迭。时任公安部副部长的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被任命为新成立的中国海警局局长兼国家海洋局副局长、党组副书记，并明确为正部级。
 <br/>
 <br/>
 这个任命就有些蹊跷了。此前孟宏伟分管国际合作、出入境管理、边防，让其兼任海警局局长属于同类扩展，本不出奇。但国家海洋局是个副部级机构，副局长官座更只有正司局级，让孟平级兼任都足够〝高配〞，更遑论把孟提拔为正部级呢？
 <br/>
 <br/>
 孟宏伟的这一步反常升级显得相当〝没理由〞。在习近平初上台立足未稳的当口上，把孟宏伟这个周永康派系的人赶鸭子上树，急速推到高位，幕后之手是谁，令人生疑。
 <br/>
 <br/>
 此后公安部高层〝习家军〞不断冒头，习近平的福建嫡系王小洪、邓卫平相继进京〝插入〞公安部。王小洪不但延续了傅政华以公安部副部长兼任北京市公安局局长的做法，还更进一步，被任命为国安办常务副主任，明确为正部级。这样一来，公安部副职中，就罕见的出现了四名正部级，分别是傅政华（常务副部长）、黄明（中央610办主任）、孟宏伟、王小洪。这种〝高配〞格局是公安部成立以来没有过的。
 <br/>
 <br/>
 藉助打老虎和机构改革，周家党和习家军不断此消彼长。十九大后，中央610办并入中央政法委，黄明调到应急管理部做书记，傅政华也收拾包袱去了司法部，王小洪升任公安部常务副部长。这时，在王小洪之外的唯一正部级〝老人〞孟宏伟俨然成了钉子户。
 <br/>
 <br/>
 很多人把2017年底孟宏伟被免去国家海洋局副局长及中国海警局局长，看作是其失势的信号。这未免有失偏颇。真实的原因是中国海警局转隶军委下武警部队领导，办公室都没了，这当然会导致孟宏伟帽子被摘。
 <br/>
 <br/>
 孟真正失势的信号其实出现在2018年3月，当时许甘露从河南进京，出任公安部副部长兼国家移民管理局（出入境管理局）局长，出入境管理是孟的传统地盘，这一下子就被习家军夺去了。次月，孟宏伟的公安部党委委员被剥夺，成为一个空头副部长，这就大大不妙了。这时孟的倒台已是迟早的问题了。
 <br/>
 <br/>
 孟宏伟中箭落马，公安部党委委员中至此再无十八大之前的老面孔，公安部也恢复到只有一名正部级副职的老格局。但这是否意味着中南海对公安部的清洗就结束了呢？未必。
 <br/>
 <br/>
 周永康把持公安系统足足10年，冰冻三尺之寒无法指望一夜间就消融。更关键的是，当初那只把孟宏伟推上来的看不见的手还〝活着〞呢。
 <br/>
 <br/>
 如果〝刀把子〞的隐患已除，就不用公安部长赵克志反复讲要清除周永康〝余毒〞，王小洪也不必升任公安部常务副部长了，却还死死抓住北京市公安局的把手不放。这种〝肌紧〞状况，说明中南海的大脑神经目前依然紧绷。
 <br/>
 <br/>

 <br/>
