### 台铁普悠玛翻车事故 司机意识清楚可说话
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车昨天发生重大死伤意外，事故列车尤姓司机员在罗东圣母医院治疗中，，院方上午表示，尤姓司机员意识清楚，可以讲话。至于事故当下状况，有待检察官厘清。
 <br/>
 <br/>
 圣母医院院方今天上午表示，尤员有头部外伤、第三根肋骨骨折等伤势，院方先前是有开立受伤证明，今天状况比昨天改善，意识清楚，可以讲话。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 主任秘书朱来顺在上午的记者会中表示，事故的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车司机在操作过程中曾反映有〝（气压）压力过低〞等问题。
 <br/>
 <br/>
 台铁北回线普悠玛列车（6432）树林往台东列车昨天下午在宜兰新马车站发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 翻覆事故，据内政部消防署截至上午5时最新统计，共205人死伤，包括18人死亡、187人受伤（含司机员）。
