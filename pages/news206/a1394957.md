------------------------

  中华民国
 </a>
 107岁生日，当然要许下生日愿望了，马上带您来看，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民众.html" target="_blank">
  民众
 </a>
 对国家未来的期许。
 <br/>
 <br/>
 一面面国旗，有人披在身上，有人拿在手里挥舞展示，大家脸上都充满笑容，今天是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中华民国.html" target="_blank">
  中华民国
 </a>
 107岁生日，在台北及台中市政府前参加国庆活动的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民众.html" target="_blank">
  民众
 </a>
 ，热情表达对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 的爱。
 <br/>
 <br/>
 在国庆日这天，许多受访民众，都表示，希望台湾能更好，也纷纷许下了愿望。
 <br/>
 <br/>
 民众 曾先生：〝期许台湾未来变得更好，然后我们的政府要，更加油一些，然后让台湾有更多的工作机会。年轻人更有留在台湾的工作环境。然后这个环境，会让我们台湾年轻人成长跟茁壮。〞
 <br/>
 <br/>
 民众 张先生：〝我希望(台湾)可以越来越好，希望不要有天灾啊，什么都不要有。〞
 <br/>
 <br/>
 民众 李小姐：〝安居乐业。〞
 <br/>
 <br/>
 民众 洪小姐：〝蓝绿之间不要再恶斗，然后两岸之间有更好的一个沟通管道这样子。〞
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双十国庆.html" target="_blank">
  双十国庆
 </a>
 这天，全台湾人民同庆，也期盼台湾，能够越来越好。
