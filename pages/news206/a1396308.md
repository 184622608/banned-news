### 普悠玛翻车酿惨重伤亡 蓝绿暂停竞选活动吁捐血
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】台铁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车伤亡事件，为了让政府单位全力救灾善后。民进党中央下令，党公职和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 候选人停止竞选活动；国民党主席吴敦义也吁请党籍候选人，暂停一切选举活动，协助救灾，另外，高雄市长候选人韩国瑜呼吁民众捐血中心，并祈福伤者一切平安。
 <br/>
 <br/>
 新唐人亚太台报导，国民党立院党团总召 江启臣：〝在此时此刻应该先
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暂停选举.html" target="_blank">
  暂停选举
 </a>
 的活动，让所有社会相关资源能投入在抢救上，让我们的政府专心把善后处理好，检讨事故原因，让这样事故以后别再发生。〞
 <br/>
 <br/>
 民进党立委 张宏陆：〝大家都担心，我们这些受伤的同胞们，如何能够度过这难关，所以我想任何人都没有心情去选举，我觉得这就是台湾可爱的地方，人饥己饥。等过了救人的时刻之后，一切稳定之后，我们再来谈究责，台铁确实有很多该负责的事情。〞
 <br/>
 <br/>
 21日傍晚，韩国瑜在行程当中听闻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车下午的严重事故后在脸书发文表示，〝心中的震惊与难过实在无法形容。...由于伤者超过百人，相信血库一定告急，我希望各方好友能和我一起挽起袖子献血相助。如果有任何我能帮上忙的地方，也欢迎联系我的团队人员。...今夜，让我们一起为他们祈祷！也谢谢所有的医护、救难人员，麻烦您们了！〞
 <br/>
 <br/>

   </font>
  </a>
 </center>
 <br/>
 <br/>
 不过，台湾血液基金会表示，目前宜兰捐血站库存量相当足够，呼吁民众不要集中在这几天捐血，除担心血品过期，也会有捐了之后两个月不能再捐血的空窗。希望民众按照自己的捐血间隔，定期定量的捐血，维持血液库存量平衡，就是最好的救援。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 大选倒数一个月，是各阵营冲刺期。但朝野立委，都支持两大党暂停竞选活动。要让社会资源、政府机构能全力专注抢救，防范事故再发生。
