### 台国道3号1小时3起冲撞车 13车受损2人伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国道3号.html" target="_blank">
  国道3号
 </a>
 北上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彰化路段.html" target="_blank">
  彰化路段
 </a>
 ，30日上午7点多左右，在短短的1小时内发生3起连环车祸，造成13车辆碰撞受损、2人轻伤。
 <br/>
 <br/>
 综合媒体报导，第1起发生在早上7点18分许，203公里北上路段，因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_快官匝道.html" target="_blank">
  快官匝道
 </a>
 出口车多，后方车辆未注意车前状况，发生6车连环撞，幸好没有人员伤亡。
 <br/>
 <br/>
 第2起发生在7点40分、北上192公里路段正在进行路面刨铺，后方的小货车未注意前方车速变慢，追撞4部自小客车，造成小货车侧翻于车道，自小客颜姓驾驶(23岁)、陈姓乘客(49岁)手脚挫伤、胸部颈部不适，送台中荣总医院急救，没有生命危险。
 <br/>
 <br/>
 第3起发生在194公里北上路段，疑因前方事故，驾驶未注意车前状况，造成2车追撞，无人伤亡。
 <br/>
 <br/>
 事故发生后造成后方大塞车，现场直到上午8时40分清除完毕。
 <br/>
 <br/>
 警方表示，3起车祸都与未注意前方路况有关，提醒用路人，上班时间车多，要多注意车前状况并保持耐心与安全距离，才能避免发生事故。
