### 〝潭美〞今明离台最近 恐有颱风接力生成

 【新唐人2018年09月29日讯】再来带您关心，

  潭美
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 动态。中颱潭美靠近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 ，预计周六在日本〝宫古岛〞附近转向后，影响
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冲绳.html" target="_blank">
  冲绳
 </a>
 地区，随后将从九州一路贯穿到北海道，横扫日本。往返台湾与冲绳的航班，将全数取消。接下来，太平洋可能还有颱风接力生成，距离台湾3千多公里的热低压，会生成第25号颱风〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康芮.html" target="_blank">
  康芮
 </a>
 〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潭美.html" target="_blank">
  潭美
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 周五(28日)开始转向，周六凌晨最接近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 ，但在这期间仍要小心外围环流挟带风雨，北台湾山区甚至可能达到豪雨强度。颱风从周四晚上加速，预计通过琉球后转向日本。暴风圈会掠过台湾近海，气象局表示发布海上警报机率不高。
 <br/>
 <br/>
 中央气象局技正黄椿喜：〝受到这个颱风外围环流跟东北风的影响，其实在北部的山区尤其像新竹以北的山区跟基隆北海岸这一带，不排除从今天到明天会有豪雨发生的一个机会。〞
 <br/>
 <br/>
 周五晚上到周六早上，颱风偏向西北方向移动，这段时间比较接近台湾附近，整体是风雨最强的时期，到周六晚上雨势才会有较明显的趋缓。即便颱风逐渐转向，气象局仍呼吁民众周末海边或户外活动要多加小心。而周五早上关岛东南方形成的热低压，很可能会接力潭美，成为下一个颱风。
 <br/>
 <br/>
 中央气象局技正黄椿喜：〝未来的话其实增强为颱风的机率应该还算是蛮高的不过因为它现在还是热带性低气压，发展还没有相当的明显，所以未来的预测变数应该还是蛮大的。〞
 <br/>
 <br/>
 另外，周六晚上潭美颱风远离之后，还会有些干冷空气往台湾靠近，下周东北风增强，北台湾温度可能下探到21度。
 <br/>
 <br/>
 新唐人亚太电视胡宗翰、实习记者曾馨旻台湾台北报导
