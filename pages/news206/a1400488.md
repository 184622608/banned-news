### 高市选情紧 蓝绿车队扫街冲人气拼声势
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】选战进入最后阶段，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄.html" target="_blank">
  高雄
 </a>
 市的选情，令人紧张。星期四蓝绿候选人都透过车队扫街的行程，来冲人气。希望两天后的投票结果，可以顺利拿下高票，获得胜选。另外，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 阵营，针对日前在网路指控他买票的林姓男子，则有最新的调查结果。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄.html" target="_blank">
  高雄
 </a>
 市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 ，站在车上，向车子旁的支持者打招呼。选举倒数两天，选战打得激烈。也有各种消息传出。日前有位林姓男子，先是在网路上指出韩国瑜阵营买票，后来又去小港，亲自感谢韩国瑜不告他，当众向他道歉。不过，林姓男子接受电视访问时，竟然表示自己从当兵开始就是国民党员。整个事件发展过程，引发质疑。
 <br/>
 <br/>
 国民党高雄市长候选人 韩国瑜：〝(被人家传说演很大，国民党自己演很大，韩导自己演很大？)我们再重申一遍，这个林先生是谁我不知道，他为什么写这个，或传这个我也不知道，他来道歉，就一切烟消雾散结束了，跟演什么有什么关系，我觉得选举到了太多不必要的联想，太多不必要的穿凿附会，那是没有意义的。〞
 <br/>
 <br/>
 不过，韩国瑜阵营后来声明，透过身份证号码调查，国民党内没有这位林先生的资料，确定他不是党员。并进一步质疑，林先生是深绿铁粉。
 <br/>
