### 从5只病毒到物联网 张明正：人体可能遭骇
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】近年在萤光幕鲜少露面的，网路资安科技大厂董事长暨创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张明正.html" target="_blank">
  张明正
 </a>
 ，周四(1日)为自家公司30周年活动现身，分享30年来病毒的转变，从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5只病毒.html" target="_blank">
  5只病毒
 </a>
 到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_物联网.html" target="_blank">
  物联网
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_骇客.html" target="_blank">
  骇客
 </a>
 攻击，张明正也提醒，穿戴装置的普及，甚至还将晶片植入人体，未来人体可能是骇客下一个攻击目标。
 <br/>
 <br/>
 网路资安科技公司董事长暨创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张明正.html" target="_blank">
  张明正
 </a>
 ：〝IOT等等这些OT，互相结合，以前很少听过说，什么台积电会工厂会中毒，这件事大家知道了，ATM会中毒。〞
 <br/>
 <br/>
 一语道破病毒30年来最大转变，网路资安科技公司董事长张明正指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_骇客.html" target="_blank">
  骇客
 </a>
 从好玩有趣的心态，走向勒索从中获利，像是2017年的WannaCry病毒，就有150个国家遭到攻击。随着穿戴装置的普及，甚至还将晶片植入人体，张明正提醒，未来人体可能是骇客下一个攻击目标。
 <br/>
 <br/>
 网路资安科技公司董事长暨创办人 张明正：〝它一定要经过云端，5G、AI等等这类，因为晶片已经在人的脑子里面了嘛，所以这一点一定会发生的，所以这个事情会更切身，那大家对，资安的安全已经不再是，从公司电脑这么小的事情，是会到个人的人去这样。〞
 <br/>
 <br/>
 从30年前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5只病毒.html" target="_blank">
  5只病毒
 </a>
 ，到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_物联网.html" target="_blank">
  物联网
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_AI时代.html" target="_blank">
  AI时代
 </a>
 ，骇客的攻击不再单一，张明正表示，假如能把这些数据整合，变成有意义的解决方案，就能找出商机。
 <br/>
 <br/>
 网路资安科技公司董事长暨创办人 张明正：〝假如台湾在Edge这一边，就是边缘的私有云，这边我是觉得，还是有一些机会，不过这个的那个的Window很窄，一下子就够了，所以我看，硬体软体还有Edge的结合，我是觉得是重要的。〞
 <br/>
 <br/>
 张明正指出，网路的下一步，将以5G为首，也是美中较劲的主要战场。记者也追问，面对万物都有遭骇的可能，该怎么办？张明正展露笑容的说：〝回到花莲做农夫，不要用任何有联结的东西，就没事了。〞
 <br/>
 <br/>
 新唐人亚太电视胡宗翰、李晶晶台湾台北报导
</div>
