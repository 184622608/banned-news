### 丁守中重新计票 北院查封票匦12月3日验票
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】国民党台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 昨(28日)到台北地方法院声请查封票匦并
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 ；台北地方法院今(29日)下午到台北市选委会启动重新计票的查封程序，预计将在12月3日正式验票。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 落幕，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 以3254票落败，原向法院声请重新验票，需缴交保证金新台币428万元，但他在26日宣布撤回；不过丁守中昨天召开记者会，宣布重新提司法验票，并在律师陪同下，到北院声请查封票匦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 ，并缴交保证金。
 <br/>
 <br/>
 北院下午前往台北市选委会查封票匦，预计在12月3日正式验票。
 <br/>
 <br/>
 对此，台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 阵营律师林光彦稍早表示，申请重新计票是丁先生权益，会协助法院尽速厘清真相，相信重新计票结果会跟当天结果差不多。
 <br/>
 <br/>
 林光彦说，遵守法院程序和通知，大约召集至少80至100位律师，还在陆续增加中。
 <br/>
 <br/>
 至于有传说验票程序疑似遭技术性杯葛，丁守中阵营律师叶庆元说，希望北市选委会配合，以最顺利、快速的方式完成验票工作。
 <br/>
 <br/>
 联合新闻网报导，律师叶庆元说，候选人依法验票要缴交一票3元，但今早他被法院书记官通知选委会不愿意支出其他相关费用，例如今天要找搬运工人来协助，却要求候选人要额外垫付几万元的费用，选委会似乎有点技术性杯葛；他认为法律上的规定很清楚，希望选委会依法配合，也希望不是小动作，只是误会。
 <br/>
 <br/>
 针对指控疑似有技术性杯葛验票，北市选举委员会副总干事余淑否认说，过去2004年重验总统大选时，选委会也都只有提供〝寻找〞库房、验票场地、人力的协助，〝本来就没有出钱请搬运工人，负责的应是法院〞。
