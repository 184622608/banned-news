### 韩国瑜效应助攻丁守中？ 柯文哲：不会 因中间力量稳固
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ，今天(31日)早上，出席地方重建工程上梁典礼，受访时，针对11月即将举办的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辩论.html" target="_blank">
  辩论
 </a>
 会，他说会当做准备考试读资料。另外，对于蓝营高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 ，是否可能将浅蓝的票拉回丁守中阵营，柯文哲认为不会。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ，戴着工程帽，为大同区的大龙社区及市场重建工程，主持上梁典礼。柯文哲说，这次工程属于海砂屋拆除重建公办都更，因此才会出席。不过他受访时媒体关心，选举倒数，台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辩论.html" target="_blank">
  辩论
 </a>
 会时间，终于敲定在11月10日。
 <br/>
 <br/>
 台北市长 柯文哲：〝辩论既然是选举过程当中需要的，那我就把它当作好像，要准备考试一样，你要给我两三天的时间去把资料读一读，准备一下。〞
 <br/>
 <br/>
 柯文哲说，辩论会还是要模拟，不过他还是主张，台湾不需要解释问题的人与发现问题的人，而是需要解决问题的人。另外他也被问到，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 声势一路上涨，是否会担心出现外溢效应。
 <br/>
 <br/>
 台北市长 柯文哲：〝(幕僚那边，是不是一直在盯着韩国瑜的外溢效应影响，很担心他把浅蓝的票，都召回丁守中那边去？)不会啦因为其实。其实这种所为中间力量你知道，在台北市算是比较，比较成功也比较稳固。我觉得是这样啦，台湾政治上，还是不要陷入两极对立或两元政治。〞
 <br/>
 <br/>
 柯文哲也说，他认为除了台北市，政治上最有机会突破蓝绿对立的，是新竹县。而关于他因为妇联会事件在脸书道歉，柯文哲说幕僚告诉他，不可以用救国团有限的经验，去过度推论妇联会。
