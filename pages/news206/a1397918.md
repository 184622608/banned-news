### 高雄市长选战民调拉开 蓝绿阵营造势拚场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 选战，高雄市长蓝绿阵营2日晚间造势拚场，民进党高雄市长参选人陈其迈成立后援会〝母鸡带小鸡〞同台造势，国民党参选人韩国瑜举行旗津、鼓山、盐埕（旗鼓盐）造势晚会，吸引大批支持者涌入相挺。
 <br/>
 <br/>
 据台湾竞争力论坛学会11月1日最新民调，国民党高雄市长候选人韩国瑜的支持度是43.1%，民进党的陈其迈则是32.3%，领先幅度高达10.8%。另看好度方面，韩国瑜为42.5%，陈其迈为30.7%，领先11.8%。
 <br/>
 <br/>
 而此项民调由学会于10月30至31日委托全国公信力民意调查公司进行，相较于上一次10月25日学会公布的民调，韩国瑜当时领先6.3个百分点，领先幅度进一步拉大。
 <br/>
 <br/>
 学会执行长谢明辉也根据这份民调分析，指出在短短一周内，高雄绿营大崩盘的主要两项因素造成，分别为：第一、韩国瑜10月26日凤山造势活动成功，集结8万人成功拉抬气势。第二、民进党立委邱议莹对〝陪睡说〞死不认错，引来58万网友到她脸书表达愤怒，外溢效应持续扩大，让〝高雄绿营大崩盘确定〞。
 <br/>
 <br/>

   </font>
  </a>
 </center>
 <br/>
 中央社报导，陈其迈2日晚间出席左营区大新庄后援总会成立晚会，且〝母鸡带小鸡〞与左楠区民进党市议员参选人及在地里长参选人同台造势；立委刘世芳也到场站台。
 <br/>
 <br/>
 陈其迈上台一一授旗给后援会代表并致词表示，高雄要向前走，左营是火车头，投资左营是最聪明的选择，难怪富邦集团及义联集团纷纷来左营投资，将在这里盖造大型购物商场与巨蛋商圈连成一气；他当选市长后也要打造左营火车站成为有如日本新宿车站一样，生活机能完备，繁荣地方。
 <br/>
 <br/>
 陈其迈今(3)日晚上要举办劳工之夜，成立劳工后援会，他说晚上到场的〝最大咖〞就是劳工朋友。另外，国民党云林县长候选人张丽善辞去不分区立委全心投入选举之后，陈其迈表示也将辞不分区立委，明天会正式召开记者会说明。
 <br/>
 <br/>

 </center>
 <br/>
 韩国瑜2日晚则在高雄市盐埕区中心的五福四路、大智路口封街举行旗津、鼓山、盐埕（旗鼓盐）造势晚会，大批支持者涌入相挺，将会场挤爆。
 <br/>
 <br/>
 韩国瑜好不容易的进入讲台后，民众热情升到最高点。韩国瑜表示，他40年前曾在高雄当兵，但他不来高雄不知道，一来高雄吓一跳，高雄怎么变的如此没有活力，希望市民支持当选市长拚经济，让高雄这个沉睡的巨人站起来。
 <br/>
 <br/>
 韩国瑜今天竞选拜票行程重点集中在旗鼓盐地区，下午搭乘渡轮到旗津拜票与商圈的海产等业者座谈后，持续走访当地代天宫等庙宇参拜，凝聚旗鼓盐基层支持者。
