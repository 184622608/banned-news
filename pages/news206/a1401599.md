### 为败选下台 李应元林聪贤吴宏谋请辞获准
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】台湾九合一选举，民进党遭遇重大挫败，行政院长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
  赖清德
 </a>
 12月1日召集各部会首长与政委进行检讨会议，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环保署.html" target="_blank">
  环保署
 </a>
 长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李应元.html" target="_blank">
  李应元
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农委会.html" target="_blank">
  农委会
 </a>
 主委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林聪贤.html" target="_blank">
  林聪贤
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交通部.html" target="_blank">
  交通部
 </a>
 长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴宏谋.html" target="_blank">
  吴宏谋
 </a>
 为败选扛责，请辞获准。总统蔡英文听取报告后表示〝尊重〞。对于未来内阁是否全面改组，行政院发言人表示，赖清德会在适当的时机对外说明。
 <br/>
 <br/>
 中央社报导，行政院发言人Kolas Yotaka（谷辣斯．尤达卡）表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
  赖清德
 </a>
 感谢这3位首长的努力，并了解他们愿意负起政治责任，在征询许多相关人士的意见，经过一周的思考后，决定准辞。
 <br/>
 <br/>
 总统府发言人黄重谚表示，赖清德接受3位首长请辞前，已先向总统蔡英文报告，总统表示尊重。
 <br/>
 <br/>
 民进党九合一选举挫败后，赖清德第一时间请辞获总统慰留，但包括普悠玛列车事故、菜价及空污问题，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交通部.html" target="_blank">
  交通部
 </a>
 、农业委员会、环境保护署三部会首长被点名应负责下台，赖清德11月27日在立法院受访时表示，会聆听各界声音、虚心检讨，适当时机会做必要调整。
 <br/>
 <br/>
 蔡英文今天到台中市出席选后检讨座谈会时也表示，面对这次民意提出来的警示，选后开始进行深度及全面性检讨，检讨完后将进行包括府院党的人事重新布局。
 <br/>
 <br/>
 赖清德于下午2时30分邀请政务委员及全体部会首长亲赴行政院，召开施政检讨会议。政务委员与部会首长针对政府整体施政以及主管部会政策进行检讨，会议长达3小时。
 <br/>
 <br/>
 Kolas指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李应元.html" target="_blank">
  李应元
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林聪贤.html" target="_blank">
  林聪贤
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴宏谋.html" target="_blank">
  吴宏谋
 </a>
 在九合一选举开票当天已向赖清德请辞，在检讨会中再度表达辞意，赖清德在会后准辞。这也是继中选会主委陈英钤后，选后第二波辞职的官员。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环保署.html" target="_blank">
  环保署
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农委会.html" target="_blank">
  农委会
 </a>
 、交通部将于下周二上午举行交接仪式，将由政务副首长代理；环保署由副署长蔡鸿德代理署长、农委会由副主委陈吉仲代理主委、交通部由次长王国材代理部长。
 <br/>
