### 台湾宜兰普悠玛列车翻覆出轨 18人死亡
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】10月21日，美东时间凌晨4点50分，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 从新北市开往台东的〝普悠玛列车〞发生出轨意外，目前至少造成18人死亡，175人受伤。
 <br/>
 <br/>
 载有366名乘客，行进于冬山至苏新间东正线的普悠玛列车，于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 时间周日下午4点50分在苏澳新马车站前出轨翻车，目前至少造成18人死亡，175受伤，其中10人受重伤，大量救护车赶往现场抢救，伤者已分送至附近多间医院。
 <br/>
 <br/>
 事故发生现场一片狼藉，车厢倾斜翻覆、扭曲变形，车窗破裂，断裂的铁轨插进车窗中。列车疑似在翻车时扫过整个月台，月台上的水泥杆破碎散落一地，铁轨也沿着车厢弯曲而出。
 <br/>
 <br/>
 此列车共有8个车厢全部出轨，其中5个车厢严重翻覆，第4和第5节车厢情况最为严重。
 <br/>
 <br/>
 有乘客称，列车先前行驶时就有几次紧急煞车，电力不稳的状况，驾驶也曾在事故发生之前反应行驶时动力不足，确切事故原因还在调查当中。
 <br/>
 <br/>
 台湾官方已指示全力协助救援，台铁也已成立一级应变小组。
 <br/>
 <br/>
 据了解，列车上有1名美籍乘客，其余均为台湾乘客。
 <br/>
 <br/>
 新唐人记者王愉贺综合报导
</div>
