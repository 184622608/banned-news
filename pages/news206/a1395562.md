### 华航班机降落爆胎 急停跑道影响逾3400名旅客
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中华航空.html" target="_blank">
  中华航空
 </a>
 公司一架由菲律宾马尼拉飞往台湾高雄的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CI712.html" target="_blank">
  CI712
 </a>
 航班，16日上午11时36分降落
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小港机场.html" target="_blank">
  小港机场
 </a>
 时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆胎.html" target="_blank">
  爆胎
 </a>
 ，导致飞机紧急煞停在跑道上，令机上98名旅客受到不小的惊吓，所幸没人受伤，但受事件影响机场紧急关闭跑道，直到下午5时，受影响旅客约3400人。
 <br/>
 <br/>
 民航局高雄站表示，这架华航班机中午11时38分落地后，主轮胎突然破裂，紧急停在09跑道，机场消防车及救护车随即赶到班机旁待命。
 <br/>
 <br/>
 所幸未造成不幸事故，机上98名旅客都没有受伤，随后出关静候。但因事故影响，机场预计关闭至下午2时，但因航机拖离尚未完成，延长至下午5时。
 <br/>
 <br/>
 该事件也打乱了数千名旅客行程。影响航班包括国际线入境10班、1655人，出境9班1261人。国内线离站7班、300余人，到站5班、200多人，共约3416人。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆胎.html" target="_blank">
  爆胎
 </a>
 原因正由机场与及华航相关人员进一步调查中。
