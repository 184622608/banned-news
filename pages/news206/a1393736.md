### 三叶片风扇球！强颱康芮预测4、5日最靠近台湾
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月02日讯】今年第25号
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风康芮.html" target="_blank">
  颱风康芮
 </a>
 1日晚间增为强烈颱风，中央气象局表示，目前康芮强度仍持续增强，预测4日及5日最靠近台湾，北部及东半部将有明显风雨。气象局副局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑明典.html" target="_blank">
  郑明典
 </a>
 在脸书转贴一张卫星照片显示，康芮连上了三条外侧的云系，就像〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三叶片风扇.html" target="_blank">
  三叶片风扇
 </a>
 〞。
 <br/>
 <br/>
 2日上午8时，强烈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风康芮.html" target="_blank">
  颱风康芮
 </a>
 (KONG-REY)以每小时18公里速度，向西北进行。颱风近中心最大风速每秒55公尺，瞬间之最大阵风每秒68公尺，7级风半径250公里，10级风半径80公里
 <br/>
 <br/>
 气象局预测，康芮明天会来到琉球南方海面，强度持续增强。并于4日、5日最靠近台湾。但目前康芮的路径仍有不确定性，若是颱风路径较偏西靠近台湾，影响程度会更明显，区域更扩大，也有机会发布海上警报，明(3)日的路径资讯会更明确。
 <br/>
 <br/>
 气象局副局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑明典.html" target="_blank">
  郑明典
 </a>
 在脸书转贴一张卫星照片显示，康芮连上了三条外侧的云系，他说，好像三叶片的风扇。
 <br/>
 <br/>
 郑明典表示，康芮原本强度增强速率就不慢，昨(1)日下午再经历一次更快速的增强过程，昨晚8时已经达强颱等级，〝现在云图上可以看出强颱的特征，厚实均匀的眼墙云系，清晰的颱风眼。〞
