### 投完票 姚文智回立院办公室打包
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 选举投票日加上好天气，许多民众纷纷出门投票，民进党籍台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚文智.html" target="_blank">
  姚文智
 </a>
 上午9时赴士林公民会馆投票，因已请辞立委，投完票后回
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_立法院.html" target="_blank">
  立法院
 </a>
 办公室收拾东西。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚文智.html" target="_blank">
  姚文智
 </a>
 上午9时排队近50分钟后才步出士林公民会馆，由于他已在18日请辞立法委员，必须在规定的时间内空出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_立法院.html" target="_blank">
  立法院
 </a>
 办公室，而姚文智则趁空档，投完票后就驱车回到办公室整理。
 <br/>
 <br/>
 姚文智受访表示，18日请辞后虽然已办理程序，但因为投入选战没有空，因此趁空档打包、收拾物品，在规定的时间内空出办公室；他也说，东西仍然很多，但更多的东西已经先搬到竞选办公室。
 <br/>
 <br/>
 被问到会不会觉得不舍，姚文智说，舍不得是在所难免，过去也受到很多选民的厚爱，但为了表达更重要的决心（参选台北市长），必须做出这个决定。
 <br/>
 <br/>
 姚文智指出，因为长期投入内政委员会，他担任立委期间一直在做的一件事，就是要把都市更新条例修好，而此条例在上会期就差临门一脚，相当可惜。
 <br/>
 <br/>
 但姚文智也说，目前已经到了朝野协商的阶段，希望选后能够加快脚步完成，因为台北市工业化以来，需要透过都市更新来让台北翻转、改头换面；他的幕僚事后补充，由于姚文智是共同提案人，因此这项法案仍会持续进行，不会因此停摆。
 <br/>
 <br/>
 特别的是，姚文智办公室内的许多摆设反映了他的运动兴趣，包括多颗篮球、足球，以及脚踏车；还有一幅收藏的海报，上头写着〝高雄万岁〞。
 <br/>
 <br/>
 (责任编辑：卢勇信)
</div>
