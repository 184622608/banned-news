### 姚文智：当选后不允台北街头出现五星旗
------------------------

<div class="wysiwyg">
 【新唐人2018年09月23日讯】台湾彰化二水碧云寺沦为中共基地，又是违章建筑，将被县府强制拆除。台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚文智.html" target="_blank">
  姚文智
 </a>
 力挺彰化县政府，认为如果中华民国国旗不能出现在中国，那台湾也不应该出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五星旗.html" target="_blank">
  五星旗
 </a>
 ，强调如果当选市长，要禁止中共五星旗出现在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北街头.html" target="_blank">
  台北街头
 </a>
 。
 <br/>
 <br/>
 彰化二水碧云寺近年沦为宣扬中共政权的基地，彰化县府将在下周拆除违章建筑。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚文智.html" target="_blank">
  姚文智
 </a>
 力挺彰化县府，也痛斥亲共人士魏明仁殴打县府官员是无法无天。
 <br/>
 <br/>
 台北市长候选人姚文智：〝无法无天，在历史的建物上搭盖违建，然后作为共产党的训练基地，我们其实尊重言论自由，也尊重思想的自由，但是他的行动已经触犯了我们相关的法律，它的旗帜已经失去了我们国家自主的尊严跟对等的原则，这些我希望政府能够快速的来处理。〞
 <br/>
 <br/>
 姚文智认为，如果中华民国国旗无法出现在北京、上海，那依照对等尊严，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五星旗.html" target="_blank">
  五星旗
 </a>
 也不该出现在台湾，尤其是首都台北，强调如果当选，会禁止中共五星旗。
 <br/>
 <br/>
 台北市长候选人姚文智：〝如果我当台北市长，台北市这个城市不会再允许中共的国旗。(亲共人士)在街头叫嚣，然后拿着旗帜，甚至有时候拿着旗帜做为殴打别人的工具，然后在历史建筑上，在那里迎风飘扬，完全视法律为无物，这样的现象不应该再继续。〞
 <br/>
 <br/>
 姚文智已经提案修改国旗国徽法，他认为这不是单单言论自由的问题，而是攸关国家主权尊严对等的问题。
 <br/>
 <br/>
 新唐人亚太电视林嘉韦、张东旭、刘姿吟台湾台北报导
</div>
