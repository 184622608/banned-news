### 将与韩国瑜北高合作 柯P：本来就交情不错
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 今天出席北市中高龄就业博览会，在受访时表示，将安排与高雄市长当选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 见面，他们的交情本来就还不错，也认为台北高雄的合作应该不会有政治前提。
 <br/>
 <br/>
 台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ：〝那网路声量嘛，网路民调差不多啦，要赢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 、现在要赢韩国瑜实在很奇怪。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 落幕，最新民调指出，高雄市长当选人韩国瑜以62%的满意度居冠，台北市长柯文哲以61%，拿下第二名，对此，柯文哲受访时表示，这很正常，而现在有多个县市都找上韩国瑜跨党派拚观光，媒体也问到，未来台北市与高雄市有无合作可能。
 <br/>
 <br/>
 台北市长 柯文哲：〝没问题啊，我跟他本来就交情还不错，(可是如果他是以一个九二共识的前提下去要推动)，不会啦，这个台北跟高雄的合作，应该不会要有什么政治前提吧，这还好。〞
 <br/>
 <br/>
 柯文哲说，他本来就有在约韩国瑜，找时间出来见面，但现在两个人都很忙，但日后一定会跟韩国瑜见面，而媒体也关心市府的人事动向，柯文哲表示，目前局处部分本来就缺很多个，但之后大部分都会留任，至于市政顾问蔡壁如，柯文哲说，因为四年来的工作和打这场选战太累了，让她休息一下。
 <br/>
 <br/>
 台北市长 柯文哲：〝其实选完，这个还、我们只有我们，其实要休息、给大家休息一下也好。〞
 <br/>
 <br/>
 媒体追问，副市长陈景峻是否也有提想要休息？柯文哲回应，选完给大家休息一下也好，而对于副市长林钦荣请辞，是否担心对原本的计划推动不利？柯文哲说，每一个计划都有专案负责人，只是少一个协调者，现在正在找接任人选，是否从府内找人，还要想想。
 <br/>
 <br/>
 新唐人亚太电视胡宗翰、黄庭锋台湾台北报导
</div>
