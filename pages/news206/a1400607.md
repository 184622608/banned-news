### 台湾九合一选举并公投登场 投票前必看10大须知
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】规模空前的2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 与10案公投11月24日开始投票，将选出1万1047名公职人员。每人最多可领到15张选票，再加上18岁青年首度有权公投，堪称史上最复杂选举。投票时间为上午8时到下午4时，估计选举票将在晚间9时开完，公投结果将于25日凌晨2时左右出炉。
 <br/>
 <br/>
 《中央社》整理了一整套投票须知，介绍投票流程，以及投开票日所有注意事项，供大家参考。内容如下：
 <br/>
 <br/>
 <h4>
  1. 投票时间是什么时候？
 </h4>
 <br/>
 早上8时至下午4时。4时整已在投票所排队的选民，仍可完成投票。
 <br/>
 <br/>
 <h4>
  2. 投票要带什么？
 </h4>
 <br/>
 身分证、印章、投票通知单。若没带印章，可以用签名、盖手印取代；没带投票通知单也可以投票，只是必须请选务人员在一大叠名册中找出你的名字，比较费时。
 <br/>
 <br/>
 <h4>
  3. 可以领哪些票？
 </h4>
 <br/>
 满18岁：公投票
 <br/>
 <br/>
 满20岁：3至5张选举票＋公投票
 <br/>
 <br/>
 要领取哪些选举票、哪些公投票，由选举人自行决定，领票时告知选务人员即可。
 <br/>
 <br/>
 <h4>
  4. 领票及投票流程是什么？
 </h4>
 <br/>
 进入投票所后，先领选举票、投选举票；再领公投票、投公投票。选举票、公投票都可以选择不全领，但只能领票一次，不能补领。一旦离开投票所，就不能再进入。
 <br/>
 <br/>
 公投案有3个票匦，依照票匦上的标示，将3或4张公投票投入同一票匦；别担心，投错票匦仍为有效票。
 <br/>
 <br/>
 <center>
 </center>
 <br/>
 <h4>
  5. 还有什么注意事项？
 </h4>
 <br/>
 可以携带小抄，但不能给别人看，也不能将小抄留在圈票处或投进票匦。
 <br/>
 <br/>
 不可以带手机进入圈票处（请留在投票所暂时置放区），不可以拍照。
 <br/>
 <br/>
 不可以亮票，不可以撕毁或携出选票。
 <br/>
 <br/>
 不可以携带宣传品去投票，也不可以穿戴有选举或公投相关文字、图像的服饰或物品。
 <br/>
 <br/>
 与这次公投题目相关的彩虹、台湾、中华台北等图案或字样的服饰、物品能不能穿戴呢？中选会表示，需依整体情况综合判断，最好还是避免带去投票所。
 <br/>
 <br/>
 竞选活动于11/23晚间10时截止，投票日当天不可以再有任何竞选、助选活动，也不可以用简讯、Facebook、LINE拉票。

 <h4>
  6. 选举票有哪些？
 </h4>
 <br/>
 直辖市、省辖市有3张：直辖市长、直辖市议员、村里长
 <br/>
 <br/>
 省辖县有5张：省辖县长、省辖县议员、乡镇市长、乡镇市民代表、村里长
 <br/>
 <br/>
 直辖市山地原住民区有5张：直辖市长、直辖市议员、山地原住民区长、山地原住民区民代表、村里长
 <br/>
 <br/>
 <h4>
  7. 公投票有哪些？
  <br/>
  <br/>
  第7案 反空污
 </h4>
 <br/>
 你是否同意以〝平均每年至少降低1%〞之方式逐年降低火力发电厂发电量？
 <br/>
 <br/>
 <h4>
  第8案 反深澳电厂
 </h4>
 <br/>
 您是否同意确立〝停止新建、扩建任何燃煤发电厂或发电机组（包括深澳电厂扩建）〞之能源政策？
 <br/>
 <br/>
 <h4>
  第9案 反核食
 </h4>
 <br/>
 你是否同意政府维持禁止开放日本福岛311核灾相关地区，包括福岛与周遭4县市（茨城、枥木、群马、千叶）等地区农产品及食品进口？
 <br/>
 <br/>
 <h4>
  第10案 婚姻限一男一女
 </h4>
 <br/>
 你是否同意民法婚姻规定应限定在一男一女的结合？
 <br/>
 <br/>
 <h4>
  第11案 高中前不教同志教育
 </h4>
 <br/>
 你是否同意在国民教育阶段内（国中及国小），教育部及各级学校不应对学生实施性别平等教育法施行细则所定之同志教育？
 <br/>
 <br/>
 <h4>
  第12案 立专法保障同志权益
 </h4>
 <br/>
 你是否同意以民法婚姻规定以外之其他形式来保障同性别二人经营永久共同生活的权益？
 <br/>
 <br/>
 <h4>
  第13案 东奥正名
 </h4>
 <br/>
 你是否同意，以〝台湾〞（Taiwan）为全名申请参加所有国际运动赛事及2020年东京奥运？
 <br/>
 <br/>
 <h4>
  第14案 修民法保障同志权益
 </h4>
 <br/>
 您是否同意，以民法婚姻章保障同性别二人建立婚姻关系？
 <br/>
 <br/>
 <h4>
  第15案 国中小性平教育
 </h4>
 <br/>
 您是否同意，以〝性别平等教育法〞明定在国民教育各阶段内实施性别平等教育，且内容应涵盖情感教育、性教育、同志教育等课程？
 <br/>
 <br/>
 <h4>
  第16案 以核养绿
 </h4>
 <br/>
 您是否同意：废除电业法第95条第1项，即废除〝核能发电设备应于中华民国114年以前，全部停止运转〞之条文？
 <br/>
 <br/>
 公投过了可以干嘛？挺同反同公投都过了会怎么样？公投10案议题总整理看这边。
 <br/>
 <br/>
 <h4>
  8. 预计何时开完票？
 </h4>
 <br/>
 先开选举票，再开公投票。中选会预估11/24午夜12时前宣布选举结果，25日凌晨2时前宣布完整公投结果。
 <br/>
 <br/>
 <h4>
  9. 投票日上班工资怎么算？
 </h4>
 <br/>
 公民投票日为应放假日，具投票权劳工可依法放假一天。若上班，劳工可选择加班费或补休。

 <h4>
  10. 谣言不可信
 </h4>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 网路谣言不断，外传〝今年投票使用油性印泥，不容易干〞，盖选票时不要沾太多，否则折起来时容易成废票。中选会澄清，本届选举采用的印泥与往年相同，是红色快干打印台。
 <br/>
 <br/>
 此外，网路谣传〝重新印制的公报，可能号码和标题不对〞，务必细看再盖。中选会对此表示，公民投票案的编号及主文，自公告成立后即未有任何更动，绝不会发生号码跟标题不对的情事。
