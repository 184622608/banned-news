### 亚洲最大机器人竞赛 桃园登场上万人参加
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全能机器人.html" target="_blank">
  全能机器人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际邀请赛.html" target="_blank">
  国际邀请赛
 </a>
 ，日前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桃园.html" target="_blank">
  桃园
 </a>
 体育场举行，来自多国好手参与，从VEX机器人、相扑机器人、格斗机器人，到水上田径竞赛，(19日至21日)三天的活动，吸引上万人次民众参加，不少学生及家长到现场观看。
 <br/>
 <br/>
 左挥拳、右挥拳、扑倒技、侧移，格斗机器人在参赛者的遥控下，移动快速、反应敏捷，让人目不转睛。
 <br/>
 <br/>
 参赛者 傅茂智:〝虽然一开始很紧张，一开始就稳稳打，先把比分拉开，之后就可以比较轻松的打，就让心情平复一下，我本身也很少在玩电动，所以我也觉得有点不太一样，因为你是人去看它打，跟你在玩游戏那种感觉不一样。〞
 <br/>
 <br/>
 家长 傅景鸿:〝我觉得他在这方面成长满多的，因为他本身喜欢有一些挑战的东西，藉由这个机器人了解到他可以在学校，学以致用的机电整合。〞
 <br/>
 <br/>
 VEX机器人工程挑战赛，是全世界最多人参与的机器人比赛，结合机械结构及编写程式，让学生们在实作中分工学习，并培养解决问题的能力。
 <br/>
 <br/>
 参赛队伍指导老师 林奕光:〝地上那个圆盘，把它挂在柱子上面，一个圆盘就可以得2分。它因为是既竞争又合作，所以说他们在台下的时候，基本上会彼此互相，相互的去支援这样子的一个机器人，看哪些对手他可能机器人有问题，他可以去帮他维修，可是在场上的时候，等于是在享受这样一个机器人的比赛。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桃园.html" target="_blank">
  桃园
 </a>
 市长 郑文灿:〝去年决定主办的时候，蔡逢春董事长很高兴睡不着，后来我跟他说要一千人以上报名，他说他来试试看，结果我们报名2000组而且秒杀，这次有来自全世界十几个国家的机器人都来参加了，这是一个大的国际教育的潮流。〞
 <br/>
 <br/>
 桃园市与企业合作，共同举办
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_机器人竞赛.html" target="_blank">
  机器人竞赛
 </a>
 ，有近2,000组报名参加，涵盖陆、海、空和创意机器人等领域；并结合桃园30家观光工厂，及150组新创团队参与，通过发展创客学习、IoT技术、机器人教育，也期望机器人和AI智慧成为桃园发展的一大特色。
 <br/>
 <br/>
 新唐人亚太电视胡淑霞、张年洁台湾桃园采访报导
</div>
