### 台湾国庆日花车亮相 亚运选手带头游行台北街头
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双十国庆.html" target="_blank">
  双十国庆
 </a>
 日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花车.html" target="_blank">
  花车
 </a>
 ，今天（5日）对外亮相。双十节当天将搭配
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_游行.html" target="_blank">
  游行
 </a>
 活动，由60位亚运选手搭乘的车队领军，行经台北街头。不过多名亚运夺金选手因为正在忙赛事与训练，像是戴资颖正在准备台北羽球公开赛，另外举重选手郭婞淳，也都不会现身游行队伍。
 <br/>
 <br/>
 去年国庆找来世大运选手，现身
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_游行.html" target="_blank">
  游行
 </a>
 会场。今年国庆将再邀亚运好手，加入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花车.html" target="_blank">
  花车
 </a>
 游行。双十当天经过主席台后，车队从爱国西路出发，沿路经过西门町与台北车站。不过多位金牌选手，包括戴资颖、郭婞淳都不会现身。
 <br/>
 <br/>
 国庆筹委会秘书长 陈宗彦：〝因为考虑到所有的选手，在自己的训练计画跟赛事，所以我们尊重教育部跟体育署的邀请，那也尊重选手，虽然可能有一些知名的选手，没有办法来参加，不过我想在每一个项目上面的为台湾争光的选手，都值得我们为他们加油。〞
 <br/>
 <br/>
 这次国庆花车总共24辆，虽然比去年少，但仍是各有亮点。像是观光局制作巨大喔熊，上下摆动拿着手机玩自拍，上头有还不同装扮的喔熊，呈现台湾各地风貌。外交部则是把护照还有邦交国与友好国家的国旗，都放上花车。台北市选用吉祥物熊赞，打扮成飞行员，旁边除了101大楼还有会转动的摩天轮。
 <br/>
 <br/>
 台北市观传局副局长 陈誉馨：〝当天熊赞它也会跟着它化身成飞行员的这个花车，一起参与游行的活动。〞
 <br/>
 <br/>
 24辆国庆花车不仅在双十当天游行，之后将转往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自由广场.html" target="_blank">
  自由广场
 </a>
 展示直到14号，供民众一睹主题花车。
