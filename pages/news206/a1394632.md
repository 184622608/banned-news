### 台男海钓遭浪袭坠海 漂流10小时自行上岸
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】新北市一名58岁古姓男子8日搭乘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_橡皮艇.html" target="_blank">
  橡皮艇
 </a>
 前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瑞芳区.html" target="_blank">
  瑞芳区
 </a>
 鼻头风景区距岸约500公尺处钓鱼，不料大浪袭击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_船艇翻覆.html" target="_blank">
  船艇翻覆
 </a>
 ，导致古男坠入海中。古男漂流10小时后，在 9日凌晨4时许，海巡部门在落海位置外约4公里处，发现古男已自行上岸，身体状况良好。
 <br/>
 <br/>
 综合媒体报导，海巡人员昨天晚间6时33分接获通报，两名男子分别乘坐
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_橡皮艇.html" target="_blank">
  橡皮艇
 </a>
 ，在鼻头风景区距岸约500公尺处遭大浪袭击翻覆。陈姓男子上岸，却不见58岁古姓男子。
 <br/>
 <br/>
 陈男报案时曾表示，失踪的古男有穿救生衣，希望有奇迹出现。海巡人员获报后出动空侦机、海巡、岸巡、消防人员共计20余员全力搜救。但经过整夜来回搜寻均未找到人。
 <br/>
 <br/>
 直到约过了10小时，约在今天凌晨4时左右，海巡人员在落海位置4公里外的南雅明隧道，发现一名男子向他们招手求救。经核对资料后，确认他就是坠海失踪的古男。
 <br/>
 <br/>
 据了解，古男落海后因钥匙和手机都被大浪冲走，无法与友人联系。海巡人员则表示，古男经检查身体证实无大碍，不需送医，清晨5时许，海巡人员将他送至贡寮朋友住家休息。
 <br/>
 <br/>
