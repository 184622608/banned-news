### 普悠玛翻车超速酿祸？ 检方初判车速过快
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】欢迎回来。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车出轨翻覆事故，造成严重伤亡，行政院行政调查小组．到现场实地现勘后表示，依行车纪录器数据，当时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超速.html" target="_blank">
  超速
 </a>
 开到140公里，可能是肇事主因。宜兰地检署傍晚也召开记者会，初步判定事故原因在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_车速过快.html" target="_blank">
  车速过快
 </a>
 ，机械故障可能性很小，但还须进一步监识。
 <br/>
 <br/>
 宜兰地检署主任检察官 江贞谕：〝就目前案情的状况还有事故的发生，目前初步的掌握，的确问题可能是在车速方面。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 21日在宜兰新马站翻覆出轨，造成18人不幸罹难，还有百余人轻重伤，巨大伤亡的肇因引起社会关注，宜兰地检署召开记者会说明案情，初步判定事故原因是在车速。
 <br/>
 <br/>
 宜兰地检署主任检察官 江贞谕：〝这个路段应有的行车速度确实超过了。当时的时速是绝对超过80，甚至不排除100以上。〞
 <br/>
 <br/>
 检方表示，除在普悠玛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻车.html" target="_blank">
  翻车
 </a>
 事故现场调查客观证据外，也已经陆续传唤相关车上服务员、列车长、调度员、司机等厘清案情，但也还有待监验报告出炉，才会有比较正确的结论。
 <br/>
 <br/>
 宜兰地检署主任检察官 江贞谕：〝目前就迹证跟机械看起来，机械的部分显然有瑕疵或者是不正常部分，机率是很小，所以机械的部分目前等待进一步的监识结果，我们会把其它的因素一一排除。〞
 <br/>
 <br/>
 至于尤姓司机是否有超时工作的情形，也还有待进一步厘清。
 <br/>
 <br/>
 宜兰地检署主任检察官 江贞谕：〝我们调了他的这个值班纪录表，还有跟他的长官、还有相关的服务同仁做一个了解，目前看起来是没有超时工作的迹象。〞
 <br/>
 <br/>
 至于媒体报导司机关闭ATP
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超速.html" target="_blank">
  超速
 </a>
 防护系统，才会发生这起意外，检方表示，因为事件仍在侦查中，目前不便多做说明。
 <br/>
 <br/>
 新唐人亚太电视彭正干、黄庭锋台湾宜兰报导
 <br/>
