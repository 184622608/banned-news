### 台湾小琉球迎王祭典 烧王船千人参与
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】台湾有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_烧王船.html" target="_blank">
  烧王船
 </a>
 仪式，在屏东小琉球也有三年一次的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迎王祭典.html" target="_blank">
  迎王祭典
 </a>
 。七天六夜的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迎王仪式.html" target="_blank">
  迎王仪式
 </a>
 ，高潮是最后的烧王船仪式。上千名游客参与，体验独特的海岛文化。
 <br/>
 <br/>
 吉时一到，爆破的鞭炮声震耳欲聋，随着整串鞭炮跟金纸点燃，整艘王船四周熊熊大火，烈焰漫天，火光照亮整个中澳沙滩，这是小琉球三年一次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迎王祭典.html" target="_blank">
  迎王祭典
 </a>
 的最后高潮〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_烧王船.html" target="_blank">
  烧王船
 </a>
 〞，现场涌入上千游客，小琉球全岛几乎都是靠海为生，居民对自己的文化信仰相当虔诚，因此这里的迎王祭典有着更不一样的特色。
 <br/>
 <br/>
 游客黄小姐：〝他的王船祭典跟别的迎王祭典有非常的不同，因为它在地的凝聚力非常的强，然后民众向心力，也是3年一次一定要回来我们的家乡为家乡服务。〞
 <br/>
 <br/>
 小琉球居民陈先生：〝我们大家都是，家乡的一些事情都是请王爷来处理这样，然后保大家平安，所以大家都一定会回来。〞
 <br/>
 <br/>
 琉球乡三隆宫总干事蔡文财：〝我们这个迎王祭典已经有百年历史了，大部分都是一代传一代，像祖父传给父亲，父亲会传给儿子，就是大部份都长子这样子传下来的，所以到每一次的迎王祭典都会回来。其实现在工商业，如果不能请假也会在这个礼拜，会找两天三天回来拜拜这样。〞
 <br/>
 <br/>
 一大早神轿与王船由居民合力拖行绕岛一周，王船所到之处居民都虔诚下跪，双手持香祭拜，当地居民在路边、门前提供饮料、水果慰劳辛苦工作人员，王船从早到晚，从渔村走到渔港，再走到街道上，最后回到三隆宫，等到凌晨才送到中澳沙滩，举行最后的烧王船仪式。
 <br/>
 <br/>
 琉球乡三隆宫总干事蔡文财：〝以前王船是没有在烧的，就放在那个海里面到处让它去，到处去让它开到哪去。但是我们这边的捕鱼的渔船很多，有时候会碰到，那碰到有时候不知道这个礼仪，要怎么去给它求，那有一种忌讳，那忌讳后来就演变就它烧掉这样子。〞
 <br/>
 <br/>
 小琉球人口约5千多人，庙宇就接近上百座，岛上居民看天吃饭与海搏斗，仰赖的就是自己的信念与虔诚的信仰。
 <br/>
 <br/>
 收添载工作人员：〝谢谢谢谢 感谢感谢。〞
 <br/>
 <br/>
 新唐人亚太电视胡淑霞、佘志诚屏东小琉球报导
</div>
