### 普悠玛翻车事故 司机自承关闭ATP 法院裁定50万交保
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车21日发生翻车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 不幸事故，至今共208人死伤，包括18人死亡、190人受伤。宜兰地院漏夜开庭，今(23)日清晨裁定司机员尤姓男子以新台币50万元交保。
 <br/>
 <br/>
 宜兰地检署昨天晚上10时39分左右，依业务过失致死罪嫌，有串证、逃亡之虞，向宜兰地方法院声请羁押禁见驾驶列车的尤姓司机。
 <br/>
 <br/>
 主任检察官江贞谕表示，经检察官讯问以及勾稽卷内部分事证，认为驾驶员犯罪嫌疑重大，加上他的说法跟卷内事证及证人的说法显有不符，认有串证之虞；再加上这种案件，依据社会常情及卷内的个人陈述，检察官认为尤姓司机员有逃亡之虞。
 <br/>
 <br/>
 不过，宜兰地方法院审判长黄永胜受访表示，合议庭开庭后，认为检方主张尤姓司机员有串证之虞不成立，也认为以重保、限制出境、出海，就足以保全后续侦查审理的程序执行，因此驳回检方声请。并裁定尤男以新台币50万元交保。
 <br/>
 <br/>
 <h4>
  司机关闭〝列车自动防护系统〞
 </h4>
 <br/>
 有记者问到，从卷内资料来看，案发时列车时速为何？尤姓司机员是否有关闭〝列车自动防护系统〞（ATP）以及关闭原因？
 <br/>
 <br/>
 黄永胜指出，依据检方提出的证据显示，被告超速部分犯嫌重大；被告承认在大溪站附近把ATP关掉，后来就没有开启，而根据证人的说法是说，如果驾驶员关掉ATP系统后，在到达下一站，应该要重新启动；根据检方提供的证据显示，案发时的车速约时速140公里。
 <br/>
 <br/>
 黄永胜接着说，尤姓司机员担任驾驶员有5年的时间，对于该路段限速，在职务上应该明了，而根据尤的说词，仪表板显示的时速为82、83公里，但这部分跟卷证资料所显示的行车纪录时速显然不符，检方已掌握相关证据。
 <br/>
 <br/>
 据交通部铁道局网站，列车自动防护系统（简称ATP系统）是利用铁路沿线的号志设备与轨道上的信号传输设备，以电子信号与列车进行传输，监控列车的行驶状态，并依据列车〝前行距离〞及速度持续监控，确保列车不会越过〝可移动范围〞的终点。
 <br/>

 据了解，此前，台铁曾表示，尤姓司机有5年驾驶经验，且担任运转副主任职位，是需要具有相当资历且表现很好的才能担任，因此操作上无庸置疑。
 <br/>
 <br/>
 台铁6432次普悠玛列车21日下午4时50分于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏澳.html" target="_blank">
  苏澳
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新马车站.html" target="_blank">
  新马车站
 </a>
 出轨翻覆，全列车出轨，座椅移位歪倒，多个车厢变形堆叠严重，造成18人死亡、190人受伤。并造成东部铁路交通瘫痪，22日已单线抢通。
 <br/>
 <br/>
 台铁表示，事故路段预定在24日中午之前修复完成，恢复双向通车。
