### 普悠玛列车事故 台铁抢修60小时清晨恢复双线通车

 【新唐人北京时间2018年10月24日讯】

  台铁
 </a>
 6432车次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车21日在宜兰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新马站.html" target="_blank">
  新马站
 </a>
 附近翻车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 ，造成严重死伤与东部铁路交通瘫痪。经台铁动员约700名人员、耗费约60个小时的抢修后，终于恢复双向通车。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 6432车次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车21日下午4时50分行经宜兰冬山与苏澳新站的东正线时，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新马站.html" target="_blank">
  新马站
 </a>
 时发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 翻车事故，所有8节车厢全部出轨，，造成18人死亡、190人受伤送医。
 <br/>
 <br/>
 造成东部铁路交通瘫痪，经台铁各单位全力投入抢修，原预计今(24)日中午前恢复双向通车，已提前于清晨5时54分，恢复双线通车，由4102次区间车缓慢停靠进入新马站东正线，几名乘客上下车，
 <br/>
 <br/>
 台铁局长鹿洁身特别在新马站月台上迎接。鹿洁身表示，今天凌晨3时51分东正线在抢修后开始送电、试运转，当地轨道上的ATP（列车自动防护系统）也换新，保障行车安全，4102次区间车之后顺利进入新马站。因路段刚修复，目前限制时速40公里通行，他满意抢修后通车的情况。至于事故发生原因为都已进入司法调查阶段，这方面不表示任何意见。
 <br/>
 <br/>
 中央社引述搭乘4102次区间车的旅客说，列车经过普悠玛列车事故路段时，心里不会害怕，反而感谢所有曾经抢救伤者及抢救铁路交通无名英雄的辛劳，大家才又能有便捷的铁路交通。
 <br/>
