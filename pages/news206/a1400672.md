### 九合一大选大多底定 唯台北市长选情胶着
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一大选.html" target="_blank">
  九合一大选
 </a>
 已进入尾声，全台瞩目的6都中5都战情已经底定。唯独台北市市长一职的选举，直至午夜仍然战情胶着，开票结果仍持续并呈现拉锯战的情形。
 <br/>
 <br/>
 综合台媒报导，目前，台湾6都选战中已有5都决出胜负。其中，新北市、高雄市和台中市都是国民党候选人胜选；民进党仅守住了桃园市和台南市的市长之位。
 <br/>
 <br/>
 上述5市，新北市国民党籍候选人侯友宜、高雄市国民党籍候选人韩国瑜、台中市国民党籍候选人卢秀燕，以及台南市民进党籍候选人黄伟哲、桃园市民进党籍候选人郑文灿，均已宣布当选。
 <br/>
 <br/>
 唯独台北市，现任市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 与国民党籍候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 开出的选票数量差距相当小，而且持续呈现出交替上升的拉锯战情形。有台媒甚至推测，最终该市市长选战的结果可能仅在数票之差。
 <br/>
 <br/>
 截至当地时间24日晚间10点46分，台北市1563个投开票所已有1284个完成计票作业。据《民视》开票现况，柯文哲得票数为52万6745张；丁守中获得52万7258票，双方差距仅有513票。自开票以来，这两名台北市长的候选人得票数据就始终紧咬对方。
 <br/>
 <br/>
 此外，民进党台北市候选人姚文智目前得票数202,304，因其得票数量远远落后于丁柯两位对手，已出面宣布败选，并未提选第三名就宣布退出政坛。
 <br/>
 <br/>
 其他候选人的得票情况分别为：吴蕚洋4583票、李锡锟5029票。
 <br/>
 <br/>
 据台媒报导，24日的选举过程因公投案而出现延滞，因此出现了一边开票，一边还有民众在等着领票的状况。
 <br/>
 <br/>
 根据台湾〝公职人员选举罢免法〞第69条规定：区域立法委员、直辖市长、县（市）长选举的结果，如果得票数最高与次高之候选人得票数差距在有效票数的千分之三以内时，次高票的候选人需于投票日后7日内，向第126条规定之管辖法院声请查封全部或一部分投票所的选举人名册及选举票，就查封的投票所于20日内完成重新计票，并将重新计票结果通知各主管选举委员会。
 <br/>
 <br/>
 其后，各主管选举委员会应于7日内依管辖法院重新计票结果，重行审定选举结果。审定结果，有不应当选而已公告当选之情形，应予撤销；有应当选而未予公告之情形，应重行公告。
 <br/>
 <br/>
 从目前台北市长的选战情形来看，不排除出现这种双方差距在千分之三以内的情况。届时如果票数次高的候选人请求重新验票，则将会在管辖法院民事庭裁准后，在法院及竞争双方人员在场情况下重新验票。
 <br/>
 <br/>
 此外，北区各县市选战中，新竹县国民党候选人杨文科、苗栗县国民党候选人徐耀昌、宜兰县国民党候选人林姿妙、花莲县国民党候选人徐榛蔚、基隆市民进党候选人林右昌、新竹市民进党候选人林智坚、金门县国民党候选人杨镇浯、连江县国民党候选人刘增应等人获胜。
 <br/>
 <br/>
 中南区各县市选战，则是彰化县国民党候选人王惠美、南投县国民党候选人林明溱、云林县国民党候选人张丽善、嘉义县民进党候选人翁张梁、屏东县民进党候选人潘孟安、台东县国民党候选人饶庆铃、澎湖县国民党候选人赖峰伟、嘉义市国民党候选人黄敏惠等人战胜了对手。
 <br/>
 <br/>
 （记者何雅婷综合报导）
