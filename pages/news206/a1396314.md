### 普悠玛翻覆 柯文哲下令消防局配合中央调度
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】星期日傍晚近五点，台铁一列
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车，在苏澳新马车站脱轨翻覆，造成严重伤亡。台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 在今天受访表示，第一时间已经下令北市消防局，听取中央调度。
 <br/>
 <br/>
 台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ：〝台北市最大的资源还是医疗资源，所以我们在第一瞬间，是叫消防局，因为我们现在目前台北市紧急状况，是由消防局在做统一调度，所以我们由消防局连络中央的应变中心，我们就是把指挥权来，接受他们的调度。〞
 <br/>
 <br/>
 台北市长柯文哲，针对21日傍晚发生的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车意外，表示已经在第一时间，下令提供医疗、消防协助。不过临近选举，却发生重大伤亡事故，柯P也被问到，除了总统等特定政治人物，其余政治人物前往灾难现场的适当性。
 <br/>
 <br/>
 台北市长 柯文哲：〝(如果撇开特定的人物，政治人物到现场去探视会不会，到底会不会妨碍救灾。)所以我第一瞬间，就跟那个消防局讲说，我们不要没有问人家，就一大堆救护车跑去。我们还是直接，接受中央应变中心的指挥。〞
 <br/>
 <br/>
 虽然没有明说，不过对于重大伤亡事故，柯文哲认为还是要遵循SOP。另外，柯文哲也被询问，关于他
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行动竞选总部.html" target="_blank">
  行动竞选总部
 </a>
 周末的开幕园游会违规一事。
 <br/>
 <br/>
 台北市长 柯文哲：〝公园处说你那个标题叫做，行动竞选总部开幕典礼，他说这个被认定是，是竞选行动，所以就把我们保证金没收了，另外还有一个是这样，公园里面不可以摆，只能摆舞台，我们那个，我们那台行动竞选总部，他说你那个是货车不是舞台，又被罚钱。不过是这样啦，法令如果订在那里，我们还是要想办法遵守。〞
 <br/>
 <br/>
 媒体追问，对于行动竞总违规，现在他是倾向修法，还是会对此道歉，柯文哲表示，他会想办法在目前规则下合法，但现在那台行动总部，不晓得还有哪里可以用。
 <br/>
