### 学者：中共网军登堂入室 台民主最大危机
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】台湾九合一大选前夕，台湾与香港学者对话，探讨中共锐实力影响。中研院学者指出，这次大选遭遇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共网军.html" target="_blank">
  中共网军
 </a>
 攻击，是台湾民主的最大一次危机；全球20年民主倒退，是中共利用全球化所造成，呼吁台湾社会觉醒，认清中共本质，连结国际，建立防卫机制。
 <br/>
 <br/>
 台湾民主基金会，大选前三天 (21日)座谈会，邀请观察选举的香港专家对话，面对中共复合式攻击民主，台湾如何反制。
 <br/>
 <br/>
 中央研究院台史所副研究员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴叡人.html" target="_blank">
  吴叡人
 </a>
 ：〝中共对台湾的渗透攻击，锐实力渗透攻击，已经是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_登堂入室.html" target="_blank">
  登堂入室
 </a>
 ，而且现在总算引发警觉。(锐实力)通常绝大部分，是透过在地协力者在进行的。现在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共网军.html" target="_blank">
  中共网军
 </a>
 对台湾这次选举，进行的攻击，这是有史以来，我们经验到、台湾民主经验到的一次最大的危机，没有人知道该怎么做，技术上到底该怎么办，真的是不知道，我们非常的，正在一边练兵一边打仗。〞
 <br/>
 <br/>
 学者吴叡人汇整四项防卫作为：首先，必须认识中共马克思共产极权的本质。
 <br/>
 <br/>
 成功大学政治系教授 (港裔)
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梁文韬.html" target="_blank">
  梁文韬
 </a>
 ：〝你对中共一定要用一个，比绝对不信任更不信任的态度，来面对它(中共)，你才有可能做一个正确的政治判断。〞
 <br/>
 <br/>
 第二，共同揭穿锐实力隐藏的真面目，要跨国跨境分享讯息，包括与中国大陆残存的公民社会。第三，提高台湾社会的免疫力，善用主权国家地位，建立内部防卫机制、惩罚机制。第四，各党派领袖，要重申保护民主价值。
 <br/>
 <br/>
 中央研究院台史所副研究员 吴叡人：〝现在台湾有一种，所谓的保守民粹出现，他甚至认为说，我们应该要禁止游行，然后台湾社会竟然对这个没有什么反应。不要谈党派，就谈价值，这东西是不能够被容许的。〞
 <br/>
 <br/>
 吴叡人呼吁台湾各界，要看到二十年来全球民主倒退，是三十年全球化所造成的。中共极权封闭了中国，却用锐实力对民主开放国家长驱直入，这种不对称，让香港与台湾，日本、美国、欧洲都受害。
 <br/>
 <br/>
 中央研究院台史所副研究员 吴叡人：〝保卫民主的第一件事是(人民)醒来，全球化把全世界连在一起，连在一起的时候，却让反民主的力量，可以利用这连结，来攻击脆弱的民主。〞
 <br/>
 <br/>
 华人民主学院荣誉校长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑宇硕.html" target="_blank">
  郑宇硕
 </a>
 ：〝它(中共)能动员台商回来投票，当然能动员台商来做种种的经济的捐献，也可以利用台商的网络，发挥选举方面的影响力。〞
 <br/>
 <br/>
 吴叡人说，美中冲突对立，创造了一些空间，让台湾可以喘息、重整旗鼓，建立防卫机制，并连结外国、香港与中国的民间社会。
 <br/>
 <br/>
 新唐人亚太电视陈辉模、张东旭台湾台北报导
</div>
