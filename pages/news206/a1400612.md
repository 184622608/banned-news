### 投票费时约30分钟 蔡盼选举过程顺畅 马称投票处太少
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】中华民国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 与10案公投24日上午8时到下午4时投票，将选出1万1047名公职人员。总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 今上午8时30分前往新北市永和区秀朗国小投票；希望今天选举过程都能顺畅。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 由前民进党发言人张志豪陪同来到投票所与民众一起排队，过程中笑容满面，还和民众话家常，投票过程耗时近30分钟结束。
 <br/>
 <br/>
 前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马英九.html" target="_blank">
  马英九
 </a>
 由兴安里里长陈启东陪同，一早到住家附近的中华卫理公会平安堂投票，因今年选举还合并10案公投案投票，马英九这次光投票，前后就花了36分钟。他建议，公投的投票处（票匦）太少了，未来摆设应可再增加一点。
