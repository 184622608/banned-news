### 高雄大气球游行登场 拒绝丑化抗霸凌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】〝#
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_weCARE.html" target="_blank">
  weCARE
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄大气球游行.html" target="_blank">
  高雄大气球游行
 </a>
 〞今天下午登场，主办单位上午在出发点高雄市文化中心前摆设香案祈福；总指挥、前高雄市文化局长尹立说，主轴拒绝丑化、力抗霸凌。
 <br/>
 <br/>
 这场定调为欢乐嘉年华会的大气球游行，工作人员一早即为大型气球打足气，升空的小飞象、大鲸鱼、四头龙、鹦鹉、超人鸡、千里眼、顺风耳等大型造形气球，在文化中心前的高雄市五福路上集结，吸引许多家长带小朋友拍照。
 <br/>
 <br/>
 尹立表示，欢迎市民一起参加游行，拒绝网路霸凌，用正向的力量守护高雄，拒绝污衊高雄、相信未来。网路报名已有2万多人，预估下午会有4、5万人上街，凸显人民拒绝网路霸凌及仇恨丑化的意志，期待更多人守护高雄，用双脚走出希望和相信高雄。
 <br/>
 <br/>
 尹立说，游行分4大队，前3大队以市民为主，政党和候选人编排在最后大队。欢迎政党和相关人员参加，以欢乐的嘉年华会心情共襄盛举，但游行现场不允许有任何候选人旗帜和标志出现。
 <br/>
 <br/>
 〝#
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_weCARE.html" target="_blank">
  weCARE
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄大气球游行.html" target="_blank">
  高雄大气球游行
 </a>
 〞下午1时30分将在高雄市文化中心前登场，沿五福路游行经过五福桥转公园路、蓬莱路后，进入终点铁道文化园区，全长约5公里，诉求〝拒绝丑化、力抗霸凌、反对介入、停止仇恨、守护高雄〞。
 <br/>
 <br/>
 民进党高雄市长候选人陈其迈将参加这场气球游行守护高雄活动，呼吁民众不分〝颜色〞站出来，形成守护高雄的〝暖流〞。
 <br/>
 <br/>
 游行队伍规划由50辆哈雷重机车队前导开路，分成小飞象、千里眼及鹦鹉和大鲸鱼等大队，分别由大型造型气球领军前进，沿途并有九天民俗技艺团等表演，终点铁道文化园区则有包括音乐制作人陈明章及纸风车剧团等参与演出。
 <br/>
 <br/>
 weCARE高雄联盟是由一群热爱高雄的在地艺文界人士、民间团体共同发起，并在脸书成立粉丝页，陆续有人加入并贴文分享关于高雄的种种美好；基于选举期间出现污衊高雄的政治语言、造假选举模式及网路霸凌，联盟策划这项大气球游行，希望民众走出沉默螺旋、走出来一起守护高雄。
