### 川普取得重大支持 墨同意容留庇护待审移民
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】美媒周六（11月24日）报导，据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 方面透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府已赢得墨西哥新一届政府的支持，双方已达成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_协议.html" target="_blank">
  协议
 </a>
 ，允许向美国申请庇护的大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 在墨西哥境内等待审批。
 <br/>
 <br/>
 据《华盛顿邮报》（The Washington Post，以下简称《华邮》）周六报导，一些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 官员和当选总统欧布拉多（Andres Manuel Lopez Obrador）过度团队的高级成员告诉《华邮》，新总统打算在下个月就职后实施美国提出的〝留在墨西哥〞（Remain in Mexico）计划。
 <br/>
 <br/>
 这一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_协议.html" target="_blank">
  协议
 </a>
 打破了墨西哥以往在大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 问题上保持沉默的立场，为今后中美洲移民借道墨西哥进入美国设置了有力的障碍。
 <br/>
 <br/>
 根据该计划，在边境提交庇护申请的移民在案件审批过程中必须留在墨西哥境内等待，从而使美国以往〝抓了就放〞（catch and release）的移民政策有望得以终结——该政策通常允许非法移民在待审过程中留在美国境内，从而给美国带来一系列难以控制的问题。
 <br/>
 <br/>
 即将上任的墨西哥新政府内政部长科尔德罗（Olga Sánchez Cordero）告诉《华邮》，目前，他们已同意了〝留在墨西哥〞计划。不过，她把这项计划从称为〝短期解决方案〞。
 <br/>
 <br/>
 科尔德罗在采访中表示，中长期计划则是人们不再移民。她指出，墨西哥张开手臂迎接移民，可是设想一下，如果大篷车一辆接一辆地到来，那对墨西哥来说将成为问题。
 <br/>
 <br/>
 尽管目前两国还没有正式签署协议，且美国官员表示还有很多细节需要讨论，但墨西哥政府同意暂时容留大量的中美洲移民，这对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府解决当前的移民问题来说，显然是个巨大的支持。
 <br/>
 <br/>
 欧布拉多将于12月1日举行就职典礼，预计美墨两国将会很快签署正式协议。
 <br/>
 <br/>
 （记者李佳欣综合报导）
 <br/>
