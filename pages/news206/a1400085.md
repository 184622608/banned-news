### 蜜苹果红了！ 福寿山采果、赏枫一次满足
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 中央山脉与雪山山脉群峰之间的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福寿山农场.html" target="_blank">
  福寿山农场
 </a>
 ，因海拔较高，日夜温差大，种出来的苹果果心会有一圈蜜腺，非常特别，每年吸引不少游客上山品尝；今年首度以现场报名的方式，开放游客上山采果，从十月起每天都吸引上百人体验。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福寿山农场.html" target="_blank">
  福寿山农场
 </a>
 场长 巫嘉昌：〝看到这个有没有，很明显的蜜腺，因为我们这边的自然环境，就是很适合它这个蜜腺的生长，由淀粉转变成糖，自然的果糖。〞
 <br/>
 <br/>
 受到高山气候，与日夜温差的影响，福寿山
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蜜苹果.html" target="_blank">
  蜜苹果
 </a>
 心周围布满一圈蜜腺，加上栽培期间施有机肥料，脆又甜的好滋味，每年都吸引不少游客上山采果。
 <br/>
 <br/>
 民众：〝今天很开心，六年后带着女儿，重新再来一次，享受福寿山的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_采果乐.html" target="_blank">
  采果乐
 </a>
 。〞
 <br/>
 <br/>
 民众：〝口感会酸酸甜甜的，其实是很好吃，我们觉得它是很新鲜的一个苹果。〞
 <br/>
 <br/>
 福寿山农场场长 巫嘉昌：〝 苹果的外观有时候会黑点，那是自然的现象，这个皮真的是可以吃的，只要洗一洗我们没有打蜡。我们这些苹果几乎都不套袋，接受自然的阳光照射，它行光合作用，然后自然的雨水给它滋润，所以这边是最天然的苹果的园区。〞
 <br/>
 <br/>
 福寿山农场里，有一颗〝苹果王〞树，在同一颗树上结有43种不同品种的苹果，从樱桃般的小苹果，到重达近1公斤的大苹果。近48岁的〝苹果王〞，目前也已经有第二代〝苹果王子〞树，可以说是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 农业的奇迹。
 <br/>
 <br/>
 福寿山农场场长 巫嘉昌：〝海拔高度2400公尺左右的一个农场里面，嫁接这个苹果最主要是，让我们的游客来近距离的来体验，苹果生长的情形。〞
 <br/>
 <br/>
 场长巫嘉昌表示，今年因气候好、雨水少，苹果结实累累，走在园区还能闻到阵阵的苹果香，今年农场开放民众报名体验，采果当农夫，正值山上枫叶渐渐转红，有兴趣的民众，不妨上山一趟，包你
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赏枫.html" target="_blank">
  赏枫
 </a>
 、采果，一次满足。
 <br/>
 <br/>
 新唐人亚太电视胡淑霞、张年洁台湾台中采访报导
</div>
