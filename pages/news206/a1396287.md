### 普悠玛列车翻车事故 台铁：西线恢复单线双向行车
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车事故，22日上午5时最新统计，共造成205人死伤，包括18人死亡、187人受伤(10人重伤)。凌晨5时12分抢通西线恢复单线双向行车，台铁表示将加强路线检测及进行普悠玛列车特检。
 <br/>
 <br/>
 满载366名乘客的6432次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 号，下午4时50分突然在苏澳新马车站前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 翻车，由于普悠玛并未停靠该站，疑列车翻车时扫过整个月台，导致月台上的水泥杆破碎散落一地，铁轨也沿着车厢弯曲而出。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 今天早上发出新闻稿表示，全列车旅客366人，伤亡共计205人，其中18人死亡；187人受伤。
 <br/>
 <br/>
 台铁漏夜派员抢修今天清晨5时12分抢通西线恢复单线双向行车(慢行40K/H行驶)。第一班列车4102次于5时54分通过现场。
 <br/>
 <br/>
 台铁表示，待铁道局及台铁行保会完成出事故原因调查，将责成相关单位办理改善措施，台铁并同时请相关单位加强路线检测及进行普悠玛列车特检。
 <br/>
 <br/>
 台铁表示，针对死亡旅客，先行致送慰问金每人新台币10万元；受伤旅客每人5000元。后续有关受伤医疗及死亡理赔，依照相关赔偿规定办理。
 <br/>
 <br/>
 台铁表示，指定班次对号列车车票延误超过45分钟，旅客可于1年之内至各站办理全额退费，或免费搭乘同区间同等级车种列车1次；未指定班次之对号列车车票或非对号列车车票延误超过45分钟，旅客可于1年之内免费搭乘同区间同等级车种列车1次，电子票证旅客可办理0元解卡，定期票旅客可延长使用1天。
</div>
