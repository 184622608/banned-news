### 疑借2000元遭拒 清洁工狠心掐死成大女研究生
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】国立
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_成功大学.html" target="_blank">
  成功大学
 </a>
 24岁陈姓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女研究生.html" target="_blank">
  女研究生
 </a>
 28日晚间被发现陈尸在校园。警方漏夜调阅现场周边近百支
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监视器.html" target="_blank">
  监视器
 </a>
 后，认为42岁林姓男清洁工涉涉有重嫌，今(29)日清晨埋伏逮捕。经警方调查，林嫌疑因要向陈女再借2000元(约64美元)遭拒，两人起争执，一时情绪激动掐死陈女，但真正犯案动机仍待调查。
 <br/>
 <br/>
 综合媒体报导，台南警方昨晚8时50分获报，成大社会科学院教育研究所陈姓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女研究生.html" target="_blank">
  女研究生
 </a>
 被发现倒卧在成大力行校区社会科学院大楼的个别谘询室内，警消赶到时，陈女已死亡。
 <br/>
 <br/>
 台南警方勘查，陈女趴在地上，口中塞有红色抹布，现场凌乱，有打斗痕迹，颈部有勒痕，初步研判是他杀，立刻封锁现场，遗体随后移往台南市立殡仪馆。
 <br/>
 <br/>
 警方初步调查，昨天虽是假日，但陈女和其他研究生在同栋大楼的4楼研究生办公室读书讨论，陈生离开研究室后，到了晚间迟迟没有出现，教育研究所师生动员找人，才找到陈女的遗体。
 <br/>
 <br/>
 警方获报深入调查，从现场周边
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监视器.html" target="_blank">
  监视器
 </a>
 发现，林男昨天中午在成大社科院4楼和陈女同搭一部电梯到社科院2楼；直到下午1时许再从社科院大门离开，警方认定他涉有重嫌，今天清晨6时多在台南市仁德区林男的前岳父住家前逮到人。
 <br/>
 <br/>
 台南市政府警察局第五分局副分局长谢承甫说，警方在成大资源回收场找到陈女拖鞋，认定是林男行凶后丢弃，案情急转直下，才到仁德区埋伏逮人。初步判定陈女死亡时间约在昨天中午12时多，林男被逮后也坦承犯案。
 <br/>
 <br/>
 谢承甫表示，林男行使缄默权，警方将等候律师到场后进行侦讯，目前还无法得知林男行凶动机。台南地检署检察官黄铭莹得知成大发生命案后，马上赶赴现场；目前检方与法医正在相验，确认陈女死因。
 <br/>
 <br/>

 <br/>
 <h4>
  女研究生被掐死 将解剖确认死因
 </h4>
 <br/>
 警方调查，林嫌是成大社科院的清洁工与陈女认识，之前林嫌曾向陈女借1000元。昨天中午，要再向陈女借2000元，陈女回说〝我是学生而已，之前你借1000元尚未还钱，没有钱再借了〞。两人发生口角争执，林嫌为此一时动怒掐死陈女。
 <br/>
 <br/>
 台南地检署检察官上午会同警方与死者家属在市立殡葬所相验陈女大体。检方相验后表示，检视陈女身体外观，颈部有明显掐压痕迹、口中被塞抹布，其他部位无明显伤势，已请法医研究所安排法医在明天上午解剖陈女大体，进一步确认死因。
 <br/>
