### 花之礼赞迎花博 台日四大美馆联展开箱
------------------------

<div class="wysiwyg">
 【新唐人2018年09月28日讯】文化部和台中市政府文化局携手，联合
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国立故宫博物馆.html" target="_blank">
  国立故宫博物馆
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东京富士美术馆.html" target="_blank">
  东京富士美术馆
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奇美博物馆.html" target="_blank">
  奇美博物馆
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国立台湾美术馆.html" target="_blank">
  国立台湾美术馆
 </a>
 四大馆，预定于台中市花博期间合作展出典藏精品，9月27日举行展品开箱仪式，让观众先睹为快。
 <br/>
 <br/>
 国立故宫博物院开箱作品，精选北宋青磁莲花式温碗、清18世纪的雕犀角梅树香插、雕犀角花篮、以及清高宗御笔《桃夭》收入《御笔诗经图》，展现故宫在器物及梳化上的收藏特色。
 <br/>
 <br/>
 国立故宫博物院副院长 李静慧 ：〝我们来了40组件里面，都是第一次到台中，然后有九成以上是，第一次出故宫大门。〞
 <br/>
 <br/>
 而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东京富士美术馆.html" target="_blank">
  东京富士美术馆
 </a>
 ，带来了浮世绘知名画家歌川丰国的五联幅作品《新吉原樱景》；
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奇美博物馆.html" target="_blank">
  奇美博物馆
 </a>
 则展出镇馆之宝之一，法国19世纪学院画家寇特(Pierre Auguste COT)画作〈黛妮希雅〉（Dionysia）。四大馆精选中的精华藏品首度曝光，不同的馆藏展现了不同文化与族群的特殊性。
 <br/>
 <br/>
 东京富士美术馆馆长 五木田聪：〝我们这次的展览，主要是展出从17世纪的江户时代一直到20世纪昭和时代，日本的绘画和版画，总共有25件作品。〞
 <br/>
 <br/>
 奇美博物馆副馆长 郭玲玲：〝我们为了叩合这一次的主题，奇美也挑了30件作品，其中有2件应该刚才也许大家都有看到了，就是我们的馆内最吸睛的一个雕塑，就是《花神》Flora，然后还有《春的降临》。〞
 <br/>
 <br/>
 策展人 萧琼瑞：〝每一个馆设的长期的经营当中，都展现了某一种文化上的特色。〞
 <br/>
 <br/>
 这次以〝花卉〞为主题
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合策展.html" target="_blank">
  联合策展
 </a>
 ，展品综合古今中外，包含绘画、器物、雕塑、空间装置、新媒体艺术，共168件作品，预定10月6日于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国立台湾美术馆.html" target="_blank">
  国立台湾美术馆
 </a>
 盛大开展。
 <br/>
 <br/>
 新唐人亚太电视王媛、林莉台湾台中采访报导
</div>
