### 丁守中声请重新计票 逾千投开票所20天内完成
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 落幕，台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 连任成功，国民党台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 以3254票之差落败，丁守中声请重新计票，今(3日)启动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_验票.html" target="_blank">
  验票
 </a>
 ；台北地院表示，法律规定查封后20天内要完成验票，但因有1000多个投开票所，无法预估几天内验完，但会竭力在规定时间内完成。
 <br/>
 <br/>
 中央社报导，台北地方法院法官与两造律师等人，11月29日下午到台北市选举委员会完成查封相关作业；北院法官今天早上8时起进行启封作业，并把选票从台北市选委会（9楼）搬到10楼的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_验票.html" target="_blank">
  验票
 </a>
 场地（信义区公所大礼堂）。
 <br/>
 <br/>
 北院表示，目前规划以50组人力投入，每组（桌）配置法官一人、书记官一人，以及丁阵营律师、柯阵营律师、2位选务人员。
 <br/>
 <br/>

   </font>
  </a>
 </center>
 <br/>
 北院行政庭长黄柄缙今天受访说明验票流程，包括选务人员拿出选票后，会经过法官、两造代理人面前让他们看，再由另一位选务人员收起来；验票过程中，选务人员必须配戴白色手套，避免选票受污染，选票不会经过两造阵营、法官及书记官的手。
 <br/>
 <br/>
 此外对于有争议的选票，会另外做纪录，现场法官也会做初步认定，但合议庭保留最终的判断。
 <br/>
 <br/>
 黄柄缙也说，原则上法律规定是在查封后20天内要完成验票，但北市有1000多个投开票所，无法预估在几天内可以验完，但会竭力在规定时间内完成；每天验票的时间，原则上上午部分是到中午12时30分，下午则是从1时30分至5时30分。
 <br/>
