### 陈菊、赖清德败选请辞 蔡英文慰留稳政局
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】中华民国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 民进党惨败，总统府秘书长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈菊.html" target="_blank">
  陈菊
 </a>
 、行政院长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
  赖清德
 </a>
 纷纷向总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 提出辞呈。经与蔡英文恳谈后，两人均接受慰留，决定在此艰难时刻与团队共同承担、共同努力，稳定选后政局。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
   蔡英文
  </a>
  再三慰留
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈菊.html" target="_blank">
   陈菊
  </a>
  决定留任
 </h4>
 <br/>
 陈菊24日透过脸书指出，选举结果很抱歉让支持者失望了，她已经口头向总统请辞总统府秘书长职务，深刻检讨、反省，〝感谢每一位愿意给我们机会的人民〞。
 <br/>
 <br/>
 这2天，传出陈菊辞意甚坚，并请幕僚打包办公室与台北住处，要返回高雄。
 <br/>
 <br/>
 不过，在蔡英文以考量国家政局稳定与各项重大政策的持续推动，邀集陈菊恳谈，再三慰留，陈菊决定留任，全力襄助总统在扩大社会对话以及各项革新工作上的努力。
 <br/>
 <br/>
 民进党在高雄由谢长廷担任市长8年后，再由陈菊任市长12年。黄重谚表示，现在市民选择改变，民进党虚心接受，也以最高标准来自我策进。
 <br/>

 </center>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
   赖清德
  </a>
  ：盼政务推动不中断 决定留下
 </h4>
 <br/>
 行政院长赖清德也因九合一败选请辞，不过，赖清德上午在记者会表示，每张选票都是对施政的检验，选举结果表示对施政不满意，因此他第一时间道歉，也向总统口头请辞。
 <br/>
 <br/>
 赖清德说，经过昨天下午，他跟总统讨论，最后决定留下来继续努力，一方面希望行政院政务推动不至于中断，也希望协助稳定政局，后续行政院会负起责任，滚动检讨各项政策，在短期内完成公投相关法案，积极回应人民期待。
 <br/>
 <br/>
 赖清德也恭喜新当选的县市长，行政院承诺会与每个地方政府合作，共同建设国家、造福民众。
 <br/>
 <br/>
 至于内阁是否改组，赖清德表示，〝会在适当时间做必要调整，以回应人民期待〞。
 <br/>

   </font>
  </a>
 </center>
 <br/>
 立法院长苏嘉全表示，一场选举选输不是单一事项，是非常多因素累积而成，如果大家都认为是自己的因素就辞职，〝我看大概每个人都要辞职〞，他认为现阶段还是以政局稳定为重，朝对的方向做，相信民进党还是会得到民心。
