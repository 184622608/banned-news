### 全联与桃园实物银行合作 惜食分享计划
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】台湾每年从生产到消费端，至少浪费368万吨食物，为了响应减少食物浪费，知名连锁超市与桃园市社会局实物银行合作，媒合市内36家门市的即时食材，捐赠老人供餐点及弱势孩童陪读班，帮助弱势族群。
 <br/>
 <br/>
 桃园市市长郑文灿与知名连锁超市董事长林敏雄，签署合作备忘录，提供36间门市效期内的食材，做为老人供餐点及陪读班供餐材料。
 <br/>
 <br/>
 桃园市长郑文灿:〝安家实物银行又增加了一个面向，就是生鲜食品也可以送给需要的地方，我相信这计划，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全联.html" target="_blank">
  全联
 </a>
 社桃园的36个据点都参与了，而且运送的过程很严谨，也符合规范，那都是低温配送。〞
 <br/>
 <br/>
 全联实业股份有限公司董事长林敏雄:〝真的非常感谢郑市长，能够出面跟我们社会局的古局长能出面 ，把这个实物银行带上来，因为这个是浪费，以前我的老一辈一直在讲，一粒饭都要给它吃进去，不能浪费。〞
 <br/>
 <br/>
 中华基督教救助协会媒体公关李淑美:〝 快过期的做一些整理，整理完之后会放在冰桶上面，会把它封起来，再由我们的志工们再把它带回去，之后可能第二天就可以煮给这些老人们，或者是孩子们那些陪读班的孩子们 ，能够享用晚餐。〞
 <br/>
 <br/>
 全联董事长表示，为了减少食物浪费，全联从2017年开始与中介单位，进行惜食合作，既不浪费食物，也能真正帮助到弱势族群。
 <br/>
 <br/>
 新唐人亚太电视胡淑霞、张年洁台湾桃园采访报导
</div>
