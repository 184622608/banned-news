### 卢秀燕初访花博 将设市民专用通道
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】欢迎回来。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台中市长.html" target="_blank">
  台中市长
 </a>
 当选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢秀燕.html" target="_blank">
  卢秀燕
 </a>
 ，选前承诺，如果当选市长，明年元旦起，台中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_市民.html" target="_blank">
  市民
 </a>
 可不限次数免费参观
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花博.html" target="_blank">
  花博
 </a>
 ，今天早上（1日），卢秀燕亲自勘查花博后里马场及森林园区，了解园区内的动线。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢秀燕.html" target="_blank">
  卢秀燕
 </a>
 穿着牛仔裤和蓝色T恤，跟一般游客一样背着包包，从丰原火车站搭乘台铁区间车到后里，展开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花博.html" target="_blank">
  花博
 </a>
 初体验。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台中市长.html" target="_blank">
  台中市长
 </a>
 当选人 卢秀燕：〝我已经宣布元旦，就是明年一月一日开始，我们要台中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_市民.html" target="_blank">
  市民
 </a>
 免费，我们为了让动线更加顺畅的话，初步的构想是要把市民的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专用通道.html" target="_blank">
  专用通道
 </a>
 ，跟非台中市民的通道，要把它分开来，因此今天要稍微去了解一下动线。〞
 <br/>
 <br/>
 一路上民众，见到新科市长，频频向她打招呼，不时挤上前要合照，工作人员导览参观各展区，也看了厕所和餐厅等设施。
 <br/>
 <br/>
 台中市长当选人 卢秀燕：〝明年元旦开始，因为游客应该会激增，也因此呢这个厕所够不够用，可能回去我们要再请同仁稍微评估一下。〞
 <br/>
 <br/>
 为兑现竞选承诺，让民众不要在空污之中逛花博，卢秀燕重申停止中电北送救空污。
 <br/>
 <br/>
 台中市长当选人 卢秀燕：〝在大选完之后，我有看到赖清德院长提到说，因为中部的空污，很多都是从因为为了中电北送的关系，所以台北不排除他们的核电机组，台北的电厂要能够恢复发电供电，我觉得这是好消息。〞
 <br/>
 <br/>
 卢秀燕参观世界花卉博览会后表示，肯定台中市长林佳龙团队与台中市政府同仁做得很好。
 <br/>
 <br/>
 台中市长当选人 卢秀燕：〝当然也谢谢前胡志强市长争取到台中来，更谢谢林市长的团队把它建设起来，我觉得做得很好做得满好的。〞
 <br/>
 <br/>
 选举结束，如何带领台中继续向前走，考验下任台中市长的智慧。
 <br/>
 <br/>
 新唐人亚太电视王媛、林莉台湾台中采访报导
</div>
