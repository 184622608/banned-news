### 桃园77艺文町也有日本风 历史建筑展新生
------------------------

<div class="wysiwyg">
 【新唐人2018年09月23日讯】桃园最近多了一个热门打卡景点。叫做〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桃园77艺文町.html" target="_blank">
  桃园77艺文町
 </a>
 〞，是由日治时期的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警察宿舍.html" target="_blank">
  警察宿舍
 </a>
 ，改造而来的日式文创园区，古色古香，让人有置身日本的感觉。
 <br/>
 <br/>
 日治时期
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警察宿舍.html" target="_blank">
  警察宿舍
 </a>
 的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桃园77艺文町.html" target="_blank">
  桃园77艺文町
 </a>
 〞，全区有四栋建筑物，是桃园少数保留较完整的宿舍群，市政府以修旧如旧的活化方式，保留
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_历史建筑.html" target="_blank">
  历史建筑
 </a>
 风貌。
 <br/>
 <br/>
 桃园市长 郑文灿:〝桃园77艺文町在试营运了一个月，已经有35000人来体验过，可以讲非常受到欢迎，也是历史建筑修复活化的典范，那这个名称是由市民投票产生的，77代表是中正路77巷，艺文町代表它是艺文的基地。〞
 <br/>
 <br/>
 〝桃园77艺文町〞结合日治时期建筑和中正路77巷的意涵，采取委外营运方式，提供手作坊、展览区、户外表演、美食餐厅进驻，化身多元艺术创作的交流平台，带动桃园文化资产活化。
 <br/>
 <br/>
 游客 斯小姐:〝这个空间就很舒服，进来到这房子里面，都有浓浓的桧木香，看一些手作的东西，其实就还满疗愈的，很适合来走一走。〞
 <br/>
 <br/>
 游客 张小姐:〝这边是以前警察宿舍所改建的，它改建成日式的风格，里面还满多元化的，有餐厅、有手作、有美食，环境很丰富，还满喜欢的。〞
 <br/>
 <br/>
 距离桃园火车站步行仅约五分钟的桃园77艺文町，日式木造警察宿舍，周围有多棵大树环绕，闹中取静的环境，像是城市里的市外桃源，已经成为不少民众打卡的新景点。
 <br/>
 <br/>
 新唐人亚太电视胡淑霞、张年洁台湾桃园采访报导
</div>
