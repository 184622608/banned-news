### 台北市长柯文哲人气飙升 选总统民调支持率高居榜首
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】成功连任台北市长后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 近期人气持续上升。有台媒近日针对台湾2020总统大选进行民意调查，结果无党派的柯文哲获得的支持率高居榜首。
 <br/>
 <br/>
 台湾媒体东森新闻云近日就2020年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾总统选举.html" target="_blank">
  台湾总统选举
 </a>
 的问题，进行了一次候选人支持度的民意调查。该媒体从11月27日至11月29日 通过网络问答和电话问答，搜集了2196份有效样本。最终得出的数据显示：有27.5%的受访者表示支持现任台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 竞选总统；10.3%的受访者支持国民党的新北市长朱立伦；韩国瑜得到的支持率为9.2%；现任总统蔡英文的支持率则仅为7.8%。
 <br/>
 <br/>
 此外，针对国民党下任主席候选人的民意调查显示：韩国瑜以43.9%的支持率居于榜首；其次是朱立伦的15.3%；现任主席吴敦义的支持率仅为6.6%。
 <br/>
 <br/>
 至于民进党主席的候选人民意调查则显示：赖清德以获得31.1%的支持率而居于首位，其次是高雄市长落选人陈其迈获得16.4%的支持率，现任总统府秘书长陈菊仅获得4.3％。
 <br/>
 <br/>
 今年以来，台湾舆论界敦促或表态支持柯文哲竞选台湾总统的呼声渐高，但柯文哲本人却一直表示自己还没有这方面的计划。
 <br/>
 <br/>
 日本媒体《朝日新闻》今年10月9日曾在采访柯文哲时提到，许多人呼吁他出来参选总统。当时柯仍然表示，〝一切还没有计划〞。不过他同时也分析称，自己现在之所以受到舆论的瞩目，主要是因为台湾经过两次政党轮替后，台湾民众发现不论是国民党还是民进党执政，其结果都是一样的，台湾存在的许多问题还是得不到解决。
 <br/>
 <br/>
 柯文哲说：过去台湾人没有第三个选择，现在有人把他看作一个新的选项，这对台湾政治是一个〝大冲击〞。
 <br/>
 <br/>
 两天后（10月11日）《朝日新闻》发表了一篇题为《台湾将吹起无党派旋风？》的报导。这篇报导称，台湾过去以国民、民进党为核心的两党政治正在异变，两党支持率同样低迷，而正在寻求连任的台北市长柯文哲的存在感增大，开始被视为未来可能的总统人选。
 <br/>
 <br/>
 （记者欣然综合报导/责任编辑：秦锋）
