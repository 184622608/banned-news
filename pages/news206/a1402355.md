### 华为恐成下个中兴 美中5G版图将大挪移？
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】加拿大逮捕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 CFO，业界震动。专家分析，下一步美国不排除对华为祭出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁售令.html" target="_blank">
  禁售令
 </a>
 。那么，对供应链和产业版图有哪些影响，成为后续观察的重点。
 <br/>
 <br/>
 孟晚舟在加拿大遭逮捕，时间点就在贸易战看似休兵之际，外媒提醒，美中之间不光贸易争端，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5G.html" target="_blank">
  5G
 </a>
 领域话语权争夺，才是更大关键。
 <br/>
 <br/>
 台经院产业分析师 邱昰芳：〝其实
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 ，除了透过这个关税成本的一个加征以外，美国其实采取了一系列这个非关税的一个措施。就是希望在5G即将商用化的这个时间点，能够力求保有在5G产业的一个主导权。其实是有可能再度重演，就是美国会禁止相关的美国厂商出售，包括晶片等、相关的一个零组件给中国的厂商。〞
 <br/>
 <br/>
 专家分析，美国未来不排除，比照
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴.html" target="_blank">
  中兴
 </a>
 通讯，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 制裁晶片、技术禁售，或是采取高额罚金，预料华为手机，5G布建发展，都会受到遏阻。美国国会发布报告显示，中共计划利用5G过渡期，对数十亿手机进行情报搜集、手机控制，窃取商业利益。目前美国扩大对华为5G设备抵制，美、纽、澳，确定禁用华为网路设备，日本、加拿大、英国，也可能跟进。
 <br/>
 <br/>
 台经院产业分析师 邱昰芳：〝所以你可以看到在核心的设备上面，其实华为已经面临到一定程度的一个阻碍，这也使得他们在这个5G产业的布局上面，整个脚步会相对的，出现一个落后的一个情况。它(华为)在5G设备的一个，核心设备的市场的渗透率，可能会不如原先所预期的强劲。〞
 <br/>
 <br/>
 市场调查，华为基地台设备全球市占率30-35%、光通讯设备全球市占率约 40-45%，对台湾供应链，晶圆代工、封测、消费性零组件，网通设备，影响巨大，台积电6日跌幅2.65%，联发科暴跌6.13%，大立光更是跌停，好在周五恐慌心理纾缓，股价纷纷反弹。
 <br/>
 <br/>
 台经院产业分析师 邱昰芳：〝那因为5G设备采购，那目前其实才是刚开始，其实整个规模是相对的一个有限，但是这个部份的一个版图的变化，会去影响后续，相关供应链厂商的一个订单的一个来源。〞
 <br/>
 <br/>
 美中角力转向非贸易领域，专家指出，华为全球核心供应链厂商共92家，33家来自美国，事件美中双方都会受到冲击，5G产业版图，或将有所转变，也将值得关注。
 <br/>
 <br/>
 新唐人亚太电视曾奕豪、沈唯同台湾台北报导
</div>
