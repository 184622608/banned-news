### 台湾政治现状 柯P：取决于你爸爸是谁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 的网路新节目〝阿伯面对面〞，日前推出第二集，柯P提到台湾的政治现状就是，你会不会成功，不是取决于多努力，是取决于你爸爸是谁，遭质疑是在影射
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_靠爸族.html" target="_blank">
  靠爸族
 </a>
 ，但他周六（20日）强调，不是在说政治人物，而是台湾目前普遍现象，但也直言30年内，一定不会有三级贫户当总统！
 <br/>
 <br/>
 主持人：〝一起启动，拔出我们的剑，往上挥。〞
 <br/>
 <br/>
 拔起长剑，宣示北市一年一度的商圈活动正式展开，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 还和参与学生一起来个大合照！
 <br/>
 <br/>
 台北市长 柯文哲 VS. 学生：〝(你要跟市长说什么？)柯P冻蒜。〞
 <br/>
 <br/>
 学生高喊冻蒜，柯P有点尴尬，脸上笑容却藏不住，最近他推出的网路新节目阿伯面对面，找民众上车和他聊天，获得不少好评，但二集聊到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贫富差距.html" target="_blank">
  贫富差距
 </a>
 问题时，他却冒出这些话。
 <br/>
 <br/>
 台北市长 柯文哲：〝台湾的政治现状就是，你会不会成功，不是取决于多努力，是取决于你爸爸是谁。〞
 <br/>
 <br/>
 台北市长 柯文哲：〝我没有说政治人物啦，我是说台湾目前的普遍现象啦，想想看，像类似王永庆白手起家的创业者，现在台湾如果在40岁那一级的，全部靠自己白手起家创业的有没有，很少了啦，老实讲你把柯文哲扣掉，台湾政坛可以这样一路杀上来的，不多啦。〞
 <br/>
 <br/>
 柯P用自己经历的政治路，强调说的是实话，不过，他口中的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_靠爸族.html" target="_blank">
  靠爸族
 </a>
 ，到底是谁啊？
 <br/>
 <br/>
 台北市长 柯文哲：〝哈哈哈哈，你这不是害我吗，(连什么的吗？还是陈什么的？)，没有，我在讲不是说哪个政治人物，是台湾社会普遍现象，如果你要完全白手起家满困难的，最近我对教育界演讲都说，30年内不会有三级贫户当总统，所以我说阿扁是最后一个，在可见的30年内，这是我们要去改革的社会问题。〞
 <br/>
 <br/>
 要媒体不要害他，而柯文哲第一场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_造势活动.html" target="_blank">
  造势活动
 </a>
 ，在北投
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_七星公园|违停.html" target="_blank">
  七星公园|违停
 </a>
 展开，却传出内部讨论，如果效果不好的话，后续场次可能就得喊卡！
 <br/>
 <br/>
 柯文哲说：〝大家高兴就好，现在就办个园游会，大家高高兴兴、快快乐乐就好了，老实讲我现在每次遇到要喊冻蒜，都觉得很尴尬，(内部好像评估办不好就不办第二场了？)对啊，这个本来就是这样，做得好才做，做不好就收一收，想另外一个方式。〞
 <br/>
 <br/>
 强调大家开心就好，只不过，活动都还没开始，就因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行动竞选总部.html" target="_blank">
  行动竞选总部
 </a>
 ，违规开进公园里头，上午已经被开违停红单。
 <br/>
