### 台北市长辩论 民进党姚文智谈中共威胁
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 首场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电视辩论.html" target="_blank">
  电视辩论
 </a>
 4日登场，独缺台北市长柯文哲，民进党台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚文智.html" target="_blank">
  姚文智
 </a>
 在辩论时直接谈到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 危胁，并表示国民党高雄市长候选人韩国瑜的声量，就是仰赖中共网军对台湾的渗透。
 <br/>
 <br/>
 民进党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚文智.html" target="_blank">
  姚文智
 </a>
 ：〝维护民主的价值，它就是一个最重要的一件事情，中国已经透过各种在网路世界的攻击，它有30万的网军、有200万的五毛党，每天有10万笔对我们国安单位的攻击，在这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选举.html" target="_blank">
  选举
 </a>
 ，已经变成它们的练兵场。〞
 <br/>
 <br/>
 首场台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电视辩论.html" target="_blank">
  电视辩论
 </a>
 登场，民进党台北市长候选人姚文智一开始先将炮火指向国民党高雄市长候选人韩国瑜，说
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 不断透过文攻武吓威胁台湾，代表人物就是韩国瑜，仰赖中共网军对台湾的渗透，成为台湾反智的网红，对于姚文智把炮口对准韩国瑜，国民党台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 ，拿姚文智的手势回批。
 <br/>
 <br/>
 国民党台北市长候选人 丁守中：〝刚开始的时候大家都叫我们举自己的号码，为自己加油，姚文智你们注意到他吗？他举的是4号，难道真的是在打假球吗？〞
 <br/>
 <br/>
 民进党台北市长候选人 姚文智：〝一开始的时候就比了4，就是心里一直想说，这个4号都不来，真的是太可恶了，我真的觉得是，我真的觉得辩论会不来，事实上是在修理台湾的民主。〞
 <br/>
 <br/>
 在台上误比出柯文哲的四号，竞选号次三号的姚文智连忙澄清，而柯文哲本人独缺辩论会，更是成为4位台上候选人的质疑焦点。
 <br/>
 <br/>
 国民党台北市长候选人 丁守中：〝觉得最遗憾的还是，柯文哲缺席这么重要的电视辩论，那也表示他的市政禁不起检验。〞
 <br/>
 <br/>
 无党籍台北市长候选人 李锡锟：〝他胆小鬼、跑掉了，所以当然就比较有一点像开枪没有靶一样。〞
 <br/>
 <br/>
 11月10日将再有一场台北市长候选人电视辩论，五名候选人将全员到齐，将采三段式辩论，但没有开放公民团体或媒体提问。11日、18日还有公办政见会，随着投票日接近，选民也把握机会深入检验。
 <br/>
 <br/>
 新唐人亚太电视曾奕豪、黄庭锋台湾台北报导
</div>
