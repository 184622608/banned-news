### 韩国瑜打造高雄 布局小内阁 力邀郭台铭投资
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 当选高雄市长，在胜选当天表达人事布局没有任何政治考量，也开出小内阁条件，必须清廉、专业、热情。据悉，韩国瑜昨(25日)就会晤了一些具专业背景的人士；而为了要打造高雄成为〝全台首富〞，前高雄县长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨秋兴.html" target="_blank">
  杨秋兴
 </a>
 今(26)日证实，韩国瑜已和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸿海.html" target="_blank">
  鸿海
 </a>
 董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭台铭.html" target="_blank">
  郭台铭
 </a>
 通电话力邀投资高雄，双方展现最大的诚意。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 多次主张高雄就是要面向海洋、面向国际及世界，才能拚出自己的一条路，所以对3位副市长开出的条件，要外语好、懂工程、会行销、清廉、热情不可或缺。
 <br/>
 <br/>
 自韩国瑜参选以来，前高雄县长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨秋兴.html" target="_blank">
  杨秋兴
 </a>
 就大力协助，尤其杨秋兴娴熟高雄县市合并前的县境，还有工程背景和对城市的建设方向也能补强韩国瑜，被认为有能力快速协助韩国瑜团队上手的副市长人选。
 <br/>
 不过，杨秋兴本人尚未证实。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-26/p9143731a927820676.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-26/p9143731a927820676-ss.jpg"/>
   <br/>
   <font size="-1">
    图为，前高雄县长杨秋兴接受媒体访问。(视频截图)
   </font>
  </a>
 </center>
 <br/>
 杨秋兴曾是民进党籍的高雄县长，在高雄县市合并后脱党参选高雄市长，后来又加入国民党代表国民党选高雄市长失利而淡出政坛。今年为协助国民党于高雄市长这一仗，杨秋兴向党主席吴敦义力荐韩国瑜到高雄市开疆辟土，也算是促成韩当选市长的幕后功臣之一。
 <br/>
 <br/>
 除了杨秋兴的动向受关注，前高雄市议员、前立委林国正也被认为是人选之一。林国正有农业经济、财务管理专业背景，他对高雄市的建设发展方向很清楚，论述能力强，尤其受到前立法院长王金平的肯定，前高雄市议长许昆源的倚重。
 <br/>
 <br/>
 另外，韩国瑜幕僚透露，曾为韩国瑜站台背书也公开允诺为高雄市政把脉，前行政院长张善政、前内政部长李鸿源将会担任高雄市的经济及治水总顾问。
 <br/>
 <br/>
 <center>

   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-26/p9143681a944243430-ss.jpg"/>

    国民党高雄市长候选人韩国瑜（中）19日晚间由前行政院长张善政（右2）、前内政部长李鸿源（左2）等人陪同出席2018高雄市长候选人辩论会后记者会，对辩论会内容进行说明。(中央社)
   </font>
  </a>
 </center>
 <br/>
 <h4>
  全台首富 杨秋兴：韩国瑜和
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭台铭.html" target="_blank">
   郭台铭
  </a>
  通电话
 </h4>
 <br/>
 至于韩的口号要打造高雄为〝全台首富〞。杨秋兴证实，韩国瑜已和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸿海.html" target="_blank">
  鸿海
 </a>
 董事长郭台铭通电话力邀投资高雄，双方通了近3个小时的电话，韩力邀郭台铭投资高雄。双方展现了最大的诚意，也会安排见面。
 <br/>
 <br/>
 杨秋兴说，他在八八风灾重建时和郭台铭结缘，了解郭台铭是有情义的人，也愿意参与台湾的建设及投资。他说，当时郭台铭就答应要投资高雄。
 <br/>
 <br/>
 杨秋兴表示，即使时隔多年，郭台铭也说，投资并帮助高雄是件简单的事，届时也会带来国际性或周边关联产业投资。
 <br/>
 <br/>
 高市府经济发展局表示，鸿海科技集团在高雄软体科学园区内已有投资，员工数310人，群创光电在南科路竹科学园区内有3600名员工。
 <br/>
 <br/>
