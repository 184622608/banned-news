### 美中贸易战效应 台厂自行车高阶产线移回台湾
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】台北国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自行车展.html" target="_blank">
  自行车展
 </a>
 今天盛大开展，台湾具竞争力的产业供应链，以及产业聚落，让国际自行车展，成为引领产业发展趋势的重要国际平台。外界也高度关注，美中贸易战，对产业影响，台湾自行车大厂直言，高额关税下，不可能在中国生产外销美国，会将高阶产线移回台湾。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巨大.html" target="_blank">
  巨大
 </a>
 集团董事长 杜綉珍：〝碳纤维的这个环，两个就要超过美金两千块。〞
 <br/>
 <br/>
 台湾自行车大厂巨大董座，亲自介绍旗下高阶车款，近年电动车蔚为潮流，台湾两大自行车厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美利达.html" target="_blank">
  美利达
 </a>
 以及巨大，都展示旗下最新电动自行车，看好未来商机发展。
 <br/>
 <br/>
 巨大集团董事长 杜綉珍：〝我们在电动车方面的努力，已经超过三十年。整个集团，电动车(营收占比)达到百分之四五十，应该是没有问题。〞
 <br/>
 <br/>
 台湾自行车输出业同业公会理事长 曾崧柱：〝未来十年我们更迈向，新的电动自行车蓝海到来。研发创新是自行车产业关键DNA，更是国际征战上，不可或缺的基石。〞
 <br/>
 <br/>
 台湾素有自行车王国的美誉，电动自行车出口年成长5成，中高阶自行车出口全球市占29.31％，台湾自行车整车出口平均单价已达658美元，反映台湾自行车产业在国际的竞争力，因应美中贸易战，台湾厂商调整布局，纷纷将欧美高阶产品移回台湾。
 <br/>
 <br/>
 巨大集团董事长 杜綉珍：〝本来关税以外，再加上25%，也就是等于，在中国生产自行车，输出到美国是不可能的。当然我们会先回到台湾，台湾一定越来越重要，因为高端东西在这边生产。〞
 <br/>
 <br/>
 以台湾作为重要生产据点，今年台北国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自行车展.html" target="_blank">
  自行车展
 </a>
 一大特色，就是结合智慧装置，搭配物联网（IoT）。今年台北国际自行车展（Taipei Cycle），共吸引1150家厂商参展，台湾具竞争力的产业供应链，以及产业聚落，让国际自行车展，成为引领产业发展趋势的重要国际平台。
