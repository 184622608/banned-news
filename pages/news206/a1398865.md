### 李明哲状态不明 妻子探监再度被拒
------------------------

<div class="wysiwyg">
 【新唐人2018年11月10日讯】台湾NGO工作者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李明哲.html" target="_blank">
  李明哲
 </a>
 遭中共拘捕、关押600天，家属近日两度赴中国探监遭拒。原因是李明哲先被移到燕城监狱，两周内又移回赤山监狱，但当局未对家属说明情况。李明哲妻子李净瑜今天（10日）在人权团体陪同下，召开记者会，向外界求助。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李明哲.html" target="_blank">
  李明哲
 </a>
 去年被判刑关押，妻子李净瑜今年总共5度赴监狱探望，不过10月要再探监时，却被告知，李明哲遭移往河北燕城监狱。而前一天要到燕城监狱探视时，却又被告知，李明哲被移回湖南赤山监狱。
 <br/>
 <br/>
 李明哲妻子 李净瑜：〝移监不是一个小事，如果他们大费周章，把李明哲从湖南，移监到河北，那是一个何等在一般监狱行刑法当中，何等大的工程，倘若李明哲是真的被移监过去，为什么又会在两个礼拜之后，被送回去。我的丈夫现在是身处在一个，身体我没有办法臆测的一个状态，否则为什么不让我探监。〞
 <br/>
 <br/>
 李净瑜表示，李明哲患高血压，健康情况堪虑，当局无理由移监，拒绝探视，担忧丈夫身处危险，李净瑜无助对外求援。民间团体指出，目前国际人权论坛正在台湾举办，台湾应该更积极对外说明中共侵害人权事实。
 <br/>
 <br/>
 台湾人权促进会 邱伊翎：〝我们其实应该都要透过国际的会议上面，或是跟国际组织或是跟其它国家，有这些见面跟交流机会的时候，不断的去提出来。〞
 <br/>
 <br/>
 民团呼吁国际社会关注李明哲，并要求中共当局告知家属李明哲现况，也喊话台湾政府，对中共提出严正抗议，并积极维护家属探监权利。
 <br/>
 <br/>
 新唐人亚太电视林嘉韦台湾台北报导
