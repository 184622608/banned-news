### 中国海漂垃圾来袭 金门五乡镇齐护环境
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金门.html" target="_blank">
  金门
 </a>
 与中国，最近的距离只有1.8公里，东北季风常为金门，带来福建沿海的箱网养殖泡沫塑料，及写满简体字的塑料瓶等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海漂垃圾.html" target="_blank">
  海漂垃圾
 </a>
 ，当地环保局动员2千多位民众，在五个乡镇的海滩同步进行垃圾清除，希望还原美丽的海湾。
 <br/>
 <br/>
 一根根矗立
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金门.html" target="_blank">
  金门
 </a>
 海滩的轨条砦，是早年反制共军登陆的大功臣，如今却挡不住对岸的垃圾大军。一颗颗深掩沙中的巨型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保丽龙.html" target="_blank">
  保丽龙
 </a>
 ，伴随着写满简体字的保特瓶，本该洁净雪白的金门沙滩，在这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海漂垃圾.html" target="_blank">
  海漂垃圾
 </a>
 的侵蚀下，可说是惨不忍睹。
 <br/>
 <br/>
 国小学生 李同学：〝我就觉得很可怕，刚才我同学也挖出两大块这么大块的保丽龙，然后在捡垃圾的时候也有许多保丽龙在飘，所以就觉得怎么会这样子。〞
 <br/>
 <br/>
 国小学生 陈同学：〝在（从沙子中）挖保丽龙的时候，因为它旁边有长草，然后保丽龙上面自己本身有一些刺刺的，然后在挖的时候，手可能去磨擦到，然后就刮伤了。（记者：那挖出来之后感觉？）很有成就感。〞
 <br/>
 <br/>
 国小学生 蔡同学：〝以前就是很爱看动画，所以就是以为海滩就是那种干干净净，然后没有垃圾，然后海水很漂亮这样子。可是今天去的时候看到一大堆垃圾，而且还是一整堆一整堆分散的，就是很傻眼！〞
 <br/>
 <br/>
 小小的金门岛，四周布满巨型保丽龙，陪同学生一同参与净滩的师长心中满是不舍。
 <br/>
 <br/>
 安澜国小校长 张志猛：〝最难清的应该就是那些保丽龙的浮球，因为它的体积比较大，而且它通常会有一些绳子结在一起，然后会牵绊在一起，那个在小朋友在拖在拿的时候，会比较不方便。〞
 <br/>
 <br/>
 述美国小教师 洪宇樊：〝保丽龙球很恐怖，它就是会慢慢的变成很小很小，然后它就是跟沙子（搅）和在一起，对啊！你看小朋友这么小，他的沙子里就（搅）和了这么多保丽龙的东西在里面，那等他长大了之后？对，所以我觉得净滩是很有意义的。〞
 <br/>
 <br/>
 响应〝金秋环境季〞，金门五乡镇总动员，同步在峰上海滩、后扁海滩、湖下海堤、雄狮堡海滩、南山头至青岐港一带海岸清除海漂垃圾。
 <br/>
 <br/>
 金门县环保局掩埋场场长 蔡源铭：〝金门的海漂垃圾，内容主要都是保丽龙、还有一些木材，竹木类之类的，塑胶类，就主要这三大类。那来源，都简体字嘛！那个保特瓶部分一律都是简体字嘛，那保丽龙虽然没有写文字，但是应该是对岸养殖业比较多啦！〞
 <br/>
 <br/>
 上午一个多小时，就清理出资源垃圾约2,368公斤，非资源垃圾约12,736公斤。但是令人担忧的是，下一次的涨潮或颱风，又将带来清也清不完的海漂垃圾。
 <br/>
 <br/>
 新唐人亚太电视杨忠其、林威宇、胡净妮台湾金门采访报导
</div>
