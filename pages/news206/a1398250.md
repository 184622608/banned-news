 </a>
 〞节目，日前获得中华民国中医师公会全国联合会颁发〝革新台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中医文化.html" target="_blank">
  中医文化
 </a>
 创意卓越贡献奖〞， 理事长陈旺全表示，节目对中华民族的文化，尤其是中医相关体系，都做了很多论述，同时详细说明传统中医固有诊查以及诊断方法，也邀请许多来宾，对中医药的发展与处方，做详尽的说明。
 <br/>
 <br/>
 中医师公会全国联合会第十届理事长 陈旺全：〝我们也希望中医的传统文化、中医的实证医学、中医的精准医学，逐渐的在世界上呈现，这个就是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新唐人.html" target="_blank">
  新唐人
 </a>
 电视台目前所呈现出来的节目的内容，我们也恳请新唐人电视台往后多多这方面琢磨，来为我们中医的医疗体系，以及对中医对民众的健康照护尽更大的这个力量，造福全世界的人民。〞
 <br/>
 <br/>
 新唐人电视中医制作人 林至厚：〝非常感谢中医师公会全国联合会，颁这个大奖给新唐人亚太电视台,意义重大,这也说明了，我们谈古论今话中医这个节目，已经受到中医界的肯定  ,对中医的推广也起到了很好的作用。〞
 <br/>
 <br/>
 节目制作人表示，谈古论今话中医节目针对现代疾病，与中医古籍对照分析，深入浅出的介绍病因与治疗特色，节目中教导观众如何穴道按摩、自我保健与养生药膳，提供观众身心健康的正确观念。
 <br/>
 <br/>
 新唐人电视中医制作人 林至厚：〝今后我们要更加努力，制作更优质的中医节目，来服务大众。〞
 <br/>
 <br/>
 中医师公会全国联合会第十届理事长 陈旺全：〝新唐人真的是不容易，对中医传统文化，都做很尽力的宣传，我们希望未来把中医真正在实证医学、或是精准医学上所呈现出来的效果，跟新唐人电视台相关的合作。〞
 <br/>
 <br/>
 制作人林至厚表示，很高兴再度受到中医师公会全国联合会的肯定，未来将制作更符合民众需求的节目，把传统中医瑰宝介绍给更多人认识。
 <br/>
 <br/>
