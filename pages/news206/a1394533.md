### 台试办俄罗斯人来台14天免签 踩线团来了！
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】中华民国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交部.html" target="_blank">
  外交部
 </a>
 上个月宣布，试办
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 国民来台14天
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_免签证.html" target="_blank">
  免签证
 </a>
 的措施，而俄罗斯的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_踩线团.html" target="_blank">
  踩线团
 </a>
 已经陆续来到了台湾，周六
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北101.html" target="_blank">
  台北101
 </a>
 董座亲自迎接，这次采线团代表的是莫斯科当地2500家旅行业者，不少人都说，要到101来看烟火秀。
 <br/>
 <br/>
 登上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北101.html" target="_blank">
  台北101
 </a>
 八十九楼向下望去，台北美丽的夜景一览无遗，这可是让来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_踩线团.html" target="_blank">
  踩线团
 </a>
 ，赞不绝口。
 <br/>
 <br/>
 俄罗斯旅游业者：〝太美好、太美妙、太不平凡了，是令人难忘的景色。〞
 <br/>
 <br/>
 俄罗斯旅游业者：〝如果有机会，肯定是要来这边跨年。〞
 <br/>
 <br/>
 俄罗斯旅游业者，走过了，垦丁、日月潭、花东，最后一站来到台北101，董事长张学舜更是亲自现身解说。
 <br/>
 <br/>
 台北101董事长 张学舜：〝今天的天气很好，除了看烟火，我们在这边也是看台北夜景最好的地方。〞
 <br/>
 <br/>
 而今年九月，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交部.html" target="_blank">
  外交部
 </a>
 宣布要试办俄罗斯国民来台14天
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_免签证.html" target="_blank">
  免签证
 </a>
 的措施，邀请了四团俄罗斯业者以及媒体团，而这一团踩线团可不简单，因为他们代表的可是莫斯科当地共2500间的旅行业者，背后庞大的商机，可是让人惊呼。
 <br/>
 <br/>
 台北101董事长 张学舜：〝对俄罗斯旅客的免签证政策，我想是对俄罗斯旅客来台的元年。那我们会跟欧洲的许多大使或代表，一起来举办他们国家跟台湾联谊节庆的日子，也希望两边的交流或是观光能够强化，带来更多的多元旅客。〞
 <br/>
 <br/>
 看好俄罗斯的潜在旅游商机，政府试办的这项免签措施，将创造更多元的客源，来刺激内需市场的商机。
 <br/>
 <br/>
 (新唐人亚太电视 黄亮戬 赵庭誉 台湾台北采访报导)
