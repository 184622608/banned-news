### 非苹新机十月连发 供应链仍看iPhone XR出货
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】苹果九月份发表三款新机，而非苹阵营，也是不甘示弱，十月份将近十支新机发表，台湾部分业者也顺势打入供应链。像是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸿海.html" target="_blank">
  鸿海
 </a>
 旗下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富智康.html" target="_blank">
  富智康
 </a>
 ，代工Google
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手机.html" target="_blank">
  手机
 </a>
 ，还有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大立光.html" target="_blank">
  大立光
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台积电.html" target="_blank">
  台积电
 </a>
 ，抢下韩厂、陆厂部分订单，不过整体而言，台湾供应链以苹果手机为主，第四季营运是否亮眼，也是要看苹果手机的 销售情况。
 <br/>
 <br/>
 Google Pixel 3正式发表，IP68防水防尘，还有独家的AI运算。非苹
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手机.html" target="_blank">
  手机
 </a>
 第三第四季来势汹汹！小米 周一发表MIX 3，周二，华为在伦敦发表Mate 20系列，周三Vivo 新机Z3发布，周五换OPPO K1开卖；手机新品机轮番上阵，台湾供应链，也受到瞩目。
 <br/>
 <br/>
 腾旭投资长 程正桦：〝那台湾比较能够打进去的，比如说像联发科的主要主晶片，或者说像
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大立光.html" target="_blank">
  大立光
 </a>
 的一些Lens，可能还可以打得进去。其他的一些领组件，相对不管是面板啦、PCB板等等，很多其实大陆本地都有他们自己的供应商可以做，所以台湾在所谓非苹阵营里面，比例的确是比较小一点。〞
 <br/>
 <br/>
 Google Pixel 3系列，还有宏达电下周(22日)要发表的首款区块链手机Exodus，由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸿海.html" target="_blank">
  鸿海
 </a>
 等业者操刀；而锁定印度市场的Nokia 3.1 Plus，用的是联发科Helio P22处理器晶片；LG首款的5镜头手机(V40)，台湾业者(晶技、大立光)也打入供应链。
 <br/>
 <br/>
 前外资分析师程正桦指出，非苹产品中，韩系、陆系手机，多以当地供应链生产，台湾供应链订单，以苹果
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_iPhone.html" target="_blank">
  iPhone
 </a>
 为最大宗。今年平价版的iPhone XR，扮演第四季出货要角，带动可成、铠胜、和硕、大立光、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台积电.html" target="_blank">
  台积电
 </a>
 ，成为主要受惠厂商。
 <br/>
 <br/>
 腾旭投资长 程正桦：〝手机的供应链来讲，因为台湾的供应链都比较着重在苹果的部分，所以大家还是会看苹果的销售状况。那去年大概是拉了8000到9000万台左右，今年大概也是8、9000千万台的水准，大概就是持平。所以整体来说，目前看起来，整个第四季，大概就只能寄望XR卖得好不好。〞
 <br/>
 <br/>
 各家业者抢攻欧洲市场耶诞商机，还有主要新兴市场国家的年终消费旺季，台湾供应链第四季乃至明年第一季旺不旺，就看iPhone新机，否端出亮眼成绩。
 <br/>
