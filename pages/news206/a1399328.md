 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制程与产能.html" target="_blank">

 </a>
 <br/>
 <br/>
 晶圆代工龙头台积电 季度例行董事会，核准约1,035亿元资本预算，用于兴建厂房及扩建先进制程等产能。台积电未来几年仍将维持每年100亿美元左右的资本支出，冲刺7奈米及5奈米等先进制程产能，并启动3奈米先进技术前期研发。

 2. 郭台铭重回台湾首富 Q3财报三率三升
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸿海.html" target="_blank">
  鸿海
 </a>
 第3季财报，毛利率、营益率、净利率〝三率三升〞，都优于前一季。另外，鸿海董事长郭台铭在《富比世》排行中重回台湾首富，他在美国威州的投资，是否会因美国期中选举受冲击，对此，威州议会积极力挺表示，将不会容许新任州长艾佛斯毁掉鸿海的投资计划。
 <br/>
 <br/>
 3. 强强联手 传福斯.福特结盟进军电动车
 <br/>
 <br/>
 市场盛传，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福特汽车.html" target="_blank">
  福特汽车
 </a>
 将与福斯汽车结盟合作共同研发电动车。福特之前已经表示，正计划在未来 3到5 年内投入 110 亿美元用于新科技并重塑公司，并将找同业合作，分担研发成本。

 4. 摆脱对中韩依赖 德国力推电池芯国产化
 <br/>
 <br/>
