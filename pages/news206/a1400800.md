### 九合一选举检讨 民进党中生代接班声浪起
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 惨败，兼任民进党主席的总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 辞去民进党主席一职后，民进党党务系统将改组，目前党内已经有人喊出〝中生代提早全面接班〞，党内多位要角也被点名接任党主席。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 在这次县市长选举中，于原有13个席次只剩下6个，总得票率从2014年的约47%，下滑到约39%，选票大幅流失。
 <br/>
 <br/>
 党内检讨败选原因，民进党立院党团干事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李俊俋.html" target="_blank">
  李俊俋
 </a>
 在脸书撰文谈改革理念，主张民进党中生代应该要扛起责任，中生代提早全面接班由中生代承担面对两端世代的沟通之责，同时年轻世代上场。
 <br/>
 <br/>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 在他脸书留言回应：〝我们一起来让中生代承担更大的责任，中生代加油，我们一起加油。〞
 <br/>
 <br/>
 前行政院长苏贞昌上午从土城区展开车队谢票行程，他接受媒体访问时表示，民进党经历重大挫败，必须检讨、反省、调整，党主席已经辞职负责，应该由优秀的中生代承担更大的责任，他愿意扮演协助的角色。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-26/p9143472a96753233.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-26/p9143472a96753233-ss.jpg"/>
   <br/>
   <font size="-1">
    前行政院长苏贞昌（中）26日表示，民进党经历重大挫败，必须检讨、反省、调整，党主席应由优秀的中生代承担更大的责任，他愿意扮演协助的角色。(中央社)
   </font>
  </a>
 </center>
 <br/>
 寻求连任台中市长失利的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林佳龙.html" target="_blank">
  林佳龙
 </a>
 上午受访时表示，民主政治是民意政治，反映这次选举的结果，民进党应该确实检讨与改进，其实民进党的中生代都已经在第一线扮演重要的角色，这也是持续在发展的事情。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-26/p9143471a959696214.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-26/p9143471a959696214-ss.jpg"/>
   <br/>
   <font size="-1">
    寻求连任失利的台中市长林佳龙（左）26日对于自己未来的出路，他说，选后将先沉潜，一方面思考也休息，调整步伐后再决定未来的路。(中央社)
   </font>
  </a>
 </center>
 <br/>
 被视为游系立委蔡适应说，选后党内世代交替是必然的现象。对于外界点名前行政院长游锡堃接任党主席或行政院长，蔡适应说，如果是过渡时期的承担可以，但还是要起用中生代部长与干部，引进新思维与新想法。
 <br/>
 <br/>
 前新潮流系、立委钟佳滨说，民进党面临的是改造、而非改革，因为民进党是组织有问题需要改造，但民进党的价值、目标没有错。
 <br/>
 <br/>
 他说，日前参加民进党新北市长候选人苏贞昌的造势活动，发现苏贞昌已经71岁，但20年前的50岁就当选全台湾人口最多的县长，20年后还是民进党这次表现最好的候选人，显然这个政党出了问题。
 <br/>
 <br/>
 钟佳滨表示，民进党说重视年轻人，但在政党权力分配上，无法适当的释放出来。他认为，民进党要跟企业一样交棒，交棒不成可以再换回来，就像台积电创办人张忠谋一样。
 <br/>
 <br/>
 海派立委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王定宇.html" target="_blank">
  王定宇
 </a>
 说，党团应该尽速开会稳住阵脚，让失落、痛苦的支持者一起站起来，不要让失败主义弥漫，而眼前最重要是5席立委补选要尽快站稳脚步。至于由谁接任党主席、是否由中生代接班，他认为最重要先找出问题，谁接班都是次要问题。
 <br/>
 <br/>
 <h4>
  李俊俋点名中生代：郑文灿、
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
   陈其迈
  </a>
  、林佳龙
 </h4>
 <br/>
 李俊俋25日接受媒体访问时点名郑文灿、陈其迈、林佳龙表现都很优秀，且中生代没有派系包袱，在各方面都准备好了。
 <br/>
 <br/>
 桃园市传统被视为蓝大于绿，郑文灿2014年挑战首届桃园升格桃园市长选举，险胜对手吴志扬2万9281票，这次寻求连任，以超过14万票的差距大胜国民党桃园市长候选人陈学圣，守住这个北部重要直辖市，也让他的政治行情水涨船高。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-26/p9143473a970893219.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-26/p9143473a970893219-ss.jpg"/>
   <br/>
   <font size="-1">
    以高票数连任桃园市长的郑文灿（左）26日表示，拚经济、拚建设、顾民生，是民进党未来努力方向。(中央社)
   </font>
  </a>
 </center>
 <br/>
 林佳龙在台中市长任内政绩不错，被认为是非战之罪，陈其迈经过这场高雄市长选战的洗礼，败选后展现大将之风的气度，加上两人都有丰富的党政经验，未来仍大有可为。
 <br/>
 <br/>
 连任成功的基隆市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林右昌.html" target="_blank">
  林右昌
 </a>
 表示，民进党这次选举大败反映出民意，包括中央推动政策是否周延，过程是否充分接轨民意，都需要反思，最重要接下来要痛定思痛、深切检讨反省，未来方向仍有赖党中央凝聚全党共识。
 <br/>
 <br/>
 他表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 党机器的发挥不够，过去民进党强项包括文宣、造势等，在工具与方式上都要转变，因此需要中生代党主席与秘书长来带领。
 <br/>

  <a href="http://imgs.ntdtv.com/pic/2018/11-26/p9143474a320646438.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-26/p9143474a320646438-ss.jpg"/>
   <br/>

    民进党高雄市长候选人陈其迈 （前右2）24日前往高雄市左营新庄仔青云宫的投开票所投票，向民众比出爱心手势。(中央社)
   </font>
  </a>
