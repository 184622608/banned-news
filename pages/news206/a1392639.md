### 潭美转中颱 台湾中秋假期变天
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中秋.html" target="_blank">
  中秋
 </a>
 连假登场，但好天气可能只有今天(22日)一天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_气象局.html" target="_blank">
  气象局
 </a>
 22日傍晚对新北等12县市，发布大雨特报。而从周日开始，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_锋面.html" target="_blank">
  锋面
 </a>
 及东北风报到，北部、东半部有雨，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潭美颱风.html" target="_blank">
  潭美颱风
 </a>
 最快可能23日就转为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中颱.html" target="_blank">
  中颱
 </a>
 ，颱风的外围云系，下周将为台湾北部以及东半部，带来降雨。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中秋.html" target="_blank">
  中秋
 </a>
 连假第一天，全台各地大多都是多云到晴的好天气，午后会有局部短暂阵雨，不过从周日下午开始，就要变天了。
 <br/>
 <br/>
 中央
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_气象局.html" target="_blank">
  气象局
 </a>
 预报员 吴依帆：〝周日下午开始，由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_锋面.html" target="_blank">
  锋面
 </a>
 掠过北部海面，台湾附近的东北风也会逐渐增强，所以在周日下午到晚间，北部东北部逐渐转为有短暂阵雨的天气，花东地区也会有局部短暂阵雨，云量也会增加。〞
 <br/>
 <br/>
 气象局表示，北部东半部由于云量多，看到月亮机率较低，而中南部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赏月.html" target="_blank">
  赏月
 </a>
 机率高。下周一开始受到东北风影响，周一到周三，迎风面北部东半部都会有短暂阵雨的机率，甚至有局部大雨发生，中南部影响较小，维持多云午后有局部短暂雨的天气。而紧接着还有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潭美颱风.html" target="_blank">
  潭美颱风
 </a>
 ，将影响台湾。
 <br/>
 <br/>
 中央气象局预报员 吴依帆：〝潭美颱风还是处在高海温而且风切比较弱的环境，非常适合它发展，预计有机会强度会来到中度颱风的强度。周四周五它的外围云系，会对台湾的北部东半部，带来一些比较明显的雨势。〞
 <br/>
 <br/>
 气象局以目前路径预测，潭美颱风通过台湾东部海面，之后北转机率高，但要到下周二才会有比较明确的路径。东北风报到加上颱风外围云系影响，未来一周北台湾天气都不稳定。
 <br/>
