### 吴音宁下台 北农董事会通过解职即日生效
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】台
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北农.html" target="_blank">
  北农
 </a>
 产运销公司今召开董事会，董事几乎全出席，被点名要为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 败选下台负责的北农总经理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴音宁.html" target="_blank">
  吴音宁
 </a>
 成为外界关注焦点。最新消息称，董事会已经通过解职吴音宁总经理职务。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北农.html" target="_blank">
  北农
 </a>
 下午召开第12届第7次董事会，据议程有报告和讨论事项，最后才是临时动议。若有董事提人事案，会在临时动议讨论、表决。
 <br/>
 <br/>
 兼任常务董事的全国农会总干事张永成会前受访，被问及北农总经理人事案时说，他不会提，但若农委会有提出，会附议。
 <br/>
 <br/>
 张永成被媒体询问如何评价
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴音宁.html" target="_blank">
  吴音宁
 </a>
 表现，他说，因看不出她有何表现，所以不知如何评价。
 <br/>
 <br/>
 至于吴音宁是否成为这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 败选后的战犯？张永成表示，这不是他说了算，因外界都这样传言，且北农是农产品交易平台，无论谁当农委会主委等位置，应该要让农民没有后顾之忧，让农产品可公平交易，透过太多政治斗争模式而产生总经理人选，事后造成的代价更大。
 <br/>
 <br/>
 下午4时传来最新消息，董事会提案讨论总经理人事案，通过解职总经理吴音宁，即日生效。总经理职务由主秘路全利代理。
 <br/>
 <br/>
 农委会副主委陈吉仲下午4时30分对外表示，北农董事会结束前，农委会代表刘圣鸿提出临时动议通过解职案，过程将跟台北市政府沟通说明。
 <br/>
 <br/>
 陈吉仲说，第一时间已经跟吴讨论沟通过，但她坚持程序走完，吴跟他说，为什么要自己提出?要把程序走完。
 <br/>
 <br/>
 并称农委会希望她离开，不希望外界用不专业角度检视她专业贡献，不容忍她在这个位置被非专业势力糟蹋。
 <br/>
 <br/>

   </font>
  </a>
 </center>
 <br/>
 联合新闻网报导，被点名要为民进党败选下台负责的北农总经理吴音宁今日稍早打破沉默，强调外界把一些标签往她身上贴，攻击她和农委会的副主委陈吉仲、她的家人及蔡英文总统，并说她不能向谎言低头，尤其不能向强势的谎言低头。
 <br/>
 <br/>
 知情人士表示，吴音宁的说法代表没办法接受民进党人士将农业县溃败归咎于她，甚至觉得更高层级的人想留她，但实际有〝两股力量〞考验高层留人的能耐。
 <br/>
 <br/>
 知情人士表示，陈吉仲在今年5、6月就已经劝吴音宁见好就收，吴音宁不该还有错误期待。
 <br/>
