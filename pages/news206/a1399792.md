### WeCare游行8万人上街 陈其迈陈菊合体
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】
 <b>
  新闻周刊
 </b>
 （654）选战倒数，由文化界发起的We Care气球游行守护
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄.html" target="_blank">
  高雄
 </a>
 活动，今天下午举行。活动呼吁民众不分颜色站出来，诉求拒绝丑化霸凌介入、停止仇恨，共同守护高雄。民进党高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 ，今天也在队伍中走完全程，总统府秘书长前高雄市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈菊.html" target="_blank">
  陈菊
 </a>
 ，开场时也突然现身，与陈其迈合体。
 <br/>
 <br/>
 〝weCARE大气球游行〞登场，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄.html" target="_blank">
  高雄
 </a>
 民众不分颜色站出来，诉求拒绝丑化霸凌介入、停止仇恨，共同守护高雄。
 <br/>
 <br/>
 参与游行民众：〝我们也是一直留在高雄，毕竟没有去外地工作，我们就是喜欢我们自己的家园，我们自己的高雄，看到那些新闻，我没有动摇到什么又老又穷，因为我们是一直有在进步的。]
 <br/>
 <br/>
 参与游行民众：〝爱河，我可以到那里喝咖啡呀，那以前哪，不敢靠近呀，就这样子而已。〞
 <br/>
 <br/>
 参与游行民众：〝无形中你就感觉到高雄和以前不同了，你看像交通也好，进步很多就是了。〞
 <br/>
 <br/>
 民进党高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 ，今天穿着T恤短裤，走完全程。主办方号称现场破10万人，陈其迈表示，不知道多少人，但看到人山人海，高雄人站出来表达守护民主，还高雄公道，其实高雄这几年来愈来愈漂亮，大家都以高雄为荣。
 <br/>
 <br/>
 民进党高雄市长候选人 陈其迈：〝尤其是很多的年轻的妈妈，还有年轻的爸爸都带着他们的小朋友走出来，那让我们看到说，高雄其实人民还是非常的温暖，而且非常的热情，而且对我们高雄都有一份执着，那这一份执着就是我们都非常的爱高雄，希望我们的家乡高雄能够变的更好。〞
 <br/>
 <br/>
 总统府秘书长前高雄市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈菊.html" target="_blank">
  陈菊
 </a>
 在游行开始前，突然现身，陪同陈其迈一起游行。而国民党主席吴敦义今天到高雄市云林同乡会辅选，谈到走路工事件，暴走暗指陈菊是母猪，对此，陈其迈表示，非常愤怒、难过。
 <br/>
 <br/>
 民进党高雄市长候选人 陈其迈：〝用那么不堪的字眼，怒骂一位女性，我真的非常难过也非常愤怒。法院都已经有清楚的判决，确实有走路工这件事情，那我想事实就是事实，那也不希望说国民党在选举的时候又模糊焦点。〞
 <br/>
 <br/>
 对此，总统府发言人林鹤明表示，很遗憾看到在野党主席为了辅选参选的韩国瑜而作出这样粗暴低俗的语言霸凌。国民党则回应，吴主席并没有指名道姓，一切都是媒体臆测。
 <br/>
 <br/>
 新唐人亚太电视李金菊、巫亦晴台湾高雄报导
</div>
