### 中共居住证搞统战 民团吁政府要有所作为
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】针对中共对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 居民发放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_居住证.html" target="_blank">
  居住证
 </a>
 ，经济民主连合等民间团体认为，中共此举是在国际间造成统治台湾的印象，台湾应该采取防御机制，向国际社会严正说明立场，杜绝日后中共的任何口实。
 <br/>
 <br/>
 中研院社会所副研究员 吴介民：〝中国宣称要实施对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_居住证.html" target="_blank">
  居住证
 </a>
 ，到它真正实施差不多两个礼拜，两个礼拜的时间而已，所以它算的很准，知道我们的国家有缺位的问题，然后对应能力不足，所以它几乎是一个闪电攻击。〞
 <br/>
 <br/>
 经济民主连合10日举办〝不要让台湾成为克里米亚—中国居住证的风险与因应论坛〞，邀请专家学者针对中共核发居住证问题进行报告与讨论，中研院副研究员吴介民表示，台湾在抵抗中共侵蚀主权中，常会遇到〝国家缺位〞的难题，台湾在面对各项中共政策上，就必须表明自己是国家，中研院副研究员吴叡人则认为，这是中共透过对台湾公民行使统治权达到〝国民化〞，进而达成分化台湾公民，模糊政治边界，兼并台湾的策略。
 <br/>
 <br/>
 中研院台湾史研究所副研究员 吴叡人：〝台湾人四百年来因为政治没办法搞，所以都在搞赚钱，台湾人很喜欢赚钱，这是一个历史情境造成的民族性，所以中国利用这一套东西的做法非常的有效。我们台湾的国民意识不够成熟，我们的国民只知道自己的权利，而不知道对国家、对于这个国民共同体的一个义务跟责任感。〞
 <br/>
 <br/>
 吴叡人分析，居住证是以个人短期利益瓦解台湾共同体的集体利益，台湾政府必须要有效反制，在中国发展的台湾公民，也不该装天真无辜，宣称政经可以分离，中山大学教授蔡宏政也举出中共蚕食美国的例子，说明其中的严重性。
 <br/>
 <br/>
 中山大学社会系教授 蔡宏政：〝他说跟中国现在在美国所做的各种事情相比，俄罗斯所做的事情真是小巫见大巫，那我要跟彭斯讲说，副总统先生，跟中国在台湾所做的事情的话，那你美国的事情真是小巫见大巫。〞
 <br/>
 <br/>
 蔡宏政建议台湾政府，应该要对针对中共设立〝主权风险预警原则〞，证明某项行为不会造成台湾伤害，否则就不应该准许该行为。
 <br/>
 <br/>
 新唐人亚太电视林嘉韦、黄庭锋台湾台北报导
</div>
