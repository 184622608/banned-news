### 脚踏车冲撞台中市长座车 骑士倒地送医
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台中.html" target="_blank">
  台中
 </a>
 市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林佳龙.html" target="_blank">
  林佳龙
 </a>
 (6日)上午参加活动时，发生一辆脚踏车逆向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冲撞.html" target="_blank">
  冲撞
 </a>
 林佳龙座车事故，造成脚踏车骑士倒地送医。据了解，这名伤者是已经失踪五天的黎明幼儿园园长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林金连.html" target="_blank">
  林金连
 </a>
 ，至于是否蓄意冲撞，有待厘清。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台中.html" target="_blank">
  台中
 </a>
 黎明自办重划会与黎明幼儿园争讼多年，幼儿园不服重划不愿拆除，遭重划会提告，园长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林金连.html" target="_blank">
  林金连
 </a>
 近10年开始四处奔走陈情，提出撤销强拆决议、市府出面协调与调查重划会等诉求。
 <br/>
 <br/>
 而林金连在幼儿园被划入重划区后，将幼儿园交由现任张姓园长经营，目前仍有十多名学童就学。林金连在10月31日留下一封信向政府建言后，随即行踪不明，亲友还动员协寻。
 <br/>
 <br/>
 今天林金连骑单车逆向撞台中市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林佳龙.html" target="_blank">
  林佳龙
 </a>
 座车后被送医，到院时呈现昏迷状态，院方紧急插管急救。
 <br/>
 <br/>
 台中市警局第二分局长 高志良：〝目前初步从录影带来看，这位林姓民众是在路旁，在厢型车路旁守候，等市长座车之后，在逆向行驶。我们初步研判的话，市长的座车是符合道路相关交通规定。 〞
