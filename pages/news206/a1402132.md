### 来台旅客1千万提前达标
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观光.html" target="_blank">
  观光
 </a>
 业界及政府各部门的携手合作之下，来台国际旅客在今日（12月2日）晚上达到1,000万名来台人数，不仅连续4年突破年度来台千万旅客的纪录，且较去（2017）年提前近2周达成目标，面对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际观光.html" target="_blank">
  国际观光
 </a>
 竞争下，根据统计，今年迄今新南向18国占来台旅客人次约近四分之一，尤其菲律宾及越南较去年同期更成长达50%及30%以上，成长力道强劲，仰赖政府各部门共同建置友善旅游环境，以及产业界积极提升旅游品质，让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 真正稳站千万旅客量，特别是多元客源市场的开发有了成效。
 <br/>
 <br/>
 交通部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观光.html" target="_blank">
  观光
 </a>
 局指出，近4年来台旅客前5大客源市场，包括中国大陆、日本、港澳、韩国、美国占来台旅客总人数超过7成，其中日本市场可望在今年底突破200万人次；韩国市场虽受花莲地震影响1至10月的来台人次表现，但可望维持百万人次规模；加上新南向18国客源市场可突破250万，几占总来台旅客数的95%以上。
 <br/>
 <br/>
 交通部观光局在Tourism 2020策略下，以一年一旅游主题的方式，来行销
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 观光资源，并以开拓多元市场及推广体验观光等策略主轴，来营造更友善环境及致力于拓展多元的国际客源。今年度在观光产业共同努力下，各主要客源市场表现均显突出，东南亚地区有近2成的成长幅度、加拿大及欧洲长线市场则维持6~16%的成长率，证明台湾观光资源确实具备吸引不同客源市场的魅力，也让我们更有信心开拓更多潜力市场，让市场结构可以更多元。
 <br/>
 <br/>
 展望2019年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际观光.html" target="_blank">
  国际观光
 </a>
 行销，在目前面临邻近各国积极引客竞争激烈的环境下，台湾观光将全力与地方产业结合，希望以各地小镇引导旅客深入地方，体验最在地的台湾风貌。
 <br/>
 <br/>
 （责任编辑：李红）
</div>
