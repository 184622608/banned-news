### 韩国瑜旗山大造势旗海飘扬 逾3万人挤爆会场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】距离11月24日台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 选战，只剩16天，国民党高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 〝三山造势大会〞8日举行开拔到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旗山.html" target="_blank">
  旗山
 </a>
 ，主办单位称有逾3万名支持者涌入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高美医专.html" target="_blank">
  高美医专
 </a>
 ，民众热情高喊冻蒜震天价响，旗海飘扬，而短短一百公尺的路，韩国瑜走逾10分钟。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 继上月底凤山晚会成功造势后，第二场的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旗山.html" target="_blank">
  旗山
 </a>
 造势晚会8日晚间在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美浓.html" target="_blank">
  美浓
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高美医专.html" target="_blank">
  高美医专
 </a>
 登场，原预定晚间18时30分开场，热情的支持者下午4点不到，就涌进近万人，连场外道路也被摊商堵塞。
 <br/>
 <br/>
 造势晚会支持者用汽笛、冻蒜声及国旗旗海为韩国瑜、国民党加油打气。晚间6时54分，主办单位宣称，现场已超过3万人。
 <br/>
 <br/>
 晚会由两广龙狮战鼓团开场，国民党议员候选人林义迪、朱信强、钟易仲，立委林郁方、柯志恩、江启臣、蒋万安、李彦秀轮番站台助讲，强调韩国瑜已不是韩流，而是潮牌，因韩接地气，懂民意，他的资源比头发还要少，但现在大家跟着秃子走，希望就会来招手，一定要把韩国瑜送进高雄市政府，当大家的长工，把高雄打造为全台首富。
 <br/>
 <br/>
 晚间7时28分，韩国瑜在现场群众不断高唱军歌〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_夜袭.html" target="_blank">
  夜袭
 </a>
 〞，支持者热情簇拥、冻蒜声不绝于耳下，与前立法院长王金平寸步难行，短短100公尺的路，走了逾10分钟。
 <br/>
 <br/>
 王金平表示，国民党已攻下凤山、旗山，接着下星期还要前进冈山，盼民众跨域支持，让韩国瑜改变高雄。
 <br/>
 <br/>
 韩国瑜面对群众的热情，他宣示打造高雄为全台首富。他说，从他参选以来对手从上到下，不断抹黑、抹黄，现在又抹红他、打压他，不过他说即便民进党倾全力打韩，但他有支持者全力相挺，他无所惧。
 <br/>
 <br/>
 韩国瑜允诺只要能当选市长就会带着农渔民及高雄盛产的农特产品一起找外销订单，允诺市府会当人民的后盾、靠山，带着人民一起拒绝贫穷，让大家拚命赚到金山、银山。
 <br/>
 <br/>
 他呼吁人民站出来全力相挺，未来市府会全力吸收外资、观光客，让高雄万商云集；市府也不会是官僚体系，允诺当人民的靠山。
 <br/>
 <br/>
 现场民众热情回应相挺，晚会在合唱〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱拚才会赢.html" target="_blank">
  爱拚才会赢
 </a>
 〞的歌声中落幕。
 <br/>
 <br/>
