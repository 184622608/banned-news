 </a>
 宜兰县长参选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林姿妙.html" target="_blank">
  林姿妙
 </a>
 周六(20日)在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗东镇.html" target="_blank">
  罗东镇
 </a>
 成立溪南竞选总部，国民党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄市长参选人.html" target="_blank">
  高雄市长参选人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 特地前往站台，现场挤进数千名支持者，热闹非凡。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林姿妙.html" target="_blank">
  林姿妙
 </a>
 上午在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗东镇.html" target="_blank">
  罗东镇
 </a>
 光荣路成立溪南竞选总部，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 主席吴敦义、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 、立法委员蒋万安到场为林姿妙站台，林姿妙陪同韩国瑜进到会场时，沿途受到民众热烈欢迎，短短数十公尺走了好几分钟。
 <br/>
 <br/>
 韩国瑜致词表示，虽然高雄事情很多，但一定要来为林姿妙站台，因为在林姿妙身上看到很多感动，首先，林姿妙扶养4个孩子又要服务民众，不仅孩子孝顺，选民服务也做得很棒，当罗东镇长又有口碑，〝一位伟大女性、优秀的政治人物、很棒的妈妈和非常和善的好朋友，都在林姿妙身上看到。〞
 <br/>
 <br/>
 韩国瑜说，对宜兰县民而言，一个县长比总统还重要，因为小老百姓其实要的不多，只要水沟能通、商业繁荣，农渔产品卖得出去，教育做得好，而这些都是县长的工作，呼吁宜兰乡亲把握机会选出一个发自内心爱家庭、爱社会、爱宜兰的县长。
 <br/>
 <br/>
 韩国瑜指出，为何越是〝风调雨顺〞农渔民越是流眼泪，因为农渔民面临〝两把刀〞，外销出不去，内销价格崩盘，未来他若当选高雄市长，要与林姿妙联手帮农渔民找外销订单，提升农渔民生计，让农渔民不再流泪，他保证〝说得到一定做得到〞。
 <br/>
 <br/>
 韩国瑜说，现在选举不用奥步、抹黑好像不会选，为什么每次选举一定要用奥步，就不能拥有一个干净的选举吗，大家要相信林姿妙坚定信心，不要相信没水准的奥步，让伟大的母亲、勤劳的县长带领宜兰发光发热。
