### 替郑朝方站台 蔡英文吁坚定守护民主价值
------------------------

 【新唐人2018年11月18日讯】
 <b>
  新闻周刊
 </b>
 （654）选战进入倒数，总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 周六来到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新竹县.html" target="_blank">
  新竹县
 </a>
 为民进党新竹县长候选人造势。蔡英文再度重申，年底的选战是民主保卫战，要坚定守护民主价值，台湾，不能乱。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一大选.html" target="_blank">
  九合一大选
 </a>
 最后一个超级星期六，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 辅选列车开到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新竹县.html" target="_blank">
  新竹县
 </a>
 ，为民进党新竹县长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑朝方.html" target="_blank">
  郑朝方
 </a>
 造势。蔡英文致词时重申，年底的选战，是一场民主保卫战，不仅内部有反改革势力，还有外在因素虎视眈眈，就是希望台湾乱，来打击台湾民主。
 <br/>
 <br/>
 民进党主席 蔡英文：〝还有一群人，在外面虎视耽耽，希望台湾乱一点，他们就可以证明，民主是失败的，各位，台湾的民主没有失败。这一场选举就是台湾价值的保卫战，我们团结起来，一起悍卫台湾，一起坚定的守护我最相信的民主价值，好不好。〞
 <br/>
 <br/>
 蔡英文再次强调，政府推动改革的批评声浪，她都听到了，也不断在修正和调整，但是改革方向没有错，国家正走在正确的道路上。
 <br/>
 <br/>
 民进党主席 蔡英文：〝改革的方向没有错，民主的坚持也没有错，我们的国家正走在正确的道路上是不是。〞
 <br/>
 <br/>
 蔡英文以4年前郑文灿和林智坚的例子说明，没有不能翻转的政治板块，选对的人，会做事，4年就可以改变城市，希望打造〝桃竹竹共好〞生活圈，翻转新竹县。
 <br/>
 <br/>
 新唐人亚太电视彭正干台湾新竹采访报导
