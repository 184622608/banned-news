### 破釜沉舟决心 卢秀燕辞去立法委员
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】距离投票最后4天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 台中市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢秀燕.html" target="_blank">
  卢秀燕
 </a>
 20日以〝决心〞为主题，发表重大宣布，将辞去做了20年立法委员，展现破釜沉舟决心。台中市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林佳龙.html" target="_blank">
  林佳龙
 </a>
 上午在丰原第一市场拜票，当媒体询问时表示，〝尊重她的决定，平常心看待！〞
 <br/>
 <br/>
 这一次各县市长选举，至今已有4名候选人辞去立法委员，除了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢秀燕.html" target="_blank">
  卢秀燕
 </a>
 还包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 台北市长候选人姚文智、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 云林县长候选人张丽善、民进党高雄市长候选人陈其迈。
 <br/>
 <br/>
 中午1时30分，卢秀燕展现破斧沉舟的决心、宣布辞立委拼到底，决心做个好市长《台中非赢不可》。她说，这场选举是代表乡亲与执政党当局的挑战，背负乡亲的托负。
 <br/>
 <br/>
 卢秀燕说，〝我非赢不可！中部人民受到空污、低薪、景气差之苦，如果秀燕在周六选赢，代表我们人民乡亲有翻身机会，也代表台中这个城市、我们国家有改变的机会。为展现破釜沉舟的决心，也为了展现一定会赢的决心，此刻宣布辞去立委职务，并当场签署立委辞职书。〞
 <br/>
 <br/>
 卢秀燕表示，因不分区立委可以递补，区域立委涉及改选，只会涉及提早改选；她坦承今天职辞并没有跟党中央报备，因请辞中央一定不准。但〝因为我一定会当选，所以没有考虑接棒人选。〞
 <br/>
