### 中秋节我最应景  来〝月世界〞登陆月球
------------------------

<div class="wysiwyg">
 【新唐人2018年09月24日讯】中秋连假，除了烤肉赏月外，还有什么不一样的方式过节呢？在高雄有个知名的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_月世界.html" target="_blank">
  月世界
 </a>
 〞，一片荒漠的景色，来到这里就像
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_登陆月球.html" target="_blank">
  登陆月球
 </a>
 ，在苗栗三义更有个〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火炎山.html" target="_blank">
  火炎山
 </a>
 〞，地形地貌就像置身于火星，不同的风貌让你过个特别的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中秋节.html" target="_blank">
  中秋节
 </a>
 。
 <br/>
 <br/>
 苗栗著名景点三义
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火炎山.html" target="_blank">
  火炎山
 </a>
 ，由于风化与雨水的冲刷侵蚀，切割成许多尖锐山峰的组合，彷佛像裂开凶恶的山形，加上崩塌断崖的黄色石砾，远看就像是被火烧过一般，极恶地形吸引许多登山客前来朝圣。
 <br/>
 <br/>
 登山客：〝这里是跟黄山有一点点像，我们这是小黄山。〞
 <br/>
 <br/>
 登山客 林小姐：〝因为它没有树，就感觉是很像月球表面一样，简单的形容一下像什么，剑龙的鳍。〞
 <br/>
 <br/>
 登山客 林小姐：〝很像国外的大峡谷，那就是心情不好的时候，可以来这边走走，那心胸就很开阔，那登高之后就有六百多公尺的海拔，那时候就可以更愉悦的欣赏苑里美丽的风景。〞
 <br/>
 <br/>
 同样拥有极恶的地形，高雄田寮〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_月世界.html" target="_blank">
  月世界
 </a>
 〞特殊景观充满凄凉荒漠之美，不少游客选在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中秋节.html" target="_blank">
  中秋节
 </a>
 之前，来趟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_登陆月球.html" target="_blank">
  登陆月球
 </a>
 之旅。
 <br/>
 <br/>
 香港游客：〝明天是中秋节，这里就是月世界，这好像有点关系，所以就来这边。〞
 <br/>
 <br/>
 外国游客 LUKE：〝我觉得这里很特别，跟那个月亮一样，所以我们就来了这里，看不一样的地方。〞
 <br/>
 <br/>
 游客：〝爬这个天梯上来之前，有看个有一段是嫦娥奔月，就想说中秋节来奔月一下。〞
 <br/>
 <br/>
 由泥岩、砂岩、页岩等构成的青灰岩，表面是灰白色，草木无法生长，呈现一片荒漠景象。由于泥岩中疑似有一种能发光的矿物质，在月光照射下，据传能隐约发出点点荧光，因此才被称为〝月世界〞，而老一辈的民众，则是有另一种说法。
 <br/>
 <br/>
 本地区民：〝它那个是盐份，牠(羊)自己会去吃这些盐份，这些盐份吃起来的羊特别好吃就是这样，所以冈山以前的羊肉都说，羊是从月世界抓的，所以源头就是这样，牠就是爬这些山去吃这个盐。〞
 <br/>
 <br/>
 从高处俯视壮阔的〝月世界〞，一座座光秃秃的恶地地形相当壮观；夜晚的〝月世界〞还会投射灯光，透过灯光的照射下，营造出〝月世界〞山形的另一番美貌。有心前来游玩的民众，可别错过了〝月世界〞晚间的奇特景致。
 <br/>
 <br/>
 新唐人亚太电视佘志诚、彭正干台湾高雄报导
</div>
