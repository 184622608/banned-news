### 台铁：观光列车发生死伤事故 部份旅客换乘
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 23日晚间8时30分发出简讯，一列
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观光列车.html" target="_blank">
  观光列车
 </a>
 (台北一花莲一台东一台中一台北的顺行列车)晚间7时56分在三义站开车后于西正线(第二股道)，发生死伤事故，导致列车最后车厢离月台距离约10公尺。目前尚未有进一步死伤消息。
 <br/>
 <br/>
 据悉，疑三义站开车时传出疑似有民众落轨，而其他班次列车也因此受到延误，人员伤亡状况仍待确认。
 <br/>
 <br/>
 <center>
 </center>
 <br/>
 由于明天是台湾九合一选举，不少人赶着返乡投票，事故列车车箱几乎每个车箱都客满。赶时间的旅客，车站已派换乘144次。
 <br/>
 <br/>
 (责任编辑：卢勇信)
