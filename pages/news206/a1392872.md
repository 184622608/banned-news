### 强颱〝潭美〞侵台机率降低 周五六最接近台湾
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月25日讯】今年第24号强烈颱风〝潭美〞今(25)日上午8时位在鹅銮鼻东南东方880公里的海面上，目前缓慢往西北方移动，依各方预测其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_路径.html" target="_blank">
  路径
 </a>
 ，侵袭台湾机率降低，但预计28、29日最接近台湾。气象专家吴德荣提醒，尽管
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潭美颱风.html" target="_blank">
  潭美颱风
 </a>
 登陆台湾机率低，但是对台湾威胁的程度仍有待观察。
 <br/>
 <br/>
 中央大学大气科学系兼任副教授吴德荣今天在〝三立准气象．老大洩天机〞专栏中表示，今、明两天北方的太平洋高压减弱，导引气流变慢，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潭美颱风.html" target="_blank">
  潭美颱风
 </a>
 移速减慢，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_路径.html" target="_blank">
  路径
 </a>
 偏西北转北，之后太平洋高压又略增强，导引气流再转回东南风、潭美颱风转向西北、并略加速，将通过宫古、石垣岛附近海面，逐渐接近西风带。
 <br/>
 <br/>
 他提醒，颱风路径经台湾东侧北转的机率最高，登陆台湾的机率降低，但由于途径中经多次的环流变化，路径不确定性仍大，各国模拟的误差范围仍未脱离台湾，因此仍应持续观察。
 <br/>
 <br/>

 据新唐人亚太台报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强颱.html" target="_blank">
  强颱
 </a>
 〝潭美〞，目前缓慢往西北方移动，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_气象局.html" target="_blank">
  气象局
 </a>
 指出，潭美今明两天呈现滞留状态，实际路径仍须观察，但从台湾东方海面北上机率较高，依目前预测，周五周六，是颱风最接近台湾的时候，北部东北部将有明显降雨。
 <br/>
 <br/>
 受到东北风影响，北部以及东半部，云量增多，有阵雨以及局部大雨发生的机率，中南部天气比较稳定，而在温度上，已经稍微有些凉意，尤其在北部东北部，中南部则是比较闷热，外界也高度关注，强颱潭美的最新动态。
 <br/>
 <br/>
 气象局简任技正 伍婉华：〝目前各国模式预测，我们分析以从东方海面北上机率是有比较高的，但是北上过程中，距离我们多近，必须看到时候太平洋高压甚至北方潮汐牵引力量，这部分是必须观察的。〞
 <br/>
 <br/>
 25日上午8时，潭美颱风来到鹅銮鼻东南东方880公里海面，缓慢往西北移动，
 <br/>
 <br/>
 气象局表示，今明两天潭美的导引气流不明显，中心近似滞留，对台湾的实际影响，仍需观察。
 <br/>
 <br/>
 气象局简任技正 伍婉华：〝以目前预测，我们预估星期五到星期六，是潭美颱风最接近我们的时候，接近过程中，在星期五星期六，北部东半部地区的降雨情况，尤其北部东北部降雨情形就会明显，有可能会有大雨或局部性豪雨。这是目前判断路径，未来随着颱风路径改变，北台湾的降雨程度也会有所调整。〞
 <br/>
 <br/>
 至于是否最快周四发布海上警报？气象局表示，由于颱风直接侵袭台湾机率降低，北上时距离台湾的远近也仍需观察，目前持保留态度，认为重点应该放在，颱风外围环流，对台湾北部以及东北部的影响程度。
 <br/>
 <br/>
 (新唐人亚太电视 王冠霖 张媛婷 台湾台北采访报导)
