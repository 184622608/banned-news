### 双英桃竹选战拚场 拉抬客庄选情
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】离今年底即将举行的台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 越来越近，台湾总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 和前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马英九.html" target="_blank">
  马英九
 </a>
 ，周六下午先后抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桃园.html" target="_blank">
  桃园
 </a>
 与新竹，为各自的侯选人造势。
 <br/>
 <br/>
 年底选战将近，总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 辅选行程满档。下午先到新竹市参加市长林智坚竞选总部成立大会，旋即赶往新竹县为县长候选人郑朝方站台，希望创造大新竹共好，区域联合治理，让新竹县市互相拉抬选情。
 <br/>
 <br/>
 民进党主席 蔡英文：〝4年前智坚给新竹市带来的改变，大家有目共睹，这也代表了有创意、有能力、爱家乡的年轻人绝对不会被埋没，大家说对不对？这一次，是新竹县新竹市在历史上最难得的机会，最年轻、最有冲劲、最有创意、最有行动力的两个年轻人，都在我身边。〞
 <br/>
 <br/>
 而民进党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桃园.html" target="_blank">
  桃园
 </a>
 市长候选人郑文灿今天举办北区竞选总部成立大会，蔡英文也到场为郑文灿加油打气，并强调〝大屋盖一半，师傅不能换〞，呼吁票投郑文灿
 <br/>
 <br/>
 民进党主席 蔡英文：〝选举没有那么复杂，11月24日我们把票投给认真做事，做出成绩，把票投给带领桃园进步的5号郑文灿好不好。应该是这样讲，桃园有够赞，市长选文灿，好不好。〞
 <br/>
 <br/>
 年底选战拚场，新竹县选情胶着，前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马英九.html" target="_blank">
  马英九
 </a>
 今天也到新竹县为国民党新竹县长候选人杨文科站台，并到5个乡镇扫街，就怕蓝营铁票区失守。
 <br/>
 <br/>
 前总统 马英九：〝今天来很开心，为什么呢，刚刚文科兄讲，平面媒体民调我们领先了，领先了第二位候选人，但是呢，千万千万不要骄兵的心态，领先的幅度不到3个百分点，还在误差范围内，所以我们还有很长的路要走。〞
 <br/>
 <br/>
 选战持续升温，马英九今天晚间也将陪同国民党桃园市长候选人陈学圣前进八德区夜市扫街拜票。双英今天都在新竹、桃园造势，就是要拉抬自家候选人客家庄选情。
 <br/>
 <br/>
 新唐人亚太电视柯国栋、林秋霞、胡淑霞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾新竹.html" target="_blank">
  台湾新竹
 </a>
 、桃园报导
