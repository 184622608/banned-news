### 台湾周六九合一选举 治安高度戒备
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 将于星期六举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 ，选出县市长，议员等一万多个公职人员。警政署召开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全台连线会议.html" target="_blank">
  全台连线会议
 </a>
 ，盘点
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_治安部署.html" target="_blank">
  治安部署
 </a>
 ，确保选前平安，投票平顺、投票后平稳三大目标。尤其今晚的选前之夜。
 <br/>
 <br/>
 警政署长 陈家钦：〝迅速侦办有关危安与不实讯息，例如候选人人身安全、走路工等等，企图制造恐慌及混淆视听，这些都已经构成犯罪要件。〞
 <br/>
 <br/>
 警政署保安科长 范织坤：〝我们今天在各个竞选造势晚会的活动中，我们也会针对，现场所有出入处所、周边厕所、还有舞台，都做相关的安全措施，也会利用我们刑警局、保三总队，的相关警犬、侦爆犬等，做现地的侦搜。〞
 <br/>
 <br/>
 (电访音源)高雄市警局保安科股长 高英豪(电访)：〝派遣三位，每天有两位，贴身保护市长候选人。(高雄选前之夜)压力很大，但是一定要把维安最好，最重要的，是候选人的安全维护。〞
 <br/>
 <br/>
 连线会议的萤幕上，呈现多个县市候选人扫街的即时画面。今天护送选票、选前造势，工作繁重，警方严阵以待。明天投开票日，全
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 总共6万8千名警察，全部停止休假，也规划配套保障警察个人投票权。警政署长呼吁全民冷静理性，若有任何破坏、暴力，警方将严正执法、追究到底。
 <br/>
 <br/>
 新唐人电视综合报导
 <br/>
 <br/>
