### 年底旅展准备开跑 旅游好康率先报你知
------------------------

<div class="wysiwyg">
 【新唐人2018年11月24日讯】一年一度的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北国际旅展.html" target="_blank">
  台北国际旅展
 </a>
 ，周五登场。而今年，是首度移师到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南港展览馆.html" target="_blank">
  南港展览馆
 </a>
 ，共有1700个摊位参加，业者更是纷纷祭出新优惠，要让消费者们感到更加划算。
 <br/>
 <br/>
 充满活力的民族舞蹈做开场，就是要告诉大家，一年一度的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北国际旅展.html" target="_blank">
  台北国际旅展
 </a>
 将在11/23盛大登场，首度移师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南港展览馆.html" target="_blank">
  南港展览馆
 </a>
 ，规模号称史上最大，优惠促销当然不能少。
 <br/>
 <br/>
 旅游集团总经理 游国珍：〝到韩国的景福宫去穿韩服，去游景福宫的部分，也只要7999。〞
 <br/>
 <br/>
 不管你是要到国外还是在国内旅游，都能找到好康优惠。像是五福祭出日本九州四天，未税价1万3999元起；雄狮则祭出美西三万有找；东南旅游主攻国旅，花博一日游，买一送一只要488元；六福住宿最低下杀一七折；爱泡温泉的民众，馥丽饭店打出原价1万5200的温泉客房，现在四人同行住宿券只要5999元。
 <br/>
 <br/>
 旅游集团公关处协理 王浅秋：〝现场也都会做房型的升等，就会比原本在网路上买的更有价值，今年度又碰到选举强碰的时段，不知道大家的购买力如何，所以我们基本上是自己在推估业绩的时候少了一成。〞
 <br/>
 <br/>
 台湾观光协会会长 叶菊兰：〝投票也好、投完票也好，你都可以来逛旅展轻松一下，从过去比较狭窄的世贸改到南港展览馆，所以它可以容纳的摊位更多。〞
 <br/>
 <br/>
 特别的是，今年的展览更首度开放6岁以下的小朋友入场，推出了亲子体验区，11/23~11/26，现场将有来自全球60个国家、地区，共1700个摊位，要让民众逛到心满意足。
 <br/>
 <br/>
 新唐人亚太电视林嘉韦、赵庭誉台湾台北采访报导
</div>
