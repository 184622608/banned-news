### 五千法轮功学员 自由广场排字震撼全场
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】
 <h4>
  新闻周刊（655）
 </h4>
 来自台湾及世界各国五千多位的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员，周六(11/24)在台北自由广场举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_排字.html" target="_blank">
  排字
 </a>
 活动，他们排出一本巨型的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英文版转法轮.html" target="_blank">
  英文版转法轮
 </a>
 ，从高空俯看，景象相当殊胜壮观，让许多游客震撼不已 。
 <br/>
 <br/>
 从高空俯看，一本巨大的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英文版转法轮.html" target="_blank">
  英文版转法轮
 </a>
 ，就在自由广上绽放光芒；五千多名来自台湾及世界各国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员们，在2018年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_排字.html" target="_blank">
  排字
 </a>
 活动上，共同排出一本英文转法轮的壮观图形。
 <br/>
 <br/>
 台湾法轮大法学会副理事长 黄春梅：〝今年排的是英文版的转法轮，那我们都知道转法轮其实在全球已经有四十几个国家的语言的翻译，所以我们觉得这么一本宝书，应该让世界各个民族，各国家的人都有机缘能够来看看这本书。〞
 <br/>
 <br/>
 盛大的排字活动，让现场所有游客深受震撼，也让更多人看见法轮功的平和与受迫害真相。
 <br/>
 <br/>
 民众：〝很震撼，然后就觉得哇！这些(学员)，就觉得不可思议这样。〞
 <br/>
 <br/>
 荷兰游客 Groenendaal：〝感觉非常的平和，台北的民众也都很善良。我第一次看到排字活动，是全新的体验，现场气氛很祥和舒服〞
 <br/>
 <br/>
 加拿大华人 赖先生：〝因为中国政府这样的侵犯基本人权，侵犯这个中国人民基本的信仰自由是非常违背世界潮流，反人类的一种行为，所以今天我看到这么多的法轮功朋友在这里这个进行这样的法事(排字)还是什么东西总之我感觉非常震撼非常好。〞
 <br/>
 <br/>
 排字活动汇集了世界各地的法轮功学员，许多外国学员也和大家分享他们的心情与修炼心得。
 <br/>
 <br/>
 阿根廷法轮功学员：〝很开心看到来自世界各地的法轮功学员都来参加排字活动。以前我只能从明慧网上看到台湾的排字活动，今天是第一次亲身参与，觉得很开心〞
 <br/>
 <br/>
 日本学员 藤岛太太：〝其实以前有病嘛，就是身体非常不好，就是精神比较怎么说，就挺不起来吧，总爱发脾气什么的，变好了以后，心里面自然可以就平静嘛。〞
 <br/>
 <br/>
 中国旅美法轮功学员 林先生：〝觉得我自己心性有提升，能够有一个更加宽容、包容的心，然后能够明辨是非吧，我觉得在这个社会中，在这个社会洪流中，能够把自己把握住，能够同时有一个心性的提升，我觉得是非常难得的事情。〞
 <br/>
 <br/>
 法轮功学员一年一度的排字盛会，每年都留下一幅殊胜壮观的图像，记录着法轮功学员的修炼历程，也让更多人见证法轮功的美好
 <br/>
 <br/>
 新唐人亚太电视黄亮戬、张年洁台湾台北采访报导
