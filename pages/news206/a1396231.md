### 韩国瑜游港拼经济 郑进一出江湖陪站台
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】国民党高雄市长参选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 20日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_茄萣.html" target="_blank">
  茄萣
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_兴达港.html" target="_blank">
  兴达港
 </a>
 乘动力帆船游港，并重申拚经济，带动地方观光；资深音乐人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑进一.html" target="_blank">
  郑进一
 </a>
 站台摧票。两人还高歌一曲，郑进一事后帮韩国瑜的歌艺打了90分。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 到高雄
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_茄萣.html" target="_blank">
  茄萣
 </a>
 区为第二选区市议员候选人郑东元助阵，下午到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_兴达港.html" target="_blank">
  兴达港
 </a>
 情人码头，搭乘动力帆船小艇游港。韩国瑜说，兴达港环境优美，可惜游客不足，未来要举办大型活动，盼带动观光。
 <br/>
 <br/>
 韩国瑜表示，拚经济是他的主要政见。他说，高雄市民关心的是经济问题，相信高雄有优异的经济发展条件，只是〝沉睡太久〞，只要经济发展起来就可以让人刮目相看。
 <br/>
 <br/>
 傍晚艺人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑进一.html" target="_blank">
  郑进一
 </a>
 也现身竞选会场，两人抵达时，受到民众夹道热烈欢迎，民众喊出的〝冻蒜(当选)〞声不绝于耳。
 <br/>
 <br/>
 郑进一表示，对政治没兴趣也不认识韩国瑜，但有次在电视看到看到〝秃头〞的韩国瑜，觉得怎么可能有人讲话比他还臭屁，而且臭屁得很正派，因此兴起替韩国瑜写竞选歌曲的念头。
 <br/>
 <br/>
 郑进一说，韩国瑜对城市发展有想法，让他决定要相挺，他还登上韩国瑜的宣传车与韩国瑜一起宣传拜票。
 <br/>
 <br/>
 郑进一答应要为经济状态，写一首新歌，表达高雄的心情。他说，台湾的选举一直以来都是骂来骂去，但他与韩的共同想法是，选举应该很快乐的，所以这首竞选新歌的基调会是〝选举，就要选乎欢喜〞。
 <br/>
