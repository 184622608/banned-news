### 大陆猪瘟失控 台湾行政院提高罚金防肉品入境
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】欢迎回来，中国大陆的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟疫情.html" target="_blank">
  非洲猪瘟疫情
 </a>
 ，不断蔓延，中共官方通报，有15个省市沦陷。但台湾农委会日前评估，中国境内的疫情早已全面失控。台湾行政院为了防堵病毒进入台湾，全面禁止肉品入境，违规者罚金上限最高30万。
 <br/>
 <br/>
 行政院拍版通过动物传染病防治条例，未来旅客入境时，如果有违规，携带肉品入境行为，将实行更高罚金。
 <br/>
 <br/>
 行政院农委会防检局局长冯海东：〝最高额度从一万五调高到30万，那将来我们会依据他的违规的情节的情形，来订定，修改我们现在的这些裁罚基准。〞
 <br/>
 <br/>
 中国境内非洲猪瘟全面爆发，疫情已经失控。台湾政府近期全面加强稽查，并且在星期三(7日)举行防疫演练，就怕小三通成为防疫漏洞。
 <br/>
 <br/>
 行政院农业委员会副主委黄金城：〝肉类制品是不得携带，不得携带入国内，就是所有肉类制品都不可以，因为鸡鸭鱼鹅它仍然，虽然说非洲猪瘟不会传染给那个鸡鸭鹅，但是它有禽流感的问题，所以这一些类似的(产品)，不管是熟肉或者是生肉，完全都不能带进国内。〞
 <br/>
 <br/>
 中国大陆的猪瘟疫情蔓延，光是中共官方通报，就有15个省市沦陷，再加上中共隐匿疫情，中国民众对于猪瘟病毒，有变种传染给人的可能性，一无所知，形成防疫漏洞。日前一段影片在网路上疯传，有中国民众开着车，沿途马路边都是丢弃的病死猪尸。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情爆发.html" target="_blank">
  疫情爆发
 </a>
 至今，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪肉.html" target="_blank">
  猪肉
 </a>
 价格，已经涨价两成五。而台湾如果被传播非洲猪瘟，对台湾整体产业将会造成，超过两千亿台币的经济损失。
 <br/>
 <br/>
 新唐人亚太电视综合报导
