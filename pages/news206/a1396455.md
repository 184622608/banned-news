### 花莲东部海域规模6地震 最大震度3级
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】中央气象局
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 测报中心观测，今天中午12时34分58秒，距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花莲.html" target="_blank">
  花莲
 </a>
 县政府东方104.2公里发生芮氏规模6.0地震，最大震度3级，北部地区民众有感到摇晃。
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 测报中心地震报告，这起地震震央位在台湾东部海域，距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花莲.html" target="_blank">
  花莲
 </a>
 县政府东方104.2公里，深度约31.3公里。
 <br/>
 <br/>
 最大震度出现在宜兰县武塔、宜兰县宜兰市、新北市五分山、台中市鸟石坑都为3级。
 <br/>
 <br/>
 其他地区震度为：花莲县和平、南投县合欢山、台东县长滨、台北市、桃园市、苗栗县南庄、新竹市、新竹县竹北市、新北市、彰化县员林、云林县古坑、云林县斗六市、嘉义县番路2级。
 <br/>
 <br/>
 基隆市、台中市、南投线南投市、高雄市桃源、彰化县彰化市、嘉义市、台南市楠西各1级。
 <br/>
 <br/>
 各地暂无重大灾情传出。
 <br/>
 <br/>
 (责任编辑：夏明义)
</div>
