### 花莲东海域规模6地震 专家：两板块碰撞正常能量释放
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】中央气象局
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 测报中心观测，今天中午12时34分58秒，距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花莲.html" target="_blank">
  花莲
 </a>
 县政府东方104.2公里发生芮氏规模6.0地震，最大震度3级，北部地区民众有感到摇晃。地震专家看法，这是今年第2起规模6.0地震，正常能量释放，余震发生机率不高。
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 测报中心地震报告，这起地震震央位在台湾东部海域，距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花莲.html" target="_blank">
  花莲
 </a>
 县政府东方104.2公里，深度约31.3公里。
 <br/>
 <br/>
 最大震度出现在宜兰县武塔、宜兰县宜兰市、新北市五分山、台中市鸟石坑都为3级。其他地区震度为：花莲县和平、南投县合欢山、台东县长滨、台北市、桃园市、苗栗县南庄、新竹市、新竹县竹北市、新北市、彰化县员林、云林县古坑、云林县斗六市、嘉义县番路2级。基隆市、台中市、南投线南投市、高雄市桃源、彰化县彰化市、嘉义市、台南市楠西各1级。各地暂无重大灾情传出。
 <br/>
 <br/>
 气象局地震测报中心主任陈国昌指出，这起今年以来第2起规模达到6的地震，属于隐没带碰撞。
 <br/>
 <br/>
 陈国昌表示，菲律宾海板块每年向欧亚板块推挤约8公分的位移，正常来说，每年都应该要有2、3次的规模6地震，从今年2月6日花莲地震之后，台湾就一直没有规模相似的地震产生，因此气象局私底下也担心能量释放延迟，以及能量释放的地点。
 <br/>
 <br/>
 气象局地震测报中心副主任吴健富受访则说，这次震央的位置约2、3年就会发生一次地震，上次这个区域发生规模6以上的地震是在2015年，显示这次的地震也在周期范围之内。
 <br/>
 <br/>
 吴健富也表示，从过去的观测资料来看，这个位置的地震发生余震的机率并不高，民众不用太过担心。
 <br/>
 <br/>
 前气象局长辛在勤分析，这次地震的主因是菲律宾板块碰撞欧亚大陆板块碰撞的影响，属于中层地震，地震深度31.3公里正是这两个板块碰撞最剧烈的地方，到达一个时间点，就会发生地震释放能量。
 <br/>
 <br/>
