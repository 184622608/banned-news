### 迎全球情势变化 台湾国际品牌价值正成长
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】2018年度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际品牌.html" target="_blank">
  国际品牌
 </a>
 价值调查，日前（1月30日）公布前20大名单，由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电脑.html" target="_blank">
  电脑
 </a>
 品牌大厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华硕.html" target="_blank">
  华硕
 </a>
 夺冠，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宏碁.html" target="_blank">
  宏碁
 </a>
 、研华、台达电、联发科等大厂，都榜上有名，仅管全球情势变化多端，台湾前20大品牌的总价值，更较往年成长0.2%，突显台湾品牌在国际上的，竞争力。
 <br/>
 <br/>
 2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际品牌.html" target="_blank">
  国际品牌
 </a>
 价值名单揭晓，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电脑.html" target="_blank">
  电脑
 </a>
 品牌大厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华硕.html" target="_blank">
  华硕
 </a>
 连6年夺冠，品牌市值以16.19亿美元蝉联榜首，进步最大的是中信金，增幅达13%，今年台湾国际品牌前20名总价值与去年相比，成长0.2%，仅管全球情势变化多端，台湾国际品牌，仍保持稳定成长。
 <br/>
 <br/>
 华硕电脑财务长 吴长荣：〝其实真的要去深耕每个国家，的一个特定的一个市场，那真正的去做到跟使用者的一个接触。台湾这么多的一些品牌公司，能够找寻一些合作，跟互相这个突破的一个机会，可能也是一个很好的发展方向。〞
 <br/>
 <br/>
 经济部常务次长 王美花：〝那我们就是把他推广到国际上去。〞
 <br/>
 <br/>
 根据调查，台湾前20大国际品牌保持多元领域发展，其中，消费性电子及科技领域维持成长，金融品牌增幅最大。并归纳出台湾品牌发展的4大趋势，包括智慧驱动产业革新、企业结盟引领突破、品牌创新体验升级，以及实践承诺价值跃升，也是自2008年金融海啸后，对台湾品牌的另一个反思。
 <br/>
 <br/>
 品牌价值调查机构大中华区总经理 林万足：〝这个10年的反思来看，以订阅式的经济为出发点，以客户导向为出发点，再来实际提供对消费者有用的东西或服务，这是一个非常重要的，三大的一个趋势。〞
 <br/>
 <br/>
 台湾经济研究院院长 林建甫：〝我们可以看到台湾的品牌在国际上，已经有相当大的影响力，所以我们应该持续的让台湾的品牌，能够在国际上发光发热，这对解决台湾目前的闷经济，还有薪资过低的问题，其实是有很大的帮助。〞
 <br/>
 <br/>
 专家也提醒，企业在专注技术升级、加速转型的同时，也必须以品牌角度进行思考，建议可从〝数位化、无人化、细分化、服务化、理念化及透明化〞等6个面向发展，帮助企业品牌定位加值。
 <br/>
 <br/>
 新唐人亚太电视林嘉韦、李晶晶台湾台北报导
