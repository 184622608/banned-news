### 桃园文创博览会登场 六大展区玩创意
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】在台湾，第一届桃园
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文创博览会.html" target="_blank">
  文创博览会
 </a>
 ，10月27日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桃园展演中心.html" target="_blank">
  桃园展演中心
 </a>
 登场，今年的活动，不仅整合了桃园的多元族群文化与在地艺术家、青年科技等作品，还有DIY创作体验及大师讲堂、草地音乐会活动，吸引不少孩童与家长前来。
 <br/>
 <br/>
 民众 陈先生：〝花生糖，台语呢？那个花生糖是…？（我也不知道）你不知道，啊会的先，这个卤肉是什么（卤肉），鸡卷是（鸡卷）。〞
 <br/>
 <br/>
 爸爸抓抓腮帮子，想了想，发现自己也忘了。台北科技大学的学生将桌游结合科技，把台语设计成游戏，让小朋友边玩边学台语。
 <br/>
 <br/>
 台北科技大学互动设计系 廖翊伶：〝透过我们的APP去辅助他们，去给他们语音的回馈，让他们可以比较快的去熟悉这个语言，我们设定是这个桌游可以让幼稚园玩，也可以让他们带着这套桌游，拿回去一起跟阿公阿妈一起游玩。〞
 <br/>
 <br/>
 今年博览会主题丰富多元，吃得到美食，还能逛手工艺品；业者在欧式面包里，加入原住民特有的红藜、刺葱等食材，更增添风味。皮革结合原住民传统图腾制成各式皮包，让业者在国外参展时大受瞩目。
 <br/>
 <br/>
 业者 王冠宇：〝这是完全老板的母亲自己手织的，这个菱形代表祖灵之眼，象征祖先在天上保佑你，同时也希望您不要做坏事。〞
 <br/>
 <br/>
 业者 崔静玫：〝用原住民方式，来展现欧式面包的风格，以无毒的概念为主要的一个诉求。〞
 <br/>
 <br/>
 27日起，一连7天首届桃园
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文创博览会.html" target="_blank">
  文创博览会
 </a>
 登场，从日常生活用品到工匠艺术文化、互动科技还有客家文创，六大展区近500件文创商品，开幕首日吸引不少民众到场参观。
 <br/>
 <br/>
 文创博览会策展顾问 陈俊良：〝透过展览其实当民众喜欢，引起了很多的收购，或者是很多人的收藏，那这个东西它就有市场的价值，它开始生产变成一个独立品牌，那这个其实是一个很好的管道。〞
 <br/>
 <br/>
 民众 廖先生：〝说实在的客家产业已经慢慢的（没落）了，没错，所以小朋友可能都不知道什么叫做客家，客家的文物有什么都不知道，像我是客家人，可是说真的已经没有像以前那种客家文化的东西了。〞
 <br/>
 <br/>
 桃园首届文创博览会，琳琅满目的展览，展现桃园多元文化，也看见在地文创的无限潜力。
 <br/>
 <br/>
 新唐人亚太电视胡淑霞、张年洁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾桃园.html" target="_blank">
  台湾桃园
 </a>
 采访报导
</div>
