### 知名修琴师：神韵编曲功力深厚 是学习典范
------------------------

<div class="wysiwyg">
 【新唐人2018年09月25日讯】2018美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神韵交响乐.html" target="_blank">
  神韵交响乐
 </a>
 团，9月23日晚间在台中中山堂演出，现场座无虚席，有30年资历的知名修琴师王圣哲盛赞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神韵.html" target="_blank">
  神韵
 </a>
 编曲功力深厚，学习编曲作曲的音乐人，都应来学习。
 <br/>
 <br/>
 前台湾省立交响乐团驻团修琴师 王圣哲：〝非常非常的令人感动，这要非常非常雄厚的后援，跟这个作曲，还有编曲的专业人才，才有办法做得到。〞
 <br/>
 <br/>
 台湾知名小提琴修琴师王圣哲，拥有30年修琴资历，初中时期就参加交响乐团，本身也是管弦乐团指挥，他推荐，编曲家们都应向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神韵.html" target="_blank">
  神韵
 </a>
 取经。
 <br/>
 <br/>
 前台湾省立交响乐团驻团修琴师 王圣哲：〝她会把这个很东方的弦律走一段，然后再有一段变奏，能够让曲目变化这么大，然后和声这么漂亮，尤其是用西方的技巧，并没有改变东方的技巧，我想很多学作曲的人应该来听这样子的变化，还有这样子的作曲方式和配乐的方式，这是非常成功的，功力很深厚。〞
 <br/>
 <br/>
 神韵的音符不只感动了挑剔的修琴师，王圣哲说，台下观众对神韵产生的共鸣反应，是每个艺术家都想追求的。
 <br/>
 <br/>
 前台湾省立交响乐团驻团修琴师 王圣哲：〝深深的着迷，才会有这么热情，这么热烈的安可声跟掌声，最后一首安可，那个阿里山变奏曲《高山青》，那个每一个人都跟着拍子，跟着节奏这样鼓掌，那感觉真的是台上台下融成一片，那就是我们交响乐团最期待，也是梦寐以求的那种感觉。〞
 <br/>
 <br/>
 近两个小时演奏落幕，观众长时间掌声不舍离去，指挥米兰纳切夫，为台中乐迷带来三首安可曲，还在指挥第三首《高山青》时，转身对观众用大拇指比了一个〝赞！〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神韵交响乐.html" target="_blank">
  神韵交响乐
 </a>
 在台湾创下一连十场演出票售罄的佳绩，接下来神韵交响乐将前往嘉义、台南。
 <br/>
 <br/>
 新唐人亚太电视叶锡鸿、詹咏茹、林莉台湾台中采访报导
</div>
