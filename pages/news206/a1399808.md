### 北市长候选人吴蕚洋扫街 话题离不开〝蜂蜜柠檬〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 候选人吴蕚洋因为在辩论会上唱歌，加上〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蜂蜜柠檬.html" target="_blank">
  蜂蜜柠檬
 </a>
 论〞一夕爆红。今天他上街拜票，来到北投市场，果然人气很旺，不少民众争相合照，还有人要求当场献唱一曲。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 候选人吴蕚洋，趁黄金周到北投市场拜票，人气果然很旺，马上遇到粉丝要求献唱。
 <br/>
 <br/>
 民众 VS. 台北市长候选人 吴蕚洋：〝不醉不罢休，东边那个美人，西边黄河流。〞
 <br/>
 <br/>
 就像看到偶像一样热情挡不住，整个市场就是我的KTV，除了这首《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱江山更爱美人.html" target="_blank">
  爱江山更爱美人
 </a>
 》，热门话题当然离不开〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蜂蜜柠檬.html" target="_blank">
  蜂蜜柠檬
 </a>
 〞。
 <br/>
 <br/>
 台北市长候选人 吴蕚洋 VS.民众：〝要榨给妈妈喝哦。〞
 <br/>
 <br/>
 另类扫街，没有喊冻蒜，也不说自己是几号，倒是手拿着蜂蜜柠檬，努力宣传平安健康。而市场人潮满满满，走过路过不少人抢着合照。这时候，〝社民党〞议员候选人刘仕杰也来扫街，当下顾不得继续拜票，直接当起小粉丝。
 <br/>
 <br/>
 台北市长候选人 吴蕚洋 VS.社民党议员候选人 刘仕杰：〝蜂蜜蜂蜜蜂蜜，柠檬柠檬柠檬。〞
 <br/>
 <br/>
 双方人马一起喊口号，整个市场少了选战烟硝味，气氛好欢乐，而对于周日(11月18日)政见发表会，吴蕚洋留了一个大伏笔。
 <br/>
 <br/>
 台北市长候选人 吴蕚洋：〝明天就要一个政见发表会，政见发表会是明天嘛，所以明天应该会给他一个惊喜。〞
 <br/>
 <br/>
 素人参选意外爆红，市场走一回，证明吴蕚洋已经满满粉丝，人气不比政治人物差。
