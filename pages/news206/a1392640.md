### 台湾人爱吃海鲜 贸协利用龙虾拚外交
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 人爱吃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海鲜.html" target="_blank">
  海鲜
 </a>
 ，去年进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_龙虾.html" target="_blank">
  龙虾
 </a>
 、螃蟹等甲壳类水产金额高达4亿美元，贸协特地带着业者，前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中南美洲.html" target="_blank">
  中南美洲
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_采购.html" target="_blank">
  采购
 </a>
 海鲜，因为免关税优惠，零售价降价6%，让采购团的成交金额，高达1100万美元，是2017年采购的3倍，利用龙虾拚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交.html" target="_blank">
  外交
 </a>
 ，成果相当亮眼。
 <br/>
 <br/>
 鲜红肥美的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_龙虾.html" target="_blank">
  龙虾
 </a>
 还有新鲜的冷冻白虾，一端上桌，光是气势就很惊人。装在碟子里头的和风鲜虾盘，插满国旗的百香果龙虾盅，或是咸咸甜甜的龙虾塔，小巧可爱，超适合当作下午茶，其实这些都是贸协带团去中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_采购.html" target="_blank">
  采购
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海鲜.html" target="_blank">
  海鲜
 </a>
 。
 <br/>
 <br/>
 贸协董事长 黄志芳：〝龙虾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交.html" target="_blank">
  外交
 </a>
 ，我们要继续推动。〞
 <br/>
 <br/>
 原来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瓜地马拉.html" target="_blank">
  瓜地马拉
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼加拉瓜.html" target="_blank">
  尼加拉瓜
 </a>
 签有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自由贸易协定.html" target="_blank">
  自由贸易协定
 </a>
 ，去年对尼加拉瓜采购水产金额高达5700万美元，是熟冻龙虾出口第一大市场，占比达46%，这次贸协带着10家业者，前往中美洲采购，金额再攀升3倍，因此连洪都拉斯、尼加拉瓜大使都到场致意。
 <br/>
 <br/>
 贸协董事长 黄志芳：〝后续采购量全年度会达到，4800万美金，就是今年的水产采购比去年牛肉采购，金额高了三倍出来。〞
 <br/>
 <br/>
 业者 曾福生：〝优惠来讲的话，台湾现在大概都是中美洲来的价格，看到比较多，大概将近5%、6%是跑不掉的。〞
 <br/>
 <br/>
 业者大力引进
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邦交.html" target="_blank">
  邦交
 </a>
 国水产，除了配合政府拚外交，尼加拉瓜和波士顿龙虾比一比，尼加拉瓜龙虾螯小身体大，因此肉比较多，波士顿龙虾则是相反；至于洪都拉斯白虾，和台湾东南亚养殖的草虾大小肉质，几乎相差无几。
 <br/>
 <br/>
 友邦的海鲜水产进入台湾，不只稳固双方友谊，也让依存度提高，成功发展龙虾外交。
