### 观光局补助 高雄饭店住房率提高逾90%
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱学恒.html" target="_blank">
  朱学恒
 </a>
 到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄.html" target="_blank">
  高雄
 </a>
 发鸡排，但却订不到房间，还特别在网路发文问，高雄有什么活动？有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_饭店.html" target="_blank">
  饭店
 </a>
 业者说，其实是因为观光局补助，从10月中开始住房率提高，再加上观光补助的关系，住房率提高到90%以上。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_饭店.html" target="_blank">
  饭店
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_订房.html" target="_blank">
  订房
 </a>
 代表 洪佩甄：〝主要原因，我想观光局的补助它是没有限名额的， 它只有限日期，那补助费用也是没有上限。今年又多了交通补助的关系，所以刺激更多人愿意出游的意愿。〞
 <br/>
 <br/>
 游客：〝我从头份来，本来就一直都会来。每年固定会来两次。〞
 <br/>
 <br/>
 记者 vs.游客：〝你有感觉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄.html" target="_blank">
  高雄
 </a>
 订房其实要越来越早吗？要啊！要啊！〞
 <br/>
 <br/>
 观光局的补助，住宿和交通费每人每房最高2500元，所以刺激更多人的出游意愿。饭店业者表示，从10月中开始住房率提高，再加上观光补助的关系，住房率提高到90%以上。
 <br/>
 <br/>
 饭店接待组主任 洪芸彤：〝我们从10月、10月中开始住房率开始满高，那因为有观光补助的关系，又把我们住房率提高到九成以上。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱学恒.html" target="_blank">
  朱学恒
 </a>
 本周末要到高雄发鸡排，同样订不到房间，还特别在网路发文说，高雄有什么活动吗？怎么房间这么难订。
 <br/>
 <br/>
 饭店公关 李姿莹：〝高雄其实最近艺文活动举办满多，又有一些医学研讨会在举办，到明年又有一些娱乐演唱会，所以在需求部分是有增加的。〞
 <br/>
 <br/>
 游客：〝高雄这几年发展得非常好，很值得来。〞
 <br/>
 <br/>
 高雄住宿大爆满，有人说是因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 当选了，所以游客也到了，不过高雄这几年风貌越发迷人，还曾夺下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际宜居城市金牌.html" target="_blank">
  国际宜居城市金牌
 </a>
 ，可能才是吸引游客最重要的原因。
