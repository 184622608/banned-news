### 颱风潭美午后转强颱 不排除侵台
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月24日讯】今天是中秋节，中央气象局表示，今年第24号
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潭美.html" target="_blank">
  潭美
 </a>
 最快午后就会发展为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强颱.html" target="_blank">
  强颱
 </a>
 ，是否侵台25、26日两天是关键。
 <br/>
 <br/>
 中央社报导，中央气象局预报员张承传表示，由于东北风增强，北部及东半部地区的云量增多、降雨机率提升，今天起到27日北部及东半部有断断续续的降雨，并可能伴随雷击，气象局也针对新北、基隆和宜兰发布豪雨特报，台北市为大雨特报，提醒上述区域的民众除了外出活动要携带雨具，而中南部则维持多云到晴，山区及近山区的平地有午后雷阵雨。
 <br/>
 <br/>
 气温方面，受东北风影响，北部及东北部高温降到摄氏28至30度，花东高温也是30度出头，中南部因位于背风面，仍显闷热，高温可达32、33度。
 <br/>
 <br/>
 第24号
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潭美.html" target="_blank">
  潭美
 </a>
 凌晨2时位于鹅銮鼻东南东方约1280公里的海面上，以西北西方向持续往琉球南方海面移动，预估今天午后就可能增强为强烈颱风，25、26日颱风将进入导引气流较弱区域而减速，也是颱风路径的关键期，28日起潭美可能影响台湾天气，但影响的程度仍有相当大的变数，不排除侵台，提醒民众持续留意气象局发布的最新颱风动态。
 <br/>
 <br/>
 他补充，颱风目前距离台湾仍远，但今晚起东南部沿海（含绿岛、兰屿）及恒春半岛就会开始有长浪发生，另外由于东北风增强，气象局也发布陆上强风特报，台南以北、东半部（含兰屿、绿岛）、恒春半岛及澎湖、金门、马祖沿海、空旷地区及邻近海域将有8至10级强阵风，并有较大风浪。
