### 平稳物价 台湾油价冻涨至年底
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中油.html" target="_blank">
  中油
 </a>
 表示，政府为平稳物价，避免油价短期大幅波动上扬，带动国内物价上涨影响民生，国内各式汽、柴油价格从今天凌晨零时起，不予调涨，冻涨期限，将执行到今年底止。
 <br/>
 <br/>
 现在，92无铅汽油每公升30.0元、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_95无铅.html" target="_blank">
  95无铅
 </a>
 每公升31.5元，日前已接近美国要求完全禁运伊朗石油的11月4日，市场预期心理，也推升油价，来到四年新高。
 <br/>
 <br/>
 到国际油价持续走扬影响，10月桶装
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瓦斯.html" target="_blank">
  瓦斯
 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天然气.html" target="_blank">
  天然气
 </a>
 涨势，分别调涨每公斤1.6元及2.99%涨幅，随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中油.html" target="_blank">
  中油
 </a>
 宣布到年底前国内油价冻涨，11及12月气价是否跟进？中油只强调，这月底视价格调整幅度及物价情况再做决定。
