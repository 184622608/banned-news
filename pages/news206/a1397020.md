### 台电吊桩水块 2外包水下作业员离奇失联
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台电.html" target="_blank">
  台电
 </a>
 龙井区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火力发电厂.html" target="_blank">
  火力发电厂
 </a>
 27日进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_止水块.html" target="_blank">
  止水块
 </a>
 吊桩作业，下午1点多，2名外包水下作业人员先后下水，失去音讯，岸上人员立刻报警抢救。
 <br/>
 <br/>
 据台中港消防分队指出，现场海域正值涨潮，第一个工人下水2分钟失去讯号，接着第2人下水也失联，10分钟后，岸上人员旋即通报台中港消防分队抢救，目前出动空勤直升机搜救中。
 <br/>
 <br/>
 下午3时40分左右，台中港消防分队透过水下摄影机发现其中1人受困于石头底下，下午4时多潜水人员准备下水，但现场水流过强，潜水人员无法下水，正设法全力搜寻抢救中。
 <br/>
