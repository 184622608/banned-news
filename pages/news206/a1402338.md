### 阿里山林铁106周年 与斯洛伐克林铁缔结姊妹
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】来自台湾消息。1912年通车到今年的阿里山
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林业铁路.html" target="_blank">
  林业铁路
 </a>
 ，已有106年的历史。在日前的周年庆祝活动中，林务局特别与欧洲中部的斯洛伐克、切尼赫荣铁路缔结
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姊妹铁道.html" target="_blank">
  姊妹铁道
 </a>
 ，建立合作关系。
 <br/>
 <br/>
 阿里山桧木车厢蒸汽火车，缓慢地由车库开到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里山林铁.html" target="_blank">
  阿里山林铁
 </a>
 周年庆会场，林务局局长林华庆与斯洛伐克切尼赫荣林铁主席Bilek，在会场上签署合作意向书，这是阿里山林铁与国外缔结姊妹的第9条铁路。
 <br/>
 <br/>
 林务局局长 林华庆：〝这二条铁路、也都是从原本早期是载运林木，到后来因为公路的兴起，所以就慢慢转型到现在变成是一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观光文化列车.html" target="_blank">
  观光文化列车
 </a>
 的营运。〞
 <br/>
 <br/>
 斯洛伐克切尼赫荣林铁主席 Bilek：〝我们在台湾不仅仅找到了我们一直梦寐以求的想找到的那台火车头，我们真的衷心衷心的感激感谢命运，让我们在台湾可以找到这么多好朋友、志同道合的朋友。〞
 <br/>
 <br/>
 斯洛伐克位于处欧洲中部，切尼赫荣的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林业铁路.html" target="_blank">
  林业铁路
 </a>
 ，则是一条在众目睽睽之下可以穿越足球场的铁路。蒸汽火车伴随着碧绿草原，成为世界各国游客喜爱的铁道游程。
 <br/>
 <br/>
 林务局局长 林华庆：〝除了会进行铁道或者是车辆修复技术的交流之外，我们也会开始藉由斯洛伐克跟去年所签署的英国威尔普兰菲尔铁道，我们开始进行三方的合作跟技术交流，我们也希望能够在欧洲地区拓展更多的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姊妹铁道.html" target="_blank">
  姊妹铁道
 </a>
 的关系。〞
 <br/>
 <br/>
 斯洛伐克切尼赫荣林铁主席 Bilek：〝我坚信就这种合作一定能提升两国在国际上的能见度，而且能互相促进两国人民之间的交流。〞
 <br/>
 <br/>
 切尼赫荣林铁与阿里山林铁之轨距相近，年龄相近，原本都是以运送林木为目的，现在也都是文化观光游憩列车，现在缔结为姊妹铁道，期望为双方带来更大的发展前景。
 <br/>
 <br/>
 新唐人亚太电视张金洲台湾嘉义采访报导
</div>
