### 代理民进党主席 基隆市长林右昌：以市政为主
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】台湾〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 〞大选和高达10案的公投，身兼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党主席.html" target="_blank">
  民进党主席
 </a>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 承担这次败选的政治责任，请辞民进党党主席一职。今(28日)下午，民进党举行中常会，中常委一致推举基隆市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林右昌.html" target="_blank">
  林右昌
 </a>
 担任代理主席。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林右昌.html" target="_blank">
  林右昌
 </a>
 透过幕僚发布新闻稿表示，他是现任市长，当然一切以市政为重，代理党主席一职是〝暂时性的代理职务〞，只有1至2个月时间，主要是协助党主席补选顺利，还有姚文智及卢秀燕辞职立委后，留下台北及台中两席立委党内提名作业。
 <br/>
 <br/>
 林右昌同时表示，他原本就是民进党中常委，每周都要出席中常会，暂时性的〝代理党主席〞，最主要是协调整合中央党部顺利完成补选与提名工作，并不负担民进党未来的辅选责任，所以，不会占用太多市政时间，一切还是以基隆的市政工作及发展为主。
 <br/>
 <br/>
