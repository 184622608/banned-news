### 苹果iPhoneXR降价促销  鸿海年底前裁10万人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】新唐人亚太台报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 iPhone XR遇上销售压力了吗？根据外媒报导，苹果已经与日本电信业者达成协议，将在下周由苹果补贴给电信业者以促销iPhone XR，这恐怕也是有史以来，首款上市未满月，就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降价.html" target="_blank">
  降价
 </a>
 促销的iPhone。另外iPhone最大组装厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸿海.html" target="_blank">
  鸿海
 </a>
 计划今年年底前最多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裁员.html" target="_blank">
  裁员
 </a>
 10万人。
 <br/>
 <br/>
 经济日报报导台湾业者说法，表示如果补贴策略带动iPhone XR
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降价.html" target="_blank">
  降价
 </a>
 ，预料对整体销售将有强劲动能，期待降价后带买气，对此乐观其成。
 <br/>
 <br/>
 中央社也引述消息人士证实，iPhone最大组装厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸿海.html" target="_blank">
  鸿海
 </a>
 预料明年将面临景气放缓和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 砍价压力增加，因此提出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裁员.html" target="_blank">
  裁员
 </a>
 计划，今年年底前最多裁员10万人，严阵以待未来12个月可能更险峻的市况。
 <br/>
 <br/>
 消息人士表示，这次裁员力度会比往年都还要大，但他也说，鸿海淘汰10万人，对〝拥有超过100万名员工的集团而言，没那么严重〞。
 <br/>
 <br/>
 另外，苹果的软性印刷电路板（FPC）供应商嘉联益已在台湾裁撤数百名派遣员工。
 <br/>
 <br/>
 〝日经亚洲评论〞（Nikkei Asian Review）报导，许多苹果主要供应商加快削减成本计画，为2019年需求预期走跌做准备。
 <br/>
 <br/>
 知情人士表示，他预期苹果和其他客户会要求砍价10%到20%。他说，尽管这是苹果一般会提出的要求，但在订单没有强劲增加之下，这样的要求难以实现。
 <br/>
