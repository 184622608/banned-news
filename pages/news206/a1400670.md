### 台湾县市长选举 国民党至少赢得15县市
------------------------

<div class="wysiwyg">
 【新唐人2018年11月24日讯】当地时间11月24日晚上，台湾县市长的选举成果，已经确定由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 拿下15个县市，重要的六都中，执政的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 ，丢掉了重要的台中、高雄。民进党主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 ，已经宣布辞去党主席一职。台北市长则是战况激烈，直到晚间11点50分，现任市长柯文哲与国民党候选人丁守中，还未分出胜负。
 <br/>
 <br/>
 中华民国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 ：〝（我们）不能停下脚步，我们更不能因此自怨自艾，在我辞去党主席之后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 当进行党务的改组。〞
 <br/>
 <br/>
 台湾这次六都选举结果令许多专家惊讶，包括执政的民进党丢掉原先看好的台中市、高雄市。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 台中市长候选人卢秀燕票数一路领先。
 <br/>
 <br/>
 卢秀燕：〝我所开的政见都是我深思熟虑的政见，我一定要说到做到。事实上，我今天很感动，也很感谢，但是我没有很激动，因为我知道我一定会赢。〞
 <br/>
 <br/>
 韩国瑜：〝感谢高雄市人民，愿意给我一个机会，感谢我最可敬的对手陈其迈委员，没有批判过对手一句话。〞
 <br/>
 <br/>
 而势均力敌的新北市最后确定由国民党候选人侯友宜胜选，国民党党主席吴敦义也在记者会上感谢民众的支持：
 <br/>
 <br/>
 侯友宜：〝我会用这样的精神，与大家一起打拼新北市的好日子，打拼台湾、中华民国的未来，大家说好不好？〞
 <br/>
 <br/>
 以下我们也提供新唐人亚太台稍早访问专家画面。
 <br/>
 <br/>
 东吴大学政治系教授左宜恩：〝当六都直辖市长选举结果，他（执政党）发现他丧失大部分直辖市控制权时，这些直辖市他们联合起来对中央政府产生的制衡力量会是非常巨大的。因为中央政策必须要靠地方政府的配合。主要影响政策范围在外交，尤其是二岸政策方面。〞
 <br/>
 <br/>
 新唐人记者谢心欣综合报导
</div>
