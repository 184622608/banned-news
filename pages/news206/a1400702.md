### 台湾选举公投 七成以上赞成传统婚姻
------------------------

<div class="wysiwyg">
 【新唐人2018年11月24日讯】与台湾与九合一选举同一天登场的〝平权
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公投.html" target="_blank">
  公投
 </a>
 〞、〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱家公投.html" target="_blank">
  爱家公投
 </a>
 〞，也引起台湾和国际社会广泛关注。公投结果显示，超过70%的台湾民众坚持民法婚姻应限定在一男一女的传统形式。
 <br/>
 <br/>
 与〝九合一选举〞一同进行的10项
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公投.html" target="_blank">
  公投
 </a>
 议案结果陆续公布，77%的台湾民众认为，应该坚持一男一女的传统形式。
 <br/>
 <br/>
 下一代幸福联盟主席曾献莹认为，公投结果代表主流民意。
 <br/>
 <br/>
 下一代幸福联盟主席TSENG HSIEN-YING：〝这个是所有重视家庭价值跟下一代教育全体民众的胜利。大家用一张张的同意票，告诉执政当局，什么才是主流的民意。〞
 <br/>
 <br/>
 他呼吁政府在指定相关法律和学校教材时，尊重民意。
 <br/>
 <br/>
 下一代幸福联盟主席曾献莹：〝我们在这边强烈的呼吁，政府未来在制定定相关的法律，还有国中、小教材的时候，一定要按照这次公投的结果来执行。〞
 <br/>
 <br/>
 台北市民认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统婚姻.html" target="_blank">
  传统婚姻
 </a>
 形式合乎情理。
 <br/>
 <br/>
 台北市选民FU JIAN-HUA：〝一个男的一个女的结婚，这是合情合理的嘛，这个也就是说你要延续后代嘛，繁衍物种嘛，那不能，就不需要这个东西，连婚姻法都不需要有了，对不对？那就没有意义了嘛。〞
 <br/>
 <br/>
 台湾明年5月将合法化同性婚姻，但外界认为，这次公投结果，至少展现出民意。
 <br/>
 <br/>
 新唐人记者易文、严曦综合报导
