### 日本传出两起猪瘟 台农委会宣布暂停进口
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】欢迎回来。中国非洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪瘟.html" target="_blank">
  猪瘟
 </a>
 扩散至18省市，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 持续蔓延。台湾为防堵疫情，加强入境查验。农委会主委林聪贤今明两天，安排视察机场。另外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 当地传出猪瘟，农委会也宣布，暂停
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本猪肉.html" target="_blank">
  日本猪肉
 </a>
 与肉制品进口。
 <br/>
 <br/>
 中国非洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪瘟.html" target="_blank">
  猪瘟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 扩散，农委会主委林聪贤视察小港机场防疫。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 扩及中国18省，传播到台湾的风险高，农委会加强入境查验，同时宣导旅客不得从疫区携带肉制品。
 <br/>
 <br/>
 农委会主委 林聪贤：〝中国非洲猪瘟的疫情，现在已经扩散福建的莆田，离台湾已经非常近了，我们为了要保障我们国内的禽畜产业，在疫区携带这些肉制品，我们未来要提高罚则到30万，所以请大家不要轻易的触法。〞
 <br/>
 <br/>
 林聪贤在视察过程，就有越南旅客被查获夹带肉制品，检疫人员撕开外层报纸，确定是违禁肉品，当场罚缓1万5千元。
 <br/>
 <br/>
 小港机场检疫员：〝他们都是因为台湾买不到鸡肉丝跟牛肉干，因为那个风味比较特别，所以一般来讲他们都会冒险，偷渡、运送来藏在行李箱，或随身行李偷渡进来。〞
 <br/>
 <br/>
 除了非洲猪瘟疫情，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 也传出两起日本猪瘟案例，虽然不同于非洲猪瘟，不过为保护台湾养猪产业，农委会宣布暂停进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本猪肉.html" target="_blank">
  日本猪肉
 </a>
 产品。防检局表示。日本每年输入台湾猪肉40公吨，仅占台湾进口总量0.06％，预估对台湾市场影响不大。
 <br/>
 <br/>
 新唐人亚太电视台湾高雄报导
</div>
