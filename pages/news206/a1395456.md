### 美查中共〝千人计划〞涉间谍 传台33人被吸收
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞吸收海外优秀华人专家，美国政府调查后，发现这计划与中共军方有关，有多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华裔科学家.html" target="_blank">
  华裔科学家
 </a>
 涉及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍.html" target="_blank">
  间谍
 </a>
 罪。有媒体报导，台湾有33名专家参与千人计划，引发立委关注。
 <br/>
 <br/>
 美国联邦调查局FBI，几个月来重点调查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 所谓〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞名单，大批约谈，已有人涉及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍.html" target="_blank">
  间谍
 </a>
 罪名被美方拘捕。有媒体报导，竟然有33名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾专家.html" target="_blank">
  台湾专家
 </a>
 ，被千人计划吸收。台湾政府陆委会表示，正密切了解中。
 <br/>
 <br/>
 民进党立委 张宏陆：〝美国也以间谍罪，要去审问这些人、了解相关的程序，台湾(政府)竟然都完全没有动作，而且台湾面临中共的利诱，是比美国更严峻的。这都是政府可以做的啊，你可以用更好的机会留住人才，也可以用法律的限制。〞
 <br/>
 <br/>
 国民党立委 廖国栋：〝如果是涉及到国家的机密安全，这个部分的技术的时候，我觉得我们应该就要，有一定的管理会比较好。〞
 <br/>
 <br/>
 外媒指出，美国政府相信，〝千人计划〞与中共〝科学与工业间谍〞有关，一部分还与中共军方密切相连。传出FBI要求美国研究机构，开除列千人计画的学者，还有相关学者被工作单位要求提出自清。
 <br/>
 <br/>
 民进党立委 刘世芳：〝我们自己的政府，应该能够呼吁，在我们台湾所培养出来的这些高科技、或者是相当有学术背景的人，能够分清楚说，我们服务的对象，如果是一个极权国家，然后做出来的结果，不利于所有全球的高科技发展的话，其实应该有所遏止。〞
 <br/>
 <br/>
 在美国，已有列名计划的科学家被拘捕调查，包括美国通用电气主任工程师郑小清、维吉尼亚理工大学教授张以恒、气象专家王春在，安德森癌症研究中心谢克平则涉嫌儿童色情与间谍。
 <br/>
 <br/>
 时代力量立委 黄国昌：〝我们的政府到目前为止，都还没有展现出一个，让全体国人可以放心的积极态度。机敏性的产业，它的整个发展，事实上一方面接受国家政策的援助，另方面对国家安全的维系也有非常强的作用。〞
 <br/>
 <br/>
 据中共〝千人计划〞官网资料，声称已有6千名海外人才加入。不过，在美国政府盯上后，转趋低调，网路近日传出一份中共官方文件，要求党媒不得再出现〝千人计划〞的字眼。
 <br/>
 <br/>
 新唐人亚太电视王冠霖、张东旭台湾台北报导
</div>
