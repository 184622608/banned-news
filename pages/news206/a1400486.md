### 台湾九合一选举 蓝绿选前之夜决战高雄
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】蓝绿将决战高雄港都，作为这次选举的〝一级战区〞，最后一天〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选前之夜.html" target="_blank">
  选前之夜
 </a>
 〞，两大阵营将用造势晚会大PK。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 定调〝欢乐、阳光〞的群星会，艺人轮番上阵，还找来百名秃子陪韩国瑜进场。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 则诉求，〝守护民主、决胜高雄 〞除了有陈菊等党内高层外，也找来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_詹雅雯.html" target="_blank">
  詹雅雯
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灭火器.html" target="_blank">
  灭火器
 </a>
 乐团演唱，希望号召20万支持者到场。
 <br/>
 <br/>
 国民党高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 &amp;歌手 叶启田：〝(你们会一起唱爱拚才会赢)我觉得他现在可以唱，他大概明天不能唱，因为他有经纪公司。讲好了，讲好了，真的可以唱？讲好了，这样就妥当了。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选前之夜.html" target="_blank">
  选前之夜
 </a>
 大拚场，韩国瑜准备找来277位秃子陪他一起进场，代表高雄277万人口，象征照亮高雄，另外也邀请包括郑进一、曹雅雯等艺人轮番献唱，地点就在梦时代台糖加油站旁，不邀请政治大咖，强调将时间留给市民百姓。
 <br/>
 <br/>

 <br/>
 国民党高雄青工会秘书长 曹桓荣：〝不是再用悲情激情的角度，希望大家理性一起参与快乐，充满欢笑的派对。希望把这个时间，留给我们所有的市民。〞
 <br/>
 <br/>
 民进党高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 (2018.11.18)：〝在韩流的夜袭里面，我们不怕敌人刺进我们的心，大家守护民主爱护高雄，这样对不对。〞
 <br/>
 <br/>
 陈其迈在凤山举办〝守护民主、决胜高雄〞选前之夜，邀请前市长陈菊出席，同样找来歌手登台演出，包括先前遭网路霸凌的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_詹雅雯.html" target="_blank">
  詹雅雯
 </a>
 、创作歌手陈明章以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灭火器.html" target="_blank">
  灭火器
 </a>
 乐团，最特别的桥段，还有将跟新北市长候选人苏贞昌，南北串联连线。
 <br/>
 <br/>
 陈其迈竞选办公室总督导 陈启昱：〝苏贞昌未来的苏市长，也要跟其迈进行一个现场连线。〞
 <br/>
 <br/>
 陈其迈阵营预计将吸引20万人，而韩国瑜阵营则预计将吸引10万人以上，高雄市长选情紧绷，双方比人气比气势，要在最后时刻，凝聚支持者力量。
 <br/>
 <br/>
 (新唐人亚太电视 李金菊 林宏观 台湾高雄采访报导)
