### 2018台东国际冲浪赛 精彩赛事缔造新纪录
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】2018台湾国际冲浪公开赛，在台东县
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金樽渔港.html" target="_blank">
  金樽渔港
 </a>
 登场，历经五天的激烈竞争后，WSL世界总冠军陆续出炉，尤其在同一场赛事产生双冠军，再度创下台湾冲浪赛事，新高纪录。
 <br/>
 <br/>
 来自世界各地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冲浪高手.html" target="_blank">
  冲浪高手
 </a>
 ，在东台湾蔚蓝的太平洋里，轻松驾驭长浪，展现好身手。
 <br/>
 <br/>
 游客：〝欢迎大家都可以来这里，看我们台东的好风景，跟厉害的选手，大显身手。〞
 <br/>
 <br/>
 这是2018台东国际冲浪比赛会场，从11月26日开幕后，吸引上千位民众慕名而来，一睹顶尖职业冲浪选手的风采。
 <br/>
 <br/>
 游客：〝每一年都有比赛，我觉得满感动是，这个延续下去会越做越大，让所有选手都可以，有地方可以发挥自己的场地跟舞台，是一个很棒的地方。〞
 <br/>
 <br/>
 台东县县长 黄健庭：〝我们看到这些年下来，台东在世界冲浪的知名度，然后每年来参加的选手的档次、人数、总奖金的金额攀升再攀高，毫无疑问，成为全世界最重要的冲浪比赛之一，所以我们也期许在这个比赛里面，更多的人透过这个比赛，看到台东、认识台东、愿意来台东。〞
 <br/>
 <br/>
 冲浪教练：〝夏天没有颱风来的时间点，是最适合初学者冲浪的时候，因为它又小又缓又顺又平稳。〞
 <br/>
 <br/>
 游客除了可以欣赏激烈的赛事外，场边还有初学者的教学活动，地上模拟冲浪姿势，让大小朋友玩的开心，也让外国选手看见台东之美。
 <br/>
 <br/>
 新唐人亚太电视廖俪芬台湾台东采访报导
</div>
