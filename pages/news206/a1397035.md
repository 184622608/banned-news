### 普悠玛罹难者头七 全线列车16:50齐鸣笛致哀
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】周六是台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 翻车事故后的第七天，也是18位遇难者的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_头七.html" target="_blank">
  头七
 </a>
 ，有台铁司机发起哀悼活动，今天全线火车在事发当时的下午4点50分，同时鸣笛，而台铁也将官方网站从彩色换成黑白，表达对遇难者的哀悼。
 <br/>
 <br/>
 月台传来低沉的鸣笛声，因为这一天是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 翻车事故
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罹难者.html" target="_blank">
  罹难者
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_头七.html" target="_blank">
  头七
 </a>
 ，台铁全线火车在当天事故发生的16点50分，同时鸣笛，表达哀悼。
 <br/>
 <br/>
 民众：〝一种哀悼致意的一种方式，应该说代表台铁对这件事情，是有在重视的。〞
 <br/>
 <br/>
 民众：〝这样对那些不幸的罹难者，也是一种表达哀悼的那个意思。〞
 <br/>
 <br/>
 在当天下午大约4点40分的时候，台铁432次普悠玛列车行经事发地点宜兰新马车站，先鸣长笛长达30多秒，全台1378名司机员则是在胸前配戴黄丝带，表示哀悼。民众则希望台铁能够正视问题，做出改善。
 <br/>
 <br/>
 民众：〝我觉得台铁需要做改善，就是针对普悠玛这次的事件，做一个比较完善的对社会大众的一个交代。〞
 <br/>
 <br/>
 民众：〝还没有发生普悠玛事件的时候，就没有觉得有这么的危机感，可是它发生之后我才觉得，原来他们的程序上还是一些人员的分配，其实还是有很大的缺失。〞
 <br/>
 <br/>
 台铁普悠玛列车翻覆事故，造成不少家庭破碎，全国火车驾驶产业工会希望透过列车鸣笛以及驾驶配戴黄丝带，来表达最沉痛的哀悼，同时祈求所有伤者能够早日康复。
 <br/>
 <br/>
 新唐人亚太电视曾奕豪、赵庭誉台湾台北采访报导
</div>
