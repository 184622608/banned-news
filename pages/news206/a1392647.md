### 竹市童被关阳台骨瘦如柴 饿到吃粪便 生母救援崩溃
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月23日讯】台湾新竹发生3岁男童疑被生父与同居人虐待，被以束带绑住双臂关在阳台，骨瘦如柴、全身是瘀青且裸体，见到妈妈前往探视时，竟饿到拿起地上的粪便往嘴里塞，社工与警方获报后已展开调查侦办。
 <br/>
 <br/>
 据自由时报报导，脸书《新竹爆料公社》接获网友爆料称，友人因先生外遇而离异，2个儿子交给生父和同居人抚养，女儿则是给友人抚养。
 <br/>
 <br/>
 妈妈对两个儿子仅有探视权，此前还会将小孩接回陪伴，发现小孩身上就有多处瘀青，但小孩年纪还小不会表达，遂请小孩的生父注意这个问题，后来要再探视小孩就屡遭刁难。
 <br/>
 <br/>
 日前妈妈的女儿突然说〝晚上睡觉总是梦到弟弟死掉！〞。妈妈决定直接到住处看望小孩是否出事，看到的是小孩全身瘀青裸体、骨瘦如柴地的被束带绑住，挂在阳台，而且手指甲、脚趾甲都是排泄物。
 <br/>
 <br/>
 生父的同居人发现妈妈来时，立即拿剪刀将束带剪断，被放下来的孩子竟然直接捡地上的粪便往嘴里塞！不知让孩子饿了几天，才逼得让他去捡粪便吃！妈妈整个人吓到六神无主、惊慌失措。
 <br/>
 <br/>

 <br/>
 男孩则连站都站不稳，一直哭，随后被紧急送往医院急救，发现全身上下都是伤，除了瘀青、刮伤、甚至连眼睛都有瘀血、脸部也都是瘀青。
 <br/>
 <br/>
 警方与社工已经介入侦办处理，初步发现，大儿子身上没有被虐的迹象。
 <br/>
 <br/>
 新竹市社会处表示，昨（22）日已将男童与男童哥哥安置在亲友家，状况已平安稳定。
 <br/>
 <br/>
 目前，男童父亲及当时在场的陈姓同居人已到警察局制作笔录，并于23日至地检署由检察官讯问。讯后检方依家庭暴力防制法第31条规定，对男童父亲及父之女友核发紧急家庭保护令，并谕知

 </a>
 。
