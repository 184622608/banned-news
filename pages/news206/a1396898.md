### 普悠玛翻车惩处将继续 赖揆：一定要负责
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 翻车意外进入检讨调查阶段，行政院长赖清德今天表示，行政调查完成后，行政惩处还会再继续，该负责的人一定要负起责任，惩处就是要检讨改进，避免事情再次发生。
 <br/>
 <br/>
 行政院长 赖清德：〝等行政调查的这个结果完成之后，行政的惩处还会再继续，还会再继续，该负责的人一定要负起责任。〞
 <br/>
 <br/>
 行政院长赖清德在立法院受访时表示，为了告慰亡者的在天之灵、向社会大众交代，会痛下决心推动检讨改革，解决
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 多年来的沉痾，避免不幸的事件再度发生，已经要求行政院调查小组，务必要客观公正、公开透明、秉公处理，绝不可循私护短，才能以昭公信、解决争端，踏出解决问题的第一步。
 <br/>
 <br/>
 行政院长 赖清德：〝惩处就是为了要检讨改进，避免后续事情的发生，那鹿洁身局长已经请辞获准，那交通部吴宏谋部长，待检讨完毕之后再做决定。〞
 <br/>
 <br/>
 至于未来台铁公司化会不会是检讨方向？赖清德说，台铁全体同仁应该记取惨痛教训，不能流于过去的表面检讨，至于公司化问题，就留给张景森政委所督导的台铁大体检中去讨论，原则上对台铁好，就来做。
 <br/>
 <br/>
 行政院长 赖清德：〝我们希望检讨之后的台铁公司，是一个更安全、更有效率，服务更好的方向来迈进，这样的话才有办法对得起社会大众的支持，跟他们的期待。〞
 <br/>
 <br/>
 赖清德表示，未来会成立交通运输安全委员会，把陆海空都纳入，除了负责安全查核外，如果遇重大事故，它也是一个进行调查的独立机关，这样才有办法取得社会信任，也才能经由检讨改进，让交通运输不断进步。
 <br/>
 <br/>
 新唐人亚太电视陈进交、黄庭锋台湾台北报导
</div>
