### 九合一选举并公投投票 8时到下午4时 勿带手机进入
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】台湾规模空前的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_10案公投.html" target="_blank">
  10案公投
 </a>
 24日上午8时到下午4时投票，将选出1万1047名公职人员。中选会估计，选举票晚间9时开完，公投结果要到25日凌晨2时左右才能宣布。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 包括直辖市长、直辖市议员、县市长、县市议员、乡镇市长、乡镇市民代表、直辖市山地原住民区长、直辖市山地原住民区民代表及村里长共9种选举。
 <br/>
 <br/>
 6个直辖市（台北市、新北市、桃园市、台中市、台南市、高雄市）和3省辖市（基隆市、新竹市、嘉义市）领市长、市议员和里长3张选票；其他13个县辖市领有县长、县议员、乡镇市长、乡镇市民代表和村里长共5张选票。
 <br/>
 <br/>
 此外，直辖市的山地原住民区也是领5张选票，包括市长、市议员、山地原住民区长、山地原住民区民代表、里长。
 <br/>
 <br/>
 根据中选会资料，九合一县市长选举人人数为1910万2502人，选举票达6721万3515张，将选出1万1047名地方公职人员；公投的投票权人有1975万7067人。公职人员选举可投票年龄为20岁以上；公投为18岁以上。
 <br/>
 <br/>
 中选会表示，这次在直辖市、县市设置368个选务作业中心，全台共设置1万5886个投票所，出动29万5904名选务人员，选举规模可说是史上空前。
 <br/>
 <br/>
 公投10案包括国民党提出的反空污、反深澳电厂和反核食公投，下一代幸福联盟提出的爱家3公投（反同）、婚姻平权2公投（挺同）及东奥正名公投、以核养绿公投。
 <br/>
 <br/>
 中选会提醒民众，投票时间为上午8时到下午4时止，要记得携带国民身分证、印章及投票通知单，到指定的投票所投票。投票日不能进行任何竞、助选活动，包括不能用line、脸书拉票。
 <br/>
 <br/>
 中选会说，圈选选票时，请使用选举委员会提供的圈选工具在圈选栏内圈选，切勿使用印章圈选，否则会变成无效票。
 <br/>
 <br/>
 此外，千万不要携带手机及摄影器材进入投票所，否则将处新台币3万元以上、30万元以下罚锾；也不可撕毁选票，否则将处以5000元以上、5万元以下罚锾。
 <br/>
 <br/>
 投票结束后，中选会将先开选举票，再开公投票。选举票预计晚间9时前开完，11时左右宣布最后结果，直辖市部分预估7、8时就可知道结果。公投票的现场开票作业预计午夜12时前开完，最后公布结果可能要到25日凌晨2时。
