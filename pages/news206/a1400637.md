### 投票加上好天气 陈菊、陈其迈都等逾30分钟
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 选举投票日加上好天气，许多民众纷纷出门投票，今年因公投绑大选，投开票所大多大排长龙，总统府秘书长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈菊.html" target="_blank">
  陈菊
 </a>
 与民进党高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 上午依序排队投票，都等了约30分钟以上，并鼓励民众踊跃投票。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈菊.html" target="_blank">
  陈菊
 </a>
 上午到高雄市明诚中学国小部投票，较原订时间迟到近20分钟，她说，实在太累了，起不来。
 <br/>
 <br/>
 投开票所等候投票人潮大排长龙，陈菊依照规定排队，大约等了40分钟，还因公投票多要花一些时间看。陈菊受访时鼓励民众踊跃投票，不管投谁都应勇于表达。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 的妻子吴虹由只能投公投票的女儿陪同到富民路长青中心的投票所投票。吴虹说，与陈其迈没在同一户籍，且她早上还要看诊，所以上午8时30分就先投票。
 <br/>
 <br/>
 陈其迈与就读大三的儿子陈政通一起到左营区新中街青云宫投开票所投票，陈其迈呼吁选民一定要投公投票。陈政通则说，陈其迈是个很温暖、不太会发脾气，对家人很好的父亲。
 <br/>
 <br/>
 陈其迈受访表示，昨晚选前之夜回家后，很快就睡着了，声音也沙哑，一早就起床，稍作准备就出门投票。
 <br/>
 <br/>
 陈其迈说，他这辈子生活从来没放松过，会加紧努力，让家乡更好。台湾是自由民主国家，可以百分之百行使投票权，公投票得来不易，选民投完市长、市议员、里长后，一定要记得投公投票。
