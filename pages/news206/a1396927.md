### 贸易战没影响 美光：台湾是最大生产基地
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美光.html" target="_blank">
  美光
 </a>
 科技公司扩大在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 投资，希望在美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 期间，善用全球布局。至于明年的动态存储器（DRAM）产业前景，美光高层也相当乐观。
 <br/>
 <br/>
 半导体DRAM大厂，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美光.html" target="_blank">
  美光
 </a>
 扩大在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 投资布局，不过产业却频传利空，除了DRAM价格下滑，更有韩厂下砍资本支出，美光全球营运执行副总裁巴提亚强调，明年DRAM市况，相当健康。
 <br/>
 <br/>
 美光全球营运执行副总裁巴提亚 Manish Bhatia：〝(DRAM)需求来自PC区块、移动装置区块、绘图装置区块，还有伺服器以及云端运算。我们认为产业需求相当相当健康。所以我们预期供给方面，可能每年大幅成长约20%，所以我们认为结合(供给需求)这两者情况来看，对DRAM产业都是非常好。 〞
 <br/>
 <br/>
 此外，观察美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 冲击，美光高层表示，台湾是美光最大的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生产基地.html" target="_blank">
  生产基地
 </a>
 ，并不受贸易战影响，但记忆体模组方面，较受冲击，将针对中国大陆以外生产据点，加以因应。
 <br/>
 <br/>
 美光全球营运执行副总裁巴提亚 Manish Bhatia：〝晶圆制造方面，你知道并没有任何影响，台湾已经是全球最大的据点。目前受到关税冲击影响，最主要的产品是DRAM模组(DRAM module)，所以这些产品我们会寻求全球供应链，包括台湾。〞
 <br/>
 <br/>
 美光指出，2019年资本支出，主要用于提升，持续开发DRAM和NAND记忆体技术，产能并不盲目扩大。
 <br/>
 <br/>
 美光科技副总裁徐国晋：〝整个晶圆的生产量上，我们是不会增加的，我们是透过不断的技术升级，来提供我们记忆体单元的成长。〞
 <br/>
 <br/>
 美光全球营运执行副总裁巴提亚 Manish Bhatia：〝所以我们(产能)很弹性，我们也会很有弹性，我们实际上已经下调NAND记忆体资本支出，在目前的年度计划中。〞
 <br/>
 <br/>
 针对业界传出，美光在台湾，提出高于台积电20%以上薪水来挖角，美光人资副总裁(April Arnzen)对媒体提问，并不否认，表示集团在人才招募上相当有野心（aggressive），会用有吸引力的条件，来网罗人才。
 <br/>
 <br/>
 新唐人亚太电视林嘉韦、沈唯同台湾台中报导
</div>
