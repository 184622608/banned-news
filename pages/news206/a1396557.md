### 千年神木见证爱情  15对海内外新人参加
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】号称全台最高海拔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_婚礼.html" target="_blank">
  婚礼
 </a>
 的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里山.html" target="_blank">
  阿里山
 </a>
 神木下婚礼〞，今年迈入第12年。周日(21日)上午，吸引15对国内外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新人.html" target="_blank">
  新人
 </a>
 参加，主办单位特别邀请8年前曾参加婚礼的新人，带着3个儿女〝回娘家〞，一起见证爱的誓约。
 <br/>
 <br/>
 在树龄2300多岁的香林神木下，穿着白纱礼服的新娘子，甜蜜挽着新郎倌，缓缓步入会场，完成宣读
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱情.html" target="_blank">
  爱情
 </a>
 誓言、交换爱的信物，15对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新人.html" target="_blank">
  新人
 </a>
 互许终身。
 <br/>
 <br/>
 新娘 吴宛蓉：〝就是很开心，在大自然的环境下，见证我们的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_婚礼.html" target="_blank">
  婚礼
 </a>
 。〞
 <br/>
 <br/>
 新郎 俞方中：〝非常荣幸能够跟我的老婆，在神木见证下，一起手牵手共度一生。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里山.html" target="_blank">
  阿里山
 </a>
 管理处处长 马惠达：〝今年有15对新人，11对国内、4对海外，包括这个香港、新加坡、日本、韩国；那明年呢我们希望可以扩大在海外的部分，到甚至到10对。〞
 <br/>
 <br/>
 来自日本冲绳县的石川夫妇，曾受过台湾人的帮助，感念台湾人的友善与热情，虽然脚受伤了，还是决定参加这场婚礼；另外一对2010年参加神木下婚礼的夫妻，也带着3个孩子〝回娘家〞。
 <br/>
 <br/>
 石川吉幸的太太 波平梓：〝虽然他的脚骨折了，但是我们爱台湾，还有爱台湾的人，就特地来参加婚礼；台湾的朋友特地来参加我婚礼，就感动了。〞
 <br/>
 <br/>
 新娘 徐如谊：〝99年有来参加我们神木下的婚礼，那我们觉得这种婚礼非常的温馨；那八年过后了，我们生了3个可爱的小宝贝，所以呢我们决定今年要来参加。〞
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千年神木.html" target="_blank">
  千年神木
 </a>
 下，阿里山国中小的同学们，以原民舞蹈为新人献上祝福；15对新人也在家人和朋友的祝福声中，完成终身大事。
 <br/>
 <br/>
 新唐人亚太电视张金洲、方雅娴台湾嘉义采访报导
</div>
