### 交通部次长降任台铁局长 明起采〝双驾驶〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】台铁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 翻覆意外；台铁改革势在必行，今天行政院与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交通部.html" target="_blank">
  交通部
 </a>
 同时召开记者会宣布新任局长人选，交通部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴宏谋.html" target="_blank">
  吴宏谋
 </a>
 表示，而为了展现改革的决心，特地请
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张政源.html" target="_blank">
  张政源
 </a>
 次长以政务次长的高度，来接任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁局长.html" target="_blank">
  台铁局长
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交通部.html" target="_blank">
  交通部
 </a>
 长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴宏谋.html" target="_blank">
  吴宏谋
 </a>
 ：〝非常感谢我们的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张政源.html" target="_blank">
  张政源
 </a>
 政次、张政次，他愿意来承担未来台铁的改造工作，带领台铁走出阴霾。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁局长.html" target="_blank">
  台铁局长
 </a>
 鹿洁身日前请辞获准后，行政院与交通部29日上午同时举行记者会宣布，由交通部政务次长张政源接任台铁局长，交通部长吴宏谋表示，张政源有运输专业背景，也曾担任台铁局台北站站长，有台铁背景，却没有台铁包袱，对于凝聚员工士气也有很大的帮助，是适合的人选。
 <br/>
 <br/>
 交通部长 吴宏谋：〝台铁局大家都知道，现在它是长期累积的一些，系统性、结构性的问题，真的是需要大刀阔斧，这个部份我们当然需要找对人，才能够做对事。〞
 <br/>
 <br/>
 吴宏谋表示，新任台铁局长上任后，首先就是要在人、车、路与管理部分全面改善，包括车辆安全的提升， ATP系统的体检，铁路危险路段的优化改善，还要检讨台铁内部工作分配是否合理，今年行政院已经核准晋用将近3千名台铁员工，相信会有所帮助。
 <br/>
 <br/>
 交通部长 吴宏谋：〝我们在整个ATP这个功能没有完整之前，我们用双驾驶的方式，我们总需要未来的驾驶员，他能够很专心、不要分心，所以这一部份的话，我们现在包括太鲁阁号、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 号，我们都用双驾驶的方式来进行。〞
 <br/>
 <br/>
 吴宏谋表示，为了行车安全，从30日起将改采双驾驶，人力上没有太大问题，至于普悠玛翻覆事故所凸显的过去采购、验收过程是否合乎规定等问题，交通部都会彻底调查，并尽速改善。
 <br/>
