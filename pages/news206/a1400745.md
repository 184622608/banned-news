### 韩国瑜胜选 民进党执政20年高雄变天
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一大选.html" target="_blank">
  九合一大选
 </a>
 中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 失守了执政20年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄市.html" target="_blank">
  高雄市
 </a>
 ，由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 当选市长。民进党候选人陈其迈发表感言时说，不管支持蓝绿，都是高雄市民，从明天开始为韩国瑜市长加油，为高雄加油。
 <br/>
 <br/>
 晚上8点45分，陈其迈与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 的得票差距来到10万票，陈其迈抵达竞选总部，宣布败选，并向支持群众表示感谢。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄市.html" target="_blank">
  高雄市
 </a>
 长候选人 陈其迈：〝我作为一个高雄人，不管我的工作在哪里，我以高雄人为荣，也以这个伟大城市为荣。更以所有跟其迈一起打拼的市民为荣。〞
 <br/>
 <br/>
 另一边，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 候选人韩国瑜则是自行宣布当选。
 <br/>
 <br/>
 高雄市长当选人 韩国瑜：〝广大的280万高雄市民，请你们相信新的高雄市政府的团队，请你们做我们的后盾，请你们跟我们一起来努力。[CUT打造一个崭新、耀眼、亮丽的高雄。]〞
 <br/>
 <br/>
 来到高雄只有1年2个月，韩国瑜感谢市民给他机会当市长，他说心中没有一丝喜悦，反而感到无比沉重，还要马上筹组小内阁。
 <br/>
 <br/>
 这次九合一选举，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 在高雄的20年执政画下句点。而韩国瑜是否真的符合高雄人的期待，能改变高雄，考验才要开始。
 <br/>
 <br/>
 新唐人亚太电视台湾高雄采访报导
</div>
