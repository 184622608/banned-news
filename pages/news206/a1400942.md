### 外力介入选举？台国安局：是人民对经济民生表态
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 落幕，境外势力介入选举、言论自由等议题仍受关注。国安局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭胜竹.html" target="_blank">
  彭胜竹
 </a>
 昨(26日)在立院备询时强调，国安局不会踩侵犯言论自由的红线，绝不会监控言论自由，并强调这次选举主要是内政、经济与民主层次，是人民对经济民生的表态。
 <br/>
 <br/>
 中央社报导，立法院外交及国防委员会26日审查2019年度中央政府总预算案关于国家安全局收支机密及公开部分，邀请国安局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭胜竹.html" target="_blank">
  彭胜竹
 </a>
 、副局长陈国恩、柯承亨、陈文凡等人列席。
 <br/>
 <br/>
 这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 前，有媒体引述不具名的〝国安人士〞说法，指称这次选战，会成为中共统战的成绩单，且在选举前利用网军散播假讯息扩大攻击，民进党也多次强调，有外力介入台湾选举，且指涉中共，而总统蔡英文也在替候选人站台时多次提及此事。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 立委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江启臣.html" target="_blank">
  江启臣
 </a>
 询问境外势力介入选举情形表示，除中共外，包括美国、日本等其他国家是否有介入台湾选举。彭胜竹说，是〝境外敌对势力〞，不是〝境外势力〞，主要所指的境外敌对势力，〝中共就是我们的敌对势力〞。
 <br/>
 <br/>
 彭胜竹强调，〝到目前为止，我们没有非常的明显迹象，表示说其他国家有任何对我们有敌对势力的影响〞，但国安局仍持续侦蒐相关情资。
 <br/>
 <br/>
 国民党立委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吕玉玲.html" target="_blank">
  吕玉玲
 </a>
 询问，执政党选举挫败，国安局是否有了解原因，彭胜竹表示，这次选举主要重点放在经济、民生，〝我们看到选民对经济、民生的表态〞。
 <br/>
 <br/>
 吕玉玲也问，选前有人说有些当选人是中共培植的人？彭胜竹表示，到目前为止，没有查到特定的人。
 <br/>
 <br/>
 民进党籍立委何欣纯认为，心理战、舆论战可能都会影响国安，如何因应；彭胜竹表示，会强化与宣导，让民众了解中共的敌对势力，并宣导如何辨识网路讯息的真伪。
 <br/>
 <br/>
 国安局副局长柯承亨答覆民进党立委邱志伟询问时说，国安局尊重言论自由，不会踩了侵犯言论自由的红线，〝监控言论自由，国安局不可能做这样的事情。〞
 <br/>
 <br/>
 至于国安局在选前时指称，有所谓33件外力介入选举，而调查局也指出，全台有6案中有县市首长层级涉案，还有外力、中资介入。国安局官员在回应时，推给调查局，称是吕局长答覆的议题，而侦办也由检调在处理。
 <br/>
 <br/>
 (责任编辑：夏明义)
