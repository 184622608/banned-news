### 韩国瑜高雄首场造势晚会 人山人海热情相挺
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】国民党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄市.html" target="_blank">
  高雄市
 </a>
 长参选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 首场造势晚会26日晚间在高雄
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凤山.html" target="_blank">
  凤山
 </a>
 登场，现场人山人海热情相挺，主办单位估算会场涌进逾5万人，几乎瘫痪附近交通。
 <br/>
 <br/>
 26日晚起至11月中旬，〝高雄三山之友会〞分别在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凤山.html" target="_blank">
  凤山
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旗山.html" target="_blank">
  旗山
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冈山.html" target="_blank">
  冈山
 </a>
 地区各举办一场大型造势晚会，昨日在凤山区维新桥下空地办第一场造势活动。
 <br/>
 <br/>
 主办方原本预放8千张张塑胶红椅，后来又追加到2万张红椅，晚会愈晚人潮愈多，造成附近道路塞车，交通瘫痪，不少进不了会场民众在外围大喊〝加油〞、〝冻蒜〞，主办单位估算会场涌进逾5万人。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 由前立法院长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王金平.html" target="_blank">
  王金平
 </a>
 等人陪同进场时掀起晚会高潮，现场旗海飘扬，〝韩国瑜冻蒜〞声浪高涨。
 <br/>
 <br/>
 <b>
  相关视频：
 </b>
 <center>
 </center>
 <br/>
 韩国瑜登台演说时，现场支持者欢呼声不绝于耳，此情此景韩国瑜也难掩心情的澎湃，停顿约30秒后，他一开口即连喊3次〝伟大的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄市.html" target="_blank">
  高雄市
 </a>
 人民〞。
 <br/>
 <br/>
 他说，高雄是一个巨人，有山有海，有空港、海港，土地是4个新加坡大、10个台北市大，有完整工业与农渔业，伟大的高雄巨人要清醒了，〝我们要拒绝贫穷、拒绝又老又穷，要迎向光荣富庶〞。
 <br/>
 <br/>
 他又说，高雄以前钱淹肚脐，有工作机会，外县市的人到高雄找工作，但如今没有太多工作机会提供子弟足够养分，他们被迫离乡背井讨生活；要让高雄巨人站起来，要如何拚经济？他没有高深方法与学问，他认为，只要东西卖得出去、人进得来，高雄就发大财。
 <br/>
 <br/>
 韩国瑜指出，如果他选上，明年2月就亲自陪高雄农渔民到外面抢订单，这是〝蛋白区〞；高雄的〝蛋黄区〞则要让观光客及投资客大量进来，高雄要全面开放，大大张开臂膀用热情欢迎全世界走进来。
 <br/>
 <br/>
 〝自己的高雄自己救，自己的未来自己拚〞，韩国瑜期待2018年高雄市长选举是场非常干净与健康的选举，他坚持一瓶矿泉水选到底，没有竞选总部、不插旗、没有竞选总干事、不买票。
 <br/>
 <br/>
 他认为，如果2018年选举真的是一瓶矿泉水赢得胜利，这代表高雄市民的伟大，不是他厉害，他只是个卖菜郎，高雄市长这一役全世界都在看，〝让我们一起写下选举史上这伟大的一页，让我们证明高雄是民主圣地〞。
 <br/>
 <br/>
 韩国瑜向高雄乡亲喊话，让高雄这巨人醒过来吧，再也不要碰政治了，让高雄市民头壳愈来愈聪明，经济100分，让全台湾、全世界对高雄刮目相看，〝勇敢的高雄市民起来吧！让我们一起努力吧！〞
 <br/>
 <br/>
 王金平则说，谁说韩国瑜只有〝空军〞，没有〝陆军〞，〝韩流〞已形成且锐不可挡，现场我们每个人都是韩国瑜的〝陆军〞，他呼吁大家都当〝小石头〞撑起韩国瑜这颗〝大石头〞形成山丘。
 <br/>
 <br/>
 这场造势活动，蓝营评估成功激出支持群众，象征〝高雄市长选战，已进入短兵相接的肉搏战〞。
