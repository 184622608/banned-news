### 台湾选举深化民主  江启臣：选民大于政党
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】最近，台湾九合一选举顺利举行，国民党赢得地方县市首长多数。对于这次选举，国民党政策委员会执行长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江启臣.html" target="_blank">
  江启臣
 </a>
 、驻美代表高硕泰及美方学者分别发表了不同看法。
 <br/>
 <br/>
 国民党政策委员会执行长、立法委员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江启臣.html" target="_blank">
  江启臣
 </a>
 应邀赴美，拜会美国行政部门及国会，并会晤当地侨界，介绍国民党对这次胜选的看法。他认为，这次选举是台湾民主的深化，并透露出一个重大信息，即
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选民.html" target="_blank">
  选民
 </a>
 大于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政党.html" target="_blank">
  政党
 </a>
 。
 <br/>
 <br/>
 台湾立法委员 江启臣：〝过去选民容易被政党所影响，或者操作。但是这一次选民想要展现的是，我不想再让你操作，我要当你的主人，所以我不是你的小弟啊，我是你的主人。〞
 <br/>
 <br/>
 中华民国驻美代表高硕泰日前投书华盛顿邮报指出，刚落幕的九合一选举表明，台湾选民有能力透过选举，自由、和平表达自我意志，台湾是〝民主灯塔〞。
 <br/>
 <br/>
 中华民国驻美代表 高硕泰：〝这是一个民主的历练，一个非常重要的洗礼，而且我们觉得也非常的骄傲与欣慰。但是，我们还是政府的立场不变，尤其在处理台美关系、在处理两岸问题，我想善意、承诺不变。〞
 <br/>
 <br/>
 对于这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾选举.html" target="_blank">
  台湾选举
 </a>
 成功，美国学者表示祝贺，认为选举结果是台湾民主强大的体现。
 <br/>
 <br/>
 美国约翰霍普金斯大学客座教授 卜道维：〝选举结果清楚回答了这个问题，台湾继续拥有两个强大的民主政党，这为台湾未来民主的延续提供了基础。〞
 <br/>
 <br/>
 学者认为，这次选举对美台关系影响甚微。实践已证明，无论哪一政党执政，美台之间都可维持紧密关系，而北京干预，只会有反效果。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
 <br/>
</div>
