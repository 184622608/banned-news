### 林飞帆批柯P两岸一家亲 恐成中共渗透缺口
------------------------

<div class="wysiwyg">
 【新唐人2018年09月23日讯】台湾太阳花学运代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林飞帆.html" target="_blank">
  林飞帆
 </a>
 ，日前投书〝外交家〞杂志，称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 提倡的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_两岸一家亲.html" target="_blank">
  两岸一家亲
 </a>
 〞，可能会为台湾反制中共侵略的决心，带来不确定性。对此，柯文哲今天受访时表示，两岸本来就是互相攻防，也会有各种意见。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ：〝两岸本来就是互相攻防，然后有各种意见，所以每个人讲的我们也是听一听，我们也是尊重。〞
 <br/>
 <br/>
 太阳花学运代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林飞帆.html" target="_blank">
  林飞帆
 </a>
 日前投书到国外杂志，提到台北市长柯文哲认可〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_两岸一家亲.html" target="_blank">
  两岸一家亲
 </a>
 〞，可能为台湾反制中共侵略的决心带来不确定性，因为国民党在短期内不可能重回执政，但柯文哲民粹主义的领导风格和无党派性质，却是未来十年，台湾总统的强大潜在挑战者，恐成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共渗透.html" target="_blank">
  中共渗透
 </a>
 台湾缺口。
 <br/>
 <br/>
 台北市长 柯文哲：〝我是比较务实跟对话的态度，来处理我们面对的问题，但是你不能够用务虚跟欺骗的手段，来处理我们面对的困难你知道，面对我们自己的问题，所以明白讲，你不可以一边取消征兵制，然后要降低国防经费然后喊台独，这个不负责任。〞
 <br/>
 <br/>
 对此，民进党台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚文智.html" target="_blank">
  姚文智
 </a>
 认为，即使台湾在外交处境非常困难，仍然要坚持坚守国家尊严、民主自由，才能无惧中共在各方面的挑战，人民应该拒绝，轻易接受〝两岸一家亲〞的首都市长，才能内外同时抵抗中共的压力。而北市府研考会公布最新民调，有超过6成的民众对于柯文哲的表现表示满意，国民党台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 认为，柯文哲一直在滥用行政资源，吹捧造神，是个政治骗子。姚文智则说，这民调是粉饰太平，像是为了选举需要。
 <br/>
 <br/>
 台北市长 柯文哲：〝60%的满意度叫造神？我要造神我就写95或是92%，这才是造神。〞
 <br/>
 <br/>
 柯文哲说，每个人都可以发表意见，也不见得别人要讲什么就要去评论，何必把人生搞的这么痛苦，大家快乐一点。
 <br/>
 <br/>
 新唐人亚太电视曾奕豪、黄庭锋台湾台北报导
</div>
