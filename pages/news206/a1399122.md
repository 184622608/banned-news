### 走过一甲子历史 味全埔心牧场传26亿易主
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】有65年历史的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_味全.html" target="_blank">
  味全
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埔心牧场.html" target="_blank">
  埔心牧场
 </a>
 ，传出要被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三立集团.html" target="_blank">
  三立集团
 </a>
 以26.63亿元(约8535万美元)买下，若交易完成，广达6万8千多坪的土地将易主，有员工如果获资金挹注，为民众打造更多元化游乐产业，对台湾休闲发展是一大加分，甚至可吸引国外游客访台观光。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_味全.html" target="_blank">
  味全
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埔心牧场.html" target="_blank">
  埔心牧场
 </a>
 当初配合政府酪农产业政策开发，协助培育近2千名杰出酪农，让处于亚热带的台湾生乳产量，每天可达1000到1100 吨左右。
 <br/>
 <br/>
 后来，埔心牧场转型成为观光休闲牧场，成为许多亲子出游的选择，但历经30年经营没有突破性发展。味全公司表示，适逢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三立集团.html" target="_blank">
  三立集团
 </a>
 有开发计画需求，为发挥埔心牧场土地的经济效益，促进地区繁荣发展，决定转让给三立集团，未来将更专注在食品本业的经营与发展。
 <br/>
 <br/>
 埔心牧场人士12日晚间表示，大部分员工事前都不清楚，但相关讯息传闻已久，有员工认为只要可促进台湾观光产业发展都很欢迎。如果获资金挹注，为民众打造更多元化游乐产业，对台湾休闲发展是一大加分，甚至可吸引国外游客访台观光。
 <br/>
 <br/>
 至于牧场部分是否保留，埔心牧场人士说，目前还未获相关讯息，但牧场可有更多元化发展，员工多半抱持正面肯定态度。
 <br/>
 <br/>
 住牧场附近的王姓公务人员表示，听闻相关消息相当讶异，毕竟这是历史悠久的品牌，但内部设施的确稍老旧；透过集团进驻，希望让牧场再现繁荣，保留牧场，这是〝桃园人的共同回忆〞。
 <br/>
 <br/>
 据了解，这项交易如顺利执行，初估获利金额约新台币10亿元左右，有助味全改善财务状况，每股盈余贡献约1.98元；三立集团目前尚未就此消息回应。
 <br/>
 <br/>
 (责任编辑：夏明义)
