### 爱河畔办交接典礼 韩国瑜要搭爱之船进场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】高雄市长当选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 25日将走马上任，交接小组今(3日)到市府与秘书长赵建乔为窗口的小组洽谈交接事宜。在近1个小时的首次交接事宜会议后，双方敲定在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱河畔.html" target="_blank">
  爱河畔
 </a>
 举办交接典礼，规划韩国瑜将搭乘太阳能爱之船沿爱河口进场。
 <br/>
 <br/>
 中央社报导，交接小组由召集人潘恒旭率领准高市府新闻局长王浅秋等人，下午2时30分抵达高市府四维行政中心。
 <br/>
 <br/>
 赵建乔表示，新任市长和副市长等宣誓就职交接典礼，经开会共识择定在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱河畔.html" target="_blank">
  爱河畔
 </a>
 的河东路到市中路的路段举行，舞台和交通及维安相关问题和规划大致谈妥。
 <br/>
 <br/>
 赵建乔指出，会议重点是市长当选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 规划将自高雄港栈贰库搭乘太阳能爱之船，沿爱河口抵达就职交接典礼会场，乘船等维安已特别规划，并有多个维安和检查点，安全最重要。由于会场为开放场域，也欢迎民众参加，当天节目有乐团及原住民歌唱演出。
 <br/>
 <br/>
 为确保乘船进场等程序顺利进行，赵建乔说，爱河河道管制和气象条件等都会考量，并于12月18日按照全部程序试乘演练一遍，也会有备案。
 <br/>
 <br/>
 王浅秋表示，规划在室外空间举办是希望有更多的民众参与，并希望以不同方式登场，因高雄是海洋城市，由河面搭船进场，展现不同活力和创意，有象征意义，也呼应韩国瑜先前提出的打造爱河爱情产业链，带动观光。
 <br/>
 <br/>
 至于观礼贵宾名单，王浅秋说，会有各行各业的代表，会再讨论，最近几天会敲定，韩国瑜的家人也会陪同出席。
 <br/>
 <br/>
 (责任编辑：夏明义)
