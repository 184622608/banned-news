### 台湾宜花东高屏旅宿 观光局最高补助1500元
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】中华民国交通部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观光.html" target="_blank">
  观光
 </a>
 局投入9500万元，鼓励民众自由行住宿、团体旅游，观光局29日召开记者会宣布补助内容时表示，期望透过短期奖励措施，鼓励国内民众前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宜花东高屏.html" target="_blank">
  宜花东高屏
 </a>
 地区旅游，刺激扩大在当地的旅游消费。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观光.html" target="_blank">
  观光
 </a>
 局指出，此次〝前进宜花东．高屏暖冬游〞方案分别订定〝前进
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宜花东高屏.html" target="_blank">
  宜花东高屏
 </a>
 暖冬游住宿优惠措施补助要点〞及〝奖励旅行业推广宜花东高屏暖冬旅游补助要点〞。
 <br/>
 <br/>
 观光局表示，实施期间自今年11月1日起至12月31日止；此外，考量台湾目前60岁以上的乐龄族已达总人口20%以上，平日时间较为充裕，方案中也针对乐龄族群提出加码优惠，希望为国旅市场挹注新活力。
 <br/>
 <br/>
 观光局说，自由行住宿优惠方面，国人在周日至周五期间，住宿宜、花、东、高、屏等地参与方案的旅宿，每房最高1000元，60岁以上者加码每房最高1,500元，另补助交通费用每房最高1000元。
 <br/>
 <br/>
 该局又指出，团体旅游两天一夜以上且放假日未超过一天，奖助旅行社以每位国人旅客每日最高500元，乐龄族每日最高1000元，每团补助上限为3万元，每一家旅行社以补助10团为限。
 <br/>
 <br/>
 观光局还表示，宜花东高屏观光资源丰富，除了观光局优惠方案外，部分观光旅馆也主动提供每日限量3折住宿优惠，及台东3天2夜优惠等业者措施。
 <br/>
 <br/>
 最后，观光局也说，提出观光游乐业乐龄优惠，协调观光游乐业者放宽博爱票年龄从65岁调降至60岁，也规划〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾观巴.html" target="_blank">
  台湾观巴
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乐龄旅游.html" target="_blank">
  乐龄旅游
 </a>
 行程，加强行销宣传及媒体露出，以及温泉美食推广活动，由温泉业者与相关产业结合推出住宿、伴手礼及餐饮优惠方案，促进温泉区旅游。
 <br/>
 <br/>
 (责任编辑：卢勇信)
</div>
