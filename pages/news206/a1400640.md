### 投票日好天气大排长龙 中选会：下午4时前排队都可投票
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 选举
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 日加上好天气，许多民众纷纷出门投票，今年因公投绑大选，投开票所大多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大排长龙.html" target="_blank">
  大排长龙
 </a>
 。中选会表示，下午4时已经抵达投票所的民众仍可以投票，届时队末也会有警察或选务人员确认。
 <br/>
 <br/>
 中央社报导，天气晴朗，各地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 所都出现排队人龙，且等候排队的时间比往年都还要长，至少需花半小时到一个小时，许多民众对于公投流程、多达10案的内容不熟悉是原因之一。
 <br/>
 <br/>
 中选会说明，基于选务中立，在投票所选务人员不会提供公投内容谘询，因此请民众在进入投票所前，先了解公投案相关内容，决定投票意向。
 <br/>
 <br/>
 中选会进一步说明，为了加快投票速度，对于选举人数较多的大型投票所，中选会在圈票处已增设遮屏，以方便民众圈票；今日民众投票情形踊跃，遇有排队情形，请耐心等候，进入投票所后，也请配合选务人员的引导，进行领票及投票，以节省民众宝贵的时间。
 <br/>
 <br/>
 中选会副主委陈朝建补充说明，选举人于规定时间前，到达投票所投票，即投票时间截止前已经到场排队的选举人、投票权人仍可投票，下午4时一到也会指派警卫或其他工作人员站于队末，作为确认。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 选举与10案公投今天投开票，将选出1万1047名公职人员。投票时间上午8时到下午4时，中选会估计，选举票晚间9时开完，公投结果要到25日凌晨2时左右才能宣布。
 <br/>
 <br/>
 (责任编辑：卢勇信)
</div>
