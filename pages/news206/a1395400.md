### 离奇！北市消防车撞电线杆 小队长抛出车被辗身亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】台北市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北投区.html" target="_blank">
  北投区
 </a>
 大度路今(15)日上午发生一起离奇交通事故，消防局光明分队小队长李耀文驾驶水箱车前往关渡验车厂验车，返队途中竟自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_撞电线杆.html" target="_blank">
  撞电线杆
 </a>
 ，当场无呼吸心跳，经送医抢救不治，事故原因仍待后续调查。
 <br/>
 <br/>
 综合媒体报导，现年45岁的李耀文今天上午8时45分驾驶水箱车前往关渡验车厂验车，随即在上午9时10分返队途中于大度路一段竟不明原因自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_撞电线杆.html" target="_blank">
  撞电线杆
 </a>
 。
 <br/>
 <br/>
 李耀文被摔出车外当场已失去呼吸心跳，被紧急送往荣总急救，院方于上午10时15分宣告不治。
 <br/>
 <br/>
 有目击者说，当时消防水箱车在等红灯，车头突然往前升起，司机撞破车头挡风玻璃摔在地上，水箱车仍继续前行，辗过司机后再撞上电线杆才停下来。
 <br/>
 <br/>
 警方表示，目前仍在厘清详细案发原因。
 <br/>
 <br/>
 (责任编辑：叶日观)
