### 朝鲜非核化无进展 日媒：川普政府明春重启美韩军演
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】日本共同社根据政府消息人士透露，美国川普政府自9月以来已多次告知日本政府，若朝鲜非核化没有进展，目前中断的大规模美韩联合军事演习〝有可能最快明年春天重启〞。
 <br/>
 <br/>
 共同社报导，政府消息人士12日透露，美方人士提及可能重启的演习有2个，一个是代号〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关键决断.html" target="_blank">
  关键决断
 </a>
 〞的指挥所演习，另一个是代号〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鹞鹰.html" target="_blank">
  鹞鹰
 </a>
 〞的野外机动训练。〝关键决断〞往年在2至3月前后举行，〝鹞鹰〞则于3至4月前后实施。
 <br/>
 <br/>
 而韩国国防部计划在12月1日前决定是否重启大规模演习，美国因此在期限前向日方说明可能重启演习的想法，以向朝鲜施加军事性压力。
 <br/>
 <br/>
 报导说，日方很难断定朝鲜是否积极致力于非核化，而曾表态中止军演的川普的忍耐也是有限的。
 <br/>
 <br/>
 报导说，对一向维持施压朝鲜的日方而言，这是美国不允许朝鲜拥核的态度体现，并给予正面看法。而日方也设想美国朝鲜关系若再陷紧绷，计划与美方紧密协作。
 <br/>
 <br/>
 朝鲜核子与飞弹问题方面，日本政府内部多数意见认为，若美国不施加相当程度的压力，就无法实现朝鲜的完全非核化目标。但也有人忧心〝若在美国朝鲜谈判没有进展的情况下，继续中断大规模军演，或对朝鲜有利〞。
 <br/>
 <br/>
 (责任编辑：卢勇信)
