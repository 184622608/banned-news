### 台九合一选举接近 反贿广播行动车起跑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】11月24日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 接近，法务部防范境外势力干预选举的同时，检方也大动作查贿，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台南.html" target="_blank">
  台南
 </a>
 ，今天有反贿广播行动车起跑，希望端正选风。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台南.html" target="_blank">
  台南
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地检署.html" target="_blank">
  地检署
 </a>
 检察长 郑铭谦：
 <br/>
 <br/>
 〝包括买票包括暴力介入，包括抹黑、妨害选举公正公平的这些的不法的行为，就像垃圾一样，我们用巡回的垃圾车环保车，把它载出去把它扫出去，让我们台南市的选举是干净的。〞
 <br/>
 <br/>
 台南市代理市长 李孟谚：
 <br/>
 <br/>
 〝大家通力合作，能够让我们台南的选风更加端正，给我们下一代一个，更健康干净跟阳光的环境。〞
 <br/>
 <br/>
 台南市环保局长 林健三：
 <br/>
 <br/>
 〝这一次特别增加了一个LED在我们车首这边，将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反贿选.html" target="_blank">
  反贿选
 </a>
 相关口号再进行宣导。〞
 <br/>
 <br/>
 台南地检署透过各项宣导活动，传递反贿选资讯。 民众一旦发现贿选可以检举，检举奖金从50万元到1000万元。
