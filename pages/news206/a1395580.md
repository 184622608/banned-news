### 台九合一选举接近 反贿广播行动车起跑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】经过一个月的跟监埋伏，桃园警方15日傍晚直捣平镇毒窟，现场逮捕张姓男子并查获毒品恺他命及小恶魔毒品咖啡包，讯后依毒品罪嫌送办。
 <br/>
 <br/>
 据中央社报导，桃园市警察局刑事警察大队副大队长邓中义今天受访表示，警方日前接获情资指出，平镇区一名40岁张男出手阔绰，平日就以高级宾士轿车代步贩卖毒品，刑事警察大队侦一队派员跟监埋伏，搜集张男的犯罪事证。
 <br/>
 <br/>
 邓中义表示，警方经过一个多月的搜证，昨天傍晚联合八德及平镇分局，持搜索票至张男住处搜索，并查获第三级毒品恺他命11包，重1653.55公克，小恶魔毒品咖啡包100包，重917公克，另外还有新台币22万元、电子磅秤、分装袋及点钞机等多项证物。
 <br/>
 <br/>
 邓中义指出，警方调查发现，张男平日在大卖场上班，利用下班时间兼差贩毒，却因此染上毒瘾，而结交多年的女友也在去年底罹患癌症，但张男仍继续以高级宾士轿车代步贩毒。警方讯后依毒品危害防制条例将张男移送桃园
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地检署.html" target="_blank">
  地检署
 </a>
 侦办。
 <br/>
 <br/>
