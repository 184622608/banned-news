### 台铁列车翻车事故 影片曝光：嘎嘎怪声爆火花(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车事故，22日上午5时最新统计，共造成205人死伤，包括18人死亡、187人受伤(10人重伤)。事故发生当下的监视器影片曝光，普悠玛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 撞击力之大，连月台上的柱子都跟着倒塌。
 <br/>
 <br/>
 中央社报导，当时事故发生前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车曾与区间列车交会，一名蔡姓女子转述，当时70岁的父亲搭乘区间车要回家，区间车在新马站与事故列车交会，父亲曾听到事故列车发出嘎嘎的奇怪声响，且前面车厢出现火花。
 <br/>
 <br/>
 从曝光的影片可见，普悠玛列车行经过弯处，车身没有如预期转弯，而是笔直撞上月台，车头翻覆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 ，火花四溅还扯倒月台柱子，掀起一阵浓烟。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 稍早表示，普悠玛列车驾驶在事故前，曾反应行驶时动力不足，详细原因有待进一步调查，相关单位将连夜抢修。
 <br/>
 <br/>
 凌晨5时12分台铁已抢通西线恢复单线双向行车。
 <br/>
 <br/>
