### 儿女代父鞠躬致歉 普悠玛司机员发4点声明
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车翻覆事故，司机员尤振仲儿女26日下午3时代表父亲对外宣读4点声明，对于意外发生事故造成许多家庭破碎，甚至无法弥补的憾事，内心深感万分歉疚与不安，也致上无限歉意。
 <br/>
 <br/>
 尤振仲的儿女神情严肃向媒体表示，事情发生至今，父亲对意外感到相当自责且难过，想向死伤者的家属致上最深的歉意。
 <br/>
 <br/>
 尤振仲的4点声明，包括对于这一次发生意外事故的罹难者及家属，以及当天的乘客，致上最深的歉意、对不起。对于意外发生事故造成许多家庭的破碎，甚至无法弥补的憾事，内心深感万分歉疚与不安。
 <br/>
 <br/>
 尤振仲指出，身为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车的驾驶员，他承诺不会逃避、隐匿或是说谎，并坦然面对且配合司法调查，负起应该要负的责任。
 <br/>
 <br/>
 (责任编辑：叶日观)
