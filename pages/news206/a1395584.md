### 抢年轻选票 台候选人搭上RAP嘻哈饶舌风潮
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】选举倒数40天了，年轻人手中将近700万张的选票，成为各县市首长候选人的兵家必争之地。为了取得年轻族群共鸣，也有不少政治人物，竞选歌曲就搭上了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嘻哈.html" target="_blank">
  嘻哈
 </a>
 饶舌风潮。一向受到年轻人欢迎的台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ，也没有放过小试身手的机会。
 <br/>
 <br/>
 打着节奏唱着RAP，台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 进录音室，录竞选歌曲的片段，在周一凌晨曝光。他找了专业的饶舌团队，为他量身订做歌曲，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 推出竞选歌曲后，柯文哲也搭上现在最热的饶舌风潮。
 <br/>
 <br/>
 台北市长 柯文哲：〝参观那一些我们叫次文化团体，有很多什么街舞啦、RAP啦，还有一些涂鸦，有去拜访一些次文化团体，就去拜访他们，顺便被抓去录的一个影片。〞
 <br/>
 <br/>
 受到韩国和中国大陆的饶舌选秀节目影响，台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嘻哈.html" target="_blank">
  嘻哈
 </a>
 饶舌的市场打开，是年轻群众们关注的流行文化。选举年，政治人物们为了抢年轻选票，也让嘻哈音乐烧进政坛。
 <br/>
 <br/>
 高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 竞选歌曲《高雄发大财》：〝东西卖得出去，人进得来，高雄发大财，Let's get that money babe.〞
 <br/>
 <br/>
 高雄市长候选人韩国瑜的歌曲，将政见作为副歌歌词唱出来，伴着浓厚嘻哈风格的编曲。新北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侯友宜.html" target="_blank">
  侯友宜
 </a>
 ，则是直接在活动上，现场演出，搭上日前热烧议题－－深澳电厂。
 <br/>
 <br/>
 新北市长候选人 侯友宜：〝烧炭发电，空气很脏，出门回来，要洗头发。HEY！PM2.5，别来搞鬼，撑到生病，那就不行。〞
 <br/>
 <br/>
 另外台北市长候选人丁守中，则是找幕僚与国民党政治人物来帮忙唱RAP。
 <br/>
 <br/>
 台北市长候选人丁守中竞选歌曲《台北的未来在手中》：〝台北的未来在你手中，台北的未来在我手中，他为您守候，他为您守梦，他是台北的未来丁守中。大家好，我是台北市长参选人，丁守中。〞
 <br/>
 <br/>
 竞选年底县市首长的政治人物们，接连出招，搭上嘻哈风潮，是否能成功吸到年轻人选票，40天后，就会揭晓。
 <br/>
