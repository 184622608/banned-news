### 黄金周造势 韩国瑜〝快乐出帆〞 陈其迈weCARE游行
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】国民党高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 黄金周动员造势大会17日晚间在凤山举行，蓝营预告韩国瑜将有政见铺陈，还有组曲演唱，展望高雄幸福繁荣。民进党候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 将参加民间发起的weCARE游行，周日回防冈山，与谢长廷、陈菊共同造势。
 <br/>
 <br/>
 选前黄金周末，蓝绿阵营大规模动员造势，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 造势大会活动地点在高雄市议会旁空地，这块宽敞的空地，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 阵营在10月中旬也曾在此造势。两场活动的气势和规模、人潮势必再成为比较的话题。
 <br/>
 <br/>
 韩国瑜团队公布今晚凤山造势晚会，请来艺人暖场，还有张善政、谢龙介等来宾助讲。韩国瑜将在晚间九点上台，发表政策规划。
 <br/>
 <br/>
 韩国瑜团队发言人 陈宜民：〝从凤山的三万人，到旗山的三万人，到冈山的十万人，所以再回来到凤山，我想很多全高雄市区的朋友，应该是都会来帮韩国瑜加油。〞呼吁民众为减少交通壅塞，尽量搭乘大众交通工具。
 <br/>
 <br/>
 他说，韩国瑜将在晚上会铺陈政见并透过专家背书擘画高雄的发展，同时拉抬胜选气势。由于是选前的大造势活动，主办单位16日提前在现场摆满胶椅，场面壮观。
 <br/>
 <br/>
 <center>

   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-17/p9130641a940605954-ss.jpg"/>

    国民党高雄市长候选人韩国瑜黄金周动员造势大会17日 晚上在凤山举行，现场安排组曲演唱，展望高雄幸福繁荣。（韩国瑜竞选总部提供/中央社）
   </font>
  </a>
 </center>
 <br/>
 不同的是，今天的造势活动将有代表高雄过去、现在与未来组曲的演唱，韩国瑜在脸书预告并希望民众一起唱出〝黄昏的故乡 〞、〝快乐的出帆〞与〝世界第一等〞，要和大家一起努力，拚出幸福繁荣的高雄。
 <br/>
 <br/>
 陈其迈17日则以市民的身份参加民间团体发起weCARE游行。陈其迈表示，在乎高雄，关心高雄，所有的朋友大家一起来加入11月17号weCARE，我们在乎高雄的大游行。
 <br/>
 <br/>
 18日，陈其迈将回防冈山，与谢长廷、陈菊共同造势。下周一(19日)，韩国瑜、陈其迈将再度同台，面对面进行辩论。
 <br/>
 <br/>
