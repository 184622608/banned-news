### Netflix与台湾合作 首部华语原创影集即将开拍
------------------------

<div class="wysiwyg">
 【新唐人2018年09月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际影音串流平台.html" target="_blank">
  国际影音串流平台
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Netflix.html" target="_blank">
  Netflix
 </a>
 ，在各国发展原创影集，现在触角也延伸华人影视市场，Netflix选择与台湾团队合作，推出首部华语的自制原创影集，由演员张孝全与贾静雯领衔主演。今天在台北，举办开拍记者会，Netflix亚太区原创总监，也亲自出席。
 <br/>
 <br/>
 主持人 黄子佼：〝中间空一个位置给张孝全有请，欢迎，哇，这个兄弟的阵仗太完美了，所有的女性观众都要垂涎三尺了，五位请看前方，欢迎欢迎。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Netflix.html" target="_blank">
  Netflix
 </a>
 与台湾合作首部华语自制原创影集，由台湾艺人张孝全与贾静雯领衔，与王柏杰、刘子千、周洺甫、许光汉、范晓萱，共同演出。29日，在台北，举办开拍记者会。
 <br/>
 <br/>
 主持人 黄子佼：〝所以你是黑道的女人，你是吗？(我是阿全的太太，可以透露嘛？)(你都说了)导演感觉越来越无奈了，她说完之后才问说，欸可不可以说？欸你的顺序有点怪欸。〞
 <br/>
 <br/>
 整部片由台湾才女导演陈映蓉执导并且编剧，是一部悬疑剧情的电视影集。为了融入情境，让影集更生动，剧组将剧本中的人物设定及生活背景等元素，做一番研究考察。
 <br/>
 <br/>
 导演 陈映蓉：〝田调了一下，我们就特别去剧本上写到的，该发生的环境，该发生的场合去交了一些朋友这样，那真的非常（每天都跟颜正国吃饭这样啊之类的这样）颜正国是前辈，(现在角头很厉害好不好)真的真的，致敬致敬。〞
 <br/>
 <br/>
 Netflix首部华语自制影集，全片都在台湾取景拍摄，Netflix亚太区原创总监 艾瑞卡诺斯也特别出席台北的记者会，强调平台对华语以及在地化的重视，原创影集市场竞争激烈，积极向串流媒体转型的HBO，继《冰与火之歌》后，近期也推出新创作，在新兴媒体的角逐中，就看谁能在创新时代中，抓住观众的眼球。
 <br/>
 <br/>
 新唐人亚太电视台吕复展、唐洁安(实习记者)台北采访报导
</div>
