### 排1.5小时投完票 柯文哲：程序太冗长
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 选举投票日加上好天气，许多民众纷纷出门投票，不过，因为今年公投绑大选，投票时间多半被公投票耽搁。
 <br/>
 <br/>
 中央社报导，台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 上午10时到北市图启明分馆投票，夫人陈佩琪陪同30分钟后就回家做家事，柯文哲笑说：〝我们家很务实，不会浪费时间。〞
 <br/>
 <br/>
 柯文哲足足排了1.5小时才投到票，直呼作业程序太冗长，应要E化。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴蕚洋.html" target="_blank">
  吴蕚洋
 </a>
 则是上午8时到和平东路三段的投票所投票，表示〝早起的鸟儿有虫吃〞，没想到现场已排满人。由于吴蕚洋在公视举办的辩论会上高唱〝爱江山更爱美人〞引发高知名度，现场许多民众称赞吴蕚洋有一副好歌喉。
 <br/>
 <br/>
 吴蕚洋笑说，他想唱什么歌都是神来一笔，今早投完票后，脑海里回响着周杰伦的〝星晴〞，他也不忘推广蜂蜜柠檬成效，表示希望所有人都健康快乐。
 <br/>
 <br/>
 无党籍台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李锡锟.html" target="_blank">
  李锡锟
 </a>
 上午10时到万华区投票所投票，等候40分钟才完成投票，李锡锟也呼吁民众踊跃投票，用手中一票体现民主价值。
 <br/>
 <br/>
 (责任编辑：卢勇信)
</div>
