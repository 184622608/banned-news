### 台湾选举吸引大陆民众：很民主 大陆看不到
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾九合一选举.html" target="_blank">
  台湾九合一选举
 </a>
 战火烧红宝岛，也吸引无数大陆民众〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_跨海观选.html" target="_blank">
  跨海观选
 </a>
 〞。有大陆民众称，因为在大陆看不到这种选举，所以特意来台湾感受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主.html" target="_blank">
  民主
 </a>
 的气氛。台湾选举有如政治嘉年华，〝这个选举真正是老百姓来作主〞，很民主。
 <br/>
 <br/>
 在台湾各地万头窜动的地方选战场子里，不乏中国大陆人的身影。他们都是自掏腰包到台湾自由行的〝观选人〞。
 <br/>
 <br/>
 自由亚洲11月23日报导，来自浙江的徐其红，曾经在浙江担任县级的人大代表，为了直击烟硝弥漫的九合一选举狂潮来到台湾。
 <br/>
 <br/>
 徐其红一路从台北追到高雄，哪里场子热，他就往哪里跑。在他眼中，台湾选举有如政治嘉年华。〝我觉得很有趣、很好看，所以我们就去看，看他总部怎样子的，造势怎么样子的，我们去看看，我们（中国大陆）那边看不到，所以我们到这边来看。〞
 <br/>
 <br/>
 徐其红选择坐出租车观看选举行程，台湾司机边开车、边关注选举新闻，两个人热烈讨论高雄市长候选情况。
 <br/>
 <br/>
 徐其红亲身见证了公民参与和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主.html" target="_blank">
  民主
 </a>
 价值，他说，〝这个选举真正是老百姓来作主，4年的时间你做得好你就做，做不好就下台。〞
 <br/>
 <br/>
 周曙光被外国媒体形容为〝中国公民记者第一人〞，身为台湾女婿的他定居花莲5年多，今年5月拿到了台湾身份证，成为首投族。
 <br/>
 <br/>
 周曙光说出第一次拥有投票权的心情：〝因为我在大陆是没有见过选票的，没有投票权。所以我很珍惜这次投票的机会，我做了不少的影像记录，方便我的朋友了解台湾的基层选举是如何进行的。〞
 <br/>
 <br/>
 周曙光说，中国大陆是不民主的，甚至在这几年不断施展自己的影响力，试图影响其他地方的民主。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾九合一选举.html" target="_blank">
  台湾九合一选举
 </a>
 也抓住很多在台湾求学的大陆学生的眼球。来自福建厦门的袁同学，在台北攻读数位设计领域，这是他第一次看到台湾选举。
 <br/>
 <br/>
 袁同学说：〝大街上有很多旗子，还有宣传车，公车上有很多广告，就觉得蛮新鲜的。民众的参与率也较高。从某种方面来说，也是一种民主的体现。〞
 <br/>
 <br/>
 肖同学是福建师范大学和台湾的交换生，台湾的选举文化让他大开眼界，肖同学这么形容：〝我认为是一种全民参与选举，被选举人之间，（有）一种竞争的现象，还蛮新奇的，因为没有见过。〞
 <br/>
 <br/>
 这一场选举在不同身份的大陆观选人眼里，主题就是〝民主参与〞。而这正是大陆所缺失的。
 <br/>
 <br/>
 中共自称领导人都是选举出来的，实际上中国民众根本就没有选举权，那些坐在北京大会堂的所谓〝人民代表〞，则被民间嘲讽是〝橡皮图章〞，只不过是按照中共最高层的意愿，举举手，做做样子而已。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：文慧)
