### 拚了！南投拚观光 5万张优惠序号上网抢
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月24日讯】整合产业拚观光，南投县政府10月起推出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南投感谢祭.html" target="_blank">
  南投感谢祭
 </a>
 〞，共结合200家厂商提供5万张优惠序号，开放民众上网索取，除享消费优惠及每月抽奖，还有汽车大奖。
 <br/>
 <br/>
 据中央社报导，南投县政府邀请县内观光体系旅宿业、日月潭游船业、景点游乐区、餐饮业、伴手礼观光工厂5大产业200家厂商参与，以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抢序号.html" target="_blank">
  抢序号
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_享优惠.html" target="_blank">
  享优惠
 </a>
 〞方式，透过网路供全国民众索取，第一阶段10月10日上午10时10分开放。
 <br/>
 <br/>
 县府观光处长王源钟今天说，县府带领观光旅游产业进行整合式行销，由5大产业限量释出5万张优惠序号，第一阶段10月10日上午开放索取，每一产业供应6000张，5大产业总计3万张，第二阶段抢序号则为12月12日开放2万张。优惠使用期限自10月10日到明年2月4日。
 <br/>
 <br/>
 王源钟表示，优惠序号须由消费者上网索取，除到各店家消费享优惠，每月还可抽小奖，并预定农历新年南投灯会期间抽汽车、旅馆住宿券等大奖，欢迎民众到〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南投好好玩.html" target="_blank">
  南投好好玩
 </a>
 〞网站抢优惠序号，系统预计10月1日上线，可先上网了解优惠内容。
 <br/>
 <br/>
 打造〝观光首都〞是南投县政府施政重点，王源钟表示，县长林明溱陆续实现观光政见，如瑞龙瀑布风景区、双龙瀑布水源吊桥改建、清境高空观景步道第二期等景观工程，未来规划则有南投县农产运销公司猪乐园旁4公顷苗圃，改建为休闲公园及儿童游乐园。
 <br/>
 <br/>
 另外，东埔蚋溪木屐寮滞洪生态区由经济部水利署完成整治，设有步道、生态景观、石雕，是民众亲近生态及休闲旅游好去处，王源钟说，县府未来规划导入临时购物市街、活动式亲水设施，透过举办活动的收入，作为园区维护经费，达到永续经营目标。
 <br/>
 <br/>
