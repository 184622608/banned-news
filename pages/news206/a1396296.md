### 董家嫁女搭上死亡列车 憾！17人中8死2重伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车21日下午4时50分在新马车站300公尺半径弯道弯道，发生翻车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 惨剧，至今共205人死伤，包括18人死亡、187人受伤。罹难者中有一家17人的董姓家族，有8人不幸丧生，6人受伤。
 <br/>
 <br/>
 综合媒体报导，董进兴家族一家17人从台东赴台北参加二女儿婚礼，21日吃完喜酒返程时，坐上6432次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宜兰县.html" target="_blank">
  宜兰县
 </a>
 消防局统计，18名罹难者中，董家有8人分别是董进兴、王绿云（董进兴妻）、董进发（董进兴弟）、董益良（董进兴孙）、董佳惠（董进兴孙）、董玉兰、何发仁（董玉兰夫）、何青宴（董进兴外甥女）。
 <br/>
 <br/>
 另有2人在加护病房观察，其他7人因座位在第3车厢，逃过一劫。
 <br/>
 <br/>
 苹果日报报导，董进兴是台东船长，是台东县渔业界活跃人物，其妻王绿云是成功渔会理事，过去为争取渔民权益，曾在媒体发声。未料吃完女儿喜酒后，搭上死亡列车，令喜事变成丧事。
