### 竞选唱〝夜袭〞惹议 国防部长：凝聚向心力 予以尊重
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】中国国民党籍高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 在竞选活动时唱军歌〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_夜袭.html" target="_blank">
  夜袭
 </a>
 〞引发议论，如民间团体人本认为，此举把候选人当〝军头〞，散布仇恨和军国主义。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国防部长.html" target="_blank">
  国防部长
 </a>
 严德发今(14)日表示，世界各国的部队都有军歌的教唱，目的在凝聚向心力，且军歌都有当时的历史背景，应该不至于会有这样的解读。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 在选举造势场合高唱军歌〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_夜袭.html" target="_blank">
  夜袭
 </a>
 〞，先是立委郑运鹏认为唱军歌是〝军国主义〞、〝心理变态〞，还要唱歌的人去吃排泄物；接着人本教育基金会也发起抗议连署，并称歌词〝钻向敌人的心脏〞，有杀人犯罪的嫌疑，背后的意识型态，违背这块土地追求独立、自由、多元、融合的精神。
 <br/>
 <br/>
 严德发上午赴立法院外交及国防委员会备询时受访，有媒体问及对于军歌在造势场合传唱的看法。他认为，世界各国的部队都有军歌的教唱，主要的目的是凝聚向心力、激励士气，因此对于外界传唱任何的军歌，都予以尊重。
 <br/>
 <br/>
 至于是否担忧是军国主义的复辟及扭曲社会，严德发表示，大家都很清楚台湾是爱好自由、民主的国家，且军歌都有当时的历史背景，应该不至于会有这样的解读。
 <br/>
 <br/>
 《夜袭》怎么唱！
 <br/>
 韩国瑜在造势场合唱红军歌《夜袭》，让Youtube上与〝夜袭〞相关影片的点阅率爆增，其中一则还出现超过200万人次的点阅，并有超过2千则的留言。
 <br/>
 <br/>
 网友说，就是冲着韩国瑜来的，旗山那场造势让他们想听听夜袭到底怎么唱，〝听韩市长唱过来听的+1〞、〝跟着秃子来的+1〞、〝刚在旗山晚会听韩总唱这歌 特地来听的〞、〝退伍后从没想过会去唱这一首歌...没想到旗山夜袭战用这首歌当作主题曲竟然一点也没有违和感。〞
 <br/>
 <br/>
 据了解，〝夜袭〞这首军歌最早的歌词是〝钻向共匪的心脏〞，在80年结束动员戡乱时期后，才把歌词改成〝钻向敌人的心脏〞，2014年还被总政战局列为重点推广军歌之一，连歌手陈升都曾在1994年的专辑里，翻唱收录了这首军歌。
 <br/>
 <br/>
 前总统马英九日前赴万华为台北市议员郭昭岩站台造势时，被问到有关韩国瑜造势场合唱军歌〝夜袭〞的看法，他说，夜袭是很多当过兵的人、一般人觉得最好听的军歌，不但歌词雄壮威武很有激励人心的作用，而且曲调也很美，自己离开战队40年还会唱。
 <br/>
 <br/>
 他说， 韩国瑜的〝韩流〞效应，〝方兴未艾〞一切都还在发酵中，因为韩国瑜描述高雄的话打到很多人心坎里，得到他们共鸣，才会这么受欢迎，而这也反映民众对大情势的看法，很多地方民怨沸腾、民心思变，在这样的环境下只要擦一根火柴就可以点起来。
