### 女路边等过马路 遭恍神男冲撞伤重不治(视频慎入)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】新竹县一名黄姓男子10日下午驾车行经新竹县
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竹北.html" target="_blank">
  竹北
 </a>
 地区时，疑精神不济、打瞌睡，撞到站在路边等待过马路的郭姓女子，导致郭女伤重不治。新竹检方今(12)日相验遗体，厘清案情。
 <br/>
 <br/>
 警方调查，黄姓男子10日下午1时许，开车行经
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竹北.html" target="_blank">
  竹北
 </a>
 市自强南路与文兴路路口时，先撞上停在待转区的机车骑士后，车辆再
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冲撞.html" target="_blank">
  冲撞
 </a>
 正在路边等过马路的郭姓女子，将郭女撞上夹在路边轿车，再波及3辆轿车，一旁民众见状，赶紧协助报警处理。
 <br/>
 <br/>
 警消赶抵现场，救出伤势严重的郭女，郭女一度失去意识，送往竹北东元医院急救后恢复生命迹象，转送林口长庚医院急救后，昨天仍因伤重宣告不治。
 <br/>
 <br/>
 警方调查，黄男酒测值为0毫克，侦讯时坦承因工作太累，一时恍神酿祸，警方讯后将黄男依过失致死罪嫌函送法办，警方报请检察官于今天相验郭女遗体，持续厘清案情。
