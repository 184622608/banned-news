### 少子女化浪潮 全台大专校院172个系所停招
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】台湾因应
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_少子女化浪.html" target="_blank">
  少子女化浪
 </a>
 潮和产业发展趋势，进行系科重整，2019学年度全台大专校院共有172个系所
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停招.html" target="_blank">
  停招
 </a>
 ，其中以硕博班及在职专班停招较多，学士班停招较少。
 <br/>
 <br/>
 据教育部统计，2019学年度一般大学共有96个系所
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停招.html" target="_blank">
  停招
 </a>
 ，技职校院则有76个系所，不少是进修部、在职专班。
 <br/>
 <br/>
 其中国立交通大学，电子工程系和电机资讯学士班也在停招之列。校方日前解释，目的是要整合为〝大电机系〞，因应跨领域学习的需求，让学院成为课程设计的主体，全盘规划和统筹课程。
 <br/>
 <br/>
 另据自由时报报导，创校50多年的宜兰县私立兰阳技术学院，设有7系，因学生数锐减，决议2019学年起停招6系，仅保留餐旅管理系继续招生。
 <br/>
 <br/>
 兰阳技术学院发出声明稿解释，为因应
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_少子女化浪.html" target="_blank">
  少子女化浪
 </a>
 潮和永续经营，108学年进行系科重整，减少招生系科以〝养精蓄锐〞，集中资源，以职场导向为中心，转型为小而美的技专校院。
 <br/>
 <br/>
 教育部技职司长杨玉惠5日表示，在学生减少的情况下，兰阳技术学院无法每个系都维持正常运作，才会有所调整。因宜兰县重视发展观光，因此兰阳技术学院也希望以观光产业相关系所为重点发展，逐步调整系科。
 <br/>
 <br/>
 (责任编辑：夏明义)
 <br/>
