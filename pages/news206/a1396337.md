### 中共资金干预选举 台湾调查局：迅速侦办
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】年底九合一大选，传出中资介入资助特定候选人，法务部调查局长在立法院证实，中共金援国内特定的候选人，调查局掌握情资有33件，已经移送4件给检方，朝野立委要求尽速查办。
 <br/>
 <br/>
 法务部调查局副局长 黄迪熹：〝已经有四件，移送到台北、云林、彰化、还有屏东(地检署)，有四件是用贿选的型态移送。〞
 <br/>
 <br/>
 台湾调查局证实，中共直接、间接金援，干预台湾九合一大选，已掌握33件情资，部分已分案侦办，还有四件移送地检署。
 <br/>
 <br/>
 陆委会主委 陈明通：〝境外势力提供政治献金，给我们候选人，这按照政治献金法，是不允许的。〞
 <br/>
 <br/>
 法务部调查局长 吕文忠 VS. 民进党籍立委 刘世芳：〝主要(中共)是以招待桩脚，到大陆旅游占大宗，其它也有透过金钱来支援，(来交付)，对。这一部分(资金境外交付)当然，在境外的查察，它有一定的困难度，但是我们调查局一定会运用各种方式。如果这个，我们的调查结果，会影响到他的选举结果，当然，(可能取消他的当选资格)，对啊，有可能。〞
 <br/>
 <br/>
 调查局长吕文忠日前(21日)表示，候选人不得接受大陆人民或团体的政治献金，中共就以其它方式金援，给钱的模式很多样，像中共国台办，就会透过台商送钱给特定候选人。
 <br/>
 <br/>
 法务部调查局长 吕文忠 VS. 国民党籍立委 曾铭宗：〝如果是纯台商，是可以做政治献金的，那这部分我们不会去做任何调查。（所以很清楚，这33个案件里面，都是涉及到中国大陆政府机关机构，或者(大陆)公司的问题，没有纯台商的问题。）可以这么说。〞
 <br/>
 <br/>
 朝野立委都催促侦办，调查局表示，掌握非常落实，但需要时间搜集证据。
 <br/>
 <br/>
 法务部调查局长 吕文忠 VS. 国民党籍立委 黄昭顺：〝(限期办出来，因为距离选举只有三十天。)〞
 <br/>
 <br/>
 调查局副局长 黄迪熹 VS. 民进党籍立委 张宏陆：〝（两年前就很多罗。）是。（看美国政府的态度，跟我们台湾你们现在办的态度，(台政府)太不积极了。）〞
 <br/>
 <br/>
 对于公股银行强化洗钱防制，多个部会官员也澄清，并非针对中国大陆或台商，而是基于风险管理，犯罪所得流出及流入的前三大地区，都是大陆、香港、澳门。
 <br/>
 <br/>
 新唐人亚太电视陈进交、张东旭台湾台北报导
