### 小乾坤品德列车到苗栗后龙 开启学童良善心
------------------------

<div class="wysiwyg">
 【新唐人2018年11月10日讯】新唐人〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小乾坤品德列车.html" target="_blank">
  小乾坤品德列车
 </a>
 〞，周四(8日)受邀，开往台湾苗栗后龙的新港国中小，透过活泼的动画以及游戏方式，让小朋友们了解生活常规和品德的重要。
 <br/>
 <br/>
 卡通动画《天庭小子─小乾坤》现身苗栗后龙新港国中小，小朋友们与小乾坤开心击掌，握手拍照。
 <br/>
 <br/>
 新港国小小朋友:〝小乾坤就是很可爱，也很厉害，可以杀死蛀牙虫。〞
 <br/>
 <br/>
 新港国小老师 李春芳：〝小朋友会互相的提问，就是藉着联想到我们在日常生活中，如果碰到了这样的问题，我们怎么去处理他。那孩子的反应就是非常的好，他们看到这个就很可爱，每次就说再看一次！再看一次！〞
 <br/>
 <br/>
 从事教职二十多年的李春芳老师，利用晨读时间，让班上学生观赏小乾坤影集，并且结合日常生活发生的事件，与学生讨论，让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_品德教育.html" target="_blank">
  品德教育
 </a>
 更活泼，不再只是死板说教。这一次，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小乾坤品德列车.html" target="_blank">
  小乾坤品德列车
 </a>
 能够开进新港国中小，是因后龙镇公所及万善祠所促成，希望下一代的品德教育，能够做的更好。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苗栗县.html" target="_blank">
  苗栗县
 </a>
 后龙镇万善祠百姓公管委会主委 李长峰：〝你要去和学校结合，我是以抛砖引玉的方式，来引导这些学校办。〞
 <br/>
 <br/>
 新港国中小校长 吴云道：〝以现在社会趋势越来越复杂多元的情形下，那我们学校能加强品德教育，如果是用活生生的教条式的方式，我想效果不见得会有很大的效果，那以这种方式去进行，我相信学生会慢慢被他吸引，那会有内化的作用。〞
 <br/>
 <br/>
 苗栗县后龙镇长 朱秋隆：〝我们都希望在小朋友在这个学业最基础的，学习的一个初期，能够让他自己建立一个新的而且正确的观念，首先一定要把品德这个部分，能够让他有一个很正常，很正确的一个学习环境。〞
 <br/>
 <br/>
 以趣味的生活戏剧故事，让孩子从娱乐中学习品德，〝小乾坤品德列车〞落实品德教育生活化，带给孩子更多的快乐与启发。
 <br/>
 <br/>
 新唐人亚太电视陈彦如、江淳卉台湾苗栗采访报导
</div>
