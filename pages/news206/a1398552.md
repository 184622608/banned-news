### 蔡英文主持双舰成军：勿忘守护台湾使命
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】来自台湾的消息。总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 今天上午赴海军
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_左营基地.html" target="_blank">
  左营基地
 </a>
 ，主持派里级巡防舰成军典礼，蔡英文演说时提醒国军〝三不〞，并说，中共对台的〝三战〞日益显著。来看稍早的最新谈话与画面。
 <br/>
 <br/>
 中华民国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 ：〝我也有‘三不’要期许国军，不要忘记守护台湾的使命、不要放弃追求区域和平稳定，以及不要忘记来自境外的威胁。〞
 <br/>
 <br/>
 〝是中共对台的‘三战’，日益显著。透过新的科技、网路及新媒体，数以万计的假讯息正渗入台湾的社会，持续性的网路攻击，不断地打击我们的政府和产业。〞
 <br/>
 <br/>
 蔡英文总统提出中共对台新三战，期许国军三不，稍早蔡总统也亲自登舰视导，

  铭传
 </a>
 号与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逢甲.html" target="_blank">
  逢甲
 </a>
 号，这两艘派里级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军舰.html" target="_blank">
  军舰
 </a>
 在美国海军服役时，是12艘拥有最新、最完整现代化升级的派里级舰之一，在台湾成军后，将成为海军146舰队战斗序列成员，加入海峡第一线防务任务。
