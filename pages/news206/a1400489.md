### 拉票催票 侯友宜奔三芦绿票仓扫街
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】选战倒数48小时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侯友宜.html" target="_blank">
  侯友宜
 </a>
 抢攻绿营票仓三芦地区，一大早就到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芦洲|扫街.html" target="_blank">
  芦洲|扫街
 </a>
 菜市场徒步扫街拜票，被问到最近天气不好，侯友宜说请大家帮忙拉票、催票，不要像4年前那样差点输掉，而对于选前之夜没有请党内大咖站台，侯友宜重申〝人民最大咖〞。
 <br/>
 <br/>
 摊贩 VS. 国民党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新北市长.html" target="_blank">
  新北市长
 </a>
 候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侯友宜.html" target="_blank">
  侯友宜
 </a>
 ：〝侯友宜，当选，友宜友宜友宜，当选当选当选。〞
 <br/>
 <br/>
 一走进菜市场，就收到摊贩送上凤梨，侯友宜收下祝福，笑得好开心。
 <br/>
 <br/>
 摊贩 VS. 国民党新北市长候选人 侯友宜：〝不投给你，要投给谁，喔，这样喔，谢谢。〞
 <br/>
 <br/>
 继前一天在三重、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芦洲|扫街.html" target="_blank">
  芦洲|扫街
 </a>
 ，派报澄清土地争议，隔了一天又到芦洲扫街，抢攻绿营票仓。
 <br/>
 <br/>
 国民党新北市长候选人 侯友宜：〝因为这里有很多跟我成长一样的背景，菜市场南部来的，我们要记得，4年前大家没有出来投票，我们就差点输掉了，所以我要拜托各位选民，风雨无阻要站出来投票。〞
 <br/>
 <br/>
 不断跟选民拉票、催票，不过周五（23日）选前之夜，侯友宜不打算找党内大咖站台，只请来朱立伦，相较对手苏贞昌阵营，请来总统蔡英文出马，难道不怕人气输输人一截？
 <br/>
 <br/>
 国民党新北市长候选人 侯友宜：〝因为他们讲过嘛，新北赢他们就大赢了，他们一定要非拿下新北，他们才会大赢，市民们跟我全部在那边支持相挺，面对这些等于是高层，总统级的院长级的，侯友宜永远不害怕。〞
 <br/>
 <br/>
 强调有人民当后盾，没在怕，而对于选举一路以来的风风雨雨、相互攻讦，侯友宜难得感性。
 <br/>
 <br/>
 国民党新北市长候选人 侯友宜：〝我的太太一直提醒我一件事，他说你面对的对手是老县长，是你的老长官，对手任何的抹黑奥步，无情的打击，你要记得，他曾经跟你共事过，我们要理解包容。〞
 <br/>
 <br/>
 把握选前最后时间，侯友宜一步一脚印，希望催出更多选票。
 <br/>
