### 康芮减弱为中颱 如曲棍球般北上影响韩日
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月03日讯】今年第25号
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康芮.html" target="_blank">
  康芮
 </a>
 强度减弱为中颱，3日集结在菲律宾以东海域，向西北海面移动，预料将影响冲绳，并来个转弯。韩国气象厅预测于7日上午影响
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_济州岛.html" target="_blank">
  济州岛
 </a>
 。
 <br/>
 <br/>
 下午2时，中度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康芮.html" target="_blank">
  康芮
 </a>
 以每小时16公里速度，向西北进行，近中心最大风速每秒45公尺，瞬间之最大阵风每秒55公尺，七级风半径250公里，十级风半径80公里。
 <br/>
 <br/>
 台湾中央气象局预报员林定宜表示，由于康芮颱风(KONG-REY)将经过海水温度较低的区域，因此在上午8时减弱为中度颱风，加上路径略往东偏，因此台湾发布海上颱风警报的机率再降低。
 <br/>
 <br/>
 不过，他说，尽管康芮颱风不会直接影响台湾，但明(4)日一直到5日上半天，北部、东北部将有大雨或豪雨，提醒民众留意。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/10-3/p9063072a414811621.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-3/p9063072a414811621-ss.jpg"/>
   <br/>
   <font size="-1">
    今年第25号颱风康芮影像。(中央气象局截图)
   </font>
  </a>
 </center>
 <br/>
 日本气象厅预测，虽然康芮慢慢减弱，但周四仍会以非常强的风势影响冲绳，为当地带来狂风暴雨。预测之后转向东北面，并继续进逼九州以至北日本。
 <br/>
 <br/>
 而韩国气象厅则预测，康芮将于7日(周日)上午，到达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_济州岛.html" target="_blank">
  济州岛
 </a>
 西归浦南方100公里海上处，将对济州岛近海以至南部地区带来影响，但经哪条路线登陆南韩仍有待进一步观察。
 <br/>
 <br/>
 气象达人彭启明表示，颱风康芮接近台湾的位置比上个颱风
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潭美.html" target="_blank">
  潭美
 </a>
 略为偏西，到达宫古岛附近后会逐渐北转，在北上的过程中，仍会有些飘移和变动。预计从台湾东部外海通过后，再从中国大陆东部海面北上往韩国前进，然后再转向东北朝日本而去，准备前往日、韩的民众要注意交通航班。
 <br/>
 <br/>
