### 金钟奖落谁家今晚揭晓 黄子佼与侯佩岑搭档主持
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】第53届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电视金钟奖.html" target="_blank">
  电视金钟奖
 </a>
 即将在今晚（10月6日）隆重登场。今年的主持人由妙语如珠的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄子佼.html" target="_blank">
  黄子佼
 </a>
 与已婚人妻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侯佩岑.html" target="_blank">
  侯佩岑
 </a>
 搭档，众星齐聚庆祝电视人的年度盛会。除了各界瞩目奖落谁家外，典礼还有精彩表演桥段，
 <br/>
 <br/>
 中央社报导，第53届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电视金钟奖.html" target="_blank">
  电视金钟奖
 </a>
 共有182件作品入围，角逐38个奖项。周六下午5时在星光大道率先登场，晚间7时在国父纪念馆举行颁奖典礼。出席女星包括谢盈萱、钟瑶、谢沛恩等，男星则有何润东、卢广仲、是元介、李国毅、蔡振南等人。
 <br/>
 <br/>
 除了台剧入围表现受瞩，今年综艺节目主持人奖也竞争激烈，包括综艺大哥大张菲和胡瓜师徒同时入围，还有主持经历丰富的许效舜、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄子佼.html" target="_blank">
  黄子佼
 </a>
 、马世芳与陶传正共同拚奖；此外，益智及实境节目主持人奖项也颇有看头，吴宗宪与KID组合对上徐乃麟与曾国城、胡瓜与玛莉亚（叶欣眉），和孙协志、王仁甫、许孟哲等人。
 <br/>
 <br/>
 至于颁奖嘉宾焦点，被封〝最美周芷若〞的52岁周海媚第2度参加金钟颁奖典礼，将搭档张轩睿合颁〝戏剧节目新进演员奖〞，相差27岁的颁奖组合，同台互动备受瞩目。韩星金来沅被安排搭档演员天心，颁发〝戏剧节目编剧奖〞及〝戏剧节目女主角奖〞，其他还有演员温贞菱、庄凯勋、柯淑勤及柯佳嬿等人皆担任颁奖人。
 <br/>
 <br/>
 典礼表演桥段分别有3段演出，开场〝电视美食 让创意无限〞，找来团体〝浩角翔起〞携手Janet与康妮媚演出，歌手阿达也将带来rap表演，要透过以美食为符号的电视作品，带动电视人的连结与共鸣。
 <br/>
 <br/>
 中场第二段表演节目〝原生IP 让戏剧跨界〞，找来乐团〝八三夭〞以摇滚风格重新诠释
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 经典戏剧主题曲；第三段〝多元剧种 让世界看见〞，邀请创作歌手韦礼安演唱入围今年戏剧节目奖主题曲作品，其中也包括以客语、台语、国语呈现入围作品的多元风貌。
 <br/>
 <br/>
 〝2018度电视金钟奖颁奖典礼〞晚间7时在国父纪念馆举行，可透过三立都会台、公视、三立综合台（MOD）、三立戏剧台（MOD）收看直播，海外观众可透过三立国际台、新加坡STAR HUB及北美中天台同步收看；新媒体方面，可透过全球华人影音平台Vidol、三立新闻网、YouTube同步收看；此外，手机用户可下载金钟53 APP，体验多视角、多场景等典礼直播技术。
 <br/>
 <br/>
 金钟奖是台湾传播媒体的官方荣誉奖项之一，与金马奖、金曲奖并列为三大娱乐奖项，称为〝三金〞。金钟奖更成为台湾演艺人员追求的荣誉目标，首届创始于1965年，第1届至第46届由行政院新闻局主办，第47届之后由文化部影视及流行音乐产业局接续主办。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
