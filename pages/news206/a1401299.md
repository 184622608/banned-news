### 国际寻失联李明哲 民团吁柯文哲向中共要人
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】台湾NGO工作者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李明哲.html" target="_blank">
  李明哲
 </a>
 ，被中共非法判刑一周年，却处于监禁失联状态。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际人权组织.html" target="_blank">
  国际人权组织
 </a>
 11月陆续发起紧急营救活动，台湾民间团体周三(28日)举行联合记者会，要求台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ，在双城论坛，当面向中共官员要人。
 <br/>
 <br/>
 国内外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权团体.html" target="_blank">
  人权团体
 </a>
 、艺文界联合记者会，呼吁台湾政府向国际发声，也要求台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ，在12月双城论坛，向中共要人。
 <br/>
 <br/>
 台湾人权促进会 邱伊翎：〝柯文哲市长，藉由这样一个可以跟中国高层的官方代表，见面的一个机会，也应该要去为台湾公民、以及人权工作者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李明哲.html" target="_blank">
  李明哲
 </a>
 先生说话。〞
 <br/>
 <br/>
 李明哲因为在台湾使用脸书、微信发表民主言论，在大陆被中共强制失踪至今619天，以〝颠覆国家政权〞罪名非法判刑五年，至今一周年。然而他又一次陷入失联状态。
 <br/>
 <br/>
 国际特赦组织副理事长 刘璐娜：〝他的健康状况，跟在监狱所受到的待遇，也都令人非常忧心。而这一切，也都违反了国际囚犯待遇的标准。〞
 <br/>
 <br/>
 外界担心，李明哲会不会遭遇酷刑虐待，家属两次要求探监被拒绝。国际人权联盟、国际特赦组织，陆续发起紧急救援，要求无条件释放；联合国工作小组、各国大使馆也关切。
 <br/>
 <br/>
 时代力量籍立委 林昶佐：〝像现在李明哲，这样子被不明原因移监，然后又没办法探视的案例，也是很多。例如像是法轮功学员王占青，之前他的两位七十多岁的父母，其实为了要探望他，也是到、一直被通知，一直换监狱，换到最后，到处跑，跑到最后，到了现场还是不能见。〞
 <br/>
 <br/>
 导演李惠仁说，拍摄中国环境社会问题的名摄影师卢广，11月初在新疆失联，这显示，经济、艺术与政治，是分不开的；台湾对两岸人权，不能冷漠。
 <br/>
 <br/>
 纪录片《并：控制》导演 李惠仁：〝冷漠，让台湾努力三十年，最重要的价值，民主自由、人权和法治，你(冷漠)会让它变成泡沫。〞
 <br/>
 <br/>
 民进党籍立委 尤美女：〝我们会联合国际的这些所有朋友，继续发声，我们绝对不会沉默，直到李明哲被释放出来的这一天。〞
 <br/>
 <br/>
 李明哲救援大队，将发起活动，鼓励台湾人每天写一封信给李明哲，由写信人在脸书发布，标注〝寻找李明哲〞。
 <br/>
 <br/>
 <br/>
 新唐人亚太电视陈进交、林嘉韦、张东旭台湾台北报导
</div>
