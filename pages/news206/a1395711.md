### 钓客离奇横尸堤岸 桃园警方怀疑凶手是牠
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】桃园市大园工业区海边堤岸16日发生一起离奇死亡事件。当天下午1时20分许，一名吕姓男子仰躺地上死亡，手臂上有2个孔洞，而一旁还有一尾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魟鱼.html" target="_blank">
  魟鱼
 </a>
 。警方怀疑可能是魟鱼酿灾，详细死因仍待报验。
 <br/>
 <br/>
 桃园市警方16日获报，有民众在路过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大园工业区海滨.html" target="_blank">
  大园工业区海滨
 </a>
 时，发现吕男正仰躺海边堤岸旁地上一动也不动，经过一段时间后，民众好奇的上前呼叫，才发现男子已经全身僵硬死亡。
 <br/>
 <br/>
 另有现场民众说，当天上午9时许就已看到死者躺在地上。
 <br/>
 <br/>
 警方初步观察发现，吕男的手臂上有2个疑似遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魟鱼.html" target="_blank">
  魟鱼
 </a>
 尾刺所刺伤的孔洞，而一旁还有钓竿及魟鱼一尾，魟鱼尾刺上还沾有血迹，怀疑男子是在收线时，正准备卸下鱼钩，不幸被魟鱼毒刺刺伤休克昏倒，身边没有人而一时延误就医，引发神经毒送命。
 <br/>
 <br/>
 目前已通知家属前往制作笔录。警方也表示，〝从未碰过这种意外！〞，对这样的案件相当罕见。
 <br/>
 <br/>
 魟鱼又称〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魔鬼鱼.html" target="_blank">
  魔鬼鱼
 </a>
 〞，尾部2处有刺，相当锋利，并且带有神经
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒液.html" target="_blank">
  毒液
 </a>
 ，有些毒刺可能长到超过37公分，受到威胁时，为了自卫会主动攻击，遭其尾刺攻击会引发烧灼感和阵痛。
 <br/>
 <br/>
 渔民间流传〝一魟、二虎、三臭肚〞的俗谚，形容魟鱼毒性之强，一向被视其为刺毒鱼类之首。有经验的钓客捕获到魟鱼时，怕被刺伤，都会优先将尾鳍切除，令其不会被攻击。
 <br/>
 <br/>
 台湾近海四季盛产魟鱼，过去发生过不少遭魟鱼刺伤的意外，但致死率很低。这次不幸发生命案，很可能是因为吕男遭直接刺伤主动脉造成，又延误就医所致。
 <br/>
 <br/>
