### 新人辈出 候选人：我们是国民党白色力量
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】九合一年底大选，也是政治新人的表现机会，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 在这次年底大选中，有68位议员候选人，是由40岁以下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青年.html" target="_blank">
  青年
 </a>
 人出线参选，今天，他们召开记者会，表示自己是国民党里的白色力量。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青年.html" target="_blank">
  青年
 </a>
 部主任 翁绍辉：〝现在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_年轻人.html" target="_blank">
  年轻人
 </a>
 心里想的，早就已经不是政党的色彩，而是可以象征改革力量的白色力量。我会希望说我们既然从国民党出发，我们一样穿着蓝色的衣服，然后可以带给年轻人更幸福，那我觉得说这也是一种白色力量，我们在国民党里面的白色力量。
 <br/>
 <br/>
 希望大家在这一次的选举里面，可以放下自己过去，对于政治政党的立场，的想法，的刻板印象，可以重新的愿意的，去接受我们，重新的认识我们给我们一次机会，理性的评判我们的候选人。〞
 <br/>
 <br/>
 那么记者会上，有来自全台各地的10位议员候选人露面，除了提出政见，信心喊话，也呼吁年轻人返乡
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 。
