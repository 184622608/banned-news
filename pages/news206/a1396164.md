### 台南美术馆试营运 重现古迹原始风貌
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】来关心台湾的新闻。让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台南.html" target="_blank">
  台南
 </a>
 人期待了一甲子的台南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美术馆.html" target="_blank">
  美术馆
 </a>
 ，10月17日开始试营运，馆内成立美术科学研究中心，是全台唯一有科学研究、修复画作的美术馆，这座美术馆是日治时期建筑，经过施工团队考据历史，将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_古迹.html" target="_blank">
  古迹
 </a>
 栋复旧还原，重现往日风华。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台南.html" target="_blank">
  台南
 </a>
 市美术一馆正式试营运，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美术馆.html" target="_blank">
  美术馆
 </a>
 前身为台南警察官署是日治时期建筑，外观非常雅致美丽。
 <br/>
 <br/>
 台南市代理市长 李孟谚：〝美术界期待达60年的美术馆，属于我们台南市自己的美术馆，第一馆终于在今天顺利开馆，里面的馆藏也非常丰富，有很多我们台湾前辈的画家。〞
 <br/>
 <br/>
 台南美术馆建筑师 石昭永：〝它的中间有一棵很大的榕树非常漂亮，所以我们不论是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_古迹.html" target="_blank">
  古迹
 </a>
 的维修新馆的设立，我们都希望能够绕这个榕树。〞
 <br/>
 <br/>
 南美馆以在地艺术家作品为优先典藏，发展在地艺术家资料库，目前约有1,534件各方捐赠典藏品入库，并且规画市民画廊，透过征件提供在地艺术家、创作团体申请展出。
 <br/>
 <br/>
 台南市美术学会荣誉理事长 陈泰元：〝我们台南幸福啦，我们艺术家共同的心声，把它发挥出来。〞
 <br/>
 <br/>
 台南长荣大学创办人 黄仁村：〝这是台南的资产，有过去的东西也有现在的东西，以后艺术家会很多。〞
 <br/>
 <br/>
 台南市美术馆馆长 潘襎：〝将会连结台湾四百年脉络，同时也将一盏大窗开向世界，将台湾艺术成就推向世界。〞
 <br/>
 <br/>
 南美馆首要建构台湾400年美术史，设有美术科学研究中心，并且获得中央前瞻计画补助经费，着重技术修复，未来这座场馆将成为拓展台南人文艺术的重要基地。
 <br/>
 <br/>
 新唐人亚太电视史进旺台湾台南报导
