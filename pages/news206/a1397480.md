### 普悠玛司机员痛哭跪拜 家属：不怪你 一起找真相
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】台铁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 出轨意外，造成18人死亡、200多人受伤，其中15名罹难者来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台东.html" target="_blank">
  台东
 </a>
 。台东县府上午举行罹难者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合公祭.html" target="_blank">
  联合公祭
 </a>
 。普悠玛司机尤振仲，也在家人陪同下低调现身，并向死者上香跪拜致意，他情绪激动痛哭，但家属未予责难，反安慰他要保重身体，一起找出事故真相。
 <br/>
 <br/>
 罹难者家属代表 董家小妹：〝这次的事故，让我们家族在同一时间，失去了8位最爱的至亲。多希望那只是一场梦，醒来就会回到原点。董家感谢所有对我们伸出援手，付出关爱的团队，谢谢你们。〞
 <br/>
 <br/>
 今天上午10点，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台东.html" target="_blank">
  台东
 </a>
 举办
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合公祭.html" target="_blank">
  联合公祭
 </a>
 ，行政院长赖清德、立法院长苏嘉全都到场致祭。而刚出院的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 司机员，也在儿子搀扶下，现身公祭现场吊唁。这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻车.html" target="_blank">
  翻车
 </a>
 意外，造成台东不少民众罹难与伤亡，包括董姓一家8口，以及卑南国中的师生，台东县府今天特地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降半旗.html" target="_blank">
  降半旗
 </a>
 ，向罹难者致意。
