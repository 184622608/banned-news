### 办公椅飙速大赛登场 带动文昌家具街人潮
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】日本超夯的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_办公椅.html" target="_blank">
  办公椅
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竞速比赛.html" target="_blank">
  竞速比赛
 </a>
 延烧到了台湾，今年在台北市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文昌家具街.html" target="_blank">
  文昌家具街
 </a>
 是第二度举办，那么首度登场的是团体赛，采三个人一组，第一名可以拿到奖金两万元，而举办这场活动，也是要藉此带动文昌家俱街的买气。
 <br/>
 <br/>
 鸣笛一响，民众用力划动双脚，奋力向前冲刺。从日本红到台湾的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_办公椅.html" target="_blank">
  办公椅
 </a>
 竞速赛，台北市文昌街封街二度举办，戴上头盔护具，还得小心别撞上旁边的围栏。
 <br/>
 <br/>
 单趟150公尺，团体赛三人要来回接力，去年的冠军队伍更是原班人马，要拼第一名两万元奖金。
 <br/>
 <br/>
 参赛者：〝我们团队(名称)就叫做领两万。〞
 <br/>
 <br/>
 参赛者：〝喇叭声叭下去，就是肾上腺素就爆发了。〞
 <br/>
 <br/>
 北市府商业处二度携手北市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文昌家具街.html" target="_blank">
  文昌家具街
 </a>
 发展促进会，不只要让民众玩得开心，还希望能活络文昌街的买气。
 <br/>
 <br/>
 台北市文昌家具街发展促进会理事长 刘松雄：〝利用这个时间点创造更多的参赛者，让我们商圈能够更多的曝光，那也提高我们的一个知名度。〞
 <br/>
 <br/>
 发展将近有50年的文昌家具街，是台北市著名的家具商圈，不过近几年人潮不复以往，有的店家自动将价格下修，吸引客源；有的店家则是寻求转型，引进意大利原装进口家具。
 <br/>
 <br/>
 家具业者：〝现在家具订做太多了，而且网路太透明化，很多店家就会变成销价竞争，然后店家就慢慢越来越没有利润。我们把意大利原装的东西，当成台制的价格在卖，所以销售量都还满不错的。〞
 <br/>
 <br/>
 家具业者：〝会有销价竞争，就会差在材质，就是比的就是材质。〞
 <br/>
 <br/>
 同时业者也反映，因为附近的停车位太少，希望停车规划能更完善，带动更多的采购人潮。
 <br/>
 <br/>
 新唐人亚太电视黄亮戬、赵庭誉台湾台北采访报导
</div>
