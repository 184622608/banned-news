### 天气闷热秋意迟 云林山海特产收成受影响
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】来自台湾的消息。有些民众可能有感觉，明明节气过了立冬，白天
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天气.html" target="_blank">
  天气
 </a>
 却还是异常闷热，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_云林.html" target="_blank">
  云林
 </a>
 山线的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_古坑咖啡农.html" target="_blank">
  古坑咖啡农
 </a>
 ，和海线的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌鱼子业者.html" target="_blank">
  乌鱼子业者
 </a>
 ，叫苦连天，不只产量减少，连品质都受到很大的影响。
 <br/>
 <br/>
 时序来到11月中旬，中南部白天高温持续飙上30、31度，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_云林.html" target="_blank">
  云林
 </a>
 古坑咖啡产地的咖啡豆，已开始转红，但农民的脸上看不到笑容，因为产量粗估只剩去年的一半。
 <br/>
 <br/>
 咖啡农 游木临：〝减一半，因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天气.html" target="_blank">
  天气
 </a>
 关系，还有果虫的关系，种种问题，咖啡也有大小年，去年生的多，今年一定会少一点。〞
 <br/>
 <br/>
 返乡种了二十年阿拉比卡咖啡的游先生，本来很自豪于他的田间管理，现在是苦无对策。
 <br/>
 <br/>
 咖啡农 游木临：〝（去年）我大概收成十五吨，依我们古坑乡来说，我可能属一属二，虽然种的不多，但管理很好，今年因为天气的变化，病虫害很多，所以产量少很多，剩七吨而已。〞
 <br/>
 <br/>
 不只是山线的咖啡农叫苦，连海线的乌鱼子大王曾先生，也感到无奈。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌鱼子业者.html" target="_blank">
  乌鱼子业者
 </a>
 曾焕佾：〝因为天气一直凉不下来，所以现在乌鱼稍微有一点过熟的忧虑，天气比较闷热，所以说现在这几天产出来的乌鱼子，品质没有那么好一点。〞
 <br/>
 <br/>
 成排大大小小的乌鱼子，吹着微凉的海风可以，太阳太大也不行，要不断帮它们翻身，农渔民真的是看天吃饭，要天时地利人和，一年的辛苦努力才能收成。
 <br/>
 <br/>
 新唐人亚太电视廖俪芬台湾云林采访报导
</div>
