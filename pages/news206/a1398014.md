### 韩国瑜与爱心菜贩出席公益会 陈其迈批恶质操作
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】爱心菜贩
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈树菊.html" target="_blank">
  陈树菊
 </a>
 3日与国民党高雄市长参选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 见面，陈树菊受访肯定韩有亲和力但坦言不认识韩。民进党市长参选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 则批评用消费的方式是恶质操作。韩国瑜驳斥陈对他的批评说，是受邀出席慈善团体公益会帮助偏乡小孩，澄清没有消费陈树菊。
 <br/>
 <br/>
 台东爱心菜贩
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈树菊.html" target="_blank">
  陈树菊
 </a>
 ，3日下午和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 相见，但两人都说是第一次和对方见面。仅是出席一场慈善活动。陈树菊被记者问到要投谁，陈树菊说，不认识韩国瑜，她也不是高雄人，但说韩国瑜是很尽职的生意人，第一次见面觉得有亲和力。
 <br/>
 <br/>
 韩国瑜则说第一次见到陈树菊，他说，陈树菊卖菜几十年，源源不断的爱心感动全台湾人，正好他过去也在卖菜，他感佩陈树菊的精神，希望高雄市将来也充满爱，充满温暖。
 <br/>
 <br/>

 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 4日一早出席高雄市义勇消防总队举办的〝澄清湖义消健走活动〞时受访说，国民党在这场选举一直在操作，从牙医师全联会理事长谢尚廷到高雄市总工会理事长林明章，国民党都用消费的方式，还企图要误导民众，他认为那是恶质的选举，呼吁韩国瑜应该要学会高雄人的厚道，勿把见面或参加餐会做恶质的选举操作。
 <br/>
 <br/>
 韩国瑜上午到永安区永安宫参拜，被媒体追问陈其迈批评他消费陈树菊，韩国瑜表示是受邀出席慈善团体公益会帮助偏乡小孩，澄清没有消费陈树菊，驳斥陈其迈对他的批评。
 <br/>
 <br/>
 陈其迈3日晚上于造势活动宣布辞立委，韩国瑜表示尊重；但他也说，站在高雄发展的角度，陈其迈辞去立委将使高雄市籍的立委少一席，一旦未来高雄要全力拚经济就会少了一份力量。
 <br/>
 <br/>
 外界期待韩国瑜、陈其迈进行政见辩论，韩国瑜表示，参加辩论会的门是开的，他愿意也认为应该进行一场候选人辩论会，只是双方时间无法配合。
