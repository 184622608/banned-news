### 裕隆董事长严凯泰骤逝 壮志未酬遗孀接班
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裕隆.html" target="_blank">
  裕隆
 </a>
 董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_严凯泰.html" target="_blank">
  严凯泰
 </a>
 抗癌2年半仍不敌病魔，3日过世，享年54岁。裕隆表示，家人将遵照严凯泰生前遗愿，不发讣文、不办公祭、不设灵堂、恳辞花礼，敬辞来自各界的致意。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_严凯泰.html" target="_blank">
  严凯泰
 </a>
 在2016年检查出罹患食道癌，历经2年半的抗癌仍不敌病魔骤逝。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裕隆.html" target="_blank">
  裕隆
 </a>
 内部随即启动接班机制，由遗孀
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_严陈莉莲.html" target="_blank">
  严陈莉莲
 </a>
 接班，加上老臣集团副执行长陈国荣、裕隆总经理姚振祥、裕日车总经理李振成的辅佐，未来将持续集团总管理处的领导机制。
 <br/>
 <br/>
 严陈莉莲昨天晚间也以裕隆集团执行长名义发出给全体员工的一封信，强调严凯泰未竟的使命，她会带领大家一起完成，希望大家一起加油。
 <br/>
 <br/>
 裕隆3日晚间发声明指出，集团表示悲痛与不舍，并将尽速组成治丧小组，全力协助家属处理相关事宜。
 <br/>
 <br/>
 <h4>
  25岁接班 〝少主中兴〞
 </h4>
 <br/>
 严凯泰14岁赴美留学，24岁回到台湾就担任裕隆首席副总经理，25岁正式接班，30岁就在林信义的支持下，引进国产百万房车Cefiro，引发市场热销，外界也把此役视为严凯泰〝少主中兴〞的代表作。严凯泰更循父亲严庆龄〝国车国造〞的遗志，创立自有品牌纳智捷（Luxgen）。
 <br/>
 <br/>
 严凯泰对于纳智捷的重视，可从〝我是把生命跟纳智捷画上等号〞一句话可以看出。严凯泰强调，纳智捷是年轻的汽车品牌，〝是在跑马拉松，要花20至25年去走的路〞，对于外界看衰纳智捷的品牌路，严凯泰也坚持〝自己就是要走品牌这条路，也已准备好银弹与资源。〞
 <br/>
 <br/>
 包括纳智捷的品牌发展及新店裕隆城的整体规划开发，可说是严凯泰继〝Cefiro〞之后的代表作，但在两大计画起步之际，严凯泰却壮志未酬身先死，也把接棒的任务交给妻子严陈莉莲。
 <br/>

 <br/>
 对于裕隆集团董事长严凯泰不幸病逝，与严凯泰有10多年交情的新北市长朱立伦4日表示，心情相当悲痛，也呼吁大家默默怀念，并称他会遵守与严凯泰之间，彼此的承诺。
 <br/>
 <br/>
 裕隆集团董事长严凯泰3日逝世，享年54岁。的朱立伦，今天在主持市政会议后受访表示，他以悲痛的心情呼吁大家，在心中永远怀念严凯泰，也请大家尊重严凯泰的遗愿，心中默默怀念，祝福他一路好走。
 <br/>
 <br/>
 朱立伦与严凯泰两人交情不浅，朱立伦在2010年竞选新北市长时，宣传新书〝改变人生的执行力－就要做好．朱立伦〞演讲，两人在桃园中坜中原大学座谈。
 <br/>
 <br/>
 在那场座谈中，两人不时相互调侃、妙语如珠。有读者提问若两人角色互换，彼此最想做什么？朱立伦说，最想把严凯泰的所有跑车都开一遍，且不怕周刊跟拍。严凯泰接着说，如果要他选举，他第一个想法就是想跳楼。两人的逗趣回答，引起一片笑声。
 <br/>
 <br/>
 严凯泰是国内成功的企业家，做公益、对社会付出不遗余力。被问到眼中的严凯泰是什么样的人，朱立伦语气稍微哽咽说，他将遵守与严凯泰之间的承诺。至于承诺内容是什么，朱立伦并没有明说，只表示有机会时会向外界报告。
 <br/>
