### 美航母大黄蜂战斗机坠海  2飞行员获救
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】美军
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根号.html" target="_blank">
  里根号
 </a>
 航空母舰1架舰载机
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F/A-18.html" target="_blank">
  F/A-18
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大黄蜂.html" target="_blank">
  大黄蜂
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战斗机.html" target="_blank">
  战斗机
 </a>
 ，今（12日）中午坠入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冲绳.html" target="_blank">
  冲绳
 </a>
 外海，机上2名飞行员都顺利获救。
 <br/>
 <br/>
 综合媒体报导，日本第11管区海上保安本部与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冲绳.html" target="_blank">
  冲绳
 </a>
 县警表示，这架出事战机是在冲绳北大东岛西南方约300公里海域坠海。
 <br/>
 <br/>
 美国海军表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大黄蜂.html" target="_blank">
  大黄蜂
 </a>
 战机是在执行例行任务期间，因机械故障坠毁，两名飞行员安全获救，情况良好，军方正调查坠机原因。
 <br/>
 <br/>
 据悉，事件中没有民间船隻受到影响。
 <br/>
 <br/>
 中央社报导，这架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F/A-18.html" target="_blank">
  F/A-18
 </a>
 战机是美国航空母舰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根号.html" target="_blank">
  里根号
 </a>
 （USSRonald Reagan CVN-76）的舰载机，部署于山口县的岩国基地。
