### 美中贸易战加速产业板块碰撞 台湾密切关注
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】美国总统川普26日表示，美国关税制裁，将如期进入第三阶段，包括苹果产品也可能加征关税。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 和硕董事长童子贤认为，贸易战的影响，已经不是单一厂商或部分业者，而是全面性的版块碰撞。
 <br/>
 <br/>
 一年一度资讯月，VR实境体验，各式电脑产品、还有手机打出优惠价，不过美国可能启动第三波关税，针对2670亿美元中国进口商品加征10或25%关税，甚至点名iPhone，还有大陆生产笔电、手机，可能加征10%进口关税，引发业界震撼。
 <br/>
 <br/>
 和硕董事长 童子贤：〝两边的态势似乎，还没有缓和下来，同时我们也提醒，不是单一品牌，也不是单一产业，也不是单一制造商所面临的，这是全球共同面临的贸易版块的撞击的这个事情。因为这最后的2670亿，恐怕有相当高的比例是美国自己的品牌。〞
 <br/>
 <br/>
 瑞银估计，iPhone如果被课10%关税，苹果获利将折损15亿美元；税率如果是25%，影响将扩大到38亿美元。苹果股价27日开低走低，盘中跌幅超过2%，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 苹果概念股，28日则是涨跌互见。
 <br/>
 <br/>
 经济部次长 龚明鑫：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_产业链.html" target="_blank">
  产业链
 </a>
 的移动是势在必行，只是说这个时程跟规模，就要看美国跟中国之间，在这一次的G20里面，去谈的结果是怎么样。〞
 <br/>
 <br/>
 和硕、广达、英业达等苹果供应链，近期纷纷规划，将生产线迁移至中国大陆以外的区域，广达更斥资42.8亿在桃园龟山设新厂，除了回台投资，东南亚、全球布局都要因应。
 <br/>
 <br/>
 和硕董事长 童子贤：〝这一些动作呢，配合我们在龟山原来就有相当四五千人的一个厂区呢，我想在第一阶段还可以应付，前面的这个2500亿阶段还可以应付，但是后面2670亿的这个，我们要重新做思考，目前还没有办法回答。〞
 <br/>
 <br/>
 苹果供应链业者坦言，产线搬迁不是一两天的事情，G20高峰会上，美中之间交锋角力，也成为重要观察指标。
 <br/>
 <br/>
 新唐人亚太电视胡宗翰、沈唯同台湾台北报导
