### 重返台湾市场 希尔顿酒店进驻新板特区
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】睽违15年，国际连锁饭店品牌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希尔顿酒店.html" target="_blank">
  希尔顿酒店
 </a>
 ，携手昔日战友宏国集团，重返
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾市场.html" target="_blank">
  台湾市场
 </a>
 ，首座酒店选择在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新板特区.html" target="_blank">
  新板特区
 </a>
 落脚，周四正式开幕，这也是成为新北市推动新板特区以来，第三家品牌酒店进驻。
 <br/>
 <br/>
 睽违15年，国际连锁饭店品牌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希尔顿酒店.html" target="_blank">
  希尔顿酒店
 </a>
 携手昔日战友宏国集团，重返
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾市场.html" target="_blank">
  台湾市场
 </a>
 ，首座酒店选择
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新板特区.html" target="_blank">
  新板特区
 </a>
 落脚，成为新北市推动新板特区以来，第三家品牌酒店进驻。
 <br/>
 <br/>
 台北新板希尔顿酒店总经理 汤思嘉：〝新北市政府对这个新板特区有很棒的愿景，我们第一年目标，先看7成住房率。〞
 <br/>
 <br/>
 新北市市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱立伦.html" target="_blank">
  朱立伦
 </a>
 ：〝将来也可以带动更大的商机，对我们台湾观光产业，对新北有非常大的助益。〞
 <br/>
 <br/>
 希尔顿酒店锁定商旅、度假及会展，就是看准新板地区的五铁共构交通优势、MICE会展与商务发展潜力，目前已有板桥凯撒大饭店、台北趣淘漫旅再加上台北新板希尔顿酒店，累计有1137间客房。
 <br/>
 <br/>
 台北市方面，包括台北国泰万怡酒店、金普顿大安酒店和台北中山逸林酒店等国际品牌进驻，预计第4季陆续开幕。
 <br/>
 <br/>
 宏国关系事业副董事长 林鸿道：〝台湾是一个，非常具有吸引力的一个地方，那么台湾不管你从他的这一个，地理、人文，或者是整个这一个，整个社会的环境。整个观光产业，我在想我对这里的，这个发展是非常有信心。〞
 <br/>
 <br/>
 林鸿道表示，观光旅宿业景气是跟着整体景气起伏，整体来说，乐观看好台湾观光旅宿业发展，对台湾有信心。
 <br/>
 <br/>
 新唐人亚太电视陈辉模、李晶晶台湾新北市报导
