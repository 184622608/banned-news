### 中坜仁海宫推广《悠游字在》传承中华文化
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】为了培养
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小朋友.html" target="_blank">
  小朋友
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汉字.html" target="_blank">
  汉字
 </a>
 的兴趣，桃园
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中坜仁海宫.html" target="_blank">
  中坜仁海宫
 </a>
 与新唐人亚太电视台，共同举办〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悠游字在.html" target="_blank">
  悠游字在
 </a>
 半日学堂〞，现场有40组亲子报名参加，课程中《悠游字在》以动画，呈现汉字趣味奥妙的演变，加上有趣的文化故事，及趣味的闯关游戏，让小朋友以活泼的方式，从中学会做人做事的道理，进一步提升语文能力。
 <br/>
 <br/>
 活动一开始，先学习古人如何拱手敬礼，一下子大朋友
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小朋友.html" target="_blank">
  小朋友
 </a>
 都彬彬有礼了起来。周六上午，由桃园
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中坜仁海宫.html" target="_blank">
  中坜仁海宫
 </a>
 赞助的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悠游字在.html" target="_blank">
  悠游字在
 </a>
 半日学堂〞，从认读
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汉字.html" target="_blank">
  汉字
 </a>
 的重要性，汉字如何演变，再到静心调息练写毛笔，小朋友发现原来汉字是这么深奥有趣。
 <br/>
 <br/>
 小朋友 卓训亮:〝我今天学到了汉字要怎么写，还发现不但它是用物体的形状来写，而且还要照着笔顺写，写汉字的时候握笔的方式，跟握铅笔的方式都不一样。〞
 <br/>
 <br/>
 家长 黄小姐:〝我觉得这个活动很值得推荐，因为小朋友透过卡通影片的方式，来宣导来认识汉字，认识国字，因为对他们来说现在是国字，他们不知道汉字的意思，(小朋友)刚好在学习，这个活动对他来说的话，会诱发国字汉字中文的启发。〞
 <br/>
 <br/>
 接下来的闯关游戏需要团队合作，为了展现团结小朋友们展现自己的队呼，接着要在三分钟内完成接力闯关，小朋友们发挥想像，将甲骨文演变成现在的汉字，或将拆开来的汉字拼成完整的字，随着有趣的汉字游戏，小朋友渐入佳境，最后都很快的完成任务。
 <br/>
 <br/>
 中坜仁海宫董事长 李合监:〝深深感觉我们悠游汉字的学习营，加深我们小朋友学习的兴趣，而且给他深入的印象，小学生受到我们的简体字的影响，还有火星文的影响，所以这个汉字有需要我们来给小朋友来学习。〞
 <br/>
 <br/>
 中坜仁海宫副董事长 王介禧:〝我第一次参与这样的活动，给我的感觉非常的有启发性，有启蒙，学生对国字的造形艺术有了解，我相信学校的老师方面，他们也是第一次碰到类似这样的教学，有动画也有文字造形这样的教学，很活泼的来带动。〞
 <br/>
 <br/>
 王介禧表示，中坜仁海宫一向支持文化事业推广，非常感谢新唐人亚太电视台的引见，让仁海宫有机会向新街国小的孩子们，介绍老祖宗的智慧，学习正统汉字，传承
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中华文化.html" target="_blank">
  中华文化
 </a>
 。
 <br/>
 <br/>
 新唐人亚太电视胡淑霞、张年洁台湾桃园采访报导
</div>
