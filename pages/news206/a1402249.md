### 华为财长被捕 股市下挫 供应链影响受瞩
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】华为财务长孟晚舟在加拿大被捕，牵连超过2000家供应商，其中包括台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_供应链.html" target="_blank">
  供应链
 </a>
 。针对台积电的部分，经济部长沈荣津认为，影响不大。
 <br/>
 <br/>
 经济部长沈荣津：〝这一个部分来说，应该不会影响。台积电是一个国际的一个，一个相当很卓越的公司，这个应该不会影响。华为还是应该关心的是不是制裁权(令)的问题。〞
 <br/>
 <br/>
 市场消息指出，华为财务长罪名如果确立，美国对华为，可能会进行晶片禁售制裁，让处于休兵的美中贸易战再度开打。
 <br/>
 <br/>
 中国上海A股的通信、5G族群，周四(6日)是全面重挫，开盘开低走低，平均跌幅3.5%，可说受创严重。
 <br/>
 <br/>
 此外据了解，华为也因为跟台湾科技业者，是客户关系，晶圆代工台积电，封装测试业者，精测、京元电，股价纷纷走低。华为相关基地台设备供应商，联发科、瑞昱、立积等业者，也是大跌。后续影响仍有待观察。
 <br/>
 <br/>
 新唐人亚太电视胡宗翰、黄彦菱台湾台北报导
</div>
