### 中颱玉兔环流影响台湾 一张图看共伴效应范围
------------------------

 【新唐人北京时间2018年10月30日讯】中度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_玉兔.html" target="_blank">
  玉兔
 </a>
 预测将不会登陆台湾，但今(30日)晚上起，外围环流将影响台湾，周三到周五，影响最明显，一直到周六(11月3日)迎风面北部及东半部要留意
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共伴效应.html" target="_blank">
  共伴效应
 </a>
 造成的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大雨.html" target="_blank">
  大雨
 </a>
 或豪雨。
 <br/>
 <br/>
 天气风险公司指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_玉兔.html" target="_blank">
  玉兔
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风.html" target="_blank">
  颱风
 </a>
 将登陆通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾.html" target="_blank">
  菲律宾
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吕宋岛.html" target="_blank">
  吕宋岛
 </a>
 ，周三进入南海后逐渐北转，强度也会持续减弱，但外围东南风将与东北风发生〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共伴效应.html" target="_blank">
  共伴效应
 </a>
 〞。到时候，整个北部到东半部都有雨，降雨热区落在〝北海岸、东北角、宜兰到花莲一带〞，会有持续性明显雨势。
 <br/>
 <br/>
 中央气象局预报员林定宜也表示，随着玉兔颱风登陆菲律宾吕宋岛，东北季风及颱风外围环流影响下，30日下午过后北部及东半部降雨机率逐渐提升，开始有局部短暂阵雨及较
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大雨.html" target="_blank">
  大雨
 </a>
 势发生，其他地区仍是多云到晴为主。
 <br/>
 <br/>
 他表示，午后起至11月3日迎风面北部及东半部局部地区受共伴效应影响，将有大雨或豪雨发生。
 <br/>
 <br/>
 他说，玉兔颱风将登陆吕宋岛北部，结构会受到破坏，在南海重整北上后海温低加上垂直风切大，强度会持续减弱，接近台湾时不一定仍维持颱风强度，由于路径跟强度仍有不确定性，对台湾天气具体影响程度气象局将会持续注意。
