### 美中贸易战升温 台商回流全台大猎地
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月27日讯】美中贸易战升温，不少台商准备回流，也带动台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工业用地.html" target="_blank">
  工业用地
 </a>
 需求。外资商仲业者公布最新统计，前三季土地交易冲破千亿元大关、达1,192亿元，比过去三年全年数字都高，建商猎地动能强，工业用地交易金额也持稳。
 <br/>
 <br/>
 外资商仲业者公布最新统计，前三季土地交易冲破千亿元大关、达1,192亿元，比过去三年全年数字都高，建商猎地动能强，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工业用地.html" target="_blank">
  工业用地
 </a>
 交易金额也持稳。本季工业厂房以及厂办交易合计达44亿元，占大型交易总额的54%，业者分析，今年确实有科技大厂，有买进自用工业厂房、土地的需求。
 <br/>
 <br/>
 住商不动产企研室经理 徐佳馨：〝在这波台商的一个回流来说，确实是造成工业地需求的一个增加。本地的传产跟电子业者本身扩厂，和另外过去是有一批农地农用，开设小工厂的这些业者，可能必须要去做转型，这样一个需求，种种需求之下，造成一个工业用地缺口。〞
 <br/>
 <br/>
 今年像是大立光，持续在台中猎地，购入台中工业区2,074坪厂房；台达电则锁定北部区域，今年已投入总计30.7亿元，陆续购入桃园中坜工业区扩建厂房。外资商仲业者分析，美中贸易战升级，促使科技大厂回台觅地，会让已经饱和的工业厂房以及土地供给更加吃紧，尤其新北、桃园、新竹、台中等产业群聚、发展成熟的工业区。
 <br/>
 <br/>
 住商不动产企研室经理 徐佳馨：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台商回流.html" target="_blank">
  台商回流
 </a>
 来说的话，如果五缺问题没有解决，还是会集中在某些特定的工业区，当然会造成某些工业区土地的缺口，但是如果五缺问题能够顺利解决的话，认为在土地需求上面，并没有那么不乐观。〞
 <br/>
 <br/>
 先前经济部表示，的确有厂商提出北部土地较贵的问题，建议可以考虑在中南部设厂，但有业者反映部分地区地处偏远，无法建立产业群聚效应，就看政府如何妥善因应，确切解决企业问题。
