### 【禁闻】民主的精髓！台湾蓝绿版图逆转
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 24号落幕，民进党丢失多数县市首长席位，总统蔡英文因此辞去党主席一职，针对这个结果，各方反应各异。有一名中国人发文指出，政党轮替是民主的精髓。
 <br/>
 <br/>
 面对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 蓝绿版图大逆转，中共国台办发言人马晓光25号表示，选举结果反映了广大台湾民众希望继续分享
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_两岸关系.html" target="_blank">
  两岸关系
 </a>
 和平发展的所谓〝红利〞等。他强调，将继续坚持〝九二共识〞，坚决反对〝台独〞，并欢迎台湾更多县市参与两岸城市交流合作。
 <br/>
 <br/>
 对此，台湾陆委会回应，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾地方选举.html" target="_blank">
  台湾地方选举
 </a>
 及公民投票展现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾民主.html" target="_blank">
  台湾民主
 </a>
 政治的成熟发展，中共当局应尊重并正确认识，不应该片面政治解读，涉入台湾内政，这将误判未来台海情势发展。
 <br/>
 <br/>
 针对海峡两岸的不同反应，一位来自中国的自由撰稿人〝末夏〞在台湾《上报》发表评论文章说，这本来是一桩台湾社会的民主选举，放到大陆语境下，变成反台独的结果，这是被夸张的意淫，用大陆的政治正确解读台湾民众的选举。
 <br/>
 <br/>
 旅美时事评论员蓝述表示，中共擅于炒作民族情绪，引导中国百姓热爱它，它不会错过台湾选举这个时机。
 <br/>
 <br/>
 旅美时事评论员蓝述：〝中共炒台独问题，它主要的目的就是挑动爱国情结、民族情绪。然后中共再把这种民族情绪进一步地引导，从这种所谓的爱国最后变成爱党，这个就是中共一直玩的老把戏。〞
 <br/>
 <br/>
 台湾大学国家发展研究所法学博士曾建元也表示，中共玩的台独议题，是在操弄民族主义。目的不在于台湾的领土或者是中国的统一，而在于它的政权需要。
 <br/>
 <br/>
 末夏写道，这次看起来蓝胜绿败，其实真没什么，这就是政党轮替，这恰恰是民主政治的精髓，老百姓永远可以说不，永远存在权力制约，这才是一个正常的政治生态与社会制度。
 <br/>
 <br/>
 蓝述：〝一般民主国家都是这样子，任何一个总统，到了中期选举的时候，失去一些票数也不奇怪，这跟台海两岸关系没有什么关系，更多的是在经济、交通、民众的生活等这些方面，你如何治理台湾。〞
 <br/>
 <br/>
 而针对不少台湾人抱怨经济发展欠佳，末夏在文章中指出，经济的提振对于民主制度与极权制度国家，是完全不同的体现，它背后还要关乎社会正义丶法治与全社会健康程度。在大陆，GDP很高，但那是虚胖，大陆的贫富差距与社会问题，包括各种安全隐患，是远远高于台湾与其他民主国家。如果从政治丶社会与经济全方位来看，大陆人应该羡慕台湾人。
 <br/>
 <br/>
 据了解，中共当局狂删中国网民有关台湾选举的议论，但中国网民不断变换招法，发出自己的评论。有网民说，这次选举显示了在台湾没有人民的名义，只有人民的选择。
 <br/>
 <br/>
 蓝述：〝讨论的越多，最后只可能产生一个结果，就是让更多的中国社会的民众向往民主的制度，这个向往必然会带来对中共政权的冲击，所以它当然是狂删了。〞
 <br/>
 <br/>
 也有中国网民说，面对失败，台湾执政党领导人感谢人民的指教，什么时候台湾海峡这边的执政党也感谢人民指教，而不是教训人民，中国就有希望了。
 <br/>
 <br/>
 海外华人也对台湾选举结果议论纷纷。
 <br/>
 <br/>
 美国《北京之春》杂志主编陈维健‏表示，只要是以选票获胜，都是对那个永不选举、强加民意的共产党的打击。
 <br/>
 <br/>
 旅居加拿大的民运人士盛雪写道：〝这次选举…是台湾民主进程和中共长期渗透之争。〞
 <br/>
 <br/>
 英国牛津大学和路透新闻学研究所最近发表的一份研究报告指出，中共军队掌控的社群媒体帐号所发布的贴文，今年被台湾新闻媒体大量采用，而且在使用这些帐号的照片和影片时，没有查验其真实性。
 <br/>
 <br/>
 采访/常春 编辑/陈洁 后制/钟元
