### 日月潭花火音乐会 呈现原民传统艺术之美
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】〝花响明潭〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花火音乐会.html" target="_blank">
  花火音乐会
 </a>
 ，3日晚间在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日月潭.html" target="_blank">
  日月潭
 </a>
 伊达邵码头热闹登场。以我国重要传统表演艺术为主轴，包括排湾族鼻笛与歌谣，布农族传统乐器与祭典歌谣，还有北管新生代乐团等，演出相当精彩。跟着镜头一起去欣赏。
 <br/>
 <br/>
 以鼻就笛，用身体深处吐出的鼻息，吹奏出动人的音符，排湾族鼻笛大师谢水能的拿手技艺，双管鼻笛，配合嘹亮浑厚的古调歌谣，传达心中的情感。
 <br/>
 <br/>
 排湾族鼻笛保存者谢水能：〝一个是勇士舞，一个是勇士歌，一个是情歌，大部分都是这样的。但是大部分都是情歌，古调和现代的都是情歌。我们这个鼻笛也是一样，这个鼻笛也是情歌，是男孩子去追求女朋友的乐器。〞
 <br/>
 <br/>
 打猎前的祭祀歌，打猎后报战功，还有庆祝丰收的饮酒歌等，布农族的各种祭典歌谣，是传统文化的一大特色。而传统乐器的弓琴和口簧琴，也充分展现出他们喜爱音乐的天性。
 <br/>
 <br/>
 布农文化协会弓琴演奏者 全伍阿现（Savugoz）：〝高兴的时候就弹，无聊的时候也要弹。〞
 <br/>
 <br/>
 由北管新生代圆仔花乐团伴奏，入围第27届金曲奖〝最佳原住民歌手奖〞，排湾族歌手少妮瑶・久分勒分清澈如泉的歌声，回荡在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日月潭.html" target="_blank">
  日月潭
 </a>
 畔。这场传统表演艺术音乐会，精彩纷呈，要带给现场民众，满满的回忆。
 <br/>
 <br/>
 新唐人亚太电视萧政忠、林沛芸台湾南投采访报导
</div>
