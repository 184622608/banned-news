### 中华民国107年双十国庆大典 总统府前举行(直播)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】中华民国107年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双十.html" target="_blank">
  双十
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国庆.html" target="_blank">
  国庆
 </a>
 大典，将于10月10日(周三) 早上9:05，在台湾总统府的凯达格兰大道举行。
 <br/>
 <br/>
 开场将由世界六大都的侨胞代表〝美国加州排舞协会〞带来广场舞，紧接着是振奋人心的陆海空三军联合乐仪队操演；随后有今年元旦才成立的宪兵指挥部〝快速反应连〞重机操演，这也是目前唯一使用重型机车的国军部队，以36辆时速可达180公里1300cc的重型机车，展现训练精良的各种高超技术。
 <br/>
 <br/>
 国歌领唱邀请到6位在国土边陲守护台湾海上安全的灯塔管理员，与国防大学合唱团一同领唱，共同彰显出今年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国庆.html" target="_blank">
  国庆
 </a>
 主轴〝2018台湾共好〞！之后将有一连串热闹的花车游行。最精彩压轴则是从总统府后方凌空而过的幻象2000英姿，将为今年的国庆大典画下完美的句点。
 <br/>
 <br/>
 新唐人亚太电视台全程为您进行直播，届时敬请阖家准时观赏，一同欢庆中华民国107岁生日！
