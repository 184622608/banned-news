### 政院谈联电窃密： 该办就办 美对台信任加深
------------------------

<div class="wysiwyg">
 【新唐人2018年11月03日讯】美国司法部日前起诉了福建晋华、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾联电.html" target="_blank">
  台湾联电
 </a>
 与三名涉案的台湾人，对此，财经媒体彭博社的文章警告，案件可能会影响美方对台湾产业的信任。不过，今天行政院表示，台湾司法协助美方取证，美方加强对台湾的信任，案件该办就要办。
 <br/>
 <br/>
 美国司法部1日正式起诉联电及福建晋华，窃取美国美光公司商业机密。尽管被调查的公司是台湾大企业，台湾法务部及调查局仍与美国司法部密切合作。
 <br/>
 <br/>
 行政院发言人Kolas Yotaka : 〝台湾的立场依然是勿枉勿纵，我们认为该办就要办。那在这一个过程当中，美国跟台湾互相的信任的关系，并没有因此破坏，反而因此更加强。〞
 <br/>
 <br/>
 美国在台协会也感谢台湾法务部在法律协助上的承诺，表示这是反对国家支持经济间谍活动，以国际法具体展现合作的案例，台湾的〝营业秘密法〞在调查中发挥巨大作用。
 <br/>
 <br/>
 行政院发言人Kolas Yotaka :〝这起案件，台湾的法务部，我们也提供了美方相对的协助，也促成了这个案件可以进入司法的程序。〞
 <br/>
 <br/>
 台中地检署去年依违反营业秘密法等罪，起诉两名前美光员工及联电。台中地院表示，该案去年九月初起诉，法官已开过七次庭，检辩双方争议不小，美方也曾派人来台取证。
 <br/>
 <br/>
 新唐人亚太电视邱添喜、詹咏茹、林妤台湾台中报导
</div>
