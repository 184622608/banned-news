### 台湾宜花东高屏旅宿补助 2500元怎么拿？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】中华民国交通部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观光.html" target="_blank">
  观光
 </a>
 局宣布于11月起推出〝前进宜花东‧高屏暖冬游〞方案，鼓励民众自由行住宿、团体旅游，引起热烈讨论，民众如何拿到补助，观光局表示，初期600多家参与的旅宿业者名单已经在观光局〝台湾旅宿网〞公布，供民众下载查询，从民宿到饭店通通都有，之后新增的旅宿名单将采滚动式陆续更新。
 <br/>
 <br/>
 这次活动自11月1日至12月31日止，于周日至周五期间，到宜兰（不含头城镇、礁溪乡及宜兰市）、花莲、台东、高雄、屏东旅游住宿，每人最高可补助1,500元住宿费，另外，首度补助〝旅游交通费〞每房最高1,000元。
 <br/>
 <br/>
 住宿方面会直接由饭店、旅宿业者〝直接折抵〞，而交通补助，不论是搭乘高铁、台铁、客运、船舶或国内飞机，只要〝提交票券正本〞，由旅宿业者核实垫付。
 <br/>
 <br/>
 以一房来说住宿可折抵1000元，交通费补助1000元，如有60岁以上长者，还可加抵500元，此趟可省下总计2500元。
 <br/>
 <br/>
 首波参与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观光.html" target="_blank">
  观光
 </a>
 局旅游方案的旅宿业者就多达600多家，其中包括五星级的太鲁阁晶英、台东娜路弯大酒店、理想大地、日晖国际度假村等，还有今年开幕的绿舞观光饭店与民宿业者都在其中。
 <br/>
 <br/>
 此外，像花莲翰品饭店、花莲美仑大饭店、高雄圆山大饭店、义大皇家酒店、屏东垦丁福华渡假饭店、凯撒大饭店等9家业者，主动提供每日限量3折住宿优惠，结帐时再享有观光局补助1000、1500元折抵优惠，目前参与观光局旅游方案的旅宿业者还在持续增加中。
 <br/>
 <br/>

 </center>
 <br/>
 另外，台东加码推出〝避寒台东 暖冬行动〞优惠，于107年11月1日至108年1月24日止，共有33家旅宿参与，平日2人同行〝二泊二食〞只要3,888元、5,888元、7,888元的住宿优惠，如果在12月底前入住，还可享观光局每房每日1000～1500元、交通费1000元补助优惠。
 <br/>
 <br/>
 但参加〝避寒台东 暖冬行动〞方案的旅宿业每日仅提供10个房间，想住者建议先提前预约查询。
