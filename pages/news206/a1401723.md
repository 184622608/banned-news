### 北市长选举3254票之差 北院启动验票作业
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 落幕，国民党台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 以3254票之差落败，声请重新计票，台北地方法院3日(周一)早上启动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_验票.html" target="_blank">
  验票
 </a>
 作业。丁守中表示，这是一个追求真相的机会。对于是否会担心结果翻盘，现任市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 说，其他阵营有监票部队，〝所以放心好了〞。
 <br/>
 <br/>
 中央社报导，台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 以3254票之差险胜
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 ；丁守中日前召开记者会，宣布重新提司法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_验票.html" target="_blank">
  验票
 </a>
 ，并在律师陪同下，到北院声请查封票匦重新计票，并缴交保证金新台币428万元。
 <br/>
 <br/>
 北院表示，今天在台北市选委会启动验票作业，支援的法官跨各庭，但不要求参与法官取消原庭期；目前规划以50组人力投入，每组配置法官一人、书记官一人、丁阵营律师、柯阵营律师、2位选务人员。
 <br/>
 <br/>
 北院昨(2日)公布九大验票流程，包括验票组人员坐定后，即由书记官（于协助下）领取选票箱、选举人名册、投开票所报告单，进行验票等。
 <br/>
 <br/>
 此外，当有争议选票会由选务人员举手，经现场北院巡回人员影印后交还选票原本及影本，影本由两造签名，由北院带回；原本则集中封存，置回选票箱由选委会保管。
 <br/>
 <br/>
 <h4>
  丁守中：让市民了解真相机会
 </h4>
 <br/>
 丁守中早上8时左右抵达台北市选举委员会，并接受媒体访问指出，非常感谢有那么多律师站出来，愿意一起守护台湾民主，让市民大众有一个了解真相的机会。
 <br/>
 <br/>
 丁守中也说，事实上整个选务人员都是台北市政府在选定的，大家现在也了解到，很多选务人员根本不符合资格，这也是这次（选举）会有那么多问题的原因之一；另外律师会依据验票结果，来进行后面诉讼程序。
 <br/>
 <br/>
 <h4>
  翻盘？柯文哲不担心
 </h4>
 <br/>
 对于会不会担心结果翻盘，柯文哲上午7时20分简单回答媒体问题。柯文哲说：〝这一次我们没有派监票人员，只有对手有派，所以放心好了。〞
 <br/>
 <br/>
 由于总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 11月26日致电祝贺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六都市长.html" target="_blank">
  六都市长
 </a>
 当选人，并说会尽快安排时间会面，有媒体询问柯文哲是否已排好与蔡总统碰面时间，柯文哲说〝还没〞。
 <br/>
 <br/>
