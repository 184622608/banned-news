### 2018台湾九合一选举 规模或创历史纪录
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 前夜，全台各地选前造势晚会登场。九合一选举被视为2020年总统大选的前哨战。
 <br/>
 <br/>
 周六的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 将选出1万1千多名地方官员，其中最受关注的是台北和高雄的市长选举。
 <br/>
 <br/>
 无党籍台北市长柯文哲争取连任。
 <br/>
 <br/>
 而民进党台北市候选人姚文智有蔡英文总统站台。
 <br/>
 <br/>
 台湾总统蔡英文：〝守护台湾的民主，守护台湾的主权。〞
 <br/>
 <br/>
 国民党籍候选人丁守中的造势口号是〝台北要先赢，投票救台湾〞。
 <br/>
 <br/>
 在民进党执政约20年的高雄市，国民党支持面扩大。该党候选人韩国瑜打出〝打造高雄 全台首富 世界第一等〞的口号。
 <br/>
 <br/>
 高雄市长国民党候选人韩国瑜：〝一个市长，要像超级推销员，会把高雄的好，高雄的美，高雄的产品，推销到全世界。〞
 <br/>
 <br/>
 民进党候选人陈其迈则延续爱与包容的基调，双方展开势均力敌的拉锯战。
 <br/>
 <br/>
 在台湾这个拥有2300万人口的岛屿上，预计将有1900万名选民参加11月24日的投票，这次选举有可能成为台湾有史以来规模最大的选举。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
