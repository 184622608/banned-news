### 英特尔携手五大台厂 抢攻Face ID新商机
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英特尔.html" target="_blank">
  英特尔
 </a>
 公司在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 发表全新的Intel视觉加速器，并且携手五大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台厂.html" target="_blank">
  台厂
 </a>
 业者，打造一系列多元应用，展示脸部识别、影像分析技术的庞大商机，英特尔高层更表示，台湾是英特尔在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_物联网.html" target="_blank">
  物联网
 </a>
 产业领域的重要伙伴。
 <br/>
 <br/>
 看着萤幕，一秒解锁，苹果iPhone X，靠着Face ID，也就是眼部辨识声名大噪，其实视觉影像分析，进入你我生活周遭。
 <br/>
 <br/>
 奇偶科技业务处专业开发经理 李彦霆：〝那它这个东西所主要应用，可以应用在栅栏机上面，辨识到人员才可以进入。比如说可以装在办公室里面做打卡的功能，所以员工上下班的时候也可以透过它去打卡， 〞
 <br/>
 <br/>
 识别人脸，取代上下班打卡，也兼顾保全。不只如此，摄影机捕捉男性女性顾客，还能广泛应用在零售业。
 <br/>
 <br/>
 大猩猩科技产品经理 邱文岳：〝举例来说下面这个萤幕，我们人在经过这个画面的时候呢，它同样会抓取看广告这个人，他的年龄跟性别，推荐依照年龄跟性别去推荐适当的广告。〞
 <br/>
 <br/>
 研扬科技协理 赖立凯：〝更多新的应用在各个地方，比如说我们看到工安这边，其实就有很多地方可以利用AI的方式，去做更快速的运算。它其实在做的时候，其实 就可以做到，即时的运算、即时的回馈，那也可以减少你所谓3G、4G的传输的数量。〞
 <br/>
 <br/>
 晶片大厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英特尔.html" target="_blank">
  英特尔
 </a>
 （Intel）在台发表视觉加速器设计产品（Intel Vision Accelerator DesignProducts），强化人工智慧（AI）推论与分析效能，以即时的边缘运算，不需云端连网，就能分析视觉影像，携手国内研扬、研华科技、凌华、威强电、新汉5家业者，打造各式应用场景。
 <br/>
 <br/>
 英特尔资深副总裁暨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_物联网.html" target="_blank">
  物联网
 </a>
 事业群总经理 Tom Lantzsch：〝零售业系统、制造业系统、智慧城市，这些我们可以运用电脑视觉影像运算技术。是的，我认为如果我们可以一同分享视觉处理器(Movidius)革新技术，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 业者伙伴一同携手，我认为产业前景是相当可观。〞
 <br/>
 <br/>
 首次与台湾媒体见面的兰奇表示，2017年到2020年，全球物联网市场规模将超过330亿美元，约新台币一兆元规模，而台湾都是英特尔发展物联网技术的重要伙伴，也看好未来市场商机。
 <br/>
 <br/>
 新唐人亚太电视王冠霖、沈唯同台湾台北报导
