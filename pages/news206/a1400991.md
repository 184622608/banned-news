### 大选后 学者：提防台湾成中共避税转口港
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一大选.html" target="_blank">
  九合一大选
 </a>
 ，高雄绿地变蓝天，当选人韩国瑜在胜选时，就立即表明，面对中国的态度就是：〝九二共识〞，财信传媒集团董事长谢金河，在脸书发文指出，台湾面临两条路线抉择，一是直接靠拢中国，在九二共识下讨生活，另一条是：台湾把世界变腹地，成为小而美的经济体。也有专家提醒，要提防台湾日后变成中共规避美国关税的〝转口港〞。
 <br/>
 <br/>
 2018九合一选举，掀起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾经济.html" target="_blank">
  台湾经济
 </a>
 走向的讨论，财讯传媒董事长谢金河脸书发文说，台湾拚经济面临两条路线抉择：一是直接靠拢中国，二是台湾把世界变腹地，成为小而美的经济体。AIA Capital首席经济学家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴嘉隆.html" target="_blank">
  吴嘉隆
 </a>
 认为，陆客来台的短期〝红利〞，对台湾经济带动有限，并不可行。
 <br/>
 <br/>
 AIA Capital首席经济学家 吴嘉隆：〝我们发现韩国瑜当选之后，当然是关心台湾、高雄经济，但是他的第一个做法是，成立两岸工作小组、承认92共识，要来跟大陆连结，要来跟郭台铭通电话，来招商引资，问题是这一套想法，有它的极限，我估计可能半年就把它发挥的差不多了。所以我们必须寻找新的概念、新的模式、新的策略，来提升台湾的经济能量。〞
 <br/>
 <br/>
 专家解析，美国印太战略，对准中共一带一路，不排除对峙转向军事层面，围堵中共态势将更为明朗。此外，贸易战短期内不会结束，台湾也得小心，不要成为中共规避关税的〝转口港〞，以免未蒙其利，先受其害。
 <br/>
 <br/>
 AIA Capital首席经济学家 吴嘉隆：〝台湾的政府，不管是民进党，还是国民党，要修改一个观念，就是说要积极来跟美国的新的全球战略来配合。台湾选边，几乎只能选美国，不能选大陆，选大陆的话结果是一起陪葬，一起落难，你看香港的情况，香港的情况其实很不好。香港已经越来越像
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国大陆.html" target="_blank">
  中国大陆
 </a>
 的内地城市了，所以美国可能要修改美港关系法，不再给香港关税优惠，一样的逻辑会套用在台湾。〞
 <br/>
 <br/>
 美国总统川普和中国国家主席习近平，本周末，将在阿根廷举行的G20高峰会上会面，更被外界视为，美中关系走向关键，AIA首席经济学家吴嘉隆，预测美中可能走向〝新冷战〞阶段。
 <br/>
 <br/>
 AIA Capital首席经济学家 吴嘉隆：〝美国把川习会，月底的川习会认定为最后一次机会，如果中国大陆不能全面接受美国的要求的话，接下去就是冷战。所以在美国印太战略之下，南中国海是将来的争执焦点。〞
 <br/>
 <br/>
 专家建议台湾与其倚赖中国大陆，不如思考，如何加深与欧美日的经济连结，从制造业思维转到设计师思维，藉由欧美市场，培育高附加价值产业，才不至于在国际新局势下，丧失先机。
 <br/>
 <br/>
 新唐人亚太电视林嘉韦、沈唯同台湾台北报导
</div>
