### 吴音宁下台？韩国瑜吓一跳 网友拜托她别走
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】台湾选举落幕，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北农总经理.html" target="_blank">
  北农总经理
 </a>
 动向受关注。农委会主委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林聪贤.html" target="_blank">
  林聪贤
 </a>
 今天说，台北农产运销公司总经理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴音宁.html" target="_blank">
  吴音宁
 </a>
 阶段性任务已完成。吴音宁对外界询问则暂不回应相关问题。曾任北农总经理的高雄市长当选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 说，北农风风雨雨经过两年700多天，结果会这样真的吓一跳。
 <br/>
 <br/>
 综合媒体报导，台北市长选举结果出炉，柯文哲得票数险胜对手，成功连任，外界关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北农总经理.html" target="_blank">
  北农总经理
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴音宁.html" target="_blank">
  吴音宁
 </a>
 动向。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林聪贤.html" target="_blank">
  林聪贤
 </a>
 上午赴立法院列席备询前，被媒体问到有关吴音宁相关问题。
 <br/>
 <br/>
 林聪贤说，吴音宁的阶段性任务应该已经完成，再来可以跟台北市政府好好谈总经理的人选，农委会也会配合市政府节奏。
 <br/>
 <br/>
 柯文哲受访时说，人选由农委会提出，没问题就同意，他只要求专业，还有愿意备询。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-28/p9147212a78781356.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-28/p9147212a78781356-ss.jpg"/>
   <br/>
   <font size="-1">
    台北市长柯文哲（图）在台北市政府接受媒体访问时说，人选由农委会提出，没问题就同意，他只要求专业，还有愿意备询。(中央社)
   </font>
  </a>
 </center>
 <br/>
 北农董事长、台北市工务局长彭振声则表示，总经理人选是农委会决定。
 <br/>
 <br/>
 而吴音宁至发稿时，仅透过简讯回复，相关议题目前皆不对外回应。
 <br/>
 <br/>
 这次大选民进党遭逢空前挫败，吴音宁也因菜价惨跌、农产品供需失调被指是绿营败选首要战犯，加上吴音宁坚持不到北巿议会备询，让北巿长柯文哲屡遭议员非议。
 <br/>
 <br/>
 据熟悉内情人士透露，两天前行政院和北巿府已达成让吴音宁下台的共识，但为避免把场面弄得太难看，曾建议吴音宁自行请辞。
 <br/>
 <br/>
 中时电子报报导，据转述，吴音宁表示她由北农董事会任命，若要她去职，也应由董事会决议，她不会自行请辞。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-28/p9147211a931689405.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-28/p9147211a931689405-ss.jpg"/>
   <br/>
   <font size="-1">
    国民党籍高雄市长当选人
    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
     韩国瑜
    </a>
    （图）28日被问及北农总经理吴音宁是否遭〝韩流〞灭顶而下台，韩国瑜表示，不敢这么说，自己本来就是卖菜郎，没这么厉害。(中央社)
   </font>
  </a>
 </center>
 <br/>
 高雄市长当选人韩国瑜中午在台北与媒体高层、名嘴等餐叙，会后被问到吴音宁下台一事时，顿了一下，他说，〝我吓一跳，真的吓一跳〞。
 <br/>
 <br/>
 韩国瑜说，他是卸任总经理，一概不评论，并认为未来总经理要顾及农民、消费者和劳工之间的平衡，三者都要照顾好，这是台北农产总经理未来要面对的重大任务。
 <br/>
 <br/>
 被问到过去被鲁莽拔掉北农总经理一职，继任者在这么短时间内又去职，韩国瑜说，一次人事就是一次涅盘，各有各的故事。他的离职经过风雨，过程中忍受很多，也让他成长很多。
 <br/>
 <br/>
 韩国瑜说，如果吴音宁真的去职，也祝福她一切顺利，平平安安。
