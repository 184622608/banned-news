### 台湾加入战团孤立中共？农业团体签下巨额大豆订单
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月02日讯】在中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 中，中共为报复美国政府而宣布对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 加征高额关税，导致美国大豆价格下跌了。近日，台湾农业团体与美国大豆种植户签下了高达15亿美元的大豆购买协议，被认为是接棒欧洲支持美国政府，继续孤立中共的行动。
 <br/>
 <br/>
 中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 从今年7月初正式开打，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 就被中共当局纳入了第一批报复美国的征税清单中。美国生产大豆的地区是共和党的票仓，中共此举被认为是试图通过打击美国的大豆种植户来影响该地区民众支持共和党以及川普的民意。
 <br/>
 <br/>
 由于中国过去是美国大豆出口最主要的市场，所以自从中共当局对美国进口大豆加征高达25%的关税以来，面临出口困难的美国大豆的价格，在近3个月内已下跌15％左右。
 <br/>
 <br/>
 9月26日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾农业贸易友好代表团.html" target="_blank">
  台湾农业贸易友好代表团
 </a>
 在华盛顿与美国大豆种植户团体签定了一份大豆采购意向书。这份意向书由台湾区植物油制炼工业同业公会理事长洪尧昆与美国黄豆协会理事柯文代表签署，台驻美代表高硕泰和多名美国议员出席了签约仪式。
 <br/>
 <br/>
 根据这份协议，台湾的农业团体决定在2018年至2019年期间，从美国进口价值15.6亿美元的美国大豆，比他们去年承诺的采购金额增加了30%。对于台湾在关键时刻的支持，美国政商两界均有代表发言表示感谢。
 <br/>
 <br/>
 美国大豆协会理事 Wade Cowan当时即对前往采访的媒体说：〝自从我们和中国有贸易纷争后，大豆价格下跌。可喜的是有像台湾这样的朋友站出来，购买比原来承诺的更多。〞
 <br/>
 <br/>
 美国会台湾连线共同主席，佛州众议员 Mario Diaz-Balart也在发言时说：〝我们感谢台湾，再次挺身而出，显示出美台之间的友好关系。〞
 <br/>
 <br/>
 台湾农产品访问团团长洪尧昆事后接受媒体采访时证实，这次的签单的确比去年计划的数量多买了60万吨大豆，他说：〝我们并不只是因为（美国大豆）的价格便宜，(而是)因为我们其实跟他们（美国大豆种植户）的关系更深厚嘛。〞
 <br/>
 <br/>
 台湾农业团队的这个行动，被认为是在中美贸易战中，接棒欧洲加入了孤立中共的行动之中。
 <br/>
 <br/>
 此前，自7月份美国总统特朗普与欧盟委员会主席容克达成了避免美、欧贸易冲突升级的协议后，欧洲明显加大了购买美国大豆的规模。
 <br/>
 <br/>
 据欧盟公布的数据，在截至9月中旬的过去12周里，欧盟进口的美国大豆达到147万吨，占欧盟进口大豆总量的52%，较去年同期增加133%。
 <br/>
 <br/>
 与此同时，巴西大豆在欧盟进口大豆总量中所占比重降至40%，显示美国已经取代巴西，成为欧盟最大的大豆供应国。
 <br/>
 <br/>
 欧盟数据显示，以避免贸易战以来，欧洲增购美国大豆的行动，不仅是因为美国大豆价格更便宜，这也是欧盟为了支持美国政府降低贸易战对美国豆农冲击而采取的一个行动。
 <br/>
 <br/>
 因此，这次台湾农业团体与美国方面签订大规模购买美国大豆的采购意向书，被外界视为台湾当局与欧美联手，在贸易战中孤立中共的一个迹象。
 <br/>
 <br/>
 据公开的资讯，台湾农业贸易友好代表团从1993年以来就一直就与美国保持着亲密的合作关系。每隔几年，该贸易团都会和美国签订采购意向书。2013年，该团与美国签订了2014年至2015年向美采购约15亿美元黄豆的采购意向书；2015年该团体更与美国方面签署了30亿美元的采购计划，当时有9位美国联邦参议员和8位联邦众议员到场参观礼。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：明轩）
