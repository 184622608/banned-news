### 观大选 台工商界大老：盼政党政治善性循环
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】大选后两天，国策研究院26日举办〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 与两岸走向〞座谈会、向政府建言，工商界大老
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_许胜雄.html" target="_blank">
  许胜雄
 </a>
 全程参与聆听，关心台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主.html" target="_blank">
  民主
 </a>
 发展，期盼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政党政治.html" target="_blank">
  政党政治
 </a>
 善性循环。
 <br/>
 <br/>
 多位学者深度对话分析，九合一大选的启示与影响。台下一位背影，神似工商界重量级人物，没错，三三会理事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_许胜雄.html" target="_blank">
  许胜雄
 </a>
 ，整个下午当听众，专注聆听，关心台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主.html" target="_blank">
  民主
 </a>
 ，他谈到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政党政治.html" target="_blank">
  政党政治
 </a>
 与选举的目的。
 <br/>
 <br/>
 三三会理事长 许胜雄：〝我们要使得台湾是一个民主自由的国家，而且因为透过政党政治选举的关系，使得执政的人，不能落入傲慢，使得执政的人，能具有慈悲心。使得在野的政党，去看到执政体系，就是对方有哪些不足的地方，应该去改善、应去提升，来造福这社会国家。他提出更好的对策，使得未来选举的时候，能够让选民让人民来认同、来投(票给)它，这样子政党就可以造成善性循环。〞
 <br/>
 <br/>
 许胜雄说，民意如流水，选举结果是一时的，他一直思索，如何让政党政治更好，这是台湾永续发展非常重要的根基。许胜雄还提到，假新闻摧残人性与信任。
 <br/>
 <br/>
 三三会理事长 许胜雄：〝恶质地去塑造，不管假新闻也好，或什么东西，你去摧残(本来)人性应该往善良的地方走。〞
 <br/>
 <br/>
 多位学者表示，选举结果显示，选民不是支持国民党，而是对执政成果不满；期许两大政党，都能反思提升。
 <br/>
 <br/>
 国策研究院董事长 田弘茂：〝(希望)两个大的政党，能互相激励、互相竞争，是一个民主政治，能够维持继续不断稳定的重要因素。〞
 <br/>
 <br/>
 董立文表示，台湾跟韩国，是全球第三波民主化的同班同学，三十多年年轻的民主，各有问题，台湾的问题，可以看成是在〝交学费〞。
 <br/>
 <br/>
 台湾智库谘询委员 董立文：〝每次的选举，都会被中共批评的一文不值，它常态性的在妖魔化我们台湾民主、也妖魔化美国的民主。这个两岸关系，如果把它放在是专制跟民主的对抗的话，那么对民主的信心，要更加的强韧，才挡得住中共〞前国安会谘询委员 苏进强：〝我最后的结论就是，希望民主焦虑赶快过去，不要变成民主内战。那对两党都要有一些期待。〞学者探讨，蔡政府重大改革密集进行、民生经济问题、民意的善变、中共的细致干预等因素，都影响这次大选结果。
 <br/>
