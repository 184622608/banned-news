### 中共抛〝居住证〞统战！时力提法案反制
------------------------

<div class="wysiwyg">
 【新唐人2018年09月25日讯】再来看到，中共推出所谓的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港澳台居民.html" target="_blank">
  港澳台居民
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_居住证.html" target="_blank">
  居住证
 </a>
 ，让持有居住证的人，能有与中国大陆人民相近的社会生活待遇。对此，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_时代力量党.html" target="_blank">
  时代力量党
 </a>
 直言，这是对台湾的统战，今天召开记者会，提出提出《两岸人民关系条例》修正草案，内容针对持有居住证的人，予以除籍、限缩
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公民权.html" target="_blank">
  公民权
 </a>
 。
 <br/>
 <br/>
 时代力量立委 徐永明：〝我们要有一个很强烈的反应才行，才能让我们在那边的台商、我们的台生，知道政府是有态度的，让他们可以去跟，对岸的这样一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_居住证.html" target="_blank">
  居住证
 </a>
 推动，有一个谈判的筹码。〞
 <br/>
 <br/>
 时代力量立委 高潞．以用：〝这个港澳台居住证，它不只是，不只是对台湾的统战，他甚至就是矮化我们台湾的地位，那时代力量对于就是因应的方式，我们也提出了两岸人民关系条例的部分修正，部分条文修正案。〞
 <br/>
 <br/>
 时代力量指出，今天时代力量就会将修正草案送到立院议事处，预计下周二会进程序委员会。并表示，被除籍者，只要能证明中共发的居住证已经失效，很快能恢复身份。希望行政部门也拿出态度面对，表达保护台商、台生的决心。
 <br/>
 <br/>
 新唐人亚太电视综合报导
</div>
