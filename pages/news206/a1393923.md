### 国际风筝节登场 五彩缤纷风筝齐聚北海岸
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】2018北海岸
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际风筝节.html" target="_blank">
  国际风筝节
 </a>
 ，上周日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新北市.html" target="_blank">
  新北市
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_石门.html" target="_blank">
  石门
 </a>
 白沙湾登场，吸引超过8个国家20支队伍，100多位国内外风筝好手同台较劲，各种特色风筝在北海岸上空争奇斗艳，带您一起去看看。
 <br/>
 <br/>
 大白鲨、山姆大叔与小朋友喜欢的各式卡通软体风筝，还有特色平面造型风筝、冲浪特技风筝等，一起飞上天，把北海岸天空点缀的五彩缤纷。
 <br/>
 <br/>
 泰国队选手 Ron Spaulding：〝我们热爱来台湾的风筝节，为什么我们要来？看看四周。美丽的海滩、美丽的天气、最棒的友善的人们、非常好的食物。〞
 <br/>
 <br/>
 美国队选手 李港生：〝我这次带来了山姆大叔风筝，它属于软体风筝，没有骨架的，所以最适合在这种海边放，像今天的海浪、还有这个风非常的好，所以我们非常喜欢，也喜欢这个地方。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新北市.html" target="_blank">
  新北市
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_石门.html" target="_blank">
  石门
 </a>
 区所长 吴嘉荣：〝一共有8个国家20支队伍，100多位风筝高手齐聚在这里一展身手，石门有两个特色，一个是风筝一个是肉粽，所以我们就是以五公尺大的肉粽的造型，主题的风筝来作为今天的主题。〞
 <br/>
 <br/>
 半月型的白沙湾，非常适合冲浪，所以也吸引不少风筝冲浪好手前来大展身手，利用风力拖曳滑行水上展现超高的技巧。
 <br/>
 <br/>
 风筝冲浪玩家 江茂松：〝白沙湾算是比较高阶的场所，初学的比较不适合；这个有相当的危险性，第一个风筝的拉力相当大，把你拉上去以后，下来不小心就会断手断脚，只要有教练教你怎么玩，这个很容易入手，因为它很容易入手，玩起来非常非常有乐趣。〞
 <br/>
 <br/>
 石门又称风筝的故乡，每年吸引世界各国风筝好手来此放飞，互相交流；区长吴嘉荣表示，透过这样的活动，除了让民众认识各地区不同风俗民情独特的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_风筝文化.html" target="_blank">
  风筝文化
 </a>
 ，也希望籍此行销石门在地观光及产业，带动地方经济。
 <br/>
 <br/>
 新唐人亚太电视胡淑霞、张年洁台湾新北市采访报导
</div>
