### 张忠谋APEC会彭斯 台美公开层级最高会面
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】亚太经合会（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 ）领袖峰会，周六在巴布亚新几内亚首都登场。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾代表.html" target="_blank">
  台湾代表
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张忠谋.html" target="_blank">
  张忠谋
 </a>
 跟美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 ，在场边举行双边会谈，国际媒体纷纷报导，这也成了台美公开最高层级的会面。彭斯接受美国媒体访问时表示，会把台美自由贸易协定(FTA)提案带回去。带您看到，这次APEC会议台湾有哪些重要突破！
 <br/>
 <br/>
 台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 领袖代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张忠谋.html" target="_blank">
  张忠谋
 </a>
 16号抵达巴纽，隔一天就在APEC场边跟美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 ，举行双边会谈。从照片中似乎可以看出，两人相谈甚欢，外交部也在推特上写说，会谈将强化促进包容性成长与数位发展的区域连结。
 <br/>
 <br/>
 而前一天台湾举办的双部长晚宴，台湾在南太平洋的四个友邦领袖纷纷出席，力挺意味相当浓厚。国发会主委陈美伶更指出，今年的APEC部长会议联合声明中，台湾提出的4项倡议跟6段文字，都被大会采纳。
 <br/>
 <br/>
 经济部长沈荣津：〝我们在APEC的平台，大概我们在中小企业这一块，我们相当的积极去做贡献，所以我们过去在跟越南、菲律宾、马来西亚，甚至于美国都跟我们都很紧密的在做合作、交流。〞
 <br/>
 <br/>
 行政院政委邓振中也表示，他已经向各国表达台湾希望加入CPTPP的意愿以及准备的状况，而台湾高标准的自由化也获得肯定。
 <br/>
 <br/>
 经济部长沈荣津：〝一定要穷尽一切努力，因为下一波我们一定把握这个机会。我们在法规的调适，已经都做了一些修法的动作，大概我们预计这个会期应该就可以通过。现在就是要跟国际沟通，所谓国际沟通就是这些第一轮的会员国，我们要表达我们要积极参与的意愿，那透过APEC这个平台，我觉得满好的。〞
 <br/>
 <br/>
 另外，有平面媒体报导指出，政府高层透露，张忠谋跟日本首相安倍晋三的双边会谈，将在领袖峰会的第二天，也就是18号下午举行，预计张忠谋会向安倍表达台湾极力争取加入CPTPP的意愿。
 <br/>
 <br/>
 新唐人亚太电视赵庭誉整理报导
