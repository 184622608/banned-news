### 全球孩童创意行动挑战台北登场 40国参与
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】小朋友的创意和潜力真的是无限的，带您看到，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球孩童创意行动.html" target="_blank">
  全球孩童创意行动
 </a>
 挑战，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北.html" target="_blank">
  台北
 </a>
 举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球年会.html" target="_blank">
  全球年会
 </a>
 ，有超过40个国家的小朋友参与，他们将把生活中感受到的问题，以及解决问题的故事，带上台分享给大家，一起来看。
 <br/>
 <br/>
 小朋友们在台上，用英文与来自世界各国的大小朋友交流。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球孩童创意行动.html" target="_blank">
  全球孩童创意行动
 </a>
 挑战(DFC)，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北.html" target="_blank">
  台北
 </a>
 举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球年会.html" target="_blank">
  全球年会
 </a>
 ，永龄教育基金会协助他们在台湾推动这个计划，已经第三年，今年年会，有超过40个国家的孩子参与。
 <br/>
 <br/>
 永龄教育慈善基金会董事长 郭守正：〝有参加的小朋友，他们的世界观都会因此而增进。即使英文不是那么的好，但大家都非常有自信，跟其他的国交流。感觉得到说，其实昨天晚上就像个小小联合国一样，非常的感人。大家都非常地积极想要改变这个世界，然后自己不会觉得说，自己的力量不足，而是勇于实践，然后可以做出很多很不一样的事情。〞
 <br/>
 <br/>
 周六上午的活动中，现场聚集1000人，小朋友们分组上台，分享他们发现的问题，以及解决问题的方法，试图用自己小小力量，让世界变得更好，也提升自己独立思考能力。
 <br/>
 <br/>
 永龄教育慈善基金会董事长 郭守正：〝推动这件事情，同时也发现，台湾有很多的小朋友，都有这个能力，都在做很多很有意义的事情，所以我觉得今年，特别我们邀请了全球的年会，这次有40几个国家来参加。〞
 <br/>
 <br/>
 郭守正指出，许多大人会否定小孩的能力，但小孩子其实很有能力也很有想法。发起DFC挑战的印度河滨学校校长，则表示今年是2009年来规模最大的年会，很感谢每个参与、协助计划的人。
 <br/>
 <br/>
 新唐人亚太电视林嘉韦、黄彦菱台湾台北报导
</div>
