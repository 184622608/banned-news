### 韩国瑜助选喊政治归零 陈其迈辞立委明志
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾高雄.html" target="_blank">
  台湾高雄
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_市长选战.html" target="_blank">
  市长选战
 </a>
 激烈，民进党的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 周六宣布辞去立委，力拼市长一职。同一天，国民党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 则是北上彰化助选，再喊要让高雄政治归零。
 <br/>
 <br/>
 民进党高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈其迈.html" target="_blank">
  陈其迈
 </a>
 ：〝其迈自今天开始，辞掉立法委员，我要拜托大家在这里见证，其迈的人生，未来的岁月，我的性命，只有一个任务，就是繁荣高雄、守护台湾、守着台湾的民主，这样好吗？〞
 <br/>
 <br/>
 民进党高雄市长候选人 陈其迈宣布辞立委明志，强调繁荣高雄、守护台湾是一生志业，而对手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 ，同天晚上，则是北上彰化与党主席吴敦义，一起为同党候选人站台，再喊政治归零。
 <br/>
 <br/>
 国民党高雄市长候选人 韩国瑜：〝我住在云林县住25年，彰化县，算隔壁亲家，我现在来南部高雄，大家想法都一样，我在高雄我常常说，你要搞政治，你要搞意识形态，麻烦你去台北，总统府，行政院，立法院都在台北。你南部人，搞政治要做什么？〞
 <br/>
 <br/>
 上一届高雄市长选举，陈菊拿下99万张选票，大胜国民党杨秋兴的45万张票；这次高雄市长选情，民进党不敢轻敌，总统蔡英文周日回防高雄铁票区催票。
 <br/>
 <br/>
 新唐人亚太电视叶锡鸿、林宏观彰化高雄报导
