### 台铁列车出轨 国一女不顾受伤沉着救人报案
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】台湾铁路6432次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 列车21日下午搭载旅客共366人，从树林出发开往台东，不幸在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宜兰县.html" target="_blank">
  宜兰县
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冬山.html" target="_blank">
  冬山
 </a>
 =苏新间东正线发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出轨.html" target="_blank">
  出轨
 </a>
 事故。目前已知18人罹难，上百人受伤。一名在火车上的卑南国中杨姓女学生第一时间救人，并随手捡起手机报案。
 <br/>
 <br/>
 据中央社报导，卑南国中校长游数珠和3名老师、20位学生到韩国进行交流，搭上这班列车返回台东，目前校长无法取得联络。
 <br/>
 <br/>
 交流团导游是朱姓退休公务员，记者拨打他的电话，接电话的是卑南国中一年级杨姓女学生。
 <br/>
 <br/>
 〝很恐怖，我真的不知道怎么发生，一阵恐怖后，火车就倒了，怎么发生的都不知道，车厢内座椅坏了，行李散落车厢〞。
 <br/>
 <br/>
 她说，回神后，立即挣脱出车厢，大喊救人，〝很多人受伤〞，然后又立即奔回车厢救人，她在车厢内随手捡起别人掉落的手机拨打紧急电话报案，请他们快来救人。
 <br/>
 <br/>
 她说，当时不知道这支手机是导游的，因为现场看不到导游，〝可能被压住了〞，后来有人陆续拨打这支电话进来，她才知道是导游的电话，〝现在还不知道他在哪里〞。
 <br/>
 <br/>
 她说，一年级的学生坐在第一节车厢，包括她有4人受伤，一人骨折比较严重，她则背部疼痛，脸部擦伤，目前他们在苏澳荣民医院。坐在其他车厢的3年级学长都平安。
 <br/>
 <br/>

 健保署署长李伯璋晚间表示，医院若遇到个人资料不详伤者，不分国籍均以例外就医方式抢救，若为外国人，健保署也将代垫医疗费用。
 <br/>
 <br/>
 李伯璋表示，由于地方政府都有编列紧急救难专款，呼吁医院即便遇到未纳健保的外籍人士或中国大陆人士，也应立即救治，健保署将代垫医疗费用后，再向地方政府请款。
