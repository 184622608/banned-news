### 不可错过！高雄悬日美景 11/11封街看夕阳
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】台湾南部的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄市.html" target="_blank">
  高雄市
 </a>
 每年有两次如美国纽约〝曼哈顿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悬日.html" target="_blank">
  悬日
 </a>
 〞美景，去年悬日首度在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青年路封街.html" target="_blank">
  青年路封街
 </a>
 ，但当天因天候不佳民众久等无夕阳出现，今年市府为弥补民众缺憾，将于11月11日(周日)再度封街，让民众在安全的环境下拍摄欣赏悬日。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄市.html" target="_blank">
  高雄市
 </a>
 政府观光局20日新闻稿表示，中央气象局预测11月11日日落时间为傍晚5时15分，市府当天下午4时至6时会在青年一路(民权一路至维仁街)快慢车道实施封路管制。
 <br/>
 <br/>
 观光局长曾姿雯表示，著名〝曼哈顿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悬日.html" target="_blank">
  悬日
 </a>
 〞（Manhattanhenge）可以观赏到夕阳余晖洒满东西向的街道，而高雄街道与曼哈顿拥有相同条件，因此同样能欣赏到一年两次的悬日奇景。
 <br/>
 <br/>
 曾姿雯表示，高雄获得国际知名旅游指南－孤独星球(Lonely Planet)推荐为2018年10大最佳旅游城市第5名，悬日奇景再度增加高雄的魅力。
 <br/>
 <br/>
 她说，除观赏悬日外，温暖的高雄也很适合户外活动，而这次为了配合悬日美景活动，观光局也与高雄市五星级饭店及旅宿业者合作推出超值专案，民众出示与悬日自拍照片，可享受各饭店优惠，旅客不只观赏悬日美景，也可以享受优惠住宿及前往热门景点逛逛、享用在地特色美食小吃。
 <br/>
 <br/>
 观光局也建议民众搭乘大众交通工具到附近再步行前往，如果自行开车前往民众可利用邻近的民权轻钢架停车场及四维立体停车场，同时当天公车路线将改道行驶，提醒多加留意。
 <br/>
