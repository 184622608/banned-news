### 世界杯拔河锦标赛 台夺6金1铜打平上届纪录
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月23日讯】2018世界杯室外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拔河锦标赛.html" target="_blank">
  拔河锦标赛
 </a>
 在南非开普敦举行，22日举行最后一天赛程。台北市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_景美女中.html" target="_blank">
  景美女中
 </a>
 联手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾师范大学.html" target="_blank">
  台湾师范大学
 </a>
 在〝女子540公斤锦标赛〞项目夺冠，累计这届比赛共摘下6面金牌，打平上届锦标赛纪录。
 <br/>
 <br/>
 据中央社报导，景美师大联队台北时间22日晚上8时在女子540公斤锦标赛出战南非、巴斯克、英格兰、瑞士、荷兰、津巴布韦、瑞典、苏格兰等队，以2比0全胜挺入准决赛，再以2比0力克荷兰队，进入决赛。
 <br/>
 <br/>
 在决赛中，中华台北对上劲敌瑞士队，第1局胜出后，于第2局陷入鏖战，因犯满3次犯规被判败；第3局全力激战后，终以2比1强压瑞士队，斩获金牌。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_景美女中.html" target="_blank">
  景美女中
 </a>
 秘书顾秀华指出，因为欧洲队伍室外赛很强，赛前把瑞士、瑞典队视为头号强敌，但开赛后发现，南非队除了占据地主优势，也展现强烈夺冠企图心与气势，因此本届赛事中，景美师大联队在各项目的冠亚军战多次遭遇瑞士、南非队。
 <br/>
 <br/>
 她表示，景美师大联队室外赛经验丰富，选手大多有国际赛经验，赛前训练也非常扎实。由于赛前就知道南非土质较硬，对台湾选手来说适合快攻，比较不会虚耗体力，这项策略也在赛事中发挥作用。
 <br/>
 <br/>
 顾秀华说，本届比赛景美女中共3人出赛，包括就读高三的董涵、高二的施芷芯与高一的邱思瑜。施芷芯和邱思瑜都是首次参加国际赛，但两人表现非常稳定，平常训练的扎实度都有展现出来，老师和学姐对她们都非常肯定。
 <br/>
 <br/>
 本届赛事联队夺牌项目包括：U23－560公斤混合组公开赛、U23－500公斤女子组公开赛、540公斤女子组公开赛、500公斤女子组锦标赛、U23－500公斤女子组锦标赛、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_540公斤女子组锦标赛夺金.html" target="_blank">
  540公斤女子组锦标赛夺金
 </a>
 ；U23－560公斤混合组锦标赛获得铜牌，总计夺下6金1铜。
 <br/>
 <br/>
 混合组比赛由景美、师大与国立体育大学、桃园市寿山高中联队出赛，女子组则仅由景美师大联队参赛。
 <br/>
 <br/>
 世界杯室外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拔河锦标赛.html" target="_blank">
  拔河锦标赛
 </a>
 两年举办一次。景美师大联队2010年于南非举办的世界杯室外拔河锦标赛中，夺下亚洲队伍首面世界杯室外拔河锦标赛金牌。
 <br/>
 <br/>
 2010年参赛以来，景美师大联队囊括公开赛（女子500公斤、540公斤）9金1银1铜、锦标赛（女子500公斤、540公斤）9金1银1铜，合计18金2银2铜佳绩。
