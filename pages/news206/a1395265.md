### 世界最大单一屋顶剧院！卫武营中心启用
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】历时15年打造的卫武营国家艺术文化中心，今天(13日)在台湾高雄正式开幕，总统蔡英文、总统府秘书长陈菊、文化部长郑丽君以及文化界人士，都应邀出席启用典礼，共同见证全台最大、美仑美奂的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际级艺术场馆.html" target="_blank">
  国际级艺术场馆
 </a>
 诞生。
 <br/>
 <br/>
 总统蔡英文：〝从规划到完工，卫武营的兴建一共经历了三位总统，长达15年的时间，这个场馆的落成，不仅代表了当年大家所期待的空间解严，更代表了我们对于文化平权的努力。〞
 <br/>
 <br/>
 卫武营启用典礼音乐会，在台湾唯一葡萄园式音乐厅举行，由艺术总监、知名指挥家简文彬亲自上阵指挥，演奏作曲家萧泰然〝来自福尔摩沙的天使〞、旅美作曲家黄思瑜〝客家薪传〞、知名音乐家王乙聿〝簧〞3首乐目，赢得现场观众如雷掌声。
 <br/>
 <br/>
 总统蔡英文：〝我要邀请所有的高雄市民，全台湾的民众，全亚洲、全世界的朋友一起走进卫武营，一起全新的感受这一座亚洲最大的表演场馆。〞
 <br/>
 <br/>
 蔡总统指出，高雄捷运橘线将驳二艺术特区、海洋文化及流行音乐中心、高雄市立文化中心、卫武营国家艺术文化中心、大东文化艺术中心连成一线，未来高雄会是南部的文化艺术重镇。
 <br/>
 <br/>
 新唐人亚太电视林宏观台湾高雄报导
</div>
