### 花博葫芦墩公园 四大地景新亮点
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】台中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花博.html" target="_blank">
  花博
 </a>
 三大园区之一〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_葫芦墩公园.html" target="_blank">
  葫芦墩公园
 </a>
 〞，因为有软埤仔溪贯穿其中，是花博唯一有河川的展区，市府除了打造
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地景艺术.html" target="_blank">
  地景艺术
 </a>
 ，也整治河川，希望提供民众一个干净的休憩场所。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花博.html" target="_blank">
  花博
 </a>
 三大园区之一的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_葫芦墩公园.html" target="_blank">
  葫芦墩公园
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_景观工程.html" target="_blank">
  景观工程
 </a>
 即将完工。艺术家以蝴蝶翅膀做为灵感，包覆桥体，呈现生命蜕变的喜悦；浮在水面的〝浮花映月〞，表现出植物的千姿百态；还有这个将花的视觉放大，以编织手法打造一座花朵网床，藉此拉近与民众的距离。
 <br/>
 <br/>
 地景建筑师 陈宣诚：〝蝶舞风姿，当风在吹的时候，蝶片其实是有声音，所以这些声音才是最重要的在我们生活周遭的声音。〞
 <br/>
 <br/>
 葫芦墩公园除了花卉外，周围种植大量的本土植栽和香草，其中大部分更是台中特有的大安水蓑衣，原本濒临绝种，都在这里进行复育。
 <br/>
 <br/>
 台中市水利局长 周廷彰：〝利用这次花博的工程，我们把附近的家庭污水搜集起来，送到我们的净水场，把这些净水场的污水处理完之后，我们再把它放流到我们河川，所以说，整个水质情况大幅提升。〞
 <br/>
 <br/>
 从原本水质差、老旧的水岸公园，摇身一变，成为净水绿地。加上这座园区免收门票，不只是为了花博而打造的临时花园，更要成为永续，与万物共存的休憩空间。
 <br/>
 <br/>
 新唐人亚太电视邱添喜、詹咏茹台湾台中报导
</div>
