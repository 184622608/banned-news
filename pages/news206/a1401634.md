### 国军官兵集团婚礼  70对新人搭机步入红毯
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】12月1日，台湾空军司令部在新竹空军基地为官兵举办
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集体婚礼.html" target="_blank">
  集体婚礼
 </a>
 。70对佳偶搭乘C-130运输机进场，由空军司令张哲平上将为新人证婚，现场喜气洋洋。
 <br/>
 <br/>
 空军司令部集团婚礼新人：〝我们结婚了。〞
 <br/>
 <br/>
 高雅美丽的新娘，挽着帅气挺拔的新郎，双双依序步入礼堂，接受军眷与亲朋好友的祝福。空军司令张哲平上将为70对新人证婚，特别以〝选妳所爱，爱你所选〞与〝彼此宽容，相互扶持〞期勉新人。
 <br/>
 <br/>
 空军司令 张哲平：〝今天我们这70对的新人，新郎英俊挺拔、气宇轩昂，新娘气质高雅、娇媚婉约，此时此刻，这是诸位人生中最美好的一天，让我们一起共同记忆这永恒的画面。〞
 <br/>
 <br/>
 为迎接
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_官兵集团婚礼.html" target="_blank">
  官兵集团婚礼
 </a>
 ，新竹空军基地特地将两架C-130运输机，装饰的喜气洋洋做为〝礼机〞，其中一架编号1314象征〝一生一世〞，勉励新人彼此守护、牵手共度未来的每一天。
 <br/>
 <br/>
 空军司令部集团婚礼新郎 谢孟浩：〝因为我刚好本身是新竹第二联队的，能在自己的家在这个二联队，举办那个集团结婚，然后在这边参加，我觉得是满特别、满具有意义的事情。〞
 <br/>
 <br/>
 空军司令部集团婚礼新娘 余小姐：〝觉得很特别然后很开心，以后会是一个可以纪念的一个回忆。〞
 <br/>
 <br/>
 空军司令部集团婚礼新郎 甘先生：〝感谢我们司令部的像司令还有各级的长官，可以办这个活动，让我们所有的新人就是都一个美好的回忆。〞
 <br/>
 <br/>
 70对佳偶一起步入红毯，每一对都有感人的爱情故事，很特别的一对是中日联姻，新郎倌是仪队上士班长郑智祥，他在忠烈祠担任督哨官时，认识时为游客的日籍安藤光（Hikara Ando）小姐，虽然相隔两地、语言不通，断断续续连络两年，决定携手共度一生。
 <br/>
 <br/>
 空军司令部集团婚礼新娘 安藤光：〝大家好我是安藤光，我来自日本大阪，欢迎大家可以来日本玩。〞
 <br/>
 <br/>
 空军司令部集团婚礼新郎 郑智祥：〝因为我现在也是在学习日文，然后就中文、日文还有一点英文，再加上翻译软体在这样子的沟通。〞
 <br/>
 <br/>
 婚礼场地在空军第二联队二号棚厂举行，会场设计活泼、营造喜庆气氛，舞台两侧停置两部〝幻象彩绘机〞供佳偶和眷属合影拍照，留下浪漫美好回忆。
 <br/>
 <br/>
 新唐人亚太电视柯国栋、林秋霞、彭正干台湾新竹采访报导
</div>
