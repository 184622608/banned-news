### 快讯：柯文哲险胜连任台北市长 已发表胜选感言（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾九合一大选.html" target="_blank">
  台湾九合一大选
 </a>
 已进入尾声。台北市长的选票于25日凌晨全部开完。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 经过惊险鏖战，最终险胜竞争对手丁守中，成功连任台北市长。他随即发表了胜选感言，感谢台北市民对自己的支持。
 <br/>
 <br/>
 在这一次的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾九合一大选.html" target="_blank">
  台湾九合一大选
 </a>
 的选战过程中，最具悬念的莫过于台北市长的竞选。在全台瞩目的6都选战中，除台北市外的其余5市的市长人选陆续在24日底定后，台北市市长的选战仍持续呈现拉锯战的胶着情形。
 <br/>
 <br/>
 直至25日凌晨，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 最终以3254票的差距胜选。至此，台北市长一职的选战经历10个小时的马拉松式跨夜开票才宣告结束。
 <br/>
 <br/>
 据台媒报导，在这次竞选台北市长的5个候选人中，寻求连任的现任台北市长柯文哲和他的竞争对手国民党候选人丁守中两人的得票差距一直十分接近，持续出现相互交替上升的拉锯战式的情形。
 <br/>
 <br/>
 直到24日晚间12时，柯、丁二人仍未决出胜负。期间，两人的得票数目一度仅相差8票，让开票现场的观战者的情绪激动又紧张。
 <br/>
 <br/>
 最后，柯文哲于25日凌晨，以58万820票，险胜丁守中的57万7566票，两人的选票差距为3254票。
 <br/>
 <br/>
 这次参与竞选台北市长的候选人，除了柯文哲和丁守中，还有民进党选人姚文智、国民党候选人李锡锟以及独立候选人吴蕚洋。只不过后边3位候选人的得票数量远远落后于柯、丁两位对手，已提前宣布败选。
 <br/>
 <br/>
 除台北市外，6都中的另外5都市长的人选分别为：新北市长国民党籍候选人侯友宜；高雄市市国民党籍候选人韩国瑜；台中市长国民党籍候选人卢秀燕；台南市长民进党籍候选人黄伟哲、桃园市长民进党籍候选人郑文灿。
