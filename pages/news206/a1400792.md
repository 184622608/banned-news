### 柯文哲3254票险胜 丁守中提选举无效之诉
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 ，台北市长经过长达10小时的开票后，现任市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 以3254票的差距险胜国民党籍候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 。丁守中25日到法院声请查封票匦重新计票，但今(26日)他的律师与幕僚到台北地院撤回声请，并称将依选罢免法，提选举无效之诉。
 <br/>
 <br/>
 台北市长选举，台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 以58万820票胜选连任，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 则以57万7566票居次落选，两人差距仅3254票。
 <br/>
 <br/>
 丁守中质疑台北市一边开票、一边投票，过程中存在重大违法和太多疑点，决定提选举无效诉讼并提出验票要求。256日凌晨3时15分左右，在律师陪同下到台北地方法院递状声请查封台北市全部投票所选举人名册及选举票，并要求重新计票。
 <br/>
 <br/>
 台北地方法院裁定，丁守中应在一日内缴纳428万2929元保证金，丁也刚开记者会指出，将在26日上午补缴保证金；依相关规定，若重新计票后未改变当选或落选结果，法院将不发还保证金。意味丁守中428万元保证金将付诸流水。
 <br/>
 <br/>
 今(26日)天，丁守中的律师与幕僚到北院撤回重新计票的声请，并称因为进入验票、计票程序旷日废时，而且最近这几天接到很多人对于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中选会.html" target="_blank">
  中选会
 </a>
 违法乱纪、乱无章法的事证及证据资料，接下来会直接按照选罢法第118条规定，提起选举无效之诉。
 <br/>
 <br/>
 对此，中选会副主委陈朝建表示，〝我们会深切检讨〞，并尊重丁守中提起选举无效的决定。
 <br/>
 <br/>
