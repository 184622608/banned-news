### 屡扰神韵 统促党要员承认收钱 被警驱散
------------------------

<div class="wysiwyg">
 【新唐人2018年09月27日讯】美国神韵交响乐团到台湾十个城市演出，获得社会政要名流高度赞赏，25日在嘉义市演出前，统促党主要成员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张孟崇.html" target="_blank">
  张孟崇
 </a>
 率众辱骂，被警方依法驱离。张孟崇近年多次干扰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神韵演出.html" target="_blank">
  神韵演出
 </a>
 ，2016年他和一名男子分别向蒐证人员承认，收受金钱。立委谴责，并提醒国安单位，追查干扰背后的黑手。
 <br/>
 <br/>
 嘉义市警察局第二分局指挥官：〝第二次举牌警告，停止违法集会，立即解散。〞
 <br/>
 <br/>
 警方连续两次警告违法，亲共团体25日在嘉义市演艺厅前，以高声喇叭谩骂侮辱，干扰美国神韵交响乐团的演出，闹场十多分钟，被警方依法驱离后，搭乘同一部游览车离去。
 <br/>
 <br/>
 立法院国民党团书记长 曾铭宗（2018.9.21）：〝不可以去干扰艺术团体的表演，这个是最基本的素养。〞
 <br/>
 <br/>
 立法院民进党团总召 柯建铭（2018.9.22）：〝谴责在外面骚扰(神韵)的团体，这背后有中共的北京的力量，是毋庸置疑。〞
 <br/>
 <br/>
 神韵19日在台北演出，被朱纪玮率亲共团体闹场后，国民党市议员说丢脸丢到国际了。蓝绿政要，也要求政府保护神韵不受干扰。
 <br/>
 <br/>
 而在南部，多次带头干扰的男子，叫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张孟崇.html" target="_blank">
  张孟崇
 </a>
 ，绰号老K，统促党中央委员，是地下电台主持人，脸书上许多和张安乐合照、与共产党五星旗的合照。2016年10月，张孟崇干扰神韵的现场，向蒐证员坦承收钱。
 <br/>
 <br/>
 蒐证人员(变声处理) VS. 张孟崇(2016.10.1 神韵嘉义场外)：〝（你叫什么名字？）张孟崇，中华
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_统一促进党.html" target="_blank">
  统一促进党
 </a>
 。（你哪里来的？）嘉义市（有人说他从台中来。）这样喔。（一个三千吗？）六千啦！（真的喔，你领六千喔，我知道了，谢谢你。）〞
 <br/>
 <br/>
 张孟崇在现场诋毁神韵、攻击法轮功的言论，与中共的攻击同出一辙。紧密配合张的女子，叫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪文婷.html" target="_blank">
  洪文婷
 </a>
 ，是统促党罗俊党部秘书，拿着手机拍摄。跟张来干扰的民众，大多是嘉义的高龄民众，妇女居多，有些人向蒐证人员说，〝她不知道要来干什么，人家叫就来了！〞而在2016年，一名配合干扰的男子透露，对价三千元。
 <br/>
 <br/>
 蒐证人员(变声处理) VS. 参与男子(2016.10.1 神韵嘉义场外)：〝（那你拿多少？）我三千吧。我从台中来的。（你从台中来喔？一人三千？）对啊！（哇，这么伤本喔）〞
 <br/>
 <br/>
 立院外交国防委员会委员 王定宇 (2018.9.21)：〝(有些人如果)执行中共骚乱台湾的任务，伪装成民主陈情抗议的活动，那如果事证明确的话，这些行为是比(共谍)情蒐行为更严重。〞
 <br/>
 <br/>
 立委王定宇表示，政府应追查，干扰背后，如果有中共相关单位授意、资金金援的交流，若事证明确，国安单位应该依法严办。
 <br/>
 <br/>
 新唐人亚太电视综合报导
</div>
