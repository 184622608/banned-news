### 连四年破千万观光客 美商会论坛看好台湾
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】中共紧缩陆客来台，但台湾连续第四年达成千万观光人次。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北美国商会.html" target="_blank">
  台北美国商会
 </a>
 周一(3日)举办数位观光趋势
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_论坛.html" target="_blank">
  论坛
 </a>
 ，看好台湾前景，邀请业者与观光局对话，结合数位化、专业与个性化等跨领域的合作策略，扩大国际客源。
 <br/>
 <br/>
 观光局2日宣布，出现今年第1千万名来台
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_观光客.html" target="_blank">
  观光客
 </a>
 ，连续4年站稳千万人次。观光局目标2020年，达到1140万人次。美国商会看好台湾。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北美国商会.html" target="_blank">
  台北美国商会
 </a>
 执行长 傅威廉 (William Foreman)：〝最后一件事是要有迫切感，台湾必须真正向上升级，建立在台湾已有的了不起成就上。台湾是美丽的国家，有丰富的文化，有很棒的故事。〞
 <br/>
 <br/>
 观光局表示，台湾的数字乐观，其实面临挑战，要追求质而不是量，因为世界各国竞争投入观光产业。不过台湾许多优势，包括航线中转、密集多样的观光资源，2小时交通就能看到千变的景观。
 <br/>
 <br/>
 交通部观光局副局长 陈淑慧：〝云端这件事情，是一个可以取代我们口碑的这件事。要抓住自己的在地特色，可是还是要和国际是连结的。我们不是自己说，我有国际就有国际。我想不管在语言的接待环境、跟国际化环境这件事，都应该是努力的目标。〞
 <br/>
 <br/>
 旅游平台业者表示，台湾的自由行观光，前景光明，不过，要便利国际认识。
 <br/>
 <br/>
 数位旅游平台业者 大中华区总监 Spencer Low：〝管理正确的管道，提供多语种的资讯，是非常重要的。〞
 <br/>
 <br/>
 数位旅游平台业者 总监 Peter L. Allen：〝台湾在一个小指南针中，几乎有一切东西，你都可以到达。你可以在台北购物，可以访太鲁阁峡谷，所有都很接近，而且非常容易到达、非常友善。〞
 <br/>
 <br/>
 观光局表示，今年千万人次，新南向18国占了约两成五，显现多元客源市场开发，有了成效。
 <br/>
 <br/>
 台湾旅宿生活产业协会副理事长 杨欣欣：〝虽然前阵子陆客的减少，但我觉得像从观光局推广的所谓南向政策，其实我们有感受到。希望如果再有第二、第三次的，台湾深度旅行，我会比较希望，接下来的入境的客人，他不要是团体的方式，他可以用自由行的方式。〞
 <br/>
 <br/>
 <br/>
 观光局说，规划了大量的〝借问站〞，从车站到小镇，许多小店加入，提供外文解答服务，让人人都是观光大使。
 <br/>
 <br/>
 新唐人亚太电视陈进交、张东旭台湾台北报导。
</div>
