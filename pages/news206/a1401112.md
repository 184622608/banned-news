### 难忍受肮脏胜选 北市长候选人丁守中重提司法验票
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 及10案公投11月24日投票，开票作业直至25日凌晨3时2分才结束，台北市长选举，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 以3254票微幅差距险胜，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 向法院声请验票，但今(28日)说，近期选民提供许多违规情资，将重新声请司法验票，再依结果提当选或选举无效诉讼，且不接受党内资助。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 和律师团上午在国民党中央党部记者室召开记者会，说明近期验票及提出选举无效讼诉状况。他在记者会拿出缴交保证金的支票，结束后也将和律师团赴法院缴交保证金。
 <br/>
 <br/>
 丁守中表示，他接受干净选举的失败，但无法忍受台湾民主选举还有肮脏胜选，重新验票不是为他个人选举输赢，而是捍卫台湾民主价值。
 <br/>
 <br/>
 丁守中说，选举投票结束后，接获许多选民陈情，像中正区有里长反映在投开票所发现19张废票，有民众发现士林区某国小的投票所是边投票和开票，选举当天，有选民听到现场有人叫喊〝弃姚保柯〞等言论。
 <br/>
 <br/>
 丁守中表示，本届台北市选票开票时间缓慢，像是市议员选票已统计完成，但市长票数却迟未结束，台北市选委会有行政瑕疵。
 <br/>
 <br/>
 他也发现民进党市长候选人姚文智的票数达到19万多票后，直到开票结束前，票数仅缓慢增加，明显有〝弃保效应〞。
 <br/>
 <br/>
 丁守中说，他不会接受党内和其他同志资助，保证金全自筹，相关证据已整备完成，会在法庭上公布，且提供证据选民也愿意出庭作证，因此，会重新声请由司法验票，再视验票结果，提出当选或选举无效诉讼。
 <br/>
 <br/>
 律师团提及，原决定要先验票后再提选举诉讼，但评估后，研判不会有大规模舞弊状况，经考量后先撤回，但这2天接获太多选举违规状况，重新和丁守中讨论后，决定再向法院声请重新验票，此外，也会要求扣押投开票所电磁纪录、投票纪录单等其他选举证据。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 落幕，丁守中以3254票落败，原已向法院声请重新验票，需缴交保证金新台币428万元。不过，丁守中26日宣布撤回，改提选举无效之诉，但国民党内部则认为重新验票与提选举无效之诉应同时进行，党主席吴敦义昨天也允诺可分摊一半费用。
 <br/>
 <br/>
