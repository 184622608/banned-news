### 贸易战台商回流 科技部：20家业者申请进驻
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】美中贸易战升温，不少台商希望返台设厂，行政院统筹各部会，协助厂商回流。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技部.html" target="_blank">
  科技部
 </a>
 负责六大科技产业，统筹单一窗口，进驻科学园区，现在有3家厂商通过审查，还有20多家厂商申请。
 <br/>
 <br/>
 美中贸易战，台商加速撤离中国，转向回台布局，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技部.html" target="_blank">
  科技部
 </a>
 透露，有3家业者准备进驻三大科学园区，还有20多家业者，表达意愿，产业涵盖：通信、精密机械、光电、生技产业。
 <br/>
 <br/>
 科技部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈良基.html" target="_blank">
  陈良基
 </a>
 ：〝让厂商，希望一个月内能够完成它所有进驻，一个月内能够把土地相关的这些规划能够做完，让他们顺利的进驻，所以目前我们也已经有三家厂商，已经获准进来，还有将近20家厂商在联系中。〞
 <br/>
 <br/>
 科技部盘点，国内三大科技园区，有140公顷土地可供使用，中科二林园区200公顷未来一到两年可供释出。科技部上推出单一窗口，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一站式.html" target="_blank">
  一站式
 </a>
 服务协助科技产业回台。不过立委爆料，新北市工业用土地，现在炒到每坪80万元，缺地问题，恐怕比想像还要严重！
 <br/>
 <br/>
 国民党立委 柯志恩：〝桃园、大园、新北、五股、土城，近四年来是(价格涨幅)超过50%，然后我刚刚特别提出来的，新北刚刚发布的，一坪80万。非常多的投资客已经去卡位了。〞
 <br/>
 <br/>
 民进党立委 陈亭妃：〝为什么院长要说 要解决五缺，就确实有五缺的问题嘛。〞
 <br/>
 <br/>
 经济部工业局副局长 游振伟：〝有没有具体的计画，它的投资金额、机器设备，有没有人才需求等等这一些，一个真的投资计画，还是说假借一个回台的方式来做炒作土地，我们会用个案审慎的去做一个了解跟评估。 〞
 <br/>
 <br/>
 经济部喊话，新设工业园区，保留20%产业用地，只租不售，也要减少炒作问题。财政部表示，因应
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台商回流.html" target="_blank">
  台商回流
 </a>
 ，个人股东股利所得税负从49.68%降为42.4%，有助吸引投资台湾意愿，后续也将端出资金汇回特别租税措施。
 <br/>
 <br/>
 财政部次长 吴自心：〝课税新制实施了以后，它未来营运以后，它的整个的所得税税负，会比以往要做大幅的降低。〞
 <br/>
 <br/>
 厂商供应链搬迁已经成为趋势，立委高度关心，政府如何端出办法，让台商真的有感，府院各部会动起来，希望力挺台商回流。
 <br/>
