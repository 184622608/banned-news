### 中共渗透选举 姚文智呼吁台湾人民看清楚
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】
 <b>
  新闻周刊
 </b>
 （654）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长候选人.html" target="_blank">
  台北市长候选人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚文智.html" target="_blank">
  姚文智
 </a>
 ，连续两天宣传星期天的游行活动，强调当天风雨无阻，希望有3万人一同参加。对于外部势力干预选举，姚文智再次提醒，台湾民众必须看清楚，尤其在这一场选战中，中共无所不用其极的渗透模式。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长候选人.html" target="_blank">
  台北市长候选人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚文智.html" target="_blank">
  姚文智
 </a>
 ，为〝手牵手护台北〞活动造势，号召民众参与。被问到外部势力影响选战，他说，过去中共以飞弹威胁台湾，现在则学会利用民主，扶植在地团体组织与媒体，做为中共代理人。
 <br/>
 <br/>
 民进党台北市长候选人 姚文智：〝如果没有办法看清楚，很可能就算你在各行各业再努力，就算我们的民主再自由，我们都可能一夕之间，变成85度C，变成要被辞职的马云，变成要被缴税的范冰冰。〞
 <br/>
 <br/>
 呼吁看清楚中共渗透，姚文智喊出守卫国家首都。至于高雄选情紧绷，民进党动用重兵把守，传出姚文智认为，党中央挹注台北市的资源不够，因此向蔡英文讨救兵。
 <br/>
 <br/>
 民进党台北市长候选人 姚文智：〝我从来没说过我们党的资源给的不够，事实上党有整体的策略安排，17号大家关心高雄，18号大家关心台北。蔡总统本来过去就很帮忙，你难道没看到马英九说，支持姚文智就是支持蔡英文吗？连马英九都感受到了。〞
 <br/>
 <br/>
 选战倒数一星期，姚文智18号要动员游行，21号将再次与蔡英文同台造势，选前之夜，邀请谢长廷站台，希望最后关头，催出更多选票。
 <br/>
 <br/>
 新唐人亚太电视林嘉韦、刘姿吟台湾台北报导
</div>
