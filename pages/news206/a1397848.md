### 美控联电窃密给晋华 关键技术管理受瞩
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】美国针对智慧财产权遭窃取，起诉包括中国大陆福建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晋华.html" target="_blank">
  晋华
 </a>
 ，以及台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联电.html" target="_blank">
  联电
 </a>
 ，还有三名台湾个人，涉及窃取美光商业机密，对此，联电表示将严正以对，而经济部方面高度重视，强调产业界必须重视自主技术，以及合法技术授权移转，以免涉及红线，将对产业造成冲击。美方则感谢台湾法务部协助调查。
 <br/>
 <br/>
 美国司法部长塞申斯：〝从（美国企业）美光窃取商业机密，价值87.5亿美元。〞
 <br/>
 <br/>
 根据美方起诉书，被窃取的美光商业机密，价达高达87.5亿美元，约新台币2690亿元。美方并以民事诉讼，要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联电.html" target="_blank">
  联电
 </a>
 、福建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晋华.html" target="_blank">
  晋华
 </a>
 ，禁止使用被窃机密出口产品到美国。
 <br/>
 <br/>
 和硕董事长童子贤：〝我确实没有详细的关注，所以没有办法评论。那我想智慧财产权，本来就文明社会所共同关注的。 〞
 <br/>
 <br/>
 消息一出震撼产业界，起诉书称，原台湾美光中科厂的陈姓总经理，2015年辞职加入联电，并招募原美光子公司的员工跳槽，包括何姓和王姓男子，美国检方称，何、王两人都窃取了美光DRAM设计和制造相关的重要商业机密，目的是，转移给成立才两年的福建晋华，让中国企业迅速取得DRAM技术。
 <br/>
 <br/>
 经济部长沈荣津：〝关键技术对每一个企业，一定要掌握，那要掌握的话，我们一定要有自主的关键技术，但是最重要这关键技术，我们一定要做好‘管理’，但是最重要呢，也不要透过不正常的管道。〞
 <br/>
 <br/>
 美光去年就起诉联华电子和晋华，联电则在反指控美光，侵害了存储器相关专利。联电上午发布重大讯息，表示：对于联电可能违反法律的指控，都将严正以对，并拟竭力回应该等指控。
 <br/>
 <br/>
 经济部长沈荣津：〝因为这些你如果不管理好，你去踩到红线，这个会带来困扰，这部分我一直希望，我们也会透过管道，来跟我们产业界来交流来呼吁。〞
 <br/>
 <br/>
 双方法律战越演越烈，但就怕台湾业者轻忽美国维护智财决心，损害产业利益。联电上午股价，下跌1.26%，暂收11.80元。
 <br/>
 <br/>
 新唐人亚太电视胡宗翰、沈唯同台湾台北报导
</div>
