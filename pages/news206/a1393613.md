### 梅北竹乐团悠扬乐音 快闪高铁嘉义站
------------------------

<div class="wysiwyg">
 【新唐人2018年10月01日讯】台湾嘉义县
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅北国小.html" target="_blank">
  梅北国小
 </a>
 竹乐团的小朋友们，9月29日一早到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高铁嘉义站.html" target="_blank">
  高铁嘉义站
 </a>
 快闪演出，现场演奏三首著名的台湾名谣组曲，搭配竹乐器表演，吸引不少旅客驻足观赏、拍照。
 <br/>
 清脆通透的悠扬乐音，突然从高铁车站的大厅传来，15位小朋友演奏大家耳熟能详的《桃花过渡》、《青蚵仔嫂》等三首台湾名谣组曲，让旅客惊喜不已！纷纷拿起手机拍照录影。
 <br/>
 <br/>
 民众：〝它很轻快，然后是古早的民谣的(旋律)，就觉得心情很好。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅北国小.html" target="_blank">
  梅北国小
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_学童.html" target="_blank">
  学童
 </a>
 黄郁蓁：〝心里其实很紧张，可是表演完后很开心。〞
 <br/>
 <br/>
 梅北国小学童 郑和乐：〝非常的高兴，因为可以顺利完成表演，然后把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_音乐.html" target="_blank">
  音乐
 </a>
 传达给大家。〞
 <br/>
 <br/>
 梅北国小学童 陈睿恩：〝感觉就是很与众不同，跟我们表演过的有点不一样，(哪里不同？)是人很多，而且是随处走过来的。〞
 <br/>
 <br/>
 穿着中国服的梅北国小学童，使用丰富多样的竹制打击乐器和传统的竹管乐器，演奏出来的声音清脆优美，搭配上动人的旋律，让人惊艳！
 <br/>
 <br/>
 台湾竹乐团团长 罗正楎：〝推广这个竹乐器已经满多年，那我们希望就是可以把它深耕在校园，因此呢我就从台北回到我的故乡梅山这边来，指导我们梅北国小的这群小朋友，来演练这些竹乐器；那今天刚好有非常难得的机会，可以在高铁这边来快闪活动。〞
 <br/>
 <br/>
 透过快闪方式演出，除了展现梅北国小学童学习竹乐的成果，也让大家感受到音乐的美好以及竹艺文化的传承。
 <br/>
 <br/>
 新唐人亚太电视林裕浦、方雅娴台湾嘉义采访报导
</div>
