### 普悠玛翻车抢救空拍 台铁27年最严重事故
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】早安新唐人带您关心，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 普悠玛列车昨天下午四点，发生重大出轨翻覆事故，载满366人的列车，8节车箱全部出轨。据最新统计，死亡人数为18人，另外有178人受伤，这也是台铁27年来，最严重事故。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁.html" target="_blank">
  台铁
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛号.html" target="_blank">
  普悠玛号
 </a>
 21日下午4点50分，在苏澳新马车站发生翻覆事故，从空拍画面来看，原本全长有8个车厢的列车，断裂成〝W〞型，当中有5节是整个90度 倾倒翻覆。
 <br/>
 <br/>
 宜兰县代理县长 陈金德：〝8节车厢第一时间确认，整个都救出，包括司机也救出来，不知道有没有人会压在车厢底下，因为整个车有两节翻了，要等遗体搬走大型机具吊进来，再去确认，连夜会扩大搜索。〞
 <br/>
 <br/>
 现场15人当场死亡，3人送医后死亡，截至晚间12点，死亡人数累积到18人，遗体运送完毕，确认车厢内已无乘客，警消出动大型吊车进驻现场，展开车厢吊挂作业，吊挂死伤最惨重的第6、7、8节车厢，寻找可能的生还者。普悠玛号酿成重大死伤，交通部长吴宏谋表示，将会全面检查普悠玛号列车，但顾及旅客交通旅运需求，暂时不会停驶。
 <br/>
 <br/>
 交通部长 吴宏谋：〝普悠玛号以前行车纪录都还算可以，这一次发生这么重大的交通事故，我们一定会全面体检，包括行车环境还有车辆。〞
 <br/>
 <br/>
 连夜作业，要将把翻覆的车厢都吊离，修复铁轨，也希望尽快让东部干线，恢复通行。
 <br/>
 <br/>
 新唐人亚太电视宜兰综合报导
