### 出席世华妇女年会 总统盼菁英回台投资
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界华人工商妇女企管协会.html" target="_blank">
  世界华人工商妇女企管协会
 </a>
 ，是唯一一个从台湾发展到全球的国际工商妇女社团组织。今天，该组织在圆山饭店举办年会，中华民国总统蔡英文亲自出席，并向在场菁英喊话，把握机会回台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资.html" target="_blank">
  投资
 </a>
 。
 <br/>
 <br/>
 遍布世界各地的台湾女性企业家齐聚一堂，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界华人工商妇女企管协会.html" target="_blank">
  世界华人工商妇女企管协会
 </a>
 的年会正式开幕，而身为台湾第一位女性总统的蔡英文，在场更是满面笑容。
 <br/>
 <br/>
 中华民国总统 蔡英文：〝今天这样这么美丽、这么热闹、这么热情。〞[欢呼声]
 <br/>
 <br/>
 全球侨商以及台商妇女重要的沟通平台，世华已经迈入了第25年，目前已经有65个分会，遍布全球26个国家，不仅协助台湾深化与全球的经贸交流，总统蔡英文更希望这些菁英能够，回台
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资.html" target="_blank">
  投资
 </a>
 。
 <br/>
 <br/>
 中华民国总统 蔡英文：〝国庆的前一天，国际货币基金也刚刚发布，调高了今年跟明年两年台湾经济成长率的预期、预估值，请各位把握机会，能够在这个时候回来台湾投资，让台湾更好。〞
 <br/>
 <br/>
 世界华人工商妇女企管协会总会长 黎淑瑛：〝我们现在也都配合侨委会，在做这个帮助青年创业，还有刚刚您说的这个回国投资，我们都有在评估，我相信只要是项目对了，我想姊妹们都会回来的。〞
 <br/>
 <br/>
 总统蔡英文强调，政府会努力改善台湾的投资环境，协助企业解决土地、水、电、人才跟劳工的课题，同时加速5+2产业创新计划，来协助企业提供研发能量，扩大台湾的竞争优势。
 <br/>
 <br/>
 新唐人亚太电视王冠霖、赵庭誉台湾台北采访报导
</div>
