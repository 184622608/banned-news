### 台积电运动会员工热情不减 送近5亿奖金
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台积电.html" target="_blank">
  台积电
 </a>
 今天(20日) 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新竹.html" target="_blank">
  新竹
 </a>
 县体育场举行运动会，这是创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张忠谋.html" target="_blank">
  张忠谋
 </a>
 退休后，首次由董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘德音.html" target="_blank">
  刘德音
 </a>
 主持，少了张忠谋的参与，员工热情依然不减，刘德音也大方送礼，预计发给员工将近5亿元奖金。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台积电.html" target="_blank">
  台积电
 </a>
 员工：〝董事长我爱你，CC总裁我爱你。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晶圆.html" target="_blank">
  晶圆
 </a>
 龙头台积电一年一度运动会，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘德音.html" target="_blank">
  刘德音
 </a>
 率领高阶主管团队绕场，受到员工热烈欢迎。尽管是30年来创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张忠谋.html" target="_blank">
  张忠谋
 </a>
 第一次缺席，员工们依然热情不减，准备各类精彩的表演，热力放送活力与创新精神。
 <br/>
 <br/>
 台积电董事长 刘德音：〝非常感动，台积一向就是非常团结，丝毫没有改变，反而这个奋斗心更强，我们挑战也更大，需要大家努力。〞
 <br/>
 <br/>
 感动于员工的努力，刘德音大方送礼，发给台湾3万9千名员工，每人1万2千元奖金，共计发出4.68亿元，创下历年最高金额。
 <br/>
 <br/>
 台积电董事长 刘德音：〝为了感谢大家去年一起的努力与贡献，，我和 CC(总裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魏哲家.html" target="_blank">
  魏哲家
 </a>
 )要向辛勤奉献的台积同仁宣布一个小礼物，每一个人发给新台币1万2千元的红包。〞
 <br/>
 <br/>
 第一次主持运动会，刘德音带着高阶主管领跑1200公尺，还和同仁一起大跳暖身操，展现高度的亲和力。刘德音也透露，台积电今年营收将突破台币1兆元，七奈米制程也开始量产，迈入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_半导体.html" target="_blank">
  半导体
 </a>
 发展重要里程碑，只要明年获利表现好，张忠谋就会回来参加运动会。
