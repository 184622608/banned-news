### 自责丢掉政权 马英九一天排7场扫街辅选赎罪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马英九.html" target="_blank">
  马英九
 </a>
 卸任之后是四处辅选。今天（29）先后前往嘉义县、市，陪吴育仁和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄敏惠.html" target="_blank">
  黄敏惠
 </a>
 拜票，晚间还将前往台南市，替谢龙介站台。马英九说他在拜票过程，有太多民众跟他反应，希望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 重回执政，他很感慨当初蓝营连任失利，因此现在几乎是用赎罪的心情在辅选。
 <br/>
 <br/>
 前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马英九.html" target="_blank">
  马英九
 </a>
 ：〝我们碰到一些农民、渔民、小旅馆的业者，讲到后来都要哭了，最后的结论是，你们回来好不好？真的想起来我们会有一种什么感觉？很内疚的感觉，我们卸任的时候，接任者没有办法是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 ，所以造成这么多苦痛，所以我这样拚了命在辅选，真的是有一种赎罪的感觉，我非常自责，这点没有办法找到一个能继续做的，我当台北市长的时候，我找了郝龙斌很顺利就连任了嘛，就选上了，这一次就没有办法，所以我非常自责。〞
 <br/>
 <br/>
 前总统马英九从上午十点一路辅选到晚间七点，从嘉义县、市到台南，一天排七场扫街行程。马英九说还好有运动，所以不会太累。他对嘉义市选情的看法，是审慎乐观。而国民党另一位当红高人气的韩国瑜，上午是和党籍嘉义县长候选人吴育仁一同造势，不过并没有到旁边的嘉义市辅选，对此，国民党嘉义市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄敏惠.html" target="_blank">
  黄敏惠
 </a>
 说，因为韩国瑜行程太满，之后会规划邀请韩国瑜一同站台。
