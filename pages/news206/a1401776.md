### 韩国瑜首波小内阁 副市长叶匡时 新闻局长王浅秋
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄市长.html" target="_blank">

 </a>
 当选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 今(3日)公布首波小内阁，副市长由前交通部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叶匡时.html" target="_blank">
  叶匡时
 </a>
 接任，新年过后报到；前高雄市议员林国权接原民会主委、客委会主委是美浓客家人黄永卿、新闻局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王浅秋.html" target="_blank">
  王浅秋
 </a>
 。
 <br/>
 <br/>
 对于媒体报导〝有党政高层塞主管名单〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 表示，〝绝不可能，绝无此事，连一个都不可能，所有人都是我一个一个面谈，一定要有热情，不可能接受任何党政高层安排人选。〞
 <br/>
 <br/>
 另外，前市长陈菊团队成员是否会有留任，韩国瑜透露有可能，但最重要是要有热情，且不能有太强烈的意识型态。
 <br/>
 <br/>

 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叶匡时.html" target="_blank">
  叶匡时
 </a>
 过去曾任交通部长，在确定出任高雄市副市长，还未上任就开始拚经济。昨天在脸书上打卡说自己在香港亚洲台商香江论坛演讲，顺便为高雄招商。
 <br/>
 <br/>
 叶匡时任交通部长期间，以〝美好生活的连结者〞概念作为交通部的使命，积极推动观光，2013年国外来台观光客达800万人，联合国世界观光组织也将台湾列为2013年全球前十名成长最快速的旅游地点，花东铁路电气化也是在他任内通车。
 <br/>
 <br/>
 叶匡时曾于高雄国立中山大学企业管理学系、政治大学科技管理与智慧财产研究所任教，并于今年退休。
 <br/>
 <br/>
 叶匡时产官学经历丰富，交友广阔，为人海派，但不修边幅的个性也让他在交通部长任内引发许多争议，包括ETC电子收费、国道收费员安置等，最后因为高铁财改案未过关，基于政务官为政策负责理念，请辞交通部长以示负责。
