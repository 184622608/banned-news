### 北市长政见会 吴蕚洋放话要训练柯P当总统
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】第2场台北市长公办电视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政见发表.html" target="_blank">
  政见发表
 </a>
 会18日下午登场，无党籍候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴蕚洋.html" target="_blank">
  吴蕚洋
 </a>
 再出狂言，先是一人分饰两角唱双簧，还说台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 只要能够改正思维，他可以来训练柯文哲当总统。
 <br/>
 <br/>
 台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ：〝我相信我会越做越好，现在人家批评我都是批评以前的柯文哲，他没有办法批评现在的柯文哲，因为我会不断的改进，会修正、会越来越进步，所以我要拜托大家支持，我相信我会越做越好，会做得更好、谢谢大家。〞
 <br/>
 <br/>
 台北市第2场公办电视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政见发表.html" target="_blank">
  政见发表
 </a>
 会18日登场，台北市长柯文哲第一位发言，就提到了一直被长者抗议的重阳敬老金，柯文哲表示，有些政策短期不讨喜，但长期是对的，不该用世代冲突鼓动老人，反对北市取消重阳敬老金，而国民党台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 则是直批柯文哲，任内所提的政见都跳票，并说柯文哲的北市还债说是假象。
 <br/>
 <br/>
 国民党台北市长候选人 丁守中：〝刚才柯文哲讲，说他前面做的不好再给他四年，还能够再给他四年吗？还能够再虚耗四年吗？〞
 <br/>
 <br/>
 丁守中认为，大巨蛋案持续摆烂、招商引资成绩低落，这些都让北市竞争力持续下滑，不能再如此虚耗下去，而无党籍候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴蕚洋.html" target="_blank">
  吴蕚洋
 </a>
 则是现场一人分饰两角唱双簧，还说如果当选要培养副市长来参选总统。
 <br/>
 <br/>
 无党籍台北市长候选人 吴蕚洋：〝现在很多人要拱他当总统，为了不让他左右为难，我觉得还是请大家将选票投给我，我来训练他当总统，柯文哲先生如果你来应征副市长，我一定优先考虑。〞
 <br/>
 <br/>
 而柯文哲在受访时听到吴蕚洋要训练他选总统，忍不住哈哈大笑，并回答媒体说，他晚上在北门有活动，叫吴蕚洋去看一看好了。
 <br/>
 <br/>
 新唐人亚太电视曾奕豪、黄庭锋台湾台北报导
</div>
