### 一支触控笔 也能练就一手好字
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】您知道吗？现在用一支触控笔，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平板.html" target="_blank">
  平板
 </a>
 上也能写出一手漂亮、流畅的毛笔字。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 经销商携手淡江大学以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_书法.html" target="_blank">
  书法
 </a>
 名师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张炳煌.html" target="_blank">
  张炳煌
 </a>
 ，研发出了一款数位学习系统，不需要用到笔墨纸砚，就能练习如何写书法，带您一起了解。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平板.html" target="_blank">
  平板
 </a>
 上一笔一笔的勾勒，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_书法.html" target="_blank">
  书法
 </a>
 名家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张炳煌.html" target="_blank">
  张炳煌
 </a>
 这次拿的不是毛笔，而是一支触控笔。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 经销商携手淡江大学跟书法名师张炳煌，共同研发推出了一款数位学习系统〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_e笔书画.html" target="_blank">
  e笔书画
 </a>
 App〞，不需要笔墨纸砚，就能够写出一手漂亮的毛笔字。
 <br/>
 <br/>
 书法名家 张炳煌：〝一笔一画、一撇一捺，然后你的字也不能乱写，你一定要有一定的字型去写出，那这一些的基本条件的学习，e笔是完全可以做到。将来你要发挥的时候，随着你个人的兴趣，可能他可以转进，传统的文法的书画的发展，这是我们最大的一个目的。〞
 <br/>
 <br/>
 苹果经销商总经理 程应龙：〝最新的科技，我们来结合我们最深底蕴的一个文化，我们用最简单的方式来呈现。〞
 <br/>
 <br/>
 不仅可以调整笔画的粗细、浓淡以及提按跟锋芒，还有自动调整格线的设定，来辅助点画位置，最后你还能DIY自己的专属印章，接着将完成的作品分享给其他人。不过不只有书法跟水墨的功能，喜爱绘画的民众也能大展身手。
 <br/>
 <br/>
 画家 蔡爱珠：〝画那种实际的画先做这个练习，我有时候会画两边都画一样的，然后画完这个再画这个，这样你真正去实际做画笔的画画的话，也很容易上手。〞
 <br/>
 <br/>
 而前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马英九.html" target="_blank">
  马英九
 </a>
 也受邀出席，在平板上当场挥毫。而这款App同时支援中英日韩四种语言，也要透过科技的力量，将中华传统文化推向国际舞台。
