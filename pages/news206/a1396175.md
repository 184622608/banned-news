### 全台最大农机展 农业界口碑开幕首日人潮暴
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】全台最大国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农业.html" target="_blank">
  农业
 </a>
 机械暨资材展周末（20日)在云林登场，吸引来自国内外400多家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_厂商参展.html" target="_blank">
  厂商参展
 </a>
 ，1200多个展出摊位。开展首日，人潮大爆满，预计三天展期将涌入40万参观人次。
 <br/>
 <br/>
 耕耘机轻松往前推，连小朋友操作起来都有模有样。全台最大规模的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农机展.html" target="_blank">
  农机展
 </a>
 在虎尾高铁特定区展出三天，开展首日，人潮爆满，连外县市的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农业.html" target="_blank">
  农业
 </a>
 产销班，都特地坐游览车前来参观。
 <br/>
 <br/>
 民众彭先生：〝我们每年要看先进一点的机器，然后我们回去考虑，在班里面开会以后，再来决定要不要买。〞
 <br/>
 <br/>
 民众张先生：〝种类很多，姑且看看。〞
 <br/>
 <br/>
 民众杨先生：〝现在时间就是金钱，（机器的速度）越做越快，提升效率，农业在进步！〞
 <br/>
 <br/>
 举办13年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际农业机械暨资材展.html" target="_blank">
  国际农业机械暨资材展
 </a>
 ，已经是台湾每年的农业界盛事，年年刷新参展摊商数与商机记录，去年近400家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_厂商参展.html" target="_blank">
  厂商参展
 </a>
 ，3天创下11亿元商机。云林县长李进勇表示，智慧科技，才能有效解决农业发展遇到的瓶颈。
 <br/>
 <br/>
 云林县长李进勇：〝农业的发展遇到一些瓶颈，包括人工短缺、包括产销失衡、包括天然灾害的影响等等，我们希望在下一个阶段可以推动农业4.0，就是引进智慧科技。〞
 <br/>
 <br/>
 李进勇表示，每年的展场规模日益扩大，已经是农业界的口碑。今年三天的展期，参展摊位及参观人次都创新高，估计带来的商机也许将再创新高。
 <br/>
 <br/>
 新唐人亚太电视张佑铭、陈逸文台湾云林采访报导
 <br/>
</div>
