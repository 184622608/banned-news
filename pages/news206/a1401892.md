### 北市长空白选票少60张？北院：与议员票放一起无误
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】国民党台北市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 声请
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 ，4日进入第2天，外传有市长空白选票短少60张选票一事，台北地方法院今指出，是因空白选票与市议员部分放置在一起，故未能在第一时间发现，已确认选票数量无误。
 <br/>
 <br/>
 综合媒体报导，北院昨天验完208个投开票所票箱，再加上今天上午的102个，累计验票进度达19.8%。下午将完成文山区部分后，接续验内湖区。
 <br/>
 <br/>
 北院指出，目前以50组人力投入验票作业，每组（桌）配置法官一人、书记官一人，以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁守中.html" target="_blank">
  丁守中
 </a>
 阵营律师、台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 阵营律师、2位选务人员，共有1563个投开票所的票箱。
 <br/>
 <br/>
 而为了考量中午休息时间无庸过长，明天起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 时间，更改为上午9时至中午1时、下午1时30分至5时30分。
 <br/>
 <br/>
 关于媒体报导市长空白选票短少一事，北院表示，经再次确认后，为大安区1224投开票所其空白选票与市议员部分放置在一起，故未能在第一时间发现；目前已经确认数量无误，非媒体报导消失60张。
 <br/>
 <br/>
 柯文哲律师代表指出，法院规定不得透露验票细节，会请法院再次要求，〝不要以讹传讹〞
 <br/>
 <br/>
 法院表示，实际上各投开票重新计票的结果以法院笔录记载为准，为使法院同仁能尽速在法定期间内完成重新计票，希望大家不要再听信网路消息，或两造当事人片面陈述。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 落幕，台北市长柯文哲连任成功，丁守中以3254票之差落败；丁守中日前召开记者会，宣布重新提司法验票，并在律师陪同下，到北院声请查封票匦重新计票，缴交保证金新台币428万元。
 <br/>
 <br/>
