### 法轮大法修炼心得交流会 台湾七千学员参与
------------------------

 心得

 ，今天(25日)在台北召开，约7000名法轮功学员齐聚一堂，分享在法轮大法中的修炼体悟的心得。
 <br/>
 <br/>
 2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮大法.html" target="_blank">
  法轮大法
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_修炼心得.html" target="_blank">
  修炼心得
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交流会.html" target="_blank">
  交流会
 </a>
 ，周日在台大体育馆登场，全台法轮功学员齐聚，分享在法轮大法中的修炼体悟。
 <br/>
 <br/>
 大学时一身都是病的精神科医生周明辉，在接触法轮功后，如获至宝，身体也健康了。结婚后因为无法平衡工作与家庭，常与搞太太矛盾，通过向内找放下自己的执着心，而改善了家庭关系。
 <br/>
 <br/>
 台南法轮功学员周明辉：〝其实我能够在外面的工作，很顺利是她在家把这个家顾得很好，反而是我没有替她想一想，那这样我把自己置于别人之上的心，骄傲的心，这些都是执着，应该要把它放下，把它放下来之后，夫妻之间的争执矛盾就会少很多，那这个就是法宝。〞
 <br/>
 <br/>
 用法宝让夫妻间更加和睦，也用法理归正亲子间的教养，从小与妈妈一起在大法中修炼的张可心，与带着女儿一起修炼大法的萧伊君，都在修炼过程中历经重重考验，随着小弟子提高心性，明白如何符合真善忍做一个好人，一切辛苦都是值得的。
 <br/>
 <br/>
 南投法轮功学员萧伊君：〝有一次她就说，妈妈今天有人推我，很难过就说为什么他推你 ，他说因为该往前走了，我没有往前走，他就推我，那我就说他可能不是故意的 ，所以我们没有关系，事后她就会想想，可是上次我不小心踩到他的脚，我跟他道歉，他说我才不原谅你呢，她就很难过，我就跟我女儿说那如果是你，你会怎么做 ，他就说如果是我，我会觉得说没关系，我会原谅他，原谅别人，就是善待自己，她就这样跟我说。〞
 <br/>
 <br/>
 新竹法轮功学员张可心：〝有曾经受社会中那些手机，有些报导然后就是被吸引，没有按照真善忍这个标准去做对比的话，很难分得清什么是真正的好和真正的坏。〞
 <br/>
 <br/>
 今年台湾法轮大法心得交流会，约有7000名法轮功学员参与，其中还有600多位来自各国的法轮功学员，学员们开心的表示，能够参加心得交流会收获满满，大家比学比修，期盼在修炼上共同精进。
 <br/>
 <br/>
 新唐人亚太电视胡淑霞、张年洁台湾台北采访报导
