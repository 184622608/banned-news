### 名间千人彩绘风车 打造南投美丽亮点
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南投名间.html" target="_blank">
  南投名间
 </a>
 乡今天（13日）在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_松柏岭.html" target="_blank">
  松柏岭
 </a>
 举办2018千人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彩绘风车.html" target="_blank">
  彩绘风车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嘉年华.html" target="_blank">
  嘉年华
 </a>
 活动，小朋友们发挥创意在风车上画下各式各样的图案，完成的彩绘作品将摆放在名间乡的庙宇广场上让游客参观，为在地增添美丽亮点。
 <br/>
 <br/>
 舞蹈团带来轻快的风车舞，现场千名小朋友和家长在欢乐气氛下共同
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彩绘风车.html" target="_blank">
  彩绘风车
 </a>
 ，小朋友发挥创意巧思，在风车上画上海洋、花草、还有蝴蝶等各种图案。
 <br/>
 <br/>
 记者VS.小朋友：〝我画蝴蝶，（开心吗？），开心。〞
 <br/>
 <br/>
 小朋友：〝我在风车上画海水，上面画鱼和章鱼、上面画太阳、还有画草。〞
 <br/>
 <br/>
 小朋友：〝我喜欢画一只佩佩猪在草地上玩。〞
 <br/>
 <br/>
 民众：〝我每年都有带她们来参加，因为她对这个还满有兴趣的，很喜欢画画，然后回去都还满开心的。〞
 <br/>
 <br/>
 现场准备了千余份风车材料及颜料，让小朋友们发挥想像力，完成属于个人独特风格的彩绘风车。
 <br/>
 <br/>
 名间乡长 陈文德：〝他们的创作能够把它插在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_松柏岭.html" target="_blank">
  松柏岭
 </a>
 庙宇旁边广场，让所有的香客来观赏，展现他们的创作能力。〞
 <br/>
 <br/>
 名间狮子会会长 施江烨：〝我们就把名间所有的农产做在风车上面。〞
 <br/>
 <br/>
 主办单位将小朋友的风车作品插在广场上，让色彩缤纷的风车随风转动，宣传在地农产品，也为家乡增添美丽亮点。
 <br/>
 <br/>
 新唐人亚太电视陈汉昌、林沛芸台湾南投采访报导
</div>
