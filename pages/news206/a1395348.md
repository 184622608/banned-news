### 为橘色站台造势 柯文哲回应赖揆国家价值
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】再来关注台湾的竞选情况，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ，周六（13日）来到亲民党市议员黄珊珊的竞选总部，参加成立大会，稳固结盟势力。柯文哲受访时，被记者问到，前晚行政院长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
  赖清德
 </a>
 ，在姚文智的造势晚会上指出，首都市长的价值，应该与中央一致，下面来看柯文哲的回应。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ：〝其实我认为说把人民的福祉放在，放在最优先的考量，不可能中央跟地方意见不一样。〞
 <br/>
 <br/>
 台北市长柯文哲，为亲民党市议员黄珊珊站台，受访时为中央与地方价值如何一致，下了注解，至于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
  赖清德
 </a>
 特别提到的两岸问题，柯文哲是否和中央态度一样，他说。
 <br/>
 <br/>
 台北市长 柯文哲：〝其实还是一样，把人民的福祉，当做最重要一件事情来思考的时候，其实跟中央地方没有，不会有什么叫不同调的问题。〞
 <br/>
 <br/>
 柯文哲没有多谈，不过在活动上，柯文哲与亲民党主席宋楚瑜并肩而坐，双方在这次选举达成结盟，柯文哲势力壮大，也令外界不禁联想，是否也在经营未来选总统的可能性。加上这个周末，柯爸柯妈也在不同场合，为柯文哲辅选固桩，一家人颇有齐头并进的意味。而柯妈妈也被问到儿子对未来总统大选的看法。
 <br/>
 <br/>
 柯文哲母亲 何瑞英：〝（今天早上柯文哲被问到，那他有没有支持蔡总统，他就都避答耶。）避答就好了，这他说世事难料啊，我不是说这样，我是说我们现在做好自己就好。我自己是支持蔡总统，我自己是支持她。（可是不代表柯市长的看法？）他的头脑跟我的头脑有一样吗？〞
 <br/>
 <br/>
 台北市长 柯文哲：〝（刚刚柯妈妈受访的时候，还是有提到就是说，她自己个人是支持蔡总统，继续拼连任，但她也有提到说您可能这边是，不会要再选跟总统拼的这件事情。妈妈跟你的立场是一样的吗？）没有啦我跟你讲，我因为我今天还在跑摊你知道，我晚上回家，打电话给我妈妈，问一问清楚她到底在想什么。〞
 <br/>
 <br/>
 后来再被追问，对于2020竞选总统话题从不说死，柯文哲仍然回答，他没有做这个计画。
 <br/>
 <br/>
 新唐人亚太电视胡宗翰、黄彦菱台湾台北报导
</div>
