### 九合一民进党惨败  日媒：台湾长期低薪惹民怨
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】台湾〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 〞大选和高达10案的公投，身兼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 主席总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 宣布请辞民进党党主席一职，承担这次败选的政治责任。日媒分析认为，蔡政府执政后首次大型选举惨败，造成民怨四起的原因，恐与台湾长期低薪有关。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 失守了执政20年的高雄市，由国民党的韩国瑜当选市长。民进党候选人陈其迈当天发表感言时说，不管支持蓝绿，都是高雄市民，从明天开始为韩国瑜市长加油，为高雄加油。
 <br/>
 <br/>
 日本《每日新闻》以〝民进党在‘圣地’败北，蔡政府的支持者失望〞为题，指民进党大溃败，台湾现任总统兼民进党主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 辞去党主席一职，将让下届总统大选的动向变得难以预测。
 <br/>
 <br/>
 文章中提到，1998年以后，高雄市就一直由民进党执政，这次输掉高雄市长，其他县市也大都败选，对民进党无疑是一记痛击。
 <br/>
 <br/>
 文章分析认为，民进党败北主因恐和职场长期低薪、年轻失业率有关。台湾总失业率虽然只有3.76%（今年9月），但20～24岁的年轻人的失业率却高达12.29%（今年9月）。来自高雄三民区的38岁英文补习业者许婉真感叹，〝对蔡政权有期待，但景气完全没有好转〞。
 <br/>
 <br/>
 而韩国瑜却透过向选民宣传〝自己有过经营台北农产运销公司的经验〞，喊出要让〝高雄成为台湾最富有的城市〞的愿景，并透过亲民的打扮与口吻，抓住了选民的心，掀起〝韩流〞热潮。
 <br/>
 <br/>
 此外，两岸关系也是蔡政权的弱点。中共为了打压、松动〝独立志向〞的民进党政权，刻意大幅限缩陆客来台的人数，同时减少采购台湾的农产品，导致台湾东部及农村地带对蔡英文不满。
 <br/>
 <br/>
 虽然民进党在选战中声称：〝中共为了让亲中的国民党胜出，在网路上释出假情报、假新闻，干预台湾选举〞，但并未能藉此挽回劣势。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁)
