### 新北市长候选人侯友宜扫街 跑基层找乡亲较实在
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】国民党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新北市长.html" target="_blank">
  新北市长
 </a>
 候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侯友宜.html" target="_blank">
  侯友宜
 </a>
 今天（20日）到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金山万里.html" target="_blank">
  金山万里
 </a>
 扫街，还到了行政院长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
  赖清德
 </a>
 老家附近，刚好赖清德今晚要陪苏贞昌两场合体造势。侯友宜表示万里常来，虽然苏贞昌找赖清德辅选，但他自己认为，和万里的乡亲在一起更实在。
 <br/>
 <br/>
 现场音：〝友宜友宜友宜，当选当选当选。〞
 <br/>
 <br/>
 国民党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新北市长.html" target="_blank">
  新北市长
 </a>
 参选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侯友宜.html" target="_blank">
  侯友宜
 </a>
 ，来到金山老街扫街，拍照全面换上胜利手势，也就是侯友宜的2号号次。稍早，侯友宜还到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
  赖清德
 </a>
 故乡，万里玛鋉路上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拜票.html" target="_blank">
  拜票
 </a>
 。而赖清德晚上就要跟苏贞昌连续两场同台造势，侯友宜扫街扫到赖清德家附近，难道是刻意拚场？
 <br/>
 <br/>
 国民党新北市长参选人 侯友宜：〝万里我常常来，我们都希望不管谁的故乡，在新北市的好所在，我们都给它带来繁荣，所以其实赖院长来这替老县长助选，我感觉我最重要的，还是和万里的乡亲黏一起，比较实在，因为他们才是我的大咖。〞
 <br/>
 <br/>
 绿营对新北市选情很有信心，民进党立委段宜康还说新北市选情已经黄金交叉，苏贞昌从现在开始就会领先，侯友宜不以为然。
 <br/>
 <br/>
 国民党新北市长参选人 侯友宜：〝每一步你都要充满危机感，票没开出来，民调高还低，你都要用平常心来看，不管你赢20几%、10几%，或者是什么样的交叉，记得给2号侯友宜，催票拉票〞。
 <br/>
 <br/>
 不论领先还是落后，侯友宜都说戒慎恐惧。但对手频频攻他不敢辩论，国会观察基金会董事长姚立明还批评，侯友宜政见不够具体，都是复制苏贞昌主张，才一再避免辩论。
 <br/>
 <br/>
 国民党新北市长参选人 侯友宜：〝我这么多年来在新北市，做了很多老县长没有做过的，托老、托育、三环三线、运动中心，甚至包括很多老人家要照顾的动健康等等。〞
 <br/>
 <br/>
 强调自己有独到政见，但侯友宜始终回避辩论。
