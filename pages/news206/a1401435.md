### 台大逾50年〝鹿鸣堂〞 校友抢救获登历史建筑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾大学.html" target="_blank">
  台湾大学
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鹿鸣堂.html" target="_blank">
  鹿鸣堂
 </a>
 〞原名〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侨光堂.html" target="_blank">
  侨光堂
 </a>
 〞于1967年建造完成，为早年推动侨务重要场域，也曾为重要文化、政治活动集会场所。原订7月拆除，但在台大校友发起抢救行动，当时决议就鹿鸣堂成为暂定古迹重启审议。台北市文化局今(30日)召开文资审议会，通过登录鹿鸣堂为历史建筑。
 <br/>
 <br/>
 中央社报导，文化局7月在文资会上表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鹿鸣堂.html" target="_blank">
  鹿鸣堂
 </a>
 外观采中式建筑元素，也结合西式建筑美学，且在侨务工作上具有历史意义，但文资委员会勘后，认为现况保存不佳，一楼作为餐饮区使用，经整修后已失去原貌。
 <br/>
 <br/>
 不过，由于〝抢救台大鹿鸣堂阵线〞成员提出认定鹿鸣堂历史价值的新事证，文资委员讨论后，认为侨生、侨胞活动于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侨光堂.html" target="_blank">
  侨光堂
 </a>
 的公共性，还有侨光堂当年是北市少数重要事件的集合空间，这些事证过去在审议会上没有被详细讨论，因此就鹿鸣堂文资身分重启审议。
 <br/>
 <br/>
 今天讨论后，委员认为，侨光堂于1967年建造完成，作为当年台湾与海外华侨联系活动的重要集会中心，为早年推动侨务重要场域，也曾为重要文化、政治活动集会场所。
 <br/>
 <br/>
 另外，侨光堂为大型集会堂构造，展现现代主义建筑思维，融合中式传统建筑语汇，反映1960年代中华文化复兴运动下的设计特色，具有保存价值。
 <br/>
 <br/>
 对于鹿鸣堂获登录为历史建筑，台大副总务长徐炳义指出，将鹿鸣堂登录为历史建筑后，包括古迹和历史建筑，台大校内已有44栋文化资产，但台大每年的房屋修缮经费仅新台币8000万元，只有1000万元可用于文资紧急基本修缮，经费是未来鹿鸣堂维护上的现实问题。
 <br/>
 <br/>
 他表示，2002年台大曾请专业单位鉴定，认为鹿鸣堂结构不佳，有安全疑虑，因此于2011年作出拆除决议；时隔16年，未来要如何善加修缮这栋历史建筑，台大内部研议讨论后，会再向文化局提出管理维护计画。
</div>
