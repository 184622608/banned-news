### 首场电视辩论独缺他 柯文哲：被攻击惯了
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】距离年底县市首长选战剩下不到1个月，首场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电视辩论.html" target="_blank">
  电视辩论
 </a>
 在今天下午举行，但5位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_市长候选人.html" target="_blank">
  市长候选人
 </a>
 中，独缺台北市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ，对于是否担心成为攻击目标，柯文哲受访时表示，他被攻击习惯了。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北市长.html" target="_blank">
  台北市长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯文哲.html" target="_blank">
  柯文哲
 </a>
 ：〝老实讲我被攻击习惯了啦，我看那个传统媒体已经打超过一年了，我也在想、奇怪，打超过一年还没有倒，这表示说，没有啦、反正这场选举结束之后，这个在学术上可以留下很多研究的项目。〞
 <br/>
 <br/>
 台北市长选战进入最后冲刺，首场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_市长候选人.html" target="_blank">
  市长候选人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电视辩论.html" target="_blank">
  电视辩论
 </a>
 4日下午登场，但5位市长候选人中，独缺争取连任的台北市长柯文哲，对于可能会成为其他候选人的箭靶，柯文哲表示，他被攻击习惯了，至于妻子陈佩琪将赴台中参加花博，外界解读是缓和与绿营关系，柯文哲说，是去帮忙行销台湾。
 <br/>
 <br/>
 台北市长 柯文哲：〝花这么多钱然后是让，它的目标是要对整个国外推出去的，这个等于是行销台湾，所以、所以，该可以帮忙的，帮多少算多少。〞
 <br/>
 <br/>
 至于柯文哲3日参加市议员候选人徐世勋在公园举办的活动时，送上冰冻的蒜头，象征〝冻蒜〞，引发外界质疑，是在踩议员候选人选前10天才能在公园进行竞选活动规定的红线。
 <br/>
 <br/>
 台北市长 柯文哲：〝(这个场子很明显就是选举场合啊)？对啊、不是啦、但是我，但是老实讲，我去之前一定不知道它是违法的。回去、也回去检讨看看吧，不过有时候人家请我们去我们就去，也不会去再去跟它检查你有合法没有。〞
 <br/>
 <br/>
 而对于竞选对手多次攻击他没有政绩，柯文哲3日在脸书发文反酸，柯文哲受访时也表示，那篇发文是自己写的，没有很呛，他只是说回去google查一查就好了。
 <br/>
 <br/>
 新唐人亚太电视曾奕豪、黄庭锋台湾台北报导
