### 赖揆逛台中花博 赞文资保存与环保双赢
------------------------

<div class="wysiwyg">
 【新唐人2018年11月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台中花博.html" target="_blank">
  台中花博
 </a>
 开幕头一天，行政院长赖清德也亲自到后里马场园区参观，他称赞林佳龙保留了石虎栖息地和百年马场，台中花博在上午开园后，已经涌入4万8000人。
 <br/>
 <br/>
 行政院长 赖清德：〝哇！这个比家里的椅子还要舒服。〞
 <br/>
 <br/>
 体验用马草做成的椅子，花博开幕第一天，行政院长赖清德，在台中市长林佳龙的陪同下，抢先到后里马场园区走走。
 <br/>
 <br/>
 行政院长 赖清德：〝今天天气非常好，也是第一天开幕，我迫不及待抢在第一天奔来欣赏。〞
 <br/>
 <br/>
 一路上欣赏马术游行和特技表演，还有热情民众握手拍照，对于这里就是发现一级保育类动物石虎的地方，赖清德大赞，园区保留了石虎栖息地与百年马场。
 <br/>
 <br/>
 行政院长 赖清德：〝没有说硬是把整个栖息地开发，反而是保留给石虎，除了碰到石虎以外，又碰到马场，百年马场又是文化资产，又要保护它，一个市长到底怎么样因应，结果他创造双赢。〞
 <br/>
 <br/>
 同一时间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_故宫.html" target="_blank">
  故宫
 </a>
 花蝶馆外大排长龙，都是来一睹翠玉白菜、小白菜的风采。
 <br/>
 <br/>
 民众 乔先生：〝很少这种宝物会外借，算是很稀有，算是帮这次花博争美啊。〞
 <br/>
 <br/>
 来自故宫尊贵的娇客，为花博增添亮点，翠玉白菜将等半年花博结束后才会再回到故宫，截至五点之前，三个园区已经涌入近4万8000人。
 <br/>
 <br/>
 新唐人亚太电视林莉、詹咏茹台湾台中报导
 <br/>
</div>
