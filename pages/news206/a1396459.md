### 2艘美舰穿越台海 释放信号不一般
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】日前，两艘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国军舰.html" target="_blank">
  美国军舰
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穿越台湾海峡.html" target="_blank">
  穿越台湾海峡
 </a>
 。美国国防部表示，这是例行航行，体现美国致力于使印度太平洋区域自由开放。自川普（特朗普）总统签署《台湾旅行法》后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美台关系升温.html" target="_blank">
  美台关系升温
 </a>
 。近期，美国军舰已多次航行台湾海峡。
 <br/>
 <br/>
 美国国防部10月23日证实，两艘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国军舰.html" target="_blank">
  美国军舰
 </a>
 柯蒂斯．威尔伯号驱逐舰（DDG 54）与安提坦号巡洋舰（CG 54），于22日依循国际法例行性通过台湾海峡。
 <br/>
 <br/>
 美国国防部发言人罗根（Christopher Logan）对台媒中央社表示，这两艘舰艇的通过，体现美国致力于自由开放的印度太平洋区域。美国海军将继续在国际法所允许的任何地方飞行、航行与操演。
 <br/>
 <br/>
 台湾总统府随后表示，台湾一向重视台海及区域的和平稳定，致力于维持两岸现状，总统府乐见各种有助于区域安全稳定的努力。
 <br/>
 <br/>
 自美国总统川普3月16日正式签署〝鼓励美台之间所有层级互访〞的《台湾旅行法》后，美台关系持续升温。该法案形容台湾是亚洲〝民主灯塔〞，并指〝台湾的民主成就，启发区域内很多国家和人民〞。
 <br/>
 <br/>
 近期，美国军舰多次在台湾海峡航行。7月，美国2艘伯克级驱逐舰通过台湾海峡；10月15日，美国海军科研船停泊在高雄港，中共对此表示不满。
 <br/>
 <br/>
 10月19日，澳洲又公布消息，称导弹护卫舰墨尔本号（HMAS Melbourne）早在去年就已通过台湾海峡。美国一名政府高级官员表示，澳舰通行台湾海峡，证明澳洲是美国的联盟伙伴。
 <br/>
 <br/>
 《澳洲人报》指出，美国官员讨论澳舰议题，暗示美国及其盟友正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反对中共.html" target="_blank">
  反对中共
 </a>
 对台湾的主张。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：文慧)
