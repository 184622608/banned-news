### 台国道3号新竹段连环追撞 台积电交通车多人伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国道3号.html" target="_blank">
  国道3号
 </a>
 南下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新竹香山路段.html" target="_blank">
  新竹香山路段
 </a>
 ，上午上班尖峰时间，发生5部车追撞意外，其中1部是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台积电.html" target="_blank">
  台积电
 </a>
 交通车。整起车祸造成7人受伤，其中2人伤势较严重，但暂无生命危险，详细肇事原因由国道警方调查厘清。
 <br/>
 <br/>
 事发在上午8时许，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国道3号.html" target="_blank">
  国道3号
 </a>
 南下107.6公里处，救护人车抵达现场，发现共有5部车辆连环追撞，分别是2部货车、2部轿车以及1部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台积电.html" target="_blank">
  台积电
 </a>
 交通车中型巴士。车子零件散落一地，有的车头扭曲变形，有多人受伤。
 <br/>
 <br/>
 消防局表示，货车上有2名伤者，载运台积电员工的巴士上则有5人受伤，2部自小客车无人受伤。7名伤患中有1人骨盆疑似变形、另1名头部外伤，送往新竹马偕医院，其余5名多为头部、手脚等处擦挫伤，伤势较轻微，分别送往马偕及国泰医院。
 <br/>
 <br/>
 连环车祸发生时，正值上班交通尖峰时段，一度造成后方车阵回堵，至上午9时41分排除事故，恢复正常通车，详细事故原因正由国道警方调查中。
 <br/>
 <br/>
 (责任编辑：叶日观)
</div>
