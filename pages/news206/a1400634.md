### 民众热情簇拥投票 韩国瑜：谦虚面对结果
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】中华民国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 选举24日投票，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 高雄市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 由女儿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩冰.html" target="_blank">
  韩冰
 </a>
 陪同，上午至林园中学投票所投票，受到现场民众热情簇拥〝韩总加油〞声此起彼落。他说，会谦虚地面对选举结果。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 受访表示，这次选举高雄人民的热情令人感动，所有乡亲的眼神都是炽热的，透露着期待及改变，不仅感动全台，也感动海外，这次选举连海外华人都在看。
 <br/>
 <br/>
 他说，参选一路走来该论述的，都已论述了，接下来将交由市民做神圣的决定，他会谦虚地面对今晚的开票结果。
 <br/>
 <br/>
 韩国瑜较原定时间迟到约1小时抵达投票所，他说，因为上午从高雄市区来这里一路堵车，一路人多车多，好像过年一样。
 <br/>
 <br/>
 韩国瑜说，他昨晚睡得不错，昨晚选前之夜晚会结束后回到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国民党.html" target="_blank">
  国民党
 </a>
 高雄市党部坐镇因应可能的突发状况，因此没在林园户籍地过夜。
 <br/>
 <br/>
 他说，每到选举高雄市都担心有奥步，昨晚他跟幕僚就守在市党部直到今天凌晨2时，因应可能的突发状况，投完票后也会再回到市党部待命。
 <br/>
 <br/>
 韩国瑜表示，〝君子不忘本〞，若能当选市长，就职第一天晚上就会睡在高雄果菜市场，因为他是卖菜出身的。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩冰.html" target="_blank">
  韩冰
 </a>
 则表示，陪父亲前来投票，是希望首投族能踊跃出来投票。
