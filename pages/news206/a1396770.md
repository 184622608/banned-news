### 普悠玛翻车人为疏失 台铁：司机关ATP未通报
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】火线新闻，带您锁定，造成200多人伤亡的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 翻车，累计有37国与欧盟向台湾表达慰问。台铁在今天上午，召开事故总结记者会 公布调查报告，表示事故主因是〝人为疏失〞；台铁指出，自动防护系统，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_ATP.html" target="_blank">
  ATP
 </a>
 并没有故障，尤姓司机关闭后，没有通报，并且超速行驶。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台铁局.html" target="_blank">
  台铁局
 </a>
 副局长 杜微：〝再一次的表达歉意。〞
 <br/>
 <br/>
 针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普悠玛.html" target="_blank">
  普悠玛
 </a>
 号
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻覆事故.html" target="_blank">
  翻覆事故
 </a>
 ，台铁局再次鞠躬向社会大众道歉，台铁副局长杜微表示，事发主因是司机员关闭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_ATP.html" target="_blank">
  ATP
 </a>
 导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_列车超速.html" target="_blank">
  列车超速
 </a>
 ，事后调阅无线电通联纪录，并未显示司机员有向调度员报备关闭ATP，也就没有根据SOP配合因应措施。
 <br/>
 <br/>
 台铁局副局长 杜微：〝事后有去调我们行调无线电的通联记录，那记录上并没有显示他有向我们的调度员，做任何报备的动作。真有关闭的情形依照SOP，他必须要先报备才能关，那我们后续的SOP还有辅助措施，不是说他关掉就可以开了，它有很多的限制。〞
 <br/>
 <br/>
 副局长杜微说，台铁虽要求列车赶点，但绝不允许超速赶点，加上有ATP保护，不可能超速，ATP如果故障或关闭，反而将会启动降速措施。
 <br/>
 <br/>
 台铁局副局长 杜微：〝看起来是这个司机，他有一些人为疏失的因素在里面。他把车上的列车防护装置关闭，这点是确认的，然后他在经过事故现场的时候，又超速行驶，这是这个事故的主因。〞
 <br/>
 <br/>
 台铁表示，列车在宜兰到罗东之间，主要是因为列车主风泵保护装置故障，让列车没有办法出力，主风泵是供应列车气压，气压不足就会抑制动力，是列车保安措施，对于安全没有影响，至于是否有其他肇事原因，以及谁该负责，台铁都会尊重行政院调查小组的调查结果。
 <br/>
 <br/>
 新唐人亚太电视林嘉韦、黄庭锋台湾台北报导
</div>
