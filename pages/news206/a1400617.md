### 台中市长候选人与家人投票 卢秀燕：不管结果都是祝福
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_10案公投.html" target="_blank">
  10案公投
 </a>
 11月24日开始投票。国民党籍台中市长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢秀燕.html" target="_blank">
  卢秀燕
 </a>
 在家人的陪同下前往投票。她受访时表示，对选举抱着平常心，不管结果如何都会祝福。
 <br/>
 <br/>
 中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢秀燕.html" target="_blank">
  卢秀燕
 </a>
 周六上午9时40分，在丈夫廖述嘉的陪同下，前往北区健行国小投票。她身穿衬衫、牛仔裤，与民众一同排队等候投票，也向热情民众打招呼。
 <br/>
 <br/>
 她受访时表示，觉得这场选战非常有意义，心情很期待，不管结果如何她都会祝福。她说，昨天晚上睡得很好、一觉到天亮，在宣传车站了快10天，实在是有点累，出门前家人也都与她拥抱给予鼓励。
 <br/>
 <br/>
 媒体询问她是否有带幸运小物，卢秀燕回说没有，带着平常心就好。提到喉咙声音沙哑，她表示，医生说要多休息、少讲话，但目前显然不可能，正在做的事情都是违反医生的交代，选后才能好好休息。
 <br/>
 <br/>
 对于公投是否带小抄，卢秀燕说，她跟民众一样，对这次的公投都很关心，事先都有做功课，心里已经有定数，所以没有带小抄。今天的天气很好，对民众的投票率是有帮助，但是听说每个投开票所都是大排长龙，有等待的情形，希望选务顺利。
 <br/>
 <br/>
 卢秀燕，籍贯山东诸城，生于基隆，现任立法委员至今已连任六届，公民监督国会联盟八次评鉴第一名国会优秀立委。
 <br/>
 <br/>
 规模空前的2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一选举.html" target="_blank">
  九合一选举
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_10案公投.html" target="_blank">
  10案公投
 </a>
 11月24日开始投票，将选出1万1047名公职人员。每人最多可领到15张选票，再加上18岁青年首度有权公投，堪称史上最复杂选举。投票时间为上午8时到下午4时，估计选举票将在晚间9时开完，公投结果将于25日凌晨2时左右出炉。
 <br/>
 <br/>
