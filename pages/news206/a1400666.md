### 纪念大法洪传世界  台湾排字英文《转法轮》
------------------------

<div class="wysiwyg">
 【新唐人2018年11月24日讯】尽管遭受中共打压，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 不但没有停止发展，反而洪传全世界一百多个国家和地区。周六，约5,400名来自台湾及亚洲各地区的部分法轮功学员，在台北自由广场，排出英文版《转法轮》图案，纪念这一盛况。
 <br/>
 <br/>
 来自台湾和亚洲其他地区的5,400名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员，排出英文版《转法轮》的图案。深蓝色封面、中英文的〝转法轮〞字样，还有42道绽放的金光，场面相当壮观。
 <br/>
 <br/>
 法轮功学员排字活动始于中国。1996年，贵州法轮功学员排出法轮图形，此后中国法轮功学员多次在各种活动上进行排字。
 <br/>
 <br/>
 这一传统后来被台湾学员继承下来。多年来，台湾学员每年都会举行大型排字活动，向世界洪扬
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮大法.html" target="_blank">
  法轮大法
 </a>
 的美好，抵制中共在中国对法轮功的迫害。
 <br/>
 <br/>
 从1992年在中国传出至今，法轮功已经传遍一百多个国家和地区，上亿人身心受益。尽管遭受中共的迫害，法轮功不但没有消失，反而更加蓬勃发展。
 <br/>
 <br/>
 新唐人记者林宇、谢心欣综合报导
