### 新北市长参选人苏贞昌海报背景 科技感十足
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新北市.html" target="_blank">
  新北市
 </a>
 长参选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏贞昌.html" target="_blank">
  苏贞昌
 </a>
 ，今天（20日）一早就到新店扫街催票。宣传海报以动态影片当背景，科技感十足，苏贞昌说这个是有史以来第一次两个院长同时站在台上，跟市民报告政见，希望让大家看见新北应该改变了。
 <br/>
 <br/>
 跟市场小贩一一问好，不忘转过身，对直播镜头热情介绍，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏贞昌.html" target="_blank">
  苏贞昌
 </a>
 扫街冲选情，也透过新科技帮市场老板们拉点生意。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民进党.html" target="_blank">
  民进党
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新北市.html" target="_blank">
  新北市
 </a>
 长参选人 苏贞昌：〝线上朋友，你看这么年轻，他在这里，你看这么努力，各种不同的蛋品让你参考。我们也要跟得上时代，很多人不但跟不上，这种新技术搞不清楚，甚至连观念都还非常迂腐。〞
 <br/>
 <br/>
 分享直播心得，也向对手叫阵，选战开打到现在，苏贞昌不只频频开直播跟网友互动，形象广告、政见影片也大玩科技风。
 <br/>
 <br/>
 海报上，苏贞昌正在跟行政院长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
  赖清德
 </a>
 交谈，一个点头微笑、一个指手画脚，动态感十足，恍如电影《哈利波特》中会动的照片，海报内容是要预告两人在周六（20日）晚间，将有连续两场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_造势.html" target="_blank">
  造势
 </a>
 晚会。
 <br/>
 <br/>
 民进党新北市长参选人 苏贞昌：〝赖院长是新北子弟，我又当过8年县长跟行政院长，两个人站在台上跟市民报告，这是历史上第一次，所以我们选择，就在市政府所在地板桥，同时在溪北的三重。〞
 <br/>
 <br/>
 苏贞昌打选战，不只有年轻团队拓展网路声量、行政院长赖清德同台开讲，自家立委也上广播节目助势帮腔，立委段宜康前一天说，国民党新北市长参选人侯友宜，在深澳电厂停建后表现出质疑态度，相当荒谬，加上始终不愿接受辩论，将会让新北选情出现交叉，苏贞昌从现在开始就要领先了！
 <br/>
 <br/>
 民进党新北市长参选人 苏贞昌：〝新北市民水准很高，新北市民不是傻瓜，新北市民知道未来4年的新北，要交给谁才会进步。(侯友宜)不只是政策抄袭，连布置都抄袭，因为这么生疏所以不敢面对，因此不敢辩论。〞
 <br/>
 <br/>
 新北选战进入白热化，苏阵营从上到下卯起来拚了。
