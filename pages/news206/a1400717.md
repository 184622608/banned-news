### 九合一惨败 蔡英文辞党主席慰留赖清德
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】
 <h4>
  新闻周刊（655）
 </h4>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 选举民进党惨败，党主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 24日晚间宣布负起政治责任，请辞党主席，而阁揆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
  赖清德
 </a>
 与总统府秘书长陈菊，两人提出口头请辞，是双双被慰留。
 <br/>
 民进党主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 ：〝我从现在开始辞去民主进步党的主席一职。〞
 <br/>
 <br/>
 蔡英文辞党主席，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一.html" target="_blank">
  九合一
 </a>
 惨败，负起政治责任。
 <br/>
 <br/>
 民进党主席 蔡英文：〝今天民主给我们上了一课，我们就应该要虚心接受人民更高的要求。对于民意的展现，我们当然要深自检讨。〞
 <br/>
 <br/>
 民进党将启动主席改选，与党务改革，而行政院长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
  赖清德
 </a>
 和总统府秘书长陈菊，也提出请辞，蔡英文予以慰留。
 <br/>
 <br/>
 民进党主席 蔡英文：〝而在改革的关键期，政府的行政团队则必须保持稳定，因此，虽然刚才赖院长已经向我口头请辞，但我已经请他继续留下来打拚，以确保各项重大政策持续的推动。〞
 <br/>
 <br/>
 对选举结果虚心检讨，蔡英文向支持者致歉，而下周中常会，将选出代理主席，即刻启动党务改选。
 <br/>
 <br/>
 新唐人亚太电视曾奕豪、刘姿吟台湾台北报导
</div>
