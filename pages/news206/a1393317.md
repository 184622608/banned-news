### 台中接力展出30年 167位美术家回娘家
------------------------

<div class="wysiwyg">
 【新唐人2018年09月28日讯】台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台中.html" target="_blank">
  台中
 </a>
 市葫芦墩文化中心办理〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美术家.html" target="_blank">
  美术家
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_接力展.html" target="_blank">
  接力展
 </a>
 〞，30年来，培育216位艺术创作者，这次30周年展，邀集历届参与接力的167位艺术家回娘家，展现一场跨世代的美术飨宴。
 <br/>
 <br/>
 一百多位艺术家齐聚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台中.html" target="_blank">
  台中
 </a>
 胡芦墩文化中心，展出当时创作的270件优秀作品，依年代陈列，回顾半甲子岁月艺术思潮的演变。
 <br/>
 <br/>
 策展人国立台湾师范大学美术系教授 庄连东：〝台中市文化局为艺术家办了30年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_接力展.html" target="_blank">
  接力展
 </a>
 ，也就是说一年一个月一档，一年12个这样子办了30年，这一次的30年回顾展，其实一方面要把这些，曾经在这边办过接力展的画家回娘家，大家来庆贺一下；另一个部分是希望透过一个研究的方式，把这30年来比较早期的风格，画家个人成长的历程，到现在这近十年来，所反应时代的状态来做一个梳理。〞
 <br/>
 <br/>
 台中市长 林佳龙：〝30年来从过去的台中县，到现在升格后的台中市，总共有216位的艺术家，他们一棒接一棒，累积了这么多好的作品，今天回娘家，有参与这一次的策展，就高达167位有接近八成之多。〞
 <br/>
 <br/>
 参与第一届台中县接力展的雕塑大师谢栋梁表示，30年来的创作材质和理念都在改变。
 <br/>
 <br/>
 资深雕塑家 谢栋梁：〝我转变很多，我不敢说进步，但是我每次展览，都有不同的手法，但是都是以人为主，就是表现人的一种关怀，人性的探讨，这个就没有变，到了2005年以后才开始做山水意象的系列，2005年以前都是以人体为主。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油画.html" target="_blank">
  油画
 </a>
 艺术家 庄明中：〝今年30年的回顾展，我就把民国74年的作品，跟106年的作品来做一个对照，可以看出这段时间在创作上的一个变化，跟创作理论探讨的一个不同。〞
 <br/>
 <br/>
 从展览中看到艺术家，美感意识转换的脉动及风格的演绎，也彰显大台中的艺术风华在台湾美术史上特殊的地位。
 <br/>
 <br/>
 新唐人亚太电视王媛、林莉台湾台中采访报导
</div>
