### 台湾屏东铁道文物展 唤醒往日记忆
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】搭火车可以说是许多人生活中的共同记忆，但是随着铁路
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电气化.html" target="_blank">
  电气化
 </a>
 与高架化等现代化工程，很多记忆中的事物也渐渐消失。在屏东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枋寮艺术村.html" target="_blank">
  枋寮艺术村
 </a>
 ，展出了70余件的铁道文物及图片，一起来看看。
 <br/>
 <br/>
 日治昭和时期的铁路时刻表，西海岸的铁道路线图，古色古香的票柜，这些超过半甲子的铁道文物及图片，共有70多件，彷佛让人穿梭时空走入时光隧道，唤起早期铁道的记忆。
 <br/>
 <br/>
 屏东县文化处副处长 曾龙阳：〝介绍出我们整个台湾铁路一个脉络跟历史，那重新再认识台湾铁路的面貌。〞
 <br/>
 <br/>
 这一张被二人扛起的大时钟，原本是悬挂在屏东旧火车站正门，摄影家邱才彦用快门纪录下当初的拆除画面，也感伤光阴的流逝。
 <br/>
 <br/>
 摄影家 邱才彦：〝屏东火车站要拆的时候，我想说，我们印象中最深刻的就是这个时钟，我就趁着他们累了放下，提起来比较有力，我把它按下快门，因为这张给我感觉就是说，好像时光不在了，还有另外一种感觉就是把握时光。〞
 <br/>
 <br/>
 收藏家陈朝强表示，台铁人员工作时所戴的大盘帽，上头有着各种颜色线条别具意义，也分别代表不同的工作性质。
 <br/>
 <br/>
 收藏家 陈朝强：〝像一般来讲就是红色的部份，那红色的部份一般就是站长，副站长，那就是红色的部份，那金线的部份愈粗就代表他的层级愈高。〞
 <br/>
 <br/>
 随着铁路
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电气化.html" target="_blank">
  电气化
 </a>
 、高架化，记忆中的铁道一景一物纷纷走入历史，透过展出，也让人更加珍惜身边的人与事物。
 <br/>
 <br/>
 新唐人亚太电视李金菊
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾屏东.html" target="_blank">
  台湾屏东
 </a>
 报导
</div>
