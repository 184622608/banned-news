### 联邦调查局结束对卡瓦诺的调查 两党反应大不同
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-10-05T03:40:59+08:00">
   2018年10月5日 03:40
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <p>
  星期四，在联邦调查局提交了对特朗普总统大法官提名人选布雷特·卡瓦诺性侵指控的调查报告几小时后，参议院共和党人不顾民主党人的强烈反对，表示保证会继续推进对卡瓦诺的确认投票。
  <br/>
  <br/>
  参议员有责任不披露联邦调查局调查报告的细节。这份报告只有一份副本，参议员可以在国会大厦一间安保措施严密的房间里进行审阅。看过报告的许多共和党参议员对记者表示，报告中没有卡瓦诺牵涉不当性行为的内容。
 </p>

 <p>
  参议院多数党领袖、来自肯塔基州的共和党籍参议员麦康奈尔说，联邦调查局无法查到“任何可以证实这些指控的第三方 - 没有任何证人的支持，包括那些最初提出指控的人所提到的那些具体的目击证人。”
  <br/>
  <br/>
  上个星期，克里斯丁·布莱西·福特在参议院司法委员会的听证会上说，卡瓦诺在1980年代初的一个派对上对她有性侵行为。但是卡瓦诺坚决否认。
  <br/>
  <br/>
  民主党人也没有说联邦调查局报告有哪些内容，但是他们称，白宫和参议院司法委员会的共和党人给调查人员设限，因此调查受到束缚。一些媒体报道说，福特和卡瓦诺都没有被联邦调查局问话。
  <br/>
  <br/>
  司法委员会的资深民主党成员、来自加州的黛安·范斯坦说：“这份报告最显著的部分就是它没涉及的内容。报告看起来是一个不完整调查的结果。”
  <br/>
  <br/>
  她还说：“民主党人同意调查的范围应当是有限的。但是我们没有同意，白宫可以束缚联邦调查局的双手。”
  <br/>
  <br/>
  白宫发言人拉吉·沙阿说，在“史上对最高法院大法官提名人最全面的审查”之后，白宫“完全有信心”卡瓦诺会得到确认。
 </p>
 <p>
  预计参议院将在星期五进行程序性投票，并可能在星期六对是否确认卡瓦诺出任大法官举行最终的投票。
 </p>
 <p>
  上个星期，参议院司法委员会投票批准向参议院全院推荐卡瓦诺出任大法官，但是条件是，联邦调查局要对卡瓦诺做一个补充背景调查。
  <br/>
  <br/>
  参院民主党人都不支持卡瓦诺的任命。两名非常关键的共和党人在审阅联邦调查局报告后对记者表示，他们目前还没有承诺是否会投赞成票。
 </p>

 <p>
  来自缅因州的参议员苏珊·柯林斯称调查是“彻底的”；来自亚利桑那州的杰夫·弗莱克说：“我们没有看到额外的证实指控的信息。”
  <br/>
  <br/>
  星期三发布的一项由全国公共广播电台、马里斯特学院（Marist）和美国公共电视网新闻一小时节目共同进行的民调显示，47%的受访者说他们对卡瓦诺持负面看法，36%的受访者说，他们对卡瓦诺持正面看法。还有18%的受访者表示不确定或没有听说过卡瓦诺。
  <br/>
  <br/>
  在另一个问题上，45%的受访者认为，他们相信福特在讲述高中时发生的事情时，讲的是真话；33%的人说，他们相信卡瓦诺。
  <br/>
  <br/>
  共和党人在参议院占据51个席位，民主党人有49个席位。如果出现平票，身为共和党人的副总统彭斯可以投票，因此参议院的共和党人至少要有50票才能最终确认卡瓦诺出任大法官。
  <br/>
  <br/>
  如果保守派的上诉巡回法院法官卡瓦诺得到确认，他将接替退休的安东尼·肯尼迪大法官出任最高法院大法官。应当有九名大法院的最高法院目前只有八名大法官。
