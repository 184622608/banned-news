### 匹兹堡宗教领袖们联合谴责仇恨犯罪
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-10-31T02:45:40+08:00">
   2018年10月31日 02:45
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <span class="dateline">
  匹兹堡 —
 </span>
 <p>
  宾夕法尼亚州匹兹堡市上星期六成为又一个发生滥杀枪击事件的美国城市。一名枪手闯入“生命之树”犹太教堂，朝大部分是老年和残疾人的信众射击。枪手然后朝赶到现场的警察开枪，打伤了警员。匹兹堡三位宗教领袖联合发出号召，强调匹兹堡城不会被仇恨击溃。
 </p>
 <p>
  匹兹堡星期一的夜晚。社区人士聚集在被警察用胶带围住的追思场地，悼念星期六松鼠山“生命之树”犹太教堂滥杀枪击案的受害者。松鼠山是个以犹太裔为主的社区。对于匹兹堡地区这三位宗教领袖来说，星期六早晨与往常没有什么不同。
  <br/>
  <br/>
  基督教女牧师李迪·巴罗说：“我已经起来了。家人还没醒，我在看脸书。”
 </p>
 <p>
  伊斯兰教阿訇哈姆扎·佩雷斯说：“我在厨房，正准备喝咖啡。”
 </p>
 <p>
  犹太教拉比罗恩·西蒙斯说：“我正准备去南希尔斯教堂，参加一个安息日的祷告仪式。”
 </p>
 <p>
  几分钟后，他们开始意识到，这个星期六不太寻常，因为满大街都是急救车。
  <br/>
  <br/>
  犹太社区中心“慈爱与公民参与中心”的拉比罗恩·西蒙斯说：“急救车一辆接一辆地从我身边冲了过去。“
  <br/>
  <br/>
  “时代之光”清真寺的阿訇哈姆扎·佩雷斯说：“各种信息开始从四面八方涌来—”
  <br/>
  <br/>
  “宾夕法尼亚西南基督之家”的女牧师李迪·巴罗说：“很快就看出，这起灾难很严重。“
  <br/>
  <br/>
  枪击开始不久，这些宗教领袖们就碰面了。
  <br/>
  <br/>
  “时代之光”清真寺的阿訇哈姆扎·佩雷斯说：“我立即给女牧师李迪打电话—“
  <br/>
  <br/>
  犹太社区中心“慈爱与公民参与中心”的拉比罗恩·西蒙斯说：“我拿起电话，找到了李迪。我说，李迪，你应该开始想想了。我们该做些什么？我不知道什么时间，但会发生什么？”
  <br/>
  <br/>
  他们三人策划举行了一次跨越信仰的悼念烛光晚会。有数千人参加。很多人站在雨中祷告。这三位宗教领袖说，匹兹堡以蓝领职业道德著称，这种情况很正常。
  <br/>
  <br/>
  “宾夕法尼亚西南基督之家”的女牧师李迪·巴罗说：“匹兹堡韧性十足。我们过去就闯过艰难的时光，有重建的传承。”
  <br/>
  <br/>
  拉比罗恩·西蒙斯说：“匹兹堡的意志坚硬如钢。”
  <br/>
  <br/>
  阿訇哈姆扎·佩雷斯说：“匹兹堡很美。匹兹堡让你有主人翁的感觉。”
 </p>
 <p>
  “宾夕法尼亚西南基督之家”的女牧师李迪·巴罗说：“匹兹堡是一座移民城市。匹兹堡族裔混杂，充满活力，珍惜并拥抱多元化。”
 </p>
 <p>
  犹太社区中心“慈爱与公民参与中心”的拉比罗恩·西蒙斯说：“因为多样化，匹兹堡居民的品行更为多姿多彩。”
 </p>
 <p>
  “时代之光”清真寺的阿訇哈姆扎·佩雷斯说：“匹兹堡就是如此的坚强。”
  <br/>
  <br/>
  在“歌颂慈悲”这个全国穆斯林团体的帮助下，匹兹堡的穆斯林社区在一天之内为星期六暴力事件的受害者募捐了七万美元。
  <br/>
  <br/>
  拉比罗恩·西蒙斯说：“我们此时站在一起，因为不管在哪，我们的世界都需要团结。”
 </p>
 <p>
  这三位宗教领袖坚持集体接受采访。他们熟知彼此，共事多年。他们表示，打击仇恨与偏执，爱与团结是他们手中最好的武器。
 </p>
 <p>
 </p>
