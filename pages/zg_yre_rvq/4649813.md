### 加州酒吧枪击案 枪手打死12人
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-11-08T21:01:31+08:00">
   2018年11月8日 21:01
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <p>
  美国南加州文图拉县（Ventura County）政府表示，一名枪手11月7日星期三晚上在一个酒吧舞厅打死12人。
  <br/>
  <br/>
  文图拉县警长吉奥夫·迪恩告诉媒体， 派到现场的警察当中有一人被击中。任职29年的朗·赫勒斯警官后来在医院伤重不治身亡。
  <br/>
  <br/>
  迪恩警长说：“我告诉他妻子他是一个英雄。”
  <br/>
  <br/>
  这次袭击还造成了数人受伤，但是迪恩说确切的数字还不清楚，因为一些受伤的人自己去了医院。
  <br/>
  <br/>
  调查人员在联邦调查局帮助下确定枪手的身份和动机。
 </p>
 <p>
  迪恩说嫌疑人在酒吧里死亡，身上有枪伤，但不清楚是谁开的枪。
  <br/>
  <br/>
  目击者说，当时的情景非常混乱，人们纷纷躲避或试图逃离。
 </p>
</div>
