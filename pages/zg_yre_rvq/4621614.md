### 美国会中期选举战场州两党抢攻移民选票
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-10-20T09:24:40+08:00">
   2018年10月20日 09:24
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <span class="dateline">
  洛杉矶 —
 </span>
 <p>
  一个是在韩国出生长大的亚裔移民女性，一个是在2010年赢得奖金2亿6千多万美金彩票的慈善家，今年美国国会中期选举中，角逐加州第39选区联邦众议员席位的两位候选人有着吸引媒体版面的特殊背景。不过，要赢得这场竞争激烈的选举，抢攻移民的选票是他们的决胜关键。
  <br/>
  <br/>
  来自韩国的金映玉（Young Kim）代表共和党角逐加州第39选区联邦众议员席位。
  <br/>
  <br/>
  作为一名亚裔、移民、女性候选人，金映玉的这三种特色似乎并非典型“共和党”。金映玉说，正是她与众不同的经历使得她更加了解这个选区的需要。
  <br/>
  <br/>
  她说：“我个人和我的背景给了我独特的观点。我不需要再上一堂特殊的地缘政治课来了解亚洲对美国来说有多么重要，亚洲的贸易伙伴有多重要。”
  <br/>
  <br/>
  金映玉在韩国出生长大，初中在关岛完成，高中在夏威夷就读，一直到大学才搬到南加州。
  <br/>
  <br/>
  她所在的这个选区是全美族群最多元的选区之一。亚裔、拉美裔和白人各占了三分之一。来自民主党的竞争对手、2010年彩票得主吉尔·西斯尼洛斯（Gil Cisneros）也深刻了解选区里多元族裔所代表的意义，要赢得选举，他必须努力与各社区接触。
  <br/>
  <br/>
  和对手相比，西斯尼洛斯少了一项亚裔的背景，但是，他的竞选团队中多了不同族裔的面孔。
  <br/>
  <br/>
  他说：“我们会走出去，用多种语言和民众沟通。如果你来到我们的办公室，你会听到汉语、越南语、卡塔洛语，你能听到西班牙语，英文，你甚至能听到一些粤语。我们和每一个人交谈。”
  <br/>
  <br/>
  亚太裔是美国成长最迅速的少数族群，选民对参与政治的热情也与日俱增，这使得民主共和两党开始重视亚太裔选民，特别是在这个亚太裔已非少数族裔的选区。
  <br/>
  <br/>
  金映玉说：”我理解我们这个选区里充满了组成我们这个伟大国家的多样性。我们称这个地方为环太平洋的门户，所以选民要找一个了解这些议题的代表，而我一直都致力于关注这些议题。”
  <br/>
  <br/>
  西斯尼洛斯说：“搬进这个选区的移民人口正改变这个选区的人口结构，这是一个以少数族裔成为多数的选区，大量的拉美裔、亚裔人口。当这样的情况发生时，这个选区就变得更偏向于民主党，这也随着时间在发生变化。”
  <br/>
  <br/>
  据“亚太裔美国人投票促进会”(Asian and Pacific Islander American Vote)调查显示，亚太裔选民最关注的议题依序为经济、健保、教育和枪支管控。
  <br/>
  <br/>
  西斯尼洛斯认为，无论什么背景和种族，移民关心的问题都是一样的。
  <br/>
  <br/>
  他说：“无论你的背景是什么，来自什么文化，你担心的就是你的医疗保险，担心共和党是否会把我的社保福利金拿走，是否会拿走我的医疗保险。这些是民众讨论的，也是他们为什么支持我的原因。”
 </p>
 <p>
  一般传统认为，美国的亚太裔移民对参与政治较为冷感，但今年的选情发生了明显变化。无党派组织“亚太裔美国人投票促进会”10月份所公布的民调发现，亚太裔选民对2018年中期选举，展现了高于以往的热情。
  <br/>
  <br/>
  加州大学洛杉矶分校政治系助理教授娜塔莉·增岗说：“相较于其他族群，亚裔美国选民通常比较认为他们是独立或非党派倾向。基于这个原因，亚裔美国选民也通常较易于成为潜在或摇摆选民，他们更有可能被影响，变得倾向于民主党或共和党。”
  <br/>
  <br/>
  金映玉说：”亚裔美国人开始意识到，与其躲在后面什么都不做，怨天尤人，我们现在必须主动参与，没有其他人可以为我们做决定，我们也不会再继续不停抱怨，我们要一起参与其中，一起做出改变。”
  <br/>
  <br/>
  这个选区过去四十年多来一直由共和党人主控，众议院外交委员会主席埃德·罗伊斯今年稍早宣布退休，两党候选人争抢他空出来的席位。民主共和两党都看准了亚太裔选民对政治的开放程度，积极抢攻移民选票。这个选区成为了本届国会中期选举中两党投入资金最多的选区之一。显然，在这个战场州，移民已经成为了两党的决胜关键。
  <br/>
  <br/>
 </p>
</div>
