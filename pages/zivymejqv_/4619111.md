### 国际刑警组织拒邀台湾以观察员身份参与年会 
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-10-19T00:54:02+08:00">
   2018年10月19日 00:54
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <span class="dateline">
  华盛顿 —
 </span>
 <p>
  据中央社报道，台湾外交部星期四表示，国际刑警组织秘书长办公室回覆台湾申请以观察员身分参与大会，拒绝邀请台湾，称“中国是唯一代表”。台湾外交部表达强烈不满和遗憾。
  <br/>
  <br/>
  报道表示，国际刑警组织第87届大会11月18到21日将在阿联酋的迪拜召开。台湾内政部警政署9月致函国际刑警组织秘书长史托克，申请以观察员身分出席，希望有意义参与各项活动及机制，包括使用全球警察通讯系统。
  <br/>
  <br/>
  外交部发言人李宪章表示，台湾争取参与国际刑警组织的诉求具有正当性与必要性，已获友邦和美国等理念相近国家的支持。台湾政府将持续争取国际支持，呼吁国际刑警组织尽快就台湾参与做出务实可行的安排，以确保全球犯罪防制体系不会出现漏洞。
  <br/>
  <br/>
  台湾曾是国际刑警组织成员，但中国1984年在第53届大会上获准加入后，希望将中华民国更名为“中国台湾”，台湾拒绝接受而退出。
  <br/>
  <br/>
  国际刑警组织总部设在法国里昂，前身为1923年成立的国际刑事警察委员会，1956年改为现名，是促进全球警察合作、分享信息，支援预防及打击犯罪的一个平台，现有192个成员国。
 </p>
