 【大纪元2019年01月10日讯】
</p>
<p>
 天灭中共必有方，神助人力倒红墙。
 <br/>
 四面路阻难抬脚，八方烽火火正旺。
 <br/>
 调兵遣将剿红魔，且看神佛主天纲。
 <br/>
 恶做绝时因果报，费尽心思也空忙。
 <br/>
 中共犯下滔天罪，亿万冤魂悲声扬。
 <br/>
 五千文明一脚踹，神传文化一扫光。
 <br/>
 西来幽灵堂上供，欺宗灭祖断炎黄。
 <br/>
 黄赌毒假齐泛滥，打压正信丧天良。
 <br/>
 人无善念坑拐骗，道徳沦丧乱纲常。
 <br/>
 欲将中华带何处？毁灭人类祸心藏。
 <br/>
 法徒慈悲劝
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%89%E9%80%80.html">
  三退
 </a>
 ，恶世救人百苦尝。
 <br/>
 也有精英直言谏，机会给你险自扛。
 <br/>
 手握权柄本天赋，正用邪用当思量。(注1)
 <br/>
 历史关头选去留，一念成败主存亡。
 <br/>
 堂堂戈尔等你做，
 <a href="http://www.epochtimes.com/gb/tag/%E8%A7%A3%E4%BD%93%E4%B8%AD%E5%85%B1.html">
  解体中共
 </a>
 有神帮。(注2)
 <br/>
 振擘一呼万众应，名垂青史万古芳。
 <br/>
 神佛眷顾传天意，指条大路通天畅。
 <br/>
 走与不走自做主，神看人心不勉强。
 <br/>
 恶念一生下地狱，善念一出步天堂。
 <br/>
 天堂地狱咫尺间，只上莫下休彷徨。
</p>
