### 云鹤天：忆青春
------------------------

<p>
 【大纪元2018年10月23日讯】
</p>
<p>
 小学
 <br/>
 小学上学没书读
 <br/>
 老师油印做成书
 <br/>
 三天两头去游行
 <br/>
 经常开会斗地主
 <br/>
 三餐汤水无干粮
 <br/>
 家徒四壁茅草屋
 <br/>
 破衣烂衫鞋露趾
 <br/>
 天天要跳忠字舞
</p>
<p>
 中学
 <br/>
 中学徒步去县城
 <br/>
 早晚两头见星星
 <br/>
 两个窝头手绢包
 <br/>
 仅有干粮给学生
 <br/>
 今天工厂去学工
 <br/>
 明天下乡去学农
 <br/>
 中学几乎不上课
 <br/>
 说是教育要革命
</p>
<p>
 大学
 <br/>
 恢复高考上大学
 <br/>
 离开农村心愉悦
 <br/>
 一日三餐一斤粮
 <br/>
 饿得前心后心贴
 <br/>
 弟弟老山去打仗
 <br/>
 复员回来手残缺
 <br/>
 父母抚摸儿残臂
 <br/>
 老泪纵横心滴血
</p>
<p>
 今天
 <br/>
 如今退休回到家
 <br/>
 两鬓斑白添银发
 <br/>
 妹妹去年癌症走
 <br/>
 癌症村里新坟加
 <br/>
 弟弟昨天去维权
 <br/>
 头破血流满脸花
 <br/>
 掩卷举目望窗外
 <br/>
 恶藏灯红酒绿下
</p>
