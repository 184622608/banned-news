### 鲁泰：民族之栋梁
------------------------

<p>
 【大纪元2018年10月29日讯】
</p>
<p>
 八十四岁，教授文广。
 <br/>
 一直敢言，正直脊梁。
 <br/>
 文广文广，齐鲁栋梁。
 <br/>
 老而凛凛，威风浩荡。
 <br/>
 与老为敌，红朽木亡。
 <br/>
 国香桂香，七十四双。
 <br/>
 同籍德州，同修德康。
 <br/>
 同遭迫害，同法一乡。
 <br/>
 凌雪寒梅，坚定溢香。
 <br/>
 疾病全无，衰身复康。
 <br/>
 江蟾错位，横逆肮脏。
 <br/>
 法轮大法，世界之光。
 <br/>
 各族受益，百国洪扬。
 <br/>
 尊老爱幼，人间正常。
 <br/>
 唯党利大，尽毁朝纲。
 <br/>
 内斗内斗，惨烈异常。
 <br/>
 宗庙文革，祖变奶娘。
 <br/>
 战天战天，党自飙狂。
 <br/>
 婴儿毒奶，疫苗殇殇。
 <br/>
 无德无德，家庭跄跄。
 <br/>
 幸遇九评，开我心房。
 <br/>
 三退三退，神清气爽。
 <br/>
 洞彻红朝，毁人超狼。
 <br/>
 终极终极，目的真相。
 <br/>
 知之得救，择善久长。
 <br/>
 传真传真，乾坤朗朗。
</p>
<p>
 注：“国香桂香”指山东德州市法轮功学员
 <a href="http://www.epochtimes.com/gb/tag/%E5%88%98%E6%A1%82%E9%A6%99.html">
  刘桂香
 </a>
 和
 <a href="http://www.epochtimes.com/gb/tag/%E9%AB%98%E5%9B%BD%E9%A6%99.html">
  高国香
 </a>
 。
</p>
<p>
 （明慧网通讯员山东报导）二零一八年九月二十四日，是中国传统节日——中秋节，也是亲人团聚的日子。而中共邪党法院却对坚持信仰修炼法轮功、现年七十四岁的
 <a href="http://www.epochtimes.com/gb/tag/%E9%AB%98%E5%9B%BD%E9%A6%99.html">
  高国香
 </a>
 老太太非法判刑四年，罚款二万元。
</p>
<p>
 山东省德州市陵城区法轮功学员高国香本应该和其他老人一样，享受天伦之乐，度过幸福的晚年，可是她却心系那些仍被中共谎言毒害、不明真相的中国同胞，希望他们得到法轮大法的救度，远离邪党，退党保命。（节选）
</p>
