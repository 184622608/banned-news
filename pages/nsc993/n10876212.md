 【大纪元2018年11月27日讯】
</p>
<p>
 加拿大，来牵头。欧盟澳，把信投。
 <br/>
 陈全国，疆党酋。集维人，中营沟。
 <br/>
 锁定陈，各国揪。藏惊天，食人幽。
 <br/>
 全国虎，蟾帮瘤。红潮泛，经地髅。
 <br/>
 左盖头，画白骨。活摘明，暗惑诱。
 <br/>
 器官卖，钱肥流。一带路，建红楼。
 <br/>
 小国套，白狼畴。大国钻，主流垢。
 <br/>
 苏一百，未停偷。僵统战，尸魔纠。
 <br/>
 毛邓江，习难收。何近平，马下勾。
 <br/>
 古到今，传统流。三尺头，神划猷。
 <br/>
 西死豹，东皮裘。东西难，六四蚪。
 <br/>
 法轮功，九二授。得者益，日众酬。
 <br/>
 三代妒，九头兽。中西谶，恐怖厚。
 <br/>
 十九年，善恶仇。秦城糠，癌殃喉。
 <br/>
 夕日薄，大连苟。苏家屯，集中钩。
 <br/>
 骨渣蝇，神鬼抽。梅花五，夏雪后。
 <br/>
 秦皇岛，看守所。马桂兰，大法徒。
 <br/>
 开膛后，内脏取。一八秋，人祸巨。
 <br/>
 死后生，西耶稣。天滴露，菩萨柳。
 <br/>
 九评醒，三退友。三亿增，方舟救。
 <br/>
 明共的，脱难走。陆驱鬼，万王佑。
 <br/>
 五千史，传统纽。神韵复，中土秀。
