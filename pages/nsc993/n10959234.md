### 理悟：北大教授为习近平指路
------------------------

<p>
 【大纪元2019年01月08日讯】
</p>
<p>
 山穷水尽无生门，教授指路有阳春。
 <br/>
 不知当政可入耳？金玉良言无价珍。
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E5%8C%97%E5%A4%A7%E6%95%99%E6%8E%88.html">
  北大教授
 </a>
 <a href="http://www.epochtimes.com/gb/tag/%E9%83%91%E4%B9%9F%E5%A4%AB.html">
  郑也夫
 </a>
 ，双目如电洞古今。
 <br/>
 一针见血谈政改，敢为民言见慈心。
 <br/>
 中共乱政七十载，遗害无穷种祸根。
 <br/>
 纠错机制全丧失，结束专制利万民。
 <br/>
 历史舞台虽壮美，非供红魔舞妖群。
 <br/>
 鼓漏弦断该退场，引领淡出福子孙。
 <br/>
 避免暴力和平度，最少动荡社会稳。
 <br/>
 奇功一件载史册，万般和顺天地人。
 <br/>
 舒怀言志安危抛，教授风范中华魂。
 <br/>
 该听不听机缘过，身败名裂自悔恨。
 <br/>
 有道中共权钱握，屠刀在手人性泯。
 <br/>
 历史舞台怎肯退？权贵利益令头昏。
 <br/>
 水能载舟亦覆舟，民已觉醒无可忍。
 <br/>
 弃党三退三亿二，民心所向定乾坤。
 <br/>
 天灭中共是必然，幻想保党自欺心。
 <br/>
 生死存亡天有道，自选位置慎思忖。
</p>
<p>
 题注：【大纪元2019年01月03日讯】（大纪元记者梁义报导）近日，网上传出北大知名教授
 <a href="http://www.epochtimes.com/gb/tag/%E9%83%91%E4%B9%9F%E5%A4%AB.html">
  郑也夫
 </a>
 分析中共政改为何难产的文章。文章表示，政改从未迈出一步，是因为执政党魁意识到，政改的每一项都是在削弱其政党，并直指当今的领导人应引领其党淡出历史舞台，结束专制符合广大人民的利益。
</p>
<p>
 文章还说，经济体制改革之前，之后，中共的大多数方针政策都很难说是符合广大民众的根本利益的，但当前有一项符合中国人民共同利益的事情，就是共产党“和平地“，即以避免暴力的、最少社会动荡的方式，淡出历史舞台。
</p>
<p>
 作者认为，今后中共的领导人所能做的唯一可望载入史册的大事情，就是引领这个党淡出历史舞台。
</p>
<p>
 作者直言，在中共执政的70年历史中，这个党给中国人民带来太多的灾难。演化到今天，其权力的结构和生态决定了，它已经不能为中国社会输送优秀的各级领导者，它几乎完全丧失了自我纠错的机制。结束专制符合中国广大人民的利益。
</p>
