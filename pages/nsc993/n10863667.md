### 万古缘：人莫贪
------------------------

<p>
 【大纪元2018年11月20日讯】
</p>
<p>
 <a href="http://www.epochtimes.com/gb/tag/%E8%B4%AA.html">
  贪
 </a>
 财使人丑，
 <br/>
 魅鬼牵着黑心走。
 <br/>
 虎口臭，
 <br/>
 脸皮厚，
 <br/>
 梦里紧把钱袋搂。
 <br/>
 死也不撒手。
</p>
<p>
 <a href="http://www.epochtimes.com/gb/tag/%E8%B4%AA.html">
  贪
 </a>
 色使人丑，
 <br/>
 淫鬼牵着魂儿走。
 <br/>
 风雨骤，
 <br/>
 赛禽兽，
 <br/>
 色胆包天持多久？
 <br/>
 钢刀断贼首。
</p>
<p>
 贪杯使人丑，
 <br/>
 酒鬼牵着命儿走。
 <br/>
 喝不够，
 <br/>
 肝坏透，
 <br/>
 乱伦乱性乱插手。
 <br/>
 地狱不如狗。
</p>
<p>
 贪欲使人丑，
 <br/>
 改邪归正大道走。
 <br/>
 人长寿，
 <br/>
 作良莠，
 <br/>
 除恶扬善去腐朽，
 <br/>
 天地福分有！
</p>
