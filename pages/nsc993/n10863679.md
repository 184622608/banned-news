### 伍新：还牙改送牙 纸虎惧拔牙
------------------------

<p>
 【大纪元2018年11月20日讯】
</p>
<p>
 中美贸战打，
 <br/>
 张口牙还牙。
 <br/>
 满嘴金牙假，
 <br/>
 闭口改送牙。
 <br/>
 还牙说不屑，
 <br/>
 报告呈纸牙。
 <br/>
 纸牙鼠磨牙，
 <br/>
 不嫌酸倒牙。
 <br/>
 舞爪狂张牙，
 <br/>
 真怕真还牙。
 <br/>
 邪从不压正，
 <br/>
 纸虎惧拔牙。
 <br/>
 虽仍闲磕牙，
 <br/>
 足令笑掉牙。
</p>
<p>
 注：据新唐人北京时间2018年11月18日讯，大陆新闻解读（578）【严真点评】
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1.html">
  中共
 </a>
 变乖巧了？
</p>
<p>
 向美提交经济改革书面报告：那个要以牙还牙，以眼还眼的厉害国在姿态上乖巧了许多。
</p>
<p>
 虽然向美国提交的这个改革计划书，不可能满足川普提出的所有要求，因为那将动到它所谓的“核心利益”。但就这个乖巧的姿态本身，恐怕将成为
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1.html">
  中共
 </a>
 滑稽表演史上，比较有娱乐“笑”果的一个桥段。
</p>
