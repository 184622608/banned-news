### 甘楠：红蓝青山 谁更霁霁
------------------------

<p>
 【大纪元2018年11月06日讯】
</p>
<p>
 一九二八，湘鄂血流。红染山河，代代囚囚。
 <br/>
 一九四九，古谶囚幽。蛰起赤龙，陆亿髅髅。
 <br/>
 金门史记，两岸猿啼。传统行舟，一脉继继。
 <br/>
 史记金门，毛炮轰始。红蓝青山，谁更霁霁。
 <br/>
 台湾商品，享誉国际。诚信为本，收获累累。
 <br/>
 台湾信誉，无需诋毁。功到自然，红潮诡诡。
 <br/>
 大家都懂，休内安外。进化华为，晋华水水。
 <br/>
 大家皆懂，谁是痞子。无神偷心，阴暗鄙鄙。
 <br/>
 双十一路，一百零七。血与火炼，善恶笔笔。
 <br/>
 红朝一路，民族痛疾。危害全球，真相怵怵。
 <br/>
 一路民国，台湾绮丽。何止信誉，德惠仔仔。
 <br/>
 一路红朝，亿亡血洗。礼崩乐坏，老无委委。
 <br/>
 中山蒋公，前仆后继。传统秉承，后果喜喜。
 <br/>
 毛鳖江蟾，前标后傀。宗庙皆砸，罪恶垒垒。
 <br/>
 中山蒋公，经国福祉。一蓝一绿，艾灸芷芷。
 <br/>
 毛鳖江蟾，卖国红史。一断一贪，五岳蠡蠡。
 <br/>
 法轮长春，功高盖世。福泽众生，百国比比。
 <br/>
 蟾迫法轮，法徒难系。囹圄活摘，器官集集。
 <br/>
 时间越久，凌寒香弥。普世价值，普天垂垂。
 <br/>
 越久时间，曝党终极。毁人洪水，猛兽蚀蚀。
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E5%85%A8%E7%90%83%E6%B8%85%E5%85%B1.html">
  全球清共
 </a>
 ，顺天之美。华夏子孙，斧钺旗旗。
 <br/>
 台湾漂红，真言三字。广传九评，世界期期。
 <br/>
 清共全球，毁人主义。终极目的，务必知知。
 <br/>
 漂红台湾，大陆驱鬼。三退潮尾，清风吹吹。
</p>
