### 冀旦：一带本应有路
------------------------

<p>
 【大纪元2018年11月29日讯】
</p>
<p>
 粮本饥肠温，穿衣布票存。社单商品票，火热资台昏。
 <br/>
 现代化空想，承包产党恩。只因六四制，诱惑开红门。
 <br/>
 经济外商冲，人民私有饨。香蕉干劲起，豆腐饱吃盆。
 <br/>
 口欲美中美，钱迷浑上浑。不分猫黑白，难识山中荪。
</p>
<p>
 电话手机呼，汽车高速闻。舞厅夜市现，共企酒鸡熏。
 <br/>
 老大地盘肉，股东党代群。公房私有变，全部党囊荤。
 <br/>
 生产唯偷能，环污成本坟。访民开发巨，娱乐补红纷。
 <br/>
 农舍空空野，劳工厂厂勤。剩余价值榨，代表肚肥醺。
</p>
<p>
 物欲横流中，迷失方向寻。身心劳损大，道德滑危森。
 <br/>
 大法长春始，北京中南针。法徒日日众，得道人人心。
 <br/>
 妒火泽民烧，祚庥罗干襟。自焚焦点祸，伪火浇油淋。
 <br/>
 产业器官摘，白衣尸体侵。江包胡习接，渎职位权今。
</p>
<p>
 闻报秦皇岛，法徒马桂兰。开膛看守所，内脏医公安。
 <br/>
 案例多多叠，含冤隐隐瞒。二零零二案，张市润娟叹。
 <br/>
 请愿北京至，广州夺命残。皮肤被剥掉，内脏全掏剜。
 <br/>
 双目被挖掉，一堆尸骨滩。非人底线党，一路马毛肝。
</p>
<p>
 劳教所终寿，丝汗一带绸。虎蝇公审透，蟾蟹解西幽。
 <br/>
 天赐归天路，马迷入马沟。全球清共美，川普兵团优。
 <br/>
 贸战路通畅，顺天行道筹。九评三退涌，目的终极流。
 <br/>
 马恩毁人列，斯毛蛤习骝。天堂有路一，神韵巡回修。
</p>
<p>
 注：“润娟”指——河北省张家口市法轮功学员
 <a href="http://www.epochtimes.com/gb/tag/%E9%83%9D%E6%B6%A6%E5%A8%9F.html">
  郝润娟
 </a>
 （女），家住广州白云区。因四次到北京为法轮功请愿被抓，2002年遭二十二天
 <a href="http://www.epochtimes.com/gb/tag/%E9%85%B7%E5%88%91.html">
  酷刑
 </a>
 摧残后死在广州白云看守所。
</p>
<p>
 家属去认尸时，见遗体已面目皆非，内脏全掏空，皮肤被剥光，眼睛被挖掉，只剩下一堆尸骨肉，还带有鲜红的血迹。
</p>
<p>
 两次看过遗体后，家属认为那不是
 <a href="http://www.epochtimes.com/gb/tag/%E9%83%9D%E6%B6%A6%E5%A8%9F.html">
  郝润娟
 </a>
 ，只好把她两岁的儿子带来验血，最后证实那是郝润娟。摘自大纪元报导“法轮功学员马桂兰被
 <a href="http://www.epochtimes.com/gb/tag/%E8%BF%AB%E5%AE%B3%E6%AD%BB.html">
  迫害死
 </a>
 内脏被取走”
</p>
