### 伍新：解“自力更生”
------------------------

<p>
 【大纪元2018年09月28日讯】【解体
 <a href="http://www.epochtimes.com/gb/tag/%E5%85%9A%E6%96%87%E5%8C%96.html">
  党文化
 </a>
 】之六：习惯了的党话（上）：中共盗用国家政权对民族文化和语言的深刻伤害，可以说前无古人。语言是思维的工具。被党文化严重污染的语言，严重地损害了人们反思中共、反思党文化、构思民族未来的能力。很多人都发现一个奇特的现象：人们用中共制造的语言批判中共，愤怒声讨中共的文章仍然称中共建政为“解放”，有人在“退党声明”中仍然说“我是‘生在新中国，长在红旗下’的。”让人简直难以分辨他到底是要唾弃中共，还是要感激中共。在中共政权摇摇欲坠、人民亟待回归正常人类文化的今天，认清附着在民族语言上的党话，清除党话，已经成为刻不容缓的任务。
</p>
<p>
 ——题记
</p>
<p>
 中美贸战，越打越怂。自产大豆，惨遭霜冻。
 <br/>
 天灭中共，处处逼命。党魁复哼，自力更生①。
 <br/>
 万般无奈，万分悲鸣。自力更生，成语简明。
 <br/>
 凭己之力，更新运程。用作党话，压根欺蒙。
 <br/>
 西来邪灵，两手空空。从无到有，总吹自能。
 <br/>
 从小到大，全算党功。自自是党，并非民众。
 <br/>
 自力假意，党不下力。一路杀骗，流氓耍惯。
 <br/>
 更生更伪，为把人毁②。延安本穷，红寇突增。
 <br/>
 一群蝗虫，啃苦百姓。雷霹县头，农妇发生。
 <br/>
 雷怎不去，霹毛泽东？言出被抓，手下逞凶。
 <br/>
 毛贼窝火，强忍吞声。恐激民反，自身无容。
 <br/>
 假扮笑脸，买心松绳。惊魂难定，夜做噩梦。
 <br/>
 再生奸计，挥笔题词：自己动手，丰衣足食。
 <br/>
 兵不抗日，生产运动。说是自力，全是壮丁。
 <br/>
 抓来骗来，为党打工。大面生产，鸦片业隆。
 <br/>
 南泥湾里，罂粟花盛。自力更生，韬光窑洞。
 <br/>
 邪党自力，一味扩充。抗日末尾，毛张大嘴。
 <br/>
 自力更生③，磨刀篡位。自力何使？内战挑之。
 <br/>
 更生谁更？红旗舞疯。夺得大权，洗脑门关。
 <br/>
 连番运动，血雨腥风。跃进大闹，饥荒虐暴。
 <br/>
 六亿肚叫，四千万殍。谎称天灾，罪责推掉。
 <br/>
 当时实景，风顺雨调。自力更生，口号高声。
 <br/>
 农学大寨，工学大庆。党为称雄，输出革命。
 <br/>
 亚非拉美，外援不停。赶造核弹，霸王强弓。
 <br/>
 自力更生，自力索命。自力亦假，贪人之功。
 <br/>
 两弹一星，专家现成。民国培养，诱逼入营。
 <br/>
 更生何见？举国寒饥。党力自增，躲过险关。
 <br/>
 自力更生，借力诓情。爱国主义，恨外为据。
 <br/>
 实无所爱，党偷索爱④。邪灵附体，寄生借力⑤。
 <br/>
 自力更生，今非昔同。自力哪借，谁还卖命？
 <br/>
 十官九贪，一行不通。专家队伍，砖家走红。
 <br/>
 裸官流长，移民潮汹。怨声载道，抗暴浪高。
 <br/>
 日暮途穷，借力逃生。逃乃妄念，也难苟延。
 <br/>
 在遭天谴，谁又能拦？九评⑥扒衣，巨著⑦起底。
 <br/>
 真相看明，众叛亲离。三退⑧先驱，早超三亿。
 <br/>
 良知快醒，勿失良机。
</p>
