### 海网：一洋破鞋怎合重脚？
------------------------

<p>
 【大纪元2018年12月17日讯】
</p>
<p>
 <a href="http://www.epochtimes.com/gb/tag/%E5%BE%B7%E5%9B%BD%E6%80%BB%E7%BB%9F.html">
  德国总统
 </a>
 演讲稿，
 <br/>
 理论自信不让瞧。
 <br/>
 内紧外松佯无视，
 <br/>
 紧报删帖封牢牢。
 <br/>
 最知
 <a href="http://www.epochtimes.com/gb/tag/%E9%A9%AC%E5%85%8B%E6%80%9D.html">
  马克思
 </a>
 底细，
 <br/>
 主义浩劫害乡里。
 <br/>
 驱逐出国亡命徒，
 <br/>
 孤魂漂泊野鬼族。
 <br/>
 脚印西德从无一，
 <br/>
 全德今都当敝履①。
 <br/>
 鞋子合脚不合脚，
 <br/>
 自己穿着才知道②。
 <br/>
 只是党妈无可告，
 <br/>
 一洋破鞋合众脚？
 <br/>
 缠足残女批声嚎，
 <br/>
 红朝男女足统削。
 <br/>
 德人甩弃破马靴，
 <br/>
 生穿硬套称合脚。
 <br/>
 马屎③更当马奶酪，
 <br/>
 兑水强灌熏洗脑。
 <br/>
 只是末日正临近，
 <br/>
 裸官移民脚声悄。
 <br/>
 更有漂红复天足，
 <br/>
 弃鞋三退④党远抛。
 <br/>
 残脑跛脚破鞋路，
 <br/>
 尽头唯余恶者倒。
</p>
<p>
 注：
</p>
<p>
 1、据大纪元2018年12月09日讯，从12月5日开始，
 <a href="http://www.epochtimes.com/gb/tag/%E5%BE%B7%E5%9B%BD%E6%80%BB%E7%BB%9F.html">
  德国总统
 </a>
 施泰因迈尔（Frank-Walter Steinmeier）正在中国进行为期5天的国是访问，昨天（12月7日）他去了四川大学并发表了演讲。路透社从德国大使馆那里得到了一份施泰因迈尔的演讲稿，里面显示他提醒中国年轻人，“
 <a href="http://www.epochtimes.com/gb/tag/%E9%A9%AC%E5%85%8B%E6%80%9D.html">
  马克思
 </a>
 主义曾给德国带来浩劫”。施泰因迈尔并没有在演讲中就具体问题对中共政府提出批评，但他提到了东德的历史，同样是没有自由，充满了压迫。他说，“正因为如此，我们对那些持非主流观点、属于少数民族或实践个人信仰的人所受到的待遇非常敏感和关注。无论哪里出现个人权利受到限制的问题，德国都会感到担忧和不安。”
</p>
<p>
 旅居德国的社会学家王容芬曾去马克思的墓地参观，她发现墓碑上这么写着：“马克思1845年被德国驱逐，亡命布鲁塞尔、巴黎，最后落脚伦敦⋯⋯100年间，德国没纪念过这个亡命徒”，“西德依旧没马克思一个脚印儿”。
</p>
<p>
 这就是马克思家乡人对它的基本评价。王容芬说，“马克思实在不是个东西，不仅把燕妮•马克思坑得太惨，还把燕妮带过去的女仆搞大了肚子，生下孩子又不认账。乱七八糟的。”
</p>
<p>
 2、据新唐人2013年03月25日讯，23号，中共总书记习近平在俄罗斯发表演讲宣称：鞋子合不合脚，自己穿着才知道，一个国家的发展道路，只有这个国家的人民才知道。这被外界认为，是习近平在为中共一党专制作辩护。很多大陆网民质疑这一“鞋子合脚论”，反过来说：“不合脚的鞋子，还要坚持穿，这才是最痛苦的！”
</p>
<p>
 美国中文杂志《北京之春》主编胡平向《新唐人》表示，“鞋子合不合脚，穿着才知道。”这句话本来是西方的谚语，常常被引用来论证“主权在民”。
</p>
<p>
 《北京之春》杂志主编胡平：“现在
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1%E4%B8%93%E5%88%B6.html">
  中共专制
 </a>
 的问题是，它一贯冒充人民、代表人民，在那里发号施令。所以习近平举这句话，完全是牛头不对马嘴。他不知道从哪听来一些东西，这句话本身是为民主辩护、是反对专制的。”
</p>
<p>
 3、马屎：马克思的作品。“据我所知，在著名作家中，只有马克思称自己的作品为‘屎’、‘污秽之书’。他自觉、蓄意地将秽物给予他的读者。怪不得他的某些信徒，比如罗马尼亚和莫桑比克共产党，强迫囚犯们吃自己的屎尿。“（理查德•沃姆布兰德《卡尔•马克思的成魔之路》）
</p>
<p>
 “马克思成魔之路现在也并非什么秘密，马克思的这些作品、通信和西方学者提供的大量考证都是公开的，只是共产党国家故意忽视罢了。据《马克思与撒旦》（Marx and Satan）一书的作者理查德•沃姆布兰德（Richard Wurmbrand）说，他曾联系过莫斯科马克思学院，被告知马克思的作品共有100卷之多，其中只有13卷被公开印发。”（《共产主义的终极目的——中国篇》第二章）
</p>
<p>
 对于这些（包括对于马克思把自己的作品称为“屎”、“污秽之书”），现在的中共不止“故意忽视”，而且在极力封锁。
</p>
<p>
 4、三退：用真名或化名退出中国共产党的党、（共青）团、（少先）队邪教组织。有关资料显示，截止到2018年3月23日，在大纪元退党网站（http://tuidang.epochtimes.com/）声明“三退”总人数突破三亿。截至2018年12月15日，三退人数已超过三亿二千二百二十多万。
</p>
<p>
 “‘青山遮不住，毕竟东流去。’邪不胜正，看似猖獗的所有邪恶表象都是暂时的，一切都掌握在神的手中。2004年，《大纪元时报》发表的《九评共产党》开启了中国的‘三退’大潮，数亿人退出了中共的党、团、队组织，这是中国人驱除共产邪灵附体的自救之举。人只要主动‘三退’，神就会将邪灵附体瞬间清除，这个生命就将属于未来！”
</p>
<p>
 “解体共产邪党，清理人间的共产主义邪恶因素，全面反思近二百年来人类社会的堕落和魔变，成为今天人类的当务之急。归正人心，净化社会，回归传统，重建信仰，重新体认与神的联系，找回与神的纽带，这是每个人的责任，也是每个人得救的希望所在！神的慈悲与威严同在！神在看着每个人的内心。一个人在此时此刻的抉择和所为，就会决定他（她）的未来。”（《共产主义的终极目的——中国篇》结语）
</p>
