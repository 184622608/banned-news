### 甘楠：又到猪年
------------------------

<p>
 【大纪元2019年01月01日讯】
</p>
<p>
 三皇五帝，二十五史。老子孔子，百家子子。
 <br/>
 三才五行，诸佛慈悲。十方世界，道法济济。
 <br/>
 太平御览，通鉴资治。永乐大典，皇家之之。
 <br/>
 太平广记，有神宗祠。四库全书，钦定期期。
 <br/>
 文以载道，虞舜孝悌。炎黄春秋，孟颜师师。
 <br/>
 德以配位，禅让禹启。五千传承，神州仪仪。
 <br/>
 二百年前，撒旦马厮。社会共产，恶行鬼鬼。
 <br/>
 一百年前，列宁狞厉。催生中共，全民饥饥。
 <br/>
 跃进地狱，饿殍野遗。土灶钢渣，草民离离。
 <br/>
 文革浩劫，砸庙像推。毛不万岁，无主痴痴。
 <br/>
 八十年代，四化大旗。肚皮刚饱，天门洗洗。
 <br/>
 九十年代，猫领腐败。公变私有，江帮疥疥。
 <br/>
 股市房市，银行蟾资。电信江绵，一贪丕丕。
 <br/>
 闹市集市，江家画地。粮田建厂，蝌蚪滋滋。
 <br/>
 法轮清流，沁人心脾。道德回升，明善追追。
 <br/>
 功法健身，内外兼修。旭日东升，光明洲洲。
 <br/>
 蟾王妒嫉，抓狂迫害。自焚活摘，器官卖卖。
 <br/>
 六一零办，公检法司。政法委干，为牢自自。
 <br/>
 飘洋移植，世贸钱催。海伍志军，画血帔帔。
 <br/>
 过海移植，中兴华为。晚舟开来，鬼妖姬姬。
 <br/>
 雪亮工程，民血民脂。集中器官，何止维维。
 <br/>
 民心雪亮，“钱袋子”吹。党喉毒霾，吹牛皮皮。
 <br/>
 川普行者，七寸点指。看似随意，党却衰衰。
 <br/>
 川普团队，火眼目垂。习江分清，党终刈刈。
 <br/>
 天剑九评，三退潮起。三亿有二，尽收尾尾。
 <br/>
 巨著终极，魔鬼透视。截至毁人，神箭矢矢。
 <br/>
 清清小河，彼自天来。绿树成荫，圣主栽栽。
 <br/>
 唯唯神韵，此花在开。修复传统，六出苔苔。
</p>
