### 苏醒：他和她均能上中加史书
------------------------

<p>
 【大纪元2018年12月26日讯】
</p>
<p>
 美澳新后，日本筹谋。拆机华为，零件暗沟。
 <br/>
 多余零件，晚舟莫羞。世界已醒，魔鬼在偷。
 <br/>
 中兴华为，地下洞蚯。引尔跟进，红霾粒酬。
 <br/>
 此引非蚓，彼君根逑。生机坤图，盎然山畴。
 <br/>
 苹果伊甸，撒旦蛇勾。马厮拜之，列斯毛收。
 <br/>
 叶落袭秋，红染神州。亿冤亡故，十四亿楼。
 <br/>
 价高十万，八千里抠。机配摄像，一机一手。
 <br/>
 中南视墙，偷窥全球。美盟来袭，诱惑狗狗。
 <br/>
 打压苹果，三星难受。抗韩抗日，按需铜臭。
 <br/>
 加冻公主，党火焚舟。加人陆失，蒸发比牛。
 <br/>
 青牛伴老，文明悠悠。红牛虎头，属蛇尾囚。
 <br/>
 佳人加姐，
 <a href="http://www.epochtimes.com/gb/tag/%E6%9E%97%E8%80%B6%E5%87%A1.html">
  林耶凡
 </a>
 游。生在红潮，成于加眸。
 <br/>
 明辨善恶，大法中修。身心净化，内刚外柔。
 <br/>
 法徒器官，妖党利牟。耶凡从演，揭露西幽。
 <br/>
 影片问世，触目醒洲。血淋淋债，世界马瘤。
 <br/>
 十七年前，牡丹江漏。
 <a href="http://www.epochtimes.com/gb/tag/%E7%8E%8B%E6%99%93%E4%B8%9C.html">
  王晓东
 </a>
 亡，十三天殴。
 <br/>
 三十六岁，肚子瘪掊。像个拉锁，一大刀口。
 <br/>
 活摘案例，多多不否。法徒修善，魔鬼红寇。
 <br/>
 加人
 <a href="http://www.epochtimes.com/gb/tag/%E5%AD%99%E8%8C%9C.html">
  孙茜
 </a>
 ，胡润豪留。法轮功炼，富德行辀。
 <br/>
 二零一七，京警带走。至今关押，多方在救。
 <br/>
 晚舟保释，加人红扣。
 <a href="http://www.epochtimes.com/gb/tag/%E5%BA%B7%E6%98%8E%E5%87%AF.html">
  康明凯
 </a>
 囹，可领红兽？
 <br/>
 加拿大鹅，秋冬装秀。北京店缓，美豆加饾？
 <br/>
 赖昌星昔，红通鸡猴。谁不懂驴，东施甚尤。
 <br/>
 九评天降，醒脑最优。三退方舟，彼名网邮。
 <br/>
 三亿二余，脱党相柳。神韵复开，佳人菅畤。
</p>
<p>
 注：孔壬的封国在雍州以西，孔壬和相柳狼狈为奸，残害生灵，暴虐百姓，凭恃险阻，对抗朝廷，干扰治水。于是禹杀死相柳，缉拿孔壬押解至帝都。摘自“【千古英雄人物】尧舜禹(6) 大禹治黄河”
</p>
