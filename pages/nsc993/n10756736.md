 【大纪元2018年10月03日讯】

 <br/>
 跳楼上官，
 <br/>
 娘子军红专。
</p>
<p>
 八零走穴初偿数钱，
 <br/>
 走入秦城为何遭陷？
 <br/>
 远华红演，
 <br/>
 几泪几落难！
</p>

 <a href="http://www.epochtimes.com/gb/tag/%E5%BA%86%E4%BA%B2%E7%8E%8B.html">

 <br/>
 行宫诸院，
 <br/>
 几多类汤灿。
</p>
<p>
 政法委公安局腐烂，
 <br/>
 几多被跳楼自杀案！
 <br/>
 东生永康，
 <br/>
 鸨母裤衩伞？
</p>
<p>
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BA%AA%E5%A7%94.html">
  中纪委
 </a>
 内鬼多起谳，
 <br/>
 甚至赃款达叉叉万。
 <br/>
 红始淫乱，
 <br/>
 谁在给白眼！
</p>
<p>
 内鬼何止贪官党员，
 <br/>
 毛割外蒙认贼苏联。
 <br/>
 邓六四难，
 <br/>
 卖国梳头蟾。
</p>
<p>
 入主谁人眼泪涟涟，
 <br/>
 政令僵灭不出中南。
 <br/>
 活摘泛滥，
 <br/>
 暴利徒器官。
</p>
<p>
 人皮何祚庥与罗干，
 <br/>
 蟾帮富可敌国惯犯。
 <br/>
 红朝何安？
 <br/>
 红楼迷梦连。
</p>

 <br/>

 <br/>

 <br/>
 毒到家中饭。
</p>
<p>

 <br/>
 洞大赌洞党支共铲。
 <br/>
 云山雾范，
 <br/>
 冰花冰化缘。
</p>
<p>
