### 理悟：“十•一”国人苦难日
------------------------

<p>
 【大纪元2018年09月28日讯】
</p>
<p>
 又逢
 <a href="http://www.epochtimes.com/gb/tag/%E5%8D%81%E4%B8%80.html">
  十一
 </a>
 苦难日，中共邪党头上骑。
 <br/>
 生死存亡红魔管，命运握在党手里。
 <br/>
 毛贼城楼一声喊，灾难降临华夏地。
 <br/>
 血旗飘飘阴风刮，无产专政屠刀利。
 <br/>
 反天反地反人类，杀敌杀友杀自己。
 <br/>
 谁沾邪党谁倒楣，羊入虎口命危矣。
 <br/>
 运动杀人八千万，人心魔变互为敌。
 <br/>
 互相残杀表红心，毛魔装神收渔利。
 <br/>
 为保皇位千年固，发动文革整异己。
 <br/>
 十年浩劫民遭难，杀人如麻当儿戏。
 <br/>
 活人推入天坑中，阴深恐怖不见底。(注1)
 <br/>
 老少不分一勺烩，妇幼难逃论阶级。
 <br/>
 杀人取乐数道县，花样翻新天下奇。(注2)
 <br/>
 惨不忍睹非人为，红魔充胆被附体。
 <br/>
 杀人立威党传统，六四屠城鬼神泣。
 <br/>
 杀人能保江山稳，百姓命贱草不及。(注3)
 <br/>
 最毒当数江蟾蜍，打压大法妖风起。
 <br/>
 打死法徒算自杀，就地火化无痕迹。
 <br/>
 活人开膛挖心肝，器官移植换钱币。
 <br/>
 血腥惨烈十九载，至今杀人仍继续。
 <br/>
 邪党罪恶罄竹难，冤魂讨命声色厉。
 <br/>
 百姓绵羊变醒狮，抗暴连连烽烟起。
 <br/>
 司机罢工潮水涨，老兵维权浪滔急。
 <br/>
 强拆访民不断溜，厕所革命表民意。(注4)
 <br/>
 九评掀起三退潮，三退勇士超三亿。
 <br/>
 四面楚歌红船漏，八方喊打响国际。
 <br/>
 穷途末路眼发蒙，经济崩溃无底气。
 <br/>
 无耐紧掏百姓兜，非洲拉硬大撤币。
 <br/>
 瞻前前头一片黑，顾后后面是绝壁。
 <br/>
 哎哟哟，路在那？天灭中共没有戏。
 <br/>
 怀抱马列不撒手，天网一收下地狱。
 <br/>
 死期将近哪里逃？你命掌在神手里。
 <br/>
 国人莫忘苦难日，解体中共废“
 <a href="http://www.epochtimes.com/gb/tag/%E5%8D%81%E4%B8%80.html">
  十一
 </a>
 ”。
 <br/>
 谁弃邪党谁得救，谁掀红船丰碑立。
</p>
<p>
 注1：【大纪元2018年09月25日讯】1967年10月2日至3日，在不足24小时中，广西桂林地区全州县三江公社民兵营长黄天辉率队将当地93名地富分子推入无底天坑——黄瓜冲。其中一个大地主家族中老少共76人，包括外嫁的女儿及所生的孩子，小的不足1岁，老的65岁，全部被推入坑中。
</p>
<p>
 注2：人民报消息，文革湖南道县
 <a href="http://www.epochtimes.com/gb/tag/%E5%A4%A7%E5%B1%A0%E6%9D%80.html">
  大屠杀
 </a>
 ，从1967年8月13日到10月17日，历时66天，共死亡4,519人，其中被杀4,193人，被迫自杀326人。全地区非正常死亡9,093人，其中被杀7,696人，被迫自杀1,397人，全地区直接或间接杀人者15,050人。其中被“诛九族”灭门的有117户，被杀者年纪最大的78岁，最小的才10天。书中记载了十多种匪夷所思的杀人方式，杀人者不满足于把人处死，而是变换花样，享受处死过程的快感和折磨的乐趣。杀人的同时还伴随着抄没财产、强奸妻女等暴行。一时间，尸体漂满了潇水。血腥之气长久不散。
</p>
<p>
 注3：邓小平说：杀20万，保20年稳定。
</p>
<p>
 注4：媒体报导，各大城市不断出现把“打倒共产党“等标语写在厕所里。厕所无监控，邪党很是头疼。
</p>
