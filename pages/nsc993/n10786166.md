 <br/>
 西安八路假统战几何新四扩编。

 <br/>
 谈判代表潘汉年。

 红朝窃国逼蒋公入台，

 掐指整整六十九年。

 大陆运动灾劫，
 <br/>
 文革六四法轮功连番。

 集中营，
 <br/>
 贩卖国人器官。
 <br/>

 <br/>
 血腥房地产。
 <br/>
 宝岛经国推多党，
 <br/>
 与韩国经韩战。
 <br/>
 先后踏上民主，
 <br/>
 两地富而有德如镜鉴。
 <br/>
 二零一八年十月十三日，
 <br/>
 亚洲十二国法轮功学员聚首南韩。
 <br/>
 反中共迫害游行，
 <br/>
 内有约三百名来自台湾的学员。
 <br/>
 持续十九年的迫害，
 <br/>
 更显大陆红潮凶险。
 <br/>
 九评引发三退，
 <br/>
 人数已超三亿一千万。
 <br/>
 共产主义的终极目的，
 <br/>
 毁人更加彰显。
 <br/>
 今年大纽约双十节大游行，
 <br/>
 喜见“民国粉” 大陆青年。
 <br/>
 高举“三民主义光复大陆”，
 <br/>
 “
