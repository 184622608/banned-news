### 关乐：党妈袒护国外撒野者只是利用与坑害
------------------------

<p>
 【大纪元2018年10月04日讯】
</p>
<p>
 恐风凛凛牛哄哄，
 <br/>
 凶掌啪啪抽义工。
 <br/>
 莫当大裤衩尽黄，
 <br/>
 没见党喉舌血红。
 <br/>
 骂人粗口久忍苦，
 <br/>
 打人巴掌常拍疼。
 <br/>
 扣帽打骂家挨惯，
 <br/>
 出洋钻孔放风疯。
 <br/>
 自由世界享自由，
 <br/>
 土共花开更土共。
 <br/>
 内外有别党教会，
 <br/>
 双面两手模仿成。
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E5%85%9A%E6%96%87%E5%8C%96.html">
  党文化
 </a>
 就这黑窑，
 <br/>
 又红又砖又臭硬。
 <br/>
 一般家长会训斥，
 <br/>
 唯有党妈佯护宠。
 <br/>
 兔死鹰犬下锅烹，
 <br/>
 工具用完即弃扔。
 <br/>
 别说小小无名辈，
 <br/>
 英模党魁亦非葱。
 <br/>
 害人一害害到死，
 <br/>
 但愿有良能清醒。
 <br/>
 东西南北鉴泾渭，
 <br/>
 善恶正邪看分明。
 <br/>
 穷途末路更倡狂，
 <br/>
 党遭天谴近坟坑。
 <br/>
 漂红三退神护祐，
 <br/>
 瞎步党尘白搭命。
