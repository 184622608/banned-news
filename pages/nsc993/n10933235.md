### 陈达：拒洋正神拜洋魔
------------------------

<p>
 【大纪元2018年12月26日讯】
</p>
<p>
 洋发洋装洋皮鞋，
 <br/>
 洋房洋路洋汽车。
 <br/>
 洋手机加洋喇叭，
 <br/>
 洋脑
 <a href="http://www.epochtimes.com/gb/tag/%E6%8A%B5%E5%88%B6.html">
  抵制
 </a>
 <a href="http://www.epochtimes.com/gb/tag/%E8%80%B6%E8%AF%9E%E8%8A%82.html">
  耶诞节
 </a>
 。
 <br/>
 高喊爱国拒洋货，
 <br/>
 其实盘点大陆节，
 <br/>
 洋节远远超土节。
 <br/>
 元旦初称洋历节，
 <br/>
 三八洋红妇女节。
 <br/>
 五一洋工闹事节，
 <br/>
 六一染红儿童节。
 <br/>
 七一共产洋党节，
 <br/>
 八一红军洋兵节，
 <br/>
 十一马列殖民节。
 <br/>
 泱泱古国五千载，
 <br/>
 而今何须建国节？
 <br/>
 本是篡权建邪政，
 <br/>
 中国只被挂特色。
 <br/>
 马恩列斯洋鬼节，
 <br/>
 排挤改造中古节。
 <br/>
 土洋一体非水火，
 <br/>
 不共戴天是正邪。
 <br/>
 为何
 <a href="http://www.epochtimes.com/gb/tag/%E6%8A%B5%E5%88%B6.html">
  抵制
 </a>
 <a href="http://www.epochtimes.com/gb/tag/%E8%80%B6%E8%AF%9E%E8%8A%82.html">
  耶诞节
 </a>
 ？
 <br/>
 拒洋正神拜洋魔。
 <br/>
 也是邪党自露怯，
 <br/>
 文化自信骗自个。
 <br/>
 中华自古多教存，
 <br/>
 独有红朝一齐灭。
 <br/>
 非是力大无匹敌，
 <br/>
 唯恐照妖镜光烈。
 <br/>
 迫害大法亦因此，
 <br/>
 因此生路全堵绝。
 <br/>
 也是内斗高级黑，
 <br/>
 习近平脸给扇破。
 <br/>
 扩大开放刚喊过，
 <br/>
 立马关门紧收缩。
 <br/>
 核心一尊何粉饰，
 <br/>
 不就马屎○注洋烂货？
 <br/>
 基础一群洋脑残，
 <br/>
 漂红向善即反戈。
 <br/>
 三退已超三亿二，
 <br/>
 五毛队伍日败落。
 <br/>
 好人良知一清醒，
 <br/>
 炎黄子孙拒马列。
 <br/>
 中华复兴携四海，
 <br/>
 地球村过除魔节。
</p>
<p>
 注：马屎：马克思的作品。“据我所知，在著名作家中，只有马克思称自己的作品为‘屎’、‘污秽之书’。他自觉、蓄意地将秽物给予他的读者。怪不得他的某些信徒，比如罗马尼亚和莫三比克共产党，强迫囚犯们吃自己的屎尿。“（理查‧沃姆布兰德《卡尔‧马克思的成魔之路》）
</p>
