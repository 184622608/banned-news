 刚一过招原形露，吹牛幻想不当饭。

 嘴上功夫坑蒙拐，贼手三只偷抢骗。
 <br/>
 外强中干是虚胖，见硬缺钙一软蛋。
 <br/>

 <br/>
 红楼梦破路在哪？两腿乱蹬无明天。
 <br/>
 向内看，上下乱，经济崩溃炊烟断。
 <br/>
 贪官污吏照样肥，党捅窟窿民买单。
 <br/>
 抗暴烽火压不往，此起彼伏遍地燃。
 <br/>
 打倒
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1.html">
  中共
 </a>
 大街喊，手刃党腿屡不鲜。
 <br/>
 亦有党官苦难言，跳楼自杀无路选。
 <br/>
 更见三退大潮涌，觉醒民众跳红船。
 <br/>
 向外看，党胆寒，四海浪起倒共产。
 <br/>
 川普神助开头炮，彭斯助阵正上演。
 <br/>
 非洲不顺黑脸对，白装笑脸抛橄榄。
 <br/>
 活摘追查扣扣紧，人权恶棍万众谴。
 <br/>
 军属医院忙重组，更名难藏罪滔天。
 <br/>
 纸里包火挺几时？越遮越藏越昭眼。
 <br/>
 哎呀马恩列斯毛，党要绝根你不管。
 <br/>
 更有祸首江蛤蟆，不干好事惹后患。
 <br/>
 又添雾霾京津冀，老天脸色不好看。
 <br/>
 天灭中共事不虚，一个跟头党玩完。
