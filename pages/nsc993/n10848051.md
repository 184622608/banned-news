### 甘楠：笑红朝高考政审
------------------------

<p>
 【大纪元2018年11月13日讯】
</p>
<p>
 台湾人民，水热火深。挣扎美帝，日落沉沉。
 <br/>
 解放台湾，涉水金门。一看没戏，空炮频频。
 <br/>
 尼布楚条，索额图任。鄂罗斯定，国土印印。
 <br/>
 尼布楚约，红朝不认。先毛后江，卖疆盆盆。
 <br/>
 海棠图时，外蒙饥赈。台湾铸钱，清命薪薪。
 <br/>
 公鸡图时，蒙内亡人。跃进人祸，公社浑浑。
 <br/>
 文革反骨，开挖祖坟。战天斗地，宗庙烬烬。
 <br/>
 反骨中共，破坏人伦。共产共妻，马列孙孙。
 <br/>
 耀邦紫阳，早殇软禁。六四屠城，蛤蟆淋淋。
 <br/>
 紫阳耀邦，渐出课本。少奇未血，真相闷闷。
 <br/>
 屠夫中共，国际义愤。制裁联合，中南震震。
 <br/>
 中共屠城，对内深隐。国门微起，亿币允允。
 <br/>
 洗脑央视，杜宪类贫。白骨万变，不离泯泯。
 <br/>
 殃视潮二，台湾歌唇。掀起波澜，丽军纭纭。
 <br/>
 只要残喘，歌舞马论。赌色誓诺，统战螓螓。
 <br/>
 残喘只要，党能蟾临。三英亿鹰，部队裙裙。
 <br/>
 香港美第，钱能领引。闷声发财，镰斧刎刎。
 <br/>
 美帝台湾，不深不慎。帕瓦罗蒂，蛙声阵阵。
 <br/>
 人群渐大，修炼法轮。功传世界，福亿身身。
 <br/>
 渐大人群，遵德有神。罗庥泽民，鬼妒焚焚。
 <br/>
 应劫九九，四二五醺。七二零附，魔党捆捆。
 <br/>
 九九应劫，法国诺谶。七月恐怖，近廿准准。
 <br/>
 法徒天安，坚定修心。自焚伪火，蝌蚪殒殒。
 <br/>
 天安法徒，囹圄至今。世人尊敬，法之茵茵。
 <br/>
 全球清共，美台牡牝。终止活摘，驱鬼刃刃。
 <br/>
 清共全球，天地共愤。昭雪亿冤，亡灵滚滚。
 <br/>
 毁人红潮，目的哪遁。自由无界，翻墙浚浚。
 <br/>
 红潮毁人，九评甘醇。驱邪三退，城镇村村。
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E6%94%BF%E5%AE%A1.html">
  政审
 </a>
 蟾帮，雷霆万钧。天钩天安，宣读本本。
 <br/>
 妖党公审，同流毒疹。莫毁前程，择善浈浈。
</p>
