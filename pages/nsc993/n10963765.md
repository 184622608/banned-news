### 关乐：统一大业天意光复（外一首）
------------------------

<p>
 【大纪元2019年01月09日讯】
</p>
<p>
 中国人不打中国人，
 <br/>
 却老嚷向台湾进军①。
 <br/>
 党话不是中国人话，
 <br/>
 流氓逻辑浑不怕浑。
</p>
<p>
 一九四五台湾收复，
 <br/>
 一直是中国人居住。
 <br/>
 收复台湾出师何名？
 <br/>
 马列殖民扩犯国土。
</p>
<p>
 联合国立一九四五，
 <br/>
 排挤民国中共混入。
 <br/>
 混入假称席位恢复，
 <br/>
 亦为被逐自留埋伏。
</p>
<p>
 分裂两岸大罪已铸，
 <br/>
 五千文明毁罪难赎。
 <br/>
 中华正统宝岛留根，
 <br/>
 统一大业天意光复②。
 <br/>
 中共主打中国人
</p>
<p>
 中国人不打中国人，
 <br/>
 共产党主打中国人。
 <br/>
 马列子孙非中国人，
 <br/>
 害死八千万中国人。
</p>
