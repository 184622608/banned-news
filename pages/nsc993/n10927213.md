### 陈达：中共叫嚣“两个没有”不过是哀鸣而已
------------------------

<p>
 【大纪元2018年12月23日讯】
</p>
<p>
 没有可以奉为金科玉律的教科书，
 <br/>
 那么
 <a href="http://www.epochtimes.com/gb/tag/%E9%A9%AC%E5%85%8B%E6%80%9D%E4%B8%BB%E4%B9%89.html">
  马克思主义
 </a>
 著述究竟算何物？
 <br/>
 马克思自己都说那是屎污秽之书，
 <br/>
 为什么中共还用其邪恶主义指路？
 <br/>
 没有可以对中国人民颐指气使的教师爷①。
 <br/>
 那么马恩列斯毛邓江凭何总教导人们说？
 <br/>
 “定于一尊”危乎高哉又是为何让人尊何？
 <br/>
 这“两个没有”本身又从何说起冲谁扯？
 <br/>
 明明是指桑骂槐否定五千年神传文化，
 <br/>
 继承文革却不敢打文革旗帜心虚胆怯。
 <br/>
 明明是嘲讽川普正义援手的热诚挽救，
 <br/>
 无理决绝而不敢公开对决还打滚撒泼。
 <br/>
 然而党叫嚣“两个没有”并非自有妙计，
 <br/>
 末日邪灵越怕失去一切越哀鸣领导一切②。
</p>

</p>

 <a href="http://www.epochtimes.com/gb/tag/%E6%94%B9%E9%9D%A9%E5%BC%80%E6%94%BE.html">
  改革开放
 </a>
 四十周年大会上的讲话》，二〇一八年十二月十八日）
</p>
<p>
 2、“第一，必须坚持党对一切工作的领导，”“中国共产党领导是中国特色社会主义最本质的特征，是中国特色社会主义制度的最大优势。党政军民学，东西南北中，党是领导一切的。”“在坚持党的领导这个决定党和国家前途命运的重大原则问题上，全党全国必须保持高度的思想自觉、政治自觉、行动自觉，丝毫不能动摇。”（同上）
