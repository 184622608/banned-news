------------------------

<p>

 <a href="http://www.epochtimes.com/gb/tag/%E8%A7%A3%E4%BD%93%E5%85%9A%E6%96%87%E5%8C%96.html">

 </a>
 】之六：习惯了的党话（上）：中共盗用国家政权对民族文化和语言的深刻伤害，可以说前无古人。语言是思维的工具。被党文化严重污染的语言，严重地损害了人们反思中共、反思党文化、构思民族未来的能力。很多人都发现一个奇特的现象：人们用中共制造的语言批判中共，愤怒声讨中共的文章仍然称中共建政为“解放”，有人在“退党声明”中仍然说“我是‘生在新中国，长在红旗下’的。”让人简直难以分辨他到底是要唾弃中共，还是要感激中共。在中共政权摇摇欲坠、人民亟待回归正常人类文化的今天，认清附着在民族语言上的党话，清除党话，已经成为刻不容缓的任务。

 ——题记
</p>
<p>
 薄王孙毒肃清，一字留也不成①。切割如此彻底，重庆又演何戏？
 <br/>
 肃清某某余毒，似乎毫不含糊。名义一概反腐，巨贪每抓必肃。
 <br/>
 别的罪行不问，肃毒调门另谱。怎老玩这名目？终究有何目的？
 <br/>
 迫害暗暗升级，殃民狠狠加厉②。防火墙封愈严，墙外推特也断③。
 <br/>
 政法网络沙皇，倒不倒怎一样？政治流氓奸雄，抓不抓咋相同？
 <br/>
 为何单抓其贪？因其跟党抢钱。不问别的罪行，因那党还有用。
 <br/>
 肃清周永康毒，枉法恐怖如故。肃清鲁炜余毒，网控严厉如故。
 <br/>
 肃清薄王孙毒，打黑唱红如故。肃清余毒栈道，洗刷影响陈仓。
 <br/>
 洗刷什么影响？党的形象损伤。清毒是个名义，切割为党洗地。
 <br/>
 洗地保党画皮，党妈伪装干洗。教出一帮贪官，反腐再证党廉。
 <br/>
 肃清贪官余毒，顶雷暗保红毒。训出一帮暴徒，打黑证党清白。
 <br/>
 肃清打手余毒，替罪保党黑帅。带出一帮色狼，扫黄证党反黄。
 <br/>
 肃清淫官余毒，共妻旗偃泛黄。领出一帮骗子，示一再行骗百。
 <br/>
 肃清骗子余毒，打假证党实在。不断肃清余毒，反腐越反越腐。
 <br/>
 肃清余毒风吹，打黑越打越黑。肃清余毒流长，扫黄越扫越黄。
 <br/>
 肃清余毒招花，打假越打越假。归根折腾什么？证党伟光正确。
 <br/>
 从来不肯认错，认错怕被追责。更惧追根溯源，认清邪灵嘴脸。
 <br/>
 魔鬼假扮正神④，罪错全推给人⑤。肃清余毒此招，红染缸里搅绕。
 <br/>
 搅的眼花心乱，残脑毁人不倦。反腐前腐后继，清毒接连不断。
 <br/>
 本来都是党毒，无一非被毁族。狼奶才是毒源，全由狼外婆灌。
 <br/>
 切割转移视线，毒源安然焕然。肃清被毁余毒，党文化毒再输。
 <br/>
 流氓耍的贼溜，防不胜防妖术。洗脑终为何事？改造非人害死⑥。
 <br/>
 好人赶快清醒，找读九评⑦巨著⑧。认清邪党本质⑨，三退⑩喜踏新途。
