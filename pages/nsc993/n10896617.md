 支离破碎要散架，红楼必倒难支撑。
 <br/>
 四面楚歌人心散，内斗火拼声连声。

 别想再有好日子，邪党不倒祸不停。
 <br/>
 天要灭共谁能躲？厄运时时伴尔行。
 <br/>
 做恶不信天有眼，三尺头上有神灵。

 罪盈恶满天清算，一桩一件必算清。
 <br/>
 杀人如麻八千万，遍地冤魂齐讨命。

 <br/>
