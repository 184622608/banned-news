### 冀旦：戊戌之倒共九十天倒数
------------------------

<p>
 【大纪元2018年12月28日讯】
</p>
<p>
 不变国清民，变平安社邻。数天九十倒，剪辫九三伸。
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%89%E9%80%80%E5%A4%A7%E6%BD%AE.html">
  三退大潮
 </a>
 泛，九评天降津。十数三亿二，呼唤良知纯。
 <br/>
 不变道传宗，变平明泽豳。仲勋三字学，经典炎黄淳。
 <br/>
 拜马其谁祖，远平红尾巾。黄皮肤种族，西马幽蒙瑉。
</p>
<p>
 文革十年红，十红元帅濒。少奇床上扼，宗庙土中抡。
 <br/>
 七六陨唐震，首三红落嶙。国锋难姓毛，叶邓耀阳春。
 <br/>
 旧势八蟾权，京红六四殷。蟾王江泽弄，原壁火龙坟。
 <br/>
 国土鸿毛割，陈黄英祖醺。一权枪党国，蟾子一贪蚊。
</p>
<p>
 九二清风徐，法轮功福身。长春一启始，世界各邦巡。
 <br/>
 罗干政公痿，祚庥妖笔皴。进京四二五，妒嫉蟾王颦。
 <br/>
 七二零妖飙，淳风诺氏吟。预言中外古，剧本戏台今。
 <br/>
 入扣丝丝对，开心字字琛。先贤遗指点，真相法中寻。
</p>
<p>
 活摘天安焚，喉殃造慌闻。害人终害自，癌报果癌群。
 <br/>
 薄谷立军咬，康生裤衩熏。狱中狱外鼓，利外利中荤。
 <br/>
 川普彭斯团，清红振臂纶。晚舟落更浅，拿大正非深。
 <br/>
 资料漏于党，美元世贸唇。拜江楼墅数，民币银行银。
</p>
<p>
 共产极终明，毁人马接迎。主流渗透密，魔鬼暗中狞。
 <br/>
 媒体真言吐，纪元唐视诚。翻墙无界网，门外自由亨。
 <br/>
 川普天天推，有缘日日睛。一元圣诞美，神韵五城呈。
 <br/>
 元旦喜讯更，巡回六出行。五千传统复，文化神传明。
</p>
