### 臧海清：“我是三十五年的党员”
------------------------

<p>
 【大纪元2018年11月15日讯】
</p>
<p>
 “我是三十五年的
 <a href="http://www.epochtimes.com/gb/tag/%E5%85%9A%E5%91%98.html">
  党员
 </a>
 ”！
 <br/>
 二零一八减去后是一九八三，
 <br/>
 那时入党政审很严，
 <br/>
 不过正逢毛狱走入邓猫钱！
 <br/>
 如果又红又专，
 <br/>
 毛时代入党简单。
 <br/>
 如果黑五类，
 <br/>
 没有耀邦解救，
 <br/>
 入党太难！
 <br/>
 猫时代你入党是为钱还是权，
 <br/>
 还是觉得入党安全？
 <br/>
 那么入党六年，
 <br/>
 六四不知你是否心惊震胆！
 <br/>
 估计你不会追随紫阳，
 <br/>
 遑论你是否官倒腐贪！
 <br/>
 卖国贼蟾，
 <br/>
 税收拿给书记经理一身的
 <a href="http://www.epochtimes.com/gb/tag/%E5%85%9A%E5%AE%98.html">
  党官
 </a>
 ，
 <br/>
 养了一国的公母蝌蚪白手套，
 <br/>
 书记在里手套房产，
 <br/>
 不知你是书记还是手套？
 <br/>
 真的没有什么值得骄傲！
 <br/>
 红五类已低端，
 <br/>
 房奴需付出三代血汗！
 <br/>
 你入党十六年，
 <br/>
 正逢迫害法轮功，
 <br/>
 你供人猿，
 <br/>
 还是真的见善！
 <br/>
 大法百国洪传，
 <br/>
 莫失机缘。
 <br/>
 六一零政法委十九年来罪恶累累，
 <br/>
 酷刑活摘黑白倒颠，
 <br/>
 你是否已经洞察恶源？
 <br/>
 迫害的罪魁薄熙来周永康李东生们恶报入狱，
 <br/>
 请相信冥冥中有更高的主宰，
 <br/>
 不是西来马列幽灵魔王撒旦，
 <br/>
 请珍惜天降九评，
 <br/>
 仔细读一读，
 <br/>
 共产主义的终极目的在毁人，
 <br/>
 退出党队团，
 <br/>
 这一刻，
 <br/>
 就是脱离红三十五年的新纪元。
</p>
