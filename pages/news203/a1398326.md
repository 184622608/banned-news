### 3分钟看懂中期选举
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 每4年举行一次，那么这次中期选举的影响如何呢，来看我们的〝中期选举指南〞。
 <br/>
 <br/>
 【选什么？】
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，每四年一次。本次将改选国会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 全部435席，改选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 100席中的35席。在州长级官员选举中，将改选36个州和3个美国领地共39位州长级官员，以及6600多个州立法机构席位。
 <br/>
 <br/>
 【选战现状？】
 <br/>
 <br/>
 参议院中，目前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 占51席，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 49席。本次改选35席，其中26席来自民主党。民主党6年前获得这26席，不过其中10席所在的州，在2年前总统大选时倒向共和党的川普。民主党想守住这26席甚至扩大优势的难度很大。多数民调认为，共和党有很大胜算继续主导参院。
 <br/>
 <br/>
 众议院部分，选情胶着。多家民调分析，两党的〝安全席位〞都未达到218席的多数党门槛，并且相差不到10席。因此30~40个摇摆选区的席位，将产生决定作用。
 <br/>
 <br/>
 【重要性？】
 <br/>
 <br/>
 参议院和众议院共同起草和制定法律。参议员还有权确定或驳回总统对官员和法官的任免。
 <br/>
 <br/>
 改选后的国会，如果对总统的政策主张和人事提名进行牵制，将影响白宫的执政效率。也将影响川普政府能否继续透过降低税收、削减福利、减少监管，来降低美国民众对政府的依赖，破除社会主义等左倾政策。
 <br/>
 <br/>
 州长则是各州主要发言人、监督州法律、可任免地方官员和法官，另外也对2020年总统大选的募集资金和拉抬选情方面，有很大影响。
 <br/>
 <br/>
 6600多个州立法机构席位，影响也不容小觑。不仅影响本州医保、教育等领域的立法和政策走向，而且主持本州国会选区重新划分，影响未来总统选举和国会众议院议席的归属。
 <br/>
 <br/>
 【参与度？】
 <br/>
 <br/>
 传统上，中期选举投票率较低，只有40%左右。但今年，选民热情高涨。截至本周一，美国49个州至少有3640万人提前投票，其中30个州的早期投票人数，就超过2014年投票总人数。专家预计，最终将有45%注册选民投票，是半个世纪来最高纪录。
 <br/>
 <br/>
 【开票时间？】
 <br/>
 <br/>
 多数州会在美东时间6号晚8点结束投票，到美东时间7日凌晨0点，阿拉斯加州投票站关闭后，整个中期选举投票落下帷幕。
 <br/>
 <br/>
 除了个别国会选区的计票可能需要数日，大多数投票结果，将在11月7日中午之前明朗。
</div>
