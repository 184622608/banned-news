### 川普驳首席大法官言论 吁法官尊重宪法原意
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】周四（11月22日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统就非法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 的裁决问题驳斥美国最高法院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首席大法官.html" target="_blank">
  首席大法官
 </a>
 ，并批评位于加州的美国第九巡回上述法院是个〝灾难〞，同时援引公众意见，呼吁整编或拆分该法院。
 <br/>
 <br/>
 周四，针对美国第九巡回上诉法院（the Ninth Circuit Court of Appeals）保护非法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 问题，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 驳斥美国最高法院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首席大法官.html" target="_blank">
  首席大法官
 </a>
 罗伯茨（John Roberts）的相关评论。
 <br/>
 <br/>
 川普指出，罗伯茨可以说他自己想说的话，但第九巡回上诉法院是个〝完全而彻底的灾难〞。它已经失控且声名狼藉，被推翻的裁决多达79%，比全国的任何一个法院都多，而且经常被利用来起到保护非法移民的效果。
 <br/>
 <br/>
 川普警告，法官不能针对边境及任何地方的安全问题进行立法，他们在这方面什么都不懂，反而正让美国陷于不安全之中。
 <br/>
 <br/>
 川普接着以大写字母形式在推文中强调，必须允许美国的执法人员做他们该做的工作，否则，美国民众面临的只能是喧嚣、混乱、伤害、死亡。川普呼吁法官尊重宪法原意。

 <br/>
 周一，由前总统奥巴马任命的美国加州北区地方法院（US. District Court for the Northern District of California）法官泰格（Jon S. Tigar）裁定，禁止川普推行拒绝非法入境移民申请庇护的行政令。这道
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁令.html" target="_blank">
  禁令
 </a>
 无疑给川普政府拦阻数千将至的大篷车移民设置了新的障碍。
 <br/>
 <br/>
 川普周二在接受媒体采访时批评，法院的裁决是个〝耻辱〞，并指泰格是一名〝奥巴马法官〞（Obama judge）。
 <br/>
 <br/>
 周三，罗伯茨罕见公开批评川普，称〝我们没有‘奥巴马法官’或‘川普法官’，也没有‘布什法官’或‘克林顿法官’，我们拥有的是一群非凡的尽职尽责的法官〞，并赞扬这些法官正尽最大努力保证人们在法律面前平等。
 <br/>
 <br/>
 罗伯茨的评论随即遭到川普反驳。川普指出，法院的确拥有〝奥巴马法官〞，如果第九巡回上诉法院确实如罗伯茨所说的是个〝独立的司法机构〞，那就太棒了。但实际情况却令人震惊，该法院那些裁决正把美国置于巨大危险之中。
 <br/>
 <br/>
 川普接着表示，很多人对第九巡回法院表示不满，并讨论把该法院拆分，然后把法官们分配到第二、第三巡回法院中去。
 <br/>
