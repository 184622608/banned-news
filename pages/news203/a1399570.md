### 美国网络鞋店推新招：买一送一给流浪汉
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】看到街上那些无家可归的人，相信很多人都会奉献一份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱心.html" target="_blank">
  爱心
 </a>
 。美国一间鞋店日前推出一个更特别的行销方式，就是消费者每买一双鞋，他们就会送一双给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_流浪汉.html" target="_blank">
  流浪汉
 </a>
 。
 <br/>
 <br/>
 据《Good News Network》报导，这家商店叫〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Brethren Shoes.html" target="_blank">
  Brethren Shoes
 </a>
 〞，创办人Tian Mu因为常在美国洛杉矶看到住在街头的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_流浪汉.html" target="_blank">
  流浪汉
 </a>
 ，有些伤感，因此想出这个点子。他表示，这些无家可归的人让人很同情，但他仍相信人性本善以及人类的慷慨。他认为顾客在购物的同时，还能帮助有需要的另一个人，这是双赢。
 <br/>
 <br/>
 〝Brethren Shoes〞的鞋子种类多元，款式也多种多样，从皮鞋到靴子都有，但价格都是每双66美元。假如顾客愿意主动捐鞋，也可以用33美元的慈善价买鞋。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
