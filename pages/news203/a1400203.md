### 加州大火感人一幕 校车司机冒死救22学生冲出火海
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】南北加州野火已经烧了近两周，共79人丧生，近千人失踪。现场惨不忍睹，但在火灾中，也有很多可歌可泣的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英雄.html" target="_blank">
  英雄
 </a>
 事迹传出。一位小学
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_校车司机.html" target="_blank">
  校车司机
 </a>
 就冒死带着22名学生逃出火海，被人们誉为英雄。
 <br/>
 <br/>
 据〝CBS新闻网〞报导，11月8日，加利福尼亚州坎普野火在天堂镇（Paradise）开始燃烧，得到消息的镇上居民都在紧急撤离。
 <br/>
 <br/>
 天堂镇黄松小学（Ponderosa Elementary School）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_校车司机.html" target="_blank">
  校车司机
 </a>
 凯文・麦凯（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Kevin McKay.html" target="_blank">
  Kevin McKay
 </a>
 ）没有离开，因为小学里还有很多学生家长没有前来领走孩子，他觉得确保其他孩子安全是他的责任。
 <br/>
 <br/>
 在认识到迅速蔓延的大火将马上扑到小学之后，41岁的麦凯立即让22名学生和2名老师上了校车，并迅速开车离开。
 <br/>
 <br/>
 麦凯担任黄松小学校车司机仅几个月，那一天让他永远难忘，麦凯回忆说：〝我当时就知道大火会迅速扩散到那里。〞
 <br/>
 <br/>
 他告诉CBS新闻记者德马科・摩根，当时他真的是被吓坏了。
 <br/>
 <br/>
 在校车上，麦凯告诉老师艾比・戴维斯（Abbie Davis）和玛丽・路德维格（Mary Ludwig）安慰学生并分散他们的注意力。老师戴维斯回忆说：〝我一直忍着没哭。因为我们一直专注于照顾那些孩子，是巨大的责任感才使我们能够专注。〞
 <br/>
 <br/>
 路德维格和戴维斯两名老师也都表示，当时她们非常担心自己的安全。路德维格回忆说：〝我们都努力不让对方哭。我们只是牵着手……我们只是祈祷着。〞
 <br/>
 <br/>
 7岁的温迪・里维斯（Wendy Reeves）说，当时她无法忽略外面的大火。她说：〝当我们离开学校后，就看到了大火，之后我们还看到了更多大火。〞
 <br/>
 <br/>
 随着野火的浓烟将附近地区覆盖，校车内也开始冒进浓烟。一些孩子说他们感到无力和恶心。麦凯说：〝浓烟进到车里后，我们都一直在咳嗽，我的眼睛也被熏得很疼。我知道我们必须做点什么。〞
 <br/>
 <br/>
 麦凯立即脱下衬衫，并让老师们把它撕成布块，浇上水后，向孩子们展示如何利用湿布帮助呼吸。麦凯说：〝当时那是我们能做出的最好选择。〞
 <br/>
 <br/>
 麦凯开了5个小时的车，路上经常碰到无法通行的情况。野火的烟雾模糊了他的视线，看不到前面的路。一些撞毁和烧毁的车还停在路上。但是最后，他终于把孩子们送到了大约30英里以外的安全地方。
 <br/>
 <br/>
 麦凯说：〝我们一直等到每个孩子都被稳妥地接走之后才离开，最终每个孩子都和他们的父母在一起了。〞
 <br/>
 <br/>
 麦凯的行为受到来自全美各地人们的赞扬。虽然有人称麦凯为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英雄.html" target="_blank">
  英雄
 </a>
 ，但他哽咽着说，其实，那天还有很多人做得更多，〝一路上看到我们的消防员们都朝着相反的方向冲去，那才真的是伟大。〞
 <br/>
 <br/>
 事后人们才知道，麦凯当时在得知自己的家已经被大火吞噬之后，仍留在学校救出了22名学生和2位老师。所幸，他的妻子和家人已经撤离到安全地点。
