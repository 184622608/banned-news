### 众筹捐流浪汉原是骗局 40万捐款全须退
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】周四，费城伯灵顿县检察官宣布在公益网站发起为一流浪汉捐款是一场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_合谋骗局.html" target="_blank">
  合谋骗局
 </a>
 ，涉案三人被控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二级盗窃罪.html" target="_blank">
  二级盗窃罪
 </a>
 。该网站将向相关捐赠者提供全额退款。
 <br/>
 <br/>
 检察官科非纳（Scott Coffina）介绍，一对来自新泽西的情侣称，去年11月因大雪被困在费城一处高速路上，一个流浪汉Johnny用身上仅有的20美金帮他们买了汽油。他们随后在网站
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_GoFundMe.html" target="_blank">
  GoFundMe
 </a>
 .com上建募款名目〝Paying it Forward〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_为Johnny捐款.html" target="_blank">
  为Johnny捐款
 </a>
 ，以示回报。
 <br/>
 <br/>
 伯灵顿县检察官Scott Coffina：〝整个活动都是骗局。〞
 <br/>
 <br/>
 检察官指出，情侣在高速路上没有耗尽汽油，流浪汉也没有花最后二十美元帮助他们。
 <br/>
 <br/>
 伯灵顿县检察官Scott Coffina：〝然而，D'Amico，McClure和Bobbitt三人合谋编造一个虚假的，感觉良好的故事。〞
 <br/>
 <br/>
 涉案三人被控以欺骗阴谋手段的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二级盗窃罪.html" target="_blank">
  二级盗窃罪
 </a>
 ，他们欺骗超过14,000名捐赠者，骗取近40.3万美元捐款。网站将向该案捐赠者提供全额退款。
 <br/>
 <br/>
 检察官希望杜绝此类事发生，但他认为慷慨善心值得赞扬。
 <br/>
 <br/>
 世上难事多，对需要帮助的人慷慨援助是应该赞扬的。〞
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
