### 司法部监督19州投票 川普警告舞弊后果
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 已进入倒计时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周一（11月5日）警告，任何人在选举中舞弊都将面临严重后果，甚至遭到司法诉讼。
 <br/>
 <br/>
 周二，美国将迎来直接影响未来几年内民众利益的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，尤其是在当前两党之争分外激烈的情况下，每张选票或许都会关系到今后美国的政治走向。
 <br/>
 <br/>
 周一上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在大选前最后一天发出警告：
 <br/>
 <br/>
 〝已强烈通知执法部门，密切关注在周二选举（或提前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 ）中可能出现的任何非法投票。被抓获的任何人都将在法律允许的范围内接受最高刑事处罚。谢谢！〞
 <br/>
 <br/>

 <br/>
 周一上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部.html" target="_blank">
  司法部
 </a>
 长塞申斯（Jeff Sessions）宣布，司法部正在部署大量人力，将在19个州的35个投票点
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监督.html" target="_blank">
  监督
 </a>
 选举，以防止欺诈行为，保护公民的选举权。
 <br/>
 <br/>
 塞申斯在声明中指出，选举权是宪法赋予公民的权利，在保护这项权利行使的过程中，司法部被赋予了不可或缺的作用。
 <br/>
 <br/>
 塞申斯还表示，司法部今年利用各种民事及刑事法律工具，去保护数百万美国人畅通无阻地投票，涉及范围达170,000多个选区。美国公民通过投票选举他们中意的政府官员来管控国家，而投票过程中的欺诈行为是不能容忍的，同时也在腐化选举的公正性。
 <br/>
 <br/>
 川普过去曾一再声称美国选举中广泛存在投票舞弊的现象。川普就职总统后，曾要求美国行政部门调查2016年大选中的选民欺诈现象，包括是否有人冒充公民身份投票、以及选民是否为已经死亡甚或去世多年的人投票等不法行为。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
