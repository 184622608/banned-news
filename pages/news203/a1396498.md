### 库洛德确认11月川习会 披露预期〝最好结果〞内容
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】白宫首席经济顾问库德洛（Larry Kudlow）周二（23日）确认，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）将在11月阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会上，与中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 会面，并坦言二人或可达成的〝最好结果〞。
 <br/>
 <br/>
 库德洛周二告诉媒体，两位领导人将在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会期间〝碰面一次〞（will meet for a bit），但他没有透露其它细节。
 <br/>
 <br/>
 库德洛并不预期两国首脑会晤能取得重大突破。他说，如果能〝就一些基本原则和贸易规则〞达成广泛协议，包括窃取知识产权、强制转让技术和农产品关税，〝这可能是最好的结果〞。
 <br/>
 <br/>
 他预期在两人会面前，美中两国官员会先进行幕僚层级的会谈。
 <br/>
 <br/>
 这将是自去年11月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 访华至今，两国领导人之间的再次面对面会晤。但外界对会晤是否有助于美中之间缓和贸易紧张局势，仍持观望态度。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易.html" target="_blank">
   美中贸易
  </a>
  纠纷陷入僵局
 </h4>
 <br/>
 到目前为止，美中双方已互相征收关税，美国对价值2,500亿美元、近6成中国进口商品加征关税，而中方也对美国产的1,100亿美元、超过7成的进口商品加征关税。
 <br/>
 <br/>
 双方已经过4次贸易对话，但都陷入停滞。美方指，中方对美国的核心诉求——3零2停1允许——没有进行明确回应。〝3零〞是〝寻求零关税、零非关税壁垒、零补贴〞，〝2停〞是〝停止盗窃知识产权和技术转移〞，〝1允许〞就是〝允许美国人在中国拥有自己的企业〞。
 <br/>
 <br/>
 〝我们的问题摆在桌面上，我将很乐意看到他们（中共）回应，〞库德洛说，〝到目前为止，它们还没有（回应）。〞
 <br/>
 <br/>
 他还表示，在领导人峰会召开之前，他将参加美中两国官员的工作级别会议。
 <br/>
 <br/>
 <h4>
  川习二次会面 或非贸易会谈
 </h4>
 <br/>
 香港《南华早报》上周五（19日）引述消息人士的话说，川普和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 已〝暂时〞同意11月在G20集团峰会上会面。G20是世界主要经济体领导人年度会议，今年的会议日期是11月30日至12月1日。
 <br/>
 <br/>
 报导称，双方会面日期定在11月29日，在阿根廷布宜诺斯艾利斯举行的G20峰会正式开始前一天。
 <br/>
 <br/>
 美国新闻网站Axios周日（21日）也引述知情人士消息，川普无意放宽对中国商品的关税，川普与中共的贸易战正处于〝开始的开始〞阶段，同时川普团队对总统与习近平下个月在G20峰会期间的会晤不抱太大期望。
 <br/>
 <br/>
 知情人士表示，G20会晤的目的是川普和习近平重新联系，并在不断升级的贸易战中〝互相感受〞一下。〝这是国家元首会议，而不是贸易会议。〞知情人士说。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：）
