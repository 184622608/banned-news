### 美国将退出万国邮联 大陆寄美邮资看涨
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务院周三宣布，起启动退出万国邮政联盟程序。白宫表示，万国邮政联盟的现行费率制度对美国不公平。
 <br/>
 <br/>
 白宫表示，退出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万国邮联.html" target="_blank">
  万国邮联
 </a>
 是为了停止不公平的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邮政费率.html" target="_blank">
  邮政费率
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 制造商造成的伤害。
 <br/>
 <br/>
 在现行国际邮政体系下，从发展中国家寄送小包裹到美国，比从美国寄送到这些国家，运费更便宜。而邮递到美国的国际包裹中，来自中国的占大多数。
 <br/>
 <br/>
 据美国全国制造商协会统计，去年财年，美国邮政损失大约1.7亿美元。
 <br/>
 <br/>
 白宫官员表示，纠正邮费费率的不公平，还可以有效改善假货和毒品芬太尼的泛滥。
 <br/>
 <br/>
 中共外交部称，对此感到遗憾。
 <br/>
 <br/>
 中国电子商务和物流公司股价连续两天全面下跌，反映美国退出万国邮联，将推升中国企业寄产品给美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邮资.html" target="_blank">
  邮资
 </a>
 。
 <br/>
 <br/>
 阿里巴巴集团控股股价周三收盘下跌0.98%至148.16美元，中通快递股价昨大跌4.2%，中国第二大电子商务平台京东下跌3.86%。
 <br/>
 <br/>
 根据万国邮联的规定，退出程序为期一年。期间，美国国务院将通过双边或多边谈判。
 <br/>
 <br/>
 白宫发言人桑德斯表示，如果谈判成功，美国将撤销退出万国邮联的通知，留在联盟内。
 <br/>
 <br/>
 新唐人记者李梅、明玉综合报导
</div>
