### 美国零售行业销售连续两年暴增  今年超7200亿美元
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】美国零售业假期
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_销售.html" target="_blank">
  销售
 </a>
 连续两年持续增长，今年，预计增长4.3%～4.8%，总数超过7200亿美元。并且越来越多的人使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络购物.html" target="_blank">
  网络购物
 </a>
 。
 <br/>
 <br/>
 随着感恩节的过去，零售业的假日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_销售.html" target="_blank">
  销售
 </a>
 预测也新鲜出炉，今年假日销售最高的是日用品，电子产品则紧随其后。
 <br/>
 <br/>
 全国零售联合会资深主任 Ana Serafin Smith:〝我们预测今年的增长是4.3%～4.8%，如果要按照数字来说的话，总数达到了7200亿美元〞
 <br/>
 <br/>
 并且节假日期间也创造了很多新的就业岗位。
 <br/>
 <br/>
 全国零售联合会资深主任 Ana Serafin Smith:〝从11月和12月，我们预测会增加65万个就业岗位。〞〝零售业会从7月份开始招人。〞
 <br/>
 <br/>
 越来越多的人在网络进行购物，并且使用手机购物的年轻人，达到了88%。
 <br/>
 <br/>
 随着川普上任的两年来，美国经济持续增长，去年的零售业增幅达到了5.3% 创下了历史纪录。
 <br/>
 <br/>
 全国零售联合会资深主任 Ana Serafin Smith:〝这实在太高了，去年我们的销售额增长了5.3%，这是我们从没见过的〞
 <br/>
 <br/>
 新唐人记者王真纽约报导
</div>
