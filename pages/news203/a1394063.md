### 美国副总统彭斯讲话 多方面批中共
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】周四上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国副总统.html" target="_blank">
  美国副总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 就川普政府的对华政策发表演讲，指责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 通过军事、外交和贸易打击川普政府。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国副总统.html" target="_blank">
  美国副总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 ：〝美国人曾希望经济开放，能使中国成为我们以及世界的伙伴，但是，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 选择经济侵略，反过来让其军事上胆子大了。北京也没有像我们希望的，为其国民带来更多自由。过去，北京曾经对努力尊重人权表现出兴趣，但在最近几年，中共突然掉转方向，控制并压制其人民。今天，中共成为了一个空前监视（民众）的政权，越来越昂贵和强制，往往是利用了美国的技术。〞
 <br/>
 <br/>
 彭斯指出，中共正在以〝更加主动和强制的方式干涉美国国内政策和政治，〞瞄准〝将在2018年大选中发挥重要作用的行业和州〞，作为对川普贸易关税的回应。
 <br/>
 <br/>
 他说，美国情报界已经确定中共正在瞄准美国州和地方政府及官员，企图利用联邦和地方政府之间在政策方面的分歧。
 <br/>
 <br/>
 彭斯也警告其它国家，警惕与中共做生意，避免进入中共影响力轨道。
 <br/>
 <br/>
 彭斯同时也警告美国商界业要警惕中共利用市场准入为槓杆，试图改变企业的行为，以迎合中共的口味。
 <br/>
 <br/>
 彭斯还指责中共说服三个拉美国家切断与台湾的关系，明确美国不会对中共在南中国海的恐吓行动退缩。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
</div>
