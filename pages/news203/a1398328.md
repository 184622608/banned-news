### 从川普期中考看美国未来 专家解析2018中期选举
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 各地投票点，选民非常踊跃。有人说，这次选举将是是人生中最重要的一次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 。为何如此説呢？来听聼专家的解读。
 <br/>
 <br/>
 无论爱他、还是恨他，出来投票——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统，无疑是本次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的核心因素。
 <br/>
 <br/>
 川普在2017年就职演说时表示要抽干华府沼泽、把权力还给人民。上任至今不到2年，重点领域改革推进迅速，减税、发展经济；强化军力和边境安全；打击毒品；重塑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国际影响力；恢复对神的信仰。
 <br/>
 <br/>
 川普新政的推进，受益于共和党在国会两院的多数党位置。因此，本次中期，对于未来川普政策是否能顺利推进，有直接影响。而两党争夺的关键战场是众议院。
 <br/>
 <br/>
 时政评论人士 横河：〝美国政治在历史上很少有像今天这样，党派这么分明的。在国会的绝大多数关于国内政策的投票，几乎都是完全按照党派画线。如果国会能够和总统站在一起，对于完成这么困难这么重大的国家的转型，会有很大帮助。如果民主党控制众院，对于重大政策的立法会有一定的影响。〞
 <br/>
 <br/>
 国际政治风险顾问公司Corr Analytics负责人 Anders Corr博士：〝今天美国选民最关心的议题是税收和健保开支，这会决定众院由谁掌控。如果民主党控制国会，他们会阻止共和党进一步进行税改和解除健保改革、像对奥巴马健保的改变。也有很小的可能会发生弹劾总统的情况，但不会对总统职务产生影响。〞
 <br/>
 <br/>
 Anders Corr认为，即使民主党拿回众院控制权，也是有限的权力。
 <br/>
 <br/>
 国际政治风险顾问公司Corr Analytics负责人 Anders Corr博士：〝民主党重新控制众院后，他们能做的就是阻止进一步立法。但无法有效的提出新的重大立法提案。〞
 <br/>
 <br/>
 对于美中关系，专家表示，选举结果对此影响有限。
 <br/>
 <br/>
 国际政治风险顾问公司Corr Analytics负责人 Anders Corr博士：〝中国问题不是选举中的重要议题。即使民主党赢回众院，他们不会停止川普的对华严厉政策，可以预见对华的这种强硬政策至少还会持续2年直到2020年。〞
 <br/>
 <br/>
 时政评论人士 横河：〝对华政策，是两党唯一没有太大争议的地方，这个已经是美国社会的共识。在贸易战过程中，两党都是支持的。另一方面，美国总统在国际事务上，受国会的制约比较小。〞
 <br/>
 <br/>
 本次中期选举，除了是对川普总统2年执政的期中考试，也可以从中预见2020大选及美国更久远的走向。
 <br/>
 <br/>
 时政评论人士 横河：〝这次中期选举正好是在几十年来美国国内外政策重大的调整时期。从国内政治来看，美国这几十年，除了里根时期有短暂的保守主义回归以外，长期趋势是向大政府、高税收、高福利倾斜；意识形态上是偏向反传统的自由主义和进步主义。国际上，冷战结束后，美国做出过最大的误判，导致中共做大并成为共产主义的新旗手，而对整个自由世界形成了前所未有的威胁。川普总统的当选和他实行的政策，仅在2年时间之内就取得了惊人的成就，初步扭转并在继续扭转国内国际的趋势。不仅让美国重新伟大，而且在世界范围内抵制和对抗中共的影响、对抗共产主义。〞
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
