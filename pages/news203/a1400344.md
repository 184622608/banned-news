### 超市惊现圣诞老公公 神秘男子帮顾客结帐
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】圣诞节即将来及，很多能都希望自己能收到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞礼物.html" target="_blank">
  圣诞礼物
 </a>
 。在位于美国佛蒙特州德比市一家沃尔玛超市内，近日就有一名神秘男子主动充当
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞老人.html" target="_blank">
  圣诞老人
 </a>
 ，不仅慷慨为排队买单的顾客结账，还为很多预定商品的顾客也付了款。
 <br/>
 <br/>
 当天，也在超市排队等待结账的朱莉⋅盖茨（Julie Ann Gates）告诉美国广播公司ABC，她看到一名男子先是与排在她前面的男子说话。接着，这名男子又转向她并告诉她说：〝你可以再去拿一些你想要买的东西，我可以现在帮你
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结帐.html" target="_blank">
  结帐
 </a>
 ，也可以等你回来后再帮你买单。〞朱莉虽然有些不敢相信自己的耳朵，但她还是离开了队伍又去挑了一些商品。
 <br/>
 <br/>
 等她再次回到柜台时，看到那名男子还在等她，当时所有排队的人都已经离开了。朱莉不敢相信这是真的，她问这名男子为何要这么做，男子则说，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞老人.html" target="_blank">
  圣诞老人
 </a>
 会这么做的。〞
 <br/>
 <br/>
 该男子为她结账后就离开了。朱莉透露，在她前面排队的顾客购买的商品总价约为800到900美元。她说，〝他所做的才是我们过圣诞节的真正意义。〞
 <br/>
 <br/>
 朱莉补充说，〝当有人告诉我所有商品货款结清的时候，我感到这不像是真的，但事实证明这确实发生了。我相信圣诞节的精神就在我们周围，我会永远记住他。〞
 <br/>
 <br/>
 不仅如此，沃尔玛超市工作人员还透露，该男子不但为现场的顾客结帐，还为预定商品的顾客也买了单，存放在仓库内的商品都已派送到了顾客手中。不过，他没有透露这些商品的总价。
 <br/>

 <br/>
 <h4>
  西方传说中的圣诞老人
 </h4>
 <br/>
 据公开资料显示，圣诞老人（Santa Claus）又称Saint Nick、Kris Kringle、Father Christmas或Santa（爱尔兰英语的Santy），是一位专门在圣诞节前夜时悄悄赠送礼物给小孩子的神秘人物，也是节日主题的代表角色之一。他普遍被认为是基督教的圣人圣・尼古拉斯（Saint Nicholas）的衍生形像。
 <br/>
 <br/>
 传说每到12月24日晚上，有个神秘人会乘驾由9只驯鹿拉的雪橇在天上飞翔，挨家挨户地从烟囱进入屋里，然后偷偷把礼物放在好孩子床头的袜子里，或者堆在壁炉旁的圣诞树下。他在一年中的其他时间里，都是忙于制作礼物和监督孩子们的行为。
 <br/>
 <br/>
 虽然没有人真的见过神秘人的样子，但是人们会装扮成他的样子来给孩子送上礼物。他通常被描述为一位老人，头戴红色帽子，大大的白色胡子，一身红色棉衣，脚穿红色靴子的样子，拿着装有礼物的大袋子，因为总在圣诞节前夜出现派发礼物，所以习惯地称他为〝圣诞老人〞。
 <br/>
 <br/>
