### 美国将起诉阿桑奇 寻求引渡回美庭审
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】媒体披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 司法部正在准备起诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维基解密.html" target="_blank">
  维基解密
 </a>
 创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿桑奇.html" target="_blank">
  阿桑奇
 </a>
 ，并对将阿桑奇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_引渡.html" target="_blank">
  引渡
 </a>
 至美国受审表示乐观。
 <br/>
 <br/>
 《华尔街日报》周四披露，有华府消息人士表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 司法部正在寻求起诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿桑奇.html" target="_blank">
  阿桑奇
 </a>
 。
 <br/>
 <br/>
 报导说，美国检察官已经在去年就起诉阿桑奇一事展开过讨论。目前还不清楚指控的具体细节，但从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维基解密.html" target="_blank">
  维基解密
 </a>
 过去爆出的美国军事与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交机密.html" target="_blank">
  外交机密
 </a>
 来看，美国起诉阿桑奇的罪名中很可能包括间谍罪。
 <br/>
 <br/>
 2010年，维基解密公布了数十万份美军在伊拉克的机密军事文件。2016年美国大选时，维基解密又曝光了数千封希拉里竞选团队的内部电邮。
 <br/>
 <br/>
 自2012年以来，阿桑奇一直居住在厄瓜多尔驻伦敦大使馆内。尽管厄瓜多尔给了阿桑奇庇护以及公民身份，但阿桑奇仍面临来自英国警方的拘捕令，一旦离开大使馆就会被逮捕。
 <br/>
 <br/>
 去年开始，阿桑奇与厄瓜多尔之间的关系明显恶化。厄瓜多尔新任总统莫雷诺称，阿桑奇不能继续长期留在厄瓜多尔领馆。
 <br/>
 <br/>
 《华尔街日报》报导说，美国司法部正在考虑公开起诉阿桑奇，并公开相关证据的细节，以促使厄瓜多尔政府同意将阿桑奇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_引渡.html" target="_blank">
  引渡
 </a>
 到美国受审。
 <br/>
 <br/>
 不过，阿桑奇的律师表示，没有得到有关起诉的消息。
 <br/>
 <br/>
 新唐人记者林宇、张琪综合报导
</div>
