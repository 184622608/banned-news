### 斯科特重计票仍赢 川普再助选 保密州参院席位
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 联邦参议员以及州长竞选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 的结束显示，共和党参选人纷纷胜出。虽然共和党在众议院失去36席，不过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 回应，他支持的候选人绝大多数都赢得了选举。
 <br/>
 <br/>
 中期选举投票日前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在全国跑了44场活动。不过，他的助选旅程还未结束。11月26日，川普还将前往密西西比的图珀洛（Tupelo）和比洛克西（Biloxi）举办两场集会，为现任共和党籍联邦参议员史密斯（Cindy Hyde-Smith）拉票。
 <br/>
 <br/>
 密西西比州将于27日，进行一场复选投票，决定今年中期选举中联邦参议院最后一个席位的归属。史密斯将与民主党挑战者埃斯皮（Mike Espy）再次对决。
 <br/>
 <br/>
 本周二，史密斯与埃斯皮举行辩论。前副总统拜登在社交媒体上为艾斯皮发声拉票。
 <br/>
 <br/>
 而在刚过去的周末，即将卸任的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 州长、共和党籍联邦参议员候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯科特.html" target="_blank">
  斯科特
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 后胜出，他击败了现任民主党参议员纳尔逊。
 <br/>
 <br/>
 现任佛州联邦参议员 纳尔逊（民主党）：〝我没有赢得选举。我们在这长竞选中投入很多。〞
 <br/>
 <br/>
 白宫高级顾问 康威：〝在这个节日季这（佛州参议员竞选）真是场真正的胜利。总统的支持让结果不同。〞
 <br/>
 <br/>
 至此，参院民主党占47席，共和党确定赢得52席。
 <br/>
 <br/>
 另外，全美州长席位共和党占27人、民主党23人；众议院共和党200席，民主党232席。
 <br/>
 <br/>
 川普此前称中期选举是重大胜利，但有人对此质疑，因为共和党在众议院失去了36席。川普的回应是，他要的是参议院，而且他支持的参选人几乎都在选战中胜出。
 <br/>
 <br/>
 佛州选战，同样在重新计票后，共和党籍州长候选人德桑蒂斯也再次宣布获胜。
 <br/>
 <br/>
 佛州当选州长 Ron DeSantis（共和党）：〝川普在佛州有很好的选民基础，其他州也一样，他帮助这些竞选赢得胜利。〞
 <br/>
 <br/>
 德桑蒂斯明年就任州长后将任命3位联邦法官。
 <br/>
 <br/>
 同时，被斯科特起诉、涉嫌选票舞弊的布劳沃德郡选举总监布兰达斯奈普斯（Brenda Snipes）周日已递交辞呈，之前她因在2016大选时非法销毁选票而被判违反联邦法律。
 <br/>
 <br/>
 此外，同样在周末确定胜出的共和党籍乔治亚州州长候选人坎普，周一宣布了他的州长过渡团队。
 <br/>
 <br/>
 乔治亚州当选州长 Brian Kemp：〝我身后是乔治亚第一委员会。他们将和我一起工作确保乔治亚的继续伟大。过渡委员会将和我一道指定方案来增加工作、强化教育成果、降低健保成本、发扬乔治亚精神并保护乔治亚人家庭安全。〞
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
