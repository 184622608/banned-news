### 苹果发布新iPad 无Home键 比旧版快95%
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】美东时间10月30日上午，苹果公司在纽约举行了新品发布会。这次推出的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新版iPad Pro.html" target="_blank">
  新版iPad Pro
 </a>
 的突出特点是〝更轻、更薄、更快〞。
 <br/>
 <br/>
 苹果公司行政总裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库克.html" target="_blank">
  库克
 </a>
 在新品发布会上宣布，这次推出的新品包括Macbook Air、Mac mini及全新版本的iPad等产品。其中，最引人瞩目的是新版iPad，它的CPU处理器比旧版快95%。
 <br/>
 <br/>
 苹果硬件工程负责人在推介新品时说：〝新iPad从根本上改变了使用体验〞。
 <br/>
 <br/>
 据介绍，新版iPad容量高达1TB，屏幕亦较贴边采用，与iPhone XS不同的液晶体屏幕是没有〝M字额〞，并附有新版Apple Pencil，无需再将笔头插入Lightning插口，改用磁力附着iPad。
 <br/>
 <br/>
 新iPad Pro的尺寸包括11英寸和 12.9英寸两款，外形上比旧版本更轻薄。新版iPad取消了Home按键，其厚度仅5.9毫米，比老款薄了15%；体积则缩小了25%；支持Face ID，可以面部识别且具备最高安全性。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-31/p9103951a247086717.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-31/p9103951a247086717-ss.jpg"/>
   <br/>
   <font size="-1">
    苹果公司首席执行官提姆库克2018年10月30日在纽约布鲁克林音乐学院举行的Apple新品发布会上推介苹果公司的新产品。(Stephanie Keith/Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 在配置方面，新iPad Pro配备了A12X芯片，拥有7纳米技术，单核处理能力提升35%，多核提速达90%以上，其图形处理能力则比便携PC快1000倍。新iPad Pro的储存支持1TB，并配备了Type-C接口，可反向为iPhone充电。
 <br/>
 <br/>
 此外，苹果还推出了Apple Pencil，增加了无线充电功能，使用起来更加便捷。
 <br/>
 <br/>
 新iPad Pro将有银色和深空灰色两两款，并提供 64GB、256GB、512GB 以及新的 1TB 容量配置。
 <br/>
 <br/>
 其中，11 英寸 iPad Pro 无线局域网机型起售价为799美元；12.9 英寸iPad Pro 无线局域网机型起售价为999美元。
 <br/>
 <br/>
 库克在发布会上称，目前全球已卖出了4亿台iPad，去年销量超过了全球所有竞品的总和。
 <br/>
 <br/>
 （记者欣然综合报导/责任编辑：东野）
