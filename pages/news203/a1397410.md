### 匹兹堡犹太教堂枪案 川普夫妇亲往慰问
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】观众朋友大家好，欢迎收看新唐人全球新闻。10月30日，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_匹兹堡.html" target="_blank">
  匹兹堡
 </a>
 犹太教堂举行枪击案
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇难者.html" target="_blank">
  遇难者
 </a>
 的首次哀悼仪式。川普总统夫妇前来慰问。
 <br/>
 <br/>
 10月30日，川普总统夫妇专机抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_匹兹堡.html" target="_blank">
  匹兹堡
 </a>
 ，慰问被凶手枪击遇害的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受难者家属.html" target="_blank">
  受难者家属
 </a>
 ，以及数千名哀悼者。
 <br/>
 <br/>
 同一天，匹兹堡犹太教堂举行被枪击遇难的第一次葬礼哀悼仪式。来自各地的哀悼者前往犹太教堂向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇难者.html" target="_blank">
  遇难者
 </a>
 致哀。
 <br/>
 <br/>
 然而在近日，一些左派媒体试图将此次造成11人死亡的犹太教堂枪击案，以及近日针对左派人士或媒体的连环炸弹邮包案，归咎于川普总统。特别是炸弹邮包案嫌犯曾自称是川普支持者。
 <br/>
 <br/>
 川普也指出，嫌犯萨约克早就患有精神病。
 <br/>
 <br/>
 美国总统 川普：〝他（萨约克）很久以前就有精神病，你们看看他的治疗记录。他很长时间就有精神病。〞
 <br/>
 <br/>
 川普并指出，曾在去年枪击共和党议员，造成众议院多数党党鞭斯卡利斯重伤的枪击案嫌犯，是左派联邦参议员伯尼・桑德斯的支持者，但媒体却没有将枪案与桑德斯相连系。
 <br/>
 <br/>
 川普在周日推文中说：假新闻媒体总想将分裂和仇恨归咎于共和党，其实恰恰是假新闻媒体造成了分裂和仇恨。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
