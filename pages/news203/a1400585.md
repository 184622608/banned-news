### 各地美军欢度佳节 川普佛州摆感恩大餐
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】观众朋友大家好，欢迎收看新唐人全球新闻。美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 刚刚过去，各地民众欢度佳节的同时，驻扎在各处的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 ，也用自己的方式庆祝节日。请看报道。
 <br/>
 <br/>
 驻扎在美墨边境的德州部队准备了火鸡、牛排。众多士兵们相聚一起，自助享用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 大餐。
 <br/>
 <br/>
 远在欧洲罗马尼亚的美国空军基地，军人们准备了与家乡相同的蔬菜和传统火鸡，官兵共同餐叙，欢度佳节。
 <br/>
 <br/>
 而在德国的加里森安斯巴赫，美国陆军的士兵们，也选取各类蔬菜，与馅饼和火鸡一起，摆了一道自助大餐，分享欢乐时光，这是当地美国军人们多年的感恩节传统。
 <br/>
 <br/>
 美国三军最高统帅
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统，周四在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 访问了海岸警卫队之后，于当晚在棕榈滩的海湖庄园宴会厅举行盛宴，宴请亲朋和众宾客，共度感恩节。总统夫人梅拉妮亚、女儿伊万卡等亲属都出席了晚宴。
 <br/>
 <br/>
 川普周二来到佛州，感恩节假期结束后，下周一，他将前往密西西比州出席选战集会。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
