### 美国宣布退出《中导条约》  原因在于中共？
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）日前宣布，美国计划退出冷战时期与前苏联签署的《中程导弹条约》，原因是俄国违反条约规定。专家认为，川普欲施压
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 ，进而争取其合作抵制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （2018.10.20）：〝我们不会让他们违反核协议，（他们）发展武器，却不允许我们这样做。所以我们要停止协议，我们将退出。〞
 <br/>
 <br/>
 川普宣布，计划退出1987年由时任美国总统里根和苏联领导人戈尔巴乔夫签署的《中程导弹条约》，简称INF。按照条约，双方要各自销毁一定数量的中程导弹，并禁止持有、制造或试射射程介于500至5500公里的陆基弹道导弹或巡航导弹，目的是缓解冷战时期的紧张气氛。
 <br/>
 <br/>
 旅美时事评论员邢天行：〝他要寻求一个公平的原则，同时要保护美国的利益。由于这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中导条约.html" target="_blank">
  中导条约
 </a>
 对美国是一种束缚，而美国又认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 在违背条约，所以现在川普政府提出要退出中导条约。〞
 <br/>
 <br/>
 美国政府指控俄罗斯违反条约，在过去4年发展和部署新型巡航导弹，威胁欧洲国家以及与西方结盟的前苏联国家。而冷战后崛起的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 政权，又不受条约约束。
 <br/>
 <br/>
 川普说：〝如果俄罗斯做了，如果中共做了，我们却坚守条约，那是无法接受的。〞
 <br/>
 <br/>
 哈德逊研究所政治和军事分析中心主任Richard Weitz：〝这个决定有道理。影响很小。我认为俄罗斯已经预期美国会退出。〞
 <br/>
 <br/>
 Richard Weitz认为，川普以退出协议，换协商合作、欲联合俄罗斯对抗中共威胁。
 <br/>
 <br/>
 普京23号则提议，如果能和川普总统直接对话，将对美俄关系大有改善。
 <br/>
 <br/>
 俄罗斯总统 普京：〝在我看来，尽管我们各自有不同的观点，但我们必须且需要找到接触点。〞
 <br/>
 <br/>
 美国媒体引用美国官员和欧洲国家驻美外交官员的话指出，美国退出INF的部分原因是，中共不是中导条约的签署方，正不受约束地大力开发和部署新的、更致命的导弹，威胁到美国的安全。
 <br/>
 <br/>
 台湾大学国家发展研究所法学博士曾建元：〝这个条约长期忽略了对中国的发展中程导弹的约束，导致过去2、30年间，中国迅速成长，不仅针对台湾海峡部署导弹，事实上也威胁了美国。〞
 <br/>
 <br/>
 美国前亚太司令哈里斯（Harry Harris）今年三月份告诉参议院，〝我们没有可以威胁中共的陆基导弹能力，原因有很多，包括我们僵硬遵守（中导）条约。〞。美国已处于不利地位。
 <br/>
 <br/>
 川普也暗示，如果美国真的退出中导条约，他可能很快允许五角大楼开发对抗中共的导弹。
 <br/>
 <br/>
 白宫国家安全顾问博尔顿指出，中共最近暗示，它希望华盛顿留在条约内。这样可以绑住美国的手脚，而中共却可以完全不受约束。
 <br/>
 <br/>
 旅美时事评论员邢天行：〝中共／对美国的利益构成威胁，尤其对美国在太平洋地区这个利益构成了极大的威胁。美国其实也是在敲打俄罗斯，因为中俄宣称他们要加强战略协助。〞
 <br/>
 <br/>
 邢天行表示，针对中共这样一个危害世界的政权，川普正全方位追击它，退出中导条约只是川普实现目标的一个步骤，或者说是一个开始。
</div>
