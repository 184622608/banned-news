### 美国兆彩连槓3个月 头彩出现南卡州1注独得
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_兆彩.html" target="_blank">
  兆彩
 </a>
 （Mega Millions）已经3个月未开出头奖，奖金累计至16亿美元，打破历史纪录。引发全美疯狂投注，而23日晚间(当地时间)开出的头彩，已经出现在南卡罗来纳州。
 <br/>
 <br/>
 加州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乐透.html" target="_blank">
  乐透
 </a>
 公司表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_兆彩.html" target="_blank">
  兆彩
 </a>
 共5个号码加上一特别号，这次兆彩于美国23日晚上开奖，奖号为5、28、62、65、70，特别号为5。
 <br/>
 <br/>
 哥伦比亚广播公司报导，幸运儿落在南卡州。加州乐透公司官员表示，该州有8张彩票对中5个奖号。
 <br/>
 <br/>
 此次兆彩乐透自7月24日起开始连槓，引发全美疯狂投注，彩金累积至16亿美元。据估计，抱走头彩的中奖者扣除税金等，可领到约9亿1300万美元。
