### 美国5日重启对伊朗制裁 8国将得豁免
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 宣布，将于11月5日起全面恢复对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 的制裁，包括与伊朗有贸易往来和从伊朗进口原油的国家，但会有8个国家得到临时豁免。
 <br/>
 <br/>
 从下周一（11月5日）开始，川普政府将全面恢复对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 的制裁，制裁对象涵盖伊朗的航运、金融和能源等部门，并对继续从伊朗进口原油，以及继续与受到制裁的伊朗机构有商务往来的国家实施惩罚。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 财长姆努钦表示，在重新恢复的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁名单.html" target="_blank">
  制裁名单
 </a>
 上，又增加了超过700多家伊朗实体及个人。
 <br/>
 <br/>
 不过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 也提到，美国会对8个依赖伊朗原油的国家提供临时豁免。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝我们会给8个国家提供临时便利，但这只是因为他们已表明会大幅降低原油进口以及其它方面（与伊朗）的合作，并朝着（从伊朗）原油零进口的方向迈出了重要一步，这些谈判还都在继续进行中。〞
 <br/>
 <br/>
 美国国家安全顾问博尔顿也说，一些伊朗周边的国家，不大可能立即做到从伊朗零进口原油，而美国不想对这些国家造成伤害。
 <br/>
 <br/>
 美国国家安全顾问博尔顿：〝所以，我们希望达到最大化施压，但也不希望伤及友邦与盟国，我们正在想办法解决这个问题。〞
 <br/>
 <br/>
 周五当天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 能源部长多梅兹（Fatih Donmez）向媒体表示，土耳其已被告知会得到豁免。
 <br/>
 <br/>
 记者：〝土耳其在这个8个（豁免）国家中吗？〞
 <br/>
 <br/>
 土耳其能源部长多梅兹：〝是的，根据我们得到的消息，土耳其在其中，但我们还没有得到细节。〞
 <br/>
 <br/>
 预计日本、韩国、印度、中国等也都将出现在美国的豁免名单上。
 <br/>
 <br/>
 新唐人记者 乔安 综合报导
</div>
