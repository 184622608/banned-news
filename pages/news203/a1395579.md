### 微软联合创始人保罗・艾伦去世 享年65岁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微软.html" target="_blank">
  微软
 </a>
 联合创始人保罗・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艾伦.html" target="_blank">
  艾伦
 </a>
 周一（15日）下午去世，享年65岁。艾伦10月初表示，他9年前所患的非霍奇金淋巴瘤复发，并且已开始接受治疗。
 <br/>
 <br/>
 大纪元报导，保罗・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艾伦.html" target="_blank">
  艾伦
 </a>
 （Paul Allen）的妹妹乔迪・艾伦（Jody Allen），美国时间周一下午代表他的家人在一份声明说：〝我的哥哥在各方面都是一位了不起的人。虽然大多数人都知道
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保罗・艾伦.html" target="_blank">
  保罗・艾伦
 </a>
 是一名技术专家和慈善家，但对我们来说，他是一位深受爱戴的兄弟和叔叔，也是一位特别的朋友。〞
 <br/>
 <br/>
 她说，保罗的家人和朋友很幸运地感受到他的智慧、温暖、慷慨和深切关怀。尽管他的日程繁忙，但都会留些时间给家人和朋友。〝我们非常感激他每天所表现出的关爱和关注。〞
 <br/>
 <br/>
 保罗・艾伦是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微软.html" target="_blank">
  微软
 </a>
 公司的两位创始人之一。他生于美国华盛顿州西雅图，是华盛顿大学图书馆副馆长Kenneth S. Allen和Faye G. Allen的儿子。他是世界上最富有的人之一，拥有资产约227亿美元，其中包括约50亿的微软股票。
 <br/>
 <br/>
 艾伦是一名发明家、投资者、考古学家和慈善家。他的商业活动和慈善事业由他自己创办的公司管理。艾伦在技术、实业、不动产，媒体领域拥有数十亿美元的投资。艾伦还拥有2个专业球队：美式橄榄球联盟（NFL）的西雅图海鹰队，和美国职业篮球联赛（NBA）的波特兰开拓者队。
 <br/>
 <br/>
 2009年11月，艾伦被诊断患有非霍奇金淋巴瘤（non-Hodgkin‘s lymphoma），并接受化疗。
 <br/>
 <br/>
 今年10月初，艾伦表示，他9年前所患的非霍奇金淋巴瘤复发，并且已开始接受治疗。
 <br/>
 <br/>
 艾伦去世的消息传出后，苹果CEO蒂姆・库克在推特上发文，表达了对保罗・艾伦的悼念：〝我们的行业失去了一位先驱，我们的世界失去了一股善的力量，我们向保罗的朋友、艾伦家族和所有微软人致以最深的慰问。〞
