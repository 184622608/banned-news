### 美国圣诞季来临  传统点灯仪式祈神祝福
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】感恩节之后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 就开始进入热闹的圣诞节日季。过去一周，在华府和纽约都有隆重的圣诞树
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_点灯仪式.html" target="_blank">
  点灯仪式
 </a>
 。人们感谢神的恩赐，也祝愿新的一年幸福美好。
 <br/>
 <br/>
 寒冷的夜色中，川普总统夫人梅兰妮亚按下开关，在白宫椭圆形广场，点亮今年的国家圣诞树。闪亮的光芒带给人们节日的喜庆与祥和。这也是白宫延续了96年的传统。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统 Donald Trump：〝我们要一起来说，圣诞快乐！和你们在一起真的很好。〞
 <br/>
 <br/>
 在繁华的纽约洛克菲勒广场，圣诞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_点灯仪式.html" target="_blank">
  点灯仪式
 </a>
 已经成为重要的年度盛事。今年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞树装饰.html" target="_blank">
  圣诞树装饰
 </a>
 有五万多盏彩灯，最顶端的星星由300万颗水晶组成，晶莹璀灿。
 <br/>
 <br/>
 著名的华府双橡园已有130年历史。中华民国驻美代表处今年首次在这里举行圣诞点灯仪式。美国军中服务教区总主教布罗里奥(Timothy Broglio)，带领来宾们，共同为台湾人民及世界和平祈祷。
 <br/>
 <br/>
 中华民国驻美代表 高硕泰：〝希望在来年，2019年，大家平安喜乐，大家共享繁荣和平。〞
 <br/>
 <br/>
 美丽的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞季.html" target="_blank">
  圣诞季
 </a>
 节已经来到。人们在卡片上写下心中满满的感恩与祝福，希望新的一年幸福平安。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
