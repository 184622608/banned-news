### 美国中期选举在即 川普田纳西助选
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】周日晚，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和副总统彭斯在田纳西州查塔努加参加助选集会，受到当地民众热情支持。请看来自当地的采访报导。
 <br/>
 <br/>
 大批民众一大早来到集会现场，支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和共和党候选人。
 <br/>
 <br/>
 律师Chris Lanier：〝他遵守承诺，这很重要。另一位和川普类似，信守承诺的总统是里根。〞
 <br/>
 <br/>
 德国移民Wendy Madison：〝我认为，神选择了一位智者（川普），带我们走出经济萧条，希望能将我们更快地、带入更好经济时期。〞
 <br/>
 <br/>
 伊朗移民Sabrina Barahman：〝人们说你应该支持民主党人，你应该喜欢奥巴马，你应该反对（川普）政策。但我拒绝相信这些话。我喜欢川普，支持川普。〞
 <br/>
 <br/>
 营销人员Deby Moody：〝我喜欢川普，因为我们国家建立在基督教道德上，建立在基督教精神上。他也许不是每件事都做得完美，但他信守诺言，并且相信我们为神服务。〞
 <br/>
 <br/>
 前法律文书 Bobby Brown：〝两年来他为我们做了这么多，人们想帮助他（川普）。他需要共和党人掌控众议院，掌控参议院，从而推进他的计划。降低税收，帮助老兵。〞
 <br/>
 <br/>
 前法律文书Bobby Brown：〝他（川普）是改革者。主流媒体说他是（国家）分裂者，他们说错了。人们喜欢他（川普），因为他是团结（国家），而非分裂（国家）。
 <br/>
 <br/>
 集会主题曲《上帝保佑美国》的作曲和演唱歌手李・格林伍德，亲自献唱，表达对川普的支持。
 <br/>
 <br/>
 新唐人记者张婉君、陈雷、田纳西查塔努加采访报导
