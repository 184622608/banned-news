### 美众院共和党遭网络攻击 或为中期选举惨败原因
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】周二（12月4日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 全国委员会（National Republican Congressional Committee, NRCC）承认，他们在2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 前遭到了严重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络攻击.html" target="_blank">
  网络攻击
 </a>
 。
 <br/>
 <br/>
 NRCC发言人珀莱尔（Ian Prior）周二告诉福克斯新闻（Fox News），NRCC可以肯定自己受到了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络攻击.html" target="_blank">
  网络攻击
 </a>
 。珀莱尔称，由于数据库的网络安全至关重要，所以NRCC一发现受到黑客攻击便启动了内部调查并通知了联邦调查局（FBI），FBI目前正在调查该事件。
 <br/>
 <br/>
 但珀莱尔表示，为了保护调查的完整性，NRCC将不对此事件发表进一步评论。
 <br/>
 <br/>
 NRCC致力于在全国范围内选举
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 人。众议院共和党在11月的选举中失去了40个席位及多数党地位。
 <br/>
 <br/>
 美国政治新闻网站〝政客〞（Politico）周二最先报导了NRCC被黑客攻击的消息。据3名共和党高级成员透露，NRCC的4名高级助理的电邮账号被监视了数月，今年4月，NRCC的一个供应商才检测到黑客的入侵，并向NRCC及其网络安全承包商发出预警。
 <br/>
 <br/>
 但上述3名要求匿名的共和党人表示，NRCC虽然在得知攻击后便发起了内部调查并通知了FBI，但直到周一〝政客〞网站联系NRCC并询问此事时，NRCC居然还未把此事告知众议院共和党高层领导人，包括议长瑞恩（Paul Ryan）、多数党领袖麦卡锡（Kevin McCarthy）和多数党党鞭斯卡利斯（Steve Scalise）。众议院其他共和党人也不知道此事。
 <br/>
 <br/>
 NRCC主席、俄亥俄州共和党众议员斯蒂维斯（Steve Stivers）至今仍未就任何评论要求做出回应。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
