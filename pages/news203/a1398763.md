### 佛州可疑选票涌入民主党 州长下令调查
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 虽然大局已定，但仍有波澜。本次最受关注的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 州长及联邦参议员选战都有可能重新计票。佛州州长已下令执法机构，对不断增加的可疑选票进行调查。
 <br/>
 <br/>
 周四（11月8日）下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 州长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯科特.html" target="_blank">
  斯科特
 </a>
 下令佛州执法部门调查布劳沃德郡（Broward）和棕榈滩郡（Palm Beach）的选举机构，为何在投票截止48小时后，仍在收集和清点选票，而且新计入的选票都流向民主党参选人。
 <br/>
 <br/>
 以斯科特本人参与的联邦参议员选战为例，周二晚斯科特领先对手、现任民主党参议员纳尔逊约5万多张选票而宣布获胜；然而之后二人选票差距不断缩小，到周四下午，双方只差1万5千多张票，斯科特仅领先纳尔逊0.18%。
 <br/>
 <br/>
 佛州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_州长竞选.html" target="_blank">
  州长竞选
 </a>
 上也出现相似情况，共和党参选人德桑蒂斯周二晚宣布获胜、民主党对手吉卢姆承认败选，不过到周四，德桑蒂斯多于吉卢姆的选票缩小到了3万6千张左右、仅0.44%。
 <br/>
 <br/>
 佛州法律规定，如果选票差距小于等于1%需要机器重新点票；如果小于等于0.25%则需要人工重新计票。
 <br/>
 <br/>
 目前，纳尔逊和吉卢姆都要求重新计票。
 <br/>
 <br/>
 川普（特朗普）总统在推文中说：〝吉卢姆市长在选举日失败，现在布劳沃德郡让他‘起死回生’。纳尔逊败选了，现在他也要重新来过？这是对我们国家和民主的羞辱！〞
 <br/>
 <br/>
 佛州大西洋大学政治学教授Marshall L. DeRosa：〝棕榈滩郡和布劳沃德郡选举总监他们负责着这两个民主党占绝大多数县的运作。他们有备选方案，某些情况下就会启动后备方案，比如胶着的选票结果。目前我们还没有足够的信息，但有大量旁证证明布劳沃德郡存在大量（选举）舞弊。〞
 <br/>
 <br/>
 斯科特的竞选团队也提出诉讼，要求布劳沃德郡选举负责人提交收集和清点选票的报告，截至目前，该郡仍有大量早该在4天前就截止计票的先期投票和邮寄选票没有清点。棕榈滩郡则是拒绝公布相关信息。
 <br/>
 <br/>
 佛州州长斯科特：〝自普选日后的每一天，布劳沃德郡的左翼活动者不知从哪弄来越来越多的选票。他们的目标是偷偷的增加选票直到选举结果朝他们想要的发生变化。〞
 <br/>
 <br/>
 联邦参议员卢比奥（Marco Rubio）转发了一则推文，显示布劳沃德郡的选票被用私人车辆运送。
 <br/>
 <br/>
 〝两党政策中心〞民主项目主任John Fortier：〝对于州和郡县而言提供清楚的报告非常重要，特别是在选举日后仍有选票计入。让选民知道这些选票是真实的而不是有什么问题非常重要。〞
 <br/>
 <br/>
 美国总统川普：〝你看布劳沃德郡，他们的历史纪录非常糟糕。你看看与此事相关的布劳沃德郡选举总监布兰达斯奈普斯，她的纪录相当不好。突然之间不知哪来的选票。〞
 <br/>
 <br/>
 布劳沃德郡选举负责人布兰达斯奈普斯，今年因为其在2016大选时非法销毁选票而被判违反联邦法律。
 <br/>
 <br/>
 除了佛州，乔治亚州州长竞选、亚利桑那州联邦参议员竞逐，都存在类似问题。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
