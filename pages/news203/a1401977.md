### 【禁闻】美财长：见到中共具体行动才能达成协议

 【新唐人2018年12月04日讯】美国白宫证实，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统已经指定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易代表.html" target="_blank">
  贸易代表
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莱特希泽.html" target="_blank">
  莱特希泽
 </a>
 在未来90天内，就日前〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞上提出的结构性改革议题与中方谈判，美国财政部长姆钦则称，必须见到中方解决贸易纷争的具体行动，才能与中方达成真正协议。
 <br/>
 <br/>
 12月3号，美国白宫证实，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统已经指定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易代表.html" target="_blank">
  贸易代表
 </a>
 罗伯特．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莱特希泽.html" target="_blank">
  莱特希泽
 </a>
 （Robert Lighthizer）负责在未来90天内，就日前〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞上敲定的中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 、市场准入和知识产权实践等结构性改革议题，与中方谈判。
 <br/>
 <br/>
 莱特希泽是一位经验丰富的贸易谈判代表，不久前已经代表美方与加拿大、墨西哥达成新的北美自贸协定（USMCA）；在与欧盟和日本的类似谈判中，他也是主导人物。他是美国政府中最具声望的中共批评者之一。
 <br/>
 <br/>
 白宫贸易顾问纳瓦罗说，莱特希泽将继续进行有根有据的谈判，降低关税、降低非关税壁垒，并结束中共所有阻止市场准入的做法。
 <br/>
 <br/>
 美国洛杉矶加州大学经济学家俞伟雄：〝莱特希泽是非常有经验、非常专业、然后非常认真的美国的谈判代表，他在年轻的时候就代表美国跟日本谈判在80年代，他对中国以前贸易等各方面的作法都非常了解，他做代表当然就代表了川普坚强的意志。〞
 <br/>
 <br/>
 中华经济研究院特约研究员吴惠林指出，中共土匪似的重商主义贸易政策，已经严重伤害了美国经济，也伤害了世界贸易体系，因为川普政府明白中共内部分为两派，才给北京90天的时间，但是川普的决心是不会动摇的。
 <br/>
 <br/>
 中华经济研究院特约研究员吴惠林：〝这一次去谈判的当然都是所谓的鹰派，对中共非常了解，而且把中共和中国分开了，让习近平对中国在这个90天当中，怎么样让这个政体没有现在这样一种藉口，来做一些反抗。〞
 <br/>
 <br/>
 吴惠林认为，川普的贸易战本身就不是以打垮中国经济为目的，这一次给北京90天时间，体现了川普的善意。
 <br/>
 <br/>
 吴惠林：〝因为川普本身是一个商人，人家谈判的时候不会赶尽杀绝，然后给它留一些路，其实如果贸易战这样激烈打下去，中国经济垮了，对中国民众就业生活就造成很大的影响，所以要让他们怎么样有一种缓冲的余地。〞
 <br/>
 <br/>
 白宫关于美中首脑会谈的声明说，如果在90天内未能达成协议，10%的关税将会被提高至25%。
 <br/>
 <br/>
 12月3号，美国财政部长姆钦(Steven Mnuchin)又重申，美国必须在未来的90天内，见到中方解决贸易纷争的〝某些具体行动〞，才能与中方达成一项真正协议。姆钦希望中方信守诺言。
 <br/>
 <br/>
 中国金融智库研究员巩胜利：〝贸易关税就是3个问题，一个是关税、一个是补贴、一个是零壁垒，特朗普一直在推销他的零关税零壁垒零补贴，如果中国和美国搞成这个的话，那可能是全球最大的贸易，中国和美国就走在了全世界的前列。〞
 <br/>
 <br/>
 12月2号晚间，川普发布推文透露，中国已同意下调及取消从美国进口汽车的关税。目前这一关税税率为40%。
 <br/>
 <br/>
 姆钦表示，北京还在谈判桌上做出了另外1.2万亿美元的承诺。并且承诺向美国公司开放市场，保护美国技术。
 <br/>
 <br/>
 中国金融智库研究员巩胜利认为，中国买多少美国货物是很容易的事情，川习会最大的看点是结构性改革。
 <br/>
 <br/>
 巩胜利：〝结构性改革那就可以降低中国经济运行的成本，因为中国政府除了党政双合基因以外，还有它的政府的构架特别多。〞
 <br/>
 <br/>
 巩胜利说，美国和中国的经济加在一起接近全球经济的一半，川普倡导的零关税、零壁垒、零补贴的自由贸易，已经是大势所趋，如果中国能和美国合作，不但能使中国人民享受更好的生活，中国还能和美国一道主导全球的自由贸易体系。
 <br/>
 <br/>
 采访编辑/刘惠  后制/周天
