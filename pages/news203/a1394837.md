### 白宫颁布新规则 领取福利者难拿绿卡
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】周三，美国国土安全部公布《拒绝入境公共负担因素》
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新规则.html" target="_blank">
  新规则
 </a>
 ，确保申请美国签证和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绿卡.html" target="_blank">
  绿卡
 </a>
 的人〝不依赖公共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福利.html" target="_blank">
  福利
 </a>
 〞，目的在于保护美国纳税人。
 <br/>
 <br/>
 美国国土安全部表示，〝自给自足一直是美国移民法的基本原则〞。
 <br/>
 <br/>
 按照
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新规则.html" target="_blank">
  新规则
 </a>
 ，美国将首次在批准
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绿卡.html" target="_blank">
  绿卡
 </a>
 和延期签证时，考虑申请人是否领取食物券和住房补贴等。预计每年将有38万人受到影响，将给美国纳税人节省27亿美元。
 <br/>
 <br/>
 新规则只适用于申请签证或绿卡的外国人。大部分已经取得绿卡以及申请入籍的人士不受影响。
 <br/>
 <br/>
 对领取食物券、医疗保险处方药计划D部分、第八类房屋补助券等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福利.html" target="_blank">
  福利
 </a>
 的人，美国政府有权拒发绿卡或延期签证。
 <br/>
 <br/>
 国土安全部正在征询公众意见，领取〝儿童健康保险计划〞的外国人申请绿卡时是否也需要受到限制。
 <br/>
 <br/>
 新唐人记者王羽音、田瑞综合报导
</div>
