### 中兴案再现? 陆芯片厂疑窃技术 美下令限售
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国商务部.html" target="_blank">
  美国商务部
 </a>
 周一宣布，出于国安和经济安全考量，对〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 〞启动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_限售令.html" target="_blank">
  限售令
 </a>
 。晋华曾被指窃取美国最大存储芯片生产商美光科技的关键技术。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国商务部.html" target="_blank">
  美国商务部
 </a>
 周一宣布，〝福建省晋华集成电路有限公司〞参与了侵害美国国家安全的活动，从10月30日起，美国企业不经许可，不得向晋华提供原料、软体或技术。对晋华的再出口和跨国转移也必须取得授权。
 <br/>
 <br/>
 业界举例说，生产高端芯片必备的ASML光刻机，光源和光栅尺都来自美国。如果美国要求ASML不得将含美国原件的光刻机卖给晋华，晋华就将难以对光刻机进行维护和升级。
 <br/>
 <br/>
 张健：这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_限售令.html" target="_blank">
  限售令
 </a>
 等于一记绝杀，另一个对比的案件就是〝中兴〞案件。因为中国企业并不掌握所有核心技术，在这个产业链的源头，只要卡住了核心技术，中国高新企业都将面临中兴和晋华一样的困境。
 <br/>
 <br/>
 外界普遍认为，这次的限售令，和知识产权窃密有关。
 <br/>
 <br/>
 去年12月，美国最大的存储芯片生产商〝美光科技〞，在加州提起诉讼，指
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 及其技术伙伴台湾联电，透过美光员工，大量窃取美光的关键技术。加州法院正在审理。
 <br/>
 <br/>
 不过晋华随即在福建反告美光〝侵权〞。福建法院7月判决，有国资背景的晋华胜诉，美光则被要求暂停在中国销售26种芯片。
 <br/>
 <br/>
 而美光强调自己没有侵权，并指出晋华提出的都是无效专利，美光已递交强力证据要求复审。
 <br/>
 <br/>
 美国商务部周一发布声明说，福建晋华透过可能源自美国的技术，即将大量生产DRAM记忆体，这将威胁到一家美国军方供应商的生存能力，因此发布限售令。
 <br/>
 <br/>
 谢田：这实际上是美国总体战略的一部分，任何可能透过不合法的方式获取技术、来威胁美国国家安全的话，现在的政府他都要严厉追究。美国没有想去限制中国发展，现在的问题是，中共透过盗窃的方式、不公平竞争的方式，或强制转让的方式来发展，这个当然是美国政府不可能去认可的。
 <br/>
 <br/>
 这项限售令，被认为直指中共的2025计划。2025计划提出，到2020年芯片自给率达到40%，2025年达到50%。福建晋华是关键厂商之一。
 <br/>
 <br/>
 谢田：我想这个公司肯定不是唯一一家，因为中共盗取或强行索取美国科技，是全方位进行，等一个一个调查结束以后，可能到时都会出现。
 <br/>
 <br/>
 白宫首席经济顾问库德洛表示，美国已多次要求中方停止窃取知识产权，但中方至今没有回应。川普本周一再次表示，如果无法达成贸易协议，他将对所有中国进口商品加征关税。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
