### 美股暴跌 川普直言不赞同Fed升息
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】周三（10日）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 道琼工业指数重挫逾3%，标准普尔指数则创自2月以来最大单日跌幅。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）当天表示，美股抛售其实是等待已久的〝修正〞，他还说对数次升息的联邦储备理事会（Fed）的做法不敢苟同。
 <br/>
 <br/>
 FOX新闻报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周三在宾夕法尼亚州的政治集会前告诉记者：〝其实那是我们期待已久的修正。但我真的不同意Fed正在做的事情，好吗？〞〝我认为Fed已经疯了。〞
 <br/>
 <br/>
 白宫高阶官员告诉CNBC，川普周三已听取股巿暴跌的简报。官员说：〝这是牛巿修正，或许是健康的。这将会过去，美国经济依然强健。〞
 <br/>
 <br/>
 美国公债殖利率攀升，导致投资人对风险资产竞相走避，科技股全面通杀，加上对快速上涨利率的担忧，华尔街10日出现8个月来最大跌幅。
 <br/>
 <br/>
 道琼工业平均指数下跌831.83点或3.15%报25598.74点，下跌超过2.5%，是3月来最糟的一天。英特尔和微软各下跌超过3%。那斯达克综合指数暴跌315.97点或4.08%报7422.05点，跌幅超过6.5%。
 <br/>
 <br/>
 史坦普500指数下跌94.66点或3.29%报2785.68点，下跌超过3.6%，是2月以来的最大跌幅。科技类股带头下杀。大盘指数连跌5个交易日，是2016年底以来最长，并跌破重要的技术关卡50日移动均线。
 <br/>
 <br/>
 不断上涨的利率担忧和资金逃离科技股让过去几天的股市变得很艰苦。道指在过去5个交易日中下跌了4次，在此期间一跌近900点。
 <br/>
 <br/>
 亚马逊股价下跌6.15%，而Netflix下跌8.38%。脸书和苹果也各下跌超过4%。
 <br/>
 <br/>
 因预期美国经济至少还能再成长3年，Fed在9月底第三次上调短期利率，预计今年12月再升息1次，明年升息3次，2020年升息1次。届时将使隔夜借款利率来到3.4%，比Fed预估的〝中性利率〞水准高0.5个百分点，而这种紧缩的货币政策预计将持续到2021年。
 <br/>
 <br/>
 对于Fed的做法，川普在纽约的记者会上表达不满，指自己是一个低利率支持者。川普在今年稍早曾表示，美国经济正在成长，但每次成长起来，Fed就想再度升息，他对此并不满意，称他宁愿将注意力转移到美国经济的持续增长和创造更多就业机会。
 <br/>
 <br/>
 川普9日再批Fed升息，强调他不希望当前经济成长的势头放缓，〝即使只是一点点〞。
