### 川普或派1.5万美军驻守边境 规模恐超阿富汗驻军
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】为阻止从墨西哥穿越边境进入美国的中南美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统近日称，或派驻1.5万名美军驻守
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 ，规模或超过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 。
 <br/>
 <br/>
 中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 乘坐各式各样的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 涌向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 ，并欲强行闯入美国，已成为当前最让美国政府伤脑筋的一件大事。
 <br/>
 <br/>
 根据周四（11月1日）的最新消息，经过墨西哥想前往美国的大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 有4批团队，总人数可能高达6000。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 表示，〝我的政府正要完成一项计划，以终结对我们庇护制度的猖獗滥用。唯一终结这危机的永久性解决办法，来确保我们国家的主权，就是国会要克服开放边境的做法。〞
 <br/>
 <br/>
 美东时间周三（10月31日）下午，川普在白宫南草坪告诉媒体记者：〝美墨边境目前有边境巡逻队（U.S. Border Patrol）、移民与海关执法局（ICE），以及其他执法人员。除此之外，我们将再加派军力，将在边界的军人总数提高到1万至1.5万。〞
 <br/>
 <br/>
 此规模不但超越美军在叙利亚、伊拉克的驻军数量，更有可能逼近在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 驻扎的1.4万名美军。
 <br/>
 <br/>
 美阿战争是以美国为首的联军在2001年对阿富汗发起的战争，被视为反恐战争的开端，基地组织头目本拉登于2011年被美国海豹部队击毙后，美军陆续撤离。
 <br/>
 <br/>
 川普表示，大篷车即将接近边境，加派军力的目的是阻止大批中美洲公民非法进入美国。装备有器械设备的美国军人已抵达圣安东尼奥，并在德州开始行动。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
