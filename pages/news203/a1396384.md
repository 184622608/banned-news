### 中期选举前 川普称为中等收入者减税
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】去年12月，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 将公司税率从35％降至21％。预计11月初，美国政府又将推出对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中等收入者.html" target="_blank">
  中等收入者
 </a>
 的重大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_减税.html" target="_blank">
  减税
 </a>
 政策。同时也会制定针对贫困地区的投资优惠计划。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 上周六在参加内华达州埃尔科的集会后表示，政府正在研究对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中等收入者.html" target="_blank">
  中等收入者
 </a>
 的重大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_减税.html" target="_blank">
  减税
 </a>
 政策，并将在11月1日左右推出。
 <br/>
 <br/>
 美国总统川普：〝我们在考虑为中等收入的人减税。〞
 <br/>
 <br/>
 （记者）：〝是什么样的时间表？〞
 <br/>
 <br/>
 美国总统川普：〝11月1日左右吧，也许再早些。〞
 <br/>
 <br/>
 11月6日为美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 。共和党希望能够在参议院和众议院都保持多数席位，这将有利于川普推行他的政策。外界认为，新减税政策推出的时间点具有重大意义。
 <br/>
 <br/>
 去年12月，川普签署了自1980年代以来最大规模的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_税改.html" target="_blank">
  税改
 </a>
 ，将公司税率从35％降至21％，并减轻了大多数人的税收负担。
 <br/>
 <br/>
 美国众议院在今年9月底通过了第二轮减税措施，使个人减税永久化。
 <br/>
 <br/>
 根据去年通过的税改政策，个人方面的税收变化，包括降低税率和增加儿童税收抵免等等，都将在2025年到期。
 <br/>
 <br/>
 与此同时，美国财政部上周五制定了针对在贫困地区投资者的税收优惠计划指南。
 <br/>
 <br/>
 财政部长姆努钦在一项声明中表示，该计划为私人投资者提供重大税收优惠，预计贫困地区将因此而得到1000亿美元的私人投资，得以创造更多就业机会，促进经济发展。
 <br/>
 <br/>
 可投资的项目包括商业和工业房地产，住房，基础设施和初创企业。
 <br/>
 <br/>
 该项目惠及全美8761个社区，3500万人。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
