### FBI局长：中共在打未来之战 对美威胁远超俄国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】美国联邦调查局（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 ）局长克里斯托弗．雷10日在参议院国土安全委员会上表示，中共在很多方面是美国面对的最广泛、最复杂和最长远的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反间谍威胁.html" target="_blank">
  反间谍威胁
 </a>
 ，俄罗斯只关注当前，而中共在打明日之战。
 <br/>
 <br/>
 据路透社报导，克里斯托弗．雷在参议院作证时说，在前苏联垮台后，俄罗斯只是试图维持目前的局面，〝俄罗斯在面对今天的战役，而中国（中共）在打的则是明天的战争〞。
 <br/>
 <br/>
 雷指出，中共正构成严重威胁，尤其是美国的商业秘密、创意和创新技术，北京正在加大力度，以网络入侵、对美投资、企业收购和供应链威胁等不道德的方式获取。
 <br/>
 <br/>
 雷说，中共对美的商业间谍威胁，从大城市或郊区，从大公司到新创企业，从化学产业到农业，已影响到美国每一地区和每一个领域。
 <br/>
 <br/>
 雷称，面对这方面的威胁，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 各部门除了运用反间谍技术展开反击外，也在继续努力向私企和学界发出相关警示信息。
 <br/>
 <br/>
 9日，美国资安公司CrowdStrike Inc公布最新报告指，中国现在已经取代俄罗斯，成为对西方国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络攻击.html" target="_blank">
  网络攻击
 </a>
 的最大国家。在过去一年半里，中共黑客活动大幅增加，频繁窃取美国与西欧国家的生物科技、国防、采矿业、医药等领域的商业资讯。
 <br/>
 <br/>
 日前，白宫安全顾问博尔顿宣布，川普（特朗普）政府将协调各政府部门，对来自中共的网络攻击展开大规模反击。该反击行动将推翻前政府禁令，允许向对方发动主动攻击。
 <br/>
 <br/>
 另外，FBI正在对涉嫌盗窃美国尖端科技的中共〝千人计划〞展开调查。9月时，有网络消息传出，FBI已将参与〝千人计划〞的美国华裔学者列为调查重点，并按名单〝一个一个查〞。此前已有多名学者因涉嫌间谍罪被捕。
 <br/>
 <br/>
 另有多方消息指，FBI开始造访美国一些研究机构，有机构在FBI造访后开除〝千人计划〞学者，也有相关学者被要求向学校与工作单位提出自清。
 <br/>
 <br/>
 此外，白宫近来接连抨击中共干预美国中期选举。美国网络安全公司〝火眼〞透露，中共黑客开始加大力度，对美国一些政治团体和智库展开攻击。
 <br/>
 <br/>
 日前，美国副总统彭斯发表演讲，谴责中共在各个层面对美国的侵蚀和伤害，称在中方从新尊重美国之前，美方绝不让步。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
