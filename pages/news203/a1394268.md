### 手握抗共贸易联盟 美方回应中共：北京知道我们要求
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国家经济会议主席.html" target="_blank">
  美国国家经济会议主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 近日谈及美中贸易问题时表示，美国正与一些国家形成一个〝抗中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易联盟.html" target="_blank">
  贸易联盟
 </a>
 〞，如果中方想要认真谈判，美国也会随时做好准备。他说，中方完全知道美国的要求。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 （Larry Kudlow）10月4日出席了华府经济俱乐部举行的一场活动。在这场活动中，他表达了上述观点。
 <br/>
 <br/>
 据《世界日报》报导，库德洛坦言美国政府与墨西哥、加拿大达成新版贸易协定后，仍在继续与日本、欧盟对话，美国政府正与这些国家迈向形成一个〝愿意对抗中国（中共当局）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易联盟.html" target="_blank">
  贸易联盟
 </a>
 〞，接下来美国将集中精力对付中国（中共）。
 <br/>
 <br/>
 库德洛在这次活动中再度批评中共这些年来的贸易举措，包括窃取智慧财产权、强迫技术转移、对美产品征收高额关税等问题，他说：〝我们为什么要接受（这些不公平的做法）呢？〞
 <br/>
 <br/>
 此前，中共驻美大使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔天凯.html" target="_blank">
  崔天凯
 </a>
 10月3日在NPR总部接受采访时说，中国〝准备好达成一份协议〞，结束与美国的贸易战。但他同时又指责美国政府〝一直改变立场〞而错过了达成协议的机会。
 <br/>
 <br/>
 针对崔天凯的上述抱怨，库德洛在华府经济俱乐部的活动上回应说：〝我不同意（崔天凯的）这个说法，他们知道我们的要求。〞他并进一步表示，如果北京方面能够提出一些具体的回应，〝那就来谈吧〞。
 <br/>
 <br/>
 当有媒体记者追问，〝何谓具体回应？〞库德洛回答说：在技术转移、智财权、产品关税等方面，中方清楚地知道美方的要求。
 <br/>
 <br/>
 库德洛还表示，美国政府已准备好随时可以对话，但中方回应的关税措施令美国政府非常失望，导致美中贸易谈判进展缓慢。
 <br/>
 <br/>
 活动结束后，有媒体在采访库德洛时询问他：台湾是否能参与他所说的对抗中国（中共）的贸易联盟？库德洛并未针对台湾作出回应，只是强调说，美日欧三方会议、美墨加协定，都展现了大家〝团结一致〞避免受到不公平贸易行为的影响。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 今年7月6日正式爆发后，中美双方首先互相对价值500亿美元的商品加征高达25%的关税；9月24日，美国又向中国另外2,000亿美元的商品征税，中国也对美国另外600亿美元的商品征税，同时  为表达对美国加征新的关税的不满，中共国务院副总理刘鹤也取消了原定于9月末的访美贸易谈判之行。
 <br/>
 <br/>
 10月2日，美国总统川普（特朗普）在密西西比州的一次公开集会上说：〝你知道吗？他们（中共）想要谈判，他们想要达成协议。中国在过去几个月里被重创。他们的股市暴跌了30%，而自我当总统以来，我们的股市上涨了55%。〞
 <br/>
 <br/>
 川普还说：〝我希望我们能够同中国达成协议，但是老实说，谁知道呢？但是无论最后他们打算怎样，我们都会很开心。〞
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：李泉）
