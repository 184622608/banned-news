### 美国司法部：总统任命代理部长合法
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国司法部.html" target="_blank">
  美国司法部
 </a>
 公布内部法律意见，支持马修・惠特克被合法任命为代理司法部长。 川普总统周四连发推文，澄清对任命合法性的质疑，驳斥媒体不实报道。
 <br/>
 <br/>
 司法部法律顾问办公室公布了20页法律意见，表示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统任命.html" target="_blank">
  总统任命
 </a>
 惠特克代理司法部长，无需通过参议院确认。
 <br/>
 <br/>
 有司法部官员表示，前司法部长塞申斯下台前，白宫曾联系法律顾问办公室，了解职位空缺时的人选问题。
 <br/>
 <br/>
 法律顾问办公室认为，根据〝空缺改革法案〞，总统可不通过参议院确认，提升某人。
 <br/>
 <br/>
 川普总统周四连发推文，澄清对惠特克任命合法性的质疑，驳斥媒体不实报道。
 <br/>
 <br/>
 推文说：〝穆勒调查的内幕一团糟。他们没有发现串通，骑虎难下。他们对人叫嚣，威胁任命交出他们想要的东西。他们是民族的耻辱，不顾有多少生命毁掉。〞
 <br/>
 <br/>
 〝这些愤怒的人，包括非常矛盾的鲍勃・穆勒，他曾为奥巴马工作八年。他们从不看另一边犯下的所有恶行。美国历史上从来没有的〝猎巫行动〞！〞
 <br/>
 <br/>
 川普同时批评媒体不顾事实，尽管政府和国家都运行良好，但假新闻疯狂地抹黑总统。
 <br/>
 <br/>
 新唐人记者毕心慈、张琪综合报导
</div>
