### 侨报董事长被杀案开庭 嫌犯需缴300万美元保释金
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】洛杉矶县高等法院位于阿罕布拉市的分院，周一（26日）对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈忠启.html" target="_blank">
  陈忠启
 </a>
 涉嫌枪杀洛杉矶《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侨报.html" target="_blank">
  侨报
 </a>
 》董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢一宁.html" target="_blank">
  谢一宁
 </a>
 一案进行开庭审理。陈忠启面临最高入狱50年刑期，当天法官判决其保释金为300万美元。
 <br/>
 <br/>
 周一上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈忠启.html" target="_blank">
  陈忠启
 </a>
 （Chen，Zhong Qi）在阿罕布拉市（Alhambra）高等法院首次出席提审。据陈忠启的律师Donald Marks称，陈忠启在本案中主张不认罪。辩方与检方律师在当天庭审中主要对保释金金额的设置做了激烈辩论。
 <br/>
 <br/>
 辩护律师为其辩解称，陈忠启在案发前一天曾与公司高层发生争执，当天晚间喝了酒，整夜没睡，隔天与死者争吵时所配带的手枪，只是他个人用来自卫的武器。
 <br/>
 <br/>
 公诉方认为，因陈忠启可能对公共安全造成威胁，又有双重国籍，有逃回中国的嫌疑，最终法官将保释金定在了300万元。并且需戴上电子脚镣以便监控，而陈忠启的美国护照也被法院没收。该案预定将于12月18日上午8时30分再次开庭。
 <br/>
 <br/>
 检方律师Presciliano Duran表示，目前以谋杀罪起诉陈忠启，最高可判其入狱50年。
 <br/>
 <br/>
 该案案发于11月16日，在位于中共海外党媒洛杉矶阿罕布拉市（Alhambra）的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侨报.html" target="_blank">
  侨报
 </a>
 》大楼内，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢一宁.html" target="_blank">
  谢一宁
 </a>
 身中数枪当场身亡。《侨报》员工、56岁的嫌犯陈忠启当天下午被警方逮捕。
 <br/>
 <br/>
 11月19日，陈忠启以100万美元交保获释后，20日再度被捕入狱。经过阿罕布拉警局和洛县地检署共同调查后，将其保释金提高至600万美元。陈忠启在21日凌晨4点转押洛杉矶县监狱。
 <br/>
 <br/>
 因谢一宁和《侨报》背后都有中共官方元素，该命案迅速在华人圈引发震蘯。
 <br/>
 <br/>
 据悉，谢一宁1960年出生，原籍潮州。1982年新闻系毕业后进入中新社任记者，1987年外派担任《中新社》驻美国白宫记者。其正式头衔为《中新社》驻美国特派员。1991年，担任美国《侨报》首席记者，1992年起任该报美西版总裁，后任董事长至今。
 <br/>
 <br/>
 《中新社》全称为《中国新闻社》，是中共中央级喉舌媒体，中共统战部直属事业单位。中共自己曾承认全球80%的华文媒体接受《中新社》的供稿。
 <br/>
 <br/>
 《自由亚洲》电台17日报导说，谢一宁的朋友陆先生透露，谢一宁是《侨报》董事长，他的前妻是该报总经理；而他现任妻子是位于洛杉矶的凤凰卫视主播郭晶晶。
 <br/>
 <br/>
 凤凰卫视是中共的〝第二个央视〞。美国凤凰台是香港凤凰卫视的美国分台。香港凤凰卫视是中共在香港设立，中共前党魁江泽民之子江绵恒是大股东。
 <br/>
 <br/>
 美国独立非盈利机构詹姆斯通基金会（Jamestown Foundation），早在2001年曾披露，中共做出了巨大努力介入北美的中文媒体，美国的中文报纸：《世界日报》、《星岛日报》、《明报》和《侨报》等都受中共直接或间接的控制。
 <br/>
 <br/>
 而《侨报》则是直接受中共控制，是中共国务院侨办的，其资金和主要编辑、管理人员全部来自北京。该报的特点是大量及时地报导来自中共的消息，代表着中共官方的声音和观点。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
