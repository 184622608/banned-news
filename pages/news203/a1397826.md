### 美劳工部报出重大利好数据  川普称赞〝难以置信〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】美国劳工部周五（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_10月.html" target="_blank">
  10月
 </a>
 2日）报告，美国10月份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新增工作.html" target="_blank">
  新增工作
 </a>
 岗位250,000个，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业率.html" target="_blank">
  失业率
 </a>
 稳定、保持49年来的最低点——3.7%，同时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工资.html" target="_blank">
  工资
 </a>
 增长率也出现了较大幅度的提升，多年来首次突破3%。
 <br/>
 <br/>
 经济学家此前预测，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_10月.html" target="_blank">
  10月
 </a>
 份将新增就业岗位190,000个，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业率.html" target="_blank">
  失业率
 </a>
 稳定在3.7%，维持9月份创下的自1969年以来的最好记录。不过，统计结果显然比预期的还好。
 <br/>
 <br/>
 根据劳工统计局（Bureau of Labor Statistics）发布的数据，9月份美国新增就业岗位134,000个，低于预期，主要原因在于当月〝佛罗伦斯〞（Florence）飓风猛烈侵袭了北卡罗来纳州和南卡罗来纳州。飓风登陆及洪水肆虐期间，很多酒吧和餐馆停业，岗位需求急剧减少，对当月工作造成了很大影响。
 <br/>
 <br/>
 10月份，就业状况显然出现良好反弹，工作岗位比专家所预期的多出60,000个。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新增工作.html" target="_blank">
  新增工作
 </a>
 岗位遍及各个经济领域。其中，制造业增加了32,000个工作，耐用品生产行业、特别是运输设备生产行业尤为突出。与专家们的预测相左，川普政府征收钢铝进口税并没有给制造业、特别是使用金属的工业造成多大压力。
 <br/>
 <br/>
 尽管人们担心房地产市场已经放缓，但建筑业的增长却出人意料，新添了30,000个工作岗位。
 <br/>
 <br/>
 采矿业10月份也增加了5,000个工作。
 <br/>
 <br/>
 其它新增工作所涉及的行业包括：运输和仓储，42,000个；医疗保健，36,000个；休闲和招待，42,000个；专业和商业服务，35,00个。
 <br/>
 <br/>
 在美国近两来蓬勃发展的经济中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工资.html" target="_blank">
  工资
 </a>
 增长是最弱的一环。不过，10月份工资增长率达到了3.1%，是经济衰退后多年来首次突破3%。
 <br/>
 <br/>
 周五，众议院议长瑞恩（Paul Ryan）在《华盛顿观察家报》（The Washington Examiner）发表专栏文章指出，良好的经济指标显示，川普时期的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 人正在兑现他们在2016年大选时做出的承诺。
 <br/>
 <br/>
 他表示，美国的工资和福利正以10年来最快的速度增长，各个家庭也都直接感受到了经济增长带给他们的影响。
 <br/>
 <br/>
 瑞恩敦促选民，要在下周二选举日那天出来投票支持共和党，以便共和党能够在成功的基础上继续保持经济强劲，并为更多的美国人拓展机会。
 <br/>
 <br/>
 川普周五上午在推文中分享了劳工部报告的好消息，同时表示，〝这些数字令人难以置信。保持下去，投票支持共和党人！〞
