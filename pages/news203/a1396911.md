### 卡舒吉儿子抵美 CIA局长返美汇报川普
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】来关注沙特记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉被杀案.html" target="_blank">
  卡舒吉被杀案
 </a>
 的最新进展。美国中央情报局局长哈斯佩尔，已经从土耳其返回美国，向川普总统报告了案件调查情况。同时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 的长子，获准离开沙特抵达美国。他之前被禁止出境。
 <br/>
 <br/>
 美国中情局局长吉娜・哈斯佩尔，返回美国后向川普做了简报。哈斯佩尔周一前往土耳其，调查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 遇害真相。据报，哈斯佩尔在土耳其听到一份现场录音。不过，这一消息没有得到白宫证实。
 <br/>
 <br/>
 同时，卡舒吉的长子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_萨拉．卡舒吉.html" target="_blank">
  萨拉．卡舒吉
 </a>
 ，获得沙特官方批准离开沙特，和家人一起抵达美国，和他母亲等亲人团聚。
 <br/>
 <br/>
 萨拉・卡舒吉拥有美国和沙特双重国籍，之前因为父亲对沙特王室、特别是王储的批评，被禁止离开沙特。
 <br/>
 <br/>
 离开沙特前两天，萨拉・卡舒吉与沙特王储萨勒曼握手的照片传出。照片上，萨拉冷眼看着萨勒曼。外界怀疑，萨勒曼可能下令杀害卡舒吉。
 <br/>
 <br/>
 不过，萨勒曼王子坚决否认他对案件知情。他在周三首次就事件公开发声，誓言将把凶手绳之以法。
 <br/>
 <br/>
 美英两国已经宣布，对涉入卡舒吉案的沙特官员实施制裁，禁止其入境。
 <br/>
 <br/>
 川普之前接受《华尔街日报》专访时表示，希望萨勒曼王子与事件无关，但要看调查结果。
 <br/>
 <br/>
 土耳其总统埃尔多安，周五再度敦促沙特当局，公开卡舒吉的遗体，追查下达命令的人。
 <br/>
 <br/>
 目前，沙特司法部长莫杰布已经前往土耳其处理事件。
 <br/>
 <br/>
 新唐人记者林宇、张琪综合报导
</div>
