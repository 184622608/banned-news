### 布县藐视裁决继续计票 佛州州长下令强制执行
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】周日（11月11日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 州长斯科特（Rick Scott）向该州布劳沃德县（Broward）和棕榈滩县（Palm Beach）法院提出两项紧急动议，同时下令执法部门采取法律行动，以保证
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 工作顺利进行。
 <br/>
 <br/>
 据《国会山报》（The Hill）11月11日报导，斯科特的动议要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 执法部门和县警扣押并保护布劳沃德县所有未在使用的投票机、选票和制表设备。
 <br/>
 <br/>
 斯科特的竞选团队还向布县法院提起诉讼，请求法官禁止该县选举委员会在周六中午的最后期限后所计的选票算入最终计票中。
 <br/>
 <br/>
 斯科特竞选团队的发言人哈特林（Chris Hartline）在声明中表示，布县和棕榈滩县的选举监督员已经表现出公然无视佛州选举法，这使佛州政府尽一切可能防止欺诈并确保
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 得以负责任地操作变得更为重要。
 <br/>
 <br/>
 哈特林声明，当选参议员斯科特将继续为保护佛州选民的意志而奋斗。
 <br/>
 <br/>
 周六，佛州州务卿戴兹特（Ken Detzner）宣布，鉴于该州州长和参议员选举的计票结果非常接近，因而启动重新计票机制，重新计票结果将于美东时间下周四（11月15日）下午3点前出台。
 <br/>
 <br/>
 佛州规定，如果获胜候选人所赢得的选票高出对手不足0.5%时，将启动强制性重新计票。如果相差幅度低于0.25%，重新计票工作必须手工完成。
 <br/>
 <br/>
 选举夜过后，倾向于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 的布劳沃德县和棕榈县不但迟迟不能完成计票工作，而且在接下来的两天中选票突然激增，且多数流向民主党候选人。
 <br/>
 <br/>
 至周六下午，斯科特的领先优势已被压缩为0.15%，德桑蒂斯的优势也降低到0.41%。
 <br/>
 <br/>
 由于质疑布县和棕榈县选举舞弊，周四晚，面对日益严重的事态，斯科特提起诉讼，并下令佛州执法人员展开调查。
 <br/>
 <br/>
 令人震惊的是，布县选举监督员斯奈普斯（Brenda Snipes）不服从法院判决，当佛州共和党众议员盖茨（Matt Gaetz）按法官命令赶到布县选举总部要求审阅并复印选举记录时，却被斯奈普斯拒之门外。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
