### 寄毒信给川普 嫌犯购蓖麻籽：为第三次世界大战做准备
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】五角大楼日前收到有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒邮件.html" target="_blank">
  毒邮件
 </a>
 后，特勤局披露，一封寄给总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）的信件也含有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓖麻毒素.html" target="_blank">
  蓖麻毒素
 </a>
 〞。联邦调查局（FBI）随后逮捕了涉嫌犯案的海军退役成员艾伦。艾伦告诉调查员，购买蓖麻籽是为第三次世界大战做准备。
 <br/>
 <br/>
 美联社10月6日报导，艾伦（William Clyde Allen III）自称，曾向一些外国领导人寄过类似信件，包括英国女王伊丽莎白二世、俄罗斯总统普京和空军部长等，但目前尚不清楚这些信件是否被发现。
 <br/>
 <br/>
 检方指控艾伦5项罪名，包括将生物毒素用作武器和邮件威胁。此案预计将由大陪审团进行审理，艾伦可能会在10月18日的听证会上面临额外的指控。如果指控的各项罪名成立，艾伦将面临终身监禁。
 <br/>
 <br/>
 艾伦现年39岁，1998年10月至2002年10月在海军服役，他在犹他州有犯罪记录，包括虐待儿童和企图加重攻击。法院文件显示，艾伦告诉调查人员，他在电子商务网站亿贝购买蓖麻籽，用以在〝第三次世界大战爆发时保护我们的国家〞。 目前，美国执法部门仍在调查他寄送信件的动机。
 <br/>
 <br/>
 在5日开庭审理时艾伦没有提出抗辩，他的代理律师也没有发表评论。警方也没有对他的精神问题发表评论。
 <br/>
 <br/>
 美国白宫和国防部本月早些时候拦截多封疑似含有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓖麻毒素.html" target="_blank">
  蓖麻毒素
 </a>
 （ricin）的信件。
 <br/>
 <br/>
 美国司法部盐湖城办公室表示，联邦调查局（FBI）3日在针对寄疑似内含蓖麻毒素邮件给总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 一事，在犹他州洛根（Logan）一所住宅逮捕了艾伦。
 <br/>
 <br/>
 另一个执法单位消息表示，艾伦是因涉嫌寄发有问题邮件给国防部官员和川普而遭到调查。
 <br/>
 <br/>
 五角大厦的邮件检查设施1日至少拦截遭到当局怀疑内含蓖麻毒素的两件包裹，收件人分别是国防部长马蒂斯（Jim Mattis）和海军作战部长理查森（John Richardson）。
 <br/>
 <br/>
 据了解，在毒信事件事发的同一天，德州共和党参议员克鲁兹（Ted Cruz）位于休斯敦的竞选办公室也收到了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_有毒信件.html" target="_blank">
  有毒信件
 </a>
 ，两名工作人员因接触到信封中的白色粉末而被送院救治。而两件毒信事件是否有关联也尚不得而知。
 <br/>
 <br/>
 美国国防部官员3日稍早表示，经化验后寄到五角大厦的可疑包裹内含可萃取蓖麻毒素的蓖麻籽，而不是致命的蓖麻毒素。
 <br/>
 <br/>
 据悉，蓖麻毒素从蓖麻籽中提取而得，为剧毒物质，少量摄取、吸入或注射即可致人在36～72小时内中毒死亡。它比氰化物强6000倍，目前还没有解毒剂。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
