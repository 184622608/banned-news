### 炸弹包裹嫌犯佛州落网 面临五项控罪
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】观众朋友大家好，欢迎收看新唐人全球新闻。一周来，美国的民主党高层或相关人士不断收到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸物包裹.html" target="_blank">
  爆炸物包裹
 </a>
 ，周五，警方在佛罗里达州种植园市逮捕了涉案嫌犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞亚克.html" target="_blank">
  塞亚克
 </a>
 。
 <br/>
 <br/>
 被捕的嫌犯为56岁的西萨・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞亚克.html" target="_blank">
  塞亚克
 </a>
 （Cesar Sayoc），公开资料显示，塞亚克现居住在佛州阿文图拉，此前曾有多次犯罪记录，其中包括炸弹威胁。
 <br/>
 <br/>
 就在同一天，警方也先后拦截要寄给新泽西民主党参议员布克（Cory Booker）、前国家情报总监克拉柏（James Clapper）和加州民主党联邦参议员贺锦丽（Kamala Harris）的炸弹包裹，使这礼拜全美的炸弹包裹事件达到13起，此前收到包裹的人士包括前总统奥巴马，前副总统拜登及前民主党总统候选人希拉里等。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国司法部.html" target="_blank">
  美国司法部
 </a>
 长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞申斯.html" target="_blank">
  塞申斯
 </a>
 周五宣布，已对塞亚克提起五项控罪。
 <br/>
 <br/>
 美国司法部长塞辛斯：〝他（塞亚克）今天已被控五项联邦罪名，包括跨州运输爆炸物，非法邮寄爆炸物，威胁前总统等人，威胁跨州通讯，以及攻击现任及前任联邦官员。
 <br/>
 <br/>
 如果这些罪名全部成立，塞亚克将面临最高58年监禁，由于调查尚未结束，对塞亚克的控罪还有可能会增加或变动。
 <br/>
 <br/>
 目前，执法部门仍无法确定塞亚克的作案动机。
 <br/>
 <br/>
 此前曾指示执法部门不遗余力抓捕嫌犯的美国总统川普，在得知嫌犯被捕后，对FBI及司法部的高效工作表示称赞。川普再次重申，不容许政治暴力在美国立足。
 <br/>
 <br/>
 美国总统川普：〝我们绝不容许政治暴力在美国扎根，不允许这种情况发生。我承诺将会竭尽总统所能，现在就制止其发生。〞
 <br/>
 <br/>
 川普同时强调，美国人必须团结一致，向世界展示美国人的平和、关爱与和谐。
 <br/>
 <br/>
 新唐人记者乔安综合报导
</div>
