### 坎普当选乔治亚州长  民主党对手不服欲上诉
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】周五，乔治亚州〝州长选举〞的计票风波，暂时告一段落。共和党候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎普.html" target="_blank">
  坎普
 </a>
 ，宣布胜选。民主党候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艾布拉姆斯.html" target="_blank">
  艾布拉姆斯
 </a>
 ，虽然承认败选，但誓言，将针对选举过程，起诉乔治亚州政府。
 <br/>
 <br/>
 民主党候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艾布拉姆斯.html" target="_blank">
  艾布拉姆斯
 </a>
 ，周五举行记者会承认败选。不过，她批评选举制度存在漏洞，将对此进行上诉。
 <br/>
 <br/>
 乔治亚州州长民主党候选人 艾布拉姆斯：〝未来几天内，我们将对乔治亚州提起重大联邦诉讼，控诉对这次选举显而易见的管理不当，保障未来的选举免于违宪操作。〞
 <br/>
 <br/>
 数个月来，艾布拉姆斯指责共和党候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎普.html" target="_blank">
  坎普
 </a>
 利用州务卿身份干预选举，但坎普对此坚决否认。
 <br/>
 <br/>
 艾布拉姆斯承认败选后，坎普在脸书发表胜选声明。
 <br/>
 <br/>
 根据非官方的统计，坎普目前已经获得了该州50.3%的选票，因超过半数得以直接胜选。
 <br/>
 <br/>
 美国总统川普周五发表推文，恭喜坎普赢得州长选举，同时也称赞了他的对手艾布拉姆斯努力有天分，政治前景光明。
 <br/>
 <br/>
 至于其他有计票争议的选战，共和党候选人德桑蒂斯经过重新计票，当选佛州州长。而佛州参议员选举则因差距过于接近，仍在人工计票中。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
