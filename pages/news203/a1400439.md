### 边境危机似升级 川普：关闭美墨边境全线
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】感恩节当天，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在佛州渡假庄园表示，目前南部边境有失控的危险，在考虑关闭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 全线。他已经下令国防部提供更多协助，确保边境安全。
 <br/>
 <br/>
 来自中美洲的移民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 一辆辆抵达墨西哥北部边境。
 <br/>
 <br/>
 墨西哥蒂华纳居民 Jesus Olvera：〝大家不习惯看到这种状况，像大篷车移民，这么巨大的人数。这让蒂华纳变得不安全和不稳定，这是我看到的。 〞
 <br/>
 <br/>
 美国国土安全部的信息显示，已经有超过6千大篷车成员聚集在与美国接壤的墨西哥边境城市，他们在当地落脚并获得食物、水及衣物等援助。不过，随着一辆接一辆大篷车登陆，墨西哥政府周三逮捕了第四辆抵达的大篷车上的大约213名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 。
 <br/>
 <br/>
 大篷车成员 洪都拉斯人 Denis Martínez：〝我们加入大篷车，但有一些牵头的人沿途寻求帮助。所以是他们、他们决定的。感谢他们让我们来到这儿。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周四表示，边境情况不容乐观，正在考虑要关闭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 全线。
 <br/>
 <br/>
 美国总统 川普：〝我们发现（边境）情况有点要失控、我们的人可能会被攻击伤害，将会关闭边境一段时间直到情况得到控制。我指的是边境全线。全部边境。〞
 <br/>
 <br/>
 川普已下令国防部提供更多援助、确保边境安全。
 <br/>
 <br/>
 国防部长 马蒂斯（2018.11.21）：〝总统确实看到边境巡逻需要增援。我们昨天深夜获得进一步授权来完成新任务。目前正在评估和准备。我也已经跟国土安全部的人谈过此事。〞
 <br/>
 <br/>
 前一天，国土安全部长尼尔森在南部边境视察时表示，国防部会提供支持，直到目前的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境危机.html" target="_blank">
  边境危机
 </a>
 得到解决。
 <br/>
 <br/>
 目前美墨边境有大约5800名军人，他们的主要任务是提供建墙、加固设施等工程类支持。新任务可能会包括处理边境暴力冲突。此前有6名墨西哥边境执法人员在冲突中受伤。
 <br/>
 <br/>
 盖洛普（11月1日-11日）民调显示，移民与犯罪外国人议题已经跃居美国人关注首位，人数从上个月的13%增长为21%。
 <br/>
 <br/>
 新唐人记者李兰、特约记者 Kimberly Hayek 美国纽约、墨西哥蒂华纳连线报导
</div>
