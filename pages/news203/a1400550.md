### 大篷车移民逼近边境 川普：或将关闭边境
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】数百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 ，周四从墨西哥的蒂华纳，前往美国圣伊西德罗入境口岸，要申请美国庇护。周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统表示，如果发现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境.html" target="_blank">
  边境
 </a>
 失控，将关闭美国边境。
 <br/>
 <br/>
 墨西哥在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境.html" target="_blank">
  边境
 </a>
 口岸部署了防暴警察，并设置路障。虽然墨西哥为移民提供工作许可，但部分移民仍要前往美国。
 <br/>
 <br/>
 洪都拉斯移民 JAVIER AGUIRRE：〝我们不是来蒂华纳找工作，我们来这是为了去美国。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统表示，只有合法入境的移民，才能申请庇护。申请过程可能需要六个月。
 <br/>
 <br/>
 墨西哥方面认为，移民需要了解，要想越过边境，必须登记名字，清楚每天的人口配额。
 <br/>
 <br/>
 墨西哥国家人权委员会官员 EDUARDO SISOSA：〝他们缺乏很多信息，（有了这些信息）他们可以看到多种选择。〞
 <br/>
 <br/>
 美国总统川普周四表示，如果发现边境失控，美国将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关闭边境.html" target="_blank">
  关闭边境
 </a>
 。
 <br/>
 <br/>
 美国总统川普：〝我们发现事情发展到这种程度：（边境）可能失去控制，或者我们的民众会受到伤害。我们会关闭边境一段时间，直到我们能控制。〞
 <br/>
 <br/>
 川普表示，如果关闭整个边境，墨西哥将蒙受损失。
 <br/>
 <br/>
 周五一早，川普发推文，呼吁两党合作，保护边境。
 <br/>
 <br/>
 推文说：〝共和党人和民主党人必须合作，最终（达成）主要的边境安全方案，包括为边境筑墙提供资金。经过40年的谈话，最终该采取行动了。修复边境，一劳永逸，现在就做！〞
 <br/>
 <br/>
 墨西哥移民官员估计，可能有一万难民将聚集在美墨边境城市蒂华纳和墨西加利。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
