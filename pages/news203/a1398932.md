### 加州南北野火肆虐已致25人丧生
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】连日来美国加州多处爆发的野火，导致死亡人数升至25人，随着调查人员到达被野火摧毁的地区，死亡人数或再增加。
 <br/>
 <br/>
 11月8日开始出现的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山火.html" target="_blank">
  山火
 </a>
 在加州北部布尤特县和南部的文图拉和洛杉矶县周遭地区不断燃烧，连续三天火势不减，过火面积不断扩大，到10日已达到425平方公里，一些社区被夷为平地，摧毁建筑物超过6700处。目前，南、北加州合计共有25万多人被疏散。
 <br/>
 <br/>
 根据加州林业与防火局的数据，本次山火目前已成为有记录以来造成死亡人数第三多的山
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火灾.html" target="_blank">
  火灾
 </a>
 难。
 <br/>
 <br/>
 加州迄今已投入至少810万美元灭火。加州当局表示，自8日以来，已投入数千消防员全力救火，动用了灭火专用飞机、铲土机等大型机械，更多的消防员将前往火灾地区。天气预报显示，到周一每小时的风速将达到每小时80公里，这增加了火势进一步扩散的风险。
 <br/>
 <br/>
 美国林业局的消防队长史密斯：〝我们最大的担心就是风，风势增长的地方对火情真的很有影响，我们尽可能控制火势。〞
 <br/>
 <br/>
 与此同时，山火已造成严重空气污染，多个城市开始向市民发放面罩，以减少污染对人体健康造成的危害。
 <br/>
 <br/>
 新唐人记者严枫、田瑞综合报导
</div>
