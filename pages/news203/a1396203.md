### 中期选举迫近   川普身处政坛〝颱风眼〞
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】
 <b>
  新闻周刊
 </b>
 （650）四年一次的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 将在11月6号举行。中期选举历来被认为是对现任总统两年执政成绩的〝期中考核〞。而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 就任后，美国国会两党分歧加剧，更加凸显这次中期选举的重要性。
 <br/>
 <br/>
 还有半个月的时间，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 就将举行。国会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 全部435席和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 100席中的35席，都将进行改选。此外，还有39个州的州长和其他地区性选举也将举行。
 <br/>
 <br/>
 随着中期选举临近，总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 的集会行程也越加密集。仅在两周之内，川普就接连造访爱荷华、宾夕法尼亚、俄亥俄、肯塔基、蒙大拿和亚利桑那州。川普还将在周六和下周一分别在内华达州和德克萨斯州为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 候选人助阵。
 <br/>
 <br/>
 这次选举被视为对川普执政满意度的一次公投，以及2020年大选的风向标。对美国乃至世界未来的政治走向，至关重要。分析认为，川普在这次中期选举中，处于〝颱风眼〞的位置。
 <br/>
 <br/>
 《标准周刊》副主编 Ethan Epstein：〝他是这次选举的中心，尽管他的名字不在选票上。他想以此激发对他的支持。另外一个激发他助选的因素是，他不想失去共和党对众议院的控制。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 方面，要想在众议院或者参议院夺回多数党地位，需要在参议院至少增加两席，在众议院至少要增加24席。
 <br/>
 <br/>
 分析认为，经济议题是对总统川普有利的因素。今年第二季，美国经济创下近四年来最快增速，国内生产总值环比增长4.1%，是2014年第三季以来的最佳表现。
 <br/>
 <br/>
 美国总统 川普：〝我们国家的失业率降到3.7%，是50年来的最低点。家庭收入中位数达到最高点，非常好，这意味着你们比以往任何时候都赚得更多的钱。〞
 <br/>
 <br/>
 此外，移民议题也成为川普反复批评民主党的要点。本月中美洲再次出现〝大篷车〞难民潮，川普已经表态，必要时将派军队驻守边境。
 <br/>
 <br/>
 美国总统 川普：〝我们在这里説话的时候，民主党正在公开邀请数百万的非法移民，让他们来破坏我们的法律、进犯我们的边境，并击溃我们的国家。〞
 <br/>
 <br/>
 随着中期选举临近，川普多次在社交媒体贴图，提醒选民注册中期选举，号召支持者走出来投票。
 <br/>
 <br/>
 而根据皮尤研究中心的报告显示，川普是激励选民出来投票的重要因素。在乔治亚州，早期投票率和四年前相比增长了230%。在亚利桑那，选民的注册人数也是四年前的两倍以上。
 <br/>
 <br/>
 同时，共和党内大部分获得川普背书的候选人，都在初选中获胜。川普的〝光环效应〞，成为此次中期选举的一大特点。
 <br/>
 <br/>
 根据民调结果，选情激烈的选区包括，佛州现任共和党籍州长斯科特挑战民主党人纳尔逊的国会参议员位置。密苏里州民主党参议员麦卡斯基尔，与共和党候选人霍利民调几乎持平。
 <br/>
 <br/>
 分析认为，从选区形势看，共和党在参议院改选中占据有利地位。而众议院由于全部改选，形势较难预测，也是民主党寄托希望的所在。
 <br/>
 <br/>
 另一方面，美国也在严防外国势力企图干预选举结果，尤其是来自北京和莫斯科的威胁。周五，《美国之音》引用美国政府多个执法部门发表的联合声明，其中表达了对俄罗斯、中国、伊朗等国家破坏美国民主制度的担心。
 <br/>
 <br/>
 美国总统川普在接受采访时表示，他认为来自中共的威胁最大。
 <br/>
 <br/>
 美国总统 川普：〝他们（俄罗斯）是干预了，但我认为中国（共）也干预了。坦白说，我认为中国（共）的问题更大。〞
 <br/>
 <br/>
 有专家认为，中共与俄罗斯不同之处，在于企图掩盖对海外的政治渗透，以达到长期的政治目标。对美国的民主制度来説，中共的威胁要远远超过俄罗斯。
 <br/>
 <br/>
 撰稿：林宇
