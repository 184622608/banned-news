### 贸战〝停火〞协议细节 中美说法现4大差异
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞在布宜诺斯艾利斯落幕之后，中美双方均通报了会晤的成果，均称双方达成暂时停火的协议，但有英媒梳理发现，美中两国对协议内容的说法存在四大不同。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）与中国国家主席习近平，于12月1日晚，在阿根廷布宜诺斯艾利斯举行了晚餐会后，中美双方均通报了这次会晤的成果，均称就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 达成〝暂时休战〞的协议，但是舆论注意到双方披露的内容存在明显区别。
 <br/>
 <br/>
 路透社3日盘点了美中两国对协议内容的说法存在的4大不同。
 <br/>
 <br/>
 一是，对90天的停火限期说法不同。
 <br/>
 <br/>
 白宫声明称，如果在90天后双方仍无法达成协议，美国将对中国2000亿美元商品的关税提高至25%。而中方通报只称中方会〝使美方的合理关切得到逐步的解决〞，但对于90日限期则只字不提。
 <br/>
 <br/>
 二是，对中国经济的构造改革的说法不同。
 <br/>
 <br/>
 白宫声明称，在强制技术转移、知识产权保护、非关税壁垒、网路攻击、服务以及农业等领域，川普与习近平就迅速推进构造改革达成一致。而中共外交部长王毅对此并没有具体提及，只是在记者会上说，中方将扩大从美国进口商品的规模，努力修正中美之间的贸易不平衡状况。
 <br/>
 <br/>
 三是对美国高通对荷兰恩智浦半导体（NXP）发起的收购案说法不同。
 <br/>
 <br/>
 白宫声明称，高通若再次申请，中方将允许这一收购案。而中共高官在记者会上没有提及这一议题，只有中国社交媒体有所提及。
 <br/>
 <br/>
 四是，对中国进口美国商品的说法不同。
 <br/>
 <br/>
 白宫声明表示，中方将〝立刻〞开始大量进口美国的农产品、能源以及工业制品，但未披露进口规模。而中共通报中没有提及扩大进口的美国商品种类，以及何时开始进口。
 <br/>
 <br/>
 对于美中双方达成暂时停火协议，有中国学者的观点认为，北京形同被判三个月缓刑，中共官媒不愿向国人如实汇报，正是因为这对中共是致命的，这种隐瞒不报说明北京当局不想真正做出变革。
 <br/>
 <br/>
 但也有人认为，北京隐瞒关键细节，与中国国内及中共政坛日趋分裂有关，习近平不想在进行变革之前，触怒仍掌握经济命脉的中共权贵家族。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
