### 近半美军预期明年陷冲突  对手是中俄
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】因中共对美国民主自由构成威胁，川普政府把中共和俄罗斯、伊朗、朝鲜一样看待。美国已将大国竞争调整为战略首要重点。一份民意调查显示，近半
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 官兵预期美国将很快陷入一场大战，对手是中俄。
 <br/>
 <br/>
 据美国《军事时报》（Military Times）16日报导，该报与锡拉丘兹大学退伍军人和军人家庭研究所合作，9月20日至10月2日对美国军方人员进行了自愿和保密的在线民意调查。
 <br/>
 <br/>
 该调查问卷涵盖了从太空部队到他们对总统的支持以及美国是否应继续向伊拉克派兵等话题。调查显示，约46%的军人预期美国将很快陷入一场大战，并预期全球不稳定，特别是对中俄的话题不断增加。
 <br/>
 <br/>
 报告称，在2017年9月进行的类似调查中，同样的问题只有5%认为会发生冲突。虽然仍然有50%的人认为美国明年不会陷入重大冲突，但这一比例也较去年的调查明显下降，去年的调查显示，有2/3的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 官兵认同〝不太可能发生战争〞。
 <br/>
 <br/>
 美军的预期源于中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 高烧不退，美中两国可能濒临全面战略对抗。一位川普（特朗普）总统的资深幕僚告诉《路透社》，中共为了影响美国的大众舆论，增进中共政权与共产党的利益，几乎动用了一切可以利用的手段，涵盖了政治、经济、商业、军事与资讯科技等多个方面。
 <br/>
 <br/>
 《BBC》外交事务记者罗宾斯（James Robbins）认为，中共干预美国的选举和民主进程的作法已经越过了〝底线〞。正是因为对美国民主自由构成威胁，川普政府才把中共和俄罗斯、伊朗和朝鲜一样看待。
 <br/>
 <br/>
 近年来，为了应对亚太地区越来越大的安全挑战，美军经常以轮调的方式将B-2战略轰炸机派驻关岛。本月初，美国空军在夏威夷完成了首次常态化部署B-2隐形战略轰炸机，此举将对确保印太地区的自由与开放发挥重要作用。
 <br/>
 <br/>
 美国之音报导引述防务专家们表示，美军正分散在太平洋地区的军事资源，以减少受到中共军方攻击的风险，但如果发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事冲突.html" target="_blank">
  军事冲突
 </a>
 ，美军仍然需要迅速从不同地方调派资源，因此B-2战略轰炸机和MV-22型鱼鹰倾转旋翼机的远程飞行能力非常重要。
 <br/>
 <br/>
 而另一方面，北约规模最大的海陆空三军演习，称为〝三叉戟接点〞计划，将于10月25日到11月7日举行。10月17日，美国海军陆战队已抵达冰岛，之后将前往挪威。
 <br/>
 <br/>
 这将是冷战结束以来，北约最大的军演。北约过去多年一直削减军费，并将注意力集中在比较遥远的中东等地。这次军演表明，北约重新将目光投向了自家后院，加强了对俄罗斯的防御。
 <br/>
 <br/>
 （记者李红报导）
</div>
