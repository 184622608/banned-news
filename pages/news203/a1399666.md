### 美国今年首场冬季风暴 至少7人死亡
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】观众朋友大家好，欢迎收看新唐人全球新闻。周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 迎来第一场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冬季风暴.html" target="_blank">
  冬季风暴
 </a>
 ，影响区域巨大，造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停电.html" target="_blank">
  停电
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交通瘫痪.html" target="_blank">
  交通瘫痪
 </a>
 等诸多问题。
 <br/>
 <br/>
 猛烈的暴雪，令人措手不及。
 <br/>
 <br/>
 在世界上最繁忙的大桥之一，纽约乔治华盛顿大桥严重塞车，人们被困数小时。
 <br/>
 <br/>
 纽约市民Smit Patel：〝嘿伙计们，我站在华盛顿大桥上，你看多拥堵，桥上出了场大车祸。〞
 <br/>
 <br/>
 纽约市最大的交通枢纽之一，港务局巴士总站，也在周四深夜停摆。
 <br/>
 <br/>
 新泽西州长Phil Murphy：〝巨大的混乱，是因为公交没办法穿过哈德逊河过来载客，所以公交和乘客数量脱节的情况如此高。〞
 <br/>
 <br/>
 周五，对新泽西州长的批评声不断，该州发生了约1000起车祸。
 <br/>
 <br/>
 新泽西州长Phil Murphy：〝暴雪侵袭的猛烈程度，超出任何人或任何组织的预测，这是事实。〞
 <br/>
 <br/>
 在新泽西的一所中学，学生只能留在学校过夜。
 <br/>
 <br/>
 马里兰州府报告，在6小时内已经有至少300起车祸。
 <br/>
 <br/>
 目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雪.html" target="_blank">
  暴风雪
 </a>
 造成至少7人死亡，其中主要是车祸事故。
 <br/>
 <br/>
 而在东海岸，也造成超过2000架航班被取消，8000多架被延迟。
 <br/>
 <br/>
 飞机延误乘客Breanna Taylor：〝我们被困在飞机上2小时。〞
 <br/>
 <br/>
 这场早冬大雪的打击让美东各地措手不及，破坏性超出预料，让许多民众惊叹不已。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冬季风暴.html" target="_blank">
  冬季风暴
 </a>
 艾弗里（Avery）沿东海岸行进，超过8000万民众受到北极寒流（Arctic blast）影响，冬季风暴警报从密苏里州圣路易斯，穿越阿巴拉契亚山脉，直达纽约和新英格兰地区。
 <br/>
 <br/>
 据poweroutage.us称，截至周五早些时候，约有30万户
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停电.html" target="_blank">
  停电
 </a>
 ，主要在弗吉尼亚州，俄州和宾州。
 <br/>
 <br/>
 这场早冬暴雪，为纽约和宾州部分地区，带来超过1英尺积雪，也为各地带来创纪录降雪量。
 <br/>
 <br/>
 纽约中央公园降下6.4英吋积雪，创下136年以来11月份单日降雪最高纪录。
 <br/>
 <br/>
 新泽西迎来创11月单日纪录的6～8英吋积雪。
 <br/>
 <br/>
 华盛顿DC为1.4英吋，打破112年前0.2英吋的纪录。还有巴尔的摩、费城等也纷纷刷新纪录，带来1.7英吋和3.6英吋降雪。
 <br/>
 <br/>
 新唐人记者宇亭纽约报导
</div>
