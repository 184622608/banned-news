### 彭斯展开亚洲行 将出席东盟APEC等会议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】周一，美国副总统迈克・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 及夫人卡伦・彭斯抵达日本，开始亚太访问之旅，彭斯将参加在巴布亚新几内亚召开的APEC
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_会议.html" target="_blank">
  会议
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 此行将代表美国总统川普，出席〝东盟峰会〞、东亚峰会，以及在巴布亚新几内亚召开的APEC
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_会议.html" target="_blank">
  会议
 </a>
 。
 <br/>
 <br/>
 彭斯还将会见日本首相安倍晋三、印度总理穆迪以及澳大利亚总理莫里森等国家首脑，并进行一系列双边会议。
 <br/>
 <br/>
 APEC会议主办国、巴布亚新几内亚周日表示，准备工作已经完成。
 <br/>
 <br/>
 巴布亚新几内亚APEC会议部长JUSTIN TKATCHENKO：〝巴布亚新几内亚可能很长、很长时间内不会（再有类似活动）。我们必须确保做对了，而且做得很好。〞
 <br/>
 <br/>
 巴布亚新几内亚购买了一批用来接送各国首脑的豪华车，修建了新道路。巴布亚新几内亚是亚太地区最贫穷的国家。
 <br/>
 <br/>
 11月16日至18日，包括美国副总统彭斯和中国的习近平在内的15,000多名各国代表，将在巴布亚新几内亚出席APEC会议。
 <br/>
 <br/>
 为保障APEC会议安全，澳大利亚向巴布亚新几内亚首都、莫尔兹比港派遣了1500名国防军人。
 <br/>
 <br/>
 会议期间，澳大利亚战斗机将在空中巡逻。
 <br/>
 <br/>
 澳大利亚陆军少校 Paul McLachlan：〝我们和巴布亚新几内亚的关系，可能是我们最重要的地区防御关系。当巴布亚新几内亚需要我们，我们能来到这就很重要。〞
 <br/>
 <br/>
 为防止中共渗透，澳大利亚正在加强在南太平洋地区的军事地位。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
</div>
