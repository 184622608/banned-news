### APEC会议美中激烈交锋  法媒:川普彭斯施双重心理战
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】日前，在巴布亚新几内亚举行的亚太经合组织（APEC）峰会上，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 和中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在发言时，围绕贸易、地区安全等一系列敏感议题，互相发出了措辞尖锐的批评。法媒评论称，在阿根廷〝川习会〞前，美国政府开始对北京实施双重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心理战.html" target="_blank">
  心理战
 </a>
 。
 <br/>
 <br/>
 为期两天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 17日在巴布亚新几内亚开幕，在当天举行的工商领袖会议上，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 和中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 展开了激烈的交锋。
 <br/>
 <br/>
 据《美国之音》报导，美国副总统彭斯在APEC峰会上发表讲话时，措辞严厉地抨击了中共政府通过对外大举贷款令其他国家陷入债务陷阱的做法。他暗指北京推行的〝一带一路〞是一条有去无回的〝单行道〞。他并称，我们不会拿出〝单行道〞计划。
 <br/>
 <br/>
 彭斯警告与会各国不要被中国〝不透明支票〞外交所诱惑。他表示，中共政府提供的融资，不仅是存在〝不透明〞的问题，更严重的是，中方捆绑的贷款条件通常都会给接受援助的国家带来〝令人眩晕的巨额债务〞。他奉劝各国不要接受可能损害自己国家主权的外国援助。
 <br/>
 <br/>
 彭斯还当场呼吁亚太各国站在美国一边。他说：〝我们不会将合作伙伴淹没在债务海洋中，我们不会强制，也不会腐蚀贿赂，不会损害你们的独立。〞
 <br/>
 <br/>
 在发言中，彭斯还提到了加征关税的问题，他直言：〝我们已经对2500亿美元的中国输美产品加征关税，而这个数目有可能翻倍。〞彭斯明确指出，〝我们希望情况有所改善，但只要中国立场不变，美国的行为也不会变。〞
 <br/>
 <br/>
 就在彭斯发表上述前，习近平刚刚在APEC峰会上发言称，美国政府提出的〝美国优先〞政策是〝保护主义〞和〝单边主义〞。他还说，国与国之间如果走上对抗道路，无论是冷战、热战，还是贸易战，都不会有真正的赢家等。
 <br/>
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-18/p9131501a784812855.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-18/p9131501a784812855-ss.jpg"/>
   <br/>

    美国副总统迈克．彭斯2018年11月17日在莫尔兹比港举行的亚太经济合作组织（APEC）峰会首脑会议上发表了主旨演讲。(SAEED KHAN/AFP/Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 <br/>
 针对中美两国首脑在APEC峰会上的激烈交锋，法新社评论说，这次亚太经合峰会的背景是中美两国争夺太平洋影响力之战。
 <br/>
 <br/>
 法广则发文分析称，彭斯这次在APEC峰会上的表现，与前两天美国总统川普在谈及中美贸易冲突问题时比较温和的态度形成了鲜明对比，显示川普和彭斯在G20峰会到来前，分别〝唱起了红脸、黑脸〞。
 <br/>
 <br/>
 今年的20国集团峰会将于11月30日在阿根廷举行，届时川普与习近平将藉此机会举行一对一的双边谈判，外界认为，这是中美两国政府避免贸易战继续升级为新冷战的一个重要契机。
 <br/>
 <br/>
 本周五，川普对外证实，中方近日已向美方递交了一份〝相当完整的清单〞，以回应美国提出的改革要求消息。但他表示，因为清单中〝漏掉了四、五件大事〞，自己现在还不能接受这份清单。
 <br/>
 <br/>
 川普同时表示，他相信美中双方最终可以达成一个双方都能够接受的协议。他说，他可能不会对中国商品征收更多关税，他相信在与中国达成的任何协议中，被遗漏的项目都将得到解决。
 <br/>
 <br/>
 然而一天之后，彭斯却在APEC会议上发表了对中共态度十分强硬的讲话，誓言〝中国不改变，美国不转向。〞
 <br/>
 <br/>
 而此前，彭斯还通过《华盛顿邮报》放话称，〝北京应就在美方向其所提出要求的经济、军事和政治行为做出彻底的改变〞。他并指出，〝阿根廷峰会是中国避免与美国发生冷战的最好机会〞。
 <br/>
 <br/>
 法广针对上述情况分析称，如若说川普近日的发言还略显温和的话，彭斯在对华纠纷的态度上则十分强硬，川普与彭斯的这种态度上的差异，是两人在G20峰会到来前对北京实施的〝双重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心理战.html" target="_blank">
  心理战
 </a>
 〞，其最终目的则是要促使中方按照美方的要求改变贸易现状。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
