### 哈佛歧视案下周开审   川普政府挺亚裔
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】下周，广受瞩目的〝哈佛大学歧视亚裔申请人〞一案将在美国波士顿联邦地方法庭开审。该案涉及到美国平权行动政策(affirmative action, 简称AA)的适用问题，预计将对美国的教育系统有重要影响。
 <br/>
 <br/>
 6月15日，学生公平招生组织(SFFA)，针对哈佛大学入学政策对美国亚裔学生存在歧视一案，提交证据。下周一(10月15日)，波士顿联邦地方法庭就将开始审理此案。专家认为，这场官司恐怕会旷日持久，并极可能上诉到最高法院。
 <br/>
 <br/>
 平等机会中心总裁兼总顾问 Roger Clegg：〝我充满希望，最终此案结果会是，我们的教育系统不允许高校歧视，无论基于肤色、或他们祖先来自的地方。〞
 <br/>
 <br/>
 除大学外，亚裔家长担忧，歧视现象在美国中学也有存在。马里兰州蒙郡是全美最好的公校系统之一，但其资优班入学标准近两年来发生变化，种族平衡成为考虑因素之一，亚裔学生比率随之下降两成。
 <br/>
 <br/>
 美国马里兰州州长 Larry Hogan：〝我有三个亚裔女儿，都很聪明，受到很好教育。没有任何人应在任何地方因种族受到歧视。我知道在一些大学、一些地方发生了这种状况，但不会在马里兰州发生。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 上任后，推翻了奥巴马时代有关种族意识招生政策的指导方针，并先后对哈佛、耶鲁大学入学歧视问题展开调查。同时美国教育部要求，所有接受联邦资金补助的学校，必须严格遵守最高法院对大学
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_择优录取.html" target="_blank">
  择优录取
 </a>
 的相关规定。
 <br/>
 <br/>
 平等机会中心总裁兼总顾问 Roger Clegg：〝1964年的民权法案，禁止接受联邦资金的机构有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_种族歧视.html" target="_blank">
  种族歧视
 </a>
 行为，只有少数大学是私立并拒绝联邦资金。但即使这样的大学，也受其它联邦法律限制(禁止歧视)。〞
 <br/>
 <br/>
 上周六，新任最高院大法官卡瓦诺宣誓就任。未来，保守派占多数的最高院将对此案持何种立场，令人关注。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
