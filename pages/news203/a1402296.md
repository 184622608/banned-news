### 博尔顿：不会对中共贸易行为视而不见
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】周四（12月6日），美国国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 （John Bolton）在接受媒体采访时表示，在未来几个月的美中谈判中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易.html" target="_blank">
  贸易
 </a>
 紧张局势仍是谈判关注的焦点，美国不会对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 不守规则的贸易行为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视而不见.html" target="_blank">
  视而不见
 </a>
 。
 <br/>
 <br/>
 据美国全国公共广播电台（NPR）12月6日报导，周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 告诉该媒体，当有些国家把窃取竞争对手的知识产权作为国家政策奉行时，美国不该对此听之任之。
 <br/>
 <br/>
 博尔顿是在回答关于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司首席财务官孟晚舟被捕的问题时做出上述表述的。
 <br/>
 <br/>
 针对美国要求逮捕并引渡孟晚舟一事，NPR的早间综合新闻节目〝Morning Edition〞采访了博尔顿，但博尔顿拒绝透露孟晚舟被捕的原因。不过他表示，美国早就在关注华为公司窃取知识产权的问题。
 <br/>
 <br/>
 博尔顿告诉主持人英斯克普（Steve Inskeep），华为是美国早已关注的一家技术公司，另外还有一些公司也在为美国所关注。
 <br/>
 <br/>
 由于孟晚舟被捕后要求签署了〝禁止报导令〞，并将于周五出席保释听证会，所以，不仅是博尔顿，连加拿大司法部也拒绝对事件加以评论。
 <br/>
 <br/>
 孟晚舟被捕正值〝川习会〞同一天，美中双方达成暂时停火协议，但商定将于未来90天内决定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易.html" target="_blank">
  贸易
 </a>
 战的最终走向。
 <br/>
 <br/>
 博尔顿指出，最主要的是，要保护美国的就业机会，并且不让美国公司再像过去那样遭受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 政府所施加的不公平待遇。
 <br/>
 <br/>
 博尔顿指责，中共并不是靠公平竞争打败其它国家，而是在窃取它国的技术。
 <br/>
 <br/>
 美国副总统彭斯（Mike Pence）不久前在美国智库哈德逊研究所（Hudson Institute）演讲时指出，美国本来希望经济自由化能使中共加强与美国及世界的合作，但相反，中共却选择了经济侵略，并反过来不断壮大自己的军事力量。
 <br/>
 <br/>
 博尔顿表示，他认为国家安全所需要的并不是这些，而是无论中共经济如何增长，都要通过遵守规则来实现。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
</div>
