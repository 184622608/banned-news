### 川金二会开始选址  川普：3、4个待选地点
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）周二（9日）在白宫表示，他和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 的第二次峰会即将举行，白宫已在考虑3到4个会议地点。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （Donald Trump）周二在白宫椭圆形办公桌前告诉记者，国务卿蓬佩奥（Mike Pompeo）访问平壤时，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 举行了很棒的会面。目前，美朝双方官员已开始筹备第二次川金峰会。
 <br/>
 <br/>
 川普说：〝我们正在讨论3到4个地点，时间不会太久〞。他提到，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川金二会.html" target="_blank">
  川金二会
 </a>
 的地点可能是〝与新加坡不同的地方〞。
 <br/>
 <br/>
 川普强调，他与金正恩建立了良好的工作关系。在谈到金正恩时，川普说，〝我喜欢他，他也喜欢我〞，〝我们最终将在美国国土和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 国土举行更多的会议〞。
 <br/>
 <br/>
 具有历史性意义的首场〝川金会〞，于今年6月12日在新加坡举行。当时，在峰会结束后的记者会上，川普曾表示，双方达成了一份〝非常全面〞的联合声明，保证了双方对朝鲜半岛〝彻底无核化〞的坚定承诺。
 <br/>
 <center>
  <br/>

   <font size="-1">
    2018年6月12日，川普总统与金正恩签署联合声明。（SAUL LOEB/AFP/Getty Images）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 上周日（10月7日），蓬佩奥国务卿结束了第四次访朝。他在8日的记者会上表示，与朝鲜的无核化会谈取得重大进展，朝鲜领导人金正恩已经答应让国际核视察团，进入朝鲜进行核验证。
 <br/>
 <br/>
 此外，蓬佩奥还透露，美朝双方就〝川金二会〞达成协议。目前，两位领导人都认为可以往下一步前进了，〝尽管现在还没到那个阶段，但将来会的。〞
 <br/>
 <br/>
 金正恩上月在一封信函中，也曾提议同总统川普举行第二次峰会。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：东野）
