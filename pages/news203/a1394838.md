### 迈克尔登陆前升至4级  佛州百年来最强飓风
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】周三（10月10日），即将登陆美国佛罗里达的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 （Hurricane Michael）已增至
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_4级.html" target="_blank">
  4级
 </a>
 ，正以非常危险的态势向墨西哥沿岸推进。气象学家表示，迈克尔可能会成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 百年来所遭遇的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最强飓风.html" target="_blank">
  最强飓风
 </a>
 ，当地正面临生命和财产的威胁。
 <br/>
 <br/>
 周三早上，原来预测将达到3级的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 登陆前已迅速增至
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_4级.html" target="_blank">
  4级
 </a>
 ，目前正以最大持续风速145英里/小时的速度向前推进，阵风最高达172英里/小时，浪高31英尺，且在继续加强，预计将在周三中午左右登陆佛罗里达狭长地带（Florida Panhandle，也称西北佛罗里达或西佛罗里达）。
 <br/>
 <br/>
 根据美国国家飓风中心（National Hurricane Center）在美东时间周三上午10点的报告，迈克尔已抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 巴拿马市（Panama City）西南偏南约65英里处，且正以13英里/小时的速度向北推进。
 <br/>
 <br/>
 飓风中心指出，佛罗里达狭长地带沿岸的水位正在快速上升。
 <br/>
 <br/>
 福克斯新闻（Fox News）高级气象学家迪恩（Janice Dean）预测，迈克尔将于美东时间周三下午1点至2点在巴拿马市海滩（Panama City Beach）与阿巴拉契科拉（Apalachicola）之间登陆。
 <br/>
 <br/>
 迪恩周三表示，迈克尔非常危险，是灾难性且威胁生命的超强飓风。佛罗里达狭长地带从未遭遇过4级飓风，迈克尔很可能成为历史上在10月份袭击美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最强飓风.html" target="_blank">
  最强飓风
 </a>
 。
 <br/>
 <br/>
 科罗拉多州立大学气象学家克洛茨巴赫（Phil Klotzbach）也表示，自1851年以来，佛罗里达狭长地带曾9次遭遇大型飓风攻击，但没有一次达到4级。
 <br/>
 <br/>
