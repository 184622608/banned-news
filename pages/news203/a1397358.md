### 川普受福克斯采访：若谈不拢再征关税
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】周一晚，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 接受福克斯新闻采访时表示，将在贸易问题上和中国达成协议，如果无法达成协议，将对另外2670亿美元中国商品征收关税。
 <br/>
 <br/>
 美国总统川普：〝中国确实一直在经济上伤害美国。我们会赢得（贸易战）。会这样，我们会赢。〞
 <br/>
 <br/>
 川普表示，相信会达成协议，但现在中国还没有准备好。
 <br/>
 <br/>
 川普：〝我认为，我们会和中国达成很棒的协议。〞
 <br/>
 <br/>
 川普和习近平在下个月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上是否举行会谈，白宫之前表示还在计划中，但也可能有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 谈，将排除贸易议题，不过完全不谈也不可能。
 <br/>
 <br/>
 如果川习会谈失败，华盛顿准备在12月初宣布，对所有剩余的2670亿美元中国进口商品
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加征关税.html" target="_blank">
  加征关税
 </a>
 。
 <br/>
 <br/>
 美国总统川普：〝2,500亿美元关税。如果我们不能达成协议，接下来还会有2670亿美元关税。〞
 <br/>
 <br/>
 迄今，川普政府已对价值2500亿美元的中国商品加征关税，中国对价值1100亿的美国商品实施报复性关税。
 <br/>
 <br/>
 在采访中，川普指责媒体虚假报导，将连环炸弹客和他相提并论。
 <br/>
 <br/>
 美国总统川普：〝《华盛顿邮报》头条提到我，将我的名字和疯狂炸弹客联系起来，说川普炸弹客什么的。他们抓到他（炸弹客）时，我（的名字）出现在头条新闻。他们不说炸弹客抓到了，却在头条提到我。〞
 <br/>
 <br/>
 新唐人记者毕心慈、宇微综合报导
</div>
