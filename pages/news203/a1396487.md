### 美国富豪索罗斯住宅邮箱发现爆炸物
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】周一，美国亿万富翁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_索罗斯.html" target="_blank">
  索罗斯
 </a>
 纽约
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_住宅邮箱.html" target="_blank">
  住宅邮箱
 </a>
 中，发现含
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸物.html" target="_blank">
  爆炸物
 </a>
 的包裹。FBI介入调查后，表示对公共安全没有威胁。
 <br/>
 <br/>
 周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_索罗斯.html" target="_blank">
  索罗斯
 </a>
 位于纽约州凯托纳的家中，工作人员在邮箱中发现可疑包裹，里面装有疑似
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸物.html" target="_blank">
  爆炸物
 </a>
 装置，事发时索罗斯并不在家。
 <br/>
 <br/>
 拆弹小组技术人员到场后，引爆了可疑包裹，FBI 反恐调查小组随后介入调查。
 <br/>
 <br/>
 FBI在官方推特上表示，〝我们在对纽约州贝德福德一处住宅周围，进行调查。公共安全没有受到威胁，我们目前没有更多评论。〞
 <br/>
 <br/>
 索罗斯是美国民主党支持者，向民主党提供大量捐款。
 <br/>
 <br/>
 十月初，美国国会对川普提名的大法官卡瓦诺表决过程中，有愤怒的抗议者对议员吼叫。之后，他们被发现去找索罗斯要钱。
 <br/>
 <br/>
 索罗斯在他的祖国匈牙利也受到批评。
 <br/>
 <br/>
 新唐人记者毕心慈、明玉综合报导
