### 美军官阿富汗遇害 生前最后贴文谈选举
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】美国军官泰勒近日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 遇袭身亡，周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 这一天，他的遗体被运回美国。他生前在社交媒体上的最后一则贴文，就是呼吁选民出来投票。
 <br/>
 <br/>
 上周六，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 首都喀布尔巡逻时，美国国民兵军官泰勒突然被一名阿富汗士兵开枪打死，攻击者随即被其他阿富汗士兵击毙。据悉，袭击者是渗透到阿富汗军队的塔利班成员。
 <br/>
 <br/>
 泰勒于2013年当选为犹他州北奥格登市市长。作为国民兵军官，他于今年1月放下市长公职，前往阿富汗执行为期一年的部署任务。
 <br/>
 <br/>
 上个月，泰勒在脸书上发出生前最后一则贴文，是几张10月20日阿富汗选举的照片，他在贴文中写到：〝下周，美国人要为自己的选举投票了，我希望每位美国人都行使宝贵的投票权。无论共和党还是民主党赢，我们都记住，我们作为美国人，有更多的东西让我们团结起来，而不是分裂。〞
 <br/>
 <br/>
 周二，他的父母，妻子和两个最年长的儿子去多佛空军基地迎回了他的遗体。泰勒有7个未成年的子女，最小的不满一岁。
 <br/>
 <br/>
 泰勒遗孀 詹妮・泰勒：虽然布伦特・泰勒是在阿富汗的土地上阵亡，但他是为了美国和阿富汗两国的自由和民主的成功而死。
 <br/>
 <br/>
 犹他州长赫伯特称赞泰勒〝无私地服务于〞国家。
 <br/>
 <br/>
 犹他州州长 赫伯特：泰勒市长是在做这样的事情：代表他的国家，他的家庭和我们的州，去实践〝为同胞过上更好的生活而努力〞。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
