### 中共可能被逐WTO？川普重申对中共开铡关税
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统与中国国家主席习近平，下周将于阿根廷G20峰会会面。昨天川普再次重申，如果和中共达不成协议，还有更多在关税等着。而白宫经济顾问日前还说，中共加入WTO后就一直〝行为不检〞，他暗示可以研究将中共驱逐出WTO来解决问题。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 :〝美国在中国问题上处理得非常好，中方非常想和我们达成协议。他们虽然没说，但其实非常想。〞
 <br/>
 <br/>
 美国总统川普，20日出发前往佛州前，在白宫接受记者提问。他说中方非常想要达成贸易协议，但若无法达成，美国将对更多中国商品加征关税。
 <br/>
 <br/>
 美国总统 川普:〝如果我们没达成协议，我还有另外价值2500亿美元的关税。而且相信我，我会实施。〞
 <br/>
 <br/>
 此外，白宫经济顾问之一的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈塞特.html" target="_blank">
  哈塞特
 </a>
 （Kevin Hassett），日前接受BBC访问时表示，川普总统在贸易方面的强硬战略，正在发挥作用。中共自从加入WTO后就一直〝行为不检〞，WTO令美国失望。他还暗示，可以研究将中共驱逐出WTO来解决问题。
 <br/>
 <br/>
 针对美国的301条款调查报告，WTO 21日同意对美国的指控进行争端调查，以确定中共政策是否违反WTO条款，明年可作出裁定。
 <br/>
 <br/>
 新唐人亚太电视综合报导
