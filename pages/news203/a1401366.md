### 科恩认罪曾向国会撒谎 川普：他为换减刑
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的前私人律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科恩.html" target="_blank">
  科恩
 </a>
 周四承认，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查中，他曾向国会提供虚假信息，此事涉及川普在俄国的房地产生意。
 <br/>
 <br/>
 周四，在纽约曼哈顿法院开庭时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科恩.html" target="_blank">
  科恩
 </a>
 出人意料地承认，他2017年在参议院情报委员会作证时，就〝莫斯科
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 大楼计划〞一事作了虚假陈述。
 <br/>
 <br/>
 他当时说这个地产项目早在2016年1月就中止了，但实际直到2016年6月才中止。而2016年11月以前，川普都是在进行总统选战。
 <br/>
 <br/>
 科恩表示，这样做是为了和川普的政治辞令保持一致。
 <br/>
 <br/>
 周四，在启程前往阿根廷前，川普对此事做出回应。
 <br/>
 <br/>
 美国总统 川普：当我竞选总统时，这并不意味着我不被允许做生意。我赢得大选后，很明显，我停止做生意了。
 <br/>
 <br/>
 另外，莫斯科的川普大厦最终没有修建。川普说最重要原因是当时他集中精力在竞选上。
 <br/>
 <br/>
 美国总统 川普：这个项目我们没有做，我没有做。
 <br/>
 <br/>
 科恩此次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_认罪.html" target="_blank">
  认罪
 </a>
 ，表明他愿意和调查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 的特别检察官穆勒配合，以换取法庭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_减刑.html" target="_blank">
  减刑
 </a>
 。
 <br/>
 <br/>
 今年八月，科恩已就违反竞选财务法等罪名认罪。这些罪名将在两周后宣判。
 <br/>
 <br/>
 美国总统 川普：他是个软弱的人，他想要的就是减刑。所以他在这个项目上撒谎，这个项目是人人都知道的。
 <br/>
 <br/>
 川普发推文表示：(通俄门)调查浪费了4千多万美元之后，只证明了一件事情：没有和俄罗斯勾结。太荒谬了！
 <br/>
 <br/>
 周四早上，联邦特工突袭搜查了芝加哥市议员伯克的办公室。伯克是一位律师，曾为川普打理川普大厦地产税多年，今年早些时候双方中止了合作关系。但目前联邦特工行动的动机仍不明朗。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
