### 一块丑石头挡门30年 一检验竟价值10万美金
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】美国密西根州中部的艾德莫尔（Edmore）小镇有一名农场主，曾把一大块形状奇怪的石头用作
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_挡门石.html" target="_blank">
  挡门石
 </a>
 整整30年，直到今年年初一个偶然的机会促使他把这块石头送去中密西根大学（CMU）进行检验，结果发现这竟然是一块价值10万美元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陨石.html" target="_blank">
  陨石
 </a>
 ，CMU日前对外发布了相关消息。
 <br/>
 <br/>
 〝中密西根大学〞10月3日发布的新闻稿称，一名农场主今年初拿来一块
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_挡门石.html" target="_blank">
  挡门石
 </a>
 请求检验，经过该校质学教授西贝斯库（Mona Sirbescu） 的仔细检验，发现这是一块重约22.5磅的大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陨石.html" target="_blank">
  陨石
 </a>
 。这块陨石含有88%的铁和12%的镍，是密西根州发现的第6大陨石，预估价值10万美元。
 <br/>
 <br/>

    密西根州一名农场主曾把一块奇怪的石头用作挡门石30年，一个偶然的机会促使他把石头拿去检验，结果发现那是一块价值10万美元的陨石。（视频截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 <br/>
 据报导，这么农场主人1988年购买农场和房屋时，原屋主曾带他去看棚屋后方一块又大又丑的奇怪的石头。当时，原屋主告诉他， 1930年代某一晚曾经有陨石掉落在附近，隔日他与家人去找到了这块疑似陨石的石头，然后就把它当作收藏品保留了下来。
 <br/>
 <br/>
 农场主当时对这个故事并不以为意，他在买下房屋并继承这块陨石后，便将它当作挡门石搁置了30年。
 <br/>
 <br/>
 直到今年1月，农场主从新闻上读到有人在贩售小块的陨石，这才让他想起原屋主的话，于是农场主决定把这块〝挡门石〞拿去检验看看是不是陨石，结果让他十分惊喜。
 <br/>
 <br/>
