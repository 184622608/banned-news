### 纽约男自制90kg炸弹被捕 计划选举日引爆
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】美国联邦调查局（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 ）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 分部于周三（10日）通报，嫌犯保罗・罗森费尔德已被逮捕，他涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自制炸弹.html" target="_blank">
  自制炸弹
 </a>
 ，并计划于11月6日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 当天，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华盛顿国家广场.html" target="_blank">
  华盛顿国家广场
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_引爆.html" target="_blank">
  引爆
 </a>
 。
 <br/>
 <br/>
 现年56岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 男子罗森费尔德（Paul Rosenfeld）周三被控非法制造、运输及接收爆炸品等罪名。
 <br/>
 <br/>
 控罪书显示，罗森费尔德因不满现时的选举制度，而策划这起自杀式爆炸。此前，他在通讯中表明，有意制造一个爆炸装置，然后运到华盛顿，并在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 当天
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_引爆.html" target="_blank">
  引爆
 </a>
 装置并自杀。
 <br/>
 <br/>
 另有美媒披露，嫌犯保罗・罗森费尔德（Paul Rosenfeld）在网路上购买了大量火药，并重复制造小型炸弹进行引爆实验，甚至将犯罪计画透过信件及手机短讯，告诉了宾夕法尼亚州的一名居民。这名收信者随后向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 举报，整起事件才得以曝光。
 <br/>
 <br/>
 警方于本周二（9日）前往嫌犯位于曼哈顿郊区塔潘（Tappan）的家中进行搜索，发现一个超过90公斤的爆炸装置，里面装有用于枪械和大炮的黑色粉末。
 <br/>
 <br/>
 警方表示，他已经承认计划在11月6日中期选举当天，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华盛顿国家广场.html" target="_blank">
  华盛顿国家广场
 </a>
 引爆炸弹自杀，〝以宣扬他的政治理念〞。
 <br/>
 <br/>
 FBI纽约分部副部长威廉・斯威尼（William Sweeney）说：〝如果计划成功，将会夺走无数行人的性命，造成无可估计的破坏〞、〝所幸有市民及早报案，加之多个调查单位与FBI联手调查，才阻止了这起恐怖计划。〞
 <br/>
 <br/>
 FBI指出，罗森费尔德的〝激进政治主张〞是提倡一种〝抽签选举制（sortition）〞的政治制度，即政府官员应由随机抽签来决定，而非民选。纽约检察官贝尔曼（Geoffrey Berman）说，罗森费尔德计划引爆炸弹自杀，是为激起公众对其〝主张〞的关注。
 <br/>
 <br/>
 另据俄罗斯〝卫星通讯社与广播电台〞（简称卫星社，Sputnik news agency &amp; radio）周四（11日）报导，罗森费尔德被拘捕后向警方坦白，除了在中期选举当天引爆炸弹外，他还曾计划于今年8月和9月自焚。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：东野）
</div>
