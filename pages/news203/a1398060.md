### 川普移民政策获赞 佛州选民吁保护边境
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】中期选举在即，周六，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在佛州彭萨科拉为共和党助选，约有5000人参加了当天集会，一些人与台记者分享了他们的心声。来看报导。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 支持者 Randy Fox：一个国家不控防自己的边境，就不是一个国家。如果我们需要100,000个墨西哥人，去加州中央谷地采水果，那可以，我们应在有边境保护前提下，让这100,000人进来，因为是我们决定需要谁。但现在是谁都随便来。
 <br/>
 <br/>
 上周，五角大楼遵照川普命令，派遣多达7,000多名美军士兵前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 ，协助国土安全部阻挡非法越境。
 <br/>
 <br/>
 周日，已有不少美军已抵达边境，架设铁丝围栏等设施。与此同时，一部分来自中美洲的大篷车移民已抵达墨西哥城，并继续向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 边境行进。外界估计，目前大篷车移民约有4000人左右。
 <br/>
 <br/>
 川普支持者 William Beasley：川普说世界在笑话美国，因为我们边境法很弱，我们移民法也很弱，这真的太可笑了。听听川普怎么说的，因为他是对的。
 <br/>
 <br/>
 比斯利先生并很欣赏川普用推文直接与民众互动的做法。
 <br/>
 <br/>
 川普支持者 William Beasley：有关川普的推文，一些人觉得不舒服，一些人觉得奇怪。但是（你）读读他的推文，他是在传递自己的信息，他的看法，他有关美国方向的领袖观点，直接呈现给你，（首次有总统这么做），我们生在了不起的时代。
 <br/>
 <br/>
 川普支持者 Deborah Minda：他是个商人，他知道这个国家需要什么。
 <br/>
 <br/>
 川普支持者 Linda Norton：真的是这样，他已经让美国再次伟大了。
 <br/>
 <br/>
 新唐人记者王琼、雷跃佛州采访报导
</div>
