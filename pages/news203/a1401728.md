### 美加州致命野火 失踪人口从1200人降至25人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】美国加州约2周前发生最致命野火，导致死亡人数高达88人，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪人口.html" target="_blank">
  失踪人口
 </a>
 则从1200多人降至25人。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎普大火.html" target="_blank">
  坎普大火
 </a>
 从11月8日窜出火苗、延烧人口近2万7000的天堂镇以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪人口.html" target="_blank">
  失踪人口
 </a>
 数字即大幅波动。天堂镇位在旧金山北方280公里。
 <br/>
 <br/>
 加州标特郡（Butte County）警局1日晚间发布声明说，让北加州山间社区天堂镇（Paradise）几乎全毁的坎普大火（Camp Fire），死亡人数维持在88。
 <br/>
 <br/>
 而此前失踪人口从1200多人，到11月30日通报的失踪人口数字为49人，再下修至25人。
 <br/>
 <br/>
 有关单位解除天堂镇以北马加里亚（Magalia）地区部分社区的撤离令，但仍禁止进入天堂镇。
 <br/>
 <br/>
 目前大火起因仍在调查当中，不过有通报显示，火苗开始窜烧前后，太平洋瓦斯电力公司（PG&amp;E）他们位于起火点附近的设备有问题。
 <br/>
 <br/>
 (责任编辑：卢勇信)
