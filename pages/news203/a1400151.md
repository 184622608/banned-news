### 库德洛驳斥萧条论 乐观看美国经济前景
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 周二开盘大幅走低，有投资者悲观看待
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 走势，但被白宫经济顾问驳斥，白宫仍积极看待美国的经济前景。
 <br/>
 <br/>
 因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技股.html" target="_blank">
  科技股
 </a>
 的抛售潮，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 周一全线下跌，纳斯达克大跌3%，标普500指数下跌1.6%。
 <br/>
 <br/>
 美股周二开盘大幅走低，科技股继续下挫。分析人士指，市场对经济增长的疑虑，以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 的担忧，进一步加重压力。
 <br/>
 <br/>
 还有投资者认为，美国当前的经济形势，与上世纪30年代的大萧条时期非常相似，甚至断言美股处于熊市。
 <br/>
 <br/>
 白宫首席经济顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 ，严厉驳斥这一说法。
 <br/>
 <br/>
 白宫首席经济顾问库德洛：〝我认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 ，基础经济已复苏并将保持下去。我看到一些非常奇怪的报导，说经济大萧条即将到来，一派胡言。我个人以及当局的看法是，大萧条离我们远着呢，根本看不见。〞
 <br/>
 <br/>
 另外，针对美中贸易争端，库德洛说，川普总统对达成协议持乐观态度，但前提是必须符合美国的最佳利益。他将陪同川普总统，出席本月底在阿根廷召开的G20峰会。
 <br/>
 <br/>
 新唐人记者曹菁综合报导
</div>
