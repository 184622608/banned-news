### 川普休斯顿集会 民众提前24小时排队
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】周一晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统将在休斯顿举行集会，支持泰德・克鲁兹竞选连任联邦参议员。集会报名人数远远超过会场容量，从昨天早上10点，就有川普的支持者在会场外排队。请看报导。
 <br/>
 <br/>
 这次集会从可容纳七千人的NRG Arena（NRG竞技场），转到可容纳一万八千人的丰田中心（Toyota Center）。周日，报名人数已经超过10万。
 <br/>
 <br/>
 记者：距离周一晚上的集会还有24小时，但在Toyota Center外面已经排了100多人。他们带上了帐篷、食物。而在附近的酒店也有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的支持者提前入住，等待明天的集会。
 <br/>
 <br/>
 排在队首的女士，从早上十点就来到了这里。
 <br/>
 <br/>
 物流经理 Lisa Brewer：〝我来支持我的总统，这是我来的目的。来支持总统，我们都应如此。〞
 <br/>
 <br/>
 记者：这些支持者们不仅来自休斯顿，也有的来自奥斯汀、圣安东尼奥，以及路易斯安那等外州。
 <br/>
 <br/>
 路易斯安那退休机械工程师 John Sellers：〝我开了约400英里的车到这儿。我很坚决，一定要进入会场。〞
 <br/>
 <br/>
 民众称赞川普做了很多实事，给美国带来希望。
 <br/>
 <br/>
 奥斯汀IT咨询师 Cecil Oh：〝我以前对政府印象很糟，认为他们很腐败，有很多问题。在奥巴马之后，他（川普）让我再次为国家自豪。〞
 <br/>
 <br/>
 华裔 Nancy Rawis：〝少了很多税单，节约了很多钱。〞
 <br/>
 <br/>
 退伍军官 Tom Bills：〝他通过了退伍军人选择法案，我就医更便利了。〞
 <br/>
 <br/>
 民众认为，川普抵制社会主义的行动，利于美国的发展。
 <br/>
 <br/>
 墨西哥裔企业主 Martha Doss：〝我们应努力工作，从而去获得，我不赞同派发。资本主义最有利于我们的国家，尤其可以对比像委内瑞拉、古巴这些国家，社会主义是行不通的。〞
 <br/>
 <br/>
 新唐人记者语清、洪宇美国休斯顿采访报导
</div>
