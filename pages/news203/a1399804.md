### 习近平两张照片爆红  APEC峰会自备防弹专车
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】2018年亚太经济合作组织（APEC）领导人峰会，正在巴布亚新几内亚召开，17日晚各国元首穿上当地的红色传统服装出席晚宴时，美媒记者抓拍到了有趣的瞬间，令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的两张相片爆红。此外，外媒还发现习在当地乘坐的是国产防弹专车。
 <br/>
 <br/>
 11月17日，正在巴布亚新几内亚访问的美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）在APEC领导人峰会上，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 进行了激烈交锋。
 <br/>
 <br/>
 习近平谈了对贸易战和一带一路的看法。彭斯则反驳，在中共改变行为方式前，美国不会改变征收关税的政策，并且明确表示，中共的〝一带一路〞是给合作国家设立的债务陷阱。
 <br/>
 <br/>
 《美联社》报导，二人舌战之后，有人看见彭斯与习近平对谈1分半钟。
 <br/>
 <br/>
 报导说，17日晚宴上，各国元首身着巴布亚新几内亚的红色传统服装，准备拍摄大合照。当习近平到达时，彭斯正与加拿大总理杜鲁多（Justin Trudeau）聊天。
 <br/>
 <br/>
 习近平与其他国家领导人打招呼时，停下来与彭斯短暂交谈，聊了约1分半钟，过程两人都面带微笑，翻译也有陪同，不过两国官员都未透露两人谈了什么内容。
 <br/>
 <br/>
 当晚，在Twitter疯传两张照片。其中一张照片显示，各国元首穿上巴布亚新几内亚的红色传统服装出席晚宴，在准备拍摄大合照时，彭斯碰到习近平两人对视了一下，彭斯的眼神很特别。
 <br/>
 <br/>
 另一张是各国领导人站好位置之后，在等待拍照期间。习近平被拍到头微微上抬，用力紧闭双眼，好像很陶醉的样子。有媒体调侃习好像正在做〝中国梦〞！
 <br/>
 <br/>
 两张照片放在一起，看上去非常有趣。不少网民调侃习：〝你们都假装看不见我，我也看不见你们所有人〞。也有人称赞，照片拍得十分传神。
 <br/>
 <br/>
 据悉，习近平紧闭眼睛的照片，是美国《华盛顿邮报》记者Josh Rogin所摄。他留言说：〝看看我今晚在巴布亚新几内亚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 上拍到的习近平照片。〞
 <br/>
 <br/>
 另外，外媒还发现习近平在当地乘坐的是国产防弹专车。
 <br/>
 <br/>
 据《路透社》报导，在习近平抵达巴布亚新几内亚的当天（15日），习的两辆国产专车也由中方空运至当地。该车是由红QL5改装而成，除了加入防弹结构，还配备4公升引擎，外貌类似劳斯莱斯。
 <br/>
 <br/>
 按国际惯例，一个国家首脑到访，都是由对方国家负责包括安保在内的全部接待事宜，习自备国产防弹专车引发舆论关注。
 <br/>
 <br/>
 《苹果日报》17日称，巴布亚新几内亚治安不是很好，所以中方特意将防弹专车空运到当地。这是中方第二次空运专车到外国，供习在访问时使用。2014年他出访新西兰时，中方也是将两辆国产防弹车运到了奥克兰。
 <br/>
 <br/>
 有美媒报导说，北京对APEC峰会期间的维安极度重视，主要是担忧暗杀、绑架和恐怖袭击。此次习近平慎之又慎，在评估中南海核心智囊的建议后，习首次劝阻了原本一起随访的第一夫人彭丽媛。
 <br/>
 <br/>
 本次APEC峰会，丁薛祥、杨洁篪、王毅和何立峰等人随行。报导披露，此次中南海还特别大幅增补了负责习近平贴身安全的保镖数量，所有保镖均由习近平前任〝大内总管〞栗战书亲自把关。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
