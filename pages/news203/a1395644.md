### 美国法官干预行政助攻左派选战 司法部反击
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部.html" target="_blank">
  司法部
 </a>
 长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞申斯.html" target="_blank">
  塞申斯
 </a>
 日前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保守派.html" target="_blank">
  保守派
 </a>
 智库传统基金会发表演讲，论述〝司法侵略〞问题，他认为地方法官通过禁令等方式干涉行政部门运作有违宪法。塞申斯决定动用紧急上诉干预，在最短时间内让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院.html" target="_blank">
  最高法院
 </a>
 介入裁决。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部.html" target="_blank">
  司法部
 </a>
 长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞申斯.html" target="_blank">
  塞申斯
 </a>
 ：〝一种新的战术，司法激进主义，正在出现，我们称它为司法侵略。实际上，维权倡导者希望法官能够为他们达成他们无法通过选票实现的目标。从根本上说这不是民主。〞
 <br/>
 <br/>
 塞申斯提到，商务部长罗斯曾要求在2020人口普查中恢复询问公民身份的案例。多个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_左派.html" target="_blank">
  左派
 </a>
 州起诉川普政府此举不合法。纽约地方法院法官弗曼（Jesse M. Furman）裁定在诉讼中可以要求罗斯接受质询。
 <br/>
 <br/>
 塞申斯认为，这种做法有违美国既有法律对行政部门的运作进行保护的相关规定，同时也违背宪法保护的三权分立。
 <br/>
 <br/>
 司法部长 塞申斯：〝联邦和州政府的立法机构与行政部门，是宪法授权的分支机构来做这些事情。如果这些分支机构的做法让一个不负责任的法官感到不满，并不是意味着他们需要司法专业知识或建议来解决问题。〞
 <br/>
 <br/>
 佛州大西洋大学政治学教授Marshall L. DeRosa：〝普查问题之所以重要是因为它涉及国会议员的代表问题。如果一个区域人口众多是由于比如非法移民原因，如果把他们也计算在内，那么那个州就会有更多的国会议员代表，其他人口减少的州相应也会减少席位。〞
 <br/>
 <br/>
 他认为民主党通过这种方式来扩大支持，像DACA争论，旅行禁令的争议等，都是如此。
 <br/>
 <br/>
 塞申斯通过紧急动议上诉至
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院.html" target="_blank">
  最高法院
 </a>
 ，这样可以大大缩短中间过程。
 <br/>
 <br/>
 佛州大西洋大学政治学教授Marshall L. DeRosa：〝塞申斯之所以采用这种主动的方式，因为他想在2020大选前就此问题有一个结果。〞
 <br/>
 <br/>
 人口普查中的公民身份问题在问卷中早有历史，但2010年奥巴马政府决定不再询问这一问题。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
