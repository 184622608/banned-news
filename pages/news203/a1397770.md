### 普选临近 《纽约邮报》背书共和党州长候选人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】11月6日纽约州的选举在即，民主党籍候选人现任州长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库默.html" target="_blank">
  库默
 </a>
 ，与共和党籍州长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克・马利纳罗.html" target="_blank">
  马克・马利纳罗
 </a>
 的竞选将见分晓。《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约邮报.html" target="_blank">
  纽约邮报
 </a>
 》10月28日发表社论，为纽约州共和党籍州长候选人马利纳罗背书，还表达了他们对现任州长库默的失望。
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约邮报.html" target="_blank">
  纽约邮报
 </a>
 》（New York Post）发表社论，以《投票给改革者马利纳罗，而不要投给腐败份子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库默.html" target="_blank">
  库默
 </a>
 》（Vote for reformer Marc Molinaro, not corruptocrat Andrew Cuomo）为题，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约州长.html" target="_blank">
  纽约州长
 </a>
 共和党候选人马利纳罗（Marc Molinaro）大力背书，并表示，他有望反转纽约州腐败的趋势。
 <br/>
 <br/>
 《纽约邮报》曾在八年前为现任州长库默背书，但他们认为，库默没有做到当初的承诺，却渐渐走向迂腐和偏袒，库默任内，曾爆发多起身边助手的贪腐案。
 <br/>
 <br/>
 相反的，《纽约邮报》认为，共和党籍州长候选人马利纳罗可以带领纽约州走向新方向。
 <br/>
 马利纳罗提出要在五年内降低30%的资产税，并表示不会在没有任何监管的情况下，就把大笔的钱交给企业做建设项目。
 <br/>
 <br/>
 马利纳罗还提出重新设立反腐委员会（Moreland Commission），禁止与州府有合作关系的个人，向候选人提供竞选资金。
 <br/>
 <br/>
 《纽约邮报》赞赏马利纳罗有着远大的目标，作风却非常实际。
 <br/>
 <br/>
