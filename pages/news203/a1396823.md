### 五角大楼拟派800士兵增援边境 川普劝移民返乡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五角大楼.html" target="_blank">
  五角大楼
 </a>
 目前已决定，派遣一支800人的部队前往美墨边界，阻挡企图非法入境的数千〝大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 〞。周四（10月25日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 规劝这些移民回身返乡，寻求通过合法途径进入美国。
 <br/>
 <br/>
 周四中午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 发推劝说妄图非法入境的洪都拉斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 ：
 <br/>
 <br/>
 〝致那些在大篷车里的人：转身回去吧，我们不会让人非法进入美国的。回到你们的国家，如果你们愿意，可以像其他数百万人那样做，申请公民身份！〞
 <br/>
 <br/>
 <blockquote class="twitter-tweet" data-lang="en">

 <br/>
 <div style="clear:both;display:block;">

 <br/>
 就在川普发表这篇推文前几小时，美国官员刚刚确认，国防部长马蒂斯（Jim Mattis）将派800名美军士兵前往南部边境，帮助国土安全部和国民警卫队拦截抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 的〝大篷车移民〞。
 <br/>
 <br/>
 预计马蒂斯最早会在周四签署命令。目前尚不清楚他准备把那只部队部署在边境。
 <br/>
 <br/>
 美国联邦法律规定，禁止动用现役军人参与国内的执法行动，除非国会特别授权。
 <br/>
 <br/>
 不过，当川普得知仅上周末就有大约7000洪都拉斯移民越过墨西哥南部边境并继续向北挺进后，周一便在推特上表示，他已警告军方这是需要应对的〝国家紧急情况〞（National Emergy），暗示将派遣军队增援边境巡逻部门。
 <br/>
 <br/>
 如果不即刻采取有效措施，接下来的情况可能更为严重。据墨西哥主流日报〝El Universal〞周一报导，正向美国进发的非法移民已高达14,000人，其中多数已强行进入墨西哥境内，预计不久就将抵达美国南部边界。
 <br/>
 <br/>
 近一时期，已有2,100名国民警卫队员增援移民及海关执法部门，帮助处理非法移民大规模入境问题。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
