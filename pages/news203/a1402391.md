### 美国11月就业报告 失业率再创历史低点
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国劳工部.html" target="_blank">
  美国劳工部
 </a>
 最新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业报告.html" target="_blank">
  就业报告
 </a>
 显示，11月份美国雇主增加了155,000个工作岗位，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业率.html" target="_blank">
  失业率
 </a>
 连续三个月保持五十年来的低点。
 <br/>
 <br/>
 新唐人记者Jeremy Sandberg:〝美国最新的月度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业报告.html" target="_blank">
  就业报告
 </a>
 今天出炉，就业数据显示经济正在稳步增长。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国劳工部.html" target="_blank">
  美国劳工部
 </a>
 报告称，11月份美国雇主增加了155,000个工作岗位，低于预期的20万个工作岗位。
 <br/>
 <br/>
 虽然实际数字低于预期，但这可能进一步支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美联储.html" target="_blank">
  美联储
 </a>
 明年减少加息的次数。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业率.html" target="_blank">
  失业率
 </a>
 连续第三个月保持在3.7％这个五十年来的低点。
 <br/>
 <br/>
 每小时平均工资比去年增加了3.1％，与10月份的数字持平，是自2009年以来的最高水平。
 <br/>
 <br/>
 数据显示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 正在健康稳步的扩张，今年夏季和秋季的年增长率达到3.9％，是四年来最好的六个月增速。
 <br/>
 <br/>
 分析师预计，由于去年减税政策的影响逐渐减弱，2019年美国经济增长将放缓但仍会保持稳健。
 <br/>
 <br/>
 11月的增加的工作机会来自医疗保健公司，会计和工程等部门。制造业公司雇用了27,000名新员工，这是七个月来最多的，这表明中美贸易紧张局势并未削弱工厂的招聘人数。
 <br/>
 <br/>
 新唐人记者Jeremy Sandberg:〝持续的就业增长带动各行业的失业率降至历史最低水平。〞
 <br/>
 <br/>
 新唐人记者Jeremy Sandberg纽约报导
