### 迈克尔〝后遗症〞显现 德州面临洪水威胁
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】飓风
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 六天前离开美国，不过后续影响依然严重，直到现在一些地区仍面临着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪水.html" target="_blank">
  洪水
 </a>
 威胁。各州的救灾善后工作继续进行。
 <br/>
 <br/>
 无人机航拍视频显示，受灾最重的地区之一，在佛州墨西哥海滩市，最高时速155英里的飓风让这里大部分房屋消失得无影无踪。
 <br/>
 <br/>
 因为搜救工作尚未完成，一些居民到周三才能回家。15.8万人仍然没有电使用。
 <br/>
 <br/>
 联邦紧急事务管理局在佛州和乔治亚州建立42个救灾物品分发点。但在受灾最严重的地区，居民仍通过空投食物和水来解决生活问题。
 <br/>
 <br/>
 截至周二下午，佛州、乔治亚、南卡和北卡四州因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 遇难的已达20人。
 <br/>
 <br/>
 同时，迈克尔带去的大量
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降雨.html" target="_blank">
  降雨
 </a>
 使河水继续上涨。周二，美国国家气象局称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德州.html" target="_blank">
  德州
 </a>
 拉诺河水位达到40英尺，远超18英尺的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪水.html" target="_blank">
  洪水
 </a>
 警戒线，接近历史最高记录41英尺。奥斯汀西北65英里处的一座桥被冲毁。
 <br/>
 <br/>
 断桥附近地区已下达疏散令，数所学校被关闭，150多个地势低洼的路口被封。
 <br/>
 <br/>
 预计周三水位才会降回到洪水警戒线以下。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
