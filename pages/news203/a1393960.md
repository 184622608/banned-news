### 梅拉妮亚非洲行造访遗迹 缅怀奴隶遭遇
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】美国第一夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉妮亚.html" target="_blank">
  梅拉妮亚
 </a>
 首次访问非洲。周三，她在第一站加纳访问了当地的古迹海岸角城堡。
 <br/>
 <br/>
 美国第一夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉妮亚.html" target="_blank">
  梅拉妮亚
 </a>
 首先参加了海岸角的欢迎仪式，在得到当地首领的授权后，她随着向导穿越炮台，来到了17世纪建成的海岸角城堡，这是当初进行奴隶交易的地方，梅拉妮亚听取了有关城堡的介绍。
 <br/>
 <br/>
 美国第一夫人梅拉妮亚：〝讲述得很好，（故事）非常感人。〞
 <br/>
 <br/>
 梅拉妮亚还在城堡的留言簿上签字，并敬献花圈。
 <br/>
 <br/>
 美国第一夫人梅拉妮亚：〝这是一个非常特别的地方。我永远不会忘记这些难忘的经历和故事。真的，非常感人。〞
 <br/>
 <br/>
 此次非洲之旅，是梅拉妮亚以美国第一夫人身份首次外访。而她的丈夫，美国总统川普，自上任以来尚未对非洲进行访问。
 <br/>
 <br/>
 前一天，梅拉妮亚还慰问当地儿童，分发礼物。
 <br/>
 <br/>
 接下来，梅拉妮亚还会访问马拉维、肯尼亚和埃及，推广她发起的〝做到最好〞活动，促进更多人关注儿童成长。预计周五完成非洲之旅。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
