### 迈克尔飓风增为四级 预计周三登陆佛州
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】新闻开始首先来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国飓风.html" target="_blank">
  美国飓风
 </a>
 消息。周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔飓风.html" target="_blank">
  迈克尔飓风
 </a>
 增强到四级，威胁佛罗里达州和南卡罗来纳州，有可能成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 史上最大飓风。美国总统川普连发推文，提醒民众做好防备。
 <br/>
 <br/>
 国家飓风中心消息，飓风迈克尔周二晚间风速已高达每小时193公里，而且不断加强，预计周三午间登陆，可达四级飓风的威力。
 <br/>
 <br/>
 飓风中心专家表示，飓风将在登陆地区掀起最高4米的海浪。预计周三晚间扫过美国东南沿海，周四离开。
 <br/>
 <br/>
 周三早上，国家飓风中心在推特上每小时更新飓风消息，并提醒民众注意安全。
 <br/>
 <br/>
 电力公司已做好准备，确保飓风过后，能迅速恢复供电。
 <br/>
 <br/>
 海湾电力公司发言人 Gordon Paulus:〝我们大约有2000名外援，加上1100名雇员，以及系统内330名合同工，我们约有超过4000名工人帮助恢复电力。我们还会调遣更多人来。〞
 <br/>
 <br/>
 佛罗里达沿岸的居民忙着把门窗钉上木板，门口放上沙袋，给汽车加满油。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 巴拿马市居民 Don Walton: 〝该做防范的地方，我们都做了，我们已经准备好了。我们现在就是在回家休息之前，放松几个小时。我们会及时上路离开这里，我们知道严重性，已经准备好离开。〞
 <br/>
 <br/>
 也有居民因为房屋位置高，不打算撤离。
 <br/>
 <br/>
 巴拿马市居民 Stephanie Emanuel：〝我们家族在这里居住了有100年了，我见过比这更糟的。风暴潮从来没能淹到我们的房屋。〞
 <br/>
 <br/>
 美国总统川普从周三连续转发推文，提醒飓风警报区的居民，迅速撤离到安全地带。
 <br/>
 <br/>
 新唐人记者安心、明玉综合报导
</div>
