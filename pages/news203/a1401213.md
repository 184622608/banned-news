### 国家圣诞树点灯 川普总统夫妇出席仪式
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】观众朋友大家好，欢迎收看新唐人全球新闻。周五，美国总统川普及第一夫人在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 外主持了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家圣诞树.html" target="_blank">
  国家圣诞树
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_点灯仪式.html" target="_blank">
  点灯仪式
 </a>
 ，这是美国总统连续第96年延续这一传统。
 <br/>
 <br/>
 〝3 2 1 圣诞快乐 谢谢！〞
 <br/>
 <br/>
 周三晚，美国总统川普的夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼娅.html" target="_blank">
  梅拉尼娅
 </a>
 按下了圣诞树灯光的开关，第96次在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 外点亮了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家圣诞树.html" target="_blank">
  国家圣诞树
 </a>
 。
 <br/>
 <br/>
 这一白宫传统起源于1923年的美国柯立芝总统。
 <br/>
 <br/>
 今年，这棵处理在白宫后公园的巨大圣诞树，来自科罗拉多州。它的周围环绕着56株小树，代表着美国的州、领地与华盛顿特区。
 <br/>
 <br/>
 美国总统 川普：〝我们要一起来说，圣诞快乐！和你们在一起真的很好。〞
 <br/>
 <br/>
 由于左派思想影响，美国一些地区不再说圣诞快乐，而是用节日快乐取代。川普上任后，誓言恢复这一传统用。
 <br/>
 <br/>
 新唐人电视综合报导
