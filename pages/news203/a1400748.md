### 飞行六个月 NASA〝洞察号〞周一登陆火星
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】美国航空航天局NASA的无人探测器〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洞察号.html" target="_blank">
  洞察号
 </a>
 〞，经过长达6个月的飞行后，预计周一将在火星着陆，是继2012年探测车好奇号后登陆火星后，首个抵达火星的探测器。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洞察号.html" target="_blank">
  洞察号
 </a>
 〞发射后历时六个月的飞行，会以时速1.98万公里进入火星大气层，为避免过热，机身有隔热层保护，一旦探测器降到足够低的速度，会打开降落伞，之后速度减慢至每小时8公里。NASA称这个过程为〝恐怖6分半钟〞。在人类过去43次火星任务中，仅18次能成功，登陆火星的成功率仅40%，而全部成功的任务都是来自美国。
 <br/>
 <br/>
 〝洞察号〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_登录火星.html" target="_blank">
  登录火星
 </a>
 后，将伸出1.8米长的机械手臂，挖掘火星地表下方土壤，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_探测火星.html" target="_blank">
  探测火星
 </a>
 上的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地壳震动.html" target="_blank">
  地壳震动
 </a>
 及温度，以便更完整
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_了解火星.html" target="_blank">
  了解火星
 </a>
 。
 <br/>
 <br/>
 工程师巴克拉什：〝〝洞察号〞是首次研究火星内部深处的任务。我们过去发送的火星探测器和着陆器，是探索火星地质和表面的化学性质，那里有什么样的岩石。这次我们将探索深处的内部，并试图了解火星〝地震〞的活跃程度。〞
 <br/>
 <br/>
 〝洞察号〞的主要仪器包括一个由法国国家太空研究中心（CNES）制成的地震仪，〝洞察号〞机身上也有6个地震感测器，可以侦察到火星上最细微的震动；同时可以深挖地下3至5公尺，是首次有仪器能准确探测火星地表下温度。
 <br/>
 <br/>
 〝洞察号〞结合美、法、德科学机构心血，这项耗资达9.9亿美元的任务的目的是搜集有关火星的资料，为日后派人上火星做准备，NASA希望于2030年前后展开人类探索火星任务。
 <br/>
 <br/>
 新唐人记者严枫、田瑞综合报导
