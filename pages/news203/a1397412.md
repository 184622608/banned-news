### 川普助共和党选战 保边境成重要着力点
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】距离美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 还有7天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 还将亲赴11场助选活动。在周二的推文中，他并背书了十几位参选人，他们大多在移民问题上表现出色。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （2018.09.21）:〝2018年（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ）要出来投票，因为你是为我投票。〞
 <br/>
 <br/>
 中期选举普选日临近，川普的助选活动一个接一个；推特上今天至少背书了13人。川普提到的不少人，在边境与移民问题上表现出色。
 <br/>
 <br/>
 7月众议院决定2019年给国土安全部划拨50亿美元预算用于边境安全，包括建造200英里新围墙。
 <br/>
 <br/>
 刚过10月中旬，突然大量中美洲移民试图穿越墨西哥进入美国，至今至少两批共5-7千人压向美墨边境，被称为〝大篷车移民〞。
 <br/>
 <br/>
 美国总统 川普（2018.10.20）：〝民主党人要搭大篷车（移民）。他们喜欢大篷车（移民）。很多人说我也在想谁启动了大篷车（移民）？〞
 <br/>
 <br/>
 副总统彭斯日前从洪都拉斯总统处得知，大篷车移民受到了社会主义国家的资助。
 <br/>
 <br/>
 美国副总统 彭斯（2018.10.23）：〝他告诉我通过墨西哥边境、正朝着（美国）南部边境前进的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 ，是由左翼团体组织、受委内瑞拉资助的。〞
 <br/>
 <br/>
 川普已下令，本周将有5千200名军人赴美墨边境，阻止意欲非法入境的中美洲移民。
 <br/>
 <br/>
 美国总统 川普：〝你看大篷车上，有大量、大比例的男性，年轻的、强壮的男性。很多坏人、其中有很多坏人、帮派成员，我们不能让他们进来。〞
 <br/>
 <br/>
 川普表示不会有〝抓了就放〞的情况，实际上，军队不会允许那些人进入美国。
 <br/>
 <br/>
 专家认为，川普直接调用军队的做法与前任总统非常不同。
 <br/>
 <br/>
 哥伦比亚大学政治学教授夏皮罗（Robert Y. Shapiro）：〝这个做法非常不寻常，特别是派遣的军人数量更不寻常。他通过派遣军队传递出他对这个问题的担忧
 <br/>
 <br/>
 他认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 议题是2016年助川普胜选的政策之一，川普也希望这次中期选举对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 能同样奏效。
 <br/>
 <br/>
 今年中期选举的结果，特别是共和党能否控制国会众议院，将会对未来2年川普政策的推进产生重大影响，包括非法移民、健保、减税、经济等。
 <br/>
 <br/>
 周二，美国消费者信心指数达到2000年来的最高水平，川普在推文中说，〝股市自我主政后大幅上升，但现在有点停顿——人们想知道中期选举会发生什么。如果你想要自己的股票跌，我强烈建议你投票给民主党。他们喜欢委内瑞拉的金融模式，高税收和开放边境！〞
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
