### 竞争最激烈美国中期选举 周二投票开始
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】周二，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 日，被认为是对美国、以至国际社会的重要时刻，直追两年前的美国总统大选，更有西方媒体称之为〝一生中最重要的中期选举〞。
 <br/>
 <br/>
 中期选举周二
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 ，参议院将有35席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_议员改选.html" target="_blank">
  议员改选
 </a>
 ，众议院435名议员全数改选，另外还有36州进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_州长改选.html" target="_blank">
  州长改选
 </a>
 。
 <br/>
 <br/>
 选民关注的重点包括健保、移民、假新闻、枪枝管制以及经济等等，年轻人更关注工作机会和经济前景。
 <br/>
 <br/>
 外界认为，这次中期选举是对川普执政两年业绩的检验。川普上任近两年来，成功重振经济、大量创造就业、推行减税，广泛赢得选民支持。
 <br/>
 <br/>
 选民BRIAN ROTHMANN：〝我相信更多人会认真对待川普，因为他会兑现所有的政策，信守所有的承诺。奥巴马只会对经济表现与一些事抢功劳，事实上与他一点关系也没有。〞
 <br/>
 <br/>
 本次选举，众议院全部席位和参议院1/3席位将全部改选。共和党是否能继续主掌两院，对川普下一步施政和美国社会的走向，都极为关键。
 <br/>
 <br/>
 据统计，这将是半个世纪以来，选民参与度最高，竞争最激烈的中期选举。
 <br/>
 <br/>
 新唐人记者李梅、田瑞综合报导
</div>
