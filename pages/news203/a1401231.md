### 川普书面回复穆勒 两关键问题曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】美媒披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统本月早些时候书面回答了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查的两个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关键问题.html" target="_blank">
  关键问题
 </a>
 ，一是他事前并不知道其长子2016年6月在川普大厦会晤俄罗斯律师，二是前川普选战顾问斯通并没通知他维基解密计划公布有损于希拉里选战的信息。
 <br/>
 <br/>
 据美国有线电视新闻网（CNN）11月28日报导，是两名消息人士透露的上述信息。其中一名消息人士称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的回答没有任何引证，且川普明确表示，他正按照自己的最佳记忆回答问题。
 <br/>
 <br/>
 目前特别检察官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 （Robert Mueller）所领导的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查已进入收尾阶段，穆勒已开始准备最终报告。
 <br/>
 <br/>
 维基解密（WikiLeaks）公布有关希拉里（Hillary Clinton）丑闻的电邮和小川普（Donald Trump Jr.）会晤俄国人这两件事，一直被视作穆勒调查的两个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关键问题.html" target="_blank">
  关键问题
 </a>
 。穆勒企图以这两个问题为突破点，调查川普选战是否勾结俄罗斯干扰了2016年美国大选。
 <br/>
 <br/>
 在与穆勒团队进行了数月的紧张谈判后，川普于11月20日提交了自己的书面答覆。
 <br/>
 <br/>
 川普的律师们力争不让川普回答所谓可能妨碍司法公正的问题，该问题是由川普开除前联邦调查局（FBI）局长科米（James Comey）引起的。
 <br/>
 <br/>
 川普的法律团队也反对川普接受穆勒的面询，以免陷入穆勒所设置的伪证陷阱。
 <br/>
 <br/>
 CNN所说的两名消息人士也未向CNN提供任何直接的引证。
 <br/>
 <br/>
 2016年6月9日，川普总统的长子小川普和女婿库什纳（Jared Kushner）等人，在川普大厦会晤了俄罗斯女律师维塞尼茨卡雅（Natalia Veselnitskaya）。
 <br/>
 <br/>
 但值得一提的是，在这之前和之后，这名女律师都曾与福森公司的创建人之一辛普森（Glenn Simpson）会面，而正是福森公司雇用前英国间谍斯蒂尔（Christopher Steele）编篡了黑川普档案，并导致FBI启动了通俄门调查。
 <br/>
 <br/>
 川普和小川普都一再公开表示，在川普大厦的会晤发生前，川普并不知晓此事。
 <br/>
 <br/>
 关于维基解密事件，今年7月5日，美国一名地方法院的联邦法官驳回了民主党的诉讼。
 <br/>
 <br/>
 民主党起诉川普选战及其前顾问斯通（Roger Stone），指控他们勾结维基解密和俄罗斯政府，在2016年美国大选期间公布了被黑客袭击的民主党全国委员会（DNC）电邮。但该法官裁决，民主党的指控证据不足，从而将案件驳回。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
</div>
