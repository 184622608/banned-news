### 白宫发函通知 CNN记者将再次取消采访证
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】美国有线电视新闻网（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN.html" target="_blank">
  CNN
 </a>
 ）的记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿科斯达.html" target="_blank">
  阿科斯达
 </a>
 （Jim Acosta）被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 取消采访通行证后，在联邦法官的要求下，上周五白宫暂时恢复了阿科斯达的采访证。不过，事情并未就此落幕，周一，白宫正式发函通知阿科斯达，将在法院的临时令到期后，再次取消他的白宫采访证。
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN.html" target="_blank">
  CNN
 </a>
 报导称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿科斯达.html" target="_blank">
  阿科斯达
 </a>
 19日收到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 的来函通知，在法院临时令规定的14天期限到期后，也就是本月底，白宫将再次取消阿科斯达的采访证。
 <br/>
 <br/>
 白宫今天通知CNN记者阿科斯达（Jim Acosta），在法院临时令到期后，将再次取消他的白宫采访证。为此，CNN已向法院申请紧急聆讯。
 <br/>
 <br/>
 美东时间11月7日，CNN记者阿科斯达在美国期中选举后的第一次白宫记者会上，接二连三地向美国总统提问，期间更参杂着针对川普的尖锐质问与激烈辩论。
 <br/>
 <br/>
 川普在回答他的2个提问后示意换其他记者提问，但阿科斯达坚持要继续提问，并拒绝交出手中的话筒。当一名年轻的白宫女实习生试图从他手中拿走话筒时，阿科斯达抬起胳膊阻挡，不当碰撞到实习女生的手腕。
 <br/>
 <br/>
 当天晚间，白宫于对外宣布，阿科斯达当天在白宫的表现〝很不专业〞、〝不适当〞，尤其对女实习生的举动很粗鲁，因此决定取消阿科斯达的白宫采访证。
 <br/>
 <br/>
 CNN其后于12日向法院提告，控告白宫的上述决定违反了宪法第一修正案中关于保障言论与新闻自由的规定，要求白宫恢复CNN白宫首席记者阿科斯达(Jim Acosta)的白宫采访证。
 <br/>
 <br/>
 联邦法官凯利（Timothy Kelly）接手该案后，经过初步调查研判，于16日援引第5修正案有关〝政府决策应经由一定的法律正当程序〞的规定，要求白宫暂时恢复阿科斯达的采访证，有效期为14天。凯利解释，他同意白宫方面的主张，该宪法第一修正案的有关规定，并不适用于本案的情况。
 <br/>
 <br/>
 上周在法院的临时禁止令裁决出炉后，川普公开表态说，白宫将制定适当程序与采访规则。他当时表示，如果有必要，会继续走法律诉讼程序。
 <br/>
 <br/>
 接获法官裁决后，白宫发言人桑德斯发表了一个声明，宣布临时恢复阿科斯达的记者出入证。她同时强调，白宫将研拟新规范，以确保未来白宫记者会的秩序与公平性。
 <br/>
 <br/>
 桑德斯还特别说明，法官的裁决确认，第一修正案并没有赋予任何记者有进入白宫采访的绝对保障权。
 <br/>
 <br/>
 川普总统随后也在椭圆形办公室接受媒体提问时说，政府将会针对媒体记者进入白宫采访的问题写出明确的规范。他同时表示，〝政府将与CNN持续法律战，我们法庭见，而且会赢。〞
 <br/>
 <br/>
 稍后，川普在接受福克斯新闻主播华里斯(Chris Wallace)访问又再次强调，法官做出上述裁决，旨在要求白宫建立程序规范，但白宫的规则也是要维护的，如果有人不守规则，白宫还是会把他赶出去。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：程以仁）
</div>
