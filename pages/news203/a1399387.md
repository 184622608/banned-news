### 美参院拒绝立法保穆勒 司法部认定代理部长合法
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】周三（11月14日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 多数党领袖麦康奈尔（Mitch McConnell）拒绝了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_立法.html" target="_blank">
  立法
 </a>
 保护特别检察官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 （Robert Mueller）免遭开除的提议。司法部当天也发表备忘录，认定川普总统任命
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代理司法部长.html" target="_blank">
  代理司法部长
 </a>
 的程序
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_合法.html" target="_blank">
  合法
 </a>
 。
 <br/>
 <br/>
 周三，麦康奈尔驳回了亚利桑那州共和党参议员弗莱克（Jeff Flake）的动议。一向与川普不睦的弗莱克提议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_立法.html" target="_blank">
  立法
 </a>
 保护
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 免遭解雇，但该动议涉嫌违宪，引发很大争议并遭麦康奈尔拒绝。
 <br/>
 <br/>
 弗莱克与几名参议员共同提出了上述动议，藉口是川普要求司法部长塞申斯（Jeff Sessions）辞职引发了他们的担忧，他们认为这是川普准备开除穆勒的一个信号。
 <br/>
 <br/>
 穆勒正在领导所谓川普选战勾结俄罗斯干扰2016年美国大选的调查，塞申斯在中期选举结束的第二天辞职后，美国有线电视新闻网（CNN）等多家媒体报导，穆勒已开始撰写通俄门调查的最终报告。
 <br/>
 <br/>
 塞申斯辞职后，川普宣布由塞申斯的幕僚长惠特克（Matthew Whitaker）出任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代理司法部长.html" target="_blank">
  代理司法部长
 </a>
 ，司法部同时宣布，由惠特克接替副部长罗森斯坦（Rod Rosenstein）监督穆勒调查。
 <br/>
 <br/>
 由于惠特克此前曾公开批评穆勒调查，因此，他刚一上任就被推到风口浪尖上。民主党猛烈攻击他，并指控川普任命他的决定是违宪的。
 <br/>
 <br/>
 弗莱克提议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 迅速通过法案，以保证穆勒不被解雇。但是，麦康奈尔的回答非常干脆——〝我反对〞。
 <br/>
 <br/>
 麦康奈尔事后告诉记者们，他经常和川普谈到相关问题，而川普从没表示过他打算阻止穆勒调查。
 <br/>
 <br/>
 麦康奈尔承认，他知道川普对穆勒调查的感受，但川普并没说过因此要解雇穆勒。
 <br/>
 <br/>
 另外，司法部周三发表了一份备忘录，确认川普任命惠特克出任代理部长的决定完全
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_合法.html" target="_blank">
  合法
 </a>
 。
 <br/>
 <br/>
 备忘录指出，虽然宪法规定总统提名的主要官员须获得参议院批准，但惠特克只是临时性官员，因而不存在违宪问题。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
</div>
