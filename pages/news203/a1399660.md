### 法官下令暂时恢复CNN记者证 无关第一修正案
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】美国联邦地区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 临时恢复CNN驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 记者阿科斯塔的记者证。川普表示正在完善相关规定，白宫必须要有礼仪规范。
 <br/>
 <br/>
 周五，联邦地方法院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 蒂莫西．凯利（Timothy Kelly）下令，要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 立刻临时性恢复CNN驻白宫记者吉姆．阿科斯塔（Jim Acosta）的白宫记者证。
 <br/>
 <br/>
 CNN驻白宫记者 Jim Acosta：〝感谢法官今天所做的决定。回去工作了。〞
 <br/>
 <br/>
 阿科斯塔（Jim Acosta）9天后重新回到白宫。11月7日中期选举后的白宫新闻会上，他与总统川普发生激烈对话，其白宫记者证随后被撤销。
 <br/>
 <br/>
 阿科斯塔当天提问，似乎并不关注川普总统的回答，而是就移民大篷车是否构成入侵等问题与川普进行辩论；多个问题后川普让其坐下，但其仍坚持提问并用手阻挡白宫工作人员，此时下一位记者已经起身等待提问。
 <br/>
 <br/>
 美国总统 川普：〝礼仪。你不能问3-4个问题还站在那不坐下。礼仪、要遵守礼仪。我们需要充分的新闻自由，这非常重要，对我而言更加重要。但是你的行为方式要有对白宫的尊重。〞
 <br/>
 <br/>
 白宫发言人桑德斯在声明中说正在完善相关规定和程序以确保未来的新闻发布会公平而有序。在白宫行事必须要有礼仪规范。
 <br/>
 <br/>
 媒体研究中心主席博兹尔（Brent Bozell）在声明中指CNN记者阿科斯塔其实是名左翼活动者，很多场合他做的事都是在干扰总统、并不是像一名记者那样提出问题。CNN正如
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 担忧的，已经不是一个新闻机构，而变成了极左派的先锋队。
 <br/>
 <br/>
 白宫高级顾问 康威：〝据我所知，CNN没有任何损失，他们大部分时间仍在反川普。我在白宫看到了一些CNN的人。他们有通行证，他们在这儿。〞
 <br/>
 <br/>
 白宫撤销阿科斯塔的记者通行证，CNN13日起诉川普政府的做法违反宪法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第一修正案.html" target="_blank">
  第一修正案
 </a>
 保护言论和新闻自由。不过，联邦法官下令暂时恢复其记者证的依据，却非基于第一修正案的言论自由。而是宪法第五修正案：不经正当法律程序，不得被剥夺生命、自由或财产。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
