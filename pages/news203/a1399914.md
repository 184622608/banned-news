### 创纪录助学捐款 彭博捐18亿给大学母校
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】前纽约市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭博.html" target="_blank">
  彭博
 </a>
 18日表示，他将捐赠18亿美元给母校
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_约翰霍普金斯大学.html" target="_blank">
  约翰霍普金斯大学
 </a>
 ，据信这是美国大学院校史上最高额助学
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_捐款.html" target="_blank">
  捐款
 </a>
 。美国顶尖学校的高等教育费用，每年动辄超过5万美元，对大部分家庭来说难以负担。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭博.html" target="_blank">
  彭博
 </a>
 （Michael Bloomberg）投书〝纽约时报〞（New York Times）写道：〝我很幸运：我父亲是一名记帐员，每年赚的钱从未超过6000美元。但透过国防助学贷款，且在校园内工作，我才能读得起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_约翰霍普金斯大学.html" target="_blank">
  约翰霍普金斯大学
 </a>
 （Johns Hopkins University）。〞
 <br/>
 <br/>
 〝霍普金斯大学文凭帮我打开大门，让我实现美国梦，不然我根本就没有机会。〞
 <br/>
 <br/>
 彭博在大学毕业后的第一年就回馈母校，虽然仅是捐5美元给约翰霍普金斯大学。自此之后，他已陆续捐赠了15亿美元供研究、教学和助学。
 <br/>
 <br/>
 这笔新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_捐款.html" target="_blank">
  捐款
 </a>
 超过以往，彭博表示，他捐款帮助符合资格的中低收入学生，使他们更容易读得起大学。
 <br/>
 <br/>
 彭博写道：〝我想让给我机会的母校，能永久为他人敞开同样的机会之门。〞
 <br/>
 <br/>
 76岁彭博，1964年毕业于约翰霍普金斯大学，是全球金融服务和媒体公司〝彭博有限合伙企业〞（Bloomberg LP）的创办人，也是全球最有钱人之一。
 <br/>
 <br/>
 彭博在2002年至2013年担任纽约市长，且多年来一直考虑角逐白宫大位，包括下一届2020年总统大选。
 <br/>
 <br/>
 (责任编辑：卢勇信)
