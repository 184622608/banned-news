### 芝加哥摩天大楼电梯急坠84层 钻墙3小时救出6人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芝加哥.html" target="_blank">
  芝加哥
 </a>
 100层楼高的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摩天大楼.html" target="_blank">
  摩天大楼
 </a>
 16日中午，发生电梯高速坠落84层楼的意外，电梯里有6个人受困。所幸在消防员花了3小时后，才将受困乘客救出。
 <br/>
 <br/>
 据《ABC》报导，16日中午12时30分左右，包含1名孕妇在内的6人走出95楼的餐厅搭乘电梯，结果电梯一路坠落到11楼。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芝加哥.html" target="_blank">
  芝加哥
 </a>
 消防局获报后，连忙派员前往抢救，发现电梯竟有两条钢缆断裂。
 <br/>
 <br/>
 消防员透露，现场状况相当危险，钢缆断裂意味着他们无法从电梯上方救人。但11楼附近没有出口，必须在混凝土墙上打洞，再撬开电梯门，花了3个小时，才将受困乘客救出，所幸无人伤亡。
 <br/>

 <br/>
 <br/>
 发生事故的电梯今年7月才通过最近一次检查，目前事故电梯与相邻两个电梯已停止乘坐。当局对此已展开调查。
 <br/>
 <br/>
