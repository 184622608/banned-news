### 加州野火酿79死逾千人失踪 雨季将临或引土石流
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】美国加州北部遭野火肆虐的地区，处处是焚毁残骸及呛鼻烟雾。截止周日（18日），全加州已有至少79人丧命，近1300人失踪。而当地的雨季即将来临，这虽有助灭火但专家担忧可能会引发致命
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土石流.html" target="_blank">
  土石流
 </a>
 。
 <br/>
 <br/>
 北加州近日多处野火已经夺去至少76条人命，其中坎普野火（Camp Fire）持续延烧，重创山区小镇天堂镇（Paradise），昔日风景优美的天堂镇，眼下只剩一片焦土和废墟。当局虽已寻获数以百计火灾发生时仓皇四散的民众，但17日失踪人数仍飙升至1276人。
 <br/>
 <br/>
 而在南加洛杉矶郊外，伍尔西大火Woolsey Fire有82%的火势得到了控制。大火燃烧了9万8362英亩的土地，来自火灾的浓烟一直笼罩在该州的部分地区，这也被称为〝世界上最脏的空气〞。伍尔西大火在当地夺去至少3条性命。
 <br/>
 <br/>
 复原监识小组持续在烧焦残骸中采集检体，再以DNA鉴定确认死者身份。
 <br/>
 <br/>
 不过，好消息是，本周南北加州都有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降雨.html" target="_blank">
  降雨
 </a>
 ，这可能助灭火。
 <br/>
 <br/>
 国家气象局（National Weather Service）下属单位天气预报中心（Weather Prediction Center）表示，天堂镇等内华达山脉山麓小丘地区预计20日晚上开始降雨，雨势将持续至23日，雨量将达102毫米。
 <br/>
 <br/>
 坎普野火虽在旧金山以北约280公里延烧，但仍让旧金山的空气充斥不利健康的烟雾。
 <br/>
 <br/>
 〝旧金山纪事报〞（San Francisco Chronicle）及其他媒体报导，根据美国环境保护署（Environmental Protection Agency）的测量结果，当地空气品质不利健康，若干体育赛事取消，当局也建议老人和儿童留在屋内，避免外出。如今这座西岸大城也将降雨，对提高空气质量会有很大的帮助。
 <br/>
 <br/>
 路透社报导，国家气象局旗下单位、马里兰州大学园（College Park）气象预报中心的气象学家柏克（Patrick Burke）表示，〝雨势将持续至23日，计可累积76毫米的雨量，一些地区还可能出现102毫米或更多的雨量。〞
 <br/>
 <br/>
 他说下雨将有助清除烟雾，改善空气品质，也为消防员带来急需的援助。但由于持续多日的山火将植被烧毁，柏克表示，下雨也可能〝雪上加霜〞，在植被遭烧毁的山坡，附着力不足，难使土壤及碎石不移位，恐引发危险
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土石流.html" target="_blank">
  土石流
 </a>
 。
 <br/>
 <br/>
 柏克说，加州首府沙加缅度（Sacramento）北部等南加州一带，预计本周将降下最多50毫米的雨量。
 <br/>
 <br/>
 当局在南加州文图拉县、洛杉矶县等〝伍尔西大火〞燃烧区域发布土石流预警，提醒土石流可能形成新的灾难。
