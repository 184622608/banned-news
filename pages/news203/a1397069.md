### 宾州犹太教堂遇袭 11人死亡 川普谴责
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】周六上午，美国宾夕法尼亚州匹兹堡的一座犹太教堂，遭到枪手袭击，造成多人死亡，具体伤亡数字仍在统计之中。美国总统川普第一时间发声，谴责这一针对犹太人的仇恨袭击。
 <br/>
 <br/>
 事件发生在匹兹堡松鼠山的〝生命之树〞犹太教堂附近。该地区是美国最大的犹太社区之一。
 <br/>
 <br/>
 一名白人枪手向人群射击。有执法官员称，枪手曾高喊〝所有犹太人都必须去死〞。
 <br/>
 <br/>
 匹兹堡警方公布，11人死亡，6人受伤。
 <br/>
 <br/>
 匹兹堡市公共安全部门负责人希斯里奇：〝在最初警方和嫌犯交火中，两名警察受伤，另外的交火中，有两名警察受伤，他们是特警。〞〝有11人死亡，没有儿童。〞
 <br/>
 <br/>
 警方发言人  Chris Togneri：〝目前枪击嫌犯已经被拘押，教堂内有多位受害者，〞
 <br/>
 <br/>
 匹兹堡市公共安全部门负责人希斯里奇宣布，枪手是46岁男性白人罗伯特・鲍尔斯，目前已被送往医院。
 <br/>
 <br/>
 美国总统川普第一时间发推文，对事件表示关注，并说〝事件比想象的更严重〞。
 <br/>
 <br/>
 川普很快又发表视频，严词谴责这一仇恨犯罪。
 <br/>
 <br/>
 美国总统  川普：〝肮脏的、恶毒的反犹太仇恨必须受到谴责和抵制，不论出现在哪里。美国不能容忍任何反犹太主义、以及针对任何宗教、种族的仇恨或偏见。〞
 <br/>
 <br/>
 事发地点附近的卡耐基梅隆大学校区封闭，警方通告附近居民留在室内。
 <br/>
 <br/>
 新唐人记者林宇、尚敬综合报导
