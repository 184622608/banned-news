### 川普连发4推文谈川习会 3次提及习近平
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】美东时间12月3日清早，美国总统川普接连发了四条推文，谈及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 上达成的协议，其中三次提及北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 。
 <br/>
 <br/>
 早上4时54分，川普发出当天第一篇推文称，〝我与中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在阿根廷举行了一次非凡的会晤。与中国的关系向前迈了一大步！将会发生非常棒的事情。我们正在尽最大努力达成协议，不过当协议达成时，中国同样能从中收获颇多。让竞争公平！〞
 <br/>
 <br/>
 7分钟之后，他又发文称，〝农民将是我们与中国达成协议的最大受益者，也将最快从中受益。他们（中国）计划立即开始购买农产品。我们生产世界上最佳、最清洁的产品，这正是中国想要的。农民，我爱你们！〞
 <br/>
 <br/>
 5时18分，他又发表一篇推文说，〝习主席与我有着非常强大的个人关系。他和我是我们两个伟大国家之间唯一能够就贸易及更多问题实现重大积极改变的两个人。一个解决朝鲜问题的方案对中国和所有国家来说都是好事！〞
 <br/>
 <br/>
 5时30分，他又表示，〝我相信，在未来的某个时候，习主席和我，以及俄罗斯总统普京将开始谈判，讨论真正阻止已然重大且失控的军备竞赛。美国今年花费了716亿美元军费。这很疯狂！〞
 <br/>
 <br/>
 此前一天，川普还曾有一篇推文谈及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 ，透露北京已承诺削减和取消针对美国汽车的进口关税。
 <br/>
 <br/>
 北京这一承诺，没有写入白宫12月1日晚关于川习会的声明当中。
 <br/>
 <br/>
 12月1日晚宴上，中美首脑达成了暂缓贸易战升级的临时协议。根据白宫声明，北京承诺从美国购买大量农业、能源、工业和其它产品，管控此前大举输送美国的毒品，从新考虑高通收购案等；作为交换，美方同意将2019年1月1日提升中国产品关税税率的计划搁置90天。
 <br/>
 <br/>
 不过在这90天内，中方必须与美方开始就强制技术转让、知识产权保护、非关税壁垒、网络入侵和网络盗窃等问题展开谈判。如果届时不能达成协议，美方将重启关税升级计划。
 <br/>
 <br/>
 评论人士指，中方同意购买美方农产品和能源，等于取消了前两轮对美国的贸易报复，而美方前两轮贸易措施原封未动，还保留了继续加征关税的主动权。
 <br/>
 <br/>
 在白宫声明和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普推文.html" target="_blank">
  川普推文
 </a>
 中，毫不吝啬对北京的〝溢美之词〞，再次体现了今年美中贸易谈判中〝中方要面子，美方要里子〞的一贯传统。
 <br/>
 <br/>
 据香港《南华早报》之前的消息，川习会后没有记者会，也没有安排联合声明，其目的就是方便美中双方事后各自对川习会做出自己的阐述。
 <br/>
 <br/>
 川习会后，美方第一时间宣布川习会取得〝成功〞，并列举了北京的一系列让步。
 <br/>
 <br/>
 而中方随后也开始宣传北京的〝胜利〞，声称阻止了美方1月1日升级关税战的计划，并称双方还将继续寻求〝取消已经加征的关税〞。
 <br/>
 <br/>
 不过，中共官方回避了北京做出了哪些让步，也没有提及关键的90天限期，大陆网络上与此相关的评论也遭到封杀。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
