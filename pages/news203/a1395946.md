### 200年来首次 〝穆桂英〞挂帅美陆军司令部
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】美国最大的司令部——陆军司令部（FORSCOM）近日宣布，副司令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_劳拉・理查森.html" target="_blank">
  劳拉・理查森
 </a>
 中将将获晋升陆军司令部代理指挥官。她将成为200多年来陆军史上首位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女司令.html" target="_blank">
  女司令
 </a>
 ，统领77.6万名军人及9.6万名人员。
 <br/>
 <br/>
 WTVD报导，陆军参谋长马克・米利（Mark A. Milley）将军当地时间周二（16日）下午主持了指挥交接仪式，宣布由理查森（Laura J. Richardson）做代理指挥官。
 <br/>
 <br/>
 米利说，〝你将在相当长的时间内掌控这个命令。它将以数月来衡量，而不是几天或几周。我们知道你会做得很好，因为我们知道FORSCOM的每个人都会象
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_劳拉・理查森.html" target="_blank">
  劳拉・理查森
 </a>
 和艾布拉姆斯一样做得很好。〞
 <br/>
 <br/>
 理查森16岁时就获得飞行员驾照，后来毕业于州立丹佛都会大学，获得心理学学士学位。
 <br/>
 <br/>
 她1986年入伍，担任陆军航空兵飞行员。理查森曾说：〝很多人都说，若你能晋升至上将级别，就好比你能穿过针孔。〞但是理查森中将却连连突破女性的职业天花板，连破纪录。
 <br/>
 <br/>
 在32年的行伍生涯中，理查森驾驶过大名鼎鼎的〝黑鹰〞直升机，2002至2004年赴伊拉克，担任一支部队的指挥官，并于2003年登上美国《时代》杂志的封面，标题是《当妈妈上了战场》，讲述这名女飞行员和丈夫都在战场，家中只留下14岁的女儿。
 <br/>
 <br/>
 2011年她晋升为准将，2012年成为第一骑兵师（也被称为美国第一队）的首位女性副总指挥官。2013到2014年，她担任阿富汗国际安全援助部队的通信副主管。2017年，理查森被任命为位于北卡罗来纳州布拉格堡的FORSCOM的首位女性副司令，成为仅次于罗伯特・艾布拉姆斯上将的二号人物。现在，随着艾布拉姆斯调任驻韩国美军司令，她接掌了帅印。
 <br/>
 <br/>
 （记者李红报导）
