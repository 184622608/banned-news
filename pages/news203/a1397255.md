### 枪杀犹太人 嫌犯受审 川普将往当地慰问
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】上周六，宾州发生了美国史上，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹太人.html" target="_blank">
  犹太人
 </a>
 最致命的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪杀案.html" target="_blank">
  枪杀案
 </a>
 。周一，该案
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌犯.html" target="_blank">
  嫌犯
 </a>
 出庭受审。
 <br/>
 <br/>
 被指控上周六在宾州匹兹堡一间犹太教堂制造枪杀
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹太人.html" target="_blank">
  犹太人
 </a>
 案的凶嫌鲍尔斯，周一出庭受审。
 <br/>
 <br/>
 宾州检察官指称，鲍尔斯不得保释，联邦地方法官还将于11月1日举行第二次听证。
 <br/>
 <br/>
 宾州法院检察官 Scott Brady：〝届时我们将有机会提供证据证明，Robert Bowers枪杀了11名在教堂从事宗教活动的人。他并枪伤了其他6人，包括4名回应枪击的警察〞。
 <br/>
 <br/>
 周一白宫也为此召开记者会。
 <br/>
 <br/>
 白宫发言人 桑德斯：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统的几名孙辈就是犹太裔，他的女儿（伊万卡）是信仰犹太教的美国人，女婿是大屠杀受害者的后裔。明天（周二），总统与第一夫人将前往宾州，表达对当地美国人的支持，并分担匹兹堡社区的哀伤。〞
 <br/>
 <br/>
 同仇恨谋杀完全不同的境界是，匹兹堡阿勒格尼综合医院的犹太医生和护士们放下个人感受，还将医治这个声称〝要杀死所有的犹太人〞的凶手。
 <br/>
 <br/>
 许多人在教堂外表达了对无辜
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇害者.html" target="_blank">
  遇害者
 </a>
 的爱和哀悼。
 <br/>
 <br/>
 犹太教堂教友RABBI JEFFREY MYERS：〝这是我的忧虑：这有很多的仇恨。但最终，爱会战胜仇恨。我们不会让仇恨占上风。〞
 <br/>
 <br/>
 同一天，另一件仇恨犯罪的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌犯.html" target="_blank">
  嫌犯
 </a>
 ，前段时间向左派人物或媒体接连邮寄14枚炸弹邮包的萨约克（Cesar Sayoc），被下令不得保释。而CNN在亚特兰大的总部又收到了类似的炸弹邮包。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
