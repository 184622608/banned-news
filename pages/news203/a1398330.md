### 2018中期选举开跑 川普坐镇白宫观战
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】观众朋友大家好，欢迎收看新唐人全球新闻。周二是美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 日，国会大多数席位都要改选，全美各地不少选民早早就赶去投票，而出席了一连串助选集会的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统，周二选择在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 观战。
 <br/>
 <br/>
 经过数月的激烈造势活动，2018年美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 正式登场。
 <br/>
 <br/>
 周二清晨，各州投票站陆续开放，迎接选民。
 <br/>
 <br/>
 在弗吉尼亚州的一处投票站，民众早已排起长队，等待入场。
 <br/>
 <br/>
 佛州迈阿密的选民直到下午，还在投票站外排队等待投票。
 <br/>
 <br/>
 佛州州长候选人竞争激烈。共和党前联邦众议员德桑蒂斯和民主党佛州首府塔拉哈西市市长吉隆姆都亲赴投票站，为自己投票，并受到支持者们的欢迎。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周二当天没有公开行程。不过他也没有閒着，在推特上再次转推了此前背书过的共和党候选人。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 表示，总统当晚将和家人及朋友一起，在白宫观看选举结果。
 <br/>
 <br/>
 不过，本次选举也不是一帆风顺，除了东部一些区域〝天气〞来捣乱，有的地方投票机器也出现故障。
 <br/>
 <br/>
 乔治亚州斯涅乐市一处投票站的机器故障，导致选民长时间等待。选民安达丽娅・伍兹表示，她看到有20多人没投票就离开了。选民路易斯称自己等待投票时间达到5个小时，她甚至怀疑这并非意外。
 <br/>
 <br/>
 纽约市则有几个投票点的读票器损坏，导致投票拖延。其中，曼哈顿上西城投票点六个读票器中，有两个发生故障，让等待人群大排长龙。
 <br/>
 <br/>
 过去一个多月来，川普总统马不停蹄参加各地竞选集会，为共和党候选人造势。
 <br/>
 <br/>
 投票前一天，川普还进行了〝马拉松式〞闪电战，赶赴三州参加了三场集会。在密苏里州的最后一场集会，他呼吁全体美国人团结。
 <br/>
 <br/>
 美国总统 川普：〝我们是一个民族，一个家庭，一个光荣的国家，在上帝的引领之下。〞
 <br/>
 <br/>
 新唐人电视综合报导
