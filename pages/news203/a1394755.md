### 国际货币基金发布报告：中共损失超美国至少一倍
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际货币基金.html" target="_blank">
  国际货币基金
 </a>
 （IMF）9日公布最新报告称，一场全面的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 将削弱全球经济增长，但不会令全球经济停止增长。2018年美国经济增长超乎寻常地强劲之后，明年开始放缓，而中国的损失将比美国严重一倍。
 <br/>
 <br/>
 中美贸易持续升级外加付债升高，IMF10月9日将2018年，2019年全球经济成长率双双下调0.2个百分点，对2018年，2019年和2020年全球经济增长率的基准预测均为3.7%。
 <br/>
 <br/>
 IMF公布的世界经济展望（WEO）最新报告中，是将美国总统川普（特朗普）政府已经实施的贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，以及来自中国和其他国家的报复的影响，纳入一起计算的。
 <br/>
 <br/>
 IMF警告，如果
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 进一步升级，中美两国2019年的经济增长都会受到不利影响，其中，中国蒙受的损失将更大。同时，全球经济也将受到冲击。
 <br/>
 <br/>
 IMF公布的图表中，下调了美中2019年的经济增长预估。图表显示，2019年，美国的经济成长将从2.5%降到1.6%，中国将从6.2%降到4.6%。这将是中国工业化以来的最低增长率。
 <br/>
 <br/>
 目前，美国已经对大约一半的中国对美出口产品出台了加征关税措施。川普总统表示，还要对全部中国商品都加征关税；根据IMF的估算，在这种局面下，美国2019年的GDP将会因此损失0.9%，而中国的经济增长则将损失1.6%。
 <br/>
 <br/>
 IMF同时也指出，测算使用的经济学模型并不十分精确，与最终的实际结果可能有偏差。
 <br/>
 <br/>
 在全球层面上，关税壁垒会对市场信心造成不利影响，股市将出现波动。根据IMF的测算模型，如果中美双方都对对方的全部产品加征关税，全球经济增长将在2020年损失0.8个百分点，而长期的损失则大约为0.4个百分点。
 <br/>
 <br/>
 而目前已经出台的关税措施，则会让中美两国2019年的经济增长各损失0.2个百分点。
 <br/>
 <br/>
 预计在当前的条件下，2018年全年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 增长率将会在6.6%左右，略低于2017年的6.9%。而到了2019年，增长率将会进一步放缓到6.2%。除了外部需求降温，中国国内加强金融监管也会对经济增长造成不利影响。
 <br/>
 <br/>
 从中期来看，随着中国向可持续增长模式的转型，中国的经济增长率将会逐步下落到5.6%。
 <br/>
 <br/>
 长期来看对美国经济的影响将接近一个百分点，而对中国来说，这则是美国的一倍。
 <br/>
 <br/>
 在最糟糕的情况下，日本的2019年增长率将从0.9%减慢到0.4%。欧元区的增长率将从1.9%降至1.5%。尽管欧元区受贸易战的影响较小，但是经济增长的放缓将会加重一些欧洲国家的失业问题，特别是那些因举债过高而经济脆弱的国家。
 <br/>
 <br/>
 但IMF的报告也称，2018年美国经济增长超乎寻常地强劲，预计达到2.9%，而不是原来估计的2.4%。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际货币基金.html" target="_blank">
  国际货币基金
 </a>
 首席经济师莫里斯-奥布斯费尔德（Maurice Obsfeld）说，眼下美国经济的强劲是因为川普政府的各种减税政策的刺激。美国2017年年底通过的税务改革，除了减少所得税之外，最重要的是将企业税从35%降到21%。这个措施导致美国经济增长超常。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
