### 川普赴加州慰问 行前答记者问
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】周六，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 总统赴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州慰问.html" target="_blank">
  加州慰问
 </a>
 野火受灾民众，同时与加州州长布朗及当选州长纽森商讨森林管理等问题。行前，他就一些主要议题回答记者提问。
 <br/>
 <br/>
 川普表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州野火.html" target="_blank">
  加州野火
 </a>
 灾情严重，失踪人数比想像的要多。
 <br/>
 <br/>
 美国总统 川普：〝我要和消防员和一线人员待在一起。〞
 <br/>
 <br/>
 川普说，他将分别前往两处野火灾区，与加州州长以及当地官员一起处理问题。
 <br/>
 <br/>
 美国总统 川普：〝我们有很多事要谈，会谈到森林管理的问题——这个问题我已经说了很久。多年前早就应该做了。〞
 <br/>
 <br/>
 川普之前批评加州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环保政策.html" target="_blank">
  环保政策
 </a>
 导致森林缺乏管理，成为酿成火灾的主要原因之一。他在推文中说〝每年都有数十亿美元的（联邦）资金注入，却还是失去那么多生命，原因是森林严重管理不善。〞
 <br/>
 <br/>
 前一天，川普发布《重大灾害和紧急声明》和3项《火灾管理援助拨款》，为加州支付75%用于派遣消防资源、清理火场、安置灾民及控制灾情影响等方面的开支。
 <br/>
 <br/>
 谈到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民问题.html" target="_blank">
  非法移民问题
 </a>
 ，川普强调要改变〝抓了就放〞的执法模式；保护合法移民的权力。
 <br/>
 <br/>
 美国总统 川普：〝有数百万人在排队等着合法进入我们的国家，那些人有优先权，他们必须优先（被考虑）。 我们在南部边境有伟大的军事力量。我们不允许人们非法进入我们国家，而且我们不会抓到了他们就放掉。））〞
 <br/>
 <br/>
 对于《纽约时报》报导，说他与副总统彭斯貌合神离，川普再次批评纽时编造故事，也再次说假新闻是人民的敌人。
 <br/>
 <br/>
 美国总统 川普：〝他有百分百的忠诚。那是虚假故事。我怀疑纽时根本没有消息来源。典型的纽时编造故事。我对麦克彭斯没有半点怀疑。〞
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
