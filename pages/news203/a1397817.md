### 应对难民潮 川普：拟下周颁行政令
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 继续逼近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 ，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 周四发表讲话，详尽阐述了美国当前面临的边境危机。川普表示，下周可能发布行政令，全面应对这一危机。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 继续向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 前进。尽管川普多次警告，并已经派遣军队前往南部边境，阻止难民非法入境。但大篷车北上的步伐没有停止。
 <br/>
 <br/>
 对此，川普周四发表演説，总结了美国当前面临的边境危机。
 <br/>
 <br/>
 美国总统  川普：〝吸引非法移民入境的最大漏洞，就是利用欺诈和无依据的庇护申请，以能够进入我们伟大的国家。 我们看到庇护申请令人瞠目结舌地增加了1700%，自从2010年以来。〞
 <br/>
 <br/>
 白宫同一天发布声明，2013年以来，全家非法入境遭到拘留的人数，从2013年的14,855人，猛增到2018年的107,212人，增幅高达620%。
 <br/>
 <br/>
 而来自中美洲的难民申请，符合条件获得批准的只有15%。
 <br/>
 <br/>
 川普在演説中，批评大篷车难民拒绝留在墨西哥，已经证明他们来到美国并不是为了得到庇护。
 <br/>
 <br/>
 美国总统 川普：〝墨西哥已经同意接收他们，并鼓励他们留下来。但他们不想留下，他们想来美国。所以这已经和安全无关了，而庇护是为了保障安全。〞
 <br/>
 <br/>
 川普还表示，在墨西哥边境缴获海洛因的比例，已经从2003年的10%上升到90%。与边境安全问题相伴而来的是毒品危机。
 <br/>
 <br/>
 白宫声明说，2017财年缴获的芬太尼数量高达2370磅，足以杀死所有美国公民。
 <br/>
 <br/>
 川普在回答记者提问时表示，下周将发布行政令，全面应对当前的大篷车难民危机。
 <br/>
 <br/>
 川普再次警告，难民如果硬闯边境，像在墨西哥那样投掷石块，他已经告诉军队，将视为对美军开火。
 <br/>
 <br/>
 川普已经将向美墨边境派出最多15,000人的军队，阻止大篷车难民非法入境。部分部队和装备已经抵达德克萨斯州的圣安东尼奥。
 <br/>
 <br/>
 新唐人记者林宇、宇微综合报导
</div>
