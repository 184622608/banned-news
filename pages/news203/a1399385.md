### 加州野火肆虐50人丧生 首府发N-95型口罩挡浓烟
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】连日来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州野火.html" target="_blank">
  加州野火
 </a>
 肆虐，受灾遇难人数刷新历史纪录，目前南北加州报共有50人丧生。火势持续燃烧，加上大风助长使灾区空气质量极差，加州首府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙加缅度.html" target="_blank">
  沙加缅度
 </a>
 市政府近日免费提供可过滤悬浮粒子的立体口罩给民众，防护等级与消防员相同。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州山火.html" target="_blank">
  加州山火
 </a>
 在大风助长下来势十分凶猛，火势蔓延得极其迅速。干燥强烈的圣塔安那季候风（季节演变风）从内陆吹袭，使烟雾不断扩散。
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙加缅度.html" target="_blank">
  沙加缅度
 </a>
 邮报》（The Sacramento Bee）报导，大火燃烧后，沙加缅度（Sacramento）街区空气质量指数一度达到367。根据环境保护局的说法，这对所有人而言都是危险的，无论年龄或身体健康状况如何。烟雾迫使沙加缅度地区的几所学校关闭。
 <br/>
 <br/>
 沙加缅度市政府为市民提供7,000个特别为野火烟雾设计的立体口罩，民众可于早上8时至晚上8时前往市内所有消防局索取。唯一例外的是16号站，目前已关闭。当局称，如果消防局没有人手应付居民索取口罩，可能正在紧急服务，请民众耐心等候。当局也向无家者分发口罩，并正在努力争取更多口罩。
 <br/>
 <br/>
 该市提供的口罩是经美国环保署批准的N-95型口罩，旨在保护肺部免受野火烟雾中的小颗粒污染。
 <br/>
 <br/>
 美国疾病控制和预防中心报告说，在五金店常见的口罩是用于捕获大颗粒，如锯末等，并不会在烟雾弥漫的天气中保护肺部。不过，沙加缅度公共卫生局建议，不要长时间使用N95口罩。

 截至周二（11月13日），北加州〝坎普野火〞（Camp Fire）的遇难人数增加到48人，成为加州史上最致命的火灾。坎普野火烧毁的天堂镇（Paradise），距离沙加缅度约130公里。坎普野火燃烧范围达到13万亩，目前有35%火势得到控制，逾5,100名消防员投入抢救。
 <br/>
 <br/>
 在天堂镇，法医队伍正在废墟中寻找遇难者遗骸。目前仍有228人失踪，预计确认的死者人数还会上升。同时，400名美国国民警卫队也加入了救急队伍。
 <br/>
 <br/>
 而南加州的〝伍尔西野火〞（Woolsey Fire）火势仍然不减，目前已致2人死亡、3名消防员受伤。伍尔斯野火延烧面积超过9万7千英亩，40%火势得到控制。近3,600名人员参与灭火。伍尔斯野火摧毁了有百年历史的派拉蒙牧场（Paramount Ranch），圣莫尼卡山脉国家保护区有83%面积已毁。
 <br/>
 <br/>
 目前，南北加州的野火已造成近7,000幢建筑被毁。据研究机构Enki Research的估计，野火给加州政府、保险公司及业主带来了至少190亿美元的损失。
 <br/>
 <br/>
 川普（特朗普）总统已宣布加州山火为重大灾害，将全力提供联邦资源救助。他周二（13日）在白宫表示，〝我们为失去的生命而哀悼，为（野火的）受害者祈祷， 受害者比任何人想像的还要多。我们将尽一切力量支持和保护处于险境的同胞。〞
 <br/>
 <br/>
 加州大火令两大公共电力公司太平洋瓦电（PG&amp;E）和爱迪生国际（Edison International）再次成为焦点。外界质疑，大风吹断电线再一次引发大火。去年10月的北加州酒乡大火事件中，太平洋瓦电被索赔170亿美元。
