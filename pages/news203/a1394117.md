### 彭斯演说全面揭中共 透析美中真实关系
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】观众朋友大家好，欢迎收看新唐人全球新闻。周四，美国副总统彭斯在哈德逊研究所发表讲话，全面剖析了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 真相，以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对外扩张、对内镇压的实质。被认为是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 对中国的行动方案。
 <br/>
 <br/>
 彭斯副总统本次以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 〞为主题的演讲，全面论述了近70年来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对美国的影响及蚕食。
 <br/>
 <br/>
 美国副总统 彭斯：〝北京动用整个政府机构，通过政治、经济和军事及宣传手段推动其影响力，并在美国牟利。中国（中共）也比以往更积极地运用这种手段来干涉美国国内的政策和政治、施加影响。〞
 <br/>
 <br/>
 彭斯在演讲中，用1949年将美中关系划线，他认为，1949以前美中互惠，但1949以后发生消极变化。
 <br/>
 <br/>
 美国副总统 彭斯：〝但在1949年掌权后不久，中国共产党就开始追求专制的扩张主义。〞
 <br/>
 <br/>
 彭斯并批评中共执政，说一套，做一套。
 <br/>
 <br/>
 美国副总统 彭斯：〝自由的梦想对中国人来说仍然遥不可及。虽然北京仍然口头上说“改革开放”，但邓小平的著名政策现在却是有名无实。到2020年，中国统治者的目标是实施奥威尔式的极权体系，控制人类生活的方方面面，即所谓的‘社会信用系统’。〞
 <br/>
 <br/>
 彭斯指出中共提出的〝中国制造2025〞及〝金钱债务外交〞都是其对外扩张、意欲控制地区及全球的具体行动方案。回顾近几十年美中关系，特别是美中贸易，中共利用一系列不公平手段，包括征收关税、操控汇率、强迫转让技术、盗窃知识产权等，实现GDP增长奇迹并成为世界第二大经济体。他提醒想要进入中国市场的企业要三思而后行，并批评谷歌屈从中共的做法。
 <br/>
 <br/>
 美国副总统 彭斯：〝谷歌应该立即停止开发‘蜻蜓’应用程序，这会加强共产党的审查并损害中国客户的隐私。〞
 <br/>
 <br/>
 美国副总统 彭斯：〝为了回应川普总统所采取的强硬立场，北京正进行总动员来破坏对总统的支持，破坏我们的议程，和我们国家最珍视的理念。〞
 <br/>
 <br/>
 彭斯直言，中共想要颠覆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 。
 <br/>
 <br/>
 美国副总统 彭斯：〝这位总统不会退缩；美国人民不会动摇。〞
 <br/>
 <br/>
 针对副总统彭斯的讲话，哈德逊研究所中国战略中心主任白邦瑞（Michael Pillsbury）在回复本台时表示，〝这是美国政府对待中国（中共政权）的行动方案。川普政府不会继续容忍中共的例行做法：窃取我们的尖端技术，从世界最大的贸易顺差中获益，同时干预我们的政治，比俄罗斯有过之而无不及。〞
 <br/>
 <br/>
 他认为，重塑美中关系、共同迎接未来是美国所希望的，但需要中国真正的改变。
 <br/>
 <br/>
 美国副总统 彭斯：〝让我们以决心和信念追求和平与繁荣的未来。相信成事在天——蒙神的恩典，美国和中国将共同迎接未来。〞
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
