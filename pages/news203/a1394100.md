### 纽约豪掷130亿美元翻修 JFK机场将焕然一新
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】纽约州长库默周四公布了一项基建工程计划，将花费巨资，全面翻新纽约肯尼迪国际机场。
 <br/>
 <br/>
 纽约州长库默（Andrew Cuomo）宣布启动一项130亿美元的项目，计划将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约肯尼迪机场.html" target="_blank">
  纽约肯尼迪机场
 </a>
 焕然一新，增加运客量。其中120亿美元将来自私人资金，10亿出自纽新港航局。
 <br/>
 <br/>
 纽约州长库默：〝你飞到世界各处旅行，看看别的国家的建筑，再回到美国，你就明白了我们落在后面了。〞
 <br/>
 <br/>
 新计划最大的亮点是在机场南、北两边，修建两座新的国际航站楼，增加400万平方英尺的服务面积，并集中地面交通，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻修.html" target="_blank">
  翻修
 </a>
 跑道。
 <br/>
 <br/>
 目前，肯尼迪机场每年服务6000万旅客，库默表示，完工后在现基础上每年可多服务至少1500万旅客。
 <br/>
 <br/>
 工程将在2020年开始，第一批新的登机门会在2023年开放，全部工程将在2025年完工。
 <br/>
 <br/>
 州长库默曾在2015年启动40亿美元的项目，升级拉瓜迪亚机场，其第一期工程将在2019年完工。
 <br/>
 <br/>
 新唐人记者韩瑞纽约报导
</div>
