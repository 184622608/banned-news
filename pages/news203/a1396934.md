### 中情局前特工：邮包炸弹无引爆系统 旨在引发恐慌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】美国寄送炸弹邮包的嫌犯周五已在佛州迈阿密地区落网，但嫌犯作案的动机尚不明确。一名中情局前特工向媒体披露，已经被查获的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邮包炸弹.html" target="_blank">
  邮包炸弹
 </a>
 均无引爆系统，显示嫌犯的目的是想引发社会恐慌。炸弹邮包事件发生后，美国左派媒体迅速把矛头指向总统川普，而右派媒体则怀疑这是民主党因为害怕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 失败而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自导自演.html" target="_blank">
  自导自演
 </a>
 的一出苦情戏。
 <br/>
 <br/>
 从本月24日起的三天内，美国民主党一众政要和商家大佬接二连三地接到疑似装有爆炸物的邮包，联探员立即对此案展开了追查，很快就发现，这些炸弹邮包的源头均来自佛罗里达，随后又追踪到了迈阿密邮件中心，并成功在26日一早将寄送这些邮包的嫌犯抓获。
 <br/>
 <br/>
 在此过程中，虽然陆陆续续有民众党的知名人士先后接到可疑的邮包，并在一定程度上引发了社会的恐慌情绪，美国联邦调查局最初形容这些可疑的邮包时，称其为〝具破坏力的装置〞。但外界很快就发现，所有的炸弹邮包都没有发生爆炸。这个显得有些蹊跷的现象引起了外界的好奇。
 <br/>
 <br/>
 日前，美国中情局（CIA）的一名前特工接受路透社采访时透露，所有被截获的可以邮包经技术人员检测后发现，那些〝炸弹〞都没有引爆系统。也就是说，这些炸弹理论上是无法引爆的。
 <br/>
 <br/>
 这名前特工说：〝他们想做到的是引发恐慌、制造混乱，但无意伤人。因为若然他们有意伤人，炸弹理应会爆炸。〞
 <br/>
 <center>
  <br/>

   </font>
  </a>
  <br/>
 </center>
 <br/>
 随即，也有美国官方专家对外透露，那些炸弹邮包里的装置虽然有烟火粉、计时器及电池，但安装的方式并不想让人打开时就爆炸。
 <br/>
 <br/>
 有媒体也提到，可疑包裹里的管状型炸弹大多没有引爆装置，虽然设计相当粗糙，但专家认为，歹徒并不是〝省油的灯〞。
 <br/>
 <br/>
 炸弹专家Samenow在接受媒体采访时说：〝歹徒非常不简单，他们精心策划过，歹徒清楚决定要把炸弹寄给谁。〞
 <br/>
 <br/>
 对于这名〝炸弹客〞寄送这么多恐吓包裹的动机，外界则出现了各式各样的分析与揣测，甚至在左、右两派媒体之间引发了一场口水战。
 <br/>
 <br/>
 以CNN为首的左派媒体几乎在事件刚一发生，就直接把矛头指向了现任美国总统川普，抨击他平日里有〝过激言论〞激发了极端份子的这种暴力恐吓行动。但支持川普和共和党的福士电视台主持人道布斯则坚定地认为，寄送这些不会爆炸的炸弹，明显是有人试图影响
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的选情。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
