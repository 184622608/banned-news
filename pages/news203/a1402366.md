### 孟晚舟铤而走险酿祸？内部会议称可接受违规风险
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 之女、CFO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 日前在加拿大被捕，罪名是违反美国对伊朗的制裁令。而港媒披露华为近日一次会议记录，印证了美方指控。在会上，孟晚舟称可接受〝暂时违规〞的风险，任正非则毫不隐讳称遵守美国管制令〝没有意义〞。
 <br/>
 <br/>
 据香港《南华早报》12月7日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 内部员工近日透露了华为10月29日的一份会议记录。
 <br/>
 <br/>
 根据记录，当天，担任华为首席财务官的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 ，和父亲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 一起参加了华为内部员工问答环节，说明了公司在国外营运时对于合规的立场。
 <br/>
 <br/>
 孟晚舟表示，外国规范有〝红线〞、〝黄线〞之分，前者是〝必须严格遵守〞的；而对于后者，公司很难〝严格遵守〞，例如外国的劳动法等，公司可以将〝藐视黄线规范〞视为〝既定成本〞。
 <br/>
 <br/>
 孟晚舟接着说，在红黄线之外，还有一种情况是，外部规则明确且没有争议，但在实际营运上仍〝完全无法配合〞。这时，公司经过〝合理决策〞，〝可以接受暂时违规的风险〞。
 <br/>
 <br/>
 任正非则在会议上强调，对于一些〝敏感国家〞，也要留意美国和欧盟的相关规范。但他没有指明哪些是〝敏感国家〞。
 <br/>
 <br/>
 但他表示，公司不能因为美中贸易战就〝束缚自己的手脚〞，否则就无法继续生产，那么合规〝又有什么意义〞？
 <br/>
 <br/>
 报导指，华为拒绝回应这份会议纪录。
 <br/>
 <br/>
 华为被指具有中共军方背景，多年来帮助中共在海外从事网络间谍活动，并向朝鲜、伊朗、古巴等受美国制裁的国家秘密提供技术支持。
 <br/>
 <br/>
 根据美媒报导，美国商务部2016年已就与伊朗的违规交易对华为和中兴展开调查。中兴当年就遭到美国制裁。
 <br/>
 <br/>
 今年4月，美国商务部对中兴下达禁售令的同时，美国司法部也对华为展开刑事调查。
 <br/>
 <br/>
 路透社12月6日报导，对华为的调查，包含了其通过滙丰银行同伊朗进行非法交易。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
