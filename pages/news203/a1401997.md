### G-20峰会间隙 川普批中国脏空气飘到美国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）上周五(11月30日)在阿根廷出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G-20峰会.html" target="_blank">
  G-20峰会
 </a>
 的间隙上接受美媒采访时，谈到气候问题时称，美国在环境方面做得很好……而中国和其他国家的脏空气飘到了美国，让美国花了很多钱。
 <br/>
 <br/>
 美国之音12月4日报导称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 上周五在阿根廷首都布宜诺斯艾利斯举办的20国集团（G20）峰会隙上，接受了美国之音特约撰稿人格莱塔・范・萨斯特伦的专访，谈到即将举行的美中峰会以及美中贸易等多项话题。
 <br/>
 <br/>
 期间，萨斯特伦提到：〝全球经济的问题之一是气候变化。此间也在进行讨论。您在气候变化及其对全球经济影响方面的立场是什么？〞
 <br/>
 <br/>
 川普回答说：〝很简单，我希望有地球上最洁净的空气和最洁净的水。我希望有水晶般洁净的水，而这是我们所拥有的。我们在环境方面做得很好，这也是我希望的。但是我不会为了试图维持某些可能根本无关紧要的标准而让国家破产。〞
 <br/>
 <br/>
 川普说，〝您看看中国，看看其它国家，他们的空气肮脏，他们的坏空气飘到了美国。人们不想谈论这点，但是它飘了过来。所以我们让空气清洁，但他们没有，而这花费很多的钱。事实是，我们是绝对洁净的，但我们不会花费几万亿美元去做对别人有好处可对我们却没好处的事情。您知道，我有个非常简单的政策，它叫做‘美国优先’。同时，我们会是世界了不起的邻居。但我们必须公平对待自己。就应该是这样的。〞
 <br/>
 <br/>
 萨斯特伦又问：〝您会向习主席提到气候变化吗？〞
 <br/>
 <br/>
 川普说：〝我们会谈这个问题的，我们会谈很多问题。我们谈论的大话题是贸易，这是大家想听的。要知道，他必须在他的气候问题上拿出一些行动。他的气候问题有点难。但我肯定他会的。我认为他是…他是我的朋友，是好人。但是你也知道，我们有那么一点纠纷。还是那句话，多少年来我们国家一直在被占便宜，这种情况不会继续下去了。〞

 在气候大会召开之际，美国国务卿蓬佩奥周二在参加北约峰会前，在布鲁塞尔向外交官发表讲话时提到，美国退出巴黎气候协定是因为协定缺乏公平的条款。
 <br/>
 <br/>
 蓬佩奥说：〝我们退出巴黎协定，因为其中的条款对美国不公平。现有的条款是从美国那里抽钱，补贴中国这样污染严重的国家。〞
 <br/>
 <br/>
 G20期间，川普与中国国家主席习近平进行了会谈，白宫贸易顾问纳瓦罗（Peter Navarro）接受美媒采访时，透露了G20峰会后〝川习会〞的一些细节。纳瓦罗称，美方在〝川习会〞上没有做出任何让步，只是给了中共90天时间，让他们去做过去20年来早就该做的事情而已。
 <br/>
 <br/>
 川习会后，美中将立即展开新一轮贸易谈判。川普选择对中共立场强硬的莱特希泽监督谈判，这预示了美国的坚定的谈判立场。
 <br/>
 <br/>
 川普12月4日在推特上宣布，〝与中方的谈判已经开始〞。谈判将自12月1日〝川习会〞起的90天内结束，除非期限延长。
