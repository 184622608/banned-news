### 川普谈全球气候变暖：远不及科学家所说的严重
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统日前在接受媒体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_采访.html" target="_blank">
  采访
 </a>
 时表示，虽然他不反对人类行为会影响气候的观点，但是他认为这种影响力远不像一些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科学家.html" target="_blank">
  科学家
 </a>
 所说的那么严重，而他更关注如何确保空气清洁。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 是在上周接受美国Axios新闻网站专访时做出上述表示的。该 访谈视频周日晚在HBO电视网播出。
 <br/>
 <br/>
 川普告诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_采访.html" target="_blank">
  采访
 </a>
 他的记者斯万（Jonathan Swan），他认为人类活动当然会对全球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_气候变暖.html" target="_blank">
  气候变暖
 </a>
 造成一定影响，但他指的是，只是某些污染物会增加、某些事情会发生而已。
 <br/>
 <br/>
 斯万原本认为，他带来一份最近公布的《美国全国气候评估》（U.S. National Climate Assessment, NCA）副本，这可能会令川普更难质疑其中所发现的问题。不过，他的意图显然没有实现。
 <br/>
 <br/>
 斯万表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科学家.html" target="_blank">
  科学家
 </a>
 们说，过去50年来，使全球变暖的压倒性原因是人类的活动。川普则回答说，〝我不一定同意（这种结论）〞。
 <br/>
 <br/>
 2017年发布的最新《全国气候评估》表示，对于近年来的全球气候变暖因素，除了人类所排放的温室气体外，〝没有令人信服的替代解释〞。
 <br/>
 <br/>
 该报告 是由美国联邦机构和一些非政府组织的气候科学家共同编写的。
 <br/>
 <br/>
 去年，川普的批评者担心川普会审查《全国气候评估》甚至阻止其发表，但是，川普却按时公布了该报告，结果反而令批评者们感到难堪。
 <br/>
 <br/>
 在接受Axios采访期间，川普对这一问题显然并不热心。他表示，他同意人类活动会影响气候的说法，但是，其影响程度可能远远低于《全国气候评估》和其他许多科学家所说的那样。
 <br/>
 <br/>
 川普还指出，即使在气候科学界，对于全球变暖的看法也并不完全统一。尽管可能有多数科学家支持《全国气候评估》的发现和联合国的评估，但也有一些科学家对此提出异议。
 <br/>
 <br/>
 川普表示，他希望将所有观点都纳入政府的气候报告中。
 <br/>
 <br/>
 川普指出，确实有许多科学家对人类行为导致全球变暖的观点提出异议，而他想做的只是，〝确保我们拥有最清洁的空气〞。
 <br/>
 <br/>
 2017年6月1日，川普宣布美国退出联合国气候协议《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎协定.html" target="_blank">
  巴黎协定
 </a>
 》（Paris Agreement），兑现了他的竞选承诺。
 <br/>
 <br/>
 《巴黎协定》是联合国气候峰会于2015年12月通过的一项决议。按照该协议，各缔约国同意自愿减少温室气体的排放量，以对抗全球气候变化。
 <br/>
 <br/>
 2016年4月22日，共有171个成员国在联合国总部签署了上述协定。
 <br/>
 <br/>
 2016年美国大选期间，川普曾批评所谓气体排放影响气候变化的说法是个〝骗局〞、损害了美国经济，并誓言如果能当选总统，他将退出《巴黎协定》。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
