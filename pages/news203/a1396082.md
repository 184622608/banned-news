### 美中期选举 经济议题仍是选战核心
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 临近。由于国内经济
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_增长强劲.html" target="_blank">
  增长强劲
 </a>
 ，选民基本没有感受到贸易战带来的负面影响。而在各州选战中，经济议题依旧是核心话题。
 <br/>
 <br/>
 今年第二季，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 创下近四年来最快增速，国内生产总值(GDP)环比增长4.1%，是2014年第三季以来的最佳表现。这也让共和党在中期选举中也获得更大动能。
 <br/>
 <br/>
 在临近首都华盛顿的马里兰州，已执政4年的共和党州长豪根(Larry Hogan)正在争取连任，经济牌也是他的主打。
 <br/>
 <br/>
 美国马里兰州州长Larry Hogan：〝我们连续四年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_减税.html" target="_blank">
  减税
 </a>
 ，减少规章限制，让马州变得更适合企业经营，所以我们不仅没有减少、反而增加了十万人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业.html" target="_blank">
  就业
 </a>
 ，工资水准上升9%，我们的增长率高于全美绝大多数的州。我也很骄傲，马州在支持少数族裔和妇女企业方面，排名全美第一。〞
 <br/>
 <br/>
 与豪根竞争州长的民主党人吉勒斯，曾任全国有色人种促进协会主席和首席执行官。教育和就业是他竞选中的最关切议题。
 <br/>
 <br/>
 美国马里兰州州长候选人Ben Jealous：〝我有两个孩子在公立学校，并一直经营小企业。在豪根任州长期间，我们在全美公学排名中，从第一降到第六。我们的地区在就业增长和收入增长上停滞不前。如果有和维州同样的增长率，马州的就业人数会比现在增多五万。〞
 <br/>
 <br/>
 吉勒斯批评州长豪根，在吸引高科技和初创企业投资方面，缺乏有效策略，输给了比邻的维吉尼亚州。他也提出改善方案。
 <br/>
 <br/>
 马里兰州州长候选人Ben Jealous：〝我们会把更多的政府津贴用于TEDCO，这样就可投资于马州各地的初创公司。我们还要做一件事，就是把‘技术转化办公室’从商务厅提出来，直接置于州长办公室之下。〞
 <br/>
 <br/>
 根据考夫曼指数报告，马州新创公司增长率高出维州10％，但维州的创业增长，总体依然排名全美第一，马州居第三。不过，州长豪根强烈捍卫马州在信息产业上的成就。
 <br/>
 <br/>
 马里兰州州长Larry Hogan：〝我们在这方面彻底击败维州，马州有12000个信息公司，我们是美国的网络之都，我们的信息公司和就业者数量仅次于硅谷，我们马上可以在信息产业中，再雇用八万至十万人，只要能找到受过适当训练者的话。〞
 <br/>
 <br/>
 目前，美国国内经济
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_增长强劲.html" target="_blank">
  增长强劲
 </a>
 。如何让本州民众享受到经济成长的最大利益，也是各州候选人最为着力的议题。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
 <br/>
</div>
