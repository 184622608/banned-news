### 民主党欲调查代理司法部长 司法部驳回
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】尽管美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 就川普总统任命
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代理司法部长.html" target="_blank">
  代理司法部长
 </a>
 一事强硬施压，但司法部周二（11月20日）表示，司法部内部运转良好，并再次重申该项任命是符合美国法律和惯例的。
 <br/>
 <br/>
 周一，三名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 参议员联名向联邦法院提交了一项
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诉讼.html" target="_blank">
  诉讼
 </a>
 ，指控川普总统任命惠特克（Matthew Whitaker）担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代理司法部长.html" target="_blank">
  代理司法部长
 </a>
 的决定〝违宪〞。
 <br/>
 <br/>
 这三名民主党参议员包括康涅狄格州的布鲁门斯（Richard Blumenthal）、罗德岛州的怀特豪斯（Sheldon Whitehouse）和夏威夷州的广野庆子（Mazie Hirono）。
 <br/>
 <br/>
 周二，参议院少数党领袖舒默（Chuck Schumer）也致信司法部监察机构，要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_调查.html" target="_blank">
  调查
 </a>
 白宫与惠特克之间是否存在〝非法或不当的沟通〞。
 <br/>
 <br/>
 民主党宣称，他们担心川普会利用惠特克监察
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查而使特别检察官穆勒（Robert Mueller）今后的调查朝着有利于自己的方向发展。他们的理由是，惠特克在成为代理司法部长前曾批评过穆勒调查。
 <br/>
 <br/>
 舒默在信中表示，近来有报导称惠特克与白宫之间有〝不寻常〞的接触，这引发了他的高度关注——惠特克是否与白宫沟通并试图破坏或阻挠穆勒调查？舒默说，这可能已经违反了法律或政策。
 <br/>
 <br/>
 中期选举结束后的第二天（11月7日），司法部长塞申斯（Jeff Sessions）应川普的要求辞职，塞申斯的幕僚长惠特克被川普任命为代理司法部长。同一天，司法部宣布，惠特克将接替副部长罗森斯坦（Rod Rosenstein）监督穆勒调查。
 <br/>
 <br/>
 针对民主党的反对，司法部于11月14日发表了一份备忘录，确认川普对惠特克的任命完全合法。
 <br/>
 <br/>
 备忘录指出，虽然宪法规定总统提名的主要官员须获得参议院批准，但惠特克只是临时性官员，因而不存在批评者所指控的违宪问题。
 <br/>
 <br/>
 周二，司法部再次声明，惠特克的任命符合美国法律和历史惯例。
 <br/>
 <br/>
 司法部发言人库佩克（Kerri Kupec）指出，在美国历史上，由未经参议院确认的官员临时履行参议院所确认的官员的职责的情况屡见不鲜，超过160例。对川普任命惠特克横加指责，实际上是忽略了美国几百年来的实践和先例。
 <br/>
 <br/>
 穆勒办公室也在法律文件中表示，惠特克的任命对他们的调查工作并无影响。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
</div>
