### 川普周一休斯顿集会 订票爆棚全城安保
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 下周一将在休斯顿举行大型集会，支持共和党联邦参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泰德．克鲁兹.html" target="_blank">
  泰德．克鲁兹
 </a>
 (Ted Cruz)竞选连任。原定场地可容纳七千人，报名的却有七万。休斯顿警察局长表示，已经准备好各种安保力量，保证集会的成功举行。请看报导。
 <br/>
 <br/>
 能容纳一万八千人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丰田中心.html" target="_blank">
  丰田中心
 </a>
 (Toyota Center)，已有七万人报名参加总统川普下周一晚间在休斯顿举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超级集会.html" target="_blank">
  超级集会
 </a>
 。警方将从周一早上5点45关闭丰田中心附近的一定区域，以及部分市中心道路。
 <br/>
 川普的超级集会（MAGA Rally)原订于可容纳七千多人NRG Arena(NRG 竞技场）举行，因报名人数过多，周四宣布将地方改为丰田中心，截至周五上午，报名人数7万多，超出场地可容纳人数近三倍。休斯顿警察局长表示，本次安保联合了联邦、德州、郡，市的资源，已经做好交通与安保的全面部署。
 <br/>
 <br/>
 超级集会计划于周一晚上6点半开始，丰田中心周边范围将于周一早上5点半开始封锁，休斯顿三个机场也会处于高度警戒状态。
 <br/>
 <br/>
 由于丰田中心四周不能停车，人们到达会场的可以乘坐轻轨或者走路，警方估计将有数万人在会场外面。
 <br/>
 <br/>
 进入会场不必出示身分证，但是不可以携带武器、背包、大包，可拆卸镜头的专业照相机，三脚架，单脚架，GoPro，自拍棒等物品。
 <br/>
 <br/>
 这是川普就任总统以来第二次在休斯顿出席的公众活动。上一次时在去年哈维飓风时期视察灾情。
 <br/>
 <br/>
 新唐人记者易永琦美国休斯顿采访报导
</div>
