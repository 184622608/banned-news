### 大篷车压境墨西哥 当地居民接力抗议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 的情况。五角大楼决定，在圣诞节前调回部署在边境的军队。与此同时，很多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 已经抵达与美国接壤的墨西哥边境，并引发当地墨西哥人的强烈不满。
 <br/>
 <br/>
 根据美国国土安全部的消息，在位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 、墨西哥一侧的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蒂华纳.html" target="_blank">
  蒂华纳
 </a>
 （Tijuana）圣伊西德罗过境点，已经聚集了6千来自中美洲的大篷车成员，其中500多人是罪犯。而这些人的目的是寻求美国给予政治庇护。
 <br/>
 <br/>
 大篷车成员来自洪都拉斯Luis Fernando：〝我正在等排号，希望能拿到庇护。〞
 <br/>
 <br/>
 不过，蒂华纳人首先对这些远道而来的陌生人表达不满，连日在街头抗议。
 <br/>
 <br/>
 蒂华纳居民Elvia Villegas：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 他在防卫他的边境，不像墨西哥，政客腐败，对边境安全无作为。他们不关闭边境。给蒂华纳带来麻烦。〞
 <br/>
 <br/>
 墨西哥人Rodrigo Melgoza：〝移民，可以！非法移民，不！他们不能侵犯墨西哥和墨西哥人的主权，正如这些人在边境的所为。〞
 <br/>
 <br/>
 墨西哥人Guadalupe Arangure：〝他们来到这里、违反我们的法律、侵犯我们的边境。他们唯一做的就是制造混乱和抱怨。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 就是入侵，不要听信那些歪曲的说法，那说法不对。把他们送回去，先考虑墨西哥人！〞
 <br/>
 <br/>
 美国国土安全部官员表示，大篷车成员主要是青壮年男性，大约占到75%；但妇女儿童却被推倒了前台。
 <br/>
 <br/>
 预计未来两周，还会有上千大篷车成员抵达蒂华纳。
 <br/>
 <br/>
 蒂华纳当地警察局长Mario Martinez：〝我们从没遇到过这种状况。情况非常复杂。〞
 <br/>
 <br/>
 蒂华纳居民Magdalena Baltazar：〝如果他们来为了工作，他们应该去工作；如果他们只是为了获得帮助，那应该到自己的国家去。他们说有更多大篷车会来，蒂华纳真的无法承受这些只为来这寻求帮助的人。〞
 <br/>
 <br/>
 此前副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 获得消息，大篷车获得左翼金主资助。
 <br/>
 <br/>
 美国总统川普11月9日签署一项公告，暂停为非法越过美墨边境的外国人提供政治庇护申请，有效期为90天。但是，北加州地方法院联邦法官蒂卡尔周二发布临时禁令，阻止了这项政令。临时禁令立即生效，并将于12月法庭聆讯后作最终判决。
 <br/>
 <br/>
 新唐人记者李兰、特约记者Kimberly Hayek纽约、蒂华纳连线报导
</div>
