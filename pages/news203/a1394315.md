### 分析：彭斯讲话透两大用意 中共进退两难
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】美国副总统彭斯（Mike Pence）日前发表的对华政策讲话，引发舆论广泛关注。有分析认为，彭斯的演说透出两大用意，其一，代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）总统向中共〝宣战〞，但保留回旋余地；其二，阐明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遏制中共.html" target="_blank">
  遏制中共
 </a>
 战略不是川普个人意愿，而是美国的国策。美国如此强硬的政策，令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共进退两难.html" target="_blank">
  中共进退两难
 </a>
 。
 <br/>
 <br/>
 当地时间10月4日，美国副总统彭斯在华盛顿发表演讲，在政治、经济、军事、外交等多个层面，对中共进行严厉谴责。
 <br/>
 <br/>
 有分析指，美中不仅深陷贸易战，随着最近几次预定会谈的破局，与双方军事活动增加，两国紧张局势已升至〝危险层级〞。而彭斯的对华政策演说，无疑是对中共进行全方位〝开火〞。
 <br/>
 <br/>
 外界认为，彭斯的演说有如代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府向中共公开〝宣战〞，其用意有两层：一，主帅川普暂不出马，保留回旋余地；二，阐明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遏制中共.html" target="_blank">
  遏制中共
 </a>
 战略不是川普的个人意愿，而是美国政府的国策。
 <br/>
 <br/>
 分析指，中美对抗至今，中共已陷入进退两难。不妥协，难以抵御川普政府的贸易制裁加码；妥协的话，又无法对被中共自己煽动的民粹主义至上的中国民众交代。
 <br/>
 <br/>
 《经济日报》10月5日刊文称，川普没有亲自上场与北京撕破脸，而是派彭斯扮演黑脸，等于是给北京当局保留了情面与谈判空间。
 <br/>
 <br/>
 彭斯在演讲结尾时也表示，〝（中美）竞争并不总是意味着敌意〞，川普总统希望共同促进双方的繁荣与安全，而不是分离。北京领导人现在仍可以改变路线，回归当初的〝改革开放〞精神。
 <br/>
 <br/>
 而在另一方面，彭斯的演说也阐明，遏制中共战略并不是川普的个人意愿，即使川普总统卸任，或期中选举后国会主控权易手，遏制中共的策略也不会改变。
 <br/>
 <br/>
 事实上，在彭斯演讲之前，川普政府已经在军事、政治、经济、外交等多个领域，接连推出反制中共的举措，显示中美关系正在走向全方位对抗。
 <br/>
 <br/>
 今年美国最新国防法案中，已明确与中共〝长期战略竞争〞是美国根本性的优先战略，美国将在军事、经济、外交和法治领域和中共长期对抗。
 <br/>
 <br/>
 日前，中共驻美大使崔天凯在讲话中，释放中共选择和美国〝合作〞、〝不对抗〞的信号。但白宫官员则当场回绝称，美国已选择了〝竞争〞。
 <br/>
 <br/>
 对于彭斯的演说，多名海外学者以〝新冷战宣言〞来比拟。旅美学者吴祚来在推特发表评论指，彭斯的演讲是一篇宣言书，再进一步就是战书。中美关系正在发生巨变，美国未来或会强力遏制中共扩张。
 <br/>
 <br/>
 值得一提的是，彭斯在演说中多次刻意以〝中国共产党〞取代中国，凸显了中美意识形态对立。
 <br/>
 <br/>
 他在演说中还列举中共种种恶行，包括台海军事行动、干预美国选举、试图让美国换总统、对美国进行政治渗透和网路攻击等等。而且，他每指出中方一种恶劣行径时，都点名一次〝中国共产党〞。
 <br/>
 <br/>
 例如彭斯说，中共花费数十亿美元在美国与其它国家宣传，并点名在美国许多主要城市能看到的中国环球电视网（CGTN），是依共产党大佬的命令行事。
 <br/>
 <br/>
 台媒称，如今，中共在与美国的对抗中已进退两难，看不到两全其美的解决方案。中共如退让，难以被中共自己煽动的民族主义至上的民众交代；而越晚妥协，川普政府加码越多、施压越重，美中关系也面临更糟的局面。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
</div>
