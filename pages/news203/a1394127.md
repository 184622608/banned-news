### 中期选举关乎美国未来 选民热情空前高涨
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】备受世界关注的美国国会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 定于11月6日。观察人士认为，这次选举不仅是对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）总统，也是对美国与世界新关系的公投。民调显示，美国选民热情空前高涨。
 <br/>
 <br/>
 11月6日，美国将举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，不仅要改选众议院所有435名众议员和参议院33名参议员，还要改选36个州的州长以及其它大小地方选举。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 从9月29日开始在美国5个州进行了为期一周的集会活动，旨在中期选举前夕站台支持共和党候选人，确保共和党人守住国会多数党优势。
 <br/>
 <br/>
 川普表示，他上任以来，美国失业率创历史新低，更多美国人有了工作。新签署的美国-墨西哥-加拿大贸易协定，是美国取得的又一个胜利，让美国人得到公平贸易，以及更多工作机会。
 <br/>
 <br/>
 集会现场座无虚席，川普步入会场时，受到全场民众热烈欢迎。他呼吁民众在11月的中期选举中都出来投票。
 <br/>
 <br/>
 民意调查公司皮尤研究中心称，因选举将决定对国会的政治控制，许多美国选民的热情都被调动起来了。近3/4的登记选民表示，他们关注的是共和党是否会保持对参众两院的控制权。
 <br/>
 <br/>
 皮尤的调查研究发现，大多数（近60％）登记选民表示，对川普的看法将影响到他们把票投给谁，其余的人表示他们对川普印象好坏不会影响到他们的决定。
 <br/>
 <br/>
 皮尤中心的民调发现，目前美国选民的投票热情是20多年来历次中期选举中最高的。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
