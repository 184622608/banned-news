### 川普签新移民公告 禁非法入境者申庇护
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】周五，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）签署了一项
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民公告.html" target="_blank">
  移民公告
 </a>
 ，自本周六开始的90天内，将禁止通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 非法进入美国的移民寻求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_庇护.html" target="_blank">
  庇护
 </a>
 。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：〝我刚刚签署了关于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_庇护.html" target="_blank">
  庇护
 </a>
 的公告，非常重要的。人们可以进入美国，但必须通过入境口岸。我们希望人们来我们国家，但他们必须合法进入美国。〞
 <br/>
 <br/>
 这项不具追溯效力的禁令，杜绝了从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 进入美国的非法移民申请庇护的可能，虽然有效期只有90天，但之后可根据国务卿、国土安全部长及司法部长的建议进行延长，直到与墨西哥达成有效阻止非法移民涌入美国的协议。
 <br/>
 <br/>
 川普同时呼吁民主党配合，支持新移民法，以保护美国不再受非法移民潮的冲击。
 <br/>
 <br/>
 就在周五当天，又有数百名大篷车移民，离开墨西哥城的临时庇护所，继续北上美国的行程。
 <br/>
 <br/>
 此外，美国第九巡回上诉法庭周四裁定，要求川普政府继续执行前总统奥巴马制定的DACA计划。但川普的回答有些让人意外。
 <br/>
 <br/>
 美国总统 川普：〝好消息是，DACA案昨天被第九巡回法庭否决，这是我们一直最终等待的结果，因为之后我们就可以上诉到最高法院，我们希望把DACA案带到最高法院。现在DACA案有望进入最高法院，并得到公正裁决。〞
 <br/>
 <br/>
 新唐人记者乔安综合报导
</div>
