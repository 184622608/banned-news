### 川普抵达加州野火灾区 慰问灾民
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】周六傍晚，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 ，与加州州长布朗和侯任州长纽森一起，视察灾情。
 <br/>
 <br/>
 昔日风景优美的天堂镇，眼下只剩一片焦土和废墟。看到这场景，川普表示非常痛心。
 <br/>
 <br/>
 美国总统川普：〝看看这里的情况，谁也想不到会这样。联邦政府支持你们，我们互相支持。看了让人很痛心。至于（被夺去的）生命，还不太清楚。有了一个数字，但还有很多人没有算进去。〞
 <br/>
 <br/>
 川普说，这还不是受灾最重的地区。下一步，他将和环境小组一起，尽快开始重建工作。
 <br/>
 <br/>
 美国总统川普：〝我们必须处理好，维护好，我们将和环境小组合作。我想大家都看到了光明。我想不会再有这么严重的事了。我们必须要尽快去做。〞
 <br/>
 <br/>
 川普嘱咐布朗州长，一定照顾好灾区居民。
 <br/>
 <br/>
 美国总统川普：〝会做好的。但现在我们要照顾好心灵受伤的民众。失去亲人的家庭，失去了很多人。但我们有杰出的人员在做（救助）工作，我们会比其他任何人做的都好。〞
 <br/>
 <br/>
 随后，川普还将到另外一处
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火灾区.html" target="_blank">
  野火灾区
 </a>
 查看。
 <br/>
 <br/>
 新唐人记者易文、尚敬综合报导
</div>
