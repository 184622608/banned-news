### 匹兹堡教堂血案11死 凶嫌出庭承认29项罪状
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_匹兹堡.html" target="_blank">
  匹兹堡
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹太教堂.html" target="_blank">
  犹太教堂
 </a>
 日前发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，造成11人死亡6人受伤。凶嫌罗伯特．鲍尔斯被控29项联邦重罪。周一（29日）他在联邦法院首次出庭，承认所有罪状。
 <br/>
 <br/>
 现年46岁的鲍尔斯（Robert Bowers）周六坐着轮椅出庭，签署了与他被控罪名有关的法院文件，他也承认了解那些罪名。
 <br/>
 <br/>
 鲍尔斯被控29项联邦重罪，其中11项为使用枪枝武器造成伤害，7项为对警方动用武力，还有11项为因受害人的宗教信仰而对其谋杀。
 <br/>
 <br/>
 联邦检察官布雷迪（Scott Brady）发表法院简短声明称，下次开庭时间是11月1日，届时联邦政府将提出证据，证明鲍尔斯杀害了11名正在参加宗教活动的信徒，并射伤了6人。
 <br/>
 <br/>
 宾州西部地区联邦检察官 Scott Brady表示，〝在敬神的场所发起攻击，尤其令人发指。那是神圣的地方，是慈悲且祥和的场所，是人们聚集在一起，表达最珍视信仰的地方。〞
 <br/>
 <br/>
 该事件发生在上周六（27日）犹太教的安息日，当时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹太教堂.html" target="_blank">
  犹太教堂
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生命之树.html" target="_blank">
  生命之树
 </a>
 〞正在进行一场新生儿命名仪式。鲍尔斯使用3个格洛克17手枪和一支AR-15自动步枪攻击人群，造成11人死亡，6人受伤，伤者包括4名警察。
 <br/>
 <br/>
 鲍尔斯在与特警部队的枪战后被捕，因受伤被送医救治、情况稳定。
 <br/>
 <br/>
 当局周日公布的死者名单中最年轻的54岁，最年长的97岁，包括一对兄弟和一对夫妻。
 <br/>
 <br/>
 美国总统川普（特朗普）周六在第一时间发推文，对事件表示关注，并说〝事件比想象的更严重〞。当天他在伊利诺伊州的一场竞选集会上，又严词谴责这一仇恨犯罪。
 <br/>
 <br/>
 〝生命之树〞犹太教堂已经有超过150年历史，教堂所在的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_匹兹堡.html" target="_blank">
  匹兹堡
 </a>
 松鼠丘地区（Squirrel Hill）从1920年就有众多犹太人聚居。
 <br/>
 <br/>
 全美的犹太教堂和犹太中心在此枪击事件后，都加强部署警力巡逻。
