### 苹果发表会将登场 7款预测新品一睹为快
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】苹果第二场秋季发表会，在纽约时间10月30日登场。媒体盘点了发表会的七款
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新品.html" target="_blank">
  新品
 </a>
 ，下面带您一睹为快。
 <br/>
 <br/>
 苹果银白色的大logo，就高挂在纽约布鲁克林音乐学院的墙面上，因为Apple今年第二场产品发表会，即将在这里盛大展开。而产品的最大亮点，莫过于两款不同萤幕尺寸的全新iPad Pro。
 <br/>
 <br/>
 科技网站编辑Russell Holly：〝我觉得最大的亮点是两款iPad，一个大尺寸跟一个比较小尺寸，这两款将会移除一般在iPad上会有的实体Home键，另外与刚发表的新款iPhone相似之处是，上方的感测器可以进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脸部辨识功能.html" target="_blank">
  脸部辨识功能
 </a>
 ，或许也将连结新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Apple Pencil.html" target="_blank">
  Apple Pencil
 </a>
 的功能。〞
 <br/>
 <br/>
 外媒整理的七大预测
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新品.html" target="_blank">
  新品
 </a>
 还包括，市场预期的第二代Apple Pencil，预估将加强类似AirPods的蓝牙配对功能；还有苹果正在研发的升级版iPad mini，将具备全新A12X处理器以及低成本的Retina萤幕；而13吋的新款Macbook Air，将搭载Retina萤幕，体积也将更轻薄；另外，12吋的Macbook则预计采用Intel第八代处理器，延长笔电的续航力；分析师也认为这次iMac的产品将会再升级，效能再提升；而今年苹果也有机会推出针对专业用户的新款Mac Mini，不过外传高规版的Mac mini可能配置更高的尺吋，预计采用Intel第八代处理器。
 <br/>
 <br/>
 新唐人记者王愉贺、赵庭誉纽约报导
</div>
