### 川普宣布修改庇护规定 防止移民欺诈入境
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周四（11月1日）宣布，向美国政府申请
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_庇护.html" target="_blank">
  庇护
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 必须在边境口岸合法地亲自递交申请。这一规定是川普政府针对南部边境面临的危机、防止移民进行庇护欺诈而采取的应对措施。
 <br/>
 <br/>
 周四下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在白宫宣布对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_庇护.html" target="_blank">
  庇护
 </a>
 政策进行重大调整。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新规.html" target="_blank">
  新规
 </a>
 不再允许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 入境提出申请后即离开、并借等待审批的时间而获取立足。
 <br/>
 <br/>
 川普表示，仍允许在官方入境口岸现身的移民提交庇护申请，但拒绝偷偷入境的移民所提出的申请。他表示，这一规定将会改变现在的庇护模式，从而降低继续向美国边境进发的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民人数。
 <br/>
 <br/>
 川普指出，那些能够进入美国境内的大篷车移民将被收容在〝帐篷城〞（tent-city）拘留设施中。川普派往美墨边境的军队搭建起这些设施，目的是防止移民入境后失控逃脱。
 <br/>
 <br/>
 川普说，他将终结〝抓了就放〞（catch-and-release）的做法，该惯例导致近年来非法移民问题越来越严重。成千上万的非法移民被边境巡逻人员逮捕后获得释放，然后进入美国境内等待庇护听证。
 <br/>
 <br/>
 针对法律性质疑，川普指出，庇护新规是完全合法的。
 <br/>
 <br/>
 川普没有具体说明新规的具体内容以及何时推出，但他表示，他希望将在下周签署一份完整的行政令。
 <br/>
 <br/>
 川普指出，在当前的制度下，移民利用欺诈性或没有存在价值的理由申请入境美国，再利用移民律师教给他们的训练好的语言申请庇护。
 <br/>
 <br/>
 川普表示，一些移民在嘲弄美国的移民法，滥用庇护政策，并直指大篷车移民并非合法的寻求庇护者。
 <br/>
 <br/>
 他指出，庇护并非是给生活贫困的人所提供的项目，全世界有几十亿人生活贫困，美国不可能全部容纳他们。
 <br/>
 <br/>
 川普还表示，解决移民危机唯一的长期方案以及保证美国持久维护国家主权的唯一方式，就是国会攻克阻挠，废除开放边界的政策，只有这样，才能从根本上解决问题。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
 <br/>
