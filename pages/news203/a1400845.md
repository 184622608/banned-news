### 500难民试图非法闯关 美国短暂关闭边境
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】周日，五百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 ，试图跨越美墨边境，美国临时关闭了圣伊西德罗入境口岸数小时。墨西哥表示将遣返试图非法入境美国的难民。
 <br/>
 <br/>
 周日早上，聚集在美墨边境的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 准备冲击边境。墨西哥警察尝试驱散他们，但仍有约500名难民试图非法跨越边境。
 <br/>
 <br/>
 一些难民试图闯过边境，另一些穿过边境口岸附近的运河。
 <br/>
 <br/>
 当地时间上午11点半左右，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国边境.html" target="_blank">
  美国边境
 </a>
 执法人员设置路障和铁墙，并临时关闭了圣伊西德罗边境口岸的汽车和人行通道，下午五点边境重新开放。
 <br/>
 <br/>
 据报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法闯关.html" target="_blank">
  非法闯关
 </a>
 的难民试图向美国一方丢掷石块，迫使美方使用催泪瓦斯驱赶他们。
 <br/>
 <br/>
 有消息说，美国总统川普已经和墨西哥达成协议，难民在庇护申请批准前必须留在墨西哥。
 <br/>
 <br/>
 不过，墨西哥政府表示，还没有和美国达成协议。
 <br/>
 <br/>
 周一，川普发推文，呼吁墨西哥政府遣返移民中的罪犯。
 <br/>
 <br/>
 推文说：〝墨西哥应该将挥舞旗子的移民送回国，其中很多是冷酷无情的罪犯。用飞机，公车或者任何方式，但他们不会进入美国。如果需要，我们将永久关闭边境。（美国）国会，出资建墙！〞
 <br/>
 <br/>
 墨西哥当局稍后表示重新控制了边境，并将遣返试图非法越境的中美洲难民。
 <br/>
 <br/>
 近日，数千名难民来到蒂华纳，并举行抗议。当地居民表示，生活受到影响。
 <br/>
 <br/>
 蒂华纳居民：〝我们好像生活在战区。成千上万的中美洲（难民）到这，试图非法越境进入美国。〞
 <br/>
 <br/>
 当地官员表示，每天有大约7万辆汽车和2万多行人通过圣伊西德罗的入境口岸。
 <br/>
 <br/>
 新唐人记者毕心慈、李静明综合报导
</div>
