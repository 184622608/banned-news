### 白邦瑞曝中共百年倒美计划 双面谍假投诚误导白宫
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】外界普遍认为美中对抗之势已不可逆转。美媒文章指，这是美国被中共欺骗多年后的真正醒觉，北京诈术再难奏效。文章引述美高官著作揭露，中共一直在秘密执行击倒美国的百年计划，还曾派双面间谍让美国相信中共很快将政改。
 <br/>
 <br/>
 自由亚洲电台12月5日发表署名文章指，美中关系转向对抗，并非完全是因为川普（特朗普）上台，其实华府从亲中到反共的转变早已开始，其动因来自美国政府对中共多年欺骗的彻底醒觉，因此很难再有逆转。
 <br/>
 <br/>
 文章说，美国此前一直帮助中国发展，期望中共政府会改良，但是近年来华府官员逐渐觉醒，认识到中共其实一直在欺骗美国，并且从没放弃击倒美国的秘密计划。
 <br/>
 <br/>
 文章提到2014年出版的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_百年马拉松.html" target="_blank">
  百年马拉松
 </a>
 ：中国称霸全球的秘密策略》一书，揭露了美国的觉醒过程。
 <br/>
 <br/>
 该书作者是川普多次提及的华府〝中国通〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白邦瑞.html" target="_blank">
  白邦瑞
 </a>
 （Michael Pillsbury），他是美国一名资深的外交、情报人员，并一度是美国共和党内位高权重的亲中派。
 <br/>
 <br/>
 白邦瑞在《百年马拉松》中严正宣示：美国的亲中派都彻底错了。中共自1949年开始，便在秘密部署一百年之内击倒美国，到2049年成为世界霸主，但表面上一直伪装成和美国合作的弱小国家。美国现在才发现中共的长期野心，可能已经太晚了。
 <br/>
 <br/>
 书中引用大量的秘密情报和中共变节者的证词，揭露中共的野心和策略。其中对1990年代初两名中共〝变节者〞的描述，非常形象地展示了中共为欺骗美国如何处心积虑。
 <br/>
 <br/>
 书中称这两人为白先生和绿小姐，他们都来自中共高层，却向美方提供了完全相反的情报。
 <br/>
 <br/>
 白先生指，中共的反美鹰派已在党内斗争中胜出，并得到邓小平支持。因此，中共并无诚意走向真正的市场经济，更不用说政治上的自由化和民主化。
 <br/>
 <br/>
 相反，绿小姐则告诉白邦瑞，中共现在是改革派当道，正在努力以美国为榜样改革中国经济，之后政治改革就会自然出现。
 <br/>
 <br/>
 书中说，当时华府奉行帮助中共发展经济的政策，因此白宫高层选择性地相信绿小姐之类的情报。但后来绿小姐被揭发是双重间谍，并被中情局逮捕。之后形势的发展证明，白先生提供的才是准确情报。
 <br/>
 <br/>
 白邦瑞指，中共的精细骗术，再加上华府一厢情愿的主观愿望，导致美国遭中共愚弄多年。其中一个典型例子就是，美国帮助中共加入世贸，希望这会加速中国改革，结果是中共藉此壮大经济实力后，反而对美国恩将仇报。
 <br/>
 <br/>
 白邦瑞警告，过去几十年中共操弄全世界帮它发展经济，现在则利用这样得来的经济实力，明目张胆地控制世界舆论、勒索小国、扩张军备。
 <br/>
 <br/>
 白邦瑞在书中敦促华府，应该及早放弃幻想，将中共列为竞争对手，而不再是合作对象。
 <br/>
 <br/>
 今年8月，川普政府通过的新国防法案中，已经明确将美中关系定性为〝长期战略竞争〞，并将其列为美国的根本战略。这显示白邦瑞4年前的呼吁已经奏效。
 <br/>
 <br/>
 另外，这次美中对抗的导火索——美中贸易战，被认为剧本来自2011年出版的《致命中国：中共赤龙对人类社会的危害》一书。此书作者是美国经济学家彼得．纳瓦罗（Peter Navarro）和葛瑞格．奥特瑞（Greg Autry），前者现任川普的贸易顾问。
 <br/>
 <br/>
 该书相信列举了中共如何使用八种违规的贸易手段摧毁美国经济，也可被视为美国识破中共骗局的代表作之一。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
