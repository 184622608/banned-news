### 绿卡门槛抬高 收入达贫穷线250%可加分
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】美国国土安全部近日公布《拒绝入境公共负担因素》新规则，确保申请美国签证和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绿卡.html" target="_blank">
  绿卡
 </a>
 的人〝不依赖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公共福利.html" target="_blank">
  公共福利
 </a>
 〞，目的在于保护美国纳税人。有美媒称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府还拟为申请绿卡的合法移民设下更高门槛，两口之家年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_收入.html" target="_blank">
  收入
 </a>
 至少4.1万，逾半申请人恐被驳。
 <br/>
 <br/>
 为了不让移民人口成为仰赖税金资助福利计划的〝公共负担〞(public charge)，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府推出一系列移民政策革新。
 <br/>
 <br/>
 美国国土安全部周三（10日）公布《拒绝入境公共负担因素》新规则，确保申请美国签证和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绿卡.html" target="_blank">
  绿卡
 </a>
 的人〝不依赖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公共福利.html" target="_blank">
  公共福利
 </a>
 〞。国土安全部表示，〝自给自足一直是美国移民法的基本原则〞。
 <br/>
 <br/>
 按照新规则，美国将首次在批准绿卡和延期签证时，考虑申请人是否领取食物券和住房补贴等。预计每年将有38万人受到影响，将给美国纳税人节省27亿美元。
 <br/>
 <br/>
 川普政府研拟的最新改革中，绿卡申请者如果家庭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_收入.html" target="_blank">
  收入
 </a>
 水平超过联邦贫穷线250%，即2018年个人收入为3万350美元、4口家庭为62,750元，申请绿卡时会加分。反之，如果失业、办理移民身份时申请免除费用、在就医时没有私人的医疗保险公司，则会减分。
 <br/>
 <br/>
 新草案有60天的征集公众意见期，到12月10日截止。一旦最终的规定颁布，将在60天后生效。
 <br/>
 <br/>
 根据草案，曾使用或随时可能使用医疗照顾(Medicaid)、医疗保险处方药D计划(Medicare Part D)、粮食劵(Supplement Nurtrition Assistance Program, SNAP)，以及Section 8、房租津贴等房屋补助计划等福利者，在申请签证，或调整合法居留身分(绿卡)时，移民局可将其视为〝公众负担〞，认定资格不符而拒发签证。
 <br/>
 <br/>
 这项新规适用于3类人：第一，申请美国签证者(Applicants for Admission)，任何这类申请人，如被认定可能在任何时候成为公众负担，将可予以拒签。
 <br/>
 <br/>
 第二，申请延长在美国的停留期限及申请改变身分者(Extension of Stay and change of status applicants)，必须证明以前及现在没有使用公共福利，且以后也不会使用到这些公共福利。
 <br/>
 <br/>
 第三，对于已经在美国要申请调整身分者(Adjustment of Status Applicants)，在申请合法居留时，移民局有权检视并决定他们提出的文件，是否足够证明其不会成为公共负担。
 <br/>
 <br/>
 不过，难民、政治庇护或基于人道主义的永久或临时身分申请人，则不在新规定范围内。
 <br/>
 <br/>
 美国移民律师夏渢表示：〝难民，包括一些特别列出来的国家，包括：伊拉克、阿富汗这些国家的难民，都特地是区别在外，他们是不受这个规定限制的。〞
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
