### 选民：艾维乐胜选对保SHSAT至关重要
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】竞选连任的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_州参议员.html" target="_blank">
  州参议员
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艾维乐.html" target="_blank">
  艾维乐
 </a>
 （Tony Avella）2日前往选区内大学点26大道的〝永康成人日间护理中心〞拜票，受到热烈欢迎。支持者表示，艾维乐胜选对保住特殊高中入学考试SHSAT、挡住民主党极左派对社区的损害至关重要。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艾维乐.html" target="_blank">
  艾维乐
 </a>
 呼吁支持者6日出来投票，让他可以连任继续为大家服务。耆老们对艾维乐说，感谢他勤勤恳恳为社区工作，祝愿他连任成功。
 <br/>
 <br/>
 在与耆老们的合照结束后，艾维乐扶起旁边的一个华人老太太，并搀扶着她走到自己的座位上。艾维乐的支持者Jenny看到这个细节很感动，她表示：〝（艾维乐）一点没有作秀的感觉，为这样的民选官员点赞！〞
 <br/>
 <br/>
 Jenny表示，他们到大学点大道华人商家前张贴艾维乐的中文竞选海报时，华人商家都表示支持艾维乐竞选连任，〝因为他们不喜欢在College Point建流浪汉收容所。〞
 <br/>
 <br/>
 多年来推动华人投票的社区人士罗维宗表示，明年市长白思豪改革特殊高中入学考试SHSAT的提案将在州议会投票，州参议会是挡住该提案的重要一环。所以，一直坚定捍卫SHSAT的艾维乐能否胜选，对能否保住SHSAT至关重要。
 <br/>
 <br/>
 罗维宗说，挑战艾维乐的刘醇逸是民主党极左派，主张大麻合法化，无视大麻对社会及青少年的危害；另外，第11选区的地税已经很高，还在不断上涨，而刘醇逸还主张地税上涨无上限，全然不考虑民众的负担。这些都是左派的〝共产主义〞思想，只想千方百计搜刮民脂民膏，不考虑民众利益。
 <br/>
 <br/>
 罗维宗还指出，在刘醇逸当年竞选市长时，为了拉拢非裔、西语裔的选票，提出改革SHSAT。在本次竞选中，刘醇逸不敢得罪非裔、西语裔，始终不开口承诺支持SHSAT的单一考试，没有参加过任何一次抗议废除SHSAT的集会及游行。如果刘醇逸当选，大家可以想像后果。当前州参议会共和党和民主党的比例为32比31，也就是民主党的参议员一个都不能再多，这是保卫SHSAT、反对大麻合法化的最后防线。
 <br/>
 <br/>
 据悉，为了孩子和社区的未来，不少第11选区的华人妈妈们，近期不仅积极当义工为艾维乐助选，而且以前从不投票的她们，在本次普选日将会第一次出来投票。
 <br/>
 <br/>
 ──转自《大纪元》
 <br/>
