### 汇聚东西结善缘 Destiny礼品店开幕
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】在美国，感恩节标志着采购季节开始。日前，一家名为Destiny的精美礼品店在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大华府地区.html" target="_blank">
  大华府地区
 </a>
 正式
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开幕.html" target="_blank">
  开幕
 </a>
 。本地政要及商会领袖出席剪彩，感谢他们为社区带来最美好的礼物与祝福。
 <br/>
 <br/>
 感恩节之际，维吉尼亚州Fair Oaks商场内，一家汇聚东西方
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统文化特色.html" target="_blank">
  传统文化特色
 </a>
 的礼品店Destiny亮丽登场。本地政要及商会领袖出席剪彩，感谢商店为北维州地区带来最美好的礼物与祝福。
 <br/>
 <br/>
 亚美商会董事会主席 元姬：〝祝贺我们的社区，得到这样美好的祝福。我们能够分享这样美好的文化。我看到这里有舞蹈，有音乐，有艺术，并看到有很多的朋友。〞
 <br/>
 <br/>
 维州费尔法克斯市议员 李尚炫：〝我们要祝贺这样的商家，能在这里成功经营，并接触各种各样的人群。祝贺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Destiny礼品店.html" target="_blank">
  Destiny礼品店
 </a>
 ，也感谢大家来支持生意。〞
 <br/>
 <br/>
 从碧绿高洁的翠竹，到意境悠远的山水画，从传统木雕摆件，到西方铜质雕塑，从喜马拉雅盐灯，到印度手工神秘盒，店内商品每一件都独具特色，并强调传统与自然。
 <br/>
 <br/>
 店内还销售法轮大法书籍，让人们有难得的机会，了解博大精深的修炼文化，并与佛法结缘。
 <br/>
 <br/>
 维州州众议员  Kenneth Plum：〝今天，我在店里很好地参观了一下。店员介绍我这些物品的意涵，对于打坐的影响，如何在打作中使用等等。我所知有限，但可以说是个学生，并对学到的很有兴趣。〞
 <br/>
 <br/>
 Destiny礼品店所在的费尔法克斯(Fairfax)市，比邻美国首都华府，经济发达并充满多元文化。店主也希望，未来能在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大华府地区.html" target="_blank">
  大华府地区
 </a>
 开设更多分店，服务社区。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
