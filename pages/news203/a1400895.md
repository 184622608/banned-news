### 延续购物热潮 网购星期一销售额料创新高
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】26日是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网购星期一.html" target="_blank">
  网购星期一
 </a>
 ，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 加州最大的配送中心，工人们迎来最繁忙的一天，来看记者的报导。
 <br/>
 <br/>
 传送带上，以动画片《圣诞怪杰》为主题的绿色包装盒凸显著浓浓的节日气氛。在旧金山湾区特雷西市（Tracy）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 配送中心，每800名工人被分成一个轮班，不停工作，迎接该电商平台历史上销售额最大的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网购星期一.html" target="_blank">
  网购星期一
 </a>
 。
 <br/>
 <br/>
 亚马逊发言人Cedric Ross：〝去年的网购星期一，亚马逊在全球发送了8300万件货物，也就是说每秒发送961件。我们希望今年比这好要多。〞
 <br/>
 <br/>
 数据分析公司奥多比（Adobe Analytics）的统计显示，刚结束的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑色星期五.html" target="_blank">
  黑色星期五
 </a>
 ，全美在线销售额达到62.2亿美元，比去年增长23.6%。而今年的网购星期一预计将达到78亿美元的销售新高。
 <br/>
 <br/>
 记者：〝对喜欢网络购物的人士、特别是年轻族群来说网购星期一才是真正
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_购物季.html" target="_blank">
  购物季
 </a>
 的开始。像我身后这些电子商品、时尚服装和玩具等各类产品，在亚马逊上都有大幅折扣。〞
 <br/>
 <br/>
 亚马逊上，包括自家产品在内的电子设备，最高有50%的优惠。其他零售业者也抓住机会，抢占商机。
 <br/>
 <br/>
 护肤品公司创办人Alex McIntosh：〝因为我们卖节日礼物，这有礼物套装，今天的销售额会比平常会高出5到10倍。〞
 <br/>
 <br/>
 亚马逊发言人Cedric Ross：〝如果你是亚马逊的Prime会员，可享有免费两日送货，有些货物，今天就能收货，当日送货。〞
 <br/>
 <br/>
 全美零售业联合会预计，今年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电子商务.html" target="_blank">
  电子商务
 </a>
 的消费者数量将达到7500万人，网购冲击实体店面的趋势将继续扩大。
 <br/>
 <br/>
 新唐人记者古馨乐、林骁然加州特雷西采访报导
</div>
