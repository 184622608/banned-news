### 川普评奥巴马多年前移民主张：我100%赞成
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】周二（10月23日），美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在推特上晒出一段前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 多年前的演讲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视频.html" target="_blank">
  视频
 </a>
 ，对其当时的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民主张.html" target="_blank">
  移民主张
 </a>
 点赞，并暗讽民主党的移民政策受政治需要驱使，前后不一。
 <br/>
 <br/>
 周二下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在推文中贴出一段
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 （Barack Obama）就任总统前的讲话
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视频.html" target="_blank">
  视频
 </a>
 ，并写道：
 <br/>
 <br/>
 〝我100％赞同奥巴马总统！〞
 <br/>
 <br/>
 视频剪辑中的奥巴马当时尚是伊利诺伊州民主党参议员，他所倡议的移民政策与当前民主党的主张迥然不同，估计川普的〝赞扬〞只会令民主党人感到尴尬。
 <br/>
 <br/>
 奥巴马在该演讲中警告，非法移民对美国的法制构成了威胁，对合法移民来说也是不公平的。
 <br/>
 <br/>
 他说，美国人民是慷慨好客的，但是对于那些非法入境、不尊重美国法制、漠视守法者的人，美国不能接受。
 <br/>
 <br/>
 奥巴马指出，不能允许外国人没有证件许可、未经检查地偷偷进入美国，他们逃避合法进入美国，而合法移民却在耐心地排队等待。
 <br/>
 <br/>

 <br/>
 奥巴马的上述主张刚好是川普正在倡议的，但是，如今很多民主党人都在驳斥它，并称之为种族主义者的政治〝狗哨〞。
 <br/>
 <br/>
 川普晒出奥巴马以往的视频，其实也在暗讽民主党支持非法移民是用心叵测——为争夺选票不顾美国民众的利益和安全。川普曾多次公开揭露这一问题的实质。
 <br/>
 <br/>
 美国当前正面临来自非法移民的严重威胁，仅上个周末，就有大约7000名洪都拉斯移民跨越墨西哥南部边界向北挺进。据墨西哥主流日报〝El Universal〞周一报导，目前向美国进发的非法移民已高达14,000人，其中多数已强行进入墨西哥境内，预计不日即将抵达美墨边界。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
