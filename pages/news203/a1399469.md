### 川普白宫讲话 推动监狱改革《第一步法案》
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周三在白宫发表讲话，推动众议院通过的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第一步法案.html" target="_blank">
  第一步法案
 </a>
 》。这项法案被认为是几十年来美国最重要的监狱制度改革法案，受到两党支持。
 <br/>
 <br/>
 站在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 身后的是促成推动监狱制度改革法案的部分幕后功臣，包括执法人员、教育及信仰团体。他们与川普政府合作，达成了两党协议。
 <br/>
 <br/>
 美国总统川普：〝今天，我很高兴的宣布支持这项两党法案，将使我们的社区更安全，并为刑满释放人员提供第二次机会，这非常重要。〞
 <br/>
 <br/>
 川普表示，美国现在经济强劲，法案鼓励刑满释放人员去找工作，开始新生活。
 <br/>
 <br/>
 美国总统川普：〝首先，它将为低风险的囚犯提供新的激励措施，以学习找工作所需的技能，改掉旧习惯，获释后守法。这些激励措施将鼓励他们参加职业培训、教育和信仰课程。〞
 <br/>
 <br/>
 重要改革还有，安排犯人出狱后就近获得家庭社区关怀、合理的量刑改革，同时将危险和暴力犯罪分子赶出街头。
 <br/>
 <br/>
 美国总统川普：〝在许多方面，我们对真正恶劣的罪犯更加强硬。不幸的是，有许多坏人。但我们以不同方式对待不同罪犯。〞
 <br/>
 <br/>
 对两党共同合作达成协议，川普感到非常高兴。
 <br/>
 <br/>
 美国总统川普：〝当共和党人和民主党人对话、辩论和寻求共识时，我们可以取得突破，推动我们的国家前进，为我们的公民服务。〞
 <br/>
 <br/>
 目前，《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第一步法案.html" target="_blank">
  第一步法案
 </a>
 》仍在参议院修改，若通过，将成为美国数十年来最重要的监狱制度改革法案。
 <br/>
 <br/>
 新唐人记者李梅、田瑞综合报导
</div>
