### 三大难题棘手 中美90天〝缓刑〞谈判不被看好
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】12月1日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 上，北京用一系列让步，换来暂缓贸易战升级的90天谈判期。但外界对这90天谈判普遍不看好。港媒指，三大棘手问题决定谈判难有成果。而荷兰一名金融主管表示，如果谈判有任何实质进展，就吃掉自己的帽子。
 <br/>
 <br/>
 随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 结束，美中90天谈判即将展开。美媒消息，中共副总理刘鹤将作为习近平特使，在未来数周内率领30人代表团赴华盛顿磋商。
 <br/>
 <br/>
 港媒指，根据当前中美关系判断，这场谈判将极端艰巨，其中存在三个棘手问题：
 <br/>
 <br/>
 一个是中方所能给出的让步清单，很难满足美方的要求。
 <br/>
 <br/>
 根据白宫声明，双方谈判的重点为有关中共〝强制性技术转让、知识产权保护、非关税壁垒、网络入侵及盗窃、服务业及农业〞等方面的结构性改革，如果90天内不能达成协议，美方会继续增加关税。
 <br/>
 <br/>
 而中方除了形式上不断承诺〝加强保护知识产权〞之外，迄今从未就这些关键问题显示出让步迹象。外界分析，如果按美方要求进行结构性改革，实现真正的公平贸易，就将危及中共政权的安全，因此北京只能〝揣着明白装糊涂〞，一再宣称〝不知道美国到底想要什么〞。
 <br/>
 <br/>
 第二，中美间还有其它棘手问题，和贸易战互相缠绕。
 <br/>
 <br/>
 其中包括南海问题、台湾问题、新疆人权问题以及美国对中共的科技封锁等。
 <br/>
 <br/>
 川习会前，美国副总统彭斯首次将南海自由航行列入川习会达成协议的条件之一，但此事源起于11月9日中方在第二轮美中外交安全对话上的挑衅。不过，最终川习会上没有提及南海问题。
 <br/>
 <br/>
 而北京方面，则不断将贸易问题牵扯上其它中美矛盾以混淆视听，掩盖自己在贸易战道义层面的明显劣势。另外，其它无关的政治问题也被北京拉来充当贸易战筹码，其中典型例子是朝核问题，这也让解决贸易战变得更为复杂。
 <br/>
 <br/>
 第三，美国国内强硬派声音高涨。
 <br/>
 <br/>
 伴随着贸易战不断升级，遏制中共已成为美国朝野和两党的共识。川普（特朗普）政府如果有意对中共软化态度，将面临来自国内各方的制约。
 <br/>
 <br/>
 据《华尔街日报》消息，在川习会前，以贸易代表莱特希泽、国家安全顾问博尔顿以及白宫贸易顾问纳瓦罗等为代表的白宫强硬派，都强烈反对和北京达成这种临时协议，因为担心会因此落入中方的陷阱，陷入几乎不会取得任何成果的无休止谈判中。
 <br/>
 <br/>
 此外，川普还面临来自民主党的压力。美国众议院的民主党领袖佩洛西一直是中共痛恨的强硬反共人士，而参议院的民主党领袖舒默日前也公开表态将致信川普，希望他不要软化对中共的立场。
 <br/>
 <br/>
 在美国内部制约下，白宫很难对中共做出任何妥协。如果北京90天后不能答应美方的贸易要求，白宫继续拖延贸易战的可能性已经很小。
 <br/>
 <br/>
 法媒指，北京争取来这90天谈判，等于给自己套上一个紧箍咒。
 <br/>
 <br/>
 前美国财政部中国专家、洛杉矶TCW集团基金经理人骆文杰（David Loevinger）表示，虽然美中贸易战暂缓升级，但90天后要看到根本改变相当困难，〝美中贸易战的达摩克里斯之剑仍然悬在那里，闪烁着寒光。〞
 <br/>
 <br/>
 荷兰合作银行集团（Rabobank ）驻香港亚洲金融市场研究部门主管艾佛瑞（Michael Every）更打赌称，〝如果（贸易战暂缓）意味着任何实质进展，我就吃掉我的帽子。〞
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
