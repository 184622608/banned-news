### 中期选举关键战 专家：民调数字谨慎参考
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】2018
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，全美众多国会及州长席位要改选，下面来看本台记者盘点的关键选战以及对当前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民调.html" target="_blank">
  民调
 </a>
 的分析。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）在周一推文中指出CNN
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民调.html" target="_blank">
  民调
 </a>
 在打〝唱衰〞牌：〝看到CNN民调的唱衰数字及错误基调真是可笑。周二见分晓。不要落入‘唱衰’把戏的陷阱。出来投票！〞
 <br/>
 <br/>
 多数分析认为，本次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 可以继续控制国会参议院。目前共和党占51席位，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 49席位；今年民主党26席需重选、共和党9席。目前竞争激烈的选举在内华达州、亚利桑那州、印第安纳州、佛州、密苏里州。
 <br/>
 <br/>
 密苏里州共和党联邦参议员参选人霍利（Josh Hawley），目前较对手民主党现任参议员麦卡斯基尔（Claire McCaskill），在民调结果上平均值（REALCLEARPOLITICS AVERAGE）领先0.5%。
 <br/>
 <br/>
 佛州联邦参议员席位挑战者、现任州长斯科特（Rick Scott）与民主党籍现任参议员纳尔逊（Bill Nelson）平均值落后2.9%。
 <br/>
 <br/>
 州长选举，目前共和党33位州长中26名需要重选；民主党16位州长中9名需要重选。佛州州长最受关注，目前民主党人吉卢姆（Andrew Gillum）平均领先共和党联邦众议员德桑蒂斯（Ron Desantis）3.2%。乔治亚州长参选人、共和党人肯普（Brian Kemp）平均领先民主党大力扶持的艾布兰（Stacey Abrams）2.8%。
 <br/>
 <br/>
 〝美国企业研究所〞高级研究员Karlyn Bowman：〝最重要（影响选情）的因素是投票率，看看出来投票的人，是否民主党选民会比共和党的更热情、更愿意投票。〞
 <br/>
 <br/>
 美国民主研究所主任Patrick Basham：〝让选民出来投票的活动很关键。另一个超越其他的因素是川普本身，多大程度上选民会支持（他）来改变命运；或者多少人会说我不喜欢这个人、一定的去投票反对他的参选人。〞
 <br/>
 <br/>
 国会众议院，目前共和党占有235席，民主党193席。民主党要夺回众院控制权，必须至少从共和党手中赢得23席。根据Fox选情分析，目前选情胶着的有29个选区，加州5个选区（国会第10/25/39/45/48区）格外值得关注。川普今日推文背书了加州联邦众议员罗拉巴克（Dana Rorabacher）和纽约州联邦众议员谭妮（Claudia Tenney） 。
 <br/>
 <br/>
 目前多数民调显示，民主党可能夺回众院控制权，其平均值领先7.3%；而民调公司〝拉斯穆森〞（Rasmussen Reports）5日报告众议院选情，共和党领先1%（误差范围正、负2%）。
 <br/>
 <br/>
 美国民主研究所主任Patrick Basham：〝总的来说我觉得不要太依赖民调，因为它们有一些错误和2016年时差不多。不过最近一两天的民调中显示两党选情胶着，我觉得最后这些会被证明更准确一些。〞
 <br/>
 <br/>
 本次中期选举，从前期投票率来看，选民热情度颇高，川普和经济是推动选民投票的主要因素。民主党主打健保、移民牌，共和党则有经济繁荣、减税及稳固边境安全的绩效做依靠。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
