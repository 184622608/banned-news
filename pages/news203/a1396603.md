### 花数百万买到赝品 美圣经博物馆撤下5件死海古卷残片
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 华盛顿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣经博物馆.html" target="_blank">
  圣经博物馆
 </a>
 去年11月盛大开幕，创办人耗巨资买下的古董，却有5件经鉴定后确认是假货。馆方周一（22日）宣布以后将不再于博物馆内展示。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华盛顿特区.html" target="_blank">
  华盛顿特区
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣经博物馆.html" target="_blank">
  圣经博物馆
 </a>
 耗资5亿美元打造，于去年11月开业，其中收藏了博物馆创办人兼富商格林（Steve Green）购入的16块
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_死海古卷残片.html" target="_blank">
  死海古卷残片
 </a>
 （Deas Sea Scrolls）。这些残片被认为是最古老的旧约圣经抄本，真迹大多在以色列展示。但在开馆前，就有人质疑并非真品。
 <br/>
 <br/>
 于是，馆方将5件死海古卷残片送到德国一个材料分析研究所。学者利用3D智慧显微镜进行测试，并对纸莎草上的墨水和沉积物进行材料分析。学者们推测，伪造者会在古老的纸莎草或皮革碎片上书写，使古卷残片看起来像真的，直到经过墨水测试才能分辨真伪。
 <br/>
 <br/>
 德国学者对这些残片进行测试后，发现其中5件显示出与古代来源不相符的特征。馆方最近收到报告后，进一步引起对5件残片真实性的怀疑，因此将不再于博物馆展出。
 <br/>
 <br/>
 博物馆说，在其总共16个卷轴中，还有2个是与上述5个是同批次的货，所以也不再展出，另外9个也将进行额外的测试鉴定。
 <br/>
 <br/>
 博物馆首席策展人柯罗哈（Jeffrey Kloha）表示，〝虽然我们希望测试能有不同的结果，但这是一个教育大众关于验证稀有圣经文物真实性、详细的测试过程及我们对透明度的承诺的机会。〞
 <br/>
 <br/>
 其实，去年加拿大西三一大学死海古卷专家戴维斯（Kipp Davis）就是曾试着警告包括格林家族在内的基督徒关于残片伪造品。
 <br/>
 <br/>
 他认为，受到信仰激励而收集文物的福音派人士或其他人应该加倍谨慎，古董商会趁机占他们的便宜。戴维斯认为，博物馆收藏的16件残片中，只有2～4件可能是真品。
 <br/>
 <br/>
 《独立报》称，格林可能为购入这16块死海古卷残片花费约数百万美元。
 <br/>
 <br/>
 这座距离国家广场（National Mall）2个街区的博物馆高8层楼，陈列着全球最大的〝犹太妥拉经卷〞（Torah）、打造圣经诺亚方舟故事（Noah's flood）场景，并重现耶稣时代中东村落景象。
 <br/>
 <br/>
 但就在博物馆开张的几个月前，格林旗下的Hobby Lobby公司曾因非法进口超过5500件文物，包括自伊拉克走私的楔形文字泥板，引发外界抨击。
 <br/>
 <br/>
 为免遭追究责任，Hobby Lobby同意放弃这些文物，并支付300万美元和解金。
 <br/>
 <br/>
 （记者李红报导）
