### 两地四天活动纪念老布什 含两场追悼会
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】美国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 于12月1日在休斯顿去世，享年94岁。川普（特朗普）总统下令12月5日为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全国哀悼日.html" target="_blank">
  全国哀悼日
 </a>
 ，并举行一连四天的悼念活动。
 <br/>
 <br/>
 周一，美国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 的灵柩在21响礼炮声中，被抬上空军一号，并于下午送抵美国首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华盛顿DC.html" target="_blank">
  华盛顿DC
 </a>
 ，供美国政界及民众缅怀和送行。老布什长子、前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小布什.html" target="_blank">
  小布什
 </a>
 及夫人劳拉，还有其他家人也都随飞机一道抵达。
 <br/>
 <br/>
 包括周一，美国政府和他的家人将为前总统老布什举行一连四天的悼念活动：
 <br/>
 <br/>
 周一下午，老布什的遗体送抵美国国会大厦的圆形大厅（United States Capitol rotunda）安放，一直到周三早晨。
 <br/>
 <br/>
 周二全天，圆形大厅将向公众开放，以让人们缅怀老布什总统。
 <br/>
 <br/>
 周三上午11:00，在华盛顿国家大教堂（National Cathedral in Washington, DC）的国葬
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_追悼会.html" target="_blank">
  追悼会
 </a>
 后，老布什的遗体将被送回德州。
 <br/>
 <br/>
 周四上午11:00，德州圣马丁圣公会教堂（St. Martin's Episcopal Church in Houston）将举行第二场追悼会，而后老布什将被安葬在德州卡城（College Station）的老布什总统图书馆（George H.W. Bush Presidential Library）。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
