### 习近平再遇尴尬事 川普顾问揭穿中共谈判招术
------------------------

<div class="wysiwyg">
 【新唐人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 时间2018年11月14日讯】近期中美高层恢复接触，为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞铺路之际，习近平再遭遇尴尬事。川普经济顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 日前在一场演讲中，批评中共的策略就是谈判，且从不遵守协议，即使中美达成贸易协议，美国也很难相信北京会认真执行。
 <br/>
 <br/>
 《华尔街日报》12日报导称，在中美两国领导人，定于11月底G20峰会期间会晤之际，美国财政部长努钦（Steven Mnuchin），已经恢复与中方贸易谈判负责人、副总理刘鹤的对话，讨论一个将缓解双方贸易紧张关系的协议。
 <br/>
 <br/>
 一些对中共持强硬立场的美国官员说，在美中两国领导人会晤前，中方将拿出一项提议。
 <br/>
 <br/>
 他们认为，美中两国最多可能会就贸易争端达成某种休战协议，美国会暂停对中国产品加征关税。而习近平已要求各部委提出对谈判的具体期望，以便双方能够在G20峰会后继续谈判。
 <br/>
 <br/>
 中共国家副主席王歧山，日前在新加坡出席一次国际会议时也表示，中方准备好，寻求双方都能接受的贸易解决方案。
 <br/>
 <br/>
 中共海外党媒爆料，为避免刺激美国，近日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 下达数条指令，包括禁止对川普（特朗普）搞〝人身攻击式〞；贸易战必须等商务部回应后才能报导等。旨在为〝习川会〞铺路。
 <br/>
 <br/>
 然而，在〝习川会〞之前，习近平再遇尴尬事。川普经济顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 （Peter Navarro），9日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华盛顿.html" target="_blank">
  华盛顿
 </a>
 的一次演讲中，对中共严厉批评且不肯相信北京会认真执行双方达成的任何协议。
 <br/>
 <br/>
 纳瓦罗说，美国和许多国家都有贸易纠纷和相关谈判，但和北京谈判即使有结果，也很难保证得到执行。他举例说，中共加入世贸组织时的多项承诺，在16年过渡期后却都没有实行。
 <br/>
 <br/>
 他说，中共模式和前苏联的模式看似不同，实际上都是由国家掌控的经济，只是中共模式多了些市场和资本主义因素。他说，中共的经济获益于窃取他国的技术。
 <br/>
 <br/>
 纳瓦罗认为，最大的问题是信任问题，当美国提出一长串结构性的问题时，中共则予以否认，〝如果他们连你的担忧都不承认，你怎么和他打交道？那是爱丽丝在漫游奇境。〞
 <br/>
 <br/>
 纳瓦罗还批评中共的策略就是谈判，也是他们的擅长的游戏。他说，始于布什政府时期的美中对话就是中听不中用。
 <br/>
 <br/>
 他举例说，当时一个中共高官在华盛顿向美国作出两项承诺：一是南海人工岛礁非军事化；二是不会再对美国企业进行电脑黑客攻击。结果，两年后中共在南海岛屿上武装到牙齿，而由政府支持的对美国企业的电脑攻击，只停了6个月。
 <br/>
 <br/>
 除纳瓦罗之外，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 出访亚洲之前，于9日在《华盛顿邮报》撰文，批评有的国家（中共）在国内打压人民，对外侵犯邻国主权，美国不会容许印太地区出现这种威权及侵略行为。
 <br/>
 <br/>
 美国之音报导说，战略与国际研究中心的资深中国问题专家张克斯（Christopher K. Johnson）说，彭斯文章中有一个引人注目的信息，就是美国将会坚决反对区内的威权主义、侵略，无疑就是针对中共。
 <br/>
 <br/>
 张克斯说：〝美中贸易争端吸引了很大的注意力，我们有时候好像忘记战略关系其实也有很严重的恶化。〞他说，美国政府和商业界的一个共识是不存在两国首脑握手就能解决的事。
 <br/>
 <br/>
 CSIS的贸易问题专家威廉・莱茵施也认为，美中两国贸易存在大量细节问题，不能指望一次会晤和一场晚宴就能达成有意义的协议。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
