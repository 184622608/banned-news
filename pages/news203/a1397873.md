### 中期选举倒数四天 众议院关键战盘点
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 还有四天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统及两党代表人物都在积极拉票。而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 被普遍认为是本次选举的主战场，下面来了解其中的关键之战。
 <br/>
 <br/>
 国会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 435个席位今年全部改选，目前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 占有235席，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 193席。民主党要夺回众院控制权，必须至少从共和党手中赢得23席；其中1／3席位在加州。民主党为这7个国会众议员席位已经投入了6-7千万。
 <br/>
 <br/>
 加州共和党全国委员会主席Shawn Steel：〝现在我认为他们肯定能赢的只有一席，另外三场竞选则难以预料。我看见的每一个存在竞争的席位、对共和党来说比较悬的席位，都只有正负3%的差距。共和党有大概40%的几率保住众议院多数。民主党，如果他们赢，也不会赢很多。所以他们不会得到很多权力。〞
 <br/>
 <br/>
 美东蓝州纽约，国会第19选区现任共和党众议员法索（John Faso）与民主党挑战者德尔加多（ Antonio Delgado）的竞争也相当激烈，10月下旬民调法索落后5%，10月中旬民调法索领先1%。法索主张废除奥巴马健保但反对税改，因为担心会导致纽约州提高州税和地方税率。
 <br/>
 <br/>
 西弗吉尼亚州国会第三选区共和党众议员米勒（Carol Miller）面对州参议员、民主党人奥杰达（Richard Ojeda）的挑战。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 儿媳Lara Trump周四前往西弗州支持米勒等共和党参选人。这个铁锈带（rust belt）选区的结果，可以测试出这个摇摆选区在2016大选中投票给川普后，两年来的民意是否有变化。目前不同民调结果显示，共和党米勒领先。
 <br/>
 <br/>
 全美20个国会摇摆选区的战况，对谁能主导众议院有关键影响。
 <br/>
 <br/>
 美国民主研究所主任Patrick Basham：〝如果共和党在这次选举中生存下来、重新控制国会，川普第一任期内后两年的执政可以和，像有些人说的，可以和他前两年执政一样成功。如果民主党在下周选举中胜利赢得众议院控制权， 他们会把逆转川普政策作为重大议程去推进，我可以预见如果届时共和党和川普试图反对他们的任何议程，他们会有怎样的激烈表现。〞
 <br/>
 <br/>
 周五晚，川普在副总统彭斯及篮球教练耐特（Bobby Knight）陪同下在印第安纳州助选；下午川普在西弗吉尼亚州的亨廷顿举行MAGA（Make America Great Again）集会。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
