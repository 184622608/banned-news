### 川普宾州演说 呼吁支持共和党爱国者
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】周三，美国总统川普在宾州和数千名民众集会，并发表演说，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 议员中期选举助力。
 <br/>
 <br/>
 来自宾州和附近支持川普的民众，热情迎接川普的到来。
 <br/>
 <br/>
 川普首先向遭受〝迈克尔〞飓风袭击的佛州民众表示慰问，并表示行政部门和佛州及北卡州州长密切合作，为灾区提供支援。
 <br/>
 <br/>
 美国总统川普：〝我们有充足存水和食物，我们有电力公司和成千上万的电力工人。〞
 <br/>
 <br/>
 川普还提到刚刚就职的最高法院大法官卡瓦诺。
 <br/>
 <br/>
 美国总统川普：〝卡瓦诺现在和尼尔・戈萨奇坐在一起，保卫你们的权利，保卫你们的宪法，以及你们的天赋自由。〞
 <br/>
 <br/>
 川普邀请
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 议员巴勒塔和伊利地区国会议员迈克・凯利上台，为两人中期选举助阵。
 <br/>
 <br/>
 美国总统川普：〝这是一个你们要投票给他的了不起的人，我邀请他上来，迈克。〞
 <br/>
 <br/>
 美国总统川普：〝如果你想拯救美国生活，拯救美国就业，就像现在一样，使经济情况比历史上任何时期都好，你需要投票支持这名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱国者.html" target="_blank">
  爱国者
 </a>
 巴勒塔。〞
 <br/>
 <br/>
 川普还概述了政府取得的一系列成就，包括削减税收、促使宾州煤炭行业复兴、重新谈判北美自由贸易协定以及遏制非法移民等。
 <br/>
 <br/>
 美国总统川普：〝我们创造了420多万新就业，使400多万人摆脱了（粮食券）救济，〞
 <br/>
 <br/>
 据悉，川普将很快前往佛州灾区。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
