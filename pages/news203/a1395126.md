### 迈克尔飓风过后满目疮痍 至少11人死
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔飓风.html" target="_blank">
  迈克尔飓风
 </a>
 周三横扫美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛罗里达州.html" target="_blank">
  佛罗里达州
 </a>
 狭长地带，暴雨伴随每小时250公里的狂风，所经之处，满目疮痍。目前，迈克尔飓风至少造成7人死亡。
 <br/>
 <br/>
 飓风扫过，墨西哥海滩小镇因为靠近风眼，受灾最重。
 <br/>
 <br/>
 周四，佛州州长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯科特.html" target="_blank">
  斯科特
 </a>
 乘坐直升机，查看灾情。
 <br/>
 <br/>
 从高空望去，很多房屋屋顶掀翻，有的整个被摧毁、夷为平地，树木吹倒，受灾区域犹如经历了一场战争，满目疮痍。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛罗里达州.html" target="_blank">
  佛罗里达州
 </a>
 州长 斯科特:〝特别是对墨西哥海滩的破坏，是灾难性的。向上帝祈祷，让所有人都活下来。〞
 <br/>
 <br/>
 斯科特慰问投入救灾的国民警卫队，称赞他们英勇无私。
 <br/>
 <br/>
 佛罗里达州州长 斯科特:〝这是国民警卫队的所为，当他人处于危险之中（时所做的），他的母亲现在就在医院里。〞
 <br/>
 <br/>
 电力公司已开始全力恢复电力。
 <br/>
 <br/>
 不过，很多道路被飓风毁坏，路面断裂成碎块，凹凸不平，给救援工作带来很大困难。
 <br/>
 <br/>
 阿拉巴马州的海湾城市莫比尔也受灾严重，救援人员用直升机救助受困居民。
 <br/>
 <br/>
 据报，目前至少11人丧生。川普总统形容，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔飓风.html" target="_blank">
  迈克尔飓风
 </a>
 犹如致命子弹，杀伤力强。
 <br/>
 <br/>
 美国总统 川普：〝有一点我们可以确切的说，就是迈克尔的速度非常快，象子弹飞过，是致命的子弹，力量非常大。风大的史上少见，我们从来没见过。〞
 <br/>
 <br/>
 〝我们为失去生命的人祈祷，也为他们的家人祈祷。我们也心系损失财产的数千民众，很多被飓风彻底摧毁。〞
 <br/>
 <br/>
 他还表示，救援工作已全面展开，并称赞他们表现出色。
 <br/>
 <br/>
 飓风迈克尔已转为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热带气旋.html" target="_blank">
  热带气旋
 </a>
 ，向乔治亚州和南北卡罗莱纳州方向推进。
 <br/>
 <br/>
 新唐人记者安心、天羽综合报导
</div>
