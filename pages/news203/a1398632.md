### 美85岁最高法院大法官 摔断3根肋骨送医
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】85岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国最高法院大法官.html" target="_blank">
  美国最高法院大法官
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金斯伯格.html" target="_blank">
  金斯伯格
 </a>
 周三（7日）不慎跌倒，跌断3根肋骨，周四（11月8日）早上送医。
 <br/>
 <br/>
 根据法院当地时间周四公布的消息，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金斯伯格.html" target="_blank">
  金斯伯格
 </a>
 （Ruth Bader Ginsburg）周三在办公室摔了一跤，之后她回家了，但晚上感觉不适，于周四早上去了乔治华盛顿大学医院就诊。声明中说，检查结果显示，她左侧3根肋骨骨折，送医后进行观察和治疗。
 <br/>
 <br/>
 1993年美国最高法院怀特大法官（ByronR.White）退休后，当时的克林顿总统提名金斯伯格担任大法官。她是美国历史上第二位女性、第一位犹太人女性最高法院大法官。属于自由派。首位女性大法官是桑德拉・戴・奥康纳。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国最高法院大法官.html" target="_blank">
  美国最高法院大法官
 </a>
 是终身制，若有大法官离世或自行退休，便由现任总统提名继任人，故此外界高度关注各大法官的健康状况，以及他们是否有退休意向。
 <br/>
 <br/>
 金斯伯格1999年被查出患有结肠癌。她今年8月曾表示，她会再做5年大法官，到90岁退休。不过她也透露，2016年如果是民主党候选人希拉里当选总统，自己可能春季就已宣布退休了。
 <br/>
 <br/>
 此前，美国总统川普（特朗普）曾表示希望金斯伯格早日退休，以便他能任命一位新的保守派大法官。
 <br/>
 <br/>
 川普在上任后的近2年内，已经成功提名2名保守派大法官，分别是戈萨奇（Neil Gorsuch）及卡瓦诺（Brett Kavanaugh），他们分别接替离世的大法官斯卡利亚（Antonin Scalia）及自行退休的肯尼迪（Anthony Kennedy）。这也让最高法院稳稳的向右转。此前最高院内部一直存在着保守派和自由派的席位之争。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
