### 美首次引渡中共国安官员 震慑经济间谍
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 司法部宣布，首次将一名中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国安官员.html" target="_blank">
  国安官员
 </a>
 从比利时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_引渡.html" target="_blank">
  引渡
 </a>
 到美国受审。他被指控涉嫌从事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济间谍.html" target="_blank">
  经济间谍
 </a>
 ，窃取美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_航空.html" target="_blank">
  航空
 </a>
 业的商业机密。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 司法部公布的起诉书显示，中共国安部下属的江苏省国安厅六局副处长徐延军，涉嫌从2013年起，以多种伪装身份，接触美国顶尖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_航空.html" target="_blank">
  航空
 </a>
 航天企业的专家，窃取商业机密。他的目标包括为美国民用和军用飞机提供引擎的通用电气航空公司。
 <br/>
 <br/>
 今年4月，徐延军前往比利时，准备与通用航空一名工程师见面。美国联邦调查局和比利时政府配合，将徐延军抓捕，10月9号
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_引渡.html" target="_blank">
  引渡
 </a>
 回美国受审。
 <br/>
 <br/>
 这是美国第一次成功引渡中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济间谍.html" target="_blank">
  经济间谍
 </a>
 。目前美国与全球109个国家有引渡条约，学者关注本案是否具有示范意义。
 <br/>
 <br/>
 谢田：中共间谍可能以后会慢慢发现，只要踏出国门的话，如果这个国家跟美国有引渡条约的话，他们就有真实的危险。这样中共间谍活动就有很大的局限。对中共是一个相当大的震慑。
 <br/>
 <br/>
 夏业良：如果美国联合西方的这些国家，形成一致性的防范系统和网络的话，那么中共的间谍人员和线人，以后就难以开展活动。
 <br/>
 <br/>
 徐延军目前被控〝窃取商业秘密〞等4项罪名，如果罪名成立，将面临最高25年监禁和罚款。
 <br/>
 <br/>
 联邦调查局官员普利斯塔普表示，徐延军案暴露出，对美经济间谍直接受中共政府监督管理。
 <br/>
 <br/>
 司法部助理部长德默斯说，美国〝不能容忍一个国家窃取美国的军力和智慧成果，不能容忍有国家不劳而获〞。
 <br/>
 <br/>
 夏业良：国安厅的是〝正牌间谍〞，这样的一个部门现在被直接抓到了，是一个鲜明的信号，就是说美国不会再容忍这样的（经济间谍）行为。这是一个非常好的警示。
 <br/>
 <br/>
 9月25号，还有一名居住在芝加哥的中国公民季超群被捕。起诉书指，季超群曾向江苏省国安厅官员A某，提供了8名美国航空航天专家的个人资料，而A某不晚于今年4月被抓。综合判断，A某就是徐延军。
 <br/>
 <br/>
 谢田：美国（防范）中共盗取科技的网，越收越紧。中共它会发现，它的业余间谍，都会非常危险。
 <br/>
 <br/>
 川普政府已多次批评，中共利用经济间谍、黑客窃密、强制技术转移、收购敏感企业等方式，不公正地获取美国尖端科技，削弱美国的经济和国防实力。美国最近已加强执法，完善立法，力图填补漏洞。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
