### 前白宫美女主任加盟福克斯 出任首席通讯官
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】数月前辞职的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前白宫通讯主任.html" target="_blank">
  前白宫通讯主任
 </a>
 霍普．希克斯（Hope Hicks）日前再度出山，将出任保守派媒体巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福克斯.html" target="_blank">
  福克斯
 </a>
 （Fox ）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首席通讯官.html" target="_blank">
  首席通讯官
 </a>
 兼执行副总裁。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福克斯.html" target="_blank">
  福克斯
 </a>
 官方周一（10月8日）宣布，已聘用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前白宫通讯主任.html" target="_blank">
  前白宫通讯主任
 </a>
 希克斯担任公司的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首席通讯官.html" target="_blank">
  首席通讯官
 </a>
 （chief communications officer , CCO）兼执行副总裁。
 <br/>
 <br/>
 福克斯有限广播公司经过精简整顿，目前已成为一个更富于效率的新公司，包含福克斯新闻（Fox News）、福克斯体育（FOX Sports）、福克斯商业新闻（FOX Business）及其它一些连锁机构。
 <br/>
 <br/>
 周一，希克斯已抵达洛杉矶准备就职。一旦福克斯集团敲定2019年初向娱乐巨头迪士尼公司（Walt Disney Company）出售电影和电视资产，希克斯的任命就将开始生效。
 <br/>
 <br/>
 同时，福克斯还任命了另一名副总裁，即通用电气公司旗下运输业务〝GE Transportation〞的奥布莱恩（Danny O’Brien），他将驻守华盛顿打理福克斯与政府的关系。
 <br/>
 <br/>
 福克斯首席法律和政策官员维特‧狄恩（Viet Dinh）在声明中表示，希克斯和布莱恩是公认的公共事务专业人士及领导者，他们会共同定义并向相关社区投射福克斯的声音。
 <br/>
 <br/>
 白宫新闻发言人桑德斯在得知这一消息后，在推特上祝贺希克斯：
 <br/>
 <br/>
 〝福克斯不会找到比希克斯更聪明或更有才华的人了。真为我的朋友高兴。他们能拥有你简直超级幸运，而东海岸已错过了你。〞
 <br/>
 <br/>

 <br/>
 希克斯今年29岁，当过模特，是跟随川普时间最长、也是川普最忠诚最得力的助手之一。她是川普长女伊万卡（Ivanka Trump）的朋友，早在川普宣布竞选总统之前就开始与其合作， 2015年7月加入川普选战，去年9月起正式担任白宫通讯主任一职。
 <br/>
 <br/>
 希克斯生于有数十年公关经验的家庭，公关能力十分出色。
 <br/>
 <br/>
 不过，2月27日，希克斯就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 问题接受了众议院情报委员会长达9小时的面询，期间，希克斯承认他为川普偶尔撒过〝善意的谎言〞（white lies），但关于通俄门调查，她从未做过任何伪证。
 <br/>
 <br/>
 据福克斯新闻（Fox News）报导，消息人士称，川普当晚告诉希克斯，他认为她关于〝善意的谎言〞之说不是个好主意。不过，川普并没有训斥希克斯。
 <br/>
 <br/>
 但希克斯次日还是决定辞职。她在辞职声明中表示：〝没有足够的言语来表达我对川普总统的感激之情。我希望总统及其政府在继续领导我们的国家时一切顺利。〞
 <br/>
 <br/>
 白宫发言人当天也发表了总统的声明。川普表示，希克斯很出众，在过去的三年里工作非常令人满意，她聪明伶俐、思想丰富，是位真正杰出的人士。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
