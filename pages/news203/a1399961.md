### 佛州计票风波落幕 布县选举官败阵辞职
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】周日（11月18日），依据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 结果，该州参议员和州长选举的战果终被共和党收入囊中。几小时后，挑起计票争端的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布劳沃德县.html" target="_blank">
  布劳沃德县
 </a>
 （Broward）选举监督员斯奈普斯（Brenda Snipes）也递交了辞呈。
 <br/>
 <br/>
 周日，在11月6日选举夜之后仍悬疑了十余天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 参议员选举终于落下帷幕，共和党至此大获全胜，一举拿下参议员和州长两项重要选举，使得佛州这一著名紫州向红色倾斜。
 <br/>
 <br/>
 值得一提的是，当佛州现任州长斯科特（Rick Scott）宣布自己在参议员竞选中获胜、其民主党对手尼尔森（Bill Nelson）也承认自己败选之后，不过几小时的时间，早已声名狼藉的斯奈普斯也知趣地递交了辞呈。
 <br/>
 <br/>
 斯奈普斯早有涉嫌选举舞弊的不良记录。据美国保守派网站〝Gateway Pundit〞11月9日公布的调查记录，她早在本次选举前就曾14次遭到欺诈指控，但却始终没有下台，而且一直在历次选举中制造事端。
 <br/>
 <br/>
 布县是民主党占优势的选区，但斯奈普斯的欺诈行为过于明目张胆。根据斯科特选战11月8日的声明，布县至选举夜共收到634,000 张选票；但至美东时间8日晚8:00，选票竟激增到712,840张。由于突然多出的选票大多流向民主党候选人，导致两名共和党人的优势被巨幅压缩，并最终启动了强制性
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 。
 <br/>
 <br/>
 在重新计票过程中，斯奈普斯仍企图左右选举结果。布县在提交机器重新计票结果时故意迟后2分钟，导致布县的重新计票结果错过了截止时间而被废弃、转而使用此前的计票结果。而布县之所以这样做，是因为该县重新计票结果发现，斯科特又胜出对手1,000余张选票。
 <br/>
 <br/>
 此外，布县还〝弄丢〞了2,000多张选票。
 <br/>
 <br/>
 斯奈普斯从2003年开始担任布县选举监督员，多年来广受争议。一名法官今年5月裁决，该县毁掉了民主党众议员舒尔茨（Debbie Wasserman Schultz）与对手在党内初选中的选票，破坏了法律。之后，佛州政府开始监察斯奈普斯的工作。
 <br/>
 <br/>
 斯奈普斯还在2017年的一次法庭听证中承认，其所在的选举部门曾允许非公民和重犯投票。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
