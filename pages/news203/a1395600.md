### 挑拨白宫？中共大使称不知谁是美方决策者
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】中共驻美国大使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔天凯.html" target="_blank">
  崔天凯
 </a>
 近日在接受美国媒体采访时声称，在贸易问题上不知道美国方面的最终决策者究竟是谁，此言令舆论界一片哗然，引发各种解读，其中有观点指崔意在挑拨白宫内部。随后白宫新闻发言人正面澄清称，总统川普（特朗普）是〝最终决策者〞，莱特希泽是〝首席谈判代表〞，而主导美国贸易政策的是一群〝对贸易政策有深刻理解的人〞。
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福新周日新闻.html" target="_blank">
  福新周日新闻
 </a>
 》本月14日播出了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专访.html" target="_blank">
  专访
 </a>
 中国驻美大使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔天凯.html" target="_blank">
  崔天凯
 </a>
 的节目。崔天凯在那次访谈节目中强调称，美国是发起贸易战的一方，而中方并不希望打这场贸易战，只是被迫〝防守〞的一方。他矢口否认中方曾窃取美国的智慧财产权以及干涉美国内政，他还声称自己对川普政府的决策核心〝感到困惑〞，不知道美国政府的贸易决策到底听谁的。
 <br/>
 <br/>
 福克斯的主持人华莱士对崔天凯进行专访的过程中出现了下面这样一段对话：
 <br/>
 <br/>
 华莱士问：〝你清楚川普总统在贸易问题上听谁的意见吗？是白宫首席经济顾问库德洛，财长姆努钦这样的温和派，还是白宫国家贸易委员会主任纳瓦罗这样的强硬派？〞
 <br/>
 <br/>
 崔天凯回答说：〝这得你告诉我。〞
 <br/>
 <br/>
 华莱士对崔的上述说法表示很诧异，他反问：〝你不清楚？我认为，向北京报告美国总统听信谁的话，这显然是你作为中国大使的分内事。〞
 <br/>
 <br/>
 崔天凯于是辩解称，他和其他国家驻美的大使们一样找不到头绪，并不知道美国贸易政策的〝最终决策者是谁〞。他还说：〝当然，总统应该是最终拿主意的。但在决策中，谁发挥了作用、发挥了什么样的作用？这个问题有时候很令人困惑。〞
 <br/>
 <br/>
 这档节目播出后，多家中国大陆媒体和海外西方主流媒体均在相关报导中突出了崔天凯的上述说辞，并纷纷从不同的角度进行分析解读。
 <br/>
 <br/>
 有美国媒体随即发文分析了川普上台以来，在处理贸易冲突的过程中，不同阶段发挥主要作用的究竟是哪几位政要人物；更有舆论认为崔天凯的这番说辞，疑似暗藏挑拨白宫内部持不同政见者关系的不良用心。
 <br/>
 <br/>
 美国〝政客〞新闻网站（Politico）曾发文分析称，通常情况下，美国贸易代表办公室在与其他国家的贸易谈判中处于领导地位。但在川普政府的早期，在美国贸易代表莱特希泽上任之前，美国财政部长姆努钦和商务部长罗斯负责对外的经济对话。尽管后来有了莱特希泽，但他似乎也只是起了支持作用。
 <br/>
 <br/>
 随后，白宫新闻发言人桑德斯正面对外澄清称，美国的贸易政策方面，〝总统是最终的决策者〞，莱特希泽是〝首席谈判代表〞，美国政府还有一群〝对贸易政策有深刻理解的人〞 组成的团队，他们都是整个团队的一部分。
 <br/>
 <br/>
 外界还注意到，崔天凯在那次专访中，对今年11月底川普可能与中国国家主席习近平会面的预期持明显乐观的态度。他声称川普和习近平两人有〝良好的相互理解与工作关系〞，又表态说自己也希望并确信这种良好的关系可以〝持续〞。
 <br/>
 <br/>
 不过，华莱士在专访崔天凯之前还访问过白宫的经济顾问库德洛，而这位白宫政要对中美两国关系现状与贸易战的走向显然没有这么乐观。
 <br/>
 <br/>
 当时华莱士问库德洛：〝鉴于美中两国在贸易、军事、和政治领域的一系列冲突，两国关系是否已经进入了所谓的新冷战时期？〞
 <br/>
 <br/>
 库德洛则回答说：这个问题应该留给历史学家来判断。不过，在经济层面，美国政府现在对中共政权开出的条件〝相当不满意〞，美国不能容忍中共继续盗窃美国的智慧财产权，或强迫美国公司转让技术。
 <br/>
 <br/>
 库德洛还举出例证说：就在最近，中共官方还公布了一项新的规定，允许中共的〝执法人员〞可以进入在华美国公司的网络去窃取技术。
 <br/>
 <br/>
 对于川普会否在今年11月底的G20峰会上与习近平举行双边会晤，以及他们二人能否为两国的贸易冲突找到新转机的问题，库德洛并未给出肯定的答覆，他仅表示两人有可能见面，但他同时强调称，美中近来的关系并没有朝向〝正面发展〞。
 <br/>
 <br/>
 (记者唐迪综合报导/责任编辑：程以仁）
