### 美中期选举倒数8天 最胶着选战一览
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】距离美国中期选举，只剩8天，下面带您来了解几个关键选区的选情。
 <br/>
 <br/>
 川普今早推文，〝佛州人要做出选择，要么是哈佛／耶鲁毕业的德桑蒂斯，一位出色的国会众议员、也将会成为一位出色的州长；或者是那
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 人，一个小偷和把塔拉赫西市搞得一团糟的那个市长，该市据说是全美最腐败的城市之一。〞
 <br/>
 <br/>
 佛州州长候选人，民主党人吉卢姆（Andrew Gillum）与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 联邦众议员德桑蒂斯（Ron Desantis）的选情正发生变化，民调越来越有利共和党人。
 <br/>
 <br/>
 类似状态也体现于该州民主党联邦参议员纳尔逊（Bill Nelson）与共和党挑战者、现任州长斯科特（Rick Scott）之间的竞争。
 <br/>
 <br/>
 目前佛州的投票情况，共和党选民超过民主党。民主党不断加大对佛州的投入；川普总统则会在本周两次前往佛州助选。
 <br/>
 <br/>
 另一场竞争激烈的联邦参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选战.html" target="_blank">
  选战
 </a>
 在德州，昆尼派克大学（Quinnipiac University Poll）周一民调显示，共和党参议员克鲁兹领先挑战者、民主党人奥罗克( Beto O'Rourke) 5个百分点（51-46）。
 <br/>
 <br/>
 德州联邦参议员 克鲁兹：〝我们看到德州超高的投票率，这是个好兆头。〞
 <br/>
 <br/>
 根据州务卿办公室数据，德州今年的先期投票率，截至目前，已超过2014年中期选举；前5个大县的投票率几乎都翻倍。
 <br/>
 <br/>
 目前被认为可能最难分胜负的州长选举是威斯康辛州，现任共和党州长沃克（Scott Walker）寻求三连任，其支持度在上周总统川普亲临助选后，缩小了与民主党挑战者埃弗斯（Tony Evers）的距离，从9月底10%缩小到目前的3.6%。
 <br/>
 <br/>
 而目前最胶着的国会众议员竞选，要算弗吉尼亚布拉特（Dave Brat）与民主党人斯潘伯格（Abigail Spanberger）的较量。
 <br/>
 <br/>
 两位现任国会众议员，共和党人罗斯福斯（Keith Rothfus）与民主党的蓝姆
 <br/>
 <br/>
 新唐人记者李兰综合报导
