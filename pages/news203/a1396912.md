### 中共间谍徐燕军被控5罪 两辩护律师背景惹关注
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】日前从比利时引渡到美国的中共情报官员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐燕军.html" target="_blank">
  徐燕军
 </a>
 （Yanjun Xu，音译），被指盗窃美国航空技术，遭美方起诉五项罪名。英媒还披露，为其辩护的律师团队中，竟包括两位美国前联邦检察官，目前不知是何人为其聘请。
 <br/>
 <br/>
 BBC引述美国法庭文件报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐燕军.html" target="_blank">
  徐燕军
 </a>
 被控共五项罪名，包括两项经济间谍罪、两项偷窃贸易机密和一项刑事没收（criminal forfeitures）罪名。
 <br/>
 <br/>
 报导还披露，为徐燕军辩护的三名美国律师中，竟包括两位经验丰富的美国前联邦检察官，他们曾经主管国家安全、反恐和刑事案的工作。
 <br/>
 <br/>
 报道指，这两名律师均来自塔夫特律师事务所（Taft Stettinius &amp; Hollister）。
 <br/>
 <br/>
 其中，罗伯特・麦克布莱德（Robert McBride）曾任美国联邦检察官，属于司法部前雇员。根据该律师事务所资料，麦克布莱德曾在北肯塔基州东司法辖区担任美国联邦助理律师长达15年，还曾任该区的刑事案件主管、国家安全检察官和反恐顾问委员会的协调员。
 <br/>
 <br/>
 另一位律师罗夫・科纳（Ralph Kohnen），亦曾是联邦检察官和刑事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辩护律师.html" target="_blank">
  辩护律师
 </a>
 ，在2001年更升任美国联邦检察官在俄亥俄州辛辛那提市办公室的副刑事主管。他还多次获得联邦调查局（FBI）、美国财政部颁发的奖项和表彰。
 <br/>
 <br/>
 报道指，美国前联邦检察官加入私营律所的例子并不少见，但此案涉及中共间谍，案情敏感，因此特别引人关注。
 <br/>
 <br/>
 报导说，目前尚不清楚是哪个机构或个人代表徐燕军聘请了该律师团。
 <br/>
 <br/>
 美国检方的起诉书指，徐燕军是中共国安部下属江苏国安厅六局副处长，曾使用化名，以〝江苏省科学技术协会〞官员的身份，自去年3月起多次诱导和收买美国通用电气航空(GE Aviation)一名华裔工程师为其提供公司机密。
 <br/>
 <br/>
 今年4月，徐燕军在比利时被美国FBI诱捕，10月初引渡至美国受审。这是美国首次引渡中共间谍。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
