### 美经济强劲  库德洛：小企业明年增长或超3%
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，经济与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健保.html" target="_blank">
  健保
 </a>
 依然是民众最关注的议题。白宫经济顾问认为，目前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 强劲，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小企业.html" target="_blank">
  小企业
 </a>
 将会有更高的增长率。
 <br/>
 <br/>
 记者：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小企业.html" target="_blank">
  小企业
 </a>
 能否期待明年有3%的增长？〞
 <br/>
 <br/>
 白宫首席经济顾问 Larry Kudlow：〝是的，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 预测。可能会更高。〞
 <br/>
 <br/>
 目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 持续表现强劲，市场信心走强，包括蓝领在内、就业机会普遍增多，小企业也在成长。
 <br/>
 <br/>
 白宫首席经济顾问 Larry Kudlow：〝2018年的最大事件，就是(美国)经济迈向繁荣。之前几乎所有人都说这不可能。〞
 <br/>
 <br/>
 今年，美国前三季度经济成长率分别为2.2%, 4.2%和3.5% 。不过，民主党依然批评说，税改并未让中低阶层受益。
 <br/>
 <br/>
 美国维州民主党联邦众议员 Don Beyer：〝川普减税的问题是，83%甚至更多获利，都给了最富的1%人口，绝大多数美国人并未受益。〞
 <br/>
 <br/>
 白宫首席经济顾问 Larry Kudlow：〝蓝领工人的薪资成长高过白领阶层。不是说其薪资高过白领，而是薪资增长速度更高。〞
 <br/>
 <br/>
 根据劳工部最新数据，2017年9月到今年9月，美国工人工资增长2.9％，通胀调整后，工资全年增长0.6％，成为2016年以来的最大增幅。
 <br/>
 <br/>
 美国商会调查也显示，小企业信心指数创下30年最高记录，税收负担不再是企业的主要担懮，相反，劳动力紧张已成为业主面临的首要问题。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健保.html" target="_blank">
  健保
 </a>
 也是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 两党针锋相对的议题。
 <br/>
 <br/>
 美国维州民主党州参议员 Jennifer Wexton：〝他们令债务爆增，现在他们又要削减社会安全福利和医疗补助。〞
 <br/>
 <br/>
 白宫首席经济顾问 Larry Kudlow：〝不，我们没有计划触动主要的福利系统，但确实有计划取消奥巴马健保的剩余部份，并提供更加以市场为导向、激励型的健保。〞
 <br/>
 <br/>
 针对美中贸易战，库德洛认为，这对美国经济影响很小。事实已证明，没有中国，美国经济依然可以成长良好。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
