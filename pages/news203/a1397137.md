### 调查〝活摘〞12年 麦塔斯：恶者须绳之以法
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纪录片.html" target="_blank">
  纪录片
 </a>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活摘.html" target="_blank">
  活摘
 </a>
 》，揭露了正在中国大陆发生的反人类罪行：活摘器官交易。26日，在美国阿拉巴马州举办的一场放映会上，加拿大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权律师.html" target="_blank">
  人权律师
 </a>
 大卫・麦塔斯表示，参与这个罪恶的人必须绳之以法。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纪录片.html" target="_blank">
  纪录片
 </a>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活摘.html" target="_blank">
  活摘
 </a>
 》（Human Harvest）在阿拉巴马州莫比尔市的春山学院放映。
 <br/>
 <br/>
 影片介绍了获得诺贝尔和平奖提名的大卫・乔高（David Kilgour）和大卫・麦塔斯（David Matas），努力调查和揭露发生在中国的活摘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 器官的罪恶。
 <br/>
 <br/>
 观看纪录片《活摘》学生 Jarvis Williams：〝我听了真的很震惊。他的故事非常有力，让我警觉起来。不能让这样的事情在世界上发生。〞
 <br/>
 <br/>
 加拿大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权律师.html" target="_blank">
  人权律师
 </a>
 大卫・麦塔斯表示，他们在这件事上的努力，已有12个年头了。现在世界上许多组织和他们一起采取行动，来揭露和制止罪恶，但是他认为这还远远不够。
 <br/>
 <br/>
 加拿大人权律师 David Matas：〝中国的这一切必须被透明化，在中国犯下活摘器官罪行的人必须绳之以法，同时，中共应配合国际上对活摘器官的调查。还有许多的改变需要去做。〞
 <br/>
 <br/>
 到会的学生们对活摘真相感到非常震惊，并希望采取行动。
 <br/>
 <br/>
 观看纪录片《活摘》学生 Isabelle Delplanche：〝我为此非常的受触动，我一时还没办法理解到底发生了什么。这些人们都遭受了什么？老实说，我有点喘不过气来。我觉得很难过，一时还很难平静，我想多做些什么来帮助他们。〞
 <br/>
 <br/>
 学生们认为这件事非常急迫，得呼吁更多人关注。
 <br/>
 <br/>
 观看纪录片《活摘》学生 Hannah Fox：〝尽可能让更多的人知道这件事，因为我以前从来没有听说过，我相信还有很多人还没有听说过。如果能够让更多人知道会很好，传递这个信息，让更多的人知道正在发生的事情。〞
 <br/>
 <br/>
 新唐人记者王琼、岳磊阿拉巴马州莫比尔报导
</div>
