### 美媒称白宫开始起草谈判条款  川普：习近平很想达协议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】有消息指美国总统川普（特朗普）周三与中国国家主席习近平通话后，已要求内阁开始草拟可能的中美协议条款。当天，川普在一个竞选集会上表示，北京方面很想和美国达成一项协议，习近平本人也很想这样做。
 <br/>
 <br/>
 《彭博社》2日援引四名消息人士爆料称，川普1日与习近平通电话后，已经要求美国内阁的重要官员开始起草可能的贸易协议条款。估计中美双方有意于本月末在阿根廷举行的20国集团峰会(G20)上，通过两位元首的会谈，就解决双边贸易争端达成某种共识。
 <br/>
 <br/>
 未具名的消息人士称，有多个部门参与了草拟贸易协议文本的工作。
 <br/>
 <br/>
 川普本人则也在Twitter 上发帖称，自己与习近平进行了〝非常良好〞的交谈，着重讨论贸易问题，谈话进展得〝很顺利〞。
 <br/>
 <br/>
 就在周四当天，川普在密苏里州出席一次竞选集会时也提到了中美贸易战的问题，他说：〝他们（北京当局）很想和我们达成一项协议。 习近平很想这样做，他们所有的人都想这么做。〞
 <br/>
 <br/>
 对于川习周四的通话，中美双方的媒体均有报导。其中有媒体的报导说，两位领导人除了讨论如何解决贸易冲突的问题外，还就朝鲜弃核的问题交换了意见。中共官媒则报导称，川普在通话中建议双方〝经常展开直接对话〞，并为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间的会晤〝进行积极筹备〞。
 <br/>
 <br/>
 G20峰会将于今年11月30日至12月1日在阿根廷首都布宜诺斯艾利斯举行。
 <br/>
 <br/>
 受到上述消息的刺激，人民币急弹逾200点；离岸人民币汇(CNH)价一度增长了0.31%，现报6.9017；在岸人民币(CNY)现报6.8934。此前受贸易战阴影的影响，人民币汇率在岸价曾跌回6.93的水平。
 <br/>
 <br/>
 与此同时，香港恒生指数递增了四个百分点，是2015年以来增幅最大的一次；上交指数也攀升了两个百分点。
 <br/>
 <br/>
 不过，《华盛顿时报》11月2日却报导称，川普前一天接受该报独家专访时表示，尽管中美两国政府在贸易问题上〝有进展〞，但他他并不预计两国会在本月底的20国集团元首峰会期间宣布达成贸易协议。
 <br/>
 <br/>
 川普说：〝最终我们会跟中国达成一个非常合理的协议。我可以告诉你，他们希望跟我们达成协议。〞
 <br/>
 <br/>
 在过去的几个月间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易冲突.html" target="_blank">
  中美贸易冲突
 </a>
 不断升级，美国政府已经宣布对价值2500亿美元的中国输美产品征收惩罚性关税，不久前川普还警告北京，如果中美仍然无法达成令美方满意的协议，美国将对剩下所有的中国输美产品征税。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
