### 老布什遗体送抵华盛顿 川普5日出席葬礼
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】美国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 上周在德州休斯敦的家中逝世，他的遗体周一（3日）运到埃林顿机场，并由短暂改名为〝41号空中特别任务〞的空军一号运送到华盛顿。总统川普（特朗普）5日将出席葬礼。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 (George Herbert Walker Bush)在当地时间11月30日深夜去世，享寿94岁。周一在埃林顿机场，老布什的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灵柩.html" target="_blank">
  灵柩
 </a>
 在21响礼炮声与军乐队演奏中，被抬上空军一号，超过100人在跑道上送别老布什。
 <br/>
 <br/>
 因为老布什是第41任总统，所以空军一号暂时改名为〝41号空中特别任务〞，老布什的长子、另一位前总统乔治布什和夫人劳拉，以及其他布什家族成员及好友也登机，一同前往华盛顿。
 <br/>
 <br/>
 老布什灵柩周一下午送抵华盛顿DC，供美国政界及民众缅怀和送行。当天川普和第一夫人梅拉尼娅当天前往美国国会大厦圆形大厅瞻仰老布什灵柩，并敬礼，向已故前总统表达敬意。
 <br/>
 <br/>
 丧礼将于5日在华府国家大教堂(Washington National Cathedral)举行，成为第4位在大教堂举行葬礼的美国总统。
 <br/>
 <br/>
 灵柩周四（6日）运返家乡，下葬在老布什过去数十年祈祷的地方、圣马丁圣公会教堂（St Martin's Episcopal Church）。
 <br/>
 <br/>
 老布什遗愿表明要邀请川普参加丧礼，川普已经答应出席。
 <br/>
 <br/>
 基于对老布什的尊重，川普还取消了原订1日在阿根廷举行的G20高峰会议记者会，并派出可以担任空军一号的747专机到德州载运老布什灵柩。
 <br/>
 <br/>
 白宫表示，川普宣布5日为全国为老布什致哀日。
 <br/>
 <br/>
 川普在阿根廷告诉媒体，〝我们会以3天时间哀道，也会以3天时间为一位真正伟大人物的一生表示庆祝。〞
