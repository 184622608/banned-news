### 费城老兵节游行 天国乐团金秋展风采
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】周日，在美国《独立宣言》的发源地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_费城.html" target="_blank">
  费城
 </a>
 ，举行了年度退伍军人节游行，向军人和军人家属致敬。游行上，由法轮功学员组成的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天国乐团.html" target="_blank">
  天国乐团
 </a>
 ，第一次受邀参加了庆祝活动。
 <br/>
 <br/>
 晴空下，150多个组织、3千多人参加了游行。浩浩荡荡的队伍穿过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_费城.html" target="_blank">
  费城
 </a>
 市中心。
 <br/>
 <br/>
 150人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天国乐团.html" target="_blank">
  天国乐团
 </a>
 是唯一的华人队伍，也是人数最多的。经过独立宫广场前的主席台时，主持人对壮观的演奏连连赞叹。
 <br/>
 <br/>
 天国乐团周指挥：〝我们参加这个游行是希望以音乐形式告诉世人法轮大法的美好。今天除了传统曲目，包括《法轮大法好》、《神圣的歌》、《法轮圣王》、《凯旋》等，还排练了美国特别有名的《星条旗永不落》，一般现在的游行很少演奏这种传统的进行曲，但是我们天国乐团为了能够恢复
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统文化.html" target="_blank">
  传统文化
 </a>
 ，带来了东方音乐也带来了西方传统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经典音乐.html" target="_blank">
  经典音乐
 </a>
 ，希望大家喜欢。〞
 <br/>
 <br/>
 业主格里芬：〝天国乐团很精彩，演奏非常激动人心。非常感人，真是美好的一天。〞
 <br/>
 <br/>
 她得知本周日天国乐团还将参加纽约老兵节游行时非常高兴，希望再去观赏。
 <br/>
 <br/>
 费城居民Amber：〝我认为需要让自己的孩子了解，军人为国家做出的牺牲，感激为我们国家和其他国家的和平而战的人们。〞
 <br/>
 <br/>
 新唐人费城记者站报导
</div>
