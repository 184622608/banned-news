### 7岁男孩以2200万收入 荣登YouTube频道主冠军宝座
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】年仅7岁的美国男孩莱恩在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_YouTube频道主.html" target="_blank">
  YouTube频道主
 </a>
 收入排行榜中，以2200万美元收入，从去年的第8名，荣登今年第一名宝座。第2名则与他相差50万美元。
 <br/>
 <br/>
 综合媒体报导，《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富比士.html" target="_blank">
  富比士
 </a>
 》(Forbes)杂志截至今年6月1日的记录，莱恩(Ryan)的频道〝Ryan ToysReview〞拥有逾1700万订户，浏览次数累积逼近260亿次，无人能出其右。
 <br/>
 <br/>
 据悉，莱恩3岁时喜欢看YouTube上的玩具开箱影片，当时问妈妈为什么他没有YouTube频道，莱恩妈妈因此兴起拍片念头，并于2015年3月开设〝Ryan ToysReview〞，随后造就1730万人订阅的莱恩王国，其中大多数粉丝都是跟他同龄的小学生，而每款经他称赞过的玩具，都成为热销商品。
 <br/>
 <br/>
 《富比士》指出，开箱影片会如此广受欢迎，是民众观看这类影片，犹如身历其境的喜悦与惊喜，尤其是当影片中的产品独特稀有时，更能吸引民众的目光。
 <br/>
 <br/>
 在《富比士》公布的资料中，莱恩频道收入达2200万美元，排名从去年的第8名，年收入1100万美元，一举跃升至第1名。
 <br/>
 <br/>
 今年8月，莱恩还与零售商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沃尔玛.html" target="_blank">
  沃尔玛
 </a>
 （Walmart）合作，推出〝Ryan's World〞独家品牌玩具和童装，并与爸爸拍摄畅游沃尔玛等影片，上传至今3个月内已经吸引了1400万人次收看，有望再度入围下年度YouTube收入排行榜。目前《富比士》公布的收入数字，未计算税款以及其他佣金和法律服务等开支。
 <br/>
 <br/>

 <br/>
 另外，因莱恩未成年，据法律规定，有15%的收入是直接存入所谓的〝童星专用帐户（Coogan Account）〞，直到莱恩满18岁才能动用。
 <br/>
 <br/>
 其他名列《富比士》排行榜的频道主，包括去年以1650万美元登上榜首的电玩玩家米道顿（Daniel Middleton），今年滑落至第4名；因闯日本自杀森林事件，网红罗根保罗（Logan Paul），全年收入1450万元，跌落第10名；他的弟弟杰克保罗（Jake Paul）则连跳6级，差莱恩50万美元，屈居第2位。
