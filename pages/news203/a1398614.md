### 替换塞申斯 川普称无意结束穆勒调查
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】美国中期选举日刚过，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）就要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部长.html" target="_blank">
  司法部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞申斯.html" target="_blank">
  塞申斯
 </a>
 辞职，由惠特克暂为代理。外界关注，川普下一步是否会开除
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 从而结束〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 〞调查。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 任命惠特克代理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部长.html" target="_blank">
  司法部长
 </a>
 ，意味着之前由司法部副部长罗森斯坦负责监督的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查，今后将由惠特克负责。而惠特克曾质疑通俄门调查的范围。
 <br/>
 <br/>
 不过，法律专家认为，惠特克不会结束通俄门调查。
 <br/>
 <br/>
 前司法部雇员、〝传统基金会〞司法领域专家 Hans von Spakovsky：〝我认为惠特克不会结束调查，他可能会比罗森斯坦发挥更多监督作用，但认为他会立刻结束调查的想法是错的。〞
 <br/>
 <br/>
 川普在周三的记者会上也表示，不会终止通俄门调查，但强调不存在通俄。
 <br/>
 <br/>
 美国总统 川普（2018.11.07）：〝我马上可以解雇任何人，但我没有想结束它（通俄门调查），政治上我不想这么做。调查是耻辱，根本就不应该开始，因为根本没有通俄。〞
 <br/>
 <br/>
 白宫高级顾问 康威周四表示，川普团队非常配合调查，并希望看到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 的调查报告。
 <br/>
 <br/>
 白宫高级顾问 康威：〝总统和他的团队非常配合穆勒的调查。我们完全没有阻碍调查。就我了解，33名证人、33个人被传唤，有上百万页的报告。让我们看看穆勒先生的报告。〞
 <br/>
 <br/>
 不过，民主党人要求惠特克也像
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞申斯.html" target="_blank">
  塞申斯
 </a>
 一样回避，甚至用可能导致〝宪政危机〞来制造舆论压力。
 <br/>
 <br/>
 前司法部雇员、〝传统基金会〞司法领域专家 Hans von Spakovsky：〝宪政危机的说法太过夸张，是一种政治说辞，不存在宪政危机。总统有宪法赋予的权力来决定谁来掌管司法部。〞
 <br/>
 <br/>
 目前，外界普遍猜测新泽西前州长克里斯蒂也许是司法部长人选之一。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
