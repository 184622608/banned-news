### 川习通话余音未了 美司法部发动重磅新攻势
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】11月1日，中美首脑的〝友好〞通话刚刚结束几个小时，正当中共媒体努力〝展望中美和解〞的时候，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国司法部.html" target="_blank">
  美国司法部
 </a>
 就宣布设立专项计划，以打击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共商业间谍.html" target="_blank">
  中共商业间谍
 </a>
 活动，并称〝不能再忍受〞中共的盗窃行径。
 <br/>
 <br/>
 美东时间11月1日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国司法部.html" target="_blank">
  美国司法部
 </a>
 长塞申斯（Jeff Sessions）宣布，他已下令司法部设立〝中国计划〞（China Initiative），专门打击中共针对美国的经济间谍活动。
 <br/>
 <br/>
 在新闻发布会上，塞申斯还宣布了一个新的诉讼案，指控一家中共国企和一家台湾公司合谋盗窃美国美光公司价值87.5亿美元的芯片技术。9月以来，美国司法部已对3起涉及中共国安人员的商业间谍案提起诉讼。
 <br/>
 <br/>
 塞申斯表示，中共正在以偷盗手法爬上经济发展的阶梯，让美国蒙受损失，而且其针对美国的经济情报活动还在迅速增加。
 <br/>
 <br/>
 塞申斯说，〝够了就是够了。我们不能再忍受了。〞
 <br/>
 <br/>
 就在几个小时前，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （Donald Trump）刚刚和北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 通过电话。川普在推特上说，双方的讨论向着月底川习会的方向〝进展良好〞。
 <br/>
 <br/>
 不过，就在同一天，美国国务卿蓬佩奥（Mike Pompeo）就对美国媒体强调，对中国商品加征关税，不仅对科技公司有利，亦对美国农业和工业有益。
 <br/>
 <br/>
 中美双方都在承受贸易战压力，有着结束贸易战的意愿。但是川普多次强调，绝不能再忍受中共的〝经济侵略〞，必须达成公平对等的贸易协议，解决中美贸易不公的问题。
 <br/>
 <br/>
 在北京的一再拖延之下，华盛顿的态度越来越强硬，贸易战不断升级。伴随着副总统彭斯全面指控中共的重磅演讲，中美正走向经济、军事、外交、科技领域的全面对抗。外界纷纷解读，新冷战时代已经开启。
 <br/>
 <br/>
 在贸易战方面，川普采取了与对付朝鲜类似的〝极限施压〞战略，即保留谈判机会的同时，不断加大施压。不过，华盛顿留给北京的谈判机会已越来越少。美媒消息指，白宫拒绝重启谈判，除非北京拿出真正解决美方关切的具体方案。
 <br/>
 <br/>
 目前，美国司法部正在推进对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共商业间谍.html" target="_blank">
  中共商业间谍
 </a>
 活动的严厉打击。港媒认为，关税战高潮已过，川普或将对中共科技产业实施新一轮〝制裁战〞。
 <br/>
 <br/>
 日前，美方已经宣布对福建晋华集团的禁售令。据信福建晋华就是美国司法部长刚刚宣布的新诉讼案被告之一。
 <br/>
 <br/>
 海外观察家指出，虽然川普一直希望和习近平保持良好个人关系，但在美国三权分立的背景下，遏制、反击中共已成为美国主流社会整体共识，在北京发生彻底改变之前，美方的相应遏制措施仍可能不断出台。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
</div>
