### 联邦报告：每个非法移民消耗美国纳税人7万元社会成本
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】近日，数千名来自中美洲的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 涌入墨西哥，并长途跋涉奔向美墨边境。正当美国政府想方设法欲阻止这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无证移民.html" target="_blank">
  无证移民
 </a>
 闯入美国境内之际，美国移民政策研究院(MPI)公布了两份联邦报告，其中分析非法移民给美国社会带来的附加成本时指出，每个非法入境的无证移民，平均一年耗费美国纳税人近7万美元。
 <br/>
 <br/>
 中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 乘坐各式各样的大篷车，大规模涌向美墨边境，并欲强行闯入美国，已成为了当前最让美国政府伤脑筋的一件大事。
 <br/>
 <br/>
 日前，移民政策研究院(MPI)公布的一份调查报告中，提及了两个联邦政府的报告，分别从不同的角度，指出了大量非法移民涌入美国的严重后果。
 <br/>
 <br/>
 其中，第一份报告指出，大量合法的和非法的移民涌入美国，导致美国每年增加的人数，足以让费城的人口翻上一番。
 <br/>
 <br/>
 第二份报告则分析了外来移民给美国社会带来的成本支出。
 <br/>
 <br/>
 报告指出，大量外来移民（包括合法的与非法的）持续涌到美国，导致美国社会依靠纳税人上缴的税收支持的执法以及社会服务等方面的开支不断上升，即使这些移民中合法获得身份的人工作缴税也无法抵消。
 <br/>
 <br/>
 报告称，每一个非法进入美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无证移民.html" target="_blank">
  无证移民
 </a>
 ，平均每一年要花费美国社会6万9570美元。也就是说，每增加100万非法移民，美国社会就要多支出696亿美元。
 <br/>
 <br/>
 与此同时，移民政策研究院(MPI)的调查报告指出：目前生活中全美国的〝定锚子女〞 (anchor babies)就有大约400万人之多。这些由非法滞留于美国的外国人所生的孩子，因为在美国领土上出生而获得了美国公民的身份，他们的父母亲也因此得以继续留在美国生活，因此这些孩子被统称为〝定锚子女〞。
 <br/>
 <br/>
 据MPI30日公布的统计资料：仅2016年度，全美就共有407万6000名18岁以下的〝定锚子女〞。其中双亲都是无证移民的，有132万8000人；双亲之一是无证者的，有1839万9000人；由无证单亲抚养的定锚子女也高达90万9000人。
 <br/>
 <br/>
 MPI表示，每隔几年，政治人物就会辩论是否废止美国宪法第14条修正案所赋予的与生俱来公民权，而就在不久前，美国总统川普（特朗普）还曾对外表示，国会应修宪来废止这条修正案。
 <br/>
 <br/>
 报告认为，如果美国政府废除双亲中至少有一人是无证移民的〝定锚婴儿〞的公民权，那么美国的无证移民人口总数量，将从2015年的1100万，骤然增至2050年的2400万。
 <br/>
 <br/>
 （记者李明报导/责任编辑：戴明）
