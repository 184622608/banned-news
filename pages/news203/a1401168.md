### 共和党赢密州选举 参院力量对比再拉差距
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】周二（11月27日）晚，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 人海德-史密斯 （Cindy Hyde-Smith）在密西西比州联邦参议员决选中获胜，为共和党再拿下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 一席，并为未来川普总统提名的法官获得成功确认铺平了道路。
 <br/>
 <br/>
 周二晚，在11月6日选举日未决出胜负的密州参议员选举再决雌雄，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 现任参议员海德-史密斯终于以明显的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_优势.html" target="_blank">
  优势
 </a>
 战胜其民主党对手埃斯皮（Mike Espy），给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 中期选举画上句号。
 <br/>
 <br/>
 密州本次参议员选举是该州几十年来最激烈的竞选之一。海德-史密斯与非裔候选人埃斯皮在大选夜所获得的选票均未过半，且两人得票率相差不足1个百分点，因而必须举行决选。
 <br/>
 <br/>
 周二晚所开出的95%的选票显示，海德-史密斯以54.4%的得票率击败埃斯皮（得票率为45.6%），终于拉开与对手的差距并赢得选举。
 <br/>
 <br/>
 海德-史密斯的胜选扩大了共和党在参议院的优势。 从明年1月起，共和党和民主党在参议院的席位之比将从当前的51:49变为53:47，共和党稳居多数优势。
 <br/>
 <br/>
 决选前，川普总统再赴密州，出席了两场为海德-史密斯助选而举行的竞选集会。
 <br/>
 <br/>
 周二晚，川普在推特上祝贺海德-史密斯获胜：
 <br/>
 <br/>
 〝恭喜参议员海德-史密斯在伟大的密西西比州大获全胜。我们都为你骄傲！〞
 <br/>

 <br/>
 在密州州长的提名下，海德-史密斯于2018年4月填补前密州共和党参议员柯克兰（Thad Cochran）辞职后留下的空缺，成为该州第一位女性国会代表。
 <br/>
 <br/>
 海德-史密斯的胜选意义重大。从某种程度上说，他铺平了未来川普提名的法官在参议院的确认之路，或使共和党避免再次面对今年最高法院大法官卡瓦诺（Brett Kavanaugh）确认过程中所遭遇的艰苦局面。
