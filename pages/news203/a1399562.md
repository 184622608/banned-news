### 警示中朝？美双航母打击群在菲律宾海演习
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】美国海军周四（15日）表示，〝里根〞号航母和〝斯坦尼斯〞号航母以及约150架战机正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾海.html" target="_blank">
  菲律宾海
 </a>
 举行〝复合式〞战备演习。此次演习是美军航母一个月内第二次驶入该海域。因时机敏感，外界猜测是在警告朝鲜与中共不要轻举妄动。
 <br/>
 <br/>
 中央社报导，美国第七舰队15日称，部署于日本的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根号.html" target="_blank">
  里根号
 </a>
 （USS Ronald Reagan）及美国西岸的史坦尼斯号（USS John C. Stennis），及约150架战斗机正在中国南方的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾海.html" target="_blank">
  菲律宾海
 </a>
 （Philippine Sea）一带进行海陆和反潜作战演练，打击范围涵盖朝鲜。
 <br/>
 <br/>
 以〝里根〞号为中心组建的〝第五战斗群〞是目前美国海军实力最强的海军战斗群，其中重达10万吨的〝里根〞号搭载着约75架飞机，包括F/A-18大黄蜂战机、E-2鹰眼预警机和SH-60海鹰巡逻直升机等。航母周围还配备有能发射巡航导弹的军舰和为战斗群提供防卫的驱逐舰。
 <br/>
 <br/>
 暂时未知此次行动何时结束。军方称，这次演习旨在展示海军操作多个航母战斗群，作为协同攻击部队的独特能力。
 <br/>
 <br/>
 美国海军第七舰队副司令索约尔（Phil Sawyer）介绍说：〝集结两组航空母舰打击群，将提供海军无与伦比的战斗力、极高的作战灵活度及覆盖区域的能力。〞
 <br/>
 <br/>
 他说，美军增加在印太区域的航母部署，反映美方对该区自由及开放的承诺，重申海军将继续提供安全保障，促进地区稳定及繁荣。
 <br/>
 <br/>
 美国海军指挥官托马斯少将（Rear Adm. Karl Thomas）向媒体表示，这一名为〝利剑〞（Keen Sword）的演习旨在展现〝美国和日本对维护一个自由和开放的印太地区的承诺〞。
 <br/>
 <br/>
 他说，〝显然现在有很多事情正在朝鲜半岛上发生，我对（美朝）对话感到鼓舞，〞托马斯说：〝我们和日本、菲律宾以及这一地区的各国海军的工作是为了寻求稳定。〞
 <br/>
 <br/>
 值得关注的是，这次演习的时机也相当敏感，适逢美国副总统彭斯（Mike Pence）访问亚洲，亚太经合组织峰会（APEC）召开之际，美国或将在会上抨击中共在军事、海外投资和贸易方面破坏国际规则。
 <br/>
 <br/>
 而且美国与朝鲜的非核化谈判也没有取得进展的迹象。去年，美国在和朝鲜关系紧张时也曾有3艘航空母舰展开类似演习。
 <br/>
 <br/>
 值得一提的是，美国盟邦加拿大也受邀参加了此次〝利剑〞演练，这是其首度参加该演习。
 <br/>
 <br/>
 美国海军上尉哈尼芬（Captain Pat Hannifin）对此表示：〝美军每天都在寻求发展，并将局势的发展融入计划和训练中来〞，他说，〝我们还将最亲近的盟国融入进来，例如日本和加拿大，所以这是一个持续的发展，我们和全球的伙伴及盟友保持密切合作。〞
 <br/>
 <br/>
 此次演习是美军航母一月内第二次驶入菲律宾海，〝里根〞号航母战斗群在3周前和日本海上自卫队举行了两国本年度最大型的军事演习。两国当时出动了数十艘军舰，数百架军机，及5.7万水兵及其他人员参与了演习。
 <br/>
 <br/>
 斯坦尼斯号航母战斗群驶入菲律宾海前，曾在夏威夷附近与卡尔文森号航母战斗群共同行动及演习。美军于过去数年亦多次在西太平洋展开〝双航母〞行动，地点包括朝鲜半岛水域、日本海、南海、东海及菲律宾海，涉及驻日航母及从美国西岸出发的航母。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
