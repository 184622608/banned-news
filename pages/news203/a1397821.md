### 南早：川习会计划有变 或改G20后晚宴
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】《南华早报》此前消息，中美首脑拟在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 前的11月29日，在阿根廷会晤。但在11月1日的〝友好通话〞后，该媒体又引述消息称，两人或改成G20峰会后举行〝会晤加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晚宴.html" target="_blank">
  晚宴
 </a>
 〞。
 <br/>
 <br/>
 香港《南华早报》11月2日援引消息人士报导，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （Donald Trump）提出在G20领导人峰会后，在阿根廷布宜诺斯艾利斯宴请北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，北京也表示接受邀请。但现在还不清楚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晚宴.html" target="_blank">
  晚宴
 </a>
 的具体议程。
 <br/>
 <br/>
 报导称，同峰会间隙的对话相比，峰会后的会晤晚宴能够提供更多的交谈时间，和更加有利于谈判的氛围。
 <br/>
 <br/>
 据公开资讯，2018年二十国集团（G20）领导人峰会，将于11月30日至12月1日在阿根廷举行。
 <br/>
 <br/>
 10月20日，《南华早报》曾报导，中美双方暂定11月29日会晤。
 <br/>
 <br/>
 白宫官员称，这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 将是一次短暂的会晤。美媒消息指，如果北京不能回应美方的贸易关切，川习会或将排除贸易议题。
 <br/>
 <br/>
 不过，在北京10月31日的政治局会议承认经济下行压力的次日，中美首脑进行了几个月来的首次通话。川普在推特形容，这是一次长时间的讨论，向着月底川习会的方向〝进展良好〞。
 <br/>
 <br/>
 与此同时，中共高层和官媒也一改此前强硬态度，努力释放〝中美友好〞的信息。
 <br/>
 <br/>
 这些消息给中美和解带来某种希望，中国和香港股市、人民币汇率都有所反弹。不过，对于双方是否能真正和解，外界有不同的看法。
 <br/>
 <br/>
 有分析认为，北京的经济陷入困局，或被迫接受美方要求。也有人认为，这不过是中方的又一次拖延战术。
 <br/>
 <br/>
 此前多次谈判中，美方都曾向中方提交了具体的贸易要求，但每次北京都对其中的关键部分表示拒绝，理由是涉及中方〝根本利益〞。
 <br/>
 <br/>
 据新华社通稿，在11月1日的通话中，习近平也向川普表示，〝中方已就中美关系多次阐明原则立场〞，并希望双方经济团队加强接触磋商，以达成一个〝双方都能接受的方案〞。
 <br/>
 <br/>
 目前美国经济强势，遏制中共已是朝野和两党的共识，川普也多次强调必须解决中美贸易不公。外界普遍认为，川普对中共让步的可能性不大。
 <br/>
 <br/>
 而对北京而言，如果顽抗到底，将意味着经济最终将崩溃；但如果接受美国要求，就必须彻底改变经济体制，这等于动摇中共统治的根基。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
