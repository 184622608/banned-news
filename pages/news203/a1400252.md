### 大篷车移民藏500罪犯 川普:不会让他们来
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】六千多名中美洲大篷车难民，在美墨边境城市蒂华纳聚集。美国国土安全部周二在圣地亚哥表示，他们中藏有500名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罪犯.html" target="_blank">
  罪犯
 </a>
 。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 再次强调，不会让非法移民轻易进入美国。
 <br/>
 <br/>
 美国国土安全部部长尼尔森，周二在圣地亚哥视察边境墙时表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 不是像一些人所说，大部分是逃离暴力的妇女儿童，而是大部分是单身男子，其中有500多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罪犯.html" target="_blank">
  罪犯
 </a>
 。
 <br/>
 <br/>
 美国国土安全部部长尼尔森:〝此外，我们已确定大篷车中有500多名罪犯，包括已知的帮派成员。〞
 <br/>
 <br/>
 美墨入境关口附近，几英里的边界墙上新安装了一圈一圈的剃刀铁丝网，防止移民非法越境。
 <br/>
 <br/>
 尼尔森对试图非法入境的移民发出警告。
 <br/>
 <br/>
 美国国土安全部部长尼尔森:〝毫无疑问，我们非常认真。你不会非法进入我们国家的。〞
 <br/>
 <br/>
 周二，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 也强调，现在的政策不同以往，不再容许非法移民。
 <br/>
 <br/>
 美国总统川普:〝现在的规矩不是，我们可轻易过到美国去。我们不会让人进来。现在是〝抓住就拘留〞，不是多年来的〝抓住就释放〞。〞
 <br/>
 <br/>
 针对旧金山第九巡回上诉法院周一的临时限制令，阻止川普取消非法入境移民申请庇护资格，川普表示，将会寻求最高法院裁决。
 <br/>
 <br/>
 美国总统川普:〝我要说的是，第九巡回上诉法院 - 如果每人都可在第九巡回上诉法院起诉的话，这是非常不公平的。 〞
 <br/>
 <br/>
 川普表示，他相信最高法院最终会驳回第九巡回上诉法院的临时限制令，就像之前的旅行禁令在最高法院通过一样。
 <br/>
 <br/>
 新唐人记者王羽音、张琪综合报导
</div>
