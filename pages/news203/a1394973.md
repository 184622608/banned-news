### 美中暗战明朗化 美首引渡中共间谍意在杀鸡儆猴？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】今年美国首开先例，在欧洲诱捕中共情报官员，并引渡回美国受审。目前中美对抗已经公开化，港媒指，美国此举在于杀鸡儆猴，阻止中共继续窃取美资
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商业机密.html" target="_blank">
  商业机密
 </a>
 。
 <br/>
 <br/>
 美国司法部日前公布，被捕官员是中共江苏省国家安全厅副处长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐彦钧.html" target="_blank">
  徐彦钧
 </a>
 （Yanjun Xu，或译
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐燕军.html" target="_blank">
  徐燕军
 </a>
 ），他被指控涉嫌从事经济间谍活动，窃取美国多家公司的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商业机密.html" target="_blank">
  商业机密
 </a>
 。而中方至今未回应这宗案件。
 <br/>
 <br/>
 美方诉状显示，徐彦钧是被FBI诱至比利时抓捕，然后再引渡到美国受审。这是美国首次采取类似行动。
 <br/>
 <br/>
 港媒引述分析指，中美关系迅速恶化，美方多次公开谴责中共盗取美国知识产权，美方破例采取这种行动，对中共有强烈警告的意味。
 <br/>
 <br/>
 美国司法部国家安全部门在通报案情时，直指此案并非独立事件，而是中共以牺牲美国利益为代价、取得发展的经济政策之一。〝美国不能容忍一个国家窃取知识成果，不能容忍一个国家不劳而获。〞
 <br/>
 <br/>
 因此，有分析指，美方跨国抓捕中方情报人员，已向中共发出明确讯号，如果中方不停止类似间谍活动，美方将继续采取同类行动对中共情报官员进行扫荡行动。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-11/p9075651a366309192.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-11/p9075651a366309192-ss.jpg"/>
   <br/>
   <font size="-1">
    美国司法部通告徐彦钧被控罪。（司法部网页截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 根据美国司法部公开资料，自2013年起，徐彦钧以多种伪装身分，接触联系美国航太与航空业界人士，邀请这些人赴中国〝学术交流〞。
 <br/>
 <br/>
 其中，从2013年12月起，徐彦钧以南京科技促进协会副秘书长的身分，开始接触一位美国通用电气航空（GE Aviation，又译
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奇异航空.html" target="_blank">
  奇异航空
 </a>
 ）的华裔工程师，之后多次从该工程师处获取通用电气航空的机密技术资料，并向其提供各种好处。
 <br/>
 <br/>
 不过，徐彦钧并不知道，通用电气航空数月来一直在配合FBI的调查。今年4月，徐彦钧邀约该工程师在比利时见面，交付公司内部机密文件，徐在比利时遭FBI抓捕。
 <br/>
 <br/>
 据《今日美国报》，对于经济间谍活动，美国最高可处15年徒刑；盗窃商业机密最高则可处10年徒刑。除此之外还可能另有罚款。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
