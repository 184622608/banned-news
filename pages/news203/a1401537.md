### 7.0强震 阿拉斯加紧急状态 道路现裂缝机场关闭
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿拉斯加.html" target="_blank">
  阿拉斯加
 </a>
 州第一大城安克拉奇（安克拉治）周五（11月30 日）发生7级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 ，约7千到1万户家庭断电。当局一度发布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海啸.html" target="_blank">
  海啸
 </a>
 警报，但不久后解除。远在阿根廷的川普（特朗普）总统发推文称，愿上帝保佑。
 <br/>
 <br/>
 美国地质调查局称，7级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 发生于格林尼治时间11月30日17时29分，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿拉斯加.html" target="_blank">
  阿拉斯加
 </a>
 麦肯锡角以南，震中位于安克拉奇（Anchorage）以北8.1英里（约13公里）处，深度为26.7英里（约42.9公里）。国家气象局说，依其里氏震级标准，这次地震达7.2级。
 <br/>
 <br/>
 地震发生后，当局针对库克湾（Cook Inlet）和金奈半岛（Kenai Peninsula）发布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海啸.html" target="_blank">
  海啸
 </a>
 预警，不过，随后在确认无海啸风险后，取消了预警通告。
 <br/>
 <br/>
 安克拉奇警政部门表示，强震对该市〝基础设施造成重大破坏〞，〝多栋民宅和建筑物受损〞，〝许多道路和桥梁关闭〞，民众〝若无必要开车，请远离马路〞。
 <br/>
 <br/>
 阿拉斯加州州长沃克（Bill Walker）宣布该州进入紧急状态，并称已与白宫联络，以及与应变人员合作，以保障该州居民的安全。
 <br/>
 <br/>
 白宫新闻秘书桑德斯（Sarah Sanders）表示，正在阿根廷参加20国集团（G20）高峰会的川普总统，已听取地震简报，并持续追踪灾情。
 <br/>
 <br/>
 川普还发推文称〝伟大的阿拉斯加州居民，你们遭到了大地震的重创。请遵照帮助你们的训练有素专业人士的指示应变。 联邦政府将不遗余力（提供协助）。 上帝保佑！〞
 <br/>

 <br/>
 <br/>
 目前还不清楚这次地震是否有居民伤亡，许多居民在地震发生时逃到户外。随后发生余震，强度达5.0以上。
 <br/>
 <br/>
 当地部分居民在社交媒体上传视频或图像，从中可见，一家杂货店的商品散落在地上、学校教室灯具摇晃、橱柜门被震开、道路隆起、地板龟裂。有人发推文说，他以为这次没命了。

 <br/>
 <br/>
 安克拉奇当地电视台KTVA发布影片显示，一间房间剧烈摇晃，灯光忽明忽灭，民众躲在桌下，不料天花板突然掉落。
 <br/>
 <br/>
 安克拉奇每日新闻（Anchorage Daily News）表示，地震造成报社受损，墙壁龟裂，天花板损坏。
