### 中期后美媒再炒通俄门 川普发推质疑调查
------------------------

 【新唐人2018年11月15日讯】美国媒体藉

  川普
 </a>
 任命代理司法部长一事炒作川普要开除
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 ，民主党人称要保护穆勒调查。川普则通过社交媒体表达自己的意见。
 <br/>
 <br/>
 已耗时1年半但仍然没有找到证据证明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 竞选团队与俄罗斯勾结干预美国大选的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 〞调查，因为中期选举平静些许日子后，又被美国大媒体带回到人们的每日生活中。
 <br/>
 <br/>
 CNN和《纽约时报》等报导，从获得的电子邮件通讯来看，罗杰斯通（Roger Stone）、川普竞选前高级顾问，是川普前选战经理班农与维基解密创始人阿桑奇的中间人。后者在2016总统大选期间，向公众披露了希拉里克林顿竞选团队和民主党全国委员会的电子邮件，显示希拉里竞选中的一些可疑做法。而班农等人在维基解密披露电邮前已经知道此事。不过，目前这些信息是否属实没有被确认。
 <br/>
 <br/>
 川普周四连发数则推文，对通俄门调查表达质疑。他说〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 调查的内部运作完全是混乱的。他们因没有发现任何通俄证据而变得非常疯狂。他们对人咆哮、想通过威胁得到他们想要的答案。他们是美国的耻辱。他们不在意有多少人的生活因此被摧毁。这些是愤怒的人群，包括为奥巴马工作了8年的高度被争议的鲍勃穆勒。他们完全不看另一方的所有不好的行为和罪行。美国历史上绝无仅有的猎巫（调查）！〞
 <br/>
 <br/>
 在另一则推文中川普质问，为什么穆勒一直保护希拉里、科米、麦凯布、史卓克等人。
 <br/>
 <br/>
 联邦参议院多数党领袖Mitch McConnell：〝我们知道总统对穆勒调查的看法，但他从来没有说过要停止它（调查）。我从来没有听到任何人说他们想结束它。〞
 <br/>
 <br/>
 另外，对于川普任命惠特克为司法部代理部长，美国大媒体的报导、发生在华府和纽约的团体抗议、民主党人的言行都在强化一个概念，那就是川普打算解雇穆勒。理由是在惠特克担任新闻评论员时，多次表态批评通俄门调查。
 <br/>
 <br/>
 民主党人甚至要求惠特克也应该像塞申斯一样从监督通俄门调查的位置上退出、回避。
 <br/>
 <br/>
 2017.7.26 CNN Tonight新闻评论节目中表示，塞申斯无需开除穆勒，只要限制其预算即可削弱调查。
 <br/>
 <br/>
 2017.8.7 CNN At This Hour穆勒调查川普家族财务跨越红线、超过通俄调查范围。
