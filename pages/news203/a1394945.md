### 差5公尺到地表 美墨边境疑毒贩挖跨国隧道
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】
 <br/>
 】美国边境执法部门近日在美墨边境以南221英尺处，发现一处尚未完工的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_隧道.html" target="_blank">
  隧道
 </a>
 ，内建集水坑泵、照明系统与通风设施，另有直达美国境内的铁轨。据信是由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒贩.html" target="_blank">
  毒贩
 </a>
 与人蛇集团所打造来从事不法勾当。
 <br/>
 <br/>
 据美国海关与边境保护局（U.S. Customs and Border Protection）公布的报告，墨西哥警方与军方9月19日，在美墨边境以南221英尺(约67.36)处，发现这隧道的起点，隧道向北由地底穿越两国的边界后，在墨西哥境内下加利福尼亚州（Baja California）哈昆姆（Jacume）的一栋住宅被发现。
 <br/>
 <br/>
 这个隧道尚未在美国境内挖掘出口，不过隧道内已建置两个集水坑泵、一座太阳能发电的照明系统与通风设施，还有直达美国境内的铁轨。
 <br/>
 <br/>
 美国国土安全部（U.S. Department of Homeland Security）、美国缉毒局（Drug Enforcement Administration）与边巡单位证实，隧道的入口处在地底下31英尺(约9.4公尺)处，全长627英尺（约191公尺），出口处在美国境内，不过尚未凿穿至地面，目前已向上延伸至距离地表约15英尺(约4.6公尺)处。
 <br/>
 <br/>
 探员称，挖掘这样一条复杂的隧道，需要花费很多时间和金钱。从隧道的规模和使用技术来看，应该是毒贩准备向美国境内走私毒品的。
 <br/>
 <br/>
 美国总统川普为防范非法移民、贩毒与人口贩运，下令在美国与墨西哥边境建围墙。
