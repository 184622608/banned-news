### 美中期选举临近 民调：民主党重掌参院无望
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 将近，愈来愈多迹象显示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 无望成为国会参院多数党，而且要想争取控制众院也是难上加难，或将由少数几个席位来决定胜负。
 <br/>
 <br/>
 美国全国广播公司新闻网数据分析实验室（NBC News Data Analytics Lab）在分析由政治数据与技术领先提供商TargetSmart所收集整理的数据后，10月22日公布：结果显示，在7个关键州的早期投票中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 占据优势，参众两院的控制权在选举后或仍为共和党所掌握。
 <br/>
 <br/>
 民主、共和两党的高层人士都认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 要成为参院多数党之路几乎没有希望。而要成为众院的多数党也是困难重重，最终会由少数几个席位来决定。相反，共和党人的支持率持续升温。
 <br/>
 <br/>
 民主党的民调专家马斯林(Paul Maslin)称，想要在参院取得大多数，就必须先拿下原属民主党的席位，然后在支持共和党的州再挖多个席位，可惜根据现在的情况来判断，民主党基本上难以如愿。
 <br/>
 <br/>
 多个民调显示，共和党支持者愈接近11月6日的投票日，情绪愈高涨。共和党顾问艾里斯(Whit Ayres)表示，在任命卡瓦诺为联邦最高法院大法官的过程中，反而更刺激了共和党的支持者。他估计，投票结果共和党不但可望维持参院的大多数，甚至还可能会增加1～2个席次。
 <br/>
 <br/>
 按目前的情势看，共和党愈来愈乐观，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 最近的种种表现，都得到了选民的支持，从朝鲜到北美自由贸易协定，再到中美贸易战，民众普遍认为：川普正在成功地维护美国的利益。另外强力支持保守派法官卡瓦诺出任大法官、对非法移民逼近边界的强硬姿态等等也都让选民更加支持川普。
 <br/>
 <br/>
 在11月6日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 上，美国选民将重选出众议院和参议院两院的议员。其中众议院所有435个议席都将重选，参议院则是100个席位中的35席将面临重选。中期选举的结果对当届总统余下任期的执政、以至两年后的大选都将产生不小的影响。
 <br/>
 <br/>
 随着中期选举即将进入倒数计时，川普竞选活动团队说，为了保住共和党在国会的多数席位，川普将在未来两周全力助选，计划再参加至少10场MAGA竞选活动，为共和党候选人助选。川普的竞选活动团队表示，川普还同意从其竞选资金中，拨出600万美元做电视和数字广告，定于下周一（29日）开始推出，并持续到11月6日。此外，也将提拨300万美元给RNC，用于共和党的宣传活动。
 <br/>
 <br/>
 川普竞选活动团队告诉福克斯新闻，自今年夏天以来，川普至少参加了20场〝让美国再次伟大〞（Make America Great Again，MAGA）集会活动，除了为11月6日中期选举的共和党候选人助选外，同时加强其与基层选民的接触，以为即将到来的2020年总统大选暖身。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
