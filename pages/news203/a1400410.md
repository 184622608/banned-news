### 川普感恩节讲话 祝福美国人民
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】周四上午，美国总统川普发表视频，向全国人民祝贺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 快乐。
 <br/>
 <br/>
 川普代表家人，祝全国人民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 快乐！
 <br/>
 <br/>
 美国总统川普：〝祝所有人感恩节快乐！从建国开始，美国人民就聚在一起，纪念并
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感谢神.html" target="_blank">
  感谢神
 </a>
 赐予我们福祉。〞
 <br/>
 <br/>
 川普特别向20万驻守海外的美国人表达祝福，感谢他们的服务。
 <br/>
 <br/>
 他还向飓风、野火、以及暴力事件中幸存的人们和家人表示慰问。
 <br/>
 <br/>
 美国总统川普：〝特别是那些幸存者和家属，他们在可怕的仇恨犯罪中幸免遇难。我们祈祷，为灾难性的暴风雨和加州野火肆虐中幸存的人们。〞
 <br/>
 <br/>
 川普重申
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节精神.html" target="_blank">
  感恩节精神
 </a>
 。
 <br/>
 <br/>
 美国总统川普：〝我们感谢神对美国人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保佑.html" target="_blank">
  保佑
 </a>
 ，能和家人一起欢聚，以及英雄们的牺牲，他们让美国安全、强大而自由。谢谢。神保佑你们。祝大家感恩节快乐！〞
 <br/>
 <br/>
 新唐人记者易文、明玉综合报导
