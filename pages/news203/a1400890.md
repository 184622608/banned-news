### 大篷车边境抗议欲闯关 执法人员使用催泪弹
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】周日，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 的数百
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 试图强行闯入美国，美方执法人员驱散人群并关闭了边境口岸。事件引发美墨两国，加强
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境执法.html" target="_blank">
  边境执法
 </a>
 。
 <br/>
 <br/>
 对于试图冲向隔离围栏的墨西哥一侧
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 ，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境执法.html" target="_blank">
  边境执法
 </a>
 人员发出警告。
 <br/>
 <br/>
 周日，数百滞留在墨西哥蒂华纳的大篷车移民进行了抗议活动，之后一些人试图闯关。
 <br/>
 <br/>
 来自危地马拉的大篷车成员Manuel Véliz：〝我们进行了一场和平示威，不过情况变得有点紧张，但没有打斗发生。〞
 <br/>
 <br/>
 美国边境执法人员使用催泪弹驱散人群，边境及海关执法局表示执法人员认为当时情况对他们构成人身威胁。随后美方再次短暂关闭了圣伊西德罗入境口（San Ysidro Port of Entry）。墨西哥称有39人被逮捕。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）：〝他们不得不使用（催泪弹），因为他们受到一些非常强硬的人群冲击，因此使用了催泪弹。〞
 <br/>
 <br/>
 总统川普周一再通过社交媒体发声，要求墨西哥将非法移民送回去；特别是那些挥舞着本国国旗的中美洲人，其中大部分是罪犯。
 <br/>
 <br/>
 对于美国大媒体报导由于严格执法导致边境线上不断上演的骨肉分离的情况，川普在推文中指出CBS〝60分钟〞新闻节目使用一张2014年的老照片来说假故事。
 <br/>
 <br/>
 美国总统川普：〝奥巴马执政时就是‘分离政策’，我们用的是相同政策，但我试图用不同的方式处理。〞
 <br/>
 <br/>
 他说：〝我试图让这些人全家在一起，但问题是，你一旦这样做，更多的人会冲击边境。〞
 <br/>
 <br/>
 来自洪都拉斯的马丁内斯迫于在国内找不到工作与父亲一起登上了大篷车，现正在墨西哥蒂华纳等待机会。他的母亲和妻子几个月前已经成功进入美国并定居下来。
 <br/>
 <br/>
 来自洪都拉斯的大篷车成员Frank Martinez：〝我先找份工作再看看。听说去招聘会可以拿到合法的（待在墨西哥的）许可，然后就可以跨境进入美国。看看怎么样吧，接着就是申请政治庇护。〞
 <br/>
 <br/>
 记者：〝如果你无法进入美国，你要去哪？〞
 <br/>
 <br/>
 来自洪都拉斯的大篷车成员Frank Martinez：〝那就非法入境。〞
 <br/>
 <br/>
 随着感恩节假期结束议员们回到国会，川普要求国会批准建设
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境墙.html" target="_blank">
  边境墙
 </a>
 的资金。也再次重复永久关闭边境是可用的选项。
 <br/>
 <br/>
 新唐人记者李兰、特约记者Kimberly Hayek纽约、蒂华纳连线报导
</div>
