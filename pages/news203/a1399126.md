### 大陆留学生代考现象盛行 催生职业〝枪手〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】越来越多的大陆学生到海外留学，近年来，随着大陆学生的涌入，留美学生中雇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪手.html" target="_blank">
  枪手
 </a>
 代写作业甚至
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代考.html" target="_blank">
  代考
 </a>
 现象越来越严重。
 <br/>
 <br/>
 美国国土安全部数据显示，截至2018年7月，超过34万的大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_留学生.html" target="_blank">
  留学生
 </a>
 正在美国学习，绝大多数正在攻读本科或以上学位。从国别来看，大陆人占据了美国国际学生总数的三成，位居第一。
 <br/>
 <br/>
 据自由亚洲电台11月12日报导，美国高校蕴藏着富饶的教育资源，大陆的父母不惜花费一百多万元人民币把子女送到美国上大学。但是，一纸文凭是否表示真才实学，值得怀疑。因为在大陆留学生中，一直存在着雇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪手.html" target="_blank">
  枪手
 </a>
 代写作业甚至
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代考.html" target="_blank">
  代考
 </a>
 的现象。
 <br/>
 <br/>
 枪手服务在大陆留学生中是公开的，据一位刚从纽约大学游戏设计专业毕业的大陆留学生李龙霄回忆说，2011年首次来到美国上学时，他对这类枪手服务还鲜有耳闻，自从2013年以后，这些生意变得随处可见，而且大量在大陆留学生的各种微信群里面打广告。
 <br/>
 <br/>
 枪手琳达（化名）是美国某高校英语专业的硕士毕业生，受雇于美国波士顿的一家枪手公司。琳达表示，她的公司瞄准的是大陆留学生市场。他们的顾客包括波士顿大都会区以及其他地区的大陆学生。
 <br/>
 <br/>
 关于收费标准，琳达介绍，如果只是一篇普通论文，会收取每页25到35美元不等的代写费用。此外，她还提供整门课作业的代写服务，比如大一新生必修的英语写作课。她举例说，如果学生需要她代写这门课定期布置的作业、课堂汇报内容和零星的论文，她会按每堂课一学期1000美元左右的价格收取服务费。在此基础上，负责经营这家公司的朋友还会额外向这些留学生收取一笔数目不明的中介费。
 <br/>
 <br/>
 琳达还补充：她时常会根据学生的身份调整代写风格，甚至有时候会刻意添加语法错误，以显示较低的写作水平，避免教授起疑。
 <br/>
 <br/>
 尽管接踵而来的代写给琳达带来了可观的收入，她还是不免为大陆留学生感到遗憾，因为他们自愿放弃了提升语言能力、融入当地文化的良机。
 <br/>
 <br/>
 自由亚洲电台援引美国圣母大学人类学教授苏珊．布卢姆（Susan D.Blum）的观点表示，枪手市场钟爱大陆留学生可以追溯到他们的母国。她说，大陆腐败文化横行，不少人的是非观比较模糊。如果他们在国内就经常不按规矩行事，这些人到了国外未免也会想方设法打擦边球。
 <br/>
 <br/>
 一位主修国际关系的乔治．华盛顿大学的大四学生透露，她身边多位大陆同学都雇用过枪手。其实她认识的很多学生完全有能力自主完成作业，但他们可能只是嫌麻烦，所以出钱买作业。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
