### 火鸡豌豆仔和萝卜丝 获得川普总统赦免
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】周四就是美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 了。一年一度的总统赦免
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火鸡.html" target="_blank">
  火鸡
 </a>
 仪式，周二在白宫举行。两只幸运火鸡获得了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统的赦免。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：〝今天的幸运儿同时也是我们的嘉宾，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_豌豆仔.html" target="_blank">
  豌豆仔
 </a>
 〞和它的伙伴〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_萝卜丝.html" target="_blank">
  萝卜丝
 </a>
 〞。台下的孩子们非常了解。获胜者是由白宫网站举办的公平公开投票中选出。〞
 <br/>
 <br/>
 据了解，39磅重的〝豌豆仔〞和41磅的〝萝卜丝〞，将在弗吉尼亚理工大学度过余生。
 <br/>
 <br/>
 川普总统随后发表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 祝词，感谢神的赐予，让美国国泰民安。
 <br/>
 <br/>
 美国总统 川普：〝我们此刻暂停手边工作，感谢神赐予我们的一切。在给予这么多，这么多，这么多的祝福之后，我们心存感恩。〞
 <br/>
 <br/>
 仪式结束后，川普总统启程前往佛州海湖庄园，与家人共度感恩节。
 <br/>
 <br/>
 美国总统 川普：〝我代表全体川普家族成员，祝所有美国人度过一个非常、非常愉快的感恩节。〞
 <br/>
 <br/>
 新唐人记者曹菁综合报导
</div>
