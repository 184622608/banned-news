### 美驻联合国大使黑利辞职 川普接受辞呈
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】美国驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 大使妮基．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑利.html" target="_blank">
  黑利
 </a>
 （Nikki Haley）提出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 ，周二（10月9日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统接受了她的辞呈。
 <br/>
 <br/>
 据福克斯新闻（Fox News）10月9日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周二上午在白宫椭圆形办公室宣布了这一消息。川普告诉在场记者，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑利.html" target="_blank">
  黑利
 </a>
 做出了〝令人难以置信的工作〞，是一个〝极好的人〞。
 <br/>
 <br/>
 川普称，早在6个月之前，黑利就告诉她〝可能会在年底〞休息一下，但表示她愿意工作到2018年底。
 <br/>
 <br/>
 最先报导这一消息的美国Axios.com网站称，黑利在上周赴白宫面见川普时就讨论了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 问题。黑利辞职的消息令很多美国高级外交官员感到震惊。
 <br/>
 <br/>
 一些来自国务院的消息人士告诉福克斯新闻，黑利直到9日上午才告诉自己的助手们她的打算。
 <br/>
 <br/>
 美国驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 代表团没有立即对此做出评论。
 <br/>
 <br/>
 黑利，1972年1月20日出生于美国南卡罗来纳州，印度裔美国人。2011年起担任南卡州州长，并成为美国最年轻的州长。
 <br/>
 <br/>
 2017年1月24日，即川普就职典礼后第4天，美国参议院确认黑利为第29任美国常驻联合国大使。
 <br/>
 <br/>
 黑利在2016年大选共和党内部初选时曾批评川普，但川普就任后，她成为川普政府的核心人物之一，在伊朗、朝鲜及其它国际问题上努力推进川普的政策和目标，是美国政府内的强硬派成员。
 <br/>
 <br/>
 在联合国工作期间，黑利尖锐批评联合国管理不善以及对以色列存在偏见。她促成了美国削减对联合国所分摊的会费，并退出联合国人权理事会（U.N. Human Rights Council）。
 <br/>
 <br/>
 黑利批评，联合国人权理事会是〝政治偏见的污水坑〞。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
