### 美军边境演习  警告难民勿涉险闯关
------------------------

<div class="wysiwyg">
 【新唐人2018年11月10日讯】再来关注中美洲大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 情况。目前，已有数百大篷车难民向美墨边境挺进。另有数千难民可能在周末跟进。周五，美军在美墨边境举行演习。美国边防官员警告继续北上的大篷车难民，不要冒险硬闯美墨边境，即使进入美国，也只会遭遇逮捕和遣返。
 <br/>
 <br/>
 美军周五在靠近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 城市华雷斯城的边境，举行演习。视频显示，直升机飞过连接美国和墨西哥的一座步行天桥。
 <br/>
 <br/>
 士兵随后从直升机上跳下，使用烟雾弹等武器执行任务。
 <br/>
 <br/>
 这座步行天桥是连接美国艾尔帕索市与墨西哥华雷斯城的边境通道。许多期望进入美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 在此驻扎。
 <br/>
 <br/>
 对于美军的演习，有墨西哥当地居民表示理解。
 <br/>
 <br/>
 华雷斯城居民Raul Mandujana：〝我认为这是美国政府的预防措施，虽然给华雷斯城带来一些混乱，但这是必要的。〞
 <br/>
 <br/>
 美国总统川普周五发布公告，在90天内，禁止从墨西哥非法入境美国的难民申请庇护。
 <br/>
 <br/>
 目前，已有数百大篷车难民率先离开墨西哥城，向美墨边境挺进。另外数千难民可能会在周末跟进。
 <br/>
 <br/>
 美国边防官员对此警告说，不要冒险非法进入美国。
 <br/>
 <br/>
 圣地亚哥边巡长官罗德尼・斯科特：〝任何翻过护栏，不经过入境口岸穿越边境的人，都面临逮捕和全面的法律诉讼，这些人（大篷车）也不例外。〞
 <br/>
 <br/>
 圣地亚哥边巡长官斯科特警告说，墨西哥已经为大篷车难民提供合法庇护身份，他们如果进入美国希望得到庇护，将反而面临起诉和遣返。
 <br/>
 <br/>
 斯科特希望这些难民慎重考虑，不要将他人，尤其是孩子置于危险的旅程之中。
 <br/>
 <br/>
 新唐人记者林宇、尚敬综合报导
</div>
