### 川普批通用汽车美国撤厂 瞄准小汽车〝鸡税〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】周三（11月28日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 连发推文评论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通用汽车.html" target="_blank">
  通用汽车
 </a>
 公司从美国撤厂问题，并分析指出，如果美国对进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小汽车.html" target="_blank">
  小汽车
 </a>
 征收〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸡税.html" target="_blank">
  鸡税
 </a>
 〞，那么通用汽车事件也许就不会发生。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通用汽车.html" target="_blank">
  通用汽车
 </a>
 （General Motors）决定关闭其设于俄亥俄、密歇根和马里兰州的汽车制造工厂，引发了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统的极大不满。周二晚，川普已发推指出，自己对通用汽车非常失望，因为他们停止了一些在美国的业务，却保留了设于墨西哥和中共的工厂。
 <br/>
 <br/>
 川普愤而表示，如果通用汽车不想把工作岗位留给美国，那么他将砍掉给通用汽车公司的所有补贴，因为那是美国纳税人资助的，而且他也要保护美国工人的利益。

 周三早上，川普在对相关问题进一步分析后，就美国的关税制度提出自己的看法。
 <br/>
 <br/>
 他指出，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_轻型卡车.html" target="_blank">
  轻型卡车
 </a>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸡税.html" target="_blank">
  鸡税

 〞。如果美国对进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小汽车.html" target="_blank">
  小汽车
 </a>
 也这样做，那么将有更多的小汽车在美国制造，而通用汽车也不会关闭其在国内的工厂。

 <br/>

 <br/>
 川普所说的〝鸡税〞（Chicken Tax，也叫鸡肉关税），是指的二战后欧洲国家对美国鸡肉所征的高额关税，旨在打击美国廉价鸡肉的出口、保护本地鸡肉的消费。上世纪60年代初，这一措施给美国家禽业造成数千万美元的损失。
 <br/>
 <br/>
 1963年，美国总统林登．约翰逊（Lyndon B. Johnson）宣布，对包括轻型卡车在内的4种进口商品征收25%的关税，以回应欧洲国家对美国鸡肉征收高额关税，这就是所谓的〝鸡税〞。
 <br/>
 <br/>
 这项措施的结果是，世界各地的轻型卡车在美国的销量都大受影响。此后几十年来，美国对轻型卡车所征收的〝鸡税〞一直保留。
 <br/>
