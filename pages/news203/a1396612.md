### 患失智症 美最高法院首位女大法官宣布退出公众活动
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】美国最高法院首位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女性大法官.html" target="_blank">
  女性大法官
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧康纳.html" target="_blank">
  欧康纳
 </a>
 12年前退休后，一直关注公民教育。周二（23日）她宣布自己正在与失智症搏斗，医生说她可能患有阿兹海默症。她已〝无法再参与公开活动〞。
 <br/>
 <br/>
 《纽约时报》（NYT）报导，现年88岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧康纳.html" target="_blank">
  欧康纳
 </a>
 （Sandra Day O'Connor）23日在一封公开信中称，日前医生告诉她患了早期失智症，可能是阿兹海默症。她说，近日自己身体状况愈来愈差，以后可能不能出席公开活动了。
 <br/>
 <br/>
 她说，她将继续与她挚爱的朋友和家人在亚利桑那州凤凰城（Phoenix）生活。
 <br/>
 <br/>
 信中写道：〝尽管我生命的最后一个章节可能会因为罹患失智症而有些难堪，但对此生受到无数的恩赐，我心中的感谢与感激之情丝毫没有减少。〞
 <br/>
 <br/>
 欧康纳说，她从没想过，〝一个来自亚利桑那州沙漠地区的年轻女牛仔，有朝一日会成为美国最高法院第一位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女性大法官.html" target="_blank">
  女性大法官
 </a>
 〞。
 <br/>
 <br/>
 她说：〝我希望我鼓舞了年轻人参与公民活动，也帮助了可能在求职生涯遇到障碍的女性铺平道路。〞
 <br/>
 <br/>
 就在欧康纳发表公开信的前一天，美联社报导称，欧康纳的儿子杰‧欧康纳( Jay OConnor)说，母亲现在常常有短期记忆方面的困扰，而且髋部问题让她大多数时间都必须坐轮椅，她如今已经很少出远门。
 <br/>
 <br/>
 欧康纳毕业于史丹佛法学院(Stanford Law School)，在亚历桑纳州参议院期间也是州内首位女议长。
 <br/>
 <br/>
 于1981年9月获时任共和党总统里根（Ronald Reagan）提名，她正式成为美国史上第一位最高法院女性大法官。
 <br/>
 <br/>
 担任最高法院大法官期间，对于有关平权法案、选举献金法案等都具有关键影响。
 <br/>
 <br/>
 2005年，欧康纳宣布她的先生约翰‧欧康纳三世(John OConnor III)患有阿兹海默症，身体状况非常不好。她于2006年1月退休照顾约翰。之后，她投身自己创立的非营利组织iCivics，目标透过线上游戏和校园计划，协助推动公民教育。
 <br/>
 <br/>
 对于欧康纳宣布患有失智症，最高法院首席大法官罗伯兹 (John Roberts)发表声明称，听到这个消息他感到很难过。声明中称，虽然欧康纳将不再参加公开活动，〝但无论什么疾病或状况，都带不走她曾经披荆斩棘，给后人开辟道路的启发意义。〞
 <br/>
 <br/>
 目前美国最高法院9席大法官中有3位是女性，分别是金斯柏格（Ruth Bader Ginsburg）、索托玛约（Sonia Sotomayor）和凯根（Elena Kagan）。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
