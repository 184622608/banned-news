### 蓬佩奥访朝有进展 川普：期待再会金正恩
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）周日（10月7日）发推文表示，国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 和朝鲜（北韩）领导人金正恩会面，就新加坡〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川金会.html" target="_blank">
  川金会
 </a>
 〞达成的协议取得进展，他期待在不久后和金正恩再次见面。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 稍早在推文中还贴出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 当地时间7日与金正恩会谈时的照片。
 <br/>
 <br/>
 蓬佩奥周日结束对东京的访问后前往平壤，金正恩亲自相迎。两人进行了两个小时的私人闭门会议。
 <br/>
 <br/>
 在私下会谈后，金正恩在平壤的百花园国宾馆设宴款待蓬佩奥，并说他〝非常高兴〞跟蓬佩奥在这样一次〝愉快的会议〞之后共同进餐。
 <br/>
 <br/>
 金正恩说这已经是蓬佩奥第4次来平壤访问。他相信蓬佩奥〝不会有第一次来时的陌生感〞。蓬佩奥表示同意并称，〝今天早上我们的访问非常成功，川普总统发来了问候，我们早上（谈话）非常成功。〞蓬佩奥说。
 <br/>
 <br/>
 金正恩则表示这是〝非常美好的一天，这将为两国带来美好的未来〞。
 <br/>
 <br/>
 蓬佩奥表示双方继续朝着朝鲜实行无核化方向迈进。
 <br/>
 <br/>

 <br/>
 陪同蓬佩奥访问平壤的一名美国官员周日表示，蓬佩奥这次访问朝鲜的成果〝超越上次〞，达成一些进展，但还须进一步努力，〝未来会是一段漫长而艰辛的路。〞
 <br/>
 <br/>
 官员没有具体说明朝鲜是否提出哪些具体承诺。
 <br/>
 <br/>
 这是蓬佩奥3月以来第4度访问平壤。他原本计划8月下旬成行，但川普在最后一刻以非核化问题迟无进展为由，要求蓬佩奥取消行程。
 <br/>
 <br/>
 金正恩6月12日在新加坡与川普进行首次会谈时承诺，努力实现无核化，但平壤的行动并未达到华盛顿要求其放弃核武器的步骤，包括申报所有核设施和导弹设施清单。
 <br/>
 <br/>
 蓬佩奥近日表示，对朝鲜的国际制裁必须保持原样，直到其放弃核武器。
 <br/>
 <br/>
 在结束平壤之行后，蓬佩奥于周日晚些时候抵达韩国、会见韩国总统文在寅，并与韩国外交部长康京和举行会谈。
 <br/>
 <br/>
 当天韩国总统办公室表示，蓬佩奥跟朝鲜达成协议，要尽快举行第二次〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川金会.html" target="_blank">
  川金会
 </a>
 〞。韩方说， 预计美朝双方将很快举行工作会谈，来确定川金会的细节，如日期和地点等。
 <br/>
 <br/>
 周一，蓬佩奥将前往此次东北亚之旅的最后一站、北京。
 <br/>
 <br/>
 日前，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 发表重磅讲话，重整美中政策，细数中共以全方位手段，试图影响美国政治与民主体制，破坏国际规则与秩序，美中关系紧张情势不断升高。专家认为，彭斯的讲话矛头直指中共，同时表达了对中国人民的关切。
 <br/>
 <br/>
 蓬佩奥在这样的背景下访华，格外引发关注。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
