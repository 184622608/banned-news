### 加州侨报办公楼爆枪击案 传死者为董事长谢一宁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 阿罕布拉一家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人媒体办公楼.html" target="_blank">
  华人媒体办公楼
 </a>
 ，周五（11月16日）发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 。一名华人男子被枪杀，56岁的枪手被捕。有消息称，死者是该报董事长谢一宁。
 <br/>
 <br/>
 据帕萨迪纳当地媒体pasadenastar news网站报导，当地时间周五早上9点半之前，有人报警称，在阿罕布拉（Alhambra）2121 W. Mission Road上发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 。警方尚未确认该媒体出版物的名称，但地址显示是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侨报.html" target="_blank">
  侨报
 </a>
 〞（China Press）办公楼。
 <br/>
 <br/>
 当警方抵达枪击案现场后，拘留了现场的几个人，但后来确认枪手是其中的56岁的陈中奇（Zhong Qi Chen 音译）。警方说，在现场找到了枪手行凶的枪枝。
 <br/>
 <br/>
 <center>
 </center>
 <br/>
 警察卡尔（Steven Carr）表示，有一名男子在现场被宣布死亡，没有其他人受伤。
 <br/>
 <br/>
 卡尔说，〝目前没有任何其他受害者或任何其它暴力事件的证据。这似乎是一起职场暴力事件。我们正在调查心怀不满的员工与同事之间可能发生的纠纷。〞
 <br/>
 <br/>
 凶案发生后，洛杉矶媒体界深感震惊。有传死者为侨报最高主管，但卡尔说明，由于该案当前尚在调查之中。唯一能公布的消息，是枪手和受害者的关系为同事关系。
 <br/>
 <br/>
 据悉，死者为40～50岁的中年男子。至于枪手行凶动机等内情，警方仍须厘清案情。
 <br/>
 <br/>
 周五早上前来〝侨报〞上班的员工因为枪击案导致办公楼被封锁，只得站在外面等候。
 <br/>
 <br/>
 阿罕布拉市位于南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 洛杉矶以东14英里，是一个华人聚集的城市。该市华人人口占37%，华人是人口比例最多的一个族裔。
