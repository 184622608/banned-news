### FBI十大通缉首犯 波士顿黑帮老大命丧监狱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】美国联邦监狱管理局周三（30日）表示，现年89岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波士顿黑帮老大.html" target="_blank">
  波士顿黑帮老大
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴尔杰.html" target="_blank">
  巴尔杰
 </a>
 在西维吉尼亚州联邦监狱内死亡，联邦调查局（FBI）正在调查其死亡原因。
 <br/>
 <br/>
 联邦监狱管理局（Federal Bureau of Prisons）称，没有其他囚犯或监狱人员受伤，但没有说明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴尔杰.html" target="_blank">
  巴尔杰
 </a>
 （James "Whitey" Bulger）的死因。
 <br/>
 <br/>
 美国多家媒体报导，巴尔杰可能是遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谋杀.html" target="_blank">
  谋杀
 </a>
 。据一名了解此事的监狱员工透露，联邦调查局以谋杀案作为调查。
 <br/>
 <br/>
 一名监狱的消息人士告诉《TMZ》，3名囚犯将巴尔杰的轮椅推到监视器拍不到的死角，用包在袜子里的锁头狠打他的头部，还试图挖出他的眼睛。监狱的一名员工说，巴尔杰前一天才坐在轮椅上被转移到西维吉尼亚州黑泽尔顿联邦监狱，就在狱中离奇身亡。
 <br/>
 <br/>
 巴尔杰的辩护律师布瑞南（Henry Brennan）表示，他无法证实或否认这些报导。
 <br/>
 <br/>
 据FBI资料显示，巴尔杰是波士顿知名爱尔兰裔黑帮〝冬山帮〞（Winter Hill）的老大，在1970到1980年代曾犯下多达19起重罪，包括谋杀、毒品买卖、敲诈勒索洗钱等刑事案件，被警方长年通缉。
 <br/>
 <br/>
 巴尔杰称霸波士顿黑社会25年。1994年12月，巴尔杰在接到FBI内鬼通知他即将被起诉后逃离波士顿，数个月后他的女友葛瑞格也离开波士顿与他会合。2002年泄密给巴尔杰的前FBI探员被判刑逾10年。
 <br/>
 <br/>
 巴尔杰名列FBI十大通缉要犯，悬赏金仅次于全球头号恐怖分子欧萨玛．本拉登（Osama bin Laden），本拉登遭击毙后，升格为十大要犯之首。
 <br/>
 <br/>
 期间，全球许多地方都传出民众见到巴尔杰，最可靠的一次是在2002年，巴尔杰与女友在伦敦旅游。据称，他的足迹遍布美国、欧洲、加拿大及拉丁美洲。
 <br/>
 <br/>
 巴尔杰逃亡16年后，于2011年6月落网，2013年8月陪审团裁定其谋杀、敲诈、洗钱、贩毒以及非法藏械等31项罪名成立，其中包括有涉嫌参与11起谋杀案，被判处两个无期徒刑外加5年监禁。
 <br/>
 <br/>
 而他生平为非作歹事迹也被名导演马丁史柯西斯（Martin Scorsese）2006年翻拍成警匪片〝神鬼无间〞（The Departed）。该片由杰克尼克逊（Jack Nicholson）、李奥纳多狄卡皮欧（Leonardo DiCaprio）及麦特戴蒙（Matt Damon）主演，还荣获奥斯卡最佳影片奖。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
