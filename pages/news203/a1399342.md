### 中期选举后选民〝公投〞 川普支持率再升2%
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】周三（11月14日）公布的一项最新民调显示，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 结束刚刚一周左右，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支持率.html" target="_blank">
  支持率
 </a>
 再度上升了2个百分点。这一被视作全民〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公投.html" target="_blank">
  公投
 </a>
 〞的民调结果说明，美国民众对川普的满意度处于上升趋势。
 <br/>
 <br/>
 据《国会山报》（The Hill）14日报导，由国会山电视台（Hill.TV）与HarrisX民调公司合作进行的民意调查发现，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 当前的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支持率.html" target="_blank">
  支持率
 </a>
 比
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 前又上升了2个百分点，已高达47%。
 <br/>
 <br/>
 其中，23%的受访者表示 〝强烈〞（strongly）支持川普，24%的人表示〝有些〞（somewhat）支持川普。
 <br/>
 <br/>
 根据该民调10月31日公布的调查结果，川普当时的支持率是45%。
 <br/>
 <br/>
 周三，美国政治新闻和民调数据整合网站RealClearPolitics（RCP）公布的数字很相近——川普的支持率为43%。
 <br/>
 <br/>
 中期选举之后的民调结果，通常可以被视作对现任总统前两年工作满意度的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公投.html" target="_blank">
  公投
 </a>
 〞。
 <br/>
 <br/>
 此次中期选举的结果，民主党重掌众议院，而共和党则保住了参议院多数党地位、且优势稍有扩大。
 <br/>
 <br/>
 川普在大选结束后曾指出，105年来，现任总统所在的党派能够保住参议院主导权的情况只有5次，因而，中期选举对共和党来说是成功的。
 <br/>
 <br/>
 参议院负责审核政府官员和司法人员的任命，其作用对政府机构的运行来说举足轻重。
 <br/>
 <br/>
 民主党民意测验专家西拉斯．李（Silas Lee）告诉国会山电视台，在很大程度上，这是一次公投——针对川普的工作和川普的个人表现及风度的公民表决。
 <br/>
 <br/>
 该民调是由国会山电视台与HarrisX民调公司于11月9日至10日对1,000名注册选民调查的结果，误差幅度不超过下上3.1%。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
