### 政绩卓然川普肯塔基竞选集会民情振奋
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 即将在11月6日举行。周六晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肯塔基州.html" target="_blank">
  肯塔基州
 </a>
 里士满，参加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竞选集会.html" target="_blank">
  竞选集会
 </a>
 为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党议员.html" target="_blank">
  共和党议员
 </a>
 站台，川普总统入主白宫两年来取得的政绩，受到了支持者们欢呼。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竞选集会.html" target="_blank">
  竞选集会
 </a>
 在东肯塔基大学的体育馆举行，热情选民坐满了会场。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：〝我们国家的失业率降到3.7%，是50年来的最低点。家庭收入中位数达到最高点，非常好，这意味着你们比以往任何时候都赚得更多的钱。〞
 <br/>
 <br/>
 在集会期间，川普总统还谈到了迈克尔飓风造成的破坏。他称赞联邦政府，执法部门，地方官员和联邦紧急事务管理局在受灾地区的恢复工作。
 <br/>
 <br/>
 美国总统川普：〝（我感谢）所有那些在实地支持灾后恢复工作的人员。我们有几千辆电动卡车，全部准备就绪，在跟进工作，清理道路，不容易。〞
 <br/>
 <br/>
 对于不久前卡瓦诺获大法官提名后在参院的听证过程，川普总统表示〝极端的民主党人对卡瓦诺法官所做的是个国家耻辱〞。
 <br/>
 <br/>
 川普表示，在他还没有对大法官人选做出决定之前，民主党就开始攻击，而无论是谁。
 <br/>
 <br/>
 美国总统川普：〝他们只是想阻挠，抵抗，拖延，和摧毁。民主党已经完全被他们对权力的欲望所吞噬。〞
 <br/>
 <br/>
 在11月6日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 上，美国选民将重选出众议院和参议院两院的议员。其中众议院所有435个议席都将重选，参议院则是100个席位中的35席将面临重选。中期选举的结果对当届总统余下任期的执政、以至两年后的大选都将产生不小的影响。
 <br/>
 <br/>
 新唐人记者严枫、天羽综合报导
</div>
