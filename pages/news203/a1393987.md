### 维基百科洩共和党议员资讯 美国会〝内鬼〞被捕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】美国国会警方周三（10月3日）逮捕了一名民主党工作人员，指控他涉嫌上周在网上偷传共和党参议员的个人信息，以便给支持最高法院大法官提名人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 （Brett Kavanaugh）的议员施加压力。
 <br/>
 <br/>
 周三，国会警方证实，他们已逮捕27岁的国会民主党工作人员科斯科（Jackson A. Cosko），指控他涉嫌犯有威胁、干扰证人、未经授权访问政府电脑、身份盗窃、入室盗窃、非法进入、公布受限制的个人信息等罪行，触犯了联邦及当地法律。
 <br/>
 <br/>
 根据科斯科在社交媒体领英（LinkedIn）上的自我介绍，他是一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党职员.html" target="_blank">
  民主党职员
 </a>
 ，直到今年早些时候，一直在参议院工作。网上记录显示，他是新罕布什尔州(新罕布夏州)民主党参议员哈桑（Maggie Hassan）的法律通讯员。
 <br/>
 <br/>

   </font>
  </a>
 </center>
 <br/>
 上个月，他还被列入德州民主党众议员希拉‧杰克逊‧李（Sheila Jackson Lee）的员工联系名单中。
 <br/>
 <br/>
 李公开反对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 ，并且出席了上周卡瓦诺和其指控者、加州教授福特（Christine Blasey Ford）的听证会。她上周还因向福特的律师递交了一张纸条而引起轰动。
 <br/>
 <br/>

    图为，德州民主党众议员希拉‧杰克逊‧李。（MANDEL NGAN/AFP/Getty Images)
   </font>
  </a>
 </center>
 <br/>
 近几周来，已有好几位共和党参议员的个人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_资讯.html" target="_blank">
  资讯
 </a>
 ，包括家庭地址、手机号码等被公布到网上。尤其是上周四，卡瓦诺的听证会结束后不久，许多共和党参议员的信息便被公布在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维基百科.html" target="_blank">
  维基百科
 </a>
 上，以便激进的自由派人士向这些议员施压、阻挠卡瓦诺的确认。
 <br/>
 <br/>
 目前信息遭到洩露的共和党参议员包括参议院多数党领袖麦康奈尔（Mitch McConnell）、南卡罗来纳州参议员格雷厄姆（Lindsey Graham）、犹他州的两名参议员哈奇（Orrin G. Hatch）和麦克‧李（Mike Lee）等。
