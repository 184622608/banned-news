### 川普：俄罗斯违反协议 美国退出《中导条约》
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】美国总统川普周六宣布，美国正在退出30多年前与俄罗斯签署的《中程导弹条约》，因为俄罗斯多年来一直违反协议。
 <br/>
 <br/>
 周六，川普结束在内华达州的竞选集会后，登上空军一号之前表示，俄罗斯多年来，一直没有遵守《中程导弹条约》，而美国一直单方面执行协议。
 <br/>
 <br/>
 美国总统 川普：〝俄罗斯违反协议。我们不会让他们违反核协议，去发展武器，却不允许我们这样做。我们一直遵守协议，我们执行了协议，但俄罗斯没有遵守协议，很遗憾，所以我们将停止协议，我们将退出。〞
 <br/>
 <br/>
 川普还说，除非俄罗斯、中国和美国一起协商，都不研发这些武器，否则美国只能退出这一条约。
 <br/>
 <br/>
 《中程导弹条约》是1987年12月，美国前总统里根和苏联前总书记戈尔巴乔夫签署的。
 <br/>
 <br/>
 条约规定，美国和苏联双方全部销毁所拥有的中程导弹、中短程导弹及其发射装置和辅助设施，双方不得再生产、试验中程导弹和中短程导弹，以缓解冷战时期的紧张气氛。
 <br/>
 <br/>
 该条约使中共成了冷战末期全球唯一能出口该类型导弹的政权。1988年，中共与沙特阿拉伯达成目前唯一一笔中程导弹国际贸易。
 <br/>
 <br/>
 白宫官员认为，《中程导弹条约》不允许美国开发新武器，使美国处于不利地位，而中共正不受约束地研发和部署相关武器，威胁到美国的安全。
 <br/>
 <br/>
 下周，白宫国家安全顾问博尔顿访问莫斯科时，将向俄罗斯领导人通报退出条约事宜。
 <br/>
 <br/>
 新唐人记者易文、天羽综合报导
