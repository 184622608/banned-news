### 哈佛被控歧视亚裔案开审 数百亚裔学生上街举牌声援
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】美国名校
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈佛大学.html" target="_blank">
  哈佛大学
 </a>
 被指在招生过程中歧视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚裔学生.html" target="_blank">
  亚裔学生
 </a>
 的案件，备受各界瞩目。该案于周一（15日）在波士顿地区联邦法院开审，预计将审理3周时间。开审前，美国多个民间组织在14日发起集会，举牌声援
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚裔.html" target="_blank">
  亚裔
 </a>
 学生。
 <br/>
 <br/>
 大纪元报导，〝录取公平学生组织（Students for fair admissions, SFFA）〞控告
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈佛大学.html" target="_blank">
  哈佛大学
 </a>
 歧视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚裔.html" target="_blank">
  亚裔
 </a>
 一案，要求哈佛及其他大学依据个人能力而非种族因素录取学生。
 <br/>
 <br/>
 10月14日中午，以SFFA和〝亚裔美国人教育联盟（Asian American Coalition for Education，AACE）〞为首的亚裔组织和社区代表，在波士顿科普利广场（Copley Square）举行集会。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/10-16/p9082042a394468959.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-16/p9082042a394468959-ss.jpg"/>
   <br/>
   <font size="-1">
    AACE主席赵宇空（左）感谢SFFA创办人兼主席布朗姆（Edward Blum，右）为对亚裔教育平权的帮助。（刘景烨／大纪元）
   </font>
  </a>
 </center>
 <br/>
 周日上午12点，数百名SFFA的支持者齐聚科普利广场，举起〝我有美国梦〞、〝哈佛放弃种族刻板印象〞等标语，抗议哈佛大学的招生政策。包括AACE主席赵宇空、联邦参议员独立候选人希瓦（Shiva Ayyadurai）在内的多位亚裔社区代表登台演说。一些亚裔大、中学生及哈佛学生代表也分享了他们的〝美国梦〞和对亚裔受到歧视问题的看法。
 <br/>
 <br/>
 SFFA创办人兼主席布朗姆（Edward Blum）说，〝我们并非在此批判哈佛或其他大学。我们聚在这里，是因为哈佛的录取政策存在歧视。〞
 <br/>
 <br/>
 布朗姆提到，亚裔申请人一般拥有较高的学术分、课外活动评分、教师评分等成绩，却常常在主观性较强的〝个人评分〞，例如积极性格、勇气等评估中被打低分。SFFA分析了哈佛大学6年的录取数据，以及数千份电子邮件及文件，认为哈佛大学在录取过程中有意歧视亚裔申请人。
 <br/>
 <br/>
 布朗姆认为，哈佛设置了种族配额，把种族因素当作了录取过程中的〝主导因素〞。若这一指控属实，哈佛则违反了2013年最高法院关于种族与大学录取的判决（Fisher v. Univ. of Texas-Austin）。
 <br/>
 <br/>
 布朗姆举例说，在20世纪20年代，犹太学生曾一度占据哈佛新生人数的1/4。当时的哈佛校长在一封信中表示对犹太学生泛滥的担忧。哈佛因而开始采用〝综合录取政策〞，并在一年内降低了犹太学生的比例。布朗姆认为，现在这种政策正被用来针对亚裔。
 <br/>
 <br/>
 布朗姆说，〝你的种族不应被利用来帮助或伤害你人生中的努力。你的种族不应成为求职、升职的因素，不应影响学生申请大学。〞
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/10-16/p9082043a423453109.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-16/p9082043a423453109-ss.jpg"/>
   <br/>
   <font size="-1">
    印度裔高等教育谘询师Vijay Jojo Chokalingam。（刘景烨／大纪元）
   </font>
  </a>
 </center>
 <br/>
 印度裔高等教育谘询师Vijay Jojo Chokalingam则激动地说，根据美国医学院协会（American Association of Medical Colleges）的数据，同样具备GPA 3.1和AMCAT 31分，亚裔申请人被学校录取的概率只有18%；白人概率为28%，拉美裔为48%，非裔则高达76%。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/10-16/p9082044a411149239.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-16/p9082044a411149239-ss.jpg"/>
   <br/>
   <font size="-1">
    一位华人在科普利广场呼吁哈佛取消亚裔配额。（刘景烨／大纪元）
   </font>
  </a>
 </center>
 <br/>
 来自宾州的侯先生到美国数十年，他认为，保证优秀的亚裔学子能得到最好的教育，并不只为亚裔权益，也是为了美国国家的长远发展。侯先生说，〝运动员参加奥运会、宇航员上太空时，你不会考虑他们代表什么族裔，你只要他们是最优秀的。〞
 <br/>
 <br/>
 从纽约赶来声援的高先生表示：〝党派提出‘多元化’是为了赢得其他族裔的选票，真正的精英却因此无法得到最好的教育，其实是破坏了整个美国的教育系统。〞
 <br/>
 <br/>
 从大陆来旅游的郭先生被集会吸引驻足。他表示，在大陆，民众能走上街头发声的机会非常少。他说，〝中国人要有不同的声音，一些不对的事情才能被及时的纠正和制止。〞
 <br/>
 <br/>
 同日，哈佛大学〝反歧视行动（Affirmative Action）〞的一百多名支持者们也在剑桥市的哈佛广场集会，支持大学采取〝包容录取政策〞。
 <br/>
 <br/>
 哈佛大学越南人协会的主席Thang Diep在欢呼声中登台。他分享了自己从越南移民到美国后，饱受歧视和心理创伤的成长历程，并表示〝反歧视行动〞正是对不同族裔之历史背景的理解和包容。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/10-16/p9082045a581836115.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-16/p9082045a581836115-ss.jpg"/>
   <br/>
   <font size="-1">
    哈佛大学越南人协会的主席Thang Diep在哈佛广场分享自己艰辛的成长经历。（刘景烨／大纪元）
   </font>
  </a>
 </center>
 <br/>
 Thang Diep说，〝支持‘反歧视行动’意味着积极挑战刻板印象，认真检视亚裔美国人中，不同民族群体和不同经济背景群体的差异。这也表明，不同的移民历史，会带来不同的艰苦抗争。〞
 <br/>
 <br/>
 6月15日，SFFA针对哈佛大学入学政策对美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚裔学生.html" target="_blank">
  亚裔学生
 </a>
 存在歧视一案，提交证据。15日该案在波士顿联邦地方法庭就将开始审理，预计将审理3周时间。
 <br/>
 <br/>
 该案涉及到美国平权行动政策(affirmative action, 简称AA)的适用问题，预计将对美国的教育系统有重要影响。
 <br/>
 <br/>
 哈佛大学的律师团队对此回应称，这起诉讼则代表了对其他许多大学的〝攻击〞。这些大学或许认为考虑申请人种族因素只是招收多元化学生的一种方式。
 <br/>
 <br/>
 （记者李红报导/责任编辑：范铭）
