### 章莹颖案新进展:嫌犯扮警察诱拐  浴室发现证据
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】时年26岁的中国女学者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_章莹颖.html" target="_blank">
  章莹颖
 </a>
 在美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇害案.html" target="_blank">
  遇害案
 </a>
 的调查工作，日前再有新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进展.html" target="_blank">
  进展
 </a>
 。有线民向检方提起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新证词.html" target="_blank">
  新证词
 </a>
 ，爆出嫌犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克里斯滕森.html" target="_blank">
  克里斯滕森
 </a>
 作案时的更多细节。
 <br/>
 <br/>
 联邦检察官表示，这位线民是梅肯郡监狱（Macon County Jail）的一名囚犯，2017年时，他与凶嫌布伦特．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克里斯滕森.html" target="_blank">
  克里斯滕森
 </a>
 （Brendt Christensen）被关押在一起，两人因此建立了关系。后来，相关执法部门才从这位线民口中取得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新证词.html" target="_blank">
  新证词
 </a>
 。
 <br/>
 <br/>
 检方表示，克里斯滕森曾亲口告诉线民：〝我（当时）向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_章莹颖.html" target="_blank">
  章莹颖
 </a>
 出示了一枚警徽，并告诉她自己是警察，来诱使章莹颖上车。〞
 <br/>
 <br/>
 另外，今年8月，辩方律师还指出，警犬在克里斯滕森公寓的浴室发现了尸体反应，律师会请专家作证上述发现是否与章莹颖的DNA与血型吻合。
 <br/>
 <br/>
 检察官表示，截至目前，辩方律师还没有提供专家证词，因此无法回应。
 <br/>
 <br/>
 <h4>
  事件回顾：
 </h4>
 <br/>
 美国伊利诺大学香槟分校的中国访美学者章莹颖，2017年6月9日下午乘坐香槟-厄巴纳公共交通区厄巴纳线城巴前往校外公寓，计划签署公寓租约。当她在位于北古德温大道（North Goodwin Avenue）和西克拉克街（West Clark Street）拐角处的一个公交站等待公交车时，搭乘了一辆黑色土星轿车，之后便失去一切音讯与踪迹。
 <br/>
 <br/>
 在案发3天后（6月12日），联邦调查局（FBI）寻找到该犯罪车辆，但嫌犯克里斯滕森一直否认自己曾出现在案发现场。
 <br/>
 <br/>
 6月16日，FBI寻求其女友T.E.B.秘密协助，并为她佩戴了窃听装置。在T.E.B.提供的一段录音中，嫌犯克里斯坦森提到了自己绑架并杀害了章莹颖，并描述了章莹颖遭到袭击时是如何反抗的；而另一段录音则是嫌犯和女友参加章莹颖的平安祈祷会时录制的。他当时甚至向女友指出了人群中的哪些人，是自己的〝理想受害者人选〞。
 <br/>
 <br/>
 在祈祷会次日（6月30日），克里斯坦森被FBI以绑架罪名逮捕。
 <br/>
 <br/>
 7月12日，美国联邦大陪审团正式以绑架案起诉克里斯滕森。起诉书指控被告克里斯滕森在案发当天〝故意且非法地抓走、监禁、诱骗、绑架了章莹颖，之后出于自己的利益和目的，将章莹颖困住并使用恶劣的手段进行进一步侵害。〞
 <br/>
 <br/>
 随着调查的深入，在掌握了更多证据材料后，检方决定将该案改为以〝谋杀罪〞起诉，并对克里斯滕森寻求死刑判处。
 <br/>
 <br/>
 10月3日，斯普林菲尔德联邦大陪审团重新起诉克里斯滕森。新的起诉书指控被告克里斯滕森三项罪名：第一项是绑架致人死亡罪，另外两项是向联邦调查局特工提供虚假陈述。
 <br/>
 <br/>
 大陪审团还特别针对第一项指控发布了特别调查结果通知书，指控被告在实质策划和预谋后蓄意谋杀了章莹颖，而且采用了〝令人发指的、残忍的、道德败坏的手段犯下罪行〞，并指控嫌犯〝酷刑或身体上严重虐待受害人〞。
 <br/>
 <br/>
 上述两项向联邦调查局特工作虚假陈述的罪名若被定罪，可判处五年监禁；而如果绑架致人死亡罪被定罪，被告将面临死刑或终生监禁。
 <br/>
 <br/>
 今年1月19日，在联邦司法部长杰夫．塞申斯（Jeff Sessions）指令下，联邦检察官提交了对此案嫌犯寻求死刑的意向书，并宣布将在本案中寻求死刑判决。
 <br/>
 <br/>
 在这份声明中，检察官宣布，他们怀疑克里斯滕森除杀害章莹颖外，还参与了2013年伊利诺伊州中部一起伤害与性侵案件，并称克里斯滕森曾承认〝有不止一名受害者〞，他甚至还表示过自己〝渴望以杀手身份留名〞。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
