### 中期选举落幕尘埃未定   佛州现计票风波
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】
 <b>
  新闻周刊
 </b>
 （653）11月6号，美国举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，最终
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 成功保住参议院多数党地位，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 则在众议院翻盘成为多数党。尽管选举已经过去4天，但在佛罗里达、乔治亚和亚利桑那州，四场关键的参议员和州长选战相继爆出计票争议，也让中期选举迟迟无法尘埃落定。
 <br/>
 <br/>
 11月6号的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，改选国会众议院全部435席，参议院100席中的35席。在州长级官员选举中，将改选36个州和3个美国领地共39位州长级官员，以及6600多个州立法机构席位。
 <br/>
 <br/>
 此次选举的最大看点，在于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 能否保住众议院和参议院的多数党席位。不过，中期选举历来存在〝钟摆效应〞，新任总统所在党通常很难在中期选举中同时赢得参众两院。
 <br/>
 <br/>
 截止到周六更新的最新结果，共和党在参议院赢得51席，保住多数党地位，并有希望扩大席位优势。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 和独立党派共赢得46席，另有三个席位仍未确定最终结果。
 <br/>
 <br/>
 众议院方面，民主党已经赢得435席中的226席，成为新的多数党。共和党赢得198席。尚有11个席位没有公布最终结果。
 <br/>
 <br/>
 参加了44场助选活动的美国总统川普，认为中期选举对他是个巨大的成功。
 <br/>
 <br/>
 周三早晨，川普发推文说，〝收到很多人的众多祝贺，祝贺我们昨晚取得重大胜利，包括来自等待和期待达成贸易协议的国外友人。〞
 <br/>
 <br/>
 周三中午，川普举行了两个半小时的记者会，阐述对选举结果的看法以及未来的施策方针。
 <br/>
 <br/>
 川普强调了共和党在参院扩大席位优势的重要性，认为这对未来两党之间的协商与合作有利。
 <br/>
 <br/>
 美国总统 川普：〝从未来的协商角度来看，从达成协议的角度看。如果我们所占的多数只多出1、2票，反而容易陷入僵局。我确信，可以和民主党很好的合作，可以提出很多好的提案，获得两党通过。〞
 <br/>
 <br/>
 民主党众议院领袖佩洛西，在民主党赢得众议院后发表讲话，其中也提到会寻求两党合作，巩固美国的民主制度。
 <br/>
 <br/>
 民主党众议院领袖 南希・佩洛西：〝我们会问责，会争取两党合作，为各方实现公平。我们有责任找到共识，在可能的情况下立足于这些共识，我们必须去尝试。〞
 <br/>
 <br/>
 相较以往，此次中期选举投票率大幅上升，从过去的40-42%，提高到46-48%。专家认为，最终结果两党可谓各有所获。
 <br/>
 <br/>
 美国企业研究所美国政治研究员 Michael Barone： 〝民主党赢得众议院多数，这很重要，让他们有机会推动其政策，可抵制其所不同意的川普政策。共和党赢得更多参议院席位，让他们更容易确认法官提名，包括最高院大法官，如果再有空缺，总统会再提名。〞
 <br/>
 <br/>
 这次选举中，川普成为推动选民出来投票的重要动力。三分之二选民表示，投票时有将川普作为一个考虑因素。同时，选举结果再次反映出美国城乡之间存在重大的政治分化。
 <br/>
 <br/>
 跨党派政策中心民主项目主任 John Fortier：〝响村地区越来越倾向共和党，郊区和城市则更倾向民主党。〞
 <br/>
 <br/>
 另一方面，在佛罗里达州，现任共和党州长斯科特已经认定自己以微弱优势获胜，赢得联邦参议员席位。但此时却爆出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_计票风波.html" target="_blank">
  计票风波
 </a>
 。
 <br/>
 <br/>
 周四下午，斯科特下令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 执法部门调查布劳沃德郡和棕榈滩郡的选举机构，为何在投票截止48小时后，仍在收集和清点选票，而且新计入的选票都流向民主党参选人。
 <br/>
 <br/>
 以斯科特本人参与的联邦参议员选战为例，周二晚斯科特领先对手、现任民主党参议员纳尔逊约5万多张选票而宣布获胜；然而之后二人选票差距不断缩小，到周四下午，双方只差1万5千多张票，斯科特仅领先纳尔逊0.18%。
 <br/>
 <br/>
 佛州州长 斯科特：〝自普选日后的每一天，布劳沃德郡的左翼活动者不知从哪弄来越来越多的选票。他们的目标是偷偷的增加选票直到选举结果朝他们想要的发生变化。〞
 <br/>
 <br/>
 除了佛州参议员选举之外，佛州州长、乔治亚州州长、亚利桑那州参议员选战，也都出现了类似的计票争议。这也使得共和党宣布在参议院扩大席位优势的局面，仍然迟迟未能兑现。
 <br/>
 <br/>
 分析认为，计票机制牵扯美国民主制度的根基。计票过程如果出现舞弊，其影响甚至可能超过中期选举的结果本身。
 <br/>
 <br/>
 撰稿：林宇
