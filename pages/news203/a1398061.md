### 第二轮对伊朗制裁 美称将稳定国内油价
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 接受福克斯新闻采访，重申美国恢复对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗制裁.html" target="_blank">
  伊朗制裁
 </a>
 的原因和目的，他强调，要确保美国消费者的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 ，不受对伊朗制裁的影响。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 ：这个恐怖的政权，威胁着以色列，通过资助黎巴嫩真主党，这个政权还继续资助胡塞武装，（使其）能够向利雅德和迪拜发射导弹，这都要停止。这种行为必须改变。美国制裁将在今天（周日）午夜生效。
 <br/>
 <br/>
 周五，美国国务院宣布，于11月5日起全面恢复对伊朗的制裁。国务卿蓬佩奥表示，这是迄今为止针对伊朗政权最严厉的制裁，而制裁只有一个目的，就是抑制这个全球头号援助恐怖主义的国家干坏事的能力。
 <br/>
 <br/>
 5月8日，川普总统宣布退出奥巴马政府于2015年主导并签署的《伊朗核协议》。
 <br/>
 <br/>
 川普政府认为，这项协议并没有抑制伊朗发展核武器的野心，相反却成了它的保护伞。而且伊朗同时还在援助恐怖组织。
 <br/>
 <br/>
 退出协议后，美国着手恢复对伊朗的制裁，并分两轮进行。
 <br/>
 <br/>
 8月7日，美国开始了对伊朗的第一轮制裁。包括限制伊朗购买美国货币，交易黄金等贵金属，以及限制其它国家销售汽车零部件、商用客机给伊朗。
 <br/>
 <br/>
 而11月5日生效的第二轮制裁，以限制伊朗销售原油、石化产品出口为主。但美国政府承诺，将保护美国消费者的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_油价.html" target="_blank">
  油价
 </a>
 不受冲击。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：确保美国消费者不受影响，（我们）会让布仑特原油与今年5月保持同样价格，即总统宣布退伊朗协议时价格等同。
 <br/>
 <br/>
 蓬佩奥表示，第二轮制裁生效前，一些国家已经开始行动，减少了购买伊朗原油。蓬佩奥透露，目前伊朗已经每天减少出口超过一百万桶原油，并将继续减少。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：有几个国家已大幅减少伊朗原油进口，但他们需要多些时间才能实现〝零〞进口，我们将对这类国家颁布豁免。
 <br/>
 <br/>
 第二轮制裁生效，意味着继续从伊朗进口原油，以及继续与受到制裁的伊朗机构有商务往来的国家会受到美国惩罚。美国并在原来制裁名单上，再增加了超过700多家伊朗实体及个人。
 <br/>
 <br/>
 但美方表示，给8个依赖伊朗原油的国家提供临时豁免。蓬佩奥专访时拒绝透露是哪8个国家，不过土耳其此前宣称自己是获得豁免的国家之一。
 <br/>
 <br/>
 本轮制裁，正值1979年11月4日伊朗人质危机周年。当时，伊朗新政权开始号召伊朗人反美反以色列，并在当天占领美国驻伊朗大使馆，66名美国外交官和平民被扣留为人质。所以这一天也被认为是美国和伊朗化友为敌的转折点。
 <br/>
 <br/>
 新唐人记者李大宇综合报导
