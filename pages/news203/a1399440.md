### 疑涉家暴  告川普成人片艳星律师被捕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】因帮成人片
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艳星.html" target="_blank">
  艳星
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风女.html" target="_blank">
  暴风女
 </a>
 丹尼尔斯与美国总统川普（特朗普）打官司而出名的律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艾维纳提.html" target="_blank">
  艾维纳提
 </a>
 ，周三（14日）在洛杉矶因被控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_家暴.html" target="_blank">
  家暴
 </a>
 遭逮捕。
 <br/>
 <br/>
 洛杉矶警局周三发推文称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艾维纳提.html" target="_blank">
  艾维纳提
 </a>
 （Michael Avenatti）下午已遭警方登记在案，保释金为5万美元。
 <br/>
 <br/>
 警方发言人警官Tony Im表示，案件中的受害者有明显的受伤情况。但拒绝透露受害人的相关信息。
 <br/>
 <br/>
 多名高级执法官员告诉NBC新闻，艾维纳提被捕是因一项重罪指控。到当地时间14日下午，艾维纳提仍被拘留，他的政治行动委员会发言人拒绝发表评论。
 <br/>
 <br/>

 根据名人八卦网站TMZ报导，艾维纳提遭控周二与身份不明女性发生争执，并在周三与对方第2次冲突后被捕。
 <br/>
 <br/>
 TMZ最初指称女方为艾维纳提的分居妻子丽莎史多利（Lisa Storie Avenatti），但她告诉美国国家广播公司（NBC），艾维纳提没有对她施行过暴力。
 <br/>
 <br/>
 47岁的艾维纳提在声明中否认有不法行为。如果被控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_家暴.html" target="_blank">
  家暴
 </a>
 罪成立，艾维纳提将面临最高4年的牢狱之灾。
 <br/>
 <br/>
 艾维纳提因帮成人片
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艳星.html" target="_blank">
  艳星
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风女.html" target="_blank">
  暴风女
 </a>
 丹尼尔斯（Stormy Daniels）在几个月内不停地追击川普及其亲信，并在受访时辱骂川普和在推文中设局坑川普及其律师而出名。他甚至还声称要竞选2020年总统。佛蒙特州民主党取消了定于16日和17日的举行活动，并开始接受退票。艾维纳提原定安排在这两天的活动中发表演说。
 <br/>
 <br/>
 丹尼尔斯是美国成人电影演员，真名叫斯蒂芬妮．克利福德。她声称2006年曾和川普发生关系。2011年，她准备在一次采访中讲述这段往事时，一名男子在拉斯维加斯的一个停车场威胁她，让她不要提这件事。克利福德还公布了男子的肖像素描。
 <br/>
 <br/>
 川普在推文中回应，克利福德的指控是骗局，〝多年前一个不存在的男子肖像素描，全是骗局，把假新闻媒体当傻子耍弄（但是他们都知道）！〞
 <br/>
 <br/>
