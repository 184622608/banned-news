### 记者被失踪 川普警告严惩 沙特股价开盘狂泻7%
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】对于美国《华盛顿邮报》特约专栏记者，疑似遭沙特阿拉伯当局跨国杀害一案，美国总统川普（特朗普）警告，如消息属实，将对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙国.html" target="_blank">
  沙国
 </a>
 施加〝严厉惩罚〞，沙国股市周日（14日）应声暴跌7%。
 <br/>
 <br/>
 川普在美东时间10月13日播出的哥伦比亚广播公司60分钟节目采访中表示，《华邮》记者卡舒吉（Jamal Khashoggi）生还的希望渺茫，假如调查发现沙特阿拉伯当局参与了卡舒吉的失踪事件，美国〝会非常沮丧和愤怒〞，将会对其进行〝严厉的惩罚〞。
 <br/>
 <br/>
 话音刚落，沙特阿拉伯证交所
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全股指数.html" target="_blank">
  全股指数
 </a>
 （TASI）周日（14日）早盘一度大跌7.02%至7001.94点，抹去今年来所有涨幅，且该指数186档成分股几乎全数下挫，仅3档个股幸免。创下2015年8月以来最大单日跌幅。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙国.html" target="_blank">
  沙国
 </a>
 股市以周日为每周第1个交易日，每周五和周六休市。TASI继上周四（10月11）收盘重挫3.88%，周日开盘再度暴跌，仅2个交易日该国股市市值就蒸发近500亿美元。
 <br/>
 <br/>
 伦敦Think Markets UK首席市场分析师Naeem Aslam表示，人们在谈论地缘政治情势甚至变得更加恶劣，且沙国再次展现出顽固的姿态。外国投资人不会坐以待毙。在他看来，目前投资人对沙国股票弃如敝屣。
 <br/>
 <br/>
 迪拜Think markets市场策略师Mohammed Zidan也表示，因卡舒吉失踪的不确定性因素，受政治和经济因素影响，沙国股市下跌是恐慌性抛售的结果。
 <br/>
 <br/>
 59岁的卡舒吉毕业于美国印第安纳州立大学，一向大胆公开批评沙特阿拉伯政府的政策，但也一直担心自己会遭受报复，一年前逃离了沙特。
 <br/>
 <br/>
 10月2日，卡舒吉前往沙特驻土耳其领事馆办理结婚文件，其后神秘失踪。土耳其当局相信，卡舒吉在领事馆内被绑架、谋杀；沙特方面则表示绑架指控毫无根据。沙特官员表示，卡舒吉已离开领事馆，但在领事馆外等他的成吉兹表示，卡舒吉后来就不曾再出现。
 <br/>
 <br/>
 美国和土耳其官员证实，土耳其政府已经把相关视频和录音证据交给美国，指证卡舒吉在伊斯坦布尔的沙特领事馆内遭到扣留、审问、虐打，最终被肢解杀害。而沙特一直否认绑架或暗杀了这名流亡记者。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
