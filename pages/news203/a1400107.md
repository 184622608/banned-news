### 芝加哥医院枪击案 四人死 一警察殉职
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】周一下午，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芝加哥.html" target="_blank">
  芝加哥
 </a>
 一家医院发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，枪手和一名警察在内的四人死亡，凶手身份还没有确认。警方初步断定，枪手作案动机与家庭暴力有关。
 <br/>
 <br/>
 周一下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芝加哥.html" target="_blank">
  芝加哥
 </a>
 市南部的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_慈爱医院.html" target="_blank">
  慈爱医院
 </a>
 〞发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 。
 <br/>
 <br/>
 枪手在医院外停车场射杀一名医生。警察随后赶到，与枪手交火。之后枪手冲入医院，射杀一名助理药剂师。双方再次交火，一名警察不幸殉职，枪手身亡。目前不清楚枪手是自杀还是被击毙。
 <br/>
 <br/>
 目击者Dennis Burke：〝人们从我身后沿街奔跑，每个人（都跑），医生还有好像是病人，多大年龄都有。真是吓人。〞
 <br/>
 <br/>
 据报道，遇害医生和药剂助理师分别是38岁的奥尼尔和25岁的莱斯。枪手据称是遇害医生的前未婚夫。
 <br/>
 <br/>
 周一晚间，芝加哥警方和救援队伍，以及大批市民举行仪式，悼念遇害警察和医护人员。一队警车开道，警察和悼念人群立在道路两侧默哀。
 <br/>
 <br/>
 殉职警察是金默内，现年28岁，去年初刚刚加入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芝加哥警察局.html" target="_blank">
  芝加哥警察局
 </a>
 。
 <br/>
 <br/>
 芝加哥警察局长表示，金默内是英雄，挽救了很多生命。
 <br/>
 <br/>
 芝加哥警察局局长EDDIE JOHNSON：〝毫无疑问，在我心中所有警察都是英雄，他们挽救了许多生命，因为我们不知道（枪手）准备伤害（多少人）。〞
 <br/>
 <br/>
 芝加哥市长RAHM EMANUEL：〝芝加哥市失去了一名医生、一名药剂助理师和一名警察。〞
 <br/>
 <br/>
 金默内是芝加哥警察局今年殉职的第二名警察。
 <br/>
 <br/>
 新唐人记者毕心慈、宇微综合报导
</div>
