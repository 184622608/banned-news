### 五角大楼报告 警告美军事物资对华依赖
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】五角大楼周五发布了一份报告，指出北京对关键技术和材料供应的控制，可能对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国家安全.html" target="_blank">
  美国国家安全
 </a>
 至关重要的产业构成重大风险。
 <br/>
 <br/>
 这项报告研究由美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 去年在行政命令中授权调查编写，确定了存在的〝漏洞〞，这些漏洞可能影响到支持美国军方必不可少的关键材料和其他物资的供应。
 <br/>
 <br/>
 报告称：〝中共侵略性的产业政策已经削弱了美国的一些防御性物资的能力，包括用于军事的太阳能电池，平板飞机显示器以及稀土元素的加工〞。
 <br/>
 <br/>
 其中，中国控制着世界绝大多数用于生产高科技防御设备的〝稀土材料〞。从2013年到2016年，中国占所有稀土进口美国的78％。根据美国地质调查局的数据，美国在2017年没有在国内开采任何稀土，而是进口了大约1.5亿美元。
 <br/>
 <br/>
 报告还指出：〝中共还严重威胁到美国的其他物资，包括机械工具、生物材料、陶瓷和复合材料等先进材料的生产和加工；以及印刷电路板和半导体的生产。〞
 <br/>
 <br/>
 白宫贸易顾问彼得纳瓦罗指出：〝对美国工业基础的核心威胁来自中共。〞
 <br/>
 <br/>
 同一天，美国国防部颁布声明，川普总统将采取措，以填补以上漏洞。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
