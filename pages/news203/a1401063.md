### 奥巴马圣诞树挂毛像遭批 爱国主义现重返白宫
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】周二（11月27日），美国自由派媒体纷纷批评第一夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼娅.html" target="_blank">
  梅拉尼娅
 </a>
 （Melania Trump）设计的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞装饰.html" target="_blank">
  圣诞装饰
 </a>
 。然而，一家保守派博客回顾了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 时期的一桩装饰丑闻——把带有中共杀人狂领袖毛泽东形象的装饰品挂到白宫圣诞树上。
 <br/>
 <br/>
 周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼娅.html" target="_blank">
  梅拉尼娅
 </a>
 向外界展示了她设计和布置的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞装饰.html" target="_blank">
  圣诞装饰
 </a>
 ，主题是〝美国宝藏〞（American Treasures），琳琅满目的装饰品多达数百种，重在体现美国的传统。
 <br/>
 <br/>
 梅拉尼娅在推特上宣布，人民之家——白宫已经做好准备，庆祝圣诞节及假日来临。
 <br/>
 <br/>
 但是，正如经常挑剔梅拉尼娅的着装一样，左媒对于第一夫人设计的白宫圣诞装饰也刻薄地指指点点，Slate 网站甚至将其中的红色圣诞树称为〝她的红色死亡圣诞树〞。
 <br/>
 <br/>
 对比白宫以往的圣诞装饰，美国保守派博客〝Gateway Pundit〞周一指出，2009年，时任总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 （Barack Obama）及第一夫人米歇尔（ Michelle Obama）所装饰的白宫才真正令人震惊。他们不仅把奥巴马的图像与最受美国人民爱戴的林肯总统等前辈的图像一样装饰在小球上并悬挂在相邻位置，甚至还挂上了带有杀人狂、前中共领袖毛泽东图像的装饰球。
 <br/>
 <br/>

 <br/>
 <br/>
 根据各媒体当年的报导，奥巴马在白宫圣诞树上挂中共独裁者图像的行为引发了巨大争议，甚至连自由派媒体《洛杉矶时报》（Los Angeles Times）都在博客中评论，这是巨大的〝圣诞装饰丑闻〞。
 <br/>
 <br/>
 但周一发表的白宫装饰视频显示，与去年一样，梅拉尼娅的装饰传统、美丽、典雅，今年甚至还挂上了带有爱国主义色彩的圣诞球——上面带有美国首任总统乔治．华盛顿（George Washington）的图像。
