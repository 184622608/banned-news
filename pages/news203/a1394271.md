### 美参议院结束辩论 卡瓦诺提名周六表决
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】观众朋友大家好，欢迎收看新唐人全球新闻。周五，美国联邦参议员投票决定，结束对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 争议指控的辩论，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官.html" target="_blank">
  大法官
 </a>
 任命的最后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_确认投票.html" target="_blank">
  确认投票
 </a>
 将在周六晚进行。
 <br/>
 <br/>
 国会参议员Shelley Moore Capito：〝本次投票结果，51票赞成，49票反对。动议通过。〞
 <br/>
 <br/>
 联邦参议员以51对49票通过表决，同意结束对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 争议指控的辩论。周六晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 将全体表决卡瓦诺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官.html" target="_blank">
  大法官
 </a>
 的提名。
 <br/>
 <br/>
 4位态度摇摆的议员，除共和党籍的穆考斯基（Lisa Murkowski）投了反对票，科林斯（Susan Collins）、弗雷克（Jeff Flake）和民主党籍的曼钦（Joe Manchin）都投票赞成结束辩论，因为FBI针对卡瓦诺的第七次补充背景调查的结论是：福特博士等人的指控没有得到证实。
 <br/>
 <br/>
 对于周六的最终表决，弗雷克、科林斯、曼钦在周五的程序性投票结束后先后确认，将支持卡瓦诺出任大法官。
 <br/>
 <br/>
 共和党联邦参议员 Susan Collins:〝总统先生，我将投票确认大法官卡瓦诺。〞
 <br/>
 <br/>
 周四，卡瓦诺本人在《华尔街日报》发表专栏文章，再次重申其对大法官应保持独立的观点，并就自己在福特指控听证会中表现出的情绪及一些尖锐的话语进行说明。他表示那是他无比沮丧和痛苦心情的表现，无法被证实的错误指控不仅伤害了他、也伤害了他生命中最重要的5位家人。
 <br/>
 <br/>
 不过，抗议者反对卡瓦诺的活动，仍然激烈，在国会山内外，纽约的川普大楼前，都有人举牌抗议，但共和党质疑这些抗议的动机。
 <br/>
 <br/>
 联邦参议院司法委员会主席葛雷斯利：〝我们看到的是，从2016年11月6日大选后的抵制力量，在国会山集中体现了出来。他们在鼓动暴民统治。〞
 <br/>
 <br/>
 总统川普周五发推文，批评左派金主索罗斯等人花钱雇人抗议。他说：那些粗鲁无比的在电梯里尖叫的人是专业抗议者，为了让参议员难堪。再看看那些订制的抗议标语。索罗斯这些人为其买单。
 <br/>
 <br/>
 卡瓦诺是继戈萨奇后，川普提名的第二位保守派大法官，他一旦就职，将令保守派在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院.html" target="_blank">
  最高法院
 </a>
 占据优势。
 <br/>
 <br/>
 民主党人从提名出炉时就誓言抵制，他们担心支持堕胎的罗诉韦德案等判决将被推翻。
 <br/>
 <br/>
 白宫发言人桑德斯：〝正如我说的，他（卡瓦诺）的经历超过了以往任何一个被提名人。一旦投票确认，希望我们能够走回一起、作为一个国家，向前迈进。〞
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
