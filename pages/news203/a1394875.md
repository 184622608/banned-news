### 著名歌手威斯特力挺川普  二人将共进午餐
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】41岁的美国最热门
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_饶舌歌手.html" target="_blank">
  饶舌歌手
 </a>
 、唱片制作人肯伊・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威斯特.html" target="_blank">
  威斯特
 </a>
 ，日前再次公开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_力挺.html" target="_blank">
  力挺
 </a>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 。白宫现已确认，两人将在本周四（11日）中午
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共进午餐.html" target="_blank">
  共进午餐
 </a>
 。
 <br/>
 <br/>
 直言不讳的肯伊・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威斯特.html" target="_blank">
  威斯特
 </a>
 （Kanye Omari West）一直是美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的头号粉丝，他将于周四在白宫与川普
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共进午餐.html" target="_blank">
  共进午餐
 </a>
 。川普今年稍早时，已与他的妻子金・卡戴珊（Kim Kardashian）碰过面。
 <br/>
 <br/>
 此前，威斯特曾多次在公开场合
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_力挺.html" target="_blank">
  力挺
 </a>
 川普总统。今年9月，威斯特现身综艺节目〝周六夜现场〞（Saturday Night Live）时，头戴一顶写有川普签名的〝让美国再次伟大〞红色帽子。

 <br/>
 现场直播节目结束后，肯伊・威斯特没有放下麦克风，并随即发表一场关于政治的即席演说，重申支持川普的立场，因而跃上媒体头条。
 <br/>
 <br/>
 不过，不久后，他删除了个人IG和Twitter帐号，不少人猜测这与他公开支持川普总统，而受到舆论压力有关。
 <br/>
 <br/>
 肯伊・威斯特在今年4月时，还曾在推特上发表一系列支持川普的言论，并称〝他（川普）是我的兄弟〞。该推文的留言及转推数仅在几小时内破万，留言中亦不乏对威斯特坦诚说出心声的支持。川普本人也已在推特上向威斯特表达感谢（〝Thank you Kanye, very cool!〞）。
 <br/>
 <br/>
 生于1977年的肯伊・欧马立・威斯特，又名Ye，是一位美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_饶舌歌手.html" target="_blank">
  饶舌歌手
 </a>
 、唱片制作人、作家和歌手。目前为止，威斯特共发行了五张专辑，依序分别是《远离校园》（The College Dropout，2004年）、《迟来的注册》（Late Registration，2005年）、《毕业特典》（Graduation，2007年）、《心碎节拍》（808s &amp; Heartbreak，2008年）和《我的奇特幻想》（My Beautiful Dark Twisted Fantasy， 2010年），每一张专辑在销售上也都相当成功。
 <br/>
 <br/>
 威斯特还经营着自己的一家唱片厂牌〝GOOD Music〞。2008年5月16日，肯伊・威斯特被MTV电视台选为该年度〝最热门饶舌歌手〞。
 <br/>
 <br/>
 此外，威斯特曾在2015年的MTV音乐录影带大奖上，宣布自己将角逐2020年美国总统，但现已将口号改为〝角逐2024年美国总统〞。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：东野）
