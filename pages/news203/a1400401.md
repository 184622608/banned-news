### 美国感恩节 人们感恩并分享所有
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】今天是美国的感恩节，全国放假，家家户户团聚，感谢神赐予人们丰收平安。
 <br/>
 <br/>
 美军驻韩国士兵，首先迎来感恩节。他们在韩国平泽市陆军基地，度过了以哈利・波特为主题的感恩节午餐。
 <br/>
 <br/>
 陆军基地餐厅准备了烤火鸡、土豆泥、南瓜派等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统感恩节大餐.html" target="_blank">
  传统感恩节大餐
 </a>
 ，希望士兵们能一解相思之苦。
 <br/>
 <br/>
 美军驻韩国士兵 AUSTIN DONEL-ANTHONY：〝这有些帮助，因为我们在一起，不是独自一人，这是（过感恩节）的目的，不要孤单一人。〞
 <br/>
 <br/>
 士兵们举杯，一起祝福感恩节快乐!
 <br/>
 <br/>
 美国西岸的加州刚刚遭受史上最惨烈的野火侵袭。在洛杉矶市区，好莱坞艺人捐款送感恩节食物给游民和其他需要帮助的人。
 <br/>
 <br/>
 好莱坞女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蜜妮卓芙.html" target="_blank">
  蜜妮卓芙
 </a>
 每年都带儿子一起来，她说希望能让孩子体验感恩节精神——施舍比享受更有福。
 <br/>
 <br/>
 好莱坞演员蜜妮卓芙：〝你回馈，不管是什么，哪怕一点点，尽可能的给予，尽你所能这样做，他是社区一员，我希望他在成长中能学到这一点。〞
 <br/>
 <br/>
 蜜妮卓芙感谢投入加州救灾灭火的消防员、志愿者及救难人员，赞扬他们是英雄。
 <br/>
 <br/>
 好莱坞演员蜜妮卓芙：〝我非常感激，真的非常感激，我的许多朋友在大火中失去了家园，幸而他们都很平安。我真的很感谢那些人救了我的社区，他们是英雄。〞
 <br/>
 <br/>
 感恩节当天，美国总统川普也在推特上，祝福大家感恩节快乐!
 <br/>
 <br/>
 新唐人记者李梅、天羽综合报导
