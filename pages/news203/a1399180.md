### 〝漫威之父〞斯坦・李逝世 各地粉丝悼念
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】周一，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_漫威之父.html" target="_blank">
  漫威之父
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯坦・李.html" target="_blank">
  斯坦・李
 </a>
 在洛杉矶一家医疗中心去世，享年95岁。美国各地〝漫威迷〞发起悼念活动。
 <br/>
 <br/>
 纽约时代广场附近的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中城漫画店.html" target="_blank">
  中城漫画店
 </a>
 〞内，漫画书迷们赞许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯坦・李.html" target="_blank">
  斯坦・李
 </a>
 是一个传奇。漫画店老板表示，斯坦・李不仅创作了许多经典漫画人物，他本人也很亲切。
 <br/>
 <br/>
 〝中城漫画店〞老板GERRY GLADSTON：〝他不仅创造了全世界喜爱的（漫画）人物，这些也是历史上最受欢迎的虚构角色。他本人也很可爱，是一个用不疲倦的自我推动者。〞
 <br/>
 <br/>
 周一，漫威迷穿着漫威经典人物服装，也在好莱坞星光大道发起悼念活动。
 <br/>
 <br/>
 漫威迷Eduardo Lezcano：〝斯坦・李是一切。他是传奇。〞
 <br/>
 <br/>
 一些美国演员和导演也表示了对斯坦・李的悼念。
 <br/>
 <br/>
 美国知名配音演员Josh Gad：〝我认为他留下来的（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_漫画作品.html" target="_blank">
  漫画作品
 </a>
 ），会比他的一生流传得更久。不仅在未来几十年，而且在此之后很长时间（流传）。〞
 <br/>
 <br/>
 美国演员、导演Rob Reiner：〝他是孩子们观看的这些东西（漫威作品），伟大的创作者之一。〞
 <br/>
 <br/>
 斯坦・李1961年合作创立的〝漫威影业〞，成为美国流行文化的重要标志之一。近年他的许多漫画作品被改编成电影，票房常年火爆。其代表作有《蜘蛛侠》、《绿巨人》、《钢铁侠》和《复仇者联盟》等。
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
</div>
