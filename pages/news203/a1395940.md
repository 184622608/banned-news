### 美国退出万国邮联 中美小件邮费看涨
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 宣布退出〝万国邮政联盟〞，因其现行的〝终端费〞制度对美国不公，滋长了不公平的贸易模式。据悉，中国寄到美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小件包裹.html" target="_blank">
  小件包裹
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邮费.html" target="_blank">
  邮费
 </a>
 在两年内将会提高。
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 周三宣布，退出万国邮政联盟（UPU）。
 <br/>
 <br/>
 按照UPU现行规定，发展中国家向发达国家寄送小件邮包，只需支付低廉的〝终端费〞给收件国的邮政部门，用于完成最终投递。
 <br/>
 <br/>
 中国目前已是世界第二大经济体，但UPU仍将其列为发展中国家。中国向美国支付的终端费，低于美国邮政境内投递的成本。去年美国邮政补贴了约1.7亿美元用于处理中国包裹。
 <br/>
 <br/>
 唐靖远：现在的〝终端费〞制度是2000年左右制定的，那个时候跨境电商还没有兴起。现在情况已经完全不同了，全球化和电子商务时代的兴起，中国又是最大的跨境电商。这种规模效应，就给美国邮政带来很重的经济负担。
 <br/>
 <br/>
 白宫贸易顾问纳瓦洛批评，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万国邮联.html" target="_blank">
  万国邮联
 </a>
 的费率，也导致全球电商市场扭曲。上个月他在《金融时报》撰文说：〝中国业者常透过网购平台（如阿里巴巴或亚马逊）向美国销售仿冒品〞。
 <br/>
 <br/>
 纳瓦洛：这很荒谬，现在有个问题，从洛杉矶寄包裹到纽约，比从上海寄到纽约还贵。
 <br/>
 <br/>
 唐靖远：要知道今年中国电商在美销售额已经达到800亿美元，这种透过电商零售来低价倾销的方式已经不容小觑了。现在美国想调高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小件包裹.html" target="_blank">
  小件包裹
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邮费.html" target="_blank">
  邮费
 </a>
 ，可能也有想降低伪劣品价格优势的因素。同时让两国零售电商在相同的邮费基础上竞争。目标还是公平贸易。
 <br/>
 <br/>
 目前运往美国的包裹，约60%来自中国。根据UPU规定，启动退出程序后，仍有一年谈判时间。
 <br/>
 <br/>
 纳瓦洛：我们所做的，就是允许实行自行费率，也就是说，美国邮政能按包裹（运费）成本收费。
 <br/>
 <br/>
 匿名白宫官员对美国媒体说，现行邮费至少六个月不会变。如果谈判成功，美国仍将留在UPU内。
 <br/>
 <br/>
 但不论去留，中国运往美国的小包裹运费增加，都已在所难免。川普已要求，最迟2020年1月1日前，启用自订费率。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
