### 民主党狂争中期选举 数州允许非公民投票
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 已进入倒计时阶段，国会两党选情胶着，竞争极为激烈。日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 为赢得胜利，竟在一些州允许甚至动员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非公民.html" target="_blank">
  非公民
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 ，给中期选举再添复杂因素。
 <br/>
 <br/>
 德克萨斯州州务卿巴勃罗斯（Rolando Pablos）周一（10月22日）声明，德州政府已收到大量呼吁和投诉，当地民众对于有人把预先填写的选民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_登记.html" target="_blank">
  登记
 </a>
 申请表发送给不合格选民的情况深感不安。
 <br/>
 <br/>
 巴勃罗斯指出，德州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 向当地民众发送邮件，敦促人们报名参加选举，并警告居民如果不迅速行动就会错过最后期限。邮件中包含一份选民申请表，上面已填好了预先核查过的收信人的年龄和公民身份。
 <br/>
 <br/>
 更令人不安的是，其中一些邮件寄给了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非公民.html" target="_blank">
  非公民
 </a>
 ，这些人实际上没有资格
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 。
 <br/>
 <br/>
 这些信封中甚至还装入了用以回复的信封，上面写好了收信的选举官员的地址。
 <br/>
 <br/>
 这些形式让一些居民误认为，这些信件是德州官方发出的。
 <br/>
 <br/>
 鼓励他人提交欺诈性登记表是犯罪行为，巴勃罗斯表示，他已要求德州司法部长调查此事。
 <br/>
 <br/>
 周一，新罕布殊尔也发生支持不合格选民投票的情况。该州激进主义法官布朗（Kenneth C. Brown）颁布一份临时禁令，限制执行共和党所支持的一项法律。该法律规定，新选民在注册时须出示居住证明。
 <br/>
 <br/>
 布朗称，要求出示居住证明会延长排队时间，使学生、残疾选民及其他人更难投票。
 <br/>
 <br/>
 布朗在裁决中写道，〝如果法律威胁要剥夺个人的投票权，唯一可行的补救办法就是禁止其执行〞。
 <br/>
 <br/>
 该法官还说，登记表太复杂了，人们无法理解。
 <br/>
 <br/>
 新罕布什尔州的民主党人则对布朗的裁决表示欢迎。
 <br/>
 <br/>
 加利福尼亚州旧金山市（San Francisco）做得更为过火。
 <br/>
 <br/>
 加州本来就被称为民主党〝大本营〞。早在今年7月，加州选举部就向非法外侨发放了选民登记表，要求他们在11月的选举中投票选出旧金山教育委员会（San Francisco Board of Education）成员。
 <br/>
 <br/>
 旧金山是加州第一个允许包括非法外侨在内的非公民投票的城市，虽然这项规定以关心孩子的教育为藉口、且仅限于选举教育部门官员。
 <br/>
 <br/>
 不过，旧金山选举部表示，至本周一的加州选民登记截止日期，只有35名非公民报名参加投票。
 <br/>
 <br/>
 民主党主张对非法移民实行宽松政策，所以，他们把非法移民视作自己潜在的选民。加州和纽约州两个最大的蓝州为维护民主党选票，甚至分别起诉川普政府在2020年人口普查中恢复询问受查者公民身份问题的决定。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
