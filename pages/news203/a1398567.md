### 加州酒吧枪击案至少13死 枪手身份曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】周三（11月7日）午夜，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 千橡市（Thousand Oaks）爆发大规模
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，一名枪手潜入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒吧.html" target="_blank">
  酒吧
 </a>
 ，向正在欢度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_音乐节.html" target="_blank">
  音乐节
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大学生.html" target="_blank">
  大学生
 </a>
 们射击，导致至少13人死亡，21人受伤，包括枪手及1名以身殉职的即将退休的警察。
 <br/>
 <br/>
 周三晚11:20左右，加利福尼亚路德大学（California Lutheran University）等三所大学的数百名学生正聚集在千橡市一家名为〝Borderline Bar &amp; Grill〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒吧.html" target="_blank">
  酒吧
 </a>
 内欢度〝乡村之夜〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_音乐节.html" target="_blank">
  音乐节
 </a>
 ，一名身穿黑衣的青年男子突然悄悄入内，并持一把半自动手枪向人群射击。文图拉县（Ventura County）警方称，枪手至少射出30发子弹，并使用烟雾弹迷惑人们的视线。
 <br/>
 <br/>
 警长迪恩（Geoff Dean）在周四早上召开的新闻发布会上表示，警方和公路巡逻人员接报后几分钟便赶至现场。据他描述，当时的场景非常可怕，到处都是鲜血。
 <br/>
 <br/>
 警官赫鲁斯（Ron Helus）在事件中殉职。赫鲁斯已在执法部门工作29年，即将退休。他是事发后第一位抵达现场的警察，听到枪声后迅速与一名巡警冲进酒吧，之后被射中数枪，送院急救后不治而亡，于周四早间去世。
 <br/>
 <br/>
 据福克斯新闻（Fox News）周四上午报导，枪手大卫．龙（David Long），现年28岁，是一名前海军陆战队员。执法人员在酒吧入口附近的一间办公室内发现了他的尸体，并判断他是作案后自戕而亡。
 <br/>
 <br/>
 迪恩称，龙患有创伤后应激障碍（post-traumatic stress disorder，简称PTSD，又称创伤后遗症），在过去几年中，因交通事故等问题，警方和他有过几次〝接触〞，知道他的情况。
 <br/>
 <br/>
 迪恩表示，警方最后一次和他接触是在今年4月。当时警察接到报警电话后赶到他家，发现他正在发怒，行为很不理智，所以其家人不得不打电话叫来警察帮助处理。但警察放过了他，他们认为，龙的行为不足以导致被拘留。
 <br/>
 <br/>
 据龙2017年3月在一个特种部队论坛中所公布的个人信息，他曾在阿富汗服役，是美国海军陆战队的一名步兵机枪手，从军4年半，后来还在日本冲绳（Okinawa）担任过教练。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
