### 川习会晤在即 俄媒：美国释放危险信号
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】近日，有俄罗斯媒体发表报导讨论中美关系，并藉机重炒美军可能于本月底在台湾海峡举行大规模军演的话题，声称这是个〝危险的信号〞。
 <br/>
 <br/>
 〝今日俄罗斯〞电视台11月14日发表的一篇报导称，中国国家主席习近平与美国总统特朗普（Donald Trump）在G20峰会上的会晤，为此现阶段美国政府发表的〝所有声明〞，都在施压北京必须在谈判中让步。
 <br/>
 <br/>
 该报导并引述俄罗斯人民友谊大学教授塔夫罗夫斯基（Yuri Tavrovsky）表示，美军11月底将在台湾海峡举行大规模军演，这是一个〝非常危险的信号〞。
 <br/>
 <br/>
 有关美军将在台湾海峡举行大规模军演的说法，最早是一个多月前由美国有线电视新闻网（CNN）披露出来的。
 <br/>
 <br/>
 今年10月3日，CNN在一篇报导中表示，有多名美国国防官员向该媒体透露，美军正在制定一个秘密计划，打算在今年11月份举行为期大约一周的全球性军事演习，以此向中共展示美军具有同时在多个前线对抗敌军的能力。
 <br/>
 <br/>
 据称，美国海军太平洋舰队已经起草一份秘密提案，打算把军演的地点主要安排在南中国海和台湾海峡进行，让美国的军舰和战机在南中国海和台湾海峡进行〝自由航行行动〞，以宣示〝美国拥有公海上自由航行的权利〞。
 <br/>
 <br/>
 报导称，军演行动的范围，也可能会扩展到美国西海岸和南美洲，因为中共正在那里扩大投资。
 <br/>
 <br/>
 不过，一位不具名的美军官员也对CNN表示，上述计划目前还〝只是一个想法〞，而美国防部长马蒂斯（James Mattis）和参联会主席邓福德（Joseph Dunford）正在考虑这些行动一旦实施可能产生的外交影响。
 <br/>
 <br/>
 当时五角大楼和太平洋海军司令部拒绝对CNN的这篇报导进行置评。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：明轩）
