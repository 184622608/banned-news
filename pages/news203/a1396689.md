### 可疑包裹寄奥巴马 特工:不爆炸本身就是巨大线索
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】周三（10月24日），美国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 （Barack Obama）等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 政要相继遭到可疑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_包裹.html" target="_blank">
  包裹
 </a>
 威胁。美国有线电视新闻网（CNN）也收到了类似的爆炸装置。但寄发这些包裹似乎别有用意，许多分析人士表示质疑。
 <br/>
 <br/>
 周三，几个装有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸物.html" target="_blank">
  爆炸物
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_包裹.html" target="_blank">
  包裹
 </a>
 分别寄给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 众议员沃特斯（Maxine Waters）、奥巴马政府时期的司法部长霍尔德（Eric Holder）和前中情局（CIA）局长布伦南（John Brennan）。自由派媒体CNN位于纽约的编辑部也收到类似装置。
 <br/>
 <br/>
 周二晚，前国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希拉里.html" target="_blank">
  希拉里
 </a>
 （Hillary Clinton）也险些收到类似包裹。不过，美国特勤局（U.S. Secret Service）先后成功拦截了寄给希拉里和奥巴马的爆炸物。
 <br/>
 <br/>
 消息传出后，正在争夺中期选举的美国两党均发出谴责。川普总统当天上午也转发了副总统彭斯（Mike Pence）的推文，并表示他完全赞同彭斯的立场。
 <br/>
 <br/>
 彭斯指出，这些懦弱的行为非常卑鄙，美国不能容忍，并将把责任人绳之以法。

 <br/>
 奥巴马等人收到爆炸物的消息几乎成为周三所有美国媒体的头条新闻。
 <br/>
 <br/>
 本月初，特勤局成功拦截了几封寄给川普、国防部长马蒂斯（Jim Mattis）和海军作战部长理查森（John Richardson）的有毒信件。该事件一度引发巨大关注。
 <br/>
 <br/>
 周一，民主党的重要金主、亿万富翁索罗斯（George Soros）也收到类似的爆炸装置。
 <br/>
 <br/>
 但分析人士认为，在距中期选举还有两周之际发生多起此类事件，显然具有政治意义。
 <br/>
 <br/>
 调查发现，这些可疑邮件都很容易被发现，且均〝有惊无险〞、无一爆炸。其中，索罗斯和CNN并没有特勤局所安装的邮件筛选系统， 但也最终安然无恙。
 <br/>
 <br/>
 值得关注的是，这些爆炸物指向的目标都是民主党政要或是激进的反川普人士。它们虽然无一爆炸，但想要达到的目的却显然具有爆炸性。
 <br/>
 <br/>
 艾美奖（Emmy Award）得主、前高级新闻记者豪斯利（Adam Housley）在推文中说，联邦特工告诉他，这些装置都未爆炸本身就是个〝巨大线索〞。
