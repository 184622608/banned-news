### 川普提名内阁新人选 司法部长料会获各方认同
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）确认了美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部长.html" target="_blank">
  司法部长
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻联合国大使.html" target="_blank">
  驻联合国大使
 </a>
 的新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人选.html" target="_blank">
  人选
 </a>
 ，同时宣布周六将公布参谋长联席会议主席人选。新司法部长提名人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威廉・巴尔.html" target="_blank">
  威廉・巴尔
 </a>
 是老布什执政时期的司法部长，很多方面持保守司法观点。
 <br/>
 <br/>
 在老布什执政时期1991-1993年间，曾担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部长.html" target="_blank">
  司法部长
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威廉・巴尔.html" target="_blank">
  威廉・巴尔
 </a>
 （William Barr），周五获
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 提名司法部长一职。
 <br/>
 <br/>
 美国总统川普：〝我想确认比尔・巴尔，全美最受尊敬的法学家之一、律师、老布什时期的前司法部长，非常出色的人。共和党人及民主党人都尊重他。他将被提名为美国司法部长。〞
 <br/>
 <br/>
 专家认为川普提名巴尔，会获得更多支持。
 <br/>
 <br/>
 乔治梅森大学法学院教授Francis H. Buckley：〝在以前（布什）行政期间，他是一个非常勇敢的司法部长。很正直、很保守的一个人
 <br/>
 <br/>
 哥伦比亚大学政治学教授Robert Y. Shapiro：〝这是一个相当不错的提名。他挑选的这个人有非常好的经验，能够与共和党建制派很好的合作；对川普而言也是好的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人选.html" target="_blank">
  人选
 </a>
 ，因为他很多方面的观点都是传统理念，比如在法律秩序上、犯罪问题上、行政权力看法上、公民权利方面的议题等。〞
 <br/>
 <br/>
 川普希望提名能很快在参议院获得确认。专家认为，巴尔的确认没有问题。
 <br/>
 <br/>
 乔治梅森大学法学院教授Francis H. Buckley：〝他是司法部长的最佳人选，标竿一样的人物。而且他与现在的代理司法部长的背景不同。所以他被确认应该没有问题。〞
 <br/>
 <br/>
 哥伦比亚大学政治学教授Robert Y. Shapiro：〝而且他会有更多的信誉，在通俄门调查、穆勒及其他一些问题上，不会像塞申斯那样受到那么多批评。〞
 <br/>
 <br/>
 巴尔在老布什时期担任司法部长时，通俄门特别检察官穆勒是他的手下，担任刑事司负责人。
 <br/>
 <br/>
 专家认为，穆勒的调查报告很可能会让民主党人失望。
 <br/>
 <br/>
 川普一直批评通俄门调查为〝猎巫〞，在今日推文中川普说〝我们要对穆勒的报告提出反诉报告。这样的事情不应该再发生在未来的美国总统身上！〞
 <br/>
 <br/>
 除司法部长提名外，另一个重要内阁成员变更提名是美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻联合国大使.html" target="_blank">
  驻联合国大使
 </a>
 ，国务院发言人诺尔特（Heather Nauert）将接替黑利（Nikki Haley），明年出任联合国大使一职。诺尔特曾是FOX新闻主播、此前没有外交经验。
 <br/>
 <br/>
 川普还透露周六将宣布下任参谋长联席会议主席人选，目前外界预测将由陆军参谋长马克・米利（Mark Milley）将军接替现任的邓福德（Joseph Dunford）将军。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
