### 北加坎普野火全面控制 南加马里布居民返家
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】在大雨的帮助下，25日加州消防局宣布，北加坎普大火受到全面控制，共造成87死249失踪。而在南加州，随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伍尔西山火.html" target="_blank">
  伍尔西山火
 </a>
 被扑灭，越来越多的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马里布.html" target="_blank">
  马里布
 </a>
 居民也在25日重返家园。
 <br/>
 <br/>
 美国加州消防当局25日表示，加州史上最致命且最具破坏性的坎普大火（Camp Fire）延烧逾2周，共造成87死249失踪，火势终于在消防人员努力下受到全面控制。
 <br/>
 <br/>
 加州森林防火厅（Cal Fire）在推特（Twitter）25日发布最新公告：〝#坎普大火…现在百分之百被控制住了。〞
 <br/>
 <br/>
 坎普大火的焚烧面积达到15万3336英亩，并烧毁近1万4000间民房和数百栋其他类型建筑物。
 <br/>
 <br/>
 坎普野火形成的滚滚浓烟，迫使旧金山部分学校在本月稍早停课，而且市区缆车和恶魔岛（Alcatraz Island）也全部关闭。
 <br/>
 <br/>
 加州当局已经允许居民返回部分受灾最严重的地区，以查看他们的财产毁损状况。

 另外，南加州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伍尔西山火.html" target="_blank">
  伍尔西山火
 </a>
 （Woolsey Fire）自11月8日在文图拉（Ventura）县爆发，后延烧至洛杉矶县，共延烧9万8362英亩地，烧毁1643座建筑，有另外364座建筑遭破坏。共有3人在大火中丧生，另外还有3名消防员受伤，21日火势得到了控制。
 <br/>
 <br/>
 根据洛杉矶警局消息，当地时间24日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马里布.html" target="_blank">
  马里布
 </a>
 （Malibu）部分路段已逐渐开放，让更多居民返家。25日上午9时许，马里布西湖大道（Westlake Boulevard）部分路段开放给当地居民进入。上午10时，Encinal Canyon部分路段也已经开放。
 <br/>
 <br/>
 来自火灾的浓烟一直笼罩在该州的部分地区，被称为〝世界上最脏的空气〞。洛县消防局发言人Pono Barnes表示，该地区的面貌被彻彻底底地改变了。他也督促返家居民，应做好预防泥石流措施，部分地区恢复了自然气供应。居民应当通知自然气公司或者相关专业人员恢复通气。
 <br/>
 <br/>
 目前仍有消防员等工作人员在灾区工作。随着越来越多民众返回居所，洛县警局官员提醒居民，由于电力工作人员在灾区作业，会有断断续续停电现象。
 <br/>
 <br/>
 另外，位于23525 Civic Center Way，以及Conrad L. Hilton Foundation的阿古拉山（Agoura Hills）和马里布灾害救援中心，已经为受灾居民开放，灾民可以在这里申请保险赔偿，灾后援助等。
