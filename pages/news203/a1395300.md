### 川普俄亥俄州助选 民众排斥极左 支持爱国主义
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】周五晚，川普在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄亥俄州.html" target="_blank">
  俄亥俄州
 </a>
 黎巴嫩市举行竞选造势活动。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 说，美国美国再次赢得尊重。现场不少民众表示，走向极左对国家很危险，他们支持川普的议程。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄亥俄州.html" target="_blank">
  俄亥俄州
 </a>
 做为摇摆州，是历次选举中两党激烈争夺的对象，这次是川普总统从上次竞选总统以来，第24次在俄亥俄州举办集会。
 <br/>
 <br/>
 川普首先公布，布朗森牧师获释的好消息。
 <br/>
 <br/>
 美国总统 川普：〝我非常自豪的告诉大家，今天早些时候，我们确保了布朗森牧师从土耳其获释。〞
 <br/>
 <br/>
 川普鼓励民众出来投票。
 <br/>
 <br/>
 美国总统 川普：〝美国正在走向繁荣昌盛，美国再次赢回自己。最近我到处走也经常说民主党变成了一个（制造）罪恶之党，没有人质疑我，包括那些假新闻媒体。我们需要共和党控制参众议院，我们需要你们的选票。〞
 <br/>
 <br/>
 今年面临选举的俄亥俄州共和党议员们悉数到场。
 <br/>
 <br/>
 国会众议员 Jim Renacci：〝我告诉我的妻子、家族和朋友给你投票，希望你改变华盛顿。你没有让我失望。〞
 <br/>
 <br/>
 支持川普的民众，一大早就赶来参加集会。不少人是因为对民主党走向激进的极左方向感到担忧和抵制。
 <br/>
 <br/>
 民众 Cindy Foltz（女儿）：〝她做了多年的民主党人，不过几年前转入共和党。〞。
 <br/>
 <br/>
 Johnson（母亲）：〝因为（民主）党内不同信仰的人意见不同，它让我们失望了。〞
 <br/>
 <br/>
 民众 Bob Banks：〝我最不想要的就是国家变成社会主义、接近共产主义。〞
 <br/>
 <br/>
 老兵 Greg Rice：〝我看到国家被偏自由的左派分裂。终于川普来了、扭转局势，让国家更好更强大。〞
 <br/>
 <br/>
 民众 Jim Liles ：〝他让美国再次获得尊重，那是之前8年我们失去的东西。〞
 <br/>
 <br/>
 不少人是因为川普热爱美国而支持他。
 <br/>
 <br/>
 民众 Holly Jackson（母亲）和 Rylynn Jackson（儿子）：〝让我们的军队更加大，很多原因（支持川普）。你呢？家庭学校，还有她刚才说的那些原因。〞
 <br/>
 <br/>
 退休消防员 Bryan Torner ：〝消防员和警察都支持川普，因为他真的在意我们。〞
 <br/>
 <br/>
 民众 Mark Arnold：〝他是我见过的最爱美国的人。〞
 <br/>
 <br/>
 民众 Joey Colegate：〝因为他要抽干沼泽。〞
 <br/>
 <br/>
 民众 Urbancic ：〝他做出承诺、然后他去兑现。〞
 <br/>
 <br/>
 上任20个月，川普政府已经实现了289项在2016总统大选中的承诺。
 <br/>
 <br/>
 新唐记者陈雷、葛安晨、张清韵俄亥俄州黎巴嫩市报导
</div>
