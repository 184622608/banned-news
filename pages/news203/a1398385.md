### 美期中选举大致底定 国会或呈分裂局面
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】美国2018
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_期中选举.html" target="_blank">
  期中选举
 </a>
 大致底定，目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 稳住
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 则拿下过半席次，形成国会分裂的局面。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_期中选举.html" target="_blank">
  期中选举
 </a>
 ，改选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 435个全部席位、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 100个席位中的35席，以及50个州中的36个州长和州议会。
 <br/>
 <br/>
 当地时间7日凌晨14时55分的战况：参议院改选方面，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 51席：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 44席。众议院方面，共和党193席：民主党217席。州长改选，共和党25席：民主党21席。
 <br/>
 <br/>
 这次期中选举前，共和党掌握236席，民主党人占193席，另有6席空缺。
 <br/>
 <br/>
 现任众议院议长是代表威斯康辛州的共和党籍莱恩（Paul Ryan），这回他未寻求连任。而民主党籍高龄78岁的前议长裴洛西（Nancy Pelosi），目前是众院少数党领袖，或可望回锅众院议长。
 <br/>
 <br/>
 不过，裴洛西在8年前失去众议院，让裴洛西招致不少党内批评，不少年轻民主党议员公开与她划清界线，表明不愿继续力挺，裴洛西能否顺利高举议长槌，还有待观察。
 <br/>
 <br/>
 <b>
  委员会主席
 </b>
 <br/>
 中央社报导，在委员会主席方面，共和党仍是参议院多数党，军委会主席殷荷菲（Jim Inhofe）9月初才接替因脑癌过世的麦凯恩（John McCain）的遗缺，〝原则〞不受部分席次改选影响。
 <br/>
 <br/>
 外委会主席寇尔克（Bob Corker）去年9月就宣布将退休，不再参选。依照惯例，共和党排名第二、代表爱达荷州的里契（James Risch）可望接任外委会主席职务。
 <br/>
 <br/>
 但媒体也指出，里契在外委会并不活跃，反而更专注于情报委员会（Intelligence Committee）。现任情委会主席柏尔（Richard Burr）预计2020年退休，若里契决定等候情委会主席一职，佛罗里达州参议员卢比欧（Marco Rubio）有望接下外委会主席职务。
 <br/>
 <br/>
 众议院方面，由于民主党重新拿下多数党席次，各委员会主席也将由民主党排名议员出任。
 <br/>
 <br/>
 军委会目前排名最高的民主党议员为华盛顿州第9选区选出的史密斯（Adam Smith）。史密斯所在选区亚裔比例超过2成，对少数族裔现况十分熟悉。
 <br/>
 <br/>
 (责任编辑：卢勇信)
