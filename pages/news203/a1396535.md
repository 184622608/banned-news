### 美中期选举  川普外交政策为共和党加分
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】距离美国中期选举只有两周了。虽然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交政策.html" target="_blank">
  外交政策
 </a>
 不是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选民.html" target="_blank">
  选民
 </a>
 最关注的议题，但专家表示，贸易和移民政策依然会影响很多选民的决定，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）的政策显然会给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 加分不少。
 <br/>
 <br/>
 美国中期选举，通常也被视为对现任总统满意度的公投。根据媒体最新民调，只有3%
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选民.html" target="_blank">
  选民
 </a>
 认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交政策.html" target="_blank">
  外交政策
 </a>
 和反恐是他们最关切议题，而对经济和健保问题的关注则远超于此。
 <br/>
 <br/>
 美国企业研究所民调专家 Karlyn Bowman：〝有些议题是次要问题，如同慢慢在火上炖，但随时可能沸腾，变为重要议题。外交政策和反恐问题正是如此。〞
 <br/>
 <br/>
 最近，数千名来自洪都拉斯的大篷车难民试图越过墨西哥边境，进入美国。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统连发推文，批评洪都拉斯、瓜地马拉和墨西哥，阻止难民不利，并宣布南部边境进入国家紧急状态。面对紧迫的拉美非法移民问题，专家认为，这可能在中期选举中对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 更有利。
 <br/>
 <br/>
 前美国维州联邦众议员 Tom Davis：〝选民们了解必须要有一定法律，拥有边境安全，不能让人们随意越境，仅仅因为他们想要过来。这有可能引发很多坏的事情。〞
 <br/>
 <br/>
 专家认为，从移民、到朝核、到贸易问题，川普总统的外交政策凸显出他〝美国优先〞的竞选承诺，受到选民广泛认可。尤其是《美墨加贸易协定》的签署为川普政府加分不少。
 <br/>
 <br/>
 川普政府前白宫副幕僚长 Rick Dearborn：〝甚至是在劳工和环境问题上，在最近谈成的《美墨加贸易协定》中，民主党和关注者都对结果感到惊喜。所以这些对他们促动很大。〞
 <br/>
 <br/>
 目前中共押宝民主党会在中期选举后掌控众议院，美中贸易紧张形势会有所变化，但专家认为，这只是中共幻想。
 <br/>
 <br/>
 川普政府前白宫副幕僚长 Rick Dearborn：〝我认为民主党会和他们的想要公平竞争环境的选民站在一起。〞
 <br/>
 <br/>
 前美国维州联邦众议员 Tom Davis：〝总统大选开始前，川普总统大约还有一年时间。其间，压力会在中共一方，如果中国GDP没有增长的话。看起来，他们有不少内部问题。〞
 <br/>
 <br/>
 虽然传统上，中期选举结果往往对现任总统所在政党不利，但是目前美国经济表现强劲、就业数字看好。专家认为，共和党依然有望同时保住参众两院的多数席位。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
