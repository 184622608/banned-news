### 老布什国葬前夕 川普偕夫人亲访布什家族致意
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼娅.html" target="_blank">
  梅拉尼娅
 </a>
 亚4日亲自拜访布什家族表达哀悼之意，前总统小布什与夫人萝拉在冷冽的午后于布莱尔宾馆外迎接。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 稍早推文表示，他将前往总统贵宾招待所布莱尔宾馆（Blair House）拜访〝很棒的布什家族〞。布莱尔宾馆和白宫只有一街之隔。
 <br/>
 <br/>
 前总统小布什（George W. Bush）在冷冽的午后于布莱尔宾馆外，迎接川普和第一夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼娅.html" target="_blank">
  梅拉尼娅
 </a>
 （Melania）入内。稍早，小布什夫人萝拉（ Laura Bush）在现任第一夫人梅兰妮亚接待下，参观白宫的耶诞装饰。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/12-5/p9157201a227016589.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/12-5/p9157201a227016589-ss.jpg"/>
   <br/>
   <font size="-1">
    美国总统川普和夫人梅拉尼娅亚4日亲自拜访布什家族表达哀悼之意。(NICHOLAS KAMM/AFP/Getty Images)
   </font>
  </a>
 </center>
 <br/>
 <br/>
 老布什（George H.W. Bush）国葬仪式将于明(5日)举行，据报届时川普将坐在第一排，但不会致悼词。
 <br/>
 <br/>
 圆形大厅外，美国民众顶着寒风排队，在挑高的国会山庄圆顶之下，瞻仰老布什遗容，民众静静走过覆盖着国旗的灵柩旁。
 <br/>
 <br/>
 高龄95岁的鲍勃．多尔（Bob Dole）在他人搀扶下，从轮椅缓缓起身，向曾在1988年共和党党内总统提名击败他的老布什灵柩致敬。
