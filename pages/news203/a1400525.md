### 川习会准备好了 川普：我比谁都了解数据
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】今天欧洲3大股市收黑。美国股市则是因为感恩节假期休市，而美国总统川普（特朗普），下周将与中国国家主席习近平会晤，他表示，自己已经准备好了。
 <br/>
 <br/>
 美国总统川普，到佛罗里达州度过感恩节假期，21日上午他回答随行记者提问，谈到下周将赴阿根廷出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 ， 并与中国国家主席习近平会谈，川普信心满满。
 <br/>
 <br/>
 美国总统 川普:〝我已经准备好了，我毕生都在为会谈准备。我不需要坐在书桌前读书，我比任何人都了解任何内容，任何数据，我的直觉一直都是对的。我们做得很好，而且中国非常想和我们达成协议。〞
 <br/>
 <br/>
 川普还说，正是因为实施关税，中共才这么想和美国达成协议。也再次重申，明年1月将对中国2500亿美元商品关税税率调升至25%。
 <br/>
 <br/>
 美国总统 川普:〝自从我上任总统后，美国增加了数兆美元收入，而中国则是损失数兆美元。美国是真正的经济强权，比我上任总统前还要更强大。〞
 <br/>
 <br/>
 有分析师认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 不会在解决贸易冲突上有重大进展，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 在未来几个月〝暂时中断〞，但在美国2020年总统大选前不会结束。
 <br/>
 <br/>
 <br/>
 新唐人亚太电视综合报导
</div>
