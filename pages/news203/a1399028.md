### 加州三场野火同燃 至少31人丧生
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】首先来关注加州大火。目前，美国北加州和南加洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火.html" target="_blank">
  野火
 </a>
 肆虐，死亡人数上升到至少31人，加州州长布朗请求川普总统发布〝重大灾难声明〞，川普已批准联邦援助。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南加州圣安娜.html" target="_blank">
  南加州圣安娜
 </a>
 周日再度刮起强风，助长了〝伍尔西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火.html" target="_blank">
  野火
 </a>
 〞的火势，使其摧毁了马利布市的豪宅和郊区住宅，造成至少2人丧生，超过25万人疏散。此外，被称为〝小山〞的野火也在洛杉矶附近燃烧。
 <br/>
 <br/>
 而加州北部天堂镇〝营火〞也因为强风和干旱天气，使山火继续蔓延。
 <br/>
 <br/>
 野火已造成至少31人死亡，其中29人在北加州丧生。
 <br/>
 <br/>
 布尤特县警长暨验尸官Kory Honea：〝目前还有228人经确定下落不明。〞
 <br/>
 <br/>
 当局扩大失踪者和尸体的搜查，并呼吁DNA实验室和人类学家协助监别受害者。
 <br/>
 <br/>
 8000多名消防人员投入加州这场有纪录以来破坏性最大的野火。大火已摧毁了超过6,700栋建筑。
 <br/>
 <br/>
 加州州长杰里．布朗请求川普总统发布〝重大灾难声明〞，川普已批准联邦援助。
 <br/>
 <br/>
 布朗表示，目前还不知道这场火灾的原因，但他警告这可能将是一种新的异常状态，表示加州未来继续面临野火威胁，还可能更糟。
 <br/>
 <br/>
 而加州官员警告，周日和周一的强风将助长目前的火势，令火势更加难以控制。
 <br/>
 <br/>
 新唐人记者 李梅 综合报导
