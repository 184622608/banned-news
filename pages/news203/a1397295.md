### 大篷车难民潮有增无减 墨西哥爆冲突
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】中美洲大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 继续向美国挺进，近日于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 南部稍事休息时，因争抢食物而爆发冲突，继而误传出有人掳走儿童，酿成打斗。同日，墨西哥政府首次向非法移民大军提供援助。
 <br/>
 <br/>
 已向北方前进逾2周的大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 ，上周六（27日）于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 瓦哈卡州爆发冲突。塔帕纳特佩克市（Tapanatepec）在派发食物时有难民煽动攻击维持排队秩序的一名男子。男子逃走后，又有传言称他掳走儿童，结果被众人殴打受伤入院治疗。
 <br/>
 <br/>
 与此同时，再有新的2队移民大军在危地马拉及萨尔瓦多出现，进一步向美国边境逼进。危地马拉警方出动催泪弹，阻止冲击墨西哥边境的非法移民。
 <br/>
 <br/>
 这些中美洲移民主要来自洪都拉斯，一些移民计划申请政治庇护，理由是他们在洪都拉斯受到帮派威胁，或是受到尼加拉瓜政府对反对派的攻击威胁。但随着这个群体变得越来越大，人群也变得越来越多样化。人群中抽大麻、乱抛垃圾、浪费食物的行为引来批评浪潮。绝大多数大篷车成员拒绝在墨西哥申请避难，担心这一过程可能会导致他们被拘留或被驱逐出境。
 <br/>
 <br/>
 据报导，大篷车最初形成时规模约200人，进入墨西哥边境时增加到4,000人，穿越墨西哥时增加到约7,000人。目前，这些非法移民日夜兼程继续北上，人数已经过万。该队伍中包括许多曾经被美国驱逐出境的人。先前从美国被驱逐的人明确表示，他们打算非法越境，避开巡逻人员进入美国。
 <br/>
 <br/>
 墨西哥当局26日曾表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 如果愿意留在墨西哥南部两州，将为他们提供医疗、子女教育和临时工作。但被难民拒绝，他们坚持要朝美国行进。
 <br/>
 <br/>
 墨西哥移民保护局27日首次向这些非法移民提供食水和顺风车，此举或进一步激发美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）针对墨西哥的制裁。
 <br/>
 <br/>
 美国国防部长马蒂斯周日（28日）访问捷克期间透露，美军已开始运送围栏等装备到南部边境，应付即将从墨西哥前来的约7000名非法入境者。国防部官员透露，军方可再增派5000军人支援。
 <br/>
 <br/>
 川普周一（29日）发推文称，难民大军中混入很多黑社会成员及非常坏的人，他呼吁难民回国，称他们不会获准非法进入美国，直指〝这是入侵美国，我们的军队等待你们〞。
