### 蓬佩奥连续强硬表态：全面反击中共窃取知识产权
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 近期多次就中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的问题强硬表态。日前他对美国媒体直言，中国可能是美国面临的最大国安挑战，美国政府正在对中共政府各方面的恶行发起全面反击。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 10月31日接受了〝劳拉英格拉汉姆秀〞节目主持人英格拉汉姆（Laura Ingraham）的专访。
 <br/>
 <br/>
 当被问到美国如何应对中国窃取智慧财产权的问题时，蓬佩奥提到了美国政府本周指控10名试图窃取涉及敏感的商业航空与技术数据的中国人的事件，以及美国商务部将中国半导体生产商福建晋华列入禁售黑名单并限制其对外出口的事件。
 <br/>
 <br/>
 他表示，在川普总统的领导下，美国政府正在进行多方努力，准备全面反击中国（中共）政府，以促使中国在商业和国际法规则方面表现得像一个正常国家。他说，反击中国（中共）将从贸易开始，川普总统希望与中国进行公平和互惠的贸易。
 <br/>
 <center>
  <br/>

   </font>
  </a>
  <br/>
 </center>
 <br/>
 此前，蓬佩奥还曾经于10月26日接受美国电台主持人修伊特（Hugh Hewitt）的采访。当时他也曾坦言，中共盗窃知识产权，推行不公平贸易，对外军事扩张等行为，对美国人民的利益造成了威胁。他当时强调说：〝每一个挑战都会得到美国强而有力的回应，我们会在每一个领域进行应对。〞
 <br/>
 <br/>
 蓬佩奥还批评中共到处以投资贿赂其他国家的领导人。他说：〝美国欢迎在公平及互惠的基础上，与中国展开商业竞争，但当中国透过贿赂其他国家的领导人，以换取有损该国人民的基建项目时，我认为这种‘金钱帝国’的建立对这些国家是坏事，也必定对美国利益构成危险，我们每次都会反击。〞
 <br/>
 <br/>
 他还谈及宗教自由，提到了新疆的〝再教育营〞以及中共对基督徒等宗教信仰者的打压。他表示，川普总统已经要求美国政府严肃处理这些问题。今后美国政府对于那些不尊重人权的政府将不仅仅是口头批评，而是会动用美国和世界的影响力来应对这些对基本人权的挑战。
 <br/>
 <br/>
 据公开的资讯，美国自宣布对中国商品征收关税以来，从政治、经济和军事上对中共打出了一系列超级组合拳以反击来自中共的威胁。
 <br/>
 <br/>
 首先在贸易方面，川普政府在过去几个月内，先后宣布对总计2500亿美元的中国商品征收关税；10月10日，川普政府宣布一项扩大政府对外国投资美国公司审查的试点项目和临时法规，对外国投资在电信、半导体和计算机等多个具体技术行业实施额外限制。
 <br/>
 <br/>
 美国政府还针对那些具有战略动机、对美国技术优势和国家安全造成威胁的外资，精心制定了试点项目行业清单。
 <br/>
 <br/>
 在美国与墨西哥以及加拿大重新达成的〝美墨加自贸协议〞（USMCA）中，特别加入了一条〝毒丸条款〞，规定协议中的任何成员国，如果与〝非市场经济国家〞达成自贸协定，其它成员国将有权退出新协定。外界认为这个条款主要是针对的中共政府。
 <br/>
 <br/>
 在军事方面，川普总统今年8月签署生效的《2019财年国防授权法案》中坦言，美国的首要任务是〝与中共长期的战略竞争〞。该法案还要求扩充五角大楼向国会提交的有关中共军事的年度报告，其中提供中共势力通过媒体、文化机构、商业和学术界对美国进行渗透的详细信息。
 <br/>
 <br/>
 美国国家情报总监科茨（Dan Coats）曾公开表示，中共代表了一个更广泛的，更系统的威胁，包括恶意网络攻击，媒体操纵和施压策略，目标群体是企业和大学生，旨在提升中共的形象。
 <br/>
 <br/>
 在国家安全政策方面，美国国家安全委员会已经要求政府机构制定各种方案，限制中国科技公司参与即将在美国推出的新5G网络建设。
 <br/>
 <br/>
 川普总统最近签署的国家网络安全战略，授予美国多个政府部门及执法机构更大的权限，以便让这些机构可以主动出击，清除中共等境外势力针对美国的网络犯罪和国家攻击。
 <br/>
 <br/>
 为了防范包括中共在内的外来势力干扰美国选举，川普甚至在今年9月签署了一份行政令，允许美国政府对干涉美国选举的外国实体或个人实施制裁。
 <br/>
 <br/>
 （记者唐迪综合报导/责任编辑：戴明）
