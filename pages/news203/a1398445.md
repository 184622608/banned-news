### 民主党掌控众院 对川普内政外交影响多大？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】周二（11月6日）晚间，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 初步结果出台，共和党保住并扩大了其参议院优势，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 则时隔8年重掌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 。该选举结果势必在一定程度上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_影响.html" target="_blank">
  影响
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府的内政外交，但预计影响程度非常有限。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 之所以引发全球性关注，是因为其对内势必
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_影响.html" target="_blank">
  影响
 </a>
 民众的切身利益，对外则会通过外交政策影响其它很多国家的政治、经济运行。
 <br/>
 <br/>
 就对内影响而言，中期选举后，最先波及的就是通俄门调查、移民政策、医保改革等问题。
 <br/>
 <br/>
 事实上，早在大选前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 大谈〝蓝波〞之说时，他们就一再威胁称，一旦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 民主党大权在握，他们就将启动针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府的若干项调查，甚至扬言弹劾总统。
 <br/>
 <br/>
 有鉴于此，川普于周三上午发推指出，民调显示多数美国选民反对所谓的穆勒（Robert Mueller）调查，并再次批评这是〝猎巫行动〞。
 <br/>
 <br/>
 <blockquote class="twitter-tweet" data-lang="en">
  <p dir="ltr" lang="en">

  </p>

  <a href="https://twitter.com/realDonaldTrump/status/1060194964351660033?ref_src=twsrc%5Etfw">
   November 7, 2018
  </a>
 </blockquote>
 <br/>
 <div style="clear:both;display:block;">
 </div>
 <br/>
 不过，其实从川普当选伊始，民主党就开始筹划如何赶他下台，但从2017年5月至今，尽管特别检察官穆勒绞尽脑汁调查了17月有余，但无论是关于所谓的〝通俄〞、还是〝妨碍司法〞，都没有找出任何令人信服的证据。因此，虽然民主党已对弹劾动议叫嚣两年，但始终无法如愿。即使如今掌控了众议院，毫无根据的指控也难以成立。
 <br/>
 <br/>
 但是，由于美国任何立法都必须获得参众两院通过，所以，在民主党成为众议院多数党后，川普要想在移民、医改等方面通过自己的立法议程，势必会增加难度。
 <br/>
 <br/>
 就在中期选举前一周内，川普接连提出两项有关移民政策的重大改革：一是打算签署行政令废除公民出生地原则（birthright citizenship），二是针对大篷车移民问题宣布修改庇护规定，防止移民欺诈入境。
 <br/>
 <br/>
 川普后来在幕僚的建议下表示将把废止公民出生地原则问题提交最高法院裁决，但在改革庇护政策及相应的筑边境墙资金上，预计将会遭到众议院民主党阻挠。
 <br/>
 <br/>
 另外，在2019财年的预算问题上，估计川普政府与民主党掌控的众议院之间还会有一番较量。川普政府因预算问题曾两度关门，后来川普虽然签署了众议院拟定的综合预算法案，但他曾警告众议院议长瑞恩（Paul Ryan），〝我再也不会签这样的法案了〞。
 <br/>
 <br/>
 在税改问题上，虽然民主党反对减税，但在给民众和企业带来巨大好处的减税已广受欢迎的情况下，民主党想通过众议院立法来改变当前的状况恐怕是在痴人说梦。
 <br/>
 <br/>
 不过，共和党继续掌控参议院并至少已增加两个议席，这对政府工作的正常运行非常重要。参议院负责审批政府和司法官员的任命，但由于民主党的阻挠和拖延，川普提名的政府官员及法官很多尚未得到批准，大大影响了川普政府的正常行政运转。
 <br/>
 <br/>
 共和党在参议院的优势增强后，官员审批过程预计可加速，而当再次遇到类似最高法院大法官卡瓦诺（Brett Kavanaugh）之确认的问题时，共和党也不至于为每票必争而耗尽心力。
 <br/>
 <br/>
 正因如此，在精力有限的情况下，川普在选举前把助选重点放在了参议院上。
 <br/>
 <br/>
 选举夜，川普针对开票结果发出的第一条推文就是：〝今晚取得巨大成功。谢谢大家！〞
 <br/>
 <br/>
 <blockquote class="twitter-tweet" data-lang="en">
  <p dir="ltr" lang="en">
   Tremendous success tonight. Thank you to all!
  </p>
  — Donald J. Trump (@realDonaldTrump)
  <a href="https://twitter.com/realDonaldTrump/status/1060022696703070208?ref_src=twsrc%5Etfw">
   November 7, 2018
  </a>
 </blockquote>
 <br/>
 <div style="clear:both;display:block;">
 </div>
 <br/>
 在对外政策方面，虽然民主党夺取众议院可能会对川普有一定影响，但预计影响程度非常有限。
 <br/>
 <br/>
 首先，美国实行立法、司法、行政三权分立的原则，立法机构并不能直接干预政府推行政令。
 <br/>
 <br/>
 其次，由于川普上任后在解决朝核威胁、打击恐怖主义等一系列问题上取得突破性进展，因此，美国的国际威望大增，民主党也不可能使川普回到奥巴马时期所推行的对朝绥靖政策的老路上去。
 <br/>
 <br/>
 在退出伊朗核协议问题上，虽然以奥巴马为首的一些民主党人表示反对，但他们无法改变川普的决定，只能发发批评之声而已。
 <br/>
 <br/>
 当前最受关注的既涉及外交、又关系美国民众利益的是美中贸易冲突问题。中共非常关注此次美国中期选举，就是因为其在走投无路的情况下寄希望于国会民主党能够掣肘川普。
 <br/>
 <br/>
 但是，中共显然打错了算盘。一个不争的事实是，美国两党在对华立场上渐趋一致，预计民主党掌控众议院不会影响川普的强硬对华政策。
 <br/>
 <br/>
 事实上，美国参众两院的民主党领袖舒默（Chuck Schumer）和佩洛西（Nancy Pelosi）都支持川普对中共采取强硬政策，他们在关税、贸易、知识产权、人权等问题都一再批评中共，支持鹰派立场。
 <br/>
 <br/>
 当然，由于民主党在选举中拿下了众议院，川普在其本届任期内势必面对更多困难、甚至会给其2020年的竞选连任之路增加难度，但就整体而言，的确如川普所说，共和党在本次选举中取得了很大成功。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
