### 大篷车接近边境 彭斯：委内瑞拉资助
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】来自洪都拉斯的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 难民，人数已超过7千人。目前，他们跨过墨西哥边境，继续向美国南部边境前进。美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 周二表示，有左翼组织和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_资助.html" target="_blank">
  资助
 </a>
 这场大篷车运动。
 <br/>
 <br/>
 周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 难民抵达墨西哥南部边境城市维斯特拉，人数增加到大约七千到一万人。他们已疲劳不堪，在城市广场上休息，补充食物和水。他们还将继续向北，朝着美国南部边境行进。
 <br/>
 <br/>
 周二，在白宫记者会上，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 回答记者提问，表示大篷车难民有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_资助.html" target="_blank">
  资助
 </a>
 。
 <br/>
 <br/>
 美国副总统彭斯：〝我和洪都拉斯总统赫尔南德斯通过话，他告诉我通过墨西哥边境、正朝着（美国）南部边境前进的大篷车难民，是由左翼团体组织、受委内瑞拉资助的。〞
 <br/>
 <br/>
 彭斯说，美国执法部门每天都要阻止恐怖份子进入美国，难保没有这些人混入难民中。
 <br/>
 <br/>
 美国副总统彭斯：〝美国每天都要介入和阻止10多名恐怖份子或是有嫌疑的恐份子进入我们国家，所以很难确保不断增长的大篷车难民中没有中东人。〞
 <br/>
 <br/>
 川普总统之前也推文表示，有罪犯和不知底细的中东人混在难民中。
 <br/>
 <br/>
 周三一早，川普连发推文，以欧洲为例，佐证严守边境的重要。
 <br/>
 <br/>
 推文说：〝那些想要为非法移民争取权益的人，好好看看过去五年欧洲的情况吧。一片混乱！他们只希望能重新做出决定。〞
 <br/>
 <br/>
 另一则推文说：〝我们是伟大的主权国家。我们有强大的边境，永远不允许非法进入美国。〞
 <br/>
 <br/>
 川普周二的推文，还引用了前总统奥巴马在担任伊利诺州参议员时的一段讲话。
 <br/>
 <br/>
 美国前总统奥巴马：〝那些非法入境者，这样做是不尊重法律规定，不顾那些守法的人。我们不能允许人们涌入美国，没有被发现，没有移民证件，没有调查（背景）。〞
 <br/>
 <br/>
 川普说：〝我百分百赞同奥巴马总统所言〞。
 <br/>
 <br/>
 新唐人记者安心、张琪综合报导
</div>
