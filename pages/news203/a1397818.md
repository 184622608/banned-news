### 川普密苏里州助选演说 民众支持
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】周四晚，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_密苏里州.html" target="_blank">
  密苏里州
 </a>
 参加了两场助选集会，谈到移民、美中贸易等话题。当地大批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民众.html" target="_blank">
  民众
 </a>
 赶到会场，表达对川普的支持。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 首先提到，移民政策要改革。
 <br/>
 <br/>
 美国总统川普：〝每年有成千上万非法移民生育的孩子，自动成为美国公民，都是因为这种疯狂政策（出生公民权），我们能结束这个政策。〞
 <br/>
 <br/>
 川普表示，他和习近平已经通话，谈了贸易等问题。
 <br/>
 <br/>
 美国总统川普：〝我说我们得达成公平的（贸易）协议，他（习近平）也这么想，他们都这么想。〞
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民众.html" target="_blank">
  民众
 </a>
 表示，赞同川普和他的政策。
 <br/>
 <br/>
 办公室经理Julie Bacues：〝除了里根总统，他是第一个告诉美国人真相的总统。真相是：我们的政客所做的一切不是为了我们，不是为了美国的利益，而是为了自己。川普总统立下承诺，履行承诺，他每一天都在为我们工作。〞
 <br/>
 <br/>
 办公室经理Julie Bacues：〝为了神的荣耀，川普来到这里，（神）选中并把他放在这个位置。他不把荣耀归于自己，而是归于神。〞
 <br/>
 <br/>
 工具制造商David Hardman：〝我做了近三十年生意，对这个国家来说川普是最好的。我们想边境安全，我们想业务增长，我们想就业重新回来，我们有非常好技术员工，我们需要让他们就业。我们爱川普。〞
 <br/>
 <br/>
 软件工程专业大学生Jack Bell: 〝很多人就是政治正确，而川普说到做到。他是我们这代人需要的强有力领导者，我们需要更多人来鼓舞我们。大学教授非常偏向民主党，误导了很多人。〞
 <br/>
 <br/>
 新唐人记者陈虎、陈晓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_密苏里州.html" target="_blank">
  密苏里州
 </a>
 哥伦比亚报导
</div>
