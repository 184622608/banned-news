### G20峰会美中休战？ 专家悲观预期
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间会晤，商讨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易争端.html" target="_blank">
  美中贸易争端
 </a>
 的化解方案，全球对此高度关注。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 11月30日开幕。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 定于12月1日举行工作晚宴。在此之前，美中经贸团队已多次接触，草拟协议，供双方首脑会面时定夺。
 <br/>
 <br/>
 多家外媒披露，北京期望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 能达成一份贸易战〝休战〞协议。
 <br/>
 <br/>
 谢田：〝中共现在最迫切的要求，就是停止关税战升级。因为现在10%的关税，已经导致产业链的转移，投资的撤出。中共现在最害怕的就是，从明年1月1号开始（关税税率）从10%涨到25%，或者把剩下$2670亿也加上关税。〞
 <br/>
 <br/>
 据《华尔街日报》和《金融时报》报导，中方提出，如果美国暂缓施加进一步的关税，北京将同意放宽采购美国农产品和能源产品的限制；并愿意就〝结构性问题〞和美国谈判。例如〝中国制造2025计划〞，以及对大型国企的控制和补贴等。此前中方一直拒绝讨论这些问题。
 <br/>
 <br/>
 但美国专家担忧，中共还是想〝以拖待变〞。
 <br/>
 <br/>
 俞伟雄：〝其实从川普上台后，美方已经跟中方在谈所有这些问题，已经谈了2年了。中国以〝暂缓实施〞来做接下来谈判的前提，我现在不清楚还可以谈什么东西？（美国的要求）已经很清楚了。重点是中国有没有诚心诚意要来做这个改变。〝以拖待变〞，过去中国政府已经用了很久了。〞
 <br/>
 <br/>
 谢田：〝如果你看中共对国内的宣传，号召要‘自力更生、勒紧裤腰带’，你就知道中共并没有做真正让步的安排。它是想打长期战，或者在中美贸易崩盘的时候走另外一条路。〞
 <br/>
 <br/>
 《金融时报》报导，北京提出，如果川习会达成休战协议，中方代表团将于12月12日到15日，前往华盛顿展开新一轮谈判。
 <br/>
 <br/>
 而川普周四表示，他不确定是否会接受协议。
 <br/>
 <br/>
 俞伟雄：〝即使有一些协议达成，我觉得重点还是美方1月1号这个关税是不是不实施了，我觉得那是一个重点。因为表面上大家都可以讲，我们这次G20的会议进行的很好，〝但是我还是实施加税〞，那就表示其实没有什么真正具体的协议达成，中共没有释出具体的诚意跟承诺来改变。〞
 <br/>
 <br/>
 《华尔街日报》报导说，川习会前，中国经济乌云压顶，周五公布的制造业PMI跌至50.0的荣枯线，标志着中国经济放缓加剧。如果无法达成休战协议，2019年中国经济下行的风险将大幅上升。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
