### 美高院做出有利裁决 川普打击选举作弊获重要胜利
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】周一（10月22日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院.html" target="_blank">
  最高法院
 </a>
 裁定，暂停执行地方法院就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人口普查.html" target="_blank">
  人口普查
 </a>
 问题质询
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商务部长.html" target="_blank">
  商务部长
 </a>
 罗斯（Wilbur Ross）的裁决。这是川普政府在选举权问题上取得的一项重要司法胜利。
 <br/>
 <br/>
 周一晚，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院.html" target="_blank">
  最高法院
 </a>
 临时叫停了一项
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 地方法院对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商务部长.html" target="_blank">
  商务部长
 </a>
 罗斯的裁决。该法院的一名联邦法官此前裁定，原告可就商务部决定在2020年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人口普查.html" target="_blank">
  人口普查
 </a>
 中询问受查者是否具有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公民身份.html" target="_blank">
  公民身份
 </a>
 一事质询罗斯。
 <br/>
 <br/>
 今年4月，一些利益集团与纽约州司法部长联合向位于曼哈顿的一家联邦法院提起诉讼，指控川普政府在2020年的人口普查中增加了公民身份问题。由于负责这项工作的美国人口普查局（U.S. Census Bureau）受商务部监管，所以原告把矛头指向了罗斯。
 <br/>
 <br/>
 按照美国宪法的要求，每十年要进行一次人口普查，以便在各州中分配国会众议院的席位。
 <br/>
 <br/>
 原告声称，公民身份问题〝将致命地破坏人口数量的准确性〞。
 <br/>
 <br/>
 纽约是众所周知的由民主党占优势的蓝州，商务部的决定或将影响民主党选票。因为在某种程度上，非法外侨、非居民或少数族裔可能会因害怕被捕或被驱逐出境而避免参与投票，而这部分人刚好是民主党的支持者，是其潜在的选民。
 <br/>
 <br/>
 商务部2012年的一份报告显示，在2010年的人口普查中，大约有150万少数族裔居民没有被计入。
 <br/>
 <br/>
 纽约地方法院法官弗曼（Jesse M. Furman）裁定，原告可以在诉讼中质询罗斯。在另一份文件中，弗曼甚至指责罗斯可能已采取歧视性行动。
 <br/>
 <br/>
 该案目前定于11月5日开始审判。
 <br/>
 <br/>
 美国商务部3月26日宣布，将在2020年人口普查中恢复询问受查者是否具有公民身份。这一决定随即遭到民主党激烈反对，几个支持民主党的蓝州甚至马上扬言要起诉川普政府。
 <br/>
 <br/>
 从1950年起，美国在进行全面人口普查时不再询问公民身份问题，政府从而无法确切估计美国境内到底有多少公民、多少非公民和非法居留者。
 <br/>
 <br/>
 白宫新闻发言人桑德斯（Sarah Sanders）在商务部宣布决定后指出，在人口普查中询问公民身份是一个标准惯例，有助于保护美国公民的选举权。她还表示，实际上，在1965年以后的历次人口普查中，也都包含公民身份问题，只有2010年奥巴马执政时除外。
 <br/>
 <br/>
 大量证据证实，2016年美国总统大选中存在选民欺诈现象，有不少人冒充公民投票。川普当选后曾多次表示，2016年大选的选票有问题，去年1月，他曾签发行政令予以调查。
 <br/>
 <br/>
 美国司法部长塞申斯（Jeff Sessions）本月15日发表讲话，称弗曼的裁决是新型司法激进主义的例证，地方法院法官通过禁令等方式干涉行政部门运作是违背宪法的。
 <br/>
 <br/>
 塞申斯还宣布决定动用紧急上诉进行干预，以便在最短时间内让最高法院介入裁决。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
