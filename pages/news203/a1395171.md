### 【禁闻】千人计划又一例 美华人学者被疑间谍
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】直到今年4月为止，55岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢克平.html" target="_blank">
  谢克平
 </a>
 还是美国德州大学安德森癌症研究中心的肠胃病学教授，但现在他被怀疑是中共的经济间谍。谢克平是中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞的学者之一，此前还因为涉嫌儿童色情被捕。
 <br/>
 <br/>
 NBC新闻10月9号引述几名美国官员的话说，联邦调查局正在调查德州大学安德森癌症研究中心的教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢克平.html" target="_blank">
  谢克平
 </a>
 ，他涉嫌将机构的尖端研究传递给中共政府。
 <br/>
 <br/>
 谢克平是中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞的学者之一。据中共官方媒体报导，谢克平通过〝千人计划〞受聘于同济大学上海东方医院，也曾参加2009年在北京举行的〝第一届海外高层留学人员座谈会〞。
 <br/>
 <br/>
 而美国情报部门认为〝千人计划〞与间谍活动有关。NBC新闻引述一份非机密FBI备忘录中说，〝中共的千人计划，通过经济间谍活动和盗窃知识产权，对美国企业和大学构成严重威胁。〞
 <br/>
 <br/>
 华府人权律师叶宁：〝谢克平遭到调查实际上是一个警讯，这就说明美国现在开始重视，开始觉醒的过程。开始对中共对美国全面的锐实力入侵和渗透呢，开始进行调查，这一切还都只是刚刚开始。〞
 <br/>
 <br/>
 谢克平此前被德州警方逮捕是因为涉嫌儿童色情。NBC新闻说，法庭文件显示，休斯顿警方1月份开始调查谢克平，是否涉及〝篡改政府记录〞（Tampering with a Government Record）的欺诈行为。文件没有说明具体涉嫌欺诈的内容，但是记载，警察在搜查谢克平的住家及办公室时，查获了两个装有各种儿童色情图片的设备。
 <br/>
 <br/>
 之后谢克平交出护照，获得保释。他在4月份辞去了年薪25万美元的工作。
 <br/>
 <br/>
 法庭文件中没有提到间谍调查。联邦调查局休斯敦办事处发言人加尔扎（Christina Garza）对调查没有证实也没有否认。
 <br/>
 <br/>
 华府人权律师叶宁分析，儿童色情在美国属于联邦重罪惩罚的对象，但显然休斯顿警方不会无缘无故针对谢克平这样白领阶层的非现行刑事犯罪进行突然侦查。
 <br/>
 <br/>
 叶宁：〝虽然表面上看起来是德州警方抓的人，但实质上联邦在这个里面是有份的。但是呢，联邦现在很显然在这个事情当中，他不想在一开始就给外界一个感觉，他卷入的太多。〞
 <br/>
 <br/>
 谢克平的律师梅斯（Nathan Mays）告诉NBC新闻，执法部门在谢家总计没收了88个设备，包括手机、硬盘、记忆棒和各种类型的电脑。两个硬盘被标记为〝龙虎一〞及〝龙虎二〞。估计查封了约40TB的数据。
 <br/>
 <br/>
 此前已经有多名加入〝千人计划〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华裔科学家.html" target="_blank">
  华裔科学家
 </a>
 ，因涉嫌转移美国敏感技术而遭到美国逮捕或判刑。谢克平在8月份受到指控，而8月初FBI在休斯敦学术界召开了一个史无前例的会议，召集德州医学中心和全州的100名学术界领军人物，警告他们要提防来自外国对手，尤其是来自内部的威胁。
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝针对这些涉及到千人计划学者的调查呀，针对去突出人物进行逮捕，他其实释放出的信号就是，美国政府现在对中共在科技领域、在教育领域、在知识产权领域，会采取一系列的强硬反击的措施。〞
 <br/>
 <br/>
 叶宁表示，正是因为中共近年来抛弃〝韬光养晦〞加剧扩张，才引发了美国的警觉，美国政府的全面遏制措施，和中共的扩张野心〝对头撞〞，今后估计还会有类似案件出现。
 <br/>
 <br/>
 而唐靖远认为，美国副总统彭斯此前的讲话标志着美国对中共策略的全面转变，随着美国社会对抵制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共渗透.html" target="_blank">
  中共渗透
 </a>
 形成共识，相信这一基本国家战略会长期持续，不会受到国内政治变化的影响。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/陈建铭
