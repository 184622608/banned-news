### 川普回应包裹案：绝不让政治暴力在美国扎根
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】周五，美国执法人员逮捕了〝炸弹邮包〞案
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌犯.html" target="_blank">
  嫌犯
 </a>
 。对于该男子是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支持者.html" target="_blank">
  支持者
 </a>
 ，川普总统表示，这并不能困扰他，但他指出，他绝不容忍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治暴力.html" target="_blank">
  政治暴力
 </a>
 在美国扎根。
 <br/>
 <br/>
 周五，本周来制造了一系列
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_可疑包裹.html" target="_blank">
  可疑包裹
 </a>
 案的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌犯.html" target="_blank">
  嫌犯
 </a>
 谢亚克（Cesar Sayoc）在佛州被捕。据目前已知的资讯，谢亚克现年56岁，有犯罪前科，但他却是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支持者.html" target="_blank">
  支持者
 </a>
 。
 <br/>
 <br/>
 得知嫌犯被捕后，川普在推特上盛赞，美国各执法部门的工作、技术、决心都非常了不起，简直令人难以置信。
 <br/>
 <br/>
 <blockquote class="twitter-tweet" data-lang="en">
  <p dir="ltr" lang="en">
   I want to applaud the FBI, Secret Service, Department of Justice, the U.S. Attorneys’ Office for the Southern District of New York, the NYPD, and all Law Enforcement partners across the Country for their incredible work, skill and determination!
  </p>
  — Donald J. Trump (@realDonaldTrump)
  <a href="https://twitter.com/realDonaldTrump/status/1055881545628704769?ref_src=twsrc%5Etfw">
   October 26, 2018
  </a>
 </blockquote>
 <br/>
 <div style="clear:both;display:block;">
 </div>
 <br/>
 川普在当天白宫举行的非裔少年领袖论坛上演讲时表示，这些恐怖行为是卑鄙的，在美国没有地位，〝我们绝不让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治暴力.html" target="_blank">
  政治暴力
 </a>
 在美国扎根〞。
 <br/>
 <br/>
 他还承诺，作为总统，他将尽其所能，立即制止这种恐怖行为。
 <br/>
 <br/>
 当有记者问及嫌犯是川普支持者这一点是否会令其困扰时，川普回答——不会。
 <br/>
 <br/>
 川普表示，他只是听说谢亚克喜欢他胜过喜欢其他人，但他并没看到这一点。
 <br/>
 <br/>
 谢亚克有犯罪历史，曾被指控犯有重大盗窃罪和分发类固醇重罪，而且早在2002年8月，他就曾被指控犯有威胁使用破坏性装置的罪行。
 <br/>
 <br/>
 显然，谢亚克是个有犯罪倾向的人，而这与他是否喜欢或支持川普无关。
 <br/>
 <br/>
 调查发现，谢亚克的白色面包车上贴有支持川普和副总统彭斯（Mike Pence）的图片。网上还晒出了他头戴〝川普帽〞参加支持川普的集会的视频。
 <br/>
 <br/>
 另外，他的脸书和推特账号显示，他激烈批评希拉里（Hillary Clinton）等民主党人，具有鲜明的党派性。
 <br/>
 <br/>
 谢亚克被捕后，脸书和推特公司很快移除了他的账号。
 <br/>
 <br/>
 谢亚克是川普支持者，这立即成为自由派攻击川普的口实和焦点。谢亚克被称为〝MAGA Bomber〞（‘让美国再次伟大’轰炸机），并在社交媒体上迅速流传。
 <br/>
 <br/>
 从前总统奥巴马（Barack Obama）和前国务卿希拉里（Hillary Clinton）受到可疑包裹威胁后，川普多次强调美国人民团结的重要性，并一再重申将坚决制止政治暴力的决心。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
