### 川普降温川习会：预计中美不会在G20达成协议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】中美首脑通话后，中共媒体开始努力〝展望贸易和解〞。而美媒日前刊登对美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的专访内容，无异于给中共宣传泼了一盆冷水。报导称，川普表示，不认为双方会在本月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间达成贸易协议。
 <br/>
 <br/>
 美国《华盛顿时报》美东时间周五（11月2日）报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在前一天接受该报独家专访时说，尽管中美两国在贸易问题上有所进展，但他并不认为双方会在本月底的G20首脑峰会期间宣布达成协议。
 <br/>
 <br/>
 川普说：〝最终我们肯定会跟中国达成一个非常合理的协议。我可以告诉你，他们希望跟我们达成协议。〞
 <br/>
 <br/>
 就在《华盛顿时报》发出这篇报导的当天早些时候，彭博新闻社援引四位知情人士的消息称，川普希望在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上与北京达成协议，而且已经指示美国核心官员起草美方可能提出的协议条款。
 <br/>
 <br/>
 但美国消费者新闻与商业频道（CNBC）记者埃蒙・杰弗斯随后在推特上透露，一名白宫高官对他说，跟北京的谈判〝还有很长的路要走〞。
 <br/>
 <br/>
 杰弗斯还在CNBC电视节目上表示，彭博新闻社记者可能确有白宫消息来源，但他所得到的消息是华盛顿不会跟北京很快达成协议。
 <br/>
 <br/>
 周五上午的美国股市，也随着上述几则消息应声起落。
 <br/>
 <br/>
 遏制中共已是美国朝野的共识，与北京和解的消息或能提振美国股市，但不利于川普。
 <br/>
 <br/>
 目前，随着美国政治博弈的激化，美国媒体的报导也呈现出越来越混乱的局面。
 <br/>
 <br/>
 今年8月，在中共商务部副部长王受文赴美谈判前夕，美国《华尔街日报》引述消息称，中美双方正在规划一个谈判〝路线图〞，准备在11月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 上结束贸易战，王受文赴美是在为此做准备。
 <br/>
 <br/>
 不过，川普次日就接受路透社访问，否认美方有什么中美谈判〝时间表〞，并表示他对王受文来访并不抱希望。
 <br/>
 <br/>
 贸易战以来，川普一直强调，中美必须达成公平对等的贸易协议。
 <br/>
 <br/>
 而据中共官媒通稿，在周五的中美首脑通话中，北京也向川普表示，〝中方已就中美关系多次阐明原则立场〞。这似乎也传递了中方在关键问题上不会让步的信息。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
