### 俄战斗机拦截美侦察机 五角大楼：不安全且不专业
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】美国国防部官员与海军发表声明称，周一（5日），美国一架海军侦察机在飞越黑海上空的国际领空时，遭到一架俄罗斯战斗机以不安全且不专业的动作拦截。
 <br/>
 <br/>
 五角大楼发言人埃里克・帕洪说，美国这架海军EP-3侦察机当时是在希腊索达湾（Souda Bay）外海执行任务。而且EP-3执勤时应答机是开通的，但俄罗斯苏-27战机（SU-27）并没有尝试与其进行沟通，就以〝非常近〞的距离掠过侦察机之后打开后燃器，气流使EP-3侦察机整个机体发生震颤。
 <br/>
 <br/>
 他说：〝这类行为绝对是没有理由的。〞
 <br/>
 <br/>
 美国海军在声明说：〝这架苏恺27型战机高速飞至任务机正前方，将我们的机组人员置于危险境地，属于不安全的互动。这架出动拦截的苏恺27型战斗机后来还再次飞近EP-3侦察机，并在倾斜转弯时启动后燃器。在第一次互动后，EP-3机组人员回报遇到乱流，第二次则感受到震动。〞
 <br/>
 <br/>
 声明中说，这一行为是不负责任的，并〝使美军驾驶员和机组人员陷入危险。我们期望他们（俄罗斯驾驶员）的行为符合为保证安全和防止事故而制定的国际标准。〞
 <br/>
 <br/>
 官员目前还无法估计俄罗斯战机距离美国侦察机有多近，但从俄方飞行行为可以判定这次遭遇属不安全的互动。官员目前还不知道苏恺27战机是否搭载了武器。
