### 川普在安全社区会演讲 敦促尊重执法者
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统应邀在堪萨斯市2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国安全社区会议.html" target="_blank">
  美国安全社区会议
 </a>
 上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演讲.html" target="_blank">
  演讲
 </a>
 ，指出战略核心是恢复对执法者的尊重。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 指出政府正在快速降低犯罪率，这是八年来第一次召开全美安全会议。
 <br/>
 <br/>
 美国总统川普：〝在我任职总统以前的两年，暴力犯罪率增加了7％，谋杀案增加了20％以上。但我们正在很快扭转犯罪率，快得超过想像。〞
 <br/>
 <br/>
 川普指出太长时间以来，政治人士和激进人士抨击和诽谤执法系统，让犯罪分子过得轻松，而让守法人过得困难。
 <br/>
 <br/>
 美国总统川普：〝我们的战略核心是恢复对执法者的尊重。我们不会容忍攻击那些保护着我们社区的英雄。我们将不会允许这些发生。〞
 <br/>
 <br/>
 同时，川普在会上称赞了新提名的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部长.html" target="_blank">
  司法部长
 </a>
 威
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_廉姆・巴尔.html" target="_blank">
  廉姆・巴尔
 </a>
 (William Barr)。
 <br/>
 <br/>
 巴尔曾是老布什时期的司法部长。1991年他一度担任代理司法部长，上任仅三天，就遇到121名古巴囚犯在监狱劫持9名人质。 他迅速指示FBI袭击监狱，成功救出所有人质。
 <br/>
 <br/>
 美国总统川普：〝他（原先）任职期间，坚定不移地按法律行事，这是在场各位都愿意听到的。〞
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
