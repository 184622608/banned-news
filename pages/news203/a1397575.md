### 左媒选前打〝反川牌〞 专家警告其难奏效
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】连日来，美国左派媒体经常把社会恶性事件的责任推向川普（特朗普），专家认为，这是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_左媒.html" target="_blank">
  左媒
 </a>
 想间接打击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选情.html" target="_blank">
  中期选情
 </a>
 ，恐怕难以奏效。
 <br/>
 <br/>
 本周CNN有两档节目引发争议，先是节目嘉宾说川普总统比ISIS还恶劣，但CNN主持人对此评论无动于衷；接着则是该台的明星主播发表〝白人男子是美国最大恐怖威胁〞的言论。
 <br/>
 <br/>
 伴随匹兹堡犹太教堂的枪声，左派媒体又发起一轮对川普的舆论战。
 <br/>
 <br/>
 《纽约时报》29日的系列文章，将匹兹堡犹太教堂枪击案归咎于川普，说他是〝反犹太阴谋论推动者〞。
 <br/>
 <br/>
 但周二，川普与第一夫人梅兰妮亚，携犹太裔女婿库什内及信仰犹太教的女儿伊万卡亲赴宾州匹兹堡慰问。
 <br/>
 <br/>
 美国民主研究所主任 Patrick Basham：〝近日美国主流媒体试图将所有坏消息都和川普联系起来，并通过某种方式暗示这些与川普及他过去的论调有关。〞
 <br/>
 <br/>
 除了匹兹堡枪击案，邮寄炸弹包裹的嫌犯名字也与川普名字同时出现在主流媒体大头条上。26日，当炸弹邮寄包裹的嫌犯身份萨罗克（Cesar Sayoc）被确认后，纽时新闻报导标题为〝在萨罗克炸弹客身份前，他是川普的超级粉丝〞；27日《华盛顿邮报》的新闻标题则是〝包裹炸弹客如何从川普那里获得启发〞。
 <br/>
 <br/>
 美国民主研究所主任 Patrick Basham：〝反川普媒体试图间接辅助民主党的竞选，通过将反川普与竞选最大化关联的方式。不过我不认为这会有效，正如2016年他们没有成功一样。〞
 <br/>
 <br/>
 多方分析，民主党及反川普媒体炒作的〝蓝色浪潮〞目前还没出现。而实际情况是，连加州这样的民主党大本营，有不少选区的国会众议员的争夺都势均力敌。
 <br/>
 <br/>
 美国民主研究所主任 Patrick Basham：〝反川普媒体意识到选举结果可能与他们在今年所渲染的画面不同。所以未来几天直到下周二的选举日，我们可以预见大多数主流媒体上的反川普报导会持续增加。〞
 <br/>
 <br/>
 大选日倒数6天，川普马不停蹄。从周三（10月31日）至11月5日，川普6天要走访8个州参加11场助选活动，11月5日更是一天三场连轴转。
 <br/>
 <br/>
 Harris Poll的民调，川普是今年选民出来投票的首要激励因素，无论他们是支持还是反对。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
