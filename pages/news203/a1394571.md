### 飓风迈克尔形成 佛州26县进入紧急状态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】新的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风迈克尔.html" target="_blank">
  飓风迈克尔
 </a>
 行程，将袭击佛州，当地已经开始准备预防，并有26个县进入紧急状态。
 <br/>
 <br/>
 周一，迈克尔已演变成1级飓风，最大持续风速至少75英里，可能在周三早间登陆佛州，届时很可能演变成3级飓风，给当地带去4-12英寸降雨，并可能激发危及生命的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪水.html" target="_blank">
  洪水
 </a>
 。
 <br/>
 <br/>
 迈克尔周日过境墨西哥和古巴，给当地带去强风豪雨，随后向佛州方向移动。
 <br/>
 <br/>
 周日，佛州州长宣布，迈克尔将极其危险，他已要求超过500名国民警卫队员待命，并备好了救援用的直升机，船隻和发电机等。
 <br/>
 <br/>
 佛罗里达州州长Rick Scott（10.07.2018）：〝今天我宣布佛州26个县进入紧急状态，在佛州大弯区走廊地带，迎战风暴（飓风）迈克尔。〞
 <br/>
 <br/>
 佛罗里达州立大学已关闭其主校区、塔拉哈西（Tallahassee）和巴拿马市（Panama City）校区。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
