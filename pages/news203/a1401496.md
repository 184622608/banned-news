### 阿拉斯加突发7级地震 部分地区发海啸预警
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】周五（11月30日）上午，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿拉斯加.html" target="_blank">
  阿拉斯加
 </a>
 州安克雷奇市（Anchorage）附近发生两次较大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 ，震级分别为里氏7.0级和5.8级，导致房屋震坏、公路塌陷，科迪亚克市（Kodiak）居民因担心海啸而纷纷逃往高处。
 <br/>
 <br/>
 据美国地质调查局（U.S. Geological Survey）监测，首次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 为7.0级，震中位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿拉斯加.html" target="_blank">
  阿拉斯加
 </a>
 最大城市安克雷奇北部大约7英里（12公里）处。该市约有300,000居民，地震发生时正是上班时间，人们纷纷逃到屋外或躲在桌下藏身。

 目前还没有关于伤亡情况的报告，但社交媒体上公布的照片显示，一些建筑物震出裂缝，房屋天花板震落，道路塌陷，安克雷奇机场附近的立交桥坍塌了很大一部分。
 <br/>

 <br/>
 当地电力公司称，地震还导致7,000 至10,000用户遭遇断电。断电导致交通路口的红绿灯熄灭，交通陷入混乱。不过，电力公司称，地震并没有损坏发电基础设施。
 <br/>
 <br/>
 有人在首次地震结束后返回室内，但大约5分钟后又发生了5.8级余震，人们再次急忙跑到街上。
 <br/>
 <br/>
 阿拉斯加南部沿海地区的库克湾（Cook Inlet）和南基奈半岛（Southern Kenai Peninsula）的部分地区发布了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海啸预警.html" target="_blank">
  海啸预警
 </a>
 。科迪亚克岛上的警察要求人们立即向高处疏散，因为预计10分钟后可能会有海啸袭来。
 <br/>
 <br/>
 正在阿根廷参加G20峰会的川普总统接到简报后迅速做出回应，他在推特上呼吁震区居民听从救援人员指令，并表示联邦政府将不遗余力地给予支援。
