### 最新证据：美国一通讯公司被中共黑客入侵
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】美媒报导，最新证据显示，美国一家主要通讯公司8月份在系统中发现被操纵过的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恶意软体.html" target="_blank">
  恶意软体
 </a>
 ，并将其移除。一名安全专家说，这是中共试图操纵运往美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍芯片.html" target="_blank">
  间谍芯片
 </a>
 ，实施窃取情报行动的最新证据。
 <br/>
 <br/>
 10月4日，彭博社报导，美国硬体公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超微.html" target="_blank">
  超微
 </a>
 (SuperMicro)的供应系统遭到中共军方情报人员的渗透，它们生产的设备被植入微型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍芯片.html" target="_blank">
  间谍芯片
 </a>
 ，旨在获取商业和政府数据。
 <br/>
 <br/>
 9日，又有最新消息指，美国一家大型通讯公司也曾被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共黑客.html" target="_blank">
  中共黑客
 </a>
 侵入。透露此消息的是一名网路安全专家犹西．阿普尔鲍姆。
 <br/>
 <br/>
 报导称，这位前以色列军队情报官员提供了文件、分析和其他证据，详细说明中共情报官员如何在2013到2015年两年里，要求合同公司在超微公司的伺服器主板中，放置恶意芯片。而这些主板被提供给美国政府和公司之后，就可以向中共偷偷传输信息。
 <br/>
 <br/>
 美国主要有四大通信公司占据主导地位，分别是Verizon，At&amp;t，Sprint和T-mobile。不过，报导没有提供这家被侵入的大型通讯公司的名字，因为阿普尔鲍姆与他们签有保密协议。
 <br/>
 <br/>
 事实上，中共长期以来被怀疑通过在中国制造的硬体，从事大规模间谍行动。美国科技使用的大多数电子元件都是在中国制造。
 <br/>
 <br/>
 过去一年，包括华为和中兴等元件制造商，以及监控摄像头制造商海康威视，都受到美国政府的强烈怀疑和审查。
 <br/>
 <br/>
 彭博社在10月4日的报导中还表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 网路服务和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 运营的数据中心设备，也被中共情报部门植入间谍芯片，可能受到中共政府的监视。此消息导致两家公司股票大跌。
 <br/>
 <br/>
 美国调查人员发现：那些间谍芯片在设备制造过程中被插入设备，并且是由中共军方人员实施。在Supermicro，中共间谍似乎发现了实施〝供应链攻击〞的一条完美通道。
 <br/>
 <br/>
 一名美国官员说，调查人员发现，中共的这个行动最终影响到30家公司，包括一家大银行、政府承包商、和世界上最有价值的公司——苹果。
 <br/>
 <br/>
 苹果是Supermicro的重要客户，曾经计划从它那里订购3万台伺服器，以建立新的全球数据中心网路。据苹果公司3名内部人士说，在2015年夏天，苹果在Supermicro主板上发现了恶意芯片。次年便切断了跟Supermicro的关系。
 <br/>
 <br/>
 不过，包括苹果和亚马逊在内的多家公司，都否认报导的相关内容。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
</div>
