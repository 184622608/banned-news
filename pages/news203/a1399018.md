### 传习近平托基辛格带话愿让步 俄媒：假动作
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】北京正在运作缓和中美对抗，日前又邀请〝老朋友〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基辛格.html" target="_blank">
  基辛格
 </a>
 访华。日媒分析，中方想让基辛格充当和事佬，向美方传达准备做出一定让步的信息。俄媒则认为，北京或做出〝放弃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞的假动作，以寻求停止贸易战。
 <br/>
 <br/>
 美国前国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基辛格.html" target="_blank">
  基辛格
 </a>
 （Henry Kissinger）日前访问了中国，并获得了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 、王岐山和刘鹤等北京高层的连番接见。中方多次强调中美合作的〝重要性〞，并展示了希望与美国〝磋商和解〞的意愿。
 <br/>
 <br/>
 日本《外交学者》网站11月10日文章指，在谈话的字里行间，习近平都在让基辛格给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）传话，表明中方愿意在11月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 上与美国达成协议，包括在贸易上做出一些让步，只要不触及会损害中共政权的根本问题。
 <br/>
 <br/>
 韩国《朝鲜日报》11月9日报导也称，习近平的发言传递了在川习会上可以在某种程度上接受美方要求的妥协信息。但习近平也表示要美国〝尊重中国选择的发展道路〞，这表明中方不会在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞等关键问题上让步。
 <br/>
 <br/>
 另外，中美本月上旬在华盛顿举行了第二轮外交安全对话。这也被解读为中方有意缓解双边关系的信号。美媒此前消息，这次本拟10月召开的对话曾被中方一度取消。
 <br/>
 <br/>
 《海峡时报》11月11日引述分析指，近期的时局发展似乎显示，北京准备在贸易问题上做出〝明显更多的让步〞，希望在某种程度上能安抚美国人，满足美方的要求。
 <br/>
 <br/>
 不过，外界普遍认为，〝中国制造2025〞计划牵扯的问题，事关中共的经济体制和政治根基，中方不会真正放弃。
 <br/>
 <br/>
 日前，美国总统川普表示北京〝放弃了中国制造2025〞，一度引发舆论关注。
 <br/>
 <br/>
 俄罗斯《独立报》引述俄国专家分析，北京可能在假装放弃该计划，并希望川普接受这种形式上的〝胜利〞，和中方达成协议。
 <br/>
 <br/>
 分析并指，中美贸易战爆发后，中方就叫停了对〝中国制造2025〞的宣传，但这并不意味着真正放弃了在科学和技术领域的发展计划。北京可以单独执行这些具体的项目，而不再谈论整体的计划。
 <br/>
 <br/>
 对于北京这种〝让步姿态〞能否换取华盛顿的和解，外界普遍不看好。有分析认为，即便双方能达成某种临时协议，也无法从根本上缓解贸易战，双方利益和价值观的根本冲突还会令对抗继续升级。
 <br/>
 <br/>
 日前，白宫强硬派、贸易高级顾问纳瓦罗（Peter Navarro）表示，由于中共一贯言而无信，所以即使美中达成协议，也无法相信中共能够真正执行。
 <br/>
 <br/>
 在刚刚结束的美国中期选举中，川普所在的共和党保住了参议院，但失去了众议院的多数党地位。不过，抵制中共已是两党共识，而且众议院的民主党领袖佩洛西（Nancy Pelosi）也是一名对中共强硬派人士。
 <br/>
 <br/>
 外界分析，这种局势会让中美达成贸易协议更加困难。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
