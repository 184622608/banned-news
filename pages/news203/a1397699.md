### 共和党中期选举保卫战  6战役20个摇摆区是关键
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 即将于11月6日登场，共和、民主两党都在紧锣密鼓地为这次选战作最后的冲刺，舆论界则纷纷从不同的角度来分析这次选战当前的战况与动向。有分析认为，共和党能否成功保住对国会的控制权，关键要看6场战役的战况，能否继续保住众议院的多数席位，关键要看20个摇摆选区的动向。
 <br/>
 <br/>
 据了解，通过在今年的中期选举，美国人将投票决定联邦参议院三分之一的席位和联邦众议院的全部席位，以及36州的州长和华盛顿等市的市长。
 <br/>
 <br/>
 目前的基本格局是：参议院方面，共和党共握有51席，民主党握有47席，另有2席独立党派（亲民主党派）席位。而这次需要重选的35个席位中，共和党只需要捍卫9个席位就稳操胜券。因此外界普遍判断，民主党这一次的联邦参议院选战将是其史上最艰难的一次。
 <br/>
 <br/>
 众议院方面，目前共和党握有235席，民主党握有193席，另有7席无投票权。众议院的435个席位全部需要重选，在这场选战中夺下218席的党派就能以众议院多数党的优势，在未来国策的制定中发挥重要作用。
 <br/>
 <br/>
 从近期美国各大媒体的民意调查讯息来看，共和党能否继续掌控国会，关键要看德州、密苏里、北达科他、印第安纳、亚利桑纳和佛罗里达州，这6州的选战情况。其中，德州的民主党众议员欧路克(Beto O'Rouke)，挑战声望颇佳的共和党联邦参议员克鲁兹(Ted Cruz)的战役最受外界瞩目。
 <br/>
 <br/>
 此外，在亚利桑纳州的第二选区、明尼苏达州两席联邦众议员的选战中，两党候选人的声望和民意支持几乎都势均力敌，也是外界特别关注的两场重要战役。
 <br/>
 <br/>
 至于众议院会否通过这次中选而改变共和党目前占据的优势，则需要观察20个摇摆选区的选战情况。
 <br/>
 <br/>
 这些摇摆选区包括：纽约和明尼苏达各三个选区；加州、佛罗里达、俄亥俄、德州和宾州各有两个选区；肯塔基、北卡罗来纳、新泽西和维吉尼亚各一个选区。
 <br/>
 <br/>
 政情分析人士指出，民主党这次如果想坐上多数党的大位，需要在选战中拿夺回至少23个席位。共和党要保住多数党的大位，则需要在上述20个摇摆选区中拿下12个席位。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
