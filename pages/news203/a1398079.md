 今年10月，被指与州长库默关系密切的前伊利郡民主党主席、54岁的佩金（Steven Pigeon），在联邦法庭承认，曾参与向州长库默提供非法的政治献金而触动联邦法律。
 <br/>
 检方的起诉文件指出，2014年加拿大蒙特利尔一家网上博彩公司Amaya的CEO巴阿佐夫（David Baazov），以一名中间人的名义，向当时竞选连任纽约州长的库默团队捐款25,000美元，佩金则参与其中并帮助隐瞒巴阿佐夫的身分。

    今年5月，州长库默签署一项行政命令，允许2万4千名假释犯拥有投票权，这些人中很多是性侵犯，而州法律是禁止性罪犯出现在校园内（投票点）的。

 <br/>
 另外，今年5月，州长库默签署一项行政命令，允许2万4千名假释犯拥有投票权，这些人中很多是性侵犯，而州法律是禁止性罪犯出现在校园内（投票点）的。
 <br/>
 <br/>
 这项行政命令已经在党内初选执行。对此，反对声音不断，已经有长岛地区的学校表示，不愿再继续作为投票站点。
 <br/>
 <br/>
 他们认为，包括性罪犯在内的假释犯在任何时间，都不应该进入校园。根据现行法律，假释犯投票必须得到假释官以及校方的同意，才能在校园投票站里投票。学校代表在听证会上表示，他们从来没有收到类似的征求通知。他们并认为库默将政策置于民众的安全之上。
 <br/>
 <br/>
 触动州法律的事件，让目前寻求第三届任期的库默，面临质疑。◇
