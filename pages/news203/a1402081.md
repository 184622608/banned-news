### 老布什国葬 历任总统齐聚 小布什致悼词
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】美国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国葬.html" target="_blank">
  国葬
 </a>
 仪式，周三上午11点在华府的华盛顿国家大教堂举行。从川普到卡特所有现任和前任总统都出席了葬礼。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小布什.html" target="_blank">
  小布什
 </a>
 在仪式上为父亲献上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悼词.html" target="_blank">
  悼词
 </a>
 。
 <br/>
 <br/>
 周三是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 的全国哀悼日。上午10点开始，老布什的灵柩从国会的圆形大厅，移动到华盛顿国家大教堂。
 <br/>
 <br/>
 记者：我身后就是葬礼车队。老布什不仅是第41届美国总统，也担任过两届副总统，前中情局局长，和国会众议员。他是第43届总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小布什.html" target="_blank">
  小布什
 </a>
 的父亲。
 <br/>
 <br/>
 美东时间11点，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国葬.html" target="_blank">
  国葬
 </a>
 仪式正式开始。
 <br/>
 <br/>
 现任总统川普，以及目前仍然在世的美国前总统，包括奥巴马、小布什、克林顿以及卡特，都出席了老布什的葬礼。
 <br/>
 <br/>
 共有四人致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悼词.html" target="_blank">
  悼词
 </a>
 ，除了小布什之外，另外三人分别是加拿大前总理穆罗尼，老布什的好友、美国前国会参议员辛普森，以及老布什传记作者米查姆。他们以不同身份，回顾老布什从二战海军飞行员到总统的一生。
 <br/>
 <br/>
 周一，川普与夫人在老布什的灵柩前哀悼，并以军礼致敬。周二再度访问布什家族表达哀悼。
 <br/>
 <br/>
 周三的国葬仪式为华府的系列悼念画上句号。老布什的灵柩将运往休斯顿，周四再次举行悼念活动后，在德州农工大学校园内的布什总统图书馆下葬。
 <br/>
 <br/>
 新唐人记者林宇、宇微综合报导
