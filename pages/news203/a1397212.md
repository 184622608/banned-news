### 媒体狂轰一周失败 川普支持率超奥巴马同期6%
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】上周，美国主流
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_媒体.html" target="_blank">
  媒体
 </a>
 借〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸弹邮包.html" target="_blank">
  炸弹邮包
 </a>
 〞事件狂攻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统，但周一（10月29日）发布的最新民调显示，川普的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支持率.html" target="_blank">
  支持率
 </a>
 已再次击破50%的高点，高出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 同期6%。
 <br/>
 <br/>
 周一，美国民调公司〝拉斯穆森报告〞（Rasmussen Reports）的每日跟踪调查结果显示，50%的可能选民支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的工作表现，不支持的人只占49%。
 <br/>
 <br/>
 相比之下，前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 （Barack Obama）在2010年10月29日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支持率.html" target="_blank">
  支持率
 </a>
 只有44%，低于川普同期支持率6个百分点。
 <br/>
 <br/>
 值得一提的是，川普高达50%的支持率是在美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_媒体.html" target="_blank">
  媒体
 </a>
 的又一轮狂轰乱炸之下获得的。上周，由于十几名民主党高级政客和显要人士受到〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸弹邮包.html" target="_blank">
  炸弹邮包
 </a>
 〞威胁，美媒把事件责任转嫁给川普，指责这是由于他〝攻击媒体〞、言论上〝纵容暴力行为〞所致——虽然他们自己的报导一再被川普批评为〝假新闻〞。
 <br/>
 <br/>
 据investors.com网站10月10日报导，媒体研究中心（Media Research Center）最近公布的一项调查结果显示，92%的电视新闻对川普的报导都是负面的，而媒体对川普在经济上取得的巨大成就几乎根本就不关注。
 <br/>
 <br/>
 川普上任后，其支持率一路走高，并保持在相对稳定的状态。
 <br/>
 <br/>
 拉斯穆森民调4月2日发布的调查结果显示，当时川普的支持率即已达到50%，而奥巴马在其第一届任期内的同期支持率仅为46%。
 <br/>
 <br/>
 半年多之后，该民调显示，尽管遭到主流媒体狂攻，川普的支持率仍高居50%不下，而一直受到自由派媒体支持的奥巴马，其同期支持率却下降了2个百分点。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
</div>
