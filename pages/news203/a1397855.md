### 美10月新增25万工作 时薪涨幅刷新纪录
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】美国劳工部周五发布最新就业报告，10月份新增就业高于预期，工资增长率也达到2009年以来最大涨幅。
 <br/>
 <br/>
 据美国劳工部的报告，全美非农就业人数十月份增加25万人，高于预估的19万人。当月失业率为3.7%，是49年来最低点。
 <br/>
 <br/>
 十月份平均小时工资增加5美分，涨幅0.2%。年增率由九月份的2.8%上升到3.1%。这是2009年4月以来最大涨幅。
 <br/>
 <br/>

 </a>
 经济顾问委员会主席哈塞特)表示，川普总统的政策让大批失业者重回就业市场。

 <br/>
 <br/>
