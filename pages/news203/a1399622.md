### 佛州启动人工重新计票 民主党恐难挽败局
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】周四（11月15日）晚，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 根据该州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议员.html" target="_blank">
  参议员
 </a>
 竞选的机器重新计票结果，宣布从当晚开始启动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人工重新计票.html" target="_blank">
  人工重新计票
 </a>
 。尽管民主党人利用多种手段试图扩大自己的优势，但从目前的计票结果看，其挽回败局的希望已非常渺茫。
 <br/>
 <br/>
 周四晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 州长斯科特（Rick Scott）下令，从当晚7点开始，启动全州范围的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人工重新计票.html" target="_blank">
  人工重新计票
 </a>
 。
 <br/>
 <br/>
 由于竞选双方的选票差距不足0.5%，佛州从上周六开始启动了3项选举的机器重新计票机制。至本周四，计票结果显示，共和党在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议员.html" target="_blank">
  参议员
 </a>
 和州长两项重要竞选中仍处于领先地位。其中，共和党州长候选人德桑蒂斯（Ron Desantis）所获得的选票高出其民主党对手吉卢姆（Andrew Gillum）33,683张，估计其最终胜选已无悬念。
 <br/>
 <br/>
 不过，鉴于现任州长、共和党参议员候选人斯科特（Rick Scott）高出其对手、现任民主党参议员尼尔森（Bill Nelson）的选票数只有12,603张，所以需要启动人工计票。
 <br/>
 <br/>
 佛州规定，如果获胜候选人所赢得的选票高出对手不足0.5%时，将启动强制性重新计票。如果相差幅度低于0.25%，重新计票工作必须手工完成。
 <br/>
 <br/>
 根据选举夜的非正式计票结果，斯科特领先尼尔森50,000多张选票。但是在接下来的几天中，倾向于民主党且迟迟不完成计票工作的布劳沃德县（Broward）和棕榈滩县（Palm Beach）突然多出数万张选票并多数流向民主党候选人，导致两位共和党人的优势被巨幅削弱，至上周六下午，斯科特的领先幅度已被压缩为0.15%，德桑蒂斯的优势也降低到0.41%。
 <br/>
 <br/>
 不过，机器重新计票的结果，斯科特的优势有所扩大，增加了865票。周四晚，斯科特再次宣布自己获胜。但是，由于尼尔森仍不承认失败，所以佛州开启了第二次重新计票——人工计票。
 <br/>
 <br/>
 选举的最终结果将在11月20日州选举计票委员会开会后裁定，州务卿戴兹特（Ken Detzner）将出具选举结果的书面证明。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
