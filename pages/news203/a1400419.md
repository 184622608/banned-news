  边境墙
 </a>

 <br/>
 建边境墙是川普2016年 大选的首要承诺之一，但民主党对此激烈反对。
 <br/>
 <br/>
 感恩节（周四）上午，川普就大篷车移民发表评论时告诉记者，如果南部边境的中美洲移民陷于失控状态，美国将把美墨两国之间的全部入境口岸关闭。
 <br/>
 <br/>
 川普同时警告，这意味着在边境重新开放前，墨西哥将无法再把汽车卖入美国，墨西哥的经济利益势必受到严重伤害。
 <br/>
 <br/>
 感恩节上午，在佛州海湖庄园（Mar-a-Lago）度假的川普还表示，如果
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国会.html" target="_blank">
  国会
 </a>
 不批准筑墙资金，他可能在12月份关闭政府。
 <br/>
 <br/>
 川普告诉记者，边境墙只是边境安全的一部分，不过是很重要的一部分，而且可能是最重要的一部分。
 <br/>
 <br/>
 建边境墙本来是川普打算上任后首先解决的问题，但是，众议院议长瑞恩（Paul Ryan）始终没有促成国会解决筑墙资金问题。
 <br/>
 <br/>
 今年3月，参众两院4名两党领袖达成妥协并秘密起草了一份综合预算草案，且直到政府面临关门的前2天才出示。由于时间有限，国会匆匆表决，川普也无足够时间仔细审阅。川普为保障国防安全，虽一度犹豫，但最终还是勉强签署了法案。
 <br/>
 <br/>
 川普签署法案后，罕见受到其支持者激烈反对，有些批评甚至极为犀利。为兑现自己的竞选诺言，川普在被迫答应据称只能修修〝边境篱笆〞的区区16亿美元拨款后，紧急采取补救措施，声明将利用巨额军费中的一小部分去贴补边境墙建设。
 <br/>
 <br/>
 （记者李佳欣综合报导）
