 经过机器和人工的两轮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 佛州联邦参议员候选人以超过10,000票的领先优势，于周日下午正式宣布获胜。
 <br/>
 <br/>
 他击败的是佛州现任联邦参议员、民主党人尼尔森，这也意味着，现年76岁的尼尔森将可能结束近40年的政治生涯。
 <br/>
 <br/>
 川普在推文中祝贺斯科特说：斯科特是一名了不起的州长，也将成为一名代表佛州人民的更好的参议员。
 <br/>
 <br/>
 佛州于11月6日中期选举后开始重新计票，第一轮机器计票在11月15日下午截止后，斯科特领先尼尔森的得票率仍在0.25%以内，因此佛州再启动人工计票，并于周日中午截止。
 <br/>
 <br/>
 过程中，尼尔森和民主党多次就重新计票中的情况提起诉讼，但大多数被联邦法官否决。
 <br/>
 <br/>
 此外，佛州另一场进入重新计票的州长选举，也于周六得出结果。共和党佛州州长候选人德桑蒂斯（Ron DeSantis）击败民主党人吉卢姆（Andrew Gillum）取胜。
