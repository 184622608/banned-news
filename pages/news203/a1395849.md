### 身负重伤救数十位战友  80岁越战老兵获川普授勋
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】前美国海军陆战队枪炮军士长约翰・坎利在越战期间抵抗越共军队，不顾自己身受重伤，救出数十位战友。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）周三（10月17日）下午在白宫授予他美国军方最高荣誉——国会勋章。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统美东时间周三在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_授勋.html" target="_blank">
  授勋
 </a>
 仪式上说：〝50年前，在越战战争期间最为漫长与血腥的一场战役——顺化战役中，一名美国海军陆战队员以无与伦比的勇气参加了作战，这位英勇的海军陆战队员就是军士长约翰・坎利（JOHN CANLY）。〞
 <br/>
 <br/>
 川普说，〝约翰参加了连续7天的持续作战，挽救了20多名海军陆战队员的生命。在战斗结束时，美国海军陆战队击败了共产党（军人），夺回了这座城市。〞
 <br/>
 <br/>
 川普接着说，约翰的战友们把他称为〝海军陆战队勇士〞，他的战友们说，他比生命还高大，超越了死亡。
 <br/>
 <br/>
 川普说，正因为有像约翰这样的人，美国才可成为捍卫和平、正义和自由的最伟大的力量。
 <br/>
 <br/>
 今年80岁的坎利退役前在美国海军陆战队服役28年，曾被派往韩国和日本。在越战期间参加作战5年。
 <br/>
 <br/>
 1968年1月30日，在越南当人们正在庆祝黄历新年时，数千名北越共产党军人在全国各地发动突袭，并占领了美军的重要据点顺化市。当时约翰・坎利所在的海军陆战队第一团负责夺回这座城市，150名海军陆战队员参加作战，对抗6000多名越共军。
 <br/>
 <br/>
 越共军用机关枪、迫击炮和火箭弹向美国海军陆战队员发动猛烈攻击，坎利的战友帕特・弗拉利被一枚火箭弹击中，随后一辆坦克向他开去，坎利冲向敌军，救回战友。
 <br/>
 <br/>
 在顺化战役初期，美国海军陆战队指挥官受了重伤，约翰・坎利接替指挥官，带领美军士兵完成了战斗。当天傍晚，他们攻入了被6000多名越共士兵占领的这座城市。
 <br/>
 <br/>
 在接下来的巷战中，坎利带领海军陆战队袭击敌军据点，他多次不顾个人安危，在自己严重受伤的情况下，冒着枪林弹雨救出战友。
 <br/>
 <br/>
 当年一起英勇作战的30多名美国海军陆战队员周三也来到白宫，参加授勋仪式。弗拉利也是其中的一位，他们一起感谢这位救命英雄。
