### 美墨边境一卡车逃避追捕 翻车后3人死
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】周四，一辆载有11人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡车.html" target="_blank">
  卡车
 </a>
 以超过100英里的时速，试图逃离美国加州边境人员的追逐，途中失控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻车.html" target="_blank">
  翻车
 </a>
 ，导致3死7伤。
 <br/>
 <br/>
 加州高速公路巡警表示，事故发生于周四下午约4点半，在圣地牙哥附近的坎波镇（Campo），紧邻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 。
 <br/>
 <br/>
 据目击者称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡车.html" target="_blank">
  卡车
 </a>
 当时以超过一百英里的时速逃避边境巡逻人员的追捕。巡警在路面设置了〝钉子带〞，以阻止卡车行进，而高速行驶的卡车当场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻车.html" target="_blank">
  翻车
 </a>
 ，车上11人中，3人死亡，7人受伤，司机被逮捕。
 <br/>
 <br/>
 目前，还不清楚追捕这辆卡车的原因。但政策要求，只有必要性大于危险性时，才会发生这种追捕的情况。
 <br/>
 <br/>
 今年6月，在德州边境也发生过类似事件，车上绝大多数是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 。当时有舆论对事故表示遗憾，也有人认为应当更有效地阻止非法入境，才能避免此类悲剧。
 <br/>
 <br/>
 同一天，边境另一侧的墨西哥城市蒂华纳开辟了第二个避难所，以容纳聚集在当地的众多大篷车移民。而他们的滞留引发不少当地人的抗议。这些大篷车移民目前还在想办法进入美国。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
