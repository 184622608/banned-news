### 赶赴生日派对礼车发生车祸 20人遇难
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 州警方周日表示，周六晚间，一辆加长豪华
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_礼车.html" target="_blank">
  礼车
 </a>
 和另一辆汽车在纽约上州相撞，造成20人死亡，是纽约州九年来最严重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_车祸.html" target="_blank">
  车祸
 </a>
 。
 <br/>
 <br/>
 事故发生在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 州30号公路，靠近纽约上州奥本尼以西38英里的舒哈里小镇。
 <br/>
 <br/>
 警方表示，出事的是一辆豪华加长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_礼车.html" target="_blank">
  礼车
 </a>
 ，从高速公路冲向路边一个商家，撞到停车场的一辆车，两名行人被撞身亡。
 <br/>
 <br/>
 纽约州警副督察 Chris Fiore：〝总共20人遇难，都是成年人。礼车内18人，包括司机全部遇难，两名行人被撞身亡。〞
 <br/>
 <br/>
 目击者称，事发时听到巨响，树都在晃。
 <br/>
 <br/>
 目击者 Linda Riley:〝我们听到爆炸声，现在我们知道是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_车祸.html" target="_blank">
  车祸
 </a>
 ，这些树都在晃，就听见‘嗖’的一声，我们用最快速度从车里跑出来，还以为是树倒了，有爆炸，不知道是怎么回事。〞
 <br/>
 <br/>
 事发后，至少6辆救护车和两架直升机赶到现场救援。遇难者中有四个姐妹，其中三个与丈夫一起，为了给最小的妹妹庆祝生日，租用礼车赶赴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生日派对.html" target="_blank">
  生日派对
 </a>
 。
 <br/>
 <br/>
 姐妹们的姨母在接受采访时声音哽咽。
 <br/>
 <br/>
 遇难者亲人 Barbara Douglas：〝她们（四姐妹）很有爱心，都是好孩子。她们可以为你做任何事。她们之间非常亲密，爱她们的家人和父母。〞
 <br/>
 <br/>
 她认为姊妹们已经很负责任，租用礼车，不自己开车。没想到悲剧还是发生了。
 <br/>
 <br/>
 警方表示，正在调查这起当地九年来最严重的车祸，礼车公司、司机、车辆情况、以及天气路况等，都在调查范围之内。
 <br/>
 <br/>
 新唐人记者安心、宇微综合报导
</div>
