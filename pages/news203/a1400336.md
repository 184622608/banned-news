### 美空军B-52轰炸机破坏力惊人 曾是毛泽东外号
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国空军.html" target="_blank">
  美国空军
 </a>
 两架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_B-52轰炸机.html" target="_blank">
  B-52轰炸机
 </a>
 越过中国，在南中国进行常规
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_训练.html" target="_blank">
  训练
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞行.html" target="_blank">
  飞行
 </a>
 引起关注。B-52战略轰炸机所到之处被视为美国施加政治影响的地方。据报，B-52曾是毛泽东外号，意指毛象B-52一样，有巨大的破坏威力。
 <br/>
 <br/>
 美国之音21日报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国空军.html" target="_blank">
  美国空军
 </a>
 两架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_B-52轰炸机.html" target="_blank">
  B-52轰炸机
 </a>
 19日越过中国，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南中国海.html" target="_blank">
  南中国海
 </a>
 宣称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_主权.html" target="_blank">
  主权
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_领空.html" target="_blank">
  领空
 </a>
 进行常规
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_训练.html" target="_blank">
  训练
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞行.html" target="_blank">
  飞行
 </a>
 ，展示美国在这片水域，继续
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自由航行.html" target="_blank">
  自由航行
 </a>
 和飞行的决心。
 <br/>
 <br/>
 美国太平洋空军部队20日表示，两架美国B-52H轰炸机，从关岛的安德森空军基地起飞，参加南中国海附近的例行训练任务。空军发布的声明称，这一任务符合国际法和美国对自由、开放的印度太平洋的长期承诺。
 <br/>
 <br/>
 同一天，美国海军还宣布，里根号航母和斯坦尼斯号航母以及其战斗群，在菲律宾海域共同执行任务，此举也同样彰显美国对海上自由的承诺。
 <br/>
 <br/>
 此前，中共称为了给11月底的〝川习会〞铺路，批准了刚刚在菲律宾海结束演习的美国4艘军舰21日停靠香港。
 <br/>
 <br/>
 美国里根号航母和斯坦尼斯号航母和B-52H轰炸机等，均是美国全球军事打击能力的象征，它们所到之处也是美国施加政治影响的地方。
 <br/>
 <br/>
 英媒BBC报导说，B-52在中国也是大名鼎鼎，中共建政后的第一代〝最高统帅〞林彪垮台后揭发材料显示，林彪之子林立果和他所在空军的小圈子，私下给中共第一代党魁毛泽东取外号为B-52。
 <br/>
 <br/>
 据猜测把毛泽东比作B-52，意指毛有巨大的破坏威力，而且象B-52一样高高在上。
 <br/>
 <br/>
 据国家利益2018年初报导，目前军中服役的B-52战略轰炸机型号仅为H型，又名同温层壁垒，因为武器协约而只能携带常规武器或带核弹头的空基巡航导弹（ALCM）。
 <br/>
 <br/>
 B-52性能非常优良，在本世纪20年代新型战略轰炸机B-21突袭者正式服役后，B-52仍可以继续使用；而且在稍作升级后，其服役期可延至40年代，服役年限将达90年。
 <br/>
 <br/>
 美军目前装备三大战略轰炸机，除了B-52之外，其余两个分别为B-1枪骑兵（B-1 Lancer） 超音速可变后掠翼重型远程战略轰炸机和B-2幽灵隐形轰炸机（B-2 Spirit）。
 <br/>
 <br/>
 B-52是其中研发和服役时间最早的轰炸机，其载重量也最大。B-52为波音设计生产的8部发动机、亚超音速、远程喷气式重型轰炸机，共生产744架。其中的D型、G型及H型性能最优。
 <br/>
 <br/>
 B-52时速可超过1000公里，最大高度达到1.5万米。1952年B-52首次飞行，其设计目的为抗击共产阵营头目苏联。
 <br/>
 <br/>
 1962年12月10日，B-52H在未重新加油的情况下，从日本冲绳嘉手纳空军基地飞抵西班牙托雷洪，飞行距离达12,532英里。空军对B-52的航程很满意，因为如果再次加油，可以覆盖欧亚大陆的任何地点。
 <br/>
 <br/>
 B-52于1955年正式服役后，在20世纪参加越南战争，及在海湾战争中发挥主力作用。21世纪，B-52参与打击中东的伊斯兰国，以及阿富汗战争。可谓身经百战，常胜不败。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
