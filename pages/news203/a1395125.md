### 方济各接受华盛顿总主教乌尔辞职
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】周五，罗马天主教教宗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_方济各.html" target="_blank">
  方济各
 </a>
 ，接受美国华盛顿-总主教唐纳德．乌尔的辞职。乌尔成为全球天主教会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵丑闻.html" target="_blank">
  性侵丑闻
 </a>
 中，宣布辞职的最高级别神职人员。
 <br/>
 <br/>
 乌尔现年77岁，1988年到2006年间，担任美国宾州匹兹堡教区的主教。乌尔被指这期间，掩盖包庇了所辖教区主教的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵丑闻.html" target="_blank">
  性侵丑闻
 </a>
 。
 <br/>
 <br/>
 今年八月，宾州大陪审团报告揭露，过去70多年来，300多名宾州神父涉嫌对1000多人性侵，其中多数是儿童。涉案神职人员来自宾州六个教区，其中包括乌尔负责的匹兹堡教区。
 <br/>
 <br/>
 其中，前美国华盛顿教区红衣主教西奥多．麦卡里克涉嫌性侵案，最受关注。麦卡里克被控在过去数十年间，涉嫌性侵儿童和成人。
 <br/>
 <br/>
 今年7月，教宗方各济接受了麦卡里克的辞职。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
