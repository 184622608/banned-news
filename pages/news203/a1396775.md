### 大篷车难民继续北上 部分儿童出现病症
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】周四，中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 已抵达离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 南部边境约160公里处的小镇。白宫顾问康威表示，川普总统和彭斯副总统一直关注大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 问题，希望了解难民中到底都有什么人。
 <br/>
 <br/>
 周四，越过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 边境的洪都拉斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 ，继续向美国南部边境前进。部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 周三晚间，已抵达墨西哥南部小镇马帕斯特佩克。
 <br/>
 <br/>
 墨西哥官方统计，这批难民有3600多人。
 <br/>
 <br/>
 当地政府开放了六个庇护所，提供食物和水，以及医疗服务。
 <br/>
 <br/>
 马帕斯特佩克市政主席 Carla Gamboa:〝我们从没见过一下来这么多移民。这里总有不同国家的移民路过，因为有移民中转站。但是，从没见过这么多人。我想这对我们行政管理是个挑战。〞
 <br/>
 <br/>
 连日旅途奔波，很多难民疲惫不堪，年幼儿童患上流感。
 <br/>
 <br/>
 马帕斯特佩克社会事务办公室主任 Paulina Gonzalez:〝我们看到许多儿童发高烧、患上流感、咳嗽、脱水。〞
 <br/>
 <br/>
 周三，白宫顾问康威((Kellyanne Conway))表示，川普总统和彭斯副总统一直关注大篷车难民问题，希望了解难民中到底有什么人。
 <br/>
 <br/>
 白宫顾问 康威：〝总统每天都听简报，副总统也一样，他们和国土安全部等部门保持紧密联系，以便确切了解情况，我想也许你们不感兴趣，到底大篷车难民中有谁？谁要来这里？我们有权知道这些。〞
 <br/>
 <br/>
 周四，川普推文再次批评民主党的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民政策.html" target="_blank">
  移民政策
 </a>
 ，让美国很难阻挡移民非法进入边境。
 <br/>
 <br/>
 推文说：〝国家边境巡逻委员会的布兰登・贾德，在接受福克斯新闻采访中说的很对。民主党的(移民)法律让我们很难阻挡人们(非法)进入边境。必须改变。我正在调动军队，应对国家紧急状态。必须阻止他们！〞
 <br/>
 <br/>
 新唐人记者安心、宇微综合报导
 <br/>
 <br/>
