### 大篷车移民压境 川普政府：非法越境者不得申请庇护
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】面对大批中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 压境，川普（特朗普）政府周四（8日）颁布最新难民庇护申请程序，规定非法越境的移民不得在美国寻求庇护。但此规定不具有追溯力，也就是说不适用于已经入境的非法移民。
 <br/>
 <br/>
 美国司法部代理部长惠特克（Matthew Whitaker）和国土安全部部长尼尔森（Kirstjen Nielsen）周四（11月8日）共同宣布这项暂行最终规定（Interim Final Rule），所有违反总统暂停或限制公告、通过美国与墨西哥南部边境（非法）进入美国的外国人，基于美国《移民和国籍法》（INA）第212(f)条或第215(a)(1)条，不符合申请庇护的资格。
 <br/>
 <br/>
 联合声明中称：〝根据美国移民法，在总统认为符合国家利益的情况下，总统有权暂停或限制外国人进入美国。今天颁布的这一重要规定将适用于违反总统暂停或限制令、通过美国南方边境进入的外国人，国会明确授权给总统决定限制庇护资格的权力。〞
 <br/>
 <br/>
 声明称，因太多外国人来美申请庇护，美国的庇护制度已不堪重负，这些申请让国家资源背负沉重负担，并使美国政府无法迅速处理那些真正应获得庇护资格人士的申请。
 <br/>
 <br/>
 声明写道，〝今天，我们用国会授予我们的权力，禁止违反总统暂停入境或其他限制令的外国人寻求政治庇护。〞
 <br/>
 <br/>
 同时，声明还援引美国《移民和国籍法》（INA）的相关规定：
 <br/>
 <br/>
 《移民和国籍法》第212(f)条规定，如果总统发现任何外国人或任何类别的外国人进入美国将不利于美国的利益，他可以在他认为必要的期间，宣布暂停所有外国人或任何类别的外国人作为移民或非移民入境，或对外国人入境施加他认为适当的任何限制。
 <br/>
 <br/>
 《移民和国籍法》第215(a)条规定，除非依据合理的规则、条例、命令以及总统授权的限制和例外情况下，任何外国人离开或进入、试图离开或进入美国都属于〝非法〞。
 <br/>
 <br/>
 《移民和国籍法》第208(d)(5)(B)条规定，国会授权司法部长〝可以通过法规规定对庇护申请施加其它条件或限制。〞
 <br/>
 <br/>
 司法部和国土安全部的联合声明中称，该新规适用于总统公告，不具有追溯力，也就是说不适用于已经入境的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 。
 <br/>
 <br/>
 外界认为，美国政府的新规主要是针对来自洪都拉斯等中美洲国家的万人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 队，在中期选举前他们以最快的速度北上，向美国边境挺进，人数从高峰期的7千人降至4千人。这已是今年的第二波难民潮，第一波发生在今年春季。
 <br/>
 <br/>
 庇护是美国政府基于对外国人因种族、宗教、国籍、特定社会群体或政治见解，逃避本国迫害而酌情提供的救济形式。因申请人数众多、案件积压严重，截至2018年1月21日，美国移民局已经积压了31.1万起庇护案件，移民局局长李・西斯纳（Lee Francis Cissna）称到了〝灾难性〞的积压水平。
 <br/>
 <br/>
 从1月底开始，美国移民局对政治庇护申请恢复过去的〝后进先出〞原则，优先集中处理21天内提交的政治庇护申请。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
