### 转机隐现？白宫经济顾问:新关税只是重要假设
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】白宫经济顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 日前谈及中美贸易冲突的问题时表示，没什么是一成不变的，如果中美双方能够进行有希望的政策讨论，华府可能会撤回部分关税。与此同时，有消息指中国国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 将出席11月初在新加坡举行的首届中美创新经济论坛，外界猜测，在中国经济下滑压力山大之际，王或将走上前台为解决中美贸易冲突寻求转机。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
   库德洛
  </a>
  谈贸易战：没什么是一成不变的
 </h4>
 <br/>
 美国国家经济委员会主任库德洛（Larry Kudlow）周三对外表示，对于华府会否继续对中国商品加征新关税的问题，川普（特朗普）总统还没有定案。如果美中双方能进行〝有希望的政策讨论〞并成某种友好协议，那么华府也有可能会撤回部分关税。
 <br/>
 <br/>
 他接着解释称，进一步向中国商品征税并不是美方的一个〝承诺〞，而只是一个非常〝重要的假设〞。
 <br/>
 <br/>
 他强调说：〝这将视乎政策谈判的结果，而不是任意的时间表。〞
 <br/>
 <br/>
 今年11月30日，美国总统川普将与中国国家主席习近平在阿根廷首都布宜诺斯艾利斯（Buenos Aires）出席20国集团峰会〝G20〞期间会面，外界普遍预估他们二人将就中美贸易冲突等重要议题进行面对面商讨。
 <br/>
 <br/>
 因此，有媒体提问：如果这一次的川习会谈没能缓解两国的贸易紧张局势，川普会不会如期推行新一轮加征关税的政策？库德洛则回应称：〝我会说没什么是一成不变的。〞
 <br/>
 <br/>
 此前，川普曾于10月29日在福士新闻频道的The Ingraham Angle节目上接受专访时，就中美贸易战的动向透出口风。他说：〝我认为我们将和中国达成绝佳协议，必须要是绝佳的协议，因为他们已经掏空了我们的国家。〞
 <br/>
 <br/>
 同一天，彭博新闻援引未具名的消息人士表示，如果川普和习近平11月的会谈不能缓解贸易战局势，美国政府准备于12月初宣布对所有其余进口自中国的商品征收关税。
 <br/>
 <br/>
 <h4>
  中国经济数据严峻 北京口风放软
 </h4>
 <br/>
 当地时间10月31日，习近平在北京主持召开中共中央政治局会议时称，中国的经济形势正在发生变化，中国国内企业发展遇到困难，经济下行压力有所加大，政府需要及时采取对策来应对这种情况。
 <br/>
 <br/>
 海外中文媒体随即在相关报导中分析称，这是自中美贸易战爆发以来，中共领导人首次承认了对当前国内经济下行趋势的担忧，中国经济的实际状况可能比官方所承认的状况更为严峻。
 <br/>
 <br/>
 《德国之声》当时也援引上海华东理工大学经济系宏观经济教研室主任沈凌分析称，中国领导层在此时此刻改变口风，触发点是中国今年第三季度的经济数据不佳。
 <br/>
 <br/>
 沈凌说：〝经济学界大概半年前已经觉得，中国经济并不如官方所说般乐观。这是第一次民间和官方判断一致，所以我们并不感到意外。〞
 <br/>
 <br/>
 他进一步表示，明年中国的GDP增长率还会继续下滑，唯一转机恐怕只有川普与习近平11月底的这次会谈。他说：〝明年走势完全取决于年底的中美谈判，中国经济短期内的最大波动因素就是美中贸易问题，如果中美达成妥协，明年走势会好一点，不然非常不乐观。〞
 <br/>
 <br/>
 <h4>
  为中美贸易冲突寻求专机？
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
   王岐山
  </a>
  将出席〝
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美新经济论坛.html" target="_blank">
   中美新经济论坛
  </a>
  〞
 </h4>
 <br/>
 与此同时，外界注意到，中国国家副主席王岐山下周将访问新加坡，出席于11月6日至7日举行的首届〝中美创新经济论坛〞，并将首次就中美关系在国际平台发表演讲，预计将谈及中美贸易战的话题，向外界表达中国将继续改革开放的意愿。
 <br/>
 <br/>
 据了解，〝中美创新经济论坛〞是由前纽约前市长布隆伯格（Michael Bloomberg）创立的一个经济论坛，基本宗旨是促进中美关系的发展，而中国智库〝中国国际经济交流中心〞也是这个论坛的合办方。
 <br/>
 <br/>
 有消息指，布隆伯格曾明确提出，希望中方能够派出高官来出席这个论坛，以此凸显这一论坛开幕的历史性和现实意义。至于美国方面，则会有美国总统川普的前战略顾问班农（Steve Bannon）以及美国商界代表出席这次论坛，届时王岐山将与他们举行会谈。
 <br/>
 <br/>
 鉴于上述情况，有分析认为，在经历了几个月的贸易冲突与你来我往的博弈较量后，中方显然已经底气不足，几乎已没有回击的力量。在这样的背景下，此前一直没有出面介入中美贸易冲突的王岐山开始走向前台，显示北京高层或已准备对美方作出某种妥协，所以在川习11月底会谈前，先让王岐山出面来铺路，为解决中美贸易冲突寻求新的转机。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
