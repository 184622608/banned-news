### 数千法国民众抗议燃油税上涨 爆发冲突要总统下台
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】周六（11月24日），数千名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 民众上街
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃油税.html" target="_blank">
  燃油税
 </a>
 上涨，并与警方发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冲突.html" target="_blank">
  冲突
 </a>
 且要求总统马克龙（Emmanuel Macron）下台。
 <br/>
 <br/>
 周六，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 首都巴黎及许多其它城市，数千名身穿黄马甲的法国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 者连续第8天走上街头，抗议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃油税.html" target="_blank">
  燃油税
 </a>
 上涨。
 <br/>
 <br/>
 抗议者们的愤怒愈加强烈，诉求已从要求降低燃油税扩大到呼吁总统马克龙辞职，并声称法国统治阶层已与民众脱节。
 <br/>
 <br/>
 黄马甲已成为抗议者的一种标志，起源于法国司机开车时按要求必备的黄色莹亮背心。
 <br/>
 <br/>
 法国当局称，仅涌到巴黎香榭丽舍大街（Champs-Elysees）抗议的民众就高达至少8,000人， 遍及全国的抗议者总计达到了81,000人。而上周六，抗议者一度达到244,000人，警方则出动了3,000 安全部队设防。
 <br/>
 <br/>
 据法国《世界报》（Le Monde）报导，本周六，警方使用了烟雾、催泪瓦斯、甚至水枪等，试图驱散抗议人群。
 <br/>
 <br/>
 在抗议者与警方的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冲突.html" target="_blank">
  冲突
 </a>
 过程中，至少有8人受伤，其中包括2名警察。另外还有几十名抗议者被拘留。
 <br/>
 <br/>
 在满是受游客欢迎的奢侈品的商业区La Madeleine，店铺因抗议活动而被迫关闭。
 <br/>
 <br/>
 随着警方使用催泪瓦斯，抗议者疾呼〝马克龙辞职〞。抗议区飘满黑烟并被完全封锁。
 <br/>
 <br/>
 法国交通部长鲍恩（Elisabeth Borne）称，目前法国的柴油税已上涨了7欧分（差不多8美分），并且将在未来几年中持续上涨。汽油税将上涨4欧分。
 <br/>
 <br/>
 但马克龙表示，对于减少依赖化石燃料和资助可再生能源投资来说，增加燃油税是必须的。
 <br/>
 <br/>
 （记者李佳欣综合报导）
 <br/>
</div>
