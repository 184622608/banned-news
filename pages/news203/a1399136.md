### 加州最猛野火酿44死 好莱坞大砍星光熠熠首映
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州野火.html" target="_blank">
  加州野火
 </a>
 持续延烧，目前总计至少44死，仍有200多人下落不明。死亡人数可能还会进一步提升。另外还有数千栋民宅焚毁，其中包括不少名人宅邸也付之一炬，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_好莱坞.html" target="_blank">
  好莱坞
 </a>
 周一（12日）大砍星光熠熠的首映活动。
 <br/>
 <br/>
 路透社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国电影学会.html" target="_blank">
  美国电影学会
 </a>
 （AFI）电影节发言人当地时间周一表示，女星珊卓布拉克（Sandra Bullock）新片〝蒙上你的眼〞（Bird Box，暂译）和男星安德鲁加菲（Andrew Garfield）新片〝解码游戏〞（Under the Silver Lake）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红毯.html" target="_blank">
  红毯
 </a>
 活动取消。
 <br/>
 <br/>
 AFI电影节也在11日已取消了柯恩兄弟（Joel and Ethan Coen）执导的〝西部老巴的故事〞（The Ballad of Buster Scruggs）红毯活动。
 <br/>
 <br/>
 此外，派拉蒙影业（Paramount）也已取消由萝丝拜恩（Rose Byrne）和马克华柏格（Mark Wahlberg）主演的〝速成家庭〞（Instant Family）的红毯与派对活动。
 <br/>
 <br/>
 坎普野火（Camp Fire）8日爆发以来，已将7100多栋民宅和其他建筑物夷为平地，最新公布的死亡人数统计增至42人，成为加州史上最致命野火。加上南加州的2人在大火中丧生，目前加州总计至少44死，仍有200多人下落不明。
 <br/>
 <br/>
 南部的〝伍尔西大火〞（Woolsey Fire）在8日晚间开始燃烧，9日朝滨海富裕城市马里布和太平洋推进。
 <br/>
 <br/>
 因马里布住着多个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_好莱坞.html" target="_blank">
  好莱坞
 </a>
 名人，大火肆虐也导致他们的房屋受损，主演《斯巴达300勇士》的杰拉德．巴特勒（Gerard Butler）得知自己的家付之一炬后，与烧到只剩断垣残壁的房子合照，并在推文中说，在撤离后回到马里布的家…野火带来的灾难令人心碎。
 <br/>

 以校园青春影集〝飞越比佛利〞（Beverly Hills,90210）走红的美国女星香侬道荷堤（Shannen Doherty）表示，她心如刀割，因为她失去了2011年与丈夫结婚时的房子。
 <br/>
 <br/>
 卡戴珊家族的金卡戴珊（Kim Kardashian）、实境秀节目〝比佛利娇妻〞明星凯尔理查兹等人也都纷纷撤离。歌星雪儿发推文表示，她很担心房子不保，但也知道自己爱莫能助，她自1972年起就一直住在马里布。
 <br/>
 <br/>

 <br/>
 <br/>
 不过，也有部分名人于11日回家查看灾情后表示，住宅没有受殃及。其中包括知名歌手Lady Gaga的豪宅就丝毫无损。
