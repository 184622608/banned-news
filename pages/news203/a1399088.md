### 加州野火烧太快 消防员先救人要紧
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州野火.html" target="_blank">
  加州野火
 </a>
 已造成至少31人死亡，其中29人在北加州丧生。而南加州圣安娜周日再度刮起强风，助长了〝伍尔西野火〞的火势，造成至少2人丧生，超过25万人疏散。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州山火.html" target="_blank">
  加州山火
 </a>
 因火势太快太猛，消防员只能先救人，眼看小镇被吞噬。
 <br/>
 <br/>
 南加州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伍尔西山火.html" target="_blank">
  伍尔西山火
 </a>
 （Woolsey Fire）8日起自南加州西米谷(Simi Valley)的伍尔西峡谷窜烧，到11日，山火已经蔓延8万3000英亩，仅10%火势受到控制。消防人员趁10日风势减弱挖掘防火壕并喷洒大量阻燃剂。
 <br/>
 <br/>
 美联社报导，北加州的坎普山火（Camp Fire）截至11日仅有25%火势得到控制，这场大火在24小时内烧毁6453栋房屋和260间商务建筑，3天来已知至少有29人丧生、228人下落不明。失踪者可能是因仓皇逃难而失去音讯，但也可能不幸罹难。
 <br/>
 <br/>
 坎普山火已追平1933年洛杉矶郡葛瑞菲斯公园(The Griffith Park)大火造成29人死亡的纪录。因死亡人数可能还会增加，坎普山可能成为加州史上死亡人数最多的林火。
 <br/>
 <br/>
 据悉，因野火延烧太快太猛，消防员在最初24小时内只能救人，根本来不及灭火。人们眼睁睁地看着重灾区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天堂镇.html" target="_blank">
  天堂镇
 </a>
 （Paradise）被火舌吞没，这与消防员平时的日常工作不同。
 <br/>
 <br/>
 在内华达市加州森林防火厅服役18年的消防员薛利(Thor Shirley)称，他从来没有见过这么惨重的山火。他说，〝你只能保护性命，这很挫折，因为你想同时拯救财产与性命，但是你只能根据情况调整，救人要紧。〞
 <br/>
 <br/>
 他的小队来回奔波，总共救了14人，包括卧病在床的病患、医护人员和警察。
 <br/>
 <br/>
 消防员佩克(Casey Peck)的消防车则困在数十辆汽车与人群间不能前进，车上的漆和水管都被熔化了，甚至高温还摧毁了可以承受900℉高温的释压阀，旁边的车辆无一幸免。
 <br/>
 <br/>
 4名从着火车辆逃出的人拍打消防车车门，佩克立即将他们拉进车内，其中一名护士的裤管还着着火。车内消防员将防火毯压在车窗上抵御高温，等待大火平息。
 <br/>
 <br/>
 佩克10日谈到当时的情景时说，〝信仰，我想我此生从未如此奋力祷告过。〞
 <br/>
 <br/>
 滨海小镇天堂镇有2万7000名居民，而且当地众星聚居。如今却成为遍地灰烬的焦黑炼狱。
 <br/>
 <br/>
 消防队队长塞瑟(Alejandro Saise)领着队员视察天堂镇灾区时表示，〝我们走过原有20栋房子的地区，但现在只剩下一栋了。〞
 <br/>

 <br/>
 国际消防员协会(IAFF)现役消防员代表艾鲍达拉(Tim Aboudara)说：〝天堂镇已经基本从地图上消失了。〞
 <br/>
 <br/>
 8000多名消防人员投入加州这场有纪录以来破坏性最大的野火。大火已摧毁了超过6,700栋建筑。
 <br/>
 <br/>
 加州州长布朗(Jerry Brown)请求川普（特朗普）总统发布〝重大灾难声明〞，川普已批准联邦援助。
 <br/>
 <br/>
 布朗表示，目前还不知道这场火灾的原因，但他警告这可能将是一种新的异常状态，表示加州未来继续面临野火威胁，还可能更糟。
 <br/>
 <br/>
 而加州官员警告，周日和周一的强风将助长目前的火势，令火势更加难以控制。
