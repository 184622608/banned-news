### 美中期选举三大推演皆不利中共 库德洛:北京在作糟糕赌博
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 已经拉开帷幕，执政共和党与在野民主党在国会参众两院的势力版图，持续引发外界关注。目前，更多的分析人士认为，中期选举无论结果如何，对中共来说都不乐观。
 <br/>
 <br/>
 在周二所有选举活动结束时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 50个州以及华盛顿特区将选出35席国会参议员、435名国会众议员、36个州的州长以及各州州内诸多立法席位。
 <br/>
 <br/>
 <h4>
  共和党众议员失势？仍存变数
 </h4>
 <br/>
 根据目前的选前民调推算，共和党继续保持在参议院优势的可能性较大，而民主党赢回众议院的优势，看起来似乎更大一些。
 <br/>
 <br/>
 对此，时事评论员文昭指出，共和党是否真会在众议院失去多数党地位，目前仍存有诸多变数。
 <br/>
 <br/>
 文昭认为，这种民调只是给出一个趋势的分析，正如〝2016年美国大选，多数民调机构就打偏了，那一次大多数民调机构都预测是希拉里・克林顿（Hillary Clinton）当选，没想到是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）当选了，所以这一次会不会又马失前蹄呢？〞
 <br/>
 <br/>
 同时，文昭也指出，这一次的误差风险不会像2016年美国大选时那么高，〝因为川普的很多支持者（2016年时）是隐形的，就是他不敢在公开场合表达支持川普，因为川普的很多观点是所谓‘政治上的不正确’，你要公开表达支持川普，会在生活中被人围攻的，所以很多人不敢表达。但是经过快两年的时间，川普的很多支持者已经从‘潜水’状态浮出水面，已经敢公开表达了。〞
 <br/>
 <br/>
 也有舆论认为，如果上述情况出现，总统川普针对中共的经济制裁政策将持续，在进一步的施压下，中共很可能会被迫回到谈判桌边。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
   中期选举
  </a>
  无益改善
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系.html" target="_blank">
   中美关系
  </a>
 </h4>
 <br/>
 另有政治观察家认为，美国政界已形成统一立场，即对中共必须强硬。因此，目前，国会里不仅听不到表扬中共的声音，相反地，对中共采取强硬态度已经成为两党〝政治正确〞的共识。
 <br/>
 <br/>
 圣路易斯华盛顿大学东亚系副教授马钊认为，中期选举对改善中美局势，不仅没有太大帮助，甚至可能让中美关系恶化。
 <br/>
 <br/>
 马钊形容说：〝不论中期选举哪一个党胜利，甚至不论下届总统是谁，中美关系都非常麻烦。〞他认为，〝中期选举后，中美进入新竞争关系这一格局不会调整。〞
 <br/>
 <br/>
 <h4>
  弹劾川普？民主党无胜算
 </h4>
 <br/>
 另一方面，有评论认为，即使民主党在此次中期选举中，赢回对众议院的控制权，但几乎可以肯定的是，川普在未来两年任期内，并没有可能被弹劾下台。
 <br/>
 <br/>
 文昭说：〝众议院只要有相对多数，他们就可以发出（对川普）弹劾的提议。〞〝但是总统要被罢免，至少要得到参议院三分之二以上的参议院的同意。但目前，共和党掌握参议院仍然是大概率，所以哪怕民主党能够发起弹劾动议，实际上能够通过的可能性也是很小的，其实就是几乎不可能。〞
 <br/>
 <br/>
 不过，文昭指出，民主党试图弹劾川普的用心，确可能对川普在未来两年的施政带来干扰。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
   库德洛
  </a>
  ：北京在作糟糕赌博
 </h4>
 <br/>
 然而，中共一直期待共和党在中期选举中失利，进而为中美关系带来转机。〝北京似乎认为等过了11月的中期选举，川普总统会变弱，进而减轻（对中共的）贸易打击。这是个很糟糕的赌博。〞白宫国家经济委员会主任库德洛（Larry Kudlow）7月接受美媒采访时说。
 <br/>
 <br/>
 上月，库德洛在接受福克斯新闻（FOX News）采访时，再度明确表示，〝如果这是他们（中共）的打算，那他们最好另打算盘，不管是谁控制（国会），我认为，美国对中国（中共）的政策都会非常近似。〞
 <br/>
 <br/>
 库德洛说：〝不管中期选举结果如何，即使是民主党，也不会对中国（中共）（的不公平贸易）进行放任。他们也很强硬，这是美国为自己国家的工人、制造业和整个经济维护贸易权力的时代。〞
 <br/>
 <br/>
 〝如果中共认为在中期选举后，（美国贸易政策）会有什么改变的话，那是一个误判。〞库德洛还谈到，G20的时候，川普总统也许会会晤习近平，〝我非常怀疑中美之间是否可以（达成协议），因为我们提出条件后，中方没有回应。〞
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
