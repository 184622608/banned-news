### 美封锁十年签证断中共关系网 陆专家赴美〝空前困难〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】有港媒周四披露，美国政府近日已撤回了部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系专家.html" target="_blank">
  中美关系专家
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十年签证.html" target="_blank">
  十年签证
 </a>
 ，现在中国专家赴美变得空前困难。有海外亲共媒体称，中国（中共）在美关系网被断。
 <br/>
 <br/>
 香港《南华早报》11月22日的一篇报导引述消息人士透露，美国驻北京的大使馆，已撤回了部分已颁发的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十年签证.html" target="_blank">
  十年签证
 </a>
 ，而中招者基本都是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系专家.html" target="_blank">
  中美关系专家
 </a>
 〞。
 <br/>
 <br/>
 报导并透露说，现在美方负责给签证把关的人员安排发生了改变。一位中美关系研究人员称，以后，赴美国的签证都要与总领事〝面签〞，而美国大使馆对此没有给出任何的理由。
 <br/>
 <br/>
 此外，中国的专家学者们发现，赴美签证的审核过程及时间更长了，有的研究人员因此不得已取消了原定的赴美行程。
 <br/>
 <br/>
 上文提到的研究人员还说，据他的了解，现在被取消十年签证的人数还不多，范围则集中在美国问题研究机构的中国专家群体。
 <br/>
 <br/>
 该报导并提到，国际经济交流智库中国部的首席经济师陈文玲（音译），近日在某个论坛上发言时坦言，中国的那些专家们现在想要和美国政府官员、美智库以及美国境内企业交流〝空前的困难〞 。
 <br/>
 <br/>
 据公开的资讯，美中两国政府2014年曾共同决定，给持有护照的商人、游客设立特惠通道，颁发十年期内能在美中之间多次往返的签证（简称〝十年签证〞），以免去需要不断申请签证的烦扰。但现在，这条便利通道很可能被美国政府单方面封锁。
 <br/>
 <br/>
 上述消息在海外中文媒体上公开后，有中国网友留言评论称，美国政府此举是在打击那些以〝研究〞和〝交流〞为幌子，配合中共政府到美国来给美国政府制造麻烦的机构和个人。
 <br/>
 <br/>
 还有中国网友在海外中文媒体上相关报导后留言评论称：美国政府现在是用〝以其人之道，还治其人之身〞的方法来对付中共，因此〝那些在媒体网络上为中共背书的‘聪明人’，是不是该提前做好准备？〞
 <br/>
 <br/>
 该网友并表示，如果美国政府也〝以毒攻毒〞，效仿中共在中国大陆压制外国媒体的做法，也对那些在美国和西方国家替中共效力的〝华人〞、媒体、网络工作者统统拒绝签证，或者把跳得最厉害的那种人驱逐出境，让那些想做美国梦的〝聪明人〞，回到〝党妈妈〞的怀抱里去做〝自力更生的中国梦〞，这恐怕是中共最害怕的举措，因为这样做相当于〝拿下了中共的笔杆子〞。
 <br/>
 <br/>
 （记者黎明综合报导）
