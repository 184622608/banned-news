### 宾州参、众议会通过友台决议案
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宾州.html" target="_blank">
  宾州
 </a>
 参、众议会10月16日分别通过第SR465号及第HR1149号决议案，除重申台美之经济、安全及文化伙伴关系外，并支持进一步加强台湾及宾州双边友好情谊。此两项决议案系由宾州州参议员Gene Yaw及州众议员Jeff Wheeland分别领衔提案。
 <br/>
 <br/>
 该两决议案强调双方现有在各面向之紧密关系： 2017年台湾为美国第11大贸易伙伴；台湾为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宾州.html" target="_blank">
  宾州
 </a>
 在亚洲之第7大出口市场；台湾企业且为宾州创造数千个就业机会；双方2015年签订生技合作了解备忘录及驾照相互承认协定；宾州2005年于台湾设立贸易投资办事处等；基此，州议会将续致力加强双边各项交流。
 <br/>
 <br/>
 驻纽约经济文化办事处徐大使俪文当日应邀偕政治组吕组长志坚及魏副领事浩威前往议会观礼。获宾州副州长兼参议会议长Mike Stack及众议会议长Mike Turzai致意欢迎，众议会议长Mike Turzai并邀请徐大使于议会致词。徐大使致词时表示，感谢美联邦政府与国会、宾州议会及州政府对台湾之长期支持，台湾为美国经贸、文化、教育及印太地区安全重要伙伴，诚盼双方进一步扩大经贸、文化、人民交流，共创互惠双赢。徐大使此行受到州参、众议员热烈欢迎，友我决议均无异议通过。
 <br/>
 <br/>
 宾州参、众议会自2009年后，再度于今年共同通过决议案，重申对台湾深挚友谊、承诺加强与台湾经贸伙伴关系，深具意义。
