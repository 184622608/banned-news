### 川普明尼苏达演说 背书多名共和党参选人
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】周四晚，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 第二次来到明尼苏达州，在竞选集会发表演说，为当地多位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党参选人.html" target="_blank">
  共和党参选人
 </a>
 助选。近万名民众举着标语，欢迎川普。
 <br/>
 <br/>
 明尼苏达州近万名民众，举着支持川普和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党参选人.html" target="_blank">
  共和党参选人
 </a>
 的标语，欢迎川普。
 <br/>
 <br/>
 川普在演讲中，首先提到最高法院大法官提名人卡瓦诺确认受阻一事。他表示，尽管民主党极力阻挠，卡瓦诺的支持率反而快速上升。
 <br/>
 <br/>
 美国总统 川普：〝大家只要看看过去三、四天的民调，表明他们（民主党人）的怒火刺激其抵抗，结果适得其反，达到了前所未有的程度。从没见过支持率上涨11、10、14，没人见过。〞
 <br/>
 <br/>
 他还表示，美国的非裔、西语裔和亚裔的失业率都创下历史新低，妇女的失业率也达到65年来的新低。
 <br/>
 <br/>
 谈到与中国的贸易关系，川普表示，首先要维护美国人的利益。
 <br/>
 <br/>
 美国总统 川普：〝多年来，中国（中共）一直在侵蚀我们，夺去我们的工厂，用我们的钱建设中国，却牺牲我们的工人和公司。我们不会再这么做了。不会了。〞
 <br/>
 <br/>
 周四，川普发推文，呼吁选民支持国会参议员参选人卡琳．霍斯利, 国会众议员参选人吉姆．黑格多恩和皮特・斯托伯，同时背书竞选连任的国会众议员杰森．路易斯和汤姆．艾穆尔。
 <br/>
 <br/>
 川普还鼓励选民在11月的选举中，一定要出来投票。
 <br/>
 <br/>
 新唐人记者安心、天羽综合报导
</div>
