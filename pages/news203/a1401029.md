### 争取参院最后一席 川普赴密西西比助选
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】周一，美国总统川普展开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的最后一场助选之旅，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_密西西比.html" target="_blank">
  密西西比
 </a>
 州的图珀洛和比洛克西出席两场集会，为现任共和党联邦参议员海德-史密斯拉票。
 <br/>
 <br/>
 周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_密西西比.html" target="_blank">
  密西西比
 </a>
 州进行复选投票，决定今年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 中联邦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 最后一个席位的归属。史密斯将与民主党挑战者埃斯皮再次对决。
 <br/>
 <br/>
 在史密斯的陪同下，川普在图珀洛(Tupelo)的集会上现身，他称赞史密斯是〝难以置信的优秀领导者和坚持不懈的冠军〞。
 <br/>
 <br/>
 美国总统川普：〝我到这里，是来要求密西西比人民再次让史密斯当选参议员，这样我们才能让美国再次伟大。〞
 <br/>
 <br/>
 川普还提到，目前几千名中美洲人在美墨边境试图非法进入美国，他表示不会再让非法入境的事情发生。
 <br/>
 <br/>
 美国总统川普：〝如果你们想要更多的工作，并减少犯罪，那么你必须投票给共和党人。〞
 <br/>
 <br/>
 而当川普进入比洛克西(Biloxi)的密西西比海岸体育馆参加第二场集会时，伴随着他的不仅有掌声，还有从天而降的人工雪花。几十棵圣诞树前站着木偶人，给现场增添不少节日气氛。
 <br/>
 <br/>
 最近民调显示，史密斯支持率为54%，领先对手埃斯皮10个百分点。
 <br/>
 <br/>
 目前共和党占有52个参院席位，民主党则为47席，若史密斯获胜，共和党将超出民主党6个席位。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
