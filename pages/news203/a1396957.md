### 纽约再现炸弹包裹 曼哈顿中城邮局拦截
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】本周全美已经累积13起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸弹包裹事件.html" target="_blank">
  炸弹包裹事件
 </a>
 ，其中的一个包裹，周五早上，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曼哈顿.html" target="_blank">
  曼哈顿
 </a>
 被发现，是指名寄送给前国家情报总监克莱柏。
 <br/>
 <br/>
 记者：〝我们现在在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曼哈顿.html" target="_blank">
  曼哈顿
 </a>
 西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_52街邮局.html" target="_blank">
  52街邮局
 </a>
 ，一个准备要寄送给美国前国家情报总监克莱柏（James Clapper）的炸弹包裹，就在这里先被拦截了。〞
 <br/>
 <br/>
 可疑包裹在早上8点多钟被发现后，当局马上封锁邮局、疏散人员，包裹很快就被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约警方.html" target="_blank">
  纽约警方
 </a>
 的防爆车取走处理。包裹内容与之前多起事件中的管状土制炸弹类似，且外观都是以马尼拉纸包装。
 <br/>
 <br/>
 炸弹原先是要寄到位于哥伦布圆环的CNN纽约办公室。
 <br/>
 <br/>
 纽约民众Amanda Armbruster：〝我就住在那边，那边都被封锁了，一片混乱。我很高兴他们发现了包裹，竟然是通过邮件的方式寄送，我不敢相信这些事的发生。〞
 <br/>
 <br/>
 在全美这一系列
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸弹包裹事件.html" target="_blank">
  炸弹包裹事件
 </a>
 中，就有3个包裹是寄到纽约曼哈顿。
 <br/>
 <br/>
 纽约市长Bill de Blasio：〝这几天这些炸弹攻击让纽约市和美国经历了许多挑战。好消息是，现在有个嫌犯已经被逮补，是个重大的突破，这带给了我们希望，希望这个危机能够赶快结束。〞
 <br/>
 <br/>
 周五当天，在佛州和加州还有另外两起炸弹包裹被拦截。
 <br/>
 <br/>
 纽约民众Larry Waller：〝我觉得很遗憾，有人用这种极端方法表明立场，给无辜的人带来危险，这种做法是错的，解决政治问题有其他办法。〞
 <br/>
 <br/>
 截至周五的全美13起炸弹包裹事件中，没有造成任何人员伤亡。
 <br/>
 <br/>
 新唐人记者王愉贺、王真纽约报导
</div>
