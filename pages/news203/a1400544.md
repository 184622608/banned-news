### 〝黑色星期五〞提前降临   感恩节现购物潮
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】又到了美国〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑色星期五.html" target="_blank">
  黑色星期五
 </a>
 〞购物季。如今，越来越多的商家在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 当天晚上就开门营业，让〝黑色星期五〞的购物热情，提前降临。
 <br/>
 <br/>
 零售巨头沃尔玛全美各家店铺，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 当晚6点就开门营业。购物人潮蜂拥而入，将〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑色星期五.html" target="_blank">
  黑色星期五
 </a>
 〞变成〝黑色星期四〞。
 <br/>
 <br/>
 如同往年，电器和数码产品是黑色星期五打折的重头戏。今年的黑色星期五，40寸高清电视在沃尔玛只要99美元，65寸4K超清智能电视只卖398美元。还有许多笔记本电脑、平板电脑以及智能手机也有不菲的折扣。
 <br/>
 <br/>
 电子产品零售巨头百思买今年也在感恩节提前开门。在德州的一家店铺，感恩节下午人们就早早排起了长队。
 <br/>
 <br/>
 到开门的时间，人们欢呼涌入店铺，享受〝黑色星期五〞的购物乐趣。
 <br/>
 <br/>
 黑色星期五购物者雪瑞儿・普拉特：〝我们每年都会购物，这已经成为节日的传统。我们给孩子们买礼物，给家人买礼物。我们买的礼物，许多人都买不起，除非有两百、四百、六百美元的折扣，所以感觉很棒。〞
 <br/>
 <br/>
 在纽约梅西旗舰店，傍晚5点一到，等候多时的顾客像洪水一般涌入店内，冲向他们的目标。
 <br/>
 <br/>
 有人双手捧满了战利品，而收银台前也排起了结帐的长龙。
 <br/>
 <br/>
 纽约民众崔小姐：〝自己有能力赚一点钱，就可以给爸妈买一点东西。应该的，这些都是应该的，比起他们做的，这些实在是微不足道。〞
 <br/>
 <br/>
 记者：〝虽然今年的感恩节很冷，但是能感受到民众的热情。截至晚上六点多，已经有人收获满满，走出店外。〞
 <br/>
 <br/>
 据美国零售业协会估计，今年平均每位消费者会在感恩节到圣诞节期间消费超过1,000美元，总零售额可达到7,210亿美元。
 <br/>
 <br/>
 分析师预计，〝黑色星期五〞当天的销售额，将比去年有所增长。
 <br/>
 <br/>
 新唐人记者柯婷婷、林宇、天羽综合报导
</div>
