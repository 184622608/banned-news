### 美国老牌零售商西尔斯 即将申请破产
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 老牌百货公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西尔斯.html" target="_blank">
  西尔斯
 </a>
 公司即将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_申请破产.html" target="_blank">
  申请破产
 </a>
 保护，西尔斯每年需要超过10亿美元才能维持运营，但连年销售业绩低迷。
 <br/>
 <br/>
 受亚马逊等网络购物业者的冲击，曾是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 最大零售商的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西尔斯.html" target="_blank">
  西尔斯
 </a>
 （Sears），近年销售业绩惨淡，最快在本周五
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_申请破产.html" target="_blank">
  申请破产
 </a>
 保护。
 <br/>
 <br/>
 今年第二季度，西尔斯损失了5.08亿美元。相比去年同期多了6600万美元。 在8月4日结束的第二季度，经营超过12个月的分店销售额也下降了3.9％。
 <br/>
 <br/>
 品牌策略师 购物专家Trae Bodge：〝我想他们是现代化改组上出了问题，不像其他一些企业跟上了时代步伐。例如沃尔玛，显然不像西尔斯那样老牌，但它们一直随时代转型，西尔斯则有点停滞不前。〞
 <br/>
 <br/>
 西尔斯股价下跌36％，现在只有37美分，而该股票在十年前交易价格超过100美元。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
