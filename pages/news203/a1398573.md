### 洛杉矶千橡市酒吧枪击案 12人死
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】周三深夜，美国加州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洛杉矶.html" target="_blank">
  洛杉矶
 </a>
 千橡市一家酒吧，发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，导致12人死亡，其中包括嫌犯和一名警察。警方表示嫌犯身份已经确认。
 <br/>
 <br/>
 周三晚11点20分，距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洛杉矶.html" target="_blank">
  洛杉矶
 </a>
 西部40英里的一个酒吧，发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，导致12人死亡，其中包括一名警察，嫌犯也死在酒吧内。
 <br/>
 <br/>
 警方确认，嫌犯是一名29岁男子，案发时使用了烟雾装置和.45口径手枪。
 <br/>
 <br/>
 警方表示，还不清楚嫌犯是否与恐怖主义有关联。
 <br/>
 <br/>
 加州文图拉县警官 Geoff Dean：〝我们不清楚是否与恐怖主义相关。〞
 <br/>
 <br/>
 警方表示，当时酒吧里有100多人，很多是大学生。警方接到报案，三分钟内赶到现场。
 <br/>
 <br/>
 文图拉县警长贺拉斯 首先冲进前面，不幸被枪手击中，身中数枪，同伴随后冲进酒吧，嫌犯自杀身亡。
 <br/>
 <br/>
 贺拉斯随后被送进医院，由于伤势过重，不幸殉职。贺拉斯从警29年，明年即将退休。
 <br/>
 <br/>
 川普总统周四，发推文表彰警方的勇气。
 <br/>
 <br/>
 川普推文还说：〝警方展现了极大勇气。加州高速巡逻警察三分钟内赶到现场，第一名（进入酒吧）的警察多次被击中。这名警官在医院殉职。神保佑所有遇害者和他们的家人。感谢执法者。〞
 <br/>
 <br/>
 目击者回忆，枪响时，人们四处逃命，场面混乱。
 <br/>
 <br/>
 目击者Teylor Whittler：〝我听到枪声，我转过头就看见，（枪手）他连接开了几次枪。没一会儿，所有人都在喊‘趴下’。〞
 <br/>
 <br/>
 案发酒吧每周三举行〝大学乡村之夜〞活动，酒吧附近是加州路德大学和加州州立大学。
 <br/>
 <br/>
 新唐人记者毕心慈、宇微综合报导
