### 脸书成立新战争工作室 应对中期选举
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 在即，社交媒体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脸书.html" target="_blank">
  脸书
 </a>
 在硅谷总部设立〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战争工作室.html" target="_blank">
  战争工作室
 </a>
 〞，希望杜绝虚假帐户，打击假新闻，严防有人企图利用脸书干预中期选举。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脸书.html" target="_blank">
  脸书
 </a>
 管理层表示，成立新战争室是为保护脸书选举平台的安全。
 <br/>
 <br/>
 脸书产品管理总监查克拉巴蒂：〝这实际上是我们整个公司两年努力的结果，在人员和技术方面投入大量资金，以确保平台以及选举安全。〞
 <br/>
 <br/>
 查克拉巴蒂表示，最近六个月，阻止了13亿个假帐号。
 <br/>
 <br/>
 脸书产品管理总监查克拉巴蒂：〝这一点非常重要，因为我们通常能够在第一时间，阻止创建假帐户，能够在它们造成任何伤害之前阻止它们。〞
 <br/>
 <br/>
 这个团队共有2万人，24人小组在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战争工作室.html" target="_blank">
  战争工作室
 </a>
 〞24小时坐镇。
 <br/>
 <br/>
 脸书选举战争室主任 Lexi Sturdy：〝作战室这些人代表威胁情报部门、数据科学家、研究等团队，我们在这个会议室以简化沟通和决策。〞
 <br/>
 <br/>
 他们的首要任务是打击假新闻，严防有人干预
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 。
 <br/>
 <br/>
 脸书选举战争室主任 Lexi Sturdy：〝因此在内部，我们会监控诸如选举相关内容的用户报告，内容是否违反了我们的社区标准，例如仇恨言论或压制选民。〞
 <br/>
 <br/>
 除了脸书，推特也采取行动，防范被利用来干预中期选举。
 <br/>
 <br/>
 推特近日公布了超过1千万条与外国试图干涉美国选举有关的推文。
 <br/>
 <br/>
 新唐人记者李梅、张琪综合报导
</div>
