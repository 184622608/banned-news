### 西尔斯破产 将关142家门店或有清仓销售
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】因巨额债务负担和亏损，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国百货公司.html" target="_blank">
  美国百货公司
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西尔斯.html" target="_blank">
  西尔斯
 </a>
 周一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_申请破产.html" target="_blank">
  申请破产
 </a>
 。 今年年底前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Sears.html" target="_blank">
  Sears
 </a>
 将关闭142家分店，预计很快将开始清仓销售。
 <br/>
 <br/>
 自2012年以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西尔斯.html" target="_blank">
  西尔斯
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Sears.html" target="_blank">
  Sears
 </a>
 ）已经亏损超过60亿美元。
 <br/>
 <br/>
 西尔斯（Sears）销售官员Wendy Liebmann ：〝品牌的价值很小，我指的是店里的一些有匠心的货品和一些真正的标志性品牌。〞
 <br/>
 <br/>
 近年，西尔斯一直努力应对危机，但并不成功。
 <br/>
 <br/>
 西尔斯（Sears）销售官员Wendy Liebmann ：〝Sears和Kmart里有的东西太多是很平淡无奇的，那些过去曾是标志性品牌的影响力也因此在流失。〞
 <br/>
 <br/>
 Liebmann认为西尔斯远远落后于数字时代。
 <br/>
 <br/>
 西尔斯（Sears）销售官员Wendy Liebmann ：〝今天那么多人的生活方式和购物方式变换多样。西尔斯依然活在一个非常古老的世界。除此之外，我认为他们的业主、管理层和董事会的注意力太集中在物业地产方面了。〞
 <br/>
 <br/>
 西尔斯曾在美国商界称霸一时，如今破产，100多家门店面临关闭，不少商品可能要随之清仓处理。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
