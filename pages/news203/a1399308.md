### 南加州又添新火 全州野火死者升至50
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】周二，肆虐南北加州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火.html" target="_blank">
  野火
 </a>
 仍未得到有效控制，目前报告的死亡人数已经上升到50人。同时，在洛杉矶以东50英里的方塔纳市又爆发了一处新的野火。
 <br/>
 <br/>
 周二晚间，圣贝纳迪诺县的方塔纳市市，又燃起一处
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火.html" target="_blank">
  野火
 </a>
 。火势借风力迅速蔓延。消防员赶到现场，零点左右暂时将野火面积控制在20英亩，目前居民没有接到疏散通知。
 <br/>
 <br/>
 在北加州肆虐的〝营地〞野火和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南加州.html" target="_blank">
  南加州
 </a>
 的〝伍尔西野火〞，仍未得到有效控制。目前，〝营地〞野火造成的死亡人数上升到48人，整个加州已有50人不幸丧生。
 <br/>
 <br/>
 在天堂镇，法医队伍正在废墟中寻找遇难者遗骸。目前仍有228人失踪，预计确认的死者人数还会上升。
 <br/>
 <br/>
 同时，400名美国国民警卫队也加入了救急队伍。
 <br/>
 <br/>
 加州国民警卫队准将海什默：〝我们的人数正在增加，我们刚刚开始接管交通管制工作，以减轻警察和执法部门的负担，让他们能去做其他工作。我们从这开始，可能会逐渐接手其他任务。〞
 <br/>
 <br/>
 在南加州马里布市，野火延烧到当地有名的野生动物牧场。工作人员经过努力，成功疏散了长颈鹿等珍贵的野生动物，但牧场的大部分设施却付之一炬。
 <br/>
 <br/>
 截至周二，北加州〝营地〞野火面积已扩大到13万英亩，目前有35%火势得到控制。在南加州，〝伍尔西野火〞面积超过9万7千英亩，40%火势得到控制。
 <br/>
 <br/>
 新唐人记者林宇、张琪综合报导
</div>
