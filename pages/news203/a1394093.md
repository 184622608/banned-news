### 梅拉妮亚抵马拉维 捐书140万册
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】10月4日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国第一夫人.html" target="_blank">
  美国第一夫人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉妮亚.html" target="_blank">
  梅拉妮亚
 </a>
 抵达非洲之行第二站
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马拉维.html" target="_blank">
  马拉维
 </a>
 ，探访学校，同参与实施美国援助学校计划的官员座谈。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国第一夫人.html" target="_blank">
  美国第一夫人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉妮亚.html" target="_blank">
  梅拉妮亚
 </a>
 抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马拉维.html" target="_blank">
  马拉维
 </a>
 ，继续推动她针对儿童的〝做最好自己〞（Be Best）的倡议。
 <br/>
 <br/>
 梅拉妮亚观赏了学生歌舞，聆听课堂教学，一起阅览他们的教科书，并与实施美国援助的马拉维学校计划的官员座谈。
 <br/>
 <br/>
 美国第一夫人梅拉妮亚：〝我希望来这，看看美国提供给孩子们的成功教育项目。谢谢你们培养他们做到最好，让他们做最好的自己，并使后代们成长为有修养的人。〞
 <br/>
 <br/>
 在现场，梅拉妮亚还给孩子们分发书籍，此次她一共捐出140万本书。过去几年，马拉维当地学校得益于美国资助的一项阅读计划，已经得到了960万册书籍。
 <br/>
 <br/>
 梅拉尼亚在非洲的首次亲善访问得到了川普总统在推特上的赞扬：〝我们了不起的第一夫人在非洲做得真棒。人们喜欢她！真是一件好事！〞
 <br/>
 <br/>
 接下来，梅拉妮亚还将访问肯尼亚和埃及，预计周五返回美国。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
