### 遏止中美洲难民涌入  川普或关闭南部边界
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 最近两日连发推文，抨击民主党糟糕的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民政策.html" target="_blank">
  移民政策
 </a>
 。面对正在由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲.html" target="_blank">
  中美洲
 </a>
 不断向北挺进的大批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民潮.html" target="_blank">
  难民潮
 </a>
 ，川普表示，如果墨西哥无法遏止中美洲难民北上，美国就将〝关闭南部边界〞。
 <br/>
 <br/>
 为躲避纷乱和贫穷，近日，大批来自危地马拉、洪都拉斯及萨尔瓦多等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲.html" target="_blank">
  中美洲
 </a>
 国家的民众，拖家带口，穿越丛林，希望穿越危地马拉和墨西哥，逃到美国寻求庇护。
 <br/>
 <br/>
 周四（18日），总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 连发三则推文，要求墨西哥及中美洲各国家阻止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民潮.html" target="_blank">
  难民潮
 </a>
 跨越边境。
 <br/>
 <br/>
 川普说：〝我正在观察民主党领导的来自危地马拉、洪都拉斯和萨尔瓦多对我们国家的攻击（因为民主党人主张开放边界以及支持现行软弱无力的移民法）。这些国家为数众多的人民，包括许多罪犯，通过墨西哥进入美国，而他们的总统没有采取更多行动加以阻止。〞
 <br/>
 <br/>
 〝这些国家几乎无法控制其人民（向北移动），因此除了全面终止对这些国家提供援助资金，我必须强烈要求墨国力阻这个移民车队。如果墨国办不到，我会派遗美军并且关闭我们的南部边境！〞
 <br/>
 <br/>
 〝我们国家南部边境遭到攻击的问题，包括犯罪及毒品的大举入侵，身为总统的我，相当重视这个问题，且其重要性远超过贸易或USMCA（美墨加新自贸协定）。希望墨国能阻止大批移民车队通过其北部边境。民主党的错误在于支持软弱无力的法律！〞
 <br/>

 另据美联社报导，奉川普指示，国务卿蓬佩奥（Mike Pompeo）周四将前往巴拿马城、周五前往墨西哥城，分别和巴拿马及墨国总统会面。
 <br/>
 <br/>
 国务院一位高级官员透露，蓬佩奥此行的目的是，与两国总统讨论正在进行的安全合作项目，包括打击在边境有关现金、武器、毒品的非法流动、阻止非法移民入境、加强对贪腐和侵犯人权人士的归责，以及将跨国犯罪组织成员绳之以法。
 <br/>
 <br/>
 该官员还表示，移民车队将是会谈焦点，〝我们认为这是共同的问题，将继续与该地区各国密切合作，以解决造成非法移民入境美国的问题，包括经济、安全和治理等各方面的潜在问题。〞
 <br/>
 <br/>
 自2015年以来，美国政府向危地马拉、洪都拉斯和萨尔瓦多等国家提供了超过26亿美元的援助资金。川普近日再次威胁，如果这些国家放任大规模移民非法入境美国，美国将切断对这些国家的所有援助。
 <br/>
 <br/>
