### 孟晚舟被捕受审 美国智库：对中共警讯
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 财务长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被抓一事，美国有关专家认为，此案是美方向中共发出的强烈信号，美国不会容忍非法行为。
 <br/>
 <br/>
 12月1日，正当美中首脑在阿根廷就贸易问题紧张谈判之际，中共科技巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司财务长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_温哥华被捕.html" target="_blank">
  温哥华被捕
 </a>
 。白宫国家安全顾问博顿否认川普总统事先知情，但专家分析，这一行动时间很可能是有意选择。
 <br/>
 <br/>
 美国传统基金会经济学家Steve Moore：〝这个(抓捕的)时间可能是有意的，要让中共知道，如果我们达成协议，我们要确保协议会得到强有力的执行。〞
 <br/>
 <br/>
 哈德森研究所高级研究员Arthur Herman：〝这是一个重大决定，抓捕华为创始人的女儿。这是严肃的一步，但是早就该这么做了。违背对伊朗的制裁当然是严重的事情，但这只是华为参与（非法行为）的一小部份。〞
 <br/>
 <br/>
 彭博社报导，华为被查原因，除涉嫌违反美国对伊朗的贸易制裁外，恐怕还包括图谋美国学术机构最新技术。
 <br/>
 <br/>
 实际上，美国国会早就点名华为涉嫌间谍活动，情报部门也高度关注。参议院情报委员会副主席沃纳 (Mark Warner) 6日推文说，华为与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴.html" target="_blank">
  中兴
 </a>
 一样对美国国家安全构成威胁。大量证据表明，没有一家中国大公司是独立于中共政府和共产党。
 <br/>
 <br/>
 哈德森研究所高级研究员Arthur Herman：〝华为、中兴只是中共政府控制的延伸，代表其官方目的和计划。〞
 <br/>
 <br/>
 目前美中之间在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5G.html" target="_blank">
  5G
 </a>
 市场竞争激烈，而华为作为中共扶持的领头羊，设备安全性令人担懮。日本媒体报导，出于安全考量，日本政府下周一就将出台规则，禁止政府采购华为及中兴设备。这是继美国、澳大利亚、新西兰之后，采取类似行动的又一民主国家。
 <br/>
 <br/>
 哈德森研究所高级研究员Arthur Herman：〝各国刚刚开始认识到，在未来移动技术中，使用华为设备与安装的真正危险性。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
