### 中期选举完而未结 佛州备受瞩目
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 普选日已过去一周，虽然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 控制众议院、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 控制参议院的大局看似已定，但仍有一些选战尚未决出结果。
 <br/>
 <br/>
 微弱的差距、不明来历的选票、选举舞弊起诉——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 联邦参议员席位的竞争，目前正在重新计票过程中。
 <br/>
 <br/>
 原本上周二
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 人斯科特以领先
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 人纳尔逊5万多张选票宣布胜选。不过布劳沃德郡（Broward）和棕榈滩郡（Palm Beach）在截止日期后持续清点出新选票，使得二人差距不断缩小，达到州法律规定的0.5%的重新计票标准线以下。佛州州务卿上周六宣布机器重新计票，本周四是截止日。
 <br/>
 <br/>
 仍在履行州长职务的斯科特下令州执法部门调查布劳沃德郡和棕榈滩郡的不明选票是否涉及选举舞弊，并增派执法人员赴布劳沃德郡监督重新计票。
 <br/>
 <br/>
 而民主党人则要求法庭延长重计票的截止日以有足够时间清点所有的邮寄选票。
 <br/>
 <br/>
 川普对此高度关注，他在周二的推文中称〝民主党人纳尔逊打算何时承认败选呢？靠布劳沃德郡和棕榈滩郡是找不到足够选票的。〞
 <br/>
 <br/>
 佛州州长职位的竞争，也因为相同原因，面临重新计票的情况。
 <br/>
 <br/>
 另一个悬而未决的州长竞争在乔治亚州共和党人坎普与民主党人艾布兰之间。普选日前夕，联邦法院地区法官梅伊（Leigh Martin May）下令乔治亚州不得作废那些涉及选票签名与档案记录不符的缺席选票，而是发出临时选票让选民重新填写并解释原因。12日，联邦法官托滕贝格（Amy Totenberg）要求州选举官员重新核查这些临时选票。最早周五有最终结果。
 <br/>
 <br/>
 密西西比州参议员席位的竞逐仍未完结，27日面临复选投票，现任共和党参议员史密斯（Cindy Hyde-Smith）和民主党人埃斯皮（Mike Espy）在普选中得票率都未超过50%。史密斯持共和党传统理念——低税收、小政府。埃斯皮则是获得左派大金主索罗斯的资助（根据政治献金观察网站OpenSecrets数据），范士丹等激进民主党参议员也出资支持；他赞同全民健保及堕胎、反对建边境墙等。
 <br/>
 <br/>
 截至目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 后的全美州长席位，共和党占25人、民主党23人；联邦参议院共和党51席、民主党47席；众议院共和党200席，民主党225席。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
