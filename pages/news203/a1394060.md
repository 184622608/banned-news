### 彭斯演讲稿提前曝光 从4方面对中共开火
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】美国副总统彭斯周四上午11时在哈德逊研究所发表演说，路透社提前曝光了彭斯演讲的重点内容。据透露，彭斯将从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海军舰对峙.html" target="_blank">
  南海军舰对峙
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_干预美国选举.html" target="_blank">
  干预美国选举
 </a>
 、台湾政策、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 这四大方面，对中共作出严厉抨击。
 <br/>
 <br/>
 据路透社披露，彭斯首先将针对南海争端，提到美中军舰在南海一度发生对峙的事件，并再次强调，美国海军将继续在国际法允许和美国国家利益要求的〝任何地方飞行、航行和行动〞。
 <br/>
 <br/>
 美国迪凯特号（USS Decatur）驱逐舰周日在南沙群岛的南薰礁（美称Gaven Reef）和赤瓜礁（美称Johnson Reef）12海里范围内〝自由航行〞时，曾遭中共一艘军舰近距离追逼而导致双方一度发生对峙。
 <br/>
 <br/>
 其次，彭斯将在谈话中抨击中共干预美国的选举。据称，中共不但针对美国2018年的期中选举，瞄准了扮演重要角色的企业和各州选民，意图影响选举结果，而且早在2016年美国总统选举中，中共就曾瞄准了超过80%投票给川普的美国各州选民，现在中共正试图扭转这些选民的看法。
 <br/>
 <br/>
 第三，彭斯还将批评中共威逼利诱三个拉美国家与台湾断交的做法。彭斯谴责中共的这些做法对台湾海峡的稳定造成了威胁。
 <br/>
 <br/>
 提前曝光的演讲稿中，彭斯说，〝虽然我国政府将继续尊重我们的一个中国政策，正如三个联合公报和与台湾关系法所反映的那样，但我还想说，台湾拥抱民主，为所有中国人展示了一条更好的道路。〞
 <br/>
 <br/>
 第四，彭斯还将谈到中共政府正试图通过向亚洲、非洲、欧洲甚至拉丁美洲的政府提供大笔贷款，推行〝债务外交〞来扩大其全球影响力的问题。
 <br/>
 <br/>
 （记者欣然报导/责任编辑：秦锋）
</div>
