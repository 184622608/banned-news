### 高层对美立场混乱？蓬佩奥访华罕见一幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】有港媒25日披露，美国务卿蓬佩奥10月访问北京期间，遭到中方冷落，特别是与中共外长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 会晤后，中方甚至没带他吃顿饭，中共如此失礼的做法，甚至引发高层内部官员的不满。由此可见中共在对美关系方面，高层政见不一，已经乱作一团。
 <br/>
 <br/>
 香港《南华早报》25日引述熟悉内情的消息人士说，当美国务卿蓬佩奥（Mike Pompeo）10月8日访问中国时，他受到了冰冷的待遇。
 <br/>
 <br/>
 蓬佩奥是在美国副总统彭斯刚刚发表了全面性抨击中共的演讲、两国关系日益恶化的情况下访问北京的。
 <br/>
 <br/>
 中共官员在公开场合十分直接的指责美国升级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 。蓬佩奥希望会晤习近平也被拒绝了。之后蓬佩奥会晤了中共外长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 ，会晤持续不到一个小时，王毅对美国进行了多方指责，他把贸易摩擦、信任遭到破坏、台湾问题等矛盾都归咎于美方， 并要求〝美方停止这种错误言行〞。
 <br/>
 <br/>
 这位消息人士说，会晤之后，王毅和他的团队甚至没有带蓬佩奥吃顿饭，这非常失礼。而中共的一些国防官员和其他官员，都不认可王毅处理蓬佩奥访问的方式。
 <br/>
 <br/>
 蓬佩奥访问中国前的10月4日，美国副总统彭斯发表美国政府对华政策的演说，对中共内外政策提出严厉批评，除了贸易战，他还指责了中共对美国在军事、人权、宗教信仰、科技、意识上进行渗透，以及干预美国政治等多方面的恶行。
 <br/>
 <br/>
 在中美贸易战持续升温的背景下，美中进入新冷战状态，而中共内部各级官员更是乱作一团。
 <br/>
 <br/>
 港媒苹果刊发评论说，贸易战开打以来，中共政经形势不妙，股市也持续下跌，金融形势险峻。而中共内部的倾轧更可怕，很多被掩盖的问题暴露出来，就连中共内部团结一致的假象也被戳破。
 <br/>
 <br/>
 至今中共仍没有统一的应对贸易战的策略，各级官员自然乱作一团，都在忙于卸责。
 <br/>
 <br/>
 纽约时报也曾报导称，在中美贸易战中，尽管中共可以依赖其极权政治架构来杜绝异见者声音，确保执政权力，并且可以企盼〝美国的选民因为贸易战痛苦而将川普赶下台〞，但是共产党本身也有其致命的缺点。
 <br/>
 <br/>
 报导引述广州暨南大学国际关系教授陈定定的说法称，中美贸易战中，中共政府内部有两种声音，一方建议应该更强硬，对川普要反守为攻，一方则认为应该对美国的要求作出让步。
 <br/>
 <br/>
 报导引述分析称，这反映了北京政权的政治弱点，就是内部只能〝议而不决〞。而另一种分析则认为，内部的争论不休，或说明最高层的〝权威〞已经不足以统一党内认识。
 <br/>
 <br/>
 中美贸易战以来，中国股市连番下跌，外企出逃、失业率上升、消费断崖式的跳水，使中共政权陷入内外交困的险境。而贸易战压力更让中共内部陷入末日恐慌，这也给中共党内各种势力制造了机会，进一步搅乱中共政局，引发内部分裂。
 <br/>
 <br/>
 尽管中共官媒密集释放信号，为贸易战和经济政策定调。不过，海内外党媒舆论的混乱，各部门政策和声音的不协调，似乎透露出中共内部依然没有〝统一认识〞。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
