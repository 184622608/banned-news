### 雪城举办中华民国国庆升旗庆典 多位政要出席
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雪城.html" target="_blank">
  雪城
 </a>
 （Syracuse）市政府及〝纽约上州台湾同乡会〞于本月12日上午共同举办庆祝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中华民国.html" target="_blank">
  中华民国
 </a>
 第107年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国庆.html" target="_blank">
  国庆
 </a>
 升旗典礼，驻纽约台北经济文化办事处处长徐俪文大使偕教育组杨主任敏玲、侨教中心黄主任正杰及政治组吕组长志坚等同仁驱车前往出席。典礼由〝纽约上州台湾同乡会〞会长刘忠钦主持，新任雪城市市长Ben Walsh、Onodaga 郡郡长Ryan McMahon及纽约州众议员Pam Hunter等当地政要，以及来自雪城、罗彻斯特（Rochester）、优地加（Utica）、绮色佳（Ithaca）等地台湾侨胞及留学生代表共百余人出席。

 <br/>
 徐大使致辞时表示，此次是雪城市政府自1995年李登辉前总统访美以来，连续第23年在市政府广场举办升旗典礼，庆祝中华民国国庆，意义非凡。感谢Walsh 市长及此间政要、侨胞及留学生对台湾之坚定支持。台湾与美国不仅经贸、文化及教育等实质关系密切，更重要的是共享自由、民主与人权的普世价值。徐大使另强调美国国务院近日在多次不同场合表示，台湾是美国重要伙伴、民主成功的故事以及世界良善的力量，台湾期盼与美国及纽约州深化经贸等各项领域的实质关系。
 <br/>
 <br/>
 Walsh市长致辞肯定台美双边关系之正向发展，并推崇我侨民及留学生对雪城经济及多元文化的卓越贡献，另强调台湾民主成就及经济发展令人称羡，雪城重视与台湾关系，至盼加强双边交流。W市长并特别颁发贺状祝贺中华民国107年国庆，并宣布2018年10月12日为雪城之〝台美友谊日 〞(Taiwan-American Friendship Day)。
 <br/>
 <br/>
 徐大使乙行另藉此次辖访纽约上州雪城的机会，拜会水牛城（Buffalo）市长Byron Brown及罗彻斯特市长Lovely Warren讨论加强双边合作机会，并与当地侨胞及留学生互动交流。
