### 终结冷战 老布什病逝 戈尔巴乔夫赞其贡献
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】美国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 ，于当地时间，周五晚间10点10分，以94岁高龄，与世长辞。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ，以及前苏联领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戈尔巴乔夫.html" target="_blank">
  戈尔巴乔夫
 </a>
 ，都发表声明，表达哀悼之意。
 <br/>
 <br/>
 布什家族发言人麦格拉斯，周五深夜发布声明，对外宣布了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 过世的消息。
 <br/>
 <br/>
 老布什在1989年至1992年间担任美国第41任总统。他执政期间，经历了柏林墙倒塌、天安门六四事件、苏联解体、海湾战争等重要历史事件，领导美国在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冷战.html" target="_blank">
  冷战
 </a>
 结束时期向冷战后过渡。
 <br/>
 <br/>
 94岁的老布什是美国历史上最长寿的总统。他曾担任里根总统的副总统8年，同时也是小布什总统的父亲。
 <br/>
 <br/>
 1992年，老布什在竞选连任时败给了民主党候选人克林顿，成为最后一位曾参加过二战的美国总统。
 <br/>
 <br/>
 目前，正在阿根廷参加G20峰会的美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 及夫人，第一时间发表声明，悼念老布什。
 <br/>
 <br/>
 川普在声明中説，〝布什总统真实、风趣，对信仰、家庭和国家的责任感从未动摇，他激励了数代美国同胞为国服务，用他的话说，成为‘千万点光芒’将美国的伟大、希望与机遇照向世界。〞
 <br/>
 <br/>
 老布什执政时期的苏联领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戈尔巴乔夫.html" target="_blank">
  戈尔巴乔夫
 </a>
 ，也向老布什的家人表达悼念之意。戈尔巴乔夫表示，他和老布什经历了巨变的时代，需要领导人有重大的责任感。他深切感激老布什和夫人一生中传达的关切和善意。
 <br/>
 <br/>
 老布什曾是二战飞行员，还以跳伞庆祝了包括90岁在内的多个生日。退休后，他仍然热心参加各种公共活动。
 <br/>
 <br/>
 与老布什结发73年的前第一夫人芭芭拉于今年4月17日辞世，两人共育有6个孩子。老布什的遗体将和夫人以及早夭的女儿，一同下葬在德州农工大学的布什总统图书馆旁。
 <br/>
 <br/>
 新唐人记者林宇综合报导
