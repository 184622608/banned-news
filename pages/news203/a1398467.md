### 应川普要求 司法部长塞申斯辞职
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】周三中午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）记者会上，媒体追问内阁成员和白宫人员的去留，川普不置可否。但稍后，川普发推文，宣布司法部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞申斯.html" target="_blank">
  塞申斯
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 。
 <br/>
 <br/>
 媒体报导，美国司法部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞申斯.html" target="_blank">
  塞申斯
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统的要求下请辞。
 <br/>
 <br/>
 川普发推文，宣布任命新的司法部长，并感谢塞申斯在任期间的工作。
 <br/>
 <br/>
 推文说：〝我们很高兴地宣布，司法部长塞申斯的幕僚长，马修．惠特克将成为新的美国代理司法部长。他将很好地为我们的国家服务。〞
 <br/>
 <br/>
 川普还说，惠特克暂时代理司法部长，直到继任人选出台。
 <br/>
 <br/>
 白宫表示，代理司法部长将负责司法部所有事宜，包括〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 〞调查。
 <br/>
 <br/>
 惠特克曾任艾奥瓦州检察官，曾批评〝通俄门〞调查范围太大。
 <br/>
 <br/>
 最近一年来，川普曾在公开和私下里多次批评塞申斯，对所谓〝通俄门〞的相关调查采取回避态度。
 <br/>
 <br/>
 今年9月，川普甚至表示〝我没有司法部长。〞显示双方关系已十分紧张。
 <br/>
 <br/>
 塞申斯成为中期选举后首位下台的内阁成员。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
