### 洩露资讯者入室被抓 威胁公开参议员孩子信息
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】美国国会警方日前指控，国会民主党工作人员科斯科（Jackson A. Cosko）私自在网上公布了至少3名共和党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议员.html" target="_blank">
  参议员
 </a>
 的个人资讯，并威胁证人如果其说出他的行为，他将洩露参议员的孩子们的信息。
 <br/>
 <br/>
 据福克斯新闻（Fox News）周四（10月4日）报导，27岁的民主党国会职员科斯科（Jackson A. Cosko）周三被逮捕，警方指控他涉嫌犯有威胁、干扰证人、未经授权访问政府电脑、身份盗窃、入室盗窃、非法进入、公布受限制的个人信息等多项罪行，触犯了联邦及当地法律。
 <br/>
 <br/>
 科斯科近来曾为德州民主党众议员希拉‧杰克逊‧李（Sheila Jackson Lee）工作，这名非裔女议员激烈反对最高法院大法官提名人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 （Brett Kavanaugh）、支持指控卡瓦诺性侵的加州教授福特（Christine Blasey Ford）。
 <br/>
 <br/>
 国会警方指控，上周四卡瓦诺的听证会结束后，科斯科在网上公布了一些共和党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议员.html" target="_blank">
  参议员
 </a>
 的住址和电话，其中包括南卡罗来纳州参议员格雷厄姆（Lindsey Graham）、犹他州参议员哈奇（Orrin G. Hatch）和麦克‧李（Mike Lee）。
 <br/>
 <br/>
 根据国会警察队长贝尔（Jason Bell）的宣誓声明，本周二，一名证人看到科斯科在他以前工作过的一名参议员的办公室里操作电脑，而前一天，又有两名参议员的个人信息被公布在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维基百科.html" target="_blank">
  维基百科
 </a>
 上。
 <br/>
 <br/>
 科斯科曾为数名民主党参议员工作过，包括加州的范士丹（Dianne Feinstein）、新罕布什尔州的哈桑（Maggie Hassan）、前加州参议员波克塞（Barbara Boxer）等。
 <br/>
 <br/>
 消息人士告诉福克斯新闻，科斯科是在哈桑的办公室被抓的。当时他未经同意就进入他曾经工作过的地方并擅自使用电脑。哈桑的发言人称，哈桑强烈谴责科斯科的行为。
 <br/>
 <br/>
 贝尔说，科斯科在哈桑办公室遇到一名工作人员后就出去了。之后，那名工作人员通知了警察。几小时后，这名证人收到一封电邮，写信人威胁他说，如果他把这事告诉别人，自己将洩露一切，包括一些参议员的孩子的健康信息和社会安全号码。
 <br/>
 <br/>
 （记者李佳欣编译报导/责任编辑：东野）
