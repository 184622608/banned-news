### 美国中选：川普内华达助选 移民问题有对策
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）这周末，马不停蹄的到访西部几个城市，为共和党在中期选举中助阵。周六下午，川普总统，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内华达州.html" target="_blank">
  内华达州
 </a>
 埃尔科参加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竞选集会.html" target="_blank">
  竞选集会
 </a>
 。他在集会上表示，已经了解如何解决复杂的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民问题.html" target="_blank">
  移民问题
 </a>
 ，但没有提到具体细节。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内华达州.html" target="_blank">
  内华达州
 </a>
 埃尔科的共和党集会中批评民主党根本不理会非法移民给美国带来的伤害、给移民带来的伤害，并表示共和党将解决这个问题。
 <br/>
 <br/>
 川普总统：〝坦诚的说，我已经弄清楚（解决方法）。你们会很开心的。我们会让很多人开心。我们会让很多人开心。在选举之前，我想我会保持低调。我喜欢我们现在选举的势头。〞
 <br/>
 <br/>
 川普还表示美国人将在11月6日决定对国会的控制权。
 <br/>
 <br/>
 川普总统：〝你们的选票将决定谁将控制国会，每一个美国人的选择将再清楚不过了。〞
 <br/>
 <br/>
 川普为内华达州共和党参议员迪恩．海勒（Dean Heller）和亚利桑那州（Arizona）的两届共和党代表玛莎．麦克萨利（Martha McSally）助选，他鼓励选民们出来投票。
 <br/>
 <br/>
 川普总统：〝如果你想要拯救你的国家，如果你想保护你的家人，如果你想捍卫美国的法律，边界，主权甚至你的尊严。那你今天必须出去，投票。〞
 <br/>
 <br/>
 目前中期选举的注册选民，已经超过上次同一时期的两倍。有报道分析，川普的影响力，可能是促成这一现象的重要原因之一。
 <br/>
 <br/>
 新唐人记者严枫、雨薇综合报导
</div>
