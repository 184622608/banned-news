### 洛杉矶千橡市酒吧枪击案 烛光悼念遇难者
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】周四晚间，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洛杉矶.html" target="_blank">
  洛杉矶
 </a>
 2千多民众举行集会，悼念周三〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒吧枪击案.html" target="_blank">
  酒吧枪击案
 </a>
 〞的12位遇害者。联邦调查局搜查了枪手的住所，警方公布，嫌犯曾在美国海军陆战队服役，其作案动机还没有确认。
 <br/>
 <br/>
 周四晚，大批民众排队进入举行悼念活动的剧院，剧院内摆放着鲜花。人们拿着蜡烛，相互拥抱、哭泣。
 <br/>
 <br/>
 周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洛杉矶.html" target="_blank">
  洛杉矶
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千橡市.html" target="_blank">
  千橡市
 </a>
 一家酒吧正在举行〝大学乡村之夜〞活动时，发生枪击案，包括一名警察在内的12人遇害，另有十多人受伤。
 <br/>
 <br/>
 警方接到报案后，迅速赶到现场。文图拉县警长贺拉斯首先冲进酒吧前门，不幸被枪手击中，身中数枪，不幸殉职。贺拉斯从警29年，明年即将退休。
 <br/>
 <br/>
 集会时，民众起立鼓掌，表达对警察的敬意。
 <br/>
 <br/>
 千橡市市长安迪・福克斯表示，经历枪击案后，千橡市会〝更有爱心、更有同情心，更团结，更充满希望〞。
 <br/>
 <br/>
 千橡市市长安迪・福克斯：〝今晚我们是一座受伤的城市。但我们也是充满爱心、同情和团结的社区。我们是一个充满希望的社区。〞
 <br/>
 <br/>
 遇难的12人中，有当地大学生，还有去年拉斯维加斯枪击案的幸存者，让人倍感伤心。
 <br/>
 <br/>
 警方周三公布，枪击案的枪手是28岁的伊恩・朗，他是海军陆战队退伍军人。
 <br/>
 <br/>
 周四，美国联邦调查局检查了伊恩・朗的家，调查其犯案动机。
 <br/>
 <br/>
 嫌犯邻居表示，过去一年，伊恩・朗经常和母亲在家中激烈争吵。警方曾怀疑，伊恩・朗是否患有创伤后遗症。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
</div>
