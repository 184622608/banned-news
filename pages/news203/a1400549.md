### 川习会临近  川普：已做好充分准备
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】一周后，美国总统川普和习近平将再次会面。之前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC会议.html" target="_blank">
  APEC会议
 </a>
 上中美交锋激烈。另外，美国计划明年初增加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，中美贸易摩擦将升级。在这种局势下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 能取得什么成果，引发各界关注。
 <br/>
 <br/>
 11月30日到12月1日，二十国集团峰会将在阿根廷首都布宜诺斯艾利斯召开。期间，川普和习近平将谈论中美贸易等问题。
 <br/>
 <br/>
 川普周四表示，对于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 他已做好〝充分准备〞。
 <br/>
 <br/>
 美国总统川普:〝我已经准备好了，我毕生都在为会谈准备。我不需要坐在书桌前读书，我比任何人都了解任何内容，任何数据，我的直觉一直都是对的。我们做得很好，而且中国非常想和我们达成协议。〞
 <br/>
 <br/>
 为了让中共改变不公平贸易行为，今年美国已经对价值2500亿美元的中国商品加征
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 。
 <br/>
 <br/>
 北京对1100亿美国商品加征报复性关税。
 <br/>
 <br/>
 2019年1月1日，美国计划将对2000亿美元商品的关税从10%增加到25%。
 <br/>
 <br/>
 美国总统川普:〝我可以这样说，由于加征关税，中国（中共）非常希望达成协议。明年1月1日，10％的关税增加到25％。所以他们将支付巨额资金，这对我们国家来说是非常好的。〞
 <br/>
 <br/>
 据白宫调查，中共通过不公贸易行为窃取美国知识产权，给美国每年造成数千亿美元损失。
 <br/>
 <br/>
 川普政府今年5月初，向中方提出希望改善的八大要求，包括2020年前将对美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易顺差.html" target="_blank">
  贸易顺差
 </a>
 减少2000亿美元、取消贸易壁垒、终止工业补贴、停止窃取知识产权及强制技术转让等行为。
 <br/>
 <br/>
 中共上周向美国提出回应文件，不过川普表示，这虽然是〝一份非常完整的清单〞，但〝四五项很重要的要求，中方没有回应〞。
 <br/>
 <br/>
 周五，中共商业副部长王受文声称，要与美国进行〝平等互利〞的贸易谈判。
 <br/>
 <br/>
 新唐人记者王羽音、尚敬综合报导
</div>
