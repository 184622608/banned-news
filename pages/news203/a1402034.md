### 80余中国城市重污染预警 川普批评脏空气飘来美国了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 当前遭遇今年以来最严重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴霾.html" target="_blank">
  阴霾
 </a>
 天气，多达82个城市已发布了重污染预警。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）近日在谈及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空气污染.html" target="_blank">
  空气污染
 </a>
 问题时，批评中国空气污染严重，脏空气已经飘到了美国，让美国花费很多的钱。
 <br/>
 <br/>
 美媒报导说，11月30日，在阿根廷首都布宜诺斯艾利斯举办的二十国集团（G20）峰会间隙上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统接受了著名评论员，美媒特约撰稿人格莱塔・范・萨斯特伦的专访。
 <br/>
 <br/>
 采访中川普谈到了中美峰会、中美贸易等关键议题，当被问到气候问题时，川普说：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 在环境方面做得很好，拥有最纯净的空气和最洁净的水，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 和其他国家的脏空气飘到了美国，让美国花费很多的钱。〞
 <br/>
 <br/>
 川普还说，〝人们不想谈论这点，但是它飘移了过来。所以我们让空气清洁，但他们没有，而这花费很多的钱。〞
 <br/>
 <br/>
 美国政府11月23日发布了一份超过1000页的重要《气候评估报告》（National Climate Assessment）。报告中预估，因气候变暖而造成的数千亿美元损失。
 <br/>
 <br/>
 2014年9月，时任美国总统的奥巴马，就曾在联合国气候变化大会上，要求中国加入全球遏制气候变暖行动，在减少中国的温室气体排放上做出与美国一样的努力。
 <br/>
 <br/>
 奥巴马在演讲前表示，他认为作为世界最大的两个经济体和碳排放国，美国和中国对于遏制全球碳排放有责任起带头作用。
 <br/>
 <br/>
 他说：〝这是大国必须做的。〞
 <br/>
 <br/>
 由于中国对煤炭使用的增加，中国已经成为世界最大的温室气体排放者。由于中国空气严重污染，中国上百万孩子不得不在室内游戏，在过去几年，成千上万的中国人游行抗议中国对煤炭的依赖，认为是燃烧煤炭使中国空气成为全球最肮脏的空气。
 <br/>
 <br/>
 进入2018年，中共官方为了应对持续升高的中美贸易战火，8月份中共提出了〝6个城市需要在秋冬取暖季将钢铁产量减半和其它城市也需要将钢铁产量限制在至少30%之内〞的规定也已废弃。
 <br/>
 <br/>
 9月27日，中共又出台了《京津冀及周边地区2018-2019年秋冬季大气污染综合治理攻坚行动方案》，该方案提出，对钢铁、建材、铸造、有色、化工、焦化等高污染和高耗能行业的秋冬季停产限制取消，对环保标杆企业，更可以不限产。
 <br/>
 <br/>
 最近京津冀区域的企业频频发生爆燃事故，也说明了高污染行业，仍在正常生产。如11月30日，河南范县一家化工厂发生爆燃事故。11月7日，河北新乐化肥厂爆炸。11月28日，张家口发生爆燃事故。
 <br/>
 <br/>
 而上述相关规定还导致，近一段时间以来，80多个城市再度遭遇严重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴霾.html" target="_blank">
  阴霾
 </a>
 围城。
 <br/>
 <br/>
 四川维权与环保人士谭作人批评，中共政府为了转嫁中美贸易战的危机，不仅不考虑老百姓的利益，甚至不惜让老百姓承担环境污染所造成的后果，导致中国成为癌症第一大国。
 <br/>
 <br/>
 资料显示，中国每年因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空气污染.html" target="_blank">
  空气污染
 </a>
 死亡的人数达35.8万人，中国新增癌症病例和癌症死亡人数都高居世界首位。
 <br/>
 <br/>
 旅居德国的著名水利专家王维洛曾表示，就生态环境而言，中国式的发展是吃祖宗饭造子孙孽。这个交换是一个与魔鬼做的交易，中国失去了未来发展的潜力。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
