### 斥蒙州民主党政治暗钱 自由意志者弃竞选挺共和党对手
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】目前距美国中期选举仅剩5天，周三（10月31日），蒙大拿州选情突变，自由意志党（Libertarian）候选人布雷肯里奇（Rick Breckenridge）因对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 人使用政治〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暗钱.html" target="_blank">
  暗钱
 </a>
 〞的行径不满，宣布退出竞选并为其共和党对手背书。
 <br/>
 <br/>
 周三，蒙大拿州中期选举出现戏剧性变化，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自由意志者.html" target="_blank">
  自由意志者
 </a>
 布雷肯里奇公开批评他所收到的一封非法竞选邮件并宣布退出竞选。该邮件试图说服选民勿投票支持共和党参议员候选人罗森戴尔（Matt Rosendale），而支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 候选人泰斯特（Jon Tester）。
 <br/>
 <br/>
 目前，罗森戴尔与泰斯特正处于竞选热战之中，双方差距甚微。民调结果显示，罗森戴尔只在调查数据的误差范围内稍稍落后于自己的民主党对手，但最终结果胜负难料。
 <br/>
 <br/>
 布雷肯里奇没有机会在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蒙州.html" target="_blank">
  蒙州
 </a>
 选举中胜出，但他在选前最后关头所做出的决定却是蒙州政治史上前所未有的——放弃竞选转而支持自己的共和党对手，因为他反对政治〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暗钱.html" target="_blank">
  暗钱
 </a>
 〞（dark money）。
 <br/>
 <br/>
 促使布雷肯里奇做出这一决定的是一封来自未知组织的匿名竞选邮件，该信违反了蒙州及联邦竞选法，恶意诋毁罗森戴尔，力图把支持布雷肯里奇的选票拉给民主党人泰斯特、并破坏罗森戴尔的获胜机会。
 <br/>
 <br/>
 但是，事与愿违，该信反而把自由意志党的选票推向了共和党人。布雷肯里奇认为，这封信企图影响选举，因此他决意为罗森戴尔背书，尽管他并不赞成罗森戴尔的某些观点。
 <br/>
 <br/>
 布雷肯里奇告诉美联社（AP），估计他只能获得3%-4%的选票，但是，他希望能够帮助罗森戴尔，和他团结起来共同打击政治〝暗钱〞。
 <br/>
 <br/>
 泰斯特是现任民主党参议员，目前正竞选第三次连任。他被认为是不信守竞选承诺的国会议员，其批评者众多。
 <br/>
 <br/>
 此外，泰斯特还被指抹黑川普在今年3月底提名的退伍军人事务部长杰克逊（Ronny L. Jackson）。
 <br/>
 <br/>
 海军上将杰克逊曾先后担任小布殊（George W Bush）、奥巴马（Barack Obama）和川普3位总统的白宫医生，医术精湛，但因在今年1月的总统体检报告中敢于为川普说真话而招致民主党不满，左媒随之对其发起大肆攻击。
 <br/>
 <br/>
 在接受参议院审查期间，泰斯特声称，据传杰克逊工作时〝一再喝醉〞，而且他还发现杰克逊有个绰号叫〝Candyman〞，并用这个名字给病人提供不出具药方的药物，比如安眠类药物安必恩（Ambien，音译）等。
 <br/>
 <br/>
 杰克逊否认了上述指控，但是，在民主党的大肆攻击下，他不堪承受巨大的压力，最终决定退出提名。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
