### 刘强东性侵案：两度施暴仍获释 真正原因曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】中国京东集团创始人刘强东涉性侵案又有新消息。美媒披露，刘强东当晚实施强奸时，受害女生曾一度挣脱，但刚穿好衣服，又被刘拉回推倒。但该女生因为害怕一直不敢提出指控，直到刘获释后才决定诉诸法律。
 <br/>
 <br/>
 美国明尼苏达州明尼阿波利斯市的《星坛报》，11月2日披露了刘强东案受害女生的短信、部分证词以及其它文件，还原了当晚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东性侵.html" target="_blank">
  刘强东性侵
 </a>
 案发的过程。
 <br/>
 <br/>
 报导指，8月30日晚宴之后，受害女生刚走出餐厅，就被刘强东的司机拉进一辆黑色SUV，刘强东也上了车。刘强东的助理坐在前排，司机带他们去了南明尼阿波利斯的一处豪宅。受害女生下车后发现不对，拒绝进入豪宅，僵持一阵后，他们又回到了车内。
 <br/>
 <br/>
 她后来发信息跟朋友说，〝在车里，他开始对我动手动脚，我让他停下，他不听……〞
 <br/>
 <br/>
 大概晚上11点的时候，司机把他们送到女生的公寓。她下车后，刘强东也跟着下车，并告诉司机和助理一会就回来。女生打开公寓门后，刘强东跟了进去。
 <br/>
 <br/>
 受害女生告诉警方，刘强东进入公寓后，不顾她的反抗试图脱掉她的内衣，还把她推倒在床上。她挣扎很久，终于挣脱，并跑到客厅，穿上衣服。
 <br/>
 <br/>
 但是她说，〝最后，他还是把我推倒在床上，然后压在我身上，他很重。我想推开他，但是他太重了，然后……他就……〞
 <br/>
 <br/>
 〝刘强东在我床上，〞凌晨2点左右，她给朋友发短信称，〝他昨晚强行睡了我，我没办法逃跑，〞她又写道，〝我不是自愿的，我反抗了。〞
 <br/>
 <br/>
 朋友建议她报警，但受害女生害怕报复，〝他有钱有势，你别小看他，别报警。〞
 <br/>
 <br/>
 根据消息人士透露，朋友随后来到她的公寓，在外面报了警。但凌晨3点，警方到达公寓时，受害女生表示她不希望警方介入调查。
 <br/>
 <br/>
 因此，警方只是带走刘强东，并将其送回了酒店。
 <br/>
 <br/>
 天亮后，朋友劝说她去医院做性侵检查。大学博士课程的工作人员获悉性侵事件后，再次报警。当晚，他们在学校找到受害女生，后者讲述了事情经过。刘强东被等候的警方带走。
 <br/>
 <br/>
 但消息人士表示，受害女生那时仍旧十分担心害怕，不愿警方展开进一步调查。次日，刘强东获释。
 <br/>
 <br/>
 在刘强东获释当晚，经过朋友们进一步劝导，受害女生最终决定起诉刘强东，并重新联系了警方。第二天，刘强东返回中国。
 <br/>
 <br/>
 此前，外界曾就刘强东迅速获释做出各种解释。但据上述最新消息，受害女生因顾虑而不愿出面指控刘强东，或许是其中一个关键原因。
 <br/>
 <br/>
 据此前报导，21岁的受害女生是中国籍留学生，父亲是湖北富商。
 <br/>
 <br/>
 港媒指，案发后，中共外交部介入，逼迫女生回国。而刘强东回国后，曾在湖北省委高层陪同下高调露面，外界猜测或籍此向女方家族施压。
 <br/>
 <br/>
 据《星坛报》透露，亨内平县（Hennepin）检察官办公室目前正在评估，是否对刘强东提出性侵指控，但没有为何时做出决定设定时间表。
 <br/>
 <br/>
 据此前美媒报导，刘强东今年在明尼苏达大学参加博士课程期间，涉嫌在8月30日晚性侵该校女生，于8月31日晚间被警方拘捕，9月1日下午获释，9月2日返回中国。
 <br/>
 <br/>
 官方文件显示，刘强东被控明尼苏达州性犯罪五级中的一级重罪。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
