### 保护美国安全 川普促改出生即公民政策
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 去佛罗里达州助选前，接受媒体采访时表示，取消在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国出生.html" target="_blank">
  美国出生
 </a>
 的孩子自动获取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公民权.html" target="_blank">
  公民权
 </a>
 的政策很重要。这一计划被认为是中期选举前，川普政府在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民问题.html" target="_blank">
  移民问题
 </a>
 上的又一个有力举措。
 <br/>
 <br/>
 川普回答记者提问说，取消在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国出生.html" target="_blank">
  美国出生
 </a>
 即是公民的政策不需要修改宪法。
 <br/>
 <br/>
 美国总统川普：〝这不是修宪，不需要修宪，对美国出生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公民权.html" target="_blank">
  公民权
 </a>
 问题。我认为，可通过国会投票，或者我理解，和一些有才干的法律学者会面后，我认为可以颁布行政令。〞
 <br/>
 <br/>
 但考虑长期有效，川普更希望通过国会修改在美国出生即是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公民政策.html" target="_blank">
  公民政策
 </a>
 。
 <br/>
 <br/>
 美国总统川普：〝但我更愿意通过国会，因为这将永久有效。但我们肯定可以，我非常相信可以颁布行政令来做这件事。〞
 <br/>
 <br/>
 川普表示，前总统奥巴马以行政令执行〝达卡〞，自己也可以颁布行政令取消出生即是公民政策。
 <br/>
 <br/>
 美国总统川普：〝如果他（奥巴马）可以执行〝达卡〞，我们也可通过行政令做这件事。〞
 <br/>
 <br/>
 川普表示，取消出生即是公民政策，可以阻止更多非法移民进入。
 <br/>
 <br/>
 美国总统 川普：〝一个之前没来过美国的人，来这里生了孩子，这孩子立刻就成了美国公民，通过移民链或其他途径，许多人通过这个孩子都来到美国。这太荒谬了。〞
 <br/>
 <br/>
 川普还提到，前参议院少数党领袖、民主党人哈里・瑞德在1993年，就提出终止出生即是公民政策。
 <br/>
 <br/>
 前参议院少数党领袖 哈里・瑞德：〝没有一个理智的国家会这么做，是不是？一个人违反法律、未经许可进入我们国家，生了孩子，我们奖励这个孩子成为美国公民，保障他享受全套社会公共服务。〞
 <br/>
 <br/>
 川普之前表示，修改美国出生即是公民政策是保护美国安全，而对这一政策的法律解释将由最高法院来厘清。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
</div>
