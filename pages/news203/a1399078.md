### 佛州重新计票继续 斯科特提〝欺诈指控〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 结果
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 。共和党籍州长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯科特.html" target="_blank">
  斯科特
 </a>
 提起诉讼，指责有两个郡的选票存在欺诈，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统则认为该维持最初的选举结果。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 联邦参议员候选人，共和党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯科特.html" target="_blank">
  斯科特
 </a>
 （Rick Scott）与民主党的尼尔森（Bill Nelson）；州长候选人共和党德桑蒂斯（Ron DeSantis）与民主党吉卢姆（Andrew Gillum），以及该州农业厅长等选战。由于他们的选举结果差距都低于0.5％，因此佛州全部67个郡从上周六开始
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 ，而且必须在15日下午3点前得出结果并交给州务卿。
 <br/>
 <br/>
 重新计算的选票，一半都在布劳沃德（Broward）和棕榈滩（Palm Beach）两个郡，这里集中了很多民主党选民。
 <br/>
 <br/>
 这两个郡的选举监督机构由民主党控制。佛州州长斯科特（Rick Scott）对他们分别提起诉讼，指责他们在选举日截止时间后仍不断报告出新增的选票，他称其为〝肆无忌惮的欺诈〞。
 <br/>
 <br/>
 民主党否认〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欺诈指控.html" target="_blank">
  欺诈指控
 </a>
 〞，其佛州州长参选人吉卢姆并在重新计票后宣布，此前的〝认输宣言〞作废。目前，他比共和党德桑蒂斯只少33,700票。
 <br/>
 <br/>
 不过，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在推文中指出：〝佛州选举应该判定斯科特（Rick Scott）和德桑蒂斯（Ron DeSantis）获胜，因为无端冒出大量新选票，而且许多选票都已缺失甚至被伪造。一个可信的选票计算已不再可能，大批选票已不可靠。一定要维持最初结果！〞
 <br/>
 <br/>
 此外，乔治亚州州长选战与亚利桑那州联邦参议员选战，也都未得出最终结果。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
