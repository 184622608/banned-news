### 中期选举前夕 川普多州穿梭助选
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】11月5日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 前一天，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在中西部3个城市，一连举办了3场集会，为共和党候选人做最后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选冲刺.html" target="_blank">
  助选冲刺
 </a>
 。当地民众数万人参加集会，表达对川普的支持。
 <br/>
 <br/>
 在印第安纳州韦恩堡，一位建筑公司老板，连续参加了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在五个州的助选集会。
 <br/>
 <br/>
 建筑公司老板Labasse：〝我参加了五个州（竞选集会）〞
 <br/>
 <br/>
 建筑公司老板Labasse：〝（支持川普），因为我们想要更多工作，工资更高的工作，更安全。民主党人只想提高税收，开放边境。〞
 <br/>
 <br/>
 建筑公司老板Labasse：〝他（川普）赞成（边境）安全，反堕胎，信仰神。他为了你们的信仰自由，你可以选择（自己的）信仰，免遭迫害。他为了（民众的）高薪工作，他为了土地所有者，他为了（边境）安全。〞
 <br/>
 <br/>
 郡议会议员Tina Wenner：〝我来美国40多年，对我来说，川普是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最好的总统.html" target="_blank">
  最好的总统
 </a>
 。〞
 <br/>
 <br/>
 郡议会议员Tina Wenner：〝他不总是〝政治正确〞，因为他不是政客，他就是我们中的一员。〞
 <br/>
 <br/>
 在俄亥俄州克里夫兰市的世博中心，约有一万民众到现场支持川普。
 <br/>
 <br/>
 退休警察John Mott：〝他几乎兑现了他的每一个承诺，唯一没有兑现的承诺是由于国会不批准造成的。例如建墙。他要建墙，我支持建墙，很多美国人支持建墙，但国会，民主党人不让他建墙。〞
 <br/>
 <br/>
 退休钢铁厂工人 Mark Fedarko：〝我家是合法地从欧洲移民来的。非法越境过来的人，采取了错误的方法，是非法的，他们是罪犯。 其实，坐大篷车越境，就不是大篷车的问题了，这是入侵。 我们需要保护边界并将其关闭。 我们不知道谁在越过边界。我们终于有川普总统挺身而出，着手解决这个问题了。许多美国人已经无法忍受了。我们要阻止这种非法移民。〞
 <br/>
 <br/>
 在密苏里州，民众凌晨4点半就来排队等候入场。
 <br/>
 <br/>
 电台播音员 Trey Windhorst：〝他站在政局中，讲了美国人一直在思考的问题，我们终于没有了办公室里的政客。〞
 <br/>
 <br/>
 新唐人记者陈雷、婉莹、明光、陈虎、陈晓美国中部采访报导
</div>
