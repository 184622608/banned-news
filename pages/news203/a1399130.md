### 南加山火原因 疑爱迪生电力线路故障所致
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】北加州坎普山火死亡人数已增至42人，成为加州史上死亡人数最多的林火。而南加州伍尔西大火也已经夺走两条生命，超过25万人撤离家园。加州公共事业委员会（CPUC）认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伍尔西山火.html" target="_blank">
  伍尔西山火
 </a>
 起火原因很可能是爱迪生电力公司的电缆出了故障。
 <br/>
 <br/>
 据CBS报导，CPUC正在调查，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伍尔西山火.html" target="_blank">
  伍尔西山火
 </a>
 （Woolsey Fire）的起火原因是否是爱迪生电力公司（Southern California Edison，SCE）造成的。
 <br/>
 <br/>
 据CPUC称，爱迪生电力公司于上周四（8日）告诉CPUC，在伍尔西山火起火点附近的变电箱发生〝回路〞，检测到不寻常电流。2分钟后，伍尔西山火开始燃烧。
 <br/>
 <br/>
 CPUC发言人Terrie Prosper告诉城市新闻网，他们正与加州紧急事件办公室（Cal OES）、加州消防局（Cal Fire）一起密切关注火势的发展。
 <br/>
 <br/>
 Terrie Prosper说，CPUC将结合爱迪生电力公司的调查报告与他们自己的调查，来评估该公司的电力设备是否对本次山火造成影响。CPUC的调查包括对起火点、当地植被、电力设备的维护和紧急预备方案的检查。不过，目前调查人员仍无法进入山火起火点进行调查。
 <br/>
 <br/>
 就在1个多月前，南加爱迪生公司称，托马斯大火至少有一处起火点的点燃与该公司的设备有关。托马斯大火于去年12月4日在文图拉县和圣芭芭拉县爆发，造成28.2万英亩土地及1,063幢建筑被烧毁。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
