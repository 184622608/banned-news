### 盘点川习会六个敏感细节  中共不敢让百姓知道
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】美中首脑在阿根廷的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间举行会晤后，海内外媒体均对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 谈成果进行了报导，但是，其中有六个细节，因过于敏感，中共不敢让老百姓知道，官媒报导中有意忽略了。
 <br/>
 <br/>
 川普与习近平12月1日在阿根廷二十国集团（G20）峰会期间举行会谈，会谈结束后，中共很快通过央视发布消息称，美中两位领导人达成重要共识。约两个小时后，川普一行登上〝空军一号〞离开布宜诺斯艾利斯时，白宫新闻发言人桑德斯发布会谈结果声明。
 <br/>
 <br/>
 随后，中外媒体都对〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞的成果进行了报导，但报导内容明显不同。希望之声盘点了中共不敢让百姓知道六个细节，因过于敏感，被中共官媒在报导中有意忽略了。
 <br/>
 <br/>
 <h4>
  一，习近平首提与川普友谊
 </h4>
 <br/>
 中共官媒体习近平谈〝川习友谊〞的话没有报导。
 <br/>
 <br/>
 当地时间12月1日晚，习近平与川普在阿根廷共进工作晚宴的时候，习近平首次提及他与川普的个人友谊。
 <br/>
 <br/>
 习近平说，这次会晤体现了他和川普的个人友谊，〝很高兴同总统先生会晤。自上次见面以来，世界形势又出现了不少新变化。我愿利用今晚的机会，同总统先生就共同关心的问题交换看法，并规划好下一阶段美中关系。〞
 <br/>
 <br/>
 川普也说，他同习近平有着一段〝不可思议的友谊〞，〝这段关系非常特殊。我认为，我同习主席的关系，是我们最终可能会做一些对中国和美国都有利的事情的首要原因。〞
 <br/>
 <br/>
 《华尔街日报》报导说，在〝川习会〞晚宴前的谈判中，美方官员对中共谈判人员的表现很不满意。但是，当川普和习近平会面时，气氛立即发生了变化。
 <br/>
 <br/>
 <h4>
  二，川习会谈延长至少1小时
 </h4>
 <br/>
 川习会谈判持续2个半小时，比原订时间延长至少1小时，这个细节，中共官媒相关报导中没有提及。
 <br/>
 <br/>
 港媒称，这次晚宴比原计划延长了约一个小时，持续2个多小时。晚宴结束后，宴会厅内传出一阵掌声。会谈前剑拔弩张的气氛，被热烈的掌声代替，结局皆大欢喜，显示双方对会谈结果还算满意。
 <br/>
 <br/>
 2017年4月，习近平首次访美时，两人便一见如故，首次〝川习会〞原订两人谈两次，各谈15分钟，最后两人谈了5个小时；第二次，川普访问北京时，本来安排了25分钟的晚餐，持续了至少两个小时。
 <br/>
 <br/>
 当时西方媒体非常惊讶于川普对习近平表现出来的友善与宽容。纷纷以川普讲话中的那句〝我不怪中国了〞，以及川普对习近平的称赞〝过分热情〞等进行报导。
 <br/>
 <br/>
 而此次G20〝川习会〞结束后，川普更是称赞：〝这是一场惊人且富有成效的会议，为美国和中国提供了无限的可能性。我非常荣幸能与习主席合作。〞
 <br/>
 <br/>
 <h4>
  三，中共承诺进行结构性改革谈判
 </h4>
 <br/>
 中共外交部长王毅对此并没有具体提及，只是在记者会上说，中方将扩大从美国进口商品的规模，努力修正中美之间的贸易不平衡状况。双方同意相互开放市场，在中方推进新一轮改革开放中，使美方的合理关切得到逐步解决。
 <br/>
 <br/>
 而白宫声明称，川普总统和习近平同意立即就中国的结构性改革开始谈判，包括在强制技术转移、知识产权保护、非关税壁垒、网路攻击、服务以及农业等领域。
 <br/>
 <br/>
 这些正是美国的核心问题。中共损害美国的知识产权由来已久。美国知识产权被盗问题委员会发表一份报告称，仿冒、盗版以及盗窃商业机密等与知识产权相关的问题，每年给美国经济造成的损失在2250亿美元至6000亿美元之间。
 <br/>
 <br/>
 其中，仅盗窃商业机密一项就令美国经济损失1800亿到5400亿美元。中共是制造这些问题的〝元凶〞，美国没收的假冒商品87%来自中国。如今中共就此关键问题同美方谈判，这也是中共作出的重大让步。为了面子，中共不敢把这个实情告诉中国百姓。
 <br/>
 <br/>
 <h4>
  四，中美90天的停火限期
 </h4>
 <br/>
 〝川习会〞后，白宫声明称，中美双方约定90天的谈判期限，如果在90天后双方仍无法达成协议，美国将对中国2000亿美元商品的关税提高至25%。
 <br/>
 <br/>
 对于90日限期这一问题，中方通报中只字未提，只是称中方会〝使美方的合理关切得到逐步的解决〞。中共商务部副部长王受文对记者说，中美将加紧谈判取消现行的25%关税，但也没提到90日限期。
 <br/>
 <br/>
 对于美中双方达成暂时停火协议，有中国学者认为，这90天的谈判期限，形同北京被判三个月缓刑，正是因为这对中共是致命的，中共不愿让老百姓知道。
 <br/>
 <br/>
 <h4>
  五，美国高通并购案
 </h4>
 <br/>
 白宫声明称，习近平承诺，如果高通收购恩智浦的案件再次提交，他对批准这项此前未获批准的交易持开放态度。
 <br/>
 <br/>
 但中共高官在记者会上没有提及，只有一家中共国营媒体在中国社交平台微信上，援引了白宫声明提到的高通恩智浦收购案。
 <br/>
 <br/>
 由于高通和恩智浦在9个国家都占有较大市场，这起并购案必须向9个国家的反垄断机构申报批准。8个国家都已经同意的高通并购案，由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 爆发，中共报复美国而阻挠高通公司的这个并购案，因中共一家不同意该案泡汤。
 <br/>
 <br/>
 <h4>
  六，中国同意大量进口美国商品
 </h4>
 <br/>
 白宫声明表示，中方将〝立刻〞开始大量进口美国的农产品、能源以及工业制品，以减少美中两国之间的贸易不平衡。
 <br/>
 <br/>
 而中共通报中没有提及扩大进口的美国商品种类，以及何时开始进口等。中共官媒也没有提到具体购买美国的哪些商品，以及购买美国大量商品的具体时间。
 <br/>
 <br/>
 12月3日，在推特上有推友列出一份更详细的表格，让人一目了然的了解美中停火协议中，美中不同表述之处，高达10多项。
