### 呼吁消灭以色列 CNN评论员惹众怒遭解雇
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】周四（11月29日），美国有线电视新闻网（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN.html" target="_blank">
  CNN
 </a>
 ）宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_解雇.html" target="_blank">
  解雇
 </a>
 其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_评论员.html" target="_blank">
  评论员
 </a>
 希尔（Marc Lamont Hill），因其周三在联合国演讲时呼吁消灭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 并支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力抵抗.html" target="_blank">
  暴力抵抗
 </a>
 而引发了一场舆论风暴。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN.html" target="_blank">
  CNN
 </a>
 发言人周四表示，CNN已与希尔解除工作合同。
 <br/>
 <br/>
 希尔一直是名反犹太主义者。周三，他受邀在联合国举行的一个有关巴勒斯坦问题的会议上发言，但他在演讲过程中发表过激言论，主张巴勒斯坦人使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力抵抗.html" target="_blank">
  暴力抵抗
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 ，并敦促基层民众、当地和国际社会采取行动，建立起一个〝从河流到大海〞的自由的巴勒斯坦。
 <br/>
 <br/>
 〝从河流到大海〞是巴勒斯坦恐怖主义组织哈马斯（Hamas）已使用几十年的战斗口号，意思是要将巴勒斯坦国从约旦河扩展到地中海。
 <br/>
 <br/>
 这一主张其实就是号召摧毁以色列，因为以色列正好位于约旦河与地中海之间，而且拥有世界上44%的犹太人。
 <br/>
 <br/>

 <br/>
 希尔的言论引发了各界的尖锐批评。保守派利用社交媒体纷纷指责，国际性犹太人非政府组织及美国公民权利组织〝反诽谤联盟〞（Anti-Defamation League）、为纪念在二次世界大战中被纳粹杀害的犹太人而成立的国际人权组织〝西蒙．维森塔尔中心〞（Simon Wiesenthal Center）等都对希尔的演讲进行了强烈谴责，以色列官方也迅速做出回应。
 <br/>
 <br/>
 希尔是CNN的非裔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_评论员.html" target="_blank">
  评论员
 </a>
 ，同时也是费城天普大学（Temple University）的媒体研究和城市教育学教授，并曾在亚特兰大的莫尔豪斯学院（Morehouse College）和纽约的哥伦比亚大学（Columbia University）执教。
 <br/>
 <br/>
 希尔是一名自由派分析人士，2010年，他也曾被雇用他的福克斯新闻（Fox News）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_解雇.html" target="_blank">
  解雇
 </a>
 。
 <br/>
 <br/>
