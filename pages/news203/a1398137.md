### 大篷车移民将至 美边境拉铁丝网严阵以待
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】大批中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 不日将抵达美国南部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边界.html" target="_blank">
  边界
 </a>
 ，为了防止移民非法越境，目前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 已沿边界拉起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_铁丝网.html" target="_blank">
  铁丝网
 </a>
 ，以弥补边防人员及设备不足所带来的困境。
 <br/>
 <br/>
 据《纽约邮报》（New York Post）上周六（11月3日）报导，新到达的驻防
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 已沿美墨边境拉起带刺的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_铁丝网.html" target="_blank">
  铁丝网
 </a>
 ，以防止将至的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 非法潜入美国境内。
 <br/>
 <br/>
 五角大楼表示，已到达德州里奥-格兰德（Rio Grande）的美军正沿美国境内一侧的美墨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边界.html" target="_blank">
  边界
 </a>
 安设铁丝网，他们与美国海关和边境保护局（U.S. Customs and Border Protection）的巡逻人员合作，正在沿界河架设1,000英尺长的篱笆，这些临时障碍物安装在麦卡伦-伊达尔戈国际大桥（McAllen-Hidalgo International Bridge）下面。
 <br/>
 <br/>
 上周六，川普在出席蒙大拿州竞选集会时表示，他已看到那些拦阻非法移民的铁丝网，那景象〝很美〞。
 <br/>
 <br/>
 海关和边境保护局的一名官员在电邮中告诉《纽约邮报》，这些铁丝篱笆是针对大篷车移民所做的〝必要准备〞中的一部分。
 <br/>
 <br/>
 从10月下旬国防部开始部署驻防美军开始，至今已有大约900名军人抵达边境，并开始了先期工作。川普总统决意把大篷车移民阻挡在边界外，避免出现像在墨西哥南部那样强行突破边界的情况。
 <br/>
 <br/>
 美军此举实际上是弥补边境缺乏墙体防护的不得已措施。川普一再呼吁国会拨款建筑边境墙，但在民主党的阻挠下，资金远远不能满足需要。川普在给各共和党候选人背书时一再强调这一问题的重要性，并呼吁美国选民出来给共和党投票。
 <br/>
 <br/>
 截至周日，已进入墨西哥境内且继续北进的大篷车移民已高达约1,2000人。面对如此众多甚至还在增加的移民数量，即使川普派驻1,5000军队驰援边境的承诺已经兑现，也很难完全依靠人力看管漫长的边界线。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
