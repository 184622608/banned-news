### 打击中共经济间谍 塞申斯宣布新行动

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共经济间谍.html" target="_blank">
  中共经济间谍
 </a>
 活动。他还公布，一个联邦大陪审团起诉了中国公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 及其台湾合作伙伴联华电子，窃取美国公司的商业秘密。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国司法部.html" target="_blank">
  美国司法部
 </a>
 长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞申斯.html" target="_blank">
  塞申斯
 </a>
 周四批评中共违背承诺，持续对美国进行商业间谍活动。
 <br/>
 <br/>
 美国司法部长 塞申斯：〝中国（共）对美国的商业间谍活动一直在增长，而且是迅速增长。〞
 <br/>
 <br/>
 他说，美国不会再忍气吞声，现在是中共改变做法的时候了。
 <br/>
 <br/>
 美国司法部长 塞申斯：〝这是不能接受的。中国加入守法国家行列的时候到了。国际贸易对中国有利，但必须停止欺骗。〞
 <br/>
 <br/>
 塞申斯说，川普总统领导下，美国政府现在挺身应对中共的商业间谍活动。
 <br/>
 <br/>
 美国司法部长 塞申斯：〝我们不会让主权受到侵犯，不会让我们的知识产权被盗，不会让我们的人民被剥夺辛苦挣来的繁荣。我们希望在诚实交易的基础上实现公平贸易，建立良好关系。〞
 <br/>
 <br/>
 塞申斯还宣布，一个联邦大陪审团对中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 集成电路有限公司，及其台湾合作伙伴联华电子以及三名台湾人提出起诉，指控他们窃取美国美光公司)的商业秘密，让中共当局受益。
 <br/>
 <br/>
 美国司法部长 塞申斯：〝如果罪名成立，被告将面临高达15年的监禁，500万美元的罚款，公司可能被没收，还有超过200亿美元的罚款。〞
 <br/>
 <br/>
 塞申斯说，已下令启动一项新的行动计划，打击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共经济间谍.html" target="_blank">
  中共经济间谍
 </a>
 活动，同时应对中共在美国校园宣传等不断演变的新威胁。
 <br/>
 <br/>
 新唐人记者王羽音、张琪综合报导
