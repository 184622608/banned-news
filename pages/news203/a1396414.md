### 美大学生派对嗨翻 下一秒踩破地板酿30伤（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南卡罗来纳州.html" target="_blank">
  南卡罗来纳州
 </a>
 克莱门森大学的学生与居民，近日在公寓里开趴庆祝开学，大家跳到正嗨时，下一秒地板突然破裂，人群直接掉落到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地下室.html" target="_blank">
  地下室
 </a>
 ，造成至少30人受伤。
 <br/>
 <br/>
 从现场的视频中可见，一群
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大学生.html" target="_blank">
  大学生
 </a>
 正在屋内欢庆开学，下一秒，地板突然裂开了一个大洞，欢乐和庆祝的场面立刻充满恐慌和尖叫声。
 <br/>
 <br/>
 克莱门森警察局局长迪克森（Jimmy Dixon ）告诉WYFF，共有30人被送往3家地区医院，大多都是骨折和撕裂伤，所幸，没有人有生命危险。救援人员赶到出事公寓时，看到伤者们扶着自己骨折的手臂，许多人双腿都是血。
 <br/>
 <br/>
 当天参加派对的亚当斯（Jaylen Adams）表示，当人们正在欢庆时，他觉得地板像蹦床一样上下起伏，所以想和朋友到阳台上去，结果还没等他们离开地板就破了，所幸他没有掉下去。
 <br/>
 <br/>
 美联社报导，大二学生莱莉莎（Larissa Stone）回忆，意外发生时整个房间都挤满了人，而且当时正在播放一首大家都非常喜爱的歌曲，每个人都跟着音乐狂跳，节奏很快，下一秒地板却直接垮掉，不少人受伤。很多人甚至直接压在其他人身上，〝有人流血，我的鞋子上还沾到了血，真的很可怕。〞
 <br/>
 <br/>
 另一名学生提到，当下听到〝碰！〞的一声，尖叫声随之而来，没有掉进洞里面的人尽可能靠到边上站着，全都吓呆了。有的人还试图想要把下面的人拉起来。参加派对的雷洛伊（Leroy Pearson ）称，他当时想去帮忙那些受伤的人，看到他们似乎有人脚踝和腿都折断了，〝这真的是太疯了。〞
 <br/>
 <br/>
 屋主史兰（Tal Slann）称，出事房屋是2004年到2005年建成的，但没有表示房子内有限制人数。他也只知道公寓内有派对在举办，没多久地板就塌了，但没有人有生命危险。
 <br/>
 <br/>
 史兰说，房子所在的社区名为〝木头地〞（The Woodlands ），已经寄信告知所有居民发生了这样的意外。
 <br/>
 <br/>
 克莱门森大学表示，此次活动是由Kappa Alpha Psi兄弟会赞助的年度开学派对，由学生和非学生参加。在周日（21日）的一份声明中，该大学表示正在努力确定所涉及的克莱门森的学生，并通知他们就读的其他机构。
 <br/>
 <br/>
 警方正在进一步调查事发原因。
