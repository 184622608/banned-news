### 山火高温可熔铁 北加天堂小镇秒变恐怖炼狱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】美国加州近日发生的山火已导致逾2百人失踪，至少已造成31人死亡。加州布特郡警方日前对外透露，绝大多数罹难者遗骸，是被消防人员在被烧毁的房屋或汽车中找到的，这意味着这些不幸罹难的人，很可能是在正准备逃离的时候被大火所吞没。
 <br/>
 <br/>
 美国加州山火在大风助长下来势十分凶猛，火势蔓延得极其迅速。迄今为止，这场加州85年来最致命的火灾已造成至少31人死亡，其中有两名遇难者是在南加伍尔西大火（Woolsey Fire）中丧生的，而其余的29人则死于北加州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎普大火.html" target="_blank">
  坎普大火
 </a>
 (Camp Fire)，这场大火几乎把天堂城（City of Paradise）变为了人间炼狱。
 <br/>
 <br/>
 加州布特郡(Butte County) 警长哈尼亚(Kory Honea)11日晚间对媒体表示，大火已至少造成约230人失踪。而搜救团队在该地区已找到的29具遗体中，绝大多数死者是在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天堂镇.html" target="_blank">
  天堂镇
 </a>
 及附近地区严重烧毁房舍里被找到，或者在被大火烧焦的汽车旁被找到。
 <br/>
 <br/>
 据判断，有的罹难者可能是赶着上车的时候被浓烟呛昏而最终被大火吞噬。
 <br/>
 <br/>
 哈尼亚表示，现在警消人员面临的最大挑战就是寻找罹难者的遗骨。据他亲自上山查看的情况，许多地方已经烧到面目全非，很难确定断垣残壁底下到底有没有人体遗骸，由于大火温度非常高，连钢铁都被烧到熔化，遇难者的遗体有可能完全被烧光。
 <br/>
 <br/>
 据警方介绍，在部分案例中，警消人员所找到的罹难者遗骸已经烧到只剩碎骨，由于骨片非常细小，法医办公室调查人员必须使用网篮，慢慢筛选骨块。
 <br/>
 <br/>
 为了确认罹难者身分，一座DNA移动实验室已经进驻灾区，目前警方也在寻求法医专家(forensic anthropologists)的协助。
 <br/>
 <br/>
 有消息称，搜救人员11月9日下午在马里布穆赫兰（Mulholland）高速公路33000街区的一条住宅车道上，从一辆车里发现了两具严重烧伤的遗体。
