### 最后8小时跑3州后坐镇白宫 川普：我在选票上
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 周二（11月6日）投票，为刺激选情，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）总统周一（5日）接连出席了3场竞选集会，并向支持者喊话表示，〝某种程度而言，我在选票上〞。在以顶尖团队圆满结束竞选集会后，川普周二坐镇白宫密切关注投票结果。
 <br/>
 <br/>
 白宫新闻发言人桑德斯（Sarah Sanders）周二上午表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在连续出席8州的11场竞选集会及此前数周的助选后，当天将坐镇白宫，拨打电话监督各地选情。
 <br/>
 <br/>
 川普在选前最后一天，8小时内连跑3州出席造势活动，从俄亥俄、印第安纳到密苏里州，到处都是潮涌般欢迎川普的人群。
 <br/>
 <br/>
 据《华盛顿邮报》等媒体报导，在启程前，川普5日上午举行全国电话会议时向支持者说，〝某种程度而言，我在选票上。不管我们是否这么认为，媒体大多认为这是对我以及我们行动的公投。〞
 <br/>
 <br/>
 网路新闻媒体Buzzfeed写道，川普告诉支持者，〝明天的选举非常重要，因为这将总结我们所做的一切事情。〞
 <br/>
 <br/>
 川普还说，许多报导将这场选举视为对他的信任公投，他不知道是否如此，但这就是媒体操作的方式，如果共和党选情不佳，媒体会写成世界末日。而如果共和党选得好，媒体也不会归功于他们。
 <br/>
 <br/>
 川普5日并撰写专文刊登在《福克斯新闻》网站，呼吁选民投给共和党，继续现在的经济成就。川普强调，因为共和党的领导，美国经济表现强劲，工作机会增加、失业率不断下降，这一切都是因为共和党看重美国劳工与家庭。
 <br/>
 <br/>
 周一早上约6点40分，川普到达印第安纳韦恩堡（Fort Wayne）集会场，为竞选印州联邦参议员的共和党候选人麦克‧布朗（Mike Braun）助选，观众们举牌欢呼。在约1小时的演讲中，川普呼吁选民投票给共和党。在列举多项经济成绩包括失业率创新低、大规模税改后，川普说：〝当你周二进入投票站时，你面临一个简单的选择，投给共和党是继续我们现在的荣景，投给民主党就是让这个繁荣快速崩溃。〞
 <br/>
 <br/>
 〝民主党议程会带来社会主义的噩梦，共和党议程是让人实现美国梦。〞
 <br/>
 <br/>
 接着，在密苏里的演讲中，川普还邀请一个颇有人气的顶级助选团队登台，其中包括美国拥有听众最多的电台主持人林堡（Rush Limbaugh）、福克斯新闻主持人汉尼提（Sean Hannity）、葛来美奖得主格林伍德（Lee Greenwood）、白宫新闻发言人桑德斯、白宫顾问康威（Kellyanne Conway）、共和党全国委员会主席麦克丹尼尔（Ronna McDaniel）以及第一千金伊万卡（Ivanka Trump）等。
 <br/>
 <br/>
 川普在选民的热情欢呼中完成了他在选举前的最后一场演说，周二凌晨回到白宫并密切关注投票结果。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 各州的投票截止时间不同，印第安纳州以及肯塔基州将在美东时间18:00最早结束投票，最晚的是阿拉斯加，要到美东时间24:00才结束投票。多数州会在美东时间20:00结束投票。
 <br/>
 <br/>
 根据往届选举情况来看，各州计票结果通常在美东时间11月6日20:00左右相继出炉，媒体出口民调最早会在11月7日6:30左右出炉，最终结果将在11月7日午前明朗。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
