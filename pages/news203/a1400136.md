### 川普经济支持率再创新高 远超40年来所有美国总统
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】周一（11月19日）发布的一项最新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民调.html" target="_blank">
  民调
 </a>
 结果显示，美国民众对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济支持率.html" target="_blank">
  经济支持率
 </a>
 再创新高，以53%的比率远超近40年来所有前美国总统的平均经济支持率。
 <br/>
 <br/>
 周一公布的盖洛普
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民调.html" target="_blank">
  民调
 </a>
 （Gallup Poll）显示，53%的受访者对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 处理经济的方式感到满意，比8月份所创下的最高纪录又提高了3个百分点，再创历史新高。
 <br/>
 <br/>
 盖洛普民调指出，从里根（Ronald Reagan）时期到奥巴马（Barack Obama）执政，所有前美国总统获得的平均
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济支持率.html" target="_blank">
  经济支持率
 </a>
 只有45%，川普显然远远超出了这项纪录。
 <br/>
 <br/>
 川普通常把密切关系民生的失业率和新增工作岗位数量作为衡量经济状况的重要指标。
 <br/>
 <br/>
 根据美国劳工部11月2日公布的报告，美国10月份新增工作岗位250,000个，失业率维持9月份所创下的49年来的最低点——3.7%，同时，工资增长率也较大幅度地提升，多年来首次突破了3%。
 <br/>
 <br/>
 该盖洛普民调是在11月1日至11日期间对1,037 名受访者进行调查的结果，时间段跨越中期选举前后，误差幅度不超过4%。
 <br/>
 <br/>
 周二，里根总统的前经济顾问拉弗（Art Laffer）在接受福克斯新闻（Fox News）《美国新闻室》（America’s Newsroom）节目采访时表示，川普政府在管理经济的过程中，〝几乎每件事〞（almost everything）都做对了。
 <br/>
 <br/>
 拉弗特别赞扬了川普改革税法、废除监管法规、任命鲍威尔（Jerome Powell）为美联储主席等决定，并表示川普并不需要在经济方面再多做什么，因为良好的机制已经形成了。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
