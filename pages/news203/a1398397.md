### 半岛无核化生变？蓬佩奥与金英哲会谈延后
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】美国国务院周三（7日）宣布，国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 与朝鲜（北韩）第2号人物金英哲原定本周在纽约举行的会谈延后。国务院并未进一步透露其他细节。
 <br/>
 <br/>
 美国国务院周三在声明中还说，等到〝双方时间允许时〞将重新安排会面，且既有的对话仍将持续进行。
 <br/>
 <br/>
 声明称，美国仍将致力落实川普（特朗普）总统与朝鲜领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 ，今年6月在新加坡峰会上商定的承诺。
 <br/>
 <br/>
 路透社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 （Mike Pompeo）原本希望与金英哲举行会谈，为〝川金二会〞铺路，并持续推动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜半岛.html" target="_blank">
  朝鲜半岛
 </a>
 非核化。
 <br/>
 <br/>
 美国国务院本周曾表示，两人会晤的议程是〝讨论如何就新加坡峰会联合声明的4大要点取得进展，包括朝鲜落实最终、全面且可验证的非核化。〞
 <br/>
 <br/>
 川普今年6月在新加坡与金正恩举行峰会，双方签署了一项概况声明，要求朝鲜半岛实现去核化。但在此后的几个月中，朝鲜终止核武器计划的进展有所放缓。
 <br/>
 <br/>
 蓬佩奥上周五（11月2日）他在《福克斯电视台》上说，在他与金正恩的会谈中，金正恩明确表示愿意去核。
 <br/>
 <br/>
 但蓬佩奥也表示，对朝鲜去核的核查至关重要。他说：〝我们必须确保做到这一点。我们不能仅仅听任何人说的话。还有大量工作，但是我相信，我们将维持经济压力，直到金委员长履行了他今年6月在新加坡对川普总统做出的承诺。〞
 <br/>
 <br/>
 对此，朝鲜外交部在当天发布声明扬言，如果美国不解除经济制裁，朝鲜将考虑恢复其原来的政策，同时发展核武器和经济。
 <br/>
 <br/>
 这之前朝鲜外交部美国研究所所长权正根也在朝鲜官媒《朝中社》发表评论，并暗示将重启经济与核武的并进路线。
 <br/>
 <br/>
 蓬佩奥周日（11月4日）回应说，〝我对此（威胁）并不担忧〞，〝我们的重点十分明确。我们清楚我们的谈判对手，清楚我们的谈判立场。川普总统十分清晰地阐述了他的立场。〞
 <br/>
 <br/>
 蓬佩奥说，美国不只是要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无核化.html" target="_blank">
  无核化
 </a>
 ，还要经过美方的验证，然后才能商讨解除制裁的问题。
 <br/>
 <br/>
 美国主张，作为无核化的第一步，朝鲜需要先提交核武器和核设施等的清单，以及废除这些设施和武器的时间表，终战宣言也只会在弃核之后宣布。而朝鲜要求先发布朝鲜战争终战宣言，再解除对朝制裁，然后才愿意提交清单和弃核时间表。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
