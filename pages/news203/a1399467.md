### 佛州法官废签名匹配法 更多选票流入民主党
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 周四（11月15日）裁决，选举官员须在11月17日之前纠正数千张签名有问题的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选票.html" target="_blank">
  选票
 </a>
 。这项裁决显然有利于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 候选人，而佛州两名共和党候选人的胜利将面临更大挑战。
 <br/>
 <br/>
 这项裁决是由前总统奥巴马任命的联邦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 沃克（Mark Walker）做出的。周四上午，他做出这一裁决，支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 人的诉求。
 <br/>
 <br/>
 但是，按照
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 法律，邮寄
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选票.html" target="_blank">
  选票
 </a>
 上的签名如果与选民所登记的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_签名不符.html" target="_blank">
  签名不符
 </a>
 ，应该废弃不计，以保证选票的真实性、有效性。
 <br/>
 <br/>
 沃克的裁决显然推翻了佛州的原有法律规定，使选举条例向着有利于民主党候选人的方向倾斜。
 <br/>
 <br/>
 一直在密切关注计票过程的佛州共和党参议员卢比奥（Marco Rubio）周二晚揭露，民主党律师已提请法官修改佛州法律，企图以更改原有选举规则的方式使选票流向民主党候选人。
 <br/>
 <br/>
 卢比奥指出，佛州法律规定，邮寄选票上的选民签名应该与选民本人的签名相符，可是民主党律师却要求法官抛弃这条法律，而命令佛州选举部门把不符合这一规定的选票也计算在内。
 <br/>
 <br/>
 但卢比奥警告和担心的事情最终还是发生了。自由派法官沃克周四命令州务卿戴兹特（Ken Detzner），要求他通知各县选举官员，佛州这项关于〝签名匹配〞的法律过去是被〝违宪应用〞了。
 <br/>
 <br/>
 沃克要求，选举官员必须给选民提供给多时间纠正签名错误，截止时间为周六晚5点。
 <br/>
 <br/>
 这项诉讼是由在初步计票结果中败北的佛州民主党参议员候选人尼尔森（Bill Nelson）提起的，旨在通过计算不合格选票来增加自己的获胜机会。
 <br/>
 <br/>
 沃克称，在佛州67个县中，有45县拒绝了总共大约4,000张签名不符的选票。
 <br/>
 <br/>
 虽然这几千张选票不足以弥补尼尔森与其共和党对手、现任州长斯科特（Rick Scott）之间的差距，但这一裁决将会进一步削减斯科特的优势，给重新计票结果增加更大变数。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
