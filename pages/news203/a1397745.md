### 美起诉晋华联电窃取美光机密 价值近90亿美元
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】周四，美国联邦大陪审团对中国福建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晋华.html" target="_blank">
  晋华
 </a>
 集成电路有限公司和台湾联电电子股份有限公司以及三名台湾人提出了起诉，指控这两家公司及同谋者从事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济间谍活动.html" target="_blank">
  经济间谍活动
 </a>
 ，盗窃了美国美光公司的商业机密。据指，美光被窃的技术资料价值高达87.5亿美元。
 <br/>
 <br/>
 据《美国之音》报导，美国司法部长杰夫．塞申斯（Jeff Sessions）11月1日宣布了上述起诉的消息。他强调说：〝这起以及最近其它案件显示，中国增加了针对美国的间谍活动，而且一直在迅速增加。我在这里要说‘够了’。司法部将本着操守和专业精神，大力起诉这类非法活动。〞
 <br/>
 <br/>
 塞申斯进一步表示，在这次共谋盗窃美光技术之前，中国企业并没有掌握动态随机存储器(DRAM)技术。而联电与福建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晋华.html" target="_blank">
  晋华
 </a>
 合伙，好让中国能够从美国企业窃得这种技术，用来和美国竞争。他直言：〝这是厚颜无耻的阴谋。〞
 <br/>
 <br/>
 据了解，这份起诉书是2018年9月27日向法院提交的，美国司法部11月1日对外公布。
 <br/>
 <br/>
 根据司法部公开的起诉书，被告之一台湾人陈正坤，原系美光公司在台湾的子公司美光记忆体(MMT)的总裁，而MMT负责为美光公司制造至少一种DRAM芯片。
 <br/>
 <br/>
 2015年陈正坤从MMT辞职后，立即开始为联电工作。在陈的安排下，联电与福建晋华合作，由福建晋华提供资金，联电将DRAM技术转移给福建晋华进行量产，技术由两家公司共享。陈后来成为福建晋华的总裁，并负责其DRAM生产设施。
 <br/>
 <br/>
 此后，陈正坤又从美企MMT挖走了不少雇员，其中就包括本案的另外两名被告——台湾人何健庭(音译)和王永明(音译)，而何、王二人在离开MMT之前，都盗窃了几项与DRAM设计和生产有关的商业秘密并交给联电。其中，王某盗窃了美光机密和专有文档900多份，把这些资料存在USB外盘或个人云存储内。
 <br/>
 <br/>
 另据台湾《中央社》报导，美光科技公司被盗的上述商业机密，估计价值达87.5亿美元。
 <br/>
 <br/>
 依照美国法律，如果被判罪名成立，三名台湾人被告将因经济间谍罪名而面临最长15年的监禁及5百万美元的罚款，因窃取商业秘密罪名面临10年监禁。如果晋华公司和联电公司被判有罪，则每家公司将面临被没收非法所得的处罚，外加最高逾200亿美元的罚款。
 <br/>
 <br/>
 联邦调查局局长克里斯托弗．雷说：〝没有任何国家比中国对我们的观念、我们的创新和我们的经济安全构成更广泛、更严重的威胁了。中国政府执意要获取美国技术，他们不惜动用各种手段，从境外投资、收购公司和网络入侵到获取公司现任或前任雇员的服务以掌握内部信息。如果中国获得了一家美国公司最重要的技术—这种使其成为行业龙头的技术- –该公司将蒙受重大损失，我们的国家安全甚至也有可能受到影响。我们致力于与我们的联邦、州、地方和私营部门的合作伙伴密切合作，抗衡来自中国的这一威胁。〞
 <br/>
 <br/>
 司法部除了宣布上述刑事起诉外，还针对这起盗窃美企商业机密事件提起了民事诉讼，申请禁制令，阻止联电和福建晋华将使用美光公司（Micron）失窃技术所生产的产品输往美国。
 <br/>
 <br/>
 此前，福建晋华公司曾于今年1月向福建当地的法院起诉美光侵权，并最终打赢了这场官司，福建的法院下令美光停止在中国销售十余种固态硬碟、记忆卡和晶片。
 <br/>
 <br/>
 路透社报导称，美国正扩大反制中国对美国企业的间谍活动。对此，塞申斯声明说，鉴于中国（中共）的间谍活动快速增加，美国政府将采取新的主动行动反制。
 <br/>
 <br/>
 美国联邦调查局（FBI）局长瑞伊（Christopher Wray）也发布声明说：〝中国（中共政府）对我们的创意、创新和经济安全构成广泛而严重的威胁，没有其他国家比得上。〞
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：戴明）
