### 美中期选举临近 民主党高层担忧〝蓝波〞逆转
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】随着美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 临近，以自由派为主的各种民意测验显示，民主党似乎将控制众议院。但一些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党高层.html" target="_blank">
  民主党高层
 </a>
 人士周一（10月22日）警告，不要过于相信
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民调.html" target="_blank">
  民调
 </a>
 结果，他们担忧，今年的国会选举会再次出现像2016年总统大选那样令人始料不及的结果。
 <br/>
 <br/>
 据福克斯新闻（Fox News）22日报导，现距离11月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 还有两周时间，虽然众议院少数党领袖佩洛西（Nancy Pelosi）几天前宣称，即将出现民主党〝海啸〞，但民主党全国委员会（DNC）主席汤姆．佩雷斯（Tom Perez）周一表示，他不用〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓝波.html" target="_blank">
  蓝波
 </a>
 〞一词谈论民主党可能会取得的胜利。
 <br/>
 <br/>
 佩雷斯告诉美国有线电视新闻网（CNN），〝我们一直都知道这次选举将会很接近——我不使用‘蓝波’一词〞，而只是谈论组织的必要性。
 <br/>
 <br/>
 不过，作为民主党领袖，佩雷斯仍然对民主党能够掌控众议院充满信心。另外，他相信在一些州长的选举中，民主党也会取得胜利。
 <br/>
 <br/>
 随着选举日渐临近，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民调.html" target="_blank">
  民调
 </a>
 显示，两党支持率之差也在逐渐缩小，尤其是近来川普频繁出席各地的竞选集会并为各州共和党候选人背书，使得美国注册选民的人数远远高于往届同期水平。
 <br/>
 <br/>
 佛蒙特州资深民主党参议员桑德斯（Bernie Sanders）甚至对〝蓝波〞之说提出更多质疑。
 <br/>
 <br/>
 周一，桑德斯在接受《国会山报》（The Hill）采访时表示，〝我知道很多人都在谈论这股‘蓝波’及所有这类的东西，但我不相信。〞
 <br/>
 <br/>
 桑德斯是2016年总统大选民主党党内初选时希拉里（Hillary Clinton）最有力的竞争对手，但在希拉里的不公平排挤下败北。
 <br/>
 <br/>
 桑德斯认为，选举当晚，人们会发现双方情况非常接近，而且，可能会有一些选票最终决定民主党能否控制众议院。
 <br/>
 <br/>
 目前的民调倾向于共和党仍控制参议院，但民主党在众议院竞选中占据优势。但民调的预测在2016年大选中曾遭到重创，因此，〝蓝波〞之说很可能是自由派媒体再次为民主党制造的声势。
 <br/>
 <br/>
 共和党全国委员会（RNC）主席麦克丹尼尔（Ronna McDaniel）周一在接受福克斯新闻《美国新闻室》（America's Newsroom）节目采访时表示，她认为共和党将控制参议院，而且可能也将继续控制众议院，不过这取决于选民的投票结果。
 <br/>
 <br/>
 根据华尔街日报/ NBC新闻（Wall Street Journal/NBC News）民调上周末发布的报告，目前48%的登记选民支持民主党赢得众议院控制权，而支持共和党的只有41%，双方相差7个百分点。但该民调9月份的调查显示，当时民主党的支持率领先共和党12个百分点，显然两者差距正在缩小。
 <br/>
 <br/>
 但是，普遍认为比较客观的盖洛普民调（Gallup Poll）9月24日公布的调查结果显示，共和党的支持率上升到45%，已超出民主党1%，达到其7年来最高点，比其去年9月的36％增加了9个百分点。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
