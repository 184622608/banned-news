### 【禁闻】川普拟废止落地国籍 评：剑指中共
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】最近，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 计划签署行政命令，停止给予非美国公民和非法移民在美国所生孩子的公民身分。川普还特别谈及，那些美国敌人和独裁者在美国生的孩子，怎么能自动成为美国公民呢？这被一些观察人士解读为：剑指中共。请看报导。
 <br/>
 <br/>
 日前，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在密苏里州参加中期选举的助选活动时，再次谈及非法移民等问题。他表示，赴美产子已经形成了一个巨大产业。他特别点名中国大陆，说：目前来自中国大陆的赴美产子人数最多。
 <br/>
 <br/>
 川普认为，美国现有的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出生公民权.html" target="_blank">
  出生公民权
 </a>
 〞政策荒谬，必须停止。他特别举例说：〝你是我国（美国）的敌人，你是一个想要发动战争的将军，你是一个我们痛恨并反对我们的独裁者。那个独裁者的妻子在美国的土地上生了孩子。恭喜你，你的儿女现在是美国公民了。有人会认为这合理吗？！〞
 <br/>
 <br/>
 川普的这番话引起关注。有人认为，川普似有剑指中共之意，间接讽刺了中共高官把妻子、儿女变成美国公民。川普所言的独裁者、敌人、战争挑衅者，应该就包括中共官员。
 <br/>
 <br/>
 也有网民表示，川普再一次强调了法律和规则。他提出，当规则被恶人乱用的时候，是否要改变规则。
 <br/>
 <br/>
 早在川普竞选美国总统之时就曾表示，将强力整顿非法移民给美国社会带来的各种问题，包括废除外国人在美国出生即自动取得美国公民身分的政策，也就是所谓的〝出生公民权〞、又称〝落地国籍〞政策。
 <br/>
 <br/>
 按照美国现有法律，不管
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_婴儿.html" target="_blank">
  婴儿
 </a>
 的父母是合法居民还是非法移民、游客，在美国境内出生的婴儿将自动成为美国公民。
 <br/>
 <br/>
 川普上台后，一直致力于收紧移民政策，打击非法移民。10月29号，川普在接受美国媒体AXIOS采访时说，他计划签署一项行政命令，取消落地国籍。
 <br/>
 <br/>
 网路自由观察人士古河：〝这个方式其实是一种釜底抽薪的做法。中共的高官天天口口声声要‘解放’台湾，要对美国进行颠覆计画，把美国搞乱搞倒，那么现在把你（在美国出生）的子女统统取消美国国籍，别在美国卧底。〞
 <br/>
 <br/>
 网路自由观察人士古河表示，川普所要发布的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政令.html" target="_blank">
  行政令
 </a>
 所针对的对象包括中共高官，这也从一个侧面表明美国对中共几十年来的绥靖政策彻底终结。
 <br/>
 <br/>
 一般而言，美国的〝出生公民权〞政策的依据是美国1868年宪法的第14条修正案:〝所有在美国出生或在美国归化，并受美国司法管辖的人，都是美国公民以及其所居住州的居民，任何州都不能制定或执行任何削弱美国公民权利或豁免权的法律。〞
 <br/>
 <br/>
 因此，一些人认为，川普要改变这一政策，首先就要面临法律挑战，需要修改宪法修正案。但也有一些人认为，目前的政策曲解了宪法修正案的内容，修正案强调〝受美国司法管辖〞，也就是只有〝那些对美国有充分政治效忠的人〞，包括绿卡持有者和公民才适用这一修正案。
 <br/>
 <br/>
 川普对媒体表示，可以通过国会修改法案来改变〝出生公民权〞政策，但他同时也谘询了法律顾问，被告知总统可以通过签署行政令来改变这一政策。
 <br/>
 <br/>
 美国华府人权律师叶宁：〝（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赴美生子.html" target="_blank">
  赴美生子
 </a>
 ）生下来的小孩从行政命令生效以后，这个小孩要去申请美国护照可能就申请不到。〞
 <br/>
 <br/>
 《大纪元》新闻网报导，如果川普的这个行政令能够落实，将成为美国移民政策改革中的重大事件，也将阻止中国人赴美产子的浪潮。
 <br/>
 <br/>
 据不完全统计，10年之间，大陆赴美生子人数翻了上百倍不止。2007年，大陆每年赴美生子的人数只有几百人，2010年升至5,000人，到2016年则超过8万人次，可谓数量惊人。
 <br/>
 <br/>
 采访/陈汉 编辑/李谦 后制/周天
