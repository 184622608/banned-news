### 加州枪击案 警察火拼枪手殉职 民众夹道致敬
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】美国南加州千橡镇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_音乐吧.html" target="_blank">
  音乐吧
 </a>
 7日深夜发生枪击案，造成至少13人死21人受伤。第一时间赶赴现场救人的警官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赫卢斯.html" target="_blank">
  赫卢斯
 </a>
 ，以身挡子弹不幸殉职，周四（8日）他的遗体送往法医部门时，民众沿途向其致敬。
 <br/>
 <br/>
 〝橘郡纪事报〞（The Orange County）报导，周四上午
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赫卢斯.html" target="_blank">
  赫卢斯
 </a>
 （Ron Helus）的遗体送往文杜拉郡法医署，车队行经101号高速公路，高架桥上站满了民众与执法人员，向赫卢斯的英勇行为致敬。

 〝洛杉矶时报〞（Los Angeles Times）报导，周三晚11时20分夜店传出枪响之后，赫卢斯与一名加州高速公路巡警最早抵达现场，4分钟之内就从前门进入并与歹徒交火。
 <br/>
 <br/>
 后援警力到达时，赫卢斯已经身中数枪，送到当地医院之后不治身亡。15分钟后第2批警力到达时，枪击屠杀已画上句号。警方在酒吧入口附近发现了枪手，28岁的前海军陆战队员大卫．龙（David Long）的尸体。据信他是作案后自戕而亡。

 文杜拉郡（Ventura County）警长狄恩（Geoff Dean）表示，赫卢斯冲进枪击现场前正在与妻子通电话，他告诉妻子说：〝亲爱的，我要处理一通电话，我必须挂断电话了，我爱你。〞
 <br/>
 <br/>
 狄恩说：〝他们知道必须采取行动，他们进入现场，而且执行了该做的任务。〞狄恩形容，赫卢斯是一名勤奋、敬业的警官，为了工作付出一切，他以英雄之姿死去，拯救了其他人的性命。
 <br/>
 <br/>
 文杜拉郡警局队长库瑞杰安（Garo Kuredjian）说，枪击案发生之后，一名年轻人特地走向他道谢，〝毫无疑问，你们这名警官救了其他人，使更多人免于丧命。〞
 <br/>
 <br/>
 库瑞杰安说：〝对家属来说，稍微引以慰藉的是知道他们的父亲、丈夫冲入险境，拯救生命。〞

 洛杉矶电视主播在连线的时候形容，赫卢斯的英勇行为，让酒吧里许多人躲过这场枪击案，平安回家，但他却再也回不了家。
 <br/>
 <br/>
 赫卢斯服务警界29年，原本计划明年退休。他与妻子育有一儿，根据社群网站上的照片，他与儿子最喜欢从事的活动是钓鱼。
 <br/>
 <br/>
 枪击案造成13人死亡（包括赫卢斯和枪手），另有12人受伤。
