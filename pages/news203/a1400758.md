### 川普再警告关闭边境 墨否认达成协议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】周末，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统再次警告，如大篷车移民难以控制，他将关闭美国南部入境口岸。川普指责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 及中美洲国家在移民问题上做法不当，但墨西哥方面则
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_否认.html" target="_blank">
  否认
 </a>
 已与美国达成容留移民的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_协议.html" target="_blank">
  协议
 </a>
 。
 <br/>
 <br/>
 周六（11月24日）晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 发推强调，在美墨边境递交庇护申请的中美洲移民，在案件审理过程中不允许进入美国。
 <br/>
 <br/>
 〝南部边境的移民在其庇护申诉得到批准前，不得进入美国。我们只接受合法进入我国的人。除此之外，我们还有非常强大的政策——抓着就扣。不会‘释放’到美国⋯⋯〞
 <br/>
 <br/>

 </a>
 。如果出于任何原因有必要的话，我们将关闭我们的南部边界。在经过数十年伤害之后，让美国再忍受这种代价高昂且危险的局面，没门儿！〞
 <br/>
 <br/>
 <blockquote class="twitter-tweet" data-lang="en">

 </blockquote>
 <br/>
 <div style="clear:both;display:block;">
 </div>
 <br/>
 这是川普近日第二次公开表示，他可能因大篷车问题而关闭美墨边界。
 <br/>
 <br/>
 感恩节（周四）上午，川普告诉记者，如果墨西哥当局不能控制其境内的大篷车移民，他将全线关闭美墨边境，直到情况恢复正常为止。
 <br/>
 <br/>
 川普同时警告，如果边境口岸关闭，墨西哥将无法向美国出售汽车，其经济利益也必将遭受损失。
 <br/>
 <br/>
 据《华盛顿邮报》（The Washington Post）周六报导，即将上任的墨西哥新政府内政部长科尔德罗（Olga Sánchez Cordero）告诉该媒体，他们已同意美方提出的〝留在墨西哥〞（Remain in Mexico）计划，允许向美国申请庇护的大篷车移民在墨西哥境内等待审批。
 <br/>
 <br/>
 但周六晚，科尔德罗发表声明说，新一届墨西哥政府并未与美国达成任何
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_协议.html" target="_blank">
  协议
 </a>
 。
 <br/>
 <br/>
 周日清晨，川普在推文中表示，如果墨西哥早在大篷车移民到达美墨边境前就阻止他们，或者这些中美洲移民的来源国不让这些移民大军形成，那才是非常聪明的做法。
 <br/>
 <br/>
 川普指出，这是这些中美洲国家所采用的方式，即让某些人离开他们国家并倾入美国。但美国不会再容忍这种情况继续下去了。
 <br/>
 <br/>
 川普再次批评，是民主党制造出的这一问题。他重申，他不许大篷车移民过境。
 <br/>
