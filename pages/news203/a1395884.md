### 英勇抵抗越共 80岁越战老兵获川普授勋
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】周三，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 向前海军陆战队员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_约翰・坎利.html" target="_blank">
  约翰・坎利
 </a>
 授予军方最高荣誉——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国会勋章.html" target="_blank">
  国会勋章
 </a>
 ，表彰他在越战的英勇行为。
 <br/>
 <br/>
 美国总统川普：〝我非常荣幸，向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_约翰・坎利.html" target="_blank">
  约翰・坎利
 </a>
 中士颁发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国会勋章.html" target="_blank">
  国会勋章
 </a>
 。〞
 <br/>
 <br/>
 约翰・坎利现年80岁，出生于阿肯色州，是第300位获国会勋章的海军将士。
 <br/>
 <br/>
 川普盛赞约翰・坎利，在越战最血腥的战役——〝顺化战役〞中，表现英勇。
 <br/>
 <br/>
 美国总统川普：〝约翰经过七天残酷的战斗，挽救了20多名海军陆战队员生命。〞
 <br/>
 <br/>
 1968年，北越共产党人违反〝休战条约〞，在全国发动袭击，占领美军重要据点顺化市。
 <br/>
 <br/>
 约翰・坎利率领150人，对抗6000多名越共。战斗中，他不顾自己受重伤，多次救出战友。
 <br/>
 <br/>
 越战后，约翰・坎利继续服役，训练了成千上万海军陆战队员。川普说，战友称赞约翰・坎利〝超越了死亡〞。
 <br/>
 <br/>
 美国总统 川普：〝美国成为世界上捍卫和平、正义和自由的最伟大力量，正是因为有你和像你一样的人。〞
 <br/>
 <br/>
 30多名曾参与顺化战役的前海军陆战队员来到白宫，见证授勋仪式。
 <br/>
 <br/>
 新唐人记者毕心慈、张琪综合报导
</div>
