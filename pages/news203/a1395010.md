### 受邀入白宫赴宴 威斯特:戴川普帽感觉如超人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】周四（10月11日），美国饶舌歌手肯伊．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威斯特.html" target="_blank">
  威斯特
 </a>
 （Kanye West）赴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 与川普总统共进
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_午餐.html" target="_blank">
  午餐
 </a>
 。除了就美国的团结等问题发表自己的看法外，这位支持川普的黑人巨星还表示，当他带上〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普帽.html" target="_blank">
  川普帽
 </a>
 〞时，感觉自己简直就像 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超人.html" target="_blank">
  超人
 </a>
 〞一样。
 <br/>
 <br/>
 周四上午，美国著名饶舌歌手、唱片制作人及作家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威斯特.html" target="_blank">
  威斯特
 </a>
 如约到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 会晤川普。在于川普共进
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_午餐.html" target="_blank">
  午餐
 </a>
 前，他先参观了椭圆形办公室，并坦言〝我喜欢（川普）这家伙〞，且跟川普热情拥抱。
 <br/>
 <br/>
 面对在场记者，威斯特谈了自己对刑事司法改革和团结美国民众的看法。
 <br/>
 <br/>
 他告诉记者们，〝川普现在正在其英雄之旅上〞。
 <br/>
 <br/>
 威斯特赞颂川普与金正恩谈判、促使朝鲜放弃了核武计划：〝你阻止了战争。〞
 <br/>
 <br/>
 川普赞同他的说法：〝我们确实阻止了战争——拯救了数百万生命。〞
 <br/>
 <br/>
 威斯特希望可以促成美国人更加团结，不过其有些建议可能很难实现。比如，他鼓励川普与在演奏国歌时下跪的始作俑者、前美国国家橄榄球联盟（NFL）旧金山49人队（San Francisco 49ers）四分卫凯普尼克（Colin Kaepernick）一同参加下届超级碗（Super Bowl）比赛，并建议两个人都戴上写有〝让美国伟大〞的帽子。
 <br/>
 <br/>
 威斯特建议川普把他2016年的竞选口号 〝让美国再次伟大〞（Make America Great Again）稍微修改一下，去掉〝再次〞一词，因为他认为，这可能让美国黑人联想到种族主义和奴隶制。
 <br/>
 <br/>
 川普没有当众回应他的建议。
 <br/>
 <br/>
 威斯特表示，他不支持在芝加哥实行〝拦截搜身〞（Stop and Frisk）政策，认为这一政策无益于改善当地关系，并表示劝阻川普是他的当务之急。
 <br/>
 <br/>
 川普周一在奥兰多（Orlando）举行的国际警察局长协会（International Association of Chiefs of Police）年会上表示，他支持在枪击案频发的芝加哥市实行〝拦截搜身〞政策。他提到，前纽约市长朱利安尼（Rudy Giuliani）就曾制定这样一套强有力的方案，从而提升了纽约市的治安。
 <br/>
 <br/>
 对于威斯特表示出的反对意见，川普表示，他愿意考虑不同意见，如果可以用不同方式去做，他会采纳，他对此持完全开放的态度。
 <br/>
 <br/>
 威斯特还表示，他的一些好莱坞合伙人吓唬他，不让他戴着写有〝让美国再次伟大〞的帽子在公共场合露面。但他告诉川普，当他戴上〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普帽.html" target="_blank">
  川普帽
 </a>
 〞时，他感觉自己就像
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超人.html" target="_blank">
  超人
 </a>
 （Superman）一样——那可是他最喜欢的超级英雄。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
