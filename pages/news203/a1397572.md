### 川普与美国工人代表会面 推动工人就业
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】周三下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在白宫与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工人代表.html" target="_blank">
  工人代表
 </a>
 会谈，并向与会者介绍〝对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国工人.html" target="_blank">
  美国工人
 </a>
 承诺〞计划的进展。
 <br/>
 <br/>
 今年7月，白宫宣布了一项〝对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国工人.html" target="_blank">
  美国工人
 </a>
 的承诺〞（Pledge to America's Workers）计划，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统随后签署行政令创建美国工人全国委员会（National Council for the American Worker)。
 <br/>
 <br/>
 川普表示，目前已有160多家公司和组织承诺为美国人提供639万个工作岗位，并认为该计划能为美国人提供更多更好的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业.html" target="_blank">
  就业
 </a>
 机会，也能帮助雇主找到合适的工人。
 <br/>
 <br/>
 美国总统川普：〝我觉得他们的培训，会比政府做得更好。他们可以专程去公司培训员工。但我们作为政府不能去这样做，像你们这些了不起的公司那样去培训。 〞
 <br/>
 <br/>
 会上，川普谈到，目前美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业率.html" target="_blank">
  失业率
 </a>
 为3.7%，是50年来的最低点。
 <br/>
 <br/>
 美国总统川普：〝自大选以来，我们增加近60万个制造业工作，而且还要比这多得多。〞
 <br/>
 <br/>
 新唐人记者蔡美音综合报导
</div>
