### 美国土安全部：大篷车移民来自20国 含270名罪犯
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国土安全部.html" target="_blank">
  国土安全部
 </a>
 周四（11月1日）指出，目前向美国进发的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 来自至少20个国家，而且其中270余人有犯罪前科。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国土安全部.html" target="_blank">
  国土安全部
 </a>
 发表声明，称目前墨西哥境内的两组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 中，有的是黑帮成员、有的有犯罪历史，共计270余人。
 <br/>
 <br/>
 国土安全部没有告知其消息来源，一名发言人表示，该情报涉及执法方面的敏感信息。
 <br/>
 <br/>
 声明表示，国土安全部会继续关注正前往美国的大篷车移民，这些移民来自至少20个国家，而不只是洪都拉斯和危地马拉。
 <br/>
 <br/>
 声明指出，实际上，这些移民中混杂着数百名有犯罪前科的，包括已知的黑帮成员。其中有些是暴力犯罪分子 ，其罪行包括使用致命武器重度攻击他人、武装抢劫、性侵儿童及女性等。
 <br/>
 <br/>
 一名国土安全部发言人告诉《华盛顿观察家报》（The Washington Examiner），他们已掌握情况的这270人不仅仅是遭到了犯罪指控，而是被宣判有罪的。
 <br/>
 <br/>
 国土安全部还表示，他们也在继续观察这些移民的背景。这些人来自20多个国家，除了中美洲的洪都拉斯和危地马拉，还包括索马里、印度、海地、阿富汗和孟加拉等国。
 <br/>
 <br/>
 国土安全部指出，他们对这群人中的大部分都一无所知，必须准备好捍卫美国的边界并执行美国的法律，以保护美国的公民。据他们估计，两支大篷车队伍加起来共有6,500人左右。
 <br/>
 <br/>
 预计第一组大篷车移民将于11月3日抵达美墨边界，最迟不会超过11月12日，具体时间取决于他们是步行还是乘车。
 <br/>
 <br/>
 五角大楼周一宣布，将把派往美墨边境的驻军由原来的800人增加到大约5200人。
 <br/>
 <br/>
 周三，鉴于移民与墨西哥执法人员多次发生冲突且移民的情况复杂，川普总统表示，他决定再次增加兵力，派驻15,000人保守边界。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
