### 民调：共和党支持率急剧上升 中期选举前景乐观
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】美媒发布的最新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民调.html" target="_blank">
  民调
 </a>
 结果显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 在一些战场州（battleground）已赢得优势，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 后或更有力地掌控参议院。
 <br/>
 <br/>
 据福克斯新闻（Fox News）10月3日公布的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民调.html" target="_blank">
  民调
 </a>
 结果，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 在5个战场州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支持率.html" target="_blank">
  支持率
 </a>
 正一路上扬。
 <br/>
 <br/>
 该民调显示，与9月初相比，表示对即将到来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 〝非常〞感兴趣的共和党人，在亚利桑那州的人数
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上升.html" target="_blank">
  上升
 </a>
 了2个百分点，在印第安纳州上升了9%，在密苏里州和北达科他州分别上升了8%，在田纳西州甚至上升了11%。
 <br/>
 <br/>
 共和党支持率在上述几州的增长已经消除了民主党上个月在数州的优势。
 <br/>
 <br/>
 共和党的支持率急剧增长与最高法院大法官提名人卡瓦诺（Brett Kavanaugh）的确认有关。由于参议院两党的席位之比是51:49，导致两党在卡瓦诺的确认上每票必争，其紧张程度令人咂舌。
 <br/>
 <br/>
 从目前的状况来看，民主党不择手段反对保守派大法官确认的做法已伤及不少民主党参议员候选人，直接降低了他们的支持率，而民主党阻挠卡瓦诺入职最高法院的目标基本上已完全失败。
 <br/>
 <br/>
 在福克斯的抽样调查中，只有一名民主党候选人领先，而且是唯一一名在自己的选区中支持率超过50%的民主党竞选人。
 <br/>
 <br/>
 美国中期选举将从11月6日开始举行，届时，不仅众议院所有435名众议员都要改选，参议院的33个席位及36个州的州长职位也要重新洗牌。
 <br/>
 <br/>
 总部设于华盛顿的独立民调和智库机构皮尤研究中心（Pew Research Center）最近调查发现，美国选民目前的投票热情是20多年来历次中期选举中最高的。因中期选举将决定对国会的控制权，所以近3/4的登记选民表示，他们很关注共和党是否能保持在两院的优势。
 <br/>
 <br/>
 皮尤的调查还发现，近60％的选民表示，对川普的看法将会影响他们的投票，其余人则表示，对川普的印象不会影响到他们的决定。
 <br/>
 <br/>
 川普近来在竞选集会上一再敦促其支持者，选举时一定要走出家门，不要把投票机会放弃。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
</div>
