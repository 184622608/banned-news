### 参访肯尼亚 美第一夫人喂小象 与儿童共舞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国第一夫人.html" target="_blank">
  美国第一夫人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼娅.html" target="_blank">
  梅拉尼娅
 </a>
 周一（10月1日）独自启程前往非洲访问。她的5天4国之旅的关注重点是促进儿童健康和福利发展。周五（5日），她来到肯尼亚野生动物，并亲自喂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小象.html" target="_blank">
  小象
 </a>
 配方奶，神情轻松自若展现亲和力。
 <br/>
 <br/>
 周五，内罗毕国家公园的野生动物园之旅，开启了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼娅.html" target="_blank">
  梅拉尼娅
 </a>
 （Melania Trump）对肯尼亚的访问。梅拉尼娅通过这个活动强调了该地区的野生动植物保护工作。
 <br/>
 <br/>
 据美联社报导，梅拉尼娅当天给公园里养的两只
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小象.html" target="_blank">
  小象
 </a>
 喂了配方奶，期间她拍了拍一只小象的背，并抚摸了另一只小象的耳朵。
 <br/>
 <br/>
 在野生动物园期间，梅拉尼娅拿起手机拍照，并用双筒望远镜观察斑马、长颈鹿、黑斑羚、河马和犀牛。此外，她当天还参观了为阻止象牙交易而烧掉105吨象牙的地方。

 之后，梅拉尼娅访问了肯尼亚利穆鲁的鸟巢孤儿院，并和那里孩子们一起跳舞。
 <br/>
 <br/>
 梅拉尼娅对孤儿院的工作人员表示感谢，她说，感谢你们所做的，感谢你们照顾他们。
 <br/>
 <br/>
 梅拉尼娅给孩子们读故事后，作为回报，一名小男孩也给她讲了故事。接着，一群孩子围着梅拉尼娅唱歌，她也与孩子们一起跳舞，其乐融融。
 <br/>
 <br/>
 肯尼亚第一夫人玛格丽特・肯雅塔（Margaret Kenyatta）陪同梅拉尼娅参加了狩猎旅行，预计肯雅塔夫妇将于周五晚些时候参加传统表演。

 <br/>
 这是梅拉尼娅第一次到访非洲，也是她第一次以美国第一夫人的身份单飞国际旅行。梅拉尼娅5天非洲之旅的第一站是加纳，参观了当地的一家医院。后来她前往第二站马拉维的一家小学参观了语言课程，肯尼亚是第三站，最后一站将是埃及。
 <br/>
 <br/>
 梅拉尼娅这次非洲行，主要目的是为推广儿童福利计划。她作为12岁儿子的母亲，今年在美国推出了一项名为〝Be Best〞的计划，专注于儿童福祉问题。本周的行程将是她在国外推广该计划及实现其目标的第一个阶段。
 <br/>
 <br/>
 他的丈夫川普（特朗普）总统发推文表示：〝我国优秀的第一夫人梅拉尼娅在非洲表现非常好。受人民爱戴，她也爱他们！这是件美好的事。〞
 <br/>
 <br/>
 48岁的梅拉尼娅已经陪同川普多次出访，到访国家包括沙特阿拉伯、以色列、意大利、比利时、法国、日本、韩国和英国。她曾在7月份参加川普与俄罗斯总统普京在芬兰举行的峰会，但由于手术后需要恢复，她没有参加6月份川普与朝鲜领导人金正恩的会晤。
 <br/>
 <br/>
 在梅拉尼娅飞越大西洋的前几天，川普在联合国宣布他和梅拉尼娅〝热爱非洲〞。
 <br/>
 <br/>
 梅拉尼娅也在9月26日联合国大会的与会者招待会上表示：〝我将于10月1日开始对非洲4个美好而迥异的国家进行单独访问。我们来自不同国家，每个国家都面临着各自的挑战，但我衷心希望，把青年一代培养成幸福、健康和有道德责任感的人，这一美好意愿，能够将我们所有人团结在一起。〞
