### CNN记者对女实习生动手 白宫发布影片还原真相
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】美国有线电视新闻网（CNN）的记者周三（11月7日）与美国总统川普（特朗普）发生激烈对话，并一度拒绝交出麦克风，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 撤销了他的媒体通行证。白宫发言人桑德斯强调，无法容忍阿科斯达对一名女实习生动手，并在推特上传影片佐证。
 <br/>
 <br/>
 美东时间11月7日晚间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 对外宣布取消了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN记者.html" target="_blank">
  CNN记者
 </a>
 阿科斯达（Jim Acosta）进入白宫的记者通行证，指其当天在记者会上对白宫一名年轻的女实习生〝动手〞。
 <br/>
 <br/>
 桑德斯（Sarah Sanders）还在推特上传1段15秒的影片，慢动作重播白宫实习生欲将阿科斯达的麦克风拿走的过程，表示阿科斯达在记者会上与实习生发生肢体接触，白宫无法容忍此行为。
 <br/>
 <br/>

 <br/>
 当天在记者会上，阿科斯达不断地向川普质疑，为何把中美洲的大篷车移民描述为〝入侵〞？〝因为我们对‘入侵’有不同理解〞，川普说。阿科斯达问总统是不是在〝妖魔化〞外来移民？
 <br/>
 <br/>
 川普否定了〝妖魔化移民〞的说法，解释道:〝不，不，我希望他们来，但他们要合法地进来，你知道他们必须按照程序进来。我要走程序，我要让人们进来。我们需要人。〞
 <br/>
 <br/>
 在总统说话过程中，阿科斯达打断总统，川普制止他，并解释说〝等等，你知道我们为什么需要人吗？因为成千公司进入（美国），我们需要（劳动力）。〞
 <br/>
 <br/>
 阿科斯达接着就移民问题发问，并再次重复〝那些（移民）不是入侵……〞，川普说，〝说实话，应该是我管理国家，你们管理CNN。〞阿科斯达继续纠缠，总统停止他的发问，并说，〝够了。〞
 <br/>
 <br/>
 在这过程中，白宫女工作人员走过来想拿下阿科斯达的麦克风，但是他躲闪不给，并挡开工作人员的手臂，还转换话题，向川普提出〝通俄门〞方面的问题，川普简单回答他之后说，〝够了，把麦克风放下。〞
 <br/>
 <br/>
 阿科斯达仍然不停，继续发言，在激烈的冲突中川普离开发言台，等阿科斯达坐下后，川普走回到麦克风，对着阿科斯达说，〝CNN应该自感羞耻，因为你为其工作。你是一个粗鲁的、糟糕的人。〞
 <br/>
 <br/>
 当川普回答其他记者的时候，阿科斯达又几次站起来试图打断和发问，川普转过来对他说，〝当你报导假新闻的时候—-CNN多次这样炮制（假新闻），你们就是人民的敌人。〞
 <br/>
 <br/>
 当晚，阿科斯达想再回到白宫做连线时，在供记者出入的宾夕法尼亚大道入口被特勤局拦下。他发推文说：〝我刚才进入白宫时被拒，特勤局告诉我，我不能入内进行晚上8时的节目。〞
 <br/>
 <br/>
 CNN透过声明抨击白宫的这个决定〝威胁我们的民主〞。
 <br/>
 <br/>
 桑德斯稍后在声明中证实阿科斯达的推文，称中止阿科斯达的记者通行证是因为他〝对一个试着做本分工作的白宫实习生动手〞，说这种行为〝绝对无法接受〞。
 <br/>
 <br/>
 桑德斯在声明中说，〝川普总统相信新闻自由，并期待和欢迎（媒体）提出棘手问题。〞但是她解释，一些记者不合适的作法是〝绝对不可接受的〞。
 <br/>
 <br/>
 桑德斯说，〝这种行为也十分不尊重其他记者同行，不给他们机会提问。川普给记者的采访自由比史上任何总统都要多。〞
 <br/>
 <br/>
 桑德斯表示，情况并不是CNN所说的那样。事实是，川普总统在7日当天的活动中，用了一个半小时回答了35名记者的68个问题，其中也包括回答CNN好几个问题，充分体现了对新闻自由的支持，但即使这样CNN还攻击总统，说他不支持新闻自由。
 <br/>
 <br/>
 桑德斯说：〝CNN以做出这种行为的员工为傲，这个事实不但恶心，还证明了他们恶劣的不尊重其他任何人，包括替这个政府效力的年轻女性。〞
