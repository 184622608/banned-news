### 亚利桑那选情异常 机器故障投票叫停
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】周二（11月6日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚利桑那.html" target="_blank">
  亚利桑那
 </a>
 州一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 占优势的选区出现问题，因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_机器故障.html" target="_blank">
  机器故障
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 站暂停投票并打发选民返回等待，需电话通知后才能再回投票站投票。
 <br/>
 <br/>
 据美国保守派网站〝Gateway Pundit〞报导，周二，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 战事正酣，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚利桑那.html" target="_blank">
  亚利桑那
 </a>
 州马里科帕县（Maricopa county）的一个选区居然出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_机器故障.html" target="_blank">
  机器故障
 </a>
 ，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 被迫取消。
 <br/>
 <br/>
 该选区为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 优势选区。周二上午，位于该县皇后湾图书馆（Queen Creek Library）的投票站竟然告诉选民们一个令人震惊的消息——仅有的两台指定用于打印选票的打印机全都坏了。
 <br/>
 <br/>
 一位选民告知当地共和党主席霍夫曼（Jake Hoffman），投票站工作人员要求在场的150至200名选民在一张便签纸上写下自己的姓名和电话号码，然会回去等待电话或短讯通知，等打印机恢复正常后才能再次赴投票站投票。
 <br/>
 <br/>
