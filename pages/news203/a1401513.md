### 7级地震袭阿拉斯加 尚无人员伤亡报告
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】观众朋友大家好，欢迎收看新唐人全球新闻。周五，7级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强震.html" target="_blank">
  强震
 </a>
 袭击美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿拉斯加.html" target="_blank">
  阿拉斯加
 </a>
 南部，造成路面塌陷，上万家庭断电。截至发稿，还没有伤亡报告。此前的海啸警报也已取消。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国地质调查局.html" target="_blank">
  美国地质调查局
 </a>
 称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_7级地震.html" target="_blank">
  7级地震
 </a>
 发生在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿拉斯加.html" target="_blank">
  阿拉斯加
 </a>
 麦肯锡角以南，震中位于安克拉奇以北8.1英里处，深度为26.7英里。
 <br/>
 <br/>
 画面显示，地震使房屋吊灯剧烈摇晃，超市货架上的商品散落满地。
 <br/>
 <br/>
 地震还破坏了街道路面，造成交通阻塞。
 <br/>
 <br/>
 阿拉斯加市政电力公司表示，约7千到一万户家庭断电。但地震对发电基础设施还没有造成损害。
 <br/>
 <br/>
 有当地高中生表示，校方及时疏散他们，现在都很安全。
 <br/>
 <br/>
 远在阿根廷的美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）发推文祝福阿拉斯加，称这场地震很大，联邦政府会不遗余力帮助民众。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
