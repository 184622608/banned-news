### 黑利轰媒体不公:黑人教堂遭枪击并未谴责奥巴马
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】周一（10月29日），即将卸任的美国驻联合国大使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑利.html" target="_blank">
  黑利
 </a>
 （Nikki Haley）指出，因宾州犹太
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_教堂枪击案.html" target="_blank">
  教堂枪击案
 </a>
 而谴责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统是不合理的，因为几年前南卡州一座黑人教堂遭到同样袭击时，舆论并没有谴责时任总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 （Barack Obama）。
 <br/>
 <br/>
 周一晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑利.html" target="_blank">
  黑利
 </a>
 发推驳斥对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的攻击。她表示，她经过费力比较，发现上周六发生的宾州匹兹堡（Pittsburgh）犹太
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_教堂枪击案.html" target="_blank">
  教堂枪击案
 </a>
 与2015年发生的南卡州查尔斯顿（Charleston）黑人教堂枪击案非常相似。当时美国的种族分裂现象非常严重，但是舆论并没有指责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 ，而只是关注遇害者及其家属的境遇。
 <br/>
 <br/>
 黑利呼吁，现在应尊重在犹太教堂枪击案中受害的家庭，并停止谴责。
 <br/>
 <br/>

 <br/>
 上周六上午，一名反犹太主义的白人枪手在匹兹堡的〝生命之树〞犹太教堂向参加宗教活动的人群开枪射击，并大喊〝所有犹太人必须死亡〞，造成11名人死亡，6人受伤，其中包括4名警察。
 <br/>
 <br/>
 事件发生后，美国左派
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_媒体.html" target="_blank">
  媒体
 </a>
 立即把责任推给川普，攻击他宣扬白人民族主义，从而激励了极端分子攻击民主党和少数族裔。
 <br/>
 <br/>
 匹兹堡市民主党籍市长佩杜托（William Peduto）甚至明确表示不欢迎川普前往吊唁遇难者，声称川普的到来将会〝分散〞人们对葬礼的注意力。
 <br/>
 <br/>
 左派媒体的攻势使得一些共和党人担心，川普到匹兹堡慰问可能会给他们带来政治压力。该市重要的共和党人罗德迪（Jim Roddey）表示，他希望一直活跃在中期选举中的川普不要去匹兹堡并出席集会。
 <br/>
 <br/>
 上周，自由派媒体借〝炸弹邮包〞案攻击川普，指责他批评民主党和媒体的言论〝鼓励了暴力〞。
 <br/>
 <br/>
 川普周一在推文中指出，媒体不正确甚至骗人的报导激发了国内的愤怒，假新闻媒体是民众真正的敌人。他呼吁媒体停止敌意和不公正的报导，这样才能迎来各方的和平与和谐。
