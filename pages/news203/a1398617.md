### 极左翼闯福克斯主持人家砸门毁车 警方以〝仇恨罪〞调查
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】周三（11月7日）晚，一些〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反法运动.html" target="_blank">
  反法运动
 </a>
 〞（Antifa）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴民.html" target="_blank">
  暴民
 </a>
 闯到福克斯新闻（Fox News）主持人卡尔森（Tucker Carlson）家毁车砸门，目前警方已以涉嫌〝仇恨罪〞（hate crime）立案调查。
 <br/>
 <br/>
 据美国保守派网站〝每日电讯〞（The Daily Caller）10月8日报导，周三晚上，华盛顿DC的一些左翼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴民.html" target="_blank">
  暴民
 </a>
 到福克斯新闻《今晚塔克．卡尔森》（Tucker Carlson Tonight）节目的主持人家寻釁闹事，抗议卡尔森的保守派政治观点。
 <br/>
 <br/>
 〝每日电讯新闻基金会〞（The Daily Caller News Foundation）获得的一份警方报告显示，华府警察怀疑该事件系出于反政治（anti-political）动机的〝仇恨罪〞，并且正在调查当晚聚集在卡尔森家外进行暴力抗议的左翼极端分子。
 <br/>
 <br/>
 这些暴民在卡尔森家外的车道上喷上无政府主义标识，并在他家的前门及他家车道上所停靠的车辆上喷上一些关于卡尔森政治派别的标志。
 <br/>
 <br/>
 该行动是由〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反法运动.html" target="_blank">
  反法运动
 </a>
 〞的一个名为〝Smash Racism DC〞的团体组织的。他们把卡尔森家位于华府的住址公布在网络上，并安排一组暴民前去卡尔森家，大骂他是〝种族主义人渣〞（racist scumbag），并要求他逃离这座城市。
 <br/>
 <br/>
 卡尔森是〝每日电讯新闻基金会〞的联合创始人，事发时，他正在福克斯新闻的工作室工作。
 <br/>
 <br/>
 警方称，当时卡尔森家中只有其妻一人在家。她听到有人使劲敲击她家的前门，后来发现一大群人堵在她家门外胡闹，于是赶紧打电话报警。
 <br/>
 <br/>
 卡尔森告诉《华盛顿邮报》（The Washington Post），后来有人开始撞门，结果把前门撞出了裂缝。
 <br/>
 <br/>
 警察到场后清点，发现闹事的暴民约有20来人。
 <br/>
 <br/>
 警方还在现场截获了6张手写的海报。视频显示，暴民们手举标志，高喊〝卡尔森，我们要战斗！我们知道你晚上睡哪儿！〞
 <br/>
