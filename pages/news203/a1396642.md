### 美国威力球彩票开奖 中奖号码揭晓
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】周二晚间，美国〝超级百万〞彩票再次开奖，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_头奖.html" target="_blank">
  头奖
 </a>
 号码揭晓。累计奖金高达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_16亿美元.html" target="_blank">
  16亿美元
 </a>
 ，创下世界记录。
 <br/>
 <br/>
 昨晚的开奖吸引了全美各地的彩票迷。
 <br/>
 <br/>
 摇出的中奖号码是5 28 62 65 70，威力球号码是5。
 <br/>
 <br/>
 猜中全部六个号码的幸运儿，税后可以获得9.7亿美元，可选择一次性领取5.48亿美元现金，也可以选择29年内，分期领完这笔奖金。
 <br/>
 <br/>
 开奖前，人们络绎不绝来购买彩票，希望幸运降临。说到获奖后的梦想，也是五花八门。
 <br/>
 <br/>
 彩票购买者MARK PRYSLER：〝可能买许多房子，环游世界，就是享受生活吧。〞
 <br/>
 <br/>
 彩票购买者JEFF WILLIAMS SENIOR：〝我会买一架波音747飞机，退役飞机，努力改造、翻新，把它变成家。〞
 <br/>
 <br/>
 彩票购买者LONNIE SHEPARD：〝华盛顿失去了许多新街头乐队音乐，我想把这些音乐找回来。我想找一个地方，不同乐队可（在那）展示才华。〞
 <br/>
 <br/>
 媒体报导，周三，南卡罗来纳州一位居民中了

  头奖
 </a>

 <br/>
