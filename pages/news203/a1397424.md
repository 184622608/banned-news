### 摄影师意外拍到绝美求婚照 寻主角一周终于现身
------------------------

 【新唐人北京时间2018年10月31日讯】日前一名男子在美国加州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_优胜美地国家公园.html" target="_blank">
  优胜美地国家公园
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悬崖.html" target="_blank">
  悬崖
 </a>
 上单膝跪地向女友求婚的情景，意外被摄影师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迪佩尔.html" target="_blank">
  迪佩尔
 </a>
 拍下，并将其PO上网络，请全球网友协寻照片的主角，如今这对情侣终于现身！
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迪佩尔.html" target="_blank">
  迪佩尔
 </a>
 （Matt Dippel）日前到国家公园要给他的朋友拍照时，在取景窗中意外看到一对男女走上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悬崖.html" target="_blank">
  悬崖
 </a>
 ，之后男方单膝下跪向女友求婚，在夕阳的照映下，格外动人。他立即拍下这张浪漫指数破表的照片。
 <br/>
 <br/>
 迪佩尔表示，其实，这个悬崖常有人求婚、示爱，但他第一次捕捉到这个难得的画面，于是将照片PO上脸书，希望透过网路平台找到照片中的男女主角。该照片也被网友广传，许多网友称，〝好想知道主人公到底是谁？〞
 <br/>
 <br/>

 事过近一周后，摄影师终于如愿找到了这对情侣。据《ABC》报导，他们就是贝尔（Charlie Bear）与未婚妻梅莉莎（Melissa）。
 <br/>
 <br/>
 贝尔表示，他和梅莉莎非常喜欢徒步旅行，因为他想要给女友一次永生难忘的求婚经历，所以他选择在优胜美地悬崖单膝跪地求婚。后来他们在浏览一个地方新闻的报导时，无意间发现当时的求婚场景被拍下，这才与摄影师联系。
 <br/>
 <br/>
 <center>
 </center>
 <br/>
 不过，拍照地方虽然风景壮观又梦幻，但因没有设置围栏，观光客们一个不小心很有可能就会掉下7500英尺高（约2300公尺）的悬崖，发生憾事。
 <br/>
 <br/>
 公园发言人盖迪曼（Scott Gediman）称，国家公园拥有美丽的山谷、瀑布等壮丽景色，游客可以步行到悬崖边缘，俯瞰整个区域美景。但因国家公园仅有一小处设有栏杆，游客步行到悬崖边缘时，是完全没有安全措施遮蔽的，一不小心就会落入谷底的花冈岩裂缝死亡。
 <br/>
 <br/>
 据《CBS》报导，该公园在25日时还发生一对夫妇坠谷身亡的事件。而截至25日，今年已有超过10人在此地丧生，且多数都是坠谷身亡。
 <br/>
 <br/>
