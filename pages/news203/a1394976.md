### 川普签署两项法案 降低药品价格
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 周三签署两项
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法案.html" target="_blank">
  法案
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降低药品价格.html" target="_blank">
  降低药品价格
 </a>
 ，终止让患者处于弱势的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_封口条款.html" target="_blank">
  封口条款
 </a>
 〞,保护患者利益，兑现竞选承诺。
 <br/>
 <br/>
 白宫周三发表声明，川普签署了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最低价格法.html" target="_blank">
  最低价格法
 </a>
 〞和〝患者知情权药物价格法〞，进一步
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降低药品价格.html" target="_blank">
  降低药品价格
 </a>
 ，减轻患者自付费用，让患者知道一样的药品，到哪里买更便宜。
 <br/>
 <br/>
 川普在随后的发布会上表示，药品价格公开，不仅让患者受益，也促进药品市场公平竞争。
 <br/>
 <br/>
 美国总统川普：〝民众将能够看到定价。他们将能够看到应该去哪里买。当他们开始不去某些药店时，那些药店将会降价。〞
 <br/>
 <br/>
 川普表示，从上任第一天，就把降低药品价格作为政府首要事项。
 <br/>
 <br/>
 美国总统川普：〝这对我们来说非常重要。我告诉你们，从第一天起就一直是首要事项。〞
 <br/>
 <br/>
 他说，去年，美国食品药品监督管理局采取史上最重要举措，审核批准了1000多种低成本非专利药，为美国人节省了90亿美元。
 <br/>
 <br/>
 美国总统川普：〝我们采取行动，制止医院多收老年人的健保费，仅今年一年就将节省数亿美元。〞
 <br/>
 <br/>
 川普还说，在国际贸易中，也将保护美国人利益，推行公平贸易，同时推行短期健保计划，取代奥巴马健保。
 <br/>
 <br/>
 最新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法案.html" target="_blank">
  法案
 </a>
 公布后，有15家大型制药企业宣布降低价格，或推迟涨价。
 <br/>
 <br/>
 美国总统川普：〝15家制药公司降低了定价，推迟价格上涨，或是在年底之前冻结价格上涨。〞
 <br/>
 <br/>
 川普表示，未来还会采取进一步措施，为患者节省更多开支。
 <br/>
 <br/>
 新唐人记者李梅、李静明综合报导
</div>
