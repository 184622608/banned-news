### 记者被撤通行证 CNN提吿 白宫反驳
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN记者.html" target="_blank">
  CNN记者
 </a>
 阿科斯塔的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通行证.html" target="_blank">
  通行证
 </a>
 被吊销，CNN称做法违宪，因此提出起诉，被白宫反驳。
 <br/>
 <br/>
 11月7日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN记者.html" target="_blank">
  CNN记者
 </a>
 阿科斯塔在记者会上不断追问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统移民和通俄门问题，并一度拒绝交出麦克风。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 因此撤销了他的媒体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通行证.html" target="_blank">
  通行证
 </a>
 。
 <br/>
 <br/>
 13日，CNN起诉川普政府，要求恢复阿科斯塔通行证。白宫回应说：CNN又在〝哗众取宠〞，将强力抵制这起诉讼。
 <br/>
 <br/>
 美国总统 川普（11/09/2018）：〝我认为记者阿科斯塔是一个非常不专业的人。〞
 <br/>
 <br/>
 川普认为白宫是神圣之地，人们理应尊重白宫和总统。
 <br/>
 <br/>
 美国总统 川普：〝当你在白宫时-这对我来说是一个非常神圣的地方。这是非常与众不同的地方。你必须尊重白宫。你必须尊重总统。〞
 <br/>
 <br/>
 新唐人记者李子玉综合报导
