### 疑用恐吓手段逼伪证 通俄门穆勒团队遭起诉
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】下面来关注美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查。近日，一位保守派作家科西起诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 调查团队用恐吓方式逼他做
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伪证.html" target="_blank">
  伪证
 </a>
 。请看报道。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查的证人之一、保守派作家科西（Jerome Corsi）周一（12月3日）对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 （Robert Mueller）的调查团队提出刑事和道德诉讼（criminal and ethics complaint），称他们逼他做
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伪证.html" target="_blank">
  伪证
 </a>
 来攻击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 。78页文件已被递交给多位执法部门高官，包括司法部代理部长惠特克（Matthew Whitaker）和司法部监察长霍洛维茨（Michael Horowitz）等。
 <br/>
 <br/>
 通俄门调查的证人之一、保守派作家 Jerome Corsi：〝我很配合、给了他们所有他们要的，包括电脑、时光机备份、自愿交出了电邮、推特帐号，所有他们想要的我都给了。但第一天进行了记忆测试，我没做好，突然之间，我余生就要在监狱中度过了。这是他们恐吓人的招数，我相信他们是有意而为。〞
 <br/>
 <br/>
 穆勒团队正准备起诉科西，理由是他在2016年发送电子邮件提醒川普前竞选顾问史东（Roger Stone），维基解密（WikiLeaks）将公开希拉里竞选团队及民主党全国委员会（Democratic National Committee）的电邮。科西本人对此否认。
 <br/>
 <br/>
 对于穆勒团队的调查方式，川普前竞选主席曼纳福特（Paul Manafort）的律师把其被调查时的情形透露给了川普私人律师朱利安尼，朱利安尼11月末公开说：穆勒团队对待曼纳福特像对待一个恐怖份子，他被单独监禁、反覆被审问。
 <br/>
 <br/>
 川普私人律师 朱利安尼：〝特别检察官（穆勒）越线了，他用恐吓的方式让人们说出他认为的真相。说到曼纳福特先生，他已经被调查了5-6个月，反覆被询问，他们希望他给出不利于总统川普的信息，但曼纳福特说那些内容并不是事实。〞
 <br/>
 <br/>
 而川普前律师科恩（Michael Cohen）于上周认罪，认罪前穆勒调查团队花了70小时听取其汇报。科恩的认罪内容是：对于川普竞选总统期间是否在莫斯科有地产生意，科恩对国会撒了谎。不过这与穆勒团队针对曼纳福特的起诉一样，与通俄门调查的初衷没有关系。
 <br/>
 <br/>
 川普总统在3日的推文中说：〝穆勒，与大家所认为的非常不同；他与失控的愤怒民主党人组成的团队并不需要真相，他们想要的是谎言。而事实真相对他们而言非常可怕！〞
 <br/>
 <br/>
 有司法专家认为穆勒团队在对待川普与希拉里时使用了双重标准。时任国务卿并负责〝外国投资委员会〞的希拉里，帮助俄罗斯国家原子能公司（Rosatom）获得了美国〝铀壹公司〞（Uranium One）拥有的20%铀控制权的交易、被称为〝铀矿门〞。但当时身为FBI局长的穆勒并没有对此展开调查。
 <br/>
 <br/>
 周二，穆勒团队对前国家安全顾问迈克尔・弗林提出量刑建议。弗林于2017年12月认罪向FBI谎报他与俄罗斯的接触。本月18日将在美国哥伦比亚特区地方法院被判刑。
 <br/>
 <br/>
 新唐人记者李兰综合报导
