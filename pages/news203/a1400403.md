### 川普警告墨西哥控制移民 否则全线关闭美墨边境
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周四（11月22日）表示，如果墨西哥当局不能控制滞留境内的大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 ，他将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全线.html" target="_blank">
  全线
 </a>
 关闭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 ，直到情况恢复正常。
 <br/>
 <br/>
 感恩节上午，正在佛州海湖庄园（Mar-a-Lago）度假的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 再就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 问题发表评论。他告诉记者，如果美国发现抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 的大篷车移民陷于失控状态，美国将把两国之间的全部入境口岸都关闭一度时间。
 <br/>
 <br/>
 川普指出，目前滞留在墨西哥蒂华纳（Tijuana）的大篷车移民中有很多犯罪分子，如果墨西哥当局在抑制这些中美洲移民涌入美国的问题上无所作为，那么美国将暂停与墨西哥之间商贸往来，直到边境地区恢复正常状态。
 <br/>
 <br/>
 川普表示，美国关闭边境口岸，这意味着在边境重新开放前，墨西哥将无法再把汽车卖入美国，而这将严重伤害墨西哥的经济利益。
 <br/>
 <br/>
 他还指出，由于情况严重，他两天前已命令关闭了部分边境口岸，并允许军队在必要时使用武力阻挡强行进入美国的大篷车移民，包括使用杀伤性武器。
 <br/>
 <br/>
 川普同时威胁，如果国会不同意给他足够的拨款用于边境墙建设，他可能会让政府关门。
 <br/>
 <br/>
 据《华盛顿邮报》（The Washington Post）周三报导，按照新计划，申请庇护的大篷车移民必须在墨西哥境内等待案件审理。
 <br/>
 <br/>
 新措施允许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_庇护申请.html" target="_blank">
  庇护申请
 </a>
 者在案件审批过程中留在美国，但必须提出合理的理由，证实自己在墨西哥境内会担心遭受迫害、因此申请入境美国。
 <br/>
 <br/>
 该报导说，此项措施将从周五开始生效。而一名墨西哥官员表示，按照墨西哥当前的法律，当局是不允许向它国申请庇护的人在案件审批过程中留在墨西哥等待的。
 <br/>
 <br/>
 不过，《华邮》称，一名美国国土安全部官员告诉他们，美国政府没有打算立刻实施新的措施。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
