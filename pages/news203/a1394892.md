### 飓风迈克尔登陆佛州 最初时速155英里
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】周三下午，迈克尔飓风以四级的规模登陆美国佛罗里达州，几小时后才降为三级。联邦官员称这是一场百年来最大飓风。
 <br/>
 <br/>
 美东时间周三下午1:40左右
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风迈克尔.html" target="_blank">
  飓风迈克尔
 </a>
 登陆巴拿马城，最高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_风速.html" target="_blank">
  风速
 </a>
 每小时155英里，掀起的海浪可达14英尺。
 <br/>
 <br/>
 〝联邦紧急事务管理署〞署长 Brock Long：〝对于这一区域，这应该是自1851年来遭遇的最大的飓风。〞
 <br/>
 <br/>
 虽然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 周二已下达强制撤离令，但仍有部分居民未能及时逃离危险区域。
 <br/>
 <br/>
 美国总统 川普：〝一些地区非常穷，对于穷人来说没有足够钱撤离，我们我们派巴士过去。〞
 <br/>
 <br/>
 由于一线救援人员无法在飓风过境时做出响应，佛州州长斯科特（Rick Scott）要求无法撤离的民众立刻找到避难所（https://www.floridadisaster.org/info/），并准备了100万加仑的水和150万份食物；同时他启动了佛州赈灾基金（bit.ly/2e3J6NH），民众可以捐款。
 <br/>
 <br/>
 佛州州长 Rick Scott ：〝记住，我们可以重建你的房屋，但无法让你重生。认真对待。让家人安全。佛州不会轻视这场风暴，每个佛州家庭也应该同样对待。〞
 <br/>
 <br/>
 除了海岸警卫队，佛州还部署7000警力配合应战。
 <br/>
 <br/>
 总统川普在推文中说，〝佛州，我们和你并肩作战！〞
 <br/>
 <br/>
 海湾电力（Gulf Power）发言人Jeff Rogers表示，迈克尔过后预计会有22万5千居民断电，直接受影响地区需要数周才能恢复。为保障安全，工作人员只有在风速降至每小时30英里后才能开始工作。
 <br/>
 <br/>
 迈克尔在佛州登陆，随后还会向北过境乔治亚、南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北卡.html" target="_blank">
  北卡
 </a>
 等州。
 <br/>
 <br/>
 乔治亚州长迪尔宣布108县进入州紧急状态，但未发出强制撤离令。
 <br/>
 <br/>
 乔治亚州长 Nathan Deal：〝这场飓风与我们记忆中的不同，它的破坏程度会极大，保障人身安全头等重要。保护好自己和家人。帮助那些需要帮助的人。〞
 <br/>
 <br/>
 北卡州长库珀（Roy Cooper）也宣布66个县进入紧急状态。预计北卡中部部分区域，降雨可达7英寸。对于刚刚经历飓风〝佛罗伦斯〞重创的北卡，〝迈克尔〞无疑是雪上加霜。杜克能源（Duke Energy）公司预计，迈克尔将在南、北卡导致30万到50万例断电。
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
