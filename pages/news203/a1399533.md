### 1500大篷车抵边境 美国防部：5800士兵严阵以待
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】超过1500名中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民，周四（15日）抵达墨西哥接壤美国的边境城市蒂华纳。美国国防部严阵以待，目前有5,800名美军镇守
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民分别乘坐22辆巴士抵达蒂华纳，当局预计稍后会有更多巴士抵达，人数可能会超过3000人。多数移民表示，在等待车队组织者到来再计划下一步行动，包括寻求庇护或一同突破美军防线。
 <br/>
 <br/>
 带着两名孩子的洪都拉斯妇人索托（Carmen Soto）称，自己已经迫不急待要看到边境了。这像一场没有完结的旅程，〝但感谢上帝，我们终于来到蒂华纳了。〞
 <br/>
 <br/>
 墨西哥当局为这些移民设立了临时庇护中心，提供食物及医疗服务。另外还有一些人在沙滩及广场留宿。蒂华纳市政府稍早曾承诺，为他们提供3000份工作给想留下的人。
 <br/>
 <br/>
 14日，国土安全部长尼尔森与国防部长马蒂斯共同前往边境探望美军，鼓励将士守边关。他们都肯定了部署军队以保护边境的做法。目前有约5,800名军人派驻在边境。
 <br/>
 <br/>
 国防部长马蒂斯告诉美军，〝新闻报导怎么说的都有，你们就专注于你们军官的指令。〞
 <br/>
 <br/>
 美国副总统彭斯10月25日在接受采访时表示，大篷车是由海外的左翼集团资助。为了控制非法入境，川普（特朗普）总统11月9日签署公告，90天内暂停为非法越过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 的外国人提供庇护申请 。
