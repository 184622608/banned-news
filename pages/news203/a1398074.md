### 捍卫华人价值 华人助选共和党
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】前年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国总统大选.html" target="_blank">
  美国总统大选
 </a>
 ，由一批高学历高技术华裔移民组成的助选团，曾捐款雇用小飞机，在曼哈顿上空打出〝纽约华裔支持川普〞横幅，引人注目。而今年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，也有一批背景类似的第一代华裔高知群体和中产阶层群体，坚定地站在了纽约共和党候选人这边，他们的参政热情，前所未见的高。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-5/p9111862a802133071.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-5/p9111862a802133071-ss.jpg"/>
   <br/>
   <font size="-1">
    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人助选团.html" target="_blank">
     华人助选团
    </a>
    将
    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
     民主党
    </a>
    与共和党的根本分歧点整理出表格。
   </font>
  </a>
 </center>
 <br/>
 这几天，各路候选人为11月6日的普选冲刺阶段，在华埠、皇后区和布碌仑华社，无论是共和党籍州长候选人马利纳罗（Marc Malinaro）、共和党竞选纽约联邦参议员的候选人萧法莉（Chele Farley）、竞选连任的第11选区共和党籍国会议员邓诺允（Dan Donovan），还是竞选连任的第22选区共和党籍州参议员高顿（Martin Golden）出来拉票，周围都有相当数量和比例的华裔中产阶层群体〝啦啦队〞，大多是第一代中国移民。
 <br/>
 <br/>
 杨汉青居美30年，在纽约开电召车公司，原来是民主党，现在转为了共和党。他说，这些年来看到民主党宣传扩大政府职能，而很多机构的工作重复，职能重复、工作重合，到处花钱，将酒店租用来做临时收容所，安置单身流浪汉等等，让他产生强烈的反感，〝我们来美国，每个人都希望勤劳，但民主党的政策催生了不想多工作的福利懒汉，搞得美国社会风气很不好，我非常生气。因此一定要支持共和党。〞
 <br/>
 <br/>
 记者采访几名华人助选团义工，大家的看法很类似。当谈到为何支持共和党，许多人开始语速变快，声音急促地说，〝在纽约州参议院，反对白思豪违宪歧视华人的全是共和党议员，支持白思豪的全是民主党议员，如果让民主党控制参议院，可想而知会发生什么，支持民主党所有的议案，搞种族配额，搞政治正确，支持大麻合法化，引进庇护大量的西语裔无证移民，纳税人交更多的税提供福利……万一让民主党一党专政，美国就要变成共产国家！〞
 <br/>
 <br/>
 华人社群特别关注几个议题，包括教育跟移民法例等，许多华人助选团义工都认为，共和党的政策更加合符华人的利益。他们将民主党与共和党的根本分歧点整理出表格，认为共和党的财政哲学就是小政府、低税收、有信念信仰，强调家庭、自食其力、对社会比较保守，一些华人对此深感认同。
 <br/>
 <br/>
 在极度尖锐对峙的美国政治氛围下，今年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 选情激烈，改选的众议员、部分参议员与州长，将对总统余下任期的执政，甚至两年后的大选产生不小的影响。在共和党参议员高顿选区，华裔助选团非常有组织，他们打出〝支持高顿，就是保卫我们自己和华人孩子们的未来〞口号，在网页上、在社交媒体上广泛宣传，点燃了更多华人的参政、议政热情。
 <br/>
 <br/>
 就像法拉盛华人（工商）促进会总干事杜彼得昨天给大纪元的投稿中所言，他说，华人在纽约不只是为保护SHSAT（特殊高中考试录取制）而战，更为了自己的〝价值〞。
 <br/>
 <br/>
