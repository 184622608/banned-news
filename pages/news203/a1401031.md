### 川普：合法入境是底线 暴力非法越界者必须驱逐
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】日前，美墨边界曾有数百移民试图强行冲过边界进入美国境内事件，为阻止这种以暴力方式非法穿越边界的行为，美方边境人员被迫使用了催泪弹。周一，美国总统川普（特朗普）告诫墨西哥政府，如果墨方不驱逐那些包括冷血罪犯的凶悍移民群体，美方将长久封闭边界关卡。
 <br/>
 <br/>
 本周一（26日），有媒体记者针对此前一天美国边防人员发射
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_催泪瓦斯.html" target="_blank">
  催泪瓦斯
 </a>
 和橡皮子弹以阻挡数百移民试图强行冲入美国境内的行为向川普提问，川普回答说：〝他们必须用 (催泪瓦斯)，因为一些非常凶悍的人正冲向他们。这是底线：除非合法入境，没有人能进入我们国家。〞
 <br/>
 <br/>
 川普同时否认了外界有关边境人员用催泪瓦斯对付儿童的传言，他说：〝我们没有这样做。〞
 <br/>
 <br/>
 同一天，川普还发推文告诫墨西哥政府，〝驱逐那些包括冷血罪犯的移民群体，否则会看到这些对经济至为重要的通行管道永久封闭。〞他同时在推文中向美国国会国会喊话〝国会，提供筑墙经费！〞
 <br/>
 <center>

   </font>

 </center>
 <br/>
 似乎是对川普警告的一种回应，墨西哥政府日前誓言要驱逐大约500名试图〝以暴力方式非法穿越边界〞进入美国的人。
 <br/>
 <br/>
 当地时间11月25日，数百滞留在墨西哥蒂华纳的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 进行了抗议活动，之后有部分人试图
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强行闯关.html" target="_blank">
  强行闯关
 </a>
 进入美国境内，现场一度乱成一团，美方执法人员动用了催泪瓦斯来驱散人群，并临时关闭了圣伊西德罗入境口。不过，美墨边界的关卡口岸26日又恢复了开放。
 <br/>
 <br/>
 大篷车移民强行闯关事件发生后，国土安全部长尼尔森曾对外证实，有边境人员被移民投掷的东西击中。她表示，那些采取了危险行动的人〝不符合和平寻求庇护〞的要求，美国不能容忍这种〝无法无天〞的行为。尼尔森并誓言要制裁任何〝摧毁联邦产业、危害我们边界人员，或破坏我们主权的人〞。
 <br/>
 <br/>
