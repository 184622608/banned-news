### APEC召开前夕 川普白宫谈美中贸易
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 峰会本周末进行，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 将发表演说，会触及中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 计划。而周五，总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）透露，中共在贸易上有让步迹象。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：〝中国想要（和美国）达成协议。他们发来一份他们愿意做的事情的清单，一个很长的清单，但我还是不能接受。〞
 <br/>
 <br/>
 川普认为这份涉及142项的清单已经相当全面，但还是漏了四五项重要问题，川普没有透露细节，但表示会要求中方把漏掉的加进去。
 <br/>
 <br/>
 中共给美国商品设立关税和非关税壁垒，以及知识产权盗窃等不公平竞争行为，导致川普政府接连施加惩罚性关税。
 <br/>
 <br/>
 美国总统川普：〝我们已对价值2500亿美元的（中国）商品征收关税，如果我们愿意的话还有2670亿美元（可征税）。我们可能不必这样做，（因为）中国愿达成协议。〞
 <br/>
 <br/>
 与此同时，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 也于当地的周五晚，与夫人抵达澳大利亚凯恩斯。虽然亚太经合组织峰会将在邻近国家巴布亚新几内亚举行。但出于安全考虑，多国领袖都选择下榻在澳大利亚。
 <br/>
 <br/>
 彭斯将在峰会上发表重要讲话，具体阐述中共的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞计画如何让有关国家陷入债务困境。
 <br/>
 <br/>
 中国的习近平则于周四抵达巴布亚新几内亚。拥有美中俄日以及台湾的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 21名成员，要在本周末进行连续两天的会议。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
