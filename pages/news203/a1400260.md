### 路透：中国民企暗挺川普 盼市场更自由开放
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】虽然中国商界表面上和中共官方保持一致，对美国启动贸易战表示〝谴责〞，但路透文章披露，很多中国私企高管私下里都支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 向北京施压，希望贸易战能给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国民企.html" target="_blank">
  中国民企
 </a>
 带来更自由、宽松的生存环境。
 <br/>
 <br/>
 这篇由路透专栏作家考克斯（Rob Cox）21日发表的评论文章指出，中国民营企业家们支持贸易战，是因为他们期盼市场更自由开放，但北京却缺乏这方面改革的动力。
 <br/>
 <br/>
 文章说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 对中方实施的惩罚措施和开出的贸易要求，大多数都未必直接对中国企业有利，但川普几乎可说是促使北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开放市场.html" target="_blank">
  开放市场
 </a>
 的唯一压力来源。
 <br/>
 <br/>
 文章指，北京的政治经济政策都有左转倾向，包括强化对企业的控制，铁腕干预经济领域。当局宣传的所谓经济改革也陷入困境，北京已没多少动力去耗费政治资本推动棘手的改革。
 <br/>
 <br/>
 因此，中国大陆许多民间企业不再看好北京的经济政策，转而寄希望于川普的贸易大刀能够迫使北京实施他们本不愿意采取的市场开放措施，这些措施将提升民营企业的的竞争力，特别是跟国企的竞争。
 <br/>
 <br/>
 文章指出，事实上，贸易战开打以来，北京也确实宣布了少数开放市场的措施，包括对外国车商和金融业进入中国放宽限制等。
 <br/>
 <br/>
 虽然北京一再宣称这是中方〝改革的既定步骤〞，但显然都是美国施压的结果。
 <br/>
 <br/>
 另外，北京高层近日高调表态民企是〝自己人〞，在中共各部门和全国各地掀起了一场〝支持民企〞的政治运动。北京相继推出减税和放开贷款融资的措施，改善了民企的经营环境。
 <br/>
 <br/>
 外界认为，贸易战让北京陷入经济危局，当局不得不给民企放宽生路，以避免经济快速崩溃。
 <br/>
 <br/>
 而事实上，北京的经济政策早就开始左转。薄熙来倒台前在重庆〝唱红打黑〞，公然抢劫民企财产，当时得到中共高层相当一部分人的公开支持。多年来，北京的经济政策都偏向国企，不断压缩私营经济的生存空间，令外界产生北京将重回〝计划经济〞的担忧。
 <br/>
 <br/>
 近日，一篇主张〝私营经济应该离场〞的博文掀起民间新一轮恐慌。随后北京高层接连表态〝一如既往地支持私营经济〞。但是，这仍然无法打消民企的忧惧。
 <br/>
 <br/>
 日前，浙江省委书记车俊一句〝要把民营企业家搞得香香的，把民营经济做得壮壮的〞，再次引发舆论关注。
 <br/>
 <br/>
 网友纷纷评论，这句话马上让人联想起〝烤乳猪〞，洩露了中共政府要把民企〝养肥了再杀〞的真实心态。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
