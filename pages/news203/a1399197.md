### 立法保护穆勒？参议员克鲁兹：做法违宪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】共和党国会参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泰德．克鲁兹.html" target="_blank">
  泰德．克鲁兹
 </a>
 日前在接受美国媒体采访时表示，国会不应透过立法来保护特别检察官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 的调查，而早前禁止穆勒被撤职的法案其实已经违宪。参院多数党领袖麦康诺也表态称，立法来保护穆勒是不必要的。
 <br/>
 <br/>
 泰德．鲁兹(Ted Cruz) 本月11日在哥伦比亚广播公司(CBS)〝面对国家〞(Face The Nation)节目中受访时表达了上述观点，他并直言：〝司法委员会通过了一项法案，试图使特别检察官不可能被撤职。我认为该法案违宪，与宪法第二条不一致。〞
 <br/>
 <br/>
 此前，司法部长塞申斯（Jeff Sessions）应川普总统的要求于11月7日辞职。同日，川普宣布委派惠特克(Matt Whitaker)出任代理司法部长。司法部随即声明，副部长罗森斯坦（Rod Rosenstein）不再负责监督通俄门调查，转而由惠特克接替这项工作。
 <br/>
 <br/>
 由于惠特克此前曾经公开斥责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 的调查已跨越授权范围逼近〝红线〞，民主党便以此为据开始向司法部道德主管施压，企图让他出面提议惠特克回避通俄门调查。支持惠特克的共和党议员则认为，民主党的意图是想阻止忠于川普的惠特克去监督“通俄门”的调查。
 <br/>
 <br/>
 亚利桑纳州共和党参议员佛莱克(Jeff Flake)和康州民主党参议员库恩斯(Chris Coons)则提出了让总统不能开除穆勒的法案。参院少数党领袖舒默11日也声称，如果惠特克不回避对俄罗斯的调查，他将寻求在国会必须通过的法案中，列入保护穆勒调查团队的条文。
 <br/>
 <br/>
 对此，参院多数党领袖麦康诺(Mitch McConnell)公开表态称，通过立法来保护穆勒是不必要的。
 <br/>
 <br/>
 共和党参议员葛理汉(Lindsey Graham) 11日也在接受媒体采访时表态称，他认为惠特克不必回避穆勒的通俄案调查。
 <br/>
 <br/>
 葛理汉在〝面对国家〞的节目中对主持人玛格丽特‧布伦南(Margaret Brennan)说：〝我昨天和惠特克谈过，我们会安排时间在下周见面，我相信他的任命合法，并认为他不必回避。我认为穆勒的调查会获准继续进行，直到有坚实的结论。惠特克不会对穆勒施加任何政治影响，要他做调查以外的事，我确信穆勒可以不受干预的完成工作。〞
 <br/>
 <br/>
 葛理汉接着又补充说，他相信总统将于〝明年初〞任命一位司法部长，而惠特克的代理职务只有几个月，所以惠特克根本不必回避穆勒的通俄案调查。
 <br/>
 <br/>
 据公开的资讯，今年4月26日，美国参议院司法委员会曾通过一项法案，为设在司法部的特别检察官办公室提供了额外的保护，规定特别检察官只能由司法部长或指定人员以〝不当行为、失职、无行为能力，利益冲突或其他正当理由〞解雇。
 <br/>
 <br/>
 舆论认为，主要由民主党议员提出的该法案，旨在防止负责调查俄罗斯干涉2016年美国大选的特别检察官穆勒被任意解职。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
