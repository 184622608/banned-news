### 蓬佩奥与朝鲜会谈推迟 原因未透露
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国务院.html" target="_blank">
  美国国务院
 </a>
 宣布，推迟原定于周四举行的、国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 与朝鲜外长的会谈。韩国政府表示，会议推迟不影响第二次美朝峰会。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国务院.html" target="_blank">
  美国国务院
 </a>
 周三声明：〝将在我们双方时间允许的情况下〞，重新安排会议。
 <br/>
 <br/>
 声明还说：〝正在进行的通话将继续进行。美国仍将专注于，履行川普总统和金正恩六月新加坡峰会上达成的承诺。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 原定周四在纽约，会见朝鲜高级官员金英彻，商讨美朝第二次峰会，及推动朝鲜去核化进程。
 <br/>
 <br/>
 韩国青瓦台发言人金宜谦表示，会议推迟不会影响美朝第二次峰会。
 <br/>
 <br/>
 有消息人士称，会议推迟表明，朝鲜不愿配合美国的期望，比如在第二次峰会前，开放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核试验.html" target="_blank">
  核试验
 </a>
 场所用来检查。朝鲜则希望美国作出妥协，比如放宽部分惩罚性制裁。
 <br/>
 <br/>
 白宫一直坚持，除非朝鲜放弃核武器，并通过国际组织复核，否则不会放弃制裁。
 <br/>
 <br/>
 新唐人记者毕心慈、张琪综合报导
</div>
