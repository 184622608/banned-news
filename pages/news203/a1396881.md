### 卡瓦诺风波未平 美参议员吁调查指控者是否涉伪证
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】对于色情女星丹尼尔斯的律师阿维纳蒂，上个月与他的客户斯沃特尼克，指控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院大法官.html" target="_blank">
  最高法院大法官
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 30多年前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵.html" target="_blank">
  性侵
 </a>
 案，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 司法委员会主席格拉斯利怀疑其作伪证，周四（25日）呼吁司法部对其启动刑事调查。
 <br/>
 <br/>
 美国之音报导，共和党议员格拉斯利（Chuck Grassley）希望检察官能够查明朱莉・斯沃特尼克（Julie Swetnick），以及她的律师迈克尔．阿维纳蒂（Michael Avenatti）在上月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 （Brett Kavanaugh）听证会期间是否向国会作出了虚假陈述。
 <br/>
 <br/>
 格拉斯利还希望司法部调查这两人是否试图妨碍他领导的司法委员会对斯沃特尼克和其他人对卡瓦诺所提指控的调查。他说:〝要让法律起作用，我们不能对潜在的违反行为视而不见。〞
 <br/>
 <br/>
 上个月，在卡瓦诺的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 确认听证进程期间，斯沃特尼克在宣誓不作伪证的陈述中称，当年在有卡瓦诺参加的一个派对上，她遭到了轮奸。斯沃特尼克说，这一事情发生在他们1980年代上高中的时候。卡瓦诺否认这一指控。
 <br/>
 <br/>
 斯沃特尼克声称，在1981至1982年间，她曾参加过10多次有卡瓦诺在场的聚会。在这些聚会上，他目睹了卡瓦诺及其高中好友查治（Mark Judge）等人努力使女孩们喝醉并失去判断力，然后遭到很多男孩轮奸。
 <br/>
 <br/>
 美国国会共和党参议员格雷厄姆（Lindsey Graham）9月26日发表声明，就斯沃特尼克的指控做出回应。他认定指控者是在荒谬抹黑，同时警示共和党人不要被其戏弄。
 <br/>
 <br/>
 格雷厄姆发表一系列推文称，他很难相信有人会继续参加这种聚会——在两年中举行过10多次，而女孩们经常被轮奸却不报案。一个有理性的人为什么会围着这样的人转？为什么有人会把她们自己及朋友置于危险之中？难道没有责任去警告他人吗？
 <br/>
 <br/>
 格雷厄姆表示，他非常推崇允许人们表达心声，但是，他不打算被戏弄，也不打算让自己的智力被阿维纳蒂这样的世人侮辱，他更不会参与这种不顾可信度的大规模〝品格暗杀〞行动。
 <br/>
 <br/>
 格雷厄姆当天还声明称，斯沃特尼克等人的指控与卡瓦诺的生活完全不符。所有与卡瓦诺一起工作或为他工作过的女子都赞许他的品行。他是一个正派的男人，过着体面的生活，但现在正被阿维纳蒂这类人抹黑。
 <br/>
 <br/>
 美国联邦调查局(FBI)10月初结束了对卡瓦诺的第七次调查，所有针对卡瓦诺的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵.html" target="_blank">
  性侵
 </a>
 指控都没有被证实，卡瓦诺的提名投票在10月6日通过，为持续数周的风波画上句号。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
