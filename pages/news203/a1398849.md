### 两天摧毁六千房屋   北加州火情危急
------------------------

<div class="wysiwyg">
 【新唐人2018年11月10日讯】周四开始，在美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北加州.html" target="_blank">
  北加州
 </a>
 爆发的〝营地〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火.html" target="_blank">
  野火
 </a>
 ，继续快速蔓延，仅两天时间，就摧毁了超过6千栋的房屋。目前已有，至少9人死亡，35人失踪。
 <br/>
 <br/>
 一天以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北加州.html" target="_blank">
  北加州
 </a>
 的〝营地〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火.html" target="_blank">
  野火
 </a>
 迅速蔓延。起火48小时内，就已经覆盖了140平方英里的面积，摧毁超过6,700栋建筑，成为加州历史上最具破坏力的野火。
 <br/>
 <br/>
 当局已派出超过3千名消防员灭火，但火势扩散的速度太快，仍然只有5%的火势受到控制。
 <br/>
 <br/>
 巴特县治安官霍尼表示，发生了最坏的情形。火势蔓延之快，甚至来不及通知每户人家疏散。
 <br/>
 <br/>
 巴特县治安官霍尼：〝一旦看出哪个社区有危险，我们会立即开始发布疏散命令，通过我们现有的机制实施。但我们无法一家一家的敲门通知，因为火势蔓延的太快了。〞
 <br/>
 <br/>
 当局表示，在周五一天就救出了大量民众，同时出动直升机，在临近的马格利亚社区救出5名被困民众。
 <br/>
 <br/>
 巴特县治安官霍尼：〝灾情开始迅速扩大，我们救出数以万计的民众，将火情通知给他们。〞
 <br/>
 <br/>
 目前，在北加州有大约5万2千名民众被疏散。而在南加州，灾情同样不容乐观。目前，野火已经延烧至洛杉矶西部的马里布景区，许多好莱坞明星居住于此。
 <br/>
 <br/>
 当地官员在推特上发布警告说，火势目前处于失控状态，已经进入马里布的居民区，所有人必须立即疏散。
 <br/>
 <br/>
 许多名人在社交媒体上发文，表示家园面临被野火摧毁，他们已经疏散至安全地区。周三刚刚发生枪击事件的千橡市也接到了疏散命令。
 <br/>
 <br/>
 目前，南、北加州合计共有25万人遭到疏散。气象学家警告，强风和干燥天气导致的危险火势，可能要一直持续到下周。
 <br/>
 <br/>
 新唐人记者林宇综合报导
