### 美媒曝白宫幕僚长曾与共官爆肢体冲突
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 令中美关系逐渐迈向全面
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对抗.html" target="_blank">
  对抗
 </a>
 、全面新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冷战.html" target="_blank">
  冷战
 </a>
 时代。有美媒刊文称，川普（特朗普）对华政策愈加强硬，其团队中的鹰派人员也占据了上风，在去年秋季，白宫幕僚长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凯利.html" target="_blank">
  凯利
 </a>
 曾与一名中共官员发生肢体冲突。
 <br/>
 <br/>
 美国总统川普正在对中共采取史上最强硬政策，他的幕僚中有诸多鹰派人物，包括纳瓦罗、莱特希泽、博尔顿、白邦瑞等等。然而，美国政界为什么会对中共变得日益强硬？
 <br/>
 <br/>
 《华尔街日报》引述知情人士的话透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凯利.html" target="_blank">
  凯利
 </a>
 对中共的看法与美国参谋长联席会议主席邓福德（Joseph Dunford）一样，也是受到此前经历的影响，开始对中共的态度转向强硬。
 <br/>
 <br/>
 据报，邓福德2017年曾为了促成美中两国，建立一个正式的军事沟通机制而访问北京。之后他对这一点有了更清楚的认识。当时一位助手放在酒店房间的平板电脑被人篡改，这动摇了美国军方对与中共往来的固有看法。
 <br/>
 <br/>
 报导说，在川普去年访问北京的时候，白宫幕僚长凯利跟中共官员发生肢体冲突，并且拒绝中方的道歉。
 <br/>
 <br/>
 当时这名中共官员企图获取〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核足球.html" target="_blank">
  核足球
 </a>
 〞，一个必须随时在总统身边的45磅重铝制公文包。这个公文包装有总统如何下达核攻击命令的信息和指南。当时川普身边一名军官将这包握在手上。
 <br/>
 <br/>
 2月中，美国媒体Axios也曾引述多位消息人士透露，去年11月9日，一名美国军方助理携带着，总统发射核武认证码的黑色公事包，正准备跟随川普进入北京大会堂时，被中方保镖拦截。
 <br/>
 <br/>
 同行的白宫幕僚长凯利得知情况后，走过来告诉这名军事助理及周围美国官员〝我们进去〞。
 <br/>
 <br/>
 不料，一名中方保镖突然抓住凯利的手臂，试图阻止他前行。凯利推开他继续走进大会堂。这时，随行的美国特勤局人员冲上去，把这名中方保镖按倒在地。
 <br/>
 <br/>
 据悉，事后一名中共高级安全官员，就此事对川普团队表示道歉，美国政府也对这场冲突采取了冷处理。
 <br/>
 <br/>
 凯利告诉同事，他拒绝接受道歉，除非中共高级官员来到华盛顿，站在美国国旗下表示悔悟。
 <br/>
 <br/>
 据悉，引发冲突的是被外界昵称为〝足球〞(football)的核密码箱约20公斤重，装有与五角大楼的指挥中心联络并启用核武的设备。美国总统出访时，都会有总统的军事助理提着〝足球〞贴身随行，这是惯例。
 <br/>
 <br/>
 美媒报导说，美国官员在川普访华之前，美方已向中方通报了美国安保惯例，以便中方了解并遵守。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：范铭）
