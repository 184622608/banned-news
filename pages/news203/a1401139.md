### 促成美墨加协定 墨国将颁雄鹰勋章给川普女婿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】因美国总统川普（特朗普）的女婿、白宫幕僚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库什纳.html" target="_blank">
  库什纳
 </a>
 ，为美、墨、加之间达成新的贸易协议做出了重大贡献，墨西哥政府周二（11月27日）表示，将授予其阿兹特克雄鹰勋章，该勋章是墨西哥授予外国人的最高荣誉。
 <br/>
 <br/>
 据墨西哥媒体Reforma报导，墨国总统潘尼亚尼托（Enrique Pena Nieto）将于30日在阿根廷举行的G20高峰会上，给库许纳（Jared Kushner）颁发阿兹特克雄鹰勋章（Aztec Eagle）。届时美、墨、加三国领导人将签署新版NAFTA，即美—墨—加协定（USMCA）。
 <br/>
 <br/>
 USMCA挽回了由美加墨三国参与的NAFTA的格局，墨国外交部赞扬USMCA是〝国家经济稳定和数百万墨西哥劳工安定的泉源〞。
 <br/>
 <br/>
 墨西哥外交部在一份声明中说，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库什纳.html" target="_blank">
  库什纳
 </a>
 在整个过程中发挥了重要作用，他的参与对于展开《北美自由贸易协定》（NAFTA）重新谈判至关重要，避免了美国单方面退出，他的持续及有效参与是取得成果的关键。〞
 <br/>
 <br/>
 具有24年历史的NAFTA，因让美国制造业工作流失，川普曾批评其为〝史上最糟协议〞，并要求对协议进行重大修改。但是，美墨和美加有关NAFTA的重新谈判，一度曾陷于停滞，甚至破裂，是库什纳力促谈判继续进行。
 <br/>
 <br/>
 路透社此前引述多名消息人士的话透露，库什纳与墨西哥外交部长维德加雷（Luis Videgaray）和加国总理杜鲁道（Justin Trudeau）的助手们的友谊，化解了几次谈判破裂所带来的危局，使得美国和加拿大最终达成NAFTA新协议。这被外界认为是川普的缓和与盟友贸易纠纷的努力取得重大进展，将让美国能够集中力量加强对北京的贸易施压。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
