### 派美军防非法越境 川普称士兵不会开枪
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军士兵.html" target="_blank">
  美军士兵
 </a>
 已陆续抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 ，以防止中美洲大篷车移民非法闯进美国领土。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统表示，即便到时难民有过激行为，美军也不会开枪。
 <br/>
 <br/>
 目前，来自中美洲的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 ，一部分已抵达墨西哥中部，仍在向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 行进。
 <br/>
 <br/>
 此前，发生过大篷车难民们扔石块打伤墨西哥边防人员的事件，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军士兵.html" target="_blank">
  美军士兵
 </a>
 若遇到这种情况不会开枪。
 <br/>
 <br/>
 川普总统：〝不，他们（军人们）不会开枪。但我不希望这些非法入境者扔石头。〞
 <br/>
 <br/>
 川普也指出，如果他们那样做，也要付出代价。
 <br/>
 <br/>
 川普总统：〝他们对墨西哥军人的所作所为是羞耻的。他们用石块打他们，有些人受伤很重，而且他们扔石头打边境人员的脸。（如果）他们对我们对人这样做，他们将被逮捕。他们将被逮捕关押很长时间。〞
 <br/>
 <br/>
 目前，派往边境的美军士兵已抵达美墨边境附近，他们有来自空降师、军警大队的士兵，也有参加过反恐、阿富汗战的工程兵等。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
