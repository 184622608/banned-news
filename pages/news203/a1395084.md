### 迈克尔横扫 佛州首当其冲 F-22基地建物几乎全毁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 迈克尔挟带高达时速240公里强风威力，10日下午在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥海滩.html" target="_blank">
  墨西哥海滩
 </a>
 登陆，造成当地一处海岸社区房舍几乎全毁，而拥有全美最多F-22隐形战机部署的廷德尔空军基地遭到正面吹袭，一架展览用的战机被吹翻，基地广泛受到灾难性的破坏。
 <br/>
 <br/>
 对于自然地形宛如一把锅柄的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 锅柄地带(Florida Panhandle)来说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 迈克尔堪称是史上威力最强的飓风。首当其冲的佛州巴拿马市(Panama City)南方约40哩的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥海滩.html" target="_blank">
  墨西哥海滩
 </a>
 ，正是迈克尔登陆的地点。
 <br/>
 <br/>
 美国有线电视新闻网(CNN)11日清晨出动直升机，从空中鸟瞰飓风肆虐后留下满目疮痍的景象，主播布鲁克‧巴德温(Brooke Baldwin) 从直升机上表示，受到飓风迈克尔影响，通往墨西哥海滩的道路目前已经根本无法通行，当地社区某些地方着火，一栋栋房屋变得破碎不堪。
 <br/>
 <br/>
 佛州坦帕湾时报(Tampa Bay Times)报导，在墨西哥海滩的〝房子几乎都遭到彻底毁灭，飓风过后如今还能看得到的，只剩冰箱跟马桶而已。〞
 <br/>
 <br/>
 而拥有55架F-22隐形战机部署的廷德尔空军基地（Tyndall Air Force Base）则受到广泛及灾难性的破坏，虽然早在飓风来袭前，绝大部分战机及另外17架教练机早已飞往俄亥俄州的空军基地暂避。
 <br/>
 <br/>
 但基地所有建筑物几乎出现不同程度的损毁，而有少部分未能飞行的F-22，以及多架F-16战机需停泊在机库内避风。
 <br/>
 <br/>
 从视频中可见，基地损毁情况很严重，很多建筑物屋顶被吹走，树木、电杆倾倒，供水及供电中断。在一个屋顶被吹毁的大型机库内，可看到一架F-16战机内里布满残骸，一架在基地入口展览的F-15战机被吹倒。
 <br/>
 <br/>
 基地指挥官表示，很多建筑物〝完全损失〞，还不知基地何时恢复运作。
 <br/>
 <br/>
 目前，飓风迈克尔已造成至少6人死亡，在袭击佛州和乔治亚州后，已降为热带风暴，继续向北移动。
