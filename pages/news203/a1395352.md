### 里根总统图书馆 全息影像让人身临其境
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】在美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州西米谷.html" target="_blank">
  加州西米谷
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根总统图书馆.html" target="_blank">
  里根总统图书馆
 </a>
 ，首次应用了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全息影像.html" target="_blank">
  全息影像
 </a>
 技术，通过里根总统的三维全息影像，将参观者带回历史现场，给游人带来身临其境的感受。
 <br/>
 <br/>
 里根总统被认为是美国历史上一位伟大的总统。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全息影像.html" target="_blank">
  全息影像
 </a>
 三个场景使用的音频取自里根1981年-1989年担任总统期间的真实讲话。
 <br/>
 <br/>
 一个场景显示里根在1994年的竞选活动中在火车上向人群挥手。另外两个描绘了他在椭圆形办公室和他的加利福尼亚农场的场景。里根总统于2004年逝世。
 <br/>
 <br/>
 罗纳德-里根总统基金会和研究所执行主任约翰-修普希（JOHN HEUBUSCH）：〝我认为人们会被这个令人难以置信的影像所吸引。这将是第一次，如果你从来没有和里根总统在一个房间里，这会让你感到你就和他一起站在那。他以三维全息形式站在那里，如此逼真，令人惊讶。所以这是一个让人们参观的有趣展览。〞
 <br/>
 <br/>
 罗纳德-里根总统基金会和研究所执行主任约翰-修普希（JOHN HEUBUSCH）：〝我们尽量选择各种场景，总统欢迎你走进你期待见到他的椭圆形办公室，还有被称为他的西部白宫的农场，他喜欢在那里放松。最后我们也想把他放到1984年的竞选状态中，他因令人难以置信的人气，即将赢得压倒性的选举。〞
 <br/>
 <br/>
 罗纳德-里根总统基金会和研究所执行主任约翰-修普希（JOHN HEUBUSCH）：〝对于每一个场景，我们都有一个与里根总统身形相同的演员（做动作模仿），因为里根总已经不再试了，我们不得用矽胶重塑他的形像，然后我们必须让那个模型、那个脸动起来，那就是你看到的，一个三位数字化的里根总统形像。〞
 <br/>
 <br/>
 里根总统的全息图是使用美国全系影像公司（Hologram USA）的技术创造的。
 <br/>
 <br/>
 美国全系影像公司高级副总裁大卫-纳斯鲍姆（DAVID NUSSBAUM）：〝博物馆的参观者将与世界上最知名的人物之一几乎鼻子对鼻子（的接触），所有要做到信息准确，然后将这些信息呈现在这个空间，看起来他真的站在那里，绝不能搞砸了。我们必须得做好。〞
 <br/>
 <br/>
 新唐人记者徐哲、雨薇综合报导
</div>
