### 共和党抓住机会 参院再确认15名川普提名法官
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】周四（10月11日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 又
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_确认.html" target="_blank">
  确认
 </a>
 了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_15名.html" target="_blank">
  15名
 </a>
 川普总统提名的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 ，解决了大约1/3的积压案。这是参议院共和党继确认最高法院大法官卡瓦诺（Brett Kavanaugh）之后取得的又一重大成果。
 <br/>
 <br/>
 周四晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 又批准了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_15名.html" target="_blank">
  15名
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普提名.html" target="_blank">
  川普提名
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 。共和党抓住数名民主党参议员需返乡竞选连任的机会，迫使民主党领导人达成协议，在中期选举到来前完成了一部分积压已久的工作。
 <br/>
 <br/>
 在获批的15名法官中， 3名为巡回上诉法院法官，12名为地区法院法官。
 <br/>
 <br/>
 很多投票结果仍接近沿党派划线，还有一些是通过口头表决批准的。这是继上周末卡瓦诺的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_确认.html" target="_blank">
  确认
 </a>
 后参议院首次批准新法官。
 <br/>
 <br/>
 这次投票是在特殊背景下完成的。因为有6名民主党参议员、特别是红州的参议员，迫于压力要返回家乡、在中期选举来临前的最后阶段全力竞选。
 <br/>
 <br/>
 参议院多数党领袖麦康奈尔（Mitch McConnell）抓住这一机会，迫使长期以来一直在抵制和拖延的民主党答应了先确认一批被提名的法官。
 <br/>
 <br/>
 一些共和党人本来希望参议员们都留在华府，以便使49名已进入全院表决阶段的被提名法官全部都有机会在选举前得到确认。但最终，仅能达成15名的协议。其实这已是能实际确认的最高数量，即使在接下来的几周中参议员把全部时间都投进去，民主党能够同意的数量也就这么多了。
 <br/>
 <br/>
 即便如此，自由派活动人士仍对民主党领导人同意投票感到愤怒。
 <br/>
 <br/>
 而保守派活动家虽然希望能够确认更多法官，但对于有15人得到批准还是很欣慰。
 <br/>
 <br/>
 获得确认的3名新巡回法院法官分别是：美国第3巡回法院法官波特（David Porter），以50:45票通过；美国第9巡回法院法官尼尔森（Ryan Douglas Nelson），通过票数为51:44；美国第2巡回法院法官沙利文（Richard J. Sullivan），以79:16的高票通过表决。
 <br/>
 <br/>
 就在对上述15名法官进行确认投票前，参议院司法委员会又批准了8名法官提名人进入全院表决程序。这意味着，还有34名法官提名人等待确认，但是需要等到11月选举结束、参议院回归工作以后了。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
