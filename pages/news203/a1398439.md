### 中期选举结束 大篷车移民嘎然止步逾百人消失
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 前夕，拉美〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 〞大举向着美墨边境进发，一度让边境气氛十分紧张。随着中期选举结束，向美进军的移民〝大篷车〞不走了，更有近百人扔下车在墨西哥〝就地消失〞。这种蹊跷的现象，令有关〝民主党势力鼓动非法移民搅局中期选举〞的疑云更为浓厚。
 <br/>
 <br/>
 11月6日，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 投票与统计程序完成，民主党重新夺回了众议院的多数席位，共和党则增长了参议院席位，成功保住了对参院的控制权。
 <br/>
 <br/>
 随着中期选举落幕，一度气势汹汹的〝大篷车〞移民大军也在美墨边境嘎然止步。
 <br/>
 <br/>
 据《墨西哥太阳报》11月7日的报导，约4500名洪图拉斯移民选择在墨西哥逗留，有人已经放弃去美国的念头，表示〝想留在墨西哥〞，或是〝想去加拿大〞。
 <br/>
 <br/>
 墨西哥《先锋报》和《至上报》也报导了类似的情况，指这些搭乘大篷车长途跋涉奔向美国的移民大军内部，现在对〝目的地的选择〞出现了严重的分歧。其中有移民表示，只要到达墨西哥，就算〝成功〞了；有人明确表示，〝墨西哥机会很多，我打算留在这里。〞
 <br/>
 <br/>
 《先锋报》更报导了一个蹊跷的现象，自11月3日起，陆续有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 在抵达墨西哥后就〝神秘失踪〞。尤其在11月7日（美国中期选举刚刚结束），就有80名移民扔下2辆大篷车〝就地消失〞，不知所踪。像这样的奇怪现象，据统计总人数已达100人。报导称，〝这令墨西哥当局很困扰。〞
 <br/>
 <br/>
 墨西哥移民部门的一位官员在接受西班牙埃菲社采访时也谈到了这种现象，他抱怨说：〝这些人到底想不想继续北上，还是想留下来，最好能通知下我们。〞
 <br/>
 <br/>
 综合墨西哥当地媒体的报导，目前大约有4500名移民在墨西哥城的一个体育场内扎营。移民代表11月7日对墨西哥有关机构表示，他们需要48小时来进行一下内部讨论，以便决定这些移民的最终去向。
 <br/>
 <br/>
 当地时间8日，移民代表将和墨西哥〝准总统〞奥夫拉多尔（刚在大选中获胜）会谈。
 <br/>
 <br/>
 洪都拉斯总统赫尔南德斯11月6日宣布，已有大约6500名大篷车移民返回国内。
 <br/>
 <br/>
 早前当大篷车移民〝大军〞正向美国边境进发之际，美国国内有声音指责这一波非法移民大潮是支持民主党的金主，为了搅局中期选举，不惜出钱资助这些拉美的非法移民北上，意图乘机制造事端，以便给共和党以及川普政府制造压力，但这个说法尚没有确凿的证据来证实。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：东野）
