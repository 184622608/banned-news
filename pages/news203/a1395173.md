### 飓风迈克尔席卷美国南部致13死 度假小镇秒变废墟　
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】4级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风迈克尔.html" target="_blank">
  飓风迈克尔
 </a>
 近日吹袭美国东南部几个州并引发强风和暴雨，迄今为止已发现有至少13人因这次风灾而死亡，其中包括一名在弗吉尼亚州殉职的消防员。飓风登陆点附近的著名度假胜地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥海滩.html" target="_blank">
  墨西哥海滩
 </a>
 （Mexico Beach）遭受迈克尔肆虐后已面目全非，几乎成为一片废墟。
 <br/>
 <br/>
 当地时间10月10日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风迈克尔.html" target="_blank">
  飓风迈克尔
 </a>
 在美国佛罗里达州海边小城
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥海滩.html" target="_blank">
  墨西哥海滩
 </a>
 附近登陆，最大持续风速为249公里/小时。之后，迈克尔继续向东北方向移动，继续穿越北卡罗来纳州中部和东部，于周四晚上横扫弗吉尼亚州东南部。目前，已降级为热带风暴的迈克尔进入西大西洋。
 <br/>
 <br/>
 迈克尔席卷而过后，美国气象频道新闻航拍的画面上可看到，曾经十分美丽的度假小镇墨西哥海滩已几乎变成了一片废墟。原先排列停泊在海边的游艇，像废弃的玩具一样支离破碎地堆叠在一起。目前救援人员已进入该镇评估受损情况。

 在飓风来临之前，当地政府早前已下令当地民众疏散撤离，不过全镇仍有285人拒绝离开。飓风过后，成千上万的美国国民警卫队、执法人员、医护人员和电力抢修人员进入受灾社区对受困的民众进行救援。
 <br/>
 <br/>
 沿海城镇巴拿马城受灾情况也比较严重。镇上到处都是从海滩吹来的沙砾、各种碎片、电线，还有高大的树木倒伏在街道中央。
 <br/>
 <br/>
 有媒体报导称，巴拿马城有一座在去年飓风艾玛吹袭时曾作为避难所的室内球场，这次遭迈克尔袭击后，其外墙已被完全摧毁。
 <br/>

 <br/>
 <br/>
 威廉姆・布洛克・朗表示，另一个优先事项是恢复佛罗里达州、阿拉巴马州、佐治亚州和南卡罗来纳州近80万户家庭和企业的电力供应。
