### G20峰会开幕  川普会晤阿根廷总统
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】周五，备受瞩目的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 在阿根廷首都布宜诺斯艾利斯开幕。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ，首先与东道主，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷总统.html" target="_blank">
  阿根廷总统
 </a>
 马克里举行了双边会谈。
 <br/>
 <br/>
 周四晚间，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 与夫人抵达阿根廷，开启了48小时的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 行程。
 <br/>
 <br/>
 川普着陆后发推文说，〝刚抵达阿根廷，将有两天忙碌的日程安排，自始至终排满了重要的会议。我们伟大的国家将会得到极好的展示，会很有成果！〞
 <br/>
 <br/>
 峰会开幕前，川普首先受到了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷总统.html" target="_blank">
  阿根廷总统
 </a>
 马克里的欢迎。两人举行了正式的双边会谈。会谈开始前，两位元首发表了简单的致辞。
 <br/>
 <br/>
 马克里在致辞中表示，美国是阿根廷最大的投资国之一，同时美国人也是阿根廷最大的游客群体。马克里感谢美国过去一年来对阿根廷的重要支持，表示两国将在教育、国防以及打击贩毒等领域继续合作。
 <br/>
 <br/>
 阿根廷总统马克里：〝在此重要时刻，我感谢美国和美国政府给与我国的重大支持，尤其在去年我们经历困难的时刻。〞
 <br/>
 <br/>
 川普则在致辞中提到，两人的家族之间有多年的私交和生意往来，没想到此刻代表各自的国家进行会面。
 <br/>
 <br/>
 美国总统川普：〝我们互相认识很久了。我们将探讨很多好事，对阿根廷和美国都如此，包括贸易、购买军备以及其他方面。〞
 <br/>
 <br/>
 马克里是川普此次G20峰会举行正式会谈的首位元首。
 <br/>
 <br/>
 新唐人记者林宇、张琪综合报导
</div>
