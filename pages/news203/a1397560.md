### 【禁闻】盗取飞机引擎数据 10名中共间谍遭美起诉
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】
 <b>
  盗取
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞机引擎.html" target="_blank">
   飞机引擎
  </a>
  数据 10名
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
   中共间谍
  </a>
  遭美起诉
 </b>
 <br/>
 <br/>
 10月30号，美国司法部发布一份起诉书，起诉10名中国人，指控他们侵入法国航太和美国公司的计算机系统，窃取敏感商业航空数据。
 <br/>
 <br/>
 被起诉的10人中，两人是中共江苏省国安厅的情报人员。司法部说，中共情报人员和他们招募的黑客与公司内部人士合谋，窃取商用客机使用的涡轮发动机数据。
 <br/>
 <br/>
 美国司法部表示，在江苏国安厅招募的那家法国航空航天制造商员工的协助下，这些黑客侵入了该公司电脑，还侵入了制造该涡轮发动机部件的其它公司，包括总部在美国亚利桑那州、马萨诸塞州和俄勒冈州的航空航天公司。而与此同时，一家中共的国有航空航天企业正在开发类似的引擎。
 <br/>
 <br/>
 <b>
  福建建瓯
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
   法轮功学员
  </a>
  叶国华被
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迫害.html" target="_blank">
   迫害
  </a>
  致死
 </b>
 <br/>
 <br/>
 中共迫害法轮功再添血债，9月11号，又有一位法轮功学员被迫害致死。
 <br/>
 <br/>
 据〝法轮大法明慧网〞报导，这名法轮功学员是福建省建瓯市东游中学教师叶国华，他在今年1月被当地警察非法抓捕，关押到建瓯看守所。9月8号，家属突然接到警方通知，说叶国华得了急病，在医院抢救。但家人见到他时，他已经昏迷，并且全身瘀肿。问医生他得了什么病，医生只是含糊其词的说了一句：心、肺、肾等内脏全坏了。
 <br/>
 <br/>
 家属质问警方是否殴打所致，警方矢口否认。
 <br/>
 <br/>
 事后，有朋友打电话询问看守所工作人员，对方回答说：〝人是死了，但你千万不要过问和打听这件事。〞
 <br/>
 <br/>
