### 参院共和党再增一席 政府预算仍待过关
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 最后一席的归属尘埃落定，密西西比州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 籍参议员史密斯成功连任。与此同时，新的政府预算案要在国会闯关，边境墻拨款成焦点。
 <br/>
 <br/>
 密西西比州复选，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 参议员史密斯击败民主党人埃斯皮，成功连任。
 <br/>
 <br/>
 密西西比州参议员 史密斯：我希望每个人都知道，无论你今天投票给谁，我总是会代表每一位密西西比人。
 <br/>
 <br/>
 现在参院共和党以53：47领先民主党6个席位。
 <br/>
 <br/>
 同时，不出外界预料，国会众议院于周三提名了民主党人佩洛西为众院议长，该提名将在明年1月3日正式表决。
 <br/>
 <br/>
 目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 后的国会新格局已基本成形。但在新国会明年1月就职前，当前国会还要面对12月7日截止的政府预算案表决。
 <br/>
 <br/>
 中期选举期间，上万名中美洲大篷车移民向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 聚集，川普政府部署约5800名士兵协助边防，阻止非法越境。
 <br/>
 <br/>
 与此同时，修建美墨边境墙的呼声再次高涨。
 <br/>
 <br/>
 目前，川普正寻求国会批准50亿美元的建墙款。
 <br/>
 <br/>
 周二，川普表示，如果接下来总计约3120亿美元的政府预算中不包含建墙款，那么他与共和党将不支持预算案。这意味着，联邦政府面临12月7日后再度关门。
 <br/>
 <br/>
 众议院共和党党鞭斯卡利斯呼吁两党合作，并认为有希望通过预算案。
 <br/>
 <br/>
 众议院共和党党鞭 斯卡利斯：不是说大家在政府是否关门上计较，而是我们是否达成共识保护美国安全。因此我们会进行这方面协商。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
