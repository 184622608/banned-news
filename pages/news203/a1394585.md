### 芝加哥凶杀案减少 川普谢警察捍卫治安
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际警察局长协会.html" target="_blank">
  国际警察局长协会
 </a>
 ，在150个国家拥有3万多名会员，正值该协会成立125周年之际，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统在其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_年会.html" target="_blank">
  年会
 </a>
 上，感谢警察捍卫安全、制止犯罪所做的贡献。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统：〝代表所有美国人，我对你们每一天保护我们的家人、捍卫城市治安、 制止犯罪分子，表达我们永远的感激。〞
 <br/>
 <br/>
 虽然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芝加哥.html" target="_blank">
  芝加哥
 </a>
 刚经历枪击事件，但警方上周表示，与2017年前9个月相比，今年该市的凶杀案减少了102例，射击受害者减少了近500人。
 <br/>
 <br/>
 川普总统指出，他执政将代表法律、秩序和正义，他认为参议院最新确认的大法官卡瓦诺就是美国宪法的忠实捍卫者。
 <br/>
 <br/>
 川普总统：〝就在两天前，参议院确认了一位极为合格的被提名人，他将成为忠实的美国宪法捍卫者。〞
 <br/>
 <br/>
 正值飓风袭击佛州，川普总统还告知佛州和地方官员采取一切必要的预防措施。
 <br/>
 <br/>
 川普总统：〝我告诉州长斯科特，我们为你们准备好了。我们之前应对飓风做得很好。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际警察局长协会.html" target="_blank">
  国际警察局长协会
 </a>
 （IACP）在150个国家拥有30,000多名会员，是全球警务领域公认的领导者。自1893年以来，该协会一直代表执法部门发表意见，在全球警务工作中提升领导力和专业水平。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
