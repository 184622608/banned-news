### 感恩节前夕 暴风雪侵袭美东北
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 期间，美国东北部将遭遇寒冷天气。部分地区预计将迎来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降雪.html" target="_blank">
  降雪
 </a>
 ，或将为假日季出行的民众造成困扰。
 <br/>
 <br/>
 据AccuWeather报导，新一轮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降雪.html" target="_blank">
  降雪
 </a>
 将为美东部分地区带来影响。
 <br/>
 <br/>
 从本周二（11月20日）晚间开始，大片降雪将从中西部以北和五大湖区域伸延开来，周三抵达东北大部分地区。
 <br/>
 <br/>
 预计周二部分地区会迎来1～3英吋降雪，而强风暴，则会为纽约卡茨基尔（Catskills）到缅因州沿海地区，带来3～6英吋降雪。
 <br/>
 <br/>
 AccuWeather气象学家表示，如果风暴在海岸附近加强，不排除最高带来6英吋降雪。
 <br/>
 <br/>
 据AccuWeather高级气象学家亚历克斯・索斯诺斯基（Alex Sosnowski）说，〝我们担忧与北极冷空气相关的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雪.html" target="_blank">
  暴风雪
 </a>
 ，将在周三晚间，抵达美国纽约市北部和西部郊区、以及费城、华盛顿DC、巴尔的摩。〞
 <br/>
 <br/>
 周三，大湖效应带来的危险暴风雪，可能会快速导致路况安全问题，对假日季成千上万的驾驶员造成影响。特别是在天空突然白茫茫一片的时候。
 <br/>
 <br/>
 在纽约市内，阵风会拉低实际体感温度，让人觉得更加寒冷。
 <br/>
 <br/>
 严酷寒冷的暴风雪天气，将在周四得到好转，但气温并未因此升高，干燥天气将继续维持。
 <br/>
 <br/>
 新唐人记者宇亭纽约报导
</div>
