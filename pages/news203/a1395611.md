### 投下关键选票支持卡瓦诺 参议员柯林斯家收毒信
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】本周一，美国共和党籍参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯林斯.html" target="_blank">
  柯林斯
 </a>
 （Susan Collins）的家中收到一封含有蓖麻毒素的信件，警方获得报警后，随即前往现场清理了受毒信影响的地方，目前当局仍在检验信件并调查来源。此前，柯林斯曾投下关键一票支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 出任美国大法官。
 <br/>
 <br/>
 据美国媒体ABC新闻报导，美国共和党参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柯林斯.html" target="_blank">
  柯林斯
 </a>
 在缅因州班戈（Bangor）的家中15日收到一封毒信，柯林斯接到毒信后向警方报案，地方警局于当地时间15日下午1时39分派员前往柯林斯的家中处理。
 <br/>
 <br/>
 柯林斯办公室的新闻主任克拉克（Annie Clark）日前接受媒体采访时表示，寄信的人声称用蓖麻毒素污染了这封信，而目前柯林斯家中受影响的区域已被专业人员清理干净，她和丈夫可以继续留在家里。
 <br/>
 <br/>
 现在班戈警方尚未对外透露这封毒信的内容，仅称有关机构仍在检验信件并调查其来源。
 <br/>
 <br/>
 报导指出，美国大法官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 被川普提名时，曾遭他人指控年轻时涉及性侵，但卡瓦诺本人完全否认了相关指控，并认为那是政敌试图阻止他就任美国大法官的政治性攻击。FBI一度介入调查，最后得出的结论是相关指控〝查无实证〞。在参议院就卡瓦诺出任大法官事项投票表决时，共和党参议员柯林斯（Susan Collins）是关键选票之一，她当时投下赞成票力挺卡瓦诺上位。
 <br/>

 <br/>
 据了解，在柯林斯遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓖麻毒信.html" target="_blank">
  蓖麻毒信
 </a>
 攻击之前，10月1日美国国防部和特勤局接连截获了三封可疑信件，这三封信件被检测出含有致命的蓖麻毒素，而收件者分别是国防部长马蒂斯、海军作战部长约翰‧理查森上将和总统川普。所幸的是，这些信件在进入五角大楼和白宫之前被及时发现有问题，没有造成人员伤害。
 <br/>
 <br/>
 经过两天的调查，10月3日，犹他州洛根市一名海军退伍军人因涉嫌向美国总统和国防部高官邮寄了上述毒信而被拘捕。
 <br/>
 <br/>
 资料显示，蓖麻毒素是从蓖麻籽中提取的物质，毒性比氰化钾高6000倍，只要摄取、吸入或注射小剂量的蓖麻毒素便足以致命，而且蓖麻毒素没有解药。
 <br/>
 <br/>
 据美媒的报导，涉嫌投送毒信的艾伦现年39岁，他曾于1998年至2002年在海军服役，担任伤害控制见习消防员。但这名男子退役后曾于2004年因涉及两名女童的性虐待案而被起诉，他后来承认了罪名较轻的忽视和虐待指控，不必被登记为性犯罪者。
 <br/>
 <br/>
 同一年，一名妇女在艾伦涉及的另一案件中向法院申请了保护令，并获得批准。2008年，艾伦又因为严重袭击未遂而入狱两年半，于2011年获释。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
