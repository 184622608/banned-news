### 中期选举：共和党参院保卫战 决胜关键在这3州
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 正在进行中。共和党团评估这次选战中的参议院控制权保卫战选情乐观，预估共和党在参院里不但能够继续保住多数掌控，还有很有希望增加席次。
 <br/>
 <br/>
 目前，美国公民对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的投票正在进行中。最新民调也显示，田纳西州的两党候选人支持度不相上下，内华达和亚利桑那州的选情也陷入胶着苦战。不过，从整体战况来分析，共和党在参议院选举方面的选情仍十分乐观。
 <br/>
 <br/>
 美国总统川普近日先后奔赴俄亥俄州、密苏里州和印第安纳州出席造势活动，为这些州的共和党参选人助威。川普呼吁各地支持共和党候选人的民众踊跃投票。
 <br/>
 <br/>
 有分析指出，密苏里州、印第安纳州都是川普2016年总统大选时获得压倒性胜利的选区，如果共和党想要在这次的中期选举中稳住参议院的多数优势，这两州再加上俄亥俄州是决战的关键。
 <br/>
 <br/>
 从目前选情来看，共和党在北达科塔州、密苏里州与印第安纳州均可望增加席次，唯蒙大拿州、佛罗里达州的选情呈现〝五五对决〞的紧绷态势。
 <br/>
 <br/>
 法新社11月4日发文分析称，民主党可能在这次中期选举中丢掉其在密苏里、北达科他和印第安纳这3州的联邦参议院席次。
 <br/>
 <br/>
 该报导表示，在密苏里州，寻求连任的65岁民主党籍联邦参议员麦卡斯基尔（Claire McCaskill），面临共和党籍州检察长霍利（Josh Hawley）的强力挑战。川普亲自为38岁的霍利助选，称呼他是共和党的明日之星。而2016年大选时，川普在密苏里曾大胜18个百分点。
 <br/>
 <br/>
 在北达科他州，寻求连任的民主党63岁的联邦参议员海蒂‧ 海特坎普(Heidi Heitkamp)选情告急。根据民调统计，她的支持度一直落后于57岁的共和党候选人克雷默(Kevin Cramer) ，差距平均约在11%左右。
 <br/>
 <br/>
 此外，新泽西州64岁的民主党籍联邦参议员梅南德兹（Bob Menendez）因遭贪污指控已陷入困境。虽然本案最后因陪审团成员意见分歧，以无效审判结果落幕，他的控罪因而得以撤销，但面对64岁共和党籍企业家修根（Bob Hugin）挑战，梅南德兹寻求连任的机会越来越渺茫。
 <br/>
 <br/>
 有舆论认为，川普政府上台以来，美国民众对美国经济的评价日渐乐观，加上总统特朗普对待移民跟边境安全的强硬立场正获得越来越多选民的认同，很可能因此而拉抬共和党选情。
 <br/>
 <br/>
 《华盛顿邮报》和ABC新闻网日前公布的最新民调显示，65%的美国民众认为美国经济状况良好或优秀，认为经济不好的仅有34%。上次美国民众对经济如此乐观是在2001年。
 <br/>
 <br/>
 不过，共和党在参院的明显优势在最后阶段也出现了一些悬念，主要是在内华达和亚利桑那州两党候选人都陷入了势均力敌的苦战。最新民调显示田纳西州的两党候选人的支持度也不相上下。
 <br/>
 <br/>
 至于众议院方面的选情，11月4日《华盛顿邮报》和ABC新闻网公布的两项最新民调显示，共和党正在迅速拉近和民主党的差距，众院多数席位最终〝花落谁家〞尚难定论。
 <br/>
 <br/>
 （记者欣然综合报导/责任编辑：东野）
