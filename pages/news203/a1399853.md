### 彭斯曝APEC与习两次交谈内容：让他改变贸易政策
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】在刚刚结束的亚太经济合作会议（APEC）峰会上，北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 和美国副总统曾激烈交锋。而一些媒体则聚焦两人在峰会上的短暂〝亲切〞交流。日前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 曝光了他两次和习交谈的内容，是要求北京改变贸易政策。
 <br/>
 <br/>
 《法新社》11月18日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 透露，他在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 议间曾两次与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 进行〝坦诚交谈〞。他告诉习近平，美国有兴趣与中方建立更好的关系，但中方必须先改变其贸易政策。
 <br/>
 <br/>
 此次APEC峰会上，习近平和彭斯针锋相对的演讲，成为舆论关注的焦点。
 <br/>
 <br/>
 习近平在演讲中不点名批评贸易〝保护主义〞，并宣称〝无论热战、冷战和贸易战都没有赢家〞。
 <br/>
 <br/>
 而与会的彭斯则反击称，北京拒绝开放市场，才是真正的贸易保护主义。彭斯还辛辣地抨击北京的〝一带一路〞对沿线国家来说是一条〝不归路〞，并宣布美方将支援印太地区基础建设，但不会推出带有约束力的〝带〞和单方向的〝路〞。
 <br/>
 <br/>
 另外，许多海外媒体报导指，在领导人拍合照现场，两人也曾有短暂交流。
 <br/>
 <br/>
 报导称，按照历届APEC惯例，主办国会为与会各国领袖提供当地民族服装，并拍合照。当习近平抵达拍摄现场时，彭斯正与加拿大总理杜鲁多谈话。习近平与其他领袖打完招呼后，便停下来与彭斯交谈，并由翻译员进行翻译。会谈持续了约1分半钟，过程中都〝面带微笑〞。
 <br/>
 <br/>
 不过，双方礼节性的〝交流〞，显然并不影响实质上的激烈对抗。由于中美贸易方面的巨大分歧，这一届APEC峰会破例没有发表联合公报。
 <br/>
 <br/>
 除了抨击中方经济政策外，彭斯在峰会演讲中，再次明确了美国加入印太地区政治竞争的计划。
 <br/>
 <br/>
 他说，〝美国有一个‘自由开放的印度太平洋’的愿景。现在美国对亚太地区投资总额已经突破1.4万亿美元，超过中国、日本和韩国的总和。美国对亚太地区的投资将继续增长。〞
 <br/>
 <br/>
 彭斯承诺，〝美国可以提供更好的选择，不会把我们的伙伴淹没在债务危机之中，我们不会强迫或损害你们的独立性。〞
 <br/>
 <br/>
 美国《政治》新闻网指，虽然彭斯没有点名北京，但他敦促亚洲各国考虑华盛顿的印太战略，而该战略得到了澳大利亚和日本的支持，是替代中方〝一带一路〞的首选方案。
 <br/>
 <br/>
 多家西方媒体认为，习近平和彭斯此次的演讲表现出中美之间的矛盾正在扩大，也让外界对本月底的中美首脑会晤进一步降低了期望值。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
