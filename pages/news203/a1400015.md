### 白宫恢复CNN记者通行证 制定3项规则约束记者
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 周一（11月19日）宣布，恢复美国有线电视新闻网（CNN）记者阿科斯塔的记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通行证.html" target="_blank">
  通行证
 </a>
 ，但已制定新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_规则.html" target="_blank">
  规则
 </a>
 ，用以规范白宫记者在总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新闻发布会.html" target="_blank">
  新闻发布会
 </a>
 上的礼仪。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 新闻发言人桑德斯（Sarah Sanders）周一宣布，白宫不再寻求永久吊销阿科斯塔（Jim Acosta）的记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通行证.html" target="_blank">
  通行证
 </a>
 ，而且当天下午已通知阿科斯塔和CNN，阿科斯塔的通行证已经恢复。
 <br/>
 <br/>
 但桑德斯同时指出，白宫也已通知阿科斯塔，今后参加白宫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新闻发布会.html" target="_blank">
  新闻发布会
 </a>
 必须遵守新制定的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_规则.html" target="_blank">
  规则
 </a>
 。
 <br/>
 <br/>
 CNN立即对桑德斯的发言做出回应，声明放弃针对白宫的诉讼。
 <br/>
 <br/>
 关于记者在总统新闻发布会上应遵守的规则，桑德斯列举如下：
 <br/>
 <br/>
 一、一名记者只能提出一个问题，然后将发言机会留给其他记者；
 <br/>
 <br/>
 二、由总统或其他白宫官员决定提问，可能允许提出一个或多个后续问题；只有提出后续问题的要求得到允许，提问者才能发言；
 <br/>
 <br/>
 三、要遵守〝停车让道〞的规则。这种情况发生时，正在提问的记者要把麦克风交给白宫工作人员，以供下一位提问者使用；
 <br/>
 <br/>
 桑德斯指出，如果记者不遵守上述规则中的任何一条，都可能会导致其通行证被暂停或吊销。
 <br/>
 <br/>
 白宫在致阿科斯塔的信中告诫他，如果他将来拒绝遵守这些规则，白宫将采取相应行动。川普总统知道并赞成这一决定。
 <br/>
 <br/>
 11月7日，在白宫举行的新闻会上，阿科斯塔以尖刻的提问持续纠缠川普总统，一名白宫女实习生上前试图拿下他手中的麦克风以阻止他继续发问，但他躲闪不给，并以手臂挡开年轻的女工作人员。川普当场斥责阿科斯塔粗鲁，白宫当晚宣布吊销阿科斯塔的记者通行证、禁止他进入白宫。
 <br/>
 <br/>
 CNN之后就此案提起诉讼。11月16日，华盛顿地区法院的联邦法官凯利（Timothy J. Kelly）裁决，白宫应立即恢复阿科斯塔的记者通行证。该裁定基于宪法《第五修正案》，凯利称，白宫在吊销阿科斯塔的新闻通行证前，没有给予阿科斯塔履行正当程序的机会。
 <br/>
 <br/>
 但凯利做出的只是一项临时性裁决，有效期仅14天，而关于最终是否取消阿科斯塔的通行证，原定未来几周内进一步听证。
 <br/>
 <br/>
 白宫周一宣布以实行新规则代替吊销阿科斯塔的通行证，或许更有利于真正规范白宫新闻发布会的礼仪。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
