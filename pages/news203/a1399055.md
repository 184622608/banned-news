### 企图干预监督穆勒调查 民主党施压司法部
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】为阻止川普任命的代理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部.html" target="_blank">
  司法部
 </a>
 长惠特克（Matthew Whitaker）监督
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 目前正向司法部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_道德主管.html" target="_blank">
  道德主管
 </a>
 施压，企图让他出面提议惠特克回避通俄门调查。
 <br/>
 <br/>
 据《华盛顿时报》（The Washington Times）11月12日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 高层表示，他们认为惠特克绝对不能负责监察特别检察官穆勒（Robert Mueller）的调查工作，并且希望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部.html" target="_blank">
  司法部
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_道德主管.html" target="_blank">
  道德主管
 </a>
 洛夫特斯（Lee J. Lofthus）也赞同这种观点。
 <br/>
 <br/>
 这封发给洛夫特斯的信件由参议院少数党领袖舒默（Chuck Schumer）和众议院民主党领袖佩洛西（Nancy Pelosi）主笔、并经参众两院的司法委员会和情报委员会、以及众议院监察委员会的民主党议员共同签名，旨在施压洛夫特斯介入阻止惠特克的工作。
 <br/>
 <br/>
 这些民主党议员指责惠特克过去针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查所做的评论。惠特克去年曾建议，司法部长应该卡住特别检察官的调查资金，从而限制其持续调查的能力。
 <br/>
 <br/>
 民主党在信中措辞强硬地要求，洛夫特斯须〝立即书面通知我们〞，他本人或司法部的其他道德官员是否已经建议惠特克回避监督
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒调查.html" target="_blank">
  穆勒调查
 </a>
 。
 <br/>
 <br/>
 这些民主党议员还要求，洛夫特斯须向他们提供迄今为止司法部已提供给惠特克的道德指导。
 <br/>
 <br/>
 11月7日，即中期选举结束次日，司法部长塞申斯（Jeff Sessions）应川普总统的要求辞职。同日，川普宣布由惠特克出任代理司法部长，司法部则声明，副部长罗森斯坦（Rod Rosenstein）不再负责监督通俄门调查，转而由惠特克接替这项工作。
 <br/>
 <br/>
 惠特克曾在小布殊总统（George W. Bush）任内供职司法部，担任爱荷华州检察官。塞申斯出任司法部长后，委任他担任自己的幕僚长。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
