### 美国务卿：美将建立国际新自由秩序
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 ，周二在布鲁塞尔发表外交政策演说，指出现有国际秩序正遭到中共、伊朗等国的破坏。美国将推动建立新的世界秩序。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 ：〝伊朗和中共，他们真正在破坏国际秩序。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务卿蓬佩奥，4号在布鲁塞尔发表演说。他表示，二战后，以多边主义为基础建立的国际机构，初衷是维护世界和平繁荣和人权，但目前一些国家不断违规，挑战现有国际秩序。
 <br/>
 <br/>
 蓬佩奥：〝中共经常利用世界贸易组织的规则漏洞，强加市场限制，强制技术转让和窃取知识产权。它知道世界舆论无力阻止其奥威尔式的侵犯人权行为。〞
 <br/>
 <br/>
 唐靖远：〝比如世界贸易组织这个多边组织，那些严格履约的成员国，一部分经济主权权力受到限制和弱化。而那些钻漏洞、违背协议的国家，世贸组织对它们却没有强有力的约束和惩罚机制。〞
 <br/>
 <br/>
 李天笑：〝比如说世界银行和国际货币基金组织，是援助需要开发、需要援助的国家的组织。但是现在的情况是，像中共政权已经可以通过自己能力，在国际资本市场上进行融资，甚至大规模向其它国家输出资金、来达到自己的目的。已经有这种能力，还继续向世界银行、国际货币基金组织申请贷款、取得优惠利益等。〞
 <br/>
 <br/>
 蓬佩奥敦促，改革联合国、世界银行等国际组织。不能继续纵容破坏规则者。
 <br/>
 <br/>
 蓬佩奥：国际机构必须帮助促进加强自由世界的安全和价值观的合作，否则就必须改革或淘汰（这些机构）。如果（这些机构的）条约被违反，违约者必须受到抵制。
 <br/>
 <br/>
 他警告，一些强权正试图改变现有国际秩序。
 <br/>
 <br/>
 蓬佩奥：急迫的威胁是，我们正在面对一些强权国家和人。他们的野心是，想重塑国际秩序，以他们反自由的形象。
 <br/>
 <br/>
 唐靖远：现有不完善的多边机制，客观上造成全球财富非正常的，向低自由少人权的国家转移。反过来这些国家再利用经济实力，扩大影响更多的多边机制，来变异普世价值观，并扭曲全球事务治理方式。而且随着各国将更多主权让度给国际组织，也有演变成共产主义所倡导的世界政府的风险。
 <br/>
 <br/>
 李天笑：〝美国通过蓬佩奥发出的信息是，美国要改革目前已经不适应国际现状的国际组织，使它能够满足大多数国家（倡导的）公平、公开、透明、自由的原则，实现那些国际组织本来应该起到的作用。〞
 <br/>
 <br/>
 蓬佩奥表示，川普政府将带领美国，重返传统的世界领导者角色，并联合盟友，重塑一种尊重国家主权，维护公民权利的新的世界秩序。
 <br/>
 <br/>
 蓬佩奥：我们正在凝聚全球的高尚国家，以建立一种新的自由秩序。以避免战争，实现所有人的更大繁荣。我们渴望使国际秩序服务于我们的公民，而不是控制他们。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
