### 卡瓦诺宣誓就职大法官 总统川普祝贺
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】美国联邦参议院周六下午以50票对48票确认了对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官.html" target="_blank">
  大法官
 </a>
 提名。之后，卡瓦诺在最高法院宣誓成为美国历史上第114位最高法院大法官。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 的提名在参议院获得确认后，多数党领袖麦康奈尔（MITCH MCCONNELL）在新闻发布会上称赞听证最终坚守住了〝无罪假定〞的原则。
 <br/>
 <br/>
 参议院多数党领袖麦康奈尔：〝今天对美国来説是美好的一天，也是参议院重要的一天。我们坚守住了无罪假定原则。我们没有被在议会大厅和到共和党议员家外围堵的示威者们所吓住。我为参议院共和党党团感到无比骄傲。我也感谢曼钦参议员对我们的帮助。〞
 <br/>
 <br/>
 川普总统当天下午6点50分左右签署了对卡瓦诺的任命令，几分钟后卡瓦诺完成了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宣誓就职.html" target="_blank">
  宣誓就职
 </a>
 。
 <br/>
 <br/>
 当晚前往堪萨斯州托皮卡（Topeka）市参加中期选举竞选活动的川普总统在机场接受了记者采访。
 <br/>
 <br/>
 美国总统 川普：〝我刚刚向他（卡瓦诺）表示了祝贺，得之不易。谁会想到他会经历这些发生的事情。一切指控都没有旁证。我要感谢联邦调查局。我认为联邦调查局的工作很出色，他们工作很努力，也很迅速。我听説报告很详细、透彻，非常专业。所以我要感谢他们。我要感谢司法部，因为他们也和联邦调查局一起非常努力地工作。非常迅速。〞
 <br/>
 <br/>
 〝所以我们有一个伟大的新的最高法院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官.html" target="_blank">
  大法官
 </a>
 ，他将在那里很多年。我们非常非常为他骄傲，也为他和他的家人不得不经历的忍受。这是一个伟大的证明，也是对我们国家的证明。〞
 <br/>
 <br/>
 新唐人记者徐哲、雨薇综合报导
</div>
