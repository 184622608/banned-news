### 川普威斯康辛州集会 呼吁政治文明
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】周三晚，美国总统川普出席威斯康辛州集会，为共和党人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选.html" target="_blank">
  助选
 </a>
 。提到当天发生的可疑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_包裹事件.html" target="_blank">
  包裹事件
 </a>
 ，川普谴责暴力行为，表示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治暴力.html" target="_blank">
  政治暴力
 </a>
 是对民主的攻击。周三，CNN总部发现可疑包裹，之前警方截获邮寄给前总统奥巴马、前国务卿希拉里的可疑邮件。
 <br/>
 <br/>
 演说开始，川普提到可疑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_包裹事件.html" target="_blank">
  包裹事件
 </a>
 ，重申保护美国人民安全。
 <br/>
 <br/>
 美国总统川普：〝作为总统，我的最高职责是保护
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国安全.html" target="_blank">
  美国安全
 </a>
 。我们是这么说的，也是这么做的。联邦政府正在积极调查，会找出责任人，希望很快将他们绳之以法。〞
 <br/>
 <br/>
 川普表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治暴力.html" target="_blank">
  政治暴力
 </a>
 是对民主的攻击。
 <br/>
 <br/>
 美国总统川普：〝任何政治暴力的威胁和行为，都是对民主本身的攻击。没有哪个国家能够通过容忍暴力，或容忍以暴力进行政治恐吓和操控，取得成功。我们希望各方，能在和平与和谐中走到一起。我们能做到，我们能做到，我们能做到。〞
 <br/>
 <br/>
 川普还提到媒体责任，呼吁建立新的政治文明。
 <br/>
 <br/>
 美国总统川普：〝作为更大范围全国行动的一部分，消除分歧、团结民众，媒体也有责任树立文明基调，制止无休止的敌意和长期的负面（报导），停止刻意安排的错误攻击和报导，（媒体）必须这样做。〞
 <br/>
 <br/>
 川普还提到，民主党的移民法律，鼓励大篷车难民非法闯入美国，必须改变。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
