### 恶魔岛历史悬案 警公布3囚制假人头越狱模型
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】美国旧金山湾阿尔卡特拉斯岛（Alcatraz）又称〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恶魔岛.html" target="_blank">
  恶魔岛
 </a>
 〞因四面峭壁、深水，联外交通不易，被设为联邦监狱，过去关押过不少重刑犯，也发生过不少
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_越狱.html" target="_blank">
  越狱
 </a>
 行动，但约半世纪前，仅有3名囚犯逃狱成功，至今这3名囚犯仍未抓回。15日联邦调查局（FBI）对外公布当年囚犯越狱的细节，并展示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_3D假人头.html" target="_blank">
  3D假人头
 </a>
 模型。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恶魔岛.html" target="_blank">
  恶魔岛
 </a>
 〞在1933年至1963年间被设为联邦监狱，是高度设防监狱。1962年，3名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_越狱.html" target="_blank">
  越狱
 </a>
 犯分别是莫里斯（Frank Morris）、以及约翰（John Anglin）与克伦斯（Clarence Anglin）兄弟，利用肥皂、头发、卫生纸等材料制作出逼真的假〝人头〞放在枕头及被单之间，假装有人在床上睡觉，然后耗时数月偷挖地道逃狱。
 <br/>
 <br/>
 最后3人从监狱通风口逃出后消失在世界上，是恶魔岛监狱中的有史以来的悬案。
 <br/>
 <br/>
 FBI探员宾纳特（John Bennett）表示，当年的〝假人头〞太脆弱，且是逃狱事件的证物，因此实验室团队特意扫描真品，以3D列印制成仿品，在利用人员捐出及监狱理发厅廊收集到的头发，重现当年的假人头样貌。
 <br/>
 <br/>
 目前假人头只对媒体公开，未来将会开放给民众参观。宾纳特说，〝此举不仅是尽职调查悬案，更是为了警告其他逃犯，我们不会因为时间久远而放弃追捕。〞
 <br/>
 <br/>
 这段逃狱故事，1979年被拍成电影《亚特兰翠大逃亡》（Escape From Alcatraz）。
