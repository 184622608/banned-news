### 反垄断 法院：高通须授权专利技术给对手
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】针对美国联邦贸易委员会（FTC）于2017年初提起针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高通.html" target="_blank">
  高通
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反垄断.html" target="_blank">
  反垄断
 </a>
 诉讼，联邦法官周二 (6日) 裁定，芯片销售商高通必须将部分技术授权给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英特尔.html" target="_blank">
  英特尔
 </a>
 等竞争对手。
 <br/>
 <br/>
 美国加州北区地方法院法官Lucy Koh，周二做出初步裁决，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高通.html" target="_blank">
  高通
 </a>
 公司(Qualcomm)须将部分技术授权给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英特尔.html" target="_blank">
  英特尔
 </a>
 (Intel）等竞争对手，包括制造行动数据机晶片所涉及的一些专利。该专利主要是让智慧手机能够连结无线网路。该案预计将于明年正式开庭审理。
 <br/>
 <br/>
 Koh表示，高通在文件中已无争议地表明，数据机晶片是手机的核心组件，由于它提供数据机晶片的标准，高通公司不能歧视晶片供应商。
 <br/>
 <br/>
 10月，高通与FTC曾联合要求法官，在他们进行和解谈判时，先推迟此项裁定最多30天。但Koh否决了该提议，并于周二发布了自己的判决。
 <br/>
 <br/>
 目前尚不清楚该裁决会对高通与FTC的和解谈判产生何种影响，但高通的股价在新闻发布后下跌约0.3%。不过，在尾盘有所拉高，收涨0.25%，至63.63美元。
 <br/>
 <br/>
 高通公司以专利授权为主要的收入来源，但是包括苹果等一些大客户，都对其提出诉讼。高通则一直为其商业模式辩护，在全球范围内受到监管方的挑战。
 <br/>
 <br/>
 韩国与台湾之前都做出不利高通的判决，高通也对判决进行了上诉。
 <br/>
 <br/>
 台湾公平会与高通公司于今年8月9日达成诉讼和解，高通将进行5年期产业投资方案，并就〝台湾营运与制造工程暨测试中心（COMET）〞、〝5G技术与产品开发〞、〝协助台湾OEM厂商全球市场拓展与新兴产品开发〞、〝研发及创新育成〞等4大关键领域进行投资，直接投资总额约达7亿美元，并缴纳约9300万美元罚款。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
