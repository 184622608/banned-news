### 白宫报告痛批社会主义：应丢入历史垃圾箱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】 白宫近日发布的一份报告措辞严厉地批评了社会主义的经济政策。报告指出，社会主义的制度曾经给中国、苏联、古巴等国家造成饥荒惨剧，又让原本很富裕的委内瑞拉陷入赤贫。由于机制上的根本缺陷，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_社会主义经济政策.html" target="_blank">
  社会主义经济政策
 </a>
 即使在民主制度下实施也会带来灾难。如果美国也走那样的路线，将使美国经济受到重创。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫经济顾问委员会.html" target="_blank">
  白宫经济顾问委员会
 </a>
 （CEA）10月23日发布了一份题为《社会主义的机会成本》（〝The Opportunity Costs of Socialism〞）的报告，概述了〝社会主义〞意识形态给宏观经济带来的机会成本，并进而对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_社会主义经济政策.html" target="_blank">
  社会主义经济政策
 </a>
 提出了尖锐的批评。
 <br/>
 <br/>
 这个报告首先发出警告称，社会主义的许多观念已在美国的政治讨论中卷土重来，一些自称社会主义者提出的政策提议，正在国会和选民中获得支持。而当前的这些社会主义提倡者，与传统上的社会主义支持者一样，都提出了相似的一些目标展望。
 <br/>
 <br/>
 报告表示，那些支持社会主义的人士都把市场经济中的收入分配形容为不公平〝剥削〞的结果，宣传应该强化政府对国民经济的控制权来纠正结果。这些人提出的解决方案主要是实行所谓〝单一支付者制度〞和高税率政策，他们并鼓吹应该让国民免费从政府那里获取物品和服务（即按需分配）。
 <br/>
 <br/>
 报告进一步分析指出，由于社会主义的经济政策对生产和创新提供的物质激励十分有限，而〝免费〞分配物品和服务的做法，又让产品的价格无法真实地反应生产成本和消费者的需求等重要的经济信息，所以那样的经济政策只能给国家带来灾难。
 <br/>
 <br/>
 这份报告以毛泽东时代的中国、古巴和苏联等几个社会主义农业国为例进行分析，指这些非民主的政府在强行夺取了对整个国家的农业控制权后，向国民承诺由政府提供充足的食品，但实际情况是导致粮食生产的大幅降低，最终让〝数千万人饿死于饥荒〞。
 <br/>
 <br/>
 报告对不同程度采纳了社会主义理念的当代工业国家，包括委内瑞拉和北欧进行了分析。
 <br/>
 <br/>
 原本很富裕的委内瑞拉在奉行了社会主义的经济政策后，让国家经济陷入了崩溃的边缘。报告认为，这个当代的教训表明，即使在民主制度下，通过和平方式在工业化国家实行社会主义经济政策，因为根本的激励机制被政策扭曲了，再加上大型国有组织和中央控制资源导致的经济信息缺失等问题，最后的结果仍然是灾难性的。
 <br/>
 <br/>
 报告直言，经过研究估算，如果美国也实施委内瑞拉那种程度的社会主义政策，国内的生产总值（GDP）将下降至少40%，美国人每年的人均收入也将减少24000美元。
 <br/>
 <br/>
 至于采纳了部分社会主义理念的北欧国家，在对劳动力市场进行了较多的监管和征税后，北欧国家国民的生活水平也明显有所下降。目前，北欧国家的生活水平至少比美国低15%。
 <br/>
 <br/>
 报告坦言，如果美国现在也去实行北欧当前的政策，那么中等工资收入的美国家庭每年将被多征2000至5000美元的净转移税。
 <br/>
 <br/>
 报告最后引用前英国首相戴卓尔夫人的话总结道，〝社会主义政府总是花光其他人的钱〞，反过来，这些国家的繁荣之路就是给〝人民更多选择、让他们用自己的方式花自己的钱〞。
 <br/>
 <br/>
 同一天的下午，白宫官方推特帐号上发文称，〝抛开政治不谈，社会主义应丢入历史的垃圾箱。〞
 <br/>

 事实上，早在一个月前，美国总统川普（特朗普）就曾经于9月25日在联合国大会上，向在场的世界各国领袖们发表了一次震撼性演讲，他在呼吁国际贸易体制改革的同时，痛批了社会主义的罪恶。
 <br/>
 <br/>
 川普的演讲也以中共的南美盟友委内瑞拉为例指出，〝不久之前，委内瑞拉还是这个星球上最富有的国家之一。今天，社会主义使这个石油资源丰富的国家破产，并使其人民陷入赤贫之中。实质上，不管是什么地方尝试了社会主义或共产主义，都导致了苦难、腐败和衰变。〞
 <br/>
 <br/>
 当时川普还直言不讳地指出，〝社会主义对权力的渴望导致了扩张、入侵和压迫。世界上所有国家都应该抵制社会主义，抵制这种制度给每个人带来的苦难。〞
 <br/>
