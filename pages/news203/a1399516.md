### 川普白宫慰问老兵 称继续改善军人待遇
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】周四，是美国很多地方的〝感谢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵日.html" target="_blank">
  老兵日
 </a>
 〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统这一天接连出席两场活动，表达对美国军人的敬意。
 <br/>
 <br/>
 周四下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 举行仪式，对美国退伍军人表达感谢。
 <br/>
 <br/>
 美国总统 川普：〝去年我签署了立法，允许每个退伍军人享受美国军人，能在生活的方方面面享受到权利法案的福利。〞
 <br/>
 <br/>
 川普总统今年早些时候还签署行政令，以帮助军人家属在政府中担任公职。川普上台后，一直把改善
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军人待遇.html" target="_blank">
  军人待遇
 </a>
 作为执政重点之一。
 <br/>
 <br/>
 美国总统 川普：〝事实上，我已经做了很多。我都，我都（可以走了），再见了…… 我们还要继续做。〞
 <br/>
 <br/>
 今年的11月15日，是美国很多地方的〝感谢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵日.html" target="_blank">
  老兵日
 </a>
 〞，在华府，白宫和司法部都举办了相应活动。
 <br/>
 <br/>
 美国总统 川普：〝你们是了不起的人，我知道你们的工作有多难。但你们克服了所有（困难），我很荣幸能和你们在一起，愿神保佑美国。〞
 <br/>
 <br/>
 当天早些时候，川普与第一夫人还前往在华府的海军陆战队军营，与士兵们握手、交谈，向他们支援近日华府一场火灾救援表达感谢。白宫表示，这次慰问几周前就开始策划。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
