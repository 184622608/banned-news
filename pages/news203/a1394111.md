### 川普颁新版国家反恐战略 七年来最全面
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 颁布新版《国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反恐战略.html" target="_blank">
  反恐战略
 </a>
 》。这是2011年以来，美国首次全面阐明并更新反恐战略。
 <br/>
 <br/>
 新版《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反恐战略.html" target="_blank">
  反恐战略
 </a>
 》的重点是从源头打击恐怖分子，隔离恐怖分子及其财政物资和后勤支援，并整合美国的反恐工具，使其现代化。
 <br/>
 <br/>
 新版战略将重新加强非军事能力，发挥美国政府机构人员，如执法，外交和财政部官员的重要作用。
 <br/>
 <br/>
 该战略还强调强大边界，国界安全，保护关键基础设施。并致力于扩大美国国内外的伙伴关系，鼓励合作伙伴在反恐活动中给予帮助，包括北大西洋公约组织（北约）同盟国和合作伙伴合作。
 <br/>
 <br/>
 此前，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统推动下，美国和全球盟友摧毁了ISIS，使其濒临彻底灭亡。超过750万人因此脱离魔掌。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
