### 3小偷扭屁股匍匐前进 〝奇葩〞潜入餐厅偷钱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德州.html" target="_blank">
  德州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小偷.html" target="_blank">
  小偷
 </a>
 3人组日前闯入一间餐厅行窃，由于他们是一个接一个扭着屁股，匍匐前进，一路从门口爬进办公室行窃，因动作相当〝奇葩〞被称为〝毛毛虫小偷〞，3人共偷走约450美元现金，目前警方正在追查嫌犯下落。
 <br/>
 <br/>
 从视频中可见，3名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小偷.html" target="_blank">
  小偷
 </a>
 穿着宽松帽T运动服，一个接一个，趴在地上小心翼翼匍匐前进。犹如屋内如好莱坞电影情节般，架设了雷射防盗器，电影里神偷大盗必须巧妙躲避雷射。
 <br/>
 <br/>
 但屋内并没有高科技的防盗装置，3名小偷就这样扭屁股爬行，目的是为了躲开监视器，从门口爬过用餐处和吧台，进到办公室行窃，最后逃之夭夭。
 <br/>
 <br/>
 奇怪的动作让监视器全程录了下来。视频被公开后，这3名小偷被戏称为毛毛虫小偷。目前警方正全力追查嫌犯下落。
