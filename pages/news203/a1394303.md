### 美国大叔撞脸川普 你看像吗？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】近日，一段
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大叔.html" target="_blank">
  美国大叔
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_撞脸.html" target="_blank">
  撞脸
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）总统的影片在网络上广传。视频中的大叔穿着衬衫、戴着海军帽在商店内看彩券，其侧脸、发型及动作都与川普有些相似，引起网友热议。
 <br/>
 <br/>
 推特用户〝波兰爱国者〞（PolishPatriot）2日称，他见到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 分身，并贴出1段6秒的影片。片中可见，这名戴着海军帽，穿着法兰绒衬衫的大叔正在商店内看彩券，他的侧脸不仅与川普相似，连发色也是金发，但他没有讲话。
 <br/>
 <br/>
 影片上传网络后在推特及Reddit论坛等社交平台广传，至今已有超过94万次点击，4351次分享，9227个点赞。网友纷纷留言称，〝他的侧脸真的很像〞。但也有人认为是假的，〝感觉是美国知名演员克莱（Kevin Kline）扮演的〞、〝看那啤酒肚就觉得不像〞。
