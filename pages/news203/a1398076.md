### 共和党参议员高顿拜票 华人助选团跟随
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】寻求第八次连任的第22选区州参议员高顿（Marty Golden）4日到布碌仑金皇庭酒家，接受美国香港总商会背书。他表示若连任，必将在纽约州参议会阻止白思豪企图废除SHSAT的计划，捍卫华人的权益。而当天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人助选团.html" target="_blank">
  华人助选团
 </a>
 跟随他向餐馆顾客一个个拜票，希望华人投他的票。
 <br/>
 <br/>
 这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人助选团.html" target="_blank">
  华人助选团
 </a>
 称高顿为〝华人之友〞，他们列数高顿的政见，包括：拨款112亿巩固学校，反对市长取消特殊高中考试；反对库默推出的取消〝小费抵薪〞（tip credit）提案；反对娱乐大麻合法化；反对关闭雷克岛监狱，然后在市区兴建监狱；反对假释杀警凶手和暴力罪犯等，称赞他〝和华人的价值观一致〞，此外他深入社区，历年举办中国黄历新年会、亚裔传统节、龙狮节、敬老节等，为华人的利益鼓与呼。
 <br/>
 <br/>
 <center>

    高顿和他的华人助选团。

 </center>
 <br/>
 华人助选团的周国东说，现在纽约的大气候，州长和市长都是民主党，纽约民主党在州众议会拥有66个议席，在州参议会仅以一个席位之差控制州参议会，纽约市的州参议员中，只有高顿是唯一的共和党籍参议员，高顿是保住SHSAT的最关键一票，因为如果失去高顿，州参议会将会是极左民主党一党独大。
 <br/>
 <br/>
 他说，参议会的共和党人是阻止一党专政的〝防火墙〞，民主党上一次在2008年短暂赢得参议会后，所幸两名民主党议员与共和党合作，才没让民主党控制参议会，〝现在共和党每一席都至关重要〞。
 <br/>
 <br/>
 华人助选团在〝我们华人为什么必须坚定支持Marty Golden 高顿参议员连任〞传单中指出，〝现在民主党推出郭纳德挑战高顿，民主党的大佬们都来布碌仑，甚至极左的民主党刘醇逸也从皇后区跑来挺极左的郭纳德，上下合力围剿和华人站在一起的高顿参议员，目的就是用民主党取代高顿，将来白思豪更多对华人不利的提案，就可以在纽约州参议院畅通无阻。〞
