### 美饶舌明星成〝川粉〞 设计T恤吁黑人脱离民主党
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】美国知名非裔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_饶舌歌手.html" target="_blank">
  饶舌歌手
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肯伊威斯特.html" target="_blank">
  肯伊威斯特
 </a>
 （Kanye West）是美国总统川普（特朗普）的超级粉丝。最近为了力挺川普，肯伊威斯特设计了一款T恤，呼吁黑人脱离民主党，引发舆论界的广泛关注。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肯伊威斯特.html" target="_blank">
  肯伊威斯特
 </a>
 最近模仿英国〝脱欧〞（Brexit）的单词构词法，以〝黑人退出〞（black exit）的词组为基础创造了单词〝Blexit〞，并设计出了一系打印有〝Blexit〞字样的T恤衫，意思是呼吁美国的黑人们退出支持民主党的阵营。
 <br/>
 <br/>
 据公开的资讯，〝Blexit〞T恤分别有蓝色、橘色、紫色可供选择，价钱大约为25至28美元，另外还有2款丹宁布制的棒球帽，价钱为1顶25美元。
 <br/>
 <br/>

 肯伊威斯特从2016年美国总统大选时就一直相当支持川普，被外界形容为〝死忠〞川普的超级粉丝。就在本月中旬，川普曾特别邀请了肯伊威斯特在白宫共进午餐，两人相处得非常愉快。
 <br/>
 <br/>
 此前，肯伊威斯特为支持川普，还曾经专门与T.I. 合作创作了饶舌歌曲〝Ye vs. the People〞。
 <br/>
 <br/>
 路透社今年5月初发表的一次民意调查统计数据显示，在知名饶舌歌手肯伊威斯特(Kanye West)推文挺川普后，非裔男性对川普的支持率〝显著上升〞。
 <br/>
 <br/>
 这份4月29日进行的民调结果显示，黑人男性支持川普的比率已达到22%，比前一周上升了11%；同时还有7%的受访者表示自己对总统有着〝复杂感受〞(mixed feelings)，与一周的1.5%前相比也提高了许多。

 今年9月底，肯伊威斯特在参加综艺节目〝周六夜现场〞(Saturday Night Live)前不久，宣布自己将改名为Ye，而且肯伊威斯特在节目尾声演唱新专辑Ye里的曲目时，特意头戴一顶有川普签名的〝让美国再次伟大〞红色帽子。
 <br/>
 <br/>
 在现场直播节目结束后，肯伊威斯特没放下麦克风，而是就此发表一场关于政治的即席演说，重申自己对川普的支持，并批评民主党企图使人依赖政府的福利金。
 <br/>
