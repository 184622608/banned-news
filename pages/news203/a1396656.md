### 美智库报告：中共不公贸易夺美340万工作机会
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 公布的一份最新报告显示，自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 加入世界贸易组织（WTO）以来，全美共流失了340万个工作机会，其中多数是由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制造业.html" target="_blank">
  制造业
 </a>
 受到中方严重打击造成的。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 智库经济政策研究所（Economic Policy Institute，简称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_EPI.html" target="_blank">
  EPI
 </a>
 ）周二（23日）发布题为《中国代价深化》（The China Toll Deepen）的最新报告，指出自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 2001年加入WTO到2017年间，逐年增长的美中贸易逆差对美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制造业.html" target="_blank">
  制造业
 </a>
 造成的冲击最大，这期间全美流失的工作岗位数量高达近340万个，其中制造业流失250万个工作岗位，占比为74.4%。
 <br/>
 <br/>
 上述数字是通过在美中贸易中因进口而失去的工作机会，减掉由出口而增加的工作机会得出的。
 <br/>
 <br/>
 报告还显示，2001年到2017年，美中贸易逆差已经导致全美每个州和选区都有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工作流失.html" target="_blank">
  工作流失
 </a>
 的现象。其中以加州流失最多，共失去了56.2万个工作；其次是德州，流失了31.4万个工作；然后是纽约州流失了18.3万个，伊利诺伊州流失14.8万个，宾夕凡尼亚州流失13.61万个。
 <br/>
 <br/>
 报告说，中共自2001年进入了WTO之后，其经济就开始迅速发展。从2001年到2017年，美中贸易逆差从830亿美元增至3750亿美元。其中，在计算机和电子零件行业的逆差增长最多，随之流失的工作机会也最触目惊心，总计减少了120.9万个工作岗位，约占总量的36%。
 <br/>
 <br/>
 研究者还发现，从2001年到2011年，美中贸易逆差的持续增长，直接导致美国工人每年损失约370亿美元的收入。仅2011年一年，来自中国及其它低工资国家进口产品的竞争，导致全美非大学毕业工人的工资减少了1800亿美元。
 <br/>
 <br/>
 EPI的报告得出结论说：〝美中贸易必须进行根本的改变。〞〝我们应当将解决中国（中共）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不公平贸易.html" target="_blank">
  不公平贸易
 </a>
 、廉价劳工以及环境标准、结束货币操纵，作为我们贸易和经济的首要任务。〞
 <br/>
 <br/>
 该报告主笔者之一、EPI贸​​易和制造政策研究主任罗伯特‧斯科特（Robert Scott）告诉财经网站〝市场观察〞（Market Watch），造成工作减少的原因之一，是对美国境内生产商品的需求降低。〝大部份的工作机会流失，都出现在制造业，情况非常严重〞，但〝制造业的劳工失去了工作，可能就会退出劳动力市场。〞
 <br/>
 <br/>
 〝中国的经济正在放缓，唯一能够保持活力的是出口〞，斯科特说，〝如果我们能够减少美中贸易逆差，我认为这是可以做得到的，那么高科技行业将会招聘更多员工。〞
 <br/>
 <br/>
 Matthews Asia投资策略师罗斯曼（Andy Rothman）表示，进口货物价格低，必然会造成美国国内工作机会的流失，然而最大的问题是〝（先前的）政府没有帮助业者妥善应对〞。
 <br/>
 <br/>
 他说：〝肯尼迪总统知道国际贸易将创造赢家和输家，并制定了《贸易调整援助法案》（Trade Adjustment Assistance Act），以帮助失业工人〞，不过，〝我们的国会没有为这个法案提供足够的资金。〞
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：东野）
</div>
