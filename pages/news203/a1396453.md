### 加州小飞机故障迫降高速公路 完美避开车阵吓坏驾驶（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣地亚哥.html" target="_blank">
  圣地亚哥
 </a>
 一架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小飞机.html" target="_blank">
  小飞机
 </a>
 ，近日因为引擎故障，但附近并无适合的降落点，只好紧急迫降在洲际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高速公路.html" target="_blank">
  高速公路
 </a>
 上，过程中驾驶完美避开车阵，吓坏过路的车主。
 <br/>
 <br/>
 根据警方调查，25岁飞行教练姆诺（Ryan Muno）和36岁的学员称，19日上午11点左右，他们在回程途中驾驶的单引擎教练机引擎出现故障，失去动力。姆诺随即冷静接手处理，在判断无法抵达附近的机场后，他决定紧急降落在埃尔卡洪（El Cajon）的8号州际公路上。
 <br/>
 <br/>
 姆诺靠着精湛技术成功避开电线及招牌，完美降落，并跟着车流缓缓前进、未造成事故。不少路过的车主经过飞机时都放慢速度，并对飞机比出大拇指为其喝采。
 <br/>
 <br/>
 目击者称，当时这架飞机降落的方式相当平稳而且缓慢，没有造成人员受伤。警方到场后也对飞行教练的处理方式给予肯定。至于飞机引擎故障的原因，目前还在进一步调查中。
 <br/>
 <br/>
 据悉，姆诺过去是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣地亚哥.html" target="_blank">
  圣地亚哥
 </a>
 州立大学的棒球队内野手。他的母亲表示，姆诺梦想是成为职业棒球员，但因受伤未能如愿，现在他希望成为一名优秀的飞行员。
