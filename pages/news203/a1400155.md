### 非法器官移植仍继续 国际谴责中共
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】有调查团体在过去2个月暗访发现，大陆多家医院仍在从事非法器官移植。各界持续批评中共强摘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员等良心犯器官，呼吁将责任人绳之以法。
 <br/>
 <br/>
 英国BBC近期播出调查报导，披露中国器官移植，等待供体的时间，远远少于国际平均值，一名病患2小时就等到了鲜活器官，供体来源不明。中共前卫生部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄洁夫.html" target="_blank">
  黄洁夫
 </a>
 ，不愿作出解释。
 <br/>
 <br/>
 〝我打电话给一家中国医院，获得迅速移植肝脏的机会，这怎么可能？〞
 <br/>
 <br/>
 〝我不想听，也不想答这些问题。〞
 <br/>
 <br/>
 黄洁夫今年7月曾在央视宣称，从1980年代到2015年，中国依赖死刑犯器官做移植。
 <br/>
 <br/>
 不过官方数据显示，1999年中共迫害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 后，中国器官移植数量突然暴增3倍以上，同一时期死刑犯人数没有明显增加。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汪志远.html" target="_blank">
  汪志远
 </a>
 ：〝按照‘国际大赦’调查的数量，2000年前后的10年中，（中共）平均每年处决了1600个死囚犯。但是（2000年后）中共宣布他们每年器官移植都是上万。所以这是不可能的。〞
 <br/>
 <br/>
 多个国际独立调查员和人权组织，通过电话暗访、大数据分析、论文比对等方式，交叉验证，确认中共1999年迫害法轮功后，将被非法关押的法轮功学员等良心犯，视为活体器官库，进行强制验血，器官配型，按需杀害。
 <br/>
 <br/>
 汪志远：〝在2006年左右，有20多家器官移植医院的医生，直接承认，他们用了法轮功学员器官。遍布中国将近20个省市。〞
 <br/>
 <br/>
 〝追查迫害法轮功国际组织〞调查认为，这样的暴行还在持续。
 <br/>
 <br/>
 汪志远：〝2018年9月份到11月份，我们抽查（暗访）了，北京，郑州，湖南，湖北，浙江，上海，广州，相关器官移植科室的主任。大概有10多个科主任直接承认了，现在来了（做移植）平均等待时间一至两周。用法轮功学员器官，他们还在做。这个报告我们即将要发表。〞
 <br/>
 <br/>
 仍然有人成为受害者，我们越早采取行动，越能使人们不再受害。
 <br/>
 <br/>
 10月16日，多位英美人权律师、医学专家等，在伦敦成立〝独立人民法庭〞，专责调查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强摘器官.html" target="_blank">
  强摘器官
 </a>
 刑事犯罪，12月将举行首次公听会。
 <br/>
 <br/>
 10月23日，加拿大参议院通过S-240法案，将贩运人体器官定为刑事罪行。参与者不能进入加拿大，即使入境也会受到起诉。法案已送交国会，有望成立法律。
 <br/>
 <br/>
 该法案将使得通过强摘获得器官成为违法行为。/最大的问题是中共政权。
 <br/>
 <br/>
 10月至今，在法国、英国、德国、澳洲、美国国会及人权组织的活动上，中共强摘器官问题，也持续受到关注。
 <br/>
 <br/>
 〝法轮功面临严酷迫害。关于强摘器官有明确报导。(中共)杀害他们，并把其器官贩卖。〞
 <br/>
 <br/>
 〝我们需要不断地谈论这个话题，需要让这个话题出现在政治议程上。〞
 <br/>
 <br/>
 〝中国的这一切必须被透明化，在中国犯下活摘器官罪行的人必须绳之以法。〞
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
