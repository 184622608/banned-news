### 美国将制裁杀害沙特记者责任人
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 周二下午在记者会上说，美国已经确认几名沙特官员对杀害记者负有责任，将收回他们的签证，并考虑制裁。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 ：〝我们发现多人牵涉其中，包括沙特情报部门、王室、外交部、和其他部门的人，与记者卡舒吉之死有关。我们会采取行动。〞
 <br/>
 <br/>
 蓬佩奥表示，将取消这些人的签证，与财政部合作，启动人权问责法和其他措施让涉事者承担责任。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝我们要明确指出，美国不会容忍用残忍手段让记者卡舒吉消声。我们与沙特仍是伙伴关系，但川普总统和我对目前状况并不满意。〞
 <br/>
 <br/>
 蓬佩奥指出，将在保护美国利益的同时，惩戒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_责任人.html" target="_blank">
  责任人
 </a>
 。
 <br/>
 <br/>
 新唐人电视综合报导
