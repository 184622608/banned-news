### 贸易战再发一弹 美国将对中国钢轮课重税
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】本周三，美国商务部宣布了一项涉及中国企业的调查结果，初步裁定涉案的中国企业向美国市场倾销钢制轮毂，美国将对这些享受了不公平补贴的中国钢制轮毂征收
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反倾销.html" target="_blank">
  反倾销
 </a>
 税。舆论认为，这是美国政府在对中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 中再发的一弹。
 <br/>
 <br/>
 美国商务部本周三（24日）发布了一项声明，指该部根据印第安纳州埃文斯维尔的Accuride Corp以及俄亥俄州亚克朗的Maxion Wheels Akron LLC这两家公司的投诉，针对一些来自中国的进口钢制轮毂展开了调查，现在作出初步裁决，判定涉案的中国企业享受了中共政府的不公平补贴，然后向美国市场倾销钢制轮毂产品。
 <br/>
 <br/>
 美国商务部表示，这些产品的倾销幅度高达231.7%，该部将指示美国边境官员按这一比例向涉案的中企收取保证金。
 <br/>
 <br/>
 商务部的这项声明还表示，最终裁定将在2019年1月初发布。如果商务部的最终裁定确认初裁结果，美国国际贸易委员会(ITC)将在大约2月21日前做出最终损害裁定。
 <br/>
 <br/>
 舆论认为，这是美国政府在对中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 中新发的〝炮弹〞，估计会令中美贸易冲突再度升温。不过，迄今为止中方尚未对此作出正式的回应。
 <br/>
 <br/>
 此前不久，中共官媒曾发文宣称，中方将〝不屑于〞继续对美采取以牙还牙的方式进行报复。对于美方这次对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_钢制轮毂倾销案.html" target="_blank">
  钢制轮毂倾销案
 </a>
 的裁定，中方将作出怎样的反应，外界也在观察之中。
 <br/>
 <br/>
 （记者李明报导/责任编辑：明轩）
</div>
