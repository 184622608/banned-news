### 疑防遭利用干预美中期选举 脸书移除逾800帐户专页
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】美国社交网站
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脸书.html" target="_blank">
  脸书
 </a>
 （Facebook）本周四宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移除.html" target="_blank">
  移除
 </a>
 逾800个专页及帐户。该网站安全政策总监对外解释称，被移除的账户散播敏感政治内容，或利用脸书吸引网民登入其他附属的广告链接。为防止有人利用这个平台散播虚假资讯而误导民众，脸书公司采取了上述举措。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脸书.html" target="_blank">
  脸书
 </a>
 网络安全政策总监戈雷彻（Nathaniel Gleicher）及产品经理罗德里格斯（Oscar Rodriguez）日前对外界证实，该网站11日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移除.html" target="_blank">
  移除
 </a>
 了559个专页及251个帐户，其中包括多个左翼或右翼的专页。
 <br/>
 <br/>
 他们表示，脸书网站管理人员发现有不少假帐户或同名帐户，在多个专页发布大量内容，吸引网民浏览脸书以外的网站，这种做法违反了该网站有关禁止滥发垃圾内容的规则因而被移除。
 <br/>
 <br/>
 脸书方面特别提到了2015年由俄罗斯商人卡祖恩（Artur Khachuyan）创立的数据库服务供应商 SocialDataHub（SDH），指该公司未经授权收集脸书用户资料，违反了脸书的用户守则和规例。
 <br/>
 <br/>
 据了解，SDH过往曾为俄罗斯政府提供数据分析服务，而这次脸书封杀的账号中，就包括了俄罗斯数据库服务供应商 SocialDataHub超过66个帐号、专页和应用程式。
 <br/>
 <br/>
 有美国媒体报导称，脸书在矽谷总部成立了〝战情室〞，以防止有心人士利用脸书来干预即将到来的巴西大选第2轮投票及美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 。
 <br/>
 <br/>
 脸书创办人兼行政总裁扎克伯格(Mark Elliot Zuckerberg)近期曾发文表示，为对抗有心人士为影响选情而操纵这个平台，脸书已做足准备，最近还成功阻止外国势力锁定数个国家的行动。
 <br/>
 <br/>
 （记者欣然综合报导/责任编辑：程以仁）
</div>
