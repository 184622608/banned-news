### 川普为中期跑44场集会 盛况推文全记录
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 周二投票，而从今年3月开始，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）共为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 举办了44场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选集会.html" target="_blank">
  助选集会
 </a>
 ，并用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推文.html" target="_blank">
  推文
 </a>
 记录和展示集会盛况。
 <br/>
 <br/>
 周日，在田纳西州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 竞选造势集会上，总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和副总统彭斯同时现身，美国歌手格林伍德的一曲《上帝保佑美国》，将现场气氛推向高潮。
 <br/>
 <br/>
 从3月10日到周一，川普在全美一共举办了44场有关中期选举的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选集会.html" target="_blank">
  助选集会
 </a>
 。而周一，川普更是以72岁年龄，连续在俄亥俄，印第安纳和密苏里州举办三场集会。而川普经常将集会上支持者人数之多、场面之热烈的盛况，在自己的推特上呈现。
 <br/>
 <br/>
 周日，川普结束在田纳西州的助选集会后，发布了四幅照片。川普作为集会主角成为照片主体，以热情且人数众多的支持者为背景，足以显示集会盛况。但以这种方式呈现的照片在主流媒体的报导中往往较少见到，但却是川普推特中的主流风格。
 <br/>
 <br/>
 例如还有在11月3日的蒙大拿州集会，11月1日的密苏里州集会，等等。集会人群头戴的红色MAGA帽则构成了川普集会最标志性的景观。
 <br/>
 <br/>
 此外，伴着众多等待人群时，川普总统座驾〝空军一号〞在日落时分抵达的景象。
 <br/>
 数以万计的集会人群，在门外排队入场的超长队伍，都成了川普
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推文.html" target="_blank">
  推文
 </a>
 上展现人气的选择。
 <br/>
 <br/>
 川普并在自己11月1日的推文中，历数自己过去两年实现的减税、任命大法官、重振经济、重建军队等政绩，呼吁选民投票给共和党。
 <br/>
 <br/>
 美国总统川普：〝你们得去投票。但是我也得说，空气中弥漫的这种热烈气氛，是我和你们自2016年大选以来从未见过的。〞
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
