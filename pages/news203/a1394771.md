### 谁是美国下任驻UN大使 川普：拟提名鲍威尔
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】美国总统川普（特朗普）周二（9日）宣布，接受了驻联合国（UN）大使黑利的辞呈并否认外界他将任命女儿伊万卡出任此职的揣测。透露将考虑任命高盛主管、前副国家安全顾问鲍威尔接替黑利的职位。
 <br/>
 <br/>
 川普周二宣布，现任驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国大使.html" target="_blank">
  联合国大使
 </a>
 黑利（Nikki Haley）将于年底前辞职。随后，他在离开白宫前往爱荷华州参加活动时告诉记者上述消息。
 <br/>
 <br/>
 如果川普提名鲍威尔担任驻联合国大使，她还需要得到参议院的确认。
 <br/>
 <br/>
 鲍威尔（Dina Powell）在川普就任第一年担任副国家安全顾问，负责国家战略，是中东外交行动的要角。她于去年12月离开白宫，今年稍早返回她工作10多年的高盛。她曾在小布什总统时期担任国务院高阶官员。
 <br/>
 <br/>
 川普的数位顾问都称赞鲍威尔的工作表现，并且认为她归队的可能性极大。
 <br/>
 <br/>
 3名知情人士告诉CNBC新闻，在黑利周二表达辞意的前几天，白宫高级官员已经与鲍威尔接触，讨论她接替黑利担任驻联合国大使的可能性。一位不愿透露姓名的人士说，鲍威尔是川普的重要候选人之一。
 <br/>
 <br/>
 白宫发言人桑德斯曾说，鲍威尔一直是本届川普政府中值得信赖的重要顾问，她一直计划在做完一年后返回纽约，在民间继续支持川普总统的议程，并致力于中东政策。
 <br/>
 <br/>
 鲍威尔生于埃及，能说流利的阿拉伯语。她随着家人移民到美国，在白宫任职期间，深入参与川普政府中东事务，协助库什纳（Jared Kushner）恢复以色列－巴勒斯坦和平进程，也曾在川普的总统政权移交时期担任伊万卡的顾问。
 <br/>
 <br/>
 对于黑利的辞职，川普周二透露，黑利早在6个月前就曾告诉他，她想休息一下，但她会工作到今年年底。川普表示黑利将协助他挑选最终的继任人选。他还称赞黑利是一位〝非常特别〞的人，她的工作〝非常出色〞，她本人也〝非常棒〞，他说他们一起〝解决了很多问题〞。
 <br/>
 <br/>
 因黑利当天与川普讨论离职事宜时，称赞过伊万卡（Ivanka Trump）和他的丈夫库什纳，外界揣测川普可能会任命女儿伊万卡为下任驻联合国大使。
 <br/>
 <br/>
 但川普否认了此说法，他说，虽然伊万卡是极佳人选，但如果由他委任她，将会被指责〝任人唯亲〞。
 <br/>
 <br/>
 对于外界传言黑利要角逐2020年总统大选，黑利表示，〝我没有要角逐2020年总统大选〞，她会全力支持并帮助川普竞选连任。她也感谢川普让她有机会担任联合国大使、为国服务，是〝毕生荣耀〞。
 <br/>
 <br/>
 在担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国驻联合国大使.html" target="_blank">
  美国驻联合国大使
 </a>
 前，黑利是南卡罗来纳州州长。她表示，6年州长职务再加上2年的联合国大使，经历这样密集繁重工作后，她认为〝是该告一段落的时候了〞。同时，她也感谢家人的全力支持。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
