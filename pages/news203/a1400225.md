### 白宫记者晚宴换主讲嘉宾 川普：明年或出席
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】明年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫记者协会.html" target="_blank">
  白宫记者协会
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晚宴.html" target="_blank">
  晚宴
 </a>
 将不再邀请谐星担任主讲嘉宾，而是改邀专门研究美国总统生平的历史学家契尔诺担纲。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）周二（20日）发推称，可能会出席明年的晚宴？
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周二发推文说，所谓的喜剧演员沃尔夫（Michelle Wolf）去年在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫记者协会.html" target="_blank">
  白宫记者协会
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晚宴.html" target="_blank">
  晚宴
 </a>
 表现奇差。〝（2019年举行的晚宴）将是数十年来，他们首度邀请作家（担任主讲嘉宾）而非谐星。这是让这个垂死的夜晚和传统重整旗鼓的良好一步！也许我会参加？〞
 <br/>

 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫记者.html" target="_blank">
  白宫记者
 </a>
 协会之所以做出上述决定，是因为担任今年晚宴主讲嘉宾的谐星沃尔夫（Michelle Wolf）开了一连串低俗玩笑，引爆争议。包括针对个人的下流贬损言语，就连在场的白宫新闻秘书桑德斯（Sarah Sanders）也遭无情嘲讽。
 <br/>
 <br/>
 时任白宫记者协会会长塔莱夫（Margaret Talev）说，沃尔夫的行为〝不符合〞颂扬新闻自由和〝重视礼节〞的活动精神。
 <br/>
 <br/>
 将于2019年4月27日举行的白宫记者协会晚宴的嘉宾演说，将由史学家兼传记作家契尔诺（Ron Chernow）担纲，他曾撰写过美国开国总统华盛顿（George Washington）、第18位总统葛兰特（Ulysses S Grant）和美国首任财政部长汉密尔顿（Alexander Hamilton ）等人的生平。
 <br/>
 <br/>
 白宫记者协会主席诺克斯（Olivier Knox）说：〝我们庆祝自由独立新闻媒体对于这个国家健全的重要性的同时，我期待听到朗（契尔诺）把这个不同寻常的时刻放进美国历史脉络中。〞
 <br/>
 <br/>
 契尔诺也发表声明表示：〝虽然我从没被误认成谐星，但我保证自己的历史课不会枯燥乏味。〞
 <br/>
 <br/>
 据公开的资讯，由记者协会在白宫组织的每年一次的年度晚宴，是持续了将近一百年的传统，该活动通常是在每年4月的最后一个周六举行。近百年来的历任美国总统中，尼克松、卡特、里根和肯尼迪都曾因各自不同的原因缺席过白宫记协晚宴。
 <br/>
 <br/>
 通常来说，美国国家元首在参加白宫记协晚宴时，都会发表一番内容轻松的讲话。从1983年开始，美国总统在记协晚宴上发表的演讲开始以嘲讽媒体并自嘲为特色，主持人藉机嘲讽总统也逐渐成为传统。
 <br/>
 <br/>
 但有时候对总统的嘲讽会引发不满而出现尴尬场面。在2006年的记协晚宴上，脱口秀主持科尔伯特（Steven Colbert）对时任总统小布什的嘲讽，就导致小布什的几名幕僚大为反感而中途离场以示抗议。
 <br/>
 <br/>
 川普从参加总统大选以来，不论是选战期间还是就职总统以后，美国不少媒体一直对他保持着批评、挑剔甚至是抹黑的立场。川普为此常常批评一些主流媒体的报导预设立场，不公正不真实。
 <br/>
 <br/>
 川普上任以来，连续2年拒绝出席白宫记者协会晚宴，而是选择前往2016年胜选关键州与民众一起欢庆美国再次伟大。2017年他去的是宾州，今年4月则选择前往密歇根州华盛顿镇，与民众一起庆祝美国经济复苏。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
