### 〝活摘〞阿拉巴马州大学放映 观众要为法轮功发声
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】纪录片《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活摘.html" target="_blank">
  活摘
 </a>
 》（Human Harvest），记录了两位诺贝尔和平奖提名人调查在中国发生的活摘器官及非法交易的故事。10月25日，该片在阿拉巴马州莫比尔市（Mobile）春山学院学生中心进行了放映。观众们感到震惊，也意识到需要做些什么来制止。
 <br/>
 <br/>
 反映中国大陆非法器官摘取情况的纪录片《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活摘.html" target="_blank">
  活摘
 </a>
 》（Human Harvest）这次来到了阿拉巴马州莫比尔市的春山学院。学校的教职员工观看了影片。
 <br/>
 <br/>
 观众Daniel Matthews：〝在全球医学圈有很多组织机构，应该理解必须停止去中国的器官移植旅游，如果他们真的是去那里为了拿到器官。这必须停止。而在美国国内，应该从经济上施压中国。〞
 <br/>
 <br/>
 观众Sarah Hyde：〝我希望在美国为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 发声，让人们真的了解并认识到这是一个问题，应该为此而做些什么。〞
 <br/>
 <br/>
 医生反强摘组织Glynn Weldon Gilcrease：〝今晚我们说到已经制定的一些相关法律，州和联邦层面的。我们看到了变化和更多认知，至少在政治层面。但是，在医学领域对这个情况的了解依然很匮乏。〞
 <br/>
 <br/>
 电影观摩会后，医生反强摘组织与观众交流，并邀请到中国大陆强摘器官的主要受害群体——法轮功学员，讲述亲身经历。
 <br/>
 <br/>
 法轮功学员罗伊：〝当我开始他们我经历的迫害的时候，我当时就感觉几乎是难以控制自己的情绪。但是，总体的感觉，我非常高兴，能够有这样一个机会，来这里，告诉这些年轻的美国人真相和我所经历的迫害。〞
 <br/>
 <br/>
 罗伊5年前从大陆劳教所被释放后来到美国寻求庇护。因为坚持修炼法轮功，罗伊不仅失去了大学教师的工作，还被中共非法判刑、劳教。在劳教所罗伊不仅遭受各种酷刑还反覆被体检。
 <br/>
 <br/>
 观众Bob Poirier：〝这（强摘器官）是邪恶。我希望能让更多人了解。〞
 <br/>
 <br/>
 新唐人记者王琼、岳磊、李兰阿拉巴马莫比尔采访报导
</div>
