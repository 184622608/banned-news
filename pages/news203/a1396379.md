### 美支付非法移民生育费惊人 远超筑边境墙资金
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】根据美国人口普查局（Census Bureau）日前的数据分析，美国政府为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 生育子女所支付的费用数额巨大，正超过国会今年计划给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境墙.html" target="_blank">
  边境墙
 </a>
 的拨款。
 <br/>
 <br/>
 周一（10月22日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在推特上转发了保守派媒体《华盛顿观察家报》（The Washington Examiner）10月9日的一篇报导，再次指出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 问题的严重性。
 <br/>
 <br/>
 该报导称，一份新的报告揭示，2014年，非法移民在美国出生的孩子高达297,000人，耗费美国纳税人24亿美元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_资金.html" target="_blank">
  资金
 </a>
 。
 <br/>
 <br/>
 令人震惊的是，这个数字居然比参议院今年批给川普筑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境墙.html" target="_blank">
  边境墙
 </a>
 的费用高出8亿美元。
 <br/>
 <br/>
 在民主党的阻挠下，参议院共和党几经努力，才得以拨给川普政府16亿美元用于建筑美墨边境墙。
 <br/>
 <br/>
 根据非营利组织移民研究中心（Center for Immigration Studies）发布的这份报告，如果把非法移民分娩可能所花的医疗费用和她们非法或合法生育的子女的早期护理费用加在一起，共计高达53亿美元。这笔钱相当于川普所要的筑墙总资金的1/5。
 <br/>
 <br/>
 目前已经确定，中期选举结束后，将就资助建边境墙问题展开一场全国性辩论。如果届时川普总统无法得到高于参议院所批准的16亿美元的筑墙资金，部分政府机构可能会再次关门。
 <br/>
 <br/>
 多年来，移民问题已给美国纳税人造成巨大经济负担。川普政府正在修改部分移民政策，从严考虑对领取福利的移民发放绿卡或批准入籍。
 <br/>
 <br/>
 另一方面，非法移民正企图大规模入境美国。据墨西哥主流日报〝El Universal〞10月22日报导，目前向美国挺进的非法移民已高达14,000人，其中多数已强行进入墨西哥境内，预计不日即将抵达美墨边界。
 <br/>
 <br/>
 （记者李佳欣编译报导/责任编辑：东野）
 <br/>
