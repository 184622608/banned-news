### 中期选举 休斯顿集会 川普挺克鲁兹
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周一晚出席休斯顿举办的竞选集会，支持共和党的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克鲁兹.html" target="_blank">
  克鲁兹
 </a>
 竞选连任联邦参议员。报名参加集会的人数超过预期，主办方更换场地，但仍有大量民众无法进场，在场外观看集会直播。来看记者发自现场的报导。
 <br/>
 <br/>
 记者：〝 丰田中心只能容纳一万八千名观众，但到目前为止，报名的人数已超过十万。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在民众的欢呼声中，步入会场。民众打着支持川普和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克鲁兹.html" target="_blank">
  克鲁兹
 </a>
 的标语，鼓掌欢呼，气氛热烈。
 <br/>
 <br/>
 川普说，美国的就业率创历史新高，与墨西哥、加拿大签订新的贸易协定，重申面对大篷车移民潮，将采取必要措施，保护美国边境安全。
 <br/>
 <br/>
 川普在演讲中赞扬克鲁兹，鼓励德州民众投他一票。
 <br/>
 <br/>
 美国总统川普：〝我认为他已经拉开竞争了，马上开始了。我认为，他们想看到我支持克鲁兹，使得我百分之百（支持他）。〞
 <br/>
 <br/>
 记者：〝民众们表示非常荣幸的听到总统川普的演讲。他们还表示，川普对于美国来说是一位非常伟大的总统。他们也希望2020年川普能够连任美国总统。〞
 <br/>
 <br/>
 集会民众 Susan Engelman ：〝川普将成为美国历史上最棒的总统，即使现在或以后还没有人这样评价，但（他确实是）。他非常了不起，不到两年时间就有如此成就。〞
 <br/>
 <br/>
 集会民众 Joann Lombard ：〝我们很喜欢他。我们认为他很了不起，有很好的主意。他许诺什么，他都会兑现。这非常重要。〞
 <br/>
 <br/>
 集会民众 Paul Ochoa：〝这展现出他是多么伟大的总统，把失业率降低那么多。〞
 <br/>
 <br/>
 集会民众：〝我们很支持他的主张，建造边境墙，我们确实需要移民法律，让我们的边境更安全。〞
 <br/>
 <br/>
 集会民众：〝这给我们很大的鼓舞，让我们能在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 中积极投票。〞
 <br/>
 <br/>
 集会民众 Corina Borel：〝我相信他为我们国家所做的事，我们非常受鼓舞，期待2020年总统选举。〞
 <br/>
 <br/>
 克鲁兹与民主党参选人国会众议员奥罗克的选战，被认为是中期选举的一大看点。最新民调显示，克鲁兹领先奥罗克7个百分点。
 <br/>
 <br/>
 新唐人记者易永琦美国休斯顿采访报导
</div>
