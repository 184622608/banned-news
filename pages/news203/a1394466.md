### 【禁闻】英媒：白宫曾讨论禁止中国公民赴美留学
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】英国媒体日前报导，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 官员早前曾向川普总统建议，停止向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国公民.html" target="_blank">
  中国公民
 </a>
 提供学生签证。虽然该提议被暂时搁置，但一些美国官员仍继续力主采取更强硬立场。
 <br/>
 <br/>
 英国《金融时报》10月2号引述多名知情人士报导，今年早些时候在美国有关部门讨论如何应对中共间谍时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 助理斯蒂芬‧米勒（Stephen Miller）向川普和其他官员建议，应禁止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国公民.html" target="_blank">
  中国公民
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赴美留学.html" target="_blank">
  赴美留学
 </a>
 ，引发美国高层激烈辩论。
 <br/>
 <br/>
 今年春季在一次椭圆形办公室会议上，米勒与持相反观点的美国官员，包括美国驻华大使布兰斯塔德（Terry Branstad）进行了辩论。
 <br/>
 <br/>
 布兰斯塔德在会议前表示，米勒的计划会对较小的高校造成较大冲击，而不是资金充裕的常春藤联盟名校。美国驻华使馆官员们也强调经济方面的考虑。
 <br/>
 <br/>
 尽管布兰斯塔德最终说服了川普，决定先不采纳米勒的建议，但报导称，包括白宫贸易顾问、对华鹰派人士纳瓦罗（Peter Navarro）在内的一些官员，继续力主采取更强硬立场。美国政府内部对中共间谍威胁的担忧十分普遍，包括对美国大学里的中国学生和科研人员。
 <br/>
 <br/>
 全美中国学生学者自治联合会现任理事陈闯创：〝来到美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国留学生.html" target="_blank">
  中国留学生
 </a>
 他们在学习了美国的先进科技之后，他们没有接受美国更先进的民主制度，他们缺乏对美国政治的认同，导致他们受到中国（中共）利益的诱惑，他们回到中国，甚至在美国就帮助中国的企业和国家（政府）窃取机密。要么是呢他们自己实在是认同中国（中共）的体制，回去为中国（中共）效劳。〞
 <br/>
 <br/>
 今年6月，美国国务院宣布收紧对部分中国留学生的签证，以降低敏感学科情报被窃取的风险。
 <br/>
 <br/>
 而多名涉嫌转移美国敏感技术的华人学者、工程师，近期陆续遭到美国逮捕、判刑或拒签。有消息指，中共的〝千人计划〞已被FBI紧盯。
 <br/>
 <br/>
 10月1号，中国籍女子陈思因非法向中国出口敏感技术，被判刑46个月，陈思2007年来美国读会计专业研究生。
 <br/>
 <br/>
 陈闯创：〝这项关于中国留学生的签证限制，其实他们达成一个妥协，不是说所有的出生在中国大陆的人都不能在美国就读这些高科技的行业，有三类人是例外的。第一类是在国外获得难民、逃离中共的中国人。第二类和第三类是说，要么是获得外国国籍，要么是获得外国永久居留权，像绿卡这样的。我觉得加一个这样的区分还是比较好的。〞
 <br/>
 <br/>
 全美中国学生学者自治联合会现任理事陈闯创认为，中共利用留美学生来窃取机密、影响舆论，美国作为被动方，对中国留学生施加一定的签证限制是合理的。但如何既能保证美国安全，又能给予一定例外，还是很关键的。例如有一些对中共持不同态度，受到惩罚的中国学生，他们不应受到限制。
 <br/>
 <br/>
 美国华府人权律师叶宁表示，相关提议将来能否成为全面禁止令，取决于中美之间是否会发生全面的对抗。
 <br/>
 <br/>
 美国华府人权律师叶宁：〝如果有危机升级失去管控的话，那肯定会全面禁止两国之间文化交流的项目。但是如果没有的话，我估计这个审查和监控肯定会严格，但是全面禁止的可能性不大。〞
 <br/>
 <br/>
 今年2月，美国FBI局长雷（Christopher Wray）表达了对中共〝学术间谍〞的担忧。
 <br/>
 <br/>
 10月1号起，美国执行移民新规，扩大遣返范围，对欺诈、犯罪的外国人，他们的被遣返会导致失去美国合法身分。
 <br/>
 <br/>
 陈闯创：〝对于那些伤害美国利益的华人依法的实施处罚。对于那些有利于美国国家安全的，你像不管是参与到民主政治运动中，还是说像法轮功的朋友、其他宗教信仰，维族，藏族的朋友实施更大的保护以后，扭转华人社会上亲共的、恐共的、被中共收买的那样一个氛围还是比较容易的。我是希望尽早看到美国政府这么做。〞
 <br/>
 <br/>
 据报导，米勒的反对者们担心，川普总统可能会重新考虑相关提议，特别是他对中共采取越来越强硬的立场。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/肖颜
