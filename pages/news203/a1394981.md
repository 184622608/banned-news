### 迈克尔飓风重创佛州 至少两人死亡
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】首先来关注飓风消息。周三下午，伴随着狂风暴雨，飓风迈克尔横扫佛罗里达狭长地带，成为该地区有史以来最强大的风暴。目前已有两人死亡，沿岸37.5万人接到撤离警告。
 <br/>
 <br/>
 飓风迈克尔登陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 巴拿巴市，风速高达250公里，威力相当于四级飓风。
 <br/>
 <br/>
 暴雨加狂风，场面触目惊心。
 <br/>
 <br/>
 大树连根拔起。
 <br/>
 <br/>
 庭院和街道上的大树，被大风吹倒。汽车泡在洪水中。
 <br/>
 <br/>
 房屋建筑被摧毁，电线刮断。
 <br/>
 <br/>
 巴拿马市居民罗林斯用手机拍下飓风肆虐后的景象。
 <br/>
 <br/>
 洪水淹没街道，吹倒的大树砸坏房屋，庭院里的栅栏、门板都被飓风毁坏，屋顶铁皮剥落。人们忙着清理被毁坏的铁架。
 <br/>
 <br/>
 巴拿马市居民拉蒙尼卡屋前的大树被吹断，差一点砸到房屋。
 <br/>
 <br/>
 巴拿马市居民 Andrew Lamonica: 〝这是64年来我第一次经历。我是从电力公司退休的，见过很多次风暴。这是我见过最严重的一次。总之，这里还好，你看看它干的，毁坏一颗大树就像折断一根牙签一样容易。〞
 <br/>
 <br/>
 当地警方通报，巴拿马市一名男子因房屋被大树砸毁，不幸身亡。另外还有一人死亡。
 <br/>
 <br/>
 川普总统周三晚推文，并发视频说：〝佛罗里达高速路巡逻队从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 各地赶赴狭长地带，帮助受飓风迈克尔影响的居民。如果你见到他们，别忘了和他们握手，说声谢谢。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔飓风.html" target="_blank">
  迈克尔飓风
 </a>
 登陆数小时后，减弱到三级，风速降到每小时150公里，向佐治亚州西南地区推进。
 <br/>
 <br/>
 新唐人记者安心、明玉综合报导
</div>
