### 佩洛西宣布角逐众院议长 党内已获压倒性支持
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】美国国会众院民主党领袖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佩洛西.html" target="_blank">
  佩洛西
 </a>
 (Nancy Pelosi)日前明确表态将角逐众院议长。虽然有17名众院民主党人联署公开信对此表示反对，但舆论普遍认为，除了佩洛西，在民主党人中很难找到第二个大家都接受的众院议长人选。
 <br/>
 <br/>
 民主党在今年的期中选举获得了众议院的230个席位，因此有望以多数席位的优势重掌众议院，而新一届众院预定明年1月3日选出下任议长。
 <br/>
 <br/>
 曾在小布什总统任内当过众院议长的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佩洛西.html" target="_blank">
  佩洛西
 </a>
 (Nancy Pelosi)15日宣布，自己将参与竞争出任下任众院议长。她同时对外表示，自己现在并获得民主党人的〝压倒性的支持〞。
 <br/>
 <br/>
 据美联社报导，按规则，佩洛西要确保胜选，至少需要获得218票支持。考虑到共和党人不大可能把选票投给民主党的候选人，因此佩洛西需要得到绝大多数民主党议员的支持。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-17/p9129821a204309493.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-17/p9129821a204309493-ss.jpg"/>
   <br/>

    美国众议院少数党领袖南希佩洛西2018年11月15日在华盛顿特区国会山举行的每周新闻发布会上发表讲话。(Alex Wong/Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 <br/>
 事实上，民主党中的确有不少人都支持佩洛西角逐众院议长。他们认为，作为众院议长将肩负为重大法案把关的责任，而佩洛西在这方面拥有傲人的资历。
 <br/>
 <br/>
 支持者认为，佩洛西在众院有30年丰富经验，曾是拨款委员会和情报委员会的成员，也担任过民主党党鞭。她曾在众院任内经历了6任总统，经历过不少〝大风大浪〞，对于美国的政治体制有充分的了解，也富有运筹帷幄的经验，具备横跨各个领域的领导力。而在这些方面，其他挑战者无法与之抗衡。
 <br/>
 <br/>
 不过在民主党内部，以麻州联邦众议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莫尔顿.html" target="_blank">
  莫尔顿
 </a>
 (Seth Moulton)为首的17位众院民主党人却联署了一份公开信，明确表态反对佩洛西重掌众议院，他们当中，俄亥俄州联邦众议员玛西亚福(Marcia Fudge)更放话称，有其他众议员敦促她出来竞选议长。
 <br/>
 <br/>
 对此，佩洛西回应称，〝如果有任何人对女性参与其中心怀不满，那是他们的问题，不是我的问题。〞她说：〝我想要争取民主党人的支持票，赢得众院议长职务。〞
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
