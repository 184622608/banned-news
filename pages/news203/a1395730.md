### 穿越时空网络爆红 川普与前任总统谈笑风生画作
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）上周日（10月14日）在白宫接受 〝60分钟〞节目访谈时，白宫墙上挂着一幅共和党总统们在一起谈笑风生的画作，引起广大网友的热议。川普曾表示自己非常喜欢这张画，并亲自打电话向画家致谢。
 <br/>
 <br/>
 这幅颇具穿越时空感的画作名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党俱乐部.html" target="_blank">
  共和党俱乐部
 </a>
 〞（The Republican Club），是由美国当代画家安德鲁・汤马斯（Andy Thomas）所作。原版其实已经流传颇长一段时间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 是被新加入的角色。
 <br/>
 <br/>
 在这幅画中，川普与林肯（Abraham Lincoln）、尼克森（Richard Nixon）、里根（Ronald Reagan）、老罗斯福（Teddy Roosevelt）、老布什（George H. W. Bush）与小布什（George W. Bush）父子等多位前总统欢聚一堂。他们边喝酒水边聊天，脸上露出灿烂的笑容。
 <br/>
 <br/>
 川普则居中而坐，身穿白衬衫配红领带，面带微笑。因川普滴酒不沾，所以他面前摆放的是一杯加冰的可乐。
 <br/>
 <br/>

 <br/>
 节目播出后，眼尖网友发现了这幅画作，并上传网络，引起人们热议。
 <br/>
 <br/>
 汤马斯在接受福克斯采访时表示，他的太太偶然间在社交媒体上得知，川普把这幅画挂在白宫里让他们感到非常欣喜。
 <br/>
 <br/>
 汤马斯说，这幅画是由共和党籍加州联邦众议员艾沙（Darrell Issa）转交给川普。川普当时还亲自打电话给他。汤马斯说，〝他（总统）非常客气，（对作品有）很多赞扬。总统对这幅画赞不绝口，还说看过很多关于自己的画像，但很少有满意的。〞
 <br/>
 <br/>
 汤马斯透露，其实要画好川普很难，因为他金头发、白皮肤，脸上没有很深的阴影，所以画他是一个〝挑战〞。
 <br/>
 <br/>
 汤玛斯告诉〝时代杂志〞，自己并不懂政治，对于如何把川普嵌入共和党总统之中， 让他费尽心思，但不管如何安排，他都认定要把川普摆在中间位置最好。
 <br/>
 <br/>
 汤马斯来自密苏里州西南部，除了〝共和党俱乐部〞，他也画了另一幅〝民主党俱乐部〞（The Democratic Club）。
