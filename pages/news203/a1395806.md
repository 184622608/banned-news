### 美国地方报业大量关闭 研究：新闻沙漠来临
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】研究显示，自2004年至今，美国已有超过1800个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新闻刊物.html" target="_blank">
  新闻刊物
 </a>
 停刊，或是几乎没有公共事务的报导。部分不上网的民众难以接触地方教育与税收资讯。
 <br/>
 <br/>
 据《法新社》报导，美国北卡罗来纳大学传播研究所的这项研究称，过去15年来，包括日报与周报，有1/5的地方报业面临关门危机。全美有3143个郡只有一家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地方新闻.html" target="_blank">
  地方新闻
 </a>
 ，且多为周报。有200个郡甚至完全没有地方新闻。
 <br/>
 <br/>
 目前仍有7100家地方新闻艰难维生，但因为新闻从业人员减少，只是打着老招牌在苦撑。
 <br/>
 <br/>
 据美国独立民调机构与智库〝皮尤研究中心〞（Pew Research Center）今年公布的调查结果显示，自2008年以后，美国新闻业的聘用人员减少了23%。
 <br/>
 <br/>
 报告称，即使网路上有大量的国内和国际新闻，但报导地方税收、教育等议题的新闻正在减少，地方媒体正处于危机之中。而那些最没机会接触到地方新闻的少数人，也是最穷、教育水准最低的群体。低收入社群正面临〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新闻沙漠.html" target="_blank">
  新闻沙漠
 </a>
 〞危机。
 <br/>
 <br/>
 在广告与读者不断流向广告费用便宜且新闻不收费的网路之际，美国报业正面临转变的关键时刻。
 <br/>
 <br/>
 (记者李红报导/责任编辑：范铭）
</div>
