### 宾州是焦点？美中期选举众院席位争夺激烈
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】11月6日，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的大幕将正式拉开，共和党与民主党对众议院席位的争夺，是外界关注的重头戏，各种分析与预测纷纷出笼。日前，有媒体报导称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宾州.html" target="_blank">
  宾州
 </a>
 将成两党争夺对众议院控制权的决胜点。不过，也有分析认为，目前两党在多州的竞争都很激烈，而摇摆不定的29个选区是关键。
 <br/>
 <br/>
 路透社日前报导称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宾州.html" target="_blank">
  宾州
 </a>
 的国会众议员选举，可能是决定哪个党会取得众院主导权的决胜点。报导引述匹兹堡共和党策略师狄凡尼(Mike DeVanney)称，〝宾州将是指标。这里将决定众议院由谁掌握。〞
 <br/>
 <br/>
 报导分析称，宾州的民主党支持度低落曾经是2016年川普能够当选美国总统的一大原因。当年在匹兹堡(Pittsburgh)外的宾州第12选区两度连任众议员罗斯福士(Keith Rothfus)，其所在的选区在2016年总统大选时，川普的得票率胜出20个百分点。
 <br/>
 <br/>
 但如今宾州选区范围在法院命令下大改，罗斯福士被划入了新的宾州第17选区，而这里的选民中近半是民主党选民，仅36%是共和党人，因此形式可能发生改变。
 <br/>
 <br/>
 该报导并援引维吉尼亚大学政治研究中心的观点称，在宾州的18席联邦众议员席次中，民主党目前拥有6个国会众院议席，而该党派有可能在这次选举中赢得九席以上的席位。
 <br/>
 <br/>
 不过，这也只是一家之言。
 <br/>
 <br/>
 无党派选举分析机构库克政治报告（Cook Political Report）11月2日的最新报告却指出，今年众议院435个席位全部需要选举，而其中大约有75个席位的竞争都很激烈，尤其29个选情摇摆不定的选区的席位争夺最为激烈。
 <br/>
 <br/>
 此外，在两党传统上占据优势的地方，也有部分选区的选情比较胶着，主要包括：西部加州（10、25、39、45和48选区）、东北部的宾州、纽约（19和22选区）和新泽西（3和7选区）以及中西部的爱荷华、伊利诺伊和明尼苏达等地区。
 <br/>
 <br/>
 最终鹿死谁手，尚需拭目以待。
 <br/>
 <br/>
 （记者文轩综合报导/责任编辑：东野）
