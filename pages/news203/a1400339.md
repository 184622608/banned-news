### 台北经文处赴侨教中心受理海外就养荣民文件验证
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】驻纽约台北经济文化办事处为服务辖区内年长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_荣民.html" target="_blank">
  荣民
 </a>
 ，由领务秘书于今日(11月17日)下午2时30分至4时30分，在位于法拉盛之华侨文教服务中心受理〝旅居海外长期居住就养荣民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_验证.html" target="_blank">
  验证
 </a>
 声明书〞验证申请案，以减轻辖区内年长荣民赴位于曼哈顿之经文处申办舟车劳顿。
 <br/>
 <br/>
 每年12月是中华民国国军退除役官兵辅导委员会受理海外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_荣民.html" target="_blank">
  荣民
 </a>
 提缴就养
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_验证.html" target="_blank">
  验证
 </a>
 声明书的期限。凡是曾经向国内荣民服务机构完成旅居海外长期居住就养报备手续的荣民，都可以向经文处申办〝旅居海外长期居住就养荣民验证声明书〞验证，以便向国内相关安养单位请领就养金，不必亲自返国办理。
 <br/>
 <br/>
 除今日以外，经文处另将于12月1日周六下午2时30分至4时30分再次在华侨文教服务中心受理相关验证，尚未申请者请把握本年最后一次机会，备妥有效中华民国护照正本及影本各1份，并携带填妥之〝文件证明申请表〞及〝旅居海外长期居住就养荣民验证声明书〞至华侨文教中心亲自签字并缴交15美元验证费用，经文处于受理后将代为转寄退辅会分转各县市安养单位核办。
 <br/>
 <br/>
 华侨文教服务中心地址是133-32 41Rd. Flushing, NY11355，如有其他问题，可电洽212-486-0088向经文处领务组询问。
 <br/>
 <br/>
