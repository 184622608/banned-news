### 川普爱荷华演説：民主党已走上极端
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】周二，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱荷华州.html" target="_blank">
  爱荷华州
 </a>
 集会，为共和党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 助阵。在演説中，川普表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 已经走上极端，就是想把他弹劾下台。
 <br/>
 <br/>
 川普在演説中，祝贺卡瓦诺克服困难，成功就任最高法院大法官。
 <br/>
 <br/>
 美国总统 川普：〝如今已经就职的卡瓦诺法官和戈萨奇法官一起，致力于保护你们的权利、宪法和神赋予的自由。〞
 <br/>
 <br/>
 川普特别感谢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱荷华州.html" target="_blank">
  爱荷华州
 </a>
 参议员、参议院司法委员会主席格拉斯利，以及其他共和党参议员，共同抵制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 人针对卡瓦诺个人名誉发起的运动。
 <br/>
 <br/>
 川普直言，民主党人就是要推翻现任政府。
 <br/>
 <br/>
 美国总统 川普：〝上周他们（民主党人）说，我们要弹劾他（卡瓦诺）。弹劾他什么？再说，得先弹劾我吧，不是得我先来吗？尽管我们没做任何错事，还创造了国家历史上最好的经济形势之一。〞
 <br/>
 <br/>
 川普说，民主党已经走上极端，把权力交给他们是危险的。
 <br/>
 <br/>
 美国总统 川普：〝你们不能把火柴递给纵火犯，也不能把权利交给愤怒的左派一伙，他们已经成了乌合之众。〞
 <br/>
 <br/>
 集会上，川普还点名批评几位民主党议员的所为，尤其是在推动卡瓦诺提名风波中起到关键作用的加州参议员范士丹。
 <br/>
 <br/>
 民众呼喊：〝把她关起来。〞
 <br/>
 <br/>
 美国总统 川普：〝我想他们说的是范士丹。你相信吗？〞
 <br/>
 <br/>
 由于集会地点靠近内布拉斯加州，许多邻州选民也赶来表达对川普的支持。川普集会后在推特发布现场照片，感谢两州选民，再次呼吁他们出来为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 投票。
 <br/>
 <br/>
 新唐人记者林宇、天羽综合报导
</div>
