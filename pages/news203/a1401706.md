### 时代广场求婚 戒指掉天井 纽约警察找到
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】在男士单膝跪地向女士
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_求婚.html" target="_blank">
  求婚
 </a>
 后，总要浪漫地给女方戴上订婚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戒指.html" target="_blank">
  戒指
 </a>
 。11月30日午夜前，纽约
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_时代广场.html" target="_blank">
  时代广场
 </a>
 再次出现这浪漫一幕，可令人哭笑不得的是，男方求婚成功后，竟把戒指掉进了路边的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天井.html" target="_blank">
  天井
 </a>
 。
 <br/>
 <br/>
 从纽约
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_时代广场.html" target="_blank">
  时代广场
 </a>
 的监视录像中我们可以看到，男子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_求婚.html" target="_blank">
  求婚
 </a>
 成功后，发现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戒指.html" target="_blank">
  戒指
 </a>
 不见了。他起身后向未婚妻说些什么，然后立即俯下身，把手伸进
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天井.html" target="_blank">
  天井
 </a>
 里去找。可能发现还够不到，男士干脆趴了下去。
 <br/>
 <br/>
 据媒体报导，男士最终没有够到戒指。他在求婚时，把订婚戒指摆在了路边天井的井盖上，而这个天井是格子状的，戒指从格子缝隙掉了下去。
 <br/>
 <br/>
 不过幸运的是，纽约警察注意到了这件事，他们后来前去寻找，不仅找到了戒指，还把它洗净装好。
 <br/>
 <br/>
 看！纽约警局的新闻推特于周六下午贴出了这枚定情信物，并写下了联络电话寻找失主。
 <br/>
 <br/>
 周日上午10点多，好消息到了！纽约警局贴出推文，称找到了这对未婚夫妇，他们来自英国，现已回家，纽约警方已对归还戒指做好了安排。
 <br/>
 <br/>
 新唐人记者李大宇综合报导
 <br/>
