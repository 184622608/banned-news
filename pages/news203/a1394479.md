### 美军B-2隐形轰炸机 首次常态化部署夏威夷
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 印太司令部近日宣布，美国空军刚刚在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_夏威夷.html" target="_blank">
  夏威夷
 </a>
 完成了首次常态化部署B-2隐形战略轰炸机，此举将对确保印太地区的自由与开放发挥重要作用。
 <br/>
 <br/>
 据美国之音报导，近年来，为了应对亚太地区越来越大的安全挑战，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 经常以轮调的方式将B-2战略轰炸机派驻关岛。这次在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_夏威夷.html" target="_blank">
  夏威夷
 </a>
 珍珠港希卡姆（Hickam）美军联合军事基地的部署，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_B-2轰炸机.html" target="_blank">
  B-2轰炸机
 </a>
 首次在印太地区的常态化部署，显示美军继续实施将最先进武器优先派驻印太地区的计划。
 <br/>
 <br/>
 据美国空军官方网站周四（10月4日）发表的新闻稿中称，作为美军战略司令部展开的轰炸机特遣部队部署的一部分，今年8月15日至9月27日间，美国空军三架B-2〝幽灵〞（Spirit）隐形战略轰炸机和200名空军人员，在夏威夷和印太其他地区展开了数以百次的飞行训练，包括与F-22隐形战斗机的伴飞、武器装载和热机加油训练。
 <br/>
 <br/>
 太平洋空军总部的空军与网路行动主管威廉斯少将（Maj.Gen.Stephen Williams）表示：〝B-2轰炸机首次部署（珍珠港希卡姆基地），突显美军可以在世界任何地方投射武力的战略灵活性。B-2与主要区域合作伙伴执行了例行空战任务并展现了综合能力。这次的演练有助于确保印太地区的自由与开放，让美国通过全球部署和军事力量一体化表明对盟友和伙伴的承诺。〞
 <br/>
 <br/>
 这次演练的其中一个关键整合涉及分配到夏威夷国民警卫队第154联队199飞行中队的B-2幽灵轰炸机和F-22猛禽战机。
 <br/>
 <br/>
 据悉，B-2〝幽灵〞战略轰炸机是美军最先进的远程轰炸机，可以不用加油持续飞行近7000英里（约1.12万公里），主要基地位于密苏里州的怀特曼空军基地（Whiteman Air Force Base）。
 <br/>
 <br/>
 美国空军B-2隐形轰炸机已经完成了对投掷新型多功能B61-12核重力炸弹的测试。此类炸弹提高了准确性，将各种攻击选项集成到一枚炸弹中，在同一领域领先中俄，改变了核武器战略格局。也使B-2轰炸机准备好应对核战争。
 <br/>
 <br/>
 F-22与B-2相同，都具备隐形能力，这使得它们成为护卫与轰炸的完美搭档。F-22同时也能为B-2提供战场信息与引导，此种训练有助于提高飞行员之间的凝聚力。
 <br/>
 <br/>
 除了空中行动，这次部署还集中进行了热机加油训练，这是一种飞机着陆后快速加油的方法，即轰炸机降落后在不关闭引擎的状态下直接进行加油作业。这项技术能最大限度利用轰炸机在地面停靠的时间，节省后勤工作时间，缩小遭到导弹攻击的空窗期，确保美国空军轰炸机部队的有效作战力量，并时刻保持战备状态。此外，B-2还与KC-135空中加油机进行了空中加油演练。
 <br/>
 <br/>
 上周，美国海军陆战队8架MV-22型鱼鹰倾转旋翼机成功完成了跨太平洋5300海里的远程飞行。分析人士说，美军的这项能力对于应对南中国海潜在的军事冲突具有重要的战术意义。
 <br/>
 <br/>
 报导引述防务专家们表示，美军正分散在太平洋地区的军事资源，以减少受到中共军方攻击的风险，但如果发生军事冲突，美军仍然需要迅速从不同地方调派资源，因此B-2战略轰炸机和MV-22型鱼鹰倾转旋翼机的远程飞行能力非常重要。
 <br/>
 <br/>
 美国空军目前拥有20架B-2隐形轰炸机，其中大部分位于密苏里州的怀特曼空军基地。B-2可以飞到50,000英尺高度，并携带40,000磅的有效载荷，包括常规武器和核武器。
 <br/>
 <br/>
 为了能够应对复杂的高端现代威胁，美国空军对上世纪80年代投入使用的B-2轰炸机进行了一系列升级改造。
 <br/>
 <br/>
 B-2隐形轰炸机正在改进其数字武器集成、新的计算机处理能力，以获得比现有系统快1000倍的系统，而下一代传感器旨在帮助飞机躲避敌方防空防御。
 <br/>
 <br/>
 其中一项主要修改旨在改进轰炸机的防御管理系统，这是一种帮助B-2识别和避开敌方防空系统的技术，可使用各种天线、接收器和显示处理器完成这个功能。
 <br/>
 <br/>
 此外，B-2还计划与最新型下一代隐形轰炸机B-21 Raider一起飞行。许多空军开发商表示，这个平台将在2020年代中期做好准备，其中包括新一代隐形技术，这些技术大大扩展了现有隐形轰炸机的作战范围和能力。
 <br/>
 <br/>
 事实上，空军领导人已经表示，B-21将能够随时在世界范围内控制任何目标。
