### 台媒：川普小输为赢 中南海忧惧加深
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 结果尘埃落定，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 重夺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 控制权，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 保住
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 优势，有台媒说，这其实是正常的，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 这次选举算〝小输为赢〞。但中共政情却依然扑朔迷离，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 面临抉择，中南海忧惧加深。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的最终结果是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统所在的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 ，失去对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 的控制权，赢得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 222席，增加了29席；共和党则减少了36席，达到199席。
 <br/>
 <br/>
 台湾《信传媒》9日发表评论认为，这其实是很正常的，因为只要哪一党当上总统，另一党通常在国会期中选举都会过半，甚至大胜，这几乎是〝钟摆效应〞下的惯例，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 的奥巴马在两次期中选举中都让共和党大胜。
 <br/>
 <br/>
 评论说，让人意外的是共和党在参议院竟然继续过半。这次选举川普算〝小输为赢〞，川普通过期中选举考验，他连任总统的机会更大了一些。而中共现在要紧张了，本来期待共和党大败，以此改变川普对中共的强硬立场，结果事与愿违。
 <br/>
 <br/>
 香港苹果日报评论说，无论共和党在中期选举中输或赢，都不会改变中美走向冲突的趋势。民主党在对华政策上与川普是一致的。如果北京寄望于众议院拖垮川普对中共的打压，到头来哭都没地方哭。
 <br/>
 <br/>
 台媒评论说，当美国政情大势底定，中共政情却依然扑朔迷离。中共现在正面临〝路线矛盾〞，与美国的贸易战究竟是〝战〞还是〝和〞？中共一直在等美国期中选举的结果出来之后再做决定，现在仍在斟酌中。
 <br/>
 <br/>
 时政评论员周晓辉分析说，曾以为川普坐不满一届总统，就可能被弹劾的北京当局，现在通过美国中期选举已意识到，川普2020年连任总统不再是不可能的事情。
 <br/>
 <br/>
 首先，共和党在参议院赢得多数党地位，确保了川普在人事任命上，不必受到民主党的掣肘。而民主党掌控众议院后，虽然可能在政府财政预算、与俄罗斯缓和等问题上发难，但此举不利于两年后的大选，因为川普大可将诸多责任推到民主党的不配合上，因此左右为难的民主党或许会在一些问题上与共和党、川普合作。
 <br/>
 <br/>
 其次，中期选举中两个重要摇摆州佛罗里达州、俄亥俄州州长，都在川普的大力支持下取得胜利，而民主党2020年竞选总统的热门候选人则纷纷落马，这对川普而言，绝对是个好消息。
 <br/>
 <br/>
 再者，有望明年接任众议院议长的民主党领袖佩洛西，自1986年首次当选众议员后，一直对中共持强硬立场，因此，北京将其视为〝敌对人物〞，未来与其打交道会相当头疼。
 <br/>
 <br/>
 总之，美国两党对北京均持有强硬立场并未因选举而发生改变，坚定反共的佩洛西将出任众议院议长并继续在人权问题上强势发声，川普将极有可能连任总统，以上三点只能加深中南海的忧惧，再加上贸易战和人权问题，北京何去何从？值得关注。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉)
