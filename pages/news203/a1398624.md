### 惠特克接任〝通俄门〞调查 穆勒着手写最终报告
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】美国司法部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞申斯.html" target="_blank">
  塞申斯
 </a>
 周三（7日）向总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）提交辞职申请后，川普发推文宣布惠特克担任代理司法部长。司法部当天发表声明称，调查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 案将由惠特克接任。
 <br/>
 <br/>
 周三下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞申斯.html" target="_blank">
  塞申斯
 </a>
 （Jeff Sessions）向白宫幕僚长凯利（John Kelly）递交了辞呈。他承认，他是应
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的要求辞职的。
 <br/>
 <br/>
 川普一直对塞申斯在处理调查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 的举措不满，因司法部任命的特别检察官穆勒（Robert Mueller）在经过近2年调查后，查无任何结果。而塞申斯一直回避通俄门调查，下放予副司法部长罗森斯坦跟进，川普批其为自保而不作为。同时，在移民以及行政令问题上，川普与塞申斯的意见有时也不统一。
 <br/>
 <br/>
 塞申斯辞职后，川普发推文宣布塞申斯的幕僚长惠特克担任代理司法部长后，司法部发言人发表声明称，副部长罗森斯坦（Rod Rosenstein）将不再负责监督通俄门调查，转而由惠特克（Matthew Whitaker）接替这项工作。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-9/p9118042a471693555.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-9/p9118042a471693555-ss.jpg"/>
   <br/>
   <font size="-1">
    司法部发言人7日发表声明称，副部长罗森斯坦（Rod Rosenstein 图）将不再负责监督通俄门调查。图为资料图片。（JIM WATSON/AFP/Getty Images）
   </font>
  </a>
 </center>
 <br/>
 惠特克曾是爱荷华州南区的检察官，2014年6月竞选爱荷华州联邦参议员失败。他曾公开批评穆勒调查通俄门去得太远。
 <br/>
 <br/>
 惠特克今年7月在CNN接受访问时表示，如果塞申斯被换下，可以预见的情况是，总统会行使休会任命，而新司法部长只需减少调查拨款，便可停止调查。对于通俄门调查，他也曾为该传媒撰写专栏，题目是〝穆勒对川普的调查去得太远〞。
 <br/>
 <br/>
 上个月，美国媒体就传出川普曾与惠特克会谈，而惠特克开始为担任司法部长一职进行〝准备工作〞。川普当时接受福克斯新闻采访时没有否认这一传闻。
 <br/>
 <br/>
 川普周三表示，将在稍后的日子里提名司法部长，新司法部长提名人将需要通过参议院确认，届时必定广受关注。
 <br/>
 <br/>
 另一方面，多名消息人士告诉CNN，通俄门调查已进入收尾阶段，穆勒已开始撰写最终报告。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-9/p9118041a479193354.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-9/p9118041a479193354-ss.jpg"/>
   <br/>
   <font size="-1">
    自2017年5月罗森斯坦指派穆勒（图）调查通俄门事件开始，至今未找到川普团队通俄的证据。图为资料图片。（Andrew Burton/Getty Images）
   </font>
  </a>
 </center>
 <br/>
 自2017年5月罗森斯坦指派穆勒调查通俄门事件开始，调查已持续近2年，但穆勒至今仍未发现任何川普团队勾结俄罗斯干扰2016年美国大选的证据。
 <br/>
 <br/>
 穆勒无论是调查前川普团队竞选主席马纳福特（Paul Manafort）、前国家安全顾问弗林（Michael Flynn），还是在压力下已与穆勒合作、对川普倒戈相向的川普律师科恩（Michael Cohen），都没能找出任何川普选战通俄的证据。
 <br/>
 <br/>
 川普一再批评这是一场政治迫害，并呼吁尽早结束这场导致美国政治分裂并耗资巨大的〝猎巫〞行动。
 <br/>
 <br/>
 同时，美国民众也对此调查表示厌倦。据美国有线电视新闻网（CNN）8月公布的民调结果显示，66%的受访者希望特别检察官穆勒在中期选举前结束通俄门调查，其中，共和党受访者中有72%的人持这种观点，民主党受访者中比率也高达57%。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
