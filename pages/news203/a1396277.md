### 川普近日将再推减税政策 中产阶级受益
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）近日表示，政府正在研拟对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中产阶级.html" target="_blank">
  中产阶级
 </a>
 的重大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_减税.html" target="_blank">
  减税
 </a>
 政策，预计将在11月6日美国中期大选前或更早推出。
 <br/>
 <br/>
 综合《路透社》和《华盛顿邮报》的报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周六（10月20日）在参加内华达州埃尔科（Elko）的集会后，在登上空军一号之前告诉现场记者，〝我们正在考虑对中等收入群体进行非常重要的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_减税.html" target="_blank">
  减税
 </a>
 。〞
 <br/>
 <br/>
 川普透露，这次〝不是为企业而是为中等收入者（减税）。〞他的政府正在对这个减税政策进行深入研究。
 <br/>
 <br/>
 对于减税政策将于何时公布，川普说，可能在11月1日左右，也许可能还会更早一点。但他没有对减税政策透露更多细节。
 <br/>
 <br/>
 白宫没有对就新减税政策回复《华盛顿邮报》的置评要求。但一位熟悉此事的官员表示，川普已经督促国会领袖们在中期选举前推出这套减税方案。
 <br/>
 <br/>
 外界认为，这项新的减税政策的推出正值中期选举前的关键时期，具有重大意义。美国将在11月6日进行中期选举。随着中期选举的临近，川普总统的助选活动也越来越频繁。川普和共和党人都希望共和党能够在参议院和众议院继续占据多数，这将更有利于川普今后推行他的政策。
 <br/>
 <br/>
 众议院筹款委员会(House Ways and Means Committee)主席布拉迪（Kevin Brady）的发言人达穆森（Rob Damschen）表示，川普政府持续努力在减税与就业法案成功的基础上再接再厉，并不断改善勤劳家庭和美国小企业的税法。
 <br/>
 <br/>
 去年12月，川普签署了自1980年代以来最大规模的税改。这项减税政策将公司税率从35％降至21％，并暂时减轻了大多数人的税收负担。
 <br/>
 <br/>
 美国众议院在今年9月底通过了第二轮减税措施，使个人减税永久化。根据去年通过的税改政策，个人方面的税收变化，包括降低税率和增加儿童税收抵免等等，都将在2025年到期。
 <br/>
 <br/>
 美国政府也正在开始筹备总统的2020财年预算提案，将于明年初把预算提案提呈给国会。
 <br/>
 <br/>
 10月17日，川普在白宫举行的内阁会议上提出〝5美分计划〞，要求在场的内阁成员和部长削减至少5％的预算，并取消了减少工作岗位的不必要规则。
 <br/>
 <br/>
 川普说：〝我希望你们再来（开会）时，都削减5％预算。去除脂肪，清除浪费。〞
 <br/>
 <br/>
 〝这是我对坐在这张桌旁的每个人提出的一个非常非常重要的要求，这是数额巨大的资金，这是我们都能做的事。〞
 <br/>
 <br/>
 当天参加内阁会议的官员包括美国内政部、交通部、农业部、环境保护部的部长，以及预算管理办公室主任、小企业局局长、美国贸易谈判代表莱特希泽、白宫幕僚长凯利、首席经济顾问库德洛等。
 <br/>
 <br/>
 不过，他也补充说军事预算大部分将免于削减，明年国防预算可能会从目前的7160亿降至7000亿。川普强调国防开支最为重要。
 <br/>
 <br/>
 川普预测，美国经济有巨大的潜力。他在会上再次提到过去多年来不公平贸易给美国带来的巨额经济损失。他说，这一切不会再发生。美国修复了与加拿大、墨西哥、韩国的贸易协议，欧盟也重回谈判桌。美国与中共的贸易谈判仍在进行中，不过中共还没有为达成交易做好准备。
 <br/>
 <br/>
 川普说表示，〝我不是在指责中共，也不是指责欧洲，我没有指责任何人。我指责我们的政客，指责我们的领导者，他们允许这些事发生，这毫无理由。〞
 <br/>
 <br/>
 （记者李红报导/责任编辑：祝馨睿）
