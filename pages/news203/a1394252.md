### iPhone XS再曝新故障  扬声器无法正常工作
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_iPhone XS.html" target="_blank">
  iPhone XS
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_iPhone XS Max.html" target="_blank">
  iPhone XS Max
 </a>
 推出后，深得用户喜爱。但日前，这两款苹果新产品被发现除了存在严重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_充电.html" target="_blank">
  充电
 </a>
 问题外，还存在着另一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_缺陷.html" target="_blank">
  缺陷
 </a>
 。
 <br/>
 <br/>
 据《福布斯》杂志周四（4日）的一篇调查报告显示，大量的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_iPhone XS.html" target="_blank">
  iPhone XS
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_iPhone XS Max.html" target="_blank">
  iPhone XS Max
 </a>
 用户说，他们手机上的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扬声器.html" target="_blank">
  扬声器
 </a>
 无法正常工作。
 <br/>
 <br/>
 报导引用了国外著名论坛网站Reddit以及Twitter、Apple支持社区、粉丝网站iMore和其它多个网站上的用户报告，称他们的iPhone XS和iPhone XS Max的顶级扬声器，会在播放音乐时突然停止工作。
 <br/>
 <br/>
 有一个用户在推特上抱怨说：〝iPhone XS使用耳机扬声器和底部扬声器来创造立体声效果，使用iPhone XS Max几天后，耳机扬声器就停止工作了。重启手机似乎解决了这个问题，但是没过多久问题就会重演。〞
 <br/>
 <br/>
 在另一个受影响的用户发布的视频中，可以清楚地看到手机扬声器的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_故障.html" target="_blank">
  故障
 </a>
 问题。
 <br/>
 <br/>
 另一个受影响的用户表示，即使更新到最新的iOS 12.1测试版，也没有解决扬声器的问题。
 <br/>
 <br/>
 不过，iPhone XS和iPhone XS Max的电话功能并没有受到影响，显示该故障可能来自软件问题。
 <br/>
 <br/>
 截至目前，苹果公司还没有针对上述问题作出回应。
 <br/>
 <br/>
 <h4>
  iPhone XS存在根本性
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_缺陷.html" target="_blank">
   缺陷
  </a>
  ：
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_充电.html" target="_blank">
   充电
  </a>
  问题
 </h4>
 <br/>
 9月底，《福布斯》杂志曾刊文说，Apple的许多新iPhone都无法充电。在Apple支持社区，Reddit、MacRumors论坛，Twitter和YouTube上有很多用户投诉了该问题。
 <br/>
 <br/>
 文章作者凯利（Gordon Kelly）说，〝数百个〞帖子详细地说明了iPhone XS和iPhone XS Max在待机状态下插入充电器时无法充电的情形。此外，一些手机会转为死机状态，必须重新启动。
 <br/>
 <br/>
 在测评机构Unbox Therapy制作的一段视频中，赫尔森特格（Lewis Hilsenteger）也发现了这个问题。但目前还没有解决方案。
 <br/>
 <br/>
 Apple支持社区的一张帖文称，Apple的技术支持人员〝意识到〞了这个问题，并建议用户更换手机。这可能表明，是硬件上出了问题。
 <br/>
 <br/>
 现在看来，购买iPhone XR似乎是个明智的决定。它不仅以更便宜的价格提供iPhone XS和iPhone XS Max的核心功能，而且颜色更多变，电池寿命更长，并且是与最新版本的iOS 12.1系统同时推出。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：东野）
</div>
