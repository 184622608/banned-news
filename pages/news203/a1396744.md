### 连截可疑包裹 川普吁媒体停止敌意 不容暴力立足
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】美国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 等民主党政要，以及美国有线电视新闻网（CNN）周三（24日）相继收到可疑爆炸包裹，成为新闻头条。总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）当天呼吁媒体停止敌意，并表示任何政治暴力在美国都无法立足。
 <br/>
 <br/>
 大纪元报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周三在白宫签署有关打击鸦片类药物（Opioid Drugs）滥用的法案时，针对多起爆炸包裹事件首次公开发表评论说，〝美国人民的安全是我的最优先要务。〞
 <br/>
 <br/>
 他也呼吁民众要团结一致，并谴责所有形式的暴力行为，指这类暴行在美国〝没有立足之地〞。
 <br/>
 <br/>
 川普说，他已听取了联邦调查局（FBI）、司法部、国土安全部，以及特勤局的简报，联邦执法人员正在进行重大调查，行政部门会投入所有资源，尽速将歹徒绳之以法。
 <br/>
 <br/>
 川普说，〝这种令人震惊的行为，相对于我们美国人所珍惜且视为神圣的一切事物而言，都是令人憎恶的。〞他说，他的行政团队对所发生的事情感到非常不安和气愤，并发誓要追究到底，

 <br/>
 周三，在威斯康辛州莫西尼（Mosinee）造势集会上川普也提及此事，并再度呼吁媒体停止他们〝永无止尽的敌意〞和〝假攻击〞。他也表示，公众人物有责任缓和他们的言词。
 <br/>
 <br/>
 川普说，大家不应随便将政敌比作历史恶人，但可惜的是人们这么做了，而且还经常这么做。
 <br/>
 <br/>
 川普强调，〝我们不该在公共场所成群攻击他人或破坏公共财产，总有方法解决我们的分歧，那就是和平地投票。〞
 <br/>
 <br/>
 他说：〝在国家加强努力消弭大家的分歧并让人民团结时，媒体也有责任树立文明格调，停止无止尽的敌意和一昧消极、经常制造假攻击和假故事。他们必须停下来。〞
 <br/>
 <br/>
 川普发表上述评论是针对本周起，包含有爆炸装置的多个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_可疑包裹.html" target="_blank">
  可疑包裹
 </a>
 被寄送到民主党高级政客或幕僚家中，消息几乎成为周三所有美国媒体的头条新闻。
 <br/>
 <br/>
 周三上午，特勤局表示，已经拦截下了寄送给克林顿和奥巴马的包裹，这些包裹里面含有爆炸物。
 <br/>
 <br/>
 美国有线电视新闻网CNN的员工当天在时代华纳楼下的邮件收发室里，也发现了可疑的爆炸装置，时代华纳的火灾警报响起，指示所有员工撤离建筑物。
 <br/>
 <br/>
 CNN说，这个可疑邮包是要寄给前中央情报局局长布伦南（John Brennan，目前他担任NBC新闻的分析师）的。在纽约市政府的新闻发布会上，纽约警察局局长奥尼尔（James O’Neill）说，到现场处理的官员确认了那个邮包〝看起来像是一个爆炸装置〞，另外还有一个装有白色粉末的信封。
 <br/>
 <br/>
 一名执法的消息人士告诉福克斯新闻，位于马里兰州国会山（Capitol Hill）的安检设施，也截住了一个要寄给加州联邦众议员瓦特斯（Maxine Waters）的可疑包裹，警方已将其送到FBI。
 <br/>
 <br/>
 加州民主党联邦参议员哈里斯（Kamala Harris）的发言人称，有人在哈里斯位在圣地亚哥的办事处附近发现可疑包裹，随后办公大楼的人们被撤离。 这些可疑包裹并不是要寄给哈里斯或她的办公室。
 <br/>
 <br/>
 佛州民主党联邦众议员舒尔茨（WassermanSchultz）及前司法部部长霍尔德（Eric Holder）在佛州的办公室位处同一栋大楼。周三上午一个可疑包裹寄给霍尔德办公室，包裹上的寄件人地址是舒尔茨办公室，因此霍尔德办公室将这个包裹送到寄件人地址。目前这个包裹已交给警方处理。
 <br/>
 <br/>
 周一，民主党的重要金主、亿万富翁索罗斯（George Soros）也收到类似的爆炸装置。
 <br/>
 <br/>
 调查发现，这些可疑邮件都很容易被发现，且均〝有惊无险〞、无一爆炸。分析人士认为，在距中期选举还有两周之际发生多起此类事件，显然具有政治意义。
 <br/>
 <br/>
 （记者李红报导/责任编辑：文慧）
