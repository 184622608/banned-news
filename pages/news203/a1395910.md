### 川普退出万国邮政联盟 美各界一致叫好
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】白宫本周三宣布，美国政府从即日起正式启动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退出万国邮政联盟.html" target="_blank">
  退出万国邮政联盟
 </a>
 程序。白宫说，万国邮政联盟的现行费率制度让中国和其他国家受益，却让美国制造商遭受不公平对待。这个决定获得了美国各界罕见高度一致的认可或赞许。
 <br/>
 <br/>
 10月17日，美国国务院宣布正式起启动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退出万国邮政联盟.html" target="_blank">
  退出万国邮政联盟
 </a>
 程序。白宫称，退出万国邮联是为了停止不公平的邮政费率对美国制造商造成的伤害。
 <br/>
 <br/>
 对于华府的这个决定，美国国内各界罕见一致叫好。
 <br/>
 <br/>
 美国全国制造商协会公开评论称：过去，万国邮政联盟协议导致大量中国冒牌货及危险有毒产品涌入美国，如果美国政府能够与中方重新达成一个现代化及更公平的邮政安排，将令美国制造业及制造业工人大大受益。
 <br/>
 <br/>
 新泽西州一间出售家庭用品的小企业的东主在接受媒体采访时表示十分赞同美国政府的这个决定，他说：〝美国政府怎可能（继续去）补贴中国（企业），却让我没有生意可做呢？〞
 <br/>
 <br/>
 即使是过去对川普政府的贸易政策报有诸多不满的美国媒体，这次也形容退出万国邮政联盟是一个〝合乎逻辑的决定〞；曾表态反对美国加征关税的商家也承认说，通过退出万国邮政联盟解决〝终端费〞收费不公的问题后，可阻止中国的冒牌货充斥美国市场。
 <br/>
 <br/>
 有消息指，促使美国总统川普（特朗普）作出退出万国邮政联盟决定的人，是华府鹰派人物、白宫国家贸易委员会主任纳瓦罗。
 <br/>
 <br/>
 美国退出万国邮政联盟的消息传出后，中国跨境电子商阿里巴巴，以及中通快递和百世等物流股的股价当天应声下跌。截至17日收盘，阿里巴巴的股价跌0.98%、百世跌2.38%、中通则暴跌4.2%，创下3周以来最大跌幅。
 <br/>
 <br/>
 据了解，在现行国际邮政体系下，从发展中国家寄送小包裹到美国的运费，比从美国寄送到这些国家更便宜，而近几年邮递到美国的国际包裹，大多数来自中国。
 <br/>
 <br/>
 据《美国之音》报导，美国政府一位高级官员曾在某次白宫电话简报会上说，加入万国邮政联盟对美国邮政〝造成了巨大的成本〞，而美国邮政还要向美国企业收取更高的运费，从而对就业也产生了影响。
 <br/>
 <br/>
 该官员并举例说：在现行的邮政制度下，一个一磅重的包裹，在美国国内邮寄，运费大概在10-13美元，但是从中国邮寄到美国的话，最后在美国境内的投递可以可获得2.5美元的补偿。如果是4.4磅重的包裹，美国的寄件方需支付20-25美元，而中国的寄件方只需支付5美元。而造成上述明显不公平现象的根源，是万国邮联多年前制定的终端费（terminal dues）体制。
 <br/>
 <br/>
 所谓〝终端费〞是指一国为国际邮件到达另一国家后支付一定的投递费用，但这个费用并不反映对方国家投递邮件的实际成本。按照万国邮联的复杂规定，各成员国按经济发达水平分成两档，发达国家是〝目标国〞，发展中国家是〝过渡国〞，万国邮联允许〝过渡国〞获得补偿，因此支付的终端费会远低于实际投递成本，而中国就在〝过渡国〞之列，美国则属于〝目标国〞。
 <br/>
 <br/>
 据美国全国制造商协会统计，由于上述不公平的规则，美国邮政仅去年财年就损失大约1.7亿美元。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
