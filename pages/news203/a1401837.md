### 贸易战只是未升级 中共本质障碍真停火
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞落幕，接下来约90天谈判，中共必须做出美国要求的结构性改变。有专家认为，中共恐怕很难做到。
 <br/>
 <br/>
 12月1日晚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 后，中共官媒抢先发布消息，称〝达成重要共识〞，但只强调美国在1月1日后不会额外加征关税，却闭口不提其〝前提条件〞和重要细节。白宫发表的声明则列出部份协议内容。双方还同意就〝结构性改革〞问题展开为期90天的谈判。
 <br/>
 <br/>
 美中商会高级副总裁Erin Ennis：〝90天不加征关税，至少是我们期待的结果之一，但我们要等看到具体谈判内容，及双方是否能够对谈判结果达成一致。我们还在等着看。〞
 <br/>
 <br/>
 专家认为，虽然谈判避免了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 进一步升级，但接下来北京将面对更大挑战，即结构性问题谈判。
 <br/>
 <br/>
 前副总统拜登副国家安全顾问Jeffrey Prescott：〝在美国真正关切的问题上，如国有企业、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强迫技术转让.html" target="_blank">
  强迫技术转让
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 ，及中国市场的其它限制，会谈并没有具体结果。也不清楚周末情况是否有变，能让未来90天的谈判更容易。〞
 <br/>
 <br/>
 结构性问题实际在美中之间已持续多年，但这次川普（特朗普）政府只给中共90天谈判期。北京在这些问题上能有多大让步远，专家并不乐观。
 <br/>
 <br/>
 战略与国际研究中心经贸专家William Reinsch：〝他们（川普政府）希望的结构性改变对于中共来说是不可能做到的，因为美国所要的改革，其结果将威胁到中共对于国家的控制。而这点是中共不会放弃的。〞
 <br/>
 <br/>
 前副总统拜登副国家安全顾问Jeffrey Prescott：〝即使实现这些，你也不会在其实现前看到结果。我们从长期经验中看到，这些问题上，将有一个非常困难的逐步实现的过程。〞
 <br/>
 <br/>
 财长姆努钦周一说，他对未来三个月美中谈判达成协议表示乐观，但同时警告，这必须是一个真正协议，不能是缩水的软承诺，需要有具体日期以及行动项目。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
