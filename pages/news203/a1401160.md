### 密西西比参议员决选再投票 共和党保住席位
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】周二，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_密西西比.html" target="_blank">
  密西西比
 </a>
 州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联邦参议员.html" target="_blank">
  联邦参议员
 </a>
 决选，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 候选人辛迪・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海德史密斯.html" target="_blank">
  海德史密斯
 </a>
 击败民主党候选人迈克・埃斯皮，共和党在参议院对民主党的优势，扩大到53比47。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海德史密斯.html" target="_blank">
  海德史密斯
 </a>
 是前密州农业厅厅长，今年3月被州长任命填补
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 参议员席位。
 <br/>
 <br/>
 民主党籍竞选对手迈克・埃斯皮，曾任联邦众议员多年，还曾担任克林顿政府的农业部长。
 <br/>
 <br/>
 中期选举中，两人的选票都未超过半数，得票相差仅一个百分点，因而再次投票。
 <br/>
 <br/>
 海德史密斯胜选后表示，是密州人遵循的传统价值使她获选。
 <br/>
 <br/>
 共和党候选人辛迪・海德史密斯：〝这次胜利在于我们的传统价值，在于我们所有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_密西西比.html" target="_blank">
  密西西比
 </a>
 人最看重的：信仰和家庭。这也是我要带到华府的，我要代表所有密西西比人体现这些价值。〞
 <br/>
 <br/>
 海德史密斯向媒体透露，她将关注监狱改革和经济两大问题。
 <br/>
 <br/>
 周一，川普总统参加了两场支持海德史密斯的助选集会，并在周二晚间，发推特祝贺她胜选。
 <br/>
 <br/>
 川普推文：〝祝贺辛迪・海德史密斯在密西西比州的胜利。我们都为你感到骄傲！〞
 <br/>
 <br/>
 <br/>
 新唐人记者毕心慈综合报导
