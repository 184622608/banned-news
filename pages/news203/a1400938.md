### 北京官场局势微妙 有人暗挺川普
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】〝川习会〞11底在G20峰会期间举行，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 将迎来关键节点。有海外党媒暗示，北京政坛局势微妙，有人支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普），掣肘习近平。而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国民营企业.html" target="_blank">
  中国民营企业
 </a>
 也纷纷倒戈，支持川普对中共进行更严厉的贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 。
 <br/>
 <br/>
 11月30日至12月1日，二十国集团峰会（G20）将在阿根廷首都布宜诺斯艾利斯召开，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和习近平将在峰会期间会晤。中方近期在贸易战方面态度已软化，从最初〝以牙还牙〞转为〝推动达成双方都能接受方案〞。
 <br/>
 <br/>
 有港媒26日称，〝川习会〞前，中共已经做出了可以做的让步，以〝不抵抗〞为底线换取〝美中双赢〞。
 <br/>
 <br/>
 中共海外党媒多维新闻，则接连刊登两篇评论文章暗示，在〝川习会〞召开前，北京政坛有人在支持川普，有人在掣肘习近平。
 <br/>
 <br/>
 <h4>
  谁在掣肘习近平？
 </h4>
 <br/>
 多维发表的题为〝中美周旋，谁在掣肘习近平 〞一文写道：来自中共官方的声音再次令人感到迷惑，这些声音抛开浓厚的民族主义执政色彩，与美国政府内部的实用派类似，多指向中美关系需要向好的方向转变，中美存在的问题将获得渐进式的解决。
 <br/>
 <br/>
 中共〝让步论〞也在一部分中共官员中游走，从官方到民间希望结束中美贸易的动荡感受似乎更为迫切，这一点不仅出现在外交领域，在经济领域更为突出。
 <br/>
 <br/>
 有中共发改委官员称，中共在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革.html" target="_blank">
  改革
 </a>
 领域需要美国的先进经验，中美合作是发展的大势，两国间的贸易冲突升级不能带来任何好处，中国各地方政府对贸易领域的纠纷降温也表现出强烈期待。
 <br/>
 <br/>
 文章说，部分中共官员认为，来自美国的贸易施压，对推动中国深化改革有一定的积极影响。
 <br/>
 <br/>
 <h4>
  谁在支持川普？
 </h4>
 <br/>
 多维发表的另一篇题为〝北京观察：谁在支持川普〞的时评则写道：〝在中国，不仅对官方在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 的策略选择的批评之声公开出现，甚至有中共官员公开表示自己欣赏川普，还有媒体称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国民营企业.html" target="_blank">
  中国民营企业
 </a>
 暗中支持川普。〞
 <br/>
 <br/>
 英媒《路透》21日撰文称，大陆民间企业界有一部分人，暗暗支持川普向北京当局施压并促其改革。这些大陆企业家期盼大陆市场更自由开放，但国内无人能向最高层施压加速这方面改革。
 <br/>
 <br/>
 因此，在他们眼里，川普几乎成为北京高层唯一的改革压力来源。中国改革派人士也都期待，川普能成为刺激中国经济改革的外部驱动力。
 <br/>
 <br/>
 11月18日，在中国第九届财新峰会上，龙永图批评在中美贸易战中，中共高层的策略失误。龙永图曾经是中共加入世贸谈判的代表，其意见观点具有影响力。
 <br/>
 <br/>
 在同一论坛上，中国发展研究基金副理事长、央行货币政策委委员刘世锦，也罕见批评中共，不能把计划经济遗留物当成体制优势加以固守，要建设高标准市场经济，实行高水平对外开放。
 <br/>
 <br/>
 截止目前，持续了近一年的中美贸易战火仍在持续升温，中国经济持续下滑，美国经济呈现强劲态势。川普政府此前已对2,500亿中国商品课征10%到25%的惩罚性关税。未来征税的规模还可能扩大一倍。
 <br/>
 <br/>
 美国副总统彭斯警告中共，必须全面改变其行为作法，才能避免与美国发生新冷战。他说，美国在过去25年帮助〝重建〞了中国，但中共没有像美国希望的那样，让自己的人民迈向更大的自由，而是朝着控制和压迫中国民众的方向急转弯。
 <br/>
 <br/>
 现在美国真的觉醒了，因此，川普总统在经济、政治、外交、军事等领域展现出的全新姿态，绝非中共诋毁的〝单边保护主义〞，而是反击邪恶共产主义的进取之姿。
 <br/>
 <br/>
 在这场正邪对抗中，国际社会也都已看清，川普政府对中共的这轮贸易战，不仅是在为美国的利益而战，也是在为中国人民的利益而战。因此中共官场及民企有人暗中支持川普也不足为怪了。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：范铭）
