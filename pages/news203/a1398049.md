### 威斯康辛州汽车撞人4死1伤 嫌犯自首
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】周六，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威斯康辛州.html" target="_blank">
  威斯康辛州
 </a>
 哈利湖附近，发生一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交通事故.html" target="_blank">
  交通事故
 </a>
 ，一名男子驾车撞向路边行人后逃逸，车祸造成4名学生死亡， 另有1人受重伤。
 <br/>
 <br/>
 当地哈利湖警局官员表示，肇事者驾驶的是一辆黑色福特皮卡。车祸发生在一个小山坡前。
 <br/>
 <br/>
 哈利湖警局警长 Daniel Sokup：〝我们希望大家在社交媒体上转发这一则信息，如果你知道谁有这辆车，你的邻居或者是谁，去看一眼，确认一下，如果发现与肇事车辆描述相符，请立即通知我们。〞
 <br/>
 <br/>
 最新消息显示，嫌犯已经自首，名叫科尔顿・特鲁（Colton Treu）, 21岁，是当地齐佩瓦市人。警方称，将以4项谋杀罪名起诉他.
 <br/>
 <br/>
 一位目击者表示，4名死者均为齐佩瓦市4年级学生，事发当时他们正在清理路边的垃圾。
 <br/>
 <br/>
 新唐人记者陈杰综合报导
</div>
