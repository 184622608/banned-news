### 通俄案调查备忘录即将公布 穆勒团队遭起诉
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】耗时已久的〝通俄案〞调查进入收尾阶段。日前有知情者披露，卷入该案的佛林、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马纳福.html" target="_blank">
  马纳福
 </a>
 、柯恩这三名关键人士的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_调查备忘录.html" target="_blank">
  调查备忘录
 </a>
 将于近日公布。与此同时，保守派作家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科西.html" target="_blank">
  科西
 </a>
 起诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 团队，指控他们用恐吓方式逼他做伪证。
 <br/>
 <br/>
 日前，美国媒体世界日报援引消息人士透露，特别检察官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 (Robert Mueller)手下的联邦检察官，已经告知〝通俄案〞被告的辩护律师，全案调查已经进入〝收尾阶段〞。这场拖延已久的调查，估计将会在未来几周内宣告终结。
 <br/>
 <br/>
 这个案件中最受瞩目的三大关键人物是前任美国国家安全顾问佛林(Michael Flynn)、前任川普竞选总干事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马纳福.html" target="_blank">
  马纳福
 </a>
 (Paul Manafort )和前任川普私人律师柯恩(Michael Cohen)。
 <br/>
 <br/>
 消息人士表示，穆勒调查团队本周将提交对上述三名关键人士进行调查的备忘录。其中，佛林的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_调查备忘录.html" target="_blank">
  调查备忘录
 </a>
 将在本月4日晚间公布，而马纳福与柯恩的调查备忘录将于本月7日公布。
 <br/>
 <br/>
 此前，马纳福同意配合穆勒团队调查后，曾有消息指穆勒团队可能会以保密方式向法院提交有关马纳福的调查备忘录，以避免让马纳福卷入的其他罪行曝光。但不久前穆勒团队指称马纳福对调查人员〝说谎〞，宣布他与检方达成的认罪协议破裂。随后穆勒的发言人卡尔(Peter Carr)对外表示，马纳福的调查备忘录〝将会公开〞，不过某些内容可能会经过删减，或者以保密附件的形式另外提交。
 <br/>
 <br/>
 针对上述情况，有分析人士指出：马纳福备忘录被公开〝意味着调查行动已经到了不再需要保留案情内容以便起诉更多被告的阶段〞。
 <br/>
 <br/>
 而就在上述消息传出的前一天（12月3日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查的证人之一、保守派作家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科西.html" target="_blank">
  科西
 </a>
 （Jerome Corsi）宣布对穆勒调查团队提出刑事和道德诉讼。他并宣布，长达78页的文件已递交给包括司法部代理部长惠特克（Matthew Whitaker）和司法部监察长霍洛维茨（Michael Horowitz）在内的多位执法部门的高官。
 <br/>
 <br/>
 科西对前往采访他的媒体说：〝我很配合、给了他们所有他们要的，包括电脑、时光机备份、自愿交出了电邮、推特帐号，所有他们想要的我都给了。但第一天进行了记忆测试，我没做好，突然之间，我余生就要在监狱中度过了。这是他们恐吓人的招数，我相信他们是有意而为。〞
 <br/>
 <br/>
 有关俄罗斯干预2016年美国总统大选的调查从，2017年5月开始，迄今已经持续进行了18个月，却始终没有发现任何川普在总统大选过程中通俄的证据。川普一再批评这是一场政治迫害，并呼吁尽早结束这场导致美国政治分裂并耗资巨大的〝猎巫行动〞。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
