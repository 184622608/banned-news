### 躲过赌城大屠杀 27岁男命丧加州枪击案
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 一间
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒吧.html" target="_blank">
  酒吧
 </a>
 周三（7日）深夜发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，导致13人遇难，其中包括一名27岁男子者奥尔凡诺。据悉，去年他曾经历
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉斯维加斯.html" target="_blank">
  拉斯维加斯
 </a>
 赌城大屠杀，当时他侥幸生还，没想到时隔一年后再遇死神。
 <br/>
 <br/>
 美国南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 千橡镇音乐吧7日深夜发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，造成至少13人死21人受伤。27岁的奥尔凡诺（Telemachus Orfanos）就是其中一位。
 <br/>
 <br/>
 奥尔凡诺在当地高中毕业后，在美国海军服役。退役后他在加州两间
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒吧.html" target="_blank">
  酒吧
 </a>
 担任服务员及产品推销员，包括案发的Borderline Bar &amp; Grill酒吧。一年前，他与朋友前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉斯维加斯.html" target="_blank">
  拉斯维加斯
 </a>
 参加音乐节活动时，也遇上枪手扫射群众，事件共造成58死、500多人伤，震惊全美。他当时幸运的逃过一劫。
 <br/>
 <br/>
 没想到一年过后，命丧加州酒吧。他的母亲得悉哀讯后，伤痛不已说，〝他避过了拉斯维加斯枪击案，回到家，但昨晚他没有再回来。〞她也表示，希望控枪。
 <br/>
 <br/>
 奥尔凡诺的朋友亦也在社交网哀悼好友的死：〝不敢相信你已经离开了，我会记得你总在我伤心时引我发笑。〞
