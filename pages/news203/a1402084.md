### 川普评巴黎暴乱：美国人不为它国污染付钱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】周二（12月4日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统发推祝贺法国解决了因燃油税涨价而引发的暴乱，并再次批评《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎协定.html" target="_blank">
  巴黎协定
 </a>
 》（Paris Agreement），声明他不会像法国政府那样，让美国民众为它国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_污染.html" target="_blank">
  污染
 </a>
 付钱。
 <br/>
 <br/>
 周二下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 祝贺法国政府顺应民意，解决了因燃油税涨价而引发的国内危机。
 <br/>
 <br/>
 〝我很高兴，我的朋友马克龙和巴黎的抗议者已同意我两年前得出的结论。 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎协定.html" target="_blank">
  巴黎协定
 </a>
 》存在致命缺陷，因为它提高了责任国家的能源价格，同时掩饰一些世界上最严重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_污染.html" target="_blank">
  污染
 </a>
 源。〞
 <br/>
 <br/>
 〝我想要清洁的空气、干净的水，并一直在改善美国环境方面阔步前进。但美国纳税人 ——以及美国工人 ——不应该为清理其它国家的污染而付出代价。〞
 <br/>
 <br/>
 法国民众已连续数周抗议燃油税上涨，且抗议规模不断升级。在刚刚过去的这个周末，巴黎的抗议已演变为暴乱，致使象征法国辉煌历史的凯旋门遭到破坏，抗议者还直接要求法国总统马克龙（Emmanuel Macron）下台。
 <br/>
 <br/>
 面对著名的〝光明之城〞（City of Lights）变为〝火力之城〞（City of Fire），周二，法国政府终于松口，同意6个月内暂停增加燃油税，并讨论进一步解决措施。
 <br/>
 <br/>
 法国〝黄背心〞抗议活动于11月17日爆发，之后愈演愈烈。上周六（12月1日），一些抗议者在巴黎市中心焚烧了数十辆汽车和街边建筑物，同时掠夺商店、砸坏窗户，甚至一些珍贵的文物也未能幸免。冲突中，有400多人遭到警方逮捕，100多人受伤，马克龙政府则面临严重的执政威胁。
 <br/>
 <br/>
 事件起因于马克龙的环境政策。他为履行联合国关于气候变化的《巴黎协定》，今年已将法国每公升的柴油税调涨了6.2％，并声称未来几年中还将持续上涨。
 <br/>
 <br/>
 马克龙认为，调高燃油税将有助于减少法国对化石燃料的依赖，增加对可再生能源的投资。但抗议者认为，马克龙以环保为名强推改革，忽略了法国人民、尤其是中下阶层民众的真正需求。
 <br/>
 <br/>
 与马克龙截然相反，川普无视联合国及专家的所谓评估和施压，指出即使在科学界对于气候变暖原因也有异议。他坚持兑现自己的竞选承诺，于2017年6月1日宣布退出《巴黎协定》。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
