### 防移民再次疯狂破境 美现役军人边境演习
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】为了防止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 失控并大规模入境，周三（12月5日），一些美国现役部队已抵达亚利桑那州边境地区，会同边境执法人员共同训练，做好应对移民风暴的战术准备。
 <br/>
 <br/>
 据《华盛顿观察家报》（The Washington Examiner）12月5日报导，鉴于美国海关和边境保护局（U.S. Customs and Border Protection, CBP）执法人员在应对可能再次发生的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 风暴时会遭遇困境，周三，一些现役美国部队已抵达亚利桑那州南部的两个入境口岸，准备从事防御训练。
 <br/>
 <br/>
 得到增援的是诺加利斯（Nogales）和德孔西尼（DeConcini）两个入境口岸。抵达那里的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_现役军人.html" target="_blank">
  现役军人
 </a>
 将与海关执法人员一起举行大规模的防御演习，以有效应对可能到来的危机。
 <br/>
 <br/>
 为防止数百甚至数千在边境口岸申请庇护的中美洲移民强行入境——就像11月25日在蒂华纳（Tijuana）和圣地亚哥（San Diego）之间所发生的情况那样，美国现役军人和边防人员采取了所有的必要步骤，以确保入境口岸安全。
 <br/>
 <br/>
 CBP在声明中概括了边防人员训练和演习的主要内容。声明指出，这些训练是为了应对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民可能大规模涌向边界以及攻击美国边防人员而进行的，以便此类情况发生时，边防人员能迅速做出战术上的反应。
 <br/>
 <br/>
 目前滞留在墨西哥蒂华纳市的中美洲移民有6,000人左右。10天前，约有500至1000名移民试图使用武力制服边境巡逻人员。
 <br/>
 <br/>
 事件过后，圣地亚哥的首席巡逻官员斯科特（Rodney S. Scott）表示，美国执法人员被迫使用催泪瓦斯对付攻击者，因为这些移民在试图突破边界时，采用投掷石块和瓦砾、嘲笑等方式攻击执法人员。
 <br/>
 <br/>
 （记者李佳欣编译报导/责任编辑:东野）
