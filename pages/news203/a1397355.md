### 视频：2018年神韵艺术团新秀技巧表演
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神韵.html" target="_blank">
  神韵
 </a>
 艺术团年轻的领舞演员们近日来展示了高超的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国古典舞.html" target="_blank">
  中国古典舞
 </a>
 技术、技巧。中国古典舞是传承了五千年的中国文化中的一种表演艺术形式。自2006年起，总部设立于纽约的神韵艺术团，每年在五大洲巡回演出，令中国古典舞这种艺术形式为世人所熟知。
 <br/>
 <br/>
 身韵、身法、技巧加上表演是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国古典舞.html" target="_blank">
  中国古典舞
 </a>
 的特点和基础。视频中展示的只是中国古典舞的技巧组合，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神韵.html" target="_blank">
  神韵
 </a>
 演出也不是一回事。神韵演出还包含现场乐队伴奏、服装、天幕、舞蹈和舞剧，每年在全世界的180多个城市巡回演出。
 <br/>
 <br/>
 根据神韵官网信息显示，2019年新一季的演出，即将于今年12月12日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 拉开序幕：
