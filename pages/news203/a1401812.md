### 习近平开绿灯高通不领情：已弃购恩智浦
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】跨国半导体和电信设备企业美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高通.html" target="_blank">
  高通
 </a>
 公司发布公告，就中国国家主席习近平在G20峰会上对该公司的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恩智浦并购案.html" target="_blank">
  恩智浦并购案
 </a>
 作出的积极表态表示感谢，但同时重申，该公司的这项并购案已终止，也无意重启这个并购案。
 <br/>
 <br/>
 在阿根廷首都布宜诺斯艾利斯刚刚结束的美中首脑会谈上，习近平曾表态称，如果先前未获准的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高通.html" target="_blank">
  高通
 </a>
 收购恩智浦案再次提交，他对批准这个收购案〝持开放态度〞。
 <br/>
 <br/>
 这个消息传出后，外界开始关注这宗号称〝半导体史上最大收购案〞会否因此而死灰复燃。
 <br/>
 <br/>
 对此，高通公司发布一份新闻稿称，该公司并购恩智浦的交易已结案，现在公司正全力推动5G研发进度。
 <br/>
 <br/>
 新闻稿写道，〝本公司得知总统川普（特朗普）和中国国家主席习近平对高通早先提出的并购恩智浦之议发表看法，深表感谢。但这个交易已经过期，原先拟定的交易也已终止。〞
 <br/>
 <br/>
 稍早前，高通曾开出380亿美元的高价来收购恩智浦，但被中共有关监管机构以〝反垄断〞为由一直拖延审查进度，不予批覆。
 <br/>
 <br/>
 今年7月26日，高通宣布终止收购恩智浦计划，为此该公司向恩智浦支付了20亿美元的〝分手费〞。同时，高通公司的董事会批准了300亿美元的股票回购计划。
 <br/>
 <br/>
 就在高通公司宣布终止这项收购计划的第二天（27日），中共市场监管机构突然对外放话称，高通的并购案仍有通融余地，希望有关各方能找到处理垄断问题的解决方案，并希望继续与高通公司就此进行沟通。不过，高通公司对此并未作出响应。
 <br/>
 <br/>
 分析人士认为，高通的这项并购案原本对于双方来说是一次双赢的机会，但在中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 日益升级的背景下，这个并购案成为了美中两国政府角力的一枚棋子。现在虽然美中双方达成了临时停火协议，但美方明确提出了3个月的暂停期限。在这90天内，要指望中方真的在〝结构性改革〞方面作出实质性的改变，可能性并不高。高通公司对于接下来的中美贸易谈判或缺乏信心。为了避免并购案再次成为中美贸易较量的棋子而无谓折腾，高通公司作出了果断放弃并购计划并另谋发展的决定。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
