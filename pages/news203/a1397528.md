### 川普访匹兹堡受热情接待 批媒体耍政治手腕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统夫妇周二（10月30日）赴宾州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_匹兹堡.html" target="_blank">
  匹兹堡
 </a>
 吊唁犹太
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_教堂枪击案.html" target="_blank">
  教堂枪击案
 </a>
 受害者，受到当地民众热情接待。周三，川普批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_媒体.html" target="_blank">
  媒体
 </a>
 挑拨事端，利用不实报导玩弄政治手段，以达到政治攻击的目的。
 <br/>
 <br/>
 周三清晨，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 发推揭露自由派
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_媒体.html" target="_blank">
  媒体
 </a>
 再次进行不实报导，故意错误描述民众对教堂枪击事件的立场。
 <br/>
 <br/>
 〝昨天在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_匹兹堡.html" target="_blank">
  匹兹堡
 </a>
 ，我和梅拉尼娅受到很好的接待。在一个非常悲伤、庄严的日子里，总统办公室受到了极大的尊重。我们得到了如此热情的接待。我们没有看到小规模的抗议，抗议的地点离我们很远。假新闻的报导恰恰相反 ——可耻！〞
 <br/>
 <br/>
 周二，川普携第一夫人梅拉尼娅（Melania Trump）赴匹兹堡，悼念犹太
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_教堂枪击案.html" target="_blank">
  教堂枪击案
 </a>
 的死难者，并慰问家属、伤员、医护人员和执法官员。他们受到当地人员的热情接待。
 <br/>
 <br/>
 川普的长女伊万卡（Ivanka Trump）和女婿库什纳（Jared Kushner）、财政部长姆努钦（Steven Mnuchin）也一同前往。他们都是犹太人或犹太教信徒。
 <br/>
 <br/>
 在枪击案中遇难的戈特弗里德（Richard Gottfried）博士的遗孀与川普会晤了长达一小时的时间。白宫新闻发言人桑德斯（Sarah Sanders）周三表示，这名受害家属告诉川普，她想见到川普并让他知道，匹兹堡民众
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欢迎.html" target="_blank">
  欢迎
 </a>
 他。
 <br/>
 <br/>
 虽然在川普访问期间有些人在教堂附近抗议，但主持悼念活动的犹太拉比和大多数当地民众都欢迎川普的到来。一名医务人员对到访医院的川普夫妇表示感谢，并向第一千金致意——〝我爱你，伊万卡！〞
 <br/>
 <br/>
 匹兹堡市民主党籍市长佩杜托（William Peduto）藉口〝唯一关注的是葬礼〞，拒绝会晤川普总统。
 <br/>
 <br/>
 枪击案发生后，民主党和左派媒体立即把责任转嫁给川普，攻击他宣扬白人民族主义、从而激励了极端分子攻击民主党和少数族裔。
 <br/>
 <br/>
 川普周一在推文中指出，正是媒体不正确甚至骗人的报导激发了国内的愤怒，假新闻媒体才是民众真正的敌人。他呼吁媒体停止敌意和不公正的报导，这样才能迎来各方的和平与和谐。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
</div>
