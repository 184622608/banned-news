### 黑利年底离任联合国大使 2020不参选总统
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统宣布，他批准美国驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国大使.html" target="_blank">
  联合国大使
 </a>
 妮基・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑利.html" target="_blank">
  黑利
 </a>
 （Nikki Haley）的辞职申请。黑利将在年底离任。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国大使.html" target="_blank">
  联合国大使
 </a>
 妮基．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑利.html" target="_blank">
  黑利
 </a>
 ，在我看来与众不同。她的工作相当出色，她是个了不起的人。她想
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离职.html" target="_blank">
  离职
 </a>
 一段时间休息一下。〞
 <br/>
 <br/>
 46岁的黑利于2016年11月被任命为美国驻联合国大使，经手俄罗斯、伊朗和朝鲜等棘手的国际事务。而此前，她还担任过六年南卡州长。
 <br/>
 <br/>
 联合国大使妮基．黑利：〝这是（连着）八年的紧张工作，我是个坚信凡事要有限期的人。我认为一个人必须非常无私，才能知道什么时候你该站到一边，让别人来接手工作。〞
 <br/>
 <br/>
 对于黑利离职后的去向，此前有传闻说她可能准备参加2020年的总统大选。
 <br/>
 <br/>
 联合国大使妮基．黑利：〝不，我不会参加2020年的总统大选。但我可以承诺我会支持他（川普），我将在下次大选中支持川普总统连任。〞
 <br/>
 <br/>
 关于接替黑利的人选，川普随后在面对媒体时提到了大女儿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊万卡.html" target="_blank">
  伊万卡
 </a>
 ，他说：我如果提名伊万卡做联合国大使，可能被指责为向着自己家人，但有谁比她更胜任呢？
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
