### 法官临时裁决恢复CNN记者通行证 白宫回应
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】周五（11月16日），美国一名联邦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 裁决，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 应立即恢复美国有线电视新闻网（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN.html" target="_blank">
  CNN
 </a>
 ）记者阿科斯塔（Jim Acosta）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_记者通行证.html" target="_blank">
  记者通行证
 </a>
 。白宫表示将执行法官的裁定，但将着手改变规则，让新闻发布会今后能够公平有序地举行。
 <br/>
 <br/>
 周五，一名联邦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 裁决，允许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN.html" target="_blank">
  CNN
 </a>
 明星记者阿科斯塔临时恢复获得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_记者通行证.html" target="_blank">
  记者通行证
 </a>
 。
 <br/>
 <br/>
 做出这项裁决的是华盛顿地区法院的联邦法官凯利（Timothy J. Kelly）。他把原定于周四下午举行的听证会推迟到周五上午，并批准了CNN申请的临时限制令，命令白宫恢复阿科斯塔的通行证。
 <br/>
 <br/>
 周五，白宫对法官的裁决做出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_回应.html" target="_blank">
  回应
 </a>
 。白宫新闻发言人桑德斯（Sarah Sanders）声明指出，法院在裁决中明确表示，并不存在可进入白宫的绝对《第一修正案》权，白宫会依照裁决临时恢复阿科斯塔的〝硬通行证〞（hard pass），但将进一步制定规则和程序，以确保未来的新闻发布会公平有序。白宫必须规定礼仪规范。
 <br/>
 <br/>
 凯利做出的只是一项临时性裁决，未来几周还将举行进一步听证，以确定取消阿科斯塔的通行证是否侵犯了宪法第一和第五修正案所赋予他的权利。凯利裁决令的有效期只有14天。
 <br/>
 <br/>
 凯利的裁决基于宪法《第五修正案》，他声称，白宫吊销阿科斯塔的新闻通行证前，没有给予阿科斯塔履行正当程序的机会。
 <br/>
 <br/>
 11月7日，在白宫举行的新闻会上，阿科斯塔以尖刻的提问持续纠缠川普总统，一名白宫女实习生上前试图拿下他手中的麦克风以阻止他继续发问，但他躲闪不给，并以手臂挡开年轻的女工作人员。川普当场斥责阿科斯塔粗鲁，白宫当晚宣布吊销阿科斯塔的记者通行证、禁止他进入白宫。
 <br/>
 <br/>
 （记者李佳欣综合报导）
