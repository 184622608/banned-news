### 美中竞争加剧  G20峰会不会改变
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】除
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易.html" target="_blank">
  美中贸易
 </a>
 争端，目前中共在所谓〝灰色地带〞对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 构成严峻挑战。专家预计，美中竞争加剧的总体趋势，不会因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 有所改变。
 <br/>
 <br/>
 伴随美中之间竞争加剧，介于和平与战争之间的所谓〝灰色地带〞也更受关注。这其中包含如网络攻击、信息战、经济胁迫与政治胁迫等。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国家安全局局长 Michael Rogers：〝他们认为可以采取这些行为，而不触及美国的战略门槛，付出极小代价。他们认为这是风险较小的方式。〞
 <br/>
 <br/>
 专家认为，灰色地带竞争，手段与过去冷战相比有很多不同，但目标都是要削弱美国及盟友，而公众防范意识却相对较低。长期以来，中共就是在各方面寻找美国弱点，进行攻击。
 <br/>
 <br/>
 美国国家民主研究所总裁 米德伟：〝他们寻找一些方式，可以打击美国和其盟友，打击他们的开放社会。他们以鼓动全社会的方式来做。〞
 <br/>
 <br/>
 副总统彭斯日前接受媒体专访时表示，中共要想避免和美国展开〝全面冷战〞，必须先改变其行为模式。其中包括改变猖獗的知识产权盗窃，尊重国际规则和规范，停止限制国际水域的航行自由，及放弃干预西方国家政治等。这些也正是所谓〝灰色地带〞。
 <br/>
 <br/>
 本周末G20期间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 即将登场。不过专家认为，美中竞争态势并不会因双方会晤有所改变。
 <br/>
 <br/>
 澳洲麦考瑞大学教授 亚太战略专家 季北慈：〝总体来看，我不认为会有根本上或战略性改变，未来美中关系会更具挑战性或更困难。〞
 <br/>
 <br/>
 专家指出，美国从未想要遏制中国，但中共恶意行为已对美国构成挑衅。美中全方位竞争的局面已经形成。
 <br/>
 <br/>
 美国国家民主研究所总裁 米德伟：〝美国会做出反应，只要中共继续损害美国的民主，和在全球的民主利益，及人权、开放、透明及自由市场。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
