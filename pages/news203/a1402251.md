### 首名大篷车〝锚孩〞诞生 落地公民权再惹争议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】上周二（11月27日），一名洪都拉斯孕妇非法攀爬边境围栏进入美国后，不足24小时便诞生了一名男婴。这名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民诞生的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_锚孩.html" target="_blank">
  锚孩
 </a>
 〞再度引发了关于美国出生地原则（也称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落地公民权.html" target="_blank">
  落地公民权
 </a>
 ）的争论。
 <br/>
 <br/>
 11月26日，19岁的洪都拉斯孕妇塞拉诺-埃尔南德斯（Maryury Elizabeth Serrano-Hernandez）在美国圣地亚哥（San Diego）生下一名婴儿。她离开本国时已经怀孕7个多月，其20岁的丈夫奥尔蒂斯（Miguel Ortiz）和3岁的长子也都在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民队伍中。
 <br/>
 <br/>
 塞拉诺-埃尔南德斯一家上月下旬抵达墨西哥边境城市蒂华纳（Tijuana）后，不想在墨西哥等待庇护审批，而是翻越围栏潜入美国。
 <br/>
 <br/>
 他们于26日非法入境，当晚被美国边境人员发现后，办理了庇护申请手续并被拘留。27日，他们的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_锚孩.html" target="_blank">
  锚孩
 </a>
 〞（anchor baby）便在圣地亚哥的一家医院降生了。
 <br/>
 <br/>
 据美国Univision电视网报导，塞拉诺-埃尔南德斯称，孩子的出生对她家的旅途跋涉是个〝巨大回报〞。
 <br/>
 <br/>
 3名美国巡逻人员发现塞拉诺-埃尔南德斯一家翻过边境围栏后，曾要求他们返回蒂华纳。但他们拒绝返回，并申请庇护。
 <br/>
 <br/>
 他们现在由圣地亚哥的一个家庭予以安置，但他们希望搬到俄亥俄州哥伦布市，因为那里有他们的家人，他们希望在那里等待他们的庇护审批程序。
 <br/>
 <br/>
 塞拉诺-埃尔南德斯的新生婴儿据信是近期在美国诞生的第一个大篷车〝锚孩〞（anchor baby），并再度引发了关于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落地公民权.html" target="_blank">
  落地公民权
 </a>
 问题的关注。
 <br/>
 <br/>
 美国公民出生地原则（birthright citizenship，也称〝出生地国籍〞、〝出生公民权〞或〝落地公民权〞），按照现行的美国移民与国籍法，通常认为是指任何在美国领土上出生的人都享有美国公民权。
 <br/>
 <br/>
 虽然普遍认为该原则的法律依据来源于美国宪法第14修正案，但对于相关条例的解释存有争议。美国保守派社会及政治评论家、专栏作家、律师安．库尔特（Ann Coulter）早在2015年8月就曾发表专栏文章评论说，宪法第14修正案其实不保证落地公民权。
 <br/>
 <br/>
 库尔特还指出，尽管很多人歇斯底里地批评时任总统候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的移民主张，但在她看来，川普是唯一一名在移民问题上观点清晰而又符合逻辑的总统候选人。
 <br/>
 <br/>
 川普10月29日在接受新闻网站节目〝Axios on HBO〞专访时表示，他准备签署行政令废止公民出生地原则，以遏制包括〝链式移民〞（chain migration）在内的移民泛滥。
 <br/>
 <br/>
 川普指出，随便什么人来美国生的孩子都能获得公民身份和福利，这太荒谬了，必须停止。
 <br/>
 <br/>
 10月31日，川普又连发多篇推文，批评公民出生地原则给美国带来诸多危害，誓言废除这项不合理的移民政策，并把此案提交美国最高法院裁决。
 <br/>
