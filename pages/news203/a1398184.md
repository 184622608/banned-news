### 聚焦众院争夺战 民主党忧川普效应重现
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】美国国会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 即将登场，聚焦众院争夺战。总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）赞扬共和党人在参众两院的表现都非常好，民众对共和党人非常支持。而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 人则越来越担心川普效应，会让2016年大选时民主党惨败的场景重现。
 <br/>
 <br/>
 11月6日，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 将会改选众议院435名议员全数改选，参议院将有35席议员改选，众议院另外还有36州进行州长改选，是关乎美国未来政治版图的大事。
 <br/>
 <br/>
 共和党在参议院拥有51席次，需要改选的议员仅9席。而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 拥有49席议员（含两名无党籍议员），其中26席需要改选，面临较大异动风险。加上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 上任近两年来，成功重振经济、大量创造就业、推行减税，广泛赢得选民支持。因此，目前各界多数看好共和党在参院能取得过半席次、维持多数党地位。
 <br/>
 <br/>
 而众议院则是这次中期选举的关键战场。众议院共有435名议员席次，目前共和党拥有235席众议员，民主党拥有193席，另有7席悬缺。想要取得多数党地位，就必须至少赢得218席。
 <br/>
 <br/>
 尽管目前有民调显示，共和党在众议院选举的支持度暂时落后民主党，加上〝钟摆效应〞的预期心理，民主党虽然宣称此次选举将出现〝蓝色浪潮（Blue Wave）〞，但由于2016年总统大选曾发生严重的民调失准乱象，加上此次选情激烈，可能促使更多人出面投票，川普效应可能再一次改变历史。
 <br/>
 <br/>
 支持该党女权倡议的组织EMILY's List主席施罗克称，民主党迄今仍忘不了2016年的惨败。两年前该党总统候选人希拉里在政治期货预测中有91%笃定当选的大热下，最后惨败在川普手上。
 <br/>
 <br/>
 中期选举的结果对当届总统余下任期的执政、以至两年后的大选都将产生不小的影响。而对民主党而言，假如无法重夺众院，该党势必在路线上进一步迷失，要在2020年挑战川普连任就更渺茫。
 <br/>
 <br/>
 目前美国在川普治理下，展开全面政治与社会改革，引领美国向传统回归，触动了诸多政坛旧势力与泛左派团体的利益，也引发左派政客与媒体的猛烈反击。因此，这次中期选举对美国、乃至国际社会的重要性，或许不下于两年前的总统大选，更被部分西方媒体称为〝一生中最重要的中期选举〞。
 <br/>
 <br/>
 由于关系重大，两党共投入超过50亿美元催谷选情，打破历届中期选举纪录。
 <br/>
 <br/>
 美国选民也是热情高涨，已经有2200万到3000万人提前投票。根据提前投票的14个州的情况来看，共和党在12个州领先，民主党只在俄勒冈和爱荷华州领先。有华人学者之前分析，根据数据模型，如果民主党没有8%的民调领先就会输，现在看来这位华人学者是很靠谱的，共和党继续掌控众议院和参议院，美国大选的钟摆效应就会停摆。
 <br/>
 <br/>
 从3月10日到周一（5日），川普在全美一共举办了44场有关中期选举的助选集会。周一当天，川普更是以72岁年龄，连续在俄亥俄，印第安纳和密苏里州举办3场集会。
 <br/>
 <br/>
 周日，川普临行前接受媒体采访时表示，共和党人在参众两院的表现非常好，民众对共和党人非常支持。
 <br/>
 <br/>
 川普直言不讳的说，虽然媒体不愿报导美国经济的良好发展，但现在经济情况是有史以来最好的。美国10月失业率3.7%，经济成长率第三季达3.5%，成为川普辅选的利器。
 <br/>
 <br/>
 选民更表示，一些媒体在妖魔化川普和共和党，他们不相信这些假新闻。
 <br/>
 <br/>
 川普并在自己11月1日的推文中，历数自己过去两年实现的减税、任命大法官、重振经济、重建军队等政绩，呼吁选民投票给共和党。并将集会上支持者人数之多、场面之热烈的盛况，在自己的推特上呈现。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
