### 90天缓刑计时提前 中共38部委联手严惩知识产权失信者
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】日前，中美达成共识暂缓加征
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，给北京3个月的缓冲谈判时间，来解决贸易平衡、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 等问题。为了回应美方，4日，中共发改委等联合数十个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_部委.html" target="_blank">
  部委
 </a>
 联合发布通知，惩戒知识产权领域严重失信者等。
 <br/>
 <br/>
 美国总统川普（特朗普）与中国国家主席习近平，日前在阿根廷G20峰会间隙的工作晚宴上，就持续数个月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，达成〝停战90天的〞共识。
 <br/>
 <br/>
 白宫声明宣布，美国同意在明年1月1日，将对价值2千亿美元的中国产品的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 税率维持在10%。如果双方在未来90天内未能达成协议，10%的关税将会被提高至25%。
 <br/>
 <br/>
 根据白宫声明，中方同意从美国大量进口美国农产品、能源、工业用品外，还同意就美国最为关切的涉及结构性改变的〝强制技术转让、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 保护、非关税壁垒、网路入侵和网路盗窃〞等进行谈判。
 <br/>
 <br/>
 美国白宫3日晚间，更正了稍早前由首席经济顾问库德洛发布的，美中贸易战90天的缓冲期从明年1月1日算起的媒体采访文字记录，白宫更正的期限为今年12月1日开始。
 <br/>
 <br/>
 为了回应美方，在白宫公布90天的计时日期后，4日，中国国家发展改革委等38个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_部委.html" target="_blank">
  部委
 </a>
 ，联合公布《关于对知识产权（专利）领域严重失信主体开展联合惩戒的合作备忘录》。
 <br/>
 <br/>
 《备忘录》要求对重复专利侵权行为、不依法执行行为、专利代理严重违法行为、专利代理人资格证书挂靠行为等6种知识产权领域严重失信行为开展联合惩戒。
 <br/>
 <br/>
 惩戒措施包括限制乘坐飞机火车，限制买房，限制在一定范围的旅游、度假等消费行为。国家知识产权局将加大监管力度，从重处罚违法行为。
 <br/>
 <br/>
 《备忘录》还要求12月底前，实现知识产权领域严重失信主体的联合惩戒。明确联合惩戒对象为知识产权（专利）领域严重失信行为的主体实施者。
 <br/>
 <br/>
 对于知识产权领域严重失信行为，在联合惩戒措施方面，《备忘录》列举共33条惩戒措施，对联合惩戒对象可采取一种或多种惩戒措施。
 <br/>
 <br/>
 惩戒措施还包括限制政府性资金支持，对政府性资金申请从严审核，或降低支持力度，限制补贴性资金和社会保障资金支持，失信情况记入金融信用信息基础资料库及互联网征信系，供金融机构融资授信时审慎性参考。
 <br/>
 <br/>
 《备忘录》由国家发展改革委、央行、国家知识产权局，中共中央组织部、中央宣传部、中央编办、中央网信办，最高法院、科技部、民政部、财政部、人力资源社会保障部等38个部门和单位联合签署。
 <br/>
 <br/>
 尽管中共开始为90天的停火协议对知识产权严重失信主体联合惩戒，但外界仍不看好。
 <br/>
 <br/>
 中国独立经济学家巩胜利接受自由亚洲采访时说，知识产权问题和失信问题之间存在着微妙的区别，政府出台的文件似乎只是走个过场。
 <br/>
 <br/>
 他说，〝侵犯知识产权的意思是侵犯了别人的发明、技术。比如专利和名称；失信则是信用的问题，主要是指货币、金融和债务的问题。这两个应该是有关联，但不是相同的主题。〞
 <br/>
 <br/>
 根据美国贸易代表办公室发布的 〝301〞调查更新报告，中共继续维持原有政策，开展以及支持通过网路入侵来盗窃美国的商业和学术机密。
 <br/>
 <br/>
 巩胜利说：〝中国以前都没有商标、知识产权的概念，是在中共加入世贸组织以后才开始慢慢向国民普及，这和中国的国民教育、政治和经济体系有很深的渊源，不是一时半会可以解决的。
 <br/>
 <br/>
 美国荣鼎集团的中国宏观经济分析师朱鸣岐曾向BBC表示，两国在90天内要谈判的话题，并未涉及中方坚持不让步的国企、产业政策改革等议题。
 <br/>
 <br/>
 他说，就算90天内两国能就知识产权保护、网路盗窃等达成共识，也不能排除华府未来要求北京在更深层的结构性问题上做出让步。
 <br/>
 <br/>
 另外，由于早前习近平在与川普会晤时提出的〝百日计划〞并未实现，外界对90天之后的中美谈判结果仍持怀疑态度。有评论人士表示，不管中共是否是真心回应美国的需求，美国给出的三个月限期，可能是其最后的机会了。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明）
