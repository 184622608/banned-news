### 布朗森牧师回美国 川普白宫接见
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】周六下午，获释的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_布朗森牧师.html" target="_blank">
  布朗森牧师
 </a>
 平安回到美国。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 总统在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 与布朗森会面。
 <br/>
 <br/>
 川普祝贺布朗森回到美国，和家人团聚。
 <br/>
 <br/>
 美国总统川普：〝对安德鲁（布朗森）来说，这一天很有意义。因为你们听我之前说过，从土耳其监狱到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 ，只用不到24小时。〞
 <br/>
 <br/>
 川普告诉布朗森，他妻子不断的打电话求助，也向她表示感谢。
 <br/>
 <br/>
 川普还说，国务卿蓬佩奥等官员一直在就此事与土耳其交涉，但不会也没有付赎金。
 <br/>
 <br/>
 美国总统川普：〝我们一直在交涉，漫长而艰苦，我们没有付赎金，也不会再付赎金。〞
 <br/>
 <br/>
 川普也感谢土耳其总统的帮助，并表示，这对美国和北约国家以及土耳其建立良好甚至强大的关系，具有引导作用。
 <br/>
 <br/>
 美国总统川普：〝我要感谢土耳其总统埃尔多安，我们在谈，这事之前我们一直关系很好，但我仍然奇怪没有在几个月前解决。〞
 <br/>
 <br/>
 布朗森感谢川普总统、彭斯副总统、国务卿蓬佩奥以及所有帮助他获释的人。他还单腿跪下，祈祷神赋予川普超常的智慧。
 <br/>
 <br/>
 基督教福音派牧师 布朗森：〝上帝，我请求你，让你的圣灵在川普总统身上展现，赋予他超凡的智慧，成就你对这个国家以及对他的安排。〞
 <br/>
 <br/>
 外界认为，布朗森的获释是川普外交的又一个胜利，对共和党赢得中期选举又加一分。
 <br/>
 <br/>
 新唐人记者易文、张琪综合报导
</div>
