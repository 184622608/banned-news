### 可疑包裹引发舌战 左派大肆攻击川普
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】从周一（10月22日）开始，一些装有爆炸装置的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_包裹.html" target="_blank">
  包裹
 </a>
 接连寄往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 政要或幕僚家中。虽然这些装置都未爆炸且仍处于调查之中，但左派却藉机挑起新一轮论战，大肆攻击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在挑动暴力。
 <br/>
 <br/>
 周四，前美国副总统拜登（Joe Biden）和好莱坞明星罗伯特‧德尼罗（Robert De Niro）也成为可疑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_包裹.html" target="_blank">
  包裹
 </a>
 指向的目标。
 <br/>
 <br/>
 美国联邦调查局（FBI）正对案件进行调查。其中，寄给拜登的含爆炸物的包裹有2个，虽然发件地址不同，但包裹类似，与近几天发现的其它可疑包裹也都很相似。
 <br/>
 <br/>
 本周来，已有数名显要人物收到同类可疑包裹，而他们无一例外都是敌视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 者，其中包括前总统奥巴马（Barack Obama）、前国务卿希拉里（Hillary Clinton）、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 众议员沃特斯（Maxine Waters）、前司法部长霍尔德（Eric Holder）、前中情局（CIA）局长布伦南（John Brennan）、亿万富翁索罗斯（George Soros）等。
 <br/>
 <br/>
 美国有线电视新闻网（CNN）驻纽约总部周三也收到了可疑包裹，该包裹是寄给常在CNN做嘉宾的布伦南的。CNN随后上演了一幕戏剧化的疏散行动。
 <br/>
 <br/>
 尽管美国情报部门和联邦执法部门尚未得出调查结论，但一些民主党人及自由派
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_媒体.html" target="_blank">
  媒体
 </a>
 已迫不及待地开始攻击，把事件的责任推向川普。
 <br/>
 <br/>
 周三，川普呼吁美国人团结，但民主党参议院领袖舒默（Chuck Schumer）和众议院领袖佩洛西（Nancy Pelosi）却回击川普〝言辞空洞〞，要求他推翻自己〝纵容暴力行为〞的言论。
 <br/>
 <br/>
 在美国最高法院大法官卡瓦诺（Brett Kavanaugh）被确认前后，在民主党的支持下，各种激进的抗议行为空前高涨，民主党的重要金主索罗斯甚至被证实暗地花钱买人发起抗议行动。不少行动上升为暴力行为，川普及共和党高层强烈批评这种失控的暴民政治。
 <br/>
 <br/>
 已被吊销安全许可的布伦南，藉机在推特上要求川普〝停止指责他人〞，并攻击川普鼓励暴力。
 <br/>
 <br/>
 CNN总裁扎克 （Jeff Zucker）甚至发表声明，称事件与川普及白宫新闻发言人桑德斯（Sarah Sanders）不断〝攻击媒体〞有关。
 <br/>
 <br/>
 周四，桑德斯在接受《福克斯与朋友们》（Fox &amp; Friends）节目采访时表示，扎克的声明是〝绝对可耻的〞，他没有根据地指责，并把出现这种卑鄙行为的责任推给川普和她自己。
 <br/>
 <br/>
 川普周三晚在威斯康星州出席集会时表示，〝任何政治暴力行为或威胁都是在攻击民主本身〞。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
