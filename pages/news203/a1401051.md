### 川习G20再会面 川普用关税促协议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 、习近平即将在阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 会面，这是美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 以来双方的首次会晤。白宫经济顾问库德洛周二表示，川普总统持开放态度，但底线不变。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 以来，中国经济持续下行；随着2019年的到来，美国对2千亿中国商品的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 税率将从10%上调至25%，这是目前中共最急于解决的问题。
 <br/>
 <br/>
 对于此次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 是否能达成协议，白宫经济顾问库德洛周二表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统对与习近平达成协议持开放态度，但前提条件必须达成。
 <br/>
 <br/>
 白宫经济顾问 库德洛：〝必须解决知识产权盗窃问题，必须解决强制技术转让，必须解决重大关税和非关税壁垒，必须解决所有权问题。〞
 <br/>
 <br/>
 前一天，川普对媒体表示，谈判的底线没有变。
 <br/>
 <br/>
 美国总统 川普：〝这是底线，中国必须公平对待我们，以前它们从来没有这样做。他们必须公平对待美国。〞
 <br/>
 <br/>
 对于推迟提高对华商品关税的可能性，川普在回答华尔街日报采访时表示，〝极不可能〞（highly unlikely）接受北京方面的这个要求，同时他也说除了上调2千亿中国商品的进口税率，如若达不成协议，还将对另外2,670亿美元中国商品加征10%或25%的关税。而这批商品清单中包含了像苹果手机和笔记本电脑在内的产品，对美国消费者将产生影响。
 <br/>
 <br/>
 白宫经济顾问 库德洛：〝考虑我们的经济规模和实力，我们可以很好的解决这个问题。如果我们确实有自由贸易系统，或者我们朝着自由贸易系统、真正的自由贸易系统的方向前进，我们将受益匪浅，坦率地说，我们将受益、他们将受益、全世界其他地区都将受益。〞
 <br/>
 <br/>
 库德洛表示，达成零关税、零补贴、零非关税壁垒的公平自由贸易是川普真正想要的。然而中国没有遵守任何相关国际规则。
 <br/>
 <br/>
 日前，美国贸易代表办公室更新了301调查报告，结论是中方几个月以来没有任何改变。
 <br/>
 <br/>
 白宫国家安全顾问 博尔顿：〝不期待这次会晤会达成具体协议，但对于双方代表会有一个指导性纲领出炉。〞
 <br/>
 <br/>
 美国股市三大指数周二均有所回升。
 <br/>
 <br/>
 中国问题专家白邦瑞（Michael Pillsbury）今日在推文中说，〝川普在中国关税（问题上）的长期态度表明，华尔街过于乐观。〞
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
