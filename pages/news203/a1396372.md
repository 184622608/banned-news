### 1分钟喷射45万加仑水柱 NASA测试系统
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】为准备
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_火箭发射.html" target="_blank">
  火箭发射
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国宇航局.html" target="_blank">
  美国宇航局
 </a>
 NASA上周（10月15日）测试降温系统，一分钟内喷射出45万加仑的水柱，场面壮观。
 <br/>
 <br/>
 佛州肯尼迪太空中心的发射台，喷出巨大水流，释放出大约45万加仑的水量，水柱瞬间冲到100英尺。
 <br/>
 <br/>
 这是一个点火超压保护和降噪散热系统（Ignition Overpressure Protection and Sound Suppression water deluge system)。之后，巨大的水流很快被各种设施排散。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国宇航局.html" target="_blank">
  美国宇航局
 </a>
 表示，这一测试是为发射其自行研发的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太空发射系统.html" target="_blank">
  太空发射系统
 </a>
 〞做准备，执行深层空间探测任务。
 <br/>
 <br/>
 NASA将发射目前飞得最远的无人宇宙飞船，由最强大的火箭推进器推动升空。
 <br/>
 <br/>
 新唐人电视综合报导
