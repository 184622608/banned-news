### 中美通报川习会结果 中方避提关键内容
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】举世瞩目的中美首脑会晤落幕，中方以多项让步换取了贸易战暂停升级。中美各自通报会晤结果，主要内容大同小异，但中方隐瞒了北京只获得90天谈判时间的关键信息。
 <br/>
 <br/>
 阿根廷当地时间12月1日晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晚宴持续两个多小时后结束。中美双方事后都宣布，会晤取得了〝成功〞。
 <br/>
 <br/>
 北京时间12月2日，新华社报导中共外交部长王毅在记者会上通报的川习会内容。
 <br/>
 <br/>
 据王毅的说法，川习会谈及了中美关系问题、中美贸易战问题、台湾问题和朝鲜半岛问题等。
 <br/>
 <br/>
 其中，双方首脑都同意〝搞好中美关系〞，并准备适当时候再次进行互访。
 <br/>
 <br/>
 有关贸易战问题，王毅表示，中方愿意扩大进口，从美国购买更多的商品，〝逐步缓解〞贸易不平衡问题。中美同意相互开放市场。中方在新一轮改革中，将〝逐步解决〞美国的合理关切。
 <br/>
 <br/>
 王毅说，中美达成共识，停止互相加征新的关税。双方还将朝着取消所有加征关税的方向加紧磋商，尽早达成〝互利共赢的协议〞。
 <br/>
 <br/>
 中共商务部副部长王受文在随后的吹风会中说，美方同意2019年1月1日不会将加征的10%关税提升至25%，并且不再加征新的关税。至于已经加征的关税，将加紧谈判，朝着取消的方向努力。
 <br/>
 <br/>
 和美方有关川习会的声明比较，中方通报大幅〝缩减〞了中方让步的内容，并隐瞒了美方只给予中方90天谈判时间的信息。
 <br/>
 <br/>
 在川习会当晚，白宫就发表声明称，这是一次〝高度成功的会谈〞，并高度赞扬了中方的〝人道主义姿态〞与合作的〝诚意〞。
 <br/>
 <br/>
 根据白宫声明，北京承诺管控中国此前大举〝出口〞美国的毒品——芬太尼；大量进口美国农业、能源、工业和其它产品，其中农产品是〝立即购买〞；〝重新考虑〞此前刻意阻挠的高通-恩智浦收购案；协助美国实现朝鲜无核化等。
 <br/>
 <br/>
 作为交换，美方同意在2019年1月1日，将原定对价值2千亿美元中国产品加征关税从10%提升至25%的计划暂时搁置，并立即开启对中共贸易政策进行结构性改变的谈判，谈判内容包括强制技术转让、知识产权保护、非关税壁垒、网络入侵和网络盗窃、服务业和农业等。
 <br/>
 <br/>
 不过，白宫声明指，只会给予中方90天时间进行谈判。如果届时未能达成协议，10%的关税将会被提高至25%。
 <br/>
 <br/>
 外界评论认为，中共进行结构性改革的诚意值得怀疑，90天所谓〝谈判〞可能只是贸易战暂时停止升级。而对于中美双方来说，都有一定理由宣传自己〝获胜〞，虽然双方的理由多少不一。
 <br/>
 <br/>
 对中方而言，暂缓贸易战本就是这次出席川习会的唯一目的，用此前已经承诺的让步换取90天时间缓解中国经济下行的压力，虽然并无实质帮助，但也算是一次形式上的〞〝胜利〞。
 <br/>
 <br/>
 而对美方来说，用90天暂时停止升级换取了一次中共进行结构性改革的可能性，并且解决了当下一些棘手的问题，包括美中贸易逆差和大豆滞销等。
 <br/>
 <br/>
 美国前财政部的中国事务高级协调员洛文杰（David Loevinger）表示，虽然这次会晤取得了暂免关税战升级的成果，但是很难看出90天后会有根本性变化。
 <br/>
 <br/>
 前美国财政部官员、现为美国外交关系委员会资深研究员特塞尔（Brad Setser）表示，美方这次关税暂停也没有给予中方太多，并且在贸易战持续的过程中赢得了将丰收的大豆运往中国的机会。
 <br/>
 <br/>
 另外，在川习会前，川普一再强调他和习近平的个人友谊。因此也有分析认为，川普顶住来自白宫强硬派和美国民主党的压力，给予中方90天时间，是想再为习近平争取一个做出正确选择的机会。
 <br/>
 <br/>
 随着美中贸易战持续，外界基本已达成共识，认为贸易战已经演化成两种政治体制的根本性对抗。如果中共体制不变，美中长期对抗已经不可逆转，虽然中间可能出现技术性暂缓。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：云涛）
