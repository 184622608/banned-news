### 民调：54％美国人看好川普2020年连任总统
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】美国中期选举结束后，共和党牢牢掌握住了参议院的控制权。随后外界开始关注现任美国总统川普（特朗普）能否在2020年的总统大选中胜出的问题。美国知名的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民意调查.html" target="_blank">
  民意调查
 </a>
 机构近日进行的民意调查显示，超过半数的美国人相信川普能够成功连任美国总统。
 <br/>
 <br/>
 一项最新的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民意调查.html" target="_blank">
  民意调查
 </a>
 显示，超过半数以上的美国人认为，现任总统川普（特朗普）能够在2020年的总统大选中胜出，成功连任。
 <br/>
 <br/>
 美国民意调查机构〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉斯穆森报告.html" target="_blank">
  拉斯穆森报告
 </a>
 〞（Rasmussen Reports）日前公布了一项民意调查的结果，54%的美国人认为川普能够在2020年总统大选中胜出而成功连任。
 <br/>
 <br/>
 据了解，这项调查是于今年11月28至29日进行的。Rasmussen Reports对1000名美国民众进行民意调查，询问他们是否认为川普能够连任美国总统。调查的结果显示，高达54%的美国人认为，无论自己投给谁，2020年美国总统的选举，川普将会再次当选。
 <br/>
 <br/>
 认为川普不太可能成功连任美国总统的人数约占40%，其余受访者则表示难以对此做出判断。
 <br/>
 <br/>
 早在今年7月中旬，川普在访问英国期间就曾在接受英媒《周日邮报》专访时，被问到是否会在2020年竞选连任的问题，当时川普就回答说〝我完全打算这样做（竞选连任）。〞
 <br/>
 <br/>
 川普接着又表示，相信民主党内没有适当人选可与自己匹敌。他说：〝我没有看到任何能击败我的人，我完全了解他们，我看不到任何人。〞
 <br/>
 <br/>
 事实上，Rasmussen Reports针对川普能否连任美国总统的问题已经进行过多次民意调查。而今年8月份有个调查的统计数据显示，相信川普能胜出的比率是39%；11月份的相关统计数字是47%；现在这一统计数据则已超过半数，明显呈现出不断上升的趋势。显示川普正获得越来越多的美国人的认同与支持。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：东野）
