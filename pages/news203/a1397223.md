### 时间敏感 习近平将于美选举前一天发表重要讲话
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】中共官方宣布，北京最高领导人习近平将于北京时间11月5日，出席首届中国国际进口博览会开幕式，并发表主旨演讲。而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 将于当地时间11月6日举行。习近平在美国选举前夕发表演讲，势必引起关注。
 <br/>
 <br/>
 10月29日，在中共商务部的媒体吹风会上，副部长傅自应透露，首届中国国际进口博览会（简称进博会）将于11月5日到10日在上海举行。11月4日，习近平夫妇将举行欢迎宴会。11月5日上午，习近平会出席开幕式并发表主旨演讲。
 <br/>
 <br/>
 近期以来，中美贸易战步步升级，中国经济下行压力加大，习近平已经很长时间没有发表公开演讲。在刚刚结束的〝南巡〞期间，包括港珠澳大桥开通仪式上，习近平也没有〝重要讲话〞。
 <br/>
 <br/>
 上海举行进博会的消息，是习近平去年亲自宣布的。中共副总理胡春华强调，该会展由习近平〝亲自谋划、亲自提出、亲自部署推动〞。由此可见习近平对这次博览会的重视。
 <br/>
 <br/>
 韩联社29日引述北京消息人士透露，此次博览会是为迎接所谓〝改革开放40周年〞而开的，在中美贸易战激化的背景下，习近平可能在开幕式演讲中发布〝扩大对外开放〞的具体信息。
 <br/>
 <br/>
 据外媒报导，此次博览会，参会的大部分是第三世界国家，西方主要国家的首脑几乎无一出席。
 <br/>
 <br/>
 另外，据中共商务部宣布的日程安排，在博览会期间，习近平还将会见与会的外国首脑。这似乎证实了中共十九届四中全会将推迟到11月10日之后的传言。
 <br/>
 <br/>
 路透社日前消息指，四中全会将在11月举行，但会安排在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 之后。按惯例，中共本次全会的中心议题是经济政策，会期一般4天左右。
 <br/>
 <br/>
 消息指，四中全会的主题就是〝落实改革〞，但目前中美贸易谈判结果未知，北京希望在〝外部大的不确定因素稍有明确结果〞后，再确定后续的〝改革方案〞。
 <br/>
 <br/>
 该消息验证了此前外界的判断，即北京在〝豪赌〞美国中期选举。但外界普遍认为，这是北京的误判，因为遏制中共已是美国朝野共识，中期选举不会对川普（特朗普）的贸易政策有太大影响。
 <br/>
 <br/>
 消息还指，四中全会要讨论的经济政策，大方向仍然是〝经济上向市场放权，政治上更强调集权〞，以推动越来越难的〝经济改革〞。但多有海外学者分析，如果不进行政治转型，北京所谓的〝经济改革〞很难再有出路。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
