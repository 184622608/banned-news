### 中期选举结果出炉   川普称巨大成功
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】周三，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的结果揭晓。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 在参议院扩大了领先优势，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 重新夺回众议院多数党地位。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周三凌晨发表推文，祝贺共和党获得巨大成功。
 <br/>
 <br/>
 截至周三清晨，大部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 结果已经揭晓。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 在参议院赢得51席，并在三个尚未公布最后结果的选战中领先，基本确定将扩大参议院的多数党优势。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 和独立党派共赢得45席。最后一个密苏里州参议员席位，因为无人得票过半，将在本月27号举行第二轮决选。
 <br/>
 <br/>
 众议院方面，民主党已经赢得435席中的219席，确定夺回多数党地位。共和党已赢得193席。在尚未公布最终结果的23个席位中，共和党在14个选战中领先，预计民主党最终将在众议院拥有20席左右的微弱多数。
 <br/>
 <br/>
 在州长选举中，民主党从共和党手中夺回了7个州长职位。不过，在全国范围内，共和党州长数量仍然略微占优。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周二晚间发推表示，中期选举取得巨大胜利。
 <br/>
 <br/>
 周三早晨，川普再发推文说，〝收到很多人的众多祝贺，祝贺我们昨晚取得重大胜利，包括来自等待和期待达成贸易协议的国外友人。现在我们都可以回到岗位，把事情做完！〞
 <br/>
 <br/>
 美国中期选举历来存在〝钟摆效应〞，新任总统所在党通常会在首个中期选举中损失一些国会席位。
 <br/>
 <br/>
 川普在一则推文中引用历史数据，表明过去105年间，只有五次在任总统能够在中期选举中增加参议院议席。
 <br/>
 <br/>
 对于民主党夺回众议院，并有可能针对现任政府展开调查，川普警告说，〝如果民主党人认为他们能够浪费纳税人的钱在众议院调查我们，那我们也将被迫考虑在参议院调查他们，包括泄露机密信息以及其他种种。你要来这一套，我们也可以！〞
 <br/>
 <br/>
 新唐人记者林宇、明玉综合报导
 <br/>
