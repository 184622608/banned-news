### 用催泪瓦斯对付非法移民该轰？ 奥巴马早有先例
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 目前正激烈抨击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_催泪瓦斯.html" target="_blank">
  催泪瓦斯
 </a>
 对付非法入境的移民，但却故意忽略这种做法早在2013年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 执政时期就有过类似先例。
 <br/>
 <br/>
 周日（11月25日），为应对数百名大篷车移民强行入境，美国边境执法人员使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_催泪瓦斯.html" target="_blank">
  催泪瓦斯
 </a>
 加以阻拦。但此举招来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 激烈反对，他们抨击边境人员的执法行为违背了道德。
 <br/>
 <br/>
 滞留在蒂华纳（Tijuana）的中美洲移民上周末开始使用暴力，他们突破墨西哥防暴警察的封锁，然后冲向圣地亚哥（San Diego）边界，并向边防人员投掷石块，企图强行进入美国。
 <br/>
 <br/>
 事件发生后，民主党人藉机打出人道主义的幌子，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府发动猛烈的政治攻击。夏威夷州民主党参议员布莱恩．夏兹（Brian Schatz）甚至在推特上把催泪弹比作化学武器。之后，他又指控边境巡逻人员专门向孩子喷洒催泪瓦斯。
 <br/>
 <br/>
 美国穆斯林人权活动家卡西姆．拉希德（Qasim Rashid）甚至进行恶意地比较，把川普政府使用催泪瓦斯对比世界上的独裁政权对自己的国民使用致命毒气，比如叙利亚阿萨德（Bashar Assad）政权和伊拉克萨达姆．侯赛因（Saddam Hussein Abd al-Majid al-Tikriti）政权的暴行。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 时期的国家安全顾问本．罗兹（Ben Rhodes）甚至也指责，川普政府使用催泪瓦斯是不道德的。
 <br/>
 <br/>
 具有讽刺意味的是，奥巴马政府早在2013年就曾使用过类似方式拦阻移民非法入境。
 <br/>
 <br/>
 据《圣地亚哥联合论坛报》（The San Diego Union-Tribune）2013年11月25日报导，当时，曾有100多名移民在圣伊西德罗入境口岸（San Ysidro Port of Entry）附近向美国边境人员投掷石块和瓶子，边防人员以发射胡椒球作为回应，迫使这些移民回到墨西哥境内。
 <br/>
 <br/>
 而在5年后的同一天，当数百名中美洲移民冲向边境围栏企图非法入境、并且同样向边防人员投掷石块时，美国边防人员在情况更为严重的情况下采取与当年类似的方式抵挡移民非法侵入国境，却招致民主党及自由派媒体的严厉谴责。
 <br/>
 <br/>
 另外，当年事件发生时，奥巴马也曾强烈呼吁加强边界安全，并敦促国会就移民改革出台方案。
 <br/>
 <br/>
 但在今天，民主党已不再呼吁制止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 ，而是以人道主义为藉口，把批评之声引向自己的政治对手——川普。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
