### 老布什总统去世 家乡民众纪念
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】94岁的美国前总统老布什，周五晚在美国德州休斯顿市的家乡安详
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_去世.html" target="_blank">
  去世
 </a>
 。当地民众前往老布什夫妇的住所、以及他的雕像前，留下了鲜花和花环，以示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纪念.html" target="_blank">
  纪念
 </a>
 。
 <br/>
 <br/>
 今年初，和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什总统.html" target="_blank">
  老布什总统
 </a>
 结婚73年的妻子芭芭拉也以92岁的高龄辞世。两人辞世前后仅相距8个月。
 <br/>
 <br/>
 老布什从20岁第二次世界大战的英雄到第41任美国总统期间，在近一个世纪的生涯中扮演了许多重要公共角色，包括国会议员，中央情报局局长，联合国和中国大使，以及两任副总统。
 <br/>
 <br/>
 但他的同事和朋友们却说，这位曾祖父谦卑，谦虚，总是彬彬有礼。
 <br/>
 <br/>
 敬献鲜花的当地民众表示，他们非常怀念老布什。
 <br/>
 <br/>
 德克萨斯州休斯顿居民麦克-彼得森：〝毫无疑问，他会非常被怀念。我认为他很出色的是，无论你是民主党人还是共和党人，他把人们聚集在了一起。〞
 <br/>
 <br/>
 德克萨斯州休斯顿居民麦克-彼得森：〝我们休斯顿非常想念他的原因之一是，你会在棒球比赛、足球比赛，或者一些当地餐馆看到他。他是个很谦虚的人。他会被非常怀念。〞
 <br/>
 <br/>
 德克萨斯州休斯顿居民科瑞-扎恩科：〝无论你是否同意他的政策，他热衷和想要为美国民众服务，这真的令人难以忘怀。〞
 <br/>
 <br/>
 德克萨斯州休斯顿居民汤姆-曼尼斯：〝我自己是个保守派。看到两派政治立场的人们走到一起，由衷怀念他，真是很好。〞
 <br/>
 <br/>
 老布什在1989年至1992年间担任美国第41任总统。他执政期间，经历了柏林墙倒塌、天安门六四事件、苏联解体、海湾战争等重要历史事件，领导美国在冷战结束时期向冷战后过渡。之前，他曾担任里根总统的副总统8年。同时他也是第43任总统小布什的父亲。
 <br/>
 <br/>
 新唐人记者徐哲、天羽综合报导
