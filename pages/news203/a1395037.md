### 川普律师拟书面回答穆勒问题 通俄门调查或接近尾声
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】美媒周四（10月11日）报导，为尽快结束
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查，川普总统的律师已准备
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_书面回答.html" target="_blank">
  书面回答
 </a>
 特别检察官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 （Robert Mueller）提出的问题，这一进展预示着持续了近一年半之久的穆勒调查已接近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尾声.html" target="_blank">
  尾声
 </a>
 。
 <br/>
 <br/>
 据美国有线电视新闻网（CNN）11日报导，消息人士透露，川普的律师团队已准备
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_书面回答.html" target="_blank">
  书面回答
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 的提问，这标志着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普律师.html" target="_blank">
  川普律师
 </a>
 与穆勒团队之间的谈判已取得重大进展，同时也示意
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查可能已进入最后阶段。
 <br/>
 <br/>
 围绕川普总统是否接受穆勒团队的面询或是提交书面答覆，双方已争执数月。
 <br/>
 <br/>
 由塞库洛（Jay Sekulow）和朱利安尼（Rudy Giuliani）所领导的川普法律团队已经压缩了提交川普回答的问题。
 <br/>
 <br/>
 塞库洛告诉CNN，他们正与穆勒团队继续讨论，目前不想对此进行评论。
 <br/>
 <br/>
 穆勒于2017年5月17日被指定为特别检察官，负责调查川普选战是否勾结俄罗斯干扰了2016年美国总统大选。在长期调查仍找不出任何证据后，穆勒开始把调查重点转向川普是否妨碍了司法公正，尤其是针对前联邦调查局（FBI）局长科米（James Comey）被川普解雇的事件。
 <br/>
 <br/>
 科米是穆勒的好友，于2017年5月9日被川普开除。
 <br/>
 <br/>
 据称川普的律师们避免让川普回答关于妨碍司法公正的问题。朱利安尼强调，川普有权开除科米，只要他认为理由适当，因为科米是受总统雇用的内阁成员。
 <br/>
 <br/>
 《纽约时报》（The New York Times）今年4月曾公布了一份清单，上面罗列了近50个穆勒提交给川普律师团队的问题，涵盖范围广泛，包括川普选战的活动，还包括解雇前白宫国家安全顾问弗林（Michael Flynn）等问题。
 <br/>
 <br/>
 川普总统多次公开表示，他对会晤穆勒持开放态度，但他的律师们强烈建议他拒绝面谈，以免陷入穆勒所设置伪证陷阱。
 <br/>
 <br/>
 同时，川普的法律团队也在准备反报告，质疑穆勒调查的合法性。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
