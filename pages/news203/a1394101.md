### 美法院裁定监督延期 中兴将接受美国三重监管
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】作为美国放生的条件，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴通讯.html" target="_blank">
  中兴通讯
 </a>
 公司7月被迫接受美国商务部官员的监督。日前美国法院裁定，将法院此前派驻中兴的监察官任期延长两年，并另外指定一名和商务部监察官拥有同样权限的监督员。
 <br/>
 <br/>
 路透社消息，美国得克萨斯州达拉斯市一位地区法官金基德（Ed Kinkeade）周三（10月3日）裁定，因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴通讯.html" target="_blank">
  中兴通讯
 </a>
 违反了缓期处罚规定，该法官把他任命的监察官任期延长至2022年。
 <br/>
 <br/>
 这名监察官的职责是评估中兴是否遵守美国出口管制规定，任期原定2020年结束。
 <br/>
 <br/>
 中兴周四向香港股票交易所通报了美国法院的裁决。除了延长监察官任期，中兴表示还必须另外聘请一名由法庭指定的监督员，拥有和美国商务部派驻监督员同等的权力。
 <br/>
 <br/>
 中兴因为违反美国出口管制规定，将美国产品和技术非法出售给伊朗，于2017年3月接受美国的缓期处罚，并支付了8.92亿美元罚金。
 <br/>
 <br/>
 但美国商务部今年初认定，中兴公司违反了缓期处罚的规定，没有处罚涉事的35名员工，因而对中兴发出为期7年的禁售令。在中方斡旋下，美国商务部7月将禁售令修改为10亿美元罚款加4亿美元保证金，以及全部更换中兴管理层。
 <br/>
 <br/>
 作为解除禁售令的条件之一，中兴还同意聘请一名美国商务部指定的合规监察官，为期10年，该监察官直接向美国商务部汇报。
 <br/>
 <br/>
 美国法官延长对中兴的监督期，也是基于商务部处罚中兴同样的理由。
 <br/>
 <br/>
 根据美国法院裁决，中兴很快将被迫接受来自美方的第三名监察员，接受美方三重监督。
 <br/>
 <br/>
 中美7月有关中兴的和解协议，因为涉及接受美国官员到中企执法，曾有观点认为，此举涉嫌〝丧失主权〞，北京可能不会接受。
 <br/>
 <br/>
 中兴禁令解除后，美国商务部长罗斯表示，商务部仍然有三项措施保护美国国家安全：对中兴暂时搁置的禁令、4亿美元保证金，以及向中兴派驻监察人员。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
