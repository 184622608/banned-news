### 中兴通讯疑助委国监视国民 美参议员要求调查
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】中国电信大厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴通讯.html" target="_blank">
  中兴通讯
 </a>
 公司疑协助
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 设置资料库以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监视国民.html" target="_blank">
  监视国民
 </a>
 行为，美国联邦2名参议员今(28日)要求川普政府着手调查。
 <br/>
 <br/>
 路透社报导，美国民主党籍参议员范荷伦（Chris Van Hollen）和共和党籍参议员卢比奥（Marco Rubio）写信要求国务卿、财政部长和商务部长判定，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴通讯.html" target="_blank">
  中兴通讯
 </a>
 是否与美国制裁的个人合作、非法使用美国零件，或协助委国政府藐视民主进程或人权。
 <br/>
 <br/>
 目前包括美国国务院、商务部、中兴通讯与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 资讯部均未回应要求出面置评。
 <br/>
 <br/>
 穷于应付恶性通膨和经济崩落的委国总统马杜洛（Nicolas Maduro）则认为，美国制裁是华府颠覆他左派政府的〝经济战〞部分手段。
 <br/>
 <br/>
 中兴通讯遭许多西方国家官员指控协助中国出口监控策略和设备给全球独裁主义政府。此前中兴通讯因违反美国制裁措施，对伊朗、朝鲜非法出售商品，今年以来已支付美国政府10亿美元罚款。
 <br/>
 <br/>
 (责任编辑：夏明义)
