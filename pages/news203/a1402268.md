### 四天悼念活动结束 老布什安葬德州
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】美国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灵柩.html" target="_blank">
  灵柩
 </a>
 从华盛顿DC送回德州，周四，在休斯顿的追悼仪式后，老布什遗体会送往布什总统图书馆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安葬.html" target="_blank">
  安葬
 </a>
 。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灵柩.html" target="_blank">
  灵柩
 </a>
 送抵德州休斯顿的圣马丁主教教堂。
 <br/>
 <br/>
 老布什的长子、前总统小布什等众多家属及来宾共几百人，出席了在这里举行的第二场追悼仪式。
 <br/>
 <br/>
 老布什的长孙，德州土地管理局局长发表演说，向祖父表达敬意。
 <br/>
 <br/>
 老布什长孙 George P. Bush：〝他经常谈到永恒的信条：责任、荣誉和国家。这个延续超过240年的美国价值观。但这不是他随便说说，而是他身体力行。〞
 <br/>
 <br/>
 仪式结束后，载着老布什灵柩的专列离开休斯顿。
 <br/>
 <br/>
 沿途之上，当地居民聚集到专列经过的铁路旁，挥手送他最后一程。
 <br/>
 <br/>
 专列将驶往德州卡城的布什总统图书馆。周四晚，老布什的遗体将被安放在那里，与他今年4月去世的妻子和一个早逝的小女儿一同
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安葬.html" target="_blank">
  安葬
 </a>
 。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
