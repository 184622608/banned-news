### 俄女干预期中选举 美司法部首次起诉
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】美国司法部针对11月国会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_期中选举.html" target="_blank">
  期中选举
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网路干预.html" target="_blank">
  网路干预
 </a>
 动作，起诉俄罗斯〝网路研究社〞的财务负责人库斯亚诺娃。据悉，库斯亚诺娃自2014年以来一直在管理一项网路干预行动的财务，这些行动目的是在美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选举.html" target="_blank">
  选举
 </a>
 期间散播假消息并造成社会分歧。而这起起诉事件成为今年第一位因期中选举而遭起诉的人士。
 <br/>
 <br/>
 路透社报导，44岁的库斯亚诺娃（Elena Alekseevna Khusyaynova）遭起诉同一天，美国执法和情报部门也发出警告，宣称俄国、中国、伊朗和其他外国机构企图干预11月6日美国国会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_期中选举.html" target="_blank">
  期中选举
 </a>
 。
 <br/>
 <br/>
 美国检方指出，库斯亚诺娃自2014年以来一直在管理拉〝拉赫塔计划〞（Project Lakhta）干预行动的财务，这些行动目的是在美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选举.html" target="_blank">
  选举
 </a>
 期间散播假消息并造成社会分歧。2016年1月至2018年6月期间，拉赫塔计划提出的所有业务预算超过3500万美元。
 <br/>
 <br/>
 而这项计划是俄国圣彼得堡〝网路研究社〞（IRA）的广泛政治干预行动。检方提到，〝网路研究社〞是由俄国总统普京（Vladimir Putin）亲密盟友普里格钦（Yevgeny Prigozhin）所掌控。
 <br/>
 <br/>
 普里格钦与其他12名〝网路研究社〞员工因涉嫌在2016年美国总统大选期间散播假消息，已在今年2月遭到美国检方起诉。
 <br/>
 <br/>
 报导说，他们当时在脸书（Facebook）、推特（Twitter）及Instagram等社群平台散播数以百万则贴文，目的在于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_煽动.html" target="_blank">
  煽动
 </a>
 政治阵营与社会团体之间的敌意。
 <br/>
 <br/>
 起诉书还详述俄方在社群媒体使用假帐号干预选举的新案例，例如在种族、枪枝权利、选举舞弊和其他争议问题上造成社会分歧。
 <br/>
 <br/>
 美国密西根州前联邦检察官麦奎德（Barbara McQuade）表示：〝此案显示来自俄国的威胁尚未结束，这是一场真正的宣传战。〞
 <br/>
 <br/>
 (责任编辑：程以仁)
