### 军事圆桌会议 川普聚焦航太与重工
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】川普(特朗普)总统在助选途中，周五在亚利桑那州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_路克空军基地.html" target="_blank">
  路克空军基地
 </a>
 （Luke Air Force Base）参加了一个国防圆桌会议。他强调要让美国军事力量达到无人可以匹敌。
 <br/>
 <br/>
 亚利桑那州的军事工业在全美排第三位，提供了76,000个工作，创造经济产值115亿美元。
 <br/>
 <br/>
 国防部副部长 Patrick M. Shanahan（夏纳汉）：〝我想最重要的一点是，充足的预算让我们的军力更现代化、具有竞争力。〞
 <br/>
 <br/>
 美国总统 川普：〝你说到竞争，我想说甚至没有人可以赶上我们、和我们竞争。因为我要做的就是让我们的军力无人可以匹敌。 〞
 <br/>
 <br/>
 波音、洛克希德．马丁（Lockheed Martin）、诺斯洛普．格鲁曼(Northrop Grumman)公司、通用动力（General Dynamics）等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国防军工企业.html" target="_blank">
  国防军工企业
 </a>
 受邀参与了圆桌会议。
 <br/>
 <br/>
 波音公司CEO米伦伯格感谢川普总统投资国防，重视军工基地及促进制造业的各项政策。
 <br/>
 <br/>
 波音公司CEO Dennis Muilenburg：〝特别是税改、减少规管并持续下去，这非常重要，这让我们在国防领域可以进行投资和创新。〞
 <br/>
 <br/>
 川普签署的2019财年《国防授权法案》，通过了7160亿美元国防预算。川普还给美国军人涨工资，并重建核武库让其更加现代化。
 <br/>
 <br/>
 美国总统 川普：〝军队对我而言非常重要。我们在很短的时间内就会让人无法企及。我们不会想用核武。如果需要用，当然最好是用不到。〞
 <br/>
 <br/>
 他鼓励军工制造，并将他们出售给盟友。今年5月以色列使用F-35战斗机在中东执行空袭，让F-35首次参与实战。沙特则签署了购买美国1千1百亿美元军事装备的合同。
 <br/>
 <br/>
 另外，川普再次提到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太空部队.html" target="_blank">
  太空部队
 </a>
 ，并说相信国会很快会通过议案。
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
