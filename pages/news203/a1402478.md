### 中共高官改口:中国强大只是幻象 必须捍卫中美关系
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月09日讯】伴随着北京向华盛顿低头求和，中共官方言论在全面降调。一度声称将贸易战打到底的驻美高官，最新表态更为〝大胆〞。驻美公使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克新.html" target="_blank">
  李克新
 </a>
 称，所谓〝中国强大〞四成是幻象。驻美大使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔天凯.html" target="_blank">
  崔天凯
 </a>
 则表示，必须挫败〝摧毁中美关系〞的企图。
 <br/>
 <br/>
 12月5日，美中政策基金会举行23周年年会，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共驻美大使.html" target="_blank">
  中共驻美大使
 </a>
 馆公使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克新.html" target="_blank">
  李克新
 </a>
 在致辞中称，对中国〝强大〞影响力的看法，有60%是〝基于现实〞，20%纯属〝外界想像〞，剩下20%是中方自己创造的〝民族主义泡沫〞。
 <br/>
 <br/>
 李克新没有透露他如何得出的这个比例。但是，称〝中国（共）强大〞有四成是虚幻，该言论出自中共高官之口，显得不同寻常。
 <br/>
 <br/>
 李克新还批评称，那些声称〝中共通过百年马拉松超越美国〞的书籍，认识太过〝肤浅〞。他强调，尽管中共取得了〝巨大经济进步〞，但许多地区仍十分贫困，各地区发展差距惊人，而且保护产权和改善空污都是有待解决的难题。
 <br/>
 <br/>
 对于中美90天的贸易战谈判，李克新表示，只要不涉及中共〝主权〞，北京方面〝愿意妥协〞。
 <br/>
 <br/>
 在同日举行的美中贸委会成立45周年庆典上，中共驻美大使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔天凯.html" target="_blank">
  崔天凯
 </a>
 表示，中美两国工商界需要的是市场准入而不是壁垒，是开放窗口而不是铁幕，〝有人企图发动〞贸易战、关税战、网络战、冷战甚至热战，以〝百年马拉松〞之名进行战略对抗，〝绝对不会得逞〞。
 <br/>
 <br/>
 他声称，要挫败那些欲〝将中美关系45年心血毁于一旦的企图〞。
 <br/>
 <br/>
 崔天凯还盛赞上周六〝坦诚而友好〞的美中首脑会晤，指双方首脑已同意推进所谓〝协调、合作、稳定的中美关系〞。据此看来，崔天凯口中〝企图摧毁中美关系〞的人，不是指美国总统川普。
 <br/>
 <br/>
 与崔天凯此前〝北京不怕贸易战〞的言论相比，他这番〝坚决不要贸易战〞的貌似强硬但失去指责目标的表态，更象是为北京即将做出的贸易妥协辩护。
 <br/>
 <br/>
 12月6日，中共商务部再度就美中贸易谈判表态，其间没有提及华为CFO被抓事件，而是再次强调川习会的〝成功〞，并承诺切实推进与美国间达成的关于农业、能源、汽车等领域的协议，还宣称中方〝有信心〞能在90天内与美国就贸易问题达成一致。
 <br/>
 <br/>
 中方这次表态，等于变相承认了美方此前透露的中方同意大量购买美国农产品和能源产品、取消美国汽车进口关税等。
 <br/>
 <br/>
 根据白宫官员的说法，中方在川习会上还提出了高达1.2万亿美元的采购计划，以及在保护知识产权、降低贸易壁垒方面也有所承诺。
 <br/>
 <br/>
 不过，中方迄今对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_90天谈判.html" target="_blank">
  90天谈判
 </a>
 的重点——结构性改革依然只字不提，特别是有关政府补贴、强制技术转移、拆除关税壁垒等关键问题。
 <br/>
 <br/>
 而根据白宫声明，在这90天谈判期内，美中必须就中共结构性改革达成协议，否则将继续升级关税战。而且，美方提出的贸易要求没有任何让步的迹象。
 <br/>
 <br/>
 因此，尽管中方一再放低姿态，做出兑现承诺、愿意妥协的表示，但多有分析指，这或许只是巨大压力下的一种拖延战略，如果中共拒绝就关键问题做出让步，90天达成协议的可能性不高。
 <br/>
 <br/>
 另外，虽然美国在贸易方面对中共暂缓出招，但在其它领域遏制中共的措施并没有停步。抓捕孟晚舟，就是制裁中共盗窃知识产权和对中共进行科技围剿的一个重大举措。
 <br/>
 <br/>
 在今年8月出台的美国新国防授权法案中，已明确与中共的〝长期战略竞争〞是美国的根本战略。外界分析，美中长期对抗已不可逆转。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：）
