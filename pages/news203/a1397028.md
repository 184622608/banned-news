### 美军反弹道飞弹系统测试拦截成功
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】美国国防部表示，军方26日在夏威夷西岸外海成功的测试
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反弹道飞弹系统.html" target="_blank">
  反弹道飞弹系统
 </a>
 。官员说，这是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_标准3型.html" target="_blank">
  标准3型
 </a>
 第二次成功拦截，使美国飞弹防御局更有信心解决以前导致拦截失败的设计缺陷，成功的测试使系统更接近量产。
 <br/>
 <br/>
 综合媒体报导，美军飞弹
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驱逐舰.html" target="_blank">
  驱逐舰
 </a>
 芬恩号（USS John Finn）以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神盾作战.html" target="_blank">
  神盾作战
 </a>
 系统（Aegis system）探测及追踪目标靶弹后，发射
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_标准3型.html" target="_blank">
  标准3型
 </a>
 （SM-3 Block IIA）拦截弹，并且成功拦截。
 <br/>
 <br/>
 这次是标准3型拦截飞弹进行第4次拦截测试，此前3次仅于去年2月首次测试时进行成功，随后2次均以失败告终。
 <br/>
 <br/>
 美国导弹防御局(MDA)局长格里夫斯中将(Lt. Gen. Samuel Greaves)表示，这是标准3型拦截飞弹一次重要的里程碑，取得一个极好的成就。
 <br/>
 <br/>
 <b>
  相关视频：
 </b>
 <center>
 </center>
 <br/>
 不久前，美国因为中共和俄罗斯的飞弹威胁，有意退出《中程导弹条约》，且与朝鲜无核化持续进展的情况下，此次美军成功测试
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反弹道飞弹系统.html" target="_blank">
  反弹道飞弹系统
 </a>
 ，证明美国有能力击落来自朝鲜和伊朗等流氓国家的新中程飞弹。
 <br/>
 <br/>
 据悉，此拦截弹由美国军事承包商雷神公司（Raytheon）和日本三菱重工共同研发，预计将在罗马尼亚、波兰、日本的神盾系统上装备，能够从海上或陆地上发射拦截飞弹。
 <br/>
 <br/>
