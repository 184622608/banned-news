### 美联邦法官要求进一步调查希拉里〝邮件门〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】日前，美国一名联邦法官要求就司法部和国务院对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希拉里.html" target="_blank">
  希拉里
 </a>
 ．克林顿〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邮件门.html" target="_blank">
  邮件门
 </a>
 〞案件的处理情况展开进一步的调查，向公众提供更多细节。
 <br/>
 <br/>
 据《华盛顿邮报》报导，美国联邦地方法院法官罗伊斯．兰伯思（Royce C. Lamberth）12月6日发表了一份措辞严厉的意见书，要求有关机构对司法部和国务院当初处理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希拉里.html" target="_blank">
  希拉里
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邮件门.html" target="_blank">
  邮件门
 </a>
 〞案件的情况作进一步的检查，搞清楚是否有人故意逃避相关的公共记录法律，恶意向法院隐瞒他们已获得希拉里删除的邮件这一事实达数月之久。
 <br/>
 <br/>
 兰伯思在意见书中表示，虽然美国政府声称信息透明，但面对当今对政府公开透明的原则最严重侵犯的事件，（奥巴马政府时期）国务院和司法部的做法远远达不到相关的法律要求。
 <br/>
 <br/>
 兰伯思指出，尽管川普总统在竞选期间多次抨击希拉里没有公开她的电子邮件，但〝当时的司法部〞让事情变得更糟。有人说司法部和国务院没有义务搜索不为政府所有的记录，但兰伯思对这种说法提出质疑。
 <br/>
 <br/>
 兰伯思直言：最好的情况是，政府的行为是〝因无能而产生的疏忽〞；〝最坏的情况是，国务院和司法部的工作人员合谋阻止公众对希拉里的审查，绕开《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信息自由法.html" target="_blank">
  信息自由法
 </a>
 》，欺骗法庭。〞
 <br/>
 <br/>
 希拉里．克林顿在2009年至2013年担任国务卿的4年时间里，曾无视存在洩露国家机密的巨大风险，擅自使用私人的电子邮件服务器及个人电子邮件账户来处理政府事务，却不使用在联邦服务器上维护的官方国务院电子邮件帐户。而这些官方通信中，包括上千封后来被国务院归类为国家机密的电子邮件。
 <br/>
 <br/>
 2015年3月希拉里的这种做法被曝光后，在美国朝野引发巨大争议。希拉里因此被保守派组织司法观察等起诉，指控她违反了美国《信息自由法》（FOIA）。希拉里此后宣称已永久性删除了私人邮箱中的所有电邮，不能向调查委员会提交有关邮件。
 <br/>
 <br/>
 2015年9月下旬，彭博社在一篇报导中引述消息人士透露，联邦调查局（FBI）已从希拉里‧克林顿任国务卿期间所用的电脑服务器中，成功恢复了曾被她删除的电子邮件。这使得最终公布这名民主党总统竞选人的邮件往来变成可能。
 <br/>
 <br/>
 美国国务院曾经在网站上公布了希拉里近8000个与公务有关的电子邮件，并确定其中数十个邮件含保密信息。那些与公务有关的邮件内容包括：日程安排、新闻简报、人员更换、发言稿、要求助手技术支持等。
 <br/>
 <br/>
 自从邮件门事件曝光以来，希拉里的民调便不断下滑，民众质疑她的诚信，批评她鲁莽的做法会危及国家敏感信息的安全。
 <br/>
 <br/>
 FBI经过一年多的调查后于2016年7月5日宣布调查结束。当时身为FBI局长的詹姆斯．科米发表声明称，希拉里在处理她的电子邮件系统上〝极其粗心大意〞，但还不属于犯罪性质，调查的结论是建议不对希拉里提起指控。
 <br/>
 <br/>
 第二天（7月6日），时任美国司法部长洛丽泰．林奇宣布司法部门正式停止调查希拉里〝邮件门〞事件，不会对其提起任何诉讼。
 <br/>
 <br/>
 不过，在FBI对希拉里的服务器上的电子邮件进行调查的过程中，确认至少有113封电子邮件所载信息在当时是秘密，其中 65封电子邮件属于〝机密〞，22封邮件属于〝绝密〞，3封可为机密，但缺少机密标题，仅被用圆括号中的小写〝c〞标记。后来，希拉里私人服务器上的近2100封电子邮件又被国务院追溯标记为机密。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：东野）
