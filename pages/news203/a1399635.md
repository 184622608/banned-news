### 杜鲁门-里根自由奖得主：退党大潮影响巨大
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】日前， 美国共产主义受难者基金会，向为波罗的海三国摆脱共产专制做出杰出贡献的政治家，颁赠了本年度〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜鲁门.html" target="_blank">
  杜鲁门
 </a>
 -
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根自由奖.html" target="_blank">
  里根自由奖
 </a>
 〞。获奖人士称赞，中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退党大潮.html" target="_blank">
  退党大潮
 </a>
 影响巨大。
 <br/>
 <br/>
 日前，美国共产主义受难者基金会将今年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜鲁门.html" target="_blank">
  杜鲁门
 </a>
 -
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根自由奖.html" target="_blank">
  里根自由奖
 </a>
 ，分别颁赠给来自立陶宛、拉脱维亚和爱沙尼亚的三位政治家，表彰他们为三国摆脱前苏共统治，实现民族独立所做的贡献。
 <br/>
 <br/>
 共产主义受难者基金会主席 Lee Edwards：〝这三国站得很高，在反对共产主义暴政的战斗中是巨人。因此我们认为，如果他们能这样站起来反抗，美国和其它西方民主国家就更该这么做。〞
 <br/>
 <br/>
 获奖的欧洲议会议员图恩‧克莱姆(Tunne Kelam)来自爱沙尼亚，被视为是杰出的反共斗士。对于中国目前的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退党大潮.html" target="_blank">
  退党大潮
 </a>
 ，克莱姆给予高度评价。
 <br/>
 <br/>
 欧洲议会议员 Tunne Kelam：〝很多人都知道退党正在发生。其数字与影响力一定是非常巨大的。〞
 <br/>
 <br/>
 截止目前为止，在海外大纪元网站上声明退出中共党、团、队组织人数已经超过三亿一千九百万。曾亲身经历苏联解体和东欧巨变的克莱姆议员，对此表示非常鼓舞。
 <br/>
 <br/>
 欧洲议会议员 Tunne Kelam：〝这是最务实、最现实的方式来削弱共产专制。如果成百上千万的人都在这样做，这令人充满希望。这是一个全新的开始，必将产生很积极的影响。〞
 <br/>
 <br/>
 新唐人记者杨立新、王凯迪美国首都华盛顿报导。
</div>
