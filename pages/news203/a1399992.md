### 川普夫妇迎白宫圣诞树 开启快乐节日季
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 即将到来，周一，美国总统川普和夫人，第二次在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 迎接白宫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞树.html" target="_blank">
  圣诞树
 </a>
 。一起来看一看。
 <br/>
 <br/>
 周一中午，伴随着节日音乐，两匹高头大马拉着马车和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞树.html" target="_blank">
  圣诞树
 </a>
 抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 北门，似乎宣告了美国2018感恩与圣诞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_节日季.html" target="_blank">
  节日季
 </a>
 的正式开始。
 <br/>
 <br/>
 自1966年以来，白宫圣诞树都由〝全美圣诞树大赛〞的优胜者提供。今年被选中的是一株19.5英尺的弗雷泽冷杉，来自北卡罗莱纳州。它稍后会竖立在白宫蓝厅展示。
 <br/>
 <br/>
 此外，川普总统即将于周二在白宫玫瑰园举行一年一度的火鸡赦免仪式。
 <br/>
 <br/>
 今年，最幸运的火鸡将从〝豌豆〞和〝胡萝卜〞间产生，他们来自南达科他州的农场。不过，赦免仪式后，它们都将被送往弗吉尼亚理工学院安度余生。目前，它们在华府一家酒店等待出场。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
