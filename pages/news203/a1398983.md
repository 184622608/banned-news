### 加州野火令人心碎 〝300壮士〞男星豪宅烧到只剩钢筋
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】美国南加州野火肆虐，已造成31人死亡、200多人失踪，30万人被迫从家中撤离。好莱坞巨星〝300壮士〞（300）杰拉德．巴特勒花645万美元在马里布购入的豪宅，也被烧成废铁。他发推文说这是〝整个加州的心碎时刻〞。
 <br/>
 <br/>
 据《今日美国报》（USA Today）报导，主演《斯巴达300勇士》的杰拉德．巴特勒（Gerard Butler）得知自己的家付之一炬后，与烧到只剩断垣残壁的房子合照，并在推文中说，在撤离后回到马里布的家…野火带来的灾难令人心碎，感谢加州消防队的付出，希望民众可以募款给警消单位救援。
 <br/>
 <br/>
 画面中可见，在他身后一辆货车被烧成灰烬，而整栋豪宅也被烧到只剩钢筋，且现场仍弥漫着些微浓烟。
 <br/>
 <br/>
 他还在Instagram限时动态影片中说：〝欢迎来到我马里布（Malibu）的家。〞

 </a>
 〞（Woolsey Fire）在8日晚间开始燃烧，9日朝马里布和太平洋推进，当局疏散马里布、卡拉巴萨斯（Calabasas）、阿古拉山（Agoura Hills）等其他附近区域民众。
 <br/>
 <br/>
 路透社报导，又发现6名受害者遗体后，目前死亡人数攀升至31人。巴特县（Butte County）警长哈尼亚（Kory Honea）表示，仍有200多人下落不明。
 <br/>
 <br/>
 加州迄今已投入至少810万美元灭火。加州当局表示，自8日以来，已投入数千消防员全力救火，动用了灭火专用飞机、铲土机等大型机械，更多的消防员将前往火灾地区。天气预报显示，到周一风速将达到每小时80公里，这增加了火势进一步扩散的风险。
 <br/>
 <br/>
 与此同时，山火已造成严重空气污染，多个城市开始向市民发放面罩，以减少污染对人体健康造成的危害。这次山火可能会成为美国加州史上最严重的野火
