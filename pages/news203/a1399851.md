### 重新计票结果不变 共和党赢2州州长竞选
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】周六（11月7日），美国两场悬而未决的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_州长竞选.html" target="_blank">
  州长竞选
 </a>
 终于出台结果，佛罗里达和乔治亚州经过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 ，确认两名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 人分别当选下届州长。
 <br/>
 <br/>
 周六，佛州民主党州长候选人吉卢姆（Andrew Gillum）承认自己败选，这场在11月6日大选夜后出现争议并一度成为关注焦点的选举终于尘埃落定。
 <br/>
 <br/>
 佛州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 州长候选人德桑蒂斯（Ron Desantis）本已在大选夜宣布胜利，但在随后的两天，由于倾向于民主党的布劳沃德县（Broward）和棕榈滩县（Palm Beach）突然多出大量选票且多数流向民主党，其民主党对手转而拒绝失败并要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 。
 <br/>
 <br/>
 重新计票从11月10日开始。周四（11月15日），机器重新计票结果显示，共和党在佛州参议员和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_州长竞选.html" target="_blank">
  州长竞选
 </a>
 中仍处于领先地位。其中，德桑蒂斯所获得的选票高出吉卢姆33,683张，估计最终
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胜选.html" target="_blank">
  胜选
 </a>
 已无悬念。
 <br/>
 <br/>
 周六晚，吉卢姆在脸书（Facebook）上承认自己失败，并祝贺德桑蒂斯当选州长。
 <br/>
 <br/>
 吉卢姆志在成为佛州第一任黑人州长，但重新计票结果，他的最后一线希望落空了。
 <br/>
 <br/>
 周六，乔治亚州州务卿根据该州州长选举的认证结果，宣布共和党候选人坎普（Brian Kemp）胜选。前一晚，民主党候选人艾布拉姆斯（Stacey Abrams）也承认自己不敌对手。
 <br/>
 <br/>
 乔治亚州重新计票的结果，坎普的得票率为50.22%，超过了半数，而艾布拉姆斯的得票率只有 48.83 %。不过，艾布拉姆斯虽然宣布结束自己的竞选，但仍对州选举规定表示不满，表示将就相关问题提起法律诉讼。
 <br/>
 <br/>
 目前中期选举结果的关注焦点集中在佛州参议员竞选上。11月15日出具的机器重新计票结果显示，共和党参议员候选人斯科特（Rick Scott）所获的选票高出民主党候选人尼尔森（Bill Nelson）12,603张。由于双方相差幅度低于0.25%，佛州已从当晚7点开始启动全州范围的人工重新计票。
 <br/>
 <br/>
 佛州参议员选举结果应在11月20日出台。
 <br/>
 <br/>
 （记者李佳欣综合报导）
