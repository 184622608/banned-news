### 彭斯提6大底线迫中共改正  APEC会议不会晤习近平
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习川会.html" target="_blank">
  习川会
 </a>
 〞月底登场，近日，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 明确提出，美中贸易谈判中方需要遵守的6条底线，并称中共只有全面改变才能避免
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冷战.html" target="_blank">
  冷战
 </a>
 。17日彭斯和习近平都将出席APCE峰会，外界预计，双方将再次激烈交锋，但有消息称，双方不会安排见面。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习川会.html" target="_blank">
  习川会
 </a>
 〞将于11月月底在阿根廷的二十国集团（G20）峰会期间登场，中美内部正就此积极准备。
 <br/>
 <br/>
 美媒报导，近日，美国商务部长罗斯（Wilbur Ross）在华府一场座谈会上证实，中共国务院副总理刘鹤即将启程前往华盛顿，希望能与美方人员磋商之后，拟订一份非正式协议。
 <br/>
 <br/>
 至于中方代表抵美的确切时间，罗斯则没有说明。
 <br/>
 <br/>
 中美展开新一轮磋商之际，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 于13日在新加坡接受《华盛顿邮报》专访时，对中共发出警告，他强调，中方必须全面改变其行为作法，才能避免与美国发生新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冷战.html" target="_blank">
  冷战
 </a>
 。
 <br/>
 <br/>
 彭斯明确指出，〝习川会〞上除了讨论中美两国贸易纠纷问题之外，中方也必须在保护知识产权、禁止强迫技术转让、取消限制市场准入、遵守国际规则、及保障国际水域航行的自由等问题上让步。
 <br/>
 <br/>
 他还警告，如果中方的提议未能令美方满意，美国已经准备好在经济、外交及政治等，多个层面上对中共施加更大压力。因美国经济强劲到可以抵御这种升级，而中共经济则不那么持久。
 <br/>
 <br/>
 彭斯此番发言，被认为是对即将赴美进行磋商的中共副总理刘鹤施加更大压力。
 <br/>
 <br/>
 彭斯在出访亚洲之前，于9日也曾罕见刊文批评，有国家（中共）在国内打压人民，对外侵犯邻国主权，美国不会容许印太地区出现威权及侵略行为。
 <br/>
 <br/>
 11月12日起，彭斯已开启了为期一周的亚洲之行。在访问日本、新加坡、澳大利亚后，他将于17日出席在巴布亚新几内亚举行亚太经济合作组织（APEC）领导人峰会。习近平也将于17日至18日出席APEC峰会。
 <br/>
 <br/>
 外界预计，即将召开的APCE峰会上，美中双方将再次激烈交锋。但也有报导称，双方不会安排见面。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
