### 入墨移民过万 川普批中美洲领导人盗用外援
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周日（11月4日）表示，一些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲.html" target="_blank">
  中美洲
 </a>
 国家领导人可能盗用了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外援.html" target="_blank">
  外援
 </a>
 ，他誓言大幅削减给这些国家的援助，以应对日益膨胀的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 大潮。
 <br/>
 <br/>
 周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在参加乔治亚州的一个竞选集会时指出，美国向洪都拉斯和萨尔瓦多等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲.html" target="_blank">
  中美洲
 </a>
 国家提供了巨额援助，但是，美国政府很快就会停止这样做了。
 <br/>
 <br/>
 川普指出，这些国家的政府并没有帮助美国阻止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 潮，因为他们不愿被这些移民咒骂。
 <br/>
 <br/>
 在数千移民大举进入墨西哥后，10月22日，川普曾在推特上表示，危地马拉、洪都拉斯和萨尔瓦多三国没有做到阻止其国民离开本国并向美国非法挺进，所以美国将开始大规模削减提供给这些国家的例行援助。
 <br/>
 <br/>
 川普表示，虽然这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民声称，他们因为惧怕本国的贫穷和暴力而向美国政府寻求庇护，但是，这种试图强行入境的行为已构成了〝入侵〞，而且其中还混杂了许多犯罪分子。
 <br/>
 <br/>
 川普说，副总统彭斯（Mike Pence）已经应他的要求给这三国的领导人打过电话，就移民问题进行交涉。
 <br/>
 <br/>
 川普指出，美国每年向这些国家提供数亿美元的援助，而这笔钱可能只是被盗用了而已，这是美国不能接受的。
 <br/>
 <br/>
 〝他们什么都没做〞，川普谴责。
 <br/>
 <br/>
 根据美国政府所报告的部分数据，2017年，美国向萨尔瓦多提供的援助至少有1.15亿美元；危地马拉接受的援助更多，至少有2.49亿美元；洪都拉斯也得到1.75亿美元的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外援.html" target="_blank">
  外援
 </a>
 。
 <br/>
 <br/>
 目前已有四批大篷车移民大规模进入墨西哥，截至周日，总计已超过12,000人。
 <br/>
 <br/>
 美国国土安全部上周四声明，向美国进发的大篷车移民来自至少20个国家，其中270余人有犯罪前科。
 <br/>
 <br/>
 川普总统上周表示将派1.5万名美军驻守美墨边境，并且宣布调整庇护政策，要求向美国政府申请庇护的移民必须在边境口岸合法地亲自递交申请。新规不再允许移民入境后提交申请即可离开，然后借等待听证的时间而在美国立足。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
</div>
