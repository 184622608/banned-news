### 川普签署法案 抵制阿片类药物滥用
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】周三，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_签署法案.html" target="_blank">
  签署法案
 </a>
 ，抵制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿片类药物.html" target="_blank">
  阿片类药物
 </a>
 滥用，并提供更广泛的预防、治疗和康复服务。这项法案获得国会两党共同支持。
 <br/>
 <br/>
 川普这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_签署法案.html" target="_blank">
  签署法案
 </a>
 ，恰逢他宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿片类药物.html" target="_blank">
  阿片类药物
 </a>
 危机称为全国突发公共卫生事件一周年。川普表示，一年来，动用整个联邦政府来应对这场危机，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芬太尼.html" target="_blank">
  芬太尼
 </a>
 查获量比他上任前两年的四倍还多。
 <br/>
 <br/>
 美国总统川普：〝过去一年，我们动员了整个联邦政府来应对这场危机。我们获得了60亿美元的新资金，用于打击阿片类药物，这是有史以来收到的最多资金。〞
 <br/>
 <br/>
 据联邦疾病防治中心统计，全美有260多万人用药成瘾，每年7万人死亡，其中芬太尼是最具危害性的阿片类药物。
 <br/>
 <br/>
 川普政府重点查缉贩毒、贩运和走私者。
 <br/>
 <br/>
 美国总统川普：〝我们正在关闭在线网络，打击国际货运，并以前所未有的方式追捕外国贩子。〞
 <br/>
 <br/>
 川普说，去年，美国移民和海关执法局和边境巡逻队，抓获了280万吨非法和致命毒品。
 <br/>
 <br/>
 川普还提到，今年8月，司法部首次起诉了两名非法贩运芬太尼到美国的中国人。
 <br/>
 <br/>
 川普政府还重点帮助年轻人，防止药物滥用。
 <br/>
 <br/>
 美国总统川普：〝我们开展了最大规模的活动，首先让美国年轻人不会上瘾。如果我们能做到，那些广告片很好，他们非常重要。〞
 <br/>
 <br/>
 美国联邦政府官员、国会议员、地方官员，边境执法人员，以及阿片类药物成瘾患者和他们的家人，聆听了川普的演讲。
 <br/>
 <br/>
 另外，沃尔玛、亚马逊、脸书、谷歌等21家美国大公司主管，也现场承诺，帮助解决阿片类药物危机。
 <br/>
 <br/>
 新唐人记者郝丽、李梅、明玉综合报导
</div>
