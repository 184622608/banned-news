### 美参议员：抓捕孟晚舟是联邦执法 与贸易战无关
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】中国华为集团副董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在加拿大被捕后，外界有舆论猜测这个行动与中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 有关。为此，白宫方面已澄清川普（特朗普）总统事前对这项行动并不知情。日前，又有美国参议院的议员发声，指此案系联邦检方正常的执法行为，与中美贸易冲突无关。
 <br/>
 <br/>
 美国参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢比奥.html" target="_blank">
  卢比奥
 </a>
 （Marco Rubio）12月6日在推特上发文称，〝华为副董
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 的逮捕与中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 无关，这是联邦检察官针对涉嫌违法的行为所采取的行动，而不是贸易纠纷中的槓杆，（美国司法机构）与中国（中共政府）的做法不同的是，在有证据证实她（孟晚舟）有罪之前，都是采取无罪推定。〞
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/12-8/p9162161a844030329.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/12-8/p9162161a844030329-ss.jpg"/>
   <br/>
   <font size="-1">
    美国参议员卢比奥（Marco Rubio）12月6日推文称，逮捕孟晚舟是联邦检察官针对涉嫌违法的行为所采取的行动，与中美贸易战无关。（网络截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 同一天，针对外界有关逮捕孟晚舟是否有政治企图的揣测，加拿大总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特鲁多.html" target="_blank">
  特鲁多
 </a>
 也在蒙特利尔的记者会上对媒体表示，逮捕华为首席财务官孟晚舟不涉及政治问题。
 <br/>
 <br/>
 〝我们是一个司法独立的国家，此案中有关机构在没有任何政治参与或干涉的情况下作出了决定。〞特鲁多说：〝我们只提前了几天收到此事在运作的通知。〞
 <br/>
 <br/>
 特鲁多还特别指出，加拿大政府尊重独立的司法程序，他本人也无意就此事件与中方进行任何直接或间接的对话。
 <br/>
 <br/>
 孟晚舟在加拿大被捕并将引渡回美国接受审判的消息传出后，多家美国媒体曾引述消息人士披露，孟晚舟是因为涉嫌利用全球银行体系逃避美国的制裁法令与伊朗进行非法交易而被逮捕，而汇丰银行发行华为公司在该银行的账号有可疑交易是这个案件的导火线。
 <br/>
 <br/>
 孟晚舟一旦被引渡回美国后，她将在纽约东区法院面临指控。
 <br/>
 <br/>
 （记者李明综合报导/责任编辑：戴明）
