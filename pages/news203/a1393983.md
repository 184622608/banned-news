### 白宫发言人：《纽时》误导性攻击 创信用新下限
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】美媒《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约时报.html" target="_blank">
  纽约时报
 </a>
 》近日发文攻击美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普），指其和兄弟姊妹们协助父母
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃税.html" target="_blank">
  逃税
 </a>
 。对此白宫发言人公开回应称，川普的父亲早已过世近20年，并炮轰《纽时》的报导是〝误导性攻击〞，其信用已达〝全新下限〞。
 <br/>
 <br/>
 白宫发言人桑德斯（Sarah Sanders）日前针对《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约时报.html" target="_blank">
  纽约时报
 </a>
 》发表的一篇攻击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 及其家人的报导作出回应。他表示，川普的父亲佛瑞德・川普（Fred Trump）过世已近20年，看到〝失败的纽约时报〞对川普家庭这种〝误导性攻击〞实在令人感到难过。
 <br/>
 <br/>
 桑德斯又进一步说明，有关川普父母向儿女们成立的公司赠送房地产的文件，都是通过了美国国家税务局（IRS）的审查的。他说：《纽约时报》等媒体的信用已达〝全新下限〞，因为他们只是致力于去攻击总统和他的家人，而不是坚持报导真相的新闻原则。
 <br/>
 <br/>
 此前，《纽约时报》曾在一篇报导中爆料，指川普与兄弟姊妹成立假公司，以接收父母数百万美元赠与的名义来设法让税务局大幅低估其房地产估值，以规避赠予税和遗产税。
 <br/>
 <br/>
 上述报导发表后，川普的律师很快便公开警告说，《纽时》的这篇报导是〝错误的报导〞，对公众起到误导的作用。舆论界也有人指出，这可能是民主党在期中选举前发起的新一轮抹黑行动。
 <br/>
 <br/>
 连中国网民也有人对《纽时》的上述报导提出质疑。
 <br/>
 <br/>
 网友sanpablo 在相关报导下留言议论说：〝这个肯定是诬陷。 川普在当选总统前，民主党政府会不查川普家人报税情况 ？正因为查不出来，导致川普顺利当选美国总统。目前《纽约时报》发表这个文章，目的是恶心川普。〞
 <br/>
 <br/>
 还有中国网民留言说：〝连范冰冰在腐败大国都因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃税.html" target="_blank">
  逃税
 </a>
 被消失，你以为美国联邦税局都是在玩吗？发这样新闻的人是对美国税务局的侮辱。〞
 <br/>
 <br/>
 (记者闻轩综合报导/责任编辑：东野）
</div>
