### 美国中西部降大雪 取消1200多航班
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 长周末，暴风雪袭击了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中西部.html" target="_blank">
  美国中西部
 </a>
 地区。导致道路关闭，电力中断，1200多架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_航班.html" target="_blank">
  航班
 </a>
 取消，约5000多航班延误 ，部分地区学校关门。
 <br/>
 <br/>
 周一早上，美国国家气象局取消了密苏里州东北部、芝加哥大都会区和密歇根州东北部的暴风雪警报，但警告说，每小时速度高达45英里的强风将会彻夜吹落积雪。
 <br/>
 <br/>
 国家气象局推文说：〝周日早上6点到今早（周一）6点，从堪萨斯州北部到芝加哥北部郊区，降雪量达8到12英寸。〞
 <br/>
 <br/>
 周一凌晨四点，芝加哥北部城市埃文斯顿警察局表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大雪.html" target="_blank">
  大雪
 </a>
 导致警察局断电两小时。
 <br/>
 <br/>
 周日晚间，国家气象局报告，堪萨斯城国际机场降雪量达5.3英寸，伊利诺伊州的罗克福德降雪至少7英寸。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_航班.html" target="_blank">
  航班
 </a>
 跟踪服务FlightAware表示，周日有1270架航班被取消。周一早上，仍有约500架往返芝加哥奥黑尔国际机场的航班取消。
 <br/>
 <br/>
 周日，堪萨斯州州长杰夫・考尔耶宣布该州进入紧急状态。
 <br/>
 <br/>
 周日，积雪导致多起交通事故。内布拉斯加州的80号州际公路等多条公路被迫关闭。
 <br/>
 <br/>
 堪萨斯州、密苏里州、爱荷华州和伊利诺伊州部分地区的学校，周一停课。
 <br/>
 <br/>
 新唐人记者毕心慈、宇微综合报导
</div>
