### 中共盗取知识产权 美国务卿：将会全面反击
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 国有企业中国电信，被爆出通过自己的网络系统，劫取加拿大和美国的互联网流量。随着越来越多中共窃取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 恶行曝光，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国务卿.html" target="_blank">
  美国务卿
 </a>
 蓬佩奥说：将会全面反击。
 <br/>
 美中贸易战之际，越来越多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 窃取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 恶行曝光。
 <br/>
 <br/>
 根据加拿大环球邮报报导，网络安全研究人员调查，中共国有企业中国电信（China Telecom）通过自己的网络系统，劫取加拿大和美国的互联网流量，以从事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍活动.html" target="_blank">
  间谍活动
 </a>
 和窃取知识产权。中国电信在2010年，也曾受到美国国会监管机构指控劫取互联网流量。
 <br/>
 <br/>
 美国国务卿蓬佩奥，31日接受英格拉汉姆（Laura Ingraham）广播节目专访时表示，从长远来看，中共可能是美国面临的最大国安挑战。
 <br/>
 <br/>
 美国国务卿 蓬佩奥(2018.10.8)：〝我遗憾我们两国之间的战略对话，你们都选择不进行。这是对我们来说都是一个重要的机会，来讨论更广的范围、重要的议题，还有我们两国人民之间存在的机会。〞
 <br/>
 <br/>
 上个月访问北京时，就提到中美双方存在许多分歧，美方非常关注北京的行动。
 <br/>
 <br/>
 专访中他还提到，在川普总统的领导下，美国正进行多方努力，准备全面反击中共政府，说服中共遵守国际法规。本周美国指控了10名试图窃取敏感数据的中国人，并将中国半导体生产商福建晋华，列为出口管制清单。
 <br/>
 <br/>
 新唐人亚太电视综合报导
