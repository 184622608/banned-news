### 应对中俄威胁 美将提早发射高超音速武器
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】为应对中俄
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高超音速武器.html" target="_blank">
  高超音速武器
 </a>
 的威胁，美国国防部正在研究方案，计划在太空部署
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_导弹感应器.html" target="_blank">
  导弹感应器
 </a>
 ，甚至拦截导弹。国防部副部长沙纳汉近日称，军方正在积极研发高超音速武器，并可能将早于预期进行发射。
 <br/>
 <br/>
 据CNBC报导，负责有关美国未来战争计划的美国国防部副部长沙纳汉（Patrick Shanahan）近日表示，美军计划提早发射，且要多发射，这可能要早于大家的预期。
 <br/>
 <br/>
 沙纳汉称，美国研发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高超音速武器.html" target="_blank">
  高超音速武器
 </a>
 的计划，将被列入申请未来国防预算的清单。他说，现在正在确定美国需要多少（高超音速武器）、如何使其更加模组化、以及如何在伴随而来的工业基础上站稳脚步。
 <br/>
 <br/>
 高超音速武器是以5马赫或更高的速度飞行的飞弹，速度至少为音速5倍，这意味着高超音速武器飞行秒速高达一英里（约1.6公里）。
 <br/>
 <br/>
 今年9月初，美国国防部负责国防研究与工程的副部长格里芬（Michael Griffin）曾表示，美国国防部需要至少200亿美元来研发高超音速武器。
 <br/>
 <br/>
 美国总统川普（特朗普）自上任以来，一直在加强美国军事力量。川普在今年6月下令五角大楼成立一支独立的太空部队，使其成为美国武装部队的一个新军种。〝这将是非常重要的事情。〞他说，美国在太空的主导地位正在受到来自中俄的挑战。
 <br/>
 <br/>
 今年3月，俄罗斯总统普京在国情咨文中，以影片和电脑模拟画面介绍新武器，包括核动力巡弋飞弹、高超音速武器及新型洲际弹道飞弹等。今年8月，中国也声称成功测试首架高超音速飞机。
 <br/>
 <br/>
 美国对中俄两国高超音速武器快速发展感到忧虑。军事专家表示，不同于传统弹道导弹的飞行轨迹，高超音速滑翔导弹可以飞入太空、再返回地球大气层内，美国目前的导弹防御雷达无法侦测这类武器。
 <br/>
 <br/>
 美国国防部认为，只有在太空中部署感应器才能够有效侦测到这类威胁。军方不仅需要发展新武器时代的美国军火库，而且需要制定解决方案，来防御敌方以前所未有的速度发射的导弹。
 <br/>
 <br/>
 美国2018国防授权法要求美国国防部研究在太空建立导弹感应系统和拦截系统的可能选项。
 <br/>
 <br/>
 根据白宫的资料，美军军方将获得3%的资金增加，意味着军队将在2019年获得6740亿美元的支持，使美国保持其在海陆空3方面的优势。
 <br/>
 <br/>
 2019年，美军军火库将添加200多架新的战机，130多辆升级版坦克，以及十几艘新型海军战舰。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
