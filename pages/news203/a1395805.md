### 生鸡肉含沙门氏菌 全美近百人感染 纽约上榜
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】美国疾病控制和预防中心周三发布警告，在全美29个州销售的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生鸡肉.html" target="_blank">
  生鸡肉
 </a>
 中，发现有抗药性
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙门氏菌.html" target="_blank">
  沙门氏菌
 </a>
 。目前，已有近百人感染病菌，多人住院。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 州也发现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感染病例.html" target="_blank">
  感染病例
 </a>
 。
 <br/>
 <br/>
 美国疾病控制和预防中心（US Centers for Disease Control and Prevention, CDC）表示，含有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙门氏菌.html" target="_blank">
  沙门氏菌
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生鸡肉.html" target="_blank">
  生鸡肉
 </a>
 涉及不同类型和品牌，无法锁定某一个供应商，而且发现地点也比较分散，遍布全美29个州。
 <br/>
 <br/>
 目前，已有至少92个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感染病例.html" target="_blank">
  感染病例
 </a>
 ，21人住院治疗，尚未发生死亡案例。其中感染病例最多是宾州11人，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 州10人，新泽西州9人。
 <br/>
 <br/>
 在检验中，还发现用生鸡肉制成的猫食、生鸡肉产品，以及活鸡身上，都有感染沙门氏菌案例。而且病菌有抵抗多种抗菌药物的能力。
 <br/>
 <br/>
 感染病菌后，在12到72小时之内病情会发作，有胃痛、抽筋、腹泻、发烧等症状。不进行治疗，一般四到七天会有好转。如果病情加重，要入院治疗。一般5岁以下小孩和65岁以上老人对这种病菌免疫力最低。
 <br/>
 <br/>
 美国疾病控制和预防中心提醒民众，处理生鸡肉后，要即时把手和厨具清洗干净，防止病菌传播。一定要把鸡肉做熟后再吃，烤箱温度至少要165华氏度。不要给猫、狗等宠物吃生鸡肉做成的食物，这会导致宠物和人一起感染生病。
 <br/>
 <br/>
 新唐人记者安心纽约报导
</div>
