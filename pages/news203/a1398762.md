### 川普答记者：未与代理司法部长谈〝通俄〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）与第一夫人启程前往巴黎，参加周末在法国举办的一战停战100周年纪念活动。临行前，川普对媒体说，没有与代理司法部长谈论〝通俄门〞问题。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：〝我没有和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_惠特克.html" target="_blank">
  惠特克
 </a>
 讨论（通俄门）。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞申斯.html" target="_blank">
  塞申斯
 </a>
 在中期选举后辞去司法部长，而换上公开批评通俄门的惠特克(Matt Whitaker)做代理部长，令民主党怀疑这与通俄门调查有关。同时，一些州的检察长正施压惠特克，要他像其前任一样，回避通俄调查。
 <br/>
 <br/>
 美国总统川普：〝惠特克有很棒的声誉，这正是我想要的。〞
 <br/>
 <br/>
 另一职位，也引起关注。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑利.html" target="_blank">
  黑利
 </a>
 大使曾表示她将在年底辞职。川普说他有意让国务院发言人诺尔特接任联合国大使，也考虑多位人选，会在接下来的几个星期做出决定。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超大基石油管.html" target="_blank">
  超大基石油管
 </a>
 建成后，将方便加拿大的原油运入美国，川普政府认为该项目会创造就业并降低油价。
 <br/>
 <br/>
 一个有关美国能源基础设施，48000人工作机会的项目也引起关注。
 <br/>
 <br/>
 11月9日，川普批评法官决定暂停建造Keystone XL石油管道建设项目，该项目将重质原油从加拿大运往美国。2017年3月，川普批准Keystone XL项目，该项目曾被奥巴马否决，并长期遭到环保人士的反对。
 <br/>
 <br/>
 该项目为加拿大TransCanada Corp公司的石油管道项目，将建造一条把加拿大阿尔伯塔（Alberta）省生产的重油运送美国的输油管道，总长1897公里、总投资61亿美元。川普称这个项目可以降低消费燃油价格、创造就业、同时降低美国对进口石油的依赖。
 <br/>
 <br/>
 在川普批准该项目后不久，几个环保团体和牧场主提起诉讼。
</div>
