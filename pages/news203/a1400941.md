### 前美ICE局长：国会应对大篷车问题负责
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】周一（11月26日），前美国移民和海关执法局（ICE）代理局长霍曼（Thomas Homan）批评指出，由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国会.html" target="_blank">
  国会
 </a>
 不作为，造成了数千名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民威胁美国边境的局面。
 <br/>
 <br/>
 霍曼周一在接受福克斯新闻（Fox News）采访时指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国会.html" target="_blank">
  国会
 </a>
 对于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 压境的局面负有责任。如果国会早前弥补有关移民政策的《佛洛雷斯协议》（Flores settlement agreement）中的漏洞，今天的大篷车现象就不会出现。
 <br/>
 <br/>
 霍曼讽刺民主党参议员艾米．克罗布彻（Amy Klobuchar），当她批评某人造成了当前局面时，需要先照照镜子反观自己。
 <br/>
 <br/>
 霍曼抱怨说，他只听到有人谈论问题，但是，解决措施在哪里？
 <br/>
 <br/>
 《佛洛雷斯协议》和《2008年威廉．维波福士保护人口贩卖受害者再授权法案》（William Wilberforce Trafficking Victims Protection Reauthorization Act of 2008）规定，来自墨西哥和加拿大以外国家的家庭及无人陪伴的未成年人如果非法进入美国，不能被拒绝入境或逮捕，且必须被允许申请庇护。
 <br/>
 <br/>
 根据这项法案，大多数偷渡的家庭和儿童在被抓后都被释放，并被告知将来按照庇护申请赴移民法庭接受听证。但实际上，这些人多数都没有在听证会上露面，并在美国境内消失了踪迹。
 <br/>
 <br/>
 此类庇护案件有4/5最后都会被驳回，因为它们并不符合庇护标准。
 <br/>
 <br/>
 霍曼指出，民主党呼吁取缔ICE，且有更多城市不与ICE合作抓捕非法入境的犯罪分子，这让那些试图非法进入美国的人更有理由如此去做，因为他们会受到美国执法部门的保护。
 <br/>
 <br/>
 霍曼称赞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统的处理方式。川普周一上午警告，如果墨西哥不驱逐那些聚集在美墨边境的中美洲移民，他可能会永久性地关闭美国南部边界。
 <br/>
 <br/>
 霍曼说，川普的提议〝绝对完美〞，因为墨西哥需要采取行动。关闭边境给墨西哥造成的伤害会远远大于给美国造成的伤害。
 <br/>
 <br/>
 （记者李佳欣编译报导/责任编辑：程以仁）
</div>
