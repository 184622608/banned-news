### 加州预计11月底扑灭大火 已知77死失踪993人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】截至周一（19日）中午，美国北加州史上最致命
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎普野火.html" target="_blank">
  坎普野火
 </a>
 已至77人死亡。有关当局仍在废墟中寻找993名失踪者的踪迹。预计本周将有大的降雨，专家担忧恐引发泥石流。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎普野火.html" target="_blank">
  坎普野火
 </a>
 （Camp Fire）肆虐山城天堂镇，至少夺走77条人命。巴特郡（Butte）警长办公室周一在声明中称，失踪人数从原本1276人，下降至993人。但声明中没有透露其他细节。
 <br/>
 <br/>
 巴特郡警局表示，失踪人员名单是根据火灾引发混乱后的第一时间，从特殊热线、电子邮件及紧急号码911收到的讯息汇编。部分名列失踪名单者可能幸免于难，但可能因他们缺乏电信服务或不知道有人在寻找他们，而没有通知家人或官员。
 <br/>
 <br/>
 天堂镇远离都会区，隐蔽在山林之间，是加州人退休后首选的居住地之一，拥有27,000居民。然而，美丽舒适的家园几乎被大火夷为平地。
 <br/>
 <br/>
 到周一，〝坎普〞野火已燃烧了12天，过火面积超过15万英亩，已有66%的火场得到控制，当局预计将于11月底彻底扑灭大火。目前，起火原因还在调查中。
 <br/>
 <br/>
 除了人员伤亡外，烈火也导致加州史上最惨重财产损失，至少1万3600栋房屋等建筑遭烧毁，数以千计流离失所灾民需要长期避难处所。许多灾民暂居亲友家中，其他人则搭起帐篷或露宿车内。
 <br/>
 <br/>
 因雨季来临，预计该地区本周将有降雨，这对灭火有很大的帮助，但因植被被烧毁，专家担忧会发生洪水和土石流，这将使接获疏散命令的4.6万人更加痛苦。
 <br/>
 <br/>
 而在南加洛杉矶郊外，伍尔西大火（Woolsey Fire）夺去至少3条性命。已有82%的火势得到了控制，大火燃烧了9万8362英亩的土地。
 <br/>
 <br/>
 预计当地本周也有大的降雨，当局在南加州文图拉县、洛杉矶县等〝伍尔西大火〞燃烧区域发布了土石流预警，提醒土石流可能形成新的灾难。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
