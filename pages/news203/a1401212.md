### 通用减本土制造增对华投资 川普或加汽车税
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通用汽车.html" target="_blank">
  通用汽车
 </a>
 本周一宣布关闭北美5家工厂，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 对此相当不满，周三他在推文中表示，正在研究新的汽车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，保护美国制造业。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通用汽车.html" target="_blank">
  通用汽车
 </a>
 宣布在北美关闭的5家工厂，4座分布在俄亥俄、密歇根和马里兰州，1家在加拿大安省奥沙瓦（Oshawa）；裁员人数大约1万5千人。但同时，该公司将扩大在中国生产电动车及对墨西哥增加投资。
 <br/>
 <br/>
 白宫经济顾问 库德洛：〝非常的失望。通用更愿意在中国制造那些特定车型而不是在美国，这让人很失望。我们正在考虑对电动车及其他车型的补贴是否适合。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 对通用的做法一再表达不满，并公开施压通用CEO巴拉（Mary Barra）要求其研发适合的车型在俄亥俄州开设新厂房，并停止在中国制造汽车。
 <br/>
 <br/>
 中国问题专家、专栏作家章家敦（Gordon Chang）：〝通用汽车（的做法）是对商业环境及贸易战的一种回应。我觉得川普总统要求通用只在美国设厂不太现实，重点是川普要能确保通用在中国设厂不会得到激励。这也正是目前川普政府所做的，在中国制造的汽车进口美国时要有相应的条件。〞
 <br/>
 <br/>
 目前通用汽车在中国年产销约四百万辆汽车，占其全球产销的40%；去年该公司在中国的利润大约为20亿美元。
 <br/>
 <br/>
 有分析师认为在全球化背景下，靠近客户端建厂有利于盈利并可就近取材、降低物流成本。
 <br/>
 <br/>
 另外，自从美国加征钢铝进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 并基于〝301条款〞调查报告对中国实施关税制裁以来，中国对通用等美国产进口汽车也上调关税至40%。专家认为这也是通用做出此决定的原因。今年较早前通用下调了2018年盈利预测。
 <br/>
 <br/>
 不过，川普在周三的推文中表示，小型卡车之所以在美国受青睐，是因为施加了25%的进口关税。已经开始调研是否要对所有进口车都设置相应的关税。
 <br/>
 <br/>
 目前，美国进口汽车关税为2.5%。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
