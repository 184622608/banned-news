### 律师：美国公民入籍等候延长 纽约近2年
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】美国移民及公民服务局近日表示，部分州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入籍.html" target="_blank">
  入籍
 </a>
 申请，等待时间最长可达两年，远远超过平均6个月的等待时间。那么，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约州.html" target="_blank">
  纽约州
 </a>
 的情况怎样呢？来听一听律师的分析。
 <br/>
 <br/>
 根据美国移民及公民服务局（USCIS）的数据，在奥巴马政府期间，等候裁决的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入籍.html" target="_blank">
  入籍
 </a>
 申请案件，积压量增加了一倍以上，从2010年9月的29万个增加到2017年初的将近70万个。
 <br/>
 <br/>
 美国各地入籍申请的平均等待时间超过10个月，其中，亚特兰大长达22个月，德州部份地区更是长达26个月。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民律师.html" target="_blank">
  移民律师
 </a>
 张德海：〝政府可能把一些人力资源抽调到其他方面，比如以前已经行政关闭的案子，政府要考虑是不是把一些案子重新打开，这是一个原因。还有一个原因就是现在公民入籍是从严把关，所以以前没有去审查的东西，现在是审查的比以前要更仔细，这也是一个原因。〞
 <br/>
 <br/>
 那么，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约州.html" target="_blank">
  纽约州
 </a>
 目前的入籍等候时间需要多久？
 <br/>
 <br/>
 美国移民律师 张德海：〝纽约州目前平均处理公民入籍的时间是，根据移民局的网站，公布出来的数据是11.5个月到22个月，差不多要接近两年的时间。〞
 <br/>
 <br/>
 张德海律师提醒，等待入籍期间有一些事项需要注意。
 <br/>
 <br/>
 美国移民律师 张德海：〝有一点非常重要，不能有犯罪纪录。如果要出境旅游，这个时候比如说是申请绿卡的话，要确认回美纸是不是还有效？另外，出境旅游之前一定要跟自己的律师沟通好，确认一下。这样回到美国的时候不会有任何的问题。〞
 <br/>
 <br/>
 总统川普上任后，2017财年共有超过76万个移民入籍成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国公民.html" target="_blank">
  美国公民
 </a>
 ，是2013年以来的最大规模。其中，来自加州、纽约州、佛罗里达州和德州入籍的人数最多。
 <br/>
 <br/>
 新唐人记者柯婷婷、韩瑞纽约报导
</div>
