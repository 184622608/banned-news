### 谈判不确定性盖过乐观情绪 美媒:川普将持强硬立场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】川习会后，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 已指定强硬派人物莱特希泽主导美中90天谈判，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 也将参与。《华尔街日报》5日消息，川普将在谈判中采取强硬立场，很可能继续升级关税，谈判结果的不确定性已经盖过了川习会刚结束时的乐观情绪。
 <br/>
 <br/>
 报导还说，事实上，在新一轮谈判中，美方的首席谈判代表将是美国贸易代表莱特希泽，而他是白宫鹰派的主将之一。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 还在4日的推文中透露，此次美中谈判，莱特希泽将与财政部长姆努钦、白宫经济顾问库德洛和贸易顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 密切合作。
 <br/>
 <br/>
 这显示，最不受中共欢迎的超级强硬派纳瓦罗也将参加谈判。此前，纳瓦罗出席阿根廷川习会的消息曾令股市下跌。
 <br/>
 <br/>
 除此之外，川普和白宫高层官员已连日喊话施压，展示美方未来90天将采取的强硬态度。
 <br/>
 <br/>
 4日一早，川普在推特上表示，他〝预计〞中方将立即开始购买更多美国农产品，并同时展示强硬姿态说，〝我是一个关税人，当有人或国家来掠夺我国的巨大财富时，我将让他们为此付出代价。〞
 <br/>
 <br/>
 同一天，白宫安全顾问博尔顿呼吁立法，以禁止窃取美国知识产权所生产的商品和服务进入美国。
 <br/>
 <br/>
 纳瓦罗则对美媒强调北京领导人习近平在川习会上亲自做出的各项承诺，督促北京兑现承诺之意明显。
 <br/>
 <br/>
 就连对北京态度相对温和的姆努钦和库德洛，表态也越来越严厉。库德洛说，〝我希望我们达成协议，但必须是好的协议，而且要能核实。〞姆努钦则质疑，〝90天结束时会达成一项真正的协议吗？〞
 <br/>
 <br/>
 不过，在对北京密集施压的同时，白宫也表示对达成协议持开放态度。
 <br/>
 <br/>
 川普4日宣布90天谈判期从12月1日开始计算，3月1日结束。但他也提到〝延长期限〞的可能性。
 <br/>
 <br/>
 姆努钦也对福克斯（FOX）商业新闻网称，〝这将是一个真正的协议，不是说我们可以在90天内完成所有事情，而是我们希望取得很大进展。〞
 <br/>
 <br/>
 观察人士分析，白宫表态显示了川普政府的两手准备：这90天期限可能是用来判断北京诚意，如果发现中共耍诈，短短90天的休战不会对美方部署造成实质影响；如果发现北京确有诚意，美方可能延长谈判，以真正达成协议。
 <br/>
 <br/>
 不过，许多分析人士认为，北京如果真的进行结构性经济改革，会动摇中共统治根基，因此中共90天内不大可能做出实质让步。
 <br/>
 <br/>
 自由亚洲电台12月3日引述中共对外贸易部门一位匿名官员宣称，美方〝需要调整谈判内容〞，别指望中共做出〝影响社会制度〞的结构性改革。他说，中共在贸易领域已经〝无路可走〞，美方不能把北京逼入〝死胡同〞。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
