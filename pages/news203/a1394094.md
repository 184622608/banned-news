### 〝摇摆〞参议员赞FBI调查 或投票支持卡瓦诺
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】美国最高法院大法官提名人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 （Brett Kavanaugh）的性侵指控调查已经结束，周四（10月4日），几名被视为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摇摆票.html" target="_blank">
  摇摆票
 </a>
 的共和党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议员.html" target="_blank">
  参议员
 </a>
 对联邦调查局（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 ）的调查结果表示满意，暗示他们可能会在确认表决时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支持.html" target="_blank">
  支持
 </a>
 卡瓦诺。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 的性侵指控调查周三完成，白宫随后把调查结果复印一份送交国会。
 <br/>
 <br/>
 周四，此前一直没有表态的缅因州共和党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议员.html" target="_blank">
  参议员
 </a>
 柯林斯（Susan Collins）表示，在阅览FBI的报告后，她认为这〝似乎是一次非常彻底的调查〞。
 <br/>
 <br/>
 推动再次调查卡瓦诺的关键人物、亚利桑那州共和党参议员弗莱克（Jeff Flake）对FBI的调查也比较满意，他表示，FBI并没发现〝新的确凿信息〞。
 <br/>
 <br/>
 不过人们对弗莱克的立场仍有猜测，因为他在卡瓦诺听证会后一度翻来覆去上演〝戏剧〞，而且在周四的表态中，他补充说他还要进一步阅览资料。
 <br/>
 <br/>
 上周五，弗莱克先是声明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支持.html" target="_blank">
  支持
 </a>
 卡瓦诺，但在自由派的压力下，很快又要求FBI进行至少为期一周的调查，并表示只有这样他才能〝自在地〞投票表决。
 <br/>
 <br/>
 阿拉斯加州共和党参议员穆考斯基（Lisa Murkowski）周四表示，她还没有做出决定。但她做出这一表述时尚未看到FBI的调查结果，她说她将尽快返回国会查看报告。
 <br/>
 <br/>
 北达科他州民主党参议员海特坎普（Heidi Heitkamp）则宣布投反对票。正在竞选连任的她因为卡瓦诺问题的冲击，支持率已急剧下降，目前落后于其共和党对手10个百分点。不知她做出这一决定是否是因为她看到获胜的希望大势已去、索性站队于自己的党派，以免面临更大压力。
 <br/>
 <br/>
 北达科他州是支持共和党的红州。2017年4月，海特坎普在川普提名的第一位保守派大法官戈萨奇（Neil Gorsuch）确认表决时投了赞成票。
 <br/>
 <br/>
 另外，处于红州的西维吉尼亚州民主党参议员曼钦（Joe Manchin）的立场尚不清楚，不过，去年他给戈萨奇投了支持票。
 <br/>
 <br/>
 还有消息人士称，白宫正试图说服印第安纳州民主党参议员唐纳利（Joe Donnelly）放弃反对立场。他正在红州竞选连任，去年，他也给戈萨奇投了赞成票。
 <br/>
 <br/>
 卡瓦诺要获得确认，必须得到51票支持。目前参议院两党的席位之比是51:49，共和党只占微弱优势。当然，如果投票持平，副总统彭斯（Mike Pence）将以参议院议长的角色投下决定性一票。
 <br/>
 <br/>
 参议院已决定周五进行程序性投票，周六进行全院表决。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
</div>
