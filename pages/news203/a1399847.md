### 阔别三年 百年玩具店纽约重新开张
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】有156年历史的著名玩具店FAO施瓦茨纽约曼哈顿店，2015年关闭，周五重新开张。新店里有哪些新鲜玩具？一起去看看。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_百年玩具店.html" target="_blank">
  百年玩具店
 </a>
 FAO施瓦茨在阔别三年后，新的旗舰店周五在曼哈顿洛克菲勒中心开业。
 <br/>
 <br/>
 超大型填充动物玩偶、互动游戏区、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三层钟塔.html" target="_blank">
  三层钟塔
 </a>
 ，2万平方英尺店面挤满了来自北美各地的顾客。
 <br/>
 <br/>
 除了电影《飞进未来(Big)》中汤姆‧汉克斯与罗伯特‧劳吉亚跳舞的钢琴键盘，新店还升级了〝小熊工作室〞,〝探索科学实验室〞, 和〝马文魔术间〞。
 <br/>
 <br/>
 还有不少扮演成玩具角色的雇员为大小朋友们服务。
 <br/>
 <br/>
 除了时下热门的高科技玩具，百年老店也特意为那些传统玩具保留了空间。孩子可以和父母亲人一起，用玩具构筑属于他们的世界。
 <br/>
 <br/>
 FAO施瓦茨德国移民施瓦茨在是1862年创立的，此前是玩具反斗城的家庭成员。
 <br/>
 <br/>
 2015年，由于租金不断上涨，FAO施瓦茨关闭了位于曼哈顿第五大道中央公园前的6万平方英尺店铺；后被ThreeSixty买下成为旗下品牌。
 <br/>
 <br/>
 新唐人记者李兰、尚敬综合报导
</div>
