### 加州野火累计达66死 避难所爆发疫情
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】南北加州两场山火持续燃烧了一周，截至周四（11月15日）死亡人数已增至66人，失踪人数更跃升至631人。雪上加霜的是，一些撤离者又面临新的威胁，避难所爆发疫情，病患被隔离。
 <br/>
 <br/>
 北加州燃烧了近一周的〝坎普野火〞继续肆虐，消防人员发现另外7具遗体，当地死亡人数增加到63人。而南加州的〝伍尔西大火〞则造成3人死亡，整个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州山火.html" target="_blank">
  加州山火
 </a>
 死亡人数上升到66人。目前，5.2万人处于撤离状态。
 <br/>
 <br/>
 当地卫生官员表示，其中一些撤离者面临新的威胁。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴特郡.html" target="_blank">
  巴特郡
 </a>
 卫生部证实，避难地点奇科（Chico）爆发诺如病毒（Norovirus）疫情，出现肠胃炎症状的病患被隔离。
 <br/>
 <br/>
 他们证实，周四奇科（Chico）邻近教堂的一个庇护所有179人避难，其中有21人感染诺如病毒，需要隔离。诺如病毒是一种传染性病毒，可引起严重的呕吐和腹泻。

 据巴特郡警局与加州森林防火厅(Cal Fire)发布的消息，北加州这场野火累计已烧毁1万1862栋建筑，其中至少9700栋是独栋屋(single family home)。
 <br/>
 <br/>
 巴特郡（Butte County）警长哈尼亚（Kory Honea）说，新发现7具遗体，其中天堂镇（Paradise）有3具，在马格利亚（Magalia）发现另外3具，而还有一具则在Concow发现。
 <br/>
 <br/>
 当地官员说，北加州地区仍然下落不明的人数已增加到631人，约有52,000人住进了避难所。
 <br/>
 <br/>
 哈尼亚15日晚在记者会上表示，以义工为主的400多位搜救人员正在重灾区天堂镇清查约1万栋被毁房屋，寻找未被发现的遇难者，失踪名单已经公布在巴特郡警局官网，并持续更新中。
 <br/>
 <br/>
 哈尼亚也提醒居民上到警局官网看失踪者名单，确定其中是否有自己的名字。他说，〝这是一个随时变化的名单，有时人数会增加，有时会减少。〞失踪名单上部分人可能已经逃离火场，但也可能〝很多人不知道我们正在寻找他们。〞
 <br/>
 <br/>
 哈尼亚前几天向全加州征求资源与人员协助后，超过450位加州搜救队员前往天堂镇支援，这些人来自全加州所有58个郡。加州州法规定每个郡的警长都必须维持一组由志愿者组成的搜救队，现在看来这种制度确实在关键时刻发挥效用。
 <br/>
 <br/>
 加州州长紧急服务办公室的搜救协调官Ben Ho表示，这些义务搜救队员男女都有，来自不同年龄层，〝是地方社区的缩影〞。
 <br/>
 <br/>
 加州森林防火厅一位官员15日在记者会上透露，调查人员发现可能有第二个起火地点，但原因仍在调查中，但他没有公开任何资讯。

 截至周四晚，加州林业和消防部门（Cal Fire）表示，〝坎普野火〞的火势已有40%得到控制。〝伍尔西大火〞有57%的火势得到控制。
 <br/>
 <br/>
 〝坎普山火〞大部分死者和失踪者都是居住在重灾区天堂镇所在的巴特郡，以年长者居多，估计他们可能因为没能迅速逃离火场或受困于汽车内而遇难。
 <br/>
 <br/>
 从上周四（8日）开始燃烧的〝坎普野火〞已经成为加州史上最致命的山火，也是加州史上规模最大的失踪人口搜索行动，是911事件后全美规模数一数二的搜索行动。
