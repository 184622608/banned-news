### 黑色星期五将至 购物季消费额或创纪录
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑色星期五.html" target="_blank">
  黑色星期五
 </a>
 即将到来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络星期一.html" target="_blank">
  网络星期一
 </a>
 紧随其后，一连数天都是购物黄金日。之后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞.html" target="_blank">
  圣诞
 </a>
 月购物狂欢正式登场。
 <br/>
 <br/>
 美国经济持续景气，消费者信心指数达到18年新高，失业率为3.7%，为49年来最低。不同的预测显示，今年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑色星期五.html" target="_blank">
  黑色星期五
 </a>
 的消费水平也将创纪录，每人平均花费将达到800到1000美元。从11月1日到12月31日，消费额将有史以来第一次超过一万亿美元。
 <br/>
 <br/>
 网上消费预计增长15%，其中有40%将使用手机购买。
 <br/>
 <br/>
 美国购物狂潮即将开始，商家会推出很多打折促销活动。但购物前仔细调研才更能找到物美价廉的商品。
 <br/>
 <br/>
 例如，调查显示，近一半网购者希望可以免运费，但像梅西百货可能只给购物超99美元的消费者免运费，否则要收10.95美元。
 <br/>
 <br/>
 另外，有些商品，黑色星期五找不到便宜货，但在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞.html" target="_blank">
  圣诞
 </a>
 节前会更优惠。比如玩具，圣诞节前一周价格可能会下降一半。其他如冬季服装或床上用品，一般是1月份才能找到更便宜的货品。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
