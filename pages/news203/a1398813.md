### 马克龙吁建欧洲部队 川普：侮辱美国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 日前呼吁欧洲应组建一支联合部队，防范俄、中甚至美国，对此，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 9日抵达法国后，在推特贴文表示说：这是〝非常侮辱〞的言论。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 (Donald Trump另译特朗普)今天偕同妻子梅拉尼娅（Melania Trump）搭机前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 ，准备参加11日在巴黎举办的第一次世界大战结束百年纪念仪式。此活动预计全球有70个国家领袖将出席，包括俄罗斯总统普京。
 <br/>
 <br/>
 川普专机降落巴黎后，即在推特贴文说：〝法国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 建议欧洲建立军队保护自己，以防范美国、中国和俄国。非常侮辱，欧洲或许应先支付他们的北大西洋公约组织（NATO）费用，这部分美国已资助了太多！〞
 <br/>
 <br/>
 马克龙办公室不愿就川普推文置评，两人已排定在10日进行双边会谈及共进午餐。
 <br/>
 <br/>
 马克龙6日曾告诉法国的欧洲第一电台（Europe 1）：面对重新崛起的俄国，欧洲需要一支真正的军队，以减少对于美国的防卫依赖。而〝当涉及中国、俄国，甚至是美国时，我们必须保护自己。〞
 <br/>
 <br/>
 马克龙表示，假如我们决定不要拥有一支真正的欧洲军队，我们就无法保护欧洲民众。
 <br/>
 <br/>
 另外，德国总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨克尔.html" target="_blank">
  墨克尔
 </a>
 也曾公开表达疑虑，忧心欧洲还能否在发生战事时仰赖美国协助。
 <br/>
 <br/>
 马克龙的言论获得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 执行委员会（European Commission）高层呼应。路透社报导，欧盟执委会主席荣科（Jean-Claude Juncker）长期支持欧盟应拥有更多共同防御能力。
 <br/>
 <br/>
 欧盟执委会首席发言人席纳斯（Margaritis Schinas）表示，欧盟正努力在国防采购和研究上进行合作，以及发展欧盟军事维和能力。
 <br/>
 <br/>
 彭博（Bloomberg）表示，川普已多次要求北约国家多负担自身的防卫，甚至质疑美国是否还需留在北约。不过，白宫国家安全顾问博尔顿曾称，华府关注欧洲的稳定，川普亦无意退出全球事务。
 <br/>
 <br/>
 川普下周将出席在巴黎凯旋门的纪念活动，但不会参与同日举行的巴黎和平论坛，亦不会在法国与普京举行双边会谈。
 <br/>
 <br/>
 (责任编辑：程以仁)
</div>
