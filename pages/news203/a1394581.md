### 〝空军一号〞上会川普 罗森斯坦官位保住
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】周一，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）在启程去国际警察局长协会年会之前，接受媒体短暂采访，一起来看一下。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国司法部.html" target="_blank">
  美国司法部
 </a>
 二号人物
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗森斯坦.html" target="_blank">
  罗森斯坦
 </a>
 ，周一随
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 搭乘〝空军一号〞专机，前往佛罗里达奥兰多市，参加国际警察局长协会年会。
 <br/>
 <br/>
 启程前，记者问川普是否会开除罗森斯坦。
 <br/>
 <br/>
 美国总统川普：〝不会，我不会。〞
 <br/>
 <br/>
 9月21日，《纽约时报》报导，罗森斯坦曾在内部谈话中，提出让FBI局长候选人偷录与川普的对话，以便找出川普说话的漏洞，赶他下台。媒体曾一度传言罗森斯坦乌纱帽不保，但罗森斯坦否定了这篇报导。
 <br/>
 <br/>
 周一在空军一号上，川普和罗森斯坦进行了约半小时的谈话。川普后来表示，感谢罗森斯坦同行，并且两人谈得非常好。
 <br/>
 <br/>
 周一答记者问时，川普还谈到了新任最高法院大法官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 。
 <br/>
 <br/>
 卡瓦诺就任前受到了充满争议的性侵指控，对他的表决过程也不断遭受一些左派团体的侵扰。
 <br/>
 <br/>
 美国总统川普：〝他们(民主党人)是在折磨他（卡瓦诺）和他的家人。我想，这是我见过的最可耻的表演之一。〞
 <br/>
 <br/>
 川普认为，人们已经从卡瓦诺的任命过程中看到了民主党人的不诚实，而这会影响到11月6日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 。
 <br/>
 <br/>
 美国总统川普：〝民主党的基本盘已经变得太左了，（如果民主党当权）我们最终将成为委内瑞拉。我想，很多民主党选民将在11月6日投共和党的票。〞
 <br/>
 <br/>
 10月6日，卡瓦诺获参议院表决通过，成功就任大法官。周一晚间，川普还为他举行入职仪式，而后卡瓦诺将于周二正式在最高法院开始工作。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
