### 弗林供出川普？白宫否定有证据指向通俄
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】周二，美国〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 〞特别检察官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 的团队表示，可能不会对前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 国家安全顾问迈克尔．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弗林.html" target="_blank">
  弗林
 </a>
 ，提出监禁指控。弗林曾因在俄罗斯问题上向FBI撒谎，而辞去白宫职务。
 <br/>
 <br/>
 周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 团队发表备忘录，建议法官不判
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弗林.html" target="_blank">
  弗林
 </a>
 入狱。
 <br/>
 <br/>
 穆勒团队表示，他们与弗林进行过19次闻讯，弗林提供了大量资讯，包括〝（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ）过渡团队和俄罗斯政府官员接触的内容和背景的第一手信息〞。
 <br/>
 <br/>
 但是，这份备忘录对外界高度保密，只有个别字句露了出来，大部分被黑色条块盖住。
 <br/>
 <br/>
 川普总统律师朱利安尼（Rudolph Giuliani）周二晚向媒体表示，他已看过全部备忘录，根本没有任何内容暗示川普与俄罗斯官员有共谋，并暗讽穆勒团队的声明是受到别有用心的媒体鼓动。朱利安尼指出，整个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查就是想拉川普下台的左派策略。
 <br/>
 <br/>
 川普总统在12月3日的推文中称赞前顾问史东（Riger Stone），拒绝配合他所认为会对抗川普的调查取证行动。川普说：史东的声明表示他不会被失控的检察官逼迫去制造谎言，真高兴知道还有这样有〝胆识〞的人。
 <br/>
 <br/>
 接下来，弗林将于12月18日接受审判。
 <br/>
 <br/>
