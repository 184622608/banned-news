### 佛州或临13年来最强飓风 川普发文预警
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】周二（10月9日），2级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 （Michael）正迅速增强，预计周三将登陆美国佛罗里达州。美国气象部门、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 和联邦政府都敦促当地居民做好防范准备，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统也嘱咐民众听从当地官员指挥，保证人身安全。
 <br/>
 <br/>
 据美国国家气象频道（The Weather Channel）9日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 正在迅速增强，预计周三将达到至少3级，并在佛罗里达狭长地带（Florida Panhandle，也称西北佛罗里达或西佛罗里达）登陆。
 <br/>
 <br/>
 根据气象预测，迈克尔登陆时，将带来危及生命的破坏性风暴潮和强降雨。迈克尔向内陆推进后，美国东南海岸的其它地区也将面临大雨和强风侵袭。
 <br/>
 <br/>
 周一下午，位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 塔拉哈西（Tallahassee）的国家气象局（National Weather Service）办公室在其地区预测中表示，迈克尔飓风可能会发展成为墨西哥湾东北部海岸的灾难性事件。迈克尔还可能成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_13年.html" target="_blank">
  13年
 </a>
 来在墨西哥湾沿岸佛罗里达西北登陆的最强飓风。
 <br/>
 <br/>
 根据气象频道目前的预测，迈克尔飓风最有可能在周三中午前后的几小时内登陆，登陆地点可能在佛罗里达狭长地带和大弯（Big Bend）地区之间。
 <br/>
 <br/>
 最早从周二晚间起，墨西哥湾东北部的天气将出现恶化。
 <br/>
 <br/>
 登陆后，迈克尔会减速并向美国东南部的内陆地区推进。周三晚至周四晚，将出现疾风和大雨。
 <br/>
 <br/>
 周日及周一两天，佛州州长斯科特（Rick Scott）向总计35个县发布了紧急状态令，要求沿海低洼地区的居民撤离。
 <br/>
 <br/>
 周一，斯科特再次要求当地居民做好最后的防范工作，称迈克尔是极具破坏潜力的〝怪兽风暴〞（monstrous storm），并因支持居民撤离而取消了佛州的公路通行费。
 <br/>
 <br/>
 美东时间周二下午5点（北美中部夏令时间下午4点），美国国家飓风中心（National Hurricane Center）官方推特公布的最新情况显示，佛州西北部海岸、大弯和自然海岸（Nature Coast）的部分地区将面临危及生命的风暴潮。
 <br/>
 <br/>
 另外，迈克尔带来的强降雨可能会引发威胁生命的暴洪，影响所及地区从佛罗里达狭长地带和大弯地区延伸到乔治亚州、南北卡罗来纳州的部分地区和弗吉尼亚州东南部地区。
 <br/>
 <br/>
 还有，从佛州东北部到北卡州的美国东南海岸的部分地区，都将受到热带风暴侵袭。
 <br/>
