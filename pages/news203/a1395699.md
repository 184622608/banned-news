### 飓风迈克尔袭美 至少30人遇难千人失踪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 上周重创
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国东南部.html" target="_blank">
  美国东南部
 </a>
 ，截至16日已知4州至少30人遇难。搜救人员仍在佛罗里达狭长地带的破瓦残砾中，寻找上千名下落不明的民众。
 <br/>
 <br/>
 4级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 （Hurricane Michael）本月10日以每小时250公里的风速在佛州墨西哥湾沿岸登陆，引发海水暴潮，房屋损毁严重。
 <br/>
 <br/>
 受灾严重的佛罗里达州狭长地带（Florida Panhandle），清理修复工作正在展开。佛州警方告诉媒体，贝郡（Bay County）寻获12具遗体后，佛州死亡总数已升至20人。
 <br/>
 <br/>
 另外，加上乔治亚州发现1人死亡，北卡罗来纳州3人遇难，维吉尼亚州有6人丧生，已知4州共有30人遇难。
 <br/>
 <br/>
 截至当地时间周二（16日），佛州仍有近13万7000户住家和商家断电。
 <br/>
 <br/>
 联邦紧急事务管理局在佛州和乔治亚州建立42个救灾物品分发点，但在受灾最严重的地区，居民仍通过空投食物和水来解决生活问题。
 <br/>
 <br/>

 </center>
 <br/>
 为搜索和救助被困灾民，直升机、重型设备和搜索犬都投入救援工作。佛州州长斯科特表示，目前有1700名搜救人员，在搜查灾区内2万5000所房屋。
 <br/>
 <br/>
 休士顿搜救志工网CrowdSource Rescue发动上百名志工在佛州搜寻超过1135名与亲友失联的民众。共同创办人马尔凯（Matthew Marchetti）表示，死亡人数可能还会攀升，〝我们每找到一个人，可能就还有3人没找到。〞
 <br/>
 <br/>
 周一上午，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）和第一夫人梅拉尼娅赶赴佛罗里达和乔治亚州，查看灾情。
