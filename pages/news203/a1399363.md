### 国会两党领袖多连任 众院议长1月票选
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】新一届美国参众两院的议员大部分已尘埃落定，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 在两院的领袖也已选举出台。
 <br/>
 <br/>
 第116届美国众议院议员周三在华盛顿国会大厦外合影。
 <br/>
 <br/>
 天气已经相当寒冷，有些议员衣着单薄，她们手挽手互相取暖。
 <br/>
 <br/>
 拍摄集体照是新议员入职周的一系列活动之一。
 <br/>
 <br/>
 这届众议院成员的平均年龄更年轻，成分也更多元化。
 <br/>
 <br/>
 女性议员的数量达到1974年以来最多，她们尤其在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 中占有举足轻重的地位。
 <br/>
 <br/>
 麦卡锡成功连任下一届众议院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 领袖。而今年中期选举后，共和党变成少数党。瑞安的议长席位将由民主党人取代。议长选举将于明年1月3日在众议院进行，获得至少218票者当选，南希．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佩洛西.html" target="_blank">
  佩洛西
 </a>
 是有力竞争者之一。
 <br/>
 <br/>
 参议院少数党领袖、民主党人舒默赢得连任。
 <br/>
 <br/>
 参议院少数党领袖 舒默：〝参院民主党的团结一直是我们的力量所在，这将在第116届国会中得以延续。〞
 <br/>
 <br/>
 参议院多数党领袖、共和党人麦康奈尔也赢得连任。他说，二战后大部分时间是两党分掌两院，两党合作不是问题。
 <br/>
 <br/>
 参议院多数党领袖 麦康奈尔：〝我将与众议院佩洛西和众议院民主党人讨论我们可以一起做些什么。〞
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
