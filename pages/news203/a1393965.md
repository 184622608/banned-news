### 促成北美贸易新协定 英媒：川普女婿建头功
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】美国和加拿大的贸易谈判经过长达一年的艰辛拉锯战，终于在今年9月30日达成新协议，这是美国重返北美自由贸易协定取得的一个重大胜利。日前，有英国媒体报导了美国贸易代表团分别与墨西哥和加拿大谈判过程中的一些内幕情况，指川普的女婿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贾里德・库什纳.html" target="_blank">
  贾里德・库什纳
 </a>
 在其中发挥了十分关键的作用，建下了〝头功〞。
 <br/>
 <br/>
 据英国《每日邮报》报导，被外界视为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美加谈判.html" target="_blank">
  美加谈判
 </a>
 中最重要人物的美国贸易代表罗伯特・莱特希泽（Robert Lighthizer），本周一（10月1日）在白宫玫瑰园中介绍美加贸易谈判取得成功的情况时表示，这一次的美加贸易谈判的设计师是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贾里德・库什纳.html" target="_blank">
  贾里德・库什纳
 </a>
 （Jared Kushner）。莱特希泽说：〝我之前已经说过了，我再说一遍，如果不是贾里德，这个协议就不会发生。〞
 <br/>
 <br/>
 现年37岁的贾里德・库什纳是美国总统川普（特朗普）的女儿伊万卡的丈夫，他在从政前是一名成功的房地产商人，是帮助川普成功获选美国总统的功臣之一。在川普正式就任美国总统之初，身为白宫高级顾问的库什纳就开始参与处理美国贸易问题，特别是涉及加拿大和墨西哥的贸易谈判。
 <br/>
 <br/>
 《每日邮报》的报导称，库什纳在维持北美自由贸易协定的过程中扮演着〝重要角色〞。报导引述一位接近谈判的消息人士称，库什纳与墨西哥外交部长路易斯・维德加雷（Luis Videgaray）从华尔街时期就建立起的友谊，帮助化解了美墨关系中的几次危机，并最终促成了今年8月达成的美墨贸易协议。
 <br/>
 <br/>
 维德加雷本人也曾在接受路透社采访是坦言，美墨贸易谈判不止一次面临破裂，但每次库什纳都能找到弥合裂缝的方法，让谈判维持下去。
 <br/>
 <br/>
 报导并引述一位匿名的消息人士称，在川普上任美国总统初期，加拿大总理贾斯汀・特鲁多（Justin Trudeau）最信赖的的两名顾问 —— 办公厅主任凯蒂・特尔福德（Katie Telford ）和私人秘书杰拉尔德・巴茨（Gerald Butts）都曾飞往华盛顿与库什纳会面，他们与库什纳建立了友谊。当美国贸易代表莱特希泽在谈判中与加拿大外交部长克里斯蒂娅．弗里兰(中文名〝方慧兰〞)产生情绪对立时，库什纳与特尔福德以及巴茨的这种联系的作用便凸显出来。
 <br/>
 <br/>
 报导引述华盛顿的一位知情人士说，〝曾经有一段时间，传统的谈判渠道似乎已经崩溃，因为他们无法解决一些重大的问题，乳制品就是其中之一。〞为了解决这个令谈判双方陷入僵局的重大难题，库什纳把莱特希泽与特尔福德和巴茨拉到一起举行了一次电话会议，最终他们四人共同解决了一些悬而未决的问题，使得谈判取得突破，并最终促使美加双方成功达成了新的协议。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
</div>
