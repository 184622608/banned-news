### 美国超级百万奖金累积至9.7亿 周五晚开奖
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】美东时间周五晚上11点，美国的超级百万彩票或将诞生历史上第二高奖金。由于上一次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开奖.html" target="_blank">
  开奖
 </a>
 没有人6个数字全中，所以本周五的奖金飙升到9.7亿美元。
 <br/>
 <br/>
 周五，纽约宾夕法尼亚车站的一家出售彩票的小商店里人流不断。这里曾经多次卖出过中奖彩票，吸引不少怀着发财梦的人赶来试运气。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彩票玩家.html" target="_blank">
  彩票玩家
 </a>
 Shearrod Duncan：〝（如果中奖）可能会买栋房子，去渡假，就不会再上班啦。〞
 <br/>
 <br/>
 彩票玩家 Mel Kleinman：〝我们希望中奖，希望很多人都能中奖，但如果我们中奖，想为慈善机构捐很多钱。〞
 <br/>
 <br/>
 今年7月24日以来，超级百万始终没有人中过大奖，致使奖金不断累计，目前已飙升到接近10亿美元。
 <br/>
 <br/>
 不过与钜额奖金形成强大反差，中奖概率微小到了不足三亿分之一。
 <br/>
 <br/>
 如果这次有人中大奖，就可以选择一次性领取5.48亿美元，也可以选择在29年内分期领完这9.7亿美元的钜款。
 <br/>
 <br/>
 如果仍没人中奖，下周二奖金将飙至16亿美元，创下美国最高乐透奖金的历史记录。目前的最高记录是2016年创下的15.86亿美元的威力球奖金。
 <br/>
 <br/>
 超级百万彩票可以在全美44个州，华盛顿特区以及美属维京群岛买到。
 <br/>
 <br/>
 新唐人记者乔安综合报导
