### 〝经济铁幕〞将临？中期选举后美中更难达成协议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 尘埃落定，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 所在共和党保住了参议院，但失去众议院主导权。美学者分析，因为民主党阻挠，川普将更难和中共达成贸易协议。美前财长则认为，全球〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济铁幕.html" target="_blank">
  经济铁幕
 </a>
 〞可能降临，美中或进入〝全面冷战〞。
 <br/>
 <br/>
 台湾中央社此前引述消息指，北京从来没有象今年这样关注中期选举。报导指，北京仍期待民主党能在国会取得优势，以箝制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 施政。
 <br/>
 <br/>
 不过，英国广播公司（BBC）引述华府保守派智库美国企业研究所研究员史剑道（Derek Scissors）的分析指，即使民主党掌控参众两院，因为民主党支持遏制中共的贸易政策，更会抓住任何机会阻挠川普政府与北京达成贸易协议。
 <br/>
 <br/>
 美国外交关系协会副总裁林德赛（James Lindsay）亦撰文表示，若华盛顿停止中美贸易战，民主党无论从政策还是党派政治来考量，都有动机批评川普政府没有达成对美国有利的协议。
 <br/>
 <br/>
 11月7日，美国前财长保尔森（Hank Paulson）在新加坡举行的彭博创新经济论坛上表示，美中利益渐行渐远，对国际体系中一些根本性问题看法相反，已令美中关系变得高度紧张。
 <br/>
 <br/>
 他指出，中共在网络上的行径，以及南海的军事化行动，令美国意识到中共已是一个主要的长期对手。即使在经济领域，双方的巨大分歧也已不仅仅局限于贸易问题。
 <br/>
 <br/>
 〝除非这些涉及面更广和更深层次的问题得到解决，我们将面临两国关系的严冬。〞保尔森断言。
 <br/>
 <br/>
 他呼吁北京〝大胆〞开放经济，引入外企竞争，施行市场经济，并保护知识产权。他并警告称，如果中方不迅速行动，对美中经济割离的呼声可能越来越强。
 <br/>
 <br/>
 保尔森认为，美中经济的紧张关系已达到临界点，如果双方在货物、资金、技术和人员四方面持续脱鈎，将阻碍全球经济的流动，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济铁幕.html" target="_blank">
  经济铁幕
 </a>
 〞有可能降临，并使经济全球化后退。
 <br/>
 <br/>
 据大纪元刊载的《魔鬼在统治着我们的世界》一书揭露，经济全球化本是现代工业发展的自然过程，但是近几十年来已被共产主义所劫持，不但挽救了六四屠杀后行将崩溃的中共政权，还给全世界带来贫富分化、经济独裁、国家主权丧失等问题，正在向建立共产主义预谋的全球性极权政府的方向前进。
 <br/>
 <br/>
 川普上台后，不断〝退群〞，重整国际规则，遏制全球化的继续推进。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
