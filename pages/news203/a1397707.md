### 中共影响下 美大学自我审查或成流行病
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】最近，美国康奈尔大学中断了与中国人民大学的学术合作。相关报告指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 影响之下，美国精英大学和学者的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自我审查.html" target="_blank">
  自我审查
 </a>
 普遍存在。专家呼吁各大学联手应对。
 <br/>
 <br/>
 捍卫言论与学术自由，一直是美中交流中的敏感议题。美国《新共和》杂志9月初发表学者石宇（Isaac Stone Fish）的调查报告指出，美国精英大学在研究和涉及中国问题时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自我审查.html" target="_blank">
  自我审查
 </a>
 几乎已成为一种〝流行病〞。
 <br/>
 <br/>
 汉学家林培瑞认为，自我审查的最严重代价是，影响美国公众对于中国的正确理解与认知。
 <br/>
 <br/>
 美国加州大学河滨分校教授  林培瑞：〝他描写中国要是只用一些北京欢迎的字眼，或者北京欢迎的题目，那当然是给学生和民众一种中国不全的印象。只给它好的一面，差的一面看不到。〞
 <br/>
 <br/>
 石宇在调查报告中指出，目前西方学者和机构增加自我审查，有三个主要原因。一是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 影响力上升，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大学.html" target="_blank">
  美国大学
 </a>
 担心，一旦惹恼中共，再也不能进入中国进行研究，损失会更大。第二是中共对国内外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_言论自由.html" target="_blank">
  言论自由
 </a>
 的打压越来越严重。第三，美国大学在经济上越来越依赖于中共。
 <br/>
 <br/>
 美国加州大学河滨分校教授 林培瑞：〝也许冒险你上黑名单，或者你下次到中国去，有些研究机会不给你了。这代价是有的。可是归根结底，最重要的任务是说真话。〞
 <br/>
 <br/>
 曾任康奈尔大学校董的中国问题专家章家敦指出，中共直接干预影响美国校园，主要是通过中国学生学者联合会(缩写CSSA)和孔子学院两个机构。他们的一些活动不仅侵犯学术自由，也损害到美国的主权和法律。
 <br/>
 <br/>
 人权观察中国部主任 Sophie Richardson：〝我认为大学要履行其责任，保护其学生和学者的学术自由，这是最重要的。〞
 <br/>
 <br/>
 美国加州大学河滨分校教授 林培瑞：〝关键可能是要教育那些大学里头的校长、副校长，它(自我审查)的代价在哪。〞
 <br/>
 <br/>
 普林斯顿大学政治与公共事务副教授 Rory Truex：〝我们希望美国大学能够更多携手，当学者受到打压时，能帮助他们，进行良好的研究。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
