### 川习会结果 美中各表 专家解析
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 后，美中双方公布的同一份协议内容，却存在着不小的差异。有关专家对此作出了解读。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 后，白宫宣布，2019年1月1日，暂不将2000亿中国进口商品的关税税率，从10%调升到25%。双方将立即就强制技术转让、知识产权保护等〝结构性改变〞展开谈判。如果90天后无法达成一致，美国将把关税升至25%。而中方同意，立即购买美国农产品，并购买大量农业、能源、工业等产品，减少双方贸易不平衡。
 <br/>
 <br/>
 美国记者：〝过去中方多次违背协议，这次是什么让你受到了鼓舞呢？〞
 <br/>
 <br/>
 库德洛：〝我想是习主席对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统的陈词，我们出席周六晚宴（川习会）的人，都觉得（陈词）难以置信，不同以往。其详细程度非同小可，这是我们从未从（中方）口中听过的。所以我觉得有希望。但我想提醒的是，他们（中方）不能慢吞吞地行动，拖延，绕弯子。〞
 <br/>
 <br/>
 谢田：〝美国政府比较用善意解决，但他们实际上没有放弃原则，只是给了（北京）一个喘息的机会，考虑如何改变中国的经济结构。〞
 <br/>
 <br/>
 不过中共党媒《人民日报》和官媒《新华社》，在稍后宣布协议内容时，只字未提〝90天限期〞，而是宣称双方同意〝停止加征新的关税〞。
 <br/>
 <br/>
 唐靖远：〝中共这样做，一个是片面宣传对中共有利的信息，营造中共并未败退而是所谓双赢的假象；二是不排除它在给自己预留后路，一旦90天后谈判失败，它可能宣称是美方毁约升级关税。〞
 <br/>
 <br/>
 另外中共宣称下轮谈判方向是〝取消所有加征关税〞，不同于美国主张的谈判方向为〝结构性改变〞。还有多项细节，美中表述都存在差异。
 <br/>
 <br/>
 唐靖远：〝中共这样做，显示出北京当局面临内部强大压力，无法公开承认当前被‘缓期执行’的困境，不过如果连承认谈判结果的魄力和诚意都没有，那么接下来有没有魄力和诚意进行大刀阔斧的结构性改革，就很难让人感到乐观。〞
 <br/>
 <br/>
 谢田：〝现在最关键的问题是，能不能让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 认识到，中共在经济上在统治上都受到了最大的挑战，能不能通过改变中国的政治结构，经济结构，来应对这个难关。川普政府的智慧在于，他在其他所有方面都没有放松。比如限制美国科技非法流向中国，技术间谍，网络攻击，实际上美国没有任何放松，都在同时给中共施加压力。〞
 <br/>
 <br/>
 美国媒体周一报导，川普总统已任命被视为强硬派的美国贸易代表莱特希泽，监督与中方的新一轮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易谈判.html" target="_blank">
  贸易谈判
 </a>
 。中方可能由刘鹤率队，下周前往华盛顿谈判。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
