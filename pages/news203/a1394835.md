### 川普再警告 对2670亿中国商品加税
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 周二在白宫表示，美国近期向中国产品征收关税和其他措施，如果北京进行报复，他将对额外价值2670亿美元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国商品.html" target="_blank">
  中国商品
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加征关税.html" target="_blank">
  加征关税
 </a>
 。
 <br/>
 <br/>
 川普周二在白宫与美国驻联合国代表黑利一起宣布黑利辞职消息时，有记者问到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易.html" target="_blank">
  美中贸易
 </a>
 问题。
 <br/>
 <br/>
 川普表示，多年来美中贸易存在不平等，美国不会再这样了。
 <br/>
 <br/>
 美国总统川普：〝我们和中国关系很好。但他们许多年来，每年从我们国家拿走5,000亿美元。我们不能再让这事发生。〞
 <br/>
 <br/>
 美国总统川普：〝我们帮着重建了中国。如果我们没有那样做，中国不会是现在这样。对我来说没问题，但我们不会再这样做了。〞
 <br/>
 <br/>
 川普将之前的美中贸易关系比喻为单行道。
 <br/>
 <br/>
 美国总统川普：〝我们不能只有单行道，应该是双行道。过去25年来一直是单行道，我们得将它改成双行道，我们也会因此获益。〞
 <br/>
 <br/>
 川普认为，中国已采取报复手段。
 <br/>
 <br/>
 美国总统川普：〝他们已经报复过了。他们拿走了5,000亿美元。这一点，我认为这已经是终极报复。〞
 <br/>
 <br/>
 不过，他表示北京想达成协议，但还没有做好准备。
 <br/>
 <br/>
 美国总统川普：〝我说他们没准备好。我们取消了一些（和中国）的会谈，因为他们没准备好达成协议。〞
 <br/>
 <br/>
 川普警告，如果北京继续报复，将对另外2670亿美元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国商品.html" target="_blank">
  中国商品
 </a>
 征收关税。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
