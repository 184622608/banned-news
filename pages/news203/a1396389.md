### 美国退出〝中导条约〞施压俄罗斯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】观众朋友大家好，欢迎收看新唐人全球新闻。上周六，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）明确提出要退出里根时期签署的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中程导弹条约.html" target="_blank">
  中程导弹条约
 </a>
 》。专家认为，川普欲施压
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 、进而争取其合作抵制中共。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （2018.10.20）：〝我们不会让他们违反核协议、去发展武器，却不允许我们这样做。所以我们要停止协议，我们将退出。〞
 <br/>
 <br/>
 川普20日宣布要退出《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中程导弹条约.html" target="_blank">
  中程导弹条约
 </a>
 》因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 违反了协议。俄罗斯外长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉夫罗夫.html" target="_blank">
  拉夫罗夫
 </a>
 随后表示，希望美方解释清楚。
 <br/>
 <br/>
 对于川普的表态，专家认为俄罗斯早有预期。
 <br/>
 <br/>
 哈德逊研究所政治和军事分析中心主任Richard Weitz：〝这个决定有道理。影响很小。我认为俄罗斯已经预期美国会退出。〞
 <br/>
 <br/>
 Richard Weitz认为，川普以退出协议换协商合作、欲联合俄罗斯对抗中共威胁，但短期内他认为俄罗斯不会接受这一提议。
 <br/>
 <br/>
 周一，美国国家安全顾问博尔顿先后与俄罗斯联邦安全局长派楚希夫（Nikolai Patrushev）和俄罗斯外交部长拉夫罗夫（Sergey Lavrov）会晤。此次访俄，博尔顿会沟通核武条约、叙利亚、伊朗、朝鲜等问题。
 <br/>
 <br/>
 俄罗斯联邦安全局在会晤后的声明中说，〝双方讨论了莫斯科与华盛顿在战略问题上展开对话的前景〞。
 <br/>
 <br/>
 哈德逊研究所政治和军事分析中心主任Richard Weitz：〝双方关系不好，所以谈不上对话，只是互通信息而已。叙利亚问题上，双方都有军事力量在那所以需要处理争端。〞
 <br/>
 <br/>
 周二，博尔顿将与普京会晤。
 <br/>
 <br/>
 哈德逊研究所政治和军事分析中心主任Richard Weitz：〝俄罗斯把自己和其他所有人的关系搞得很糟，所以他们愿意接受中共，目前俄罗斯拒绝接受中共是当前俄美巨大威胁的说法，从而与美国合作应对中共。同时，中共与大部分周边国家关系也不好，像印度、越南等，所以中共政府也是尽可能与俄罗斯保持良好关系。〞
 <br/>
 <br/>
 1987年美俄双方签署了《中程导弹条约》，禁止美国和俄罗斯拥有、生产或试飞搭载核弹头的路基巡航导弹，射程为300英里至3400英里之间。
 <br/>
 <br/>
 不过代表美国签字的里根总统当时更倾向于太空防御核导弹计划。川普政府正在力促国会尽快通过组建美国太空军的计划。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
