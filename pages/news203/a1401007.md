### 抗议中共审查版搜素引擎 谷歌员工发网络公开信
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】周二（11月27日），一些谷歌（Google）员工在网上发表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公开信.html" target="_blank">
  公开信
 </a>
 ，声明反对公司开发满足
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 需要的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_审查.html" target="_blank">
  审查
 </a>
 版搜索引擎、从而为中共压迫弱势群体提供技术支持。
 <br/>
 <br/>
 谷歌代号为〝蜻蜓计划〞（Project Dragonfly）的研发项目今年夏天被曝光，并引发了美国社会的极大关注，公司内部的反对也极为强烈。继9月13日美国网络媒体〝BuzzFeed News〞报导谷歌公司已有7名员工因此辞职、1700多员工签名反对后，两个多月来，公司内部的反对声浪有增无减。
 <br/>
 <br/>
 周二，一封由部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌员工.html" target="_blank">
  谷歌员工
 </a>
 联合签名的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公开信.html" target="_blank">
  公开信
 </a>
 发表在网志发布平台〝Medium 〞上。这些员工表示，他们之中的很多人之所以接受公司的雇用，是因为他们赞同公司的价值观，其中包括公司先前对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_审查.html" target="_blank">
  审查
 </a>
 和监控体系的立场、以及谷歌被视作一家愿意把价值观置于利润之上的公司。
 <br/>
 <br/>
 这些员工在公开信中指出，但一年来，他们对公司开发〝蜻蜓计划〞等项目、支持施虐者感到非常失望，所以他们不再相信谷歌的确如他们当初所认为的那么美好。
 <br/>
 <br/>
 截至该公开信发表，已有11名谷歌员工在信上签名。
 <br/>
 <br/>
 〝蜻蜓计划〞是谷歌公司秘密开发的针对中共市场的审查版搜索引擎，旨在配合中共监控网络。该计划被曝光后，人权组织及美国政界也都表示强烈关注和反对。
 <br/>
 <br/>
 谷歌的母公司Alphabet的董事长约翰．汉尼斯（John Hennessy）上周承认，与中共做生意需要在〝核心价值观〞上妥协。
 <br/>
 <br/>
 而在周二的公开信中，员工们集体指责说，几个月来，数千名谷歌员工已表达了自己对〝蜻蜓计划〞的反对立场，国际人权组织和调查记者也发出了警告，要求公司取消该项目。但是，公司领导层的反应至今仍不令人满意。
 <br/>
 <br/>
 这些员工说，他们反对那些帮助有权者压迫弱势群体的技术，不管这些技术用于何地。适应中共的〝蜻蜓计划〞将会树立一个危险的先例。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
