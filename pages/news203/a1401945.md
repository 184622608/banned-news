### 参院介入调查 要求FBI解释突袭铀矿门举报人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法委员会.html" target="_blank">
  司法委员会
 </a>
 主席格拉斯利（Chuck Grassley）日前致信联邦调查局（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 ）局长克里斯托弗‧雷（Christopher Wray），要求其就FBI突击搜查希拉里
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_铀矿门.html" target="_blank">
  铀矿门
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_举报人.html" target="_blank">
  举报人
 </a>
 住所事件做出解释。
 <br/>
 <br/>
 据美国保守派网站〝每日电讯〞（The Daily Caller）12月3日报导，在该媒体11月29日报导
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 突袭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_铀矿门.html" target="_blank">
  铀矿门
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_举报人.html" target="_blank">
  举报人
 </a>
 事件后，11月30日，格拉斯利致信雷，要求其解释相关问题。
 <br/>
 <br/>
 11月19日上午，16名FBI特工突袭了司法部认可的举报人丹尼斯‧内森‧凯恩（Dennis Nathan Cain）位于马里兰州的住所，并劫持了有关克林顿基金会（Clinton Foundation）和〝铀壹公司〞（Uranium One）的文件。
 <br/>
 <br/>
 这些文件披露，希拉里（Hillary Clinton）在担任国务卿期间，曾促成美国〝铀壹公司〞将大部分所有权卖给俄罗斯国家原子能公司（Rosatom），从而让俄罗斯掌握了美国20%以上的铀供应、危害美国国家安全。作为利益回报，俄国官员随后向克林顿基金会输送了几百万美元。。
 <br/>
 <br/>
 FBI早在2009年就掌握了俄国核工业部门通过行贿等方式换取美国铀矿资源的证据，但仍坐视美俄于2010年达成了铀交易。之后，由于这项协议倍受争议， FBI在时任局长穆勒（Robert Mueller）的领导下故意继续调查了将近4年，藉此向美国国会和公众隐瞒黑幕。
 <br/>
 <br/>
 凯恩在为一名FBI承包商工作时发现了令人震惊的相关文件，之后他把文件转交给司法部总监察长霍洛维茨（Michael Horowitz），霍洛维茨又派高级官员亲自将这些文件交给参众两院的情报委员会。
 <br/>
 <br/>
 格拉斯利11月30日也给霍洛维茨发了一封信，向他索要凯恩的文件并询问相关情况。
 <br/>
 <br/>
 凯恩的律师索卡拉斯（Michael Socarras）表示，凯恩曾告诉领导搜查的FBI特工，他是依据法律受到保护的举报人，但16名FBI特工仍在他家里搜查了长达6小时，凯恩则坐在客厅里恐惧地面对那一场面。
 <br/>
 <br/>
 索卡拉斯指出，凯恩在举报过程中所做的一切都是遵照合法程序的。格拉斯利在信中质问FBI局长雷，FBI是否认为凯恩的举报行为应当受到保护。
 <br/>
 <br/>
 格拉斯利还要求雷告知突袭凯恩的依据，并提供搜查令原件的副本以及所有的证明文件。
 <br/>
 <br/>
 搜查令是由美国巴尔的摩地区法院（U.S. District Court for Baltimore）的联邦法官加拉格尔（Stephanie A. Gallagher）签发的，她是前总统奥巴马（Barack Obama）提名的法官。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
