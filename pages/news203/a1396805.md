### 抵制中共渗透海外  专家吁支持独立敢言媒体
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】最近，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主国家.html" target="_blank">
  民主国家
 </a>
 的干预渗透日益引起国际关注，其中对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_媒体.html" target="_blank">
  媒体
 </a>
 的影响控制是重要一环。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专家.html" target="_blank">
  专家
 </a>
 呼吁，民主国家应给予海外独立敢言媒体更多支持，以捍卫其自由价值观和公民权利。
 <br/>
 <br/>
 近十年来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 耗资450亿人民币，打造所谓〝大外宣〞工程，扩大对海外的宣传渗透，并打压异议声音。
 <br/>
 <br/>
 自由之家东亚高级研究员Sarah Cook：〝一方面，中共投入数十亿美元，来扩张其官方
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_媒体.html" target="_blank">
  媒体
 </a>
 ，同时压制海外的批评声音和报导。〞
 <br/>
 <br/>
 共产主义受难者基金会中国研究员Peter Mattis：〝中共在澳洲和其它地方采取的最有效手段，就是控制中文媒体。现在只剩一家以及法轮功媒体不受控制。〞
 <br/>
 <br/>
 长期以来，大纪元、新唐人等独立媒体都是中共打压的主要对象，包括初期在国内抓捕记者，到持续干扰卫星租用，再到威胁广告客户。不过中共的持续打压，不仅没有让这些媒体消音，反而跃升为海内外最受欢迎的中文媒体，显示民众对真实信息的强烈渴望。
 <br/>
 <br/>
 新唐人制作人及大纪元资深编辑Jan Jekielek：〝大纪元时报和新唐人电视的中文网站，是仅有的排名进入美国前1000名的中文媒体。虽然网站被中共封锁，但我们的中文网，四分之一流量都来自中国的VPN。〞
 <br/>
 <br/>
 美国副总统彭斯日前讲话，直指中共对美国的干预渗透。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专家.html" target="_blank">
  专家
 </a>
 认为，这不只涉及国家安全，更是捍卫公民权力和自由价值观的问题。他们也建议美国通过加强相关立法与执法、增加透明度等方式，反制中共媒体渗透。
 <br/>
 <br/>
 自由之家东亚高级研究员Sarah Cook：〝还有就是提供资金和能力，来帮助美国的(独立)中文媒体，提供不同于中共官媒如CCTV的选择。〞
 <br/>
 <br/>
 加州大学柏克莱分校信息学院教授萧强：〝唯有当我们反击，捍卫我们自己和我们的自由之时，中共专制政权才会得到教训。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
