### 川普为共和党人助选 不担心选情
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】周日美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ，前往乔治亚州，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 人竞选助阵。川普临行前接受媒体采访时表示，共和党人在参众两院的表现非常好，民众对共和党人非常支持。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 直言不讳的说，虽然媒体不愿报道美国经济的良好发展，但现在经济情况是有史以来最好的。
 <br/>
 <br/>
 美国总统川普：〝周五报告显示，就业人数又增长250,000名。十月份非常好，工资上涨3.1%.〞
 <br/>
 <br/>
 川普还说，媒体报导大篷车难民，这没问题，但他认为，边境现在加强了管理，不会再像民主党那样敞开大门。
 <br/>
 <br/>
 美国总统川普：〝我们的边境工作非常好。没有人做到我们这样。民主党人想要开放边境，使成千上万（难民）涌入美国。不会让它这样。〞
 <br/>
 <br/>
 川普表示，每次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 人竞选集会，都有大批民众到场支持，和2016年总统大选时一样。
 <br/>
 <br/>
 美国总统川普：〝民众热情高涨，因为经济情况良好，民众热情高涨，因为边境（安全），民众热情高涨，因为减税、再减税。〞
 <br/>
 <br/>
 川普表示，不担心共和党的选情，如果需要，政府将对中产阶级再减税10%。
 <br/>
 <br/>
 美国总统川普：〝如果我们不能掌控众议院，但这不太可能，我们还将对中产阶级再减税10%。〞
 <br/>
 <br/>
 美国周一重启对伊朗制裁。川普表示，就任总统前，伊朗几乎要在一年内占领中东，但现在没人这样想了。
 <br/>
 <br/>
 美国总统川普：〝已经生效的对伊朗的制裁，是我们国家历史上最严厉的制裁。〞
 <br/>
 <br/>
 川普这次是为布莱恩・坎普竞选乔治亚州州长助阵。
 <br/>
 <br/>
 新唐人记者毕心慈、李静明综合报导
</div>
