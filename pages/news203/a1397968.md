### 解读川习会中美分歧大 库德洛:会谈若失败攻势将更猛
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】美国总统川普与中国国家主席习近平近日通话后，中美两国政府均对外释放了比较积极的讯号。海内外专家学者纷纷发表意见，对此进行分析解读。有观点认为，中美双方存在的矛盾其实仍然很大，虽然川习的会面有助于缓和双方关系，但短期内解决贸易战的可能性很低。白宫经济顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 （Larry Kudlow）则在表示谨慎乐观的同时发出警告，如果这次的协议尝试失败，美国政府的攻势将更加猛烈。
 <br/>
 <br/>
 中美两国元首前日通电话后，彭博社引述4名〝知情人士〞爆料称，川普希望本月底在阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上与习近平会谈能达成某种贸易协议，并已经指示美国主要官员开始起草美方可能提出的条款。
 <br/>
 <br/>
 不过，随后美国媒体CNBC（消费者新闻与商业频道）却引述一位匿名白宫官员称，彭博社的上述消息不属实，中美之间的贸易谈判〝还有很长的路要走〞 。
 <br/>
 <br/>
 与此同时，中共官媒新华社对外放话称，期待中美领导人在通话中达成的〝共识〞能化为实在的行动。中共外交部也对外表示，两国领导人在通话中所要求的结果是，通过对话磋商解决彼此关切，〝达成双方都能接受的方案〞。
 <br/>
 <br/>
 就在川普与习近平通话的同一天，白宫经济顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 对外表示，川普和习近平通电话，代表美中关系正在〝融冰〞，G20峰会上的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞有望打破两国间的诸多僵局。如果美中双方能敲定一个大家都满意的协议，美方也可能会撤回部分关税。
 <br/>
 <br/>
 不过，库德洛同时也警告说，如果北京方面未能就美国认为非常重要的一系列问题提出解决办法，美国政府的贸易战攻势将〝更猛烈〞。
 <br/>
 <br/>
 上述消息引起外界的广泛关注，并纷纷对中美贸易冲突的走向做出分析与预估。
 <br/>
 <br/>
 香港《明报》引述中国问题专家分析称，一直表态不愿打贸易战的中国政府届时可能会做出很大让步，虽然不至于对美方要求照单全收，但至少可能同意50%。
 <br/>
 <br/>
 报导并表示，估计中美之间应该已经敲定清单，而且是让川普满意的内容，因此才会宣布这次会面，否则不如不见，因此有理由推断〝川习会〞将就贸易战达成共识或协议。
 <br/>
 <br/>
 不过，北大国际关系教授梁云翔却认为，对于中美贸易战能否在短期内收场的问题不可太过乐观。他表示，中美之间的贸易摩擦并不单纯是经济问题。〝美国觉得中国是要通过不公平贸易发展起来要挑战美国。中国也认为如果完全答应美国的条件，会影响到中国的政治和战略方向。〞
 <br/>
 <br/>
 《美国之音》则在相关报导中引述兰德公司高级政策分析师莫里斯（Lyle Morris）的推文指出：川普总统称为解决美中贸易问题的努力正在良好地进行，这里可能有两个解释，第一，习近平向他暗示了中国会对美国的要求作出重大让步；第二，总统抢先一步，但被误导而期待习近平会作出重大让步。
 <br/>
 <br/>
 莫里斯并表示，他会〝坚实的〞选择第二个解释。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
