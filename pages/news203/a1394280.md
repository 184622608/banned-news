### 美国最被讨厌城市大排名 达拉斯居首纽约第六
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】一项最新的调查研究，对全美数十个大城市的生活质量不佳做出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_排名.html" target="_blank">
  排名
 </a>
 ，德州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_达拉斯.html" target="_blank">
  达拉斯
 </a>
 排名最高，成为全美最惹人嫌的大城市，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 市排名第六位。
 <br/>
 <br/>
 芝加哥营销公司Digital Third Coast的一项调查，采访了全美人口最多的24个大城市的2500名居民，对各大城市的〝讨厌〞程度进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_排名.html" target="_blank">
  排名
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 市在24个城市中排名第六，居民抱怨最多是市内噪音、装修噪音、烹饪气味过大、垃圾太多、以及私人物品占用公共区域。
 <br/>
 <br/>
 其他城市中，德克萨斯州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_达拉斯.html" target="_blank">
  达拉斯
 </a>
 尴尬登上榜首，其次是迈阿密、奥斯汀、费城、洛杉矶。旧金山排在第七位。
 <br/>
 <br/>
 另外54%的受访者表示，曾与邻居发生争执，而争执发生最多的10个城市中，有8个位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 南部，研究分析，炎热的气候也可能是引发争端的原因之一。
 <br/>
 <br/>
 新唐人记者韩瑞纽约报导
</div>
