### 中期选举结果揭晓 亚欧股市反应不一
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 结果周三出炉，欧州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市反应.html" target="_blank">
  股市反应
 </a>
 积极，而亚洲市场，因为美朝关系生变反应不一。专家认为，中期选举的结果不会改变贸易战。
 <br/>
 <br/>
 民主党重掌美国众议院，有机会阻止川普总统的一些施政计划。不过，瑞士宝盛银行首席经济学家科尔表示，美国贸易和外交决策来自政府，而不是国会，所以美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 结果对市场和贸易战影响很小。
 <br/>
 <br/>
 瑞士宝盛银行首席经济学家 科尔：〝影响市场的主要问题，比如贸易战，都不会因(选举)结果而改变。〞〝对市场而言，相关的是(美国)贸易冲突，税改进展等。现在川普有一个民主党掌权的国会，这个新局势不会带来什么变化。〞
 <br/>
 <br/>
 周三，泛欧斯托克600指数上涨1.1%。所有欧洲主要国家和行业指数早盘时都处于上升阶段。
 <br/>
 <br/>
 亚洲股市不仅受到美国中期选举结果的影响，还受到另外一个消息的冲击。周三，美国宣布无限期推迟国务卿蓬佩奥和朝鲜官员的会晤，这一消息打击了亚洲股市。
 <br/>
 <br/>
 日本日经指数收盘下跌0.3％，至22,085.80点，韩国KOSPI指数下跌0.5％至2,078.69点，香港恒生指数小幅上涨0.1％，报26,147.69点。
 <br/>
 <br/>
 美元兑大多数主要货币走弱。
 <br/>
 <br/>
 新唐人记者王羽音、宇微综合报导
