### 川普辟谣手机被窃听问题:《纽时》报导太离谱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】周四（10月25日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统就所谓其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手机.html" target="_blank">
  手机
 </a>
 被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_窃听.html" target="_blank">
  窃听
 </a>
 一事辟谣，澄清自己只有一部政府手机且很少使用，不存在使用通讯工具不当的问题，并批评《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约时报.html" target="_blank">
  纽约时报
 </a>
 》的报导过于离谱。
 <br/>
 <br/>
 周四早上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统发推表示，《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约时报.html" target="_blank">
  纽约时报
 </a>
 》（The New York Times，以下简称《纽时》）报导失实，他使用通讯工具的习惯并非如该报导所说的那样。
 <br/>
 <br/>
 〝那些所谓的川普专家写了一篇关于我使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手机.html" target="_blank">
  手机
 </a>
 情况的文章，冗长无聊，发表在《纽约时报》上。内容很不准确，而我没时间放在这上、去纠正它。我只使用政府电话，只有一部政府手机且很少使用。报导太离谱了！〞
 <br/>
 <br/>

 <br/>
 《纽时》周三晚报导，〝美国情报机构官员〞称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 和俄罗斯的间谍一直都在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_窃听.html" target="_blank">
  窃听
 </a>
 川普的手机，尤其是中共，妄图通过搜集川普与朋友交谈的内容，来研究他的想法并找出应对贸易战的策略。
 <br/>
 <br/>
 《纽时》还称，这些官员透露，川普共有3部手机，其中2部是官方手机，1部是私用iPhone，而美国国家安全局（ National Security Agency）对川普私人手机的安全漏洞尚未处理。
 <br/>
 <br/>
 不过，川普在推文中简单澄清了事实，虽然他表示自己不愿多花时间放在辟谣上。
 <br/>
 <br/>
 《纽时》的报导显然在指责川普使用手机不当，这使人们很容易联想到2016年大选前发生的希拉里（Hillary Clinton）电邮门事件。希拉里一直表示，联邦调查局（FBI）在大选前10多天重启电邮门调查，这是导致她败选的致命原因。
 <br/>
 <br/>
 但是，希拉里使用私人服务器收发政府邮件一事属实，而《纽时》的消息则出自不具名的〝现任或前任美国官员〞。
 <br/>
 <br/>
 川普一再讽刺《纽时》是〝失败的《纽约时报》〞 （Failing New York Times），并特别批评其报导中不明身份的所谓〝消息人士〞。
 <br/>
 <br/>
