### 佛州重新计票再生变 法官裁决棕榈滩延后5天
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】周二（11月13日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 裁决，允许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_棕榈滩.html" target="_blank">
  棕榈滩
 </a>
 县（Palm Beach）延长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 时间，于11月20日之前提交结果。这一裁定给佛州选举结果增加了更多变数。
 <br/>
 <br/>
 据《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_棕榈滩.html" target="_blank">
  棕榈滩
 </a>
 邮报》（The Palm Beach Post）13日报导，周二下午，一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 莱昂县（Leon）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 裁定，棕榈滩县提交
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 结果的时间可以在本周四下午3点的截止期限后再延长5天，即于11月20日之前完成。
 <br/>
 <br/>
 按照佛州法律，州选票审核委员会必须在11月20日之前提交选举结果，莱昂县法官的裁决很可能会推迟这一进程，使下届佛州参议员、州长和农业厅长3个位置在未来几天继续保持悬疑状态。
 <br/>
 <br/>
 佛州民主党参议员候选人尼尔森（Bill Nelson）向塔拉哈西市（Tallahassee）的美国地方法院提起诉讼，要求给予全州67个县的所有选举监督员更多时间重新计票。
 <br/>
 <br/>
 此诉讼涉及的三项选举包括：尼尔森与现任州长、共和党候选人斯科特（Rick Scott）之间的参议员竞选，民主党籍的塔拉哈西市长吉卢姆（Andrew Gillum）与共和党联邦众议员德桑蒂斯（Ron Desantis）之间的州长竞选，以及民主党候选人弗莱德（Nikki Fried）与共和党人考德威尔（Matt Caldwell）之间的农业厅长竞选。
 <br/>
 <br/>
 佛州共和党参议员卢比奥（Marco Rubio）周二上午揭露，布劳沃德县（Broward）和棕榈滩县故意拖延重新计票，目的在于最终窃取佛州的两项重要选举。
 <br/>
 <br/>
 佛州州务卿戴兹特（Ken Detzner）上周六宣布启动全州范围的重新计票，迈阿密-戴德县（Miami-Dade County）周六晚即开始工作，至周一中午，该县800,000多张选票已有半数以上审核完毕。然而，《迈阿密先驱报》（Miami Herald）报导，被斯科特控告涉嫌选举欺诈的布县直到周二上午还未启动重新计票，有意拖延计票进程。
 <br/>
 <br/>
 卢比奥指出，布劳沃德县和棕榈滩县之所以拖延计票，目的在于等待法官裁决选举夜7:00后收到的选票也能够入账，而这是违法佛州法律的。
 <br/>
 <br/>
 周二晚，卢比奥再次发推披露，事实上，民主党律师们已经提请法官修改佛州法律，企图以更改原有选举规则的方式使选票流向民主党候选人。
 <br/>
 <br/>
 卢比奥举例说，佛州法律规定，邮寄选票上的选民签名应该与选民本人的签名相符，可是民主党律师却要求法官抛弃这条法律，而命令佛州把不符合这一基本规定的选票也算在其内。
 <br/>
