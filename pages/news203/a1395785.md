### 沙特记者失踪案 川普等待国务卿归来
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国务卿.html" target="_blank">
  国务卿
 </a>
 蓬佩奥就记者卡舒吉在沙特领事馆失踪一事，先去会见了沙特国王和王储，10月17日又和土耳其总统及外长会面。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统表示，他将等待蓬佩奥归来后的全面报告。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统：〝我想弄清楚到底发生了什么。〞〝这个周末就会知道，等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国务卿.html" target="_blank">
  国务卿
 </a>
 蓬佩奥回来，我们会做一个长谈。〞
 <br/>
 <br/>
 川普总统表示已要求土耳其提供任何有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特记者.html" target="_blank">
  沙特记者
 </a>
 卡舒吉失踪的音频或视频证据。
 <br/>
 <br/>
 沙特记者卡舒吉10月2日进入沙特驻土耳其领馆后失踪。土耳其当局16日号首次进入领馆进行了9小时搜查。
 <br/>
 <br/>
 美国国务卿蓬佩奥在访问了沙特之后，周三又和土耳其总统及外长会面。
 <br/>
 <br/>
 土耳其外长 Mevlut Cavusoglu：〝我们之前说过，沙特必须毫不拖延、在各方面配合我们。〞
 <br/>
 <br/>
 卡舒吉生前是〝华盛顿邮报〞专栏作者，长期批评沙特王储萨尔曼和沙特政府。
 <br/>
 <br/>
 新唐人记者毕心慈、李子玉综合报导
