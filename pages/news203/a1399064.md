### 人民选择奖颁奖礼 明星吁关注加州野火
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】周日，美国好莱坞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民选择奖.html" target="_blank">
  人民选择奖
 </a>
 在加州洛杉矶的圣塔莫尼卡举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颁奖仪式.html" target="_blank">
  颁奖仪式
 </a>
 ，有多位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_影视名人.html" target="_blank">
  影视名人
 </a>
 提到了加州当地的大火。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民选择奖.html" target="_blank">
  人民选择奖
 </a>
 是好莱坞历史最悠久的奖项之一，所有获奖人都由观众投票选出。
 <br/>
 <br/>
 今年，漫威大片《复仇者联盟3：无限战争》击败了包括《黑豹》在内的多部强片，夺得年度电影奖。
 <br/>
 <br/>
 另外，韩国人气组合防弹少年团摘下〝年度团体〞、〝年度歌曲〞、〝年度音乐录影带〞和〝年度社群名人〞四个奖项。
 <br/>
 <br/>
 在发奖仪式上，多位获奖者提到了加州的野火。
 <br/>
 <br/>
 实景电视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_明星.html" target="_blank">
  明星
 </a>
 金．卡戴珊（Kim Kardashian）和她的家人获得〝最佳真人秀节目奖〞，但她的家庭也因为野火而不得不逃离家园，同时，她家所在的千橡市，也是近期震惊全美的酒吧枪击案的事发地。
 <br/>
 <br/>
 实景电视明星 金．卡戴珊：〝我们的心碎了，因为可怕的枪击案，也为那些失去生命和家园的人，以及目前正因灾难性火灾而被疏散的成千上万的人。〞
 <br/>
 <br/>
 被称为好莱坞喜剧女王的玛丽莎．麦卡锡获得〝人民偶像奖〞，她敦促观众为灾区捐款。
 <br/>
 <br/>
 演员 玛丽莎．麦卡锡：〝给洛杉矶消防局基金会的任何捐款，1美元，10美元，50美分，都将得到充分利用，都将发挥作用。〞
 <br/>
 <br/>
 多名好莱坞明星遭遇野火灾情冲击，演员巴特勒在社交媒体上贴出了他的家被大火烧毁后的景象。
 <br/>
 <br/>
 设计师维多利亚．贝克汉姆获得了〝人民时尚偶像奖〞，她在加州洛杉矶居住过。
 <br/>
 <br/>
 设计师 维多利亚．贝克汉姆：〝我的心真的和那些受到可怕火灾影响的每个人在一起，我向那些消防员表示钦佩，他们的付出不可想象。〞
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
