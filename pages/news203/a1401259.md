### GM计划关厂大裁员 川普放话要停止补助
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通用汽车.html" target="_blank">
  通用汽车
 </a>
 即将在北美关闭工厂并大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裁员.html" target="_blank">
  裁员
 </a>
 的消息惹恼了美国总统川普（特朗普），他周二在推特上放话称，美国政府救了通用汽车，该公司却押宝中国，政府将考虑停止所有给予该公司的补助。此言一出，通用汽车的股价应声下跌。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通用汽车.html" target="_blank">
  通用汽车
 </a>
 (General Motors)近日宣布要关停北美五座工厂，并计划
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裁员.html" target="_blank">
  裁员
 </a>
 1万5千。针对这个大整顿的计划，川普总统27日发表推文表达不满，〝美国救了通用汽车，这就是我们得到的感谢！〞川普写道，〝我们正考虑要删掉所有给予通用汽车的补助，包括电动车补助。〞
 <br/>
 <br/>
 川普还在推文中指出，〝通用汽车很多年前就押宝中国，在那边盖工厂，在墨西哥也有，我不觉得这笔投资最后能够获利。我在这里是为了要保护美国劳工。〞
 <br/>
 <center>
  <br/>

   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-29/p9148591a129343630-ss.jpg"/>

    美国通用汽车(General Motors)近日宣布要关停北美五座工厂，并计划裁员1万5千，这个消息惹恼了川普，他发推文表示正考虑取消联邦对通用汽车的补助。(Bill Pugliano/Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 <br/>
 川普的上述推文发出后，很快就在美国的股市上产生了明显的后效应，通用汽车的股价当天便应声下跌约2.6%。而此前一天（26日），通用汽车刚刚宣布该公司的精简成本计划时，还曾经让股价因此而一度走高。
 <br/>
 <br/>
 眼看势头不妙，通用汽车公司在川普推文发表之后赶紧也发表了一份声明，声称该公司未来将继续在美国维持坚实的制造部门。
 <br/>
 <br/>
 这份声明还特别表白说，通用汽车从2009年起，已经在美国国内投资了220亿美元，接下来还打算在电动车、自驾车领域增加新的就业机会。
 <br/>
 <br/>
 与此同时有消息称，加拿大总理杜鲁多(Justin Trudeau)已经打电话给川普，讨论通用汽车裁员与关厂的动作，因为通用汽车宣布的裁员计划已经在加拿大引起了政界和劳工界领袖的反弹。
 <br/>
 <br/>
 （记者李明报导/责任编辑：文慧）
