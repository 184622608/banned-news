### 美国流失太空制空权？ 太空军加速推进
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太空军.html" target="_blank">
  太空军
 </a>
 〞计划正快速推进，预计明年进入国会表决议程。专家认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太空制空权.html" target="_blank">
  太空制空权
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 至关重要，也是美国〝以实力维护和平〞的重要砝码。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 副总统 彭斯：〝太空和陆海空一样，是战争领域。〞
 <br/>
 <br/>
 美国副总统彭斯周二表示，白宫从2019年开始，将与国会合作，对2020年组建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太空军.html" target="_blank">
  太空军
 </a>
 的议案进行表决。
 <br/>
 <br/>
 专家指出，在现代军事中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太空制空权.html" target="_blank">
  太空制空权
 </a>
 至关重要，将决定地面战争的胜负。
 <br/>
 <br/>
 中国问题专家 马杰森博士：〝美国现在是全球军事部署，全球打击，全球性的决策。很多军事行动都是靠卫星拍到某一个地方的照片，然后导弹发射以后靠卫星制导，整个通讯系统各方面，全是依赖于外太空的卫星技术。如果任何一个国家对美国的外太空实施打击，对美国的军事、政治、经济都可能是瘫痪性的。〞
 <br/>
 <br/>
 中共近年加大太空军事化力度。从2007到2013年，中共3次测试反卫星导弹，击落了报废的气象卫星。
 <br/>
 <br/>
 美国军事专家Bill Gertz，2016年分析指出，中共正在打造4种太空战武器系统，除了反卫星导弹，还有激光武器、GPS干扰机和杀手卫星。有能力摧毁500枚美国卫星。
 <br/>
 <br/>
 中共官媒〝中国日报〞早在2008年就宣称，美国的太空优势正在流失。
 <br/>
 <br/>
 马杰森博士：〝某种意义上讲，美国不是先发性的，它是应对性的。前几届（美国）总统知道有这样的危险，但基于各种原因，并没有（应对）行为。那么这一届的川普总统，把这个提到一个非常高的高度。从根本上保护美国现有的军事政治利益。〞
 <br/>
 <br/>
 《纵览中国》主编 陈奎德：〝我比较支持的一个基本哲学就是，像中共，像很多专制国家，它们基本上只懂得一种语言，就是实力的语言。它如果认为它的实力已经和你平等，它一定要用这个实力破坏世界秩序。所以美国一定要有以实力做后盾的和平。〞
 <br/>
 <br/>
 外界认为，美国希望保护自己的卫星免受攻击，并发展出能准确攻击对方卫星的技术，增加震慑力。彭斯也重申，和平来自实力。
 <br/>
 <br/>
 陈奎德：〝美国作为全球第一的军事力量，在全球秩序里面，起了一个基本上的稳定作用。如果没有美国作为全球第一的军事力量，这个全球秩序恐怕要发生一些我们预料不到的改变。〞
 <br/>
 <br/>
 彭斯表示，太空军初期不会是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 第6军种，而是比军种低一级，类似军方特种部队司令部的架构。未来可能再扩充。目标是让美国重回太空主导地位。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
