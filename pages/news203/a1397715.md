### 川普否定蓝色浪潮 参院选举关键战盘点
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】距离美国2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 还有5天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统及两党高层都积极奔走，为各自的国会或州长的参选人拉票。外界普遍认为，本次中期选举，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 将很可能在参议院扩大优势。下面一起来看，当前参议院的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关键选战.html" target="_blank">
  关键选战
 </a>
 。
 <br/>
 <br/>
 本次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，国会参议院1/3、也就是33个席位将改选。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 今晚将在密苏里参加助选活动，这是继9月21日他第二次赴密苏里，这次是为州总检察长、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 联邦参议员参选人霍利（Josh Hawley）站台。目前霍利较对手民主党现任参议员麦卡斯基尔（Claire McCaskill），在民调结果上只有2%的差距。奥巴马则奔赴好莱坞为麦卡斯基尔募款。霍利要为中产家庭争回权益，比如减税；麦卡斯基尔主张加税和开放边境。川普2016年在密苏里州胜出19个点。
 <br/>
 <br/>
 另一个相当胶着的参院席位争夺在佛州，现任州长斯科特（Rick Scott）任期已满、今年竞逐联邦参议员，挑战民主党籍现任参议员纳尔逊（Bill Nelson）。目前二人的民调差距在2%。民主党想要保住现有这个席位，而共和党则将此看作在参院多赢一席的机会。目前，斯科特已投入超过6400万在这次竞选中。
 <br/>
 <br/>
 田纳西州共和党联邦参议员布莱克本（Marsha Blackburn）与民主党挑战者、前州长布雷德森（Phil Bredesen）的民调差距为6.8%。月初川普赴田纳西约翰逊市为布莱克本站台。布莱克本在国会职涯初期，以领导众议院共和党人调查〝美国计划生育协会〞（Planned Parenthood）而闻名。如果布莱克本能胜出，将成为田纳西州历史上第一位当选的女性联邦参议员。
 <br/>
 <br/>
 德州共和党联邦参议员克鲁兹目前民调领先对手奥罗克（Beto O'Rourke）6.5%，虽然民调始终领先，但差距并未拉大至两位数。政治数据分析机构TargetSmart的报告显示，大量30岁以下的选民支持奥罗克。
 <br/>
 <br/>
 美国总统 川普：〝坦白说，我认为已经不存在蓝色（民主党）浪潮了。我们做的相当不错。看样子我们可以赢下参议院，这非常重要。〞
 <br/>
 <br/>
 前副总统拜登今天则先后去到北达科他州及密歇根州分别为民调落后的现任民主党籍参议员海特坎普（Heidi Heitkamp）和斯塔贝诺（Debbie Stabenow）助选。
 <br/>
 <br/>
 美国民主研究所主任 Patrick Basham：〝普通美国人、大多数美国人并不关心谁会赢的政治斗争的胜利，他们在意的是自己会得到什么、国家会怎样。川普执政二年，经济繁荣，大部分人认为美国变得更好。这个大环境让民主党难以实施他们的计画，他们自己也没有有力的理念政策支撑、积极的办法拿出来。他们所做的就是批评川普和共和党人。〞
 <br/>
 <br/>
 新唐人记者李兰纽约报导
