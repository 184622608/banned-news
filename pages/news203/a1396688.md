### 连环可疑包裹 专家：试图撕裂美国
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 还有13天。川普总统针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_可疑包裹.html" target="_blank">
  可疑包裹
 </a>
 事件明确表示，任何形式的政治暴力在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 都没有土壤。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统 川普：〝我只想告诉你们，现在我们必须团结，大家要在一起，发出非常明确，强有力且准确无误的信息——任何形式的政治暴力或威胁在美利坚合众国都没有土壤。 〞
 <br/>
 <br/>
 川普已要求执法机构全面调查，并会在合适时间公布于众。
 <br/>
 <br/>
 本周起，包含有爆炸装置的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_可疑包裹.html" target="_blank">
  可疑包裹
 </a>
 被寄送到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 高级政客或幕僚家中，包括
 <br/>
 <br/>
 22日 民主党大金主索罗斯位于纽约卡托纳（Katonah）县的豪宅；
 <br/>
 23日 克林顿和希拉里在纽约查巴克（Chappaqua）县的住址；
 <br/>
 24日 奥巴马在华府的住址；
 <br/>
 <br/>
 周三，CNN位于曼哈顿时代华纳大厦的办公室也收到可疑包裹，导致紧急疏散。
 <br/>
 <br/>
 特勤局拦截下了寄送给克林顿和奥巴马的包裹。
 <br/>
 <br/>
 佛州大西洋大学政治学教授Marshall L. DeRosa：〝事件刚发生，还不知道是谁干的。但是很遗憾有这样的事，尤其现在离选举这么近。就是疯子或疯狂的组织，想撕裂美国人，我觉得很悲哀。〞
 <br/>
 <br/>
 美国执法领域专家 Rogers Canfield博士：〝无论是谁干的，处于何种目的，都是犯罪。时间点上相当巧妙，分散了大家对持续发生的暴民袭击共和党人的关注。〞
 <br/>
 <br/>
 不久前确认最高法院大法官卡瓦诺的过程，左派的表现被川普及共和党批评为国家耻辱。卡瓦诺就职后，希拉里10月9日在接受CNN采访时表示对共和党〝不能文明。〞奥巴马执政时期的司法部长Eric Holder则说要踢走共和党。
 <br/>
 <br/>
 这种鼓动暴力的言论再次被川普批评为〝暴民政治〞。
 <br/>
 <br/>
 佛州大西洋大学政治学教授Marshall L. DeRosa：〝不过迄今为止，做法上欠文明的是民主党，共和党一直保持文明。是民主党人在组织暴民，骚扰个人，打人，吓唬人。所以现在还不知道谁在后面主使。〞
 <br/>
 <br/>
 美国执法领域专家 Rogers Canfield博士：〝这件事真正的意义在于，我们需要文明和礼仪，无论大家观点有多么不同。〞
 <br/>
 <br/>
 新唐人记者李兰纽约报导
