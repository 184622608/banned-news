### 川习会生变？习近平刘鹤G20前分头访欧惹猜疑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】中美在APEC峰会上的激烈交锋，被怀疑或预示了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上中美首脑会晤的气氛。日前官方消息指，本应积极准备此次会晤的中方高层
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 ，在G20前将分头访问欧洲，引发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 生变的猜疑。美智库日前称，会晤有取消的可能。
 <br/>
 <br/>
 周一（11月19日），西班牙王室宣布，北京最高领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将从11月28日起访问西班牙，但没有说明访问几天，仅表示习近平将会见西班牙国王和首相，并拜访国会，以及在皇宫举行晚宴。
 <br/>
 <br/>
 次日（20日），中共外交部宣布，中共国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 将于11月25日至28日访问德国，并出席第八届中欧论坛汉堡峰会。
 <br/>
 <br/>
 而今年的G20高峰会将于11月30日至12月1日在阿根廷举行。香港《南华早报》此前首先报导消息指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 初步安排在G20开幕前一天（11月29日），随后又引述新消息称，会晤改成了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 后，于12月1日在当地举行晚宴。
 <br/>
 <br/>
 习近平28日访问西班牙的行程，凭空为川习会增加了变数。而关于刘鹤的行程，在中方向美方提交让步清单后，曾有消息指刘鹤将亲赴美国谈判，为川习会铺路。不过随后又有消息指，刘鹤和美方的谈判将改在G20的主办地。
 <br/>
 <br/>
 不过，中方突然宣布刘鹤25日访问德国，令人怀疑他是否还会赶赴阿根廷谈判。
 <br/>
 <br/>
 观察人士指，离G20峰会已不足十天，在中美对北京让步清单存在巨大分歧的情况下，刘鹤还要拿出至少3天访德，至少说明中方或已无意为川习会做认真准备。
 <br/>
 <br/>
 法媒指，中方或籍此向美国表达不满和强硬态度，中美贸易战又有山雨欲来之势。
 <br/>
 <br/>
 美媒报导，在美中关系紧张不断加剧和两国贸易谈判不确定之际，刘鹤访德旨在加强同欧洲国家的关系，以对抗美国的压力。
 <br/>
 <br/>
 在上周的APEC峰会上，美国副总统彭斯对北京进行了不留情面的抨击，引发与会国对北京的围堵，最后令峰会联合公报流产。此外，彭斯还宣布联合日本和澳大利亚，反击中共的〝一带一路〞，并从中方手中夺取了对巴布亚新几内亚一个海军基地的开发权，令北京铩羽而归。
 <br/>
 <br/>
 外界分析，APEC峰会的紧张气氛或将延续到十几天后的G20峰会。
 <br/>
 <br/>
 日前，美国智库战略暨国际研究中心（CSIS）就G20峰会举行吹风会，该智库高级顾问古德曼（Goodman）认为，川习会可能达成〝停火协议〞，但也有取消的可能性。
 <br/>
 <br/>
 此前美方曾放风称，如果中方对美方贸易要求的回应不能令人满意，川习会可能剔除贸易话题，而只聚焦朝鲜问题等争议不大的议题。
 <br/>
 <br/>
 彭斯则警告，中方如果不能在保护知识产权、强迫技术转移、限制外企进入中国市场、尊重国际规则及在国际水域航行自由等问题上做出显著的、实质的让步，美方准备在经济、外交和政治上对北京施加更大压力，新冷战就将开启。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
</div>
