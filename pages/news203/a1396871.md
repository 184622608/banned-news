### 可疑邮包满天飞 洛杉矶时报也中镖
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】近日，一些装有爆炸装置的包裹接连寄往民主党政要或幕僚家中，引发各界质疑。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洛杉矶时报.html" target="_blank">
  洛杉矶时报
 </a>
 称，24日该报也收到4个可疑信封，警方前往调查证实是虚惊一场。
 <br/>
 <br/>
 位于艾尔塞冈度（El Segundo）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洛杉矶时报.html" target="_blank">
  洛杉矶时报
 </a>
 发言人曼宁（Hillary Manning）24日说，过去也曾有类似古怪邮包寄到报社并接受检查，但并未引起任何安全顾虑。因为24日纽约CNN与圣地亚哥联合论坛报（San Diego Union-Tribune）和其他几个地方都接连收到可疑邮包，该报工作人员询问收发室员工是否发现〝奇怪邮包〞，收发室员工才说有收到4个信封。
 <br/>
 <br/>
 这4个信封捆绑在一起，收信人为〝乐慈百利坊甜品〞（knots berry farm sweets）， 回邮地址则是洛杉矶时报，信封上的标签是有人用印好的地址粘贴上的。但该报称，这些信并非从洛杉矶时报寄出。
 <br/>
 <br/>
 报社保安人员当天上午约11时15分报警，随后，警方疏散报社大楼部分人员，并封锁附近一些街道。
 <br/>
 <br/>
 洛杉矶县公共卫生局、艾尔塞冈度市警局和洛杉矶县警局联合组成一个调查危险物品小组，他们把4个信封放在报社大楼外的一个箱子，监测它们是否会释放有毒物，结果显示，这些信封没有排出有毒气体或辐射物质，也没有炭疽。
 <br/>
 <br/>
 直到中午过后，警方确定报社内没有发现爆炸物，才解除街道封锁，让员工返回大楼。
 <br/>
 <br/>
 本周起，包含有爆炸装置的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_可疑包裹.html" target="_blank">
  可疑包裹
 </a>
 接连被寄送到民主党高级政客或幕僚家中，从24日到25日，包括美国前总统奥巴马和克林顿以及前国务卿希拉里在内的多位民主党政要，先后收到了疑似炸弹的邮包。
 <br/>
 <br/>
 民主党重量级金主索罗斯(George Soros)和美国有线电视新闻网(CNN)纽约分社也收到了疑似有爆炸物的包裹。此外，还有寄给民主党加州联邦众议员华特斯(Maxine Waters)的可疑包裹在国会邮件收发处被发现；另有2个分别寄给前副总统拜登和影星劳勃狄尼洛的可疑包裹也被有关机构发现。
 <br/>
 <br/>
 24日上午9时前不久，圣地亚哥联合论坛报也因收到一个可疑邮包而被迫疏散编辑部。 民主党加州联邦参议员贺锦丽（Kamala Harris）的办公室也在同一座大楼。一个专门检查危险物品的小组前往调查后证实邮包无害。
 <br/>
 <br/>
 这些可疑包裹引发了社会疑虑与担忧。多名共和党要员已相继发声，谴责寄送炸弹邮包的暴力行径。有舆论质疑：恰逢期中选举即将开始之际，这些明显带有恐怖主义色彩的行动，显然是在给共和党和川普添乱，或许这恰恰就是寄送邮包的人挑起事端想达到的目的。
 <br/>
 <br/>
 美国司法决定立案调查这起连环可疑包裹事件，定义为恐怖主义袭击。截至发稿，没有人声称犯案，也没有人被捕。
 <br/>
 <br/>
 总统川普（特朗普）周三呼吁美国民众团结一致，并表示〝任何形式的政治暴力威胁行动，在美国都无法立足。〞他也呼吁媒体停止敌意和〝假攻击〞。他也表示，公众人物有责任缓和他们的言词。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
