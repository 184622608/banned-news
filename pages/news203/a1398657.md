### 为21岁庆生遇屠杀 加州枪击案幸存者忆惊魂夜
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千橡镇音乐吧.html" target="_blank">
  千橡镇音乐吧
 </a>
 7日深夜发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击.html" target="_blank">
  枪击
 </a>
 案，导致至少13死21伤。21岁的大学生王娜莉当时正在现场与朋友庆生，回忆自己侥幸逃离现场的惊魂过程，仍让她不寒而栗。
 <br/>
 <br/>
 王娜莉（Nellie Wong）告诉洛杉矶时报，当天她邀请朋友一起到音乐吧庆生，她还刻意穿上棕色牛仔靴和牛仔外套。当她独自坐在桌子旁时，突然听到枪响。她立即拿起夹克，跌跌撞撞地躲在高脚凳旁。慌乱中还把膝盖给刮破。这时，她在烟雾中看到了穿着黑色连帽衫和黑裤子，还戴着黑围巾的枪手冲了进来。
 <br/>
 <br/>
 王娜莉说，〝谢天谢地，他没有看到我。〞她立刻停止移动，屏住呼吸。并一直躲在酒吧里的矮墙后方，直到警方帮助她离开现场。
 <br/>
 <br/>
 另一位幸存者，王娜莉的朋友、19岁的沃瑞尔（Chyanne Worrell）说，她当时躲在桌子底下，听到有人说要她快跑，立即跑了出去。她说，〝感谢上帝，让我成功逃出。〞
 <br/>
 <br/>
 幸运的是，她们5个当晚在酒吧庆生的朋友都逃过一劫。沃瑞尔回忆当时紧张的心情说，在找到彼此之前，感觉就像永远，当时真的担心死了。
 <br/>
 <br/>

 <br/>
 <br/>
 事发于当地时间周三晚11:20左右，加利福尼亚路德大学（California Lutheran University）等3所大学的数百名学生正聚集在千橡市一家名为〝Borderline Bar &amp; Grill〞的酒吧内欢度〝乡村之夜〞音乐节。
 <br/>
 <br/>
 一名身穿黑衣的青年男子突然悄悄入内，并持一把半自动手枪向人群射击。文图拉县（Ventura County）警方称，枪手至少射出30发子弹，并使用烟雾弹迷惑人们的视线。
 <br/>
 <br/>
 事件导致至少13人死亡，21人受伤，包括枪手及1名以身殉职的即将退休的警察。
 <br/>
 <br/>
 据福克斯新闻（Fox News）周四上午报导，枪手是28岁的大卫．龙（David Long），是前海军陆战队员。警方在酒吧入口附近的一间办公室内发现了他的尸体，据信他是作案后自戕而亡。
 <br/>
 <br/>
