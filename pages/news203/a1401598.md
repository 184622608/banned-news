### 助〝一马〞洗钱干扰调查 美司法部前官员认罪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】美国司法部前官员希金巴坦因协助马来西亚国家投资基金〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一马.html" target="_blank">
  一马
 </a>
 〞发展公司（1MDB）弊案，11月30日认罪。
 <br/>
 <br/>
 路透社引述美国司法部声明报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一马.html" target="_blank">
  一马
 </a>
 公司的设计者曾聘请当时任职司法部资深国会事务专家的希金巴坦（George Higginbotham），以协助从国外银行帐户转汇数千万美元到美国。
 <br/>
 <br/>
 报导中说，他所转汇的资金随后作为游说用途，以解决美国司法部在调查一马发展公司被挪用数十亿美元阴谋的民事与刑事诉讼。
 <br/>
 <br/>
 希金巴坦承认曾密谋针对这些资金的来源及目的，对银行作出虚假陈述。
 <br/>
 <br/>
 司法部也宣称，希金巴坦并未参与调查，无法对调查产生任何影响。
 <br/>
 <br/>
 马来西亚前政府除了在大马募集〝一马〞国家建设开发基金外，还在美国及其它国家大量募资，后经现任政府及法部介入调查后发现，前首相兼前财政部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳吉布.html" target="_blank">
  纳吉布
 </a>
 （Najib Razak）将庞大的〝一马〞基金转到自己的私人帐户。
 <br/>
 <br/>
 由于该基金曾在六个国家发行债劵集资，其中还三度在美国发债募资，因此美国有许多受害者，于是美国司法部开始对〝一马〞丑闻展开包括刑事与民事上的调查。
 <br/>
 <br/>
 美国司法部调查后发现，从2009年至2014年间，纳吉布及其他涉案高层官员，总共从〝一马〞基金中盗走约45亿美元的资金。
 <br/>
 <br/>

 美国司法部还提出另一份起诉书，要求冻结大马富豪刘特佐（Jho Low）在美国银行存放7,300万美金的户头。据希金巴坦供称，这是他协助刘特佐洗钱到美国银行存放。
 <br/>
 <br/>
 承办此案的美国检察官已在本月初公布两名涉及〝一马〞洗钱丑闻的高盛银行高层主管，并对他们提起刑事起诉。
