### 匹兹堡犹太教堂枪击案 川普严词谴责 民众烛光夜悼
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】在美国宾夕法尼亚州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_匹兹堡.html" target="_blank">
  匹兹堡
 </a>
 市，周六发生的犹太教堂枪击事件后，匹兹堡和华盛顿特区民众，当晚举行了烛光守夜活动，纪念受害者。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统当天发表讲话，严厉谴责这一仇恨犯罪，白宫降半旗为受害者默哀。
 <br/>
 <br/>
 10月27日（星期六）上午，46岁的白人枪手罗伯特-鲍尔斯在美国宾夕法尼亚州城市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_匹兹堡.html" target="_blank">
  匹兹堡
 </a>
 的〝生命之树〞犹太教堂，向参加教堂活动的人群开枪射击，并大喊〝所有犹太人必须死亡〞，造成11名信徒死亡，另有6人受伤，其中包括4名警察。被送往医院的受伤者中包括1名61岁妇女、1名70岁男子、1名55岁警官。
 <br/>
 <br/>
 涉嫌枪手枪手鲍尔斯在与特警部队的枪战后被捕，因受伤被送医救治、情况稳定。鲍尔斯曾在网上发表很多反犹太人的帖子，并在周六当天早晨开枪之前还发过一帖。联邦检察官指控他犯有包括暴力和枪支罪在内的29项罪名，并违反了美国民权法。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在第一时间发推文，对事件表示关注，并说〝事件比想象的更严重〞。
 <br/>
 <br/>
 当天他在伊利诺伊州的一场竞选集会上，又严词谴责这一仇恨犯罪。
 <br/>
 <br/>
 美国总统 川普：〝这个邪恶的反犹攻击是对我们所有人的攻击，是对人性的攻击。反犹太仇恨不能被忽视、不能被容忍、不允许持续。它必须受到抵制和谴责，不论它丑陋的出现在哪里。〞
 <br/>
 <br/>
 匹兹堡和华盛顿特区的民众星期六晚上冒雨为受害者举行烛光守夜活动。
 <br/>
 <br/>
 匹兹堡居民琳达-马汀（LINDA MARTIN）：〝我们必须阻止（枪击）。我们的领导者们必须阻止（枪击）。这需要领导才能阻止，而我们的领导者们却没有阻止。我们是一个文明社会。我们必须保持文明。〞
 <br/>
 <br/>
 华盛顿特区的烛光守夜则在白宫前举行。反诽谤联盟和犹太公共事务委员会（The Anti-Defamation League and Jewish Council for Public Affairs）称这次枪击是美国历史上对犹太人社区最致命的攻击。
 <br/>
 <br/>
 大华府地区犹太社区关系委员会副主任西格尔（GUILA FRANKLIN SIEGEL）：〝安息日早晨在犹太教堂遭到攻击，是每个犹太人最可怕的噩梦。当你和家人在一起礼拜，在和平的一天，在休息日，和社区一起的时候。〞
 <br/>
 <br/>
 白宫降半旗为遇难者致哀。
 <br/>
 <br/>
 新唐人记者徐哲、天羽综合报导
