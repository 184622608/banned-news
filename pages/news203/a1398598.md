### 美中期选举城乡差异大  未来两党仍可合作
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 基本尘埃落定。共和党与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 可谓各有所获。专家认为，这次选举再次反映出美国城乡之间巨大的理念差异。不过未来两年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_两党.html" target="_blank">
  两党
 </a>
 之间依然有合作空间。
 <br/>
 <br/>
 这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 相对以往，总体投票率大幅上升，从过去的40-42%，提高到46-48%。专家认为，最终结果
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_两党.html" target="_blank">
  两党
 </a>
 可谓各有所获。
 <br/>
 <br/>
 美国企业研究所美国政治研究员 Michael Barone：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 赢得众议院多数，这很重要，让他们有机会推动其政策，可抵制其所不同意的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政策。共和党赢得更多参议院席位，让他们更容易确认法官提名，包括最高院大法官，如果再有空缺，总统会再提名。〞
 <br/>
 <br/>
 虽然这次选举民主党重掌众议院，但专家认为，左派媒体预期的蓝潮很大程度上并未实现。
 <br/>
 <br/>
 美国企业研究所美国政治研究员 Michael Barone：〝民主党曾期待蓝潮，但他们只得到全食蓝潮。他们只在有全食食品店的地方(大城市)做得不错。〞
 <br/>
 <br/>
 这次选举也被视为对川普施政的一次公投。三分之二选民表示，投票时有将川普作为一个考虑因素。同时，选举结果再次反映出美国城乡之间在理念上存在很大差异。
 <br/>
 <br/>
 跨党派政策中心民主项目主任 John Fortier：〝响村地区越来越倾向共和党，郊区和城市则更倾向民主党。〞
 <br/>
 <br/>
 虽然未来两年，两党将分别掌控国会参众两院，专家也预测，川普依然可以运用其交易艺术，寻求两党合作空间。
 <br/>
 <br/>
 跨党派政策中心民主项目主任 John Fortier：〝川普总统在有些方面是可交易的，他喜欢达成协议。如在基础建设问题、建设桥梁公路上，可能总统会愿意和佩洛西议员做些交易，达成对彼此都有利的结果。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
