### 普选在即 刘醇逸自曝被FBI〝红钱行动〞调查
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】近日，在一场纽约的辩论会上，丑闻缠身的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘醇逸.html" target="_blank">
  刘醇逸
 </a>
 ，自曝几年前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 对他展开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_调查.html" target="_blank">
  调查
 </a>
 的细节。
 <br/>
 <br/>
 在10月30日〝贝赛小丘公民协会〞（Bayside Hills Civic Association）举办的辩论会上，第11选区州参议员民主党候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘醇逸.html" target="_blank">
  刘醇逸
 </a>
 （John Liu）在会上自曝，他在2013年竞选市长时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 对他秘密
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_调查.html" target="_blank">
  调查
 </a>
 的行动，名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红钱行动.html" target="_blank">
  红钱行动
 </a>
 〞（Operation Red Money），当时FBI怀疑中共在暗中支持刘醇逸，并影响选举。
 <br/>
 <br/>
 刘醇逸当时的竞选募款人，时任美国福建同乡会副会长潘心武，竞选财务主管、北京同乡会会长侯建利的女儿侯佳（Jenny Hou）先后因非法募款被逮捕。随后，潘心武被判刑4个月及三年监外看守，侯佳被判刑10个月及三年监外看守，轰动纽约政坛。
 <br/>
 <br/>
 刘醇逸2013年竞选市长失败后，在2014年转战第11选区，挑战州参议员艾维乐，失败告终。
 <br/>
 <br/>
 在今年11月6日的普选中，刘醇逸作为民主党候选人，再次竞选第11选区纽约州参议员职位。
 <br/>
 <br/>
 他提倡的政见包括娱乐大麻合法化、不逮捕公开吸食大麻者、支持征收地税不设上限、支持纽约州堕胎合法化，至于特殊高中考试制度（SHSAT），刘醇逸表面支持，却不认为考试是唯一的录取标准。
 <br/>
 <br/>
 现任纽约州参议员艾维乐在普选日，将作为独立党（Independence）和妇女平等党（Women's Equality）的候选人登上选票，不过，选民需要注意，只能勾选其中一个党派，否则选票将会作废。
 <br/>
 <br/>
 新唐人记者柯婷婷纽约报导
