### 川普印第安纳州助选 当地人：川普代表人民声音
------------------------

<div class="wysiwyg">
 【新唐人2018年11月03日讯】周五晚，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 和副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 ，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印第安纳州.html" target="_blank">
  印第安纳州
 </a>
 印第安纳波利斯出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选集会.html" target="_blank">
  助选集会
 </a>
 ，并发表演说。当地民众表示，川普代表了人民的声音、坚守着美国的价值观。
 <br/>
 <br/>
 周五晚上，川普在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印第安纳州.html" target="_blank">
  印第安纳州
 </a>
 印第安纳波利斯，首先提到最新的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业报告.html" target="_blank">
  就业报告
 </a>
 。
 <br/>
 <br/>
 美国总统 川普：〝仅上个月一个月，我们就增加了25万个工作，将近60万美国人重返工作岗位。你注意到了吗，在伟大的印第安纳州，我们让众多钢铁工人返回工作岗位，这是之前没有人能想像的。〞
 <br/>
 <br/>
 川普执政至今，创造了450万个新工作机会。
 <br/>
 <br/>
 副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 表示，相信川普政府会给美国人带来安全和繁荣。
 <br/>
 <br/>
 美国副总统 彭斯：〝我相信我们能让印第安纳更安全、更繁荣。这个人能让美国再次伟大。〞
 <br/>
 <br/>
 当地民众挤满了体育场，表达对川普的支持。
 <br/>
 <br/>
 退休职业女性 Ponny：〝他代表了人民的价值观、知道什么对人民最好。他不介意人们说他什么、对他做什么，他依然为我们而战、为了自由、为了边境、为了经济，还有信仰自由。〞
 <br/>
 <br/>
 印第安纳波利斯市长参选人 John Schmitz：〝他爱美国，他真的在意每个个体生活的情况。〞
 <br/>
 <br/>
 印第安纳波利斯人 Tony Adrian（民主党人）：〝没想到川普吸引了这么多的人。 一些关键的社会议题，包括移民、福利、债务和社会安全等，我觉得最起码这一点川普做的不错，他提出这些问题，我们需要讨论这些问题、需要有对话。〞
 <br/>
 <br/>
 受访者都提到川普代表了美国的传统价值观，包括努力工作、重视小商业、合法移民等。
 <br/>
 <br/>
 印第安纳波利斯人 退伍军人 Chris：〝我觉得认为移民可以来美国，但他们不能非法进来。不是来着白吃白喝，而要努力工作、感恩并回报美国。我来这是因为我相信美国曾经伟大、现在仍然伟大、并能更伟大如果我们做正确的事。〞
 <br/>
 <br/>
 印第安纳波利斯当地学生：〝我们非常认同川普推进的政策，比如必须的强有力的边境政策、扶持小商业也非常重要。〞
 <br/>
 <br/>
 印第安纳波利斯人 制造业小业主 Carlos：〝我带着孩子来的，因为觉得让他们了解政治运作很重要，知到从政的人他们的观点；了解这套系统、虽然不完美、但各司其职；尽量不随人摇摆，而是要看事实，让他们自己判断决定。 〞
 <br/>
 <br/>
 今天川普在蒙大拿州和佛州还有两场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选集会.html" target="_blank">
  助选集会
 </a>
 。
 <br/>
 <br/>
 新唐人记者唐明镜、陈虎、李兰印第安纳州采访报导
</div>
