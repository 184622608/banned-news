### 蓬佩奥:美将强力回应中共每个恶行 包括迫害宗教信仰
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月28日讯】美中对抗正走向新冷战，美方言辞也越来越直接。美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 日前表示，美国将强力回应中共在每个领域的不轨行为，而不再仅仅是口头批评。他特别提到，川普（特朗普）总统已要求严肃处理中共对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教信仰.html" target="_blank">
  宗教信仰
 </a>
 的迫害。
 <br/>
 <br/>
 据《美国之音》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 周五（10月26日）接受美国电台采访时谈到，川普政府开始意识到并将严肃应对美中竞争关系。
 <br/>
 <br/>
 蓬佩奥指出，中共利用〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞贿赂世界各国，损害各国人民以及美国的利益；中共的贸易政策、南海军事扩张、暴力镇压宗教人士、偷盗知识产权、政府补贴、强迫技术转让等，全部都是恶意行为，与美国价值观抵触。
 <br/>
 <br/>
 他说，美国政府将对中共的这些行为给予强烈的反击，并会持续下去。他还强调，美国会在每一个领域强力应对中共的每一个挑战。
 <br/>
 <br/>
 蓬佩奥特别提到新疆的〝再教育营〞及中共对基督教等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教信仰.html" target="_blank">
  宗教信仰
 </a>
 的打压，并透露川普总统已经要求美国政府严肃处理这些问题。
 <br/>
 <br/>
 他说，这意味着将不再仅仅是批评那些不尊重人权的国家，而是要动用美国和世界的影响力来应对这些对基本人权的挑战。
 <br/>
 <center>
  <br/>

   <br/>
   <font size="-1">
    美媒报导截图。
   </font>
  </a>
  <br/>
 </center>
 <br/>
 除了新疆少数民族和基督徒，美国政府还多次公开提及中共对法轮功的残酷迫害。
 <br/>
 <br/>
 美国国会暨行政当局中国委员会（CECC）10月10日发布的年度报告指，今年是17年以来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人权.html" target="_blank">
  中国人权
 </a>
 状况最糟的一年。其中提到中共对法轮功学员的监禁和虐待，以及活摘器官暴行。
 <br/>
 <br/>
 日前，美国参议院司法委员会主席查克・格拉斯利在美国《政治杂志》发表文章，谴责中共严重侵犯宗教群体的人权，并称为宗教自由而战应是美中关系的核心部分。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
