### 【微视频】美国中期选举不考川普而是考美国人
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】2018年11月6号美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 将会改选众议院、部分州长和部分参议员，是关乎美国未来政治版图的大事，美国选民也是热情高涨，已经有2200万到3000万人提前投票了。根据提前投票的14个州的情况来看，共和党在12个州领先，民主党只在俄勒冈和爱荷华州领先。咱们微视频之前也说过一个华人学者的分析，那就是根据数据模型，如果民主党没有8%的民调领先就会输，现在看来这位华人学者是很靠谱的，共和党继续掌控众议院和参议院，美国大选的钟摆效应就会停摆。
 <br/>
 <br/>
 在11月6号之前还有很多因素在左右大选的情况，咱们着重说两个媒体忽悠的例子。首先，11月3号《华盛顿时报》说，指控大法官性侵的其中一个女人已经承认编造谎言。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在推特上发表了评论，而当初对卡瓦诺炮轰的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_左派.html" target="_blank">
  左派
 </a>
 媒体集体沉默。为什么呢？在国会说谎不光是道德问题，而且是法律问题，据说FBI已经介入调查。同时，利用这件事情不断为难卡瓦诺的民主党参议员就处于危险的境地，做了这种事情还有脸竞选吗？所以左派媒体尽量淡化这条新闻。
 <br/>
 <br/>
 这件事情也为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国人.html" target="_blank">
  美国人
 </a>
 敲响了警钟，媒体和某些政客被社会主义已经渗透到了令人发指的地步，为了达到掌权的目的，编造谎言搞宣传，这已经彻头彻尾的共产党行为了。卡瓦诺一个大男人在参议院当着妻子和女儿的面被人污蔑，脸憋得通红，这是何等的耻辱！咱们也是希望卡瓦诺经过这次磨练能坚定的保卫美国传统价值。
 <br/>
 <br/>
 第二是中美洲非法移民大篷车挺进美国事件。媒体说上万名洪都拉斯等中美洲国家的人乘坐大篷车浩浩荡荡的向美国进发。10月25号，美国副总统彭斯认为大篷车是由海外的左翼集团资助的。他说，〝大篷车是由洪都拉斯的左翼团体组织的，部分由委内瑞拉资助，这是洪都拉斯总统（胡安‧奥兰多‧埃尔南德斯）说的，他亲自告诉我的〞。
 <br/>
 <br/>
 这个事件有三个疑点，首先是大篷车的汽油和租用费用；其次是事件发生在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 之前的时间点；最后是移民们不肯留在墨西哥接受救济，目的地非常确定就是美国。从这三点来看这些人不是难民而是有目的地的移民。左派挑起这个事，就是让川普总统两面为难，如果川普不肯接受这批人，就会丧失拉丁美洲裔美国人的选票；如果接受了，就会丧失了维护美国的立场。
 <br/>
 <br/>
 左派媒体把各种稿子都准备好了。总统派军队阻拦，那就就用冷血、无情的稿子赚取公众的同情票；总统软化处理，那就是打爱国牌。左派是不管美国未来的，或者说他们希望美国越来越来乱、越来越穷。因为贫穷和混乱才能忽悠民众接受共产主义，现在美国人有财产和法治，不好忽悠。
 <br/>
 <br/>
 在这各种忽悠因素的作用下，这次中期选举主要是考验政客吗？不是的，考验
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国民众.html" target="_blank">
  美国民众
 </a>
 的，大家能不能不被左派忽悠就成了这次考试的根本，这是大选最后结果的变数。
</div>
