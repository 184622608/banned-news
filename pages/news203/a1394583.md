### 在美记者致信川普 吁保护记者在华家人
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大纪元时报.html" target="_blank">
  大纪元时报
 </a>
 》英文版记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾铮.html" target="_blank">
  曾铮
 </a>
 ，日前致信
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统，讲述中共剥夺她母亲来美探亲的权利。她呼吁白宫，保护记者在中国的家人，并帮助结束中国的人权迫害。
 <br/>
 <br/>
 彭斯：〝我们很高兴看到更多的记者报导真相，不用恐惧也没有偏袒〞。
 <br/>
 <br/>
 美国副总统彭斯，鼓励记者报导真相而无需恐惧。不过，《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大纪元时报.html" target="_blank">
  大纪元时报
 </a>
 》英文版驻华盛顿DC记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾铮.html" target="_blank">
  曾铮
 </a>
 ，在发表了一系列揭示中共迫害人权及渗透美国的深度报导后，中共剥夺了曾铮母亲来美国旅行的权利。
 <br/>
 <br/>
 曾铮因信仰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 ，被中共列入入境黑名单无法回国。她的母亲则已高龄76岁，越来越难承受长途旅行，她将这次访美视为母女最后一次团聚的机会。
 <br/>
 <br/>
 曾铮：〝10月1号那天早晨，打开电脑一看，她给我发了一个短信，就是说‘我在上海浦东机场被拦住了，海关告诉我，我的护照被绵阳市公安局取消了，我在这里走投无路了’。等她回到四川绵阳的家里，我才打电话过去，她就在电话里一直地哭。然后有个时候给我写个Email说，‘看来只能把我逼疯逼死算了，这辈子我们还能再相见吗？’非常非常难过。〞
 <br/>
 <br/>
 曾铮今年7月加入英文《大纪元时报》。她目前无法确认，中共注销剪毁她母亲的护照，是中共系统收紧护照政策的一部分，还是因为她信仰法轮功、并报导有关中国的真实消息，而又一次遭到中共打压。无独有偶，法轮功学员王晓丹长期在国际社会讲述中国人权现状，2016年，她父亲的护照也被中共注销。
 <br/>
 <br/>
 曾铮：〝作为法轮功学员和法轮功学员的家属，这么多年不能够享有正常的回国的权利，或者是拥有一本护照的权利。中共一切都是这种黑箱操作，到现在它就告诉你‘取消了’，为什么？没有任何人给你解释。甚至没有任何人提前知会你一声。所有往返的奔波、买机票花的钱等于都白费了，然后最后一刻遭受这么大的精神打击。〞
 <br/>
 <br/>
 10月7日，曾铮致信
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统，呼吁帮助她母亲重新获得护照；并请求调查绵阳市公安局长王明华，是否符合美国《马格尼茨基人权问责法》制裁的人权恶棍条件，如果符合，呼吁将他列入制裁名单。
 <br/>
 <br/>
 曾铮：〝这个迫害进行这么多年了，一定是要有个人，来对每一项具体的迫害人权的罪责承担责任。如果不承担责任，就这么不了了之，那么还有更多的人选择继续作恶。〞
 <br/>
 <br/>
 她也呼吁川普帮助，结束中共对法轮功长达19年的迫害。
 <br/>
 <br/>
 曾铮：〝前几天副总统彭斯在发表美国对中国的战略的讲话的时候，提到希望中国人民能拥有宗教的自由。那么我也希望川普总统关注过去十九年以来，中共对法轮功的这种最残酷的镇压。如果（对）法轮功的迫害不能够结束，那么中国的宗教自由根本就无从谈起。〞
 <br/>
 <br/>
 曾铮曾是中国国务院发展研究中心的研究人员。2000年，她因修炼法轮功，被中共劳教一年。她出国后，长期撰写专栏讲述有关中国的真实消息。中共国安人员一直向她的母亲间接施压。
 <br/>
 <br/>
 她的遭遇并非个案。美国自由亚洲电台5名记者报导新疆集中营后，中共抓捕了他们20多位家人。
 <br/>
 <br/>
 曾铮：〝以文明的方式来行事，他才会被这个世界接纳。现在用这样的方式来应对国际局势的变化，真的除了加快共产党的灭亡之外，没有其它任何帮助。〞
 <br/>
 <br/>
 新唐人记者林澜、约克采访报导
</div>
