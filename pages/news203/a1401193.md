### 川普前竞选总干事认罪协议瓦解 穆勒通俄调查遭遇重挫
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】美国特别检察官罗伯特．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 办公室最近指控保罗．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马纳福特.html" target="_blank">
  马纳福特
 </a>
 撒谎，违反了他此前与联邦检察官达成的控辩协议，并要求法院重新考虑对他的量刑时间。舆论认为，这意味着马纳福特将丧失在任何案件中充当证人的资格，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 的调查，这对穆勒来说是一次重大挫折。
 <br/>
 <br/>
 据《美国之音》11月28日的报导，美国特别检察官罗伯特．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 的办公室最近对外宣称，曾经担任川普竞选团队总干事的保罗．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马纳福特.html" target="_blank">
  马纳福特
 </a>
 (Paul Manafort)向调查人员撒谎，违反了与联邦检察官达成的控辩协议。
 <br/>
 <br/>
 报导表示，穆勒办公室本月26日晚间提交的法庭文件声称，由于马纳福特〝在各项事务上向联邦调查局和特别顾问办公室撒谎，因而触犯了联邦法律〞。
 <br/>
 <br/>
 穆勒团队并未详述马纳福特撒谎的内容，但他们要求法院根据这一新的情况来考虑对马纳福特的〝量刑时间〞。根据联邦指导原则，马纳福特面临的判刑可能会比之前的更严厉。
 <br/>
 <br/>
 针对特别检察官方面的上述指控，辩护团队回应称，马纳福特〝相信自己提供了真实的信息〞，并试图〝履行他的合作义务〞 。
 <br/>
 <br/>
 报导表示，近两个月来，马纳福特一直被关押在监狱，他与特检官团队至少会面了9次，但调查人员周一提交的文件中却声称，他们无法从马纳福特处获得任何可以核实的信息。
 <br/>
 <br/>
 由于被指控金融犯罪，马纳福特将面临可能长达10年的监禁，而指控他的特别检察官穆勒，正在调查川普竞选班子是否与俄罗斯联手干预了美国2016年大选。
 <br/>
 <br/>
 之前，马纳福特曾同意与检方合作，〝充分、真实和完整地回答问题〞。作为交换，检方撤销了对他的部分指控，并可能会要求对他从轻宣判。
 <br/>
 <br/>
 从目前的情况来看，显然马纳福特给出的回答并不是穆勒团队希望获得的答案，因此宣布与他的合作协议作废。
 <br/>
 <br/>
 华府新闻网站The Hill日前发文分析指出，马纳福特的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_认罪协议瓦解.html" target="_blank">
  认罪协议瓦解
 </a>
 ，对特别检察官穆勒的通俄案调查是一个大的打击。
 <br/>
 <br/>
 因为在穆勒调查的〝通俄〞案中，马纳福特被视为极具潜在价值的证人，穆特团队原本指望马纳福特作为证人，可以指控川普的女婿库什纳、川普长子小唐纳2016年6月曾在川普大楼与一名俄罗斯律师会面，当时那名俄国律师声称拥有民主党总统候选人希拉里‧克林顿的黑材料。但如今穆勒小组公开宣布马纳福特撒谎，等于把他拖下了证人席。
 <br/>
 <br/>
 事态发展至此，意味今后马纳福特的证词将不能用来起诉任何人，他也不太可能在任何审判中出庭作证。
 <br/>
 <br/>
 针对上述情况，美国总统川普27日早晨连发多条〝推特〞抨击穆勒团队。其中一条川普写道，〝虚假的政治猎诬还在继续，但穆勒和他的愤怒的民主党人团伙只看事情的一面而不看另一面。等到事实出来之时，可以看到他们对待人们的方式是多么可怕和恶毒，由于人们拒绝撒谎的而要毁掉他们的生活。〞
 <br/>
 <br/>
 川普还在推文中批评说，〝假新闻媒体将鲍勃．穆勒奉为圣徒，但他实际上恰恰相反，他给我们的刑事司法系统造成了极大损害。〞
 <br/>
 <br/>
 上周，川普向特检官团队提交了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 〞调查问题的书面答覆，预计穆勒将在短期内结束他的调查。
 <br/>
 <br/>
 据公开的资讯，马纳福特是华盛顿特区政坛资深圈内人。他1949年4月1日出生于康涅狄格州，乔治城大学法律系毕业，曾效力于共和党前总统里根、老布殊。2016年4月，马纳福特特应聘加入川普总统竞选团队，担任竞选团队的总干事。
 <br/>
 <br/>
 美国司法部2017年5月17日任命FBI前局长穆勒为特别检察官，调查川普的竞选团队成员是否与俄罗斯政府有过不正当接触，是否存在合谋或利益输送等情况。之后，马纳福特成为了穆勒调查的一个重要目标。同年7月，FBI人员突击搜查了马纳福特特位于弗吉尼亚州的住宅。
 <br/>
 <br/>
 2017年10月30日，美国司法部特别检察官罗伯特．米勒宣布，保罗．马纳福特、里克．盖茨（Rick Gates）和乔治．帕帕佐普洛斯等三人，在2016年大选期间和俄罗斯有不正当接触，涉嫌犯罪。三人受到的指控还包括洗钱、密谋反美、作伪证和伪报、瞒报银行信息等。
 <br/>
 <br/>
 其后，帕帕佐普洛斯承认曾向联邦调查局调查人员作伪证，但马纳福特和盖茨对指控予以否认。
 <br/>
 <br/>
 2018年8月21日，由12人组成的一个评审团在维吉尼亚州亚历山德里亚裁定，马纳福特被指控的所得税报税造假、银行诈骗等8项罪名成立，而这些起诉的罪名与穆勒对2016年大选川普涉俄的调查并非无关联。马纳福特则否认了所有的指控。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
