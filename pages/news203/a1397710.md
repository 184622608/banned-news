### 大篷车难民仍北进 川普提改革庇护制度
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】近日，在墨西哥向美国行进的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 估计最多达到6000人。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）已下令向墨西哥边境加派军人到1万5千人。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：〝没有国家会允许失控的大批人群涌向边境，但这正在发生着。〞
 <br/>
 <br/>
 根据周四的最新消息，经过墨西哥想前往美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 有四个团，总人数可能高达6000。
 <br/>
 <br/>
 美国总统 川普：〝我的政府正要完成一项计画，以终结对我们
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_庇护制度.html" target="_blank">
  庇护制度
 </a>
 的猖獗滥用。唯一终结这危机的永久性解决办法，来确保我们国家的主权，就是国会要克服开放边境的做法。〞
 <br/>
 <br/>
 川普批评民主党政策是造成边境开放的主因，呼吁选民在中期选举时为共和党投票。
 <br/>
 <br/>
 目前，美墨边境已有国土安全部安排的边境巡逻队（U.S. Border Patrol）、移民与海关执法局（ICE）等部门的执法人员。国防部表示，最多可派8000人到边境支援执法，而目前已有超过7000名士兵部署到边境。而川普总统则希望最多可以派遣15,000名士兵。
 <br/>
 <br/>
 美国国防部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蒂斯.html" target="_blank">
  马蒂斯
 </a>
 ：〝我们派兵员过去是支持国土安全部，他们需要军人辅助边境行动。我们在风暴灾害后也这样做，我们现在这样做是支持国土安全部。〞
 <br/>
 <br/>
 装备有器械设备的美国军人已抵达圣安东尼奥，并在德州开始行动。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
