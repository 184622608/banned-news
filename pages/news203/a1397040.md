### 蓬佩奥谈一带一路：输出贿赂  伤害百姓
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 ，周五接受电台节目采访，他表示中共使用贿赂手段推广〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞项目，损害当地民众利益，美国将对此，予以强力反制。
 <br/>
 <br/>
 周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 接受了著名电台节目主持人休伊特的专访，其中重点谈到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 对北京的政策。
 <br/>
 <br/>
 蓬佩奥对此表示，本届政府自始至终，都对来自中共的威胁有清醒的认识。他还重点回答了针对〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞问题。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝我提醒其他国家，我们欢迎与中国进行公平互惠的商业竞争。但当中共使用贿赂手段笼络高层领导人，以换取伤害当地民众的基础建设项目时，如此打造财富帝国的主意，我认为对每个国家都是有害的，也对美国的国家利益造成威胁。我们会在每个回合对此予以反制。〞
 <br/>
 <br/>
 被问到中共在新疆建立集中营大规模关押维吾尔人，蓬佩奥表示，中共当局明显加强了对中国民众宗教自由的打压，川普政府将会采取实际行动，反制中共的做法。
 <br/>
 <br/>
 美国国务卿蓬佩奥：〝马上就是世界宗教自由日了，美国国务院会就此发表许多言论。〞
 <br/>
 <br/>
 周六恰逢世界宗教自由日，美国国务院发表公告，再次强调保护宗教自由，是川普政府外交政策的核心部分。
 <br/>
 <br/>
 新唐人记者林宇综合报导
