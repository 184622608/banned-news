### 有人暴力挑衅制造口实？美共和党领袖谴责炸弹邮包
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】最近两天，接连有疑为炸弹的邮包被寄送给民主党多名政要和商界大佬，引发了社会疑虑与担忧。多名共和党要员已相继发声，谴责寄送
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸弹邮包.html" target="_blank">
  炸弹邮包
 </a>
 的暴力行径。有舆论质疑：恰逢期中选举即将开始之际，这些明显带有恐怖主义色彩的行动，显然是在给共和党和川普添乱，或许这恰恰就是寄送邮包的人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_挑起事端.html" target="_blank">
  挑起事端
 </a>
 想达到的目的。
 <br/>
 <br/>
 从24日到25日，包括美国前总统奥巴马和克林顿以及前国务卿希拉里在内的多位民主党政要，先后收到了疑似炸弹的邮包。民主党重量级金主索罗斯(George Soros)和美国有线电视新闻网(CNN)纽约分社也收到了疑似有爆炸物的包裹。此外，还有寄给民主党加州联邦众议员华特斯(Maxine Waters)的可疑包裹在国会邮件收发处被发现；另有2个分别寄给前副总统拜登和影星劳勃狄尼洛的可疑包裹也被有关机构发现。
 <br/>
 <br/>
 值得庆幸的是，所有这些〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸弹邮包.html" target="_blank">
  炸弹邮包
 </a>
 〞都在第一时间被发现有问题，并未发生任何爆炸事件，也无人因此受伤，这让公众在惊疑之余松了一口气。
 <br/>
 <br/>
 针对这一系列诡异的事件，国会共和党领袖纷纷作出公开的回应，同声谴责寄送炸弹包裹的暴力行径，表态对这种行为绝对不能容忍。
 <br/>
 <br/>
 美国众议院共和党党鞭史卡利斯(Steve Scalise)24日在推特上发文说，〝这些预谋攻击已经超越犯罪的层次，根本是恐怖的行为，暴力与恐惧是我们政治不能容许的，在我们社会的任何一个角落也无法容许。〞
 <br/>
 <br/>
 史卡利斯2017年曾在国会棒球赛练球时遭遇枪击而受重伤，因而他对炸弹包裹事件感到十分愤慨。他在推文中表示，一定要查出究竟是谁做出这些恐怖行径，并且予以法律制裁。
 <br/>
 <br/>
 其推文称，〝我有高度信心我们的执法警察人员一定会成功达成使命；做为一个国家，我们必须获得共识，那就是这种铤而走险的方式，绝对不能变成一种新的常态。〞
 <br/>
 <br/>
 由于迄今为止被寄送炸弹邮包的人，几乎全都是民主党政要或曾经高调批评过川普的企业大佬或媒体，所以外界更倾向于怀疑炸弹可能来自极右派激进份子。更有声音指责川普，认为他在公开场合的一些言论比较〝激进〞，可能是诱发这种暴力事件的一个诱因。
 <br/>
 <br/>
 对此，共和党犹他州联邦参议员哈契(Orrin  Hatch)24日对外回应说，导致发生类似炸弹邮包这种极端事件的因素可能很多，并不一定与川普的言论有什么关联。现在的社会情况已经变得相当复杂，〝双方民众都应该缓和下来〞。
 <br/>
 <br/>
 哈契明确地表示：〝我并不觉得总统有什么错。我觉得他所处的立场是很艰难的，面对四面八方的攻击，我觉得总统也应该有权力可以畅所欲言才对。〞
 <br/>
 <br/>
 美国总统川普25日则在推特上发文称，〝我们今天在社会上所看到的怒气，很大一部分是主流媒体刻意造假和不精确的报导所挑起，这也就是我所说的假新闻。〞
 <br/>
 <br/>
 同时，也有舆论认为，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 即将举行的关键时刻，有人如此明目张胆地集中向川普的政敌寄送炸弹邮包，这分明是在给共和党以及川普制造麻烦。与其说寄送炸弹的人是在威胁民主党政要，不如说是有意
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_挑起事端.html" target="_blank">
  挑起事端
 </a>
 ，为攻击川普与共和党制造口实。
 <br/>
 <br/>
 还有一种观点认为，寄送炸弹邮包的人很可能是想要激化民主党和共和党之间的矛盾与分歧，故意制造出具有轰动效应的极端事件，以便挑起两党的相互攻击，乃至激发出整个社会的极端情绪，让美国乱成一锅粥甚至诱发暴力骚乱，从而达到其不可告人的政治目的。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：戴明）
