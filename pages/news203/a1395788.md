### WEF全球竞争力排名 美国重登榜首
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】世界经济论坛周三公布了〝2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球竞争力.html" target="_blank">
  全球竞争力
 </a>
 报告〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>

 <br/>
 <br/>
 世界经济论坛在〝有利环境〞、〝人力资本〞、〝市场〞及〝创新生态系统〞等四大类别下，列出12项支柱及98个指标，对全球140个经济体进行评估。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 在企业文化、劳动市场及金融体制等项目的竞争力表现优异。总分85.6，击败德国、日本、瑞士等国
 <br/>
 <br/>
 亚洲排名靠前的是日本（第5名）、香港（第7名）、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 （第13名-79.3分）、韩国（第15名）。中国排在（第28名，72.6分）。
 <br/>
 <br/>
 台湾在创新能力的表现上高居全球第四位，仅次于德国、美国及瑞士。
