### 美公布新反恐战略  加强意识形态反恐
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】周四，美国总统川普颁布了新版的美国《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家反恐战略.html" target="_blank">
  国家反恐战略
 </a>
 》，和以往相比，新战略中特别强调了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意识形态反恐.html" target="_blank">
  意识形态反恐
 </a>
 的重要性，扩大了反恐战略覆盖的范围。
 <br/>
 <br/>
 美国总统川普，周四发表声明，介绍了刚刚签署生效的新版《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家反恐战略.html" target="_blank">
  国家反恐战略
 </a>
 》。
 <br/>
 <br/>
 川普表示，这是2011年后美国首份详尽的反恐战略文件，其中规划了如何应对日益复杂、变化的恐怖主义威胁。
 <br/>
 <br/>
 白宫国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 ，周四接受电话采访，解释国家反恐新战略。
 <br/>
 <br/>
 美国国家安全顾问 博尔顿：〝我们不会把注意力放在某个单一组织上，而是会打击所有恐怖分子，只要他们有能力和意图伤害美国、美国人和美国的利益。〞
 <br/>
 <br/>
 博尔顿表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新反恐战略.html" target="_blank">
  新反恐战略
 </a>
 的具体措施包括，断绝恐怖分子背后的经济和军事支持、反恐装备现代化、重点保护美国基础设施、加强对恐怖袭击的准备和应对、反制恐怖组织招募人员、加强国际伙伴的反恐能力等。
 <br/>
 <br/>
 博尔顿特别强调，和上届政府不同，新的反恐战略确立了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意识形态反恐.html" target="_blank">
  意识形态反恐
 </a>
 的重要性。此举很大程度上拓宽了反恐策略的覆盖面。
 <br/>
 <br/>
 美国国家安全顾问 博尔顿：〝我想总统很久之前就有这个看法，不认可意识形态上的斗争，我们就无法妥善应对恐怖威胁。〞
 <br/>
 <br/>
 博尔顿点名伊朗在全球范围资助恐怖主义活动。美国驻日内瓦裁军大使伍德周四在记者会上，呼应博尔顿的説法。
 <br/>
 <br/>
 美国驻日内瓦裁军大使 罗伯特．伍德：〝我们会继续加强对伊朗的制裁，11月你们将看到新措施。〞
 <br/>
 <br/>
 伍德表示，美国非常关切伊朗在叙利亚境内的活动，不希望看到这种状况继续下去。
 <br/>
 <br/>
 新唐人记者林宇、李静明综合报导
 <br/>
