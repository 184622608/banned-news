### 嫌犯当庭逃走 美国法官立脱法袍追逐逮人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国华盛顿州.html" target="_blank">
  美国华盛顿州
 </a>
 奇黑利斯法庭日前在开庭审理案件时，两名嫌犯当庭逃走，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 巴扎德见状二话不说直接脱掉法袍一路紧追，最后擒住其中一名犯人。
 <br/>
 <br/>
 据《美联社》报导，当时在奇黑利斯（Chehalis）出庭受审的共有4名嫌犯，其中两名嫌犯分别是雅各布森（Tanner Jacobson 22岁）和霍华德（Kodey Howard 28岁），突然从法庭后门
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃跑.html" target="_blank">
  逃跑
 </a>
 ，一路疾奔，从4楼向下一路逃出一楼安全门。
 <br/>
 <br/>
 法巴扎德（R.W. Buzzard）见状立即脱掉法袍从后方紧追，终于在一楼安全门门口擒住跑得较慢的霍华德。而雅各布森虽然跑得较快，但还是在几个街区外被逮捕归案。两名嫌犯除了原本的罪行之外，目前又面临了二级逃狱罪。
 <br/>
 <br/>
 据ABC报导，巴扎德表示，他一路喊着要他们停下来，因为这会增加他们的罪行。
 <br/>
 <br/>
 他说，4年前他也经历过类似事件，一名诉讼人攻击律师，他不得不将其摔在地板上。
 <br/>
 <br/>
 巴扎德也表示，会努力使法庭变得更安全，但为更多员工申请预算是一项挑战。
 <br/>
 <br/>

 </center>
