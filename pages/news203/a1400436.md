### 川普感恩节送祝福 电话慰问全球美军
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】周四，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在佛州海湖庄园与家人共度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 ，同时，他也在这一天给美国人民和军人送去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_节日祝福.html" target="_blank">
  节日祝福
 </a>
 ，来看报导。
 <br/>
 <br/>
 周四一早，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在推特发表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 讲话，为所有美国人送上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_节日祝福.html" target="_blank">
  节日祝福
 </a>
 。
 <br/>
 <br/>
 美国总统 川普：〝祝所有人感恩节快乐！从建国开始，美国人民就聚在一起，纪念并感谢神赐予我们福祉。〞
 <br/>
 <br/>
 依往年惯例，川普随后与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 五个军种代表通话，慰劳驻扎在世界各地的美国军人。
 <br/>
 <br/>
 美国总统 川普：〝我要向我们伟大的战士们问好，感恩节快乐。你们每一个人都是伟大的爱国者，代表着美国武装部队的每一个军种。你们为了保卫国家离开了你们的家人和所爱的人，梅拉尼娅和我对于你们付出的巨大牺牲深表感谢。〞
 <br/>
 <br/>
 除了慰问海外驻军，川普还亲自前往当地海岸警卫队，探望在假日期间值班的队员。
 <br/>
 <br/>
 美国总统 川普：〝我很荣幸和你们在一起，非常感谢你们（的付出），让我们好好庆祝一下。〞
 <br/>
 <br/>
 合照期间，一名队员为了寻找合适的拍摄角度，花了不少时间。川普鼓励他说，今天时间充裕，可以慢慢来，引众人会心一笑。
 <br/>
 <br/>
 新唐人记者曹菁综合报导
</div>
