### 沃伦以原住民血统争2020换回各方批评
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】今年寻求连任的美国马萨诸塞州联邦参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沃伦.html" target="_blank">
  沃伦
 </a>
 近日成为话题焦点，其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_原住民血统.html" target="_blank">
  原住民血统
 </a>
 随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_DNA测试.html" target="_blank">
  DNA测试
 </a>
 的公布引发巨大争议。请看报导。
 <br/>
 <br/>
 联邦参议员伊丽莎白・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沃伦.html" target="_blank">
  沃伦
 </a>
 （Elizabeth Warren）周一拿出一份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_DNA测试.html" target="_blank">
  DNA测试
 </a>
 报告，证明她一直宣称的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_原住民血统.html" target="_blank">
  原住民血统
 </a>
 。
 <br/>
 <br/>
 此举迅速吸引了公众注意并引发各界回应。
 <br/>
 <br/>
 联邦认证的原住民部落切罗基国（Cherokee Nation）国务卿Chuck Hoskin Jr.当天发声明称，沃伦的做法在破坏部落利益。
 <br/>
 <br/>
 〝切罗基国〞国务卿Chuck Hoskin Jr.：〝这个所谓的DNA测试说明不了一个人是原住民公民或他们的一员。对于公民身分我们有法律规定。〞
 <br/>
 <br/>
 沃伦的DNA测试显示她可能有1/1024的原住民血统，也就是0.09%。不过2014年的一份研究显示，欧洲裔美国人整体测试结果含有0.18％的美洲原住民血统，是沃伦的2倍。
 <br/>
 <br/>
 切罗基国公民、国会众议员 Markwayne Mullin（2018.10.16）：〝对我及其他原住民、特别是切罗基国人而言真的感到不爽。她是公众人物并持续不断的利用这一点（她的原住民血统），她藉此引发辩论为其2020年挑战
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 、竞选总统做准备。〞
 <br/>
 <br/>
 对于沃伦将自己标签为少数族裔，川普在竞选总统时就持批评态度。
 <br/>
 <br/>
 总统候选人 川普：〝她在参议员位置上什么都没做。他们说她是一名超级自由派。她的职业生涯全是欺骗因为她称自己是原住民。〞
 <br/>
 <br/>
 沃伦因为原住民身分获得哈佛大学录取先机；2012年她首次竞选联邦参议员时也以此来收集支持。
 <br/>
 <br/>
 美国总统 川普（2018.10.15）：〝我绝不会道歉的，我又不欠她。她应该对国民道歉。〞
 <br/>
 <br/>
 川普在推文中感谢切罗基国对沃伦说法的揭露。他说：〝沃伦被敲打，甚至是被左派敲打。她谎称印第安人血统的说法只能骗了低智商的人。〞
 <br/>
 <br/>
 根据RealClearPolitics的数据，最近一次民调显示沃伦领先共和党对手迪尔25个百分点。
 <br/>
 <br/>
 麻萨诸塞州共和党籍联邦参议员候选人 Geoff Diehl （2018.10.15）：〝 她全力在竞选总统，无意作为马萨诸塞的参议员代表来服务这个州。〞
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
