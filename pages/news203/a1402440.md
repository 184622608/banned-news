### OPEC减产之际 美国变身石油净出口国产量第一
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】美国上周打破近75年来仰赖国外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进口石油.html" target="_blank">
  进口石油
 </a>
 的历史，成为石油净出口国，朝向美国总统川普提出的〝能源独立〞主张发展。不过，考量到单周石油数据的波动性，美国大部分时间很可能仍将维持少量石油净进口国身分。
 <br/>
 <br/>
 美国多年来朝石油净出口国方向努力，上周之所以达成目标，因当时石油进口量锐减，石油出口量则冲上历来最高水准。
 <br/>
 <br/>
 据美国能源情报署（EIA）的资料，上周美国每日净出口21万1000桶原油和汽柴油等炼油品，一反今年来每日平均净进口约300万桶油，且更远离2005年超过每日1200万桶的净进口油品需求。
 <br/>
 <br/>
 彭博报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_页岩油.html" target="_blank">
  页岩油
 </a>
 (SHALE OIL)革命使得美国一跃成为全球最大石油产国，超越俄罗斯和沙乌地阿拉伯。从德州和新墨西哥州的二叠纪（Permian）地区、北达科他州的巴肯（Bakken）到宾州的马塞勒斯（Marcellus），数以千计的油井源源产出石油。
 <br/>
 <br/>
 战略能源暨经济研究公司（Strategic Energy &amp; Economic Research）总裁林奇（Michael Lynch）表示：〝美国正成为全球举足轻重的能源强国。但转变是随时间逐渐形成，因此我不认为这会引发巨大变革。尽管如此，石油输出国家组织（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_OPEC.html" target="_blank">
  OPEC
 </a>
 ）在思考减产时，无疑必须将这点纳入考量。〞
 <br/>
 <br/>
 石油输出国家组织（OPEC）和产油盟邦7日达成减产协议，明年起每天减产120万桶原油，以提振油价，但协议并不涵盖伊朗。
 <br/>
 <br/>
 美国能源情报署指出，若依据单周石油数据，美国自1991年以来一直是石油净进口国；若依据单月数据，则自1973年以来美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进口石油.html" target="_blank">
  进口石油
 </a>
 一直多于出口。石油史学家利用美国石油业协会（API）的统计资料收集到更久远的全年数据后，发现美国自1940年代中期以来一直是石油净进口国。
 <br/>
 <br/>
