### 美军抵达冰岛 北约最大军演即将登场
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】北约规模最大的海陆空三军演习，称为〝三叉戟接点〞计划，将于10月25日到11月7日举行。周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国海军陆战队.html" target="_blank">
  美国海军陆战队
 </a>
 已抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冰岛.html" target="_blank">
  冰岛
 </a>
 ，之后将前往挪威。请看报导。
 <br/>
 <br/>
 这是冷战结束后北约最大规模的联合军演，将有近三十个国家的4万多名军人参加。
 <br/>
 <br/>
 演习主要地点在挪威，覆盖范围往东部延伸到芬兰和波罗的海，向西到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冰岛.html" target="_blank">
  冰岛
 </a>
 。但会避开挪威最北端和俄罗斯交界的地区，以免产生紧张局势。
 <br/>
 <br/>
 这次为期两周的演习将测试北约的物流和战斗能力。
 <br/>
 <br/>
 周三，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 抵达冰岛，在当地为军演做准备。之后前往挪威。
 <br/>
 <br/>
 美国军官 Rex Van Koevering：〝我们正在为‘三叉戟接点’军演做排练，寻找可以让我们的重型直升机着陆的地方。〞
 <br/>
 <br/>
 美国是军事力量最强的北约国家。美国海军驻欧洲和非洲司令詹姆斯・法戈称，以〝杜鲁门〞号为首的美军航母羣也将参加这次军演。
 <br/>
 <br/>
 美国军官 Rex Van Koevering：〝我非常高兴能为冰岛和北约其他国家的人民工作。〞
 <br/>
 <br/>
 这将是冷战结束以来，北约最大的军演。北约过去多年一直削减军费，并将注意力集中在比较遥远的中东等地。这次军演表明，北约重新将目光投向了自家后院，加强了对俄罗斯的防御。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
