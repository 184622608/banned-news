  彭斯
 </a>
 的哥哥

  葛瑞格．彭斯
 </a>
 顺利当选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印第安纳州.html" target="_blank">
  印第安纳州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联邦众议员.html" target="_blank">
  联邦众议员
 </a>
 。
 <br/>
 <br/>
 共和党籍的葛瑞格．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Greg Pence）周二（6日）当选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印第安纳州.html" target="_blank">
  印第安纳州
 </a>
 第6选区众议员。他的弟弟麦克．彭斯（Mike Pence）在成为美国副总统前，也曾担任印第安纳州第6选区众议员，之后还担任该州州长。
 <br/>
 <br/>
 中期选举开票还不到1/5，61岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_葛瑞格．彭斯.html" target="_blank">
  葛瑞格．彭斯
 </a>
 便以68%得票率宣布胜选。
 <br/>
 <br/>
 葛瑞格．彭斯在胜选演说中表示：〝如同许多人一样，我持续受到川普（特朗普）总统的鼓舞。〞
 <br/>
 <br/>
 他说，〝我支持总统为了中产阶级着想的政策议题，并确保合理政策获得国会通过且完成立法。〞
 <br/>
 <br/>
 今年有35个参议员席位要选举，其中民主党需要更换26个席位、共和党需要更换9个席位。要成为多数党，必须拥有至少51票。435个众议院席位全部竞选。
 <br/>
 <br/>
 截止美东时间23:42，参议院改选方面，共和党50席：民主党38席。众议院方面，共和党128席：民主党114席。州长改选，共和党17席：民主党13席。
 <br/>
 <br/>
