### 佛州周日开始重新计票 最晚15日出结果
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】周日，美国佛罗里达州正式开始对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的重新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_计票.html" target="_blank">
  计票
 </a>
 。其中争议不断的布劳沃德郡，却拖延了两个小时才开始。
 <br/>
 <br/>
 周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 超过一半以上的郡，都已经开始重新计算选票。
 <br/>
 <br/>
 其中，最具争议的布劳沃德郡（Broward），因为制表机故障，在延后两小时后，才开始重新计算多达70万的选票。
 <br/>
 <br/>
 此前，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 占优势的布劳沃德郡，发现了数量不详的未计数早期选票，共和党质疑，该郡很可能存在选举舞弊现象，官员却拒绝提交共和党合法索要的记录。
 <br/>
 <br/>
 周六，川普总统就发推文表示：〝（民主党）试图窃取佛州的两场重大选举！我们正在密切关注！〞
 <br/>
 <br/>
 此次，佛州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的参议员和州长的竞选结果，共和党候选人皆以小于0.5个百分点的差距，领先民主党候选人。
 <br/>
 <br/>
 佛州法律规定，得票差距在0.5%以下，就达到重新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_计票.html" target="_blank">
  计票
 </a>
 标准。
 <br/>
 <br/>
 这次佛州重新计票，最晚要在15日向州府上交统计结果。
 <br/>
 <br/>
 新唐人记者王愉贺粽合报导
</div>
