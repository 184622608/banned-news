### 匹兹堡枪击案嫌犯坐轮椅出庭 川普夫妇亲往案发城市慰问
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_匹兹堡.html" target="_blank">
  匹兹堡
 </a>
 日前造成11死6伤的反犹太
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，嫌犯鲍尔斯周一（29日）坐轮椅出庭，一直目无表情，法官下令不准保释。周二（10月30日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）总统夫妇乘专机抵达匹兹堡，慰问遇害的受难者家属，以及数千名哀悼者。
 <br/>
 <br/>
 宾州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹太教堂.html" target="_blank">
  犹太教堂
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 嫌犯、46岁的卡车司机罗伯特・鲍尔斯（Robert Bowers）周一出庭应讯。他面临仇恨罪等29项联邦刑事指控。其中11项为使用枪枝武器造成伤害，7项为对警方动用武力，还有11项为因受害人的宗教信仰而对其谋杀。
 <br/>
 <br/>
 受伤后需要坐轮椅的鲍尔斯，周一下午约1点30分被推入法庭，并坐在两名被委派代表他的公共律师中间。庭审期间他只和两名律师进行了短暂交谈、看文件并向法官证明身份，期间还数次简短回答法官的问题。法庭职员一度打开他其中一只手的手铐让他签署文件，直到离开法庭他都一直目无表情。
 <br/>
 <br/>
 控方目标是希望判处鲍尔斯死刑，检察官巴拉迪称事件是〝可怕的暴行〞。他说，〝大家可以放心，我们有多名检察官正努力确保正义得到伸张。〞
 <br/>
 <br/>
 联邦检察官布雷迪（Scott Brady）周一发表法院简短声明称，下次开庭时间是11月1日，届时联邦政府将提出证据，证明鲍尔斯杀害了11名正在参加宗教活动的信徒，并射伤了6人，到时控方会向他详细解释全部29项控罪。
 <br/>
 <br/>
 美国司法部长塞申斯周一在波士顿表示，这是对人类所有信众、以及美国价值观的攻击。
 <br/>
 <br/>
 塞申斯说，〝这不仅是对犹太信徒的攻击，也是对所有人类信仰者的攻击。这是对美国保护信仰者的价值观的攻击。这不能也不会被容忍。这是对我们本身的直接威胁。〞
 <br/>
 <br/>
 该枪击案事发于上周六（27日），鲍尔斯携带半自动式攻击型步枪进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_匹兹堡.html" target="_blank">
  匹兹堡
 </a>
 一家名为〝生命之树〞的犹太教堂，向教堂内的犹太人开枪，造成11人死亡，6人受伤。鲍尔斯被认为有强烈的反犹太倾向，并曾在社交网络上发布仇恨言论。
 <br/>
 <br/>
 事件中不幸遇难的罗森塔尔兄弟于周二（10月30日）举行了葬礼。当天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 夫妇乘专机抵达匹兹堡，慰问被凶手枪击遇害的受难者家属，以及数千名哀悼者。白宫表示两人希望〝向匹兹堡表达支持以及和当地市民一起哀悼〞。
 <br/>
 <br/>
 同一天，匹兹堡犹太教堂举行被枪击遇难的第一次葬礼哀悼仪式。来自各地的哀悼者前往犹太教堂向遇难者致哀。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
