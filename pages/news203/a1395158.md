### 美国又出重拳 限制向中共国企输出核技术
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】周四（10月11日），美国出台
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_限制令.html" target="_blank">
  限制令
 </a>
 ，原则上禁止对中国输出核产品及技术，以杜绝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 非法获取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核技术.html" target="_blank">
  核技术
 </a>
 以提高其军力并削弱美国工业。
 <br/>
 <br/>
 《华尔街日报》报导，这道
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_限制令.html" target="_blank">
  限制令
 </a>
 将立即生效，内容包括禁止出口核产品及技术到中国最大的核电公司，即中国广核集团（China General Nuclear Power Co. , CGN）。
 <br/>
 <br/>
 限制令并非全面禁止输出，而是代表川普（特朗普）政府加强管制核产品输出到中国，以保证
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 正确地使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核技术.html" target="_blank">
  核技术
 </a>
 。此外，限制令也不涵盖传统的核子反应炉，而是针对一些更可能被应用在军事用途上的新型技术。
 <br/>
 <br/>
 美国官员表示，有证据显示，中共正持续且迅速地以非法手段获取美国军用、商用技术，并用于在南海人工岛屿上兴建浮动核能发电设施、航空母舰和潜艇；中共也不正当地将美国的核能技术转移给其他国家。
 <br/>
 <br/>
 官员表示，鉴于中共及其国有企业的非法活动，国家安全委员会（National Security Council，NSC）重新检讨美国对中共的核政策，并且认为有必要进行改革。
 <br/>
 <br/>
 周四稍早，前美国助理国务卿、现任伍德罗威尔逊国际学者中心（Woodrow Wilson International Center for Scholars）亚洲项目主任邓志强（Abraham Denmark）在演说中警告说，核问题在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 中被忽视，它将成为一个潜在的爆发点。
 <br/>
 <br/>
 <h4>
  美祭出限制令 事出有因
 </h4>
 <br/>
 早在2016年，美国曾起诉华裔核技术工程师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_何则雄.html" target="_blank">
  何则雄
 </a>
 （Szuhsiung〝Allen〞Ho）及CGN。根据起诉文件，美国指控CGN多年来协助北京当局试图取得美国机密文件，手段是对美国核能专家施压，要求他们提供美国的核电设备、使用手册和软件等机密文件。对拒绝照办的专家，CGN即不再邀请或聘其为顾问。
 <br/>
 <br/>
 何则雄则被控与他人合谋，从1997年至2016年4月，在没有依法经美国能源部批准的情况下，在中国参与一种特殊核材料的研发和生产。
 <br/>
 <br/>
 据法庭文件，何则雄帮助CGN招募美国的核技术工程师，以降低设计和制造核反应堆部件的时间和成本。何则雄于2017年1月认罪，并被判处24个月监禁。
 <br/>
 <br/>
 至于能否对CGN以外的中国公司出口核产品或技术，美国官员表示，将根据具体情况逐一审查。
 <br/>
 <br/>
 美国官员还表示，新规定是为了保护国家安全，以及维护美国核工业基地的完整性；为了降低新规定对美国工业的影响，相关部门将努力为美国公司开辟其它出口市场。
 <br/>
 <br/>
 据统计，2017年美国向中国出口了价值1.7亿美元的核技术。核电专家表示，中国是少数几个扩大核电投资的国家之一，市场规模可能达到数十亿美元。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：秦锋）
