### 美国中期选举开始计票  纽约州州长库默连任　
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】美国期中选举于当地时间6日登场，目前已经开始计票，全球紧盯战况。各州投票于美东时间6日晚间6时陆续结束，结果已一一揭晓。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约州.html" target="_blank">
  纽约州
 </a>
 民主党籍现任州长库莫（Andrew Cuomo）确定连任州长。
 <br/>
 <br/>
 2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 中，参议院共35席需要改选。要成为多数党共和党仅需要再拿下9席，而民主党需要26席。众议院全部435席全部改选，要成为多数党，两党都需要获得至少218席。
 <br/>
 <br/>
 与此同时，美国50个州里的34个州将会在中期选举中选出4年任期的州长。佛蒙特州和新罕布什尔州则会在中期选举及总统选举年中选出两年任期的州长。所以，中期选举中会选出36位州长。此外，选民们还将参与地方议会选举，重新改选6千6百多个州立法和行政机构官员和成千上万个县、市的议会职位。
 <br/>
 <br/>
 美国有2亿3千多万名合格选民。
 <br/>
 <br/>
 美国中期选举将是影响美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）后半总统任期政令推行的关键之战。目前，川普所在的共和党在国会两院均占据多数党的优势地位，这是帮助川普税改等多项政令可以顺利立法的关键保障，进而为2020年总统选举打下基础。
 <br/>
 <br/>
 目前，在36州州长改选的战况中，已有24州结果出炉。
 <br/>
 <br/>
 其中共和党确定当选的有：麻萨诸塞州州长贝克（Charlie Baker）、麻里兰州州长霍根（Larry Hogan）、田纳西州州长比尔．李（Bill Lee）、阿肯色州州长哈金森（Asa Hutchinson）、德州州长阿博特（Greg Abbott）、怀俄明州州长高顿（Mark Gordon）
 <br/>
 <br/>
 民主党确定当选的有：罗德岛州州长瑞芒多（Gina Raimondo）、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约州.html" target="_blank">
  纽约州
 </a>
 州长库莫（Andrew Cuomo）、宾州州长沃夫（Tom Wolf）、伊利诺州州长普立兹克（J.B. Pritzker）。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
</div>
