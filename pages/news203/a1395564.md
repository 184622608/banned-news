### 撑不下去了？两艘满载美国大豆货船驶向中国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】在美中贸易持续升温之际，又有两艘满载
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 的货船驶向中国港口。迹象表明即使美中全面对抗，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国买家.html" target="_blank">
  中国买家
 </a>
 仍需要美国大豆。
 <br/>
 <br/>
 彭博社10月15日报导，这两艘满载
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 的货船，在10月11日结束的那一周装船完毕。其中〝劳拉星〞号（Star Laura）货轮驶离墨西哥湾的港口，将于月底抵达中国青岛港。而〝金色皇后〞号货轮，则从美国太平洋西北海岸起航，预计于12月到达中国。
 <br/>
 <br/>
 美国农业部最新的农作物出口检验数据显示，从墨西哥湾的港口运往中国的大豆约6.5万吨，从华盛顿州普吉特海湾（Puget Sound）港口运往中国的大豆约6.9万吨。
 <br/>
 <br/>
 自7月6日美中贸易战正式开打之后，中共向美国大豆征收25%的关税，以报复美国对340亿美元中国商品的关税。近几个月来，中国转向巴西购买大豆。
 <br/>
 <br/>
 根据上周的美方出口销售报告，自9月1日营销年开始截至10月4日，美国仅向中国出口了约6.7万吨大豆。
 <br/>
 <br/>
 一些分析人士认为，最新的数据显示，在巴西大豆供应逐渐下降、新一季大豆要等到明年年初才能丰收之际，中国不得不向美国购买大豆。
 <br/>
 <br/>
 资料显示，美国和巴西的大豆收获季节不同，南半球的巴西大豆收获季节是2月份到5月份，而美国的收获季节是9月份到12月份。
 <br/>
 <br/>
 根据全球油籽专业分析机构〝油世界〞分析数据，9月到12月期间，南美对中国的大豆出口量最高也只能达到1030万吨。但今年冬季中国至少需要消耗掉3000多万吨的大豆。缺口达2000多万吨。
 <br/>
 <br/>
 斯特林营销公司（Sterling Marketing）分析师道森（Mike Dawson）8月底曾说，今年末，巴西将不会有足够的大豆来供应中国。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国买家.html" target="_blank">
  中国买家
 </a>
 将不得不恢复购买美国大豆。要么中国商家忍受着关税带来的高价，要么中共解决贸易纠纷。
 <br/>
 <br/>
 中国每年要消费世界上三分之一的大豆。大豆被用来喂猪，而猪肉是中国人餐桌上的主食。
 <br/>
 <br/>
 根据美国农业部的数据，中国每年购买美国的大豆，约占美国大豆出口量的5成。美国去年向中国出口大豆达到124亿美元。美国大豆在中国是优质猪饲料，也用来制作食用油。如果中共继续维持25%的关税，它将进一步加大中国通膨。
 <br/>
 <br/>
 富国投资研究所房地产策略主管拉弗吉（John LaForge）8月曾说，无法保证中共对大豆的需求将迫使它在贸易谈判中让步。中共可以暂时动用紧急储备，帮助中国买家避免购买美国大豆。但是这只是临时解决办法。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：赵云）
