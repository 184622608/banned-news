### 川普划出川习会谈判底线：舍此不能达成协议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 即将于本月30日开幕，中美两国首脑将趁出席这次峰会的机会举行双边会晤，为解决中美贸易冲突寻找出路。日前，美国总统川普（特朗普）在接受美国媒体采访时，亮出了自己与习近平谈判的底线，那就是中国市场必须对美开放，如同美国市场向中方开放那样。
 <br/>
 <br/>
 从本周一（26日）开始，阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 已经进入筹备的最后阶段。各国领导人的峰会即将于11月30日上午在布宜诺斯艾利斯科斯塔萨尔格罗（Costa Salguero）会展中心拉开帷幕。美国总统川普和中国国家主席习近平也即将在G20峰会上会面，届时他们两人举行的双边会晤是否成功，对中美贸易关系的走向将产生重大影响。
 <br/>
 <br/>
 11月26日，川普在记者会上谈到中美贸易战时，表达了比较乐观的态度。他表示，自己有可能在阿根廷的G20峰会上同习近平达成一份协议。不过，他同时强调，美国政府的底线是〝中国必须公平对待我们〞。
 <br/>
 <br/>
 同一天，川普在接受《华尔街日报》采访时也明确表示，〝对我来说，唯一可以接受的协议就是中国向美国开放市场，接受美国的竞争。他们必须让中国对美国开放。否则，我不认为能达成一份协议。至于其他国家，他们自己决定。〞
 <br/>
 <br/>
 此外，川普还表示，北京当局还必须遏制其窃取美国知识产权的行为。
 <br/>
 <br/>
 当时，《华尔街日报》的记者问道：〝在这次G20峰会上，中国的首要目标就是让你推迟或叫停从2019年1月1日开始的25%的关税，你愿意这么做吗？〞
 <br/>
 <br/>
 川普回答说：〝我认为非常不可能。〞
 <br/>
 <br/>
 他进一步解释说，在美中之间还没有达成协议之前，如果美国政府就停止提高关税比例等政策，中共将会继续敲诈美国，就像他们过去那样继续任意妄为。
 <br/>
 <br/>
 川普表示，如果有必要的话，他还可以向中国另外2,670亿美元的商品征税，还可以提高关税比例。这样即使美国政府没有与中共达成协议，美国照样也会也会挣到数十亿美元。
 <br/>
 <br/>
 他说：〝现在有很多钱流入我们的国库，1月1日开始，会有更多的钱流入。故事是这样的：如果我们没有达成一份协议，我会对中国另外2,670亿美元的商品征收10%到25%的关税。〞
 <br/>
 <br/>
 今年夏天，川普曾警告北京，美国政府将从2019年1月1日开始，把对2,000亿美元中国输美商品征税的比例从10%提升至25%。
 <br/>
 <br/>
 就在前不久，北京方面向美国政府发出了签订一项协议的建议，但华府方面却认为，北京的这项建议漏掉了几项至关重要的内容，所以美国政府还不能接受。
 <br/>
 <br/>
 不过，《华尔街日报》的报导称，包括美国财政部和白宫国家经济委员会(National Economic Council)官员在内的其他人士一直想看看这项建议是否能成为签订一项协议的基础。
 <br/>
 <br/>
 川普在这次接受采访的过程中还提到，美国对中国商品提高关税，将会迫使企业把生产线转移到不会面临这些关税的地方，而他建议这些企业在美国建厂和生产。
 <br/>
 <br/>
 此外，在即将举行的G20会议上，美国将与墨西哥和加拿大签署经过重新谈判的北美自由贸易协定(North American Free Trade Agreement)，这项协定已更名为《美国-墨西哥-加拿大协定》(U.S.-Mexico-Canada-Agreement)。而川普政府将这项协定列为优先任务的一个原因是，希望鼓励美国公司至少将工厂从中国迁往北美。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
