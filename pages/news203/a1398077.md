### 中期选举倒数一天 州选举局：准备好了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 倒数剩一天。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约州选举.html" target="_blank">
  纽约州选举
 </a>
 局发言人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康克林.html" target="_blank">
  康克林
 </a>
 （John Conklin）表示，为防止黑客入侵投票系统做手脚，这次选举，该州投入使用光学扫描仪的投票机，另使用纸质选票进行备份。他表示〝已尽一切努力做好准备〞。
 <br/>
 <br/>
 根据Press-Republican报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康克林.html" target="_blank">
  康克林
 </a>
 指出，这些新式投票机不会连接互联网；州府机构与各地方选举委员会密切合作，检查投票设备的可靠性、评估潜在的风险；他们会在选举前后对机器进行测试，并于选举后进行随机审核，以验证机器的准确性。
 <br/>
 <br/>
 州选举局（State Board of Elections）预计，即使在恶劣的天气下，周二（6日）选民的投票率会出现异常高涨，〝我们一直在准备额外的选票〞。
 <br/>
 <br/>
 民众前往投票时，如果遇到投票障碍，例如：发现名字不在选民名单中，可以拨打Common Cause New York的选举保护热线（866-687-8683）；他们将汇集问题做成报告。
 <br/>
 <br/>
 ──转自《大纪元》
 <br/>
 <br/>
