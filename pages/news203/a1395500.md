### 传记者失踪案有进展 沙特将承认死于刑讯
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】周一（10月15日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_记者.html" target="_blank">
  记者
 </a>
 卡舒吉（Jamal Khashoggi）一案有了最新进展，据传无可逃避的沙特政府正准备承认，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 的卡舒吉误死于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刑讯.html" target="_blank">
  刑讯
 </a>
 。
 <br/>
 <br/>
 据美国有线电视新闻网（CNN）15日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 政府正在准备一份报告，将承认两周前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 的卡舒吉在审讯过程中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_死亡.html" target="_blank">
  死亡
 </a>
 ，然后被沙特方面转移出土耳其。
 <br/>
 <br/>
 报导称，消息人士透露，该报告计划给出如下结论：卡舒吉的死是审讯不当造成的错误后果，而该行动在执行时〝未获许可〞（without clearance），因此参与事件的人将对此负责。
 <br/>
 <br/>
 卡舒吉失踪事件已导致沙特与西方陷于外交危机之中。目前，英、法、德都要求沙特对事件进行〝可信的调查〞；美国总统川普则更为强硬，他表示，如果发现是沙特政府谋杀了卡舒吉，美国将严厉制裁沙特。
 <br/>
 <br/>
 川普总统周一下午表示，他已看到沙特将做出报告的报导，但他不知道这是真的还是谣言。
 <br/>
 <br/>
 川普说，〝我还不知道。我必须看看他们怎么说〞。
 <br/>
 <br/>
 川普当时正在乔治亚州小镇华纳．罗宾斯（Warner Robins）慰问受到迈克尔飓风（Hurricane Michael）伤害的居民，他告诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_记者.html" target="_blank">
  记者
 </a>
 ，〝没有人知道这是否是官方报告。到目前为止，只是传说报告将要出来〞。
 <br/>
 <br/>
 川普表示，他非常渴望彻底查清卡舒吉事件的情况，并且指出，沙特正与土耳其合作，以确定到底发生了什么，而且很多人都在参与这项调查。
 <br/>
 <br/>
 周一早上，川普在推文中表示，他已致电沙特国王，但对方断然否认自己了解情况。
 <br/>
 <br/>
 川普随即派遣国务卿蓬佩奥（Mike Pompeo）启程前往沙特，会见沙特国王并进一步敦促彻查事件。
 <br/>
 <br/>
 蓬佩奥已于当天中午离开华盛顿飞往沙特首都利雅得（Riyadh）。
 <br/>
 <br/>
 卡舒吉是《华盛顿邮报》（The Washington Post）专栏作家，经常批评沙特王储，去年逃离沙特。
 <br/>
 <br/>
 10月2日，卡舒吉前往沙特驻土耳其伊斯坦布尔领事馆领取文件，但之后离奇消失。
 <br/>
 <br/>
 据土耳其媒体报导，卡舒吉已在领事馆遇害，尸体已转移，而〝暗杀小组〞来自沙特。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
</div>
