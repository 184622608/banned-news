### 俄媒评川习会：根本分歧仍在 贸战停火不会持久
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】12月1日的中美首脑会议上，北京以多方让步换取了90天暂停贸易战升级的时间。但俄媒指，美中有关贸易问题的根本分歧仍在，贸易战暂缓的时间不会长。
 <br/>
 <br/>
 根据白宫12月1日当晚发表的声明，北京承诺从美国购买大量农业、能源、工业和其它产品，管控此前大举输送美国的毒品，从新考虑高通收购案等；作为交换，美方同意将2019年1月1日对2,000亿美元中国商品提高税率的计划搁置90天。
 <br/>
 <br/>
 不过在这90天内，中方必须与美方开始就强制技术转让、知识产权保护、非关税壁垒、网络入侵和网络盗窃等问题展开谈判。如果届时不能达成协议，美方将重启关税升级计划。
 <br/>
 <br/>
 俄罗斯《独立报》12月2日分析称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 协议并不是一个突破，中美之间在贸易政策方面的根本分歧仍然存在，没有迹象表明任何一方已准备放弃自己的立场。
 <br/>
 <br/>
 报导引述俄罗斯国立高等经济学院教授波尔坦斯基（Alexei Polttanski）的分析指，中美这次暂缓贸易战升级，还无法变成永久性休战。毕竟在之前，中美双方也曾达成过短暂协议，之后又很快转向对抗。
 <br/>
 <br/>
 波尔坦斯基称，中美之间的根本分歧不仅是贸易逆差，而是美方要遏制中共的野心。中共的〝中国制造2025〞规划显示，北京依然不会放弃称霸世界的目标。
 <br/>
 <br/>
 波尔坦斯基指出，贸易冲突是一个看得见的战场，中美贸易战暂缓不会持续很长时间。
 <br/>
 <br/>
 事实上，十几年来，中共正在利用不公平的贸易拖垮美国经济，同时在政治、经济和军事领域不断对外扩张，其压倒美国、统治世界的野心已经彰显无疑。这已经对西方自由社会形成致命的威胁，越来越引起国际社会的警惕。
 <br/>
 <br/>
 而川普政府提出〝美国优先〞，重振美国经济，正是对中共拖垮美国的反击。同时，作为西方国家的龙头，美国也担负起了遏制中共威胁的重任，并得到了越来越多西方国家的响应。
 <br/>
 <br/>
 外界认为，只要中共体制不变，美中长期对抗已不可逆转，虽然其间有可能出现一些技术性暂缓。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
