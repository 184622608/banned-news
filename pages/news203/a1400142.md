### 佩洛西拚众院议长面临挑战 川普发文力挺
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】民主党资深国会众议员兰希．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佩洛西.html" target="_blank">
  佩洛西
 </a>
 正在力争担任下届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众院议长.html" target="_blank">
  众院议长
 </a>
 ，但在民主党内遭到十余名新众议员的公开反对，佩洛西虽然表态不惧挑战，但舆论认为民主党恐因众院议长之争而陷入混乱。当此之际，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 发推文，罕见表态愿支持佩洛西竞选众院议长。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 日前发推文表态支持民主党众议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佩洛西.html" target="_blank">
  佩洛西
 </a>
 (Nancy Pelosi)担任下届议长，称自己可以帮她争取共和党支持。川普甚至还在推文中透露，包括纽约州众议员里德(Tom Reed)在内的一些共和党议员，有可能对佩洛西投下赞成票。
 <br/>
 <br/>
 其后，里德的发言人雷纳特(Will Reinert)对媒体解释说，〝总统了解众院面临破裂的情况，众议员（里德）愿意支持任何一位允诺改革并能代表美国人民的众院议员。〞
 <br/>
 <br/>
 此前，佩洛西曾对外表示，她和众院的其他民主党议员愿意与川普合作，以便推动降低处方药价、重建美国的基本设施，以及保护年轻无证移民免于遭到驱逐等事项。
 <br/>
 <br/>
 不过，佩洛西的发言人哈米尔(Drew Hammill)日前在谈及佩洛西竞争议长一事时却表示，自己对佩洛西靠民主党的选票来赢得议长充满信心。
 <br/>
 <br/>
 现年78岁的佩洛西是位传奇议员，从2003年以来一直是众院民主党的领袖，2007至2010年民主党控制众院期间，佩洛西就曾出任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众院议长.html" target="_blank">
  众院议长
 </a>
 。今年11月6日民主党在中期选举中夺回众院的控制权后不久，佩洛西就对外宣布自己将争取担任议长。目前，佩洛西已获得民主党内大多数成员的支持，其中包括61位女性议员的支持。
 <br/>
 <br/>
 不过，近日在民主党内出现了反对派，至少有16名民主党议员已表态反对佩洛西出任议长。他们的理由是民主党需要〝新的领导阶层〞来带头才能获得新生。有舆论称，民主党在未来六周内，可能为领导权之争爆发激烈内斗，围绕是否支持佩洛西出任众院议长的问题，幕后的攻防战已经开打。
 <br/>
 <br/>
 佩洛西本月19日已对外表明，将支持自己的副手侯耶尔和克莱本续任众院民主党第二和第三号领导人。她同时表示不惧竞争对手的挑战。
 <br/>
 <br/>
 众院1月3日将正式选举新的领导人，届时佩洛西在全部435票中至少必须获得218票才能获胜。民主党目前在众院握有232席，因此虽然公开反对她竞选议长的只有16人，真正具有挑战力的对手也还未出现，但也已经对佩洛西的议长之战形成了威胁。
 <br/>
 <br/>
 佩洛西上周会晤了进步派国会党团(CPC)成员，CPC共同主席、华盛顿州联邦众议员蜜拉‧贾亚帕(Pramila Jayapal)，现在又利用感恩节假期私下会晤新科民主党议员，希望能获得更多人的支持。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
