### 参院通过程序投票 卡瓦诺确认进入最后一步
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】周五（10月5日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通过.html" target="_blank">
  通过
 </a>
 了确认最高法院大法官提名人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 （Brett Kavanaugh）的程序性投票，对他的确认表决至此已进入倒计时，最终结果周六将见分晓。
 <br/>
 <br/>
 周五上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 的程序性投票扫清了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 确认的最后一道障碍，以51: 49
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通过.html" target="_blank">
  通过
 </a>
 表决，顺利进入最后一关。
 <br/>
 <br/>
 值得关注的是，投票并没有严格按照党派划线。西维吉尼亚州民主党参议员曼钦（Joe Manchin）投了赞成票，而阿拉斯加州共和党参议员穆考斯基（Lisa Murkowski）则投了反对票。
 <br/>
 <br/>
 两张共和党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摇摆票.html" target="_blank">
  摇摆票
 </a>
 起到了关键的推动作用——缅因州参议员柯林斯（Susan Collins）和亚利桑那州参议员弗莱克（Jeff Flake）都投了支持票。周四，他们都表示认同FBI的调查，暗示他们可能支持卡瓦诺确认。
 <br/>
 <br/>
 川普总统对参议院通过程序性投票表示赞赏，他在推文中说：
 <br/>
 <br/>
 〝非常自豪，美国参议院投票‘赞成’，推进卡瓦诺法官的提名！〞
 <br/>
 <br/>
 <blockquote class="twitter-tweet" data-lang="en">
  <p dir="ltr" lang="en">
   Very proud of the U.S. Senate for voting “YES” to advance the nomination of Judge Brett Kavanaugh!
  </p>
  — Donald J. Trump (@realDonaldTrump)
  <a href="https://twitter.com/realDonaldTrump/status/1048226225196552193?ref_src=twsrc%5Etfw">
   October 5, 2018
  </a>
 </blockquote>
 <br/>
 <div style="clear:both;display:block;">
 </div>
 <br/>
 不过，周六的最终表决结果仍难肯定。柯林斯虽然同意结束辩论并进入最终表决程序，但她仍未明确表示是否支持卡瓦诺确认，她说她周五下午才会宣布自己的最终决定。预计她的声明将在下午3点出台。
 <br/>
 <br/>
 弗莱克则表示，〝除非发生重大变化〞，他打算投赞成票。
 <br/>
 <br/>
 参议院司法委员会主席格拉斯利（Chuck Grassley）呼吁共和党参议员投票支持卡瓦诺，以此向民主党的暴民统治（mob rule）说〝不〞。他指出，民主党以党派划线极力抵制共和党的政策，这一切目前就发生在国会。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
