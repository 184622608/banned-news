### 卡瓦诺就职  堕胎健保法可能重塑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】周六（10月6日），美国参议院以50对48投票确认美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）提名的最高法院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官.html" target="_blank">
  大法官
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 ，川普当天签委任令后卡瓦诺立即就职。外界认为，这对未来的堕胎权、同性恋权利、拥枪权等案件的判决，都有重大影响。
 <br/>
 <br/>
 根据联邦上诉法院的司法审理记录，对于联邦监管权力、枪枝、堕胎、宗教自由和移民等法律争议案，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 将与其他4位保守派
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官.html" target="_blank">
  大法官
 </a>
 保持一致的立场。今年只有53岁、毕业于耶鲁大学法学院的卡瓦诺，未来的堕胎权、同性恋权利、拥枪权等案件的判决，都有重大影响。
 <br/>
 <br/>
 <h4>
  移民
 </h4>
 <br/>
 他曾反对给予巴西劳工特别签证，入境美国从事美国劳工能够胜任的工作。此外，他裁定一项工会选举结果无效，因为这个选举容许非法移民参与投票。
 <br/>
 <br/>
 <h4>
  行政权
 </h4>
 <br/>
 卡瓦诺曾撰文表示起诉现任总统可能使政府为之瘫痪，因此总统在职期间应豁免受到起诉或民事控告。
 <br/>
 <br/>
 <h4>
  堕胎
 </h4>
 <br/>
 卡瓦诺立场保守，被认为是虔诚的天主教徒。为此，他在堕胎问题上的立场引发关注。卡瓦诺称最高法院1973年裁定堕胎合法化为〝重要判例〞，不过他的背景显示他可能进一步限制堕胎权利，支持者和批评者都预计他会这样做。
 <br/>
 <br/>
 卡瓦诺2015年在审理有关奥巴马医改将避孕纳入健保的规定时，裁定这个措施侵犯了宗教组织的权利。去年，卡瓦诺在一项判决中反对允许非法青少年移民进行堕胎的规定。
 <br/>
 <br/>
 <h4>
  同性恋
 </h4>
 <br/>
 卡瓦诺将接替的今年退休的大法官肯尼迪，2015年投下关键性的第5票，推翻对同性恋婚姻的禁令。肯尼迪也为这项加强同性恋权利的裁决撰写意见书。
 <br/>
 <br/>
 肯尼迪经常对社会议题投下决定性的一票，使高院以5对4做成裁决，并巩固保守派对高院的控制。
 <br/>
 <br/>
 卡瓦诺没有对同性恋权利做过裁决，但他认为宪法没有明文订定的权利，只有在〝根源于国家历史和传统〞的情况下才能获得承认，而他把高院对同性婚姻合法化的裁决同样称为〝重大判例〞。
 <br/>
 <br/>
 <h4>
  健保
 </h4>
 <br/>
 保守派的首席大法官罗伯兹支持大致维持奥巴马健保的裁决，使共和党人不满。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 把废除奥巴马健保列为最优先事项。卡瓦诺一直拒绝表明他在这个问题上的立场。
 <br/>
 <br/>
 尽管他过去曾在堕胎、奥巴马健保等案件中，一度采取较为温和立场，并因此遭到保守派人士质疑，但整体而言，卡瓦诺仍被视为立场坚定的保守派，这亦是川普提名他的理由。
 <br/>
 <br/>
 <h4>
  卡瓦诺坚定的保守派
 </h4>
 <br/>
 53岁的卡瓦诺毕业于耶鲁大学法学院，担任过肯尼迪大法官的法律助理，并为第三和第九巡回上诉法院的法官当办事员，在小布什总统任内曾担任白宫律师，有在白宫工作的丰富经验。
 <br/>
 <br/>
 自2006年起，卡瓦诺一直担任华盛顿特区巡回上诉法院法官，积累了稳固的保守司法记录。他撰写了300多条意见，其中10余条获得最高法院批准，成为国家法律。
 <br/>
 <br/>
 卡瓦诺具有完整法律资历，川普7月9日在白宫东厅引荐他时表示，〝卡瓦诺拥有无可挑剔的资历，在整个法律界，他被认为是法官中的法官，当代最敏锐的法律学家，没有人比他更有资格出任大法官。〞
 <br/>
 <br/>
 卡瓦诺替代的是今年6月27日宣布退休的最高法院大法官肯尼迪。肯尼迪一直被认为是保守派法官中的摇摆票。而卡瓦诺是坚定保守派，他的入职，标志着最高法院9名大法官形成稳定的、保守派占多数的5-4格局，最高法院正式〝右转〞。
 <br/>
 <br/>
 卡瓦诺一向被认为是恪守法律的原旨派法官，他获川普提名时也强调，〝法官必须立场独立，必须解释法律，而非创造法律。法官必须依据法令书写的原貌来解释，法官必须依据宪法的原貌来解释，并以历史、传统、判例为佐证。〞
 <br/>
 <br/>
 （记者李红报导）
