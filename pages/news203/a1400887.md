### 美2020国防预算或缩减 更专注合理分配
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国防部.html" target="_blank">
  美国国防部
 </a>
 目前正在制定2020财年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国防预算.html" target="_blank">
  国防预算
 </a>
 草案，总额度有可能下降至$7000亿美元。专家认为，当前最重要问题是如何让预算分配更合理，有效提升美军战力。
 <br/>
 <br/>
 2019财年，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国防预算.html" target="_blank">
  国防预算
 </a>
 为7,160亿美元，创下9年来最大增幅。但不久前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统要求内阁各部门，将预算缩减5%。他还说，明年国防预算可能会降至$7000亿美元。
 <br/>
 <br/>
 中期选举后，预计新一届国会将就国防预算展开一翻角力。参议院共和党认为，2020年国防开支应进一步增加，而众议院民主党则表示，明年财政赤字逼近一万亿美元，应更多缩减国防预算。
 <br/>
 <br/>
 布鲁金斯学会安全与战略高级研究员Frank A. Rose：〝两党有很多机会，在国防预算上可相互妥协，问题是他们是否有此政治意愿。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五角大楼.html" target="_blank">
  五角大楼
 </a>
 已准备了两份不同预算报告，将在12月1日前完成。一份是早先编制的、总额为$7330美元；另一份则按川普总统要求、总额为$7000美元。
 <br/>
 <br/>
 专家预计，年度国防预算最终可能变化不大。如何让支出分配更合理、有效提升战力才是目前的首要问题。
 <br/>
 <br/>
 前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国防部.html" target="_blank">
  美国国防部
 </a>
 政策次部长Jim Miller：〝这表示更多拨款在研发和采购，不只是发展日常军事竞争能力，而是在有潜在冲突的领域。〞
 <br/>
 <br/>
 面对中、俄两大战略竞争对手，哪些应作为美国军费开支的优先项目，专家有不同建议。
 <br/>
 <br/>
 前美国国防部政策次部长Jim Miller：〝面对持续竞争，所谓的灰色地带，如网络、中国的南海军事因素等，我们要在灰色地带进行有效竞争，同时我们要威慑和阻止其它争端。〞
 <br/>
 <br/>
 布鲁金斯学会安全与战略高级研究员Frank A. Rose：〝一是战略核威慑能力的现代化，二是应对不对称战力的能力，三是保证我们拥有现代化的常规部队。〞
 <br/>
 <br/>
 新成立的美国国防战略委员会本月中向国会提交一份报告，警告美国的军力优势在消失，现有资源已无法实现其战略目标，必须增加五角大楼开支。周二(11月27日)，参议院军事委员会将就此展开听证。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
