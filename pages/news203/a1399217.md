### 加州野火肆虐死亡增至44人 损失逾190亿美元
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】大家好，欢迎收看全球新闻。连日来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州野火.html" target="_blank">
  加州野火
 </a>
 肆虐，使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇难人数.html" target="_blank">
  遇难人数
 </a>
 刷新历史纪录，南北加州合计44人。总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 已将加州野火列为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重大灾害.html" target="_blank">
  重大灾害
 </a>
 〞。
 <br/>
 <br/>
 当局周一晚报告，在北加州受灾最严重的〝天堂镇〞（Paradise），从烧毁的汽车和房屋中，又搜到13具遗骸，令〝营地野火〞（Camp Fire）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇难人数.html" target="_blank">
  遇难人数
 </a>
 增加到42人，成为加州史上最致命的火灾。而南加州的伍尔西野火（Woolsey Fire），火势仍然不减，目前已致2人死亡，3名消防员受伤。总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周二在白宫表示，已宣布加州山火为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重大灾害.html" target="_blank">
  重大灾害
 </a>
 ，将全力提供联邦资源救助。
 <br/>
 <br/>
 美国总统 川普：〝我们为失去的生命而哀悼，为（野火的）受害者祈祷，受害者比任何人想像的还要多。我们将尽一切力量支持和保护处于险境的同胞。〞
 <br/>
 <br/>
 此前，川普批评加州过于严格的环保政策令森林管理不善，导致野火频发。而州长布朗则将此归咎于气候变迁。
 <br/>
 <br/>
 记者 VS 加州州长 布朗：〝去年立法者在野火问题上花了很多时间和精力开会，为什么还不足以防止这种灾难？〞〝大火是史无前例的，就像我说的，这是一个新的非常态的现象。〞
 <br/>
 <br/>
 目前，南北加州的野火已造成近7000幢建筑被毁。据研究机构Enki Research的估计，野火给加州政府、保险公司及业主带来了至少190亿美元的损失。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州大火.html" target="_blank">
  加州大火
 </a>
 令两大公共电力公司太平洋瓦电（PG&amp;E）和爱迪生国际（Edison International）再次成为焦点。外界质疑，大风吹断电线再一次引发大火。去年10月的北加州酒乡大火事件中，太平洋瓦电被索赔170亿美元。
 <br/>
 <br/>
 截至周二中午，〝营地野火〞和〝伍尔西野火〞（Woolsey Fire）的过火面积分别达到12.5万英亩和9.6万英亩，已有30%以上的火场得到控制。空气品质监测显示，湾区的空气污染将持续到周五。
 <br/>
 <br/>
 新唐人记者林骁然旧金山报导
</div>
