### 截堵大篷车 军事设备已抵德州
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】川普政府周一宣布派驻5,200名现役军人到美墨边境，帮助截止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德州.html" target="_blank">
  德州
 </a>
 本地民众也拿起枪枝，协助巡逻。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 目前在南墨西哥，据报已从7000人减少到了4000人。他们还在用各种方式向美国行进，有的乘坐卡车，有的步行。
 <br/>
 <br/>
 周三，并有另一批约2000人的大篷车队伍从中美洲国家萨尔瓦多出发，准备前往美国。
 <br/>
 <br/>
 与此同时，被派往边境守卫的5000多美军也在积极筹备。
 <br/>
 <br/>
 美国军方将首先集中精力〝加强〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德州.html" target="_blank">
  德州
 </a>
 的边界安全，其次是亚利桑那州和加州。
 <br/>
 <br/>
 派驻边境的军人已经将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事设备.html" target="_blank">
  军事设备
 </a>
 装载到肯塔基州的运货飞机上，并在德克萨斯州卸载。
 <br/>
 <br/>
 同时，美墨边境的德州本地平民，也拿起枪枝，准备协助军队等边防人员守卫他们的土地。
 <br/>
 <br/>
 在美国边境，民兵和志愿者帮助巡逻的习惯已经有几十年了。通常平民在边境巡逻，一旦发现非法越境者，就会联系边境巡逻队逮捕他们。
 <br/>
 <br/>
 川普总统29日发推文指出：〝许多帮派分子及一些坏人也在混入大篷车难民队伍。〞并重申不会让他们进来。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
