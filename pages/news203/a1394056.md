### 美国南卡州7名警察执法遭枪击  1人殉职
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】周三晚，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南卡罗来纳州.html" target="_blank">
  南卡罗来纳州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警察执法.html" target="_blank">
  警察执法
 </a>
 中遭遇枪击，1人殉职，6人受伤。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌犯.html" target="_blank">
  嫌犯
 </a>
 已被捕，案件详情还在调查中。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 发生在南卡州佛罗伦萨郡西部的一个高档社区。三名警察执行搜查令时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌犯.html" target="_blank">
  嫌犯
 </a>
 埋伏在屋内向警察射击。
 <br/>
 <br/>
 佛罗伦萨郡警长Kenney Boone：〝警察到达时并不清楚嫌犯的火力，他们以为只是随机执行搜查令。〞
 <br/>
 <br/>
 赶来支援的四名警察也遭枪击，一名警察不幸殉职，六人受伤。
 <br/>
 <br/>
 佛罗伦萨郡警长Kenney Boone：〝（嫌犯）的火力射程在几百码，他有优势。警方无法靠近倒下的同伴。〞
 <br/>
 <br/>
 随后大批警察赶来支援。
 <br/>
 <br/>
 在与警方对峙的两个小时中，嫌犯劫持儿童做人质，所幸儿童没有受伤。为保护人质，警方不得不出动防弹车营救伤员。
 <br/>
 <br/>
 川普总统随后发表推文，为殉职警察祈祷。
 <br/>
 <br/>
 推文说：〝今晚我为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南卡罗来纳州.html" target="_blank">
  南卡罗来纳州
 </a>
 佛罗伦萨郡警察办公室和警察局祈祷，并致以哀思，我们永远感激执法人员一天24小时、一周7天、一年365天的付出。〞
 <br/>
 <br/>
 南卡州长麦克马斯特也发表推文，表彰执法人员。
 <br/>
 <br/>
 推文说：〝佛罗伦萨传来的消息真是令人震惊。我们执法人员真正无私而勇敢，就像祈祷一样真实有力。〞
 <br/>
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
