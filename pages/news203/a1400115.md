### 怂恿移民非法越境？自由派法官禁川普庇护令
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】美国政府10天前宣布拒绝非法越境者申请庇护，但周一（11月19日），加州旧金山市一名自由派
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 裁定，临时禁止该行政令执行。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府11月9日声明，抵达美墨边境的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民如非法跨越边境，将失去申请庇护的资格。
 <br/>
 <br/>
 但是，由前总统奥巴马任命的北加州的美国地区法院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 泰格（Jon S. Tigar）在旧金山举行了一场听证会，然后于周一发布了一道临时禁令，禁止川普政府推行拒绝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法入境.html" target="_blank">
  非法入境
 </a>
 移民申请庇护的行政令。
 <br/>
 <br/>
 这项诉讼案是由美国公民自由联盟（American Civil Liberties Union）和宪法权利中心（Center for Constitutional Rights）两个非营利组织提起的，他们在川普发布禁令后很快将政府告上了法庭。
 <br/>
 <br/>
 美国公民自由联盟的律师阿兹米（Baher Azmy）称，如果移民是在边境口岸之间穿越，就有权申请庇护。
 <br/>
 <br/>
 根据美国国土安全部（DHS）的报告，估计每年通过官方入境口岸进入美国的庇护申请者就有70,000人，而非法入境者一般会遭逮捕，然后再寻求庇护或其它形式的保护。
 <br/>
 <br/>
 周一，美国边境官员关闭了连接墨西哥和加州的最重要的圣伊西德罗入境口岸（San Ysidro Port of Entry），以便加紧安装障碍物、防止即将到来的数千大篷车移民非法越境。
 <br/>
 <br/>
 目前美国边防检查员正在该入境口岸处理打算从蒂华纳（Tijuana）进入圣地亚哥（San Diego）的中美洲移民的庇护申请，但每天只能处理100多份申请，而预计将至的移民大军至少会有几千人，有些先期到达的已开始试图攀爬篱笆强行入境。
 <br/>
 <br/>
 泰格的临时禁令将在全国范围内执行，有效期截至12月19日，届时将举行另一场听证会，或由法院进一步裁定。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
