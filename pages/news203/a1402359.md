### 川普将提名政治新手接任联合国大使
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】媒体披露，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ，将提拔国务院女发言人诺尔特，接替黑利担任美国驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国大使.html" target="_blank">
  联合国大使
 </a>
 。诺尔特此前只有很少的政治决策经验，因而引发关注。
 <br/>
 <br/>
 今年48岁的诺尔特，自去年4月起出任美国国务院发言人。之前，她是福克斯新闻频道的主播和记者，曾多次采访军政界高层人士，报道重大事件。
 <br/>
 <br/>
 今年3月，诺尔特被任命为负责公共外交事务的国务次卿。此前，她并没有相关从政经验。这一点可能会在参议听证中面临质询。
 <br/>
 <br/>
 现任美国驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国大使.html" target="_blank">
  联合国大使
 </a>
 黑利，今年10月提出辞呈，将在年底离职。黑利任职期间，在制裁朝鲜、支持以色列等议题上，不断表达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府的强硬立场。
 <br/>
 <br/>
 有分析认为，任命
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治新手.html" target="_blank">
  政治新手
 </a>
 诺尔特驻扎联合国，或许意味着川普将在以后的任期中，进一步将联合国的地位边缘化。
 <br/>
 <br/>
 川普本人此前多次发声，批评联合国运作不利。蓬佩奥几天前也再次质疑联合国是否发挥了应有的职能。
 <br/>
 <br/>
 新唐人记者林宇、天羽综合报导
</div>
