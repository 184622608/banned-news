### 选举证明中共票仓战失效 华日:美国农户并未抛弃川普
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】中美贸易战中，中共不顾自身损失，坚持向美国农产品加征关税，希望打击通过川普（特朗普）的〝票仓〞逼停贸易战。不过《华尔街日报》11月9日报导，美国中期选举结果显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国农民.html" target="_blank">
  美国农民
 </a>
 并没有表现出对川普的明确不满。
 <br/>
 <br/>
 报导指，北京原本以为自己加征的关税已经触到川普的痛点，但周二的中期选举结果显示，中共可能高估了自己的影响力。
 <br/>
 <br/>
 在关税战中，中共针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 、猪肉和高粱等农产品进行反击，其想法是打击川普在农村地区的支持基础。此举确实让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国农民.html" target="_blank">
  美国农民
 </a>
 遭受损失，比如美国大豆对华出口剧减，大豆价格也大幅下降。此外，美国大豆销售季刚刚开始，许多农户将面临更艰难时刻。
 <br/>
 <br/>
 不过，报导指出，中期选举结果表明，川普的农村选区并没有因此受到太大影响。在美国中西部和东南部大豆高产县的19场众议员选举中，共和党胜出10场，民主党只胜9场，这很难解读成农民对川普贸易政策的明确不满。
 <br/>
 <br/>
 此外，美国中部十个农业州的选举结果显示，这十个州中只有三个出现了参议院、众议院和州长的席次变化。
 <br/>
 <br/>
 虽然共和党在众议院的席位出现小幅下滑，但是2016年大选后确立的驴象两党的政治版图并没有发生太大变化，东西海岸的传统民主党控制区和中部的传统共和党控制区大都没有出现偏离。
 <br/>
 <br/>
 除了农业州的基本盘，川普对伊利诺伊州等地煤钢产业的扶持，也令他继续保持这些地区选民的支持。
 <br/>
 <br/>
 《华尔街日报》报导指，中期选举的结果，显然不是11月底川习会达成贸易和解的征兆。目前几乎没有迹象显示，中共会在美国的核心要求上作出让步，或者中共的反击足以让川普改变立场。到了12月初，人们或许将再次感受到贸易寒风的冰冷。
 <br/>
 <br/>
 彭博社11月7日的报导也指出，选举结果显示，中美贸易战并没有对受战事影响州的共和党候选人产生不利影响。相反，多名反对贸易战的候选人败北，支持对中共征税的候选人却胜出。因此，选举结果不会让川普软化贸易战的强硬立场。
 <br/>
 <br/>
 在Morning Consult今年8月公布的调查结果显示，从1月1日到7月22日期间，美国19个农业大州中，有10个州对川普的支持率明显上升，还有7个州的支持率坚挺不降。
 <br/>
 <br/>
 美国农业部长珀杜（Sonny Perdue）说，〝在感情上，他们（农民）是川普在贸易、劳工、监管等政策上的支持者，因为他们拥有川普所支持的价值观。〞
 <br/>
 <br/>
 另外，时事评论员田园博士9月对《大纪元》分析称，无论美国农民还是中产阶级，都看到川普的政策对美国长期发展是有益的。
 <br/>
 <br/>
 虽然中共关税令美国农民受损，但田园认为，实质上损失并不象中共宣传的那么大，美国大豆也不会烂在仓库里。因为大豆是中国的〝刚需〞，欧洲和巴西的大豆产量加在一起，都无法填补美国大豆造成的缺口，中国市场最终还是要消化美国大豆。
 <br/>
 <br/>
 事实上，美国大豆价格的下降，触发了其它国家抢购和囤积美国大豆的风潮。其中，许多国家的中间商通过把美国大豆转卖给中国获利。
 <br/>
 <br/>
 此外，川普政府也在积极扩展美国农产品的国际销路，今年还给农民补贴了120亿美元。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
