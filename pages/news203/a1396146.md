### 美国安顾问访俄 拟告知将退出1987年飞弹条约
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】美国总统川普的国家安全顾问博尔顿，预计下周前往莫斯科举行会谈，届时有可能谈及的议题，包括朝鲜、乌克兰和叙利，另外还将向莫斯科当局传达，华府计划退出双方1987年签订的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中程核飞弹条约.html" target="_blank">
  中程核飞弹条约
 </a>
 。
 <br/>
 <br/>
 路透社报导，1987年签订的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中程核飞弹条约.html" target="_blank">
  中程核飞弹条约
 </a>
 （Intermediate-Range Nuclear Forces Treaty，INF），是当时美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根.html" target="_blank">
  里根
 </a>
 （Ronald Reagan）和苏联领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戈巴契夫.html" target="_blank">
  戈巴契夫
 </a>
 （Mikhail Gorbachev）达成这项里程碑条约，美国国会次年审查通过。
 <br/>
 <br/>
 如今美方认为俄方已经违反协定。〝纽约时报〞（NYT）报导，川普政府一名资深官员表示，两任政府已试图要求俄方遵守协定，但尽管〝我们反对，俄罗斯仍继续生产和推出遭禁止的巡弋飞弹，并对透明度的要求视而不见。〞。而白宫官员并未否认这则报导。
 <br/>
 <br/>
 报导说，退出INF将对美国在亚洲的国防政策有重大冲击，对区域内主要战略对手中共也会带来不小影响。
 <br/>
 <br/>
 因中共不隶属于这项协定，且不惜成本积极发展传统飞弹，作为〝反介入/区域阻绝〞（anti-access/area denial，A2/AD）战略的一环。但INF却禁止美国持有射程介于500至5500公里的陆基弹道飞弹或巡弋飞弹。
 <br/>
 <br/>
 前美军太平洋司令哈里斯（Harry Harris）顾问、目前担任新美国安全中心（Center for a New American Security）副研究员的塞耶兹（Eric Sayers）表示，这对亚洲有重大影响，因〝这将赋予五角大厦和印度太平洋司令部新的传统选项，以恢复区域军事平衡。〞
 <br/>
 <br/>

    图为，美国总统川普的国家安全顾问博尔顿。(MANDEL NGAN/AFP/Getty Images)
   </font>
  </a>
 </center>
 <br/>
 预计博尔顿（John Bolton）将和俄国官员讨论朝鲜、乌克兰和叙利亚等重大议题外，也将谈及这项中程核飞弹条约。届时会告诉莫斯科当局，华府打算退出这项条约。
 <br/>
 <br/>
 俄罗斯新闻社（RIA）则引述克里姆林宫表示，俄罗斯总统普京计划会见博尔顿。报导中说，博尔顿预定22至23日在莫斯科出席会谈。
 <br/>
