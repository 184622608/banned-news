### 川普夫妇访迈克尔灾区 促改善灾民温饱
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】观众朋友大家好，欢迎收看新唐人全球新闻。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风迈克尔.html" target="_blank">
  飓风迈克尔
 </a>
 上周袭击了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国东部.html" target="_blank">
  美国东部
 </a>
 四个州。周一，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）和夫人亲自前往灾区探望民众。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统和夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼娅.html" target="_blank">
  梅拉尼娅
 </a>
 （Melania Trump）周一飞抵佛州艾格林空军基地，开始了一天的灾区视察。这个基地位于佛罗里达遭飓风侵袭的狭长地带，离飓风登陆地点约100英里。佛州州长斯考特到场迎接。
 <br/>
 <br/>
 川普首先拜访了受到飓风猛烈袭击的林恩港市，并和夫人一起为灾民分发瓶装水。
 <br/>
 <br/>
 川普称赞，联邦紧急事务管理署，地方政府和执法部门的工作〝非同反响〞，并表示，当前首要任务是保障居民的正常生活。
 <br/>
 <br/>
 美国总统川普：〝他们一些人连家的影子都看不到了。 你可能都不知道。他们的家被彻底摧毁了。我们最重要的事是（提供）食物，水和安全。〞
 <br/>
 <br/>
 截至目前，佛州至少有约19万人还没有电力供应。州长表示，将竭尽所能让这个城市恢复原状。他还向总统和第一夫人表示感谢。
 <br/>
 <br/>
 佛罗里达州长斯考特：〝我要感谢总统不厌其烦接我的电话，并亲自前来。我要感谢第一夫人到这里支持巴拿马城的居民。〞
 <br/>
 <br/>
 五天过去，四级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风迈克尔.html" target="_blank">
  飓风迈克尔
 </a>
 留下的痕迹仍触目惊心。CBS新闻在推特上公布了无人机拍摄的佛州墨西哥海滩市的灾后重建工作。昔日房屋已变成一堆堆散乱在地基上的木板，各式家俱和日用品散落其间。即便依然屹立的房屋，也遭受了不同程度破损。而据CNN报导，该市仍有3人失踪。
 <br/>
 <br/>
 周一，飓风在佛罗里达、弗吉尼亚、乔治亚和北卡州造成的死亡人数已上升到18人，而这个数字很可能还会增加。
 <br/>
 <br/>
 周一稍晚，川普和夫人拜访乔治亚州灾区后，就启程返回华盛顿。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
