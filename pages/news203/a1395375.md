### 川普首次放话 马蒂斯可能离职
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在周日播出的〝CBS60分钟〞访谈节目中，首次提到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国防部长.html" target="_blank">
  国防部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蒂斯.html" target="_blank">
  马蒂斯
 </a>
 可能
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离职.html" target="_blank">
  离职
 </a>
 。印证了外界的一些猜测。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在预先录制的访谈节目中说：〝我和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蒂斯.html" target="_blank">
  马蒂斯
 </a>
 关系很好，两天前还一起午餐。他有可能
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离职.html" target="_blank">
  离职
 </a>
 。说实话我觉得他像民主党人。但是马蒂斯将军是一个好人，我们相处融洽。〞
 <br/>
 <br/>
 五角大楼发言人 陆军上校 曼宁对此回应：〝马蒂斯防长一直专注工作，保证美国是世界上最强大的军事力量。〞
 <br/>
 <br/>
 2016年，川普任命马蒂斯为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国防部长.html" target="_blank">
  国防部长
 </a>
 。身为海军陆战队退役上将，马蒂斯曾在奥巴马时期担任美国中央司令部司令。
 <br/>
 <br/>
 美国国防部长马蒂斯：〝请相信，对美国和我们盟友的任何攻击，将以失败告终。〞
 <br/>
 <br/>
 美国媒体分析，马蒂斯与川普总统在不同问题上存在分歧，他倾向于保留奥巴马时期签订的伊朗核协议。而川普一直称这个协议极其糟糕，今年早些时候恢复了对伊朗的制裁。
 <br/>
 <br/>
 此外，川普一直要求北约盟国贡献更多国防预算。而马蒂斯认为与北约盟友的关系有助于维护国际秩序。
 <br/>
 <br/>
 他还认为全球暖化问题是实质的威胁，但是川普认为那是环保组织的政治工具。
 <br/>
 <br/>
 为此，川普称马蒂斯更像民主党。
 <br/>
 <br/>
 川普表示：〝我们的内阁很棒。我对有些人不太满意，有的人我觉得不怎么样。但是有的人非常出色。〞
 <br/>
 <br/>
 目前马蒂斯还没有辞职迹象，他周二将再次访问越南。
 <br/>
 <br/>
 米拉・里卡德尔（Mira Ricardel）今年五月作为副国家安全顾问进入白宫后，外界就猜测川普可能在中期选举后让马蒂斯离职。
 <br/>
 <br/>
 新唐人电视综合报导
