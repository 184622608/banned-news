### 演说时遭〝枪击〞 里根总统的反应你想不到
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】美国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根.html" target="_blank">
  里根
 </a>
 （Ronald Wilson Reagan，或译
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雷根.html" target="_blank">
  雷根
 </a>
 ）被视为该国最伟大的总统之一，素以高明而极具说服力的演说闻名。而在一次公开演说中，这位颇具幽默感的总统遭遇〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击.html" target="_blank">
  枪击
 </a>
 〞，仍不改其幽默本色，做出让很多人都意想不到的临场反应。
 <br/>
 <br/>
 1987年6月12日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根.html" target="_blank">
  里根
 </a>
 搭乘专机抵达德国柏林滕珀尔霍夫机场（Tempelhof Airport），参加柏林建城750周年的庆祝活动。
 <br/>
 <br/>
 里根在该机场的美国空军基地对数百名美国人发表演说。会场的四周装饰着很多不同颜色的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_气球.html" target="_blank">
  气球
 </a>
 ，十分热闹。
 <br/>
 <br/>
 正当里根在发表演说时，一个气球突然爆掉，发出〝碰！〞的一声，听起来就像有人开枪进行刺杀。此时他中断演说，面不改色地说：〝没打中我。〞（Missed me.）
 <br/>
 <br/>
 里根这个幽默的反应让在场观众哈哈大笑，随即报以热烈的掌声。而他的回应也让人想起他曾在1981年3月30日遭遇真实
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击.html" target="_blank">
  枪击
 </a>
 的往事。当时才上任两个多月的他在华府一家酒店被一名精神异常的男子开枪射中肺部，导致严重内出血，但因抢救及时得以迅速康复。
 <br/>
 <br/>
 据合众国际社报导，里根在柏林这场美国式的庆祝活动中也与现场观众合唱《祝你生日快乐》替柏林庆生，而当年已有76岁的他还幽默地告诉众人说：〝你不常参加比我老的某人（或某事）的庆生会。〞
 <br/>
 <br/>

 <br/>
 （〝没打中我。〞这句话在5分21秒处）
 <br/>
 <br/>
 同样在1987年6月12日当天，里根也在东西柏林交界的布兰登堡门（Brandenburg Gate）前发表著名的历史演说，呼吁苏联领导人进行政治与经济改革，并推倒柏林围墙。
 <br/>
 <br/>
 里根在该演说中说的〝推倒这堵墙！〞（Tear down this wall!）这句话，成了他的经典名言，为世人广为传颂。
 <br/>
 <br/>

 <br/>
 <br/>
 而在里根发表这篇演说29个月后，柏林围墙真的倒塌了，苏联不久后也随之解体。1990年9月，已卸任美国总统职务的里根再度造访柏林。他亲自举起鎚子，象征性地敲击柏林围墙的遗迹，为这段历史做见证。3270039
