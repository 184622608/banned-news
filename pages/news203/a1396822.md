### 川普〝历史性措施〞 降低处方药价
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】周四，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 宣布，将采取一系列措施降低
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_处方药价.html" target="_blank">
  处方药价
 </a>
 格，保护美国消费者的利益。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：〝同一家药厂，同样的包装，同一个地方制造的同样药品，部分国家的药价是我们的20%，还有更低的。〞
 <br/>
 <br/>
 川普表示，过去几十年来，美国民众在处方药上的支出，远高于德国和法国等欧洲国家。针对27种药物调查，发现某些情况下，美国人的花费是其他国家的四倍。
 <br/>
 <br/>
 美国卫生及公共服务部，将出台新的〝国际药品价格指数〞支付模型，分阶段下调美国药的成本，最终与欧洲国家的价格一致。
 <br/>
 <br/>
 卫生部门估计，新的支付模式将在五年内节省172亿美元。2019年春季，将对海外定价模式制定正式法规。
 <br/>
 <br/>
 美国总统川普：〝过去几十年来，其他国家一直在操纵药价系统，让美国患者的负担越来越重。我们采取的每一个行动都是将美国放在第一。〞
 <br/>
 <br/>
 新唐人记者曹菁综合报导
</div>
