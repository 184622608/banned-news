### 迈阿密飞墨西哥班机传安全疑虑 美航：不可信威胁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国航空.html" target="_blank">
  美国航空
 </a>
 一架编号257的班机，周三（24日）计划飞往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 市，但因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安全问题.html" target="_blank">
  安全问题
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈阿密国际机场.html" target="_blank">
  迈阿密国际机场
 </a>
 被警方调查。

 <br/>
 稍后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国航空.html" target="_blank">
  美国航空
 </a>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 市编号257班机有关的安全疑虑是〝应可立即排除的不可信威胁〞。
 <br/>
 <br/>
 路透社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈阿密国际机场.html" target="_blank">
  迈阿密国际机场
 </a>
 （Miami International Airport）稍早在推特上说，这架美国航空班机乘客已经全部撤离，警方正在调查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安全问题.html" target="_blank">
  安全问题
 </a>
 。
