### 佛州2县故意拖延重新计票 卢比奥曝真实原因
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 共和党参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢比奥.html" target="_blank">
  卢比奥
 </a>
 （Marco Rubio）周二（11月13日）揭露，该州布劳沃德县（Broward）和棕榈滩县（Palm Beach）故意拖延
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 ，目的在于最终窃取佛州的两项重要选举。
 <br/>
 <br/>
 周二，最先披露布劳沃德县涉嫌选举舞弊的共和党参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢比奥.html" target="_blank">
  卢比奥
 </a>
 再次揭露，布县和棕榈滩县故意耽误
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 ，目的是取得这两个倾向于民主党的县的法官裁决，以算入在投票截止日期以后出现的不合规选票，从而增加民主党候选人胜选的可能性。
 <br/>
 <br/>
 周六，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 州务卿戴兹特（Ken Detzner）宣布，鉴于该州州长和参议员选举的计票结果非常接近，因而启动全州范围的重新计票，各县重新计票结果须于美东时间周四（11月15日）下午3点前出台。
 <br/>
 <br/>
 但周日，棕榈滩县公然违抗州务卿命令，声称该县的计票设备老旧，无法在周四的截止期限前完成重新计票，并要求戴兹特延长截止时间，但遭拒绝。
 <br/>
 <br/>
 据美国保守派网站〝Gateway Pundit〞周二报导，虽然福克斯新闻（Fox News）此前报导布县已从周日上午11:30开始启动重新计票（推迟了4.5小时），但周二上午，福克斯表示赞同《迈阿密先驱报》（Miami Herald）的消息，即至周二上午布县根本还没启动重新计票工作。
 <br/>
 <br/>
 迈阿密-戴德县（Miami-Dade County）周六晚即已启动重新计票，该县共有800,000多张选票，至周一中午， 计票工作已经过半。
 <br/>
 <br/>
 卢比奥指出，布劳沃德县和棕榈滩县之所以拖延计票，旨在等待法官裁决，以便该县在选举夜7:00后收到的有利于民主党的临时选票也能够入账。
 <br/>
 <br/>
 但是，这种做法是违法佛州法律的。
 <br/>
 <br/>
 卢比奥一针见血地指出，这两县破坏法律的选举官员给民主党律师们窃取选举制造了机会。这些律师抵达佛州并非为了确保每张选票都被计入，而是想尽办法增加民主党候选人的选票、同时拉低其竞选对手的选票数量。
 <br/>
