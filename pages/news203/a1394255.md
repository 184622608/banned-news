### 摇摆票宣布立场 卡瓦诺确认大局已定
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】周五（10月5日）下午，美国共和党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议员.html" target="_blank">
  参议员
 </a>
 柯林斯（Susan Collins）发表声明，宣布支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 （Brett Kavanaugh）就任最高法院大法官；随后，民主党参议员曼钦（Joe Manchin）也声明支持卡瓦诺，这两张关键
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摇摆票.html" target="_blank">
  摇摆票
 </a>
 的定位基本稳定了周六表决的大局。
 <br/>
 <br/>
 周五下午，缅因州共和党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议员.html" target="_blank">
  参议员
 </a>
 柯林斯发表了长达45分钟扣人心弦的演说，阐明了她对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 确认问题的立场——她将在周六表决时支持卡瓦诺。
 <br/>
 <br/>
 柯林斯在参议院发表声明期间，现场气氛异常紧张。开场仅几秒钟，自由派抗议者便打断了她的发言，并冲她高喊，敦促她投反对票。
 <br/>
 <br/>
 参议院已连续几天被自由派抗议者淹没。同时，当地抗议首领还指示抗议人群冲击柯林斯位于缅因州的办公室以及另一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摇摆票.html" target="_blank">
  摇摆票
 </a>
 参议员穆考斯基（Lisa Murkowski）位于阿拉斯加州的办公室。
 <br/>
 <br/>
 柯林斯的发言冗长，详细解释了她最终
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_决定.html" target="_blank">
  决定
 </a>
 支持卡瓦诺的原因。
 <br/>
 <br/>
 柯林斯批评民主党的阻挠行动，嘲讽卡瓦诺的确认过程类似〝阴沟级政治运动的漫画〞（caricature of a gutter-level political campaign）。
 <br/>
 <br/>
 柯林斯特别谈到了遵守参议院正当程序的传统以及无罪推定的重要性，并指出，针对卡瓦诺的未经证实的性侵指控不应阻止他在最高法院任职。
 <br/>
 <br/>
 在柯林斯宣布决定后不久，西维吉尼亚州民主党参议员曼钦（Joe Manchin）也声明，他将投票支持卡瓦诺。他随后受到抗议者攻击。
 <br/>
 <br/>
 此前摇摆不定的亚利桑那州共和党参议员弗莱克周五也表示，〝除非发生重大变化〞，他打算周六投赞成票。
 <br/>
 <br/>
 虽然共和党参议员穆考斯基周五下午表示她将投反对票，但目前的赞成和反对票数之比是51:49，如果没有特殊情况发生，共和党对表决结果显然已稳操胜券。
 <br/>
 <br/>
 卡瓦诺的确认投票将于周六晚举行，大约从美东时间下午5点开始。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
