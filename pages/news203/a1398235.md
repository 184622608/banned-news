### 响应国家征召 美犹他州市长在阿富汗遇袭身亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹他州.html" target="_blank">
  犹他州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北奥格登市.html" target="_blank">
  北奥格登市
 </a>
 市长泰勒，响应国家第4次征召，赴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 协助当地陆军卫队进行训练，不料上周六(3日)在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_喀布尔.html" target="_blank">
  喀布尔
 </a>
 的基地，遭一名阿富汗成员袭击身亡，得年39岁。
 <br/>
 <br/>
 泰勒（Brent Taylor）生于亚利桑那州，2001年911恐攻事件后，和5个兄弟一起入伍。2009年成为居民约1.9万人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北奥格登市.html" target="_blank">
  北奥格登市
 </a>
 （North Ogden）市议员，2013年被选为市长，四年后成功连任。
 <br/>
 <br/>
 北奥格登市民大多肯定泰勒，称他一直努力改善城市，与他政见不同的人也说，泰勒处事公允，不会把自己的共和党意识形态融入施政。
 <br/>
 <br/>
 泰勒自退伍后，仍作为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹他州.html" target="_blank">
  犹他州
 </a>
 陆军国民兵的一员，两度被召去伊拉克、一次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿富汗.html" target="_blank">
  阿富汗
 </a>
 。今年1月，他响应第4次征召，透过脸书向市民表示，必须离开一年至阿富汗，协助训练该国陆军突击队；他离开该市时，数百名居民在街上为他送行。
 <br/>
 <br/>
 身为犹他州国民卫队少校，泰勒被派去阿富汗，是为了协助当地陆军卫队进行训练。上周六，阿富汗部队一名成员忽然袭击美军，造成泰勒当场死亡，另有一名同袍受伤，杀手当场被撃毙。随后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塔利班.html" target="_blank">
  塔利班
 </a>
 （Taliban）承认犯案。
 <br/>
 <br/>
 据了解，驻阿富汗美军人数，自2011年的10万人，现下降至不到1.4万人。美军在该地的主要工作是为阿富汗军队提供建议和训练，非与极端组织塔利班战斗。
 <br/>
 <br/>
 泰勒与妻子结婚15年，共育有7名子女，长女年仅13岁，么女11个月大。对于泰勒捐躯，家属表示心痛，〝但不后悔〞。
 <br/>
 <br/>
 泰勒上月28日还透过脸书发布讯息，谈及阿富汗进行了8年来首次的议会选举，并鼓励美国公民在期中选举踊跃行使投票权，〝无论共和党赢，或是民主党赢，我们都要记得，要让更多美国人团结，而不是分裂。天佑美国〞。
 <br/>
 <br/>
 泰勒遗体将送回美国空军基地，北奥格登市政府为他降半旗以示尊敬，民众也纷纷到他的脸书留言追悼。
 <br/>
 <br/>
 (责任编辑：卢勇信)
</div>
