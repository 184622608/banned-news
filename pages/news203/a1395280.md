### 博尔顿:中共是〝世纪重大议题〞 将有更强硬措施
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】近期，时不时就有美国政要通过接受媒体采访或借出席公开活动之机，谈论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 的问题，并向北京方面公开喊话。日前，美国国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 （John Bolton）再次就中美贸易冲突强硬表态。他表示，美国要对中采取更强硬的措施，中共在贸易、国际、军事和政治领域上都应该作出调整。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 是上周五在接受Hugh Hewitt Show电台的采访时作出了上述表态。
 <br/>
 <br/>
 据路透社报导，博尔顿10月12日在Hugh Hewitt Show电台播出的一档节目中说，美国政府将中国视为〝本世纪的重大问题〞，而总统川普认为，中共占国际秩序便宜的时间太长了，却一直没有足够多的美国人站出来应对，〝现在是时候这么做了〞 。
 <br/>
 <center>
  <br/>

   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-14/p9079201a819111809-ss.jpg"/>
   <br/>
   <font size="-1">
    美国国家安全顾问约翰博尔顿10月12日接受媒体采访时表示，美国要对中采取更强硬的措施，中共在贸易、国际、军事和政治领域上都应该作出调整。图为博尔顿于2018年10月3日在华盛顿特区白宫举行的新闻发布会上发表讲话。(Alex Wong/Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 <br/>
 博尔顿表示，川普对中共政府的强硬态度让对方 〝感到困惑〞。他说：〝他们（中共）从未见过有美国总统如此强硬。我认为，他们的行为需要在贸易领域、国际、军事和政治领域以及一系列领域进行调整。〞
 <br/>
 <br/>
 他还指出，中共过去在贸易和商业方面，通过违反国际规范而获得了可观的经济利益并增长了军事实力，〝若他们被放回适当的位置，不再被容许窃取我们的科技，他们的军事实力将会大大降低，我们今天看到的很多紧张局势就会减少。〞
 <br/>
 <br/>
 博顿透露说，美国政府准备采取更多行动，限制对中共出口〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_敏感高科技.html" target="_blank">
  敏感高科技
 </a>
 〞。最后他证实，川普总统可能会与中国国家主席习近平在G20峰会上会晤并讨论上述议题。
 <br/>
 <br/>
 此前，美国财长姆钦昨在印尼峇里岛出席国际货币基金组织（IMF）和世界银行年会期间接受CNBC访问时，也曾透露中美双方正计划两国首脑在下月阿根廷布宜诺斯艾利斯G20峰会期间会晤的问题。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：明轩）
