### 芝加哥华埠慈爱医院爆枪击案 酿4死
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】美国伊利诺州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芝加哥.html" target="_blank">
  芝加哥
 </a>
 官员周一（19日）表示，当地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_慈爱医院.html" target="_blank">
  慈爱医院
 </a>
 下午发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，造成4人丧生，其中包括一名枪手和一名警察。
 <br/>
 <br/>
 事发在19日下午3时30分左右，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芝加哥.html" target="_blank">
  芝加哥
 </a>
 南部的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_慈爱医院.html" target="_blank">
  慈爱医院
 </a>
 (Mercy Hospital)一名歹徒先在慈爱医院外的停车场开枪，接着冲进医院大厅扫射。
 <br/>
 <br/>
 事发时人在急诊室的怀特（Steven White）告诉美媒，他听到若干枪响，赶紧朝紧急窗口冲，他看到枪手就在那里朝警察开枪，枪手一身军事风格。
 <br/>
 <br/>
 他说，当他往外看时，看到有一个人倒卧在医院大门口，枪手转身再次开火，接着往院内跑。
 <br/>
 <br/>
 怀特说：〝我无法形容我的感受。连到医院接受治疗都有危险，真悲哀。〞
 <br/>
 <br/>
 另一位附近自行车店经理希梅内斯（Sue Jimenez）说，她听到多声间断枪响，枪手还一度连开5枪，警方抵达后枪声就停了。
 <br/>
 <br/>
 警方发推文称，警察希梅内斯（Samuel Jimenez）殉职；芝加哥消防局则发推文证实枪手同样身亡。另医院2名女性员工伤重不治。
 <br/>
 <br/>
 芝加哥警方消息来源表示，凶嫌为一名32岁男性，尚未就枪手动机发表评论。
 <br/>
 <br/>
 慈爱医院位于25街与南密西根大道间，距芝城华埠约5分钟车程，许多华人都会到该医院就诊，附近一些老公寓也有狠多华人聚居。
