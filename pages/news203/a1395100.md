### 纽约月子中心刺婴案 华人月嫂屡自杀未遂
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法拉盛.html" target="_blank">
  法拉盛
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_月子中心.html" target="_blank">
  月子中心
 </a>
 刺杀婴儿案，震惊整个社区。华人月嫂王玉芬行凶后曾试图自杀，被捕后又2次自杀未遂。其家人称，之前她就告诉家人不想活了。
 <br/>
 <br/>
 据纽约时报报导，事发于9月21日凌晨3点多，地点在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法拉盛.html" target="_blank">
  法拉盛
 </a>
 一幢三层高的砖砌公寓楼里。
 <br/>
 <br/>
 52岁的王玉芬（Yufen Wang，音译）用刀刺向在摇篮中熟睡的3名婴儿和旁边的2名大人，随后她将刀转向自己的脖子和手腕，试图自杀。
 <br/>
 <br/>
 王玉芬来自中国大陆福建省的一个贫困地区。她在丈夫彼得获得美国工作签证后，于2010年带着两个成年的儿子来到纽约。
 <br/>
 <br/>
 彼得和两个儿子一起在餐馆工作。王玉芬开始做家政保洁和照料孩子的工作。大约2年后，她开始在美心公司位于法拉盛市中心附近的第一个分部工作。
 <br/>
 <br/>
 但因为她的丈夫有腿疼的毛病，家庭的重担落在了王玉芬身上。在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_月子中心.html" target="_blank">
  月子中心
 </a>
 照顾完新生儿之后，回到家里，她还得照顾3个幼小的孙辈。
 <br/>
 <br/>
 王玉芬的儿子丹尼（Danny）说，大约6个月前，他们就发现王玉芬变了。她总是忘带钥匙和手机，变得无法集中精力，无法入睡，体重也减轻了很多。后来她的精神状态明显恶化了。
 <br/>
 <br/>
 她的儿子说，大约2～3个月前，王玉芬告诉丈夫，她的人生毫无意义，她不想活了。彼得坚持要她休息2周，因为他们都知道，王玉芬在中国的姐姐曾经自杀未遂。
 <br/>
 <br/>
 丹尼说，母亲去看了医生，开了安眠药和镇静剂。但之后美心的老板又叫她回去上班，从晚上8点上到早上8点，不料却发生刺婴事件。王玉芬被指控犯有5项谋杀未遂罪，她否认指控，并在里克斯岛接受自杀监护。
 <br/>
 <br/>
 王玉芬的律师计划为她做精神障碍辩护。她的家人和律师说，王玉芬在监狱里变得更加离群和烦躁，她还企图撞墙自杀，在第一次尝试自杀失败后，她还试图咬掉自己的舌头，这是一种华人特有的自杀方式。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美心月子中心.html" target="_blank">
  美心月子中心
 </a>
 〞实际上是私人住宅改造而成，在纽约和新泽西有约40家这样的月子中心，法拉盛就有近20家，它们只在网上做广告。
 <br/>
 <br/>
 当地的职业中介机构表示，美心月子中心创办人从事该行业已有近10年，事发地点是其2016年开设。据了解，当年两人以150万美元买下了这幢楼。
 <br/>
 <br/>
 报导称，一位邻居表示，她看见经常有客人出入，有时候乘坐的是高级轿车。
 <br/>
 <br/>
 这些月子中心逃避城市和州许可的分类和分区法规。它们不具备日托中心的资格，因为有婴儿母亲在场；它们也不需要医疗执照，因为所有者提供中式的营养看护。
 <br/>
 <br/>
 虽然美心月子中心的邻居提出过投诉，称其作为酒店在运营，但城市建筑检查员三次被拒绝进入，导致投诉被自动关闭。
 <br/>
 <br/>
 行凶事件发生后，警察关闭了美心月子中心，但不到3周，该中心似乎又重新开放。透过窗户可以看到一些女人，一堆尿布放在外面。
 <br/>
 <br/>
 纽约市议员顾雅明和法拉盛民选社区领袖玛莎（Martha Flores-Vasquez）案发后召开记者会，敦促州、市政府对月子中心加强规管，并对月嫂制定培训规范，同时要求持证上岗。
 <br/>
 <br/>
 他们也提醒，儿童照护这类机构在雇请员工时，尤其要注意考察应聘者的背景及精神、心理、身体状体。同时，雇主不能让月嫂工作太长时间，从事月嫂的人一般年纪较大，如果劳累，容易造成情绪、心理不稳定，对婴儿、孕产妇不利。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
