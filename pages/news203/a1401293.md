### 美国国会听证  促制止中共迫害宗教
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 到来之际，美国国会举行听证会，关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教信仰.html" target="_blank">
  宗教信仰
 </a>
 团体的严酷打压，并呼吁川普总统在即将举行的川习会上，直接敦促中共停止迫害。
 <br/>
 <br/>
 美国会及行政部门中国委员会主席-参议员 Marco Rubio：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教信仰.html" target="_blank">
  宗教信仰
 </a>
 及团体的镇压是真实的、邪恶的、骇人听闻到令人无法忽视。〞
 <br/>
 <br/>
 美国联邦参议员 Steve Daines：〝在中国绝大部份地区，宗教自由状况处于可怕困境中。
 <br/>
 <br/>
 美国国会及行政当局中国委员会举行的听证会上，议员们关切中共对宗教信仰团体日益严酷的迫害，以及利用统战部对宗教与民族事务进行更直接管控。
 <br/>
 <br/>
 美国会及行政部门中国委员会主席 参议员 Marco Rubio：〝法轮功学员被送入法制教育中心进行洗脑灌输。〞
 <br/>
 <br/>
 美国会及行政部门中国委员会共同主席 众议员 Chris Smith：〝高智晟、江天勇和其他很多人，被消失、被关押及遭受酷刑，由于站出来反对迫害。〞
 <br/>
 <br/>
 出席作证的包括曾三次被中共关入新疆再教育营的维吾尔族穆斯林，也有宗教问题专家、学者。在揭露残酷迫害的同时，他们提醒国际社会，不要对中共抱有任何幻想。
 <br/>
 <br/>
 澳大利亚战略政策研究所研究员 Samantha Hoffmanand：〝问题不在于中国或中国人民，而是在于中共。除非中共的本质发生改变，否则这一问题(宗教迫害)绝不会消失。〞
 <br/>
 <br/>
 日前，中共驻美大使崔天凯扬言，如果美国对涉嫌侵犯人权的中共新疆官员进行制裁，中共将〝按比例〞进行报复。
 <br/>
 <br/>
 美国会及行政部门中国委员会主席 参议员 Marco Rubio：〝底线就是，美国的外交政策与人权承诺，不会决定于是否会受到报复。我才不在乎他们的报复。这不应阻止我们做正确的事，即呼吁人们关注迫害。〞
 <br/>
 <br/>
 对即将在阿根廷举行的川习会，国会议员希望川普总统，直接敦促中共，停止宗教迫害。
 <br/>
 <br/>
 美国会及行政部门中国委员会共同主席 众议员 Chris Smith：〝我们在私下，也在公开场合向总统传递这一信息，提出人权问题，提出对信仰者的镇压问题，法轮功、基督徒、西藏佛教徒、以及目前对维吾尔族穆斯林的可怕镇压，并对中共政府说，停止迫害！〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导。
</div>
