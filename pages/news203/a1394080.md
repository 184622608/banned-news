### 彭斯批中共窃美技术 敦促Google停止研发〝蜻蜓〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】周四（10月4日），美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 在美国智库〝哈德逊研究所〞（Hudson Institute）发表重要演说，披露
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 干涉美国内政、外交、经贸及军事等各方面，并严厉批评中共盗窃美国技术，同时敦促
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Google.html" target="_blank">
  Google
 </a>
 停止帮助中共审查互联网。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）在演说中谴责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的互联网审查和宗教迫害政策。他提到，中共当局迫害人权，迫害中国宗教信仰人士，同时在新疆建立〝集中营〞进行洗脑活动；中共正在利用美国的技术，来监视本国的人民。他说，之前的美国政府忽视中共的行为，但那些日子已经过去了。
 <br/>
 <br/>
 彭斯还表示，如果进入中国市场意味着要向中方转交知识产权，或受北京胁迫，那么，美国企业应慎重考虑。
 <br/>
 <br/>
 他说：〝例如，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌.html" target="_blank">
  谷歌
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Google.html" target="_blank">
  Google
 </a>
 ）应该立即终止研发‘蜻蜓’搜寻应用程式，因为它会加强中共在互联网方面的审查，牺牲中国民众的私隐。〞
 <br/>
 <br/>
 彭斯还谴责了中共瓦解台湾邦交国的行为，他强调，〝我们虽然坚持‘一个中国’的政策，但是，我们认为台湾的道路是对的。〞
 <br/>
 <br/>
 <h2>
  彭斯：中共正在干涉美国内政
 </h2>
 <br/>
 彭斯在讲话中指责中共企图打击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统，并警告中共说，其若不改变不公平的贸易行为，美国将会征收更多的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 。
 <br/>
 <br/>
 彭斯说：〝这位总统不会退缩，美国人民不会动摇。即使我们希望改善与北京的关系，我们的政府也将继续坚定地保障我们的安全和经济。我们的政府将继续采取果断行动，以保护美国的利益，美国的就业机会和美国的安全。〞
 <br/>
 <br/>
 〝我们将继续要求与中国建立自由，公平和互惠的经济关系。我们将要求北京消除其贸易壁垒，履行其义务，完全开放其经济，就像我们开放我们的经济一样。我们将继续对北京采取行动，直到结束知识产权盗窃。〞
 <br/>
 <br/>
 彭斯继续说：〝我们的国家安全战略写道，竞争并不总意味着敌对〞，〝正如川普总统所说，我们不想让中国市场遭受损失。事实上，我们希望他们兴旺繁荣。但美国想让北京追寻自由、公平和互惠的贸易政策。我们将会坚持并要求他们这样做。〞
 <br/>
 <br/>
 不过，〝可悲的是，到目前为止，中共统治者拒绝走这条道路。美国人民有权利知道（真实情况）。〞彭斯在讲话中指出，中共正在以全政府的方式，包括政治、经济和军事工具，以及宣传来提升其影响力，并使其在美国的利益受益。中共也正在以〝更加主动而强制的方式，干涉美国国内政策和政治。〞彭斯说。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：东野）
</div>
