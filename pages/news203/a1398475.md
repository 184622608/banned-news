### 专家谈中期选举结果 看美国政治分裂
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】这次美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的结果，对美国的发展意味着什么，来听聼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专家.html" target="_blank">
  专家
 </a>
 的分析。
 <br/>
 <br/>
 在周三中午召开的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 后的新闻发布会上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）带了一条蓝色领带，并强调〝团结〞一词。
 <br/>
 <br/>
 执政党在中期选举中丢掉国会席位在美国历史上似乎已成规律，〝两党政策中心〞民主项目主任 John Fortier认为，川普政府未来会在外交政策上更侧重。
 <br/>
 <br/>
 〝两党政策中心〞民主项目主任 John Fortier：〝他们会在外交政策上花更多时间，因为这不需要国会的太多授权，他们可以依据自己的经验而行。〞
 <br/>
 <br/>
 川普在记者会问答环节宣布，中共将取消〝中国制造2025〞计划。
 <br/>
 <br/>
 国际政治风险顾问公司Corr Analytics负责人 Anders Corr博士：〝我认为在中国问题上两党没有太大分歧。总体上在美国，民主党选民和共和党选民对中国都持审慎态度，希望能够更多的保护美国经济及政治系统。未来两年美国对华还会保持强硬态度。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专家.html" target="_blank">
  专家
 </a>
 认为，整体上，本次的选举结果，反映出美国在政治上无法调和的现状。
 <br/>
 <br/>
 〝美国民主研究所〞主任 Patrick Basham：〝有人认为当国会内部两党对立时，是最好的状态，因为两党都要妥协，国家会更好。但我不得不说，近年来、特别是今天，两党在所有主要议题上都太对立了，而且都没有妥协的意愿，他们各自的选民也不同意他们妥协，不妥协保持对立，导致僵局。缺少妥协而导致的僵局反映出美国政治上的分裂。〞
 <br/>
 <br/>
 一些具体选战则反映出美国面临的威胁，比如纽约的社会主义参选人奥卡西奥-科特兹成功当选国会众议员。
 <br/>
 <br/>
 时政评论人士 横河：〝这次选举，共和党建制派逐渐和川普力量结合，民主党只有一小部分脱离建制派、快速转向极左的社会主义，尤其在传统的左派大本营，主要在纽约、加州、还有一些大学。大学里的左派洗脑已经不只一代人，越战时期反战的嬉皮士运动，大学也是主要阵地。〞
 <br/>
 <br/>
 虽然民主党的众议院会对川普新政制造阻碍，但除了国会立法机构，川普要推进改革，还有其他工具可以借助，比如行政令、行政规定、或者通过总统的话语特权直接告诉美国人重大议题是什么。另外共和党对参院的控制权也能在行政上帮助川普。
 <br/>
 <br/>
 〝两党政策中心〞民主项目主任 John Fortier：〝共和党在参议院保持多数，这样共和党可以继续确认法官和行政人员的任命。〞
 <br/>
 <br/>
 有人说，2020大选的竞争从今天已经开始，不少人认为本次中期选举的结果是2年后大选的风向标，不过也有专家有不同看法。
 <br/>
 <br/>
 〝两党政策中心〞民主项目主任 John Fortier：〝中期选举的结果并不能作为两年后总统大选的风向标，我们看到里根、克林顿和奥巴马在首次任期的中期选举中都丢掉很多席位，但在大选中却大获全胜。2020会是不同的情况，所以今天民主党控制众院，不意味着川普在2020会不利，我们还要看。〞
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
