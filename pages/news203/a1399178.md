### 北加州野火死者数升至42 刷新加州纪录
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州野火.html" target="_blank">
  加州野火
 </a>
 持续燃烧，其中，北加州再发现13具罹难者遗体，使死亡人数达到42人，成为加州历史上最致命的野火灾害。此外，南加州也有2人死亡。
 <br/>
 <br/>
 南北
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州野火.html" target="_blank">
  加州野火
 </a>
 持续燃烧，尚未获得控制，死亡与失踪人数继续攀升。
 <br/>
 <br/>
 加州巴特县警长哈尼亚：〝今天，再发现13具罹难者遗体，使得（北加州）死亡总数达到42人。就我所知，这是美国史上最致命的野火灾害。〞
 <br/>
 <br/>
 北加州野火已造成位于旧金山东北部180英里、拥有2万7千名居民的天堂镇全毁。
 <br/>
 <br/>
 接下来干燥气候还将持续，除非奇迹出现，降雨机会渺茫。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家气象局.html" target="_blank">
  国家气象局
 </a>
 气象学家布伦恩：〝我知道很多人都很关心下次何时下雨。我们没有看到任何降雨迹象，下周或可能感恩节都不会有。〞
 <br/>
 <br/>
 消防员华伦火发生后第三天投入灭火，因为前两天他位于天堂镇的家也陷于火场。
 <br/>
 <br/>
 加州林业和消防局大队长麦特．华伦：〝很多屋主，接近一半的人失去了房屋，我们都遭受了严重的财产损失，但幸运的是我们留下了大部分的房屋建筑。〞
 <br/>
 <br/>
 麦特．华伦表示，他把妻子安顿好之后，就投入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_救灾.html" target="_blank">
  救灾
 </a>
 的招唤。
 <br/>
 <br/>
 华伦和其他工作人员在大火到来之前，砍伐森林，点燃较小的火焰，移走干枯的树木和植披，阻止大火蔓延。
 <br/>
 <br/>
 消防官员周一在南加州大火周边地区发布了更多疏散令，警告居民保持警戒，因为强风引发了新的火灾。
 <br/>
 <br/>
 至少2人在洛杉矶市西部爆发的火灾中丧生，370栋房屋被毁，20多万人撤离家园。
 <br/>
 <br/>
 洛杉矶消防队长奥斯比表示，消防人员在长约20英里，宽约14英里的所谓〝伍尔西火灾〞中抢救了大约5万7千所房屋。
 <br/>
 <br/>
 火灾范围从洛杉矶西北部到郊区，通过圣莫尼卡山脉，到达马里布海岸，只有10%火势获得控制。
 <br/>
 <br/>
 附近另一场规模较小的希尔火灾 烧毁了4,530英亩土地，已有75%火势获得控制。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普总统.html" target="_blank">
  川普总统
 </a>
 已宣布加州进入野火紧急状态，并批准联邦援助，协助受灾居民度过难关。
 <br/>
 <br/>
 新唐人记者李梅综合报导
</div>
