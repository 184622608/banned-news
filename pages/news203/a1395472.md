### 川普评中期选举：将有大事发生
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周一（10月15日）表示，众多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支持者.html" target="_blank">
  支持者
 </a>
 涌向他的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竞选集会.html" target="_blank">

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大事发生.html" target="_blank">
  大事发生
 </a>
 〞，暗示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 将迎来共和党胜利的〝红波〞。
 <br/>
 <br/>
 周一清晨，针对他每次出席集会都场场爆满的情况，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 发推评论：
 <br/>
 <br/>
 〝我集会上的人群比以往任何时候都大得多，包括2016年竞选。这些大型会场从没有一个座位空着，还有好几千人在会场外观看屏幕。热情和兴致穿透屋顶。将有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大事发生.html" target="_blank">
  大事发生
 </a>
 ——看吧！〞
 <br/>
 <br/>
 目前离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 到来还有3周时间，美国两党正就国会控制权问题加紧竞争。
 <br/>
 <br/>
 川普在中期选举前积极出席各地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竞选集会.html" target="_blank">
  竞选集会
 </a>
 ，全力支持共和党人竞选。上周，他先后赴爱荷华、宾夕法尼亚、俄亥俄和肯塔基州出席集会，而每场集会的会场都被数千名热情的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支持者.html" target="_blank">
  支持者
 </a>
 挤得水洩不通。
 <br/>
 <br/>
 据保守派博客Gateway pundit报导，上周六，有人为能亲眼看到川普在肯塔基的演讲，居然在华氏温度46°的低温天气中排队等候了6个小时。

 <br/>
 各地民众高度支持的热情让川普相信，他会证明权威人士和民意测验所宣扬的〝蓝波〞其实是错误的。自由派一直在制造声势，称民主党将在中期选举后控制众议院，甚至还可能控制参议院。
 <br/>
 <br/>
 美国中期选举将从11月6日开始举行，届时，不仅众议院所有435名众议员都要改选，参议院的33个席位及36个州的州长职位也要重新洗牌。
 <br/>
 <br/>
 总部设于华盛顿的独立民调和智库机构皮尤研究中心（Pew Research Center）最近调查发现，美国选民目前的投票热情是20多年来历次中期选举中最高的。近3/4的登记选民表示，他们很关心共和党能否保住在参众两院的优势。
 <br/>
 <br/>
 皮尤调查还发现，近60％的登记选民认为，对川普的看法会影响他们的投票。
