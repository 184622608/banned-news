### 共和党抓紧调查执法偏见 众院传讯前司法部长和FBI局长
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】周三（11月21日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 司法委员会发出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传票.html" target="_blank">
  传票
 </a>
 ，要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前司法部长.html" target="_blank">
  前司法部长
 </a>
 林奇（Loretta Lynch）和前联邦调查局（FBI）局长科米（James Comey）就2016年美国大选问题到国会作证。
 <br/>
 <br/>
 周三晚，弗吉尼亚州共和党议员、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 司法委员会主席谷德拉特（Bob Goodlatte）传讯林奇和科米，要求他们赴国会听证，回答司法委员会所质疑的关于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_2016年大选.html" target="_blank">
  2016年大选
 </a>
 的一些问题。
 <br/>
 <br/>
 由于在中期选举中赢得众议院，预计明年1月重掌大权的众议院民主党将终止共和党目前正在进行的一项重要调查，即在所谓川普选战勾结俄罗斯干扰2016年美国大选的通俄门调查中，司法部和FBI是否持有政治偏见并影响了调查的公正性。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传票.html" target="_blank">
  传票
 </a>
 要求，科米和林奇须分别于12月3日和4日到众议院司法委员会接受闭门听证。
 <br/>
 <br/>
 科米周四在推特上承认他已收到传票，但他表示拒绝出席，理由是他〝抵制〞闭门听证。
 <br/>
 <br/>
 科米不久前已拒绝了一次国会的传讯。
 <br/>
 <br/>
 根据国会的调查，科米在2016年大选前两周出于压力重启电邮门调查，但实际上，他在调查结束前两个多月就已为民主党总统候选人、前国务卿希拉里（Hillary Clinton）起草好免责声明，并在声明中刻意修改字眼，把希拉里使用私人服务器收发政府邮件一事从轻定性，把原来的〝严重疏忽〞（grossly negligent）换成〝非常粗心〞（extremely careless），致使希拉里免遭指控。
 <br/>
 <br/>
 2016年，时任司法部长的林奇则因与前总统克林顿（Bill Clinton）进行的一次臭名昭着的停机坪会晤，而被质疑与克林顿夫妇达成了秘密交易。据说她因为袒护希拉里，还曾威胁报复与希拉里有关的美俄铀交易案件中的线人。
 <br/>
 <br/>
 （记者李佳欣综合报导）
