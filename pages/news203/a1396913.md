### 白宫邀普京访华府 二次〝双普会〞有望
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】周五，美国国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 表示，白宫邀请俄罗斯总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 访问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华府.html" target="_blank">
  华府
 </a>
 。之前，普京邀请美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 11月11日在巴黎会面。如果成行，将是7月16日芬兰会面后，双方的第二次峰会。
 <br/>
 <br/>
 周五，美国国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 访问格鲁吉亚时表示，白宫邀请
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 明年访问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华府.html" target="_blank">
  华府
 </a>
 。具体时间和议程尚未确定。
 <br/>
 <br/>
 美国国家安全顾问 博尔顿：〝我们邀请普京总统明年访问华盛顿，进行一天的磋商。〞
 <br/>
 <br/>
 博尔顿还表示，他之前访问俄罗斯时，和几位俄罗斯内阁成员讨论了俄罗斯入侵格鲁吉亚的问题。
 <br/>
 <br/>
 2008年，格鲁吉亚和俄罗斯为争夺南奥塞梯的控制权爆发战争。之后，属于格鲁吉亚的20%的领土在俄罗斯的支持下宣布独立。
 <br/>
 <br/>
 博尔顿提到，针对该事件，美国已经对俄罗斯进行了制裁，这些制裁措施一直有效。
 <br/>
 <br/>
 他还说，美国正在考虑是否针对前俄罗斯间谍斯克里帕尔和他的女儿在英国被俄罗斯特工下毒一事，对俄罗斯实施新的制裁。
 <br/>
 <br/>
 今年7月，川普和普京在芬兰首都赫尔辛基进行首次会谈。
 <br/>
 <br/>
 对于美国此次的访问邀请，俄罗斯尚未给出答覆。
 <br/>
 <br/>
 如果成行，川普将成为布什之后第一位在白宫接见俄罗斯总统的美国总统。
 <br/>
 <br/>
 新唐人记者王羽音、宇微综合报导
</div>
