### 美国副总统彭斯在哈德逊研究所演讲视频（中文字幕）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 10月4日星期四在华盛顿智库
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈德逊研究所.html" target="_blank">
  哈德逊研究所
 </a>
 就美国政府的中国政策发表长篇演说。他严厉抨击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 试图破坏美国的民主制度，引起国际社会极大关注。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 的讲话涉及话题广泛，包括美中贸易争端、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在南中国海的军事化、对台湾的打压，还有中共对国内民众的监控和压制以及在美国和海外施加影响力等等。他在讲话中引用了大量情报评估和公开信息，并谈到了许多具体例证。
 <br/>
 <br/>
 彭斯同时毫不客气的批评前几届美国政府，称他们忽视了中共的行动甚至〝助长〞了中共。但是他说：〝这样的日子结束了。〞
 <br/>
 <br/>
 彭斯这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演讲.html" target="_blank">
  演讲
 </a>
 引发国际社会极大反响，舆论界普遍将其视为美国对中共〝新冷战〞的一篇檄文。本台特地将彭斯演讲视频制作中文字幕，以飨读者。
