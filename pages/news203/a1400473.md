### 专家：习近平暗示三种战争 中美关系阴云密布
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】日前在亚太经济合作组织（APEC）峰会上，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 激烈交锋。中美分歧甚至导致APEC首脑宣言流产。有政治评论家说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系.html" target="_blank">
  中美关系
 </a>
 发展的前景阴云密布。习在峰会上已暗示，中美之间可能爆发〝热战、冷战、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 〞这三种战争。
 <br/>
 <br/>
 澳大利亚媒体《澳大利亚人报》评论称，由于美中之间的紧张局势加剧，在巴布亚新几内亚召开的亚太经合会议（APEC）峰会，18日在混乱中结束，而双方〝戏剧性的〞分歧，导致该峰会近29年来首次没能发表联合公报。
 <br/>
 <br/>
 文章称，在美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）指责中共，企图〝将我们的伙伴淹没在债务海洋〞后，中美两国未能在联合公报中，就三个有争议的部分做出让步，此举导致各国领导人在峰会结束后未能达成共识。
 <br/>
 <br/>
 彭斯和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 17日在峰会上激烈交锋。习近平谈了中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 和北京所倡议的〝一带一路〞等。彭斯则点名批评中共的贸易政策，及中共的〝一带一路〞是给合作国家设立的债务陷阱。
 <br/>
 <br/>
 旅美政治评论家陈破空对法广分析说，此次各国领导人首次未能达成一致、发表共同宣言，令峰会成果黯然失色。不仅因美中分歧，其他各个成员国都与中共有分歧，在围堵中共方面，西方国家已达成了共识。
 <br/>
 <br/>
 陈破空说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系.html" target="_blank">
  中美关系
 </a>
 发展的前景，可以说是阴云密布。对于未来数年，中美两国关系的发展走向，他分析说，看点在于11月底在阿根廷举行的中美两国元首会谈的情况。
 <br/>
 <br/>
 一种乐观的说法认为：可以在贸易战上实现停火协议，如果中方可以做出重大让步的话；中间的说法是：可能举行一个会谈，有其他的议题进行严肃的谈判；不看好的观点认为：即便达成协议，中共也未必去执行和遵守。
 <br/>
 <br/>
 因此，在经济上也可能达成一定的妥协，但是中共可能会继续使用这种不遵守或者拖延战等方式来敷衍美国。
 <br/>
 <br/>
 陈破空说，在政治上，中美双方完全没有对话的余地。中共会继续坚持他的一党专政，美国则会对中共的一党专政保持高度地警惕。因为外交是内政的延伸，这种模式对世界构成的威胁，在历次大战或者冷战中已经显现。
 <br/>
 <br/>
 他举例说，像当年纳粹德国对世界的威胁，以及前苏联对世界的威胁。现在共产党用一党专政的方式控制13亿人，恐怕这是中美之间无法解脱的死结。
 <br/>
 <br/>
 尽管中共高层，如习近平和王岐山，请基辛格或者保尔森这些人到中国去调解中美关系，但是他们的说法也都发生了改变。
 <br/>
 <br/>
 基辛格对中共的告诫是：中美关系不可能回到从前。中共必须脱离旧制度。保尔森对中共的告诫是：必须遵守世界贸易组织的规则，也就是必须履行入世承诺。
 <br/>
 <br/>
 中共是否能做到这些？陈破空表示，多数人看来，中共做不到、也不愿意做。因此在这样的情况下，中美对立、对抗、甚至于敌对的大趋势已形成。所以习近平在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 上讲〝热战、冷战、贸易战没有赢家〞，已经在暗示中美之间，这三种战争都可能爆发。
 <br/>
 <br/>
 关于亚太经合组织峰会上，中美两国激烈交锋方面，陈破空说，从彭斯和习近平分别在会议上的演讲可以看出来。彭斯主要还是继续不点名地谴责、批评中共破坏世界贸易经济体系，包括大规模地窃取知识产权，强迫技术转让，单向的由政府来补贴工业。人为地制造贸易逆差，忽视劳工权益等等。
 <br/>
 <br/>
 而且彭斯还直接批评说，极权政治在亚太地区应该是没有立足之地。或者说：极权政治、有些国家对其他国家输出极权政治，对区域安全和稳定的危害，就直指中共。
 <br/>
 <br/>
 同时彭斯也批评了中国的一带一路，说这是一个〝束缚他国的袋子〞，而且是一个〝债务海洋〞。因此，美国和国际社会认为：〝一带一路〞这样不公开、不透明的为他国单向服务是对世界经济的损害。
 <br/>
 <br/>
 而习近平的演讲也不点名地批评美国，说美国要把自己的模式强加于人，或者说是美国的做法是对自由经济的伤害等等。陈破空说，实际上，这种说法用在美国身上并不成立，用在中共身上倒是更为合适。特别是说〝大国霸道〞、或者〝不讲道理〞等等，对中共来说更为合适。
 <br/>
 <br/>
 陈破空说，习近平发言时所讲的话，主要是讲给中国国内人听的。习在APEC峰会上继续地兜售一带一路，但是一带一路在中国国内也不受欢迎。因为中国老百姓认为这是一个〝大撒币〞的工程，如果这些钱用在中国，〝全面小康〞早就实现了。但似乎中共官方并不愿意把这个钱用在国内。如果中共不改变的话，〝一带一路〞会走向一条死胡同。
 <br/>
 <br/>
 在西太平洋或者说亚太地区，中共不仅在经济上挑衅美国，试图阻止本地区的经济事务，中共还试图在军事上、在地缘政治上胁迫他国。但是中共的这种价值体系，一党专政的模式，在国际上不得人心，走不出国门。因此中共用金钱、间谍等各方面的手段对周边国家的选举进行干涉。
 <br/>
 <br/>
 在这样的情况下，美国越来越提出了中共模式问题。彭斯13日在新加坡东盟峰会上提出批评〝对外扩张和侵略战争在亚太地区没有立足之地〞。
 <br/>
 <br/>
 APEC峰会上，彭斯进一步抨击：极权政治在亚太地区没有立足之地、没有存活的机会。直指北京的政治制度，这恰恰是北京最敏感的地方，也是它的底线。
 <br/>
 <br/>
 因此北京对此作出了毫不掩饰的回应，他们在中美贸易谈判中声称：中共一党专政的道路不能动摇。
 <br/>
 <br/>
 陈破空认为，这说明中共为了维护它的专制政权，会继续推行对内镇压、对外威胁的政策。因此亚太地区必然会成为美中博弈的大战场，而且这个战场已延伸到、从印度洋到太平洋的广阔地区。因此形成了美国、日本、印度和澳大利亚的另一层防线。
 <br/>
 <br/>
 这条防线也可以倒过来证明，中共力图突破第一岛链、第二岛链的封锁，美国和国际社会在加强对中共的这种红色怪兽的封锁。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
