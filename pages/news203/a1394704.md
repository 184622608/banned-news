### 英媒：美要中方G20前开〝让步清单〞 否则川习无会
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 近期陷入僵局，外界都在猜测中美首脑在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上的会面或许能给贸易谈判带来转机。不过，日前有英国媒体引述知情人士透露，美国官员已警告中共政府，如果中方不能首先提出详细的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_让步清单.html" target="_blank">
  让步清单
 </a>
 〞，美国总统川普（特朗普）将不会在下月30日举行的G20峰会上与习近平开展贸易谈判。
 <br/>
 <br/>
 英国《金融时报》日前报导称，有两名为中美两国谈判简报的消息人士透露，美国官员已经明确警告中共政府，如果北京当局没有提供详细
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_让步清单.html" target="_blank">
  让步清单
 </a>
 ，川普将不会在下个月出席20国集团（G20）峰会时与习近平举行贸易会谈。
 <br/>
 <br/>
 两位消息人士表示，中方声称他们早已经准备好了一份清单，但如果不能确保华府在〝稳定的政治情势〞下获得接受，或者没有一个真正可以代表川普政府谈判的负责人，中方就不会交出这份清单。
 <br/>
 <br/>
 报导表示，现在美国方面对北京当局始终不愿讨论与经济和贸易政策相关的实质〝结构性问题〞十分不满，认为这是导致中美贸易冲突无法取得突破的原因所在；而北京方面则指责美国之前派出的官员无法谈判具拘束力协议。
 <br/>
 <br/>
 有媒体在相关报导中称，自从美国副总统彭斯发表了强烈批评中共政府的演讲后，中方开始猜测，或许彭斯将会是下一次带头与中方进行贸易谈判的美方负责人。
 <br/>
 <br/>
 今年10月1日，白宫首席经济顾问库德洛（Larry Kudlow）接受FOX Business访问时曾表示，中美之间的贸易谈判近期没有太大进展，川普总统对此〝不满意〞，但美国政府也并不急于与中国（中共政府）达成贸易协议。
 <br/>
 <br/>
 库德洛表示，目前美中谈判的重点问题，是中共方面窃取美国企业的知识产权以及强制美国企业的技术转移的问题，而关键问题在于企业所有权，美国政府认为，美国公司不应必须通过创办让中资企业持有更大份额的合资企业才能在中国运营，而应该让在中国运营的美国企业可以拥有自己公司的全部所有权。
 <br/>
 <br/>
 当时库德洛还指出，美国与加拿大达成了贸易协议，向中方传递了一个重要的讯息，那就是北美在贸易议题上是站在〝同一阵线〞的。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 正式爆发前，美国方面曾于今年5月初第一轮贸易谈判时，向中共国务院副总理刘鹤提出了一份多达140项具体要求的清单，从要求中共政府撤销市场准入壁垒到向美国长期采购能源、农产品等方方面面。
 <br/>
 <br/>
 而中方在今年8月份曾表示，可以接受上述清单中大约1/3的要求，也愿意讨论清单中另外1/3的要求，而剩余的条款则是中方禁区，不能讨论，其中包括美国要求中共政府开放国内云端运算市场给外国企业等内容。
 <br/>
 <br/>
 中共官员称，他们原本准备于9月份再与美方进行第5轮贸易谈判，但于美国政府突然宣布对超过半数的中国输美产品征收关税后，这一轮贸易谈判被取消。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：明轩）
