### 律师解读 川普或颁布行政令 取消在美出生公民权
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】周一，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 接受HBO电视频道采访时表示，将发布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政令.html" target="_blank">
  行政令
 </a>
 ，不再允许在美国出生的婴儿自动成为美国公民。来听听律师的详细解读。
 <br/>
 <br/>
 美国移民律师 张钧：〝签署这项行政命令，主要目的就是为了限制几类人，一类人就是父母双方，连美国的永久居民都不是，既不是公民，也不是绿卡。到美国来旅游的，甚至包括学生，在这里只是短期的身分，在这里生了孩子，孩子就变成了美国公民。21岁或者21岁以后，反过来又可以申请自己的父母；申请自己的兄弟姐妹。〞
 <br/>
 <br/>
 根据美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宪法.html" target="_blank">
  宪法
 </a>
 第14条修正案，〝出生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公民权.html" target="_blank">
  公民权
 </a>
 〞是指在美国出生、并受美国司法管辖的人，将自动成为美国公民。
 <br/>
 <br/>
 要取消〝出生公民权〞，需要通过修改宪法。
 <br/>
 <br/>
 美国移民律师 张钧：〝不管是谁想要改变这个宪法，或者是宪法修正案，就需要走一个非常复杂的一个修改宪法的程序。首先只少要有三分之二的众议院和参议院两院的同时提名，通过这个提案之后，到各州去表决，要四分之三以上的州同意了，这个修正案的增加或者删减内容才能成为宪法的一部分。〞
 <br/>
 <br/>
 不过，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 表示，无需修改宪法，颁布行政命令即可实行。
 <br/>
 <br/>
 如果川普的这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政令.html" target="_blank">
  行政令
 </a>
 能够落实，将是美国移民政策的巨大改革，影响重大。
 <br/>
 <br/>
 新唐人记者柯婷婷、韩瑞纽约报导
</div>
