### 韩媒预测贸易战3大结局 中南海最惊惧第3种
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】中美贸易战持续升级且已蔓延到其它领域，有韩媒预测接下来可能出现的三种结局，认为中美贸易战全面扩大后，将导致中国资本市场被制裁，这是中南海最担心的一种结局。
 <br/>
 <br/>
 尽管
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 打得如火如荼，但中共始终不愿放弃长年的不公平贸易行为，不愿开放市场自由竞争，因而采取拖延战术，寄希望于美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，看选举结果能否削弱川普政府实力再来行动。
 <br/>
 <br/>
 美国副总统彭斯4日发表了川普政府对中共政策的演讲，严厉抨击中共对内压迫中国人民，对外渗透操纵全世界的不端行径，而且还点名中共干预美国的国内政策和中期选举的恶劣行为。
 <br/>
 <br/>
 彭斯表示，川普政府不会退缩，美国将对中共的行动展开决定性的回击。
 <br/>
 <br/>
 10月9日，韩国《中央日报》报导，美国总统川普（特朗普）对华发动的贸易战，中共处于〝守〞势，然而内伤不断的中共，已然下定长期抗战的决心，正在忙着准备对策。
 <br/>
 <br/>
 报导认为，中美贸易战很可能会导致以下三种结局。
 <br/>
 <br/>
 <h4>
  一是，中共在关税战中做出妥协。
 </h4>
 <br/>
 美中贸易战从7月6日正式开打后，川普不断强势加码，除了已有500亿美元商品征收25%关税外，9月24日起又追加2000亿美元商品，征收10%关税，自明年1月1日起，这2000亿美元商品的关税将升至25%。
 <br/>
 <br/>
 川普政府更表明，一旦中共针对美国农民或产业进行报复，美国将再祭出2670亿美元的商品清单，加征高关税，等于中国所有出口美国商品全部覆盖在关税炮火之下。
 <br/>
 <br/>
 对于美方继续加征2000亿关税，中共只征收了600亿美元的关税，韩媒认为，这体现了中共寻找妥协的一面，如果中共同意美国要求的减少贸易赤字的话，损失也不小。
 <br/>
 <br/>
 <h4>
  二是，中共对全部美国进口的产品征收关税。
 </h4>
 <br/>
 <br/>
 这意味着将引发美国向中共继续征收2,670亿美元的关税，即中美贸易战会继续到底。 这样一来，投资受限、人员交流等的限制措施也会跟着出台。如果关税战没起到作用，还有可能导致特定商品的紧急进口限制等政策出台。
 <br/>
 <br/>
 届时中国电子产品及日用品、建筑材料等主要出口产品都会受限，导致制造业供给发生重大变化。外国投资企业也会纷纷退出中国，人民币贬值，资金会外流。同时，中国国内依靠出口的企业都会陷入破产危机，最少会有500万农民工失业。
 <br/>
 <br/>
 <h4>
  第三种应是中共最担心的一种结果。
 </h4>
 <br/>
 <br/>
 那就是中美冲突不仅局限于贸易领域，还会不断延伸到中国金融、资本市场的制裁和控制，甚至到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率战.html" target="_blank">
  汇率战
 </a>
 。这样很有可能会导致出现资产冻结，以及特定商品的世界贸易被禁等状况的出现。
 <br/>
 <br/>
 原因是美国拥有着可掌控贸易支付和兑税交易的两大金融系统——全球金融控制系统和金融支付结算系统。美国财政部可通过海外资产控制办公室进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融制裁.html" target="_blank">
  金融制裁
 </a>
 。
 <br/>
 <br/>
 如果美国以国家安全为由将中共列入金融制裁范围，对其资本市场进行控制的话，中共的对外经济会陷入困境。
 <br/>
 <br/>
 对于美方的贸易制裁，尽管中共一开始便高调宣称〝奉陪到底〞，但贸易战一开打，中国股市萎靡不振、人民币大贬、物价上涨、资本外逃，许多外企、台企甚至中国本地民企也相继撤离中国。中共担忧的〝规模性失业〞、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_债务危机.html" target="_blank">
  债务危机
 </a>
 目前已在各地陆续涌现。
 <br/>
 <br/>
 川普追击不舍的贸易战术，被认为是川普战术里目前最具威力的重拳，而金融战则是川普紧接着贸易战之后的下一记重拳。
 <br/>
 <br/>
 9月20日，川普签署行政命令，授权国务院、财政部联合执行《反击美国对手制裁法案》，未来可对总统下令制裁的个人或对象进行权限范围内的最大制裁，包括：禁止金融交易与转账、冻结在美资产，及进一步对受制裁者的主管、官员进行惩罚等。
 <br/>
 <br/>
 金融制裁范围广泛，小至冻结个人或企业资产，大到禁止外国个人与实体与中国交易往来，甚至取消中共通过美元进行外汇交易结算等，其影响力与伤害力将难以估算，分析认为这对中共政权与权贵集团将更有〝切肤之痛〞。
 <br/>
 <br/>
 有观点认为，若美国全方位对抗中共，必将诱发大规模民变，中共政权将就此终结。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：祝馨睿）
