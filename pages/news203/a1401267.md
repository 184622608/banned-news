### FBI全球通缉令 美籍华裔女撞死人逃至澳洲被捕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】拥有美国国籍的陈姓华裔女商人，去年1月涉嫌在加州撞死一名骑单车的男子后逃逸，5日后，经香港潜逃至澳洲，随后被美国联邦调查局（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 ）发布全球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通缉令.html" target="_blank">
  通缉令
 </a>
 ，今年4月在澳洲阿德莱德遭拘捕，案件将于下月审理，陈女或被引渡至美国受审。
 <br/>
 <br/>
 据东网报导，事发于去年1月30日上午7时许，33岁陈姓华裔女商人（Andrea Dorothy Chan Reyes）驾驶一辆白色私家车，逆向撞倒46岁罗德里格斯（Agustin Rodriguez Junior）后，一直拖行罗德里格斯约270米，再加速逃离现场。
 <br/>
 <br/>
 事后，罗德里格斯证实死亡，而陈女则将私家车驶到一间修理厂，谎称〝撞了一头鹿〞，更换了挡风玻璃。陈女也在事发后5日潜逃至香港。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 寻线展开调查，找到陈女的私家车，并在车上发现与死者相符的DNA。随即于去年9月发布全球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通缉令.html" target="_blank">
  通缉令
 </a>
 。
 <br/>
 <br/>
 至今年4月，陈女在阿德莱德北部一处民宅被捕，当时她正与新相识的男友同居。
 <br/>
 <br/>
 根据法庭文件显示，陈女在菲律宾出生，曾就读于香港汉基国际学校，于2009年取得南加州大学马歇尔商学院工商管理硕士学位，并拥有美国国籍。目前，陈女被还押于阿德莱德女子监狱。
 <br/>
 <br/>
 (责任编辑：夏明义)
