### 川普政策转变 扩大外援 遏制中共影响力
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】川普总统近日签署法律，扩大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 对外援助力度，为亚非拉国家提供透明互惠的基础建设投资选择。此举被视为，正面应对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 计划。
 <br/>
 <br/>
 川普总统日前签署法律，决定整合多个政府机构，成立新的外援机构〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 国际发展金融公司〞。
 <br/>
 <br/>
 新机构将获得600亿美元资金，负责向发展中国家的能源、港口、供水等基础设施建设项目提供贷款，并为愿意到发展中国家开拓业务的企业提供保险。
 <br/>
 <br/>
 发起相关法案的国会众议员泰德・约霍表示，这是对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 的反击〞。
 <br/>
 <br/>
 〝中共正在有效利用其金融发展机构，在发展中国家扩大自己的影响力，甚至影响到了西半球的国家，例如海地。〞
 <br/>
 <br/>
 中共的一带一路计划，5年来，向沿线国家投资超过700亿美元。
 <br/>
 <br/>
 美国智库〝新美国安全中心〞的报告指出，中共借一带一路名义，修建军民两用设施，提升中共军队的全球行动能力；
 <br/>
 <br/>
 同时，中共向一些国家提供远超其支付能力的基础建设贷款，一旦该国无力偿还，中共就获得对该国的战略影响力。例如斯里兰卡因无力还贷，被迫将一个战略港口，租给中共长达99年。
 <br/>
 <br/>
 报告还警告，中共在多个国家的投资进程不透明，孳生腐败。并支持一带一路沿线的威权国家打压人权。还计画将监控技术向欧亚非多国输出。使威权主义在全球上升。
 <br/>
 <br/>
 杨建利：越来越多的国家已经感受到中共的负面影响，已经给很多国家造成了安全的隐患，也给很多正在发展中的不民主国家带来一种殖民的政策，也改变了很多民主国家的生活方式。
 <br/>
 <br/>
 白宫官员强调，美国的海外投资重在透明、可持续、创造当地就业、带动私人投资，使他国经济进入良性循环。和〝一带一路〞形成鲜明对比。
 <br/>
 <br/>
 我们将给其他国家一个公正、透明的选择，以取代中共的〝债务陷阱外交〞。
 <br/>
 <br/>
 陈破空：一个是价值的对比，民主价值和专制价值的对比；第二个，（美国）市场经济，跟中共权力经济的对比；我相信假以时日，印太国家会作出自己的正确选择。
 <br/>
 <br/>
 川普从竞选以来，一直主张削减对外援助。美国新的外援机构成立，标志着川普在外援政策上的重大转变。
 <br/>
 <br/>
 杨建利：美国在全球范围内反制中共的影响，已经全面开始。而且会逐步的深入和扩展下去。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
