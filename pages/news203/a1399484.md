### 中共对美书面回应细节曝光 仅限经贸领域部分妥协
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】路透社15日报导，中方已经就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易谈判.html" target="_blank">
  中美贸易谈判
 </a>
 向美方书面提交了让步方案，但离美方要求相去甚远。当天晚些时候，自由亚洲电台曝光了北京方案的更多具体内容，但多是重复此前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降低关税.html" target="_blank">
  降低关税
 </a>
 、部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开放市场.html" target="_blank">
  开放市场
 </a>
 等承诺，不涉及南海问题。
 <br/>
 <br/>
 自由亚洲引述知情人士消息报导，中方回应的主要内容包括，扩大降低美国货品关税的范围，特别是消费品和汽车关税，〝争取〞接近美国进口关税的水平。
 <br/>
 <br/>
 另外，中方还承诺对在华美企提供〝与国内企业同等待遇〞，加快审批美资项目，加快开放金融市场。
 <br/>
 <br/>
 美国CNN同日引述消息指，中方书面回应中并未包含白宫反覆提出的几项必含内容，包括强制技术转让和盗窃知识产权等。这份方案和北京此前公开发表的声明几乎没有差别，不过是用新瓶装旧酒，并稍微修改了一下酒的配方。
 <br/>
 <br/>
 稍早前，彭博社透露，中方在书面回应中，还承诺放宽某些行业的外资股权上限，不过文件多是老调重弹，缺乏新内容。
 <br/>
 <br/>
 关于方案中所谓对美〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开放市场.html" target="_blank">
  开放市场
 </a>
 〞，中共早在此前历次谈判中已有承诺，而且在金融领域已经就降低外资持股比例出台了有限的具体措施。但随着中美谈判搁浅，这些已公布的措施又成了一纸空文。
 <br/>
 <br/>
 但在有关经济体制的结构性改革，特别是〝中国制造2025〞等关键问题上，中方从无让步的迹象。
 <br/>
 <br/>
 路透社引述一名华盛顿消息人士说，中方回应离达成协议还很遥远，因为双方〝完全不在同一个世界〞（not in the same universe）。
 <br/>
 <br/>
 日前，美方副总统彭斯警告，中方如果不能在保护知识产权、强迫技术转移、限制外企进入中国市场、尊重国际规则及在国际水域航行自由等问题上做出显著的、实质的让步，美方准备在经济、外交和政治上对北京施加更大压力，新冷战就将开启。
 <br/>
 <br/>
 除了重申对贸易公平的具体要求外，彭斯喊话中还将南海问题（即〝国际水域航行自由〞）列入了达成贸易协议的条件清单。
 <br/>
 <br/>
 11月9日的中美第二轮外交安全对话之后，美国务院已发表声明，要求中方撤除部署在南沙群岛的导弹。
 <br/>
 <br/>
 中方提交的书面方案中，并未就此做出回应。
 <br/>
 <br/>
 自由亚洲引述分析称，这份书面回应或只是一个技术性妥协。因为不但国际社会怀疑中方做出实质性让步的诚意，连中共体制内也不看好。
 <br/>
 <br/>
 分析指，在中共的政治体制下，对美妥协存在巨大的政治风险。未来中美贸易战将伴随着中共内部的权力斗争继续升级。
 <br/>
 <br/>
 另有分析认为，中美对抗表现是全方位的。而在贸易以外的国际政治领域，包括人权问题、南海问题、台湾问题、地缘政治问题上，中共不会做出退让。
 <br/>
 <br/>
 日前，中共高阶智囊、中国人民大学的美国研究中心主任时殷弘表示，北京当前应对美国做出〝空前让步〞，重拾〝韬光养晦〞，以争取五六年时间〝调整经济〞，然后再卷土重来。
 <br/>
 <br/>
 港媒指，这可能是北京当前策略的真实写照。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
