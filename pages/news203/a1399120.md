### G20川习会前北京禁令曝光：避免刺激美国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】G20
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 前，中美高层恢复接触，就贸易等问题进行对话为川习会铺路。中共海外党媒爆料，为避免刺激美国，北京近日下达数条指令，包括禁止对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）搞〝人身攻击式〞；贸易战必须等商务部回应后才能报导等。
 <br/>
 <br/>
 美国中期选举结束后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）7日在白宫举行的新闻会上说，〝中国制造2025〞非常具有威胁性。中方已经放弃〝中国制造2025〞。
 <br/>
 <br/>
 彭博新闻分析称，〝中国制造2025〞是阻碍解决美中贸易紧张局势的绊脚石之一，该计划改变了与美国的经济关系。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 持续升温，美国封杀诸如中兴通讯、福建晋华等中国科技企业，被认为是剑指〝中国制造2025〞的战略，正所谓〝打蛇打七寸〞。
 <br/>
 <br/>
 中共官媒9日发表社评称，自中美贸易战全面开打后，〝中国制造2025〞这个词语，已经从中共官方文件和媒体上逐渐消失了。这首先是中共不希望刺激美国的表现。
 <br/>
 <br/>
 中共海外党媒引述知情人的话称，事实上，中共早在贸易战开打之后，就不再大张旗鼓地宣传〝中国制造2025〞。
 <br/>
 <br/>
 报导说，另外〝为避免刺激美国〞，在中共的宣传系统内，有数条指令已经下达，包括禁止对川普搞〝人身攻击式〞报导；贸易战等必须等到中共商务部、外交部回应后才能报导。
 <br/>
 <br/>
 观察家普遍认为，贸易战的确给中国当头一棒，清醒之后，北京方面低调处理〝中国制造2025〞是一种〝韬光养晦〞的战略体现。
 <br/>
 <br/>
 不过，时政评论员夏小强分析说，对于中共来讲，实行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_2025计划.html" target="_blank">
  2025计划
 </a>
 旨在通过盗窃手段获得技术，来超越取代美国，维持中共统治而已。让中共放弃2025计划，等于放弃了中共掌控中国经济发展的工具，也等于削弱中共政权的稳定和统治，因此它不可能从根本上放弃2025计划。
 <br/>
 <br/>
 另有评论人士认为，中共口头答应川普放弃2025计划，只是战略，是在美国压力下，不再大力宣传，转入地下低调实施。包括流传出的中宣系统一些禁令，实际目的是〝避免刺激美国〞，尽量在G20
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 期间取得一些成果，缓解贸易战的压力。
 <br/>
 <br/>
 《华尔街日报》12日报导称，在中美两国领导人定于11月底G20峰会期间会晤之际，美国财政部长努钦（Steven Mnuchin），已经恢复与中方贸易谈判负责人、副总理刘鹤的对话，讨论一个将缓解双方贸易紧张关系的协议。
 <br/>
 <br/>
 一些对中共持强硬立场的美国官员说，在美中两国领导人会晤之前，中方将拿出一项提议。他们认为，美中两国最多可能会就贸易争端达成某种休战协议，美国会暂停对中国产品加征关税。但即便是有限的停火协议可能也很难达成。
 <br/>
 <br/>
 他们还披露称，中国国家主席习近平已要求各部委提出对谈判的具体期望，以便双方能够在G20峰会之后继续谈判。
 <br/>
 <br/>
 中共国家副主席王歧山日前在新加坡出席一次国际会议时也表示，中方准备好寻求双方都能接受的贸易解决方案。
 <br/>
 <br/>
 7日川普在白宫举行的新闻会上说，中国经济已经大大下滑，〝我不希望他们下滑。我们（川普和习近平）将举行一次很好的会议，我们将会看看我们能做些什么。〞〝我们将会尝试与中方达成协议，因为我希望和习主席以及和中国都有良好的关系。〞
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
