### 代司法部长或限制通俄调查 川普称不介入
------------------------

<div class="wysiwyg">
 【新唐人2018年11月18日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周日接受福克斯新闻采访。他表示，如果司法部代部长惠特克，削减通俄门调查经费，他将不会介入。川普还提到，将继续与沙特维持友好的国际关系。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在福克斯新闻的访谈中，提到惠特克（Matthew Whitaker）曾批评穆勒的〝通俄门〞调查范围太大，并提出要缩减
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒团队.html" target="_blank">
  穆勒团队
 </a>
 的经费。但川普表示不会介入惠特克的决定。
 <br/>
 <br/>
 美国总统 川普：〝这是他（惠特克）的决定。我认为他很掌握政治，也在这方面很精明，他是一个聪明人，一个受敬重的人。他会做出对的事情。〞
 <br/>
 <br/>
 美国总统川普一直批评〝通俄门〞调查为一场政治陷害，称自己从未与俄罗斯政府存在共谋2016大选的问题。中期选举后，回避
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄调查.html" target="_blank">
  通俄调查
 </a>
 的前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部长.html" target="_blank">
  司法部长
 </a>
 塞申斯（Jeff Sessions）辞职，对通俄调查持批评态度的惠特克（Matthew Whitaker）出任了代理司法部长。
 <br/>
 <br/>
 此外，川普还提到了沙特记者卡舒吉（Jamal Khashoggi）的谋杀案，他批评整起事件〝暴力、恶毒且可怕〞。上周，川普政府并对17名涉嫌此案的沙特人提出制裁。但川普表示，愿意与沙特继续维持良好关系。
 <br/>
 <br/>
 美国总统 川普：〝有人真的知道事情真相吗？我们发布了严厉的制裁，大范围制裁了很多沙特人。不过与此同时，我们还是同盟国，而我们将在这一点上维持，两国之间关系一直都很好。〞
 <br/>
 <br/>
 新唐人电视王愉贺综合报导
</div>
