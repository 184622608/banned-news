### 网攻全球体坛 美起诉7名俄罗斯特工
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】美国司法部周四（10月4日）公布，起诉7名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 军事情报局骇客，罪名是骇客攻击国际反兴奋剂机构。并详述莫斯科当局2016年涉嫌发动攻击，以报复全球体育界的制裁。
 <br/>
 <br/>
 美国司法部助理部长德梅斯（John Demers）周四称，今天起诉了一些在（荷兰）海牙被捕的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 特工，以及他们在莫斯科的同事。他说，起诉的这7个人，涉嫌对国际世界反禁药组织（WADA）进行骇客攻击，因为该机构谴责俄罗斯政府支持运动员使用兴奋剂，并支持禁止俄罗斯运动员参加国际体育比赛。
 <br/>
 <br/>
 中央社报导，起诉书中称，WADA在2016年7月公布的报告，说明俄罗斯如何有系统地掩盖俄罗斯运动员服用禁药行径。俄军事情报局（GRU）骇客侦测总部设于蒙特娄的WADA的弱点。
 <br/>
 <br/>
 WADA在报告中建议禁止俄罗斯队伍参加里约热内卢夏季奥运。2016年7月26日，国际奥林匹克委员会（International Olympic Committee, IOC）执行委员会通知各体育协会，请他们自行对服用禁药采取行动，导致111名俄罗斯运动员遭禁止参加里约奥运。
 <br/>
 <br/>
 结果引发的是一场网路灾难，根据起诉书，一天后，俄罗斯军事情报局骇客攻击WADA网站，发动〝分散式阻断服务攻击〞（distributed denial of service）造成严重后果。
 <br/>
 <br/>
 IOC展开行动后的两周内，俄罗斯军事情报局情报员诱使WADA员工交出内部通讯与WADA电脑系统的登入帐号与密码。他们的主要策略是使用鱼叉式网路钓鱼，假冒WADA技术长的名义将电邮寄给员工，要求他们点开连结。
 <br/>
 <br/>
 起诉书说，里约奥运期间，两名俄罗斯军事情报局特工骇入IOC、WADA、CAS与USADA官员下榻饭店WiFi系统，取得一名IOC官员的安全资料库登入凭证，查看医疗和药检资料。
 <br/>
 <br/>
 在几周后的里约帕拉林匹克运动会期间，他们使用相同手法取得一名USADA官员的大型资料库登入帐号。
 <br/>
 <br/>
 全球体育机构之后遭受长达一年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网攻.html" target="_blank">
  网攻
 </a>
 ，到了2017年1月，俄罗斯军事情报局骇客已能进入CCES、FIFA与IAAF系统。
 <br/>
 <br/>
 俄国研发网路钓鱼与欺骗攻击，将恶意软体植入目标电脑，并利用公共WiFi系统窃取登入凭证与官员电子邮件等方式入侵。受害机构不只IOC和WADA，还包括国际足球总会（FIFA）、国际田径总会（IAAF）、美国反禁药组织（USADA）、加拿大运动道德中心（CCES）和体育仲裁法庭（CAS）。
 <br/>
 <br/>
 规模堪比俄罗斯对美国2016年大选发动的网攻，导致数以百计全球顶尖运动员的药物检测与药物使用资讯外洩。受波及的包括美国女子网坛姐妹花大、小威廉丝（Serena and Venus Williams）、体操好手拜尔斯（Simone Biles）与英国自由车选手弗鲁姆（Chris Froome）。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
