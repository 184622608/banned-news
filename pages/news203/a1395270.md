### 怀疑违法贩卖   美政府质询电子烟商
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】随着青少年吸
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电子烟.html" target="_blank">
  电子烟
 </a>
 的问题，越来越严重，美国食品和药物管理局，周五致信给21家电子烟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生产商.html" target="_blank">
  生产商
 </a>
 ，要求他们，提供合法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贩卖.html" target="_blank">
  贩卖
 </a>
 的相关证明。
 <br/>
 <br/>
 美国食品与药物管理局，于2016年8月禁止未经当局许可发售新的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电子烟.html" target="_blank">
  电子烟
 </a>
 产品。不过，此前有媒体披露，多家烟草公司在规定实施后，仍然研发了高尼古丁含量的新电子烟产品。
 <br/>
 <br/>
 这些产品据说模仿了在年轻人中相当流行的电子烟JUUL。尽管JUUL官方网站声明购买者需要年满21岁，但青少年偷吸的现象仍然愈演愈烈。据统计，美国高中生吸电子烟的比例，从2011年的1.5%，大幅上升到2017年的11.7%。
 <br/>
 <br/>
 在信件中，美国食品与药物管理局要求21家电子烟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生产商.html" target="_blank">
  生产商
 </a>
 出示证据，证明他们的产品在2016年规定颁布前已经在市场上贩售，否则，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贩卖.html" target="_blank">
  贩卖
 </a>
 这些产品就属于非法。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
