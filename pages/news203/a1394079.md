### 美军计划全球军演震慑中共 美媒：准备摊牌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】 中美两国军舰近日在南海险些相撞并发生对峙后，有美国媒体周三披露，美国海军正在制定一项计划，拟在全球多个地方举行军事演习以震慑中共。有美媒认为，这很可能是美国政府准备向中共当局〝摊牌〞的前兆。
 <br/>
 <br/>
 据美国有线电视新闻网（CNN）10月3日报导，有多名美国国防官员向该媒体透露，美军正在制定一个向中共展示军事实力的秘密计划。有人建议在今年11月的某一周内，同时在全球各地展开军事演习行动，以此向中共展示美军具有同时在多个前线对抗敌军的能力。
 <br/>
 <br/>
 据称，这项计划首先将由太平洋舰队在靠近中国本土的地方实施，包括让美国军舰和飞机在南中国海和台湾海峡进行〝自由航行行动〞，以展示〝美国拥有公海上自由航行的权利〞。  同时，这项拟为期一周的全球性大规模军演，还将展开一系列其他行动，行动范围将扩展到美国西海岸和南美洲，因为中国正在那里扩大投资。
 <br/>
 <br/>
 此前，美国驱逐舰迪凯特号9月30日在南沙群岛南薰礁（Gaven）、赤瓜礁（Johnson）12海里的范围内航行时，被中共海军170舰进逼，双方最近距离仅剩41公尺，两舰险些相撞，由此一度发生对峙。因此，有舆论认为，美军将让美国军舰和飞机在南中国海和台湾海峡继续〝自由航行〞纳入其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球军演.html" target="_blank">
  全球军演
 </a>
 的计划之中，应该是有针对性采取的一项报复措施。
 <br/>
 <br/>
 据称，这个全球军演的建议目前主要是美国军方推动，美国防部长马蒂斯（James Mattis）和参联会主席邓福德（Joseph Dunford）正在考虑这些行动一旦实施可能产生的外交影响。例如：如果把美军从诸如中东这些具有潜在威胁的地区撤出来参加军演，有可能让美国在这些地区缺乏军力以应对突发事件。
 <br/>
 <br/>
 这个建议还仅仅是在美军内部进行讨论的阶段，不过美国《商业内幕》分析称，美此举是准备同中国（中共）进行〝严肃的摊牌〞，CNN则认为军演的时间恰逢美国中期选举，因此这个安排完全可能具有影响美国国内政治目的。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
