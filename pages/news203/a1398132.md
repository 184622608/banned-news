### 川普乔治亚州助选 民众：不信假新闻
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】美国中期选举前最后一个周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乔治亚州.html" target="_blank">
  乔治亚州
 </a>
 梅肯市的集会上，为共和党州长候选人布莱恩・坎普
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选.html" target="_blank">
  助选
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民众.html" target="_blank">
  民众
 </a>
 表示，一些媒体报导不实消息，他们不相信那些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假新闻.html" target="_blank">
  假新闻
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 称赞共和党候选人坎普是〝伟大的领导人〞，胜选后将帮助推进政府议程。
 <br/>
 <br/>
 美国总统川普：〝这是我们一生中最重要的选举之一。这是一次非常重要的选举。我不说它和2016年那次一样重要，但真的很重要。〞
 <br/>
 <br/>
 美国10月失业率3.7%，经济成长率第三季达3.5%，成为川普辅选的利器。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乔治亚州.html" target="_blank">
  乔治亚州
 </a>
 选民认为，川普的竞选承诺说到做到，值得支持。
 <br/>
 <br/>
 乔治亚选民BRIAN ROTHMANN：〝我相信更多人会认真对待他，因为他会兑现所有的政策，信守所有的承诺。〞
 <br/>
 <br/>
 乔治亚选民BILLIE ROBINSON：〝川普是美国总统。他关心我们，关心乔治亚，关心美国的每个人。他是我们需要的人。〞
 <br/>
 <br/>
 川普总统表示，美国现在经济发展最好，呼吁选民投票给共和党。
 <br/>
 <br/>
 美国总统川普：〝这次选举将决定，我们是否能在我们建立的基础上，创造非凡的繁荣。(闪白)美国现在拥有我国历史上最好的经济，它可以很快被摧毁，这就是为什么你必须去投票。〞
 <br/>
 <br/>
 选民表示，一些媒体在妖魔化川普和共和党，他们不相信这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假新闻.html" target="_blank">
  假新闻
 </a>
 。
 <br/>
 <br/>
 乔治亚选民SHANNON VOLKODAV：〝我相信这个系统。我是一个美国人，我相信我们的政府会做正确的事情。很多媒体报导，坦白说我并不相信。〞
 <br/>
 <br/>
 乔治亚选民KEN LAMBERT：〝事实上，是共和党在兑现征收经济关税，他们也正在就移民问题进行公开讨论。〞
 <br/>
 <br/>
 新唐人记者李梅、田瑞综合报导
