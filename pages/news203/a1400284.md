### 流亡富商曝王健死亡疑点 揭海航是情报渗透平台
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】流亡美国的中国富豪
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭文贵.html" target="_blank">
  郭文贵
 </a>
 ，11月20日在纽约举行记者会，对中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海航.html" target="_blank">
  海航
 </a>
 原董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王健之死.html" target="_blank">
  王健之死
 </a>
 提出更多新疑点，并表示海航是中共国安部在美国安插的情报平台，也是洗钱工具。
 <br/>
 <br/>
 记者会上播出了有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王健之死.html" target="_blank">
  王健之死
 </a>
 的一段监控视频。视频显示，当地时间7月3日上午10时36分左右，王健和一名女翻译及三名保镳离开旅馆，前往法国博尼约（Bonnieux）的景点，10时50分左右，女翻译向旅馆餐厅老板报案，说王健坠墙。
 <br/>
 <br/>
 记者会上提出的疑点包括：
 <br/>
 <br/>
 一、女翻译向餐厅老板报案时，使用的是法文未来式〝王健‘将’坠墙〞，报案时的哭泣是因为〝恐惧〞而非〝难过〞。
 <br/>
 <br/>
 二、餐厅老板表示，当他赶赴现场时，保镳仅剩两人，另外一人不知去向。他还发现，王健附近的地面上以及一位保镳的手中都有针灸用的针头。
 <br/>
 <br/>
 三、根据餐厅老板的说法，11时30分时，王健仍然活着。下午2时，王健被送往停尸间。警方在12时09分左右赶到时，两名保镳及翻译都在旅馆。从11时30分至下午2时这段时间内，王健被如何处置情况不明。
 <br/>
 <br/>
 四、事发后中共领事馆人员全面介入调查，法国警方警告当地人士噤声，不少现场监控录像也遭到删除。
 <br/>
 <br/>
 此外，郭也表示，王健因为〝知道太多〞，所以非死不可。
 <br/>
 <br/>
 他公布了一些中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海航.html" target="_blank">
  海航
 </a>
 公司近年来迅速扩张的资料，以及资产非正常转移过程中的疑点，并指海航并非正常企业，其实归属中共安全部主管，是中共在美国甚至全世界进行间谍渗透的〝情报平台、战略平台、情报网路以及洗钱工具〞，并建议美国联邦调查局对此展开调查。
 <br/>
 <br/>
 今年7月3日，王健在法国普罗旺斯一处旅游景点意外坠墙死亡，引起外界关注。法国警方调查后表示，王健是拍照时失足落下十米高墙死亡。但多方消息显示，此事疑点重重，很多人怀疑王健是他杀。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
