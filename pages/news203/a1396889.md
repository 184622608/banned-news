### 男童邀32同学庆生被放鸽子 NBA连送大礼羡煞球迷
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】美国亚利桑那州的一位妈妈，日前邀请32名同学及家长一起为其6岁的儿子泰迪庆生，但客人们不知为何全部缺席。妈妈把泰迪一个人坐在空荡餐厅的照片放到社交媒体后，意外获
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_NBA.html" target="_blank">
  NBA
 </a>
 邀请免费观看比赛，并请7000球迷帮他庆生，羡煞球迷。
 <br/>
 <br/>
 生日当天，为款待泰迪（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Teddy.html" target="_blank">
  Teddy
 </a>
 ）的同学及家长，妈妈特意替他订了披萨。没想到，当天除了1名家长告知不能前往外，其他人全部爽约，让泰迪度过了一个〝最孤独的生日宴〞。
 <br/>
 <br/>
 泰迪的妈妈马济妮（Sil Mazzini）坦言，这让她替泰迪感到难过，好在泰迪还小，没有把事情太放在心上。
 <br/>
 <br/>
 妈妈见泰迪孤单一人坐在长桌边，心里很不是滋味，拍下照片上传网络。不料，照片曝光后，不但吸引大量网友送上温暖的生日祝福，连亚利桑那州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_NBA.html" target="_blank">
  NBA
 </a>
 太阳队也答应免费招待泰迪入场观赛，并送上特制礼物。
 <br/>
 <br/>
 太阳队邀请泰迪观看25日右超级篮球巨星詹姆士（Leborn James）率领的湖人队对上凤凰城太阳队比赛。赛前，太阳队不但让吉祥物替泰迪换上印有泰迪名字的特制球衣以及生日皇冠，让全场7000多名观众帮他庆生，更让他与NBA球星击掌并一同观看赛前热身。
 <br/>
 <br/>
 之后，NBA太阳球队在推特上发了几张小泰迪兴奋不已的照片，其中一张小泰迪身穿印有数字6（代表6岁生日）的特制球衣，笑容灿烂的站在球场上留影，羡煞不少球迷。
 <br/>

 <br/>
 <br/>
 @TJ说：〝I cried when  I saw the picture of him sitting alone heartbroken at his birthday party...I'm so happy to see that smile on his face...(当我看到他一个人孤零零的坐着的那张照片时，心都碎了。现在很高兴看到他脸上灿烂的笑容……〞
 <br/>
 <br/>
 不过，对于小泰迪的遭遇也有网友质疑，为何32名同学一起放鸽子？有网友留言说：〝是不是没沟通好？〞
 <br/>
 <br/>
 还有网友称：〝这种情况在美国比较少见。我孩子在学校曾被两个同学打过，但他们邀请我们去参加他们生日会，我们也去了，毕竟小朋友的生日宴，对他们来说是意义重大的。〞
 <br/>
 <br/>
 其实，事发原因似乎并不重要了，重要的是，小泰迪如今重新露出天真灿烂的笑容，你说呢？
