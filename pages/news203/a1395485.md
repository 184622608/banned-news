### 川普：普京有可能卷入毒杀事件 中共是更大威胁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 日前在接受美国〝哥伦比亚广播公司〞（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CBS.html" target="_blank">
  CBS
 </a>
 ）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专访.html" target="_blank">
  专访
 </a>
 时表示，他认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 有可能卷入了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒杀.html" target="_blank">
  毒杀
 </a>
 事件，并称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对美国大选的干涉更甚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
   中共
  </a>
  干涉美国大选 更甚
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
   俄罗斯
  </a>
 </h4>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 上周日（14日）在接受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CBS.html" target="_blank">
  CBS
 </a>
 新闻节目〝60分钟〞（60 Minutes）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专访.html" target="_blank">
  专访
 </a>
 中，谈及俄罗斯、中国、朝鲜以及美国内阁等等问题。
 <br/>
 <br/>
 川普表示，俄罗斯总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普京.html" target="_blank">
  普京
 </a>
 （Vladimir Putin）是个非常强硬的人，而今年7月在芬兰赫尔辛基（Helsinki）举行的〝双普会〞，是个虽然艰难但却非常好的会谈。
 <br/>
 <br/>
 当被记者问及〝总统是否也认为，俄罗斯总统普京卷入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒杀.html" target="_blank">
  毒杀
 </a>
 事件〞时，川普表示：〝他有这个可能性。〞〝当然他们不应该这么做。〞
 <br/>
 <br/>
 至于俄罗斯试图影响美国2016年总统大选，川普指出，自己并不认为只有俄罗斯而已，〝我认为中国（中共）也进行了干涉〞，可能还有其他国家，但他强调，〝老实说，我觉得中国（中共）问题更大。〞
 <br/>
 <br/>
 今年3月4日，前俄罗斯军事情报官员谢尔盖．斯克里帕尔（Sergei Skripal）和他33岁的女儿尤莉娅（Yulia Skripal）在英国中毒险些丧命。
 <br/>
 <br/>
 3月12日，英国首相特雷莎・梅（Theresa Mary May）向国会议员表示，造成斯克里帕尔父女中毒的物质，是俄罗斯研发的军用神经毒剂诺维乔克（Nowitschok），并指俄罗斯有很大可能需要为此事件负责。英国调查结果亦显示，〝该行动很可能获得（俄罗斯）政府层面的批准。〞
 <br/>
 <br/>
 不过，俄罗斯方面始终否认对毒杀事件负有责任。
 <br/>
 <br/>
 <h4>
  川普：美国只想与中方达成〝公平协议〞
 </h4>
 <br/>
 至于中美贸易关系，川普在采访中表示，中美之间进行的只是小型冲突（skirmish），而非〝贸易战〞（trade war），而美国终会赢得胜利。
 <br/>
 <br/>
 他还强调，自己无意影响中国经济发展，只想求得公平交易。
 <br/>
 <br/>
 他说：〝我不想这样，我想要他们（中共政府）与我们进行谈判，达成公平的协议。我要他们和我们一样开放市场，我们的市场是开放的。〞
 <br/>
 <br/>
 川普也表示，自己与中国国家主席习近平有良好的关系，但他强调，他已经告诉习近平一件事，那就是〝中国（中共）不能继续以贸易或其它任何方式，每年从美国赚走5000亿美元〞。
 <br/>
 <br/>
 川普并再次提出警告说，目前，美国已经展开新一轮关税征收，但如果中共再次对美国农民或其它行业进行报复，他将会继续追加关税。
 <br/>
 <br/>
 谈到朝鲜问题，川普说，他知道金正恩在侵犯人权，但自己（当前）只能努力将美国受到的威胁减少。川普还表示，〝我和他（金正恩）相处得很好，我们有很好的关系。〞
 <br/>
 <br/>
 谈到美国内阁是否存在问题？川普在专访时透露，国防部长马蒂斯（James Mattis）可能打算离职。
 <br/>
 <br/>
 川普说：〝如果你想知道原因，我认为他有点像民主党员。〞但他强调，〝马提斯将军是个好人，我们相处得非常好〞，两人会共进午餐，但〝在某种程度上，每个人都会离职，这就是华府〞。
 <br/>
 <br/>
 川普还说：〝我现在有待命的人，他们会非常杰出。〞
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：东野）
