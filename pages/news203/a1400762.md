### 台媒：中美关税战是否升级 关键不在中共妥不妥协
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】美方不断警告，如果月底美中首脑会晤达不成协议，将对更多中国产品加征关税。但台媒文章指，从美方态度看，贸易战仍将继续，美方是否继续加税，中共妥协与否已不是关键，川普（特朗普）将主要考量此举对美中双方的实际影响。
 <br/>
 <br/>
 台湾《上报》11月24日的这篇评论文章称，中共不断利用不公平贸易手段掠夺他国，特别是美国的经济利益，成为美国总统川普上台后面临的一个棘手问题。
 <br/>
 <br/>
 文章指，美国中期选举并没有给川普执政带来挑战。日前美国副总统彭斯开列美中达成贸易协议的六大条件，以及美国司法部本月起诉福建晋华公司盗窃美国知识产权，显示美国政府对中共的态度越来越强硬。
 <br/>
 <br/>
 此外，中美在今年APEC峰会上激烈交锋，导致峰会宣言流产。美国商务部长罗斯近日确认，美中不可能在明年1月前达成全面协议，因此美方将按原计划在明年1月把对2,000亿美元中国商品加征关税的税率从10%增加到25%。
 <br/>
 <br/>
 文章指，这都显示G20上的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 很难有实质的内容产生，贸易战仍将继续。美方是否继续增加关税，不再取决于中共是否妥协，而将主要考量进一步行动对中共的打击力度，以及对美国国内产生的影响。
 <br/>
 <br/>
 据彭博社日前引述欧盟的〝欧洲经济和财政政策研究网络〞（EconPol Europe）最新论文指，若明年1月美国对2,500亿美元中国商品全部加征25%关税，美国企业和消费者只会承担其中的4.5%，另外20.5%将由中国制造企业承担。
 <br/>
 <br/>
 论文说，川普政府在这2,500亿美元的加税清单中，选择了具有最高〝价格弹性〞的产品，或者具有很高替代性的中国产品。这不仅能够将对美国消费者与企业的负面影响降至最低，也迫使中国出口商为留住买家而削减售价，从而为美国政府带来184亿美元的净收益。
 <br/>
 <br/>
 另外，这些关税还会将中美贸易逆差降低17个百分点。而降低中美贸易顺差，也是美国启动关税战的一个主要目标。
 <br/>
 <br/>
 但论文也指出，随着中美贸易战持续升级，美国政府要挑选出更多高进口弹性产品将越来越困难，如果继续对剩余的中国产品加税，对美国消费者的影响将会增大。
 <br/>
 <br/>
 但此前也有权威研究报告称，如果美国决心继续升级关税战，中方承受的损失将远大于美方。
 <br/>
 <br/>
 截至目前为止，虽然美方加征的关税遭到美国一些相关产业的反对，但并没有对美国消费者指数产生明显的影响。外界分析，美国经济强劲的背景，也会加大川普政府继续升级关税战的信心。
 <br/>
 <br/>
 不过，美国对剩余2,670亿美元中国商品加征关税的计划，显得比较审慎。近期以来，美方对中方的施压，开始从关税更多地扩展到技术、军事、政治和外交等领域。
 <br/>
 <br/>
 但是，根据川普政府最近释放的信号，如果此次G20的中美首脑会晤失败，美方很快推出2,670亿加税计划的可能性大增。
 <br/>
 <br/>
 川普20日在白宫接受媒体访问时表示，〝如果我们（川习会）没达成协议，我还有另外价值2500亿美元的关税〞，〝相信我，我会向这些商品加征关税，因为中国（中共）许多年来一直在敲诈我们国家，有我在，他们别想再敲诈我们。〞
 <br/>
 <br/>
 据此前消息，美方或将在明年初推出对剩余中国产品加征25%关税的计划。日前美国贸易代表办公室表示，计划并没有改变。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：云涛）
