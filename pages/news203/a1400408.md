### 北加州降雨火势控制85% 感恩节持续搜救
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】北加州近日下起大雨，缓解野火
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灾情.html" target="_blank">
  灾情
 </a>
 ，火势控制住85%。据最新统计，还有563人下落不明，救灾人员在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 期间继续执行任务。
 <br/>
 <br/>
 两周前爆发的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北加州野火.html" target="_blank">
  北加州野火
 </a>
 ，烧毁了大约芝加哥城市大小的240平方英里的土地，摧毁了超过13,000栋房屋。
 <br/>
 <br/>
 当地近日下起大雨，缓解
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灾情.html" target="_blank">
  灾情
 </a>
 ，目前85%的火势已控制住。
 <br/>
 <br/>
 警方周三晚表示，目前大火已造成83人丧生，563人下落不明，这个数字还在不断变化。
 <br/>
 <br/>
 加州巴特县警长哈尼亚：〝截至今晚，下落不明的人数为563人。这一数字从昨晚开始，减少了307人。我想再次告诉大家，这个很重要，要记住这是一个不断变化的数字。你会看到它上升或下降。〞
 <br/>
 <br/>
 消防搜救人员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 期间不休息，继续执行搜寻任务。
 <br/>
 <br/>
 加州巴特县警长哈尼亚：〝我们仍将继续搜寻，继续执行任务。我们有830人，包括执法、搜救和志愿人员在现场搜寻遗体。〞
 <br/>
 <br/>
 虽然大雨缓解了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火灾情.html" target="_blank">
  野火灾情
 </a>
 ，但气象部门表示，一些地区可能伴随时速45英里的强风，有爆发山洪和泥石流的危险，也增加搜救难度。
 <br/>
 <br/>
 大雨也让因火灾失去家园的人们更艰难，灾民们祈祷早日恢复正常生活。
 <br/>
 <br/>
 北加州野火灾民AMY SHEPPARD：〝我只是祈祷这些都能实现，我们能一起建造另一个社区，重建我们的城镇，修复家园。〞
 <br/>
 <br/>
 由于正值感恩节出行高峰，降雨天气迫使旧金山国际机场取消了40多个航班。
 <br/>
 <br/>
 新唐人记者李梅、宇微综合报导
