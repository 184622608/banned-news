### 出席非裔青少年领袖论坛 川普期待未来出总统
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】周五，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统.html" target="_blank">
  总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在非裔少年领袖论坛上演讲，告诫年轻人，不要妒嫉；应树立正确目标并绝不放弃。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周五中午来到美国非裔少年代表中，为论坛致词。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统.html" target="_blank">
  总统
 </a>
 川普：〝看看这些美丽、英俊的脸。（相信）有一天，你们当中3-4个人会站在这里。对此我丝毫没有怀疑，毫不怀疑。你们谁想有一天当总统？〞
 <br/>
 <br/>
 川普鼓励他们坚持独立思考。
 <br/>
 <br/>
 美国总统 川普：〝他们说美国优先是种族主义，我说种族主义，什么叫种族主义？我相信在这里的各位都同意，你们生活在美国，美国优先，不对吗？〞
 <br/>
 <br/>
 非裔少年代表：USA 美国（优先）；USA 美国（优先）⋯⋯
 <br/>
 <br/>
 美国总统 川普：〝保守运动，是走向容忍、团结和进步。民主党只知道抵制。我们专注于结果，我们要结果。他们攻击我们，因为我们说真话、改变人们的思想、每天都在印证我们的政策是正确的，我们做的是对的事好的事。我也被攻击、你们也被攻击，我一直被攻击。〞
 <br/>
 <br/>
 川普也告诉年轻人不少智慧的处世经验。
 <br/>
 <br/>
 美国总统 川普：〝妒嫉非常的不好，卑鄙又令人讨厌。妒嫉的人永远成不了事。永不放弃，最重要的是不放弃，但要确定你的目标是正确的。〞
 <br/>
 <br/>
 上季度美国经济增长3.5%。川普就任将近两年，非裔美国人失业率、贫穷水平等都创下历史新低。
 <br/>
 <br/>
 新唐人记者李兰综合报导
