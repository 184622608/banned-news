### 参议院投票通过 确认卡瓦诺大法官提名
------------------------

<div class="wysiwyg">
 【新唐人2018年10月06日讯】周六下午，美国联邦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 全体投票，确认通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院大法官.html" target="_blank">
  最高法院大法官
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 的提名。这位保守派大法官很快将正式就职。这对未来美国的价值取向，将产生深远影响。
 <br/>
 <br/>
 美国副总统、兼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 议长彭斯在宣布投票结果之前，首先询问议员们是否确认自己的投票。
 <br/>
 <br/>
 美国副总统 彭斯：〝在座各位议员，有没有人想要投票或更改投票？如果没有，本次投票，50票赞成，48票反对。马里兰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院大法官.html" target="_blank">
  最高法院大法官
 </a>
 提名确认。〞
 <br/>
 <br/>
 结果宣布后，议程被走廊的抗议声多次打断。
 <br/>
 <br/>
 随后，参议院多数党领袖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麦康奈尔.html" target="_blank">
  麦康奈尔
 </a>
 宣布，投票结果将报告给总统。
 <br/>
 <br/>
 参议院多数党领袖麦康奈尔：〝我要求考虑动议且已经过考虑，并且摆到桌面上。参议院的投票将立即通报给总统。〞
 <br/>
 <br/>
 川普总统随即发推文，表示祝贺。
 <br/>
 <br/>
 推文说：〝我欢迎并祝贺美国参议院确认我们的重要提名，卡瓦诺法官进入美国最高法院。今天晚些时候，我将签署任命令，他将正式宣誓就职。非常高兴！〞
 <br/>
 <br/>
 51位共和党参议员中，蒙大拿州参议员史蒂夫．戴恩斯因参加女儿婚礼，没有参加投票。
 <br/>
 <br/>
 川普推文说：〝史蒂夫已准备好做他要做的，但我们已经够数了。〞川普向戴恩斯一家表示祝贺。
 <br/>
 <br/>
 至此，美国最高法院9位大法官中，左右派比例恢复到4：5，保守派占多数。这对未来美国社会的司法、价值取向等，将产生深远影响。
 <br/>
 <br/>
 新唐人记者易闻、田瑞综合报导
</div>
