### 纳瓦罗或参加G20川习会 股市震荡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】美国总统川普（特朗普）和中国国家主席习近平，将在全球20国首脑峰会（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 ）上进行双边会晤，讨论美中贸易争端。据报导，对北京鹰派人物
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 将出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 ，投资人担心达成协议希望不大，股市随之震荡。
 <br/>
 <br/>
 白宫官员周四（29日）告诉CNBC，白宫贸易政策顾问、对北京现政权鹰派人物彼得・
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 （Peter Navarro）将出席川普与习近平的重要会晤。
 <br/>
 <br/>
 路透社也引述一位美国政府官员的话透露，纳瓦罗将出席在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会期间举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 。
 <br/>
 <br/>
 <h4>
  股市震荡起伏
 </h4>
 <br/>
 监于纳瓦罗对美中贸易的长期强硬态度，他出席会议的消息降低了在晚宴上达成贸易协议的可能。
 <br/>
 <br/>
 道琼工业平均指数因联准会(Fed)的会议纪录连续3天上涨，而纳瓦罗的确定出席，让道指短暂下跌超过160点，跌至当天的最低。
 <br/>
 <br/>
 29日，道琼工业平均指数下跌27.59点或0.11%收2万5338.84点。史坦普500指数下跌5.99点或0.22%收2737.8点。那斯达克综合指数下跌18.51点或0.25%收7273.08点。
 <br/>
 <br/>
 《道琼》报导，美国和中国正在寻求降低贸易紧张局势的协议，股市交易变得更加波动。
 <br/>
 <br/>
 <h4>
  纳瓦罗：川普为不公平贸易发声
 </h4>
 <br/>
 纳瓦罗是《致命中国：中共赤龙对人类社会的危害》的共同作者。书中强调中共利用贸易优惠给美国带来威胁。
 <br/>
 <br/>
 2012年，川普为《致命中国》纪录片写了推荐语，他说：〝《致命中国》说得很对，这部重要的纪录片用充分的事实、数据和洞察力，描述了我们与中国之间存在的问题。我强烈推荐大家观看。〞
 <br/>
 <br/>
 纳瓦罗11月9日称，〝川普具有勇气和智慧，为环球精英发声，为全球那些受到不公平贸易行为影响的国家发声。〞他主导所有的贸易谈判，任何与中共合作想要左右川普的华尔街〝亿万富翁〞，都是〝没有登记的外国代理人〞。
 <br/>
 <br/>
 他说，〝川普在解决这个问题上做得非常出色，他不需要华尔街的帮助，也不需要高盛的帮助，他现在不需要这些帮助。〞
 <br/>
 <br/>
 <h4>
  纳瓦罗参会 对中方释何信号
 </h4>
 <br/>
 CNBC报导，纳瓦罗参加川习晚餐会晤，可以被视为是在向中方发出信号，表明贸易鹰派在美国政府内部更具主导地位。
 <br/>
 <br/>
 熟悉贸易谈判的人士告诉《南华早报》，纳瓦罗此前在北京举行的美中贸易谈判中对中共态度强硬，要求中共从根本上改变其产业政策。
 <br/>
 <br/>
 G20峰会将于11月30日～12月1日在阿根廷举办。川普政府内阁的几位成员，包括财政部长姆钦、贸易代表莱特希泽，以及国务卿蓬佩奥和国家安全顾问博尔顿将出席峰会。
 <br/>
 <br/>
 <h4>
  美中各自诉求
 </h4>
 <br/>
 周六，在布宜诺斯艾利斯举行的川习晚宴将为美中之间不断升级的贸易战提供一些答案。这也是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 爆发以来，川普和习近平的首次会面。
 <br/>
 <br/>
 白宫首席经济顾问拉里・库德洛（Larry Kudlow）11月27日表示，目前中方对川普提出的贸易要求做出的回复不令人满意，但川习会提供了一个打破贸易僵局的机会。他还说，要想结束贸易战，北京还需要做更多。
 <br/>
 <br/>
 川普政府至今已向2,500亿美元中国商品加征10%到25%的惩罚性关税，大约占美国去年自中国进口商品总额（5,060亿美元）的50%。中共则对大约1,100亿美元美国商品实施等值的报复性关税，约占其去年自美国进口商品总额（1,300亿美元）的85%。
 <br/>
 <br/>
 对于本周的川习会，川普政府的目标是北京政权必须提出具体行动方案，取消其不公平贸易措施、停止窃取美国知识产权、不再强制技术转让、取消扭曲市场的产业补贴政策及非关税壁垒措施，以及进一步开放市场等。
 <br/>
 <br/>
 中共官员表示，他们的首要任务是说服华府暂停明年1月1日提高2,000亿美元中国商品的关税。对此，川普26日告诉《华尔街日报》，美方不太可能让步。
 <br/>
 <br/>
 （记者李红报导/责任编辑：戴明）
