### 川普成白宫真正发言人 11天回答300多个提问
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国总统.html" target="_blank">
  美国总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）最近敞开心怀，在过去11天，他回答了记者大约300个提问，基本上取代了白宫发言人桑德斯的位置。
 <br/>
 <br/>
 据ABC周四（10月18日）报导称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在过去11天回答了记者大约300个提问，超过史上任何
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国总统.html" target="_blank">
  美国总统
 </a>
 在同等时间段的回答提问。而自从劳工节以来，桑德斯仅仅举行了两场新闻发布会。
 <br/>
 <br/>
 自从10月6日以来，川普已经在南草坪跟白宫记者举行了6场即兴新闻发布会，跟电视媒体举行了至少8场采访，跟平面媒体举行了5场采访，在白宫举行了18场活动。在每一场活动中，川普都回答了记者提问。
 <br/>
 <br/>
 川普还接受了部分媒体的长时间采访，包括接受〝CBS60分钟〞的采访，跟〝福克斯朋友〞节目进行43分钟的交谈。
 <br/>
 <br/>
 美国驻联合国大使黑利上周宣布将辞职时，川普一反过去的行事作为，邀请记者到椭圆形办公室，回答了18个提问。之后川普还问：〝还有什么问题吗？〞直到没有记者再举手才结束。而过去的发布会川普没有回答多少提问就会离开。
 <br/>
 <br/>
 ABC称，他们没有找到其他美国总统在任何11天内比川普回答的提问更多的。
 <br/>
 <br/>
 媒体对川普最近敞开心怀的回答提问都很欢迎。川普在最近一次采访中也承认，他让媒体更容易接触到自己，是为了11月份的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 。川普正积极在全美国为共和党助选。共
 <br/>
 <br/>
 从9日开始，川普就在Instagram上持续贴图，号召选民出来投票。美国中期选举的〝先期投票〞和邮寄投票已在20个州展开。
 <br/>
 <br/>
 根据乔治亚州务卿办公室等多方数据，今年这个州的先期投票率比4年前增长了230%（296,601 vs 99,881）。近邻田纳西州，州府所在的Davidson县周三、〝先期投票〞首日，有10,249人参与，仅仅比2016年总统大选少了2千人，在中期选举中实属罕见。
 <br/>
 <br/>
 皮尤研究中心的报告显示，川普是激励选民出来投票的重要因素。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
