### 川习会前 美301报告：中共未改不公贸易
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 将在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 上有缓和机会吗？
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 贸易代表署，发布〝301条款〞调查的更新报告，明确指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 并没有彻底改变不公平、不合理以及扭曲市场的作法。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 贸易代表署，发布〝301条款〞（Section 301）调查的更新报告，美国贸易谈判代表莱特海泽在声明中说，更新报告是作为政府加强管控和落实法规的一部分，调查结果显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 并未彻底改变不公平、不合理以及扭曲市场的作法。
 <br/>
 <br/>
 美国副总统 彭斯：〝美国不会改变做法，直到中方改变其行为方式。〞
 <br/>
 <br/>
 报告还提到，中共仍继续以政策，支持透过网路的窃取美国的智慧财产权，以及歧视性技术许可。并继续以投资限制，来对美国科技公司施压，以移转技术。
 <br/>
 <br/>
 新美国安全中心亚太安全项目高级主任 Patrick Cronin：〝中共认为美国虚弱，会在压力下屈服，所以重要的是美国要强硬，并坚持强硬，即使其它国家有担懮。〞
 <br/>
 <br/>
 G20峰会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 即将到来，美国再次对中方施压，而中共是否能在月底前，提出川普政府能接受的改革清单，外界持续关注。
