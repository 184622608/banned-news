### 令人心碎！消防员遭酒驾撞死 未婚妻墓前孤影拍婚纱照
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】美国印第安纳州一名消防员与女友订下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_婚约.html" target="_blank">
  婚约
 </a>
 ，要在今年9月29日结婚，但消防员却不幸在去年11月遭人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒驾.html" target="_blank">
  酒驾
 </a>
 冲撞当场死亡。女友在原本要举行婚礼的当天，独自穿婚纱来到男友的墓前拍照，画面令人鼻酸，看哭众网友。
 <br/>
 <br/>
 据《福斯新闻》报导，当时撞死27岁墨菲（Kendall Murphy）的驾驶，血液酒精浓度为0.21％，几乎是法定限制的3倍之多。
 <br/>
 <br/>
 墨菲就此与女友帕吉特（Jessica Padgett）阴阳两隔。帕吉特在他们订下结婚的当天，请来摄影师克内普（Mandi Knepp）为她拍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_婚纱照.html" target="_blank">
  婚纱照
 </a>
 ，帕吉特除了请来墨菲的家人合照，也一一的与他消防装备、消防靴合照。
 <br/>
 <br/>
 而一张她在墨菲墓前独自思念的孤影，更令网友心动，不禁落泪。网友留言形容说自己〝泪流满面，喔，我的天啊！〞、〝我...我怎么哭了！〞、〝这画面好美，但我心碎了...〞、〝一个哭泣的天使...〞。
 <br/>
 <br/>

 <br/>
 <br/>
 美国殉职消防员基金会表示，墨菲是无私的人，特别喜欢和家人与朋友共度时光。
