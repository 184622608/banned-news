### 佛州宣布重新计票 7500名志愿者赶赴监督
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 结果因争议巨大而返回待定状态。周六（11月10日），佛州州务卿宣布，将对两场州长和参议员选举
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统警告他正“密切关注”事态发展，7,500名计票
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_志愿者.html" target="_blank">
  志愿者
 </a>
 也已赶到佛州参与监督。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 州务卿戴兹特（Ken Detzner）周六宣布，鉴于该州州长和参议员选举的计票结果非常接近，因而启动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 机制，重新计票结果将于美东时间下周四（11月15日）下午3点前出台。
 <br/>
 <br/>
 涉及重新计票的两场竞选分别是民主党参议员候选人尼尔森（Bill Nelson）与现任州长、共和党参议员候选人斯科特（Rick Scott）之间的竞选，以及两位州长竞选人——民主党籍的塔拉哈西市（Tallahassee）市长吉卢姆（Andrew Gillum）与共和党联邦众议员德桑蒂斯（Ron Desantis）之间的竞选。
 <br/>
 <br/>
 另外，关于农业长官的竞选也要重新计票。
 <br/>
 <br/>
 迈阿密-戴德县（Miami-Dade County）的选举官员说，他们已在周六晚上启动“机器重新计票”，即将纸质选票装入扫描机器。整个处理过程需要持续数日。
 <br/>
 <br/>
 佛州规定，如果获胜候选人所赢得的选票高出对手不足0.5%时，将启动强制性重新计票。如果相差幅度低于0.25%，重新计票工作必须手工完成。
 <br/>
 <br/>
 11月6日选举夜，佛州共和党州长候选人和参议员候选人双双胜出，他们的对手也承认自己败选。但是，在接下来的周三和周四两天，随着迟迟没有结束计票的布劳沃德县（Broward）和棕榈滩县（Palm Beach）多出大量选票且多数流向民主党，共和党参议员卢比奥（Marco Rubio）首先发出警告——佛州民主党可能正在窃取选举。
 <br/>
 <br/>
 截至结束投票的周二选举夜，布劳沃德县共收到了634,000 张选票；但至美东时间周四晚8点，该县的选票竟激增为712,840张，而这些突然冒出来的选票多数都是投给民主党候选人的。
 <br/>
 <br/>
 棕榈滩县也在选举日后突然多出15,000张选票。
 <br/>
 <br/>
 选举夜的非正式计票结果显示，斯科特领先尼尔森50,000多张选票，但是周三和周四，布劳沃德县和棕榈滩县的计票结果却开始巨幅削弱斯科特的优势。一些民主党律师也抵达佛州，要求重新计票。
 <br/>
 <br/>
 而共和党则质疑，这两个县很可能存在选举舞弊现象。
 <br/>
 <br/>
 美国保守派网站〝Gateway Pundit〞周五公布的调查记录显示，布劳沃德县选举监督员斯奈普斯（Brenda Snipes）以往便有不良记录，在此之前曾14次被控选举舞弊。2017年，她在法庭上承认，她所主管的选举部门曾私自同意非公民和重犯投票。
 <br/>
 <br/>
 州长斯科特已于周四晚下令，要求执法部门就可能存在的选举欺诈展开调查。
 <br/>
 <br/>
 至周六下午，斯科特的领先已被压缩为0.15%，德桑蒂斯的优势也降低到0.41%。
 <br/>
 <br/>
 周六，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统发推表示：
 <br/>
 <br/>
 “试图窃取佛州的两场重大选举！我们正在密切关注！”
