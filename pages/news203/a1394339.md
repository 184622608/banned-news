### 英媒：白宫在特殊节日里开火 让北京无比难堪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）4日发表了40分钟的演讲痛批中共的各项国内和国际政策，引爆舆论。英媒说，在正值中共纪念其建政69年节日之际，彭斯向中共开火，火力之猛，让北京无比难堪。
 <br/>
 <br/>
 英媒《电讯报》10月5日报导说，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 指责中共，试图从军事、经济和政治方面损害川普（特朗普）政府，这一系列指责，给本来就已经脆弱的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 雪上加霜。
 <br/>
 <br/>
 报导说，眼下中美两国间的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 正在升级，在正值中共纪念其建政69年节日之际，这些指责让北京无比难堪。
 <br/>
 <br/>
 英媒BBC也称，10月初，正在休十一长假的中国人，被美国副总统彭斯的主旨演讲震惊了，这样的唇枪舌剑，为中美贸易战的硝烟，又增添了更加浓厚的政治阴云。
 <br/>
 <br/>
 10月4日，彭斯在华盛顿智库哈德森研究所发表了40分钟的演讲，阐述了川普政府最新、最完整的对中政策。细数中共的种种罪状，对中共进行了全方位的抨击，内容涵盖中共的贸易、军事、人权以及干预美国政治等多个领域。
 <br/>
 <br/>
 虽然演讲中彭斯一直痛批中共的恶行，但在最后他也表示，相信美国人民和中国人民之间的持久友谊，希望共同促进中美两国的繁荣，并称〝中国人民理应得到更多〞。
 <br/>
 <br/>
 彭斯要求北京用实际行动〝重新尊重美国〞，并警告，在美中关系达到〝公平、对等和尊重我们主权〞之前，美方绝不会让步。
 <br/>
 <br/>
 外界分析，彭斯选择在10月4日发表这样一篇猛烈抨击中共的演讲，最直接的原因是：在中美贸易战爆发，中共处境极为不利的情况下，开始干涉美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，希望川普失去参众两院的支持。彭斯指责中共希望换一个总统。
 <br/>
 <br/>
 有评论分析，从彭斯副总统的讲话可以看到，中美关系发生巨大变化。美国对中国由失望到绝望，现在通过强力遏制中共的扩张，迫使其遵守文明准则。
 <br/>
 <br/>
 也有评论人士说，彭斯的演讲，不只是为了打赢期中选举，这个演讲可以和71年前丘吉尔的〝铁幕演说〞相提并论。
 <br/>
 <br/>
 英国已故首相丘吉尔1946年3月5日在美国威斯敏斯特学院发表的反苏联、反共产主义的演说，利用〝铁幕〞一词，意指苏联和东欧社会主义国家〝用铁幕笼罩起来〞，该演说被称为〝铁幕演说〞，被认为是正式拉开了美苏冷战的序幕。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：赵云）
