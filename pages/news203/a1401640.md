### 宾州西切斯特圣诞游行 天国乐团送祝福
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】切斯特郡（Chester County）是宾州创建之初三个古老的郡之一，已有336年的历史。11月30日傍晚，首府西切斯特市举行了传统的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞游行.html" target="_blank">
  圣诞游行
 </a>
 。由法轮功学员组成的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天国乐团.html" target="_blank">
  天国乐团
 </a>
 再次应邀参加。
 <br/>
 <br/>
 冬日雨后的傍晚有点冰冻，不过仍有4万多民众涌上街头，参加当地年度圣诞活动的经典开场节目——西切斯特（West Chester）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞游行.html" target="_blank">
  圣诞游行
 </a>
 。
 <br/>
 <br/>
 游行在主干道〝市场街〞，唐宁镇(DowningTown）高中乐团等当地学校、商会及企业、舞蹈团体等都组队参加。
 <br/>
 <br/>
 观众 Don：〝这是我们第一次看游行，相当棒。〞
 <br/>
 <br/>
 来自纽约的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天国乐团.html" target="_blank">
  天国乐团
 </a>
 作为唯一的华人团体也应邀参加。主持人介绍，天国乐团成立于2005年，由法轮大法学员组成。法轮大法是促进身心健康的功法，老少皆宜。
 <br/>
 <br/>
 整齐的队伍，亮丽的乐音，让许多观众耳目一新。
 <br/>
 <br/>
 观众 Amanda：〝他们穿着亮丽色彩的服装，这是我最喜欢的部分。〞
 <br/>
 <br/>
 当地民众非常感谢天国乐团远道而来，听到令人振奋的演奏感到幸运。
 <br/>
 <br/>
 这个游行充满传统的节日气息，也被《今日美国》选为全美能看到圣诞老人的十大活动之一。
 <br/>
 <br/>
 观众 Amanda的女儿：〝我觉得正走过来的是我最喜欢的。〞
 <br/>
 <br/>
 观众：〝圣诞老人。〞
 <br/>
 <br/>
 观众：〝圣诞老人，我们都在等待他。〞
 <br/>
 <br/>
 除了游行外，还有已持续10年的暖场经典项目——2英里的Jingle Elf跑走活动。
 <br/>
 <br/>
 美国网上购物电视频道QVC的总部就设在西切斯特，圣诞节期间会给上亿的电视观众重播游行全程录像，给数千万家庭增添节日的喜悦。
 <br/>
 <br/>
 新唐人记者李和平宾州西维切斯特市报导
</div>
