### 港媒：中美贸谈突然重大调整 或影响川习会
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】为了给11月底举行的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞铺路，近期中美高层恢复接触。消息人士透露，中美贸易代表团原定于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华盛顿.html" target="_blank">
  华盛顿
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谈判.html" target="_blank">
  谈判
 </a>
 ，但在最后一刻将地点改为第三方国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 ，这将对习近平产生更加直接的影响。
 <br/>
 <br/>
 11月18日，香港《南华早报》援引一名消息人士的话称，中美贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谈判.html" target="_blank">
  谈判
 </a>
 团队，在最后时刻修改了原始计划，将对话安排在第三方国家而不是在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华盛顿.html" target="_blank">
  华盛顿
 </a>
 。
 <br/>
 <br/>
 报导引述一位消息人士称，在北京与华盛顿方面进行了〝工作层面的讨论〞之后，中方负责人刘鹤，原计划飞往华盛顿，与可能由美国财长努钦（Steven Mnuchin）率领的美国代表团进行谈判，但目前已经进行了调整。
 <br/>
 <br/>
 消息人士表示，〝最新计划是两国贸易谈判团队在布宜诺斯艾利斯会面。〞 这一变化的部分原因，是为了确保会谈能够对11月底在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 的G20峰会期间，习近平与美国总统川普（特朗普）的会晤产生直接作用。
 <br/>
 <br/>
 报导说，如果上述消息得到确认，这一变化表明中美首脑会晤的分量将提升。会晤期间或有更多重大的事项提上日程。这也是两个最大经济体的领导人，首次在第三国进行双边会晤，而此次贸易对话的推进，将对〝习川会〞产生更直接的影响。
 <br/>
 <br/>
 《华尔街日报》12日称，在中美两国领导人定于G20峰会期间会晤之际，美财长努钦恢复与刘鹤的对话，讨论一个将缓解双方贸易紧张关系的协议。
 <br/>
 <br/>
 一些对中共持强硬立场的美国官员说，在美中两国领导人会晤前，中方将拿出一项提议。
 <br/>
 <br/>
 他们认为，美中两国最多可能会就贸易争端达成某种休战协议，美国会暂停对中国产品加征关税。而习近平已要求各部委提出对谈判的具体期望，以便双方能够在G20峰会后继续谈判。
 <br/>
 <br/>
 为了给〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞铺路，中共日前向美国政府提交了一份贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_让步清单.html" target="_blank">
  让步清单
 </a>
 。
 <br/>
 <br/>
 据《路透社》引述一名美国官员透露，中共向美国递交的清单有142个项目，主要分为3类：即中共愿意进一步谈判采取行动的问题，已经着手解决的问题，及被视为不会谈判的禁区问题。
 <br/>
 <br/>
 报导引述一名消息人士透露，中共的让步方案中，很多都是重新列出北京以前做出的调整，而且这些文件并未涵盖放弃〝中国制造2025〞计划上的承诺。
 <br/>
 <br/>
 CNN报导称，中共开出的条件并没有包括白宫的核心要求，例如解决强迫技术转移和偷窃知识产权等问题。
 <br/>
 <br/>
 一名消息人士透露，中方给的东西并不是什么新的，双方仍处在僵局之中。通道现在再次打开了，但是仍有很多工作要做。
 <br/>
 <br/>
 美国总统川普16日表示，中共递交了一份很大的清单回应美国对贸易改革的要求，有很多我们要求的东西，但是还有4、5件事被漏掉了。
 <br/>
 <br/>
 川普说，中共想要达成协议，〝但是，我仍然认为不可接受。我认为，我们在中国方面做得非常好。〞中美贸易不公平，中方对美国征收了巨额的关税，而美国没有对中方征收。〝现在一切都将改变。〞
 <br/>
 <br/>
 17日在亚太经济合作组织（APEC）领导人峰会上，美国副总统彭斯称，中共设置的贸易壁垒史无前例，包括关税、盗用知识产权、强逼技术转让等，美国已对中共采取决定性的措施，对2500亿美元的中国产品征收关税，并可能将征税规模扩大一倍。
 <br/>
 <br/>
 彭斯说，美国不会改变这一应对方式，除非中共改变其行为方式。彭斯还警告中共，必须全面改变其行为作法，才能避免与美国发生新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冷战.html" target="_blank">
  冷战
 </a>
 。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
