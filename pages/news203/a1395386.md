### 【禁闻】美国会简报会 聚焦中国宗教信仰自由
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】日前，一个主题为〝捍卫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人权.html" target="_blank">
  中国人权
 </a>
 的最后尝试〞的会议，在美国国会山举行，会议主持者透露，中美双边关系基本上都在重新谈判之中，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教自由.html" target="_blank">
  宗教自由
 </a>
 将是所有谈判的先决条件。
 <br/>
 <br/>
 10月11号，美国国会〝国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教自由.html" target="_blank">
  宗教自由
 </a>
 核心小组〞（IRFC）及非政府机构〝国际宗教自由圆桌会议〞，在美国华盛顿国会山共同主持召开简报会，主题是: 〝捍卫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人权.html" target="_blank">
  中国人权
 </a>
 的最后尝试会议（A Final Attempt to Defend Human Rights in China.）〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国政府.html" target="_blank">
  美国政府
 </a>
 官员、非政府组织、国会山工作人员以及媒体参加了会议。其中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 发言人张而平、基督教代表Kunrui Li、维吾尔代表Omer Kanat，以及人权援助协会康（Kang)女士，在会议上介绍了中共对不同宗教信仰团体的持续打压和迫害情况。
 <br/>
 <br/>
 法轮功发言人张而平：拿迫害法轮功的案例来说，他们有610办公室，610办公室被中央政治局掌控，政治局的人直接控制610办公室，因为610办公室下令到省、县和市所有相关部门，不仅直接控制资深负责人，而且控制所有关键部门或单位 。
 <br/>
 <br/>
 法轮功发言人张而平说，中共迫害法轮功已经持续19周年了，截至2018年10月11号，经确认并记录在案的，已经有4,258名法轮功学员被迫害致死，由于受中共审查制度和调查犯罪的困难条件所限，实际被迫害致死的人数要高出许多倍。张而平呼吁美国继续调查在中国仍在发生的强摘器官暴行，并采取一切必要手段制止这一反人类罪行，停止中共打压中国公民的信仰。
 <br/>
 <br/>
 美国国际宗教自由委员会研究与政策副主任Tina L.Mufford表示，鼓励美国政府所有机构持续参与进来讨论中国人权问题。
 <br/>
 <br/>
 华盛顿国际宗教自由圆桌会议共同主席兼协调员Greg Mitchell，会后向新唐人透露，现在美国正在重建与中国的关系，诸如国家安全、经济贸易等一切双边关系，都将从新谈判，所有协定都将注入宗教自由成分。
 <br/>
 <br/>
 华盛顿国际宗教自由圆桌会议共同主席兼协调员Greg Mitchell：如果我们重新谈判新的贸易政策并且没有关税战争，那么就需要遵循现在的问题，以便没有关税战争。其中一个条件应该是他们需要开始改善宗教自由，作为新的贸易协定的一部分谈判,  否则我们不会签字。
 <br/>
 <br/>
 国际宗教自由核心小组顾问Miranda Robinson向新唐人表示，这一次会议之所以取名〝捍卫中国人权的最后尝试〞，是因为没有全面的压力，中共政府不可能真正改善中国人权，而只是在继续欺骗中加重迫害。
 <br/>
 <br/>
 国际宗教自由核心小组顾问Miranda Robinson：我认为这是审视中国人权的最后一次尝试，因为情况正在不断恶化。如果没有中国允许的社交媒体，世界就不会对此采取后续行动。
 <br/>
 <br/>
 Miranda Robinson透露，现任美国政府非常关心中国人权，他们在观察，在倾听，会尽最大努力帮助中国人民。
 <br/>
 <br/>
 据称，这次会议，是为接下来的联合国审议中国人权做准备。
 <br/>
 <br/>
 采访/王凯迪 编辑/刘惠 后制/肖颜
