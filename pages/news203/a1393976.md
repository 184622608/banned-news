### 反卡瓦诺抗议或有黑幕 美媒起底索罗斯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】近日，两组女性抗议者分别拦阻多位美国参议院议员，试图说服他们改变对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 的支持。《福克斯新闻》报导，这两组人都来自同一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_左派.html" target="_blank">
  左派
 </a>
 组织，而多个反卡瓦诺的左派组织都得到了富豪
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_索罗斯.html" target="_blank">
  索罗斯
 </a>
 的资助。
 <br/>
 <br/>
 上周五，美国参议院司法委员会投票通过对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 的表决，但当时出现戏剧性一幕。原本无条件背书卡瓦诺的共和党联邦参议员弗雷克提出条件，要求FBI介入对卡瓦诺性侵指控的调查，才会在之后的参院全体表决中投票赞成。
 <br/>
 <br/>
 弗雷克承认，他的态度转变，是因为当天他一次乘坐电梯时，两名女性抗议者把他堵在电梯里四分钟，对他说，如果他支持卡瓦诺提名，就说明〝性侵对他来说是无所谓的事情〞。这四分钟改变了他的态度。
 <br/>
 <br/>
 而后来美国媒体起底，两名女性抗议者之一是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_左派.html" target="_blank">
  左派
 </a>
 倡议组织〝普及民主中心〞官员阿齐拉。
 <br/>
 <br/>
 周一，在里根国际机场也出现类似一幕。三名女性拦住了美国参议院共和党领袖麦康奈尔。
 <br/>
 <br/>
 她们中的科德是〝普及民主中心〞的种族正义运动负责人。
 <br/>
 <br/>
 据福克斯新闻报导，最近参与干扰卡瓦诺表决程序的左派组织，都得到过亿万富豪
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_索罗斯.html" target="_blank">
  索罗斯
 </a>
 的资助。记录显示：
 <br/>
 <br/>
 〝普及民主中心〞于2016年和2017年获索罗斯捐款150万美元；
 <br/>
 <br/>
 〝普及民主行动中心〞过去两年获索罗斯120万美元。而该组织在9月初的卡瓦诺听证会上组织过抗议活动。
 <br/>
 <br/>
 1630基金会也在2016年和2017年获捐款73万美元，该基金会资助〝要求正义〞组织刊登反卡瓦诺的电视广告。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
