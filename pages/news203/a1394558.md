### 川普堪萨斯州助选 批民主党极端分子沦左倾暴徒
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）6日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_堪萨斯州.html" target="_blank">
  堪萨斯州
 </a>
 举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选.html" target="_blank">
  助选
 </a>
 活动上，再次称赞最高法院新任大法官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 的优秀人格，并批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 中的极端分子已经蜕变成〝一群愤怒的左翼暴民（an angry left-wing mob）〞。他呼吁更多的民众积极投票给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 人，以保住美国的传统价值。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选.html" target="_blank">
  助选
 </a>
 活动于6日晚间在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_堪萨斯州.html" target="_blank">
  堪萨斯州
 </a>
 首府土匹卡（Topeka, Kansas）的展览中心（Expo Center）举行。
 <br/>
 <br/>
 6日傍晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 发推文说：〝你不会把火炬交给纵火犯，你也不会想把权力交给愤怒的左翼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴徒.html" target="_blank">
  暴徒
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 人已经变得过于极端和危险，以至于不适宜当政。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 人相信法治 - 而不是暴民的统治。投票给共和党！〞

 川普呼吁，为了防止极端民主党掌权，大家应该在一个月后积极投票给共和党，让更多的共和党人进入国会。
 <br/>
 <br/>
 就在助选活动开始前的几个小时，美国副总统彭斯（Mike Pence）在参议院宣布：〝投票结果，50票赞成，48票反对，任命马里兰州法官布雷特‧M‧
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 （Brett M. Kavanaugh）出任美国最高法院副法官的提案，确认通过。〞
 <br/>
 <br/>
 这是总统川普的另一场胜利。自去年1月上任以来，川普提名了两位大法官，都获得参院确认。
 <br/>
 <br/>
 川普在当晚的集会上表示，自己在参加集会前，已经就卡瓦诺的任命签了字。他告诉集会现场的民众：这是〝一个历史性的夜晚〞，并感谢共和党参议员〝未屈服于民主党无耻的政治攻势，以及诋毁（卡瓦诺）人格的行为〞。
 <br/>
 <br/>
 美联社报导称，川普在搭乘〝空军一号〞前往堪萨斯州途中，曾对记者表示，在对卡瓦诺进行人格攻击过程中，一些女性〝在许多方面（表现得）比男性还要粗暴、强劲〞，但也有许多女性〝对于卡瓦诺遭遇的一切感到愤怒〞。
 <br/>
 <br/>
 川普说：〝有非常多的女性（对这一结局）非常高兴，因为他们想到了自己的儿子、夫婿、兄弟等男性亲友。〞
 <br/>
 <br/>
 川普6日还表示，卡瓦诺案尘埃落定，〝这是几年来未见的里程碑〞。未来，共和党人会〝表现得非常好〞。
 <br/>
 <br/>
 外界认为，卡瓦诺出任大法官后，将改变未来几十年最高法院的判决，回归崇尚美国传统价值的保守派理念。
 <br/>
 <br/>
