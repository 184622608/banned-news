### 川普视频讲话 批民主党〝全民医保〞计划
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】十一月开始，美国民众将开始申请新一年的健保计划，川普政府的健保政策备受关注。周二，川普总统发布了一则短视频，批评许多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 人支持的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全民医保.html" target="_blank">
  全民医保
 </a>
 〞计划，将会把美国经济拖垮。
 <br/>
 <br/>
 随着中期选举临近，医保成为美国两党辩论的焦点议题之一。川普周二在视频中，再次阐述了他的观点。
 <br/>
 <br/>
 美国总统川普：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全民医保.html" target="_blank">
  全民医保
 </a>
 是彻头彻尾的谎言。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 人要（让政府）完全接管医疗保险。我要告诉你们，这对你们和家人来説是最糟糕的事。〞
 <br/>
 <br/>
 由民主党参议员桑德斯提出的〝全民医保〞法案，计划把目前仅针对老年人、残疾人和一些重病患者的政府医保计划Medicare，推广到所有人。
 <br/>
 <br/>
 目前，参议院少数党领袖舒默等民主党议员支持这一法案。
 <br/>
 <br/>
 川普一周前在《今日美国》发表专栏文章，批评〝全民医保〞计划将拖垮美国医保体系，最后导向社会主义。川普在视频中重申这一观点。
 <br/>
 <br/>
 美国总统川普：〝你们将必须纳两倍甚至三倍的税，甚至还不够支付全民医保的费用。这将给国家带来灾难，把我们国家变成委内瑞拉，但这不会发生，因为我不会允许它发生。〞
 <br/>
 <br/>
 周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫网站.html" target="_blank">
  白宫网站
 </a>
 发表了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国医疗保险.html" target="_blank">
  美国医疗保险
 </a>
 和医疗补助服务中心主管魏玛的文章。
 <br/>
 <br/>
 文章引用最新数据，说明川普政府成功扭转了医保保费连年大幅上涨的势头，稳定了被奥巴马健保破坏的医保市场。
 <br/>
 <br/>
 其中，2013年到2017年，联邦平价医疗保险的平均费用上涨了105%，而2018年，保险公司的报价五年来首次下降，幅度为1.5%。
 <br/>
 <br/>
 不过文章也认为，目前医保系统对纳税人的负担仍然过于沉重，本届政府还有许多工作要做。
 <br/>
 <br/>
 新唐人记者林宇、张琪综合报导
</div>
