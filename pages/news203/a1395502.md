### 〝60分钟〞专访 川普再点名中共干预选举
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】周日，总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 接受了美国CBS电视〝60分钟〞节目专访，谈到了一系列热点问题，其中，川普再次点名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 干预美国的选举，而且称中共的干预比俄罗斯更加严重。
 <br/>
 <br/>
 在周日播出的访谈中，史塔尔把朝核问题的焦点放在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 表现出的善意。
 <br/>
 <br/>
 美国总统 川普：〝我和他相处得不错，我们的默契感也不错。看看我们最大化施压。（朝鲜）没有再（导弹）威胁、没有再威胁。〞
 <br/>
 <br/>
 专家认为，川普对金正恩本人释出善意，实际上是美国要与朝鲜建立一种与以往不同的关系。
 <br/>
 <br/>
 国家利益中心国防研究主任 Harry Kazianis：〝大家都有点追求巨大、戏剧化的进展，而我认为它（第二次川金会）的意义在于对话变成了常规化，这才是关键，在此基础上才能寄期望于未来的更大进展，但必须先建立起关系。〞
 <br/>
 <br/>
 中期选举将至，史塔尔提到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查，川普表示不会以行政干预调查，但继续重申自己的立场。
 <br/>
 <br/>
 美国总统 川普：〝没有任何形式的勾结，没有勾结（俄罗斯）。他们（俄罗斯）干预了，但中国也干预了我们的大选。坦白讲，中国的问题更严重。〞
 <br/>
 <br/>
 川普也提到正考虑对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 实施新一轮关税。
 <br/>
 <br/>
 国家利益中心国防研究主任 Harry Kazianis：〝美国与中国（中共）之间的竞争可追溯多年甚至数十年。之前没有总统触碰这个问题，也没有让大家知道这有多重要。无论你喜不喜欢川普，你都要给他赞，因为这确实是一个非常重要的外交议题，而川普真正在解决这个问题。我非常赞赏。〞
 <br/>
 <br/>
 副总统彭斯月初发表演讲，向美国大众揭露中共对美的全方位渗透、侵入和扩张。《华尔街日报》发表文章说，美中正逐渐迈向新冷战
 <br/>
 <br/>
 国防部亚太安全事务前助理部长 Wallace Gregson中将：〝现在把美国经济及国家安全与中共威胁联系起来，像冷战时期对苏联的做法，我认为是正确的。〞
 <br/>
 <br/>
 此外，川普访谈中提到国防部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蒂斯.html" target="_blank">
  马蒂斯
 </a>
 有可能离职，备受外界关注。川普称马蒂斯像民主党人。在组建太空军、退出伊朗核协议及对北约的态度上，马蒂斯与川普存在分歧。但川普也说，他与马蒂斯相处融洽。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
