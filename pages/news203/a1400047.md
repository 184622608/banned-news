### 感恩节前夕 美西10万磅生碎牛肉召回
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 前夕，美国农业部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_食品安全局.html" target="_blank">
  食品安全局
 </a>
 宣布，主要牛肉供应商Swift的产品，感染了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大肠杆菌.html" target="_blank">
  大肠杆菌
 </a>
 ，该公司被责令召回近10万磅发往加州、内华达州、俄勒冈州、犹他州和华盛顿州的牛肉。
 <br/>
 <br/>
 美国农业部食品安全检查局，上周四（11月15号），在Swift牛肉公司取样时，检测到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大肠杆菌.html" target="_blank">
  大肠杆菌
 </a>
 后，周五召回被运往加州、内华达州、俄勒冈州、犹他州、和华盛顿州零售商进行进一步加工的生碎牛肉。
 <br/>
 <br/>
 这批生碎牛肉生产于10月24号，大约99,260磅。
 <br/>
 <br/>
 Swift牛肉公司属于JBS USA的一部份，十月因沙门氏菌，召回25州的690万磅的牛肉。有246人受到影响，国家疾病控制中心指其中59人住院。
 <br/>
 <br/>
 节假日即将来临，美国农业部食品安全检查局，提醒民众，为大家庭准备晚餐时，一定要注意食品的准备，洗手、生肉与熟肉不要混用切菜板，小心交叉感染，另外要将将生碎牛肉煮熟，注意温度要达到160华氏度以上。
 <br/>
 <br/>
 新唐人记者古馨乐综合报导
</div>
