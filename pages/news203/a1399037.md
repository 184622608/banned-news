### 中期选举打破中共幻想 财新网含蓄质疑北京强硬路线
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 后，川普（特朗普）政府并没有软化中美贸易战的立场。北京则开始放软姿态，寻求和解。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财新网.html" target="_blank">
  财新网
 </a>
 日前发文〝该有个中美都能接受的方案了〞，虽然符合高层近期表态，但其口吻也被指隐晦质疑北京一直奉行的路线。
 <br/>
 <br/>
 周一（11月12日），《财新周刊》发表社评称〝该有个中美都能接受的方案了〞。社论开篇即表明态度，认为应该〝以更多精力弥合双方具体分歧〞，以期早日达成〝阶段性成果〞。
 <br/>
 <br/>
 文章并称，本月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 是〝一个难得的时间窗口〞，以展示两国决策层解决争端的〝诚意和决心〞。文章还说，〝全球对此寄予厚望〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财新网.html" target="_blank">
  财新网
 </a>
 被指和中共国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 关系密切，在王岐山任中纪委书记时，曾经一度是〝反腐风向标〞，但在中共十九大后似乎已淡出舆论中心。在川习会前敏感时刻，财新网突然发出这样一篇指向性明确的社论，迅速引起舆论关注。
 <br/>
 <br/>
 努力〝推动达成一个双方都能接受的方案〞，这句话出自北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 11月1日和美国总统川普通话时的表态。不过，财新网社论似乎更强调要在月底达成协议。
 <br/>
 <br/>
 自由亚洲电台引述香港一名大陆访问学者分析，这篇社论应该说对于北京迄今为止的一些做法，提出了不同意见。
 <br/>
 <br/>
 该匿名学者认为，财新敢于在这么重大的议题上发出一篇社论，应该是得到高层同意或默许。不过，这名学者也表示，文章到底是不是代表高层意见，是不是放风试水，很快就能知道答案。
 <br/>
 <br/>
 11月6日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 结果，外界普遍认为对中共不利，令北京对中期选举的期待落空。随后，北京对美国的态度进一步软化。
 <br/>
 <br/>
 日前，北京邀请美国前国务卿基辛格访华。日媒指，中方希望基辛格充当〝和事佬〞，向川普传递准备做出某种让步的信息。
 <br/>
 <br/>
 10日，一贯对外强硬的党媒《环球时报》也强调不与美国对抗，将持续〝改革〞，并努力向美国阐明态度，降低美方的疑虑。
 <br/>
 <br/>
 不过，习近平在8日和基辛格的谈话中，依然要求美国〝尊重中国选择的发展道路〞。在5日进博会开幕式上，习近平还将中国经济比作〝不怕巨浪的大海〞。这些表态依然在某种程度上保持了此前〝以牙还牙〞的强硬。
 <br/>
 <br/>
 而王岐山6日在新加坡新经济论坛上，则表示愿意和美国展开磋商，并没有强调中方的〝不妥协〞。也有分析认为，习近平和王岐山在唱〝黑白脸〞。
 <br/>
 <br/>
 对于月底川习会的结果，自由亚洲引述分析指，中国经济形势严峻，中方或希望通过部分妥协达成一个象征性的协议，但双方的巨大分歧依旧。如同美朝川金会达成象征性协议之后的局势发展，中美关系还会再度恶化。
 <br/>
 <br/>
 亦有分析指，中美贸易战短期内完全结束的可能性甚低，中方不大可能完全接受美方要求，而美方朝野对中共强硬立场的一致，也会令川普政府很难和中方和解。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
