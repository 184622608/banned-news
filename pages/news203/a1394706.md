### 美国盯紧中共千人计划 华裔教授因儿童色图被控
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】被中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞招募的美国华裔高阶人才，已被美国FBI盯紧调查，多人涉嫌间谍罪被抓。已被FBI控制多时的德州大学教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢克平.html" target="_blank">
  谢克平
 </a>
 ，日前以涉嫌持有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_儿童色情图片.html" target="_blank">
  儿童色情图片
 </a>
 被控开庭。
 <br/>
 <br/>
 据NBC报导，德州大学安德森癌症中心终身教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢克平.html" target="_blank">
  谢克平
 </a>
 ，今年8月被休士顿的联邦南区法院以涉嫌〝持有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_儿童色情图片.html" target="_blank">
  儿童色情图片
 </a>
 〞罪名提控，10月3日出庭。谢克平目前已交出护照，并交保候审。
 <br/>
 <br/>
 谢克平律师的梅斯(Nathan Mays)称，谢克平否认犯下该罪名。
 <br/>
 <br/>
 报导指，谢克平也因涉嫌为中共当间谍，仍正被FBI调查中。
 <br/>
 <br/>
 据《大纪元》此前报导，数月前，安德森癌症中心有四名华人教授，因为牵扯中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞、〝长江计划学者〞被FBI调查。这些人都是正教授，自己有实验室。其中一人被抓，一人被撤销系主任职务，两人被要求辞职。
 <br/>
 <br/>
 其中被抓的教授就是谢克平。他不仅涉及到千人计划，而且涉及到洩密问题。他太太也在安德森癌症中心工作，夫妻俩一起被抓。后来鉴于人道主义考虑，警方让他太太回家照顾孩子，但是谢某一直被拘押。
 <br/>
 <br/>
 55岁的谢克平是〝千人计划〞学者之一，2016年曾列名上海同济大学东方医院的骨干领导人才。
 <br/>
 <br/>
 NBC报导指出，谢克平是今年1月因为〝窜改政府文件〞(tampering with GovernmentRecord)被调查。警方取得谢的住家和办公室搜索令，取走数十个设备与档案，其中两个有儿童色情图片。
 <br/>
 <br/>
 今年4月，谢被解职。
 <br/>
 <br/>
 此前，美方曾以间谍罪拘捕多名华裔科学家，这些人都名列〝千人计划〞名单。网友戏称，中共的〝千人计划〞随时变〝入狱计划〞。
 <br/>
 <br/>
 9月时，有网络消息传出，FBI已将〝千人计划〞学者列为调查重点，并按名单〝一个一个查〞。
 <br/>
 <br/>
 另有美国研究机构传出在FBI造访后开除〝千人计划〞学者，也有相关学者被要求向学校与工作单位提出自清。
 <br/>
 <br/>
 所谓〝千人计划〞，是2008年起中共多部门联合发起的招揽人才计划，目标是海外华裔尖端人才。至今已有近8000名海外专家入选该计划，多数为华裔且来自美国。
 <br/>
 <br/>
 评论人士指，千人计划本意就是目标明确地从美国盗取知识情报。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
