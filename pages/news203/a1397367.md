### 川普拟签行政令 废止美国公民出生地原则
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周一（10月29日）表示，他计划签署
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政令.html" target="_blank">
  行政令
 </a>
 ，以结束
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国公民.html" target="_blank">
  美国公民
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出生地原则.html" target="_blank">
  出生地原则
 </a>
 （birthright citizenship，也称〝出生地国籍〞）。这一动议是对美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 政策的重大改革，但预计会引发法律纠纷。
 <br/>
 <br/>
 美国在国籍问题上奉行出生地制，即在美国领土上出生的孩子（外国外交官的孩子除外）自动拥有美国国籍。该项法律的实施由1868年制定的宪法第14修正案保障。
 <br/>
 <br/>
 据美国Axios新闻网站10月30日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周一在接受〝Axios on HBO〞专访时表示，他准备废止非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国公民.html" target="_blank">
  美国公民
 </a>
 和非法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 在美国出生的婴儿的公民权，以遏制包括〝链式移民〞在内的移民泛滥。
 <br/>
 <br/>
 川普指出，他之所以选择采用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政令.html" target="_blank">
  行政令
 </a>
 方式而非国会制定宪法修正案的方式，是因为在目前民主党阻挠移民改革的背景下，国会很难通过法案。
 <br/>
 <br/>
 世界各国所奉行的国籍制度共有两种，一种是血统制，一种是出生地制。美国属于后一种情况。
 <br/>
 <br/>
 除了美洲，其它国家基本上都采用血统制。川普曾批评出生地制〝荒唐〞，并倡议〝它必须终止〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出生地原则.html" target="_blank">
  出生地原则
 </a>
 给美国带来非常复杂的移民难题，因为非法移民在美国出生的孩子都具有美国公民身份。
 <br/>
 <br/>
 川普早在2015年竞选总统初期就提出过废止宪法第14修正案，这也是他的强硬移民主张的一部分。不过他在距中期选举还有一周时再次提出这一动议，必然会引发巨大争议。
 <br/>
 <br/>
 如果川普避开国会立法转而采用颁布行政令方式，预计会面临法律挑战，而且他是否能赢得胜利也未可知。
 <br/>
 <br/>
 目前，面临至少7000洪都拉斯移民涌往美国边境，川普正在积极寻求有效的解决措施。周一下午，五角大楼宣布，将派大约5200名军人前往美墨边境支援边境巡逻人员和海关和边境保护局（CBP），拦截企图非法入境的大篷车移民并处理相关问题。
 <br/>
 <br/>
 保守派媒体《华盛顿观察家报》（The Washington Examiner）10月9日报导，根据非营利组织移民研究中心（Center for Immigration Studies）新发布的一份报告，仅2014年，在美国出生的非法移民的孩子就高达297,000人，其生育费耗费了美国纳税人24亿美元的资金。令人震惊的是，这一数字居然比参议院今年批给川普筑边境墙的费用高出8亿美元。
 <br/>
 <br/>
 该报告称，如果把非法移民分娩可能花费的医疗费用和她们非法或合法生育的子女的早期护理费用加在一起，共计高达53亿美元。这笔钱相当于川普所要的筑墙总资金的1/5。
 <br/>
 <br/>
 〝Axios on HBO〞对川普的采访视频将于美东时间周日晚上6:30在HBO电视网上播放。
