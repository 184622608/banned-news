### 美国301最新报告：中共未作根本改变
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】周二，美国贸易代表办公室发布了最新的301调查报告，其中明确指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的贸易不公行为没有从根本上改变。由于报告发布时间正值
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 前夕，引发外界关注。
 <br/>
 <br/>
 美国贸易谈判代表莱特希泽用一句话总结了调查报告的内容。莱特希泽说：〝这份更新的报告显示，中国没有从根本上改变其不公平、不合理以及扭曲市场的行为〞。
 <br/>
 <br/>
 周二发布的报告共53页，分为六个章节。报告第二章指出，〝中国继续未经允许，进入美国商业网络，并用黑客手段盗窃知识产权和敏感商业信息〞，甚至频率还有所增加，手段也更加复杂。
 <br/>
 <br/>
 报告第三章到第五章，详细介绍了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 强迫技术转让、对外企的歧视性许可限制、以及在外国收购投资的情况。
 <br/>
 <br/>
 由于报告发表时机敏感，正值
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和习近平会晤前夕。分析认为，这再次呼应了美国副总统彭斯几天前在APEC峰会上的讲话，警告中共。
 <br/>
 <br/>
 美国副总统 彭斯：〝美国不会改变做法，直到中方改变其行为方式。〞
 <br/>
 <br/>
 上周，川普在白宫透露，中方向美方发送了一份解决贸易摩擦的改革措施清单，但川普表示清单上漏掉了四、五件重大事项。
 <br/>
 <br/>
 美国总统 川普：〝他们发来一份他们愿意做的事情的清单，一个很长的清单，但我还是不能接受。〞
 <br/>
 <br/>
 周二，川普再次强调，如果中方不能达成协议，还将对另外价值2500亿美元中国商品征收
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 。
 <br/>
 <br/>
 美国总统川普：〝我还有价值2500亿美元的关税，如果我们不能达成协议。相信我，我会征税的，因为中国（中共）一直宰我们国家，很多很多年了。他们在我这儿宰不了了。〞
 <br/>
 <br/>
 川普和习近平将于月底在阿根廷的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上会面。白宫首席经济顾问库德洛周二已经表示，他认为这次峰会是解决中美贸易纠纷的关键时刻。
 <br/>
 <br/>
 新唐人记者林宇、天羽综合报导
</div>
