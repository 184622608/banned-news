### 美炸弹包裹案取得重大进展 1名嫌犯被捕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】周五（10月26日），美国佛罗里达警方逮捕一名与本周系列〝炸弹邮包〞案相关的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌犯.html" target="_blank">
  嫌犯
 </a>
 。通过对包裹上的生物资讯进行分析，并借助先进的邮政系统设备，调查工作已取得重大进展。
 <br/>
 <br/>
 周五上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 警方确认，本周多名民主党政要和幕僚收到含爆炸装置的邮包威胁，这些包裹大多从佛州寄出，目前警方已逮捕一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌犯.html" target="_blank">
  嫌犯
 </a>
 。
 <br/>
 <br/>
 其中一些包裹是通过美国邮局（USPS）寄出的，这为执法人员展开调查提供了重要线索。
 <br/>
 <br/>
 美国邮局使用一种创新的成像系统，可以拍摄全国各地所处理的每一封邮件。
 <br/>
 <br/>
 有几个包裹寄自佛州奥帕洛卡（Opa-locka）市的一家邮局。联邦调查局（FBI）称，这些邮件都选用非常结实的内含泡沫包装的信封，以防止信封内的东西被弄毁。信封上贴的地址都是用列印机打出来的，而非手写。
 <br/>
 <br/>
 官方人士告诉福克斯新闻（Fox News），位于弗吉尼亚州匡提科（Quantico）的FBI实验室正分析信封上的邮票，并且已取得重大进展。
 <br/>
 <br/>
 FBI对信封和里面的邮寄物也都进行了细致分析，里面很可能包含了非常有价值的DNA资讯。调查人员和炸弹专家说，一些微小的生物物质——汗液、皮肤细胞、唾液、头发、指纹等，通常被用作查找嫌疑人的路径。
 <br/>
 <br/>
 目前，在华盛顿、FBI总部及其地方分支机构，共有24个团队在参与调查该事件。
 <br/>
 <br/>
 警方将在美东时间下午2:30召开新闻发布会，通报案件进展情况。
 <br/>
 <br/>
 从周一开始，已有数名民主党政要或自由派显要人士收到含爆炸装置的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_可疑包裹.html" target="_blank">
  可疑包裹
 </a>
 ，其中包括前总统奥巴马（Barack Obama）、前国务卿希拉里（Hillary Clinton）、民主党众议员沃特斯（Maxine Waters）、前司法部长霍尔德（Eric Holder）、前中情局（CIA）局长布伦南（John Brennan）、亿万富翁索罗斯（George Soros）、前美国副总统拜登（Joe Biden）和好莱坞明星罗伯特‧德尼罗（Robert De Niro）等。
 <br/>
 <br/>
 执法部门披露，周五上午，前国家情报总监（DNI）克拉珀（James Clapper）新泽西州民主党参议员布克（Cory Booker）也受到了含爆炸装置的包裹。目前发现的可疑包裹已达13个。
 <br/>
 <br/>
 第13个〝炸弹邮包〞是准备寄给加州参议员哈里斯（Kamala Harris）的，但周五被截获。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
