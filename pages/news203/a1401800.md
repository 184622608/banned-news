### 川普： 中国同意取消对美汽车加征关税
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】〝川习会〞，双方同意贸易战休兵90天，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周一推文表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 已同意，将削减并移除对美国汽车加征的关税，此举将大幅缓解美国汽车制造业蒙受的损失。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周日晚间推文：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 已同意削减和移除对美国进口车加征的关税，目前课征40%的关税。〞
 <br/>
 <br/>
 中国今年5月宣布汽车进口关税从25%调降至15%，不过美中贸易战开打后，7月份又将美国进口车关税调升至40%。
 <br/>
 <br/>
 现在，中国同意移除25%的报复性关税，再削减原本的进口关税，将大幅缓解美国汽车制造业蒙受的损失，特斯拉、美国制宝马汽车与福特林肯汽车都将受惠。
 <br/>
 <br/>
 这是〝川习会〞后，中国除了立即购买美国农产品之外，再次对美国让步。
 <br/>
 <br/>
 中方还承诺立即就美方所关切的中共强制技术转让、知识产权保护、非关税壁垒、及网络盗窃进行结构性改革谈判。
 <br/>
 <br/>
 美方承诺对中国进口到美国的2,000亿美元商品，暂停调高关税，但前提是双方必须在90天内完成协议内容。
 <br/>
 <br/>
 全球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融经济学家.html" target="_blank">
  金融经济学家
 </a>
 彼得．狄克森表示，〝川习会〞后，美中贸易战获得某种程度的缓解，但是美国并没有退缩的意思。
 <br/>
 <br/>
 全球金融经济学家彼得．狄克森：〝美国可能不愿退缩，并准备对另外价值2,000亿美元的货物征收关税。因此，即使基本上双方达到预期的停火，市场可以乐观起来，但明显的并没有消除所有问题。〞
 <br/>
 <br/>
 彼得．狄克森认为，除非中国不再强制外国企业技术转让并重视保护知识产权，否则贸易战仍会升级。
 <br/>
 <br/>
 全球金融经济学家彼得．狄克森：〝如果我们看不到中方改变这方面的行为，那可以预期贸易问题会再次爆发。〞
 <br/>
 <br/>
 新唐人记者李梅、田瑞综合报导
</div>
