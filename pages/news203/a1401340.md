### 美内政部长访加州 称野火致数十亿损失
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 火灾在上周末得到完全遏制。日前，美国内政部长和农业部长视察灾区，同当地官员商议赈灾事宜。
 <br/>
 <br/>
 北
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火.html" target="_blank">
  野火
 </a>
 摧毁了天堂镇及周边社区的近14,000栋房屋，至少88人死亡。
 <br/>
 <br/>
 本周，美国内政部长辛克与农业部长珀杜共同探访了灾区。辛克说从未亲眼见过这样的灾难。
 <br/>
 <br/>
 美国内政部长Ryan Zinke：〝让我们一起努力确保火灾不再发生。我们来这里显然是因为这是总统的首要任务，让我们解决它。〞
 <br/>
 <br/>
 辛克指出要减轻火灾损失，及早治愈被烧过的树木，而不让百万英亩的树木患病或死亡。据估计，仅北
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州野火.html" target="_blank">
  加州野火
 </a>
 ，损失就可达数以十亿计美元。
 <br/>
 <br/>
 美国农业部长珀杜 Sonny Perdu：〝我们可以做到对森林更好的管理，以防止和杜绝当前这种破坏。〞
 <br/>
 <br/>
 截至本周一，美国政府已向北加州灾区提供超过2000万美元的援助。联邦紧急事务管理署（FEMA）已拨款1550万美元用于住房援助，包括酒店房间代金券。并已发出500万美元赈灾，帮助约17,000注册申请者。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
