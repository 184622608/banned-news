### 中美贸战〝停火〞90天 刘鹤将率30人赴美谈判
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】中美两国元首在阿根廷达成共识，双方暂停加征关税，给北京90天缓冲谈判时间。接下来将重启中美高层
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易谈判.html" target="_blank">
  贸易谈判
 </a>
 。有美媒披露，中共国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 ，在数周内将率领一个30人的团队赴美进行新一轮磋商。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）与中国国家主席习近平，在阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 间隙的工作晚宴上，就持续数个月的贸易战，达成〝停战90天的〞共识。
 <br/>
 <br/>
 中共官媒2日报导，中美两国元首会晤达成共识，同意停止相互加征新的关税，双方就如何妥善解决存在的分歧和问题提出了一系列建设性方案，中方愿意根据国内市场和人民的需要扩大进口，包括从美国购买农产品和燃料，逐步缓解贸易不平衡问题。
 <br/>
 <br/>
 12月4日，新媒《南洋视界》称，中美两国达成贸易战停火90天的协议，接下来将重启中美高层
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易谈判.html" target="_blank">
  贸易谈判
 </a>
 。
 <br/>
 <br/>
 报导称，习近平11月底离开北京，出访时的团队名单中没有中共国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 ，但刘鹤在G20〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞意外现身。有消息称，〝川习会〞确定〝休战〞，刘鹤可能再赴美谈判，时间或许在12月12日至15日。
 <br/>
 <br/>
 今年以来，刘鹤就贸易战问题已经两次访美，与美方进行了三轮谈判均无果。
 <br/>
 <br/>
 《华尔街日报》报导称，此次刘鹤将继续担任中方最高贸易谈判代表，在数周内率领一个30人的团队赴美就贸易进行新一轮磋商。
 <br/>
 <br/>
 此前消息人士透露，刘鹤对贸易谈判缺乏进展以及美国不断施压感到沮丧。刘鹤认为，贸易战中共不能打，要打必输，前景不堪设想，会引发中国经济总崩溃。
 <br/>
 <br/>
 自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 开打以来，美国已经宣布对总值2500亿美元中国商品征收关税。G20〝习川会〞后，根据白宫发表的声明，川普同意在2019年1月1日，不把价值2000亿美元中国产品的关税税率从10%提高到25%。
 <br/>
 <br/>
 而北京同意购买大量的美国产品，除了立即购买的农产品，其他还包括仍未商定但数量非常大的能源、工业和其它产品等。
 <br/>
 <br/>
 如果双方未能在90天内达成协议，美国将把10%的关税提高至25%。
 <br/>
 <br/>
 美国荣鼎集团的中国宏观经济分析师朱鸣岐向BBC表示，中国购买大批美国商品，是中方示好的信号，尤其是北京承诺马上重启进口美国农产品，让川普能对农业州有所交代。
 <br/>
 <br/>
 不过，朱鸣岐认为，这个以购买美货换关税暂缓的协议，只是治标不治本，没有提供平衡中美贸易差距的持续动力。中美经济的基本面维持不变，贸易逆差预料将长期持续。而且两国在90天内要谈判的话题，并未涉及中方坚持不让步的国企、产业政策改革等议题。
 <br/>
 <br/>
 他说，就算90天内两国能就知识产权保护、网路盗窃等达成共识，也不能排除华府未来要求北京在更深层的结构性问题上做出让步。
 <br/>
 <br/>
 美国白宫贸易顾问纳瓦罗（Peter Navarro）在接受美国国家公共电台（NPR）采访时表示，美国贸易代表莱特希泽，将在90天时间内与中方进行贸易谈判。
 <br/>
 <br/>
 纳瓦罗说：莱特希泽是美国贸易代表办公室有史以来最强硬的谈判人。他将按照章节原原本本地降低关税，减少非关税壁垒，并结束妨碍市场准入的所有这些结构性做法。
 <br/>
 <br/>
 《华尔街日报》引述熟悉决定的消息人士说，这一个决定，让中方惊讶，因为他们一直主要是和美财长努钦打交道。 努钦同金融市场关系密切，他一直在努力寻求解决方案。但中美前四轮贸易谈判均无果。
 <br/>
 <br/>
 而莱特希泽则一直施压对中方征收更多的关税，让北京做出改变，以解决两国产生分歧的根本性问题。这位消息人士说，选择莱特希泽给了川普更多信心，美国将对北京采取更加强硬的路线。
 <br/>
 <br/>
 莱特希泽说，如果中国在90天后未能提出令人满意的条件，那么关税税率将再次上涨25%。
 <br/>
 <br/>
 这意味着对中方来说，虽然获得了90天的〝缓刑〞时间，但是仍然是祸福难料，因为美国方面换上了一位更加强硬的官员负责这90天的谈判。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
