### 川习再通话 北京左右摇摆 华府两手准备
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】观众朋友大家好，欢迎收看新唐人全球新闻。周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 时隔半年后首次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通话.html" target="_blank">
  通话
 </a>
 。周五，川普重申，美中必须达成对各方都公平的协议，否则，可能将对更多中国商品加征
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 。
 <br/>
 <br/>
 中共官媒称，11月1日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 应约与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通话.html" target="_blank">
  通话
 </a>
 。双方都表示，期待11月底在阿根廷举办的G20峰会上见面；并同意双方经济团队展开实质性的磋商。
 <br/>
 <br/>
 俞伟雄：〝不管台面上讲谁主动，我觉得不重要，我觉得重点是，现在中国有这个急迫性，要来跟美国好好地谈。〞
 <br/>
 <br/>
 在川习通话前，中共政治局开会，罕见承认经济下行压力扩大。
 <br/>
 <br/>
 10月份，中国GDP增速、制造业PMI、股市汇市等，都一片惨淡。在华南设厂的美国企业，已有超过70%考虑减缓投资或撤出中国。
 <br/>
 <br/>
 川普：〝昨天我和习主席对话，他们非常想达成协议。他们经济非常非常糟糕，我想说，我们要达成一个对美国有益的协议，同样也对他们有益。〞
 <br/>
 <br/>
 美中可能重启正式经贸会谈，白宫首席经济顾问库德洛重申，美方立场坚定，一定要让中方在知识产权窃取、强制技术转让、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 及网络安全等方面，拿出满意方案。
 <br/>
 <br/>
 俞伟雄：〝川普总统的想法是非常坚定的，是非常符合先进国家的市场原则的，WTO这个原则的。我真的是希望，中国这是一个历史的契机，化危机为转机，基本上可以100%的接受美方的要求都不是吃亏的事情。因为我觉得那些都是很正确、很正当的要求。〞
 <br/>
 <br/>
 不过外界观察到，北京不断释放矛盾信息。例如10月中旬刚宣称，考虑对国企〝竞争中性〞，即不在信贷等方面给国企特殊优惠，被认为有意向市场化迈出一步。但10月底就又宣布，向一汽集团提供一万亿元的巨额授信，显示出政府仍强力主导产业计划。
 <br/>
 <br/>
 俞伟雄：〝那中国这边，我觉得有时候感觉它是进一步退两步。想要在过程中拖延，来赚取一些机会。〞
 <br/>
 <br/>
 北京左右摇摆，华盛顿则做好两手准备。如果谈判不成，就加征更多关税。
 <br/>
 <br/>
 川普：〝我们已对$2500亿中国商品加征25%关税，可能再对$2670亿商品加征25%或更多关税。我们成效显著。我相信，他们想达成协议。〞
 <br/>
 <br/>
 俞伟雄：〝现在有一个大限，就是明年1月1号开始，中国出口到美国的2000亿（美元）的商品，关税增加，要从10%到25%。很多厂商都在做准备，移厂，迁出等等各方面。再走下去，（北京）机会越来越小了。〞
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
