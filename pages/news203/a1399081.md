### 全美聚焦:如佛州不能按时结束计票 后果如何？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 选举上周六（11月10日）开始
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 ，但备受争议的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_棕榈滩.html" target="_blank">
  棕榈滩
 </a>
 县（Palm Beach）却表示，该县可能无法在截止日期前完成工作。目前，佛州重新计票已成为全美关注的焦点。
 <br/>
 <br/>
 据《华盛顿观察家报》（The Washington Examiner）12日报导，如果
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_棕榈滩.html" target="_blank">
  棕榈滩
 </a>
 县不能在周四（11月15日）下午3点的最后期限前完成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 很可能排除该县选票而进入最终计票程序。
 <br/>
 <br/>
 佛州州务卿戴兹特（Ken Detzner）周六宣布启动重新计票机制，佛州67个县须全部在截止日期前完成任务。但周日，棕榈滩县抗拒州务卿命令，声称该县的计票设备老旧，无法在周四的期限前完成重新计票，并要求戴兹特延长截止日期，但被其拒绝。
 <br/>
 <br/>
 棕榈滩县的声明使重新计票形势变得更加复杂，共和党人认为，该县选举部门在有意抵制共和党的胜利。
 <br/>
 <br/>
 根据佛州法律，如果棕榈滩县果真不能在截止日期前完成计票，他们将不得不使用上周六提交给州务卿的计票结果。
 <br/>
 <br/>
 棕榈滩县共和党主席巴内特（Michael Barnett）周日在接受美国有线电视新闻网（CNN）采访时表示，如果该县错过了最后期限，对共和党来说将会是个好消息，因为共和党候选人处于领先地位。
 <br/>
 <br/>
 不过虽然如此，棕榈滩县毕竟在选举日后突然多出15,000张选票，且多数流向民主党。这些被川普总统称之为〝不知从哪儿冒出来的选票〞严重削弱了共和党候选人的优势，并最终导致重新计票。
 <br/>
 <br/>
 涉及重新计票的两场重要竞选分别是民主党参议员候选人尼尔森（Bill Nelson）与现任州长、共和党参议员候选人斯科特（Rick Scott）之间的竞选，以及两位州长竞选人——民主党籍的塔拉哈西市（Tallahassee）市长吉卢姆（Andrew Gillum）与共和党联邦众议员德桑蒂斯（Ron Desantis）之间的竞选。
 <br/>
 <br/>
 另外，关于农业长官的竞选也要重新计票。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
