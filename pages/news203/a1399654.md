### 格拉斯利转管财政 参院司法委换掌门
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】周五（11月16日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法委.html" target="_blank">
  司法委
 </a>
 员会主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_格拉斯利.html" target="_blank">
  格拉斯利
 </a>
 （Chuck Grassley）宣布，他明年将就任参议院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财政.html" target="_blank">
  财政
 </a>
 委员会（Senate Finance Committee）主席一职。外界揣测，新的司法委主席将由南卡罗来纳州共和党参议员格雷厄姆（Lindsey Graham）出任。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_格拉斯利.html" target="_blank">
  格拉斯利
 </a>
 周五宣布了他将离开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法委.html" target="_blank">
  司法委
 </a>
 员会并就任新职的消息。他将接替即将退休的犹他州参议员哈奇（Orrin Hatch），担任下届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财政.html" target="_blank">
  财政
 </a>
 委员会主席。
 <br/>
 <br/>
 格拉斯利在担任司法委员会主席期间，主导了两名最高法院大法官的确认程序，同时也创下记录——在新总统的第一届国会批准了最多的巡回上诉法官提名人。
 <br/>
 <br/>
 格拉斯利曾两次担任参议院财政委员会主席，最近的一次是在2003年1月至2007年1月期间。他此番重回财政委员会，仍担任主席角色。
 <br/>
 <br/>
 格拉斯利卸任司法委员会主席之职后，接替他的很可能是现任参议院司法委员会成员、南卡罗来纳州共和党参议员格雷厄姆。格雷厄姆近日曾暗示，如果他主掌司法委，他将改变一些做法，包括将采取严厉措施、制止民主党阻挠总统提名人的做法。
 <br/>
 <br/>
 格雷厄姆称赞格拉斯利对司法委员会的领导很成功，称他是一名楷模、稳健地领导着司法委的工作。
 <br/>
 <br/>
 格雷厄姆是参议院的强硬保守派，在确认最高法院大法官卡瓦诺（Brett Kavanaugh）的艰难过程中，他一再激烈批评民主党发起的各种破坏和阻挠行动，并敦促党内同僚团结一致通过确认。
 <br/>
 <br/>
 格雷厄姆在中期选举前曾公开呼吁司法部长塞申斯（Jeff Sessions）在选举结束后下台。11月11日，他在接受哥伦比亚广播公司（CBS）《面向全国》（Face the Nation）节目采访时表示，他支持惠特克（Matthew Whitaker）担任代理司法部长，他认为川普总统的任命是适当的、合法的。
 <br/>
 <br/>
 面对民主党要求惠特克回避
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查的压力，格雷厄姆告诉《面向全国》的主持人，他认为惠特克无须回避监督穆勒（Robert Mueller）调查。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
