### 佛州空军基地遭飓风破坏 战机损毁
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】上周三登陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 西北部的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 ，不仅给当地民众带来巨大损失，也使座落在此的廷德尔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空军基地.html" target="_blank">
  空军基地
 </a>
 受到重创。周日，国防部公布了一段灾后视频。
 <br/>
 <br/>
 廷德尔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空军基地.html" target="_blank">
  空军基地
 </a>
 （Tyndall Air Force Base）座落在佛罗里达州西北部，巴拿马城外的墨西哥湾沿岸，首当其冲对上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 ，受到大规模损毁。
 <br/>
 <br/>
 这段美国国防部最新公布的视频，可以看到飞机库受到大部分损坏，军事基地的房屋多处坍塌，房屋玻璃被震碎，掀翻的汽车残骸仍在马路边，大量的树木倒塌、折损，现场如同战后废墟，一片狼藉、了无生气。
 <br/>
 <br/>
 空军基地在飓风登陆前撤离，周三（10月10日）下午，迈克尔以4级飓风登陆墨西哥湾沿岸，时速达每小时155英里。
 <br/>
 <br/>
 飓风造成多架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战机.html" target="_blank">
  战机
 </a>
 遭到损毁。据纽约时报消息，此前空军基地有55架F-22战机，其中至少33架提前转移到了俄州。这种战机的单个造价是$3.39亿。但官方并未透露其他22架F-22的去向，只表示有留下一部分战机。空军基地官员表示，此次基地遭到了〝灾难性〞的破坏。
 <br/>
 <br/>
 新唐人记者宇亭综合报导
