### 美中长期利益不相容 政体差异是关键
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】近期，川普政府对华政策的战略转向，引起华府政策圈对于美中未来关系的热议。前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国安会.html" target="_blank">
  国安会
 </a>
 亚洲事务主任麦艾文周二(10月30日)在华府辩论会上表示，美中之间的长期利益越来越不相容，其中政治体制差异是背后重要因素。
 <br/>
 <br/>
 前美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国安会.html" target="_blank">
  国安会
 </a>
 亚洲事务高级主任麦艾文：〝我们要阐明的是，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中利益.html" target="_blank">
  美中利益
 </a>
 在很大程度上越来越不相容。〞
 <br/>
 <br/>
 目前，美中之间无论在经贸、安全、科技及人权等各领域，都存在诸多竞争与分歧。不仅川普政府和共和党把中共视为最重要竞争对手，就连奥巴马时代的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 国安会官员都认为，美中之间长期利益越来越不相容，而背后重要因素就是政治体制差异。
 <br/>
 <br/>
 前美国国安会亚洲事务高级主任麦艾文：〝美国的政治体制被中共认为是天然的威胁。美国媒体和非政府机构，都在持续批评中共。美国政治领导人绝不会承认中共领导者是中国人民的合法代表。〞
 <br/>
 <br/>
 意识形态一直在美中竞争中扮演重要角色。专家认为这是核心问题，也反映在双方对国际秩序的看法根本对立。
 <br/>
 <br/>
 布鲁金斯学会美欧中心主任 Thomas Wright：〝中共担心，自由社会体制会颠覆中共 ；美国则担忧，如中共令世界变得对极权政权更安全，就会影响自由的世界秩序。〞
 <br/>
 <br/>
 目前在半导体、人工智能、5G等高技术领域，美中之间竞争日益激烈。针对中共窃取美国技术与知识产权，川普政府已采取关税等严厉手段回应。专家认为，美中双方谁能赢得这一竞赛，会导致根本不同的结果。
 <br/>
 <br/>
 布鲁金斯学会美欧中心主任 Thomas Wright：〝如果他们(中共)精通这些技术并出口，我们会看到自由社会的持续倒退。〞
 <br/>
 <br/>
 过去多年来，美国政府对中共都是采取〝接触〞战略。麦艾文认为，片面强调合作，淡化分歧的做法，只是导致中共更自大、并采取更强势、更冒险的作为。
 <br/>
 <br/>
 前美国国安会亚洲事务高级主任麦艾文：〝如果向中共压力低头，就会受到中共更大压力。〞
 <br/>
 <br/>
 作为民主党政策智囊，麦艾文的态度也反映出，美国两党对于中共政权已有了日益清楚的认识与共识。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
