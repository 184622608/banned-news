### 美两党中期选举平分秋色 中共沦最大输家
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 尘埃落定，共和党保住参议院，民主党则赢得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 多数党地位。港台媒体指，选举结果应该令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）比较满意，但对中共非常不利。
 <br/>
 <br/>
 香港《经济日报》11月8日评论文章指，民主党夺回
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 ，会对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 施政有所牵制，但这并不适用于川普对中共的政策，因为遏制中共已是美国政界的共同立场。而且，民主党的本质及立场决定了如果其掌控众议院，其实更不利于中共。
 <br/>
 <br/>
 文章引述美国智库的观点指，相对于共和党，民主党更不倾向于自由贸易，更反对不公平的贸易关系。
 <br/>
 <br/>
 另外，自从美国和中共建交后，向来都是共和党政府对北京比较温和。而民主党政府在传统上更注重针砭中国的人权问题，至今依然如故。
 <br/>
 <br/>
 随着民主党重夺众议院，民主党领袖佩洛西（Nancy Pelosi）很可能成为新的众议院议长。她长年谴责中共侵犯人权，北京肯定不乐见她主掌众议院。
 <br/>
 <br/>
 文章指，民主党重夺众议院，很可能施压白宫对中共更强硬，甚至会督促川普政府把人权问题与贸易问题捆绑在一起。中共未来将同时面对反共情绪高涨的国会，以及鹰派云集的白宫，陷入双线作战的困境。
 <br/>
 <br/>
 而《端传媒》7日文章称，民主党在选前宣称将掀起政坛〝蓝潮〞，结果只拿回众议院，两党都没有获得决定性胜利。川普没有输，而中共更没有赢。
 <br/>
 <br/>
 文章说，美国执政党在中期选举中历来处于天然劣势，最近80年的16次中期选举中，时任总统所在政党只有两次增加了众议院席位。因此，民主党夺回众议院本是意料中事，在这次中期选举考试中，民主党仅仅达到了及格水平。
 <br/>
 <br/>
 《苹果日报》8日文章分析，民主党在开票初期就输掉参议院的多数议席，众议院虽然算是取回多数，但更象是中期选举对执政党不满的例行反弹。共和党仍然在众议院保留200个以上的席位，未来拿回多数党地位并不难。
 <br/>
 <br/>
 文章认为，共和党顶住民主党攻势，令中期选举没有出现一面倒的结果，这说明川普的政治本钱比一般人估计的要雄厚。对共和党而言，川普路线成为吸引选票的护身符，令该党不致一败涂地，这决定了未来两年里，共和党将不会再拖川普的后腿，而是更积极的配合他的路线。川普将有更大空间和实力来打他的〝美国优先〞、〝美国再强大〞牌。
 <br/>
 <br/>
 而民主党直到目前还没有能力动员更多人离开川普阵营，中期选举中也没有出现足以挑战川普的新锐，2020年大选不容乐观。民主党不但难以夺回总统宝座，还大有可能再次失去众议院。
 <br/>
 <br/>
 台湾联合新闻网7日文章则指，这次中期选举的结果，客观上更有利于川普连任。
 <br/>
 <br/>
 文章提到，两党分治的政府，更容易让选民接受。而且，有了共和党1990年代末弹劾克林顿的前车之鉴，民主党基层如果施压众议院弹劾川普，无论是否付诸行动，其造成的紧张纷扰都对川普有利。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
