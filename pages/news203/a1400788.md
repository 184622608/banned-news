### 〝川习会〞高兴太早？四五件大事或已定性
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】自11月初川习通电话后，坊间一直对习近平与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）将于11月底在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 时会面，及可能达成贸易协议持乐观态度。然而最近一系列事态，显示出双方的不和谐，其中包括四五件大事不被美方所接受。中美能否达成协议，出现巨大不确定性。
 <br/>
 <br/>
 20国集团峰会（G20）将于11月30日至12月1日在阿根廷首都布宜诺斯艾利斯召开，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和习近平在峰会期间的会晤，将是美中贸易的关键时刻。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞能否在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间举行，能否达成休战协议，目前均是悬念
 <br/>
 <br/>
 11月24日，《美国之音》报导，〝川习会〞可能会因为《中国制造2025》问题而破裂，川普希望中国放弃这项野心勃勃的计划，但习近平是不会同意的。
 <br/>
 <br/>
 《中国制造2025》是中共企图主导未来科技的一项野心计划，是阻碍解决美中贸易紧张局势的障碍之一。中共要从西方获得技术，真正的目标是占领西方，特别是占领美国市场，击垮美国的高科技产业的野心计划。
 <br/>
 <br/>
 为了给〝川习会〞铺路，中共在口头上承诺撤下〝中国制造2025〞计划，以安抚川普政府，但外界认为，中共不可能从根本上放弃2025计划。
 <br/>
 <br/>
 11月16日，川普在白宫记者会上披露，中共近期向美国提交了一份贸易改革清单，〝那是一份很大的清单〞。川普认为这是好事，但现在他不能接受，因为清单〝漏掉了四五件大事〞。
 <br/>
 <br/>
 对中共清单中漏掉的四五件大事，有分析人士称，美方的要求包括：关税、知识产权、技术转让和政府补贴等内容，同时还包括〝一带一路〞、南海问题、〝中国制造2025〞以及改变中国发展方式等。这些中共不可能全盘接受。
 <br/>
 <br/>
 据《路透社》引述一名美国官员透露，中共向美国递交的清单有142个项目，但不包括白宫的核心要求，例如解决强迫技术转移和偷窃知识产权、〝中国制造2025〞等问题。
 <br/>
 <br/>
 一名消息人士透露，中方给出的并不是什么新的东西。目前，双方仍处在僵局之中，仍有很多工作要做。
 <br/>
 <br/>
 《南华早报》报导称，在这种氛围之下，中方经贸负责人，国务院副总理刘鹤，原定赴华盛顿进行的新一轮贸易谈判，现在搬到了布宜诺斯艾利斯，由川普与习近平定夺。
 <br/>
 <br/>
 此外，11月17日在巴布亚新几内亚举行的APEC峰会上，习近平与美国副总统彭斯（Mike Pence）在各自的演讲中针对中美贸易争端展开激烈交锋。
 <br/>
 <br/>
 彭斯直接点名批评中共设置的贸易壁垒史无前例，并称中共的〝一带一路〞是给合作国家设立的债务陷阱。在中共改变行为方式前，美国不会改变征收关税的政策。
 <br/>
 <br/>
 彭斯还说，11月底在阿根廷举行的〝川习会〞将讨论5大议题，包括中共不正当贸易行为的关税壁垒、强制要求的技术转移、侵犯知识产权、航行的自由和人权问题。
 <br/>
 <br/>
 这意味美中能否达成协议并实现最终和解，已经出现巨大不确定性。
 <br/>
 <br/>
 《卫报》说，将世界两大经济体领导人的G20对决称为〝世纪大战〞一点都不离谱。习近平和川普能否解决他们的分歧将产生重大影响，或将让世界带入一个新的不确定时代。一些分析人士则警告，将会开启一场新的冷战。
 <br/>
 <br/>
 彭斯在新加坡接受专访时警告中共，必须全面改变其行为作法，才能避免与美国发生新冷战。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
