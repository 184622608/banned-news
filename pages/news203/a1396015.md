### 巡航菲律宾海  美航母直升机坠甲板数人受伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】美国军方周五（19日）发布声明称，海军一架直升机当天坠落在里根号航空母舰甲板上，造成数人受伤。事发时，里根号正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾海.html" target="_blank">
  菲律宾海
 </a>
 执行巡逻任务。
 <br/>
 <br/>
 中央社报导，这架MH-60
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海鹰直升机.html" target="_blank">
  海鹰直升机
 </a>
 （MH-60 Seahawk）周五上午起飞后不久坠落。第七舰队在声明中说：〝所有受伤人员状况稳定，里根号（USS Ronald Reagan）的医务人员正进行评估。虽然一些人员将被医疗后送上岸，但伤者都无生命危险。〞声明中没有说明受伤人数及相关细节。
 <br/>
 <br/>
 声明说，事发原因正在调查中。
 <br/>
 <br/>
 核动力航母里根号在太平洋各地执行巡逻任务。意外发生时，里根号正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾海.html" target="_blank">
  菲律宾海
 </a>
 （Philippine Sea）执行〝例行任务〞。军方称，里根号依然具备〝全面执行任务的能力〞。
 <br/>
 <br/>
 这艘载有70多架飞机、重达97,000吨的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_里根号航母.html" target="_blank">
  里根号航母
 </a>
 6月26日在菲律宾马尼拉湾停靠。是今年以来，美国军方在该海域部署的第三艘航空母舰。
 <br/>
 <br/>
 里根号装备有北约海麻雀导弹、滚动式机身导弹以及各种枪枝和电子战设备。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
