### 洛杉矶法官驳回艳星指控川普案
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】美国加州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洛杉矶法官.html" target="_blank">
  洛杉矶法官
 </a>
 周一驳回了成人电影演员丹尼尔斯对川普总统的指控，裁决说，川普的推文受美国宪法第一修正案言论自由的保护。
 <br/>
 <br/>
 丹尼尔斯是美国成人电影演员，真名叫斯蒂芬妮．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克利福德.html" target="_blank">
  克利福德
 </a>
 。她声称2006年曾和川普发生关系，2011年，她准备在一次采访中讲述这段往事时，一名男子在拉斯维加斯的一个停车场威胁她，让她不要提这件事。克利福德还公布了男子的肖像素描。
 <br/>
 <br/>
 川普在推文中回应，克利福德的指控是骗局。
 <br/>
 <br/>
 推文说：〝多年前一个不存在的男子肖像素描，全是骗局，把假新闻媒体当傻子耍弄（但是他们都知道）！〞
 <br/>
 <br/>
 丹尼尔斯随后控告川普的推文构成诽谤罪，要求川普支付13万美元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_封口费.html" target="_blank">
  封口费
 </a>
 。
 <br/>
 <br/>
 法官奥特罗驳回了丹尼尔斯的指控，指出川普的推文受第一修正案言论自由的保护。
 <br/>
 <br/>
 判决书说，〝法庭同意川普的论证，被控的推文只是‘修辞夸张’，通常是与美国政治和公共言论有关。第一修正案保护这种修辞言论。〞
 <br/>
 <br/>
 法官还判丹尼尔斯支付川普的律师费。丹尼尔斯的律师表示会继续上诉。
 <br/>
 <br/>
 新唐人记者安心、张琪综合报导
</div>
