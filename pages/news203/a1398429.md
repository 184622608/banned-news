### 美国中期选举 各地选民踊跃投票
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】据统计，周二的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选民.html" target="_blank">
  选民
 </a>
 参与度增加，亚太裔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 热情高涨。民调显示，高达76%的亚太裔选民表示要参加投票。来看当天美国各地的，选民投票情况。
 <br/>
 <br/>
 华府亚太裔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 联盟表示，美国亚太裔人口目前已超过2100万，成为竞选中两党争取的力量。
 <br/>
 <br/>
 亚太裔投票联盟执行理事陈棣芳：〝我们看到更多的志工和组织，在中期选举前，协助
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选民.html" target="_blank">
  选民
 </a>
 注册和鼓动选民出来投票。总的来看，我们接到很多热线电话，寻问基本信息，如投票地点在哪里，投票时间等。〞
 <br/>
 <br/>
 周二，在纽约华埠最大票仓、孔子大厦投票站，投票人络绎不绝。
 <br/>
 <br/>
 孔子大厦投票站协调人郑永佳：〝我很忙，你可以看到我很忙。通常没有这么忙的。来投票的人数很高，中午12点就有300人投票，我预计今天会有1000人来投票。我想他们是来表达自己的见解，所以他们出来投票。〞
 <br/>
 <br/>
 纽约选民韦晶晶：〝今天是投了共和党州长（参选人）莫里纳罗的票，还有投了法雷（的票）。〞
 <br/>
 <br/>
 在加州旧金山硅谷圣塔克拉拉县，选民登记人数创历史新高。
 <br/>
 <br/>
 圣塔克拉拉县选民登记处主任 Shannon Bushey：〝圣塔克拉拉县的选民注册数量破纪录，超过88.7万人登记成选民。〞〝这次选举我们达到的另一个历史新高是提早投票的人数，到昨天为止共收到25.7万多个邮寄选票。〞
 <br/>
 <br/>
 美国出生的冯博洋，父母是中国大陆移民。他赞赏川普总统对内减税、对外推动朝鲜弃核等政策。认为在加州左倾政治风气下，没有给川普一个正确评价。
 <br/>
 <br/>
 选民冯博洋：〝我在这里我知道，这是个很民主党的地方，我也看到，如果你（的想法）是毛泽东有一点右边，好多人都认为你是纳粹，还是法西斯，还是一个非常不好的人。〞
 <br/>
 <br/>
 在加州洛杉矶，选民早早赶到投票站投票。
 <br/>
 <br/>
 退休国会首席谈判代表 C.G. Frienden：〝很开心看到经济在回升，虽然我不应该太强调政治，但我真的十分开心我们有川普这样的总统。〞〝我认为他真的很在乎美国，他在帮助美国的经济和人民的福祉。〞
 <br/>
 <br/>
 陈先生经商多年，很高兴看到美国逐渐回归传统和普世价值。
 <br/>
 <br/>
 商人陈怡：〝我从原来的民主党转移到共和党，而且因为加州共和党的力量比较弱，所以我觉得我这一票还是很重要的，所以我就赶来投票。〞
 <br/>
 <br/>
 周三，选举结果初步揭晓，共和党在参议院的优势进一步扩大，民主党掌控众议院。
 <br/>
 <br/>
 新唐人记者站采访报导
</div>
