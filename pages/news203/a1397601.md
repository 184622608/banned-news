### 〝大篷车〞非法移民压境 川普拟建〝帐篷城〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 继续向美国边境推进。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）日前对媒体表示，他计划在美国南部边境打造全新的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_帐篷.html" target="_blank">
  帐篷
 </a>
 城市〞，安置那些被拘留等待庇护的移民。
 <br/>
 <br/>
 为阻止大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 入境美国，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 29日曾表示，他计划在本周末前向美墨边境部署5200名士兵。他周三（10月31日）离开白宫准备参加佛罗里达州的集会时告诉记者，美国可以向墨西哥边境派遣1万5000名军人。川普警告非法移民请回去，称美国不会接受他们的非法过境。
 <br/>
 <br/>
 10月29日，福克斯新闻的Laura Ingraham询问总统，如果移民向当局投降并要求庇护，他会做什么，这是常见的。
 <br/>
 <br/>
 川普称，如果〝大篷车〞移民跨过边境，就会被逮捕。而如果他们申请庇护，〝我们将把他们关押，直到受审为止。〞
 <br/>
 <br/>
 川普说，他会下令建造一个〝非常好〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_帐篷.html" target="_blank">
  帐篷
 </a>
 城市，〝我们将在各处设置帐篷。他们要等待，如果他们没有得到庇护，他们就得离开。〞但他强调，不会花费〝数亿美元〞建造房屋来收容任何等待庇护期间被拘留的人。
 <br/>
 <br/>
 近日，数千名来自洪都拉斯、危地马拉和萨尔瓦多等中美洲国家民众穿越边界进入墨西哥，计划穿越墨西哥北上前往美国。也因他们中很多人乘坐大篷车前进，所以被称为是〝大篷车〞移民。
 <br/>
 <br/>
 大篷车难民目前在南墨西哥，据报已从7000人减少到了4000人。他们还在用各种方式向美国行进，有的乘坐卡车，有的步行。周三，另一批约2000人的大篷车队伍从中美洲国家萨尔瓦多出发，准备前往美国。
 <br/>
 <br/>
 更为严重的是，10月28日在墨西哥特昆乌曼和危地马拉的接壤地段，另一批约1000人的中美洲难民突破了边境门，同防卫警察发生冲突，他们向工作人员和联邦警察投掷石块，鞭炮，玻璃瓶。导致6名警察受伤，一名洪都拉斯难民死亡。
 <br/>
 <br/>
 据英文《大纪元》报导，副总统迈克・彭斯（Mike Pence）引用与洪都拉斯总统胡安・奥兰多・埃尔南德斯（Juan Orlando Hernández）的谈话称，数千名大篷车至少部分是由洪都拉斯左翼活动分子组织，据称这些人是由委内瑞拉资助的。他们是来自洪都拉斯，萨尔瓦多和危地马拉的100多万移民中的一部分，这些移民自2013年以来一直试图非法进入美国。
 <br/>
 <br/>
 据了解，这些在边境附近被捕的人通常面临〝加急撤离〞，国土安全部（DHS）官员可以迅速将他们送回原籍国。
 <br/>
 <br/>
 但如果这些人要求庇护，政府可以选择将他们拘留，等待移民法官审理。但由于申请案件大量积压，大多数人都会在几年后才能得与法官见面。根据司法部长杰夫塞申斯的说法，这些人中有许多人没有出现，几乎有一半的人甚至没有填写并提交庇护申请。
 <br/>
 <br/>
 根据对锡拉丘兹大学联邦数据的分析，如果他们真正申请庇护，有4/5的中美洲人最终将被拒绝庇护。
 <br/>
 <br/>
 （记者李红报导/责任编辑：戴明）
