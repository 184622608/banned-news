### 美国中期选举冲刺  众议院成焦点战役
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】
 <b>
  新闻周刊
 </b>
 （652）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 将在下周二11月6号举行。国会两党以及总统川普本人，都在为选战做最后的冲刺。分析普遍认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 是这次中期选举的关键战役。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 能否守住众议院多数党席位，将在很大程度上影响未来两年川普施政的走向。
 <br/>
 <br/>
 国会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 435个席位今年全部改选，目前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 占有235席，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 193席。民主党要夺回众院控制权，必须至少从共和党手中赢得23席；其中1/3席位在加州。民主党为这7个国会众议员席位已经投入了6-7千万美元助选。
 <br/>
 <br/>
 加州共和党全国委员会主席Shawn Steel：〝现在我认为他们肯定能赢的只有一席，另外三场竞选则难以预料。我看见的每一个存在竞争的席位、对共和党来说比较悬的席位，都只有正负3%的差距。共和党有大概40%的几率保住众议院多数。〞
 <br/>
 <br/>
 全美20个国会摇摆选区的战况，对谁能主导众议院有关键影响。
 <br/>
 <br/>
 除了加州以外，纽约州第19选区现任共和党众议员法索与民主党挑战者德尔加多的竞争也相当激烈。最新民调中，德尔加多略微领先。
 <br/>
 <br/>
 此外，西弗吉尼亚州国会第三选区共和党众议员米勒，也面临民主党人奥杰达的挑战。该摇摆选区2016年大选中投票给川普，此次选举结果，被认为可在一定程度上反映出两年来的民意是否出现变化。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 此次有1/3席位改选，其中大部分是民主党选区。共和党对保住参议院多数党形势普遍持乐观态度，并认为可以扩大领先的优势。
 <br/>
 <br/>
 Amy Walter（Cook Political Report主编）：〝如果民主党想增加席位，甚至于仅仅是维持现状，依我看民主党不减少席位就已经很不错了，他们都需要赢得田纳西和德州的选举，川普在这两个州很受欢迎。这是参议院的选举格局，对川普非常有利。〞
 <br/>
 <br/>
 在密苏里州，共和党联邦参议员参选人霍利，与民主党现任参议员麦卡斯基尔，民调上只有2个百分点的差距。
 <br/>
 <br/>
 佛州州长斯科特挑战民主党籍现任参议员纳尔逊，选情也十分激烈，两人民调差距细微。共和党有可能在此位置上夺下一个参议员席位。
 <br/>
 <br/>
 随着选举日临近，美国总统川普也加快了助选的步伐。从10月31号到11月5号，川普6天要走访8个州参加11场助选活动，选举日前一天更是安排了三场活动。
 <br/>
 <br/>
 分析普遍认为，当前美国良好的经济形势是对共和党有利的因素。创下历史记录的低失业率，成为川普口中的王牌。
 <br/>
 <br/>
 美国总统 川普：〝仅上个月一个月，我们就增加了25万个工作，将近60万美国人重返工作岗位。你注意到了吗，在伟大的印第安纳州，我们让众多钢铁工人返回工作岗位，这是之前没有人能想像的。〞
 <br/>
 <br/>
 最近的大篷车难民潮，也让边境和移民问题成为中期选举的焦点议题。川普本周提出多项改革方案，包括取消出生公民权，禁止非法入境者申请庇护等。
 <br/>
 <br/>
 同时，川普也毫不讳言，来自国会民主党人的阻力是他实施移民改革的一大障碍。
 <br/>
 <br/>
 美国总统 川普：〝你可以提出最好的边境（保护）方案，最好的移民方案，但你不会从民主党那得到任何一票。他们有糟糕的政策，在许多场合下他们是糟糕的政客，但有一点我得称许他们，他们总是抱团投票，团结一致。〞
 <br/>
 <br/>
 美国民主研究所主任Patrick Basham：〝如果民主党在下周选举中胜利赢得众议院控制权， 他们会把逆转川普政策作为重大议程去推进。〞
 <br/>
 <br/>
 分析认为，当前美国政治光谱出现两极分化现象。一方对川普非常支持，另一方则极力反对川普。这一点在卡瓦诺大法官提名风波中展现的淋漓尽致。因此有人认为，11月6号的投票，可能是近年来对美国最重要的一次中期选举。
 <br/>
 <br/>
 撰稿：林宇
