### 佛州布县选举官15次被控舞弊 川普:历史可怕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】周四（11月8日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 计票出现巨大争议，其中至少两县的选票被指数额有诈，佛州州长已下令调查。历史记录显示，布劳沃德县（Broward）选举监督员斯奈普斯（Brenda Snipes）过去曾多次遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舞弊.html" target="_blank">
  舞弊
 </a>
 指控，这次再次成为丑闻焦点。
 <br/>
 <br/>
 截至周四晚上8点，尚未完成计票的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 布劳沃德县已激增近2万张选票，佛州共和党籍州长斯科特（Rick Scott）周四晚已下令执法部门展开调查。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周五告诉白宫记者们，布县
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选举官.html" target="_blank">
  选举官
 </a>
 员斯奈普斯（Brenda Snipes）有着〝可怕的历史〞，而且突然之间，不知从哪儿冒出来大量支持民主党的选票。
 <br/>
 <br/>
 斯科特于周四晚正式对布劳沃德县和棕榈滩县（Palm Beach）提起诉讼，指控这两县的选举部门与民主党人合作，企图利用〝猖狂的欺诈〞来窃取佛州选举。
 <br/>
 <br/>
 佛州是著名的摇摆州，其选举结果往往会影响美国的政治力量对比。而在刚刚结束的2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 中，两名共和党人双双赢得州长和参议员地位，而民主党则失去现有的参议院一席。
 <br/>
 <br/>
 周四晚，早就声名狼藉的斯奈普斯居然告诉采访她的记者们，布县之所以仍未完成计票，是因为他们的选民太多了。
 <br/>
 <br/>
 斯科特在诉讼书中指出，斯奈普斯不愿披露选举记录，以公开该县共有多少选民投票、多少选票已审查完、多少选票仍在等待审查，而这些〝不确定〞令人担忧——该县选举过程是否合法？
 <br/>
 <br/>
 斯科特的竞选团队声明，周二选举夜，布县共收到了634,000 张选票；但至美东时间周四晚8点，选票竟激增到712,840张。而据该县的选举部门公布，这些突然多出来的选票基本上都是投给民主党候选人的。
 <br/>
 <br/>
 斯奈普斯从2003年开始担任布县选举监督员，多年来广受争议。今年5月，一名法官裁决该县毁掉民主党众议员舒尔茨（Debbie Wasserman Schultz）与对手在党内初选中的选票，破坏了法律。之后，佛州政府表示，他们正在监察斯奈普斯的工作。
 <br/>
 <br/>
 在2017年的一次诉讼中，斯奈普斯在法庭上承认，其办公室曾允许非公民和重犯投票。
 <br/>
 <br/>
 美国保守派网站〝Gateway Pundit〞周五公布的调查记录显示，在斯科特提起诉讼前，斯奈普斯已14次受过选举
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舞弊.html" target="_blank">
  舞弊
 </a>
 的指控。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
