### 亚太裔中期选举热情高 独立选民大增
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】今年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 ，两党竞争异常激烈。专家认为，占美国总人口6%的亚太裔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选民.html" target="_blank">
  选民
 </a>
 ，无论参选还是投票的热情都空前高涨，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_独立选民.html" target="_blank">
  独立选民
 </a>
 数量也大量增加。
 <br/>
 <br/>
 过去近20年，美国亚太裔人口增长迅速，至今已超过2100万，成为竞选中两党都在争取的政治力量。本次中期选举，亚太裔投票热情空前高涨。据9月份一项民调，高达76%的亚太裔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选民.html" target="_blank">
  选民
 </a>
 表示，会参加选举投票。
 <br/>
 <br/>
 亚太裔投票联盟执行理事陈棣芳：〝我们看到更多的志工和组织，在中期选举前，协助选民注册和鼓动选民出来投票。总的来看，我们接到很多热线电话，寻问基本信息，如投票地点在哪里，投票时间等。〞
 <br/>
 <br/>
 虽然传统上亚太选民更倾向于支持民主党，但不同族裔也有很大差异。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_独立选民.html" target="_blank">
  独立选民
 </a>
 大量增加已成为新趋势。亚太裔投票联盟统计，选前还有25%
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚太裔选民.html" target="_blank">
  亚太裔选民
 </a>
 ，尚未决定投票支持哪个党派候选人。
 <br/>
 <br/>
 亚太裔投票联盟执行理事陈棣芳：〝越南裔和菲律宾裔更加支持共和党。华裔以往更加支持民主党，但我们现在看到，他们不明确倾向哪一政党。所以，我们看到尚未决定投票意向的选民在增加。〞
 <br/>
 <br/>
 川普当选总统后，不断推动各项改革。民众对政治活动参与度比以往显著提高。本次中期选举，也有更多亚太裔、尤其是女性候选人参选。
 <br/>
 <br/>
 亚太裔投票联盟执行理事陈棣芳：〝如在加州橙县，越南裔人口很多。这次选举，有24名越南裔候选人出来竞选公职，其中13人都是一个姓氏。〞
 <br/>
 <br/>
 调查还显示，就业、健保、教育以及控枪，都是亚太裔选民关注议题，投票倾向也更注重传统价值观。
 <br/>
 <br/>
 观察家认为，目前两党竞争激烈，实际差距可能比人们看到的民调数据小得多。至于选举结果，观察家也大但预测，共和党很可能继续赢得两院多数。
 <br/>
 <br/>
 美国霍华德大学政治学副教授 Michael Fauntroy：〝民主党将在众议院增加六席，参议院减少一席。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
