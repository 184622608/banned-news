### 精彩又刺激！青年开飞机环游世界 误闯台领空遭F-16战机〝拦截〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】来自美国路易斯安那州，只有18岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安德鲁斯.html" target="_blank">
  安德鲁斯
 </a>
 近日完成了驾机
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环游世界.html" target="_blank">
  环游世界
 </a>
 的纪录，耗时76天。他受访时提到，为了避免恶劣天气误入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 领空，还遭到空军
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-16战机.html" target="_blank">
  F-16战机
 </a>
 拦截，相当刺激。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 空军司令部官员周三（10日）接受中央社访问时证实了该事件，称此事件发生在9月19日下午，站管雷达队发现异常后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_F-16战机.html" target="_blank">
  F-16战机
 </a>
 紧急起飞前往目的地、并按程序查证，事后回报这名轻型飞机的驾驶者并没有恶意，研判是迷航，因此F-16战机紧急起飞，并导引他离开领空。
 <br/>
 <br/>
 18岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安德鲁斯.html" target="_blank">
  安德鲁斯
 </a>
 （Mason Andrews）最近挑战史上最年轻驾驶飞机
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环游世界.html" target="_blank">
  环游世界
 </a>
 者，在10月5日完成这趟奇妙旅程，回到路易斯安那州的门罗市（Monroe）。他受访时谈到旅程惊魂记时提到，曾在经过日本冲绳时，因天气恶劣，他的无线电还断线了，所以误入了台湾领空，然后被台湾F-16〝拦截！〞
 <br/>
 <br/>
 安德鲁斯表示，当时真的有些害怕，〝但可以看到战机真的很酷〞，本来他很想拍张照片，但是为了不激怒F-16的飞行员就没有拍照，他说，〝我知道他能看到我〞。之后在F-16引领下他飞回日本领空，结束这个意外插曲。
 <br/>
 <br/>
 令他欣慰的是，虽然没有拍到F-16战机的照片，但他从空中拍下了台湾中央山脉的壮观景象，还PO在Instagram上，分享这段他遇到战机的故事。
 <br/>
 <br/>
 旅程中还有许多意想不到的意外，安德鲁斯提到，原本预计在菲律宾待上两晚，但正巧遇上了燕子与山竹两个超强颱风，前后把他困在苏比克湾长达19天，〝这是我看过最强劲的颱风了〞。另外他飞经过沙特阿拉伯上空时，遇到了沙尘暴，在空中被困住长达10个半小时，最后才安全通过。
 <br/>
 <br/>
 虽然旅途中有很多惊险，但这趟环游世界的飞行还是让他见识到许多美景，其中一个就是飞经过巴黎时，从空中看到了艾菲尔铁塔，他形容，〝这真的是整趟旅程的亮点，我当晚降落在巴黎，看着城市被路灯点亮，走在塞纳河畔时才知道，自己刚刚飞越了大西洋呢！〞
 <br/>
 <br/>
 目前，安德鲁斯的航空记录尚未正式发布，他已把资料交给相关单位认证。他打算申请最年轻飞行环游世界、飞越大西洋与太平洋等纪录，而且他很有信心可以拿到认证，他说，〝我知道我做到了〞。
 <br/>
 <br/>
 安德鲁斯结束这趟旅程回到美国时，年龄是18岁又163天，比先前来自澳洲的纪录保持人史马特（Lachlan Smart）年轻了71天。
