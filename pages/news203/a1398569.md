### 弹劾川普希望落空 穆勒撰写通俄门调查最终报告
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】中期选举结束后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查再次成为美国国内政治的风向标。但美媒周四（11月8日）报导，调查已进入尾声，特别检察官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 （Robert Mueller）已开始撰写
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最终报告.html" target="_blank">
  最终报告
 </a>
 。
 <br/>
 <br/>
 据美国有线电视新闻网（CNN）周四报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 已经开始撰写
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查——即所谓川普选战勾结俄罗斯干扰2016年美国大选的调查报告。
 <br/>
 <br/>
 11月6日中期选举结束，共和党继续掌控并加强了在参议院的优势，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 则时隔8年重掌众议院。这一选举结果重新点燃了一些民主党人的希望，他们期待可以利用众议院制衡甚至弹劾总统，而通俄门调查则是赶川普下台的最现成工具。
 <br/>
 <br/>
 但目前看来，这一希望可能很快就会彻底落空。
 <br/>
 <br/>
 周四，即大选结束的第二天，因回避通俄门调查而遭川普批评的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部.html" target="_blank">
  司法部
 </a>
 长塞申斯（Jeff Sessions）终于应川普要求辞职。同一天，司法部发言人声明，副部长罗森斯坦（Rod Rosenstein）将不再负责监督通俄门调查，转而由川普新任命的代理司法部长惠特克（Matthew Whitaker）接替这项工作。
 <br/>
 <br/>
 司法部的声明无疑在民主党圈内投下一颗巨型炸弹，他们要求甚至威胁惠特克回避通俄门调查，但曾公开批评穆勒调查的惠特克拒绝了民主党的要求。
 <br/>
 <br/>
 多名消息人士告诉CNN，通俄门调查已进入收尾阶段，穆勒已开始撰写
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最终报告.html" target="_blank">
  最终报告
 </a>
 。
 <br/>
 <br/>
 目前尚不清楚穆勒是从何时开始草拟报告的，但不久前，川普的法律团队已同意书面回答穆勒的问讯。CNN称，预计川普团队将在本月底给出答覆。
 <br/>
 <br/>
 自2017年5月罗森斯坦指派穆勒调查通俄门事件开始，该调查已经持续了18个月，却始终没有发现任何川普选战通俄的证据。川普一再批评这是一场政治迫害，并呼吁尽早结束这场导致美国政治分裂并耗资巨大的〝猎巫〞行动。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
