### 〝大篷车〞陆续抵达 马蒂斯鼓励将士守边
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】周三，美国国防部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蒂斯.html" target="_blank">
  马蒂斯
 </a>
 和国土安全部长尼尔森，共同视察了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 的军队部署，以保护边境安全。
 <br/>
 <br/>
 国土安全部长 尼尔森：〝美国国土安全部同国防部有着很棒的合作关系。我们很荣幸能与你们合作，我们为你们自豪。〞
 <br/>
 <br/>
 14日，尼尔森与国防部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蒂斯.html" target="_blank">
  马蒂斯
 </a>
 共同前往边境探望美军，他们都肯定了部署军队以保护边境的做法。目前有约5,800名军人派驻在边境。
 <br/>
 <br/>
 国防部长 马蒂斯：〝新闻报导怎么说的都有，你们就专注于你们军官的指令。〞
 <br/>
 <br/>
 同一天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民仍在前往美国途中，他们得到了一个身份不明的墨西哥商人赞助的汽车，因此这些中美洲人得以加快速度前往与圣地牙哥接壤的墨西哥蒂华纳市（Tijuana）。据目击者称，周二，已有约400人到达那里。
 <br/>
 <br/>
 10月25日，美国副总统彭斯在接受采访时表示，大篷车是由海外的左翼集团资助。为了控制非法入境，川普总统11月9日签署公告，90天内暂停为非法越过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 的外国人提供庇护申请 。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
