### 可能撤换3至5名阁员 川普卖关子
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）在周日（18日）播出的访谈节目中透露，虽然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内阁.html" target="_blank">
  内阁
 </a>
 表现非常好，但也可能撤换3～5名的内阁官员。期间他还提到和白宫幕僚长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凯利.html" target="_blank">
  凯利
 </a>
 相处很好，也很尊重国土安全部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼尔森.html" target="_blank">
  尼尔森
 </a>
 ，但两人都有弱点与可改进处。
 <br/>
 <br/>
 在〝福克斯电视台〞周日上午播出的〝福克斯周日新闻〞节目中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 谈到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内阁.html" target="_blank">
  内阁
 </a>
 团队成员的表现与变动。期间他提到白宫幕僚长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凯利.html" target="_blank">
  凯利
 </a>
 （John Kelly）和国土安全部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼尔森.html" target="_blank">
  尼尔森
 </a>
 （Kirstjen Nielsen），也谈到对两人的期许。川普表示，他很尊重凯利和尼尔森，但两人也都有各自的弱点。
 <br/>
 <br/>
 主持人华勒斯（Chris Wallace）问川普，是否还如过去曾说的，希望凯利留任到2020年，川普说，〝我不会（这么说）〞，随后解释，要等等看，他和凯利处得很好，他很喜欢凯利做的某些事情，但〝有些事是他（凯利）不擅长的，但那不是他的错。凯利工作真的非常勤奋，在许多方面都表现的〝相当优异〞。
 <br/>
 <br/>
 华勒斯问，具体有哪些事是凯利不擅长的？但川普没有回答这个问题，他说，他还没考虑过凯利去留这件事，凯利留任到2020年还是有可能发生，〝让我们看看下面会发生什么事〞。
 <br/>
 <br/>
 在谈到尼尔森时，川普说，他非常尊重尼尔森，〝她很聪明，但我希望她能更强势一些，（她的职务）应极为强势。〞他也希望尼尔森在这艰难时期，能更强悍执行移民政策，并加强边境安全。
 <br/>
 <br/>
 川普在节目中还说，他非常乐观，白宫的运行就像一台运行良好的机器，做得很好，〝我有很棒的团队，我会做一些改变但不是很多。我对我的内阁非常满意，除了几个例外，即便如此，我也不会不高兴。〞
 <br/>
 <br/>
 他也提到，虽然有考虑要更换3到5位的内阁人选，但最后可能只换2个，而在人事安排上他想保有灵活性。
 <br/>
 <br/>
 目前外媒还未点名其他可能被换的人选。
