### 收到川普的简讯了吗 3日全美总统警报测试
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】如果你生活在美国，周三下午有没有收到〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统警报.html" target="_blank">
  总统警报
 </a>
 〞呢？这个系统允许总统办公室直接向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国手机用户.html" target="_blank">
  美国手机用户
 </a>
 发送
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_紧急短信.html" target="_blank">
  紧急短信
 </a>
 ，来看报导。
 <br/>
 <br/>
 10月3日下午2:18分，全美国到处都听到手机警报响起的声音，一看，会发现手机上显示的是〝总统警讯〞（Presidential Alert）的测试讯息，并写着〝这是国家无线紧急警报系统的测试，不需要采取任何行动〞。
 <br/>
 <br/>
 纽约民众 Bolle：〝我感觉到手机的震动，所以我很好奇是什么。结果是来自总统的简讯。〞
 <br/>
 <br/>
 原来，这是由美国联邦紧急事务管理局（FEMA）和美国联邦通信委员会（FCC）发出的，用来测试新的〝WEA无线紧急警报系统〞（Wireless Emergency Alerts），这个系统将允许总统办公室，直接向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国手机用户.html" target="_blank">
  美国手机用户
 </a>
 发送短信。
 <br/>
 <br/>
 白宫发言人Sarah Sanders：〝这是第一次全国性的WEA测试，检测整体操作的成熟度，全国讯息发送的结构稳定性，并决定是否需要改善技术问题。〞
 <br/>
 <br/>
 这个警报系统与即时通报孩童遭到绑架的〝安柏警报〞(Amber Alert）和危险天气警报类似。
 <br/>
 <br/>
 纽约民众 Jerry Brown：〝这是他们第一次这样做，我感觉这主意挺好的。是政府与老百姓之间的沟通。〞
 <br/>
 <br/>
 也有些手机用户反应，没有收到警报，专家列出可能的原因为，小型且当地的电信商可能没有参与WEA计划，而手机机型也要能够与WEA相容，其功能要是开启的状态，且在收讯范围之内才收的到警报。
 <br/>
 <br/>
 首次的测试，有些手机用户反应收到时间较晚，也有用户反应收到三次以上的警报。
 <br/>
 <br/>
 新唐人记者王愉贺、王真纽约报导
</div>
