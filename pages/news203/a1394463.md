### 中共驻美大使方寸大乱  受访讲话矛盾百出
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】海外多家媒体日前刊出中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻美大使.html" target="_blank">
  驻美大使
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔天凯.html" target="_blank">
  崔天凯
 </a>
 ，在接受美媒采访时乱了阵脚，说话时语言前后矛盾，人也显得〝形单影只〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔天凯.html" target="_blank">
  崔天凯
 </a>
 在10月3日接受美国全国公共广播电台（National Public Radio，NPR）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专访.html" target="_blank">
  专访
 </a>
 时，被问到中共是否愿意展示更大的开放性，允许北京不喜欢的学者或者记者，进入西藏等所谓〝敏感〞的地区采访？崔天凯回答先是使用中共套话，称〝中国对美国学生、教授、记者或学者都是开放的〞，但话锋一转，〝一些其它地区，像西藏，因为海拔很高，那里的气候可能非常恶劣。〞
 <br/>
 <br/>
 NPR主持人随即表示，美国记者和学者可以克服高海拔环境后，崔天凯则硬生生回应称，〝不是每个人都可以很快适应高原气候和自然条件。〞
 <br/>
 <br/>
 崔天凯把中共当局限制外国人进入西藏，归结于高原恶劣的环境和气候，令人啧啧称〝奇〞。
 <br/>
 <br/>
 NPR主持人继续发问（美国人）是否可以前往新疆时，崔天凯仍重复表示，〝整个中国都对世界开放，当然新疆也不例外〞，但话锋再转，〝新疆有一个特别的问题，就是恐怖主义……〞
 <br/>
 <br/>
 当被问及中国的言论自由时，崔天凯表示，〝如果浏览过中国社交媒体，就会看到上面有各种各样的讨论，可见中国言论的自由程度〞，但他只字不提中共利用网警封堵、删除微博敏感评论，甚至将网民微博空间全部清空，或干脆把用户帐号注销。
 <br/>
 <br/>
 <h4>
  中共官员愈来愈孤立
 </h4>
 <br/>
 旅居美国的台湾记者吕佳颖10月6日在《上报》（UP Media）报导引述一名智库朋友的话说，就在美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）进行公开演讲的当晚（4日晚），德国大使馆举行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国统一日.html" target="_blank">
  德国统一日
 </a>
 〞庆祝酒会，参与人员包括美国官员、各国使节、智库学者等，当然也包括崔天凯。
 <br/>
 <br/>
 这位智库朋友当晚也有幸参加了酒会。他说：〝我注意他（崔天凯）大约10分钟，但是没有人跟他说话……〞，〝崔天凯先是自己看表演，之后才去拿东西，一下自己站在那低头猛吃，一下抬头看表演，期间没有美方人士和其它国家的使节跟他攀谈。〞
 <br/>
 <br/>
 他用〝形单影只〞来形容崔天凯当时的尴尬处境。
 <br/>
 <br/>
 文章还透露，自从美国总统川普（特朗普）9月26日在联大会议上，指控中共介入美国大选，并透露彭斯一周后将公开演讲后，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻美大使.html" target="_blank">
  驻美大使
 </a>
 馆〝整整一个星期都在加班〞。文章称，可能崔大使们想透过许多关系，来摸清彭斯到底想说些什么。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：祝馨睿）
</div>
