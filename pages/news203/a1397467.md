### 华府罕见要求也门内战停火 30天内和平会谈
------------------------

 </a>

  沙特阿拉伯
 </a>
 与

 </a>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青年运动.html" target="_blank">

 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停火.html" target="_blank">
  停火
 </a>
 ，并于未来一个月就持续3年的也门内战坐上谈判桌。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蒂斯.html" target="_blank">
  马蒂斯
 </a>
 （Jim Mattis）在设于华盛顿的美国和平研究所（US Institute of Peace）表示，华府观察到这场冲突〝够久了〞；他相信沙特与阿拉伯联合大公国(阿联酋)就对抗什叶派
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青年运动.html" target="_blank">
  青年运动
 </a>
 （Huthi）叛军，已经准备好会谈，并需要推动和平进程。
 <br/>
 <br/>
 马蒂斯更设下冲突各方未来30日谈判，也呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_也门.html" target="_blank">
  也门
 </a>
 内战有关各方，于11月在瑞典与联合国代表特使葛瑞菲斯（Martin Griffiths）会晤，共商解决方案。马蒂斯认为这是唯一的可行方法。
 <br/>
 <br/>
 据悉，上个月，青年运动叛军拒绝到日内瓦，声称联合国无法保证可让他们安全回到也门首都沙那，导致联合国主导的和平会谈无法顺利举行。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特阿拉伯.html" target="_blank">
  沙特阿拉伯
 </a>
 2015年与盟邦支持四面楚歌的也门总统哈迪（Mansour Hadi）与青年运动之间的冲突，迄今已有近万人丧命，导致也门现在处于饥荒边缘。
