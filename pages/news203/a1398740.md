 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 州长下令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_调查.html" target="_blank">
  调查
 </a>
 该州存在的选举舞弊现象，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统也表示支持此项调查。周五，川普再次发表多篇推文，誓言曝光
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 在计票中的欺诈行为。

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统宣布，执法部门正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_调查.html" target="_blank">
  调查
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 布劳沃德县（Broward）和棕榈滩县（Palm Beach）发现的涉嫌选举欺诈的腐败丑闻，并指出佛州选民支持斯科特（Rick Scott）。

 <br/>
 美国各州均在中期选举日（11月6日）结束投票，但佛州布劳沃德县却至今仍未完成计票，且在选举结束后两天内突然多出数万张
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选票.html" target="_blank">
  选票
 </a>
 ，引发了两党极大关注。
 <br/>
 <br/>
 周四，佛州共和党参议员卢比奥（Marco Rubio）发推警告，该州未来的州长位置可能正攥在布劳沃德县选举监督员斯奈普斯（Brenda Snipes）手中，并指责她违反了佛州及联邦的法律。
 <br/>
 <br/>
 布劳沃德县是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 占优势的选区，且经常成为佛州67个县中计票最慢的一县。该县的选举部门也多次因不法行为而受到指责。
 <br/>
 <br/>
 在周二的州长选举中，共和党候选人德桑蒂斯（Ron DeSantis）以0.62%的微弱优势战胜民主党候选人吉卢姆（Andrew Gillum），当选佛州下任州长。吉卢姆周二晚本已承认自己败选，但随着周四布劳沃德县的计票结果出台，他的立场立即逆转。
 <br/>
 <br/>
 共和党籍州长斯科特在该州参议员竞选中战胜了竞选连任的民主党候选人尼尔森（Bill Nelson）。但尼尔森为求反败为胜，已聘请Perkins Coie法律事务所的律师埃利亚斯（Marc Elias）参与重新计票工作。
 <br/>
 <br/>
 值得一提的是，在2016年大选期间，埃利亚斯曾担任希拉里（Hillary Clinton）竞选团队及民主党全国委员会（DNC）的法律代理，他从2016年4月开始持有福森公司（Fusion GPS），而正是这家公司雇傭前英国间谍斯蒂尔（Christopher Steele）编制了朝名昭着的黑川普档案，并导致联邦调查局（FBI）启动了所谓〝川普选战勾结俄罗斯干预美国大选〞的通俄门调查。
 <br/>
 <br/>
 斯科特指出，埃利亚斯正采用攻击性技巧〝窃取〞选举。
 <br/>
 <br/>
 卢比奥周四在推特上揭露，民主党的律师们正抵达佛州，而他们的目的不是确保每张选票都得到正确计算，而是企图改变选举结果。

 <br/>
 卢比奥还指出，身为选举监督员的斯奈普斯居然表示她不知道究竟还有多少选票等待计数，而且也不定期报告结果，但过程中却在快速削减共和党人的票数优势。
 <br/>
 <br/>
 选举夜的非正式计票结果显示，斯科特领先尼尔森将近60,000张选票，但是至周三和周四，棕榈滩的计票结果却开始巨幅削弱斯科特的优势，一些民主党律师也抵达佛州，声称共和党犯规并要求重新计票。
 <br/>
 <br/>
 而布劳沃德县的情况更加可疑。斯科特的竞选团队声明，周二选举夜，该县共收到了634,000 张选票；但美东时间周四凌晨1点，选票突然变为 695,700张；周四下午2:30，这一数字再摇身变成707,223；周四晚8点，选票竟激增到了712,840张。而据该县的选举部门公布，这些突然多出来的选票基本上都是投给民主党候选人的。
 <br/>
 <br/>
 州长斯科特周四下午就布劳沃德县和棕榈滩县的计票问题召开新闻发布会，并于当晚下令该州警方立即投入调查。他指出，棕榈滩县也在选举日后突然多出15,000张选票，这两个县很可能都存在选举舞弊现象。
 <br/>
 <br/>
 周五，川普总统连发推文，就佛州计票问题发表一系列评论，并且指出，虽然民主党派出了自己〝最好的窃取选举律师〞、并开始〝奇迹般地〞帮民主党候选人〝发现〞选票，但是，〝别担心，佛罗里达—— 我会派出更好的律师去曝光欺诈！〞
