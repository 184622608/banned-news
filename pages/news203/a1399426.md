### 加州山火烧焦核污染区引争议 官员：未发现有毒物
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】美国加州南部遇野火肆虐，大火也烧焦了被核污染的前火箭测试与研究场所的部分地区。但据加州公共卫生官员说，有毒化学物质没有被释放到空气中。
 <br/>
 <br/>
 伍尔西林山火（Woolsey Fire）于上周四（8日）在文图拉县（Ventura County ）圣苏珊娜野外实验室（Santa Susana Field Lab）附近爆发，在洛杉矶和文图拉县烧毁了超过97,000英亩的土地，导致至少2人死亡。
 <br/>
 <br/>
 来自多个机构的公共卫生官员收集了该区的空气样本，并在该设施周围进行辐射探测。
 <br/>
 <br/>
 据国家广播公司（NBC）报导，洛杉矶县公共卫生部周二（11月13日）表示，上周末在该实验室附近进行初步测试发现没有明显的辐射水平。
 <br/>
 <br/>
 负责监督该研究所清理的加州有毒物质控制部门也表示，〝以前处理过的放射性和危险的材料不受火灾影响。〞
 <br/>
 <br/>
 圣苏珊娜野外实验室建于20世纪40年代，位于西米谷（Simi Valley）和查茨沃斯（Chatsworth）之间，占地约2,900英亩，过去用于从事火箭引擎及核研究。近几十年来，圣苏珊娜野外实验室被用作前洛克丹公司（Rocketdyne）的火箭测试和研究实验室，一直是周围社区争议的焦点。
 <br/>
 <br/>
 那里曾发生过核反应堆事故，并在1959年发生了部分核熔毁。核熔毁就是核反应堆因无法及时冷却而熔化造成的损毁。核燃料熔毁后可引发具有放射性的物质外洩，并给人类及其它生物留下严重后遗症。
 <br/>
 <br/>
 当局在继续使用该场所的同时，因发现大面积的化学和放射性污染，所以许多建筑物后来被废弃。1996年，波音公司收购了洛克丹公司，成为该场所的主要拥有者。
 <br/>
 <br/>
 美国能源部表示，活动人士对进展缓慢的修复过程感到沮丧，其中包括拆除废弃和受污染的建筑物。
 <br/>
 <br/>
 西米谷居民，同时也是洛克丹公司清理联盟（Rocketdyne Cleanup Coalition）的联合创始人梅森（Marie Mason），一直关注可能接触有害毒素的活动，她呼吁有关人员对空气测试进行独立分析。
 <br/>
 <br/>
 梅森在一份声明中说，〝我们感到愤怒的是，经过这么多年，加州有毒物质控制部门几十年的拖延……可能导致更多毒素泄露，我们一直担心圣苏珊娜野外实验室会发生火灾，而现在大规模的野火已经发生了。很久以前，圣苏珊娜野外实验室可能应该已被清理过了。〞
 <br/>
 <br/>
 实验室周围居民和相关组织努力了30年，一直要求对实验室进行高标准的清理。他们担心该实验室潜在的化学物质及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核辐射.html" target="_blank">
  核辐射
 </a>
 物质污染会影响健康。加州一项州研究曾发现，圣苏珊娜野外实验室的工作人员患癌症的比率较高。而且从1988年到1995年，生活在实验室2英里范围内的居民与居住在5英里远的居民相比，其罹患甲状腺癌、上消化道癌、膀胱癌、血癌和淋巴癌的比率高出60%以上。
 <br/>
 <br/>
 （记者李红报导/责任编辑：文慧）
