### 犹太教堂枪案 民众悼念 塞申斯谴责暴力
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】周一晚间，美国各地举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_烛光守夜.html" target="_blank">
  烛光守夜
 </a>
 ，悼念宾州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹太教堂枪击事件.html" target="_blank">
  犹太教堂枪击事件
 </a>
 的遇难者。司法部长塞申斯表示，这不只是攻击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹太信徒.html" target="_blank">
  犹太信徒
 </a>
 ，这是对人类所有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信仰.html" target="_blank">
  信仰
 </a>
 者的攻击。川普总统周二下午前往宾州，慰问受害家属。
 <br/>
 <br/>
 宾州犹太教堂枪击案嫌犯、46岁的卡车司机罗伯特・鲍尔斯，周一出庭应讯。他面临仇恨罪等29项联邦刑事指控。
 <br/>
 <br/>
 美国司法部长塞申斯周一在波士顿表示，这是对人类所有信众、以及美国价值观的攻击。
 <br/>
 <br/>
 美国司法部长塞申斯：〝这不仅是对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹太信徒.html" target="_blank">
  犹太信徒
 </a>
 的攻击。这是对所有人类
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信仰.html" target="_blank">
  信仰
 </a>
 者的攻击。这是对美国保护信仰者的价值观的攻击。这不能也不会被容忍。这是对我们本身的直接威胁。〞
 <br/>
 <br/>
 周一晚间，华盛顿特区、宾州约克和美国其他城市都举行了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_烛光守夜.html" target="_blank">
  烛光守夜
 </a>
 ，悼念遇难者。各地政要和宗教人士到场声援信仰自由，谴责暴力。
 <br/>
 <br/>
 宾州州长沃尔夫：〝这种卑鄙和懦弱的反犹太主义行为，在我们的社会没有立足之地。这种行为与我们的本质背道而驰。〞
 <br/>
 <br/>
 悲剧发生后，社区民众自发的以各种形式悼念受害者。商店老板发起〝爱比仇恨更强大〞活动，让人们留言，记录哀悼与关怀。
 <br/>
 <br/>
 几个穆斯林团体联合募款，为受害者家属支付葬礼和医疗费用，目前已经筹集了超过16万4千美元。
 <br/>
 <br/>
 新唐人记者李梅、田瑞综合报导
