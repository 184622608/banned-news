### 领先7关键州早期投票  共和党或再掌两院
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】随着美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 日益临近，民主党所鼓吹的〝蓝波〞已渐趋渺茫。周一（10月22日）发布的最新分析结果显示，在7个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关键州.html" target="_blank">
  关键州
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_早期投票.html" target="_blank">
  早期投票
 </a>
 中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 占据优势，参众两院的控制权在选举后或仍为共和党所掌握。
 <br/>
 <br/>
 周一，美国全国广播公司新闻网数据分析实验室（NBC News Data Analytics Lab）在分析由政治数据与技术领先提供商TargetSmart所收集整理的数据后，公布了一项令民主党人气馁的结果——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 在多数
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关键州.html" target="_blank">
  关键州
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_早期投票.html" target="_blank">
  早期投票
 </a>
 中都处于优势地位。
 <br/>
 <br/>
 数据显示，在亚利桑那、佛罗里达、乔治亚、印第安纳、蒙大拿、田纳西和德克萨斯等7个战场州，支持共和党的选民超过了支持民主党的人数。
 <br/>
 <br/>
 数据分析所涵盖的几个摇摆州中，只有内华达州的民主党支持率超过了共和党，两者分别为45%与38%，相差7个百分点，仅略大于两党在佛州的差距——6个百分点。
 <br/>
 <br/>
 在其它5州，共和党的支持率最小的高出民主党9%，最大的则高出民主党33%（田纳西州，分别为63%和30%）。
 <br/>
 <br/>
 这8个州中，有7个州正进行事关参议院控制权的关键竞选。
 <br/>
 <br/>
 这一最新的分析结果让〝蓝波〞吹嘘不攻自灭。迹象显示，不仅此前预计的共和党掌控参议院不难实现，甚至众议院领导权也将继续留在共和党手中。
 <br/>
 <br/>
 目前距
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 仅剩两周时间，川普总统更加积极地投入助选。川普选战周二宣布，再投600万美元购买竞选广告，川普也将在11月6日前再出席至少10场竞选集会，以其势不可挡的号召力全力支持各地共和党候选人。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
</div>
