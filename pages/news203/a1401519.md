### 喜达屋连锁酒店遭骇 5亿用户或受影响
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】自2014年以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万豪国际.html" target="_blank">
  万豪国际
 </a>
 旗下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_喜达屋连锁酒店.html" target="_blank">
  喜达屋连锁酒店
 </a>
 的数据库被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑客入侵.html" target="_blank">
  黑客入侵
 </a>
 ，最多有5亿名顾客的个人信息或洩露。周五，万豪国际股价下跌5.59%
 <br/>
 <br/>
 记者Jeremy Sandberg：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万豪国际.html" target="_blank">
  万豪国际
 </a>
 集团旗下的连锁酒店集团喜达屋在过去的四年里，其客房预订系统遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑客入侵.html" target="_blank">
  黑客入侵
 </a>
 ，大约5亿客户记录受到影响。〞
 <br/>
 <br/>
 万豪国际股价在周五开盘时下跌5％。
 <br/>
 <br/>
 此次黑客入侵事件始于2014年，在万豪提出收购喜达屋之前。万豪最终于2016年完成收购喜达屋。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_喜达屋连锁酒店.html" target="_blank">
  喜达屋连锁酒店
 </a>
 品牌包括喜来登，利兹卡尔顿和威斯汀。万豪品牌的连锁酒店都没有受到骇客威胁。
 <br/>
 <br/>
 这次喜达屋客户信息洩露的规模可能是史上第二大，仅次于2013年，雅虎30亿用户信息遭洩露事件。
 <br/>
 <br/>
 英国监管机构和至少美国三个州已开始调查这起骇客入侵事件。
 <br/>
 <br/>
 骇客访问了大约3.27亿客户的护照信息，电话号码和电子邮件地址，包括一些信用卡号码。
 <br/>
 <br/>
 由于公司尚未完成全面调查，因此确切的受影响顾客的数目还不能确定。9月万豪收到内部安全工具发出的关于未授权第三方试图访问的警报，发现未授权的第三方已经复制并加密了某些信息，万豪直到上周才成功解密相关信息，并确定喜达屋客户预定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_数据库被骇.html" target="_blank">
  数据库被骇
 </a>
 客入侵。
 <br/>
 <br/>
 记者Jeremy Sandberg：〝万豪专门为此建立了一个网站和电话客服中心，任何认为自己的个人信息受到影响的人可以与之联系。〞
 <br/>
 <br/>
 新唐人记者Jeremy Sandberg纽约报导
