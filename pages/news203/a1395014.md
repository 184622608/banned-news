### 美国国土安全部：构成公共负担 影响绿卡或签证
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国土安全部.html" target="_blank">
  国土安全部
 </a>
 在近日颁布新规定，将构成〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公共负担.html" target="_blank">
  公共负担
 </a>
 〞和享受〝公共福利〞，列入申请
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绿卡.html" target="_blank">
  绿卡
 </a>
 或
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_签证.html" target="_blank">
  签证
 </a>
 的考量因素，请看报导。
 <br/>
 <br/>
 根据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国土安全部.html" target="_blank">
  国土安全部
 </a>
 在10月10日颁布的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公共负担.html" target="_blank">
  公共负担
 </a>
 不予接受〞（Inadmissibility on Public Charge Grounds）草案。申请临时或永久
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绿卡.html" target="_blank">
  绿卡
 </a>
 、非移民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_签证.html" target="_blank">
  签证
 </a>
 调整为绿卡、非移民签证延期或转换签证的申请者，都属于新规定的范围内。
 <br/>
 <br/>
 若被认定构成公共负担，申请绿卡或签证时，将有可能被拒。
 <br/>
 <br/>
 美国移民律师 夏渢：〝因为现在审核的项目越来越多，并且表格里面的问项肯定也会越来越多，那么势必会增加每一个案子的审理过程，所以等待的期间会变长。〞
 <br/>
 <br/>
 〝公共负担〞的范围从原来的现金补助（SSI）、白卡（Medicaid）中的长期护理，扩大到粮食券（SNAP）、白卡（Medicaid）非紧急医疗、红蓝卡（Medicare）的D部分，以及第八类房屋补助券（Section 8 housing voucher）。
 <br/>
 <br/>
 美国移民律师 夏渢：〝目前还是没有必要停止（福利），因为还是在60天内征求公众意见，并没有真正的实行。但如果说你真的打定主意要留在美国的话，在没有必要的情况下，我认为是没有必要去大量的申请政府补助。〞
 <br/>
 <br/>
 不过，难民、政治庇护或基于人道主义的永久或临时身分申请人，则不在新规定范围内。
 <br/>
 <br/>
 美国移民律师 夏渢：〝难民，包括一些特别列出来的国家，包括：伊拉克、阿富汗这些国家的难民，都特地是区别在外，他们是不受这个规定限制的。〞
 <br/>
 <br/>
 至于是否构成公共负担，则根据申请者的年龄、健康、家庭状况、财务状况、教育及技能等衡量。
 <br/>
 <br/>
 如果家庭收入水平超过联邦贫穷线250%，即2018年个人收入为3万350美元、四口家庭为62,750元，申请绿卡时加分。反之，如果失业、办理移民身份时申请免除费用、在就医时没有私人的医疗保险公司，则会减分。
 <br/>
 <br/>
 美国移民律师 夏渢：〝最好的办法就是，如果家庭成员报税够高，或者家庭成员有稳定的收入，稳定的工作的话。填写一张I864表格（经济担保书），去支持申请中的家庭成员，即便他（申请人）可能是工作能力不好，但是我的工作能力可以，我会去在经济上支持他。〞
 <br/>
 <br/>
 新草案有60天的征集公众意见期，到12月10日截止。一旦最终的规定颁布，将在60天后生效。
 <br/>
 <br/>
 网址：https://www.regulations.gov/
 <br/>
 <br/>
 新唐人记者柯婷婷、韩瑞纽约报导
</div>
