### 解决两岸问题  吴新兴：自由人权是基础
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】日前在美中对话中，国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 公开表达了美国对民主台湾的支持，和对中共打压宗教信仰自由的关切。周六，正在华府访问的中华民国侨务委员长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴新兴.html" target="_blank">
  吴新兴
 </a>
 也表示，自由民主、尊重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权.html" target="_blank">
  人权
 </a>
 是解决两岸政治分歧的最重要基础。
 <br/>
 <br/>
 美中外交与安全对话日前在华府举行。会后联合记者会上，国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 明确表达美方在台湾问题上的立场。
 <br/>
 <br/>
 美国国务卿 Mike Pompeo：〝关于美国与‘民主台湾’的坚定关系，我重申美国政策没有变化。美国关切中共不断增加对他国的胁迫，压缩台湾国际空间。〞
 <br/>
 <br/>
 针对蓬佩奥的讲话，中华民国驻美副代表黄敏境，周六在华府同乡会午宴上，特别向美方表达感谢。
 <br/>
 <br/>
 中华民国驻美副代表 黄敏境：〝感谢您，国务卿蓬佩奥！感谢您支持台湾，感谢您支持民主！〞
 <br/>
 <br/>
 出席同一活动的中华民国侨委会委员长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴新兴.html" target="_blank">
  吴新兴
 </a>
 ，也解释了蔡政府为何无法接受中共的所谓〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九二共识.html" target="_blank">
  九二共识
 </a>
 〞与〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一中原则.html" target="_blank">
  一中原则
 </a>
 〞。
 <br/>
 <br/>
 中华民国侨委会委员长 吴新兴：〝这一中国原则，就完全排除了中华民国的发言机会，没有尊重中华民国政府，也漠视台湾人民心声。事实上，我们认为两岸应该坐下来谈。只有透过对等的、互相尊重、有尊严的对话跟协商，才能够真正解决两岸之间的政治分歧。〞
 <br/>
 <br/>
 本轮对话中，蓬佩奥还谈到，美国对于中共打压宗教信仰自由的关切。
 <br/>
 <br/>
 美国国务卿 Mike Pompeo：〝美国和国际社会将继续表达，我们对于中共压迫宗教信仰团体的忧虑。〞
 <br/>
 <br/>
 中华民国侨委会委员长 吴新兴：〝自由民主、尊重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权.html" target="_blank">
  人权
 </a>
 是解决两岸政治分歧最重要的一个基础。我们觉得这一点，中国大陆领导人应该好好去思考。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
