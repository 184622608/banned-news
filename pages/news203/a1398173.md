### 美中将达贸易协议？ 川普提出核心诉求
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】上周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 与习近平通话后，川普表示有意和北京达成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易协议.html" target="_blank">
  贸易协议
 </a>
 。白宫首席经济顾问库德洛，周日披露了川普的核心诉求。
 <br/>
 <br/>
 4号，白宫首席经济顾问库德洛告诉媒体，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统有意愿和北京达成协议，但有一个核心要求。
 <br/>
 <br/>
 库德洛：〝这份协议必须效力强大，并且符合
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 利益，这是总统的核心要求。〞
 <br/>
 <br/>
 库德洛说，美国已就中方的强制技术转移、知识产权窃取、高关税、在华外企所有权不足等问题，向北京提出了诉求。
 <br/>
 <br/>
 库德洛：〝如果他们回应这些诉求，那么总统会考虑，跟他们达成一份符合美国利益的协议。〞
 <br/>
 <br/>
 与此同时，美国没有派任何官员，出席中共5号在上海举办的首届〝进口博览会〞。另外七大工业国集团，以及中共主要的贸易伙伴韩国澳洲的政要也全数缺席。
 <br/>
 <br/>
 《彭博社》说，外界期望北京解决盗窃知识产权等问题，北京的回应却是〝我多买你点商品〞，凸显出北京政策和外界期待之间，有很大差异。
 <br/>
 <br/>
 谢田：美国已经明确表示了，美国不会满足于中国承诺的〝多进口几百亿、几千亿美元的产品〞。美国现在的要求非常清楚，要全面彻底的，从根本上，从制度上解决中美贸易的障碍。
 <br/>
 <br/>
 10月初，美国曾在〝美墨加协定〞中加入一项重要的〝毒丸条款〞，限制成员国与〝非市场经济国家〞签订自贸协议，折射出美国的决心。
 <br/>
 <br/>
 横河：根据这个条款，加拿大已经暂停了和中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易协议.html" target="_blank">
  贸易协议
 </a>
 讨论。美国也是签订协议的一方，当然也必须受到条款制约。如果中共不能作出有效的步骤、而不是承诺，就是你作出行动来，达到市场经济国家的要求的话，美国也没有办法和中共达成贸易协定。
 <br/>
 <br/>
 库德洛8月份曾披露，无论美国中期选举结果如何，川普都不会改变解决
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易.html" target="_blank">
  美中贸易
 </a>
 问题的立场。
 <br/>
 <br/>
 横河：我认为美国不会走到这一步，就是需要用〝毒丸条款〞来制约自己，因为迄今为止，我们没有看到过，川普总统在兑现竞选承诺，和实现〝让美国重新伟大〞的过程中，有改变其基本立场的迹象。
 <br/>
 <br/>
 《南华早报》披露，川普与习近平约定，12月1日在G20峰会后将会面并举行晚宴，详谈中美贸易等话题。
 <br/>
 <br/>
 谢田：中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的实质，涉及到中国经济体制的问题，实际上就是中共政权对经济的掌控、对财富的掌控、对国家和社会的控制能力的问题。就是共产党生死存亡的问题。习近平到这个月底，必须作出一系列决定，来决定跟川普怎么谈。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
