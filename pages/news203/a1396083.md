### 佛州选情激烈 飓风效应或影响投票率
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】刚经历飓风袭击的美国佛州，下星期将迎来今年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_先期投票.html" target="_blank">
  先期投票
 </a>
 〞。目前2个席位的争夺值得关注。
 <br/>
 <br/>
 目前，佛州州长和联邦参议员的竞逐呈现胶着状态。
 <br/>
 <br/>
 现任州长斯科特（Rick Scott）任期已满，今年竞逐联邦参议员席位，挑战寻求四连任的民主党人纳尔逊（Bill Nelson）。根据民调机构St.Pete周二发布的数据，斯科特在注册选民中领先纳尔逊2个百分点、达到49%。他应对迈克尔飓风的表现提升了支持度。
 <br/>
 <br/>
 〝美国企业研究所〞高级研究员Karlyn Bowman：〝佛州的选举、州长和参议员的竞选，可能是美国竞争最激烈的角逐之一。〞
 <br/>
 <br/>
 州长职位的竞争者，现任国会众议员共和党的德桑蒂斯（Ron Desantis）仅落后民主党的塔拉赫西市长吉卢姆（Andrew Gillum）1个百分点。吉卢姆被川普称作是〝失败的社会主义市长〞，他主张增收企业税、扩大医疗补助（白卡）的受益人群、加强控枪。德桑蒂斯在周四杰克逊维尔（Jacksonville）的活动中则重申，会继续斯科特的经济政策——减少管制、降低税收以增加就业。
 <br/>
 <br/>
 川普最近常用委内瑞拉来提醒美国人，激进左派如果当选，美国将会步委国后尘。
 <br/>
 <br/>
 由于飓风迈克尔的灾后效应仍在持续，受灾选区的投票点除了被毁损外，有的被用做飓风庇护所，有的用来储存物资。目前大面积的停电的11个县中有8个县是共和党的票源。
 <br/>
 <br/>
 〝美国企业研究所〞高级研究员Karlyn Bowman：〝这些被飓风灾区的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票率.html" target="_blank">
  投票率
 </a>
 会受到影响。但他们在恢复中，距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 还有两周时间，相信佛州会尽全力让投票可以进行。〞
 <br/>
 <br/>
 州长斯科特放宽了这8个县的投票规则，以减少飓风的影响。州务卿也正在考虑建议，是否把受灾区域并入其他选区。
 <br/>
 <br/>
 佛州普选的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_先期投票.html" target="_blank">
  先期投票
 </a>
 〞从10月27日起至11月3日。前副总统拜登下周一和二会在佛州坦帕、奥兰多和杰克逊维尔三地助选。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
