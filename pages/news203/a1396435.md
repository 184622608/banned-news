### 难民涌入列〝国家级警戒〞  川普将断中美洲3国金援
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】据悉，2018财政年度，共约10.8万个无证移民家庭被美国政府羁押。上周末，非法跨越墨西哥南部边境的移民飙升至7000人左右。对此，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）周一（22日）连发推文称，难民涌入美国是〝国家级警戒〞，并宣布即刻中止或削减对中美洲3国的金援。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周一透过推特表示，他已经向美国边境巡逻队（The U.S. Border Patrol ）与军方下令，特别注意入境美国的篷车，因为那里面很可能就有中美洲国家的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 趁机偷渡入美国。
 <br/>
 <br/>
 他除了将大量难民涌入国内列为〝国家级警戒〞，也践行此前对中美洲国家发出的警告，宣布由于危地马拉、洪都拉斯、萨尔瓦多这3个中美洲国家，皆未能阻止国民非法入境美国，即刻停止或削减对这些国家的经济援助。
 <br/>
 <br/>
 据《美联社》报导，这3个国家去年总共收下了美国高达5亿多美元的金援。
 <br/>
 <br/>
 英国广播公司（BBC）报导，2017年，洪都拉斯获得美方援助1.75亿美元，危地马拉得到逾2.48亿美元，萨尔瓦多1.15亿美元。这些国家充斥着帮派犯罪、贩毒、失业、贫穷、饥饿等问题，当局本身的腐败与失能，也是导致其人民外逃的原因之一。
 <br/>
 <br/>
 《华盛顿邮报》报导，仅上个月美国边境执法人员就逮捕近1.7万名非法移民，创有史以来单月最高逮捕纪录。美国政府在2018财政年度，共羁押约10.8万个无证移民家庭，人数逾39.6万，较2017年多30%。而且目前仍有大量移民正离开洪都拉斯向北挺进，导致美国南部边境告急。
 <br/>
 <br/>
 川普周一在推文中再次提醒美国选民，中期选举事关国家的前途和命运，呼吁民众投票支持共和党而不是造成非法移民威胁的民主党。
