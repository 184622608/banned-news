### 华日:中共欲谈判被拒 美方要求北京拿出具体方案
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】华尔街日报周四（25日）引述一名不具名官员的话称，美方拒绝重启与中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易谈判.html" target="_blank">
  贸易谈判
 </a>
 ，直至北京提出一项具体方案，以解除华盛顿对中共的不满，包括中共强迫外企技术转让以及中美存在的其它贸易争端。
 <br/>
 <br/>
 报导引述多位匿名美国官员的话说，中方曾要求美国财政部副部长马尔帕斯（David Malpass）恢复谈判，但遭到拒绝。马尔帕斯表明，要等到北京提出一份正式提案后再谈。
 <br/>
 <br/>
 不过，中共驻美国大使崔天凯则向《华尔街日报》表示，中方希望在提出具体方案前，先展开进一步协商。目前，双方的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易谈判.html" target="_blank">
  贸易谈判
 </a>
 陷入僵局。
 <br/>
 <br/>
 自9月24日起美中互祭新一波关税以来，美国已对价值2,500亿美元、近6成中国进口商品加征关税，而中方也对美国产的1,100亿美元、超过7成的进口商品加征关税。
 <br/>
 <br/>
 而即将于下月（11月）底举行的阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会上，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 将与中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 第二次会见。但据知情人士透露，川习此次会见只是建立私人关系，并非贸易谈判。
 <br/>
 <br/>
 新闻网站Axios引述知情人士透露，川普总统无意放宽对中国商品的关税。〝他（总统）希望他们（中共）从2000亿美元的中国商品关税中受到更大的压力，〞一位了解川普总统的消息人士表示，总统认为〝关税持续时间越长，他就会拥有更多的（谈判）筹码。 〞
 <br/>
 <br/>
 消息人士还透露，川普与中共的贸易战正处于〝开始的开始〞阶段。川普的团队对总统与习近平下个月在G20峰会期间的会晤，并不抱太大期望。
 <br/>
 <br/>
 报导称，到目前为止，川普经济团队没有对〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞的议程做过实质性的规划，因为这次会议的目的，是川普和习近平重新联系，并在不断升级的贸易战中〝互相感受〞一下。
 <br/>
 <br/>
 知情人士说：〝这是国家元首会议，而不是贸易会议。〞
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
</div>
