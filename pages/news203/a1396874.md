### 福克斯商业网主播连发推文 称炸弹包裹是阴谋论
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】近日，一些可疑包裹接连寄往民主党政要及部分媒体，引发广泛关注。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福克斯商业网.html" target="_blank">
  福克斯商业网
 </a>
 新闻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_主播.html" target="_blank">
  主播
 </a>
 道布斯，25日连发推文称该事件其实有着不为人知的〝阴谋论〞内幕。
 <br/>
 <br/>
 25日上午，道布斯(Lou Dobbs)在推特发文称：〝假新闻，假炸弹。这么多的造假，到底谁可能从中获益呢？〞推文最后还加注了〝让美国再度伟大〞(MAGA)以及〝美国优先〞(America First)、〝道布斯〞( Dobbs)关键字。但推文很快遭删除。
 <br/>
 <br/>
 不久，道布斯又发布第二则内容相近的推文称，假新闻已经成功改变了舆论的方向，原本对于非法移民、边境安全出现漏洞等问题的注意，现在都变成讨论〝可疑包裹〞。不过，稍后道布斯又将第二则推文移除。
 <br/>
 <br/>
 道布斯的推特发文是否触犯员工守则规范，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福克斯商业网.html" target="_blank">
  福克斯商业网
 </a>
 (Fox Business Network)发言人没有发表评论，仅表示道布斯后来已经说明了其立场。
 <br/>
 <br/>
 道布斯在最新推文中称，他将在自己主持的晚间节目中讨论联邦调查局(FBI)、美国菸酒枪炮及爆裂物管理局(ATF)为何没有针对可疑包裹对社会大众发布警告，以及不管这些土制炸弹是真是假，很明显就是为了影响中期选举的。这一切都是为了什么呢？
 <br/>
 <br/>
 周三（24日）上午，美国特勤局表示，截获试图送到白宫和送给前总统奥巴马、克林顿夫妇的可疑包裹，这些包裹里面含有爆炸物。
 <br/>
 <br/>
 当天，美国有线电视新闻网CNN的员工在时代华纳楼下的邮件收发室里，也发现了可疑的爆炸装置，时代华纳的火灾警报响起，指示所有员工撤离建筑物。
 <br/>
 <br/>
 CNN说，这个可疑邮包是要寄给前中央情报局局长布伦南（John Brennan，目前他担任NBC新闻的分析师）的。在纽约市政府的新闻发布会上，纽约警察局局长奥尼尔（James O’Neill）说，到现场处理的官员确认了那个邮包〝看起来像是一个爆炸装置〞，另外还有一个装有白色粉末的信封。
 <br/>
 <br/>
 一名执法的消息人士告诉福克斯新闻，位于马里兰州国会山（Capitol Hill）的安检设施，也截住了一个要寄给加州联邦众议员瓦特斯（Maxine Waters）的可疑包裹，警方已将其送到FBI。
 <br/>
 <br/>
 加州民主党联邦参议员哈里斯（Kamala Harris）的发言人称，有人在哈里斯位在圣地亚哥的办事处附近发现可疑包裹，随后办公大楼的人们被撤离。 这些可疑包裹并不是要寄给哈里斯或她的办公室。
 <br/>
 <br/>
 佛州民主党联邦众议员舒尔茨（WassermanSchultz）及前司法部部长霍尔德（Eric Holder）在佛州的办公室位处同一栋大楼。周三上午一个可疑包裹寄给霍尔德办公室，包裹上的寄件人地址是舒尔茨办公室，因此霍尔德办公室将这个包裹送到寄件人地址，包裹已交给警方处理。
 <br/>
 <br/>
 洛杉矶时报称，24日该报也收到4个可疑信封，警方前往调查证实是虚惊一场。
 <br/>
 <br/>
 周四（25日），美国前副总统拜登（Joe Biden）和好莱坞明星罗伯特‧德尼罗（Robert De Niro）也成为可疑包裹指向的目标。
 <br/>
 <br/>
 可疑包裹事件引发了社会疑虑与担忧。多名共和党要员已相继发声，谴责寄送炸弹邮包的暴力行径。有舆论质疑：恰逢期中选举即将开始之际，这些明显带有恐怖主义色彩的行动，显然是在给共和党和川普添乱，或许这恰恰就是寄送邮包的人挑起事端想达到的目的。
 <br/>
 <br/>
 川普（特朗普）总统周三针对可疑包裹事件明确表示，任何形式的政治暴力在美国都没有土壤。呼吁美国民众团结一致。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
