### 老布什遗体暂置国会 华府各界前往吊唁
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】从周一晚开始，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统和华府各界人士，纷纷前往国会圆形大厅，向刚刚去世的美国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什遗体.html" target="_blank">
  老布什遗体
 </a>
 致敬。
 <br/>
 <br/>
 周一晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统夫妇在前总统老布什的灵柩前致敬。
 <br/>
 <br/>
 副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 也前往致敬，并代表川普总统发表了讲话。
 <br/>
 <br/>
 美国副总统 彭斯：〝在他94年时光中，老布什总统从未惧怕过探险，始终响应为国家效力的召唤。〞
 <br/>
 <br/>
 参议院多数党领袖麦康奈尔追忆老布什生平。老布什自愿服役，并在18岁时成为史上最年轻的海军飞行员之一。在从政后，历经了冷战和苏联解体。
 <br/>
 <br/>
 参议院多数党领袖 麦康奈尔：〝稳健之手，坚守方向，这就是乔治．布什几十年来给予我们的。〞
 <br/>
 <br/>
 前来致敬的还有民主党下任众议院议长候选人佩洛西等多位议员。
 <br/>
 <br/>
 第二次世界大战的退伍军人，现年95岁的前参议院多数党领袖鲍勃．多尔（Bob Dole），也乘作轮椅前来。
 <br/>
 <br/>
 周二，也有不少民众前来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吊唁.html" target="_blank">
  吊唁
 </a>
 老布什总统，其中有小孩，还有亚裔面孔。
 <br/>
 <br/>
 周三上午11点，还将举行追悼会，之后布什总统灵柩将被送回德州。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
