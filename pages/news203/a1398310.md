### 美国中期选举 全球紧张观战 心情各异
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）上台后，强势恢复美国传统价值观，逆转全球政治左倾。11月6日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 ，关系到川普是否将永久改变美国，也事关整个世界的政局，因此受到全世界的紧张关注，各国立场各异。
 <br/>
 <br/>
 美国史上最分裂、投入最多的中期选举，6日开始投票。这次选举将改选美国众议院全部435个议席，以及参议院100席中的35席，还有36个州的州长，也包括各州议会。
 <br/>
 <br/>
 美东地区投票所几乎都在美东时间清晨6时开放，投票时间多为12小时，最后收尾的阿拉斯加州投票所，在美东时间7日凌晨1时才关闭。选举最终结果大约在6日午夜揭晓。
 <br/>
 <br/>
 《金融时报》说，若
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 获胜，将代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 新政获得认可，全世界就将必须在气候变迁、武器管制、难民移民等方面进行实质调整。
 <br/>
 <br/>
 观战的各国，对美国两党选情抱有不同的期待。
 <br/>
 <br/>
 遭川普政府制裁或加征关税的伊朗与中共，热切希望川普遭到打击，从而减低自身受到的压力。
 <br/>
 <br/>
 而川普力图削减美欧贸易逆差、减少北约军费和遏止全球化，也令欧盟不满。许多欧洲国家也希望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 能挽回一局，阻止川普推行新政。
 <br/>
 <br/>
 不过，也有一些国家希望共和党大胜，以证实他们站在历史的正确一方。
 <br/>
 <br/>
 巴西总统当选人波索纳洛表示，川普是他的〝楷模〞。
 <br/>
 <br/>
 即使在欧洲，意大利、匈牙利和波兰等国政府也在帮川普加油。
 <br/>
 <br/>
 沙特阿拉伯希望川普继续帮他们对抗伊朗，以色列同样感激川普帮助抵制巴勒斯坦，印度则盼望川普继续压制中共。
 <br/>
 <br/>
 俄罗斯也不希望民主党取胜后，拿〝通俄门〞做文章，令俄国陷入尴尬境地。
 <br/>
 <br/>
 根据各方预测，共和党在参议院保持优势已无悬念。
 <br/>
 <br/>
 中共党媒6日分析，表面上两党在众议院争夺激烈，但如果抽去美国主流民调中的水分就会发现，民主党要夺回参议院已经希望渺茫。
 <br/>
 <br/>
 美国27个州的提前投票数据显示，共和党扭转了过去在提前投票中都落后于民主党的趋势，今年的共和党选民在关键州提前投票中的表现远超预期，在许多州反而领先民主党。
 <br/>
 <br/>
 前民主党总统候选人桑德斯和民主党全国委员会主席佩雷斯都已表态，不会再有所谓的〝蓝潮〞。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
