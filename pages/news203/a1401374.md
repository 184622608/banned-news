### 川普G20双边会7场变6场 与北京不急于协议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】周四上午，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和夫人启程前往阿根廷参加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 。川普两天的日程中原订有7场双边会晤，不过，与普京的会晤突发变化。请看报道。
 <br/>
 <br/>
 11月30日-12月1日，G20两天会议，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 原计划与7国首脑会晤，但在周四前往阿根廷的途中，川普发推文表示取消与俄罗斯总统普京的此次会面。
 <br/>
 <br/>
 由于上周日俄罗斯在双方共有海域亚速海扣押了乌克兰船舰及官兵且至今仍未释放。川普决定取消与普京的此次会晤，表示待事态解决后再尽快安排会面。
 <br/>
 <br/>
 另一场焦点会面在12月1日晚，川普与习近平的此次会晤，是否能成为美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 至今的一个拐点，备受瞩目。对于是否能达成纲领性协议，川普表示压力在中共一边。
 <br/>
 <br/>
 美国总统川普：〝我们非常接近与中国达成某些实际措施，不过我不一定想那么做。通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 和税收，成百上千亿美元进入美国。所以我还没想好。但我可以告诉你我认为中国想要达成协议。我持开放态度，不过坦白说，我觉得目前的局面不错。〞
 <br/>
 <br/>
 目前中共面对的是2019年伊始，2千亿美元出口美国的商品税率将从10%上调至25%；另外2,670亿美元的商品关税之剑也已悬在头上。
 <br/>
 <br/>
 川普日前透露他极不可能接受中方提出的推迟惩罚性关税的要求。
 <br/>
 <br/>
 在G20前，美国副总统彭斯与习近平在APEC领袖峰会上针锋相对；彭斯直接使用〝冷战〞来警告中共如果不作出实质改变，美国将采取的对策。
 <br/>
 <br/>
 对于美方强调的中国现有的关税及非关税壁垒、知识产权盗窃、公海海域争端甚至侵犯人权等问题，北京会做出怎样的妥协，有待观察。
 <br/>
 <br/>
 另外，川普总统原订与土耳其总统埃尔多安、韩国总统文在寅的正式会晤，白宫周四宣布改为非正式会谈。
 <br/>
 <br/>
 日本首相安倍晋三、德国总理默克尔及东道主阿根廷总统马克里（Mauricio Macri）也将分别与川普会晤。
 <br/>
 <br/>
 除了6场双边会晤，川普与沙特王储萨勒曼（Mohammed Bin Salman）将如何互动也颇受关注。
 <br/>
 <br/>
 《华盛顿邮报》记者卡舒吉（Jamal Khashoggi）遇害后，目前在如何对待沙特问题上，川普内阁与部分立法者立场不同，一些参议员希望通过撤销在也门的军事支援来制裁沙特；川普此前一直表示他不能不考虑盟友沙特在〝美国优先〞战略上的角色。
 <br/>
 <br/>
 国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭培奥.html" target="_blank">
  彭培奥
 </a>
 与国防部长马蒂斯周三在参院做简报，彭培奥表示现在参议员提出的制裁方案影响国家安全利益、破坏外交努力。
 <br/>
 <br/>
 美国国务卿彭培奥：〝我们认为参议院正在商讨的内容不合时宜。现在我们正处于关键时刻，美国特使马丁格里菲斯12月将汇集各相关方，希望达成也门停火协议，这是我们在外交上争取了几个月的事情，现在我们正处在关键节点上。〞
 <br/>
 <br/>
 美国国防部长马蒂斯：〝我们没有证据显示王储涉及其中。〞
 <br/>
 <br/>
 为保障G20会议安全进行，马克里将30日定为法定假日。
 <br/>
 <br/>
 新唐人记者李兰综合报导
</div>
