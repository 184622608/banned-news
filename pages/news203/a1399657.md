### 加州野火已66人遇难 天堂镇烧成〝地狱〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州野火.html" target="_blank">
  加州野火
 </a>
 肆虐，把原本繁华美丽的小镇，变为废墟，周五，因此死亡的人数，也攀升到66人。
 <br/>
 <br/>
 比尤特县警察局长Kory Honea：〝我再次非常遗憾的汇报，我们又找到了另外7具遗体。〞
 <br/>
 <br/>
 警方周四晚公布了新发现的7名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罹难者.html" target="_blank">
  罹难者
 </a>
 。至此，北加州〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎普野火.html" target="_blank">
  坎普野火
 </a>
 〞已导致63人死亡，加上在南加州〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伍尔西大火.html" target="_blank">
  伍尔西大火
 </a>
 〞中丧生的3人，整个加州已有66人因野火死亡。
 <br/>
 <br/>
 比尤特县警察局长Kory Honea：〝此外，我们还在搜寻的失踪人数已经增加到了631人，此前这一数字为501人。〞
 <br/>
 <br/>
 包括志愿者在内的数百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_搜救.html" target="_blank">
  搜救
 </a>
 人员，仍在不停的寻找失踪人员及遇难者遗体。
 <br/>
 <br/>
 由于一些遗骸被严重烧毁，已无法通过DNA来识别身份，专家不得不通过牙科记录，外科手术记录等辅助方式来确认遇难者身份。
 <br/>
 <br/>
 加州州立大学人类学教授Jesse Dizard：〝我们可能永远都无法识别有些被发现的遗骸，但我和我的同事们希望在可能的情况下，给那些不幸罹难者的家属提供一些有意义的结果，〞
 <br/>
 <br/>
 这场加州史上最致命的野火，将原本风光秀美的〝天堂镇〞变成了人间地狱，整片的居民区被夷为平地，除了黑色灰烬，再看不到其它颜色。
 <br/>
 <br/>
 至今为止，在〝坎普野火〞中被烧毁的建筑高达8700栋，超过220平方英里的土地化为灰烬。
 <br/>
 <br/>
 受野火影响，旧金山湾区也笼罩在浓重烟雾之中，空气质量严重恶化。自周四开始，大部分学校宣布临时关闭，市民外出时也带上口罩，避免吸入空气中的有害物质。
 <br/>
 <br/>
 美国总统川普（特朗普）计划于周六前往加州，探望野火灾民。
 <br/>
 <br/>
 白宫顾问康威：〝我相信他会感谢那些勇敢奔赴火场，在坎普野火中将人们从家中救出来的救援人员，当然也会探望那些失去至亲的人们，也许还会与一些官员会面。〞
 <br/>
 <br/>
 新唐人记者乔安综合报导
</div>
