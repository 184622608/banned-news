### 遭奥巴马妻新书攻击  川普不屑:为了卖个好价钱而已
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】美国前总统奥巴马的妻子米歇尔在即将出版的新书中，提及有关奥巴马出生地的争议，并籍此对现任美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）进行了猛烈攻击，誓言自己永远不会原谅川普。川普则在上周五公开回应称，米歇尔不过是为了买书多挣点钱而在书里炒作争议话题。
 <br/>
 <br/>
 米歇尔‧奥巴马(Michelle Obama)的自传《蜕变》(Becoming)即将出版上市。在这本书中，米歇尔对现任美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 进行了强烈的抨击。
 <br/>
 <br/>
 据日本放送协会（NHK）周一的报导，米歇尔‧奥巴马在她的新书中指称，川普当年〝散播〞奥巴马并非出生于美国本土的传言，导致她的家庭面临〝安全威胁〞。米歇尔并声称〝永远不会原谅川普〞。
 <br/>
 <br/>
 米歇尔还在这本书中声称，川普多次发表了〝蔑视女性〞的言论，自己每念及至此都气得〝浑身发抖〞。
 <br/>
 <br/>
 11月9日，川普在动身前往巴黎前经过白宫南草坪时，有媒体记者针对米歇尔的上述言论提出的质询。〝一定是为了回忆录卖个好价钱，有好销路，才这么炒作吧。〞川普答覆说：〝她（米歇尔）写书赚很多钱，他们总是希望书里要有一些争议内容。〞
 <br/>
 <br/>

   </font>

 </center>
 <br/>
 <br/>
 〝说到安全问题，我也拿一些争议内容做为回敬，我永远不会原谅奥巴马对我们美军的所作所为。（在奥巴马治下）军队被耗尽，我现在必须重新进行整顿。〞川普进一步回击道：〝奥巴马对美军所做的那些事，对这个国家所有人来说，都变得非常不安全。〞
 <br/>
 <br/>
 川普针对奥巴马的上述批评，被认为是暗指奥巴马在任期间批准的2011年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_预算控制法案.html" target="_blank">
  预算控制法案
 </a>
 (Budget Control Act of 2011)。该法案曾引发的巨大争议，反对者认为法案中对美国军事支出预算进行深层次削减等内容会对美国军队造成严重伤害。
 <br/>
 <br/>
 另据公开的资讯，米歇尔‧奥巴马这次与书商签订的出书合约利润非常可观，合约内容包括：米歇尔与她的丈夫奥巴马两人都将各自出版一本新书，为此，书商必须预先向奥巴马夫妇支付6500万美元的酬劳。奥巴马自传据称预定于2019年发行。
 <br/>
 <br/>
 （记者何雅婷综合报导）
