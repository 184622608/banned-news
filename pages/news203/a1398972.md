### 美国失业率创新低招工难 改请银发族〝出江湖〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】自美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）上任以来增加了近60万个制造业工作，失业率创50年来最低点。不过，这也让一些速食连锁店面临缺工的窘境，业者为解决此难题出新招，改变以往招募对象年龄层，从主打青少年员工改为招募银发族。
 <br/>
 <br/>
 今年第二季，美国经济创下近4年来最快增速，国内生产总值(GDP)环比增长4.1%，是2014年第三季以来的最佳表现。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 10月31日在白宫与工人代表会谈，并向与会者介绍〝对美国工人承诺〞计划的进展。他说，目前美国的失业率为3.7%，是50年来的最低点，〝自大选以来，我们增加近60万个制造业工作，而且还要比这多得多。〞
 <br/>
 <br/>
 美国国内经济增长强劲，就业机会增加，让美国人有了更多的选择。但同时，一些速食业者也因招不到人，不得不跳到第一线帮忙。
 <br/>
 <br/>
 据《地球突击队》报导，纽约一家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Subway.html" target="_blank">
  Subway
 </a>
 业主卡普洛（Jeffrey Kaplow）表示，为了补足人力，他曾尝试不同方法来招聘员工，包括在网站上登广告以及问其他倒闭的Subway分店员工，是否愿意转职。但尽管如此，仍找不到足够人力，无奈之下他也只好上到第一线帮忙。
 <br/>
 <br/>
 而在北卡罗莱纳州拥有3间Subway加盟店的老板米勒（Keith Miller）也遇到过同样问题。他表示，以往寒暑假都会有很多青少年前来应征，但这两年所收到的履历数量都却变得很少，好像是这些青少年暑假都不出来打工一样。
 <br/>
 <br/>
 为了解决缺工问题，这些速食业者想到另一人群，那就是银发族。此一改变，不但解决缺工问题，还给银发族提供了重新投入职场的机会。
 <br/>
 <br/>
 聘雇银发族群的好处多多。包括业者只需要用支付给年轻人的薪水，就能得到有工作经验的银发员工；而且这些银发员工比年轻人稳定，使公司流动率降低；另外，银发员工待人处事技巧，也较年轻人好。
 <br/>
 <br/>
 而对于这些银发族来说，退休后因缺乏生活重心，让他们感觉自己变老，似乎无所事事，因此重出江湖对他们来说可能是个不错的选择。
 <br/>
 <br/>
 报导称，在德州炸鸡店任职的63岁的威廉姆斯（Stevenson Williams）表示，退休生活让他感觉自己老了，他很高兴能在炸鸡店工作，他表示很喜欢这个工作气氛，也很享受和顾客的互动。威廉姆斯从原本只能做些清扫和洗碗的工作已经升级为分店经理。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
