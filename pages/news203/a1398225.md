### 2018中期选举为什么重要
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】今天的选举和每四年的普选一样，之所以被称为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 〞，是因为这样的选举发生在此届的美国总统的任职中间时期。所以，虽然候选人追逐的都是立法院的席位或者州长等地方官职，但是美国人普遍认为这次投票实际上是一次对那个没有出现在候选人名单上的名字——现任美国总统的公投。
 <br/>
 <br/>
 而本次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 又和历次同类选举不一样，它对美国人以及纽约州人意义更加重大，理由如下：
 <br/>
 <br/>
 根据《财富》杂志及《福克斯》新闻的综合报导，11月6日的选举要选出一个新的联邦众议院以及三分之一的参议院席位，这个结果将决定2020年的总统选举力量构成以及2022年国会的重新划分，甚至决定了一直到2030年的美国立法院的力量分配。也就是说，这次选举将决定参众两院是不是将被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的共和党主导，或者相反，将由民主党取得至少对其中一个立法院的领导地位。如果民主党占上风，总统的很多政策无疑会遭到制肘。
 <br/>
 <br/>
 《华盛顿邮报》分析，周二（6日）的普选关系到美国的医保和移民政策，关系到经济，关系到华盛顿和各州的权利交替。虽然上周五的劳工部的报告显示，在川普执政的两年期间，美国经济不论在就业率的提高、工资的增长、失业率的降低和劳动力的参与方面都呈现出〝好得不能再好的形势〞，但是两党选民对川普的态度以及各自向对方转变的不确定性，都〝注定了这次选举是一代人才有一次的事情〞。
 <br/>
 <br/>
 对于各地方州的政策而言，本次中期选举将决定堕胎法、医保、教育、税收以及劳动法等的命运。在纽约州，根据《VOX》媒体的综合分析，民主党目前以104对41票的位置稳操州众议院的主动权；而在州参议院，因为一个民主党和共和党合作，造成共和党仅以一席之差占领导地位，现在共和党参议员与民主党参议员的比例为32比31。所以，如果选举中由另一个民主党候选人替换掉一个共和党的话，民主党就会占领参议院，从而把持纽约州的两个立法院。
 <br/>
 <br/>
 所以，上述媒体把纽约州列为本次中期选举中九个最重要的州之一。
 <br/>
 <br/>
 根据州选举委员会的数据，纽约州登记为民主党的选民人数为580万，共和党为260万。但是4日的一份最新锡耶纳大学的民调显示，更多人认为纽约州现在发展的方向不对（47%比40%）；而且，在过去的八年中，对民主党州长库默持不满意态度的选民人数第一次超过了赞成他的人数（50%比46%）。
 <br/>
 <br/>
 所以，纽约人在这次普选中怎么投票，将决定州参议院花落谁家，在某种程度上也等于决定自家的命运。
 <br/>
 <br/>
 ──转自《大纪元》
