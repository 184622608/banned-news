### 美国航母抵达香港 设禁飞区五天
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】美国航空母舰里根号战斗群，周三抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 。外界认为，中方同意美方军舰停靠香港，意味美中紧张关系有机会缓和。
 <br/>
 <br/>
 航空母舰〝里根号〞刚结束美国海军在菲律宾海域的军事演习，停靠
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 青衣海域。〝里根号〞配备约70架军机，还有三艘军舰随行。
 <br/>
 <br/>
 香港民航宣布，21日到25日，青衣对开和尖沙咀渡轮码头一带，将设立临时飞行限制区。
 <br/>
 <br/>
 美国海军少将托马斯表示，希望藉此机会增进了解。
 <br/>
 <br/>
 美国海军少将 KARL O.THOMAS：〝要了解人民，最好是成为其中一员，能够四处走一走，所以我认为，好处是有机会看到一些优秀的年轻船员，出去玩玩，有所互动。〞
 <br/>
 <br/>
 北京允许美国海军军舰靠港访问，引发外界猜测，美中关系是否进一步缓和。上个月，北京拒绝了美国两栖攻击舰黄蜂号停靠香港。
 <br/>
 <br/>
 11月底，川普和习近平将在阿根廷20国集团（G20）峰会期间会面，讨论中美贸易议题。
 <br/>
 <br/>
 新唐人记者易文、明玉综合报导
