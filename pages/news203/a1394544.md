### 美10年来最严重车祸 短讯显示车有问题
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】上周六（10月6日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 州发生美国近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_10年.html" target="_blank">
  10年
 </a>
 来最严重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_车祸.html" target="_blank">
  车祸
 </a>
 ，一辆加长豪华轿车撞车，造成20人死亡。一名遇难者的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_短讯.html" target="_blank">
  短讯
 </a>
 显示，该车在事发前已显现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_问题.html" target="_blank">
  问题
 </a>
 ，但最终没能在下坡的路口停住，导致车上人员无一幸免。
 <br/>
 <br/>
 上周六下午2点左右，在位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 上州奥伯尼（Albany）以西30多英里外的斯科哈里县（Schoharie County）的纽约州30号高速公路（State Route 30）上，一辆载有多名乘客的加长轿车在高速下坡时与另一辆停在附近的空车相撞，造成20人死亡，酿成近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_10年.html" target="_blank">
  10年
 </a>
 来美国最重大的交通事故。
 <br/>
 <br/>
 出事的2001款福特豪华游览轿车其实是临时替换的，因为车上乘客所租借的大巴车在到达目的地前出现了故障，但没想到车上的遇难者在劫难逃，在换车后，最终还是没能免于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_车祸.html" target="_blank">
  车祸
 </a>
 。
 <br/>
 <br/>
 在20名死者中，18人死于车内，2名是路人。
 <br/>
 <br/>

 美国国家运输安全委员会（NTSB）主席苏瓦特（Robert Sumwalt）表示，这是美国自2009年2月以来最致命的一起交通事故。
 <br/>
 <br/>
 一名名叫瓦莱丽．阿贝林（Valerie Abeling）的妇女告诉《华盛顿邮报》（The Washington Post），她34岁的侄女麦高恩（Erin McGowan）及其6月份才婚配的丈夫均死于车祸。
 <br/>
 <br/>
 阿贝林说，她的女儿事发前收到麦高恩的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_短讯.html" target="_blank">
  短讯
 </a>
 ，说他们所乘坐的那辆车情况似乎很糟糕，20分钟后，车上的所有乘客便全部死于车祸。
