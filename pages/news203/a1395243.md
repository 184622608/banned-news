### 美首席中国问题权威：川习会前提是北京重大让步
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】美媒称中美首脑计划在11月底举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_20国峰会.html" target="_blank">
  20国峰会
 </a>
 期间会晤。这让外界在持续升温的中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 中，似乎看到了一线缓和的迹象。但美国首席中国问题权威披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 的前提是，中方必须在贸易议题上做出实质性让步。
 <br/>
 <br/>
 华尔街日报12日报导，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）和中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，计划在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_20国峰会.html" target="_blank">
  20国峰会
 </a>
 期间举行会晤。
 <br/>
 <br/>
 20国集团会议将于11月30日至12月1日，在阿根廷布宜诺斯艾利斯举行。中方希望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晤，对中美贸易关系产生积极的影响。此次川习会的中方负责人是中共国务院副总理刘鹤。
 <br/>
 <br/>
 对此，外界认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系.html" target="_blank">
  中美关系
 </a>
 在激烈对抗中，似乎又现出一线缓和的希望。
 <br/>
 <br/>
 但中国战略研究中心主任白邦瑞(Mike Pillsbury)，在接受福克斯财经频道(Fox Business)采访时表示，川普同意与习近平会晤的主要原因，是中方有望在贸易问题上做出一些重大让步。
 <br/>
 <br/>
 他说：〝我所得知的一些消息是，中方正在研究一套方案，他们试图了解川普总统的底线在哪儿，但显然还不确定。〞
 <br/>
 <br/>
 白邦瑞是川普总统〝首要中国问题权威〞的保守派智库——哈德逊研究所(Hudson Institute)中国战略研究中心主任。
 <br/>
 <br/>
 白邦瑞表示，中方正在努力，但目前还没有满足美方的要求。他对川习会是否能取得突破，抱谨慎乐观态度。他说这将取决于中方是否会做出重大让步。
 <br/>
 <br/>
 正在印度尼西亚巴厘岛出席国际货币基金组织(IMF)会议的美国财长姆努钦，对美联社表示，要确保11月份的川习会能够成行，仍有很多工作要做。
 <br/>
 <br/>
 美国国家安全顾问博尔顿（John Bolton）12日在播出的电台访问中说，中共在贸易、国际事务、军事和政治领域的行为需要调整，两国领袖11月底会面期间，将讨论这些问题。
 <br/>
 <br/>
 博尔顿还批评中共在南海的行为危险，强调美国会捍卫国际海上通道的开放。9月底，中共驱逐舰在南海南薰礁附近驶近美国海军驱逐舰狄卡特号，几乎相撞。
 <br/>
 <br/>
 对于中共经济及军事的扩张，博尔顿说，中共是违反国际贸易和商业规范得来的。如果他们不被允许窃取美国的技术，他们的军事能力将会大大降低。今天大家看到的很多紧张局势将会减少。
 <br/>
 <br/>
 他还说，川普总统相信中共长期从国际秩序中占便宜，过往少有美国人出来阻止，现在是时候要做了。
 <br/>
 <br/>
 川普在接受福克斯新闻采访时表示，针对中国贸易，美国还有更多招数。他说，中共好日子过得太久了，〝我估计他们认为美国人是傻瓜。美国人不是傻瓜。〞
 <br/>
 <br/>
 白邦瑞对福克斯表示，川普总统的理想目标是在互惠的原则下，实现美中贸易的平衡，并大幅增加美国就业。
 <br/>
 <br/>
 他说，中方可能会出于面子，既不承认，也不解决美方提出的强制技术转让和盗窃知识产权问题。但如果中方能够提出在两年内，连续大幅削减美中贸易赤字的方案，并有助于美国增加就业，川普总统才可能会接受并就此给美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 暂时划上句号。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：赵云)
</div>
