### 飓风迈克尔过后 救灾恢复 川普查看灾情
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】首先来关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 消息。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 飓风退去，受灾严重的佛罗里达州狭长地带，清理修复工作正在展开。周一上午，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和第一夫人梅拉尼亚赶赴佛罗里达和乔治亚州，查看
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灾情.html" target="_blank">
  灾情
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 过后，很多居民返回家中，开始清理毁坏的房屋。
 <br/>
 <br/>
 为搜索和救助被困灾民，直升机、重型设备和搜索犬都投入救援工作。佛州州长斯科特表示，目前有1700名搜救人员，在搜查灾区内两万五千所房屋。
 <br/>
 <br/>
 佛罗里达州州长 斯科特:〝我的想法是，搜救工作第一，我们还在做这件事。第二，要确保我们有食物和水，以及燃料。我们正在做。〞
 <br/>
 <br/>
 〝第三，我们要给所有郡县提供发电机，确保交通指示灯恢复正常，即时那里的电力还没有恢复。〞
 <br/>
 <br/>
 斯科特还表示，现在电力公司有一万七千名工人在工作，灾区逐渐恢复电力。但是，互联网和手机通讯还没恢复。
 <br/>
 <br/>
 受灾严重的墨西哥海滩小镇，约有1000名居民，至少有250人选择留下，目前有46人下落不明。
 <br/>
 <br/>
 联邦紧急事务管理署管理员 Brock Long：〝很多生命损失是可以避免的，可以不发生。这是我们最痛心的。我们可以重建被飓风毁坏的一切，不管多长时间。但我无法让你复活。〞
 <br/>
 <br/>
 联邦紧急事务管理署表示，搜救工作过后，还需要很长时间帮助居民修缮房屋。
 <br/>
 <br/>
 联邦紧急事务管理署管理员 Brock Long：〝最艰巨的任务不仅是要抢救生命，帮助数千避难所里的居民找到安居之地，最艰难的长期任务就是修复房屋。需要好几个月。〞
 <br/>
 <br/>
 目前，迈克尔飓风已造成至少20人死亡。随着搜救工作的延伸，这一数字可能还会攀升。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统和梅拉尼亚周一赶往灾区，查看
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灾情.html" target="_blank">
  灾情
 </a>
 。
 <br/>
 <br/>
 新唐人记者安心、宇微综合报导
