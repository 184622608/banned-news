### 电线故障酿加州山火？如属实PG&E将赔68亿恐关门
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】南北加州两场山火持续燃烧了一周，死亡人数已增至66人，另有631人失踪，是加州大火史上死亡人数最多的一次。在两场大火分别发生之时，有2家供电公司都在原始起火点有故障报告纪录。相关部门正在对其调查，如调查属实，两家电力公司恐支付高额罚款。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太平洋瓦电.html" target="_blank">
  太平洋瓦电
 </a>
 （PG&amp;E）11月9日披露了一则在坎普野火（Camp Fire，又译〝野营大火〞）发生前15分钟，该公司向加州公共设施委员会发出的巴特县（Butte County）有一条输电线路发生故障的报告，故障发生之处正好位于加州防火局确定的火灾始发地。
 <br/>
 <br/>
 巴特县地检长迈克．拉姆齐（Mike Ramsey）11月9日表示，地检署正在与加州防火局讨论保存火灾现场证据的事宜，这些证据有可能会用于刑事调查。
 <br/>
 <br/>
 根据《圣荷西水星报》获得的证据，在11月8日火灾发生之前，火源附近的小镇帕尔加（Pulga）的一位小商业主贝特西．安．考利（Betsy Ann Cowley）在火灾发生前接到供电局的电子邮件通知，说太平洋瓦电公司正在处理附近的一处输电线路火花故障。
 <br/>
 <br/>
 据湾区新闻集团（Bay Area News Group）监控的无线电，以及向加州监管机构发出的一则警告，均显示在火灾被首次报告前15分钟左右，高压输电线路发生危险。在距离帕尔加东北1英里处，靠近〝坡〞大坝（Poe Dam）的火灾初发现场，当消防队员们抵达后，发现有掉下来的高压线，火灾沿着高压线下面迅速蔓延。
 <br/>
 <br/>
 虽然调查结果还未出炉，但火灾受害者代理律师团，11月13日已向旧金山高等法院对太平洋瓦电提出起诉，指控该公司失职，对电力设备、高压电线管理不善，造成高压输电线路故障，而引发了这场加州有历史纪录以来破坏力最强的火灾。
 <br/>
 <br/>
 目前具体索赔金额尚未确定，但索赔将涵盖受害者包括房屋、财产、个人物品等损失，也包括撤离、安置的费用，以及受害者的心灵创伤等。
 <br/>
 <br/>
 到11月15日，坎普野火已至少导致63人死亡，直接经济损失估计达190亿美元。
 <br/>
 <br/>
 太平洋瓦电发言人表示，他们正在审查考利的证词。太平洋瓦电在提交给美国证券交易委员会（SEC）的文件中显示，他们已将2018年8月1日至2019年7月31日的山火责任保险理赔金额调高至14亿元，而且已经从循环信贷借得34亿6000万元现金。
 <br/>
 <br/>
 但据穆迪投资公司（Moodys）本周提出的初步估算，此次大火造成的损失高达68亿元，太平洋瓦电投保的理赔金不足以负担全部损失，更何况还有法律诉讼费和罚金等，要想走出困境，估计可能还需要加州政府再度伸出援手。
 <br/>
 <br/>
 由于对2017年的火灾调查已确认，太平洋瓦电电力设施是导致一些火灾的原因，其中包括酒乡纳帕（Napa）附近的一些大火。太平洋瓦电已面临数十亿美元的赔偿诉讼，但今年加州议会推出有争议性法案，允许该公司将2017年火灾赔偿诉讼由用户分摊，以避免公司破产，但只限于2017年的火灾赔偿诉讼。
 <br/>
 <br/>
 若调查确定坎普大火也是由于电力设施导致，该公司恐面临更大数额的赔偿，恐无力支付。
 <br/>
 <br/>
 太平洋瓦电公司股票持有者从11月9日起开始大规模抛售该公司股份，PG＆E的股价已下跌近50％。
 <br/>
 <br/>
 11月12日，加州公共设施委员会（CPUC）正式对供电公司发起调查，除北加州的太平洋瓦电外，可能导致南加州大火的供电公司〝南加州爱迪生〞（Southern California Edison）也在调查之列。
 <br/>
 <br/>
 据CPUC称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱迪生电力公司.html" target="_blank">
  爱迪生电力公司
 </a>
 于上周四（8日）告诉CPUC，在伍尔西山火起火点附近的变电箱发生〝回路〞，检测到不寻常电流，2分钟后，伍尔西山火开始燃烧。
 <br/>
 <br/>
 CPUC发言人特丽．普罗斯帕（Terrie Prosper）表示，调查将重点审查电力设施是否有不符合规范之处，以及是否符合火灾危险区域所要求的规范。普罗斯帕说，除了在加州防火局（Cal Fire）允许下赴火灾起点调查外，其它方面的调查内容还包括设施维护、周围的植被控制状况，以及紧急状况防范和应对等。
 <br/>
 <br/>
 就在1个多月前，南加爱迪生公司称，托马斯大火至少有一处起火点的点燃与该公司的设备有关。托马斯大火于去年12月4日在文图拉县和圣芭芭拉县爆发，造成28.2万英亩土地及1,063幢建筑被烧毁。
