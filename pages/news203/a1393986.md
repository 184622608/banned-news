### 民调：美中贸易战有助川普争选票
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】美中贸易战正如火如荼展开，双方经过四轮协商，依然无果。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国总统.html" target="_blank">
  美国总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）近日更直指中共欲干预美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，攻击他的贸易政策。但民调显示中美贸易战可能有助于川普争取选票。
 <br/>
 <br/>
 11月6日，美国将举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，不仅要改选众议院所有435名众议员和参议院33名参议员，还要改选36个州的州长以及其它大小地方选举。
 <br/>
 <br/>
 在中期选举中，如果共和党人能保持国会控制权，将有利于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府在移民、减税、制定新的贸易协议，以及投资基础设施等方面政策的实施。所以，中期选举的结果将不仅影响美国的未来、世界的未来，而且对中国的未来也会有很大的影响。
 <br/>
 <br/>
 美中贸易战正如火如荼展开，双方经过四轮协商，依然无果。
 <br/>
 <br/>
 9月26日，川普在主持联合国安理会会议时，直言不讳地指责中共一直试图干预美国即将举行的中期选举。川普说，他是第一位挑战中共贸易政策的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国总统.html" target="_blank">
  美国总统
 </a>
 ，因此中共〝不想要我或我们（共和党）赢〞。
 <br/>
 <br/>
 而据民调显示，中美贸战更有助川普争取选票。以位于美国中西部的威斯康辛州为例，该州是国会中期选举两党必争之地，当地正受到来自亚洲制造业的挑战，尤其是生产线搬至中国。
 <br/>
 <br/>
 在2016年总统大选时，川普以高出1%的选票胜出当时的民主党总统侯选人希拉里。
 <br/>
 <br/>
 8月份，由波士顿萨福克大学所做的民意调查显示，威斯康辛州只有31%受访者支持川普向〝盟友〞欧盟、加拿大、或墨西哥开征关税，但却有39%支持对中共加关税，其中82%是川普的支持者。
 <br/>
 <br/>
 有分析认为，川普若要留住支持者，就不能对中共放软手脚。
 <br/>
 <br/>
 另外，哈佛大学美国政治研究中心(Center for American Political Studies, CAPS) 和哈里斯民调公司(Harris Poll)合作，在9月26日-27日和29日-30日分两个阶段进行了民意调查，抽样受访者人数分别为1228人和1330人。
 <br/>
 <br/>
 该民调是在卡瓦诺(Brett Kavanaugh)大法官提名听证会前后分别进行的调查，综合调查结果的数字显示，目前川普的支持率为46%。
 <br/>
 <br/>
 民调显示，在经济政策上，57%的美国人对川普的经济方针以及在创造新就业岗位所实施的政策表示支持，另有56%的受访者对川普打击恐怖主义的作为表示支持。
 <br/>
 <br/>
 川普从9月29日开始在美国5个州进行了为期一周的集会活动，旨在中期选举前夕站台支持共和党候选人，确保共和党人守住国会多数党优势。
 <br/>
 <br/>
 川普表示，他上任以来，美国失业率创历史新低，更多美国人有了工作。新签署的美国-墨西哥-加拿大贸易协定，是美国取得的又一个胜利，让美国人得到公平贸易，以及更多工作机会。
 <br/>
 <br/>
 他呼吁民众在11月的中期选举中，都出来投票。
 <br/>
 <br/>
 (记者李红报导/责任编辑：范铭）
