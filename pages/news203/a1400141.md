### 前瞻G20峰会 美中或难缓解经济紧张关系
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 日前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚太经合组织峰会.html" target="_blank">
  亚太经合组织峰会
 </a>
 上，针对中共不公平贸易政策展现强硬立场。专家认为，月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 ，美中无论达成何种协议，总体上，紧张的经贸关系还将持续。
 <br/>
 <br/>
 过去一周，从东盟及东亚峰会，再到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚太经合组织峰会.html" target="_blank">
  亚太经合组织峰会
 </a>
 ，美中之间在经贸、投资及安全问题上，分歧明显。
 <br/>
 <br/>
 美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 措辞严厉，批评中共的不公平贸易政策，要求中方做出改变。
 <br/>
 <br/>
 美国副总统彭斯：〝美国不会改变做法，直到中方改变其行为方式。〞
 <br/>
 <br/>
 北京也用强硬措辞进行回击。
 <br/>
 <br/>
 新美国安全中心亚太安全项目高级主任Patrick Cronin：〝技术上来讲，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC.html" target="_blank">
  APEC
 </a>
 太有吸引力了，美中都无法忽视，要显示其坚定信念，说明我们不会退缩。〞
 <br/>
 <br/>
 月底
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间，川普与习近平将举行会谈。上周，北京终于应美方要求，书面提交了142项贸易改革清单。但川普指出，中方仍然漏掉了4或5个重要项目。媒体披露，G20峰会前，美中贸易官员还将在阿根廷进行谈判，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 铺路。
 <br/>
 <br/>
 新美国安全中心亚太安全项目高级主任Patrick Cronin：〝将会是某方面、部份性的协议。不一定会是中共喜欢的那种，大的框架、原则性协议，其中问题都和谐处理。这不大可能是美国的方式。因为川普总统在这些问题上，更注重交易性。他会继续对经济关系的很多方面不满意。〞
 <br/>
 <br/>
 专家预计，G20峰会期间，不少国家将公开支持美国公平、对等、互惠的贸易原则，而中共也将拉拢一些国家对美国施压。
 <br/>
 <br/>
 新美国安全中心亚太安全项目高级主任Patrick Cronin：〝中共认为美国虚弱，会在压力下屈服，所以重要的是美国要强硬，并坚持强硬，即使其它国家有担懮。〞
 <br/>
 <br/>
 专家认为，无论美中在G20峰会上达成何种协议，双方总体上在经贸领域的紧张关系还将持续。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
