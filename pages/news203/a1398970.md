### 外太空看加州野火 天堂镇成火海 滚滚浓烟向西飘
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】连日来美国加州多处爆发的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火.html" target="_blank">
  野火
 </a>
 ，导致死亡人数升至25人，25万人逃离家园。美国国家航空暨太空总署（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_NASA.html" target="_blank">
  NASA
 </a>
 ）近日发布卫星空拍图显示，北部巴特县的天堂镇已成为火海。
 <br/>
 <br/>
 美国加州南北共有3股大火狂烧，除了〝坎普
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火.html" target="_blank">
  野火
 </a>
 〞（Camp Fire）肆虐天堂镇（Paradise）北部布巴特县（Butte County）外，还有位于南部文图拉县（Ventura County）的〝伍尔西大火〞（Woolsey Fire）和〝希尔大火〞（Hill Fire）。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_NASA.html" target="_blank">
  NASA
 </a>
 的8号卫星于8日上午10时45分空拍的坎普野火图像显示，天堂镇燃起红橙的火焰，火势不断向南延烧，成为一片火海。到了9日NASA则以Terra卫星空拍整个加州，这次还拍到南方的希尔野火与伍尔西野火，滚滚浓烟不断向西飘散。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/11-12/p9122172a830297909.jpg" target="_blank">

   <br/>

   </font>
  </a>
 </center>
 <br/>
 大火从11月8日早上6:30左右开始，在加州北部巴特县（Butte County）和南部的文图拉和洛杉矶县周遭地区不断燃烧，连续3天火势不减，过火面积不断扩大，到10日已达到425平方公里，一些社区被夷为平地，摧毁建筑物超过6700处。
 <br/>
 <br/>
 目前，南、北加州合计共有至少25人遇难，随着调查人员到达被野火摧毁的地区，死亡人数或再增加。另外有25万多人被疏散。气象学家警告，强风和干燥天气导致的危险火势，可能要一直持续到下周才能完全控制。
 <br/>
 <br/>
 加州迄今已投入至少810万美元灭火。加州当局表示，自8日以来，已投入数千消防员全力救火，动用了灭火专用飞机、铲土机等大型机械，更多的消防员将前往火灾地区。天气预报显示，到周一风速将达到每小时80公里，这增加了火势进一步扩散的风险。
 <br/>
 <br/>
 美国林业局的消防队长史密斯表示：〝我们最大的担心就是风，风势增长的地方对火情真的很有影响，我们尽可能控制火势。〞
 <br/>
 <br/>
 与此同时，山火已造成严重空气污染，多个城市开始向市民发放面罩，以减少污染对人体健康造成的危害。
