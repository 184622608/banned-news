### 大火肆虐北加州 9死25万人撤离 强风持续至下周
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月11日讯】美国加州北部自周四（8日）起遭遇多起大型野火肆虐，至少9死多人受伤，近25万人紧急撤离。川普（特朗普）总统周五（9日）宣布联邦政府将向灾区提供资助。他周六（10日）发推文称，起火原因是森林管理严重不善，需立即补救。
 <br/>
 <br/>
 加州北部和南部均发生多起大火，目前有3起野火正在蔓延，除了〝坎普野火〞（Camp Fire）肆虐天堂镇（Paradise）北部巴特县（Butte County）外，还有位于南部文图拉县（Ventura County）的〝伍尔西大火〞（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Woolsey Fire.html" target="_blank">
  Woolsey Fire
 </a>
 ）和〝希尔大火〞（Hill Fire）。
 <br/>
 <br/>
 川普周四宣布加州进入紧急状态，联邦政府将向灾区提供资助。周六在巴黎访问的川普发推文说，除了森林管理非常糟糕外，加利福尼亚州没有理由发生这些〝大规模的、致命的和昂贵的〞森林大火。每年都有数十亿美元拨出救灾，有这么多人丧失生命，这些都是森林〝严重管理不善〞造成。现在需要补救，否则不会再有更多的联邦经费！

 <h4>
  火势1天增大4倍
 </h4>
 <br/>
 〝坎普野火〞在周四开始燃烧，到周五时火势已经增大了4倍。当地政府官员说，只有5%的火势得到控制。当局在周四就已下令全县强制撤离。
 <br/>
 <br/>
 巴特县当局表示，遇难者大多是加州首府萨克拉门托以北145公里的天堂镇的居民，他们在试图逃离大火时被烧死在汽车里或汽车旁边。当局在一架焚毁的车辆里发现5具遗体。
 <br/>
 <br/>
 加州森林防火厅(Cal Fire)发言人麦克连(Scott McClean)说：〝我们只发现5人死亡，是因为他们还困在路上。〞
 <br/>
 <br/>
 因野火来得太快，镇上多处交通受阻，警方与消防队无法逐一清查受灾房屋或车辆内是否还有人伤亡。
 <br/>
 <br/>
 巴特县治安官Kory Honea表示，据报还有35人失踪。大火烧毁了巴特县的90,000英亩土地，至少有52,000名居民被迫撤离家园。

 <br/>
 <br/>
 55岁的天堂镇中学老师凯斯勒(Marc Kessler)表示，〝天空都变黑了，你搞不清楚是白天或晚上，天上降下好多烟灰，然后到处都起火了。〞

 <h4>
  伍尔西大火威胁马里布
 </h4>
 <br/>
 洛杉矶西北方另一场山火，则继续向南加州名人聚居的高级海滨小镇马里布蔓延。
 <br/>
 <br/>
 周四，文图拉县共燃起两场大火，希尔火灾延烧了近7000英亩。伍尔西火灾也在干燥天气和大风的助长下迅速蔓延，一个晚上就破坏了多幢住宅，并引发卡拉巴萨斯（Calabasas）、千橡市（Thousand Oaks）、马里布等市的部分地区被迫疏散。
 <br/>
 <br/>
 伍尔西火起火仅几个小时后就大爆发，而且没有停止的迹象。大火穿过了分隔西米谷和千橡的脊岭Albertson Motorway，烧到了China Flat处，在Cheeseboro和Palo Comado峡谷之间。
 <br/>
 <br/>
 周五，马里布数千人试图从太平洋海岸公路沿线（PCH）逃离。大火还威胁到文图拉县和洛杉矶县的其它地区，已有约7.5万户被下达疏散令。101高速路在自由峡路（Liberty Canyon Road）的双向出入口均关闭。目前还没有伤亡报告。
 <br/>
 <br/>
 当天，火灾还摧毁了橡树园（Oak Park）、千橡市，、贝尔峡谷（Bell Canyon）和其它文图拉县社区的数十所房屋，目前火势没有任何放缓迹象。马里布市府当天上午要求全市居民立即撤离。
 <br/>

 <br/>
 洛县消防官员告诉KNBC记者，大风推动下火势就像爆炸一样，这是新常态。碰到这样的大风，控制大火就非常困难。文图拉县消防局推特说，燃烧的植被和建筑助长了火势蔓延。
 <br/>
 <br/>
 当局说，由于当地圣安娜风非常强烈，目前伍尔西火基本零控制。周五早上，天气预报山谷和沿海瞬间风力时速为40～50英里，山区为60～70英里。国家气象台发布的大风红旗警报效期持续到周五晚上。
 <br/>
 <br/>
 文图拉县官员表示， 一直奋战希尔火灾的消防员连夜被调去扑灭伍尔西大火。估计希尔火会一直烧到海边。
 <br/>
 <br/>
 据当地媒体报导，Hollytree大道5600街区有几人报告被大火困住。当局已经加派了10组灭火队，帮助保护受大火威胁的建筑。现场消防员说，如果不是有额外援助，他们的灭火资源早就耗尽了。
 <br/>
 <br/>
 据《洛杉矶时报》报导，周五凌晨1点，橡树园地区火情剧烈。当地居民维恩（Douglas Wayne）说，他远远看到大火已经烧到他住了17年的房子后面的山坡。维恩说，多年前也有一场大火威胁该社区，但那时没有风，大家都坐在外面，看着大火烧。这次不同，很快就冒烟，然后立刻疏散警报就来了。
 <br/>
 <br/>
 他说：〝我们没有龙卷风或飓风灾害，但有地震和大火。这个（大火）真的很吓人。〞

 <h4>
  强制疏散地区
 </h4>
 <br/>
 强制疏散地区包括：整个马里布市，以及101高速以南，从文图拉边界到Las Virgenes/Malibu Canyon，朝南一直到太平洋。Liberty Canyon以西一直到Decker Canyon，朝南一直到太平洋高速。东从Las Virgenes/Malibu Canyon，西从Decker Canyon，一直到马里布的太平洋高速。
 <br/>
 <br/>
 整个橡树公园地区和西湖村（Westlake Village），还有千橡市的部分地区，包括千橡大街北边到Sunset Hills、 橡树公园地区向西到23号高速之间的地区。
 <br/>
 <br/>
 Bell Canyon：Saddlebow路位于Maverick Lane和Morgan Road之间路段。Bard Lake以南、23号高速以东地区。23号高速以西、E. Olsen Road以南、Pederson Rd以北地区。101高速以南、Reino Rd.以东、Potrero Rd.以北、洛县文图拉县边界以西的地区。
 <br/>
 <br/>
 目前洛杉矶警局已经实施战术警惕，确保充足资源帮助疏散和道路关闭。警局发推文警告：如果您住在受影响地区，并已被下达疏散令，请立即离开。
 <br/>
 <br/>
 洛县消防局发推文说：居民需密切关注疏散令，大风推火，非常危险。
 <br/>
 <br/>
 洛县消防队长伊姆布兰达（Tony Imbrenda）说，马里布的官员正迅速组织疏散。但因人们急着逃离，导致交通堵塞。他说，太多人滞留，希望人们为了安全尽快打包出城。
 <br/>
 <br/>
 目前，南、北加州合计共有25万人遭到疏散。气象学家警告，强风和干燥天气导致的危险火势，可能要一直持续到下周。
