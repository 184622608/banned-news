### 川普效应 中期选举早期投票率走高
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_先期投票.html" target="_blank">
  先期投票
 </a>
 〞和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邮寄投票.html" target="_blank">
  邮寄投票
 </a>
 已在20个州展开。皮尤研究中心的报告显示，川普是激励
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选民.html" target="_blank">
  选民
 </a>
 出来投票的重要因素。
 <br/>
 <br/>
 从9日开始，川普总统在Instagram上持续贴图，号召
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选民.html" target="_blank">
  选民
 </a>
 出来投票。
 <br/>
 <br/>
 根据乔治亚州务卿办公室等多方数据，今年这个州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_先期投票.html" target="_blank">
  先期投票
 </a>
 率比四年前增长了230%（296,601 vs 99,881）。近邻田纳西州，州府所在的Davidson县周三、〝先期投票〞首日，有10,249人参与，仅仅比2016年总统大选少了2千人，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 中实属罕见。
 <br/>
 <br/>
 川普今天发推支持维吉尼亚州国会众议员布拉特（Dave Brat），称他在边境问题、治理犯罪及宪法第二修正案拥枪权上都很有力。他批评民主党人打开边境，并称如有必要将会调动军队保卫南部边境。
 <br/>
 <br/>
 在联邦参议员竞选中，新泽西州寻求三连任的民主党籍联邦参议员梅南德兹（Bob Menendez）选情严峻，他与共和党挑战者休金（Bob Hugin）的民调差距目前仅相差8%，被认为是共和党一个可能的机会。
 <br/>
 <br/>
 同样寻求三度连任的密苏里州联邦参议员麦卡斯基尔（Claire McCaskill）近日拒绝普通人在她竞选活动中摄像。因为卧底调查组织Project Veritas本周开始陆续发布〝卧底〞视频，显示麦卡斯基尔对选民隐瞒了真实的观点。
 <br/>
 <br/>
 在一段卧底视频中，其办公室雇员表示，她主张弹劾川普总统；禁止使用半自动步枪甚至是手枪，观点上与奥巴马几乎完全一致。但这些都不能让她的共和党选民知道。
 <br/>
 <br/>
 目前，麦卡斯基尔与共和党挑战者霍利（Josh Hawley）的民调不相上下。
 <br/>
 <br/>
 对于民主党不断炒作的通俄门调查，选民似乎没有那么关心。
 <br/>
 <br/>
 爱达荷州Boise市选民 建筑工人 Justin Burkee：〝我认为需要更多的事实证据。〞
 <br/>
 <br/>
 爱达荷州Boise市选民 健保人员 Shawn Redetzki：〝我无法根据不明确的信息作出决定。〞
 <br/>
 <br/>
 爱达荷州Boise市选民 Noel Burkovsky：〝我们偶尔听到一点点，但是我认为通俄门是胡说八道。〞
 <br/>
 <br/>
 爱达荷州民主党籍国会众议员参选人 Muffy Davis：〝我没有听过一个选民、没有人问我关于通俄门调查的事。他们关心的是，至少我看到是这样，他们在意如何谋生，做自己喜欢做的事情，供养孩子、为他们提供更好的生活和教育。〞
 <br/>
 <br/>
 新唐人记者李兰综合报导
