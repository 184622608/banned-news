### 北加州野火71人死千失踪 毁近万房屋
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】加州历史上最具破坏性的〝营地〞野火继续延烧。巴特县治安官周五更新了最新的伤亡数据，已有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_71人丧生.html" target="_blank">
  71人丧生
 </a>
 ，超过一千人失踪，被摧毁的房屋接近一万栋。
 <br/>
 <br/>
 巴特县治安官哈尼亚，周五对外公布了〝营地〞野火最新的统计数据。当天又发现八具尸骸，死亡人数上升到71人。
 <br/>
 <br/>
 巴特县治安官 哈尼亚：〝今天发现八名死者，七人在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天堂镇.html" target="_blank">
  天堂镇
 </a>
 ，所有尸骸都在室内。另有一人在马格利亚。〞
 <br/>
 <br/>
 哈尼亚表示，当天确认了102名失踪人员的生存状况，目前失踪名单中还有1,011人，比前一天增加380人。名单还在随时变化，并将每天更新。
 <br/>
 <br/>
 当地消防官员表示，目前〝营地〞野火面积为14万6,000英亩，50%的火势得到控制。野火摧毁了9,700栋民宅，另有216栋建筑受损。当局派出5600多搜救人员，可能是加州历史上最多的。
 <br/>
 <br/>
 文图拉县消防总管 考夫曼：〝派出救援野火的总人数为5,632人。截至到今天（周五）晚18点01分，疏散47,200人，住进避难场所1,164人。〞
 <br/>
 <br/>
 周六下午，美国总统川普前往加州与灾民会面，视察灾情。
 <br/>
 <br/>
 川普在推文中说：〝期待见到我们勇敢的消防员、急救人员和紧急事务管理署员工，以及许多勇敢的加州人。我们一直和你们在一起，愿神保佑你们所有人。〞
 <br/>
 <br/>
 部分丧失家园的天堂镇居民，目前在一处沃尔玛停车场上搭帐篷度日。有居民表示，大火让他一无所有，但也得到很多人的帮助。
 <br/>
 <br/>
 流离失所的天堂镇居民 格莱布：〝我知道许多人没能逃离，一对夫妇在我面前遇难，你知道，作为一个男人会感到无助，无能为力，看着他们在面前遇难。我尽力了。〞
 <br/>
 <br/>
 目前，加州国民警卫队仍然不懈地在废墟中寻找尸骸。目前，当地的通讯仍很不稳定，许多人可能未能及时通知当局而被列入失踪名单。
 <br/>
 <br/>
 新唐人记者林宇、尚敬综合报导
