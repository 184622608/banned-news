### 【禁闻】峰会前施压？纳瓦罗直言不信任中共
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】本月底，美国总统川普和中国国家主席习近平将在二十国集团（G20）峰会上会面。美中双方是否能就贸易纠纷达成实质协议已经成了各方关注的焦点。不过，美国白宫贸易委员会主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 日前表示，即使与中方达成协议，也无法信任中国（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 ）会真正执行协议内容。
 <br/>
 <br/>
 9号，美国安全和国际研究中心（CSIS）在华盛顿举办了一场有关经济安全和国家安全的讨论会。美国白宫贸易委员会主任彼得．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 （Peter Navarro）受邀演讲。
 <br/>
 <br/>
 纳瓦罗在会上表示，美国和多个国家都有贸易谈判，但和中国的谈判即使有结果，也很难保证得到真正执行。他直言，美国和加拿大、墨西哥、日本、欧洲等国家和地区，是可信任的，他们会遵守协议，但是中国（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 ）却与众不同，无法信任。
 <br/>
 <br/>
 纳瓦罗举例说明，2015年一位中共高官在华盛顿曾向美国政府做出承诺，一个是南中国海人工岛礁非军事化，一个是不会再对美国企业进行黑客攻击。但实际结果是，两年后，中共在南海岛屿上已经部署了大量的武器装备，而对美国企业的电脑攻击，只暂停了几个月。此外，当初中共加入世贸组织时曾做出多项承诺，但都没有兑现实行。
 <br/>
 <br/>
 时事评论员田园博士：〝纳瓦罗是少数西方学者里面对中共的理解最精准、最到位、也是最彻底、对中共本质看得最清楚的学者之一，确实是目前川普政府在对中国贸易问题上的很大的一个智囊。〞
 <br/>
 <br/>
 纳瓦罗是一名经济学家，目前兼任白宫贸易和工业政策办公室主任、以及总统助理职务。他曾出版多部重要著作，其中《致命中国》一书，揭示了中共不遵循国际贸易规则，以政府补贴、窃取知识产权等国家行为，来主导、控制中国的贸易和工业发展。
 <br/>
 <br/>
 田园：〝最后的结果就是中国（中共）崛起，美国在这种不公平的体制下造成极大的伤害，然后中共真有可能成为独霸全球的强权和力量。〞
 <br/>
 <br/>
 美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 上个月发表演讲，对中共提出全面尖锐批评，标志着美国对中共政策将发生根本性转变。
 <br/>
 <br/>
 彭斯17号将出席亚太经合组织（APEC）峰会，并就美国的〝印度-太平洋〞战略发表重要演讲。《美国之音》报导，预计彭斯将宣布：美国针对印太各国，将提供600亿美元的经济援助，以抗衡中共的〝一带一路〞计划。
 <br/>
 <br/>
 另外，本月底，二十国集团（G20）峰会将在阿根廷举行，美国总统川普将与中国国家主席习近平会面。美国总统川普早先曾表示，不排除对中国商品征收更高关税。
 <br/>
 <br/>
 田园：〝目前川普总统也好，彭斯副总统还有其他官员发表的一些讲话，都在预示着美国政府高层对中共进行施压。〞
 <br/>
 <br/>
 时事评论员田园博士表示，本月底的川习会能够达成实质协议的可能性很小。很可能美国真的会升级关税制裁，之后中共会经历经济寒冬与各种危机，那时候才可能进行有成效的谈判。
 <br/>
 <br/>
 北京时政观察人士华颇则认为，最终，中共有可能做出一些重大让步，但对于中国制造2025计划，则不会放弃。
 <br/>
 <br/>
 北京时政观察人士华颇：〝比如说在关税方面、贸易准入、知识产权的保护等等方面都会做出一些重大让步。〞
 <br/>
 <br/>
 在周五的研讨会上，纳瓦罗还展示了一张图表，列出了中国产业所谓〝赶超计划〞的策略和具体方法，这其中包括〝不择手段获得先进核心技术〞。
 <br/>
 <br/>
 纳瓦罗强调，全球自由贸易的基础是公平和公开，当有些国家刻意采取不平等政策、并且还暗中进行侵害的时候，保护本国利益，并不是所谓的〝贸易保护主义〞，而是一种正当的政府行为，因为经济的安全就是国家安全。
 <br/>
 <br/>
 采访/陈汉 编辑/李谦 后制/周天
