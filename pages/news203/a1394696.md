### 迈克尔飓风持续增强 佛州进入紧急状态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】二级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 预计周二晚间增强为三级，将在佛罗里达州登陆。佛州州长宣布，35个县进入紧急状态，呼吁民众做好准备。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热带风暴.html" target="_blank">
  热带风暴
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 横扫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美洲.html" target="_blank">
  中美洲
 </a>
 后，增强为2级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 ，强度还在持续增加。
 <br/>
 <br/>
 美国国家飓风中心预报，迈克尔可能在周二晚间增强到3级，在美国佛罗里达州登陆，并向美国东岸移动。
 <br/>
 <br/>
 佛罗里达州州长斯科特宣布35个县进入紧急状态，呼吁民众做好准备，及时疏散。
 <br/>
 <br/>
 佛罗里达州州长 斯科特：〝飓风迈克尔是大风暴，可能会给部分地区或州带来严重破坏，特别是在狭长地带。〞
 <br/>
 <br/>
 斯科特形容迈克尔是滔天风暴，巨浪最高将达到4米左右，可能给佛州狭长地区带来数十年来最严重的破坏。
 <br/>
 <br/>
 佛州民众抓紧之间，出门加油、储备食品、饮用水等，做好应对飓风的准备。
 <br/>
 <br/>
 登陆佛罗里达之后，预计风暴将在周三、周四沿着大西洋海岸向东北方向移动，将影响卡罗莱纳州。
 <br/>
 <br/>
 周二一早，川普总统转发国家飓风中心推文，提醒民众注意飓风消息。
 <br/>
 <br/>
 国家飓风中心还警告飓风登陆地区民众，做好准备，保护生命和财产安全。
 <br/>
 <br/>
 新唐人记者李梅、张琪综合报导
