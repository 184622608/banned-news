### 梅拉尼娅回应印字大衣风波：反击左翼传媒
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】美国第一夫人梅拉尼娅（Melania Trump）近日接受美国媒体采访时，首次针对此前她身穿一件印有〝我真的不在乎〞字样的外套所引发的风波作出正面回应。她表示，那是针对左派媒体抹黑自己的报导进行的反击。
 <br/>
 <br/>
 梅拉尼娅10月13日在接受美国广播公司（ABC）采访时表示，稍早前她曾前往探望被拘留的中美洲移民儿童，在上飞机时，穿上了一件背后印有〝 I don't really care do u? 〞字样的外套出现在镜头中，当时这样做是为了反击批评她的左翼媒体。
 <br/>
 <br/>
 梅拉妮亚说：〝它是穿来给一些人、以及那些批评我的左媒体看的。我想让他们看到我不在乎。你想怎么批评我都可以，但不会阻我去做我认为对的事。〞
 <br/>
 <br/>
 她进一步表示：〝我想向他们（左翼传媒）展示我并不在乎，我宁愿他们关心我所做的事，而不是我的衣着。〞
 <br/>
 <br/>
 梅拉妮亚还进一步表示，左派传媒曾大肆宣传她在白宫活得不开心，甚至抹黑她的婚姻生活，形容她生活得〝很悲惨〞。这些不实报导令她感到愤怒，所以身穿写有上述字样的外套出现在镜头前以示反击。

 今年6月21日，身为第一夫人的梅拉尼娅・川普启程前往德克萨斯州，计划到那里的一座儿童收容机构去做人道主义访问。当天她身穿一件橄榄绿色Zara军装外套走上飞机，而那件夹克的背后印上了那句引发争议的句子，这一切清楚地通过媒体拍摄的镜头展示在公众眼前。
 <br/>
 <br/>
 随后，左派媒体开始那这件事做文章，把梅拉尼娅穿的这样一件外套，与她即将去看望那些来自墨西哥边境非法移民的子女联系起来，指其〝别有用意〞。梅拉尼娅则在接受媒体采访时解释说：〝我当然不会冲着孩子们穿这件外套。我只是穿着它上下飞机。〞
 <br/>
 <br/>
 而就在近日，梅拉妮亚1999年接受美国广播公司访问的视频再度曝光。当时年仅26岁的梅拉妮亚与川普约会一年多，她在访问中称，川普将成为一名伟大的总统。她当时就说：〝他非常聪明，知道如何做生意，他将是一名伟大的领袖。〞
 <br/>
 <br/>
 梅拉妮亚还表示，自己会想像，如果有朝一日自己成为了美国第一夫人，自己将会像积琪莲和贝蒂一样，是一位传统的第一夫人，会认真履行很多的社会义务。
 <br/>
 <br/>
