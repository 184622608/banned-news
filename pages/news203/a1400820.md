### 北京透露〝不抵抗〞底线 换取川习会〝双赢〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】为了给11月底召开的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞铺路，近期中方在贸易战方面态度软化，从最初〝以牙还牙〞，转为〝推动达成双方都能接受方案〞。有港媒26日称，中共还做出了可以做的让步，以〝不抵抗〞为底线，换取〝美中双赢〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 即将迎来关键节点，中国国家主席习近平与美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普），即将在阿根廷首都布宜诺斯艾利斯举行的二十国集团（G20）峰会期间会面。
 <br/>
 <br/>
 为了给〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞铺路，中美双方自11月开始释放对话信号，以缓和当前局势。近期中方对贸易战的态度，已从最初〝以牙还牙〞转为了〝推动达成双方都能接受方案〞。
 <br/>
 <br/>
 有匿名中共高层官员对媒体说，习近平最近特别强调，要控制好反击贸易战的影响，避免对中共继续开放产生负面效应。
 <br/>
 <br/>
 川普透露，北京迫切希望在此次〝川习会〞能达成某种协议，因此已经向美方缴交一份关于贸易纠纷的让步清单。但川普也表示，他还不能接受这份清单，因为漏掉了4、5项重要内容。他还强调，未来美国与中共要达成协议必须是公平互惠的。
 <br/>
 <br/>
 对于中共向美国低头，中共官媒集体沉默不作声。苹果日报说，中共向美低头交清单，这在中共历史上极为罕见，而大陆官媒对此事只字不报，显示中共在掩耳盗铃。
 <br/>
 <br/>
 有北京政治观察家分析，中方放低姿态的背后，有近期和长期两方面考量：从近期来看，今年年底北京开始进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革.html" target="_blank">
  改革
 </a>
 开放40周年的宣传期，与美国摩擦升级，不利于传递其扩大开放的氛围。
 <br/>
 <br/>
 从远期来看，正如外界所言〝中美已经回不到过去〞，经过这一年争拗，中共已接受这一现实，放弃了以反击迫使美国回到贸战之前的想法，准备好迎接新形态的中美关系，以〝不抵抗〞为底线，寻找这一世界上最重要双边关系的新平衡。
 <br/>
 <br/>
 《香港经济日报》26日引述中国人民大学国际关系学院教授时殷弘的分析称，中美贸易战有两个层面，一层是贸易方面的高关税征收博弈，另一层是高科技封锁，长期不会改变。
 <br/>
 <br/>
 时殷弘说，中共希望在关税层面的贸易战上，能有一个较长时间的休战期，以争取一个压力较小的外部环境，加速经济优化等方面的调整，而今中方已经做出了可以做的让步，主要集中在巨量进口美国商品和大幅放宽对美市场准入。
 <br/>
 <br/>
 但美国想要其他的，包括中共停止盗窃美国知识产权、中共政府停止向中国企业提供特惠和补贴、中共停止强迫在华美企转让技术等，而美国的核心要求是要中共改变工业政策。
 <br/>
 <br/>
 之前就有中国学者透露，中共的谈判底线是不阻碍中共经济的转型升级，而美方要求北京按照市场经济标准，停止政府补贴企业开发高技术的〝2025计划〞，或〝2035计划〞，以及北京不承认的盗窃知识产权问题等，这些应该是触到了中共的底线。
 <br/>
 <br/>
 美国贸易代表莱特希泽（Robert Lighthizer）21日还在白宫批评中共，基本上没有改变其技术转让、知识产权和创新相关的行为、政策和做法，最近几个月，中共还采取了更进一步的不合理行动。这给中美贸易战因〝川习会〞停止的期望泼上冷水。
 <br/>
 <br/>
 美国副总统彭斯曾表示，两国间的贸易战不会在APEC峰会后偃旗息鼓，不可能在明年一月前达成交易。美国仍然计划在明年1月提高对中国商品的征税额度。
 <br/>
 <br/>
 经济学者杰森表示，不管〝川习会〞能否谈妥，中共让步是迟早的事。但是即使中共作出全面妥协，它也不会告诉老百姓真实情况。无论它做出什么样的让步， 它都会说〝美中双赢〞。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
