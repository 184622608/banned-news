### 见识5级飓风威力！夏威夷小岛从地图上消失
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】5级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瓦拉卡.html" target="_blank">
  瓦拉卡
 </a>
 本月初以每小时220公里的风速横扫法国军舰环礁后，无人居住的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_夏威夷小岛.html" target="_blank">
  夏威夷小岛
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东岛.html" target="_blank">
  东岛
 </a>
 几乎被吞没。这座小岛虽然地处偏远，却是夏威夷僧海豹、夏威夷绿蠵龟的重要栖地。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瓦拉卡.html" target="_blank">
  瓦拉卡
 </a>
 (Walaka)在5级制的沙费尔-辛浦森制（Saffir-Simpson scale）属于最高级别5级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 ，是侵袭这个太平洋地区史上第2强的飓风。
 <br/>
 <br/>
 美国官员表示，当瓦拉卡本月稍早侵袭一座偏远的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_夏威夷小岛.html" target="_blank">
  夏威夷小岛
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东岛.html" target="_blank">
  东岛
 </a>
 （East Island）后，这座岛屿大体上从地图上消失，本周已派出海洋废弃物的工作人员前往小岛评估受灾情况。

 帕帕哈瑙莫夸基亚国家海洋保护区（Papahanaumokuakea）发声明表示：〝东岛似乎已在水下。〞
 <br/>
 <br/>
 美国鱼类暨野生动物管理局（US Fish and Wildlife Service）卫星影像显示，当飓风瓦拉卡肆虐这座岛屿时，几乎所有沙子都从这座小岛流失。
 <br/>
 <br/>

 <center>
 </center>
 <br/>
 东岛是一座低洼岛屿，位于檀香山市西北方约800公里，偏远而杳无人烟，是法国军舰环礁（French Frigate Shoals）第2大岛。这座岛屿长约800米、宽约120米。主要由松软的沙子和砾石组成，是受威胁物种绿蠵龟筑巢之处，以及濒危的夏威夷僧海豹栖地。
 <br/>
 <br/>
 美国商务部、内政部、夏威夷州和夏威夷事务办公室（Office of Hawaiian Affairs）管理这个保护区。
 <br/>
 <br/>
 夏威夷大学（University of Hawaii）气候科学家佛莱契尔（Chip Fletcher）本周告诉夏威夷当地媒体Honolulu Civil Beat，他得知这个消息大吃一惊，直呼，〝噢，我的天呀，它消失了。〞
 <br/>
 <br/>
 佛莱契尔说，将近96％的夏威夷绿蠵龟都会迁徙到东岛及附近的岛屿上，包括法国军舰环礁，以便在繁殖季节安全筑巢。不过他也说，幸运的是，当瓦拉卡来袭时，大部分龟卵都孵化了，幼龟已进入海中。
 <br/>
 <br/>
 而全球可能只剩下1400只的夏威夷僧海豹，虽然大部分时间都躺在海滩上，但也能在狂风暴雨中潜入海里。弗莱彻认为这两种动物估计不会遭遇毁灭性的打击，不过，这还是需要生物学家最后的评估。
 <br/>
 <br/>
 其实，这也不是夏威夷一带第一次出现小岛消失的情形。小岛〝特里〞(Trig)也在今年9月涨潮的时候被淹没。1990年代，小岛〝鲸鱼滑板〞(Whale-Skate)消失后就再也没有浮起来了，该岛隶属法国军舰环礁、同样也是夏威夷僧海豹重要栖地。
 <br/>
 <br/>
