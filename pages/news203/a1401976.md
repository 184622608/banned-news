### 贸易战〝缓刑〞细节 川普公布美方谈判阵容
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 后，美中将立即展开新一轮贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谈判.html" target="_blank">
  谈判
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 选择对中共立场强硬的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莱特希泽.html" target="_blank">
  莱特希泽
 </a>
 监督谈判，这预示了美国的坚定的谈判立场。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 12月4日在推特上宣布，〝与中方的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谈判.html" target="_blank">
  谈判
 </a>
 已经开始〞。谈判将自12月1日〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞起的90天内结束，除非期限延长。
 <br/>
 <br/>
 川普提醒中方立刻兑现川习会上的口头协议，〝立即购买美国农产品等其他产品〞。接下来，谈判如能达成〝公平协议〞，实现所有必要事项，他会〝愉快地签署〞；如果无法达成协议，他警告：〝记住，我是关税强人〞。
 <br/>
 <br/>
 库德洛：我们必须确保任何协议都被真正履行，这非常重要。这一次，就像我说的是有希望的。我们（谈判）涉及的事项更多，中方比以往更加合作。
 <br/>
 <br/>
 川普还宣布了美方谈判阵容，包括对中共态度强硬的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莱特希泽.html" target="_blank">
  莱特希泽
 </a>
 和纳瓦罗。值得注意的是，今年前4轮美中谈判，都是由〝温和派〞姆努钦与罗斯负责，这次换成〝强硬派〞莱特希泽督阵。
 <br/>
 <br/>
 白宫官员表示，莱特希泽将密切注视中共近期的举动，并进行评估。
 <br/>
 <br/>
 库德洛：我们将密切注视（中方举动）。你知道的，在监控和评测方面，莱特希泽将承担绝对关键的角色，在这一领域，没人比他更胜任。
 <br/>
 <br/>
 《华尔街日报》引述知情人士说，莱特希泽出任首席谈判代表，让川普更有信心，确保美国推行强硬立场。
 <br/>
 <br/>
 莱特希泽2001年就反对中共加入世贸组织；过去2年他主持编写301调查报告，指出中共长期有系统的盗窃美国知识产权；他曾表示，他出任美国贸易代表的主要目的，是寻求重新定位对华政策，确保美国不会成为中国现有体制的受害者。他多年的谈判经历显示，他倾向于达成具有法律效力的协议，而不是空泛的承诺。
 <br/>
 <br/>
 陈破空：按照纳瓦罗的描述说，莱特希泽是我们团队中最强势的，最会谈判的高手，说他将据法律，因为他本身是个大律师，逐章逐章的，逐节逐节的跟中方谈判关税、非关税壁垒、以及市场准入，促使中方结构性的改变。
 <br/>
 <br/>
 不过美国联邦参议员鲁比奥发推特提醒，中共擅长拖延战术，总是谈判，从不改变。他期待G20峰开启了新的方向，但他怀疑中共仍在故技重施。
 <br/>
 <br/>
 多位美国专家也认为，美国寻求中方进行〝结构性改变〞，等于要求中共放弃其经济特权，这对美国有利，也对中国民众有利，但北京是否有意愿和决心这么做，仍有待观察。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
