### 美国土安全部长：大篷车移民将不能入境
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国土安全部长.html" target="_blank">
  国土安全部长
 </a>
 尼尔森（Kirstjen Nielsen）周日（10月28日）警告，那些正穿越墨西哥向北进发的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 将不能进入美国。
 <br/>
 <br/>
 尼尔森告诉福克斯新闻（Fox News），有合法入境美国的方式，而选择非法入境的人将被制止。
 <br/>
 <br/>
 目前至少有7000
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪都拉斯.html" target="_blank">
  洪都拉斯
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 正从墨西哥境内向美国边界挺进，虽然墨西哥方面日前表示，如果这些移民愿意留在墨西哥南部两州，墨西哥政府将为他们提供医疗、子女教育甚至临时工作，但绝大多数移民拒绝了这一提议，坚持朝美国行进。
 <br/>
 <br/>
 这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 仍然希望，可以利用美国宽松的移民法律寻求在美国立足。
 <br/>
 <br/>
 这些移民通常能采用的策略就是声称他们惧怕被遣返回国，如果能获批准，他们就有机会申请庇护。
 <br/>
 <br/>
 入境的移民通常都能立即得到释放，然后获得机会申请彻底庇护。其中一部分人很快就作为非法移民消失了，另一些逃避听证、同样也消失了。
 <br/>
 <br/>
 尼尔森指出，大篷车压境只是边境危机的一次重复，而川普总统有权选择任何一种方式阻止非法移民入境。
 <br/>
 <br/>
 不过，尼尔森拒绝透露，川普总统是否打算对申请庇护的移民关闭边界，比如可能发布类似于此前的阻止来自以穆斯林为主的国家的公民入境美国的旅行禁令。
 <br/>
 <br/>
 尼尔森表示，她对大篷车移民传递总的信息是：〝别来。不会允许你们入境。有正当的方式移民美国，但这种方式不是。〞
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
</div>
