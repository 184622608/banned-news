### 美国土安全部长视察边境墙 与各方合作
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】10月26日，美国国土安全部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼尔森.html" target="_blank">
  尼尔森
 </a>
 来到加州边境小城加利西哥，视察这里已经完工的部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境墙.html" target="_blank">
  边境墙
 </a>
 。
 <br/>
 <br/>
 根据白宫年初提交国会的计划，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境墙.html" target="_blank">
  边境墙
 </a>
 第一阶段工程新建墙体316英里、高度18到30英尺；还包括407英里围栏的加固工作。
 <br/>
 <br/>
 针对向美墨边界移动的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 ，国土安全部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼尔森.html" target="_blank">
  尼尔森
 </a>
 表示，美国有各种选项，阻挡非法入境者。
 <br/>
 <br/>
 美国国土安全部长Kirstjen Nielsen: 〝如果没有合法权利进入美国，就不会允许进入。如果偷偷溜过我们的边境，将被逮捕和遣返。〞
 <br/>
 <br/>
 装甲车和骑着马匹的巡逻兵在附近移动。边境上各种高墙已初具规模，还有照明灯柱。
 <br/>
 <br/>
 美国国土安全部长Kirstjen Nielsen: 〝需要注意，我们与墨西哥保持着实时的密切合作，与联邦内部各机构也紧密沟通，确保每一个应对方案都能实现。〞
 <br/>
 <br/>
 国土安全部还要求国防部给予援助，提供车辆路障和行人路障。
 <br/>
 <br/>
 新唐人记者李子玉报导
</div>
