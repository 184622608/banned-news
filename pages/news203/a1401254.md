### 全美治安最佳10城 湾区菲利蒙入选
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】个人理财网站WalletHub日前公布2019年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全美.html" target="_blank">
  全美
 </a>
 治安最佳和治安最差的城市名单，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湾区菲利蒙.html" target="_blank">
  湾区菲利蒙
 </a>
 入选十大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_治安最佳城市.html" target="_blank">
  治安最佳城市
 </a>
 之一。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州洛杉矶.html" target="_blank">
  加州洛杉矶
 </a>
 则被列入治安最差的城市名单。
 <br/>
 <br/>
 WalletHub按照年度犯罪数量、枪击死亡人数以及盗窃和诈骗等进行比对，评比出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全美.html" target="_blank">
  全美
 </a>
 城市治安排行榜。湾区第四大城市菲利蒙（Fremont）进入全美治安最佳的前十名城市名单，位列第10位。
 <br/>
 <br/>
 排名的靠前的城市为：佛蒙特州南伯灵顿（South Burlington）、夏威夷州珍珠市（Pearl City）、马里兰州哥伦比亚（Columbia）以及维吉尼亚州乞沙比克（Chesapeake）等城市。
 <br/>
 <br/>
 此前，菲利蒙还被WalletHub评比为全美最幸福城市榜首。居民对收入、就业和自然环境等因素都感到满意。
 <br/>
 <br/>
 加州的洛杉矶则被列入治安最差的前十名城市名单。11月7号，洛杉矶以北的千橡市发生酒吧枪击事件，导致12人丧生。这份治安最差榜单中，除了洛杉矶还有内华达州拉斯维加斯、纽约市、德州休士顿、密苏里州圣路易斯以及乔治亚州亚特兰大等城市。
 <br/>
 <br/>
 新唐人记者景雅兰综合报导
</div>
