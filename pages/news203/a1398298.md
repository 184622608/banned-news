### 川普俄州助选最后冲刺  民众支持建墙
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】2018年11月5日，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的前一天，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在美国中西部3个城市一连举办了3场让美国再次强大（MAGA）的演讲集会，为共和党候选人助选作最后冲刺。第一站集会在俄亥俄州克里夫兰市的世博中心（IX Center）召开。约有一万民众到现场支持川普。随着中美洲的大篷车车队向美国南部边境逼近，所有的受访者都表示支持川普总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_建墙.html" target="_blank">
  建墙
 </a>
 。
 <br/>
 <br/>
 退休警察John Mott：〝他兑现了他的每一个承诺，唯一没兑现的承诺是因为国会不批准。例如
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_建墙.html" target="_blank">
  建墙
 </a>
 。他要建墙，我支持建墙，很多美国人支持建墙，但国会，民主党人不让他建墙。〞
 <br/>
 <br/>
 退休装饰师Caroline Kobac：〝每个国家都必须有边境。作为公民，我们想要移民，但我们希望他们合法地来，这样才公平。〞
 <br/>
 <br/>
 退休钢铁厂工人Mark Fedarko：〝我家是合法地从欧洲移民来的。非法越境过来的人，采取了错误的方法，是非法的，他们是罪犯。 其实，坐大篷车越境，就不仅是大篷车的问题了，这是入侵。 我们需要保护边界并将其关闭。 我们不知道谁在越过边界。我们终于有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统挺身而出，着手解决这个问题了。许多美国人已经无法忍受了。我们要阻止这种非法移民。〞
 <br/>
 <br/>
 职业培训和发展公司项目总监John Roberts：〝他们让非法移民不断的涌入美国，增加纳税人的负担。这必须停止。他（川普）终于发声了，受够了。 美国优先。〞
 <br/>
 <br/>
 家庭主妇Tonya Depont：〝我绝对是赞成移民的。 我不反对移民，但我希望他们合法移民。 因此，如果我们建成了那堵墙，它会让我们更容易守护边界，因为我们不希望有非法入境，非法的移民只会伤害我们的国家。〞
 <br/>
 <br/>
 退休警察John Mott：〝我们不能开放边界。那就像我走到任何其他国家，未经审查，然后说，我们将在你们的选举中投票。我们不能允许这样做。川普总统并没有说你不能进入美国。 他说，如果你要来美国，你必须合法地来到这里。这就是他所说的，这是非常公平的。其他的国家也都这么做。〞
 <br/>
 <br/>
 新唐人记者婉莹、明光美国俄亥俄州克里夫兰市采访报导
</div>
