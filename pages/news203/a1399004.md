### 加州野火夹攻 父载3岁女儿逃跑影片爆红
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】美国加州近日3起野火肆虐，已知至少31人丧生、200多人失踪。重灾区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天堂镇.html" target="_blank">
  天堂镇
 </a>
 一名爸爸近日开车逃命时，边观察路况边哼歌安抚3岁女儿。影片被妻子上传脸书后，网友直呼，好贴心的父亲。
 <br/>
 <br/>
 加州南北部近日多处野火肆虐，其中属北加州的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎普野火.html" target="_blank">
  坎普野火
 </a>
 〞（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Camp Fire.html" target="_blank">
  Camp Fire
 </a>
 ）灾情最为严重，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天堂镇.html" target="_blank">
  天堂镇
 </a>
 （Paradise）6700多栋民宅和店家全部烧毁。
 <br/>
 <br/>
 据〝今日美国报〞（USA Today）报导，天堂镇居民艾伦（Joe Allen）在山火发生后开往加州契科（Chico）途中，有一段公路两旁可见熊熊烈火，现场浓烟弥漫。但艾伦一路保持冷静，还哼歌安抚3岁女儿。
 <br/>
 <br/>
 艾伦的行车纪录影片显示，艾伦3岁的女儿奥莉维亚（Olivia）看到两边的大火，一度紧张地高声说道：〝哎呀，这里为什么会有火，我们要被烧到了。〞
 <br/>
 <br/>
 不过，艾伦并未慌张，而是体贴地安抚女儿说没事，〝我们会安然度过的〞，他还哼着〝宝贝，一切都会没事的〞。
 <br/>
 <br/>
 穿过烟雾缭绕的公路，他们的车终于到达了安全路段。奥莉维亚则高兴的对爸爸连喊〝你办到了〞。艾伦也激动的对女儿说：〝是我们一起办到了。〞
 <br/>
 <br/>
 艾伦的妻子8日将影片上传脸书后，已获得超过11万次点击，1500多次分享。网友们也纷纷留言祝福他们，直呼艾伦是好爸爸，好贴心。
 <br/>
 <br/>
 据悉，艾伦的妻子当时则开另一辆车载着他们的另外一个女儿。
 <br/>
 <br/>
 <center>
 </center>
 <br/>
 美国加州近日南北共有3股大火狂烧，除了〝坎普野火〞（Camp Fire）肆虐天堂镇外，还有位于南部文图拉县（Ventura County）的〝伍尔西大火〞（Woolsey Fire）和〝希尔大火〞（Hill Fire）。连续3天火势不减，过火面积不断扩大，到10日已达到425平方公里，一些社区被夷为平地。
 <br/>
 <br/>
 加州迄今已投入至少810万美元、数千消防员、灭火专用飞机、铲土机等大型机械全力灭火。天气预报显示，到周一风速将达到每小时80公里，这增加了火势进一步扩散的风险。预计火势要到本月底才能完全得到控制。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
