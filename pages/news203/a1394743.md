### 大法官卡瓦诺首日履职 左派议程将受挫
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】周二，已宣誓就职的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院.html" target="_blank">
  最高法院
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官.html" target="_blank">
  大法官
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 首日办公，最高法院的保守派也正式占据多数。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 的公开宣誓就职典礼周一晚于白宫举行。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院.html" target="_blank">
  最高法院
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官.html" target="_blank">
  大法官
 </a>
 卡瓦诺：〝我，布雷特．卡瓦诺宣誓， 作为美国最高法院大法官，遵守美国宪法和法律。请上帝帮助我。〞
 <br/>
 <br/>
 宣誓环节由前任大法官安东尼．肯尼迪主持，卡瓦诺曾担任过他的助理。
 <br/>
 <br/>
 美国总统 川普：〝这一仪式事关我们将成为什么样的国家，我们的后代将继承的未来。〞
 <br/>
 <br/>
 肯尼迪于7月31日正式退休，川普（特朗普）提名保守派卡瓦诺填补空缺。他的任命将确立保守派在最高法院占有5席的多数地位，这让美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_左派.html" target="_blank">
  左派
 </a>
 非常忧心。共和党认为，卡瓦诺提名确认过程中遭遇的种种麻烦，包括无法被证实的性侵指控，都是左派等抵抗势力的政治把戏。
 <br/>
 <br/>
 美国总统 川普：〝我代表国家向卡瓦诺及他的家人道歉，为他们不得不承受的巨大痛苦和折磨。玛格丽特和丽莎，你们的父亲是个好人。现在他填补美国最高法院的空缺，在那里他会捍卫美国人民的权利和自由。〞
 <br/>
 <br/>
 正式
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_履职.html" target="_blank">
  履职
 </a>
 前，卡瓦诺已经招募了4位女性法官助理，是首位聘用全女性团队的大法官。根据下级法院的上诉，卡瓦诺可能会面对的案件包括DACA合法性的问题、堕胎争议问题、宪法保护的宗教自由与维护同性恋的〝反歧视法〞之间的矛盾等。他的前任肯尼迪大法官的立场经常左右摇摆，例如支持堕胎、同性婚姻等。
 <br/>
 <br/>
 乔治梅森大学法学院教授 Francis H. Buckley：〝社会主义、自由主义的日程不会进一步推进，像肯尼迪法官在任时的情况会彻底结束。但是否会颠覆既有的自由派判例，有待观察。对待堕胎等的决定，不会出现戏剧性的逆转。〞
 <br/>
 <br/>
 哈佛大学法学教授 Alan Dershowitz：〝我认为不会有革命性的变化，而是慢慢改变。因为川普要任命的人就是要改变最高法院保守派与自由派的力量对比。所以关键不在卡瓦诺，而是川普。〞
 <br/>
 <br/>
 川普周一表示，卡瓦诺提名确认的过程，会让很多民主党选民在11月投票给共和党，因为民主党基本盘已经变得太左倾。
 <br/>
 <br/>
 哈佛大学法学教授 Alan Dershowitz：〝如果民主党坚持弹劾和调查（卡瓦诺），这会伤到民主党自己。〞
 <br/>
 <br/>
 乔治梅森大学法学院教授 Francis H. Buckley：〝我认为这将有助于共和党，他们把其称之为‘卡瓦诺激励效应’。我想很多人会清醒，看到民主党所为。他们所做的是想抹去川普在大选中异军突起的印象。〞
 <br/>
 <br/>
 曾精准预测2016大选的拉斯穆森民调公司（Rasmussen polls）10月8日公布， 认同卡瓦诺听证会上批评〝确认过程变成了国家耻辱〞的民众达到56%，其中民主党人占40%。同时，川普总统的最新民调支持度上升到51%。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
