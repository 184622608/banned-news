### 挟持儿童还向警方开火 南卡州6警受伤1人殉职
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】美国南卡罗莱纳州（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南卡州.html" target="_blank">
  南卡州
 </a>
 ）城市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛罗伦斯郡.html" target="_blank">
  佛罗伦斯郡
 </a>
 于周三（3日）发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 ，一名挟持儿童在家中的男子，与警方对峙期间向警方开火，造成6名警察受伤，另有1人殉职。事发2小时后男嫌犯被捕。
 <br/>
 <br/>
 CBS报导，佛罗伦斯市（Florence County）政府发言人表示，警方获报后前往疑犯家中，但对方突然开枪，导致3名警察受伤。警方被迫以防弹汽车掩护暂时撤离。其后，又有4名接报支援的警察中枪，其中1人不幸殉职。
 <br/>
 <br/>
 佛罗伦斯市发言人约翰．乌克拉（John Wukela）告诉美联社，被杀害的警察叫特伦斯．卡拉威（Terrence Carraway），他在这支部队工作了30年。
 <br/>
 <br/>
 当局暂未公布死伤者及疑犯身份。
 <br/>
 <br/>
 川普（特朗普）总统在晚间新闻发布会后发推文，为佛罗伦斯的警察祈祷，〝我们永远感谢我们的执法人员，感谢他们24/7/365所做的事。〞
 <br/>

 联邦参议员史考特（Tim Scott）透过社群媒体表示：〝我们为佛郡受伤的执法人员及他们的家人祈祷。〞
 <br/>
 <br/>
 南卡州长麦克马斯特形容这是令人震惊的消息。他表示，执法部队内的男女，无私及勇敢的行为都是真实存在的。
 <br/>
 <br/>
 佛罗伦斯警察局局长艾伦．海德勒（Allen Heidler）说，他的一名军官在事件中丧生，他希望人们为失去了他所知道的最勇敢的警察的家人祈祷。海德勒说，忍住眼泪。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛罗伦斯郡.html" target="_blank">
  佛罗伦斯郡
 </a>
 紧急事故管理部门也发推文表示，枪击事件已结束，嫌犯已被逮捕，〝请民众勿前往复古地（Vintage Place），因为警方仍在进行犯罪调查。〞
 <br/>
 <br/>
 复古地位于霍夫梅耶路（Hoffmeyer Road）附近，是该市西部的一个高档社区。佛罗伦斯位于默特尔海滩西北约70英里处。该地居住着约37,000人，位于95号和20号州际公路的交会处。该地区最近受到飓风及大洪水的袭击。
