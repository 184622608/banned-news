### 川普谴责仇恨犯罪 提出恢复死刑
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】周六在伊利诺伊州的助选集会上，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 严词谴责这一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_仇恨犯罪.html" target="_blank">
  仇恨犯罪
 </a>
 。他甚至提到要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恢复死刑.html" target="_blank">
  恢复死刑
 </a>
 ，以制止这样的悲剧。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：〝对于这样的犯罪行为，不管是这个案件，还是其他案件，还是针对别的团体。我们必须重新启用死刑。他们必须为此付出最高代价。他们必须付出最高代价。他们不能这么做。他们不能对我们国家作出这种事情。我们必须有个明确的底线，而且要发出强大的声音：再也不要发生！〞
 <br/>
 <br/>
 川普对现场的支持者强调，要保护信仰自由和传统价值。
 <br/>
 <br/>
 美国总统川普：〝我们保护信仰自由，我们相信祈祷的力量。我们要捍卫宪法，捍卫我们的文化传统。美国的价值把我们凝聚在一起。在本次选举中，我们一定要保卫这些价值。〞
 <br/>
 <br/>
 川普表示，本次中期选举可以说是美国历史上最重要的选举。而心怀仇恨的人是愚蠢的。
 <br/>
 <br/>
 新唐人电视综合报导
