 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票站.html" target="_blank">
  投票站
 </a>
 ，为您带来中期选举第一手画面。来看报导。
 <br/>
 <br/>
 记者：我现在在位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洛杉矶.html" target="_blank">
  洛杉矶
 </a>
 亚凯迪亚市的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票站.html" target="_blank">
  投票站
 </a>
 。加州投票站的开放时间是当地的早上7点到晚上8点，有不少
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选民.html" target="_blank">
  选民
 </a>
 很早就赶来投票。目前，加州有至少5个国会选区的选情比较受到关注，对两党争取成为众议院多数党相当重要。
 <br/>
 <br/>
 退休国会首席谈判代表 C.G. Frienden：〝很开心看到经济在回升，虽然我不应该太强调政治，但我真的十分开心我们有川普这样的总统。〞
 <br/>
 <br/>
 私立学校执行董事 Vivian Chang：〝主要我想看到人民团结，无论结果如何，我知道每个人经历不同，也有不同想法，我非常支持大家团结并同心协力，相互尊重，也尊重我们的执政者，大家要好好相处。〞
 <br/>
 <br/>
 俄亥俄州共和党选民 Gary Smith：〝大多数我支持的参选人，都支持川普，这不仅因为川普是世界上最棒的人，更因为他有效率，他能把事情做成。而且他能说到做到，这非常了不起。〞
 <br/>
 <br/>
 俄亥俄州独立选民 Wilma Snyder：〝我是个基督徒，基督教价值于我非常重要，我投给那些与我有相同见解的人。〞
 <br/>
 <br/>
 记者：这里是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约曼哈顿.html" target="_blank">
  纽约曼哈顿
 </a>
 中城的投票站，从早上就不断有人进出，投票踊跃。今年纽约州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，最受瞩目的当属纽约州长的选举，共和党参选人莫里纳罗挑战竞选连任的州长库默，目前，库默民调领先。
 <br/>
 <br/>
 纽约选民 Jennifer：〝今天是很重要的一天，所有美国人都要出来投票。非常关键，是改变我们国家的超级关键时刻。〞
 <br/>
 <br/>
 纽约选民 Lisa Moravek：〝因为这是我们的公民义务，我对美国的领导者非常关注。确保能代表我价值观的人当选。〞
