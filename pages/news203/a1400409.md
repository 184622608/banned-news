### 美墨边境 马蒂斯：军队能保护边境
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】周三，美国国防部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蒂斯.html" target="_blank">
  马蒂斯
 </a>
 表示，白宫授予他新的权力：使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军队.html" target="_blank">
  军队
 </a>
 保护〝海关和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境.html" target="_blank">
  边境
 </a>
 保护局〞工作人员。
 <br/>
 <br/>
 美国国防部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蒂斯.html" target="_blank">
  马蒂斯
 </a>
 周三表示，政府赋予他新的权力。
 <br/>
 <br/>
 美国国防部长马蒂斯：〝总统认为，确有必要支援
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_边境.html" target="_blank">
  边境
 </a>
 巡逻队。我们昨晚收到进一步指示，授权实施其他措施。〞
 <br/>
 <br/>
 据报，这可能意味着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军队.html" target="_blank">
  军队
 </a>
 在发生暴力事件时，可以暂时拘留移民。不过马蒂斯解释说，他不打算利用军队扣押移民，即使拘留〝几分钟〞。
 <br/>
 <br/>
 马蒂斯表示，将增派5764名联邦军人到南部边境。他提到，在危地马拉边境，有六名墨西哥警察受伤。
 <br/>
 <br/>
 美国国防部长马蒂斯：〝这发生在危地马拉边境。因此，担忧是合理的，总统（认为）我们需要支援边境。〞
 <br/>
 <br/>
 周四，川普总统发表推文说：〝将通过电话会议，和世界各地的美军交谈。感谢海岸警卫队爱国者和他们的伟大工作。〞
 <br/>
 <br/>
 另外，川普推文批评联邦第九巡回上诉法庭和罗伯茨法官，推翻的裁决达79% ，超过任何其他巡回法庭。
 <br/>
 <br/>
 推文说：〝法官们不能制定边境或其他任何地方的安全法律。他们对此一无所知，让我们国家不安全。不许允许我们优秀的专业执法人员进行工作！否则，只有混乱、无序、以及伤亡。〞
 <br/>
 <br/>
 周二，美国官员表示，白宫正在起草议案，考虑在海关和边境保护局工作人员不堪重负的情况下，让军队为移民进行医疗检查。
 <br/>
 <br/>
 周三，更多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 从墨西哥边境城市墨西加利，涌入蒂华纳，当地难民人数增加了几百人，收容所不堪重负。
 <br/>
 <br/>
 新唐人记者毕心慈、天羽综合报导
</div>
