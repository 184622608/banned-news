### 川普谈判前不知孟晚舟被捕 美官员：这是司法部独立行动
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】应美国的引渡要求，加拿大执法机构12月1日抓捕了华为公司高管
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 。由于同一天美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 正在布宜诺斯艾利斯与中国国家主席习近平为缓和贸易冲突进行会谈，外界质疑川普是否提前知晓这次抓捕行动。日前，包括白宫贸易顾问纳瓦罗在内的多名美国高官都对媒体表示，当时正参与谈判的川普等人并不知情。
 <br/>
 <br/>
 当地时间12月6日晚间，白宫贸易顾问纳瓦罗（Peter Navarro）接受福克斯商业新闻网（FBN）《黄金时间》主持人里根（Trish Regan）的专访时，主动澄清说， 12月1日在阿根廷参加与中方举行的谈判晚宴的美方人员，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统和整个团队，都不知道司法部当天要采取抓捕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 的行动。
 <br/>
 <br/>
 〝当天入席之前，我们团队里的人反覆碰头，库德洛、努钦、博尔顿、蓬佩奥、莱特希泽、我还有总统，以及库什纳。〞 纳瓦罗回忆说：〝这件（抓捕的）事从没出现在我们的谈话里，我要告诉你，再追问也没有意义，司法部没有通知我们这些人。没通知有什么错呢？司法部的行动是独立的，这就是事情的真相。〞
 <br/>
 <br/>
 主持人里根却追问：川普总统正在进行一次异常重要的贸易谈判，难道就没人想到应该把这个〝异常关键〞的抓捕行动提前告知他？
 <br/>
 <br/>
 纳瓦罗回答称：〝不是那样的，在我们这个民主国家，司法部就是独立行事的，这就是民主的运转方式。〞
 <br/>
 <br/>
 对于这两件事恰恰发生在同一个时间点的问题，纳瓦罗表示，这么恰合的原因可能是因为美国与中共政府之间〝不公平竞争行为的矛盾扎堆出现〞，每天一开工〝这个部门、那个部门都正好要对中国出台举措〞这样的事情太多了。
 <br/>
 <br/>
 最后纳瓦罗还特意强调说，12月1日的晚宴〝意义非凡〞，证明了现任美国总统的力量，也见证了川普与习近平两人的友谊。
 <br/>
 <br/>
 事实上，不仅纳瓦罗明确澄清了川普事前对抓捕孟晚舟的行动并不知情，之前路透社也曾引述白宫一位官员表示，川普在与习近平会晤之前，并不知道美国执法机构向加拿大提出了引渡孟晚舟的要求。
 <br/>
 <br/>
 而且，还有另一名美国官员也强调，对孟晚舟的调查与拘捕是司法部的事，并不是白宫事先策划的。该官员还分析称，〝拘捕孟晚舟可能加大中美达成广泛贸易协议的难度，但并不一定会破坏这一进程。〞
 <br/>
 <br/>
 据公开的资讯，在阿根廷参加中美晚宴对谈的美方代表有，美国总统川普（Donald Trump）白宫国安顾问博尔顿（John Bolton）、国务卿蓬佩奥（Mike Pompeo）、财长努钦（Steve Mnuchin）、贸易代表莱特希泽（Robert Lighthizer）、国家经济委员会主任库德洛（Larry Kudlow），白宫首席贸易顾问库什纳（Jared Kushner）以及白宫贸易顾问纳瓦罗。
 <br/>
 <br/>
 <br/>
 有一个例外的情况是，美国全国公共广播电台（NPR）的美国国家公共广播电台（NPR） 记者Steve Inskeep12月6日在推特（Twitter）发文称，身为白宫国安顾问的博尔顿接受采访时承认，他事先通过司法部的例行简报得知孟晚舟将被逮捕。不过，他表示不知道总统川普事先是否知情。他说：〝这种事情经常发生，肯定不是每个案例都会上报给总统。〞
 <br/>
 <br/>
 当被追问孟晚舟被捕是否与阿根廷的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞晚宴同时进行时，博尔顿表示：〝就执法行动而言，你很难准确知道将要发生的情况，因为它取决于很多条件。〞
 <br/>
 <br/>
 由于舆论界有声音质疑抓捕孟晚舟的行动带有某种政治意图，加拿大总理特鲁多日前也在蒙特利尔对媒体澄清说：〝有关机构在没有任何政治介入或干涉的情况下做出了这一决定（抓捕过境的孟晚舟）。〞
 <br/>
 <br/>
 据公开的资讯，美国有关机构至少从2016年就开始调查华为公司是否违反了美国对伊朗的制裁，而近期的调查包括华为利用全球银行系统来避开美国的制裁令而与伊朗进行非法交易的问题。
 <br/>
 <br/>
 舆论认为，现在白宫努力撇清与孟案的关系，显示美方不希望华为案干扰到未来大约90天的中美贸易谈判。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
