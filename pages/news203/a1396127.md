### 〝不会蛀牙棒棒糖〞13岁执行长登亚马逊热销榜
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】美国一名年仅13岁的少女，靠着销售〝不会让你蛀牙的〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_棒棒糖.html" target="_blank">
  棒棒糖
 </a>
 ，成为一间年收入过亿元的棒棒糖公司执行长。公司卖出的棒棒糖更在今年夏天还登上亚马逊（Amazon）的热销榜，媒体节目也争相邀访。
 <br/>
 <br/>
 这家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_糖果.html" target="_blank">
  糖果
 </a>
 公司〝Zollipops〞位于底特律，市值超过数百万美元，公司执行长就是年仅13岁的女孩摩尔斯（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Alina Morse.html" target="_blank">
  Alina Morse
 </a>
 ）。公司以贩售〝不会让你蛀牙的〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_棒棒糖.html" target="_blank">
  棒棒糖
 </a>
 闻名遐迩。
 <br/>
 <br/>
 据《FOX》报导，摩尔斯称，她7时和父亲去银行办事，一名银行服务人员给了她一些棒棒糖，但她的父亲告诉她，不要吃太多糖果，因为这会伤害她的牙齿。当时她就问父亲，为什么不做一些健康且对牙齿有益的棒棒糖呢？〝这样我就可以吃糖果了。〞
 <br/>
 <br/>
 也就是在那时，摩尔斯有了想要自己开发对牙齿无害的糖果。不只是停留在想，她还采取行动上网找资料并询问店家，甚至还找上牙医、保健专家等等。
 <br/>
 <br/>
 起初，她们用自家的烤箱、微波炉做实验，之后父亲还直接带着摩尔斯找工厂合作，进行了上百次的实验，终于在她9岁时，成功研发出以木糖醇以及赤藻糖醇天然代糖为原料的糖果，可以减少吃糖导致的牙菌斑和口腔细菌，制造出不会蛀牙的棒棒糖。
 <br/>
 <br/>
 成功以后，摩尔斯还和父亲讨论起要利用零售商贩售他们开发的糖果，并在父亲的帮助下开了自己的公司，成为Zollipops的首席执行长。
 <br/>
 <br/>
 该公司还开发出一样不会伤害人牙齿的太妃糖，甚至还有洁牙糖，深受广大民众特别是小朋友的喜爱。产品在全美超过7500家商店上架，包括全食超市（Whole Foods）、克罗格（Kroger）、沃尔玛（Walmart）等，今年夏天还登上了亚马逊（Amazon）无糖类（sugar free）糖果热销榜。公司仅今年的零售额预计就在500到600万美元之间。
 <br/>
 <br/>
 摩尔斯表示，她成功的秘诀就是能够接受不同的建议，〝努力工作，进行各种尝试、以及相信大家且永不放弃。〞
 <br/>
 <br/>
 摩尔斯的父母则表示，女儿提出了很多关于产品的问题，〝我认为大多数优秀的CEO，就是擅长问问题，对任何事都充满好奇心，并勇于去求知，我们想帮助她找到答案，让她完成想要东西〞
 <br/>
