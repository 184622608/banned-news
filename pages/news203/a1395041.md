### 华日曝川普与习近平再会晤  专家小组正做准备
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】尽管美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 已经上升到全面对抗。但华尔街日报消息称，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）和中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，计划在11月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_20国峰会.html" target="_blank">
  20国峰会
 </a>
 期间举行会晤。目前美国已经组成专家小组为此次会晤做准备。
 <br/>
 <br/>
 11月30日至12月1日，20国集团会议将于阿根廷布宜诺斯艾利斯举行。
 <br/>
 <br/>
 《华尔街日报》引述消息人士的话称，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，计划在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_20国峰会.html" target="_blank">
  20国峰会
 </a>
 期间举行会晤。中方希望川习会晤，对中美贸易关系产生积极的影响。
 <br/>
 <br/>
 报导说，目前美国已经组成专家小组为此次会晤做准备，小组成员包括美国37任总统理乍得．尼克松的孙子，律师克里斯托弗．尼克松．考克斯。而中方负责人是中共国务院副总理刘鹤。
 <br/>
 <br/>
 10月2日，白宫首席经济顾问库德洛也曾表示，川普愿意在20国集团会议上，与习近平谈一谈。
 <br/>
 <br/>
 美国国务卿蓬佩奥10月8日对中国进行了访问，就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系.html" target="_blank">
  中美关系
 </a>
 及共同关心的国际和地区问题交换意见。
 <br/>
 <br/>
 对于此次中美高层重启对话，美国国务院发言人希瑟．奈特表示：在任何关系之中，都会遇到低点，但是，关系会随时间高低起伏，所以，美国肯定有很多东西跟北京谈。
 <br/>
 <br/>
 外界认为，中美关系在激烈对抗中，似乎又现出一线缓和的希望。
 <br/>
 <br/>
 在此之前，中美两国领导人已经进行了两轮会晤。第一次川习会晤，于2017年4月，在川普佛罗里达豪华私人庄园举行，当时川普对外宣告了他与习近平之间的〝友情〞。
 <br/>
 <br/>
 第二次川习会晤，于去年11月，在北京举行，习近平以从未给过任何一个外国元首的礼遇和荣誉，超规格接待了川普。之后，川普在多个公开场合，称赞习近平并说习是〝非常好的朋友〞。
 <br/>
 <br/>
 但美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 从7月6日正式开打后，川普不断强势加码，除了已有500亿美元商品征收25%关税外，9月24日起又追加2000亿美元商品，征收10%关税，自明年1月1日起，关税将升至25%。
 <br/>
 <br/>
 由于中共针对美国农民或产业进行报复，日前美国又祭出2670亿美元的商品清单，加征高关税。随着美中贸易战火日渐升高，两国关系日益紧张，已从贸易战上升到全面对抗。
 <br/>
 <br/>
 川普在9月26日在联大记者会上，指控中共企图干预美国11月中期选举，并首次坦言他跟习近平的交情可能受到损害，他两次提到，也许习近平不再是朋友了。
 <br/>
 <br/>
 10月2日，川普总统在美国的一次集会上，又一次指责中共，过去每年都从美国拿走5000亿美元，这简直太疯狂了，是时候叫停了。
 <br/>
 <br/>
 同日的另一集会上，川普说：中共〝正在榨干我们。他们拿走我们的钱重建自己。他们每天造一架飞机。每天都会造很多大桥，比华盛顿的大桥还大。〞〝我喜欢习近平主席。但是我说过，这个局面不能再持续下去了。〞
 <br/>
 <br/>
 分析人士认为，无论是习近平还是川普，12月的会谈都是关键环节，一旦共和党在11月中期选举中获胜，对于中共将是一个巨大打击，加之中共试图通过绑架14亿民众，来对抗美国之举已经失灵，许多中国民众暗中支持川普打垮中共，解体中共。另外，贸易战令中国经济下行压力巨大，分析认为，内外交困下习近平也期望能坐下来与川普心平气和的谈一谈。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明)
