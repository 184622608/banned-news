### 川普在任20个月 已实现289项选战承诺
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）上任至今已经满20个月，统计数据显示，在不到两年的时间里，川普政府已经实现了289项在大选中的承诺。
 <br/>
 <br/>
 这289项成就分属经济增长、就业、去监管、减税、边境安全、司法、以及贸易等18个大类别，其中包括173项重大成就和116项小型成就。
 <br/>
 <br/>
 而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 上任第二年的成就，比第一年超出一倍以上，许多在第一年还处于提案或启动阶段的项目，已经在第二年得以实现。
 <br/>
 <br/>
 其中的一些显著成就包括：
 <br/>
 <br/>
 当选至今，美国新增就业超过400万，就业人数达到史上最高，失业率则降至近50年来最低。
 <br/>
 <br/>
 2018年第二季度经济增长达到4.2%。
 <br/>
 <br/>
 超过4500亿美元的海外资金回流美国，美国三大股指不断刷新史上新高。
 <br/>
 <br/>
 实施大规模去监管行动。
 <br/>
 <br/>
 完成美国史上最大规模减税。
 <br/>
 <br/>
 提名坚守宪法的最高法院大法官，完成对戈萨齐及卡瓦诺两位最高法院大法官的确认。
 <br/>
 <br/>
 与加拿大、墨西哥达成新的北美自由贸易协定，并分别与韩国、日本及欧盟达成或展开新的自贸协定谈判。
 <br/>
 <br/>
 对中共多年来的不平等贸易及侵犯知识产权行为采取强硬措施，向2500亿美元的中国商品加征关税。
 <br/>
 <br/>
 退出伊朗核协议。
 <br/>
 <br/>
 承认耶路撒冷为以色列首都并搬迁大使馆。
 <br/>
 <br/>
 与金正恩会面，以最大化施压迫使朝鲜同意完全弃核。
 <br/>
 <br/>
 全面击溃伊斯兰国恐怖组织ISIS等等。
 <br/>
 <br/>
 周五，川普在推特上以〝有诺必践〞来总结过去20个月所取得的成就。
 <br/>
 <br/>
 至此，在不到两年的时间里，川普已经超越前总统里根在同时期所取得的成就，不仅对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 选民是强大激励，也为2020年竞选连任奠定了坚实基础。
 <br/>
 <br/>
 新唐人记者乔安综合报导
</div>
