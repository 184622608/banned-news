### 美国惊现大量可疑包裹 目的何在？
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】连日来，10多个可疑炸弹包裹邮寄给多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 高层以及支持民主党的知名人士，还有媒体大楼。涉案嫌犯已在10月26日被捕。适逢美国中期选举，事件引发广泛议论。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统谴责嫌犯的行动，表示绝不能让政治暴力在美国扎根。
 <br/>
 <br/>
 近期，含有爆炸装置的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_可疑包裹.html" target="_blank">
  可疑包裹
 </a>
 被寄送到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 高级政客或幕僚家中。
 <br/>
 <br/>
 22日，民主党大金主索罗斯位于纽约卡托纳县的豪宅收到可疑包裹。索罗斯当时没在家，拆弹小组技术人员到场后，引爆了包裹，没有人员伤亡。
 <br/>
 <br/>
 23日，克林顿和希拉里在纽约查巴克县的住址收到类似包裹，写明希拉里是收件人。
 <br/>
 <br/>
 24日，可疑包裹出现在前总统奥巴马在华府的住址。
 <br/>
 <br/>
 警察拦截下了寄送给希拉里和奥巴马的包裹。
 <br/>
 <br/>
 之后，民主党成员前副总统拜登，前司法部长霍尔德，前中情局局长布伦南，前国家情报总监克拉柏，国会众议员舒尔茨和沃特斯，参议员布克和贺锦丽，以及支持民主党的好莱坞影星罗伯特・德尼罗等人也都成为收件人。
 <br/>
 <br/>
 值得庆幸的是，所有这些〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸弹邮包.html" target="_blank">
  炸弹邮包
 </a>
 〞都在第一时间被发现，没有爆炸，也无人受伤。
 <br/>
 <br/>
 24日晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在威斯康辛州集会演说中，提到可疑包裹事件，重申保护美国人民安全。
 <br/>
 <br/>
 美国总统川普：〝作为总统，我的最高职责是保护美国安全。我们是这么说的，也是这么做的。联邦政府正在积极调查，会找出责任人，希望很快将他们绳之以法。〞
 <br/>
 <br/>
 同时，川普谴责暴力行为，表示政治暴力是对民主的攻击。
 <br/>
 <br/>
 美国总统川普：〝任何政治暴力的威胁和行为，都是对民主本身的攻击。没有哪个国家能够通过容忍暴力，或容忍以暴力进行政治恐吓和操控，取得成功。〞
 <br/>
 <br/>
 目前，距离美国中期选举只剩下十几天，这些〝炸弹邮包〞的出现，为这场关键选战蒙上阴霾。
 <br/>
 <br/>
 佛州大西洋大学政治学教授Marshall L. DeRosa：〝但是很遗憾有这样的事，尤其现在离选举这么近。就是疯子或疯狂的组织，想撕裂美国人，我觉得很悲哀。〞
 <br/>
 <br/>
 不过也有分析认为，这些〝炸弹邮包〞，并没有爆炸或引发伤亡，即嫌犯可能不想伤人，只是想达成某种〝恐吓〞，或者大规模影响舆论与民心的效果。
 <br/>
 <br/>
 美国执法领域专家 Rogers Canfield博士：〝无论是谁干的，处于何种目的，都是犯罪。时间点上相当巧妙，分散了大家对持续发生的暴民袭击共和党人的关注。〞
 <br/>
 <br/>
 26日，警方在佛罗里达州种植园市逮捕了56岁的涉案嫌犯塞亚克。
 <br/>
 <br/>
 公开资料显示，塞亚克现居住在佛州阿文图拉，此前曾有多次犯罪记录，其中包括炸弹威胁。
 <br/>
 <br/>
 美国司法部长塞申斯26日宣布，已控告塞亚克五项罪名。
 <br/>
 <br/>
 美国司法部长 塞辛斯：〝他（塞亚克）今天已被控五项联邦罪名，包括跨州运输爆炸物，非法邮寄爆炸物，威胁前总统等人，威胁跨州通讯，以及攻击现任及前任联邦官员。〞
 <br/>
 <br/>
 如果这些罪名全部成立，塞亚克将面临最高58年监禁，由于调查尚未结束，对塞亚克的控罪还有可能会增加或变动。
 <br/>
 <br/>
 目前，执法部门还没有确定塞亚克的作案动机。
 <br/>
 <br/>
 得知嫌犯被捕后，川普再次重申，不容许政治暴力在美国立足。
 <br/>
 <br/>
 美国总统 川普：〝我们绝不容许政治暴力在美国扎根，不允许这种情况发生。我承诺将会竭尽总统所能，现在就制止其发生。〞
 <br/>
 <br/>
 川普强调，美国人必须团结一致，向世界展示美国人的平和、关爱与和谐。
 <br/>
 <br/>
 撰稿：梁东
