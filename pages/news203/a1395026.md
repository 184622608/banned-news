### 全民医保是社会主义圈套？川普撰文论述
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统周三在《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_今日美国.html" target="_blank">
  今日美国
 </a>
 》发表专栏文章，尖锐批评了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党人.html" target="_blank">
  民主党人
 </a>
 提出的Medicare for All〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全民医保计划.html" target="_blank">
  全民医保计划
 </a>
 〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在文章中说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党人.html" target="_blank">
  民主党人
 </a>
 提出的〝Medicare for All〞的设想，其实质是建立一个政府管理运营的单一支付健保系统，排除一切私人健保方案和雇主提供的健保计划。
 <br/>
 <br/>
 美国企业研究所健保政策专家 美国国会预算办公室前助理主任 Joseph Antos：〝在医疗健康方面，确定的是，能够进入该领域的资金会变少；投资会减少；创新活力降低；新疗法的研发减少；等待服务的时间会变长。〞
 <br/>
 <br/>
 川普指出，对老年人而言，他们将无权选择自己想要的方案。一旦实施，头10年的花费就会达到32.6万亿美元。
 <br/>
 <br/>
 美国企业研究所健保政策专家 美国国会预算办公室前助理主任Joseph Antos：〝这个方案会导致联邦支出巨幅增加，这意味着更多的税收；而人们不一定能获得需要的保障计划。〞
 <br/>
 <br/>
 川普认为，这样导致的后果可能是大家都没有医保，最终连现在的〝联邦医疗保险〞（Medicare）这项福利计划都会消失。他并进一步指出，那是左派激进份子实现其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_社会主义.html" target="_blank">
  社会主义
 </a>
 目标的第一步。
 <br/>
 <br/>
 美国企业研究所健保政策专家 美国国会预算办公室前助理主任Joseph Antos：〝（Medicare for All）是不是社会主义（模式的制度）见仁见智吧。但这确实是联邦延深其控制力的表现，这会遭到抵制。〞
 <br/>
 <br/>
 在发表专栏文章的同一天，川普签署了两项法案，〝最低价格法〞和〝药物价格患者知情权法〞，让患者知道如何买到更便宜的药；降低处方药和生物技术仿制药（Generic Drug）价格，包括Medicare计划中处方保险部分（Medicare Part D）的药价，保护老年人福利。
 <br/>
 <br/>
 美国总统 川普：〝我们采取行动阻止医院对Medicare计划里的老年人过度收费，单单今年，这就将为他们节省数亿美元药费。 针对Medicare计划处方保险部分，我们运用新工具来帮助谈判降低药价。由于我们采取行动增加竞争并降低成本，Medicare计划处方保险部分的保费将连续第二年大幅下降。〞
 <br/>
 <br/>
 新法的通过是5月川普宣布的降低药价蓝图方案的一部分，包括辉瑞制药在内的15家药厂降价、冻结或推迟涨价。川普表示共和党人正在为将既往病纳入承保而努力。
 <br/>
 <br/>
 川普最后在专栏文章中提醒选民，如果民主党在中期选举后控制众议院，美国将可能走向社会主义——〝社会主义造成苦难、腐败和腐朽。〞
 <br/>
 <br/>
 新唐人记者李兰纽约报导
</div>
