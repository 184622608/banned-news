### 德州参议员席位之争 克鲁兹领先7个百分点
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】距离11月6日美国中期选举只有三周了，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 票仓德州的最大看点之一，是共和党参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克鲁兹.html" target="_blank">
  克鲁兹
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥罗克.html" target="_blank">
  奥罗克
 </a>
 之间的较量。目前民调，克鲁兹领先对手7个百分点。
 <br/>
 <br/>
 20年多来，德州一直是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 阵营，联邦参议员席位也由共和党执掌。2016年川普以九个百分点的优势击败对手希拉里。
 <br/>
 <br/>
 周二晚间，共和党参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克鲁兹.html" target="_blank">
  克鲁兹
 </a>
 和他的挑战者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥罗克.html" target="_blank">
  奥罗克
 </a>
 ，进行了选前最后一场公开辩论。最新民调显示，克鲁兹领先奥罗克7个百分点，
 <br/>
 <br/>
 很可能成功连任。
 <br/>
 <br/>
 共和党联邦参议员 泰德．克鲁兹：〝我尊崇宪法第一修正案保障的言论自由。国会不应限定产品规则，政府没有资格去限定内容和网络内容。我不愿意，我想德州人也不愿意，看到脸书谷歌等大公司删除网上的不同意见。〞
 <br/>
 <br/>
 奥罗克放弃了联邦众议员席位，与克鲁兹竞争。过去三个月筹款3800多万美元，打破参议员筹款记录。他的竞选预算将近2300万美元，是克鲁兹1130万美元的两倍多。克鲁兹指出是全美的左派人士在资助奥罗克。
 <br/>
 <br/>
 民主党期待着通过扩大拉美裔人口争取更多选票。川普总统推文说:
 <br/>
 <br/>
 〝数千人乘坐大篷车，畅通无阻地来从南部边境到我们国家。而民主党人不批准立法来保护我们国家，真是难以置信。这是共和党人的中期选举的重要议题！〞
 <br/>
 <br/>
 川普下周一将前往休士顿最大的NRG体育馆，为克鲁兹站台。2016年克鲁兹曾参选总统，但是川普胜出后，转而支持川普的政策。
 <br/>
 <br/>
 今年中共操控的媒体购买墨西哥电台，向美国广播。克鲁兹积极反对。
 <br/>
 <br/>
 川普称赞克鲁兹：〝泰德．克鲁兹为德州做了很多，大幅减税、减少法规，让德州的就业率达到历史新高。他维护宪法第二修正案，而奥罗克会全部否定。我一直鼎力支持克鲁兹！〞
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
