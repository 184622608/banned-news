### 美拟调查所有中国留学生背景 包括社群言论和电话记录
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】有美国媒体引述消息来源爆料称，出于对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 活动日益猖獗的担忧，美国政府欲加强对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国留学生.html" target="_blank">
  中国留学生
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_背景审查.html" target="_blank">
  背景审查
 </a>
 ，以防止引〝谍〞入室。而正在考虑的措施中，可能包括调查中国留学生的电话记录及社群言论，以便从中寻找其是否与中共政府组织有特殊关联。
 <br/>
 <br/>
 路透社11月30日在一篇报导中援引未具名的美国官员和国会消息人士透露了上述消息。报导表示，今年6月，美国国务院把航空、机械人和先进制造技术等领域学习的中国研究生的签证有效期，从原来的五年缩减至一年，主要就是为了〝遏制攸关国家安全的各项领域中的间谍与盗取知识产权活动〞。
 <br/>
 <br/>
 据称，川普政府正在考虑，是否在中国学生进入美国院校之前对他们进行额外的审查。包括检查学生们的电话记录，搜索他们在中国和美国社交媒体平台上的个人账户，以寻找他们赴美意图的可疑之处，以及是否与中共政府组织有关联。
 <br/>
 <br/>
 报导援引一位美国政府高层官员表示，美国执法部门曾经向政府人士提供过有关如何察觉间谍活动和网络盗窃的培训，现在估计他们也将向院校人士提供这类培训。
 <br/>
 <br/>
 〝中国派到这里的每位留学生都必须经过相关机构和政府的审批程序。〞一位高官对路透说：〝（美国）国务院帮助确保收到签证的学生符合条件，不会对美国国家利益构成风险。〞
 <br/>
 <br/>
 该官员表示，美国政府所采取的任何新的措施，都是为了求得一种平衡，既能防止间谍活动，又不要吓跑有才华的学生，而这会影响大学的财务收入，或阻碍科技创新。
 <br/>
 <br/>
 报导认为，加强对中国学生的审查是美国政府对抗中共的全面行动中的一部分。
 <br/>
 <br/>
 今年8月7日，美国总统川普在在新泽西一家高尔夫俱乐部宴请十几位美国顶尖企业高管和白宫高级官员时，曾不点名地说，来自某个国家的留学生〝几乎每一个人都是间谍〞。消息传出后，外界几乎不约而同地判断，川普说的〝这个国家〞就是中国。
 <br/>
 <br/>
 有人说这是对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国留学生.html" target="_blank">
  中国留学生
 </a>
 的成见、偏见。但问题是，为什么人们在这个问题上会出现如此惊人一致的判断呢？俗话说〝冰冻三尺，非一日之寒。〞 近些年来不断有中国留学生受控于中共做了不符合其学生身份之事的消息传出。
 <br/>
 <br/>
 据《美国之音》报导，今年4月，共和党籍众议员拉马尔．史密斯（Lamar Smith）在一次众议院的听证会上指出，中共学术间谍渗透美国各个高校获取科学技术，威胁美国的国家安全和经济安全的问题日益严重。
 <br/>
 <br/>
 一个比较典型的例子是：发明喷气式冲浪板的中国亿万富翁刘若鹏，就被指控盗窃了美国科学家的〝隐形斗篷〞等知识产权。
 <br/>
 <br/>
 据NBC的报导，刘若鹏2006年进入杜克大学，成为了〝超材料〞的专家史密斯（David Smith）博士的学生后，又带了2名中国人访问史密斯的实验室，并在3到6个月的时间，参加了几个项目，其中包括隐形斗篷。然后，他们趁史密斯不在时，偷偷对实验室进行拍照，并将照片和所有制造隐形斗篷设备的参数带回中国。
 <br/>
 <br/>
 刘若鹏2009年回中国后就创办起了自己的科技公司，现在市值60亿美元。2010年FBI启动了对他的调查。
 <br/>
 <br/>
 时事评论家杨恒均在接受《大纪元》的采访时曾表示，中共控制海外华人，把留学生当成私产，早已是公开的秘密，世上没有不透风的墙。中国人本来很聪明，因此海外留学生和华人华侨在各国科技、商场和重要机构都扮演着重要角色。但是因为常常被中共利用的原因，一旦受到监视或排挤，对整个华人都是沉重的打击。这意味着任何先进的科研部门和重要机构，都将可能跟中国留学生和华人子女们无缘。
 <br/>
 <br/>
 时事评论员蓝述则表示，海外华人和留学生被中共操控的举动，其实早被美国反间谍机构盯上了，只是没有动手。爱国没有错，想为国家做贡献也没有错，但不要干偷鸡摸狗的事，给中共当枪使。
 <br/>
 <br/>
 蓝述说：〝中国现在需要的不仅仅是科学技术，更需要价值理念和制度体系，这两者是不可偏废的。否则自以为干好事，实际是在作恶。〞
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
