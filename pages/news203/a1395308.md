### 又生变？美财长：G20川习是否会晤未定
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】美媒透露中美首脑计划在11月底在阿根廷举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_20国峰会.html" target="_blank">
  20国峰会
 </a>
 期间会晤。中方希望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晤，对中美贸易关系产生积极的影响。但美国财政部长13日透露，美中两国领导人是否会在G20峰会期间会晤，尚无定论。
 <br/>
 <br/>
 10月13日，正在印度尼西亚巴厘岛出席国际货币基金组织(IMF)会议的美国财长姆努钦（Steven Mnuchin），向美国媒体CNBC透露，美中两国领导人是否会于11月底在阿根廷举行的G20峰会期间会晤，尚无定论。
 <br/>
 <br/>
 姆努钦表示，要确保11月份的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 能够成行，仍有很多工作要做。但是否与中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 会晤，将由美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）自己决定。
 <br/>
 <br/>
 华尔街日报12日报导，川普和习近平计划在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_20国峰会.html" target="_blank">
  20国峰会
 </a>
 期间举行会晤。
 <br/>
 <br/>
 11月30日至12月1日，20国集团会议将在阿根廷布宜诺斯艾利斯举行。报导称，中方希望川习会晤，对中美贸易关系产生积极的影响。
 <br/>
 <br/>
 川普总统的〝首要中国问题权威〞保守派智库——哈德逊研究所(Hudson Institute)中国战略研究中心主任白邦瑞(Mike Pillsbury)，在接受福克斯财经频道(Fox Business)采访时表示，川普同意与习近平会晤的主要原因，是中方有望在贸易问题上做出一些重大让步。
 <br/>
 <br/>
 他说：〝我所得知的一些消息是，中方正在研究一套方案，他们试图了解川普总统的底线在哪儿，但显然还不确定。〞他续说，中方正在努力，但目前还没有满足美方的要求。而川习会是否能取得突破，这将取决于中方是否做出重大让步。
 <br/>
 <br/>
 白邦瑞还表示，川普总统的理想目标是在互惠的原则下，实现美中贸易的平衡，并大幅增加美国就业。中方可能会出于面子，既不承认，也不解决美方提出的强制技术转让和盗窃知识产权问题。但如果中方能够提出在两年内，连续大幅削减美中贸易赤字的方案，并有助于美国增加就业，川普总统才可能会给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 暂时划上句号。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：赵云)
