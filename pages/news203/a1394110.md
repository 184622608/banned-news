### 中共恶意芯片监控美企 美全面应对
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恶意芯片.html" target="_blank">
  恶意芯片
 </a>
 一事，专家认为，这表明中共已对美国构成最为严峻的安全挑战。
 <br/>
 <br/>
 针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 军事情报部门，在一家美国硬件公司生产的服务器中，植入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恶意芯片.html" target="_blank">
  恶意芯片
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国政府.html" target="_blank">
  美国政府
 </a>
 目前正在进行调查。这次共近30家公司受到影响，其中包括科技巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 ，以及一家大型银行和多家美国政府承包商。
 <br/>
 <br/>
 美国传统基金会安全问题专家 成斌：〝这件事影响巨大。如果他们感染硬件，就能绕过防病毒软件，自动制造出后门及安全漏洞。〞
 <br/>
 <br/>
 通常黑客攻击是通过软件方式进行，专家认为，如此大规模、国家级操控的硬件植入极其罕见，尤其中国是全球最重要的手机和电脑生产地。
 <br/>
 <br/>
 美国传统基金会安全问题专家 成斌：〝中国是全球贸易参与者，对此也需要全球性的应对。〞
 <br/>
 <br/>
 专家认为，目前中共已对美国构成最为复杂、严峻的安全挑战。川普政府也正在采取各种积极应对措施。
 <br/>
 <br/>
 美国传统基金会安全问题专家 成斌：〝这届政府采取了多种往届政府故意忽视的措施，尤其是金融与经济手段。〞
 <br/>
 <br/>
 周四，美国副总统彭斯发表重要讲话，披露中共对美国多方面的恶意影响活动，并发出严厉警告，其中包括由中共安全机构策划的对美国技术的全面盗窃。
 <br/>
 <br/>
 美国传统基金会安全问题专家 成斌：〝(问题是)美国会对此采取怎样的行动，针对中共在硬件芯片层面，影响全球供应链。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
