### 深圳官方赴美考察得结论：川普下台政策也不会变
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】中美对抗正从贸易战扩展到其他领域，海外舆论指
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 时代来临。中共体制内对此也有了一定认识。日前深圳市委智囊团报告，共和党在美国选举中稳操胜券，即使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）6年后下台，政策也不会变，中方应该尽快研究新冷战策略。
 <br/>
 <br/>
 自由亚洲电台10月15日报导，中美贸易战背景下，广州第124届广交会门庭冷落。珠三角地区正面临严峻的贸易形势。
 <br/>
 <br/>
 报导披露，中共深圳市委8月曾派出研究人员到美国考察，深圳市政府政策研究室日前已完成考察报告，并提交市委。
 <br/>
 <br/>
 报告指，现在美国的行政、立法、国防，在遏制中共的立场上高度一致。美国对中政策已全面改变，从以往的〝接触战略〞变成〝全方位围堵〞。
 <br/>
 <br/>
 报告强调，美国商界也普遍认为在中国遭到不公正对待。过去在中美关系紧张时，总有美国企业家出面斡旋。但是，这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 启动贸易战，反而得到美国企业家的普遍支持。
 <br/>
 <br/>
 报告认为，美国共和党在中期选举获胜已没有悬念，川普连任不成问题，其对中政策至少会延续6年。即使川普下台后，已经形成的战略也不会有大的调整。
 <br/>
 <br/>
 报告得出结论指，美国启动贸易战的意图，就是要将世界两大经济体彻底分割，重整全球产业链，重塑世界经济版图。
 <br/>
 <br/>
 报告也提醒深圳政府，美国可能在半年至一年内，在高科技层面对中共实施全面封锁。中方必须在全面封锁前的空窗期做好准备，以〝超常规手段〞加速引进科研仪器设备，并把投资在房地产的资金转移至高科技领域。
 <br/>
 <br/>
 报告还建议，中共政府要清醒认识到中美关系的严峻性，要以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 思路〞在各方面制定〝战事策略〞。包括以优厚条件加速引进〝战略性科技人才〞，特别是非华裔人才，但〝不宜大张旗鼓〞。还要鼓励华人学者〝加入其他国籍〞，参与国际科技活动。
 <br/>
 <br/>
 自由亚洲引述北京大学教授梁云祥表示，北京学术界圈子早已传阅了这份报告。梁云祥认为报告反映的只是地方政府的观点，并指目前用〝新冷战〞来形容中美关系〝为时尚早〞。
 <br/>
 <br/>
 其实，此前早就有中共体制内学者向北京表达过类似的观点，但并没有形成高层的认识。
 <br/>
 <br/>
 外媒消息，6月时，北京内部曾传阅安信证券首席分析师高善文的文章。文中指，中共官员对于即将到来的贸易战，似乎并没有思想准备。
 <br/>
 <br/>
 8月时，网络再传高善文的演讲稿指，他今年年初访问美国时就发现，当时华盛顿圈子里对美中贸易战的整体策划基本上已到了尾声阶段，但中共圈子里没有任何警惕意识。
 <br/>
 <br/>
 外界普遍认为，北京低估美国朝野反共意识的一致，是导致贸易战一再升级的原因之一。但直到今天，北京的误判似乎仍在继续。
 <br/>
 <br/>
 目前，川普和共和党的国内支持率不断上升。但据美国国土安全部部长披露，北京正在进行一项〝前所未见的努力〞，企图干扰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 。外界分析，北京仍在〝豪赌〞美国中期选举，期望民主党获胜，扭转美国贸易策略。
 <br/>
 <br/>
 日前还有中共体制内声音称，川普最多再掌权6年，中共〝完全有能力〞挺过去。
 <br/>
 <br/>
 美国副总统彭斯10月4日的演讲，凸显了和中共对抗的决心，被指〝新冷战宣言〞。但《金融时报》文章称，一位中共高级官员透露，彭斯的讲话在中共体制内引起了〝困惑〞。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
