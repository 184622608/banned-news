### 韩媒：华府不满韩国放宽对朝鲜制裁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜半岛局势.html" target="_blank">
  朝鲜半岛局势
 </a>
 升温，韩国似乎对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜制裁.html" target="_blank">
  朝鲜制裁
 </a>
 有所放松。据韩媒周一（29日）报导，美国专家警告，华府对韩国〝超速〞放宽对朝鲜制裁行为的反感程度，远超过韩国所感知。
 <br/>
 <br/>
 韩国《朝鲜日报》29日以〝美国官僚对文在寅总统的‘超速’感到担忧甚至愤怒〞为标题，在外交安保版头条报导称，韩美有关解除对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜制裁.html" target="_blank">
  朝鲜制裁
 </a>
 和韩朝经合等问题产生的分歧严重。
 <br/>
 <br/>
 报导说，美国智库传统基金会的首席研究员布鲁斯‧克林诺（Bruce Klingner），最近在会见韩国外交部记者团时称，尽管美国支持文在寅总统和他的努力，但从与美国政府有关人士对话得知，有部分人对文在寅的朝鲜政策感到担忧，甚至愤怒。该记者团是应美国国务院邀请访问华盛顿的。
 <br/>
 <br/>
 报导称，曾在美国中央情报局（CIA）担任朝鲜半岛分析官的克林诺也告诉记者团：〝华盛顿方面曾多次向文在寅传递强硬资讯，希望他在韩朝关系上‘放缓速度’。〞
 <br/>
 <br/>
 克林诺还说，文在寅想向朝鲜提供的经济优惠清单已经很长，而且项目还在不断增多，这很可能违反联合国安理会的制裁和美国法律。
 <br/>
 <br/>
 报导称，美国智库外交关系协会（CFR）首席研究员史奈德（Scott A.Snyder）透露，韩美出现意见分歧可追溯到平壤的韩朝高峰会谈，或更早以前。
 <br/>
 <br/>
 他还提及，韩国政府有意重启朝鲜开城工业园区和金刚山旅游，提议韩朝铁路对接，设立开城韩朝联络办公室和签署韩朝军事协议等。
 <br/>
 <br/>
 美国总统川普（特朗普）一直强调，只有朝鲜切实实践无核化，才能缓解对其制裁。
 <br/>
 <br/>
 朝鲜煤炭去年伪装成俄罗斯产的煤炭，运往韩国港口卸载，美国国务院8月罕见的发布韩文版〝对朝鲜制裁警报〞。韩国媒体报导，此举是对韩国可能违反联合国制裁规定而发出警告。
 <br/>
 <br/>
 韩联社10月11日引述韩国政府有关人士的话称，对朝制裁需要与美方配合，履行军事协议需要与联合国军司令部方面的合作，韩国政府很清楚这些情况，在与美国进行沟通和协调方面不会有一丝怠慢。
 <br/>
 <br/>
 10月15日，韩国和朝鲜在板门店韩方一侧的〝和平之家〞举行了第五次高级别会谈。韩朝双方在会谈中商定本月底举行体育会谈，启动铁路、公路连接项目，尽快举办将军级军事会谈，平壤艺术团赴韩演出等一系列话题。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
