### 川普以顶级团队结束助选 投票日坐镇白宫监督
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】周二（11月6日），美国国会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 日来临，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统则在紧锣密鼓地为各地共和党人助选、并以顶尖团队圆满结束竞选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集会.html" target="_blank">
  集会
 </a>
 后，于白宫密切关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 结果。
 <br/>
 <br/>
 白宫新闻发言人桑德斯（Sarah Sanders）周二上午表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在连续出席8州的11场竞选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集会.html" target="_blank">
  集会
 </a>
 及此前数周的助选后，当天将坐镇白宫，拨打电话监督各地选情。
 <br/>
 <br/>
 桑德斯说，川普还会与自己的政治团队会面，以便及时获知选情变化。
 <br/>
 <br/>
 桑德斯还表示，川普和第一夫人梅拉尼娅（Melania Trump）已邀请家人和朋友赴白宫，于晚间一同观看开票结果。
 <br/>
 <br/>
 川普和梅拉尼娅都已通过邮件，在他们的永久居住地纽约市投出自己的选票。
 <br/>
 <br/>
 川普已一再告诉选民，尽管他可能不会作为总统参加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 ，但他希望他的支持者们像他一样积极参与选举。
 <br/>
 <br/>
 川普尤其关注参议院的选情，他先后到数州为陷入激烈竞争的共和党人助选，力求增加共和党在参议院的议席，同时也努力保住共和党在众议院的多数党地位。
 <br/>
 <br/>
 为刺激选情，川普周一接连出席了三场竞选集会，从俄亥俄、印第安纳到密苏里州，到处都是潮涌般欢迎川普的人群。
 <br/>
 <br/>
 在密苏里的演讲中，川普还邀请一个颇有人气的顶级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选团队.html" target="_blank">
  助选团队
 </a>
 登台，其中包括美国拥有听众最多的电台主持人林堡（Rush Limbaugh）、福克斯新闻主持人汉尼提（Sean Hannity）、葛来美奖得主格林伍德（Lee Greenwood）、白宫新闻发言人桑德斯、白宫顾问康威（Kellyanne Conway）、共和党全国委员会主席麦克丹尼尔（Ronna McDaniel）以及第一千金伊万卡（Ivanka Trump）等。
 <br/>
 <br/>
 川普在选民的热情欢呼中完成了他在选举前的最后一场演说，周二凌晨才回到白宫。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
</div>
