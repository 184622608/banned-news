### 川普自曝访华演讲曾突然变调 习近平满脸愕然
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】10月27日，美国总统川普在伊利诺伊州的选举造势集会上，再次提及中美贸易，并回忆起2017年访华期间的一次演讲，引起哄堂大笑。
 <br/>
 <br/>
 川普在伊利诺伊州集会上说，多年来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易逆差.html" target="_blank">
  美中贸易逆差
 </a>
 平均达到5,000亿美元，但〝我告诉习主席，我很喜欢他〞，〝我说我不怪你们〞。
 <br/>
 <br/>
 川普接着说，自己当时正在北京发表讲话，激烈地谴责中方是如何欺诈美国的，〝然后我突然意识到我是在中国〞。
 <br/>
 <br/>
 讲到这里，台下爆出一阵笑声。
 <br/>
 <br/>
 川普接着回忆，〝我看向习主席，他抬头向上看，露出这到底是怎么一回事的表情。〞
 <br/>
 <br/>
 川普说，〝然后我看着他，我说，但是我不怪你们。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 感觉好多了。但是他想知道原因。我说，我不怪你们，因为我们的领导人没有做出行动，他们不应该让这种事情（贸易逆差）发生。〞
 <br/>
 <br/>
 据陆媒此前报导，2017年11月9日，川普在北京出席与中美商业领袖的会晤活动时，曾经强调中美贸易赤字的不公平，但他随后表示，〝这不怪中国〞，〝事实上我想要指责过去的美国政府，是他们默认了这种脱离控制的贸易逆差，让这种情况得以存在并持续增长。〞
 <br/>
 <br/>
 进入2018年后，川普政府逐步启动了针对北京的贸易战，中美关系开始转入紧张。但他每次提及中美贸易不公时，仍然沿用以前的习惯，同时表达对北京领导人习近平的尊重。
 <br/>
 <br/>
 不过，北京一直拒不纠正其不公平的贸易政策，川普政府逐步加大施压。随着贸易战步步升级，中美对抗已经转向全面冷战。中美首脑如果再次会面，其尴尬程度显然要超过一年前北京的那次演讲。
 <br/>
 <br/>
 据美媒消息，川普已确定让副总统彭斯代替他出席11月的亚太经合组织峰会。外界分析，这是在贸易冲突中有意回避和习近平的会面。
 <br/>
 <br/>
 不过，在11月底的G20首脑峰会上，两人将不可避免地有一次接触。
 <br/>
 <br/>
 但彭博社消息称，如果北京不能提出解决美方贸易关切的具体方案，G20的短暂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晤中，将不会包括贸易议题，而只关注朝核等公开分歧不大的话题。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
