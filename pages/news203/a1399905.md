### 佛州人工计票出台结果 川普赞胜选参议员
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】经过两轮重新计票，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议员.html" target="_blank">
  参议员
 </a>
 选举周日（11月18日）终于出台结果，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 候选人斯科特（Rick Scott）战胜对手尼尔森（Bill Nelson），最终赢得了这场苦战的胜利。
 <br/>
 <br/>
 周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人工重新计票.html" target="_blank">
  人工重新计票
 </a>
 结果显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 现任州长、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议员.html" target="_blank">
  参议员
 </a>
 候选人斯科特所获得的选票仍高出其对手、现任民主党参议员尼尔森10,000余张。这一结果使得尼尔森挽回败局的希望完全落空。
 <br/>
 <br/>
 斯科特当天发表声明表示，他已和尼尔森谈过，尼尔森大方地承认失败，他也对尼尔森多年来为公众的付出表示感谢。
 <br/>
 <br/>
 11月6日选举夜的初步选举结果显示，斯科特领先尼尔森50,000多张选票。但是在接下来的几天中，倾向于民主党且迟迟不完成计票工作的布劳沃德县（Broward）和棕榈滩县（Palm Beach）突然多出数万张选票并多数流向民主党候选人，导致斯科特的优势被巨幅削弱，至11月10日下午，斯科特的领先幅度已被压缩为0.15%，强制性重新计票随之启动。
 <br/>
 <br/>
 15日，根据机器重新计票的结果，斯科特的优势不但仍旧保持而且还有所扩大，斯科特再次宣布自己获胜。但由于尼尔森仍不承认失败，且双方相差幅度低于0.25%，所以佛州开启了第二次重新计票——人工计票。
 <br/>
 <br/>
 选举夜过后，一群民主党律师抵达佛州，企图通过更改规则的方式改变选举结果。但事与愿违，不但共和党最终仍赢得了佛州参议员和州长两项重要选举的胜利，而且，导致重新计票风波的声名狼藉的布劳沃德县选举监督员斯奈普斯（Brenda Snipes）也在人工计票结果出台几小时后被迫辞职。
 <br/>
 <br/>
 川普总统周日发表推文，祝贺斯科特来之不易的胜利。
 <br/>
 <br/>
 〝从第一天开始，里克．斯科特就没动摇过。他是一位伟大的州长，而且将成为代表佛州人民的甚至更为伟大的参议员。祝贺里克，开展了一场如此勇敢而成功的竞选！〞
