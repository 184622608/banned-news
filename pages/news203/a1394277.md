### 【禁闻】彭博：美国主板遭中共间谍植入芯片
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】《彭博商业周刊》日前披露，美国的服务器主板龙头〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_超微电脑.html" target="_blank">
  超微电脑
 </a>
 〞在中国的分包商遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共军方渗透.html" target="_blank">
  中共军方渗透
 </a>
 ，生产的主板被植入微型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恶意芯片.html" target="_blank">
  恶意芯片
 </a>
 。目前已知近30家公司受到波及，其中包括科技巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 。
 <br/>
 <br/>
 据《彭博商业周刊》10月4号发表的长篇报导，2015年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 计划并购影音软体新创公司Elemental，对该公司进行安全评估时，意外发现Elemental的服务器主板上，比原本的设计，多出了一颗只有白米般细小的芯片。亚马逊得知情况后，立即向美国政府通报情况。美国随即展开秘密调查，并证实这颗芯片会自动打开一个隐藏的后门，允许黑客藉此管道进入到服务器中。因此，调查单位认为，有心人士已经藉由这个方式渗透到美国核心的网路系统内。
 <br/>
 <br/>
 报导说，Elemental的服务器由超微公司负责组装，广泛用于美国国防部数据中心、CIA无人机操作，以及海军战舰网络，然而，超微的主板几乎全部在中国制造。美国调查人员监听发现，中共政府透过中间人，与超微在中国的四家分包工厂接洽，并要求更改主板设计，期间提供贿赂，并威胁不配合就关厂。一旦安排好细节，这位中间人会将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恶意芯片.html" target="_blank">
  恶意芯片
 </a>
 交给工厂进行装嵌。
 <br/>
 <br/>
 旅美时事评论员邢天行表示，硬件攻击的危害性非常大，因为被植入的恶意芯片要查找起来非常困难。
 <br/>
 <br/>
 旅美时事评论员邢天行：〝在底层硬件上植入间谍的晶片之后，查找起来是非常困难的。芯片有这种建立后门的功能，它平时是处于一种休眠状态，一旦计算机在运行当中，某一些数据触及了他设定的东西之后，它就开始启动，开始搜集相关数据。那你平时，即使你在外部用网络监视你这个动态的信息流，你也是查找不出来的。它启动之后，你的最关键的数据已经被它盗取，发送之后你才能够发现，但是那个时候，损失已经造成了。〞
 <br/>
 <br/>
 报导引述十几位不具名的美国情报界和科技界人士称，美国政府正在调查这起发生在2015年的黑客攻击，并认为中共军队的一个情报部门应对此负责。报导说，中共的目标，是长期获取高价值的企业机密，和敏感的政府网络资讯。
 <br/>
 <br/>
 旅美时事评论员郑浩昌表示，这件事比较像是中共总参三部，也就是技术侦察部所为。2014年有5名中共的军方黑客被美国起诉，他们当时所在的部队就隶属总参三部。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝总参三部主要手段就是监听，但是它在面对美国的时候，始终有个头痛的问题，就是和美国隔着一个太平洋，监听难度比较大，所以它具备通过硬件直接入侵美国本土的动机。而总参三部又有总参信息化部之类的给它提供技术支持，总体的硬件技术能力比国安要强不少，所以它又具备了在硬件上做手脚的能力。最后呢，因为党管一切，只要上头发话，中国的企业都会乖乖就范，所以它又具备了做手脚的制度环境。〞
 <br/>
 <br/>
 超软是全球最大的服务器主板供应商之一，客户遍布全球100个国家，一名前美国情报官员形容，攻击超微主板有如攻击全世界。目前已发现有近30家公司受到恶意芯片的影响，其中包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 、亚马逊、一家大型银行和多家美国政府承包商。不过，苹果和亚马逊都否认遭到中共黑客芯片攻击。
 <br/>
 <br/>
 郑浩昌：〝这种事情，我觉得还是真实发生了，不是空穴来风。商家否定这个可能是因为有商业利益在里面。〞
 <br/>
 <br/>
 邢天行：〝超微产品是比较低价位，很多公司就愿意用它，在硅谷也有许多伺服器用超微的，最终导致一旦出问题了，它这个危害波及面也大。〞
 <br/>
 <br/>
 目前全球有75%的手机和90%的个人电脑在中国生产，报导说，这次发现的重大安全隐患，可能促使企业将生产链撤离中国。
 <br/>
 <br/>
 采访/常春 编辑/陈洁 后制/周天
