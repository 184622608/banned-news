### 川普：美国和沙特关系破裂会让中共高兴
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】周二，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 谈到，美国与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 的关系不会因卡舒吉事件改变。作为总统，他首先考虑的是国家利益。制裁沙特会让美国经济受损，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 高兴。
 <br/>
 <br/>
 周二，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 回答记者提问时表示，在美国与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 关系上，他首先考虑的是美国利益。
 <br/>
 <br/>
 美国总统川普：〝我们不准备放弃数千亿美元订单，让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 或其他人得到它。对我来说，非常简单，就是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国优先.html" target="_blank">
  美国优先
 </a>
 〞。〞
 <br/>
 <br/>
 川普说，美国和沙特关系破裂，油价将飞涨。
 <br/>
 <br/>
 美国总统川普：〝沙特阿拉伯，如果我们与他们断交，我认为你们的油价将飞涨。他们帮助我（将油价）降下来。〞
 <br/>
 <br/>
 川普还表示，保持和沙特的关系，有利于制衡
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 。
 <br/>
 <br/>
 美国总统川普：〝坦率地说，未来某个时期，也许伊朗会发生积极变化，但是我们也需要保持平衡。以色列也需要帮助。如果我们放弃沙特阿拉伯，那将是一个可怕的错误。〞
 <br/>
 <br/>
 川普表示，自己和沙特没有私人业务往来。
 <br/>
 <br/>
 美国总统川普：〝沙特阿拉伯和我没有任何关系，我考虑的是‘美国优先’。〞
 <br/>
 <br/>
 川普还说，如果美国不和沙特做生意，沙特会从俄罗斯和中国购买武器装备。
 <br/>
 <br/>
 美国总统川普：〝他们（沙特）从美国购买数千亿美元的东西。如果我说我们不想接受你的业务，如果我说我们要切断它，他们会从俄罗斯和中国，买到这些设备，武器装备和其他东西。〞
 <br/>
 <br/>
 川普还表示，如果美国和沙特断交，俄罗斯和中共高兴。美国通过贸易谈判，有效制约了中共和俄罗斯。
 <br/>
 <br/>
 美国总统川普：〝俄罗斯和中共会非常高兴，因为我们对中国（贸易战）做得很好，我们对俄罗斯在内的所有（国家）做得都很好，我会继续这样做。〞
 <br/>
 <br/>
 川普还说，他不会破坏和沙特的关系，因为那样会破坏世界和美国的经济。
 <br/>
 <br/>
 新唐人记者毕心慈、尚敬综合报导
</div>
