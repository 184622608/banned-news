### 威斯康星共和党连夜投票 限制州长权力
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】川普在2016年总统大选拿下关键摇摆州威斯康辛州，而当地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 籍州长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沃克.html" target="_blank">
  沃克
 </a>
 却在今年11月的中期选举中败给民主党对手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃弗斯.html" target="_blank">
  埃弗斯
 </a>
 。如今，州议会的共和党议员连夜投票提案，准备削弱即将上任的新州长的权力。
 <br/>
 <br/>
 周二夜间，由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 控制的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威斯康星.html" target="_blank">
  威斯康星
 </a>
 州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 以17对16票通过提案，削弱新州长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_埃弗斯.html" target="_blank">
  埃弗斯
 </a>
 制定法规政策的权力，并赋予议会任命该州经济发展局董事会成员的权力，直到9月1日。
 <br/>
 <br/>
 这一提案周三将在州议会表决。
 <br/>
 <br/>
 明年1月，民主党籍的埃弗斯将宣誓就任州长，未来两年内，共和党占据州议会两院的多数席位。
 <br/>
 <br/>
 威斯康星议会发言人弗斯表示，这一举动是根据新形势重新分配权力。
 <br/>
 <br/>
 威州议会发言人 弗斯：〝我们要确保每个部门的权力尽可能相等。〞
 <br/>
 <br/>
 民主党人则认为，共和党人是因为中期选举失利而想要夺回更多地盘。
 <br/>
 <br/>
 威州议会民主党代表 泰勒：〝共和党人不应想着扩大自己的势力，他们在州长选举失败后感到震惊。〞
 <br/>
 <br/>
 共和党籍议员也寻求重新设定2020年初选日期，以此缩短提前投票的时限，外界认为，这将有利于共和党候选人。
 <br/>
 <br/>
 如果威州议会通过这项提案，其他共和党执掌议会，民主党人当选州长的几个州，有可能参考这一模式。
 <br/>
 <br/>
 这将对〝平价医保〞等问题产生较大影响。
 <br/>
 <br/>
 新唐人记者王羽音、张琪综合报导
</div>
