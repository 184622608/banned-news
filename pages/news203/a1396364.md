### 中共威胁2.0版 川普政府积极应对
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】最近，美国高级安全官员公开表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 是美国迄今所面临的最大反
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍.html" target="_blank">
  间谍
 </a>
 威胁。专家认为，美国政府对中共的长期误判，是导致其威胁不断升级的原因，不过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 正在扭转这一形势。
 <br/>
 <br/>
 《华盛顿自由灯塔》资深编辑 Bill Gertz：〝我们是否误判了中国？我的回答很响亮，‘是的’。〞
 <br/>
 <br/>
 美国著名专栏作家比尔．格茨(Bill Gertz)曾在2000年发表了调查撰写的《中国威胁》一书。他认为，长期以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的宣传、收买与影响，导致美国政府一直忽视中共对美国安全构成的严重威胁。
 <br/>
 <br/>
 《华盛顿自由灯塔》资深编辑 Bill Gertz：〝前国务卿赖斯否认中国是威胁，甚至曾说，我们希望一个‘强大的中国’。猜猜结果如何？我们真的有了个‘强大的中国’，并正在影响美国制度的各个层面。〞
 <br/>
 <br/>
 比尔．格茨说，他在《中国威胁》一书中谈到的很多问题，包括中共的军事扩张、情报搜集和技术窃取，在过去18年中均呈现爆炸势增长，如今可称之为〝中国威胁2.0版〞。
 <br/>
 <br/>
 情报专家也指出，近年来，盗窃经济和商业机密，已成为中共对美情报活动重点。同时，中共以类似人海战术的方式(沙粒理论)、分层搜集情报，并利用各种民间机构为掩护，完全不同于传统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍.html" target="_blank">
  间谍
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 纽约分局反间谍特工主管 Charles McGonigal：〝从2000年以来，我们就看到与经济间谍相关的盗窃行为急剧增加，并受中共政府掌控。〞
 <br/>
 <br/>
 〝共产主义受难者基金会〞中国研究员 Peter Mattis：〝美国的情报界绝对不会说，布鲁金斯学会、波音、JP摩根、麻省理工(负责窃取情报)，这不是美国的情报运作方式。但在沙粒理论指导下，中共情报系统包括所有类似机构。〞
 <br/>
 <br/>
 目前，中共影响力渗透已引起美国政府和国会高度关注。川普总统不久前在联合国安理会上公开点名，中共试图影响美国中期选举。上周五，美国多个重要执法部门发表联合声明，要严阵以待，对抗外国干扰美国选举的威胁，其中特别点名中共、俄罗斯及伊朗。
 <br/>
 <br/>
 战略与国际研究中心技术政策副主任 William Carter：〝我们还没有好的防御方式；他们尚未受到威慑；以往的得手令他们更胆大。(因此)我们将看到这种影响力操作继续存在。这也是我们担忧的。〞
 <br/>
 <br/>
 《华盛顿自由灯塔》资深编辑  Bill Gertz：〝但我认为，如果川普坚持其方向，他最终会在对抗中获胜。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
