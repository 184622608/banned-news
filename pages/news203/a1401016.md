### 纽约上州圣诞点灯 法轮功学员受邀游行
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】上周末，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约上州.html" target="_blank">
  纽约上州
 </a>
 Middletown市庆祝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞树.html" target="_blank">
  圣诞树
 </a>
 点灯仪式和游行。去看看他们的节日传统。
 <br/>
 <br/>
 Middletown市庆祝2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_圣诞树.html" target="_blank">
  圣诞树
 </a>
 点灯仪式，这是他们第23年的传统。圣诞树是一个年轻家庭捐献的。
 <br/>
 <br/>
 Middletown市民Justin和Kim Foglia：〝我们捐赠了圣诞树。 我们不久前搬到这里，我们想为社区做一些事情。〞
 <br/>
 <br/>
 当天城市的庆祝活动还有游行，大约4千人观赏。
 <br/>
 <br/>
 Middletown市长Joseph DeStefano：〝我们超过500人参加游行，大约4千人在今天庆祝。〞
 <br/>
 〝市长经济发展团队尝试在市中心创造一些愉悦将人们带到社区，庆祝节日。〞
 <br/>
 <br/>
 在西方人占96%的Middletown市，各族裔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 组成的游行队伍为整个游行增加了一道道亮丽有特色的风景线。 市长赞赏所有游行团体展现的传统。
 <br/>
 <br/>
 Middletown市长Joseph DeStefano：〝我们赞赏所有参加的不同团体，童子军，法轮大法到消防局、警察局。这是对社区的庆祝。 〞
 <br/>
 <br/>
 新唐人记者站纽约橙县报导
</div>
