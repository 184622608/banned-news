### 亚马逊总部即将搬来纽约 商家居民持续抗议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】自从亚马逊第二总部即将搬来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 的消息公布后，纽约部分居民为此一直争议不断，周一，在曼哈顿的6大道34街又有居民抗议亚马逊进驻纽约。
 <br/>
 <br/>
 抗议居民拿着横幅，和画着生气涂鸦的邮件在6大道34街聚集。之后抗议民众拿着标语沿街游行，直达亚马逊在34街的书店，并冲入书店进行抗议。
 <br/>
 <br/>
 很多抗议民众认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约.html" target="_blank">
  纽约
 </a>
 市府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违规操作.html" target="_blank">
  违规操作
 </a>
 ，没有经过任何社区谘询而直接操作，给亚马逊接近30亿美元的补贴。
 <br/>
 <br/>
 抗议者 Skipp Roseboro：〝亚马逊要来这里建总部，社区没有机会参与讨论，亚马逊会收到30亿的补贴，没有人有机会提出质疑。〞
 <br/>
 <br/>
 同时还有小商业主到场抗议。
 <br/>
 <br/>
 小商业主 Marni Halasa：〝我主要是来帮助小商业主，像亚马逊这样的大公司，会对小商业影响很大，对我而言是一个很大的问题，这就是我为什么要穿成这样来这里。〞
 <br/>
 <br/>
 抗议者 Jason Walker：〝我们希望组织给州府纳税的纽约人，让我们的纳税钱，用在自己身上，而不是补贴大公司。〞
 <br/>
 <br/>
 新唐人记者王真纽约报导
</div>
