### 中共疑窃美绝密卫星技术  案涉北京前高层
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】中共政府被曝通过投资一家洛杉矶的初创公司，向美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波音公司.html" target="_blank">
  波音公司
 </a>
 订购一颗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卫星.html" target="_blank">
  卫星
 </a>
 ，意图取得波音公司的卫星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绝密技术.html" target="_blank">
  绝密技术
 </a>
 。美媒称，此事件中负责洽谈的是中共前国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_耿飈.html" target="_blank">
  耿飈
 </a>
 之子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_耿志远.html" target="_blank">
  耿志远
 </a>
 。
 <br/>
 <br/>
 《华尔街日报》5日报导称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波音公司.html" target="_blank">
  波音公司
 </a>
 位于洛杉矶的工厂内，即将完成制造一款使用美国军方依赖的限制级技术的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卫星.html" target="_blank">
  卫星
 </a>
 ，这个卫星的订购方，是寻求改善非洲网路访问的美国本地初创公司Global IP。
 <br/>
 <br/>
 报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国东方.html" target="_blank">
  中国东方
 </a>
 资产管理公司的子公司，2015年通过在英属维京群岛的境外投资公司Bronzelink，为总部位于洛杉矶的初创公司Global IP投资了1.75亿美元，并收购了该公司75%的权益，以支持公司向波音订购卫星。
 <br/>
 <br/>
 中国东方归中共财政部所有，负责洽谈的公司高层是前中共国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_耿飈.html" target="_blank">
  耿飈
 </a>
 之子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_耿志远.html" target="_blank">
  耿志远
 </a>
 。然而，在2016年8月签约前，Global IP中方董事毫无预警地造访洛杉矶，要求检视合约中的卫星设计，遭到两位创始人的拒绝。
 <br/>
 <br/>
 Global IP的两名创始人萨德和贾韦德表示，在意识到Bronzelink并未真正独立于中共政府后，两年以来一直告诉波音公司，中共政府的资金正在支持他们订购这个卫星的项目。
 <br/>
 <br/>
 最后两名创始人选择了辞职，并于2017年向中国东方旗下子公司发起诉讼。诉讼书中称，国有中国实体以欺诈手段接管了卫星项目，使创始人面临违反美国法律的风险。两人寻求获得损害赔偿。
 <br/>
 <br/>
 华尔街日报称，事实上，根据公司记录、法庭文件以及该项目知情人士所掌握的信息，这颗卫星得到了中共政府大约2亿美元的资金。这些资金从一家中国国有金融公司流入该卫星项目，这笔复杂交易通过数家离岸公司，将中共的资金引入了波音。
 <br/>
 <br/>
 目前，这颗卫星即将在波音位于洛杉矶的工厂内建造完成，即将开展测试，最快将在明年春季由美国太空技术探索公司（Space X）发射升空。
 <br/>
 <br/>
 尽管这颗卫星最初的目的是改善非洲的网路服务，但分析人士担心，卫星进入轨道后，中共政府将更改其用途，中共发展的太空计划需要先进的卫星技术，但美国的法律禁止对中共出口卫星技术。
 <br/>
 <br/>
 曾任美国国家情报工作主管的布莱尔上将说：〝让我困惑的是，波音会在知道中国政府参与其中的情况下继续推进该项目。他们经常处理此类项目，他们知道美国这方面法律的内容和用意。〞
 <br/>
 <br/>
 波音公司在一份声明中称，他们制造这枚卫星是获得了美国商务部出口批准的。但是波音拒绝说明他们在申请批准时，是否告诉美国商务部负责人有关这个项目的资金来源。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
