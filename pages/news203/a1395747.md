### 德州中期选举辩论 川普推文挺克鲁兹
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】美国中期选举，各州选战激烈。德克萨斯州近30年来，一直是共和党占上风，民主党没有在州一级政府选举中获胜过。周二晚，民主党候选人贝托与竞选连任的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党参议员.html" target="_blank">
  共和党参议员
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克鲁兹.html" target="_blank">
  克鲁兹
 </a>
 ，进行首场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电视辩论.html" target="_blank">
  电视辩论
 </a>
 。
 <br/>
 <br/>
 民主党候选人奥罗克展开攻势，指责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克鲁兹.html" target="_blank">
  克鲁兹
 </a>
 不诚实。克鲁兹回应说，事实胜于雄辩。
 <br/>
 <br/>
 德州参议员 克鲁兹：〝很显然奥罗克众议员的民调机构让他出来攻击我。如果他想侮辱我，骂我是说谎者，这没关系。但是，大家知道，约翰・亚当斯 有句名言 〝事实难以改变。〞〞
 <br/>
 <br/>
 克鲁兹举例说，奥罗克曾投票赞成每桶油收10美元的税，法案虽没通过，但对以石油产业为生的德州人来说，影响很大。
 <br/>
 <br/>
 双方在建造边境墙议题上，看法不同。克鲁兹表示会支持川普建墙，奥罗克则寻求其他方案。
 <br/>
 <br/>
 双方还就全民健保、堕胎法规、气候变化、大法官卡瓦诺的提名等议题展开辩论。
 <br/>
 <br/>
 周三一早，川普发推文力挺克鲁兹。
 <br/>
 <br/>
 推文说：〝看了昨晚的辩论。贝托・奥罗克想要更高税收、更多法规，与泰德・克鲁兹意见相左。克鲁兹是了不起的德州人的代表、是他们想要的。〞
 <br/>
 <br/>
 川普的另一则推文说：〝泰德・克鲁兹为德州做了很多，包括大幅减税和减少法规，这让德州的就业率达到历史最高。他谨慎维护第二修正案，奥罗克会全部否定。我一直鼎力支持泰德！〞
 <br/>
 <br/>
 根据最新民调，克鲁兹大比分领先奥罗克。分析认为，辩论会后，克鲁兹的领先优势还会加大。
 <br/>
 <br/>
 新唐人记者安心田瑞综合报导
</div>
