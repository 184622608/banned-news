### 希拉里助民主党人竞选 纽时编委会成员〝要她闭嘴〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】纽约时报的一名编委会成员近日撰文称，前国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希拉里.html" target="_blank">
  希拉里
 </a>
 ・克林顿在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 中已成为〝广泛不受欢迎〞人物，他们希望她远离民主党的中期竞选活动，否则可能帮助激活
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 选民的热情。
 <br/>
 <br/>
 这篇题为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希拉里.html" target="_blank">
  希拉里
 </a>
 ・克林顿的分心大师班〞（Hillary Clinton’s Master Class in Distraction）的文章发表于10月17日。文章还配以副标题说
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 需要聚焦在中期选举上（Democrats need to be focused on the midterms）。文章的作者是纽时编委会成员米歇尔・科特尔（Michelle Cottle）女士，她在文章中明确表达出，希拉里没有为民主党的中期选举起到好作用。
 <br/>
 <br/>
 文章说，希拉里是民主党内最具代表性也是最具争议的人物，她最近的言论已经分散了选民对民主党中期选举的注意力。民主党需要谈论选民关心的医保和就业问题，或选民强烈关注的其他问题，而没有必要谈论弹劾现任美国总统川普（特朗普）的问题，也没必要谈论民主党参议员伊丽莎白・沃伦（Elizabeth Warren）的DNA测试问题。
 <br/>
 <br/>
 对于希拉里10月9日接受美国有线电视新闻网（CNN）采访时说，不能对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 采用〝文明的方式〞，科特尔说，民主党需要努力争取独立人士和摇摆派选民，而希拉里这样的言论令人深感不安，不能为选民带来鼓舞人心的消息，反而极可能会激活更多的共和党选民。
 <br/>
 <br/>
 文章中说，希拉里被很多民主党人认为是〝终极笨蛋〞（ultimate boogeyman），共和党人在遇到麻烦需要激励他们的选民时，没有比提一下〝歪曲的希拉里〞（Crooked Hillary）更好的办法了。
 <br/>
 <br/>
 科特尔接着说，更不幸的是，希拉里在不到一周的时间里又为川普和共和党提供了一份〝更加美味的中期选举礼物〞。
 <br/>
 <br/>
 10月14日，希拉里在哥伦比亚广播公司（CBS）的〝周日早晨〞节目说，她的丈夫，前总统比尔・克林顿没有为和白宫实习生莫妮卡・莱温斯基的性丑闻而选择辞职是对的，也不应该被弹劾，并称这不是滥用权力。希拉里还说性丑闻事件中的女主角当时是个成年人。
 <br/>
 <br/>
 科特尔说，当希拉里称莱温斯基女士是一名成年人时，似乎是说克林顿性丑闻事件具有了〝技术合法性〞一样。科特尔认为，希拉里在这个问题上走得太远了，特别是克林顿对待这些性丑闻的反应是试图诋毁事件中女性的声誉，这已不是秘密。
 <br/>
 <br/>
 现年72岁的柯林顿曾多次身陷性丑闻，包括阿肯色州政府前职员宝拉．琼斯（Paula Jones）控告时任阿肯色州长的柯林顿1991年对她性骚扰，获赔85万美元。
 <br/>
 <br/>
 去年又有4名女性分别宣称，欲控告柯林顿10多年前对其进行性攻击。而希拉里竟欲挖受害者〝黑料〞阻丑闻曝光。
 <br/>
 <br/>
 已经75岁的胡安妮塔・布罗德里克（Juanita Broaddrick）女士多年来一直指控克林顿在1978年强奸了她，她抨击民主党不理睬她的指控，对待性侵指控采取双重标准。
 <br/>
 <br/>
 科特尔在文章中称，希拉里为克林顿的辩护显示她无可置疑地是克林顿的根本同谋。
 <br/>
 <br/>
 保守派战略师克里斯・巴伦（Chris Barron）接受《福克斯新闻》采访时提到这篇文章，巴伦认为，希拉里不会在乎《纽约时报》和民主党对她的警告，她从来做的事都是符合她自己最佳利益的，即使牺牲了她的政党和政治盟友也不在乎，这就是克林顿夫妇的做事方式。
 <br/>
 <br/>
 媒体评论家杰弗里・麦考尔（Jeffrey McCall）则认为，希拉里的评论和状态让选民们想起为什么她不能赢得2016年总统大选，即使对她有那么多有利的因素。
 <br/>
 <br/>
 麦考尔认为，《纽约时报》内部已经看明白了，希拉里这位〝民主党老兵〞越露脸可能对民主党后来者的竞选活动变得越困难。
 <br/>
 <br/>
 美国将在11月6日进行中期选举。随着中期选举的临近，川普的助选活动也越来越频繁。川普和共和党人都希望共和党能够在参议院和众议院继续占据多数，这将更有利于川普今后推行他的政策。
 <br/>
 <br/>
 目前中期选举的注册选民，已经超过上次同一时期的两倍。有报导分析，川普的影响力，可能是促成这一现象的重要原因之一。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
