### 川普：民主党若在众院开展调查 将被反击
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】11月6日美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，共和党在参议院胜选，民主党则拿下了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 的主导权。美国总统特
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）周三（11月7日）发推文，警告民主党如果在众院开展调查，共和党将被迫反击。
 <br/>
 <br/>
 早在大选前民主党大谈〝蓝波〞之说时，他们就一再威胁称，一旦民主党掌控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 ，他们将启动针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 个人财务状况以及通俄案的调查，甚至扬言要弹劾总统。
 <br/>
 <br/>
 川普于周三上午连发推文称，民调显示多数美国选民反对所谓的穆勒（Robert Mueller）调查，并再次批评这是〝猎巫行动〞。
 <br/>

 <br/>
 川普在另一篇推文中说，如果民主党要浪费纳税人金钱，在众议院对他们展开调查，共和党将被迫考虑在参议院对民主党洩露机密资料的相关事件进行调查。
 <br/>
 <br/>
 川普还在推文中赞扬民主党领袖佩洛西（Nancy Pelosi），值得该党推选为众议院议长，并表示，如果她有困难，部份共和党议员可投票支持佩洛西。

 <br/>
 民主党众议员佩洛西可望明年1月再次出任众议院议长，她保证民主党未来将扮演制衡的角色，但也强调她会与川普合作。
 <br/>
 <br/>
 佩洛西在记者会上表示：〝如今的重点不在党派，而在必须恢复宪法对于川普政府的制衡。〞她也表示：〝民主党领导的国会将致力寻求解决方案，让大家团结，因为我们分裂得够多了。〞
 <br/>
 <br/>
 川普周三（7日）在白宫召开新闻发布会。他说，在刚刚结束的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 中，共和党扭转了历史，获得巨大胜利。川普呼吁两党摒弃党派之分，团结合作，继续为人民带来福祉，延续美国的经济奇迹。
 <br/>
