### 美国中期选举前瞻  股市规律有利共和党
------------------------

<div class="wysiwyg">
 【新唐人2018年11月03日讯】下周二，美国将迎来中期选举，总统川普在本周表示，若
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 在中期选举中获胜，将不利于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市.html" target="_blank">
  美国股市
 </a>
 。历史记录则显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 执政期间，股市表现，在一定程度上，优于民主党执政。
 <br/>
 <br/>
 路透社周五发表文章，分析了中期选举结果对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市.html" target="_blank">
  美国股市
 </a>
 可能造成的影响。
 <br/>
 <br/>
 目前，舆论多数认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 将保住参议院多数党地位，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 则有望在众议院重新成为多数党。
 <br/>
 <br/>
 而从过去50年间的历史记录来看，共和党担任总统和控制国会期间，标普500指数每年平均增加12%，而在民主党执政期间，该指数每年平均增幅为9%。而在总统和国会两院分属两党控制的情况下，股市增长仅为7%。
 <br/>
 <br/>
 分析认为，共和党倾向于为企业减少负担的减税政策，通常对刺激股市增长更加有利。在川普就任总统期间，标普500指数上涨了超过20%。
 <br/>
 <br/>
 川普本周二在推文中说，〝如果你想要自己的股票下跌，我强烈建议你投票给民主党〞。股市已经成为川普为共和党中期选举拉票的一张王牌。
 <br/>
 <br/>
 高盛也在本周发布报告指出，如果民主党在中期选举胜出，将会增加未来两年川普施政的不确定性，因而可能引发股市波动。
 <br/>
 <br/>
 新唐人记者林宇综合报导
