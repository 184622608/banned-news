### 卡舒吉案调查继续  美英宣布制裁嫌犯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】周二，土耳其继续针对沙特记者卡舒吉遇害事件进行调查。同时，美国和英国相继宣布，对涉嫌杀害卡舒吉的沙特官员进行制裁，取消他们的入境签证。
 <br/>
 <br/>
 土耳其警方，周二从一辆沙特领事馆的汽车后面，发现两个手提箱和一些其他物品。警方对这些物品进行了检查，目前尚不清楚这些物品是否属于卡舒吉。
 <br/>
 <br/>
 土耳其总统埃尔多安，周三再次强调，将把相关嫌犯绳之以法。
 <br/>
 <br/>
 土耳其总统 埃尔多安：〝我们绝不允许任何掩盖罪行之举，并将保证那些人，包括下达命令的人和命令执行者，不会逃脱正义的制裁。〞
 <br/>
 <br/>
 美国总统川普，在周二的军事将领会议上表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特王储.html" target="_blank">
  沙特王储
 </a>
 在电话中坚称他与事件无关，而是低级官员策划了这次事件。
 <br/>
 <br/>
 美国总统 川普：〝掩盖罪行很糟糕，处死很可怕。根本不应该处死或掩盖，这件事根本就不该发生。〞
 <br/>
 <br/>
 美国国务卿蓬佩奥，周二宣布已经确认，沙特王室、外交部、情报部门，以及其他部门的多名沙特官员，涉嫌杀害卡舒吉，将禁止这些人入境美国。
 <br/>
 <br/>
 美国国务卿 蓬佩奥：〝我们正在采取适当措施，包括取消他们的入境签证等。我们也在和财政部合作，检查是否可以动用全球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马格尼茨基法案.html" target="_blank">
  马格尼茨基法案
 </a>
 ，制裁这些个人。〞
 <br/>
 <br/>
 周三，英国首相特雷莎・梅也宣布，将禁止所有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉案.html" target="_blank">
  卡舒吉案
 </a>
 嫌犯进入英国。
 <br/>
 <br/>
 这一事件也在西方各国引发是否中止对沙特军售的争议。蓬佩奥表示，美国与沙特目前仍保持伙伴关系，但美国绝不会容忍沙特让卡舒吉消声的残忍手段。
 <br/>
 <br/>
 新唐人记者林宇、天羽综合报导
</div>
