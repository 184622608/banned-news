### 川普或阻所有大篷车移民入境 拒绝庇护申请
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统目前正考虑颁布一道
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政令.html" target="_blank">
  行政令
 </a>
 ，阻止所有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 入境，包括利用美国法律漏洞申请庇护的人。
 <br/>
 <br/>
 美国政治新闻网站〝政客〞（Politico）周四（10月25日）晚最先报导了这一消息。据消息人士透露，白宫正考虑发布新规，以限制某些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 申请庇护。这些规定即使不能阻挡所有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民入境，也将有效拦截大多数移民涌入美国。
 <br/>
 <br/>
 不过，据福克斯新闻（Fox News）周五报导，这只是白宫所考虑的选项之一，最终决定尚未做出。
 <br/>
 <br/>
 一名白宫官员告诉福克斯，不会向走私者或大篷车移民预测会不会采用某种确切策略。
 <br/>
 <br/>
 新的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政令.html" target="_blank">
  行政令
 </a>
 可能类似于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 之前所发布的积极阻止移民的执行令，比如旨在阻止来自以穆斯林为主的国家的公民入境美国的旅行禁令。
 <br/>
 <br/>
 目前距中期选举还有10天，阻止中美洲移民入境可能会给共和党带来巨大挑战。但是，当前正向美国进发的洪都拉斯移民高达约14,000人，而且其中多数已强行进入墨西哥境内，正向美国南部边界进军。
 <br/>
 <br/>
 一名美国官员周四透露，国防部长马蒂斯（Jim Mattis）将派800名美军士兵前往美墨边境，帮助国土安全部和国民警卫队拦截抵达边境的〝大篷车移民〞。
 <br/>
 <br/>
 国土安全部本周确认，目前企图入境美国的大篷车移民中，夹杂着黑帮成员及其他危险的犯罪分子。
 <br/>
 <br/>
 周四，美国全国边境巡逻委员会（NBPC）工会主席加德（Brandon Judd）在接受保守派网站〝每日电讯〞（The Daily Caller）采访时表示，这些大篷车移民扛着他们的国旗而来，虽然他们把这当成一种想要逃脱本国压迫的标志，但他们毕竟在扛着国旗，而且很骄傲。
 <br/>
 <br/>
 加德表示，〝你可以按照你的意愿称呼它，但是边境巡逻人员称之为入侵〞。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
</div>
