### 希拉里为有腐败嫌疑参议员拉票  出门车撞立柱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】周二晚间，美国前国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希拉里.html" target="_blank">
  希拉里
 </a>
 ・克林顿(Hillary Clinton)在前往民主党参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲍勃・梅嫩德斯.html" target="_blank">
  鲍勃・梅嫩德斯
 </a>
 筹款活动的途中发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_车祸.html" target="_blank">
  车祸
 </a>
 ，她的座驾意外撞向了停车场中的一根水泥立柱，其后希拉里只好步行到现场为梅嫩德斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉票.html" target="_blank">
  拉票
 </a>
 。
 <br/>
 <br/>
 当地时间10月16日晚间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希拉里.html" target="_blank">
  希拉里
 </a>
 前往新泽西州出席民主党参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲍勃・梅嫩德斯.html" target="_blank">
  鲍勃・梅嫩德斯
 </a>
 (Bob Menendez)的募款活动时发生了一场小型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_车祸.html" target="_blank">
  车祸
 </a>
 。随后，共和党参议院委员会在官方推特上公布了希拉里发生车祸的消息，并将其座驾撞车时的现场监控视频在网络上公开。
 <br/>
 <br/>
 从视频画面来看，事发时希拉里乘坐的特勤局面包车正驶向一个停车场，车辆在左转时突然〝咣当〞一声撞上停车场中的一根水泥柱。当时，附近并没有其他车辆和障碍物对希拉里的座驾造成干扰。
 <br/>
 <br/>
 撞车事件发生后，这辆面包车先倒退，然后向前开出一小段距离停下，希拉里才从车中走出。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-19/p9086101a103009546.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-19/p9086101a103009546-ss.jpg"/>
   <br/>
   <font size="-1">
    美国前国务卿希拉里・克林顿10月16日晚间发生车祸后从其座驾中走出来。（视频截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 据《每日邮报》报导，车祸发生后希拉里拒绝就医，并步行进入筹款活动的会场去为鲍勃・梅嫩德斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉票.html" target="_blank">
  拉票
 </a>
 。希拉里在筹款宴会上大声向现场的支持者呼吁〝我们可以相信鲍勃・梅嫩德斯〞，然后鼓励大家都去投票。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-19/p9086102a379359484.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-19/p9086102a379359484-ss.jpg"/>
   <br/>
   <font size="-1">
    2018年10月16日晚间，希拉里在新泽西州出席了民主党参议员鲍勃・梅嫩德斯(Bob Menendez)的募款活动并为其拉票。（网络图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 上述消息在网络上公开后，有不少网民在相关消息下议论纷纷。有网友调侃称：希拉里・克林顿作为美国前第一夫人和前国务卿，她乘坐的车辆应该是特勤局提供的，为她驾车的司机应该也是特勤局的保镖，估计驾车技术是过硬的，在停车场中并无其他突发状况时怎么会一头撞上了水泥柱呢？
 <br/>
 <br/>
 另据公开的资讯，民主党参议员鲍勃・梅嫩德斯2015年曾被美国司法部指控涉嫌贪污腐败。当时他被指控利用职务之便收受了总价值约100万美元的贿赂，其中包括奢侈礼物、免费乘坐私人飞机出行旅游以及超过75万美元竞选献金。梅嫩德斯则否认所有指控，最终他得以脱罪。
 <br/>
 <br/>
 这次梅嫩德斯筹款是为了参与今年11月举行的中期选举，他将面对共和党人鲍勃・哈金（Bob Hugin）和自由派候选人穆雷・萨布林（Murray Sabrin）的挑战。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：东野）
