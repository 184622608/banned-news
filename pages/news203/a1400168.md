### 雪上加霜 加州山火再遭暴风雨 失踪者身分恐成谜
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】北加坎普野火至少造成77人死亡，受灾地区迄今仍有近1000人下落不明。雪上加霜的是，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雨.html" target="_blank">
  暴风雨
 </a>
 预计将于本周侵袭重灾区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天堂镇.html" target="_blank">
  天堂镇
 </a>
 ，雨水可能冲走死者的骨骸碎片，让搜寻人员更难辨认死者身分。
 <br/>
 <br/>
 坎普野火（Camp Fire）肆虐山城
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天堂镇.html" target="_blank">
  天堂镇
 </a>
 已经13天，预计21～22日将迎来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴风雨.html" target="_blank">
  暴风雨
 </a>
 。这对灭火虽有帮助，但因植被被烧毁，专家担忧会发生洪水和土石流，使接获疏散命令的4.6万人更加痛苦。
 <br/>
 <br/>
 而且，辨识火灾死者身分本来就已经很难，如果暴雨再冲走遇难者的遗骸，灰烬融入泥水，让搜救人员更加难以辨认死者身份。
 <br/>
 <br/>
 巴特县(Butte County)警长哈尼亚（Kory Honea）沮丧的说，在暴风雨来临前完成遇难者遗体确认工作几乎是不可能的。
 <br/>
 <br/>
 目前由400多名消防员和志愿者组成的搜救队伍，正在天堂镇搜寻被烧毁房屋中的遗骸。每队搜寻小队带着一只颈圈上系有铃铛的寻尸犬，在火场中搜寻遇难者遗骸。如果他们什么也没有发现，就会在房子附近用橘色漆划个圈。
 <br/>
 <br/>
 50岁志工帕纳克(Robert Panak)告诉美联社，〝我只是往好的方面想，希望带给受害家属一点慰藉和结束。〞
 <br/>
 <br/>
 大火过后，现场足以测试DNA的遗骸不多，搜救人员目前正通过牙齿、骨头碎片等核查死者的DNA，从而确认其身份。哈尼亚表示，这项工作的难度已经非常大，再加上暴雨，可能永远无法知道确切的死亡人数。
 <br/>
 <br/>
 来自天堂镇的萨迪亚・昆特（Sadia Quint）说，她的叔叔可能在大火中遇难了，但没有人能确定这件事情。她说，〝我们目前只知道，他的房子和汽车都被烧毁了。我们还在等警长办公室能带来确切的消息，告诉我们是否能找到他的遗体。〞
 <br/>
 <br/>
 另外，南加洛杉矶郊外，伍尔西大火（Woolsey Fire）夺去至少3条性命。已有82%的火势得到了控制，大火燃烧了9万8362英亩的土地。
 <br/>
 <br/>
 截至19日，加州南部林火造成341栋建筑部分损毁、1500栋建筑全毁，灾损评估已完成95%。灾区电力逐渐恢复，最晚复电的区域包括布洛德海滩(Broad Beach)以及马里布(Malibu)海滨的百万豪宅区。
 <br/>
 <br/>
 预计当地本周也有大的降雨，当局在南加州文图拉县、洛杉矶县等〝伍尔西大火〞燃烧区域发布了土石流预警，提醒土石流可能形成新的灾难。
