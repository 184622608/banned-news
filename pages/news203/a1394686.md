### 卡瓦诺宣誓就职 川普代表全国人民道歉
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院大法官.html" target="_blank">
  最高法院大法官
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 ，周二正式上班。在周一晚的宣示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就职典礼.html" target="_blank">
  就职典礼
 </a>
 上，川普（特朗普）总统对民主党极力阻挠给卡瓦诺一家带来的困扰，代表全美人民向卡瓦诺及其家人道歉。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 手抚《圣经》，庄严宣誓。
 <br/>
 <br/>
 美国最高法院退休大法官 肯尼迪: 〝作为美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最高法院大法官.html" target="_blank">
  最高法院大法官
 </a>
 。〞
 <br/>
 <br/>
 美国最高法院大法官 卡瓦诺:〝作为美国最高法院大法官。〞
 <br/>
 <br/>
 美国最高法院退休大法官 肯尼迪: 〝遵守美国宪法和法律。〞
 <br/>
 <br/>
 美国最高法院大法官 卡瓦诺:〝遵守美国宪法和法律。〞
 <br/>
 <br/>
 美国最高法院退休大法官 肯尼迪: 〝所以， 请上帝帮助我。〞
 <br/>
 <br/>
 美国最高法院大法官 卡瓦诺:〝所以， 请上帝帮助我。〞
 <br/>
 <br/>
 美国最高法院退休大法官 肯尼迪：〝祝贺你。〞
 <br/>
 <br/>
 他感谢他家人和美国人民的支持，表示将严守美国宪法，确保法律面前人人平等。
 <br/>
 <br/>
 美国最高法院大法官 卡瓦诺：〝美国宪法和法律保护所有人，所有信仰和所有背景的人，我向最高法院的每个当事人保证，以尊重和开放的心态听取他们的论证。我向所有美国人保证，做一个独立、公正的法官，致力于〝法律面前，人人平等〞。〞
 <br/>
 <br/>
 川普总统向卡瓦诺表示祝贺，再次批评民主党违反了〝罪名未经证实之前，任何人都是无辜的〞法律原则，并为卡瓦诺一家因此受到的伤害致歉。
 <br/>
 <br/>
 美国总统 川普：〝我代表国家向卡瓦诺及其家人道歉，他们被迫忍受巨大痛苦和折磨。站出来为国效力的人，理应得到公正、有尊严的评价，而不是基于谎言和骗局、诋毁个人名誉的政治运动。〞
 <br/>
 <br/>
 周二一早，川普发推文说：〝昨晚真是美好的一晚，在白宫表彰卡瓦诺和他的家人，我们国家为他们感到非常自豪。〞
 <br/>
 <br/>
 周二，卡瓦诺来到美国最高法院，正式开始工作。
 <br/>
 <br/>
