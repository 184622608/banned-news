### 加州野火59死130人失踪 火势仅控制约35％
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】美国加州史上灾情最惨重的大火已经肆虐一周，截至周四（15日）死亡人数已上升到59人，约130人失踪，9000多户住宅被烧毁，5.2万人被疏散。
 <br/>
 <br/>
 周四，气温下降给灭火提供了便利条件，消防人员已成功控制约35％区域的火势，并减缓了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坎普野火.html" target="_blank">
  坎普野火
 </a>
 〞（Camp Fire）大火蔓延人口稠密的奥罗维尔（Oroville）的势头。该区在已遭焚毁的天堂镇（Paradise）南方。当局已投入超过5600名警消救灾。
 <br/>
 <br/>
 截至到当地时间周四，至少59人因这次毁灭性野火丧命。当局公布的名单显示，仍有130人下落不明。救援小组周四带着寻尸犬在天堂镇挨家挨户寻找失踪人员。
 <br/>
 <br/>
 大部分失踪者是这次火灾的重灾区北加州巴特县（Butte County）天堂镇的居民。
 <br/>
 <br/>
 巴特县警长哈尼亚（Kory Honea）周四晚上告诉记者，他们出动461名搜救人员和22只寻尸犬参与寻找失踪居民的行动。同时当局也正在加快DNA鉴定，以确认罹难者身分。
 <br/>
 <br/>
 哈尼亚说，如果具名确定家人已经罹难可以尽快提供DNA样本给警长办公室。
 <br/>
 <br/>
 天堂镇位于内华达山（Sierra Nevada）山麓，在加州首府沙加缅度（Sacramento）北方约130公里处。小镇约有2万6000居民，当地受到许多退休人士以及好莱坞明星的喜爱，警长办公室通报的失踪者有许多是70～90岁的老人。
 <br/>
 <br/>
 上周爆发的〝坎普野火〞让天堂镇几乎从地图上消失。强风助长火势迅速蔓延，天堂镇几乎每户民宅都被摧毁。
 <br/>
 <br/>
 〝坎普野火〞已经焚烧逾13万5000英亩（约540平方公里）的土地，造成超过7600栋建筑物遭焚毁。

 据悉，除了在第一线的救火人员，当局还派遣60支调查队伍进入天堂镇、康考镇（Concow）等周边范围进行灾后调查及搜索遗体。
 <br/>
 <br/>
 加州森林防火厅（Cal Fire）救灾指挥官乔许．毕沙夫（Josh Bischof）表示，〝目前消防人员与部属的推土机，可以确保火势不会进入奥罗维尔。但其后方已经焚毁的地区，温度仍然很高，还是十分危险。〞
 <br/>
 <br/>
 另外，对于〝坎普野火〞起火原因，日前传出疑似人为疏失造成，加州〝太平洋瓦电公司（PG＆E）〞的监督报告显示，在11月8日早上6点15分，在巴特县有条输电线发生故障，15分钟后接到第一起野火警报。
 <br/>
 <br/>
 对此PG＆E表示，相关的起火原因仍有待调查结果出炉，但加州部分受害者已经联合向PG＆E提出控诉。如果最后调查结果要PG＆E对火灾负责，预料赔偿金将高达68亿美金，公司可能会因此倒闭。受此影响，PG＆E的股价已下跌近50％。
