### FBI结束卡瓦诺调查 参议院周内表决
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】美国联邦调查局结束了对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官提名.html" target="_blank">
  大法官提名
 </a>
 人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 性侵指控的调查，周三向国会提交报告。国会参议院多数党领袖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麦康奈尔.html" target="_blank">
  麦康奈尔
 </a>
 表示，周四，参议院议员将阅读调查报告，周五对卡瓦诺的提名进行投票表决。
 <br/>
 <br/>
 美国参议院多数党领袖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麦康奈尔.html" target="_blank">
  麦康奈尔
 </a>
 ：〝这是第七次，联邦调查局对法官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 进行的第七次背景调查。这是我们国家历史上任何一个最高法院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官提名.html" target="_blank">
  大法官提名
 </a>
 人，都没有经历过的最详尽、最令人疲倦的调查之后，再追加调查的结果。〞
 <br/>
 <br/>
 参议院多数党领袖麦康奈尔提议，参议员利用周四一天阅读调查报告，晚间结束讨论。
 <br/>
 <br/>
 美国国会参议院多数党领袖麦康奈尔：〝所以我提议对卡瓦诺的提名讨论今晚结束，进入下一步程序，我本周稍早说过。〞
 <br/>
 <br/>
 周五，参议院将对麦康奈尔的结束讨论动议进行投票。如果通过，最早可在周六，参议院对卡瓦诺大法官提名投票表决。
 <br/>
 <br/>
 周四一早，川普总统发推文表示，大法官卡瓦诺提名受到民主党的极力阻挠，但是美国选民看的最清楚。
 <br/>
 <br/>
 推文说：〝卡瓦诺法官受到苛刻、不公正的对待，这在选民中产生难以置信的积极影响。人民比政客们看得更清楚。最重要的是，（卡瓦诺）这位杰出人士不能被卑鄙可耻的民主党，以及未经证实的指控诋毁。〞
 <br/>
 <br/>
 川普还赞叹美国人民对卡瓦诺的支持和信任。
 <br/>
 <br/>
 他说：〝对卡瓦诺法官有这么高的热情和能量。看看这能量，看看这些民调。正在发生一件大事。他才华出众的优秀人才。全国人民都支持他。〞
 <br/>
 <br/>
 新唐人记者安心、李静明综合报导
</div>
