### 美大豆船改运中国邻国疑转口 中共哑巴吃黄连？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】路透社日前报导，近期有多艘原定从美国开往中国的大豆船，却改道他国，认为这是原买家将大豆转卖他人。而此前有猜测称，中国大豆缺口巨大，或通过第三国转运美国大豆，以逃避中共加征的关税。
 <br/>
 <br/>
 路透社周四（11月1日）引述路孚特(Refinitiv)Eikon上的船运数据称，9月末在西雅图装载3.6万吨美国大豆的Star Laura货船，原定在本周三抵达中国青岛，却在周四凌晨将目的地改成韩国丽水(Yeosu)。
 <br/>
 <br/>
 报导指，过去一个月里，至少还有两艘原本开往中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆船.html" target="_blank">
  美国大豆船
 </a>
 改变了航向，驶向他国。
 <br/>
 <br/>
 其中，10月21日在西雅图装载近7万吨美国大豆的Audacity货船，周日将其目的地从青岛改为越南Phu My；9月末在路易斯安娜装货的Golden Empress，也在10月23日把目的地从青岛改为新加坡。
 <br/>
 <br/>
 报导说，这说明原买家已将大豆转售给了其他市场。
 <br/>
 <br/>
 值得注意的是，这三艘大豆船的原目的地都是青岛。而青岛是中国猪饲料的主要产地之一。
 <br/>
 <br/>
 10月初，中国饲料工业协会公开〝征求意见〞，计划降低猪饲料中粗蛋白质的最低含量。10月26日，官方正式公布了降低蛋白质含量后的新标准。大豆是饲料中蛋白质的主要来源。中共官媒此前报导，通过降低饲料中的蛋白质含量，每年可降低大豆需求1千万吨左右。
 <br/>
 <br/>
 此外，还有猜测称，中国企业或从其他国家转运美国大豆，以填补中国市场的巨大缺口。
 <br/>
 <br/>
 今年8月，中共农业农村部副部长韩俊对官媒表示，目前国内大豆产需缺口9千多万吨。多家外媒报导，中共或将不得不再次进口美国大豆，以满足国内需求。
 <br/>
 <br/>
 今年7月6日，中共开始对美国大豆加征25%的关税。因为中国大豆严重依赖进口，此前大豆进口关税很低。7月初，中共还将来自韩国、印度、孟加拉、老挝和斯里兰卡等国大豆的进口关税，从3%下调至零。
 <br/>
 <br/>
 有专家指，如果有必要，中共可能允许企业通过第三国转运进口美国大豆，以规避加征的关税。这意味着，美国大豆出口仍然不愁买家，但中国进口商却不得不忍痛承受比过去更高的价格。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
