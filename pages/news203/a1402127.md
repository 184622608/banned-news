### 告别老布什 国家大教堂举行追悼仪式
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】周三，美国前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 的国葬仪式在华盛顿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家大教堂.html" target="_blank">
  国家大教堂
 </a>
 举行，老布什的家人，美国现任和前任总统，以及世界各地知名人士前来悼念。
 <br/>
 <br/>
 八名军人缓步将覆盖着美国国旗的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老布什.html" target="_blank">
  老布什
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灵柩.html" target="_blank">
  灵柩
 </a>
 从国会大厦抬出。军乐队奏起了乐曲〝向统帅致敬〞。
 <br/>
 <br/>
 礼炮声响起，一层层烟幕让场面更显肃穆。
 <br/>
 <br/>
 老布什的五名子女，目送灵柩被抬上汽车。这其中有他的长子，前总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小布什.html" target="_blank">
  小布什
 </a>
 和夫人劳拉。
 <br/>
 <br/>
 周一晚开始，老布什的灵柩一直安放在国会圆顶大厅，除了政界人士，也有不少普通民众来向他表达最后的敬意。
 <br/>
 <br/>
 现在，老布什的灵柩被护送到华盛顿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家大教堂.html" target="_blank">
  国家大教堂
 </a>
 。
 <br/>
 <br/>
 老布什不仅是第41届美国总统，也担任过两届副总统，前中情局局长，和国会众议员。他是第43届总统小布什的父亲。
 <br/>
 <br/>
 老布什的灵柩送抵大教堂后，由军人抬入室内。
 <br/>
 <br/>
 现场等候的，还有德国总理默克尔和约旦国王及王妃等外宾。
 <br/>
 <br/>
 美国现任总统川普和前总统卡特，克林顿，奥巴马都参加了悼念仪式。
 <br/>
 <br/>
 川普和第一夫人梅拉尼娅来到座位后，与身边的奥巴马夫妇分别握手。
 <br/>
 <br/>
 小布什总统则与每一位总统及总统夫人一一致意。
 <br/>
 <br/>
 随后，小布什在悼念仪式上发表演说，缅怀父亲。
 <br/>
 <br/>
 美国前总统 小布什：他向我展示了，一名用诚信来服务国家，用勇气来领导民众，用心中的爱来行事的总统是什么样子。
 <br/>
 <br/>
 悼词最后，小布什流下泪水，向父亲告别。
 <br/>
 <br/>
 美国前总统 小布什：你是一个伟大，高尚的人，是儿子、女儿所能拥有的最好的父亲。在我们悲痛之余，我们可以笑着相信你已经可以重新拥抱妹妹、握着妈妈的手了。
 <br/>
 <br/>
 老布什的夫人芭芭拉・布什（Barbara Bush）今年4月才刚去世。小女儿罗宾1953年死于白血病，年仅三岁。
 <br/>
 <br/>
 悼念仪式后，老布什在华府的最后一次行程结束。他的灵柩被送回德州，周四将在当地举办最后一场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_追悼仪式.html" target="_blank">
  追悼仪式
 </a>
 ，随后将长眠在德州卡城的布什总统图书馆。他的妻子和小女儿罗宾也安葬在此。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
