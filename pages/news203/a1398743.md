### 美官员警告中共违反骇客协定：制裁时刻并未过去
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】美国国家安全局高官日前对外表示，在过去的一年间，来自中国甚至来自中共政府的对美网络
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_骇客攻击.html" target="_blank">
  骇客攻击
 </a>
 再度增多，已违反了中美之间达成的网络安全协定，美国政府不排除就此对中方进行制裁。
 <br/>
 <br/>
 据《路透社》报导，美国国家安全局官员乔伊斯（Rob Joyce）11月8日在三藩市出席Aspen网络峰会时表示，在过去一年里，美国政府发现来自中国、甚至是来自北京政府的窃取美国商业机密资料的网络间谍活动再度增多，中国骇客甚至曾经试图攻入美国能源网络。
 <br/>
 <br/>
 乔伊斯表示，这显然超出了三年前美中之间达成的网络安全协议规定的界限。他说：〝我们确实看到他们在去年的行为造成侵害，我们非常关切那些令人不安的趋势〞。
 <br/>
 <br/>
 〝我们在出现网络非法行为的背景下动用过制裁〞乔伊斯警告说，〝制裁的时刻并没有过去。〞
 <br/>
 <br/>
 2015年，时任美国总统的奥巴马与中国国家主席习近平曾经签署过一份网络安全协议，两国政府当时相互承诺，不会蓄意支持窃取企业机密与商业资讯的骇客行动。此后，中方通过网络攻击盗窃美国企业商业机密的活动曾大幅减少，但现在这种
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络盗窃.html" target="_blank">
  网络盗窃
 </a>
 活动又再次变得猖獗。
 <br/>
 <br/>
 《华尔街日报》曾报导称，据美国政府的某些统计，中共政府通过发动网络攻击获招募一线人员对美进行的窃取机密资料的行动，让美国每年的经济损失高达数千亿美元。
 <br/>
 <br/>
 近期，美国政府明显加大了打击中共窃取美商业资料与技术的力度。
 <br/>
 <br/>
 就在11月30日，美国司法部刚刚起诉了包括中共情报人员和骇客在内的10名中国籍人士，指控他们涉嫌在2010年到2015年间，共谋入侵电脑窃取美国与法国航太商业飞机〝涡轮发动机〞的科技资料。
 <br/>
 <br/>
 这是美国从今年9月份以来，第3度指控中共情报员窃取商业机密。
 <br/>
 <br/>
 上周，时任美国司法部长的塞申斯（Jeff Sessions）也曾表示，中国（中共政府）没有遵守与美方签署过的协议，为此美国方面将拟定〝中国计划〞（China Initiative），专门打击中共政府针对美国的经济贸易间谍活动。
 <br/>
 <br/>
 虽然塞申斯现在已经辞职，但美司法部的国家安全部门负责人John Demers本周四还对外表示，上述〝中国专项行动〞不会受塞申斯辞职的影响。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
