### 美中期选举后 对华政策继续强硬
------------------------

  美国中期选举
 </a>
 落幕。专家认为，虽然民主党掌控众议院，但会继续支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）的强硬
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对华政策.html" target="_blank">
  对华政策
 </a>
 。白宫贸易顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 周五也再次批评中共毫无诚信。
 <br/>
 <br/>
 对于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 ，中共一直高度关注，希望选举结果削弱总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 地位，减轻美国对华
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 力度。不过，华府专家认为，虽然民主党重新掌控众议院，但他们会继续支持川普的强硬
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对华政策.html" target="_blank">
  对华政策
 </a>
 。
 <br/>
 <br/>
 战略与国际研究中心中国项目主任张克斯：〝人们在思考中国问题时已有根本性的改变。所以民主党在很多时候，听起来和他们在政府对手很相像。〞
 <br/>
 <br/>
 周五，美中第二轮外交与安全对话在华府举行。同一时间，白宫贸易与制造业政策办公室主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 表示，美中之间的最大问题就是中共没有诚信。过去多年来，中共从不遵守和美国达成的协议，即使双方达成贸易协议，也很难相信中共会认真执行。
 <br/>
 <br/>
 白宫贸易与制造业政策办公室主任Peter Navarro：〝每个世界上我们能信任的国家，我们都能达成协议，但是中共却不同。他们想让我们来到谈判桌前，听起来很有道理，用他们的方式讲话，而他们的行为却依然如故。〞
 <br/>
 <br/>
 11日，美国副总统彭斯将启程，前往新加坡出席美国-东盟峰会及东亚峰会，并参加在巴布亚新几内亚举行的亚太经合首脑会议。白宫特别提到他将传达的信息，即〝美国不会容忍印太地区任何国家的威权主义、侵略和对他国主权的漠视。〞这被认为是直指中共。
 <br/>
 <br/>
 战略与国际研究中心高级副总裁Matthew Goodman：〝我预计副总统彭斯会强调，遵守国际规则和行为规范，尤其是在安全事务上，特别是海上安全。〞
 <br/>
 <br/>
 战略与国际研究中心中国项目主任张克斯：〝(美中)战略关系也在严重恶化。白宫声明可能会进一步推动更大的战略不稳定性。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
