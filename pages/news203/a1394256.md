### 梅拉尼娅访肯尼亚 斑马小象也来〝迎接〞
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】周五，美国第一夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼娅.html" target="_blank">
  梅拉尼娅
 </a>
 来到非洲之行的第三站
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肯尼亚.html" target="_blank">
  肯尼亚
 </a>
 ,紧凑但轻松的行程，与华盛顿剑拔弩张的氛围形成鲜明对比。
 <br/>
 <br/>
 周五，美国第一夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梅拉尼娅.html" target="_blank">
  梅拉尼娅
 </a>
 来到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肯尼亚.html" target="_blank">
  肯尼亚
 </a>
 ，展开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲之旅.html" target="_blank">
  非洲之旅
 </a>
 的第三站。
 <br/>
 <br/>
 在负责救援动物孤儿的大卫-薛德里克野生动物信托所,梅拉尼娅面带微笑地喂一头小象喝配方奶。
 <br/>
 <br/>
 之后,梅拉尼娅来到奈洛比国家公园，用望远镜观看斑马，长颈鹿，黑斑羚，犀牛和河马等非洲动物。
 <br/>
 <br/>
 2016年，肯尼亚政府曾在这里焚烧了重105吨的走私象牙，以及重1.35吨的走私犀牛角，以打击非法狩猎和走私。
 <br/>
 <br/>
 稍后,梅拉尼娅探视了当地一家孤儿院。
 <br/>
 <br/>
 现场，还有两个孩子一边跳舞，一边把美国第一夫人带到座位上。梅拉尼娅还亲自给孩子们读了一则故事。
 <br/>
 <br/>
 美国第一夫人 梅拉尼娅：〝狮子和老鼠。你们听说过这本书吗?〞
 <br/>
 <br/>
 梅拉尼娅此行还去了肯尼亚国家剧院，与孩子们一起欣赏了戏剧表演。
 <br/>
 <br/>
 梅拉尼娅非洲行是她首次单独出访，主要为了推广儿童福利计划。肯尼亚之前，她已经访问了加纳和马拉为。之后她还将拜访埃及。预计本周末返回美国。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
