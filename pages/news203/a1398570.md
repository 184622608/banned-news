### 川普声明：纪念共产主义受害者
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】周三是美国〝全国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产主义受害者.html" target="_blank">
  共产主义受害者
 </a>
 纪念日〞。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫网站.html" target="_blank">
  白宫网站
 </a>
 发表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统的声明，铭记一亿多人被〝共产主义极权〞残杀和迫害，重申将继续推动自由理想并为所有人争取机会。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 声明说，〝1917年俄罗斯十月革命以来，我们目睹了共产主义暴政意识形态所造成的暴虐、压迫和死亡。
 <br/>
 <br/>
 〝共产主义把人类与生俱来的人权置于其声称的全体幸福之下，结果是消灭宗教自由，剥夺私有财产、言论自由，甚至经常夺取生命。〞
 <br/>
 <br/>
 声明认为，共产主义的暴行证明，〝共产主义和对共产主义的追求，终将破坏人类精神和福祉。〞
 <br/>
 <br/>
 去年，在俄罗斯爆发〝十月革命〞，建立世界第一个共产党政权百年之际，川普政府宣布11月7日为〝全国纪念
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产主义受害者.html" target="_blank">
  共产主义受害者
 </a>
 日〞。
 <br/>
 <br/>
 另据《大纪元时报》报导，目前，中国共产党成为世界最大的共产政权，其建政以来，有六到八千万无辜生命被杀害。
 <br/>
 <br/>
 川普声明说，〝我们一起哀悼，众多民众蒙受了共产主义所造成的无法承受的损失，我们再次承诺，将继续推动自由理想并为所有人争取机会。〞
 <br/>
 <br/>
 新唐人记者李梅、张琪综合报导
</div>
