### 佛州拒付退休金 舞弊选举官收回辞呈
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 计票引发了巨大争议，上周，即将离任的州长斯科特决定撤换涉嫌计票欺诈的布劳沃德县选举监督员斯奈普斯，使佛州选举再次成为关注焦点。
 <br/>
 <br/>
 上周五（11月30日）晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 州长斯科特（Rick Scott）下令，立即停止声名狼藉的布劳沃德县（Broward）选举监督员斯奈普斯（Brenda Snipes）的工作，并指控她在选举工作中滥用职权、不称职和疏忽职守。
 <br/>
 <br/>
 虽然斯奈普斯已递交辞呈、决定明年1月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退休.html" target="_blank">
  退休
 </a>
 ，但斯科特在声明中表示，美国纳税人不该必须为已宣布辞职的官员支付退休金。
 <br/>
 <br/>
 斯科特决定，由自己的前法律总顾问、现佛罗里达企业公司（Enterprise Florida, Inc.）主席兼CEO安东纳奇（Peter Antonacci）取代斯奈普斯的职位，而这一决定将会威胁到斯奈普斯能否顺利退休并拿到每月上万美元的退休金。
 <br/>
 <br/>
 对于用美国纳税人的钱供养一名屡次破坏选举法并遭到指控的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选举官.html" target="_blank">
  选举官
 </a>
 员，很多民众表示不满。
 <br/>
 <br/>
 周六，斯奈普斯做出回应。她宣布收回辞呈，她的律师还在当天举行的新闻发布会上表示，他们将为此〝战斗到最后〞。
 <br/>
 <br/>
 斯奈普斯从2003年起即担任布县首席选举官，但早在今年11月选举前，就因涉嫌选举舞弊而遭到过14次指控。
 <br/>
 <br/>
 上个月的选举夜，倾向于民主党的布县共收到了634,000 张选票；但两天后，迟迟不完成计票工作的布县宣布，选票已激增到712,840张，且多出的选票基本上都是投给民主党候选人。
 <br/>
 <br/>
 由于布县和棕榈滩县（Palm Beach）都出现突然冒出大量选票且流向民主党候选人的问题，导致佛州联邦参议员和州长两项重要选举最终因竞选对手间得票率低于幅差范围而重新计票。
 <br/>
 <br/>
 斯奈普斯屡次违反选举法肆意行事、企图影响选举结果的行径，不仅一度引发了人们对佛州本次选举结果的担忧，而且还使人们忧虑，此类情况是否会在2020年总统选举中重演、从而影响美国大选。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
</div>
