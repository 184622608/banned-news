### 艾维乐胜选对保SHSAT至关重要 纽约华裔选民投票热
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】中期选选举6日进行中，一直坚定捍卫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特殊高中入学考试.html" target="_blank">
  特殊高中入学考试
 </a>
 (
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_SHSAT.html" target="_blank">
  SHSAT
 </a>
 )的艾维乐能否连任纽约州参议员，对是否能保住SHSAT至关重要。众多布鲁克林的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华裔选民.html" target="_blank">
  华裔选民
 </a>
 纷纷出来投票。他们称，取消SHSAT课题，是促使他们出来投票的重要原因。
 <br/>
 <br/>
 《世界日报》报导，成为美国公民已有5年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华裔选民.html" target="_blank">
  华裔选民
 </a>
 张Joanne说，这次是她第二次投票，因为刚成为选民时并没有对美国政治或政策多加关注，直到近两年孩子上学后，她才发现很多政策与教育密切相关。她提及华裔最为关注的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_SHSAT.html" target="_blank">
  SHSAT
 </a>
 议题时表示〝非常不赞同改革〞，同时她也希望更多华裔能站出来投票，用自己的公民权捍卫自己的权益。
 <br/>
 <br/>
 华裔选民甄思宗认为，投票没有党派之分，民众选择能为自己谋利的民选官员才是上策。甄思宗认为，时下对纽约华人最重要的议题就是SHSAT，自己的孙子也即将面对考试。他希望更多华裔选民能为教育着想，用投票的方式来表达自己的意见。他表示，投票是行使公民权利的重要方式，也是维护家庭利益的途径。
 <br/>
 <br/>
 仍在考取成为美国公民的华裔民众卢彪，一直积极让妻子注册选民。6日他陪同妻子一起去投票，他说，〝如果没人肯投票，就无法作出改变〞。
 <br/>
 <br/>
 卢彪也表示，纽约众多政策对华人不利，其中SHSAT课题对华裔学生非常不公平，如果真的改革成定案，日后将更难以反对。他说，虽然自己的孩子已经长大，不需要再面对这项考试，但对后代依旧会促成极大的影响。因此，他也不断鼓励身边亲友出来投票，发表自己的心声。
 <br/>
 <br/>
 《大纪元》报导，多年来推动华人投票的社区人士罗维宗表示，明年市长白思豪改革
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特殊高中入学考试.html" target="_blank">
  特殊高中入学考试
 </a>
 SHSAT的提案将在州议会投票，州参议会是挡住该提案的重要一环。所以，一直坚定捍卫SHSAT的艾维乐（Tony Avella）能否连任州参议院，对是否能保住SHSAT至关重要。
 <br/>
 <br/>
 罗维宗说，挑战艾维乐的刘醇逸是民主党极左派，主张大麻合法化，无视大麻对社会及青少年的危害；另外，第11选区的地税已经很高，还在不断上涨，而刘醇逸还主张地税上涨无上限，全然不考虑民众的负担。这些都是左派的〝共产主义〞思想，只想千方百计搜刮民脂民膏，不考虑民众利益。
 <br/>
 <br/>
 罗维宗还强调，在刘醇逸当年竞选市长时，为了拉拢非裔、西语裔的选票，提出改革SHSAT。在本次竞选中，刘醇逸不敢得罪非裔、西语裔，始终不开口承诺支持SHSAT的单一考试，没有参加过任何一次抗议废除SHSAT的集会及游行。如果刘醇逸当选，大家可以想像后果。当前州参议会共和党和民主党的比例为32比31，也就是民主党的参议员一个都不能再多，这是保卫SHSAT、反对大麻合法化的最后防线。
 <br/>
 <br/>
 据悉，为了孩子和社区的未来，不少第11选区的华人妈妈们，近期不仅积极当义工为艾维乐助选，而且以前从不投票的她们，在本次普选日将会第一次出来投票。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
</div>
