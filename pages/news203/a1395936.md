### 美中期选举临近  专家忧中共政治渗透
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】随着美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 临近，如何防止外国势力干预
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国民主.html" target="_blank">
  美国民主
 </a>
 程序成为关注的焦点。专家认为，中共对美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治干预.html" target="_blank">
  政治干预
 </a>
 和渗透方式，与俄罗斯完全不同，对美国构成的威胁更大。
 <br/>
 <br/>
 不到三周，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 就将开始。本月初，副总统彭斯的美中关系演讲，引发各界对于中共干预
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国民主.html" target="_blank">
  美国民主
 </a>
 制度的关切。
 <br/>
 <br/>
 澳洲教授《无声入侵》作者 Clive Hamilton：〝副总统彭斯的演讲非常重要，因为他让人们突然意识到，可能我们都被俄罗斯问题分散了注意力。〞
 <br/>
 <br/>
 专家提醒，不应把俄罗斯干预美国2016年大选，和中共对美国的影响力渗透同等看待。俄罗斯只是寻求短期影响力，并希望引起外界瞩目；而中共一直努力掩盖其政治影响力，同时以腐败、恐吓等手段，实现其长期政治目的。
 <br/>
 <br/>
 澳洲教授《无声入侵》作者 Clive Hamilton：〝北京是以长期影响力为战略目标。它不是以特别党派为目标，不在乎谁当政。它想影响所有的政治势力。这也是为何北京对美国民主的威胁要远超莫斯科。〞
 <br/>
 <br/>
 近年来，中共对澳洲、新西兰及加拿大的干预渗透已引起国际广泛关注。专家指出，中共对美国的渗透同样非常严重，而美国社会对此普遍缺乏认识，对中共的揭露还处于初始阶段。不过，〝五眼联盟〞国家情报机构已开始加强这方面合作。
 <br/>
 <br/>
 战略与国际研究中心高级副总裁 James A. Lewis：〝‘五眼联盟’已把这个放在工作计划中，因为对他们来说是很主要的安全议题。这是中共更广泛情报活动的一部份，给西方制造了一个新的威胁领域。〞
 <br/>
 <br/>
 针对即将到来的美国中期选举，专家认为，外国影响力操作已成为最关切议题，超过对选举系统安全的担忧。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
