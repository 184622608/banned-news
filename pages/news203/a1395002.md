### 川普：未经美允许 韩国不会放松制裁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 政府是否会放松对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 制裁的质疑，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周三（10日）表示，韩国不会在未经美国准许的情况下，放松对朝鲜的制裁。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周三在白宫被问及有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 正在考虑取消部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对朝制裁.html" target="_blank">
  对朝制裁
 </a>
 时，他说：〝未经我们批准，他们不会这样做。未经我们批准，他们什么都不会做。〞
 <br/>
 <br/>
 他鼓励美国盟国继续对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 实施制裁，直到确认朝鲜实现无核化。川普政府此前也一直强调，只有朝鲜实践无核化，才能缓解对其制裁。
 <br/>
 <br/>
 韩国外交部长官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康京和.html" target="_blank">
  康京和
 </a>
 （Kyung-wha Kang）周三在接受国会对外交部的国政监查时表示，外交部正在同有关部门讨论是否需解除〝5・24对朝制裁〞。
 <br/>
 <br/>
 〝5・24对朝制裁〞是韩国政府在2010年〝天安舰事件〞发生后推出的单边制裁政策。当时，天安舰遭朝鲜潜艇发射的鱼雷击中，造成45位官兵死亡，韩国因此对朝鲜实施单边制裁。制裁内容为除了开城工业园区以外，禁止韩方人员访朝，禁止朝鲜船舶进入韩方海域，暂停韩朝两国间贸易往来，以及禁止对朝进行新投资等。
 <br/>
 <br/>
 康京和的回答被认为韩国正考虑自行放松针对朝鲜的制裁，以鼓励朝鲜去核化。不过，这番话随后招致一些保守派议员的批评。
 <br/>
 <br/>
 在后来的答辩中，虽然康京和改口称，并不是政府层面的正式讨论，并表示朝鲜应先对那次攻击行动道歉，最后，康京和承认失言，〝对于我的发言引起误会一事，再次深表歉意。〞但是仍然引发争议。
 <br/>
 <br/>
 韩联社周四（11日）引述韩国政府有关人士的话称，康京和的发言有可能引发误会，但韩美之间的磋商不会遇阻。对朝制裁需要与美方配合，履行军事协议需要与联合国军司令部方面的合作，韩国政府很清楚这些情况，在与美国进行沟通和协调方面不会有一丝怠慢。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：东野）
</div>
