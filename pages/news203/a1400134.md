### 华为中兴芯片危机？美〝技术锁喉〞剑指制造2025
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】日前，美国公布一份迄今最严格的技术出口管制先期通知，涵盖几乎所有科技前沿领域。美媒指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴.html" target="_blank">
  中兴
 </a>
 等中国高科技公司将面临又一次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芯片危机.html" target="_blank">
  芯片危机
 </a>
 。港媒指，美方的〝技术锁喉〞直指〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞，将正式启动中美科技战。
 <br/>
 <br/>
 11月19日，美国商务部工业安全署出台通知，宣布将对人工智能和机器学习技术、定位导航、数据分析、机器人、高超音速空气动力学、先进材料、脸部和声纹等先进监控技术等14个高科技前沿领域实施出口管制。
 <br/>
 <br/>
 目前该措施还在征询公众意见期，12月19日之后，美国政府将根据收集到的意见评估和更新管制清单，并正式公布实施。根据今年刚出台的《出口管制改革法案》，出口管制施行后，所有相关商品和技术的出口，都要经过商务部的审核批准。
 <br/>
 <br/>
 《美国之音》报导指，该措施对中国企业的影响最大，特别是中国高科技公司赖以生存的美国芯片，将面临新的封锁。许多有政府背景的中国科技巨头，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴.html" target="_blank">
  中兴
 </a>
 和海康威视等，都可能再次面临
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芯片危机.html" target="_blank">
  芯片危机
 </a>
 。
 <br/>
 <br/>
 中国所有的高科技产业均高度依赖美国技术和零部件，特别是芯片。一旦美方下达禁售令，则意味着对相关企业宣判〝死刑〞。近日有消息指海康威视将遭到美国出口管制，其股价立即从今年最高点暴跌超过30%。
 <br/>
 <br/>
 港媒分析称，该管制措施明显直指〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞计划，将遏制中国高科技产业的发展。
 <br/>
 <br/>
 按照中共官方说法，〝中国制造2025〞旨在发展〝制造业强国〞。但外界认为，该计划真正意图是取得尖端科技领域的霸权，配合中共政治扩张和渗透。该计划重点支持的产业也多集中在高科技领域。
 <br/>
 <br/>
 港媒还称，美方的出口管制一旦实施，意味着正式启动针对中共的科技战。而中共在科技战中处于明显劣势。仅以人才为例，全球共有367所具有AI研究方向的高校，美国拥有168所，而中国只有20家，甚至低于加拿大的22家。
 <br/>
 <br/>
 外媒近期曝光一份中共深圳市政府报告指，美国将在未来半年到一年内，对中共实施全方位的技术封锁。报告建议中共政府利用这个〝空窗期〞，抓紧不择手段地引进先进设备和人才。
 <br/>
 <br/>
 《华尔街日报》此前报导指，川普（特朗普）政府已经制定一套全新策略，将协调各部门联合行动，包括利用出口管制、法律控诉等工具，用来对付所有象福建晋华一样依赖盗窃美国技术而生存的中国企业。
 <br/>
 <br/>
 根据美国对福建晋华的起诉书，台湾联华电子前高管陈正坤出身美国美光公司，从美光盗窃动态随机存储器（DRAM）技术后，又将技术转移给福建晋华。而此前中国并不掌握DRAM技术。
 <br/>
 <br/>
 <b>
  附美国政府拟实施出口管制的14项技术清单：
 </b>
 <br/>
 <br/>
 1、生物技术，例如纳米生物学；合成生物学；基因组和基因工程；神经科学。
 <br/>
 <br/>
 2、人工智能（AI）和机器学习技术，例如：神经网络和深度学习（例如，脑模拟、时间序列预测、分类）；进化和遗传计算（例如遗传算法、遗传编程）；强化学习；计算机视觉（例如，物体识别、图像理解）；专家系统（例如决策支持系统，教学系统）；语音和音频处理（例如，语音识别和制作）；自然语言处理（例如机器翻译）；规划（例如，调度、博弈）；音频和视频处理技术（例如，语音复制、deepfakes）；AI云技术；AI芯片组。
 <br/>
 <br/>
 3、定位、导航和定时（PNT）技术。
 <br/>
 <br/>
 4、微处理器技术，例如：片上系统（SoC）。
 <br/>
 <br/>
 5，先进的计算技术，例如：以内存为中心的逻辑（Memory-centric logic）。
 <br/>
 <br/>
 6、数据分析技术，例如：可视化；自动分析算法；语境感知计算。
 <br/>
 <br/>
 7、量子信息和传感技术，例如：量子计算；量子加密；量子传感。
 <br/>
 <br/>
 8、物流技术，例如：移动电力系统；建模和模拟系统；资产总体可见度；基于配送的物流系统（DBLS）。
 <br/>
 <br/>
 9、增材制造（例如3D打印）。
 <br/>
 <br/>
 10、机器人，例如：微型无人机和微型机器人系统；集群技术；自组装机器人；分子机器人；机器人编译器；智能微尘。
 <br/>
 <br/>
 11、脑机接口，例如：神经控制接口；意识-机器接口；直接神经接口；脑-机接口。
 <br/>
 <br/>
 12、高超音速空气动力学，例如：飞行控制算法；推进技术；热保护系统；专用材料（用于结构、传感器等）。
 <br/>
 <br/>
 13、先进材料，例如：自适应僞装；功能性纺织品（例如先进的纤维和织物技术）；生物材料。
 <br/>
 <br/>
 14、先进的监控技术，例如：面印和声纹技术。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
