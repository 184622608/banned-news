### Alphabet董事长很纠结：谷歌回归中国将违背核心价值
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】美国网络巨擘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌.html" target="_blank">
  谷歌
 </a>
 （Google）的母公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Alphabet.html" target="_blank">
  Alphabet
 </a>
 的董事长约翰．汉尼斯日前承认，谷歌如果想要回到中国市场，将不得不违背公司的〝核心价值〞，为此他感到很纠结。
 <br/>
 <br/>
 据《美国之音》11月24日的报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Alphabet.html" target="_blank">
  Alphabet
 </a>
 董事长约翰．汉尼斯日前在接受《彭博社》采访时说，〝任何在中国做生意的公司都要违背他们的核心价值。因为中国的法律跟我们自己国家的法律有些不同。〞
 <br/>
 <br/>
 汉尼斯坦言自己为此很纠结。他说：〝让我纠结的问题是：我们是不给中国公民一个像样的有能力的搜索引擎，即使它在某些情况下受到限制和审查更好，还是干脆让他们用一个不怎么好的搜索引擎？这会改善他们的生活质量吗？〞
 <br/>
 <br/>
 据了解，自2000年以来，〝不作恶〞 （Don't be evil）一直是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌.html" target="_blank">
  谷歌
 </a>
 公司行为准则中首要的一条。长期以来，这一准则已深深融入谷歌的公司文化中。曾有美媒报导称，这句话的一个版本已经成为谷歌在其山景城总部接送员工的班车上的wifi密码。
 <br/>
 <br/>
 由于严重不满中共政权强迫谷歌配合中共的网络审查，加上谷歌在大陆受到的网络攻击日益加剧，谷歌于2010年撤回在大陆的搜索服务。当时，几乎全世界都在称赞谷歌秉持了〝不作恶〞原则的道德风范。
 <br/>
 <br/>
 2015年，当谷歌重组成立新的母公司Alphabet时，这条基本原则被修改为〝做正确的事〞（Do the right thing），但〝不作恶〞依然是该公司行为准则和价值观的重要组成部分。
 <br/>
 <br/>
 但今年4、5月间，有媒体曝光该公司新领导阶层为了公司盈利而打算重返大陆市场。
 <br/>
 <br/>
 有美国媒体8月份报导，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重返中国市场.html" target="_blank">
  重返中国市场
 </a>
 ，谷歌正在秘密打造一个能够配合中共网络审查要求的具有过滤功能的搜索应用程序。这个代号为〝蜻蜓〞的新搜索引擎，将配合中共的要求而过滤被中共排斥的网站和关键字。
 <br/>
 <br/>
 这一消息被曝光后，很多国际人权组织和美国政界人士先后发声，呼吁谷歌坚守其〝不作恶〞的核心价值观，不要配合中共限制言论自由的做法，取消向中共政府妥协的发展计划。
 <br/>
 <br/>
 同时，数千谷歌员工以联署公开信的方式在网络上公开了一封请愿书，告诫公司高层，配合中共的审查制度将引起道德危机。员工们要求谷歌管理层必须让相关计划的决策过程更开放透明，使员工了解自己为公司制造的是什么，让员工有更多发言权和监督权，并建立与其相关的问责制度。
 <br/>
 <br/>
 曾经参与过中国业务的3名谷歌员工告诉《路透社》，目前谷歌的领导阶层可能认为，在中国提供有限的搜索结果，比起不提供任何信息更好。
 <br/>
 <br/>
 此外，有消息指，谷歌公司公布的最新版的行为准则，在最后仍然提及了公司座右铭。行为准则的最后一句写道：〝记住……不作恶。如果你看到一些你认为不正确的事情——大声说出来！〞
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：祝馨睿）
