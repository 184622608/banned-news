### 川普搭空军一号返国 预告明年初第二次〝川金会〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 结束阿根廷20国集团（G20）峰会行程，搭机返国时表示，他希望明年初跟朝鲜领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 进行第二次峰会，时间可能在1月或2月，正考量〝3个地点〞。
 <br/>
 <br/>
 路透社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 搭乘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空军一号.html" target="_blank">
  空军一号
 </a>
 （Air Force One）返回华盛顿时，在机上告诉记者，〝我认为我们将很快举行峰会，可能在1月、2月。〞并透露，未来某个时间点，他将邀请
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 到美国参访。
 <br/>
 <br/>
 今年6月12日，川普和金正恩在新加坡举行历史性会谈。
 <br/>
 <br/>
 G20峰会期间，川普相继与韩国总统文在寅和中国国家主席习近平会晤。
 <br/>
 <br/>
 白宫为此发声明，川普和文在寅一致认为有必要〝继续强力执行现有制裁，以确保朝鲜了解非核化是唯一的路〞。还指出，他们和金正恩将致力〝实现朝鲜半岛非核化〞。
 <br/>
 <br/>
 韩国青瓦台发言人尹永灿（Yoon Young-chan）也表示，川普〝请韩国和美国密切合作，以利下次峰会成为朝鲜半岛非核化进程的另一座历史里程碑〞。
 <br/>
 <br/>
 据了解，11月上旬，美国国务卿蓬佩奥（Mike Pompeo）原本预计会见朝鲜高官，但朝鲜坚持美国需在减轻制裁情况下，华府对此无预警宣布会议延期。
 <br/>
 <br/>
 (责任编辑：卢勇信)
 <br/>
</div>
