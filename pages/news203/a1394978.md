### 史上第一例 中共间谍被引渡美国受审
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】周三，美国司法部公布，一名中共国安官员，被指控-涉嫌窃取美国公司商业机密，从比利时被引渡到美国受审。这起引渡案史无前例。
 <br/>
 <br/>
 美国司法部表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共江苏省国家安全厅.html" target="_blank">
  中共江苏省国家安全厅
 </a>
 副处长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐延军.html" target="_blank">
  徐延军
 </a>
 ，又名曲辉或张辉，周二被引渡到美国。
 <br/>
 <br/>
 检方指控徐延军为中共窃取美国航空业机密，而且并非单一事件。徐延军今年4月在比利时被捕。
 <br/>
 <br/>
 总部位于美国俄亥俄州辛辛那提的电视台WCPO-TV曝光了徐延军的照片。
 <br/>
 <br/>
 徐延军将在辛辛那提的联邦法院接受审判。
 <br/>
 <br/>
 这是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 首次被引渡到美国受审。联邦调查局反间谍部门助理主任普里斯泰普表示，这起案件，暴露了中共当局直接监督对美国的经济间谍活动。
 <br/>
 <br/>
 起诉书指，从2013年到今年4月被捕为止，徐延军试图对美国航空航天领域大企业窃取商用机密，主要针对的是美国技术专家。
 <br/>
 <br/>
 新唐人记者李梅、严曦综合报导
</div>
