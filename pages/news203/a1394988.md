### 谷歌报告洩密 承认弃言论自由审查网络
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌.html" target="_blank">
  谷歌
 </a>
 公司（Google）一份洩密的内部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_报告.html" target="_blank">
  报告
 </a>
 证实，谷歌高层认为，美国所尊崇的互联网
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_言论自由.html" target="_blank">
  言论自由
 </a>
 传统已不再可行。报告承认，谷歌企图开发配合
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 审查的搜索引擎。
 <br/>
 <br/>
 据布莱巴特新闻（Breitbart News）10月9日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌.html" target="_blank">
  谷歌
 </a>
 向该媒体洩露的一份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_报告.html" target="_blank">
  报告
 </a>
 显示，谷歌的确在有选择地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_审查网络.html" target="_blank">
  审查网络
 </a>
 言论。这份名为《优秀审查员》（The Good Censor）的长达85页的报告承认，谷歌及其它技术平台正〝控制绝大多数在线对话〞，并〝转向审查〞以应对全球不受欢迎的政治事件。
 <br/>
 <br/>
 文件列举了一些所谓这样的事件，包括2016年美国大选及〝德国另类选择党〞（Alternative fuer Deutschland, AfD）的兴起。
 <br/>
 <br/>
 该报告将谷歌描述为〝编辑〞和〝出版商〞 中的〝文明〞保证者。这一点很重要，因为谷歌、YouTube和其它科技巨头公开宣称他们不是出版商而是中立平台。根据《通信规范法》（Communications Decency Act）第230条的规定，他们据此拥有特殊的法律豁免权。
 <br/>
 <br/>
 Google承认，第230条旨在确保它们能够维护中立的自由表达平台。
 <br/>
 <br/>
 <b>
  谷歌搜索在2016年大选时倾向希拉里
 </b>
 <br/>
 <br/>
 尽管谷歌自称它在2016年大选期间曾抑制所有人的负面言论、而不仅仅是针对希拉里（Hillary Clinton）的，但是，可以很容易地发现，谷歌所抑制的言论其实主要是支持希拉里的党内对手桑德斯（Bernie Sanders）和最终对手川普的。
 <br/>
 <br/>
 心理学家爱泼斯坦（Robert Epstein）的独立研究结果也表明，谷歌的搜索结果显示，它确实在2016年支持了希拉里。
 <br/>
 <br/>
 谷歌的报告提到，谷歌曾两次将〝俄罗斯干预〞（美国大选）的字样与川普的照片并放在一起。在这一点上，谷歌承认，技术平台正在改变政策，取代了国会而对外国干涉采取行动。
 <br/>
 <br/>
 <b>
  谷歌计划开发配合
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
   中共
  </a>
  审查的搜索引擎
 </b>
 <br/>
 <br/>
 尚不清楚谷歌 〝优秀审查员〞到底想审查谁，但可以看到的是，谷歌花费了大量时间和努力去做这件事情，进行了〝好几个层面的研究〞，谘询了很多专家，其中包括来自5大洲7个国家的35名文化观察家和7名文化领导人。
 <br/>
 <br/>
 该报告给出的结论是，科技公司如果要〝全球扩张〞，就不得不审查他们的平台。谷歌表示，它计划开发一个审查版的搜索引擎，以便进入中共市场。
 <br/>
 <br/>
 （记者雨轩编译报导/责任编辑：东野）
 <br/>
