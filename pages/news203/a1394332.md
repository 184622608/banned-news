### 补强军品产业弱点 川普令生产海水锂电池
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】在美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 强烈抨击中共隔天，国防部公布一份超过280项可能影响美军重要军品与零组件供应的漏洞，说明中共对美国国防工业构成的威胁。总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 5日致信联邦众议院相关委员会表示，监于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海水锂电池.html" target="_blank">
  海水锂电池
 </a>
 相关的国防工业基础不足之处，国防部将购入设备试产这种对反潜作战至关重要的电池。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 5日在致联邦众议院金融服务委员会（House Financial Services Committee）及参议院银行委员会（Senate Banking Committee）的信函中写道，开发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海水锂电池.html" target="_blank">
  海水锂电池
 </a>
 (lithium sea-water batteries)及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_燃料电池.html" target="_blank">
  燃料电池
 </a>
 的计画，〝对国防的重要性不可言喻〞。
 <br/>
 <br/>
 在信中并明确表示，国防部将开发及购买设备及材料试产这两种电池。
 <br/>
 <br/>
 白宫5日表示，将扩大〝对工业基础的直接投资，以解决关键瓶颈，支援脆弱的供应商〞。
 <br/>
 <br/>
 中央社报导，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五角大厦.html" target="_blank">
  五角大厦
 </a>
 5日公布一份长达150页、提出超过280项，针对美国制造业与国防工业基础与供应链弹性评估报告，其中包括可能影响美军重要军品与零组件供应的漏洞，报告中也花了相当篇幅说明，中共对美国国防工业构成的威胁。
 <br/>
 <br/>
 报告显示，当局寻求补强攸关国家安全的核心产业弱点，报告也建议扩大对关键部门的直接投资。
 <br/>
 <br/>
 报告中的〝竞争国家产业政策〞列举6个面向，其中4项直接点名中共，对中共的军事扩张与现代化、经济侵略战略、软实力战略与军事研发与开支战略进行说明。
 <br/>
 <br/>
 报告也指出，虽然多个国家推行的经济政策有损美国制造业，但没有一个国家像中共成功锁定美国工业基础。中共是许多美军武器与飞弹所需关键原材料的唯一或主要供应来源，中共对其他国家，特别是非洲与拉丁美洲国家的投资以换取当地资源和市场准入，也增加美国经济与国家安全威胁。
 <br/>
 <br/>
 报告也指出，中共加入世界贸易组织（WTO）并进入美国市场之后，美中贸易逆差持续扩大，中共对经济竞争环境的非市场扭曲必须结束，否则美国将面临失去技术优势与工业能力风险，这些能力让美国维持军事优势，即使中共试图提升自我军事能力以达到美国水准。
 <br/>
 <br/>
 报告也详细说明中共的经济侵略战略，其中之一是北京提出的〝中国制造2025〞，锁定人工智慧、机器人、自主与新能源汽车、高性能医疗器材与其他对国防至关重要的产业。
 <br/>
 <br/>
 中共为获取发展这些产业所需的先进技术，透过合法与非法手段，包括外国直接投资与创投、透过个人收集、间谍与逃避美国出口管制限制等方式，获取智慧财产权与关键技术。
 <br/>
