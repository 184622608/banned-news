### FBI局长：13个炸弹邮包中炸弹都是真的
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】持续5天的13起向美国民主党政要和商界大佬寄送〝炸弹邮包〞案，周五(26日）一名嫌犯在迈阿密地区落网。FBI局长称，13个收到的邮包炸弹都不是假炸弹。
 <br/>
 <br/>
 当地时间10月26日早间，联邦警察在佛罗里达州劳德岱堡（Ft.Lauderdale）附近的种植园市（Plantation）逮捕了一名涉嫌寄送爆炸包裹的男子。该男子是56岁的塞亚克(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Cesar Sayoc.html" target="_blank">
  Cesar Sayoc
 </a>
 )，有犯罪前科。此人有个住址在佛州艾文杜拉（Aventura），不过他与纽约也有关系。
 <br/>
 <br/>
 联邦调查局局长雷伊(Chris Wray)周五表示，FBI大约是在24～36小时之间锁定嫌犯身份。
 <br/>
 <br/>
 他说，这是全国性、超大规模的侦案，相信执法当局没有抓错人，因为加州众议员华特斯(Maxine Waters)华府办公室收到的邮包上留下的嫌犯指纹和DNA与塞亚克的一致。至于嫌犯作案动机，因仍在调查，雷伊没有详细说明。
 <br/>
 <br/>
 雷伊还说，查获的13个炸弹经过联邦调查员及实验室专家确定，都不是假炸弹。
 <br/>
 <br/>
 不过，美国中情局（CIA）的一名前特工日前接受路透社采访时透露，所有被截获的可疑邮包经技术人员检测后发现，那些〝炸弹〞都没有引爆系统。也就是说，这些炸弹理论上是无法引爆的。
 <br/>
 <br/>
 这名前特工说：〝他们想做到的是引发恐慌、制造混乱，但无意伤人。因为如果他们有意伤人，炸弹理应会爆炸。〞
 <br/>
 <br/>
 司法部长塞申斯(Jeff Sessions) 周五召开记者会说，塞亚克被提控5个联邦重罪，包括跨州运输爆炸物，非法邮寄爆炸物，威胁前总统等人，威胁跨州通讯，以及攻击现任及前任联邦官员。一旦罪成，将面临58年刑期。塞申斯说，不能容忍这种枉法行为，〝不论你政治信仰为何，只要你违法，我们就会起诉你。〞
 <br/>
 <br/>
 由于调查尚未结束，对塞亚克的控罪还有可能会增加或变动。目前，执法部门仍无法确定塞亚克的作案动机。
 <br/>
 <br/>
 对谢立克提控的纽约南区联邦法院检察长柏曼(Geoffrey Berman)说，这起炸弹攻击事件是破坏民主的懦夫行为，他感谢各伙伴机构通力合作，才能将嫌犯绳之以法，保障百姓安全。
 <br/>
 <br/>
 塞亚克在被捕后，其拥有的白色箱型车也被FBI作为重要证物运走。该车车窗有一个印有〝Swamp〞的海报上贴着十几位他不满的自由派政治人物。其中一个海报上的照片包括收到炸弹邮包的喜莱莉.克林顿、CNN记者、第13个邮包的收件人加州参议员贺锦丽(Kamala Harris)，都被圈上红色枪靶标记。
 <br/>
 <br/>
 周五，在警方先后拦截要寄给新泽西民主党参议员布克（Cory Booker）、前国家情报总监克拉柏（James Clapper）和加州民主党联邦参议员贺锦丽（Kamala Harris）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸弹包裹.html" target="_blank">
  炸弹包裹
 </a>
 ，使这礼拜全美的炸弹包裹事件达到13起，此前收到包裹的人士包括前总统奥巴马，前副总统拜登及前民主党总统候选人希拉里等。
 <br/>
 <br/>
 虽然该案件还在调查中，但左派媒体几乎在事件刚一发生，就直接把矛头指向了现任美国总统川普（特朗普），抨击他平日里有〝过激言论〞激发了极端份子的这种暴力恐吓行动。而福克斯电视台主持人道布斯则认为，寄送这些不会爆炸的炸弹，明显是有人试图影响中期选举的选情。
 <br/>
 <br/>
 道布斯发推文称，他将在自己主持的晚间节目中讨论联邦调查局(FBI)、美国烟酒枪炮及爆裂物管理局(ATF)为何没有针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_可疑包裹.html" target="_blank">
  可疑包裹
 </a>
 对社会大众发布警告，以及不管这些土制炸弹是真是假，很明显就是为了影响中期选举的。这一切都是为了什么呢？
 <br/>
 <br/>
 司法部已宣布，将在美东时间26日下午2时30分就此案件召开记者会。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
