### 【禁闻】美副总统发表演讲 多方面谴责中共
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】
 <b>
  美副总统发表演讲 多方面谴责
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
   中共
  </a>
 </b>
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国副总统.html" target="_blank">
  美国副总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 10月4号在哈德逊研究所发表有关美中关系问题的演讲。
 <br/>
 <br/>
 彭斯说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 正以全政府的方式，包括政治、经济和军事以及宣传，来提升其影响力并在美国谋取利益。
 <br/>
 <br/>
 彭斯谴责中共在中国国内建立集中营迫害维族人和本国人民；在国际上勾结美国的敌人与美国为敌，孤立台湾，拉拢拉美国家与台湾断交；使用〝债务外交〞输出资本，造成参与国巨额债务；偷窃美国科技和军事技术；在南中国海和东中国海进行军事化；发起前所未有的行动, 影响美国公众舆论、影响2018年选举和2020年总统选举前的环境等。
 <br/>
 <br/>
 彭斯还点名批评谷歌和中共合作，呼吁谷歌立即终止研发〝蜻蜓〞搜寻应用程式。因为它会加强中共的网络审查，因而牺牲中国民众的权利。
 <br/>
 <br/>
 彭斯敦促中共回到正轨上来。
 <br/>
 <br/>
 <b>
  彭博：美政府和公司遭中共植入间谍
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芯片.html" target="_blank">
   芯片
  </a>
 </b>
 <br/>
 <br/>
 彭博社10月4号报导，中共军队的情报部门在一家硬件公司生产的设备中植入了恶意芯片，这种芯片只有一粒米大小，全球最少有30家企业受影响，包括苹果、亚马逊在内的科技巨头，美国中央情报局、国际太空站、美国海军等机构都在使用这些设备。
 <br/>
 <br/>
 美国政府正在调查这起事件，并认为中共军队的一个情报部门应该对此负责。多名知情人士说，调查人员发现那些芯片是在中国分包商的工厂里插入的。
 <br/>
 <br/>
