### 中期选举余音 佛州参议员、州长选举重新计票
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】在美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 中，佛罗里达州州长的竞选，选情胶着，周六提交的非正式结果显示，先前已宣布胜选的共和党候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德桑蒂斯.html" target="_blank">
  德桑蒂斯
 </a>
 ，领先民主党候选人吉隆姆不到0.5个百分点。而据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 竞选法，这个结果需要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 。在佛州的国会参议员选举中，也出现了类似情况，需重新计票。
 <br/>
 <br/>
 先前已宣布胜选州长的前国会议员罗恩-
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德桑蒂斯.html" target="_blank">
  德桑蒂斯
 </a>
 （Ron DeSantis）星期六（11月10日）通过社交媒体表示，对选举结果有信心。
 <br/>
 <br/>
 前共和党国会议员德桑蒂斯：〝今天中午，来自全美各地的选举监督员向国务卿（蓬佩奥）递交了选举汇报。这些结果清新明确，就像选举之夜的结果一样。我很荣幸获得佛罗里达选民的信任，让我担任下一任州长。〞
 <br/>
 <br/>
 德桑蒂斯还表示，遵守法律很重要；他补充道，随着选举结束，现在是全州凝聚在一起，我们准备服务于佛罗里达州的时候了。
 <br/>
 <br/>
 民主党候选人、佛罗里达州塔拉哈西市（Tallahassee）市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉卢姆.html" target="_blank">
  吉卢姆
 </a>
 （Andrew Gillum）在确定需要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重新计票.html" target="_blank">
  重新计票
 </a>
 后，星期六在新闻发布会上宣布，撤回他在星期二选举之夜的败选声明，但同时也强调，他将接受重新机票后的任何结果。
 <br/>
 <br/>
 佛罗里达州长候选人吉卢姆：〝让我明确地说，我毫不妥协地、毫无歉意的改变我败选的言辞。我们要计算每一张选票。我也承认，我的命运可能改变、也可能不改变。〞
 <br/>
 <br/>
 周六，在国会参议员竞选中领先的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 州长斯科特也发现他的领先优势降低至不到0.5%。斯科特要求佛州警长检视重新计票中可能的违规行为。此外，斯科特对两个县的民主党竞选监督员提起诉讼，指控他们违反了竞选法。
 <br/>
 <br/>
 新唐人记者徐哲、田瑞综合报导
</div>
