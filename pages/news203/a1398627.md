### 白宫：CNN被吊销通行证的记者对女实习生动手
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】白宫周三晚间宣布取消美国线电视新闻网（CNN）一名记者的白宫通行证，理由是这名记者〝对一个试着做本分工作的白宫女实习生动手〞。CNN却声称是因为记者的提问具有〝挑战性〞触怒了白宫。白宫发言人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桑德斯.html" target="_blank">
  桑德斯
 </a>
 公开批评CNN的所作所为〝恶心〞。
 <br/>
 <br/>
 美东时间11月7日晚间，白宫对外宣布取消了CNN记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿科斯达.html" target="_blank">
  阿科斯达
 </a>
 （Jim Acosta）进入白宫的记者通行证，指其当天在记者会上对白宫一名年轻的女实习生〝动手〞。
 <br/>
 <br/>
 CNN随即发表声明，表示要〝全力支持阿科斯达〞。这份声明辩解说阿科斯达之所以被取消通行证件是白宫报复他当天在记者会上提出了〝具挑战性问题〞，并抨击白宫的这个决定〝威胁我们的民主〞。
 <br/>
 <br/>
 白宫发言人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桑德斯.html" target="_blank">
  桑德斯
 </a>
 （Sarah Sanders）则声明称，中止阿科斯达的记者通行证，是因为他在记者会上〝对一个试着做本分工作的白宫实习生动手〞。
 <br/>
 <br/>
 桑德斯说：〝这种行为完全令人无法接受，（阿科斯达）也十分不尊重其他记者同行，不给他们机会提问。总统川普给记者的采访自由比史上任何总统都要多。〞
 <br/>
 <br/>
 桑德斯表示，情况并不是CNN所说的那样。事实是，川普总统在7日当天的活动中，用了一个半小时回答了35名记者的68个问题，其中也包括回答CNN好几个问题，充分体现了对新闻自由的支持，但即使这样CNN还攻击总统，说他不支持新闻自由。
 <br/>
 <br/>
 桑德斯说：〝CNN以做出这种行为的员工为傲，这个事实不但恶心，还证明了他们恶劣的不尊重其他任何人，包括替这个政府效力的年轻女性。〞
 <br/>
 <br/>
 据了解，中期选举落幕的第二天（11月7日），美国总统川普举行了一次全国直播记者会。在这次记者会上，CNN记者阿科斯达就大篷车难民问题对川普提出一连串的质问。
 <br/>
 <br/>
 阿科斯达问：〝你为什么把他们（大篷车移民）描绘成入侵者？〞
 <br/>
 <br/>
 川普回答说：〝因为我认为这是入侵，我们观点不同。〞
 <br/>
 <br/>
 阿科斯达又问：〝你是否认为妖魔化移民.....？〞
 <br/>
 <br/>
 川普表示：我希望他们来，但他们应该合法地进来，必须按照程序来。事实上有数百公司搬进美国来，所以美国是需要人的。
 <br/>
 <br/>
 阿科斯达立即呛声说：〝但是他们还在千里之外呢。〞
 <br/>
 <br/>
 川普接着反击道：〝坦率地说，你应该让我来治理国家，你管CNN，如果你做得好，你们的评级会好得多。〞
 <br/>
 <br/>
 随后，阿科斯达不顾川普要求其坐下，仍然紧握麦克风继续提问。这时，白宫一名女实习生上前试着从阿科斯达的手中拿走麦克风，但阿科斯达坚持不肯放手，导致两人发生短暂肢体冲突。
 <br/>
 <br/>
 当晚，阿科斯达想再回到白宫做连线时，在供记者出入的宾夕法尼亚大道入口遭特勤局拦下。于是他发推文称，〝我刚才进入白宫时被拒，特勤局告诉我，我不能入内进行晚上8时的节目。〞
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：戴明）
