### 50年最强飓风迈克尔登陆佛州 全州进入紧急状态
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】4级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 〞（Hurricane Michael ）10月10日下午登陆佛罗里达州狭长地带后威力继续增强，风速已达时速250公里，成为50年来吹袭美国本土的最强飓风。迈克尔所到之处，带来足以对生命造成严重威胁的风暴雨，目前佛州已全州进入紧急状态。
 <br/>
 <br/>
 据路透社报导，位在迈阿密的美国国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风.html" target="_blank">
  飓风
 </a>
 中心（NHC）10日表示，朝佛罗里达州墨西哥湾沿岸移动的飓风
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 （Michael），今天在距佛州巴拿马市（Panama City）东南方约30公里处时，持续风速已高达时速250公里，威力已增强为〝极度危险〞的4级飓风。随着迈克尔外围环流开始触陆，部分地区预料将出现高达4公尺的巨浪，稍后这股飓风登陆的佛州狭长地带（Florida Panhandle）或大弯地区（Big Bend），恐将遭遇致命性威胁，该地区将面临前所未见的风灾。
 <br/>
 <br/>
 美国总统川普日前已宣布佛州进入紧急状态，强调联邦政府将全力协助地方政府救灾。
 <br/>
 <br/>
 与此同时美国国家气象局（National Weather Service）驻佛州首府塔拉哈西（Tallahasee）办公室也提前发布了公告，极力呼吁民众遵守政府的通知立即撤离。
 <br/>
 <br/>
 国家气象局的声明称，〝飓风迈克尔会带来前所未见的风灾，我们过去遭遇的任何状况都无法相比。千万别拿生命冒险，如果你接获撤离通知，就立即撤离。〞
 <br/>

 <br/>
 美国国家飓风中心（NHC）主任格雷厄姆（Ken Graham）在接受媒体采访时也表示，飓风佛罗伦斯（Florence）上月在变弱后才登陆北卡罗莱纳州，但这次的迈克尔在登陆前却不断在增强，其破坏力也将大大增强。除了首当其冲的佛州外，阿拉巴马州、乔治亚州及南、北卡罗莱纳州都会下起大雨。他说，地方政府已出动2500名国民警卫军进行戒备。
 <br/>
 <br/>
 NHC还在通告中表示，预测飓风迈克尔将在星期五（12日）成为后热带气旋，并向大西洋西部移动。
 <br/>
 <br/>
 另据路透社的报导，飓风迈克尔近逼美国佛州之前，在中美洲已导致至少13人死亡。
 <br/>
 <br/>
