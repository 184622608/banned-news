  大篷车
 </a>

 </a>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入侵.html" target="_blank">

 移民问题的立场。

 周一下午，五角大楼宣布，计划在西南边境部署5200名军人，以抵挡数千名将抵达美墨边境的洪都拉斯移民。
 <br/>
 <br/>
 美国国防部上周批准800士兵增援边境，但本周决定进一步增兵，共计派出5200人，阻拦将在边境申请庇护的大篷车移民。
 <br/>
 <br/>
 一名国防部官员称，具体人数可能稍高或稍低，人员主要来自现役陆军，可能还包括一些陆军预备队和海军陆战队成员。组成人员包括军警、规划人员、工程师、飞行员、厨师和医务人员等。
 <br/>
 <br/>
 这些军人计划将先被分派到各个入境口岸，支援边境巡逻人员、美国海关和边境保护局（U.S. Customs and Border Protection, CBP）处理大篷车到达后的相关问题。
 <br/>
 <br/>
 CBP局长麦克莱南（Kevin McAleenan）表示，他要确保CBP能够保卫边境口岸，防止大群人一下子涌入。
 <br/>
 <br/>
 上周末，墨西哥政府宣布向已进入墨西哥的中美洲移民提供庇护、工作、教育和医疗服务，但他们拒绝并继续向北进发，誓言设法入境美国。
 <br/>
 <br/>
 更为严重的是，第二批试图前往美国的中美洲移民在墨西哥和危地马拉边界遭墨西哥防暴警察拦截，之后双方爆发冲突，一些难民强行突破边境大门，导致1名移民死亡，6名警察受伤。
 <br/>
 <br/>
