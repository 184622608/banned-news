### 〝一个协议两种表述〞  一文看懂川习会后美中不同表述
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】美中首脑在阿根廷举行的晚餐会上，就贸易冲突达成了〝暂时休战〞协议，但舆论注意到双方披露的协议内容有明显区别。对此，有英媒梳理了中美说法的四大不同，而有推文则列出表格，让人对中美表述上的诸多区别之处一目了然。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）与中国国家主席习近平于12月1日在阿根廷二十国集团（G20）峰会期间举行会谈，会谈结束后，中共很快通过央视发布消息称，美中两位领导人达成重要共识，
 <br/>
 <br/>
 《路透社》3日盘点了美中两国对协议内容的说法存在的4大不同。
 <br/>
 <br/>
 1，对中国经济的构造性改革的说法不同。
 <br/>
 <br/>
 白宫声明称，川普总统和习近平同意立即就中国的结构性改革开始谈判，包括在强制技术转移、知识产权保护、非关税壁垒、网路攻击、服务以及农业等领域。
 <br/>
 <br/>
 而中共外交部长王毅对此并没有具体提及，只是在记者会上说，中方将扩大从美国进口商品的规模，努力修正中美之间的贸易不平衡状况。双方同意相互开放市场，在中方推进新一轮改革开放中，使美方的合理关切得到逐步解决。
 <br/>
 <br/>
 2，对90天的停火限期说法不同。
 <br/>
 <br/>
 白宫声明称，如果在90天后双方仍无法达成协议，美国将对中国2000亿美元商品的关税提高至25%。而中方通报只称中方会〝使美方的合理关切得到逐步的解决〞，但对于90日限期则只字不提。
 <br/>
 <br/>
 报导提到中共外长王毅没有提到期限。但是有中共官方媒体提到了期限。中共商务部副部长王受文对记者的话说，中美将加紧谈判取消现行的25%关税，但没提到时间。
 <br/>
 <br/>
 3，对美国高通对荷兰恩智浦半导体（NXP）发起的收购案说法不同。
 <br/>
 <br/>
 白宫声明称，习近平说，如果高通收购恩智浦的案件再次提交，他对批准这项此前未获批准的交易持开放态度。而中共高官在记者会上没有提及这一议题，至少有一家中共国营媒体在中国社交平台微信上，援引了白宫声明提到的高通恩智浦收购案。
 <br/>
 <br/>
 4，对中国进口美国商品的说法不同。
 <br/>
 <br/>
 白宫声明表示，中方将〝立刻〞开始大量进口美国的农产品、能源以及工业制品，但未披露进口规模。而中共通报中没有提及扩大进口的美国商品种类，以及何时开始进口。
 <br/>
 <br/>
 12月3日，名为〝Jan Luo〞的推友在推特上列出一份更详细的表格，让人一目了然的了解一个协议美中不同表述之处。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/12-4/p9155301a11581274.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/12-4/p9155301a11581274-ss.jpg"/>
   <br/>
   <font size="-1">
    中美双方就贸易战达成了〝暂时休战〞协议后，舆论注意到双方披露的协议内容有明显区别。（Jan Luo推特图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 对于美中双方达成暂时停火协议，有中国学者认为，北京形同被判三个月缓刑，中共官媒不愿向国人如实汇报，正是因为这对中共是致命的，这种隐瞒不报，或许说明了北京当局不想真正做出变革。
 <br/>
 <br/>
 由于早前习近平在与川普会晤时提出的〝百日计划〞并未实现，外界因此也对此次中美90天后的谈判结果抱持怀疑。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 前，已有多家媒体报导说，中共急切希望〝川习会〞达成〝停火〞协议，不再升级贸易冲突。为此，北京可能做出更多让步，来换取美国暂缓进一步加征关税。
 <br/>
 <br/>
 不过，出于面子和维持政权稳定的考虑，北京越是做出让步，越不会对外发布〝川习会〞的真实结果。
 <br/>
 <br/>
 时政评论周晓辉分析说，不管中共是否是真心回应美国的要求，美国给出的三个月限期，此次可能已是其最后的机会了。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：范铭）
