### 签新法促选举安全 川普答完穆勒书面提问
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）总统表示，已经书面回答了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 特别检察官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆勒.html" target="_blank">
  穆勒
 </a>
 的提问，指出没有通俄之事。当天，川普还签署了〝网络安全和基础设施安全法〞，将建立
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络安全局.html" target="_blank">
  网络安全局
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 表示，律师没有帮助他，所有问题都是他本人回答。
 <br/>
 <br/>
 川普总统：〝我被问一系列问题，我回答得很轻松〞
 <br/>
 <br/>
 川普说，已完成书面回答，但因为忙还没有提交。
 <br/>
 <br/>
 川普认为该调查正在结束中，确信一切安好。
 <br/>
 <br/>
 川普总统：〝因为没有通俄之事。〞
 <br/>
 <br/>
 川普总统指出捕风捉影的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治陷害.html" target="_blank">
  政治陷害
 </a>
 〞不该再发生。
 <br/>
 <br/>
 川普总统：〝捕风捉影的政治陷害，像我说的，应该永远不再发生。（但）它还在继续。 〞
 <br/>
 <br/>
 川普是在签署〝网络安全和基础设施安全法〞的白宫仪式上，回答记者提问。
 <br/>
 <br/>
 该法案将指令国土安全部设立〝网络安全和基础设施安全局〞，其将与各级政府部门合作，全力保护民用和联邦网络，还有包括有关政治选举机构在内的关键基础设施。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
