### 川普：动用一切资源打击人口贩卖
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】周四，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在白宫打击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人口贩卖.html" target="_blank">
  人口贩卖
 </a>
 年会上发表演説。川普表示，将动用一切
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_可用资源.html" target="_blank">
  可用资源
 </a>
 ，打击愈演愈烈的人口贩卖现象。
 <br/>
 <br/>
 周四，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府15个部门官员，参加了有关打击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人口贩卖.html" target="_blank">
  人口贩卖
 </a>
 的年会。会上，川普为在这一领域中做出杰出贡献的个人颁奖。
 <br/>
 <br/>
 美国总统  川普：〝（人口贩卖）是亟待解决的人道主义问题。我的政府正致力于动用所有的资源应对这一威胁，援助人口贩卖受害者和幸存者，并把那些罪恶滔天的人口贩子绳之以法。〞
 <br/>
 <br/>
 周四，白宫发布公告，总结过去一年川普政府打击人口贩卖的成绩。今年4月，川普签署历史性法案，打击涉嫌性交易的人口贩卖网站，将参与者绳之以法。
 <br/>
 <br/>
 去年，司法部将500名涉嫌人口贩卖的被告人定罪，42个参与儿童性交易的人口贩卖公司被调查取缔。国土安全部调查了800多个人口贩卖案件，逮捕1,500人次，找到500名受害者。
 <br/>
 <br/>
 川普表示，美国也在全世界范围内不遗余力地打击人口贩卖。
 <br/>
 <br/>
 美国总统  川普：〝在世界上，我们也担当了打击人口贩卖罪行的领导者角色。我们已经向国际终结现代奴隶制度组织捐助数百万美金。如果你相信标题所言，是为了终结现代奴隶制。〞
 <br/>
 <br/>
 白宫报告明确表示，将抵制来自独裁主义国家的劳工贩卖、涉嫌性交易以及儿童交易的人口贩卖。
 <br/>
 <br/>
 美国国务院〝2018年度人口贩卖报告〞，将中国、朝鲜等23个国家列为人口贩卖最严重的国家。
 <br/>
 <br/>
 根据这份报告，全球大约有2490万人口贩卖受害者。
 <br/>
 <br/>
 新唐人记者林宇、张琪综合报导
</div>
