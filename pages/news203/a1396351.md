### 白宫将公布最新非洲战略 抗衡中共
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】美国官员表示，美国将公布最新的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲战略.html" target="_blank">
  非洲战略
 </a>
 ，侧重在非洲的投资和创造就业机会，抗衡中共〝一带一路〞。
 <br/>
 <br/>
 十月份，川普签署了《国际发展投资法案》，设立〝美国国际发展金融公司〞，为向非洲投资的美国公司提供贷款。同时拨出600亿美元，帮助
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲建设.html" target="_blank">
  非洲建设
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基础设施.html" target="_blank">
  基础设施
 </a>
 。
 <br/>
 <br/>
 美国国际发展署署长格林表示，美国和中共当局对非洲的投资根本上不同，北京热衷于提供对非洲未来发展不可持续的财政贷款，而美国希望投资能带来改革。
 <br/>
 <br/>
 美国国务院非洲事务助理国务卿纳吉表示，非洲国家希望美国企业入驻，认为美国的投资能创造很多就业机会。
 <br/>
 <br/>
 今年9月，中共当局宣布，再向非洲提供600亿美元支持。西方国家看到，中共当局的投资是有条件的，让非洲国家未来陷入债务危机。
 <br/>
 <br/>
 新唐人记者毕心慈、田瑞综合报导
</div>
