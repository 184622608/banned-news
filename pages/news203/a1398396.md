### 美〝禁售令〞再度发威 晋华公司瘫痪 员工找退路
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】中国半导体制造商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 因涉嫌盗窃美国公司商业机密，被美国商务部下达〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁售令.html" target="_blank">
  禁售令
 </a>
 〞，美国司法部日前也对晋华提起刑事及民事诉讼。在美国的严厉制裁施压下，晋华疑似步中兴后尘，已濒临崩溃。据知情人士透露，目前，晋华晶元生产线已全部瘫痪，一些员工打算离职，寻找退路。
 <br/>
 <br/>
 10月29日，美国商务部发表声明，限制对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 集成电路有限公司(Fujian Jinhua Integrated Circuit Co.)的出口。美国商务部长罗斯在声明中表示，福建晋华生产能力的扩大，很可能得益于〝源自美国的科技〞。
 <br/>
 <br/>
 11月1日，美国司法部对福建晋华及其合作伙伴台湾联华电子，以及三名台湾人提出起诉，指控这两家公司及涉案人窃取传送和持有美国美光公司的商业机密。
 <br/>
 <br/>
 美方起诉书指，中共本不掌握DRAM技术。该技术是晋华和台湾联华电子从DRAM巨头美光公司盗窃而来，随后被中共国务院列入〝国家经济重点〞。但福建晋华日前仍声明称，〝不存在窃取其他企业技术的行为〞。
 <br/>
 <br/>
 据台媒报导，从美国下达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁售令.html" target="_blank">
  禁售令
 </a>
 的那一天起，晋华的美国半导体合作方驻厂人员已全撤出，并中断所有电话、邮件往来。所有设备、生产运作都已经全面停止，已经下单却未出货的机台全数暂停出货。晋华原本预计年底试产的DRAM生产线立即全部瘫痪。
 <br/>
 <br/>
 此外，晋华的DRAM技术合作开发伙伴，台湾联电公司10月31日宣布，该公司决定停止与晋华的DRAM开发项目。
 <br/>
 <br/>
 台湾《自由时报》称，由于设备含有美国原件，欧洲、日本的半导体企业也已暂停对福建晋华的供应。
 <br/>
 <br/>
 晋华公司被起诉后，美国最大存储晶元生产商美光公司（Micron）11月1日发表声明称，该公司对于美国司法部的做法感到欣慰，晋华窃取商业机密的犯罪行为，理应得到公正处理。
 <br/>
 <br/>
 早在去年12月，美光公司就在加州起诉福建晋华和联华电子盗取其商业机密。但晋华否认了相关指控。此案目前仍在加州法院审理中。
 <br/>
 <br/>
 然而，令人意外的是，今年1月，美光反被福建晋华和联华电子在中国境内控告侵犯专利，并被福建法院判处禁售令，美光部分产品在中国境内暂时被禁售。
 <br/>
 <br/>
 外界认为，北京当局这一裁决激怒了美国政府，成为晋华公司遭美国政府制裁并被提起诉讼的直接导火线。
 <br/>
 <br/>
 最新消息称，在美国出台的禁令下，福建晋华原计划今年底试产的系统内存（DRAM）或已无法实现。目前，晋华疑似步中兴后尘，濒临崩溃。
 <br/>
 <br/>
 据知情人士透露，一些晋华员工已开始为自己寻找后路，打算离职。晋华的生产也被曝全面停止。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴通讯.html" target="_blank">
  中兴通讯
 </a>
 因违反美国对伊朗和朝鲜的制裁，对其出售产品被美国处罚，曾一度被禁售美国产品，中兴在短时间内即面临破产边缘，后缴纳10亿美元罚金，被暂时解除禁令，才得以死而复生。
 <br/>
 <br/>
 福建晋华和中兴通讯都是〝中国制造2025〞计划企业，官方对福建晋华的DRAM生产线投资56.5亿美元，中共发展国有晶元是该计划的核心内容。
 <br/>
 <br/>
 有分析人士称，晋华事件也反映出，中共盗窃美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 已成常态。如果中共其他晶元企业受到类似禁令，那么中共的晶元项目将可能面临〝末日〞，因为全球晶元制造商都无法摆脱对美国的依赖。
 <br/>
 <br/>
 政事评论人士唐靖远认为，美国对中兴还只是行政处罚，而对晋华还有刑事、民事起诉，性质更严重。或面临高额罚款，罚金可能会超过200亿美元。
 <br/>
 <br/>
 另外，晋华被处罚的原因是其涉嫌盗窃美国知识产权，而中共在科技领域国企的发展大多都是依靠此方法，除了福建晋华之外，长鑫正在合肥兴建价值80亿美元的DRAM生产线、长江存储拟在武汉投资240亿美元生产NAND flash（储存型快闪记忆体）等，他们都位列〝中国制造2025〞计划。因此，美国严惩晋华，对其他国企的震慑力度就相当大。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
