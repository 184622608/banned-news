### 加州山火增至31死 风助火势继续蔓延
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】自上周四以来，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州大火.html" target="_blank">
  加州大火
 </a>
 持续蔓延。截至周一，死亡人数上升至31人，另有300多人下落不明。而在干燥强风天气下，周一南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 又新添两场火情。请看记者在前线的报导。
 <br/>
 <br/>
 记者：我现在位于靠近马里布地区的千橡市，根据消防单位稍早公布的情况，目前伍尔西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野火.html" target="_blank">
  野火
 </a>
 ，火势只有20%受到控制，延烧的面积长约20英里，大约有40英里的宽度。而今天每小时40英里的风速，可能意味着火势将很有可能进一步延烧。
 <br/>
 <br/>
 距离洛杉矶以北90公里的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文图拉县.html" target="_blank">
  文图拉县
 </a>
 （Ventura County），伍尔西（Woolsey Fire）和希尔（Hill Fire）两场野火在干燥的强风助力下，房屋被烧毁，至少导致两人死亡，超过百人失踪。有3名消防员在灭火时被受伤。大火对多个社区造成威胁，迫使26.5万人撤离家园。在受灾最严重的海滨小城马里布（Malibu），有好莱坞明星的住宅也遭焚毁，整个小城一半地区断电。由于火场地区植被干枯，且风势强烈，周一上午，南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 又发生两起新的火情。
 <br/>
 <br/>
 加州文图拉县消防局长Mark Lorenzen：〝很显然，这让我们意识到，我们仍处于重大火灾天气，已发生的大火已不再是唯一需要关注的。〞
 <br/>
 <br/>
 加州洛杉矶县消防局长Daryl Osby：〝消防员现在首先关注的是生命第一，其次是保护财产，然后是控制火线。〞
 <br/>
 <br/>
 在北加州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴特县.html" target="_blank">
  巴特县
 </a>
 （Butte County），营地野火（Camp Fire）已发展成加州历史上最具破坏力的大火，大火蔓延至457公里，卷起的烟尘向西北飘散至旧金山湾区，空气污染严重。当局统计，大火已烧毁了6400多栋民宅以及260栋商业建筑，至少导致29人死亡，仍有228人报告失联。拥有2.7万人口的天堂镇（Paradise）几乎被完全摧毁。
 <br/>
 <br/>
 截至周一，当局已投入近8000消防人力投入灭火，犹他州、华盛顿州、俄勒冈州等6个州也前来救援。总统川普已同意州长布朗的请求，宣布加州进入野火紧急状态，并批准联邦援助。
 <br/>
 <br/>
 新唐人记者王姿懿、林骁然加州洛杉矶、旧金山连线报导
</div>
