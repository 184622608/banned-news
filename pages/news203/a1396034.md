### 川普蒙大拿集会  助阵共和党中期选举
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】周四，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 出席蒙大拿州集会，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 候选人助阵。川普在演説中谈到最近的难民潮，再次强调保卫美墨边境。
 <br/>
 <br/>
 川普周四下午飞抵蒙大拿州西部的米苏拉市，出席助选集会。
 <br/>
 <br/>
 这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，蒙大拿州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗森戴尔.html" target="_blank">
  罗森戴尔
 </a>
 将挑战民主党候选人泰斯特，竞选争国会参议员。
 <br/>
 <br/>
 川普呼吁民众站出来投票，支持共和党候选人。
 <br/>
 <br/>
 美国总统 川普：〝这次选举事关（大法官）卡瓦诺、（移民）大篷车、法治、秩序以及常识。这次选举将是这样。〞
 <br/>
 <br/>
 谈到最近再次从危地马拉出发，涌向美国的数千人移民大篷车，川普表示，将会保护美国的边境。
 <br/>
 <br/>
 美国总统 川普：〝各种原因，四千人的大篷车，不过我想感谢墨西哥政府，因为他们希望在进入墨西哥之前就拦住他们。你们知道，我会派军队保卫南部边境，如果必要的话。〞
 <br/>
 <br/>
 在一个月前的最新民调中，罗森戴尔落后泰斯特2到4个百分点。
 <br/>
 <br/>
 川普在集会前后，连发两条推文，力挺罗森戴尔。
 <br/>
 <br/>
 川普同时也表示支持共和党众议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_詹福特.html" target="_blank">
  詹福特
 </a>
 。詹福特将在11月的中期选举中，谋求连任。
 <br/>
 <br/>
 美国总统 川普：〝选择哪个再清楚不过了，民主党制造乱民，共和党创造就业。〞
 <br/>
 <br/>
 集会后，川普在推特上发布这次野外集会的照片，感谢热情的蒙大拿民众。
 <br/>
 <br/>
 新唐人记者林宇、天羽综合报导
</div>
