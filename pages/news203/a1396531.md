### 新规再松绑奥氏健保束缚 州和企业更自主
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】近期，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 加紧推行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新规定.html" target="_blank">
  新规定
 </a>
 ，让美国的地方政府和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_企业.html" target="_blank">
  企业
 </a>
 有更多自由，选择〝奥巴马健保〞外的健保计划。从本周开始，又有两项新规出炉。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普政府.html" target="_blank">
  川普政府
 </a>
 推出两项
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新规定.html" target="_blank">
  新规定
 </a>
 ，将各州及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_企业.html" target="_blank">
  企业
 </a>
 逐步从奥巴马健保的束缚中解脱出来。
 <br/>
 <br/>
 联邦医疗保险与医疗补助服务中心（CMS）周一宣布了一项即刻实施的新〝联邦豁免〞项目，各州通过申请新豁免，可以选择奥巴马健保市场之外的保险计划，包括对年轻人和健康者更具吸引力的短期健保方案，而不必再受制于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥氏健保.html" target="_blank">
  奥氏健保
 </a>
 对保障项目一致性的规定。同时，也可保障存在既往病症的保户选择需要的保险项目 。
 <br/>
 <br/>
 〝美国企业研究所〞高级研究员 Karlyn Bowman：〝健保议题在每一个选举中都是关键议题，是一个非常敏感的政治议题。奥巴马提出可负担健保法案时曾获得支持，但之后人们转而批评，因为做得太过了。所以在全国那些竞争激烈的国会众议员竞选中，共和党都在说我们会确保既往症人群的保险权益。〞
 <br/>
 <br/>
 另一项新规定允许企业使用〝医疗偿付方案〞（health reimbursement arrangements,HRAs）中的免税资金，为购买私人健保公司计画的雇员报销医疗费，通常是保险方案中的免配额和共付额部分，最高每人每年可报销1800美元。
 <br/>
 <br/>
 除了引入私人保险公司增加竞争外，本月中旬，川普签署协会健保计划（Association Health Plans，AHP），允许小企业组团跨州购买健保，使小企业更容易获得可负担健保。
 <br/>
 <br/>
 对于民主党推动的〝全民健保〞，川普多次指出那是一个谎言，最终会让所有人失去保险。俄勒冈州选择加入奥氏健保中的扩大低收入健保（Medicaid Expansion），令该州在今年不得不增加税收来支付政府补贴保费所导致的财政缺口。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
