### 中期选举后第一棒 美对中国铝产品课税 最高176.2%
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】美国共和党在中期选举中失去众议院主导权。但外界普遍认为，美国对中共态度将更强硬。就在中期选举结束的当天，美方宣布对中国产铝产品加征关税，以打击中共对美倾销。次日，美国又对另一种中国商品展开反倾销调查。
 <br/>
 <br/>
 美国商务部本周三（11月7日）宣布，对特定的进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国铝产品.html" target="_blank">
  中国铝产品
 </a>
 加征反倾销、反补贴税，征税额度在96.3%到176.2%之间。美国国际商务委员会将在12月20日对此做出最终决定。
 <br/>
 <br/>
 这是自1985年以来，美国第一次在开展双反（反倾销、反补贴）调查之后，裁定施加的关税。
 <br/>
 <br/>
 美国商务部表示，2017年美国进口的中国铝片价值大概在9亿美元。美国铝业集团曾发表联合声明指出，近十年以来，中国铝制品进口提高了大约750%，而其中91%的增幅发生在2014年到2017年这三年间，认为这是中共利用政府补贴对美国低价倾销的结果，严重损害了美国的工业。
 <br/>
 <br/>
 11月7日凌晨，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 投票刚刚结束。最终川普（特朗普）所在的共和党保住了参议院的多数党地位，却失去了众议院主导权。但外界认为，遏制中共已是美国两党共识，民主党主导众议院后，川普政府对中共的立场可能会更加强硬。
 <br/>
 <br/>
 除了7日裁定对中国铝产品加税，美国商务部8日又宣布启动又一项针对中国商品的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双反调查.html" target="_blank">
  双反调查
 </a>
 。
 <br/>
 <br/>
 这次的调查对象是中国和印度向美国出口的聚酯加工丝纱（polyester textured yarn）。调查目的是确定中印出口的聚酯加工丝纱是否以低于公平水准的价格销售，以及两国生产厂商是否接受了政府的不公平补贴。
 <br/>
 <br/>
 中共政府被指称以约75%到77%的倾销幅度，向美国倾销聚酯加工丝纱。此外，中共还以20种补贴项目帮助本国厂商，包括低成本投入、优惠贷款、拨款、以及所得税减免等。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：）
