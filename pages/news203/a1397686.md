### 川普或敲定新联合国大使 国务院发言人成首选
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】消息人士透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统正在挑选新的美国驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国大使.html" target="_blank">
  联合国大使
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国务院发言人.html" target="_blank">
  国务院发言人
 </a>
 诺尔特（Heather Nauert）目前是他的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首选.html" target="_blank">
  首选
 </a>
 。不过，白宫拒绝对这一消息做出评论。
 <br/>
 <br/>
 据美国有线电视新闻网（CNN）周四（11月1日）报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 告诉白宫顾问，他认为诺尔特是接替黑利（Nikki Haley）出任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国大使.html" target="_blank">
  联合国大使
 </a>
 的最佳人选，他最快将于本周宣布这一决定。
 <br/>
 <br/>
 CNN称，该消息来自两名知情人士，不过他们警告说，在川普正式宣布前，不能把这视为最终决定。
 <br/>
 <br/>
 诺尔特周一与川普会晤，之后一直保持低调。本周的两场日常新闻发布会都是由她的副手主持。
 <br/>
 <br/>
 川普早前已有几个考虑人选，其中最为满意的是前副国家安全顾问鲍威尔（Dina Powell），但她拒绝了这一提议。
 <br/>
 <br/>
 诺尔特现年48岁，曾先后担任美国广播公司新闻网（ABC News）的新闻记者和福克斯新闻（Fox News）的主持人。她从2017年4月开始担任美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国务院发言人.html" target="_blank">
  国务院发言人
 </a>
 ；自2018年3月起，她兼任负责公共外交和公共事务的代理副国务卿。
 <br/>
 <br/>
 诺尔特工作出色，但鉴于她从事外交工作的时间尚不足两年，如果川普提名她出任新任联合国大使，预计参议院民主党将会阻挠。
 <br/>
 <br/>
 白宫拒绝对这一消息做出评论。
 <br/>
 <br/>
 川普上月初在白宫椭圆形办公室宣布，他已接受黑利的辞呈，同时向在场记者称颂黑利的工作，评价黑利是个〝极好的人〞。
 <br/>
 <br/>
 川普称，黑利早在6个月之前就告诉她〝可能会在年底〞休息一下，但表示愿意工作到2018年底。
 <br/>
 <br/>
 黑利将于明年1月正式卸任。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
