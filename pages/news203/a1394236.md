### 美9月失业率仅3.7%  创50年来新低
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】美国劳工部周五（10月5日）报告，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_9月.html" target="_blank">
  9月
 </a>
 份的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业率.html" target="_blank">
  失业率
 </a>
 仅有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_3.7%.html" target="_blank">
  3.7%
 </a>
 ，创下近50年来的新低。此外，其它报告显示，由于当前美国就业形势乐观，所以申请救济的人数也在大幅减少。
 <br/>
 <br/>
 根据美国劳工统计局（Bureau of Labor Statistics）最新发布的数据，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_9月.html" target="_blank">
  9月
 </a>
 ，美国新增
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业岗位.html" target="_blank">
  就业岗位
 </a>
 134,000个，低于预期；但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业率.html" target="_blank">
  失业率
 </a>
 降至
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_3.7%.html" target="_blank">
  3.7%
 </a>
 ，创下1969年以来的最好记录。
 <br/>
 <br/>
 经济学家此前预测，9月份的失业率可能低达3.8%，而实际情况甚至更好。
 <br/>
 <br/>
 不过，9月份新增工作岗位低于经济学家预计的平均数184,000个，其中零售业的就业人数减少了20,000人。
 <br/>
 <br/>
 与去年同期相比，9月份的平均时薪增长了2.8%，符合之前的预测。平均每周工作时间没有变化，仍为34.5小时。
 <br/>
 <br/>
 值得一提的是，美国9月份所取得的经济成就是在一些不利的因素下进行的，包括美中贸易争端加剧，〝佛罗伦斯〞（Florence）飓风猛烈侵袭北卡罗来纳州和南卡罗来纳州。这些都可能对当月工作造成了影响。
 <br/>
 <br/>
 Bankrate.com网站资深经济分析师哈姆里克（Mark Hamrick）指出，飓风登陆及洪水肆虐期间，很多酒吧和餐馆都停业了，其岗位需求减少了17,000个。
 <br/>
 <br/>
 另外，零售业就业人数下降，也反映出亚马逊等电商企业给美国传统零售业所带来的日益激烈的竞争。
 <br/>
 <br/>
 劳工部周四的一份报告还显示，9月份美国首次申请失业救济的人数约为 207,000人 ，比上一周减少了8,000人，超出了经济学家的预期。他们此前预测，这个数字将继续保持在215,000左右。这显示，美国当前的劳动力市场更加强大，能迅速吸纳新的员工。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
