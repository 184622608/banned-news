### 有货就下手 中国代购团在Costco简直〝抢疯了〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】美式卖场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_好市多.html" target="_blank">
  好市多
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Costco.html" target="_blank">
  Costco
 </a>
 ）最近在美国上架一款Tommy Hilfiger的外套，遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国代购团.html" target="_blank">
  中国代购团
 </a>
 盯上，有目击者说，现场简直〝抢疯了〞，也有消费者将一名华人女子抢衣的画面录影上传网路。
 <br/>
 <br/>
 综合媒体报导，影片中可以看到，这名洋溢着得意笑脸的中国女子直接将所选的外套丢到地上，整个人又扑倒在整堆衣服上宣示〝主权〞。另有多名中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代购.html" target="_blank">
  代购
 </a>
 者，也是每人一车或是地上一堆的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Tommy Hilfiger外套.html" target="_blank">
  Tommy Hilfiger外套
 </a>
 ，引来周围许多外国人侧目。
 <br/>
 <br/>
 据了解，这件Tommy Hilfiger三合一外套，在美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_好市多.html" target="_blank">
  好市多
 </a>
 只要54.99美元，但在中国微信朋友圈则喊到一件800元人民币（约116.5美元），获利翻倍。
 <br/>
 <br/>
 当时现场目击者说，〝抢疯了，只要有货就下手，管他是谁，先抢再说〞。
 <br/>
 <br/>
 <b>
  相关视频：
 </b>
 <center>
 </center>
 <br/>
 为此，美国好市多部分门市针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国代购团.html" target="_blank">
  中国代购团
 </a>
 的行为，要求买Tommy Hilfiger外套的消费者，签署一份协议书，明令购买外套的顾客不能退货、禁止出口到外国，并将相关规则告知美国政府，但不同门市视情况有不同规范。
 <br/>
 <br/>
 此前有媒体报导，芝加哥寇驰（Coach）专卖店举办的〝代购专场特卖会〞上，上万件商品在短短2小时内被抢购一空，而到场的100多位代购中，有99%为华人，中国顾客的跨洋消费实力着实令举办方惊讶。
 <br/>
 <br/>
 有人说，很多中国人靠代购赚了不少。不过，上周上海浦东机场严查代购，有消息说一个航班就查出100多人，不少人被罚得血本无归，甚至跪地求饶。
 <br/>
 <br/>
 而海外中国人在超市购物的不文明行为，近来也常被爆光。去年8月，4名讲粤语的妇女在多伦多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Costco.html" target="_blank">
  Costco
 </a>
 内挑拣樱桃，把好的樱桃放在一个箱子里，一些微烂的樱桃则被她们拨到地上；同年夏天，还有两名妇女被拍到跳进西瓜箱里〝拍西瓜〞，完全不顾周围人的眼光……
 <br/>
 <br/>
 今年8月份，有网友拍到一名身穿紫色T恤的中国妇女，拿一个粉红色塑胶袋，站在Costco food court的自助饮料机前，明目张胆地装着冰块，最后包了3大袋离开；还有妇女被拍到在Costco装走免费的洋葱。
 <br/>
 <br/>
 另近期，在加拿大的好市多一名妇女偷拿大量的塑胶袋和厨房纸巾，影片被中国网友迅速转发。
 <br/>
 <br/>
