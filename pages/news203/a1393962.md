### 出访东亚四国 美国务卿：不放松对朝经济制裁
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 将于10月6日至8日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出访.html" target="_blank">
  出访
 </a>
 日本、朝鲜、韩国和中国。周三，蓬佩奥在国务院记者会上强调，不会放松对朝鲜的经济制裁。
 <br/>
 <br/>
 国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 表示，他很高兴重返亚洲，继续推进川普（特朗普）总统与金正恩在新加坡峰会上达成的无核化协议，同时强调不会放松对朝经济制裁。
 <br/>
 <br/>
 美国国务卿 Michael Pompeo：〝经济制裁将继续。这一核心条件，给予我们实现无核化的能力，并没有改变。〞
 <br/>
 <br/>
 川普总统日前表示，不想为朝鲜弃核设定时间表。蓬佩奥对此表示赞同。他还谈到，9月下旬联合国安理会上，中俄两国都曾提出，要求考虑减轻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对朝制裁.html" target="_blank">
  对朝制裁
 </a>
 。
 <br/>
 <br/>
 美国国务卿 Michael Pompeo：〝(但是)中俄都支持，维持联合国安理会决议及相关制裁。这是个全球性的承诺。〞
 <br/>
 <br/>
 专家认为，蓬佩奥本次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出访.html" target="_blank">
  出访
 </a>
 亚洲四国非常重要，尤其是在朝核问题上，美国需要继续与日韩两国协调一致。
 <br/>
 <br/>
 战略与国际研究中心韩朝主任 车维德：〝这些会谈的结果将非常重要，决定是否有第二次川金会。〞
 <br/>
 <br/>
 外交关系协会日本问题专家 Sheila A. Smith：〝当然我们在亚洲的区域盟友，在首尔和东京的，都期待与他的对话，揭示我们(在朝核问题上)的现况。我确实认为，我们在协调上有所进展，如果仔细观察的话。〞
 <br/>
 <br/>
 本周日，蓬佩奥将在平壤与金正恩再次会面。专家认为，朝鲜目前在无核化上，可能有一些表面让步，但依然不会采取实质步骤。
 <br/>
 <br/>
 战略与国际研究中心高级副总裁 Michael Green：〝如果蓬佩奥的出访，导致(朝鲜)一些有限行动，如允许检查人员进入宁边，朝鲜继续冻结(导弹)试验，我们也该采取一些相应的适度步骤，但不是宣布无核化胜利之类的。〞
 <br/>
 <br/>
 下周一，蓬佩奥还将到访北京。目前，美中之间在贸易与安全方面均关系紧张，原本预定10月中举行的外交安全对话已被取消。在朝核问题上，双方互动也令人关注。
 <br/>
 <br/>
 战略与国际研究中心韩朝主任 车维德：〝贸易上关系的确紧张。但是，我们都要能同时应对诸多问题。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
