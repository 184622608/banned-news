### 美国会通报卡瓦诺调查结果 指控者承认诬告
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 日前表示，调查发现，对最高法院大法官提名人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 （Brett Kavanaugh）的几项性侵指控都没有任何事实依据，其中一名女子已明确承认自己做出了虚假指控。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 司法委员会主席格拉斯利（Chuck Grassley）上周五（11月2日）向参议院共和党报告，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 的性侵指控进一步调查后发现，卡瓦诺的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_指控者.html" target="_blank">
  指控者
 </a>
 无一能够提供证据证实她们的指控真实。
 <br/>
 <br/>
 格拉斯利根据一份长达414页的调查报告指出，调查人员问讯了40多名证人，监察社交媒体和新闻媒体的报导并评估相关信息，对几项指控分别进行了广泛调查，最终，司法委员会和联邦调查局（FBI）得出的结论是，没有任何证据可以支撑这些指控。
 <br/>
 <br/>
 在上述接受问讯的证人中，包括指控者加州教授福特（Christine Blasey Ford）所提供的3名证人。福特称，这3人都参加了事发当天的聚会，但3人都否认了她的说法。
 <br/>
 <br/>
 上个月，格拉斯利要求FBI调查朱莉．斯沃特尼克（Julie Swetnick）指控卡瓦诺参与轮奸的案件。值得一提的是，斯沃特尼克与指控川普的色情女星丹尼尔斯（Stormy Daniels）聘请了同一名律师，都是臭名昭彰的自由派人士阿维纳蒂（Michael Avenatti）。
 <br/>
 <br/>
 报告指出，事实上，证据显示斯沃特尼克和阿维纳蒂似乎犯了伪证罪，他们密谋向司法委员后做出实质虚假的指控，并妨碍委员会调查。
 <br/>
 <br/>
 在司法委员会的问讯下，另一指控卡瓦诺性侵的女子朱迪．蒙罗-雷顿（Judy Munro-Leighton）上个月承认，她当年根本就不认识卡瓦诺。
 <br/>
 <br/>
 蒙罗-雷顿表示，匿名信并不是她写的，她只是以此作为〝策略〞，以期〝获得关注〞。
 <br/>
 <br/>
 今年9月，就在参议院成功完成了对川普提名的大法官卡瓦诺进行听证、并准备进入最后的程序——确认表决前，加州女子福特突然以一桩所谓36年前的性侵事件指控卡瓦诺，并得到民主党的大力支持。随后，又有2名自由派女子指控卡瓦诺，导致卡瓦诺的确认在最后关头陷入危机。
 <br/>
 <br/>
 该事件最终在FBI重启对卡瓦诺的背景调查后结束，卡瓦诺以50:48票赢得了参议院全院表决通过。但是，针对一系列涉嫌伪证的指控，司法委员会共和党呼吁FBI进行调查。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
