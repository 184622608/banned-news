### 驻纽约经文处将办理行动领务服务荣民
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】为免大纽约地区年长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_荣民.html" target="_blank">
  荣民
 </a>
 舟车劳顿，驻纽约台北经济文化办事处延续多年作法，与中华民国旅美老兵自强同心会合作，订本(107)年11月17日(周六)及12月1日(周六)下午2时30分至4时30分在纽约华侨文教服务中心(地址:133-32 41 Rd. Flushing, NY 11355)，由领务秘书受理〝旅居海外长期居住就养荣民验证声明书〞及〝旅居海外长期居住就养荣民(注销)报备声明书〞之文件证明申请案件，欢迎辖区内的荣民朋友多加利用此一服务。
 <br/>
 <br/>
 经文处表示，每年12月是中华民国国军退除役官兵辅导委员会受理海外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_荣民.html" target="_blank">
  荣民
 </a>
 提缴就养验证声明书的期限。凡是曾经向国内荣民服务机构完成旅居海外长期居住就养报备手续的荣民，都可以向经文处申办上述两声明书验证，以便向国内相关安养单位请领就养金，不必亲自返国办理。
 <br/>
 <br/>
 荣民可于上述时间携带：(一)有效中华民国护照正本并附影本乙份、(二)文件证明申请表、(三)〝就养荣民验证声明书〞或〝(注销)报备声明书〞、(四)最近个人生活照片乙张，到场办理上述文件之验证，所需费用每份美金15元。
 <br/>
 <br/>
 符合请领就养金之海外荣民倘无法于上述时间到场申办，仍可选择由本人亲自前往经文处办理、委托他人至经文处申办及邮寄办理等任一种方式申办上述验证，详细说明可至经文处网站(www.taipei.org)→领务(护照、签证、文书认证及各项表格下载) →文件证明→旅居海外长期居住就养荣民验证声明书文件证明申请案应注意事项)下载，经文处将于收妥声明书后予以验证并转报退辅会处理，以照顾辖区内退休荣民朋友之权益。
 <br/>
 <br/>
