### 因库克一句话 扎克伯格要求FB高管都用安卓手机
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】有消息指美国社交网站
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脸书.html" target="_blank">
  脸书
 </a>
 （Facebook）的CEO马克．扎克伯格已要求他的管理层团队只使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安卓.html" target="_blank">
  安卓
 </a>
 （Android）手机。据信，这是因为此前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果公司.html" target="_blank">
  苹果公司
 </a>
 的CEO蒂姆．库克在接受媒体采访时暗批Facebook〝拿用户私人生活做交换〞而惹恼了扎克伯格。
 <br/>
 <br/>
 据The Verge报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脸书.html" target="_blank">
  脸书
 </a>
 首席执行官马克．扎克伯格(Mark Zuckerberg)以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安卓.html" target="_blank">
  安卓
 </a>
 操作系统在全球拥有更多用户为由，下令脸书的高级管理团队只能使用安卓手机。
 <br/>
 <br/>
 不过，据指促使扎克伯格作出这一决定的真正原因，恐怕与苹果首席执行官蒂姆．库克（Tim Cook）此前曾在接受媒体采访时暗批脸书有关。
 <br/>
 <br/>
 据公开的资讯，稍早前有媒体媒体在采访库克时提问：如果类似脸书用户信息数据被剑桥分析公司滥用的事件发生在他身上，那么他会如何去做？
 <br/>
 <br/>
 当时库克自信地回答说：〝我不会面临这种状况。我们关心用户体验，我们不会拿你的私人生活作交换。〞
 <br/>
 <br/>
 这句话被外界解读为暗批脸书靠拿用户的户私人生活资讯来赚钱。
 <br/>
 <br/>
 随后，扎克伯格也在接受媒体采访中不点名地批评库克的上述说法〝太荒谬〞、〝极其肤浅〞。
 <br/>
 <br/>
 当时扎克伯格回击说：〝我们千万不要患上‘斯德哥尔摩综合征’，让那些为了向你收取更高费用而努力工作的公司来说服你，让你相信它们更关心你。〞疑似暗批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果公司.html" target="_blank">
  苹果公司
 </a>
 只是想方设法从客户那里赚得更多的钱。
 <br/>
 <br/>
 不过外界发现，脸书的区块链主管大卫．马库斯（David Marcus）和AR/VR副总裁安德鲁．博斯沃思（Andrew Bosworth）等多位高管，至今仍然使用iPhone在社交媒体推特(Twitter)上发帖。
 <br/>
 <br/>
 对此有中国大陆媒体分析称，这有可能扎克伯格的命令未得到彻底执行，但也可能是这些脸书的高管们在工作中按老板要求使用了安卓手机，但在他们个人的私生活中却仍然在使用iPhone。
 <br/>
 <br/>
 （记者欣然综合报导/责任编辑：东野）
</div>
