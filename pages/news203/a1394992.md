### 美第一夫人：我是世上最受网络霸凌的人之一
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第一夫人.html" target="_blank">
  第一夫人
 </a>
 梅拉尼娅（Melania Trump）本周出访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲.html" target="_blank">
  非洲
 </a>
 时表示，她希望反击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络霸凌.html" target="_blank">
  网络霸凌
 </a>
 ，部分原因在于，她是在网络上最受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欺负.html" target="_blank">
  欺负
 </a>
 的人之一。
 <br/>
 <br/>
 梅拉尼娅在独访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲.html" target="_blank">
  非洲
 </a>
 期间接受了美国广播公司（ABC）记者拉马斯（Tom Llamas）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_采访.html" target="_blank">
  采访
 </a>
 。拉马斯周四（10月11日）参与了ABC《早安美国》（Good Morning America）节目的制作，并描述了他采访梅拉尼娅的情况。
 <br/>
 <br/>
 当被问及为何要对付
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络霸凌.html" target="_blank">
  网络霸凌
 </a>
 （cyberbullying）时，梅拉尼娅告诉拉马斯：〝我可以说，我是世界上最受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欺负.html" target="_blank">
  欺负
 </a>
 的人——其中之一，如果你真的看到人们在说我什么。〞
 <br/>
 <br/>
 今年5月，梅拉尼娅推出一个聚焦于儿童福祉和情感健康的〝做最好〞（Be Best）项目，该项目尤其关注社交媒体、网络霸凌和在线活动对儿童的影响。
 <br/>
 <br/>
 梅拉尼娅表示，应该培养孩子在社交和情感方面的行为，以便他们长大后知道怎样处理这些问题。
 <br/>
 <br/>
 拉马斯说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第一夫人.html" target="_blank">
  第一夫人
 </a>
 告诉他，她长期受到网络骚扰，即使在她准备从事慈善活动时。
 <br/>
 <br/>
 当拉马斯问她是否盼望再做6年第一夫人时，梅拉尼娅只是表示，她很喜欢在华府度过的时光，也喜欢住在白宫。
 <br/>
 <br/>

 <br/>
 梅拉尼娅近日参观肯尼亚的内罗毕国家公园（Nairobi National Park）时，戴了一顶白色遮阳帽，批评者居然称，其风格具有非洲殖民统治的历史内涵。
 <br/>
 <br/>
 梅拉尼娅对此表示，她希望人们关注于她在干什么，而不是穿什么。
 <br/>
 <br/>
 梅拉尼娅的服饰此前也经常遭到反对者挑剔和批评，虽然这位美国第一夫人曾经当过模特，在穿着打扮上其实很在行。
 <br/>
 <br/>
 梅拉尼娅在访谈中还表示，她认为她的丈夫所做的工作令人难以置信，她希望美国人民取得成功，因此，无论川普总统做什么，她都会支持他。
 <br/>
 <br/>
 ABC采访梅拉尼娅的视频将于周五晚上对观众播放。
