### 不是男就是女 川普政府拟归正跨性别官方定义
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】 有消息指美国政府正在研议去除上一届政府对跨性别者（transgender people）的官方定义，将以人出生时的生理性别为准则来作出判定。舆论称，这是川普（特朗普）政府继推翻奥巴马政府颁布的〝跨性别厕所令〞后，进一步归正美国官方有关性别的定义，使其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_回归传统.html" target="_blank">
  回归传统
 </a>
 。
 <br/>
 <br/>
 《纽约时报》21日报导称，该媒体获得的一份美国卫生及公共服务部（HSS）的备忘录显示，目前正有数个政府部门在展开行动，将改变美国官方之前对跨性别者的定义，重新建立一个以生理性别为依归的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性别定义.html" target="_blank">
  性别定义
 </a>
 。
 <br/>
 <br/>
 这份备忘录显示，未来根据教育法修正案第九条（Title IX），将定义性别只有男性、女性，因为人出生时就已经因为生殖器官而决定性别，此点无法改变。
 <br/>
 <br/>
 据披露，川普政府提出将性别定义为男性或女性，由一个人与生俱来的生殖器性别来决定，不可改变，而任何关于一个人性别的争论都必须通过基因测试来澄清。
 <br/>
 <br/>
 上述政府部门在备忘录中明确指出，〝性别意味着一个人的男性或女性身分，是基于出生时可识别的、不可改变的生物特征〞。〝一个人的出生证明上的性别，如果没有可靠的遗传医学证据，将作为定义一个人的性别的确凿证据。〞
 <br/>

 此前，川普政府已经推翻了上一届的奥巴马政府颁布的〝跨性别厕所令〞。有舆论认为，川普政府目前的作为，是在努力归正奥巴马时期所指定的跨性别者的性别定义，重新归于〝传统的、生物性的、不可改变〞的性别定义。
 <br/>
 <br/>
 据了解，奥巴马政府颁布的〝跨性别厕所令〞规定，不论是生理上还是心理上的〝跨性别者〞 ，只要自认为自己是什么性别，就可以照这个性别进厕所或浴室，而不需要学校许可，不需要家长提出申请，也不需要医生证明等等。
 <br/>
 <br/>
 该法令从发布之日起就备受争议，尤其遭到了宗教信仰团体的强烈反对。
 <br/>
 <br/>
 批评者认为，上述法令在判定人的性别时模糊了传统的性别定义的法律概念，很大程度上采取了由个人主观选择作为衡量标准，而不是由人出生时的性别来作出判断，这使得原本很简单的男女概念的传统区分，突然变得非常复杂和随意，对教育和医疗保健在内的联邦计划造成了很大的干扰，导致了混乱和负面的政策后果。
 <br/>
 <br/>
 早在2017年7月下旬，川普政府已经明确规定，美国军队将不〝接受或允许〞跨性别者服役。对于这项政策，川普曾解释说，美军〝必须把精力集中在决定性和压倒性的胜利上〞，无暇顾及此类人群。
 <br/>
 <br/>
