### 川普庆印度排灯节 提名DC巡回上诉法官
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在白宫与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度裔.html" target="_blank">
  印度裔
 </a>
 共同庆祝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_排灯节.html" target="_blank">
  排灯节
 </a>
 ，同时提名了一名新的DC巡回上诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度裔.html" target="_blank">
  印度裔
 </a>
 同庆传统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_排灯节.html" target="_blank">
  排灯节
 </a>
 ，他亲手用蜡烛一根一根点燃灯火，仪式象征光明战胜黑暗，善良战胜邪恶。
 <br/>
 <br/>
 美国总统 川普总统：〝美国是一片拥有信仰的土地。我们真的很幸运，将这些美妙的传统编织进我们国家的锦绣生活。〞
 <br/>
 <br/>
 川普总统指出，印度是民主国家，美印两国共同开创自由、繁荣与和平。
 <br/>
 <br/>
 美国总统 川普总统：〝这是一个令人欢欣鼓舞的时刻，让我们所爱之人、邻里和社区更加紧密，这些闪亮的灯光提醒我们寻求智慧、感恩，并永远珍惜和热爱照亮我们生命的人。〞
 <br/>
 <br/>
 川普同时提名了一位印度裔美国人担任DC巡回上诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 。
 <br/>
 <br/>
 美国总统 川普总统：〝我提名Neomi Rao担任DC巡回上诉法官，原先卡瓦诺法官的职位。〞
 <br/>
 <br/>
 新唐人记者李子玉综合报导
