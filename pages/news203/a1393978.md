### 虚假证词不断揭穿 福特诚信倍受质疑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】随着美国联邦调查局（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 ）就最高法院大法官提名人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 （Brett Kavanaugh）的性侵指控启动调查，迄今已有越来越多的证据显示，指控卡瓦诺的加州教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福特.html" target="_blank">
  福特
 </a>
 （Christine Blasey Ford）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_证词.html" target="_blank">
  证词
 </a>
 前后矛盾，其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诚信.html" target="_blank">
  诚信
 </a>
 受到更深程度的质疑。
 <br/>
 <br/>
 就目前所披露的细节而言，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福特.html" target="_blank">
  福特
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 的性侵指控正在持续崩溃。
 <br/>
 <br/>
 美国政治新闻和民调数据整合网站RealClearPolitics（RCP）10月2日报导，福特表示，她之所以突然记起卡瓦诺性侵她的具体年代和季节，是由于2012年她家房屋装修时，两扇前门激起了她对当年事件的回忆。
 <br/>
 <br/>
 福特称，她于是肯定，卡瓦诺〝性侵〞她发生于1982年夏天。
 <br/>
 <br/>
 然而，记录证实，福特所说的那次装修发生于2008年，而不是2012年。而且，安装第二扇门也并非一个受过创伤的女人出于担忧不能逃走而提出的奇怪要求，而是为了能够出租部分房屋从而赚取额外收入——而且，这可能违反了当地的分区法（zoning laws）。
 <br/>
 <br/>
 福特的前男友提供的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_证词.html" target="_blank">
  证词
 </a>
 显示，身为心理学教授的福特曾利用她的专业知识帮助一位朋友准备如何通过测谎仪检测。如果确实如此，福特可能犯了伪证罪。
 <br/>
 <br/>
 不过，福特的这位朋友周二否认了上述说法。
 <br/>
 <br/>
 福特的前男友还指控，在他们分手近一年后，福特曾用他的信用卡消费了600美元，而且拒不承认。直到他威胁将付诸欺诈调查时，她才承认了这件事。
 <br/>
 <br/>
 还有，福特声称她开始不愿赴国会作证是因为她害怕飞行，并因此把卡瓦诺的听证会拖延了一周。但其前男友表示，他们曾一起去夏威夷旅游，并一起坐过飞机，而且她从未表示过对飞行感到恐惧。
 <br/>
 <br/>
 更重要的是，其前男友还披露，他认识她10年、恋爱6年，但从未听她说起过任何性侵经历，也没听她提及害怕封闭的空间。
 <br/>
 <br/>
 参议院调查人员已依据福特前男友的证词，要求福特的律师公布福特接受测谎的全部信息，以及她的治疗记录。
 <br/>
 <br/>
 但直到目前为止，福特的法律团队仍拒绝提供相关文件。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
