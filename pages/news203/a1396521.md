### 川普还将赴10场集会力挺共和党 〝美国优先〞议程势头强劲
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】目前距美国中期选举仅剩两周时间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统正积极投入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选.html" target="_blank">
  助选
 </a>
 。川普选战周二（10月23日）宣布，川普又投入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_600万.html" target="_blank">
  600万
 </a>
 美元购买竞选广告，并将在11月6日前再出席至少
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_10场.html" target="_blank">
  10场
 </a>
 竞选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集会.html" target="_blank">
  集会
 </a>
 ，全力支持各地共和党候选人。
 <br/>
 <br/>
 周二，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 2020年选战经理帕斯卡尔（Brad Parscale）声明，为领导共和党在11月6日的选举日取得胜利，川普正全力以赴，加大力量为各地共和党人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选.html" target="_blank">
  助选
 </a>
 。
 <br/>
 <br/>
 帕斯卡尔指出，川普的求胜精神正激励着全国各地的美国人投票，并使〝美国优先〞（America First）的议程保持强劲势头。
 <br/>
 <br/>
 在未来两周内，川普将参加至少
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_10场.html" target="_blank">
  10场
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集会.html" target="_blank">
  集会
 </a>
 。下周，他至少将赴威斯康星、北卡罗来纳和伊利诺伊州为共和党人助选。
 <br/>
 <br/>
 川普选战还宣布，他们从周一起已投入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_600万.html" target="_blank">
  600万
 </a>
 美元购买电视及数字广告，并另行支付300万美元供共和党全国委员会（RNC）支持各地的竞选。
 <br/>
 <br/>
 据帕斯卡尔估计，川普选战已为中期选举花费2000万美元，其中包括对候选人的个人捐款以及总统集会的旅行和活动费用。
 <br/>
 <br/>
 川普选战的高级顾问、川普总统的二儿媳拉拉‧川普（Lara Trump）也发表声明表示，川普团队正在全力以赴，以激励选民在即将到来的中期选举中支持共和党人。
 <br/>
 <br/>
 RNC主席麦克丹尼尔（Ronna McDaniel）周一在接受福克斯新闻《美国新闻室》（America's Newsroom）节目采访时表示，川普参加竞选活动的热情高于以往任何一位总统。
 <br/>
 <br/>
 周二，《华盛顿邮报》（The Washington Post）与乔治梅森大学沙尔政策与政府学院（George Mason University’s Schar School of Policy and Government）合作开展的〝华邮/沙尔学院〞民调（Washington Post/Schar School poll）公布，在全美69个争夺最为激烈的众议院选区中，民主党人的支持率为50%，共和党人的支持率为47%，选情相当胶着。
 <br/>
 <br/>
 民主党全国委员会（DNC）佩雷斯（Tom Perez）和佛蒙特州民主党参议员桑德斯（Bernie Sanders）周一都表示，他们对〝蓝波〞之说持谨慎态度。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
