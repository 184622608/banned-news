### 【禁闻】川普欲退出《中导条约》 评：剑指中共
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）日前宣布，美国计划退出冷战时期与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 签署的《中程核飞弹条约》，因为俄国违反条约规定。有评论指出，美国其实是项庄舞剑，意在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 。
 <br/>
 <br/>
 10月20号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在内华达州参加选举造势活动后，登上空军一号前告诉记者：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 违反协议，他们多年来一直违反协议。……我不知道为什么奥巴马总统不谈判或退出。我们不会让他们违反核协议，他们出去发展武器，而我们不被允许。〞
 <br/>
 <br/>
 川普所指的协议，是已故美国总统里根和苏联领导人戈尔巴乔夫于1987年签署的《中程核飞弹条约》，简称INF，双方依约要各自销毁一定数量的中程飞弹，并禁止持有、制造或试射射程介于500至5500公里的中程和中短程常规和核导弹，目的是缓解冷战时期的紧张气氛。
 <br/>
 <br/>
 美国政府指控俄罗斯在过去4年发展和部署新型巡航导弹，威胁欧洲国家以及与西方结盟的前苏联国家，已违反条约。
 <br/>
 <br/>
 川普对记者说：〝我们是那些留在协议中的人，我们尊重协议，不幸的是，俄罗斯没有履行协议。所以我们将终止协议，我们将退出。〞
 <br/>
 <br/>
 旅美时事评论员邢天行：〝这是跟他现在一贯的思维是一致的，也就是说，他要寻求一个公平的原则，同时要保护美国的利益。由于这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中导条约.html" target="_blank">
  中导条约
 </a>
 对美国是一种束缚，而美国又认为俄罗斯在违背条约，所以现在川普政府提出要退出中导条约。〞
 <br/>
 <br/>
 川普还把矛头指向没有加入INF的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 。
 <br/>
 <br/>
 他说：〝如果俄罗斯做了，如果中共做了，我们却坚守条约，那是无法接受的。〞
 <br/>
 <br/>
 《纽约时报》引用美国官员和欧洲国家驻美外交官员的话指出，美国退出INF的部分原因是中共正不受约束地大力研发和部署相关武器，并威胁到美国的安全。
 <br/>
 <br/>
 旅美时事评论员邢天行表示，美国退出中导条约，其实主要是针对中共，因为现在真正对美国构成威胁的是中共。
 <br/>
 <br/>
 邢天行：〝他是感觉到，中共利用中导条约，它没有签署这样一个漏洞，在任意的用它自己开发的武器，对美国的利益构成威胁，尤其对美国在太平洋地区这个利益构成了极大的威胁。美国其实也是在敲打俄罗斯，因为中俄宣称他们要加强战略协助。〞
 <br/>
 <br/>
 台湾大学国家发展研究所法学博士曾建元：〝这个条约长期忽略了对中国的发展
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中程导弹.html" target="_blank">
  中程导弹
 </a>
 的约束，导致过去2、30年间，中国迅速成长，不仅针对台湾海峡部署导弹，事实上也威胁了美国。〞
 <br/>
 <br/>
 有分析指出，中程核飞弹条约恐在美国退出后面临崩解命运，可能加速美、俄、中三大超级强权军备竞赛，再现冷战时期的对峙局面。
 <br/>
 <br/>
 对此，邢天行持不同看法。
 <br/>
 <br/>
 邢天行：〝我觉得他退出条约的本身，并不是说他是要宣战，也不是说他是想回到冷战时期，他实际的目的就是希望能够签署一个新的条约，就是把俄罗斯和中共双方都在条约里边，加以去限制。〞
 <br/>
 <br/>
 曾建元：〝更重要就是说，中共的武器它会外销，它会武器扩散的这个问题，我想，这是个大问题，所以应该利用这个机会去思考怎么样把中国纳入这些重要的国际的战略武器的规范当中。〞
 <br/>
 <br/>
 邢天行表示，针对中共这样一个危害世界的政权，川普正全方位追击它，退出中导条约只是川普要实现目的的一个步骤，或者说是一个开始，如果中共做出不符合国际原则和规则的行为，美国将采取一些行动去制止和限制。
 <br/>
 <br/>
 采访/常春 编辑/陈洁 后制/葛雷
