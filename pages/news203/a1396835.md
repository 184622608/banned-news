### 期中选举 众院议长：美国价值与社会主义抉择
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 临近，众议院多数党领袖麦卡锡表示，美国现在面临的问题，不是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 、民主党之间竞争的问题，而是美国价值理念与社会主义之间的抉择。
 <br/>
 <br/>
 国会众议院多数党领袖麦卡锡（Kevin McCarthy）周二在接受Breitbart News Daily采访时说〝不再是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 人与民主党人之争，而是美国人与社会主义的较量。民主党明确的在走向社会主义〞。
 <br/>
 <br/>
 2020川普竞选财务委员会成员 楼新跃：〝美国和社会主义的竞争，这是非常非常重大的一个政策宣示。表明共和党已经认识到，现在民主党的主流就是主张在美国搞社会主义的这样一个阵营。他告诉美国人民，你投民主党，就是在主张美国走社会主义道路。〞
 <br/>
 <br/>
 目前在美墨边境，数千〝大篷车难民〞正奔向美国；而在庇护地区，旧金山已经允许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 参与学区选举投票。
 <br/>
 <br/>
 2020川普竞选财务委员会成员 楼新跃：〝加州就更可笑了。最新通过的非法移民也可以去投票，不需要身份证。你谁都可以去投票，国家不就乱了嘛。按照美国宪法规定，投票权只有给合法而且是美国公民才有投票权。〞
 <br/>
 <br/>
 众院多数党领袖麦卡锡表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 后如果民主党重新掌控众议院，裴洛西将成为议长，可以预见，届时开放边境、奥巴马健保、非法移民等议程将会落实。而过去两年川普政府创造的经济繁荣将被逐步抵消。
 <br/>
 <br/>
 为争取众议院多数党地位，近日，共和党众议院全国委员会（the National Republican Congressional Committee）及两个支持总统议程的超级政治行动委员会（PAC）机构投入百万美元，在维州、佛州、乔治亚、密歇根、纽约州及北卡投放电视广告，为共和党国会众议员参选人助阵。
 <br/>
 <br/>
 本周，纽约州国会众议员卡特科（John Katko）与民主党挑战者巴尔特（Dana Balter）周三晚进行了辩论，目前在这个2016大选投票给希拉里的区域，共和党的卡特科民调领先。
 <br/>
 <br/>
 新唐人记者李兰纽约报导
