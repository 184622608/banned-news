### 川普签法案 EB-5投资移民签证延至12月7日
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）日前签署法案，将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_EB-5.html" target="_blank">
  EB-5
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资移民.html" target="_blank">
  投资移民
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_签证.html" target="_blank">
  签证
 </a>
 计划的有效期，延长到12月7号。
 <br/>
 <br/>
 被称作〝百万富翁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_签证.html" target="_blank">
  签证
 </a>
 〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_EB-5.html" target="_blank">
  EB-5
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资移民.html" target="_blank">
  投资移民
 </a>
 签证计划，是由前总统奥巴马设立，目地是吸引更多海外投资。这一计划，原本到今年9月底到期。总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 签署法案将这一计划延长到12月7号。
 <br/>
 <br/>
 EB-5签证计划每年名额为一万个，根据美国国务院的数据，2015年EB-5绿卡超过八成都发给了中国投资者，成为华人取得美国绿卡的快速通道。
 <br/>
 <br/>
 然而，申请EB-5造假和欺诈案件也随之增多，使这一签证计划引起很大争议。
 <br/>
 <br/>
 根据美国国土安全部的数据，这个计划实施五年内，被查出19起〝经证实的国家安全隐患〞。
 <br/>
 <br/>
 川普政府将这一计划延期至中期选举之后，如果更多共和党参选人在选举中胜出，这项移民签证计划有望得到改革。
 <br/>
 <br/>
 根据美国商业部的统计，这一签证计划在2012到2013财政年度创造了58亿美元的收入，创造了超过17万4千个工作岗位。
 <br/>
 <br/>
 新唐人记者安心纽约报导
</div>
