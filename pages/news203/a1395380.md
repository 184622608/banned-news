### 肯塔基人说川普：政策依民意 有诺必实行
------------------------

<div class="wysiwyg">

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在肯塔基州的集会，吸引了至少6000多名支持者。本台记者深入人群，做了独家采访，一起去看看。

 <br/>
 〝我不认为他是共和党人，我也不认为他是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 人。〞
 <br/>
 <br/>
 〝他（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ）所做的是人民想要的，不是他自己想要的。〞
 <br/>
 <br/>
 〝川普先生，代表了大多美国人的声音。〞
 <br/>
 <br/>
 〝川普讲话直截了当，真的是说出很多美国公民的心声。〞
 <br/>
 <br/>
 〝我支持他，因为他是（很久以来）首个信守诺言的总统，践行选战承诺，他也是长期以来首个政绩卓着的总统。〞
 <br/>
 <br/>
 现场民众的谈话表明，川普说到做到，以及为美国人发声的做法，深得民心。
 <br/>
 <br/>
 〝他言行一致，他说到做到。〞
 <br/>
 <br/>
 〝我们喜欢川普的态度，努力工作，我们必须努力工作使美国再次伟大，不是在那伸手站着。〞
 <br/>
 <br/>
 民众普遍表示，川普当选后，收入得到提升。
 <br/>
 <br/>
 〝我收入至少增长了五倍，自从川普当选以后。〞
 <br/>
 <br/>
 〝极大提高，我的生意好了很多。〞
 <br/>
 <br/>
 〝川普真的使美国经济发展了，像我这样的少数族裔，有了更多（就业）机会。〞
 <br/>
 <br/>
 〝从就业到经济，没有总统能把美国经济变成今天这样（好），而且川普做总统还不是很久。〞
 <br/>
 <br/>
 一些人表达了对民主党的失望，特别是他们认为民主党此前阻碍大法官卡瓦诺的提名。
 <br/>
 <br/>
 〝我不是共和党，我注册的是独立选民。但我两党都研究，但民主党变得太疯狂了。〞
 <br/>
 <br/>
 〝民主党变得更像是社会主义（政党），因此我更换党籍了。〞
 <br/>
 <br/>
 〝我想人们开始走出来了，特别是卡瓦诺听证后，很多人离开民主党，因为卡瓦诺的事情。〞
 <br/>
 <br/>
 〝我们有太多不好的政策，民主党搞的，让大家都失望。但川普当选后，他让大家都很振奋。〞
 <br/>
 <br/>
 现场，也有不少川普的年轻支持者。还有支持者创作了称赞川普的歌曲。

 <br/>
 〝川普是个好总统，我妈妈支持他，我要早一点跟着学。〞
 <br/>
 <br/>
 〝川普让我有机会给他投票，川普政府促使我有一份适合的工作，不再依赖政府补助，我真的很感激。 〞
 <br/>
 <br/>
