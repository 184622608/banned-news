### 共谍频曝光 川普酿制裁战？美方:才刚开始
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】美国司法部门日前又公布一批盗窃航空技术的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 ，并称才刚开始。多名参议员要求财政部制裁中共。同时，美国已对涉嫌盗窃美国技术的福建晋华启动禁售令。港媒指，关税战高潮已过，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 或拟展开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁战.html" target="_blank">
  制裁战
 </a>
 。
 <br/>
 <br/>
 美国联邦检方当地时间周二（10月30日）指控，10名中国江苏省的情报官员及其手下黑客，涉嫌在2010年至2015年期间入侵美国多家航空企业和法国一家与美方合作的企业，这些企业生产喷射发动机零组件。而就在这期间，中国一家国企〝开发出〞性能相彷的商用喷射发动机。
 <br/>
 <br/>
 不过，这10名被告仍在海外，目前不在美方控制之下。
 <br/>
 <br/>
 这是9月以来，美方通报第三宗涉及江苏省国家安全厅的商业间谍案件。
 <br/>
 <br/>
 9月下旬，在美国陆军预备役服役的中国公民纪超群（音译，Ji Chaoqun）被捕，其涉嫌替江苏国安听在美招募工程师和科学家。
 <br/>
 <br/>
 10月初，美国FBI在比利时诱捕的江苏国安厅官员徐延军（音译，Yanjun Xu）被引渡到美国受审，罪名是自2013年至今年3月，利用中间人窃取美国航空航天公司的商业秘密。
 <br/>
 <br/>
 2015年上半年，美方也曾对中共大规模黑客入侵提出指控。当年9月，北京和华盛顿签署协议，承诺不向对方开展经济间谍活动。但多方信息显示，中共并没有因此停止通过间谍活动和黑客攻击盗窃美国商业机密。
 <br/>
 <br/>
 《华尔街日报》今年10月30日报导，预计未来几天，美国检方还将对另一批中共指使的黑客提出指控，这些黑客涉嫌入侵美国信息技术服务提供商。此后几个月，美方还会提出几宗商业间谍案件，以彰显中共违反中美协议。
 <br/>
 <br/>
 美国司法部国家安全部门负责人表示，〝这才刚开始。〞
 <br/>
 <br/>
 报导还称，美国两党共8名参议员已致信财政部长梅努钦，敦促总统签署行政令，制裁中共窃取美国知识产权并影响美企竞争力。
 <br/>
 <br/>
 前美国政府官员称，若发现中共违反了2015年协定，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府应予以强力回应。
 <br/>
 <br/>
 除了指控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 和黑客外，美国日前还对涉嫌盗窃美国芯片技术的福建晋华集团启动了禁售令。
 <br/>
 <br/>
 港媒指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关税战.html" target="_blank">
  中美关税战
 </a>
 高潮已过，美方这一系列动作，或许是川普政府启动新战术、对中共展开一轮制裁动作的前奏。
 <br/>
 <br/>
 目前，美国已针对每年2,500亿美元的中国商品加征关税，还警告12月将开始推进对剩余2千多亿全部加税。但北京直到目前，仍然拒绝回应美方对贸易公平的要求。
 <br/>
 <br/>
 但川普多次表示，他认为美中最终将达成公平对等的贸易协议，只是中方〝现在还没有准备好〞。
 <br/>
 <br/>
 美国新闻网站 Axios 在9月24日曾引述两名匿名白宫消息人士透露，川普政府已搜集到中共非法活动的〝数以吨计〞的证据，以便向外界公开指证北京，并准备联合多部门，推出大规模的、实质性的反击行动。
 <br/>
 <br/>
 10月流出的一份深圳官方对美国的考察报告指，美国可能在半年至一年内，在高科技层面对中共实施全面封锁。
 <br/>
 <br/>
 川普政府一直将〝中国制造2025〞视为主要威胁。而美媒披露，中共还同时启动了〝中国标准2035〞计划。这意味着中共企图在主导高科技领域后，还要从新制定规则，威胁西方国家的安全和主权。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
