### 佛州选举官拒交记录 选民爆出转移选票视频
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】周五（11月9日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 裁定，赋予共和党人〝立即〞获得布劳沃德县（Broward）选举信息的权利，但该县
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选举官.html" target="_blank">
  选举官
 </a>
 拒不执行，不肯提交记录。一些选民则在社交媒体上晒出选举部门转移选票的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视频.html" target="_blank">
  视频
 </a>
 ，提供该县舞弊的证据。
 <br/>
 <br/>
 周五下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 法院紧急审理州长斯科特（Rick Scott）提交的诉讼案，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法官.html" target="_blank">
  法官
 </a>
 菲利普斯（Carol-Lisa Phillips）裁定，布劳沃德县
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_选举官.html" target="_blank">
  选举官
 </a>
 员拒绝移交共和党索要的记录，该行为违反了佛州宪法及佛州公共记录法。
 <br/>
 <br/>
 菲利普斯于是裁决，布县选举监督员斯奈普斯（Brenda Snipes）须在美东时间周五晚7点前允许共和党人审阅并复印其索要的记录。
 <br/>
 <br/>
 但是，当佛州共和党众议员盖茨（Matt Gaetz）随后赶到布县选举总部时，却被拒之门外。斯奈普斯拒绝向州官员提交选举记录。
 <br/>
 <br/>
 盖茨随即决定去法院提请动议，指控斯奈普斯藐视法院。
 <br/>
 <br/>
 盖茨还表示，虽然斯奈普斯的办公部门不允许他进入选举总部，但是他却看到民主党律师们整天进入该总部的办公室。
 <br/>
 <br/>
 盖茨说，周五他在该处没看到任何一名共和党律师。
 <br/>
 <br/>
 斯奈普斯早就是备受争议的地方选举官员，此前就已因涉嫌选举欺诈受到过14次指控。2017年，她在法庭上承认，其办公室曾允许非公民和重犯投票。
 <br/>
 <br/>
 据美国保守派博客〝Gateway Pundit〞周五报导，佛州选民诺克斯（JoAnn Knox）拍摄到一段
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视频.html" target="_blank">
  视频
 </a>
 ，内容显示，布县官员在选举夜正把大量选票转移到租用的厢式大卡车中，而且是在没有任何安保人员监督的情况下转移选票。
 <br/>
