### 中期选举在即 川普马不停蹄多州助选
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】下周二即将迎来美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在周末继续前往多州，为共和党候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选.html" target="_blank">
  助选
 </a>
 。并在西弗吉尼亚州、印第安纳州、蒙大拿州和佛罗里达州的竞选集会演讲。参加集会的民众表示，川普总统代表了人民的声音、坚守着美国的价值观。
 <br/>
 <br/>
 周六
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统先在蒙大拿州的贝尔格莱德参加了一个竞选集会。虽然天气渐冷，在停机坪举行的集会上人们热情不减。川普对支持者说，下星期二将举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 是〝我们一生中最重要的选举之一〞，将决定是否让经济继续繁荣，还是让激进的民主党人破坏美国未来。
 <br/>
 <br/>
 川普：〝这次选举将决定我们是否继续我们已经取得的非凡繁荣，或者决定我们是否让激进的民主党人控制国会，并破坏我们的经济和我们国家的未来。美国现在拥有我们历史上最好的经济。〞
 <br/>
 <br/>
 川普：〝信心正在飙升。你们看到了信心水平。〞〝 美国正在蓬勃发展。共和党通过了对工薪阶层大幅度减税法案，并很快将推出对中产阶级减税10%。〞
 <br/>
 <br/>
 当天随后，川普总统又飞到佛罗里达的彭萨科拉(Pensacola)参加另一场集会，为竞选联邦参议员的现任州长斯科特(Rick Scott)、以及竞选州长的德桑迪斯(Ron DeSantis)
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选.html" target="_blank">
  助选
 </a>
 。
 <br/>
 <br/>
 当地民众挤满了会场，表达对川普的支持。川普执政至今，创造了450万个工作机会，失业率达50年来最低，并为工薪阶层大幅度减税。
 <br/>
 <br/>
 川普总统周五在西弗吉尼亚州和印第安纳州的竞选集会也同样吸引了众多的支持者。
 <br/>
 <br/>
 退休职业女性 Ponny：〝他代表了人民的价值观、知道什么对人民最好。他不介意人们说他什么、对他做什么，他依然为我们而战、为了自由、为了边境、为了经济，还有信仰自由。〞
 <br/>
 <br/>
 印第安纳波利斯市长参选人 John Schmitz：〝他爱美国，他真的在意每个个体生活的情况。〞
 <br/>
 <br/>
 印第安纳波利斯人 Tony Adrian（民主党人）：〝没想到川普吸引了这么多的人。 一些关键的社会议题，包括移民、福利、债务和社会安全等，我觉得最起码这一点川普做的不错，他提出这些问题，我们需要讨论这些问题、需要有对话。〞
 <br/>
 <br/>
 印第安纳波利斯人退伍军人 Chris：〝我觉得认为移民可以来美国，但他们不能非法进来。不是来白吃白喝，而要努力工作、感恩并回报美国。我来这是因为我相信美国曾经伟大、现在仍然伟大、并能更伟大如果我们做正确的事。〞
 <br/>
 <br/>
 印第安纳波利斯人制造业小业主 Carlos：〝我带着孩子来的，因为觉得让他们了解政治运作很重要，知到从政的人他们的观点；了解这套系统、虽然不完美、但各司其职；尽量不随人摇摆，而是要看事实，让他们自己判断决定。 〞
 <br/>
 <br/>
 川普星期一和星期二还将在乔治亚，田纳西州，俄亥俄州，印第安纳州和密苏里州开展竞选活动。
 <br/>
 <br/>
 新唐人记者徐哲、雨薇、唐明镜、陈虎报导
 <br/>
</div>
