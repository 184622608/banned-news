### 【禁闻】川普电视受访 再批中共干预美国政治
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 10月14号接受美国哥伦比亚电视台《60分钟》节目专访。他指出，相对于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 干预2016年美国总统大选，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 干预
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国政治.html" target="_blank">
  美国政治
 </a>
 是一个〝更大的问题〞。
 <br/>
 在14号的访谈中，主持人问到，是否认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 干预了美国2016年大选。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：〝没有任何形式的勾结，没有勾结（俄罗斯），他们（俄罗斯）是干预了，但我认为中国（共）也干预了。坦白说，我认为中国（共）的问题更大。〞
 <br/>
 <br/>
 9月26号川普也曾在联合国安理会上公开表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 正企图干预美国11月中期选举。
 <br/>
 <br/>
 10月4号，副总统彭斯发表讲话，指中共正在利用可能引起意见分裂的议题，如贸易关税问题，以推动北京的政治影响力。其中包括奖赏或胁迫美国的工商企业、电影制片厂、大学、智库、学者、记者、地方、州和联邦政府官员等人。
 <br/>
 <br/>
 旅美政经分析人士秦鹏：〝俄罗斯主要是在网络上做一些见不得人的手脚。而中共是动用了各种手段，用彭斯的话讲，是全政府行动，包括对美国在华企业施压，通过华尔街以及华盛顿建制派在背后捣鬼，让中国驻美国大使馆的官员亲自出马，发动在美的华人行动，包括对民主党议员送钱、助选，甚至像《中国日报》亲自购买美国爱荷华州的报纸打广告，对美国农民煽动对川普仇恨等等。〞
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝从川普反覆提及北京干预美国内政来看，他对中共登广告影响美国中期选举的做法是相当不满的，但他实际又没有打算硬碰硬进行到底。〞
 <br/>
 <br/>
 在访谈中，川普还说，美国可能对中国征收新一轮关税。到目前为止，美国对中国进口商品加征了两轮关税，受影响的中国商品总值达2500亿美元。
 <br/>
 <br/>
 旅美政经分析人士秦鹏表示，川普在征收中国2,000亿美元产品关税时已发出声明表示，如果中共报复，将采取更多措施，包括征收新一轮价值2,670亿美元的中国产品关税。
 <br/>
 <br/>
 秦鹏：〝但是众所周知，中共再次进行了报复，对美国600亿美元产品征收关税，这相当于不认错却错上加错，所以川普要兑现其事前的声明；第二个原因，中共最近被发现干涉了美国大选，这被认为触动了美国的民主底线，所以也要采取惩罚措施，关税我想是其中的一个方式。〞
 <br/>
 <br/>
 不过，当川普被问到是否想通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 让中国经济陷入衰退时，他否定并强调说，美国是要通过谈判，达成公平的协议，和促进中共开放国内市场，正如美国对外开放市场一样。
 <br/>
 <br/>
 川普还说，虽然他跟中国领导人习近平很合得来，但他告诉习近平，不能再让中共通过贸易和其它方式每年从美国拿走5,000多亿美元。
 <br/>
 <br/>
 秦鹏：〝这就是川普在阐述他一直宣称的对等，reciprocal，也就是说美国本来对中国就有极低的关税，没有非关税壁垒，所以希望中共也这样做。〞
 <br/>
 <br/>
 不过，旅美时事评论员郑浩昌指出，中共官员理解不了川普一再宣称的〝对等〞关系。
 <br/>
 <br/>
 郑浩昌：〝川普的每一步加关税实际上都是讨价还价的筹码而已，目的只是想达成一个合理的价格。所以双方的对立是有转机的，问题要看习近平能否走出退让的一步。但是长期在党文化下生活的中共官员理解不了这点，要让他们放下戒心，坐下来认真谈判，现在还真成了个问题。〞
 <br/>
 <br/>
 白宫首席经济顾问库德洛14号接受〝福斯周日新闻〞（Fox News Sunday）访问时表示，美、中两国领袖可能11月会在阿根廷举行的G20峰会举行双边会谈。
 <br/>
 <br/>
 这一会晤被认为是解决两国贸易争端的最佳机会，不过也有分析认为，两人见面只是一个象征意义，不会为中美关系带来实质改变。
 <br/>
 <br/>
 采访/陈汉 编辑/陈洁 后制/钟元
