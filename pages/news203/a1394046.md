### 疑寄蓖麻毒素邮件给川普 FBI逮捕39岁嫌犯
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五角大楼.html" target="_blank">
  五角大楼
 </a>
 日前收到有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒邮件.html" target="_blank">
  毒邮件
 </a>
 后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特勤局.html" target="_blank">
  特勤局
 </a>
 周二（2日）披露，一封寄给总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的信件也含有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓖麻毒素.html" target="_blank">
  蓖麻毒素
 </a>
 〞，在周一(1日)被及时截获，没有送达白宫。联邦调查局（FBI）随即介入调查后，3日逮捕涉嫌犯案39岁海军退役成员艾伦。
 <br/>
 <br/>
 美国司法部盐湖城办公室表示，联邦调查局（FBI）3日在针对寄疑似内含
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓖麻毒素.html" target="_blank">
  蓖麻毒素
 </a>
 邮件给美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 一事，在犹他州洛根（Logan）一所住宅逮捕39岁海军退役成员艾伦（William Clyde Allen III）。
 <br/>
 <br/>
 发言人表示，当局预计5日对他提出指控。
 <br/>
 <br/>
 另一个执法单位消息表示，艾伦是因涉嫌寄发有问题邮件给国防部官员和总统川普而遭到调查。
 <br/>
 <br/>
 美国国防部官员3日稍早表示，经化验后，本周寄到五角大厦的可疑包裹内含可萃取蓖麻毒素（ricin）的蓖麻籽，而不是致命的蓖麻毒素。
 <br/>
 <br/>

 <br/>
 五角大厦的邮件检查设施1日至少拦截遭到当局怀疑内含蓖麻毒素的两件包裹，收件人分别是国防部长国防部长马蒂斯（Jim Mattis）和海军作战部长理查森（John Richardson）。
 <br/>
 <br/>
 蓖麻毒素是从蓖麻籽萃取，只要摄取、吸入或注射很小剂量就可致命。它比氰化物强6000倍，目前还没有解毒剂。
 <br/>
 <br/>
 据了解，在毒信事件事发的同一天，德州共和党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议员.html" target="_blank">
  参议员
 </a>
 克鲁兹（Ted Cruz）位于休斯敦的竞选办公室也收到了有毒信件，两名工作人员因接触到信封中的白色粉末而被送院救治。
 <br/>
 <br/>
 克鲁兹当前正与德州民主党参议员候选人奥罗克（Beto O’Rourke）展开激烈的竞选角逐。目前尚不清楚中毒者的伤势，而两件毒信事件是否有关联也尚不得而知。
