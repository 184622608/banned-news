### 路透:罗斯披露〝毒丸〞条款 川普下大棋全球围堵中共
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】美国商务部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗斯.html" target="_blank">
  罗斯
 </a>
 周五在华盛顿接受路透采访时透露，新签署的美墨加贸易协定中有一项〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒丸条款.html" target="_blank">
  毒丸条款
 </a>
 〞，可以起到防止中共在国际贸易中强制技术转让及行业补贴等行为的〝合法化〞的作用，而且这项条款可能会在未来美国与诸如日本和欧盟等其它国家和地区达成的贸易协定中被复制。
 <br/>
 <br/>
 据路透社10月5日报导，美国商务部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗斯.html" target="_blank">
  罗斯
 </a>
 周五接受该媒体采访时表示，在新签署的美墨加贸易协定中，一项旨在阻止与中国达成贸易协议的〝毒丸〞(poison pill)条款，可能会在未来美国与诸如日本和欧盟等其它国家和地区达成的贸易协定中被复制。
 <br/>
 <br/>
 报导称，根据这项新的贸易条款，如果美国-墨西哥-加拿大贸易协定(USMCA)中的任何一个国家与一个〝非市场经济国家〞签署贸易协定，另外两个国家可以在六个月内自由退出，并达成双边贸易协定。
 <br/>
 <br/>
 〝这是合理的，相当于一剂毒药，〞罗斯表示，该条款是〝一项堵住贸易协定中漏洞的举措〞，那些贸易协定的漏洞，使中共的不公平贸易、以不正当手段获取知识产权以及行业补贴等行为〝合法化〞。
 <br/>
 <br/>
 罗斯还表示，他预计在11月6日国会中期选举之前，美国与中共政府的贸易谈判不会有很大进展。
 <br/>
 <br/>
 当被问及该条款是否会在未来的贸易协定中被复制时，罗斯说：〝我们将拭目以待。我们在与墨西哥和加拿大达成的协定中加入了这一条款，这当然是有益的，不管我们与其他国家达成的协议中是否会包含此条款。〞
 <br/>
 <br/>
 罗斯接着又补充说，有了这一先例，在其他贸易协定中加入此条款变得更加容易。他说：〝人们会明白，这是达成协议的先决条件之一。〞
 <br/>
 <br/>
 罗斯的上述谈话引起广泛关注，有媒体将其解读为〝川普布大棋盘联手全球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_封堵中国贸易.html" target="_blank">
  封堵中国贸易
 </a>
 〞。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-7/p9068601a142148167.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-7/p9068601a142148167-ss.jpg"/>
   <br/>
   <font size="-1">
    图为美国和加拿大达成新版自由贸易协定后，美国总统唐纳德川普2018年10月1日在华盛顿特区的白宫玫瑰园举行的新闻发布会上发表讲话。(Chip Somodevilla/Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 《共同社》也报导称，美国商务部长罗斯欲在与日本或欧盟（EU）的贸易协定中，加入〝事实上阻止与不实行市场经济的国家之间缔结自由贸易协定（FTA）的条款〞。报导认为，美国政府此举旨在孤立〝不纠正不公平贸易惯例〞的中共政府。
 <br/>
 <br/>
 该报导并指称，日本政府当下正力争实现包括中国在内的总计16国参加的东亚区域全面经济伙伴关系协定（RCEP），美国政府推出的上述〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒丸条款.html" target="_blank">
  毒丸条款
 </a>
 〞可能会牵制RCEP的推进，而这个问题或将成为今后磋商中争论焦点之一。
 <br/>
 <br/>
 早在今年9月中旬，美中第4轮中阶官员在华府会未能取得任何进展而谈草草结束后，欧盟及日本贸易官员隔天就抵达华府，与美国贸易官员展开了3边论坛，并坦言这次会谈的目的就是要通过3方的合作来对抗〝实施不公平贸易的那个国家〞。
 <br/>
 <br/>
 当时，英国《金融时报》报导称，美中贸易战出现〝关键转向〞，不仅欧盟和日本都加入了美国在WTO（世贸组织）针对中共〝强迫技术转让〞提出控告，而且今年6月份的G7（七大工业国集团）会议后，川普政府与欧盟和墨西哥、加拿大相继展开了贸易战休兵的协议谈判，〝显示美国正纠集盟友合作，共同对抗中共的‘国家资本主义’〞。
 <br/>
 <br/>
 报导引述康乃尔大学教授普拉萨（Eswar Prasad）称，华府的上述举动〝让中共非常焦虑〞。
 <br/>
 <br/>
 事实上，美国贸易代表莱席尔与欧盟贸易执行委员马姆斯壮和日本经济产业大臣世耕弘成，今年5月也曾发表联合声明称，〝没有国家可透过强迫合资事业、外国股许可权制、行政审查及执照核准流程等手段，要求或施压外国公司技转给该国公司〞。马姆斯壮更早在去年底就曾经在阿根廷WTO会议期间直言，〝这（强迫企业技术转让的做法）不是秘密，我们认为中共就是罪魁祸首〞。
 <br/>
 <br/>
 （记者唐迪综合报导/责任编辑：明轩）
