### 彭斯谈太空部队：以实力促和平
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】周二，在美国华府举行的国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太空.html" target="_blank">
  太空
 </a>
 委员会会议上，美国副总统彭斯讲述了美国成立第六军种——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太空部队.html" target="_blank">
  太空部队
 </a>
 的进展。
 <br/>
 <br/>
 美国副总统彭斯：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太空.html" target="_blank">
  太空
 </a>
 也是兵家必争之地，就像陆地，空中和海洋一样。美国在太空将像在地球上一样占据主导地位。〞
 <br/>
 <br/>
 彭斯说，现在的太空和几十年前不一样了，越来越拥挤，并且充满对抗。
 <br/>
 <br/>
 美国副总统彭斯：〝多年来，从俄罗斯、中国到朝鲜、伊朗，这些国家都在寻找武器，使我们的导航和通信卫星失明失效。他们从地面发动电子攻击。最近我们的对手一直努力将新的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战争武器.html" target="_blank">
  战争武器
 </a>
 带入太空。〞
 <br/>
 <br/>
 彭斯表示，俄罗斯和中国积极在太空进行部署，将太空变成了一个战争领域。
 <br/>
 <br/>
 美国副总统彭斯：〝我们的政府已采取果断措施，加强美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事力量.html" target="_blank">
  军事力量
 </a>
 ，确保我们在太空中的重要国家利益。〞
 <br/>
 <br/>
 美国目前有五大军种，分别为陆军、空军、海军、海军陆战队和海岸防卫队。
 <br/>
 <br/>
 今年6月18日，美国总统川普下令五角大楼成立第六个军种——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太空部队.html" target="_blank">
  太空部队
 </a>
 。
 <br/>
 <br/>
 另外，川普还希望恢复美国的太空计划，重返月球，并很快将人送上火星。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
