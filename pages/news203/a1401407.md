### 美国又出重拳 进口中国汽车或课税40%
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 持续升级，美方又祭出一重拳，反击中共长期对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进口车.html" target="_blank">
  进口车
 </a>
 征收
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高额关税.html" target="_blank">
  高额关税
 </a>
 的行为。美国贸易代表莱特希泽日前发表声明说，目前中共对美国汽车的税率是40%，根据总统的指示，美方或将对进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国汽车.html" target="_blank">
  中国汽车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 做出对等处理。
 <br/>
 <br/>
 美国之音11月29日报导说，美国贸易代表莱特希泽28日发表声明说，中共对美国汽车征收极高的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，美方将从可行的手段中寻找对策。
 <br/>
 <br/>
 他在声明中说：中共对美国汽车征收关税的政策尤其令人震惊，目前对美国汽车的税率是40%，比对其他贸易伙伴的15%汽车税率高出一倍以上，也比美国对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国汽车.html" target="_blank">
  中国汽车
 </a>
 收取的27.5%关税高出50%以上。
 <br/>
 <br/>
 莱特希泽表示：〝根据总统的指示，我将权衡所有可行的工具，对汽车关税做出对等处理。〞
 <br/>
 <br/>
 他续说，中共以国家主导的侵略性的工业政策，给美国工人和制造商造成严重损失。尽管美国不断向中共提出这些问题，但是到目前为止中共尚未提出有意义的改革建议。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 至今已经持续近一年，迹象表明关税对中国工业生产造成打击，减缓了中国机器人和集成电路产量增长，关税也给中国靠出口为主的省份造成冲击，中国小企业主生存艰难。
 <br/>
 <br/>
 〝商业内幕〞称，中方正在面临失败，美国正在赢得胜利。中共在这场贸易战中要比美国失去更多，因为中国每年向美国出口价值5300亿美元的商品。而中国每年仅进口1870亿美元的美国商品。因此，中共的报复范围有限。
 <br/>
 <br/>
 当前美国已对中国共计约2500亿美元的商品加征了关税，并警告如果双方无法达成协议，将于2019年1月1日起开始把这些产品的关税提高到25%， 甚至将对中国所有进口商品征税。
 <br/>
 <br/>
 外界正密切关注，美国总统川普和中国国家主席习近平在G20峰会期间的会晤，是否能够解决两国的贸易纠纷。
 <br/>
 <br/>
 不过从前一阶段美中各种反应来看，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞要想达成协议，难度非常大。
 <br/>
 <br/>
 彭斯（Mike Pence）在不同场合炮轰中共，特别是在APEC会议上直接〝呛声〞北京，言辞相当激烈，随后贸易代表办公室更新了〝301调查〞报告，莱特希泽指出中共正在变本加厉，〝根本不想改变〞扭曲市场的贸易政策。
 <br/>
 <br/>
 即便是〝鸽派〞人物库德洛（Larry Kudlow）也不再〝温和〞，警告中共，G20是关键点。如果中共不能让美国满意，美国有可能在峰会上〝摊牌〞，甚至有可能在贸易问题上〝直接冲突〞。
 <br/>
 <br/>
 而美国商务部11月19日发布了一份〝历来最严格〞的技术出口管制先期通知，将对14项涉及国家安全和前沿科技的技术进行管制。这等于对中共的〝技术锁喉〞。
 <br/>
 <br/>
 种种迹象显示，美中双方贸易战的火药味表现出越发浓厚的气息。外界分析认为，如果〝川习会〞能够止息贸易战，甚至达到双赢，只有一条路：中国进行结构改革。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
