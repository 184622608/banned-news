### 川普斥责奥巴马政府〝无能〞 致中共肆意妄为
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周三（10日）在听取国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访华.html" target="_blank">
  访华
 </a>
 简报后表示，正是美国前政府的〝无能〞，导致中共如今恣意妄为，包括将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海.html" target="_blank">
  南海
 </a>
 军事化。
 <br/>
 <br/>
 美国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 （Mike Pompeo）周一（8日）抵达北京，和中共外长王毅和中央政治局委员杨洁篪进行了会谈。有外界注意到，蓬佩奥当面向中方传递出两个重要信号：① 美国并不反对中国发展；② 希望中方继续推动朝鲜半岛无核化进程。
 <br/>
 <br/>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周三在听取蓬佩奥访问北京的简报后，严厉批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥巴马.html" target="_blank">
  奥巴马
 </a>
 （ Barack Obama）政府〝无能〞，没有阻止中共在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海.html" target="_blank">
  南海
 </a>
 的势力扩张，导致几天前中共公然在南海公海领域挑战美国的舰艇。
 <br/>
 <br/>
 上月底（9月30日），美国海军驱逐舰〝迪凯特号〞（USS Decatur DDG-73）正在南海的南沙群岛附近行使自由航行权。
 <br/>
 <br/>
 根据美方公布的消息，〝迪凯特号〞在驶入南沙群岛的南薰礁（Gaven）和赤瓜礁（Johnson）12海里内海域时，一艘中共驱逐舰驶近，双方距离仅45码 （约41米）。迪凯特号因而采取紧急措施以避免碰撞。
 <br/>

 <br/>
 美国太平洋舰队发言人布朗（Charles Brown）在随后的一则声明中表示，〝美国海军船舰和飞机定期航行于印太地区，包括南海。正如过去几十年来一样，我们的部队将在国际法允许的任何地方继续飞行、航行和行动。〞
 <br/>
 <br/>
 中共国防部10月2日已经承认，中方曾派出军舰靠近美舰，以〝警告〞其离开。
