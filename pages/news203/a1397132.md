### 火灾重建后启用  美京中华会馆揭幕
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】周日(10月28日)上午，位于美国华府中国城的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美京中华会馆.html" target="_blank">
  美京中华会馆
 </a>
 举办了大厦重建启用典礼。中华民国驻美代表高硕泰、与华府市议员、副市长等共同出席并剪彩。
 <br/>
 <br/>
 10月28日，华府中国城
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美京中华会馆.html" target="_blank">
  美京中华会馆
 </a>
 ，迎来崭新一天。历经近4年的恢复整修，这座2015年初、因火灾而关闭的建筑，终于重新启用。
 <br/>
 <br/>
 中华民国驻美代表高硕泰与华府市议员、副市长及社区领袖，共同出席并剪彩。
 <br/>
 <br/>
 随后，来宾们一起步入装修一新的会议室。中华民国侨委会委员长吴新兴向会馆赠送了新匾额及国父孙中山铜像。
 <br/>
 <br/>
 美京中华会馆主席 李灿民：〝这太好了，我们能够把会议室重建好，21套房间重建好。这将是个很棒的建筑。期待未来在这里，可以使用很多很多年。〞
 <br/>
 <br/>
 美京中华会馆一直以来支持中华民国(台湾)。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻美代表处.html" target="_blank">
  驻美代表处
 </a>
 也是会馆长期合作伙伴。
 <br/>
 <br/>
 中华民国驻美代表 高硕泰：〝副总统彭斯最近强调台湾。他相信台湾拥抱民主价值，为全球华人展现了更好道路。现在美京中华会馆代表着美国大华府地区华人的信念和声音。〞
 <br/>
 <br/>
 美京中华会馆成立于1951年，2001年迁入目前会址，周边中国城也是市区内日益繁荣的地段。
 <br/>
 <br/>
 华府市议员 Jack Evans：〝中国城这些年真的很繁荣，是大华府地区很多人的必来之地。华人社区成为华府发展的重要部份。〞
 <br/>
 <br/>
 华府副市长 Kevin Donahue：〝很荣幸能代表市长来到这里。希望这一建筑未来很多年继续成为中国城及华府的中心。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
