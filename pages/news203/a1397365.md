### 假新闻泛滥 桑德斯为川普辩护
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】周一，白宫发言人桑德斯针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_媒体报导.html" target="_blank">
  媒体报导
 </a>
 ，用〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸弹包裹.html" target="_blank">
  炸弹包裹
 </a>
 〞案和犹太教堂枪击案抹黑川普，批评媒体不负责任。川普总统也连发推文，抨击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假新闻.html" target="_blank">
  假新闻
 </a>
 。
 <br/>
 <br/>
 周一，美国白宫发言人桑德斯批评媒体不负责任，将两起案件的责任归咎于川普总统。
 <br/>
 <br/>
 白宫发言人桑德斯：〝像你们这样的新闻机构不负责任，将‘炸弹邮包’案（归咎于总统），这不是总统发出的。不仅指责总统，还将这些令人发指的行为，归咎于其政府成员。这太过分了，这完全不负责任。〞
 <br/>
 <br/>
 上周五被捕的〝炸弹邮包〞案嫌犯谢亚克据证实，是注册民主党人，但媒体一直称他是注册共和党人。上周六犹太教堂枪杀案后，媒体继续责怪总统。
 <br/>
 <br/>
 桑德斯说，川普总统所做的事，90%的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_媒体报导.html" target="_blank">
  媒体报导
 </a>
 都是负面的。
 <br/>
 <br/>
 白宫发言人桑德斯：〝我认为，总统许多时候将国家团结到一起。我再次提醒，总统做的第一件事是谴责袭击者，而媒体做的第一件事是责怪总统。你们（不良媒体）在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家分裂.html" target="_blank">
  国家分裂
 </a>
 这件事上，有重大责任。尽管国家表现得好极了，但总统所做的一切，90%报导是负面的。〞
 <br/>
 <br/>
 川普周一也连发推文，指责媒体报导断章取义，用虚假信息误导民众。
 <br/>
 <br/>
 推文说：〝CNN和其他
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假新闻.html" target="_blank">
  假新闻
 </a>
 媒体故意不准确报导，指责我说‘新闻是全民公敌’。错了！我说的是‘假新闻（媒体）是全民公敌’，非常显著的区别。你们发布虚假信息，这不好！〞
 <br/>
 <br/>
 另一则推文说：〝我提到假新闻媒体是全民公敌，但不诚实的记者只引用了‘媒体’一词。我们伟大的民众收到了如此多假新闻，他们感到愤怒，大失所望。他们明白了，完全理解！〞
 <br/>
 <br/>
 川普表示，假新闻必须停止。
 <br/>
 <br/>
 川普推文：〝我们能和平地将各方团结起来。假新闻必须停止！〞
 <br/>
 <br/>
 新唐人记者毕心慈、张琪综合报导
</div>
