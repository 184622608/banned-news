### 美中期选举 前议长：主流媒体可能再次误判
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 还有11天。周四(10月25日)，前众院议长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金里奇.html" target="_blank">
  金里奇
 </a>
 表示，主流媒体对于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）长期以来都是负面报导。中期选举中，媒体可能再次引导人们对形势做出误判。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 临近。目前几个主要战场州的早期投票数据显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 选民数量远超民主党。而民主党所期望的〝蓝色浪潮〞并未出现。
 <br/>
 <br/>
 前爱达荷州共和党主席Steven Yates：〝我从全美各地友人中听到的是，过去几个月媒体中报导的一些不愉快事件，让很多保守派和共和党选民更可能参与投票。〞
 <br/>
 <br/>
 曾在2016年大选中协助
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的前众院议长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金里奇.html" target="_blank">
  金里奇
 </a>
 认为，长期以来，主流媒体对川普都是一面倒的负面报导，令保守派人士非常不满。中期选举中，这些媒体很可能再次对形势做出误判。
 <br/>
 <br/>
 前美国众院议长Newt Gingrich：〝我经过2016一整年，当时每个人都在说杰布‧布什会是(共和党)提名人，而希拉里会成为总统。但没有建制派的人想到，布什没有被提名，希拉里没成为总统。因为川普真是非常聪明！〞
 <br/>
 <br/>
 目前，很多分析认为，共和党很可能继续保持参议院多数，而众议院两党谁能胜出，还难以预料。
 <br/>
 <br/>
 前爱达荷州共和党主席Steven Yates：〝基本上，有100个或更少一些的(众议员)选区竞争激烈。其选区版图非常奇怪，很难预测那些区的选民多少会出来投票，而这也是最终决定因素。〞
 <br/>
 <br/>
 金里奇认为，即使共和党不能保持众议院多数，川普依然可以继续实现他的政策目标。
 <br/>
 <br/>
 前美国众院议长Newt Gingrich：〝如果就参议院来看，川普会得到所有提名的法官和官员，他会继续重塑美国政府。如果民主党会掌控众议院，他们很可能会胜出很少，并可能无法如意地在众院运作。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
