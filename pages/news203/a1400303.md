### 加州大火谁有责？美内政部炮口对准激进环保组织
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州大火.html" target="_blank">
  加州大火
 </a>
 已持续燃烧整整两周，死亡人数已升至79人，约1千人失踪，逾7千多座建筑被焚毁。这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山火.html" target="_blank">
  山火
 </a>
 的破坏力如此之大，令外界十分震惊，纷纷质疑为何加州近些年山火总是反覆发生？日前，美国内政部长在谈及加州大火的起因时表示，有部分原因应归咎于激进环保主义者。
 <br/>
 <br/>
 加州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山火.html" target="_blank">
  山火
 </a>
 从11月8日爆发后，严重摧毁了拥有近27000人的比尤特县天堂镇。美国内政部部长瑞安．辛克日前在访问遭加州山火袭击的社区后接受媒体采访时，指责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_激进环保组织.html" target="_blank">
  激进环保组织
 </a>
 对政府森林管理的阻挠，在某种程度上为发生这样的森林大火埋下了祸根。
 <br/>
 <br/>
 据路透社报导，美国内政部部长瑞安．辛克周二亲自走访了遭山火肆虐的社区后对媒体表示，激进环保组织一直试图阻止森林砍伐等管理行为，他们对政府的森林管理横加阻挠，甚至提起诉讼要求政府停止对森林进行管理，而这些阻挠是导致加利福尼亚州历史上最致命山火发生的原因之一。
 <br/>
 <br/>
 辛克在一次关于加州山火的电话会议上对媒体说，〝激进的环保组织宁愿烧毁整个森林，也不愿砍掉一棵树或者让森林变得稀疏。〞
 <br/>
 <br/>
 辛克没有具体说明他所指的〝激进环保组织〞究竟是哪些。不过他同时也承认，气温升高、历史上罕见的干旱状况以及森林中有大量的死树和垂死的树木等因素，也是此次山火大爆发的罪魁祸首。
 <br/>
 <center>

   </font>
  </a>
  <br/>
 </center>
 <br/>
 <br/>
 环保组织〝生物多样性中心〞 的项目主任兰迪．斯皮瓦克承认，从2009年到2017年期间，联邦政府涉及加利福尼亚州的一系列森林管理决定，曾遭到38起诉讼，而他所在的组织就曾经针对森林管理问题起诉过政府。
 <br/>
 <br/>
 不过，斯皮瓦克否认他们的行为对山火的发生负有责任，他强调气候变化和易发生山火的森林地区的增加才是造成这次破坏的主要原因。
 <br/>
 <br/>
 （记者竺颖综合报导）
