### 迈克尔离开美国土地 灾后重建任务繁重
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】
 <b>
  新闻周刊
 </b>
 （649）飓风
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 给美国佛州等地带去了前所未有的严重破坏。截至目前，仍有115万多户家庭没有通电，死亡人数上升到17人，并可能会继续攀升。联邦政府已提供了多项灾难援助。来看详细报导。
 <br/>
 <br/>
 美东时间10月10日下午，飓风
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈克尔.html" target="_blank">
  迈克尔
 </a>
 以四级的规模登陆美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛罗里达州.html" target="_blank">
  佛罗里达州
 </a>
 走廊地带，最高时速150英里，掀起14英尺的海浪。
 <br/>
 <br/>
 暴雨加狂风带来的伤害触目惊心，曾经充满活力的社区一片狼藉。
 <br/>
 <br/>
 洪水淹没街道，大风刮断电线，剥落屋顶。被吹倒的大树砸坏房屋，庭院里的栅栏、门板都被飓风毁坏。
 <br/>
 <br/>
 美国总统川普形容，迈克尔犹如致命子弹，杀伤力强大。
 <br/>
 <br/>
 美国总统 川普：〝有一点我们可以确切的说，就是迈克尔的速度非常快，象子弹飞过，是致命的子弹，力量非常大。风大的史上少见，我们从来没见过。〞
 <br/>
 <br/>
 从高空望去，很多房屋整个被摧毁，受灾区域犹如经历了一场战争，满目疮痍。
 <br/>
 <br/>
 墨西哥湾居民 ：〝我们认识的所有人都失去了他们的房子，所有人。也失去了工作。〞
 <br/>
 <br/>
 当居民返家时，眼前的惨景让他们相拥而泣。
 <br/>
 <br/>
 美国总统 川普：〝我们为失去生命的人祈祷，也为他们的家人祈祷。我们也心系损失财产的数千民众，很多被飓风彻底摧毁。〞
 <br/>
 <br/>
 登陆后，迈克尔减弱为热带气旋，向乔治亚州和南北卡罗莱纳州方向推进。让还未从飓风佛罗伦斯的灾害中恢复的地区再次遭遇最多9英寸的降雨。迈克尔最后进入大西洋。
 <br/>
 <br/>
 飓风过后，当局全力投入救援工作。不过，很多道路被飓风毁坏，路面断裂成碎块，凹凸不平，困难很大。救援人员动用直升机救助受困居民。
 <br/>
 <br/>
 佛州州长斯科特慰问投入救灾的国民警卫队，称赞他们英勇无私。
 <br/>
 <br/>
 佛罗里达州州长 斯科特：〝这是国民警卫队的所为，当他人处于危险之中（时所做的），他的母亲还在医院里。〞
 <br/>
 <br/>
 电力公司全力投入电力抢修，不过仍有115万多户家庭断电，弗吉尼亚38万户， 北卡37万户，佛州30万户。
 <br/>
 <br/>
 乔治亚州农业部表示，农作物，动物和基础设施都遭受了巨大损失。
 <br/>
 <br/>
 乔治亚州农业部表示，农作物，动物和基础设施都遭受了巨大损失。
 <br/>
 <br/>
 11日，川普总统接连批准重大灾难声明，允许住房和城市发展部（HUD）向受灾居民提供联邦资金和援助。流离失所的房主和低收入租房者将得到援助。
 <br/>
 <br/>
 HUD也宣布了一系列救济措施，包括立即取消抵押品赎回权，以及为居民提供抵押和家庭康复保险。
 <br/>
 <br/>
 12日，川普总统在推特宣布，下周将访问佛州和乔治亚州。
 <br/>
 <br/>
 陆续有企业开始营业，包括超市、加油站和餐馆。更多道路解除封闭，预计学生可以在15日返回学校。
 <br/>
 <br/>
 <br/>
 撰稿：梁东 剪辑：杨月
