### 迈克尔横扫佛州乔治亚 社区若战后废墟
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】观众朋友大家好，欢迎收看新唐人全球新闻。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风迈克尔.html" target="_blank">
  飓风迈克尔
 </a>
 周三袭击了美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乔治亚州.html" target="_blank">
  乔治亚州
 </a>
 ，造成至少6人死亡。目前飓风已经降为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热带风暴.html" target="_blank">
  热带风暴
 </a>
 ，继续向北移动。
 <br/>
 <br/>
 高压输电塔被吹倒，小汽车从岸边吹到了河里，卡车原地卧倒。
 <br/>
 <br/>
 屋顶被掀翻，狂风将房子里的一切都毁掉了，曾经充满活力的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_社区.html" target="_blank">
  社区
 </a>
 只剩下一片狼藉，树木倒下，道路被毁，如同战争过后的废墟。这就是四级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风迈克尔.html" target="_blank">
  飓风迈克尔
 </a>
 周三在墨西哥湾沿岸登陆后，给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乔治亚州.html" target="_blank">
  乔治亚州
 </a>
 留下的痕迹。
 <br/>
 <br/>
 佛州巴拿马市居民：〝无法形容的糟糕，如同地狱。你往窗外看，看到有居民房子飞过，看到有树飞过。〞
 <br/>
 <br/>
 周四，当居民返家时，不能相信眼前的景象，有人不禁相拥而泣。
 <br/>
 <br/>
 墨西哥湾居民：〝我们认识的所有人都失去了他们的房子，所有人。也失去了工作。〞
 <br/>
 <br/>
 佛州州长斯科特 称飓风〝绝对是个怪物〞。
 <br/>
 <br/>
 迈克尔是美国有史以来第三强的风暴，截至发稿，死亡人数已达6人。其中，佛州一名男子在家里被倒下的树压死。乔治亚州一个金属车棚被风吹起，砸进一家住宅，导致一名11岁女孩死亡。
 <br/>
 <br/>
 飓风造成佛州和乔治亚州35万人无电可用，美国总统川普称赞当地的电力公司行动迅速。
 <br/>
 <br/>
 美国总统 川普：〝很多地方已经重新开始供给电力。〞
 <br/>
 <br/>
 不过还有些地区可能要几周后才能恢复供电。
 <br/>
 <br/>
 目前，飓风迈克尔已减弱为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热带风暴.html" target="_blank">
  热带风暴
 </a>
 ，继续威胁南卡和北卡。当地面临洪水、龙卷风和强风的威胁，而这里许多地区尚未从飓风佛罗伦斯的洪水灾害中恢复。
 <br/>
 <br/>
 美国国家飓风中心表示，周四和周五，风暴将掠过北卡和弗吉尼亚州，给当地带去最多9英寸的降雨，最后进入大西洋。
 <br/>
 <br/>
 新唐人记者王羽音综合报道
