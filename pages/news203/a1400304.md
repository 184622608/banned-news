### 天使母亲急呼边境安全 川普欲诉第九巡回庭
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】观众朋友大家好，欢迎收看新唐人全球新闻。目前，中美洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 已有超过六千人抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 ，本台特约记者前往美国南部边境，采访了美国〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天使母亲.html" target="_blank">
  天使母亲
 </a>
 〞组织成员，她们呼吁加强边境安全，不要让悲剧再发生在其他美国人身上。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天使母亲.html" target="_blank">
  天使母亲
 </a>
 〞成员 Mary Ann Mendoza：〝他是我的儿子。他是亚利桑那州梅萨市的一名警官。2014年5月12日他被一个非法入境的外国人惯犯杀害。〞
 <br/>
 <br/>
 2014年玛丽安门多萨（Mary Ann Mendoza）的儿子，布兰登门多萨军士（Sgt. Brandon Mendoza）被有犯罪纪录的非法移民Raul Silva-Corona撞死，时年32岁。之后玛丽安加入〝天使母亲〞组织，这些母亲都有家人死于非法移民之手。
 <br/>
 <br/>
 他们聚集在美国南部边境，要求严格执法，阻止罪犯进入美国。
 <br/>
 <br/>
 〝天使母亲〞成员 Mary Ann Mendoza：〝我们聚集在这里不仅是为了逝去的生命，也是为了那些在非法移民罪行中幸存的人，为了不让这样的事发生在多一个美国人身上。〞
 <br/>
 <br/>
 与此同时，乘坐大篷车的中美洲人正在不断地抵达与圣地牙哥接壤的墨西哥城市蒂华纳，目前已有超过6200人。
 <br/>
 <br/>
 预计将总计有约1万名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车移民.html" target="_blank">
  大篷车移民
 </a>
 聚集到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 。
 <br/>
 <br/>
 这些人在边境等待、希望拿到美国的庇护。
 <br/>
 <br/>
 国土安全部部长 尼尔森：〝根据过往经验和报告，我们认为大部分大篷车成员不具备庇护资格。〞
 <br/>
 <br/>
 2017年纪录显示，这类寻求庇护的人中只有不到9%符合资格，也就是说10个申请人中不到1人符合庇护资格。他们来美国或者为了获得经济帮助、或者是与已经在美的家人团聚。
 <br/>
 <br/>
 国土安全部部长 尼尔森：〝我们国家南部边境正在经历史无前例的危机。危机是由不完善的联邦法律以及第九巡回法庭误判所致。这个位于旧金山的法庭抹煞了总统的判断，总统被充分授予来决定对于一个主权国家而言，什么是真正的威胁国家安全的危机。这是非常危险的判决。〞
 <br/>
 <br/>
 联邦第九巡回上诉法院法官蒂卡尔周二发布临时禁令，阻止了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统本月发布的一项政令，暂停受理非法越过美墨边境的外国人的政治庇护申请；临时禁令已经生效。
 <br/>
 <br/>
 美国总统 川普：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第九巡回庭.html" target="_blank">
  第九巡回庭
 </a>
 的禁令，让我们不那么容易（做该做的事）。我要说的是，第九巡回法庭的做法非常不公平。每一个在第九巡回庭的起诉都在攻击我们，让我们不得不上诉至最高法院，像旅行禁令，最后还是我们赢了。〞
 <br/>
 <br/>
 针对第九巡回庭的临时禁令，川普政府表示将再次上诉至最高法院。
 <br/>
 <br/>
 川普也不断在呼吁民主党人配合，尽快完善目前的移民和边境安全立法，堵住漏洞。
 <br/>
 <br/>
 〝天使母亲〞成员 Mary Ann Mendoza：〝他们说边境上的亲子分离，我想让我的美国同胞知道，永久的分离是什么模样，我再也无法拥抱我的儿子，不能告诉他我爱他，不再有机会跟他道别、提醒他小心开车，再也不能打电话给他⋯⋯他永远的离我而去。公墓是我唯一能看他的地方。〞
 <br/>
 <br/>
 新唐人记者李兰、特约记者Kimberly Hayek美国纽约、墨西哥蒂华纳连线报导
</div>
