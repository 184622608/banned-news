### 川习会涉142项细节 川普看好与习关系
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】周一，美国财长姆努钦指出，美中之间在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞上达成了较为细致的框架协议。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统也发推文，积极看待美中贸易前景。
 <br/>
 <br/>
 美国财政部长 姆努钦：〝这是我们第一次就142种不同结构性项目交换细节。习主席非常明确地向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统介绍了他们准备在这些议题上做些什么，这些问题涵盖了很多。〞
 <br/>
 <br/>
 姆努钦并强调，除了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，美国更重视解决知识产权等问题。
 <br/>
 <br/>
 美国财政部长 姆努钦：〝更重要的是保护智慧产权，消除强制企业合资，保护美国技术货币价值。〞
 <br/>
 <br/>
 周日晚，川普并发推文表示，中方同意降低对美国进口车的关税。
 <br/>
 <br/>
 中国今年5月宣布汽车进口关税从25%调降至15%，不过美中贸易战开打后，7月份对美国进口车附加了25%的报复性关税，加在一起高达40%。目前，中方同意取消25%报复性关税，并再调降原有的15%关税，但降多少还是未知。
 <br/>
 <br/>
 周一，川普又连发推文。
 <br/>
 <br/>
 除了称赞美中关系在阿根廷〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞后获得显著改善，中方将大量购买美国农产品，使美国农民受益。
 <br/>
 <br/>
 川普并强调他与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的〝特殊关系〞，将为贸易，乃至朝鲜问题的解决产生积极影响。
 <br/>
 <br/>
 最后，川普还希望有朝一日，能与习近平和俄罗斯总统普京，探讨解决〝军备竞赛〞问题。
 <br/>
 <br/>
 金融经济学家彼得・狄克森（Peter Dixo）表示，如果贸易谈判帮助中国顺应自由社会价值，中国将得到真正的进步。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
