### 美国务院官员：将为全世界宗教自由发声
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】周四，美国〝国际宗教自由事务大使〞布朗巴克发表了题为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界宗教自由日.html" target="_blank">
  世界宗教自由日
 </a>
 和美国宗教自由政策〞的讲话，表示美国要为全世界所有人的信仰自由发声。
 <br/>
 <br/>
 10月27日是美国通过《1998年国际宗教自由法案》20周年纪念日。为了推动宗教自由，美国今年早些时候将这一天定为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界宗教自由日.html" target="_blank">
  世界宗教自由日
 </a>
 〞。
 <br/>
 <br/>
 在周四的讲话中，美国〝国际宗教自由事务大使〞布朗巴克强调，宗教自由是天赋人权，关乎人的尊严。
 <br/>
 <br/>
 美国国际宗教自由事务大使 布朗巴克：〝美国的基本信念是，每个人是有尊严的，有权做出自己的选择。这是神赋予的权利，任何政府都无权干涉。〞
 <br/>
 <br/>
 他认为，会有越来越多的国家支持宗教自由。因为这样有利经济发展和社会安全。
 <br/>
 <br/>
 美国国际宗教自由事务大使 布朗巴克：〝人们有权选择自己的信仰，但是在新疆却不能这样。在中国其他地方也无法实现，基督徒、法轮功、佛教徒都没有信仰自由。〞
 <br/>
 <br/>
 今年7月美国在华府主办了第一届〝促进宗教自由部长级大会〞。布朗巴克表示，明年将继续举办。美国将与各国合作，举办地区性的宗教自由论坛。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
