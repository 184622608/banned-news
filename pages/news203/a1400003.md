### APEC峰会公报〝难产〞 川习会蒙阴影？
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】亚太经合会APEC领导人非正式会议周日落幕。由于中美之间的分歧，峰会首次没有达成〝领袖共同宣言〞。这也令外界担忧G20峰会上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 谈的前景。
 <br/>
 <br/>
 APEC成立25年来，首次没有达成联合公报。主办国巴布亚新几内亚的总理表示，分歧主要来自美中〝两大经济体〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 一名高级官员向《华尔街日报》透露，争议主要围绕一句话：〝我们反对贸易保护主义，包括所有不公平贸易行为〞，美国等20个APEC成员都同意在公报中加入这句话，唯独中共反对。
 <br/>
 <br/>
 谢田：这个〝Unfair Trade〞不公平贸易，是川普总统在过去大概半年之内，多次批评中共贸易政策的时候所用的词汇。中共对这个非常敏感。中共在反对这句话的时候，实际上有点不打自招，就是承认了，它在从事很多不公平贸易。
 <br/>
 <br/>
 在为期2天的峰会期间，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 措辞严厉，批评中共的不公平贸易政策、一带一路债务陷阱等做法。他要求中方做出改变。
 <br/>
 <br/>
 彭斯：（中共）从美国身上获得利益太多年了，这样的日子已经结束了。美国不会改变做法，直到中方改变其行为方式。
 <br/>
 <br/>
 北京也用强硬措辞回击美国。专家认为，北京的强硬姿态有2种考量。
 <br/>
 <br/>
 谢田：中共现在采取这种强硬的立场，主要原因是它在对内宣传的时候，要保持它有一个在国际上的强硬姿态，第二确实是对G20峰会摆出一些姿态，美中双方在互相探测底牌。
 <br/>
 <br/>
 彭斯会后告诉媒体，他和习近平私下有2次短暂交谈。彭斯表示：美国希望与中国构建更加良好的关系，但中方必须做出改变，习近平回应：我认为双方间的对话是很重要的。
 <br/>
 <br/>
 再有十多天，川普和习近平就将在阿根廷G20峰会期间会谈。
 <br/>
 <br/>
 上周，北京终于应美国要求，书面提交了142项贸易改革清单。但川普表示，中方仍漏掉了4或5个重要项目。彭博社披露，其中包括〝中国制造2025〞产业政策等。美国要求放弃，但中方没有回应。
 <br/>
 <br/>
 谢田：他（川普）就是要迫使他（习近平），要放弃中共对经济的控制。现在看来，习近平显然是在极大的压力之下，试图有走出这一步的迹象，但是畏畏缩缩又不敢走。
 <br/>
 <br/>
 据《南华早报》披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易.html" target="_blank">
  美中贸易
 </a>
 官员，将于G20峰会开幕前，在阿根廷会面谈判，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 最后铺路。
 <br/>
 <br/>
 美国商务部长罗斯表示，双方最多达成框架协议，美国将按计划，明年初把对华2000亿产品的关税，从10%调升到25%。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
