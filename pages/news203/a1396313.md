### 贸战又一颗子弹上膛 美拟降〝汇率操纵〞门槛
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】美国财政部长姆钦（Steven Mnuchin）21日表示，对改变美国判断汇率操纵的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_指标.html" target="_blank">
  指标
 </a>
 持开放态度。标准一旦修改，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 对华发动的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，将有机会将中共正式列为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率操纵国.html" target="_blank">
  汇率操纵国
 </a>
 。
 <br/>
 <br/>
 由于目前美国对一个经济体标签为〝滙率操纵〞的标准门槛极高，美财政部在日前发表的半年度报告，再次没有将中共列为滙率操纵国，避免了升高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 。
 <br/>
 <br/>
 但这份长达34页的报告批判中共，凸显美国对巨额贸易逆差的关注，且对中共不公开汇率干预作法表示〝深感失望〞。
 <br/>
 <br/>
 但如果美国修改〝滙率操纵〞的认定标准，将大大增加美国将中共认定为滙率操纵国的可能性。
 <br/>
 <br/>
 彭博资讯21日报导，姆钦可能考虑利用1988年的一项贸易法案，以货币操纵的广义定义，来界定贸易伙伴是否操纵汇率，即使被贴上这一标签的国家并不符合2015年法律中的具体标准。
 <br/>
 <br/>
 另一个方式是调整一个国家是否进行竞争性货币贬值的界定标准。
 <br/>
 <br/>
 美国是否会给贸易伙伴贴上〝不正当操纵滙率〞的贴签，取决于3项
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_指标.html" target="_blank">
  指标
 </a>
 ：
 <br/>
 <br/>
 1、该经济体对美国贸易盈余超过一年200亿美元；2、该经济体经常项目盈余占GDP比重至少3%；3、该经济体持续单边干预滙率，通过滙率干预买入的外汇超过GDP的2%。
 <br/>
 <br/>
 上述3项标准的框架由美国国会提供，但具体门槛由财政部自行斟酌。
 <br/>
 <br/>
 姆钦21日在耶路撒冷受访时表示：〝我们可能检讨是否改变判断标准。我们一直在评估这些事。〞
 <br/>
 <br/>
 是否将中共列为滙率操纵国，是目前两国经贸关系中最为敏感的关键议题之一。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 竞选时就曾表示，要宣布中共为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率操纵国.html" target="_blank">
  汇率操纵国
 </a>
 ，此次如果认定标准发生变化，将可能改变两国贸易的整体态势，贸易摩擦可能会进一步加剧。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：赵云）
</div>
