### 为孤立中共 川普特意在美墨加协定中加了一条
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】日前，美国与墨西哥、加拿大的新贸易协议敲定。外界分析，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）政府逐个解决与盟友的贸易争端，目标是合围北京。而美墨加贸易协定中新加入的一个条款，更加佐证了这一观点。
 <br/>
 <br/>
 英国路透社10月2日消息，新达成的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨加贸易协议.html" target="_blank">
  美墨加贸易协议
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_USMCA.html" target="_blank">
  USMCA
 </a>
 ）中有一项条款规定，如果其中任一成员国与〝非市场化〞国家达成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自由贸易协议.html" target="_blank">
  自由贸易协议
 </a>
 ，则另外两个成员国可以在六个月后退出，并建立自己的双边贸易协定。
 <br/>
 <br/>
 根据这项条款，美墨加协议的成员国如果要进行此类磋商，必须提前三个月通知其他成员国。
 <br/>
 <br/>
 此条款中提到的〝非市场化〞国家，明显剑指北京。
 <br/>
 <br/>
 报导指，该条款曾在加拿大引发争议。这一规定符合美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孤立中共.html" target="_blank">
  孤立中共
 </a>
 的意愿，因为它能有效防止中共将墨西哥和加拿大作为〝后门〞，将中国商品免关税销往美国。
 <br/>
 <br/>
 报导还引述分析指，该条款令中共与加拿大和墨西哥达成自由贸易协定的希望落空。对墨西哥和加拿大而言，不大可能冒着失去美墨加贸易协议的风险，去和中共签署任何贸易协议。
 <br/>
 <br/>
 报导指，如果在美国与欧盟和日本的自贸谈判中，也加入这个条款，会导致中共在全球贸易系统中被孤立。
 <br/>
 <br/>
 目前日欧自贸协议已经达成，美欧自贸谈判正在进行，美日自贸谈判也即将展开。外界认为，谈判的成功基本不容质疑，只是各方妥协多少以及时间长短的问题。
 <br/>
 <br/>
 有分析认为，川普政府的目标是联合欧、日，组建全球最大自贸区，联手施压北京，迫使其调整在贸易政策、政府补贴及盗窃知识产权方面的不公平做法，实现市场化，并遵守国际规则。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
