### 川普助选马不停蹄 造访西部多个州
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的临近，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_助选.html" target="_blank">
  助选
 </a>
 活动也越来越频繁。周五，川普造访邻近墨西哥的亚利桑那州，与选民会面。川普在演説中，再次批评民主党的移民政策。
 <br/>
 <br/>
 周五晚间的集会在亚利桑那州第三大城市梅萨举行，到场民众热情迎接
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的到来。
 <br/>
 <br/>
 川普会后也在推特上发布短视频，显示大批民众在会场外围，赶来聼他的演説。
 <br/>
 <br/>
 亚利桑那州与墨西哥相邻，川普重点谈到大篷车难民潮并批评民主党的移民政策。
 <br/>
 <br/>
 美国总统 川普：〝我们在这里説话的时候，民主党正在公开邀请数百万的非法移民，让他们来破坏我们的法律、进犯我们的边境，并击溃我们的国家。〞
 <br/>
 <br/>
 川普也再次强调，不管对手如何阻拦，也要把边境墻修起来。
 <br/>
 <br/>
 美国总统 川普：〝他们（民主党）不想让我们修边境墻，尽管他们知道这是正确的。他们会拼死阻拦我们，不让我们把墻建起来。但我们还是开始修了，我们将完成建墙。〞
 <br/>
 <br/>
 亚利桑那州被认为是这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的关键州之一。民主党希望能在这里取得突破。共和党参议员弗雷克此前宣布退休，空出的位置将由两位女性候选人，共和党的麦莎利与民主党的西内玛角逐。
 <br/>
 <br/>
 川普在集会上，力挺麦莎利，呼吁民众出来投票。
 <br/>
 <br/>
 当地媒体报道，目前中期选举的注册选民，已经超过上次同一时期的两倍。报道分析，川普的影响力，可能是促成这一现象的重要原因之一。
 <br/>
 <br/>
 川普随后马不停蹄赶往内华达州，参加另一场集会。下周一将再度造访德州，为克鲁兹的参议员竞选活动站台。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
