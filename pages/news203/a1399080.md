### 声乐大赛高手云集 知名钢琴家巫漪丽称赞
------------------------

<div class="wysiwyg">
 【
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新唐人.html" target="_blank">
  新唐人
 </a>
 2018年11月12日讯】由新唐人主办的〝第七届全世界华人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美声.html" target="_blank">
  美声
 </a>
 唱法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_声乐大赛.html" target="_blank">
  声乐大赛
 </a>
 〞，11月10号在纽约落下帷幕。共有20名选手获得大赛奖项。他们表示通过参赛，收益良多。国际知名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_钢琴家.html" target="_blank">
  钢琴家
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巫漪丽.html" target="_blank">
  巫漪丽
 </a>
 称赞大赛是一个很好的平台。
 <br/>
 <br/>
 国际知名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_钢琴家.html" target="_blank">
  钢琴家
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巫漪丽.html" target="_blank">
  巫漪丽
 </a>
 ，也是中国第一代钢琴家、中国钢琴启蒙人之一，这次为新加坡参赛选手担当伴奏，观看了大赛的整个过程，她赞誉大赛为选手们提供了很好的平台。
 <br/>
 <br/>
 国际知名钢琴家巫漪丽：〝大家互相观摩，总是可以得益的。我觉得办的很不错，各方面都很有成效。〞
 <br/>
 <br/>
 大赛中获奖的各位选手也表示，非常感谢大赛给予他们难得的学习和交流机会。
 <br/>
 <br/>
 大赛金奖得主Simeng Wu:〝一个鼓励吧，鼓励更上一层楼，努力做好自己，努力唱好歌，用歌声感动更多的人。选手之间互相鼓励非常重要，有一个很好的友善的气氛。〞
 <br/>
 <br/>
 大赛银奖得主王佩颖：〝对华人在世界舞台上演出，我觉得是一个很好的平台。可以让大家来参赛，来唱自己中文的歌曲。可以让老外也看到，我们的歌曲，我们的风格，真的像歌剧一样，是不一样韵味的美。〞
 <br/>
 <br/>
 大赛银奖得主万雪峰：〝我们选手之间会交流心得，就会分享我们唱歌一些经验和经历，我也和别人分享我的东西，这样就成为一个很好的学习过程。〞
 <br/>
 <br/>
 大赛铜奖得主蔡孟融：〝这次选手每个人的程度都非常非常的好，我很荣幸和大家一起唱歌，一起分享和交流，唱歌和音乐的感觉。〞
 <br/>
 <br/>
 大赛优秀奖得主李兆钧：〝所以我就特别感谢能有这次机会，让喜欢古典音乐的朋友，有这个舞台，能参加这个比赛。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新唐人.html" target="_blank">
  新唐人
 </a>
