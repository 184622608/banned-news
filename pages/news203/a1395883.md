### 白宫高官：中共未回应美方要求 贸易谈判中断
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】中共外宣仍在热炒11月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 可能终止贸易战。而美方早已公开表态，川习会的前提是中方有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重大让步.html" target="_blank">
  重大让步
 </a>
 〞。不过，白宫官员日前透露，中方并未回应美方的要求，贸易谈判又回到中断状态。
 <br/>
 <br/>
 17日，白宫首席经济顾问库德洛（Larry Kudlow）在接受福克斯（FOX）新闻采访时，再次谴责中共的不公贸易行为，并透露目前中方尚未回应美方在贸易谈判方面的要求。
 <br/>
 <br/>
 对于川普（Donald Trump）总统及北京领导人习近平是否会在11月底的G20峰会上举行会晤，库德洛仅表示〝有可能〞。
 <br/>
 <br/>
 随后，川普在福克斯商业新闻表示，他不认为中方准备好重启谈判了。他说，〝中方想达成协议，我告诉他们，你们还没有做好准备。〞
 <br/>
 <br/>
 17日，美商务部长罗斯（Wilbur Ross）对美国消费者协会与商业频道（CNBC）证实了库德洛的消息。他表示，〝我不知道能否将之（目前局势）称为仍在持续中的僵局〞， 〝现在我们（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易谈判.html" target="_blank">
  中美贸易谈判
 </a>
 ）似乎处于中断状态〞。
 <br/>
 <br/>
 此前美媒报导，白宫温和派和北京都有团队在准备11月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 ，但美方开出的条件是中方必须有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重大让步.html" target="_blank">
  重大让步
 </a>
 〞。
 <br/>
 <br/>
 中共一些外宣媒体摘录美媒消息的一部分，称美方只要取得一个〝小胜利〞，就会同意叫停贸易战。
 <br/>
 <br/>
 但川普近期则一直表示〝中方还没有做好（妥协的）准备〞，同时继续其〝极限施压〞。
 <br/>
 <br/>
 外界分析，川普要求中美贸易实现真正的公平对等，这等于逼迫中共改变经济体制，从而动摇中共统治的根基，因此北京让步的可能性已不大。北京之所以同意11月底谈判，不过是寄希望于共和党在中期选举落败，川普会做出某种让步。
 <br/>
 <br/>
 10月初，美国副总统彭斯（Mike Pence）发表演讲，全面谴责中共在各个领域对美国的侵蚀和伤害。评论人士纷纷表示，中美对抗已不再局限与经贸领域，新冷战时代即将到来。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
