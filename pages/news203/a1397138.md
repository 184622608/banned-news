### 川普伊州助选 数万民众到场支持
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周六在伊利诺伊州参加的助选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集会.html" target="_blank">
  集会
 </a>
 ，吸引数万民众参加。很多人表示支持川普的政策，希望他在2020年成功连任。
 <br/>
 <br/>
 为表达对总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的支持，很多民众早早赶到在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集会.html" target="_blank">
  集会
 </a>
 现场外，排起长队。
 <br/>
 <br/>
 进入十月份以来，川普马不停蹄已赶赴10多场竞选集会，帮助共和党参选人。阐述他上任以来在就业、经济、对外贸易方面取得的成绩。
 <br/>
 <br/>
 现场观众不时以掌声和欢呼声，对川普的政治观点和决策表示赞同。
 <br/>
 <br/>
 教育工作者 Maria Moore：〝我认为川普是比希拉里・克林顿更好的人选。我赞同他的政治观点，移民系统非常扭曲，我们需要建造边境墙，这样对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 经济复兴有益。〞
 <br/>
 <br/>
 核电工程师 Eric Cermik：〝他出台有利商业的政策，成功减少部分商业法规，推动了美国经济向前发展。我关心的很多议题他都站在我认为对的一边。〞
 <br/>
 <br/>
 注册护士 Timmy Foltez：〝他对我们的经济有很大好处，他说到做到。他为人诚实，我知道有些媒体不这么认为。他带来很多工作机会，他保护我们，他一心为了美国，把美国利益放在第一位。〞
 <br/>
 <br/>
 有民众认为川普带回了美国的传统理念和价值。
 <br/>
 <br/>
 退休机长 Mark：〝他代表了美国的传统价值，这个国家在过去十年间向左偏的太远，我们处于即将失去宪法赋予权利的危险之中。〞
 <br/>
 <br/>
 医疗业者 Sherian Kanpp：〝他把神的概念带回美国人的生活，我们国家现在需要的就是神的护佑。〞
 <br/>
 <br/>
 神职人员 Beth Swartzendruber：〝他对美国人有一颗真心，我非常自豪，他能让一些持不同观点的人尊重法律，我们可以再次拥有真理和正义，还有边境和平。〞
 <br/>
 <br/>
 集会民众还表示为有川普这样的总统感到自豪，希望他在2020年成功连任。
 <br/>
 <br/>
 集会民众 Gibson Westerman：〝说实话，我们需要这样的领导人，告诉我们需要知道的和想要知道的事。〞
 <br/>
 <br/>
 密苏里州选民 Fred Wilferth ：〝我们一直在走下坡路，需要往回走，他是可以带我们往上走的人。〞
 <br/>
 <br/>
 集会民众 Lily Kanpp：〝他是一个了不起的人，他在帮助美国再次强大。〞
 <br/>
 <br/>
 退休矿工 Jack Kash：〝我们非常荣幸有他当我们的总统。〞
 <br/>
 <br/>
 集会民众 Connie Madison：〝我希望在2020年，他能再当选我们的总统。我为他感到非常自豪。〞
 <br/>
 <br/>
 神职人员 Beth Swartzendruber：〝这是南伊利诺州快乐的一天，神佑川普，神佑美国。〞
 <br/>
 <br/>
 距离中期选举还有一周，川普在未来几天，至少还会连续参加十场竞选集会。
 <br/>
 <br/>
 新唐人记者苏月、陈虎伊州墨菲斯伯勒报导
</div>
