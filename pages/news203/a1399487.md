### 快讯：金正恩视察新型高科技武器测试
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 官媒周五（11月16日）报导，朝鲜领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 近日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视察.html" target="_blank">
  视察
 </a>
 了一种新型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高科技.html" target="_blank">
  高科技
 </a>
 武器的测试工作。这是一年来金正恩首次公开检阅朝鲜的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武器测试.html" target="_blank">
  武器测试
 </a>
 现场。
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 中央广播电台（The Korean Central Broadcasting Station）当地时间周五报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视察.html" target="_blank">
  视察
 </a>
 了位于某国防研究所内的一个试验场，检查了一种并未指明的新战术武器的测试。
 <br/>
 <br/>
 报导称，在该国执政党——朝鲜劳动党的领导下，朝鲜已长期从事这项〝最先进武器〞的研发工作。该武器的出现意义重大，能够〝完全保护我们的领土〞，并〝显著提高了我人民军的战斗力〞。
 <br/>
 <br/>
 报导宣称该〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高科技.html" target="_blank">
  高科技
 </a>
 战术武器〞的测试非常成功，达到了所有高级、强大的设计指标的要求。
 <br/>
 <br/>
 不过，该报导始终没有具体说明，朝鲜测试的到底是何武器。
 <br/>
 <br/>
 这是自去年11月以来金正恩首次前往武器试验场进行实地视察。去年11月，他观测了Hwasong-15洲际弹道导弹的测试。
 <br/>
 <br/>
 今年6月，美国总统川普在新加坡会晤了金正恩，此后，朝鲜在兑现无核化承诺的进程中虽然时进时退，但朝鲜官媒公然报导金正恩检阅
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武器测试.html" target="_blank">
  武器测试
 </a>
 尚属首次。
 <br/>
 <br/>
 11月2日，朝鲜外交部突然发表声明警告，称如果美国不解除对它的经济制裁，它就将恢复发展核武器。但美国随后以恢复小型的美韩联合军演予以回应。
 <br/>
 <br/>
 11月7日，川普总统表示，他〝不急于〞解决朝鲜问题，因为对朝制裁仍在持续，而朝鲜则已停止发射导弹和火箭，朝鲜所扣押的美国人质也已返回家园。
 <br/>
 <br/>
 不过川普同时表示，他会在明年早些时候会晤金正恩。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野)
 <br/>
 <br/>
