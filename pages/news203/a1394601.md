### 川普：不会解雇罗森斯坦 谈得很好
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周一（10月8日）表示，他不打算解雇司法部副部长罗德．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗森斯坦.html" target="_blank">
  罗森斯坦
 </a>
 （Rod Rosenstein），并表示他与罗森斯坦谈得很好。
 <br/>
 <br/>
 周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 明确表示他没有开除
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗森斯坦.html" target="_blank">
  罗森斯坦
 </a>
 的打算，从而结束了几周来外界对罗森斯坦前途的揣测。
 <br/>
 <br/>
 自从《纽约时报》（The New York Times）9月21日报导，罗森斯坦曾于2017年春天推动对川普秘密录音、妄图抓住把柄后再援引宪法第
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_25修正案.html" target="_blank">
  25修正案
 </a>
 发动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内阁政变.html" target="_blank">
  内阁政变
 </a>
 逼川普下台后，罗森斯坦的仕途前景似乎已经很不看好，共和党议员也已敦促传讯罗森斯坦到国会作证。
 <br/>
 <br/>
 不过，《纽时》的报导发表后，四面楚歌的罗森斯坦曾求见川普，寻求机会为自己辩解。川普因当时正参加在纽约举行的联合国峰会、随后又观看最高法院大法官提名人卡瓦诺（Brett Kavanaugh）的听证会，所以一再推迟与罗森斯坦的会面。
 <br/>
 <br/>
 不过期间川普曾表示，他与罗森斯坦在电话中进行了沟通，他暂时没有解雇罗森斯坦的打算。
 <br/>
 <br/>
 周一，当准备前往佛州奥兰多（Orlando）的川普登机时，有记者问及他是否想开除罗森斯坦。
 <br/>
 <br/>
 〝不，我不会。〞川普明确表示。
 <br/>
 <br/>
 川普是前去参加一个有美国及国际警察首领们参加的国际会议的，并准备在会上发言。罗森斯坦作为司法部副部长，与他一同前往。
 <br/>
 <br/>
 白宫新闻发言人吉德利（Hogan Gidley）表示，川普与罗森斯坦交谈了45分钟，内容涵盖执法人员、边境安全、芝加哥的暴力犯罪和司法部的一般工作。白宫幕僚长凯利（John Kelly）和首席司法部副部长助理奥卡拉汉（Edward O’Callaghan）也参与了会谈。
 <br/>
 <br/>
 川普说，他和罗森斯坦一起飞往奥兰多，媒体很想知道〝你们谈了些什么〞，但他只会说，〝我们谈得很好〞。
 <br/>
 <br/>
 川普近来对罗森斯坦的态度有很大转变，不再像过去那样激烈批评他。罗森斯坦负责监督特别检察官穆勒（Robert Mueller）的通俄门调查，而川普一直谴责通俄门调查是一场〝猎巫〞的政治迫害。
 <br/>
 <br/>
 飞机起飞前，川普告诉在场记者：〝我以前不了解罗德，但我已了解他了。我跟他相处得很好。〞
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
