### 美中贸易战暂时缓解 中共不改再升关税
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】G20落幕，美中领导人之间的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞也顺利完成，双方达成框架性协议。但专家认为，这只是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 到一定程度的缓解，如果
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在90天限期没有实质改变，一样要面对美国更大强度的施压。
 <br/>
 <br/>
 时政评论人士 哥伦比亚大学政治学博士 李天笑：昨天（12月1日）的会谈啊，实际上是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 利用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，向中方施压，和善意的两方面的结果。这个压力啊，从今年7月开始，就通过贸易战的方式，加大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 惩罚的力度。迫使习近平意识到问题的严重性。特别是现在中国经济，现在也发生了实质性的压力和冲击。包括GDP、外贸领域、贸易链的转移，还有失业人口，等等一系列的问题出现，使得习近平这边压力特别大。昨天（12月1日）这个协议，从美国的声明来看，它只是一个缓解，这个缓解是有限期的。换句话说，贸易关税的这把剑还悬在头上，也就是说，90天的限期一旦过了以后，如果中方没有进行实质性的、结构性的改变的话，那么还是要增加到25%的关税。
 <br/>
 <br/>
 同时，美国密苏里州共和党联邦参议员布仑特（Sen. Roy Blunt）表示，贸易战对美国大豆农民有影响，但他仍认为川普对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的贸易施压是对的。
 <br/>
 <br/>
 密苏里州共和党联邦参议员 布仑特：总统敦促中方达成更好和更公平的贸易协议，是正确的目标。
 <br/>
 <br/>
 新唐人纽约记者站采访报导
