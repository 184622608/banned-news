<p>
 <strong>
  退出中共邪党！
 </strong>
 <br/>
 我们在年轻时，因受中共邪党的骗，积极追求所谓的“进步”，稀里糊涂的宣誓加入了中共邪党。
 <br/>
 但是这些年，亲眼目睹中共邪党对反腐败大学生的6.4大屠杀，对受冤屈访民的残忍打压，对善良无辜的法轮功学员残酷迫害，使我们认清了中共邪党的恶魔真面目！
 <br/>
 我们认为中共邪党不亡，中华民族和炎黄子孙就永远没有好日子过！
 <br/>
 我们在网上郑重声明：我们要正式退出中共邪党！与中共邪党一刀两段，从此清清白白做人，永永远远唾弃中共邪党！
 <br/>

<p>
 <strong>
  中国共产党少年先锋队を脱出
 </strong>
 <br/>
 初中二年级，对中共抱有幻想的我，欲加入共青团。然而当时的班主任收到了礼物，并要求另一位同学入团，结果我未能成功入团。后借着赴日留学的机会看到了日本的活力，呼吸到了自由的空气。并接触了在大陆根本无法想像的YouTube和谷歌。在我留学的高中里，我遇到了一位来自高雄的同学，我们聊了很多，发现台湾人并不像大陆媒体说的那么激进，那么无脑。后来我又在网路上看到了许多大陆禁闻，后又通过《
 <a href="http://www.epochtimes.com/gb/tag/%E4%B9%9D%E8%AF%84.html">
  九评
 </a>
 》系列，辛先生演讲系列了解到邪党的本质，本人在此宣布：退出我在小学一年级时加入的少先队组织，并以后坚决不入邪党。
