### 80岁航天专家熊辉丰冤狱4年身体极度虚弱　
------------------------

<p>
 【大纪元2018年10月23日讯】“我身上是清白、干净、透明的，任何污水都沾不上。” 现年80岁的中国
 <a href="http://www.epochtimes.com/gb/tag/%E8%88%AA%E5%A4%A9%E4%B8%93%E5%AE%B6.html">
  航天专家
 </a>
 <a href="http://www.epochtimes.com/gb/tag/%E7%86%8A%E8%BE%89%E4%B8%B0.html">
  熊辉丰
 </a>
 ，四年来，一直不接受中共对他的非法冤判，坚持申诉。
</p>
<p class="p3">
 <span class="s1">
  <a href="http://www.epochtimes.com/gb/tag/%E7%86%8A%E8%BE%89%E4%B8%B0.html">
   熊辉丰
  </a>
  先生，因不放弃对法轮功的信仰，2014年8月被中共警察抄家、绑架，2015年12月被非法判刑7年半，2018年，在滨海监狱中，他的身体已极度虚弱。
 </span>
</p>
<p class="p2">
 <span class="s1">
  熊辉丰的女儿到监狱看望父亲。每每看到女儿时，熊辉丰都老泪纵横，令人心碎。
 </span>
</p>
<p class="p2">
 <span class="s1">
  明慧网报导，此前，熊老先生在南开区看守所被非法关押期间，牙齿全部坏掉，早已无法正常饮食，每天只能吃泡水的饼干，身体非常消瘦，很难起床活动，过大年之前还有过“感冒发烧”情况发生。
 </span>
</p>
<p class="p2">
 <span class="s1">
  2018年3月29日，律师陪同熊辉丰的家属去天津市监狱管理局，办理相关手续要求释放熊老先生回家。因是司法日当天，狱政科的人都去参加电话会议。律师和熊辉丰的家属只得到信访办反映情况，递交材料。接待的是梁姓工作人员。
 </span>
</p>
<p class="p2">
 <span class="s1">
  首先，律师介绍了熊辉丰以往的经历与现实的情况，指出把无罪之人非法关押并转化思想是邪恶专制的一套表现，接着律师又从历史层面与政治层面讲述了对法轮功的迫害完全是江泽民、周永康出于个人邪恶心理实施的。对方表示，准备向有关部门转达。但至今无果。
 </span>
</p>
<p class="p2">
 <span class="s1">
  熊辉丰，退休前曾任航天部八三五八研究所副所长、研究员、中国宇航学会理事，是享受国家特殊津贴的专家。
 </span>
</p>
<p class="p2">
 <span class="s1">
  熊辉丰因在航天科学研究工作中的杰出成就，获得1985年度“国家科学技术进步奖”三等奖、1993年度“光华科技基金奖”二等奖，以及若干航天部的二等功、三等功等殊荣，在航天部及研究所享有极高声誉。
 </span>
</p>
<p class="p2">
 此外，熊辉丰还曾向“中国青少年发展基金会”捐款，至少资助了22个贫困地区的孩子完成学业，但他从未向他人提及过此事，直到家人收拾其书柜时发现25封来自河南、湖北的受助学生、学生家长、希望小学校方及上级机关的信件，才得知该事。
</p>

 <a href="http://i.epochtimes.com/assets/uploads/2018/10/1601081409532192-600x450.jpg">

 </a>

 </figcaption><br/>

<p class="p1">
 <span class="s1">
  2015年8月14日，南开法院非法庭审熊辉丰。辩护律师为熊辉丰做无罪辩护，当庭指出“该案侦查取证违法，事实不清。同时认为法轮功是一种宗教，没有对社会造成任何危害，该案适用法律（刑法三百条）错误，故应依法宣告被告人无罪。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  法院公诉人和法官无言以对，此次非法庭审不了了之。
 </span>
</p>
<p class="p1">
 <span class="s1">
  2015年11月25日，南开法院推延通知律师，对熊辉丰第二次庭审，并强行下达判决书。
 </span>
</p>
<p class="p1">
 在熊辉丰被非法关押期间，老伴刘元杰女士含冤离世。
</p>
<p class="p2">
 熊辉丰的老伴刘元杰女士 ，是八三五八研究所的高级工程师，多次获得航天部的二等功、三等功，为国家的飞航导弹领域做出重要贡献。
</p>
<p class="p2">
 <span class="s1">
  在中共对法轮功的迫害中，刘元杰由于多年来反复被迫害，加上警察不断恐吓威胁，特别是2014年对年近八旬的熊辉丰先生的再次绑架，对刘元杰的身心造成严重打击。她从此日渐消瘦，精神状态越来越差，于2015年3月3日凄然离世。
 </span>
</p>
<p class="p2">
 <span class="s1">
  已是耄耋之年的熊辉丰老人，即使遭受诬陷，身卧牢笼，依然信念笃定，秉持
  <a href="http://www.minghui.org/mh/glossary.html#1">
   <span class="s2">
    法轮大法
   </span>
  </a>
  真善忍的理念，做好人。
 </span>
</p>
<p class="p2">
 <span class="s1">
  他于2016年在狱中还向天津高等法院递交了申诉信，并表示将继续上诉。#
 </span>
</p>
