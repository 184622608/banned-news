### G20前中美双方频繁出招 川习会前途未卜
------------------------

<p>
 【大纪元2018年11月23日讯】（大纪元记者骆亚、周慧心采访报导）
 <a href="http://www.epochtimes.com/gb/tag/%E5%B7%9D%E4%B9%A0%E4%BC%9A.html">
  川习会
 </a>
 已经进入倒计时，在美国多方面持续施压下，中共内部出现对美贸易战政策的批评声音。另外，有消息指习近平将在28日访问西班牙，而本应为川习会铺路的刘鹤转而访问德国。川习会是否添变数引发各界高度关注。
</p>
<h4 class="p1">
 <span class="s1">
  G20前美方一连串施压
 </span>
</h4>
<p class="p1">
 <span class="s1">
  英国BBC在11月21日报导，美国总统川普（特朗普）的经济顾问委员会主席凯文·哈塞特（Kevin Hassett）暗示，可以研究将中国驱逐出世界贸易组织（WTO），因为中共作为WTO的一员“行为不端”，而WTO也让美国失望。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他还表示，川普在国际贸易方面的强硬战略正在发挥作用，非常希望川习在G20峰会有一个富有成效的会谈。
 </span>
</p>
<p class="p1">
 <span class="s1">
  11月20日，美国贸易代表办公室（USTR）发布更新版的“301调查报告”，贸易代表莱特希泽透过声明表示，中方并没有从根本上改变其不公平、不合理，且扭曲市场的举措。
 </span>
</p>
<p class="p1">
 <span class="s1">
  11月17日，美国副总统彭斯在亚太经合组织（APEC）首席执行长（CEO）峰会上的发言中，点名批评中共贸易政策，称中共设置的贸易壁垒史无前例，包括关税、盗用知识产权、强迫技术转让和不公平的补贴。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说，美国已经对中共贸易政策采取了决定性措施，对2,500亿美元中国产品征收关税，并可能将征税规模扩大一倍。彭斯说，美国不会改变这一应对方式，直到中方作出改变。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  对美政策 中共内部现分歧
 </span>
</h4>
<p class="p1">
 <span class="s1">
  在此背景之下，中共前首席贸易代表龙永图以及另一位财经高官刘世锦，最近公开批评中国应对
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BE%8E%E8%B4%B8%E6%98%93%E6%88%98.html">
   中美贸易战
  </a>
  的政策，呼吁中国实行高水平的对外开放。
 </span>
</p>
<p class="p1">
 <span class="s1">
  据香港《南华早报》报导，已经退休的龙永图在“财新—工银国际2018国际投资论坛”上表示，中国在
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BE%8E%E8%B4%B8%E6%98%93%E6%88%98.html">
   中美贸易战
  </a>
  中，对美国包括大豆在内的农产品加征报复性关税，是个策略失误。
 </span>
</p>
<p class="p1">
 <span class="s1">
  另据路透社报导，中国发展研究基金会副理事长、中国人民银行货币政策委员会委员刘世锦，在同一论坛上表示，中国若建设高标准市场经济，实行高水平对外开放，在应对中美贸易摩擦、国内稳预期、提信心等方面，都可以变被动为主动。他还表示，中国不能在市场经济改革方面走回头路。
 </span>
</p>
<p class="p1">
 对此，美国南卡罗莱纳大学艾肯商学院终身教授谢田分析认为，龙永图可能是另一派的代言人，他敢于批评中共现政权的政策，也是中共内斗激化的一个迹象。
</p>
<p class="p1">
 <span class="s1">
  对于刘世锦的言论，谢田对大纪元记者表示，这也许是中共在探风向。“美国副总统彭斯，还有美国高官都表示，除非中共改变方针政策，不然美国不惜把中共踢出WTO，或者另起炉灶，重建一个新的WTO，在这种强大的攻势之下，我觉得中共出现这种说法，这个苗头应该是比较可能的。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  但是他认为，中共不会真正妥协，因为如果
 </span>
 <span class="s1">
  按照川普的要求，
 </span>
 <span class="s1">
  就意味着中共失去对中国经济、社会的控制，意味着，失去它的专制权力。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他举例说，中共在11月12日提交的142项书面回复中，包括三部分：一部分中共愿意改进，一部分不能做到，还有一部分可以谈判。“实际上这个已经表明中共的立场了，这个可能是中共的底线。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“川普已经明确表示不会接受这个，要不就全盘接受，要不就是贸易战全面升级，甚至断绝中美之间的贸易，对中共进行经济上最后的打击。所以我想11月底可能是习近平最后一个机会，来决定他到底要不要放弃中共。”
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  <a href="http://www.epochtimes.com/gb/tag/%E5%B7%9D%E4%B9%A0%E4%BC%9A.html">
   川习会
  </a>
  再添变数？
 </span>
</h4>
<p class="p1">
 <span class="s1">
  距离在阿根廷举行的G20峰会还有8天，各界都在关注川习会是否会让贸易战“停火”，亦或是愈演愈烈。多数学者不看好这次的会面。
 </span>
</p>
<p class="p1">
 <span class="s1">
  旅居美国的宪政学者王天成表示，中美双方都希望结束贸易战，尤其是贸易战给中共带来很大的冲击。
 </span>
 <span class="s1">
  中共承受压力很大，有做出妥协的动力，贸易战对中国经济的冲击、对股市的冲击都是很大的，中共政权本身也有危机感，总是怕出问题。
 </span>
</p>
<p class="p1">
 <span class="s1">
  不过，他认为中共从来不会考虑老百姓和国家的利益，只有对其政权有利的情况下，它才会妥协。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“中美贸易战现在可以说已经探底了，该出的牌已经出的不少了，但G20川习会前景还不明朗。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  联合新闻网引述西班牙国家报（El Pais）报导，习近平将于11月28日访问西班牙。至于访问日程，没有具体说法，仅表示习近平会和西班牙国王、首相会面，并拜访国会以及在皇宫进行晚宴。
 </span>
</p>
<p class="p1">
 <span class="s1">
  报导称，由于习近平出访西班牙的时间正好在G20峰会登场前，就算习近平利用时差赶场，也相当紧凑，让外界好奇川习会能否如期登场。
 </span>
</p>
<p class="p1">
 <span class="s1">
  另一值得注意的是，原本被媒体报导将直飞阿根廷为川习会“暖身”的中国国务院副总理刘鹤，将于25日至28日访问德国。中国外交部20日公布刘鹤访德行程，刘鹤是否将前往阿根廷，外界也存在疑问。
 </span>
</p>
<p>
 《香港经济日报》分析认为，在川习会举行前夕，中共高层先后“闪电”访问欧洲国家，显然不是偶然；尽管外界暂无法得知中方的真实战略考量，但客观上，中方此际访欧、展示中国与欧洲的关系，无疑可作为川习会时，中方向美方要价的筹码。
</p>
<p class="p2">
 <span class="s1">
  原上海某大学理学教授“草祭”则发推文认为，中美幕僚未谈，川习如何会？就算川习会，也顶多是在G20上一起吃个饭，不会有任何结果。他认为，G20后，中美面临全面摊牌，贸易战变冷战！
 </span>
</p>
