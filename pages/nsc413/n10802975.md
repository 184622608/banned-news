### 股市再现跌势 刘鹤救市为何不管用
------------------------

<p>
 【大纪元2018年10月23日讯】（大纪元记者骆亚、周慧心采访报导）上周五刘鹤和财金系统三大巨头齐声向市场喊话，
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%9B%BD%E8%82%A1%E5%B8%82.html">
  中国股市
 </a>
 在周一短暂回升后，周二再次下挫，
 <a href="http://www.epochtimes.com/gb/tag/%E6%B2%AA%E6%8C%87.html">
  沪指
 </a>
 再度失守2,600点关口。学者认为，官员喊话救市只能是暂时缓解，中国经济内忧外患，投资人已经失去信心。
</p>
<p class="p1">
 <span class="s1">
  著名独立中国经济学人巩胜利在10月22日接受大纪元采访时表示，“光喊是没用的，要采取一些措施，比如中国现在企业的税收是全球最高的之一，还有让老百姓享受实实在在的实惠，这很关键。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“周一股市，稍微可能涨一下，明天后天接着往下跌，这就是最好的回答。股市被全球称为‘经济发展晴雨表’，它都会反映到股市，你看今天话喊完了，没有行动的话，股市会继续掉，人民币会继续贬值。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  果然，股市在周一短暂回升后，周二（10月23日），
  <a href="http://www.epochtimes.com/gb/tag/a%E8%82%A1.html">
   A股
  </a>
  又全线大跌，三大指数都重挫超2%，上证综指收盘报2,595.08点，跌幅2.26%；深证成指收盘报7,574.99点，跌幅2.24%。
 </span>
</p>
<p class="p1">
 <span class="s1">
  《苹果日报》10月22日刊登卢峯的评论文章，也认为这次救市顶多只能buy time，纾缓一下问题，药效很快就过去。
 </span>
</p>
<p class="p1">
 <span class="s1">
  文章写说，“事实上内地经济从超高速增长转向中速甚至中低速增长是个相当痛苦的过程，盈利追不上债务负担，利率上升更如百上加斤，加上中美贸易战打击经济前景及民众信心，企业债务、地方债务问题只有更纠结，更难以清理。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  10月19日，刘鹤在接受《人民日报》、中共央视、新华社联合采访时表示，贸易战引起的担忧被夸大了。他说：“心理影响大于实际影响。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  巩胜利则认为，中国经济现状是内忧加外患，“2008年，中国货币发行只有30万亿到40万亿之间，现在已经突破到180多万亿，今年年底就要突破200万亿。但是这些都不够，因为美元还在继续加息。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“中国货币现在发行量已经最大了，经济的GDP数字跑到了最小，中美贸易战2,000亿才刚刚开打，明年1月1日开始这2,000亿（商品的关税）可能会从10%增加到25%。现在中国数据几乎全部下走。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  对于中共官媒“信心比黄金重要”的宣传，巩胜利表示，现在中共央视每天的新闻都说美国怎么输、中国怎么样赢，“它用经济数据给人们上了一场活生生的政治课。它的GDP达到21世纪以来的最低点。人民币贬值差0.024就到7了，所有人看到数据都知道是怎么一个结果，你天天说好有用吗？”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“全中国的媒体都被封锁了，只能讲正面的话，像我刚才讲的数据它都不让说。”
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  姓党的股市没有未来
 </span>
</h4>
<p class="p1">
 <span class="s1">
  巩胜利表示，中共试图用党来控制股市，他举例说，中证监主席刘士余曾在公开场合强调，在上市公司治理架构中突出党的建设，无论是国有控股上市公司，还是非国有控股上市公司，都要加强党的建设，不能有丝毫的含糊。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “党本来就是市场经济多余的一个政治概念，党派出的代表，不仅要管经济、股市，还要进入香港上市企业股市改革的班子里去，而且要挂帅。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  “做生意讲政治，不讲经济、不讲市场行情，这样的股市有未来吗？所以不管是（股权）质押也好，转换也好，上市公司越是党的领导，股市一定是灾难，因为全世界讲政治的股市，都没有未来。”巩胜利说：“何况股市里的党组织，这个成本，谁来掏钱呀？”
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  民企：官员讲话是“绝唱”
 </span>
</h4>
<p class="p1">
 <span class="s1">
  刘鹤在10月19日的访问中把金融机构、官员不支持民企、中小企说成是政治问题，“那些为了所谓‘个人安全’、不支持民营企业发展的行为，在政治取向上存在很大问题，必须坚决予以纠正。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  此外，中国银行保险监督管理委员会主席郭树清、中国人民银行行长易纲、中国证券监督管理委员会主席刘士余，在同一天也齐声透过媒体向金融市场喊话，宣示会落实多项措施支持民营企业。
 </span>
</p>
<p class="p1">
 <span class="s1">
  对此，上海企业家胡力任表示：“这些事情我认为是绝唱。因为人心散了，现在所有的民营企业家都在一个政治高风险的状态下运作，大家认为很有可能回到文革时代的那套做法。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他对大纪元记者说：“那民营企业的老板将来都是众矢之的，将来都是被打击的对象。所以说现在跑的跑、躲得躲。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他介绍说，以前民营企业的老板都是很高调的，现在已经没有人敢出来讲话了，因为说不定哪天有句话讲错了，这个人就成了被抓捕的对象。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “随时随地公安会找到你、税务会找到你，上面稍微有一些风吹草动，一下子就把你逮起来了，所以非常难非常难。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“我身边一些上市公司的老板他们搞股权质押，主要是想套现让自己的小孩带走往外跑，逐步逐步地套现之后，他就离开中国。因为你留在中国，你就变成一块肥肉，每个人都可以来宰割你。”#
 </span>
</p>
