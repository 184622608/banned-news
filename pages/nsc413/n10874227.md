 我20年前生活在一个小山村，那边住着一家四口，一对夫妻一双儿女，大女儿十几岁，小儿子八九岁，人都挺好，是好人哪。有一天突然大抓捕，抓捕炼法轮功的，把他们一家四口都抓去了。后来，这一家人就这样凭空消失了。他爷爷奶奶经常来村里挨家挨户的找啊，来询问。那么多年了，还在找啊！拄著拐棍，满头白发，痛苦的表情。。。现在我想起来都揪心哪！
 <br/>
 这一家人生不见人，死不见尸的，就在这红旗下消失的，你上哪去说理呀？
 <br/>
 我现在才转过弯来，这一家人肯定是被活摘了，或者被关进集中营了。
 <br/>
 我年轻时跟着恶党欺压百姓，抢人家的东西，收保护费。我们小山村的生活本来就苦，还要收这个费那个费的，也不给票据。一月挣300元工资。现在想想真是后悔哪。
 <br/>
 现在，我和朋友们看一些海外资讯，也要拉上窗帘，戴上耳机。满大街的监控，员警和治安大妈。在微信上说句话被说我心里阴暗，天天派人看着我。这国家不就完了吗？！
 <br/>
 那些被关押在监狱的好人（法轮功学员），吃不饱，受的罪比平民在监狱多百倍哪！
 <br/>
 我要为那些冤死的人，为那些受欺压的人发一发声，帮助这些还在监狱里受迫害的人们。
 <br/>
 我们现在人人都要站出来呐喊，都要声张正义，不能缩在壳里装王八不吱声，不要总以为迫害离我们很远，这样下去，这不早晚要轮到我们吗？
 <br/>
 现在我郑重声明，坚决退出共产党组织，我要让全世界都知道，这共产党太坏了，比纳粹还坏呀！
