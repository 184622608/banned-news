### 冤狱12载又遭诬判 原广州大学副教授上诉
------------------------

<p>
 【大纪元2018年11月02日讯】广州市白云区
 <a href="http://www.minghui.org/mh/glossary.html#1">
  <span class="s2">
   <a href="http://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
    法轮功
   </a>
  </span>
 </a>
 学员、原
 <a href="http://www.epochtimes.com/gb/tag/%E5%B9%BF%E5%B7%9E%E5%A4%A7%E5%AD%A6%E5%89%AF%E6%95%99%E6%8E%88.html">
  广州大学副教授
 </a>
 <a href="http://www.epochtimes.com/gb/tag/%E7%8E%8B%E5%AE%B6%E8%8A%B3.html">
  王家芳
 </a>
 ，今年8月24日被广州市荔湾区法院
 <a href="http://www.epochtimes.com/gb/tag/%E8%AF%AC%E5%88%A4.html">
  诬判
 </a>
 三年半。2017年10月27日，
 <a href="http://www.epochtimes.com/gb/tag/%E7%8E%8B%E5%AE%B6%E8%8A%B3.html">
  王家芳
 </a>
 在流花湖公园游玩时，遭越秀区六榕派出所警察绑架、构陷。
</p>
<p class="p2">
 <span class="s1">
  明慧网报导，王家芳不服一审判决，向广州市中级法院提出上诉。她认为，一审过程中公检法人员严重违法，自己没有罪，要求无罪释放。
 </span>
</p>
<h4 class="p5">
 <span class="s1">
  搜查程序违法
 </span>
</h4>
<p class="p5">
 <span class="s1">
  2017年10月27日，在没有《立案决定书》的情况下，越秀区公安分局警察到王家芳住处抄家。当时，王家芳坚决抵制，但警察强行抄家。
 </span>
</p>
<p class="p5">
 <span class="s1">
  王家芳多次向越秀区公安分局、越秀区检察院和荔湾区法院投诉，但没有得到答复。
 </span>
</p>
<h4 class="p5">
 <span class="s1">
  拘留程序违法
 </span>
</h4>
<p class="p5">
 <span class="s1">
  《刑事诉讼法》第八十三条规定：拘留后二十四小时内，应该通知被拘留人的家属。越秀区公安分局不但不依法通知王家芳的家属，反而出具一份《呈请暂不通知家属报告书》。
 </span>
</p>
<h4 class="p5">
 <span class="s1">
  定案依据非法
 </span>
</h4>
<p class="p5">
 <span class="s1">
  定案用的《鉴定意见书》，案卷里根本没有。案卷里只有一份广州市公安局反邪教支队的《认定意见书》。
 </span>
</p>
<p class="p5">
 <span class="s1">
  《认定意见书》不属于《刑事诉讼法》第四十八条规定的八种证据类型，出具鉴定意见，对鉴定机构以及鉴定人的鉴定资质、鉴定程序和鉴定标准有明确要求，而《认定意见书》不满足上述要求。
 </span>
</p>
<p class="p5">
 <span class="s1">
  因此，广州市公安局越秀区分局将《认定意见书》归入鉴定意见明显错误。而这份《认定意见书》也没有向王家芳宣读，因为侦查人员宣称的宣读日期在看守所的“提讯表”里没有记录，侦查人员明显在伪造文书。
 </span>
</p>

<h4 class="p5">
 <span class="s1">
  适用法律错误
 </span>
</h4>
<p class="p5">
 <span class="s1">
  王家芳坚持对荔湾区检察院对自己“利用邪教组织破坏法律实施”的指控不成立，
  <a href="http://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
   法轮功
  </a>
  教人向善，自己也没有破坏哪条法律的实施。
 </span>
</p>
<p class="p5">
 <span class="s1">
  2018年4月9日非法庭审时，王家芳要求检察员李凡出示证明法轮功与邪教有关联的法律文件。李凡搪塞说：到时候就拿出来了。但到现在李凡也没有拿出来。
 </span>
</p>
<p class="p5">
 <span class="s1">
  王家芳要求将抄走的法轮功书籍全部归还给她。
 </span>
</p>
<p class="p5">
 <span class="s1">
  她说：邪不胜正是真理！历史决不会允许凌驾于法律之上的任何人、任何事长期存在下去，每个人都要为自己的行为负责，为自己的将来负责，为子孙后代负责。
 </span>
</p>
<p class="p5">
 <span class="s1">
  她坚信，当历史翻过这一页时，她无愧于自己，因为她坚持了正义，守住了良知，选择了善！
 </span>
</p>
<p class="p5">
 <span class="s1">
  今年45岁的王家芳曾是广州大学数学系副教授，撰写过“模糊集类上的Fuzzy测度”、“Fuzzy集的基数问题”等学术论文，是一位学生喜欢的好老师。
 </span>
</p>
<p class="p5">
 <span class="s1">
  王家芳的儿子小时候患有先天性心肌类疾病，多方求医无法医治，修炼法轮功后，她儿子得到奇迹般康复。
 </span>
</p>
<p class="p5">
 <span class="s1">
  中共

 </span>
</p>
