### 川习会间孟晚舟被捕 美方释放何信号？
------------------------

<p>
 【大纪元2018年12月07日讯】（大纪元记者骆亚、周慧心采访报导）正当川普和习近平在阿根廷会面时，加拿大政府应美国要求在温哥华国际机场逮捕了中国
 <a href="http://www.epochtimes.com/gb/tag/%E5%8D%8E%E4%B8%BA.html">
  华为
 </a>
 公司副董事长兼首席财务官（CFO）
 <a href="http://www.epochtimes.com/gb/tag/%E5%AD%9F%E6%99%9A%E8%88%9F.html">
  孟晚舟
 </a>
 ，有评论认为，美国此时逮捕孟晚舟是在释放信号。
</p>
<p class="p1">
 <span class="s1">
  <a href="http://www.epochtimes.com/gb/tag/%E5%AD%9F%E6%99%9A%E8%88%9F.html">
   孟晚舟
  </a>
  被逮捕的时间正是
  <a href="http://www.epochtimes.com/gb/tag/%E5%B7%9D%E4%B9%A0%E4%BC%9A.html">
   川习会
  </a>
  、中美达成休兵90天的时机，由于美方要求引渡，这也让
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BE%8E%E8%B4%B8%E6%98%93%E6%88%98.html">
   中美贸易战
  </a>
  更显扑朔迷离，中美贸易休战是否会增添新变数？
 </span>
</p>
<p class="p1">
 <span class="s1">
  哥伦比亚大学政治学博士王军涛在接受大纪元记者采访时表示，这个事件与贸易战是相对独立的事件，但是它给中共传递了两个非常强烈的信号。
 </span>
</p>
<p class="p1">
 <span class="s1">
  第一，美国不会因为贸易上的利益，而对其它方面放松，“你（中共）不要以为你拿一万二千亿的订单来，或者想落实贸易条款，就能让美国放松你对美国国家安全方面的威胁，让美国在台海局势、南海这些方面放松防范和压力，美国实际上是要给出这样一个信号。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  第二，另一个信息是告诉中共政府，贸易战只是暂时休战，现在是要看中共能不能够落实在
  <a href="http://www.epochtimes.com/gb/tag/%E5%B7%9D%E4%B9%A0%E4%BC%9A.html">
   川习会
  </a>
  中习近平答应要做的事情。
 </span>
</p>
<p class="p1">
 <span class="s1">
  王军涛分析说，作为
  <a href="http://www.epochtimes.com/gb/tag/%E5%8D%8E%E4%B8%BA.html">
   华为
  </a>
  的官员，孟晚舟可能直接涉入对伊朗出售禁运品，或者因为她在华为公司供职，她知情，所以美国需要她去接受调查和作证。
 </span>
</p>
<p class="p1">
 <span class="s1">
  路透社2013年曾报导，孟晚舟曾是香港天通科技有限公司（Skycom Tech Co. Ltd.）的董事会成员，该公司试图向伊朗最大的移动电话运营商出售被美国禁运的惠普（HP）计算机设备。天通科技的销售提案至少有13页被标记为“华为机密”，并有着华为公司的标识。
 </span>
</p>
<p class="p1">
 <span class="s1">
  时事评论员陈思敏认为，华为的财务体系，最清楚的人莫过于孟晚舟。因为她从1993年加入华为，到2011年坐上集团财务长的大位，都是财务这条线贯穿。可以说孟晚舟一手打理了华为庞大集团的财经系统、主导整合国际财务体系，以及相关业务的内控内稽。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  美国开启引渡模式
 </span>
</h4>
<p class="p1">
 <span class="s1">
  加拿大《环球邮报》（The Globe and Mail）报导，加国司法部发言人Ian McLeod说：“孟晚舟已于12月1日在温哥华被捕，美国政府要求引渡她，保释聆讯将在星期五进行。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  今年10月9日，一名中共情报官员在比利时被捕，并被引渡到美国接受间谍指控。而孟晚舟也可能被引渡到美国，有网民认为，这是美国开启引渡模式。
 </span>
</p>
<p class="p1">
 <span class="s1">
  旅美学者吴祚来在其推特上表示，抓孟晚舟是一个重大而致命的信号：
 </span>
 <span class="s1">
  “对中共特殊企业的打击，不仅有核心技术层面，市场控制，还会精准到个人，在世界任何一个角落，都可能落网。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  “中共开始把一些私企纳入自己人系列。”
 </span>
 <span class="s1">
  “而一些特殊企业则负有使命与任务，它们得到中共超级支持，也为这些企业精英埋下隐患。”
 </span>
 <span class="s1">
  “川大爷也开始精准打击敌对势力。”
 </span>
</p>

 </span>
</p>
<h4 class="p1">
 <span class="s1">
  中方未在第一时间抗议
 </span>
</h4>
<p class="p1">
 <span class="s1">
  12月1日，正当川普和习近平正在阿根廷布宜诺斯艾利斯共进晚餐之时，孟晚舟遭逮捕。而中共大使馆在12月6日才发表声明。
 </span>
</p>
<p class="p1">
 <span class="s1">
  《香港01》报导说，当记者向中共驻温哥华总领事馆查询孟晚舟之事之时，一名领事反问记者“你们是怎么知道这件事的”，并称如有可对外公布的消息，会对外发放。
 </span>
</p>
<p class="p1">
 <span class="s1">
  从领事的表态中可以看出，他们早知此事，但并不想提前曝光。川习会后，
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BE%8E%E8%B4%B8%E6%98%93%E6%88%98.html">
   中美贸易战
  </a>
  暂停，中美双方都希望释放中美关系可控的利好信息。尽管12月1日当天孟晚舟已经被捕，但在川习会以及中美贸易战持续发酵的当口，中方或不想提前放大华为CFO孟晚舟被捕一事。#
 </span>
</p>
