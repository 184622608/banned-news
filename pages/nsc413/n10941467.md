### 退休上将刘亚洲小吃店过圣诞节讥新闻联播
------------------------

<p>
 【大纪元2018年12月30日讯】
 <span class="s1">
  （大纪元记者骆亚报导）中共要求民间不过所谓洋节过中国节日的做法，
 </span>
 <span class="s1">
  引发舆论的批评。近日，网传中共退休上将
  <a href="http://www.epochtimes.com/gb/tag/%E5%88%98%E4%BA%9A%E6%B4%B2.html">
   刘亚洲
  </a>
  的
  <a href="http://www.epochtimes.com/gb/tag/%E6%96%B0%E5%B9%B4%E5%AF%84%E8%AF%AD.html">
   新年寄语
  </a>
  视频，其公开过
  <a href="http://www.epochtimes.com/gb/tag/%E5%9C%A3%E8%AF%9E%E8%8A%82.html">
   圣诞节
  </a>
  并嘲讽中共央视新闻联播。民间认为这凸显中共内部分歧白热化。
 </span>
</p>
<p class="p3">
 <span class="s2">
  29
 </span>
 <span class="s1">
  日，大陆资深媒体人高瑜女士推出一段视频，内容是退休上将、前中共国家主席李先念的女婿
  <a href="http://www.epochtimes.com/gb/tag/%E5%88%98%E4%BA%9A%E6%B4%B2.html">
   刘亚洲
  </a>
  与友人在四川的街边小吃店的相关视频。这段视频并
 </span>
 <span class="s1">
  不完全是现场原始镜头，有一定的后期剪辑、制作，加了音乐、旁边、及字幕。
 </span>
</p>
<p class="p3">
 <span class="s1">
  镜头呈现四川的一排小吃店，随后聚焦其中一家，只见穿便装的刘亚洲对着镜头说：“今天是
  <a href="http://www.epochtimes.com/gb/tag/%E5%9C%A3%E8%AF%9E%E8%8A%82.html">
   圣诞节
  </a>
  吧，我们在路边小店欢庆圣诞。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  随后声音就变成旁边，镜头扩大到整个小店的一角，两个圆台面，分别坐了一些食客，小吃店内的画面。
 </span>
</p>
<figure class="wp-caption aligncenter" id="attachment_10941745" style="width: 450px">
 <a href="http://i.epochtimes.com/assets/uploads/2018/12/e3e41de0d7f8c64f3086860565b853d5.jpg">
  <img alt="" class=" wp-image-10941745" height="338" src="http://i.epochtimes.com/assets/uploads/2018/12/e3e41de0d7f8c64f3086860565b853d5.jpg" width="450"/>
 </a>
 <br/><figcaption class="wp-caption-text">
  12月25日刘亚洲与友人在四川峨眉山路边小吃点过圣诞节，并发
  <a href="http://www.epochtimes.com/gb/tag/%E6%96%B0%E5%B9%B4%E5%AF%84%E8%AF%AD.html">
   新年寄语
  </a>
  。（大纪元合成图）
 </figcaption><br/>
</figure><br/>
<p class="p3">
 <span class="s1">
  刘亚洲的旁白：“（这里”硬件最差，软件最好，最破烂的门面里，有最美的味道。看上去越高贵，可能越底下。看上去越香，可能越臭。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  一番与友人互动后，刘亚洲说：“有个人批评自己的朋友：你每天在手机上看那么多负面新闻不怕心里阴暗吗？这个朋友反驳他说：徐才厚雷打不动每天看新闻联播，怎么不见道德高尚？”
 </span>
</p>
<p class="p3">
 <span class="s1">
  最后其以“新年快乐”四字结束这段新年寄语视频。
 </span>
</p>
<p class="p5">
 <span class="s1">
  有网友表示，“话里有话，好像是事先准备好的，说明中共内斗白热化了。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  有网友认为，人不在官位才能说点真话。还有说，刘亚洲被贬反倒回归人性了。
 </span>
</p>
<p class="p5">
 <span class="s1">
  但也有网友认为这是刘亚洲在中共体制内不得宠，发发牢骚罢了。有网友怀疑是其继续忽悠，“我们屁民搞不懂。只希望，你们真的能给百姓选票，否则，就是装
 </span>
 <span class="s3">
  ……
 </span>
 <span class="s1">
  ”
 </span>
</p>
<p class="p5">
 <span class="s1">
  就刘亚洲讥讽新闻联播，有网友进一步表示，“每天雷打不动制作新闻联播的，自己都知道每天在忽悠。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  因近期中共严控网路言论，有网友还恶搞说：“公然妄议新闻联播的效力，这个够得上寻衅滋事了吧？都这么萧
 </span>
 <span class="s4">
  杀了，还不明了啥叫
 </span>
 <span class="s3">
  ……
 </span>
 <span class="s1">
  这还了得！”
 </span>
</p>
<p class="p5">
 不过刘亚洲在民间口碑并不好，
 <span class="s1">
  北京的一名女
  <a href="http://www.epochtimes.com/gb/tag/%E7%BA%A2%E4%BA%8C%E4%BB%A3.html">
   红二代
  </a>
  对其
 </span>
 <span class="s1">
  新年寄语不以为然，她向大纪元记者表示：“他表面给人以改革者的姿态出现，但实际做的事情并不怎么样，表里不一。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  对方还表示：“你要是干净的话，也不会上缴多占的钱款和房子。你如果两袖清风、一身正气别人也没话说。但你自己不正，说的再漂亮，别人也会认为你是两面派。因为个人不干净的原因被习提前下课，还假装是个改革派。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  对方还指刘亚洲，“打着中央的名义，明明执行曾庆红个人的指示，折腾我老伴
 </span>
 <span class="s2">
  ……
 </span>
 <span class="s1">
  ”
 </span>
</p>
<p class="p1">
 <span class="s1">
  有网友称其为“党的最忠诚战士。就是他最坚决拥护党指挥枪，坚决反对军队国家化，
 </span>
 <span class="s1">
  忽悠一下屁民而已。”
 </span>
</p>
<p class="p1">
 还有网民表示：“他再怎么说，也是属于党内狗咬狗而已。没有人会相信中共改革了，坐等中共政权垮台。”
</p>
