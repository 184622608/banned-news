### 学者热评：压力下中共会进行结构性改革？
------------------------

<p>
 【大纪元2018年12月08日讯】（大纪元记者骆亚、周慧心采访报导）
 <a href="http://www.epochtimes.com/gb/tag/%E5%B7%9D%E4%B9%A0%E4%BC%9A.html">
  川习会
 </a>
 后中美双方达成90天贸易战停火协议，中方承诺立即与美方展开
 <a href="http://www.epochtimes.com/gb/tag/%E7%BB%93%E6%9E%84%E6%80%A7%E6%94%B9%E9%9D%A9.html">
  结构性改革
 </a>
 谈判。什么是结构性改革？中共会履行承诺吗？对此学者、分析人士各有解读。
</p>
<p class="p1">
 <span class="s1">
  <a href="http://www.epochtimes.com/gb/tag/%E5%B7%9D%E4%B9%A0%E4%BC%9A.html">
   川习会
  </a>
  后，白宫发表声明表示：川普（特朗普）总统和习近平主席同意立即开始有关结构性改变的谈判。这些结构性改变涉及强制技术转让、知识产权保护、非关税壁垒、网络入侵和网络盗窃、服务业和农业。双方同意，他们会努力在未来90天内完成谈判。如果在这个期限内双方无法达成协议，美国会将10%的关税提高至25%。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  什么是
  <a href="http://www.epochtimes.com/gb/tag/%E7%BB%93%E6%9E%84%E6%80%A7%E6%94%B9%E9%9D%A9.html">
   结构性改革
  </a>
  ？
 </span>
</h4>
<p class="p1">
 <span class="s1">
  原香港城市大学政治学教授郑宇硕认为，所谓结构性的改革就是：中国是不是真正的市场经济？（如果不是，就需要进行结构性改变，转向市场经济。）中共政府、国企在经济上有这么大的影响力，显示中国不是真正的市场经济。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他对大纪元记者说：“（中共）它是不是愿意真的进行市场经济的改革，我们看不到它有这样的决心。现在看中国（中共）能让步多少，关键还是知识产权的保护、技术转让等等关键性的问题，网络的保护也是一个。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  中国经济学家、天则经济研究所创办人茅于轼认为，中国经济的问题在于国营企业。就像嫌疑人不能当法官，裁判员不能踢足球一样的道理，政府不能经营企业。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他对美国之音表示：“国企就是裁判员踢足球，国家是一个管理机构，你怎么跑到市场上赚钱呢？中国在贸易上所有的矛盾归根究底是中国国企的本质所决定的。中国说国企改革说了几十年了，没有找到办法，就因为国企本身是不合逻辑的东西，你怎么改啊？只有一个改法，就是把国企改成民企。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他表示，由于垄断缺少竞争，这些国企普遍效率不高，但是因有垄断地位，靠损害消费者的利益赚钱不少。这些大国企所赚的钱都成为政府的收入。所以中国政府除了税收还有国企的垄断利润，非常有钱。
 </span>
</p>
<p class="p1">
 <span class="s1">
  悉尼科技大学中国问题专家冯崇义教授也表示，真正的市场经济，政府就应该是游戏规则的维护者，政府不能直接参与经济。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “中国共产党领导一切，那就是裁判直接参与比赛，到场上当运动员了。国企与私企没有同等的竞争条件，市场的基本原则就违背了。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他对大纪元记者说：“比如说国企，这种信贷上的方便，各种补贴，就违反了市场公平竞争的规则，私人企业需要购买专利，而国企不用买专利，国家补贴给他。这样的话，企业跟企业之间就没有同等的待遇，这个市场经济就立不起来。正常的市场经济，不允许有一个庞大的，由国家铺贴的国企，在市场上欺行霸市，来垄断。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他介绍说，在毛时代，国企一统江湖，国企本身没有效率，所以让国家经济濒临崩溃，全民陷于贫困。邓时代经济自由化才带来发展。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “但是共产党本身的解释不一样，它说是中国特色，就是它有国企在里头，它的解读就是‘我们是举国体制，可以集中力量办大事’。它是把国企当成它一党专政的经济基础。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  在中美贸易战中，川普一直强调取消政府补贴。哥伦比亚大学政治学博士王军涛对大纪元记者表示，中国出口产品都会获得政府的补贴，价格当然就会比美国的产品更有竞争力。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “还有就是它用了人家东西不给钱，它还派间谍到别的国家偷技术，同时对美国到中国投资的公司强制进行技术转让。”他认为这些都是（中共）政府干预经济的结果。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  中共真会进行结构性改革？
 </span>
</h4>
<p class="p1">
 <span class="s1">
  冯崇义认为，中共不会进行结构性改革，它只是在想办法拖延。“一旦全面地（实施）市场经济，就削弱了它的经济基础，那么党的垄断也彻底失去了它的合法性。就跟别人一样，没有‘特色’了，就没有共产党的必要了，所以它是一个通盘考虑，要做大做强国企，那不仅仅是经济上的问题。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他表示，国企是中共吸血的工具。“它把纳税人的钱通过金融系统补贴给国企，这相当于是剥削、是吸血、掠夺民脂民膏的机制。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“国企控制上游产业：能源、金融、交通、航空等，民企也好、外企也好，就要低头，让你永远长不大，要你永远要依赖党国政府，要是真的把国企取消，党的专制也就崩塌了。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  冯崇义表示，中共是个极端狡猾的政权，这次中共的妥协条件其实是为川普量身订做的。因为川普的票箱来自农业，它答应要多购买农产品。而毒品是共和党非常关注的政治议题。
 </span>
</p>
<p class="p1">
 <span class="s1">
  中方这次所做的四方面承诺之一就是管制芬太尼出口美国。这种致命的合成鸦片类药物每年导致数万名美国人死亡。
 </span>
</p>
<p class="p1">
 <span class="s1">
  冯崇义表示，这次中共是选择这些能够取悦于川普和共和党的一些议题来做让步，还是缓兵之计。
 </span>
</p>
<p class="p1">
 <span class="s1">
  郑宇硕也认为中共政府是不会真的变成市场经济的，“要是真的市场经济了，国有部门就不能控制经济了。中共现在的态度就是且战且退，希望做一点点让步，暂时避免全面的贸易战，也希望争取一点时间，能追上去，缩小双方的差距。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  王军涛也不认为中共会进行结构性改革，因为谈判期间双方一直都各说各话。美国谈到贸易逆差，中共就说要购买一万二千亿的美国产品来弥补逆差。但美国表示贸易逆差只是结果，根本原因是美国产品进不去中国，中共设立很高的壁垒，这个壁垒除了关税之外还有非关税，就是市场准入。比如金融、互联网，这些很赚钱的领域，中共不让美国进去。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “到最后会怎么谈，我觉得如果川普总统完全要坚持他的要求的话，那双方还得打，打到最后，不只是贸易，打的是政治、军事。”#
 </span>
</p>
