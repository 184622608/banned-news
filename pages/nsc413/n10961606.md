### 体制内专家：北京将妥协 达川普最低要求
------------------------

<p>
 【大纪元2019年01月08日讯】（大纪元记者周慧心综合报导）对于中共
 <span class="s1">
  副总理
  <a href="http://www.epochtimes.com/gb/tag/%E5%88%98%E9%B9%A4.html">
   刘鹤
  </a>
  出现在中美副部级贸易谈判会场，北京“中央级智囊”时殷弘教授称，这表示中方高度重视这次谈判，且会做出一些妥协，“至少可达到川普的最低要求”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  据《明报》1月8日报导，中国人民大学国际关系学院教授时殷弘表示，副总理
  <a href="http://www.epochtimes.com/gb/tag/%E5%88%98%E9%B9%A4.html">
   刘鹤
  </a>
  现身，显示中方有诚意与美谈判。他表示中国会在金融体制、工业政策等方面做出妥协，“至少可达到特朗普（川普）的最低要求”。他认为双方都期待这次谈判可以解决一些问题，中方是面对经济下行及深化改革开放等；美国也要解决包括股市下跌等问题。但今次只是高层谈判的前奏，就算有一些成果，“可能也不会公布”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  《香港经济日报》说，时殷弘属于中共的“中央级智囊、美国研究权威”，其意见在某种程度上反映了北京最新的策略。
 </span>
</p>
<p class="p1">
 <span class="s1">
  对于刘鹤出现在低级别贸易谈判现场，海外中国问题评论人士文昭表示，这与中共酷爱面子的一贯态度相比可谓有巨大反差。
 </span>
</p>
<p class="p1">
 <span class="s1">
  文昭在其“谈古论今”节目中说，中美贸易开战以来，中共从“以牙还牙”、“不惜一切代价”，到以副总理之身屈尊参加一个副部长级会谈，这印证了川普之前的讲话：“他们（中国）的经济现在不好，这给了他们更大的谈判动力。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他表示，两天会议过后，有两个渠道可以推测中共做了多大的让步。一是看美方官员对媒体的放风。
 </span>
 <span class="s1">
  “结果是对美国有利的话，川普也乐于做这样的放风，有助于安抚动荡的股市。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  再是看刘鹤带团访美是不是很快成行。如果很快成行，说明美国已经得到了明确的信息，中方会做出足够的妥协，“因为川普的态度比较明确的是不做无结果的谈判，只有美方判断中方会朝美方的要求靠拢，双方主要官员的会谈成功概率比较高，刘鹤访美才会立即发生。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“所以刘鹤今年访美时间的缓急也是判断习近平是不是已经做好准备向美国做出大幅度妥协的标志。”
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  美国继续强硬
 </span>
</h4>
<p class="p1">
 <span class="s1">
  《华尔街日报》曝光中美首日磋商的内容，美方向中方代表团施压，要求给出具体方案或时间表、确保中方会履行承诺。
 </span>
</p>
<p class="p1">
 <span class="s1">
  报导引述了解谈判情况的人士的话说，当天以副贸易代表格里什（Jeffrey Gerrish）为首的美国谈判代表，针对中方的承诺对中方代表施压，要求提出能确保北京方面兑现承诺的方式。例如，明确购买项目和购买期限。
 </span>
</p>
<p class="p1">
 <span class="s1">
  报导说，美方谈判代表希望北京保证不会利用政府对许可证、环境监管及其它领域的权力，来阻碍美国企业进入中国市场。川普政府对执行贸易协议的细节如此关注，凸显中共糟糕的承诺兑现记录。美方官员指，尽管中共在过去几年的谈判中也做出过类似的开放市场的承诺，但美国和其它外企在中国却遭遇日益增多的障碍。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  中共“放软”示好
 </span>
</h4>
<p class="p1">
 <span class="s1">
  据路透社报导，
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BE%8E%E8%B4%B8%E6%98%93%E8%B0%88%E5%88%A4.html">
   中美贸易谈判
  </a>
  的第一天，1月7日上午，中国国有企业购买了至少3艘、约18万吨美国大豆船货。这是过去一个月内中方第三次大规模采购美国大豆。
 </span>
</p>
<p class="p1">
 <span class="s1">
  报导引述两位交易员的话表示，这些大豆船货将在1月至3月期间，主要从太平洋沿岸西北部的港口发运，少量将从美国墨西哥湾的港口出口。
 </span>
</p>
<p class="p1">
 <span class="s1">
  2018年7月6日中方报复美国提高
  <a href="http://www.epochtimes.com/gb/tag/%E5%85%B3%E7%A8%8E.html">
   关税
  </a>
  ，其中大豆、汽车各加征25%的关税。12月1日，川习会后，中方做出包括承诺购买美国大豆在内的让步。
 </span>
</p>
<p class="p1">
 <span class="s1">
  中共财政部12月14日在网站上公告，对原产于美国的汽车及零组件等211个项目，暂停加征
  <a href="http://www.epochtimes.com/gb/tag/%E5%85%B3%E7%A8%8E.html">
   关税
  </a>
  三个月。时间自2019年1月1日起至2019年3月31日。随后中粮和中储粮12月19日在官网发出声明，证实已经开始采购美国大豆。
 </span>
</p>
<p class="p2">
 <span class="s1">
  此次谈判后，预计刘鹤将率领中国代表团赴华盛顿，与美国贸易代表莱特希泽（Robert Lighthizer）和财长梅努钦（Steven Mnuchin）举行进一步会谈。#
 </span>
</p>
