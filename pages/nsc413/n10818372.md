### 央行警告做空者 人民币汇率再跌穿6.97
------------------------

<p>
 【大纪元2018年10月30日讯】（大纪元记者骆亚、周慧心采访报导）今天（30日），离岸人民币兑美元跌破6.97关口，低见6.9774，最新报6.9727。上周五（26日）中共央行副行长对海外
 <a href="http://www.epochtimes.com/gb/tag/%E5%81%9A%E7%A9%BA%E4%BA%BA%E6%B0%91%E5%B8%81.html">
  做空人民币
 </a>
 者发出警告，
 <a href="http://www.epochtimes.com/gb/tag/%E4%BA%BA%E6%B0%91%E5%B8%81%E6%B1%87%E7%8E%87.html">
  人民币汇率
 </a>
 短暂回升后再次下跌。
</p>
<p class="p1">
 <span class="s1">
  人民币兑美元今年已累计贬值近7%，10月25日、26日，离岸人民币对美元汇率连续突破6.95、6.96、6.97关口，逐渐逼近备受关注的7元关口。
 </span>
</p>
<p class="p1">
 <span class="s1">
  上周五，中共央行副行长潘功胜在新闻发布会上表示：“对于那些试图
  <a href="http://www.epochtimes.com/gb/tag/%E5%81%9A%E7%A9%BA%E4%BA%BA%E6%B0%91%E5%B8%81.html">
   做空人民币
  </a>
  的势力，几年之前我们都交过手，彼此也非常熟悉，我想我们应该都记忆犹新。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  随后在岸、离岸人民币对美元汇率双双反弹，一度涨超300个基点。
 </span>
</p>
<p class="u02q31-0 gvqXdj sc-gqjmRU gBjLGB">
 但周一再次进入跌势。今天（周二），人民币又再低收，人民币中间价今天下调197点，报6.9574，为2008年5月21日以来新低。
</p>
<p class="u02q31-0 gvqXdj sc-gqjmRU gBjLGB">
 在岸价今早曾低见6.9741，收市报6.9613，较上日低收53点；离岸价今日亦失6.97，低见6.9774，截止收稿时报6.9727。
</p>
<h4 class="p1">
 <span class="s1">
  做空造成
  <a href="http://www.epochtimes.com/gb/tag/%E4%BA%BA%E6%B0%91%E5%B8%81%E8%B4%AC%E5%80%BC.html">
   人民币贬值
  </a>
  ？
 </span>
</h4>
<p class="p1">
 <span class="s1">
  对于潘功胜对海外做空人民币者的警告，学者表示，不是因为做空人民币造成
  <a href="http://www.epochtimes.com/gb/tag/%E4%BA%BA%E6%B0%91%E5%B8%81%E8%B4%AC%E5%80%BC.html">
   人民币贬值
  </a>
  ，而是海外投资者看衰中国经济。
 </span>
</p>
<p class="p1">
 <span class="s1">
  但是为什么有人做空中国经济？做空人民币呢？美国南卡罗莱纳大学艾肯商学院教授谢田表示，就是因为中国经济出现了问题。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “中国经济出现问题，有多方面的原因，从出口衰退到产能过剩等。”他对大纪元记者说：“中美贸易战，中国对美国出口贸易大幅缩减，贸易顺差消失，人民币一定会下跌。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他表示，中共用外汇储备托人民币、托市，用政府控制的方法在操纵汇率，但是中国的外汇储备是有限的，早晚有一天会支撑不住的。
 </span>
</p>
<p class="p1">
 <span class="s1">
  很多投资人、投机者正是看到中国经济的危机，所以才会做空人民币。他说：“做空的人会从中获利，所以才会有人前仆后继地去做空人民币。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  前北大教授、中国学者夏业良认为，人民币贬值主要是由于自身经济造成的。“在于一种逆市场化的行为，就是国进民退，让民营企业遭受重创，国企又不能真正创造价值，所以造成了整个经济的衰退。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他对大纪元记者说：“由于经济的衰退，外汇储备大幅度的减少，大家对人民币的信心，对未来的这种上涨的趋势不看好，所以这个贬值是理所当然的。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  之前有分析认为，中共任由人民币贬值，以对抗贸易战带来的出口压力。夏业良表示，人民币贬值有助于出口，是在正常贸易的情况下，“但是现在的情况是全面的贸易战，不仅是美国，是整个西方国家对中国（中共）都要进行制裁。所以靠人民币贬值提高出口这个目的是达不到的。”
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  中共为何警告做空者？
 </span>
</h4>
<p class="p1">
 <span class="s1">
  既然人民币贬值不是由于做空造成，为何中共央行副行长潘功胜要以警告方式来制止人民币继续贬值呢？
 </span>
</p>
<p class="p1">
 <span class="s1">
  谢田教授表示，做空是一种投资或者投机方式，不只是人民币，有人也做空美元，或者做多美元，这个本来就是正常的。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “做空的人自己是有风险的，比如说他认定人民币会下跌，他现在做空。做空是有期限的，比如在一星期、一个月或者多长时间之内，如果到时候，人民币没有下跌，这些人会损失很多钱。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“这本来是正常的商业投资、投机行为，这根本不足为虑，但是中共很心虚，所以特别害怕别人做空人民币，或者做空中国的经济，这是中共虚弱的本质造成的。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他表示，中共好像对做空人民币，或者是唱衰中国的人特别憎恨。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “我听说，中美两国之间正在筹备G20上的‘川习会’，中方有一个要求，不让彼得・纳瓦罗参加，因为他写了一本《致命中国》的书，中共非常不快。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“中共对这些批评的声音，唱衰的声音，或者指中共要灭亡的声音，恼羞成怒。这实际上说明，这些说法点到了中共的死穴。”
 </span>
</p>
<p class="p1">
 《致命中国》（Death by China）以事实、数据来分析，从货币操纵、不平等贸易政策和有毒商品，三个角度来说明中共统治的中国，是如何危害世界和美国的。据说，选前川普指责中共的内容来自此书。
</p>
<p class="p1">
 该书的作者是彼得
 <span class="s1">
  ・
 </span>
 纳瓦罗（Peter Navarro）和安一鸣（Greg Autry），其中彼得
 <span class="s1">
  ・
 </span>
 纳瓦罗被川普提名担任美国“白宫国家贸易委员会”的主席。
</p>
<h4 class="p1">
 <span class="s1">
  人民币破7对中国有何影响？
 </span>
</h4>
<p class="p1">
 <span class="s1">
  大陆一位不方便署名的学者对大纪元记者表示，这次做空人民币，成功的可能性很大，因为中共需要通过贬值来应对出口困境，再加上手里的子弹不多了，一旦破7，防线就溃败了。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说：“一旦汇率破7，势必带动新兴市场货币竞相贬值，导致人民币国际化更加艰难。更重要的是，美国很可能把中国列为货币操纵国，引发更大规模的报复！因此现阶段保7是一场艰难的拉踞战，也是一场生死攸关之战。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  谢田教授认为，破7本身不会对中国经济造成什么特别的影响。它实际上是心理关口。“我觉得破7是早晚的事，破7以后人们可能会说7.5是另一个关口，或者8是另外一个关口。这实际上都没有什么太大意义，但是人民币的看低、贬值的趋势是肯定的。”
 </span>
</p>
<p class="p1">
 《华尔街日报》报导称，如果人民币兑美元汇率真的破7，如果中国企业和个人想在人民币进一步贬值之前将资本转移至海外，则可能导致人民币遭到进一步抛售。美元上一次升破人民币7元是在2008年5月的境内市场交易中。离岸人民币交易2010年才启动。#
</p>
