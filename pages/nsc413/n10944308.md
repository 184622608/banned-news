### 大陆民众恭贺法轮功创始人李大师 新年快乐
------------------------

<p>
 【大纪元2018年12月31日讯】
 <span class="s1">
  （大纪元记者骆亚、易如采访报导）
 </span>
 <span class="s2">
  2018
 </span>
 <span class="s1">
  年最后一天，历史将翻过新的一页。
  <a href="http://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
   法轮功
  </a>
  群体反迫害已持续近二十个年头，越来越多明白真相的世人公开站出来支持反迫害，并在新年之际通过大纪元平台向法轮功创始人
  <a href="http://www.epochtimes.com/gb/tag/%E6%9D%8E%E6%B4%AA%E5%BF%97.html">
   李洪志
  </a>
  大师和
  <a href="http://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
   法轮功
  </a>
  学员恭贺新年快乐。
 </span>
</p>
<p class="p5">
 <span class="s1">
  产业经济研究专家、上海企业家胡力任新年之际表示，“恭祝
  <a href="http://www.epochtimes.com/gb/tag/%E6%9D%8E%E6%B4%AA%E5%BF%97.html">
   李洪志
  </a>
  大师和所有法轮功弟子们新年快乐，让真善忍传遍全世界每个角落。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  “到目前为止，我所接触的每个法轮功弟子都非常的诚恳、善良、友好，只可惜魔鬼中共把法轮功弟子丑化了”。他真诚希望秉承真善忍的弟子能将法轮功传播到世界每个角落，让全世界人民都能了解真实的法轮功。
 </span>
</p>
<p class="p5">
 <span class="s1">
  “法轮功强身健体、修炼内心，对炼功者有很多益处。自魔鬼中共镇压法轮功开始，李大师能带领弟子不畏强权反迫害，反中共就是对全世界作出巨大贡献。”
 </span>
 <span class="s1">
  “法轮功学员已成为揭露魔鬼中共最有力、最彻底的群体。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  近二十年来，法轮功学员一直用和平理性，非暴力方式在全世界范围内讲清真相。
 </span>
</p>
<p class="p3">
 <span class="s1">
  上海市民殷先生一直关注法轮功受迫害问题。今年，他在台湾景点看到法轮功学员对大陆游客讲真相。
 </span>
 <span class="s1">
  “中共当局对法轮功群体完全不讲法，不在宪法框架内来讨论问题。很邪恶。”
 </span>
</p>
<p class="p8">
 <span class="s1">
  “法轮功倡导真、善、忍，百姓做善事、讲真话，无论在什么社会都是正确的。现在中国社会就容不下这样讲真话、讲善的这些人，只能证明这个社会、这个政权的假、恶、暴。”
 </span>
</p>
<p class="p8">
 <span class="s1">
  法轮功学员都是一群好人，如果今后社会改变的话，他们会发挥更大的作用。“祝福他们在新的一年好运，不再受迫害。祝愿法轮功学员的师父李洪志先生身体健康。我也希望我们的社会能够更多的公平和正义。”
 </span>
</p>
<p class="p8">
 <span class="s1">
  他表示，“请代我向所有新唐人、大纪元、希望之声的工作人员拜个年
 </span>
 <span class="s1">
  ，大陆上海有你们的一个读者、观众，一直在观注、支持你们。”
 </span>
</p>
<p class="p9">
 <span class="s1">
  中共迫害法轮功是本世纪最大的人权灾难已成共识，大陆知名网路作家荆楚先生向大纪元记者表示：“中共对法轮功的迫害就等于对全人类的迫害，我希望国内同胞看到更多真实的情形，不要被中共的宣传洗脑，接受单方面的灌输而颠倒黑白。”
 </span>
</p>
<p class="p8">
 <span class="s1">
  “我接触的法轮功学员都是诚实善良的人。法轮功提倡真、善、忍，这是这社会人们和谐相处，有利于社会进步和社会文明的基本价值。共产党提倡的你死我活、残酷斗争、无情打击的这套仇恨理念跟法轮功恰恰相反，把人变成了畜生野兽，因而导致社会败坏。
 </span>
 <span class="s1">
  ”荆楚说。
 </span>
</p>
<p class="p8">
 “我支持法轮功学员要求正本清源、要求还李大师清白。希望李洪志先生身体健康，新年快乐！向他问好。”
</p>
<p class="p8">
 <span class="s1">
  安徽的王先生认为中共迫害法轮功是一场政治运动，二十年中，
 </span>
 <span class="s1">
  法轮功学员经历了很多苦难与不幸，很多大陆民众对此都有所了解，比如黑监狱、学习班、活摘器官等等。
 </span>
</p>
<p class="p8">
 他赞誉法轮功在反迫害过程中所表现出来的精神“可喜可贺”。“中国大陆没有任何一个团体可以表现出这样的气质，我支持法轮功学员反迫害，继续努力。”
</p>
<p class="p8">
 <span class="s1">
  “李洪志先生提倡的真、善、忍不仅对社会进步有意义，也对中国民众的启蒙做出贡献。” 他说。
 </span>
</p>
<p class="p8">
 <span class="s1">
  “如果没有真，所有的假、恶、臭就可能呈现在世人面前。如果没有真，任何武器都不可能把邪恶打倒。善是人的本性，如果没有善，岂不人人为恶，这个社会就会崩塌。与人为善这是做人的基本准则。”
 </span>
</p>
<p class="p8">
 <span class="s1">
  “在
 </span>
 <span class="s2">
  2019
 </span>
 <span class="s1">
  年最不平凡的这一年即将到来之际，祝愿李洪志先生、也祝愿所有的大法弟子身体健康、万事如意。希望在不久的将来、在神州大地再次见到他们的身影。”王先生说。
 </span>
</p>
<p class="p8">
 <span class="s1">
  前中共总书记赵紫阳的政治秘书鲍彤先生透露，有关部门要求他在
 </span>
 <span class="s2">
  1
 </span>
 <span class="s1">
  月
 </span>
 <span class="s2">
  12
 </span>
 <span class="s1">
  日前不要接受媒体采访，但他表示还是想说一句新年祝福：“希望包括法轮功学员在内的全天下中国人新年身体健康、新年快乐！”
 </span>
</p>
<p class="p8">
 <span class="s1">
  上海市民汪建华新年之际通过大纪元平台送上祝福：“祝贺李洪志先生在
 </span>
 <span class="s2">
  2019
 </span>
 <span class="s1">
  年健康，快乐！”
 </span>
</p>
<p class="p8">
 <span class="s1">
  辗转抵达旧金山不足两周的中国八九民运学生领袖、知名维权人士赵常青先生说：“祝福李洪志先生在新的一年身体健康、生活愉快、一切都好！也祝福各位法轮功学员同胞在
 </span>
 <span class="s2">
  2019
 </span>
 <span class="s1">
  年健康、快乐、幸福、平安！”#
 </span>
</p>
