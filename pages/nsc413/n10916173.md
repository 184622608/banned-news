### 中美贸易战 专家解析中共“结构性改革”
------------------------

<p>
 【大纪元2018年12月17日讯】（大纪元记者骆亚、周慧心采访报导）12月18日，
 <span class="s1">
  中共改革40周年大会将在北京人民大会堂举行，中共官媒将同步转播，预期习近平将宣布新的市场开放计划。
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BE%8E%E8%B4%B8%E6%98%93%E6%88%98.html">
   中美贸易战
  </a>
  下中共真的会履行承诺——进行
  <a href="http://www.epochtimes.com/gb/tag/%E7%BB%93%E6%9E%84%E6%80%A7%E6%94%B9%E9%9D%A9.html">
   结构性改革
  </a>
  吗？引外界关注。
 </span>
</p>
<p>
 中共已故总书记赵紫阳前秘书鲍彤则表示“改革已死”。
</p>
<p class="p2">
 <span class="s1">
  BBC中文网12月12日刊出赵紫阳前秘书鲍彤谈改革40周年的文章，他在文中表示，可以说“改革”已经死了、或变质了、异化了。
 </span>
</p>
<p class="p2">
 <span class="s1">
  他写道：“巧妙的办法是打着‘完善’、‘深化’、‘规范’诸如此类的旗号，把不愿做奴隶的人的手脚重新捆绑起来。密密麻麻的审批应运而生。立项必须是‘自己人’，不是自己人就必须寻租。党国批准了，诸事顺遂。党国不同意，不给你许可证，生意做不成，贷款没指望，打官司一定输⋯⋯”
 </span>
</p>
<p class="p2">
 <span class="s1">
  悉尼科技大学中国问题专家冯崇义教授也同意鲍彤的说法，“这是自由学派很多学者的共同判断，改革早就死了。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  他对大纪元记者说，在八九年中共镇压学生运动之后，原来改革的概念就死了。“后面所做的所谓改革，就是用放开市场来挣钱、收买人心。现在重提改革是因为现在中国经济下行，它没钱了，它要去一带一路、要维稳、要收买人心，它没钱了。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BE%8E%E8%B4%B8%E6%98%93%E6%88%98.html">
   中美贸易战
  </a>
  导致中国经济泡沫破灭，据中共国家统计局公布的数字，中国11月的消费品零售总额同比仅增长了8.1%，创下2003年5月以来的新低水平。上个月的工业产出同比增长了5.4%，为三年来最小增幅。
 </span>
</p>
<p class="p3">
 <span class="s1">
  中共虽然被迫同意进行
  <a href="http://www.epochtimes.com/gb/tag/%E7%BB%93%E6%9E%84%E6%80%A7%E6%94%B9%E9%9D%A9.html">
   结构性改革
  </a>
  ，但是“此改革”非“彼改革”，冯崇义教授表示，狡猾的中共在偷换概念。
 </span>
</p>
<p class="p3">
 <span class="s1">
  他说，川普所说的结构性改革是让中国像西方一样的自由经济，而中共所说的结构性就是谈了很多年的供给制改革。
 </span>
 <span class="s1">
  “中国的经济里有三部分，一是国企来控制包括能源、交通、金融在内的上游产业；二是引进外国技术和资金来发展高新技术的中游产业；三是服务业和下游的加工业就是民企。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  而中共所谓的改革就是以前一些企业、银行不让进入，现在可以放开，只是分享一些机遇跟你一起来垄断中国的市场，撷取高额利润，“在银行金融这块做一些改革，然后混合所有制，让社会上的资金，西方的资金进来，帮它们把这些僵尸企业救活，让它们挣钱，这是它们理解的结构。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  他表示：“从这点上你可以看到共产党的狡猾，它就玩这些概念，你讲的结构不是它讲的结构，所以现在还是用那种方式，它就是回到原点上去，不存在往前走改革的事情。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  前副团级退役军官胡伟
 </span>
 <span class="s1">
  认为中共同意改革只是“缓兵之计”，
 </span>
 <span class="s1">
  中美贸易战，它们被迫去提出改革的构思，实际上都是“做给美国人看”，它能够真正落实下来的改革是很少。“中国的经济改革已经走到了一个死胡同，再往下改的任何一个步骤都会触及到它的红线，政治底线。”
 </span>
</p>
<p class="p2">
 <span class="s1">
  他对大纪元记者表示，跟美国、国际社会形成这么严重的对立，中共妥协只是为了换得一个喘息的时机，
 </span>
 <span class="s1">
  “它不可能现在给你制定一个文件就永远走下去了，说不定到明年川普不在了，它马上把这个文件毁掉、再倒回来。”
 </span>
</p>
<p class="p4">
 冯崇义教授表示，川普施加的压力不会逼中共就范，就是改革已死，到了现在这个情况下，你要有一整套的市场机制。“真的改革、真正意义上的改革就是宪政转型，就是整个法律制度、整个一党专政取消掉。”#
</p>
