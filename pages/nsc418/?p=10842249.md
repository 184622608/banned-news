### 加拿大拨款8600万加元 打击团伙和枪支暴力
------------------------

<p>
 【大纪元2018年11月10日讯】美国加州发生13人死亡的酒吧枪击案，似乎加快了加拿大阻止枪支犯罪的步伐。
</p>
<p>
 为了阻止走私枪支进入加拿大并加强对枪支犯罪的调查，加拿大政府决定在今后五年内向加拿大
 <a href="http://www.epochtimes.com/gb/tag/%E8%BE%B9%E5%A2%83%E6%9C%8D%E5%8A%A1%E7%BD%B2.html">
  边境服务署
 </a>
 和加拿大皇家骑警
 <a href="http://www.epochtimes.com/gb/tag/%E6%8B%A8%E6%AC%BE.html">
  拨款
 </a>
 8600万加元。
 <a href="http://www.epochtimes.com/gb/tag/%E5%8A%A0%E6%8B%BF%E5%A4%A7%E5%85%AC%E5%85%B1%E5%AE%89%E5%85%A8%E9%83%A8.html">
  加拿大公共安全部
 </a>
 长古德尔（Ralph Goodale）和边境安全部长布莱尔（Bill Blair）星期四（11月8日）宣布了这个消息。
 <br/>
 加广版权所有www.rcinet.ca微信:radio-canada
 <br/>
 加拿大年
 <a href="http://www.epochtimes.com/gb/tag/%E8%BE%B9%E5%A2%83%E6%9C%8D%E5%8A%A1%E7%BD%B2.html">
  边境服务署
 </a>
 将获得5150万加元，用于建立一个全天候的嗅探犬训练中心，在邮件分拣中心和航空货运设施增加嗅探犬和X光检查，并加强关于检查入境车辆内所藏物品的培训。
 <br/>
 加广版权所有www.rcinet.ca微信:radio-canada
 <br/>
 主管边境安全和打击有组织犯罪的内阁部长布莱尔。（CP/Adrian Wyld）
 <br/>
 余下3450万加元将用于加强加拿大皇家骑警对枪支犯罪的调查和相关训练。
 <br/>
 加广版权所有www.rcinet.ca微信:radio-canada
 <br/>
 这笔钱出自去年联邦财政报告中打击团伙和枪支暴力的3.27亿加元预算。布莱尔说，加拿大的枪支文化和美国有根本的不同，但是加拿大人对本国枪支犯罪的增加越来越担心。
 <br/>
 加广版权所有www.rcinet.ca微信:radio-canada
 <br/>
 去年加拿大边境人员共查获467次枪支偷运，比上一年增加了19%。枪支暴力犯罪也连续三年呈上升趋势。
</p>
