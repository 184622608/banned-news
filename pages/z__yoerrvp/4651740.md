### 德国外长将在访华期间提出维吾尔族人问题
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-11-10T00:47:20+08:00">
   2018年11月10日 00:47
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <p>
  德国外长马斯即将访问北京。他的发言人11月9日星期五表示，在北京期间，他将提出新疆的人权问题。
 </p>
 <p>
  土耳其安纳多卢通讯社报道说，德国外长发言人星期五在柏林的一次记者会上说，“从我们的观点来看，新疆的局势令人担忧。马斯外长将与中国外长提出这个问题。”
  <br/>
  <br/>
  马斯外长将从星期天开始访问中国。
  <br/>
  <br/>
  中国在新疆设立所谓的再教育职业培训营，羁押可能上百万穆斯林的报道在国际间引起广泛的关注。
  <br/>
  <br/>
  中国政府坚称没有在那里的设立侵犯人权的集中营，但中国政府及其支持者迄今为止没有说明为什么在那些所谓的培训中心的人不能见家人，不能见律师，不能知道什么时候可以回家，为什么不能让国际媒体记者和国际调查人员前去实地查看那里究竟正在发生什么。
  <br/>
  <br/>
  从新疆出逃的为维吾尔族人向外界报告说，那里的所谓培训中心就是监狱，在里面挨打挨饿是家常便饭，在押的人被强迫洗脑，强迫学汉语，学唱歌颂中国共产党及其领袖习近平的红歌。
  <br/>
  <br/>
  在德国外长访问中国期间，德中双方还将审议双方定于今年12月举行的下一轮人权对话的准备工作。
  <br/>
  <br/>
  本星期四，德国议会一些议员呼吁中国结束对新疆自治区维吾尔族穆斯林的压迫性政策。
 </p>
 <p>
