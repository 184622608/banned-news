### 越南新任国家主席阮富仲的内政外交策略 与中国合作但不显得亲中
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-10-26T20:50:13+08:00">
   2018年10月26日 20:50
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <span class="dateline">
  台北 —
 </span>
 <p>
  越南共产党的新领导人将拥有史无前例的权力，在就任越南领导人后，他将在与老对手中国维持稳定关系的同时，与美国加强贸易和安全联系。
  <br/>
  <br/>
  越南国会星期二选举阮富仲为越南国家主席。上个月，越南前主席、曾在公安系统任职的陈大光因病逝世。阮富仲至少将担任主席到2021年。
  <br/>
  <br/>
  越南专家表示，作为越南国家和共产党的领导人，阮富仲将把自己丰富的政治经验应用在外交和经济政策领域，特别是阮富仲将能够接触到更多的外国首脑，而这些外国首脑通常不把党的领导人视为重要政治人物。
  <br/>
  <br/>
  越南河内的研究机构湄公经济（Mekong Economics）的首席分析师亚当·麦卡提说：“作为总书记，国际上难以确定要怎么对待你，但作为主席，你就受到了承认，他们知道主席是什么。”
  <br/>
  <br/>
  麦卡提说：“作为主席和总书记，他预计会有大得多的权力，在访问其他国家时，他也会因此被视为更重要的角色，而不仅仅是一个主席。”
 </p>
 <p>
  <strong>
   74岁的老共产党
  </strong>
 </p>
 <p>
  根据越共网站的信息，今年74岁的老共产党阮富仲来自“贫农”家庭，毕业于河内综合大学语言文学系。他51年前加入越共，已担任五届政治局成员。
  <br/>
  <br/>
  在越共统治越南42年的历史里，只有越共创立者胡志明同时担任过总书记和主席职务，但当时越南还没有今天的外交和世界经济地位。
 </p>
 <p>
  一般而言，总书记和主席共享权力。阮富仲的角色与中国国家主席习近平类似，习同时是中国共产党的总书记。
  <br/>
  <br/>
  华盛顿智库战略与国际研究中心东南亚项目副主任穆雷·希伯特（Murray Hiebert）表示，阮富仲是越共领导层“同侪之首”，又没有其他的挑战者，所以自然而然当上了主席。希伯特说，阮富仲可以更容易地在政府中实现他的政策。
 </p>
 <p>
  <strong>
   更强的经济和外交行动
  </strong>
 </p>
 <p>
  分析人士表示，阮富仲的当选应该不会改变政府长期的野心，比如打造出口型经济以及在中国和西方盟友间找到外交平衡。
  <br/>
  <br/>
  越南经济中心胡志明市的一名不愿透露姓名的商人对美国之音表示，越南的普通人对领导层的变化没有什么兴趣，因为这种变化没有在医疗、商品价格等方面带来什么实际的变化。
  <br/>
  <br/>
  但阮富仲的双重身份将让他能更容易地接触到美国领导人。由于特朗普对双边合作显示出兴趣，阮富仲预计将推动和美国的双边贸易。去年，特朗普退出了跨太平洋贸易协定，协定的缔约国包括越南。
  <br/>
  <br/>
  越南把美国视为其第一大出口市场，年出口量330亿美元，占越南总出口的20%。
  <br/>
  <br/>
  但澳洲新南威尔士大学的荣休教授卡尔·塞耶（Carl Thayer）表示，阮富仲可能需要克服美国人认为共产主义越南缺乏市场经济的看法。
  <br/>
  <br/>
  还有学者表示，越南可能请求美国给予更多军事援助，随着美国政府2016年废除了军售禁令，这个选择的可能性也在增加。
  <br/>
  <strong>
   <br/>
   对抗中国
  </strong>
 </p>
 <p>
  学者们认为，阮富仲主席将用既有的党对党和国对国渠道与中国合作，但不会显得亲中。在过去，党和政府的对话是由不同人负责的。
  <br/>
  <br/>
  塞耶说，没有哪个越南人是“亲中的”，但有些人可能比别人更愿意和中国合作。塞耶说：“在我看来，越南对华政策是确定的，越南可以使用这些政策提供的对华关系特殊渠道。”
 </p>
 <p>
  中国是越南第二大出口国，也是越南工厂使用的原材料的最大供应国，推动着快速增长的越南经济。但越中两国在1970年代打了一场边境战争，现在又在南中国海有主权争议。
  <br/>
  <br/>
  中国在1970年代控制了有争议的帕拉塞尔群岛（中国称西沙群岛），两国目前在争夺海底油田开采权。
  <br/>
  <br/>
  “（阮富仲）预计会继续试图平衡对华和对美关系，”希伯特说，“随着中国在南中国海越来越强势，他会向中国寻求贸易和投资，向美国寻求安全支持。”
  <br/>
  <br/>
  <strong>
   铁腕反腐
  </strong>
 </p>
 <p>
  越南的反腐运动从2016年开始为外界所知，越南当地媒体VnExpress International报道称，反腐运动预计将在阮富仲任内加强。报道引述了“许多”针对商业巨头和政府领导人的打击。
  <br/>
  <br/>
  塞耶说，阮富仲反对过度奢侈的生活方式，认为那是腐败的代表。
 </p>
</div>
