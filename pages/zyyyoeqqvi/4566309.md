### 毛忌日韶山警方抓捕佳士声援团成员
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-09-11T20:48:00+08:00">
   最后更新： 2018年9月11日 20:48
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <span class="dateline">
  北京 —
 </span>
 <p>
  持续数月的深圳佳士工人组建工会维权和大学生声援行动半个月前被镇压后，9月9日又有十余名声援学生和维权工人在毛泽东故乡韶山祭拜被警方抓捕。打出毛泽东旗号看来不仅没能成为这些维权者的保护伞，反而在同情者当中引起争议。
  <br/>
  <br/>
  9月9日，部分声援佳士工人组建独立工会的工人及高校学生到湖南韶山，举行纪念前中共领导人毛泽东忌日的活动。佳士声援团在网络上发布的图片显示，数十名工人和学生在韶山毛泽东铜像广场打出横幅并向毛泽东铜像献花。警察随后赶来对声援者进行盘查，并强行带到韶山冲派出所。截止到发稿时，这批学生仍然与外界失联。
  <br/>
  <br/>
  美国之音记者打电话到韶山冲派出所查询，派出所工作人员回答说“这个问题他无权透露相关信息”。韶山市委值班人员周先生表示，去韶山纪念毛泽东要符合公安维稳的相关规定。韶山市政府工作人员刘女士表示，在韶山纪念毛泽东被抓，这样的事情不存在。刘女士还表示，网络上造谣的人很多，不应该相信，要看官方的说法。
  <br/>
  <br/>
  8月24日，当局对佳士声援团强制清场后，参与声援的学生陆续被国保遣返回家，中国官媒发动猛烈舆论攻势，将佳士工人维权活动归咎于“境外势力”，只字不提学生和包括体制内资深党员和退休干部在内的社会各界关注和参与，引起学生声援团和一些资深体制内人士公开表达异议。
  <br/>
  <br/>
  佳士工人要求组建工会的维权活动持续数月，得到了一些前中共党内官员和高校学生的支持。左派网站乌有之乡的网民也公开声援佳士工人维权。乌有之乡负责人范景刚对美国之音表示，他了解发生在韶山的事件，但是现在不便对此发表评论。
 </p>

 <p>
  参与声援行动的中山大学硕士毕业生、维权工人沈梦雨、北大毕业生岳昕，以及部分左派青年目前仍然失联。部分到深圳参与声援的学生在网络上公开了警察暴力清场的经过。参与佳士维权的工人余浚聪、米久平、刘鹏华、李展一周前已被警方以“涉嫌寻衅滋事罪”为由正式逮捕。
  <br/>
  <br/>
  左派人士认为，学生到韶山纪念毛泽东被抓是当局针镇压佳士事件行动的一部分。目前中国各大高校陆续开学，有左派人士认为声援学生长时间缺课可能受到学校开除学籍的处分。左派圈内也发生了意见分歧，部分人士主张继续维权，也有人认为应该暂时保存实力做长远打算。
 </p>
 <p>
  许多自由派人士本来同情并支持学生声援工人维权，但是对于左派学生尊毛的口号和行为表示不能接受，并提出批评，指称这些学生和工人过于幼稚，没有认清毛泽东独裁本质及其对当今中共专制的后遗影响。
  <br/>
  <br/>
  7月末以来，中国各地多所高校左派学生自发组成声援团，参与声援的学生被警察抓捕后均被遣返受到严密监管，而部分工人则受到刑事指控。据悉，目前至少有16名佳士建会工人以及声援团的工人、大学生成员和红色刊物编辑人员，被警方等强力部门正式逮捕、刑拘、监视居住和软禁，人民大学2016级学生杨舒涵返校上课路上被不明身份人员阻拦。
  <br/>
  <br/>
  中共领导人习近平上台以来，对社会控制日趋收紧。习近平提出改革开放前后两个三十年不能互相否定。中国最新修订的历史教科书重新评价毛泽东，将毛泽东发动文化大革命的“错误”二字删除， 官方稍早前发行的历史教科书将原来定性为“十年浩劫”的文革修改为“艰辛探索”。
  <br/>
  <br/>
  在中国当局大力倡导“红色旅游”的背景下，韶山的旅游业发展日益红火。每逢清明节、毛泽东的生日和忌日，大批左派人士、弱势民众前往韶山“朝圣”，颂扬他们所说的“人民领袖”，怀念贪腐现象相对稀少的毛时代。 官方尽管处境尴尬，但始终对民间通过这种“借古讽今”来反贪倡廉的方式保持低调。
 </p>
