### 特朗普嘲笑指控卡瓦诺性侵的妇女令议员不满
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-10-04T06:52:45+08:00">
   2018年10月4日 06:52
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <span class="dateline">
  国会山 —
 </span>
 <p>
  特朗普总统受到一些关键的共和党参议员的批评，原因是他嘲笑指控大法官提名人卡瓦诺性侵的福特教授。
 </p>
 <p>
  联邦调查局正在就有关卡瓦诺法官当年有性侵行为的指控进行调查，卡瓦诺否认他曾经在高中和大学攻击任何女性。
 </p>
 <p>
  “特朗普的言论完全是错误的，”代表缅因州的联邦参议员苏珊·柯林斯在国会山对记者说。她是两名没有明确表明将投赞成票还是反对票的共和党参议员之一。 另一位关键的共和党参议员丽莎·穆尔科斯基称特朗普的这番言论“完全不合适，而且在我看来是不可接受的”。
  <br/>
  <br/>
  特朗普在密西西比州的一次集会上面对数千名支持者，戏拟了福特在上周参议院听证会上对问题的回应。她作证指控卡瓦诺在1982年的一次高中派对上酒醉侵犯了她。
  <br/>
  <br/>
  “我喝了一瓶啤酒”，特朗普模仿福特的样子说。“你怎么到家的？我不知道。你怎么去派对的？我不知道。这个派对在哪里？我不知道。事情发生在多少年前？我不知道。我不知道，我不知道！楼上，楼下，到底在哪？我不知道，”他说。
  <br/>
  <br/>
  福特告诉参议院司法委员会说，尽管有记忆空缺，但是她“100%肯定”喝醉酒的卡瓦诺将她压在床上，乱摸她，而且把手放在她的嘴上来阻止她呼救。卡瓦诺在听证上说，他从未侵犯过任何人，并说他受到“政治打击”，试图毁掉他的名誉。
  <br/>
  <br/>
  特朗普的这番言论招致国会山的很多但绝非所有议员的批评。白宫发言人桑德斯说，特朗普总统只是在陈述本案事实，并对他提名的卡瓦诺充满信心。
 </p>
 <p>
  参议院少数党领袖查克·舒默抨击总统对福特的嘲笑“应受谴责，不符合总统行为，也不符合人们彼此尊重的起码文明。”
  <br/>
  <br/>
  舒默补充说，“长期以来，性侵受害者都不敢站出来，因为她们惧怕有势力的男人会会高声盖住她们，毁掉她们的人品。美国的总统，这位世界上权力最大的人，现在正在用最卑劣的方法来证实着数百万女性的恐惧。”
  <br/>
  <br/>
  “无论何时何地都不该有这种言论，”共和党参议员杰夫·弗莱克周三对全国广播公司(NBC)“今日”节目说，“在一个政治集会上谈论这么敏感的问题根本就不妥。”
  <br/>
  <br/>
  在回应这些批评时，总统顾问凯莉安·康威说，白宫超乎寻常地对福特表示了尊重。
 </p>
 <p>
  “我们所有人都把她当成法贝热彩蛋一样对待，”康威对记者说。
 </p>
 <p>
  “法贝热彩蛋“是俄罗斯著名珠宝匠法贝热为沙皇制作的珍贵工艺品。
  <br/>
  <br/>
  特朗普总统提名卡瓦诺法官出任联邦最高法院的大法官，如果参议院予以确认，他将代替退休的大法官安东尼·肯尼迪。目前美国最高法院只有八名大法官。
  <br/>
  <br/>
  共和党人目前以51比49的优势主控参议院，而下个月的中期选举将决定哪个党将控制国会两院。
  <br/>
  <br/>
  目前，没有民主党参议员表示支持卡瓦诺，也没有任何共和党参议员公开宣布要投反对票。
 </p>
 <p>
 </p>
