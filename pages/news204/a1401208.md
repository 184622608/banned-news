### 【禁闻】人类基因编辑背伦坏理 贺建奎被划清界线
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】当中国的科学家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 宣称〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑.html" target="_blank">
  基因编辑
 </a>
 婴儿〞已经诞生，而受到国际及国内舆论强烈杯葛的同时，不仅
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党媒.html" target="_blank">
  党媒
 </a>
 立刻撤下类似报喜讯的报导，之前核准这项研究的单位及主管人员，也迅速跟贺建奎划清界线，甚至指称审查文件是贺建奎假造。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 ，2010年获得美国莱斯大学生物物理学博士学位，在美国史丹福大学任博士后研究。2012年被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 引进回国，在南方科学技术大学成立个人实验室，进行基因测序方向的研究。同年7月，贺建奎在学校的支持下，创办了〝深圳市瀚海基因生物科技有限公司〞。
 <br/>
 <br/>
 贺建奎接受媒体采访时表示，他因专心经营自己的企业而〝停薪留职〞。
 <br/>
 <br/>
 贺建奎的基因生意越做越大。媒体报导，贺建奎至少是7家公司的股东、6家公司的法定代表人，并且是其中5家公司的实际控制人。7家公司的总注册资本为1.51亿人民币，其中深圳市〝南科生命科技公司〞注册资本最高，而这家公司24.5%股份，由南方科技大学旗下的深圳市〝南科大资产经营管理公司〞持有。
 <br/>
 <br/>
 中国临牀试验注册中心在线实验注册信息显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑.html" target="_blank">
  基因编辑
 </a>
 实验，已经在去年3月7号获得批准，批准这项实验的机构是深圳〝和美妇儿科〞医院伦理委员会。
 <br/>
 <br/>
 今年，〝瀚海基因〞完成2.18亿元〝A轮〞融资，准备实现大批量投产。南科大为此专门允许贺建奎停薪留职，全力发展自己的事业。
 <br/>
 <br/>
 不过，当〝基因编辑婴儿〞受到舆论强烈杯葛时，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党媒.html" target="_blank">
  党媒
 </a>
 《人民网》删除了一则相关〝喜讯〞报导。同时，当初所有涉及的单位，快速跟贺建奎划清界线，并异口同声的表示这件事与他们无关，不是他们出资的钱，文件是贺建奎造假的。
 <br/>
 <br/>
 有媒体曝光〝基因编辑婴儿〞计画经费来自南方科技大学，参与实验的夫妇可获人民币28万元，包括住宿费、误工费、照顾费等。
 <br/>
 <br/>
 什么是基因编辑？就是科学家利用基因编辑工具，进行删除、修改或取代植物、动物甚至
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人类.html" target="_blank">
  人类
 </a>
 的活细胞DNA。
 <br/>
 <br/>
 美国西达赛奈医学中心博士后科学家陈力：〝那个基因是一个受体，它在细胞的表面上担任，然后那个艾滋病的那个HIV病毒能识别这个受体，然后就能进入这个细胞内了，但是如果它这个受体没了，艾滋病毒就识别不了这个细胞了，就像门上有一个把手，你把门把手给去掉了，那我们就开不进门而且进不去了。〞
 <br/>
 <br/>
 美国西达赛奈医学中心博士后科学家陈力接受记者采访时表示忧心，他认为，人类如果开了基因编辑这扇大门，恐怕无法再关上了。
 <br/>
 <br/>
 陈力：〝如果人类开了这一个门的话，这是第一次尝试，那么就会有第三、第四次，以后越来越多的尝试。那么那么你动越多的基因，产生这个畸形儿童或者畸形人类的这个可能性就越大，可能他不抵抗艾滋病了，可能其它更严重的病就出来了。〞
 <br/>
 <br/>
 基因的改变让人体产生什么样的后果，目前人类无法预知。
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝那么这个人类将来他如果一旦存活下来，他长大了，将来他会结婚生子，会传宗接代，那么他就会把他被污染的、被改动的、被改变、被编辑过的这些基因，他会通过遗传的方式扩散到这个人类社会当中，那么这个对整个人类的基因安全可以说都是巨大的威胁。〞
 <br/>
 <br/>
 唐靖远认为，这项技术如果在中共体制下，被大规模使用的话，后果不堪设想。
 <br/>
 <br/>
 唐靖远：〝我们打一个比方，就是如果说毛泽东他掌握了这样的技术，他手里有这样的技术资源的话，他完全可能利用这个技术制造出一个加强版的毛泽东来接他的位置，也就是说这个对整个人类社会，对整个人类的文明，可以说是非常可怕的一种灾难性的后果。〞
 <br/>
 <br/>
 目前传出，贺建奎团队所申请的另一项基因编辑研究的伦理审查，正在进行中。他们计划对400个人类胚胎进行操作。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/周天
