### 传范冰冰向高层求助遭拒：〝小范！你适可而止〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 逃税案有了定论，缴纳8.8亿元人民币可免除牢狱之灾。随着案件的落幕，更多细节也随之曝光出来。知情人士5日对台媒爆料，范冰冰一开始曾向中共高层求助，却遭拒绝，还遭警告：〝小范！你适可而止〞。
 <br/>
 <br/>
 台湾《三立新闻》10月5日消息称，记者日前走访软禁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 的招待所，红色砖瓦大厦看起来像一般民宅的社区，里面据传疑似有一个北京公安招待所，而且还有一个五星级套房，范冰冰两度进出此地，前后大约被软禁了2个月。
 <br/>
 <br/>
 此前知情人爆料，范冰冰遭软禁的秘密招待所，内部设备高级如豪宅。但范冰冰全天受专人监管，24小时睡觉不准关灯，禁用手机，手也不能偷放进棉被里，所有费用全都得自行支付，每天租金约1000元（人民币，下同），据报，当年知名影星刘晓庆也曾被关在同一招待所。
 <br/>
 <br/>
 台媒《ETtoday》报导说，范冰冰逃税事件刚爆发时，她的经纪人穆晓光，本打算将责任全往自己身上揽，当时，还未被调查的范冰冰便透过各种管道，希望能在事情更严重前赶紧解决，没想到事件如滚雪球般越闹越大。
 <br/>
 <br/>
 报导说，当时范冰冰曾打电话给中共高层，并开出巨额酬劳，盼望对方能帮她脱离险境，但对方不愿意伸出援手，仅回答〝小范，你适可而止〞。之后，范冰冰遭到调查。
 <br/>
 <br/>
 至于这位中共高层是谁，报导中没有提示。
 <br/>
 <br/>
 范冰冰在公众视野里消失4个月后，中共官媒10月3日报导，税务机构查实，范冰冰及其旗下企业，累计偷税漏税2.55亿处罚5.96亿，滞纳金0.33亿，共需补缴8.83946亿。
 <br/>
 <br/>
 同日，范冰冰在微博发致歉信说，接受税务机构作出的处罚决定，将〝尽全力克服一切困难，筹措资金、补缴税款、缴纳罚款〞。
 <br/>
 <br/>
 10月4日，知名体育博主〝辣笔小球〞在个人微博上爆料称，范冰冰最初是在江苏南京德基20楼普慈堂被抓的。
 <br/>
 <br/>
 抓捕范冰冰的行动是北京警察主导，南京警察协助，警察从20楼给范冰冰戴上头套后，直接坐电梯去停车场。
 <br/>
 <br/>
 一同被捕的还有算命人沈爱栩和德基老板吴铁军。在抓获范冰冰后，南京警察直接开车将其押到北京。直到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_补税.html" target="_blank">
  补税
 </a>
 金额达成一致前，范冰冰一直被羁押在北京。
 <br/>
 <br/>
 〝辣笔小球〞是陆媒《经济观察报》驻上海记者，出生地是江苏南京。
 <br/>
 <br/>
 美国有线电视新闻网（CNN）称，范冰冰自6月以来就不曾露面，她的失踪震惊娱乐圈。此案明显是用来警告其他名人。
 <br/>
 <br/>
 《苹果日报》刊发评论文章说，中共官方想储子弹留待日后贸易战等庞大开支，故〝处处掠夺〞，反正范冰冰入狱亦无好处，故想借范来警告其他艺人及公司补交税款，填补国库。
 <br/>
 <br/>
 而央视原主持崔永元公开爆料范冰冰偷漏税这一事件，习近平当然不会错过，藉此整肃江派势力，令曾家等人交出娱乐圈控制权，因此暂时毋须将这一行业一棍打死。
 <br/>
 <br/>
 此外，范冰冰并非曾家圈子的核心人物，又是当红女星，她失踪期间，就受到不少国际传媒关注，对她下手，对娱乐圈起到了足够的警告作用。
 <br/>
 <br/>
 网民质疑，范冰冰偷漏税这么多，现在才要求补税，当地税务机关哪去了？上班睡觉去了？为什么没有一个税务官员被追责？这可要好好查一查。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
