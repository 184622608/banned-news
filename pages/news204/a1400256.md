### 意D&G广告惹〝辱华〞争议 中共官媒热炒被疑转移视线
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】意大利时尚品牌Dolce &amp; Gabbana的一段新广告，因被疑似嘲讽中国人用筷子吃食物的方式而被指〝辱华〞。其后网上又传出一张疑似D&amp;G设计师在社交媒体上大骂〝中国是屎〞的网络截图，致使事态急剧升温。不过，也有中国网友表示，广告虽有调侃意味但性质似乎并没有中共官媒所说的那么严重，中共热炒这个话题恐怕有藉机转移民众视线的意图。
 <br/>
 <br/>
 D&amp;G近日在社交媒体上发布了新的系列视频广告。这组题为〝起筷吃饭〞的新广告，原意大约想表达中国和意大利饮食文化相结合，但视频画面上华裔女模特儿笨拙地拿着筷子，用一些别出心裁的动作来吃意大利薄饼和意粉的模样，以及旁白者所使用的调侃的语气，引起了部分中国网友的反感，在中国大陆网络社交媒体上遭到舆论批评，被指有歧视中国传统饮食文化，嘲笑中国人不懂如何用正确的方式吃意大利美食的意思。
 <br/>
 <br/>
 中国大陆网民的留言中，有人批评广告中的模特儿的表演〝矫揉造作〞，有抹黑中国人之嫌；也有网友质疑旁白配音听起来就像是在〝恶搞〞，从画面到声音都让人感觉〝很不爽〞，认为这当中暗含着〝种族歧视〞或〝文化偏见〞。
 <br/>
 <br/>
 其间，网络上又传出一段疑似D&amp;G设计师在社交媒体上与其他网友争辩时，用英文写出〝中国是屎国，我们（Dolce &amp; Gabbana）没有你也活得很好〞等句子的截图，顿时让整个事态急剧升温。
 <br/>
 <br/>
 D&amp;G公司随后发表了紧急声明，声称〝品牌和Stefano Gabbana的IG帐号被盗〞，指那段辱骂中国的言论是被别有用心的盗号者所发。声明又称〝我们已透过法务部门进行调查。对于未经授权的贴文所引起的不愉快，我们深感抱歉，我们对于中国及中国人无比尊重〞。
 <br/>
 <br/>
 被盗号的设计师Stefano Gabbana随后也在Instagram再三声明称，〝那不是我，我的帐号被盗，现在找回来了。〞他还说：〝自己喜欢中国、喜欢中国文化，对发生的事情感到抱歉。〞
 <br/>
 <br/>
 但D&amp;G公司和Stefano Gabbana的解释在中国大陆网络上产生的影响可谓是〝杯水车薪〞，众网友抵制D&amp;G的呼声已经铺天盖地，而且中共官媒已经直接上阵，拿这个话题大作文章。
 <br/>
 <br/>
 中共喉舌媒体人民网和新华网都发表了一篇专门针对〝D&amp;G辱华〞广告的评论文章，以尖刻的言辞指责D&amp;G公司及其设计师〝侮辱〞其他国家文化的言行是〝对人类共同价值观的挑战〞；中共旗下的共青团中央则发微博威胁，外企来华投资兴业必须〝尊重中国〞，同时配发了一张写有〝自取其辱〞字样的图片。
 <br/>
 <br/>
 另有一些和中共保持立场一致的文人发文称，D&amp;G此次辱华事件是为了迎合千禧一代而不惜牺牲自己的〝底线〞，文章并大骂D&amp;G公司〝在辱华的边缘疯狂试探〞，〝最终导致价值观沦丧〞。
 <br/>
 <br/>
 D&amp;G原拟本月21日晚在上海举办一次规模不小的秀，按计划将有近400模特儿、40位明星和1500嘉宾参与。但在中国网络舆论的激烈反对声中，章子怡、李冰冰、陈坤、黄晓明、王大陆、迪丽热巴、王俊凯、陈学冬等艺人，陆续公开表态将杯葛D&amp;G的这场活动，甚至将继续抵制D&amp;G的任何产品。
 <br/>
 随后有消息指，中共文旅部已经下令，要求D&amp;G本月21日晚间的这场大秀取消。
 <br/>
 <br/>
 不过，也有中国网友表示，仔细看过D&amp;G的那段争议广告后，虽然感觉是有调侃贬低的意味，会令中国人感觉不爽，但中共官媒和水军乘机大打〝爱国牌〞来热炒这个话题，感觉就像前不久中共官媒公然炒作〝义和团〞一样，更像是因为在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 中吃了瘪而蓄意找点由头来炒作〝民族意识〞，通过煽动民众的仇外民族情绪来转移矛盾焦点。
 <br/>
 <br/>
 此前，中共党媒曾刊文大赞清朝〝义和团〞是反帝爱国运动，是〝民族意识觉醒〞，展现了百年前中国民众〝团结御侮的决心〞云云。
 <br/>
 <br/>
 当时中共官媒的这些论调，在海外就受到不少学者的批评。舆论认为，当年的〝义和团〞只不过是被清廷暗中利用的〝炮灰〞，现在中共陷入贸易战的泥沼无力自拔，于是搬出义和团精神来忽悠国人，试图鼓动诸如〝小粉红〞、〝自干五〞之流的〝爱国贼〞替中共当〝炮灰〞。
 <br/>
 <br/>
 事实上中共常常会利用某个事件，来煽动狭隘的民族主义情绪，转移国内民众的视线，这可以说是中共惯用的技俩。中共往往要对台湾发难时，就会炒作〝台独〞话题；要掩盖西藏侵害人权的真相时，就炒作〝藏独〞话题；要在香港挑事，就会炒〝港独〞话题；要与美国或西方打冷战，就炒作〝辱华〞话题，煽动仇外情绪。
 <br/>
 <br/>
 而自从今年4月美中双方发生贸易冲突以来，中国大陆的股市、汇市、进出口、固定资产投资和消费等纷纷下挫；被称为〝三驾马车〞的出口、消费和投资，全部都出现了断崖式的下降，这些都严重影响着中国经济的增长。
 <br/>
 <br/>
 外媒评论称，北京正在下一盘〝非常艰难的棋〞，因为贸易战已经使中国原本就很沉重的债务负担下更深层次的问题变得更加复杂。
 <br/>
 <br/>
 今年9月下旬，国际谘询机构〝中国褐皮书国际〞（CBB）公布的最新季度报告指称，自从贸易战开打以来，中国制造业的营收明显下滑，利润增速骤减。
 <br/>
 <br/>
 该报告并表示，在美国对中国输美的产品实施加征关税的措施之前，中国制造业本就已经陷入困境，而中国官方的统计数据也没有反映出企业借贷激增的现象。现在经受贸易战的打击后情况更糟，如果中美之间无法达成贸易协议
 <br/>
 <br/>
 与此同时，外媒也纷纷发现，美国政府在对中国进口商品加征关税时，反覆强调是为了打击中共政府长期以来采取的〝不公平贸易行为〞，包括盗取知识产权、强迫外资企业转让技术等等，但中共官方引导的舆论风向，却更多着重于反思中方的宣传策略是否恰当，而少提及〝不公平贸易行为〞的问题。这意味着，中共的根本基点并没有放在如何根本上来解决问题，而更着重的是挖空心思琢磨怎样才能继续忽悠美国和西方其他大国。
