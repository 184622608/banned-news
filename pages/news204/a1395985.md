### 港媒接北京〝特急〞通知 习近平动向引关注
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】香港媒体接到北京〝特急〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_采访通知.html" target="_blank">
  采访通知
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 23日在珠海举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开通仪式.html" target="_blank">
  开通仪式
 </a>
 。有消息称，这可能与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的动向有关，据传习近平19日至23日将南下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 ，出席港珠澳大桥开通仪式。另外，中共正在筹备四中全会，最快于20日召开。
 <br/>
 <br/>
 中共驻香港联络办公室17日下午向香港媒体发出〝特急〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_采访通知.html" target="_blank">
  采访通知
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 23日举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开通仪式.html" target="_blank">
  开通仪式
 </a>
 。采访地点在珠海。主办方还要求记者自费每晚600元人民币。获邀的嘉宾必须提前一日抵达珠海，并被安排入住一间指定的酒店。
 <br/>
 <br/>
 不过，对于外界关注的中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 是否参加开通仪式，通知未提及。
 <br/>
 <br/>
 甚至连港珠澳大桥正式通车时间亦成为〝机密〞，由于今次大桥的保安工作森严，正式通车会否与开通仪式同日进行，仍是未知之数。
 <br/>
 <br/>
 据了解，有香港人大代表已收到运房局邀请函，出席港珠澳大桥位的开通仪式。邀请函称，开通仪式将于23日上午举行，嘉宾会由香港乘船前往珠海，但没有透露主礼嘉宾。
 <br/>
 <br/>
 有港媒分析，从入住酒店的安排，都反映出保安级别极高，料与习近平会现身仪式有关。
 <br/>
 <br/>
 此前已多家港媒报导，习近平将于19日至23日南下广东视察，20日到访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中山大学.html" target="_blank">
  中山大学
 </a>
 ，并在23日担任港珠澳大桥通车仪式的主礼嘉宾。
 <br/>
 <br/>
 中山大学消息人士透露，习近平的确切行踪尚不清楚，该校目前确实正为习到访做相关准备工作。
 <br/>
 <br/>
 一位不愿具名的消息人士对自由亚洲说，他也听说习近平20日到广州，因中山大学有几个校区，珠海好像也有两个，现在还不知道习近平去哪一个校区。中山大学消息人士说，习将在校园与学生有一个见面会。
 <br/>
 <br/>
 香港01报导说，中共军方也密锣紧鼓进行筹备，经陆空运送大批军备抵港。有网民18日目睹两架运输机突然抵达香港机场，并卸下多辆军车。当日另有10多辆军车，在香港警方护送下，经三号干线浩浩荡荡驶出九龙。
 <br/>
 <br/>
 有军事专家认为，军方多以〝粮草先行〞的策略行事，估计是为习近平南下做先行准备。
 <br/>
 <br/>
 对于外界关注的港珠澳大桥向社会公众开通日期，中共国务院港澳事务办公室主任张晓明17日在北京与香港新闻界聚会时，私下披露大桥开通日期是10月28日，即是比通车仪式迟5天。
 <br/>
 <br/>
 耗资人民币1,200亿元的港珠澳大桥，全长约40公里，连接香港、珠海和澳门，该项工程历时近9年。舆论认为，习近平南下出席大桥通车仪式，除了表现中央对香港及广东经济发展的高度重视，也在纪念中共改革开放40周年。
 <br/>
 <br/>
 9月20日，媒体亦曾报导，习近平南下广州视察，并出席在深圳举办的改革开放40周年纪念大会。但相关消息至今也未获证实。
 <br/>
 <br/>
 习近平上一次考察广东的时间是2012年12月，在习5天视察的行程中，多次强调〝力推改革，坚持开放〞，说改革开放是〝决定中国命运的关键一招〞，此次习近平会留下什么政治语言，外界拭目以待。
 <br/>
 <br/>
 不过，多家港台媒体报导，目前中共正在秘密筹备第十九届四中全会，最快将于10月20日召开。日本首相安倍晋三将于25日访问中国，考虑到国内外的日程，预计四中全会会在安倍晋三访华前举行。
 <br/>
 <br/>
 鉴于上述时间点于习近平南下视察时间点上有冲突，相关消息是否属实仍是未知数。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：赵云）
