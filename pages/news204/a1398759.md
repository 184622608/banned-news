 <br/>

  西安民生
 </a>
 公司投资员工
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海航.html" target="_blank">

 </a>
 讨钱
 <br/>
 <br/>
 西安民生百货管理有限公司工会早前推介员工给海航集团集资，导致参与投资的员工血本无归，11月8号，数十名投资受害者，前往海航物流公司要钱无果，愤而当街拉横幅、喊口号抗议。
 <br/>
 <br/>
 2. 无锡多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 京沪遭截访
 <br/>
 <br/>
 因上海〝进博会〞和无锡〝双11〞国际马拉松开跑，多名无锡访民近期在北京和上海遭截访、失踪，其中，无锡抗强拆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 倪满良被拘留。
 <br/>
 <br/>
 3. 上海维权人士
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑监狱.html" target="_blank">
  黑监狱
 </a>
 探友 遭殴打
 <br/>
 <br/>
 11月8号，上海维权人士任迺俊和徐佩玲，前往看望因〝进博会〞被囚禁在中兴路1039号黑监狱的另一维权人士孙洪琴，结果，在黑监狱遭到看守殴打，报警后，警察不抓打人凶手，反而将他们带走关押。
 <br/>
 <br/>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">

 </a>

 <br/>
 <br/>
 709案辩护律师余文生被关押已经10个月，11月7号，他的妻子许艳和三位自聘的辩护律师一起，再次前往徐州看守所申请探视，再次遭到拒绝。而余文生案也被徐州检察院第三次延期起诉。
