### 【禁闻】美方推动川习会协议 中方静悄悄
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞在阿根廷开过之后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 官员已经多次发出推动双方实行协议内容的声音。不过北京方面静悄悄，党媒对中共做出的重大让步只字不提，所谓的海外中文媒体还一度出现批评习近平的文章。中共是否会兑现承诺，全球都在关注。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 有线电视新闻网CNN12月4号报导，白宫在周一更正，美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的〝90天缓冲期〞是从今年12月1号开始计时，而不是白宫首席经济顾问库德洛之前说的，从明年1月1号开始。
 <br/>
 <br/>
 上周六，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和中国国家主席习近平在阿根廷会晤，达成了〝90天缓冲期〞的协议。美国宣布暂缓从明年1月1号起提高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 。而中国同意从美国购买数量庞大的农业、能源、工业和其它产品。另外，两国将在90天内进行谈判，对中共强制技术转让、知识产权保护、非关税壁垒、网路入侵和网路盗窃等进行〝结构性改变〞。如果双方在期限内无法达成协议，美国仍将上调关税到25%。
 <br/>
 <br/>
 美方立即推动实施〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞的协议，财政部长姆努钦 (Steve Mnuchin)周一说，中国承诺会有一笔超过1.2万亿美元的额外贸易。而库德洛则表示，中国也承诺立即解除关税和非关税壁垒，包括取消美国农产品25％的惩罚性关税和降低对汽车的40％关税。他预计这些关税将降到零。
 <br/>
 <br/>
 不过路透社说，这些承诺并没有书面协议，具体细节也还没敲定。
 <br/>
 <br/>
 而北京方面静悄悄没有回应。党媒对中共做出的重大让步只字不提。
 <br/>
 <br/>
 美联社的报导说，有鉴于双方此前的贸易谈判进展缓慢，未来几个月的前景令人质疑。
 <br/>
 <br/>
 川普12月5号发推特说，〝看看与中国形成一个真实有效的交易是否真的可行，中国应该立即开始购买农产品。但如果不是这样，请记住我是一个关税人。 〞
 <br/>
 <br/>
 旅美时事评论员夏小强认为，大宗购买美国产品的承诺，中方应该会兑现。
 <br/>
 <br/>
 旅美时事评论员夏小强：〝其实购买美国的商品还没有触及到结构性的改变，因为这只是一个一次性购买的行动。这个我觉得中共随时都可以去做出兑现。〞
 <br/>
 <br/>
 但他认为，关键在于90天内，要中共真正承诺经济〝结构性改变〞是非常难的。
 <br/>
 <br/>
 夏小强：〝打比方说‘保护知识产权’，因为中共一直靠盗窃美国的高科技，和知识产权来获得经济生存的，一旦它失去这种方式，中共就没有自主创新的能力，它就很难有所发展。再比方说‘停止对国有企业补贴’，中国国有企业在世界自由贸易市场的竞争中，很快就会倒闭的。〞
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院的谢田教授认为，中共实际上做不到〝结构性改变〞。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝我觉得中共做不到。因为真正做到的话，那就意味着中共就放弃了对中国经济的控制。他们统治的经济基础就没有了。那时候都没有金钱支持的话，也没有这个外汇存底来支持的话，中共政权也就延续不下去了。〞
 <br/>
 <br/>
 在此之前，中共党媒多次高喊〝有信心〞、〝有能力〞打贸易战，但抵抗至今仍逃不开美方最初要求的〝结构性改变〞。有网友在推特上说：〝撒币国抵抗了半年多，目前全部放弃抵抗。还病死了一国的猪，整残了中兴和华为，弄废了无数交流学者和留学生。六元一斤的美国鲜猪肉不要，硬买十二元一斤的俄国瘟猪肉，这是什么精神？这是无知无畏的共产主义精神！〞
 <br/>
 <br/>
 夏小强认为，这次习近平亲自出面，说明他个人是想和美国达成一定的协议，缓解贸易冲突。但中国经济结构性的改变必然会动到一些利益集团的蛋糕，因此习近平很可能面临党内博弈。
 <br/>
 <br/>
 而总部设在北京的所谓海外华文媒体〝多维新闻〞，12月3号突然在旗下的《多维月刊》刊载一篇批评习近平的文章。但文章很快被撤下，将批评、质疑习近平的句子全部删除后，又重新上线。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制 /周天
