### 【禁闻】11月6日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 开幕 一男子市中心拦车队
 <br/>
 <br/>
 11月5号，中共举办的首届〝进博会〞在上海浦东会展中心开幕，一名男子在市中心交通要道，冲向路过的车队，准备拦车，但立即遭警察带走。视频拍摄者说，这名男子是金融投资受害难友。
 <br/>
 <br/>
 2. 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 上街举牌 责问警方
 <br/>
 <br/>
 上海维权人士杨秀婷今年3月在北京被警方绑架，之后被带回上海关黑监狱15天，期间食物被下毒。杨秀婷自由后，到上海市公安局杨浦分局要求公开她被关黑监狱时的询问笔录，但一直没有结果。最近她多次上街举牌责问警方：〝为何不敢公开？〞
 <br/>
 <br/>
 3. 合肥
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退休工人.html" target="_blank">
  退休工人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_市总工会.html" target="_blank">
  市总工会
 </a>
 维权 无果
 <br/>
 <br/>
 11月5号，安徽省合肥市的三名退休职工到市总工会要求与工会主席见面，通报企业破产、改制导致职工权利被损害的情况，要求市总工会出面，维护工人权利，但工会主席不肯接待他们。
 <br/>
 <br/>
 4. 湖北维权人士
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲍乃刚.html" target="_blank">
  鲍乃刚
 </a>
 被批准逮捕
 <br/>
 <br/>
 湖北省京山市维权人士鲍乃刚，因长期在网络上发表批评时政的言论，今年10月前往北京打工期间，被京山市公安局警察押回当地刑事拘留。11月3号又被以涉嫌〝寻釁滋事罪〞批准逮捕。
</div>
