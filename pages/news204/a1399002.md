### 佳士声援团遭算帐 北大校园现绑架案
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】深圳
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佳士.html" target="_blank">
  佳士
 </a>
 工人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_声援团.html" target="_blank">
  声援团
 </a>
 再度遭到当局打压，深圳警方继8月底在惠州抓捕50多名学生及工人后，11月9号晚，在广州、上海和北京多地再抓走十多人。其中一名北京大学毕业生在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北大校园.html" target="_blank">
  北大校园
 </a>
 内遭到暴力绑架，引发学生不安。
 <br/>
 <br/>
 北大历史学系2015级本科生的于天夫11月11号发文称，9号晚10点半左右，他在大学百年讲堂路口遭到至少5名黑衣人殴打，看到北大药学院2014级的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张圣业.html" target="_blank">
  张圣业
 </a>
 还被拽上他们的汽车。张圣业是〝寻悦〞行动的发起者之一。〝寻悦〞是寻找北大校友顾佳悦的意思，她因声援深圳
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佳士.html" target="_blank">
  佳士
 </a>
 工友失踪。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳佳士.html" target="_blank">
  深圳佳士
 </a>
 工人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_声援团.html" target="_blank">
  声援团
 </a>
 成员孟先生:〝张圣业在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北大校园.html" target="_blank">
  北大校园
 </a>
 里，那些保安不起作用，保护不了，听说是被广东的带走。〞
 <br/>
 <br/>
 文中还提到，容许黑社会行为发生的，竟然是负责保卫北大校园的保卫部以及相关责任方，他们大摇大摆从学校南门驶出，没有任何的阻拦。
 <br/>
 <br/>
 孟先生:〝北大有一个学生写了一个当天晚上的事情，然后北大保卫部发出一个通报，说呢他是警方抓这个涉嫌犯罪的人员，是校外人员，不是北大的学生和老师。〞
 <br/>
 <br/>
 继8月底，深圳警方在惠州大肆抓捕50多名学生及工人后，11月9号晚上，分别在广州、上海和北京多地带走十多名佳士声援团成员。
 <br/>
 <br/>
 大陆网友李先生:〝我们现在没听说有活动，以前有早就抓走了，当地的警方他们不会讲，我们老百姓更不知道，这些东西都是不可能对外公开的。〞
 <br/>
 <br/>
 今年7月，深圳佳士科技工人争取成立工会，多名工人被捕，获得北大、清华等高校学生声援，但声援团活动遭到打压，各地被抓和失踪的至今已有几十人，其中核心成员沈梦雨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被失踪.html" target="_blank">
  被失踪
 </a>
 三个月、岳昕被失踪两个月。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
</div>
