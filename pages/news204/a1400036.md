### 中美贸谈生变 传北京最后1分钟做绝望决定
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】为了给11月底的〝川习会〞铺路，中美双方近日恢复了接触。但中方向美方递交了一份贸易清单，美方仍不满。加之刚闭幕的APEC峰会上中美激烈交峰。在这种氛围之下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易.html" target="_blank">
  中美贸易
 </a>
 谈判恐怕难有作为。消息称，中方在绝望之下，在最后一分钟取消了原定于华盛顿举行的贸易商谈。
 <br/>
 <br/>
 港媒《南华早报》11月19日称，在北京与华盛顿方面进行了〝工作层面的讨论〞之后，中方经贸负责人，中共国务院副总理刘鹤，将率经贸团队赴阿根廷，而不是华盛顿进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易.html" target="_blank">
  中美贸易
 </a>
 谈判。
 <br/>
 <br/>
 报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中方代表团.html" target="_blank">
  中方代表团
 </a>
 最后一分钟取消赴华府商谈，双方将直奔第三方国家阿根廷谈判，并由〝川习会〞定夺，美中贸易战前途未卜。
 <br/>
 <br/>
 报导18日引述一位消息人士表示，〝最新计划是两国贸易谈判团队在布宜诺斯艾利斯会面。〞 这一变化的部分原因，是为了确保会谈，能够对11月底在阿根廷的G20峰会期间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 与美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）的会晤产生直接作用。
 <br/>
 <br/>
 报导说，如果消息得到确认，这一变化表明，中美首脑会晤的分量将提升。〝川习会〞期间或有更多重大的事项提上日程。这也是两个最大经济体的领导人，首次在第三国进行双边会晤。
 <br/>
 <br/>
 11月30日至12月1日，G20峰会在阿根廷布宜诺斯艾利斯登场。这将是美中贸易代表首次在第三国商谈，会谈进展将直接影响〝川习会〞。
 <br/>
 <br/>
 美中经贸代表团都未公开新一轮磋商细节，但总统川普16日披露，中共递交了一份很大的清单，回应美国对贸易改革的要求，有很多我们要求的东西，但是还有4、5件事被漏掉了。
 <br/>
 <br/>
 川普说，中共想要达成协议，〝但是，我仍然认为不可接受。我认为，我们在中国方面做得非常好。〞中美贸易不公平，中方对美国征收了巨额的关税，而美国没有对中方征收，现在一切都将改变。
 <br/>
 <br/>
 《路透社》引述一名美国官员透露，中共向美国递交的清单有142个项目，但其中并没有包括白宫的核心要求，例如解决强迫技术转移和偷窃知识产权等问题。
 <br/>
 <br/>
 一名消息人士透露，中方给的东西并不是什么新的，双方仍处在僵局之中。通道现在再次打开了，但是仍有很多工作要做。
 <br/>
 <br/>
 在这种氛围之下，可以想像刘鹤的美国之行恐怕仍将难有作为，因此，双方也只好把谈判桌从华盛顿搬到布宜诺斯艾利斯由川普与习近平定夺。
 <br/>
 <br/>
 17至18日的亚太经合组织峰会（APEC）期间，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）表示，中美两国目前的分歧始于贸易问题、关税及配额，以至强制技术转移、窃取知识产权、航行自由及人权问题，而习近平回应说，中美对话十分重要。
 <br/>
 <br/>
 彭斯在即将回国时18日对媒体透露，G20峰会期间，川普和习近平将讨论5大议题，包括：中共的不正当贸易行为的关税壁垒、强制要求的技术转移、侵犯知识产权、航行的自由和人权问题。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
