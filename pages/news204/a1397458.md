### 川普说美中将有很好的协议 新关税也已备下
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美首脑.html" target="_blank">
  中美首脑
 </a>
 将在阿根廷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上会晤之际，总统川普（特朗普）预期美中两国能就贸易问题谈成〝很好的协议〞，但如果会谈失败，他将对2,670亿美元的中国商品加征关税。
 <br/>
 <br/>
 20国集团会议将于11月30日至12月1日在阿根廷布宜诺斯艾利斯登场，届时川普和中国国家主席习近平可望藉此场合会面。
 <br/>
 <br/>
 川普10月29日晚间接受福克斯新闻（Fox News）采访时说，有可能与中方就贸易达成〝重大交易〞。他同时也警告说，如果有必要，还准备对中国输美产品征收数十亿美元的额外关税。
 <br/>
 <br/>
 川普说：〝我们即将与中国达成很棒的协议，必须要这样，因为是他们把我们国家掏空了。如果我们达不成协议，我准备再对中国2670亿美元的输美产品征税。〞
 <br/>
 <br/>
 10月21日，美国Axios新闻网站引述一名了解川普想法的消息人士称，川普认为他的关税持续的越长，他就会拥有更多的筹码。
 <br/>
 <br/>
 一名熟悉川普谈话的消息人士称，川普同中共的贸易战，还处在〝开始的开始〞。川普的团队并不对暂定于11月的G20〝习川会〞抱有太大的期待。
 <br/>
 <br/>
 彭博社说，如果在11月底的20国集团峰会期间，川普与习近平谈不拢，那么川普就将在12月初，开始对中国输美产品征收新关税。根据一贯在宣布的60天后实施的做法，美国的新一轮关税措施将在2019年2年初中国黄历新年前后实施。
 <br/>
 <br/>
 《华尔街日报》引述中美官员的话说，华府已经要求中方就强制技术转移及其他中美经贸议题，交出具体的解决及让步方案，否则美方拒绝重新展开贸易谈判。
 <br/>
 <br/>
 报导称，中方把美国提出的要求分为142项，并表明可以谈判其中122项，余下则因涉国家安全而拒绝让步，但没有进一步说明如何处理每项要求。
 <br/>
 <br/>
 美方官员形容这属〝概念〞，不足以令谈判有进展，要求中方提交详细及具体的方案。
 <br/>
 <br/>
 多名消息人士称，美财长团队对中共说，谈论购买更多的美国产品没有意义，因为关键的优先议题必须解决，如偷窃知识产权和市场准入权等结构性的问题。消息人士还称，川普并不为中共的报复措施感到担心，他准备全力出击。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明)
