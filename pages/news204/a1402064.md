### 福州数百黑衣保安进村逼迁  打伤村民
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】福建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福州市.html" target="_blank">
  福州市
 </a>
 盖山镇浦下村
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旧城改造.html" target="_blank">
  旧城改造
 </a>
 项目，正好位于市政府的江南中央商务区项目拆迁范围 ，因为赔偿不公，拆迁遭到村民抵制。11月29号，几百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑衣保安.html" target="_blank">
  黑衣保安
 </a>
 闯进村民家中打人，几名村民被打得头破血流。
 <br/>
 <br/>
 浦下村村民12月5号告诉《新唐人》，11月29号早上，几百名穿黑色制服、手持盾牌、头戴钢盔的保安闯进浦下村，村民都站在自家门口，或爬上楼顶，防止他们
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 ，一名妇人被强行拖倒在地，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑衣保安.html" target="_blank">
  黑衣保安
 </a>
 靠近房子时，遭到村民扔砖块反击，这些保安直接冲到村民家中，见人就打，几名村民被打得头破血流。一辆过路的游览车司机也被打破头，鲜血直流。
 <br/>
 <br/>
 现场视频：〝把我们打得这么厉害，打得这么厉害，太狠了，黑鬼来打得这么厉害，强拆我的房子，黑鬼把这里，你看出不去了，还打人呀，打我们老百姓啊，叫我去派出所，怎么走？〞
 <br/>
 <br/>
 11月30号，村里被断电、断水，数百名村民到拆迁办讨说法、围堵副镇长，结果来了二、三百名特警护送副镇长离开。
 <br/>
 <br/>
 浦下村村民刘先生：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旧城改造.html" target="_blank">
  旧城改造
 </a>
 啊，拆迁那个都是城管很多，叫城管和保安先打老百姓，这是他地方政府的政策嘛。〞
 <br/>
 <br/>
 该村的房子大都是无产权或小产权的房子，当地政府以违建的名义进行强拆，补偿非常低，给外地村民的赔偿更低，而大部分村民都是外地户口，租约未到期的租户则没赔偿 ，因此老百姓都怨声载道。
 <br/>
 <br/>
 浦下村工厂老板颜先生：〝这个拆迁，搬迁好多好多，我们工厂是租赁租在那个村，他们，他们现在搬迁，我们就工厂搬走就是了，你没办法的事情。拆迁一拆的话，达不到他们要求的，有的赔偿都没有的话，当然他就愁了，这很正常。〞
 <br/>
 <br/>
 此前在7月17号，也有上千名黑衣保安闯进村中，与村民发生冲突，6名村民被打伤。
 <br/>
 <br/>
 公开资料显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福州市.html" target="_blank">
  福州市
 </a>
 政府进行江南中央商务区建设，浦下旧改项目，正好位于这个地块内，该项目征收浦下村集体土地16.67亩，用途为商业用地、二类居住用地及其配套。
 <br/>
 <br/>
 新唐人记者 熊斌、钟元采访报导
 <br/>
</div>
