### 美媒：川普向习近平透露一消息 中方惊讶
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】在阿根廷举行的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞上，中美双方同意贸易冲突休战90天。在未来的90天内，中美将就双边贸易结构性改革达成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_协议.html" target="_blank">
  协议
 </a>
 。在这一谈判中，美方团队将由贸易代表莱特希泽主导。据消息人士透露，当
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在布宜诺斯艾利斯会晤中告知习近平这个决定时，让中方相当惊讶。
 <br/>
 <br/>
 近日，美国白宫贸易顾问纳瓦罗（Peter Navarro）在接受美国国家公共电台（NPR）采访时表示，美国贸易代表莱特希泽将在90天时间内与中国方面进行贸易谈判。
 <br/>
 <br/>
 纳瓦罗说：莱特希泽是美国贸易代表办公室有史以来最强硬的谈判人。他将按照章节原原本本地降低关税，减少非关税壁垒，并结束妨碍市场准入的所有这些结构性做法。
 <br/>
 <br/>
 12月3日，《华尔街日报》引述熟悉决定的消息人士说，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）任命对华强硬派、美国贸易代表莱特希泽负责在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞后与北京谈判。这一个决定，在布宜诺斯艾利斯的会晤上，川普已经通知了习近平，这让中国方面惊讶，因为他们一直主要是和美财长努钦（Steven Mnuchin）打交道。
 <br/>
 <br/>
 过去数月以来，北京一直在同努钦打交道。努钦同金融市场关系密切，他一直在努力寻求解决方案。但中美前四轮贸易谈判均无果。
 <br/>
 <br/>
 而莱特希泽则一直施压对中方征收更多的关税，让北京做出改变，以解决两国产生分歧的根本性问题。 报导称，努钦和莱特希泽曾就如何与北京打交道才最好爆发过冲突。
 <br/>
 <br/>
 这位消息人士说，选择莱特希泽给了川普更多信心，美国将对北京采取更加强硬的路线。在莱特希泽的任命上，白宫顾问、川普的女婿库什纳（Jared Kushner）在劝说川普时发挥了重要作用。
 <br/>
 <br/>
 美国哈德逊研究中心中国战略研究主任白邦瑞（Michael Pillsbury）说，这个绝对给对话设置了一个全新的方向。
 <br/>
 <br/>
 英媒《路透社》分析认为，莱特希泽领导谈判标志着川普政府改变了此前的与中方贸易谈判方式。莱特希泽是一位经验丰富的贸易谈判代表，刚刚与加拿大和墨西哥达成新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_协议.html" target="_blank">
  协议
 </a>
 ，是川普政府中最具声望的中国批评人士之一。
 <br/>
 <br/>
 莱特希泽说，如果中国在90天后未能提出令人满意的条件，那么关税税率将再次上涨25%。
 <br/>
 <br/>
 阿根廷当地时间12月1日晚，习近平和川普在布宜诺斯艾利斯举行会晤。当天川普率领9人的顾问团队与习近平会晤，团队中除白宫贸易鹰派顾问纳瓦罗（Peter Navarro），还包括莱特希泽。
 <br/>
 <br/>
 莱特希泽和纳瓦罗都被认为是鹰派中的鹰派人物。纳瓦罗曾出版《即将到来的中国战争》（The Coming China Wars）和《致命中国》（Death by China）等书，并对中共持强烈批评态度。他是中美关税战的发起者之一。因此两人的加入会晤，被认为意在向北京方面展示川普政府的决心。
 <br/>
 <br/>
 根据白宫在会晤后发布的声明，习近平承诺减少中美间的贸易不平衡，并将从美国采购大量农产品等产品。川普则同意2019年1月1日暂时对2,000亿美元中国进口商品的关税维持在10%。但如果双方不能在90天内就双边贸易的结构性改变达成协议，对这批商品的关税仍将提高到25%。
 <br/>
 <br/>
 这意味着对中方来说，虽然获得了90天的〝缓刑〞时间，但是仍然是祸福难料，因为美国方面换上了一位更加强硬的官员负责这90天的谈判。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：范铭）
