### 妻子并非彭丽媛妹妹 吉利李书福间接回应传闻
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】中国最大的民营汽车制造企业
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉利汽车.html" target="_blank">
  吉利汽车
 </a>
 早年间曾得到时任浙江省委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的大力扶持。近年来外界流传着一种说法，指吉利汽车的创办人兼董事长李书福的妻子是名为彭丽鹃，系习近平妻子彭丽媛的妹妹。最近李书福在联交所申报一项相关权益时公开了妻子的姓名，显示其妻子并不姓彭，间接否定了前述传闻。
 <br/>
 <br/>
 据香港联合交易所的最新股权披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉利汽车.html" target="_blank">
  吉利汽车
 </a>
 的主席李书福今年10月10日申报与妻子共同持仓，涉及490万股股份的权益，在场外平均代价为每股13.6741元人民币，持股量由44.41%增至44.47%。
 <br/>
 <br/>
 消息称，有关权益为好仓、非上市衍生工具-以现金交收，如相关股份价格高于某个水平，有权向另一人收取款项。
 <br/>
 <br/>
 外界注意到，这项申报者提及的相关权益共同持有人李书福的妻子名为Li WANG，并不姓彭。吉利汽车的发言人日前更向美国彭博社表示，公司正〝试图纠正
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李书福妻子.html" target="_blank">
  李书福妻子
 </a>
 是中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 夫人姊妹的谣言〞。
 <br/>
 <br/>
 据了解，滙丰银行旗下杂志〞Week in China〞2017年7月曾在一份报导中声称，〝眼尖的网民发现李书福妻子的名字和习近平妻子彭丽媛妹妹的名字重名（彭丽娟）。这种巧合听上去无关紧要，但是对一些人而言，这正支撑了吉利得到高层支持的说法。〞
 <br/>
 <br/>
 上述说法引发外界的猜测，认为李书福很可能与习近平是姻亲关系。
 <br/>
 <br/>
 据公开的资讯，李书福1963年6月25日生于浙江省台州，1997年创办了中国民营汽车制造企业吉利汽车并担任董事长。经过30年的发展，吉利集团从汽车行业的无名小卒发展到中国唯一一个进入世界500强的民营车企。2016年李书福以305亿财富排名胡润百富榜第50位。
 <br/>
 <br/>
 《德国之声》今年2月6日曾报导称，习近平2002年刚刚上任浙江省委书记时曾参观了吉利公司，并当场表态称：〝像吉利这样的企业，如果我们不加以大力扶持，那我们还去扶持谁呢？〞
 <br/>
 <br/>
 从2006年以后，吉利集团开始在海外展开了一系列收购行动：2006年，浙江吉利控股集团在海外投资，成为了英国出租汽车生产商锰铜控股的最大股东；2010年，浙江吉利控股集团收购瑞典沃尔沃汽车公司；2017年，收购马来西亚宝腾汽车49.9%的股份以及莲花汽车51%的股份。
 <br/>
 <br/>
 2017年11月13日，宣布与美国Terrafugia飞行汽车公司达成最终协议，将收购Terrafugia的全部业务及资产，并由吉利控股集团总裁高级顾问余宁担任董事长。同年12月27日，浙江吉利控股集团与欧洲基金公司CevianCapital达成一致，将斥资32.5亿欧元、约39亿美元收购其持有的沃尔沃集团股票，项目交割后，吉利控股将持有沃尔沃集团8.2%的股权，成为其第一大股东。
 <br/>
 <br/>
 2018年2月24日戴姆勒披露，中国吉利汽车董事长李书福透过一家投资基金Tenaclou3 Prospect Investment Ltd，入股戴姆勒的9.69%股权。以市值计，该批股权的价值为90亿美元（75亿欧元），成为单一大股东。
 <br/>
 <br/>
 （记者欣然综合报导/责任编辑：明轩）
