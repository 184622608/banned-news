### 网曝刘强东案受害女涉嫌诬陷罪遭收押 美警回应
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东.html" target="_blank">
  京东
 </a>
 的创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 在美国涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 一案仍在调查中，16日网络消息称，刘强东案涉案女生因为涉嫌诬陷罪已经被当地警方刑拘，随后，美国明尼阿波利斯市警方发言人对此做出了回应。
 <br/>
 <br/>
 10月16日网络消息称，美国明尼苏达州警方称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 案涉案女子因为涉嫌诬陷罪已经被当地警方刑拘。
 <br/>
 <br/>
 对此，陆媒记者向美国明尼阿波利斯市警方查询，警方发言人约翰・埃尔德（John Elder）回复称，该消息不属实，目前亨内平县检方仍未公布是否会对刘强东提起诉讼。
 <br/>
 <br/>
 明尼苏达亨内平县警方9月21日宣布，警方已完成对刘强东案件的初步调查，接下来将由检方接手，审查案件全部证据，并最终决定是否起诉。
 <br/>
 <br/>
 根据美国警方9月4日公布的报告，刘强东涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 ，发生在8月31日凌晨1时，当天刘强东被捕入狱，16个小时后获释，0保释金。
 <br/>
 <br/>
 警方对刘强东一案的描述是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一级性犯罪.html" target="_blank">
  一级性犯罪
 </a>
 ：强奸既遂〞（CSCR-1,CSC-RAPE-COMPLETE）。据警察局的拘留记录，刘强东的状态为〝释放有待正式控告〞。 根据明尼苏达州的法律，一级强暴罪只要最后被定罪，刑期至少12年起。
 <br/>
 <br/>
 有美媒体分析说，如今唯一能够帮刘强东脱罪的，就是女方承认当晚的行为是自愿的。否则，刘强东被判强奸罪成立的概率很大。
 <br/>
 <br/>
 有华裔刑案律师认为，刘强东遭起诉的可能性超过90%，案件一旦开始会涉及重新逮捕等问题。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：戴明）
