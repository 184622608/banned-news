### 【禁闻】85岁老母发视频 吁国际营救黄琦
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四天网.html" target="_blank">
  六四天网
 </a>
 〞创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 的母亲最近在网络公开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视频.html" target="_blank">
  视频
 </a>
 ，呼吁国际社会营救她的儿子，敦促中共政府将黄琦释放。黄琦由于公开访民上访情况在2016年底被捕，在狱中遭到虐待，身体健康恶化。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四天网.html" target="_blank">
  六四天网
 </a>
 〞创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 母亲蒲文清：〝从小到大我都没打他一次，他却在里面受到如此的殴打、虐待。我心里非常非常的想念我儿子，因为我儿子是为广大的中国老百姓说真正的话，说出他们的冤情。他没有罪。〞
 <br/>
 <br/>
 〝六四天网〞创办人黄琦的母亲蒲文清近期在网络公开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视频.html" target="_blank">
  视频
 </a>
 ，呼吁外界关注黄琦。
 <br/>
 <br/>
 现年55岁的黄琦是四川人，1999年创办〝六四天网〞网站，2006年改组为〝中国天网人权事务中心〞，近20年来发布了大量中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弱势群体.html" target="_blank">
  弱势群体
 </a>
 的维权新闻。黄琦因此长期受打压，两次被判刑。
 <br/>
 <br/>
 蒲文清：〝他以前是经商的，我们家庭条件很好的。他由于看见目前的现状，他就把所有的家产都变卖了，跟他爱人到成都成立一个寻人事务所。无私的帮助被拐卖的妇女儿童。以后就相继成立六四天网，就为弱势群体发声。〞
 <br/>
 <br/>
 2016年11月28号，黄琦再次被警方带走，被指控〝非法为境外提供国家秘密〞关押至今。
 <br/>
 <br/>
 蒲文清：〝本身他们所谓的洩露国家秘密这个文件，实际上是陈天茂一个普通访民信访诉求办理情况记载。他们列为国家机密的文件，这是捏造，陷害我儿子，让我儿子入狱，至今已经快两年了。〞
 <br/>
 <br/>
 此前，黄琦的代理律师李静林曾对《自由亚洲电台》透露，对黄琦所谓涉密的指控，源于绵阳公安局前警察陈天茂，因在执行任务时发生车祸致残，遭公安局解除工作，工伤处理不公，陈天茂多次上访。
 <br/>
 <br/>
 2016年4月，陈天茂将政法委关于其上访的一份报告公开，经〝六四天网〞报导后，引起海外媒体跟进，因而成为检方指控黄琦的所谓〝罪证〞。
 <br/>
 <br/>
 黄琦前代理律师隋牧青：〝很明显不成立，它定的时候都不是所谓的机密，是他们发布以后，他们才定为机密，有关机关要打击黄琦这样的一个说法，就成了国家秘密，这种构陷迫害太过于赤裸裸。〞
 <br/>
 <br/>
 六四天网义工蒲飞：〝当局拒绝改变他们的错误做法，就是黄琦从事的维权工作触动了当局的一些基本利益吧！〞
 <br/>
 <br/>
 而陈天茂及维权人士杨秀琼，也和黄琦一起被指控〝非法为境外提供国家秘密〞合并案。但日前四川绵阳当局又将合并案拆成两起，重新起诉。陈天茂的罪名减轻，杨秀琼却不在重新起诉书中。
 <br/>
 <br/>
 而黄琦的罪名不但没有变化，他的辩护律师之一刘正清上周还接获绵阳市检察院通知，称将对黄琦追加变更起诉，加控他〝故意洩露国家机密罪〞，并称他因多次触犯相关罪名，所以应当〝从重处罚〞。
 <br/>
 <br/>
 在近两年的关押中，黄琦多次传出遭到虐待。
 <br/>
 <br/>
 蒲文清：〝我儿子身患绝症，在看守所内受到虐待，不给钱给他买（卫生）纸，连揩屁股都没有卫生纸。干部支持里边的刑犯毒打黄琦，每天毒打几次。〞
 <br/>
 <br/>
 黄琦患有脑积水、心脏病、肺气肿和肺炎等疾病。但当局拒绝让他保外就医。
 <br/>
 <br/>
 蒲文清：〝我儿子目前在狱中血压升高、全身浮肿酸痛，肌酐、尿酸超正常值一倍以上，肾功能衰竭由三期、四期，我儿子进入尿毒症了。如果得不到透析，我儿子会死在狱中。〞
 <br/>
 <br/>
 黄琦案引发国际关注，蒲文清上周到达北京。驻北京的英国、德国及瑞典外交官向蒲文清了解案情进展。此后蒲文清向最高检察院和最高法院递交信访材料，但都遭到拒绝。
 <br/>
 <br/>
 蒲文清：〝我85岁了，我活到哪一天是说不准的。但我死去的时候我见不到（儿子）怎么办哪？政府这样做事，我们‘一案两命’。我请求国际社会组织促进中国释放我无罪的、患绝症的儿子，回来与妈见一面。〞
 <br/>
 <br/>
 蒲文清呼吁国际社会关注黄琦，敦促中共政府无罪释放他。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/陈建铭
