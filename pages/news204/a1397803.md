### 内蒙公安厅副厅长自缢 死前见了傅政华
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】11月1日，内蒙古自治区公安厅副厅长、呼和浩特市副市长、公安局局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李志斌.html" target="_blank">
  李志斌
 </a>
 在办公室
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自缢身亡.html" target="_blank">
  自缢身亡
 </a>
 。据称，在李志斌自杀前，内蒙古公安、政法系统接连遭清洗，多名高官落马。就在他自杀的前几天，见了中共司法部部长傅政华。
 <br/>
 <br/>
 公开资料显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李志斌.html" target="_blank">
  李志斌
 </a>
 1967年出生，仕途一直在内蒙古。曾先后担任过包头市公安局副局长、党委副书记，赤峰市副市长、市公安局党委书记、局长等职务。
 <br/>
 <br/>
 2016年1月起，李志斌担任内蒙古自治区公安厅党委委员、副厅长，呼和浩特市副市长、市公安局党委书记、局长、督察长。
 <br/>
 <br/>
 李志斌最后亮相是在10月28日至29日，他陪同中共司法部部长傅政华、副部长熊选国等人在呼和浩特市视察工作。
 <br/>
 <br/>
 值得一提的是，李志斌自杀前的一周内，内蒙古公安、政法系统多名官员密集落马，而这些人均与内蒙古公安厅前厅长赵黎平有关联。
 <br/>
 <br/>
 2017年5月26日，赵黎平以故意杀人、受贿等多项罪名被执行死刑，是中共〝十八大〞后首个被处死的高官。
 <br/>
 <br/>
 赵黎平被处死后，内蒙古公安系统强烈地震，有多名高官接连落马。
 <br/>
 <br/>
 10月30日，内蒙古自治区纪委监委发布消息，内蒙古自治区公安厅原党委委员、副厅长孟建伟涉嫌严重违纪违法被查。
 <br/>
 <br/>
 10月29日，内蒙古自治区呼和浩特市公安局党委委员、副局长杜宝君，也因涉嫌违纪被捕。
 <br/>
 <br/>
 10月25日，内蒙古人大常委会原副主任邢云涉嫌严重违纪违法接受调查。邢云曾担任包头市委书记、内蒙古政法委书记等职。
 <br/>
 <br/>
 邢云是赵黎平的〝老上级〞，而孟建伟和杜宝君都是赵黎平的旧部。
 <br/>
 <br/>
 内蒙古是江派前常委刘云山的发迹地，多名落马官员都与刘云山有关联。同时，内蒙政法系统还积极跟随江泽民集团迫害当地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员，孟建伟、赵黎平、邢云、李志斌等人都在海外追查迫害法轮功国际组织的追查名单上。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
</div>
