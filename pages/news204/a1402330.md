### 盘点任正非的三任妻子 最小的比孟晚舟还小
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创始人任正非的女儿、华为副董事兼首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 ，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 被捕事件发酵之际，任正非也遭起底。陆媒盘点任正非的三任妻子，最小的是80后，比孟晚舟还年轻。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 政府以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 涉嫌违反美国对伊朗的出口禁令，日前逮捕了华为创办人任正非的长女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 。孟晚舟是华为副董事长、全球首席财务官。孟晚舟一案7日举行保释听证，美国正寻求将她引渡回美国受审。
 <br/>
 <br/>
 孟晚舟是任正非与第一任妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟军.html" target="_blank">
  孟军
 </a>
 所生的长女，毕业于华中理工大学，拥有硕士学位，1993年进入华为工作，1997赴华中理工大学读硕士，一年半后重新回到华为。
 <br/>
 <br/>
 孟晚舟在华为蛰伏了20年，直到2013年1月21日，她才以华为首席财务官（CFO）身份首度亮相于公众面前，至此，孟晚舟的名字才真正和任正非女儿的身份联系起来。
 <br/>
 <br/>
 陆媒称，任正非一共有三任妻子，分别是孟军，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚凌.html" target="_blank">
  姚凌
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏薇.html" target="_blank">
  苏薇
 </a>
 。任正非第一任妻子孟军育有一女一子，孟晚舟和孟平。孟晚舟随母姓。
 <br/>
 <br/>
 网传孟军的父亲孟东波曾经担任过副省长，孟东波的领导杨超则担任过中共国家领导人的秘书，任正非正是靠第一个老婆孟军的父亲飞黄腾达。
 <br/>
 <br/>
 1982年中共大裁军，孟军来到深圳做了南油集团的高管，任正非也来到深圳，做了下属一家电子厂的领导。坊间传闻，任正非擅用职权，玩弄厂里的漂亮年轻女性。
 <br/>
 <br/>
 在公开场合，任正非称，两人是因理念不合而离婚。而内部传闻称，出身高干家庭的孟军无法忍受丈夫的风流成性，断然离婚。
 <br/>
 <br/>
 任正非的第二任妻子叫姚凌，是任正非的秘书，比任正非小十几岁。在昆明生下了女儿姚安娜 (Annable Yao)，直到 Annable Yao8个月大的时候，才来到深圳，与任正非一同生活。Annabel Yao现年20岁，因今年参加巴黎名媛舞会首次亮相。姚安娜也是随母姓。
 <br/>
 <br/>
 任正非的第三任妻子也是任的秘书，是80后美女苏薇，比任正非小几十岁，比孟晚舟还要年轻。据悉，苏薇是是四川人，成都电子科大的硕士。大陆知名品牌营销专家李光斗称，目前任正非在深圳的饮食起居，都由苏薇照顾。
 <br/>
 <br/>
 此次孟晚舟的被捕事件，据称对现年74岁的任正非构成打击。任正非30多年前创办了华为。孟晚舟被视为华为的潜在接班人选。
 <br/>
 <br/>
 3月底，任正非卸任副董事长，由孟晚舟接任。孟晚舟在公司内的职级、地位与一直被视为任正非接替者人选的三名轮值董事长接近。
 <br/>
 <br/>
 华尔街日报引述一位知情人士称，孟晚舟在香港天通科技公司（Skycom Tech）任职期间的行为，现正受到美国政府的调查。香港的企业纪录显示，她在2008年2月成为Skycom Tech的董事，次年4月辞职。
 <br/>
 <br/>
 孟晚舟2007年担任了华为旗下一家控股公司的董事会秘书。旗下有香港天通科技公司（Skycom Tech），后者在伊朗有业务，其雇员曾表示他们为〝华为-Skycom〞工作。
 <br/>
 <br/>
 华为作为全球最大电信网路设备制造商之一，自2016年以来一直受到美国司法当局的暗中调查，认为其违法了美国的出口和制裁法律。而孟晚舟这次能否脱身，将攸关华为的未来命运。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
