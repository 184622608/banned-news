### 北京锋锐律所遭注销执业许可证
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】北京锋锐律师事务所日前被北京司法局注销〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_执业许可证.html" target="_blank">
  执业许可证
 </a>
 〞。锋锐律师事务曾经代理大量敏感案件，在2015年的709事件中，该所多名律师被抓。
 <br/>
 <br/>
 锋锐律师事务所合伙人刘晓原告诉《新唐人》，今年3月，北京市司法局吊销了锋锐律师事务所执业证。11月，司法局再发出文件，表示已注销锋锐的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_执业许可证.html" target="_blank">
  执业许可证
 </a>
 。
 <br/>
 <br/>
 锋锐律师事务所合伙人刘晓原：〝 11月9号经朝阳区司法局报经北京市司法局审查批准，注销了北京锋锐律师事务所的职业许可证。这也就意味着北京锋锐律师事务所已经不存在了。〞
 <br/>
 <br/>
 刘晓原表示，〝吊销〞执业证，这个律所还存在，只是不能从事任何法律业务，而〝注销〞执业许可证就意味着律所不存在了。北京市司法局声称锋锐律所疏于管理，以致于有律师涉嫌犯罪、被定罪，所以做出了这些动作。
 <br/>
 <br/>
 〝锋锐〞由律师周世锋创办，最初只是个人律师事务所，到后来改为合伙制，全盛时期旗下有60多名律师，曾代理法轮功、 三鹿毒奶粉等敏感案件。2015年〝709大抓捕〞中，事务所至少有9名律师及行政人员被捕。
 <br/>
 <br/>
 刘晓原透露，2015年7月大抓捕事件发生后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财务.html" target="_blank">
  财务
 </a>
 帐本、办公的档案全部被扣押，银行帐户被查封，锋锐律师事务所其实已经名存实亡了。
 <br/>
 <br/>
 刘晓原：〝没调离的律师都不能正常的办理案件，特别是新的案件。你要接受一个委托要签订协议，也要收律师费，但是都不可能进行了，因为财务、办理案件的相关材料全部被扣押。 2015年12月底要交2016年的办公租房的租金，已经没有任何经济能力交，租房都退出来了。〞
 <br/>
 <br/>
 刘晓原表示，由于无法正常执业，无法通过年检，除了合伙律师和涉案律师外，其余律师已到其它律所执业，现在锋锐被注销执业许可证，剩下的律师也需要调离。
 <br/>
 <br/>
 刘晓原：〝我们获得的消息都不是很好，能否顺利调动，现在还不能肯定。也许在调动过程中会遇到种种阻力，让律师事务所不接收我们。〞
 <br/>
 <br/>
 已经〝被失业〞3年4个月的刘晓原表示，为办理转所手续，自己日前到北京律协递交申请，请协会为他出具执业经历证明，结果被告知要经协会领导研究再作答覆。他说，执业经历证明也要经领导研究，那接下来的多个证明材料更难办了。
 <br/>
 <br/>
 新唐人记者陈汉、陈洁、施宜君采访报导
