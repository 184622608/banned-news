### 〝基因编辑婴儿〞闯祸 海外实验室拒收大陆学生
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】大陆〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 〞诞生的消息令世界震惊之余，更引发国际社会对于中共科学界无底线的警惕。据悉，该消息曝光后，海外一些生物、遗传领域科学实验室，已开始拒收大陆的学者或学生。
 <br/>
 <br/>
 南方科技大学副教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 ，11月26日接受美联社专访时，曝出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 〞的爆炸性新闻，震惊全球科学界，包括122名海内外华人科学家在内，都对其〝疯狂〞行为进行了严厉的谴责。
 <br/>
 <br/>
 当天，神学家、德国伦理理事会主席彼得．达布洛克（Peter Dabroc）对德国之声表示，〝基因编辑婴儿〞对科学界是一场巨大的灾难。
 <br/>
 <br/>
 他呼吁，政界必须尽快关注这一话题。〝鉴于中国所出现的发展势头，有必要考虑建立起类似于‘国际原子能机构’那样的监管机构的可能性。〞
 <br/>
 <br/>
 贺建奎的母校美国莱斯大学也迅速做出回应，对其给予严厉谴责，并声明已对贺建奎的导师、生物工程教授Michael Deem启动全面调查。
 <br/>
 <br/>
 有评论认为，国际社会早已掌握基因编辑技术，但大多数国家都严令禁止，国外科学家也都严格遵守伦理底线，不在人体上来实施。唯独中共治下的学者冲破禁区，做了别人不做的事情，这是受中共无神论洗脑后，人伦道德底线丧失的一个典型例子。
 <br/>
 <br/>
 此事件已引发国际社会对大陆科学界无底线的警惕。据陆媒报导，日前，一些正向国外生物、遗传领域，申请做访问学者的人陆续收到通知，说这些实验室不再接收大陆学者、学生了。
 <br/>
 <br/>
 11月29日，〝基因编辑婴儿〞的制造者贺建奎，在一片质疑、谴责甚至唾骂声中，在香港第二届人类基因组编辑国际峰会露面。
 <br/>
 <br/>
 令人意外的是，贺建奎丝毫没有对这项实验对道德人伦方面造成的恶劣影响，表示歉意，相反却为此感到〝骄傲〞。
 <br/>
 <br/>
 贺建奎此言，立即遭到与会的诺贝尔奖得主大卫．巴尔的摩（David Baltimore）的斥责，他指责贺建奎的作法是〝不负责任〞。
 <br/>
 <br/>
 峰会闭幕后发表一份公开声明，对贺建奎的研究给予严厉谴责，要求其切实以负责任的态度做出相关善后工作，防止这项研究给人类基因库带来污染和不确定性的危害。
 <br/>
 <br/>
 在国际舆论强大的施压下，贺建奎在南方科大的实验室，已被当局查封。该项目涉及的多家官方和非官方机构也都迅速出面〝撇清关系〞，贺建奎本人也把这起事件归结为个人原因。不过，有评论指出，〝基因编辑婴儿〞并非贺建奎个人行为，背后另有推手。
 <br/>
 <br/>
 中国问题专家横河表示，〝基因编辑婴儿〞绝不是贺建奎个人行为，因为〝基因编辑婴儿〞要有相当规模、相当级别的医院才能做这件事，而且整个过程牵涉庞大的资金。
 <br/>
 <br/>
 〝从立项到批准，到资金，到医疗的设备和医疗的病房等等这一系列的问题，绝对不是哪一个人，甚至都不是哪一个公司或者是哪一个小医院能够完成的。所以他的背后肯定有超出深圳这个医院和超出他所在的学校的背景在。〞
 <br/>
 <br/>
 化名〝草祭〞的原上海某大学理学教授在推特上指出，〝基因编辑婴儿〞实验是由中共上层推动，南方科技大学负责实施，贺建奎具体执行的一项秘密计划。贺建奎是中共通过〝千人计划〞从海外引进的人才，长期在南科大留职停薪做研究，该计划上亿以上研究经费，不是一个副教授能独立争取到的，背后一定有中共科技部支持。
 <br/>
 <br/>
 事实上，基因编辑已被中共列入2016年制定的《〝十三五〞国家战略性新兴产业发展规划》。不难看出中共是这项计划的幕后推手。
 <br/>
 <br/>
 横河表示，〝在无神论的宣传下培养了几代无神论者，导致了他们可以拿人随便做实验，没有一个道德障碍。〞再加上中共当局的鼓励，这就是为什么这个事情会在中国发生，而没有在世界其它的国家发生的根本原因。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
