### 风水师：港珠澳大桥〝两蛇对冲〞 中共别有用意（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】被中共视为〝形象工程〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 24日正式开通，然而在风水界却流传一种说法，港珠澳大桥撼动了珠三角的巨水龙脉，中共以此桥破坏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港风水.html" target="_blank">
  香港风水
 </a>
 ，大桥从建设之初，香港就是非、天灾不断。
 <br/>
 <br/>
 <h4>
  大桥破坏
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港风水.html" target="_blank">
   香港风水
  </a>
 </h4>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 历时9年兴建，花费逾1200亿人民币，连接珠海、香港、澳门三地，被称作世界上最长的跨海大桥，中共称此桥为〝国之重器〞。
 <br/>
 <br/>
 然而，港珠澳大桥似乎并未给中共带来好运，在23日举行开通仪式的当天，中港两地股市大跌，给陷入阴霾的中共经济，迎头浇下一盆冷水。
 <br/>
 <br/>
 不仅如此，在风水界流传一种说法，港珠澳大桥是中共用来压制香港的风水阵，大桥的建立掀起一场中港风水大战。
 <br/>
 <br/>
 台湾东森电视新闻近日报导称，港珠澳大桥改变香港风水格局。风水师们认为，珠江是一条大龙脉，珠江水龙脉汇集了广州、深圳、佛山、中山、东莞、惠州、珠海、香港、澳门等地方的水龙，形成一条通天巨龙，而港珠澳大桥撼动了珠三角的巨水龙脉。
 <br/>
 <br/>
 中共自称港珠澳大桥是一条龙，声称大桥不会阻断珠江龙脉，还会把龙脉引致香港，使香港更旺。然而，大桥的设计与建立却与中共的说法完全不同，风水师们认为，大桥根本不是龙，而是蛇。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-25/p9096062a566471518.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-25/p9096062a566471518-ss.jpg"/>
   <br/>
   <font size="-1">
    在香港与澳门对接的过港隧道口，形成〝
    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_两蛇对冲.html" target="_blank">
     两蛇对冲
    </a>
    〞之势，将使香港和澳门产生是非纷争，更破坏香港风水。（视频截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 在香港与澳门对接的过港隧道口，两个人工岛形似两个蛇头。风水师指，这是〝两蛇对冲〞之势，将使香港和澳门产生是非纷争，更破坏香港风水。
 <br/>
 <br/>
 香港的地形酷似〝龟〞，属福地、旺地，而蛇是克制龟的。风水师认为，酷似两条长蛇的港珠澳大桥，是中共用来压制香港的一个风水阵，会给香港招来天灾是非。
 <br/>
 <br/>
 大桥施工9年来，共发生9次致命工业意外，至少18人丧生，工伤的数字超过600宗，被坊间谑称为〝奈何桥〞。民间传说中，奈何桥是通向地狱的入口。
 <br/>
 <br/>
 更为蹊跷的是，大桥主体工程完工后，天鸽颱风就来了，对香港经济造成巨大伤害，并导致多人遇难。在此之前，很少有颱风直扑香港。而且此后，香港颱风不断，近期又迎来山竹颱风的侵袭。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 香港风水师王亭之认为，以大玄空风水角度看，港珠澳大桥是一条水路，把水路引离香港，而且俨如把维港从西北处打横切开。西北是香港的发源地，大桥会把由西北至北的元气泄去，对香港大大不利。
 <br/>
 <br/>
 而且从实际情况来看，有了大桥后，在澳门设办事处便和香港没太大分别，香港的转口地位也会因此下降。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西九毒蛇阵.html" target="_blank">
   西九毒蛇阵
  </a>
 </h4>
 <br/>
 风水师们还指出，不仅港珠澳大桥是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以蛇克龟.html" target="_blank">
  以蛇克龟
 </a>
 〞的风水阵，中共还在香港西九龙布局了一个〝西九毒蛇阵〞。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-25/p9096063a480181931.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-25/p9096063a480181931-ss.jpg"/>
   <br/>
   <font size="-1">
    今年9月底开通的香港西九龙高铁车站，外表看似时髦，但香港民众指出，这个高铁车站的造型酷似香港墓地，非常触霉头。（视频截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 今年9月底开通的香港西九龙高铁车站，外表看似时髦，但香港民众指出，这个高铁车站的造型酷似香港墓地，非常触霉头。而且，在高铁车站两边，还兴建有博物馆和公园。
 <br/>
 <br/>
 风水师指，这三点也形成一个蛇头，并且直冲着对面的中山公园，中山公园有孙中山的雕像，寓意香港民主、自由。这个〝西九毒蛇阵〞被指是中共用来破坏香港民主自由独立的又一个风水阵。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中银大厦.html" target="_blank">
   中银大厦
  </a>
  刀插香港
 </h4>
 <br/>
 此外，中共央行设在香港的中银大厦，也被指有风水设局。中银大厦外形酷似一把〝尖刀〞，直插香港金融中心，并且与香港总督府对冲。中银大厦于1985年4月18日动土启造后，时任港督尤德爵士在1986年12月5日访问北京期间，心脏病发猝死，成为唯一一位于任内逝世的港督。
 <br/>
 <br/>
 风水师指，中银大厦给香港其他银行带来不小冲击，在其旁边的汇丰银行，经风水师指点，特意在楼顶设置了一座〝风水炮〞，以对抗中银大厦这把〝刀〞，汇丰银行业绩随后开始回升。
 <br/>
 <br/>
 <h4>
  风水师预言香港局势
 </h4>
 <br/>
 且不论风水阵对香港产生的作用如何，单从现在的形势来看，中共对香港的管控越来越严。自2014年香港爆发雨伞运动后，香港多所大学的校长被更换，换成认同中共执政理念的校长，而且许多大学学者、教授也被换。中共试图从教育下手，改变香港年轻人对自由民主的追求和认知。
 <br/>
 <br/>
 已故前辈术数家吴师青曾指出，香港整体的风水格局是一个天市垣格局（即商业大都会格局），但是自然环境上，香港的格局并不完美，在西南方西博寮海峡方向直穿南中国海，中间没有岛屿遮挡。在风水上，对这种格局称为水大无收，即朝向这方的水没有兜收。西南方是坤卦，坤主民，即指一般的市民，现象就是当流年飞星转变触动时，居住在香港的市民，会大批迁移，香港市民会迁到别处，也会有大批外来客迁入香港。
 <br/>
 <br/>
 自1997年香港移交主权后，许多香港富豪和民众纷纷移民其他国家，而大批中共官员及富豪〝白手套〞等人却汇集香港，搅乱香港政商两界，正应了吴师青的预言。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
