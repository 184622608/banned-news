### 非洲猪瘟疫情持续扩大到中国南方
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 在中国大陆持续扩大，已经传到南方。中共官方连日通报，最近三天，云南、浙江和湖南都传出非洲猪瘟疫情。另外，一名中国游客带香肠闯关日本机场，被验出含非洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪瘟病毒.html" target="_blank">
  猪瘟病毒
 </a>
 。
 <br/>
 <br/>
 中共农业农村部新闻办公室10月23号发布，湖南省益阳市桃江县和常德市桃源县的两个养殖场，查出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 。
 <br/>
 <br/>
 湖南益阳市袁先生：〝猪瘟扩散的情况非常严重了，而且湖南是生猪养殖大省，像这样的一些地方都有爆发猪瘟，证明应该是蔓延到比较严重的程度，官方的媒体，很多真实的讯息都没有报导。最早爆发是在北方，蔓延的速度非常之快，而且现在到了南部省份了，向全国蔓延的势头已经形成了。〞
 <br/>
 <br/>
 官方10月22号发布，浙江台州市三门县一家养殖专业合作社发生猪非洲猪瘟疫情。这是浙江省出现的第二起非洲猪瘟疫情，在8月17号，台州市南边的乐清市也发生过非洲猪瘟疫情。
 <br/>
 <br/>
 当地民众表示，由于政府封锁消息，报导和宣传太少，很多人对疫情意识薄弱。
 <br/>
 <br/>
 浙江台州市林先生：〝猪瘟不得了，不当回事，吃的人还多啊，没死人啊，猪瘟这个事，我们还没在意。〞
 <br/>
 <br/>
 21号，官方通报，云南昭通市镇雄县牛场镇、母享镇两养猪场发生非洲猪瘟疫情，共有545只发病猪全部死亡。
 <br/>
 <br/>
 自8月3号沈阳爆发中国首例非洲猪瘟以来，已有河南、黑龙江、内蒙古、吉林、天津、山西、安徽、江苏、浙江、云南11个省份出现疫情，其中辽宁疫情最多，已通报14起，造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪价.html" target="_blank">
  猪价
 </a>
 暴跌，养猪户苦不堪言。
 <br/>
 <br/>
 辽宁沈阳市养猪户王女士：〝现在不太好卖，外市现在都不行，出不了沈阳市，大概5块左右一斤现在，不挣钱，不挣钱就保个本，要是有买主的话就卖，猪的吃食，一天成本不少呢，所以说该出栏的时候就应该卖出去，不卖的话成本太高。〞
 <br/>
 <br/>
 另外，日本共同社报导，本月1号，海关从抵达北海道新千岁机场的中国旅客的行李中查获香肠。经过检验，这些香肠验出有非洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪瘟病毒.html" target="_blank">
  猪瘟病毒
 </a>
 基因。据了解，非洲猪瘟病毒能在肉品中存活近三年，甚至可以在香肠、火腿、腊肉等加工肉品中存活140天。
 <br/>
 <br/>
 新唐人记者熊斌、周天采访报导
</div>
