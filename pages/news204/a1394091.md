### 【禁闻】范冰冰缴钜款免牢狱 中共要钱无人能逃
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】大陆知名女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 因〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞被判决，要补缴人民币8.8亿元而避
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_免牢狱.html" target="_blank">
  免牢狱
 </a>
 之灾。中国大陆影视行业的〝严冬〞即将到来。有评论表示， 中共要钱，不管你是哪个阶层无人能逃。
 <br/>
 <br/>
 消失123天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 ，发表一封
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_悔过书.html" target="_blank">
  悔过书
 </a>
 ，写着〝没有党就没有我〞，引爆舆论界。
 <br/>
 <br/>
 10月3号，范冰冰在个人微博上发表600多字的道歉信，透过公开信表示，她将按照税务部门的最终处罚决定，补缴税款、缴纳罚款。
 <br/>
 <br/>
 今年5月，曾任职央视主持人的崔永元，将一组〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞在网上曝光，暗指范冰冰拥有灰色收入，有严重偷税漏税的嫌疑。民众举报范冰冰〝阴阳合同〞涉税问题后，引起中共相关当局注意。
 <br/>
 <br/>
 江苏省税务局在9月30号向范冰冰下达正式的处罚决定书。据了解，当局对范冰冰这起逃避事件惩处超过8.8亿元罚款。监于范冰冰未曾因为逃税而受到刑事处罚，只要她在限期内缴纳罚款，就可以免去刑事责任，否则将移送公安机关处理。
 <br/>
 <br/>
 旅美作家张林：〝范冰冰这个应该算是走运，比起刘晓庆来说，还算是走运。因为按照中国的刑法，贪污几十万都可以判十几年。〞
 <br/>
 <br/>
 中共国家税务局还声称，要规范影视行业税收秩序。2018年年底前主动到主管税务机关补缴税款的，将免予行政处罚，不罚款，否则将追究法律责任。
 <br/>
 <br/>
 流亡美国的湖北知名企业家 肖运军：〝没有一个人逃得了，它(中共)是通过各种形式，各种花样来抢夺私人财产，不管企业大小。她只不过是演艺圈的一个代表，通过她来警示其他的明星，让他们主动把钱交出来。〞
 <br/>
 <br/>
 张林：〝管妳是范冰冰，管你是谁，想要收割你，就收割你，你就是韭菜。包括官员也是，也是在党和国家领导人(眼里)看来，都像肥猪一样，养肥了什么时候需要宰，就拉出来，就砍上了，别说什么民营企业、外资企业，它需要宰的时候一样宰。〞
 <br/>
 <br/>
 旅美作家张林表示，中共宰的民企太多了。
 <br/>
 <br/>
 张林：〝民企被宰的太多了，像以前薄熙来在重庆唱红打黑的时候，抓了多少人。中共这几十年来的首富，大部分都要被共产党给收割了，太多了，包括王健林，万达，中国的首富，现在都消声匿迹了，也不知道他到底还有没有钱，谁都不知道，不关心那个。在中国，什么民营企业家，就是一只黑猪而已。需要你的时候，让你赚点小钱，不需要的时候，就把你宰了。〞
 <br/>
 <br/>
 肖运军：〝就凭我这个年代内地看到的事情，就是这样的。可能这场运动，在中国（中共）国有化的运动和扫黑运动，会持续很久，因为它的经济来源没有了，只有通过这种方式，扫黑的方式，国有化的方式，抢劫私人的财产。现在中国的老百姓会受到各式各样的掠夺，他们连逃的机会都没有，钱更不用说，出不来。〞
 <br/>
 <br/>
 有网友说，〝我一直以为都在为自己美好的未来打工，原来我是为充实党的口袋在打工。〞
 <br/>
 <br/>
