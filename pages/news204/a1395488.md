### 【禁闻】陆股跌不停 上市公司高管爆离职潮
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】中国大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_A股.html" target="_blank">
  A股
 </a>
 股价表现不佳，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上市公司.html" target="_blank">
  上市公司
 </a>
 也掀起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高管离职.html" target="_blank">
  高管离职
 </a>
 、降薪、裁员潮。目前，中国大陆3,551家A股上市公司，大约每9家就有一位董事长离职。
 <br/>
 <br/>
 中国大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_A股.html" target="_blank">
  A股
 </a>
 有多黑？房企龙头万科A股，已逼近跌停。10月11号，中国股市再现千股跌停，这是10年来首次十一长假后的〝开门黑〞。
 <br/>
 <br/>
 不只
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股灾.html" target="_blank">
  股灾
 </a>
 ，现在更出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上市公司.html" target="_blank">
  上市公司
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高管离职.html" target="_blank">
  高管离职
 </a>
 潮。根据东方财富证券旗下的〝Choice〞金融数据平台统计发现，今年7月份以来，A股上市公司公告董事长辞职的有将近49家。
 <br/>
 <br/>
 而《中国基金报》通过〝Wind〞数据库的统计发现，今年以来，共有376家上市公司的385位董事长公告离职。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田认为，这件事情并不单纯。因为正常国家如果公司业绩不好，股市下滑，一般是执行长（CEO）辞职，或者是总执行长辞职。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝因为执行公司经济状况不好，或者盈利不高，或只是股市下滑的话，实际上是总裁和CEO的责任，因为他们是在实际运作这个公司，董事会只是一个公司股东的代表。明明是总裁和CEO的责任，出了问题为什么董事长辞职，所以这个奇怪的现象，这也是在中国畸形的商业环境下才出现的。〞
 <br/>
 <br/>
 旅美时事评论员唐靖远观察，当前，中国企业高管们也出现悲观潮。
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝无论现在是国营企业还是民营企业，这些比较有实力的，这些能够上市的这些公司的这些高管，他们集体对中国当前的经济前景已经是一种非常悲观的情绪。〞
 <br/>
 <br/>
 据了解，除了董事长辞职，上市公司的高管们也出现离职潮。今年已有1,000多家A股上市公司的超过8,000名董事高管离任。
 <br/>
 <br/>
 谢田：〝因为我们现在也看到了很多中共对民企开刀的问题，很多民企的董事长失踪了，被抓起来了，被诱捕或者是被退休了，这种情况都有，这种现象已经造成董事会人人自危。〞
 <br/>
 <br/>
 统计显示，在高管与董事离任人数高于去年的样本中，民营企业占60%；尤其在市值小于50亿的上市公司中，今年民营企业离职的高管与董事人数占比接近70%。
 <br/>
 <br/>
 谢田：〝可能这些人他们已经意识到中共的第二次所谓的公私合营，第二次剥夺资本家的资产，这是第二次共产化，很可能就要发生。在这种情况下，他们作为股东代表可能知道，也许他们事先得到什么消息，也可能他们觉得前景不妙，才会出现这种大规模的董事长辞职的现象。〞
 <br/>
 <br/>
 有分析指出，这次中国A股暴跌主要是受美股下跌的影响。但对比中国股市与美国股市，美股从2008年的6,000点，暴涨到今年的26,000点，而跟跌不跟涨的中国A股却从6,000多点，暴跌到2,600点以下。
 <br/>
 <br/>
 近期，美国股市已从大跌中大幅回档，而中国则是累创新低深度下挫。在美中贸易战的背景下，中国经济下行趋势越来越明显。
 <br/>
 <br/>
 唐靖远：〝尤其是到了明年1月份之后，那么川普最新的这二千亿清单的关税他要增长到25%，其实这个它对中国股市，对中国实体经济的打击都会非常大。它确实也说明了中国当前的这种经济，尤其是在中共体制之下的这种经济状况，它现在这个危机是越来越加深。〞
 <br/>
 <br/>
 中国财经界的高手纷纷出逃。之前大陆流行的一个段子说，中国的首富们约香港的李嘉诚一起赛跑，中国的首富们包括王健林、马云、马化腾、刘强东和许家印，他们都嘲笑李嘉诚年纪大，腿脚不利索，肯定跑不快，结果却是李嘉诚跑了第一。大家都在奇怪为什么他能得第一，李嘉诚大笑说，谁叫你们听到枪声才跑！
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/钟元
