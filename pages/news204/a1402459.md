### 〝健康猫〞APP爆雷　坑煞数万体育界人士　
------------------------

<div class="wysiwyg">
 【新唐人2018年12月08日讯】一款名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健康猫.html" target="_blank">
  健康猫
 </a>
 〞的APP，吸引体育界人士， 在该平台注册成为私教，并鼓励私教借钱刷单创造业绩，获得高额补贴，今年8月〝健康猫〞爆雷，受害人估计有几万人，涉及金额30亿元，其中很多体育学院学生负债累累，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信用破产.html" target="_blank">
  信用破产
 </a>
 。
 <br/>
 <br/>
 据了解，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健康猫.html" target="_blank">
  健康猫
 </a>
 〞是一款O2O
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健身APP.html" target="_blank">
  健身APP
 </a>
 ，撮合健身爱好者和私人教练约课，于2015年3月上线，在全国高校体育学院宣传，吸引了大量体育专业师生、运动员等在该平台注册为私教，私教可获得学员支付的课时费外，还可获得1.5%、5%到10%不等的高额课时补贴，但私教都没有实际教课，靠刷单获得补贴。
 <br/>
 <br/>
 〝健康猫〞受害人杨女士：〝咱这个主要是，也是一个体育平台，像我们都是搞体育的，受害的人确实应该是挺多的，然后不完全统计应该不少。〞
 <br/>
 <br/>
 今年5月，平台承诺的补贴没到位 ，却以提高补贴和以上市为诱饵，鼓励私教借钱加码投入。
 <br/>
 <br/>
 一名匿名的受害人是刚从体育学院毕业的研究生，他负债60万元。
 <br/>
 <br/>
 〝健康猫〞匿名受害人：〝钱到最后投的多也是因为在五月份，忽然补贴提高了，它宣传的更好了嘛，说是要上市，我们好多都借了信用卡，还有借了同事的、同学的，从6月1号或者是5月那个时间段，有的钱就已经开始不到，所以当时好多人受不了，坚持到8月10几号才开始报警。〞
 <br/>
 <br/>
 该平台宣传称，共有25万名私教注册，但受害人估计只有几万人，每人投入数十万到数百万元不等，涉及金额30亿元。
 <br/>
 <br/>
 〝健康猫〞匿名受害人：〝我估计好多都是学生的话，只有几万，到后期我看别人发出来一个表，那里面就说，亏钱的说不定只有一、两万人，但是好多都是那种包括在校的大学生，还有退休的老师，还有刚毕业的。现在好多人征信都黑了，并且有的天天吃馒头、咸菜，小孩有的都生病了。〞
 <br/>
 <br/>
 今年8月20号，〝健康猫〞平台运营方广州大象科技公司公告说 ，由于私教恶意刷单
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_涉嫌诈骗.html" target="_blank">
  涉嫌诈骗
 </a>
 ，所以停止提现。8月27号，广州公安抓获〝健康猫〞创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨骅力.html" target="_blank">
  杨骅力
 </a>
 等9人，后来7人以涉嫌集资诈骗罪批捕，并对该公司以涉嫌非法吸收公众存款立案侦查。有消息称，大部分资金被藏匿国外，受害人希望能早日追赃还款。
 <br/>
 <br/>
 新唐人记者熊斌、舒灿采访报导
</div>
