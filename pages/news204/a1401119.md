### 张家口爆炸案：货车烧成空壳 司机睡觉来不及撤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】河北省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张家口.html" target="_blank">
  张家口
 </a>
 市一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_化工厂.html" target="_blank">
  化工厂
 </a>
 附近28日凌晨发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸.html" target="_blank">
  爆炸
 </a>
 ，现场画面惨烈，一排货车烧成了空壳。目击村民说，发生爆炸时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司机.html" target="_blank">
  司机
 </a>
 正在车内睡觉，来不及撤。截至当日下午，死亡人数升至23人，受伤人数22名。
 <br/>
 <br/>
 11月28日0时41分，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张家口.html" target="_blank">
  张家口
 </a>
 市桥东区河北盛华化工有限公司附近发生的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸.html" target="_blank">
  爆炸
 </a>
 起火事故，截至下午，有23人死亡，22名伤者仍在医院救治。
 <br/>
 <br/>
 附近居民说，事发时正在睡觉，被爆炸声惊醒，〝当时已经睡了，爆炸后玻璃震了一下，我还以为是发生了地震〞。
 <br/>
 <br/>
 距事发地2.6公里处的一商铺老板说：〝事发时爆炸声巨大，当时睡觉被惊醒，受此影响店铺目前已关门。
 <br/>
 <br/>
 现场画面显示，整栋大楼被烧焦，一排货车烧成了空壳。多辆消防车停在路边，有消防人员给冒烟的车辆喷水。一辆轿车横在路中央，已烧得焦黑。目击村民说，事发时，停靠在路边运送危化品的货车起火。
