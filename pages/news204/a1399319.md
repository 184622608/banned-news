### 北京再陷严重阴霾
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】周三（11月14日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 出现严重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴霾.html" target="_blank">
  阴霾
 </a>
 ，空气污染指数PM2.5达到311，属于〝对健康有害的严重污染〞。
 <br/>
 <br/>
 由于严重的空气污染，周三（11月14日）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 再次笼罩在厚厚的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴霾.html" target="_blank">
  阴霾
 </a>
 当中。
 <br/>
 <br/>
 到中午12点，北京环境保护监测中心报告显示，全市空气污染指数平均值已经达到311，属于〝对健康极度有害的严重污染〞。北京市民对如此严重的空气污染表示担忧。
 <br/>
 <br/>
 北京市民丁琬云：〝我觉得还是比较担心的。首先是对我的健康来说吧，因为说实话，现在北京这个天气，我觉得对呼吸道和肺部都非常的不好。所以这种天气我更希望在家里呆着，好在家里有空气净化器。〞
 <br/>
 <br/>
 北京市民玉芳无奈的表示，面对阴霾她只能选择逃离。
 <br/>
 <br/>
 北京市民玉芳：〝就是每年一到这个天气，就要躲避一下，到墨尔本去找儿子（儿）媳妇去了。〞
 <br/>
 随着冬天的来临，在中国的北方，城市供暖的需求导致阴霾更加频繁的出现。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
