### 百度纵容山寨签证网 复旦教授中招投诉无门
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】首先来关注中国的消息。大陆搜索引擎
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_百度.html" target="_blank">
  百度
 </a>
 的竞价排名一直为人诟病。最近因百度把多个山寨外国官方签证网站，排到正规领事馆签证服务之前，连上海复旦大学中文系教授严锋也中招，而且投诉无门。
 <br/>
 <br/>
 复旦大学中文系教授严锋11月29号发微博说，自己在网上办理赴土耳其的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电子签证.html" target="_blank">
  电子签证
 </a>
 时，通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_百度.html" target="_blank">
  百度
 </a>
 搜索到一个名为〝土耳其签证中心〞的网站，花了129美元办理了签证。后来却发现，排名前两位的〝土耳其签证中心〞都是第三方中介，让他多花了近一倍的费用办理签证，而且投诉无门。
 <br/>
 <br/>
 百度搜索靠前的山寨网站业务还包括柬埔寨、肯尼亚、美国等国家的电子签证，网站英文域名都显示为〝某国名-onlinevisa〞，中文名称显示为〝XX国签证中心〞。
 <br/>
 <br/>
 在〝百度评价〞上，有大量的网友对这些山寨站进行投诉，抱怨缴纳费用之后，没有成功为他们办理电子签证或者美国签证更新，耽误了行程。
 <br/>
 <br/>
 有多年领队经验的导游说，这些代办机构有时为了图方便，填写虚假资料蒙混过关，有可能导致签证申请人被拒绝入境，甚至惹上官司等，而且用户的个人资料也有可能泄露给第三方。
 <br/>
 <br/>
 大陆法学博士张杰表示，百度这类问题存在多年了，像两年前的〝魏则西事件〞发酵后，百度一度承诺不再将造假广告、山寨网站等参与到百度的竞价排名中，但过后仍列入搜索结果的第一位。
 <br/>
 <br/>
 大陆法学博士张杰：〝为什么百度能够这样？很重要的原因就是中国互联网由于进行封锁，把西方高品质的都排除了，比如说谷歌啊，所以没有竞争嘛，有竞争才有服务嘛，所以在这种情况下，网民实质上是没有选择的。〞
 <br/>
 <br/>
 时事评论员凌小辉则表示，百度是中共控制的一个工具，用来欺骗大陆民众的。
 <br/>
 <br/>
 新唐人记者常春、李沛灵采访报导
