### 与贸战息息相关 四中全会推迟创30年记录
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】有关中共十九届四中全会，官方至今没有任何消息。港媒指，今年的四中全会将是30年来召开最晚的一次，其原因和中美贸易战有关。
 <br/>
 <br/>
 香港《明报》10月30日文章指，中共本届四中全会看来已经很难在10月启动，打破了近30年来召开最晚的纪录。过往六届四中全会，最晚的一次是2014年10月20日召开的十八届四中全会，最早的一次是1989年因为六四事件提前在6月23日召开的十三届四中全会。
 <br/>
 <br/>
 根据中共官方日前宣布的日程安排，上海的中国国际进口博览会（简称进博会）会期是今年11月5日至10日。出席的国外领导人中，俄罗斯、巴基斯坦、萨尔瓦多、多米尼加、古巴等5国首脑，还将顺道正式访华，日期从10月31日到11月8日。
 <br/>
 <br/>
 文章称，据此判断，四中全会最快也要在11月10日才能召开。
 <br/>
 <br/>
 不过文章指，李克强和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将分别于11月14日和17日赶赴菲律宾的东亚系列峰会和巴布亚新几内亚的APEC峰会，如果为期4天的四中全会在10日开幕，日程安排将〝非常紧凑〞。
 <br/>
 <br/>
 但是公开信息显示，APEC峰会将与11月12到18日在巴布亚新几内亚召开。据此前外媒报导，习近平还邀请太平洋岛国领袖在APEC峰会前往巴国，举办一场太平洋岛国领导人峰会。
 <br/>
 <br/>
 果真如此的话，四中全会或将推迟到习近平出访回国之后。
 <br/>
 <br/>
 有外媒消息称，四中全会可能在11月底召开，或者继续向后推延。
 <br/>
 <br/>
 据菲律宾媒体报导，习近平在APEC峰会后还要顺道访问菲律宾。但中共官方还没有公布习近平访菲的具体行程。
 <br/>
 <br/>
 不过，习近平已确定将出席11月30日至12月1日在阿根廷召开的G20峰会。美媒消息，11月29日，中美首脑还将在阿根廷有短暂会晤。
 <br/>
 <br/>
 在如此紧凑的日程安排下，四中全会举办日期难以确定。
 <br/>
 <br/>
 《明报》分析，从中共外交部和商务部透露的信息看，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海进博会.html" target="_blank">
  上海进博会
 </a>
 应该是今年中共最重要的主场外交活动之一，而且与中美贸易战息息相关。
 <br/>
 <br/>
 路透社日前消息指，四中全会将在11月举行，但会安排在美国中期选举之后，因为四中全会的主题就是〝落实经济改革〞，北京希望在〝外部大的不确定因素稍有明确结果〞后，再确定后续的〝改革方案〞。
 <br/>
 <br/>
 多有分析指，北京在〝豪赌〞美国中期选举。但外界普遍认为，遏制中共已是美国朝野共识，中期选举不会对川普（特朗普）的贸易政策有太大影响。因此，所谓〝等待中期选举结果〞或者只是北京托词。
 <br/>
 <br/>
 香港《苹果日报》总结了对会期延迟的三大猜测：一是习近平权力失控问题还没有得到解决；二是北京对当前困境难以拿出有效对策；三是北京出现严重误判，在等待美国中期选举结果。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
