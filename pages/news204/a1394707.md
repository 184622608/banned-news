### 【禁闻】〝千人计划〞被美紧盯 中共急雪藏？
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】自美国联邦调查局开始全面排查中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞在美人员后，有消息传，大陆大专院校开始大规模删除涉及〝千人计划〞和相关人员名单的信息。有人提醒在美学者，要做出正确的选择，不要成为中共的牺牲品。
 <br/>
 <br/>
 最近，有来自微信上的截图显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共教育部.html" target="_blank">
  中共教育部
 </a>
 发出紧急通知，要求各高校把网站上有关〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞的信息全部删除，保证不出现〝千人计划〞相关介绍与表述，教育部还要对全国高校的网站进行检查。不仅学院网站上的相关信息要全部删除，各系、中心和研究院也要将各自单位的〝千人计划〞信息全部删除。
 <br/>
 <br/>
 10月4号，一份署名〝千人计划青年计划评审工作小组〞9月29号回复某单位的信件截图，出现在网上。信件要求要〝做好〞海外人才的安全保障工作，要各单位在通知面试答辩时，不要使用邮件，要采用电话、传真等方式。并且以回国参加学术会议、论坛等名义参加，文字通知中也不能出现〝千人计划〞字样。
 <br/>
 <br/>
 所谓的〝千人计划〞是中共在2008年底启动的，以优厚待遇吸引海外高层次人才回大陆的计划。该计划是由中共组织部、教育部、统战部等组成工作小组协调。
 <br/>
 <br/>
 中共官方声称，〝千人计划〞已经引进6000多名高层次人才。
 <br/>
 <br/>
 旅美学者吴祚来：〝人才流动，人才全球化，市场化也是一个必然的结果，但是，中共它这个人才和整个大的方向发生了一个变化，它利用人才不是用来造福整个国家，尊重人权，它就像当年希特勒那样，它把整个国家人才的力量和经济的力量，都用于制造一种军国主义，如果中国（中共）由这些一流的人才来加盟，来做大，它的后果是不堪设想的。〞
 <br/>
 <br/>
 旅美学者吴祚来以大陆的摄像头为例，指出中共用最先进的摄像头，用来监控人民、迫害维权人士。
 <br/>
 <br/>
 吴祚来：〝它制造了《一九八四》奥威尔小说里所说的那样一种‘老大哥无时无刻不在控制你’。利用全球最先进的力量做这样一件事情。那么美国现在开始在经济每一个方面，对中国进行围剿，所以千人计划是这么庞大的一个计划，美国对它进行的一个反击、调查，它是必然的。〞
 <br/>
 <br/>
 中共此前曾一度高调宣传〝千人计划〞。
 <br/>
 <br/>
 但美国联邦调查局开始全面排查中共〝千人计划〞在美人员，至今已经有多名被中共引入〝千人计划〞的华人学者、工程师，因为涉嫌转移美国敏感技术而遭到美国逮捕、判刑，或者是拒发签证。
 <br/>
 <br/>
 此前还有消息称，〝千人计划〞的在美人员人人自危。
 <br/>
 <br/>
 随后才出现了大陆大专院校大规模删除涉及〝千人计划〞和相关人员名单的消息。
 <br/>
 <br/>
 旅美时事评论员郑浩昌认为，中共不太可能轻易取消〝千人计划〞。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝中共的东西往往都有明暗两层的用途。千人计划在明处是人才培养，在暗处就是盗窃西方的技术。像孔子学院，明处是推广汉语，暗处是中共意识形态渗透。当暗处的用途被盯上之后，它为了避免更大的损失，往往就会毁灭罪证。死不改悔是中共这个犯罪集团一贯的表现，所以它把地面的事情转入地下的可能性更大。〞
 <br/>
 <br/>
 吴祚来规劝在美学者，为自己选择正确的未来，不要成为中共的牺牲品。
 <br/>
 <br/>
 吴祚来：〝这样的一些华人的科学家，他们应该去看一看，1950年代那些科学家，帮助中共那些科学家的下场，无数爱国的科学家在50年代回去报效祖国，最后数以千计、数以万计的科学家，都死在甘肃的夹边沟。被打成特务，被打成涉嫌海外敌对势力，打成右派被迫害致死。〞
 <br/>
 <br/>
 吴祚来表示，美国对〝千人计划〞的个人进行个案调查，是非常必要的。他希望这些人，能够人人警醒，回头是岸。因为当暴政极权政府觉得你没用时，就会抛弃你。
 <br/>
 <br/>
 采访编辑／常春  后制／钟元
