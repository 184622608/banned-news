### 郑晓松死亡之迷 澳门官员：聚餐时中联办突然离席
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】中共官方20日称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办主任.html" target="_blank">
  中联办主任
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 因患〝抑郁症〞坠楼身亡，但澳门官员与中共官方的说法不一，澳门立法会主席提及郑晓松坠亡当晚有中联办人员突然离席，但未有透露原因。
 <br/>
 <br/>
 中共国务院港澳事务办公室官网21日公布，驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 特别行政区联络办公室主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 因〝患忧郁症〞，于20日晚在其澳门住所坠楼身亡。
 <br/>
 <br/>
 香港电台报导说，澳门立法会主席贺一诚对郑晓松堕楼身亡，感到突然和可惜。他又透露，20日晚与澳门中联办人员吃饭时，中联办人员突然离席，未有透露原因。
 <br/>
 <br/>
 中国人权民运信息中心21日报导，消息人士披露，郑晓松近日被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 官员问话，这可能是他
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀.html" target="_blank">
  自杀
 </a>
 主因之一。
 <br/>
 <br/>
 《苹果日报》报导说，中央巡视2016年对港澳办展开专项巡视，发现国务院港澳办部分官员〝出问题〞，包括任用程序不严格、贯彻中央的决策部署不扎实。
 <br/>
 <br/>
 前澳门
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办主任.html" target="_blank">
  中联办主任
 </a>
 的国务院侨务办公室前副主任李刚，去年8月落马被查，揭示港澳系统贪腐严重。而郑晓松之死的最大疑点是澳门司警强调案件仍在调查，但港澳办已定调郑死于抑郁症。
 <br/>
 <br/>
 澳门司法警察局10月22日公布调查结果，指郑晓松的死因无可疑，没有任何刑事成分。
 <br/>
 <br/>
 司警局局长薛仲明则表示，郑晓松坠楼事件虽然无可疑，但已转交调查科跟进，《苹果日报》报导质疑，薛仲明未有提及官媒声称的〝抑郁症〞，是否等于不承认官媒〝死因〞？
 <br/>
 <br/>
 22日有海外爆料者称，郑晓松〝实际上是被干掉的〞，并指郑晓松最好的哥们，是刚刚被捕的原国际刑警组织主席、中共前公安部副部长孟宏伟。
 <br/>
 <br/>
 中共官方10月7日深夜通报：孟宏伟涉嫌违法，目前正接受中共监委监察调查。这是孟妻就丈夫9月底只身返回中国失联，向法国里昂警方报警之后，中共官方的首次回应。
 <br/>
 <br/>
 分析认为，孟宏伟被诱捕回国遭中纪委调查，很可能就是习近平着手清理政法系统的一个切入点。
 <br/>
 <br/>
 海外爆料者还预言，〝香港的中联办姓王的哥们也快了！大家往下看吧！〞似乎是暗示现任香港中联办主任王志民亦将出事。接下来很多人会跳楼、自杀或被杀，特别是公检法和中纪委。
 <br/>
 <br/>
 澳门自1999年主权移交中共之后，已有5任中联办主任，多数命运不济。首任主任王启人上任后一年患癌症病逝；第三任主任李刚上任因贪腐受查；第5任主任郑晓松坠楼身亡。
 <br/>
 <br/>
 资料显示，郑晓松曾在中共外交部、新华社香港分社等部门工作。2007年至2012年任中共财政部国际司司长，2012年4月任财政部部长助理、党组成员。2013年7月任福建省副省长，2016年3月任福建省委常委、秘书长。
 <br/>
 <br/>
 2016年6月，郑晓松任中共对外联络部副部长，去年9月任澳门中联办主任。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：范铭）
