### 【禁闻】王全璋妻发公开信 谴责法官故意违法
------------------------

 【新唐人2018年11月05日讯】
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋妻.html" target="_blank">
   王全璋妻
  </a>
  发
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公开信.html" target="_blank">
   公开信
  </a>
  谴责法官故意违法
 </b>
 <br/>
 <br/>
 11月5号，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709案.html" target="_blank">
  709案
 </a>
 〞受害律师王全璋的妻子李文足在网上发表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公开信.html" target="_blank">
  公开信
 </a>
 ，谴责王全璋案的两名主审法官，指控他们〝故意违法〞。
 <br/>
 <br/>
 这两名法官是天津二中院的林昆和周虹。李文足在信中历数他们在王全璋案中阻止家属聘请的律师办案和会见当事人的事实，指控他们剥夺了王全璋的合法权利、也侵犯了家属聘请律师的权利。
 <br/>
 <br/>
 李文足说，王全璋从2015年7月10号被抓失踪至今已经3年4个月，严重超出了规定的办案期限。林昆和周虹的故意违法行为，是中国法治的耻辱。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民间智库.html" target="_blank">
   民间智库
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_学者.html" target="_blank">
   学者
  </a>
  被禁赴美开会
 </b>
 <br/>
 <br/>
 大陆民间智库〝北京天则经济研究所〞所长盛洪11月5号披露，他被禁止赴美参加该所和美国哈佛大学费正清中心合办的〝中国改革开放40年研讨会〞。
 <br/>
 <br/>
 盛洪当天对〝美国之音〞表示，他1号在北京首都机场边检时，被告知不许出境，理由是〝危害国家安全〞。
 <br/>
 <br/>
 盛洪对自己去参加一个学术研讨会，被指称为〝危害国家安全〞，感到十分震惊。
 <br/>
