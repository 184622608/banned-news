### 【禁闻】蓬佩奥吁拉美国警惕中共的〝掠夺性〞
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 近日访问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉美.html" target="_blank">
  拉美
 </a>
 国家时，呼吁拉美国家要警惕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在该地区的投资，他还批评中共开展〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_掠夺性.html" target="_blank">
  掠夺性
 </a>
 经济活动〞给其他国家带来灾难后果。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 19号在墨西哥接受美国媒体采访时，呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拉美.html" target="_blank">
  拉美
 </a>
 国家警惕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_掠夺性.html" target="_blank">
  掠夺性
 </a>
 的行为〞。
 <br/>
 <br/>
 蓬佩奥表示，中共在全球很多国家有〝掠夺性的行为〞，它们投资大笔资金，但是加上条件，让那些国家的人民在两年、五年或是十年之后，陷入很可怕的境地。
 <br/>
 <br/>
 他还说，如果中共想要投资，那很好，但是投资必须是透明公开的，而且要符合被投资国人民的最佳利益，而不应该有附加条件，在被投资国无法还债时，施加束缚。
 <br/>
 <br/>
 蓬佩奥还指出，有些中国大陆国有企业的商业活动与〝全球良好行为不符〞。
 <br/>
 <br/>
 旅美时事评论员田园：〝中国的投资不仅仅是一个简简单单的投资问题，它还带来另外很大的问题。比如说，贪污腐败、黑钱、洗钱，以及中共勾结当地政府对当地民众的经济剥削和掠夺。〞
 <br/>
 <br/>
 去年6月，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴拿马.html" target="_blank">
  巴拿马
 </a>
 与台湾断交，转投中共怀抱。中、巴建交一年后，就传出中国企业抢标巴拿马运河第四桥专案，专案价值14.2亿美元。但不久，此专案传出舞弊丑闻，巴拿马方面不得不宣布废标。
 <br/>
 <br/>
 蓬佩奥18号与巴拿马总统巴雷拉会面时说：〝要提防中共企业所带来的表面上看起来似乎非常好的交易。〞
 <br/>
 <br/>
 他指出：〝中共的国营企业并不透明，没有市场主导性，如果不是来带给巴拿马人利益，而是来带给中共政府利益的话，那就是重要问题了。〞
 <br/>
 <br/>
 蓬佩奥认为，中共的这种掠夺性经济行为对其他国家和巴拿马人是不利的。
 <br/>
 <br/>
 田园：〝巴拿马最大的经济命脉其实就是巴拿马运河。巴拿马运河需要有人给它提供安全保障。谁来提供安全保障？美国在这个地区的力量是绝对无法忽视的，美国就是不管南美洲、北美洲地区的唯一的超级强国。它对巴拿马运河的保障，很显然是第一位的。〞
 <br/>
 <br/>
 资料显示，此前，美国建造了巴拿马运河，后来免费归还巴拿马政府。因巴拿马运河也是美国商船使用的一个重要通道。
 <br/>
 <br/>
 旅美时事评论员田园指出，巴拿马希望加入西方的经济体系，成为自由民主经济繁荣昌盛的国家。但中共的投资只能给巴拿马带来专制、暴力、腐败等。
 <br/>
 <br/>
 田园认为，美国应该更深入介入这一地区，把中共的影响力排除在外。
 <br/>
 <br/>
 他表示，美国应该向巴拿马、加勒比海等国家投资。这些国家也能看到，美国的投资是公平、互利的，不会像中共的投资带来乱七八糟的东西，也不会出现血汗工厂那样的事情，可以给当地带来真正的繁荣、和平。
 <br/>
 <br/>
 其实，自中共2013年推出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞计划以来，就大量投资其他国家，同时给这些国家带来了新问题。
 <br/>
 <br/>
 中共在斯里兰卡投资数十亿美元。由于斯里兰卡无力偿还债务，不得不把投资超过10亿美元的汉班托塔港和邻近六十平方公里工业区土地的营运权，租给中国公司99年。这也让当地国民感到非常愤怒。
 <br/>
 <br/>
 尼泊尔去年也因中国公司财务违约，取消一价值25亿美元的水电项目合同。
 <br/>
 <br/>
 被视为一带一路核心的〝中巴经济走廊〞，中共计划提供600亿美元贷款。但去年底一项目因腐败丑闻而停资。今年巴基斯坦新总统上台后，出于对债务的担忧，开始消减项目投资。
 <br/>
 <br/>
 大陆时政评论员陈明慧：〝它到哪去它都不守规矩不守法律，它自己制定的法律都不会遵守，处处都是玩潜规则、黑箱操作。〞
 <br/>
 <br/>
 大陆时政评论员陈明慧表示，中共治下一切都是腐败。中共走到哪里，都会给那里带来腐败。
 <br/>
 <br/>
 田园指出，中共去哪里投资，就是想把当地的资源掠夺一空，然后再寻找下一目标，根本不会考虑当地的发展问题。
 <br/>
 <br/>
 采访/陈汉 编辑/宋风 后制/钟元
