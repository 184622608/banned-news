### 山东矿难21人死 网友先行曝光遭行拘
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】山东一名男子因为在网上发消息称，菏泽市郓城县10月20号发生的矿难，有9人确认死亡，被警方以虚构事实为由拘留。到了29号，当地官方通报称共有21人遇难。警方的这一做法引发外界斥责。
 <br/>
 <br/>
 据山东省菏泽市郓城县公安局《行政处罚决定书》显示，男子王某10月21号在〝新浪微博〞发文称，〝郓城李楼煤矿发生重大安全事故21人被掩埋，已确认9人死亡〞。警方称该微博属于虚构事实扰乱公共秩序，将王某行政拘留十天，并处罚款500元。
 <br/>
 <br/>
 不过，29号，当地官方通报称，事故共造成21人遇难，只有1人获救。
 <br/>
 <br/>
 对此，网友对警方做法提出质疑：〝矿难死亡人数是不断变化的，今天官方媒体发布死亡19人，如果明天死亡人数上升到20人，郓城警方也去把官方媒体抓起来？〞
 <br/>
 <br/>
 有网友说：这哥们太倒霉了，因〝造谣〞被拘留的处罚期限还没满，事故遇难人数就已远远超出他当初〝谣言〞里说的死了9人。
 <br/>
 <br/>
 湖南维权人士黄先生：〝他也没造谣，官方它刻意隐瞒那死亡人数嘛！〞
 <br/>
 <br/>
 北京网友马先生：〝许多网友在调侃这件事，说他报少了，所以他被抓了。老百姓说什么都不行，说多了说少了都是造谣，说对了叫泄露国家机密。没办法，在大陆是一点话语权都没有。现在大陆简直回到文革了。〞
 <br/>
 <br/>
 山东维权人士卢女士表示，既然现在官方确认死了21人，应该追责。
 <br/>
 <br/>
 山东维权人士卢女士：〝按照中国的安全管理制度的话，死了9个人的话，那要追责。死了21个人，那是哪一个领导层次的人进行追责呢？最少都是省级以上的领导。〞
 <br/>
 <br/>
 有网友质问，〝环球网、新浪等媒体造谣重庆公交车事件是女司机逆行，被警方证明是谣言——为何不抓？是否官媒有造谣特权？〞
 <br/>
 <br/>
 卢女士：〝中国一直不都是这样的吗？官媒是代表官方的语言，官方想让它做什么，官媒就是照做的。中国的媒体不会受到惩罚的。〞
 <br/>
 <br/>
 网友感叹：〝奇葩国度奇葩事！〞〝现在是比1984还要恐怖、还要寒蝉的时代。〞
 <br/>
 <br/>
 新唐人记者熊斌、王子琦、施宜君采访报导
