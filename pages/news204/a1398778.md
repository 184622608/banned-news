### 金正恩见大陆演员 孙楠深鞠躬成话题
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】近日，一支由中共官方派出的文艺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代表团.html" target="_blank">
  代表团
 </a>
 抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 进行政治
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_表演.html" target="_blank">
  表演
 </a>
 。演出结束后，朝鲜领导人金正恩会见了演出团的主要成员。男歌手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙楠.html" target="_blank">
  孙楠
 </a>
 与金正恩握手的一幕，引发舆论调侃。
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 官媒报导，此次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代表团.html" target="_blank">
  代表团
 </a>
 受朝鲜方面邀请，由中共文化旅游部部长雒树刚带队，阵容包括大陆歌手、电影和舞蹈演员等。
 <br/>
 <br/>
 朝中社称，金正恩3日在万寿台艺术剧场观看朝中艺术家联合演出，在演出前，他会见了中共文艺工作者代表团主要成员。在演出后，金正恩上台同主要演员一一握手。
 <br/>
 <br/>
 值得注意的是，演员里男歌手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙楠.html" target="_blank">
  孙楠
 </a>
 与金正恩握手过程中的一幕最抢眼。
 <br/>
 <br/>
 从视频中可见，孙楠不仅深鞠躬和金正恩握手，握完手后，还合掌鞠躬。而金正恩停留在孙楠跟前的时间也比较长，似乎还让跟前的工作人员记录下了什么内容。期间孙楠满脸堆笑，频频点头。
 <br/>
 <br/>
 网友表示，孙楠动作有些太夸张了，要知道眼前的这个金正恩，是个连自己的亲姑父和胞兄都敢杀的刽子手。
 <br/>

   <font size="-1">
    从视频中可见，孙楠表现的最抢眼。（视频截图）

  <br/>
 </center>
 <br/>
 英媒BBC评论说，中国多名商业演艺演员集体赴朝访问尚属首次，这些演员代表的选定透露出不少讯号。
 <br/>
 <br/>
 评论认为，从朝鲜角度来看，这次到访的中国演员代表具有大众知名度，这对今后金正恩进行某种政策转向会提供帮助。代表团成员也都是得到中共认可的演员。
 <br/>
 <br/>
 评论说，中朝间的文化演出一直都带有政治意义，而此次两国不仅仅满足于政治宣传，中共派出多名市场化演员，说明朝鲜已经在宣传文化方面，已做好向中共市场经济和改革开放新文化靠拢的准备，而中共给朝鲜派出了改革开放的〝样板团〞。
 <br/>
 <br/>
