### 受贸战波及 沃尔沃汽车拟转移成都生产线至瑞典
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】中国吉利汽车旗下的沃尔沃汽车（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Volvo.html" target="_blank">
  Volvo
 </a>
 ）正计划将XC60的大部分生产线从中国迁往瑞典，以避开中美之间的〝关税大战〞 。而本周三，美国贸易代表莱蒂泽刚刚公开威胁要对中国汽车征税40%，以便将美中之间的汽车关税差距拉成均等。
 <br/>
 <br/>
 日前有香港媒体披露，沃尔沃汽车有传计划将畅销型号XC60的大部分生产线，从成都转移到瑞典特斯兰大（Torslanda）的厂房去进行组装，以便出口到美国市场。
 <br/>
 <br/>
 而就在28日这天，美国贸易代表莱蒂泽突然再次向北京开火，批评中共的产业政策具侵略性和国家导向性，损害了美国工人及产业，但在美方多次向北京反映有关议题后，中方却至今未能在桌上提出〝有用的改革建议〞。
 <br/>
 <br/>
 莱蒂泽同时指出，中国对美国进口车征收高达40%关税，明显高于美国目前向中国制汽车产品征收的27.5%关税。因此美国政府将考虑把美中汽车的关税差距拉成均等。

 据了解，早在今年7月中旬，沃尔沃汽车的行政总裁萨穆埃尔松(Hakan Samuelsson)就曾对外透露，该企业正将供应给美国市场的XC 60 SUV生产线，从中国转移至欧洲，以避开美国对中国实施的新进口关税。
 <br/>
 <br/>
 他表示，沃尔沃目前在瑞典生产的XC 60主要是供应欧洲客户，而中国产出的则供应美国等其他市场。为避开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的影响，该企业将改为由欧洲厂向美国供应XC 60，中国产出则供应其他市场，有关调整现已展开。
 <br/>
 <br/>
 不过，从目前贸易战的发展情况来看，出口到美国的欧洲汽车，也可能同样面临加征关税的问题。
