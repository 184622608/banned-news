### 川普：中方让步清单漏掉几个大项目 我要加上（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】日前，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 对北京提交的贸易让步方案做出回应，称他目前不会接受，其中漏掉了几个重要项目。不过，他也对谈判进展表达了乐观态度。
 <br/>
 <br/>
 美东时间周五（11月16日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在白宫的一项法案签署现场对媒体证实，北京寄来了一份〝他们愿意做（让步）的项目的清单〞。
 <br/>
 <br/>
 川普称，这是一份内容庞大的清单，包含142个项目，但对他来说还是不能接受。
 <br/>
 <br/>
 他说，这份清单〝相当完整〞，但〝有4或5个重要项目漏掉〞，〝我认为可能也要将它们加上〞。
 <br/>
 <br/>
 稍早前，英美媒体已透露，北京提交的142个项目分为3类：中方正在解决的问题；愿意继续讨论的问题；拒绝谈判的问题。美方不接受中方排除那些〝不可谈判〞的项目。
 <br/>
 <br/>
 而且，该方案大多是重申此前北京已经做出的承诺，但仍然回避有关政府补贴、强制技术转移、盗窃知识产权以及〝中国制造2025〞等美国重点关切的问题。
 <br/>
 <br/>
 川普还对现场记者说，如果中方不能满足美方要求，美国还准备对剩下的2,670亿美元中国商品加征关税。但他也表示，〝但我们可能不会这样做。（因为）中方很希望达成协议。〞
 <br/>
 <br/>
 川普称，〝我不想让中国处于不利地位……但我们必须进行互惠交易……我认为会达成协议。我们很快就会看到。〞
 <br/>
 <br/>

 <br/>
 川普对中美贸易战前景的乐观评价，令美国股市三大指数一度应声上扬。不过，川普在讲话中更多的是强调〝美国必胜〞的信心，并没有肯定11月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 能够取得实质成果。
 <br/>
 <br/>
 美国商务部长罗斯同日对彭博社表示，美方将按原计划，在2019年1月把对2,000亿中国商品加征关税的税率从10%提升至25%，因为中美在1月前不可能达成一个全面的正式协议。
 <br/>
 <br/>
 他说，目前所做的只是为川习会做准备，如果进展顺利，这次会晤最多能就进一步谈判达成一个框架性协议。
 <br/>
 <br/>
 此前，中方一直拒绝在正式谈判前首先向美方提交书面的让步清单，因为担心这会让北京失去谈判筹码。
 <br/>
 <br/>
 因此，中方现在做出第一步妥协，交出让步方案，虽然大多是老调重弹，但也被视为解决贸易战的一个积极信号。因为中方做出答覆后，华盛顿就可以在这个基础上与北京进行有实质意义的讨价还价。
 <br/>
 <br/>
 不过，这个后续谈判的过程显然不会那么顺利，也不是短期内所能完成。
 <br/>
 <br/>
 《美国之音》引述分析称，如果中方能够表现出足够的〝诚意〞，川习会一个可能的结果是，川普总统宣布暂停明年1月1日对2,000亿中国商品提升关税，以等待中方在谈判中做出更多让步。
 <br/>
 <br/>
 不过，除了外界一些分析人士不看好北京的〝诚意〞，白宫强硬派们日前也公开表达了对中共的不信任。
 <br/>
 <br/>
 16日，主管关税的美国贸易代表办公室高调声明，否认英媒有关〝暂停对2,670亿美元中国商品加税计划〞的报导，强调原计划将维持不变。美国贸易代表莱特希泽是主张对中共强硬的代表人物之一。
 <br/>
 <br/>
 另一名强硬派人物、白宫贸易高级顾问纳瓦罗近日也表示，中共一贯言而无信，所以即使美中达成协议，也无法相信中共能够真正执行。
 <br/>
 <br/>
 自由亚洲引述分析指，北京目前的经济政策，包括〝中国制造2025〞等，已经和最高当局确立的政治路线捆绑在一起，如果做出实质改变，最高层将面临重大的政治风险。
 <br/>
 <br/>
 中共历史上，政治路线的变动，往往伴随着你死我活的权力更迭和政治倾轧。特别是，中共政坛目前暗流涌动，党内斗争空前复杂。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
