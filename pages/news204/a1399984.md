### 【禁闻】伯恩光学裁员引工潮 贸易战恶果谁先尝
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】由于苹果公司订单减少，全球最大手机屏幕供应商——港资惠州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伯恩光学.html" target="_blank">
  伯恩光学
 </a>
 公司突然宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裁员.html" target="_blank">
  裁员
 </a>
 8,000人，引发大规模
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工潮.html" target="_blank">
  工潮
 </a>
 ，当局派大批防暴警察来维稳。评论认为，中共会将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战恶果.html" target="_blank">
  贸易战恶果
 </a>
 率先转嫁给农民工这个弱势群体。
 <br/>
 <br/>
 《自由亚洲电台》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伯恩光学.html" target="_blank">
  伯恩光学
 </a>
 公司11月9号到12号连续几天爆发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工潮.html" target="_blank">
  工潮
 </a>
 。匿名工人表示，现在因订单减少，大量临时工还没有转为正职就纷纷遭到辞退。另外中介也没有取得招聘奖励，所以中介组织工人包围了工厂。当局派大批防暴警察来维稳。
 <br/>
 <br/>
 伯恩光学有限公司是一家港资企业，号称全球最大手机屏幕供应商。公司业绩60%左右来自于〝苹果〞和〝三星〞订单。近期，由于苹果公司大幅削减手机订单，导致供应商陷入困境。
 <br/>
 <br/>
 台湾企业家与社会运动人士高为邦：〝辞退了临时工为什么会引来这么多的抗议？其实我想因为现在中国的经济非常不好，倒闭的工厂一大堆，在这个时候因减产被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裁员.html" target="_blank">
  裁员
 </a>
 ，对他们的未来是非常恐慌的。因为大概就不可能再找到其它的工作。〞
 <br/>
 <br/>
 香港《苹果日报》引述微信圈的消息说，被裁员工10月份的工资还没有发放。他们不满厂方赔偿，加上年关将至，找工作不易，不少打工者等钱回家过年。示威工人连家属人数最多时达到一万人。而中国劳工通讯负责人韩东方也表示，〝去年是去产能，而今中美磨擦，后天又中欧。官方在工人权益受损情况下，没长久应对和解决机制〞。
 <br/>
 <br/>
 华裔加拿大籍作家盛雪：〝中国的农民工可以说是这个世界上非常悲惨的一个群体。中共在这方面出的成本是非常低廉的。因为这个群体我们都知道，没有自由、没有人权、没有福利保障。那么现在由于美国对中共实施的这种贸易制裁和打击，中共也就自然会把这个人群作为最先的牺牲品。〞
 <br/>
 <br/>
 10号，惠州惠阳区政府部门发通报说，伯恩方面已与相关工人〝达成初步解决方案〞，目前工人已〝疏散〞。
 <br/>
 <br/>
 但是，除了伯恩光学外，苹果代工厂〝伟创力〞旗下子公司〝伟创力塑胶科技（深圳）公司〞也因为订单锐减，从11月12号到明年2月1号，将安排员工分批〝放假〞。
 <br/>
 <br/>
 全球最大苹果代工厂〝富士康〞的工人也透露，在深圳观澜产业园区，生产量下降10%，今年产量高峰期提前个半月结束，每天有一千多人离职。
 <br/>
 <br/>
 《路透社》10月16号引述中共官方数据报导说：今年上半年中国有504万家企业倒闭。台湾媒体估计，至少有240万大陆工人已经失业。
 <br/>
 <br/>
 高为邦：〝这个大环境不好，其实对中国（中共）来讲，中美贸易战才刚刚开始而已，它一直拿不出一套办法来对付美国。所以未来的失业潮那可能会更恐怖。而这些失业的人是长期的农民工，所以回到农村大概也很难适应。〞
 <br/>
 <br/>
 中共农业农村部日前声称，大陆目前有740万人返乡创业。但外界质疑，这波所谓的〝返乡创业潮〞实质是〝大规模失业潮〞。
 <br/>
 <br/>
 盛雪：〝改革开放过程当中，这些人脱离开了原来的土地，进了城。生活可能是比以前有了改观，挣的钱多了。但事实上他们是会被随时抛弃的。他们已经无法退回到原来传统的生活和生存方式上去了。〞
 <br/>
 <br/>
 很多农民工因此陷入两难境地。
 <br/>
 <br/>
 吉林农民工王先生：〝都失业了还在外面， 没有钱花。欠我们农民工的开发商，我们吿了多少次啊！结果呢，被政府的人拿着警棍把我们打出来，也没人管。 所以有很多老百姓都心寒了，你知道吧。要不着钱肯定得回家呗。你说回到农村现在靠天吃饭肯定是挣不着钱，你都得出去干活去。〞
 <br/>
 <br/>
 华裔加拿大籍作家盛雪分析，中共会把贸易战的恶果直接转嫁给底层民众，但民众生活越来越成问题，中共也必将面对越来越多的暴力反抗事件。
 <br/>
 <br/>
 采访/陈汉 编辑/尚燕 后制/陈建铭
