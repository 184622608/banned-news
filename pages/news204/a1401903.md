### 大陆高校一副院长挂佛像 被巡视组处理
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】针对网上流传的一份有关〝某高校
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副院长.html" target="_blank">
  副院长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_办公室.html" target="_blank">
  办公室
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_挂佛像.html" target="_blank">
  挂佛像
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汽车挂件.html" target="_blank">
  汽车挂件
 </a>
 被诫勉〞处理文件，河南城建学院人事处4日回应称，这是该校发出的文件。文件显示，该校副院长刘贵明因在办公室挂带有佛像的汽车挂件，被巡视组给予政务诫勉。
 <br/>
 <br/>
 这份题为《河南城建学院关于对刘贵明有关问题的处理决定》的文件显示，该校测绘与城市空间信息学院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副院长.html" target="_blank">
  副院长
 </a>
 刘贵明，因在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_办公室.html" target="_blank">
  办公室
 </a>
 挂带有佛像的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汽车挂件.html" target="_blank">
  汽车挂件
 </a>
 遭政务诫勉。
 <br/>
 <br/>
 该份文件是11月28日发出的，内容为23日上午，河南省委第六巡视组对河南城建学院领导官员，办公用房面积及办公环境进行检查，发现刘贵明在办公室镜子前悬挂一件带有佛像的汽车内饰挂件，违反上级有关要求。
 <br/>
 <br/>
 通报称：依据中纪委国家监委，经校长办公会研究决定，给予刘贵明政务诫勉，责令其立即将饰物带离办公室。
 <br/>
 <br/>
 对此，网友纷纷留言说：〝有点小题大作，矫枉过正。一个饰件而已，个人自由。〞〝要尊重个人爱好，信仰，不能强制。〞〝挂个佛像也拿来说事，整人的人不要做的过分了。〞
 <br/>
 <br/>
 事实上，中共党官等信仰神佛早已是普遍现象。中共党媒曾发文批评〝道士做法事是迷信〞，文中罗列了一些中共官员信仰神佛烧香拜神的行为。
 <br/>
 <br/>
 如中共原铁道部部长、党组书记刘志军为求平安，长期在家烧香拜神，在办公室布置〝靠山石〞，项目开工竣工，刘都会请〝大师〞选择黄道吉日。
 <br/>
 <br/>
 原山东泰安市市委书记胡建学，曾相信〝大师〞的预测，为了当上大官，下令让正在修建的国道改线，花巨资在该地一水库上架了一座〝岱湖桥〞。
 <br/>
 <br/>
 已经停刊的《争鸣》杂志早年曾刊文透露，中共十六届中组部发放的资料显示：有2000万中共党官信仰佛教，逢年过节要到寺庙烧香拜佛，在家设有佛龛。有的党官还说，信仰佛教，精神上有个寄托。
 <br/>
 <br/>
 时评人士唐靖远认为，此事说明，中共多年来反覆对党内信徒及普罗大众洗脑，宣传无神论、进化论，走到今天，这套歪理邪说已经破产了。不仅民众不信，包括大批党徒，甚至高官，信奉科学的学者们，已经不再真正相信共产主义这些基础理论。
 <br/>
 <br/>
 这是中共在意识形态领域已经崩溃的反映，现在的中共完全只是靠着利益和放纵腐败在勉强笼络它的党徒并维持统治。
 <br/>
 <br/>
 不过，由于中共官员几乎无官不贪，它们口头喊着无神论，却用暴力强迫民众服从接受中共的理论，自己却又指望求神问鬼来保护他们非法获得的种种不义之财，不是真正的信仰神佛，真正的神明也不会答应的。
 <br/>
 <br/>
 中国问题专家李善鉴认为，如果人类要想达到真正意义的信神敬佛，还得有正确的思想指导，否则难以真正获得神佛的护佑。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
