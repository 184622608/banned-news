### 珠海航展被指〝花拳绣腿〞 落后美俄20年
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海航展.html" target="_blank">
  珠海航展
 </a>
 开幕式上，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_歼-10B.html" target="_blank">
  歼-10B
 </a>
 战机表演了3个高难度动作，受到中共官媒吹嘘。但台媒表示，歼-10B表演的这些机动动作都是〝花拳绣腿〞，美俄战机早在几十年前就已经做到。另有中共军事专家自曝，歼-10B在相关技术上仍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落后美俄20年.html" target="_blank">
  落后美俄20年
 </a>
 。
 <br/>
 <br/>
 11月6日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海航展.html" target="_blank">
  珠海航展
 </a>
 开幕式上，一架中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_歼-10B.html" target="_blank">
  歼-10B
 </a>
 战机高空表演10多分钟，并做出〝眼镜蛇机动〞、〝J形转弯〞及〝落叶飘〞3个高难度动作，中共官媒因此对歼-10B大加吹嘘。
 <br/>
 <br/>
 不过，有台湾媒体表示，歼-10B表演的这些机动动作只是〝花拳绣腿〞，早在几十年前，美俄战机就已经能够做到。这些华而不实的动作，〝只有挨打的份〞。并指中共的歼-20、歼-10B，以及采用的矢量发动机仍落后美国至少20年。
 <br/>
 <br/>
 港媒也引述澳门国际军事学会会长黄东的分析，称歼-10B表演的3个机动动作，只有第三个〝落叶飘〞用到推力矢量引擎，但跟苏-35可多次旋转相比，歼-10B只转了一下，颇有差距。
 <br/>
 <br/>
 黄东还指，在相关技术上，歼-10B落后美俄不止20年，而且中共若要量产，最快要3至5年后。
 <br/>
 <br/>
 中共前总参上校、军事专家岳刚也说，歼-10B在技术上，尚需解决控制系统、材料科学、制造技术等诸多难题。
 <br/>
 <br/>
 当日开幕式上，除了歼-10B飞行表演外，3架中共新一代战斗机歼-20也进行了6分钟的飞行表演。不过，尴尬的是，进行表演的歼-20战机，装备的仍然是俄罗斯的AL31发动机。
 <br/>
 <br/>
 据军方消息人士向香港《南华早报》透露，中共自制的发动机没有通过可靠性测试。只好找〝替身〞。消息人士还表示：〝现在非常尴尬，如果量产的话，中国可能需要向俄罗斯方面求助。〞
 <br/>
 <br/>
 据报导，珠海航展每两年举办一次，今年从11月6日到10日，展期6天。中共本希望此次航展能对外展示中共军事技术。不过，在航展首日即爆出抄袭丑闻，令中共颜面尽失。
 <br/>
 <br/>
 俄罗斯媒体报导，一家名为〝Atom〞的俄罗斯设计局发声明说，中共展出的中俄两国合资CR929客机，是盗用了他们的设计方案。
 <br/>
 <br/>
 该设计局的总经理兼总工程师马克西姆‧库津（Maxim Kuzin）表示，2015年该局在上海投标时就展示过自己的设计方案，但遭到拒绝，没想到设计却被中方剽窃。马克西姆怒斥中共〝无耻〞。
 <br/>
 <br/>
 同日，日本《产经新闻》也报导说，航展上展示的中共最新款隐身无人机〝彩虹7〞，外形涉嫌抄袭美国海军〝X47B〞无人机。并指中共有可能通过网路黑客，窃取了相关技术。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
