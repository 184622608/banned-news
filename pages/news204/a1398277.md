### 云南昆明公安局前局长刀勇落马
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】中共官媒周二通报，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_云南.html" target="_blank">
  云南
 </a>
 西双版纳州委副书记、政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刀勇.html" target="_blank">
  刀勇
 </a>
 ，因涉嫌严重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违纪违法.html" target="_blank">
  违纪违法
 </a>
 ，目前正在接受调查。刀勇曾任昆明市公安局局长。通报称刀勇是〝投案自首〞，但没有说明落马的具体原因。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刀勇.html" target="_blank">
  刀勇
 </a>
 曾任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_云南.html" target="_blank">
  云南
 </a>
 省思茅市公安局副局长、云南省纪委纪检监察一室副主任、云南省纪委纪检监察三室正处级副主任；2011年12月，任云南省纪委纪检监察二室主任；2016年4月到2017年9月，任昆明市副市长、市公安局局长；2017年9月起，任西双版纳州委副书记、政法委书记。
 <br/>
 <br/>
 刀勇在云南公检法系统任职多年，该系统是中共江泽民集团利用来迫害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 的工具。
 <br/>
 <br/>
 据明慧网不完全统计，到2018年8月，有120名法轮功学员被昆明中院非法判刑，刑期最长的7年。
</div>
