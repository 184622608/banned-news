### 非洲叫停〝一带一路〞 塞拉利昂取消机场项目
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】中共力推的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞再次受阻。西非国家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞拉利昂.html" target="_blank">
  塞拉利昂
 </a>
 日前宣布，取消中共投资的新机场项目，指其没有必要性且费用昂贵。有评论人士预言，塞拉利昂的决定，将在非洲拉开退出〝一带一路〞的序幕。
 <br/>
 <br/>
 10月10日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞拉利昂.html" target="_blank">
  塞拉利昂
 </a>
 宣布，取消中共承建、运营并提供资金的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马马哈新国际机场项目.html" target="_blank">
  马马哈新国际机场项目
 </a>
 （Mamamah International Airport）。这是非洲国家首次取消与中共的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞贷款协议。
 <br/>
 <br/>
 塞拉利昂的理由是，该项目没有任何必要性，且高达4亿美元的项目经费过于昂贵。
 <br/>
 <br/>
 该国交通与航空部部长卡隆在一份声明中指出∶经过严肃的考量和评估，政府认为现有的机场尚可使用时，继续建造新机场项目是不划算的。该国打算将目前的隆吉国际机场进行翻修，并修建一条便于通勤的桥梁。
 <br/>
 <br/>
 卡隆还明确表示，尚不清楚取消该项目合约是否会产生财务罚款等影响，但作为一个主权国家，他们要为国家做出最好的决定。
 <br/>
 <br/>
 这份被叫停的新机场建造贷款协议，是塞拉利昂前总统科罗马在其任期内与中方签署。由中铁七局集团负责建设和运营，中国进出口银行提供项目贷款，计划2022年完工。该项目一直饱受〝缺乏透明度〞的批评。
 <br/>
 <br/>
 今年3月科罗马竞选失败，现任总统比奥重新评估了前任的合作项目。他决定取消马马哈中资新机场项目。
 <br/>
 <br/>
 一位来自非洲的推特网友发表评论，〝希望所有非洲国家都能拒绝中国（共）的非法扩张和殖民地条款〞。
 <br/>
 <br/>
 中共此前为斯里兰卡一个大型机场提供了大量贷款，然而，现在该机场被称为是世界上最空的国际机场，每周只有五个航班起飞，载客数百人。而斯里兰卡另一个深水港口，也因无法偿还债务，如今已变成历时99年的中共〝租界〞。
 <br/>
 <br/>
 台湾国立大学政治学教授李酉潭向自由亚洲表示，今年8月，美国多位议员曾发出警告，中共的〝一带一路〞是带有霸权性质的扩张，在他国进行掠夺性的基础设施投资，这是一个债务外交。
 <br/>
 <br/>
 李酉潭说，当然各国在政党轮替的时候，会发现他们承担不起这样的一个债务，或者是他们也发现〝一带一路〞是扩张，并检讨这样一个状况。
 <br/>
 <br/>
 他认为，中共〝一带一路〞肯定会碰到越来越多的挑战和困难。
 <br/>
 <br/>
 北京维权人士胡佳称，很多〝一带一路〞合作国已经认识到，该项目输出的专制腐败、利用债务对其国家安全的控制等。〝人家很会就会醒悟，这样做是得不偿失的。〞
 <br/>
 <br/>
 事实上，中共力推的〝一带一路〞已连续受挫。去年年底，巴基斯坦、尼泊尔和缅甸已经确认取消，或搁置了价值近200亿美元的〝一带一路〞三大水电项目。
 <br/>
 <br/>
 马来西亚于今年8月21日宣布，由于成本高昂，取消与中共此前签订并正在建设的高铁、石油管道等两个〝一带一路〞主要基础设施项目。
 <br/>
 <br/>
 种种迹象显示，被中共视为全球战略构想的〝一带一路〞倡议，在其盟友国家也接连遭遇抵制，中共的大笔投资被质疑是一种企图长期盘踞下来的〝殖民策略〞。有专家预言塞拉利昂取消机场项目，将拉开非洲抵制〝一带一路〞的序幕。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
