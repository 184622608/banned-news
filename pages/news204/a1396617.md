### 江西千人抗议建垃圾焚烧厂遭镇压
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】江西上饶市鄱阳县游城乡村民抗议当地建造垃圾焚烧发电厂，活动已经持续一周。村民到乡政府和县政府门前请愿。10月23号，上千村民游行、堵路，遭到警察
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_镇压.html" target="_blank">
  镇压
 </a>
 ，多人被打伤和抓捕。
 <br/>
 <br/>
 当地村民10月24号告诉《新唐人》，一个月前，当地政府官员来选址，村民才知道北塘村要建垃圾焚烧发电厂。由于该项目周边有五、六个水库，距离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_垃圾焚烧厂.html" target="_blank">
  垃圾焚烧厂
 </a>
 选址处最近的只有30米，最远的120米，村民担心会污染。当时连外出打工的年轻人都赶回来抗议，地方政府宣布停止该项目。
 <br/>
 <br/>
 但是最近项目突然动工，村民于是再次发起抗争。
 <br/>
 <br/>
 游城乡村民何先生：〝垃圾焚烧发电站，离学校、居住处竟然不到300米，好多村民都到游城乡政府嘛，还到县里面的政府，附近几个村嘛，北塘村附近的土塘啊，大田、李家、江家……高升桥边那附近几个村嘛。〞
 <br/>
 <br/>
 几百名村民到乡政府打横幅、下跪；之后游行，并到县政府请愿。许多外出打工的年轻人也陆续回来。10月23号，上千名村民前往县政府门口请愿、一度堵路，100多名特警、公安前来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_镇压.html" target="_blank">
  镇压
 </a>
 ，打伤和抓捕多人。当晚警方还进村抓走多名村民。
 <br/>
 <br/>
 游城乡村民段先生：〝垃圾焚烧厂对人体有害，开始是到游城乡政府门口，然后前两天到县城了嘛，人好多。是抓了，但是具体抓多少?还是抓了之后，会怎么样处理啦？目前还不知道。〞
 <br/>
 <br/>
 公开资料显示，鄱阳县生活垃圾焚烧发电项目投资近2亿元，设计规模为近期每天处理垃圾500吨，远期达到每天处理1000吨垃圾。服务范围包括鄱阳县的24个乡镇（街道），但芦田镇、饶埠镇、昌洲乡等乡镇除外。一期工程预计在2020年6月完成，二期工程在2025年6月完成。
 <br/>
 <br/>
 新唐人记者熊斌、柏妮采访报导
