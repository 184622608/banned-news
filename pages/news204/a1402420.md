### 北京抛弃孟晚舟？美中达共识：孟案不影响贸易谈判
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】中国华为集团副董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在加拿大被捕事件，引发全球舆论关注。但北京方面显得异常低调。有分析认为，面对美国方面的引渡要求和犯罪指控，孟晚舟这个烫手山芋似乎已被北京抛弃。中美双方目前亟待解决的问题是，如何能在90天内达成贸易协议。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 12月1日被捕当天，中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 和美国总统川普(特朗普)正在阿根廷20国集团（G20）峰会举行双边会晤，并且双方最终敲定在未来90天内，解决贸易争端。
 <br/>
 <br/>
 据英国金融时报报道，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晚宴登场前，习近平就已经知道孟晚舟被捕，但是习近平想聚焦于消除贸易紧张，因此决定不在川习会上提及孟晚舟的事。
 <br/>
 <br/>
 外界发现，中美双方聚焦的重点都是美中贸易谈判。美国白宫经济顾问纳瓦罗和库德洛日前在接受美媒采访时均表示，孟晚舟被抓和贸易谈判是两回事。如果美中两国在90天里不能达成贸易协议，美国方面会按原计划提高关税。
 <br/>
 <br/>
 与此同时，中共官方对孟晚舟被抓的反应，也不是很强烈。据香港英文报纸《南华早报》报导，中共商务部发言人高峰星期四表示，他对孟晚舟的情况不了解，但是他对中美两国在90天内达成协议很有信心。
 <br/>
 <br/>
 中共外交部发言人耿爽在例行记者会上，也拒绝将这两者联系到一起。他表示，〝在这里我无意讨论关于此案的具体细节〞。
 <br/>
 <br/>
 针对孟晚舟在加拿大被捕事件，中共党媒也罕见降低了调门。《人民日报》7日仅在海外版上转发大陆外交部声明，未多作评论。
 <br/>
 <br/>
 就连一贯煽动民族情绪的中共环球网，7日凌晨社评时，也未在文中像以往一样说狠话，仅说美方很善于利用谈判之外的辅助手段向对手施压，引发各种猜测，不过，相信美国政府愿像陆方一样结束
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 。
 <br/>
 <br/>
 有分析认为，从习近平和中共官方释放出的信息来看，当前重点是如何应对〝缓刑〞90天的贸易战。而至于孟晚舟被捕一事，一旦美国方面公布确凿证据，习当局将有可能抛弃孟晚舟这个烫手山芋，趁机掌控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为公司.html" target="_blank">
  华为公司
 </a>
 ，在全球封杀华为的困境之下，将其打乱重组。
 <br/>
 <br/>
 （记者文瑞报导/责任编辑：戴明）
