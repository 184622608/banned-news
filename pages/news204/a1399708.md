### 习近平两度遭封杀  王沪宁用意引猜测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】香港武侠小说家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金庸.html" target="_blank">
  金庸
 </a>
 葬礼近日举行，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 等中共高层给金庸送了花圈，但官媒并未报导。不仅如此，习在早前一次视察中，重提〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自力更生.html" target="_blank">
  自力更生
 </a>
 〞的口号，也遭官媒删除。这究竟有什么内情？主管宣传的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王沪宁.html" target="_blank">
  王沪宁
 </a>
 ，其用意引发舆论猜测。
 <br/>
 <br/>
 香港著名武侠小说家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金庸.html" target="_blank">
  金庸
 </a>
 10月底在香港去世，香港中联办官网消息称，中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，以及诸多现任或退休中共高层对金庸的去世表示哀悼。
 <br/>
 <br/>
 据明报报导，有关习近平、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 等人给金庸
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_送花圈.html" target="_blank">
  送花圈
 </a>
 的内容，在中国大陆却全遭删节。新华社在金庸设灵当日，虽发出1,500多字的长篇报导，但却对习李的花圈只字未提。
 <br/>
 <br/>
 而中新社通稿，虽有报导习李致送花圈，但相关报导，在大陆可以浏览的中新网页上，却看不到，而在中联办官网上，只有11月2日的〝习近平对金庸逝世表示哀悼〞的消息。
 <br/>
 <br/>
 根据报导，2日当晚，中联办主任王志民专程到查家，向查夫人林乐怡转达了习近平等人的哀悼和对亲属的慰问。但这则新闻，也未上央视新闻联播。
 <br/>
 <br/>
 至于背后的原因，报导称，中共领导人的活动是报导焦点，但也讲究规格，一般来说，中共领导人只有悼念副国家级以上过世者，才可上中央级媒体。而中共领导人悼念其他逝者，一般中央级媒体不作报导。
 <br/>
 <br/>
 如中共开国元帅叶剑英之子叶选宁2016年在广州去世，习近平等高层都送了花圈，却只有广东传媒报导。
 <br/>
 <br/>
 此外，近期还爆出涉及习近平的新闻遭删的同类事件。
 <br/>
 <br/>
 《金融时报》11月13日报导说，习近平9月在黑龙江省考察中国第一重型机械集团时，向工人们表示：〝国际上，先进技术、关键技术越来越难以获得，逼着我们走
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自力更生.html" target="_blank">
  自力更生
 </a>
 的道路。〞
 <br/>
 <br/>
 报导称，央视新闻联播对习近平的发言视频进行了剪辑，删掉了讲话的第一部分。中共党报《人民日报》海外版也作出了同样的编辑。
 <br/>
 <br/>
 除了〝自力更生〞这一口号外，〝外国技术越来越难获得〞的内容也遭中共党媒删除。
 <br/>
 <br/>
 但在中共党媒删除〝自力更生〞这一口号之后，习近平南下广东视察，再次强调要〝自力更生〞。
 <br/>
 <br/>
 中共过去提及〝自力更生〞的口号，分别在1941年，1960年，1975年，都是在中共遭遇严重的政治和经济危机，在国际上处于孤立状态之下。
 <br/>
 <br/>
 习近平此次在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 发生，令中共经济萎靡情况下，重提这一口号，蹊跷被官媒删除，引发舆论对主管宣传的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王沪宁.html" target="_blank">
  王沪宁
 </a>
 用意的猜测。
 <br/>
 <br/>
 一些政治观察家分析认为，中共党媒删改习近平的讲话，反映了中共宣传部门的担忧。即习的言论，被视为承认了中共采用了强制和窃取等手段，来获取外国知识产权。而这正是美国总统提出的中共不公平贸易行为之一。
 <br/>
 <br/>
 熟悉中共体制的人士分析说，中共宣传控制，在中央顶层，由主管宣传的政治局常委把持，出于政权维稳需要，这些删改属正常，但也有可能牵涉权斗关系，比如，前任江派常委刘云山，过往不时被爆藉此封杀习近平言论，甚至搞〝高级黑〞。
 <br/>
 <br/>
 而目前掌管宣传的中共常委王沪宁比较特殊，王被视为中共的首席御用文人，有中共三朝〝帝师〞之称，以笔杆子身份，在中共十九大跃升政治局常委，成为习近平第二个任期内，主管意识形态的最高官员。
 <br/>
 <br/>
 美媒披露，王沪宁未〝入常〞时，就对涉及习近平的宣传有特别权限。王不但为习策划了理论口号，作为习的随行必备人员，他还可以管制习的照片在官媒的发布。新华社等官媒发布习的照片，都需要先经过王的认可。
 <br/>
 <br/>
 中美贸易战开打之初，中共内部矛盾激化，正是习近平热紧急降温之时。中南海知情者对美媒爆料，针对不断加剧的内忧外患，王沪宁亲自下令，严禁所有负面报导和评论发生。
 <br/>
 <br/>
 该知情者说，王沪宁7月份将中宣部和各大中共官媒高层紧急招至中南海，在这次内部会议上，王沪宁竭力强调维护中央权威和绝对领导的重要性，反覆重申中共所有宣传机器，必须无盲区和无死角，以确保习近平的形象〝毫发无伤〞。
 <br/>
 <br/>
 从朝鲜半岛危机，到中美贸易战，从一带一路，到国企、扶贫和股市，再到污染、天价药和天灾人祸……事无具细，王沪宁均要求中宣部和各大官媒严格把关，把握分寸进行报导。
 <br/>
 <br/>
 不过，事实证明，王沪宁虽为了保护习近平的权威和形像，但产生的效果却适得其反。目前中共高层实施的所有举措，几乎全部遭到国际社会诟病及中国民间的抨击。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：戴明）
