### 崔永元〝遗书〞风波未平 有人出500万买他手脚
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】女星范冰冰被罚8.8亿人民币事件已落幕，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 爆料娱乐圈偷税漏税事件引发的后效应，仍在升温。诸如〝崔永元遗书、死亡威胁〞等等，近日在网络疯传，甚至有人发出威胁，要出500万元人民币，买崔的手脚。崔永元此前发文说，愿意花费8个亿求放自己一马。
 <br/>
 <br/>
 10月14日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 在《今日头条》回应称，自己没有想过写遗书，想想也是，砸了转基因利益集团的饭碗，废了娱乐圈轻松搞钱的路径，得罪了道上的人，人身威胁警察不管，当然要和它们大战一场！
 <br/>
 <br/>
 此前的10月12日，一篇题为〈崔永元：假如我倒下〉的文章在中国互联网上广泛传播。
 <br/>
 <br/>
 文章以崔永元口吻给孩子写遗书，〝如果爸有什么不测，请你相信爸不是意外，爸是和一群人战斗倒下，爸倒下的时候是冲着枪口的方向。〞
 <br/>
 <br/>
 文章盘点崔永元这辈子干的三件事，包括：当过中共央视主持人，和转基因食品较上劲，掀开了娱乐界的黑幕。
 <br/>
 <br/>
 第一件事已经划上句号，关于第二件事和第三件事，崔永元称，自己一直努力去做。文章还介绍了他揭露娱乐圈偷税漏税潜规则的缘由。
 <br/>
 <br/>
 不少网友质疑这是一篇蹭热点的自媒体文。
 <br/>
 <br/>
 〝遗书〞风波未平，崔永元本人又在其微博发布一名自称为〝天仙哥哥V〞的神秘网友所发的威胁崔的内容，包括〝崔永元你墓地都看好啦，遗照都选好了，最好别出门〞，〝内部消息有人出500万要你手脚〞等等。
 <br/>
 <br/>
 崔永元发过上述贴文之后，他本人又回应说：〝已经做了最坏的打算，那就走着瞧吧。〞目前有关帖文已被删除。
 <br/>
 <br/>
 10月7日，崔永元还自爆，受税务总局刁难，警察三次传唤他，追问他手中〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞的来源。他自称，〝我的危险来自四面八方，不能退〞。
 <br/>
 <br/>
 崔永元4日也在微博发文称，在这5年的时间里，有一个名叫司马3忌的人，一直不停地对小崔进行骚扰和诽谤，而且还对他家人进行了人身攻击，就在最近，他实在忍受不了这样的压力，他愿意出8亿元钱给这个人，求放过。
 <br/>
 <br/>
 崔永元9月下旬还对明报说，这段时间以来，他已经收到了不下10个死亡威胁。对此，他说自己：〝已将材料备份藏于多处，如果出意外，会有人告知媒体收藏位置。〞
 <br/>
 <br/>
 在崔永元表达对自身安全的忧虑后，《检察日报》以《举报人遭威胁，法律如何保护》为题做出回应。呼吁公安机关为举报人提供适当的保护。
 <br/>
 <br/>
 上海市公安局官方微博10日也发通报称，已成立调查小组，并多次联系崔永元，相关举报问题一旦查证属实，将依法依规处理。
 <br/>
 <br/>
 有分析认为，引爆中国娱乐圈〝阴阳合同〞丑闻的崔永元，恐怕来头并不简单。另外，有人追溯崔永元爆料的过程，认为其背后似乎一直有支持、有角力。
 <br/>
 <br/>
 有北京独立学者对美媒分析说，本次由崔永元担纲的这场娱乐大戏的主角，其前台幕后应该都是中共高层政军之间的角力。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
