### 中国各地老兵再现维权潮  诉求保障待遇
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】中国各地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 再现维权潮，最近又有广东、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_河北.html" target="_blank">
  河北
 </a>
 、重庆和大连等地的大批老兵聚集在当地的政府部门维权，另外，很多到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 维权的老兵都遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遣返.html" target="_blank">
  遣返
 </a>
 。老兵们的主要诉求是要求落实优抚政策，保障老兵的待遇等。
 <br/>
 <br/>
 11月8号，一、二百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广东惠州.html" target="_blank">
  广东惠州
 </a>
 参战
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 在市政府门口维权，他们手拿〝不要让英雄既流血又流泪〞、〝还我参战老兵的待遇〞、〝落实优抚政策第三条 保障参战老兵达至当地平均生活水平以上〞的横幅表达诉求。
 <br/>
 <br/>
 现场视频:〝今天是2018年11月8号，我们广东惠州市的参战老兵在市政府门口维权现场，请大家战友、全国的战友关注，现在是上午10点，上午10点2018年11月8号，上午10点，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵维权.html" target="_blank">
  老兵维权
 </a>
 的现场，老兵维权的现场。〞
 <br/>
 <br/>
 11月7号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_河北.html" target="_blank">
  河北
 </a>
 数百老兵聚集省退役军人事务厅外维权。
 <br/>
 <br/>
 现场视频:〝反对****、反对****、反对不作为、反对不作为、反对乱作为、反对乱作为、反对*****。〞
 <br/>
 <br/>
 现场老兵高喊口号遭到制止。老兵表示他们没有信访，嘲讽信访处长为〝聊天处长〞，要求事务厅领导出来接见。
 <br/>
 <br/>
 现场视频:〝事务厅的领导出来接见、事务厅的领导出来接见、厅长接谈、厅长接谈、厅长接见、厅长接见。〞
 <br/>
 <br/>
 另外，11月7号，还有数百重庆老兵在人社局外集体维权。
 <br/>
 <br/>
 四川等地数百老兵到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 维权，被大巴送去久敬庄过夜。
 <br/>
 <br/>
 11月6号，大连老兵在市信访局外维权。
 <br/>
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
 <br/>
