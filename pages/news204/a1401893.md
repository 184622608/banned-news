### 崔永元骂农业部：没人性（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 问题，央视前主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 频频发文追打中国农业农村部。3日崔永元再次爆料
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农业部.html" target="_blank">
  农业部
 </a>
 丑闻，称其没人性。
 <br/>
 <br/>
 12月3日晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 在微博中透露：中国农业农村部食堂全部用非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 油，却怂恿老百姓吃转基因油，没人性。〞 微博中附有一段视频，曝光了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农业部.html" target="_blank">
  农业部
 </a>
 食堂内部场景。
 <br/>
 <br/>
 此前，崔永元还曾批评农业部放任中国种植转基因作物，并称自己已经掌握了农业部吃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特供.html" target="_blank">
  特供
 </a>
 的详细资料等。
 <br/>
 <br/>
 11月28日，崔永元在微博发文称，看几张来自农业部食堂的照片意思一下。如果你以为它们只是借势吃点好的喝点好的，你就错了。关键是，它们还纵容坏的，那是关乎我们及我们后代生命安全的大事，所以必须揭露！
 <br/>
 <br/>
 照片中显示，农业部机关服务局喝15年的茅台、吃野生鱼类、低标高吃宴会用的龙虾、还有特供大米、野刺生参、非转基因油、高档拉菲红酒等。
 <br/>
 <br/>
 11月27日晚，崔永元发布关于中国农业部食堂的视频，画面中有一段手机上的文字显示：张传国和团伙成员刘荣富偷偷向农业部部长韩长赋送礼。但网上没有对张传国和刘荣富的介绍。
 <br/>
 <br/>
 11月25日，崔永元在微博发文，披露中共农业部内部食堂内幕。从他发布的视频显示，中共农业部内部食堂使用的腐竹，上面标注有〝非转基因〞的字样。
 <br/>
 <br/>
 视频中一位正在熬制粥类食物的厨师，对视频拍摄者称，食堂所使用的食材、包括调味品，都是非转基因的。微博痛批，中共农业部〝只顾自己保命，不管百姓死活。〞
 <br/>
 <br/>
 11月20日，崔永元也曾在微博上发帖称，〝我已掌握农业部食堂、部长餐厅，从来不吃转基因只吃特供的详细资料。正在整理中。〞
 <br/>
 <br/>
 10月11日，崔永元微博小号发文，就转基因问题批评中国涉农业机构，他说，农业部是最最孙子的地方，现在专吃非转基因，却不允许非转基因食品标注。
 <br/>
 <br/>
 他还说，农业部特供基地在新疆，全是有机的，现在严管，不敢了。不承认以前有。他讽刺中共原农业部副部长张桃林是个骗子，因张桃林曾称，现在对转基因的监管已做到了县一级。
 <br/>

 </center>
 <br/>
 11月10日，崔永元转网友的评论并点名中共农业部称，〝农业部的特供基地在新疆吐鲁番，全是有机的。另有大鱼、大虾、各种新鲜土特产等，鸡蛋也是特供的。现在国家管得严，不敢了。顺便翻脸不承认以前有过。〞
 <br/>
 <br/>
 11月8日，崔永元曾批评农业部关于转基因食品的研究结论造假。他在微博中说：农业部说中国转基因安全最爱说自己是全世界管理最严，有专门的安全委员会。转基因中心造假、负责人李奎是转基因安委会成员，两件事说明农业部一直放屁！
 <br/>
 <br/>
 中国自1992年成为第一个引入转基因粮食到商业化生产的国家。培育出一种抗病毒的烟草。自此，中共当局开始广泛给转基因食品发安全证书，中共农业部宣称，中共对转基因技术的支持是坚定的，在中国转基因技术应有一席之地。
 <br/>
 <br/>
 而在转基因的问题上，崔永元一直持反对态度，他曾宣布自己〝自费〞100万元人民币，赴美调查转基因食品的安全问题，结果发现，在美国的超市，转基因食品是有明显标记的，和非转基因的食物是分开排放的。
 <br/>
 <br/>
 而且大部分的美国消费者对转基因的危害是有认识的。如果能够不买转基因食品的话，他们会尽量避开这类食品，因为对人体的危害性目前还没有深刻的研究。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
