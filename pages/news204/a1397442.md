### 美司法部起诉10名中共特工  内情揭晓
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】据报，美中双方发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的核心原因之一，就是美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_指控.html" target="_blank">
  指控
 </a>
 中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_盗窃.html" target="_blank">
  盗窃
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 。在此背景下，美国近日对10名中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_情报人员.html" target="_blank">
  情报人员
 </a>
 做出了指控。
 <br/>
 <br/>
 10月31日香港《南华早报》报导称，美国司法部30日起诉了多名中共特工，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_指控.html" target="_blank">
  指控
 </a>
 他们试图窃取美国公司的航空技术。
 <br/>
 <br/>
 美国联邦调查局（FBI）的一名特工说，中共支持的黑客活动是真实且没完没了的。 他还说，如今，FBI正在向中共以及其他参与了黑客行为的政府发出强烈信号。
 <br/>
 <br/>
 这是不到2个月以来，美国第三次对中共做出此类指控，美国正在加大努力阻止中共的商业间谍行为。
 <br/>
 <br/>
 10月10日，美国司法部也公布了对中共一名特工的指控。指控他试图盗取通用电气旗下GE Aviation和其它一些公司商业机密，包括串谋和企图从事间谍活动以及窃取商业秘密等。如果指控他的4项罪名成立，他可能要面临着最多25年的牢狱生活和经济处罚。
 <br/>
 <br/>
 根据美国司法部介绍，这名特工叫徐延军（音译，Yanjun Xu），是中共江苏省安全厅副处长。为了执行任务，他还曾用过不同化名，并且试图用〝江苏省科学技术协会〞（JAST）的机构关系来掩盖他的真正职务。
 <br/>
 <br/>
 美国司法部说，几十年当中，美国航空公司在科研领域花费了巨资谋求发展。而中共通过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_情报人员.html" target="_blank">
  情报人员
 </a>
 的偷窃行为，获得美国来之不易的成果。
 <br/>
 <br/>
 而徐延军事件，仅仅是被曝光的部分，法广指出，这只是中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_盗窃.html" target="_blank">
  盗窃
 </a>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 的冰山一角。
 <br/>
 <br/>
 近来，美国曝光了多起中共间谍案，涉及到美国的军事、航天等等多个领域，情况相当严重。
 <br/>
 <br/>
 几周前，中国公民季超群（Ji Chaoqun）在芝加哥被捕，罪名是没有注册为外国代理人。法庭文件显示，季超群和徐延军之间是有联系的。季超群被指控在中共江苏国安厅高级情报官员的指导下，在美国境内招募工程师和科学家充当间谍。
 <br/>
 <br/>
 《华尔街日报》引述美国官员的话说，中共偷窃美国尖端技术的专利信息，每年给美国造成上千亿美元的损失，作案的主体以〝中国（中共）和中资公司为主〞。
 <br/>
 <br/>
 这也是美国曾经多次提醒中共的问题，但多年以来，中共仍我行我素、照偷不误。甚至在盗窃知识产权的基础上，还抛出了一个产业计划〝中国制造2025〞。
 <br/>
 <br/>
 美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 持续升级的背景下，中共特工接连遭美方抓捕，有分析认为有重要风向意义，表明美方向中共发出明确讯号，如果持续通过间谍活动，刺探美方机密，将会受到严厉打击。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：戴明）
</div>
