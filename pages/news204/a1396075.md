### 【禁闻】中共曾热捧的〝英雄〞 如今成访民
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】
 <u>
  中共曾热捧的〝英雄〞 如今成访民
 </u>
 <br/>
 10月18号，推特热传一条中共〝一级战斗英雄〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐良.html" target="_blank">
  徐良
 </a>
 到北京退役军人事务部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 的消息。
 <br/>
 <br/>
 据〝自由亚洲电台〞报导，徐良的一位好友证实，网传消息属实。他表示，是他把坐在轮椅上的徐良推进退役军人事务部的，徐良像其他老兵一样，填写了登记表。但这名友人不愿透露徐良上访的具体诉求。
 <br/>
 <br/>
 徐良30年前在中共对越战争中失去一条腿，1987年登上央视春晚，与人合唱《血染的风采》而走红，成为所谓的〝明星英雄〞，在中国大陆家喻户晓。
 <br/>
 <br/>
 如今这位中共热捧的〝英雄〞也成了访民，令很多大陆民众感叹。有网友说：〝血染过了，风采不再，惨遭抛弃。〞
 <br/>
 <br/>
 还有网友说：〝为中共卖命，下场就像用过的卫生纸一样。〞
 <br/>
 <br/>
 <u>
  澳洲华人团体抵制中共歌剧上演
 </u>
 <br/>
 中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红色歌剧.html" target="_blank">
  红色歌剧
 </a>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪湖赤卫队.html" target="_blank">
  洪湖赤卫队
 </a>
 》计划11月在澳洲演出，当地华人团体〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲价值守护联盟.html" target="_blank">
  澳洲价值守护联盟
 </a>
 〞发起抵制。他们认为这是中共渗透澳洲的又一形式，是在输出中共的意识形态。
 <br/>
 <br/>
 主办方在澳洲的宣传中使用《洪湖》作为英文方面的宣传名称，把带有明显中共政治色彩的〝赤卫队〞去掉了。
 <br/>
 <br/>
 澳洲价值守护联盟指出，该歌剧中有许多美化中共〝红军〞的歌曲，而〝红军〞杀害了许多无辜的地主和农民。
 <br/>
 <br/>
 澳洲价值守护联盟的一名成员说，这是中共的统战手段，歌剧名字的修改，是在掩盖要宣传的内容，让外界以为中共是拥护自由的。
 <br/>
 <br/>
 澳洲价值守护联盟10月初曾致信有关方面，呼吁取消演出。
 <br/>
 <br/>
 编辑/周玉林
