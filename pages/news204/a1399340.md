### 南海争端再升级？美国会报告首提制裁中共高官
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】美国国会最新报告提及中共在南海的军事化行动，并首次建议国会考虑制裁参与军事化活动的中国国营企业和个人。中共高官或再次列入美方制裁名单。此前，美方已要求中共撤除部署在南沙群岛的导弹。
 <br/>
 <br/>
 美国国会下属的美中经济与安全审查委员会（USCC）周三（11月14日）提交了2018年度报告。报告提及南海问题时，再次谴责中共在该海域的军事行动，并建议对参与军事化活动的中国国营企业和个人实施制裁。
 <br/>
 <br/>
 USCC副主席白嘉玲（Carolyn Bartholomew）说，〝这些公司可能包括协助建造这些设施，或管理这些设施，或经营这些设施等。〞
 <br/>
 <br/>
 参与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海军事化.html" target="_blank">
  南海军事化
 </a>
 的多为中共军工企业。如果美国启动制裁，这些国企的高层官员或列入制裁名单。这将是贸易战以来，美方第二次对中共高官实施
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济制裁.html" target="_blank">
  经济制裁
 </a>
 。
 <br/>
 <br/>
 美国国务院9月20日宣布，美国政府将对中共中央军委装备发展部及其部长李尚福中将实施制裁，冻结其在美国司法管辖区的资产，阻止其相关的外汇交易，并限制李尚福和他负责的中共中央军委装备发展部透过美国金融体系进入全球金融市场，李尚福本人还被禁止持有美国签证。制裁原因是中共军方最近进口了被美方禁止购买的俄罗斯导弹系统。
 <br/>
 <br/>
 当时港媒文章称，冻结中共高官财产的震慑力远超关税，是掐住了中共的〝七寸〞。
 <br/>
 <br/>
 此外，美方日前已经在南海问题上对北京加大了施压。在11月9日的中美第二轮外交安全对话不欢而散后，美国务院发表声明，要求中方撤除部署在南沙群岛的导弹。这是美国首次就南海问题向中共提出此种要求。
 <br/>
 <br/>
 外界分析，中共2015年就已承诺不将南海军事化，美国不过是要中共兑现承诺。
 <br/>
 <br/>
 日前，美国副总统彭斯表示，美方已经将南海问题列为中美贸易和解的条件。这显示中美对抗已不再单纯是贸易问题。
 <br/>
 <br/>
 此外，USCC周三报告中还指出，中共的5G无线技术不断发展，连接到互联网的设备数量也不断激增，这些网络技术和设备可能被用于对美国发动网络攻击。报告建议美国政府机构警惕中共破坏美国政府系统的企图。
 <br/>
 <br/>
 报告特别提到中国电信设备巨头华为及中兴通讯。
 <br/>
 <br/>
 军事方面，报告指出，中共目前正在第二岛链范围与美国海、陆、空军力进行竞争，在2035年前或将在整个印太地区与美国展开对抗。
 <br/>
 <br/>
 台湾问题上，报告建议国会修法，防止中共在台湾问题上对美国企业施压，包括在中方实施报复时，美方启动对等报复。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：/责任编辑：明轩）
