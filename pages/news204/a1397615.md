### 肖建华吴小晖赖小民以〝金三角〞关系盘踞股坛
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】日前，独立股评人David　Webb，踢爆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_明天系.html" target="_blank">
  明天系
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融系.html" target="_blank">
  华融系
 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民生系.html" target="_blank">
  民生系
 </a>
 ，以一个〝金三角〞关系
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_盘踞.html" target="_blank">
  盘踞
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股坛.html" target="_blank">
  股坛
 </a>
 ，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港股坛.html" target="_blank">
  香港股坛
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_兴风作浪.html" target="_blank">
  兴风作浪
 </a>
 。如今，虽然三家集团公司的掌门人肖建华、赖小民、吴小晖已落网，但这一连串事件并非终局。
 <br/>
 <br/>
 有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股坛.html" target="_blank">
  股坛
 </a>
 长毛〞之称的独立股评人David Webb发表新作〝谜网26〞，踢爆了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融系.html" target="_blank">
  华融系
 </a>
 与民生银行触发香港细价股风暴、廉政公署及证监会相继介入调查，造成哄动，当中还有一只〝大老虎〞未被点相，就是去年初被带返内地调查的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_明天系.html" target="_blank">
  明天系
 </a>
 掌舵人肖建华。
 <br/>
 <br/>
 明天系掌门人肖建华，于去年中国新年大年三十，被中共警方从香港带走后，至今已失踪一年多。
 <br/>
 <br/>
 据陆媒报导，肖建华掌控着9家上市公司，控股、参股了30家金融机构，总资产近万亿元人民币。中共内部评估高达2万亿（人民币，下同）资产。
 <br/>
 <br/>
 〝明天系〞之所以能成为首家拥有金融全牌照的民营资本集团，与肖建华成为替权贵家庭牟利的中间人、〝白手套〞高度相关。只是其财技操作极为隐密，擅长化整为零，一般经旗下多间公司分别持有小量上市公司股权，以致于持股结构形如谜网。
 <br/>
 <br/>
 此举可避免按监管制度向公众披露身份，同时另找体系外的合作伙伴入股，作单一主要股东，方便运作。
 <br/>
 <br/>
 David Webb说，明天系与华融系过去几年在港股过从甚密，在2014年，当时华融正在搭建旗下首家境外上市多牌照金融控股平台，需要壳股，相中了天行国际，斥资近4.68亿元认购天行配售股份，重组成现时的华融金控。
 <br/>
 <br/>
 转手前的天行国际，曾由彭晓东持股，与官媒《第一财经》报导中提及的明天系〝发展支持部〞其中一名负责人同名。
 <br/>
 <br/>
 曾任天行董事的李海枫及孙大睿，前者是肖建华在北大法律系的师弟，曾在北大方正工作，还在明天系旗下的恒富任主席；孙大睿被《财新》踢爆曾在明天系旗下的华资实业任代理董事长。
 <br/>
 <br/>
 2014年10月，天行宣布向华融以每股0.25元大手配股前的一段时间，3人似有默契地断绝与天行之间的关系，彭晓东将佳元持股转手，孙大睿与李海枫相继辞任董事职位。
 <br/>
 <br/>
 3人身处天行董事局至华融入股前后，伴随业务可能变化的消息，天行股价不时有大幅波动；根据财新网统计，最初股价约1.6元，累积涨至9元水平，升幅逾4倍。
 <br/>
 <br/>
 而明天系与华融还有其他合作，2014年6月，华融有份出资成立的私募基金〝保华嘉泰〞，计划以每股0.36元认购江山控股约65.3亿股并入主，涉资23.5亿元。
 <br/>
 <br/>
 而肖建华的师弟李海枫，正是保华嘉泰的普通合伙人Pohua JT Capital Partners Limited董事，沾手保华嘉泰的营运管理。华融的另一只壳股华融投资（2277），也是由明天系手上收购而来。
 <br/>
 <br/>
 在〝谜网26〞中特别点名质疑民生银行，为何已有香港银行牌照，却另设民银资本。从民银资本揭出了明天系、华融系、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民生系.html" target="_blank">
  民生系
 </a>
 和安邦系互有关系。
 <br/>
 <br/>
 民银资本的前身是天顺证券，在2016年底被民生银行和华融相中，后来获入股兼改组而成。那时，安邦前董事长吴小晖尚未被捕，安邦是民行的最大单一股东，持股占17.84%；明天系旗下的华夏人寿则持5.74%，两者因此有交集。
 <br/>
 <br/>
 而安邦与华融的关系，建立的可能更早，华融2015年到香港上市时，安邦透过持股29.98%的远洋集团沾手华融IPO，涉资53亿元。
 <br/>
 <br/>
 在这一连串动作背后，都有与华融前董事长赖小民的人事关系，而且与前〝私募一哥〞徐翔也极有渊源，徐于2015年8月以个人名义进军港股，同年10月底被捕。
 <br/>
 <br/>
 徐翔被判刑后，中国证券投资基金业协会公布决定书，披露了徐曾参与操纵13间中国上市公司的股价交易，其中一间就是在深交所挂牌的中弘。
 <br/>
 <br/>
 2015年发生股灾之后，大批政界和财金界巨头遭拘捕；而香港细价股的炒卖也巧合的在过去3年间渐渐沉寂下来。
 <br/>
 <br/>
 海外多家媒体曾爆料，肖建华被称为中国股市枭雄，号称手握超过800亿人民币的调度实力，肖因涉及2015年夏秋季那场〝股灾〞而被调查。
 <br/>
 <br/>
 那场股灾被指是中共反习联盟策划的一次〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融政变.html" target="_blank">
  金融政变
 </a>
 〞。事件背后策划者被指涉江系多个新旧常委和江泽民家族。
 <br/>
 <br/>
 肖建华、吴小晖、徐翔、赖小民等人均被指涉及2015年中国股灾。
 <br/>
 <br/>
 翻查资料，徐翔2017年初被判有期徒刑5年半，肖建华同年大年三十前后被带回大陆；吴小晖今年5月被判有期徒刑18年，赖小民则在10月中被中纪委双开。
 <br/>
 <br/>
 上述一连串事件并非终局，肖建华被带回大陆当月，便传出肖案已被当局定为〝中南海头号要案〞。而肖建华抓回去调查，目的就是为了牵出更多江派要员，包括江泽民、曾庆红、张德江、刘云山等人。
 <br/>
 <br/>
 据称，肖建华被带回北京后不久，就已经供出大批江派〝大老虎〞贪腐证据。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：戴明）
