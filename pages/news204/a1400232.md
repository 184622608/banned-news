### 〝合时代〞投资受害人再到广东维权
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】11月11号和12号，几十名〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_合时代.html" target="_blank">
  合时代
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网贷平台.html" target="_blank">
  网贷平台
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资受害人.html" target="_blank">
  投资受害人
 </a>
 到广东深圳维权。〝合时代〞在7月16号停业， 老板、高管失联，平台资金去向不明，涉案金额达26亿，受害投资人有2万多。
 <br/>
 <br/>
 大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网贷平台.html" target="_blank">
  网贷平台
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_合时代.html" target="_blank">
  合时代
 </a>
 〞在今年7月的倒闭潮中停业，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资受害人.html" target="_blank">
  投资受害人
 </a>
 多次到公司所在地深圳维权。11月11号和12号，又有数十名投资受害人先后到广东省公安厅和深圳云街维权，依然没结果。
 <br/>
 <br/>
 〝合时代〞投资受害人杨女士：〝立案很久了，派出所一直没有更新，资金去哪里了也不知道。这后台比较硬嘛，好像跟中央谁有什么关连好像，像其他平台‘投之家’嘛，跟我们一起立案的，人家都赔了一些钱。深圳他不让你拍照，几百个、满多个。警察会打人的，把你抓进去或干嘛的。〞
 <br/>
 <br/>
 投资受害人透露，〝合时代〞原董事长黄林明在2016年3月时退出，以1元合同将公司转手，卷走13.6亿投入房地产，黄林明作为天朝集团董事长投资了云街一期、二期开发，因此投资受害人也多次到该楼盘售楼处维权 。
 <br/>
 <br/>
 杨女士：〝有些人，像结婚、生孩子、上有老下有小的，把全部家产投进去了，如果孩子上学或者是老人生病了，那怎么生活？有些平台很多还自杀了，因为确实家破人亡了，有些人他是瞒着家里面去投的。〞
 <br/>
 <br/>
 有〝合时代〞投资受害人表示，案件经侦没进展，他们的QQ群和微信群都被封，大陆媒体也不报导。
 <br/>
 <br/>
 〝合时代〞难友徐先生：〝我投了十万块钱，投了好几年，最近刚刚才发现‘合时代’已经那个了，我现在连报警都没有报警，我很忙，我在上海没法去，我这边没得到任何消息。〞
 <br/>
 <br/>
 据《大纪元》报导，〝合时代〞的法人经过多次变更。黄林明退出后，法人变更为马文亮，再变更为陈海伟，3个月后，2017年1月，又变更为孟德禄，平台倒闭后，孟德禄潜逃境外2个多月后回国自首。 陈海伟在7月22号带着儿子携款出逃美国，他的妻王岩和弟弟陈海涛仍逍遥法外，并涉嫌协助转移资产。〝合时代〞投资受害人曾到陈海伟河南老家拉横幅抗议。
 <br/>
 <br/>
 新唐人记者熊斌、柏妮采访报导
</div>
