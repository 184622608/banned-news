### 新西兰反对派爆政治献金丑闻 金主是华裔富商
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】新西兰上周爆发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治献金.html" target="_blank">
  政治献金
 </a>
 丑闻，最大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反对派.html" target="_blank">
  反对派
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家党.html" target="_blank">
  国家党
 </a>
 党魁布里吉斯被爆收受华裔富商张乙坤献金，并试图将献金拆成小额捐款以躲避申报，引发各界对于中共企图影响新西兰政坛的担忧。
 <br/>
 <br/>
 〝纽约时报〞（New York Times）报导，中间偏右
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家党.html" target="_blank">
  国家党
 </a>
 （National Party）前党员罗斯（Jami Lee Ross）上周向警方投诉，并在网路公布今年6月他和党魁布里吉斯（Simon Bridges）的电话录音档，当时两人讨论华商张乙坤（Zhang Yikun）的献金。
 <br/>
 <br/>
 报导说，布里吉斯收受张乙坤10万纽元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治献金.html" target="_blank">
  政治献金
 </a>
 ，且试图把资金拆成小额匿名捐款。新西兰法律规定，捐款少于1万5000纽元的捐献人身分可以保持匿名。
 <br/>
 <br/>
 不过，录音档无法确实证明布里吉斯要求把这笔献金拆成小额款项，且布里吉斯也否认这件事。新西兰警方正在调查国家党是否隐匿申报张乙坤的捐款。
 <br/>
 <br/>
 国家党声称没有做出任何违法行为，但还没解释那通电话中讨论的政治献金如何处置。
 <br/>
 <br/>
 据了解，张乙坤从中共人民解放军退役后，于2000年移居新西兰之前，担任中共省级谘询小组负责人。
 <br/>
 <br/>
 移居后，张乙坤任新西兰潮汕总会主席，这个组织成立于2014年，且是中共和新西兰之间的重要媒介。
 <br/>
 <br/>
 这起丑闻是新西兰容易受到政治干预的最新例证，中国正在太平洋地区追求更大的影响力。
 <br/>
 <br/>
 新西兰去年也爆发类似问题，国家党华裔国会议员杨健（Jian Yang）被爆曾在中国的间谍学校任教。杨健则否认自己是间谍，目前他仍是新西兰国会议员。
 <br/>
 <br/>
 分析家和盟邦担心，中共可透过廉价的政治捐献产生影响力，且不会让各界警觉新西兰政治体系遭到侵害。
 <br/>
 <br/>
 经常撰文评论中国的马丁（Miguel Martin）表示，政治捐献是中共取得〝影响新西兰政策的途径〞的权宜之计。
 <br/>
 <br/>
