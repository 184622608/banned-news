### 财新女记者采访碳九污染 被当地公安〝精准抓嫖〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】中国大陆一家媒体的女记者近日在福州泉州采访〝碳九〞泄漏事故期间，其独住的酒店房间半夜遭当地警方强行闯入〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_精准抓嫖.html" target="_blank">
  精准抓嫖
 </a>
 〞。事件公开后，引发舆论的强烈反响。日前，涉事警方终于发通报承认〝执法不当〞，但社会舆论对缺乏监督的公权力被滥用的恶劣现象的反弹并未就此平息。
 <br/>
 <br/>
 当地时间11月20日，福州省泉州市公安局发布通报，承认该市泉港公安分局的警员9天前半夜以〝抓嫖〞为由强行闯入酒店房间，骚扰正在该地对〝碳九〞泄漏事故进行调查采访的《财新周刊》女记者事件，系警方〝执法不当〞。
 <br/>
 <br/>
 这份通报中表示〝歉意〞后，声称涉事的泉港公安分局副局长、山腰派出所所长陈宾阳已被要求作检讨，并让涉事警员陈华山〝停职检查〞。
 <br/>
 <br/>
 这是在女记者被〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_精准抓嫖.html" target="_blank">
  精准抓嫖
 </a>
 〞事件曝光并引发社会舆论的强烈反响9天后，警方迫于社会舆论的压力作出的一个表态。
 <br/>
 <br/>
 但舆论认为，类似地方当局为掩盖事故丑闻而滥用权力侵犯公民权利的做法，在中国并非一时一地的偶发事件，而是在全国各地都普遍存在的。在缺乏来自公民社会的约束和监督机制下的公权力〝任性〞胡为的问题，并非一个徒具形式的道歉就可以解决的。
 <br/>
 <br/>
 据公开的资讯，〝精准抓嫖〞事件发生于本月11日深夜。当时《财新周刊》女记者周辰正在泉州市泉港地区对当地近日发生的〝碳九〞泄漏重大事故的前因后果进行调查采访。当天白天的采访过程中，周辰已发现一直有腰挂执法纪录仪的人员在跟踪自己。当晚，泉港区宣传部长邀约她见面〝聊聊〞。周辰婉拒了这个邀约，当时邀约她的官员曾半带威胁地说：〝你这是怕我们领导公关你啊！〞
 <br/>
 <br/>
 不料就在当夜11时许，周辰所住的酒店房间突被人刷卡打开，有4名身穿警察制服的男子强行闯入进行搜查。这4人自称是派出所的警员，并勒令记者交出身份证。当时已经上床准备就寝的女记者无奈只能裹着睡衣起身，从包里找出身份证递给这些人查验。
 <br/>
 <br/>
 这些警员在离开时声称只是针对酒店进行的〝例行检查〞，但酒店工作人员稍后打电话向周辰致歉时，却承认是警察直接命令前台交出周辰房间的房卡，并没有〝例行检查〞其他房间。周辰事后直言，〝没想到电视里酒店抓嫖的一幕会降临到我这个女记者头上〞。
 <br/>
 <br/>
 女记者将自己的遭遇在网络上曝光后，社会舆论一片哗然。
 <br/>
 <br/>
 无需讳言，在中国大陆，全国各地基本上一但发生类似碳九泄露这样的重大事件后，地方政府为了掩盖〝家丑〞，往往首先采取的行动就是想方设法地〝捂盖子〞，动用警力在车站机场、大路小径布控，限制任何媒体记者的采访，或肆意阻断记者接触事件当事人的渠道，之后再弄出些漏洞百出的说辞来忽悠公众。
 <br/>
 <br/>
 中国民众则对中共各级政府这种滥用权力掩盖事实真相的作法早已十分反感。因此女记者半夜被警方以〝抓嫖〞的藉口进行骚扰威胁的遭遇，获得了社会舆论普遍的同情与支持，网络舆论亦一边倒地谴责泉州地方当局与警方的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_滥权.html" target="_blank">
  滥权
 </a>
 行径。
 <br/>
 <br/>
 （记者李明综合报导/责任编辑：秦锋）
