### 《人民日报》杂志总编19楼离奇坠亡 出事前神色憔悴
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】中共喉舌《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民日报.html" target="_blank">
  人民日报
 </a>
 》附属刊物《新闻战线》总编辑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡欣.html" target="_blank">
  胡欣
 </a>
 ，6日从报社高楼层坠楼。官方仅公布其死讯，网传其跳楼消息遭秒杀。港媒引述其同事透露，事发前胡欣憔悴失眠。
 <br/>
 <br/>
 大陆社交媒体消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡欣.html" target="_blank">
  胡欣
 </a>
 6日下午1时在报社19楼一跃而下，救护车赶到时早已死亡，享年66岁。但透露消息的网民没有提及胡欣轻生的原因。
 <br/>
 <br/>
 《新闻战线》杂志与《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民日报.html" target="_blank">
  人民日报
 </a>
 》报社在同一栋建筑内。港媒引述消息指，有《人民日报》员工透露，日前在报社撞见胡欣，看见她面色憔悴，她还自称近日失眠，〝老睡不着觉〞。
 <br/>
 <br/>
 胡欣从1990年开始任职于《人民日报》理论部，从事新闻工作，并曾在《人民论坛》杂志担任总编辑，后于2008年担任《新闻战线》总编辑。
 <br/>
 <br/>
 中共宣传系统此前多年被江派人马掌控。习近平上台后，中共〝笔杆子〞一直在给其搅局。虽然当局不断整肃各大官媒，但在内斗乱局中，宣传系统很多声音依然和北京当局〝不同调〞。
 <br/>
 <br/>
 当局的反腐高压下，近期中共官员的自杀频率增高。
 <br/>
 <br/>
 胡欣并非第一个自杀的《人民日报》管理层人员。早在2012年8月，网上曾传出《人民日报》大地副刊主编徐怀谦自杀的消息。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：秦锋）
</div>
