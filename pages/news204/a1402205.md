### 西安鱼化寨村民抗议强拆遭镇压
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】陕西西安市西郊最大的城中村鱼化寨，遭到当地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政府强拆.html" target="_blank">
  政府强拆
 </a>
 ，12月2号，二、三百人上街游行示威，当晚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑保安.html" target="_blank">
  黑保安
 </a>
 进村暴打村民，被大批村民围堵。数百警察到场，对村民喷辣椒水，抓走八名村民，以涉嫌寻釁滋事的罪名拘留。
 <br/>
 <br/>
 12月2号白天，鱼化寨村民上街游行，被身穿黑色制服的保安一路跟踪，晚上他们开车进村，私闯民宅，暴打村民，砍伤一人，村民越聚越多，高喊打倒黑社会的口号，村民报警不出警，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑保安.html" target="_blank">
  黑保安
 </a>
 被围堵报警，来了几百名警察，抓走八名村民。
 <br/>
 <br/>
 鱼化寨村民周先生：〝上街有二、三百人，都是女人，男人不敢上去，黑社会的保安，黑保安那多啊，辣椒水喷，小管的那个警用辣椒水，有八个被抓，打受伤的多了，在医院。〞
 <br/>
 <br/>
 鱼化寨村民表示，鱼化寨的商户已全部被撵走，当局以违建的名义拆了很多房子，只剩下村民的房子还没拆。现在村里有一千五百户，共五、六千人，买菜要到几里外买，生活极不方便。
 <br/>
 <br/>
 周先生：〝他没有政府批文就要拆，再一个他先跟村民代表谈了，给村民每户420个平方，后来就改成330，政府出尔反尔嘛，村民就不答应了。比土匪都恶，谁敢不让拆啊，一来四、五千黑保安，外来的30万人，基本上是赶完了，成了空村子了。〞
 <br/>
 <br/>
 10月29号起，政府雇傭黑社会几千人，几天内拆了鱼化寨两条商业街，百家商户、十多年的菜市场，两间幼儿园。村民透露，强拆中打死一个，气死一个人，还有一个人自杀。
 <br/>
 <br/>
 鱼化寨高先生：〝拆迁不能做生意啊，商铺之类的都拆了，那个里面的人挺多的，加上流动人口的话，估计要在30万左右呢，只能找别的地方啊，周边房价涨的厉害，这是必然的嘛。〞
 <br/>
 <br/>
 据大陆媒体报导，鱼化寨素有西安〝小香港〞之称，占地约3000多亩。鱼化寨包括河东村、福谦堡、鱼南村、鱼西村、八家巷等村落被纳入拆迁范畴。当局计划在今年11月30号前，完成拆迁补偿协议的签订并开展拆迁行动，12月31号前，完成整村拆除及村域范围内地面附着物的清表工作。
 <br/>
 <br/>
 新唐人记者熊斌、陈杰采访报导
