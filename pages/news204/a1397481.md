### 港媒：川普发最后通牒 中南海如何接招
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）警告，中美首脑将在阿根廷G20峰会上会晤，如果会谈失败，他将对2,670亿美元的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国商品.html" target="_blank">
  中国商品
 </a>
 加征
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 。有港媒说，川普孤立中共后，又发出最后通牒，接下来，就看北京如何接招了。
 <br/>
 <br/>
 彭博通讯社引述美国官员表示，如果中美首脑在阿根廷举行的20国集团会议期间会面时，仍未能消除贸易矛盾，美方计划最快在12月初宣布对所有剩余的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国商品.html" target="_blank">
  中国商品
 </a>
 加征
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 10月29日晚间接受福克斯新闻（Fox News）采访时说，有可能与中方就贸易达成〝重大交易〞，但他也警告说，如果有必要，还准备再对中国2670亿美元的输美产品征税。
 <br/>
 <br/>
 川普政府自年初将中共公开列为竞争对手以来，全面扭转过去40年来绥靖退让的美国对华政策，改采强势对抗的追击战略，藉此争取美中两国的公平贸易关系，守护美国人民与美国权益。
 <br/>
 <br/>
 目前美方对中方征收25%高关税的商品规模是500亿美元，一旦2000亿美元措施推出，规模将达到2500亿美元。然而，如果川普再加码2670亿美元的话，美国对中方的贸易战场规模将达到空前的5170亿美元。
 <br/>
 <br/>
 而去年美国从中国进口商品的总金额约为5055亿美元，换句话说，到时候，几乎所有出口至美国的中国商品，都必须付出高额关税代价。
 <br/>
 <br/>
 香港苹果日报10月31日刊发署名评论文章说，面对川普一而再、再而三施压，北京难得没有暴跳如雷。
 <br/>
 <br/>
 中共外交部发言人耿爽后来也声称：中共不是吓大的，中方有能力、有信心打赢贸易战。
 <br/>
 <br/>
 但随着美国一波又一波的加征关税、制裁中国科技公司、制裁中央军方高层，使得中共政治、经济乱象频生。
 <br/>
 <br/>
 在美国宣布对中国2,000亿商品加征关税后，中共换了一种说法，声称不屑以牙还牙。不过，在中共外长王毅9月在联合国大会上仍宣称，中共在贸易战上不惧怕施压。
 <br/>
 <br/>
 苹果评论文章说，其实，川普一直给习近平面子，不时宣扬两人的友谊，甚至在启动贸易战后，还不顾国会议员的反对放生中共的科技巨头——中兴通讯公司。
 <br/>
 <br/>
 但是，北京却藉邀朝鲜领导人金正恩访华搅乱美朝核谈、否决刘鹤访美达成的解决贸赤问题协议，终让川普认清了中共的欺骗手段，直至联合欧盟国家围堵北京。
 <br/>
 <br/>
 文章说，北京一再错过与美国、与川普和解的机会，一方面是死要面子活受罪，宁可在4月份的博鳌论坛宣布一系列开放措施，也不愿在中美谈判时直接让步，宁可推翻刘鹤在美国达成的停战协议，转而要14亿民众共克时艰。
 <br/>
 <br/>
 而川普接连
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退群.html" target="_blank">
  退群
 </a>
 ，北京以为可以接任群主。但川普转身另建美加墨自贸群、美欧自贸群，把中共撇在群外。北京可以到中东大撒币，玩非洲万邦来朝，然而挤进北京群的多是乞丐国王，已是优劣分明。
 <br/>
 <br/>
 至于11月5至10日即将在上海举行的首届进口博览会，北京再玩八方来仪，川普更是不奉陪不给面子了，连一个高级官员都不派。而英国、法国、德国、意大利、日本等西方绝大多数国家领导人也都将缺席。
 <br/>
 <br/>
 文章说，北京把解决中美贸易争端视为彰显其国际、国内权力的踏脚石，只想着以牙还牙的痛快，浑不知没有对等报复的贸易额、科技实力、国际关系，结果是骑虎难下。
 <br/>
 <br/>
 如今
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 已经升级为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全面对抗.html" target="_blank">
  全面对抗
 </a>
 ，中美进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 时代，接下来，北京将如何接招，公众将试目以待。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉)
