### 鞍山反垃圾焚烧厂遭镇压 警察买水被拒
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】12月7号，辽宁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鞍山.html" target="_blank">
  鞍山
 </a>
 海城市腾鳌镇持续罢市、罢工、罢课，抗议建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_垃圾焚烧发电厂.html" target="_blank">
  垃圾焚烧发电厂
 </a>
 ，当局调派更多警力持续进行镇压，民众只要上街聚集就会被打被抓。为抗议警方镇压，当地商店拒绝卖水给警察，加油站也拒绝给警车加油。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鞍山.html" target="_blank">
  鞍山
 </a>
 腾鳌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_垃圾焚烧发电厂.html" target="_blank">
  垃圾焚烧发电厂
 </a>
 项目下月即将动工，当地居民走上街头抗议，5号起，当局调集大批武警抓捕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权民众.html" target="_blank">
  维权民众
 </a>
 ，爆发激烈冲突。
 <br/>
 <br/>
 据〝大纪元新闻网〞报导，6号被抓走的村民有五、六十人，所有人被迫签同意书后在有人担保的情况下才获释，大概有十多人被打伤。一名孕妇被打伤住进医院保胎，其家人也被抓走。
 <br/>
 <br/>
 腾鳌镇匿名店家：〝全部罢工，孩子上不了课现在，得有四天左右了，特警特别多，继续镇压，一上街肯定会打，你只要是阻挡道路啊啥的，他直接就打，具体抓人的数量我不太了解，但是抓了挺多人，他们太没有良心了，他不考虑老百姓，我们老百姓根本不可能卖给他东西。〞
 <br/>
 <br/>
 网传视频显示，数名穿制服的警察到一家商店买水被老板娘拒绝。
 <br/>
 <br/>
 商店老板娘：〝不卖你们水，你们走吧，咱们这儿不欢迎这样的人，你们都不为老百姓做主，还卖你什么水?〞
 <br/>
 <br/>
 7号上午，镇上居民仍坚持抗争，但害怕武警镇压，都不敢上街。
 <br/>
 <br/>
 腾鳌镇居民王先生：〝今天还是有，但是最新的我还不知道。〞
 <br/>
 <br/>
 腾鳌镇面店老板娘：〝我没看着，现在门口啥也没有。〞
 <br/>
 <br/>
 腾鳌镇是一个人口密集的地方，有12万人口，该项目选址离居民区太近，因此遭到当地民众强烈反对。
 <br/>
 <br/>
 腾鳌镇匿名店家：〝都在抗议，因为这个污染太严重了，影响我们居民的身体健康，政府打压百姓，政府想要把这个事压下来，到处都是特警，区域网全部屏蔽，不让往外发消息。〞
 <br/>
 <br/>
 公开资料显示，该垃圾焚烧发电厂项目，总投资6亿元，日处理能力将达到1500吨，将于2020年投产，地址位于鞍山市腾鳌工业园区内，距离腾鳌镇黄士村不足1公里。
 <br/>
 <br/>
 新唐人记者熊斌、钟元采访报导
 <br/>
