------------------------

 【新唐人2018年11月02日讯】下面是今天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球看中国.html" target="_blank">
  全球看中国

 <br/>
 <br/>
 1. 美国之音
 <br/>
 美国驻华大使馆官方微博周五宣布，布兰斯塔德大使将对网友的一些提问和留言作出回复。几小时后，就有数以百计的网民留言提问，到傍晚已获近千点赞、转发和留言。很多网民将矛头直指中共贪官。
 <br/>
 <br/>
 2. 明慧网
 <br/>
 去年12月，四川省公安厅和专门迫害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 的机构〝610〞指使巴中市公安局非法抄家并绑架了包括两位八旬老人在内的十多名法轮功学员。9月28日，巴州区检察院非法起诉其中十人，巴州区法院11月7日将进行所谓的〝庭审〞。
 <br/>
 <br/>
 3. 自由亚洲电台
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际人权组织.html" target="_blank">
  国际人权组织
 </a>
 自由之家周四发布《2018年网络自由报告》，中国在65个参评国家中垫底。报告指出，全球网络自由度已连续8年走跌，这与中国向全球输出其网络审查和监控模式密不可分。
 <br/>
 <br/>
 4. 大纪元时报
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无声的入侵.html" target="_blank">
  无声的入侵
 </a>
 》作者汉密尔顿（Clive Hamilton）日前在澳洲塔斯马尼亚大学演讲时，警告塔州政府提防中共渗透。他认为，中共对塔州的农业用地及靠近南极洲的地理位置感兴趣，并通过在塔州组建统战团体和支持华裔参选来扩大其政治影响力。
