### 【禁闻】中共称退保费促就业 专家：再放烟幕弹
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 下行，大陆面临
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业潮.html" target="_blank">
  失业潮
 </a>
 ，中共国务院日前出台政策，声称要给不
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裁员.html" target="_blank">
  裁员
 </a>
 或少裁员的企业，返
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业保险.html" target="_blank">
  失业保险
 </a>
 费等，以促进
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业.html" target="_blank">
  就业
 </a>
 。但专家认为，这只是一颗安抚民心的烟幕弹，在目前大量民企倒闭和裁员的情况下，无济于事。
 <br/>
 <br/>
 12月5号，中共发布了《国务院关于做好当前和今后一个时期促进
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业.html" target="_blank">
  就业
 </a>
 工作的若干意见》。
 <br/>
 <br/>
 意见称，为了促进就业，对不
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裁员.html" target="_blank">
  裁员
 </a>
 或少裁员的企业，可返还其上年度缴纳
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业保险.html" target="_blank">
  失业保险
 </a>
 费的50%；对个人和小微企业可提供政府性融资担保。同时还将扩大就业见习补贴范围，2019年1月1号开始，实施〝百万青年见习计画〞，将就业见习补贴范围，由未就业的高校毕业生扩展至16至24岁失业青年。
 <br/>
 <br/>
 清华大学副校长、经济学家杨斌：〝目前经济情况其实特别紧张，要失业率再高的话，反过来更使经济紧张。在我看来，这是信号，想作刺激经济政策，但这一两年来说了很多这种想刺激经济的政策，讲了半天没有落到实处，对民营企业的支持，只是说的空话，这次落实到什么程度，有多大力度就不好说了。〞
 <br/>
 <br/>
 前不久，中共国家税务总局原副局长许善达透露，全国的社保已经收不抵支。外界也质疑中共用什么返还失业保险费。
 <br/>
 <br/>
 大陆问题观察人士李国伟：〝实际上这个政策是没有任何意义的，它只是说返还（失业保险费），但是返还的时间、地点、人群都是不确定的，这个有可能只是个烟幕弹，只是好像告诉你我要补贴了，但实际上100个失业只有1%的人可能拿到这个补贴。〞
 <br/>
 <br/>
 目前中国裁员的消息满天飞。继万科喊出活下去口号后，据称金融、地产、互联网巨头全都卷入了裁员漩涡中。
 <br/>
 <br/>
 据〝财新网〞官方微博披露，大陆就业寒冬已经来临，4-9月份，前程无忧的招聘广告数量从285万骤减到83万条。在消失的202万条招聘广告中，500人以下的中小微企业高达656%。
 <br/>
 <br/>
 随着代工之王富士康、苹果组装大厂和硕等出口企业的生产基地转移，大陆就业人数急遽减少。
 <br/>
 <br/>
 数据显示，上半年，大陆金融行业有9家券商共裁员4186人；房地产行业在影子银行被整顿、融资成本增加后，瘦身、裁员成为也第一选择；互联网巨头也相继卷入大裁员传闻。
 <br/>
 <br/>
 清华大学副校长、经济学家杨斌认为，国进民退打压民企造成民营企业纷纷倒闭，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业潮.html" target="_blank">
  失业潮
 </a>
 来临的原因之一，而中共不自量力和美国打贸易战更使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 雪上加霜。
 <br/>
 <br/>
 杨斌：〝另外中美贸易战，本来很多企业70%靠出口，关税一升10%，那企业最后倒闭了，一倒闭不就失业了吗，再说失业之后你的消费能力不行了，缺乏消费以后就涉及到别的企业了，就循环了嘛，所以失业啊、企业凋敝、下岗都很正常啊。〞
 <br/>
 <br/>
 至于中共国务院〝意见〞所说的，给符合条件的个人和小微企业，提供创业担保贷款，大陆问题观察人士李国伟认为，那也是骗人的。
 <br/>
 <br/>
 李国伟：〝因为他原有的干了十几年的机床这个工种已经不复存在，你让他跳出来再去创业，无异于让他飞蛾扑火，因为他不会，所以返乡的创业是骗人的，骗这些中国最底层的农民工，还有这些年轻人，告诉你，你在路边卖个红薯也叫创业，城管这帮土匪这帮流氓马上来抓了。〞
 <br/>
 <br/>
 李国伟说，他所在省城那些昔日的亿万富翁，因企业倒闭现在无所事事的比比皆是，无法过年的失业人群更是不计其数，他无法想像陕西边缘地区，那些即使在所谓经济好时都穷得让他落泪的乡下人，明年将如何活下去。
 <br/>
 <br/>
 采访编辑/刘惠
