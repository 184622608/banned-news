### 河南4杂技艺人高空表演时坠地  当场吐血昏迷（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】日前，河南一家酒店举行庆祝活动，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_4名杂技艺人.html" target="_blank">
  4名杂技艺人
 </a>
 受邀表演。不料，在表演时突发意外，4人表演攀附的支架从4、5米的高空坠落，4人同时头朝下坠地受伤，其中1人当场吐血，昏迷不醒。
 <br/>
 <br/>
 据陆媒报导，11月23日上午11点左右，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_河南商丘市.html" target="_blank">
  河南商丘市
 </a>
 一家温泉酒店举行庆祝活动，酒店方请了杂技艺人现场表演，不料表演时发生意外。
 <br/>
 <br/>
 从网传的视频可见，现场一部起重机上悬挂了一个支架，4名身穿黄色表演服装的女杂技艺人，正在高空中的支架上表演。
 <br/>
 <br/>
 当她们表演倒吊动作时，整个支架突然脱落坠下。瞬间，4人头朝下，从4、5米的高空堕落。坠地后，4人倒在地上，一动不动，脚部仍被绳缠在支架上。其中一名女子更口吐鲜血，浑身抽搐，随即昏迷不醒。围观民众纷纷上前施救。
 <br/>
 <br/>
 事发后，现场民众当即报警叫救护车。4人随后送院抢救，其中一人伤势较轻，当天已出院，另外3人还在医院观察。其中一人牙齿脱落、一人上臂骨折、一人肺部挫伤，但无生命危险。目前，事件原因正在调查中。
 <br/>
