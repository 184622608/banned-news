### 江泽民大赞〝花山谜窟〞 黄山老僧透玄机
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】19年前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 到〝天下第一奇山〞黄山脚下，发现了一处结构杂乱阴森的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_石窟群.html" target="_blank">
  石窟群
 </a>
 。次年，江专程前去并题名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花山谜窟.html" target="_blank">
  花山谜窟
 </a>
 〞，且连声感叹：〝真是太绝了！〞〝这真是个大宝贝，要是宣传到国外去了不得！〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 在1999年7月正式发动迫害法轮佛法修炼人的前夕，在〝天下第一奇山〞黄山脚下，即黄山市屯溪区与歙县交界处的新安江畔，发现了一处充满神秘和诡异氛围的奇特之地，古徽州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_石窟群.html" target="_blank">
  石窟群
 </a>
 。
 <br/>
 <br/>
 2000年4月，古徽州石窟群景区破土动工，并于同年9月28日对外正式营业，江泽民次年亲自题名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花山谜窟.html" target="_blank">
  花山谜窟
 </a>
 〞。
 <br/>
 <br/>
 这个被认定为是目前中国已经发现的规模最大、面积最大的古石窟遗址，却既没有名头能够竞争自然景观，也没有机会入选人文纪录，因为谁也说不清它是什么，只能用〝神秘、诡异、奇特之地〞等字眼来描述。
 <br/>
 <br/>
 经过发掘发现，花山谜窟是由36个大小石窟组成的巨大石窟群，目前只开挖出5、6个。与敦煌石窟相比，石窟里面没有壁画和佛像，也没有文字。
 <br/>
 <br/>
 石窟内空间奇大，结构怪异、杂乱，迂回曲折，有的层层相迭，洞中套洞；有的石柱撑立，窟内有水，深不见底。水中有红、黄、蓝、白、黑游鱼，其状多怪异。
 <br/>
 <br/>
 经鉴定，石窟中的钟乳石，成长年龄已经超过1700年。由此推算，这处人工开凿的石窟，年代应在钟乳石生成之前，最晚当不超过晋朝，但它开凿的具体时间仍是谜。
 <br/>
 <br/>
 追溯1700年前的中国历史，即使在两汉盛世，要开凿如此浩大的工程，也几乎要举全国之力。石窟内凿纹（爪痕）遍布，却不知是何工具所致。
 <br/>
 <br/>
 这么庞大的规模，浩大的工程量，不亚于修筑另类长城，历史上却没留下任何文字记载。至于有人猜测石窟的用途，诸如诸侯王陵、屯兵、采石、徽商屯盐等，也只是增加说辞，也有人因花山谜窟位于北纬30度〝神秘线〞附近，而视为外星人的杰作。
 <br/>

 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千年谜窟.html" target="_blank">
   千年谜窟
  </a>
  实为〝
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蛤蟆窟.html" target="_blank">
   蛤蟆窟
  </a>
  〞
 </h4>
 <br/>
 事实上，这是一个既非自然、亦非人工的灵异构造，完全超出了人类认知的范畴，但它却和江泽民有着不小的渊源。
 <br/>
 <br/>
 说到石窟的发现与挖掘，首先开发的就是35号石窟。1999年被发现时，洞口处皆是碎石。不断向内挖，孰料越挖越多。在清理碎石时发现其中夹杂着很多牛骨，令人百思不解。故此洞被命名为〝千年谜窟〞。
 <br/>
 <br/>
 后来发现此窟是个典型的蛤蟆洞，口小肚大，有一段20米长的引洞。走到引洞出口处，豁然开朗，眼前出现一片惊人的大肚空间，总面积有4000平方米。
 <br/>
 <br/>
 开洞之初，洞内有千万只蝙蝠。人入此洞，蝙蝠惊飞，一时间若黑风黑雨搅得天旋地转，故又名〝蝙蝠厅〞。
 <br/>
 <br/>
 厅内有36间所谓的石房，最小者面积仅两平方米。石房墙壁厚薄不一，最薄处仅10厘米厚。这些石房三面封，仅临〝蝙蝠厅〞一侧有一形门洞，门洞仅容一人进出。厅边有深潭数口，水呈绿色，内有各色形状怪异的游鱼。洞内寒气阵阵，阴冷无比。
 <br/>
 <br/>
 2000年9月，第35号石窟对外开放，这样一个来历不明的诡异洞穴，却引起了江泽民的极大兴趣。官方称，2001年5月17日至24日，江到安徽视察，那时正是江镇压法轮功最凶残的时候，实际上，江就是为察看巢穴而来。
 <br/>
 <br/>
 为掩人耳目，江于19日游了黄山，还赋诗一首〈登黄山偶感〉，不仅高调公开发表，还要求收录到上海小学生的活动课教材和扩展读本中。
 <br/>

 <br/>
 <h4>
  江赞叹：真是太绝了！
 </h4>
 <br/>
 随后的两天，江仍在黄山周边活动，但官方没有任何报导。只有当地因〝总书记〞的亲临而夸耀，人们才知道江原来去了花山石窟与祖籍。
 <br/>
 <br/>
 在花山石窟的旅游页上，可见江从黄山下山后，2001年5月20日下午，就去了刚开发出的花山35号〝蛤蟆窟〞。当时江在阴冷的洞内待了很久，之后在洞口留影，并异常兴奋，那种找到巢穴的感觉从江的言辞中透露无疑。
 <br/>
 <br/>
 江赞叹：〝真是太绝了！是个谜，真是个千古之谜！这真是个宝啊！要是宣传到国外去真了不得！〞遂亲自题字将此地命名为〝花山谜窟〞。
 <br/>
 <br/>
 江的极度兴奋，也让当地官员受宠若惊。于是，江的题字置于山坡上，留影处弄个石头注明。就连江的上述语无伦次的口赞，也被刻成石碑立了起来。
 <br/>
 <br/>
 恰巧的是，江的祖籍也在黄山脚下的旌德县江村。江在花山巢穴补充能量之后，第二天进行了半小时的闪电式认祖归宗。
 <br/>
 <br/>
 根据花山谜窟的大事记，中共高层现任，没有其他人游览过被江称为〝神奇〞的〝花山谜窟〞。
 <br/>
 <br/>
 对这个被人百思不解的〝花山谜窟〞，民间江湖有许多谈论。有称是江的蟾蜍灵体在另外空间的大阴所，只不过随着江转世，带进移入了这个空间，藏身于黄山之下。时候一到，巢穴曝光，江在那里的兴奋，有了回家的感觉。

 <br/>
 <h4>
  江泽民到〝蛤蟆洞〞补充黑色能量
 </h4>
 <br/>
 《江泽民其人》一书中介绍，安徽黄山一位高僧透露，江泽民元神本是个蛤蟆，〝花山谜窟〞其实是个蛤蟆洞。迷窟的入口扁扁平平，像个倒扣的盘子，就是蛤蟆嘴；迷窟本身就是蛤蟆的肚子；迷窟顶部圆鼓凸出，上有一圈圈绿色，那便是蛤蟆后背。
 <br/>
 <br/>
 老僧说，〝蛤蟆洞〞中阴气极重，聚集了不少肉眼凡胎看不见之烂鬼，洞中所见壁画，每一幅皆与蛤蟆有关。江泽民镇压法轮功元气大伤。2001年5月，江泽民之皮囊奄奄一息，于是来到黄山这蛤蟆洞中补充黑色能量。
 <br/>
 <br/>
 老僧续说，暴君不顾民之疾苦，这江泽民游黄山乃5月下旬之事，花费了两亿元人民币。黄山地区失业者甚众，食不果腹，却专为他来修筑公路，空运水果花费数十万元。上山动用保安武警不计其数，连他上山坐的滑竿儿皆是用进口的竹子和绳子特别定制的。
 <br/>
 <br/>
 高僧还透露：江泽民元神本是个蛤蟆。由于迫害法轮功，造下无边罪业，其元神在2000年9月间已经彻底被打入无间地狱。现在，在世间上活动的不过是江的人皮，以及操纵它的烂鬼而已。
 <br/>
 <br/>
 针对江泽民当政时在外国元首面前照镜子、梳头、弹琴唱歌，以及神情恍惚，甚至半死不活等怪异表现，老僧说，这是因其人皮背后，被那些邪灵烂鬼操控。
 <br/>
 <br/>
 〝这些烂鬼各有分工，有能说的、能唱的、会弹奏乐器的，平时无事，烂鬼也不理它。因江鬼魂魄俱丧，故素常神思恍惚，似随时撒手人寰。一旦需要会见外宾、或为镇压法轮功造势，烂鬼就会摆布江鬼肉身，令其伶牙俐齿，又唱又跳。〞
 <br/>
