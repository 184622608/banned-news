### 民企退场风声鹤唳 青岛92官进驻民企
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】中美贸易战使得中国经济陷入重重危机，有关北京要推行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 〞和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_计划经济.html" target="_blank">
  计划经济
 </a>
 的说法，引发民营企业家的恐慌。青岛市近日举行全市工会干部到民企挂职的动员大会，再次引舆论质疑。
 <br/>
 <br/>
 根据青岛新闻网报导，10月8日上午，在青岛湛山花园酒店召开的动员大会，向全市工会选派干部进驻非公有制经济组织和社会组织，会后市总工会还组织92名挂职第一主席进行了工会业务培训。
 <br/>
 <br/>
 据报，青岛市总工会《关于选派工会干部到非公有制经济组织和社会组织挂职第一主席的实施意见》规定，挂职的第一主席任期为2年，每月到挂职单位工作时间不少于8天，其他时间在原单位工作。
 <br/>
 <br/>
 省、市两级工会每年为每名第一主席配备2万元人民币的工作经费。
 <br/>
 <br/>
 这一消息迅速引爆舆论，不少人感到愕然，质疑有关〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民企退场.html" target="_blank">
  民企退场
 </a>
 〞的说法并非空穴来风，青岛率先做试点，先派工会第一主席挂职，然后再派第二主席，直到私企老板〝逐渐离场〞就达到当局所要的目的了。
 <br/>
 <br/>
 近来，在中国国内，有关〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 〞的忧虑不断升温。中国财经专家吴小平9月发文称，民营公司已完成协助国有经济发展，〝应逐渐离场〞；中共人社部副部长邱小平也在一次会议上，发表〝共享民企发展利润〞的言论，被视为〝国进民退〞的最新官方信号。
 <br/>
 <br/>
 9月16日，在北京召开的中国经济50人论坛上，多位重量级经济学家发言时，批评当局，如果再次推行〝国进民退〞和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_计划经济.html" target="_blank">
  计划经济
 </a>
 ，等于是走
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_回头路.html" target="_blank">
  回头路
 </a>
 。
 <br/>
 <br/>
 9月27日，习近平在东北考察期间，访问一家民营企业时强调，中共始终〝毫不动摇地鼓励、支持、引导、保护民营经济发展〞。但在同日下午，习又强调，〝怀疑、唱衰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国企.html" target="_blank">
  国企
 </a>
 的思想和言论都是错误的〞，〝国有企业要继续做强做优做大⋯⋯〞。
 <br/>
 <br/>
 对此，有分析认为，北京什么都说到、说全了，但感觉习近平的重心还是国企。
 <br/>
 <br/>
 BBC引述中国学者吴强的话分析说，习近平虽然是对近期有关〝国进民退〞忧虑的回应。但此说法〝逻辑有问题〞，因中共没在体制上加强对民营企业的保护和尊重。外界看见的反而是相反的情况，包括在民营企业和非政府机构的党委建设。
 <br/>
 <br/>
 香港资深时事评论员刘锐绍说，习近平对民企的发展有戒心，他要〝扶持民营企业〞的说法〝有虚有实〞，也就是一方面让小型民企生存，同时尝试〝收编〞一定规模的民企，不听话的就把它打下台。
 <br/>
 <br/>
 时评人士文昭认为，中共体制在威胁着中国民企整体，那就是中共政权的财政状况，如果说中共又动了私有化国企念想，一个现实的动机就是，由于财政走向匮乏，党和当局的钱越来越不够花，对私营企业下手〝杀猪〞的动机也会越来越强。
 <br/>
 <br/>
 中共前总书记胡耀邦的长子胡德平9月27日在陆媒刊文，批评当局在民营经济遇到困难的情况下，挤压民有企业、逼其走上〝公私合营〞之路，后果不堪设想。
 <br/>
 <br/>
 文章还批评，有些官员疑似打着〝共享、民主〞的旗帜，重提〝大锅饭、铁饭碗、刮共产风〞等理念，做法与国有企业无异。
 <br/>
 <br/>
 中国改革开放40年来，民营企业在经济发展中发挥了举足轻重的作用。统计数据显示，中国民营企业创造了50%以上的财政收入，60%以上的税收，70%以上的技术创新，以及80%的就业。目前，不少企业已将企业发展规划从过去的十年，缩短至两年。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
