### 北京重度污染 甘肃县城5分钟被沙尘覆盖
------------------------

<div class="wysiwyg">
 【新唐人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 时间2018年11月27日讯】北京等地遭遇秋冬以来最严重的阴霾污染，大陆西北地区遭遇严重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙尘暴.html" target="_blank">
  沙尘暴
 </a>
 袭击。有县城5分钟即被沙尘暴侵入全城。
 <br/>
 <br/>
 秋冬季来临，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 、河北、天津、江苏等地的阴霾污染日益严重；同时，西北多地出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙尘暴.html" target="_blank">
  沙尘暴
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_甘肃.html" target="_blank">
  甘肃
 </a>
 河西走廊遭遇沙尘暴袭击，甘肃省酒泉等市相继发布沙尘暴黄色预警。
 <br/>
 <br/>
 据北京环保监测中心消息，北京26日达五级重度污染，预计27日东部和南部将达六级严重污染，为2018年秋冬季以来最严重的一次区域污染过程。
 <br/>
 <br/>
 该中心表示，此次重污染过程影响范围广，污染程度较重，为今年秋冬季以来最严重的一次区域污染过程。同时，天津、河南大部、山东西北部、辽宁西部、安徽北部等地区PM2.5浓度普遍上升至重度污染以上级别，河北省廊坊、唐山、保定、邯郸等城市空气质量降至严重污染水平。
 <br/>
 <br/>
 另外，长约近千公里的甘肃河西走廊自西向东，多地出现沙尘暴，26日傍晚，酒泉市、嘉峪关市、张掖市、武威等市已相继发布沙尘暴黄色预警。
 <br/>
 <br/>
 11月25日，甘肃省出现严重的沙尘暴天气，包括张掖、嘉峪关、酒泉、金昌等地均遭沙尘暴突袭。其中张掖市民乐县最为严重，近百米高的沙墙出现在该县的上空，风力达九至十级，仅在5分钟内，沙尘覆盖整个县，城内能见度不足十米，高速公路部分车辆停驶
 <br/>
 <br/>
 大陆官媒引述张掖市民的话说，11月25号上午还是晴天，到下午4点左右，大风裹挟着沙尘形成了一堵近百米高的沙墙向城市逼近，遮天蔽日的沙尘瞬间将城市吞没。
 <br/>
 <br/>
 有视频显示：25日下午4时许，大风卷着黄沙逼近张掖市民乐县，沙尘所到之处一片昏黄，天地不见分际，犹如世界末日，民众奔逃躲避。
 <br/>
 <br/>
 大陆城市空气质量实时发布平台的监测显示：甘肃河西地区的嘉峪关、酒泉、张掖、金昌、武威各市，空气指数在25日已达到〝爆表〞的严重污染。在空气检测中，空气质量指数设置的最高数值为500，超过500即为〝爆表〞。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
