### 不只邓超孙俪 横店会议后6明星被曝补税过亿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】中共经济下滑，影视明星沦为首轮〝剪羊毛〞的对象。在中共召开追缴影视业税款的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_横店会议.html" target="_blank">
  横店会议
 </a>
 后，有多名影视明星被约谈。其中，邓超
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙俪.html" target="_blank">
  孙俪
 </a>
 夫妇被曝补缴2.5亿人民币，而陆媒披露补税过亿的明星至少6人。
 <br/>
 <br/>
 11月底，网络传出一份《横店工作室会议内容》，其中规定影射工作室最新补税公式是70%*（80%*40%-3.5%），其结果是每个工作室需要补缴的税款大约会占总收入的20%左右，而且要从2018年追缴至2016年，而之前的征收方法是3%-7%之间。
 <br/>
 <br/>
 陆媒称，但是谁都不清楚这个一刀切的百分比是怎么来的。12月1日，中国电影导演协会发文《倡议依法合理规范影视业税务秩序》，对此提出质疑，但文章很快被删。
 <br/>
 <br/>
 根据陆媒报导，几个月前传闻补税只针对年流水500万以上的影视工作室，但现在局势已经扩大，覆盖了影视相关的所有工作室。据称账面单笔收入在20万以上的公司都接到了约谈电话。有艺人公司表示，最近几乎每天都接到来自不同地方的追缴电话。
 <br/>
 <br/>
 根据网传《横店工作室会议内容》，已有17名明星被约谈，并限期12月15日前自行补税。
 <br/>
 <br/>
 大陆艺人一般都会在税收优惠地区注册工作室，仅在横店东阳注册的影视文化工作室就有749个，其中包括黄晓明、姚晨、杨幂、李易峰、王凯、迪丽热巴等一线明星。
 <br/>
 <br/>
 账号为〝正义的化身—哒哒〞的微博11月30日爆料，邓超
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙俪.html" target="_blank">
  孙俪
 </a>
 夫妻就在被首批约谈的名单内，两人已补税2.5亿人民币。此外，Angelababy（杨颖）、吴奇隆、黄晓明、周迅、杨幂、赵薇、赵丽颖等也都在约谈名单中。
 <br/>
 <br/>
 该爆料博主还披露，被调查和补税对象为：〝所有演员第一批次 1人，第二批次 150人，其中17人为第一重点约谈对象，35人为第二重点约谈对象，第三批次200人，第四批次200人，约谈对象几乎覆盖一二三线所有以工作室形式纳税的演员，谁都别侥幸。〞
 <br/>
 <br/>
 据另一家陆媒披露，这17名被约谈的艺人覆盖大花、顶级流量、一线演员、导演等等。其中一位男明星被约谈时，在会议室〝当场就痛哭起来〞。
 <br/>
 <br/>
 报导还列举了6位需要补缴巨额税款的明星：某顶级流量年轻男星：2—3亿；某影视综艺多栖发展的综艺常驻男星：1.4亿；某专注影视作品的话题小花：1.6亿；某电视剧人气小花：单一个地区补税6000万以上，总额上亿；某大花：2亿以上；某演艺事业和商业两边抓的小生：5—6亿。
 <br/>
 <br/>
 报导指，首先被约谈的大多是多栖发展的顶级明星，尤其要补巨额税款。
 <br/>
 <br/>
 早前，范冰冰因逃税风波失踪四个多月，10月3日被当局重罚，要求她在期限内缴纳逾8.8亿元的税金、滞纳金和罚金。
 <br/>
 <br/>
 随后，中共税务总局发布了《关于进一步规范影视行业税收秩序有关工作的通知》，要求从2018年10月10日起，大陆各影视制作公司、经纪公司、演艺公司、明星工作室等企业，以及影视行业高收入者，必须在2018年12月底前完成对2016年以来的纳税情况进行〝自查自纠〞。
 <br/>
 <br/>
 根据中共官方文件，补税工作将分为四个阶段推进：第一阶段为自查自纠阶段，于2018年12月30日截止，需补缴费用和滞纳金；第二个阶段约谈补税工作室，需要在12月15日前完成；第三个阶段，税务上门辅导、检查，于2019年1月起至2019年3月截止，需补缴费用、滞纳金和罚款；第四个阶段为重点检查和税务抽查，于2019年3月起至2019年6月止，需补缴费用、滞纳金和罚款，可能会涉及刑事处罚。
 <br/>
 <br/>
 外界认为，美中贸易战开打以来，中国经济日趋恶化，国库日渐空虚，演艺圈成为当局瞄准的一大块可以宰割的肥肉。
 <br/>
 <br/>
 另外港媒分析，中国影视界背后有中共军方势力介入，也有曾庆红胞弟曾庆淮的操控，因此习近平才借演艺界开刀，藉机收回地盘。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
