### 钱塘江突发异象 惊现百米巨型移动漩涡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_钱塘江.html" target="_blank">
  钱塘江
 </a>
 浙江杭州段周三（17日）下午出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_异象.html" target="_blank">
  异象
 </a>
 ，钱江四桥附近江面突然出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巨型.html" target="_blank">
  巨型
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白色漩涡.html" target="_blank">
  白色漩涡
 </a>
 ，最大直径约为100米，持续大约20分钟后消失。事件引起市民及媒体关注。
 <br/>
 <br/>
 据浙江境河道总站
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_钱塘江.html" target="_blank">
  钱塘江
 </a>
 管理局表示，经现场勘查，漩涡最大时直径超过100米，面积超过800平方米，但未有发现传出异味或油污，初步排除是污染问题。
 <br/>
 <br/>
 网上一度传出消息指，漩涡与当地在建的地铁5号线漏水有关。不过，经核实得，5号线施工情况一切正常。
 <br/>
 <br/>
 此外，杭州燃气集团亦回应称，漩涡所在之处江底并无燃气管道。水务集团也表示，从未有在该处铺设自来水管道。
 <br/>
 <br/>
 该
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白色漩涡.html" target="_blank">
  白色漩涡
 </a>
 在持续大约20分钟后消散。水务集团派人到现场视察及检查水质，确认结果正常，与任何供排水设施无直接关联。
