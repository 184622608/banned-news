### 北京现首例非洲猪瘟 疫情蔓延陆20省市
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 在中国8月份爆发以来继续蔓延，中共农业农村部周五公布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 市也发现了首例猪瘟疫情。
 <br/>
 <br/>
 周五，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 报告了首例
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房山区.html" target="_blank">
  房山区
 </a>
 两农场共86头猪死亡。
 <br/>
 <br/>
 根据大陆公布的数据，截至目前，已有至少20个省市自治区发生73起猪瘟疫情，累计扑杀生猪60万头。
 <br/>
 <br/>
 大陆官方表示，已知扩散疫情病毒的主要方式是人员和车辆携带病毒，餐馆剩余物喂猪，及其它方式的跨区传染等。而且有的地方在饲料中也发现了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪瘟病毒.html" target="_blank">
  猪瘟病毒
 </a>
 。
 <br/>
 <br/>
 大陆民众质疑疫情已失控，中共无能应对，再加上给美国黄豆加税，造成猪饲料可能还要涨价，中国很多小型猪养殖场的生存面临极大威胁。
 <br/>
 <br/>
 同时，非洲猪瘟目前还没有疫苗防治，也有不少人担忧病毒变种。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
