### 【禁闻】10月19日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 西安丽枫酒店
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拖欠工资.html" target="_blank">
  拖欠工资
 </a>
 工人网上求助
 <br/>
 <br/>
 陕西省西安钟楼开元商场丽枫酒店拖欠装修工人工资近一年，工人多次讨要无果，10月18号再次前往酒店讨薪，并在网上发帖求助。
 <br/>
 <br/>
 2 .大同数百
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出租车.html" target="_blank">
  出租车
 </a>
 聚高速路口 拟集体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 <br/>
 <br/>
 因不满网约车泛滥，政府不退还经营权使用费，10月17号，山西大同市的数百辆出租车聚集在高速路口，准备集体前往省府太原上访，当局实行交通管制，禁止营运出租车驶入高速，拦住他们的去路。
 <br/>
 <br/>
 3. 常州纺织厂关闭车间
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工人讨薪.html" target="_blank">
  工人讨薪
 </a>
 老板不见
 <br/>
 <br/>
 江苏省常州市马杭富丽特纺织厂最近关闭部分车间，工人前往要求结算工资，发现老板已经不见人影。
 <br/>
 <br/>
 4. 在京黑省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 马志文遭遣返
 <br/>
 <br/>
 10月18号，黑龙江省延寿县访民马志文，在北京乘坐地铁时遭警察拦截扣押，之后被哈尔滨驻京办带走，当晚押回当地，目前被多人看管，限制人身自由。
