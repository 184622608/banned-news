### 槓上范冰冰？崔永元：还有更大合同没亮出来
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】中国女影星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 逃税漏税案，虽然一个月前以范上交近9亿人民币结案，但此案引发的舆论风波并为就此平息。踢爆范冰冰逃税案的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 近日遭受范的粉丝网络攻击后怒而放话称，自己手里还有一份〝数额巨大的合同〞还没亮出来，再次引发舆论的关注。
 <br/>
 <br/>
 北京时间11月4日凌晨，中共央视前主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 突然通过手机在他的小号〝小崔读书汇〞中发出一条帖文称，〝我认为有必要给范的水军泼泼冷水，还有一份数额巨大的合同我还没亮出来。〞他接着还语带威胁地警告那些水军称，〝别苍蝇一样围着乱转，再转合同就转岀来了〞。
 <br/>
 <br/>

  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-6/p9112492a566118163.jpg" target="_blank">

   <br/>
   <font size="-1">
    率先踢爆
    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
     范冰冰
    </a>
    逃税案的崔永元11月4日凌晨发帖向范冰冰的粉丝放话称，自己手里还有一份〝数额巨大的合同〞没亮出来。（网络截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 <br/>
 此帖一发出，迅速吸引了大批网友围观转发，有不少网民猜测：崔永元这是不是在向范冰冰喊话，他手中还有新证据，如果范的〝水军〞继续对他进行舆论攻击惹恼了他，他可以再次掀起风浪让范冰冰彻底翻船？
 <br/>
 <br/>
 据了解，范冰冰上月初因其名下公司偷税漏税被当局有关机构调查，范本人从公众视野消失3个月后，范冰冰以上交8.84亿元人民币的追缴税款、滞纳金和罚款得以免去牢狱之灾。
 <br/>
 <br/>
 此后，范冰冰一直十分低调，除了在微博等社交媒体上发帖向粉丝报平安并就此案表态向公众〝致歉〞外，一直没有公开露面，但有疑似范冰冰粉丝的群体开始陆续在网络社交媒体上，发帖文为范冰冰遭〝池鱼之殃〞报不平，其中有粉丝把矛头指向最初踢爆范冰冰〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞的崔永元，炮轰他拿演艺圈普遍存在的问题向最没势力的女演员发难，陷害范冰冰独自背锅。
 <br/>
 <br/>
 期间，也有不少支持崔永元的网民发帖怒怼范冰冰，其中有网友发帖称〝真有钱，刚被罚款8个亿，现在还有钱买水军洗白自己〞。崔永元近日转发了这条微博并评论说：〝太恶心了〞。
 <br/>
 <br/>
 随后，便有疑似范冰冰的粉丝发帖反呛崔永元〝更恶心〞，并重提有关崔牵涉〝400亿集资诈骗案〞以及〝为红通要犯洗白〞等网络上曾经爆出的丑闻。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-6/p9112493a15319155.jpg" target="_blank">

   <font size="-1">
    有疑似范冰冰的粉丝近日发帖呛声崔永元。（网络截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 此帖显然激怒了崔永元，他随即发帖反击称，〝我不知道这是不是范冰冰的妈还是粉丝。第一，范冰冰还欠着我的道歉，第二，你说的所有的关于我的烂话都是造谣，是方骗子、黄大娘、司马夹头和三鸡揑造并散布多次的。它们将承担法律责任，加上你吗？〞
 <br/>
 <br/>
 之后，崔永元大约感觉自己还未解气，于是又追发前述帖文，抛出了〝还有一份数额巨大的合同〞的震撼弹。
 <br/>
 <br/>
