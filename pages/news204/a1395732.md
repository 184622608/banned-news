### 呼吁释放王全璋和黄琦  香港议员中联办示威
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】香港维权人士10月16日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 门口游行示威，抗议中共当局打压维权人士，要求中共当局释放被关押在狱中的709维权律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 ，与中国《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四天网.html" target="_blank">
  六四天网
 </a>
 》负责人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港立法会.html" target="_blank">
  香港立法会
 </a>
 议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭家麒.html" target="_blank">
  郭家麒
 </a>
 、前立法会议员梁国雄等抗议人士，手持横幅标语，于10月16日下午从西环西区警署游行到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 ，抗议中共当局打压维权人士，他们要求释放被关押在狱中的709维权律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 ，以及中国维权网站《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四天网.html" target="_blank">
  六四天网
 </a>
 》负责人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 。
 <br/>
 <br/>
 今年55岁的黄琦是中国《六四天网》和《天网人权事务中心》创始人，社会公益和异议人士，曾因持续披露公共事件、人权讯息，遭当局报复，两次被捕判刑，共坐牢8年。
 <br/>
 <br/>
 两年前，黄琦再次被四川警方带走后，且被中共以涉嫌〝为境外非法提供国家机密〞批捕，最近又被追加〝泄露国家秘密〞罪名，目前被关押在四川绵阳看守所。
 <br/>
 <br/>
 10月16日，在香港带领游行的公民党议员郭家麒对媒体说，中共当局没有理由拘押黄琦。黄琦只是传播了其他中国异见人士的文章，来体现许多中国人遭受的不公平待遇，不应该受到如此严重的指控。
 <br/>
 <br/>
 黄琦的律师李静林最近与黄琦见面后透露，身患多种疾病的黄琦当时脸部和身上仍然浮肿，血压达150至190之间。
 <br/>
 <br/>
 黄琦85岁高龄的母亲蒲文清，呼吁国际社会和人权组织敦促中共当局允许黄琦保外就医，蒲文清说，自己活不了多久，希望在儿子死去的时候，在自己去世之前，母子能见上最后一面。
 <br/>
 <br/>
 黄琦案原定6月20日在四川绵阳中院开庭审理，但被取消，当局没有解释任何原因。两年多来，85岁高龄的蒲文清四处奔走呼吁，始终没能见上儿子一面。
 <br/>
 <br/>
 香港抗议者要求释放的另一位是，3年前在709大抓捕中被带走的王全璋律师。
 <br/>
 <br/>
 现年41岁的王全璋律师，主要代理、维护言论自由等敏感案件，以及替法轮功学员，无罪辩护等。2015年被中共公安带走。2017年2月，王全璋律师被天津市检察院以涉嫌〝颠覆国家政权罪〞起诉。
 <br/>
 <br/>
 被捕至今，警方一直不许家属和代理律师与王全璋见面。
 <br/>
 <br/>
 10月15日，王全璋的妻子李文足和友人，再次前往最高法院递交《督促函》，要求尽快对案件展开审理。但一行人遭到法院门口警察阻拦，不允许他们入内递交信件。
 <br/>
 <br/>
 参加抗议的公民党议员郭家麒称，中共当局没有理由羁押二人，明知黄琦身患肾脏疾病而延医〝犹如谋杀〞，必须让其得到妥善医治。
 <br/>
 <br/>
 参加抗议的前立法会议员梁国雄表示，由于黄琦和王全璋都不肯认罪，因此官方则〝不认罪就不开审〞。在场的抗议人士要求中共当局，释放黄琦、王全璋等所有仍被关押的维权人士。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
