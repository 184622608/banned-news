### 比贸战更严重  最新报告：更大危机将拖垮中国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】持续升温的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 ，使中共处于内外交困的严峻危局，然而最新一份报告指出，中共面临着比贸战更严重的危机，持续恶化的水资源危机，恐将拖垮整个中国近年来的经济成长。
 <br/>
 <br/>
 11月30日，《美国之音》报导说，过去一年，世界媒体关注的焦点莫过于美中贸易战给两国带来的影响。然而除了贸易战，中国还有一个更为艰巨的问题，那就是水资源危机，可能将这世界第二大经济体的经济增长冲走。
 <br/>
 <br/>
 根据国际环保组织绿色和平（Greenpeace）11月20日发布的一份新报告，近几十年来，中国西部冰川加速融化，现已有1/5消失无踪，此改变将大规模影响农业及城市用水。
 <br/>
 <br/>
 该组织研究员说，这些冰川被称为〝亚洲水塔〞，是南北极地区外最大的饮用水库存，同时也是亚洲数条重要河流的河水来源。
 <br/>
 <br/>
 另一份报告的作者、英国下议院外交事务委员会中国事务特别顾问彭朝思（Charlie Parton）说，中国80%可运用之水资源分布在南方和北京、上海、天津等12个北方重要省份、城市，却面临水资源严重短缺。
 <br/>
 <br/>
 但这12个省份却涵盖中国38%农业、50%的能源工业及41%人口，其中北京的人均水资源保有量，甚至仅略高于沙乌地阿拉伯。
 <br/>
 <br/>
 彭朝思认为，中国缺水不是暂时性的气候问题，也不是由于干旱，而是资源基础已经被严重过度开采而遭到破坏。
 <br/>
 <br/>
 他说，中共耗资1千亿美元打造的〝南水北调〞工程也无法解决问题，即便在2030年实现设计输水量，但这些水全部被送往北京、天津及河北省地区，也仅能达到水资源严重短缺线的2/3。
 <br/>
 <br/>
 威尔逊研究中心的环境专家吴岚则说，由于没有商业活动能在无水状况下运营，水资源缺乏的问题将重创中国地区经济，同时影响跨国企业投资意愿，对中国农工业的伤害更是不在话下。
 <br/>
 <br/>
 对于北京来说，水将是未来要面对的一个重要挑战。
 <br/>
 <br/>
 根据中共环境保护部2011年发布的《全国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地下水污染.html" target="_blank">
  地下水污染
 </a>
 防治规划》，要在2020年之前遏止地下水污染趋势，然而多年过去了，问题却越来越严重。
 <br/>
 <br/>
 10月初，《自由亚洲电台》报导说，在水资源短缺和河流污染日益严重的中国，地下水污染是当局不愿面对的棘手问题。
 <br/>
 <br/>
 中国地下水污染有多严重？中国学者表示，〝要想净化被重金属污染的地下水，需要1000年。〞
 <br/>
 <br/>
 中共环保部2015年发布的《2014中国环境状况公报》调查数据显示，大陆将近2/3的地下水和1/3的地表水，人类不宜直接接触，其中许多被农药、重金属和废水污染。
 <br/>
 <br/>
 次年北京清华大学环境学院发布的一份《水研究》报告说，在采集的中国23个省的44个城市的水质样本中，都验出了高风险致癌物质。
 <br/>
 <br/>
 去年6月绿色和平组织发布的水质报告显示，中国城市的水质危机日益严重，河流污染到人类不能碰触的程度。
 <br/>
 <br/>
 报告评估显示，在2015年上海的地表水85%不能喝，天津附近的地表水95%不能喝。另外8个省份主要河流的水有一半超过4级污染。报告说，这样的水，人类如果接触是不安全的。
 <br/>
 <br/>
 旅德环境学者王维洛说，现在中国地下水的污染，已经污染到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深层.html" target="_blank">
  深层
 </a>
 地下水。深层地下水就是在第一层的隔水，这个深层地下水跟地表水之间，很少有交换，几乎没有交换，进入到深层地下水的污染物，很难被干净的水置换出来。
 <br/>
 <br/>
 王维洛表示，深层地下水是作为战争时期最重要的水资源供应，中共等于已经自毁水源前途了。因为地表水很容易被敌方用毒药给破坏，那就不能使用了。唯一难以受破坏的就是地下水这一块，然而也已经被污染。
 <br/>
 <br/>
 中国原可提供85%的水于农田灌溉，但工业发展及都市化结果，加上工业用水欠缺效率，中国钢铁业每吨钢需要使用23吨水，大量工业和生活废水不断的被排入河流。
 <br/>
 <br/>
 所谓经济改革创造的〝中国经济奇迹〞，在付出了高污染的巨大代价之后，它又制造了多少的〝生态炸弹〞，何时会引爆不得而知。
 <br/>
 <br/>
 王维洛表示，深层地下水污染对中国人来说，就是一个真正的灾难。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
