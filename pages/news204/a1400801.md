### 中共掏空百姓财富手段曝光 农民工也难幸免
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】中共〝改革开放〞40年，房地产已成为经济发展的引擎。然而，一份最新报告称，目前中国居民财富，基本上被房地产掏空，连存款最薄弱阶层——农民工的资金也全部投入房地产市场了。
 <br/>
 <br/>
 11月24日，中国人民大学国家发展与战略研究院、经济学院与中国诚信信用管理股份有限公司联合发布了一份题为《中国宏观经济分析与预测（2018-2019）》报告。
 <br/>
 <br/>
 报告称，中国消费者在房地产去库存中债务率大幅度上升，消费基础受到严重削弱。目前居民财富基本上被房地产掏空，广大中产阶级和中下收入阶层被房地产套牢。
 <br/>
 <br/>
 报告说，3年前被房地产套牢的基本是中上收入阶层，而新一轮的去库存，特别是货币化及鼓励农民工购房，是将储蓄存款相对薄弱阶层的资金，全部投入房地产市场了。
 <br/>
 <br/>
 根据财富管理公司诺亚控股（Noah Holdings）的数据，房地产投资已经达到约占中国家庭资产的2/3的份额。同时，房地产市场在地方政府收入、银行贷款和企业投资中也扮演着重要角色。
 <br/>
 <br/>
 2018中国商业银行半年报数据显示，银行房地产贷款的投放规模同样居高不下，在26家上市商业银行中，有19家银行的房地产行业贷款余额高于2017年同期。
 <br/>
 <br/>
 2018年上半年，三、四、五线城市房地产个人贷款额的成长尤为明显。
 <br/>
 <br/>
 北京智库中国城市全球化研究所（China Institute for Globalization of Cities）执行董事秦泰乐（Thales Qin）表示，如果不增加对房地产市场的控制，那么资金仍将流入房地产市场。
 <br/>
 <br/>
 地方政府、房地产、中国民众，三者早已构成了一个内部资本循环。地方政府通过房市将债务转移给买房者。这个内循环能稳定存在的基础是高房价，因为高房价能保障资本在内循环中的高回报。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国房市.html" target="_blank">
  中国房市
 </a>
 也被称为货币蓄水池，这个内部资本循环吸收了大量货币。如3～6月期间，中共央行为刺激经济、放出了3万亿元的流动货币。房地产贷款同期增加1.7万亿元，也就是说，仅房地产一项就吸收了57%的货币放水。
 <br/>
 <br/>
 房市蓄水池内的资本循环，不但抽干了实体经济的资金，也掏空了中国人的钱包。
 <br/>
 <br/>
 如2006年中国居民负债收入比只有18.5%，2018年已高达77%。2017年中国家庭人均财富中，房产净值占比2/3。这些数据说明，中国人的财富都被锁在房子里，民众债台高筑，没钱消费了。
 <br/>
 <br/>
 中国房市被中共榨取了大半资金的同时，也将中共因腐败、低效、冗员所产生的巨大债务，通过高房价转移给了买房的中国人。
 <br/>
 <br/>
