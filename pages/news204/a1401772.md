### 湖北潜江百余老兵维权 要求解决安置问题
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖北潜江.html" target="_blank">
  湖北潜江
 </a>
 市100多名退伍军人，聚集在市政府大门口维权，要求市政府为他们解决安置问题。
 <br/>
 <br/>
 100多名退伍士官、志愿兵、参战老兵，11月30号，聚集
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖北潜江.html" target="_blank">
  湖北潜江
 </a>
 市政府大门口拉着〝还我参战安置名额〞等标语抗议。
 <br/>
 <br/>
 12月3号，湖北工商银行下岗前职工伍立娟告诉新唐人说，当天她也在市政府门口，这些老兵们从早上8点一直到中午12点，都在市政府门口拉横幅维权。
 <br/>
 <br/>
 中国工商银行下岗前职工伍立娟：〝这次政府以疏导劝说为主，因为这个市委书记是刚调来的，才半个月，对潜江很多事还不熟悉。所以这次他也没出来接待，秘书有出来，市政府的军人事务部的局长出来，还有信访局长都在现场。〞
 <br/>
 <br/>
 伍立娟说，这次政府没有动用武警、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特警.html" target="_blank">
  特警
 </a>
 到现场镇压老兵。军人事务部的局长不停的用扩音喇叭喊，要求老兵们不要堵路，局长还组织各单位维稳人员，把各地管辖区的老兵带回去。
 <br/>
 <br/>
 伍立娟：〝不像山东上次维权（政府）动用警力和武力，打维权
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 的军人，造成流血事件，还有江苏的那次，因为当时现场维权的说，你们敢动用武力，我们告诉全国的人来声援，他们可能也怕造成不好的影响。〞
 <br/>
 <br/>
 虽然这次老兵们没有被殴打，但也始终没有政府主要负责人出面解决他们的问题，当天下午老兵们去火车站，准备买票前往北京上访，最后市政府相关官员到火车站把他们都劝说回家了。
 <br/>
 <br/>
 据了解，这批老兵大多年约50岁，都是士官退伍没有获安置的人员，他们现在几乎没有任何收入，每人每月仅获得约600元人民币的军人困难补贴，而且没有养老医疗，生活拮据。
 <br/>
 <br/>
 新唐人记者熊斌、元真采访报导
</div>
