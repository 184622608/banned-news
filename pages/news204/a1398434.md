### 大陆非洲猪瘟继续蔓延  湖北现首个病例
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】中国大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 继续
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔓延.html" target="_blank">
  蔓延
 </a>
 。中共官网11月7日公布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖北.html" target="_blank">
  湖北
 </a>
 省出现首例非洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪瘟.html" target="_blank">
  猪瘟
 </a>
 疫情。至此，非洲猪瘟已蔓延至中国15个省、直辖市，发病案例达49起。
 <br/>
 <br/>
 中共农村农业部官网周三（7日）通报，经动物卫生与流行病学中心确诊，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖北.html" target="_blank">
  湖北
 </a>
 省黄冈市罗田县一处养殖场出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 。该养殖场存栏生猪821只，发病22只，死亡4只。据统计，这是大陆第49例非洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪瘟.html" target="_blank">
  猪瘟
 </a>
 疫情。
 <br/>
 <br/>
 非洲猪瘟自8月初首次出现在辽宁沈阳市沈北新区后，一路南下，仅仅一个月便扩散至河南郑州、江苏连云港、浙江温州乐清市、安徽芜湖南陵县。
 <br/>
 <br/>
 随后，该疫情继续向北、向西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔓延.html" target="_blank">
  蔓延
 </a>
 。截至目前，包括内蒙古、天津、山西、云南、湖南、贵州、重庆以及湖北在内的、共计15个省（市、区）先后沦陷。
 <br/>
 <br/>
 不过，在中共官方的疫情通报中，几乎每次均声称，〝目前，上述疫情已得到有效处置。〞
 <br/>
 <br/>
 非洲猪瘟是由非洲猪瘟病毒引起的猪和野猪的一种急性、烈性、高度接触性的传染病，可通过受污染的饲料或接触受感染的动物传播，其发病率高，染病猪只死亡率几乎100%，且没有疫苗。
 <br/>
 <br/>
 <h4>
  湖北早前已出现猪瘟疫情
 </h4>
 <br/>
 实际上，早在今年8月底，湖北省就已经发现猪瘟病例。
 <br/>
 <br/>
 8月29日，湖北黄石市阳新县一生猪养殖场爆发猪瘟疫情，此疫情具有高传染性，死亡率高。
 <br/>
 <br/>
 据陆媒〝界面新闻〞报导，有知情人士反映，阳新县政府已发布封锁令，称该县一生猪养殖场发生猪瘟疫情，疫区已被封锁。
 <br/>
 <br/>
 通告中称，2018年8月29日上午，黄颡口镇三洲村周妹生猪养殖场发生不明原因的生猪死亡。经湖北省动物疫病预防控制中心检测，4份样品均为猪瘟病毒核酸阳性，诊断结果为猪瘟疫情。疫病具有高度传染性和致死性。
 <br/>
 <br/>
 报导称，此次发生在阳新县的猪瘟，不是非洲猪疫情，而是一般的猪瘟。
 <br/>
 <br/>
 根据公开资料显示，猪瘟，又称经典猪瘟或古典猪瘟，是由黄病毒科猪瘟病毒属的猪瘟病毒引起猪和野猪的一种高度接触性传染病，具有高度传染性和致死性，猪和野猪是该病毒唯一的自然宿主。猪瘟被世界动物卫生组织列为A类16种法定传染病之一。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：秦锋）
