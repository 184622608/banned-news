### 伊朗向美供出华为违规交易 美半年前展开刑事调查
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创始人任正非之女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 12月1日在加拿大被捕，舆论再度聚焦华为与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 的违规交易。消息指，早在2016年，因为伊朗供出，美国已对华为和中兴展开调查。今年4月，美国已对华为展开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刑事调查.html" target="_blank">
  刑事调查
 </a>
 。孟晚舟被捕或与此有关。
 <br/>
 <br/>
 早在2016年，多家美媒就曝出消息指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 和中兴两家中国电信设备商因为和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 的违规交易，遭到美国商务部调查。当年，中兴即遭到商务部制裁。
 <br/>
 <br/>
 当年3月，《超讯》引述美方人士消息指，调查起因是当时美伊关系正在走向正常化，美国要求伊朗提供在美国制裁期间与伊朗违规交易的中国企业名单。伊朗供出了中共，提供的名单上华为、中兴赫然在列，铁证如山。
 <br/>
 <br/>
 报导说，美国甚至将名单向欧盟通报，对欧盟给予中共市场经济地位的决策产生了影响。
 <br/>
 <br/>
 此外，当年美国商务部公布中兴罪状时，还曾曝光截获的一份中兴内部文件。文件中揭示另一家代号F7的中国公司也在逃避美国的出口管制，和伊朗、朝鲜等遭美国制裁的国家秘密交易。而F7被曝是中兴内部对竞争对手华为的称呼。
 <br/>
 <br/>
 当时，也有说法指，是中兴的愚蠢行为〝供出〞了华为。
 <br/>
 <br/>
 另据《华尔街日报》此前报导，今年4月，在美国商务部宣布对中兴的禁售令时，美国司法部也已对华为是否违反了美国对伊朗的制裁展开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刑事调查.html" target="_blank">
  刑事调查
 </a>
 。
 <br/>
 <br/>
 报导指，这项调查将影响华为在美国已经有限的业务，还有可能对华为在欧洲等海外市场的业务产生连锁反应，最严重的结果是令华为得到与中兴类似的禁售令。
 <br/>
 <br/>
 如今，华为首席财务官、副董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被抓，或显示华为有可能面临比中兴还要严重的制裁。
 <br/>
 <br/>
 也有分析指，中兴已一蹶不振，华为成为中共对外输出网络科技垄断的龙头企业。川普政府抓捕孟晚舟，显示已经掌握华为违规交易的证据。美方在川习会同时控制孟晚舟，可能意在手握华为这个重磅筹码，迫使北京兑现川习会承诺。
 <br/>
 <br/>
 也有人认为，在川习会的贸易谈判现场之外，美国并没有放松在其它领域对中共的围剿。动刀华为，或是美国对中共科技封杀的既定步骤，与川习会的贸易协议并无直接关系。
 <br/>
 <br/>
 华为和中兴都被指有中共官方背景，除了给遭美国制裁的国家提供技术支持之外，还帮中共在海外从事网络间谍活动，窃取外国情报和尖端技术。此外，华为目前是中共对外输出5G技术的龙头，被指为中共试图垄断未来互联网的工具。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
