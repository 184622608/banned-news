### 【禁闻】美媒曝光中共渗透美国 干预大选细节
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】美国副总统彭斯10月初发表对华政策演讲时，曾提到中共企图推翻川普总统并干预美国大选。日前,  《华盛顿时报》进一步披露了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共渗透美国.html" target="_blank">
  中共渗透美国
 </a>
 、干预美国选举的新细节。
 <br/>
 <br/>
 美国《华盛顿时报》10月24号报导说，副总统彭斯在10月4号的演讲中提到，6月份，北京发布了一份题为《宣传和审查通知》的敏感文件，其中列出了中共的宣传战略。
 <br/>
 <br/>
 彭斯说,  〝北京动员了潜藏的人物、前线团体和宣传渠道，来改变美国人对中共政策的看法〞。另外，一位美国高级情报官员告诉他，俄罗斯对美国选举的干涉程度跟中共比起来，只不过是〝小巫见大巫〞。
 <br/>
 <br/>
 报导说,  彭斯提到的这份秘密文件是6月28号发出的。
 <br/>
 <br/>
 文件显示，负责贸易谈判的中共副总理刘鹤表示，〝我们跟美国没什么可谈的了，现在必须不能退后一寸，要制定对等报复措施。〞
 <br/>
 <br/>
 〝我们必须谨慎控制宣传调门儿，不要升级，不要扩大范围。而是必须进行精确打击，必须在美国的不同团体之间制造分裂，最终使他们崩溃。〞
 <br/>
 <br/>
 时事评论员夏小强：〝中共利用它们在美国的间谍和一切的资源，干扰美国的中期选举，收买媒体、渗透媒体、渗透美国的政界、商界，就是要在美国制造群体和团体之间的矛盾，甚至制造分裂，来达到干涉美国中期选举和政治的目的。〞
 <br/>
 <br/>
 报导说,  文件未写明其来自中共的什么部门，但文件中对中共的报纸、电视、广播和社交媒体网络如何报导美中贸易争端进行了严格界定。
 <br/>
 <br/>
 不仅要求报纸大力报导〝经济亮点〞，显示出中国经济的〝稳步改善〞,  而且还规定网络平台如微博和微信，都要配合中共对贸易战的宣传。
 <br/>
 <br/>
 文件最后警告说，不要提〝2025计划〞，否则会 〝产生严重后果〞。
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝这个文件可以说是客观上让中共自己暴露了它的真实的面目，等于中共自爆其丑，它其实是以对美国为主的这些西方发达国家进行这种破坏式的渗透和扩张为代价的。〞
 <br/>
 <br/>
 除了这份秘密文件,  《华盛顿时报》还披露,  中共在美国檀香山设立了两个广播电台，向当地播放中共的宣传。驻扎在夏威夷的美军太平洋司令部认为，这两个中文电台是中共的喉舌，它们支持北京的政策、间谍活动、盗窃身份和知识产权等。
 <br/>
 <br/>
 其中一个是KHCM，播放由中共国际广播电台（CRI）直接提供的节目。
 <br/>
 <br/>
 今年8月〝华盛顿自由灯塔〞也曾披露，亲共的香港凤凰卫视，试图购买南加州蒂华纳（Tijuana）附近的墨西哥西班牙语XEWW AM 690广播电台，试图在南加州进行中文广播。
 <br/>
 <br/>
 唐靖远：〝从另外一个层次面等于中共自动的暴露了它们在美国的这种渗透的程度有多深，实质上中共在美国的本土这些年它利用经济实力，同时也钻了美国民主制度的空子，在美国来收买了很多媒体，来潜在的把中共的意识形态用各种包装，把共产主义这些邪恶的这些东西，在美国进行扩散。〞
 <br/>
 <br/>
 报导引述德克萨斯州共和党参议员克鲁兹的话说, 〝中共正在发动信息战，破坏美国民主〞。
 <br/>
 <br/>
 而联邦参议员卢比奥（Marco Rubio）则督促美国联邦通信委员会(FCC)对南加州的这个出售案进行调查。
 <br/>
 <br/>
 卢比奥还表示，他将很快引进法案，要求所有由中共政权和中国共产党所拥有和受其指示或以其它方式控制的媒体机构,  注册为外国代理人。
 <br/>
 <br/>
 采访编辑/常春  后制/周天
