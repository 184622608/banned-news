### 中共再度升级思想监控  大陆书商遭审查
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对大陆出版物的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_审查.html" target="_blank">
  审查
 </a>
 再度升级。有网友在社交媒体爆料，中共审查人员日前直接前往书店，以〝审查〞为名将书籍全部搬走。大陆各地也出现图书馆、书店贴出〝关门整顿〞的告示。但相关微博帖文已被当局迅速删除。
 <br/>
 <br/>
 有网友在台湾电子布告栏〝PTT〞转贴一则消息，称一名大陆网友又遭到当地相关部门的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_审查.html" target="_blank">
  审查
 </a>
 〞。相关人员几乎把这间书店的全部书籍搬空，并直接留下一句话：〝这些（书）已经都是垃圾了。〞
 <br/>
 <br/>
 该大陆网友称，当局搬走了至少1600本书，其中不少是日本漫画。他遗憾地表示，最后能通过〝审查〞回来的书，应该不多了……
 <br/>
 <br/>
 该消息被爆出后，一度在微博引发热议。有网友称：〝已经有书店在紧急卖书了，说是‘狗奴’直接上门搬书审查。〞
 <br/>
 <br/>
 而日前，大陆各地亦有图书馆、书店，传出暂时关闭、进行整顿的消息。有网友质疑，当局对国人思想控制再度升级。
 <br/>
 <br/>
 不过，目前，有关当局加强审查书店、书籍等帖子，已被全部删除。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
   中共
  </a>
  缺乏自信  不断升级
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_思想监控.html" target="_blank">
   思想监控
  </a>
 </h4>
 <br/>
 中共对出版物的严格审查由来已久，不但各类敏感话题的读物、期刊、小说不允许发行，甚至从境外进口的原装工具书，都要被撕页或〝阉割〞。这种审查甚至已经蔓延至大陆的独立书店。
 <br/>
 <br/>
 所谓〝独立书店〞通常以特定的社群作为依托，在选择书籍和营造氛围方面，通常有自己的风格和〝精神空间〞。由于中共对独立书店的审查通常是〝运动式〞的，因此，经营者有时会等〝风头〞过后，再悄悄售卖一些被当局划为所谓〝敏感〞的书籍，包括一些曾经在中国公开出版、但后来被禁的图书，甚至还有境外出版物。
 <br/>
 <br/>
 不过，近年来，中共管控言论愈发严格。今年6月，一家位于北京海淀区北四环附近的独立书店，被当局要求下架了一批图书，涉及纪实文学、历史学术著作、宗教题材等；被迫下架的书籍在各大购书网站显示为〝缺货〞。
 <br/>
 <br/>
 而包括《圣经》和《古兰经》在内的宗教类图书，已不再被允许上架或在网上出售。
 <br/>
 <br/>
 此外，书店还收到中共指令，要求某些〝敏感题材〞的书籍，如反极权、反乌托邦的读物，不可以摆放在显眼的位置。
 <br/>
 <br/>
 为抵制西方思想入侵，中共新闻出版总署拒绝或延迟给一些〝进口〞的儿童书籍发书号，并向海外的出版商施压，利用经济利益威胁一些学术期刊删稿。
 <br/>
 <br/>
 早在2016年，中共还曾将韦氏英文词典内，有关台湾定义的页面撕掉，因为它在定义中说台湾是一个国家。
 <br/>
 <br/>
 由于中共近年来逐渐加强控制新闻、出版和言论自由，中国的出版业正日益萎缩。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
