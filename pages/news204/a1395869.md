### 〝千人公民起诉团〞因工龄归零上诉 遭驳回
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】大陆一批异议人士由于判刑导致〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工龄归零.html" target="_blank">
  工龄归零
 </a>
 〞，到了退休年龄无法享受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_养老金.html" target="_blank">
  养老金
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_医保.html" target="_blank">
  医保
 </a>
 待遇，他们近年来组成〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人公民起诉团.html" target="_blank">
  千人公民起诉团
 </a>
 〞依法维权。据维权网10月17号报导，他们的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上诉.html" target="_blank">
  上诉
 </a>
 于9月底遭到法院驳回。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人公民起诉团.html" target="_blank">
  千人公民起诉团
 </a>
 〞代表何德普、牟传珩、姜福祯等5名老政治犯，为了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工龄归零.html" target="_blank">
  工龄归零
 </a>
 〞问题，今年5月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上诉.html" target="_blank">
  上诉
 </a>
 到北京市高级法院，但9月底遭到法院驳回。
 <br/>
 <br/>
 〝千人公民起诉团〞代表姜福祯：〝我们的工龄都被剥夺了，因为坐牢的原因，我们的退休金就非常低。〞
 <br/>
 <br/>
 〝千人公民起诉团〞代表牟传珩：〝从你受处分之日起，你以前的工龄全部给你抹掉，导致了许许多多在大陆的异见人士晚年都没法退休了，没有养老
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_医保.html" target="_blank">
  医保
 </a>
 待遇。〞
 <br/>
 <br/>
 姜福祯10月18号向《新唐人》表示，中共人力资源和社会保障部〝工龄归零〞政策的依据是非法的。
 <br/>
 <br/>
 姜福祯：〝他们依据的是59年内务部一个文件，还不是文件，就一封回信吧，现在就依据这封信，所有判刑的人的工龄都被剥夺了。打官司也是输了。〞
 <br/>
 <br/>
 牟传珩：〝在这种情况下，我们联名，包括海内外已经1000多人向人力资源和社会保障部发出了一个要求信息公开的信息函，要求他提供执行59年这个旧的信件，法律依据是什么。〞
 <br/>
 <br/>
 〝千人公民起诉团〞曾两度要求人社部公开〝工龄归零〞的法律依据，都被拒绝。今年2月，几名代表到北京市二中院起诉人社部，但法院不予立案，为此，他们上诉到北京市高级法院。
 <br/>
 <br/>
 牟传珩：〝上诉了以后，高级法院又延了4个月，最近才给我们下了一个裁定，就是驳回上诉，就是说起诉权被他们剥夺了，这是完全非法的。因为国家信息公开条例上明确规定，对信息公开答覆不服的，可以提起诉讼。〞
 <br/>
 <br/>
 牟传珩指出，当局有法不依，是因为有巨大利益在其中。
 <br/>
 <br/>
 牟传珩：〝这个文件如果要被推翻的话，全国各地被他们剥夺了养老权利的人会要求他们返还，现在社保资金各地是有缺口的，所以他们宁可违法，国家它有一块巨大的部门利益在里边。〞
 <br/>
 <br/>
 1959年是中共的大跃进运动时期。姜福祯表示，〝工龄归零〞政策是针对所谓的反革命份子实施的，但是92年劳动制度改革之后，包括辞职的、辞退的、找不到档案的人，也都成为这个政策实施的对象。
 <br/>
 <br/>
 新唐人记者常春、陈洁、元真采访报导
