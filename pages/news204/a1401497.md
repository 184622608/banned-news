### 【禁闻】电动汽车全监控 逾200车商向中共交数据
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】目前，有超过200家在中国的汽车制造商，将用户的汽车定位等相关讯息向中共当局发送，中共也对中国境内的所有电动汽车制造商提出相同要求。这让中共对民众的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监控.html" target="_blank">
  监控
 </a>
 更进一步。
 <br/>
 <br/>
 《美联社》11月29号发布调查报导，披露超过200家在中国的汽车制造商，将用户的汽车定位等系列数据，传送给中共当局支持的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监控.html" target="_blank">
  监控
 </a>
 中心，却没有告知任何车主。
 <br/>
 <br/>
 这些汽车制造商包括特斯拉、大众、宝马、戴姆勒、福特、通用汽车、日产、三菱和美国上市的电动汽车初创企业NIO。
 <br/>
 <br/>
 中共官员宣称，这些数据用于分析，来改善所谓的公共安全、或促进工业发展和基础设施规划，并防止补贴计划中的所谓欺诈行为。
 <br/>
 <br/>
 不过批评人士说，在中国收集的信息超出了上述需求，不仅可以用来降低外国汽车制造商的竞争地位，还可以用于监控，特别是在对个人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_隐私保护.html" target="_blank">
  隐私保护
 </a>
 非常不足的中国。
 <br/>
 <br/>
 报导说，在电动汽车的其它主要市场，包括美国、日本和欧洲，都不会收集这种即时数据。
 <br/>
 <br/>
 澳大利亚华裔张小刚博士：〝这种汽车必须要跟一些系统进行联系，能够进行交通的控制或者最优化，中共就利用这一点强制要求汽车公司，把这些数据给它们进行监控，将来可能这些监控还要更全面的。〞
 <br/>
 <br/>
 汽车制造商表示，他们只是遵守中共政府有关〝替代能源〞汽车的法令。
 <br/>
 <br/>
 不过，全美中国学生学者自治联合会现任理事陈闯创表示，不能以此作为藉口。
 <br/>
 <br/>
 全美中国学生学者自治联合会现任理事陈闯创：〝在美国，政府想获取公民的个人隐私，尤其是数据是非常困难的，是必须通过正式的法庭发搜查令等复杂的手续才能做到的。这就说明，西方社会这些高科技公司，它们是心知肚明，对于它用户的个人隐私该进行什么样的保护。〞
 <br/>
 <br/>
 陈闯创说，西方公司不能打着服从中共命令的名义，来做助纣为虐之事，这是不可接受的。尤其是在新疆，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权危机.html" target="_blank">
  人权危机
 </a>
 已经严重到近乎种族灭绝的地步。
 <br/>
 <br/>
 新疆当局去年下令居民安装全球定位系统（GPS）设备，以便跟踪他们的车辆。
 <br/>
 <br/>
 而《华尔街日报》6月报导，中共公安部7月将推出一个系统，通过在挡风玻璃上安装无线射频识别芯片来跟踪车辆。明年1月起将强制要求所有新车安装。
 <br/>
 <br/>
 《美联社》记者走访了上海一家新能源汽车公共数据采集与检测研究中心，发现这家中心的电子屏幕，显示超过22万2千辆的上海车辆数据。这家号称是非营利机构的中心，由中共资助成立。
 <br/>
 <br/>
 报导也引述中共2016年发布的规定指出，电动车制造商会将从车上反应器收集到的车辆信息，传给各地的数据采集和监控中心。
 <br/>
 <br/>
 陈闯创：〝在新的信息化时代，数据就是新的石油，是最有价值的一个资源。中国法律对个人隐私的保障非常差，以国家安全的名义要求产生信息的公司向国家安全机关提供数据，这都是规定中的。〞
 <br/>
 <br/>
 汽车制造商们最初拒绝与上海监控中心共享数据，它们认为这些数据是宝贵的资源。不过一位政府顾问表示：〝只要提供奖励，他们就会给我们数据，因为这是利润的一部分。〞
 <br/>
 <br/>
 澳大利亚华裔张小刚博士表示，这些制造商为了一时的利益跟中共做交易，其实是对自己未来的威胁。
 <br/>
 <br/>
 张小刚：〝如果真的让中共实现这个东西，不单是对中国人是个噩梦，对西方人也是噩梦。因为凡是控制一切的政权，它们最后就会把所有的人作为机器，实现它们所谓目标的机器。我觉得应该大家向西方政府去游说、推动立法，禁止任何公司向极权政权提供这样的数据，不管你是为了什么样的市场。〞
 <br/>
 <br/>
 前美国国土安全部部长切尔托夫（Michael Chertoff）表示，中共收集的这些信息，正是它无处不在的监控的一部分。全球汽车制造商应该扪心自问，从企业价值观的角度看，这真的是我们想要做的吗？
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/钟元
