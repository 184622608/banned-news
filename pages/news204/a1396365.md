### 港媒曝习近平南巡详细路线 〝二次改革〞面临困境
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】北京最高领导人习近平日前开始了他的第二次〝南巡〞，港媒披露其详细行程。中共外宣形容这次〝南巡〞释放〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二次改革.html" target="_blank">
  二次改革
 </a>
 开放〞的信号。但有分析指，北京的所谓〝改革〞已进入死胡同。
 <br/>
 <br/>
 据党媒报导，北京时间10月22日上午，习近平在北京主持了中国工会第十七次全国代表大会开幕式。当天下午，习近平则在广东省珠海市考察横琴新区高新技术片区的粤澳合作中医药科技产业园。
 <br/>
 <br/>
 香港《南华早报》22日报导，习近平这次〝南巡〞从22日考察珠海开始，还将包括出席23日的港珠澳大桥开幕式。
 <br/>
 <br/>
 报导还引述一位消息人士称，习近平还将前往深圳前海深港现代服务业合作区。该合作区毗邻香港，包含前海湾保税港区，位于整个珠三角湾区的核心位置。
 <br/>
 <br/>
 另一位知情人士称，预计习近平还将访问广东农村循环水系统，以评估扶贫工作。
 <br/>
 <br/>
 今年是中共所谓〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 40周年〞，习近平这次南下被称为他的第二次〝南巡〞。6年前的2012年12月，刚出任中共中央总书记的习近平也曾南下广东等地，并在行程中多次强调〝力推改革坚持开放〞。
 <br/>
 <br/>
 中共外宣指，习近平〝南巡〞意在明确〝继续改革开放〞的政策。目前中共内部〝改革〞面临困境，外部又有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 压力，习近平再次南下，或许意味着中共将开启〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二次改革.html" target="_blank">
  二次改革
 </a>
 〞。
 <br/>
 <br/>
 但时政评论员唐靖远表示，中共所谓的〝邓小平路线〞已经把跛脚鸭式改革走到尽头，习近平已经没有余地。而且，邓小平的〝改革开放〞前提是和美国友好，政策是对美国开放。现在美国对中共关闭了大门，习近平已经没有了这样的条件。
 <br/>
 <br/>
 因此，唐靖远认为，习近平现在面临的是被逼触动政治体制的问题，再想要维持现状、继续走〝邓小平路线〞，已经不可能。
 <br/>
 <br/>
 路透社日前引述消息人士指，即将召开的中共十九届四中全会主要聚焦经济政策，大方向仍然是〝经济上向市场放权，政治上更强调集权〞，因为〝深化改革〞已触及体制内利益集团，难度愈来愈大，只有〝经济上放权，政治上集权〞才有可能推进〝后续的改革〞。
 <br/>
 <br/>
 党媒近年多次喊话，承认北京所谓的〝改革〞已经陷入困境。
 <br/>
 <br/>
 海外经济学者分析，中共畸形经济积累数十年的问题已经积重难返，再加上党内各大利益集团的极力抵制，北京所谓的〝经济改革〞其实已无出路。北京近年来推行〝政左经右〞，明显互相矛盾，没有多少可操作性，最终很难成功。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
