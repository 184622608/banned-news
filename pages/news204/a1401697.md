### 川习会北京转调？川普称第二次川金会即将上演
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】在阿根廷的中美首脑会晤上，中方以多项让步换取贸易战暂停升级。在返程飞机上，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 宣布将举行第二次美朝首脑峰会。朝核问题本就是中方牵制美方的一个棋子，美方重启和朝方的接触，被认为与中方态度有关。
 <br/>
 <br/>
 当地时间周六（12月1日）晚间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 结束
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 后，在返回华盛顿的空军一号总统专机上对随行记者表示，他与朝鲜党魁金正恩的第二次峰会可能会在明年1月或2月举行，双方正在考虑三个可能的会谈地点。
 <br/>
 <br/>
 他还说，在某个时间点，他可能还会在美国接待金正恩来访。
 <br/>
 <br/>
 同一天，美国国务卿蓬佩奥也表示，第二次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川金会.html" target="_blank">
  川金会
 </a>
 预计会在新年过后不久举行。他说，在这次峰会上，美国将在与朝鲜去核化谈判中寻求〝实实在在的新进展〞。
 <br/>
 <br/>
 就在川普确认第二次川金会前几个小时，美中首脑在晚餐会晤上达成暂时协议，中方以立即购买美国农产品、承诺〝放行〞高通收购案等让步，换取了对中共贸易政策进行结构性改革的90天谈判期。届时若不能达成协议，美中关税战将继续升级。
 <br/>
 <br/>
 值得注意的是，根据白宫事后发表的声明，在此次会晤中，双方还承诺将共同推进朝鲜无核化。
 <br/>
 <br/>
 声明说，〝美中领导人一致认为，朝鲜问题取得了巨大进展；川普总统将和习主席一起，并与金正恩委员长一道，努力实现一个无核的朝鲜半岛。〞
 <br/>
 <br/>
 另外，美方计划举行第二次美朝首脑峰会的时间，恰好在美中贸易战暂停升级的90天谈判期内。
 <br/>
 <br/>
 朝鲜历来是中方用来牵制美国的工具。自从今年3月首次习金会后，美朝无核化谈判的起落一直紧跟美中贸易战的节奏。近期以来，随着美中贸易战紧张升级，金正恩态度再度强硬，原定11月8日的美国高级会晤取消，美韩重启了联合军演。
 <br/>
 <br/>
 此前，川普多次暗示北京在美朝谈判背后施加负面影响，并曾在推特上一度表示将蓬佩奥访朝推迟到美中贸易战解决之后。这说明白宫已经明了，北京才是解决朝核问题的关键。
 <br/>
 <br/>
 多方消息称，在川习会前，中方加大了打击中朝边境走私的力度，此前紧张准备的中朝经济合作又有降温的迹象。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
