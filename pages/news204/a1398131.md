### 广交会闭幕 对美出口成交下降30.3%
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】一年两次、中国最大的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商品进出口交易会.html" target="_blank">
  商品进出口交易会
 </a>
 ——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广交会.html" target="_blank">
  广交会
 </a>
 ，周日在广州闭幕，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对美出口.html" target="_blank">
  对美出口
 </a>
 成交按年下降30.3%，参展商对中国明年的出口前景并不看好。
 <br/>
 <br/>
 这届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广交会.html" target="_blank">
  广交会
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对美出口.html" target="_blank">
  对美出口
 </a>
 成交27.9亿美元，按年下降30.3%。深圳市雷欧电源实业有限公司总经理表示，美中贸易战对中国影响很大。
 <br/>
 <br/>
 深圳市雷欧电源实业有限公司总经理 ABELL LU：〝很担心这种贸易战，其实来讲美国是一块，单是一个美国对我们影响不大，但是美国可以影响一批国家，那么这对我们影响很大。〞
 <br/>
 <br/>
 卢经理还说，如果各国都预期中国经济下滑，以后生意会更难做。
 <br/>
 <br/>
 深圳市雷欧电源实业有限公司总经理 ：〝特别是影响一批国家的这种预期，那么他们预期中国的经济会往下走，因为美国跟中国在对抗，如果大家的预期都改变的时候，那我们的生意会相当难做。〞
 <br/>
 <br/>
 深圳倍特力电池有限公司销售经理表示，从明年开始，影响会很大。
 <br/>
 <br/>
 深圳倍特力电池有限公司销售经理 BILL HO：〝所以说总体我们电池的影响会在明年开始，还是会受很大的影响，对这个影响，我们的策略的话，就是多开拓一些国家的市场。〞
 <br/>
 <br/>
 美中贸易战，北京一直未能就知识产权和不公平贸易行为等问题，提出解决办法，美国对中国商品加征关税。
 <br/>
 <br/>
 中国制造商在原物料和劳动力成本上升的同时，研拟对策，不排除转移生产线到其他国家。
 <br/>
 <br/>
 深圳市雷欧电源实业有限公司总经理 ：〝对我们来说，第一个，我们有没有可能降低我们的成本。第二个，我们有没有可能提高我们的技术，增加我们的竞争力。第三个，我们有没有可能转移我们的生产基地。〞
 <br/>
 <br/>
 上周，美国总统川普与习近平本人通话，商谈美中贸易问题。
 <br/>
 <br/>
 川普表示，美中双方进行了很好的对话，取得很大进展，中方非常希望达成协议。
 <br/>
 <br/>
 白宫经济顾问库德洛表示，如果北京不能就美国认为非常重要的一系列问题提出解决办法，美国的贸易战攻势只会〝更加猛烈〞。
 <br/>
 <br/>
 新唐人记者李梅、天羽综合报导
