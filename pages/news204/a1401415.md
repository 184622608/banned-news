### 〝川习会〞有三种结果?  美国四大条件曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 迎来关键时刻，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 1天后即将在阿根廷举行。日前，中共驻美大使重申，北京不想打贸易战，中美应避免全面冲突。而美方则表示，北京若想停战需满足四个条件，但白宫对北京的回应感到失望。对此，法媒分析称，此次川习会可能有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三种结果.html" target="_blank">
  三种结果
 </a>
 ，最后一种会让习近平非常尴尬。
 <br/>
 <br/>
 <h4>
  美中双方仍未有实质突破
 </h4>
 <br/>
 11月30日，二十国集团（G20）峰会在阿根廷首都布宜偌斯艾利斯召开。出席峰会的中共驻美大使崔天凯，在出行前接受路透社专访时表示，北京不想打贸易战，中美应尽一切努力避免目前的紧张局势恶化为全面冲突。
 <br/>
 <br/>
 崔天凯一方面表示，希望通过谈判解决目前的僵局；同时，他又声称，〝我们不能接受一方提出一些要求，另一方就必须满足所有这些要求。〞
 <br/>
 <br/>
 外界认为，崔天凯的言论透露出，北京口头上想要停战，但实际上并不想满足美国提出的关键要求。
 <br/>
 <br/>
 白宫首席经济顾问库德洛证实了这一点。当地时间11月27日，库德洛在G20峰会新闻发布会上表示，白宫对北京迄今在贸易问题上所作出的回应感到失望。北京日前对于美国的要求，给出一份书面清单，但这份〝老调重弹〞的清单，美方拒绝接受。
 <br/>
 <br/>
 库德洛说，〝我们发现他们的做法没什么变化，时间表是怎样的？执行机制又是怎样的？〞这些都没有。
 <br/>
 <br/>
 库德洛称，本次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 非常重要，川普（特朗普）告诉他，美中有达成协议的好机会，他对此持开放态度。但北京必须满足四大条件，包括解决窃取知识产权、强制技术转让、放宽美企在华持股比例、大宗商品的高关税和非关税壁垒等。
 <br/>
 <br/>
 <h4>
  习近平可能会在会谈中遭遇尴尬结局
 </h4>
 <br/>
 监于美中双方的表态，法广29日刊文分析称，本次G20川习会可能有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三种结果.html" target="_blank">
  三种结果
 </a>
 。
 <br/>
 <br/>
 一、美中暂时休战。据《华尔街日报》刚刚披露的消息，美中正在研究达成意向协议的可能性。根据这一协议，美方将暂停对中国课税的2000亿美元产品从元月起加征至25%，期限到春天结束。在休战阶段，美中进行新的谈判。
 <br/>
 <br/>
 不过，该报引述中美双方私下谈判的代表说，很难预测川普与习近平能否在G20达成意向协议。
 <br/>
 <br/>
 二、中方作出〝大让步〞。据香港《南华早报》引述知情人士消息，习近平将向川普承诺进一步开放市场、降低对中国国企的补助、保护知识产权等。北京不会放弃国企，但会撤销一些优惠国企的法规，让国企、民企和外企处于同一竞争水平。
 <br/>
 <br/>
 不过，上述承诺中方已说过多次，美国现在要的是具体时间表，但这些北京目前仍未能给出。
 <br/>
 <br/>
 三、双方不欢而散。库德洛表示，现在还看不出〝任何迹象表明习近平领导的政府，准备提出美国可以接受的提议。〞据称，即便在白宫内部，对这次会谈出现何种情形，都难以评定。
 <br/>
 <br/>
 《金融时报》29日引述专家分析，北京几个月来一直没有向美国提供任何东西，只是发出与6个月前基本相同的老一套提议。而川普政府已不想继续听空洞的承诺。
 <br/>
 <br/>
 川普日前已发出预告，美国将按原计划从2019年1月1日起，将对2000亿美元中国产品加征的关税从10%上调到25%。而且，如果他和习近平谈不出结果，美国将对其余的2670亿美元中国输美商品加征关税。
 <br/>
 <br/>
 报导据此分析，本次会谈的结果不至于让习近平太尴尬就已经不错了。
 <br/>
 <br/>
 11月29日，川普登飞机前往阿根廷时继续向北京施加压力，他对媒体表示，他对与北京达成商业协议持开放态度，但他不能肯定一定会达成。
 <br/>
 <br/>
 当天凌晨，川普还在推特上发文称，因为对中国（商品）征税，几十亿美元流入了美国政府，与北京的关税战还有〝一段漫长的路要走〞。暗示贸易战没有短时间结束的可能。
 <br/>
 <br/>
 值得一提的是，白宫贸易顾问，著名鹰派人物纳瓦罗此次将陪同川普与习近平谈判。纳瓦罗日前曾表示，北京不可信任，即使美国与北京达成所谓协议，北京也不一定会遵守。他指出，北京加入世贸组织WTO时的承诺，16年过去了，都没有兑现。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
