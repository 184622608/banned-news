### 网络作家吕千荣被关精神病院65天
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】安徽
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络作家.html" target="_blank">
  网络作家
 </a>
 吕千荣，因长期发表批评政府的文章，近日被江苏常州警方强制送进
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_精神病院.html" target="_blank">
  精神病院
 </a>
 ，关押65天。期间，吕千荣除了被强迫服用抗精神病药物，被删除多个社交媒体帐号，还被要求写下保证书。
 <br/>
 <br/>
 10月24号，有知情人向《新唐人》透露，8月1号，常州市警方以吕千荣〝发表对国家不利的言论〞，涉嫌〝寻釁滋事〞为名传唤他，并抄走了他的笔记本电脑和摄录器材。
 <br/>
 <br/>
 当天晚上7点左右，吕千荣被警方以精神病的名义，押到常州德安医院精神科病房，关押了65天。
 <br/>
 <br/>
 期间，吕千荣被强制每天服用抗精神病药物，派出所警察和常州市网监支队警察还多次要他删除推特、脸书、谷歌博客、博讯博客帐号，并要他写下保证书，保证不提迫害他的事，以后也不能上网发表敏感言论，还要他离开常州。
 <br/>
 <br/>
 10月16号，吕千荣获释，不过他之前被扣押的笔记本电脑和摄录器材，无法取回。
 <br/>
 <br/>
 新公民运动网编辑林云飞表示，这说明当局收紧对舆论的控制，将来这种事情还会更多。
 <br/>
 <br/>
 新公民运动网编辑林云飞：〝吕千荣网络
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_异议份子.html" target="_blank">
  异议份子
 </a>
 ，经常看到他写的一些文章，这次被送到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_精神病院.html" target="_blank">
  精神病院
 </a>
 ，中国当局就是采取舆论收缩的一个表现方式吧，对这种被关精神病院或者寻釁滋事被拘留，我觉得都是在预见之中的事情，我相信接下来这几年应该会持续的增加。〞
 <br/>
 <br/>
 四川成都访民斯毅此前也因为上访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 ，2016年7月被当地维稳部门强行送进精神病医院关押长达200多天。获释后，斯毅现在每天仍需服用多种药物来控制神经性后遗症。
 <br/>
 <br/>
 新唐人记者陈汉、陈洁、施宜君采访报导
