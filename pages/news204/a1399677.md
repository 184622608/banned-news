### 非洲猪瘟爆〝防疫漏洞〞吉林野猪染病恐侵入朝鲜
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情爆出一大难解的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_防疫漏洞.html" target="_blank">
  防疫漏洞
 </a>
 〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉林.html" target="_blank">
  吉林
 </a>
 省日前发现首例
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_野猪感染疫情.html" target="_blank">
  野猪感染疫情
 </a>
 病死案例。专家指，如果官方大规模捕杀受感染野猪，恐导致野猪逃往邻国，造成非洲猪瘟疫情侵入邻近吉林的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 。
 <br/>
 <br/>
 11月16日晚，中共农业农村部新闻办公室发布最新疫情通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉林.html" target="_blank">
  吉林
 </a>
 省白山市浑江区发生〝野猪〞感染
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 病死案例。这是中国境内首次确认〝野猪〞感染疫情死亡，之前感染非洲猪瘟疫情的都是养殖场的生猪。
 <br/>
 <br/>
 这起〝野猪〞感染疫情发生地位于长白山与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 交界处。专家指出，根据以往经验，〝野猪〞感染疫情将导致非洲猪瘟防疫举措，出现一大难解的漏洞。
 <br/>
 <br/>
 今年年初，波兰曾爆发非洲猪瘟，野猪和养殖生猪都感染疫情，官方曾大规模捕杀受感染野猪，却导致野猪逃往邻国，反造成疫情扩大。
 <br/>
 <br/>
 自非洲猪瘟今年8月在中国大陆爆发后，9月7日，联合国粮农组织就曾发出警告，在中国大陆暴发的非洲猪瘟病毒，几乎不可避免将蔓延至亚洲其他国家。
 <br/>
 <br/>
 非洲猪瘟病毒既没有疫苗也无法医治，猪只感染后死亡率高达100%，由于猪肉产品在众多亚洲国家广泛生产和食用，尤其是在东亚和东南亚地区，因此非洲猪瘟病毒在亚洲其他国家的出现〝几乎是必然的〞。
 <br/>
 <br/>
 粮农组织首席兽医官卢布罗兹表示：〝不幸的是，迄今为止我们所见到的情况还只是冰山一角。〞
 <br/>
 <br/>
 目前，韩国、日本、台湾都已检测到源自中国大陆的猪肉制品中，携带有非洲猪瘟病毒。中国大陆周边国家都在严加防控疫情侵入。
 <br/>
 <br/>
 迄今为止，非洲猪瘟疫情在大陆已经失控。16日，官方宣布疫情已蔓延至四川。四川省宜宾市高县爆发非洲猪瘟疫情，涉事的养殖场有40只猪，其中16只发病、10只死亡。
 <br/>
 <br/>
 非洲猪瘟目前已扩散及中国18个省市自治区，覆盖大半个中国。有养殖业者透露，由于成本过高，目前猪肉的生产并没有对每只猪进行病毒精准监测。这造成病死猪肉有流入市场的风险。据了解，大陆很多民众现在都已减少吃猪肉，或不吃猪肉。
 <br/>
 <br/>
 近日，大陆一位负责防控猪瘟疫情的防疫人员向外界透露，疫情比想像的要严重的多！中共官方严禁防疫人员将疫情外泄，违者视为泄密。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：戴明)
