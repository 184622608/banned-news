### 中共前军委班子几乎全军覆没 传二轮大清洗开始
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】随着中共前军委委员张阳和房峰辉，近日被开除中共党籍，褫夺上将军衔，至此中共前军委班子几乎全军覆没。有消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 正在进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第二轮.html" target="_blank">
  第二轮
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军队.html" target="_blank">
  军队
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清洗.html" target="_blank">
  清洗
 </a>
 ，这次的目标是东部和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北部战区.html" target="_blank">
  北部战区
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_领导层.html" target="_blank">
  领导层
 </a>
 。
 <br/>
 <br/>
 中共官方最近公布中共中央军委联合参谋部原参谋长房峰辉、政治工作部原主任张阳双双被开除党籍，褫夺上将军衔，官方还称，张阳以自杀方式企图逃避惩处，宣布对其追缴涉案财物。令外界再度关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在军中打〝虎〞的动向。
 <br/>
 <br/>
 10月19日，有美媒称，随着房峰辉移送审判程序，中共十八大以来，已有7名上将被查，其中有两个军委副主席、总参谋长、 总政治部主任、空军政委、武警司令等。有人戏言，习近平只身〝杀入〞军委，除了胡锦涛一人之外，前军委班子几乎全军覆没。
 <br/>
 <br/>
 报导引述消息人士透露，习近平已经开始进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第二轮.html" target="_blank">
  第二轮
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军队.html" target="_blank">
  军队
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清洗.html" target="_blank">
  清洗
 </a>
 了。下一步首当其冲的是东海和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北部战区.html" target="_blank">
  北部战区
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_领导层.html" target="_blank">
  领导层
 </a>
 ，虽然未必是战区司令，但级别一定不低。
 <br/>
 <br/>
 知情人士说，军队抓人都是由军纪委巡视组完成，巡视组掌握生杀大权，甚至专供司令政委的机密信息，巡视组也可以查阅。而官方16日宣布将房峰辉移送起诉，对军队震动很大，中共将领们都不敢肯定自己是否能平安。
 <br/>
 <br/>
 中共军权过去20多年一直被江泽民集团把持，江从1989年至2004年任军委主席。江下台前，先后提拔了心腹郭伯雄、徐才厚为其把持军权，架空了胡锦涛。
 <br/>
 <br/>
 中共十八大后，习近平上台对军队进行了强力整顿。据中共披露的数据显示，习在第一个5年任期内，共查处了近160名军级以上军官，但对外公布的人数，据不完全统计有71个〝军虎〞。
 <br/>
 <br/>
 虽然当局查处了近160名军级以上官员，但房峰辉与张阳落马后，中共军报仍强调，〝彻底肃清郭伯雄、徐才厚流毒影响〞和反腐败斗争任重道远。
 <br/>
 <br/>
 8月28日，香港《明报》称，中共南部战区政委魏亮和原广州军区司令徐粉林，传出被降级退役的消息。两人都是中共十八届中央委员，十九大时已榜上无名。
 <br/>
 <br/>
 报导中没有解释魏亮与徐粉林被降级的原因。但两人均与张阳有某种关联。
 <br/>
 <br/>
 《苹果日报》同日也引述消息披露，现任中共西部战区副司令的刘小午中将，近日也被军纪委拘查，原因或与张阳上将有关。
 <br/>
 <br/>
 7月下旬，就有爆料称，刘小午中将正在接受调查，这是自张阳和房峰辉落马后，清除张、房〝余毒〞行动的延续。报导还称，在清除郭伯雄、徐才厚〝余毒〞活动中，打落张、房两只〝军虎〞，接下来，军中还要继续清除张、房〝余毒〞。
 <br/>
 <br/>
 8月24日，港媒星岛日报引述接近军方消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东部战区.html" target="_blank">
  东部战区
 </a>
 副司令员兼参谋长、原总参二部（情报部）部长杨晖，8月22日因涉嫌严重违纪接受军纪委调查。
 <br/>
 <br/>
 军方知情者透露，杨晖涉嫌与徐才厚、郭伯雄一起组建私家军，意图对中南海不利。
 <br/>
 <br/>
 不过上述4只军老虎出事的消息，至今仍没得到官方证实。
 <br/>
 <br/>
 在8月17日至19日的中共中央军委党建会议上，习近平强调，部队要对中央绝对忠诚，全军要坚决维护中央权威和统一领导，从严治理军队腐败，反腐不会〝变风转向〞等。
 <br/>
 <br/>
 对此，有分析说，习又一次发出反腐不会〝变风转向〞信号，或意味着军队将迎来新一轮的大清洗。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：赵云）
