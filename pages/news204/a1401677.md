### 港媒：贺建奎已被〝软禁〞 基因编辑婴儿内幕流出
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】中国深圳科技大学副教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 〝基因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_编辑婴儿.html" target="_blank">
  编辑婴儿
 </a>
 〞的科研活动，遭到国内外同声谴责。近日，港媒引述内幕消息称，贺建奎现已被校方〝软禁〞。而贺建奎基因编辑婴儿的更多黑幕曝光。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 日前在香港第二届基因编辑国际峰会上称，为〝基因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_编辑婴儿.html" target="_blank">
  编辑婴儿
 </a>
 〞这项医学上的突破感到自豪。
 <br/>
 <br/>
 贺建奎称，他和他的团队为丈夫携带艾滋病毒的一对夫妇成功修改了他们的双胞胎胚胎的基因。实验共有7对夫妇参与，〝露露〞及〝娜娜〞是实验中第一对诞下的婴儿，他对自己的研究感到骄傲，若有需要，也会试着为自己的婴儿做基因编辑。
 <br/>
 <br/>
 然而，香港苹果日报12月1日称，来自深圳南方科技大学（简称南科大）的内幕消息披露，贺建奎已经于11月28日在峯会之后，被南科大的校长陈十一来港将他带回深圳，贺建奎现已被校方〝软禁〞。
 <br/>
 <br/>
 内幕消息称，该事件是一宗有意逃避西方法律、学术法规与监管制度而进行的人体胚胎基因编辑试验，主谋是几位海内外生命科学专家。
 <br/>
 <br/>
 参与者包括:贺建奎在莱斯大学的博士论文指导老师Michael Deem，及他在史丹福大学的博士后导师，团体中甚至包括哈佛大学的教授。贺建奎去年底曾亲自去了哈佛大学，向团队中的哈佛教授汇报实验进度。
 <br/>
 <br/>
 对于为何选择中国作为试验地，内幕消息称：就是看中了中共法律的漏洞和贺建奎这个大傻瓜！而基因编辑婴儿的主要资金来源，是以〝瀚海基因测试仪公司〞的名义募集的社会融资，约有2亿多人民币。这些利益集团在幕后操纵这场基因编辑实验。
 <br/>
 <br/>
 内幕消息还称，南科大即将就事件发布正式公告。
 <br/>
 <br/>
 贺建奎于10月26日对外媒公开宣布，基因编辑婴儿诞生，次日中共《人民日报》对贺建奎〝基因编辑实验成功〞的消息做出了正面报导，随即科学界内外震荡，贺建奎的研究受到强烈谴责。
 <br/>
 <br/>
 隶属美国联邦政府的美国国家卫生研究院（NIH）11月28日发表声明，对贺建奎声称实施了首例人类基因编辑实验表示〝深度忧虑〞。声明中说，贺建奎和他的团队暗地里和不负责任地实施了首例人类基因编辑，其蔑视国际伦理准则的行为令人深切不安。
 <br/>
 <br/>
 同时，香港峰会组织者发表声明，严厉指责贺建奎的人类基因编辑实验，对这项并未得到独立验证的实验深表不安。即使这项基因编辑实验得到了独立验证，这项实验的程序也是不负责任，不符合国际规范。声明要求对贺建奎声称的基因编辑实验进行独立的评估。
 <br/>
 <br/>
 此后中共官方的态度紧急发生反转，中国遗传学会、中国科协生命科学学会联合体以及中国细胞生物学学会都相继发表声明，谴责将基因修改用于生殖目的。
 <br/>
 <br/>
 贺建奎所在的南方科技大学、广东省和深圳市卫生委员会还宣布对贺建奎的实验进行调查。贺建奎所在的南方科技大学生物系全体教授，以及中国122名科学家联署声明对他进行谴责；46名律师联合声明，呼吁警方对贺建奎立案。
 <br/>
 <br/>
 随着事件的发酵，多家媒体起底，贺建奎是中共通过〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞从美国挖回来的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尖端人才.html" target="_blank">
  尖端人才
 </a>
 〞，其基因编辑项目有来自南方科技大学和深圳市科技创新委员会的经费支持。
 <br/>
 <br/>
 因此外界认为，基因编辑婴儿的诞生，与中共争夺〝科技第一〞的野心分不开。而中共在医学伦理方面立法模糊，而且需要科研人员不择手段地为其争创〝世界第一〞，是否会查处贺建奎尚待观察，其目前最大的目的只是表明与贺建奎切割。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
