### 【禁闻】美报告：中共渗透 破坏民主价值
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】长期观察中国的32位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 专家最近提出一份报告，呼吁美国民众认知北京试图扩大在美国社会的影响力。报告说，中共透过〝渗透〞的方式在操弄美国政商学界。
 <br/>
 <br/>
 在阿根廷召开的G20峰会期间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统川普和中共领导人习近平达成协议，中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 休战90天，暂时不扩大关税战场，但双方必须期限内达成协议。
 <br/>
 <br/>
 有些人认为中美贸易战升格为新冷战，美国却只赢了延期协议。对此，中国经济学者何清涟认为，从全局来看，关税之外，美国收获很多。从结果来看，美国已经通过各种外围战，达成他最重要的目标，就是：防止中国通过盗窃知识产权、实施中国制造2025计划，对美国制造业形成巨大威胁。
 <br/>
 <br/>
 旅美时事评论员邢天行表示，休战90天只是一个暂时的结果，但从中可以看到中共服软的态度。
 <br/>
 <br/>
 旅美时事评论员邢天行：〝这次‘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 ’会谈的结果，恰恰证明了中共想通过各种手段想在美国渗透达到自己的目地，彻底的宣告破产。也就是，这次的‘川习会’中共从一开始的强硬态度到服软的表现。〞
 <br/>
 <br/>
 美国财经网站CNBC报导，国际战略研究中心（CSIS）中国问题专家甘思德（Scott Kennedy），观察中美贸易战发展方向的下一个关键日期，是两周后的12月18号，因为那天是中共所谓的改革开放40周年纪念日，他认为，如果那时候中共没有重大声明，就可以确定中共没有要藉机改变经济及产业政策方向的可能。如果是这样，90天协商到期后，美国和中国之间的关税大战一定再次升温。
 <br/>
 <br/>
 在川普与习近平会面的前夕，长期观察中国的32位美国专家提出一份报告，呼吁美国民众认知北京试图扩大在美国社会的影响力。
 <br/>
 <br/>
 这份长达192页、标题为《中国影响力及美国利益：提高建设性警觉》（Chinese Influence &amp; American Interests: Promoting Constructive Vigilance）的报告指出：〝中国的活动，在广度、财力投入深度和强度上，都需要比以往更严格的审查，因为中国在许多方面都比俄罗斯更具策略且更强有力的施加影响。〞
 <br/>
 <br/>
 邢天行：〝对于这些中国专家来讲，他们能够出来发声，对中共来讲，其实打击是非常大的。因为中共它在美国想在各方面进行渗透，他们长年培植的这些人，在很大程度上是宣告失败了。〞
 <br/>
 <br/>
 值得注意的是，这批中国事务专家学者在过去都是大力主张中、美经济合作，他们大部分的人对中国抱有憧憬。
 <br/>
 <br/>
 邢天行：〝在中美贸易战当中，谁都看到了，中国这边力量是不足的，也就是美国的强大重新让世界所认识，在这种情况下，其实才真正使得原来的这些中国问题专家，去反省他们自己原来的那些误判。〞
 <br/>
 <br/>
 时事评论员田园博士表示，改变对中共的看法，亲共的华盛顿大学政治学和国际关系教授沈大伟，就是其中的一位。
 <br/>
 <br/>
 时事评论员田园博士：〝中共在所谓的改革开放之后，沈大伟是经常去中国，中共把他待为上宾。沈大伟目前都已经发生这种态度和观念上的转变，说明美国的学界、政界、商界，都已经很清楚的认识中共的真面目是什么。〞
 <br/>
 <br/>
 报告举例，中共官方媒体加强在美国的英语传播，从印刷到电子及无线电广播等。与此同时，中共又对很多独立的中文媒体作出消音，逼迫自主性审查。使用的方法包括：拉笼现存的中文媒体。但中共政府却限制在中国活动的美国媒体。
 <br/>
 <br/>
 田园博士表示，中共在全球所谓的崛起，绝对不是一个善的力量，而是恶的力量，这对于全球的和平稳定和经济发展，都是一个巨大的威胁。
 <br/>
 <br/>
 邢天行认为，川普政府对于中共的清醒认识，和打击中共不正当的作为、积极的态度和决心，导致世界格局的转变，世界的潮流已经改变方向。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/钟元
