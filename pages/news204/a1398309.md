### 监控升级 北京上海利用〝步态识别〞系统维稳
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】通过走路姿势来判断你是谁，这可不是电影里才有的黑科技，就在最近，中国一家企业推出了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_步态检索.html" target="_blank">
  步态检索
 </a>
 人工智能系统，并已被架设在北京、上海的街道上用于维稳。有异议人士批评中共对民众的监控无所不用其极。
 <br/>
 <br/>
 继利用人脸辨识系统来监控国民后，中共政府支持的企业现在又研发出了〝步态识别〞（gait recognition）系统，并已经被一些大城市用于维稳。这套系统据称光从人们的身形、走路姿势就可以辨别身分，即使把脸遮住也没用。
 <br/>
 <br/>
 据美联社报导，北京银河水滴科技公司是在中共科学院主导下诞生的企业，该公司研制的〝步态识别〞系统目前已经架设在北京、上海的街道上。
 <br/>
 <br/>
 银河水滴的执行长黄永祯接受媒体采访时称，其他的生物辨识技术需要直接接触或近距离扫描才能发挥作用，而〝步态识别〞系统最大的特点是可以远距离运作。有别于他生物辨识技术，距离50公尺远就开始辨析人物，就算是背影、人脸被遮住、光线昏暗，都不会影响系统判断。
 <br/>
 <br/>
 黄永祯还进一步强调，就算人们在系统前故意走路外八或弯腰驼背掩饰也没用，因为这套系统是运用全身所有特征下去辨别分析。
 <br/>
 <br/>
 他说：〝步态的攻击不是简单的说，比如装个瘸装个外八字，或是装个驼背，他就能够去攻击它的，因为我们用到的是全身所有的特征。〞
 <br/>

 <br/>
 美国《华尔街日报》曾撰文批评中国政府运用监视系统控制民众，报导指出，美国运用这种技术是为了辨别、找出嫌疑犯，中国却拿来用在监控人民。
 <br/>
 <br/>
 事实上，中共政府近几年大量投资于大数据监控系统、积极推动企业发展AI，在此过程中，人们发现中国人越来越没有了隐私的保障。
 <br/>
 <br/>
 针对这种情况，中国评论家石述思在接受台湾媒体的采访时表示，中共把生物识别技术用于维稳，用于加强社会管理是一个〝非常荒诞的场景〞，等同于中共政府强制人民掏钱装上探头来监控自己。
 <br/>
 <br/>
 据了解，早在2017年10月初，中共官媒就曾报导过中科院的专家正在开发研究〝步态识别〞这种新兴生物特征识别技术的消息，并公开声称这套系统将用于中共公安的监控系统，50米以内的人物，只要看走路的姿势，就能够准确的辨识出特定的对像。
 <br/>
 <br/>
 当时，北京异议人士查某在接受海外中文媒体的采访时表示，这种技术本身没有好坏，它是中性的，关键在于针对谁。他说：〝它是一把双刃剑，可以做好事，也可以做坏事。但中共不会把它用在打击真正的犯罪上面，而是把它用在监控民众身上，民众在它的面前就像是裸体一样。〞
 <br/>
 <br/>
 查某强调，中国大陆每年都有那么多的人口丢失，为什么那些严密的监控系统却不能帮助警方尽快破案？这就说明中共本身它并不是用那些技术来保护人民的生命财产安全，而是为了用它来加强对普通民众的监控，为的只是维护它政权的稳定。
 <br/>
 <br/>
