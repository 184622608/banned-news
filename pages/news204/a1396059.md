### 党内风波未平？政治老人四中全会前频露面
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】外界预料，中共十九届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 将在月底召开，经济政策将是主要议题。但中共内有改革僵局，外有贸易战高压，经济决策必将牵扯党内利益、路线以及权力之争。在此背景下，中共老人们近日频繁公开露面，引起外界关注和猜测。
 <br/>
 <br/>
 近一个月来，多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共元老.html" target="_blank">
  中共元老
 </a>
 相继露面。
 <br/>
 <br/>
 上月底，前常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 突然出现在河南嵩山少林寺。本月初，中共国家前副主席李源潮现身江苏，然后前常委、人大委员长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴邦国.html" target="_blank">
  吴邦国
 </a>
 在家乡安徽肥东高调现身。前常委、中纪委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴官正.html" target="_blank">
  吴官正
 </a>
 和政协主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李瑞环.html" target="_blank">
  李瑞环
 </a>
 先后在网络自媒体刊登文章，谈论为官处事之道或者对时政的看法。
 <br/>
 <br/>
 其中，露面最为频繁的是前总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 ，10月12日出席清华大学活动，18日又在北京登门拜访了老领导袁宝华。
 <br/>
 <br/>
 美媒评论认为，中共元老们在举行重要会议前刻意公开亮相，在某种程度上是一种政治表态。
 <br/>
 <br/>
 另有台媒分析，目前正值中美贸易战不断升级，中国经济吃紧，高层权力洗牌、政见纷争之际，中共元老们的露面和发言，还可能蕴含其它特别的意味。
 <br/>
 <br/>
 自由亚洲引述分析称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 召开的消息来得比较突然，而中共元老们在会前频频亮相，或显示党内斗争还不明朗，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 尚未完全掌握会议议程。
 <br/>
 <br/>
 另有分析说，中美开启了新冷战，北京的对策在党内备受指责。习近平需要再次强调其〝核心〞地位，同时凝聚党内不同意见应对美国。
 <br/>
 <br/>
 而台媒指，中共元老纷纷露面有两方面原因。一方面可能是习近平需要一些大佬出面为他护航和献策，另一方面也可能有元老趁机跟习近平搅局对抗。因此，这可视为北京高层局势不够稳定的迹象。
 <br/>
 <br/>
 近期露面的中共元老中，李瑞环和朱鎔基被指江泽民的对头，此前多次现身为习近平反腐打虎背书。而吴邦国、吴官正和刘云山都被指属江派人马。不过也有说法称，吴邦国曾对江泽民反戈一击，站在习近平一边。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
