### 川习会徒有虚名？美媒：白宫根本不抱期待
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 硝烟弥漫，外界关注双方拟在11月举行的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞能否终结战火。不过，美媒日前爆料称，有消息人士透露，贸易战还处在〝开始的开始〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 对两国元首的会晤，根本不抱期待。
 <br/>
 <br/>
 在中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 不断升级的背景下，将于11月在阿根廷举行的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞带来了和解的曙光。但据美国Axios新闻网站10月21日报导，3名消息人士称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统无意放缓对中共征收关税，他认为关税需要更多的时间，才能完全生效。
 <br/>
 <br/>
 其中一名熟悉川普谈话的消息人士称，贸易战还处在〝开始的开始〞。川普总统的团队并不对暂定于11月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 〝川习会〞抱有太大的期待。到目前为止，他们还没有为这次双边会晤议程做实质性规划。
 <br/>
 <br/>
 另一名知情的消息人士称，这是元首会晤，不是贸易会晤。这次会晤的目的是让川普和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在贸易战升级之际，面对面的会晤、沟通，感受对方。
 <br/>
 <br/>
 早些时候，《华尔街日报》引述消息人士称，对于G20峰会期间的川习会晤，美国已经组成专家小组为会晤做准备。
 <br/>
 <br/>
 报导称，中方期待川习会晤，能对中美贸易关系产生积极的影响。
 <br/>
 <br/>
 10月2日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫.html" target="_blank">
  白宫
 </a>
 首席经济顾问库德洛表示，川普愿意在20国集团会议上，与习近平谈一谈。〝谈话比不谈话好。但是谈话必须是严肃的。〞它得是具有实质、重要和认真的会谈。
 <br/>
 <br/>
 库德洛还表示，美国并不急于与北京达成协议。他透露，目前双方的讨论〝还不够详细〞，迄今为止的会谈主要集中在企业所有权和知识产权盗窃方面。
 <br/>
 <br/>
 英国《金融时报》10月22日报导称，库德洛指责北京至今没有采取行动，以缓解贸易紧张。这令外界对川习会的前景，产生质疑。
 <br/>
 <br/>
 事实上，川普早些时候曾在白宫表示，北京非常想谈，但他认为还不是时候，北京多年来从美国占了太多便宜，必须改变这个现状，但这需要一点时间。
 <br/>
 <br/>
 他表示，中方想要达成协议，但他们还没准备好。为此我们取消了几次会议，他们还没准备好达成协议。
 <br/>
 <br/>
 有分析认为，川普总统只是想要中共遵循公平贸易原则，但这对中共来说，改变中共经济结构，是对中共体制的致命冲击。
 <br/>
 <br/>
 目前，距离美国中期选举还有两周的时间，对这场关系川普政府施政的大选，各方都在关注。此前已有分析指，中共将在这次大选结束后，确定应对川普政府关税制裁的实质性策略。
 <br/>
 <br/>
 然而，对于美国中期大选，中共似乎不愿袖手旁观。10月19日，美国国家情报总监、司法部、联邦调查局和国土安全部四大执法部门，发表了一份联合声明，点名中共、俄罗斯和其它外国势力，干扰、破坏美国大选。
 <br/>
 <br/>
 在此之前，川普总统也曾在联合国会议期间公开指责，中共正在干预美国的中期选举，川普政府握有证据。有美国资深官员表示，中共干预大选的方法，包括针对某些地区以及州施加关税。
 <br/>
 <br/>
 自贸易战开打后，中美关系处在近50年来最为紧张的时刻。目前，中美两国军事对话已暂停，早些时候美国国务卿蓬佩奥在北京，与中共外长王毅就贸易和朝鲜无核化问题，展开舌战。
 <br/>
 <br/>
 此外，美国副总统彭斯发表的白宫对华政策演说，从多方面批评中共政策和不端行为，表明美国对中共的战略已经从接触转向对抗。
 <br/>
 <br/>
 多位专家表示，中美世界两大经济体之争，表面上看起来是贸易冲突，实质上是意识型态及价值观之间的竞争，其结果是将建立一个新的世界格局。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
