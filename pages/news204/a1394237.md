### 孔琳琳把爱国做成生意？网友：幼园性侵假疫苗咋不发飙
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】中共央视记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孔琳琳.html" target="_blank">
  孔琳琳
 </a>
 近日在英国保守党内一个讨论香港问题的会议上发飙打人，该事件在中国网络社群中引发热议。不少反感其作为的网友批评孔琳琳〝虚伪〞 自甘堕落为〝五毛〞，把爱国当成生意来作骚；更有网友质问她，当中国幼儿被性侵、毒奶粉肆虐、P2P雷爆、农民被强拆以及假疫苗泛滥时〝咋不发飙〞？
 <br/>
 <br/>
 中共央视官派常驻伦敦记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孔琳琳.html" target="_blank">
  孔琳琳
 </a>
 ，近日大闹英国保守党讨论香港问题的会议现场并掌掴工作人员。针对这起事件，中国网民在网络社交媒体上爆发了阵线两极分明的争议。支持者吹捧孔琳琳的行为是〝爱国〞；反对者则批评孔琳琳是拿〝爱国〞做生意，只不过是把高喊几句虚伪的所谓〝爱国〞口号当作升官发财的捷径而已；更多的网友则批评孔琳琳当众撒泼还动手打人的粗暴举止其实是〝丢了中国人的脸〞，也令〝记者〞这个头衔〝蒙羞〞。
 <br/>
 <br/>
 有网民发帖讽刺说，〝这泼妇成民族英雄了！〞有网友留言讥讽道：〝有当年红袖章（文革造反派）的风范。〞还有网友嘲讽道：〝国内闹一个试试？因为她知道，在英国闹贼安全，回来还能升职。〞
 <br/>
 有网评质问孔琳琳，〝幼儿被性侵、毒奶粉事件、P2P雷爆、农民被强拆……这些，女记者怒吼了吗？〞
 <br/>
 <br/>
 网名为〝阿海很快乐〞的网友则爆料称：〝 我前几天就看到了，说她儿子在学校打人被开除了，然后她就发疯了。〞
 <br/>
 <br/>
 中国著名摄影家周雁鸣也在微博爆料说，〝记者摄影圈最早爆出她的劣迹〞。〝她儿子被英国学校开除了〞，原因是〝逃课成绩差不遵守校规。〞
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-6/p9067052a836762785.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-6/p9067052a836762785-ss.jpg"/>

    中共央视记者孔琳琳2018年9月30日大闹英国保守党讨论香港问题的会议现场并掌掴工作人员。（网络截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 英文网站〝改变中国〞主编曹雅学日前坐客《美国之音》〝时事大家谈〞时表示，孔姓记者的表现让她的同业者蒙羞。
 <br/>
 <br/>
 曹雅学还详细披露了孔琳琳两年前攻击北京房山参选基层人大代表的独立候选人刘惠珍女士的往事。
 <br/>
 <br/>
 据曹雅学讲述，中共地方政府强拆受害者刘惠珍2016年以独立候选人的身份积极争取参选基层人大代表的资格，此事引起了英国媒体BBC中文网的兴趣。当BBC记者去刘惠珍家里采访她时竟被七八个黑衣打手跟踪、围堵和强行驱离。
 <br/>
 <br/>
 该事件被曝光后，孔琳琳曾接连三天在推特上发文，诬指BBC制造〝假新闻〞还企图介入中共司法，又指责刘惠珍不遵守中共的选举法律，并大骂刘〝勾结境外势力〞，还称刘为〝罪犯〞。
 <br/>
 <br/>
 当时有网友发帖警告孔琳琳，〝这么颠倒黑白，甘作中共喉舌爪牙，共党倒台后要追诉刑责的。〞 刘惠珍也曾在推特上发帖质问孔琳琳：〝你的良心到哪里去了？国家（中共政府）施行恶行，你做为记者都做了什么？！〞
 <br/>
 <br/>
 据公开的资讯，孔琳琳在英国发飙打人事件发生于今年9月30日，当时英国保守党的一个党内会议正接近尾声，保守党人权委员会副主席罗杰斯发言说，〝我批评中国当前的政府以及它对待人民的方式，但我支持中国这个国家和中国的人民，我相信香港的成功符合中国和英国的共同利益。所以，遵守中英联合声明规定的义务，保障一国两制的实行，对中国和英国都有好处。〞
 <br/>
 <br/>
 不料话音刚落，孔琳琳就站起来对着罗杰斯尖叫：〝你撒谎！你反华！你想分裂中国！你连中国人都不是。其他人都是汉奸！〞
 <br/>
 <br/>
 当时有工作人员上前告诫孔琳琳违反了会场规则，请她离场，不料她拒绝离开还出手掌掴了一名工作人员。孔琳琳在被赶离场时，还不断高叫〝英国没民主〞、〝我是记者〞、〝我有权示威〞。
 <br/>
 <br/>
 其后，孔琳琳因为普通袭击罪被警方拘捕，稍后获保释，但警方强调有关的调查仍在进行中。
 <br/>
 <br/>
 一位常常活跃于香港社会运动的青年就孔琳琳事件接受BBC中文网采访时说，〝一个来自不民主国家、从不在自己领土上示威的人，去另外一个民主国家，说自己有示威权利，这是多讽刺、多可笑的事。〞王先生说，〝如果是外国记者在中国（中共）两会上抗议、大吵大闹，后果又会是怎么样？〞
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：秦锋）
