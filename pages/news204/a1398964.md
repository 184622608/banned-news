### 【禁闻】11月11日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 新规断生路 西安数百货车主市府抗议
 <br/>
 <br/>
 西安市将于11月16号开始，对两种型号的柴油货车实施限时进入三环和绕城高速路的规定，一大批货车车主因此被断生路。11月7号、8号，大约700多名受影响的车主，集体到交警支队及市政府、信访办抗议。
 <br/>
 <br/>
 2. 江苏东方重工
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欠薪.html" target="_blank">
  欠薪
 </a>
 5年 职工集体抗议
 <br/>
 <br/>
 江苏省靖江市东方重工公司破产重组，拖欠上千名职工近5年的工资和社保金，11月8号，职工集体到公司门口维权，当局出动警察戒备。
 <br/>
 <br/>
 3. 山东技校教师罢课 抗议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 学校
 <br/>
 <br/>
 山东济宁山推技校的老师，从10月底至11月初连续多天罢课静坐，抵制企业强迫学校拆迁，学生也自发聚集抗议。
 <br/>
 <br/>
 4. 无锡抗强拆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 遭多次传唤
 <br/>
 <br/>
 因无锡举办〝双11〞国际马拉松赛跑，当地抗强拆维权人士王建芬最近多次遭派出所传唤，并被不明身份人员跟踪，11月9号，政府维稳人员甚至闯入她家中监控。
</div>
