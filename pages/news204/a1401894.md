### 东莞企业放长假 贸易摩擦后果初现
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】位于大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东莞.html" target="_blank">
  东莞
 </a>
 的代工业者〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_劲胜精密.html" target="_blank">
  劲胜精密
 </a>
 〞日前公告，部分员工〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_放长假.html" target="_blank">
  放长假
 </a>
 〞5个月。分析指，近来大陆科技代工大厂订单减少，再加上中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易摩擦.html" target="_blank">
  贸易摩擦
 </a>
 更使情况雪上加霜。
 <br/>
 <br/>
 11月26日，一份关于劲胜智能经营遇到重大困难的放假告知函在网上流传。该函件显示，因公司目前经营遇到重大困境，部分业务停产，将安排部分工作人员提早放春假，期间从12月1日到明年的4月30日。时间为2018年12月1日至2019年4月30日。
 <br/>
 <br/>
 根据公告，被放假员工在放假期间，首月会支付正常工资，剩余月份按
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东莞.html" target="_blank">
  东莞
 </a>
 市2018年最低工资标准的80%支付至长假结束。
 <br/>
 <br/>
 对此，该公司回应陆媒时表示，公告内容属实，公司对消费电子精密结构件业务部分人力进行精简，涉及18名员工。
 <br/>
 <br/>
 劲胜智能的官网近日发布说明表示，由于公司人力资源部相关人员工作失误，向员工发出告知函中关于公司经营情况的表述严重失当，不符合公司实际情况。但并未说明具体失当之处。
 <br/>
 <br/>
 世界许多手机的加工厂设厂广东东莞，劲胜智能成立于2003年，注册地在广东省东莞市，于2010年5月20日在深交所创业板挂牌上市。公司消费电子精密结构件产品的服务对象涉及三星、华为、等手机厂商。分析相信，类似
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_劲胜精密.html" target="_blank">
  劲胜精密
 </a>
 的其他企业正遭遇同样问题。
 <br/>
 <br/>
 2018年三季报显示，劲胜智能今年前三季度营业收入为41.65亿元，同比下滑10.76%；净亏损1.02亿元。
 <br/>
 <br/>
 劲胜智能曾在今年年中披露，公司消费电子精密结构件业务，受到智能手机市场增速放缓、中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易摩擦.html" target="_blank">
  贸易摩擦
 </a>
 等外部因素影响，市场竞争加剧、部分客户订单量下降。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
</div>
