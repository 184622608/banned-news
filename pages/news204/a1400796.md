### 崔永元曝农业部食堂：腐竹都要非转基因（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 近日频频发文曝光中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 内幕。25日，他在微博披露：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农业部食堂.html" target="_blank">
  农业部食堂
 </a>
 ，腐竹都要吃非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 的。只顾自己保命，不管百姓死活。〞并同时附上曝光中共农业部食堂内部场景的视频。
 <br/>
 <br/>
 11月25日13时40分，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 在微博上发文，披露中共农业部内部食堂内幕。从他发布的视频显示，中共农业部内部食堂使用的腐竹，上面标注有〝非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 〞的字样。
 <br/>
 <br/>
 视频中一位正在熬制粥类食物的厨师，对视频拍摄者称，食堂所使用的食材、包括调味品，都是非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 的。
 <br/>
 <br/>
 崔永元在文中痛批，中共农业部〝只顾自己保命，不管百姓死活。〞
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 3天前，崔永元还在微博曝光了中共农业部的特供基地：〝这个地方叫云谷山庄，也叫农业部绿化基地。在北京市密云穆家峪镇北硷厂村，离城区很远，相对也就很隐蔽。〞
 <br/>
 <br/>
 崔永元在文中质问中共农业部官员，〝咱就不说你们在这儿大吃大喝过多少全国各地送来的最好的特产，只想问农业部各位部长，你们在这儿吃过一次转基因食品吗？你们真不要脸！〞
 <br/>
 <br/>
 更早些时候，11月10日，崔永元转发网友评论并点名中共农业部称，〝农业部的特供基地在新疆吐鲁番，全是有机的。另有大鱼、大虾、各种新鲜土特产等，鸡蛋也是特供的。现在国家管得严，不敢了。顺便翻脸不承认以前有过。〞
 <br/>
 <br/>
 近期，崔永元多次在微博披露，中共农业部把转基因吹鼓上了天，但自己却吃特供。11月22日，他还在微博转发了中共退役少将彭光谦，于2013年8月21撰写的关于中国主粮转基因化的文章。
 <br/>
 <br/>
 文中指，转基因食品与肿瘤、不孕不育、胚胎异常等数十种疾病存在高度相关性。
 <br/>
 <br/>
 彭光谦当时对陆媒直言：当前转基因农作物的安全性还远未得到世界的确认，世界各国对转基因农作物商业化无不严加限制，欧盟甚至对此实行〝零容忍〞，中国却急不可耐地大面积引进和推广，这种做法十分令人费解。
 <br/>
 <br/>
 他表示，国际科技界对转基因十分慎重，近年来，已有许多严肃的科学家经过大量实验，证实转基因食品与肿瘤、不孕不育、胚胎异常等数十种疾病的高度相关性。
 <br/>
 <br/>
 而且西方国家在进行转基因的研究时，有一条决不跨过的红线，那就是不对自己国家的主粮搞转基因。
 <br/>
 <br/>
 彭光谦披露，早在2000年，全球828位科学家就向各国政府紧急呼吁，叫停转基因，指出转基因作物违反生态资源的可持续性，将使人类社会面临前所未有的危机。
 <br/>
 <br/>
 2006年，400名国际农业知识与科技促进发展评估专家，经过6年研究，得出的结论是，转基因不适用于全球的农业发展。
 <br/>
 <br/>
 针对这些已经揭示的灾难性后果和科学家们的一再警告，彭光谦质疑道，〝既然有人如此热衷于引进转基因，为什么不先在自己的机关食堂和幼儿园进行转基因食品试点，而要把13亿人民当作小白鼠呢？〞
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
