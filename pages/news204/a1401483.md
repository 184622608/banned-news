### 中兴助委国监控国民事发 美参议员吁调查违规行为
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】路透社独家报导，两名美国参议员将要求川普（特朗普）政府调查，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴.html" target="_blank">
  中兴
 </a>
 通讯是否为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 政府提供技术，帮助其监控本国民众，从而违反了美国的制裁。议员并要求，如果调查结果属实，应该再次对中兴实施
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁售令.html" target="_blank">
  禁售令
 </a>
 。
 <br/>
 <br/>
 路透社11月28日独家报导，在该通讯社揭露
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴.html" target="_blank">
  中兴
 </a>
 帮助
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_委内瑞拉.html" target="_blank">
  委内瑞拉
 </a>
 当权者监控民众之后，参议员霍伦（Chris Van Hollen）和鲁比奥（Marco Rubio）日前致信美国国务卿、财政部长和商务部长，要求确认中兴通讯是否与受到美国制裁的个人合作，非法使用美国零部件，以及帮助委内瑞拉政府侵犯人权。
 <br/>
 <br/>
 民主党人霍伦和共和党人鲁比奥一直支持美国此前针对中兴的措施。
 <br/>
 <br/>
 《美国之音》报导，目前委内瑞拉总统马杜罗正在极力应对该国恶性通货膨胀，以及自由落体式下跌的经济。
 <br/>
 <br/>
 而中兴的最大股东是中共国企，许多西方官员指责该公司帮助中共向世界各地的威权政府出口监控技术和设备。
 <br/>
 <br/>
 路透社曾在11月14日推出特别报导&lt;中兴如何帮助委内瑞拉建立中国式社会控制&gt;，披露中兴在帮助委内瑞拉当权者开发新一代身份证的同时，也为其建立了一个监控公民的系统。
 <br/>
 <br/>
 路透社记者经过几个月调查发现，委内瑞拉民众正在使用的〝祖国卡〞，是效仿中共的社会信用体系而建，里面记录了公民的家庭信息、雇主、收入、个人财产、医疗历史、社交媒体、所属政党、投票与否等完整的信息。这个系统的创建者正是中兴。
 <br/>
 <br/>
 报导指，〝祖国卡〞越来越多地与汽油价格优惠、食品补助、健保和社会项目挂鈎，国民若不加入〝祖国卡〞计划，将寸步难行。这迫使人们不得不给独裁政权投票。
 <br/>
 <br/>
 据《路透社》调查，委内瑞拉去年已委托中兴协助建立数据库，并创建行动支付系统用于该卡，中兴也已派员进驻委内瑞拉国家电信公司 Cantv 以维护该系统。
 <br/>
 <br/>
 根据中兴的一份文件，这些项目中安装了由美国戴尔公司制造的数据存储单元，这可能违反美国的进出口管制。而戴尔发言人 Lauren Lee 回应，这些数据存储单元并非由戴尔出售给委内瑞拉政府，而中兴是戴尔的客户之一。
 <br/>
 <br/>
 根据美国的进出口规定，采购美国公司元件和技术的公司，必须承诺不会将这些元件和技术转售给朝鲜、伊朗和委内瑞拉等遭美国制裁的〝流氓国家〞。
 <br/>
 <br/>
 今年4月，中兴因为和伊朗的非法交易被美国政府下达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁售令.html" target="_blank">
  禁售令
 </a>
 ，一度陷入瘫痪。3个月后，美国放生了中兴，但作为条件，中兴缴纳了10亿美元罚款和4亿美元保证金，并更换了领导层，同时接受美方官员进驻中兴进行合规监查。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
