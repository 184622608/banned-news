### 刘强东性侵案后财减三成 中国富豪榜跌至30名
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】近日公布的2018福布斯中国400富豪榜显示，京东创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 排名从去年的第16位跌至今年的第30位。自刘强东在美国涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵丑闻.html" target="_blank">
  性侵丑闻
 </a>
 曝光后，京东股票暴跌，刘强东净资产减少了34%。
 <br/>
 <br/>
 美国福布斯杂志近日公布了2018福布斯中国400富豪榜，今年45岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 身价缩水到62亿美元，排名也跌至第30位。而其掌控的京东市值今年已经跌掉一半。
 <br/>
 <br/>
 自刘强东在美国涉嫌一级强奸重罪被拘捕、指控丑闻曝光至今，京东股价已下跌25%。
 <br/>
 <br/>
 据报导，今年8月底，刘强东在美国明尼苏达大学参加一个商业管理课程期间，被指控涉嫌性侵明大一名中国女留学生。刘强东遭警方拘捕，并随后获释。目前警方已经完成调查，将案件移交给检察官办公室。但尚未对刘强东提起正式起诉。
 <br/>
 <br/>
 据美国警方消息，强奸罪是一级重罪，一旦刘强东被定罪，可能面临最高30年的刑期。
 <br/>
 <br/>
 刘强东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵丑闻.html" target="_blank">
  性侵丑闻
 </a>
 曝光后，大陆网易创始人丁磊曾在网路爆料称，刘强东是灌酒性侵惯犯。刘曾在一个多人参加的酒局上，将一名女子灌醉，并当场在酒席包间的厕所内实施性侵。
 <br/>
 <br/>
 早在2015年，刘强东也被牵扯进一起性侵事件。刘在澳大利亚悉尼一套顶层公寓举行私人宴会，一名澳洲嫩模在宴会结束后被性侵。刘强东这套顶层公寓也被其妻子挂牌出售。
 <br/>
 <br/>
 刘强东性侵丑闻也成为全球关注热点。9月17日在上海举办的2018世界人工智慧大会，作为中国电商巨头的刘强东本应是受邀嘉宾，但他最终未能出席。包括阿里巴巴CEO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马云.html" target="_blank">
  马云
 </a>
 等中国几位网路巨头都有出席。
 <br/>
 <br/>
 此外，2018年是中共改革开放40周年，北京官方推出〝百名杰出民营企业家名单〞，名单中没有万达集团创始人王健林、海航集团陈峰，以及刘强东。王健林、陈峰此前因涉嫌在海外违规并购而招致北京重罚；而刘强东则因在美涉嫌性侵一事导致名声不佳。
 <br/>
 <br/>
 另据福布斯最新的2018福布斯中国400富豪榜显示，229名中国富豪的财富都比去年缩水。重回富豪榜榜首的阿里巴巴董事局主席马云，财产为346亿美元，比上一年缩水40亿。
 <br/>
 <br/>
 排名第二的腾讯董事会主席马化腾，身家为328亿美元，比去年缩水了62亿美元。去年的首富中国恒大董事局主席许家印，今年降为第三名，财富减少117亿美元，为308亿美元，是财富降低最多的富豪。
 <br/>
 <br/>
 有分析称，自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 开打后，中共经济遭受重创，这些被称做〝红顶商人〞的中国富豪，也被贸易战火波及，生意受挫，财富被大幅缩水。马云在9月天津达沃斯峰会上曾警告大陆企业界，在中美贸易战的大背景下，经济形势会很糟糕，这时候千万不要捞浮财。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉）
