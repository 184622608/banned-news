### 美国启动中美科技谍战 重挫〝中国制造〞信心
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】日前，美媒揭露美国电路版被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造.html" target="_blank">
  中国制造
 </a>
 商植入恶意芯片，美官方高调控诉窃取美国商业机密的中共特工。港媒指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美科技谍战.html" target="_blank">
  中美科技谍战
 </a>
 开打，市场对〝中国制造〞的信心动摇，引发科网股下挫。
 <br/>
 <br/>
 《彭博商业周刊》近日接连披露，美国电路板制造商超微电脑（美：SMCI）的电板，在中国生产环节中被植入恶意芯片，藉以窃取苹果公司及亚马逊等30家美资企业的商业机密。
 <br/>
 <br/>
 报导引述知情人士指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造.html" target="_blank">
  中国制造
 </a>
 商向美国电路板植入芯片，是中共情报部门指导下的系统行为，受害的美国厂商不止一家。
 <br/>
 <br/>
 此外，一名中共情报官员涉嫌从美国通用电气航空公司窃取商业机密，日前从比利时引渡到美国，美方以经济间谍罪对其提出控诉。
 <br/>
 <br/>
 港媒日前分析，美方举动宣告
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美科技谍战.html" target="_blank">
  中美科技谍战
 </a>
 拉开帷幕，动摇了市场对〝中国制造〞的信心。这些事件凸显全球企业在中国组装智能产品的风险，再加上对科技谍战的担忧，或令科技板块面临一次减仓潮。
 <br/>
 <br/>
 上周的彭博报导，已令中国科技股急速下挫。本周随着美股大跌，中国股市跟跌更为惨烈。其中上证指数11日失守2600点大关，一日暴跌5.22%。特别是重磅股腾讯，其股价11日再创新低，全日急跌6.8%，市值跌出全球十大企业。
 <br/>
 <br/>
 中国巨头阿里巴巴董事局副主席蔡崇信也对贸易战表示担忧，称中美正朝往〝冷战或地缘政治战〞方向发展。
 <br/>
 <br/>
 对于日前全球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市暴跌.html" target="_blank">
  股市暴跌
 </a>
 的原因，业界众说纷纭。美国总统川普（特朗普）直指美联储〝疯狂升息〞是罪魁。金融学者则指，除了美联储升息之外，还有多重因素综合作用，引发这次股市动荡。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
