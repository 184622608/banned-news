### 川普:北京愿百分百合作朝核问题 韩媒质疑中共诚意
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】12月1日晚，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 后的返程飞机上表示，将与朝鲜党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 举行第二次会晤，并称〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 主席对解决朝鲜问题承诺百分之百合作〞。不过，朝核问题一直是中共试图用来牵制美方的棋子，韩媒对北京诚意表示质疑。
 <br/>
 <br/>
 韩媒《朝鲜日报》12月4日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 当晚在专机上向记者透露
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晚宴内容时，提到了北京的上述承诺。
 <br/>
 <br/>
 川普进一步表示，在朝鲜问题上，美中已协议进行〝强有力的合作〞，〝这是非常了不起的事情〞。
 <br/>
 <br/>
 他还说：〝我们在过去6-7个月的时间（与朝鲜就无核化）进行协商，请不要忘记，这并不算很长的时间〞，〝因为过去80年的时间一直在努力寻找解决方案，单论核问题也持续了20年以上。〞
 <br/>
 <br/>
 《朝鲜日报》解读，川普有关〝北京百分之百合作〞的发言，可能与其特有的夸张表达方式有关，意在向朝鲜传递〝尽快推进无核化〞的信息。
 <br/>
 <br/>
 不过，外界也有分析指，川普在川习会后突然重启与朝鲜的接触，可能说明中方确实有所承诺。
 <br/>
 <br/>
 据《美国之音》报导，川普当时在专机上还表示，与朝鲜党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 的第二次峰会可能会在明年1月或2月举行，双方正在考虑三个可能的会谈地点。而此前一个月，因为朝鲜再次变脸，美朝谈判已经中断。
 <br/>
 <br/>
 白宫就川习会发表的声明称，双方承诺将共同推进朝鲜无核化：〝美中领导人一致认为，朝鲜问题取得了巨大进展；川普总统将和习主席一起，并与金正恩委员长一道，努力实现一个无核的朝鲜半岛。〞
 <br/>
 <br/>
 不过，中共外交部长王毅2日介绍川习会情况时表示，中方希望〝并行推进半岛完全无核化和建立半岛和平机制〞，这是中方一贯主张的〝双轨并行〞原则。而川普政府主张〝先无核化、后和平机制建构〞。
 <br/>
 <br/>
 《朝鲜日报》称，在无核化和对朝制裁问题上，中美两国依然〝同床异梦〞。
 <br/>
 <br/>
 但是也有观察人士指，在美中90天谈判期内，中方有可能帮助美方向朝鲜施压，至少不会给美方搅局。川普在这90天谈判期内恢复对朝接触，可能就是出于这种考量。
 <br/>
 <br/>
 不过，90天谈判期一过，中美关系或将再次产生变化，朝核问题势必受到波及。
 <br/>
 <br/>
 根据白宫声明，在川习会上，北京承诺从美国购买大量农业、能源、工业和其它产品，管控此前大举输送美国的毒品，从新考虑高通收购案等；作为交换，美方同意将2019年1月1日提升中国产品关税税率的计划搁置90天。
 <br/>
 <br/>
 不过在这90天内，中方必须与美方开始就强制技术转让、知识产权保护、非关税壁垒、网络入侵和网络盗窃等问题展开谈判。如果届时不能达成协议，美方将重启关税升级计划。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
