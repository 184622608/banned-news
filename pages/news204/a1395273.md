### 黄琦狱中病重 八旬老母北京上访遭跟踪
------------------------

<div class="wysiwyg">
 【新唐人2018年10月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四天网.html" target="_blank">
  六四天网
 </a>
 创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狱中病重.html" target="_blank">
  狱中病重
 </a>
 ，他的母亲蒲文清为此进京，向中央各个部门反映，要求当局，允许黄琦出狱治病，但遭到绵阳当局派来的人跟踪，虽然在朋友保护下，老人已经甩开跟踪人员，但那些跟踪的人，还在北京找她们。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 的母亲蒲文清在两位朋友陪伴下到北京，11号上午刚在北京西站下火车，就有蒲文清原单位（内江一院）的职工杨夷跟踪，要她去驻京办，被她拒绝。据悉，当局总共安排了6、7个人从四川跟踪她到北京。
 <br/>
 <br/>
 陪同蒲文清进京的陈女士表示，她们甩开跟踪的人，但是那些人现在还在北京找她们。
 <br/>
 <br/>
 蒲文清友人陈女士:〝上来的时候，在车上就被跟踪了，我们两个朋友陪奶奶一块，照顾奶奶，就有身边跟踪的人给我们打电话，早上还在打，就想知道我们在哪儿，跟踪的人一直想找到我们，肯定是想带她回去。〞
 <br/>
 <br/>
 10月8号，李静林律师到绵阳市看守所会见黄琦。陈女士透露，律师说黄琦的情况不乐观，狱方几次抽血检查的结果都保密，也不对症治疗。蒲文清也身体不好，每天都要吃药，记者采访时她刚吃下14种药。
 <br/>
 <br/>
 陈女士:〝她儿子黄琦多年致力于为弱势群体发声呼吁，因此受到迫害，现在在狱中，尿毒症达到晚期，病情危重，奶奶一直要求当局允许她儿子出狱救治，一直没被允许，奶奶上来就是为她儿子喊冤。〞
 <br/>
 <br/>
 现年55岁的黄琦创办〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四天网.html" target="_blank">
  六四天网
 </a>
 〞，18年来坚持为弱势群体发声，两次判刑入狱。2016年11月，当局又以涉嫌〝非法为境外提供国家秘密罪〞，将黄琦羁押在四川绵阳看守所至今。同案被抓的杨秀琼和陈天茂被劝服认罪，两人罪名变更为量刑较轻的〝故意洩露国家秘密罪〞。杨秀琼因认罪当庭释放，黄琦至今一直坚持零口供，不认罪。
 <br/>
 <br/>
 新唐人记者熊斌、舒灿采访报导
</div>
