### 美学者犀利剖析新冷战：中共处处如翻版苏联必败
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】中美对抗日益升级，外界认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 时代来临。美国学者分析，如今的中共，无论是处境和采取的对策，都和当年的苏联非常相似，新冷战刚开局就已注定走上败北之路。
 <br/>
 <br/>
 10月16日，台媒《自由时报》引述《产经新闻》对美国加州克莱蒙特．麦肯纳学院（Claremont McKenna College）政府学教授裴敏欣的采访报导，剖析中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 。
 <br/>
 <br/>
 裴敏欣分析，当年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美苏冷战.html" target="_blank">
  美苏冷战
 </a>
 时，苏联也是企图超越美国。当时共产主义渗透欧洲，苏联经济成长率达到6%，这些如同今日的中共一般。而且，苏联当时还拥有550万兵力，核武战力超过美国。
 <br/>
 <br/>
 不过，苏联的一党专政、权力斗争与经济成长率灌水，这些与当今中共非常类似的体质，让强大的苏联最终走向毁灭。
 <br/>
 <br/>
 裴敏欣指出，苏联于1991年瓦解后，中共为避免重蹈苏联覆辙，开始强化经济，中国经济有了飞速发展。但是，当今中共对国企不愿放手，还面临债务压力与人口高龄化，再加上中美贸易战，未来经济趋缓恐难以避免。况且，由于政治体制原因，中共没有可以保持成长的经济模型，未来还得面临中美军事竞赛的庞大经费。
 <br/>
 <br/>
 裴敏欣表示，当年苏联失败的一大原因，就是经常卷入高花费的竞争中，军事费用是其中一个主要负担。中共也如出一辙，预计2025年国防预算将超过美国，2030年甚至可能超过美国的GDP，这将远远超过中国经济的承受能力。若是进入新冷战，无法排除中共会走上与苏联一般经济崩盘的道路。
 <br/>
 <br/>
 另外，苏联当年对外国投入过多资金，吸收东欧各国的债务，也是败因之一。而当今的北京力推〝一带一路〞，以背负大量坏债为代价，获取外国战略要衡和政治支持。若这种债务不断提高，可能迟早会象苏联一样走向失败。
 <br/>
 <br/>
 裴敏欣最后得出结论：美中冷战虽然才刚开始，中共已经完全走在败北的道路上了。
 <br/>
 <br/>
 另外，美苏冷战持续数十年，最终才拖垮苏联。而中共经济近年来已经陷入危局，中美贸易战一开局就瞄准了经贸领域。因此很多评论认为，新冷战开启后，中共的经济垮台不需要等待那么长时间。
 <br/>
 <br/>
 中国金融学者贺江兵甚至断言，贸易战会轻易引爆中共内债炸弹，刺破楼市和金融泡沫，中国经济未来数月内就将开始进入〝崩盘模式〞。
 <br/>
 <br/>
 今年以来，中国股市汇市跌跌不休，企业接连倒闭。学者批评，中共所谓的金融对策，实质就是加速印钞，通过各种名目释放资金，其放水规模前所未有。但是，这种引鸩止渴的救市措施恐怕很难持久。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
