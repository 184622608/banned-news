### 王全璋案律师刘卫国退出 知情者曝内幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709事件.html" target="_blank">
  709事件
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 被羁押已超过3年仍未开审。他的妻子李文足透露，代表律师刘卫国以健康为由退出该案。另有知情人士透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘卫国退出.html" target="_blank">
  刘卫国退出
 </a>
 王全璋案另有原因。
 <br/>
 <br/>
 现年41岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 ，曾代理多宗敏感案件，包括为法轮功学员出庭辩护。2015年8月他被警方带走，直到2016年1月，天津市公安局才以涉嫌〝颠覆国家政权罪〞正式逮捕他，并把他羁押在天津市第二看守所。
 <br/>
 <br/>
 2017年2月，王全璋案件移送至天津市中院，至今已超过20个月，但其家属聘请的律师从未获准与他会面。他的妻子李文足10月23日向自由亚洲电台证实，王全璋的代表律师刘卫国已退出该案。
 <br/>
 <br/>
 今年7月，自称受王全璋委托成为代表律师的刘卫国在网上表示，曾2度成功与王全璋会面。不过，刘卫国早前发声明称，因意外受伤，摔断了3颗牙齿，医生说要减少言语，因此无法继续履行辩护职责。
 <br/>
 <br/>
 熟悉刘卫国的〝中国律师后俱乐部〞发起人覃永沛认为，刘卫国是为了保住名声才决定退出。
 <br/>
 <br/>
 覃永沛：〝假如刘卫国出庭跟法院谈好条件把王全璋判缓刑出来，问题是刘卫国的角色就很不光彩。如果你一点不妥协，共产党不可能让王全璋出来。如果你不配合法院，法院不可能给你做（代理）律师。如果刘卫国出庭肯定对王全璋有利，问题损害的是刘卫国的声誉。〞
 <br/>
 <br/>
 对于刘卫国律师的事情，李文足表示没有什么可以评论的。但外界相信，他的退出，会使王全璋最终由官派律师代理出庭应讯。
 <br/>
 <br/>
 李文足认为：〝外界有什么猜测我觉得都是没有意义的。因为你不知道官方到底想怎么处理，或者什么时候处理这个事情。我能做的就是继续坚持，竭尽全力。我会始终坚持我聘请的律师。他们还在继续履行自己的职责，但是刘卫国律师不是我聘请的。〞
 <br/>
 <br/>
 另一名709律师李和平，去年被以〝颠覆国家政权罪〞判处有期徒刑3年、缓刑4年。今年6月被司法部门吊销律师执照。外界相信有关判决涉及了各方妥协。
 <br/>
 <br/>
 但覃永沛表示，这样的安排王全璋不会接受。〝我估计王全璋的性格不会接受缓刑的局面。一旦缓刑，他有几个小孩，老婆没工作。如果没能做律师，生活就非常艰难，因为他没有什么经费，没有什么钱。有罪他就不能做律师了。而且王全璋的行为光明磊落，没有找到一点违法犯罪的线索。〞
 <br/>
 <br/>
 王全璋是中国709大抓捕后至今唯一仍被羁押的维权律师，自王全璋身陷囹圄后，他的妻子李文足及亲属从未获准探望，李文足持续向外界呼吁，关注王全璋的情况。今年5月，德国总理默克尔访问北京期间，会见了王全璋的妻子李文足，对王全璋处境表示关注。
 <br/>
 <br/>
 （记者罗婷婷报导）
