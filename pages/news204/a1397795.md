### 川习〝战火〞中通话 专家：贸战倒逼中国改革
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 火持续升温的形势下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）总统与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 11月1日进行了〝长时间良好的通话〞，引发全球舆论聚焦。有专家指，贸战可倒逼北京改革，但中共恐怕不会有体制上、或社会上的变化。双方的互动不太可能改变
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美全面对抗.html" target="_blank">
  中美全面对抗
 </a>
 的格局。
 <br/>
 <br/>
 当地时间11月1日上午，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在推特上发文说，刚刚和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 主席进行了长时间和很好的通话。他们谈了许多问题，最重要的是贸易。正在安排的阿根廷G20峰会会面的讨论进展良好。他们也对朝鲜问题进行了很好的讨论。
 <br/>
 <br/>
 根据双方官方公布的信息，川普和习近平自2016年11月至2018年5月，共通电话13次。这次是今年5月以来，两人首次通话。
 <br/>
 <br/>
 目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 正处于恶化的边缘，美国已对中国2500亿美元商品加征关税，另外2670亿也很可能会兑现。外界认为，川习此次通话不同寻常，可能会使中美贸易纠纷局势发生变化。
 <br/>
 <br/>
 英国金融时报发表署名文章分析说，中美贸易战有小打、中打和大打三种可能性，而中共要度过难关，最关键的是要坚持继续开放，进行政治体制改革。
 <br/>
 <br/>
 川普政府此前也多次指出，美国要求北京在经济结构上做出实质性改变，这是解决中美贸易纠纷的关键。但外界认为，要中共改变经济结构，会对中共政权造成致命冲击，因此北京迟迟不愿做出承诺。
 <br/>
 <br/>
 还有专家认为，贸易战可倒逼中共进行改革，但变化可能不大。据自由亚洲报导，美国俄克拉荷马中心大学政治学教授李小兵表示，北京〝恐怕不会有体制上、制度上或社会上的改革，可能有一些技术上的调整，但战略上不会有变化。〞
 <br/>
 <br/>
 美国高地智库研究员秦伟平分析，这次川习通话不太可能改变美中贸易战引发的全面对抗格局。〝除非双方作出重大让步，否则谈拢的可能性非常小，贸易战和其他方面的冲突，可能会持续很长时间。〞
 <br/>
 <br/>
 目前，美国已将中共视为战略竞争对手，正在从经济、政治、军事、外交等全方位与中共对抗。此外，川普总统领导下的美国政府，对中共执政党和中国的定义区分得异常清晰。
 <br/>
 <br/>
 川普总统在今年的联合国大会上，严厉抨击残暴的社会主义和共产主义，给人民和世界带来痛苦和灾难。呼吁各国领导人共同抵制社会主义和共产主义。
 <br/>
 <br/>
 美国副总统彭斯近日发表对华政策演说时，全面抨击中共的恶行，并多次使用中国共产党一词，这非常罕见。有学者称，中美的全面对抗拉开了新冷战的序幕，是西方自由社会与共产主义的全面对抗。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
</div>
