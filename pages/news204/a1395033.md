### 【禁闻】10月11日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 庆〝双十节〞 广东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主人士.html" target="_blank">
  民主人士
 </a>
 被抓
 <br/>
 <br/>
 广东民主人士黄文勋因在网上发起〝双十节〞跑步活动，庆祝中华民国国庆，10月9号深夜被当地派出所人员抓走，11号中午仍然没有消息。他的妻子去公安局和派出所要人，遭到警方推诿。
 <br/>
 <br/>
 2. 河南近百辅警市府维权 要待遇
 <br/>
 <br/>
 河南三门峡市的近百名辅警，10月10号在市政府门前拉横幅维权，要求〝实现辅警待遇职业化保障〞。
 <br/>
 <br/>
 3. 北京维权人士美使馆听课受阻
 <br/>
 <br/>
 10月10号，北京维权人士葛志慧等人，前往美国驻华大使馆听教育知识讲座，在大使馆门口遭到中共警察阻截，被带回麦子店派出所听候发落。警察声称，因为中美贸易战，上级要求阻止民众去听讲座。
 <br/>
 <br/>
 4. 平度访民围观
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 维权 遭拘留
 <br/>
 <br/>
 山东省平度市访民刘兰香，因围观平度老兵维权，并给熟识的老兵带去食物和水果，10月8号，被警方行政拘留，罪名是〝寻釁滋事〞。
</div>
