### 【禁闻】中共黑客猖獗 惹来神秘组织狩猎
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】近期，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对外国政府和企业的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑客.html" target="_blank">
  黑客
 </a>
 攻势愈发猖獗，而美国一个名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入侵真相.html" target="_blank">
  入侵真相
 </a>
 〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神秘组织.html" target="_blank">
  神秘组织
 </a>
 自去年5月以来，不断揭露有中共政府背景的网络黑客姓名及其团队的情况。
 <br/>
 <br/>
 这个名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入侵真相.html" target="_blank">
  入侵真相
 </a>
 〞的组织，专注于挖掘与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 有关的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑客.html" target="_blank">
  黑客
 </a>
 。它揭发中共黑客的平台，包括推特和在WORDPRESS、MEDIUM上的博客，运营至今已将近一年半。
 <br/>
 <br/>
 与一般网络安全公司不公开与政府有关的黑客身份做法不同，〝入侵真相〞不仅直接点出黑客名字，贴出照片，找出隐藏在他们背后的机构。 全球网络安全专家也对〝入侵真相〞能挖到黑客们乘坐优步（UBER）的收据，感到惊叹。
 <br/>
 <br/>
 〝入侵真相〞将自己定位为APT猎手。APT特指出于商业或政治动机的网络间谍活动。
 <br/>
 <br/>
 去年5月，〝入侵真相〞曝光了一个APT3黑客团队，指出这个团队就是中国广东博御信息技术有限公司，为中共国安部服务。其中两名黑客的名字也被公布出来。
 <br/>
 <br/>
 半年后，美国联邦检察官起诉APT3团队的吴颖卓、董浩以及夏磊（Lei Xia），共谋侵犯西门子等公司。
 <br/>
 <br/>
 APT3最终解体。
 <br/>
 <br/>
 今年8月以来，〝入侵真相〞又陆续揭发另一个有中共政府背景的黑客组织APT10，并公布了其中三个黑客姓名，他们参与了2016年对MSP公司全球数千客户的攻击。最新被曝光的APT10黑客叫做安志强 。
 <br/>
 <br/>
 〝入侵真相〞认为，天津国安局通过两家幌子公司——天津华盈海泰科技发展有限公司和Laoying Baichaun设备公司操控APT10。
 <br/>
 <br/>
 而这个APT10在全球早已恶名昭彰。
 <br/>
 <br/>
 去年4月，由英国国家网络安全中心（NCSC）等单位公布的报告指出，APT10正入侵全球企业并窃取机密资讯。
 <br/>
 <br/>
 今年10月，美国国土安全部（DHS）发出技术警报，再次点名中共政府支持的APT10。
 <br/>
 <br/>
 互联网观察人士古河：〝中共的这些黑客目中无人，可以肆意的侵入其他个人组织的电脑、进行偷窃，他们觉得自己很牛，但是没有想到他们自己到处行走的足迹，被海外这些反中共黑客的组织全程的纪录下来，甚至纪录的非常详细。这个信息的公开是对中共黑客非常震撼的心理打击。〞
 <br/>
 <br/>
 互联网观察人士古河认为，〝入侵真相〞的出现，是由于整个海外大形势的转变。过去十几年间中共黑客组织发展迅猛，到处破坏和窃取西方国家的高科技、国防信息。这个反中共黑客组织的出现，是海外围剿中共黑客的一个显著特征。
 <br/>
 <br/>
 澳大利亚华裔张小刚博士：〝很多民间的技术精英自发的组织起来去对付中共这种对世界互联网全面的黑客行动，这是非常好的事情。当然前面也有，包括很多群体对中共的网络封锁进行突破，发展各种翻墙软件。现在对中共的黑客进行揭发、跟踪，把他们的底细挖出来，这个是另外的一种做法，可以说是国际上民众对中共的一种反击。〞
 <br/>
 <br/>
 澳大利亚华裔张小刚博士表示，中共四处出击，引起国际社会各个阶层的反弹。〝入侵真相〞的出现，能够遏制、反制中共对外的黑客攻击，同时也起到对中共在全球扩张链、甚至是对中共政权存在的打击。
 <br/>
 <br/>
 〝入侵真相〞今年8月曾表示，它的目标是让中共黑客对他们的非法网上活动三思而行。不过，该组织身处何方，有何背景，目前没人知道。网络专家界据说掀起了一股猜测〝入侵真相〞真实身份的热潮。
 <br/>
 <br/>
 《华尔街日报》报导，约翰霍普金斯大学教授瑞德（Thomas Rid）说，没有几个人拥有挖掘黑客身份的语言技能、工具和研究能力。〝他必定是一个专业人士。一个知道自己在做什么的人。〞
 <br/>
 <br/>
 而据网络媒体〝母板〞（Motherboard）报导，有成员自称这个组织是善于隐身的〝黑骑士〞。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/钟元
