### 习坐拥3千海外伏兵？揭秘中共军情史上最大〝潜伏案〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】中共元老叶剑英的次子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叶选宁.html" target="_blank">
  叶选宁
 </a>
 ，曾被中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太子党.html" target="_blank">
  太子党
 </a>
 阶层以及北京权贵圈内，公认为是〝太子党的老大哥〞。港媒披露，叶选宁在任职期间最大的动作，以留学方式
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_输送.html" target="_blank">
  输送
 </a>
 3000太子党成员到海外，为今后太子党执政储备人力，埋下伏兵。
 <br/>
 <br/>
 1976年叶剑英策动了一场政变，抓捕了以中共前党魁毛泽东的妻子江青为首的〝四人帮〞，最终把前中共中央军委主席邓小平扶上台。
 <br/>
 <br/>
 之后广东叶家树大根深，在中共军界及政界人脉极广。
 <br/>
 <br/>
 叶剑英的次子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叶选宁.html" target="_blank">
  叶选宁
 </a>
 生于1938年10月。据知情者介绍，叶选宁是个神秘人物，在民间知其者较少，但在中共高层军界和商界十分活跃，而且人缘关系广泛。他1968年毕业于北京工业学院，曾做过中共空军作战部副部长、全国政协委员。
 <br/>
 <br/>
 据知情者说，叶选宁的大哥、曾任广东省省长的叶选平有什么难题，还要经常向他讨主意。叶选平曾问他对上调北京有何主意，他回答：〝要当官上中央，要做人留地方。〞叶选平当上中共全国政协副主席后，一度坚持留在广东。
 <br/>
 <br/>
 由于叶选宁曾担任中共总政联络部部长，也被外界称为总政联络部的掌门人。
 <br/>
 <br/>
 在上世纪80年代，在中共最高层，最主要的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_太子党.html" target="_blank">
  太子党
 </a>
 〞圈子以叶剑英之子叶选宁、胡耀邦之子胡德平、邓小平之子邓朴方、王震之子王军、陈云之子陈元、陶铸之女陶斯亮、杨尚昆之子杨绍明为首。
 <br/>
 <br/>
 在众多中共高干子女之间，特别是邓小平、陈云、叶剑英、王震等几大中共元老家族的后代们，相互之间矛盾重重。而能够在他们之间起到调合作用和内部凝聚作用者，便是叶选宁。
 <br/>
 <br/>
 港媒披露，叶选宁在中共总政联络部任职期间的最大手笔动作，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_输送.html" target="_blank">
  输送
 </a>
 3000太子党成员到海外留学。这一行动从1990年叶选宁执掌中共总政联络部开始便陆续展开，一直持续到他1997年退役仍然没有结束。
 <br/>
 <br/>
 由于持续时间之长、涉及国家之广、派遣人数之多都史无前例，因而也被视为是中共军情史上最大规模的〝潜伏案〞。
 <br/>
 <br/>
 叶选宁由于不喜欢当时执政的江泽民，他的想法是：不希望太子党们在当时政坛上过早冒头，因为时机还不成熟；应该把大批太子党生力军转往海外，同时也远离国内环境，保存实力，以免过早曝光，昙花一现，影响未来真正〝太子党时代〞的执政大计。
 <br/>
 <br/>
 有报导说，从〝潜伏计划〞实施开始，到如今已经过去多年了。其中约一半人已经返回中国，还有一半人则留在了当地，他们或者找到工作机会，甚至打入了当地国家政府或军方机构、敏感研究所等。
 <br/>
 <br/>
 但是这些人都按照当初中共总政联络部的部署，低调生活着，避免出头露面，即使回到中国也远离仕途官场。
 <br/>
 <br/>
 2010年10月，中共十七届五中全会召开，全会以批准习近平接任中共中央军委副主席，并确立了习近平在中共十八大上的总书记接班人地位。
 <br/>
 <br/>
 据说，当时叶选宁就相当看好习近平，对他的执政寄予了厚望。所以，习上台后，他已经把手下的3000名伏兵尽数交给习近平，由习来统筹指挥。这些人无论是在中国，还是在海外，都会成为习近平最可靠的支持者，他们将会以各种不同方式，全力帮助稳固未来的习政权。
 <br/>
 <br/>
 不过，多年来，欧美多个国家都曾发布，有关中共当局利用海外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_留学生.html" target="_blank">
  留学生
 </a>
 ，从事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍.html" target="_blank">
  间谍
 </a>
 活动的报告。对于叶选宁输送的这部分太子党是否也有从事间谍活动这一计划，引起外界的考量。
 <br/>
 <br/>
