### 台媒：华为事件冲击网通厂 中方5G发展恐受挫
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在加拿大被捕后，不仅舆论界一片哗然，产业界也受到不小震荡。有台媒表示，华为集团如果被查实违规而遭美国政府制裁，势必对其他与华为有业务往来的厂商产生影响，华为自身的竞争力将遭到削弱，其5G 商转进程也将受到挫折。
 <br/>
 <br/>
 据台湾中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 涉嫌违反美国政府对伊朗制裁的相关法律规定与伊朗交易而被捕后，国际市场开始担心该事件恐将对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5G商转.html" target="_blank">
  5G商转
 </a>
 进度造成影响。
 <br/>
 <br/>
 报导表示，目前全球各大电信公司正积极进行5G商转规划。其中美国电信商Verizon、及AT&amp;T预定今年年底5G商用化，韩国3大电信公司规划明年3月推出5G服务，日本电信商则规划在2020年7月东京奥运会上展示5G商用，中国大陆也正加速5G商转的时程，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 则是中国大陆发展5G技术的带头企业。
 <br/>
 <br/>
 该报导并引述投顾的分析称，如果美国政府掌握了华为违法的确凿证据，恐会比照中兴通讯的先例，对华为进行管制处罚。
 <br/>
 <br/>
 届时，华为向美国进口高阶网通处理器的通道将受到限制，诸如路由器(router)的运算及传输晶片、PA（功率放大器）或高速传输用的雷射与光通讯零组件等等，都会受到影响，华为的5G产品推出的进程恐将因此而受挫。
 <br/>
 <br/>
 有统计数据显示，华为基地台设备全球市占率约30%至35%，是全球最大供应商之一，不过近几年该公司的产品和服务的安全性受到严重怀疑，美国、加拿大、澳洲、新西兰、英国等国都已先后点名表示不欢迎电信营运商采用华为设备。
 <br/>
 <br/>
 针对华为一旦真的出事将对台湾的关联厂商造成什么影响的问题，台湾投顾分析称，台厂主要是上游厂商受到的牵连影响较大，例如台厂中华星光来自于华为的营收贡献度超过2成，其影响程度会比较大。不过，下游包括交换器(switch)、路由器等终端产品，华为则并非台厂的主力客户，占营收比重多在10%以下，所以即使华为遭到美国制裁，对台湾相关产业链上的厂商影响程度也有限。
 <br/>
 <br/>
 （记者李明报导/责任编辑：戴明）
