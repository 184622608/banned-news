### 王全璋妻请多国大使出席王案庭审
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709.html" target="_blank">
  709
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_律师.html" target="_blank">
  律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 与英、美等驻大陆的外交人员会面，促请各国大使出席王全璋案的审讯。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709.html" target="_blank">
  709
 </a>
 大抓捕事件中被捕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_律师.html" target="_blank">
  律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 ，11月6日通过推特发布照片，称她于5日上午与英国、美国、德国、加拿大和欧盟等驻华人权官员见面，交流了王全璋案的情况，并递交了致各国大使的信件。李文足在信中促请各国大使旁听王全璋案开庭。
 <br/>
 <br/>
 李文足在此次写给诸国驻华大使的信中表示，王全璋被捕已经三年四个多月了，中共官方一直阻止家属聘请的律师会见。〝目前，此案已经严重超出规定办理期限。而最新的消息是：现在官批律师已经会见了王全璋，估计快开庭了。〞因此她希望大使们能向中方提出旁听王全璋案开庭的明确要求。
 <br/>
 <br/>
 王全璋的妻子李文足在接受自由亚洲电台采访时表示，各国一直在关注支持王全璋，在联合国人权理事会普遍审查的会议当中，美国和英国都明确提出了王全璋案，要求释放王全璋。
