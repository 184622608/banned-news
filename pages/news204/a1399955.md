### 内部激辩公开？朱鎔基旧部罕见抨击北京贸战策略
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】中美贸易战令北京陷入困境，中共内部就如何应对一直在激烈争论，但在公开言论中一般比较含蓄。而中共前总理朱鎔基的旧部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_龙永图.html" target="_blank">
  龙永图
 </a>
 ，日前在财新网论坛上公开批评北京用政治手段处理贸易问题，并直指对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 加征关税〝太草率〞。
 <br/>
 <br/>
 11月18日，在中国大陆金融媒体财新网举行的第九届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财新峰会.html" target="_blank">
  财新峰会
 </a>
 上，曾任中方入世（加入世贸组织）谈判首席代表的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_龙永图.html" target="_blank">
  龙永图
 </a>
 表示，贸易谈判要就事论事，谈关税就谈关税，如果北京一直将政治与贸易谈判掺和在一起，那中美永远都无法达成协议。
 <br/>
 <br/>
 他还说，用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 反击川普（特朗普）政府的关税措施是不明智的。因为农产品在贸易中是非常敏感的，打击农产品本应该是最后才考虑的办法。但是北京从贸易战一开始就选择了农产品。
 <br/>
 <br/>
 他指问，中国迫切需要大豆进口，为什么一开始就选择大豆？这是经过〝深思熟虑〞的吗？
 <br/>
 <br/>
 目前北京正在积极展开对美谈判，寻求贸易战〝和解〞。龙永图此时对北京贸易战策略提出直接批评，引起外界关注。
 <br/>
 <br/>
 龙永图是朱鎔基旧部，曾在朱掌管国务院期间升任外经贸部副部长，并在朱主管入世谈判时被委任为首席谈判代表。龙永图至今仍在中共对外组织和高级智库中担任要职，也是国企中国人寿的非执行董事。
 <br/>
 <br/>
 而主办此次峰会的财新传媒，被指和王岐山关系密切，曾是中纪委反腐的〝风向标〞。而王岐山也是出身于国务院系统。
 <br/>
 <br/>
 其实，中共内部对北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战对策.html" target="_blank">
  贸易战对策
 </a>
 的批评一直存在，但此前都没有象龙永图这样公开和直接。
 <br/>
 <br/>
 从年初美国酝酿贸易战时，中国专家就一再告诫，中国市场对美国大豆需求太大，不能用来作为贸易战工具。但北京出于政治考量，在7月6日的首轮关税战中就将美国农产品列入加税清单，其目的是打击川普〝票仓〞，以迫使川普停止贸易战。
 <br/>
 <br/>
 不过北京此举的结果是激怒了川普，后者在推特上批评北京〝手段卑劣〞，并警告对中方将〝不会再客气〞。随后，川普政府划拨120亿美元补贴美国农民，并开始为美国农产品积极开拓新的国际市场。
 <br/>
 <br/>
 11月初的美国中期选举结果显示，中方打击美国农产品，并没有让川普和共和党失去美国农业州的支持。但是北京方面，此举造成的大豆巨大缺口让中国市场陷入困境。
 <br/>
 <br/>
 除了针对美国农产品外，北京打〝朝鲜牌〞和〝联欧抗美〞，也被指是激化贸易战的政治手段。当时分析人士指，北京下出这些〝臭棋〞，是主动将贸易问题〝政治化〞，〝逼迫〞美国政府将中美对抗升级。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
