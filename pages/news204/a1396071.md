### 中共〝王牌〞战机云南坠毁   怪罪俄制发动机
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共空军.html" target="_blank">
  中共空军
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_歼-10战机.html" target="_blank">
  歼-10战机
 </a>
 再出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠机.html" target="_blank">
  坠机
 </a>
 事故。一架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_歼-10S.html" target="_blank">
  歼-10S
 </a>
 战机在中国南部地区进行飞行训练时，意外坠毁，已知有2人死亡。
 <br/>
 <br/>
 周四（18日）晚间，中国航空摄影师〝DS大彪〞在微博发帖证实，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共空军.html" target="_blank">
  中共空军
 </a>
 一架军机坠毁，型号疑似是一架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_歼-10S.html" target="_blank">
  歼-10S
 </a>
 双座型战机。
 <br/>
 <br/>
 网络流传出的照片显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠机.html" target="_blank">
  坠机
 </a>
 现场留有一片带有中共解放军〝八一〞军徽的军机残骸，其中的一块残骸旁还有一块带有血迹的、疑似尸块的物体。

 网上一段24秒的视频可见，事故现场聚集大量民众围观，飞机残骸火势未完全熄灭，并冒出黑烟。
 <br/>
 <center>
  <br/>
  <br/>
 </center>
 <br/>
 总部位于北京的《多维网》引述知情人士的话称，军机坠毁地点是中共空军在广东湛江的遂溪基地，主要负责中国南方地区的防务工作。
 <br/>
 <br/>
 歼-10S是中共在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_歼-10战机.html" target="_blank">
  歼-10战机
 </a>
 基础上，改进而成的双座型战机，主要用于飞行员培训，但也保留了作战功能。在过去数年中，歼-10系列战机，包括歼-10A、歼-10B、歼-10S等，都有坠毁的记录。
 <br/>
 <br/>
 中共官方曾在2015年12月公告一次歼-10S战机坠毁事故。当时，隶属中共海军东海舰队的这架歼-10S，因撞鸟而失事。而其它的坠机事故则多被指，是战机的俄制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_AL-31发动机.html" target="_blank">
  AL-31发动机
 </a>
 出现故障。
 <br/>
 <br/>
 中共官方媒体〝新浪军事〞去年12月发表文章称，〝AL-31发动机〞主轴的润滑和冷却系统经常失灵，进而可能导致轴承摩擦面变得凹凸不平，摩擦力急剧加大，旋转不稳甚至根本无法旋转，而使战机因失去动力而坠毁。
 <br/>
 <br/>
 但截至目前，尚不确定此次歼-10S战机坠毁，是否与AL-31发动机有关。
 <br/>
 <br/>
