### 【微视频】中共开始变种人类 台湾捍卫传统
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】下面请看赵培为我们带来的微视频。
 <br/>
 <br/>
 赵培：2018年11月24号，中华民国九合一选举落幕，咱们微视频没有评论结果，这是因为咱们确实不懂，甚至连候选人都没有认全，不敢妄加评论。这次选举最吸引我的是附带的公投，其中一个是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_同性恋.html" target="_blank">
  同性恋
 </a>
 婚姻和教育议题，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 选民中有67.4%反对同性恋婚姻合法化，66%反对同性恋教育，这个公投结果说明台湾人捍卫了传统道德。
 <br/>
 <br/>
 2017年台湾法官解释宪法说禁止同性恋结婚违宪。婚姻不是宪法定义的，这是一个传统，早于宪法的产生。咱们先说中华传统中的婚姻，三皇之一的〝人皇〞伏羲与女娲诚心求问上天，如果上天准许他们结为夫妻，那么就让他们各自头上的烟雾汇聚一处；如果上天不允许，就让烟雾消散。上天显吉兆让烟雾聚在一起，他们这才结为夫妻。直到现在中国婚姻仪式中还有拜天地的敬神环节，婚姻就是男女结合，就与阴阳对应上了。如果没有传统，每个名词要从新定义，人类文明推倒重来了。
 <br/>
 <br/>
 西方的传统不是这样的由来，但是形式也是男女结合受到神的祝福。可能有朋友要问了，有没有不是男女结合的，反正没有流传下来。这句话反过来看就很可怕了，可能出现过搞同性恋婚姻的族群，但是都灭绝了。这次同性恋婚姻的推手是共产主义、社会主义，俄国〝十月革命〞以后搞了〝性解放〞，废除禁止同性恋的法律，苏俄当时被鼓吹同性恋运动者认为是世界上同性性关系方面〝最开明〞的国家。不光是共产国家，自由国家里面也是这些人在搞这个事情，时间有限，咱们不多说了，具体的请看《魔鬼在统治我们的世界》这本书。
 <br/>
 <br/>
 共产主义目的是破坏家庭，让大家放弃传统道德。那么结果是什么呢？我们刚才说了，没有搞同性恋的族群从古代存续下来了，这才是可怕的后果。台湾人守护住了中华传统文明，特别是婚姻的形式，做的好。
 <br/>
 <br/>
 大陆的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 虽然也不允许同性恋婚姻，但是它的目的不是维护传统，首先是大陆百姓跟台湾一样不认同；其次是历史上中共要求多生孩子备战，同性恋不生孩子没有人上战场了；最后是现在计画生育造成了人口下降的压力，中共都鼓励二胎了，怎么可能让同性恋合法。但是，共产党在破坏家庭方面一样在做，比如说文革中让儿子批斗老子，夫妻互相揭发等等，破坏传统更加厉害。
 <br/>
 <br/>
 中共现在已经不是从毁坏社会秩序和道德来毁坏人类了，甚至是要从基因角度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_变异人类.html" target="_blank">
  变异人类
 </a>
 。11月26号，中共人民网说，深圳南方科技大学制造的基因编辑婴儿11月出生了。共产党说，震惊了科技界，这不是震惊呀，这是惊吓呀。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_编辑人类基因.html" target="_blank">
  编辑人类基因
 </a>
 不是新技术，但是涉及到人类种族生存，没有国家敢搞。这种东西不能被称为人了，这就是妖怪呀，是要被孙悟空打死的。中国大陆科学家也出来抵制这种反人类行为了，已经有122人签名了。联署抵制未必有用，真正解决问题是退党，收了中共这个妖孽。
