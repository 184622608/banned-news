### 惊人巧合 孟晚舟与张首晟同日出事 同替华为效力？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】6日有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 之女孟晚舟被捕，及在美华裔科学家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张首晟.html" target="_blank">
  张首晟
 </a>
 跳楼身亡的消息同时曝出。更巧合的是，两人同日出事，而且正值〝川习会〞举行的同一天。张首晟还被曝出曾与华为接洽研发芯片，从而引起外界各种猜测。
 <br/>
 <br/>
 12月6日，美媒报导，美国华裔科学家、斯坦福大学物理系终身教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张首晟.html" target="_blank">
  张首晟
 </a>
 ，于12月1日跳楼身亡。这位美国顶尖的华裔物理学家，所在团队因在2007年发现〝量子自旋霍尔效应〞，被《科学》评为当年的〝全球十大重要科学突破〞之一。
 <br/>
 <br/>
 张首晟于2009年获邀加入中共的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞，受聘于清华大学。此后，中国多家企业都与张首晟有过接触，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司。
 <br/>
 <br/>
 公开报导显示，2017年4月1日，正在深圳参加IT领袖峰会的张首晟，在峰会期间被华为终端公司董事长余承东专程接走。
 <br/>
 <br/>
 同年张首晟获得中共国家科学技术奖励，中共七常委中至少习近平、李克强、王沪宁等现身颁奖现场。
 <br/>
 <br/>
 巧合的是，张首晟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_跳楼自杀.html" target="_blank">
  跳楼自杀
 </a>
 当天，12月1日，华为首席财务官、副董事长孟晚舟，在加拿大被捕，她因涉嫌违反美国制裁伊朗的规定，美国方面要求引渡她。
 <br/>
 <br/>
 而且，当日正值〝川习会〞在阿根廷登场，在中美贸易战升级的背景下，与芯片相关的事件同时爆发，于是大陆网路上各种〝阴谋〞论泛起。
 <br/>
 <br/>
 数日前，美国公布的一份关于指责中共窃取美国知识产权报告里，张首晟创办的丹华资本被点名，同样引发讨论。而美国FBI严查参加中共〝千人计划〞的华裔科学家也与张首晟之死联系到一起。
 <br/>
 <br/>
 一份非机密FBI备忘录中说，中共的〝千人计划〞，通过经济间谍活动和盗窃知识产权，对美国企业和大学构成严重威胁。而有中共军方背景的华为，多年来一直被西方情报机构指控受制于中共政府，存在间谍和安全隐患。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
