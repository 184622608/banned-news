### 谁取消了中美安全对话？蓬佩奥王毅当面爆争执
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】10月8日，美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 在北京会晤中共外长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 ，双方言辞针锋相对，场面尴尬。新媒消息，双方争执的话题不只是美国副总统彭斯痛批中共的演讲，还包括第二论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美外交安全对话.html" target="_blank">
  中美外交安全对话
 </a>
 搁浅的责任问题。
 <br/>
 <br/>
 据新加坡《联合早报》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 当面要求美国停止指责中共的〝错误言行〞。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 则回应称，在这些问题上，美方和中方存在〝根本分歧〞。
 <br/>
 <br/>
 蓬佩奥还说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美外交安全对话.html" target="_blank">
  中美外交安全对话
 </a>
 为双方讨论长远问题提供了重要机会，他对中方取消对话表示遗憾。
 <br/>
 <br/>
 王毅随后回应称，安全对话不是中方取消的，〝我讲的是一个事实〞。
 <br/>
 <br/>
 在此之前，中美双方就谁取消了安全对话，已经有过一次隔空口水战。
 <br/>
 <br/>
 蓬佩奥和美国防部长马蒂斯本计划10月中旬访华参加中美外交安全对话。但两家外媒9月底引述美国官员透露，中方以〝没空〞为由，取消了这场对话。
 <br/>
 <br/>
 随后，中共官媒特地发表一篇外交部〝答记者问〞，声称是美方提议推迟了对话。之后，中共外交部重申该说法，并痛批美国〝歪曲事实、别有用心〞。
 <br/>
 <br/>
 这次对话取消的背景，是美国不断升级中美贸易战，并制裁中共军头、批准对台军售、派B-52轰炸机和导弹驱逐舰直闯南海等。而北京也推出了一些报复措施，包括禁止美国军舰访问香港、召回正在访美的海军司令沈金龙。
 <br/>
 <br/>
 而叫停第二轮中美外交安全对话，也被认为中方反击措施之一。
 <br/>
 <br/>
 此前，中共一直用〝拒绝贸易谈判〞来表明自己的〝强硬〞立场，这次突然降低调子，将叫停安全对话的〝主动权〞让给美方，显得比较罕见。
 <br/>
 <br/>
 港媒分析，中美近来军事互访频繁，说明双方都不希望贸易战演变成军事冲突。
 <br/>
 <br/>
 因此有分析指，中方不承认自己取消安全对话，或为造成双方军事紧张推卸责任。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
