### 获习王刘连番接见 基辛格建议北京放弃〝旧制度〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】日前北京又搬出〝美国老朋友〞来缓解中美关系，但这些人的表态都有些〝变调〞。美国前财长保尔森暗喻中共是魔鬼，前国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基辛格.html" target="_blank">
  基辛格
 </a>
 在接连与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 会面的过程中，建议北京〝超越自己的旧制度〞。
 <br/>
 <br/>
 11月6日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基辛格.html" target="_blank">
  基辛格
 </a>
 在新加坡的彭博新经济论坛上，见到了中共国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 。8日，基辛格到访北京，获得北京最高领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 亲自接见。10日，王岐山再次会晤基辛格。11日，中共国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 也会见了基辛格。
 <br/>
 <br/>
 在几次接见中，中方领导人都强调中美合作的〝重要性〞，并展示了希望与美国〝磋商和解〞的意愿。不过，习近平也强调要美国〝尊重中国选择的发展道路〞。
 <br/>
 <br/>
 在中共官媒报导中，基辛格也表态〝希望中美友好〞。不过，外媒披露基辛格6日在新加坡经济论坛上的讲话，似乎并不是那么太〝友好〞。
 <br/>
 <br/>
 自由亚洲9日评论文章称，基辛格那次演讲很含蓄，但有三点表达地很清楚：第一，北京需要超越自己的旧制度，才能真正引领亚洲；第二，美中关系正在从合作转为对抗；第三，美中双方都应该知道冲突的严重后果。
 <br/>
 <br/>
 法广11日评论文章指，第一点其实非常尖锐，熟读托克维尔的王岐山应该知道〝旧制度〞的含义。后面两点表面上是指〝美中两国〞，其实恐怕只是说给北京听的。
 <br/>
 <br/>
 另外，美国前财长保尔森在彭博创新经济论坛的讲话，也因其中的美中〝经济铁幕〞之说被广为流传。
 <br/>
 <br/>
 保尔森在演讲中称，中共在加入世贸组织17年之后，依然没有实现真正的对外开放市场，表现在对外国投资者进行合资和股权比例限制、以及各种非关税壁垒，强迫外企转移技术，将外国技术〝加工成中国技术〞等等。
 <br/>
 <br/>
 保尔森明确指出，这些问题是在华美企的共同抱怨。他甚至形像地将这些被迫对中共让步的美企比作浮士德。
 <br/>
 <br/>
 自由亚洲文章指，在西方文学作品中，浮士德为了利益将灵魂出卖给魔鬼，但是在出卖灵魂后也很不开心。保尔森这里明显是将中共政府比作魔鬼。
 <br/>
 <br/>
 保尔森还指出，美国对中共态度变得强硬，部分原因是中方〝对外资开放不足〞。他还说，虽然美国两党在其它所有问题上看法都不同，但对北京的看法高度一致。
 <br/>
 <br/>
 他呼吁北京〝大胆〞开放经济，引入外企竞争，施行市场经济，并保护知识产权。他并警告称，如果中方不迅速行动，对美中经济切割的呼声可能越来越强，〝经济铁幕〞就有可能降临。
 <br/>
 <br/>
 基辛格曾经与毛泽东、邓小平、江泽民、胡锦涛、习近平等五位北京最高领导人有过〝亲密交往〞；而保尔森也曾获得江泽民、胡锦涛、习近平等三位的接见，更与朱熔基、王岐山、周小川等中方经济事务负责人过从甚密。
 <br/>
 <br/>
 而这两个中方的〝多年老朋友〞，都或委婉、或直接地批评北京，呼吁中方〝大胆改变〞，也凸显了抵制中共已是美国朝野的共识。
 <br/>
 <br/>
 此前有消息指，基辛格自从川普总统上任之初，就屡次向他建议〝联俄抗共〞。
 <br/>
 <br/>
 自由亚洲文章指，中美对抗不断升级，中方的大量〝美国老朋友〞保持了沉默。
 <br/>
 <br/>
 法广文章则称，在行事果断的川普面前，北京依靠〝老朋友〞的那一套传统运作已经过时，如果真想改善中美关系，北京必须实现诺言，真正地实行市场经济。
 <br/>
 <br/>
 不过，外界普遍认为，北京如果真正转型经济制度，等于从根本上动摇其统治基础。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：云涛）
