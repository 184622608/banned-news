### 揭秘中共常委退休生活：护照被收 不准出国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】欧美总统和首相卸任后，依旧活跃在世界政治舞台上，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共领导人.html" target="_blank">
  中共领导人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退休.html" target="_blank">
  退休
 </a>
 后鲜有机会走
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出国.html" target="_blank">
  出国
 </a>
 门。有美媒指，这些人有许多秘密，不被允许踏出国门。亦有消息指出，当初
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阻止.html" target="_blank">
  阻止
 </a>
 出国，因其涉及一个难以告人的原因。
 <br/>
 <br/>
 美国《外交政策》曾引述霍普金斯大学中国问题专家主任兰普顿（David Lampton）说：〝在中国，前国家领导人几乎不
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出国.html" target="_blank">
  出国
 </a>
 。〞
 <br/>
 <br/>
 与中共高层关系密切的人士透露，如果没有获得现政治局常委的允许，前政治局常委不可以出国，这是不成文的规定。
 <br/>
 <br/>
 新加坡国立大学东亚研究所资深研究员薄智跃说，前中共中央政治局委员根本没有个人护照，他们只有官方护照，在中央办公厅。薄智跃说：〝这些人有许多的秘密，他们知道信息可能对中共的形象不利。〞
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 与谷开来》一书披露，在几年前，习近平当局准备公开审理薄熙来后，周永康曾向习近平、王岐山申请出国养病，并保证按期回国。
 <br/>
 <br/>
 但王岐山说，此举〝恐怕多有不便〞，因为法轮功学员已在澳大利亚、加拿大、美国等多个国家起诉他，出国就可能面临被抓捕。
 <br/>
 <br/>
 他说，〝你知道西方国家实行司法独立，政府不能管法官判案。你一旦到了那些国家，又没有外交豁免，法院给你送一张传票你接还是不接？法庭去还是不去？这不是很大的麻烦吗？〞
 <br/>
 <br/>
 听到王岐山的一席话后，周永康张口结舌，无言以对，只得打消了出国的念头。最终在国内国外都无法逃脱被审判。
 <br/>
 <br/>
 去年12月，有美媒刊发评论说，周永康在石油系统任职时，曾数次以商务身份造访美国，花了大把国库的银子享尽人间奢华，但等到他成为中共政权的〝正国级〞领导人之后，美国政府的大门对他永久关闭了。
 <br/>
 <br/>
 而周永康、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 、李长春等之流的中共江派党官，都因积极主导并配合中共江泽民集团迫害法轮功，导致成千上万的大陆法轮功学员被迫害致死，甚至活摘器官，而被追查迫害法轮功国际组织通告追查。
 <br/>
 <br/>
 2014年6月份，刘云山曾以所谓〝党际交流〞形式，出访欧洲四国，10天的访问的行程，从芬兰、爱尔兰、葡萄牙，一路走麦城。均遭到法轮功学员〝法办刘云山〞的呼声追着跑，最后体力不支，招架不住，取消多个行程，狼狈回国。此后再也没听说过他冒险出访欧美的消息。
 <br/>
 <br/>
 另据动向披露，贾庆林2015年7月申请今秋到东南亚旅游，并探望老朋友，但被当局拒绝。据称，贾被通知：暂时不宜外出旅游，原因一是没有先例及规则，二要考虑环境安全。
 <br/>
 <br/>
 根据明慧网报导，早在2004年9月，贾庆林在西班牙访问期间，便遭遇到当地法轮功学员向西班牙马德里国家法院提交诉状，控告贾庆林对法轮功学员犯下的群体灭绝和酷刑罪。
 <br/>
 <br/>
 报导披露，从2002年迄今，被法轮功学员在国际上刑事控告或民事起诉的中共高官有江泽民、前中共政法委书记罗干、前中共国家副主席曾庆红、前中共副总理李岚清、前政治局常委吴官正、原文化部部长孙家正、原北京市市长刘淇等30人。
 <br/>
 <br/>
 江泽民更先后在全球30个城市和地区被法轮功学员以〝群体灭绝罪、反人类罪、酷刑罪〞刑事控告或民事起诉，2009年，西班牙国家法庭以群体灭绝罪及酷刑罪起诉江泽民等五名迫害法轮功的主要责任人：罗干、薄熙来、贾庆林、吴官正，并对江泽民下全球通缉令。
 <br/>
 <br/>
 明慧网统计，从2015年至今，已有超过20万的法轮功学员及其家属向大陆最高检、最高法控告中共前党魁江泽民迫害法轮功，要求把这个首恶绳之以法。至今全球31个国家、超过260万民众支持起诉江泽民。
 <br/>
 <br/>
 法轮大法（又名法轮功）由李洪志大师自1992年5月13日在长春传出，目前洪传世界114个国家和地区，法轮功学员以〝真、善、忍〞的理念修心养性。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：赵云）
