### 中共财政部巨虎被捕 传贪腐千亿拥148情妇
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财政部.html" target="_blank">
  财政部
 </a>
 原
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副部长.html" target="_blank">
  副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张少春.html" target="_blank">
  张少春
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贪腐.html" target="_blank">
  贪腐
 </a>
 案调查终结，日前最高检察院以涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受贿罪.html" target="_blank">
  受贿罪
 </a>
 ，对其作出逮捕决定。据爆料，张少春国内外房产有130多套，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_情妇.html" target="_blank">
  情妇
 </a>
 148人，被查出的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赃款.html" target="_blank">
  赃款
 </a>
 高达近千亿人民币。
 <br/>
 <br/>
 10月19日，中共新华社报导称，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财政部.html" target="_blank">
  财政部
 </a>
 原党组副书记、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副部长.html" target="_blank">
  副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张少春.html" target="_blank">
  张少春
 </a>
 涉嫌受贿一案，由中共国家监察委员会调查终结，移送检察机关审查起诉。日前中共最高检察院以涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受贿罪.html" target="_blank">
  受贿罪
 </a>
 ，对张少春作出逮捕决定。该案正在进一步办理中。
 <br/>
 <br/>
 资料显示，张少春生于1958年，辽宁人，1994年，担任中共财政部办公厅副主任。2003年9月，担任中共政部教科文司司长。2003年，担任财政部部长助理。2006年，升任财政部副部长。2013年4月担任财政部副部长、党组副书记。
 <br/>
 <br/>
 2018年5月7日，中纪委官网通报，张少春涉嫌严重违纪违法，正接受中纪委与中共国家监委纪律审查和监察调查。
 <br/>
 <br/>
 张少春落马的前一个月已被免职。4月4日，中共国务院免去了张少春财政部副部长的职务。当时保留了其全国人大或政协职务，也没有公布其他去向，这被视为张少春可能〝出事〞的一个信号。
 <br/>
 <br/>
 被免职之前，张少春在中共财政部副部长中排名第一，工作分工是协助部长负责全面工作，分管税政司、预算司、国防司。
 <br/>
 <br/>
 张少春是继长期在财政部任职之后，调任国家统计局局长的王保安和中纪委驻财政部纪检组组长莫建成之后，财政部落马的第三个高官。但官方没有公布张少春因何原因落马。
 <br/>
 <br/>
 9月19日，网上曝光的一张截图显示，经查，张少春家中搜出美元9000万，欧元8475万，英镑5230，澳元7800万等，各类银行卡1000多张总值100多亿人民币，地下车库十多辆单价500万人民币以上豪华轿车，珠宝首饰堆积如山，20年以上茅台酒1000多瓶。
 <br/>
 <br/>
 张少春在国内外房产130多套，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_情妇.html" target="_blank">
  情妇
 </a>
 148人，被查出的资产总值600多亿人民币，境外存款300多亿人民币，其儿子儿媳侄子等名下房产70余栋，总值50多亿人民币。
 <br/>
 <br/>
 不过上述消息并没得到官方证实。
 <br/>
 <br/>
 9月20日，中纪委官网通报称，张少春违反4大纪律，包括政治纪律、组织纪律、廉洁纪律、生活纪律；违规打探有关案情，对抗组织审查；违规出入私人会所、接受公款宴请；在职务调整等方面为他人谋取利益，收受财物；搞钱色交易，收受礼品、礼金；为他人谋取利益并收受巨额财物涉嫌受贿犯罪。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：赵云）
</div>
