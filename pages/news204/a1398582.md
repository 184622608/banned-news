### 〝步态识别〞始作俑者曝光 涉香港中联办港人担忧
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】继〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人脸辨识.html" target="_blank">
  人脸辨识
 </a>
 〞监控系统之后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 最近再出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_步态识别.html" target="_blank">
  步态识别
 </a>
 〞监控阴招，用于监视国民。而该技术公司的创办人之一，竟是香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 副主任谭铁牛。消息引发港人担忧这项监控技术会被引入香港。
 <br/>
 <br/>
 据香港《苹果日报》报导，创办〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_步态识别.html" target="_blank">
  步态识别
 </a>
 〞（gait recognition）智能系统的银河水滴公司，是由中科院扶植成立的，而这项技术的首个牵头人，是香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 副主任谭铁牛。谭亦是公司创办人之一。
 <br/>
 <br/>
 官方资料显示，谭铁牛曾在中科院工作，多年来研究虹膜识别等。2016年6月，谭铁牛创办银河水滴公司，同年12月空降香港中联办，担任副主任，负责统战及教科文领域职务。而中联办网页仅列出谭的资历，未提及其在商业公司身分。但公司官微不时转载有关谭的消息，称谭为公司〝重量级顾问〞。
 <br/>
 <br/>
 港媒报导说，外界质疑谭铁牛亦官亦商，更担忧〝步态识别〞这种阴招会被引入香港。
 <br/>
 <br/>
 时事评论员刘锐绍指，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 党、政、商合一现实下，〝运作不透明〞，〝令人担心中共政商同体，经济、技术被寡头垄断，又无监督，令高科技都变成监控人民的工具〞。
 <br/>
 <br/>
 <h4>
  〝步态识别〞
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维稳.html" target="_blank">
   维稳
  </a>
  技术  已被应用于北京上海
 </h4>
 <br/>
 据陆媒报导，银河水滴公司最近推出被命名为〝水滴神鉴〞的步态识别产品，以人的步态特征为注册对象、通过步态识别技术在海量的视频中，快速进行人物检索和身分识别，并迅速实现目标人物的查找，从而使检索速度提高了6倍。
 <br/>
 <br/>
 另外，该技术号称可以在距离50米远时就开始辨析人物，就算是背影、侧身、人脸被遮住、光线昏暗等，都不会影响系统判断；对一小时视频的检索时间仅需要10分钟，而且精确度高达94%。
 <br/>
 <br/>
 银河水滴公司称，公司已与12个省市签署1.35亿人民币的定单。资料显示，今年9月该公司曾到新疆展示产品；新疆近日因〝再教育营〞而备受国际社会关注。
 <br/>
 <br/>
 美联社11月5日报导，中共公安部门已开始在北京和上海的街道上，使用这种步态识别监视技术进行维稳；新疆则准备引入。
 <br/>
 <br/>
 银河水滴CEO黄永祯称，这项技术毋须当事人配合，即使目标人物跛行、驼背也能被识别，因其是通过分析目标人物的身体所有特征达到识别目的。
 <br/>
 <br/>
 时事评论员石述思认为，生物识别技术用于维稳不可避免成为一个趋势，〝这是一个非常荒诞的场景，人民掏钱装上探头，监控自己。〞
 <br/>
 <br/>
 实际上，步态识别系统并非新技术，但目前除了中国，尚未有其它国家将它运用在监控民众上。
 <br/>
 <br/>
 美国《华尔街日报》曾撰文批评中共运用监视系统控制民众。报导指出，美国运用这种技术是为了辨别、找出嫌疑犯，而中共却拿来监控国民。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：秦锋）
