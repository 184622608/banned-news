### 幕后黑手在操纵？陆媒报导范冰冰现〝奇葩〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】大陆女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 因卷入逃漏税事件，被中共当局追缴逃漏税款超过8.8亿元人民币。海外媒体纷纷从不同角度对事件进行了跟踪报导。不过，大陆数家媒体却被网友发现了又一〝奇葩〞现象。
 <br/>
 <br/>
 中共官媒〝新华网〞10月3日公布对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃税.html" target="_blank">
  逃税
 </a>
 案的审查结果，海内外数十家媒体进行了相关报导。但有网友惊奇地发现，大陆各家媒体对艺人的报导，竟然也须统一口径，甚至连标点也不做更改。
 <br/>
 <br/>
 从网络流传的一张图片可见，多家中共党媒都发出〝订阅号〞，包括新华网、人民网、澎湃新闻、中国基金报、中国青年网等等，但这些新闻的内容竟〝完全一致〞，全部转载了新华社的通稿。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-7/p9068651a941439406.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-7/p9068651a941439406-ss.jpg"/>
   <br/>
   <font size="-1">
    大陆各家媒体对范冰冰的报导，竟然惊人地一致。（〝日人民报〞脸书截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 脸书专页〝日人民报〞讽刺说，〝美国有1万个记者，每天能发布出万条不同新闻，但中国有1万个记者，每天只能发布出同一条新闻，甚至连标点符号都是一样的！〞
 <br/>
 <br/>
 许多网友则跟贴留言：〝也太统一了吧〞、〝媒体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姓党.html" target="_blank">
  姓党
 </a>
 〞、〝连新闻都系抄人地〞、〝厉害了！我的玻璃、我的墙！〞
 <br/>
 <br/>
 还有网友调侃道：〝大陆的记者很好做〞、〝这些媒体抄袭不怕被告啊，连标点都不改一下〞。也有网友表示，幕后似乎有一双无形的〝黑手〞在操控。
 <br/>
 <br/>
