### 被中共关押律师余文生 获德法人权法治奖
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】被中共以涉嫌〝煽动颠覆国家政权罪〞和〝妨害公务罪〞逮捕的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆维权律师.html" target="_blank">
  大陆维权律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">
  余文生
 </a>
 ，日前获得2018年度〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德法人权法治奖.html" target="_blank">
  德法人权法治奖
 </a>
 〞。维权人士认为，余文生获奖实至名归。
 <br/>
 <br/>
 11月21号，德、法两国外长宣布中国709案辩护律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">
  余文生
 </a>
 、与其他国家的14名人权捍卫者，共同获得2018年度〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德法人权法治奖.html" target="_blank">
  德法人权法治奖
 </a>
 〞。这个奖项的宗旨是表彰那些在自己国家做出突出贡献的，或是在国际层面推进和保护人权、推动法治规则的人权人士。
 <br/>
 <br/>
 2018年是〝德法人权法治奖〞第三届颁奖，2016年〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709事件.html" target="_blank">
  709事件
 </a>
 〞维权律师李和平的妻子王峭岭，曾经获得首届〝德法人权法治奖〞。
 <br/>
 <br/>
 余文生的妻子许艳：〝我也是在网上看到11月21号余文生律师获得‘德法人权法治奖’，我做为他的妻子为他高兴，也非常感谢给他颁奖的所有人。我在这10个多月在为他维权的过程中，去了很多部门申请监督和写材料，或现场去交材料，所有的努力几乎都没得到回复，看到法制，这让我觉得很无助。〞
 <br/>
 <br/>
 余文生的妻子许艳说，如今余文生案被民主国家关注、肯定，对她本人来说也是一种鼓励。
 <br/>
 <br/>
 她还透露，余文生案经过多次延期后，19号再次被退回公安局补充侦查。
 <br/>
 <br/>
 湖南维权人士陈思明说 ，余文生获奖实至名归，希望民主国家继续关注余文生案件，让余文生早日获释。
 <br/>
 <br/>
 湖南维权人士陈思明：〝余文生律师他为中国的人权保障付出了很多的努力，他得到这个奖是理所当然的，当之无愧的，感谢法国和德国政府为中国大陆的人权给予了道义上的支持，我很感谢他们，同时我也认为这个奖，对于我们大陆维权人士，人权捍卫者是一个很大的鼓舞，我非常高兴。〞
 <br/>
 <br/>
 余文生是北京人权律师，长期代理宗教迫害及其他人权案件；2014年因为声援香港雨伞运动被拘留近百天。2015年709大抓捕后，余文生曾经为多位涉及〝709事件〞被关押的律师辩护。1月19号遭警方抓捕，被控涉嫌〝煽动颠覆国家政权罪〞和〝妨害公务罪〞而遭逮捕。
 <br/>
 <br/>
 新唐人记者常春、元真采访报导
