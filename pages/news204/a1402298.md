### 孟晚舟被捕事出有因 华为绝密文件被美截获
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 司法部发言人证实，中国通讯巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 副董事长孟晚舟在温哥华被捕，美国要求引渡她。有港媒称，孟晚舟被捕事出有因，美国〝盯上〞华为已经有很长时间，华为的一份绝密文件2016年被美截获，涉及规避美国的出口管制。
 <br/>
 <br/>
 孟晚舟是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的女儿，是华为未来接班人。12月6日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 媒体称，孟晚舟是在加拿大转机时，被加国政府代表美国政府暂时扣留，美国指控孟晚舟涉嫌违反对伊朗的贸易制裁，正在寻求引渡她回美国受审。
 <br/>
 <br/>
 据悉，引渡孟晚舟到美国受审，是源于1971年12月美国与加拿大在华府签订的《美加国际引渡条约》，条约规定美加两国可将其境内被对方指控或判定犯罪的人，应对方请求，移交该国审判或处罚。
 <br/>
 <br/>
 针对孟晚舟被抓事件，加拿大总理杜鲁多（Justin Trudeau）表示，政府在司法机构行动数天前就已知道消息，他称事件没有任何政治因素。
 <br/>
 <br/>
 华为董事、公共及政府事务部总裁陈黎芳称，孟晚舟面临纽约东区未指明的指控，华为目前得到的信息非常少。
 <br/>
 <br/>
 香港01报导说，美国商务部早于2016年，就公布了一份据称是中兴通讯的绝密文件，该文件详细介绍如何利用空壳公司中转等方式，规避美国的出口管制。
 <br/>
 <br/>
 文件披露，一家代号F7的中国公司也有类似做法，而F7的多个关键特征，如试图收购美企3leaf等，都和华为公司一致。这份落款时间为2011年8月的文件，遭到华府反对。
 <br/>
 <br/>
 据报〝F7〞是中兴对华为的内部称呼，是〝夫妻〞的谐音，而华为称中兴为〝26〞是〝二流〞的谐音，意指中兴是不入流的公司。
 <br/>
 <br/>
 路透社报导称，有档案显示，华为在伊朗的一家主要合作伙伴2010年年底提出，向伊朗最大的移动运营商伊朗移动通信公司（MCI）出售至少价值130万欧元、遭禁运的惠普计算机设备。 华为因此受到美国众议院情报委员会的指责。
 <br/>
 <br/>
 档案显示，在提议向伊朗移动通信公司出售产品的建议书中，至少有13页标有〝华为机密〞的字样，并带有该公的标志。华为在一份声明中称，这是一份〝招标档〞，华为在〝当地的主要合作伙伴之一〞Skycom Tech Co Ltd已将这些档提交给MCI。
 <br/>
 <br/>
 2016年底美国商务部和财政部，向华为发出行政传票，调查华为是否违反美国贸易禁令，向叙利亚、伊朗、朝鲜等流氓国家提供美国技术。2017年，美国司法部辖下的联邦调查局FBI以及美国财政部，进一步对华为展开相关刑事调查。
 <br/>
 <br/>
 2018年4月，中兴通讯公司因违规向伊朗出口产品被美国制裁。随后，美国采取了一系列行动，阻止或减少华为和中兴通讯公司进入美国市场，因为这两家公司被指控，利用其技术在美国进行间谍活动。
 <br/>
 <br/>
 4月19日，美国国会美中经济与安全审查委员会发布的报告称，中共可能支持某些中国企业进行商业间谍活动，以提高中企竞争力并促进政府利益。
 <br/>
 <br/>
 报告中点名：中兴、华为、联想、浪潮、京东方、中国科学院、中国电子科技集团（CECT）、北京华胜天成等19家公司，均属于国有控股或具备国防背景。
 <br/>
 <br/>
 美国国防部5月宣布，基于安全考虑，已要求全球美军基地停止销售华为、中兴通讯制造的手机。
 <br/>
 <br/>
 美国政府9月起开始努力劝说德国、意大利和日本等盟国官员及电信高管，弃用华为的电讯设备。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
