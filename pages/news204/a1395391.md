 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_粉黛草.html" target="_blank">
  粉黛草
 </a>

 <br/>
 <br/>
 对于记者询问〝是否知道不能跨越围栏〞？大多数人都推说：〝拍一张就走。〞还有一些人甚至回呛：〝关你什么事？〞

 <br/>
 据花田管理员郑阿姨表示，粉黛草是前年由澳洲进口的，由于它们一压就倒，而且容易长虫，这2年来，她和丈夫像养孩子一样，细心栽种。
 <br/>
 <br/>
 近日，粉黛草盛开，粉红色花海的照片被上传网络后，吸引大批人来拍照，场面完全失控。郑阿姨夫妇叫得声嘶力竭，仍制止不了拍照民众毁坏花田。不到3天时间，花田就被游客压毁得〝肢离破碎〞。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 郑阿姨的丈夫气到生病，她也忍不住哭诉，这些人就像〝蝗虫一样涌入〞。她痛斥一些游客的不文明行为：〝拍照片的人，拍出来的照片很美，可是行为却很难看！〞
 <br/>
 <br/>
 类似粉黛草花田被游客毁坏的新闻，屡屡见诸报端。去年5月，四川凉山州的高山杜鹃漫山盛放，被誉为〝中国最美杜鹃花海〞，吸引大批游客涌入观花。不过，一些不文明游客不仅折花枝拍照，还将花整株挖走；更甚者，还有人用花枝点火烤肉，现场惨不忍睹。
 <br/>
 <br/>
 对于大陆民众的这些不文明举动，有分析认为，中国人的素质真的那么差吗？其实并非历来如此。现今国人的素质低下是共产党数十年如一日破坏传统文化的结果。
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产主义的终极目的.html" target="_blank">
  共产主义的终极目的
 </a>
 》一书中阐述：中国以礼义之邦著称。先秦典籍〝三礼〞（《周礼》《仪礼》《礼记》）里详细记载了中国古人极为丰富优美的礼仪，也探讨了礼的天道根源和礼背后的哲理。这种礼的精神一直延续到中共建政之前。有人也许还记得1949年前受教育的老年人多么彬彬有礼。可悲的是，中共统治几十年后，这一切彻底被毁了。
 <br/>
 <br/>
 中共建政后对几代中国人灌输歪曲的历史，用无神论、斗争哲学洗脑，造成了如今很多中国人对传统文化一无所知，也由此带来传统价值在人心中颠覆，道德一片空白。
 <br/>
 <br/>
 中共破坏道德、让人言谈举止随心所欲、没有规矩、行为低下，把很大面积的人群变成劣等的人。
 <br/>
 <br/>
 纽约华人李静明说，有人说
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人素质.html" target="_blank">
  中国人素质
 </a>
 比较低下，不讲道德，没有诚信。但是很多人都不知道这是为什么？看了《共产主义的终极目的》这个连载文章之后才知道，中国人因为从小被灌输了党文化，根本不知道自己的行为不文明。
 <br/>
 <br/>
 李静明表示：〝这个连载文章值得每一个中国人都看一看，特别是值得每个年轻人都看一看。它能指导中国人找回那些传统。〞
 <br/>
