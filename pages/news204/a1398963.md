### 郭伯雄胞弟躲过一劫？神秘消失一年后身份曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】原中共军委副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 的胞弟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯权.html" target="_blank">
  郭伯权
 </a>
 ，在公众视野中消失超过一年后，近日以陕西省政府参事室巡视员的新职务露面。分析认为，郭伯权的平调抑或为官方的调虎离山计，秋后算账也并非无先例。
 <br/>
 <br/>
 11月9日，陕西省政府参事室发布消息：11月2日下午，陕西省委常委、常务副省长梁桂一行到省政府参事室（省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文史研究馆.html" target="_blank">
  文史研究馆
 </a>
 ）调研，省政府参事室党组副书记、馆长徐晔，巡视员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯权.html" target="_blank">
  郭伯权
 </a>
 ，党组成员、副主任张剑及副处级以上官员参加了座谈会。
 <br/>
 <br/>
 这一消息显示，此前担任陕西省政府参事室党组副书记、省文史馆馆长的郭伯权，自去年卸任后，现已改参事室巡视员。
 <br/>
 <br/>
 公开履历显示，郭伯权是陕西礼泉人，出生于1961年10月。曾在陕西省礼泉县任职近20年，曾任礼泉县副县长等职。2002年他离开礼泉县，任陕西省彬县县长。后任中共陕西省江河水库管理局（省渭河流域管理局）党委书记，中共陕西省人防办副主任，中共渭南市委常委、副市长等职。
 <br/>
 <br/>
 2013年2月，郭伯权调任陕西省民政厅厅长，成为正厅级官员。
 <br/>
 <br/>
 郭伯权是中共军委原副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 的弟弟，在郭伯雄2015年4月9日被查后，同年9月30日，郭伯权被免去陕西省民政厅厅长。此后，他又被免去陕西省老龄工作委员会办公室主任职务。
 <br/>
 <br/>
 2015年11月3日，郭伯权首次以中共陕西省政府参事室（省文史研究馆）党组副书记、省文史馆馆长的身份参加活动。但2017年他就卸任该职务，陕西官方消息披露，徐晔接替郭伯权，担任陕西省文史馆馆长一职。
 <br/>
 <br/>
 此后，郭伯权在公众视野消失超过一年，至本月才以被降级为陕西省文史馆巡视员的身份露面。
 <br/>
 <br/>
 郭伯权的哥哥郭伯雄是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 一手提拔起来的军中亲信，与另一名中共原军委副主席徐才厚联手，替江泽民把持军权，架空了时任军委主席胡锦涛的军权10年之久。
 <br/>
 <br/>
 中共十八大后，习近平对军队进行了强力反腐〝打虎〞，徐才厚于2014年3月落马，2015年3月死亡；郭伯雄于2015年落马，2016年7月被判终身监禁。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
