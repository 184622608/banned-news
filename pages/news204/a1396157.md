### 揭秘：〝六四〞前赵紫阳一句话 江泽民吓呆了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】当年，前中共总书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵紫阳.html" target="_blank">
  赵紫阳
 </a>
 因同情〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四.html" target="_blank">
  六四
 </a>
 〞学生，反对武力
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_镇压.html" target="_blank">
  镇压
 </a>
 被迫下台。〝六四〞前夕，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 因整肃《世界经济导报》，导致事态扩大，赵对江把小事化大而引发了大规模示威非常不满，言辞之厉将江吓得六神无主。江因此怀恨在心。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵紫阳.html" target="_blank">
   赵紫阳
  </a>
  秘密录音评
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
   江泽民
  </a>
  ：此人无大志
 </b>
 <br/>
 <br/>
 赵紫阳的老部下杜导正，于2010年初在港台出书《赵紫阳还说过什么？——杜导正日记》，该书记录了他与赵紫阳的谈话内容，披露了赵紫阳许多不为人知的秘密。
 <br/>
 <br/>
 书中记载，2000年6月22日，在谈到江泽民时，赵紫阳表示，他个人认为，江泽民的〝三个代表〞是要堵住政治体制改革的嘴，为一党专政制造理论根据。他还表示：〝此人（指江泽民）无大志。〞
 <br/>
 <br/>
 2014年9月11日，网上还曝出赵紫阳〝教训〞江泽民的图片。图片显示，赵紫阳正对江泽民说话，江泽民则身体大幅度前倾向着赵紫阳，一副毕恭毕敬、洗耳恭听、卑躬屈膝的奴才相。
 <br/>
 <br/>
 对此，网民嘲讽说，肢体语言是生动的、毫不掩饰的，在这张图片面前，文字语言是多余的，苍白的。网民嘲讽江泽民：〝正是‘王莽谦恭未篡时’。〞
 <br/>
 <br/>
 赵紫阳曾担任过四川省委书记、国务院总理和中共中央总书记。1989年因在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四.html" target="_blank">
  六四
 </a>
 事件中反对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_镇压.html" target="_blank">
  镇压
 </a>
 、反对戒严而被撤职，随后被软禁在北京富强胡同6号，直至2005年去世，终年85岁。软禁期间，赵紫阳的言行受到江泽民非常严格的控制。
 <br/>
 <br/>
 <b>
  江泽民封《导报》六四
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_屠城.html" target="_blank">
   屠城
  </a>
 </b>
 <br/>
 <br/>
 据《江泽民其人》第五章：封导报六四屠城 捡便宜儿皇进京（1989──1990）一文记载：
 <br/>
 <br/>
 江泽民及其亲信对于导报的粗暴处理引发了一场席卷上海乃至全国新闻界的抗议。上海市委整顿《导报》引发的风暴来临了。第二天上海街头就发生了大规模游行，公开打出了〝还我导报〞和要求恢复钦本立职务以及言论自由的旗帜和横幅。
 <br/>
 <br/>
 上海作协部分名人纷纷参加游行，北京知识界和新闻界的著名人士致电江泽民，要求收回对钦本立及《导报》的处理决定。
 <br/>
 <br/>
 在市政府门口席地而坐的学生们不时呼喊口号。一些群众说：〝学生的口号我是赞成的。现在实际上最大的失误是不推进民主进程。〞有的说：〝学生的爱国热情应该予以肯定。〞还有的说：〝这不像是动乱！〞
 <br/>
 <br/>
 据报导，当时在外滩的大学生约有八千余人。这是这次学潮中上海学生游行规模最大的一次。到晚上十时零五分，聚集在市政府门前的学生开始陆续散去。
 <br/>
 <br/>
 江泽民害怕了。对于整肃《导报》引发的抗议声浪，江泽民承认，〝后果比我们预料的要严重得多。〞有人指责他的行为引发了〝上海大规模的示威〞。事实上不止是引发了〝上海大规模的示威〞，而且促发了北京的大规模示威。
 <br/>
 <br/>
 在北京，两名记者把有1,013名首都新闻工作者签名的请愿书送交中华全国新闻工作者协会，请愿书要求与中央主管宣传工作的领导人对话。
 <br/>
 <br/>
 《中国青年报》的学校教育部兼科技部主任李大同在递交请愿书时向在场的中外记者宣布，请愿书的1,013名签名者分别来自《人民日报》、《新华社》、《经济日报》、《中国青年报》、《北京日报》、《北京晚报》等三十多家首都新闻单位。
 <br/>
 <br/>
 这份写给全国记协书记处的请愿书说，根据赵紫阳5月4日接见外宾时关于对话的谈话精神，有必要与中央主管新闻工作的领导人就新闻界发生的不正常的事件对话。
 <br/>
 <br/>
 请愿书列举了三项对话内容，其中的第一条就是关于引起海内外强烈反响的上海《世界经济导报》总编辑钦本立停职。报社实行总编负责制的说法与事实不符，这恰恰是新闻改革要解决的首要问题。
 <br/>
 <br/>
 4月27日晚，江泽民在惶恐中打电话给原中共中央组织部常务副部长、当时的中顾委委员李锐，通话达四十余分钟，江在电话中恳请李锐向北京有些朋友通融，又在电话里向李锐探询北京情况。江在电话里还以〝受不了啦〞的口气向李锐表示当时的心情。
 <br/>
 <br/>
 <b>
  赵紫阳表态：江泽民吓呆了
 </b>
 <br/>
 <br/>
 4月30日，中共总书记赵紫阳访朝归来，当晚江泽民与曾庆红飞赴北京，欲向赵紫阳汇报工作。赵很快接见他，江汇报完后问赵：〝你对我在上海处理《导报》怎么看？〞赵并未即时表态，反问江泽民：〝你看呢？〞
 <br/>
 <br/>
 江泽民支吾其词，他发现和赵紫阳隔膜已深。赵紫阳看了一眼江泽民，接着说：〝现在没有时间谈这个问题。〞
 <br/>
 <br/>
 江泽民用恳求的语气说：〝紫阳同志若不拿出意见，我和庆红同志就不好工作，也不好回上海交代。〞
 <br/>
 <br/>
 赵紫阳只好表态了：〝上海市委行事仓促地处理了《世界经济导报》的问题，把小事化大，才让自己步入了死胡同。〞说完扭身便走了。据当时在场的人士透露，江呆呆地望着赵离去的身影足足有十分钟没有说出一句话来。
 <br/>
 <br/>
 显然，赵紫阳对江泽民把小事化大致使引发了大规模示威的做法非常不满，言辞之厉让江泽民吓得六神无主。江的密友陈至立说：〝如果中央追究责任，就由我一人来承担好了，绝不牵扯你。〞从此江泽民和这个女人的关系更加亲密了。
 <br/>
 <br/>
 事后，江泽民虽然安心一点，但还是到处找关系，希望知道党内大老们是什么态度。他得到的反馈是中央意见分歧，赵紫阳的话不代表中央精神。
 <br/>
 <br/>
 5月13日，600名主要来自北京大学的学生开始在天安门广场进行绝食抗议。更多的学生市民来声援，各国记者渐渐把镜头和注意力对准这里，并纷纷指责上海市委书记江泽民破坏民主。
 <br/>
 <br/>
 在上海，4,000名学生聚集在市委门前声援北京绝食的学生们，并要求市委书记表态。江泽民对中央精神心里有了底，当然不肯露面。
 <br/>
 <br/>
 这引发了学生的极大愤慨。为了避免事态再扩大，江不得不去看望住院学生，不过这仅仅是权宜之计，这并不影响几天之后江对北京实施戒严表示坚决支持。
 <br/>
 <br/>
