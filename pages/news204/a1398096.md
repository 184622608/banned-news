### 北京给川普〝拉〞票？上海进博会时间诡异
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】习近平11月5日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海.html" target="_blank">
  上海
 </a>
 举行的首届中国国际进口博览会（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 ）上，暗批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 实施保护主义，而中共的大门会越开越大。外界分析，进博会开幕距离美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 不到48个小时，时间点巧妙。在此敏感时刻，北京当局在这种国际性会议上暗批川普，用意显然是不希望川普方面在这次中期选举中获胜。
 <br/>
 <br/>
 11月15日，中国国家主席习近平在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 上致辞时，不点名的批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 实施保护主义，强调中共开放的大门不会关闭，只会越开越大。
 <br/>
 <br/>
 彭博社5日报导，习近平演讲中虽然没有点名川普（特朗普）或美国，但是贸易争端无疑是习讲话鲜明的主题。目前中共正面临着川普与其他国家对中共削减4,230亿美元贸易顺差的压力。
 <br/>
 <br/>
 《华尔街日报》报导说，习近平承诺将进一步扩大对外开放，加大资本市场的改革力度。这表明中共将继续通过加大改革开放等措施来抵抗经济下滑压力，并缓解国际贸易摩擦。
 <br/>
 <br/>
 报导说，今年以来，中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 不断升级，加大了中国经济下滑压力，中共为此通过降税、降准、扩大投资，并加快改革开放力度、扶持科技企业等方式，以提振经济。
 <br/>
 <br/>
 美国消费者新闻与商业频道（CNBC）说，自川普当选以来，习近平第三次站到世界舞台，将中共形塑为国际开放市场与加强合作的拥护者。而习的讲话距离美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 不到48个小时，全世界都在注视着川普政府是否能通过选举获得政策推动。
 <br/>
 <br/>
 报导说，在中美贸易战不断升级的背景下，美国的中期选举也将不可避免将华盛顿的对外关系牵扯在内。
 <br/>
 <br/>
 美国中期选举将于美东时间11月6日登场，不仅要改选众议院所有435名议员和参议院33名议员，还要改选36个州的州长以及其它大小地方选举。
 <br/>
 <br/>
 中期选举的时间点，正好是川普上任约2年，因此中期选举也被视为是川普总统的〝期中考试〞，用来检测人民对总统施政的满意度。
 <br/>
 <br/>
 同时，中期选举距离2020年总统大选仅有2年光景，因此这次选举也普遍被认为是下届总统大选的风向标。
 <br/>
 <br/>
 因此，此次的中期选举，被外界视为是选择美国的未来、选择世界的未来，并将影响中共政权存亡的大选。
 <br/>
 <br/>
 在此种背景下，美国此次的期中选举，受到中共高层特别紧密的关注。
 <br/>
 <br/>
 台湾《中央社》4日引述一名与中共外交智库关系密切的人士的话透露，北京对历次美国中期选举的关切程度，〝从来没有像这次这么高〞。
 <br/>
 <br/>
 上述人士披露，中共驻美国大使馆及驻纽约、芝加哥、旧金山、洛杉矶、休士顿5个中共总领事馆，正按日搜集、汇整选情，并向大使馆回报，最后向北京呈报。
 <br/>
 <br/>
 而各个中共总领事馆，透过美国各地由中共直接成立，或立场亲北京的华人，或在当地社团、学术单位、侨社、媒体等，将搜集选情的触角向下延伸。
 <br/>
 <br/>
 搜集内容包括，媒体公开的报导、民调数据，以及各自的社群网路探听得来的消息。
 <br/>
 <br/>
 报导说，截至10月底为止，北京对美国中期选举的评估，与美国主流看法一致，即民主党有机会从共和党手中取得众议院多数席次，但参议院仍将由共和党掌握。
 <br/>
 <br/>
 这名人士说，中共在美国的搜集选情方式，基本不会引起美方特别注意。因为搜集选情掌握驻在国政情发展，本来就是各国领馆的工作，它国的使馆也在做。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉)
