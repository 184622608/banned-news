### 北京通报孟宏伟落马   国际刑警面临一大难题
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】中共公安部副部长、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 失踪数天后，北京当局7日宣布他已
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 。有评论指，一直打击各国刑事罪犯的国际刑警，正首次面临尴尬处境。
 <br/>
 <br/>
 中纪委官网10月7日通报，中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 因为〝涉嫌违法〞，目前正接受中共监委监察调查。这是孟宏伟被法国电视台《Europe 1》10月5日率先披露失踪后，北京当局首次回应孟的去向。
 <br/>
 <br/>
 另据香港《苹果日报》此前爆料，孟宏伟可能涉嫌参与2015年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北戴河政变.html" target="_blank">
  北戴河政变
 </a>
 未遂案。
 <br/>
 <br/>
 文章说，因为案件重大而紧急，北京当局才不顾影响而紧急对兼任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席的孟宏伟问话。但报导没有提供更多细节。
 <br/>
 <br/>
 据此前报导，2015年的北戴河会议前夕，中共前党魁江泽民和江派二号人物曾庆红，联合
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 的前秘书、时任河北省委书记周本顺等人，炮制了一份《河北政情通报》，他们企图在北戴河会议期间，以这份〝政治核弹〞向习近平、王岐山发难。
 <br/>
 <br/>
 但就在北戴河会议前夕，周本顺当年7月24日正在参加由中共国务院召开的〝京津冀协同发展工作推动会议〞期间，被中纪委突然带走调查。
 <br/>
 <br/>
 有评论指，这是有史以来，国际刑警组织首次面临其主席可能带有贪腐，或违纪违规诟病就职的困局与难题，但此前，该组织却并不知情。评论讽刺说，一个调查遍及所有国家刑事问题的组织，却对其主席人选毫不知情，这无疑是一种尴尬。
 <br/>
 <br/>
 〝中国（中共）又制造了一个国际笑话。〞法国塞尔奇-蓬多瓦兹大学（University of Cergy-Pontoise）教授、国际问题专家张伦，谈论孟宏伟失踪时对〝德国之声〞表示，孟宏伟任国际刑警组织主席期间失踪，这是国际刑警组织从未遇到的情形：主席去哪儿了？谁来签发〝红色通缉令〞？
 <br/>
 <br/>
 也有网友调侃道，〝国际刑警组织把个领头人给弄丢了！这还怎么为全世界执法啊？〞〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警主席.html" target="_blank">
  国际刑警主席
 </a>
 居然让共产党员担任，本来已是笑话。〞
 <br/>
 <br/>
 <h4>
  孟宏伟
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
   落马
  </a>
  〝不令人意外〞
 </h4>
 <br/>
 港媒报导曾指出，孟宏伟作为周永康的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余毒.html" target="_blank">
  余毒
 </a>
 〞被拿下，不令人意外。
 <br/>
 <br/>
 今年11月将满65周岁的孟宏伟，被认为是中共前政法委书记周永康的亲信。
 <br/>
 <br/>
 孟宏伟长期在中共公安部任职，周永康2003年3月出任公安部部长后，孟宏伟2004年4月升任副部长、党委委员，成为周永康的主要助手之一。
 <br/>
 <br/>
 周永康在中共十八大落马后，他的多名公安部旧部也被查，包括副部长李东生、常务副部长杨焕宁、政治部主任夏崇源；同时周的多名旧部傅政华、黄明等副部长也被调离。
 <br/>
 <br/>
 2017年12月，孟宏伟突然被免去中共国家海洋局副局长、中共海警局局长的兼职；2018年4月，他又被免去已担任14年的公安部党委委员一职。
 <br/>
 <br/>
 同时，孟宏伟近几年接连缺席中共公安部高层的内部会议。中共官场早已传出孟宏伟仕途〝很不妙〞的消息。
 <br/>
 <br/>
 另外，中共公安部自1999年以来，跟随江泽民集团不断地造谣、诬陷、攻击法轮功。作为中共公安部副部长的孟宏伟，已被海外明慧网列入迫害法轮功学员的涉案责任人之一。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
</div>
