### 重庆巴士坠江真相大白 与司机互殴女客曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】重庆万州一辆巴士失控冲落长江，夺去15条人命的惨剧真相大白。11月2日，官方公布事故
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_原因.html" target="_blank">
  原因
 </a>
 ，是因一名女乘客与司机互殴，导致车辆失控坠桥。事件原因公布后，疑似女乘客身份被曝光。
 <br/>
 <br/>
 11月2日，中国重庆官方举行新闻通气会，公布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万州巴士.html" target="_blank">
  万州巴士
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠江.html" target="_blank">
  坠江
 </a>
 事故
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_原因.html" target="_blank">
  原因
 </a>
 。重庆警方经调查得出结论，巴士司机冉某与女乘客刘某互殴行为严重危害公共安全，已触犯刑法，涉嫌犯罪。
 <br/>
 <br/>
 警方称，女乘客刘某现年48岁，是重庆万州区人，上车时身着浅蓝色牛仔衣服。事故原因公布后，疑似女乘客身份被曝光，一张流传网路的社交媒体微信截图显示，涉案女乘客刘某正面照被公布，在重庆万州一号馆开门市店，主要做窗帘生意，家住御景江城。
 <br/>
 <br/>
 警方公布的消息显示，10月28日上午10时许，万州22路巴士司机冉某驾车至长江二桥时，刘某原本打算在〝壹号家居馆站〞，但因此站不停车，引发刘某和巴士司机冉某互殴。导致巴士驶入逆向车道，撞向对面驶来的私家车后，撞断护栏堕入江中。
 <br/>
 <center>
  <br/>

  <br/>
 </center>
 <br/>
 事故原因曝光后，无数网民纷纷留言，痛斥女乘客的自私行为，让15人失去宝贵性命；也有人提出，车上乘客在二人争执扭打时，却没有出手劝阻，最终悲剧发生。
 <br/>
 <br/>
 中共喉舌《人民日报》在官方微博发表短评称：〝一段惊心动魄的视频，还原了悲剧诱因；一场无谓的纷争，拉十几人‘陪葬’。教训之惨重，让人不敢直视，却不得不叩问，乘客与司机互殴何以一再出现？〞
 <br/>
 <br/>
 封面新闻文章称，闭路电视曝光，真相终于大白，而视频还原的司乘纠纷，是能想到的最坏结果。一个失误，十数条人命。但真正杀人的，不只是这个矫枉过正的动作。回顾一下事故各个阶段，悲剧并非不可避免，查漏补缺，才是杜绝悲剧重演的当务之急。
 <br/>
 <br/>
 事实上，类似的事故，在重庆早几年已经发生多起。
 <br/>
 <br/>
 如2015年12月15日，《重庆晨报》官微称，万州22路公交车上，一位女乘客未听到广播坐过站，迁怒司机破口大骂，司机还嘴后，遭到女子暴打，并抢夺方向盘。最后公交车撞上行道树，车头受损，有乘客受伤。当时女乘客被刑拘。
 <br/>
 <br/>
 同年12月29日，22路公交车再遇乘客抢夺方向盘事件。该乘客也是由于坐过站，抢夺司机的方向盘，导致公交车撞到人行道上大树。
 <br/>
 <br/>
