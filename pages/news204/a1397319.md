### 360副总裁怒摔话筒 计算机大会〝一摔成名〞（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】日前在杭州举行的2018年中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_计算机大会.html" target="_blank">
  计算机大会
 </a>
 上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_360副总裁.html" target="_blank">
  360副总裁
 </a>
 兼首席安全官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谭晓生.html" target="_blank">
  谭晓生
 </a>
 ，由于上一环节拖堂及大会增加议程，在论坛上怒摔话筒及文件，引发舆论广泛关注。陆媒戏谑，360副总裁怒摔话筒的信息迅速传开，可以说一摔成名。
 <br/>
 <br/>
 10月30日，陆媒说，研发电脑安全软件的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_360副总裁.html" target="_blank">
  360副总裁
 </a>
 兼首席安全官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谭晓生.html" target="_blank">
  谭晓生
 </a>
 ，于10月27日，因不满上一个环节拖延时间，以及大会主办单位临时增加议程，竟然怒摔话筒和文件，还在台上宣称〝可以散会了！〞
 <br/>
 <br/>
 事发后，谭晓生对媒体说，摔话筒等一系列行为，不是什么大事，最后论坛举办的也很成功。
 <br/>
 <br/>
 现场视频可见，事件发生时，谭晓生在台上质问：〝临时加议程、前面的拖堂无人控制，现场的社委是如何控制的？〞
 <br/>
 <br/>
 沉默几秒后，他先拿起文件用力摔在地上，然而又将话筒拿起摔了出去。随后，他走下台说〝对不起，可以散会了〞，并向他邀请来的几位嘉宾表示道歉。
 <br/>
 <br/>
 陆媒戏谑，总裁的这一摔，真的是很潇洒呀。经常看电视的人都见识过，在会议上摔打砸甚至扭打在一起的镜头并不鲜见。不过，这种怒摔话筒的情形，恐怕还是第一次看到。
 <br/>
 <br/>
 报导说，总裁在论坛上怒摔话筒的信息迅速传开，可以说一摔成名。不过，大多数人对于总裁怒摔话筒的行为并不认同，这是不文明的表现。你可以拒绝参加会议，但你不可以在大庭广众面前耍脾气、使性子，这对其他参加会议的人也不礼貌、不尊重。
 <br/>
 <br/>
 大陆知名IT人士王冠雄说：他在数字工作时和谭晓生打过交道，谭脾气挺温和，看样子是被不专业的行为气坏了。也有说法，谭晓生请来了多位嘉宾对话，因为前面的环节严重超时，又赶上就餐时间，这就导致他们的对话听众寥寥。
 <br/>
 <br/>
 自媒体〝CEO来信〞说，这个大会的现场控制环节确实没做好，毕竟谭晓生主持的环节是27日上午的最后一个对话环节，如果前面的活动超时，那么临近中午12点钟，现场坚持听讲的人一定会少一些，毕竟大家都要去吃饭。
 <br/>
 <br/>
 2018年中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_计算机大会.html" target="_blank">
  计算机大会
 </a>
 ，于10月25日至27日在杭州举行，大会以〝大数据推动数字经济〞为主题，谭晓生原本是大会〝共享经济未来走向〞的主持人，同时他也是中国计算机学会的副秘书长。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
