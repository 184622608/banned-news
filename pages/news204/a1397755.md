### 北京密集撤换〝一把手〞  蔡奇不再是空头司令？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】北京市委书记蔡奇上任以来，密集的更换北京市的区级地方〝一把手〞，至今已撤换9名区委书记。有分析认为，这意味着他不再是〝空头司令〞。蔡奇上任之初相当于〝空头司令〞，几乎无人可用，在北京的三项政策接连受挫。
 <br/>
 <br/>
 据北京市委组织部10月29日发布任前公示，延庆区委副书记、区长穆鹏拟任区委书记。这是蔡奇自上任以来被调整的第10个北京市的区委书记。
 <br/>
 <br/>
 在此之前，已经调整撤换9名区委书记。仅4月初，北京市4个区几乎同时迎来新书记，分别为朝阳区区长王灏任朝阳区委书记，北京市委副秘书长于长辉任石景山区委书记，通州区区长张力兵任门头沟区委书记，顺义区区长高朋任顺义区委书记。
 <br/>
 <br/>
 蔡奇如此密集的更换北京市的区级地方〝一把手〞，引外界关注。有分析认为，这意味着他不再是〝空头司令〞。
 <br/>
 <br/>
 蔡奇是福建永安人，习近平福建、浙江的旧部，有近40年时间在福建、浙江两地为官。
 <br/>
 <br/>
 蔡奇的仕途加速起于2014年，以浙江省委常委兼常务副省长之职，进京担任新成立的中央国安委办公室副主任，一年后调职北京官场任代市长。这也是蔡奇仅有的在中共中央的工作经历。
 <br/>
 <br/>
 2017年5月，蔡奇担任北京市长一职仅半年即迅速升任〝一把手〞，并在中共十九大进入中共中央政治局，仕途再进一步。
 <br/>
 <br/>
 中共十九大后，11月18日，北京大兴区西红门镇新建村的平价租屋聚福园公寓发生火灾，包括8名儿童在内的19人葬身火海。
 <br/>
 <br/>
 随后，蔡奇开会部署〝全市安全隐患大排查大清理大整治专项行动〞，行动中，数万〝低端人口〞被驱逐，遭国内舆论强烈批评清理〝低端人口〞。接下来北京实施的煤改气、拆除天际线两项施政政策，也同样引起整个社会的强烈谴责。甚至中共内部的一些人也质疑蔡奇的能力，还有人幸灾乐祸，有人要追究蔡奇的责任。
 <br/>
 <br/>
 香港《东方日报》在评论文章分析说，蔡奇在京城的三把火之所以哑火，主要是他在北京市直系统并没有建立自己的人事班底，区县及委办局仍然沿用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭金龙.html" target="_blank">
  郭金龙
 </a>
 、王安顺时代的人马，而北京市委的主要成员亦多是空降而来，人员复杂。
 <br/>
 <br/>
 文章认为，蔡奇在北京相当于〝空头司令〞，主要是京城的人事盘根错节，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贾庆林.html" target="_blank">
  贾庆林
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘淇.html" target="_blank">
  刘淇
 </a>
 、郭金龙、王安顺在当地经营多年，已形成庞大的体系，蔡奇初来乍到，几乎无人可用。
 <br/>
 <br/>
 因此，蔡奇施政出现问题时，他们个个一卸了之，或作壁上观，甚至落井下石。
 <br/>
 <br/>
 文章认为，蔡奇的尴尬也是之江新军们的集体苦恼，虽然他们已上位，但没有自己的人马，因此无法如臂使指地运用权力，反而处处被绑架，动辄得咎。
 <br/>
 <br/>
 蔡奇是习近平的之江新军的代表人物，文章认为，对蔡奇的政治炮轰，真正攻击的是习近平。中共内部还存在一批反对习近平的政治山头，他们想看到之江新军施政出问题，进而否定过去五年的人事提拔路线。
 <br/>
 <br/>
 不过，蔡奇自主政北京以来，至今已撤换9名区委书记，分析认为，蔡奇换上来必定是自己人，这意味着他已经不再是〝空头司令〞。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明）
