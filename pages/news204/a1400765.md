### 【禁闻】中共制造假新闻 介入台湾九合一选举
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾九合一选举.html" target="_blank">
  台湾九合一选举
 </a>
 落幕，民进党遭到重挫，丢失了多个县市长席位。由于选前网络上充斥着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假新闻.html" target="_blank">
  假新闻
 </a>
 ，试图影响选情，国安人士警告，台湾选举结果是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 验收统战渗透的成绩单。评论人士也指出，在台湾，只要是大型选举，中共一定会干扰。
 <br/>
 <br/>
 台湾民进党高雄市长候选人陈其迈11号在旗山举办一场大型造势活动，主办单位宣称有3万人到场，但网络却流出一段段来历不明的影片，指主持人邱议莹哀求来参加造势的群众不要离开。
 <br/>
 <br/>
 这段影片迅速被许多媒体转载，逼得陈其迈阵营的民进党立委管碧玲在自己的脸书上传一段影片还原真相。
 <br/>
 <br/>
 而在台湾卷起一股〝韩流〞的国民党高雄市长候选人韩国瑜，也在一场直播中，拿着一张剪报留言，澄清自己没有收受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的金援。
 <br/>
 <br/>
 台湾大学国家发展研究所法学博士曾建元：〝这次台湾选举，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假新闻.html" target="_blank">
  假新闻
 </a>
 的问题非常的猛烈，除了有大量的来自中国大陆的网军之外，在台湾本地也查获有很多所谓的网红，在网络上制造不实的新闻。〞
 <br/>
 <br/>
 台湾国安局副局长陈文凡19号在面对立委质询〝中共是否介入引导台湾舆论？〞时表示，〝这是不需要怀疑的。〞而当另一名立委质询，国安局有没有掌握境外势力介入，包括资金、制造谣言、假讯息、恶意抹黑、网络霸凌等，陈文凡回答时明确表示〝有〞、〝确定有〞。
 <br/>
 <br/>
 国安人士表示，到了选举之时，中共会加大舆论战和心理战，透过网军散布假消息，指示中共媒体诋毁蔡政府政策，并要求亲共的台湾媒体配合，在部分台湾媒体为追求点阅率，未查证的情况下，以〝复制、贴上〞的方式，放大假消息在社会舆论上的威力。
 <br/>
 <br/>
 曾建元：〝现在的资讯传播得很快，台湾也没有资讯的管制，就让有心藉由这个自由的环境，来扰乱自由秩序的这些人士，有从中来操作的空间。〞
 <br/>
 <br/>
 旅美时事评论员蓝述：〝中共干扰台湾大选，或者说是九合一选举，任何的选举，你只要是在台湾，只要是大型的，中共它要插手，它要进行干扰，这个是从来都没有变过的。〞
 <br/>
 <br/>
 总统蔡英文直言，这次选战，有一股力量要我们闭嘴，台湾面临无孔不入的渗透和假讯息，〝如果我们低头，民主就会倒下去〞，她呼吁选民用选票当后盾，保护台湾的民主。
 <br/>
 <br/>
 蓝述：〝中共对自己的体制没有任何的信心，它害怕台湾民主社会，每一次的民主选举的成功都是对中共那个体制一种无言的鞭打，它都会让中国大陆的民众看到民主的好处，看到民主制度的优越性，所以中共它害怕。〞
 <br/>
 <br/>
 美国在台协会（AIT）主席莫健日前在接受台湾TVBS访问谈及台湾选举时说：〝很显然在台湾有外部势力试图改变舆论、传递不实讯息，这是危险的。〞该新闻播出后在TVBS网站消失了。
 <br/>
 <br/>
 党政人士表示，过去中共介入台湾选举采取直接方式，如1996年首度总统直选时，中共试射飞弹，但都造成反效果，现在中共台面上已不〝说三道四〞，但台面下却利用〝锐实力〞，以隐匿性、欺骗性手段，透过假消息与金钱收买的方式，干预台湾选举。
 <br/>
 <br/>
 蓝述：〝历年来它的方式不一样而已，Bloomberg News(彭博新闻)前不久报导过，今年大选前这一两个月，来自于中国大陆的对台湾的骇客攻击大幅度的上升。〞
 <br/>
 <br/>
 有知情人士透露，大陆涉台部门拿出中共认可的特定候选人名单，要求有在中国投资的台商以实际行动给予支持，除了要求〝票票入轨〞外，也要捐款给这些候选人。而中共平时也以金钱笼络、收买台湾意见领袖、地方桩脚、宫庙、亲共政党与团体和特定媒体等。
 <br/>
 <br/>
 国安人士警告，台湾选举结果就是中共验收统战渗透的成绩单。
 <br/>
 <br/>
 采访/常春 编辑/陈洁  后制/葛雷
