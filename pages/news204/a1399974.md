### 美发布史上最严技术管制通知 中共高技术寒冬来临？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】中共拒不放弃〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞等威胁西方安全的发展计划，美国则持续加强对北京的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_技术封锁.html" target="_blank">
  技术封锁
 </a>
 。日前，美国公布一份迄今最严格的技术出口管制先期通知，涵盖几乎所有科技前沿领域，令中国高科技公司陷入新一轮恐慌。
 <br/>
 <br/>
 这份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国商务部.html" target="_blank">
  美国商务部
 </a>
 工业安全署周一（11月19日）出台的通知，涉及人工智能、芯片、量子计算、机械人、面印和声纹技术等14个被视为有关国家安全和高科技前沿的关键领域。
 <br/>
 <br/>
 通知发出的目的是首先征询公众意见，征询截止时间是12月19日。之后，美国政府将根据意见评估和更新管制清单，并正式公布实施。
 <br/>
 <br/>
 美国商务部称，该清单是根据美国国会今年通过的《出口管制改革法案（Export Control Reform Act）》制定。出口管制开始实施后，所有涉及的商品和技术的出口，都要事先经过商务部的审核批准。
 <br/>
 <br/>
 该通知没有直接提及北京，但《美国之音》报导指，该措施对中国企业的影响最大，特别是中国高科技公司赖以生存的美国芯片，将面临新的封锁。
 <br/>
 <br/>
 在中国，许多政府背景的科技巨头，包括华为、中兴和海康威视等，都被指是帮助中共侵犯人权以及开展海外间谍活动的工具。其中海康威视是全球最大的监控设备生产商，其产品不但用于中国，还远销美国等150多个国家。
 <br/>
 <br/>
 早在美国商务部公布这一先期通知前，就有消息指海康威视将遭到美国出口管制，其股价从今年最高点已经暴跌超过30%。
 <br/>
 <br/>
 另外，中国福建晋华集团被指与台湾联华电子合谋，盗窃美国美光公司的动态随机存储器（DRAM）技术，近期遭美国下达禁售令，生产线迅速瘫痪，成为第二个中兴。同时，美国司法部还对晋华、联华电子以及相关涉案人提出指控，晋华还将面临最高200亿美元的巨额罚款。
 <br/>
 <br/>
 随后《华尔街日报》报导指，川普政府已经制定一套全新策略，包括利用出口管制、法律控诉等工具，用来对付中共盗取美国知识产权。
 <br/>
 <br/>
 报导指，针对一些典型案件，新策略将运用全面性的资源，协调各部门联合行动，对涉案的中国公司进行全方位打击。该策略将被用来对付那些依赖外国输入科技的所有中国企业。福建晋华就是其中一例。
 <br/>
 <br/>
 根据美国起诉书，台湾联华电子前高管陈正坤出身美国美光公司，从美光盗窃DRAM技术后，又将技术转移给福建晋华。此前中方并不掌握DRAM技术，但在晋华获取相关技术后，中共国务院将其列为〝国家经济重点〞。
 <br/>
 <br/>
 此前，外媒曝光一份中共深圳市政府派员赴美考察后写成的报告，其中指美国将在未来半年到一年内，对中共实施全方位的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_技术封锁.html" target="_blank">
  技术封锁
 </a>
 。
 <br/>
 <br/>
 中共为执行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞等计划，以争取高科技领域的霸权，正在以前所未有的努力抢夺和盗窃外国尖端技术，目标并不限于美国，欧盟和澳大利亚等都成为受害目标。
 <br/>
 <br/>
 日前，多家澳大利亚媒体调查发现，从去年开始，中共国安部指使黑客组织，对澳大利亚发动持续的网络攻击，以窃取澳大利亚公司的商业机密。澳大利亚政府高层也证实了该报导。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
