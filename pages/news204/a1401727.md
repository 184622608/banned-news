### 川习会关键内容消失？美媒曝北京不敢公布的细节
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）与中国国家主席习近平1日在工作晚宴上达成协议。但当天双方各自公布的会谈结果，内容却相差甚远。中共官媒对白宫声明所提到的90天期限和其他重要细节只字未提。此外，美媒还曝光了北京不敢公布的相关细节。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 与习近平12月1日在阿根廷二十国集团（G20）峰会期间举行会谈，会谈结束后，中共很快通过央视发布消息称，美中两位领导人达成重要共识，同意在1月1日后不会额外加征关税，同时双方将继续谈判，但没有提供有关共识的更多细节。
 <br/>
 <br/>
 《美国之音》报导说，在央视发布消息约两个小时后，川普一行登上〝空军一号〞离开布宜诺斯艾利斯时，白宫新闻发言人桑德斯发布会谈结果声明，涵盖了许多北京没有公布的细节。
 <br/>
 <br/>
 白宫声明宣布，美国同意在明年1月1日，将对价值2千亿美元的中国产品的关税税率维持在10%。此外，双方同意立即开始有关结构性改变的谈判。如果双方在未来90天内未能达成协议，10%的关税将会被提高至25%。
 <br/>
 <br/>
 白宫声明还说，中方同意从美国购买仍未商定但数量非常大的农业、能源、工业和其它产品，以减少我们两国之间的贸易不平衡。中方还同意将芬太尼列为〝管控物质〞，这意味着向美国出售芬太尼的人将依法受到中国最重的处罚。
 <br/>
 <br/>
 在白宫发出声明大约两小时后，新华社、央视等喉舌媒体仍然没有宣布或报导白宫声明所提到的90天期限和其他重要细节，也没有发声明否认白宫声明的准确性。
 <br/>
 <br/>
 而中共外交部长王毅和中共商务部副部长王受文则接连表示〝会谈成功〞。王毅说，〝中美双方就经贸问题达成共识：停止相互加征新的关税…双方工作团队将根据两国元首达成的原则共识，朝着取消所有加征关税的方向加紧磋商〞。
 <br/>
 <br/>
 王受文称，〝美国政府对中国2000亿美元产品征收10%的加征关税原定在明年1月1号提高到25%，现在已经决定明年1月1号仍然维持在10%。双方决定不再对新的产品加征新的关税，对于现在仍然加征的关税，双方朝着取消的方向，加紧谈判，达成协议。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 前，多家美媒曾报导说，中共急切希望〝川习会〞达成〝停火〞协议，不再升级贸易冲突。为此北京很可能做出更多让步，来换取美国暂缓进一步加征关税。不过，出于面子和维持政权稳定的考虑，北京越是做出让步，越不会对外发布〝川习会〞的真实结果。
 <br/>
 <br/>
 时政评论周晓辉评论说，从双方达成的协议内容看，北京在贸易问题上已不再像以往那样狂妄称〝奉陪到底〞，而是向美国示好，除了承诺大量进口美国农产品、能源、工业用品外，还愿意就美国最为关切的涉及结构性改变的〝强制技术转让、知识产权保护、非关税壁垒、网路入侵和网路盗窃〞等进行谈判，并给出了谈判完成日期，甚至同意如果谈判达不成协议，美国可以提高关税。
 <br/>
 <br/>
 一方面表明，北京在美国强力施压下，担心经济进一步下滑，市场信心进一步降低，且国内危机重重，自身已无太多办法应对美国加征关税后所产生的严重后果，而不得不进行的选择。
 <br/>
 <br/>
 另一方面也意味着，未来贸易战的主导权仍在美国手中。美国此次基于善意，仍选择再次相信北京，愿意就涉及结构性改变的议题谈判，但给出了三个月的时间表。一旦谈判无法达成一致，美国将再次启动加征关税。而这无疑给北京上了一个〝紧箍咒〞。
 <br/>
 <br/>
 中共官媒不敢公布相关细节，应该是担心中共党内强硬派的暗流藉此发难，也是担心民众从中看到中共的出尔反尔和虚弱。这也难免不给未来的谈判埋下阴影。
 <br/>
 <br/>
 另外，由于早前习近平在与川普会晤时提出的〝百日计划〞并未实现，也同样有理由对未来谈判结果抱持怀疑。不过，周晓辉也说，不管中共是否是真心回应美国的需求，美国给出的三个月限期，可能是其最后的机会了。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
