### 贸易争端 中共党媒突称〝不屑于报复〞有玄机
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党媒.html" target="_blank">
  党媒
 </a>
 日前刊文称，在中美贸易争端中，中方已〝不屑于〞以牙还牙的直接报复。学者认为，中共作出软化姿态，但不会就核心问题让步。
 <br/>
 <br/>
 中美贸易冲突，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党媒.html" target="_blank">
  党媒
 </a>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民日报.html" target="_blank">
  人民日报
 </a>
 》早前以强硬姿态，宣称《打，奉陪到底》《必定坚决予以大力度回击》《坚决有力，打疼美方》，到10月19号突然宣称中方〝已不屑于实施‘以牙还牙’的直接报复〞，而是将着重于〝释放自身潜力渡过难关〞。
 <br/>
 <br/>
 横河：这篇文章是放在《人民日报》第二版上，那么也就是说，至少代表了当前官方的部分看法。
 <br/>
 <br/>
 唐靖远：中共之前可能还有误判，认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 近期对中共的强硬做法，只是川普和他的鹰派内阁的少数意见。所以前几个月，中共还以为可以靠分化瓦解，定点打击川普票仓等小动作就能阻止川普。但彭斯讲话及美方系列立法执法动作，让中共终于认识到，美国对中共的态度发生了40多年来所未有的整体战略转向，对中共采取竞争和遏制，已经成为美国社会整体共识。在这种情况下，我觉得《人民日报》这篇文章说明中共已经意识到，所谓以牙还牙等报复动作无法吓退美国，而且事实上中共无法承担和美国彻底交恶的后果，所以它放弃这种短期战术，开始做长期打算了。
 <br/>
 <br/>
 中共作何打算？中共党媒近期连发文章，让民众做〝自力更生〞〝打持久战〞的准备。折射出中共不愿响应美国要求，做出〝结构性改变〞。
 <br/>
 <br/>
 谢田：结构性改变的意思是，中央政府放弃对国企、对出口、对汇率的控制。而这些国企也好、央企也好，还有强制购汇结汇，这实际上是给中共这个政权输血的基础。
 <br/>
 <br/>
 横河：美国提出来的这些要求，如果实施，对中国国家和中国人民都有好处。但是对中共的统治显然是没有好处的。因此我们没有任何理由相信中共会这样做。它们最可能采用的还是拖延战术。
 <br/>
 <br/>
 唐靖远：〝释放自身潜力渡过难关〞这话的潜台词，包括在不触及中共体制根本的前提下，加大一些领域开放程度，同时由于窃取技术难度加大，不得不加强自己创新的投入力度。总之中共最终目的，还是拖过川普任期，等待国际环境的变化。
 <br/>
 <br/>
 如果未来几周中共仍然拒绝改变，预计美国将按计划在2019年初，把2,000亿美元中国商品的关税税率，从10%上调到25%。
 <br/>
 <br/>
 美国媒体Axios报导，川普认为必须更大力度施压，让北京感到痛苦，才能真正达成协议。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
