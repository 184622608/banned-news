### 【禁闻】长春发生大规模绑架法轮功学员案
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】中共近期加强了迫害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 的力度，目前正在全国范围内大举抓捕法轮功学员，据了解，辽宁省已经有40多人被抓，而吉林省长春市仅10月12号一天就抓捕了20人。
 <br/>
 <br/>
 据《法轮大法明慧网》报导，10月10号，吉林省以省、市、区联署名义，发出一个所谓的〝打黑除恶〞公告。12号，长春市公安局国保联合各区国保、辖区派出所，对近20名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员，包括不修炼的家属，进行了绑架行动。不少学员家中的财物也被非法翻抄和抢走。
 <br/>
 <br/>
 流亡海外原长春法轮功学员唐风：〝汽车产业开发区的孙士英侄女王洪娜，还有绿园区、华联小区的法轮功学员杜河、王洪艳，在汽车厂居住的周秀芝，还有在南关区教古筝的老师赵秋悦，家住经济开发区的胡大桅夫妇，还有长春经济开发区的吴雅男，朝阳区的一名六十多岁的法轮功学员，现在他的名字我们还不太清楚，还有在南关区居住的赵旭，等等，将近20名法轮功学员被绑架。〞
 <br/>
 <br/>
 原长春法轮功学员唐风，因修炼遭到严重迫害，被迫流亡海外。他说，长春当地的迫害一直没有停止过。但这次的大规模绑架事发突然。
 <br/>
 <br/>
 流亡海外原吉林法轮功学员李秀云：〝我跟长春同修联系了一下，她12号被抓，听说是上下抓三拨的事。跟打黑也是有关系，但是好像跟（通过）宽带、电脑（有）联系的，所以说，别的同修都跟着受牵连。〞
 <br/>
 <br/>
 据唐风介绍，多年经商的法轮功学员穆君奎，以及他的公司员工张玉梅，在9月7号早上8点多，也被长春警察
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法绑架.html" target="_blank">
  非法绑架
 </a>
 。
 <br/>
 <br/>
 唐风：〝穆君奎我认识他已经有二十多年了，原来是长春市政府的一个厨师，他没修炼法轮功之前，也不是谁都敢惹他的，修炼法轮功之后，穆君奎变得非常和气的一个人，因为他坚持修炼法轮功，他的工作就被迫失去了。他一直就做这个窗簾生意，因为讲信用，诚实待人，不掺假，他的生意也就越来越好。〞
 <br/>
 <br/>
 根据《明慧网》报导，穆君奎家中和办公物品都被非法抄走。没有修炼法轮功的妻子，也在当天被绑架，直到晚上才放回。
 <br/>
 <br/>
 穆君奎遭到长春宽城分局新上任的国保队长何伟用拳狠打头和脸。宽城国保杨光则在连搧张玉梅十几个耳光后叫嚣：〝我叫杨光，你告去吧！〞
 <br/>
 <br/>
 张玉梅后来被劫持到长春第四看守所非法关押。
 <br/>
 <br/>
 近年来，打着所谓〝扫黑除恶〞名义对当地法轮功学员进行迫害的，远不只长春一地。今年8月，仅山东招远市就有超过20名法轮功学员被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法抓捕.html" target="_blank">
  非法抓捕
 </a>
 。
 <br/>
 <br/>
 唐风：〝中共往往它一贯的做法就是一边打压你，一边给你造谣诬陷，如果普通老百姓就觉得打击黑恶势力，当然是好事了，可是它以这个名义抓捕的却是以真善忍为标准做好人的这么一个群体，恰恰是颠倒黑白，荒唐至极了。〞
 <br/>
 <br/>
 原中共总书记江泽民领导下的中共当局，自1999年起，为了消灭法轮功，实施种种镇压措施，导致难以计数的修炼者死亡。时至今日，大规模的迫害和严重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酷刑.html" target="_blank">
  酷刑
 </a>
 情况仍然存在。
 <br/>
 <br/>
 《追查迫害法轮功国际组织》发言人汪志远：〝做（迫害法轮功）的目的，一是为了打压少部分人，来制造一个恐怖的场，恐吓全社会。第二个，因为迫害法轮功，尤其是大规模的活摘器官屠杀，这个罪恶是人类历史上从没有过的，可以直接导致中共的解体。所以中共对法轮功的迫害一直不敢松动。〞
 <br/>
 <br/>
 《明慧网》公布了此次参与长春大规模绑架作案的详细单位及人员信息。
 <br/>
 <br/>
 《追查迫害法轮功国际组织》发言人汪志远表示，追查国际必将追查到底，直至将每一个罪犯绳之以法。
 <br/>
 <br/>
 他也正告参与迫害者，今天中共利用他们来迫害法轮功，明天会拿他们的人头来平息民愤。因为迫害法轮功是群体灭绝罪、反人类罪，他们替中共迫害佛法修炼者，无论在道义上，还是现行中共法律下，都是非法，都是犯罪。
 <br/>
 <br/>
 汪志远呼吁那些参与迫害者，他们唯一的出路就是坦白交代自己的罪行，并收集他人参与迫害的罪证，向追查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际举报.html" target="_blank">
  国际举报
 </a>
 。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/李沛灵
