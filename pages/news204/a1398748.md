### 【禁闻】重庆高考政审风波背后：中共教育部违法
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】最近，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 官方媒体报导指称，政审材料是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_考生.html" target="_blank">
  考生
 </a>
 参加2019年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高考录取.html" target="_blank">
  高考录取
 </a>
 的必备材料，引发舆论哗然。虽然重庆教育考试院紧急出面回应说是记者〝误报〞。但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共教育部.html" target="_blank">
  中共教育部
 </a>
 一直推行的〝政审〞政策，被指违法。
 <br/>
 <br/>
 11月6号，《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 日报》报导，政审材料是参加重庆2019年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高考录取.html" target="_blank">
  高考录取
 </a>
 的必备材料，政审不合格者不能参加普通高校的录取。
 <br/>
 <br/>
 消息曝光后，〝政审〞一词很快引发舆论反弹。
 <br/>
 <br/>
 原中国人权律师滕彪表示，所谓〝政审〞，在中共建政后一直长期存在。
 <br/>
 <br/>
 原中国人权律师滕彪：〝尤其是在80年代之前非常普遍的，把很多人给排除在正常的生活之外，不能享受正常的权利。像‘黑五类’，历次政治运动当中它都有一些群体。〞
 <br/>
 <br/>
 中国问题专家横河说，从80年代到90年代，入学政审的政策似乎没那么重要，但并没有完全绝迹。中共从1999年迫害法轮功以后，重新加强了政审。
 <br/>
 <br/>
 中国问题专家横河：〝一方面它是改了名字了，它不叫政审，叫做思想政治品德考核，所以听上去就没有那么惊心动魄。另外一方面，没有轮到自己的时候（民众）他不关心，但这种政策每个人都是潜在的受害者，因为这是属于思想罪，它的标准可以根据当局的需要随时改变的。〞
 <br/>
 <br/>
 重庆教育考试院相关负责人8号回应媒体说：〝关于‘政审’是媒体记者写错了，我们叫思想政治的考核和现实表现的审核。〞
 <br/>
 <br/>
 但很快，有网友贴出11月2号重庆教育考试院微信公众号〝重庆招考〞发布的文章，其中有对〝政审怎样进行？政审主要有哪些内容？〞的解答。事件再度出现反转。
 <br/>
 <br/>
 9号凌晨，重庆教育考试院官网发出〝说明〞道歉，声称他们之前发布的信息和工作人员对记者的回应，把关不严，答覆不准确等。道歉说明中还说，2019年重庆市普通高校招生对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_考生.html" target="_blank">
  考生
 </a>
 本人的〝思想政治品德考核〞内容及方式〝均无变化〞。
 <br/>
 <br/>
 实际上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共教育部.html" target="_blank">
  中共教育部
 </a>
 从2000年到2018年，在《关于做好普通高等学校招生工作的通知》中，每年都有对考生〝思想政治品德考核〞的明文要求。
 <br/>
 <br/>
 原首都师范大学教育科学学院副教授李元华指出，所谓的〝思想政治品德考核〞，其实和〝政审〞是一个意思。
 <br/>
 <br/>
 原首都师范大学副教授李元华：〝按中共《宪法》还有《高等教育法》规定，公民就是有受高等教育的权利，并没有其它附加条件。教育部出台这个文件，本身就是违法行为。0029在国家《宪法》上任意加条件。〞
 <br/>
 <br/>
 滕彪：〝这种做法显然侵犯了《宪法》关于中国公民受教育权的规定。0515主要它的目的还是政治性的。〞
 <br/>
 <br/>
 根据中共教育部规定，考生〝反对四项基本原则或参加邪教组织；扰乱社会治安〞等，都属于所谓〝思想政治不合格〞。
 <br/>
 <br/>
 那么在这一规定下，哪些群体最容易受到影响呢？
 <br/>
 <br/>
 李元华：〝一个就是有信仰的，比如说像法轮功这种修炼团体，包括（中共）它认为的所谓地下教会，还有一些不认同它暴政，不屈从于它这种党化、奴化教育的任何人，它都可以用这个来限制你。〞
 <br/>
 <br/>
 滕彪表示，除了长期受中共迫害的法轮功修炼团体，访民家庭也会受到影响。
 <br/>
 <br/>
 还有观点认为，异议人士、以及近年来备受打压的维权律师，也难逃中共政审黑手。
 <br/>
 <br/>
 虽然重庆当局否认了政审一说，但滕彪认为事件反映了整个中国倒行逆施的可怕政治趋势。共产极权国家对民众的言论、思想、观点进行干涉，把民众分门别类，这在正常的民主国家是绝不可想像的。
 <br/>
 <br/>
 而李元华强调，所谓政审，就是斗争哲学的产物。一个公民不违法，就理应有受教育的权利。但中共说你思想跟它不同，就剥夺你的受教育权。这是违反普世价值的。
 <br/>
 <br/>
 采访/林澜、陈汉 编辑/王子琦 后制/陈建铭
