### 俄拟禁向中国出口木材 中共部长大惊失色
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】中国企业在俄罗斯滥砍乱伐，掠夺资源，引起俄罗斯社会抵制。日前，俄官方宣称可能暂停向中国出口木材。俄媒披露，俄方官员当面向中共自然资源部部长提出警告时，对方受惊色变，连忙询问如何才能避免俄方发布禁令。
 <br/>
 <br/>
 11月7日，俄罗斯自然资源与生态部部长科贝尔金在俄联邦杜马会议上透露，俄方已经向中方通报，如果森林滥伐和木材灰色出口问题不能解决，俄罗斯可能全面禁止对中国出口木材。
 <br/>
 <br/>
 他在发言中表示，中国对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯木材.html" target="_blank">
  俄罗斯木材
 </a>
 有大量需求，因此才有乱砍滥伐和灰色出口。俄方已经向告知中方，如果未来不能整顿木材出口问题，俄方将在某一个时期停止出口木材到中国。
 <br/>
 <br/>
 俄罗斯政治导航网站以&lt;中国部长在莫斯科受到惊吓后脸色大变&gt;为题报导称，科贝尔金就俄罗斯远东和西伯利亚森林资源被毁问题，当面质问中共自然资源部部长，并警告如果问题不能解决，将可能对华禁售木材。
 <br/>
 <br/>
 报导说，中共部长闻言受到惊吓，脸色大变。
 <br/>
 <br/>
 俄罗斯消息网7日题为&lt;部长要求中国重造俄森林&gt;的报导则透露，中共部长当即询问如何做才能避免俄方发布禁令，俄部长回答说，中方可以付款在当地建立10-20个种子和树苗培育基地，以帮助重新造林。
 <br/>
 <br/>
 此外，科贝尔金7日在联邦杜马会议上确认，俄中两国就俄罗斯对华木材出口问题进行了部长级对话，中方代表称愿意探讨在俄国建设重新造林设施的问题。
 <br/>
 <br/>
 近几个月以来，中国企业对俄罗斯自然资源的掠夺，成为俄国社会关注焦点之一。包括伊尔库斯科、乌兰乌德等西伯利亚和远东城市，已发生多起抗议集会活动。
 <br/>
 <br/>
 在俄罗斯媒体关注之下，俄罗斯社会批评当局容忍中方掠夺俄罗斯资源的声音也日益高涨。俄国社交媒体上，许多网民批评普京政府向中共出卖俄罗斯利益，把俄罗斯变成中国企业的原材料供应基地。
 <br/>
 <br/>
 今年6月，位于贝加尔湖畔的俄罗斯伊尔库茨克州法院裁决下令，禁止中资贝加尔湖龙泉水厂在湖边生产瓶装饮用水，理由是该企业在质量控制上违反俄方规定。
 <br/>
 <br/>
 《美国之音》引述分析称，虽然迫于国际压力，普京政府选择与中共不断发展关系，但俄方对中共觊觎俄罗斯资源仍然保持警惕。中共掠夺俄罗斯资源、甚至可能进攻俄罗斯的声音仍然经常能听见，说明俄罗斯社会的反中（共）情绪并没有发生根本改变。
 <br/>
 <br/>
 日前，俄罗斯著名媒体人维尼吉克托夫透露，普京内心也一直将中共扩张视为俄罗斯的主要威胁之一。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
