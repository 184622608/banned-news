### 广西工业园大爆炸声势骇人 当事人诡辩：粉尘泄漏
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】周四，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广西南宁.html" target="_blank">
  广西南宁
 </a>
 横县工业园区一家水泥厂突然发生疑似爆炸事件，大量粉尘腾空而起，形成巨大〝蘑菇云〞。该工业园区的管理委员会声称那只是一时的疏忽造成的〝粉尘泄漏〞，但中国网友质疑现场情况声势骇人，实在不像是简单泄漏的问题。
 <br/>
 <br/>
 当地时间11月29日，有中国网友发布视频称，当天上午11时许，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广西南宁.html" target="_blank">
  广西南宁
 </a>
 横县那阳大桥附近腾起浓烟，疑似发生爆炸。
 <br/>
 <br/>
 这段时长39秒的视频中，伴随着巨响，有大量烟尘从水泥厂里窜出。滚滚粉尘腾空而起，形成一朵巨大的蘑菇云，然后逐渐向天际与河面扩散。惊人景象让现场民众都惊呆了。
 <br/>
 <br/>
 随后有网友指出，出事的地点是横县六景工业园区中的水泥厂。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/12-1/p9151251a395668409.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/12-1/p9151251a395668409-ss.jpg"/>
   <br/>
   <font size="-1">
    广西南宁六景工业园区一家水泥厂11月29日发生疑似爆炸事件，大量粉尘突然腾空而起，形成巨大〝蘑菇云〞。（视频截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 <br/>
 当天下午1点37分，广西南宁六景工业园区管理委员会官方微博发布说明称，广西云燕特种水泥建材有限公司将水泥散装罐装船时，因阀门未关好，压缩空气把粉尘吹出，造成泄漏而形成了扬尘。声明还强调，事故现场没有任何人员因此受伤。
 <br/>
 <br/>
 不过，许多中国网友看过现场拍摄到的视频片段后，都表示情况相当惊人，看上去更像是一起爆炸事件。
 <br/>
 <br/>
 有网友发帖质疑：〝看那浓浓烟尘喷涌而出的劲道，不像是简单的泄漏，更像是爆炸。〞有网友留言反讽：〝一定无人死，几个人轻伤。〞有网友议论称：〝产能过剩！炸平再起。〞还有网友评论道：〝最近很多爆炸事件，看来中共实施计划性爆炸〞。
 <br/>
 <br/>
 （记者李明综合报导/责任编辑：秦锋）
