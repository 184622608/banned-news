### 郭泉十年刑满获释 筹划开书画店
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】原南京师范大学副教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭泉.html" target="_blank">
  郭泉
 </a>
 被中共以〝颠覆国家政权〞罪判刑10年。他在11月12号刑满
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_获释.html" target="_blank">
  获释
 </a>
 ，但仍受到当局限制，被剥夺八项自由。郭泉表示，他准备办一个书画店，卖字画维生。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭泉.html" target="_blank">
  郭泉
 </a>
 11月16号告诉《新唐人》，12号早晨出狱后，他立即到了汉中门派出所。派出所所长给他一份告知书，上面有八条禁止条款。包括：他被剥夺政治权利三年，没有选举权和被选举权；不得担任国家公职人员；还有不能集会游行、示威；没有言论自由，没有发表、出版自由等八项限制。
 <br/>
 <br/>
 郭泉表示，未来他打算卖字画维生。
 <br/>
 <br/>
 郭泉：〝今后呢，我准备办一个体私营性质的书画店，由于刚出来也没有工作。现在正在各个部门在跑，因为我不愿意用我父母的钱，我这十年让我父母担忧，让我父母为了我吃了很多苦，我也没有孝顺，所以我不能再向父母要一分钱。〞
 <br/>
 <br/>
 郭泉出狱后向朋友表示：〝这10年我没有任何变化，而且身体还比以前更加的强壮、思维更敏捷。 〞郭泉透露，在狱中，他被关在单独的牢房，没有参加劳动 。
 <br/>
 <br/>
 郭泉：〝监狱认为我一个人生活可能更合适，因为我一直是从事知识份子的工作，他们认为我在房间里面自己看看书，就算劳动改造了，所以我一天车间也没去过。〞
 <br/>
 <br/>
 2012年1月23号，郭泉的妻子李晶和当时12岁的儿子抵达美国，寻求政治庇护，并呼吁国际组织营救郭泉 。
 <br/>
 <br/>
 郭泉：〝我想说就是感谢，感谢大家对我这个十年来的关爱，也感谢你们对我家人的关爱。〞
 <br/>
 <br/>
 郭泉原来是南京师范大学的副教授，帮助老兵等弱势全体维权，因发表公开信要求宪政改革，宣布组建〝中国新民党〞，2008年被捕，之后被当局以所谓的〝颠覆国家政权〞罪判刑10年， 在南京巿浦口监狱服刑，本月12号，刑满出狱。
 <br/>
 <br/>
 新唐人记者熊斌、舒灿采访报导
</div>
