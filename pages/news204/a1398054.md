### 【禁闻】澳媒：华为参与中共网络间谍活动
------------------------

 【新唐人2018年11月04日讯】
 <b>
  澳媒：
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
   华为
  </a>
  参与中共网络
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍.html" target="_blank">
   间谍
  </a>
  活动
 </b>
 <br/>
 <br/>
 中国电信巨头〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司〞，一直在国际社会否认自己有中共官方背景，但是，澳洲媒体日前报导说，华为协助情报人员进行网络
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍.html" target="_blank">
  间谍
 </a>
 活动。
 <br/>
 <br/>
 〝澳大利亚人〞周末版（Weekend Australian）11月2号披露说，澳大利亚官员得到的秘密情报报告显示，华为公司被要求提供密码和网络详细信息，以使中共情报人员获得外国网络的访问权。
 <br/>
 <br/>
 报导说，出于安全考虑，华为已经被禁止参与澳大利亚新的5G网络建设。
 <br/>
 <br/>
 <b>
  环保组织: 中国生产违禁
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_化学品.html" target="_blank">
   化学品
  </a>
 </b>
 <br/>
 <br/>
 一家国际环保组织说，有新的证据显示，中国继续制造和使用可破坏大气臭氧层、加剧全球变暖违禁化学品。
 <br/>
 <br/>
 〝美国之音〞11月4号报导说，总部设在伦敦的〝环境调查局〞和《纽约时报》表示，他们在中国进行的大量采访、和收集到的各种文件表明，中国工厂不顾全球禁令、继续制造或使用泡沫隔热材料CFC-11，这是从上世纪80年代未就被列为全球禁止生产的氟氯烃的一种。
 <br/>
 <br/>
 报导说，今年5月，科学家监测发现，氟氯烃的排放量忽然急剧上升。调查发现占据绝对优势的来源是中国。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
   联合国
  </a>
  审议
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人权.html" target="_blank">
   中国人权
  </a>
  中共将受严厉质询
 </b>
 <br/>
 <br/>
 联合国人权理事会11月6号将对中国的人权记录进行审议，中共将在大规模关押维吾尔族少数民族事件上接受严厉质询。
 <br/>
 <br/>
 联合国的一个专家小组任务，在新疆有多达100万维吾尔和其他少数民族人士，被关押在拘留中心，接收所谓的〝再教育〞。
 <br/>
 <br/>
 据报导，各国外交官也将在6号的会议上提问。
 <br/>
 <br/>
 〝法新社〞披露说，美国将提出的一个问题是：〝过去5年来，总共有多少人被强制关押在新疆的拘留设施？〞
 <br/>
 <br/>
 另外，美国还将要求中共说明为什么将和平的宗教行为非法化，并将人拘禁在这些政治〝再教育营〞。
 <br/>
 <br/>
 美国还将特别要求中共说明：〝哪些官员负责这项政策？〞
 <br/>
 <br/>
 英国也将询问中共什么时候才会落实联合国种族歧视小组的一个建议，这个建议要求中共〝停止在任何法外拘留设施中关押未经合法指控、审判和定罪者〞。
 <br/>
 <br/>
