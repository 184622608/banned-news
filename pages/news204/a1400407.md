### 中共内部文件吁自力更生准备维稳 官员:对谈判不抱幻想
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】中美对抗气氛不断升级，月底的首脑会晤〝和解〞可能性越来越低。日前，中共内部下发文件，下令全国党政机关准备〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自力更生.html" target="_blank">
  自力更生
 </a>
 〞，应对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_企业破产.html" target="_blank">
  企业破产
 </a>
 和金融市场波动，力保所谓〝社会稳定〞。
 <br/>
 <br/>
 11月21日，海外推特传出消息指，中共向全国各厅级以上机关传达了文件，大意是可能企业会倒掉一批，股市楼市汇市会有波动，要求中共机关准备〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自力更生.html" target="_blank">
  自力更生
 </a>
 〞，维护〝社会稳定〞，并宣称在〝党中央领导下〞一定会〝战胜困难〞。
 <br/>
 <br/>
 网友纷纷表示，估计北京与川普没法谈了。
 <br/>
 <br/>
 自由亚洲电台22日证实，21日上午，中国各地厅级以上机关确已传达高层指示，敦促基层要〝自力更生、稳定企业、楼市及房价〞。消息称，有关官员提醒基层干部，美国继续加征关税〝可能会导致部分企业陷入困境〞，也〝可能造成股市和汇市波动〞，要求各级做好〝维稳工作〞。
 <br/>
 <br/>
 报导引述分析称，这说明北京高层不准备对美国做出更多妥协，已无意在十天后的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 上达成和解，哪怕是临时的〝停火协议〞。
 <br/>
 <br/>
 报导引述一位中共经贸官员私下披露，中方对贸易谈判已〝不抱幻想〞，〝未来需要自力更生〞。
 <br/>
 <br/>
 有学者分析，美国要求中方拆除贸易壁垒，彻底改变经济政策，中共如果答应的话等同自杀。因此，如果美方不降低谈判底线，中共只能硬拼到底。
 <br/>
 <br/>
 而美国总统川普和白宫官员接连喊话，中方必须做出实质让步，否则将对中国产品继续增加关税，并在各个领域加大施压。而美国副总统彭斯在上周的APEC峰会上不留情面的反击，令北京领导人败兴而归。
 <br/>
 <br/>
 西班牙王室日前透露，北京最高领导人习近平11月28日将访问西班牙，但没有说明访问几天。中方随后宣布，中美贸易谈判中方首席代表刘鹤11月25日至28日将出访德国和出席第八届中欧论坛汉堡峰会。
 <br/>
 <br/>
 而G20峰会举行时间为11月30日至12月1日，届时中美首脑就将会晤。外界分析，这说明中方已无意为川习会做出积极准备。
 <br/>
 <br/>
 近期以来，中共政治局常委接连出京调研，也被指为准备贸易战升级做摸底调查。
 <br/>
 <br/>
 美国商务部长罗斯近日表示，美方将按原计划，在2019年1月将对2,000亿美元中国商品加征关税的税率从10%提升至25%。而美国贸易代表办公室称，对另外2,670亿中国商品加税的计划正在继续推进中。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
