### 非洲猪瘟逼近北京 天津爆疫情两百活猪亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逼近北京.html" target="_blank">
  逼近北京
 </a>
 。10月12日傍晚，中共农业农村部发布通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天津市.html" target="_blank">
  天津市
 </a>
 蓟州区爆发非洲猪瘟疫情，近200只病猪死亡。官媒罕见发文，列举非洲猪瘟危害，并指出猪瘟扩散蔓延，可能会影响大陆〝猪肉市场供给〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情首次蔓延至京津冀地区。中共农业农村部在12日傍晚5点多钟，接获
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天津市.html" target="_blank">
  天津市
 </a>
 蓟州区确诊爆发非洲猪瘟疫情的报告，天津市是第9个出现疫情的省区市。
 <br/>
 <br/>
 报告指，疫情来自蓟州区侯家营镇的一个养殖场，场内有639只存栏活猪，292只发病，其中189只死亡。
 <br/>
 <br/>
 在此前一天，11日及12日，辽宁省大连及鞍山也先后爆出疫情，共计有108只活猪发病，83只死亡。
 <br/>
 <br/>
 据通报，大连市普兰店区的一个养殖场，场内有1353只存栏活猪，有20只发病，11只死亡。而位于鞍山市台安县新合镇的一个养殖场，共有120只存栏活猪，88只发病，其中72只死亡。
 <br/>
 <br/>
 大陆非洲猪瘟疫情自8月爆发以来，愈演愈烈，难以控制。疫情由北向南扩散，已蔓延至辽宁、黑龙江、河南、江苏、安徽、浙江、内蒙古和吉林8个省份和天津1个直辖市。
 <br/>
 <br/>
 天津被非洲猪瘟疫情攻陷后，被认为疫情已〝逼近京城〞。12日，中共农业农村部官网罕见首次发文列举非洲猪瘟危害，指出非洲猪瘟发病率和死亡率最高可达100%，且目前全世界没有有效的疫苗。
 <br/>
 <br/>
 文章称，若非洲猪瘟扩散蔓延，可能给中国的生猪养殖业造成极大影响，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_影响猪肉市场供给.html" target="_blank">
  影响猪肉市场供给
 </a>
 〞。
 <br/>
 <br/>
 早在8月中旬，为应对疫情，中共中央对猪瘟疫情已做出〝重要指示〞，官方成立督察组，赴各地特别是重点省份，开展非洲猪瘟防控工作督查。这意味着猪瘟疫情已经十分严重，由各地管辖改为收归中央管辖。
 <br/>
 <br/>
 对于非洲猪瘟疫情在大陆的〝失控〞，也让联合国及周边邻国非常担忧。因中共一贯对外界封锁消息，外界怀疑中共隐瞒真实疫情信息，也担忧非洲猪瘟蔓延到全球，引发〝灾难性后果〞。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
