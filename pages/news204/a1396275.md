### 习近平南下坐实？ 深圳突然全面禁空拍
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】近日，深圳突然传出全面禁止空拍的消息。在这之前，香港媒体突然接到北京〝特急〞采访通知，连接香港、澳门与广东省珠海的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 〞23日在珠海举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开通仪式.html" target="_blank">
  开通仪式
 </a>
 ，这可能与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 即将南下的消息有关。
 <br/>
 <br/>
 根据中国大型空拍机生产商DJI（大疆）的网页GEO系统显示，整个深圳突然全面禁止空拍。
 <br/>
 <br/>
 DJI网页称，GEO系统可显示全球各类飞行受限制的区域，飞行用户将即时取得相关禁飞资讯，不仅包括机场，也包括一些突发情况造成的临时限飞区域，和一些永久性禁飞区域，如监狱、核工厂等。
 <br/>
 <br/>
 再如，人流密集的城镇或野生保护区等允许飞行的区域，也可能收到飞行警示。上述无法完全自由飞行的区域，都统称为限飞区。
 <br/>
 <br/>
 而深圳全面禁空拍行动疑为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 南下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 做准备。
 <br/>
 <br/>
 多家港媒17日报导，习近平将于19日至22日南下广东视察，20日到访广州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中山大学.html" target="_blank">
  中山大学
 </a>
 ，并在23日担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 通车仪式的主礼嘉宾。
 <br/>
 <br/>
 中山大学消息人士透露，习近平的确切行踪尚不清楚，该校目前确实正为习到访做相关准备工作。
 <br/>
 <br/>
 《自由亚洲》电台18日称，中共驻香港联络办公室17日向港媒发出〝特急〞采访通知，各报纸、电台及电视台每家可派一名记者（电视台两名），采访港珠澳大桥
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开通仪式.html" target="_blank">
  开通仪式
 </a>
 。
 <br/>
 <br/>
 采访地点在珠海，主办方还要求记者自费每晚600元人民币，入住指定的一家酒店。不过，对于记者关注的习近平是否参加开通仪式，该通知中未提及。
 <br/>
 <br/>
 香港01报导说，有香港人大代表已收到运房局邀请函，出席港珠澳大桥位的开通仪式。邀请函称，开通仪式将于23日上午举行，嘉宾会由香港乘船前往珠海，但没有透露主礼嘉宾。
 <br/>
 <br/>
 报导说，中共军方也在密锣紧鼓地进行筹备，经陆空运送大批军备抵港。有网民18日目睹两架运输机突然抵达香港机场，并卸下多辆军车。当日另有10多辆军车，在香港警方护送下，经三号干线浩浩荡荡驶出九龙。
 <br/>
 <br/>
 有军事专家认为，军方多以〝粮草先行〞的策略行事，估计是为习近平南下做先行准备。
 <br/>
 <br/>
 9月20日，媒体亦曾报导，习近平南下广州、深圳等多地企业视察，并且出席在深圳举办的改革开放40周年纪念大会，当时传出深圳安保进入一级戒备。但相关消息至今都未获证实。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：祝馨睿）
</div>
