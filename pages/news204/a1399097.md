### 非洲猪瘟攻入中国17省 传播途径至今是谜
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】最近3个月来，中国大陆遭遇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 的袭击，至今仍在继续蔓延。陆媒周一报导称，非洲猪瘟如〝幽灵〞一般悄然出没，尽管防疫机构想方设法封堵，疫情仍有加速蔓延的趋势，而且其病毒的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传播途径.html" target="_blank">
  传播途径
 </a>
 至今仍不明朗。
 <br/>
 <br/>
 据陆媒介绍，自从今年8月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 首次出现在中国大陆后，短短3个多月的时间里，疫情遍及17个省区，逾8万头生猪被扑杀，而且疫情呈现加速爆发的趋势。最令防疫机构伤脑筋的是，非洲猪瘟在大陆的传播〝无声无息、踪迹不明〞，如同〝幽灵〞一般防不胜防。
 <br/>
 <br/>
 有辽宁省盘锦市养殖户接受陆媒《财新网》采访时讲述，他的猪场距离沈阳首例疫情200公里外，听说疫情后，他就立即确认了家中上百头猪当时都很健康。此后每日消毒猪圈，〝甚至大门也不出〞。但2个月后，家中猪只还是出现了疫情，不吃东西、呕吐、便血、身上发红发紫，成批死亡。
 <br/>
 <br/>
 报导引述多名动物防疫和流行病学专家表示，至今仍然不清楚非洲猪瘟的具体传染路径和介质，有些地区明明采取了严密的防控措施，但不知为何还是会蔓延。
 <br/>
 <br/>
 不过，中国动物疫病预防控制中心、中国动物卫生与流行病学中心等，参与此次疫情研究的单位却拒绝接受媒体采访，因此中国整体的防疫资讯不够明朗。
 <br/>
 <br/>
 中国农业农村部官网近日刊文称，为了防止非洲猪瘟继续蔓延，有关防疫机构采取的措施包括：要求已发生非洲猪瘟疫情的省分及其周边省份的养殖场，不得使用馊水饲养生猪；其他省份不得使用未经高温处理的馊水喂食生猪；暂时禁止饲料生产企业使用以猪血为原料的血液制品生产猪用饲料。
 <br/>
 <br/>
 农业农村部今年9月上旬曾发布特急通知，要求与发生非洲猪瘟疫情省相邻的省份暂停生猪跨省调运，并暂时关闭省内所有生猪交易市场。
 <br/>
 <br/>
 邮政局也规定，从11月1日起，一律不得收寄活猪和未经检疫的未制熟猪肉制品，严禁收寄任何猪肉类寄递物品。
 <br/>
 <br/>
 即使采取了上述防堵措施，非洲猪瘟的疫情还是不断向其他省份扩散。安徽宣城一家养殖场位于丘陵地带，环境较为封闭，该养殖场给所有生猪都喂食米糠，结果还是未能躲过一劫，于9月3日被确诊发生非洲猪瘟。
 <br/>
 <br/>
 《财新网》报导称，虽然官方已禁止跨省调运生猪，但跨省走私生猪的现象仍然存在，而且有的养殖户在发现猪只患病时，就加紧售卖了病猪，这也是疫情传播的可能途径。
 <br/>
 <br/>
 报导援引中国人民大学农业农村发展学院副院长郑风田称，非洲猪瘟的传播介质非常广泛，而且全世界至今没有有效的疫苗。
 <br/>
 <br/>
 （记者李明综合报导/责任编辑：范铭）
