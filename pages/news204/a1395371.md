### 【禁闻】港民游行反填海造地：中共才是受益者
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港民游行.html" target="_blank">
   港民游行
  </a>
  反
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_填海造地.html" target="_blank">
   填海造地
  </a>
  ：中共才是受益者
 </b>
 <br/>
 <br/>
 10月14号香港出现游行，反对行政长官林郑月娥的大规模
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_填海造地.html" target="_blank">
  填海造地
 </a>
 计划。游行由〝守护大屿联盟〞、〝土地正义联盟〞等多个民间团体发起，警方称最高峰时有5800人。
 <br/>
 <br/>
 游行队伍当天下午前往政府总部。有游行人士自制大象道具，讽刺填海工程为浪费公帑的〝大白象工程〞。
 <br/>
 <br/>
 林郑月娥日前在施政报告中提出〝明日大屿〞计划，说香港未来将斥资4千-5千亿港元，在东大屿填海1700公顷建人工岛。声称是着眼未来、为了下一代。
 <br/>
 <br/>
 但香港泛民主派质疑计划会耗掉香港的财政储备，并指出中共国企才是计划的最大受益者。
 <br/>
 <br/>
 <b>
  抚顺
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
   法轮功学员
  </a>
  金顺女被迫害致死
 </b>
 <br/>
 <br/>
 中共迫害法轮功再添血债，10月10号，抚顺市法轮功学员金顺女被迫害致死。
 <br/>
 <br/>
 据〝法轮大法明慧网〞报导，金顺女是朝鲜族女士，今年66岁。9月19号，她到抚顺市顺城区新华街道开证明时，被警方绑架后，非法关押到南沟看守所，10月6号，被看守所迫害至昏迷不醒，4天后含冤离世。
 <br/>
 <br/>
 金顺女1997年因疾病和家庭债务，在走投无路之时开始修炼法轮功，炼功之后，很快恢复健康、摆脱困境。1999年中共迫害法轮功之后，她因不放弃信仰，多次被非法抓捕。2003年被非法判刑13年，在辽宁女子监狱受尽酷刑折磨。
 <br/>
 <br/>
 <b>
  贵州4村民撕烂
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五星旗.html" target="_blank">
   五星旗
  </a>
  遭判刑
 </b>
 <br/>
 <br/>
 中共最近开始对大陆民众所谓的〝侮辱国旗〞、〝不敬国歌〞判罪。据〝自由亚洲电台〞报导，贵州省龙里县鱼洞村的4名村民，今年2月因到村委会维权时，激愤之下，将中共五星旗从旗杆上扯下后撕烂，扔到村委会院内，之后被以〝侮辱国旗罪〞起诉，日前法院分别判处他们4人8个月和9个月徒刑。
 <br/>
 <br/>
 贵州民族大学黄教授表示，中共对几个农民判重刑，已经让底层民众对法律的公平绝望。
 <br/>
 <br/>
 当地一名村民也说，这个做法太令人恐惧。
 <br/>
 <br/>
 <b>
  网红主播哼唱
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共国歌.html" target="_blank">
   中共国歌
  </a>
  被拘留
 </b>
 <br/>
 <br/>
 10月14号，大陆虎牙直播平台主播〝莉哥〞、20岁的杨凯莉，因之前在直播时哼唱了中共国歌，被一些人举报对国歌不敬，而遭上海市公安局行政拘留5天。
 <br/>
 <br/>
 有网友说，这是本年度最大黑色幽默。并讽刺说：现在的大陆民众每个人都像蜈蚣一样，不知道先迈哪条腿走路了。
 <br/>
 <br/>
 编辑/周玉林
