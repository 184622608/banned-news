### 广西高校清查全校师生手机电脑
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】广西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桂林电子科技大学.html" target="_blank">
  桂林电子科技大学
 </a>
 最近传出下发清理违禁、违法音频工作的通知，清查全校4万多名教职人员和在校学生的手机、电脑、移动硬碟及U盘等。当地维权人士批评，校方公然践踏师生的隐私权跟言论自由权。
 <br/>
 <br/>
 最近，一份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桂林电子科技大学.html" target="_blank">
  桂林电子科技大学
 </a>
 关于开展清理涉暴、涉恐、反动、淫秽等违禁、违法音频工作的通知在网上传出。通知声称，所谓的境外敌对势力通过网路和手机、大肆传播违法违禁音影资料，要对全体教职员工、在校学生的手机、电脑、移动硬碟、U盘等存储介质进行全面清查。
 <br/>
 <br/>
 通知中说，清查时间从11月7号开始，到23号为止。
 <br/>
 <br/>
 不过，桂林电子科技大学官网并没有这份文件。
 <br/>
 <br/>
 广西桂林电子科技大学匿名学生:〝我没有被查，不知道是不是抽查有人的手机。〞
 <br/>
 <br/>
 该校附近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广西师范大学.html" target="_blank">
  广西师范大学
 </a>
 的学生透露，越来越多中国高校教师和学生因为在网上发表与政府不一致的言论而被处分。像电子科大这类学校的师生思想比较活跃，所以当局要严加管控。
 <br/>
 <br/>
 广西师范大学李姓学生:〝学生反对的声音很大，电子科大可能接触的技术比较前沿，国防科技啊，做那种比较高机密的，可能保密措施要严格一点，又知道如何查看国外的消息啊，比如说访问facebook这些东西，学校可能要铲除一下隐患嘛，现在某些学院已经开始自查了。〞
 <br/>
 <br/>
 广西桂林维权人士王先生认为，校方清查学生和老师的电子设备是开历史倒车。
 <br/>
 <br/>
 广西桂林维权人士王先生:〝这是公然践踏公民隐私权跟言论自由权，因为网路发布言论的大学被解除教职的，接二连三，一而再发生啊，这种行为肯定是违法违宪，他们对于这个言论控制，越来越加强了。〞
 <br/>
 <br/>
 记者14号拨打校方电话，但都无人接听。
 <br/>
 <br/>
 该校官网显示，桂林电子科技大学是国家工业和信息化部与广西共建高校、国家国防科技工业局与广西共建高校，承担国家科技重大项目，包括国防军工科研项目，目前有学生4万多人。
 <br/>
 <br/>
 新唐人记者熊斌、柏妮采访报导
</div>
