### 【禁闻】 诱捕外逃官员 中共禁其美籍家属出境
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】《纽约时报》11月26号报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 当局为了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诱捕.html" target="_blank">
  诱捕
 </a>
 一名外逃贪官，禁止他的妻女离开中国，而他的妻女都是美国公民，目前没有证据显示他们涉及犯罪。
 <br/>
 <br/>
 今年6月，辛西娅・刘（Cynthia Liu）和弟弟维克多・刘（Victor Liu）跟随母亲桑德拉・韩（Sandra Han）返回中国，去探望生病的祖父。但几天后，警察拘留了桑德拉・韩，把她转移到一个俗称〝黑监狱〞的秘密地点关押。之后的几个月，姐弟两三次试图离开中国，但在机场被告知，他们已经被〝禁止离境〞。
 <br/>
 <br/>
 辛西娅和母亲早年归化美国国籍，而维克多出生在美国，这次他们都用美国护照入境中国。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 外交部回应此事的说法却是，三人都持有中国公民的合法有效证件。因为他们涉嫌经济犯罪，已经被警方
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_限制出境.html" target="_blank">
  限制出境
 </a>
 。
 <br/>
 <br/>
 但姐弟两发表声明说，中共警方明确表示他们两人没有被调查，也没有受到犯罪指控。将他们一家扣为人质，目的是迫使他们的父亲，就是前国营交通银行广州分行前行长刘昌明，返回中国。
 <br/>
 <br/>
 刘昌明在2007年外逃，他涉嫌违规放贷近百亿元人民币。中共在2015年开始〝天网〞行动，缉捕外逃腐败分子，并通过国际刑警组织对100名外逃贪官发出红色通缉令，其中包括刘昌明。
 <br/>
 <br/>
 但辛西娅姐弟及母亲表示，他们早在2012年已经跟刘昌明脱离关系，也没有联系。
 <br/>
 <br/>
 原福建宁德中级法院经济庭审判长李建峰告诉《新唐人》，中国法律有明确规定，什么样的人回国以后，中共政府能够限制他们的人身自由。
 <br/>
 <br/>
 原福建宁德中级法院经济庭审判长李建峰：〝要符合两个要件。一是在国外触犯了中国的法律；二，这个（罪行）追究起来要判三年以上有期徒刑的。像这种人中国政府可以对他进行立案。那么他回国以后肯定要被限制人身自由，措施之一就是限制出境。〞
 <br/>
 <br/>
 《纽约时报》说，刘家人在美国过的非常富裕，生活来源部分是通过收购房地产，尽管公共记录中没有迹象表明，这些房产是用刘昌明被指控贪污的钱购买的。
 <br/>
 <br/>
 大陆法学博士张杰表示，中共惯用株连家人的手法来迫害中国公民，但现在延伸到美国公民。
 <br/>
 <br/>
 大陆法学博士张杰：〝你拘捕一个人必须有法定的事实依据和理由，并且要经过一个法定程序。但实际上中共根本就是不会这样做的。就可以越境去抓人，然后逼迫你最后认罪等等。在国内是很正常的，延伸到美国公民。所以美国人感到很震惊。因为美国是一个法治社会。〞
 <br/>
 <br/>
 中共用〝禁止离境〞的手法扣押美国公民，成为日益紧张的美中关系中新迸发出的火花。美国媒体《Daily Beast》此前报导，过去两年至少有20名美国公民被中共当局禁止离境，中共有时把他们当〝人质〞，强迫他们在美国的家人回中国接受调查。
 <br/>
 <br/>
 美国国务院已经建议美国公民赴中国旅行时要更加谨慎，当心被〝禁止离境〞。上个月美国国务卿蓬佩奥在访华时批评了〝禁止离境〞做法，这个月他又在华盛顿的一个会议上，向负责外交的中共中央政治局委员杨洁篪提到了刘家人。
 <br/>
 <br/>
 这起事件也暴露出双重国籍的风险。中国不承认双重国籍，但不少中国人在获得了外国国籍之后，由于种种原因仍然保持中国国籍。
 <br/>
 <br/>
 李建峰：〝双国籍这种行为在国籍法上是很危险的。如果到了承认双国籍的国家，那你多多少少还受到一点领事保护。如果到了不承认双国籍的国家，你这种领事保护就很难在所在国奏效。〞
 <br/>
 <br/>
 美国国务院也提醒美国公民说，如果美、中双国籍人士持护照等中共政府旅行证件入境，一旦被逮捕、拘留或参与刑事或民事调查，美国领事官员将不被获准探视或者协助与中共政府的交涉。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/李沛灵
