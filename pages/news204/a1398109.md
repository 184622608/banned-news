### 进博会前 上海查微信禁时政消息
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】中共举办的首届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际进口博览会.html" target="_blank">
  国际进口博览会
 </a>
 11月5号在上海开幕，为了管控民间舆论，开幕前夕，上海网信办查封了一批微信公众号。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 来临前夕，上海市政府加强舆论管控，网信办查处了19个微信公众号。
 <br/>
 <br/>
 官方声称，其中4个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微信账号.html" target="_blank">
  微信账号
 </a>
 ，因假冒官方机构账号，违规发布时政新闻信息等，被关闭；另有14个教育类公众号，也因为传播所谓的不实信息、发布时政新闻信息等，被暂停更新15天或30天。
 <br/>
 <br/>
 上海访民茆先生说，上海召开个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博览会.html" target="_blank">
  博览会
 </a>
 ，不仅封号还封口，不让访民在网上发声，也不让出门。
 <br/>
 <br/>
 上海访民茆先生：〝像我们上访的进博会都不能去，一去就刑事拘留，我现在有一个朋友关在凤仙度假村里面。它把我的短讯功能封了，发短讯都发不出去，说我发乱七八糟的东西，我从来没发过，把我封掉。〞
 <br/>
 <br/>
 上海访民沈女士表示，上海召开进博会封微信帐号，与她们的冤情相比是小事。
 <br/>
 <br/>
 上海访民沈女士：〝我们现在的老百姓房子都没有，事情不解决，上访十几年0036只要解决我们内部的平安问题，吃住穿喝能得到保障，我就感觉幸运了，我已经是60岁的老太婆，吃住都不安定，再开这么多好的国际会，对我们来说有什么意义呢？〞
 <br/>
 <br/>
 上海市民秦先生两个月前就被封了一个微信帐号，召开进博会前又被封了一个。
 <br/>
 <br/>
 上海市民秦先生：〝开进博会民间窗户都不能打开，封条贴着了，微信上都有，我不能说太多，我这个电话受监控的。〞
 <br/>
 <br/>
 在中国，近年来微信已经成为了个人、企业和机构通讯的主要渠道，但由于微信用户数量庞大、搜集数据海量，越来越多的人担忧个人数据的安全。
 <br/>
 <br/>
 根据国际人权组织大赦国际2017年的一份调查报导，全球最受欢迎的11种手机通讯应用程序，加密保护用户隐私的排行列表中，腾讯公司排名垫底。
 <br/>
 <br/>
 新唐人记者熊斌、元真采访报导
 <br/>
</div>
