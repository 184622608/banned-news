### 北京回应美国要求 美中贸易战或降温
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】美中贸易谈判中断了三个月后，中共商务部周四证实，中美为解决贸易争端，恢复了高层会谈。外界认为，中方此举或将促成下个月的川习会，达成有助于中美贸易战降温的成果。
 <br/>
 <br/>
 中共商务部发言人高峰周四表示，美中双方经贸高层已经恢复接触。
 <br/>
 <br/>
 中共商务部发言人高峰：〝11月1日中美两国元首通话后，双方经贸领域高层接触已经恢复，同时，工作团队正在保持接触，以认真落实中美两国元首通话达成的共识。〞
 <br/>
 <br/>
 据报导，三名白宫知情人士周三透露，中方对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 在贸易冲突中提出的大幅度贸易体制改革，包括知识产权盗用、产业补贴以及市场准入设限等要求，已作出书面回应。
 <br/>
 <br/>
 不过，高峰周四没有对上述说法做出回应，也没有透露会谈具体细节。
 <br/>
 <br/>
 美国财政部发言人表示，财政部副部长玛尔帕斯率领的美方代表团，周二通过视频与中方讨论了贸易问题。
 <br/>
 <br/>
 美国总统川普本月初与习近平通话后表示，他认为华盛顿将与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 达成贸易协议，如果没有进展，随时准备对中国商品征收更多关税，包括从明年1月1日起，对价值2000亿美元的中国进口商品关税税率从10％增加到25％。
 <br/>
 <br/>
 如果北京进行报复，川普还将进一步对所有剩余2670亿美元的中国进口产品征收关税。
 <br/>
 <br/>
 新唐人记者李梅、宇微综合报导
</div>
