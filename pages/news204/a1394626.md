### 慎入！男子感冒鼻塞  拿长刀插头〝开孔〞通气
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】俄罗斯一名男子因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感冒鼻塞.html" target="_blank">
  感冒鼻塞
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_呼吸不畅.html" target="_blank">
  呼吸不畅
 </a>
 ，他决定〝用头呼吸〞。于是，拿了一把约20厘米的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长刀.html" target="_blank">
  长刀
 </a>
 插进
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_头部.html" target="_blank">
  头部
 </a>
 ，却意外存活。医生说，他能够活下来，这件事相当神奇。
 <br/>
 <br/>
 俄媒10月7日报导，俄罗斯男子尤里（Yury Zhokhov）觉得自己的鼻子塞住了，无法好好呼吸。10月5日，他跑到一处荒地上，拿出一把长8英寸（约20厘米）的刀，直直往自己的脑袋中心插进去，希望在头骨上开一个〝孔〞，来解决自己
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_呼吸不畅.html" target="_blank">
  呼吸不畅
 </a>
 的问题。
 <br/>
 <br/>
 随后，尤里被路过的民众看到，紧急报警。
 <br/>
 <br/>
 俄罗斯顿涅茨克地方执法部门赶到现场，看见尤里跪坐在草原上，头顶的正中间插着一把长长的刀，头皮也裂了一块很大的缝隙，场面非常惊人。
 <br/>
 <br/>
 尤里对警方说，他感冒了好几天了，鼻子塞住了，于是才决定〝用头呼吸〞，不过，刀插进
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_头部.html" target="_blank">
  头部
 </a>
 后就卡住了，自己还是无法呼吸，极为难受。
 <br/>
 <br/>
 执法人员发现尤里并未出现任何意识混乱等问题，迅速将其送往医院。
 <br/>
 <br/>
 随后，医护人员给尤里做了X光，以确定刀片插入头骨的深度。手术后，近20厘米长的刀虽已成功取出，但是尤里仍需住院治疗，以免伤口出现感染。
 <br/>
 <br/>
 医护人员表示，尤里能够活下来这事相当神奇，尤其他仍然保持着清醒的状态，而且没有任何急性脑损伤，还可以说话，真是一个奇迹！
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：祝馨睿）
</div>
