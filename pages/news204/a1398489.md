### 互联网大会不见刘强东  大佬调侃：为保护美女引导员
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】11月7日，第五届世界互联网大会在浙江乌镇开幕。性侵丑闻缠身的电商巨头京东集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 再次缺席。遭到参会大佬调侃，说主办方为了保护参会的美女引导员。
 <br/>
 <br/>
 第五届世界互联网大会在浙江乌镇开幕前，中国科技企业大佬6日晚已陆续抵达乌镇，包括阿里巴巴马云、腾讯马化腾、小米雷军、网易丁磊等人，但未见到电商巨头京东创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 的身影。
 <br/>
 <br/>
 去年的世界互联网大会上，刘强东不仅露面，还公开炮轰马云〝为富不仁〞，而此次刘强东因在美国涉性侵案，在中国舆论场上失声60多天。
 <br/>
 <br/>
 对于刘强东缺席世界互联网大会，实名认证为〝讯网云计算董事长、中科院计算机专家〞的〝赵盛烨〞在社交媒体发布两张身着蓝色服饰的美女引导员照片，并调侃说，主办方为保护这群孩子，〝应该肯定是不会找刘强东来了。〞
 <br/>
 <br/>
 刘强东因在美国涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 一案，在过去60多天里，几乎缺席中共所有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重磅会议.html" target="_blank">
  重磅会议
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重要场合.html" target="_blank">
  重要场合
 </a>
 。舆论纷纷猜测，刘强东遭北京〝抛弃〞了。
 <br/>
 <br/>
 10月24日，中共官方发布《改革开放40年百名杰出民营企业家名单》，鼓励民企老板〝不忘创业初心〞。名单中没有刘强东。
 <br/>
 <br/>
 11月1日，习近平主持召开民营企业座谈会，安抚所有民营企业和民营企业家，并称他们都是〝自己人〞。而刘仍缺席活动，被认为已被踢出习的〝自己人〞圈子。
 <br/>
 <br/>
 11月2日，美媒披露刘强东案的具体细节，称刘强东在性侵受害者时，遭到多次反抗，最后仍得逞。刘还告诉21岁的受害女大学生，可以像中共头号女间谍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓文迪.html" target="_blank">
  邓文迪
 </a>
 一样。
 <br/>
 <br/>
 据美国警方的报告，刘强东涉嫌强奸发生在8月31日凌晨1点左右，当晚11时32分被捕入狱，16个小时后获释，警方对刘强东一案的描述是〝一级性犯罪：强奸既遂〞。
 <br/>
 <br/>
 明尼苏达亨内平县警方9月21日宣布，已完成对案件的初步调查，接下来将由检方接手，审查案件全部证据，并最终决定是否起诉。
 <br/>
 <br/>
 有华裔刑案律师认为，刘强东遭起诉的可能性超过90%，案件一旦开始会涉及重新逮捕等问题。根据明尼苏达州的法律，一级强暴罪只要最后被定罪，刑期至少12年起。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：戴明）
