### 朱军猥亵案:当事女沉寂半月复发声透露现状
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】中共央视前主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军.html" target="_blank">
  朱军
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猥亵案.html" target="_blank">
  猥亵案
 </a>
 自从上月25日首次开庭后，涉事双方都一度陷入沉寂之中。日前，该案的受害女主
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弦子.html" target="_blank">
  弦子
 </a>
 （化名）在社交媒体上发帖透露了自己目前的现状。
 <br/>
 <br/>
 北京时间11月6日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弦子.html" target="_blank">
  弦子
 </a>
 的个人微博上发出一条帖文称，〝上午工作到一半请假出来和律师开会，现在见完律师又要赶回去开剧本会，来回横跨三十公里路上就要三个小时。缺席工作必须向合作伙伴寻求谅解，这只是诉讼阶段中非常琐碎寻常的一件小事，但‘小事’或许只是因为我的职业比较自由、同伴非常善良，知道‘弦子最近很忙’。〞
 <br/>
 <br/>
 弦子在微博上还表示，自己为了案件进展，拖着行李箱奔跑在北京。自己倒不怕辛苦，可是确实有身边人的生活因此事受到了影响。她披露，有姐姐因为心理状态被学校要求休学、有九五年的妹妹为了应诉需要长期请假，不得不辞掉工作。她在帖文中感叹，〝即使官司打赢，受害者也不是赢家，灯光落下后，留给她们的，是他人即使伸出援手也无法解决的千疮百孔的生活——希望赢吗？我们更希望它从未发生。〞
 <br/>
 <br/>
 此前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军.html" target="_blank">
  朱军
 </a>
 起诉弦子和〝麦烧同学〞（微博名）的名誉权纠纷案于今年10月25日在北京市海淀区法院首次开庭，当日主要进行证据交换流程。当事人弦子现身法庭，但朱军并未出现。
 <br/>
 <br/>
 庭审结束后，弦子和她的好友〝麦烧〞一同接受了媒体的采访，她们透露，在这次开庭前，她们已向法院提交了对朱军的〝反诉〞，法院会针对证据和证据内容情况来看朱军能否出庭。
 <br/>
 <br/>
 当时弦子被问及，为何4年前不去处理这个案件，却在4年后才旧事重提？弦子回应说，4年前不是她不想处理好这个事，相反，她反覆地去派出所，但并没有得到任何回复，连份书面说明都没有。
 <br/>
 <br/>
 弦子强调说：〝公众一直在说是我自己主动放弃的，并不是这样，我是被动放弃的，没有得到回复，只能放弃。4年前，如果朱军只是一个普通人的话，那这件案子可能是另一个走向。〞
 <br/>
 <br/>
 庭审结束后，朱军的代表律师离开法院时遭到大批媒体围堵与追问，但均以〝不方便〞为由拒绝回答。
 <br/>
 <br/>
 弦子与她的朋友们则在10月25日当天更新的微博中写到，：〝今天第一次出庭，比想像中难，但能坚持住，非常感谢大家。〞
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：秦锋）
