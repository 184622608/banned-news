### 秦岭别墅奢侈细节曝光 养藏獒喂西藏空运羊肉（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】近日，陆媒曝光了中国秦岭北麓最大违建别墅〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈路别墅.html" target="_blank">
  陈路别墅
 </a>
 〞的更多内幕。据披露，该别墅的房舍园林中布置了大量文物雕塑，园内主人的生活十分奢侈，招待客人动辄拿50年茅台当水喝，就连别墅内豢养的5只藏獒幼崽，吃的都是从西藏空运过来的羊肉。
 <br/>
 <br/>
 综合陆媒报导，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈路别墅.html" target="_blank">
  陈路别墅
 </a>
 〞位于秦岭西安段的鄠邑区石井镇蔡家坡村村南的山脚下，背靠朱雀森林公园、太平国家森林公园，紧邻金丝猴自然保护区，风景秀丽，南侧倚靠的山脉则是久负盛名的终南山一脉。
 <br/>
 <br/>
 这处具有园林特色的中式仿古别墅共圈占农田14.11亩，从2008年起，经过近三年的施工和装修建成。园区内建筑物占地5.4亩，鱼塘两处逾千平方米、狗舍面积达78平方米，沿着池塘和园内曲径小路边路布置着各种文物雕塑共211件，现代工艺品26件，其奢华阔气令观者瞠目。
 <br/>

 <br/>
 别墅园区内还栽种了一些名贵树木。其中，有一棵国槐，经专家鉴定已经有500多年树龄；院内还有三棵皂角树估值超过20万元人民币，直径最长的一棵有一米多，最短的也有六七十厘米。
 <br/>
 <br/>
 据《中国新闻周刊》11月13日的报导，陈路别墅的正门悬挂着写有〝望重成均〞 四字的红匾，别墅的主体建筑内，放着一尊2米多高的黄花梨达摩像，大厅墙上挂着范增的巨幅八骏图。院内还有大量磨盘、石鼓、拴马桩、古代石狮等文物。
 <br/>
 <br/>
 报导引述知情人士透露，陈路别墅的主人平日里生活十分奢侈，园内曾豢养了五只从西藏空运来的藏獒幼崽，〝由于藏獒太小，（别墅主人）同时从西藏空运来羊肉喂藏獒。等藏獒长大点适应水土后，再在本地买肉当狗食。〞
 <br/>
 <br/>
 别墅所在地的村干部李连涛也向陆媒《新京报》透露说，他自己曾多次跟支亮在别墅里一起喝酒唱歌，见到过某〝党政要员〞 三次。别墅主人招待客人时，〝50年茅台当水喝，一次一箱，喝不了让我带走。〞
 <br/>

 <br/>
 据公开的资讯，这处被称为秦岭西安段最大单体的违建别墅，最初在西安本地媒体及官方信息发布平台上被称为〝支亮超大违建别墅〞，基本讯息显示该别墅系以陕西秦悦贸易有限公司法定代表人支亮名义所建。
 <br/>
 <br/>
 但有陆媒经调查发现，名义上的别墅主人支亮出生于1980年，他当年与石井镇蔡家坡村签订土地租赁合圈地建别墅时年仅25岁，大学刚毕业不久，因此怀疑这些违建别墅的真正主人另有其人。
 <br/>
 <br/>
 后经有关机构查实，这些别墅的真正业主是一个名为陈路的人。陆媒《财经》杂志曾爆料称，陈路的父亲曾在西安市党政系统担任要职，但并未披露陈路其人和他父亲更多详细的信息。
 <br/>
 <br/>
 今年10月15日晚，陕西多家媒体及官方信息发布平台上报导有关违建别墅的消息时，才统一将这处别墅园区称为〝陈路超大违建别墅〞。
