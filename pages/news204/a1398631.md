### 监察委〝一号大案〞 西南一大学校长潜逃自杀均失败
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西南林业大学.html" target="_blank">
  西南林业大学
 </a>
 中共党委副书记、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_校长.html" target="_blank">
  校长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蒋兆岗.html" target="_blank">
  蒋兆岗
 </a>
 曾因潜逃，被云南省公安厅发布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_A级通缉令.html" target="_blank">
  A级通缉令
 </a>
 ，更被云南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监察委.html" target="_blank">
  监察委
 </a>
 定性为〝第
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一号大案.html" target="_blank">
  一号大案
 </a>
 〞。日前，中共官方公布蒋兆岗的严重违纪违法案件。蒋兆岗交代曾吞下大量安眠药自杀，但因药物过期而失败。
 <br/>
 <br/>
 11月7日，《中国纪检监察报》刊发了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西南林业大学.html" target="_blank">
  西南林业大学
 </a>
 原
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_校长.html" target="_blank">
  校长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蒋兆岗.html" target="_blank">
  蒋兆岗
 </a>
 严重违纪违法案件。
 <br/>
 <br/>
 据悉，蒋兆岗是省级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监察委.html" target="_blank">
  监察委
 </a>
 员会成立以来，全中国首例应监委要求发出的通缉令进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_追缉.html" target="_blank">
  追缉
 </a>
 的中共官员。
 <br/>
 <br/>
 中共党媒报导，5月9日，中共云南省监委决定对蒋兆岗进行监察调查。得知消息后，蒋兆岗开始逃亡。
 <br/>
 <br/>
 5月11日，云南省公安厅发布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_A级通缉令.html" target="_blank">
  A级通缉令
 </a>
 ，宣布对蒋兆岗执行通缉的决定，是由云南监察委下达的，属〝第
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一号大案.html" target="_blank">
  一号大案
 </a>
 〞。经监察委同意，对发现线索、举报查实的个人或单位，奖励人民币5万元，对抓捕归案的个人或单位，奖励人民币10万元。
 <br/>
 <br/>
 但通缉令中没有说明蒋兆岗所犯何罪，为何要对其下发A级通缉令。
 <br/>
 <br/>
 5月30日，蒋兆岗在昆明市的藏匿点被抓获归案，云南省监委对其采取留置措施。 据蒋兆岗交代，自己曾在车库内吃下20片安眠药，〝可能因为放的时间长了，安眠药失效了〞，昏睡了两天后，又醒了过来。
 <br/>
 <br/>
 公开资料显示，蒋兆岗，1964年9月生，云南玉溪元江县人。历任云南财贸学院团委副书记、书记，云南财贸学院教学服务中心经理、总务处处长、后勤产业集团总经理，云南财贸学院副院长，云南财经大学副校长等职。
 <br/>
 <br/>
 2008年10月，任中共云南省政府副秘书长。2011年2月任中共云南省农村信用社联合社党委书记。2016年2月，任中共西南林业大学党委副书记、校长。
 <br/>
 <br/>
 蒋兆岗最近一次参加公开活动是在5月7日，中共西南林业大学党委理论学习中心组召开2018年第八次集中学习会。蒋兆岗以校党委副书记、校长身份参加学习并作交流发言。
 <br/>
 <br/>
 对于蒋兆岗被通缉，陆媒称蒋兆岗可能卷入中共云南省原省委常委、副省长曹建方秘书吴敏章受贿案。吴敏章已于2017年10月，因犯受贿罪被昆明中院判处有期徒刑4年半；而吴敏章的老上级曹建方因严重违纪，受到开除党籍、由副部级降为副处级非领导职务的处分。
 <br/>
 <br/>
 同时蒋兆岗潜逃或与云南省农信社窝案有关，蒋兆岗于2011年2月至2016年1月，任中共云南省农村信用社联合社党委书记。他当初的数名同僚都已经落马被查。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：戴明）
</div>
