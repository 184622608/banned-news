### 中共公安部副部长孟宏伟被调查
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共公安部副部长.html" target="_blank">
  中共公安部副部长
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 失踪数天后，北京当局宣布其〝涉嫌违法〞，正接受调查。据报，孟宏伟涉嫌参与政变，孟宏伟妻子向法国报案有预谋。
 <br/>
 <br/>
 周一，中共外交部发言人表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 涉嫌违法，正在接受调查。
 <br/>
 <br/>
 周日深夜，中共纪委官网通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共公安部副部长.html" target="_blank">
  中共公安部副部长
 </a>
 孟宏伟涉嫌违法被查。
 <br/>
 <br/>
 中共官方还说，孟宏伟已经辞去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席职务。
 <br/>
 <br/>
 中共外交部发言人陆慷：〝对孟宏伟涉嫌违法，立案审查。他本人已经书面提交了辞去国际刑警组织主席的辞呈。〞
 <br/>
 <br/>
 孟宏伟9月底从国际刑警组织里昂总部返回中国，之后音讯全无。10月4日傍晚，孟宏伟的妻子向当地警方报案。
 <br/>
 <br/>
 据报，孟宏伟返回中国时，〝一落地〞就被中共纪检部门〝带走〞。
 <br/>
 <br/>
 孟宏伟被认为是中共前政法委书记周永康的亲信，可能涉嫌参与2015年的北戴河政变未遂案。
 <br/>
 <br/>
 有报导认为，孟宏伟夫妇早已设计好被调查后的应对计划，想让妻儿获得法国庇护身分。
 <br/>
 <br/>
 中国网民说：〝作为中共的公安部副部长，他曾经让很多人被失踪，现在终于轮到自己被失踪了。〞
 <br/>
 <br/>
 新唐人电视综合报导
