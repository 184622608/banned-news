### 【禁闻】美国中期选举落幕 北京一口气难松
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 终于落幕，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 继续主掌参议院，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 则拿下众议院。而对于期待民主党大胜，扭转川普贸易政策的中共又意味着什么呢？来看报导。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的结果在当地时间11月7号出炉。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 守住了参议院多数党的地位，并拉大了与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 的席次差距。而民主党一如预期拿下众议院，但他们期待的〝蓝色浪潮〞并没有出现。
 <br/>
 <br/>
 旅美时事评论员夏小强：〝这次美国的中期选举结果本身是没有太多意外的。因为在历史上中期选举，执政党往往会失去国会的控制。〞
 <br/>
 <br/>
 回顾美国历史上的中期选举，绝大多数在选前两年入主白宫的执政党都以失利告终，失去国会多数席位。但今年的中期选举，共和党在参议院扩张了3个席位，与民主党从2010年以来失去的18个席位形成鲜明对比。另外在众议院，今年共和党失去了26个席位，而民主党2010年在奥巴马时期损失了63席，1994年在克林顿时期损失了52席。
 <br/>
 <br/>
 夏小强：〝美国的政治传统是对政府权力的过大表示警惕。所以说在美国的政治中，往往会出现这种相互制衡的局面。也体现出美国民众对政治制衡制度的一种表现。这也是美国政治制度成熟的一种表现。〞
 <br/>
 <br/>
 外界普遍预测，未来民主党可能在众议院阻挠川普政府希望推动的法案，例如政府预算案、基础建设、健保改革等各项法案的立法，可能需要更多的谈判协商。同时，众议院也拥有对总统及政府官员的调查权、弹劾权。
 <br/>
 <br/>
 但未来预计共和党内部意见将会更加一致，川普在共和党内的号召力与影响力将扩大。同时，参议院拥有确认联邦司法任命，包括最高法院大法官提名人的权力，因此预计美国法官的立场会继续向传统理念靠拢。
 <br/>
 <br/>
 此前，中共曾期望民主党在中期选举中大胜，认为他们会扭转川普的贸易政策。如今结果出炉，多位观察人士认为，美国对中共强硬的政策已经是两党共识，民主党也会支持贸易战。
 <br/>
 <br/>
 哥伦比亚大学政治学博士李天笑：〝现在共和党和民主党实际上在美中贸易的这个问题上，观点是一致的。原来少数党领袖，现在可能是多数党，就是众议院议长佩洛西，和参议院议长麦康奈尔，这两个人都在不同的场合谈到过，要改变中美之间不平衡的交易，都支持川普对中国进行关税制裁。〞
 <br/>
 <br/>
 台湾《中央社》也引述传统基金会亚洲研究中心主任罗曼的话说，民主党赢得众院，反让美、中之间要在贸易上缓和紧张局势变得更为复杂。主要的原因是，川普现行对中共的做法有任何软化，将受到民主党人士攻击。
 <br/>
 <br/>
 而众议院民主党领袖南希・佩洛西一向对中共持强硬立场。她笃信天主教，长期以来在世界各地伸张人权，也试图改善中国的人权记录。去年川普和中国国家主席习近平会晤前，佩洛西敦促他向习近平提及中国人权问题，以支持那些因为自己的宗教和政治信仰遭到迫害的人。
 <br/>
 <br/>
 李天笑：〝佩洛西她实际上是比较重视人权的。她对中共的邪恶有一些认识。在多次法轮功反迫害的集会上她也参与的，而且在国会支持法轮功的决议案里边她也参与的。给中国人民人权，用普世价值来处理对外关系中的一些事情，在这点上她跟川普都是有共识的。〞
 <br/>
 <br/>
 川普已经向民主党提供了一根橄榄枝。他致电佩洛西，祝贺民主党赢得众议院，也发推特表示，支持佩洛西。而佩洛西在胜选演讲时也呼吁两党合作。她表示民主党将会寻求方案令全国上下团结，因为大家已经受够了分裂，美国人希望的是和平，他们想要结果。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/钟元
