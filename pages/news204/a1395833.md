### 范冰冰〝虎口脱险〞后 李晨首露面直奔女友住所
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 缴纳巨额罚金〝虎口脱险〞后，一直在美国拍戏的范冰冰男友
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李晨.html" target="_blank">
  李晨
 </a>
 ，于17日现身北京机场。陆媒说，很可能是李晨新片已杀青，他回京后直奔位于朝阳北路的范冰冰住所，这个举动力证了两人并没分开。
 <br/>
 <br/>
 10月18日，搜狐娱乐报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 男友
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李晨.html" target="_blank">
  李晨
 </a>
 于10月17日现身北京首都机场。此前他一直在美国拍戏，很可能是新片已杀青。李晨戴着口罩墨镜棒球帽，明星常备的三件套一样都不少，非常低调，从机场走出来后，坐上车匆匆离开。
 <br/>
 <br/>
 据悉，李晨随后直接回到了位于朝阳北路的范冰冰住所，这个举动也力证了两人并没有分开。
 <br/>
 <br/>
 范冰冰10月15日也曾现身北京首都机场，中港台多家媒体纷纷转发这一消息。
 <br/>
 <br/>
 从曝光的照片中可见，她戴着墨镜和黑白色棒球帽，身穿黑色宽松长外套，脚踩金色厚底鞋。她一手拿着手机，一手插在外套的口袋里，全程面无表情，一言不发，匆匆往外走。
 <br/>
 <br/>
 从画面中还看到，她身边有一名保镳，帮撑着黑伞，疑似想挡住镜头。
 <br/>
 <br/>
 这是范冰冰在公众视野消失135天后首度露面。与过去相比，她看上去似乎面容消瘦，脸部的菱角非常明显。
 <br/>
 <br/>
 中国女星杨采钰10月13日在微博PO出新戏《七日生》杀青照，画面中的李晨露出浅笑、双手抱胸。对比过去，可以明显发现他双颊凹陷，瘦到下巴尖了不少。
 <br/>
 <br/>
 不少粉丝直呼心疼：〝看来没少为女朋友范冰冰操心〞、〝8个亿不见了，换我也憔悴〞。
 <br/>
 <br/>
 5月28日，央视前主持人崔永元曝光范冰冰通过〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞逃税后，范冰冰6月初便处于〝消失〞状态。10月3日，中共税务机关称，范冰冰涉嫌偷税漏税，缴纳8.8亿元人民币罚金，可免除牢狱之灾。
 <br/>
 <br/>
 在范冰冰接到当局的天价罚单后，网传范冰冰8月份已开始抛售41套房，意图变现，李晨也挂牌出售自家的四合院，帮女友凑款。
 <br/>
 <br/>
 但由于房产中介否认消息，李晨方也回应四合院是拍摄场地而非自家房产，加上受舆论影响，范冰冰持有股票的股价几乎〝腰斩〞，因此，网友推测：范冰冰筹措近9亿元的现款谈何容易？
 <br/>
 <br/>
 再加之范冰冰事件引发系列寒蝉效应，不仅诸多代言被取消，好莱坞电影《355》准备解雇她，连她客串的《大轰炸》也在17日证实取消上映，范冰冰的未来去向成谜。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：范铭）
</div>
