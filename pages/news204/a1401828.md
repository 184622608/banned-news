### 【禁闻】川习会结果公布 中共删美使馆微信评论
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】在布宜诺斯艾利斯G20峰会期间举行的美中首脑会谈落幕后，两国公布的〝谈判结果〞大不相同。很多大陆民众想了解会谈的真实内容，纷纷在美国驻华大使馆的微信公众号上留言和评论，但不少帖子被网管删除或屏蔽。
 <br/>
 <br/>
 彭博社报导说，美国驻华使馆在其微信公众号上登出的白宫声明的中文译文，也被删减了不少。同时，帖子的转发、分享、保存功能也被关闭。
 <br/>
 <br/>
 据海外〝中国数字时代〞网站披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 官方已经下令，只准大陆媒体用新华社通稿，不准推送自媒体相关内容，而且对所谓〝恶意炒作〞〝认怂论〞相关话题的，不仅要及时清理，而且还要对账号作出处理。
 <br/>
 <br/>
 <b>
  川普：中共将降低美国
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汽车关税.html" target="_blank">
   汽车关税
  </a>
 </b>
 <br/>
 <br/>
 在美中达成双方贸易战暂停90天的协议后，美国总统川普12月2号晚间表示，中方同意削减从美国进口汽车的关税。
 <br/>
 <br/>
 他在推文中说：〝中共同意对在美国生产、出口到中国的汽车降低并取消关税。目前的关税是40%。〞
 <br/>
 川普的这一消息，使汽车生产商的股票当天大涨4-7%不等，其中包括特斯拉、福特、宝马、奔驰等。
 <br/>
 <br/>
 编辑/周玉林
