### 韩媒：习近平着急解决贸易战 会晤文在寅漏提一事
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚太经合组织峰会.html" target="_blank">
  亚太经合组织峰会
 </a>
 期间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 与韩国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文在寅.html" target="_blank">
  文在寅
 </a>
 举行了会晤。韩媒称，习近平正着急解决中美贸易战，而文在寅则希望把握美中朝之间的平衡，双方达成了〝默契〞，均未提及缓和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对朝制裁.html" target="_blank">
  对朝制裁
 </a>
 的话题。
 <br/>
 <br/>
 11月17日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 出席了在巴布亚新几内亚首都——莫尔兹比港举行的亚太经合组织（APEC）会议。期间，中韩两国元首进行了单独会面。
 <br/>
 <br/>
 韩国《中央日报》11月19日报导，这次中韩首脑会晤，中方表现出跟以往不一样的〝气流〞，两国元首罕见地均未公开提及缓和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对朝制裁.html" target="_blank">
  对朝制裁
 </a>
 的话题。
 <br/>
 <br/>
 报导称，习近平正在着急解决中美贸易战，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文在寅.html" target="_blank">
  文在寅
 </a>
 则希望在美国与中俄的对立关系中，把握平衡，双方彼此达成了〝默契〞。
 <br/>
 <br/>
 在会谈中，习近平还表示，打算于2019年访问朝鲜和韩国。有分析指，习近平将访朝时间定在2019年而不是2018年，应该是对美国有所顾虑而做出的决定。
 <br/>
 <br/>
 美国总统川普（特朗普）此前曾公开指责，每次中朝首脑会晤之后，朝鲜的态度就会发生转变。川普嘲讽说，习近平给了金正恩一些〝糖球〞。
 <br/>
 <br/>
 最近，美国国会的美中经济与安全审查委员会的一份年度报告指出，北京已放松对朝鲜的制裁。北京总是为朝鲜维系〝关键命脉〞，在执法方面存有漏洞。
 <br/>
 <br/>
 该委员会还希望，美国财政部能将与朝鲜有商业往来的中国企业、团体和个人列为美国对朝制裁的目标。
 <br/>
 <br/>
 据外交消息人士透露，考虑到这一点，习近平不希望在第二次美朝首脑会晤之前访朝，刺激川普。
 <br/>
 <br/>
 近日，美朝双方就无核化和制裁问题发生意见冲突。11月7日，川普总统在谈及朝鲜问题时明确表示，在韩半岛实现完全无核化之前，将要求所有的印度、太平洋国家维持对朝制裁施压，并强调了美国〝先无核化，后缓解制裁〞的基调。
 <br/>
 <br/>
 川普说：〝无需操之过急〞，暗示将对朝进行〝长期战〞。
 <br/>
 <br/>
 川普的强硬态度令金正恩企图利用〝无核化〞欺诈美国的伎俩失效，朝鲜经济困境仍无法缓解，金正恩对此十分恼火，急令朝鲜官员赴中国大陆求援。与此同时，原定的美朝高级别会谈也被无限期延后。
 <br/>
 <br/>
 不过，朝鲜指望北京的愿望可能会落空，北京现在已自顾不暇。在贸易战火的猛烈冲击下，中共经济、政治均面临危机。
 <br/>
 <br/>
 近日，中方正在为即将在阿根廷举行的川习会晤做准备工作，日前还向美国提交了一份让步清单。不过川普认为，中方的这份清单仍缺少4、5个关键内容，他还不能接受。
 <br/>
 <br/>
 在此之前，11月1日，川普与习近平进行了一次长时间的通话，贸易问题是重中之重，其次，在朝鲜问题上也有沟通。
 <br/>
 <br/>
 这次通话后，北京对朝鲜的态度似乎发生转变。11月14日，韩国《中央日报》引述沈阳一名消息人士透露说，北京近期向与朝鲜有经济交易的中国企业下令，〝禁止给朝鲜‘送钱’〞。
 <br/>
 <br/>
 另外，美国副总统彭斯15日在新加坡参加东盟峰会时表示，川习会应该会提及对朝鲜制裁的事宜，并讨论北京能在执行对朝制裁方面扮演的〝独特角色〞。
 <br/>
 <br/>
 当天，彭斯还会见了文在寅，文在寅保证说，韩国会继续与美国合作制裁朝鲜。
 <br/>
 <br/>
 彭斯表示，〝川普总统希望达成全面的、可查证且不可逆转的无核化(CVID)，在实现这一目标之前，我们将会保持对朝鲜的施压力度，不会重蹈过去的覆辙。〞
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：文慧)
