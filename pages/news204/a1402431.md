### 孟晚舟自己辩护求保释   声称：为了祖国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】华为副总裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 保释一案日前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 法院开庭，孟晚舟自己为自己辩护，声称自己为了祖国，不会违反法庭命令。
 <br/>
 <br/>
 综合媒体报导，12月7日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 法院开庭受理了华为副总裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 保释一案，孟晚舟被指控犯有欺诈罪，如果罪名成立，将面临最高30年监禁。同时控方还表示，孟晚舟有潜逃嫌疑，要求法院拒绝让她保释。
 <br/>
 <br/>
 短暂休庭后，孟晚舟亲自为自己辩护。她声称，为了爱她的爸爸，为了华为全球的18万雇员，为了祖国，她保证不会违反法庭关于保释的所有要求，保证绝不越雷池一步。
 <br/>
 <br/>
 值得关注的是，孟晚舟的父亲、华为董事长任正非此前在公司内部讲话中也称，在应对贸易战方面，中国最大的武器就是13亿人民的消费，中美贸易战打不起来，应该会相互妥协。
 <br/>
 <br/>
 舆论认为，不管是孟晚舟〝为了祖国〞的借口，还是其父拿〝13亿人〞来挡枪的说法，实际上都是中共权贵利益集团面临困境时，煽动民族情绪的惯常动作，其真实用意是不惜以〝祖国〞的声誉和13亿人的利益为代价，来应对自身面临的危机。
 <br/>
 <br/>
 为争取保释，孟晚舟还向法官报告她在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_温哥华.html" target="_blank">
  温哥华
 </a>
 的房产，目前她丈夫与10岁的女儿在此居住。她还有另外三个儿子，最大的20岁。
 <br/>
 <br/>
 12月1日，加拿大应美国要求，在孟晚舟入境转机时，将其抓捕。美国指控孟晚舟涉嫌违反对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 的贸易制裁，正在寻求引渡她到美国受审。
 <br/>
 <br/>
 检方表示，2009年至2014年，华为涉嫌利用一家名为〝Skycom〞的子公司在伊朗开展业务，违反美欧对伊制裁。美司法部查证华为就是Skycom，Skycom就是华为的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白手套.html" target="_blank">
  白手套
 </a>
 〞，以此控告华为欺诈。
 <br/>
 <br/>
 同时检方还称，华为高管，包括孟晚舟在内，从2017年4月开始就不再前往美国，即使孟晚舟的儿子在波士顿上学，她也没去过。从中显示孟晚舟〝做贼心虚〞。
 <br/>
 <br/>
 加拿大检方表示，美国早在今年8月已下达逮捕令，请求加方予以合作，直到今年11月才获加国相关部门批准。
 <br/>
 <br/>
 据香港01报导说，美国商务部早于2016年，就公布了一份据称是中兴通讯的绝密文件，该文件详细介绍如何利用空壳公司中转等方式，规避美国的出口管制。
 <br/>
 <br/>
 文件披露，一家代号F7的中国公司也有类似做法。
 <br/>
 <br/>
 据报〝F7〞是中兴对华为的内部称呼，是〝夫妻〞的谐音。
 <br/>
 <br/>
 （记者欧阳静报导/责任编辑：戴明）
