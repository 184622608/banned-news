### 辽宁男驾奥迪冲撞小学生 致5死18伤（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】11月22日中午，辽宁省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_葫芦岛.html" target="_blank">
  葫芦岛
 </a>
 市建昌县第二小学校门前，发生重大交通事故。陆媒称，截止下午3点多，已有5人死亡，18人受伤。肇事司机在逃逸途中被抓获。网路上还曝出肇事司机被抓捕的正面图片。
 <br/>
 <br/>
 据陆媒消息，22日中午12点半左右，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_葫芦岛.html" target="_blank">
  葫芦岛
 </a>
 建昌二小门前，有大批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小学生.html" target="_blank">
  小学生
 </a>
 过马路，期间往来车辆都有停下，先让学生过路。但一部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥迪.html" target="_blank">
  奥迪
 </a>
 A6
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房车.html" target="_blank">
  房车
 </a>
 ，却没有停下，超速过线，直接撞向正在过马路的小学生及教师。
 <br/>
 <br/>
 现场拍摄片段显示，有10多名师生被撞飞倒地，现场地上有一些儿童的鞋子、杂物，还有血迹。事发后，伤者被送到附近医院抢救。
 <br/>
 <br/>
 <br/>
 <br/>
 该校工作人员介绍，当时正值学生放学，学前班一班学生，在排队过马路期间，一辆奥迪车突然驶向人群，造成多名学生受伤，〝可能有十几人，有轻伤有重伤，具体情况还不清楚〞。
 <br/>
 <br/>
 一名目击者介绍，事发路段当时在堵车，停了几分钟就感觉不对，有救护车和警车来往，就下车走过去看了看，发现地上有孩子的鞋子、血迹，旁边的人说一个人开车撞了孩子之后跑了。
 <br/>
 <center>
  <br/>

   <br/>

    肇事司机被警方抓捕现场。（微博图片）
   </font>

  <br/>
 </center>
 <br/>
 下午两点多，警方称，目前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_撞人.html" target="_blank">
  撞人
 </a>
 男子已被抓获，该男子撞倒多名学前班儿童后，驾驶肇事车辆奥迪A6向乡下逃跑，途中被抓获。网路上曝出的肇事司机被抓捕的现场图片可见，肇事司机是一名30来岁的男子。
 <br/>
 <br/>
 党媒称，截止下午3点多，已有5人死亡，18人受伤，事故原因正在调查中。
 <br/>
 <br/>
 <h4>
  各地恶性事件频发
 </h4>
 <br/>
 事实上，上述冲撞小学生事件并非偶然事件，近年来，在中国各地频频发生开车撞人恶性事件。
 <br/>
 <br/>
 9月12日晚，湖南衡东县发生一起故意驾车伤人恶性案件，官方公告称，截止13日上午10点，已有11人死亡，44名伤者仍在医院治疗。网传至少20死。据悉，驾车男子因身患癌症，泄愤报复社会。
 <br/>
 <br/>
 8月30日，广西南宁市桃源路一辆面包车疑似失控，直接冲撞正在等红绿灯的摩托车人群，造成7人死亡，19人受伤。
 <br/>
 <br/>
 8月29日。贵州省织金县因强拆发生流血冲突，当地村民抵抗政府人员强拆房屋，驾车撞人，造成了3人死亡，10人受伤。
 <br/>
 <br/>
 8月20日，广西柳州市文昌大桥发生一起血腥袭击事件，一男子疑因感情纠葛杀死女友及其3名家人后，又驾车在闹市冲撞路人和持刀砍人，造成6人死亡12人受伤。
 <br/>
 <br/>
 6月25日上午，山东烟台一名男子驾驶叉车在路上反覆冲铲多辆车以及路人，至少造成1人身亡，10余人受伤，多辆车被毁。肇事男子被当场击毙。
 <br/>
 <br/>
 6月6日，河北唐山市一名男子开车撞马路上的行人。事故造成4死11人伤。民间目击者称，肇事男子〝见人就撞〞，还〝来回碾压五六次〞。
 <br/>
 <br/>
 4月30日晚8点左右，湖北荆门市京山县一名男子，驾车在闹市区高速行驶，且来回兜圈乱撞，最后直接冲撞路边一处人群，造成1路人及1警员身亡，3人受伤。
 <br/>
 <br/>
 2016年2月29日中午，河南南阳市第一中学门口，一辆白色比亚迪越野汽车撞向放学回家的学生，造成1名学生死亡，11名学生受伤。肇事者为南阳检察院正处级退休干部。
 <br/>
 <br/>
