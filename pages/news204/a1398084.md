### 进博会：高官陪习见外企代表 外界指老套路
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】上海首届〝国际进口博览会〞5日开幕，习近平出席开幕式并会见了境外企业家代表。中共国家副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡春华.html" target="_blank">
  胡春华
 </a>
 等人陪同习参与了会见。习强调将为境外企业赴华投资提供好的环境。致辞中则强调扩大开放，但外界认为老套，并无实质内容。
 <br/>
 <br/>
 党媒报导，11月5日，首届〝国际进口博览会〞（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 ）在上海开幕，中国国家主席习近平，会见了参加进博会的境外企业家代表。 习对这些代表们表示欢迎，强调将为境外企业赴华投资提供环境。
 <br/>
 <br/>
 中共办公厅主任丁薛祥、上海市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李强.html" target="_blank">
  李强
 </a>
 、中共外事委办公室主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨洁篪.html" target="_blank">
  杨洁篪
 </a>
 、中共国家副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡春华.html" target="_blank">
  胡春华
 </a>
 、中宣部部长黄坤明、中共外长王毅、中共公安部长赵克志、中共国家发改委主任何立峰等参与了会见。
 <br/>
 <br/>
 习近平5日早上出席进博会开幕式，做了半个小时的主旨演讲，习说，各国应该坚持开放融通，拓展互利合作空间，中共开放的大门不会关闭，只会越开越大。
 <br/>
 <br/>
 习近平也坦承，各国削减壁垒、扩大开放，国际经贸才能打通血脉；如果以邻为壑、孤立封闭，国际经贸就会气滞血瘀，世界经济也难以健康发展。各国应该坚持开放的政策取向，推动各国经济联动融通，共同建设开放型世界经济。
 <br/>
 <br/>
 习再次强调在经济全球化深入发展的今天，包容普惠、互利共赢才是越走越宽的人间正道。
 <br/>
 <br/>
 外界认为，面对各国对中共不公平贸易行为的指责，尤其是在中美贸易战的重压之下，北京欲藉进博会，打造一个进口大国的〝新〞形象，以缓解各方压力，避免进一步被孤立。
 <br/>
 <br/>
 但习的讲话被认为还是延续了过往空泛的风格，虽然表面上称，要在〝激发进口潜力〞、〝持续放宽市场准入〞、〝打造对外开放新高地〞等方面〝进一步扩大开放〞，但并无实质内容。
 <br/>
 <br/>
 彭博社在习讲话的前一天，已在文章预测说，习欲通过举办〝进博会〞以化解贸易摩擦的想法，在诸多方面体现了中共政策和外界预期之间的差距，面对盗窃知识产权和繁琐的合资要求等批评，中共从没有正面回应，给出的回答似乎总是〝买更多〞，但这只是老套路。
 <br/>
 <br/>
 在川普点燃中美贸易战火后，对中共使用了孤立政策。不仅七大工业国集团美国、加拿大、英国、法国、德国、意大利及日本的政要无一出席此次〝进博会〞，韩国、澳大利亚等中共主要贸易国家的政要也拒绝出席。
 <br/>
 <br/>
 中美贸易战已经持续近4个月，中共目前的经济政策矛盾重重、方向不清，习近平的讲话，也难以让人对中共的改变产生信心。德国BDI工业联盟的一份最新战略论文，呼吁企业减少对中国市场的依赖，显示人们对中共由政府主导的经济模式的担忧与日俱增。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉)
