### 首例基因编辑宝宝在中国诞生 科学界批〝打开魔盒〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】全球首例基因编辑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_婴儿.html" target="_blank">
  婴儿
 </a>
 ，11月份在中国大陆诞生。这对双胞胎的基因经过修改。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科学界.html" target="_blank">
  科学界
 </a>
 强烈批评相关做法存在巨大风险。
 <br/>
 <br/>
 南方科技大学副教授贺建奎日前宣布，他使用俗称〝基因剪刀〞的CRISPR-Cas9技术，为2名试管
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_婴儿.html" target="_blank">
  婴儿
 </a>
 修改了CCR5基因。CCR5基因被认为是艾滋病毒入侵机体细胞的主要辅助受体之一。敲除该基因后，据称能让婴儿天然免疫爱滋病。经过基因编辑的双胞胎露露和娜娜，11月已在中国大陆出生。
 <br/>
 <br/>
 消息一出，立刻引发全球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科学界.html" target="_blank">
  科学界
 </a>
 的强烈批评。
 <br/>
 <br/>
 我们不知道这个基因还有什么其它功能。没准这个基因缺失了，可能就没有其它免疫能力了，染其它病。我们不知道，只有时间来证明了。
 <br/>
 <br/>
 基因剪刀技术，2012年由美国科学家发现，目前处于早期研究阶段。由于一直存在脱靶效应（即误剪切其他基因），加上无法预知的负作用，因此各国科学家都谨慎使用。
 <br/>
 <br/>
 目前不清楚基因剪切的准确度，所以会担心，你把不想剪的地方剪掉了，把非常重要的基因灭活了，或可能改变其它基因表达，可能导致癌症等其他问题。
 <br/>
 <br/>
 横河：这个人工改造的基因可以遗传给后代。也就是说，这个人造的基因，有可能进入人类的基因池。人类所有基因在一起叫基因池。后果是非常严重的。改造了这个基因以后，对人会有什么影响，基本上是不知道的。太多的未知，还有太多人类根本无法控制的因素，进入了人类的基因池。
 <br/>
 <br/>
 欧美多国科学家强烈批评，大陆科学家冒然将该技术用于人体，并且操作过程不透明，让同行难以评估审议。更严重的是，项目带来的伦理危机，如同“打开了潘朵拉魔盒”。
 <br/>
 <br/>
 横河：在伦理道德上，就是人类现在可以〝定制自己〞了。马上就带来一个问题，由谁来决定，什么人，在什么情况下，可以以什么样的理由，设计出来一个什么样的目标的人？这一系列的问题，对于信神的人来说，是神的事情。现在人把神的事情接到自己手里去。也就是说，人能决定什么样的人不能出生，什么样的人能出生，改造成什么样子，这太可怕了。
 <br/>
 <br/>
 据〝中国临床试验注册中心〞数据，贺建奎的基因编辑婴儿试验，主办单位是〝南方科技大学〞与〝和美妇儿医院〞；医学伦理审议由〝深圳和美妇儿科医院伦理委员会〞通过；经费物资来源是〝深圳市科技创新自由探索项目〞。
 <br/>
 <br/>
 面对外界的强烈批评，上述三家目前都撇清和该项目的关系，声称〝毫不知情〞。
 <br/>
 <br/>
 横河：贺建奎为什么敢这样做？他有一个大的环境，第一，中共统治以后强制推行无神论，人对神没有敬畏之心；第二，中共彻底的唯物主义，它把人当成一种物质的组合状态，不是把人当成是神造的。那么同样它对人也就没有丝毫尊重。
 <br/>
 <br/>
 近年来，大陆医学伦理问题多次引起巨大争议，例如全球首例换头手术，克隆猴，以及依靠来源不明的器官发展移植技术等。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
