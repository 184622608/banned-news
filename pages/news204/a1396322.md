### 一路比拼习近平失败 薄熙来泄愤骂习是阿斗
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】重庆事件曝光了周、薄集团的政变计划后，引爆了中共高层的公然分裂与权斗。多家外媒还披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 一路比拼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，一直都很失败。为此，薄熙来泄愤骂习近平为阿斗。
 <br/>
 <br/>
 2012年2月，原重庆副市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 逃入美成都领馆后10天，在当时中共总书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 召集下的一次政治局常委会，针对是否追究
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 关于此事政治责任投票，在4人赞成、4人反对情况下，当时正在美国访问的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 投下关键赞成票，薄熙来政治生命自此告终。
 <br/>
 <br/>
 据日本朝日新闻称，除习近平外，当时投票支持追究薄责任者包含胡锦涛、温家宝、李克强、贺国强，反对者包含吴邦国、贾庆林、李长春、周永康。
 <br/>
 <br/>
 报导说，包括高干子弟在内的数名中共党内人士一致认为，如果习近平在这个问题上不点头的话，其实是很难追究薄熙来责任的。
 <br/>
 <br/>
 报导还透露，2007年6月25日下午3时，也就是中共十七大前夕，400多名中共高级干部，在北京举行了一次非公开的信任投票，目的是考察将来有可能进入中共最高决策层、年龄在63岁以下的干部的能力。
 <br/>
 <br/>
 据悉，这种形式的〝党内民主〞尚属首次，胡锦涛主持了会议。
 <br/>
 <br/>
 投票结果至今未对外公布。报导援引一名中共党内人士透露，当时的投票引发了一场风波。江派看好的时任商务部部长薄熙来，得票情况非常差，特别是军队内干部对其评价甚低，而习近平得票则位于前列。
 <br/>
 <br/>
 据称，这一结果让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 十分心慌，最终选择习近平成为胡锦涛的接班人。
 <br/>
 <br/>
 <h4>
  薄熙来骂习近平为阿斗
 </h4>
 <br/>
 对于自已落选中共接班人选，薄熙来心怀不满，更看不起胡锦涛和习近平，曾经口出狂言。在2012年2月6日，薄熙来的打手、重庆前公安局长王立军出逃美国成都领事馆后，薄辱骂胡锦涛和习近平的事情曝光。
 <br/>
 <br/>
 薄熙来曾经骂胡锦涛是〝汉献帝〞（娇生惯养，傀儡），骂习近平是〝刘阿斗〞（扶不起的君王）。
 <br/>
 <br/>
 据港媒披露，早在上世纪九十年代，时任大连市长的薄熙来就看不起胡锦涛和温家宝，并骂他们不过只是代替红二代看守江山的家奴。他也瞧不起习近平，认为习没有霸气，过于柔弱，未来势必收拾不了各路家奴。
 <br/>
 <br/>
 据报导，薄熙来作为欠有迫害法轮功血债的江派大员，早年仍被江泽民秘密选为中共接班人，主要是为了继续掩盖江十多年来对法轮功学员，包括活摘器官在内的残酷迫害真相，延续其迫害政策。
 <br/>
 <br/>
 江泽民惶恐失去核心权力后，一旦无法维持迫害将遭到清算。因此，在中共〝十六大〞退下前，江就开始布局以后的人事安排，以逃避清算。
 <br/>
 <br/>
 江曾的算盘是先在2007年阻止胡锦涛的接班人上台，在2007年-2012年期间内，让江曾真正的接班人薄熙来锻炼成熟、取得威望和权势，在2012年的十八大上，至少得到常委和中央政法委书记的位置。
 <br/>
 <br/>
 江曾还预计在十八大后，再经过二年左右的时间，利用薄熙来在全国通过〝唱红打黑〞取得的对于全国的挟持和操控，把〝重庆模式〞推向全国。
 <br/>
 <br/>
 然后再利用薄熙来掌控的全国政法委等强力政权、武警部队、以及全国众多被薄熙来掌握的军队、江泽民在军中的力量等，采取邓小平废黜华国锋等方式，罢免甚至逮捕习近平等人，到时候全国又是江曾的天下。
 <br/>
 <br/>
 但是，江曾没有想到的是自己布局多年的阴谋，却在〝十八大〞前，因王立军闯入成都美领馆，使得这一计划在执行到一半的时候全盘崩溃，最终功亏一篑。
 <br/>
 <br/>
 此政变计划因为牵涉诸多的党政军要员，以及许多内幕，成为中南海最大的〝炸弹〞。直到现在，由此事件所引发的中南海震荡仍在持续中。
 <br/>
 <br/>
