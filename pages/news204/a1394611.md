### 专家解读降准：08年后最糟 中共真的紧张了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 持续升温，中共央行日前宣布降准的消息，大陆股市应声下跌。有专家认为，目前中共正面临2008年全球金融危机以来最糟时刻。从降准的决定来看，中共是真的紧张了。
 <br/>
 <br/>
 <h4>
  中共做最坏打算
 </h4>
 <br/>
 中共央行10月7日宣布降准的消息，将从本月15日起，调降人民币存款准备金率1个百分点，这是今年以来第四次降准。估计可释出7500亿人民币的资金。外界预计央行此举将导致人民币进一步贬值。
 <br/>
 <br/>
 对于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共央行降准.html" target="_blank">
  中共央行降准
 </a>
 的消息，新加坡银行固定收益主管尼克森（Gareth Nicholson）对CNBC分析，虽然中共多次宣称〝不怕打〞贸易战，但从降准的决定来看，中共是真的有些紧张了！
 <br/>
 <br/>
 尼克森指出，〝若美国经济持续表现强劲、贸易战持续升温，将会导致更多投资人把资金撤离中国市场。〞中共正在做最坏的打算。
 <br/>
 <br/>
 在降准消息公布后，中国股市应声下跌。10月8日，大陆A股沪指、深成指、创业板三大股指齐跌，均超过3%，逾3200只个股下跌，沪指收创三个半月最大单日跌幅，外资流出近百亿人民币。
 <br/>
 <br/>
 独立金融分析师豪伊（Fraser Howie）认为，中共目前正面临2008年全球金融危机以来最糟的时刻，所有新闻对他们都是负面的。尽管中共正极力控管可能引发恐慌的言论，但它们很清楚，中国并非像往常一样了。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币贬值.html" target="_blank">
   人民币贬值
  </a>
 </h4>
 <br/>
 虽然中共央行表示本次降准不会对人民币形成贬值压力。但是大陆金融界人士却对此不以为然。
 <br/>
 <br/>
 交银国际董事总经理洪灏认为，降准加强了人民币贬值压力。
 <br/>
 <br/>
 海外评论人士文小刚认为，现在美中贸易战升级，大陆出口遭受打击，制造业等实体经济前景不妙，央行此次降准向市场投放7500亿流动性，对人民币产生贬值压力，也从侧面显示中国经济确实在下滑。
 <br/>
 <br/>
 事实上，在中共央行宣布降准之前，已经有很多国际投资机构预测人民币今年底或明年初还将贬值。国际投行摩根大通在9月底的报告中表示，如果美国在2019年对所有中国商品加征25%关税，预计人民币汇率在今年12月底将贬至7.01，并在2019年9月下探7.19。
 <br/>
 <br/>
 另据钜亨网10月2日的消息显示，德意志银行（Deutsche Bank AG）认为，人民币汇率会在明年一举跌至7.4。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国楼市.html" target="_blank">
   中国楼市
  </a>
 </h4>
 <br/>
 此外，降准不仅对股市、汇率造成影响，与中国楼市也息息相关。
 <br/>
 <br/>
 随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 开打，不少经济学者预测，中国大陆的房市泡沫面临被刺破的危机。近期，大陆多地已出现楼盘暴跌现象，环北京的燕郊楼价跌破一半，从去年的每平米3万元人民币，跌至现在的1万5千元左右。
 <br/>
 <br/>
 万科、恒大、碧桂园、泰禾、中粮等地产商，也都开始大肆降价促销，脱手楼盘。万科在厦门万科白鹭郡别墅推出107套特价别墅，原价500万人民币，现中间套一口价278万元，边套一口价298万元。
 <br/>
 <br/>
 因楼市暴跌，安徽合肥、湖南长沙、北京通州、江苏杭州等地都有业主因房价下跌而维权。
 <br/>
 <br/>
 更令人吃惊的是，大陆地产业领头羊之一的万科近期喊出，未来以〝活下去〞为最终目标的口号。显示大陆房地产业已进入寒冬，悲观情绪正在地产界蔓延。有消息称，政府已下令各地产商尽快减少库存。
 <br/>
 <br/>
 而中共央行在此刻突然宣布降准，有大陆金融专家分析，这只能是让楼市不那么快崩盘而已。因为降准会导致人民币贬值，带来通货膨胀，最终导致物价上涨，使楼价不至于狂跌。因为房价牵扯的东西太多，只能缓慢〝阴跌〞。不过，未来楼市降价几乎是无法避免的，而为此买单的又是中国的13亿老百姓。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：祝馨睿)
