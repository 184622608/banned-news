### 孟宏伟同党疑浮出 新媒：周永康非孟案主因
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共公安部.html" target="_blank">
  中共公安部
 </a>
 前副部长、国际刑警组织前主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 落马一事，引发国际关注和热议。日前，外媒有分析指，孟宏伟落马，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 或非主要因素，而孟的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_同党.html" target="_blank">
  同党
 </a>
 或将浮出台面。
 <br/>
 <br/>
 新加坡《联合早报》援引有关人士分析称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 在国际刑警组织（Interpol）主席任上突然失踪，必将引起国际舆论关注，而且，孟的妻子目前仍在法国，并已举行记者会为孟宏伟喊冤。这些后果，中共官方不可能不了解，但还是突然羁押了孟宏伟，显示孟宏伟案可能事发仓促，官方担心走漏消息，于是决定对其采取紧急措施。
 <br/>
 <br/>
 分析人士还指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共公安部.html" target="_blank">
  中共公安部
 </a>
 10月8日凌晨举行紧急会议，通报孟宏伟案情，也说明孟宏伟案来得比较突然。至于外界盛传孟宏伟因与中共前政治局常委、政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 关系密切而被查，这个因素虽然不排除，但不会是主要原因。
 <br/>
 <br/>
 分析人士称，因为周永康2015年6月被判无期，与他接近的官员早已被筛查多次。如果孟宏伟真是周永康的亲信，他不大可能在周被判刑后，还长期担任公安部副部长，尤其不会担任有可能在国外滞留不归的国际刑警组织主席。
 <br/>
 <br/>
 此外，8日凌晨的中共公安部党委会议强调，要〝组织工作专班，依法查处涉嫌与孟宏伟共同收受贿赂人员〞。该分析人士认为，这显示孟宏伟并非一个人受贿，而且这些人可能就在公安系统内部，因此要由公安部〝组织工作专班〞进行查处。
 <br/>
 <br/>
 旅美时评人、经济学者夏业良对〝美国之音〞表示，孟宏伟事件说明肯定是发生了大事情，甚至有传言称他参与推翻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的政变。他指出，正常情况下，中共一般会使用常规手段来控制官员及其家属，但是本次显然等不及孟宏伟正常回国休假以便将其妻儿一并控制。因此，他认为，肯定发生了跟习近平高层有关的、意料之外的重大事件，官方才必须紧急出手。
 <br/>
 <br/>
 时事评论员陈思敏分析提到，在通稿第四、第五段落的内容中，关于孟宏伟〝收受贿赂〞的问题还有值得注意的表述，分别是提到要〝严格要求亲属及身边工作人员〞，以及〝组织工作专班，查处涉嫌与孟宏伟共同收受贿赂人员〞。所以据此推测，不管〝共犯〞最终是否会公开落马，孟宏伟案应该是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_窝案.html" target="_blank">
  窝案
 </a>
 〞无疑。
 <br/>
 <br/>
 陈思敏指出，会议通稿最后列出参加会议的人员，是王小洪、邓卫平等9名在京公安部党委委员，独缺公安部部长助理林锐。在这场赵克志紧急召开的重要会议中，唯独林锐缺席。林目前不在北京，或许有相当紧急重要任务，但也不排除与孟宏伟案有关。
 <br/>
 <br/>
 此外，公安部会议通稿开头及通篇照例强调〝习核心〞，并明言〝绝不允许跟组织讲条件、讲价钱〞，但末段总结夹杂了一句〝严密防范各种捣乱破坏颠覆活动〞，似乎意有所指，很难不让人产生联想：既然孟的〝老领导〞周永康已经入狱服刑，是否另有相当级别的〝老领导〞关照，才让孟〝讨价还价〞？拿下孟宏伟，是否也是为了震慑其后台？
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
</div>
