### 【禁闻】10月23日退党精选
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】退出中共组织的消息在大陆民众中口耳相传，包括中共基层党委干部纷纷声明退出中共各级组织。来看他们的声明。
 <br/>
 <br/>
 陈军说：我退休前是一名基层党委的专职副书记，虽然我也不信共产党的那一套，但是很多事情身不由己，但是没有参与过迫害法轮功。
 <br/>
 <br/>
 我的表弟和他太太都是法轮功学员，他们都是名牌大学毕业，曾经有令人羡慕的工作，后来因为炼法轮功被迫害。我现在决定声明退出曾经加入过的共产党、共青团和少先队。
 <br/>
 <br/>
 我太太李祥（化名）只参加过少先队。她也要退出少先队。
 <br/>
 <br/>
 辽宁化名〝明智〞的民众说：我在省政府机构工作，深知共产党的腐败现象。对共产党彻底失望。共产党一定会完蛋。所以声明退出以前加入的共青团组织。
