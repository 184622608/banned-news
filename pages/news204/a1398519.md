### 中纪委点名：副省级官喝死陪酒人 26军军长喝死下属
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】11月7日，中纪委机关报刊文盘点习近平前5年任期内查办的违反中共官场〝铁规〞的典型案件。其中黑龙江省副省级官员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_付晓光.html" target="_blank">
  付晓光
 </a>
 被点名。付当年曾因公款消费，大量饮酒并造成陪酒人员死伤，而类似事件在中共官场层出不穷。
 <br/>
 <br/>
 11月7日，中纪委机关报再刊文盘点了2012年11月至2017年10月查办的违反中共〝铁规〞的典型案件。回顾十八大之后的5年间腐败典型案例。
 <br/>
 <br/>
 其中，黑龙江副省级官员被点名。2013年12月17日，中纪委监察部网站，发布10起违反中央八项规定精神（习八条）的典型事件，第一例是黑龙江省副省级官员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_付晓光.html" target="_blank">
  付晓光
 </a>
 因私公款消费，大量饮酒并造成陪酒人员〝一死一伤〞案件。
 <br/>
 <br/>
 据报，同年7月23日，付晓光带领包括亲属在内的6人到镜泊湖旅游，当晚，该局党委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟庆安.html" target="_blank">
  孟庆安
 </a>
 、局长孙书功，在景区的鹿苑岛宾馆设宴款待付晓光。孟庆安酒后心脏病发作死亡。另一人重伤。
 <br/>
 <br/>
 事发后，中纪委给予付晓光留党察看一年处分免去其职务，由副省级降为正局级。 付晓光成为了第一个违反〝习八条〞被点名的省部级官员。
 <br/>
 <br/>
 文章称，喝酒喝死人案例反映的中共党员干部公款大吃大喝、奢靡享乐问题，5年间，各级纪检监察机关查处了一批党员干部大吃大喝、奢靡享乐问题。
 <br/>
 <br/>
 中共官场的〝酒文化〞堪称登峰造极，民间有首打油诗，〝领导干部不喝酒，一个朋友也没有；中层干部不喝酒，一点信息也没有；基层 干部不喝酒，一点希望也没有。〞
 <br/>
 <br/>
 〝喝酒就是工作〞成为了中共官员座右铭，不会喝酒意味着仕途走到了尽头，于是诞生了〝舍命陪君子〞的〝陪酒一族〞，同时，〝陪酒致死〞的案例不断发生着。
 <br/>
 <br/>
 2016年1月微信公号〝长安街知事〞披露了〝致命饭局〞的案例。
 <br/>
 <br/>
 其一是：2015年12月31日晚，广东省潮州市公路局领导班子接受该局后勤人员宴请。据报导，酒过三巡后该局副局长王兰亭站在山崖边小便时滑落致严重受伤，该局总工程师黄某上前查看后摔落山崖当场死亡。
 <br/>
 <br/>
 事件导致多名官员被处分，其中党组书记、局长和一位副局长被行政撤职，降为正科级。
 <br/>
 <br/>
 其二是，中共国防部新闻发言人杨宇军2015年12月底证实，中共26集团军
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军长张岩.html" target="_blank">
  军长张岩
 </a>
 在军部与第39集团军的两名老部下喝酒，导致一人死亡。有关人员受到严肃查处，张岩被撤职，由正军降到副军免职。
 <br/>
 <br/>
 杨宇军说，这是一起中共〝十八大〞后〝不收敛不收手、顶风违纪〞的典型案件。
 <br/>
 <br/>
 其三是：2015年4月29日晚，永州市市长向曙光等接受由零陵机场迁建工作领导小组办公室承办的公务接待用餐，当晚14人共消费白酒、菜品合计7000余元，参加用餐的零陵区发改委副主任李春燕当晚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非正常死亡.html" target="_blank">
  非正常死亡
 </a>
 。事后，向曙光被免职。
 <br/>
 <br/>
 报导说，〝致命饭局〞出现的一个重要原因，在于买单的或是公款，或是有事相求的老板，赴宴的领导干部无需自掏腰包，当然胡吃海喝起来刹不住车。
 <br/>
 <br/>
 报导还提及了曾在网路流传的有关王岐山如何拒绝劝酒的段子，说向老王看齐一点，就不会身陷〝致命饭局〞。
 <br/>
 <br/>
 据王岐山的老友黄江南回忆，80年代他与老黄一起到河南省某县，县里的领导招待他们。当地有〝灌酒〞之风，餐桌上一位办公室主任拿起一杯酒顶在头顶，跪在地上对王岐山说，〝到我们这来，这酒一定要喝，你不喝我就不起来。〞
 <br/>
 <br/>
 而王岐山说，〝你喜欢跪着就跪着吧〞，并继续和其他吃饭的人谈笑风生，留他一个人跪那。最后这个办公室主任只得起身。
 <br/>
 <br/>
 据港媒报导，王岐山出任中纪委书记后，几乎不去任何饭局，而是邀请朋友们到家里做客，开伙煮饭，通过这种形式与朋友们交往。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
