### 江西赣州强拆房　打伤村民多人
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】10月14号上午，江西省赣州市定南县天九镇兴隆上村大河迳组，发生一起暴力
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 事件，地方官员带领七、八十人，强拆五间
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老房子.html" target="_blank">
  老房子
 </a>
 ，殴打老人和妇孺，有六、七个人受伤住院，其中两个人骨折。
 <br/>
 <br/>
 据伤者侄子杨先生透露，14号，他们接到15号要拆房子的通知，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 的理由是房子太旧不好看，而且没有一点补偿。强拆时不让搬东西，所有东西都被砸毁在里面，打伤六、七个人住院，其中有三个老人，一个70多岁，一个60多岁，一个50多岁。
 <br/>
 <br/>
 伤者侄子杨先生：〝省政府来了七、八十个人，要强拆这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老房子.html" target="_blank">
  老房子
 </a>
 ，我们家里的兄弟就是说，稻谷呢、其他的要搬出来，不让他们搬，拉了我兄弟啊，我叔叔他们就打，有六、七个人受伤了，县里住院，有两个人骨折。房子被他们拆掉了，那个稻谷也没抢出来，稻谷，还有很多东西也没抢出来。 〞
 <br/>
 <br/>
 当地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_村民.html" target="_blank">
  村民
 </a>
 表示，强拆的五间老房子，是农村的小仓库，其中一个还在种田 的76岁老人，有一千多斤稻谷，不让搬出来，都埋在里面了。
 <br/>
 <br/>
 村民李女士:〝拆的房子是属于一个储存小小的仓库，不是很烂，里面还很多稻谷啊、农具，走进去说不要拆那么快，他说你敢进去就打你，七十多岁的人，把他按在地上打，几十个人一起上来打他们，真的我看到就怕，那个场面。 〞
 <br/>
 <br/>
 当地村民还透露，村里所有当仓库的老房子都遭到强拆，村民住的房子太老旧的，说会影响村容，也会被强拆，逼得村民筹钱翻修房子。
 <br/>
 <br/>
 新唐人记者熊斌、李佩灵采访报导
</div>
