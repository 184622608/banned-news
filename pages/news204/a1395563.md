### 贸易战中寻出路？港媒：王岐山将率团出访以色列
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 愈演愈烈，由美国主导的国际联盟正在对中共进行全方位封堵，中共被迫寻找别的出路。港媒日前报导称，中国国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 近日将率团访问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 ，参加第四届以色列〝创新峰会〞，北京希望从以色列等其他国家，获得被美国封杀的高新技术。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 目前仍无缓和迹象，相反川普（特朗普）总统采用各个击破的高招，和欧盟、日本、墨西哥以及加拿大达成协议，由美国主导的国际联盟正在对中共展开围剿之势。
 <br/>
 <br/>
 与此同时，美国近期采取一系列措施，加强限制中共以各种形式接触美国军事和高新技术。如限制中共对美国高新产业的投资、严控中共对美国学术领域的渗透，并严厉打击盗窃美国军事和高科技技术的中共间谍。
 <br/>
 <br/>
 面对美国的封杀，〝自主创新〞能力薄弱的中共，只得被迫寻找其他出路。
 <br/>
 <br/>
 据香港信报10月15日报导，中国国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 将率领代表团，参加10月24日至25日，在耶路撒冷举办的第四届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 〝创新峰会〞。王岐山或将成为十多年来访问以色列的中共最高级别官员。以色列总理内塔尼亚胡(Benjamin Netanyahu)将出席两天的活动。
 <br/>
 <br/>
 报导称，王岐山此行还有一个目的，希望从以色列等其他国家，获得被美国封杀的高新技术。
 <br/>
 <br/>
 以色列〝创新峰会〞旨在突出以色列在数字健康、粮食和农业技术以及网路安全领域的技术创新。来自美国、意大利、法国、澳大利亚、韩国、墨西哥、新加坡、香港、德国、瑞士等国的代表团将出席峰会。谷歌、脸书、IBM等国际网路公司大佬也将出席峰会，阿里巴巴创始人马云也将出席并发表主旨演讲。
 <br/>
 <br/>
 王岐山最近被任命为中以创新合作联合委员会的中方负责人，内塔尼亚胡则是以方负责人。该委员会于2014年5月成立，每年举行一次会议，旨在促进中以在数个创新领域里的合作。
 <br/>
 <br/>
 随着贸易战开打，美中关系急转直下，从此前的〝接触合作〞，转变为〝全面对抗〞。目前，美国国防法案已经明确将中共视为战略竞争对手，从经济、军事、政治、外交等层面对中共进行全面反制。
 <br/>
 <br/>
 10月4日，美国副总统彭斯在智库哈德逊研究所发表对华政策演说，全方位严厉抨击中共。舆论界普遍将其视为美国对中共〝新冷战〞的一篇檄文。
 <br/>
 <br/>
 政论分析人士夏小强表示，〝美国已从政治、经济、军事全方位地向中共发出宣战的态势和讯号。预计接下来美国的中期选举共和党将获得胜利，川普将能顺利开展他所有的施政和决策，川普对中共的打击将加速进行。〞
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
