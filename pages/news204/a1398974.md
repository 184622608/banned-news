### 中美争端升级 美首次要求中方撤南沙导弹
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】中美贸易争端尚未缓解，南海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事对抗.html" target="_blank">
  军事对抗
 </a>
 又成为新焦点话题。日前，双方在中美第二轮外交安全对话上针锋相对，随后，美国发布声明，首次要求中方撤除南沙群岛的导弹。有分析指，早在4年前，中方就向美国承诺不会对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海军事化.html" target="_blank">
  南海军事化
 </a>
 ，美国此次的要求只是要中共兑现承诺。
 <br/>
 <br/>
 11月9日，中美第二轮外交安全对话在华盛顿召开，中共外事工作委员会办公室主任杨洁篪、中共国防部长魏凤和，与美国国务卿蓬佩奥（Mike Pompeo）及国防部长马蒂斯（James Mattis），在会面后召开联合记者会。
 <br/>
 <br/>
 中美双方此次对话火药味浓烈，香港《南华早报》报导，在联合记者会上，杨洁篪要求美国停止在南海派遣军舰和军机。
 <br/>
 <br/>
 马蒂斯则当面严词驳回，表示美国会继续在国际法允许的任何地方飞行、航行和行动。同时，蓬佩奥也对中共在南海岛屿进行军事化部署提出批评。
 <br/>
 <br/>
 随后，美国国务院发表声明，要求中方撤除部署在南沙群岛的导弹，这是美国首次就南海问题向中共提出此要求。有分析指，中共早在2015年就向美国承诺，不会对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海军事化.html" target="_blank">
  南海军事化
 </a>
 ，美国此次的要求，只不过是要中共兑现承诺。
 <br/>
 <br/>
 然而，中共口头上承诺不会对南海军事化，行动上却在南海岛屿开始部署导弹。美国对此深表忧虑。
 <br/>
 <br/>
 2016年2月17日，美国国务卿克里表示，〝每天都有各种证据显示，军事化活动不断在增加，令人深感担忧。〞
 <br/>
 <br/>
 而当时的中共外长王毅却辩称，中共在自己岛礁建立自卫设施，与军事化无关。
 <br/>
 <br/>
 直到今年5月，美国情报部门的报告指出，中共在4月或5月初已于南沙群岛的永暑礁、渚碧礁和美济礁上，部署反舰导弹和防空导弹系统。据悉，当中两款导弹为鹰击12反舰导弹及红旗9中远程防空导弹。
 <br/>
 <br/>
 6月2日，美国国防部长马蒂斯在香格里拉亚洲安全论坛上，指责中共在有争议的南海岛礁部署反舰导弹、防空导弹、电子干扰设备、轰炸机等一系列武器，其目的是〝恐吓与胁迫〞周边邻国。
 <br/>
 <br/>
 同日，美国总统川普（特朗普）也罕见就南海问题发推文，称对中共的做法〝非常惊讶〞。
 <br/>
 <br/>
 今年9月，美国派遣军舰到南海执行〝自由航行〞任务时，与中共军舰发生〝不安全互动〞，两国军舰差点相撞。事后，美方对此表示强烈谴责，并表示美军将继续在南海执行〝自由航行〞任务。
 <br/>
 <br/>
 种种迹象显示，美国不仅在经济领域遏制中共的侵入式贸易，在军事上也开始采取强硬对抗手段。纽约时报称，澳大利亚国立大学(Australian National University)南海问题专家布伦丹．泰勒(Brendan Taylor)认为，中美两国在南海爆发军事冲突，只是时间问题。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
