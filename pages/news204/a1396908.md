### 中共启动〝杀手鐧〞？加码贸战稀土减产36%
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】中美贸易战开打以来，对美禁售稀土一直被中共炒作成〝杀手鐧〞，但该说法已被戳破其中水分。日前英媒披露北京确已开始限制稀土产量。虽然党媒再次祭出〝吓尿体〞报导炒作，但海外舆论几乎一边倒的吐槽讥讽。
 <br/>
 <br/>
 路透社日前报导，密切追踪稀土产业的独立研究服务商Adamas Intelligence发布的数据显示，中共已把今年下半年稀土开采和冶炼配额大降36%，至4.5万公吨，创5年多来最低。而这些产量仅供满足中国国内买家的需求。北京显然希望藉此提高对国际市场的控制力。
 <br/>
 <br/>
 报导指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国稀土.html" target="_blank">
  中国稀土
 </a>
 出口量通常可供应全球约80%需求，年供应量约为15.6万吨，但出口量变动经常十分剧烈。今年上半年的开采和冶炼配额上升至7万吨，比2017年同期增加40%；今年9月出口量比8月骤升15%，但不如去年同期。
 <br/>
 <br/>
 对此，中共在香港和海外的一些外宣媒体纷纷以〝全球制造商不安〞为题报导称，中共限制稀土生产将导致稀土价格上涨，包括日本科技厂Panasonic等业者都在急于寻求其他供应源。
 <br/>
 <br/>
 不过，众多海外华人网友并不买账，在相关报导后留言讥讽中共〝吹牛不上税〞、〝五毛给自己壮胆〞等。还有网友表示，中共禁售稀土，美国可以用禁售芯片反击，把中共所谓〝高科技产业〞再次打回原形。
 <br/>
 <br/>
 其实，中共媒体宣传〝稀土杀手鐧〞已有数月之久，多家外媒已经揭破其中玄机。
 <br/>
 <br/>
 中共垄断全球稀土供应的原因，在于不计破坏环境、不惜矿工生命健康和低价对外倾销。而国外虽然同样稀土矿藏丰富，但因为要顾及保护环境和保障工人健康，政府往往限制开采；而且也基于同样的原因，国外的开采成本要高于中国，因此在价格上无法和中国稀土竞争。
 <br/>
 <br/>
 因此专家分析，如果中共限制稀土出口，定会推高稀土价格，并导致全球稀土供应链重组，但根本谈不上扼住美国或西方国家的〝喉咙〞。
 <br/>
 <br/>
 在中美关税战中，川普政府已将中国稀土从加税清单中删除，显示美国商界对此仍有所顾忌。但中共限制稀土出口，显然远远不足以令美方妥协。
 <br/>
 <br/>
 在上世纪90年代，美国曾是全球稀土第一大供应源，后来因为中国稀土在国际上以〝白菜价〞倾销，导致美国的稀土矿业骤然停产。
 <br/>
 <br/>
 中共2009年也曾一度限制稀土出口，企图抬价，但于2011年遭欧美国家向WTO申诉，结果中方败诉。
 <br/>
 <br/>
 早在今年4月，中国大陆经济学家马光远就曾公开批评，中方禁售稀土、贬值人民币和抛售美债是贸易战三大〝馊主意〞。
 <br/>
 <br/>
 马光远论及禁售稀土不可行的三个原因：一是违背WTO原则，会遭投诉；二是中国稀土价格低廉，美欧已有大量囤积；三是美国反制手段太多，〝全世界总共有的13台根伺服器，1个为主根伺服器和9个辅根伺服器都在美国，欧洲2个，日本1个。美国其实通过控制根伺服器控制了全球互联网，我们玩得起吗？〞
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
