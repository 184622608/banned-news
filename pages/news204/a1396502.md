### 【禁闻】争议不断的港珠澳大桥开通
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】周二，港珠澳大桥举行开通仪式，周三将正式通车。这座历时9年建成的大桥一直争议不断，目前大桥的配套安全问题仍令人担忧。
 <br/>
 <br/>
 在严密的保安中，港珠澳大桥10月23号在珠海举行开幕仪式，中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 出席了典礼。
 <br/>
 <br/>
 这座全长约55公里的大桥连接香港、珠海和澳门，耗时9年才建成，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大桥配套安全问题.html" target="_blank">
  大桥配套安全问题
 </a>
 仍引发担忧。
 <br/>
 <br/>
 据《自由亚洲电台》报导，香港〝民主建港协进联盟〞（简称：民建联）的立法会议员周浩鼎，21号出席电台节目时指出，目前大桥香港口岸的停车位只有650个，远少于澳门的3,000个。万一停车位爆满，市民会转到东涌泊车，容易对东涌造成挤塞。
 <br/>
 <br/>
 香港泛民主派政治组织〝支联会〞副主席蔡耀昌，指出另一个没能同步完工的配套设施问题。
 <br/>
 <br/>
 香港支联会副主席蔡耀昌：〝包括原来在香港，还要建设一个屯门到赤鱲角，连结这个大桥的另外一个通道。就可以（在）这个港珠澳大桥拿下来（建成）之后，也通过这方面到新界的西北，然后到深圳，这部分还没能够同一时间建好。〞
 <br/>
 <br/>
 由于屯门到赤鱲角的连接线没能同步开通，目前大桥的香港口岸人工岛仅靠青屿干线连接。香港一社团组织〝公共运输研究组〞成员郑衍祺，也对媒体指出，他担心大桥万一有突发意外，人工岛顿时会成为孤岛。他质疑三地没做好沟通。
 <br/>
 <br/>
 另外，尽管港珠澳大桥号称从10月24号开始通车，开放公众使用，但并不是所有的车辆都可以通行。私家车必须取得使用大桥跨境的配额，拿到〝封闭道路通行许可证〞才能通行。据了解，第一阶段港府只会分批发出一万个中、港两地车牌私家车配额，之后酌情增加。而目前拥有中、港两地车牌的私家车多达3万多辆。
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝他为什么赶期，赶到现在来仓促的通车？是因为今年正好是中共所谓改革开放四十周年。如果说赶在这样的一个时候能够拿出这样一个工程的话，对跟这个工程相关的各级的这些官员来说它是一大政绩。所以，才特意邀请习近平来出席这个剪彩的仪式。〞
 <br/>
 <br/>
 约有100人被邀请出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥开通.html" target="_blank">
  港珠澳大桥开通
 </a>
 仪式，但具体名单一直没有对外披露。不过，香港的民主派议员并没有被邀请。
 <br/>
 <br/>
 唐靖远：〝港珠澳大桥它其实被中共赋予了非常浓厚的政治工程的色彩。正是因为它是一项政治工程。所以在中共的眼中视为对立派的香港泛民（派）议员，他们就不受中共官方的待见，所以（中共）它不邀请这些人来出席大桥的开通仪式。〞
 <br/>
 <br/>
 蔡耀昌：〝中国（中共）政府对待泛民主派跟建制派是完全不同的。我想这样的理解完全是一个歧视性的。中国（中共）政府也必须要承认他是《基本法》框架之内，而且通过香港人民产生的立法会的议员。应该要一视同仁的。〞
 <br/>
 <br/>
 港珠澳大桥从兴建起就问题不断，在香港社会引发强烈反应。
 <br/>
 <br/>
 蔡耀昌：〝当中存在的问题包括严重超支，建设过程中间多年对于环境生态有影响，而且有些工人伤亡。另外我想，大家更关心就是，这样的基建建设起来，是不是真正对整个地区未来有好处？〞
 <br/>
 <br/>
 公开资料显示，港珠澳大桥总投资约1,269亿人民币。建设期间，大桥被香港媒体曝光多项质量问题，例如大桥香港接线填海工程，在2014年被发现海堤崩塌。 2015年，大桥香港口岸人工岛被揭发 〝飘移〞，部分填海用的巨型钢圆筒急剧沉降。2017年爆出混凝土压力测试造假丑闻。今年7月，香港口岸旅检大楼又被港媒揭发在暴雨时严重漏水，导致地库水浸，电机房甚至冒烟。
 <br/>
 <br/>
 另外，港珠澳大桥香港段由2012年施工至今，附近水域的中华白海豚数量下降四成，减少到47条。环保人士批评当局的环境影响评估报告失效。他们担忧海洋生态会遭到进一步破坏。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/李沛灵
