### 刘强东不止性侵女生 英媒揭更多丑闻
------------------------

 【新唐人北京时间2018年11月22日讯】中国电商巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东.html" target="_blank">
  京东
 </a>
 集团创始人兼CEO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 ，在美国明尼苏达州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 一案，22日再被英媒路透社曝出新细节。同日《金融时报》揭露刘强东更多丑闻，其中涉及京东在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双十一.html" target="_blank">
  双十一
 </a>
 超时用工，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强制劳动.html" target="_blank">
  强制劳动
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_低报酬.html" target="_blank">
  低报酬
 </a>
 等情况。
 <br/>
 <br/>
 据《金融时报》22日报导，中国〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双十一.html" target="_blank">
  双十一
 </a>
 〞销售活动期间，大量工人帮助
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东.html" target="_blank">
  京东
 </a>
 集团公司分拣、包装和运输了价值230亿美元的商品，其中有数百名是来自职业学院的学生，他们需要接受14至16小时的轮班。
 <br/>
 <br/>
 报导说，尽管他们每天有10多个小时的轮班，但拿到的工资比最低工资低得多，否则就要面临无法毕业的命运。
 <br/>
 <br/>
 山东日照科技专修学校的一位学生说，自己在昆山京东物流园区的时薪是14元人民币。而昆山市非全日制用工小时最低工资标准为18.5元人民币。
 <br/>
 <br/>
 学生们表示，全班同学都被强制参加的这种没有教育意义的实习。当他们向学校领导反映工作压力和时间过长时，得到的回复是，只要顶住压力，以后事业一定会越来越甜蜜。
 <br/>
 <br/>
 对此，京东一位发言人承认，一些临时工〝双十一〞被雇来协助为期两到三周的销售高峰。
 <br/>
 <br/>
 金融时报揭露京东用工丑闻的同一天，路透社也曝光京东创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 ，在美国明尼苏达州涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 一案的新细节。
 <br/>
 <br/>
 报导透露，刘强东涉嫌强奸当晚，强拉受害女生进浴室，女生拚命抵抗。但最后刘强东还是把她拉倒并强奸了她。女生说，她保存了留有刘强东精液的床单作为证据。
 <br/>
 <br/>
 报导说，在8月31日凌晨，警方接到报警后，前往受害女生公寓，她并未提及自己被强奸；不到24小时后，在明尼苏达大学一名管理人员的协助下，女生再度报警，这一次，她告诉警方自己被强奸了。刘强东立即被警方逮捕，在被监禁约16个小时后获释，并飞回中国。
 <br/>
 <br/>
 目前，刘强东强奸案仍未提起公诉。由于刘涉嫌罪名是最严重的〝一级强奸〞，一旦起诉并定罪，刑期至少12年起，最高可达30年。
 <br/>
 <br/>
 (记者李芸报导/责任编辑：李泉)
