### 朱军案开庭在即 重要证人被曝隐瞒细节并拒绝出庭
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】中共央视主持人朱军涉嫌性侵女实习生一案即将对簿公堂。最新消息是，原告朱军方与被告弦子和麦烧下周将在北京海淀法院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交换证据.html" target="_blank">
  交换证据
 </a>
 ，而当此关键时刻，女方日前在网络上披露称，一名知悉当年情况的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重要证人.html" target="_blank">
  重要证人
 </a>
 一直隐瞒了一个细节，而且现在还不愿为此案作证。
 <br/>
 <br/>
 10月18日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军性侵事件.html" target="_blank">
  朱军性侵事件
 </a>
 的当事人〝弦子〞的好友微博〝@麦烧同学〞发微博称，10月25日，涉案双方将在北京海淀法院进行证据交换，第一次开庭时间尚未确定。
 <br/>
 <br/>
 〝麦烧同学〞并在微博上披露，近日她与〝弦子〞 陆续收到不少匿名短讯，内容多为鼓励她们勇敢面对压力并坚持下去，取得最后的胜利；或者是向她们提供有利于反击朱军的线索。
 <br/>
 <br/>
 〝@麦烧同学〞称，她们并不知道这些短讯来自何方，但仍然感谢〝大家心里都尚存的良知、正义和勇气〞。
 <br/>
 <br/>
 此前，有媒体报导称，当事女生〝弦子〞已向法院提出要求朱军本人出庭对质。
 <br/>
 <br/>
 正当涉事双方都紧锣密鼓为即将到来的法庭较量做准备之际，〝@弦子与她的朋友们〞发微博称，当年有一位好友是此案的重要见证人，但现在这名朋友不愿出庭作证。
 <br/>
 <br/>
 据〝@弦子与她的朋友们〞10月13日在个人微博晒出的微信聊天截图显示，当事女生想请自己以前的一位朋友出庭作证，但弦子的这位朋友回复称〝非常不想回头看国内那些破事了〞。
 <br/>
 <br/>
 〝@弦子与她的朋友们〞随即表示，只要这位朋友能够与律师通过视频作证留下证言也可以，但这位朋友仍然以〝没有时间〞推托了。 〝@弦子与她的朋友们〞 表示，曾经的朋友说出这番话让她们感到非常难堪。
 <br/>
 <br/>
 据披露，这位朋友就是因为四年前陪着当事人去派出所报案的见证人。但这位朋友一直隐瞒了一个细节，当事女生在10月10日才知道这个细节。
 <br/>
 <br/>
 〝@弦子与她的朋友们〞称，这将导致其可能在法庭上面对莫须有的羞辱，然而知晓这一切，可以证明〝弦子〞清白的朋友现在却不愿意作证。
 <br/>
 <br/>
 下面是根据公开的资讯梳理的朱军涉嫌性侵事件爆发和发展的各个重要环节：
 <br/>
 <br/>
 7月26日〝@麦烧同学〞发微博率先踢爆朱军4年前涉嫌性侵当时在央视实习的女生〝弦子〞事件，后经网络〝大V〞罗昌平转发相关消息而引起社会舆论的广泛关注。
 <br/>
 <br/>
 据当事女生和她的朋友们披露，2014年6月10日，在央视《艺术人生》栏目组实习的弦子在化妆间采访朱军时，朱军试图以〝帮助留在央视工作〞等说辞诱惑女生，并在女生明确拒绝的情况下，对女生实施了抚摸和强行亲吻等猥亵举动，直到阎维文等人进入化妆间，朱军才停止猥亵。
 <br/>
 <br/>
 2014年6月11日，弦子与Z老师沟通，Z老师建议弦子报警，在Z老师、律师和同学的陪同下，弦子前往北京市公安局海淀分局羊坊店派出所报警。警方完成笔录后，派人前往央视出事现场进行调查并调取了监控视频。稍晚，警方在弦子的贴身衣物上进行DNA提取，并向弦子等人播放了监控视频。在视频回放过程中，警方指着弦子擦嘴的画面说，〝这个视频可以作为你被朱军强迫的证据〞。
 <br/>
 <br/>
 2014年6月X日（具体日期不明），羊坊店派出所的值班警官告诉弦子，现在由北京市公安局海淀分局负责处理此案。当天晚些时候，两名警官告诉弦子，〝朱军是正能量，要考虑这件事的影晌〞。
 <br/>
 <br/>
 2014年6月X日，北京市公安局海淀分局在没有告知弦子的情况下来到位于武汉的弦子父母家中，向其父母施压，并要求弦子父母写下保证书，承诺不再提及弦子曾经被朱军猥亵一事。于是此事被压下没有再提。
 <br/>
 <br/>
 2018年7月26日清早，弦子在微信朋友圈发布了第一篇鎚子便签长文，讲述了四年前的事情经过和过往四年的反思，这篇长文得到她的朋友们的转发。当天，〝麦烧同学〞在微博上发表了从朋友的朋友圈看到的弦子的长文，并注明被曝光的人是央视主持人朱军。但这篇长文很快就被中共网管屏蔽。
 <br/>
 <br/>
 7月27日，财新网记者采访了弦子、Z老师、曾陪同弦子报案的Z老师的律师朋友、阿桑、羊坊店派出所和朱军，但羊坊店派出所和朱军拒绝作出回应，随后该媒体发表了题为《女实习生指控主持人性骚扰》的报导。不过这篇报导发表数小时后也被迫删除。
 <br/>
 <br/>
 7月30日，在律师的陪同下，弦子等人前往羊坊店派出所索取4年前的调查结果，被告知案件已由北京海淀分局负责。之后，北京海淀分局只派一名警官出面，让弦子就当年猥亵事件与鎚子便签传播过程重新做了笔录并签字，之后一直没有将4年前的立案回执交予弦子。
 <br/>
 <br/>
 8月14日晚，麦烧同学被施压，〝如果不删掉和朱军有关的微博就要退掉目前所租的房子〞。这一问题在第二天媒体报道后得到解决。
 <br/>
 <br/>
 8月15日，北京星权律师事务所在其官方微博上发布律师声明，指网络上有关朱军性侵事件的信息为〝不实信息〞，宣布将向该事件的新浪微博原发者和转发用户提出起诉。
 <br/>
 <br/>
 8月16日，麦烧同学发布微博寻求更多的被朱军性骚扰的受害者站出来与她和弦子取得联系，并将该微博置顶。
 <br/>
 <br/>
 当晚，弦子以〝弦子与她的朋友们〞为名开通微博并回应朱军的律师声明，称〝麦烧同学不是造谣，所有传播那篇文章的朋友也不是造谣，因为那是真相，是我的真实经历，我万分期待朱军的律师函能给我一个契机，让真相能够被证明是真相。
 <br/>
 <br/>
 与此同时，《Vista看天下政商智库》发布《专访朱军性骚扰爆料人：若上法庭，不会退缩》，这篇报导罕见未被删除。
 <br/>
 <br/>
 8月17日，弦子发布《关于北京星权律师事务所〝律师声明〞的声明》称，希望北京星权律师事务所能够严肃认真对待事实和法律，如果北京星权律师事务所继续传播〝朱军性骚扰 (猥亵）实习生（我本人）为不实信息〞的谣言，当事人将依法对星权律师事务所及相关代理律师进行追责。 但随后弦子的微博被限制转发。
 <br/>
 <br/>
 8月20日，多家媒体发布报道《性骚扰当事人视频自述：朱军给我看手相，手伸进裙底未遂后强吻》，弦子第一次在镜头前讲述了4年前发生的事情，并希望更多受害人能够站出来维护自身合法权益。
 <br/>
 <br/>
 8月22日至23日晚，弦子不断接到异地恐吓电话，电话中以弦子母亲的安全来威胁；23日晚，弦子和她的朋友们在微博上披露了恐吓电话的录音。
 <br/>
 <br/>
 8月27日，麦烧同学和律师前往公证处，对证据进行了公证。
 <br/>
 <br/>
 8月28日，麦烧同学发布微博称，〝新浪把我微博上已被屏蔽的关于朱军的微博都解禁了〞，她认为事情可能出现了变化。
 <br/>
 <br/>
 8月29日，有消息指朱军性骚扰 (猥亵）女实习生一事可能受到了内部处理， 弦子和麦烧同学表示，〝我们想要的是法律上的交代和进步，同时要求央视作为工作场所的负责人公开调查结果〞。
 <br/>
 <br/>
 9月5日，麦烧同学在微博上发布〝从2014年到2018年，朱军被举报对女实习生性骚扰一事，发生了什么和正在发生什么〞便签长文，但发布仅半小时后，其微博被网管屏蔽。
 <br/>
 <br/>
 9月25日，麦烧同学和弦子接到北京海淀法院电话，通知已被朱军起诉。当天，弦子在接受视频采访时说，〝你站出来以后，大家会更清楚你是一个活生生的人〞。麦烧同学说，〝我们将积极应诉，让司法来决定吧〞。
 <br/>
 <br/>
 9月26日，《财经》杂志和《Vista看天下》杂志发布了对弦子和麦烧同学的采访，对于可能的结果和调解，她们表示，〝没有见到判决结果之前，我们不想去认输，也不准备去认输。〞
 <br/>
 <br/>
 10月1日，在央视60周年台庆百位主持人合唱的演唱视频中，朱军没有出现。
 <br/>
 <br/>
 10月15日，弦子及其好友向北京法院提交两份申请，一份是申请朱军本人出庭接受质询，一份是申请法院向北京市公安局海淀分局收集2014年6月朱军性骚扰案的全部案件材料及证据。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：戴明）
