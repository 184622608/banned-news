### 网贷平台〝财金圈〞投资人维权 处处受阻
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】今年年中，大陆P2P
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网贷平台.html" target="_blank">
  网贷平台
 </a>
 掀起倒闭潮。其中，北京网贷平台〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财金圈.html" target="_blank">
  财金圈
 </a>
 〞涉及6000名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资人.html" target="_blank">
  投资人
 </a>
 ，总金额2.6亿元。11月30号，网上披露，该平台有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中央财经大学.html" target="_blank">
  中央财经大学
 </a>
 和国资背景，出事后却推卸责任，投资人维权，处处受阻。
 <br/>
 <br/>
 11月30号，北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网贷平台.html" target="_blank">
  网贷平台
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财金圈.html" target="_blank">
  财金圈
 </a>
 〞受害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资人.html" target="_blank">
  投资人
 </a>
 在网上披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中央财经大学.html" target="_blank">
  中央财经大学
 </a>
 的教授大肆宣传，〝财金圈〞隶属于中央财经大学产业园重点项目。今年8月15号，宣布良性退出方案清盘。
 <br/>
 <br/>
 财金圈受害投资人郭先生：〝财经大学、国资占股，才相信嘛，没名气谁相信你？他们那边背景强大，(经侦)没有公布任何进展。闹分家、离婚的都有，生活也困难，都是不敢发这些消息、不敢传，会封锁。〞
 <br/>
 <br/>
 事发后，投资人自发组派出代表前往北京东城区金融办和东城区经侦和中央财经大学。金融办不断推委责任，东城区经侦至今不作为，中央财经大学则推卸责任。
 <br/>
 <br/>
 财金圈匿名受害投资人：〝经侦那边消息特别封锁，他们说他们是办案不追赃，说平台已经没钱了，没钱还你们了，就这么态度。9月28号，我们去北京财大那边跟他们讨个说法，他们不见人，不让进，我们难友就在门口聚集，当地朝阳派出所公安，把我们代表给拘留了一个月，有五、六个，现在都出来了。〞
 <br/>
 <br/>
 据《大纪元》报导，11月30号，有5名金融难友在天安门广场前喝农药自杀，自杀者为天津现货交易所金融诈骗受害者，他们到北京上访一个月毫无结果，走投无路。
 <br/>
 <br/>
 匿名受害投资人：〝很难维权，处处受阻，现在没门路，无处诉苦，喝农药、跳楼的、上吊的，多的去了。〞
 <br/>
 <br/>
 〝财金圈〞成立于2013年10月12号。最早是由中央财经大学商学院教授李永壮，组织学生、亲戚成立了北京厚泽国信投资管理有限公司，而后成立线上APP，改名北京财金圈投资管理有限公司。
 <br/>
 <br/>
 新唐人记者熊斌、舒灿采访报导
</div>
