### 5年后卷土重来？中共智囊吁重拾〝韬光养晦〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】中共经济陷入危局，北京一改强硬态度，正努力寻求与美国〝和解〞。中共高阶智囊日前呼吁对美国做出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空前让步.html" target="_blank">
  空前让步
 </a>
 〞，重拾〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韬光养晦.html" target="_blank">
  韬光养晦
 </a>
 〞，以争取五六年时间〝调整经济〞。港媒指，该论调或是北京当前对策的真实写照。
 <br/>
 <br/>
 日前，中国人民大学的美国研究中心主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_时殷弘.html" target="_blank">
  时殷弘
 </a>
 发表演讲，谈及对中美贸易战的看法。港媒称，时殷弘属于中共的中央级智囊、美国研究权威，其意见在某程度上反映了北京最新的策略。
 <br/>
 <br/>
 时殷弘表示，北京不应该再纠缠暂时的〝输赢〞，必须适时对美国做出一些〝空前的让步〞，包括大举增购美国产品和扩大美资中国市场准入，以争取一段时间的〝休战〞，为调整经济营造一种压力较小的环境。
 <br/>
 <br/>
 时殷弘分析，因为中方在全球化中得利太多，引发很多发达国家的不满甚至愤怒，那种对北京有利的全球化已不大可能持续下去。
 <br/>
 <br/>
 时殷弘认为，北京在过去五六年中，对外政策〝蹦得〞过高过快，已经把对手都〝动员〞起来了。因此，现在要采取〝进两步退一步〞的策略，再用五六年时间实施〝新形态的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韬光养晦.html" target="_blank">
  韬光养晦
 </a>
 〞，让对手回到被〝动员〞以前的松垮状况，然后再谋求〝新的进取〞。
 <br/>
 <br/>
 目前，北京已经放软身段，努力争取本月底中美首脑会晤的〝和解〞。英美媒体消息，北京已经迈出妥协的第一步，向美方提交了具体方案，以回应美方的贸易要求。不过，北京提出的方案多是旧调重弹，依然没有改变产业政策、放弃〝中国制造2025〞计划等关键内容。
 <br/>
 <br/>
 美国CNN引述消息指，中方书面回应中并未包含白宫反覆提出的几项必含内容，包括强制技术转让和盗窃知识产权等。这份方案和北京此前公开发表的声明几乎没有差别，不过是用新瓶装旧酒，并稍微修改了一下酒的配方。
 <br/>
 <br/>
 美国方面，副总统日前已发出最后通牒，中方如果不能在保护知识产权、强迫技术转移、限制外企进入中国市场、尊重国际规则及在国际水域航行自由等问题上做出显著的、实质的让步，美方准备在经济、外交和政治上对北京施加更大压力，新冷战就要来临。
 <br/>
 <br/>
 摩根大通预测，未来3到6个月内，中美贸易战将继续升级。
 <br/>
 <br/>
 也有分析指，川习会的〝最好结果〞是中方做出某种让步的〝承诺〞，双方达成一个临时性〝停火〞协议，但很难长久。
 <br/>
 <br/>
 白宫贸易高级顾问纳瓦罗近日表示，中共一贯言而无信，所以即使美中达成协议，也无法相信中共能够真正执行。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
