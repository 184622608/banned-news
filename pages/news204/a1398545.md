### 王全璋案或将开庭 李文足请外国大使 旁听
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】首先来关注中国的消息。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709案.html" target="_blank">
  709案
 </a>
 被抓律师王全璋的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 ，11月5号与多国大使馆官员会面，并递交了致各国大使的信件。她表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋案.html" target="_blank">
  王全璋案
 </a>
 可能即将开庭，所以请求各国大使向中共政府提出旁听的要求。
 <br/>
 <br/>
 11月5号上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 在另一位709律师家属王峭岭的陪同下，在北京与英国、美国、德国、加拿大、欧盟、瑞士等国大使馆的官员见面。
 <br/>
 <br/>
 王全璋妻子李文足：〝一起交流了王全璋最新的情况，然后我也递交了一封我写的信，就是恳请各国大使能够向中国（中共）政府提出要求去旁听王全璋的案件，这个信让他们转交，他们接受了这个信，收下了。〞
 <br/>
 <br/>
 余文生妻子许艳：〝大家的关注可能对现在失去自由的律师和律师家属都是一种帮助作用，非常感谢吧。〞
 <br/>
 <br/>
 李文足表示，王全璋被捕已经3年4个多月，中共官方一直阻止家属聘请的律师会见，而根据最新消息，官方指定的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘卫国.html" target="_blank">
  刘卫国
 </a>
 律师已在日前会见了王全璋，估计案子很快就要开庭，所以希望各国大使前往关注。
 <br/>
 <br/>
 李文足：〝刘卫国律师虽然是全璋点名的，但是王全璋点名的律师可不只这一位。我觉得这样没办法完全保障王全璋合法的权益。还有，他们是不是也会开庭。所以我现在恳请各国大使向中国（中共）政府提出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋案.html" target="_blank">
  王全璋案
 </a>
 开庭的话，他们能去旁听这个案件。〞
 <br/>
 <br/>
 李文足5号也给案件主审法官写了一封公开信，痛斥他们长期变相拖延案件办理期限。
 <br/>
 <br/>
 王全璋，现年42岁，在709大抓捕前，常年为社会弱势群体提供法律援助，也是最早为法轮功学员辩护的中国律师之一。他在2015年7月10号失踪，并在去年2月被以所谓的〝颠覆国家政权罪〞正式起诉，但案件至今仍未开庭。
 <br/>
 <br/>
 新唐人记者常春、陈洁、元真采访报导
</div>
