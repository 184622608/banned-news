### 中共四面楚歌:铁杆盟友翻脸 民主党众院力挺彭斯
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】美国总统川普对中共的反击战，不但得到美国国内政治对手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 的支持，也迎来国际上的群起响应。日前，美国众议院民主党议员发声响应副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 对北京的批评，几个此前和中方相当〝友好〞的小国也纷纷对中共说不。
 <br/>
 <br/>
 <h4>
  美国
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
   民主党
  </a>
  议员声援
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
   彭斯
  </a>
 </h4>
 <br/>
 上周，美国副总统彭斯（Mike Pence）接连出席了东盟峰会、APEC峰会等亚太地区的国际会议。在这次行程中，彭斯严厉批评中共的贸易不公和〝一带一路〞等殖民政策，并明确美方将联合澳大利亚和日本等盟友，加大对印太地区的援助，以遏制中共在该地区的政治扩张。
 <br/>
 <br/>
 在彭斯发表这些讲话的同时，身在美国的众议院民主党议员康纳利（Gerry Connolly）也接受了《华盛顿观察家报》（Washington Examiner）的采访，警告中共不要梦想把美国赶出亚太地区。康纳利是众议院外交事务委员会亚太小组的成员。
 <br/>
 <br/>
 报导称，康纳利和彭斯互相呼应，可能发展成为川普政府与明年将控制众议院的民主党之间在外交政策上罕见的共识点，那就是强硬应对中共的〝经济侵略〞。
 <br/>
 <br/>
 目前，中共正在太平洋上扩充海军力量，并宣称对绝大多少南海海域拥有主权，还在南海部署导弹和建设军事设施。康纳利称，这将引发中美严重冲突，因为美方永远不会接受中方这些举动。
 <br/>
 <br/>
 康纳利警告说，日本当年就曾犯过同样的错误（指将美国拖入第二次世界大战），并为此付出了〝可怕的代价〞。
 <br/>
 <br/>
 本月初的美国中期选举中，川普所在的共和党保住了众议院，但失去了众议院的多数党地位。不过外界分析，众议院职权更偏重外交事务，而民主党立场一般比共和党更为反共，特别是众议院的民主党领袖佩洛西（Nancy Pelosi）更是中共极为头痛的反共强硬派，美国这种新政局对北京更为不利。
 <br/>
 <br/>
 <h4>
  中共多个铁杆盟友翻脸
 </h4>
 <br/>
 中央社19日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_柬埔寨.html" target="_blank">
  柬埔寨
 </a>
 总理洪森当天表示，柬埔寨宪法绝不允许国土上有外国军事基地。这番话是在回应美国副总统彭斯对邻近南海处可能出现中共海军基地的〝关切〞。
 <br/>
 <br/>
 此前，外界盛传柬埔寨西南部海岸正在兴建一座中共海军基地。但洪森在讲话中驳斥了该说法。
 <br/>
 <br/>
 近年来，中共对洪森的强权统治提供支持，并投入大量资金。中共拥有柬埔寨60亿美元外债的近一半。作为交换，柬埔寨一直是中共的〝坚定盟友〞。
 <br/>
 <br/>
 另外，中共另一个〝坚定盟友〞马尔代夫近期变天，新总统萨利赫上台后立刻对前任总统遗留的中国债务提出批评，并宣布重新审查和中共合作的〝一带一路〞项目。马尔代夫是最早响应中共〝一带一路〞的国家之一。
 <br/>

 </center>
 <br/>
 路透社19日消息，萨利赫的顾问还表示，将退出马尔代夫和中共签订的自由贸易协定。
 <br/>
 <br/>
 17日，彭斯在APEC峰会上讽刺中共〝一带一路〞是沿线国家的一条〝不归路〞。18日，菲律宾财政部长杜明奎马上响应，表示菲律宾政府将会审查〝一带一路〞协议。
 <br/>
 <br/>
 此外，多次接受中共巨额援助的APEC峰会主办国巴布亚新几内亚，在今年峰会期间的表现也让北京受挫。
 <br/>
 <br/>
 该国不但将原打算让中共开发的马努斯岛海军基地交给了美国和澳大利亚，其外交部长还拒绝接见因峰会联合公报来访的中共官员，最后叫来保安人员将中方人员赶走。
 <br/>
 <br/>
 自由亚洲电台19日引述华人学者指出，北京近日推动〝区域全面经济伙伴关系协定〞（RCEP）遇挫，在APEC遭各国围堵，说明中方主导的所谓〝多边体制〞被美方击破，北京已陷入被动。
 <br/>
 <br/>
 人权组织〝公民力量〞创办人杨建利16日在《美国之音》访谈节目中称，美国当年领导那么多国际组织，是因为美国真正对这些国家让利并提供保护。而中共不愿意让利，只关注经济上的掠夺和军事上的扩张，所以在国际上当不成领导者。
 <br/>
 <br/>
 自由亚洲电台引述香港新世纪出版社创办人鲍朴分析，当一个超级大国撒钱的时候，很多国家都很愿意接它的钱，但拿钱后自己干自己的事，这在苏联时期是有教训的。中共用这种方式往世界上推行所谓〝中国模式〞也肯定是要失败的。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
