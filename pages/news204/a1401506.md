### 【 禁闻】路透：62%关税下 中国大买美国猪肉
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】
 <h4>
  中共绑架
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
   法轮功
  </a>
  学员 美吁停止迫害
 </h4>
 <br/>
 中共对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 的迫害今年有加剧迹象。根据〝法轮大法明慧网〞披露的信息，中共在今年的前11个月，至少制造了70起集体抓捕案件。其中11月9号，一天之内，就在黑龙江省哈尔滨市的各个区县和大庆市，非法绑架了119名法轮功学员。
 <br/>
 <br/>
 针对这个事件，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国务院.html" target="_blank">
  美国国务院
 </a>
 再次呼吁中共停止迫害法轮功。
 <br/>
 <br/>
 据〝大纪元新闻网〞报导，11月29号，美国国务院的一位发言人在给大纪元记者的邮件中表示：〝我们要求中国（中共）停止对法轮功的禁令，允许法轮功学员依据国际人权承诺自由地修炼、实践他们的信仰。〞
 <br/>
 <br/>
 <h4>
  路透：62%关税下 中国大买
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国猪肉.html" target="_blank">
   美国猪肉
  </a>
 </h4>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 在中国大陆不断蔓延，已经扩大到20个省。
 <br/>
 <br/>
 路透社11月29号报导说，尽管在美中贸易战中，中共对美国猪肉加征了62%的进口关税，但最近却从美国大量进口猪肉。
 <br/>
 <br/>
 报导说，美国农业部当天的资料显示，中国上周向美国发出了贸易战开始以来最大的一笔猪肉订单。
 <br/>
 <br/>
 美国农业部称，截至11月22号，中国购买了3,348吨今年付运的猪肉，这是今年2月以来最大的订单。同时中国还购买了预定明年出货的9,384吨美国猪肉，这在上周美国对所有国家的猪肉总销量中，占比72%。
 <br/>
 <br/>
 <h4>
  报告：
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共渗透.html" target="_blank">
   中共渗透
  </a>
  美国 破坏民主价值
 </h4>
 <br/>
 美国斯坦福大学胡佛研究所11月29号公布了一份有关中共渗透美国，破坏民主价值观的报告。
 <br/>
 这份题为〝中国影响力及美国利益：提高建设性警觉〞的报告由7个国家的32位中国问题专家撰写，其中指出,  中共正在通过〝渗透和动摇〞美籍华人社区、在美中国学生、美国民间社会组织、大学、智库、媒体和企业，〝挑战，甚至损害美国的核心自由、规范和法律〞，破坏民主价值，包括美国和世界各地的言论自由权等。
 <br/>
 <br/>
 据报导，撰写报告的专家中有很多都是公认的〝亲中〞专家,  过去都主张与中国保持良好沟通和互动，但最新的态度明显变得审慎和怀疑。
 <br/>
 <br/>
 有分析认为，这显示了包括美国在内的全球中国问题专家对中国希望的〝幻灭〞，意味着美国境内对美中关系发展的争论将发生转变。
 <br/>
 <br/>
 编辑/周玉林
