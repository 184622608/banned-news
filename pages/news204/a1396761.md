### 纽时称川普手机遭中共窃听 华春莹奇葩回应惊呆舆论
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】美媒披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 经常
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_窃听.html" target="_blank">
  窃听
 </a>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_iPhone手机.html" target="_blank">
  iPhone手机
 </a>
 ，以了解川普的想法。25日中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共外交部.html" target="_blank">
  共外交部
 </a>
 发言人华春莹回应称，如果很担心iPhone手机被窃听的话，可以改用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为手机.html" target="_blank">
  华为手机
 </a>
 。网民直呼此言论很离谱，被惊呆了。
 <br/>
 <br/>
 《纽约时报》10月24日援引多位美国退休及现任官员的话称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 一共有3部手机，其中2部是经过国家安全局加强保安及限制了部分功能的官方
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_iPhone手机.html" target="_blank">
  iPhone手机
 </a>
 ，另一部则是川普的私人iPhone。联邦情报单位通过管道以及拦截外国官员通讯等方式，得知中共和俄罗斯正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_窃听.html" target="_blank">
  窃听
 </a>
 川普的通话。
 <br/>
 <br/>
 报导说，相对俄罗斯而言，中共对于川普的通话内容更感兴趣，他们希望能从中找到解决贸易战的方法。
 <br/>
 <br/>
 消息人士说，北京精通窃听电话的技巧，并试图用以搜集川普的想法、川普和哪些人通话，进而找出影响川普的最佳方法。据了解，北京特别想要通过这个方法，防止两国贸易战的升级，并掌握与川普通话的重磅名单。
 <br/>
 <br/>
 美媒FastCompany引述白宫前首席信息官佩顿（Theresa Payton）表示，如果川普手机被监听属实，那么这可能是史上对白宫通讯的最大、最严重的一次入侵事件。〞
 <br/>
 <br/>
 10月25，在中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共外交部.html" target="_blank">
  共外交部
 </a>
 例行记者会上，有来自CNN的记者就上述监听事件提问。中共外交部发言人华春莹竟然声称，如果很担心iPhone手机被窃听的话，可以改用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为手机.html" target="_blank">
  华为手机
 </a>
 。如果还不放心，可以停用任何现代通讯设备，断绝与外界的所有联系。
 <br/>
 <br/>
 网民直呼此言论很离谱，被惊呆了，认为此回应是无耻搪塞，充分曝光了中共的流氓做法，已经不在乎外界的耻笑，更不在乎其一再包装的国际形象。还有网民抨击中共到处搞窃听，被全世界发达国家指责还不知反醒，反倒用此类荒唐言论回避事件。
 <br/>
 <br/>
 一直以来，华为和中兴等在美国引发的安全风险，已经引起了美国国会极大的关注。
 <br/>
 <br/>
 美国国会〝美中经济暨安全检讨委员会〞（USCC）4月份曾警告，中共以多种形式主导经济间谍活动，尤其试图抢占通讯产业领导地位。报告中点名华为、中兴通讯和联想等三家中国企业。尤其华为有中共军方背景。
 <br/>
 <br/>
 美国联邦通讯委员会4月也通过提案，禁止营运商使用联邦政府补贴资金采购所有可能对美国造成国安风险的网络通信设备，华为、中兴通讯等也在列。
 <br/>
 <br/>
 美国国会在8月1日通过的《2019财年国防授权法案》中明确提出，从国家安全考虑，禁止任何美国政府部门使用中国华为与中兴两家公司的产品，也禁止任何与美国政府有业务关系的实体这样做。
 <br/>
 <br/>
 法案还限制美国政府与中兴及华为的技术合约，防止国安威胁。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
