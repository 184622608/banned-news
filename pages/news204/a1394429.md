### 外媒析孟宏伟案内情：北京担心第二个王立军
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】兼任国际刑警组织主席的中共公安部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 回国失踪，引发国际关注。有海外中文媒体刊文分析，此案为北京整肃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 〝余毒〞，北京担心孟效仿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 寻求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治庇护.html" target="_blank">
  政治庇护
 </a>
 ，本拟秘密操作，但因孟妻报警而洩露。
 <br/>
 <br/>
 文章称，近年来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 在政法系统的多名心腹被抓，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 曾与周永康关系密切，也已在当局清洗范围之内。
 <br/>
 <br/>
 但是，孟宏伟作为国际刑警组织主席，拿下他并不容易。稍有不慎，孟宏伟很可能效仿当年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 投奔美国大使馆，也在法国寻求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治庇护.html" target="_blank">
  政治庇护
 </a>
 ，造成重大国际事件，让北京颜面尽失。
 <br/>
 <br/>
 文章认为，北京的原计划，应该是利用会议的名义骗孟宏伟回国，制服以后，让他通知家人随后回国，再以个人健康原因辞去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警主席.html" target="_blank">
  国际刑警主席
 </a>
 的职务。
 <br/>
 <br/>
 但是，孟宏伟长期在中共江湖中混，早已预感到危险。他虽然不大相信北京现在会对他动手，但也已做好安全预案，如果他失踪，其妻就向法国警方报案，使其成为国际事件。
 <br/>
 <br/>
 文章说，现在看来，北京对对孟宏伟的控制显然未达到预期效果，其妻已经报警，令事件曝光。
 <br/>
 <br/>
 目前局势的发展，在一定程度上验证了上述判断。孟宏伟妻子在其失联一周后向法国警方报案，令外界称奇。有分析指，此举意在引起国际社会关注，以牵制北京对孟宏伟的后续处理。
 <br/>
 <br/>
 根据公开资料，孟宏伟在去年12月就被免去中共国家海洋局副局长、中共海警局局长职务，今年4月其公安部党委委员的职务也被拿掉，只保留了公安部副部长的兼职。这显示其前途早已不妙。
 <br/>
 <br/>
 对于北京为何不顾国际影响，突然拿下国际刑警主席，外界传出多种说法。
 <br/>
 <br/>
 有港媒引述消息称，孟宏伟落马是因为经济问题和在香港购置产业等。但该说法遭到普遍质疑。
 <br/>
 <br/>
 也有港媒指，孟宏伟案涉中南海政变阴谋，事态紧急。
 <br/>
 <br/>
 还有说法认为，孟宏伟担任国际刑警主席后，在国际上有不利北京的举动，涉嫌替周永康反扑。
 <br/>
 <br/>
 日前，有一些〝特定〞政治背景的媒体放风称，孟宏伟已经〝暂时获得自由〞，还有说孟只是被〝留置〞，并未〝双规〞。
 <br/>
 <br/>
 不过，自北京当局开始〝反腐打虎〞至今，被中纪委拿下的高官，从未出现〝无罪释放〞的先例。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
