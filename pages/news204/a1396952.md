### 【禁闻】打击贩卖人体器官 加参院通过法案
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】
 <b>
  打击贩卖
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人体器官.html" target="_blank">
   人体器官
  </a>
  加参院通过法案
 </b>
 <br/>
 <br/>
 加拿大参议院日前一致通过了一项打击贩运
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人体器官.html" target="_blank">
  人体器官
 </a>
 的法案。参议员奥特拉吉安和保守党国会议员吉尼斯10月25号联合召开新闻发布会，公布了这一消息。
 <br/>
 <br/>
 他们在发布会上说，这项法案名为S-240，它修订了加拿大《刑法典》，对与贩运人体器官和组织相关的新罪行进行了规定。同时还修订了《移民和难民保护法》，规定加拿大不能让参与贩运人体器官和组织的永久居民或外国公民进入加拿大。
 <br/>
 <br/>
 <b>
  新疆
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育营.html" target="_blank">
   再教育营
  </a>
  大规模采购
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警棍手铐.html" target="_blank">
   警棍手铐
  </a>
 </b>
 <br/>
 <br/>
 中共利用〝再教育营〞在新疆关押上百万维吾尔人的事件，引起国际关注。中共坚称这些〝再教育营〞是提供职业培训的学校。但是〝法新社〞10月24号报导说，他们根据新疆政府的1500多份公开文件发现，新疆和田市的一所〝职业技能教育培训中心〞，在今年年初购买了一批与技能培训无关的物资，其中包括2768支警棍，1367副手铐，2792支胡椒喷雾器和550个电击棒。
 <br/>
 <br/>
 报导说，这只是和田地区政府自2017年初以来，为建立和管理〝职业培训中心〞所进行的上千个采购中的一部分。
 <br/>
 <br/>
 文件还显示，有上千名配有催泪瓦斯、泰瑟枪、电击枪和尖刺棍棒的卫兵，对这些所谓〝职业培训中心〞中的〝学生〞进行严密控制，而这些〝中心〞周围都有带刺铁丝网围墙。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州地铁.html" target="_blank">
   广州地铁
  </a>
  启用
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人脸识别.html" target="_blank">
   人脸识别
  </a>
  民忧社会变监狱
 </b>
 <br/>
 <br/>
 10月26号，广州的多个地铁站启用人脸识别安检系统。中共官媒报导说，这是试点，未来将向全国推广。
 <br/>
 <br/>
 有广东网民对海外媒体表示，如果在全国推广，所有民众的一举一动，都将在中共监控之下。
 <br/>
 这名人士批评说：那情形就跟电影1984一样，人一出门就无所遁形，天网摄像头、人脸识别系统、车牌识别系统、手机定位技术，公安可以随时随地的抓人。整个社会将跟监狱差不多。
 <br/>
 <br/>
 编辑/周玉林
