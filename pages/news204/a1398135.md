### 黑手染指德国支柱产业 巴符州连曝6起经济共谍案
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】伴随着中美贸易战，中共官方支持的大规模经济间谍活动，越来越受到发达国家的重视。德国汽车产业云集的巴符州，近6年曝出的8起经济间谍案件中，有6起被指与中共有关。
 <br/>
 <br/>
 据《德国之声》报导，巴符州内政部日前在回应州议会自民党党团质询时指出，从2012年至今，州刑警局共发现8起经济间谍案件，其中至少6起不排除中共间谍作案的可能性。
 <br/>
 <br/>
 内政部表示，这一数字远非间谍活动的全部，因为很多公司会选择掩盖洩密事件，以维护自身形象。
 <br/>
 <br/>
 根据内政部透露的情况，受到网络间谍威胁的不只是当地中型企业，还包括一家〝坐落于斯图加特的汽车制造商〞，也遭遇了境外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络攻击.html" target="_blank">
  网络攻击
 </a>
 。
 <br/>
 <br/>
 而奔驰、保时捷的总厂，都位于斯图加特近郊。
 <br/>
 <br/>
 州政府指出，网络经济间谍活动每年在该州造成的潜在经济损失达70亿欧元。德国联邦政府的数据则显示，全德国每年因此蒙受的直接经济损失高达500亿欧元，还不包括间接损失。
 <br/>
 <br/>
 根据此前外媒披露的消息，中共采取的商业间谍行为多种多样。其中包括军方黑客军团的大规模网络攻击，也有中共国安特工招募、培养的外企〝内鬼〞。
 <br/>
 <br/>
 中共网络攻击的一种常见形式，是有针对性的接洽外国企业的特定员工，通过电子邮件、社交网络平台发送一些包含病毒的信息，以感染对方的终端设备。
 <br/>
 <br/>
 而中共规模庞大的海外人才〝招募〞计划，包括〝千人计划〞等，则是将外国尖端技术〝引入〞中国的另一种公开的商业间谍行为。
 <br/>
 <br/>
 此外，利用中国企业输出的网络设备盗取商业资讯，也是中共网络间谍行为的典型手段。包括中国华为等出口的电信设备，多次被曝留有〝后门〞。
 <br/>
 <br/>
 今年3月，德国《经济周刊》曾经报道指出，华为等中国电信设备制造商已经在德国市场具有绝对优势地位，这让中共间谍很方便地窃取德国企业的数据。
 <br/>
 <br/>
 不久前，彭博新闻社还报导称，一批中国制造的IT硬件被植入间谍芯片，这些硬件正在亚马逊、苹果等网络巨头的办公服务器上运行。
 <br/>
 <br/>
 报导亦指出，在中国生产环节中，将间谍芯片植入电脑主板，是中共国安部门指导下的政府行为。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
