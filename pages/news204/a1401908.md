### 北京仅给空头支票？英媒：贸战将以更凶猛方式开打
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 迎来90天的休战期，但外界普遍对于贸易战的前景不看好。英媒刊登评论文章称，北京承诺的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性改革.html" target="_blank">
  结构性改革
 </a>
 〞几乎不可能得到实施，而川普（特朗普）总统仍掌握着重新开火的主动权，可以预料的是，3个月后贸易战将以更加凶猛的方式重新开打。
 <br/>
 <br/>
 12月1日，美国总统川普与中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 达成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 暂时停火的共识。英国《金融时报》12月4日刊文称，美国除了90天的最后期限外什么也没提供，川普总统占据上风，重新开火的权利仍完全掌握在他的手中。
 <br/>
 <br/>
 而中方为了换取美国不从明年1月1日起，对2000亿美元的中国输美产品加征更高关税，已同意就一份长长的让步清单进行讨论。
 <br/>
 <br/>
 根据白宫发布的声明，北京同意在短期内从美国购买〝大量农业、能源、工业和其他产品〞。北京还同意立即就〝强制技术转让、知识产权保护、非关税壁垒、网路入侵和网路盗窃、服务和农业方面的结构性变革〞展开谈判。
 <br/>
 <br/>
 美国财长姆钦12月3日接受《金融时报》采访时表示，川习晚宴上，习近平非常详细地阐述了他们的承诺。〝这是中方第一次对结构性问题和非结构性问题，提出非常长且具体的回应，川普总统于是做出如何往前推进的决定。〞
 <br/>
 <br/>
 也就是说，在贸易战导致中国经济面临巨大危机的情况下，北京首次就〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性改革.html" target="_blank">
  结构性改革
 </a>
 〞这个关键问题做出让步，才换取了美国的90天期限。如果〝结构性改革〞得到全面实施，将会从根本上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改变中国体制.html" target="_blank">
  改变中国体制
 </a>
 的性质。
 <br/>
 <br/>
 不过文章称，在北京，没有人会幻想已经避免了与美国的贸易战——尽管受到严格控制的中共官方媒体是那么报导的。
 <br/>
 <br/>
 有分析认为，进行结构性改革将会动摇中共的体制，而且不可避免会触动中共权贵集团的利益，可以预见习近平将会面临来自党内各方的压力。90天的期限其实是非常紧迫的。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田表示，只要中共的体制存在，就不可能实现非关税壁垒、网路入侵、知识产权保护等一系列结构性改变。他认为，谈判很难在90天内完成。
 <br/>
 <br/>
 谢田担心的是，中共会像以前一样玩花招，以赢得90天的喘息时间。〝在这90天的时间里，它以前2500亿的10%的关税虽然没有取消，但是更大幅度的25%关税的惩罚现在可以避免，但是川普应该很快会发现中共的承诺是靠不住的。〞
 <br/>
 <br/>
 根据协议，一旦90天期限过后，如果美国对北京方面的让步不满意，仍会将2000亿美元中国输美商品加征的关税从10%提高到25%。
 <br/>
 <br/>
 文章称，由于习近平几乎不可能实施、或者说甚至不可能同意对中国经济的如此广泛的改革，几乎可以肯定，到2019年2月底，贸易战将以更加凶猛的方式重新开打。
 <br/>
 <br/>
 另据《日本经济新闻》报导，曾担任美国助理国防部长的傅立民表示，美中关系已从根本上〝变质〞，双方已从〝竞争（rivalry）〞转变为〝敌对对立（adversarial　antagonism）〞关系。就像冷战时期的美苏关系。即使美中双方达成休战协定，但这个结果也可能很难实现，暗示贸易战很难停火。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
