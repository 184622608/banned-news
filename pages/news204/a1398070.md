### 中国乘客与司机互殴致15死 网友：在美国这是重罪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】日前中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 发生一起公车坠江惨案，造成15人死亡。经调查发现，肇事原因是女乘客与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司机.html" target="_blank">
  司机
 </a>
 间激烈的争执、互殴所致。对此，有网友称，在美国，这可是重罪，建议中国立法借鉴。
 <br/>
 <br/>
 这两天人们都在议论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆公交车.html" target="_blank">
  重庆公交车
 </a>
 坠江惨案，15条鲜活的生命，瞬间消逝了，教训极其惨痛。11月2日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 警方公布事故调查结果，证实失事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司机.html" target="_blank">
  司机
 </a>
 和一名女乘客在出事前一直在争执，并相互殴打。
 <br/>
 <br/>
 有网友贴文称，如果类似事件发生在美国，别说乘客与司机互打，只要乘客做出任何干扰司机正常驾驶的行为，包括大声讲电话、伸手触碰方向盘，屡劝不听者，司机就会请该乘客下车。
 <br/>
 <br/>
 据陆媒报导，2002年，纽约州立法规定，攻击公交车司机或地铁工作人员为刑事重罪，等同于攻击警察。公交车司机座位的上方有摄像头，车头位置有一个明显的告示写道：〝攻击巴士司机为刑事重罪，将处以7年监禁。〞
 <br/>
 <br/>
 在美国新泽西州，公交车司机座与乘客座位之间地板上有一道很粗的白线，车辆行驶过程中，所有乘客都不得跨越白线站在司机座位附近。如果有人试图跨越白线或停留在白线以内，司机会立刻发出警告。而如果有人对司机进行攻击，将被处以5年监禁和7000美元罚款。
 <br/>
 <br/>
 网友称，中国司机与乘客互殴酿悲剧，应该借鉴美国的相关规定，如果发生攻击司机的事件，可以提起刑事诉讼，以保障司机和全体乘客的安全。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
