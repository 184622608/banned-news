### 港媒：贸战刹车面临3关 G20会晤若失败将更激烈
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】即将举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20川习会.html" target="_blank">
  G20川习会
 </a>
 ，给僵持的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 带来一线曙光。但有分析认为，川习会晤有可能会使贸战减速，但要让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸战踩刹车.html" target="_blank">
  贸战踩刹车
 </a>
 则面临
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_3大难关.html" target="_blank">
  3大难关
 </a>
 。如果会晤失败，贸易争端可能会更加激烈。
 <br/>
 <br/>
 中美双方正在为11月末在阿根廷G20峰会期间举行的〝川习会〞做准备。目前，双方团队正加紧洽商条件，希望两国元首在G20会面时，不会不欢而散，至少可以达成某种协商框架。
 <br/>
 <br/>
 香港经济日报称，在两国元首会面前，美中重启外交安全对话，中共最高外交主管杨洁篪表示，中美两国官员应妥善管控分歧，精心做好准备，确保两国元首阿根廷会晤取得积极成果。
 <br/>
 <br/>
 报导称，中方强调要〝管控分歧〞，就是不想在川习会之前再生枝节。但如果进行全面评估，中美要结束
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的可能性很低。
 <br/>
 <br/>
 观察人士也普遍认为，川习会晤结束贸易战的可能性较低，但两国藉此达成协商贸易协议的框架，并非毫无可能。
 <br/>
 <br/>
 也就是说，川习会晤有可能会使贸易战减速，但要让贸战刹车则机会不大。而且，若两国元首在G20谈不拢，贸易战可能会再次激化。
 <br/>
 <br/>
 另外，报导指，北京要想让贸易战踩刹车，也不那么容易，至少面临三大难关。
 <br/>
 <br/>
 其一，北京给出的条件未必能让川普满意。会谈能否得出结果，最关键的是北京开出的条件是否足够满足川普。美国对中共发动贸易战，除了指向美中巨额贸易逆差外，还涉及知识产权、开放市场和停止〝中国制造2025计划〞。
 <br/>
 <br/>
 贸易逆差并不是最大问题，北京已多次表示愿意大量购买美国商品，以减少逆差。但余下三点，北京则不容易妥协。
 <br/>
 <br/>
 川普日前表示，北京表示愿意放弃〝中国制造2025计划〞，但中方对此至今未发表任何言论。有观点认为，北京可能是口头上放弃，行动上仍暗中进行。另有分析称，以目前美国国内政治气氛及白宫团队鹰派之强势，川普很难接受一种模糊言词的协议。
 <br/>
 <br/>
 其次，若川普降低条件接受北京妥协，可能会成为民主党另一攻击点。民主党反过来会说川普向北京投降，损害美国利益。
 <br/>
 <br/>
 第三，白宫内的鹰派也不会轻易让川普接受一份〝投降协议〞。早前彭博社曾报导，川普已指示高层官员凖备中美贸易协议的草案，意味着贸易战有停火的可能。
 <br/>
 <br/>
 但川普首席经济顾问、白宫国家经济委员会主任纳瓦罗（Larry Kudlow）对媒体否认此事。纳瓦罗表示，关于对华贸易问题〝目前未有重大的进展〞，美方尚未〝收到中国（共）对贸易问题的回答〞，暗示11月底的川习会，不会成为中美贸易战的终点。
 <br/>
 <br/>
 而且，美国商务部长罗斯、贸易代表莱特希泽、与国家安全事务助理博尔顿等对华态度强硬的官员，也不会轻易让北京过关。
 <br/>
 <br/>
 此外，美国的政治制度设计让国会对总统的国内政策有更多的制衡和监督，如果此次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20川习会.html" target="_blank">
  G20川习会
 </a>
 没有结果，贸易战可能会比以往更加激烈。美国不仅可能会从明年1月1日将关税调高至25%，更有可能会余下的2670亿美元中国商品征税。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
