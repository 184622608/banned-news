### 调查：中国高科技电子行业 员工自杀率最高
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】日前，两家非政府组织发表的共同调查报告显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国高科技.html" target="_blank">
  中国高科技
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电子行业.html" target="_blank">
  电子行业
 </a>
 员工，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀率.html" target="_blank">
  自杀率
 </a>
 最高。这些员工大多是雇佣的农民工，每周工作时间经常超过80小时，由于工作节奏紧张以及受到不公平的惩罚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心理压力.html" target="_blank">
  心理压力
 </a>
 极大。
 <br/>
 <br/>
 香港与荷兰的两家非政府组织11月14日发表共同调查报告，根据对中国大陆167起自杀与未遂事件的调查结果显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国高科技.html" target="_blank">
  中国高科技
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电子行业.html" target="_blank">
  电子行业
 </a>
 的员工，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀率.html" target="_blank">
  自杀率
 </a>
 最高。
 <br/>
 <br/>
 报告列举了职工自杀的主要原因：来自上司的压力、不合理扣工资、企业拒绝支付奖金以及对失业的焦虑等。报告认为，这些因素给员工造成了严重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心理压力.html" target="_blank">
  心理压力
 </a>
 。
 <br/>
 <br/>
 中国电子行业大多雇佣的是农民工，人数达到数千万人，雇佣他们的企业经常因侵犯劳工法而遭受非政府组织的谴责。
 <br/>
 <br/>
 2016年，荷兰非政府组织电子行业观察组织（Electronics Watch）公布的一份调查报告指出，大陆电子行业工作的工人，每周工作时间经常超过80小时，几乎是中国法定每周工作时间44小时的一倍。
 <br/>
 <br/>
 今年的调查报告并未直接披露企业的名字，只提到一家名叫F的供应商，2010年曾经发生多起职工自杀事件。
 <br/>
 <br/>
 据此前报导，在大陆有多家工厂的台湾富士康集团，2010年，有18名工人从公司大楼房顶跳下，14人当即死亡。富士康是苹果集团的一大供应商。
 <br/>
 <br/>
 今年10月，Apple Watch重庆工厂被指非法雇佣学生，苹果集团对此展开调查。这家工厂被指控雇佣学生，强迫他们像机器人一样组装电子智能手表。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
</div>
