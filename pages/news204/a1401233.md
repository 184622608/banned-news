### 消息人士曝北京让步措施 求〝川习会〞暂停关税战
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞即将于二十国集团（G20）峰会期间登场，有美媒28日称，中方盼望美方暂停关税战，为此，〝川习会〞期间北京很可能再次做出更多让步，包括购买更多的美国商品，改善知识产权法等。
 <br/>
 <br/>
 11月28日，美国政治网站报导称，北京会要求美国暂停关税战，并要求美国承认中国市场经济地位，更希望美国停止关税战，包括美国停止在2019年1月1日时将关税从目前的10%提升到25%。
 <br/>
 <br/>
 报导说，为此北京很可能再次表示购买更多的美国商品，特别是液化气，北京也会保证会改善知识产权法。不过，到目前为止，还不清楚北京要走多远才能满足华盛顿的要求。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 26日告诉《华尔街日报》说，美方唯一能够接受的协议是，中方必须同意对美国开放，与美国公平竞争。至于其它国家的要求，由他们自己决定。
 <br/>
 <br/>
 川普表示〝如果我们没有达成交易，那么我将对其它的2,670亿美元中国商品加征10%或25%的关税。〞
 <br/>
 <br/>
 11月30日至12月1日，阿根廷首都布宜诺斯艾利斯召开的二十国集团（G20）领袖峰会期间，川普与习近平将共进晚餐，讨论中美贸易问题及其它议题。
 <br/>
 <br/>
 中方盼望此次〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞双方能达成协议，缓解
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 。为了给〝川习会〞铺路，中美双方自11月开始释放对话信号，以缓和当前局势。近期中方对贸易战的态度，已从〝以牙还牙〞转为了〝推动达成双方都能接受方案〞。
 <br/>
 <br/>
 有匿名中共高官对媒体说，最近习近平特别强调，要控制好反击贸易战的影响，避免对中共继续开放产生负面效应。
 <br/>
 <br/>
 有北京政治观察家分析，经过这一年争拗，中共已接受〝中美已经回不到过去〞这一现实，放弃了以反击迫使美国回到贸战之前的想法，准备好迎接新形态的中美关系，以〝不抵抗〞为底线，寻找这一世界上最重要双边关系的新平衡。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
