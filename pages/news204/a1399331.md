### 习李同期出访 港媒：分工不同释放政治信息
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】本周，北京最高领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 和国务院总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 ，将分别出席不同地区的国家领导人峰会。港媒对两人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交分工.html" target="_blank">
  外交分工
 </a>
 做出解读。
 <br/>
 <br/>
 13日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 已抵达新加坡，未来几天将相继出席中国-东盟峰会、东盟与中日韩峰会，以及东亚峰会。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 15日也将赶赴巴布亚新几内亚，出席亚太经济合作会议（APEC峰会）。
 <br/>
 <br/>
 香港《明报》评论文章指，在中共的多边外交秀中，习近平按惯例会出席APEC峰会、G20峰会、金砖国家峰会、上海合作组织峰会，而李克强则负责东亚（东盟）系列峰会、中日韩峰会、中国与中东欧国家峰会、中欧峰会以及亚欧峰会。
 <br/>
 <br/>
 据此总结，习近平主要负责对美、俄、非洲及中亚的外交，李克强主打应对欧洲和东南亚。
 <br/>
 <br/>
 文章说，中共外交有〝大国是关键，周边是首要，发展中国家是基础〞之说。如此看来，习近平似乎负责〝大国外交〞，李克强则主管〝周边外交〞。
 <br/>
 <br/>
 不过这个分工也不那么绝对，比如中欧峰会都是由李出面。文章指，这可能是历史原因所致，因为1998年首次中欧峰会是由总理朱鎔基出席，因此成了惯例。
 <br/>
 <br/>
 另外，习近平负责金砖国家与上合组织，再加上中非论坛、一带一路等，这显示发展中国家这个〝基础〞也是习主管。
 <br/>
 <br/>
 另外，因为习身兼中共总书记，周边外交中与朝鲜、越南、老挝等共产党国家的双边关系，也是由习出面。
 <br/>
 <br/>
 文章指，中共外交的最高决策机构是中共中央外事委员会，习是主任，李是副主任，外交大局最后都是由习拍板。
 <br/>
 <br/>
 今年以来，无论习、李、副主席王岐山和外交部长王毅等人，出访频率都大增。外界分析，在中美贸易战中，北京正面临被国际孤立的局面，因此不得不到处拉拢〝盟友〞。
 <br/>
 <br/>
 日前，北京数十年的〝宿敌〞日本，也被中方拉入〝共存共荣〞的圈子中。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
