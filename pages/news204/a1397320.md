### 日本30年援华3千亿 揭鲜为人知的内幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】在日本首相安倍晋三访华前夕，日本政府宣布，将在本年度终止日本对华海外发展援助计划。此消息震惊许多中国民众。据悉，日本30年来援华金额近3千亿人民币，但由于中共的刻意隐瞒，中国民众对此知之甚少。
 <br/>
 <br/>
 10月23日，在安倍晋三访华前夕，日本外相河野太郎在记者会上正式宣布，日本对华的海外发展援助计划，将在本年度终止。他表示，日本首相安倍25日访华时，将正式告知中方这一决定。
 <br/>
 <br/>
 安倍晋三25日访华期间表示，中国现已发展成为全球第二的经济大国，日本对华经济援助（Official Development Assistance，ODA）的历史使命已经完成。
 <br/>
 <br/>
 日本停止援华计划的消息令许多中国民众震惊，而中共官方媒体对此则轻描淡写，并未报导详细内容。
 <br/>
 <br/>
 美国南卡大学教授谢田对自由亚洲表示，中共官方对外来援助历来都不予宣传，生怕民众知道真实情况。〝有一个日本人告诉我，他参加了北京中日友好医院的奠基，发现中国人，包括医院高层和一般医生，都不知道这个医院所有的钱都是日本援助资金，他感到很愤怒。即使出于礼貌的提一下，中国政府基本上都不愿意。〞
 <br/>
 <br/>
 现旅居日本的专栏作家牛彬说，过去40年，日本对中国的援助数目相当巨大。〝从79年开始，到08年的时候低息贷款结束，总共援助的资金大约3.3万亿日元（约2千亿人民币），其中无偿的有1700亿日元（约105亿人民币），技术转让资金1800亿日元（月111亿人民币），大的项目330多件。〞
 <br/>
 <br/>
 日本对华援助的项目分为几个不同的阶段，由最早的道路、发电厂、电话线路等基础设施，到后来的铁路、港口和机场，再到最后的自来水工程及环保技术和人才培训等。
 <br/>
 <br/>
 牛彬介绍说，中共改革开放之初，国际社会包括世界银行都不愿意贷款给中共，而日本政府推动日资银行向中国政府项目和企业进行大笔贷款，是中国经济启动的第一笔资金。
 <br/>
 <br/>
 名叫袁晓彬的网易专栏作家发表文章，总结出从1979年开始，日本对中国的无偿和低息贷款援助，总额达到2900亿人民币。文章援引联合国国际开发署的数据称，日本约占所有国家对华援助金额的67%。
 <br/>
 <br/>
 亚洲事务高级顾问、华盛顿战略与国际研究中心中国问题专家葛来仪（Bonnie S.Glaser）表示，在过去40年中，这一援助对中国经济转型做出了〝巨大贡献〞。她说，〝日本对此发挥了很大作用，但功劳很少算到日本头上。〞
 <br/>
 <br/>
 文章称，援助大部分以贷款名义进入中国，直到10年前不再这样操作，日本从未正式宣称援助代表了某种形式的战争赔偿，但日本的分析人士表示，至少在项目开始时确实有过这种色彩。
 <br/>
 <br/>
 牛彬表示，日本对华援助有为过去战争忏悔的因素，也有希望和中共改善关系的盼望。但这个盼望，到近年基本上彻底消失。
 <br/>
 <br/>
 〝日本社会在2012年有一个彻底反思，觉得原来以为中国经济发展可以改善人权和促进民主，结果走向了反面。后来安倍晋三上台，就成了比较强硬的态度。〞牛彬说。
 <br/>
 <br/>
 纽约评论员朱明对大纪元表示，中共向来不讲信誉，它把经济援助拿到手之后，往往挪作别的用途，不会按照援助国的意图来行事，所以还是富了政府和贪官，百姓受益很少。
 <br/>
 <br/>
 朱明说，中共过去讲什么饿着肚子发展核武器，所以它经济强大后，首先是发展技术和武器，威胁别国，西方国家的经援并没有用来富民。中共和朝鲜的做法是一脉相承的。
 <br/>
 <br/>
 虽然日本对华援助了巨额款项，但中国民众对此却知之甚少。除了日本之外，袁晓彬从各国官方对外援助的资料中发现：到2009年，中国总共接受外国官方的援助1072亿美元，其中60多亿美元为无偿援助，包括德国的12亿，美国的5.5亿，欧盟的10亿，澳大利亚的8亿等等，其中最多的还是日本。
 <br/>
 <br/>
 有中国网友开玩笑说，中国是一个奇怪的国家，对中国援助越多的国家，中国人越仇视。过去几年，中国至少发生过三次大型的反日运动，在华日本企业也感到了越来越大的压力。
 <br/>
 <br/>
 谢田教授表示，这是中共官方宣传的误导所致，其核心是中共对民主和自由社会的仇视。〝援助最多的基本上都是西方自由民主的国家，中共是专制体制，它本身对自由民主就是仇视的，所以宣传上有很多刻意的安排。〞
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
