### 忧日加入〝毒丸计划〞 中共对日方强硬举动视而不见
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】北京正在高规格接待来访的日本首相安倍晋三，党媒开足马力宣传〝中日共存共荣〞。虽然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍访华.html" target="_blank">
  安倍访华
 </a>
 前接连释放强硬信号，但北京一再委屈求全。外媒形容这让一贯反日的五毛们〝无所适从〞。日媒指，中共现在非常担心日本加入美国的〝毒丸计划〞。
 <br/>
 <br/>
 《日本经济新闻》日前评论称，中美贸易战升温，中共期待拉拢日本制美。而东京既要考虑中方期望，又要顾虑美国感受，安倍必然有所避忌，需要在中美夹缝中小心摸索。如果安倍不小心掉入北京陷阱，成为中共抗美的〝帮手〞，那日本就将变成中美关系的〝磨心〞。
 <br/>
 <br/>
 文章举例说，安倍即将出席〝中日第三国市场合作论坛〞，推进双方在发展中国家进行经济合作，这其实就是中方〝一带一路〞的变种称谓，而白宫已明确表态对抗〝一带一路〞。
 <br/>
 <br/>
 日媒认为，美加墨三国刚达成的〝贸易新协议〞（USMCA），禁止签约方与中共发展贸易关系，而美国也在努力将日本拉入这个被称为〝毒丸计划〞的圈子。北京盛待安倍，有防止日本加入的目的。
 <br/>
 <br/>
 近日，日本政府的接连动作显示，安倍在与中共靠近之前，已在多个领域宣示了日方的立场和底线，相信也有向美方表态之意。而中方的反应，似乎多显示接受的态度。
 <br/>
 <br/>
 10月14日，安倍检阅了日本最大规模的阅兵式，并宣称下一步将修宪，把自卫队列入日本宪法。而日本今年刚成立的水陆机动团，也与美军举行了夺岛演习。这些动作明显针对正在对外扩张军事势力的中共。不过，中共官方的反应相当〝克制〞。
 <br/>
 <br/>
 外界分析，美军的保护是日本应对中共威胁的安全保障，这是日本难以动摇的底线。
 <br/>
 <br/>
 10月18日，安倍内阁的3名副大臣和4名政务官参拜了靖国神社。中共党媒非但没有发声谴责，反而强调安倍〝降低参拜规格〞，没有带阁僚参加，宣称这是安倍向中方释放〝友好〞信息。
 <br/>
 <br/>
 10月23日，亦即《中日友好条约》40周年纪念日，日本突然宣布将停止对华〝政府开发援助〞（ODA），要求和中方在〝平等立场〞上讨论新的合作方法。而中方的回应，并非沉默或谴责，而是开始高调宣传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本对华ODA.html" target="_blank">
  日本对华ODA
 </a>
 的功绩。
 <br/>
 <br/>
 不过在此之前，中共对日本的表态也曾有过反击。这显示中日的靠近也是在你来我往的互相试探中进行。
 <br/>
 <br/>
 9月下旬，当日本和欧盟、美国发表部长联合声明、谴责中共贸易不公之后，北京随即以〝不方便〞为由，要求安倍将原定23日抵京的访华行程向后推延一两天。北京突然拒绝在《中日友好条约》40周年纪念日接待安倍，显然是在释放〝不友好〞的信号。
 <br/>
 <br/>
 有分析指，中方这一表态，似乎意在警告安倍不能在中美贸易战中帮助美国。
 <br/>
 <br/>
 不过，中共盗窃知识产权和强制转移技术，早已是国际社会谴责的对象。许多西方国家都表达了借中美贸易战向中共施压、要求北京遵守国际规则的意向。
 <br/>
 <br/>
 日媒也表示，安倍政府与中方开展科技合作前，首先要举行中日〝创新与知识产权对话〞，显示了日本对中共的戒心以及要求北京改变做法的意图。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
