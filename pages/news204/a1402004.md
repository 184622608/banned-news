### 传500余中国艺人已被约谈 有男星当场痛哭
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 逃税事件引发的中国影视圈税务〝地震〞持续大震荡，近日陆媒爆料有17位艺人已经被约谈之后，最新消息称，被约谈的演员已经多达551人。其中一位男星被约谈时，在会议室〝当场就痛哭起来〞。
 <br/>
 <br/>
 12月5日陆媒称，有中国影视圈业内人士表示，中共官方下决心整治娱乐圈乱象了，且不会姑息和手软，〝编剧、制片、导演、宣传、设计等各环节从业人员也被纳入〝自查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_补税.html" target="_blank">
  补税
 </a>
 〞范围内。
 <br/>
 <br/>
 一份《横店工作室会议内容》文件11月28日称，中共税务总局下达浙江税务局，要求影视工作室展开税务自查自纠，影视圈的补税工作正式实施。税务部门已在3天内约谈了17名艺人，不过并未透露姓名。
 <br/>
 <br/>
 11月30日，网上传出了17位补税演员的名单，全都是一线演员，黄晓明、杨颖夫妇、吴奇隆、周迅、杨幂、赵薇、赵丽颖等都在名单内。
 <br/>
 <br/>
 陆媒引述爆料称，〝上海沧州影视基地，税务稽查，邓超孙俪两人补了2.5亿，消息准确！〞除了邓超孙俪之外，还有很多一线演员都被谈话，黄晓明就在其中。另有消息指出，其中一位男星被约谈时，在会议室〝当场就痛哭起来〞。
 <br/>
 <br/>
 根据上述圈内人士的爆料，这次中共税务总局开始严厉打击娱乐圈逃税漏税现象，第一批次的〝1人〞是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 ；第二批次150人（其中17人为重点约谈，已经约谈了），35人为第二重点约谈对象，第三批次200人，第四批次200人。
 <br/>
 <br/>
 前前后后四个批次加起来要约谈的多达551人，数量和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 曝的585人很接近。约谈对象几乎覆盖一二三线所有以〝工作室形式〞纳税的演员。
 <br/>
 <br/>
 引发此次演艺圈大地震的导火索源自于央视前主持人崔永元的爆料，崔永元于5月底开始曝光女星范冰冰娱乐圈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 事件，之后，大陆影视娱乐演员及其公司的偷税漏税问题逐渐浮出水面。
 <br/>
 <br/>
 7月9日，崔永元又发文称，已经整理出来了585名跟华谊合作过演职人员，并把这个名单交给中共国家税务局。
 <br/>
 <br/>
 由于崔永元的爆料，范冰冰于6月起消失于公众视线，直到10月3日，中共税务机构公布案情，宣布范冰冰涉〝阴阳合同〞逃税案，缴纳约8.8亿元人民币税款，可免于牢狱之灾，至此范案终于落幕。
 <br/>
 <br/>
 然而范冰冰事件引发的巨震仍未消停，舆论场的各种声音仍是此起彼伏。大批中国影视圈人士深受范冰冰事件影响，娱乐行业税务地震暗潮汹涌。
 <br/>
 <br/>
 中国经营报9月就曾引述一位不愿具名的娱乐圈老板透露，受范冰冰事件影响，娱乐圈发生大地震，除了公布的演员工作室税收税率从6%提高到42%外，政府口头传达要求各路演员10月份必须要补缴完税收，娱乐圈一众演员为补税忙得团团转。
 <br/>
 <br/>
 他说，除了被曝光的演员外，大部分艺人都有偷税漏税的问题，歌手更严重，前200名艺人和前50间影视公司都会中招。
 <br/>
 <br/>
 另一位娱乐圈人士透露说，所有艺人补的税加起来会是一大笔，演员工作室是用来避税的，大部分演员都有偷税漏税，演员就是为了逃税才开工作室。
 <br/>
 <br/>
 一位圈内人士分析说，税不可怕，可怕的是蝴蝶效应，不但在影视娱乐圈爆发，更会席卷所有广告主。税务+限薪令会来一场风暴。因为新税制实行，近几个月来已经有多间影视宣传公司倒闭。
 <br/>
 <br/>
 曾经在中国一线娱乐节目担任导演的姜先生对德媒说，娱乐界新税制实行，是中共政府到娱乐圈〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_割韭菜.html" target="_blank">
  割韭菜
 </a>
 〞，总的来说是国库缺钱。由于当局处于中美贸易战，地方财政状况堪忧，崔永元的爆料，无疑给当局增加财政收入提供了绝佳的机会。
 <br/>
 <br/>
 之所以针对演员，他说，是因为演员们收入高，民怨也高，确实也存在逃税漏税的问题。而中共从演员大腕中割〝韭菜〞来充实国库，网友调侃演员大腕为当局〝共克时艰〞的〝功臣〞。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：戴明）
