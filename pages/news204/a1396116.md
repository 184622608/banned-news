### 女儿被罚站关押女教师7小时 派出所所副所长被免职
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南.html" target="_blank">
  湖南
 </a>
 一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_派出所.html" target="_blank">
  派出所
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副所长.html" target="_blank">
  副所长
 </a>
 因女儿上学迟到被罚站几分钟，却
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关押女教师.html" target="_blank">
  关押女教师
 </a>
 7小时事件，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_株洲县.html" target="_blank">
  株洲县
 </a>
 有关部门19日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通报.html" target="_blank">
  通报
 </a>
 ，涉事的派出所副所长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵明众.html" target="_blank">
  赵明众
 </a>
 ，因违规使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公权力.html" target="_blank">
  公权力
 </a>
 ，决定给予记大过处分、免去副所长职务，并调离公安系统。
 <br/>
 <br/>
 根据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_株洲县.html" target="_blank">
  株洲县
 </a>
 有关部门调查，8月15日8点多左右，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵明众.html" target="_blank">
  赵明众
 </a>
 的女儿在育红小学上学时，因迟到被班主任老师何某罚站后，赵明众以女儿被何某打耳光为由，安排渌口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_派出所.html" target="_blank">
  派出所
 </a>
 一名警员和一名辅警前往学校调查。
 <br/>
 <br/>
 9点左右，何某被带入派出所执法办案区的等候区。25分钟之后，何某被安排到派出所办公楼会议室，期间有辅警陪同。在会议室内，何某承认让赵明众的女儿罚站，但否认打过耳光。通过对班级部分学生调查，当时无人看见何某打人。
 <br/>
 <br/>
 在会议室内，何某及其亲属向赵明众妻子道歉，表示学生如身体有伤害，愿意承担责任。后经医院检查，确认赵的女儿身体没问题、也没有被打过的痕迹。15日14点11分，何某离开派出所。
 <br/>
 <br/>
 经调查，赵明众违规派警员处理涉及其本人家庭成员与他人的纠纷，且处置不当，决定给予赵明众记大过处分，由县公安局免去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_副所长.html" target="_blank">
  副所长
 </a>
 职务，由组织人事部门将其调离公安系统。
 <br/>
 <br/>
 大陆网媒文章18日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南.html" target="_blank">
  湖南
 </a>
 株洲育红小学一名三年级女孩，因迟到被何姓女教师罚站数分钟，女孩父亲、株洲渌口派出所一副所长知悉情况后，驱警车直入学校，将何姓教师带走，并关入审讯室7小时。
 <br/>
 <br/>
 根据何姓女教师在社媒中发布的网帖：她说自己在派出所全程被人监视，限制人身自由，没给过一口水，一粒饭……从派出所出来的那一刻，泪水从来没有停止过……我从来没有因为孩子迟到，打过哪个孩子一巴掌！我勤勤恳恳教书，为什么会受到这样的待遇……？
 <br/>
 <br/>
 事件引发舆论热议：网民抨击：〝典型的滥用公权，应该立即双开并逮捕这个副所长，执法犯法者罪加一等〞、〝可怕，让这种人手里握枪，说不定哪天不爽了，拔枪就杀人〞、〝此副所长应该马上开除公职，永远不予安排，已经伤透了所有教师的心。〞
 <br/>
 <br/>
 18日下午，株洲方面回应称，目前纪委已介入，教育局和公安局都已成立小组调查此事，涉事副所长已被停职接受调查。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：李泉）
