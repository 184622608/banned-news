### 孟宏伟被宣布辞职主席  国际刑警〝立即生效〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】截止10月7日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 失踪已超过10天，总部设在法国里昂的国际刑警组织称当日收到孟宏伟的辞呈，并宣布孟宏伟辞去主席一职〝立即生效〞。
 <br/>
 <br/>
 国际刑警组织7日在推特（Twitter）发表声明称：〝10月7日，位于法国里昂的国际刑警组织总秘书处，收到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 先生的辞呈，立即生效。〞
 <br/>
 <br/>
 同日，中纪委官网通报，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 孟宏伟涉嫌违法，目前正接受中共国监委调查。
 <br/>
 <br/>
 9月4日，孟宏伟的妻子向法国里昂警方报案说，孟宏伟9月29日返回中国后音讯全无。
 <br/>
 <br/>
 法国警方4日晚确认〝孟宏伟不是在法国境内失踪的〞、〝全案不是单纯的刑事案件〞。目前警方已对此立案，并由里昂警署展开调查。
 <br/>
 <br/>
 10月5日，港媒《南华早报》爆料，孟宏伟抵达中国后，立即被中纪委〝带走〞，目前正接受调查。据悉，他涉及经济问题及其他贪腐问题，包括他违规在香港购置房产。
 <br/>
 <br/>
 国际刑警组织6日利用执法渠道向北京了解孟宏伟的情况，表示了担心，之后，中共官方在国际舆论压力下，终于承认孟宏伟被调查。
 <br/>
 <br/>
 国际刑警组织成立于1923年，成员国有190个，是各国为打击刑事犯罪而形成的合作组织。孟宏伟于2016年底，当选国际刑警组织主席，任期将于2020年届满。
 <br/>
 <br/>
 在孟宏伟当选国际刑警组织主席之初便引来很多争议。评论认为，中共公安部有侵犯人权的不良纪录，由中共公安部副部长担任国际刑警组织主席，将影响该组织的公信力。
 <br/>
 <br/>
 孟宏伟同时还兼任中共公安部副部长。有关孟落马此前就有预兆。孟长期在公安部任职，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 交集不少。去年10月中共十九大上，郭声琨升任政法委书记后，其空缺被中共河北省委书记赵克志接替。赵克志上任后，多次要求公安部继续清洗周永康的〝余毒〞。
 <br/>
 <br/>
 2017年底，孟宏伟被免除中共国家海洋局副局长、中共海警局局长的兼任职务，2018年4月，又被免去已担任14年的公安部党委委员一职。此次如果孟宏伟作为周永康的〝余毒〞被拿下也不令人意外。
 <br/>
 <br/>
 今次孟宏伟被秘密抓捕，港媒《苹果日报》分析说，孟宏伟可能涉嫌中共政治忌讳，涉嫌北戴河政变案，因为案件重大而紧急，中共才不顾国际影响而紧急对孟宏伟问话。
 <br/>
 <br/>
 法国《世界报》评论说，孟宏伟案让中共内部的争斗与贪腐问题被搬到国际舞台检视。因此孟宏伟所涉及的问题，必定极为敏感或严重，才会让北京宁可冒着对国际出丑也要逮人的政治风险。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：祝馨睿）
</div>
