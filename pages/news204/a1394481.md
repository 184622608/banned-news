### 港媒：北京突释〝求和〞信号 拟邀川普访华
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 日前发表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对华演说.html" target="_blank">
  对华演说
 </a>
 ，对中共内外政策提出严厉批评。外界认为，美中〝新冷战〞自此拉开序幕。但10月7日，有港媒爆料，北京高层正考虑邀川普再度访华，向美国求和。
 <br/>
 <br/>
 10月4日，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）在华府发表川普政府的对华政策，对中共全方位开火，严厉抨击中共在贸易、军事、人权以及干预美国政治等多个方面的恶行。
 <br/>
 <br/>
 外界认为，彭斯公布的对华政策充分显示，美国对中共的不满，已不仅限于贸易争端，美中〝新冷战〞已经拉开序幕，两国关系急转而下。
 <br/>
 <br/>
 10月8日，香港《苹果日报》报导，据多名中共官员及智囊透露，北京当局想利用明年美中建交40周年的契机，向美国求和。
 <br/>
 <br/>
 消息人士称，中南海领导层希望于2019年1月1日，隆重纪念美中建交40周年，并希望于今年年底纪念《中美建交公报》发表40周年。
 <br/>
 <br/>
 北京高层希望借这个机会，邀请美国总统川普（特朗普）再度访华，修补中美之间的裂痕。
 <br/>
 <br/>
 消息人士透露，目前，北京中央层级的多名智囊，正在为此事进行紧张的筹备。
 <br/>
 <br/>
 消息称，这是北京当局向川普发出的〝求和〞信号。以缓解贸易战以来中美双方越来越紧张的关系。
 <br/>
 <br/>
 日前对于彭斯的演说，中共表现的惶惶不安，在大陆全面封杀。中共外交部更是罕见的一日之内做出两次回应，显示川普政府的对华政策，已对中共产生巨大杀伤力。
 <br/>
 <br/>
 卡内基国际和平基金会的副总裁包道格（Douglas Paal）认为，彭斯的演说〝是一个进行战斗的号声〞，并认定美国会赢得这场斗争。
 <br/>
 <br/>
 研究美国外交政策的专家傅克斯（Michael Fuchs）说，彭斯的演讲清楚表明，贸易战只是一个开始，川普政府的对华政策正在走向对中共更为强硬的阶段。
 <br/>
 <br/>
 傅克斯认为，今后美国可能会在关税和贸易领域，对中共采取更多的行动；而在安全领域，如南中国海和台湾问题上，美国也可能会对中共采取更为强硬的举措。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：祝馨睿)
</div>
