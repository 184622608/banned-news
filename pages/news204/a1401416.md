### 川普志在必得？美方10人全是鹰派
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】11月30日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 登场的二十国集团（G20）峰会，因中美首脑会晤变得举世瞩目。有港媒称，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 将率9人顾问团队与习近平在会议间隙的晚餐上会谈。由于美方10人全属于对北京鹰派人物，被认为是川普此次与习近平谈判志在必得。
 <br/>
 <br/>
 20国集团（G20）峰会，于11月30日至12月1日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 首都布宜诺斯艾利斯登场，会期两天已经定为阿根廷国定假日，将取消所有航班并安排转降、火车、地铁和大众运输。峰会期间，阿根廷当局将部署2.2万警力驻守市内要道。
 <br/>
 <br/>
 美国国安团队已提早12日抵达布宜诺斯艾利斯，协助阿根廷加强维安。
 <br/>
 <br/>
 在两天的峰会期间，美国海军军舰并派3架侦察机〝E-3 Awacs〞监测当地，并以透过搜集〝网路情报〞辅佐。
 <br/>
 <br/>
 此次峰会的主角之一中国国家主席习近平，已于当地时间11月29日上午乘专机离开西班牙马德里，前往阿根廷首都布宜诺斯艾利斯。
 <br/>
 <br/>
 美国白宫消息称，习近平将于当地时间12月1日晚，同美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 共进晚餐。
 <br/>
 <br/>
 《南华早报》报导，川普预计将率领9人的顾问团队赴布宜诺斯艾利斯同习近平举行会晤，团队中除白宫贸易鹰派顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 （Peter Navarro）外，白宫高级顾问、美总统川普（Donald Trump）女婿库什纳（Jared Kushner）和美国国家安全委员会亚洲事务高级主任波廷格（Matt Pottinger），预计也将陪同川普参加在阿根廷二十国集团（G20）峰会之余举行的中美双边会晤。
 <br/>
 <br/>
 其他6人包括白宫幕僚长凯利（John Kelly）、贸易代表莱特希泽（Robert Lighthizer）、国务卿蓬佩奥（Mike Pompeo）、国安顾问博尔顿（John Bolton）、白宫经济顾问库德洛（Larry Kudlow）和财长努钦（Steven Mnuchin）。
 <br/>
 <br/>
 一位消息人士称，尚不确定上述9人是否都会参加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晚宴。
 <br/>
 <br/>
 不过，据《华尔街日报》援引美国官员的话透露，库什纳将参加晚宴。中方的团队中的关键代表目前尚不明确。
 <br/>
 <br/>
 根据习近平此前参加外交峰会的惯例，除了负责中美贸易谈判的副总理刘鹤外，习的随行人员还可能包括中共办公厅主任丁薛祥、负责外交事务的政治局委员杨洁篪、外长王毅和商务部部长钟山。
 <br/>
 <br/>
 这是中美元首在贸易战爆发后的首次会晤。由于美方10人全属于对北京鹰派人物，此次〝习川会〞被认为是川普志在必得的信号。
 <br/>
 <br/>
 G20〝习川会〞将决定世界前两大经济体之间的经贸关系，也势必影响全球经济。
 <br/>
 <br/>
 自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 开打以来，美国已经宣布对总值2500亿美元中国商品征收关税。
 <br/>
 <br/>
 外界认为，此次〝习川会〞习近平要求美国延后执行25%的利率。但川普11月16日表示，他不大可能接受北京的这一要求。
 <br/>
 <br/>
 川普16日表示，如果在G20〝习川会〞上，北京方面没有任何重大让步，他将把2000亿美元中国输美商品的关税税率从10%提高至25% ，并警告还将对剩下的2670美元中国输美商品加征关税。
 <br/>
 <br/>
 被认为是鹰派中的鹰派的白宫贸易顾问纳瓦罗，曾撰写出版《即将到来的中国战争》（The Coming China Wars）和《致命中国》（Death by China）等书，并对中共持强烈批评态度。
 <br/>
 <br/>
 纳瓦罗也被认为是中美关税战的发起者之一。因此他加入会晤，被认为意在向北京方面展示川普政府的决心。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明）
