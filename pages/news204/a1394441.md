### 北京已绝望？彭斯演讲后央行降准再放水7500亿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】〝十一〞长假最后一天，中共央行宣布年内第四次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降准.html" target="_blank">
  降准
 </a>
 ，释放7,500亿人民币资金。3天前，美国副总统彭斯发表演讲痛斥中共，被指〝新冷战宣言〞。此前海外专家指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央行放水.html" target="_blank">
  央行放水
 </a>
 的速度显示〝北京已绝望〞。
 <br/>
 <br/>
 10月7日，据中共央行官网消息，从2018年10月15日起，将下调大型商业银行、股份制商业银行、城市商业银行、非县域农村商业银行、外资银行人民币存款准备金率1个百分点。
 <br/>
 <br/>
 报导称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降准.html" target="_blank">
  降准
 </a>
 所释放资金，部分用于偿还10月15日到期的约4,500亿元人民币中期借贷便利（MLF）。除此之外，降准还会再释放约7,500亿元的增量资金。
 <br/>
 <br/>
 一如既往，央行称释放资金的目的是〝支持小微企业〞。
 <br/>
 <br/>
 10月4日，美国副总统彭斯在华盛顿发表演讲，在多个层面对中共进行严厉谴责。海外时评人士纷纷表示，该演讲相当于〝新冷战宣言〞。中国网络上也开始流传：〝新冷战开始，你准备好了吗？〞
 <br/>
 <br/>
 台媒披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯演讲.html" target="_blank">
  彭斯演讲
 </a>
 前后，中共驻美大使馆已经连续加班一周。上述央行消息显示，中共国内机构也在加班加点，研究对策。
 <br/>
 <br/>
 央行宣布这一轮放水操作，距离上一轮刚刚20天。
 <br/>
 <br/>
 9月12日，美媒披露川普政府将对2,000亿中国产品加征关税。之后6天内，中共央行通过4次操作，累计净投放达5,950亿元人民币。
 <br/>
 <br/>
 美国时事评论家章家敦当时在推特评论称，〝以这个速度印钞表明中国（中共）很绝望，局势危急。〞
 <br/>
 <br/>
 大陆自媒体此前统计，为应对贸易战，仅仅今年上半年，央行就已通过各种名目放出超过4万亿资金，中国大陆将很快迎来大规模
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通货膨胀.html" target="_blank">
  通货膨胀
 </a>
 。
 <br/>
 <br/>
 每到经济陷入困境时，印钞票刺激经济都会成为中共的必然选择，但其沉重负担最后都转嫁到民众身上。日前中国房租和物价都开始全面上涨，中产阶级迎来消费降级，社会底层生活更为困难。
 <br/>
 <br/>
 但陆媒都避谈央行大印钞票的负面影响。而在一些自媒体和社交媒体上，常见民间对中共此举或明或暗的讽刺。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
