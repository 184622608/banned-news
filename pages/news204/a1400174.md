### 赵薇夫妇因收购违规5年内被禁任上市公司董事
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】本周二，上海证券交易所发布通报，宣布对违规收购的祥源文化（原名万家文化）、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_龙薇文化.html" target="_blank">
  龙薇文化
 </a>
 及有关责任人予以处分，其中女影星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵薇.html" target="_blank">
  赵薇
 </a>
 及其丈夫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄有龙.html" target="_blank">
  黄有龙
 </a>
 ，被罚5年内不得担任上市公司董事、监事或高级管理人员。
 <br/>
 <br/>
 上海证券交易所11月20日发布的通告指称，龙薇传媒作为上市公司的收购人，在自身资金准备不足、资金来源存在极大不确定性的情况下，贸然筹划收购万家文化（即祥源文化），而且并未充份提示终止风险。在股权转让过程中，相关资料披露还存在虚假记载、误导性陈述、重大遗漏及披露不及时的问题，严重损害了投资者知情权。
 <br/>
 <br/>
 通报称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵薇.html" target="_blank">
  赵薇
 </a>
 的丈夫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄有龙.html" target="_blank">
  黄有龙
 </a>
 作为龙薇传媒的代表，组织、策划、指派了相关人员进行控股权转让，并实际与万家集团实际控制人孔德永进行谈判，决策收购祥源文化控股权。
 <br/>
 <br/>
 赵薇身为龙薇传媒法定代表，在黄有龙告知其收购祥源文化控股权事项后表示同意，并在收购控股权的文件上签名，在公告发布前也看过资料披露内容。
 <br/>
 <br/>
 赵政则受黄有龙指派，代表龙薇传媒负责收购谈判、寻找资金等。祥源文化董事长孔德永全程组织、策划并参与收购事宜，是祥源文化违规行为直接负责人。
 <br/>
 <br/>
 最后，上证所决定对浙江祥源文化股份有限公司及时任董事长孔德永，西藏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_龙薇文化.html" target="_blank">
  龙薇文化
 </a>
 传媒有限公司及其直接负责人员黄有龙、赵薇、其他直接责任人员赵政予以公开谴责，并认定孔德永、黄有龙、赵薇、赵政5年内不适合担任上市公司董事、监事、高级管理人员。
 <br/>
 <br/>
 <h4>
  赵薇夫妇违规收购事件回顾
 </h4>
 <br/>
 2016年12月26日，万家文化发布公告称，龙薇传媒计划出资30.6亿元（人民币）买下万家集团的1.8亿股份，持股比例占29.13%，从而成为万家文化的实控人。
 <br/>
 <br/>
 这一纸公告，震惊了整个娱乐圈跟财经界，人们纷纷质询：赵薇用于收购的30亿资金是从哪里得来的？
 <br/>
 <br/>
 2017年1月11日，谜底揭晓，原来赵薇夫妇这次收购万家文化，仅有6000万元是自有资金，计划要向西藏银必信借15亿元，再向其他金融机构借15亿元，总共融资30亿来完成这项收购计划。而令外界更为震惊的是，他们融资贷款时抵押的竟然是还未到手的万家文化股票和赵薇个人的名誉。
 <br/>
 <br/>
 实施这起高槓杆收购的主要载体龙薇传媒，则是2016年11月2日才成立的公司。该公司刚成立一个多月，在尚未开展任何实际的经营活动，总资产、净资产、营业收入、净利润都为零的情况下，就试图通过抛出6000万元的自有资金，使用50倍的槓杆来撬动一家市值约100亿元的上市公司。这种高槓杆收购方式被网民形容为〝空手套白狼〞。
 <br/>
 <br/>
 由于这起收购案的内情被媒体曝光并在网络上引发舆论讨论热潮，证监会开始对这起事件展开调查。
 <br/>
 <br/>
 2017年11月9日，祥源文化（即万家文化）发表公告，承认该公司已收到证监会的行政处罚及市场禁入事先告知书。根据这份告知书，赵薇夫妇被指在收购上市公司过程中，涉及虚假记载、误导性陈述和重大遗漏等违规行为，被证监会给予警告，分别处以30万元罚款，且被实施五年内禁入证券市场的处罚。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：戴明）
