### 中共整顿网贷平台将再现倒闭潮
------------------------

<div class="wysiwyg">
 【新唐人2018年10月20日讯】今年6月到8月，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网贷平台.html" target="_blank">
  网贷平台
 </a>
 从最高峰的六千多家骤降至一千五百多家。大陆理财专家表示，政府整顿网贷平台的原因是，中国经济下滑，资金紧缺，想将民间资金导入国家银行，性质跟〝割韭菜〞一样。因此估计未来能生存下来的平台不多。
 <br/>
 <br/>
 今年8月，中国互联网金融协会下发《关于开展P2P网络借贷机构自律检查工作的通知》到各省市网贷整治办，要求他们在2018年12月底前完成合规检查。经过检查的平台将被〝分级分类〞处置，还要经过繁琐的手续，条件成熟的网贷机构才可申请备案。
 <br/>
 <br/>
 大陆理财专家透露，中美贸易战，中国经济下滑，资金出现紧缺，银根收紧， 政府收紧
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网贷平台.html" target="_blank">
  网贷平台
 </a>
 的政策，是想将民间资金导入国家银行。
 <br/>
 <br/>
 大陆理财专家邹先生：〝在今年的6月份到8月份这段时间，大概倒闭了六分之五的平台了，有四、五千家吧，现在剩下一千多家，最高峰的话，查到的有六千多家，因为越来越多的人把钱存到P2P里，国家那些银行怎么办？它们都没有资金投入，国家就有权的去控制，就跟割韭菜一样这种性质。〞
 <br/>
 <br/>
 据《金融时报》报导，多名业界高管表示，新的监管浪潮将迫使多数公司退出P2P行业，他们认为能够生存下来的企业可能有几百家，或者远低于50家。
 <br/>
 <br/>
 大陆理财专家也表示， 政府对网贷平台的政策在收紧，能生存下来的平台将不多。
 <br/>
 <br/>
 大陆理财专家张先生：〝今年市场不稳定，然后P2P很多平台都已经倒闭了，高获利就有高风险对不对？现在的话，国家还在整顿当中，不建议再做P2P，国内现在经济情势不好，就建议慢慢退出来吧。〞
 <br/>
 <br/>
 大陆理财专家还透露，今年8月以来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_倒闭潮.html" target="_blank">
  倒闭潮
 </a>
 中倒闭的网贷平台，很多是涉及人数较多、金额较大的平台，还有一些是地方小平台没被查到，估计有两、三千家，依然监管不到。
 <br/>
 <br/>
 新唐人记者熊斌、舒灿采访报导
</div>
