### 朝鲜核心机密外泄 祸起金正恩随身马桶
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 的健康信息被视为朝鲜核心机密。因此金正恩不论走到哪里，都会带着自己专用的随身马桶。但金正恩的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健康状况.html" target="_blank">
  健康状况
 </a>
 仍然被韩国方面掌握。韩国情报院院长近日透露，金正恩患有家族病。他自带马桶，是为了防止有关其个人身体健康的信息被泄露。
 <br/>
 <br/>
 韩国《中央日报》23日报导说，韩国国情院院长徐熏，在参加国情院国情监察会议时透露，〝金氏家族有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高血压.html" target="_blank">
  高血压
 </a>
 家族病史〞。
 <br/>
 <br/>
 为了防止有关个人身体健康的信息被泄露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 不论到哪里，都会带着专属的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_流动厕所.html" target="_blank">
  流动厕所
 </a>
 。
 <br/>
 <br/>
 曾服役朝鲜领导人卫队的一名保镖李永国（Lee Young Guk）称，朝鲜领导人拥有私人厕所，他们出访期间也带着，因他们的排泄物藏有个人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健康状况.html" target="_blank">
  健康状况
 </a>
 的信息，所以不能落入他人手中。
 <br/>
 <br/>
 《每日朝鲜》报导说，不论是金正恩的个人列车里，还是他旅行用的中小型轿车中，甚至专门在山区地形或雪地上行驶的车辆中都有洗手间。他的车队中有许多车辆，人们无法分辨他到底在其中的哪一辆车上，车队中有一辆单独的汽车是他的厕所。
 <br/>
 <br/>
 曾任金氏家族贴身护卫的脱北者李尹镐（Lee Yun-keol）也曾透露，金正恩自带厕所原因是粪便会透露他的健康状态资讯。外界曾猜测，金正恩患有痛风、高血压、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性病.html" target="_blank">
  性病
 </a>
 和其它
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心理疾病.html" target="_blank">
  心理疾病
 </a>
 等。
 <br/>
 <br/>
 金正恩6月份抵达新加坡与美国总统川普（特朗普）举行历史峰会期间，《今日美国》当时披露，金正恩不仅自带食物，还自带马桶，而且还不止一个马桶，甚至在豪华车内还有一个专门上厕所的地方，以防止情报部门试图收集有关他健康的信息。
 <br/>
 <br/>
 韩国发行量最大的报纸之一Chosunilbo报导，朝鲜给金正恩准备马桶，〝确保‘敌人’不能对最高领导人的粪便进行分析。〞而金正恩的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_屎尿.html" target="_blank">
  屎尿
 </a>
 在朝鲜都要定期接受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_检测.html" target="_blank">
  检测
 </a>
 ，以确认金正恩健康状况是否良好。
 <br/>
 <br/>
 对于共产党党魁这种防范心理，英媒BBC曾报导，早在苏联时期，苏联情报部门就曾建立秘密实验室，分析世界各地领导人的粪便。
 <br/>
 <br/>
 报导说，前苏联特工伊戈尔・阿塔马年科，在研究俄罗斯情报部门档案的过程中，发现了斯大林曾通过这种方法秘密分析毛泽东10天的排泄物。
 <br/>
 <br/>
 他介绍，建立这种分析排泄物秘密实验室的工作，是由斯大林的情报助手贝利亚负责的，目的是通过分析粪便了解被检测者的心理状态。
 <br/>
 <br/>
 如在粪便中发现高含量的色氨酸，就说明这个人心态平和，能够接近；如粪便中缺少钾，则说明此人心绪焦虑，可能有失眠问题。
 <br/>
 <br/>
 毛泽东的具体数据是什么报导没提。但苏联情报部门分析了毛10天的排泄物后，斯大林最终做出决定，和毛签署了一份协议。
 <br/>
 <br/>
 俄媒报导，斯大林的继任人赫鲁晓夫最终关闭了这个秘密实验室。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：祝馨睿）
