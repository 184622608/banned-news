### 韩媒：朝鲜副总理秘密访华 习近平即将访朝
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】韩国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文在寅.html" target="_blank">
  文在寅
 </a>
 日前透露,
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 近日将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访朝.html" target="_blank">
  访朝
 </a>
 。韩媒称，朝鲜副总理李龙男10月9日秘密访华，或前来协商习近平访朝的事宜。
 <br/>
 <br/>
 10月9日，韩联社报导，在美国国务卿蓬佩奥成功结束亚洲行之后，朝鲜半岛紧张局势有所缓解，朝鲜内阁副总理李龙男（音译）9日访问中国。
 <br/>
 <br/>
 上午10时30分左右，李龙男到达北京首都机场，朝鲜驻华大使池在龙，和中共对外联络部在机场贵宾室迎接，随后乘坐中共国宾车辆，往北京市内方向开去。
 <br/>
 <br/>
 分析认为，此次朝鲜高层官员访华，很有可能是协商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访朝.html" target="_blank">
  访朝
 </a>
 的相关具体事宜。
 <br/>
 <br/>
 《联合早报》称，韩国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文在寅.html" target="_blank">
  文在寅
 </a>
 10月8日出席内阁会议时透露，习近平计划前往朝鲜访问，但他没有进一步说明详情。
 <br/>
 <br/>
 韩国驻华大使卢英敏在8日的座谈会上说，习近平访问朝鲜事宜，属于韩国与中共紧密沟通的领域之一。
 <br/>
 <br/>
 中共〝外交部发言人办公室〞官微内容显示，中共外交部9日回应有关习近平即将访问朝鲜传闻时，只是答非所问的说，中朝是友好邻邦，双方保持着这一传统。
 <br/>
 <br/>
 朝鲜领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 今年上半年已经三次访问中国，首次访华时，已经对习近平提出了访朝邀请，据传，习接受了邀请。之后，频频传出习将访问朝鲜，参加在9月9日朝鲜建政70周年活动。
 <br/>
 <br/>
 但鉴于美中贸易战不断升级，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系.html" target="_blank">
  中美关系
 </a>
 格外紧张敏感，习近平这次访朝之旅最终未能成行。外界认为为，习近平因担心影响到中美关系，而取消了这次访朝计划。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：赵云）
</div>
