### 刘强东〝倒霉〞速度太快  政协委员头衔不保
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】自美国涉嫌性侵案发后，京东CEO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 霉运接踵而至。刘强东在连续缺席多项重要场合后，坊间又传出他已退出京东管理层，如今其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政协委员.html" target="_blank">
  政协委员
 </a>
 头衔也恐不保。有陆媒提出，刘强东应〝切割〞相关角色，包括辞去中共全国政协委员职务等。
 <br/>
 <br/>
 大陆门户网站新浪网23日刊发署名文章说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 作为一名企业家，事业正在风头上，因为私德问题翻船，其性侵案继续发酵，不守道德的刘强东应〝切割〞相关角色，包括辞去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政协委员.html" target="_blank">
  政协委员
 </a>
 职务等。
 <br/>
 <br/>
 报导说，刘强东此时主动辞去全国政协委员职务〝是明智之选〞，与〝全国政协委员〞这个角色切割后，刘强东可以静下心来打官司，同时安排好京东集团的各种〝后手〞。
 <br/>
 <br/>
 刘强东于今年3月份第一次被选为全国政协委员，至今还不满一年。
 <br/>
 <br/>
 在中国，拥有中共人大代表或政协委员的身份，对富豪从事的商业活动尤其有利，增加他们参与政府基建或国企投资的机会，又有很多机会与国家领导人会面。
 <br/>
 <br/>
 但大陆富商即使身为人大代表或政协委员，也不能确保他们的政治安全。在中国，因违法犯罪问题而被撤消政协资格的人比比皆是，比如中共〝最美政协委员〞女商人刘迎霞，2014年被撤销政协委员资格后，至今仍下落不明。
 <br/>
 <br/>
 报导表示，万一美国法院最终判刘强东有罪，或万一〝女对手〞不肯私下了结，此事不断发酵，刘强东很可能要从京东董事局主席兼首席执行官位置上下台，现在安排好〝切割〞方案，下一步京东集团的震动也可以小一些。
 <br/>
 <br/>
 8月31日，刘强东在美国涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 被捕入狱，16个小时后获释。根据美国警方9月4日公布的报告，刘强东在美性侵案属于〝一级性犯罪：强奸既遂〞。
 <br/>
 <br/>
 明尼苏达亨内平县警方9月21日宣布，已完成对案件的初步调查，接下来将由检方接手，最终决定是否起诉。根据当地法律，刘强东所涉的一级强暴罪被定罪后，刑期至少12年起。
 <br/>
 <br/>
 报导称，希望刘强东认错改错，重新做人，即使最终入狱服刑，也应该通过合适的方式维护好京东的运营，比如换一个董事局主席和首席执行官。
 <br/>
 <br/>
 自性侵案爆发后，刘强东已经从中国的政商圈消失，除了事件早期现身京东以示辟谣后，刘强东几乎缺席所有重要的场合。舆论纷纷猜测，刘强东遭北京〝抛弃〞了。
 <br/>
 <br/>
 随着刘强东性侵丑闻发酵，京东集团的股价也急转直下，持股机构数量也暴跌。2018年第二季度的持股机构达585家，而第三季度仅有100家。
 <br/>
 <br/>
 今年45岁的刘强东不仅是中国商界
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亿万富豪.html" target="_blank">
  亿万富豪
 </a>
 ，同时也是中共全国政协委员，并兼任中国民间商会副会长。作为中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红顶商人.html" target="_blank">
  红顶商人
 </a>
 ，在美国涉及强奸犯罪指控，无疑也是中共一大丑闻。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：范铭)
