### 中共清网延伸海外自媒体 要求推特文章逐条删除
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】中共最近大规模下架大陆〝自媒体〞帐户后，把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清网.html" target="_blank">
  清网
 </a>
 行动延伸到了海外自媒体。近日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推特.html" target="_blank">
  推特
 </a>
 等自媒体被当局盯上，中国各地大批维权人士被警方约谈，要求逐条删除推文，有人因拒绝删除推文而遭逮捕、扣押。
 <br/>
 <br/>
 据《德国之声》引述中国网路自由观察者野渡的话称，近期，中共网路讯息相关部门大举出动网路警察，对中外各大平台上的大陆网友的言论进行审核。
 <br/>
 <br/>
 警方近日向野渡出示一个表格，详细记录他在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推特.html" target="_blank">
  推特
 </a>
 上发表的802条〝问题推文〞，该表格把他过去几年在推特上发表的推文分类，认为他涉及所谓不法内容包括：〝鼓吹西方民主〞、〝涉及六四〞、〝煽动社会运动〞等细项，要求他逐条删除掉。
 <br/>
 <br/>
 野渡说，在他认识的网友中，过去一个月内就有超过数十人遭到警方约谈，也是要求他们删除，中共所认为的〝问题推文〞，面对当局无理施压，野渡没有配合删除推文。
 <br/>
 <br/>
 11月18日，野渡再发推文说，〝风暴确实来了，我被要求删除共计802条推文〞，网友担心他的安危，纷纷发推询问他的情况。
 <br/>
 <br/>
 《维权网》16日引述大陆多位维权人士的话披露，最近，北京、重庆、山东、湖北、福建和广东等，多地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部门.html" target="_blank">
  公安部门
 </a>
 ，展开对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Twitter.html" target="_blank">
  Twitter
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清网.html" target="_blank">
  清网
 </a>
 行动。来自重庆的刘继春及福建厦门的潘细佃等异见人士，因拒绝按当局要求删文，已被公安带走。
 <br/>
 <br/>
 而被强行删除的Twitter账号，包括2015年已入狱的异见人士吴淦，其网名是〝超级低俗屠夫〞，他的推特账号@tufuwugan，和他在推特上2015之前所发布的全部内容，已于11月9日全部被删除。
 <br/>
 <br/>
 报导说，拒绝配合中共删推文的权权人士中，重庆刘继春已被逮捕，居住福建厦门市的潘细佃，已被公安局扣押。在这场清网行动中受冲击的人数，目前还在不断增多。
 <br/>
 <br/>
 《北京之春》消息称，推特内容被中共当局删除的人士包括，学者高瑜、陈永苗、记者文涛、律师覃永沛、网路观察员古河、刘强本、维权人士何德普等人。
 <br/>
 <br/>
 消息说，中共的推特清网，能如此精确地锁定推友账号和密码，显示了当局对美国社交媒体，尤其是对推特的渗透和控制又升了一级。
 <br/>
 <br/>
 此外，在国内，中共更是疯狂删贴封号。11月12日，网信办称，自10月20日针对自媒体展开专案整治行动以来，已全网处置9,800多个自媒体账号，并对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微信.html" target="_blank">
  微信
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微博.html" target="_blank">
  微博
 </a>
 等平台提出严重警告。
 <br/>
 <br/>
 16日网信办再发消息称，约谈了10家客户端自媒体，包括：百度、腾讯、新浪、今日头条、搜狐、网易、凤凰、知乎等，要求他们按照全网一个标准，全面自查自纠。
 <br/>
 <br/>
 分析认为，当局的用意显然是抢占舆论高地，让中国民众噤声，以延续其苟延残喘的专制政权。
 <br/>
 <br/>
 中共军报2015年曾刊文称，一个国家的倾覆，始于思想的瓦解。报导举例说，突尼斯、利比亚和埃及等国家，都是被〝推特〞推倒，一夜之间垮台的值得深思！
 <br/>
 <br/>
 报导还称，〝网路时代的敌人，手中拿着的可不仅是刀枪剑戟，还有滑鼠键盘！〞并指，自媒体网路言论从根本上动摇了中共执政地位云云。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
