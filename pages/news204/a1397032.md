### 汪洋主持会议 宣布撤销孟宏伟一重要身份
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】原国际刑警组织主席、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被调查事件持续发酵，26日在中共全国政协主席汪洋主持的一次会议上，孟宏伟又一重要身份被宣布撤销。
 <br/>
 <br/>
 10月26日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 的中共第十三届全国政协委员身份被撤销。孟此前是第十三届全国政协外事委员会委员。上述决定是在中共政协第十三届全国委员会主席会议第三次集体学习上做出的。中共全国政协主席汪洋主持会议。
 <br/>
 <br/>
 中共官方10月7日深夜通报：孟宏伟涉嫌违法，目前正接受中共监委监察调查。孟宏伟落马当天，国际刑警组织宣布孟宏伟辞去国际刑警组织主席一职。
 <br/>
 <br/>
 次日凌晨，孟宏伟供职的中共公安部连夜召开会议通报其涉嫌犯罪情形。再次强调〝彻底肃清
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 流毒影响〞，同时宣布成立工作组，查处孟的同党。
 <br/>
 <br/>
 10月9日，孟宏伟公安部副部长一职，已由习近平旧部邓卫平接任。
 <br/>
 <br/>
 有关孟宏伟被逮捕的原因有很多传闻：1，政治背叛、派系权力斗争或重大失职。2，孟落马可能与其担国际刑警组织主席没完成北京交待的任务有关。3，贪腐和经济问题。4，孟宏伟被传曾参与2015年北戴河〝政变〞。
 <br/>
 <br/>
 上述诸多猜测都无法得到证实。不过，对于中共高层来说，此案是一件不容忽视的大案。
 <br/>
 <br/>
 外媒分析，北京当局未待孟宏伟任满国际刑警组织主席，即召他回国并秘密逮捕，不顾国际影响做出这一决定，表明事态紧急，北京已无法顾及其负面效应。此事也给中共官场带来另一次震慑效应。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：赵云）
