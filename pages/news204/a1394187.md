### 中美关系尚存转机？川普：愿与习近平谈一谈
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】美中新一轮关税生效之后，两国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 已经上升到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全面对抗.html" target="_blank">
  全面对抗
 </a>
 。随着11月美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 临近，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）对华展示出更强硬动作。但白宫官员称，12月的20国集团会议上，川普愿意和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 谈一谈，美中贸易谈判可能会恢复。
 <br/>
 <br/>
 在中美两国关系从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 蔓延到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全面对抗.html" target="_blank">
  全面对抗
 </a>
 的敏感时刻，10月2日，白宫首席经济顾问库德洛表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 愿意在12月阿根廷举行的20国集团会议上，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 谈一谈。
 <br/>
 <br/>
 10月3日，中共外交部宣布，美国国务卿蓬佩奥将于10月8日对中国进行访问，就中美关系及共同关心的国际和地区问题交换意见。
 <br/>
 <br/>
 对于此次北京重启对话，美国国务院发言人希瑟．奈特表示：在任何关系之中，都会遇到低点，但是，关系会随时间高低起伏，所以，美国肯定有很多东西跟北京谈。
 <br/>
 <br/>
 中美关系在激烈对抗中，似乎又现出一线缓和的希望。
 <br/>
 <br/>
 而此前，川普在9月26日在联大记者会上，指控中共企图干预美国11月期中选举，并首次坦言他跟习近平的交情可能受到损害，他两次提到也许习近平不再是朋友了。
 <br/>
 <br/>
 10月2日，川普总统在美国的一次集会上，又一次指责中共，过去每年都从美国拿走5000亿美元，这简直太疯狂了。他们不仅偷走了美国的财富，还偷走了美国作为一个国家的尊严，是时候叫停这个了。
 <br/>
 <br/>
 同日，在另一集会上，川普说：中共〝正在榨干我们。他们拿走我们的钱重建自己。他们每天造一架飞机。每天都会造很多大桥，那些大桥比华盛顿大桥还大。〞
 <br/>
 <br/>
 〝我们在做什么呢？我们在帮助他们。我们重建了中国。他们拿走了如此多的钱。我喜欢习近平主席。但是我说过这个局面不能再持续。〞
 <br/>
 <br/>
 分析认为，无论是习近平还是川普，12月的会谈都是关键环节，一旦共和党在11月美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 中获胜，对于中共将是一个巨大打击。在这个最大的赌注输掉后，习近平有可能跟川普好好谈一谈。
 <br/>
 <br/>
 另外，中共试图通过绑架14亿民众，来对抗美国发起的贸易战之举已经失灵，许多中国人暗中支持川普打垮中共，解体中共。再加之在经济下行压力巨大的情况下，习近平也有可能在12月与川普坐下来心平气和谈一谈。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
</div>
