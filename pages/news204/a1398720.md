### 百度突发技术故障死机 网民嘲讽：以为被取缔
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】中国大陆搜索引擎百度周五上午因技术故障，发生了搜索死机现象。百度因得到中共官方支持而几乎垄断了中国大陆网络搜索引擎市场，近年来该公司各种丑闻频发，在中国网民中口碑很差。因此，这次百度〝死机〞后，有网友调侃说〝百度终于被取缔了〞，大陆网络社交媒体上众网友嘲讽之声不断。
 <br/>
 <br/>
 当地时间11月9日上午约10时许，百度网站发生搜索死机现象，无法进行任何搜索的情况持续了较长时间。随后，中国网民发现，百度网盘的上传功能也出现异常。
 <br/>
 <br/>
 其后，百度公开承认该公司的搜索引擎的确发生了技术故障，并称是因为运营商DNS问题，影响部分北京地区的联通网络用户正常使用，截至上午11时半仍未解决，正在加紧修复之中。
 <br/>
 <br/>
 中国网络社交媒体上众网友纷纷拿〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_百度死机.html" target="_blank">
  百度死机
 </a>
 〞这个话题开涮。有网友调侃称，〝我还以为百度总算被取缔了。〞还有网友跟贴说，〝刚想开瓶香槟酒庆祝一下呢。〞也有网友针对百度对这次故障的解释嘲讽道，〝运营商又背锅了。〞
 <br/>
 <br/>
 百度网站近年来爆发力不少丑闻。2016年4月，陕西大学生魏则西因误信百度排名第一的医疗广告，接受治疗后癌症恶化去世，踢爆了百度与恶名昭彰的〝莆田系医院〞的利益关系，百度搜索引擎通过竞价排名收取广告资金而胡乱推介资格不足的医疗机构与各种药物广告的问题一度引发网络舆论的大反弹。百度也因此被中国网民戏称为〝百毒〞。
 <br/>
 <br/>
 今年7月初，有中国大陆民众在国产手机VIVO NEX上使用百度输入法搜寻时，偶然发现百度输入法竟有疑似偷偷录音的功能。在用户没有进行录音操作的情况下，VIVO NEX顶部通知栏竟然显示〝百度输入法正在录音〞的讯息。
 <br/>
 <br/>
 该名用户到微博上发帖询问其他网友，结果发现使用相同牌子手机的网民，不少都遇到有关情况。这则消息迅速登上微博热搜榜的前几名。
 <br/>
 <br/>
 之后百度方面则解释，这是为了方便使用语音输入法的用户，可以更快启动语音输入功能，只是被VIVO NEX〝误判〞为录音，但众网友对百度侵犯用户隐私的疑虑并未打消。
 <br/>
 <br/>
 事实上，百度涉嫌侵犯用户个资的丑闻曾多次被踢爆，有百度贴吧用户2017年网购后给王店差评，没想自己的个人资料竟被百度贴吧洩露给了卖家，由此引起一些纠纷事端。此外，百度旗下的App〝手机百度〞、〝百度浏览器〞2018年被网民揭发涉嫌监听电话，遭消保委提出民事诉讼。
 <br/>
 <br/>
 （记者欣然综合报导/责任编辑：明轩）
