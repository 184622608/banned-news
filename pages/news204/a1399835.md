 <br/>
 1. 防疫漏洞？
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉林野猪.html" target="_blank">
  吉林野猪
 </a>
 首爆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 恐侵北韩
 <br/>
 <br/>
 中国非洲猪瘟疫情爆出一大难解的〝防疫漏洞〞，吉林省日前发现首例野猪感染疫情病死案例。专家指，如果官方大规模捕杀受感染野猪，恐导致野猪逃往邻国，造成非洲猪瘟疫情侵入邻近吉林的北韩。
 <br/>
 <br/>
 2. 揭大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五星级酒店.html" target="_blank">
  五星级酒店
 </a>
 丑闻 爆料人疑遭业界封杀
 <br/>
 <br/>
 中国一位网红近日发布一段影片，曝光大陆14家五星级酒店的卫生乱象。据爆料人透露，视频曝光的五星酒店仅仅是暗访的一部分。目前此事件在业界掀起轩然大波，据称爆料人的个人信息已被酒店方洩露，其本人也遭到业界〝封杀〞。
 <br/>
 <br/>
 3. 苹果供应厂商裁员八千 广东工人围堵大门
 <br/>
 <br/>
 近期因苹果手机订单骤减，为苹果生产手机玻璃面板的港企〝伯恩光学〞在广东的工厂，近日大幅裁员工人。工人纷纷围堵在工厂门口，当局派来防暴特警将他们驱散。此外，全球最大的无人机制造商中国大疆也传裁员消息。
 <br/>
 <br/>
 4.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 复出无望？党媒罕见再发严厉批评
 <br/>
 <br/>
 中国女星范冰冰因卷入逃税案遭罚款8.8亿元人民币后，有关她将复出的消息不断传出。然而中共党媒16日再发文，呼吁加强文艺界法治建设，强调不再让有法律和道德污点的艺人出名获利。分析指出，这意味着范冰冰复出之路被堵死。
