### 【 禁闻】10月10日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 上海疫苗受害者谭华遭当局软禁
 <br/>
 <br/>
 上海疫苗受害者谭华，8月29号因联名召开记者招待会，被杨浦区延吉街道派出所警察抄家后带走,  一个多月下落不明。10月8号，她母亲华秀珍发消息说，街道和区维稳办当天通知她，说谭华是被他们软禁了。
 <br/>
 <br/>
 2. 黄琦狱中声明 若换律师将拒绝出庭
 <br/>
 <br/>
 大陆维权网站〝六四天网〞创办人黄琦，被控〝向境外提供国家秘密〞一案，再有新进展，黄琦10月8号委托前去会见的律师发出声明，说自己不会变更律师，如果当局强制变更，他将拒绝出庭。
 <br/>
 <br/>
 3. 上海村民遭入室绑架 警方不立案
 <br/>
 <br/>
 10月6号，上海市闵行区浦江镇群益村维权村民王美华，被闯入家中的一群蒙面人绑架带走，她的丈夫谈永江向当地警方报案，但警方不予立案。目前王美华下落不明。
 <br/>
 <br/>
 4. 女律师遭羞辱 警方回应引舆论哗然
 <br/>
 <br/>
 广州女孙世华律师9月20号到华林街派出所办案时，被警察围殴、脱光衣服羞辱。10月10号，广州警方发布官方通报否认事实，引发网络舆论哗然，网友纷纷要求警方公开现场监控视频。
</div>
