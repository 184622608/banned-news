### 【禁闻】12月2日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 五名金融诈骗受害者天安门
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_喝农药.html" target="_blank">
  喝农药
 </a>
 自杀
 <br/>
 <br/>
 11月30号，北京天安门广场又有5人喝农药自杀。据了解，自杀者3男2女，他们都是天津现货交易所金融诈骗受害者，已经在北京上访一个月。目前，自杀者情况不明。
 <br/>
 <br/>
 2. 久敬庄整修关闭 大批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 被送马家楼
 <br/>
 <br/>
 11月26号，北京知名黑监狱〝久敬庄访民中心〞因进行整修暂时关闭，各地访民被成批转送至另一黑监狱〝马家楼国家接济中心〞。据访民介绍，当天马家楼整排的大巴车装满访民，各地截访人员则在外面等着抓人。
 <br/>
 <br/>
 3. 广西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强征土地.html" target="_blank">
  强征土地
 </a>
 百余村民对抗千警
 <br/>
 <br/>
 11月29号，广西凭祥市板透村遭政府强征土地，一百多名村民前往阻止，被上千警察围困。警方一边放催泪弹，一边拿石头、棍子砸村民，村民则手持木棍、铁锹反抗。最后，40多名村民被抓，4人被打伤，其中一人重伤昏迷。
 <br/>
 <br/>
 4. 湖北潜江百余
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 市府维权
 <br/>
 <br/>
 11月30号，湖北潜江的一百多名退伍老兵在市政府门口维权，要求还他们士官安置名额等，官方派出信访局局长、军人事务局长等到场，但没有政府主要负责人出面解决问题。老兵们最后被维稳人员劝回。
</div>
