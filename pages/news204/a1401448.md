### 遭拖欠工资半年 东莞百余员工讨薪
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东莞.html" target="_blank">
  东莞
 </a>
 迈科科技有限公司拖欠员工工资长达半年，日前，一百多名工人前往该公司最大股东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳坪山沃尔核材厂.html" target="_blank">
  深圳坪山沃尔核材厂
 </a>
 ，在厂区前拉横幅抗议，大批警力到场戒备。
 <br/>
 <br/>
 网上视频显示，一百多名工人拉着横幅，整齐划一地高喊〝还我血汗钱，还我活路〞，大批手持盾牌警察赶到现场。
 <br/>
 <br/>
 佳士工人声援团网站发布的消息说，由于被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拖欠工资.html" target="_blank">
  拖欠工资
 </a>
 长达半年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东莞.html" target="_blank">
  东莞
 </a>
 迈科科技有限公司工人多次维权无果，11月28号，到公司最大股东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳坪山沃尔核材厂.html" target="_blank">
  深圳坪山沃尔核材厂
 </a>
 ，厂区前维权抗议。
 <br/>
 <br/>
 11月30号，新唐人致电深圳坪山沃尔核材厂，接线员起初不承认有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权抗议事件.html" target="_blank">
  维权抗议事件
 </a>
 。当问及百余人维权罢工时，她说，那不叫罢工。
 <br/>
 <br/>
 深圳坪山沃尔核材厂接线员：〝我们公司没有拖欠过我们公司员工的工资，他不是我们公司的员工，谈不上罢工，前几天是有人来（抗议），这个事情跟我们公司没有什么关系，不知道他为什么会来我们这里，他公司的股东差不多有10来个，我们只是之一。〞
 <br/>
 <br/>
 这位接线员表示，的确有大批手持盾牌警察赶到现场。
 <br/>
 <br/>
 深圳坪山沃尔核材厂接线员：〝我们工厂那么多员工都在，不可能（让）他们闹事，警察肯定是要来。也有劳动仲裁的（人）过来，他们跟公司（抗议）交谈了一下，后来就走了吧。〞
 <br/>
 <br/>
 公开资料显示，东莞市迈科科技有限公司创建于2000年，是一家专业研发、生产和销售电池及相关配套电子产品的高新技术企业，员工约4000余人。
 <br/>
 <br/>
 新唐人记者常春、李沛灵采访报导
</div>
