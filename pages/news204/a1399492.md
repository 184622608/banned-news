### 【禁闻】中共利用高中生研发杀手机器人
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 利用北京理工大学招募了一批〝最聪明〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高中生.html" target="_blank">
  高中生
 </a>
 ，参与研发人工智能〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杀手机器人.html" target="_blank">
  杀手机器人
 </a>
 〞。评论认为，这既会害了这些高中生，也给社会带来危害。
 <br/>
 <br/>
 最近，英国《每日邮报》和香港《南华早报》等媒体报导，北京理工大学公开招募了31名所谓的〝最聪明〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高中生.html" target="_blank">
  高中生
 </a>
 ，打算通过一项为期4年的〝智慧武器系统实验计划〞，把他们培养成〝世界最年轻的AI武器科学家〞。
 <br/>
 <br/>
 报导说，这31名高中生是从5,000多名候选人中挑选出来的精英人才，其中包括27名少年和4名少女，年龄都不到18岁。
 <br/>
 <br/>
 人工智慧，英语是Artificial Intelligence，缩写为AI。人工智慧武器，也被称作〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杀手机器人.html" target="_blank">
  杀手机器人
 </a>
 〞（killer robots），可以在没有人指挥的情况下，自主选择和销毁人类等目标。
 <br/>
 <br/>
 原大陆教师李言表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 能够公然地对外界公布这个项目，大张旗鼓，说明它已经不避讳、不回避它杀人嗜血的本性。不仅如此，中共还藉此倡导杀人，让全社会认可它的合理性及合法化。
 <br/>
 <br/>
 原大陆教师李言：〝如果北京理工大学的这个高中生研发智能武器是杀人的，这个项目它不仅害了这些高中生，也给社会带来危害。〞
 <br/>
 <br/>
 这些高中生想通过选拔，不仅要有优异的成绩，而且所谓的〝爱国水平〞也是被考察的方向。此外，这些学生还必须具有所谓〝创意、敢于斗争〞和迎难而上的这类特质。
 <br/>
 <br/>
 李言：〝这个要求，我们想一想意味着什么呢？就是说，这些孩子你的大脑思维只有充斥杀人，杀人的这种思维达到极致的那种状态的时候，才能不断的具有创造性，从而达到杀人的研发武器的这个结果。〞
 <br/>
 <br/>
 中共培训下一代学生将AI武器化，也引发国际科学政策专家们的忧惧。联合国大学政策研究中心新兴网络技术学者鲍威尔斯（Eleonore Pauwels）说，这令人极度不安。因为〝机器人蜂群能够在食品和生物技术供应链中，输送有害毒素。〞
 <br/>
 <br/>
 加州大学伯克利分校智能系统中心主任拉塞尔（Stuart Russell）认为，这是〝一个非常糟糕的想法〞，他表示，〝追求致命性自主武器的开发，将会严重危害到国际、国家、地方及个人的安全。〞
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝中共通过所谓的‘爱党爱国’洗脑，蒙骗了很多年轻学生为他们卖命。这些学生不仅主动投入研发可以杀人的武器，（有些）还在留学时就帮中共监视海外异议人士和法轮功学员，帮中共刺探西方国家的情报，而且毫无负罪感。所以这是一个很可悲的事情。〞
 <br/>
 <br/>
 事实上，全世界有许多科学家对于发展AI武器可能给世界带来毁灭性灾难深感忧惧。
 <br/>
 <br/>
 但中共却视而不见。今年7月，中共发布了人工智慧发展计划，声称，未来几年内，大陆人工智慧产业产值将推升到1,500亿美元。到2030年，中共将建设中国成为所谓的〝人工智慧创新中心〞。
 <br/>
 <br/>
 郑浩昌：〝因为共产主义的终结目的是毁灭人类嘛，所以中共一旦掌握了最新的人工智慧武器，就会对其他非共产国家构成严重威胁。刀呢，既是日常生活中好使的工具，但它也可成为杀人利器，所以这个不能不防。〞
 <br/>
 <br/>
 今年7月中旬，有2,400多位世界知名的AI专家联署了一份特殊的宣言，主要内容就是承诺〝永不发展致命的AI武器系统〞。
 <br/>
 <br/>
 这份《致命性自主武器宣言》（lethal autonomous weapons pledge）指出，掌控人类个体生死的决定权，决不能交给没有感情的机器。这不仅是出于道德层面的考虑，也是因为致命性的AI武器一旦拥有自主选择攻击目标的能力，那么，它们将成为暴力活动中的强大工具。
 <br/>
 <br/>
 采访编辑/常春 后制/陈建铭
