### 孟晚舟如何被抓？ 加国总理：事前接到通知
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 〝第一公主〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在加拿大被捕的消息，引发外界高度关注。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大总理特鲁多.html" target="_blank">
  加拿大总理特鲁多
 </a>
 日前就此事首次回应称，加拿大政府没有涉及这起事件，只是在事发前几天，才接到有关拘捕孟晚舟的通知。
 <br/>
 <br/>
 12月6日，特鲁多（Justin Trudeau）在蒙特利尔出席一个科技会议时，首次就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被抓一事作出回应。他表示，加拿大政府没有涉及这起事件。渥太华只是在事发前几天,才接到有关拘捕孟晚舟的通知。但他拒绝透露更多细节。
 <br/>
 <br/>
 特鲁多说：〝相关当局在此案中所做的决定没有任何政治参与或干预〞，因为加拿大政府尊重独立的司法程序，〝我们只是在事前几天接到通知。〞
 <br/>
 <br/>
 对于他是否同中共总理或大使谈论过此事，特鲁多表明，他没有就此案同任何国际同僚谈过。
 <br/>
 <br/>
 据加拿大司法部12月5日的声明称，中国电信巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 副董事长、首席财务官孟晚舟是于12月1日在加拿大温哥华被捕的，美国政府正寻求引渡她，保释听证会将在12月7日举行。
 <br/>
 <br/>
 孟晚舟身份特殊，她是华为董事长任正非的长女，外界称其是华为〝第一公主〞。此次，她因涉嫌违反美国对伊朗的出口禁令，加拿大政府应美国当局要求，将其逮捕。
 <br/>
 <br/>
 加拿大司法部称，由于孟晚舟被捕时提出的禁止报导令已生效，当局无法提供进一步信息。
 <br/>
 <br/>
 孟晚舟被捕事件曝光后，引爆舆论关注，她被捕的细节也随后流出。12月6日，华为董事、高级副总裁陈黎芳在推特说，孟晚舟是在加拿大温哥华机场转机时，被加拿大当局代表美国政府扣留的，美国要求引渡孟晚舟。
 <br/>
 <br/>
 路透社在2013年1月曾经报导，香港公司Skycom Tech涉嫌向伊朗最大的运营商，违规销售美国惠普公司的计算机设备——惠普称它禁止向伊朗出售任何其产品，而孟晚舟担任Skycom的董事职位。
 <br/>
 <br/>
 根据报导，至少有13页Skycom的商业文件打着〝华为机密〞的字样并带有华为标志。华为当时受到美国众议院情报委员会的指责，但华为对此事予以否认。
 <br/>
 <br/>
 另外，2016年，美商务部公布中国电信巨头中兴通讯公司的不当行为时，曾揭示另一家公司在逃避美国的出口管制，但是公开文件没有点名这家公司，只用了代号F7。当年，10名美国会议员在致美商务部的信函中写到，他们相信F7就是华为。
 <br/>
 <br/>
 今年4月，中兴通讯公司因违规向伊朗出口产品被美国制裁。随后，美国司法部也对华为展开调查。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
