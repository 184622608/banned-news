### 马云任上最后一年 光棍节刺激消费
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】今年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴.html" target="_blank">
  阿里巴巴
 </a>
 公司在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双11.html" target="_blank">
  双11
 </a>
 的销售金额，再创记录，预计这将是，由该公司创始人兼董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马云.html" target="_blank">
  马云
 </a>
 掌舵的最后一次双11销售活动。该公司9月份表示，现任执行长张勇将在明年接任董事长职位。
 <br/>
 <br/>
 中国〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双11.html" target="_blank">
  双11
 </a>
 〞网购交易惊人，今年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴.html" target="_blank">
  阿里巴巴
 </a>
 的〝天猫网〞在1点47分26秒，就突破1000亿人民币，比去年快约7小时。
 <br/>
 <br/>
 阿里巴巴今年虽然销售增长保持稳定，但由于中国和美国之间的贸易紧张局势加剧，经济形势不确定，年内股价下跌了16％。
 <br/>
 <br/>
 本月早些时候，阿里巴巴下调其全年收入预期，令投资者感到寒意。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马云.html" target="_blank">
  马云
 </a>
 的资产自今年减少了80多亿美元，他将在12月卸任，2020年将离开阿里巴巴董事会。
 <br/>
 <br/>
 中国的网路零售商曾因兜售低质量和假冒商品而受到抨击，分析师认为，双11鼓励购物者抢购廉价商品，导致他们购买他们也许并不需要的商品。
 <br/>
 <br/>
 新唐人记者严枫、天羽综合报导
</div>
