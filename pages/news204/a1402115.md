### 【禁闻】川普再发推 中共首度承认90天谈判期限
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】
 <b>
  川普再发推 中共首度承认90天谈判期限
 </b>
 <br/>
 <br/>
 〝川习会〞之后，各界都在关注美中在未来90天内的贸易谈判，但中共方面一直沉默。
 <br/>
 <br/>
 美国总统川普12月4号再次发推文表示，美国要么与中国达成一份真正的协议，要么就不要达成任何协议。
 <br/>
 <br/>
 川普在推文中说：〝那样的话，我们将针对中国输美产品课征巨额关税。〞
 <br/>
 <br/>
 12月5号，中共商务部终于在网站上发表文件承认，川普与习近平的会晤很成功，两国将在90天内推进谈判。并表示，北京将尽快落实已达成共识的具体事项。
 <br/>
 <br/>
 <b>
  美媒：中共〝巧取〞波音绝密
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卫星技术.html" target="_blank">
   卫星技术
  </a>
 </b>
 <br/>
 <br/>
 中共盗取美国技术的丑闻再次被曝光。《华尔街日报》12月5号报导说，中共政府通过资金控制的一家美国初创公司，向美国波音公司订购一颗卫星，意图取得波音公司的卫星绝密技术。
 <br/>
 <br/>
 报导说，波音公司位于洛杉矶的工厂内，工人们即将完成一颗新卫星的制造。这颗卫星使用了美国军方依赖的限制级技术，订购方是一家寻求改善非洲网络访问的本地初创公司。
 <br/>
 <br/>
 但事实上，根据公司记录、法庭文件以及该项目知情人士披露的情况，这颗卫星得到了中共政府大约2亿美元的资金支持。
 <br/>
 <br/>
 《华尔街日报》说，这项技术能帮助中国填补一项空白。而获得这类技术将为中共的太空计划，以及主导尖端产业和扩大在发展中国家影响力的举措提供支持。
 <br/>
 <br/>
 <b>
  德国人权专员欲访
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
   新疆
  </a>
  遭中共拒绝
 </b>
 <br/>
 <br/>
 德国一个人权代表团日前要求访问中国新疆，调查中共当局设立拘留中心，关押大批维吾尔穆斯林的情况。但遭到中共当局拒绝。
 <br/>
 <br/>
 据〝美国之音〞报导，德国人权专员科夫勒12月4号表示，他们向中共当局提出访问新疆的要求，是为了准备年度德中人权对话。
 <br/>
 <br/>
 科夫勒说，她对媒体报导的维吾尔少数民族遭受的对待很震惊，据说有100多万
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维吾尔人.html" target="_blank">
  维吾尔人
 </a>
 被关押在新疆的拘留营。
 <br/>
 <br/>
 科夫勒说，她将继续向中共当局提出访问新疆的要求，并将在拉萨12月7号和7号举行的德中人权对话会议上提出德国对中国的宗教自由、公民社会和其他人权问题的关切。
 <br/>
 <br/>
 编辑/周玉林
