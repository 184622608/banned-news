### 川普指中共不仅偷钱 还偷国家的尊严
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】自中美双方开始互征
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 逐步升级。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普（特朗普）.html" target="_blank">
  川普（特朗普）
 </a>
 2日再一次指责中共，过去每年都从美国拿走5000亿美元，他们不仅偷走了美国的财富，还偷走了美国作为一个国家的尊严。
 <br/>
 <br/>
 10月2日，川普总统在美国的一次集会上说：〝我们正在审视中国（共）和一些其他国家，我们正在纠正一些可怕的协议，这些协议偷走了我们的财富，我们的就业岗位，偷走了我们如此多的东西。事实上，还偷走了我们作为一个国家的尊严。〞
 <br/>
 <br/>
 川普还说：〝我们对中国（中共）不公平的贸易政策和行为，也采取了史上最强硬的行动。〞中共过去每年都从美国拿走5000亿美元，是时候叫停这个了。这简直太疯狂了。
 <br/>
 <br/>
 同日，川普在另一集会上说：中共〝正在榨干我们。他们拿走我们的钱重建自己。他们每天造一架飞机。每天都会造很多大桥，那些大桥比华盛顿大桥还大。〞〝我们在做什么呢？我们在帮助他们。我们重建了中国。这个局面不能再持续了。〞
 <br/>
 <br/>
 中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 已经持续了近3个月，美国已向2500亿美元的中国商品征税，中共也向美国1100亿美元的商品征税。
 <br/>
 <br/>
 川普说，如果中共敢打击美国的农民，他将会再向中共另外2670亿美元的商品征税。〝因此，我们会向5000亿美元的商品征税，如果把所有的加起来的话，最终有可能是5620亿美元。〞
 <br/>
 <br/>
 川普还说：〝你知道吗？他们想要谈判，他们想要达成协议。中国在过去几个月里被重创。他们的股市暴跌了30%，而自我当总统以来，我们的股市上涨了55%。〞
 <br/>
 <br/>
 中共外交部发言人华春莹10月3日宣布，经中美双方商定，美国国务卿蓬佩奥将于10月8日对中国进行访问。中美双方将就中美关系及共同关心的国际和地区问题交换意见。
 <br/>
 <br/>
 英媒BBC分析说，碍于中美近期的贸易纠纷，蓬佩奥访华的行程可能会出现变数，甚至能否成行也成问题。
 <br/>
 <br/>
 受贸易战的影响，中美近期多次还推迟或取消了两国官员原定的会面。美国9月扩大对中国货物征税的范围之后，中方取消副总理刘鹤赴美谈判的计划。而中共外交部9月2日宣布，应美方要求，也将推迟两国的第二轮外交安全对话。
 <br/>
 <br/>
 与此同时，川普政府全方位追击中共，除布局贸易战之外、外交战、经济战、军事战、金融战等战术，都将同步对中共进行施压。
 <br/>
 <br/>
 据美国政治新闻网站Axios引述川普政府的消息来源披露，川普当局已经拟妥一套大规模的〝秘密反制中共计划〞，准备在未来几周之内公布实施。
 <br/>
 <br/>
 这项制裁计划的规模涉及众多政府部门，不仅要揭露中共对美国的网路攻击、选举干预、商业窃密等证据，还将揭露中共如何渗透美国，并宣布美国的反击行动。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
