### 网传冯小刚被罚20亿元 当局近期公布细节
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 因逃税案被罚8.8亿人民币尘埃落定。而中共整肃娱乐圈风波持续延烧 。近日，网路疯传中国知名导演冯小刚面临20亿元人民币罚款，中共官方将在近期公布处罚细节。但冯小刚均未回应此事。
 <br/>
 <br/>
 据陆媒19日报导，知名财经大V、微博认证北京银泰安投资顾问有限公司董事长的〝老c队长〞在微博爆料：冯裤子被罚20亿人民币，近期公布。可以转发500次。
 <br/>
 <br/>
 转发500次是什么概念？如果是谣言，转500次就构成诽谤罪。截图显示，目前的转发数已经超过500次。很多媒体也跟进报导。
 <br/>
 <br/>
 冯裤子是一些网友对冯小刚的称呼。不少网路大V转发了上述消息，并且向央视前主持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 求证真伪。但直到目前，崔永元与冯小刚均未回应此事。
 <br/>
 <br/>
 期间，还有不明身份的人士向爆料人〝老C队长〞发出语言威胁称：〝信不信打死你？我们出手很重的。限你11月19日8时之前删掉冯裤子那条新浪微博，否则后果自负！！！〞
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-19/p9133061a511475089.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-19/p9133061a511475089-ss.jpg"/>
   <br/>
   <font size="-1">
    认证为某投资公司董事长、知名财经博主的大V爆料称，冯小刚被罚20亿。（网页截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 老c队长在19日回应称：真有意思，那我重说一遍好了，我说的就是大导演冯小刚，20亿，你们满意了吧？别再用小号骚扰我了，我不会理你们的~另外我8点根本起不来。
 <br/>
 <br/>
 当天华夏日报记者向〝老C队长〞发布了一条信息求证。〝老C队长〞回复说，该消息来自国税总局。而冯小刚被罚款20亿元人民币，近期国税总局也将公布出来。
 <br/>
 <br/>
 不过，一位认证为某媒体总编辑的大V转发上述消息并辟谣：20亿数字绝对是假的！
 <br/>
 <br/>
 这位叫做凯雷的粉丝数50多万，认证信息是：香港大公文汇传媒集团北京新闻中心副总编辑；头条文章作者。简介是香港文汇媒体人。
 <br/>
 <br/>
 舆论认为，如果从粉丝数和大V权重来看，这位叫做凯雷的说法应该更有信服力。但是在官方没有公布之前，所有传闻都有可能是谣言，也有可能是遥遥领先的预言。

 央视前主持崔永元5月底爆料
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 利用〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞偷税，还说涉案明星并非是一个人，而是〝一个团伙〞。冯小刚因此陷入舆论旋涡。
 <br/>
 <br/>
 6月初，崔永元又爆料冯小刚在洛杉矶买下两套价值不菲的房子。两套房产价值分别为600余万美元和300余万美元。
 <br/>
 <br/>
 10月3日，中共国税总局通报说，范冰冰因偷逃税款，缴清8.89亿人民币的罚款可免予刑责。
 <br/>
 <br/>
 10月5日，中国多个网媒称，中共红二代蔡小心爆料称，官方调查组进驻北京电视艺术中心（北视）后，疑似冯小刚因受〝惊吓〞，连夜补缴百万人民币管理费。
 <br/>
 <br/>
 据陆媒报导，除范冰冰和冯小刚之外，范的好友、中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵薇.html" target="_blank">
  赵薇
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舒淇.html" target="_blank">
  舒淇
 </a>
 ，因参与综艺节目《中餐厅》薪酬超标，也被勒令退回4000万人民币。
 <br/>
 <br/>
 曾经在中国一线娱乐节目担任导演的姜先生对德媒说，中共政府之所以到娱乐圈〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_割韭菜.html" target="_blank">
  割韭菜
 </a>
 〞，是因国库缺钱，当局处于中美贸易战，地方财政状况堪忧，崔永元的爆料，无疑给当局增加财政收入提供了绝佳的机会。而今针对演员，主要是因演员们收入高民怨也高，确实也存在逃税漏税的问题。
 <br/>
 <br/>
