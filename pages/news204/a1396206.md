### 改革遇阻？习近平重磅会议训话〝秘书帮〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在〝全国党委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘书长会议.html" target="_blank">
  秘书长会议
 </a>
 〞召开之际，再次训话〝秘书帮〞要增强
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四个意识.html" target="_blank">
  四个意识
 </a>
 ，习的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大秘.html" target="_blank">
  大秘
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁薛祥.html" target="_blank">
  丁薛祥
 </a>
 则带头表态，坚决维护习核心。这被认为习改革遇阻之下，为政令落实及有效执行而对秘书长采取的集训。
 <br/>
 <br/>
 10月20日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大秘.html" target="_blank">
  大秘
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_办公厅主任.html" target="_blank">
  办公厅主任
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丁薛祥.html" target="_blank">
  丁薛祥
 </a>
 主持召开〝全国党委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘书长会议.html" target="_blank">
  秘书长会议
 </a>
 〞。会前，习近平要求，各党委办公厅（室）要〝进一步〞增强
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四个意识.html" target="_blank">
  四个意识
 </a>
 ，即政治意识、大局意识、核心意识、看齐意识，维护中央权威和集中统一领导，〝全力〞推动中央决策部署落地。
 <br/>
 <br/>
 该秘书长会议每两年举行一次，这是首次由丁薛祥主持召开，前两次均由时任中办主任、现任中共人大委员长栗战书主持。
 <br/>
 <br/>
 丁薛祥在会上带头表态，坚决维护习核心，维护中共权威和集中统一领导。
 <br/>
 <br/>
 多维分析认为，习近平在会议上要求〝推动决策部署落地〞，是在暗示执行力不够仍是中共党内面临的最大问题之一，中共官场充斥着〝太平官〞和〝无为官〞等。
 <br/>
 <br/>
 中共各级机构中的秘书们负责处理具体事务，沟通上下关系，承担着〝舒筋活血〞的角色。一旦这一群体出现〝问题〞，则造成〝政令不出中南海〞，相关政策难以顺利在地方施行。
 <br/>
 <br/>
 外界普遍认为，中南海自己也公开承认中央被架空。习近平再提〝政令不出中南海〞，并对秘书长进行集训，折射出中南海博弈的激烈。
 <br/>
 <br/>
 而〝政令不出中南海〞要追究到中共前党魁江泽民身上。江在中共16大后，为延续其对中共最高层的控制，极力将亲信塞进中共政治局常委，并规定9常委各管一摊。
 <br/>
 <br/>
 从中共16大到18大前夕，胡锦涛和温家宝处处受制于江派，在江的〝第二权力中央〞的压迫之下，人事、宣传、军队等三项大权一直受到江的分解，实权被架空，致使〝政令不出中南海〞。
 <br/>
 <br/>
 相比18大前的〝政令不出中南海〞，迹象表明，中共官场现在受到官僚系统自上而下执行力，以及官员不作为的困扰。 上述困境使得中共各方面的改革确实面临着执行困难的问题，这表现在几个主要的方面。
 <br/>
 <br/>
 一是，各方面的改革推进和执行程度不一、不平衡。一些改革执行了，另一些改革则没有；一些部门和地方执行了，另一些部门和地方则没有动静；二是，一些改革雷声大雨点小，做了一些，遇到困难就戛然而止，半途而废。更多的改革一直只停留在纸面上，没有人去碰，也没有人敢碰。
 <br/>
 <br/>
 而此次中共秘书长会议聚焦政策落实，以及近期新一轮维护中央权威和习核心地位，与习近平4年前对秘书长进行集训，痛斥〝政令不出中南海〞的背景有些类似。
 <br/>
 <br/>
 2014年10月10日至11日，〝全国党委秘书长会议〞召开，会前习近平作出批示称，〝确保中央政令畅通、决策落地生根〞。
 <br/>
 <br/>
 习近平对秘书长集训后，陆媒门户网站盘点半年内，在31个省份中，包括省委、省政府、市委、市政府在内，有28名秘书长和副秘书长落马，多数主要涉受贿。
 <br/>
 <br/>
 秘书也被叫作〝二号首长〞，处于行权者和受权者之间的关口，是接近〝权力核心〞的必经通道，拥有一种隐性权力。党媒发文称，〝秘书腐败〞已经成为滋生腐败大案、窝案的土壤。
 <br/>
 <br/>
 而中纪委调查高官经常也都是从秘书查起，知道官员最多秘密的现任、前任贴身秘书，纪检调查组通常到最后关头或有把握的时候才会拘捕调查。
 <br/>
 <br/>
 党媒也曾结合中共前政治局常委周永康的〝秘书帮〞，将〝秘书帮〞一词多次渲染批评。如今时隔4年习近平再次喊话〝秘书帮〞，这被认为是再度释放打虎先抓秘书帮的动作之一。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：祝馨睿）
