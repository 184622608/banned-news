### 分析：反制中共已成美朝野共识 北京豪赌美大选只是幻想
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】中共干预
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 ，显示出或在期待美国共和党选举落败，从而令中美贸易战降温。但美媒分析，反制中共已成美国朝野共识，不论中期选举和2020大选结果如何，美国对中共的立场都不会大变。
 <br/>
 <br/>
 近期以来，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）和白宫官员接连谴责中共试图干预11月的中期选举。而中共官媒在美国爱荷华州地方报纸购买版面攻击川普，给美方指控提供了明证。
 <br/>
 <br/>
 此外，北京也摆出不准备在中期选举前跟美国谈判的姿态。中共方面还有〝川普执政不过几年，我们能熬过〞的论调。这显示出中共在豪赌中期选举、甚至2020大选的意图。
 <br/>
 <br/>
 不过，自由亚洲电台10月3日评论文章指，期盼共和党落败能停止贸易战，这只是北京的幻想。
 <br/>
 <br/>
 文章说，中共在美国帮助下加入世贸组织，但近二十年来一直没有兑现开放市场、公平对待外资的承诺，美国企业和政界对此已经不满很久。美国企业对中共挤压外资敢怒不敢言，多年来都在游说华府代他们出头，跟北京交涉。
 <br/>
 <br/>
 2011年1月，前北京领导人胡锦涛访美时，时任美国总统就当面要求中方处理好美商在华受政府歧视和知识产权被盗窃的问题，这是美国总统首次公开向中方表达美企的不满。
 <br/>
 <br/>
 之后，美国政府加速推动跨太平洋伙伴全面进展协定（TPP），开始时明言不会让中共参与，以迫使中共改善其市场环境。但中共一直不为所动。川普现在启动中美关税战，是美国政府的新一轮尝试。
 <br/>
 <br/>
 而美国在南中国海与中共展开军事对抗，以及提出〝重返亚洲〞，也不是始于川普任期。因此，美国要制衡中共在经济和政治上践踏国际秩序的霸道行径，早已是美国朝野的共识。
 <br/>
 <br/>
 文章说，最近美国国会通过多个反制中共的法案，都得到两党广泛支持，获大比数通过。这在民主、共和两党越来越壁垒分明的今天，简直是异数。
 <br/>
 <br/>
 文章分析，不管中期选举或2020大选结果如何，美国惩罚中共犯规的政策，都不会大变。而且，如果
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 重夺国会多数，川普可能会放弃某些会遇到民主党强力抵制的议题，而将更多精力放在两党共识较多的议题，如反制中共霸凌。
 <br/>
 <br/>
 因此，文章认为，若共和党在中期选举失利，美中对抗甚至可能还会升级。
 <br/>
 <br/>
 在此之前，已有多位时评人士指出，北京希望中期选举为贸易战解套，算盘很难打响。
 <br/>
 <br/>
 不过，也有分析指，虽然反制中共已是美国社会共识，但是相对于共和党，民主党对中共来说更易于〝沟通〞。
 <br/>
 <br/>
 从现实看，川普上台后，美国经济一路攀升。而启动贸易战解决对外贸易不公，也令川普的国内支持率节节上升。北京对共和党选举落败的企盼正在逐渐落空。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
