### 川普再警告:中共干涉美政治超俄国 或加新一轮征税
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）10月14日在接受哥伦比亚电视台访谈时，再次警告考虑对中国商品征收
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新关税.html" target="_blank">
  新关税
 </a>
 。他还警告说，相对于俄罗斯2016年干预美国大选，中共干预美国政治是一个〝更大的问题〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在访谈中说，正在考虑对中共展开新一轮关税攻势，当前的关税组合中或许还会加入更多的内容。
 <br/>
 <br/>
 10月9日，川普在白宫对记者肯定地表示，将启动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新关税.html" target="_blank">
  新关税
 </a>
 。当记者问到是否准备好征收新关税以应对来自中共的报复时，川普说：〝当然，绝对的。〞他还说，目前美中的贸易不平衡意味着〝他们（中共）已经进行了报复〞。
 <br/>
 <br/>
 另外，在14日的采访中，当谈到俄罗斯干预美国2016年大选时，川普话锋转向中共：〝我认为中国（共）也干预了。坦白说，我认为中国（共）…问题更大。〞
 <br/>
 <br/>
 近日，川普和白宫官员也接连谴责中共干预11月的美国中期选举，侵蚀美国民主制度。
 <br/>
 <br/>
 访谈中，当被问到是否想要让中国经济陷入衰退时，川普做出了否定的回答。他说：〝我想要他们同我们谈判一个公平的协议。我想要他们像我们一样开放市场。〞
 <br/>
 <br/>
 川普也提到中国股市近4个月来的暴跌，就像1929年美国大萧条开始时的情形。
 <br/>
 <br/>
 川普日前多次对媒体表示，北京很想和华盛顿谈判，但他认为北京还没有〝做好达成协议的准备〞。该表态暗示，华盛顿并不急于和北京达成和解。
 <br/>
 <br/>
 对于中共党媒热炒的11月川习会，外界普遍认为取得突破的可能性不大。
 <br/>
 <br/>
 美国政府近期的系列举措，正将中美贸易战扩展到涵盖多个领域的全面对抗。《华尔街日报》日前报导，中美正逐渐迈向新冷战，而这正是川普政府的真实战略意图。
 <br/>
 <br/>
 10月4日，美国副总统彭斯发表重磅演讲，全方位谴责中共在各个领域对美国的侵蚀和伤害，并誓言在美中关系达到〝公平、对等和尊重我们主权〞之前，美方绝不会让步。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
