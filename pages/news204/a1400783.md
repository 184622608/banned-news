### 【禁闻】11月25日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 枪击案开庭 韩金凌遭刑讯逼供
 <br/>
 <br/>
 11月23号，上海〝4．13强拆枪击案〞被告人之一的韩金凌案开庭审理，韩金凌的律师为他做无罪辩护，韩金凌当庭揭露警察刑讯逼供、看守所人员对他暴力殴打，逼迫他认罪等。法庭没有当庭宣判。
 <br/>
 2017年4月13号，韩家遭暴力强拆，他们一家采用泼洒硫酸和点燃煤气罐等手段抵抗，被警方开枪打死一人，打伤两人，并将韩金凌一家三口逮捕控罪。
 <br/>
 <br/>
 2. 江苏政府为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逼迁.html" target="_blank">
  逼迁
 </a>
 雇黑社会殴打农民
 <br/>
 <br/>
 11月24号，江苏江阴市失地农民维权代表张国璋，在无锡遭拆迁部门雇佣的黑社会人员绑架殴打，过程中，家属多次报警，警察都置之不理，张国璋最后被打至脑震荡，送医治疗。
 <br/>
 <br/>
 3. 四大银行黑龙江
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下岗职工.html" target="_blank">
  下岗职工
 </a>
 连续维权
 <br/>
 <br/>
 中国大陆四大国有银行在黑龙江的下岗职工，11月12号起在省社保局维权，到目前已经持续两个多星期，但没有结果。 有人前往香港，在香港工行总部举牌抗议。
 <br/>
 <br/>
 4. 山西民师省府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 警称要扫黑除恶
 <br/>
 <br/>
 山西省各市、县被下岗的600多名民办、代课、幼儿园教师，11月19号集体到位于太原的省政府上访，遭警方用大巴车拉到城外30多公里处遣散，警方声称要以〝扫黑除恶〞的名义处罚他们。
</div>
