### 党媒不提90天缓刑 港媒：当局面临党内巨大压力
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】12月1日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 上，北京以多方让步换取了90天暂缓贸易战升级的谈判期，被指〝死刑〞改〝缓刑〞。但中共官方对90天期限只字不提。港媒分析，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 必须面对来自党内外的压力，因此需要将川习会有所美化。
 <br/>
 <br/>
 港媒12月4日一篇评论文章指，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 而言，政策失误可能会招致批评，他必须保护自己。
 <br/>
 <br/>
 文章引述香港中文大学中国政治学教授林和立在《华尔街日报》做出的分析称，习需要应付来自党内元老的压力，有必要将此事美化为应对川普（特朗普）取得的一次〝胜利〞。
 <br/>
 <br/>
 北京一名学者则认为，北京之所以对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 后的公众舆论非常敏感，是因为中方在过去几个月持续做出强硬表态，现在的难题是如何让国内民意接受妥协。
 <br/>
 <br/>
 另有专家指，这90天谈判期限，正好落在中国新年前后，也是中共两会前夕。如何保证不冲淡新年气氛，并在两会上拿出一个不难看的〝成绩单〞，令北京的研判压力大增。
 <br/>
 <br/>
 外界普遍认为，因为习近平大权独揽，美中贸易战的压力自然也将全部落在习近平一个人身上，因为其他官员都不敢、也不愿在这件关乎中共政权存亡的大事上承担责任。
 <br/>
 <br/>
 伴随着贸易战将中共政权推入危局，中共内部暗流涌动，生死搏斗也在酝酿当中。如果哪个官员被党内指责在贸易战中做出〝错误决定〞，就可能随时面临悲惨的结局。而这巨大的风险，目前只能由〝定于一尊〞的习近平承担。
 <br/>
 <br/>
 这一点，在刚刚过去的川习会上得到充分体现。
 <br/>
 <br/>
 12月3日，白宫贸易顾问纳瓦罗（Peter Navarro）对美国全国公共广播电台（NPR）透露，川习会晚宴的前30分钟，中方与会官员中，只有习近平一个人开口发言。
 <br/>
 <br/>
 纳瓦罗说，习近平就加大购买美国商品、保护知识产权、强制技术转让等许多问题，向美方做出了承诺，还和美方讨论了汽车关税问题。
 <br/>
 <br/>
 白宫经济顾问库德洛（Larry Kudlow）则在记者会上透露，在川习会晚宴前，美方官员与中共国务院副总理刘鹤私下会面两次。在12月1日第二次会面时，刘鹤直接表示：〝我什么也不说了。习主席会出面说。〞
 <br/>
 <br/>
 库德洛说，在晚宴上，习近平亲自上阵，一开始就向美方〝推销〞达成协议的〝卖点〞，其中包括诸多细节。而这些细节，原本应该是由贸易官员掌握的内容。
 <br/>
 <br/>
 另外，在川习会后，白宫声明和川普推特上，先后透露了许多中共官方没有提及的北京让步内容，包括管控芬太尼、购买美国农产品和能源产品等商品、从新考虑高通收购案和取消美国汽车进口关税等。但中共官方尚未对此做出回应。
 <br/>
 <br/>
 彭博社12月4日引述三名中共官员透露，因为习近平还在访问葡萄牙，北京各部委的官员正在等待习回国，然后他们才能发表评论或采取行动。
 <br/>
 <br/>
 对于即将展开的中美90天谈判，多有分析指，北京很难就关键问题让步，因为这涉及到中共体制的安危。因此，90天后关税战继续升级的可能性很大。
 <br/>
 <br/>
 而贸易战的长期化，意味着中共政权可能会越来越不稳，而习近平个人面临的危机也将不断加重。
 <br/>
 <br/>
 日前，有被指编辑部设在北京的中文媒体刊文，公开要求习近平做出检讨，有海外学者对此形容为〝吹响了中共党内倒习的号角〞。但文章随后迅速遭修改，调子由批习改成了捧习。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
