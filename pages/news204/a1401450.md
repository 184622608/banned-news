### 大陆掀〝大裁员〞浪潮 大学生就业难
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】中国各地掀起企业〝大裁员〞浪潮，大量的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业.html" target="_blank">
  失业
 </a>
 人潮返回老家。就业市场也进入寒冬，很多企业停招或缩减招募应届毕业生。有大陆民众发文称：2018，没什么比活下去更重要，饭碗才是硬道理，对个人如此，对国家来说也是这样。
 <br/>
 <br/>
 11月28号，大陆微信公众号〝智谷趋势〞发表一篇题为〝2018，没什么比活下去更重要〞的长文。文章开门见山说：突然之间，〝大裁员〞的新闻满天飞。BAT、华为、京东等
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_互联网巨头.html" target="_blank">
  互联网巨头
 </a>
 相继卷入这场裁员漩涡中。沿海省份的出口代工厂纷纷关门放假，富士康被传裁员34万，金融、地产都在裁员。天风证券爬取前程无忧4~9月份的招聘广告条数后发现了，招聘广告从285万骤减至83万。
 <br/>
 <br/>
 深圳印刷业者时先生：〝今年下半年，工厂裁员这个风浪就特别的明确，很多很多的公司都裁员，工厂都缩减规模，大量的人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业.html" target="_blank">
  失业
 </a>
 ，他们都回老家。明显的现象是各个电子城比以前冷清很多，现在基本上是门可罗雀，那些餐厅啊、娱乐场所，没有这些人来这边来了，自然就凋落了。〞
 <br/>
 <br/>
 据大陆媒体报导，华为、阿里巴巴、百度、腾讯和京东等大型网络公司传出全面停招或缩减招募应届毕业生。虽然华为等多家公司否认，但已引起不安，尤其是今年中国有超过820万名大专大学毕业生，是史上新高，就业形势将更加严峻。
 <br/>
 <br/>
 大陆大学应届毕业生刘先生：〝我是学金融的，然后我知道金融这个行业，银行、券商啥的，今年确实是有减薪的，有缩的。我在准备研究生的考试，本科阶段这几年学的东西太过于浅，就不想那么早的工作，想有一个更好的平台，所以我就选择读研。〞
 <br/>
 <br/>
 〝智谷趋势〞的文章最后说，就业是最大的民生。放在经济周期里观察，一旦下岗、降薪增多，消费者的信心指数就会下滑，会越来越不敢花钱，这对于当下中国从出口驱动、投资驱动转向消费型经济搆成了挑战。只有加速转型，扩大改革，才能活在未来。
 <br/>
 <br/>
 新唐人记者熊斌、元真采访报导
