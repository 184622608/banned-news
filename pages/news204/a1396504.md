### 习近平南巡再提自力更生 党媒报导打破惯例
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】习近平22日下午考察广东珠海一家产业园，当晚入住珠海。而党媒当日傍晚就报导了相关消息，被指打破高层〝离开后才能报导〞的惯例。次日党媒再报习近平视察格力电器，号召〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自力更生.html" target="_blank">
  自力更生
 </a>
 〞。外界指，习南巡意在提振经济，应对危机。
 <br/>
 <br/>
 北京时间10月22日傍晚6时许，《人民日报》推出首篇有关习近平此次〝南巡〞的报导，发布了习当天下午考察珠海市粤澳合作中医药科技产业园的消息。随后港媒披露，习近平当晚7时左右入住珠海度假村酒店。
 <br/>
 <br/>
 香港《苹果日报》报导称，官媒即时报导习近平行踪，再次打破中共政治局常委外地考察要〝离开后才能报导〞的惯例。
 <br/>
 <br/>
 23日，新华社再次披露习近平行程，称他22日下午还考察了珠海的格力电器公司。习近平在考察时表示，发展实体经济〝至关重要〞，并指制造业的核心就是〝创新〞和〝掌握关键核心技术〞，必须靠〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自力更生.html" target="_blank">
  自力更生
 </a>
 奋斗〞和〝自主创新争取〞。
 <br/>
 <br/>
 这是贸易战以来习近平第二次提及〝自力更生〞。第一次是9月26日在黑龙江考察时提到，国际上〝先进技术、关键技术愈来愈难以获得〞，〝逼着我们走自力更生的道路〞。
 <br/>
 <br/>
 外界普遍分析，北京的所谓〝改革〞已入僵局，再加上贸易战高压，令中国经济陷入困境，习近平再次南下，意在释放〝继续改革开放〞的信息，提振市场信心。
 <br/>
 <br/>
 随着习近平〝南巡〞，中共央行宣布再次放水1,500亿人民币资金以〝支持民企〞，北京经济高官集体喊话救市，中国股市应声戏剧性高涨，沪指收复2,600点大关。
 <br/>
 <br/>
 但有专家指，北京政策短期刺激无法持久，难以挽救系统性危机。网友悲叹，股市现在上涨是〝为了更快地下跌〞。
 <br/>
 <br/>
 《苹果日报》分析，习近平南下首日就考察两个民企，明显是有意否认在推行的〝国进民退〞，给私营企业打气。
 <br/>
 <br/>
 该报评论文章指，中国经济现状让人质疑，当局可能会回归计划经济，剥夺民企财富以充实党库，股市的暴跌也让人民财富流失，失去〝共克时艰〞的意愿和能力。
 <br/>
 <br/>
 近几个月来，国有资金接连收购或入股民企，股市暴跌令许多上市公司股价逼近平仓线，面临被国有银行接管的危机。官方多次解释称，这是国企〝帮助私企度过危机〞，但网民却质疑官方藉机〝收割韭菜〞。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
