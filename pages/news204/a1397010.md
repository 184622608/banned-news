### 〝川习会〞不谈贸易？ 川普正准备全力出击
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】将于11月底举行的G20〝川习会〞能否为中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 减压，倍受外界关注。但据知情人士透露，由于北京至今未表现出认真对待合谈，白宫考虑将贸易话题从两国元首会唔中移除。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）总统认为，贸易战现在只不过是〝开始的开始〞，他正准备
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全力出击.html" target="_blank">
  全力出击
 </a>
 。
 <br/>
 <br/>
 彭博社10月27日报导，两名知情人士表示，白宫可能考虑将贸易话题，从美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和中国国家主席习近平在G20的会唔中移除。
 <br/>
 <br/>
 一名不愿透露姓名的消息人士表示，川普的很多顾问都不愿和北京谈贸易，因为北京现在还没表现出要认真对待美国协商要求的样子。
 <br/>
 <br/>
 对于如何能够停止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，白宫首席经济顾问库德洛（Larry Kudlow）近日多次重申，美国所要求的就是〝零关税、零非关税壁垒、零补贴、停止知识产权盗窃、停止技术转让，让美国人拥有自己的公司。〞川普政府只不过是要求北京遵循公平贸易原则。
 <br/>
 <br/>
 〝这些问题已经被我们提出好几个月了，到目前为止，这些问题还没有得到满足。〞库德洛指出，北京至今未能认真对待美国的要求。
 <br/>
 <br/>
 中美贸易战已经持续了4个多月，双方此前已展开多轮谈判，但因北京不愿在美国关注的关键问题上做出实质性改变，导致谈判无果。
 <br/>
 <br/>
 随着贸易战升级，中共经济遭受重创，股市、汇率双双大跌，经济放缓下行至过去10年来最低，外企纷纷撤离、制造业低迷。据大陆经济学者此前推断，中共经济如今疑似已进入〝崩盘模式〞，中共政权也处于风雨飘摇中。
 <br/>
 <br/>
 在此背景下，北京迫切希望与美国谈判，解决贸易争端。川普此前也曾公开表示，北京非常想谈，但他认为现在为时过早，因为北京还没有准备好。
 <br/>
 <br/>
 新闻网站Axios21日报导，3名消息人士称，川普总统无意放宽对中国商品的关税，贸易战现在只不过是〝开始的开始〞。川普团队对即将举行的G20〝川习会〞，不抱有太大的期待。
 <br/>
 <br/>
 〝他（总统）希望他们（中共）从2000亿美元的中国商品关税中受到更大的压力〞。一名知情人士表示，川普认为需要更多的时间，关税带来的痛苦才能逐渐显露出来，这样他就会拥有更多的（谈判）筹码。
 <br/>
 <br/>
 〝这是国家元首会议，而不是贸易会议。〞另一位熟悉川普策略的人士表示，〝川普考虑将此次会议作为与习主席的私人关系重温，而非详细地讨论（贸易）〞。
 <br/>
 <br/>
 该知情人士称，〝双方相距甚远。现在没有前进的共同基础。〞
 <br/>
 <br/>
 《华尔街日报》10月25日报导称，一名白宫高官说，如果中共想要G20期间的会晤是一个有意义的会晤，需要给出明确信息，否则很难看到这次会晤的成果。
 <br/>
 <br/>
 美财长努钦（Steven Mnuchin）团队表示，北京谈论购买更多的美国产品没有意义，因为关键的优先议题必须解决，例如偷窃知识产权和市场准入权等结构性的问题。
 <br/>
 <br/>
 另有多名接近川普的消息人士称，川普并不为中共的报复措施感到担心，他正准备
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全力出击.html" target="_blank">
  全力出击
 </a>
 。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
