### 习川晚宴结束 习近平首提与川普友谊
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】美中首脑在阿根廷的G20峰会期间举行会晤，聚焦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 问题。会晤开始时，习近平首提他与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）的个人友谊。他说此次会晤体现了他和川普的个人友谊。川普也表示，他同习近平有着一段〝不可思议的友谊〞。
 <br/>
 <br/>
 当地时间12月1日晚，习近平与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在阿根廷首都布宜诺斯艾利斯共进工作晚宴，聚焦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 等话题，会晤历时约2个半小时。
 <br/>
 <br/>
 在晚宴开始的时候，习近平首次提及他与川普的个人友谊。
 <br/>
 <br/>
 习近平说，这次会晤体现了他和川普的个人友谊，〝很高兴同总统先生会晤。自上次见面以来，世界形势又出现了不少新变化。美中作为两个大国，都有重要影响力，在促进世界和平和繁荣方面共同肩负着重要责任，合作是双方最好的选择。〞
 <br/>
 <br/>
 习近平表示，〝我愿利用今晚的机会，同总统先生就共同关心的问题交换看法，并规划好下一阶段美中关系。〞
 <br/>
 <br/>
 川普也说，他同习近平有着一段〝不可思议的友谊〞，在提及贸易问题时，川普说：〝这段关系非常特殊。我认为，我同习主席的关系，是我们最终可能会做一些对中国和美国都有利的事情的首要原因。〞
 <br/>
 <br/>
 晚宴结束后，双方没有就〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞发表声明。白宫国家经济委员会主席库德罗（Larry Kudlow）告诉媒体，川习会谈得〝很好〞。白宫发言人桑德斯（Sarah Sanders）也表示，美方将会发表声明。
 <br/>
 <br/>
 香港《南华早报》称，这次晚宴比原计划延长了约一个小时，持续2个多小时。晚宴结束后，宴会厅内传出一阵掌声。这似乎显示双方对会谈结果还算满意。
 <br/>
 <br/>
 〝川习会〞前夕，海外媒体就已经不断释出中方将加强保护知识产权、扩大开放市场等让步讯息。川普11月29日在搭机飞往阿根廷前也透露，美中双方〝很接近〞达成协议，但他不确定是否能这么做。
 <br/>
 <br/>
 美国贸易代表莱特希泽表示，他预计川普和习近平将举行的晚宴会谈会是成功的。他确信，这次〝川习会〞后，美中官员都会有〝积极的感觉〞，〝如果川习会不成功，我会感到惊讶。〞
 <br/>
 <br/>
 G20峰会期间，川普告诉媒体记者说：〝我们正在加紧努力工作，如果我们能达成协议，那会是很好的结果。〞〝我想他们（中方）想要（达成协议）。我认为我们也愿意。等着瞧吧。〞
 <br/>
 <br/>
 事实上，11月上旬，川普就曾表示，美国在往上走，中国经济在下滑。他不希望中共的经济下滑。他和习近平将在G20峰会期间举行一次很好的会晤，双方将会尝试达成协议，因为他希望与习近平及中国都有良好的关系。
 <br/>
 <br/>
 《新唐人》此前已分析报导，川习会的结果对北京来说，几乎是生死攸关的。美中贸易战持续近一年来，中国国内出现严重分裂信号，不仅北京政坛有人支持川普。甚至中国民企也纷纷倒戈，支持川普对中共进行更严厉的贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 。
 <br/>
 <br/>
 贸易战火持续升高，中共存在已久的经济危机几近全面爆发，加上美国联手西方发达国家对中共在军事、政治、意识形态等各个层面上的围堵，令中共政权面临严重危机。
 <br/>
 <br/>
 在此背景之下，川习会之前，多家美媒报导说，中共急切希望〝川习会〞达成〝停火〞协议，不再升级贸易冲突。为此北京很可能做出更多让步，包括购买更多的美国商品，改善知识产权法等，来换取美国暂缓进一步加征关税。
 <br/>
 <br/>
 不过，出于面子和维持政权稳定的考虑，北京越是做出让步，越不会对外发布〝川习会〞的结果。故此〝川习会〞至今未见中方发表任何公开消息。
 <br/>
 <br/>
 按照中共官媒公布的行程，习近平下一步将对巴拿马及葡萄牙展开国事访问。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
