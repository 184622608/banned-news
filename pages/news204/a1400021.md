### 习近平防弹专车 改装细节曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 出席亚太经合（APEC）峰会时，乘坐自带的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_防弹专车.html" target="_blank">
  防弹专车
 </a>
 ，成为舆论热点。这是习第二次在外访中乘坐自己的专车。日前，该车的一些改装细节被曝光。
 <br/>
 <br/>
 据《路透社》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 11月15日晚抵达巴布亚新几内亚。当天，他的两辆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_防弹专车.html" target="_blank">
  防弹专车
 </a>
 也由中方空运至当地。专车均由陆产红旗L5加入防弹结构改装而成，配备4公升引擎，外貌类似劳斯莱斯。
 <br/>
 <br/>
 港媒《苹果日报》报导称，由于巴布亚新几内亚治安不是很好，所以中方特意将防弹专车空运到当地，供习近平使用。2014年，习出访新西兰时，也曾使用自己的防弹车。
 <br/>
 <br/>
 按照国际惯例，一个国家首脑到访，都是由对方国家负责，包括安保。2015年9月22日，习近平访问美国时乘坐的就是美方安排的黑色凯迪拉克轿车。
 <br/>
 <br/>
 中共外交部前礼宾司代司长鲁培新此前曾表示，如果到访国家条件许可，中方会要求提供防弹专车，但如果对方无法提供，中方会使用自己的专车。
 <br/>
 <br/>
 值得一提的是，习近平此次出访巴布亚新几内亚，中共外交部仅提前三天（12日）对外公布其行程。以往习近平出访时，外交部至少在一星期之前就会公布行程。相信此次的安排与安保考虑有关。
 <br/>
 <br/>
 据悉，本次的APEC会议，美国、澳洲、新西兰三国共派出4,000名特勤人员，帮助当地提供海陆空全方位保护。但由于当地酒店房间不足，大部分与会者都在港口的三艘邮轮上住宿。
 <br/>
 <br/>
 有美媒报导说，北京对APEC峰会期间的安保极度重视，习近平慎之又慎。在出行前采纳了中南海核心智囊的建议，原本要一起随访的第一夫人取消行程。
 <br/>
 <br/>
 报导还披露，本次APEC峰会，中南海特别大幅增补了负责习近平贴身安全的保镖数量，所有保镖均由习近平前任〝大内总管〞栗战书亲自把关。
 <br/>
 <br/>
 近年来，习近平出访时的安保倍受关注。2017年6月，习近平到香港视察3天，港府动用了近10,000警力，比前任中国国家主席胡锦涛访港时安保级别严密不少，当时只动用警力3,000人。
 <br/>
 <br/>
 据港媒报导，2017年6月29日习近平抵港后，乘坐由香港警方提供的宝马七系防弹车。胡锦涛2012年访港时，乘坐的也是该款防弹车。
 <br/>
 <br/>
 据悉，宝马七系防弹车车身配备防弹装甲，分布在车门、车顶、车尾等处，能承受炸弹爆炸、自动步枪近距离扫射等。车内空调系统还有特殊的安全配备，当车辆遭受毒气攻击时，会自动关闭车窗、锁车门，同时关闭外部空气通道。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：范铭）
