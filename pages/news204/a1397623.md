### 美媒：习〝南巡〞最后一刻突改行程 以防不测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 日前罕见的低调南下广东考察三天，给外界留下多个疑点。11月1日，有消息称，习近平曾在最后一刻匆匆调整行程，取消了三大活动，在安保方面以防不测。
 <br/>
 <br/>
 11月1日，有美媒爆料称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 日前南下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 视察的路线，是事先经中南海核心团队设计，但在最后一刻，习亲自调整了〝路线图〞，突然取消了三大重要活动，包括访澳门、与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳.html" target="_blank">
  深圳
 </a>
 市民共同登临莲花山和夜游广州珠江。
 <br/>
 <br/>
 至于背后的原因，报导称，主要是防止在安保方面出现不测。因粤港澳三地，地形复杂，又联通大海，维安压力大，对习出行安全构成严重威胁。由于考虑到海上维安带来的巨大难度，和地处港澳，境内外黑帮容易联手又防不胜防，23日港珠澳大桥开通仪式，也移到了珠海边防大楼室内举行。
 <br/>
 <br/>
 10月22日至24日，习近平南下视察期间，主持了港珠澳大桥的揭幕礼。不过，习此次南下及主持大桥揭幕礼的安排都是极为低调。
 <br/>
 <br/>
 比如，官方对习近平的行程一直保持神秘，直至10月22日下午习抵达珠海，才有人透露其行踪。而在10月23日大桥揭幕仪式举行数分钟前，官方仍拒绝公布习是否会出席，更罕见的是，习一行迟到半小时才现身。
 <br/>
 <br/>
 而大桥揭幕仪式上，习上台发言只说一句，〝我宣布，港珠澳大穚正式通车〞，便走下台。揭幕仪式后，习匆匆离开现场。10月24日，习到深圳龙华区龙悦居参观，当地政府从外地调派数万警力戒备，防止有人闹事，整个屋苑等同戒严。
 <br/>
 <br/>
 去年七一，习近平参加香港主权回归20周年庆祝时。港府也曾动用了近1万名警力，飞虎队、机动部队、反恐特勤队等全部上阵，对习访港之行至少执行4层安保保护，会展一带〝水马围城〞，甚至会展对出海面也会封锁。
 <br/>
 <br/>
 为单一领导人到访，安排这种安保阵仗，这在香港历史上绝无仅有。
 <br/>
 <br/>
 深圳临近香港，香港是江派重要窝点，也是江派频频发动针对胡锦涛、习近平的搅局行动的基地。
 <br/>
 <br/>
 江泽民集团二号人物曾庆红，自2003年出任中共中央港澳工作协调小组首任组长，一直在香港培植亲信势力，操控香港黑白两道；江派前常委、江泽民心腹张德江，十九大前兼任港澳工作协调小组组长，长期掌控香港和广东一带。
 <br/>
 <br/>
 港媒称，习近平前5年反腐触动一大批既得利益集团，政治风险极大，传闻有〝大老虎〞急欲反扑。习近平到访粤港澳三地，势必要加强安保措施防止江系
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暗杀.html" target="_blank">
  暗杀
 </a>
 。
 <br/>
 <br/>
 据称，习近平接班底定后，至少遭遇超过10次的暗杀险情。事后查明全是内部人雇凶作案。而对习的人身威胁，更多被指来自内部。其中最典型的是前政法委书记周永康谋杀习近平事件。
 <br/>
 <br/>
 据报，在2012年8月的北戴河会议前后，周永康至少两次试图暗杀习近平。一次是在北戴河会议室放置定时炸弹；另外一次是趁习在301医院做体检时打毒针。由于当时形势危急，习近平一度移居西山军事指挥中心，以防不测。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
