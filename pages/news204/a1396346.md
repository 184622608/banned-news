### 习近平低调下榻珠海度假酒店 吉普开路交通管制两分钟
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】北京最高领导人习近平已开始其〝南巡〞行程。官媒22日透露习考察广州珠海的消息。港媒披露，习近平当晚下榻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海度假酒店.html" target="_blank">
  珠海度假酒店
 </a>
 ，习车队抵达时只
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交通管制.html" target="_blank">
  交通管制
 </a>
 两分钟，非常低调。
 <br/>
 <br/>
 北京时间10月22日傍晚，《人民日报》发布了习近平考察珠海市粤澳合作中医药科技产业园的消息。
 <br/>
 <br/>
 香港《苹果日报》22日报导，习近平结束在珠海的行程后，于晚上7点左右下榻珠海度假村酒店。习住在酒店南侧的别墅区，外面便是著名的情人路，隔海可以遥望港珠澳大桥和人工岛。
 <br/>
 <br/>
 报导指，该酒店区域附近自下午3点左右便开始加强警戒，附近的加油站停止营运，门口有公安车戒备。附近民众一早已经知道习近平抵达的消息，但如常在情人路上散步。
 <br/>
 <br/>
 下午6点半左右，警戒开始加强，酒店周边200米禁止停留，便衣出动检查附近的厕所和垃圾桶。7点左右，习近平车队到达，两辆黑色吉普车开道，紧跟数辆4人车和随行大巴，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交通管制.html" target="_blank">
  交通管制
 </a>
 仅两分钟左右，相当低调。
 <br/>
 <br/>
 报导说，酒店内部似乎已经清空，酒店职员却以〝没有空房〞为由拒绝记者的电话订房。
 <br/>
 <br/>
 之前，新华社22日下午发布消息说，习近平当日上午还在北京主持中国工会第十七次全国代表大会开幕式。
 <br/>
 <br/>
 但有传言说，习近平当天早上已到广东。
 <br/>
 <br/>
 而香港电台22日引述当地市民表示，自己在珠海格力电器集团工作的朋友称，习近平20日视察了集团的厂房，视察期间整个厂房的窗户都要关闭。
 <br/>
 <br/>
 报导引述当地计程车司机说，自19日起，珠海已加强安保措施，大批警察和便衣人员在主要干道巡逻把守，胜过以往领导人来访视察的规模。
 <br/>
 <br/>
 早前有多家境外媒体报导，习近平可能于20日到广东中山大学视察，并在23日担任港珠澳大桥通车仪式的主礼嘉宾。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
</div>
