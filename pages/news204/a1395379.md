### 大豆危机？中共取消发布10月农产品供需报告
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】中共对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 加征关税，令大豆进口锐减。外界分析，年底前进入巴西大豆供应淡季，中国将陷入大豆需求危机。而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共农业部.html" target="_blank">
  中共农业部
 </a>
 日前取消了10月的农产品供需预测报告，从侧面印证了这一判断。
 <br/>
 <br/>
 路透社10月11日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共农业部.html" target="_blank">
  中共农业部
 </a>
 取消了10月份的《中国农产品供需形势分析报告》。该报告本该11日早晨公布，但一名农业部官员称，因为〝技术问题〞，报告取消发布。
 <br/>
 <br/>
 报导指，该报告提供主要农作物产量、进口量和消费量的月度预测，其中主要包括大豆、玉米、棉花、食用植物油和食糖等。
 <br/>
 <br/>
 为了打击川普（特朗普）〝票仓〞，中共今年7月6日开始对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 加征25%的关税，而美国仅次于巴西，是中国进口大豆的第二大供应国。
 <br/>
 <br/>
 外界普遍分析，中共很难找到替代大豆进口源，特别是进入11月后，因为南北半球季节更替的原因，巴西大豆供应进入淡季，而美国大豆无法进入中国，将造成中国市场大豆紧缺。多家媒体分析，中共11月将被迫再次进口美国大豆。
 <br/>
 <br/>
 因此，如果此时北京发布供需报告，其中大豆等农作物的供需数据可能出现严重不平衡，打击市场信心。
 <br/>
 <br/>
 不过，目前并没有见到中共大举恢复进口美国大豆的报导。这似乎再次显示了北京对抗美国的〝决心〞。
 <br/>
 <br/>
 为了平衡大豆供应不足，中共将目标瞄准了全国4亿头猪。官方日前公布，将降低猪饲料中蛋白最低含量的标准。大豆是猪饲料中蛋白的主要来源之一。此前党媒曾透露，通过降低动物饲料中的蛋白含量，可以每年减少1,000万吨大豆的需求。
 <br/>
 <br/>
 此外，党媒还引述专家说法称，饲料中减少的蛋白含量，可以用工业蛋白来弥补，并称这只会影响猪肉质量，不会危害人类健康。
 <br/>
 <br/>
 不过海外专家指，所谓工业蛋白，就是从煤炭和石油等原料中提取加工的化学成分，只会提高猪饲料中氮元素的含量，并无法提供营养，而且危及人的健康。
 <br/>
 <br/>
 网友更将工业蛋白和此前大陆奶粉中的三聚氰胺相比，担心中国民众将来被迫食用有毒的猪肉。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
