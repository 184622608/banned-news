### 非洲猪瘟蔓延中国9省市 辽宁为重灾区
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】大陆的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情持续扩大到9个省市，其中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁.html" target="_blank">
  辽宁
 </a>
 疫情失控，已爆发10起疫情，沦为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重灾区.html" target="_blank">
  重灾区
 </a>
 ，大陆民众认为，官方封锁消息是让疫情蔓延的主要原因。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁.html" target="_blank">
  辽宁
 </a>
 省沈阳市在今年8月3号出现中国首例
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情，疫情失控持续蔓延到营口、鞍山、大连、锦州、盘锦、铁岭等地。其中营口、鞍山、盘锦为二次爆发。
 <br/>
 <br/>
 辽宁维权人士霍先生：〝辽宁已经多次发生疫情，传播的还是比较快，国内报导的还是很少，官方管控疫情讯息，这个时间很长了，领导重视不够，就是宣传的力度或是政府的管控还是欠缺。农村大部分都养猪，环境脏乱差，特别严重的，猪圈、厨房都是挨着的。〞
 <br/>
 <br/>
 天津市于10月11号首度爆发疫情，蓟州区一家猪场隔天被确诊疫情，扑杀所有生猪。据9月11号农业农村部的通知要求，与发生非洲猪瘟疫情省相邻的省份暂停生猪跨省调运，并暂时关闭省内所有生猪交易市场。这意味着京津冀地区活猪调运全面暂停。
 <br/>
 <br/>
 同时中共封锁消息，之前有民众在微信群提醒少吃猪肉就被带走。
 <br/>
 <br/>
 天津市民毋女士：〝发现猪瘟了，我们也不说，我们也不向外传，一般人谁敢说啊，说了就是制造谣言，要收拾你、制裁你，就是明知也不敢说。我们听到消息只能自己保护自己。猪肉我们不买， 我们不吃，只能这样克制自己。〞
 <br/>
 <br/>
 辽宁沦为非洲猪瘟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重灾区.html" target="_blank">
  重灾区
 </a>
 ，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪价.html" target="_blank">
  猪价
 </a>
 暴跌，最近生猪出栏均价已跌至10元/公斤，养猪户叫苦连天，有养猪户干脆不卖了。
 <br/>
 <br/>
 辽宁盘锦养猪户詹女士：〝但是很多也在养，疫情过后，猪价一定会暴涨，注重消毒都没关系，一天一消毒。我们这里先不卖，看你要什么价嘛，便宜的话我们不卖。〞
 <br/>
 <br/>
 非洲猪瘟疫情在辽宁省沈阳爆发后，接着由北向南扩散，蔓延至辽宁、黑龙江、河南、江苏、安徽、浙江、内蒙古和吉林8个省份和天津1个直辖市。
 <br/>
 <br/>
 新唐人记者熊斌、柏妮采访报导
</div>
