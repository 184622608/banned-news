### 河南大妈组团拦婚车：千元起步，不给不让走（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】01，02，……92，93……上百名大妈每个人手上都写着一个编号，聚集在一小区的大门口，这是要干什么呢？原来大妈们是在组团拦婚车。
 <br/>
 <br/>
 11月26日，大陆网路上热传一段视频，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_河南洛阳.html" target="_blank">
  河南洛阳
 </a>
 一小区内，有上百名大妈聚在小区门口拦婚车要钱，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千元起步.html" target="_blank">
  千元起步
 </a>
 ，不给钱就不让走。
 <br/>
 <br/>
 视频显示，这些大妈一开口就喊价1000元人民币，新人家属不愿意给，说最多给600。一位貌似负责的大妈不依不饶，说要再添200元。
 <br/>
 <br/>
 经过一番讨价还价，这些大妈最终要到了800元，还拿走了不少喜糖和喜烟。
 <br/>
 <br/>
 据知情人士透露，这些拦婚车的大妈们有专人组织，她们会通过门卫打听谁家结婚，然而组织起来，在结婚当日拦车要钱，一般千元起步。〝遇到好日子的时候，一天有5家结婚的，有时候最多一天拦到5家。〞
 <br/>
 <br/>
 要到钱后，这些大妈会按照事前每个人手上写的编号分钱。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 视频中还显示，附近的环卫工人也参与其中，一名女环卫工人抱怨说：〝我在这辛辛苦苦收拾东西，没有给我东西。〞
 <br/>
 <br/>
 该视频引发大陆网友围观，有网友说，像这样拦婚车要钱的，难道不违法吗？还有网友说，这不就是拦路抢劫吗？
 <br/>
 <br/>
 另有网友说，在他们当地确实有拦婚车的风俗，一般都是拿些喜糖、烟，而且办喜事的人家也会提前准备的，拦婚车的都是村里人，像这样这么多人要钱的很少见。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：文慧）
</div>
