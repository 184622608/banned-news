### 李源潮已平安着陆？十一前后接连露面引猜测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】日前，中共前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家副主席.html" target="_blank">
  国家副主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李源潮.html" target="_blank">
  李源潮
 </a>
 在中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十一招待会.html" target="_blank">
  十一招待会
 </a>
 上露面后，4日他又与妻子在江苏省盐城市现身。李源潮曾是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 唯一的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政副手.html" target="_blank">
  行政副手
 </a>
 ，自从今年两会〝裸退〞后，已很长时间没有露面。外界一度传出他被调查，但近日接连露面引起关注。有一种分析认为，这可能意味着他已经〝平安着陆〞。
 <br/>
 <br/>
 十一长假期间，中国作家高伐林在推特中披露，前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家副主席.html" target="_blank">
  国家副主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李源潮.html" target="_blank">
  李源潮
 </a>
 于9月4日在盐城市现身，到其50年前工作过的地方参观，同行的还有他的妻子高建进。
 <br/>
 <br/>
 9月30日晚，中共国务院在北京大会堂举行十一〝招待会〞，李源潮和前中央军委副主席范长龙同时现身，而且并肩而坐。两人都曾被视为〝问题人物〞，一度传出受到调查的消息。
 <br/>
 <br/>
 去年11月20日，刚满67岁的李源潮在中共十九大出局，不但没能连任政治局委员，还中央委员也被剔除。十九大之后，李源潮几乎〝被雪藏〞，一直到今年3月中共两会，李源潮一次公开露面的机会都没有。
 <br/>
 <br/>
 李源潮两会上〝裸退〞，既不是全国政协委员，也不是人大代表。两会后，李源潮也极少出席任何公开活动，直到今次才公开露面。
 <br/>
 <br/>
 分析认为，曾经作为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 唯一的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政副手.html" target="_blank">
  行政副手
 </a>
 的李源潮，此次5天内两露面，或证明其已经〝软着陆〞。
 <br/>
 <br/>
 李源潮背景跨团派、江派和太子党等多个派系。中共十九大前，就曾多次传出他因涉前中办主任令计划案而出事消息。
 <br/>
 <br/>
 2016年7月，甚至有传言说，李源潮被立案调查，限制自由，其秘书司、机和警卫已被更换，李妻高建进也被要求配合调查等。
 <br/>
 <br/>
 2017年10月30日，有港媒披露，李源潮十八大后任国家副主席其实是虚职无权，之后习中央反贪腐、打老虎，李的老婆、孩子都涉及某些大案，秘书也被带走，于是一次次传出李源潮会被拿下的消息。
 <br/>
 <br/>
 报导说，5年来李源潮鲜有公开活动，大家不知道他一天到晚都在干什么。不过，早年间北京高干子弟圈中人都知道，但凡他在工作上出了严重问题，要〝检讨错误〞时，往往会泪流满面，哽咽不已。
 <br/>
 <br/>
 大家都觉得他胆小，很怕〝过不了关〞，其提前退出中共政治权力中心也在预料之中。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿）
</div>
