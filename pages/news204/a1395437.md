### 大陆强拆再惹民愤 业主车撞拆迁队1死9伤（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】在中国大陆，因政府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 引发的血案屡屡见诸报端。日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江苏扬州市.html" target="_blank">
  江苏扬州市
 </a>
 一名遭强拆的商铺业主情绪失控，愤怒地驾车撞向拆迁人员，造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_1死9伤.html" target="_blank">
  1死9伤
 </a>
 。
 <br/>
 <br/>
 据陆媒报导，10月15日上午7时许，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江苏扬州市.html" target="_blank">
  江苏扬州市
 </a>
 杭集镇中心广场附近，一间旅游用品厂被政府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 ，现场玻璃散落一地，商铺内的东西被翻腾的一片狼藉。
 <br/>
 <br/>
 据网传视频显示，商铺业主韦某与拆迁人员发生冲突，愤怒的韦某驾驶私家车高速冲向站在路边绿化带的拆迁人员，将一群人撞倒，并扬起大量尘土。据官方通报指，事故造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_1死9伤.html" target="_blank">
  1死9伤
 </a>
 。
 <br/>
 <br/>
 事发后，大批民众在现场围观拍摄，有现场目击居民表示，韦某在该处已住了几十年，〝早上就发现有挖土机开始拆韦某的楼房，也不知道房子出了什么问题被拆。〞
 <br/>
 <br/>
 而韦某的妻子王女士则表示，清晨6点多接到电话，说有人要搬走她家的东西，她连忙从外地赶回家。王女士说，开车撞倒拆迁人员的是她的丈夫。
 <br/>
 <br/>
 目前，肇事韦某已被扣查，事件正在调查之中。但官方尚未给出韦某房屋为何被强拆的具体理由。
 <br/>
