### 彭斯演讲地点有深意 媒体：有华裔专家指点
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国副总统.html" target="_blank">
  美国副总统
 </a>
 彭斯10月4日在华盛顿智库
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈德逊研究所.html" target="_blank">
  哈德逊研究所
 </a>
 演讲，从多个层面严厉谴责中共。英媒指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯演讲.html" target="_blank">
  彭斯演讲
 </a>
 符合该智库倡导的原则，还直接引用了其高级成员的观点。另有分析指，演讲多处引述中国著作，应该出自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人智囊.html" target="_blank">
  华人智囊
 </a>
 之手。
 <br/>
 <br/>
 英国广播公司（BBC）10月6日报导指出，彭斯在演讲中特别提到，川普（特朗普）政府采取了果断行动，以美国领导为主线，应用包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈德逊研究所.html" target="_blank">
  哈德逊研究所
 </a>
 在内的各家智库长期倡导的原则和政策来回击中共。
 <br/>
 <br/>
 哈德逊研究所成立于1961年，创始人是战略分析顾问坎恩。该研究所的自身定位是，〝通过发表著作、召开大会、提供政策介绍和建议，致力于为公共政策决策者和全球政府和商界领袖提供指导。〞
 <br/>
 <br/>
 报导称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯演讲.html" target="_blank">
  彭斯演讲
 </a>
 中还提及哈德逊研究所中国项目主任白邦瑞博士的观点称，中共反对美国政府的行动和目标，正在与美国的盟友和敌人建立关系，〝这与建立任何和平和高效的中美关系的意图相矛盾〞。
 <br/>
 <br/>
 这一观点来自白邦瑞2015年出版的《百年马拉松——中国取代超级大国美国的秘密战略》。他在此书中详细论述了中共如何欺骗美国帮助其称霸世界。
 <br/>
 <br/>
 报导称，白邦瑞被称为〝中国通〞，是美国政府的高级智囊，曾被川普总统称赞为〝中国问题顶级权威〞。
 <br/>
 <br/>
 除了〝中国通〞的观点外，彭斯演讲还多次直接引用中国著作和成语。有分析指，这显示其背后还有非常熟悉中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人智囊.html" target="_blank">
  华人智囊
 </a>
 。
 <br/>
 <br/>
 据港媒报导，彭斯引用的一句中国古语〝人看眼前，天知未来〞，出自明代冯梦龙《喻世明言》（古今小说）第31卷〈闹阴司司马貌断狱〉。原意指天道报应，彭斯藉此呼吁要对上天有信心。
 <br/>
 <br/>
 彭斯还用〝化犁为剑〞一词，描述中共窃取美国知识产权。
 <br/>
 <br/>
 另外，彭斯还引述了一位五四时期文人的语句〝中国人对于异族历来只有两样称呼，一样是禽兽，一样是圣上〞，出自鲁迅《随感录四十八》。彭斯引用此语批评中共不尊重美国主权。
 <br/>
 <br/>
 报导引述评论人士分析称，彭斯引用的字句很难是出自西方人之手，只能来自真正了解中国的专家。因此川普政府应该拥有非常熟悉北京的华人智囊，甚至可能是来自北京体制内的人士。
 <br/>
 <br/>
 鲁迅以反传统的激烈言辞著称，被中共宣传成〝伟大的马克思主义者〞。但在海外的华人文化圈中，鲁迅的人品和文章并不被推崇。彭斯讲稿中将鲁迅称为〝伟大的中国作家〞，显然带有中共体制内的烙印。
 <br/>
 <br/>
 在彭斯演讲前几日，白宫国安会亚洲事务资深主任马特．波廷杰出席中共驻美使馆的〝十一〞庆典时，也曾以《论语》中的名言〝名不正则言不顺，言不顺则事不成〞，来阐述川普政府已经明确〝与中共竞争〞的重要战略。
 <br/>
 <br/>
 中美贸易战已经开始向全面对抗转变。上述迹象显示，川普政府为应对中共，已经对中国有了比较深入的研究。
 <br/>
 <br/>
 而中共方面，一直宣称〝不知道川普到底要什么〞。多家外媒引述内部消息指，北京内部对川普意图依然〝很迷茫〞，对如何应对贸易战仍在争论不休。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
