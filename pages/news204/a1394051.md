### 国资背景银豆网涉诈骗 受害人：还我血汗钱
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】首先来关注中国的消息。中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_P2P网贷平台.html" target="_blank">
  P2P网贷平台
 </a>
 今年7月份发生倒闭潮，其中有国资背景的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_银豆网.html" target="_blank">
  银豆网
 </a>
 ，被曝光诈骗数万名投资人共44亿元，伙同亿邦公司洗钱，即将在香港上市，10月3号，数十名受害人前往该公司门口维权。
 <br/>
 <br/>
 10月3号，数十名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_银豆网.html" target="_blank">
  银豆网
 </a>
 投资受害人聚集在位于杭州的浙江亿邦通信科技有限公司门口维权，但公司已紧急搬离，更名为〝万思〞，但旁边被剥掉的〝亿邦〞两字仍依稀可见。
 <br/>
 <br/>
 受害人打出：〝北京海淀公安保护黑恶势力，赃款转移亿邦还我银豆44亿〞的横幅，高喊〝还我血汗钱〞的口号，但维权行动到警察阻挡，警察抢夺横幅时，双方僵持不下，场面一度混乱。
 <br/>
 <br/>
 现场视频：〝我们这边银豆方的，调查亿邦公司的资金的流向，包括进和出；另外呢，我们要求请求杭州警方来立案调查亿邦公司所有资金走向以后，冻结企业的相关帐号；第三个，在上诉内容彻查清楚之前，冻结相关资金与股权，暂停亿邦在的香港上市计划，这是我们的诉求。〞
 <br/>
 <br/>
 当天杭州信访局、市公安局等数名官员和亿邦副总出面接待，受害人代表向他们传达了维权的诉求，得到的答覆是，杭州警方不能单独立案，要北京海淀警方发协查，他们才能查。
 <br/>
 <br/>
 记者4号打通多名受害人电话，疑似受到压力，都不敢接受采访。
 <br/>
 <br/>
 匿名银豆网投资受害人：〝投资这个东西都会有风险的，只不过是有部分的那个非法融资嘛，这块有些东西管控不好嘛。〞
 <br/>
 <br/>
 银豆网最大股东是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华信电子.html" target="_blank">
  华信电子
 </a>
 。而华信电子为国有企业，是军方设备供应商，7月18号，银豆网停止运营，资金无法兑付，全国各地受害人纷纷加入维权的行列。
 <br/>
 <br/>
 新唐人记者熊斌、施宜君采访报导
</div>
