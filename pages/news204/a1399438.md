### 习近平重提四个字  被官媒诡异剪切
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 使得中共陷入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内忧外患.html" target="_blank">
  内忧外患
 </a>
 困境的大背景下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在早前的一次视察中重提〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自力更生.html" target="_blank">
  自力更生
 </a>
 〞的口号。诡异的是，中共官媒报导居然删除了习近平这一提法，这一消息近日被英媒曝光。
 <br/>
 <br/>
 据英媒《金融时报》11月13日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 9月底到东北黑龙江省考察中国第一重型机械集团时，向工人们说，国际上的先进技术、关键技术〝越来越难以获得〞，〝逼着我们走
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自力更生.html" target="_blank">
  自力更生
 </a>
 的道路〞。
 <br/>
 <br/>
 报导称，央视新闻联播对习近平的发言视频进行了剪辑，删掉了讲话的第一部分。中共党报《人民日报》海外版也作出了同样的编辑。
 <br/>
 <br/>
 除了〝自力更生〞这一口号外，〝外国技术越来越难获得〞的内容也遭党媒删除。
 <br/>
 <br/>
 报导引述一些政治观察家的观点认为，这些删改反映了中共宣传部门的担忧，即习近平的言论，会被视为承认中共采用了强制和窃取等手段，来获取外国知识产权。这是美国川普政府提出的中共不公平贸易行为之一。
 <br/>
 <br/>
 在党媒删除〝自力更生〞这一口号之后，习近平南下广东视察，再次强调要〝自力更生〞。
 <br/>
 <br/>
 据统计，中共过去提及〝自力更生〞的口号，分别是在1941年的延安大生产运动时期，其后是大饥荒时期的1960年，以及文革末期的1975年。虽然年份不同，但有一个共同点，都是中共遇到严重的政治和经济困难，并且在国际上处于孤立状态之下。
 <br/>
 <br/>
 习近平在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 使得中共陷入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_内忧外患.html" target="_blank">
  内忧外患
 </a>
 困境的大背景下，重提这一口号，引起舆论关注。
 <br/>
 <br/>
 自由亚洲引述上海资深财经界人士方先生的话认为，习近平重提〝自力更生〞，主要原因是中美贸易战开打，原来可以轻易获得的好处没有了，没有外部资源了，经济状况逐渐恶化了，所以只能自给自足了，也就是国家被迫改变目前的外向型经济模式。
 <br/>
 <br/>
 方先生说，目前经济方面的高层人士，即使只是从个人和家庭命运的角度来看，都对目前的局势比较忧虑，更麻烦的是这种恐慌是普遍性的。
 <br/>
 <br/>
 有接近中南海高层的知情者对美媒说，除了中美贸易战愈演愈烈引发民间恐慌之外，真正黑云压顶的是中国面临的日趋严重的粮食危机。
 <br/>
 <br/>
 几十年前的〝大饥荒〞，令人记忆犹新，更令习近平挥之不去。知情者说，习最担心的是大饥荒重现！一旦历史重演，〝中国梦〞必然破碎，随之而来的是苦难降临！
 <br/>
 <br/>
 50年代未，毛泽东发起的〝大跃进〞运动，最终酿成了导致数千万人饿死的3年大饥荒，据前新华社记者杨继绳估计，大饥荒至少造成3600万民众被饿死，因饥饿少出生人数4,000万人，两者共计7,600万人。
 <br/>
 <br/>
 专家分析，当下的中国，不仅是农业粮食问题，中共数十年来的畸形发展，已经让中国各行各业的经济问题积重难返。而贸易战让中共的艰难处境更是雪上加霜。习近平此时重提〝自力更生〞，充分体现出中共目前的困境。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：方舟）
