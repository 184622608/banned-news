### 重庆客车坠江视频曝光 司机凌晨K歌疑疲劳酿祸
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆公交车坠江.html" target="_blank">
  重庆公交车坠江
 </a>
 事故最新消息称，该
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公交司机.html" target="_blank">
  公交司机
 </a>
 在事发当日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凌晨K歌.html" target="_blank">
  凌晨K歌
 </a>
 ，数小时后出事。据网传公交车坠江时的视频显示，出事公交车在行驶途中突然驶入对向车道，撞上迎面而来的私家车后，撞断护栏，坠入江中。外界怀疑，该公交司机或因疲劳驾驶酿成事故。
 <br/>
 <br/>
 北京时间10月28日，重庆万州区长江二桥上发生公交车坠江事故，一辆公交车与一辆私家车相撞后，冲破护栏掉入长江。截至目前，已有2名遇难者遗体被打捞上来。警方通报称，事发时车上有10多人。
 <br/>
 <br/>
 事故发生后，有网民爆料称，出事公交车车牌为F27085，22路车，驾驶员叫冉勇，车上可能有29人。最初，网传事故源于一辆小轿车违规调头逆行，公交车躲闪不及与其相撞后，冲破护栏掉入长江。
 <br/>
 <br/>
 不过，随后有公交车坠江瞬间的视频在网上热传，视频显示，出事公交车突然违规驶入对向车道，撞上小轿车后失控坠入江中。警方也对此予以证实。
 <br/>
 <br/>
 从现场视频中可见，被公交车撞毁的一辆红色私家车逆停在路边，一名女司机惊魂未定地坐在旁边。
 <br/>
 <br/>
 据陆媒10月29日最新消息称，出事公交车司机冉勇，今年42岁，在全民K歌APP的昵称为〝龙行天下〞。28日凌晨5时24分时，冉勇在全民K歌APP演唱歌曲《再回首》，此时，距离事发只有4个半小时。外界猜测，冉勇或因疲劳驾驶，酿成悲剧。
 <br/>
 <br/>
 检索冉勇的注册信息显示，他于2018年4月13日首次在平台K歌，第一次演唱的是《掌声响起来》。6个多月以来，冉某一共演唱了29支歌曲，拥有9,276个粉丝。在事发前一晚，冉勇在〝全民K歌〞也演唱了《再回首》这首歌。
