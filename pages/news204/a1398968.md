### 李小琳退而不休 又有新头衔曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】李鹏之女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李小琳.html" target="_blank">
  李小琳
 </a>
 退而不休。日前，在河北廊坊举行的一场〝国学智慧与大健康善行峰会〞上，李小琳又以新的身份亮相。近年来，李小琳高调支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 反腐，李鹏之子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李小鹏.html" target="_blank">
  李小鹏
 </a>
 还曾陪同习考察三峡，这些迹象显示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李鹏派系.html" target="_blank">
  李鹏派系
 </a>
 正向习靠拢。
 <br/>
 <br/>
 11月3日至4日，〝行知丝路研究院成立、陶行知国际教育基金成立暨国学智慧与大健康善行峰会〞在河北廊坊举行。中共国务院前总理李鹏之女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李小琳.html" target="_blank">
  李小琳
 </a>
 ，首次以〝陶行知国际教育基金管理委员会主席、行知丝路研究院校长〞的身份出席。
 <br/>
 <br/>
 李小琳在现场致辞说，行知丝路研究院是一个为传承中华文明，服务社会大众的教育平台，愿与大家一起为实现〝美丽中国、教育兴邦、人人健康〞的目标努力。
 <br/>
 <br/>
 这是现年57岁的李小琳退休后，曝光的又一新头衔。今年5月末，曾有〝电力一姐〞之称的〝红二代〞李小琳提前退休，不再担任中国大唐集团党组成员、副总经理职务。这离她退休年龄还差三年。
 <br/>
 <br/>
 据报，李小琳于5月18日申请退休，获得中组部同意。她退休后立即获得新的职务，担任〝丝路规划研究中心常务副理事长〞，并赴俄罗斯圣彼得堡，参加2018年圣彼得堡国际经济论坛。
 <br/>
 <br/>
 如今，李小琳又身兼〝陶行知国际教育基金管理委员会主席、行知丝路研究院校长〞的职务。
 <br/>
 <br/>
 据行知丝路研究院官网介绍：该研究院由陶行知国际教育基金主席李小琳女士，发起倡议成立并兼任校长。该教育基金的主要业务范围是，为国际商学培训、交流合作办学等。
 <br/>
 <br/>
 公开履历显示，李小琳生于1961年6月1日，现任澳门电力董事、香港中资企业协会执行董事、中共电力企业联合会常务理事和中共工商理事会常务理事等职。此外，李小琳还担任中共第十一届、第十二届全国政协委员，中共妇联执行委员。
 <br/>
 <br/>
 李小琳向来行事高调，加之是中共前领导人李鹏之女，因此有〝红色公主〞之称，倍受舆论关注。
 <br/>
 <br/>
 近年来，李小琳力挺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 反腐。2014年她在出席中电国际年中工作会议时强调，要〝懂大势、不出事、知历史〞。
 <br/>
 <br/>
 2015年博鳌论坛上，李小琳被问及对央企反腐的看法时称，她表示，〝我们大力支持反腐！〞
 <br/>
 <br/>
 李鹏的儿子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李小鹏.html" target="_blank">
  李小鹏
 </a>
 也多次发言力挺习当局反腐〝打虎〞，并在山西积极配合习近平，拿下了多名令计划、周永康的心腹。他于2008年6月由商入仕，出任山西省省长。
 <br/>
 <br/>
 2016年9月，李小鹏被调任中共交通运输部部长。今年4月24日，李小鹏陪同习近平一行考察三峡大坝。有分析认为，李鹏子女皆高调挺习，显示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李鹏派系.html" target="_blank">
  李鹏派系
 </a>
 正在向习近平靠拢。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：祝馨睿）
