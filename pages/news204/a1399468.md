### 江苏副省长缪瑞林落马 曾参与迫害法轮功
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】中共官媒周四通报，江苏省副省长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_缪瑞林.html" target="_blank">
  缪瑞林
 </a>
 ，因涉嫌〝严重违纪违法〞，目前正在接受调查。通报没有说明缪瑞林被查的具体原因，据报，缪瑞林曾和已经落马的前南京市委书记杨卫泽搭档。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_缪瑞林.html" target="_blank">
  缪瑞林
 </a>
 在江苏农业系统工作20年，曾任江苏省农林厅农业局副局长、省农业资源开发局局长等职。2004年起，先后任宿迁市常务副市长、市长、市委书记等职；2013年1月起，任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江苏副省长.html" target="_blank">
  江苏副省长
 </a>
 、南京市委副书记、代市长、市长兼南京市江北新区管委会主任；2018年1月起，任江苏省副省长。
 <br/>
 <br/>
 据报，网络一直流传有人举报缪瑞林在宿迁的违法情况。缪瑞林还与已经落马的江苏省前常委、南京市委书记杨卫泽搭档一年。
 <br/>
 <br/>
 杨卫泽被指是中共前政法委书记周永康的马仔。2016年12月14日，杨卫泽被判刑12年半。
 <br/>
 <br/>
 另据《明慧网》报导，缪瑞林任南京市市长期间，曾参与迫害当地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员。
</div>
