### 【禁闻】11月2日维权动态
------------------------

 【新唐人2018年11月02日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 〞拍照 遭追捕
 <br/>
 <br/>
 中共11月5号至10号将在上海举办首届国际进口博览会，十多名上海维权人士11月1号前往会场附近拍照后，遭到警察追捕，目前仍然在四处躲藏。
 <br/>
 <br/>
 2. 无锡维权人士遭地方当局
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法监禁.html" target="_blank">
  非法监禁
 </a>
 <br/>
 <br/>
 江苏省无锡市江阴县维权人士毛黎惠，因多次举报当地政府非法征地，被江阴申港派出所警察与村干部长期非法关押在黑监狱，10月29号凌晨逃脱后，前往无锡市公安局报案，结果还是被转交给当地继续非法关押。
 <br/>
 <br/>
 3. 陕西冤民巡视组上访 遭警阻止
 <br/>
 <br/>
 中共中央第四巡视组据称正在陕西专项巡视脱贫攻坚领域的腐败问题，10月30号，20多名访民前往巡视组接待点反映各自因案致贫致残致死的冤情，但遭到几十名全幅武装的警察阻挡。
 <br/>
 <br/>
 4. 助桃江
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肺结核事件.html" target="_blank">
  肺结核事件
 </a>
 受害者 律师遭打压
 <br/>
 <br/>
 广东维权律师陈科云，因向湖南桃江肺结核事件受害者提供法律援助，9月份遭广东省司法厅威胁吊销执照，10月31号，果然收到司法厅撤销其律师执业证的文书。
