### 晋华〝突然死亡〞 生产全停摆 美方下个目标浮出
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】中国福建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晋华.html" target="_blank">
  晋华
 </a>
 集成电路公司因涉嫌窃取美国企业机密被美国政府列入了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出口管制清单.html" target="_blank">
  出口管制清单
 </a>
 ，近日已正式提起刑事及民事诉讼。晋华公司的美国合作供应商的驻厂人员已全撤离，该公司的所有设备、生产运作已全面停止。有媒体指中国记忆体产业〝三架马车〞之一的合肥长鑫公司，可能会成为美国政府的下一个制裁目标。
 <br/>
 <br/>
 11月1日，美国司法正式对外宣布，美国联邦大陪审团已对中国福建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晋华.html" target="_blank">
  晋华
 </a>
 集成电路有限公司及其合作伙伴台湾联电电子股份有限公司，以及三名台湾人提出起诉，指控这两家公司及涉案人窃取传送和持有美国美光公司的商业机密。
 <br/>
 <br/>
 据台湾媒体《上报》报导，从美国实施出口管制的第一天起，福建晋华的美国半导体合作方驻厂人员已全撤出，并中断所有电话、邮件往来。所有设备、生产运作都已经全面停止，已经下单却未出货的机台全数暂停出货。
 <br/>
 <br/>
 报导称，晋华公司原先预计从今年9月开始量产，遭美国政府制裁后，量产阶段恐将〝遥遥无期〞了。
 <br/>
 <br/>
 据介绍，晋华公司的软件系统采购了许多美国产品，如工厂营运管制系统（MES）就是使用的美国IBM的产品，目前IBM已切断与福建晋华的任何联系。
 <br/>
 <br/>
 报导并表示，中国长江存储、合肥长鑫与福建晋华，被视为中国记忆体产业的〝三架马车〞。在福建晋华遭到美国政府控告后，有消息指同样聘雇了美光前技术和工程人员的合肥长鑫公司，有可能成为美国政府的下一个制裁目标。
 <br/>
 <br/>
 有陆媒引述来自〝供应链〞的消息称，从福建晋华撤离的美国合作企业，包括半导体设备商Applied Materials 、 Lam Research 和 Axcelis，其他与晋华公司有合作关系的美国的设备材料供应商还有 KLA-Tencor 、 Gatan 、 ECI Technology 、 Bruker 、 Engegris 等公司。
 <br/>
 <br/>
 此外，福建晋华的 DRAM 技术合作开发伙伴台湾联电公司今年10 月 31 日已宣布，该公司决定停止 DRAM 开发项目。
 <br/>
 <br/>
 晋华公司在美国被起诉后，美光公司（Micron）11月1日发表声明称，该公司对于美国司法部的做法感到欣慰，美光几十年来耗资数十亿美元用于科技研发，商业机密却被盗窃，这样的犯罪行为理应受到惩处。
 <br/>
 <br/>
 据公开的资讯，福建晋华是由中国福建省电子信息集团、及泉州和晋江两级政府2016年共同出资创办的公司。
 <br/>
 <br/>
 福建晋华一期投资370亿元（人民币，下同），其中获得中国国家集成电路产业投资基金（简称大基金）注资30亿元。
 <br/>
 <br/>
 在中共政府推动的〝中国制造2025〞的规划中，福建晋华作为中国高科技产业的半导体积体电路制造商，也被纳入了中共〝十三五〞半导体重大生产布局规划。
 <br/>
 <br/>
 2016 年，福建晋华与台湾联电达成合作协议，委托联电进行 DRAM 技术开发，而按照协议开发出的技术将由福建晋华和联电共同持有。
 <br/>
 <br/>
 2017年12月，美国美光公司向加州的一家联邦地方法庭提起诉讼，指控台湾联电和福建晋华通过收买美光科技的两位前员工盗用了该公司的内存晶片技术，并利用这些技术为福建晋华进行 DRAM 开发。
 <br/>
 <br/>
 今年1月，福建晋华和联华电子竟反过来在中国境内控告美光科技的产品侵犯他们的专利，请求福建法院禁止美光科技在中国的分公司生产、销售相关产品，并且向他们支付赔偿金。
 <br/>
 <br/>
 今年7月，福州市中级法院裁定美光子公司侵犯联电专利权，要求美光公司立即停止在中国市场销售多项动态随机存取记忆体（DRAM）与储存型快闪记忆体（NAND Flash）产品。
 <br/>
 <br/>
 这个事件引爆了美国政府的怒火，成为晋华公司遭美国政府制裁并被提起诉讼的直接导火线。
 <br/>
 <br/>
 （记者唐迪综合报导/责任编辑：明轩）
