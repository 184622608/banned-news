### 川普重启军备竞赛？中共急撇清避谈重要事实
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】日前，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）宣布退出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中程核导弹条约.html" target="_blank">
  中程核导弹条约
 </a>
 〞（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_INF.html" target="_blank">
  INF
 </a>
 ），直言警告中俄，并表态将扩充核武库。此举被指或效仿当年美苏冷战，升级美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军备竞赛.html" target="_blank">
  军备竞赛
 </a>
 。而中共外交部回应称，该条约由美俄缔结，与北京无关。
 <br/>
 <br/>
 美国广播公司新闻网（ABC News）10月22日报导指，白宫安全顾问博尔顿定于23日与俄罗斯总统和国防部长会晤，将向俄方宣布美国退出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_INF.html" target="_blank">
  INF
 </a>
 的决定。
 <br/>
 <br/>
 22日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统在搭乘〝陆战队一号〞前，在白宫南草坪与媒体对话。有记者提问，美国是否要退出INF，并准备强化美国核武库。川普回应，〝我们会强化（核武库）〞，直到〝他们（中俄）醒悟〞。
 <br/>
 <br/>
 川普指出，俄罗斯没有遵守这项协议，此事早该在几年前处理。他还表示，美国目前比其他国家更有钱，更有条件强化核武库。
 <br/>
 <br/>
 川普说，当中俄停止扩充核武库时，美国也会停止，甚至愿意裁减。但是现在中俄并没有遵守规则。
 <br/>
 <br/>
 在被问及此举是否针对俄罗斯时，川普回答说，〝这对你想到的任何一个（国家）来说都是一种警告。它包括中国（中共），包括俄罗斯和任何想要玩这种游戏的国家。你们（中俄）不能和我玩这个游戏。〞
 <br/>
 <br/>
 23日，中共外交部发言人华春莹回应辩称，中共的国防方针〝向来都以防御性〞为主，现在美国自己想废除与俄国缔结的INF，却把中方扯进来，是〝不可理喻〞，但她没有解释中共的中程导弹数量为什么会高居全世界榜首。
 <br/>
 <br/>
 20日，川普在内华达州首次宣布退出INF的声明，并谴责俄罗斯一直没有遵守INF。川普还说，除非俄罗斯、中共和美国一起协商，都不研发这些武器，否则美国只能退出这一条约。
 <br/>
 <br/>
 美国官员对《纽约时报》表示，川普宣布退出INF，不只针对俄罗斯。因为中国不是INF的签约国，中共多年来不受约束地发展中程核导弹，现在中共国已成为〝全球最大的中程弹道导弹打击力量拥有国〞。
 <br/>
 <br/>
 INF是美国前总统里根和苏联前领导人戈尔巴乔夫1987年所签署，内容是禁止双方生产或测试所有射程为500至5,500公里的陆基导弹。条约涵盖携带核弹头和常规弹头的陆基导弹。
 <br/>
 <br/>
 博尔顿22日对《华尔街日报》说，如果中共是INF的签约方，那么中共所有弹道导弹的三分之一到一半都将违反该条约。
 <br/>
 <br/>
 前美国官员邓志强（Abraham Denmark）19日在推文中说，不受INF约束的中共一直在研发部署比美国更先进的导弹。退出INF能为美国松绑，好让美军部署更有效的武器系统，以应对中共威胁。
 <br/>
 <br/>
 《国会山报》也说，退出INF标志着美国可以自由地开发类似中共部署的地基导弹，与中俄的导弹发展相匹配。
 <br/>
 <br/>
 《纽约时报》报导，美国解绑后很可能会部署一种经过重新设计、能够从陆地发射的战斧巡航导弹。消息人士称，五角大楼已经在开发核武器，以匹配和应对中共已经部署的武器。
 <br/>
 <br/>
 另据《国会山报》分析，川普宣布退出条约也可能是一种策略，目的是重新谈判一个公平协议。因为INF规定，协议的任何一方如果决定终止协议，必须提前6个月通知对方。因此，华盛顿有时间与莫斯科和北京谈判，以制定一项包括这三国在内的协议。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
