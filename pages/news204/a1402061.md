### 国家宪法日 维权人士访民遭驱赶关押
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】今年的12月4号是中国第5个〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家宪法日.html" target="_blank">
  国家宪法日
 </a>
 〞，全国多地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 在当天上街宣传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宪法.html" target="_blank">
  宪法
 </a>
 ，呼吁政府切实保障宪法赋予人民的权利，但却被置之不理、驱赶、甚至关进黑监狱。
 <br/>
 <br/>
 北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 李美青告诉《民生观察》，4号上午，20多名在京维权人士相约上街宣传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宪法.html" target="_blank">
  宪法
 </a>
 ，呼吁政府遵守宪法，切实保障公民的合法财产权及言论、集会、游行等权利，但是他们在天安门广场附近遭到巡逻警察阻拦，随后，维稳部门把部分人强行拉回家，有几名外地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 也被抓到久敬庄黑监狱关押。
 <br/>
 <br/>
 北京维权人士李美青：〝在北京维权，到处都是警察，你真想维权的话，寸步难行，就这感觉。我的心情就是很沉重，我觉得法律在中国就是一个虚无飘渺的东西，根本就没有，现在警察治国。〞
 <br/>
 <br/>
 也有在京外地访民反映，当天他们家乡的多名访民被辖区的维稳办人员堵在家里，禁止他们到北京上访。
 <br/>
 <br/>
 陕西访民王英强的女儿4号前往信访办维权。她家因遭政府长期暴力维稳，一家四口一死二残。她表示，目前尽管李克强总理已对她们的案子做了批示，但当地仍不依法办事。
 <br/>
 <br/>
 陕西访民王英强女儿：〝12月4号，我到西咸新区信访局找领导人反映，就说李克强总理把我们家的事情批示下来了，希望政府的领导能积极去督促三公司，尽快解决我们家的事情。他们仍然不同意，说就算是总理批下来，也没有批具体怎么处理，具体还是下面办事，下面想怎么弄就怎么弄。〞
 <br/>
 <br/>
 武汉维权人士蔡莲也向《民生观察》透露，武汉市30多名访民当天相约上街宣传宪法，但他们刚走近政府大楼，就被数十名警察包围，被强行搜身，宣传宪法的标语也被没收了。
 <br/>
 <br/>
 蔡莲表示，宪法的一个重要作用是限制政府公权，让政府受到有效的制约，但在现实中，政府常常权大于法，以言代法，让受害百姓有理无处讲，有冤无处申。
 <br/>
 <br/>
 新唐人记者陈汉、陈洁、柏妮采访报导
</div>
