### 重庆女子持刀袭击幼儿园14名学童受伤
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】10月26日上午9点30分左右，重庆巴南区鱼洞新世纪幼儿园发生砍杀事件。目前有10多名师生受伤，行凶者为一名妇女。
 <br/>
 <br/>
 据巴南区教委会通报，事发时，这间幼儿园教师带全体幼儿到园门外做操完毕，正带着幼儿返回园内途中，一名中年妇女突然冲入人群
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_持刀行凶.html" target="_blank">
  持刀行凶
 </a>
 。
 <br/>
 <br/>
 已知有14名幼儿受伤被送入医院，其中4人伤情较重。
 <br/>
 <br/>
 涉案的39岁女子已被拘捕, 犯案动机目前还不明确。
 <br/>
 <br/>
 据悉，这间园幼儿园有175名幼儿，17名教职员工，另有2名校园保安。
 <br/>
 <br/>
 新唐人记者尚敬综合报导
</div>
