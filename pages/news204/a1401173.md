### 贺建奎承认  还有一个基因编辑婴儿在早孕期
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】中国深圳〝产出〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 的始作俑者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 ，周三在香港举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第二届人类基因组编辑国际峰会.html" target="_blank">
  第二届人类基因组编辑国际峰会
 </a>
 上披露出惊人细节——他的团队共对约30个胚胎中的70％进行了基因编辑，还有1个在母体内处于怀孕早期阶段。而另一位外国科学家指贺某此前在其他地方透露说，他手中还有16个胚胎因工作暂停而已冻结，这些冻结胚胎未来的状况仍是个未知数。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 出生的消息在海内外引发舆论风暴后，主导进行这项实验的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 于11月28日现身正在香港举行的第二届国际人类基因组编辑峰会并发表了演讲。
 <br/>
 <br/>
 贺建奎在演讲开始时先声明，他其所属的大学对于他所进行的这项研究〝不知情〞，而他的实验内容与数据，目前也还没有接受其他专业机构或团队的验证，基因编辑婴儿〝露露〞和〝娜娜〞出生的消息在两天前被公开是一个〝意外〞。
 <br/>
 <br/>
 他说：〝首先我必须要道歉。由于实验的保密性不强，所以数据被洩露了。所以我必须要在今天这个场合，跟大家分享这个数据。〞
 <br/>
 <br/>
 据贺建奎在这次演讲中介绍，原本有8对志愿者夫妇进入了这项试验程序，都是父亲是HIV阳性、母亲HIV阴性的组合。在实验中途有 1对夫妇退出，最终有7对夫妇接受了基因编辑试验。
 <br/>
 <br/>
 他说，研究团队在小鼠中做了多代的研究，其中有3代小鼠的研究表明，〝它们的组织看上去很正常，行为也没有异常〞，于是他们决定〝推进到人类研究〞。
 <br/>
 <br/>
 他声称，在试验中对胚囊进行测序时，发现一个胚囊出现了移码变异，CCR5蛋白更短；而另一个胚囊出现了CCR5的部分删除，这个变异让CCR5蛋白变得不稳定，能减弱HIV的感染。在告知父母相应的风险后，这些胚囊被植入母体，开始怀孕。
 <br/>
 <br/>
 他披露，他的研究团队共对约31个胚胎中的70％进行了基因编辑，目标终身HIV免疫。不过，他并没有透露除了已出生的〝露露〞和〝娜娜〞外，其余经编辑后的胚胎现况究竟如何。
 <br/>
 <br/>
 在演讲后的讨论环节，斯坦福大学致力于用基因组编辑治疗儿童遗传病的Matthew Porteus教授提问：这项研究一共对多少个胚囊注射了Cas9？
 <br/>
 <br/>
 贺建奎回答说：〝注射的有更多，有31个发育到了胚囊阶段。〞
 <br/>
 <br/>
 Matthew Porteus教授接着发问：〝目前还有其他在怀孕的母亲吗？〞
 <br/>
 <br/>
 贺建奎回复说：〝还有1名，但现在还在怀孕早期。〞
 <br/>
 <br/>
 有现场的听众提问：〝能否介绍一下，具体的伦理是怎么通过的？然后能否详细介绍下未来对于这两名孩子的治疗筛查计划？〞
 <br/>
 <br/>
 贺建奎对两个问题避而不答，只承诺自己未来我会对这项研究〝保持透明和公开〞，向全世界公开孩子的生理信息，然后〝大家一起决定下一步〞。
 <br/>
 <br/>
 现场还有人提问：〝一共有8对夫妇参加实验，其中1对夫妇退出，剩下的24个胚胎呢？〞
 <br/>
 <br/>
 贺建奎说：〝生出双胞胎的这对父母，刚好是第一对怀孕的。其他人暂时推后。〞
 <br/>
 <br/>
 又有人追问：〝基因编辑婴儿是否涉及伦理问题？〞
 <br/>
 <br/>
 贺建奎声称：〝我从美国回来，不熟悉中国在这方面的规定。〞
 <br/>
 <br/>
 参与了这次会议的嘉宾威斯康星大学麦迪逊分校生物法学和生物伦理学教授R. Alta Charo其后向媒体表示，他知道〝贺博士〞也曾在别处演讲时说过，他还有16个来自其他夫妇的胚胎，他的工作暂停时，所有的胚胎都被冻结了。
 <br/>
 <br/>
 Charo对此表示十分担忧：〝这些胚胎会发生什么，甚至是谁来决定发生什么，都是未知的。〞
 <br/>
 <br/>
 人类基因组编辑国际峰会委员会成员的翟晓梅会后也对媒体表示，听完贺建奎的发言，感觉到他只是提供了一些研究的数据，但在回答问题的环节时，对伦理方面的问题并没有给予更多和深入的回应。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
