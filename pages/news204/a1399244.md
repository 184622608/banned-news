### 【禁闻】执行〝敲门行动〞 40派出所长死于非命
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】中共在2017年推出迫害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 的所谓〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_敲门行动.html" target="_blank">
  敲门行动
 </a>
 〞，这一年中，40名派出所所长死于非命，这些人基本上都是积极参与了迫害行动的，有评论说，这个现象再一次验证了中国古话讲的〝善恶有报〞的天理。
 <br/>
 <br/>
 据〝法轮大法明慧网〞报导，2017年，中共专门迫害法轮功的组织〝六一零〞、政法委、公安部推出所谓的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_敲门行动.html" target="_blank">
  敲门行动
 </a>
 〞，在全国范围内对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 进行绑架和骚扰，被迫害人数达数万人，其中，000多名法轮功学员被非法判刑，至少4名法轮功学员死亡。大陆各地的派出所参与了所有的绑架、抄家、跟踪、监视、制造假证据、栽赃陷害、刑讯逼供、非法判刑等迫害。
 <br/>
 <br/>
 公开资料显示，这一年全国在职警察的非正常死亡人数也相当惊人，达361人。其中，派出所所长、教导员死亡40人，12人因突发心肌梗塞、心脏病死亡，12人突发各种疾病猝死，11人车祸死亡，3人脑出血死亡，1人突发败血症死亡，1人伤势严重死亡。平均死亡年龄42.7岁，最大的59岁，最小的32岁。
 <br/>
 <br/>
 49岁的河北省秦皇岛市公安局抚宁分局抚宁镇派出所所长潘权，2017年11月9号，因突发心脏病死在了工作岗位上。潘权是当地迫害法轮功，实施公安部〝敲门行动〞的主要执行人。
 <br/>
 <br/>
 潘权原为抚宁县茶棚乡派出所警察，动辄抢劫法轮功学员家中财物、勒索性罚款，把法轮功学员双手戴上手铐，按在地上用木棍压在小腿上，站上两个人用力下踩。
 <br/>
 <br/>
 因迫害法轮功学员卖力，潘权由普通警察提升为派出所所长，还得到了秦皇岛市、河北省、和公安部的多次所谓表彰。
 <br/>
 <br/>
 不过，就在潘权上任的当天，他9岁的儿子在玩耍时，不小心被车撞死。不久，他的内弟利用他的权势开爆竹店，后来爆竹店爆炸，他的内弟被当场炸死。
 <br/>
 <br/>
 法轮功发言人张而平：〝中国人有句古话就是善恶有报。如果是不明真相的话，他实际上也是成了迫害的一个工具，现在在中国这么多人冒着生命危险讲真相的情况下，如果对真相处于漠然的状态，或者随从这个政府的政策对法轮功实行迫害的话，那他等于是自己选择当帮凶的工具。〞
 <br/>
 <br/>
 其实，自中共1999年迫害法轮功以来，警察死于非命的情况一直在逐年增加，公检法、甚至包括宣传教育领域，那些急切想藉着迫害法轮功往上爬的中共各级官员，莫名其妙死亡的消息也不绝于耳。还有不少被关进监狱的。
 <br/>
 <br/>
 根据中共的公开通报，从2012年开始，相继落马的政法委书记，就达60多人，包括中央政治局常委、省部级及厅局级的中共官员。而2012年引发中共政坛地震的王立军、薄熙来、周永康都是迫害法轮功最残忍的策划人和刽子手。
 <br/>
 <br/>
 张而平：〝在将近20年的迫害里，无数被迫害的法轮功学员他们的冤情是无处申冤的。特别是对法轮功学员活体器官摘除这件事情，是人类历史上罕见的这种反人类罪，在中共的迫害政策下居然持续了这么多年，大家想一想，做坏事的人不可能永远逍遥法外。〞
 <br/>
 <br/>
 法轮功发言人张而平说，天网恢恢，疏而不漏。希望那些还在参与迫害的人，认真听一听法轮功学员讲的劝善真相，不要继续行恶，把自己和家人带入深渊。
 <br/>
 <br/>
 采访编辑/刘惠 后制/王明宇
