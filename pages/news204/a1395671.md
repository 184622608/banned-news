### 中国人吃跑最豪华游轮 美〝喜悦号〞宣布：离开中国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】专为中国人量身打造的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诺唯真.html" target="_blank">
  诺唯真
 </a>
 〝喜悦号〞游轮是世界上最大最豪华的游轮之一，但进入中国市场才1年多时间，就宣布要〝离开中国！〞。有刚刚结束〝喜悦号〞旅程的中国游客自曝〝中国人吃倒了美国巨轮。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诺唯真.html" target="_blank">
  诺唯真
 </a>
 游轮（NCL）成立于1966年，是全球三大游轮品牌之一，总部位于美国迈阿密。2017年6月28日，该公司斥资数亿美元为中国人量身打造创新型豪华游轮——诺唯真〝喜悦号〞游轮在上海开启首航。
 <br/>
 <br/>
 〝喜悦号〞游轮总吨位167,725吨，最大载客量4,930人，设有1,925间奢华客房。游轮船身上的图案透出浓浓的中国风，船舱内所有设施名称、提示牌、菜单等都以中文为主，英文字体缩得很小。
 <br/>
 <br/>
 游轮上的菜肴也多是偏中式的煮法，船上所有活动的主持人都说着普通话，广播也是。只需花费2、3千元人民币，中国游客即可享受吃喝玩乐全套服务。
 <br/>
 <br/>
 不过，〝喜悦号〞游轮进入中国市场才1年多时间，近日突然宣布，将于明年4月〝离开中国！〞通告指，这是公司整体的战略规划所需。不过，有中国游客质疑，这艘美国豪华游轮是硬生生被中国人吃垮的！
 <br/>
 <br/>
 一名刚刚结束了〝喜悦号〞旅程的中国游客，分享了自己在游轮上的所见所闻。文章称，在〝喜悦号〞游轮上，24小时提供各种精致美食、饮料、冰淇淋。但中国游客真的是糟吃糟喝，浪费惊人！每个桌上都有大量遗弃的食物，整盘虾仁、鸡腿、牛排……看着令人心塞。
 <br/>
 <br/>
 文章还写道，〝勤拿少取节约食物 文明用餐礼貌相让〞这样的告示牌到处都有，但不起任何作用。餐厅里貌似见不到一个优雅的中国人……几乎每一个人都急吼吼地攫取着一盘又一盘的菜肴、水果……非要把自己肚子撑到铁饱为止。特别是中国大妈们，她们的肠胃功能令人惊叹不已。如此暴饮暴食也不怕消化系统出问题？
 <br/>
 <br/>
 有船上工作人员透露，在一次5日的航程中，游轮消耗约60吨肉类及禽类、20.5吨鱼类、7吨西瓜、1吨香蕉、11吨土豆、5.2万只鸡蛋……可是，在中国大妈毫无节制的拿取之下，80%的食物是被浪费扔掉。
 <br/>
 <br/>
 文章质疑〝看来这艘美国巨轮硬生生的是被中国人吃倒的了！〞
 <br/>
 <br/>
 该游客还指出游轮离开中国的另一个原因：指诺唯真游轮上的游客〝只浪费、不消费〞。
 <br/>
 <br/>
 文章称，在一个诺唯真游轮介绍会上，主持人提到了在另外23艘诺唯真游轮上，每个星期的啤酒消费至少5万瓶起步，但在中国运行的〝喜悦号〞上，竟然只是它们的百分之一！就更别提红酒了。
 <br/>
 <br/>
 文章调侃说，相信大家已经知道诺唯真游轮退出中国市场的真实原因了，〝中国人吃倒了美国巨轮……《厉害了，我的国》〞。
 <br/>
 <br/>
 近年来，中国游客频频因行为不文明甚至恶劣受到各国人士和媒体的抨击。有分析认为，这背后的原因是，中共建政后破坏中华传统文化，用无神论、斗争哲学对中国人洗脑，造成了如今很多中国人对传统文化一无所知，以致中国人信仰缺失、道德崩溃。
 <br/>
 <br/>
 《共产主义的终极目的》一书中阐述：中国以礼义之邦著称。先秦典籍〝三礼〞（《周礼》《仪礼》《礼记》）里详细记载了中国古人极为丰富优美的礼仪，也探讨了礼的天道根源和礼背后的哲理。这种礼的精神一直延续到中共建政之前。有人也许还记得1949年前受教育的老年人多么彬彬有礼。可悲的是，中共统治几十年后，这一切彻底被毁了。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
