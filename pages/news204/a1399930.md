### 网友恶搞毛新宇朗诵《沁园春 霾》 习曾被逗笑（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】近日，北京又遭阴霾侵袭，有网友在推特上发布一则搞笑视频，恶搞毛泽东孙子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毛新宇.html" target="_blank">
  毛新宇
 </a>
 朗诵《沁园春 霾》。据悉，数年前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共两会.html" target="_blank">
  中共两会
 </a>
 期间遭遇阴霾袭击，曾有中共政协委员在两会上，现场背了几句网友恶搞的《沁园春 霾》，连习近平也忍不住笑了。
 <br/>
 <br/>
 近年来，阴霾在中国大陆四起，北京尤为严重。这几天，北京又笼罩在令人窒息的阴霾之中。11月18日，在社交媒体推特上热传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毛新宇.html" target="_blank">
  毛新宇
 </a>
 朗诵《沁园春 霾》的一段恶搞视频，网友纷纷藉此表达自己的不满和无奈。
 <br/>
 <br/>
 被网友恶搞后的《沁园春 霾》内容是：北国风光，千里朦胧，万里烟飘，望环城内外，浓雾莽莽，鸟巢上下，阴霾滔滔，车舞长蛇,飞机难降，欲上六环把车飙，需晴日。
 <br/>
 <br/>
 空气如此糟糕，引无数美女带口罩，惜一罩掩面，白化妆了，唯露双眼，难判风骚，一代天骄，央视裤衩，只见后座不见腰，尘入肺，有不要命者，来做早操。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 据报导，这首恶搞的《沁园春 霾》还曾在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共两会.html" target="_blank">
  中共两会
 </a>
 上公开亮相。2013年中共召开两会前，阴霾再一次袭击北京。3月6日，香港《苹果日报》报导称，中科院院士、政协委员姚檀栋，在中共两会上当场背了几句网友恶搞的《沁园春 霾》。
 <br/>
 <br/>
 姚檀栋高声朗读：〝北京风光，千里雾霾，万里尘飘……空气如此糟糕，引无数美女戴口罩〞。听者包括习近平，无不被逗笑。
 <br/>
 <br/>
 据史料记载，古代〝霾〞和〝埋〞互通，故每当阴霾发作，皆主大凶不利，古代帝王皆畏惧。皇帝要斋戒，废乐，祭天，退避正殿进行自省，来化解这一〝蔽贤绝道〞的天象示警。
 <br/>
 <br/>
 古人不但将〝非正常雾天〞列入〝霾〞的范畴，还将黄雾黑气、飞沙走石甚至被现代人称为〝沙尘暴〞的极端天气，也都归入〝霾〞的范畴，都属于〝不祥之兆〞。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
