### 重庆女狂砍14幼儿 恶性事件频发陆民担忧（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】重庆市日前又发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_砍杀儿童.html" target="_blank">
  砍杀儿童
 </a>
 恶性事件，14名年仅3、4岁的幼儿，在幼稚园门外被一名持刀女子斩伤。有人说该女子是精神病人，也有人说她是因为家庭纠纷导致发狂。但官方尚未公布案发缘由。对于大陆频发的公共安全事件，不少民众认为社会怨气很深，对社会安全性感到担忧。
 <br/>
 <br/>
 10月26日上午9点半左右，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆市巴南区.html" target="_blank">
  重庆市巴南区
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新世纪幼稚园.html" target="_blank">
  新世纪幼稚园
 </a>
 门口，39岁的重庆女子刘某，持菜刀斩伤幼稚园的孩子，导致14名幼儿受伤，其中4至5人生命垂危。刘某当场被幼稚园老师和保安抓获。
 <br/>
 <br/>
 据现场目击者表示，该幼稚园有175名学生和19名教职员，案发时，老师正带着孩子在校门外做早操。
 <br/>
 <br/>
 对于凶手砍人的原因，当地法律界人士雷先生向自由亚洲称，目前只有民间说法。有人说她是精神病人，也有人说是她两口子吵了架，导致发狂。
 <br/>
 <br/>
 雷先生表示，今年频发公共安全事件，让人很担心。〝我们也是有小孩子的人，所以说我们也很担心这些问题呀。不知道什么地方就出问题了，这个随时都有隐患。前段时间也出现那个随时在街边砍人，安全性确实是一个问题呀。〞
 <br/>
 <br/>
 当地市民彭先生称，案发时，他刚好在看一家杂志关于上月12日湖南衡东县滨江广场恐袭案的报导。〝先我还在看另外一个驾自己的红色路虎车，去撞广场上的，压死10几个，发生了好多起了都。〞
 <br/>
 <br/>
 对于这次斩伤幼儿的惨案，彭先生感到很难过。他认为，频发祸及无辜平民的恐袭事件，反映中国社会怨气很深。〝经济滑坡呀，生存压力大呀，一些人的精神压力没有得到有效的管道去释放，就把愤怒发泄到无辜的人群头上。都是平民，弱势群体。哎呀，我都无语了。〞
 <br/>

 <br/>

 </center>
 <br/>
 据现场参与围堵和抓捕凶手的民众说，斩伤幼儿的女子刘某被抓住后，一直在说政府对她不公什么的，但大家都很痛恨她攻击无辜的孩子，没有人想听她说话。
 <br/>
 <br/>
 据报导，今年以来，中国北京、西安、乐山、烟台、上海、柳州、南宁、衡东、重庆等地，先后发生针对平民的暴力恐袭，迄今为止至少已造成21人死亡，125人受伤。
 <br/>
 <br/>
 针对大陆频频爆发的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恶性伤人.html" target="_blank">
  恶性伤人
 </a>
 事件，有分析认为，这与中共体制的专制、暴政不无关联。《共产主义的终极目的》一书说，共产党以〝恨〞立国，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以恶治国.html" target="_blank">
  以恶治国
 </a>
 。中共利用教育、媒体、艺术等等手段，广泛散播这种〝恨〞，中国人被从小灌输无神论和弱肉强食的斗争哲学，目睹中共政权的暴虐无道和蛮横无理，长大后必然迷信武力，思想和行为充满暴戾和攻击性。
 <br/>
 <br/>
 中国大陆的新闻网站，经常看到这样触目惊心的报导：灭门、弑父（母）、杀妻、投毒、爆炸、砍人、幼儿园老师毫无人性的虐童、暴徒到幼儿园大开杀戒、强奸幼女、强拆、城管打人等等，不一而足。日常生活中，为一点琐事而大打出手的更是司空见惯。
 <br/>
 <br/>
 文章表示，究其根源，这是因为崇尚暴力的共产党已经把〝恨〞灌进了中国人的血液。这种〝恨〞的物质使今天的许多中国人充满暴戾之气，任何时候、任何场合都可能爆发出来。其强度之大、表现方式之恶毒，甚至会使当事人感到震惊和不解。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
