### 胡锦涛大会堂设伏 两会抓捕薄熙来内幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月09日讯】重庆市公安局长王立军逃亡成都美领馆寻求庇护不久，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共两会.html" target="_blank">
  中共两会
 </a>
 期间，时任中共总书记的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 ，让时任中央
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警卫局局长.html" target="_blank">
  警卫局局长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曹清.html" target="_blank">
  曹清
 </a>
 率领中央警卫局的人，在北京大会堂部署把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 控制了起来。
 <br/>
 <br/>
 港媒报导称，周永康、曾庆红等人在中共十八大前对习近平进行过两次未遂暗杀，一次是车祸，另一次是打毒针。习近平一度〝背伤〞失踪两周，同海外政要的会晤也罕见取消。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 之前已经历过江派的多次暗杀，最为外界熟悉的是2006年5月初的一天，胡锦涛秘密到青岛视察北海舰队，突然两艘中国军舰同时向胡乘坐的驱逐舰开火，打死了驱逐舰上五名海军士兵！ 逃过暗杀的胡锦涛一个星期后才回北京。
 <br/>
 <br/>
 胡锦涛后来得知这次的暗杀行动是江泽民通过海军司令张定发所为。回京后胡马上开始抓军权，首先在2006年8月免除了张定发的海军司令员职务，海军副司令王守业被判死缓。这两人都属于江系人马，随后胡对北京卫戍区司令员和政委〝开刀〞。
 <br/>
 <br/>
 2012年2月，重庆市公安局长王立军逃亡成都美领馆寻求庇护，同年3月两会期间，被指搞政变暗杀的重庆市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 被抓捕，在抓捕薄熙来的过程中，据称当时的中央警卫局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曹清.html" target="_blank">
  曹清
 </a>
 也起着重要的作用。
 <br/>
 <br/>
 据透露，2012年3月14日上午，中共全国人大一闭幕，曹清亲自率领中央警卫局的人，在北京大会堂部署，把薄熙来控制起来了。
 <br/>
 <br/>
 王立军、薄熙来事件相继发生后，中共高层各派系间的斗争愈加激烈，直接负责中南海最高层安全的中央警卫局也被卷入其中。
 <br/>
 <br/>
 同年两会结束后，专责温家宝警卫工作的中央警卫局副局长李润田突然被免职，由副局长王庆接手。
 <br/>
 <br/>
 消息称，李润田与薄熙来、周永康早有私通。中共当时九名政治局常委前五名，每人相应配有一名副局长，专责安保。
 <br/>
 <br/>
 英媒当时的一篇报导称，至少有39人因涉及薄案被拘留，中央警卫局副局长、温家宝侍卫长李润田赫然在列。
 <br/>
 <br/>
 2015年在胡锦涛交班不足3年后的两会前，中央警卫局高层人事大动，曹清和副局长王庆双双被调离，其中曹清转任原北京军区副司令员，但王庆传出被抓。中央警卫局副局长王少军接替曹清职务。
 <br/>
 <br/>
 新唐人、大纪元早前披露，薄熙来落马的主要原因，是因参与政变夺权。该计划是由江泽民主导、曾庆红担任主谋，周永康和薄熙来实施，令计划以中办主任的身份，居中联络，暗中策应。在〝十八大〞上，由薄熙来接替周永康的职位，掌管政法委〝第二权力中央〞；待时机成熟后，联合江泽民的军中势力，意图在〝十八大〞后，赶习近平下台，推薄熙来上位。
 <br/>
 <br/>
 2012年2月6日，与薄熙来翻脸的王立军，为了免遭如英国商人海伍德式被杀人灭口，携带机密资料逃到成都美国领事馆，揭开了薄熙来、周永康等策划政变的黑幕。
 <br/>
 <br/>
 同月，时任中共国家副主席的习近平在访美期间，《华盛顿自由灯塔》曝光了王立军移交美领馆材料中其有关薄熙来、周永康联手图谋发动政变、〝十八大〞后废黜习近平的计划。美国副总统拜登向习近平出示了薄、周政变密谋的铁证。
 <br/>
 <br/>
 中共十九大后，官方多次公开批薄熙来、周永康、令计划、徐才厚、郭伯雄等人〝阴谋篡党夺权〞。
 <br/>
 <br/>
 这是继文革时期〝四人帮〞后，40年来中共官场罕见的再次出现〝篡党夺权〞的说法。间接证明了江派这些年来多次针对习近平的政变阴谋。
 <br/>
 <br/>
