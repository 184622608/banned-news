### 中共清网殃及推特 大批网民遭约谈抓捕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】继中共关闭近万个自媒体账号之后，如今开始对海外社交媒体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推特.html" target="_blank">
  推特
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Twitter.html" target="_blank">
  Twitter
 </a>
 ）进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清网.html" target="_blank">
  清网
 </a>
 。中国各地大批人权活动人士被当地警方约谈，重庆的刘继春、厦门的潘细佃等人已遭逮捕和扣押。
 <br/>
 <br/>
 11月16日，维权网引述大陆多位人权活动人士披露，在近几年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微博.html" target="_blank">
  微博
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微信.html" target="_blank">
  微信
 </a>
 大规模封禁异议和人权活动人士帐户之后，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部门.html" target="_blank">
  公安部门
 </a>
 将触角伸向海外社交媒体。
 <br/>
 <br/>
 近日，北京、重庆、山东、湖北、福建和广东等多地的公安部门，展开对海外社交媒体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Twitter.html" target="_blank">
  Twitter
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清网.html" target="_blank">
  清网
 </a>
 行动。来自重庆的刘继春及福建厦门的潘细佃等异见人士，因拒绝按当局要求删文，已被公安带走。
 <br/>
 <br/>
 而被强行删除的Twitter账号，包括2015年已入狱的异见人士吴淦，其网名是〝超级低俗屠夫〞，他的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推特.html" target="_blank">
  推特
 </a>
 账号@tufuwugan，和他在推特上2015之前所发布的全部内容，已于11月9日全部被删除。
 <br/>
 <br/>
 而拒绝配合中共当局删除推特内容的人权活动人士中，重庆刘继春已被逮捕，居住福建厦门市的潘细佃，已被公安局扣押。在这场清网行动中受冲击的人数，目前还在不断增多。
 <br/>
 <br/>
 海外政论杂志《北京之春》消息称，推特内容被中共当局删除的人士包括，学者高瑜、陈永苗、记者文涛、律师覃永沛、网路观察员古河、刘强本、维权人士何德普等人。
 <br/>
 <br/>
 消息说，中共的推特清网能如此精确地锁定推友账号和密码，显示了当局对美国社交媒体，尤其是对推特的渗透和控制又升了一级。
 <br/>
 <br/>
 有消息人士说，这一波针对境外媒体的大规模清网行动，与中共公安部门高层人事变动有关。近日原厦门市公安局长林锐，获擢升任公安部主管网安的副部长。
 <br/>
 <br/>
 林锐毕业于中国人民警官大学的计算机专业，长期从事网路安全工作，福建〝三网友案〞就是他制造的以言制罪的一个案件。
 <br/>
 <br/>
 刚调任中共公安部4个月的林锐，于11月7日被任命为中共公安副部长。
 <br/>
 <br/>
 在林锐出任公安副部长之前，中共就展开互联网自媒体的整治行动，11月12日，网信办官网称，自10月20日针对自媒体展开专案整治行动以来，已全网处置（查封）9,800多个自媒体账号，并对微信、微博等平台提出严重警告。
 <br/>
 <br/>
 网信办官网11月16日再发消息称，约谈了10家客户端自媒体，包括：百度、腾讯、新浪、今日头条、搜狐、网易、UC头条、一点资讯、凤凰、知乎，网信办要求平台企业切实履行主体责任，按照全网一个标准，全面自查自纠。
 <br/>
 <br/>
 分析认为，当局的用意显然是抢占舆论高地，让中国民众噤声，以延续其苟延残喘的政权。
 <br/>
 <br/>
 早在2015年，中共军报就曾刊文称，一个国家的倾覆，始于思想的瓦解。报导称，突尼斯、利比亚和埃及等国家政权，都是被〝推特〞推倒，一夜之间垮台的值得深思！
 <br/>
 <br/>
 军报还称，〝网路时代的敌人，手中拿着的可不仅是刀枪剑戟，还有滑鼠键盘！〞并指自媒体网路言论从根本上动摇中共的执政地位云云。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
</div>
