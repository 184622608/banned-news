### 中共间谍窃密细节被公开 对话信息曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】今年4月，美国首开先例，在比利时诱捕涉嫌窃取美国商业机密的中共情报官员，并引渡回美国。日前美方公开了其诉状，曝光了该官员的犯案过程，包括他与其美国〝合作者〞的详细对话。中共外交部在沉寂一天后表态称美方〝凭空捏造〞。
 <br/>
 <br/>
 10月10日公布的美国联邦起诉书指，嫌犯名为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐燕军.html" target="_blank">
  徐燕军
 </a>
 (Yanjun Xu音译，或译作
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐彦钧.html" target="_blank">
  徐彦钧
 </a>
 )，是中共江苏省国家安全厅六局副主任，主要职责是获取技术信息，以及欧美航空航天公司的商业机密。
 <br/>
 <br/>
 徐燕军在进行对外经济间谍活动中，使用假名(Qu Hui、Zhang Hui，音译)和假职位（江苏科技促进协会成员等）来掩盖其真实身份。
 <br/>
 <br/>
 根据起诉书，徐燕军从2013年12月起开始与中共国安部、中国高校和相关机构的多名成员密切合作，对美国等多个国家的重要航空航天公司展开调查，选择潜在的〝合作对象〞。
 <br/>
 <br/>
 与徐燕军合作的高校中，包括南京航空航天大学。该大学由中共工业和信息化部创建，与中国商用飞机公司及中国航空工业集团有合作关系。
 <br/>
 <br/>
 起诉书显示，美国通用电气航空(GE Aviation)成为徐燕军的目标之一。该公司一名现已退休的华裔工程师，从2017年3月起与南京航空航天大学的一名副主任开始通过电邮频繁联系。
 <br/>
 <br/>
 同年5月，在徐燕军的协助下，这位工程师被邀请到南京航空航天大学举办讲座。过程中，徐开始诱导工程师，将讲座内容集中在其公司最显著的研发设计和技术上。同时，徐燕军冒用该副主任的名字，给工程师发送电子邮件。
 <br/>
 <br/>
 在讲座过程中，徐燕军与工程师见面时，自称名叫Qu Hui，在江苏省对外科学技术促进协会工作。
 <br/>
 <br/>
 这次讲座，徐燕军除了替工程师支付行程费用外，还支付3500美元酬劳。行程结束时，校方邀请该工程师次年再来办讲座。
 <br/>
 <br/>
 2018年1月23日，徐燕军用假名联系该工程师，希望他能带来有关通用公司系统编程、设计方面的信息，并提供邮件地址，希望对方将相关文件发送至该地址。当工程师提出，用公司电脑发送可能无法发送时，徐回复：〝用那台电脑可能会不太合适，对吗？〞
 <br/>
 <br/>
 2月3日，该工程师向徐燕军发送了一份包含公司机密信息的摘录性文件，并问徐这份文件是否有他想要的信息。
 <br/>
 <br/>
 此外，徐燕军还向工程师发送了一系列有关飞机风机叶片制造材料的问题，工程师对其中一些问题给予了直接回复，其中也涉及商业机密。
 <br/>
 <br/>
 期间，徐燕军建议两人见面详谈，并询问能否在欧洲见面。徐燕军并要求对方将其公司电脑中的文件目录复制给他，还直接告诉对方应如何操作。
 <br/>
 <br/>
 2月28日，在徐燕军坚持下，两人进行了通话。徐燕军表示对工程师发送的文件信息非常满意，并询问对方，这些文件能否从公司电脑上直接导出，能否在欧洲开会时带上这些文件，以及届时自己能否检查这些文件，工程师都表示可以。
 <br/>
 <br/>
 随后，徐燕军还问工程师：〝如果我们之后需要一些新东西，我们见面时可以谈一谈，你觉得怎么样？不需要着急，如果我们决定一起做事情，这不会是最后一次，对吧？〞
 <br/>
 <br/>
 3月底，徐燕军到欧洲参加会议，准备与该工程师会面。
 <br/>
 <br/>
 不过在此之前，通用电气航空已经在配合美国联邦调查局（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 ）的调查，这一切都在FBI的监视中。
 <br/>
 <br/>
 今年4月1日，美方决定收网，在比利时将徐燕军逮捕。
 <br/>
 <br/>
 10月9日，徐燕军从比利时引渡抵美。这是美国首次将中共情报官员自第三国引渡到美国。
 <br/>
 <br/>
 FBI助理局长普利斯塔普（Bill Priestap）称，这是〝史无前例〞的引渡中共情报官员的行动，暴露出中共政府直接监督对美商业间谍活动。
 <br/>
 <br/>
 美方公布徐燕军间谍案后，中共对此保持了长达一天的沉默。日前外交部发言人出面表态称，这是美国〝凭空捏造〞，并称〝希望美方能依法处理此事〞。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
