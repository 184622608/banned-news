### 西安面包车与水泥罐车相撞 致12死伤（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】11月13日晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安.html" target="_blank">
  西安
 </a>
 港务区纺渭路龙王庙段一辆超载
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_面包车.html" target="_blank">
  面包车
 </a>
 与一辆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_搅拌车.html" target="_blank">
  搅拌车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_相撞.html" target="_blank">
  相撞
 </a>
 ，导致10人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_死亡.html" target="_blank">
  死亡
 </a>
 2人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重伤.html" target="_blank">
  重伤
 </a>
 ，面包车头部严重变形。
 <br/>
 <br/>
 陕视新闻14日消息称，事发于13日20时41分，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安.html" target="_blank">
  西安
 </a>
 市灞桥区纺渭路新农村附近，一辆水泥罐车与一辆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_面包车.html" target="_blank">
  面包车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_相撞.html" target="_blank">
  相撞
 </a>
 ，造成9人当场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_死亡.html" target="_blank">
  死亡
 </a>
 ，1人经抢救无效死亡，2名危
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重伤.html" target="_blank">
  重伤
 </a>
 者正在救治。水泥罐车司机已被控制。
 <br/>
 <br/>
 华商报13日称：〝今晚，一辆载有12人的陕A西安牌照五菱面包车，由南向北行驶至纺渭路窑村附近时遇路面大坑，紧急避让间与由北向南行驶的一辆搅拌罐车迎面相撞，发生事故，目前导致9人死亡3人重伤(生死不明)〞。
 <br/>
 <br/>
 初步了解：该路段无路灯，常行渣土车。路面大坑为渣土车碾压形成。事发当晚20时，面包车超载，车上12人都是附近工地的工人。
 <br/>
 <br/>
 有目击者称，涉事的陕A牌五菱面包车，核载7人，实载12人，从南向北行驶，而水泥罐车自北向南而来，结果迎面撞在一起。
 <br/>
 <br/>
 现场视频显示，面包车头部严重变形，水泥罐车停在一旁。有记者走访发现，事发地附近路段，虽已安装路灯，但均未亮。
 <br/>
 <br/>
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：戴明）
