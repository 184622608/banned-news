### 比贸战更可怕 白宫拟将中共列入货币操控国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】美国财政部一名高级官员日前透露，白宫密切关注人民币近期的贬值情况，正在考虑将中共正式列为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货币操纵国.html" target="_blank">
  货币操纵国
 </a>
 ，财政部下周将发表报告作出定案。有分析认为，一旦被美国列入〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率操纵国.html" target="_blank">
  汇率操纵国
 </a>
 〞，中共将面临严厉制裁，后果比
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 更为严重。
 <br/>
 <br/>
 据彭博社报导，10月8日，美国财政部一名官员透露，白宫正在向财长姆钦施压，要求将中共正式列入货币操控国。对此财政部正在考虑之中。
 <br/>
 <br/>
 美国财政部正密切关注人民币的走势，对人民币近期贬值感到担忧。这位官员在电话会议上对记者说，〝更广泛来说，我们对中国（共）放弃更加以市场为导向的政策，且继续依赖影响宏观经济和贸易环境的非市场机制十分关切。〞
 <br/>
 <br/>
 这位官员表示，这些担忧将在财政部定于下周发布的半年度货币报告中被提出来。但他拒绝透露报告的任何细节。
 <br/>
 <br/>
 数据显示，近6个月以来人民币兑美元重挫了9%，成为亚洲表现最差货币之一，引发外界对中共刻意令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币贬值.html" target="_blank">
  人民币贬值
 </a>
 的揣测。
 <br/>
 <br/>
 10月7日，中共央行宣布年内第四次降准，下调部分金融机构人民币存款准备金率1个百分点。降准被业界人士认为会加大人民币贬值压力。8日，人民币在岸和离岸兑美元双双跌破6.93关口。
 <br/>
 <br/>
 今年7月20日，姆钦就曾表示，在财政部关于汇率操纵半年度报告中，将审查人民币疲软问题，该报告将于10月15日发布。
 <br/>
 <br/>
 上述官员还表示，姆钦本周将赴印尼峇里岛，出席世界银行和国际货币基金组织年会。目前努钦没有与中共官员举行会晤的安排。不过，他的日程尚未最后敲定。
 <br/>
 <br/>
 有分析认为，一旦被美国列入〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率操纵国.html" target="_blank">
  汇率操纵国
 </a>
 〞正式名单，中共可能面临制裁，其程度或远远超过对商品加征关税的影响。
 <br/>
 <br/>
 事实上，美国总统川普（特朗普）一直指控中共操控人民币汇率，以逃避美国的关税制裁。川普7月份曾对美财经电视台CNBC说过，人民币汇率〝像石头一样下坠〞。
 <br/>
 <br/>
 他强调：〝中共削减人民币值，是为了弥补贸易额下降，这对我们没好处，不能让他们这么做下去，他们这么做，根本不是在平等竞争。〞
 <br/>
 <br/>
 川普还表示：北京一定在操控人民币，〝一定存在某个公式，我已派人彻查到底找出这个公式。〞
 <br/>
 <br/>
 川普早在竞选时就曾多次批评中共操纵人民币汇率，以获取贸易上的利益。随着目前美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 不断升级，有专家认为，川普政府很可能将中共列为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货币操纵国.html" target="_blank">
  货币操纵国
 </a>
 ，并采取惩罚措施。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
