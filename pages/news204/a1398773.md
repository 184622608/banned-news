  刘强东
 </a>

 </a>
 突然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离世.html" target="_blank">
  离世
 </a>
 。但目前为止，该消息尚未得到京东方面及刘强东家人的证实。
 <br/>
 <br/>
 11月10日，有网友在推特称，身陷漩涡的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 再遭重大打击，据悉年仅40岁现为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东系.html" target="_blank">
  京东系
 </a>
 〞公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股东.html" target="_blank">
  股东
 </a>
 ，亲妹
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强茹.html" target="_blank">
  刘强茹
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离世.html" target="_blank">
  离世
 </a>
 ，是医疗事故还是谋杀？为什么第二天就要火化？
 <br/>
 <br/>
 也有网友称，〝刘强东唯一的亲妹妹刘强茹是因生育难产去世。〞还有网友进一步证实，刘强茹是在南京东南大学附属中大医院，因羊水栓塞，在分娩过程中过世，年仅40岁。
 <br/>
 <br/>
 有网友提及，羊水栓塞的几率极低，而且第二天要火化，这种安排也非常蹊跷﻿。

 <br/>
 不过上述消息仅在社交网络上流传，尚未得到京东方面及刘强东家人的证实。至于刘强茹的实际年龄，也有不同的说法。
 <br/>
 <br/>
 据财经头条报导，刘强东的妹妹刘强茹今年42岁，比刘强东小2岁，在综艺节目中有过露脸。
 <br/>
 <br/>
 从刘强茹生前被媒体曝光的近照看，她十分低调朴实，穿衣打扮也很简朴。据报，刘强茹曾任一所中学的英语老师，发表过论文，从1998年毕业后，她开始做英语教师，平常非常低调，2013年，她辞去了教师的工作，在宿迁京东客服中心开起了超市。
 <br/>
 <br/>
 据说刘强茹是刘强东唯一的妹妹，近几年，刘强茹已经是刘强东旗下一家公司的股东。
 <br/>
 <br/>
 自8月底刘强东在美国涉嫌强奸案而被逮捕丑闻发酵后，京东集团的股价急转直下，持股机构数量也暴跌。刘强东的地位也正在动摇。
 <br/>
 <br/>
 在过去60多天里，刘强东除了案发初期现身京东集团〝正常开展工作〞外，几乎所有重要场合都缺席。
 <br/>
 <br/>
 11月5日在上海开幕的国际进口博览会期间，大陆企业界名人几乎全部到场，但未见到刘强东的身影。
 <br/>
 <br/>
 11月7日至9日，在浙江省乌镇举行的世界互联网大会上，也没有看到刘强东的身影。
 <br/>
 <br/>
 11月1日，习近平主持召开民营企业座谈会，安抚所有民营企业和民营企业家，并称他们都是〝自己人〞。刘仍缺席活动，被认为被踢出了习近平〝自己人〞圈子。
 <br/>
 <br/>
 除此之外，10月24日，中共官方发布《改革开放40年百名杰出民营企业家名单》，名单中也没有刘强东。
 <br/>
 <br/>
 此前在上海举行的2018世界人工智慧大会等主要活动，刘强东均缺席。
 <br/>
 <br/>
 海外舆论分析，刘强东因在美涉性侵案丑闻发酵，遭北京〝抛弃〞了。
 <br/>
 <br/>
 据美国警方9月4日公布的报告，刘强东一案属于〝一级性犯罪：强奸既遂〞（CSCR-1,CSC-RAPE-COMPLETE）。明尼苏达亨内平县警方9月21日宣布，已完成对案件的初步调查，接下来将由检方接手，审查案件全部证据，并最终决定是否起诉。
 <br/>
 <br/>
 根据当地法律，刘强东所涉的一级强暴罪被定罪后，刑期至少12年起。
 <br/>
 <br/>
