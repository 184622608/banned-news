### 孟晚舟有7本护照 曾是加拿大永久居民
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】随着温哥华保释听证会的检辩交锋，华为副董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 拥有多本护照、曾是加拿大永久居民的多重身分，也随之曝光。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 于12月1日在加拿大转机时，被加方逮捕，美国要求将其引渡至美国受审。
 <br/>
 <br/>
 当地时间12月7日，加拿大法院举行听证会，对孟晚舟举行了近6个小时的聆讯，针对孟的保释请求，法官当天并没有立即做出决定，听证会将延至下周一（9日）再度开庭。
 <br/>
 <br/>
 有消息称，孟晚舟为了获得保释，向法庭提出12项条件，包括愿戴电子脚镣、上交护照、同意皇家骑警随时搜查等，多项严格的限制人身自由的措施。
 <br/>
 <br/>
 但加国检方拒绝孟晚舟的保释请求，认为她有可能弃保潜逃。并要求法院尊重美国方面的要求，将孟引渡至美国。
 <br/>
 <br/>
 根据加拿大广播公司（CBC）取得的美国司法部证词，美方将孟晚舟描述成一位拥有庞大财力、试图逃避美方通缉的女性；还说她拥有〝来自包括中国和香港在内，多达7本的护照〞。
 <br/>
 <br/>
 美方证词指出，2017年4月左右，孟晚舟和其他华为高级主管意识到美国对华为展开刑事调查，于是开始改变旅行模式，不再前往美国，即使孟晚舟的儿子在波士顿上学，她也没去过。从中显示孟晚舟〝做贼心虚〞。
 <br/>
 <br/>
 检方在法庭上说明，孟晚舟参与计谋，让不知情的美国金融机构与伊朗交易，违反美国制裁伊朗的法律。
 <br/>
 <br/>
 加拿大当局是应美国纽约东区法院要求，逮捕孟晚舟。美方法院并要求将孟晚舟引渡到纽约受审。若在美国遭到定罪，孟晚舟面临最高30年以上刑期。
 <br/>
 <br/>
