### 崔永元深夜点名要爆料 枪口转向农业部
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 20日深夜在微博自曝，称掌握
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共农业部.html" target="_blank">
  中共农业部
 </a>
 大量不为人知的资料。由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 问题，崔永元曾多次点名农业部。有分析认为，从崔永元爆料〝阴阳合同〞对中国娱乐圈造成的冲击来看，他在转基因战场上，可能又会掀起一场风暴。
 <br/>
 <br/>
 北京时间11月20日晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 在〝小崔读书汇〞发文称，〝我已掌握农业部食堂、部长餐厅从来不吃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 只吃特供的详细资料。正在整理中。〞
 <br/>
 <br/>
 崔永元此番话有些耳熟，他此前抛出〝阴阳合同〞揭开娱乐圈偷逃税黑幕时，也有此一说。不过，这次崔永元的枪口对准的是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共农业部.html" target="_blank">
  中共农业部
 </a>
 。
 <br/>
 <br/>
 此前，由于转基因问题，崔永元曾多次炮轰中共农业部。10月25日，崔永元在微博讽刺中共农业部部长韩长赋。原因是，崔永元指出中国土地上发现了滥种的转基因作物，而韩长赋却回应说：〝我们一查，它们在国外都有安全证书。〞
 <br/>
 <br/>
 当天，崔永元在微博转发了一篇〝加拿大娱乐用大麻合法化〞的文章，他嘲讽说，以后若发现（中国）种植大麻，韩长赋依然可以说：〝我们一查，它们在加拿大和乌拉圭都有种植许可……〞
 <br/>
 <br/>
 11月8日，崔永元曾炮轰中共农业部关于转基因食品的研究结论造假。他在微博中写道：〝农业部说中国转基因安全最爱说自己是全世界管理最严，有专门的安全委员会。转基因中心造假、负责人李奎是转基因安委会成员，两件事说明农业部一直放屁！〞
 <br/>
 <br/>
 11月10日，崔永元披露中共农业部特供内幕，指农业部的特供基地在新疆吐鲁番，全是有机的。另有大鱼、大虾、各种新鲜土特产等，鸡蛋也是特供的。现在国家管得严，不敢了。顺便翻脸不承认以前有过。
 <br/>
 <br/>
 崔永元转发网友评论称，〝农业部把转基因吹鼓上了天，请国家把农业部的特供取消，立法让农业部必须吃转基因。〞
 <br/>
 <br/>
 崔永元一直关注转基因食品的安全问题，2013年，他自费赴美考察、拍摄关于转基因的记录片，片中引用了多位美国民众、专家的意见和经历，说明〝转基因食品对身体有害〞。
 <br/>
 <br/>
 该片在中国大陆引起轩然大波，崔永元的结论显然和一些力挺转基因的中共学者观点大相迳庭。2013年9月，崔永元和中共大五毛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_方舟子.html" target="_blank">
  方舟子
 </a>
 就转基因问题在互联网上展开辩论。
 <br/>
 <br/>
 今年10月26日，崔永元在接受陆媒采访时说：方舟子必须死。为什么？就是因为他这个网路集团在网路使用暴力，包括他推广有毒有害的转基因，多少条人命已经在里面了，往最少的说也得有一百条吧？
 <br/>
 <br/>
 方舟子本名叫方是民，早年因参与江泽民集团对法轮功的迫害而出名发家。他曾经攻击诺奖得主屠呦呦、学者于建嵘、IT人李开复、作家韩寒、作家李承鹏、呼吸道疾病专家钟南山等众多公众人物，以博取网络关注。
 <br/>
 <br/>
 今年7月26日，中国毒疫苗丑闻引发全民激愤，崔永元在微博怒斥方舟子为中共辩解〝洗地〞，称方舟子每年收受中共科协2000万人民币洗地费，什么都敢洗。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
