### 德法联合颁发〝人权奖〞陆维权律师余文生获殊荣
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】日前，大陆维权律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">
  余文生
 </a>
 荣获德法两国政府联合颁发的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权奖.html" target="_blank">
  人权奖
 </a>
 〞。不过，余文生目前身陷囹圄，被中共以〝煽动颠覆国家政权罪〞羁押10个多月。外界相信，获奖对他现在的处境会有正面影响。
 <br/>
 <br/>
 据自由亚洲报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">
  余文生
 </a>
 的妻子许艳获悉丈夫获奖的消息后，非常激动。
 <br/>
 <br/>
 许艳：〝非常的激动，非常的感谢。感谢德国法国关注余文生这个案件。余文生律师因为代理一些人权案件而失去自由。我认为他能得这个奖也是大家对余文生的一种鼓励。〞
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权奖.html" target="_blank">
  人权奖
 </a>
 〞旨在表扬及支持在全球范围内保护及推动人权工作的杰出人士，余文生是今年15名得奖者之一。2016年，〝709事件〞维权律师李和平的妻子王峭岭也获得该奖。
 <br/>
 <br/>
 余文生律师，现年51岁，曾代理多起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员辩护案、〝709事件〞被抓维权律师案。今年1月15日，他被当局注销律师证。2天后，他在推特发表〝关于修宪的公民建议〞，随后被当局以涉嫌〝煽动颠覆国家政权罪〞及〝妨害公务罪〞抓捕。
 <br/>
 <br/>
 他被关押在徐州看守所已超过10个月，他的妻子许艳与律师多次到看守所和检察院要求会见和阅卷，都遭到拒绝。
 <br/>
 <br/>
 本周一（19日），江苏徐州市检察院把余文生案件退回当地公安局补充侦查，许艳至今没有收到通知。
 <br/>
 <br/>
 中国维权律师关注组总干事陈悦认为，余文生获奖实至名归。〝得到这个奖对他来说是一个很大的认可。〞
 <br/>
 <br/>
 他相信，获奖会有助于改善余文生目前的处境。〝他已被羁押超过10个月。我们对他的人身安全越来越担心，就是有没有受到酷刑对待。此时获奖可以让国际社会更关注他的情况，有助他尽快获释。〞
 <br/>
 <br/>
 德国政府对余文生事件一直表示密切关注，德国总理默克尔今年5月访华期间曾经和许艳会面，就人权问题交换意见。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
</div>
