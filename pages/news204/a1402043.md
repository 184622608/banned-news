### 李克强签发文件 透露中国就业困境
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】中共国务院总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 12月5日签署新出台的文件。其中，对不裁员或少裁员的参保企业，政府将返还上年度缴纳失业保险费的50%。有分析认为，新文件的着重点在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业问题.html" target="_blank">
  就业问题
 </a>
 上，更凸显中国就业形势严峻、经济正面临困境的真实处境。
 <br/>
 <br/>
 在川习会前夕，11月30日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 在南京召开了一次经济形势座谈会，与部分省政府代表，商讨如何面对中国经济下行、国内外遭遇各种挑战的问题。
 <br/>
 <br/>
 12月5日，李克强签署政府出台的新政，要求地方政府做好促进就业的工作。其中一项规定对不裁员或少裁员的参保企业，可返还上年度缴纳失业保险费的50%。有分析认为，政府对企业裁员与否出台奖励政策，显示经济衰退引发的裁员潮，可能会导致的社会动荡，已让当局感到焦虑。
 <br/>
 <br/>
 据中共官方报导，李克强在经济会谈中说，当前国内外形势错综复杂，中国经济下滑压力加大，各级政府〝对困难和矛盾要有充分估计〞。他还强调，各级政府要把〝就业优先〞放在突出位置。
 <br/>
 <br/>
 近来，由于受贸易战影响，中共经济遭受重创，经济增速放缓至10年来最低，各地掀起企业〝大裁员〞浪潮，就业市场也进入寒冬。
 <br/>
 <br/>
 大陆微信公众号〝智谷趋势〞日前发文称：2018，没什么比活下去更重要，饭碗才是硬道理，对个人如此，对国家来说也是这样。
 <br/>
 <br/>
 文章开门见山说：突然之间，〝大裁员〞的新闻满天飞。BAT、华为、京东等网际网路巨头相继卷入这场裁员漩涡中。沿海省份的出口代工厂纷纷关门放假，富士康被传裁员34万，金融、地产都在裁员。
 <br/>
 <br/>
 前程无忧平台显示，在6个月的时间里，202万条招聘广告消失了，招聘广告从285万骤减至83万。另外，中共农业农村部官员11月8日声称，有740万人〝返乡创业〞。但外界认为，这些其实都是失业的农民工，被迫返乡。
 <br/>
 <br/>
 今年中国大陆还有超过820万名大学毕业生，是史上新高，就业形势更加严峻。还有专家称，明年才是真正的就业寒冬，还将有超过830万大学毕业生将面临就业难题。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业问题.html" target="_blank">
  就业问题
 </a>
 上，中国民营企业一直有很大贡献。不过，贸易战对中国经济的冲击，也导致中国民营经济滑入低谷。尤其是大陆近期出现〝私营经济离场论〞、〝民企帮助国企改革的历史使命已经完成〞等说法，更使民营企业倍感焦虑，对经济前景失去信心。
 <br/>
 <br/>
 因此，李克强在会谈中还强调，各地政府要支持制造业、服务业，尤其是小微企业、民营企业等实体经济，实施更大力度减税降费。
 <br/>
 <br/>
 事实上，一直以来，中共对民营企业都是施行各种盘剥、打压政策，民营企业高税负、融资难是长期存在的问题。
 <br/>
 <br/>
 有〝玻璃大王〞之称的福耀玻璃创始人、董事长曹德旺2016年对媒体说，中国企业税负比美国高35%。他说：〝在美国投资100万，企业可以赚60万，而在中国只能拿到42万，税是58万。〞
 <br/>
 <br/>
 有评论指，中国民企现在的困境，都是中共体制造成的。民企是市场经济产物，也是中国实体经济生力军，中国民企要彻底脱困，中国经济要往前走，中共不能不去除。
 <br/>
 <br/>
 值得一提的是，12月1日结束的川习会，北京当局首次同意进行结构性改革，这将会动摇中共体制的根基。外界认为，如果北京真的付诸实际行动，中共就将解体。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
