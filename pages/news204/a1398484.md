### 专家：美众议院易主 习近平更麻烦了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 后，民主党重新掌控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 ，将对总统川普（特朗普）执政起到一定掣肘。专家分析认为，这对习近平来说是个坏消息，比起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 ，民主党普遍认同对华应该采取更严厉政策。预料月底〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞更难有成果。
 <br/>
 <br/>
 11月6日美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 投票结果出炉，川普总统所在的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 ，失去对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 的控制权。这将对川普执政起到一定掣肘，外界关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 格局会否有变。
 <br/>
 <br/>
 由于中国国家主席习近平月底将在G20峰会与川普会面，近期中美已互释善意。在中美寻求贸易战解决途径之际，民主党重新掌控众议院，专家普遍认为，民主、共和两党对中国问题立场相近，料不会改变对中共强硬路线，而且民主党恐怕会阻挠川普政府与北京达成贸易协议，月底〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞更难有成果，习近平更麻烦了。
 <br/>
 <br/>
 华府智库美国企业研究所资深研究员史剑道（Derek Scissors）对英媒BBC分析认为，大部分民主党人，比共和党人更支持贸易保护主义，且会抓住每个机会，阻挠川普与中共达成贸易协议。
 <br/>
 <br/>
 班诺克本全球外汇首席市场策略师钱德勒（Marc Chandler）对美媒CNBC分析说，对中共的贸易战将由国会两党一起进行，因为中共不公贸易行为伤害美国工人，影响美国就业已经是共识。
 <br/>
 <br/>
 美国有线电视新闻网在题为〝民主党中期选举得利无法为中国止贸易战之痛〞的报导称，如果北京焦急观望美国中期选举的政客，希冀民主党得利，以此迫使白宫改变对中共策略，给北京喘息的时间，恐怕结果让北京失望了。
 <br/>
 <br/>
 报导也认为，对中共政治立场是民主、共和两党之间难得一见的共识，而且民主党普遍认同美国对中共应该采取更严厉政策，包括方方面面，如军事、贸易、情报和外交手段等。
 <br/>
 <br/>
 《经济学人》智库分析家马罗（Nick Marro）对CNN分析说：从历史上说，民主党是更加接近美国工人的党派，更不推崇对进出口贸易的宽松管制，这对习近平来说是个坏消息，美国政府日渐明显的敌意，已经让习近平政府招架无力。
 <br/>
 <br/>
 报导引述专家分析说：习处理中美关系危机的方式，以及未能摆脱掉川普的多重挑战，遭到中共党内批判，言论所指虽不具名，但习已处于防御状态。
 <br/>
 <br/>
 中共党媒也在社评中承认，对中共强硬是美国两党态度重合率最高的议题之一，民主党在人权问题上更高调。而中共恶劣的人权状况，在国际社会早已臭名远扬。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明)
