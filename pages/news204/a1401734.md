### 四川惊现碎尸案 尸块从天而降 差点砸中清洁工（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四川成都市.html" target="_blank">
  四川成都市
 </a>
 某小区惊现从天而降的人体碎尸，差点砸到扫地的清洁工，现场目击民众被吓坏，连忙报警。警方随后公布，一名身患绝症的老妇人自杀坠亡，身体被撞散。但民众对此说法有质疑，事发小区住户议论纷纷，怀疑是一名年轻女子被人杀死抛尸。
 <br/>
 <br/>
 12月2日上午8点多钟，成都锦城逸景小区发生一起惊悚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碎尸案.html" target="_blank">
  碎尸案
 </a>
 。据社区保安向陆媒介绍，当时一名清洁工正在扫地，突然有物体掉落，差点砸到该清洁工，仔细一看，惊觉是人体尸块，于是赶紧报警。
 <br/>
 <br/>
 这起空中坠尸案也吓坏目击民众，有居民拍下现场画面显示，死者身上穿的衣物散落，手、脚、身体分离，怀疑是先被人杀害后，从高楼丢下。
 <br/>
 <br/>
 当地警方接报后到现场勘察。随后，警方公布消息称，死者是一名65岁老妇人，因久病厌世从34楼跳楼自杀，坠落过程中与硬物碰撞，导致肢体分离。警方排除他杀的可能。
 <br/>
 <br/>
 不过，警方的说法遭到小区民众质疑。据网上传出的一段视频显示，不少目击民众纷纷向视频拍摄者透露当时的情形。有民众说，死者掉落的是一只红鞋，老年人不可能穿这种鞋。而且死者内裤还挂在树上，不象是自己脱的。
 <br/>
 <br/>
 还有民众说，发生坠亡事件的那栋楼，居住的都是些打工的租户。据有一位老婆婆说，坠亡的好像是个年纪约20来岁的女子，和男朋友发生了矛盾。
 <br/>
 <br/>
 不过，以上这些民众的说法，并未得到官方证实。这起惊悚的碎尸案件，仍在民众心中疑问重重。
 <br/>
