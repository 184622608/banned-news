### 川普:中方即将进口美大豆和能源 芬太尼经销商将判死刑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】根据美中首脑近日在阿根廷达成的意向，北京当局或已开始为恢复进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 和液化天然气做准备。对此，中共官方虽然尚未公开回应，但美国总统川普（特朗普）日前的推文间接证实了相关消息，他同时还强调了习近平政府将严厉打击类鸦片止痛剂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芬太尼.html" target="_blank">
  芬太尼
 </a>
 （Fentanyl）经销商的承诺。
 <br/>
 <br/>
 自从12月1日中美双方就缓和贸易冲突达成暂时休战的协议后，外界便开始密切观察北京当局即将采取的后续动作。
 <br/>
 <br/>
 美国彭博社日前引述消息人士爆料称，中共相关机构的官员已经被告知，采取必要的相关措施，为恢复大规模进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 和液化天然气等物资做准备。
 <br/>
 <br/>
 消息人士表示，目前尚不清楚中方会否降低对美国大豆和液化天然气的报复性关税，亦不清楚开始恢复进口的具体时间，但中方有可能向买家返还关税款项，因为他们为帮助国家储备大豆进行了采购。
 <br/>
 <br/>
 路透社也援引消息人士披露，中国石油贸易商〝联合石化〞已计划在2019年3月份前重新进口美国原油。该公司的高层管理人员透露，趁现在油价比较低，储存一些原油当作商业库对公司是有利的。而中国买家将在90日的限期内尽快进口美国原油，这些物资会在3月1日之前运抵中国。
 <br/>
 <br/>
 据了解，在今年夏天
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 开打后，中共政府为报复美国政府而对美国进口产生加征25%关税的第一批清单中就包括美国大豆；从9月其开始对美国液化天然气征收10%的报复性关税。在此期间，中国买家为避免多支付高额关税而纷纷转买其他国家的相关产品，但新的来源（例如巴西大豆）价格同样在上升，而且数量也无法满足中国市场的庞大需求。因此，川习达成临时休战的协议后，舆论认为中方恢复购买包括大豆在内的美国农产品以及液化天然气，其实对美中双方都是有利的。
 <br/>
 <br/>
 美国总统川普自从结束与习近平的会晤返回美国后，一连3天在Twitter上发出了多条有关中美贸易动向的推文，其中最近的一篇短文通过引述彭博社的报导片段，证实了中方已着手重新进口美国大豆和液化天然气的消息。
 <br/>

  <br/>
 </center>
 <br/>
 他在推文中表示，自己与习近平在这次〝漫长而充满希望的历史性会议上〞 讨论了所有议题，自己相信习在这次会谈中所说的〝每一句话〞，相信北京当局一从阿根廷回国就会发出〝强烈的讯号〞。
 <br/>
 <br/>
 川普还提到，习近平在阿根廷的会晤中向自己承诺，要把致命的类鸦片止痛剂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芬太尼.html" target="_blank">
  芬太尼
 </a>
 划入〝受管控物品〞，把向美国出售这种〝对所有人最严重，最危险，最容易上瘾的致命物质〞的行为，定性为刑事犯罪，并对有关的经销商和推动者动用死刑。
 <br/>
 <br/>
