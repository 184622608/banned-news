### 中共升级封锁彭斯演讲 大陆网民全力转发
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】尽管遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 官方在网络上的层层封锁、拦截及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_删贴.html" target="_blank">
  删贴
 </a>
 ，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 周四（4日）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演讲全文.html" target="_blank">
  演讲全文
 </a>
 ，今天（5日）依然成为中国网民大举转发及搜索的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热门文章.html" target="_blank">
  热门文章
 </a>
 。
 <br/>
 <br/>
 〝北京正在使用一种全政府的手段，利用政治、经济、军事工具以及宣传，在美国推进其影响和维护其利益。〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）说。
 <br/>
 <br/>
 彭斯4日在华盛顿保守派智库哈德逊研究所（Hudson Institute）发表的长篇演讲，迅速成为中国网民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热搜.html" target="_blank">
  热搜
 </a>
 的文章。尽管
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 加大了网络封堵和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_删贴.html" target="_blank">
  删贴
 </a>
 ，但网友依然想尽办法转载。有网民把彭斯的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演讲全文.html" target="_blank">
  演讲全文
 </a>
 张贴在自己的微博上供连结，也有网友将全文制作成PDF及WORD等类型档案转发，部份档案能被成功开启及阅读，但有些档案仍无法转发成功。
 <br/>
 <br/>
 看过全文的北京博士生小刘表示，彭斯提到，美国希望中共尊重私人财产、人权、个人自由及宗教自由，但〝希望落空了〞。〝难怪（全文）会被当局封杀，不意外〞。
 <br/>
 <br/>
 较多网民认为，这是美国对中共〝强硬的讯号〞。不少网友大赞彭斯的演讲〝讲的太对了〞、〝干得漂亮，不过力度还不够，继续对共匪升温〞、〝（这是）正义世界正式向邪恶轴心——中共发难！〞
 <br/>
 <br/>
 也有网友感叹〝美国二把手（彭斯）说了大实话啊〞、〝这彭斯气场不输川普（特朗普）啊！〞
 <br/>
 <br/>
 也有细心的网友发现，彭斯的演讲把〝中国人民〞与〝中共政府〞区分对待，并且把中共的官员（包括习近平）与〝中共〞区分开。
 <br/>
 <br/>
 在彭斯的整个演讲中，特别用了18次〝中国共产党〞或〝共产党〞字词，显然有意将〝中国〞与〝中共〞做出区分，这一点不但是川普政府近期格外强调的，也意味着美国绝非将中国人民视为对手或敌人，〝中国共产党〞才是美国的最大威胁，也是美国全面反击的目标。
 <br/>
 <br/>
 此外，彭斯副总统在演讲中还特别提到台湾问题。他强调，〝尽管我们政府将遵守三个联合公报和《台湾关系法》所反映的‘一个中国’政策，但美国始终相信，台湾对民主的拥抱为所有华人展示了一条更好的道路。〞
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
</div>
