### 楼盘精装涉欺诈 武汉数千业主抗议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】日前，湖北
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武汉.html" target="_blank">
  武汉
 </a>
 数千名精装房业主聚集在洪山广场维权，抗议开发商收取高额装修费，装修却存在严重质量问题，要求政府规范楼房精装修市场，监管
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欺诈.html" target="_blank">
  欺诈
 </a>
 问题。
 <br/>
 <br/>
 11月26号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武汉.html" target="_blank">
  武汉
 </a>
 精装房业主对新唐人表示，这次数千业主维权涉及近百个楼盘，因此也叫做〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_百楼联动.html" target="_blank">
  百楼联动
 </a>
 〞。
 <br/>
 <br/>
 武汉绿地城的一名业主说，业主维权主要是因为精装修质量问题。
 <br/>
 <br/>
 武汉绿地城业主 匿名：〝精装房它那个装修每个平方1800（人民币），达不到1800元的标准，肯定有蛮大的水份在里面，主要的问题就在这个收费上面。〞
 <br/>
 <br/>
 让业主们不满的问题包括：精装房的质量与开发商最初的承诺有很大落差；公共面积也按精装修收取费用；业主在交首付时就要交纳装修费用，对业主来说因此承担了高昂的房贷利息。
 <br/>
 <br/>
 武汉房地产经纪人张先生说，目前武汉地区三环以内毛坯房几乎绝种，大小楼盘几乎都是精装修，但装修出来的精装房像〝牛皮鲜〞一样。
 <br/>
 <br/>
 武汉房地产经纪人张先生：〝他的精装跟你想的不一样的，精装肯定是全部到位了，他给你的精装只是把大部分装了，反正把轮廓给你装出来了，比如家居搬进去，反正房地产的套路太深了。〞
 <br/>
 <br/>
 业主纷纷要求开发商退还精装修费，甚至还有人要求退房。
 <br/>
 <br/>
 24号，数千业主在洪山广场抗议时，警方早晨6点开始布控，上午用5辆大巴车拉走一些人，但更多的业主仍在坚守广场，抗议行动一直持续到天黑前，人们慢慢才散去。
 <br/>
 <br/>
 武汉市政府发布的文件显示，当局对这次业主维权已经事先做了维稳安排。文件要求警方对所谓的〝挑头组织者〞迅速落地查人，对静坐、列队行进、喊口号、拉横幅、穿状衣的业主，也要进行处理。
 <br/>
 <br/>
 不过，业主们表示，如果问题得不到解决，可能还会有更多业主站出来维权。
 <br/>
 <br/>
 新唐人记者熊斌、元真采访报导
