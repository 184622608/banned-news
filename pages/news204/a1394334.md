### 菲国总统惊爆：中共大使与毒贩关系密切
------------------------

 【新唐人北京时间2018年10月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾.html" target="_blank">
  菲律宾
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统.html" target="_blank">
  总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜特尔特.html" target="_blank">
  杜特尔特
 </a>
 时常曝出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_惊人话语.html" target="_blank">
  惊人话语
 </a>
 ，他日前在一场演讲中惊爆，在纳卯市一名可疑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人毒枭.html" target="_blank">
  华人毒枭
 </a>
 ，与中共驻菲律宾大使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵鉴华.html" target="_blank">
  赵鉴华
 </a>
 有往来，并指此人去年在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 访菲时，曾鞍前马后陪同。
 <br/>
 <br/>
 据中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菲律宾.html" target="_blank">
  菲律宾
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总统.html" target="_blank">
  总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜特尔特.html" target="_blank">
  杜特尔特
 </a>
 在演讲中表示，中共驻菲大使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵鉴华.html" target="_blank">
  赵鉴华
 </a>
 ，与一名疑似为毒贩的华裔富商关系密切，这名商人不但拥有特权，更与中共政府关系密切。
 <br/>
 <br/>
 根据Rappler报导，杜特尔特在10月4日的一场演讲中直言，这名华裔富商拥有大量土地、特权，成为中共总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 的代表团成员。
 <br/>
 <br/>
 杜特尔特痛批，这名华裔富商疑似涉毒，中共大使不但在他家睡觉，还与他关系密切。
 <br/>
 <br/>
 为了佐证自己的说法，杜特尔特说，自己在担任纳卯市市长时，杨麦克出席纳卯市指挥会议时，曾经询问杜特尔特愿不愿意让〝更多投资者〞进入市内，当时杜特尔特就曾经警告他远离毒品。
 <br/>
 <br/>
 杜特尔特同时也在演讲中强调，他的铁腕扫毒政策将会持续进行，绝对不会纵容任何吸食毒品的华人。
 <br/>
 <br/>
 2016年6月底，71岁杜特尔特在宣誓就职演说时，就曾重申他会致力打击菲律宾的贪污和毒品罪行，恢复法律和秩序，以及重建民众对政府的信心。
 <br/>
 <br/>
 以强硬形象闻名的杜特尔特律师出身，曾在南部大城达沃（Davao）担任市长长达20多年，主张严厉打击犯罪。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：赵云）
