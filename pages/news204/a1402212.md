### 薄熙来设3套死亡方案 王立军〝变性〞逃过追杀
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】中共重庆原市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 与其马仔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 翻脸，导致王
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叛逃美领馆.html" target="_blank">
  叛逃美领馆
 </a>
 ，引发中共高层分崩离析，薄熙来也因此落马。据海外中文媒体披露，薄熙来想要除去王立军，曾为其量身打造三套〝死亡方案〞。王立军被迫〝变性〞，化妆成女人才逃过追杀。
 <br/>
 <br/>
 2012年对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 二人来说，是一个重大的人生转折点。当年2月6日，中共前重庆公安局局长王立军夜逃美领馆，曝光了江派的代言人政法委书记周永康与薄熙来的政变黑幕，薄熙来因此倒台。
 <br/>
 <br/>
 在王立军事件背后，中共掩盖了薄熙来夫妇涉及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活摘器官.html" target="_blank">
  活摘器官
 </a>
 、非法在国际贩卖尸体等罪恶。英国商人海伍德（Neil Heywood）也因卷入薄谷开来在国际贩卖器官、尸体等事件，被其〝杀人灭口〞。
 <br/>
 <br/>
 中共喉舌2012年9月19日的报导称，当年1月，王立军向薄熙来反映其妻薄谷开来有重大杀人嫌疑时，薄熙来大怒，赏了王立军一个耳光，致使两人矛盾彻底激化。
 <br/>
 <br/>
 据海外中文媒体披露，随着薄王矛盾的不断升级，王立军身边的人不断被拘捕，其中两名被致死。王立军担心自己参与并掌握了太多薄、谷夫妇的核心秘密，也会被灭口，为自保，他秘密保留了一些内幕证据。
 <br/>
 <br/>
 当时，薄熙来确已动了杀机，为除去王立军，薄熙来的核心团队（以徐鸣、车克民为主）为〝王立军之死〞设计了三套方案:一是被黑社会报复致死，也就是〝被牺牲〞，最后给他一个英雄称号；二是畏罪自杀；三是因精神抑郁症而自杀。
 <br/>
 <br/>
 第一个方案因王立军高度警觉，制造现场的难度较大，第二个方案的不利之处在于王立军畏罪自杀，对重庆打黑是一大否定；因此，最后准备选择让王立军〝被抑郁自杀〞。在王立军出逃前，重庆医院已经开始着手给王编造〝睡不着、精神紧张〞等病历了。
 <br/>
 <br/>
 而王立军为了逃出薄熙来的谋杀阴谋，想出一招〝变性计〞。2012年2月6日，王立军化妆成女人形象，成功逃过追杀，并从朋友那里借了轿车顺利出逃到美国驻成都领事馆。
 <br/>
 <br/>
 王立军逃入美领馆后，为自保寻求政治庇护，将包括薄熙来谷开来杀死英国人海伍德的内幕资料，相关活摘法轮功学员器官及非法贩卖尸体等黑幕资料，以及周永康、薄熙来企图发动政变等内幕资料，都交给了美国领事馆。
 <br/>
 <br/>
 薄熙来获悉王立军出逃后，曾调动大批武警，试图强行攻入美领馆抢人。当时美领馆的安保人员已荷枪实弹警戒。最后，时任中共总书记胡锦涛电话命令薄熙来撤军，王立军在美领馆滞留一天后被当局带走。王立军事件被认为是导致薄熙来下台、薄谷开来被起诉，中共高层分崩离析的导火线。
 <br/>
 <br/>
 2012年9月17日和18日，王立军案在成都中级法院分别秘密和公开审理，并在24日宣判，王立军被以〝叛逃、滥用职权、受贿罪〞判处有期徒刑15年。然而中国问题专家横河认为，当局避重就轻，对王立军犯下的反人类等重罪并未提及。
 <br/>
 <br/>
 横河指出，王立军除了涉及中共当局所指的罪行外，还涉及三种重罪。首先，王立军涉嫌活体盗取人体器官。王立军当时在锦州开了一个现场心理研究中心，做了几千起器官移植，跟活摘法轮功学员器官有关系。其次，王立军在重庆打黑时，涉嫌酷刑逼供、甚至酷刑逼供致死的刑事案件；最后，有证人出来说，他建了一个女子特警队，涉嫌强奸女警官。
 <br/>
 <br/>
 重庆网民方洪（又名方竹笋）曾向外界透露，王立军到重庆出任公安局长后，成立女子交巡警平台，当中有13名研究生女警，王立军强奸了8个，方洪认识一名受害者父亲，可以证实此事。
 <br/>
 <br/>
