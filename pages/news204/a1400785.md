### 金正恩访华前留下一句话 透露对中共的真实看法
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】今年，朝鲜最高领导人金正恩已三次访华，中共与朝鲜的关系在长期冰冻之后看似已经改善。但25日有韩媒透露，朝鲜对中共积怨难解。金正恩首次访华时曾说，即便中共给朝鲜留下的伤口随着时间的流逝能抹平，但那份记忆是不会忘记的。
 <br/>
 <br/>
 韩国《中央日报》25日引述韩国成均馆大学政治外交学教授兼成均馆中国研究所所长李熙玉的社论说，金正恩三次访华后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 强调，中共致力于巩固发展
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中朝关系.html" target="_blank">
  中朝关系
 </a>
 的坚定立场不会变，中国人民对朝鲜人民的友好情谊不会变等。
 <br/>
 <br/>
 即便如此，朝鲜对中共根深蒂固的不信任也不能完全解除。从历史上来看，92年中韩建交后，朝鲜陷于一种孤立无援的地步，给中朝关系留下〝疙瘩〞。
 <br/>
 <br/>
 社评引述朝鲜一位学者的话透露，金正恩第一次访问中国前曾说：〝即便中国（中共）给朝鲜留下的伤口随着时间的流逝能抹平，但那份记忆是不会忘记的〞。
 <br/>
 <br/>
 金正恩还曾表示，〝虽然现在中国是与我们最近的国家，但以后会成为我们最应该提防戒备的国家。从历史上来看，让我们最辛苦的国家就是中国（中共）〞。
 <br/>
 <br/>
 社评说，金正恩的先辈也一直在提防中共。2000年，朝韩首脑会晤时，金正日跟金大中说：〝金日成主席说中国怀有二心，让我一直小心谨慎。在我看来中国何止是有二心，中国应该是有十个心，日后即便半岛统一，驻韩美军也不能撤走〞。
 <br/>
 <br/>
 过去的数十年间，中共一直是朝鲜最大的金主，一提到〝血盟〞、〝唇亡齿寒〞等词，就能让人想起这是中朝关系的形容词，而中共对于中朝关系一直都是这样的认知。
 <br/>
 <br/>
 但在3月中共两会期间，朝鲜给习近平送去的贺电中，已去掉了过去20年中朝两国常用的〝用鲜血凝成的〞、〝传统的〞、〝亲善合作〞等辞汇。
 <br/>
 <br/>
 而朝鲜对于中共评价也没有想像中的那么积极与正面。7月份，由韩国〝中国研究会〞举办的〝中朝关系〞演讲会上，朝鲜前驻英公使太勇浩在形容朝鲜人对于中国的评价时，概括为〝不可信〞及〝野蛮〞两词。
 <br/>
 <br/>
 他说：〝朝鲜人对中国的认知大致分为两类，一类是朝鲜精英阶层，他们普遍认为，〝中国不可信〞；另一种是朝鲜大众阶层，他们都认为中国是一个在文化上比朝鲜还落后的国家。〞
 <br/>
 <br/>
 太勇浩表示，朝鲜人对中国的〝不信任〞根深蒂固，主要源于朝鲜〝金日成革命历史〞教育，其中一半的内容是金日成的抗日斗争，另一半则是金日成如何与中共〝斗争〞，这导致朝鲜大众对中国很难有非常正面及积极的评价。
 <br/>
 <br/>
 前往朝鲜平壤为统一篮球大会取材的韩国记者也透露，朝鲜相关人士说，朝鲜有孩子的家庭，一般都不给孩子们吃中国制造的食品〞。
 <br/>
 <br/>
 朝鲜还曾一度称〝中国是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千年宿敌.html" target="_blank">
  千年宿敌
 </a>
 〞，而习近平甚至用〝最贬损的辞汇〞，表达对金正恩的讨厌之情。
 <br/>
 <br/>
 据《自由亚洲》报导，去年12月份，在朝鲜社会主义女性同盟举行的一次会议上，一名来自松坪区的高层主管说，尽管日本是朝鲜的百年敌人，中国是朝鲜千年的敌人。
 <br/>
 <br/>
 2017年9月，朝鲜的内部文件咒骂中共是〝叛徒、帮凶〞，还骂中共当上〝青蛙〞就忘了自己曾经是〝蝌蚪〞的时候，不讲〝良心〞和〝信义〞，并威胁中共将付出惨痛代价。
 <br/>
 <br/>
 虽然金正恩今年已经三次访华，但不少观点认为，金正恩只是利用中共，但他绝不信任中共，金正恩不愿意当中共的〝儿皇帝〞。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿）
