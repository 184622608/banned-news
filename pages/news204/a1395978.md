### 传《大轰炸》〝逃出生天〞 范冰冰戏份不变
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 举报电影《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大轰炸.html" target="_blank">
  大轰炸
 </a>
 》涉嫌洗钱，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 因此被罚款8.8亿人民币，17日该片导演更宣布取消上映。但有美媒称，《大轰炸》将于10月26日在美国上映，范冰冰的戏份不变。
 <br/>
 <br/>
 央视前主持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 接连在微博发文称：电影《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大轰炸.html" target="_blank">
  大轰炸
 </a>
 》牵扯使用不明的资金至少30亿（人民币，下同）以上，仅在拍摄中就使用各种肮脏手段弄走资金17亿。 他还在微博呼吁，抵制即将上映的《大轰炸》。
 <br/>
 <br/>
 对此，电影《大轰炸》导演萧锋10月17日在微博宣布取消上映。他说，〝再多的事实都于事无补，再多的清白都难挡抹黑，是到放下的时候了，只是可怜了养育八年的孩子《大轰炸》！向仍对电影有期待的观众道歉！〞
 <br/>
 <br/>
 新浪娱乐向电影宣传方求证后确认：《大轰炸》不是改档，而是取消了在院线公映的计划。
 <br/>
 <br/>
 美国媒体《Deadline》18日报导说，电影公司Lionsgate打算，10月26日在美国本土上映电影《大轰炸》，而且Lionsgate并不涉及中国发行，只会于有限度的院线放映。
 <br/>
 <br/>
 北美地区的另一家发行商Grindstone也预计，会让该片在美国院线小规模上映。中港台多家媒体转发了这一消息。
 <br/>
 <br/>
 女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 也参演了电影《大轰炸》，由于崔永元此前爆料范冰冰卷入逃税案，她因此〝消失〞4个月后，中共税务部门3日称，范冰冰在《大轰炸》拍摄中，获片酬3,000万元涉嫌偷税漏税，缴纳8.8亿元人民币的罚金可免予刑责。
 <br/>
 <br/>
 此前还传出《大轰炸》已将范冰冰的戏份全部删除，电影海报也将范冰冰除名，希望可以因此〝逃出生天〞，在中国拿到批文上映，但最后事与愿违。
 <br/>
 <br/>
 不过，据外媒推测，美国上映的版本，将会保留范冰冰的戏份。但消息尚未获得证实，因为发行商没有公布进一步的细节。
 <br/>
 <br/>
 电影《大轰炸》是由中国电影股份公司、原画（北京）影业、上影南国影业出品的历史战争片。由施建祥旗下的上海快鹿集团投资，获中共军方鼎力支持拍摄，更一度被官方列为重点电影。
 <br/>
 <br/>
 该电影于2010年开机，施建祥任总制片，斥资4亿，好莱坞巨星米路吉逊担任艺术总监，由布斯韦利士、刘烨及宋承宪主演，谢霆锋、范冰冰、范伟等30多名著名影星也有参与。
 <br/>
 <br/>
 据称该电影除获中共军方支持外，制片商中国电影股份公司更因有官方背景，而被视为后台强硬。故外界视《大轰炸》前景美好。但在电影2015年11月杀青后，传出主要投资人施建祥出事外逃，上映一直被搁置，直到今年当局才宣布8月17日上映。
 <br/>
 <br/>
 8月7日，《大轰炸》在官方微博宣布，原定于8月17日上映的《大轰炸》改档至10月26日，同时，范冰冰的名字也从海报和宣传稿中〝消失〞。但经过多次大动作的修改和处理的《大轰炸》，最后还是被取消了在院线公映的计划。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：戴明）
