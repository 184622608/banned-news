### 【禁闻】11月7日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】下面我们来关注中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 〝中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_律师后俱乐部.html" target="_blank">
  律师后俱乐部
 </a>
 〞被要求拆除招牌
 <br/>
 <br/>
 大陆被吊照律师组建的〝中国律师后俱乐部〞，被当局指称〝涉嫌违反社会组织管理规定〞。11月6号，多名警察闯入办公室，要求拆除招牌。
 <br/>
 <br/>
 2. 多国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻华外交官员.html" target="_blank">
  驻华外交官员
 </a>
 会见
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 <br/>
 <br/>
 英、美、德、加、瑞士等国的驻华外交官员，11月5号在北京与〝709案〞被捕律师王全璋的妻子李文足会面。李文足向他们介绍了王全璋案的情况，并向他们递交了致各国大使的公开信。信中请求各国大使出席王全璋案的庭审。
 <br/>
 <br/>
 3. 福州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鞭炮抓捕案.html" target="_blank">
  鞭炮抓捕案
 </a>
 唐兆星首次见律师
 <br/>
 <br/>
 福建维权人士唐兆星，9月12号因在福州第一看守所门口放鞭炮迎接朋友出狱而被拘留。11月5号，他与律师第一次在福州第二看守所会见，据律师通报，唐兆星表示自己没有犯法，整个事情完全是警察打击他们维权。
 <br/>
 <br/>
 4. 湖南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尘肺工人.html" target="_blank">
  尘肺工人
 </a>
 深圳维权遭警察殴打
 <br/>
 <br/>
 湖南一些因工作环境污染而患尘肺病的工人和已故工友的家属，11月7号到深圳市政府，要求解决工伤工亡赔偿问题，但遭警察暴力殴打和扔烟雾弹。6人因此病情加重住院，其中一人吐血。
</div>
