### 【禁闻】大陆现基因编辑婴儿 专家：后果严重
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】世界首例免疫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱滋病.html" target="_blank">
  爱滋病
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑.html" target="_blank">
  基因编辑
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_婴儿.html" target="_blank">
  婴儿
 </a>
 在中国诞生，引发轩然大波，各国专家纷纷质疑〝基因编辑〞运用在人体有悖伦理，已经远远超出了技术问题的范畴，后果将是不可预测的严重。
 <br/>
 <br/>
 中国南方科技大学副教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 对外宣布，一对名为露露和娜娜的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑.html" target="_blank">
  基因编辑
 </a>
 （gene editing）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_婴儿.html" target="_blank">
  婴儿
 </a>
 ，已经在中国大陆诞生。
 <br/>
 <br/>
 11月26号，贺建奎接受《美联社》采访时表示，在生育治疗过程中，他改动了7对夫妇孩子的胚胎，其中一位孕妇顺利生产。在参加研究的夫妇中，所有的男性都感染了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱滋病.html" target="_blank">
  爱滋病
 </a>
 毒（HIV），而所有女性并未感染。
 <br/>
 <br/>
 这对双胞胎的一个基因经过修改，使她们出生后即能天然抵抗爱滋病。这次基因手术修改的是CCR5基因，而CCR5基因是HIV病毒入侵机体细胞的主要辅助受体之一。
 <br/>
 <br/>
 基因编辑婴儿引发舆论界轩然大波。《中央社》报导，几位研究人员审查了贺建奎提供给美联社的资料，他们认为，到目前为止，还无法测试证明这一基因编辑工作可以起到完全免疫部分疾病的作用。
 <br/>
 <br/>
 基因编辑疗法存在很大的风险，一旦经过编辑就无法消除编辑可能造成的任何错误。旅美时事评论员唐靖远表示，这种研究行为根本就是拿中国人的人体来做试验。
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝他就是突破了这个禁区，应用在了人体的这个胚胎上面，我觉得，这是第一个必须要说清楚的事情。因为它涉及到非常严重的医学伦理问题、道德问题，还涉及到就是有可能对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人类.html" target="_blank">
  人类
 </a>
 自身的这种基因安全带来重大的威胁。〞
 <br/>
 <br/>
 唐靖远认为，基因编辑将导致人类成为非人类的生物。
 <br/>
 <br/>
 唐靖远：〝人会变成非人，变成掌握权势者可以任意改装的蛋白质产品，而这种产品它将不具备人的一切道德伦理，那样的人类社会就会变成一个像魔鬼一样的世界。〞
 <br/>
 <br/>
 禁止利用基因复制人类是举世共识，目前将基因编辑疗法运用在胚胎上，也是处于严格限制的实验阶段。
 <br/>
 <br/>
 目前任职于美国洛杉矶西达-赛奈医疗中心（Cedars-Sinai Medical Center），专攻食管腺癌基因病变研究的博士后科学家陈力：〝包括现在以前也有其它的团队对这方面的基因研究，但是没有万全的把握，最好不要在人身上展开研究，一般我们都是拿人的细胞在体外养，然后研究看看能不能杀死癌细胞。〞
 <br/>
 <br/>
 目前任职于美国洛杉矶西达-赛奈医疗中心（Cedars-Sinai Medical Center），专攻食管腺癌基因病变研究的生化博士陈力，接受本台记者采访时表示，人类目前的科学研究水平，对人类的基因探索是非常有限的。
 <br/>
 <br/>
 陈力：〝我们知道，那个没了那个基因人类就不会得爱滋病的话，但是我们不知道那个基因的其它功能还有什么，可能这个基因没了以后，就会引发其它的一些麻烦。〞
 <br/>
 <br/>
 有消息指出，这项试验是贺建奎和深圳〝和美妇儿科〞医院的合作项目，并通过医院伦理委员会审批。26号，中共党媒人民网还声称，这是〝划时代成就〞。不过由于这项技术存在伦理争议，人民网随即撤除报导。而深圳市卫生计生委医学伦理专家委员会说，这项试验进行前并没有报备，现在正开会研究此事。
 <br/>
 <br/>
 唐靖远：〝中共这种挑战人类道德底线和文明底线的这种行为，它其实犯了国际社会的众怒，所以在这样一种情况下，我觉得中共可能被迫的转调，马上表态，就是把贺建奎本人当作一个替罪羊一样抛出来，贺建奎的这个研究，他甚至在拿着这个人体直接来做实验，这样的行为，如果没有中共官方背书，没有中共政府在背后支持他，他根本就不可能做出这样的举动。〞
 <br/>
 <br/>
 122名中国科学家26号发表联合声明，强烈谴责所谓的〝伦理审查〞形同虚设，直接就基因编辑进行人体实验，只能用疯狂来形容，对此〝坚决反对〞。
 <br/>
 <br/>
 北京清华大学医学院教授、爱滋病综合研究中心主任张林琦表示，〝这事件已经远远超出了技术问题的范畴，后果将是不可预测的严重。〞
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/周天
