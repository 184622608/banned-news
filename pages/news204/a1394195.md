### 百人联署要求无罪释放高智晟律师
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高智晟.html" target="_blank">
  高智晟
 </a>
 遭中共当局强迫失踪1年多，至今杳无音信、生死不明，全国各地维权人士继发起〝寻找高智晟律师万里行〞的活动后，日前再发出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公开信.html" target="_blank">
  公开信
 </a>
 联署，要求中共当局立即释放高律师，目前已有一百多人参加联署。
 <br/>
 <br/>
 这封题为《强烈要求中共执政当局
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无罪释放.html" target="_blank">
  无罪释放
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高智晟.html" target="_blank">
  高智晟
 </a>
 律师》的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公开信.html" target="_blank">
  公开信
 </a>
 写道：〝释放高智晟律师人心所向。高智晟律师是正义的象征。他维护弱势苦难群体的合法利益，维护法律的正确实施，他没犯法且于国有功。强烈要求当局立即无罪释放高智晟、恢复其自由、让高智晟律师与家人团聚！〞
 <br/>
 <br/>
 这次联署是继9月20号，全国各地维权人士发起〝寻找高智晟律师万里行〞之后，再次进行的公开行动。截至10月5号上午，已有一百多人参加联署。
 <br/>
 <br/>
 上海维权人士周洪宝 :〝老百姓太苦了，改革开放就是个骗局，就是把老百姓的所有辛苦的钱，集中在他们少数的权贵集团家族手里，我房子被抢以后，我现在家破人亡，我可以说我的生活倒退了40年，中国的司法太腐败了，高律师他为正义发声，再没有像高律师这个有脊梁的人，那我们社会就完了。〞
 <br/>
 <br/>
 高智晟律师遭强迫失踪一年多来，当局至今没有任何法律手续和说明，令外界担忧。联署人李燕军表示，高智晟和709案被抓的王全璋，都是不屈服强权的律师。
 <br/>
 <br/>
 广西维权人士李燕军:〝共产党不讲法律，高智晟是无罪的啊，我们肯定要关心啊，他有罪他直接就判，开庭就是了，又不敢开庭。709被逮捕20多个人，我是其中之一，根本就没罪，律师让我认罪，我认罪，过几天马上开庭，不认罪他就不开庭，王全璋要是他认罪了，那就马上就开庭。〞
 <br/>
 <br/>
 高智晟曾被中共司法部选为〝中国十佳律师〞，从2004年12月开始，一年内多次致信中共最高层，要求停止对法轮功群体的非法迫害。2006年开始遭到中共报复，被吊销律师执业证，遭秘密绑架和酷刑折磨后判刑3年，缓刑期间失踪。短暂现身接受外媒采访后被投入新疆沙雅监狱，2014年出狱后被软禁陕北老家，2017年8月被警方带走，至今下落不明。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
</div>
