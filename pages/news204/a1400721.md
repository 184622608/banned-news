### 制造2025带来三大麻烦  美媒：习近平将会感激川普
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】中美两国首脑将在11月底在G20峰会上会晤。有美媒称，两国在《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 》计划上的分歧，将为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞蒙上阴影。而该计划给中国经济带来3大麻烦，习近平现在应该回撤《中国制造2025》，将来说不定还会因此而感激
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 。
 <br/>
 <br/>
 11月30日至12月1日，二十国集团高层峰会（G20）在阿根廷首都布宜诺斯艾利斯举行。中国国家主席习近平与美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）将在峰会期间进行会晤。
 <br/>
 <br/>
 《美国之音》11月24日称，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞可能会因为《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 》问题而破裂，川普希望中国放弃这项野心勃勃的计划，但是习近平是不会同意的。
 <br/>
 <br/>
 《中国制造2025》是中共企图主导未来科技的一项野心计划，是阻碍解决美中贸易紧张局势的绊脚石之一。
 <br/>
 <br/>
 旅美经济学家程晓农说：〝中国制造2025〞是要从西方获得技术，真正的目标是占领西方，特别是美国的市场，换句话讲，击垮美国的制造业，包括一枝独秀的高科技产业。
 <br/>
 <br/>
 《彭博社》22日刊登署名文章称，川普正在帮助中国，因为《中国制造2025》给中国经济带来的麻烦，要超过对世界经济的影响。文章列举了该计划给中国经济带来的几点麻烦。
 <br/>
 <br/>
 1，该计划将大量资金投入一个优先发展的行业，并不一定会引发竞争和创新；2，有中共保护，中国公司在国内有市场，一旦走出国门，就必须与已经成名的产品竞争，但这些公司没有成功的经验；3，该计划已经引发国外政府的警觉，开始采取措施对付此项计划。
 <br/>
 <br/>
 文章认为，如果习近平在G20峰会能够承诺回撤该计划，这不仅会缓解与美国及其他贸易伙伴的紧张关系，令外国市场对中国产品开放，同时也会促进中国企业的竞争力。文章最后说，习近平现在应该〝取悦〞川普，将来说不定还会感激他。
 <br/>
 <br/>
 川普7日在白宫举行新闻会上曾提到〝中国制造2025〞计划，他说，〝中国制造2025〞非常具有威胁性。〝我也已经告诉他们（中方）。我告诉他们‘中国制造2025’很有威胁性。〞他还说，中方已经放弃〝中国制造2025〞。
 <br/>
 <br/>
 川普表示，美国在往上走，中国经济在下滑。他不希望中共的经济下滑。他和习近平将在G20峰会期间举行一次很好的会晤，双方将会尝试达成协议，因为他希望与习近平及中国都有良好的关系。
 <br/>
 <br/>
 不过，有评论人士认为，中共只是口头上承诺从〝中国制造2025〞计划上撤下来，以安抚川普政府，但它不可能从根本上放弃2025计划。
 <br/>
 <br/>
 评论人士分析说，放弃2025计划，就等于放弃了中共掌控中国经济发展的工具，从某种意义上，等于削弱中共政权的稳定和统治。而中共的底线就是维持统治，影响到其政权和统治的事情，它绝不会干。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
