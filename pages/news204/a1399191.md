### 暗示放弃中国制造2025 环球时报发异声
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】北京最高层表态推动达成〝中美都能接受的方案〞，但也要求美方〝尊重中国选择的发展道路〞，被解读为释放核心问题不让步的信息。而被指代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共军方.html" target="_blank">
  中共军方
 </a>
 观点的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球时报.html" target="_blank">
  环球时报
 </a>
 》日前强烈暗示应该放弃〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞，引起舆论关注。
 <br/>
 <br/>
 上周五（11月9日），中共官媒《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球时报.html" target="_blank">
  环球时报
 </a>
 》发表了社论&lt;直面中美高科技之争，我们想说这些&gt;，从美国总统川普提及〝中方放弃了‘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 ’〞引发的舆论风波说起，表达了对〝中国制造2025〞的一些具体看法。
 <br/>
 <br/>
 社论称，美国启动中美贸易战的主因之一就是〝中国制造2025〞计划，因为担心中方对国企的巨额补贴会损害美企的竞争力，而且北京会鼓励盗窃美国知识产权，最终在技术上〝接管全球〞。美国的担心在其他发达国家也不同程度地存在。
 <br/>
 <br/>
 随后，社论声称，实施该计划是中方〝自己的节奏〞，美方应该放弃〝唯我独尊的心态〞，〝平等地看待中国发展技术的权利〞。
 <br/>
 <br/>
 不过，社论话锋一转，又称中方也有必要认真对待美方的抱怨，做出〝必要调整〞，比如有关国企补贴的问题以及如何对待民企的问题，可以采纳市场经济体制下的一些通行做法。因为中方在与西方的竞争中仍处于相对劣势，如果不把自己的发展计划与西方进行协调，就会面临更多困难。
 <br/>
 <br/>
 社论还特别提到，科技进步需要〝一步一步干出来〞，中国不大可能在短期内全面崛起，即使到了2025年，与世界先进水平的差距仍将很大，中方不应太过急躁。
 <br/>
 <br/>
 社论最后指，这些观点只是〝环球时报编辑部〞的一家分析，但是〝我们坚定地认为，我们讲的道理是成立的〞。
 <br/>
 <br/>
 社论中的〝我们〞，似乎有意借社论强烈表达自己的〝谏议〞。而自由亚洲电台曾引述中共体制内学者的说法指，《环球时报》一贯代表军方态度。
 <br/>
 <br/>
 这已不是《环球时报》第一次就贸易战发布明确观点。8月1日，该报也曾发表社评，提出〝我们〞的八条对美策略，建议中方退回〝守势〞。外界称这八条为〝新版韬光养晦〞。
 <br/>
 <br/>
 不过，《环球时报》这些观点，都不同程度地与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 当局的态度相左。
 <br/>
 <br/>
 贸易战爆发后，习近平〝过早放弃韬光养晦〞，已成为中共内部的批评对象。而直到目前，虽然习当局开始极力缓和中美紧张局势，但是最高层并没有释放在核心问题上让步的任何信息。
 <br/>
 <br/>
 日前，习近平在北京接见美国前国务卿基辛格时，虽然一再表明中美合作的〝重要性〞，但依然强调美国要〝尊重中国选择的发展道路〞。
 <br/>
 <br/>
 韩媒认为，这显示习当局虽然有意牺牲部分利益，但在涉及〝中国制造2025〞等关键议题上不会让步。
 <br/>
 <br/>
 有中共体制内学者分析指，〝中国制造2025〞和中共十九大确立的〝强国梦〞时间规划息息相关。取消〝中国制造2025〞，将在政治上产生巨大影响，因此当局不会放弃。
 <br/>
 <br/>
 在中共历史上，每次政治路线的改变，都伴随着你死我活的权力之争。毛泽东当年因为不得不承认〝大跃进〞错误而黯然下台，最后靠发动残酷的文革才夺回权力。
 <br/>
 <br/>
 习近平阵营因为反腐打虎和整肃权贵资本，在党内拥有许多政敌。外界分析，目前虽然习近平党政军权在握，地位暂时无虞，但是中美贸易战已经触发中共政治危机，一旦局势继续恶化，当保党成为中共〝最大利益〞的时候，反习势力或将获得反击机会。
 <br/>
 <br/>
 目前，在海内外舆论场中，习近平的修宪、政治经济的左转、〝一带一路〞等外交上的〝有所作为〞，都已成为批评的目标。另一方面，由于中共官员普遍作恶多端，习近平一些亲信旧部亦不例外，而蔡奇、刘奇和陈全国等人，更因为〝出位〞的施政而成为舆论打击焦点，其中甚至有党媒参与炒作。
 <br/>
 <br/>
 日前，海内外舆论又在〝关注〞重庆和福建的高考政审。而实际上，中共实施高考政审已有多年，并非这两地的首创。而且，最早曝出这个敏感消息的是重庆党媒《重庆日报》。
 <br/>
 <br/>
 中共政坛博弈日益激烈，暗流涌动。而观察人士指，中共已经危机重重，如果再次发生政治动荡，等待中共的可能不会是又一次权力更迭，而是政权崩溃。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：）
