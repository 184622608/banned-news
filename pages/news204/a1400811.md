### 川习会前敏感时刻 党媒宣扬第3艘航母新闻被删
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】即将登场的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 未来走向的转折点，双方若不能达成协议，或进入全面
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 局面。26日，中共党媒发布的宣扬
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共第3艘航母.html" target="_blank">
  中共第3艘航母
 </a>
 的新闻，刊登几个小时就被下线。这篇显示中共强军理念的文章，在川习会前的敏感时间点被删，引发外界诸多猜测。
 <br/>
 <br/>
 11月25日，中共党媒发文称，第3艘航空母舰〝已经在船台上有序建造〞。这是官方首次确认正在建造第3艘航母。26日，中共官媒澎湃新闻跟进报导，提及第3艘航母可能会采用核动力、电磁弹射。
 <br/>
 <br/>
 报导称，第3艘航舰是否达到美国尼米兹级航舰10万吨级的水准，是否会采用核动力和电磁弹射，这些都是网上的热议话题。
 <br/>
 <br/>
 同时，报导还引述了辽宁舰总设计师朱英富此前透露的情况，称目前中共蒸汽弹射和电磁弹射技术都在研究，这些技术一旦成熟，将会应用到未来的航母建造中。朱英富说，还有核动力，中共未来也一定要搞。
 <br/>
 <br/>
 台媒发现，这篇报导在早上9时刊出，至中午12时，网站却显示〝此文章已下线〞。不过，该报导被一些海外媒体引用。
 <br/>
 <br/>
 外界注意到，中共官方删除宣扬第3艘航母的报导，是处在一个敏感时间点。4天后，也就是11月30日至12月1日，中国国家主席习近平将出席在阿根廷举行的G20峰会，期间将与美国总统川普（特朗普）会面。双方将就美中贸易问题进行磋商。
 <br/>
 <br/>
 这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 被认为是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的关键转折点，引发全球关注。不过，外界对这次会晤能否有成果，普遍不看好。中方日前提交给美国的让步清单，仍回避美国关切的知识产权保护，强制技术转让等关键内容。
 <br/>
 <br/>
 川普表示，此清单不能接受，他希望中方将漏掉的4、5项关键内容加上。
 <br/>
 <br/>
 有分析指，这次川习会，美中双方若不能达成协议，未来美中两国很可能进入全面
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 时期。
 <br/>
 <br/>
 美国副总统彭斯13日接受《华盛顿邮报》的专访时表示，北京应把握机会，彻底改变自身行为，避免与美国爆发冷战。这是美国官方首次在公开场合提及新冷战的说法，从侧面反映，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 已十分严峻。
 <br/>
 <br/>
 彭斯指出，中方需要保护知识产权、禁止技术转移、取消限制进入中国市场、尊重国际规则及规范，以及保障在国际水域航行自由等。
 <br/>
 <br/>
 他还警告，若中方不能在川习会上，做出实质性的让步，美方将准备在经济、外交及政治上，对中方施加更大压力。
 <br/>
 <br/>
 美国总统川普日前也表示，北京非常想要达成协议，但协议必须是对美国有益的。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
