### 〝国进民退〞恐慌难平   习近平刘鹤相继发声安抚
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】有关中共当局推行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 〞和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_计划经济.html" target="_blank">
  计划经济
 </a>
 的说法，在民众中引起恐慌。近日，中共国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，就传闻相继发声安抚。但似乎并不能缓解
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民营企业.html" target="_blank">
  民营企业
 </a>
 界的恐慌。
 <br/>
 <br/>
 针对舆论近期热炒的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 〞传闻，新华社21日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 20日给一些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民营企业.html" target="_blank">
  民营企业
 </a>
 家回信，鼓励民营企业家坚定发展信心，踏踏实实办好企业。
 <br/>
 <br/>
 习近平称，改革开放40年，民营企业蓬勃发展，民营经济从小到大、由弱变强，在稳定增长、促进创新、增加就业、改善民生等方面发挥了重要作用，成为推动经济社会发展的重要力量。民营经济的历史贡献不可磨灭，民营经济的地位作用不容置疑，任何否定、弱化民营经济的言论和做法都是错误的。
 <br/>
 <br/>
 官媒19日报导称，中共国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 也表示，一些民企早前通过高负债扩张较快，由于偏离主业而遇到困难，国有银行或国企作出协助甚或重组，是帮民企度过难关，并不存在〝国进民退〞的问题。
 <br/>
 <br/>
 刘鹤说，当民企经营状况改善，国有资本可以退出，所以〝国进民退〞的说法是片面和错误的。他还称，会坚定不移贯彻基本经济制度，一方面巩固和发展公有制经济，另一方面鼓励和支持非公有制经济发展。
 <br/>
 <br/>
 刘鹤也承认，目前在执行过程中存在误解和偏差，譬如有些机构的人员认为，贷款给国企是安全，而贷款给民企在政治上有风险，这种认知和做法完全是错误。
 <br/>
 <br/>
 习近平9月底在东北考察期间也曾强调，要〝毫不动摇地〞实行〝公有制为主体、多种所有制经济共同发展〞的经济制度。始终〝关心〞、继续〝支持〞民营经济发展，对这一点民企要进一步增强信心。
 <br/>
 <br/>
 〝国进民退〞是经济学语汇，一般用于表示某领域国有资产进入，而民营资本被迫撤出。
 <br/>
 <br/>
 近几个月来，在中美贸易战升温的情况下，中共经济放缓，民营企业更陷入经营困境。北京当局近期加强了对民企的接管和控制；地方国有资产也在不断并购和入股民企；加之股市暴跌令大量民企股价逼近平仓线，面临被国有银行收回质押股权的危险。
 <br/>
 <br/>
 这令市场再次陷入恐慌，北京高层经济官员接连喊话，被认为有稳定股市之意。
 <br/>
 <br/>
 英媒BBC引述中国学者吴强的分析说，中共高层虽然对有关〝国进民退〞忧虑进行了回应，但他们说法〝逻辑有问题〞。因中共没有在体制上加强对民营企业的保护和尊重，外界看见的反而是相反的情况，包括在民营企业和非政府机构的党委建设。
 <br/>
 <br/>
 城南文史工作室负责人李道勇在评论文章中说，近年来，中共打着〝党领导〞的旗号，刻意推动国有资本〝做强、做优、做大〞，〝国进民退〞成为一种趋势，私营企业陷入前所未有的困境。
 <br/>
 <br/>
 中国民企由于银行去槓杆压力加大，上市公司纷纷陷入资本困境。很多民营企业在债务危机、股价下挫、质押爆仓、业绩疲软重压下，主动向国企〝求入股〞。
 <br/>
 <br/>
 甚至有企业愿意以每股一元的价格转让股权，主因在银行拿不到续贷，公司陷入流动性危机。在走投无路下〝投靠〞国企。
 <br/>
 <br/>
 对民企未来的走向，时政评论员文昭认为，当前中共还不会有大规模的私企国有化行动，因为当局有平衡就业和吸引外资两个问题，民营经济毕竟提供了大部分就业，而且大规模地〝杀猪拔毛〞也会把外商吓跑。
 <br/>
 <br/>
 文昭认为，这里有一个体制性原因在威胁着中国私营企业家整体，那就是中共政府的财政状况。如果说中共又动了国有化民企的念头，一个现实的动机就是由于财政走向匮乏，党和政府钱越来越不够花，对私营企业下手〝杀猪〞的动机也会越来越强。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：祝馨睿）
