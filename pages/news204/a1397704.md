### 【禁闻】王全璋案再反转 刘卫国声明继续代理

 【新唐人2018年11月01日讯】
 <b>

  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
   王全璋
  </a>
  案再反转
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘卫国.html" target="_blank">
   刘卫国
  </a>
  声明继续代理
 </b>
 <br/>
 <br/>
 〝709案〞被抓律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 一案，10月31号又传出新消息。
 <br/>
 <br/>
 10月下旬声明因身体受伤退出案件的山东律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘卫国.html" target="_blank">
  刘卫国
 </a>
 ，当天在社交媒体展示了一封据说是王全璋写给他的亲笔信。信中说，现在出现了一些新情况，要求他近期前往看守所进行沟通。与信件一起展示的还有一张天津第一看守所的〝被监管人员亲友送款凭证〞，上面显示，刘卫国当天以律师的身份，为王全璋存进了1000元现金。
 <br/>
 <br/>
 刘卫国表示，他将〝当仁不让，继续代理案件〞。
 <br/>
 <br/>
 王全璋的妻子李文足11月1号也在网上发布王全璋案最新情况说明，说她至今不知道刘卫国继续代理王全璋案一事，也不知道刘卫国是否会见了王全璋。李文足表示，她如果得到了王全璋的消息，会第一时间公开。
 <br/>
 <br/>
 网上很多人注意到，那封〝亲笔信〞中，王全璋的〝璋〞字写错了，而且名字上还按了手印，怀疑信件是否真的出自王全璋之手。
 <br/>
 <br/>
 <b>
  下中共出口监控模式 全球网络自由下降
 </b>
 <br/>
 <br/>
 总部位于美国的非政府组织〝自由之家〞，11月1号发表年度报告说，中共向新兴市场国家提供网络监控的培训和技术，出口它的网络监控模式。
 <br/>
 <br/>
 报告中说，中国是2018年侵犯网络自由最严重的国家。
 <br/>
 <br/>
 〝自由之家〞在对65个国家的网络自由状况进行研究后发现，〝数字独裁主义〞)越来越猖獗，导致全球网络自由连续八年下降。
 <br/>
 <br/>
 报告说，在这65个国家中，中共曾和其中的36个举行过有关控制信息的会议，并向一些国家的政府提供了电信监控设备。
 <br/>
 <br/>
 <b>
  〝
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双汇香脆肠.html" target="_blank">
   双汇香脆肠
  </a>
  〞被台湾验出
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟病毒.html" target="_blank">
   非洲猪瘟病毒
  </a>
 </b>
 <br/>
 <br/>
 继韩国、日本验出中国游客携带入境的猪肉制品有非洲猪瘟病毒之后，台湾10月31号也公布，在大陆制作的香肠中验出非洲猪瘟病毒。
 <br/>
 <br/>
 据报导，这些香肠是金门小三通旅客在入关前丢到农产品弃置箱内的，香肠品牌名称是中国制造的〝双汇香脆肠〞。
 <br/>
 <br/>
 〝双汇〞是中国最大的肉品加工厂，8月初沈阳爆发第一个非洲猪瘟病例，就是从双汇公司待宰的猪上验出的。
 <br/>
 <br/>
 台湾农委会副主委黄金城表示，虽然中共只通报了13个省份有非洲猪瘟疫情，但因为大陆检验不严格，实际上可能〝中国所有省份已经全部沦陷〞。
 <br/>
 <br/>
