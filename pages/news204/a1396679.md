### 【禁闻】10月24日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 武汉数百
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乐投受害人.html" target="_blank">
  乐投受害人
 </a>
 派出所维权
 <br/>
 <br/>
 10月22号，数百名武汉乐投理财的投资受害人，聚集在乐投武汉分公司辖区所属地的新华路派出所，索要〝合同诈骗〞受案回执，他们高喊口号，要求乐投还钱。
 <br/>
 <br/>
 2. 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 吴士豪
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京上访.html" target="_blank">
  北京上访
 </a>
 遭批捕
 <br/>
 <br/>
 因房屋遭政府强拆，上海居民吴士豪今年7月前往北京上访，两天后被谴送回上海，遭警方指控〝寻釁滋事〞刑事拘留，10月11号，又被正式批捕，目前吴士豪被关押在虹口区看守所
 <br/>
 <br/>
 3. 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 广场乞讨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遭刑拘.html" target="_blank">
  遭刑拘
 </a>
 <br/>
 <br/>
 上海维权人士沈金宝10月10号在人民广场向过往行人乞讨时，被警察强行带进附近的派出所，查验身份证发现他是访民身份后，警方直接将他送进浦东新区看守所刑事拘留。至10月24号，家属仍然没有收到书面通知。
 <br/>
 <br/>
 4. 姜家文派出所反洗脑被刑拘
 <br/>
 <br/>
 大陆维权人士姜家文10月15号在北京被截访押回原籍后，一直关押在丹东市七道派出所内，期间，他给一个看守他的小警察讲中国的法律现状〝反洗脑〞，22号，他因此被以〝涉嫌颠覆国家政权罪〞刑事拘留。
</div>
