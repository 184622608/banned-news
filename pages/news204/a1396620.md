### 〝非洲猪瘟〞持续蔓延 中国猪肉进口量激增
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情在中国持续蔓延，并已向南方扩散，湖南成为第12个爆发猪瘟疫情的省份。广东已禁止从湖南调入生猪，香港方面闻讯也倍感威胁。受非洲猪瘟推动进口需求，大陆猪肉进口激增。有分析师指，若疫情持续扩散，明年猪肉进口可能进一步上升。
 <br/>
 <br/>
 中共农业农村部10月23日发公告指，湖南省益阳及常德市的养猪场确诊有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 ，其中益阳桃江县养殖场有546只生猪，其中44只发病、17只死亡；而常德桃源县养殖场的268只生猪，208只发病、31只死亡。截至目前，大陆已有12个省爆发非洲猪瘟疫情。
 <br/>
 <br/>
 自由亚洲报导，据网传视频可见，桃江县有大量死猪被置于一个巨型坑内进行扑杀，现场有挖掘机及身穿全套防疫装备的工作人员。消息称，扑杀及发病的猪达700多只，养殖户损失惨重，血本无归。目前疫区有公安警车及公安人员值班驻守，非工作人员一律不得进出。
 <br/>
 <br/>
 当地居民欧先生说：我对政府很多的措施和手段，都是表示怀疑的。我也怀疑当局是不是能够有效的防控、防止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情蔓延.html" target="_blank">
  疫情蔓延
 </a>
 ，并及时向外界公布这些真相。即使当局指疫情得到控制，我还是非常担心。
 <br/>
 <br/>
 广东及香港居民也担心疫情会进一步接近。广东居民曾先生担心，疫情会蔓延到广东，听到消息后不敢吃猪肉。
 <br/>
 <br/>
 湖南邻近广东，广东猪肉主要进口自湖南。有消息指，在湖南确认爆发非洲猪瘟后，当局已停止从湖南调猪肉入广东。
 <br/>
 <br/>
 香港渔农界立法会议员何俊贤表示，如果非洲猪瘟南下广东，将会对香港构成一个很大的威胁。
 <br/>
 <br/>
 他说：（香港）猪的来源主要是广东省，而有少部分是来自江西，极少部分是来自这两个省以外的地方。猪瘟疫情一直向南，过了长江的话，就会是非常大的影响。
 <br/>
 <br/>
 他还指出，香港在应对非洲猪瘟的措施及资源不足，如果要大规模扑杀猪只，香港政府及猪场会有一定压力。
 <br/>
 <br/>
 据称，湖南生猪禁止调入广东后，广东生猪价格已出现暴涨。
 <br/>
 <br/>
 另据路透社报导，目前中国大陆正苦于控制非洲猪瘟疫情，自8月以来已经有超过40处疫情爆发，估计扑杀了约20万头猪只。
 <br/>
 <br/>
 报导称，数据显示，受疫情影响，大陆猪肉进口较前月增至94,317吨，较上年同期增加8.4%。上海汇易谘询的分析师Alice Xuan指出，〝若疫情持续扩散，明年猪肉进口可能进一步上升。〞
 <br/>
 <br/>
 值得注意的是，非洲猪瘟疫情不仅在大陆蔓延，还有扩散到周边邻国的危险。据报导，韩国此前已发现来源自大陆的猪瘟病毒。近日，日本也发现有来自大陆的猪肉制品携带猪瘟病毒。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
