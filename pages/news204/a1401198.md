### 含蓄否定〝大海论〞 京官发异声:明年将爆金融危机
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】中共十一届全国人大财经委副主任贺铿日前对外媒表示，明年中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_债务违约.html" target="_blank">
  债务违约
 </a>
 问题可能引发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融危机.html" target="_blank">
  金融危机
 </a>
 。目前北京高层虽然承认经济下行，但也多次强调中国经济的〝承受力〞。北京高官在中美首脑会晤前夕突然坦承危机，显得比较罕见。
 <br/>
 <br/>
 路透社11月27日刊出对贺铿的专访。他在专访中表示，中国经济面临的最大问题是债务，2019年中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_债务违约.html" target="_blank">
  债务违约
 </a>
 问题将更进一步暴露，有可能引发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融危机.html" target="_blank">
  金融危机
 </a>
 。
 <br/>
 <br/>
 贺铿称，中国经济明年将遭遇许多困难，包括债务违约问题，可能使金融机构的呆坏帐率快速上升至15%；中美贸易战负面影响会集中显现，外贸困难将进一步加大。
 <br/>
 <br/>
 今年5月，贺铿在北京参加〝2018中国企业信用发展论坛〞时，就曾表示中国经济处境非常危险。他提及中国企业债务率太高，到处都是僵尸企业，而地方债大概是40万亿，但地方政府没有一个想还债的。
 <br/>
 <br/>
 当时，贺铿这番言论一度引发外界热议。贺铿这次接受路透专访，是第二次公开谈及中国经济危机，但时机比较敏感。
 <br/>
 <br/>
 数日后上演的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 ，将决定贸易战走向。美方暗示，这可能是北京避免中美新冷战的〝最后机会〞。不过，中美双方目前均无明显让步的迹象，外界对此次会晤达成协议的期待值越来越低。
 <br/>
 <br/>
 目前，商业银行不良贷款余额突破2万亿元，僵尸企业资产多达3万多亿，金融风险正在迅速上升。日前中共一行两会下发文件，对数十家重点金融机构加强监管，并为其建立危机处置机制，被指意在防范即将到来的金融危机。
 <br/>
 <br/>
 因此，贺铿对明年可能爆发金融危机的担忧，显然并非危言。但作为北京一名高官，在敏感时刻对外媒坦承危机，也被猜测并非只是〝说句实话〞那么简单。
 <br/>
 <br/>
 中共全国人大财经委副主任的级别最低是副部级。但贺铿虽然位居部级官员，此前并名不见经传。
 <br/>
 <br/>
 党媒在10月底的中共政治局会议通稿中，首次公开承认经济下行压力加大。但是北京高层在11月的上海进博会上还曾发表中国经济的〝大海论〞，暗示不怕〝大浪〞、有在中美贸易战中坚持到底的〝信心〞。
 <br/>
 <br/>
 而贺铿承认明年就可能爆发金融危机，显然和〝大海论〞相悖。
 <br/>
 <br/>
 从中美贸易战开打之前，中共内部就意见不一，主战派和主和派一直在激烈争论。据编辑部设在北京的中共外宣媒体消息，就如何应对此次川习会，北京内部也曾爆发激烈冲突。
 <br/>
 <br/>
 外界认为，中共内部有关贸易战对策的巨大分歧，不止涉及中共政权本身生死存亡的问题，也与中共内部的激烈权斗息息相关。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：）
