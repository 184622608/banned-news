### 【禁闻】两美军舰穿越台湾海峡 中共低调
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】
 <b>
  两
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军舰.html" target="_blank">
   美军舰
  </a>
  穿越
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾海峡.html" target="_blank">
   台湾海峡
  </a>
  中共低调
 </b>
 <br/>
 <br/>
 两艘美国军舰10月22号穿越
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾海峡.html" target="_blank">
  台湾海峡
 </a>
 。美国国防部表示，这是例行通过台湾海峡国际水域。
 <br/>
 <br/>
 据报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军舰.html" target="_blank">
  美军舰
 </a>
 经过时，有中共军舰尾随，但全程没有跨越海峡中线。
 <br/>
 <br/>
 有军事专家表示，美军舰这次通行台湾海峡是向北京宣示台海不是中国内海的讯号。
 <br/>
 <br/>
 台湾行政院长赖清德表示，台湾尊重美国在国际海域通行的权利，也肯定美国在亚太区域和平方面所做的各种努力。他强调，这只是美国常规的演训动作。
 <br/>
 <br/>
 中共鹰派媒体《环球时报》23号就此发表了社评，但只说美军舰的行动，〝对中国不友好，是一个挑衅动作〞，不再高喊〝美军舰抵台之日，就是共军武统台湾之时〞。
 <br/>
 <br/>
 <b>
  中共高层禁女
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高尔夫选手.html" target="_blank">
   高尔夫选手
  </a>
  赴台参赛
 </b>
 <br/>
 <br/>
 因受到来自中共高层的压力，两名中国选手，被要求退出美国女子职业高尔夫协会在台湾举行的比赛，这项比赛有全球81位顶尖好手同场竞技。
 <br/>
 <br/>
 〝路透社〞10月23号透露，这两名选手分别是世界排名第九的冯珊珊和新秀刘钰。她们上个周末在上海比赛期间，被告知不得参加在台北举行的冠军赛。
 <br/>
 <br/>
 报导说，冯珊珊和刘钰是台北赛事中仅有的两名中国选手，但目前还不清楚究竟是什么人发出的退赛指示。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋案.html" target="_blank">
   王全璋案
  </a>
  8名律师被迫退出辩护
 </b>
 <br/>
 <br/>
 〝709案〞受害律师王全璋，被关押已经超过3年，但案件仍未开审。由官方指派为他辩护的律师刘卫国，日前突然表示因伤不再代理案件，成为第8个退出王全璋案的律师。
 <br/>
 <br/>
 王全璋的妻子李文足对此表示无奈，但表示，她坚持要用她自己聘请的律师为王全璋辩护。
 <br/>
 <br/>
 在官派律师刘卫国之前，已经有7位李文足自己聘请的律师，在官方的压力下都退出了王全璋案。
 <br/>
 <br/>
 王全璋是709案中最后一名仍被关押的律师，他于2015年7月被抓捕，并在2017年2月被控〝涉嫌颠覆国家政权罪〞。在被关押期间，传出曾遭受电击等酷刑折磨。
 <br/>
 <br/>
 编辑/周玉林
