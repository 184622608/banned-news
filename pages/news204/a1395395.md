### 朱鎔基罕见露面 曾骂央视〝胡说、狗屁〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清华大学.html" target="_blank">
  清华大学
 </a>
 经济管理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_学院.html" target="_blank">
  学院
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_顾问.html" target="_blank">
  顾问
 </a>
 委员会日前召开会议，90岁的中共前总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 罕见露面，引起舆论关注。过往他总在敏感时刻露面，高调挺习。朱鎔基早年还曾揶揄央视的说辞是〝胡说、狗屁〞。
 <br/>
 <br/>
 港媒10月14日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清华大学.html" target="_blank">
  清华大学
 </a>
 经济管理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_学院.html" target="_blank">
  学院
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_顾问.html" target="_blank">
  顾问
 </a>
 委员会11日召开会议，学院五任院长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 、赵纯均、何建坤、钱颖一、白重恩齐亮相合影。
 <br/>
 <br/>
 中共前总理朱鎔基是该学院首任院长，现场画面显示，朱鎔基身穿西装，脸露笑容，精神不错，旁边的赵纯均教授搀扶着他。
 <br/>
 <br/>
 舆论分析，中共卸任领导人因身份特殊，鲜少出现在公众视野，他们的露面往往被外界认为另有政治意涵。
 <br/>
 <br/>
 由于目前时值
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 ，使得北京遭到国际围困，加之中共十一长假前后，江派大员吴邦国、刘云山、范长龙等人突然接连现身，如今挺习派朱鎔基也罕见露面，是否与这波中共政坛波动有关？引发外界猜测。
 <br/>
 <br/>
 朱鎔基上一次公开露面，是中共十九大闭幕不久的10月30日上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 会见清华大学经济管理学院顾问委员会委员时，朱鎔基力挺习近平。
 <br/>
 <br/>
 在习江斗的过程中，朱鎔基多次在关键时期露面，力挺习近平的反腐〝打虎〞运动及〝习核心〞的地位。
 <br/>
 <br/>
 现年90岁高龄的朱鎔基，退休后极少在公开场合现身，据媒体披露，前几年他曾在一次座谈会上，炮轰中共教育长期规划纲要〝都是空话〞。他还嘲讽，央视新闻联播称，中国引领世界汽车业是〝胡说、狗屁〞，令在场陪同的国务委员刘延东都忍不住掩口大笑。
 <br/>
 <br/>
 尽管临别时，他叮嘱现场〝今天的话不外传〞，但现场学生仍把他的言论及相片放到了网上。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
