### 上海进博会 大批维权人士自由受限
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】首届〝中国国际进口博览会〞正在上海举办。众多上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 被限制人身自由，有的被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_软禁.html" target="_blank">
  软禁
 </a>
 家中、有的被关在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑监狱.html" target="_blank">
  黑监狱
 </a>
 、度假村，甚至遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刑事拘留.html" target="_blank">
  刑事拘留
 </a>
 。
 <br/>
 <br/>
 据《维权网》报导，上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 被告知：2018年11月5号到10号不准去〝进博会〞现场及附近拉横幅、喊口号等，否则将被公安机关从严处罚。
 <br/>
 <br/>
 目前无法统计有多少上海维权人士因〝进博会〞遭受骚扰，大多数家属受到当局警告，不准对外透露家人被关的消息。
 <br/>
 <br/>
 上海维权人士黄女士:〝反正
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刑事拘留.html" target="_blank">
  刑事拘留
 </a>
 都有六、七个了，这是知道的，还有没有知道的；那关在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑监狱.html" target="_blank">
  黑监狱
 </a>
 是无法统计的；看在家里也有；关在度假村的也有；关在监狱的也有；去旅游的也有。我们都不敢出门，出门就怕找麻烦。〞
 <br/>
 <br/>
 据《维权网》统计，因〝进博会〞遭受迫害的上海维权人士有：沈金宝、顾国平、陈建芳、孙洪琴等17人。另外丁菊英、倪明其等5人从11月2号被强迫失踪至今。
 <br/>
 <br/>
 上海访民程女士:〝都关起来了，他就是怕你们到会场上去嘛，罪名就〝寻釁滋事〞，有的在家里带出去，有的在路上走的时候都带了。我在北京就不让我回上海，不让我买票。〞
 <br/>
 <br/>
 上海访民透露，〝进博会〞过后，还要开一个党代会，估计被抓的人没那么快放出来。
 <br/>
 <br/>
 上海访民沈女士:〝这个地方不好去，我们不知道，想去玩，也要被抓，有的已经被刑拘了。把访民抓到里面去，为度假村搞活经营嘛，但是访民的经济还是没有搞活，房子到现在没落实，我们都在过水深火热的日子。〞
 <br/>
 <br/>
 〝进博会〞从11月5号到10号在上海国家会展中心举行，主要提供货物贸易展和服务贸易展。进博会举办前，上海市各部门采取严格的安保措施，邮件和铁路运输都加强安检力度，另外所有进沪列车实行〝二次安检〞等。
 <br/>
 <br/>
 新唐人记者熊斌、钟元采访报导
 <br/>
</div>
