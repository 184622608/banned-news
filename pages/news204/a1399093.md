### 全球政要云集纪念一战百年 习近平缺席有内情
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】近日，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国总统.html" target="_blank">
  美国总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）、德国总理默克尔等70多个国家的领袖齐聚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎.html" target="_blank">
  巴黎
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纪念.html" target="_blank">
  纪念
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一战.html" target="_blank">
  一战
 </a>
 终结一个世纪。但习近平缺席。为何习缺席如此重大的历史场合？有舆论认为，国际社会正在把中国和中共分开，一战
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二战.html" target="_blank">
  二战
 </a>
 的胜利与中共无关。同时也有声音质疑，正一步步与世界脱节的中共，是否要重走闭关锁国的回头路，与普世价值对抗到底。
 <br/>
 <br/>
 11月11日上午11点，法国所有的城市村庄，在同一时刻一起敲响钟声，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纪念.html" target="_blank">
  纪念
 </a>
 世界第一次大战终结100周年。全球70多个国家的元首和政府首脑，齐聚在法国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎.html" target="_blank">
  巴黎
 </a>
 的凯旋门，在庄严肃穆的礼仪中，反思那场惨烈的战争，为世界和平祈祷。
 <br/>
 <br/>
 参加纪念仪式的有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国总统.html" target="_blank">
  美国总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大总理.html" target="_blank">
  加拿大总理
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特鲁多.html" target="_blank">
  特鲁多
 </a>
 、俄罗斯总统普京，德国总理默克尔，联合国秘书长古特雷斯等70多国的领袖，唯中国国家主席习近平缺席这一重大的历史场合。
 <br/>
 <br/>
 有读者询问，是世界第一次大战跟中国无关？
 <br/>
 <br/>
 中共外交部发言人在8日的例行记者会上，回答了这一问题，他说，〝第一次世界大战是人类历史上一场深重灾难，给包括中国在内的各国带来了巨大创伤。〞
 <br/>
 <br/>
 《法广》11日刊文介绍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一战.html" target="_blank">
  一战
 </a>
 与跟中国的关系。报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一战终结.html" target="_blank">
  一战终结
 </a>
 后，属于协约国一方的中国属于战胜国之一，然而一战终结后签署的〝凡尔赛和约〞却把青岛从德国殖民者手中划给另外一个新起的列强日本。
 <br/>
 <br/>
 日本当时还向袁世凯直接提出了〝21条〞要求，并要求绝对保密。但这件事还是被泄漏了出来，次年北京爆发了〝五四运动〞。那段历史，给中国留下了耻辱的记忆。
 <br/>
 <br/>
 虽然一战的主战场在欧洲，中国不是直接的参战国，但是中国人以另一种形式参加了这场战争。一战中，14万中国劳工前往法国北方前线，为支援英法军队付出了巨大牺牲。中国劳工在法国北方抬担架、埋死尸、送补给、修铁路、修公路，许多人为之付出了生命。
 <br/>
 <br/>
 这些历史事实，欧洲也是很久以后才认识到中国人在一战中所付出的不可磨灭的贡献。
 <br/>
 <br/>
 近些年来，这些历史被中共拿来进行宣传，当年的中国劳工血泪史被描绘成中共对世界和平的贡献，把中共粉饰成和平秩序的建设者，以此标榜其〝负责任大国〞的形象。
 <br/>
 <br/>
 这次全世界70多个国家的领袖齐聚巴黎纪念一战终结百年仪式，大提琴手马友友拉起来巴赫萨拉邦德舞曲，数位中学生随后用不同语言朗诵了一战见证人留下的文字，其中一位女中学生用中文朗读了一位中国劳工停战当日写的信。
 <br/>
 <br/>
 《法广》的报导称，习近平缺席如此重大的历史场合，给人的感觉中国很遥远，一战跟中国完全没有关系一样。有关如何对待近代两次世界大战的历史，中国似乎跟世界有点脱节。
 <br/>
 <br/>
 报导说，前些年，中共举行抗战70周年纪念活动。虽然那场战争属于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二战.html" target="_blank">
  二战
 </a>
 的一部分，但如此盛大的纪念场合，出席活动的30位外国领导人中，西方国家的领袖几乎无人参加。
 <br/>
 <br/>
 值得一提的是，第一次世界大战时期的中国政府是北洋政府，第二次世界大战时期的中国政府是中华民国。
 <br/>
 <br/>
 西方国家领袖人缺席中共举办的二战纪念活动，是因为人们都知道历史，知道中共的底细。客观上反映出国际社会正在把中国和中共分开。一战、二战的胜利是中国的荣耀，但和中共没有关系。
 <br/>
 <br/>
 北京清华大学教授的历史学家秦晖，曾在《南方周末》撰文说，〝在两次世界大战中，中国都进行了正确的‘站队’，成为战胜国，这是民国外交的巨大成功。〞
 <br/>
 <br/>
 秦晖指出，二战中蒋介石领导的中华民国与英美盟国共同抗击日本，因而获得了联合国安理会常任理事国席位。
 <br/>
 <br/>
 但是中共在篡政后，把蒋介石保家卫国的抗战功绩安到自己身上，大搞抗日宣传纪念活动，自然不会被国际社会认可。
 <br/>
 <br/>
 BBC刊文称，两次世界大战，中国都是战胜国，这对〝中国崛起〞具有重大意义。但是中共篡政后却作出了完全不同的战略抉择，投靠了苏联为首的共产阵营。
 <br/>
 <br/>
 知名宏观分析师高善文曾在山西证券的演讲中说，过去40年中共只有两次〝赌对了〞，一次是1978年的对外开放，与美国共同对付苏联，使中美关系处于〝蜜月时期〞，与西方世界全面恢复交往。
 <br/>
 <br/>
 第二次是1991年苏联解体后的邓小平提出的〝韬光养晦〞，避免了成为美国的下一个目标。
 <br/>
 <br/>
 如今是北京再次抉择的关口。美中之间正在发生大规模贸易战，并引发了不同层面的摩擦与碰撞，大有〝擦枪走火〞的趋势。习近平恰在此时缺席这一重大国际场合，有声音质疑，中共是否要重回闭关锁国、与普世价值对抗到底。
 <br/>
 <br/>
 学者何清涟发推评论说，各国首脑云集法国，纪念一战终战一百周年，主题是和平、团结、联合 (力保欧盟一统)。两次大战留给我们的最重要的教训是：没有一个国家是孤立的 (美国必须与世界站在一起)。中国未来的政治诉求是要分裂成八百诸侯，各自孤立，与世界经验相反。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿）
 <br/>
