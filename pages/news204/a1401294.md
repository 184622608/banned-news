### 倪玉兰搬新家再遭监控 担心再次流离失所
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_倪玉兰.html" target="_blank">
  倪玉兰
 </a>
 几个月前再度被迫搬家。最近，公安在她新家外安装多个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监控.html" target="_blank">
  监控
 </a>
 探头，又有陌生人堵住楼道不让她外出。倪玉兰担心再次被驱赶。
 <br/>
 <br/>
 今年8月，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_倪玉兰.html" target="_blank">
  倪玉兰
 </a>
 再度被当局逼迫搬迁，还被房地产中介列入拒绝提供中介服务的〝黑名单〞，她的女儿几经周折为她找到住处。目前，倪玉兰的租房在通州区霍屯村。据了解，倪玉兰从2013年10月出狱以来，已被迫搬家21次。
 <br/>
 <br/>
 北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 倪玉兰：〝我们每一次搬家都是心惊胆战的，警察每一次都会挑拨离间。让房东和中介不让我们住，说我们是罪犯，威胁房东，说如果你们不给他们赶出去，你的住房就会被政府没收。〞
 <br/>
 <br/>
 倪玉兰透露，公安最近在她的新家外面装了3个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监控.html" target="_blank">
  监控
 </a>
 探头，其中，家门口的探头是可360度旋转的高清摄像头，还有陌生人堵在楼道口，她担心再度被驱赶。
 <br/>
 <br/>
 倪玉兰：〝最近又有陌生人堵在了我们这个楼道口，监视我们，不让我们外出，之前他们都会这么做的，对我们进行监视监控，看我们到哪去，处处都监控，以便将我们驱赶出出租房，他们一直都是这么做。〞
 <br/>
 <br/>
 除了对倪玉兰夫妇的新家进行监控外，当局对她女儿的小屋也已经监控很多年了。倪玉兰表示，这是因为当局担心他们一家与外界联系。
 <br/>
 <br/>
 倪玉兰：〝他们担心我们与外界联系，因为我们现在得到了外交官还有外媒，他们都对我们特别的关注，一直在帮助我们呼吁，所以他们有所顾忌，对我们监控。那个外交官来看望我们，他们会阻挠，外媒和我们连系采访，他们也会阻挠。〞
 <br/>
 <br/>
 现年58岁的倪玉兰曾获得〝郁金香人权奖〞。她原来是一名执业律师，因自家房屋遭到强拆而走上维权之路。她也因为帮助弱势群体维权，两次入狱并遭殴打致残，需要坐轮椅外出。2011年初倪玉兰再次被捕，之后被以所谓的〝寻釁滋事罪〞判刑。2013年出狱后，倪玉兰和丈夫频繁遭到当局
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逼迁.html" target="_blank">
  逼迁
 </a>
 、监视和驱赶，夫妇俩时常露宿街头。
 <br/>
 <br/>
 新唐人记者常春、陈洁、施宜君采访报导
</div>
