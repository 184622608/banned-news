### 港媒揭释学诚玩3P 〝佛教一带一路战略〞曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】中共派遣政客掌控和败坏宗教，以所谓〝人间佛教〞邪说，将导人向善的佛教变异成中共政治工具和败坏道德的场所。港媒日前揭露，前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛教协会.html" target="_blank">
  佛教协会
 </a>
 会长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_释学诚.html" target="_blank">
  释学诚
 </a>
 竟然在佛像前和两名女弟子淫乱，并配合中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞政策，派女弟子出国〝弘法〞。
 <br/>
 <br/>
 第5届世界佛教论坛正在福建莆田举办，该论坛本该由前中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛教协会.html" target="_blank">
  佛教协会
 </a>
 会长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_释学诚.html" target="_blank">
  释学诚
 </a>
 主持，不过他已因涉性侵多名女弟子落马。消息指释学诚已被迫辞职，并被送回老家。
 <br/>
 <br/>
 但据香港《苹果日报》26日报导，释学诚虽然在被举报后落马隐退，但他在根据地福州、莆田的地位仍稳如泰山，通过弟子间接操控多间庙宇。
 <br/>
 <br/>
 报导说，福建的极乐寺和莆田的龙泉寺，都被称作是释学诚的家寺，里面女尼众多，多是20岁出头的小姑娘。虽然释学诚性侵丑闻曝光，但这些女尼仍对释学诚深信不疑。
 <br/>
 <br/>
 报导披露，佛教经典中本有女僧尼必须跟随女法师出家的规定，但释学诚却违规大量招收女弟子。其中包括，他让很多本不符合出家条件的寺院女义工迅速〝受戒〞，成为冒牌女尼姑，却被这些女信众视为释某的恩惠。
 <br/>
 <br/>
 这些女弟子出家后，受到释学诚严密的精神控制，手段包括：财产充公、通讯控制、高强度劳动、膜拜释学诚肖像、学习释学诚语录、灌输〝男女双修〞引诱献身等。许多女尼因此成为释学诚的精神奴隶，一些企图反抗者则被压制消声。
 <br/>
 <br/>
 报导披露，根据一名出身港大博士的女弟子贤丙（化名）向北京警方提供的举报录像，前年9月一天晚上，在北京海淀区东南部魏公村一间精舍（僧团宿舍）供奉佛像的客厅内，释学诚和贤丙和贤乙（化名）两名女弟子发生淫乱行为。
 <br/>
 <br/>
 贤丙向警方指控，自己当晚是遭释学诚灌醉后性侵。贤丙报警后，释学诚私下给她36万人民币掩口费，送她去香港。而贤乙得知贤丙报警后，竟然怒不可遏，按住贤丙的头往墙上撞。
 <br/>
 <br/>
 报导还披露，释学诚善于网络炒作，也很懂得配合中共政策，甚至把佛教与北京的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞挂上〝战略关系〞，建议中共治下的佛教协助化解〝一带一路〞沿线的各种与宗教冲突，并提出〝致力建设人间佛教〞，打造〝新时代佛教〞，来弘扬〝中国佛教的声音〞。
 <br/>
 <br/>
 释学诚此前已连任多届中共全国政协常委，他在担任佛教协会会长的三年中，成为中国最具权力的〝政治和尚〞。
 <br/>
 <br/>
 报导指出，释学诚的女弟子中，约有一半被派到国内外近20个分院，成为释学诚海外〝弘法〞的主力。释学诚不选派男弟子开辟海外道场，理由是〝女众比较稳定，不会乱跑〞，而实情是这些女弟子容易被释学诚利用情欲进行精神控制。
 <br/>
 <br/>
 中共为维护其极权统治，多年来残酷打压导人向善的正统信仰。中国大陆一些仍有善根、寻求终极归宿的人们，只能选择加入被中共败坏的变异宗教中，最终悲惨地被〝改造〞成中共体制的一部分。
 <br/>
 <br/>
 而中共的〝一带一路〞，被指输出政治模式的阴谋。中共控制的〝佛教组织〞，配合中共出国〝弘扬〞淫乱和精神控制，或许也是中共对外〝文化侵略〞不可或缺的一部分。
 <br/>
 <br/>
 资料显示，今年52岁的释学诚是中国佛教界的〝最高领导人〞，他除了主持龙泉寺，还担任中共佛教协会会长、中国佛学院院长、中共宗教界和平委员会常务副主席、中共福建佛教协会会长、福建佛学院院长、福建莆田广化寺方丈、陕西扶风法门寺方丈。2008年，释学诚任中共全国政协常委并连任迄今，2018年3月还升任中共政协民族和宗教委员会副主任。
 <br/>
 <br/>
 今年8月，释学诚被实名举报性侵女弟子和涉嫌贪腐的丑闻曝光。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
