### 辽宁车撞小学生嫌犯系官二代 被捕称〝啥都认〞（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】北京时间11月22日中午12时15分，一辆轿车在辽宁省葫芦岛市建昌县第二小学校门前逆向冲入人群，陆媒指当场撞死5名小学生，18人受伤。下午消息，死者增至6人。驾车司机被曝是当地一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_村支书之子.html" target="_blank">
  村支书之子
 </a>
 ，作案原因不明。
 <br/>
 <br/>
 陆媒消息指，当时轿车是逆向行驶，直接冲向过路师生，碾压和撞飞多名儿童，随后驾车逃离，后被警方抓获。5名未成年人当场死亡，18人受伤，其中16人是未成年人，3人重伤。
 <br/>
 <br/>
 当天下午5点左右的陆媒消息指，死者已增至6人。
 <br/>
 <br/>
 报导指，事发时正值放学时间，老师带孩子在校外〝小饭桌〞用过午餐后，正带孩子返回学校，出事孩子为学前班和一年级二班的学生。
 <br/>
 <br/>
 <br/>
 <br/>
 警方消息指，肇事男子是该县老达杖子乡青牛山村
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_村支书之子.html" target="_blank">
  村支书之子
 </a>
 韩某华。他作案后驾驶黑色奥迪轿车逃走，13时左右被警方堵截抓获。目前已经排除酒驾和毒驾的嫌疑。
 <br/>
 <br/>
 韩某华被抓时未进行反抗，被带到公安局后向警方表示后悔，并称〝我认了，我啥都认〞。
 <br/>
 <br/>
 报导指，韩某华29岁，几年前结婚，这几年在沈阳打工，有一个五六岁的女儿。其父韩某升名下有一家生态休闲山庄，注册资金1000万元。
 <br/>
 <br/>
 目前暂不清楚韩某华作案原因。
 <br/>
 <br/>
 近年来，在中国各地频频发生恶意开车撞人事件。官方事后多以〝报复社会〞搪塞，拒绝公布嫌犯作案动机。
 <br/>
 <br/>
 外界分析，中共体制制造的种种社会不公令底层民众绝望，而中共多年灌输的暴戾基因，驱使这些人把仇恨宣洩在无力反抗的弱小群体身上。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
