### 持刀男闯入山东大学见人就捅  致1死5伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东青岛科技大学.html" target="_blank">
  山东青岛科技大学
 </a>
 日前发生一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_随机杀人.html" target="_blank">
  随机杀人
 </a>
 事件。一名男子手持螺丝刀闯入校园行凶，造成1人死亡，5人受伤，行凶男子被当场制服。
 <br/>
 <br/>
 据山东青岛警方官微11月26日凌晨发布通报称，25日下午5点左右，49岁的男子张某华，（黄岛区人)，持螺丝刀先后闯入山东科技大学家属区和校区捅伤6人。该校2名保安、1名教师、1名环卫工人和2位教师家属被捅伤，其中1名家属伤势过重，抢救无效死亡，其他人暂无生命危险。
 <br/>
 <br/>
 通报称，案发后，张某华当场被校园保安制服，随后被警方带走。初步查明，张某华〝近期言行异常〞，目前，案件正在调查中。
 <br/>
 <br/>
 据陆媒报导，山东科大爆发这起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_随机杀人.html" target="_blank">
  随机杀人
 </a>
 事件，引起师生恐慌。多名学生表示，幸好歹徒未能闯入教学楼，就被保安奋力制服，不然后果不堪想像。
 <br/>
 <br/>
 还有网民对警方通报嫌犯〝近期言行异常〞表示质疑，认为案情尚在调查中，警方就匆匆下结论。近期，大陆频频爆出类似随即杀入事件，官方的统一口径都是称嫌疑人〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_精神异常.html" target="_blank">
  精神异常
 </a>
 〞。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
