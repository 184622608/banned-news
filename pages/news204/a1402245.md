### 【禁闻】世界人权日前  欧美议员关注中共暴行
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界人权日.html" target="_blank">
  世界人权日
 </a>
 前夕，全球退党服务中心在美国国会内举办了〝制止中共人权暴行、声援民众退党大潮〞研讨会，同时发表英文版退党报告。多位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧美议员.html" target="_blank">
  欧美议员
 </a>
 、人权团体代表及专家到场发言或致信声援。
 <br/>
 <br/>
 全球退党服务中心主席易蓉：〝退党提供中国人民的是最根本的东西，自由与希望。〞
 <br/>
 <br/>
 美国国会内举办的退党研讨会上，资深众议员罗拉巴克出席致词。今年6月，他在众议院提出932号决议案，声援中国退党大潮，并要求中共立即停止迫害法轮功。
 <br/>
 <br/>
 美国会众院外委会资深议员 Dana Rohrabacher：〝法轮功与退党运动，多年来一直给我非常巨大的鼓舞。我和你们一样都认识到，那些以铁腕控制中国的人，不仅是本国人民的压制者，也是全世界的恶棍，整个星球的恶棍。这些恶棍会威胁到我们所有人，一旦手握权力。〞
 <br/>
 <br/>
 自2004年《九评共产党》发表至今，已有超过三亿两千万海内外民众选择退出中共党团队组织。会中，退党中心代表向即将卸任的罗拉巴克议员颁奖，感谢他对退党运动的长期支持。罗拉巴克也深表感谢。
 <br/>
 <br/>
 美国会众院外委会资深议员 Dana Rohrabacher：〝是你们给了我希望，并认识到中国和所有地方的人民，都可成为我们的盟友。我永远不会忘记，你们为世界和平和中国民众的自由所做的努力。这对全世界所有人都有着影响，尤其是对以信仰自由为首要价值的美国人。〞
 <br/>
 <br/>
 今年12月10日是联合国通过《世界人权宣言》70周年。多位法轮功学员讲述了他们与家人在中国大陆的受迫害经历，其中有饱受酷刑的艺术家，也有失去父亲的花季少女。
 <br/>
 <br/>
 与会官员及人权团体代表指出，中共践踏人权的暴行是对普世价值的公然挑战。过去19年，对法轮功的迫害更是空前残酷。
 <br/>
 <br/>
 国际宗教自由委员会政策分析专员 Dominic Nardi：〝坦白说 ，要不是我们有可信证据，很难相信，比如强摘被关押的法轮功学员器官(的暴行)。〞
 <br/>
 <br/>
 世界政治研究院院长 John Lenczowski：〝其专制系统的关键就是思想控制和言论控制。
 <br/>
 <br/>
 平等自由责进会R.E.A.L创办人 Jeffrey Imm：〝中共对普世人权的践踏，是对这些普世价值的宣战。〞
 <br/>
 <br/>
 当天活动中，美国国会资深众议员克里斯．史密斯(Chris Smith)和史蒂夫．金(Steve King)，及欧洲议会议员图恩‧克莱姆(Tunne Kelam)都发表声明，呼吁必须将迫害者绳之以法，并希望更多议员支持932号决议案及退党大潮。
 <br/>
 <br/>
 欧洲议会议员 Tunne Kelam：〝这是最务实、最现实的方式来削弱共产专制。〞
 <br/>
 <br/>
 欧洲议会议员 Ana Gomes：〝希望那些参与这次会议的人都能理解，需要接触所有的中国民众。因为仅依靠国外的人，这无法实现。必须要有互动。〞
 <br/>
 <br/>
 研讨会上，全球退党服务中心发布了英文版退党报告，内容包括退党大潮的背景、意义和影响，以及中共对自由世界构成的巨大威胁。
 <br/>
 <br/>
 全球退党服务中心宣传部主任李祥春：〝内容就是说，在中共的渗透下，我们怎么样能够让人民觉醒，让西方民众觉醒，跟中国人民一起，能够彻底地摒弃共产主义。〞
 <br/>
 <br/>
 全球退党服务中心主席易蓉：〝支持退党不仅是支持中国人民的自由，也是捍卫全世界的自由与安全。因为我们知道，共产主义的终极目的就是毁灭全人类。〞
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
