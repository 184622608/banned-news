### 摩根大通看衰川习会：未来数月贸战继续升级
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】中美首脑本月底将在二十国集团（G20）峰会上会晤，舆论关注是否带来贸易战转机。不过，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摩根大通.html" target="_blank">
  摩根大通
 </a>
 中国首席经济学家朱海斌认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 难有突破，未来数月情况可能更差，明年中国经济将面临更多风险。
 <br/>
 <br/>
 路透社报导，朱海斌13日在一次公开活动上表示，市场对这次中美元首会晤有一定程度的期望，但期望不高，因为目前看不到双方愿意让步的明显倾向。
 <br/>
 <br/>
 朱海斌说，〝按我们的判断，未来3至6个月（中美贸易关系）可能会变得很差。〞
 <br/>
 <br/>
 朱海斌预期，明年首季美国可能会对所有中国商品加征关税，贸易战对中国经济的影响，到了明年就会比较明显。
 <br/>
 <br/>
 朱海斌表示，在外部压力下，北京或相应调整国内政策，例如财政政策、去槓杆及支持民企政策，估计明年调整力度会更大，但与2008、2012及2015年的政策相比相对有限。
 <br/>
 <br/>
 他解释说，财政政策方面，若实际减税规模与公布的规模有差距，那么对经济刺激作用有限。货币政策方面，北京正陷入两难，如果继续去槓杆，很难取得理想反弹；而放弃去槓杆，债务问题会进一步蔓延，加速经济衰退。
 <br/>
 <br/>
 他举例说，北京2008年推出4万亿元人民币的刺激措施，短期内甚为见效，但10年来也造成很多负效应，包括企业债务问题上升、产能过剩、投资效率下降等。
 <br/>
 <br/>
 按照美方此前公布的计划，到了2019年1月1日，对2,000亿美元中国商品加征的关税税率将从10%提高到25%。另据消息，川普政府也正在准备继续对剩余的2前多亿中国产品加征关税。
 <br/>
 <br/>
 根据中共官方数据，目前中国产品对美出口依旧强劲。但外媒分析，这是因为中国企业赶在明年加税前加紧出货。
 <br/>
 <br/>
 据《华尔街日报》13日消息，中美贸易对话并无任何进展。美方坚持中方在谈判前先提出具体方案回应美方要求，而中方依然拒绝。
 <br/>
 <br/>
 北京目前虽然放软身段，一再强调〝中美合作〞，但依然强调美方要〝尊重中国选择的发展道路〞。韩媒指，这显示中方虽然有所妥协，但在〝中国制造2025〞等核心问题上仍然不愿让步。
 <br/>
 <br/>
 美媒引述分析指，即使这次中美能达成一个临时性协议，也难以持久。而白宫贸易高级顾问纳瓦罗（Peter Navarro）表示，中共一贯言而无信，所以即使达成协议，也无法相信中共能够真正执行。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
