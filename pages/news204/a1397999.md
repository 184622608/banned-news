### 邓朴方〝抬父压习〞惹祸？残联官网休克
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】中共前领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 之子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓朴方.html" target="_blank">
  邓朴方
 </a>
 ，早前在中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_残联.html" target="_blank">
  残联
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_换届.html" target="_blank">
  换届
 </a>
 大会上，大谈其父的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 ，在海内外引起了巨大震动，被认为是为邓家邀功向习近平叫板。近日，邓朴方的相关讲话遭当局
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_封杀.html" target="_blank">
  封杀
 </a>
 ，连带
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_残联网站.html" target="_blank">
  残联网站
 </a>
 整体休克。
 <br/>
 <br/>
 习近平10月下旬〝南巡〞广东期间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓朴方.html" target="_blank">
  邓朴方
 </a>
 在中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_残联.html" target="_blank">
  残联
 </a>
 第七次全国代表大会〞闭幕式上的内部讲话，被香港英文版《南华早报》全文发表。
 <br/>
 <br/>
 邓朴方在讲话中回顾了由其父
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 40年的〝历史进程〞，他说，以邓小平为首的老一辈人开启了改革开放的历史进程，给中国带来了翻天覆地的变化。
 <br/>
 <br/>
 他认为，我们一定要有这种实事求是的态度，保持清醒的头脑，知道自己的份量，既不妄自尊大，也不妄自菲薄，坚持立足国情，从社会主义初级阶段的实际出发谋划一切工作。
 <br/>
 <br/>
 上述讲话是9月16日在中共残联
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_换届.html" target="_blank">
  换届
 </a>
 大会上，选出邓朴方继续担任残联名誉主席时，他发表的演讲。
 <br/>
 <br/>
 中共残联官网最初全文刊登了邓朴方的上述讲话。但从10月初开始，残联官网就很难上去。被认为是高层下了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_封杀.html" target="_blank">
  封杀
 </a>
 令了。如今残联官网已无法再登陆。
 <br/>
 <br/>
 《法广》分析认为，邓朴方的讲话在海内外引起了巨大震动。官方为了不至于和邓朴方公开闹翻，又不好单独将他的这篇讲话剔除，于是采取了让残联官网整体〝休克〞的办法。
 <br/>
 <br/>
 习近平10月下旬南巡期间强调改革开放但避谈邓小平，而《人民日报》10月30日发表纪念改革开放40周年的评论文章中，同样只字未提邓小平。
 <br/>
 <br/>
 有传媒分析说，在外界质疑目前的中共领导层是否还继续坚持邓小平提出的改革开放，以及是否重新评价邓小平的改革开放〝功绩〞之际，邓朴方的讲话意在维护邓的〝改革开放〞的遗产，显示中共内部对坚持〝改革开放〞立场的分歧。
 <br/>
 <br/>
 有评论人士认为，邓朴方告诫〝要保持清醒的头脑，知道自己的份量〞，明显是说给习近平听的。邓朴方拿其父来谈改革开放，是想强调邓的功绩，并藉此向习喊话要〝知道自己的份量〞。
 <br/>
 <br/>
 美媒报导说，所谓〝人走茶凉〞，当年发动改革开放工程而改写历史的邓小平，如今〝被消失〞，做儿子的跳出来发声〝敲打〞习近平，网站都遭遇整体〝休克〞，印证了邓小平遭彻底边缘化。
 <br/>
 <br/>
 对于中共所谓的改革开放，观察人士普遍认为，在共产党领导下，任何改革都徒有虚名，而只是政治和权力运作的需要，不属于真正意义的改革。中共体制决定了任何改革都不可能成功。中共改革40年后的中国现状已经证实了这一点。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿)
</div>
