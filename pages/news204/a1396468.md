### 敏感时刻向习叫板？邓小平之子讲话罕见曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 南巡的敏感时刻，中共前领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 之子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓朴方.html" target="_blank">
  邓朴方
 </a>
 ，在中国残联换届大会的内部讲话曝光。他在讲话中大谈其父的改革开放，并强调〝要保持清醒的头脑，要知道自己的份量〞。有分析认为，邓朴方的上述讲话，疑似以父亲当年南巡后发起的改革开放，为邓家邀功，向习近平叫板。
 <br/>
 <br/>
 10月23日，中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 趁南下视察期间，主持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 的揭幕礼之际。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 之子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓朴方.html" target="_blank">
  邓朴方
 </a>
 在中共〝残联第七次全国代表大会〞闭幕式上的讲话罕见曝光。
 <br/>
 <br/>
 讲话中，邓朴方回顾了由其父邓小平开辟的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放40年.html" target="_blank">
  改革开放40年
 </a>
 的〝历史进程〞，他称1978年，以邓小平为首的老一辈人开启了改革开放的历史进程，改革开放是一场伟大革命。改革开放给中国带来了翻天覆地的全方位变化。这是社会结构、利益格局、思维方式等深层次的变化。
 <br/>
 <br/>
 他还告诫：〝我们一定要有这种实事求是的态度，保持清醒的头脑，知道自己的份量，既不妄自尊大，也不妄自菲薄，坚持立足国情，从社会主义初级阶段的实际出发谋划一切工作。〞
 <br/>
 <br/>
 上述讲话是9月16日中共残联换届大会上，选出邓朴方继续担任残联名誉主席时，他发表的演讲。
 <br/>
 <br/>
 由于正值习近平南下视察期间，上述讲话在此敏感时刻罕见曝光，引起舆论解读。
 <br/>
 <br/>
 有分析人士认为，邓朴方讲话中告诫〝要保持清醒的头脑，知道自己的份量〞，似乎是说给习近平听。邓朴方拿其父来谈改革开放，是想强调邓小平的功绩，并借此向习喊话要〝知道自己的份量〞。
 <br/>
 <br/>
 也有传媒分析说，邓朴方讲话中，为改革开放40年定的调非常高，对自己的父亲邓小平，更是给予不容任何质疑的高度肯定，虽然是对残联的讲话，但更多的是〝妄议中央〞的大政方针，多次引用邓小平的讲话，为其父的地位大声疾呼。
 <br/>
 <br/>
 改革开放是中共官方于1978年12月开始实施的一系列经济改革，由邓小平推动，这一决策改变了中共自1949年建政后经济上逐渐对外封闭的情况。
 <br/>
 <br/>
 1989年六四事件后，中共开始以〝经济发展为中心〞，试图用经济、行政和司法等领域小的变化，来掩盖其政权合法性问题。
 <br/>
 <br/>
 1992年1月18日至2月21日，邓小平视察武昌、深圳、珠海、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 、上海，发表重要〝南巡讲话〞，又称〝九二南巡〞或〝邓小平南巡〞。
 <br/>
 <br/>
 这些讲话要求中共坚持改革开放的政策，〝邓小平南巡〞成为中共经济改革的又一个转折点。此后不久，经济改革政策复苏，但政治改革进程则基本停止。
 <br/>
 <br/>
 对于邓小平南巡的原因，陆媒《中外管理》杂志透露，邓小平因担心苏联垮台的场景，在中国重现而做出的决定。
 <br/>
 <br/>
 1991年苏共垮台，邓感概：一个执政70年，号称有几千万党员的庞然大物，就这么一夜之间垮了！要知道，苏联的住房、工资、资源、生产力和社会发达程度，都比中国好得多呀！
 <br/>
 <br/>
 由苏联反观中共，邓的结论是发展才是硬道理，于是要求中共坚持改革开放的政策。
 <br/>
 <br/>
 不过，北京文史学者方滨对自由亚洲表示，中国改革开放40年，国退民进，包括党政分开。在共产党领导下，任何改革都徒有虚名，只是政治和权力运作的需要，不属于真正意义的改革。
 <br/>
 <br/>
 观察人士普遍也认为，中共不倒台，所谓的改革都是白忙一场，中共体制决定了任何改革都不可能成功。中共改革40年后的中国现状已经证实了这一点。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
