### 危害远超〝制造2025〞 中共〝标准2035〞计划曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】〝中国制造2025〞战略在过去的几个月里，早已成为美国政府在贸易战中集中火力的一个重要目标。日前，有台媒发文指出，其实，美国真正应该担心的是中共正低调推动的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国标准2035.html" target="_blank">
  中国标准2035
 </a>
 〞，这个计划的产业龙头已悄悄布局，其危害远比〝中国制造2025〞更为严重。
 <br/>
 <br/>
 台湾财经杂志《天下杂志》第659期发表的一篇文章，以今年9月在上海举行的中国国际工业博览会为切入点，分析了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国标准2035.html" target="_blank">
  中国标准2035
 </a>
 〞这项战略的发展动向。文章称，这个计划的要点是〝抢占
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_产业技术标准.html" target="_blank">
  产业技术标准
 </a>
 话语权〞，是〝真正对美国在全球市场竞争力有威胁力〞的战略计划。
 <br/>
 <br/>
 据该文透露，今年3月份〝中国标准2035〞已经正式在中国工程院启动，现在中共政府正在默默低调地推动着这个战略计划，试图从中国庞大的内需市场建立产业生态系开始，制定自己的技术标准，然后透过〝一带一路〞推向全世界。
 <br/>
 <br/>
 北京当局首先决定从IC设计、虚拟实境、5G关键零组件等领域的国家标准制定做起；接下来将要推进物联网、资讯科技设备互联与太阳能领域的国家标准，并试图把那些标准推向世界成为国际标准。
 <br/>
 <br/>
 据称，中共国家标准委工业二部主任戴红曾声称，当前，人工智能、大数据、云计算等新一代科技的全球性技术标准都还没有成形，〝这是实现中国产业和标准换道超越的良机〞。
 <br/>
 <br/>
 据文章披露，华为是中国半导体行业中最被中共官方寄予厚望的龙头企业，而汇顶（Goodix）是中国指纹辨识IC龙头，舜宇光学是中国车载镜头龙头。
 <br/>
 <br/>
 文章提到，彭博社最近已经开始察觉到，中国企业正在透过〝一带一路〞的各项基础建设计划，将各行各业的技术标准输出到沿线国家，试图主导这些市场的科技应用。
 <br/>
 <br/>
 对此，彭博社发出警告称，〝如果中国标准取代了西方标准，就会对西方企业在那些市场的获利能力带来威胁，进而改变国际市场的竞争格局。〞
 <br/>
 <br/>
 《自由亚洲电台》在相关报导中援引美国德州圣托马斯大学国际研究中心助理教授叶耀元的观点称，〝‘中国标准2035’是‘中国制造2025’的逻辑性延伸。如果说后者的目标是打造出能够支撑中国自主高新产业链的市场的话，那么‘中国标准2035’就是为了向全球推销中国的科技成果。〞
 <br/>
 <br/>
 叶耀元并进一步指出，从中共过去盗窃他国知识产权和科技的作为来看，类似偷鸡摸狗的行为恐怕还会持续下去。所以，〝中国标准2035〞看似是中国独立开创行业新秩序的一次大胆尝试，但中共的惯性决定了〝它的背后一定离不开投机取巧之举〞。
 <br/>
 <br/>
 他认为，中共很可能做的是，〝先把对方的技术偷回来，然后以对方的技术为标准再去向上开发，再发展出一条不同的路线。〞
 <br/>
 <br/>
 就在本周二（10月30日）《华尔街日报》报导称，澳大利亚一个战略政策研究所的一项最新研究报告显示，有中共军方背景的科学家正在扩展与美国顶尖高校学者之间的科研合作，但他们有时会掩盖自己的军方背景。这些中共军方派出的科研人员在美国、英国、澳大利亚等国家参与研究的领域包括量子物理、密码学和自动驾驶汽车技术等新兴技术，以及军民两用的科技领域。
 <br/>
 <br/>
 此前，海外媒体曾报导过多位中国访问学者和学生，在留美期间窃取了美国知识产权和重要技术后转送回国的例子。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：戴明）
