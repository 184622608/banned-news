### 【禁闻】10月3日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 大陆人士发起联署 要求中共释放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高智晟.html" target="_blank">
  高智晟
 </a>
 <br/>
 <br/>
 中国著名维权律师高智晟遭中共强迫失踪已经1年多，目前仍然杳无音信，大陆各地的维权人士继发起〝寻找高智晟律师万里行〞活动后，10月3号再次发表联署公开信，要求中共当局立即释放高智晟律师。
 <br/>
 <br/>
 2. 贵大发覆核结果
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨绍政.html" target="_blank">
  杨绍政
 </a>
 教授仍遭开除
 <br/>
 <br/>
 贵州大学经济学院教授杨绍政因发表批评中共〝公款养党〞等言论，遭到校方处分，去年被无限期停课，今年8月中旬被开除。杨绍政不服，向校方提出行政覆核，要求恢复教职。9月30号，校方向杨绍政发出覆核结果，决定维持原处分。
 <br/>
 <br/>
 3. 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高院.html" target="_blank">
  高院
 </a>
 喊冤 两人被拘留
 <br/>
 <br/>
 10月1号，上海市10多名访民到上海高级法院前喊冤，举牌诉说自己房屋被强拆、财产被抢夺，冤假错案投诉无门的悲惨遭遇。目前已知两名喊冤访民被拘留，其他人士下落不明。
 <br/>
 <br/>
 4. 防〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十一共振.html" target="_blank">
  十一共振
 </a>
 〞 长沙警方闹市抓人
 <br/>
 <br/>
 10月1号，是中共建政69周年，因早前有海外人士倡议〝十一共振〞抗议中共专制，因此，湖南长沙警方如临大敌，在位于市中心的五一广场和黄兴步行街商圈戒备，多位维权人士当天被抓捕。
</div>
