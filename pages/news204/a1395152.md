### 【禁闻】《天安门杀戮》 再揭中共血腥镇压铁证
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】2018年澳大利亚齐氏文化基金会第11届《推动中国进步奖》，10月10号宣布由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_《天安门杀戮》.html" target="_blank">
  《天安门杀戮》
 </a>
 编辑组获得。这本历时3年整理而成的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天安门.html" target="_blank">
  天安门
 </a>
 杀戮》，第一次披露了被军方标注为〝机密〞的内部文档，揭示了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 血腥镇压的铁证。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_《天安门杀戮》.html" target="_blank">
  《天安门杀戮》
 </a>
 这本书是来自全球各媒体、摄影和绘画
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天安门.html" target="_blank">
  天安门
 </a>
 学运的艺术家，他们所提供的经典图像集结成册，真实还原
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四.html" target="_blank">
  六四
 </a>
 天安门事件的历史现场。内容由专业媒体工作者协力企划汇编，以记实报导观点创造令人省思的空间。
 <br/>
 <br/>
 1989年4月27号，北京30万名学生冲破警察封锁线，走上街头抗议。美国摄影师安德森（ Forrest Anderson ）拍下了这个历史性的一幕。
 <br/>
 <br/>
 在中南海新华门前，一名示威者和一排守卫中南海的士兵对峙。5月13号，数百名学生自发在天安门广场静坐绝食，两天后增加至3144人。
 <br/>
 <br/>
 6月3号深夜，从西长安街向天安门广场强行开进的军队，在清除示威者设置的路障。荷枪实弹的军队开始在各地段与没有武装的学生和市民发生激烈冲突。一声枪响火光划过天空，一个身穿白衬衫的瘦小男学生腹部中弹倒地，北京城枪声开始接连不断。
 <br/>
 <br/>
 6月4号黎明时分，坦克部队冲破阻碍，占领了天安门广场。
 <br/>
 <br/>
 齐氏文化基金会创办人齐家贞：〝一大排的坦克车无情的辗过去，第二台，第三台，第四台，到处都是尸体，共产党大张旗鼓的开着坦克车，向人群辗压过去，如入无人之境，西方记者都有记载。〞
 <br/>
 <br/>
 从这张照片上可以看到，在士兵身旁的重型机枪。
 <br/>
 <br/>
 法国《巴黎竞赛》报导说：〝驻扎在天安门广场的军队，用重机枪和自动步枪向人群开枪。可怕的射击精度！从最早的死亡发现，北京的医院一直在接受子弹射在两眼之间和心脏的尸体。事实上，很明显是狙击手干的。我们找不到他……〞
 <br/>
 <br/>
 1989年6月5号，英国《每日邮报》报导说，〝这是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 元老派军镇压自己最光辉最优秀的青年：一场残忍屠杀的展示，以血腥的方式暴露其特具的权威危机，这种危机不仅冲击了当今浩瀚不安的文明，而且影响到整个共产主义世界。〞
 <br/>
 <br/>
 出版者认为，六四本质是邓小平为首的中共元老，为发动宫廷政变和军事政变而发起的血腥镇压，中共军队甚至在六四屠城中使用达姆弹。
 <br/>
 <br/>
 齐家贞：〝一颗子弹可以穿透10个人的身体。可以想像一下，如此邪恶的政府，还不要说你是杀死几百成千，几千。英国的泰晤士报，最近透露出保密文件解密，他们说的是超过1万人(死亡)。你用这样的武器，用这样的子弹，屠杀这些热血青年们，你杀一个，这个政府都是罪恶的政府，都应该下台，还坐在这里，今年7月1号，69年周年了，太长了。〞
 <br/>
 <br/>
 《天安门杀戮》这本书目前在台湾发行。旅居澳洲的作家，同时也是齐氏文化基金会创办人的齐家贞介绍，这本书是集聚一批海外专业媒体同行的力量，搜集了大量西方媒体，包括美国《纽约时报》、《时代》杂志、《新闻周刊》、英国《每日邮报》、法国《巴黎竞赛》等在六四期间的报导，以及众多知名摄影师、艺术家，冒着生命危险拍摄的画面和创作的作品，以真实的历史图文纪录，全面而深刻地揭露六四屠杀的邪恶。
 <br/>
 <br/>
 齐家贞：〝这本书的视野比较广阔，比以前更深入的揭示，六四不是突然发生，六四有它必然的根源。共产主义极端的残酷非人性本质，从斯大林嗜血成性的本质到中共49年开始，从土改杀人，从镇压反革命杀人，它假如没有前面这些杀人的嗜好，草菅人命的表现，那六四的杀人就不敢有那么猖狂。〞
 <br/>
 <br/>
 〝六四天安门〞事件死伤数字至今仍然是个谜。去年年底英国最新解密的文件则披露，中共军方至少杀害了超过1万人。
 <br/>
 <br/>
 采访/梁珍 编辑/黄亿美 后制/陈建铭
