### 中共乌镇互联网大会遭冷落  外企高管缺席
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的第五届世界
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_互联网大会.html" target="_blank">
  互联网大会
 </a>
 周三（11月7日）在浙江
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌镇.html" target="_blank">
  乌镇
 </a>
 开幕。不过值得一提的是，与前四届相比，中共出席会议的官员级别明显降低，而且，外国科技公司的高管们也几乎全部缺席会议。
 <br/>
 <br/>
 <h4>
  海外科技高管们为何缺席？
 </h4>
 <br/>
 据英国《金融时报》报导，在众多的海外科技公司当中，芯片制造商高通公司的首席执行官史蒂夫・莫伦科夫（Steve Mollenkopf）是唯一一位在开幕式上发言的外国公司高管。
 <br/>
 <br/>
 有分析人士表示，当前正处中美贸易冲突紧张之际，谁也不愿意为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 〝站台〞，因而都选择避而远之。
 <br/>
 <br/>
 总部位于北京的科技市场研究公司迈博瑞谘询（Marbridge Consulting）董事总经理马克・纳特金（Mark Natkin）指出，〝外国科技公司可能觉得，在目前的政治环境下，被视为与中国（中共）关系过于密切不是一个理想的形象。〞
 <br/>
 <br/>
 时事评论员蓝述则表示，人们对中共已经看得很清楚了，谁也不想去配合中共，不会为了一点钱就出卖自己的道德良知。生活在自由民主国家的人们，把自由看得相当重要，懂得自由的可贵。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌镇.html" target="_blank">
   乌镇
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_互联网大会.html" target="_blank">
   互联网大会
  </a>
  〝
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刷脸.html" target="_blank">
   刷脸
  </a>
  〞大行其道
 </h4>
 <br/>
 第五届世界互联网大会的一大〝亮点〞，是中共当局正式启用〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人脸识别.html" target="_blank">
  人脸识别
 </a>
 〞技术，与会的1500名中外宾客〝刷脸〞进场，〝刷脸〞看博览会展，〝刷脸〞入住、对号用餐等。
 <br/>
 <br/>
 此外，中共出席此次会议的官员级别，相比前四届明显降低了。前四届都是中共政治局常委出席，甚至习近平、李克强亲临会场，而这次却只有政治局委员与会。
 <br/>
 <br/>
 蓝述指出，中共发展高科技并不是为了造福百姓，而是为了更好地对民众实施管控，维护它的政权。它的科技越发达，对中国百姓越是坏事。与中共合作，就是在变相配合中共镇压百姓。当人们认识到这一点，自然就会远离它，这从此次互联网大会遭遇西方国家冷待也有体现。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：秦锋）
