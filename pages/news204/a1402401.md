### 【禁闻】12月7日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络作曲家.html" target="_blank">
  网络作曲家
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐琳.html" target="_blank">
  徐琳
 </a>
 被控寻釁滋事罪 判刑三年
 <br/>
 <br/>
 广州网络作曲家徐琳，因创作多首讽刺时弊的歌曲被控寻釁滋事罪。案件12月7号在广州南沙法院宣判，徐琳被判有期徒刑三年。
 <br/>
 <br/>
 2.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦母亲.html" target="_blank">
  黄琦母亲
 </a>
 北京遭拦截 被推倒在地
 <br/>
 <br/>
 大陆〝六四天网〞创办人黄琦在狱中健康恶化，他85岁高龄的母亲蒲文清一直奔走营救，12月7号，蒲文清到达北京西站时，遭到拦截，并被截访者推倒在地。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陕西景盛集资案.html" target="_blank">
  陕西景盛集资案
 </a>
 受害人省人大上访
 <br/>
 <br/>
 陕西三原县景盛集资案的上百名投资受害人，12月6号到陕西省人大常委会上访，要求三原县警方对企业法人采取拘留措施，并加紧办案，为他们追回血汗钱。
 <br/>
 <br/>
 4. 上海百人设宴 为获释
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 接风
 <br/>
 <br/>
 上海11月初召开〝进博会〞期间，警方拘捕了大量维权人士，目前这些人士陆续获得释放，12月5号，100多人在位于上海市第一百货商店7楼的顺风大酒店设宴，为这些重获自由的人士接风。
