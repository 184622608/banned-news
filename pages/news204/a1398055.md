### 【禁闻】11月4日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 上海〝进博会〞前 12
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 被抓
 <br/>
 <br/>
 11月5号至10号，中共将在上海举办首届〝进博会〞，上海当地的访民群体成为当局的维稳对象，根据访圈出的消息，到11月2号晚间，已经有4人被刑事拘留，8人被抓后失联。
 <br/>
 <br/>
 2. 广东顺德村官私卖土地 上千
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_村民抗议.html" target="_blank">
  村民抗议
 </a>
 <br/>
 <br/>
 广东佛山市顺德区北滘镇三洪奇村，因村官私自买卖村民土地，引发村民不满，11月1号，上千村民上街维权，大批警察到场暴力殴打和抓捕村民，其中2人被打伤、3人被拘留。
 <br/>
 <br/>
 3. 江西上千村民抗议建垃圾焚烧厂遭镇压
 <br/>
 <br/>
 江西九江市柴桑区赤湖工业园的上千名村民，最近一个星期持续上街，抗议政府在当地修建垃圾焚烧发电厂，11月3号，事件仍然没有平息，村民继续阻止施工，当地政府则调动大批特警镇压。
 <br/>
 <br/>
 4. 广州张广红因言获罪遭判刑
 <br/>
 <br/>
 网名〝拈花时评〞的广州居民张广红，因在网络上转发批评时政的帖子，因言获罪，11月2号，广州越秀区法院对他判刑1年半。
</div>
