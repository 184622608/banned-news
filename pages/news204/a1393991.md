### 专家评中美南海险爆热战：危险进逼迟早出事
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 加剧。近日，中美军舰在南海的军事对峙行动也随之升级。澳门军事专家黄东分析说，中美双方采用〝危险进逼〞，这种冷战时期的对抗方式，用得多了迟早会出事。
 <br/>
 <br/>
 美国国防部10月2日表示，中共军舰过分靠近在南中国海航行的美国驱逐舰，差点造成撞击。
 <br/>
 <br/>
 据美国太平洋舰队发言人、海军少校蒂姆．戈尔曼介绍，9月30日，美国驱逐舰〝迪凯特〞号(USS Decatur)在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南沙群岛.html" target="_blank">
  南沙群岛
 </a>
 ，进行旨在捍卫自由航行原则的巡逻时，遭遇中共一艘驱逐舰过分靠近，几乎造成碰撞。美舰不得不采取紧急行动避免碰撞。
 <br/>
 <br/>
 澳门国际军事会会长黄东对《明报》分析，目前中美采用的是冷战形式的对抗，在无法开枪开炮的情况下，无论是空军还是海军，双方都只好用这种危险的进逼方式来对抗，而且是愈逼愈近愈危险。
 <br/>
 <br/>
 他说，〝这种方式使用得多了，迟早会出事。目前大家只是抱了一种侥幸的心态。〞
 <br/>
 <br/>
 美国政府一位匿名官员说，美国海军所进行的是例行和定期的航行自由巡逻，正如美国过去所做的和未来将继续进行的那样。
 <br/>
 <br/>
 《路透社》认为，这是美国为了抵抗中共试图在南海战略水域限制航行自由，而采取的最新行动。中共、日本和一些东南亚国家的海军，也常常在那里行驶。
 <br/>
 <br/>
 美国批评中共，在南海建造人工岛屿，并在岛上设立军事设施。担心这些设施可能被用来限制在南海的航行自由。美国坚持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_航海自由.html" target="_blank">
  航海自由
 </a>
 行动不应该与政治纠缠在一起。
 <br/>
 <br/>
 《美联社》10月2日报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国海军驱逐舰.html" target="_blank">
  美国海军驱逐舰
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驶入.html" target="_blank">
  驶入
 </a>
 南沙群岛，和中共军舰危险地靠近美军舰一事，有可能导致两国目前已经相当紧张的关系进一步恶化。
 <br/>
 <br/>
 《自由亚洲》认为，美国此次在南海的航海显示，美中这两个世界最大经济体之间的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，正在超出贸易范围，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扩散.html" target="_blank">
  扩散
 </a>
 到美中两国关系的其它领域。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：范铭）
</div>
