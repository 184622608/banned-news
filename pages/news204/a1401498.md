### 【禁闻】中国经济寒冬 〝活下去〞互相取暖
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业市场.html" target="_blank">
  就业市场
 </a>
 景气低迷，国内经济遭逢内、外大环境因素的夹击，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活下去.html" target="_blank">
  活下去
 </a>
 〞成了互相勉励的口号。中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民大学.html" target="_blank">
  人民大学
 </a>
 就业研究所所长预测，中国将面对58年来就业总量的首次下降，而这可能只是一个开始。
 <br/>
 <br/>
 中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业市场.html" target="_blank">
  就业市场
 </a>
 出现低迷态势，市场利空消息不断。网络公众号〝智谷趋势〞28号发表一篇文章，〝2018，没什么比
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_活下去.html" target="_blank">
  活下去
 </a>
 更重要〞。
 <br/>
 <br/>
 文章说到，突然之间，〝大裁员〞的新闻满天飞。202万条招聘广告消失了，沿海省分的出口代工厂纷纷关门放假，富士康被传裁员34万，金融、地产都在裁员。
 <br/>
 <br/>
 作者引述中共人社部的说法，〝就业总量压力依然很大，就业结构性矛盾仍然十分突出。当前国内外不稳定、不确定因素也在增多，这些都会对部分企业生产经营和就业带来影响，就业形势依然严峻复杂。〞
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝我们已经很清楚的看见，工作岗位，就业机会，已经从中国转回到美国，换句话说，川普让美国再次伟大，把流失到中国的工作机会带回美国，这一系列的政策正在生效。〞
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田表示，目前看到中国有20几个省市呈报的经济增长指数，低于中共的预期。
 <br/>
 <br/>
 谢田：〝现在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 确实是进入了寒冬状态，跟美国的贸易战带来的效应，几个刺激效应，现在正一波波的展现，现在看来，中共自己的中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民大学.html" target="_blank">
  人民大学
 </a>
 就业研究所的预测，中国将迎来58年来就业总量的首次下降。〞
 <br/>
 <br/>
 中国人民大学就业研究所所长曾湘泉预计，今年中国就业总量在77,504万至77,600万人，而去年为77,640万人。他表示，今年中国就业总量将出现自1960年以来的首次下降。
 <br/>
 <br/>
 而根据目前中国的人口结构，这可能只是一个长期趋势的开始。就业总量的下降将给中国经济增长带来较大压力。
 <br/>
 <br/>
 前山东烟台大学教师张忠顺：〝贸易战到明年1月1号的时候，还要再提高一部分关税，所以现在好多外贸企业，它就是抢出口，就是在关税增加之前，先把这些出口都抢出去，抢出去之后，那个时候
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业.html" target="_blank">
  失业
 </a>
 可能更大。〞
 <br/>
 <br/>
 尽管就业总量压力增大，中共官方公布的就业数据仍然相当乐观。
 <br/>
 <br/>
 前山东烟台大学教师张忠顺认为，中共当局统计的失业率数字并不可靠，实际数据可能更大。曾湘泉在2016年4月也提出中国经济增长下行，国有企业隐形失业上升，这些都无法通过公开的失业率反映出来。
 <br/>
 <br/>
 失业率教育部公布的数据，大陆高校毕业生人数逐年走高，明年将达860万，再创历史新高。
 <br/>
 <br/>
 随着大陆今年推动机构改革，公家机关能释放的职缺也在减少。国家考试去年招录2.8万余人，今年只有1.45万余人，是近10年招收人数最低的一年。互联网行业也不乐观，阿里巴巴、百度、腾讯和京东等大型网路公司，日前都传出全面停招或缩减招募应届毕业生。
 <br/>
 <br/>
 谢田：〝进入2019年，美国已经课征了（中国）2500亿美元的商品（关税），关税会从现在的10%增加到25%，另外2600亿美元的产品关税也可能随之而来，这两项如果加上的话，那对中国的经济打击就更大，那失业的问题，通胀的问题，债务的问题，还有股市、房市泡沫的问题，可能那时候就会一泻迸发出来。〞
 <br/>
 <br/>
 失业容易就业难，中国共享单车ofo创办人兼执行长戴威28号发布内部信，表示冬天已经来临，风雪也随之而来，他呼吁员工，哪怕是跪着也要活下去。
 <br/>
 <br/>
 今年9月，中国房地产企业巨头〝万科〞集团在深圳举办秋季例会，会场上三个醒目的大字〝活下去〞，让外界注意到中国房地产的处境。而不到两个月的时间，〝活下去〞已经不是口号，而是进行式了。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/王明宇
