### 2架歼-7战机缅甸坠毁 缅军方：中共制造〝不敢买〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_缅甸军方.html" target="_blank">
  缅甸军方
 </a>
 发布声明称，从中共购买的2架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_歼-7战斗机.html" target="_blank">
  歼-7战斗机
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_缅甸坠毁.html" target="_blank">
  缅甸坠毁
 </a>
 ，导致三人死亡。6个月前，也有一架歼-7战机在缅甸发生坠毁事故。事故频发引起缅甸军方的担忧，表示中共制造的战机有一些麻烦，〝不敢买〞。
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_缅甸军方.html" target="_blank">
  缅甸军方
 </a>
 声明表示，16日清晨，4架歼-7战机从缅甸中部马圭省的空军基地起飞，执行飞行训练任务。7点30分，由于大雾弥漫，2架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_歼-7战斗机.html" target="_blank">
  歼-7战斗机
 </a>
 未能及时发现前方障碍物，在马圭省敏巫镇，撞上一个广播信号转播塔后坠毁，另外2架战机安全返回。
 <br/>
 <br/>
 据《纽约时报》报导，马圭省负责人表示，一架歼-7战机在稻田中坠毁，另外一架坠落在当地一个著名的佛塔附近。事故发生时，一名11岁女童不幸被一架飞机的碎片击中，送医不治死亡。当时该女童正和另外4名学生，在一家俬人学校外面复习功课，所幸其他4人受伤都不严重。
 <br/>
 <br/>
 缅甸空军一位官员对法新社透露，战机出事后，一架飞机的驾驶员没有来得及跳伞，在座舱内死亡。另外一架飞机的驾驶员虽然跳伞，但在落地时死亡。
 <br/>
 <br/>
 据悉，这是歼-7战机在缅甸发生的第二起坠毁事故。今年4月3日，一架歼-7战机在空中发生〝技术故障〞，在缅甸中部东吁县的稻田中坠毁，年仅30多岁的缅甸空军飞行员在送医途中死亡。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 不仅战斗机频频出事，中共制造的军用运输机也在缅甸发生恶性坠毁事故。去年，缅甸购买的中共制造运-8军用运输机，在安达曼海上空飞行时，飞入风暴云层，坠入大海，机上122名缅甸军人、家属和机组人员全部遇难。
 <br/>
 <br/>
 中共制造的飞机事故频发，引起缅甸军方的担忧。缅甸仰光的泰嘎政治研究所执行董事殷苗亨（U Ye Myo Hein）对《纽约时报》说：从飞机坠毁事故的频率中能看出，中共制造的飞机〝有一些麻烦〞。
 <br/>
 <br/>
 缅甸在上世纪90年代初期，进口了大约60架歼-7战机，而歼-7战机被指是中共对苏联米格-21战机的〝山寨翻版〞。
 <br/>
 <br/>
 殷苗亨还表示，从20世纪初，缅甸军方就不太敢买中共战机，转而从俄罗斯购买。今年8月，缅甸武装部队总司令敏昂良出席了在莫斯科举行的一个军事博览会，从俄罗斯购买了新型武器和飞机。
 <br/>
 <br/>
 事实上，中共制造的战斗机一直未能突破发动机这一关键技术，一直都是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_盗版山寨.html" target="_blank">
  盗版山寨
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯战机.html" target="_blank">
  俄罗斯战机
 </a>
 技术。今年4月，俄罗斯国防部曝光，中共已购买65亿美元俄罗斯军火，包括苏-35战机。俄媒嘲讽，中共山寨失败，只好买原装。
 <br/>
 <br/>
 另一方面，由于抄袭技术不到家，中共制造的战斗机多次发生事故。据中共官媒报导，2016年4月27日，一架歼-15战机坠毁，飞行员张超丧生；就在张超死亡的20天前，中共海军另一名舰载机飞行员曹先建驾驶歼-15也出现了事故，跳伞逃生后身受重伤。
 <br/>
 <br/>
 同年10月22日下午，一架歼-7战机在广西柳州失事坠毁，官方未通报机上人员存亡情况。
 <br/>
 <br/>
 2017年11月份，中共军队西部战区飞行中队长黄鹏驾驶歼-11B战机，在进行训练中突然发生飞行事故，年仅29岁的黄鹏遇难。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
