### 中共认怂？路透：11月未订购伊朗原油
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】美国此前警告第三国11月4日前停止进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗石油.html" target="_blank">
  伊朗石油
 </a>
 ，但中共官方一直坚称要维护中伊〝正常合作〞。不过据外媒消息，在接到美方最后通牒后，中国国有油企巨头9月就将进口伊朗石油减半，目前已停止11月与伊朗的石油交易。
 <br/>
 <br/>
 路透社10月24日引述两名直接了解情况的匿名消息人士称，因为害怕遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国制裁.html" target="_blank">
  美国制裁
 </a>
 ，中国两个最大的国有炼油商——中国石油化工集团（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中石化.html" target="_blank">
  中石化
 </a>
 ）和中国石油天然气集团公司(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中石油.html" target="_blank">
  中石油
 </a>
 )都没有订购11月交货的伊朗原油。
 <br/>
 <br/>
 一名国有石油公司的高管说，在获得美国豁免之前，没有哪家公司敢冒险采购11月的伊朗原油。
 <br/>
 <br/>
 另据路透社23日独家消息，中国昆仑银行因为美国制裁的压力，将停止处理来自伊朗的支付业务。昆仑银行是中伊之间资金流动的主要正式通道，中方几乎所有购油款项都通过昆仑银行支付。
 <br/>
 <br/>
 美国政府将于11月5日开始对伊朗实施第二轮经济制裁，以迫使伊朗从新谈判核协议。美国该轮措施，包括对继续进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗石油.html" target="_blank">
  伊朗石油
 </a>
 的第三国企业实施制裁。
 <br/>
 <br/>
 经济专家表示，美国掌控全球美元支付和结算系统，如果美国财政部对某个外国公司实施金融制裁，将令该公司的对外贸易陷入瘫痪。
 <br/>
 <br/>
 中国是伊朗最大的石油买家，因而成为美国重点关注的对象。
 <br/>
 <br/>
 此前，中共外交部等官方机构公开回应美方压力时，一直宣称中共和伊朗之间会保持〝正常的交往与合作〞，显示出拒绝美方要求的〝强硬姿态〞。
 <br/>
 <br/>
 但据路透社9月28日消息，中石化9月自伊朗进口原油的规模已经减半，原因是美国官员8月曾登门造访，发出了〝最后通牒〞。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
