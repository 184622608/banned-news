### 舆论管控又升级 网信办整肃自媒体查封近万帐号
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】中共国家网信办官网周一发布消息称，该部门从今年10月20日起，会同有关部门进行了针对自媒体展开的专项整治行动，查封了9800多个自媒体帐号，并约谈了腾讯网的微信和新浪网的微博等自媒体平台，对其〝疏于管理〞提出了严重警告。
 <br/>
 <br/>
 据这项讯息，〝唐纳德说〞、〝傅首尔〞、〝紫竹张先生〞、〝有束光〞、〝万能福利吧〞、〝野史秘闻〞、〝深夜视频〞等较知名自媒体帐号，都在这次被网信办的〝专项行动〞中被查封。
 <br/>
 <br/>
 网信办指称，这些被查处的自媒体帐号，有的〝传播政治有害信息〞，有的〝诋毁英雄人物、抹黑国家形象〞，有的〝传播虚假信息〞〝 扰乱正常社会秩序〞。网信办还扬言，下一步将继续加大管治力度，对胆敢继续违规的自媒体〝从严查处，绝不姑息〞。
 <br/>
 <br/>
 与此同时，中共喉舌媒体也相继发文，炮制出中国自媒体的所谓〝六大罪状〞，警告自媒体运营者不要踩〝政治法律红线〞，以至于有海外中文媒体报导称，中国自媒体〝冬寒骤降〞。
 <br/>
 <br/>
 中共自建立政权以来一直对国民实施严厉的思想管控。早期中共对国民的思想控制，主要通过洗脑宣传和政治运动来实现。随着时代的变迁和国际国内情况的变化，最近二十年来对网络舆论的管控表现得更为突出。从屏蔽、删帖、禁言、封号到出动警方抓人，中共无所不用其极，而监管机构对于民营媒体、自媒体的大规模整肃也已成为〝新常态〞。
 <br/>
 <br/>
 （记者李明综合报导/责任编辑：秦锋）
