 </a>

  性骚扰
 </a>
 案近日首次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开庭.html" target="_blank">
  开庭
 </a>
 ，并进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_证据交换.html" target="_blank">
  证据交换
 </a>
 。庭审结束后，当事人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弦子.html" target="_blank">
  弦子
 </a>
 （化名）和好友〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麦烧同学.html" target="_blank">
  麦烧同学
 </a>
 〞一同接受了媒体采访。弦子说，没想到当年与她一起实习的朋友成为了对方的证人。
 <br/>
 <br/>
 央视主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军.html" target="_blank">
  朱军
 </a>
 起诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弦子.html" target="_blank">
  弦子
 </a>
 和〝麦烧〞的名誉权纠纷案，10月25日，在北京市海淀区法院首次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开庭.html" target="_blank">
  开庭
 </a>
 。据悉，当天主要进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_证据交换.html" target="_blank">
  证据交换
 </a>
 流程。朱军涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性骚扰.html" target="_blank">
  性骚扰
 </a>
 事件的当事人弦子也提交了补充证据。
 <br/>
 <br/>
 庭审结束后，弦子更新微博说： 〝今天第一次出庭，比想像中难，但能坚持住，非常感谢大家。〞并附上了接受媒体采访的视频。
 <br/>
 <br/>
 视频中，弦子和〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麦烧同学.html" target="_blank">
  麦烧同学
 </a>
 〞向庭外等候的媒体表示，已向法院提交了对朱军的〝反诉〞，法院会针对证据和证据内容情况来看朱军能否出庭。
 <br/>
 <br/>
 弦子表示，〝我想过会没有人站出来，但没想到会有人站到对立面。〞
 <br/>
 <br/>
 在当天的证据交换庭上，弦子发现当年与她一起实习的朋友，为朱军提供了证词，成为了对方的证人。
 <br/>
 <br/>
 弦子表示，父母非常支持她，也相信她能得到一个公正的对待。弦子她们希望最后能赢，告诉大家这是怎么回事儿，以及女性在这个社会到底是一个什么样的地位……。
 <br/>
 <br/>
 虽然同宿舍的室友拒绝出庭作证，但是弦子及麦烧表示，对她们有利的证人还有很多。第一个是在事发后，弦子第一个倾诉对象，这位同学愿意出庭作证。
 <br/>
 <br/>
 另外还包括弦子的老师，她当时打电话联系她的老师问该怎么办，她老师和她老师的朋友都是律师，他们都愿意出庭作证。
 <br/>
 <br/>
 此次出庭，距离弦子曝料被朱军性骚扰，已经过去了92天。
 <br/>
 <br/>
 7月26日，麦烧在朋友圈发布长文，称自己朋友弦子，4年前在中央电视台实习时，遭到主持人朱军在化妆室里隔着衣服猥亵、强吻等，直到有嘉宾进来，她才得以逃离。
 <br/>
 <br/>
 事后，弦子报警，警察也进行了取证。但警方考虑朱军的〝对社会巨大正面影响力〞，规劝她放弃起诉。先是学校相关老师找该女生谈话，接着取证的警察也被更换。最后，事件不了了之。
 <br/>
 <br/>
 上述长文被大量转发，朱军性骚扰女实习生事件自此引发舆论关注。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
