### 外媒热议孟宏伟案中国式执法 学者：给美送子弹
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】担任国际刑警组织主席的中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 回国后突然人间蒸发，其妻向法国警方报警后中共官方才宣布孟涉贪腐已被调查。这种特殊的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国式执法.html" target="_blank">
  中国式执法
 </a>
 〞令一众外媒大为错愕，纷纷发文讨论该案对北京当局国际形象产生的冲击。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 今年9月底从法国回中国大陆后即与家人失去联系，危急关头更曾向其妻子发送了一把尖刀的图案。10天后孟妻向法国警方报案，法国警方随即对孟宏伟的下落展开调查。而在此期间，连国际刑警组织也不知道该组织主席的下落。
 <br/>
 <br/>
 此后，中共官方才在迫不得已的情况下对外界宣布，孟因涉及贪腐案件而被中共国家监察委扣押调查。但中共官方并没有明确指出，相关的贪腐指控究竟是与孟宏伟所担任的公安部副部长一职有关，还是与其担任的国际刑警组织主席一职有关。
 <br/>
 <br/>
 对于中共当局对一个身任国际刑警组织主席职务的官员，采取了突然袭击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘密抓捕.html" target="_blank">
  秘密抓捕
 </a>
 的方式来〝执法〞，令海外媒体大为惊叹。
 <br/>
 <br/>
 法新社10月9日发自北京的一篇报导援引多个政治观察人士的观点称，孟宏伟案的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国式执法.html" target="_blank">
  中国式执法
 </a>
 〞影响了北京的国际形象，对其寻求参与国际性组织的努力也是一个挫折。
 <br/>
 <br/>
 该报导分析称，作为世界上人口最多的国家，中国人在各大国际性组织中担任职务的数量相对较少，因此孟宏伟2016年获选担任国际刑警组织负责人时，虽然许多国际人权组织都对这一任命提出了反对和质疑，但这项任命还是被中共官方媒体宣传为〝一大外交胜利〞。
 <br/>
 <br/>
 但现在，北京当局竟然以一种〝连国际刑警组织也猝不及防〞的方式，对担任该组织主席的孟宏伟展开调查，这个异乎寻常的举动令外界十分〝错愕〞。
 <br/>
 <br/>
 法新社并援引中国问题专家、香港中文大学教授林和立的观点称，孟宏伟事件或对中国软实力产生负面影响。林和立表示：在中美贸易战正酣之际，这起事件（孟宏伟事件）对美国指责中国不遵守国际规则更添佐证。
 <br/>
 <br/>
 孟宏伟案被踢爆后，孟的妻子Grace Meng曾在法国召开记者会，呼吁国际社会和媒体关注她丈夫遭中共秘密抓捕的事件。评论人士认为，孟妻的举动，意在通过国际舆论的介入，来和北京当局讨价还价。
 <br/>
 <br/>
 英国BBC中文网日前也发文分析称，孟宏伟获委任为国际刑警组织时，曾被视为中共政府参与国际事务的〝一大突破〞，但该政府如今用国际刑警组织也〝措手不及〞的方法扣留孟宏伟，将令中国的国际形象大打折扣。
 <br/>
 <br/>
 香港浸会大学政治及国际关系学系副教授陈家洛接受英国BBC中文网采访时表示：这是一个〝国际级的大笑话〞，这对北京当局营造〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际软实力.html" target="_blank">
  国际软实力
 </a>
 〞的努力是一次〝挫败〞。
 <br/>
 <br/>
 陈家洛表示，发生了这样的事件后，将来如果中共政府还想派出代表人物到国际组织去担任什么领导职位时，国际社会将会有所保留。
 <br/>
 <br/>
 他解释说，因为其他国家会怀疑，这个人的注意力会不会仍然留在与北京政府或中共中央〝千丝万缕〞的瓜葛上，而不是他应该负责的国际组织的工作上。
 <br/>
 <br/>
 中国网友对孟宏伟事件的评论则联想到了更多方面。
 <br/>
 <br/>
 中国网友日前在海外中文媒体的相关报导下以keepitup的化名留言评论称 ：〝作为国际刑警组织主席的孟宏伟，他失踪后其妻只能向法国报警求助，令人悲哀啊。王立军当年发现自己有危险时也是往美国领事馆跑。这几年海外每有战乱、地震等发生时，中共就大肆宣传自己如何撤侨、营救，但其实在今天的中国，一个公民随时会被人间蒸发，他们的财产随时会在P2P暴雷这样的事件中灰飞烟灭，他们的孩子随时会被注射假疫苗，他们的生命和财产毫无安全可言。不少民众被它忽悠的把这个问题归咎地方部门，那个问题归咎无良厂商，但体制内的人却知道真正的原因，就是这个政党和它的制度所致。所以他们一遇到问题，反而不向‘党和政府’求救，而那些还没有遇到问题的官员和权贵，护照、绿卡早就备好，随时准备跳船而去。〞
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
