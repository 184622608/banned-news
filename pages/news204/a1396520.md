### 中共不顾影响扣查瑞银高管 德媒曝真实原因
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】近日，一名瑞士银行的财务主管到中国出差时被中方秘密扣留，引发国际舆论的广泛关注。日前，有德国媒体认为，中共官方查扣瑞银高管的真实原因，是为防止中国富人把财产转移到境外而〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杀鸡儆猴.html" target="_blank">
  杀鸡儆猴
 </a>
 〞。
 <br/>
 <br/>
 针对瑞银的一名顾问在北京被扣留事件，《南德意志报》日前报导称，瑞银虽然在上海和北京都设有分支机构，但其离岸理财的项目则由中国以外的专业人员打理，这显然让中共政府十分不满。
 <br/>
 <br/>
 文章认为，近日发生的一名长驻新加坡的瑞银女顾问在中国被扣押的事件，〝最合理的解释〞是，中共有关部门想要告诫国际大型银行和理财公司，不要动员中国人把财产转移境外，于是拿扣查瑞银女顾问来发出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杀鸡儆猴.html" target="_blank">
  杀鸡儆猴
 </a>
 〞的信号。
 <br/>
 <br/>
 瑞银高管被北京扣留事件发生后，瑞士《新苏黎世报》也曾在相关报导中分析称，在商业目的不明确的情况下向离岸帐户转移资金属于灰色地带，而以高价收购境外公司或其他投资项目做掩护转移资产到海外的做法，在中国很普遍。
 <br/>
 <br/>
 报导表示，北京过去对上述现象一直采取了比较容忍的态度，但现在看起来发生了变化。银行界人士普遍怀疑，北京为制止资产外流的势头需〝杀鸡儆猴〞，而通过留置其入境高管来向瑞银公司发出警告信号。
 <br/>
 <br/>
 现在，除瑞银之外，瑞士的另一家大型全球理财公司——瑞士宝胜银行（Julius Baer）也已要求员工无限期推迟去中国出差的计划。
 <br/>
 <br/>
 事实上，今年以来，随着中美贸易冲突的战火愈演愈烈，中国人民币对美元的汇率持续走低。眼看着人民币日益贬值，中国从富豪到平民百姓，为保住自己的财富少受损失，都开始绞尽脑汁想办法将资金转移出境，
 <br/>
 <br/>
 由于中共实行外汇管制，每人每年换汇不能超过5万美元，不少人不得不采取〝打擦边球〞甚至非法的手段，将资金转往海外。
 <br/>
 <br/>
 为防堵再次出现资金大量外逃，中共外汇管理局也不断在加大查堵
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_资金外流.html" target="_blank">
  资金外流
 </a>
 通道的力度。今年7月底，中国大陆媒体《经济日报》就曾发表报导称，中共国家外汇管理局正全面严查蚂蚁搬家、地下钱庄、内保外贷、虚构贸易合同、虚假转口贸易等五大资金非法外流渠道。
 <br/>
 <br/>
 瑞士信贷（Credit Suisse）前不久发布的《世界财富报告》显示，2000年以来，中国的个人财富递增了14倍，达到51.9万亿美元。个人资产超过10亿美元的中国人已经有3,480人。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
