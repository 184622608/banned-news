### 上千名郑州货车司机游行 多人被捕
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】11月以来，中国多地对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_排气标准.html" target="_blank">
  排气标准
 </a>
 为国III、国Ⅳ的柴油货车，实施限制进城的新规定，严重影响
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货车司机.html" target="_blank">
  货车司机
 </a>
 的生计。陕西西安和河北保定等地相继爆发维权行动，11月26号，又有上千名河南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑州.html" target="_blank">
  郑州
 </a>
 货车司机
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_游行.html" target="_blank">
  游行
 </a>
 示威，多人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被捕.html" target="_blank">
  被捕
 </a>
 。
 <br/>
 <br/>
 从11月21号起，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑州.html" target="_blank">
  郑州
 </a>
 禁止国III排放标准的柴油车（客、货）驶入，也禁止没有安装污染控制装置，或污染控制装置不符合要求的国Ⅳ排放标准的柴油车（客、货）驶入。
 <br/>
 <br/>
 新规定影响到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货车司机.html" target="_blank">
  货车司机
 </a>
 的生计。11月26号，上千名货车司机从二七纪念塔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_游行.html" target="_blank">
  游行
 </a>
 到郑州市政府，手持〝我们要养家、我们要吃饭，我们要路权、我们要生存，国III要生存、国III要吃饭〞等标语。当局出动大批警察和警车阻拦，双方爆发冲突，多人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被捕.html" target="_blank">
  被捕
 </a>
 。
 <br/>
 <br/>
 郑州货车司机王先生：〝昨天咱们在游行，大概有一、两千人，抓走大概有几人。国III的车刚买几年，说不让上路了，这办法生活，影响比较多一点，因为前几年买的车比较贵，又不让进去，找他们不给你说理。〞
 <br/>
 <br/>
 商家透露，这次郑州还规定货车要加装尾气治理设备，加装一个要三万多元，司机们怨声载道。
 <br/>
 <br/>
 污染控制装置商家张先生：〝郑州市这边他现在下了这个规定，需要自愿加装这个东西，就说这个车你只要到郑州市，经过绕城高速里面都要加装这个东西，可能上头会拍你，拍到一次就是扣三分，罚款二百。〞
 <br/>
 <br/>
 11月以来，中国多地对国III车开始了全天禁行，部分地区国IV、国V不允许进城，陕西西安和河北保定等地相继爆发维权行动。
 <br/>
 <br/>
 新唐人记者熊斌、周天采访报导
</div>
