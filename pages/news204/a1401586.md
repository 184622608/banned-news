### 〝川习会〞若失败 韩媒曝北京或故伎重施
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）与习近平将于二十国集团（G20）峰会期间举行工作晚宴，聚焦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 等话题。对于〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞结果，外界的期待值快速降低。有韩媒称，若两国无法达成贸易协定，北京或将利用民众的爱国情绪，掀起抵制运动。
 <br/>
 <br/>
 当地时间11月30日，阿根廷首都布宜诺斯艾利斯举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 ，韩国《中央日报》报导说，中国国家主席习近平与美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在G20峰会期间的谈判，将成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 选择休战还是继续的重要分水岭。
 <br/>
 <br/>
 中美贸易协商若以失败告终，贸易战会进一步激化，让世界经济的不安与不确定性更加严重。报导说，对于中共来说，最有力的反击武器是民族主义和爱国主义，会像当年反〝萨德〞时对待韩国那样，对美国制造的商品等掀起抵制购买的运动。
 <br/>
 <br/>
 报导认为，中共首先打击的对象，是在中国的麦当劳、星巴克、美国汽车、好莱坞电影等，中共对在中国国内的美国企业的管控与监测也会不断加强，中国人赴美旅游或留学的人数也会大减。
 <br/>
 <br/>
 报导说，如果最后到了〝鱼死网破〞的阶段，中共还有可能大量抛售美国国债。同时缓和对朝经济
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 ，扩大与伊朗的交易等，都将是中共对抗美国的外交牌。
 <br/>
 <br/>
 不过上述分析似乎都行不通。比如，中共想利用民众的爱国情绪，掀起反美运动之说，已经很难再奏效。因为随着互联网信息的快速传播，了解真相的大陆民众越来越多，任何人想拉13亿国民垫背，来维护权贵利益及中共极权暴政，各阶层的民众可能都会起来反对。
 <br/>
 <br/>
 之前海外党媒已暗示，不仅北京政坛有人支持川普。甚至中国民营企业也纷纷倒戈，支持川普对中共进行更严厉的贸易制裁。中国民间更是一片沸腾，纷纷期盼川普打垮中共政权，还中国人民人权，开放网路和报禁等。
 <br/>
 <br/>
 至于中方绑架在华美国企业进行贸易反击。之前也有业界学者分析说，此举正中川普下怀，有利于川普实现〝将海外低成本企业拉回美国的计划〞。
 <br/>
 <br/>
 而中共可能大量抛售美国国债反击一说，更是不成立，中共驻美大使崔天凯11月27还在罕见的承认，这非常危险，像是在玩火，中共方面应该不会有人考虑这种方式，它可能会引起反效果。
 <br/>
 <br/>
 他说，中共持有的巨额美国国债是两国经济相互依存的标志，想切断这种关系几乎是不可能的。
 <br/>
 <br/>
 据美国统计，截止到9月末，中共持有的美国国债的规模达到了11,500亿美元，是所有海外持有者中最多的一个。
 <br/>
 <br/>
 目前持续近一年的中美贸易战还在持续升温中，川普政府此前已对2,500亿中国商品课征10%到25%的惩罚性关税，未来征税的规模还可能扩大一倍。
 <br/>
 <br/>
 面对美方的持续施压，中国经济全面下滑，美国经济呈现强劲态势。
 <br/>
 <br/>
 有经济师分析认为，尽管美国在贸易战中一直处于上峰，但自贸易战以来，中共一直在将风险和危机转嫁到民众身上，还未感觉到真正的痛苦。但川普对中共是否继续抵抗以及谈判结果已经都不再那么看重，因为川普手里还有更关键的致胜筹码。
 <br/>
 <br/>
 而川普的许多策略以及行为方式，不同于史上任何一位美国总统，这给美中建交以来〝如鱼得水〞的中共，制造了前所未有的难题，中共的全球扩张野心终于踢到铁板。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
