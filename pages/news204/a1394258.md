### 【禁闻】孟宏伟失踪 中共制造国际笑话?
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
   孟宏伟
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
   失踪
  </a>
  中共制造国际笑话？
 </h4>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 担任主席的中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，据称回中国之后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 ，他的妻子日前向法国警方报案。
 <br/>
 <br/>
 法国警方表示，孟宏伟是上周离开国际刑警组织里昂总部返回中国的。
 <br/>
 <br/>
 香港《南华早报》引述消息来源说，孟宏伟上周搭乘航班抵达中国后，立即被中共纪检部门带走，目前正在接受调查。但报导没有透露他被调查的原因。
 <br/>
 <br/>
 国际刑警组织在一份声明中表示，〝这是法国和中共有关当局需要处理的事情。〞
 <br/>
 <br/>
 中共官员失踪后，家人联系外国政府的案例非常罕见。大陆有网民认为，孟宏伟的妻子打破潜规则，把他失联消息捅出去，应该是一次有准备的自救预案。
 <br/>
 <br/>
 〝德国之声〞报导说，主席在任职期间失踪，这是国际刑警组织从未遇到过的情况。并引述专家的话说：〝中共又制造了一个国际笑话〞。
 <br/>
 <br/>
 <h4>
  拒绝延长外国记者工作签证 香港首次
 </h4>
 <br/>
 英国《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融时报.html" target="_blank">
  金融时报
 </a>
 》10月5号报导说，该报亚洲新闻编辑马凯的签证延期申请，已经被香港政府拒绝，港府没有说明拒签的理由。
 <br/>
 <br/>
 马凯是香港外国记者会副主席。舆论普遍认为，签证被拒与马凯今年8月14号在香港外国记者会，邀请主张港独的民族党召集人陈浩天，出席午餐会并发表演讲有关。
 <br/>
 <br/>
 这是外国驻港记者首次被拒发工作签证。
 <br/>
 <br/>
 香港立法会议员涂谨申则对《南华早报》表示，港府由此向国际社会释放了一个信息，就是香港与中国任何其它城市没有区别。
 <br/>
 <br/>
