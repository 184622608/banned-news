### 【禁闻】军民融合瞄准高科技 方便窃取技术？
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 开展所谓〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军民融合.html" target="_blank">
  军民融合
 </a>
 〞，一年多, 调动至少3870亿元人民币投向民企，开发具有军事用途的技术。评论认为，中共一方面是要控制民企，另一方面则是要利用民企偷窃西方技术。
 <br/>
 <br/>
 据英国《金融时报》报导，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军民融合.html" target="_blank">
  军民融合
 </a>
 政策指导下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 已调动至少3870亿人民币投向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民营企业.html" target="_blank">
  民营企业
 </a>
 ，用于开发具有军用潜力的技术。
 <br/>
 <br/>
 〝军民融合〞就是把民营企业融入中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军工体系.html" target="_blank">
  军工体系
 </a>
 。这个概念中共在2015年提出，并在2017年设立了中央军民融合发展委员会，由中共最高领导人主管。在中共十九大上，军民融合被确立为发展战略，中共投入财政资金1500亿建立〝国创基金〞，用于军民融合项目。
 <br/>
 <br/>
 目前，中共官方媒体正在高调宣传这个所谓的〝军民融合〞战略。
 <br/>
 <br/>
 旅美时政评论员唐靖远认为，中共突然把历来被国企垄断的军事技术推向民营企业，说明中共完全控制了民营企业。
 <br/>
 <br/>
 旅美时政评论员唐靖远：〝也就是说，这种军工敏感技术对中共来说，它如果不能够确保绝对的控制，它根本就不会开放给民营企业的。所以我觉得客观上，等于这些民营企业事实上已经被国有化，可以说是国进民退的又一个非常典型的例证。〞
 <br/>
 <br/>
 据大陆《科技日报》报导，到今年3月份，已经有超过3000家民营企业跻身军事科技。
 <br/>
 <br/>
 〝军民融合〞也是今年珠海航展的一个重要主题，参展商都在强调自己产品与国家战略的契合度。
 <br/>
 <br/>
 一些完全与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高科技.html" target="_blank">
  高科技
 </a>
 无关的民营企业也投入到了军民融合项目中，例如曾是一家金属和矿业公司的广东宏大，已经在生产可由无人机和有人驾驶飞机携带的短程导弹和其他爆炸物，据称，其军工产业5年后有望成公司第一大业务板块。
 <br/>
 <br/>
 民营企业融入中共军工体系正在成为各省市的发展主题，并得到中共政府的政策和资金支持。
 <br/>
 <br/>
 唐靖远：〝由政府大笔注资来发展所谓的军民融合企业，说明中共发展技术的途径仍然是通过政府大规模的补贴，就是说用政府背景的这种企业和西方私人企业进行一种不正当的竞争，换句话说，就是中共不会停止被美国一直指控的这种手段，目的还是想要通过这种方式来确保它的政权的稳定，确保这些所谓的核心技术始终掌握在中共的手中。〞
 <br/>
 <br/>
 据《科技日报》报导，中国首艘国产航母的军民融合率接近80%。
 <br/>
 <br/>
 《金融时报》的报导说，中共的这种所谓军民融合已经造就了一批专门开发先进技术，诸如半导体、能量解决方案、无人机和航空航天的民营初创企业和民营公司。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝中共政治体制非常僵化，压制创新，过去几十年大批资金投入到军工生产、仿造、抄袭、剽窃，各种各样的手段，但实际上都没有成功，还有一点国有的军工企业效率非常的低下，他们也看到了民营企业在创新方面的优势，我想中共实际是再一次通过资本主义来救中共的社会主义。〞
 <br/>
 <br/>
 报导说，民营部门已经帮助国有企业将关键项目开发外包，它们往往着眼于向其他国家销售。
 <br/>
 <br/>
 旅美时事评论员蓝述：〝通过控制民企，让民企的面貌出现，在西方市场上更容易的获得外国的技术，同时也更容易进入外国的市场。〞
 <br/>
 <br/>
 不过，中共又在鼓励国有企业收购这些初创企业和民营企业。例如迈普是一家成都的初创公司，提供安全网络通信设备。2015年, 它已经被大型国有电信设备供应商中国电子信息产业集团收购。
 <br/>
 <br/>
 唐靖远：〝所以中共现在是想借用民营企业来保持创新，同时又不放松对民营企业的实际控制权，造成一种先放水养鱼，然后选择合适的时候来捕捞、收割成果的这样一个局面。〞
 <br/>
 <br/>
 不过，美国南卡罗莱纳大学艾肯商学院教授谢田认为，西方社会已经越来越警惕中国的企业，因而所谓〝军民融合〞也不会真正达到中共预期的结果。
 <br/>
 <br/>
 据多家英文媒体报导，中共的军民融合已经引起华盛顿注意。
 <br/>
 <br/>
