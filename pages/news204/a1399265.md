### 韩媒曝北京对朝政策突变：禁止给朝鲜〝送钱〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】有韩国媒体引述中国沈阳的消息人士披露，北京当局近期似乎收紧了对朝鲜的政策，命令禁止中国企业与朝鲜进行资金交易。有分析指出，相关禁令是一个多月前发出的，中方现在把这个消息释放出来，可能是在为G20峰会上的川习会面提前营造〝友善〞氛围。
 <br/>
 <br/>
 当地时间11月14日，韩国《中央日报》报导称，沈阳当地的一名消息人士13日透露说，中共政府近期向与朝鲜有经济交易的中国企业下令，〝禁止给朝鲜‘送钱’〞，因此〝现在中朝边境地区的交流比七八月份少了很多〞。
 <br/>
 <br/>
 该报导表示，中方对朝政策突然收紧是9月末到10月之间发生的，投资谘询与双方交易备忘录缔结等仍然在被允许范围，但是国际社会对朝制裁接触之前的资金交易被明令禁止。
 <br/>
 <br/>
 还有传闻称，最近从中国运往朝鲜的粮食交易似乎也被中国政府叫停，而这些粮食交易大部分都是非法的。
 <br/>
 <br/>
 外界注意到，就在一个月前（10月10日），中共外交官还曾与俄罗斯以及朝鲜的外交官员一起发表联合公告，向联合国安理会建议考虑放松对朝制裁措施，以鼓励半岛发生的积极变化，但遭到美国方面的严词拒绝。当时美国国务卿蓬佩奥在出席联合国安理会朝鲜问题部长级会议时再次明确表态说：〝必须继续加强执行安理会的制裁措施，直到（朝鲜）实现全面、可核查的无核化〞。
 <br/>
 <br/>
 中美负责外交和安全工作的部长11月9日在华盛顿进行〝2+2外交安全对话〞时，双方虽然对南海、维族人权等主要议题的分歧仍然尖锐，却唯独在朝核问题方面，就履行对朝制裁一事，双方达成了一致。
 <br/>
 <br/>
 针对中方上述态度的转变，有舆论指出，在近期举行的第二轮中美外交安全对话中，北京在朝核问题上的核心立场与美国政府的立场展现出了一致性。如今，中方人员更主动向外披露中方已收紧中朝贸易政策的资讯，表明试图跳出贸易战泥沼的北京当局，正在拿〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁朝鲜.html" target="_blank">
  制裁朝鲜
 </a>
 〞这张牌向川普政府示好，为即将在G20峰会上举行的美中首脑双边会谈提前营造〝良好的气氛〞。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：戴明）
