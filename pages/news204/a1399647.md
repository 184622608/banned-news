### 【禁闻】美中贸易战谈判 传北京服软 书面回复
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 最近警告北京，除非中共全面改变经济、军事和政治行为，否则美国将对中国展开全面性的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冷战.html" target="_blank">
  冷战
 </a>
 。对于美方坚决的态度，北京逐渐软化，有媒体传出，北京已经向华府回应书面资料。下面来看专家的分析。
 <br/>
 <br/>
 在新加坡举行的第33届〝东南亚国家协会〞峰会（ASEAN），11月15号闭幕，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 与南海问题持续受到关注。
 <br/>
 <br/>
 美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）这次代表美国总统川普出席〝东协峰会〞。彭斯强调：美国对区域的承诺〝稳定且持久〞，印太地区容不下〝帝国和侵略〞。外界认为，这是暗批中共在南海的影响力和军事扩张行动。
 <br/>
 <br/>
 就在中、美展开新一轮磋商之际，彭斯13号在新加坡接受媒体采访时表示，除非北京同意华盛顿的要求，全面改变经济、军事和政治行为，否则美国将对中国展开全面性的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冷战.html" target="_blank">
  冷战
 </a>
 。
 <br/>
 <br/>
 时事评论员田园博士：〝我认为彭斯副总统发表这个讲话，他的警告意味是非常浓厚的。就是你如果继续沿着这条路走下去，这就是你会得到的结果。但是我相信中、美两国目前都不想走到这一步。〞
 <br/>
 <br/>
 《路透社》消息，北京就美方对贸易问题所提的要求及其它议题，已经向华府提出书面回应。
 <br/>
 <br/>
 消息人士透露，所谓书面回复，只是北京略加修改先前已提出的作法，像是提高特定产业的外国投资股权上限，但不包含华府寻求中共改变〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞等产业政策的具体承诺。
 <br/>
 <br/>
 田园博士：〝中国虽然放弃了这个提法，并没有放弃想要在20年、30年之内和西方在技术方面达到平起平坐的目的，这个目的是中共绝对不会放弃的。〞
 <br/>
 <br/>
 中共商务部发言人高峰没有对上述说法做出回应，但承认双方已经恢复高层的接触。
 <br/>
 <br/>
 中共商务部发言人高峰：〝11月1日中美两国元首通话后，双方经贸领域高层接触已经恢复，同时，工作团队正在保持接触，以认真落实中美两国元首通话达成的共识。〞
 <br/>
 <br/>
 时事评论员田园博士认为，不管出现哪种情况，中共有些固有的底线是绝对不会放弃。
 <br/>
 <br/>
 田园博士：〝第一，就是大防火墙，这个东西是中共的命根子，如果没有大防火墙，中共估计倒台也就是几年的光景。还有另外一个，就是美国公司在中国市场的市场准入的问题。还有一个中共很难放弃的东西，就是它对中国企业的补贴。这些国营企业本来是在一种激烈的国际竞争环境下，根本无法生存，那么有了中共的补贴，这些企业才能苟延残喘到今天。〞
 <br/>
 <br/>
 中国问题专家章家敦也表示，中共很难在关键问题上做出令美国满意的让步。
 <br/>
 <br/>
 中国问题专家章家敦：〝但即使双方达成协议，中共也会违背的。因为中共根本没有准备要诚实地与其他国家进行交易。〞
 <br/>
 <br/>
 上个月，商务部长罗斯证实美中贸易谈判已经停止。美方表示，中方若不就美方提出的知识产权盗用、产业补贴以及市场准入设限等担忧作出切实的回应，贸易谈判就不会恢复。
 <br/>
 <br/>
 另外，由中共发起的亚太〝区域全面经济伙伴关系协定〞（RCEP），在这次东协峰会上被搁置。
 <br/>
 <br/>
 这项被视为是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 的因应制衡，随着中美贸易摩擦的不断升级和国内经济下滑，中共迫切想推动这项协议。《华尔街日报》指出，中共并试图把自己塑造成自由贸易和多边主义的卫士，而这些口号要转化结果，不是那么简单。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/周天
