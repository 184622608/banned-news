### 天津17岁少年假扮富二代 骗财百万还生了一个娃
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】天津一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_未成年.html" target="_blank">
  未成年
 </a>
 的李姓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_少年.html" target="_blank">
  少年
 </a>
 ，借长相老成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假扮.html" target="_blank">
  假扮
 </a>
 26岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富二代.html" target="_blank">
  富二代
 </a>
 ，与34岁张女交往，一年多内，骗取对方近百万人民币，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女方.html" target="_blank">
  女方
 </a>
 为他生下一个孩子。近日，李男已被批捕。
 <br/>
 <br/>
 陆媒24日报导，张女曾任职当地一家地产中介公司，后来李男到该公司应聘，两人成为了同事。 李男自称26岁，生于富康之家，自己不希望依靠家境，于是出来闯一番事业。
 <br/>
 <br/>
 后来，张女觉得李男十分体贴，慢慢对他产生了好感，两人逐渐〝坠入爱河〞。
 <br/>
 <br/>
 但是自去年8月起，李男编造一些理由向张女借钱，先是称，自己有金融方面的门路，可以弄一批抵押车倒手赚钱，于是，张女向母亲借40多万元人民币给张男周转；后来李男又称，可以办理网络贷款，张女又介绍许多亲戚朋友给李男做〝客户〞。
 <br/>
 <br/>
 今年7月，张女因为怀上李男骨肉而住院产子，在张女住院期间，李男对张女说，要办理会员卡，需要张女手机上的验证码。张女没有多想，马上将验证码告诉了李男。
 <br/>
 <br/>
 不料想，李男9月份失踪，后来透过多间银行的催款短讯，张女才得知，李男索要验证码，办理了10多张信用卡和网贷业务，共透支40多万元人民币。
 <br/>
 <br/>
 再加上张女的亲戚通过李男办贷款被骗的钱，以及张女从母亲那里借的钱，李男共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诈骗.html" target="_blank">
  诈骗
 </a>
 张女高达逾百万元人民币。
 <br/>
 <br/>
 张女发现受骗后报警。警方搜集证据进行侦查，获悉李男因涉嫌其他诈骗案件，已经于9月份落网。之后，该案移交至西青分局进一步侦办。
 <br/>
 <br/>
 据李男交代，自己并非富家子，只是一名无业游民，父亲也没有工作，家里经济状况很不好，他看到一家房产中介招聘，17岁的他凭藉自己长相老成，虚报年龄成功应聘后结识了张女，交往过程中，李男又虚构事实大肆骗钱。
 <br/>
 <br/>
 目前，犯罪嫌疑人李男已被逮捕。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：李泉）
</div>
