### 福建近70吨毒碳九泄漏 瞒报串供激怒高层
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】近日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建泉港.html" target="_blank">
  福建泉港
 </a>
 毒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碳九泄漏事件.html" target="_blank">
  碳九泄漏事件
 </a>
 再度爆出重磅消息。据中共官方通报，泄漏事件发生后，涉事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东港石化公司.html" target="_blank">
  东港石化公司
 </a>
 相关人员恶意
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瞒报串供.html" target="_blank">
  瞒报串供
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_近70吨碳九泄漏.html" target="_blank">
  近70吨碳九泄漏
 </a>
 被瞒报成7吨。目前，东港石化董事长在内的7名直接责任人，已被刑拘。
 <br/>
 <br/>
 11月4日凌晨，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建泉港.html" target="_blank">
  福建泉港
 </a>
 突发毒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碳九泄漏事件.html" target="_blank">
  碳九泄漏事件
 </a>
 ，引发严重生态灾难，当地40万民众生活堪忧。事发后，官方通报称，东港石油化工实业有限公司在执行碳九装船过程中，6.97吨碳九泄漏。
 <br/>
 <br/>
 事实上，碳九泄漏事件被严重瞒报，碳九的实际泄漏量为69.1吨，与此前官方公布的6.97吨，几乎相差十倍。
 <br/>
 <br/>
 11月25日，福建泉州政府召开新闻发布会称，碳九泄漏事件发生后，涉事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东港石化公司.html" target="_blank">
  东港石化公司
 </a>
 要求中层以上员工统一口径，隐瞒泄漏数量，并和码头船隻串供，统一口径。
 <br/>
 <br/>
 目前，碳九泄漏瞒报、串供事件中的相关责任人已被警方采取刑事强制措施，其中包括东港石化公司法人代表黄某仁等6名企业人员，〝天桐1号〞油轮的值班水手长叶某彪等4名操作人员。
 <br/>
 <br/>
 近70吨毒碳九泄漏引发一场生态灾难，相关海域水体严重污染，当地民众生活受到严重影响，现在民众出行基本还要戴口罩。
 <br/>
 <br/>
 据此前报导，碳九泄漏事件发生后，中共当局掩盖真相，未及时通报泄漏的是有毒化学品，提醒民众防护。当地许多渔民徒手去捞污染物，什么防护措施都没有，有多人身体不适，出现头晕、呕吐、手抽筋等中毒症状，被送往医院就诊。
 <br/>
 <br/>
 据官方此前通报，当地至少有52人身体不适入院，其中一人因污染物导致肺感染，正在重症室抢救。有专家担心，若此次泄漏的是毒性大的〝重整碳九〞，会有中毒、致癌等风险。
 <br/>
 <br/>
 在碳九泄漏事故发生后的第二天，泉港区环保局即发布通告称，碳九泄漏海域清理工作已基本完成，一切正常。事发后，当地泉州电视台的记者在现场进行报导，工作人员一边戴着两层口罩，一边报导说空气质量正常，遭到现场村民斥责。
 <br/>
 <br/>
 一直到8日，当地村民们向外界透露，空气中仍有刺鼻气味，海面上仍漂浮着大量污染物，更有不少村民仍陆续出现呕吐等身体不适现象。
 <br/>
 <br/>
 对于官方的欺瞒做法，当地民众十分愤怒，在网路上发布受污染视频，曝光中共谎言。更有大批民众到当地政府门前抗议。
 <br/>
 <br/>
 据悉，此次碳九泄漏事件污染范围广，周边受到影响的还有肖厝、峰尾、郭厝、南浦等村镇。村民还反映，在泉州、石狮、惠安、莆田等地也能闻到空气中的刺鼻气味。
 <br/>
 <br/>
 另外，此次事故令当地渔民及海产养殖户损失惨重，不少人家几十年积累的鱼排血本无归，损失高达数百万，还有人表示，损失大到已无法估量。
 <br/>
 <br/>
 然而，如此严重的生态事故发生后，福建泉港只有少数官员被要求做检查、或被免职，无一人被追究刑责。有评论认为，中共官方突出涉事企业瞒报碳九泄漏数量，只追究企业员工的责任，却对中共自身在处理公众事件中，一惯的欺瞒做法予以回避。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：祝馨睿)
