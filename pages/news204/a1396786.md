### 胡锡进发视频为新疆集中营洗地 网民骂声一片
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】中共新疆广设〝再教育营〞变相拘禁维族人进行洗脑教育招致国际社会的质疑与谴责。中共党媒的主编
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锡进.html" target="_blank">
  胡锡进
 </a>
 一如既往地在网络社交媒体上发帖为中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗地.html" target="_blank">
  洗地
 </a>
 。这次他抛出了一段视频，宣称是自己亲自在新疆再教育营中拍摄的日常活动情况，再次遭到中国网友一边倒的讥讽和炮轰。
 <br/>
 <br/>
 当地时间10月24日，中共党媒《环球时报》的总编辑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锡进.html" target="_blank">
  胡锡进
 </a>
 在社交媒体微博和推特上，都公布了一段视频，声称那是他近日前往新疆的所谓〝职业训练中心〞了解情况时，拍摄到的营区内的活动情况，他并强调称〝一刀未剪〞。
 <br/>
 <br/>
 视频中，几栋黄色的大楼外的操场上，一群群身穿统一制服的维族女子正在进行一些体育活动——有的在打乒乓球，有的在打篮球，还有一群女子在跳维族的民族舞蹈。
 <br/>
 <br/>
 不过，有更多身穿制服的维族女子在操场边上，姿势规规矩矩地整齐列队，一排排坐在统一的小塑料凳上旁观，操场四周围则圈着绿色的铁丝网墙。
 <br/>
 <br/>
 胡锡进同时发出帖文称，当地的官员告诉他，眼前看到的这些场面是再教育营里〝平常会从事的活动〞，虽然他没办法查证是否属实，但他自己〝持相信的态度〞。胡还特别强调说，〝教育培训中心〞并不像西方国家的媒体所想像的那么糟糕。

 不过，在推特的这段影片下方，有大批网民用中文留言，一边倒地对胡锡进的这番为中共洗地的举动冷嘲热讽或质疑炮轰。
 <br/>
 <br/>
 众网友纷纷留言质疑，〝打球和跳舞的样子都很生疏，这是拿来做做样子的吧？〞〝如果真是欢乐天堂，里面的人可以自由离开吗？〞〝那里是否允许世界各地记者前往自由采访？如果不允许，中共就是在撒谎！〞
 <br/>
 <br/>
 有网友留言议论道：〝这样的场景与中共监狱内的情形十分相似啊。只不过把监狱常用的那种裸露粗糙的铁丝网换成了经过包装的绿网，安排了一些估计是临时性的活动表演罢了。〞
 <br/>
 <br/>
 有网友嘲讽道：〝在祖国监牢里可真欢乐啊！〞、〝把集中营说的这么清新脱俗，这是放风时间吧？〞。
 <br/>
 <br/>
 还有不少网友毫不留情地炮轰胡锡进——〝昧着良心替中共说假话、打掩护，你心中都不会有愧吗？〞〝为中共洗地罔顾事实，你会有报应的。〞〝总有一天你老婆、小孩也都会进去的，你等着。〞
 <br/>
