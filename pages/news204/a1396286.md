### 澳门中联办主任敏感期坠亡 港媒揭5大疑点
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门中联办主任.html" target="_blank">
  澳门中联办主任
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 日前离奇坠楼，因其在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥通车.html" target="_blank">
  港珠澳大桥通车
 </a>
 在即，据传习近平将亲临通车仪式的敏感时间点坠亡，引发外界诸多猜测。有港媒对郑晓松坠楼事件，提出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5大疑点.html" target="_blank">
  5大疑点
 </a>
 。
 <br/>
 <br/>
 10月20日晚，现年59岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 在澳门住所坠楼身亡，由于事发相当突然，且事前没有任何先兆，该事件引发各界热议。
 <br/>
 <br/>
 公开资料显示，郑晓松1959年出生，河北石家庄人，挪威奥斯陆大学挪威语专业毕业，曾在英国牛津大学外交官研究生班学习。
 <br/>
 <br/>
 1993年，郑晓松进入外交部，2000年任职财政部，历任国际司司长及部长助理。2013年任中共福建省委常委、秘书长，2016年7月调任中共中央对外联络部副部长，2017年9月20日接替王志民，任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门中联办主任.html" target="_blank">
  澳门中联办主任
 </a>
 ，晋升正部级。此外，他还是中共第十九届中央委员。
 <br/>
 <br/>
 在郑晓松坠亡的第二天，中共国务院港澳办官网发表声明，称郑晓松患有〝抑郁症〞，引发外界质疑。
 <br/>
 <br/>
 有分析认为，近年来众多离奇坠楼或自杀身亡的中共官员，都被中共官方指患有〝抑郁症〞。而官方对自杀官员的死因，通常都是轻描淡写，调查也讳莫如深，结论更是草率武断。
 <br/>
 <br/>
 对于郑晓松坠亡事件，香港《苹果日报》21日报导，梳理出五大疑点。
 <br/>
 <br/>
 <h4>
  疑点一：中共官媒、澳门警方说法矛盾
 </h4>
 <br/>
 澳门司警局局长薛仲明表示，郑晓松坠楼事件没有疑点，已转交调查科跟进，但未提及中共官媒（港澳办）声称的，郑晓松患〝抑郁症〞的情况。显示官媒和警方说法不一。
 <br/>
 <br/>
 <h4>
  疑点二：
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥通车.html" target="_blank">
   港珠澳大桥通车
  </a>
  在即
 </h4>
 <br/>
 港珠澳大桥23日即将在珠海举行通车仪式，传闻习近平会出席，粤港澳三地高官也将出席。据称，郑晓松已被安排出席，但其却在这敏感时刻坠楼，十分蹊跷。
 <br/>
 <br/>
 <h4>
  疑点三：首名出事的十九届中央委员
 </h4>
 <br/>
 郑晓松是204名第十九届中共中央委员中，首名出事的中央委员。
 <br/>
 <br/>
 <h4>
  疑点四：上任仅一年多
 </h4>
 <br/>
 郑晓松去年9月才获任澳门中联办主任，距离他出事仅1年多时间。他上任后曾公开表示，要〝读好澳门这本书〞。就在郑晓松出事前一天(10月19日），他还在办公室会见澳门民联智库主席施家伦等一行人，并对民联智库的成立表示祝贺，外界没有发现他有任何〝抑郁症〞病发的先兆。
 <br/>
 <br/>
 <h4>
  疑点五：港澳系统贪腐不断
 </h4>
 <br/>
 中共中央巡视前年曾对港澳办党展开专项巡视，发现国务院港澳办部分官员〝出问题〞，包括任用程序不够严格、贯彻中央的决策部署不够扎实。
 <br/>
 <br/>
 在郑晓松坠楼前，在港澳任职长达13年、曾任澳门中联办主任的国务院侨务办公室前副主任李刚，去年8月落马被查。
 <br/>
 <br/>
 据总部设在香港的中国人权民运信息中心透露消息，指郑晓松近日也被中纪委官员问话，这很可能是他坠楼的原因之一。
 <br/>
 <br/>
 报导还称，郑晓松坠楼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5大疑点.html" target="_blank">
  5大疑点
 </a>
 中最大的疑点是，事发后，澳门司警强调案件仍在调查，但港澳办却已经定调，并通报郑晓松是死于抑郁症。澳门警方与港澳办对该案的表态异常，令外界质疑。
 <br/>
 <br/>
 对于郑晓松患
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抑郁症自杀.html" target="_blank">
  抑郁症自杀
 </a>
 的说法，也引起网民质疑。有网民在港媒报导后边留言：〝被自杀?〞〝共产党员，危险行业？〞〝今次晓松下次晓明，哈哈〞，〝党要你消失就消失〞。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：祝馨睿）
