### 美媒：南海前景堪忧 中美随时爆发战争
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 持续升温，美中对抗已蔓延至军事领域。有美媒称，北京和华盛顿之间的对抗存在升级变成更大危机的可能，中美爆发战争只是时间问题。
 <br/>
 <br/>
 美国《纽约时报》11月9日刊文说，美国驱逐舰〝迪凯特〞号(USS Decatur)，在南沙群岛进行旨在捍卫自由航行原则的巡逻时，遭遇中共一艘驱逐舰过分靠近，两舰一度只差41米的距离，几乎造成碰撞，美舰不得不采取紧急行动避免碰撞。
 <br/>
 <br/>
 报导说，这是美国海军对中共在南海的军事集结提出质疑以来，双方最危险的一次接触。9月30日的这场冲突，凸显出美国指挥官所担心的，即在这条存在争议的航道上，双方对峙进入了一个危险的新阶段。
 <br/>
 <br/>
 但在这一个阶段，甚至没有就防止冲突升级的基本行为准则达成冷战式协议。
 <br/>
 <br/>
 澳洲国立大学(Australian National University)南海问题专家布伦丹・泰勒(Brendan Taylor)说：中美双方正在展开一场博弈。冲突发生只是时间问题，这样的事件存在升级变成更大危机的可能。
 <br/>
 <br/>
 中共国防部长魏凤和与美国国防部长马蒂斯(Jim Mattis)预计11月16日在华盛顿会晤，以平息日益加剧的紧张局势，并降低误判风险。
 <br/>
 <br/>
 但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 及美国副总统彭斯(Mike Pence)10月初的演讲中，点名中共在内政外交、军事及宗教信仰领域的不当行径，此次演讲被认为直插中共死穴，且将美中关系带入新冷战时代，双方距离军事冲突似乎只有一步之遥。
 <br/>
 <br/>
 《美联社》10月25日报导，去年才退役美军驻欧洲部队前任司令霍吉斯（Ben Hodges），在欧洲的华沙安全论坛上说，中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军舰.html" target="_blank">
  军舰
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海对峙.html" target="_blank">
  南海对峙
 </a>
 事件，显示两国多方面面对愈来愈多竞争、关系变得紧张。
 <br/>
 <br/>
 他表示，〝你们将看到我们（美国）继续在欧洲投资、训练、轮防以及驻防，以便做好准备，10年或15年之内，我们可能要在太平洋打一场仗。〞
 <br/>
 <br/>
 霍奇斯说，中美关系紧张不仅仅与双方军舰近期在南海险撞有关，这只是紧张关系升级的其中一个讯号，美国还关注中共〝持续窃取技术〞、借投资基建试图对欧洲及非洲进行控制等。
 <br/>
 <br/>
 中美军舰在南海对峙事件不久，10月22日，美国2艘军舰刚刚航经台湾海峡。美国军方说，美舰是〝例行性经过〞国际法允许的国际水域。
 <br/>
 <br/>
 但中共国访部长魏凤10月24日在北京出席香山论坛开幕时致辞时强调，中方反对域外国家，到南海展示武力。中共军队会不惜一切代价采取行动。
 <br/>
 <br/>
 美国政府一位匿名官员说，美国海军所进行的是例行和定期的航行自由巡逻，正如美国过去所做的和未来将继续进行的那样。
 <br/>
 <br/>
 《路透社》认为，这是美国为了抵抗中共试图在南海战略水域限制航行自由，而采取的最新行动。中共、日本和一些东南亚国家的海军，也常常在那里行驶。
 <br/>
 <br/>
 美国批评中共在南海建造人工岛屿，并在岛上设立军事设施。担心这些设施可能被用来限制在南海的航行自由。美国坚持航海自由行动不应该与政治纠缠在一起。
 <br/>
 <br/>
 一系列事件导致中美的紧张关系进一步恶化之际，自10月29日开始，美国联合日本和加拿大在西太平洋继续举行〝利剑〞（Keen Sword）军事演习。
 <br/>
 <br/>
 美国第五航母打击群指挥官卡尔‧托马斯（Karl Thomas）少将在新闻发布会上说：利剑演习〝展示了美国和日本对自由开放印度太平洋地区的坚定承诺〞。
 <br/>
 <br/>
 加拿大驻日本的国防武官、舰长Hugues Canuel表示，加拿大的参与，使得往年的双边演习变成了多边演习。
 <br/>
 <br/>
 除加拿大之外，随着中共在南中国海的军事扩张，英国和法国也在这一海域增加了战舰。澳大利亚和韩国也会派遣观察员观摩此次利剑军演。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
