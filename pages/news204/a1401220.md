### 【禁闻】11月28日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 倪玉兰新居再遭监控
 <br/>
 <br/>
 北京著名维权人士倪玉兰多年来遭警方骚扰、逼迁，无法安居，五年中搬家21次。今年8月被房地产公司拉入〝黑名单〞后，女儿几经周折为她找到住处，最近发现，新住所又被人安装了多个监控探头。
 <br/>
 <br/>
 2. 广州女律师孙世华案两目击证人被批捕
 <br/>
 <br/>
 广州女律师孙世华遭警方殴打、脱衣侮辱一案，再有新的进展。11月28号，大陆民间网站〝民生观察〞发布消息说，该案两名关键目击证人梁颂基和张五洲，目前已被广州市荔湾区检察院批准逮捕，罪名为涉嫌〝寻釁滋事〞。
 <br/>
 <br/>
 3. 大陆政治犯郭泉出狱不久 又遭警方传唤
 <br/>
 <br/>
 大陆知名政治犯、南京师范大学副教授郭泉，被控〝颠覆国家政权罪〞坐牢10年，11月12号刚刚出狱，28号又被警察传唤到警局，至截稿时，仍未释放。据称，警方行为与郭泉在网上发表政见言论〝郭泉语录〞有关。
 <br/>
 <br/>
 4. 中国曹顺利人权团队第17次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上街举牌.html" target="_blank">
  上街举牌
 </a>
 <br/>
 <br/>
 中国曹顺利人权团队的部分人权捍卫者，日前第17次上街举牌，抗议中共当局对良心人士进行随意抓捕、判刑、酷刑折磨、强迫失踪等迫害，呼吁〝解除党禁报禁、立即释放全国政治犯〞。
</div>
