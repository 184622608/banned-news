### 重庆妇女持刀  砍伤14名幼童
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】10月26号上午， 中国重庆市巴南区一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幼儿园.html" target="_blank">
  幼儿园
 </a>
 门口 ，一名中年妇女持刀砍伤14名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幼童.html" target="_blank">
  幼童
 </a>
 ，其中4人伤势严重，现场血迹斑斑，孩子们不断哭喊。凶手引起公愤被民众围住。
 <br/>
 <br/>
 据巴南区公安局通报，10月26号9点半左右，39岁的刘姓女子，在巴南区鱼洞巴县大道一家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幼儿园.html" target="_blank">
  幼儿园
 </a>
 门口持菜刀行凶，导致14名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幼童.html" target="_blank">
  幼童
 </a>
 受伤。
 <br/>
 <br/>
 附近店家26号告诉《新唐人》，那是一家私立幼儿园 ，没有外墙，几名老师带着孩子刚刚做完早操，正领着孩子往回走，一名妇女突然持刀乱砍孩子，很多孩子的头部受伤，有个男孩的半边脸被削掉，凶手引起公愤， 被民众团团围住。
 <br/>
 <br/>
 附近店家刘先生:〝我们这次的事件确实比较大，说是那个女人的老公找了一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小三.html" target="_blank">
  小三
 </a>
 ，小三她的孩子就在那个幼儿园，结果那个女人发现了，就想去找那个女的和那个孩子，结果呢就发生了这件事。这个女的也是报复社会那种，她又不知道孩子是谁就乱砍人。〞
 <br/>
 <br/>
 现场视频显示，多名儿童被砍得满脸、满身是血，地上满是血迹。孩子们哭喊着，在幼儿园门口等待救治。现场民众看到受伤的孩子，都在惊呼。另一个视频显示，有儿童躺在急救床上被医护人员推走，还有两名儿童头部缠着白色纱布。
 <br/>
 <br/>
 当地民众质疑，凶手是怎么出现在幼儿园门口的？幼儿园的保安又在哪里?
 <br/>
 <br/>
 当地民众晏女士 :〝起码是安保方面有一定的问题，具体什么时间出的警，这方面的讯息，我是没有看到。如果是快速反应的话，可能不会造成这么严重的后果。像这样对无辜的人造成伤害，要么是她受到了极大的不公平，报复社会。要么她是精神方面有问题，大概是这两个方向。〞
 <br/>
 <br/>
 公开资料显示，该园幼儿有175人，有17名教职工，另有2名校园保安。
 <br/>
 <br/>
 新唐人记者熊斌、舒灿采访报导
</div>
