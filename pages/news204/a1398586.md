### 新疆逼小学生清查家中〝反党〞物品 网友：中共疯了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】中国新疆地区长期以来社会动荡不安，而中共对该地区维族人的文化、民族风俗等方方面面的严厉管控，使得该地区的民族矛盾更趋复杂化。近日，新疆小学竟然强迫在校小学生签署保证书，回家清查家中的书籍、图片与3C设备是否有违反党的要求。有网民惊叹〝中共疯了！〞
 <br/>
 <br/>
 据台湾《自由时报》报导，新疆乌鲁木齐市第79小学近日向在校读书的小学生下发了一份〝清查家庭藏书及音像视频自查情况反馈及承诺书〞，要求学生们自填班级、姓名，并承诺要清查家里的书籍、图片以及电脑、手机、音像制品等3C产品，并将清查的结果填写在学校统一印制的清单表格中向学校报告。
 <br/>
 <br/>
 这份表单上，明确要求小学生们承诺，家中所藏的书籍及3C产品等都要符合〝党的民族宗教观、历史观〞，以及马克思主义的所谓〝五观〞和新疆当局推行的所谓《纪要》的精神，并承诺家中现有的以及今后购买的书籍及3C产品，都不违背上述要求规定。最后校方还要小学生及其家长都在表单上签字担保。

 有网友惊叹，〝中共彻底疯了，竟然让还没有独立行为管控能力的小学生签保证书回家搞清查行动！〞还有网友在相关报导下留言披露，〝我有个新疆的同学假期回家说了几句不满的话，就被人举报抓进去了。〞还有网友评论称，〝不只是针对少数民族，看起来好像是针对宗教。〞
 <br/>
 <br/>
 事实上，中共地方政府通过学校教育对小学生进行意识形态的强制洗脑是一贯的作为。
 <br/>
 <br/>
 2017年12月下旬圣诞节来临之际，中国各省市的教育系统就曾经发起过一场〝抵制圣诞节〞的文革式政治运动。期间，不少地方小学也曾下发统一印制的保证书，要求小学生签名承诺不参与庆祝圣诞节的活动。从网络上曝光的一些视频片段和图片还可以看到，有的学校让小学生在课室内或操场上，集体高喊〝拒絶洋节，从我做起。〞
