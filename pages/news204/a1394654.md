### 中共降准救市无效！下周恐被列汇率操纵国
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人行.html" target="_blank">
  中国人行
 </a>
 周日宣布降准百分之一，共释放资金1.2兆元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 ，不过，昨天中国A股三大指数齐跌，都超过3%，对此，专家表示，看得出来，中国对于不利情势，呈现紧张。下星期，美国财政部将公布外汇报告，中国很可能被列入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率操纵国.html" target="_blank">
  汇率操纵国
 </a>
 。
 <br/>
 <br/>
 中国人民银行日前宣布，15日起下调部分金融机构存款准备率1个百分点，估计可释出7500亿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 ，约新台币3兆元的资金。
 <br/>
 <br/>
 新加坡银行固定收益主管 Gareth Nicholson：〝中国有点紧张，正面临不利的情势。我认为美中贸易谈判仍会有好消息，在中期选举前。但现在中国得做最坏打算，降低银行存款准备率，的确能缓和一些情绪。〞
 <br/>
 <br/>
 这是人行今年第4度降准，人民币兑美元今日持续下跌。尽管
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 多次宣称不怕打贸易战，上个月发表的《关于中美经贸摩擦的事实与中方立场》白皮书中，还提到中国经济对外部冲击具有高度弹性，但从降准的决定来看，中共是真的紧张了。
 <br/>
 <br/>
 中国金融体系分析师 Fraser Howie：〝中共当然想控管引发紧张、或近似紧张的言论，这是他们一向在做的。但现在中国的情况已经不寻常，目前正面临2008年全球金融危机以来，最糟的时刻。〞
 <br/>
 <br/>
 而人民币贬值，也让美国总统川普非常不满。彭博新闻报导，美国财政部长梅努钦，预计在下周提出的外汇报告中，决定是否将中国列入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率操纵国.html" target="_blank">
  汇率操纵国
 </a>
 中。过去半年中，人民币兑美元下滑约9%，是亚洲表现最差的国家之一。
 <br/>
 <br/>
 新唐人亚太电视综合报导
</div>
