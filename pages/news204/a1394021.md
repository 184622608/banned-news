### 升中共党旗惹众怒  台湾〝五星庙〞被夷为平地
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】历经7天拆除作业后，自称〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五星庙.html" target="_blank">
  五星庙
 </a>
 〞和〝中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_思想教育基地.html" target="_blank">
  思想教育基地
 </a>
 〞的台湾彰化
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碧云禅寺.html" target="_blank">
  碧云禅寺
 </a>
 ，于2日被夷为平地。彰化县长魏明谷表示，在这里建立中共思想教育基地，扰乱附近居民安宁。
 <br/>
 <br/>
 彰化县政府自9月26日上午10时，办理强制拆除二水
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碧云禅寺.html" target="_blank">
  碧云禅寺
 </a>
 违章建筑作业。经挖土机等大型机具，连续7天的拆除作业，至10月2日〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五星庙.html" target="_blank">
  五星庙
 </a>
 〞的最后一道墙终于倒下。
 <br/>
 <br/>
 彰化县长魏明谷前往现场视察，向在场的每一位参与拆除作业的工程人员，以及在这礼拜轮流看守现场人员安全的警察人员慰问。他也安慰被〝五星庙〞主人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魏明仁.html" target="_blank">
  魏明仁
 </a>
 赶出的住持与比丘尼。
 <br/>
 <br/>
 魏明谷说，魏明仁在这里建立中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_思想教育基地.html" target="_blank">
  思想教育基地
 </a>
 ，扰乱附近居民安宁，他的发言也损坏国家尊严，破坏人民感情。他说，〝这一面墙拆除之后，我们希望回归宁静，回到碧云禅寺香火鼎盛的荣景。〞
 <br/>
 <br/>
 他还赠送〝瓦当〞给碧云寺住持释怀宗留念，县府也答应未来将协助碧云禅寺后续工作。
 <br/>
 <br/>
 魏明谷说，庙拆了，百万人民币的拆除费，彰化县政府铁了心要找魏明仁算总帐。〝逃得了和尚，逃不了庙〞，就算魏明仁上演〝失踪记〞，产权也移交给胞姐，但是土地可以执行拍卖，县府一定会依法追讨到底。
 <br/>
 <br/>
 建设处表示，有关拆除费用，县府于10月2日已向所有权人寄发，限10月15日前缴纳处分书，倘若所有权人逾期未缴纳，将移送行政执行署强制执行。
 <br/>
 <br/>
 拆庙的几日内，魏明仁行踪成谜，他在寺庙升起中共党旗，还有统派团体在台北最热闹的西门町悬挂中共党旗。民进党立委蔡易余认为，台湾应该对这些言论和行为有所节制。
 <br/>
 <br/>
 蔡易余说，他们滥用台湾的民主自由，宣传中共的言论，现在跑到对岸变成了所谓的〝民族英雄〞的形象。这都伤害台湾的民主跟自由，中共用最廉价的方式，资助台湾反对民主自由、反对台湾主权独立性的团体，只会激起更多民众的反感。
 <br/>
 <br/>
 对于魏明仁来说，不只面临巨额的拆除费用求偿，彰化县地检署也来补刀。魏明仁因为在拆除过程殴打公务人员，10月2日，地检署已对其公然挑战公权力的犯罪行为，侦查终结提起公诉。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
