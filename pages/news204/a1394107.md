### 【禁闻】中共称降关税为人民 专家：零关税才对
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 喉舌媒体日前报导，在半年之内连降四次关税，声称是为了企业发展和人民的生活。但经济学家认为，如果真为老百姓，应该实行零关税。
 <br/>
 <br/>
 今年3月份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 降低关税，以求公平贸易后，得到的是北京对美国贸易代表的傲慢和不怕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的强硬回应，以及对美国产品加征关税的报复行动。
 <br/>
 <br/>
 10月2号，中共喉舌《人民日报》却报导称，中共从今年5月份开始出台了四次所谓的〝自主〞降低关税的措施，11月1号起还将降低1585个税目的进口关税。声称这是为了人民的美好生活、助力产业升级等等。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝这个（降低关税）就是中共比较愚蠢的地方，等于它在反击美国的时候那些子弹都打飞了，打空了，想降低关税，真想对中国老百姓有利，接受到更好的产品的话，只要答应美国的条件就好了，中美贸易战就没有了，现在做出一些姿态，但是又不接受美国的条件，就是处于一种两难的地步。〞
 <br/>
 <br/>
 加州大学洛杉矶分校（UCLA）经济学家俞伟雄认为，中共加征美国产品关税造成中国企业生产成本攀升，这是北京屡
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降关税.html" target="_blank">
  降关税
 </a>
 的真正原因。
 <br/>
 <br/>
 加州大学洛杉矶分校（UCLA）经济学家俞伟雄：〝中国现在降关税，就是美国政府要中国要做的，只不过呢，只做了一半而已，因为它并没有对美国降。〞
 <br/>
 <br/>
 除了美国开始加征关税前5月份的第一次降低关税外，中共其他的降关税，都是伴随着对美国产品增加关税进行的。俞伟雄分析说，中共这种报复性的降低关税政策，会在川普加快和其他国家签订贸易协定的过程中，使自己越来越孤立。
 <br/>
 <br/>
 美国政府的贸易行动针对的不只是中国，因为美国长期以来允许全球各国产品低关税或零关税进入美国，而美国产品进入其他国家往往要缴纳数倍甚至数十倍的关税，这种不对等贸易方式已经对美国经济造成严重打击，川普上台后要求纠正这种错误。
 <br/>
 <br/>
 目前，很多经济体已经意识到川普政策的正确性，开始跟美国合作。日前美墨加三国达成的新的北美贸易协定，更是被认为是美国与其他国家贸易协议的〝模板〞。其中的一项内容规定，如果任何协议方擅自与〝非市场经济〞国家签订贸易协定将可能使与美国的贸易协议自动作废。
 <br/>
 <br/>
 而中国经济早已被世界绝大多数经济体认定为非市场经济。
 <br/>
 <br/>
 同时，俞伟雄也认为，中共的降税措施也并不能够解决中国经济当前面临的困境，因为美国提高中国产品关税正在加快外企的撤资潮。
 <br/>
 <br/>
 根据大陆〝财新网〞的报导，中国9月份的制造业产出增长速度降到了2017年10月以来最弱，出口订单也加速下挫，已经降为2016年2月以来最低。
 <br/>
 <br/>
 外界认为，这说明贸易战给中国经济带来的后果，已经开始显现。
 <br/>
 <br/>
 至于中共声称的降税是为了〝人民的美好生活〞，贵州大学经济学院教授杨绍政指出，真正影响中国人美好生活的是中共本身，而不是关税。
 <br/>
 <br/>
 贵州大学经济学院教授杨绍政：〝全体国民的公款供养了政府人员以外，我们还供养了全世界不需要全民公款供养的中国共产党各级党委系统，我们还供养了各级民主党派，非政党的社团组织，比如说红十字会、文联、作协、妇联、共青团、公会、宋庆龄基金会等等这一系列东西，这些东西给全民带来了负担。〞
 <br/>
 <br/>
 清华大学荣誉教授李志文，早前接受新唐人记者采访时也曾指出，中国的关税其实只与太子党的利益有关，对老百姓而言，零关税最好。
 <br/>
 <br/>
 采访编辑/刘惠  后制/肖颜
