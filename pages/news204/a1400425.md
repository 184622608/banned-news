### 【禁闻】中共违反〝世贸〞条款 WTO介入调查
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】关于美、中两国能否通过双边谈判，或通过改革世界贸易组织，来解决两国的贸易争端？世界贸易组织（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 ）同意建立一个争端小组，对美国提出的指控作出裁决，确定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 政策是否违反WTO条款。预计明年可以作出裁定。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 加入世界贸易组织（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 ）后，由于违反规则和不履行承诺，引发国际社会不满。
 <br/>
 <br/>
 11月21号，在瑞士日内瓦举行的世界贸易组织会议上，美方代表指中国利用WTO推行非市场化政策，扭曲全球市场，导致大量产能过剩，尤其是钢铁和铝。
 <br/>
 <br/>
 对此，世界贸易组织同意建立一个争端小组，对美国提出的指控作出裁决，确定中共政策是否违反WTO条款。预计明年可以作出裁定。
 <br/>
 <br/>
 时事评论员田园博士认为，这种所谓的调查，除了消耗时间和金钱，并没有实质意义，因为既使有了仲裁结果，WTO也没有能力执行。
 <br/>
 <br/>
 时事评论员田园博士：〝在2008年，世界贸易组织就曾经仲裁过一次，中、美之间有关知识产权盗窃这方面的问题。最后WTO世界贸易组织作出裁判说，确实是中共侵犯了美国和其它西方国家的知识产权，这个世界贸易组织它自己没有任何的执行能力，它没有这个能力，也就是最后世界贸易组织变成了一只没有牙齿的老虎，所以它对任何人都没有威慑力的。〞
 <br/>
 <br/>
 中共坚持用〝发展中国家〞的理由，来推卸应该尽到的义务。尽管被其它国家多次投诉，但是WTO的职能只是调解纠纷。田园博士表示，10年过去了，美国每年仍然在知识产权和网络攻击方面，损失千亿美元，甚至上万亿美元。
 <br/>
 <br/>
 美国总统川普的经济顾问哈塞特接受BBC专访时表示，中国从加入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世贸.html" target="_blank">
  世贸
 </a>
 组织以后，就一直〝行为不检〞。哈塞特还说，要纠正中共政府不当贸易行为有3种方式，包括透过双边协商、改革WTO，或剔除中国在WTO会籍身分。
 <br/>
 <br/>
 哈塞特表示，WTO在帮助全球现代化方面，发挥了非常重要的历史作用，但在许多方面令美国失望。美国通常能够赢得提交给WTO的案件，但〝需要5到6年〞，那时损失早已造成。哈塞特说，对中国施加关税的目地，是在国内造成最小伤害的情况下，给中国带来最大的压力。川普总统在国际贸易方面的强硬战略正在发挥作用。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝川普对WTO采取的策略，跟他一贯的性格很像，就好比几个玩家打牌，其中一个玩家老出千，怎么办呢？第一招，就是惩罚这个玩家，把他制约住，再不行就干脆把他踢走，还不行吗，就把牌局整个推翻，另起炉灶。〞
 <br/>
 <br/>
 欧盟贸易专员马姆斯特罗姆（Cecilia Malmstrom）11月16号出席巴黎的一个会议时表示，中国（中共）必须在改革WTO的谈判中做出让步，否则就有可能看到美国放弃现行体制。
 <br/>
 <br/>
 田园博士表示，世界贸易组织已经陷入僵局，无法再继续前进。而川普政府的战略重心，早已经从世界贸易组织中转移。
 <br/>
 <br/>
 田园博士：〝最近刚通过的美、加、墨的贸易协定，就是一个很好的例子。所以在这样的条件下，这样的背景下，大家可以看到，WTO世界贸易组织它存在不存在，除了对中国这样的国家，还有像印度这样的国家还有意义之外，因为这些国家它能够利用WTO的制度漏洞去钻空子。〞
 <br/>
 <br/>
 田园博士分析认为，如果在美国的主导下，世界上主要贸易国家能够通过双边贸易谈判来解决多边贸易谈判不能解决的问题，那么世界贸易组织的存在，只能说是名存实亡。在这种情况下，如何改革世界贸易组织，如何加强世界贸易组织，其实是没有任何意义的。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/王明宇
