### 平度老兵维权第6天 300老兵饥寒交迫
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东平度.html" target="_blank">
  山东平度
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵维权.html" target="_blank">
  老兵维权
 </a>
 事件10月10号进入第6天，目前平度依然戒严，到处是特警，大部分老兵已被控制。前一晚传出，平度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 刘兰香到场探望老兵被拘，还有300名左右的老兵在场抗争，饥寒交迫。另外，9号内蒙古和江西都传出有老兵集体维权。
 <br/>
 <br/>
 老兵10月9号晚上传来最新的平度消息说，武警部队实行宵禁戒严，交通一级管制，各省援军被驱平度之外，平度人民群众送衣捐食以〝寻釁滋事〞罪名受刑，无人能接触
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退役军人.html" target="_blank">
  退役军人
 </a>
 ，致使里面战友饥寒交迫，众多伤病员，他们手机电池也已耗尽，讯息中断，目前尚剩七省300左右战士在场抗争。
 <br/>
 <br/>
 平度市民10号上午告诉《新唐人》，目前平度依然到处是特警，所有外地车都不让进。
 <br/>
 <br/>
 平度市民李女士：〝带了吃的给老兵，他们不让进，我们靠不了老兵的跟前，全部被控制，我们大街上见不到，大部分是被关起来啊，卡在外头，进不来了，要进平度拦截了，很不讲理，有一些已经被遣送回去，有一些关在西关小学、奥体中心。〞
 <br/>
 <br/>
 8号上午，平度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 刘兰香到老兵聚集静坐的广场去探望，被抓到派出所拘留。
 <br/>
 <br/>
 刘兰香丈夫史先生：〝这段时间
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵维权.html" target="_blank">
  老兵维权
 </a>
 在平度，她就去围观，在现场被当地警察带到派出所拘留，罪名是‘寻釁滋事’，刘兰香没什么过激的行为，从6号开始的，她就一直过去围观，期间也给老兵送过水。〞
 <br/>
 <br/>
 10月4号，38名平度老兵进京途中被当地公安暴力截访，6人住院，各地老兵连日赶到平度讨说法，三度遭到暴力清场。在平度被抓的参战老兵齐宝福传短信说，在平度受到法西斯待遇，两三人一间房，四面墙瓷砖地面，没吃没喝没铺设盖，席地而睡第二天爬都爬不起来。
 <br/>
 <br/>
 近来，大陆各地老兵掀起维权大潮，不断有老兵大规模聚集要求解决待遇和安置问题，9号传出内蒙古自治区政府大楼前人行道上有数千名退伍老兵维权，同一天在江西省政府大门前也有老兵集体维权。
 <br/>
 <br/>
 新唐人记者熊斌、元真采访报导
