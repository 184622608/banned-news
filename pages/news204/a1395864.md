### 崔永元曝上海警局黑幕：高层受贿百万名表
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】央视前主持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 持续〝开火〞，18日他又爆料称，当天投资控股公司董事长邵永华，是快鹿集团专门负责和上海警方以及警局高层维护关系的人，〝没少花钱，送礼的手表都是几十万上百万一块〞。
 <br/>
 <br/>
 10月18日凌晨，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 在其微博〝小崔读书汇〞发消息，询问上海警方〝警民直通车〞，有关〝彭奋、彭明达〞的事情有没有查清楚。 根据崔永元此前的爆料，彭奋、彭明达系父子关系，被指参与了电影〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大轰炸.html" target="_blank">
  大轰炸
 </a>
 〞。
 <br/>
 <br/>
 大陆网媒将彭奋、彭明达父子与进行非法金融活动的上海快鹿集团案联系在一起。据披露，彭明达目前身在新西兰，任新西兰中文电视台NCTV总裁台长。而彭奋是上海长宁区经侦队队长。
 <br/>
 <br/>
 彭奋是否对儿子的违法涉案问题，存在包庇和庇护引舆论质疑。
 <br/>
 <br/>
 崔永元在微博提供一个线索说，去找快鹿集团要角之一〝邵永华〞，他是专门负责和上海警方以及警局高层维护关系的人，〝没少花钱，送礼的手表都是几十万上百万一块〞。邵永华两年前因为公司恶性倒闭，卷款15.75亿元潜逃后失联。
 <br/>
 <br/>
 崔永元还称，上海公安局经济犯罪侦察大队〝陷在这个案子里的人太多了〞，应该让江苏省公安厅网络安全保卫总队来查。
 <br/>
 <br/>
 因崔永元爆料电影〝大轰炸〞签〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞，女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 3日被罚款超过8.8亿元人民币后，〝大轰炸〞原打算10月26日重新上映。
 <br/>
 <br/>
 由于崔永元9日公开呼吁民众抵制看《大轰炸》，并直指该片使用不明资金至少30亿，都是上海民众的养老钱，有人因看不到希望而自杀。
 <br/>
 <br/>
 该片导演萧锋17日在微博宣布该片取消上映，他还暗批崔永元，〝报应定会终归其位〞。
 <br/>
 <br/>
 崔永元随后发文说：取消上映《大轰炸》不是目的，要彻查《大轰炸》是怎样欺诈老百姓的血汗钱的，相关部门一查到底。他还宣示决不因此松手。
 <br/>
 <br/>
 电影《大轰炸》是由中国电影股份公司、原画（北京）影业、上影南国影业出品的历史战争片。由施建祥旗下的上海快鹿集团投资，获中共军方鼎力支持拍摄，更一度被官方列为重点电影。
 <br/>
 <br/>
 据陆媒报导，上海快鹿投资集团董事局主席施建祥是《大轰炸》的总制作人。该集团因涉及集资诈骗、非法吸收公众存款，已有多人遭拘捕、起诉。逃亡海外的施建祥也遭到通缉。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
