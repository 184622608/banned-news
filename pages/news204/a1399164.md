### 北京谈判方案出台？刘鹤即将访美为川习会铺路
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】11月底中美首脑会晤在即，虽然双方已重启贸易对话，但目前仍无任何进展。美媒指，中方首席谈判代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 尝试与美国财长通话无果。港媒消息，刘鹤即将亲自赴美磋商。
 <br/>
 <br/>
 香港《南华早报》11月13日引述三个消息来源指，中共国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 即将赴美，试图缓解中美贸易紧张局势，为G20峰会上的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晤铺路。
 <br/>
 <br/>
 不过消息称，刘鹤此行的时间表尚未确定。
 <br/>
 <br/>
 一名中共政府智库的人士表示，虽然现在谈判气氛有所缓和，中方比较乐观，但是对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府的态度没有把握。
 <br/>
 <br/>
 据美国《华尔街日报》13日报导，为准备川习会，刘鹤与美国财政部长姆努钦(Steven Mnuchin)已经恢复对话，尝试讨论一个缓解贸易战的协议。
 <br/>
 <br/>
 报导引述消息指，11月9日，两人已经通过电话，但没有取得任何突破。美方依然坚持要求中方必须在正式谈判前先拿出具体提议。而中方担心这样会失去谈判筹码，因此希望首先谈判，之后再拿出正式提议。
 <br/>
 <br/>
 据此前美媒报导，美方已经开出重启谈判的条件，就是中方要首先回应美方的贸易要求，拿出令美方满意的行动方案。否则11月底的川习会可能剔除贸易议题。
 <br/>
 <br/>
 分析家认为，刘鹤作为习近平指定的中美贸易最高谈判代表，不太可能空手赴美。而相关信息也显示，虽然美国总统川普表达了对川习会的乐观态度，但美方的立场并无松动。
 <br/>
 <br/>
 本月初的美国中期选举后，川普所在的共和党保住了参议院，但失去了众议院的多数党地位。不过，强硬反击中共已是美国朝野和两党的共识，而且众议院的民主党领袖佩洛西(Nancy Pelosi)更是被中共痛恨的鹰派人物。外界普遍认为，这种新的两党制衡局势，将进一步降低川普政府与中方〝和解〞的可能性。
 <br/>
 <br/>
 有外媒分析，中美和解的唯一可能是中方顺应美方提出的公平贸易要求。
 <br/>
 <br/>
 不过，北京目前依然强调美方要〝尊重中国选择的发展道路〞，显示中方虽然有所妥协，但在〝中国制造2025〞等核心问题上仍然不愿让步。
 <br/>
 <br/>
 美媒引述分析指，即使这次中美能达成一个临时性协议，也会与美朝川金会〝共识〞一样难以持久。
 <br/>
 <br/>
 今年6月，美朝首脑在新加坡会晤，发表了〝朝鲜无核化〞声明。不过，美方虽然一度暂停美韩军演，但并没有因此取消对朝制裁。在发现朝鲜推诿敷衍无核化进程后，美方再次搁置美朝对话，并重启了美韩军演。
 <br/>
 <br/>
 日前，白宫贸易高级顾问纳瓦罗（Peter Navarro）表示，中共一贯言而无信，所以即使美中达成协议，也无法相信中共能够真正执行。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
