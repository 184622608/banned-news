### 平度老兵维权第4天 局势紧张
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】山东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平度.html" target="_blank">
  平度
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵维权.html" target="_blank">
  老兵维权
 </a>
 事件已经进入第4天，10月8号，大陆各地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 仍在向平度集结，中共当局也在继续向当地增派警力。事件还在升级之中。
 <br/>
 <br/>
 10月8号，安徽退伍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 李先生告诉新唐人，事件的起因是，10月4号，38名山东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平度.html" target="_blank">
  平度
 </a>
 的退伍老兵，准备进京上访，但被当地政府派出的警员在一个高速公路的服务区拦截，警方10多人对付一人，将老兵暴力抓捕，过程中，很多老兵被打伤。
 <br/>
 <br/>
 事件引发全国老兵愤怒，各地老兵迅速向平度集结，6号早晨达到千人。下午，双方爆发激烈冲突，警方释放催泪弹、老兵则用灭火器和木棍还击。晚间，警方全面清场，坚守现场的老兵被冲散。
 <br/>
 <br/>
 安徽退伍老兵李先生：〝有的战友被打伤了，在平度的医院里面，躺了一些战友，有的是骨折、而且有的是粉碎性骨折。几十个人肯定不止嘛，因为当初38个人，不同程度的都受伤了呀，而且最近几天又有受伤的，肯定几十个人不止。〞
 <br/>
 <br/>
 现场老兵7号传出消息说，早晨8:00多，广东、河南等地声援老兵赶到，与之前被冲散的老兵集合后走上平度大街，向市政府广场进发，警方一路跟随戒备。队伍在行进中多次遭到警方冲击，老兵队伍利用木棍反抗突围。
 <br/>
 <br/>
 安徽退伍老兵李先生：〝我们知道应该目前在平度（老兵）有1000或者是2000人左右，路上的更多，路上的都被各地围追堵截。昨天突然增加了警力，而且比以前更多，把现场围得是里三层外三层的。〞
 <br/>
 <br/>
 到7号晚间，聚集的老兵更多，而警方除了之前调动青岛周边多个地市的警力外，还在不断增援。现场还出现大量武警车辆及大批武警，部分车辆车顶配备重型机枪，气氛十分紧张。
 <br/>
 <br/>
 上海退伍老兵陈先生：〝它现在是封锁外面的不让进，全国各地的都过去了嘛，它就是说外围的不让进，里面呢，基本上也不动。在高速公路上，在各个地方都在拦截嘛，不让外面的人往里面走嘛。〞
 <br/>
 <br/>
 8号早晨，网上传出一份平度保安公司的紧急通知，要求全体集合，去现场与所谓的〝邪恶势力〞斗争。
 <br/>
 <br/>
 新唐人记者熊斌、施宜君采访报导
