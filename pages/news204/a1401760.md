### 恐龙为何灭绝？网友从四川找到〝答案〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】近日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四川自贡市.html" target="_blank">
  四川自贡市
 </a>
 一条景观大道摆放了许多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_仿真恐龙.html" target="_blank">
  仿真恐龙
 </a>
 ，供市民观赏、拍照。不过，这些恐龙不久都被损毁，有网友调侃说，〝恐龙是如何灭绝的？这个谜终于解开了！〞
 <br/>
 <br/>
 自贡市大山铺恐龙化石群是中国大陆、以至世界上重要的古生物化石埋藏地之一，自贡自上世纪70年代发现恐龙化石后，恐龙便成为了这个城市的象征，该市被誉为〝恐龙之乡〞。
 <br/>
 <br/>
 在自贡市的一条景观大道上，摆放有大大小小不同体积的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_仿真恐龙.html" target="_blank">
  仿真恐龙
 </a>
 ，成为城市一大亮点。
 <br/>
 <br/>
 11月28日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四川自贡市.html" target="_blank">
  四川自贡市
 </a>
 爆料网民称，途径这条景观大道时，看到这些恐龙都被损毁。有家长带着孩子爬上恐龙拍照，也有成年人〝骑龙留影〞。
 <br/>
 <br/>
 由于仿真恐龙表面材质是人造革皮的，里面塞满海绵，所以游客的攀爬、拉扯行为很容易造成毁坏。
 <br/>
 <br/>
 据网民曝光的现场画面可见，所有的恐龙都有不同程度损毁，有些尾巴折断、有些脖子开裂、有些腿部缺失，里面的金属支架也外露。
 <br/>
 <br/>
 事件曝光后，有网民调侃称，〝自贡恐龙是如何灭绝的？灭绝之谜终于解开了！〞
 <br/>
 <br/>
 据爆料网友称，现场有禁止攀爬的提示牌，但游客均视而不见。〝其实不止我一个人进行阻止，也有人委婉地提醒，小孩子爬太高很危险，但还是有人不顾劝阻骑在恐龙上拍照〞。
 <br/>
 <br/>
 还有当地工作人员无奈地表示，被破坏的仿真恐龙是去年打造的恐龙灯饰群，但修复的进度，赶不上破坏的速度。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
</div>
