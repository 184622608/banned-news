### 港媒：习近平女儿习明泽改名姓瞿 重返哈佛读博
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 女儿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习明泽.html" target="_blank">
  习明泽
 </a>
 ，4年前在美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈佛.html" target="_blank">
  哈佛
 </a>
 大学取得学士学位后回国。9日，港媒爆料，习明泽现在又返回母校攻读
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博士学位.html" target="_blank">
  博士学位
 </a>
 ，改名姓瞿，但不知是否继续深造
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心理学.html" target="_blank">
  心理学
 </a>
 。
 <br/>
 <br/>
 香港苹果日报9日引述爆料称，从美国著名学府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈佛.html" target="_blank">
  哈佛
 </a>
 大学知情教授得知，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的独生女儿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习明泽.html" target="_blank">
  习明泽
 </a>
 ，2014年在哈佛大学
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心理学.html" target="_blank">
  心理学
 </a>
 取得学士学位后，现在又返回母校攻读
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博士学位.html" target="_blank">
  博士学位
 </a>
 ，但未知是否深造心理学。
 <br/>
 <br/>
 报导说，习明泽在美留学改名姓瞿，是她的母亲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭丽媛.html" target="_blank">
  彭丽媛
 </a>
 所取，来自彭的外公姓氏。
 <br/>
 <br/>
 此前，海外中文媒体一度传出，习近平上台后，下令中南海政要把在美留学的子女全部召回国，爆料称，这一消息不实，至少习明泽一直在2014年大学毕业后才回国，而中央警卫局曾派人到哈佛大学为习明泽当保镳，美国联邦调查局也曾派探员暗中保护。
 <br/>
 <br/>
 不过习明泽重返哈佛攻读博士消息并未得到其他渠道证实。
 <br/>
 <br/>
 早在2015年4月，美国《纽约客（The New Yorker）》杂志曾报导，习近平女儿习明泽，曾到美国就读哈佛大学，而哈佛大学校内，知道习明泽身份的不会多于十个人，显示习公主〝超低调〞。就算她从哈佛大学毕业回国后，知道的人也极少。
 <br/>
 <br/>
 消息称，出生于1992年的习明泽是从哈佛大学毕业被召回国的，回到中国后，参与辅佐父亲。她运用自己在哈佛学到的知识，提出谘询意见，尤其偏重塑造和维护父亲的形象。
 <br/>
 <br/>
 公开资料显示，习明泽小名叫〝木子〞，大名习明泽据说是习近平的父亲习仲勋给起的，〝希望她将来清清白白的做人，做个对社会有用的人。〞但习明泽到哈佛念书时，取了个化名叫〝楚晨〞（Chen Chu）。
 <br/>
 <br/>
 消息人士还称，目前在网上流传不少〝习近平爱女〞照片，但是成年照片中，无一是真正的习明泽，只有数张她幼年时的照片是真的。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：李泉）
