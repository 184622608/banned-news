### 讨好俄国？中共封杀揭沙俄侵华史微信公号
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 中，中共陷入空前孤立，极力拉拢美国可能的〝盟友〞。日前，揭露沙俄侵华历史的微博公众号突然被封杀，显示中共欲与俄罗斯抱团取暖的意图。
 <br/>
 <br/>
 据自由亚洲报导，周四（10月4日），一家名为〝俄罗斯苏联侵华史与国际关系〞的微信公众号被完全封号。
 <br/>
 <br/>
 该公号被删除的多篇文章，主要叙述〝八国联军〞时期，俄国在中国东北大肆烧杀抢劫，强行霸占了江东六十四屯。这片领土包括为人熟悉的海参威，面积相当于43个香港，又或者225个澳门。
 <br/>
 <br/>
 根据历史资料，当年清政府当年在八国列强逼迫下签订〝辛丑条约〞，其中俄国向清政府敲诈的赔款是八国之最，本息高达三万万两白银。
 <br/>
 <br/>
 此前网络公布的中国清朝疆域图显示，北部红色部分除外蒙古外都是被俄国侵占。其中大部分在江泽民当权时期以秘密协议再次得以确认，中共正式放弃追讨这些领土。
 <br/>
 <br/>
 上述
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微信公号.html" target="_blank">
  微信公号
 </a>
 被删，显示在中共当局高调渲染对俄友好之际，宣传部门对涉及两国历史的网络舆论强化了甄别。
 <br/>
 <br/>
 另外，上个月，正在中共党媒炒作所谓瑞典警察〝侮辱中国游客〞时，有中国女游客在大陆社交媒体发文，要求中共官方为俄国列车常年勒索中国游客〝主持公道〞，结果很快被删号。
 <br/>
 <br/>
 虽然中共正在积极地〝联俄抗美〞，但多名专家指，俄罗斯对中共和美国都怀有数十年的敌意，不会轻易倒向任何一方。中、俄、美之间只能是三角关系。
 <br/>
 <br/>
 日前，俄罗斯和蒙古启动联合军演，俄罗斯专家称军演的目标就是中共。而此前俄媒披露，为防止中共山寨技术，俄罗斯出口中国的苏35战机，发动机都是牢牢焊死的。
 <br/>
 <br/>
 日前，为了拉拢日本抵制美国，中共也对中日历史进行了〝过滤〞。
 <br/>
 <br/>
 8月，福建省官方下令，严禁中国渔船靠近钓鱼岛。上海师范大学原定8月10日举办关于日军慰安妇问题的国际座谈会，也被中共官方紧急叫停。
 <br/>
 <br/>
 不过，虽然中共极力促成日本首相安倍晋三10月访华，但日本9月底依然和美欧发表了直指中共贸易政策不公的联合声明，并答应启动美日自贸谈判。中共〝联日抗美〞的企图基本落空。
 <br/>
 <br/>
 日媒消息指，中共日前以〝不方便〞为由，将安倍原定23日访华的日期向后推迟一两天。10月23日是《中日和平友好条约》生效40周年，中共临时变卦避开这一天，释放的信息非常明显。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
