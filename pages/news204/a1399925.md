### 新疆前法官向百姓普法获刑10年
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】新疆退休法官黄云敏因向百姓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_普法.html" target="_blank">
  普法
 </a>
 被判刑10年，日前，他的妻子前往监狱探视，目睹黄云敏遭到狱方殴打，会见也被强行结束。家属还透露黄云敏的弟弟黄晓敏也因替哥哥说话，被关押一年多。
 <br/>
 <br/>
 据《维权网》报导，11月15号，黄云敏的妻子前去监狱探视会见，黄云敏戴着手铐脚镣前来，夫妻会见仅一分钟时间，就被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狱警.html" target="_blank">
  狱警
 </a>
 中断。
 <br/>
 <br/>
 狱警还当着黄妻的面，卡住黄的脖子，殴打他。并警告黄妻，不准向外界透露黄云敏遭到殴打的情况。
 <br/>
 <br/>
 黄云敏的哥哥黄敏青：〝前两天他的妻子去会面，大概就说了一分钟、一分半钟的话，就因为他提出来了，我不服这个判决，就说是网络不好了，就断掉了，他本来就有心脏病，我说把这个医院开具的证明，现在想送进去都送不进去。〞
 <br/>
 <br/>
 黄云敏是新疆生产建设兵团农三师图木舒克市的退休法官，因帮助农三师农场职工维权，2017年3月12号被以〝煽动民族仇恨，民族歧视罪〞拘捕。案件被检察院退回一次，换法官后，于9月19号在喀什农垦法院开庭，黄云敏要求律师和家属到庭，遭到法官拒绝，家属质疑庭审程序违法，而且一审和二审，家属都没收到任何文件，判决书也没拿到。
 <br/>
 <br/>
 黄敏青：〝17年开庭以后，当时检察官说了，并不是我们要想抓你，而是上面领导不放过你，让你一听就是他并不是因为犯罪被抓，而是因为领导要报复，因为他帮忙上访户，所告的就是这些贪官，律师就提出来质疑了，就撤诉，后来这个案子又重新起诉，还是这个罪名。〞
 <br/>
 <br/>
 家属透露黄云敏的弟弟黄晓敏也因替哥哥说话，去年5月18号被捕，以涉嫌〝寻釁滋事〞的罪名，一直被关押在成都看守所。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
