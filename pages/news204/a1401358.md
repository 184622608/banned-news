### 【禁闻】11月29日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 潜江
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 帅仁兵
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进京上访.html" target="_blank">
  进京上访
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遭遣返.html" target="_blank">
  遭遣返
 </a>
 <br/>
 <br/>
 湖北潜江访民帅仁兵，11月29号在北京中南海附近给中共领导人邮递信件时，被警察拦截送往黑监狱马家楼关押，之后被潜江驻京办主任接出，雇佣3人押送他回潜江。
 <br/>
 <br/>
 2. 江苏访民上访遭判刑 工资被取消
 <br/>
 <br/>
 江苏徐州访民吴继新2016年因为进京上访被判刑两年，今年8月29号刑满出狱，之后他发现自己的工资被当地政府取消，失去生活来源。11月28号，他又到徐州市信访局上访，要求恢复工资，但没有结果。
 <br/>
 <br/>
 3. 东莞胶带公司百名工人拉横幅讨薪
 <br/>
 <br/>
 广东省深圳市东莞迈科胶带公司拖欠员工工资半年，11月28号，一百多名员工到该公司最大的股东深圳坪山沃尔核材厂，在厂区拉横幅讨薪，大批警察手持盾牌到场戒备。
 <br/>
 <br/>
 4. 大连市大化集团
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欠薪.html" target="_blank">
  欠薪
 </a>
 5月 工人抗议
 <br/>
 <br/>
 辽宁省大连市大化集团连续5个月拖欠员工工资，停交五险一金，员工基本生活发生问题。11月27号，数十名工人集体在公司门口拉横幅抗议。
</div>
