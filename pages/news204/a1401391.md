### 人民币汇率能否保7？学者：北京进退两难
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】近日，美国联邦储备委员会主席鲍威尔释放放缓加息的信号，缓解了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率.html" target="_blank">
  汇率
 </a>
 走软的短期压力。不过有学者认为，从基本面来看，人民币在2019年仍将承受一定的贬值压力，〝保7〞将是两难的任务。
 <br/>
 <br/>
 据大陆媒体〝界面新闻〞报导称，中科院世界经济与政治研究所国际投资研究室主任张明认为，2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 兑美元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率.html" target="_blank">
  汇率
 </a>
 的贬值，很大程度上由基本面决定:
 <br/>
 <br/>
 一方面，今年以来，美国经济增速持续上升，而中国的经济增速却持续下滑，两国的经济增速已经显著缩小；第二方面，美联储今年已经加息3次，而中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央行.html" target="_blank">
  央行
 </a>
 却是降准4次，完全相反的货币政策导致中美利差无论在货币市场还是债券市场，均显著缩小；另外在股市表现方面，美国的道琼斯工业指数年初至今上涨了约2.8%，而上证综指则下跌了19%，使得中国面临的短期资本外流压力再度上升，造成新的汇率贬值压力。
 <br/>
 <br/>
 张明表示，从2019年人民币汇率的长期走势看，在经济下滑压力、中美贸易摩擦，以及地缘政治影响等因素的影响下，人民币仍将承受一定贬值压力。如果中共央行不干预，人民币兑美元汇率将会破7；但官方若干预汇率，将在人民币国际化等方面造成负面影响，〝保7〞是两难任务。
 <br/>
 <br/>
 他预测，中国的GDP增速将由2018年的6.5%至6.6%左右，下降至2019年的6.2%至6.3%，而美国的经济至少在2019年上半年仍将保持强劲成长。同时，中美两国央行货币行动上的分歧将继续。
 <br/>
 <br/>
 张明认为，为了稳住投资者信心和缓和中美贸易冲突，北京仍有通过外汇买卖、加强资本管制、干预离岸市场等手段来维持人民币汇率稳定的动机。不过，这些工具均会导致一系列负面效应，如外汇存底缩水、影响中国企业的海外投资、对人民币国际化造成负面影响、降低市场供求对汇率中间价的影响程度，并削弱离岸市场的价格发现功能等。
 <br/>
 <br/>
 此外，如果人为抑制人民币汇率贬值，还会冲击出口，不利于宏观经济增速的稳定，也会降低中国货币政策的独立性。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：戴明）
