### 封杀3天后中共突提90天期限 日媒揭原因
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】12月1日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 后，中共一直避谈自己让步的细节，并对国内刻意隐瞒封杀90天谈判期限，引发外界对北京再次欺骗美国的怀疑。白宫则不断喊话施压。5日，中共商务部突然主动提及90天期限，并承诺〝加快落实〞川习会共识。
 <br/>
 <br/>
 北京时间12月5日，中共商务部在新闻发布会上表示，〝中国将依照日程安排和规划路线，在90天内与美国方面积极展开交涉。双方已经达成共识，正在从具体的项目着手实施。〞
 <br/>
 <br/>
 发言人还强调，〝中方将从落实已达成共识的具体事项做起，越快越好。〞
 <br/>
 <br/>
 在此之前，中共官方对90天期限只字不提，只声称美方〝取消了1月1日提升关税的计划〞，并且〝新一轮谈判后有望完全取消已加征的关税〞。中国网络上所有有关90天期限的内容都遭删除，甚至连美国驻华大使馆的官微消息都被禁止转发分享。
 <br/>
 <br/>
 日本广播协会（NHK）认为，中方突然主动提起90天期限，原因是中美对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 协议的不同表述，引发了各种猜想，中方希望籍此抹去外界的不安。
 <br/>
 <br/>
 《南华早报》报导指，中方有关落实协议中〝具体〞事项的说法，可能是对美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）的回应。
 <br/>
 <br/>
 对于中方回避90天期限，只强调〝美方同意休战〞，外界普遍猜测中共或故伎重施，根本没打算和美方认真谈判，而是准备将90天后贸易战继续升级的责任推给美国的〝出尔反尔〞。
 <br/>
 <br/>
 根据美方信息，在川习会上，北京以管控芬太尼、进口美国农产品和能源产品、从新考虑高通收购案、取消美国汽车关税等让步，换取90天商讨中共结构性改革的谈判期。但是如果90天后无法达成协议，美方将继续升级关税战。
 <br/>
 <br/>
 在北京隐瞒协议细节的可疑行动持续两天后，川普12月4日开始喊话施压。
 <br/>
 <br/>
 川普称，〝我是关税人，当有人觊觎美国的伟大财富的时候，我就让他付出相应的代价。〞
 <br/>
 <br/>
 他还强调，根据双方90天谈判的结果，不排除未来仍会提高对中国产品的关税的可能性。
 <br/>
 <br/>
 川普还宣布，90天期限从12月1日算起，明年3月1日结束。这等于将谈判期限缩短了一个月。此前白宫首席经济顾问库德洛表示，90天期限从明年1月1日开始。
 <br/>
 <br/>
 12月4日，美股大跌。外界认为，这可能说明市场对90天达成协议的期望值大减。
 <br/>
 <br/>
 根据白宫声明，北京在川习会承诺〝立即购买〞美国农产品。但据彭博社报导，北京目前还没有取消针对美国大豆等农产品加征的25%关税，中国的进口商无法采取行动。
 <br/>
 <br/>
 美国农业部长珀杜对路透社表示，北京这个所谓的〝立即行动〞，要等到一个月后才会实施。而美媒消息指，美中12月中旬就将启动贸易谈判。
 <br/>
 <br/>
 外界怀疑，中方在川习会上做出各项承诺是否真有诚意。英媒指，如果中共再次欺骗美国，会让90天后重启的贸易战炮火更为猛烈。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
