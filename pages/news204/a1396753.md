### 中共高层乱作一团？港媒曝蓬佩奥访华罕见一幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】有港媒25日披露，美国务卿蓬佩奥10月访问北京期间，遭到中方冷落，特别是与中共外长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 会晤后，中方甚至没带他吃顿饭，中共如此失礼的做法，甚至引发高层内部官员的不满。由此可见中共在对美关系方面，高层政见不一，已经乱作一团。
 <br/>
 <br/>
 香港《南华早报》25日引述熟悉内情的消息人士说，当美国务卿蓬佩奥（Mike Pompeo）10月8日访问中国时，他受到了冰冷的待遇。
 <br/>
 <br/>
 蓬佩奥是在美国副总统彭斯刚刚发表了全面性抨击中共的演讲、两国关系日益恶化的情况下访问北京的。
 <br/>
 <br/>
 中共官员在公开场合十分直接的指责美国升级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 。蓬佩奥希望会晤习近平也被拒绝了。之后蓬佩奥会晤了中共外长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 ，会晤持续不到一个小时，王毅对美国进行了多方指责，他把贸易摩擦、信任遭到破坏、台湾问题等矛盾都归咎于美方， 并要求〝美方停止这种错误言行〞。
 <br/>
 <br/>
 这位消息人士说，会晤之后，王毅和他的团队甚至没有带蓬佩奥吃顿饭，这非常失礼。而中共的一些国防官员和其他官员，都不认可王毅处理蓬佩奥访问的方式。
 <br/>
 <br/>
 蓬佩奥访问中国之前的10月4日，美国副总统彭斯发表美国政府对华政策的演说，对中共内外政策提出严厉批评，除了贸易战，他还指责了中共对美国在军事、人权、宗教信仰、科技、意识上进行渗透，以及干预美国政治等多方面的恶行。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
