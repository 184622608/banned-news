### 美四大部门联合声明：中俄破坏美国选举与民主
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 日益逼近，美国四大执法部门日前发表联合声明，要严阵以待对抗外国干扰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国选举.html" target="_blank">
  美国选举
 </a>
 的威胁。声明更点名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 、俄罗斯破坏美国选举与民主。
 <br/>
 <br/>
 10月19日，美国国家情报总监办公室、司法部、联邦调查局和国土安全部发表联合声明称，〝我们担心，俄罗斯、中国（共）和其他外国势力，包括伊朗，正在努力破坏对民主制度的信心，影响公众的情绪和政府的政策。这些活动还可能试图影响2018年和2020年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国选举.html" target="_blank">
  美国选举
 </a>
 中，选民的看法和决定。〞
 <br/>
 <br/>
 声明指，这些活动可能采取多种形式，包括利用社交媒体来扩大引起分歧的问题、在英文媒体上刊登具体内容的广告，散布关于政治候选人的假新闻和外国的宣传。
 <br/>
 <br/>
 9月23日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 英文官媒《中国日报》在爱奥华州发行量最大的报纸《得梅因纪事报》（Des Moines Register）买下4个版面，刊登批评川普发动贸易战的做法，给美国农民造成损失。该州是川普和共和党的重要票仓。
 <br/>
 <br/>
 有美国资深官员表示，中共干预大选的方法，还包括针对某些地区以及州施加关税。
 <br/>
 <br/>
 9月30日，美驻华大使布兰斯塔德在《得梅因纪事报》刊文批评中共的做法，称中共一方面自己限制异议，一方面却利用美国的新闻自由，来散布攻击川普政府的宣传。
 <br/>
 <br/>
 此前，美国情报机构以及政府和民间许多人认为，俄罗斯干预了美国2016年的总统选举。俄罗斯更因此遭到美国制裁。
 <br/>
 <br/>
 而川普总统则认为，美国政府更应该把视线投向中共。他在今年联合国大会期间公开说，他认为中共正在干预美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，并提出警告。
 <br/>
 <br/>
 对于中共干预美国中期选举的目的，此前已有分析指出，北京认为是川普总统的个人原因，对中共发动贸易战。因此，北京希望大选的结果，能够影响川普的对华关税政策。
 <br/>
 <br/>
 不过，有美媒称，现在美国两党及商界，都对川普对华强硬态度表示赞许，甚至两党还在较劲，看未来谁对中共更强硬。显然，不论美国中期选举结果如何，美国政府对华政策只会有严厉和更严厉之分。
 <br/>
 <br/>
 美国副总统彭斯10月4日在华盛顿发表的白宫对华政策，宣告了美国和西方三十多年来对中共绥靖政策的终结，并预示着川普政府将在政治、经济、外交、文化和军事等全方位，对中共政权展开打击。中美关系发生了根本性的变化，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 已演变为全面对抗。
 <br/>
 <br/>
 政事评论人士夏小强日前撰文称，川普政府与中共政权的对决，是文明世界、普世价值与共产主义邪恶政权的对决，是一场对人类生存攸关的生死之战。
 <br/>
 <br/>
 文章指出，以川普为首的西方阵营，正义的力量正在不断汇聚，共产主义和社会主义政权走向坟墓已经成为定局，大势所趋，每一个国家和政府都需要表态站队，这关系到每一个国家和民族未来。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
</div>
