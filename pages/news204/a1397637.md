### 海外华人媒体身份曝光 后台直通中南海
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】中共严格限制外国游客，特别是外国媒体进入西藏。美国华盛顿邮报一名记者说，西藏是比朝鲜还难造访的地方。但对于某些海外华人媒体记者来说，西藏只是中共为他们提供免费旅游的一个地点而已。这些记者们要做的，就是在海外报导共产党所需要的宣传。
 <br/>
 <br/>
 中共军队自1950年占领西藏之后，一直严格限制藏人宗教和行动自由，并严格限制外国人进入藏区，美国的外交官员、媒体记者、人权活动工作者、游客，因此都很难进入西藏。
 <br/>
 <br/>
 然而，中共和平统一促进会（简称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_和统会.html" target="_blank">
  和统会
 </a>
 ）从2011年起，每年组织海外华文媒体记者团走访藏区。《美国之音》11月1日报导，今年夏天，十多位来自世界各地的华文媒体记者不仅在西藏畅通无阻，还受到当地中共官员的接见。
 <br/>
 <br/>
 在与记者们的见面会上，和统会副秘书长郝一峰直言，记者团此行的目的就是〝为了配合国家（中共）反藏独，尤其是反对海外藏独势力的工作〞。
 <br/>
 <br/>
 郝一峰还说，几天来，这个团队〝已经在国内外媒体发表了十篇宣传报告〞。
 <br/>
 <br/>
 葡萄牙葡新国际文化传媒总裁马丽梅已是第七次探访藏区。中共党媒《人民日报》海外版的一篇报导引述她的话说，〝有幸用人生中的七年去做一件事，去见证一个贫困落后地区的重生，是非常有意义的〞。
 <br/>
 <br/>
 同行的南美新闻网的一名记者，在参观完拉萨和日喀则的寺庙后写道：〝蓝天之下，是触手可及的信仰……和谐，宁静，平和，世俗的那些寺庙在蓝天下构筑了藏区人民内心最深刻的幸福世界。〞
 <br/>
 <br/>
 郝一峰还要求各位媒体负责人，一定要〝把西藏宣传好〞，让海外华人和海外主流媒体感受到〝没有共产党，就没有藏区今日的变化〞。
 <br/>
 <br/>
 多年来，中共一直对外宣称和统会是一个〝由赞成中国统一的各界人士自愿结成的〞非营利社会组织。但前中共统战干部、在海外出版《中共统战部揭秘》一书的程干远揭示说，和统会成立于1988年，它以民间团体的面貌出现，实际上由中共中央直接领导。
 <br/>
 <br/>
 〝和统会的会长都是由每届的政协主席来兼任的，就是政治局常委，级别定得很高。它内部的领导组织关系都是由统战部来操控的。〞程干远说。
 <br/>
 <br/>
 程干远表示，和统会成立最初主要是对台湾统战。但是最近几年，中共搞〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大统战.html" target="_blank">
  大统战
 </a>
 〞，把海外侨团动员起来，宣传共产党的成就。〝凡是有领事关系的国家，它都通过领馆做一些动员组织工作，各地侨团的分会都是由领事馆在后面操控的。〞
 <br/>
 <br/>
 过去一年来，西方政界、学界和媒体逐渐认清和统会的真实面目。2月，美国智库詹姆斯敦基金会的一份报告指出，中共官方有关该组织的表述具有误导性。报告说，对该组织的领导层架构做粗略分析显示，和统会〝直接隶属共产党统一战线工作部〞。
 <br/>
 <br/>
 8月，美国国会下属的美中经济与安全审议委员会的年度报告指出，北京把操控海外华人社区为其政治目标服务作为工作要务。因而和统会这类统战组织，在世界各地迅速壮大。
 <br/>
 <br/>
 报告指，和统会目前在90个国家设立了至少200个分会，包括在美国的33个分会。
 <br/>
 <br/>
 值得一提的是，和统会并非为海外华文媒体提供免费游的唯一机构。从2001年起，中共国务院侨务办公室、中共新闻社常年邀请海外华文媒体高管，参加业务培训班和免费旅行。
 <br/>
 <br/>
 旅居美国亚利桑那州的《亚省时报》社长甄硕钦在一篇文章中写道，〝自从有两年一届的世界华文传媒论坛和‘行走中国’采风活动组成的大家园，我们便‘有理由’隔一两年就回来走走看看，表达我们海外游子怀亲念祖的深情。〞
 <br/>
 <br/>
 甄硕钦还说，这些旅行让他为在海外讲好〝中共故事〞、传播〝中共声音〞，展示〝中国梦〞积累了丰富素材。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
