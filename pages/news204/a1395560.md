### 中南海大案震惊全球 四大佬集中〝露面〞引猜测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】国际刑警组织主席、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 日前突然被中共抓捕，令国际社会格外震惊。巧合的是，中共四名退休大佬，此时罕见的以不同方式密集〝露面〞，同样引发舆论关注。这两起事件接连发生，仅仅是时间的巧合？还是有高层权力博弈？引发猜测。
 <br/>
 <br/>
 近期，多位已退休的中共政治局常委，以不同的方式密集〝露面〞，引发舆论关注。
 <br/>
 <br/>
 如，10月15日，中国网路自媒体刊登中共前常委李瑞环的文章，聚焦这位中共前国家领导人的为官之道，以及对中共政坛的看法。李瑞环表示，告黑状的人决不能进〝领导班子〞。
 <br/>
 <br/>
 10月11日，中共前国务院总理朱鎔基，现身清华大学经济管理学院顾问委员会会议。画面显示朱鎔基身穿西装，脸露笑容，精神不错。
 <br/>
 <br/>
 10月7日，中共前常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴邦国.html" target="_blank">
  吴邦国
 </a>
 ，在祖籍合肥市肥东县高调露面，网路上还发布了一张吴邦国和妻子章瑞珍两人的合影。
 <br/>
 <br/>
 9月30日晚，中共最高检成立的〝中国反腐败司法研究中心〞公号刊发两篇中共前常委、纪委书记吴官正的文章，谈自己从中共政治局常委之位退休后的心态。
 <br/>
 <br/>
 上述老人露面引发舆论猜测。由于中共卸任领导人身份特殊，鲜少出现在公众视野，对于他们的非官方公开活动，官媒普遍不予报导，因此他们露面通常藉助非官方渠道曝光。然而，这种有意安排的行踪，往往被外界认为另有政治意涵。
 <br/>
 <br/>
 巧合的是，上述中共老人集中〝露面〞，正值国际刑警组织主席、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 闪电落马之际，但对于元老们露面是否与孟被抓有关，其中只是时间上的巧合，还是另有原因，目前尚不得而知。
 <br/>
 <br/>
 但外界认为，孟宏伟的突然落马，不论对中共内部造成的震动，以及在国际社会造成的影响，注定非同一般。北京当局不惜冒着有损国际形象的政治风险，抓捕孟宏伟，一定有着不可告人的重大隐情。
 <br/>
 <br/>
 孟宏伟9月底从法国返回中国后〝失踪〞。10月7日深夜，中纪委官网通报，孟宏伟〝收受贿赂、涉嫌违法〞，正接受调查。
 <br/>
 <br/>
 孟宏伟被调查后，孟妻葛瑞丝‧孟（Grace Meng）不但高调召开记者会为孟发声，她还接连接受美联社、CNN等媒体采访。她更称自己遭〝两组人马〞锁定。
 <br/>
 <br/>
 英媒BBC刊文分析说，孟宏伟失踪后，他的妻子向法国当局求救，这非常罕见，她之所以这样做，感觉她已别无选择。此外，孟宏伟到底做了什么？惹怒了谁？导致北京不惜在国际社会丢丑，也要将其抓捕？
 <br/>
 <br/>
 针对这一连串的疑问，外界认为，中共冒险抓捕国际刑警组织主席，意味着中共内部遇到的危机一定是非同一般。
 <br/>
 <br/>
 有专家对美国之音分析，北京当局不惜冒着损害自身国际形象的风险，抓捕国际刑警组织主席，一定是遇到十万火急的事情，不得不采取这种手段。
 <br/>
 <br/>
 中国民主运动海外联席会议主席魏京生认为，孟宏伟之所以被捕，肯定不是因贪腐，孟卷入的应该是突发事件，而且严重程度导致中共不得不紧急处理。如果孟妻没出面曝光，恐怕孟宏伟从此销声匿迹。
 <br/>
 <br/>
 经济学者夏业良说，孟宏伟事件说明中南海肯定发生了十万火急大事情，肯定发生了跟习近平高层有关的、意料之外的重大事件，官方才必须紧急出手。如果不立即办理，恐怕他直接投靠西方。消息称，他参与政变推翻习近平。
 <br/>
 <br/>
 另有媒体分析说，孟妻手中或握有中共核弹级机密。孟宏伟孤身回国前，对其处境已有充分思想准备，孟妻的行为只是按既定方针而行。
 <br/>
 <br/>
 孟作为一个老公安在中共江湖混迹多年，深知中共黑帮政治的凶险和残酷。特别
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 的党羽一个个被抓，孟宏伟作为周的亲信，他早已预感到自己的未来。
 <br/>
 <br/>
 目前，法国警方已经对孟的妻儿进行了安全保护。当年王立军手里握有谷开来杀人证据，以及薄熙来谋反证据，前往成都美国大使馆寻求庇护。而孟宏伟作为公安部副部长，应该比王立军更加老道，可以说孟与中共的博弈才刚刚开始。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：赵云）
