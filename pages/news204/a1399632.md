### 中日共荣破灭？安倍东亚峰会不点名批中共
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】北京10月高规格接待日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 ，宣传〝中日共存共荣〞。但安倍随后的系列动作表明，并不打算进入中共的〝共荣圈〞。日前，安倍在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东亚峰会.html" target="_blank">
  东亚峰会
 </a>
 不点名批评中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海军事化.html" target="_blank">
  南海军事化
 </a>
 和〝一带一路〞，中方也对日本机床展开反倾销调查。
 <br/>
 <br/>
 共同社报导，安倍15日在新加坡举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东亚峰会.html" target="_blank">
  东亚峰会
 </a>
 上表示，对南海海域违背非军事化的动向表示〝严重关切〞，并呼吁基于国际法和平解决问题，强调〝单方面改变现状的行为〞会威胁到该地区的各国利益。
 <br/>
 <br/>
 安倍还谈及自己此前提出的〝自由开放的印度太平洋〞倡议，表示将与共享该愿景的所有国家合作。
 <br/>
 <br/>
 安倍还表示，日本将推进东南亚地区的高质量的基础设施开发，并强调〝透明度〞及〝被援助国的财政健全性〞等国际标准的必要性。该表态明显在影射中共〝一带一路〞项目中的腐败问题以及令被援助国陷入债务危机的问题，显示出与〝一带一路〞竞争的意向。
 <br/>
 <br/>
 13日，安倍还在东京和来访的美国副总统彭斯共同宣布，美日将投入700亿美元资金，支持印太地区的社会基础设施开发等。外界认为，此举旨在打击中共的〝一带一路〞。
 <br/>
 <br/>
 另据《日本经济新闻》11月16日报导，中共商务部开始针对进口机床展开反倾销调查，调查对象包括日本5家机械生产企业。报导称，机床是〝中国制造2025〞的重点领域，中方对进口机床展开调查，意在为国产机床在国内增加销量，以缓解中美贸易战造成的出口下降。
 <br/>
 <br/>
 不过有分析指，中共将多家日企列入打击对象，或显示北京对〝联日抗美〞已不抱多大希望。
 <br/>
 <br/>
 其实，安倍在10月访华回国的第二天，就已表明了不与中共〝共荣〞的态度。
 <br/>
 <br/>
 10月27日，安倍从北京返回日本。28日，安倍就在自己位于山梨县的私人别墅招待来访的印度总理莫迪，随后宣布向印度高铁追加3千亿日元（约合184亿元人民币）的ODA贷款。而他访华前刚刚终止了对华ODA贷款。
 <br/>
 <br/>
 时评人士认为，这是安倍的刻意安排，旨在向外界表明，日本、印度、美国和澳大利亚的〝印太联盟〞不会因为中共的拉拢而改变。
 <br/>
 <br/>
 共同社报导，11月12日，日本、美国和欧盟联手向世贸组织（WTO）提出改革方案，建议对私下向本国企业发放政府补贴的成员国进行制裁。该条款被认为直指中共的〝中国制造2025〞计划。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
