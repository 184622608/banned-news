### 大陆免洗产品销量大增 都是一块抹布惹的祸（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】一段关于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆五星级酒店.html" target="_blank">
  大陆五星级酒店
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卫生乱象.html" target="_blank">
  卫生乱象
 </a>
 的视频，引发酒店行业
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信任危机.html" target="_blank">
  信任危机
 </a>
 和民众恐慌。近日，大陆免洗产品、隔脏睡袋等销售量大增。更有民众表示，以后外出住宿酒店，会自带杯子和床单。
 <br/>
 <br/>
 近日，大陆网民〝花总丢了金箍棒〞发布一段影片〝杯子的秘密〞，曝光14家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆五星级酒店.html" target="_blank">
  大陆五星级酒店
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卫生乱象.html" target="_blank">
  卫生乱象
 </a>
 。无一例外，所有酒店的清洁人员在清扫过程中，都用脏洗脸毛巾或浴巾擦拭杯子、洗脸台、甚至坐便器盖子。
 <br/>
 <br/>
 此外，还有清洁人员从垃圾桶捡回客人抛弃的纸杯盖，用T恤擦一擦后重复使用。这一幕幕令人恶心的画面，使五星级酒店的〝高端〞品质瞬间跌入〝低端〞。
 <br/>
 <br/>
 曝光视频的花总坦言，〝现实比视频展现出来的更加糟糕。〞
 <br/>
 <br/>
 他向媒体透露，实际上一共暗访了30多家五星酒店，涉及北京、福州、贵阳、南昌、上海等地，曝光的是比较有代表性的14家，事实上酒店卫生乱象的波及面达100%。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 这段揭露大陆酒店卫生乱象的影片，立即引发大陆民众广泛关注，不少民众决定，以后外出旅行〝靠自己〞。
 <br/>
 <br/>
 北京一家企业销售经理说，受到影片影响，她先后购买了除菌湿纸巾、便携热水壶等一系列用品。
 <br/>
 <br/>
 还有不少民众表示，以后入住酒店会用开水烫马桶盖和杯子，自备酒精清洁卫浴设施，或自带杯子和床单等。
 <br/>
 <br/>
 近日来，大陆免洗产品、隔脏睡袋等销售数量大增。一位主营旅行用品的网路店主表示，近日折叠杯、脸盆等携带型旅行用品的搜索量较往日激增。隔脏睡袋的销售数量达到往日2至3个月的销售量。
 <br/>
 <br/>
 此外，其他电商平台的杀菌湿纸巾、面纸等产品搜索量也翻倍增加。
 <br/>
 <br/>
 不过，有评论称，这些都是头痛医头、脚痛医脚的做法，大陆很多行业，尤其是服务业，乱象丛生。例如大陆餐饮业的卫生恶劣状况，多年来被媒体曝光，但依然如故。
 <br/>
 <br/>
 有网民就酒店卫生乱象一事讽刺说：〝等这风头过去了，依旧原先怎样擦就怎样擦，难不成擦的时候旁边站上个监督的？〞
 <br/>
 <br/>
 此外，一些大陆游客的低素质也遭到舆论诟病，如有的游客用酒店烧水壶煮脏内裤等恶劣行径，曾在网络曝光。
 <br/>
 <br/>
 有评论指，大陆社会这些乱象都是因为道德层面崩塌、民众失去最基本的为人处世原则所至。究其根源，是中共在夺取政权后，毁坏了5千年的中华传统文化，导致原本的礼仪之邦，频频爆出这些不文明现象。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
