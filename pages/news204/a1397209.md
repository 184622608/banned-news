### 重庆公交车坠入长江 打捞出两具尸体
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】中共官媒通报称，当地时间周日上午10点10分左右，重庆市一辆公交车从60多米高的桥上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠入长江.html" target="_blank">
  坠入长江
 </a>
 。
 <br/>
 <br/>
 当地时间周日上午，重庆市万州区一辆22路公交车，在万州区长江二桥上行驶时，突然越过中心线，与一辆对面正常行驶的小轿车相撞，冲破护栏后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠入长江.html" target="_blank">
  坠入长江
 </a>
 。
 <br/>
 <br/>
 据报，当时车上有15人，包括一名司机。
 <br/>
 <br/>
 由于公交车坠落地点水深80多米，一般的潜水员无法到达，需要使用氦氧混合潜水作业，才能潜到车辆位置。
 <br/>
 <br/>
 目前，已经打捞出两具尸体。
</div>
