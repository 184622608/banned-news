### 踢走江泽民情妇后 沈跃跃连任妇联主席
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】11月1日，中共妇联换届，中共全国人大副委员长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沈跃跃.html" target="_blank">
  沈跃跃
 </a>
 再次当选妇联主席，5年前沈出任中共妇联主席时，踢走了原妇联主席、中共前党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姘头.html" target="_blank">
  姘头
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈至立.html" target="_blank">
  陈至立
 </a>
 。
 <br/>
 <br/>
 据中共党媒报导，11月1日，中共全国妇联第十二届执行委员会举行第一次全体会议，选举中共妇联第十二届主席、副主席和常务委员。中共全国人大副委员长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沈跃跃.html" target="_blank">
  沈跃跃
 </a>
 当选中国妇联主席。
 <br/>
 <br/>
 沈跃跃于2013年5月7日，已经获任中共妇联主席一职，踢走了原妇联主席、中共前党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姘头.html" target="_blank">
  姘头
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈至立.html" target="_blank">
  陈至立
 </a>
 。2015年8月23日，沈跃跃再次顶替陈至立出任中共妇女研究会第四届会长。
 <br/>
 <br/>
 沈跃跃是深受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 重用的官员之一，曾任共青团浙江省委书记、浙江省委组织部副部长、安徽省委副书记等。胡锦涛任中共总书记时，沈跃跃出任中组部副部长、常务副部长（正部级），2013年沈才转任中共人大常委会副委员长。
 <br/>
 <br/>
 陈至立是中共前党魁江泽民的〝姘头〞之一，是江泽民在上海任职时的老部下。江泽民掌控中共大权后，1997年，陈至立由上海转任前中共国家教委党组书记、副主任。
 <br/>
 <br/>
 1998年中共两会，陈至立最低票〝当选〞中共教育部长后，推出〝教育产业化〞政策，外界认为，陈至立在学校教育上宣传中共的政治，她按照江泽民的指示，所推行的〝教育产业化〞政策不得人心、祸国殃民。
 <br/>
 <br/>
 2003年江泽民开始交权后，让陈至立当上了国务委员、国务院党组成员，还成为北京奥运会组委会第一副主席、党组副书记。
 <br/>
 <br/>
 中共〝十七大〞后，在江泽民的庇护下，陈至立出任中共妇联主席，且当上十一届中共人大常委会副委员长，从而步入了 〝党和国家领导人〞的行列。
 <br/>
 <br/>
 但中共〝十八大〞后，随着习江斗越演越烈，陈至立的妇联主席、妇女研究会会长等多个职位被剥夺。
 <br/>
 <br/>
 此外，陈至立更大的罪行是自江泽民1999年镇压法轮功之后，积极追随江在教育系统推行迫害政策。陈至立早已被海外〝追查迫害法轮功国际组织〞列为被追查的对象。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：赵云）
