### 日媒：川习会前景难料  大陆亿万富豪提心吊胆
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】 美中两国首脑11月底将在20国集团（G20）峰会期间举行重大会谈。有日媒称，〝川习会〞将针对贸易摩擦等展开磋商，能否达成某些协议，中国的亿万富翁们也在提心吊胆地关注会谈的成败。
 <br/>
 <br/>
 11月30日至12月1日，二十国集团（G20峰会）在阿根廷布宜诺斯艾利斯登场。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）和中国国家主席习近平，将在峰会召开期间举行会谈。
 <br/>
 <br/>
 为了给〝川习会〞铺路，中美双方近一段时间恢复了接触。据川普总统披露，中共递交了一份很大的清单，回应美国对贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革.html" target="_blank">
  改革
 </a>
 的要求，有很多我们要求的东西，但是还有4、5件事被漏掉了。
 <br/>
 <br/>
 《路透社》引述一名美国官员透露，中共向美国递交的清单有142个项目，但其中并没有包括白宫的核心要求，例如解决强迫技术转移和偷窃知识产权等问题。
 <br/>
 <br/>
 一名消息人士透露，中方给的东西并不是什么新的，双方仍处在僵局之中。通道现在再次打开了，但是仍有很多工作要做。
 <br/>
 <br/>
 在这种氛围之下，中美原本定于华盛顿举行的新一轮贸易商谈，改为了阿根廷。中共国务院副总理刘鹤将率经贸团队赴阿根廷，与美方进行贸易谈判，并由〝川习会〞定夺。美中贸易战前途未卜。
 <br/>
 <br/>
 南华早报称，中美贸易谈判改为阿根廷，表明中美首脑会晤的分量将提升。〝川习会〞期间或有更多重大的事项提上日程，这也是两个最大经济体高层，首次在第三国进行双边会晤。
 <br/>
 <br/>
 日本产经新闻称，此次〝川习会〞将针对贸易摩擦等展开磋商，能否达成某些协议？中国亿万富翁们或许也在提心吊胆地关注会谈的成败。
 <br/>
 <br/>
 根据瑞银财富管理（UBS Wealth Management）和普华永道（PwC）在香港等地，发布了有关世界亿万富翁的共同调查结果。中国平均每周都要诞生2名亿万富翁（多为民营企业家）。
 <br/>
 <br/>
 但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 已经使中国不少亿万富翁们的财富大缩水。相关调查结果显示，虽然2017年中国新产生106名亿万富翁，但也有51人掉队。
 <br/>
 <br/>
 在中美贸易战持续升温之下，逆风仍在加强。瑞银财富管理的经济学家韦辉廉（Philip Wyatt）说，由于经济环境骤变，中国亿万富翁的收益和投资将受到影响。
 <br/>
 <br/>
 根据美国《福布斯》杂志10月下旬发布中国400富豪排行榜显示，资产额减少最多的是恒大集团的许家印，蒸发了117亿美元，相当于总额的28％。根据截至10月的股价等计算，中国亿万富翁仅为344人，比去年减少近30人。
 <br/>
 <br/>
 面对贸易战下仍在加强的逆风，一些中国富豪和民营企业把希望寄托在了美国总统川普身上。
 <br/>
 <br/>
 据路透社21日刊文称，一部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国民营企业.html" target="_blank">
  中国民营企业
 </a>
 ，为自身利益暗中支持美国在贸易领域对中共施压，这些企业认为，此举能够倒逼北京在经济领域进行改革，向民营企业进一步开放市场，以提高民企的竞争力，以此使得民营经济得到改善。
 <br/>
 <br/>
 文章说，这种观点的背后自有其逻辑，因为过去一年来，中共宣布的一些自由化措施，以及近期习近平做出的关于支持民营企业的承诺，都像是对美国施压的回应。
 <br/>
 <br/>
 最近一段时间以来，从中共中央到地方，从多部委到官媒，密集发声新一轮〝纾困〞民营企业政策。
 <br/>
 <br/>
 对此，有财经人士评论说，在过去5年，民企实在是被中共收割得够呛，税务稽查风暴，社保检查浪潮，环保一刀切，股票质押爆仓，银行收缩贷款，等等等等诸如此类。还要叠加供给侧改革带来的原材料价格暴涨因素，民企压根连生存空间都没有。
 <br/>
 <br/>
 如今因为贸易战，外企加速转移产能，已经指望不上，也无法指望官方搞基建和国企垄断撑起中国经济，民企又有利用价值了，所以中共对民企也一直是〝两面人〞。
 <br/>
 <br/>
 对于中国民企及亿万富翁们所关注的〝川习会〞，分析认为，如果〝川习会〞能够止息贸易战，甚至达到双赢，只有一条路：中国进行结构改革。
 <br/>
 <br/>
 天则经济研究所原所长盛洪曾在《金融时报》撰文说，中共政府在高喊〝改革开放〞 ，而对知识产权的侵犯、区别对待不同企业、限制市场的自由进入和限制数据的自由流动等等，受到美国指责，这恰恰是中方需要改革的地方。
 <br/>
 <br/>
 文章说，如果北京纠正这些错误，不仅消除了美国的指责，而且对中国自身也是有好处的，这将是一个〝双赢〞的局面。
 <br/>
 <br/>
 浙江财经大学经济与国际贸易学院院长谢作诗也对大纪元表示，美国的要求其实是为了中国好，中共之所以迟迟不愿意让步，是在保护既得利益集团。
 <br/>
 <br/>
 他说，如果通过贸易战打掉了〝行政垄断〞，让既得利益集团放弃利益，换取整个社会福利水平的提高，对整个国家、整个国民来说都是好的。在美中谈判中，中方让步也是进步。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
