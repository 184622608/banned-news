### 中国男星李亚鹏欠债4千万进失信名单
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】中国男星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李亚鹏.html" target="_blank">
  李亚鹏
 </a>
 和哥哥投资房地产6年失败，一度被告上法院，遭索赔4000万人民币。陆媒称，由于李亚鹏尚未偿还债务，被列为失信名单。其经纪人回应，此为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商业.html" target="_blank">
  商业
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_合同纠纷.html" target="_blank">
  合同纠纷
 </a>
 案件，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失信人.html" target="_blank">
  失信人
 </a>
 〞一说不属实。
 <br/>
 <br/>
 中国执行信息网10月30日显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乐坛天后.html" target="_blank">
  乐坛天后
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王菲.html" target="_blank">
  王菲
 </a>
 的前夫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李亚鹏.html" target="_blank">
  李亚鹏
 </a>
 ，列入了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被执行人.html" target="_blank">
  被执行人
 </a>
 〞，原因是涉一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_合同纠纷.html" target="_blank">
  合同纠纷
 </a>
 被判支付4,000万元（人民币，下同）未有执行。但在〝失信被执行人信息〞和〝被限制消费〞讯息中并没有李亚鹏。
 <br/>
 <br/>
 李亚鹏日前接受专访畅谈慈善事业时，他公开喊话：〝我会把70%的个人财富都捐献给社会。〞如今他被列为失信名单之一，引发外界高度关注。
 <br/>
 <br/>
 法律人士称，在〝失信被执行人信息〞和〝被限制消费〞讯息中没有李亚鹏。这意味着他尚未被追究为〝老赖〞，即在法院的生效判决后，李亚鹏仍未履行判决，因此被执行法院列为被执行人，法院正在进一步追讨欠款。
 <br/>
 <br/>
 〝被执行人信息〞显示，李亚鹏身份证号码出生日期为〝1971年〞，立案时间是2018年4月9日，执行法院为朝阳法院，案号为（2018）京0105执7240号。
 <br/>
 <br/>
 案件的终审判决书显示，丽江雪山投资有限公司李亚鹏、其兄李亚炜，与北京泰和友联投资有限公司的两份民事二审判决书。
 <br/>
 <br/>
 案件由于一起合同纠纷，判决分别为2017年和今年由北京市三中院发出，前者是管辖异议的上诉，后者涉及实质性内容诉讼。
 <br/>
 <br/>
 根据三中院判决书，泰和友联一审向朝阳法院起诉，要求李亚鹏、李亚炜、中书公司（原审被告之一），连带向泰和友联支付欠款4,000万元连利息、公告费等费用。判决日期为2018年3月23日。判决生效后，原告向一审法院申请执行。
 <br/>
 <br/>
 判决书显示李亚鹏为丽江雪山投资有限公司董事长，于2016年有一次变更资讯，李亚鹏出资7,000多万元变更股权，持股比例27.84%。
 <br/>
 <br/>
 对此，李亚鹏经纪人回应说，这是一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商业.html" target="_blank">
  商业
 </a>
 合同纠纷案件，目前该案件在北京高院的申诉司法程序中，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失信人.html" target="_blank">
  失信人
 </a>
 〞一说全是子虚乌有，不属实。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：赵云）
</div>
