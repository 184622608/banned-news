### 【禁闻】西非国再拒〝一带一路〞机场项目
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】中共力推的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞政策，又遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西非国.html" target="_blank">
  西非国
 </a>
 家拒绝。已有多个国家与中共取消合作，约有300亿美元的合作项目被拒绝。许多与中共合作的国家已经认识到〝一带一路〞政策是个〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_债务陷阱.html" target="_blank">
  债务陷阱
 </a>
 〞，合作的结果恐怕会失去本国的经济利益，甚至国家主权。
 <br/>
 <br/>
 10号，西非狮子山共和国取消中共资助的一项4亿美元项目。对于这项由北京建造、管理和维护计画兴建的玛哈哈国际机场，现任总统说：〝没有给这里的民众带来经济利益或者发展。〞
 <br/>
 <br/>
 据了解，这不是第一个国家拒绝中共的大规模投资项目。
 <br/>
 <br/>
 尼泊尔上个月才取消了水坝建设项目。今年8月，马来西亚也取消了200多亿美元的铁路和输油管建设工程。巴基斯坦、尼泊尔和缅甸陆续确认取消或搁置〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞计画的三大水电项目。越南也推迟好几个经济特区的计画。
 <br/>
 <br/>
 根据《美国之音》报导，全球发展中心报告显示，与中共在一带一路合作的68个国家中，有23个陷入债务泥淖、8个因为有今后的建设项目而有债务风险。
 <br/>
 <br/>
 报导引述经济学学者克里斯托弗・波尔丁的说法，〝这些国家的人非常担心背负来自中国的债务〞。
 <br/>
 <br/>
 中共当局在2013年提出所谓的〝一带一路〞政策，目地是通过资助基础设施项目，扩大在亚非发展中国家的影响力。其中的〝一带〞是指连接亚太地区与欧洲，中间经过的中亚地区。〝一路〞就是沿着海上丝绸之路，发展中国和东南亚、南亚、中东、北非及欧洲各国的经济合作。
 <br/>
 <br/>
 中共声称，这些投资和贷款没有政治附加条件。然而美国前白宫首席策略顾问班农认为，〝这是借给你钱让你永远无法偿还，以达到控制的目地〞。
 <br/>
 <br/>
 中共的〝一带一路〞被认为是给沿线小国设置〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_债务陷阱.html" target="_blank">
  债务陷阱
 </a>
 〞，最终迫使这些国家向中共出卖经济利益甚至国家主权，在非洲一些穷国更是如此。10月11号，班农接受媒体专访，除了批评中共政府是一个极权主义之外，他还指出〝一带一路〞政策是掠夺性资本主义。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝这确实是掠夺性的。中共跟非洲打交道的时候，最早是通过收买非洲国家，所以它们在联合国或国际人权组织帮中共发声，帮中共投票、拉票。后来中国开始投资的时候，原来最早开始在赞比亚投资的时候，目地是为了当地的铜矿。〞
 <br/>
 <br/>
 中共投资非洲，战略性物资是明显的考量。例如纳米比亚的铀矿、赞比亚的铜矿、查德的石油等，处处可见中国企业的身影。
 <br/>
 <br/>
 谢田：〝中国的公司，中国的商人，他们在非洲的行为举止，都有强烈的党文化色彩，它连真正的资本主义者都不是，因为当代的资本主义者至少还要追求公平、法治和言论自由，那中国这些都没有。〞
 <br/>
 <br/>
 时事评论员田园博士分析认为，〝一带一路〞除了输出意识形态和腐败，还给周边国家带来潜在的风险。
 <br/>
 <br/>
 时事评论员田园博士：〝在各种各样因素作用下，中国在当地的投资有可能引起这些国家的混乱和不安定，甚至有可能会扶植出像塔利班或者是像伊斯兰国ISIS这样的组织出来。所以，中国拿钱到这些国家去搞输出、去投资，能带来什么样的后果？现在谁都不知道，连中国都不知道。所以这对世界的和平安宁其实是一个潜在的危险。〞
 <br/>
 <br/>
 《华盛顿时报》国安专栏作家比尔．格茨（Bill Gertz）披露，中共企图将〝一带一路倡议〞的英文写法从〝One Belt, One Road Initiative〞更改为〝Belt and Road Initiative〞，以使其更适合支持北京推动全球霸权的战略信息。
 <br/>
 <br/>
 比尔．格茨透露，为阻碍北京获得全球霸权之战略地位，五角大厦的印太司令部正在推翻〝一带一路〞。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/周天
