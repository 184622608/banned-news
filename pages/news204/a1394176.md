### 揭秘：江绵恒与江泽民三姘头的秘闻
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宋祖英.html" target="_blank">
  宋祖英
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_淫乱.html" target="_blank">
  淫乱
 </a>
 丑闻早已广为人知。除宋祖英之外，江泽民还有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李瑞英.html" target="_blank">
  李瑞英
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄丽满.html" target="_blank">
  黄丽满
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈至立.html" target="_blank">
  陈至立
 </a>
 三大情妇也不是什么秘闻。而江的三情妇与江泽民之子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江绵恒.html" target="_blank">
  江绵恒
 </a>
 的秘闻却鲜为人知。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江绵恒.html" target="_blank">
   江绵恒
  </a>
  引见
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈至立.html" target="_blank">
   陈至立
  </a>
  给
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
   江泽民
  </a>
 </h4>
 <br/>
 据报导，在江泽民的情妇里，级别最高、对江最〝忠心〞的是陈至立。
 <br/>
 <br/>
 中共文革结束后，陈至立在中科院上海硅酸盐研究所工作，与江泽民大儿子江绵恒在同一所。江泽民任中共上海市委书记后，在江绵恒的引见下，陈至立与江泽民一拍即合，相见恨晚。
 <br/>
 <br/>
 1988年，陈至立被江委以上海市委宣传部长，市委的人都知道，她的职位是用什么换来的。
 <br/>
 <br/>
 〝六．四〞前，上海《世界经济导报》因支持学运被江泽民查封。1989年5月，江泽民进京，时任总书记赵紫阳严厉批评江处理导报事件不当，江感到大祸临头。陈至立向江表示：中央怪罪下来，她一个人把责任全揽下来就是了。
 <br/>
 <br/>
 据报，陈至立和江几十年的交情不表现在男欢女爱，而是她与江在政治上的〝生死恋〞，丑陋的政治组合。凡是江要迫害的，陈至立一定变本加厉去镇压；凡是江最爱的，陈至立一定费尽心机去培植。
 <br/>
 <br/>
 1997年，江泽民调陈至立进京任教委主任。1998年，陈至立任教育部长，陈卸任教育部长后，江泽民把她升为国务委员，统管全国及全军的教育，中国的教育事业彻底跌入深渊。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄丽满.html" target="_blank">
   黄丽满
  </a>
  曾是江泽民的秘书
 </h4>
 <br/>
 海外中文网站2015年曾刊登一张照片，是江绵恒与黄丽满，以及另外两名男性的合影。从照片上看，黄丽满与江绵恒年龄差不多。
 <br/>
 <br/>
 据公开的资料，黄丽满生于1945年2月，汉族，辽宁盖县人。1992年后，任中共深圳市委副秘书长。
 <br/>
 <br/>
 估计上述照片反映的正是那段时间。不过，黄丽满与江绵恒什么关系尚未明确，而黄与江泽民勾搭成奸早已广为人知。
 <br/>
 <br/>
 据《江泽民其人》一书披露，上个世纪八十年代初，江泽民被任命为电子工业部部长，黄丽满任职于该部办公厅。当时黄丽满每天都打扮得花枝招展，在中午午休时间，黄丽满就悄悄地闪进了江的办公室。同事们只要听隔壁部长室的门锁卡哒一响，大家就都神秘地交换眼神不言语了。
 <br/>
 <br/>
 一次，中央有紧急文件送给江泽民。送信的不敢搅了江的鸳鸯梦，只好在江办公室外边焦急地等待了一个多小时。等下午上班铃打过了老半天，黄丽满才衣衫不整地从部长室里匆匆出来。送信的这才蹑手蹑脚地把中央文件交给了江。
 <br/>
 <br/>
 江泽民去上海任市长前，把黄丽满提升为电子工业部办公厅副厅长。
 <br/>
 <br/>
 1992年后，黄丽满赴深圳任职；1995年后，任深圳市委副书记；1998年后，任广东省委副书记；2001年后，任广东省委副书记、深圳市委书记；2005年1月至2008年1月任广东省人大常委会主任；2008年3月，任中共人大侨委会副主任委员。
 <br/>
 <br/>
 知情者透露，2013年底，黄丽满因涉嫌贪腐，一度被中纪委调查。据报，黄丽满任深圳市委书记时，掌控的小金库无论怎么花销，总能保持500个亿。国家审计署的调查显示，黄丽满每月津贴、福利达25～30万元，仅每月私人宴请开支、每月租用五洲宾馆高级套房就达15～20万元。
 <br/>
 <br/>
 还有资料表明，黄丽满到深圳的4年，在深圳湾、广州、北京和上海各有一幢豪宅，市值共达1,400～1,500万元，但黄享受国家津贴，实际上等于馈赠。
 <br/>
 <br/>
 据说，深圳官场上上下下都看不起黄丽满，私下里都认为黄的官帽是靠傍江泽民得来的。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李瑞英.html" target="_blank">
   李瑞英
  </a>
  退出《新闻联播》与江绵恒有关
 </h4>
 <br/>
 中共央视《新闻联播》主播李瑞英，是江泽民与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宋祖英.html" target="_blank">
  宋祖英
 </a>
 苟合之前的情妇。江曾经每次出访都要带上李瑞英。
 <br/>
 <br/>
 2014年5月28日，在央视工作近30年的李瑞英突然被曝退出《新闻联播》。同时退出《新闻联播》的张宏民称，对此事〝一言难尽〞，李瑞英则称，其它事情〝不是我们考虑的范畴〞。
 <br/>
 <br/>
 外界关注，在李瑞英〝被离职〞前的5月24日，央视《新闻联播》报导，习近平考察上海联影医疗科技公司的新闻时，出现了江绵恒和习近平一同〝露脸〞的镜头，但在新华网等中共官媒的文字报导中，江绵恒的名字被封杀。
 <br/>
 <br/>
 有评论称，央视让江绵恒和习近平在《新闻联播》画面中一同〝露脸〞，很可能被习近平当局视为重大新闻事故，即这不是习近平当局安排的，而是江派势力擅自做出的决定。据此分析，李瑞英应是被强制退下。
 <br/>
 <br/>
 近年来，随着江泽民逐渐式微，以及习近平当局反腐打虎遍及各个层面，做为江泽民的情妇们，李瑞英、陈至立、黄丽满等人的处境也都越来越不妙。
 <br/>
 <br/>
