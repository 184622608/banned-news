  港珠澳大桥
 </a>
 开通
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 抵
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海.html" target="_blank">
  珠海
 </a>
 <br/>
 <br/>
 连接香港、澳门和广东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海.html" target="_blank">
  珠海
 </a>
 的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 〞，10月23号举行开通仪式，外界盛传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将会出席。
 <br/>
 <br/>
 中共官媒22号证实，习近平当天下午已经抵达珠海。
 <br/>
 <br/>
 香港多家媒体报导说，珠海提前几天已经严格加强安保，大量警察和便衣公安在主要干道巡逻把守。曾经多次接待中共领导人的〝珠海度假村酒店〞，也戒备森严，外来人士和车辆一律禁止进入。
 <br/>
 <br/>
 即将开通的港珠澳大桥,  历时9年建造，期间丑闻不断、工程问题多多。直到开通前夕,  有香港议员仍然在担心大桥的配套安全问题，说一旦发生事故恐怕只有〝死路一条〞。
 <br/>
 <br/>
 2. 台经济部：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外资.html" target="_blank">
  外资
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对台投资.html" target="_blank">
  对台投资
 </a>
 大幅增长
 <br/>
 <br/>
 台湾经济部10月22号公布的最新数据显示，今年前三个季度的外来投资总金额已经突破72亿美元，与去年同期相比增长幅度高达38.9%。
 <br/>
 <br/>
 据报导，台湾外资的增长与各方面的投资增加和新南向政策所带来的强劲收益有关。
 <br/>
 <br/>
 台湾今年8月被全球经济分析公司〝商业环境风险情报〞（BERI)评估为排名第三的全球最佳投资地点。
 <br/>
 <br/>
 10月又在该公司的〝2018年世界经济论坛全球竞争力报告〞中，被列为与德国、美国、瑞士并列的〝超级创新国〞，整体竞争力在全球140个国家中名列第13。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瑞银职员.html" target="_blank">
  瑞银职员
 </a>
 北京被扣 各大银行发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旅行警告.html" target="_blank">
  旅行警告
 </a>
 <br/>
 <br/>
 一名瑞银集团的职员日前被北京扣押不得离境。路透社10月22号报导说，包括瑞士银行、花旗银行、渣打银行、法国巴黎银行、摩根大通银行在内的多家金融机构，都因此要求员工推迟或重新考虑赴中国出差的计画。
 <br/>
 <br/>
 据报导，这名被扣的员工是在新加坡工作的瑞银财富管理部门的经理，她在上周被中共当局要求推迟离开北京，与中共官员见面。
 <br/>
 <br/>
 目前还不清楚中共为什么要与这名瑞银员工会面。瑞银拒绝就此事发表评论。
 <br/>
 <br/>
