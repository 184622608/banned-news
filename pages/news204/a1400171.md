### 【禁闻】11月20日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 〝合时代〞投资受害者 广东维权
 <br/>
 <br/>
 大陆网贷平台〝合时代〞的数十名投资受害者，11月11号、12号先后到广东省公安厅和深圳云街一期楼盘门口维权。广东省公安厅没有官员出面接待，深圳云街则派出保安持警棍驱赶。
 <br/>
 <br/>
 南宁老兵区政府上访 与警冲突
 <br/>
 <br/>
 11月19号，广西南宁老兵因养老金待遇问题前往区政府上访，过程中与警察发生冲突，警察对老兵喷辣椒水、围殴和抓捕，老兵则用木棍反击。
 <br/>
 <br/>
 潜江
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 申请人身安全保护
 <br/>
 <br/>
 湖北潜江维权人士伍立娟11月20号向潜江市政府、市公安局寄出申请书，申请人身安全保护。她说自己因长年上访，屡遭官方报复，所以申请保护，以防止官警联手雇佣涉黑人员绑架、殴打、威胁、辱骂和跟踪她。
 <br/>
 <br/>
 广东黄文勋〝双十节〞跑步 遭逮捕
 <br/>
 <br/>
 广东维权人士黄文勋上个月以跑步的方式纪念〝双十〞中华民国国庆节，被以〝寻釁滋事罪〞刑事拘留，目前已经超过1个月，11月16号，警方通知他的家属说，他已经被正式逮捕。家属谴责警方小题大做，滥用权力。
</div>
