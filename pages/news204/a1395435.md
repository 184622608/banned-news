### 上海网贷平台〝抢钱通〞融资项目烂尾
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海网贷平台.html" target="_blank">
  上海网贷平台
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抢钱通.html" target="_blank">
  抢钱通
 </a>
 〞在今年6月出现兑付困难，人去楼空，该平台涉及3000名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资人.html" target="_blank">
  投资人
 </a>
 ，集资约人民4.5亿元，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_融资项目.html" target="_blank">
  融资项目
 </a>
 号称度假村实为烂尾楼。受害投资人报案4个月来，没有下文，怀疑上海警方不作为，包庇犯罪。
 <br/>
 <br/>
 据了解，6月23号起，上海脉麟金融运营的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抢钱通.html" target="_blank">
  抢钱通
 </a>
 平台无法还款，25号上午，〝抢钱通〞APP上的〝幸福计划〞投资产品已经下线，而在新上线的〝互市贷〞以前，〝幸福计划〞是抢钱通平台的唯一产品，融资人为黄山幸福新世界有限公司（下称黄山幸福），融资标的是该房企的养生度假房地产项目，融资总额为1.5亿元。 27号，脉麟金融办公室已人去楼空，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资人.html" target="_blank">
  投资人
 </a>
 在全国各地相继报警。
 <br/>
 <br/>
 受害投资人曲女士:〝因为这个项目是安徽省的一个重点项目，在黄山市，我们现在就是在怀疑，抢钱通这个平台就是黄山公司的一个融资平台，利用这个圈钱。６月２６号，抢钱通平台的法人于宜泓他去自首了，7月4号的时候，平台的一些主要人员，像副总徐飞就已经在等待经侦，而到现在为止，四个月过去了，经侦还不知道这个人，经侦严重的不作为，甚至我们怀疑他在包庇。〞
 <br/>
 <br/>
 黄山幸福声明，实际收到〝抢钱通〞平台提供的金额总计7,800余万元，而2016年至今，黄山幸福未偿还一分钱借款。事发有投资人去黄山维权，发现黄山幸福〝所谓价值几十亿的项目，就是一栋烂尾楼，附近都是荒地〞。
 <br/>
 <br/>
 受害投资人革先生:〝各种各样政府办的证件，还有政府的高官合影，各大媒体宣扬，老百姓他不相信这个，他相信啥呀？他们的项目就是烂尾工程，也是诈骗一类的，报案了，６月７号到现在，一点消息都没有。很多人去他们公司，去门口拉横幅嘛，去上访。我们在黄山的时候，两个人被当地的派出所拘留起来了。〞
 <br/>
 <br/>
 〝幸福计划〞产品期限分为30天、60天和半年，30天利率在10—12%，半年利率最高达到15%。据不完全统计，〝抢钱通〞平台涉及3000投资人，金额约4.5亿元，投资人投入数万元到数百万元不等，很多人生活受到影响，连维权的路费都付不出。
 <br/>
 <br/>
 受害投资人苗女士:〝我们有很多难友去了好几次，每次去经侦都是外出办案，找不着他。我们也有难友去了信访部门，也去了北京公安部投诉，但是也只是说回了一个电话，或让把材料交到上海的信访部门处理，到现在就是没有一个说法。〞
 <br/>
 <br/>
 公开资料显示，抢钱通是2016年注册于上海市闵行区的网贷平台市，注册资本为1亿元，实缴5,000万，运营主体是弘盛鼎世集团旗下子公司上海脉麟金融。
 <br/>
 <br/>
 新唐人记者熊斌、元真采访报导
</div>
