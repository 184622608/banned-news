### 洛阳88元天价面 岳云鹏:味道让人〝很想落泪〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】中国大陆相声演员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_岳云鹏.html" target="_blank">
  岳云鹏
 </a>
 近日在社交媒体上爆料称，自己在某机场餐厅吃到了88元一碗的天价〝酱油面〞，而且还被告知必须点套餐，餐巾纸则2元一张，由此引发众网友热议。店家日前则在社交媒体上回怼称，该降的是艺人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天价片酬.html" target="_blank">
  天价片酬
 </a>
 。
 <br/>
 <br/>
 大陆相声演员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_岳云鹏.html" target="_blank">
  岳云鹏
 </a>
 本月14日曾发微博爆料称，自己在河南洛阳演出时，在机场餐厅吃到了88元一碗的天价〝酱油面〞。其微博称〝一碗酱油面卖88块钱，我想单点一碗面，回答我说必须点套餐。〞
 <br/>
 <br/>
 他还随贴上载了一张〝天价酱油面〞的图片，并进一步调侃称，〝面的味道真的（让人）很想落泪。吃完了想擦擦嘴，我问老板有没有餐巾纸，他说两块钱一张。特别心疼这个餐厅老板，机场的房租应该特别贵。〞
 <br/>

   </font>

 </center>
 <br/>
 这个爆料很快就吸引了众多网友围观并引发热议。有网友乘机发帖分享了自己曾遭遇过的类似〝被宰〞的经历；有网友照片引来众多网友围观，随后有网友发现，岳云鹏指的〝某机场〞正是河南洛阳的北郊机场。
 <br/>
 <br/>
 微博引发网友热议，纷纷表示在机场有过类似的遭遇；有网友感叹说那种〝天价〞太损了，的确需要有关机关好好管理一下；还有不少网友在微博上留言追问卖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天价面.html" target="_blank">
  天价面
 </a>
 的是哪家机场。
 <br/>
 <br/>
 有网友分析指出：岳云鹏本月13日晚在洛阳新区体育馆演出，而且在爆料〝天价面〞的前面，他还发了帖文〝谢谢洛阳的朋友〞，由此推断卖〝天价面〞的机场应该是洛阳北郊机场。
 <br/>
 <br/>
 其后，洛阳市发改委对外声称，该部门已派人到洛阳机场进行调查，得出的结论是：当事人反映〝必须点套餐〞以及餐巾纸2元一张的情况〝不存在〞。调查组最后真的对该餐厅〝菜品及部分商品明码标价不规范〞的问题处以罚款1000元。
 <br/>
 <br/>
 与此同时，被爆料的店家在微博上发帖回怼岳云鹏，〝该降价的不是一碗面，也不是一张纸，而是你们艺人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天价片酬.html" target="_blank">
  天价片酬
 </a>
 ！〞
 <br/>
 <br/>
 不过，随即有网友发帖质问店家，〝扯片酬干什么？收入多就应该被宰？〞也有网友发帖感叹，〝中共整肃娱乐圈不仅宰了肥羊，还让老板开始恨艺人了。真是一箭双雕啊！〞
 <br/>
 <br/>
