### 【禁闻】港商之死 吉林9检察官被判囚 水有多深
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】两年前被香港法院通缉的香港商人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘希泳.html" target="_blank">
  刘希泳
 </a>
 ，是央视女主持人刘芳菲的丈夫。近来传出，刘希泳在检察院遭酷刑逼供致死，九名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_检察官.html" target="_blank">
  检察官
 </a>
 因此被判刑入狱。这起司法人员因执法犯法被惩罚的案例极为罕见。来看报导。
 <br/>
 <br/>
 九名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_检察官.html" target="_blank">
  检察官
 </a>
 酷刑逼供，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘希泳.html" target="_blank">
  刘希泳
 </a>
 死亡。他们全都隶属于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉林.html" target="_blank">
  吉林
 </a>
 省延边州检察院。
 <br/>
 <br/>
 10月23号法院判决，审讯小组负责人许学哲以〝故意伤害罪〞获刑13年。其他组员分别被判处1年到5年有期徒刑。
 <br/>
 <br/>
 刘希泳是香港尖沙嘴君怡酒店老板，他也经营纺织品服装进出口贸易及房地产生意。2016年11月，刘希泳被指涉嫌诈骗中国工商银行亚洲分行（工银亚洲）2亿多港币贷款，遭到香港法庭通缉，从此下落不明。
 <br/>
 <br/>
 今年4月，港媒传出，刘希泳在延边州检察院刑讯期间，遭酷刑逼供致死。
 <br/>
 <br/>
 起诉书中表示，〝为尽早取得被害人刘希泳的供述〞，审讯小组将他蒙住眼睛绑在审讯椅上，用胶带封口、皮搋子捅口鼻、钥匙捅脚心等方法逼供。四天内，刘希泳只获准休息了两个小时。在夜间也进行审问。
 <br/>
 <br/>
 去年3月19号凌晨，检察官许学哲以刘希泳不正面回答问题为由，把他铐在身后的双手用力提起，还多次压迫他的头、折弯他的腰贴向腿部，直到他身体瘫软失去知觉，抢救无效身亡。
 <br/>
 <br/>
 根据公安部的尸检报告，刘希泳胸骨、肋骨有七处骨折，因口、鼻受压等因素窒息死亡。
 <br/>
 <br/>
 北京时政观察人士华颇：〝刘希泳这个案子爆发，说是有金融方面的问题。那些办案人员就想着立功，把这个刘希泳活生生折磨致死。现在习近平，军队方面算是清理完了，现在要逐步对政法口进行清理整顿。〞
 <br/>
 <br/>
 香港《苹果日报》此前引述海外中文媒体的说法称，刘希泳案件涉大批政法系统高官，以及现任和前任常委。
 <br/>
 <br/>
 《法国国际广播电台》去年6月也曾引述时评人士的话说，刘希泳案类似于英国商人海伍德在重庆神秘死亡的案件。可能因为刘希泳知道的情况太多，而被灭口 。
 <br/>
 <br/>
 报导还说，刘希泳与现任政治局常委刘云山的儿子合作生意。表面上，他与前政治局常委贾庆林关系很近，但真正后台是前政治局常委李岚清，和前最高人民检察院检察长贾春旺。
 <br/>
 <br/>
 1990年代，刘希泳也涉及光大集团前董事长朱小华案，而被中纪委〝双规〞，一度被关押。
 <br/>
 <br/>
 华颇：〝现在说他诈骗等等，这种事情不是一般人能做出来的，水太深了。刘希泳是豪门之间倾轧的一个牺牲品吧，只不过是另一派拿出刘希泳之死，拿出死人压活人吧！〞
 <br/>
 <br/>
 中共当局罕见的公布司法人员酷刑逼供致人死亡的消息，并对他们追责，可是在此之前，延边州也传出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 张辉，吴春延等人，被酷刑迫害致死的案例，至今没有人被问责。
 <br/>
 <br/>
 有网友评论说：港商刘希泳被酷刑致死，一是，从侧面说明了十多年来，中共对法轮功学员的酷刑迫害，完全是事实。二是，警告那些还在为中共卖命的人。
 <br/>
 <br/>
 时事评论员凌晓辉：〝对刘希泳这样的中共体制内的佼佼者，都可以采取这么残酷的方式迫害致死。所以人们不难想像，也可以推断，对法轮功学员的迫害是多么的残酷。〞
 <br/>
 <br/>
 法庭宣判后，一名法律代理人表示，刘希泳家属反对法庭的判决，希望以谋杀罪罪名，定检察官们的罪。
 <br/>
 <br/>
 《法广》去年7月曾报导，检察官的家属都跑到天津去喊冤，或到北京去上访。他们认为，指使检察官们搞死刘希泳的幕后黑手，是最高检察院副检察长邱学强。
 <br/>
 <br/>
 也就是说，九名检察官执行神秘命令，成了替罪羊。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/周天
