### 纳瓦罗透露川习会细节：美中关系史上首次
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】全球瞩目的川习会出人意料地达成90天休战协议。美国白宫贸易与经济事务顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 日前对媒体透露，〝美中两位元首面对面地敲定协议的每一个细节，这在美中关系上从未发生过。〞
 <br/>
 <br/>
 当地时间12月4日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 在接受美国全国公共广播电台采访时赞赏地说，川普（特朗普）总统成功地〝吸引了中国（共）的注意〞。
 <br/>
 <br/>
 他表示，此前的政府也这样做过，就贸易问题和北京作出交涉，但北京没有把他们放在眼里，也没有落实行动。现在，是时候解决美中贸易矛盾的问题了。
 <br/>
 <br/>
 〝本届川普政府的心态、总统的领导力和专业力都很不一样〞，纳瓦罗说，另外，川普和习近平的个人关系非常好，这也给磋商提供了有力的砝码。
 <br/>
 <br/>
 纳瓦罗还透露，刚刚在阿根廷结束的川习会不同寻常。他说，〝晚宴的前半个小时到45分钟，两位总统面对面地敲定协议的每一个细节，这在美中关系上从未发生过。〞
 <br/>
 <br/>
 美国财长姆努钦也向外界透露，〝这是中国首次就结构问题和非结构问题拿出来一个很长、具体的回应〞。
 <br/>
 <br/>
 〝这是我们第一次得到他们做出的承诺。〞姆努钦12月3日在白宫接受美国全国广播公司财经频道采访时表示。
 <br/>
 <br/>
 与此同时，白宫经济顾问库德洛也在记者会上透露，在举行晚宴前，他与中共副总理刘鹤私下会面了两次，同行的还有姆努钦以及贸易谈判代表莱特希泽。
 <br/>
 <br/>
 但在12月1日进行的第二次会面中，刘鹤直接表示：〝我什么也不说了。习主席会出面说。〞
 <br/>
 <br/>
 库德洛称，这〝太不寻常了〞，他们将这些情况汇报给了川普总统。
 <br/>
 <br/>
 纳瓦罗认为，美国通过施加关税，终于让北京开始正视这些不公平的贸易问题。其中包括强制技术转移，知识产权盗窃，对于美国公司的网路袭击，中共对于国有企业的大额补贴，高关税以及非关税壁垒。这些都是北京此前一直声称，绝对不能谈的结构性问题。
 <br/>
 <br/>
 川习会于12月1日晚在掌声中结束，随后白宫发布声明称，这是一次〝高度成功的会谈〞。并宣布明年1月1日暂不对中国产品增加更多关税，而中国则同意购买大量的美国产品，包括立即购买农产品。双方还同意就〝结构性改变〞的问题展开谈判，谈判自12月1日起，期限是90天。
 <br/>
 <br/>
 不过，中共官方发布的声明比较简短，并未提及〝结构性改变〞以及90天暂停期的信息。评论人士横河分析说，北京低调处理这次对美谈判的让步，是因为北京确实作出了实质让步，但未来90天美中的谈判将会非常困难。
 <br/>
 <br/>
 纳瓦罗表示，〝谈判的结果非常明确——就是90天，90天后，我们要看到中方实际的结构变化，要看到实际的、即时的、可验证的结果。〞他呼吁〝美国人民应该坚定不移地支持总统。〞川普总统正在做近代总统所没有做过的事。
 <br/>
 <br/>
 12月4日早晨，川普总统发推文说，与中国的新一轮贸易谈判正在展开。不过他同时说，如果90天内不能达成协议，他仍然是〝使用关税的人〞。意即贸易战将重新升温，美国会提高对2000亿中国商品征税税率至25%。
 <br/>
 <br/>
 川普还表示，当一些国家（指中共）劫掠美国的巨大财富时，他们将为此付出代价。〝我们现在正在接收数十亿美元的关税，让美国再次富有。〞
 <br/>
 <br/>
 他还补充说：〝但是如果能够与中国（共）达成一项公平协议，这项协议完成我们所知道的所有必须最终完成的事情，我将会很乐意签署。让谈判开始吧。让美国再次伟大！〞
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
