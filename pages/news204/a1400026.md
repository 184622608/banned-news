### 湖南女顺产子宫被切 医院声称为了保命
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】近日，网传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南.html" target="_blank">
  湖南
 </a>
 一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_产妇.html" target="_blank">
  产妇
 </a>
 在医院生产时，不知原因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_子宫被切.html" target="_blank">
  子宫被切
 </a>
 除。该产妇的丈夫称，院方声称产妇需切除子宫保命，但家属怀疑是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_医疗事故.html" target="_blank">
  医疗事故
 </a>
 。涉事医院给予否定。目前，双方各执一词，官方已介入调查。
 <br/>
 <br/>
 11月16日，一则〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南.html" target="_blank">
  湖南
 </a>
 大众医院不告知家属原因切除
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_产妇.html" target="_blank">
  产妇
 </a>
 子宫〞的消息在社交媒体微信朋友圈流传转发。11月19日，大众医院院方回应称，子宫切除是经家属同意的。
 <br/>
 <br/>
 据陆媒报导，2018年11月1日，产妇罗某在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张家界.html" target="_blank">
  张家界
 </a>
 市永定区凤湾路口大众医院，经检查确定可以顺产。11月2日11时18分，顺产结束，医生告知母子平安，但罗某仍被留在产房中。
 <br/>
 <br/>
 据罗某的丈夫张先生称，当天，经医生许可，他将妻子从产房扶出，妻子称下体有出血，医生称是正常情况。但随后妻子在厕所中突然晕倒，又被送回产房抢救。
 <br/>
 <br/>
 据张先生回忆，3小时后，一位医生称其妻子需要输血。下午6时，由于心情焦急，他冲进产房查看妻子情况，后打电话召集家属前来医院。期间，一名医生告知他，其妻只有切除子宫才能保命，张先生就签了字。
 <br/>
 <br/>
 张先生质疑在抢救中可能出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_医疗事故.html" target="_blank">
  医疗事故
 </a>
 。他说：手术期间，医生一直未告知其产房内具体情况。事后，医生给出的切除子宫的原因是〝可能为羊水栓塞〞，他认为不能让人信服。
 <br/>
 <br/>
 张先生还说，妻子在得知
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_子宫被切.html" target="_blank">
  子宫被切
 </a>
 除后，精神状态非常差。〝两天的时间一个字没说，东西也没吃。〞
 <br/>
 <br/>
 11月19日，湖南大众医院医务科孙科长回应陆媒记者时表示，当事孕妇生产完毕后，被留在产后观察室内观察，判断是否有产后大出血或其他并发症状。后医护人员发现该产妇出血较多并晕倒，产科主任在进产房前曾和家属进行了一个简单沟通，告知家属该孕妇可能存在出血、子宫收缩乏力，医生必须做一个〝宫腔填塞〞。
 <br/>
 <br/>
 孙姓科长称，上述做法为正常处理流程。但该产妇经〝宫腔填塞〞和其他治疗方法后出血并未停止。当日下午6时左右，产科主任与家属沟通准备切除子宫，家属在同意书上签字，但未在病重通知书上签字。〝当时家属还嫌我们决定(切子宫)慢了，但我们不到万不得已是不会切除女人子宫的。〞
 <br/>
 <br/>
 就手术过程未和家属沟通这一质疑，孙科长解释:当日下午3点至6点的治疗过程中，产科主任在抢救病人，其他医务人员无权与家属沟通。而且期间由于家属情绪激动，致其他医务人员在与家属沟通时也不敢多说话。
 <br/>
 <br/>
 该科长补充，第二天，家属被告知有26000元（人民币）医药费没有结清时，提出了一系列质疑，并要求走医疗事故鉴定程序。
 <br/>
 <br/>
 目前，张家界市卫计委正针对此事展开调查。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：范铭）
