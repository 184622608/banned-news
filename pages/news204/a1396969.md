### 【禁闻】纽时黑川普 华春莹回应招骂
------------------------

<div class="wysiwyg">
 【新唐人2018年10月27日讯】《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约时报.html" target="_blank">
  纽约时报
 </a>
 》日前报导说，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的个人手机遭中共情报部门
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_窃听.html" target="_blank">
  窃听
 </a>
 ，但被川普指出错得离谱。而中共外交部发言人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华春莹.html" target="_blank">
  华春莹
 </a>
 对这个新闻的回应，则让外界跌破眼镜。
 <br/>
 <br/>
 10月24号，《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约时报.html" target="_blank">
  纽约时报
 </a>
 》援引多位美国退休及现任官员的话说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的助手多次警告他，手机通话不安全，但是川普仍不愿意放弃使用一部没有加密的苹果手机和朋友通话。
 <br/>
 <br/>
 报导说，这些官员表示，美国情报单位通过外国政府内的线人和截获的外国官员之间的通讯，得知中共和俄罗斯正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_窃听.html" target="_blank">
  窃听
 </a>
 川普的通话。消息人士表示，北京精通窃听电话的技巧，并试图得知川普如何思考、和哪些人通话，哪些论点能影响他，他会听谁的意见等，进而找出影响川普的最佳方法。据了解，北京特别想要通过这个方法，防止两国贸易战的升级。
 <br/>
 <br/>
 但是，川普随即发推文说：〝《纽约时报》所谓的川普专家写的一篇有关于我手机使用情况的文章如此不正确，我没有时间逐一纠正。我只使用政府电话，我只有一个很少使用的政府手机。这篇报导错的太离谱了！〞
 <br/>
 <br/>
 几个小时后，川普再次发推文说：〝《纽约时报》有一个新的假新闻，报导说现在俄罗斯人和中国人（很高兴他们终于将中国（中共）加进来）正在倾听我使用手机时的所有通话。我很少使用手机，当我要使用手机时，我是用政府授权的手机。除此之外，我喜欢专线（Hard Lines）电话。更多人在制造假新闻！〞
 <br/>
 <br/>
 这场美国媒体与总统之间的罗生门，很快被中共外交部发言人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华春莹.html" target="_blank">
  华春莹
 </a>
 抢了风头。
 <br/>
 <br/>
 25号，华春莹在例行记者会上回应说，川普如果很担心苹果手机被窃听的话，她建议改用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 手机。如果还觉得不放心，为了绝对安全起见，可以停止使用任何现代通讯设备，断绝与外界所有联系。
 <br/>
 <br/>
 美国方面对这番话暂时还没有反应，但是在中文社交媒体上华春莹的回应却招来了一片骂声。
 <br/>
 <br/>
 互联网观察人士古河：〝这番讲话实际上就是傻到不能再傻的地步了，因为在之前关于中兴和华为涉及到侵权，涉及到盗窃，涉及到中共军方背景，这些已经在西方世界受到了极大的关注和防范，你现在再提华为，这不是蠢上加蠢吗。〞
 <br/>
 <br/>
 澳大利亚华裔博士张小刚：〝这个是典型的拿着肉麻当有趣，闭着眼睛说瞎话。华为，澳洲已经是明确的不允许他们参加5G项目。明明知道大家都知道它是在搞监控、搞间谍活动，拿这个话来说，这个是一种恬不知耻的表现。〞
 <br/>
 <br/>
 也有网友说，面对外界质疑永远4个字：死不认账！结果是没有人愿意跟你聊天，小伙伴都散了！
 <br/>
 <br/>
 古河：〝中共通过手机搜集情报，它的手段是太多太多了。对于在中国大陆生产出售的安卓手机，没有任何一台是不装有后门的。只要用了大陆产的安卓手机，它就必然会给中共收集情报。手机已经成为一种中共获取情报的非常重要的设备。〞
 <br/>
 <br/>
 张小刚博士：〝包括在国内，国安公开这么说的，就说国内的这些人用手机基本都是透明的，全部在他们掌控之下，那么在海外用各种方式来监控、窃听，这也是非常非常普遍，所以我们在海外都非常小心。〞
 <br/>
 <br/>
 美国中央情报局、国家安全局、联邦调查局和五角大楼情报部门高级官员，今年2月在参议院情报委员会作证时指出，中国产智能手机制造商对美国用户的信息安全构成威胁，因为这些公司有机会恶意更改或盗取信息以及进行秘密间谍活动。
 <br/>
 <br/>
 美国民主党全国委员会近日也发出警告说，出于安全考虑，角逐11月国会中期选举的党内候选人不要使用中兴和华为的设备。
 <br/>
 <br/>
 采访/陈汉 编辑/陈洁 后制/葛雷
</div>
