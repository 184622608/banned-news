### 预感大事不妙！薄熙来痛哭流涕 周永康瘫倒求饶
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】中共重庆前市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 ，和中共前政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 已在秦城监狱服刑多年，有关两人丑闻再被揭。据报，当年周接到被调查的通知后，当场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_昏倒.html" target="_blank">
  昏倒
 </a>
 、瘫倒在地，不得不被人架上床；而薄熙来被〝双规〞后更是痛哭流涕，靠在沙发上求饶。
 <br/>
 <br/>
 有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 落马的细节，多家港台媒体及美媒有过详细描述，据报，由于周永康在公安武警等政法系统有大批旧部属，在抓捕周之前，为了防止有人暗中通风报信和包庇，中共高层早已作出部署，从各地调集一批新入职，全部是新面孔的公安待命。
 <br/>
 <br/>
 据美媒引述北京消息称，2013年12月1日傍晚，时任中共中央政治局委员、中央办公厅主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_栗战书.html" target="_blank">
  栗战书
 </a>
 ，带领
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中央警卫局.html" target="_blank">
  中央警卫局
 </a>
 特别行动小组，前往周永康在中南海的住处，亲自宣布政治局刚刚做出的决定。内容是对周永康和贾晓烨夫妇实施24小时严密监视起居。
 <br/>
 <br/>
 北京消息人士描述，周永康听完栗战书宣读完决定后、当场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_昏倒.html" target="_blank">
  昏倒
 </a>
 一下子瘫在地上，随后被中央警卫局特别行动小组成员架到床上。周的秘书谭红、警卫员和司机，也都在当晚被中纪委带走。
 <br/>
 <br/>
 次日，有多家港台媒体释放了〝周永康被捕〞的消息，央视英文微博还发布了〝习近平已成立一个专案组调查被指控贪污的退休领导人周永康〞的信息，彰显了周永康的一再告饶无果，当时已成为瓮中之鳖的事实。
 <br/>
 <br/>
 关于周永康的反应，也有消息说，周永康在听完栗战书的宣布后，并没有多少反应，而是他的妻子贾晓烨突然身体一歪，倒在周的脚下，随后被人扶起，为防止出现意外，周永康夫妇从吃饭到上厕所，都有人全程陪同。
 <br/>
 <br/>
 消息还说，栗战书率领的中央警卫局特别小组人员，一进入周永康住处后，就立即切断周家与外界的联系，先羁押周永康秘书谭红，随后控制周的保镖、司机与周家保母等人，当晚撤换掉周家所有工作人员。
 <br/>
 <br/>
 栗战书离开时带走周的保镖、保姆等人员接受进一步调查，而习近平等中央政治局七常委，当晚也一直等待在会议室，直到栗战书亲自汇报，特别行动小组已顺利完成任务后，才陆续离开会议室。
 <br/>
 <br/>
 除周永康之外，早前多家媒体还详细描述了，有关周的政治盟友
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 被抓捕的细节。
 <br/>
 <br/>
 据美媒披露，2012年2月，中共重庆市公安局长王立军逃亡成都美领馆寻求庇护，曝光了薄熙来、周永康等人联手政变、活摘等关键罪行。同年3月两会期间，薄熙来被抓捕。
 <br/>
 <br/>
 在抓捕薄熙来的过程中，当时的中央警卫局长曹清起了重要的作用。
 <br/>
 <br/>
 据透露，在2012年两会期间，曹清奉胡锦涛之命，亲自率领中央警卫局的人在北京大会堂部署，3月14日上午，中共人大一闭幕，就把薄熙来控制起来了。
 <br/>
 <br/>
 《薄熙来事件谜局》一书披露，薄熙来被抓一刻情绪非常激动，打了中纪委一名办公室主任一记耳光，并大喊大叫，随后，被在场的几名执法人员架走……。
 <br/>
 <br/>
 之后，薄熙来〝双规〞后的痛哭流涕丑闻也被曝光。据港媒援引前香港《文汇报》驻大连记者姜维平的文章披露，薄熙来在被〝双规〞后，起初态度十分强硬并抗拒调查。
 <br/>
 <br/>
 但是随着薄熙来的秘书、亲信、妻子等落网后纷纷倒戈，以及成都军区阮志柏自杀，其心理防线迅即崩溃。自知难逃法网的薄，痛哭流涕，并靠在沙发上求饶。
 <br/>
 <br/>
 薄熙来于2012年3月15日，被免去重庆市委书记职务，2013年9月被判处无期徒刑；周永康于2013年12月1日，被中共政治局常委会决定核查；2014年7月29日，周被立案审查；2015年6月，周被判处无期徒刑，目前两人都在秦城监狱服刑。
 <br/>
 <br/>
 周永康被视为是江泽民的〝左膀右臂〞。掌控中共政法系统长达10年，2007年被中共前党魁江泽民塞进政治局常委会。经由媒体所披露出来的周永康各种罪行和血腥黑幕触目惊心。
 <br/>
 <br/>
 中共十九大之后，官方多次公开批评薄熙来、周永康等人阴谋篡党夺权。但阴谋篡党夺权的幕后主使江泽民、曾庆红至今仍逍遥法外。
 <br/>
 <br/>
