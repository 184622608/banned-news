### 黑龙江上万下岗民办教师集体维权
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】11月5号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑龙江.html" target="_blank">
  黑龙江
 </a>
 省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下岗民办教师.html" target="_blank">
  下岗民办教师
 </a>
 约一万两千人，聚集在省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信访接待中心.html" target="_blank">
  信访接待中心
 </a>
 大门口维权。他们要求解决身分问题和养老、医疗保险问题，但遭到警方驱赶。6号，老师们仍继续维权。
 <br/>
 <br/>
 11月5号上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑龙江.html" target="_blank">
  黑龙江
 </a>
 省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信访接待中心.html" target="_blank">
  信访接待中心
 </a>
 大门口两侧的人行道上人山人海，挤满了来自各县市的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下岗民办教师.html" target="_blank">
  下岗民办教师
 </a>
 。老师们没有打横幅也没有呼口号，以防警方抓人。现场警察不停驱赶老师，不准他们在信访接待中心大门口聚集。
 <br/>
 <br/>
 黑龙江下岗民办教师表示，现在东北天气很冷了，最低温零下几度，最高才两度，老师们站了一整天，有些人感冒了。
 <br/>
 <br/>
 黑龙江下岗民办教师尹女士:〝昨天大约一万两千啊，去了就给你驱散。截访有，哪儿都有，昨天他不给打坐，他不给解决啊。（下岗民师）连个身分都没有，最基本的养老保险、医疗保险他都不承认，区里啊、市里啊、省里、北京啊......哪都去到了，从来没间断过，就是不解决。〞
 <br/>
 <br/>
 黑龙江下岗民办教师表示，她们都是有民办教师证的老师，在80年代参加工作，教龄接近20年，本来应该转正的，因为编制被转卖，被迫下岗。
 <br/>
 <br/>
 黑龙江下岗民办教师李女士:〝我们现在上访的诉求呢，是希望能够要回我们的身分，然后呢能够返岗。我们下岗，现在什么都没有。我们现在都已经过了50岁了，在外面打工很不容易，因为年龄偏大，就业呢很困难的。我们生活没有着落，有病的很多，没有办法就去找这件事情。〞
 <br/>
 <br/>
 由于老师们拒绝与信访接待中心的官员见面，要求与副厅级以上的官员谈判，因此谈判没有进行，6号，老师们仍继续维权。
 <br/>
 <br/>
 新唐人记者熊斌、周天采访报导
 <br/>
 <br/>
</div>
