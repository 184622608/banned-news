### 美媒：北京不敢向川普让步 原来是因朱鎔基
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月28日讯】中美贸易战火持续升高，美国要求中共开出解决美国担忧的具体条件，但是中方却担心陷入两难境地。有美媒评论，北京的担忧有迹可寻。当年克林顿政府将时任中共总理的朱鎔基开出的条件公诸于众，结果令朱遭到中共党内强硬派的批评和攻击。
 <br/>
 <br/>
 美国《华尔街日报》26日报导，中美两国官员均披露，在北京拿出具体方案，来解决华盛顿对中共强制技术转让和其他经济问题的抱怨之前，美国拒绝与中共重启贸易谈判。
 <br/>
 <br/>
 这个僵局有可能破坏11月底阿根廷G20峰会上的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞。中方此前希望藉峰会缓解贸易紧张关系，从而暂停美方将于明年启动的2,000亿美元中国进口产品的关税税率。
 <br/>
 <br/>
 但一名白宫高官说：如果中方想要G20期间的会晤，首先得拿出详细的提议来，否则不会与中方谈判。
 <br/>
 <br/>
 中方则称，应该是谈判之后才拿出提议方案。中共驻美大使崔天凯说，中方不确定美方是否真正在认真对待这些更详细的结构性问题，因为双方并没有展开会谈。
 <br/>
 <br/>
 《华尔街日报》引述听取中方简报的一些人士的话分析，对中方来说，提出正式方案存在几个风险。1，可能会暴露中方的谈判立场；2，中方担心
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 发推文或声明披露方案内容，藉此堵住中方的退路。
 <br/>
 <br/>
 北京的担心源于以往的教训。1999年，在中美代表团就中共加入世界贸易组织（WTO）进行谈判时，时任国务院总理的朱鎔基，提出了一个包括重大让步和重组中国经济的方案，遭到美国总统克林顿（Bill Clinton）拒绝。
 <br/>
 <br/>
 事后，克林顿政府将朱鎔基提出的方案公布于众，以此逼得中共无法故态复萌。结果这件事让朱鎔基在中共党内受到强硬派抨击。经过数月谈判，美方才说服中共接受了一个和最初方案类似的方案。
 <br/>
 <br/>
 在目前的中美贸易战中，中共内部也出现两种声音，从而在美方的压力下应对失据。
 <br/>
 <br/>
 《纽约时报》8月引述专家分析说，在中美贸易战中，中共内部存在两种声音，一方主张更强硬，对川普要反守为攻，另一方则认为应该对美国的要求作出让步。
 <br/>
 <br/>
 分析称，这反映了中共政权的政治弱点，在内部只能〝议而不决〞。而另一种分析则认为，中共内部的争论不休，或说明最高层的〝权威〞已经不足以统一党内认识。
 <br/>
 <br/>
 由贸易战带来的争论，再加上国内此起彼伏的各类大规模维权事件，已经让中共内部的批评者壮了不少胆色。报导引述清华大学教授许章润早前的一封公开信指：〝特别是此次中美贸易战，将国力的虚弱与制度软肋暴露无遗，更加强化了不安全感。〞
 <br/>
 <br/>
 尽管中共官媒目前尚未软化贸易战对峙论调，但近期中共国内政治、经济加速封闭或倒退的迹象，以及高层不断放风稳定市场的信号，都指向一个事实：川普启动的贸易战已让中共高层方寸大乱。
 <br/>
 <br/>
 中国问题专家横河认为，〝中共的决策层比一般人想像的更悲观，可能真的在准备某种程度的自力更生和部分闭关锁国〞。中共搞阴谋习惯了，根本就想不到川普会是个说到做到，什么都放在明面上的总统，所以中共过去的手段全部失灵了。
 <br/>
 <br/>
 接下来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 会如何走？大家关注11月底的G20峰会，中美两国首脑的会晤是否能缓解中美紧张关系。
 <br/>
 <br/>
 美国副总统彭斯在10月初的演讲中提到，总统川普已明确表示，除非双方达成公平和互惠的交易，否则将对中国征收更多的关税，〝可能是比现有数值增加一倍以上。〞
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：祝馨睿）
