### 浙江再爆惨剧  房主开车撞死强拆者
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】10月25号，海外社交平台推特上流传一段视频，据称是浙江省台州市路桥区因强拆导致房主开车撞死强拆人员。
 <br/>
 <br/>
 视频显示，有大约4个人躺倒在地，周围大批人围观，但没有一个人上前施救。另一个视频中，一辆银色卡车侧翻在田地边，前挡风玻璃已经碎裂脱落，车头明显凹陷。
 <br/>
 <br/>
 台州路桥区王先生说，当时，屋主开车把4个强拆人员撞了，只有一个人受了重伤。
 <br/>
 <br/>
 台州路桥区王先生：〝听他们讲把他（屋主）的房子拆掉了，准备去拆另外一家，他当时人没在家，过来看这样子了，反正人到这个地步了，气了，给他撞了，我们这个地方很普遍的，强拆人员是路桥区街道里的镇政府人员，还有保安人员，说白一点就是土匪嘛黑帮，雇他们来搞的。〞
 <br/>
 <br/>
 王先生说，事发后，当地镇政府还威胁民众，谁敢把相关事件发布到网上，就拆谁家的房子，当地已经把这起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆事件.html" target="_blank">
  强拆事件
 </a>
 定性为了交通事故。
 <br/>
 <br/>
 新唐人记者就强拆血案询问台州路桥区区长潘建华，他声称要核实记者身份之后，再经他们区委会讨论，然后才能回复。
 <br/>
 <br/>
 多年以来，中国大陆因强拆导致的血案早已屡见不鲜。10月15号，江苏扬州市一名遭强拆的商铺业主因被多人殴打，而驾车撞向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拆迁人员.html" target="_blank">
  拆迁人员
 </a>
 ，造成1死9伤。
 <br/>
 <br/>
 新唐人记者常春、施宜君采访报导
</div>
