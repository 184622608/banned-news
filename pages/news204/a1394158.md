### 五大信号显示 中美即将全面开火
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 已经蔓延至其他领域，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 前展开了全面对抗中共的态势。有港媒称，川普似乎正在为对华〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 〞拼入最后一张拼图，相信很快会对中共作出新一轮开火。其中五大信号值得关注。
 <br/>
 <br/>
 《香港经济日报》4日报导称，美国自2018年初起，陆续祭出遏制中共的措施，但仍展示有可对话的姿态。然而，随着11月美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 临近，美方近一两周态度明显变化。
 <br/>
 <br/>
 报导认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 似乎正在为对华〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 〞，拼入最后一张拼图，中美关系由此急剧转坏。
 <br/>
 <br/>
 报导分析过去美俄角力的历史，归纳出冷战的条件包括：政治及意识形态对抗、经济分隔和军事对峙及军备竞赛。
 <br/>
 <br/>
 对比最近一两周内，川普对华的一系列强硬动作，从贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孤立.html" target="_blank">
  孤立
 </a>
 中共，到公开宣示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反对.html" target="_blank">
  反对
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产主义.html" target="_blank">
  共产主义
 </a>
 、再到最新传出打算在南海与台海举行军演等等。分析认为，五大信号预示川普或已经走美国前总统里根当年针对中共展开的新冷战。
 <br/>
 <br/>
 <h4>
  一，美加达成新自由贸易协议孤立中共
 </h4>
 <br/>
 首先是美加达成新北美自由贸易协议，意味美加墨三国将签署〝美国—墨西哥—加拿大协定〞（简称USMCA），该协定消除了北美爆发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的可能，重建美国制造业。同时协定明文限制成员国与〝非市场经济国家〞。
 <br/>
 <br/>
 美国白宫首席经济顾问库德洛10月1日表示，加拿大与美国达成贸易协议，是向中方传递重要的信息，以示北美在贸易议题上站在同一阵线。美加达成协议后，川普对华施压更具信心。只要再与日本达成自贸协定后，构建对中方最大的孤立环境。
 <br/>
 <br/>
 因此，这被认为是一个明确信号，美国在经贸上要与中共脱钩，进而孤立中共。
 <br/>
 <br/>
 <h4>
  二，中美在军事上的摩擦加剧
 </h4>
 <br/>
 美国驱逐舰〝迪凯特〞号(USS Decatur)9月30日在南沙群岛，进行旨在捍卫自由航行原则的巡逻时，遭中共军舰驱逐，几乎造成碰撞。美舰不得不采取紧急行动避免碰撞。
 <br/>
 <br/>
 随着中美战略对抗升级之际，美媒CNN报导，美军计划11月份在太平洋、特别是靠近中国的南海及台海等地区，展开一连串的军事演习及行动；美方希望以一场〝全球级〞军演活动警示中共、彰显美军随时备战的能力目的。
 <br/>
 <br/>
 这次军演一旦成事，有极强象征意义，美国要更高调在亚太地区展示军力，透过与中共的军力对峙，抑制中共提升地区军事影响力的意图。
 <br/>
 <br/>
 <h4>
  三，川普在联大号召抵抗社会主义
 </h4>
 <br/>
 美国总统川普9月25日在联合国大会发表演讲，谴责社会主义和共产主义给人们带来苦难，呼吁各国抵制。
 <br/>
 <br/>
 川普以委内瑞拉为例说，〝不久之前，委内瑞拉还是这个星球上最富有的国家之一。今天，社会主义使这个石油资源丰富的国家破产，并使其人民陷入赤贫之中。〞
 <br/>
 <br/>
 川普说，几乎所有的社会主义或共产主义都经过了尝试，它导致了苦难、腐败和腐朽。社会主义对权力的渴望导致扩张、侵略和压迫。世界上所有国家都应该抵制社会主义及其给每个人带来的苦难。
 <br/>
 <br/>
 由于中共是全球最大的社会主义国家。川普表面上好像只是批评委内瑞拉，但同样是针对中共。
 <br/>
 <br/>
 <h4>
  四，美国发起〝
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融战.html" target="_blank">
   金融战
  </a>
  〞制裁中共高官
 </h4>
 <br/>
 9月20日，美国宣布制裁中共中央军委装备发展部及部长李尚福，理由是中共向俄罗斯购买武器，引中共强烈反弹并即时召回在美访问的海军总司令。
 <br/>
 <br/>
 美方对中共采取的金融制裁，被认为打到了中共的七寸。
 <br/>
 <br/>
 金融战，小从冻结众多中共官员、国企单位的在美国资产，大到禁止各国政府、企业实体或个人与中国往来，甚至是禁止中方使用美元进行外汇交易等，不一而足。
 <br/>
 <br/>
 而一旦美国限制外国与中方交易，将让众多贪官、权贵阶层断了贪腐与收入来源，届时很可能对中共政权带来巨大冲击。
 <br/>
 <br/>
 金融战的特点是杀伤力大、精准度高且兵不血刃，中共似乎也意识到这一点，不但取消了原定10月中旬在北京举办的中美第二轮外交安全对话，还即时召回在美访问的海军总司令，此举反映出中美关系正在加速恶化。
 <br/>
 <br/>
 <h4>
  五，美国高官纷纷批评中共
 </h4>
 <br/>
 美东时间10月4日，美国副总统彭斯，在美国首都华盛顿发表长达40分钟的公开演讲，严厉抨击中共干预美国中期选举和全球政治的企图。彭斯在讲话中也批评中共各项国内和国际政策，指责中共政权制定了完整的政策侵犯别国的利益等。
 <br/>
 <br/>
 川普政府高级官员此前在吹风会(指与媒体沟通)里，也力陈中方的不当行为，包括干预美国选举、影响美国学术自由、遏抑了言论自由等方面；其中，暗指中方正在威胁美国的民主体制。
 <br/>
 <br/>
 9月30日，美国驻华大使布兰斯塔德在艾奥瓦州《得梅因纪事报（Des Moines Register）》上发表专栏文章，罕见地用极强硬措词，炮轰中方〝利用美国珍贵的言论自由和媒体自由传统〞，攻击总统川普、〝欺负〞美国农民。
 <br/>
 <br/>
 布兰斯塔德过去不仅是对华温和派，更是习近平逾30年交情的〝美国老友〞。现在连最温和的布兰斯塔德也转趋强硬，说明白宫温和声音已丧失中美关系润滑剂作用，中美关系近一步恶化。
 <br/>
 <br/>
 布兰斯塔德炮轰中共事件缘于《中国日报》10月23日付费在《得梅因纪事报》刊登四版广告，批评美国的贸易行动。
 <br/>
 <br/>
 上述种种讯号显示，川普对中共会进一步加码施压，中美新一波冲突或难以逆转，且已经蔓延至其他领域。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明）
