### 柳州致18死伤案细节：凶犯拟杀10人 含自己3子女
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】11月7日，〝8.20〞案在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广西柳州.html" target="_blank">
  广西柳州
 </a>
 市中级法院开庭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_审理.html" target="_blank">
  审理
 </a>
 ，凶犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄日朝.html" target="_blank">
  黄日朝
 </a>
 承认自己原本的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杀人计划.html" target="_blank">
  杀人计划
 </a>
 是要连续杀害10人，包括他的3个子女。但在赶往下一地点继续实施杀人计划时被抓。公诉机关建议对黄日朝，判处死刑，立即执行。
 <br/>
 <br/>
 这起凶杀案件发生于8月20日晚，凶犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄日朝.html" target="_blank">
  黄日朝
 </a>
 因感情纠葛杀死女友及其3名家人后，又驾驶越野车在柳州市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文昌大桥.html" target="_blank">
  文昌大桥
 </a>
 上冲撞多辆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电动车.html" target="_blank">
  电动车
 </a>
 ，并
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_持刀砍人.html" target="_blank">
  持刀砍人
 </a>
 ，造成7人死亡11人受伤而备受关注。
 <br/>
 <br/>
 据柳州市公安局当时发布通报，犯罪嫌疑人黄某因与女友赵某发生感情纠葛，驾驶一辆黑色丰田越野车到其女友赵某及亲属住处，将赵某及其3名亲属杀害。作案后，黄某驾车逃离现场，行驶至文昌大桥中段时，因行驶受阻开车疯撞路人，造成重大伤亡。
 <br/>
 <br/>
 视频显示，黄某被抓后神情淡定，不停向警察说话，说出自己的名字，并称自己在柳州市一工厂生活区的20栋4楼杀害了一对母女。
 <br/>
 <br/>
 一名自称遇害人家属的李姓男子称，被害人是他的前妻和女儿。8月20日早上，女儿就职单位打电话给他，称女儿没有去上班，该单位的同事随后发现了母女二人遇难。
 <br/>
 <br/>
 根据公诉机关的指控，黄日朝与女死者赵某曾系男女朋友关系，并同居多年。2018年3月，两人通过社区调解协议分手。但黄日朝对赵某怀恨在心，8月20日上午，黄日朝携带尖刀，驾驶租用的丰田牌越野车来到赵某及其女儿李某的租住处，对李某、赵某进行捅刺，直至二人死亡。
 <br/>
 <br/>
 此后，黄日朝使用特别残忍的手段侮辱、故意毁坏两尸体。作案后，黄日朝将刀尖已断的凶器留在现场，驾车离开。 当日上午8时，黄日朝驾车来到太平西街购买尖刀继续作案。
 <br/>
 <br/>
 当日上午大约11时，黄日朝驾车经过文昌大桥时，因急于赶往下一地点继续实施
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杀人计划.html" target="_blank">
  杀人计划
 </a>
 ，驾车驶入非机动车道超车，连续冲撞多辆电动车，致所驾越野车被电动车卡住后方停下，黄日朝下车后又持刀连续捅刺过路群众，共造成3死11伤的严重后果。
 <br/>
 <br/>
 黄日朝承认自己原本的杀人计划是要连续杀害10人，其中还包括他的3个子女。公诉机关建议对黄日朝，数罪并罚，判处死刑，立即执行。
 <br/>
 <br/>

 <br/>
 <br/>
