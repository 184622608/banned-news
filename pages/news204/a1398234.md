### 刘强东遭北京抛弃？性侵曝光后〝失声〞60天
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】京东集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵丑闻.html" target="_blank">
  性侵丑闻
 </a>
 曝光以来，已〝失声〞60天，引发外界关注。有大陆自媒体日前刊文梳理刘强东近期行踪，指他几乎所有重要场合都缺席，疑似遭北京抛弃。
 <br/>
 <br/>
 11月4日，关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 性侵案的大陆自媒体刊文称，刘强东的个人头条号更新，停在了2018年8月30日上午10时52分。9月1日，是大陆暑假开学第一天，刘强东在美国性侵中国女留学生被捕消息不胫而走、引爆舆论。
 <br/>
 <br/>
 在此后的60天中，除案发初期现身京东集团〝正常开展工作〞外，刘强东在几乎所有重要场合都缺席、〝失声〞。
 <br/>
 <br/>
 9月17日，2018世界人工智慧大会在上海开幕，马云等网际网路大佬云集，唯独刘强东缺席，京东方面也未说明具体原因。
 <br/>
 <br/>
 9月18日-20日，天津2018夏季达沃斯论坛上，依然没有刘强东的身影。改革开放40年民营企业家百杰名单中也没有刘强东的名字。刘强东疑似被北京当局抛弃。
 <br/>
 <br/>
 有分析指，刘强东不仅是一名商人，他还顶着全国政协委员的头衔，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵丑闻.html" target="_blank">
  性侵丑闻
 </a>
 无疑使他为中共政坛带来负面影响，因而在重大场合，刘强东被〝隐身〞。
 <br/>
 <br/>
 9月21日，美国警方称已初步完成调查，刘强东被指控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_涉嫌强奸.html" target="_blank">
  涉嫌强奸
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一级重罪.html" target="_blank">
  一级重罪
 </a>
 ，案件移交检方。美国检方表示，待审查所有证据后将决定是否指控刘强东。
 <br/>
 <br/>
 9月24日中秋节的晚上，刘强东妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_章泽天.html" target="_blank">
  章泽天
 </a>
 在朋友圈首发声称，〝惟愿守得云开见月明〞。
 <br/>
 <br/>
 但案情并没有向〝云开见月明〞的方向发展。当天，路透社报导称，刘强东案的受害人在事发当晚与朋友的微信聊天被公开，聊天内容显示，刘强东强迫受害人与之发生性关系，并提到自己喝得非常醉、刘强东在车内开始摸她等细节。
 <br/>
 <br/>
 朱军性骚扰案的当事人〝弦子与她的朋友们〞看到路透社的报导后发文称，〝气到发抖，太残忍了，真的太残忍了〞。
 <br/>
 <br/>
 10月11日，在天津举办的第十届智慧山．观峰讲坛上，中国房地产企业巨头任志强点名批评刘强东〝影响了企业的社会形象〞。
 <br/>
 <br/>
 随着事件的发酵，京东股价也急转直下。京东市值缩水近千亿元，刘强东个人财富也蒸发290亿元。
 <br/>
 <br/>
 目前，投资者们正在〝抛售〞京东，持股机构数量暴跌。2018年第二季度的持股机构达585家，而第三季度仅有100家。
 <br/>
 <br/>
 11月2日，美国明尼苏达州报纸《Star Tribune》官网刊文，曝光刘强东性侵案的诸多细节。美媒援引消息人士的话称，刘强东在性侵受害者时，遭到多次反抗，但仍得逞。期间，刘强东甚至以成为西方社交名媛邓文迪那样的人为诱惑，试图蛊惑受害者。
 <br/>
 <br/>
 值得一提的是，在刘强东性侵案刚刚曝光时，网易创始人丁磊曾对此事件进行点评。他指刘强东是灌酒性侵惯犯，在一次酒局上，刘强东把一位女子灌醉，并当场在酒席包间的厕所内对其施暴。
 <br/>
 <br/>
 另外，刘强东并非首次卷入性侵丑闻，在2015年12月末，刘强东在澳洲悉尼的一套顶层豪华公寓内举办私人宴会，一名出席宴会的澳洲女模特被灌醉性侵。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
