### 【禁闻】瞒报泉港碳九泄漏 仅是企业责任？
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】中国福建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泉州市.html" target="_blank">
  泉州市
 </a>
 发生的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泉港.html" target="_blank">
  泉港
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碳九泄漏.html" target="_blank">
  碳九泄漏
 </a>
 事故〞过去二十多天后，当地政府通报事故存在恶意隐瞒的情况。不仅有毒化工原料〝碳九〞的泄漏量相差了10倍，事故原因也是人为操作不当所造成。这一消息震惊民众，质疑政府不作为，粉饰太平。
 <br/>
 <br/>
 11月25号下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泉州市.html" target="_blank">
  泉州市
 </a>
 政府召开新闻发布会，通报月初发生在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泉港.html" target="_blank">
  泉港
 </a>
 区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东港石化公司.html" target="_blank">
  东港石化公司
 </a>
 码头的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碳九泄漏.html" target="_blank">
  碳九泄漏
 </a>
 事故，存在恶意隐瞒。
 <br/>
 <br/>
 通报说，事故发生后 ，东港石化公司对外报告泄漏了6.97吨〝碳九〞，但实际泄漏了69.1吨，几乎相差10倍。而且东港石化公司内部订立攻守同盟，恶意串通、伪造证据、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瞒报.html" target="_blank">
  瞒报
 </a>
 数量。
 <br/>
 <br/>
 另外，泄漏的原因也不是东港石化所谓的〝法兰垫片老化、破损〞，而是在码头吊机长期故障的情况下，操作员违规用输油软管把岸上和船舶连接起来，并用绳索固定软管。11月4号凌晨，随着潮位降低、船重增加，船体不断下沈，软管也随之不断下拉。最终在0点58分左右，软管被拉裂，造成大量碳九泄漏。
 <br/>
 <br/>
 这一消息震惊当地民众。
 <br/>
 <br/>
 福建泉港区居民黄先生：〝当然听了就愤怒。这个（碳九）有害啊！然后那个量差别太大。你瞒报的话，包括老百姓的自我防护的一些措施和预判，形成一些误导。然后欺下瞒上，最后可能错过了一些补救的措施，负面影响很大的。〞
 <br/>
 <br/>
 不过，官方通告中只强调了涉事相关企业的责任，声称〝目前已批捕黄某仁等7名直接责任人员〞。责令泉港区委副书记、区长颜朝晖做检讨，另外几名属地官员和监管官员被免职或停职，或等待处理。
 <br/>
 <br/>
 党媒《人民日报》25号还在报导事故的处置所谓〝及时、有效〞。
 <br/>
 <br/>
 大量民众在微博表达不满，认为这么严重的瞒报事件，不可能只是企业串通。
 <br/>
 <br/>
 有网友说：〝我在化工厂工作，基层官员不会瞒报这么大的事故，因为不值而且根本瞒不住，官员不抓只抓民有啥意义。〞还有网友说，如果真和政府无关的话：〝当初环保局为何反覆检测，甚至宣告空气质量达标？政府为何对外宣称海域已经可以恢复养殖？新浪微博为何删掉群众关于碳九的维权微博，微信为何和谐我们转发的文章？〞〝政府的不作为不应该是最主要的责任吗？〞
 <br/>
 <br/>
 此外，事故后《财新周刊》女记者去现场采访时被跟踪和骚扰，警察以所谓〝抓嫖〞的理由非法查房、恐吓她。网友认为，〝完全是政府隐瞒真相阻挠记者〞，企业没有这个胆量。
 <br/>
 <br/>
 四川环保维权人士谭作人：〝一般而言，国有性质的（企业）它出了事总是会瞒报。但是它在欺骗政府的同时它也在帮助政府。为什么呢？它在帮助维稳。这个从国营企业开始已经养成了一种习惯性的动作，然后蔓延到全社会，包括民营企业。〞
 <br/>
 <br/>
 四川环保维权人士谭作人认为，社会需要真相才能真正的稳定。但中共当局习惯性的瞒报、漏报，把在法治范围内能够正常解决的事情，上升到维稳的政治问题，反而失信于民，得不偿失。
 <br/>
 <br/>
 谭作人：〝根本上讲，是老百姓就不信任政府，包括企业，为什么出了事情要瞒报呢？大家首先对政府就不信任。所以正因为这个社会缺乏这种基本的信息的透明度，缺乏这种基本的信任，我们才是一个互相加害的社会。〞
 <br/>
 <br/>
 泉州市政府还表示，泉州市下拨资金500万元，用于扶持周边海域渔民生产，但当地民众表示，受损的不止渔民，但他们没有获得补偿。
 <br/>
 <br/>
 福建泉港区居民庄先生：〝唉！反正我们这边受到很大影响。现在海鲜类的那些生意都不好做。我很多朋友他们那个酒店都关了，都没办法经营了。他不单单说你当地的海鲜不能吃，现在是连那些外地拉过来的，人家也不相信你是外地拉的，完全都不吃。〞
 <br/>
 <br/>
 〝泉港碳九实际泄漏量〞一度在11月25号登上微博热搜榜，但到了26号已经不见踪迹。
 <br/>
 <br/>
 采访/陈汉 编辑/尚燕 后制/陈建铭
