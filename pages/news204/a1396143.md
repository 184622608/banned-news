### 范冰冰携母现身北京工作室 疑为复出铺路
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】大陆女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 缴纳巨额罚金后，19日与母亲在位于北京三里屯的工作室现身，疑似为复出做准备。
 <br/>
 <br/>
 据新浪娱乐10月19日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 回到北京后，与母亲现身在北京的工作室，范冰冰穿着白色上衣和长裙，戴着蓝色贝雷帽和墨镜，背着白色帆布包，不时低头看手机，看起来精神不错。
 <br/>
 <br/>
 范冰冰与母亲到达三里屯盈科中心后，逗留到当晚9时才离开，前后近5小时。
 <br/>
 <br/>
 陆媒此前披露，范冰冰位于北京和江苏无锡的影视工作室人去楼空，文件全被清空。如今，范冰冰现身北京工作室，由于恒大旗下的影视、经纪、发行公司都设在同一大楼内，有猜测认为，她可能与大楼内的大佬一同集思广益，为复出作准备。
 <br/>
 <br/>
 范冰冰与恒大掌门人许家印关系密切早已广为流传。据称，范冰冰的北京工作室，就是许家印提供并投资装修，内部富丽堂皇似宫殿，花费过亿人民币。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-20/p9089151a726448358.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-20/p9089151a726448358-ss.jpg"/>
   <br/>
   <font size="-1">
    19日，范冰冰与母亲在其位于北京三里屯的工作室现身，时髦美艳不减，疑似为复出做准备。（微信图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 范冰冰15日也曾现身北京首都机场，中港台多家媒体纷纷转发这一消息。从曝光的照片中可见，她戴着墨镜和黑白色棒球帽，身穿黑色宽松长外套，脚踩金色厚底鞋。她一手拿着手机，一手插在外套的口袋里，全程面无表情，一言不发，匆匆往外走。
 <br/>
 <br/>
 从画面中还可看到，她身边有一名保镳，帮撑着黑伞，疑似想挡住镜头。
 <br/>
 <br/>
 这是她在公众视野消失135天后的首度露面。与过去相比，她看上去似乎面容消瘦，脸部的菱角非常明显。
 <br/>
 <br/>
 范冰冰因卷入娱乐圈逃税案，自6月初在公众视野里消失。中共税务机关10月3日宣布，范冰冰涉嫌偷税漏税，缴纳8.8亿元人民币罚金，可免除牢狱之灾。
 <br/>
 <br/>
 当天范冰冰在微博发表〝致歉信〞说，最近经历了〝从未有过的痛苦和煎熬〞，将〝尽全力克服一切困难，筹措资金、补缴税款、缴纳罚款〞。
 <br/>
 <br/>
 就在外界估算范冰冰的身家、关注她如何缴纳罚款时，消息称，为了筹款，范冰冰8月份即在房屋中介打包抛售41套房产变现，其男友李晨也挂牌变卖北京二环的四合院帮凑钱。不过，有房屋中介否认了部分消息。
 <br/>
 <br/>
 有网友表示，近9亿元对连续5年蝉联〝福布斯中国名人榜〞榜首的范冰冰，可能是小菜一碟，但也有人认为，〝流动资金和资产是两回事，就算卖房子，也不是今天挂出来，明天就有买家付钱的。〞
 <br/>
 <br/>
 10月6日，便有传言称，范冰冰已经凑够9亿元人民币现金并交给了税务局。
 <br/>
 <br/>
 另外，范冰冰参演的电影《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大轰炸.html" target="_blank">
  大轰炸
 </a>
 》17日宣布取消在中国上映。但事隔一天，有美媒18日报导说，该片将于8月26日如期在美国上映，而且保留范冰冰的戏份不变。这或意味着范冰冰复出有望。
 <br/>
 <br/>
