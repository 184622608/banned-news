### 【禁闻】国富民穷 中国财政收入增速高于GDP
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财政部.html" target="_blank">
  财政部
 </a>
 公布2018年三季度财政收支，全国财政收入同比增长8.7%。外界质疑，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 下行压力下，中国的财政收入为什么可以高于GDP增速呢？来看分析。
 <br/>
 <br/>
 根据中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财政部.html" target="_blank">
  财政部
 </a>
 公布的数据显示，今年1月到9月累计，全国一般公共预算收入145,831亿元，同比增长8.7%。
 <br/>
 <br/>
 主要收入来源包括，国内增值税、国内消费税、企业所得税、个人所得税、进口货物增值税、消费税、关税、出口退税、城市维护建设税、车辆购置税、印花税、资源税、土地和房地产相关税收、车船税、船舶吨税、菸叶税，还有今年新开增的环境保护税。
 <br/>
 <br/>
 除了关税同比下降0.9%，其它税种呈现同比增长态势。外界质疑，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 下行压力之下，中国的财政收入为何逆风高飞，甚至高于GDP增速？
 <br/>
 <br/>
 中共财政部国库司副司长杜强在发布会上表示，〝全国财政收入平稳较快增长，收入质量不断提高〞。主要是与经济运行密切相关的主要税种增收拉动。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝虽然它说是地方企业者的个人个税造成的，但是如果政府财政收入的增长快速超过经济的增长，这实际上就是一种
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 ，就是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国富民穷.html" target="_blank">
  国富民穷
 </a>
 的一个表现。当然是更多的财富被政府拿去了，税赋太高当然是民众负担加大的结果。〞
 <br/>
 <br/>
 企业的税赋重，也意味着民生更加艰难。
 <br/>
 <br/>
 大陆经济学者廖诚：〝它就是增加税种，现在经济危机再增加税种，现在老百姓已经，民企叫苦连天。实际地方债务增加比例可能超乎想像，中国现在的经济是千疮百孔，下跌的特别严重，因为有钱的人都移民，而且外资撤离加上内地企业财团也千方百计移民，现在中国是逃亡时代。〞
 <br/>
 <br/>
 国际信评机构〝标准普尔〞最新报告，中共地方政府的潜藏债务，恐怕高达数兆美元。
 <br/>
 <br/>
 为了刺激区域的经济成长，中国地方政府常透过地方政府融资工具，大举投资基础建设。标普团队表示，这些融资工具中隐藏了许多债务。
 <br/>
 <br/>
 美国UCLA安德森管理学院经济学家俞伟雄：〝过去这十年，中共不断透过举债的方式，不管是鼓励国营企业、国有银行，鼓励地方政府透过借债的方式，槓杆的方式，来投资基础建设的投资、产能的投资，然后房地产的投资，来支撑所谓的高经济成长率。〞
 <br/>
 <br/>
 报告指出，中国地方政府实际资产负债表上的债务，可能比公开揭露的高出数倍。没有登记到的部分，可能还藏有40兆人民币的债务。
 <br/>
 <br/>
 俞伟雄：〝那过去的这些债务贷放，它的投资成本越来越低，现在有很多的债务贷放没有办法回本，就是亏损的状态，现在债务渐渐到期了，越来越多的亏损浮出台面，随着中国经济成长的降缓，包括过去几个月的中美贸易冲突，我觉得这会让过去的高成长债务、高槓杆，变成中国最大的一个隐忧。〞
 <br/>
 <br/>
 虽然中共政府一再保证，中国的债务已经得到控制，但当局却没有公布隐形帐务的规模。
 <br/>
 <br/>
 谢田：〝中央政府加上地方政府潜在债务达GDP的百分之六十的话，这还是可以管控的，但实际上这个数字可能不是这样。如果地方政府通过地方融资平台、地方借贷，进一步增加利息，它的借贷债务也会增加。就是债务的增长远远超过收入的增长。〞
 <br/>
 <br/>
 标普预期，未来中共当局可能会让更多地方政府的融资工具违约，报告称，这是世界第二大经济体的〝债务冰山与巨额信贷风险〞。
 <br/>
 <br/>
 采访/常春 编辑/黄亿美 后制/周天
