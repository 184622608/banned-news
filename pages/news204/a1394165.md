### 美媒：苹果和亚马逊被中共黑客植入间谍芯片
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】据美媒披露，中共军方黑客入侵
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国超微.html" target="_blank">
  美国超微
 </a>
 硬体公司的供应系统，在其生产的设备中植入微型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍芯片.html" target="_blank">
  间谍芯片
 </a>
 ，以获取商业和政府情报。在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 的一些主板中，都发现被植入了间谍芯片，被波及的美企有近30家。
 <br/>
 <br/>
 10月4日，彭博旗下商业周刊（Bloomberg Businessweek）发布消息称，据十几位不具名的美国情报界和科技界人士透露，2015年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国超微.html" target="_blank">
  美国超微
 </a>
 硬体公司（SuperMicro）的供应系统，遭到中共军方黑客攻击，它们生产的设备被植入了微型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍芯片.html" target="_blank">
  间谍芯片
 </a>
 。
 <br/>
 <br/>
 这是目前已知的最重大的供应链攻击行为。美国政府正在调查这起黑客攻击，并认为中共军方的一个情报部门，应对此负责。
 <br/>
 <br/>
 一位美国官员称，黑客的目标是挖掘敏感的公司交易机密，以及政府承包商电脑网路所储存的国家安全资料。
 <br/>
 <br/>
 报导称，在超微电脑伺服器中发现的间谍芯片，是被人用机器焊接在主机板上联接基板管理控制器的汇流排。这样，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共黑客.html" target="_blank">
  中共黑客
 </a>
 可以和网路上的不明电脑，进行背景连线通讯（phoning home）。并且可以直接对作业系统的内核程式，输入少许指令内容，然后修改内核程式，使其接受进一步的更改。
 <br/>
 <br/>
 中共黑客只要在其中一台不明电脑设置据点，就可以进行长期探查任务，扫瞄目标网路的配置情况，并将资料传回黑客大本营。
 <br/>
 <br/>
 报导指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚马逊.html" target="_blank">
  亚马逊
 </a>
 （Amazon.com Inc）、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苹果.html" target="_blank">
  苹果
 </a>
 （Apple Inc）等大企业的资料中心伺服器中，都发现被植入微型间谍芯片。
 <br/>
 <br/>
 此消息导致亚马逊和苹果股价大跌。美东时间4日上午11点，亚马逊的股价下跌1.55%，苹果的股价下跌1.07%。
 <br/>
 <br/>
 目前，两家公司都对彭博社的报导表示质疑。超微也没有予以置评。
 <br/>
 <br/>
 对于两家公司的质疑，报导指，经过一年多调查采访，100多次的谈话，〝有17人证实，超微电脑的硬体被人动手脚，另有其他地方也遭攻击〞。
 <br/>
 <br/>
 据悉，涉及企业除亚马逊和苹果两家公司外，还有近30家美国公司，其中包括一家大型银行和多家美国政府业务承包商。
 <br/>
 <br/>
 有数名知悉调查情况的美国官员表示，美国通过追踪这些间谍芯片，一直查到为超微电脑公司生产主机板的数家转包工厂，而超微电脑公司与中国公司合作密切。
 <br/>
 <br/>
 据悉，超微电脑创立于1993年，总部位于加州圣荷西（San Jose），2015年时已在全球100个国家拥有逾900家客户。
 <br/>
 <br/>
 事实上，中共对美国政府和商家的网路攻击，多年以来从未停止，遭到美国政府的强烈谴责。
 <br/>
 <br/>
 去年4月以来，美国一个名为〝入侵真相〞的神秘组织，在WORDPRESS的博客上，揭露有中共情报部门背景的网路黑客的姓名及其团队情况。
 <br/>
 <br/>
 一个被他们称为APT3（Advanced Persistent Threat）的黑客团队，为中共国安部服务，攻击美国、英国及其他国家政府、公司和研究机构的网站，盗取工业技术和研究资料。
 <br/>
 <br/>
 今年7月以来，〝入侵真相〞又陆续揭发了另一个有中共情报部门背景的黑客组织APT10，并警告说APT10的规模比以往的类似组织都大。
 <br/>
 <br/>
 自中美贸易战开打以来，中共通过网路黑客盗取知识产权的行为显著上升。有专家认为，只要符合中共的战略和经济利益，中共的黑客行为仍将持续。
 <br/>
 <br/>
 美国副总统彭斯4日发表白宫的中国政策演说中指出，最可怕的是，中共的情报安全机构掌握了大量窃取美国科技的能力---包括最先进的军事技术。使用这些偷窃的技术，中共正大规模地化犁为剑。美国政府正在加强反制中共黑客入侵的举措。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
