### 范冰冰挨罚 崔永元首发声：还有人逃税
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】周二，中共官方通报了中国影星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃税.html" target="_blank">
  逃税
 </a>
 案件的调查情况后，最先踢爆范冰冰阴阳合同的中共央视名嘴崔永元随后在微博上发声，被认为是暗示范冰冰参演的影片《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大轰炸.html" target="_blank">
  大轰炸
 </a>
 》的相关人员中还有人存在逃税等违法问题。
 <br/>
 <br/>
 10月3日，中共官方正式就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 工作室偷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃税.html" target="_blank">
  逃税
 </a>
 问题发布通报，其中指范冰冰曾通过电影《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大轰炸.html" target="_blank">
  大轰炸
 </a>
 》获得3,000万元片酬，而她通过拆分合同偷逃税款730万元。
 <br/>
 <br/>
 3日下午，崔永元在微博上发贴称，〝事实再一次证明，（大轰炸）就是大欺诈，参加欺诈的目前已有一人受到处罚。〞
 <br/>
 <br/>
 由于崔永元是直接引爆范冰冰逃税事件的敏感人物，他之前曾多次在微博上发帖，言之凿凿地宣称自己的抽屉里装满了影视圈的逃税材料，因此有网友认为崔永元的前述帖文所说的〝参加欺诈的目前已有一人受到处罚〞，是在暗示《大轰炸》存在的问题不仅是范冰冰逃税，还有其他人逃税或存在其他〝欺诈〞行为。
 <br/>
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-4/p9063571a472147202.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-4/p9063571a472147202-ss.jpg"/>
   <br/>
   <font size="-1">
    中共官方10月3日通报了中国影星范冰冰通过电影《大轰炸》获得3,000万元片酬后逃税的情况，崔永元随后在微博上发帖，再次指控《大轰炸》涉嫌欺诈。（网络截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 <br/>
 据公开的资讯，今年5月28日，崔永元突然在微博爆料，指范冰冰接演〝大轰炸〞时先后签下了片酬为5000万元合约及1000万元的公开合约，实收6000万元，但实际只按1000万元的片酬交税，并指着这操作方式就是所谓的〝阴阳合同〞。由此引发社会舆论对影视圈偷税、漏税问题的关注和热议。
 <br/>
 <br/>
 据了解，影片〝大轰炸〞由刘烨、布鲁斯．威利斯、陈伟霆等一众中外明星出演，讲述一段以重庆大轰炸为背景的历史故事，范冰冰也在这部电影中客串了一个幼稚园院长的角色。
 <br/>
 <br/>
 该影片主要讲述了1938年2月到1944年12月，日本军队对重庆实行长期无差别轰炸期间，重庆军民的故事。电影原定今年8月17日于大陆和北美等地同步上映，之后又发布公告将上映日期后延至今年10月26日。
 <br/>
 <center>
  <br/>

   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-4/p9063551a352274850-ss.jpg"/>
   <br/>
   <font size="-1">
    中共官方10月3日通报了中国影星范冰冰通过电影《大轰炸》获得3,000万元片酬后逃税的情况，崔永元随后在微博上发帖，再次指控《大轰炸》涉嫌欺诈。图为中国影片《大轰炸》的宣传广告画面的局部。（网络图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 <br/>
 今年6月26日至27日，崔永元又在微博上几次发帖，炮轰《大轰炸》洗黑钱、偷税漏税、集资诈骗。崔还晒出了《大轰炸》的几份合同，并说：某公司号称花费3亿人民币拍摄了电影《大轰炸》，但其实花了10亿都不止。
 <br/>
 <br/>
 6月27日下午，《大轰炸》剧组在北京召开了一个说明会，公开回应最近网络上关于电影的一系列传闻，该片的导演萧锋、监制江海洋、执行制片人王丁等人出席了说明会。
 <br/>
 <br/>
 说明会的主要内容有三点：一是，剧组没有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗钱.html" target="_blank">
  洗钱
 </a>
 、偷税漏税；二是，拍摄支出超过1.5亿；三是，演员合同签署和酬金由合禾影视负责，摄制组只负责摄制经费的使用。
 <br/>
 <br/>
 这场说明会全程仅14分钟，似乎为了避免滋生更多的舆论风波而有意采取了速战速决的策略。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：明轩）
