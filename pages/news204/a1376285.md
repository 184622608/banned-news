### 揭秘：江泽民为一事指着朱鎔基鼻子咆哮内幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】据美媒披露，在中共十五大上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 为了制约时任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国务院.html" target="_blank">
  国务院
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总理.html" target="_blank">
  总理
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 ，将其心腹
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李岚清.html" target="_blank">
  李岚清
 </a>
 安排为第一副总理。后来为另一件事，江指着朱的鼻子大声咆哮〝糊涂！糊涂！糊涂！〞，最后让李岚清负责了此事。
 <br/>
 <br/>
 据悉，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 从上海开始就嫉恨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 ，当时江任上海市委书记，朱任上海市长。1989年六四后，他俩先后调任中南海，江任中共总书记，朱任副
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总理.html" target="_blank">
  总理
 </a>
 。
 <br/>
 <br/>
 《炎黄春秋》杂志前副社长杨继绳曾刊文说，江泽民上台后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 对江的〝资产阶级自由化的斗争〞论等极左的做法相当不满意。1992年初，时年88岁的邓小平决定南巡，把冷却的〝以经济建设为中心〞这个思想重新炒热。
 <br/>
 <br/>
 文章说，1992年4月，邓小平在视察首钢时讲到，朱鎔基懂经济，不服气不行。还说对朱鎔基用晚了。这加大了邓要让朱上、江下的猜测。
 <br/>
 <br/>
 江泽民当时感觉不妙，但是，邓〝换马〞的想法，在老人中得不到支持。陈云明确表示：〝江李体制不能变。〞薄一波说：〝事不过三，你已经换了三位领导人了。〞在中共元老们的干预下，江未被撤换。
 <br/>
 <br/>
 1997年邓小平病亡后，江泽民旳权力极速膨胀，既掌握军权，又是〝党核心〞的他，唯一不能插手的是牢牢掌握在〝经济沙皇〞朱鎔基手上的经济决策权，这是江无法容忍的，千方百计要夺去朱的经济大权。
 <br/>
 <br/>
 1999年4月，朱鎔基访美未获得美方支持中方加入世界贸易组织。
 <br/>
 <br/>
 同年4月25日，上万名法轮功学员到中共信访办和平上访，刚刚访美回来的朱鎔基接见了法轮功学员代表。
 <br/>
 <br/>
 《真实的江泽民》一书披露，次日，江泽民主持召开政治局常委会讨论法轮功问题，朱鎔基刚说〝让他们去炼吧……〞这时江泽民立即跳起来，指着朱的鼻子叫喊〝糊涂！糊涂！糊涂！亡党亡国啊！〞
 <br/>
 <br/>
 事情源于：朱鎔基在接见修炼群众代表时，答应了他们的三点要求：
 <br/>
 <br/>
 一）释放天津无辜被抓的法轮功学员；二）给法轮功修炼群众一个宽松的修炼环境；三）允许出版法轮功书籍。
 <br/>
 <br/>
 因为〝4･25事件〞的合理解决，使得朱鎔基在国内外的声望大升，这让江泽民更受不了。
 <br/>
 <br/>
 书中披露，同年6月10日，江泽民授意成立了迫害法轮功的〝610〞专职机构，当时政治局常委中的其他6名常委都不赞成。江让其心腹、时任第一副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李岚清.html" target="_blank">
  李岚清
 </a>
 任〝610〞领导小组组长。李成为江迫害法轮功的直接打手。
 <br/>
 <br/>
 李岚清因此在法国、比利时、台湾、德国、加拿大、智利和荷兰等国家，遭到海外法轮功学员的控告，控告其犯有酷刑罪、反人类罪和群体灭绝罪等重罪。
 <br/>
 <br/>
 同年5月8日，发生美国战机炸毁中共驻南斯拉夫大使馆事件，江泽民就以种种借口，夺去了朱鎔基的经济大权。
 <br/>
 <br/>
 <h4>
  江泽民用安排李岚清入常制约朱鎔基
 </h4>
 <br/>
 自由亚洲电台曾刊发评论文章说，江泽民1997年9月主持召开的中共十五大上，硬是要安排李岚清进入政治局常委，并在1998年3月朱鎔基接替李鹏总理职务的同时，让李接任第一副总理。
 <br/>
 <br/>
 而没有像中共十五大之前，外界传说的那样安排比李岚清年轻9岁的吴邦国，其真实动机十之八九是设计好如此安排，才能有效制约朱鎔基的目标……
 <br/>
 <br/>
 文章披露，江泽民1989年入主中南海之初，李岚清还是对外经贸部的副部长，但这名副部长的一次搬家，竟然能够惊动江泽民和李鹏双双前往恭贺乔迁之喜。
 <br/>
 <br/>
 仅从这一个生活细节上，足见他们之间的朋党关系之铁。
 <br/>
 <br/>
 当时一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国务院.html" target="_blank">
  国务院
 </a>
 办公厅工作人员透露说，李岚清与朱鎔基的关系用〝话不投机半句多〞来形容都已经不够形象，两人之间的关系紧张程度甚至已经影响到了各自手下的工作人员。
 <br/>
 <br/>
 文章说，试想，令一名与总理〝话不投机半句多〞的人出任第一副总理，如何谈得上顺利开展工作。当初赵紫阳担任总理期间，虽与副总理李鹏和姚依林时常在观念上针锋相对，但至少在表面上还能保持〝一团和气〞。
 <br/>
 <br/>
 自由亚洲另一篇评论文章中说，如果当时江泽民真心是从有利于干部新老交替、大局出发，第一副总理的位置应该是安排给下届总理的接班人温家宝（55岁），但江泽民却安排给了65岁的李岚清。
 <br/>
 <br/>
 文章认为，江泽民当时如此安排的目的，显然是希望在朱鎔基与李岚清的内耗中减弱朱对江的所谓领导核心的权力威胁。
 <br/>
 <br/>
