### 疫苗受害家长北京王府井募款被带走
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】大陆一些问题疫苗受害儿童家长，因为得不到企业和政府的赔偿，自己又没钱给孩子治疗，日前到北京闹市区王府井募捐，但几分钟之内，就被警察带走。
 <br/>
 <br/>
 10月3号，几位疫苗受害儿童家长，带着残疾的孩子，来到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京王府井.html" target="_blank">
  北京王府井
 </a>
 商业街，为孩子募集治疗费用，民众见状纷纷上前捐款，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_募款.html" target="_blank">
  募款
 </a>
 活动只持续了不到5分钟，就被警察阻止。
 <br/>
 <br/>
 江西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗受害家长.html" target="_blank">
  疫苗受害家长
 </a>
 钟爱梅：〝（警察）他是说人员太多了，怕出现什么事情，所以叫我们不要在那里。〞
 <br/>
 <br/>
 家长们10月5号对新唐人表示，他们都是迫于无奈，才会走上街头去募捐。
 <br/>
 <br/>
 重庆受害疫苗家长黄秀虹：〝因为我们这群孩子，真的医疗费用非常高啊，家里无法承担，每家都欠了很多债。〞
 <br/>
 <br/>
 钟爱梅：〝小孩子这种状况，我需要钱去治疗，我自己拿不出钱来，我就只有向社会去募捐。〞
 <br/>
 <br/>
 他们说，之所以选择王府井，正是因为那里人多，募款相对容易。
 <br/>
 <br/>
 江西疫苗受害家长 陈汉霖：〝老百姓肯定看到我们小孩那样，残疾了，或者各种情况，一般正常人都会同情的。〞
 <br/>
 <br/>
 据家长们介绍，他们当天大人、小孩加一起大概也就是7、8个人，但警察说怕人多引起踩踏事故，将她们全部带到了王府井派出所，到了晚间又送到专门关押上访人士的久敬庄
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑监狱.html" target="_blank">
  黑监狱
 </a>
 ，并通知各地驻京办来接人。
 <br/>
 <br/>
 黄秀虹：〝其中有一位家长是河南的叫高丽，他们来了8个人，其中有6个是人高马大的、都是粗汉子，就跟高丽在里面发生很多冲突，他们的意思就是想把高丽给强行的带回河南。〞
 <br/>
 <br/>
 黄秀虹说，这些政府请来的打手，与受害家长高丽在黑监狱里推推拉拉近5个小时，高丽甚至两次被逼撞铁栏杆，最后还是在十几位男性受害家长的护送下，才脱险。
 <br/>
 <br/>
 新唐人记者常春、陈杰采访报导
</div>
