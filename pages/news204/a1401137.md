### 湖北老翁杀猪祝寿 却被肥猪踢入滚水锅
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】湖北一杨姓老翁日前过80岁大寿，家人宰杀一只200多斤的大肥猪，招待前来祝寿的亲友。不料发生意外，已被宰杀的大肥猪气绝前四蹄乱踢，老翁被踢中，跌入滚烫的大热水锅中，被重度烫伤，送院救治。
 <br/>
 <br/>
 11月19日是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖北黄冈市.html" target="_blank">
  湖北黄冈市
 </a>
 罗田县的杨老翁80岁大寿。老翁的女儿向陆媒说，当天，村里来了很多亲友为父亲祝寿。
 <br/>
 <br/>
 午宴过后，家人按照杨老翁的交代，宰杀一只200多斤重的大肥猪招待亲友，并准备好一大锅滚水，用于剃猪毛。
 <br/>
 <br/>
 不料，被宰杀的肥猪在气绝前垂死挣扎，躺在地上四蹄乱踢，杨老翁不幸被踢中，跌入滚烫的大水锅里，众人连忙将他救起来，但他已被烫得浑身通红。
 <br/>
 <br/>
 当天下午，杨老翁被送往医院救治，医生检查后发现，杨老翁全身烫伤面积达60%以上，且背部和肩膊等位置达到三度烫伤，治疗可能要花费10万元人民币。
 <br/>
 <br/>
 (记者罗婷婷报导）
</div>
