### 江苏〝大妈〞闯酒店大堂跳广场舞 职员惊呆
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】近日，一群大妈在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒店大堂.html" target="_blank">
  酒店大堂
 </a>
 大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_跳广场舞.html" target="_blank">
  跳广场舞
 </a>
 的视频在网路热传。据悉，江苏南京持续遭遇阴霾天气，大妈们为躲避阴霾，闯入一家酒店，在营业的大堂热舞，酒店职员当场惊呆。
 <br/>
 <br/>
 11月27日晚，在江苏南京某小区内，一群喜爱
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_跳广场舞.html" target="_blank">
  跳广场舞
 </a>
 的大妈为躲避阴霾，遂闯入小区一家正在营业的酒店，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒店大堂.html" target="_blank">
  酒店大堂
 </a>
 大跳广场舞逾半小时。
 <br/>
 <br/>
 据网传视频画面显示，一群大妈们陆续步入酒店内，花了近十分钟在现场设置音响设备。随后，大妈们便旁若无人地播放音乐，跳起了广场舞。
 <br/>
 <br/>
 酒店职员见状后惊呆，于是连忙走上前好言相劝，但大妈们根本不予理会，酒店方无奈，只好报警求助。
 <br/>
 <br/>
 不料，警察闻讯来到酒店后，大妈们仍自顾自地跳舞。视频显示，有大妈对警察说，〝是政府要我们健身，不要小题大作。〞
 <br/>
 <br/>
 大妈们还振振有词地说，这个酒店在小区内，又是对外营业的，她们是小区业主也有使用权，可以在这里跳舞。
 <br/>
 <br/>
 最终警察也劝阻无效，在警方的协调下，酒店方只好通融、忍让，允许大妈们在酒店内跳舞。这些大妈热跳40分钟后，才扬长而去。临走前，大妈们还回呛酒店方，称阴霾过后，〝你请我们来我们都不来！〞
 <br/>
 <br/>
 该事件引发网友热议，有网友说，〝大妈们的奇葩行为，超出想像了！〞
 <br/>
 <br/>
 近日，网络有文章称，中国大妈已成公害，一些大妈在海外旅游时，身着亮眼的服装，不分场合地群起大跳广场舞；一些大妈在吃自助餐时，抢食物、霸座位，旁若无人地大声聊天，等等行为，令人侧目。
 <br/>
 <br/>
 文章最后总结，林徽因，杨绛等一代高贵，优雅，端庄的民国范儿逝去，一个时代结束了，当年的头上长角，身上长刺，满口革命的女红卫兵而今成了闻名世界的中国大妈，当然，这只是其中一部分人。但几十年的阶级斗争，造成了中国人文主义被破环，传统的礼义廉耻大滑坡，造就了这些具有中共特色的大妈们。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：文慧)
</div>
