### 10月21日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 湖北维权人士鲍乃刚进京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 遭刑拘
 <br/>
 <br/>
 湖北京山县维权人士鲍乃刚，10月18号进京上访，遭当地维稳人员强制带回，之后再被京山警方询问并抄家，20号遭刑事拘留，目前被关押在京山市看守所。
 <br/>
 <br/>
 2. P2P受害者京东总部维权 4人被抓
 <br/>
 <br/>
 10月18号，大陆40多名联璧斐讯金融受害者到京东公司总部维权，与保安发生冲突，维权者先是遭保安殴打，之后又被警察抓走4人。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北大女生.html" target="_blank">
  北大女生
 </a>
 失踪 网上发起寻人行动
 <br/>
 <br/>
 北大女生岳昕毕业后参与声援深圳佳士科技公司工人维权运动，8月月24号与其他学生一起遭警方抓捕，之后其他学生被释放，岳昕失踪。10月19号，〝佳士工人声援团〞在推特贴出〝寻找北大女生岳昕〞的寻人启事，发起网上寻人行动。
 <br/>
 <br/>
 4. 甘肃女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 交不起学费 女儿被迫辍学
 <br/>
 <br/>
 甘肃省东华镇北河村女村民罗巧玲，因居住区采煤塌陷，房子被拆除，虽然分配到一套住房，但最后却被村、镇官员盗卖，应该享受的低保、危房改造等各项救助也都被村官截留，最后只得带着11岁的女儿在北京乞讨上访，日前因她无钱交学费，孩子被迫辍学。
</div>
