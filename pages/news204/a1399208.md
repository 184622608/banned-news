 <b>
  美财长要求中共提具体方案被拒

 <br/>
 <br/>
 美财长要求中共提具体方案被拒距离美中两国领导人G20峰会不足三个星期，美国《华尔街日报》11月13号报导说，美国财政部长姆努钦与中共国务院副总理刘鹤日前恢复对话，讨论一个缓解双方贸易紧张关系的协议。据听取了谈判情况简报的知情人士透露，姆努钦与刘鹤9号通了电话。目前美方要求中共在开始贸易协议谈判之前，拿出具体的提议。但中共官员担心因此失去谈判筹码而拒绝，令美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易谈判.html" target="_blank">
  贸易谈判
 </a>
 未能在峰会前取得进展。
 <br/>
 <br/>
 <b>
 </b>
 <br/>
 <br/>
 11月13号，一份广西桂林电子科技大学下发的文件在网上流传。文件显示，该校正在开展检查所谓〝违法音视频〞活动。这项活动从11月7号开始，到23号结束，长达半个月。清查对象是全校4万多名教职人员和在校学生。检查范围包括手机、电脑、移动硬盘和U盘，特别要求要做到全覆盖，不遗漏1人。文件声称，这是为了打击境外敌对势力通过网络和手机传播涉暴恐、反动和淫秽音视频。校方向〝自由亚洲电台〞证实确有其事，并披露指令来自上级。有学者分析说，官方采取如此严厉的措施，是为了加强对学生的思想控制。大陆多位律师则在微信留言说，桂林电子科大校方涉嫌滥用职权，侵害师生权益，可能违法。
 <br/>
 <br/>
