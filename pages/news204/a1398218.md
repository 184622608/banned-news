### 港珠澳大桥成旅游〝卖点〞香港东涌被陆游客挤爆（组图）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 成为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港一日游.html" target="_blank">
  香港一日游
 </a>
 〞新卖点，通车后第二个周日，大陆游客量激增，香港东涌成为人潮挤爆的〝重灾区〞，港人投诉生活受到严重影响。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 通车后，再度发生大陆旅客挤爆香港的问题。11月4日，香港入境处公布，当日有逾10万人次经大桥出入境。
 <br/>
 <br/>
 香港01报导，香港东涌被大陆人潮挤爆。东涌位于香港新界大屿山北部，邻近香港国际机场及港珠澳大桥香港口岸，是香港出入境中的一个枢纽。
 <br/>
 <br/>
 报导引述旅游业界人士表示，一些大陆旅行团以体验港珠澳大桥为卖点，团费一般在人民币250元以内，只提供交通和一瓶矿泉水，不包括餐点。因团费低廉，吸引大量大陆游客报名，当作来香港玩一次。
 <br/>
 <br/>
 这些大桥体验团一般从珠海口岸出发，除了搭车游港珠澳大桥外，唯一到达的香港景点就是东涌的名牌折扣商场。游客通常自由活动4小时后即返程。
 <br/>
 <br/>
 因大批旅客涌入，令原本生活宁静的东涌变成旅游重地，造成噪音、卫生及逼爆商场等问题，东涌当地居民投诉，生活品质大受影响。

 <br/>
 苹果日报报导，东涌4日迎来数以万计大陆旅客大军，旅客抵达后即到商场哄抢奶粉及日用品，茶餐厅也被挤爆，大批游客排队用餐。到傍晚时分，大陆游客陆续集合于东涌巴士站候车返回大桥口岸，繁忙时间多达逾千人排长龙。直到晚上9点多，大陆游客才全部撤离。
 <br/>
 <br/>
 11月5日，香港旅游业议会主席黄进达在电台节目中表示，港珠澳大桥体验团因留港时间只有数小时，不能到铜锣湾、尖沙咀等香港市区景点游览，只能选择到东涌，导致东涌人潮激增。
 <br/>
 <br/>
 《自由亚洲》报导称，香港旅游促进会总干事崔定邦指出，按正常情况而言，大陆旅行社进入香港需事先向旅议会登记，并且在香港也需要有旅行社接待。但他指出，周日（4日）约有2万名没有登记的旅行团团员访港，或已触犯法例。
 <br/>
 <br/>
 立法会及东涌区议员周浩鼎则指出，现在大陆旅客太多，已经超出东涌的承受能力，希望政府正视这个问题。
 <br/>
 <br/>
