### 中国经济下行 河北阴霾橙色预警又来
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】首先来关注中国的消息。中国河北省环境保护厅10月31号在声明中表示，鉴于预期的阴霾，要求该省10个城市发布二级污染警报。这是两个星期内，河北省第二次发布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空气污染.html" target="_blank">
  空气污染
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_橙色预警.html" target="_blank">
  橙色预警
 </a>
 。
 <br/>
 <br/>
 河北环保厅的声明说，从10月31号下午6点到11月5号这段期间，包括唐山市在内的10个城市实施紧急减霾措施。
 <br/>
 <br/>
 重要的钢铁企业唐山钢铁集团就在唐山。根据橙色警报规定，钢铁企业必须将产量减半，而燃煤的火力发电厂也需要将产能限制在最低水平。
 <br/>
 <br/>
 河北唐山市民毛女士：〝现在，污染的工厂都在停产，这些工厂，现在国家政策都是要求它们搬迁的，搬到海边去。〞
 <br/>
 <br/>
 中共环保部在当天的例行通报会上再次警告，今年冬季中国北部和东部可能会有恶劣天气状况。
 <br/>
 <br/>
 为了治理阴霾，中国关停了部分污染企业，实施交通管制，减少北方污染多发地区的煤炭使用。不过，采取措施对抗冬季阴霾，也引发了对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 增长放缓的担忧。生态环境部长李干杰10月27号强调，反对〝一律关停〞〝先停再说〞等应对做法。
 <br/>
 <br/>
 大陆环保人士吴立红表示，中美贸易战导致中国经济不景气，中共监管部门因此放宽了对企业的监管力度。
 <br/>
 <br/>
 大陆环保人士吴立红：〝任何好的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环境保护法.html" target="_blank">
  环境保护法
 </a>
 不去执行的话，等于是在说空话，特别是现在北方已经进了冬季，用煤要增加，雾霾风险会进一步的提高，再加上中美贸易战导致国内经济不景气，可能这些企业加大力度生产，监管部门放松力度监管，它的目的还是利益。〞
 <br/>
 <br/>
 吴立红指出，没有治不好的阴霾，根据他从事环保30多年的心得，中国所有的污染都是人为造成的，为什么越治越污，还是体制有问题。如果环境保护法能真正执行到位，就不存在这样或那样的问题。
 <br/>
 <br/>
 新唐人记者常春、陈洁、元真采访报导
 <br/>
 <br/>
