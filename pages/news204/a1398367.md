### 王岐山担心被〝双杀〞 习近平有同样遭遇
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】6日召开的首届彭博新经济论坛上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 因被称是中国与世界上〝最有影响力的政治人物〞而大惊，立即脱稿论述〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_捧杀.html" target="_blank">
  捧杀
 </a>
 〞和〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_棒杀.html" target="_blank">
  棒杀
 </a>
 〞。翻查公开报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 也曾有同样遭遇，官媒甚至为此刊文称，要警惕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反改革势力.html" target="_blank">
  反改革势力
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 于11月5日至7日，应邀赴新加坡出席2018年首届彭博经济论坛（Bloomberg New Economy Forum），并对新加坡进行访问。
 <br/>
 <br/>
 6日早上，美国企业界大亨、曾任纽约市长的彭博社创办人布隆伯格（Michael Bloomberg）在新经济论坛上，介绍王岐山时，他说，〝在过去30多年，王岐山在中国政治与经济扮演着重要的角色，而他也是中国和世界最具影响力的政治人物。〞
 <br/>
 <br/>
 王岐山听闻大惊失色， 罕见的脱稿回应说，布隆伯格在介绍自己时，有很多溢美之词，〝我一生伴随着不断地从成长到成熟，特别保持着一种冷静和清醒。听到好话的时候，担心是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_捧杀.html" target="_blank">
  捧杀
 </a>
 。听到坏话的时候，我倒无所谓，那叫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_棒杀.html" target="_blank">
  棒杀
 </a>
 。〞
 <br/>
 <br/>
 他还忆述说，自己曾在美国讲过这个题目：〝有名的人、有钱的人，或者是帅哥靓女都会面对着捧杀和棒杀，所以做人啊，要保持这份冷静头脑是不容易的，因为经常有人被捧晕。这是我的一个感受。〞
 <br/>
 <br/>
 翻查公开报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 也曾有同样遭遇，中共官媒甚至为此刊文称，随着改革进入〝深水区〞，应警惕反改革的力量，并称反改革力量在〝捧杀〞习近平。
 <br/>
 <br/>
 日本《外交学人》网站曾刊出题为《Sorry, There’s No ‘Cult of Personality’ in China》（抱歉，中国没有在搞个人崇拜）的文章称，中国没有在搞个人崇拜，副标题是：〝随着改革进入深水区，我们应警惕反改革的力量〞。
 <br/>
 <br/>
 之后，中共官媒《人民日报》旗下的微信公号〝侠客岛〞 逐段翻译了该文，并将标题改为了《谁说中国在搞个人崇拜？》。
 <br/>
 <br/>
 文章批驳了日媒认为的中国出现对于习近平的个人崇拜，称如果没有强有力的领导，反腐运动不会取得如此成就，军改不会得以顺利实施。
 <br/>
 <br/>
 文章点出，随着改革进入另一个〝深水区〞阶段，应警惕反改革的力量。反改革力量一定会借改革之际抹黑习近平等人的形象，所谓的对习的个人崇拜，在中国，这一行为被称为〝捧杀〞，〝即过分赞扬或吹捧某人，最后招致众人反感，使其垮台〞。
 <br/>
 <br/>
 中共文宣系统一直被江派官员掌控，江派大员、前中宣部部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 主管的文宣系统，被认为是江派常委中阻击习近平阵营的前台人物之一。
 <br/>
 <br/>
 中南海知情人士曾对美媒披露，习近平执政的前5年，刘云山俨然成为中共党内与习近平又一分庭抗礼的权贵势力的核心。如果说周永康、薄熙来、徐才厚、令计划 〝新四人帮〞是要〝棒杀〞习，推翻习，最终取而代之；那么刘云山势力则是要〝捧杀〞习近平，最终让被扭曲了的〝习大大〞为己所用。
 <br/>
 <br/>
 中共〝十八大〞后，随着习近平的反腐〝打虎〞运动，及中共军队打乱重组的改革，习已掌控中共党政军权力，中共宣传系统也不断被清洗。尽管〝十九大〞刘离开权力中心，但分析人士认为，刘最终是否能平安着陆还是个问号。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明)
