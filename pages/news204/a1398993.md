### 港媒：北京要走出贸战困境 只有一条路
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 导致中共经济陷入困境，当局已从最初的强硬对抗，转变为向美国示好求和。港媒称，北京一直惯用的政府干预经济的刺激政策，已不能解决问题，走
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_市场自由化.html" target="_blank">
  市场自由化
 </a>
 是北京唯一的选择。
 <br/>
 <br/>
 随着中美贸易争端加剧，中共经济遭受重创、大幅放缓。11月初召开的中共政治局会议，北京当局罕见承认经济下行压力增大，并重提〝六稳〞。即稳就业、稳金融、稳外贸、稳外资、稳投资、稳预期工作。这意味着中共经济的确已面临危机。
 <br/>
 <br/>
 11月1日，川普和习近平罕见就贸易等话题进行了良好的通话。这是近5个月来，两国元首首次通话。川普随后表示，如果美中能达成交易，并且使中国更开放以及更公平，对北京也有好处。
 <br/>
 <br/>
 一直以来，川普政府所要求的就是中共经济进行结构性改革，遵循公平贸易原则，和美国公平贸易。但中共害怕自由贸易会动摇其政权统治，因此选择与美国对抗，致使现在中共经济与政局均进入两难境地。
 <br/>
 <br/>
 为应对贸易战，中共近期全面放宽宏观政策，三次削减商业银行的存款准备金率以增加流动性，批准更多地方政府发行债券，以及放宽金融监管。
 <br/>
 <br/>
 但所有近期数据表明，中共经济增长正大幅放缓、前景恶化。香港《南华早报》11月11日的报导说，中共的政府干预刺激措施，已不再能解决问题。
 <br/>
 <br/>
 中共第三季度GDP的实际增长率降至6.5％，是过去十年以来的最低点。不少经济专家认为，未来几个月中共经济环境将会变得更糟。
 <br/>
 <br/>
 美国已向价值2500亿美元的中国商品征收惩罚性关税，如果明年1月1日，美国将其中2000亿关税税率提高到25％，中国的经济情况还会更糟。
 <br/>
 <br/>
 9月份中国工业生产增长率从8月份的6.1％放缓至5.8％，这是自2015年以来首次增长率低于6％。更为严峻的是，贸易战风波已令大批外资企业把生产线从中国撤离，而且也抑制了投资者在中国投资的欲望和信心。
 <br/>
 <br/>
 10月份，中国制造业采购经理人指数（PMI））为50.2，比上月下降0.6个百分点。有分析师认为，PMI指数继续大幅下滑，表明短期经济的下行压力较高。
 <br/>
 <br/>
 在1998年和2008年的全球经济危机期间，北京也曾采取扩大信贷以及对基础设施进行资本投资来保持其经济增长。但是，面对贸易战的压力，现在这种干预不再像以前那样有效。
 <br/>
 <br/>
 报导指出，这种投资往往导致产能过剩，浪费以及随之而来的大量坏账。国际货币基金组织（IMF）此前曾多次发出警告，中共过度依赖信贷的经济策略引发巨额债务，已达到危险水平。
 <br/>
 <br/>
 哈佛大学经济学教授、IMF前首席经济学家肯尼斯．罗格夫（Ken Rogoff）近日表示，中共过于依赖举债和投资来促进经济增长，如果这种模式无法继续前进，可能会引爆下一次金融危机。
 <br/>
 <br/>
 报导说，由于过去几十年的投资热潮导致基础设施饱和，中共对基础设施的债务融资投资越来越不可持续。目前，中共经济三个增长引擎投资、出口、消费均陷入困境。因此，经济改革和市场开放已成为解决中共经济疲软的关键。
 <br/>
 <br/>
 中共只有经济转型，走
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_市场自由化.html" target="_blank">
  市场自由化
 </a>
 之路，从一个以国家主导、投资和推动出口的模式，转变为以创新、发展服务业等因素推动的模式，才能使中国公司及其产品在全球市场上更具竞争力，才能走出这场贸易战引发的经济困境。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
