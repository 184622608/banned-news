### 【禁闻】10月30日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】下面我们来关注一下中国大陆各地民众的维权情况。
 <br/>
 <br/>
 1. 湖南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 朱承志被刑事拘留
 <br/>
 <br/>
 湖南维权人士朱承志今年4月底在苏州祭奠已故异议人士林昭被抓，遭指定居所监视居住半年之后，10月29号再被苏州警方刑事拘留。
 <br/>
 <br/>
 2. 上海举办进博会 数十维权人士被抓
 <br/>
 <br/>
 中共商务部和上海市政府主办的首届中国国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进口博览会.html" target="_blank">
  进口博览会
 </a>
 ，11月5号至10号将在上海国家会展中心召开，上海当局因此加强维稳，据报导，截止到10月29号，已经有几十名维权人士被抓走。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 案 陈天茂杨秀琼被判缓刑获释
 <br/>
 <br/>
 四川维权人士杨秀琼、陈天茂与〝六四天网〞创办人黄琦因同一事件被抓，但二人被当局另案处理，日前他们已经认罪，并被判缓刑获释。而黄琦坚持不认罪，案件迟迟不审，他身患重病，在狱中不获治疗。
 <br/>
 <br/>
 4. 黑龙江村民张铁华
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宅基地.html" target="_blank">
  宅基地
 </a>
 被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强征.html" target="_blank">
  强征
 </a>
 <br/>
 <br/>
 黑龙江哈尔滨市松北区村民张铁华家的宅基地，遭当地政府强征，10月18号，在双方未达成协议的情况下，政府派人强行施工，张铁华报警后，对方虽然停止施工，但当天夜间又将两车泥石卸在她家的宅基地里。
</div>
