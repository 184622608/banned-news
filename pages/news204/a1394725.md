### 【禁闻】中共处理孟宏伟手法 自毁国际形象
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】
 <br/>
 <br/>
 1.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 处理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 手法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自毁.html" target="_blank">
  自毁
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际形象.html" target="_blank">
  国际形象
 </a>
 <br/>
 <br/>
 中共公安部副部长、国际刑警组织主席孟宏伟9月下旬失踪，连以协调国际警力合作为宗旨的国际刑警也不知道他的踪影，事件沸沸扬扬一个多星期之后，中共公布孟宏伟被中纪委扣留调查。
 <br/>
 <br/>
 10月9号，台湾中央社评论说，中共处理孟宏伟事件的手法，凸显其残酷本质，严重自毁国际形象。
 <br/>
 <br/>
 同一天，《纽约时报》也报导说，孟宏伟突然神秘失踪、被查，给中共官员，以及让中共官员担任领导职务的国际机构，留下了不确定的阴云。
 <br/>
 <br/>
 报导引述曼谷人权倡导者迈克尔．卡斯特的话说:〝想像一下，如果有一天，中国出了一位联合国秘书长，那么他有一天也会失踪。中共在国际规范的所有概念或程序之外运作，其无所顾忌的程度非常令人担忧。〞
 <br/>
 <br/>
 《纽约时报》说，中共的法制是一个不透明、高度政治化的制度，这个制度当初本应该令孟宏伟失去任国际刑警组织主席的资格。现在，中共还在试图证明自己能在全球事务中扮演更重要角色，而孟宏伟事件对这种努力，将是一次严重的、自作自受的打击。
 <br/>
 <br/>
 2. 彭博：美国拟列中共为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货币操纵国.html" target="_blank">
  货币操纵国
 </a>
 <br/>
 <br/>
 中美贸易战升级之际，白宫准备再就滙率问题向中共施压。
 <br/>
 <br/>
 据《彭博社》报导，美国财政部一名高级官员10月8号表示，美国财政部长姆钦持续受到来自白宫的压力，要求在下周公布的半年度贸易伙伴货币报告中把中共正式列为货币操纵国。美国财政部正在考虑是否可行。
 <br/>
 <br/>
 报导说，美国政府正在密切关注人民币在近半年来大幅贬值，质疑北京故意通过货币贬值来应对贸易战。
 <br/>
 <br/>
 如果被美国列入〝汇率操纵国〞正式名单，中国可能面临额外的惩罚措施或贸易制裁。
 <br/>
 <br/>
