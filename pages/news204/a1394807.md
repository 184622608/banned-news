### 崔永元自曝遭多次传唤 有人出价500万要其手脚
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】引爆中国娱乐圈〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞黑幕的央视前主播
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 ，自曝受税务总局刁难，警察已三次传唤他；另外，还有人出价500万元（人民币，下同）要其手脚。他还曾说自己愿意花费8个亿求威胁者放他一马。
 <br/>
 <br/>
 苹果日报10日称，一名叫〝天仙哥哥V〞的神秘网友，在〝今日头条〞平台发帖称，为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 选好了墓地和遗照，还贴出了图片，该网友又引述消息说，有人以500万元要崔永元手脚，并〝提醒〞崔出入要小心。
 <br/>
 <br/>
 崔永元回应说：〝已经做了最坏的打算，那就走着瞧吧。〞目前有关帖文已被删除。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-10/p9073621a812779047.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-10/p9073621a812779047-ss.jpg"/>
   <br/>
   <font size="-1">
    一名叫〝天仙哥哥V〞的神秘网友发帖称，为崔永元选好了墓地和遗照，还贴出了图片。（网络截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 之前的10月7日，崔永元在微博发贴说，他已经第三次被中共税务总局叫去问话，两名陌生面孔的警察对他补充提供的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞证据看都不看。
 <br/>
 <br/>
 崔永元还说，警察拐弯抹角地询问他，从哪里弄到的〝阴阳合同〞。他说，上海公安局经侦大队也对他，以及他所参与的所有公司进行了彻底调查，对他以前的助理也不断询查，彻夜询查。他自称自己的危险来自四面八方。
 <br/>
 <br/>
 10月4日，崔永元也在微博发文称，在这5年的时间里，有一个名叫司马3忌的人，一直不停地对他进行骚扰和诽谤，而且还对他家人进行了人身攻击，他愿意出8亿元钱给这个人，求放过自己。
 <br/>
 <br/>
 在崔永元表达对自身安全的忧虑之后，10月9日，《检察日报》以《举报人遭威胁，法律如何保护》为题做出回应。
 <br/>
 <br/>
 报导通过采访清华大学法学院教授张建伟的方式喊话说：当举报人的人身安全受到威胁，可以向公安机关报案。公安机关应查明举报人是否受到实质性威胁，以及威胁到了何种程度，进而在自己的职责范围内，为其提供适当的保护。
 <br/>
 <br/>
 10月10日，上海市公安局官方微博也发通报称，上海公安机关已成立调查小组，并多次联系崔永元，相关举报问题一旦查证属实，将依法依规处理。
 <br/>
 <br/>
 由此看来，引爆中国娱乐圈〝阴阳合同〞丑闻的崔永元，恐怕来头并不简单。如果追溯崔永元爆料的过程，不难看出，其背后似乎一直有支持、有角力。
 <br/>
 <br/>
 5月底，崔永元开始爆料中国娱乐圈以〝阴阳合同〞逃税事件，搅动大陆舆论并波及了整个娱乐圈，女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 因此被失踪4个月后，官媒10月3日称，范冰冰缴纳8.8亿元人民币罚款可免予刑责。
 <br/>
 <br/>
 在范冰冰被判罚之后，有分析说，北京当局选择罚款而非刑罚，意在解决中美贸易战引发的国库空虚问题，范冰冰只是一个开头。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
