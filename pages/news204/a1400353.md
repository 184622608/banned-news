### 一只蟑螂酿大祸！贵阳3人骨折 6人遭拘
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】近日，在贵州省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贵阳.html" target="_blank">
  贵阳
 </a>
 市，因为一只
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蟑螂.html" target="_blank">
  蟑螂
 </a>
 ，两男发生口角且扭打一团后，双方还各自叫来亲戚朋友助阵，十多名男子持钢、板凳大打出手，结果3人被打骨折，6人遭刑拘。
 <br/>
 <br/>
 据陆媒22日报导，黑衣男子赵某，10日晚上9点多，来到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贵阳.html" target="_blank">
  贵阳
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花溪区.html" target="_blank">
  花溪区
 </a>
 一家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网吧.html" target="_blank">
  网吧
 </a>
 里面，坐在了另一名男子王某的旁边，打开电脑开始上网，没过几分钟，赵某突然发现腿上有一只
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蟑螂.html" target="_blank">
  蟑螂
 </a>
 ，就顺手一弹，弹到了王某的身上。
 <br/>
 <br/>
 看到蟑螂被弹到了自己身上，王某勃然大怒，便与赵某产生了口角，双方互不相让争执逐渐升级。随即两人又扭打在一起。因各自都没占到便宜，很快也就分开了。
 <br/>
 <br/>
 然而，气不过的王某这时又叫来了同在这家网吧上网的另外几名同伴，再次殴打赵某，赵某被打后不甘心，于是便拿出手机开始叫人。
 <br/>
 <br/>
 贵阳市警方称：赵某是江西人，他就叫了网吧附近的一些亲戚朋友，来到网吧找到王某等几名男子，双方手持钢管、板凳，展开激烈打斗，10多个人从网吧内打到网吧外，持续了将近十分钟的时间，才逐渐散去。
 <br/>
 <br/>
 据了解，事发后，这些参与群殴的人都不同程度受伤，其中有3人被钢管打成骨折。目前警方抓获并刑事拘留6名主要犯罪嫌疑人，案件还在调查处理当中。
 <br/>
 <br/>
 事实上，在中国为一点小事大打出手事件层出不穷，人们早已经见怪不怪了。时评人士李林一说，其实这些事情的发生，祸根都是中共，用中共党文化教育出来的人，为一点小事
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_打群架.html" target="_blank">
  打群架
 </a>
 非常普遍，有时甚至打到了国外，把人都丢到国外去了。
 <br/>
 <br/>
 李林一说，中国有五千年的文明，本是人人称赞的礼仪之邦。中共统治大陆短短几十年后，摧毁了中国的仁、义、礼、智、信传统价值后，用无神论、丛林规则的斗争学说教育出来的中国人，不仅是男子互相为一点小事大打出手、甚至杀人，就是女子也因一点小事大打出手，完全没有了那种儒雅、文质彬彬、大家闺秀等的风范。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
