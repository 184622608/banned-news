### 【禁闻】10月28日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 上海狂犬
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗受害者.html" target="_blank">
  疫苗受害者
 </a>
 母女双双被失踪
 <br/>
 <br/>
 上海复旦大学女硕士、狂犬疫苗受害者谭华，9月初到北京维权，遭当局强迫失踪已经近两个月，她的母亲华秀珍9月中旬透露了她是被街道维稳办软禁的消息后也被失踪。母女至今没有音讯。
 <br/>
 <br/>
 2. 江苏女子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 中被逼死 尸体被扣三年
 <br/>
 <br/>
 江苏省常州市天甯区公民江国兴的妻子王平珍，3年前在政府的强拆中被逼喝农药死亡，尸体一直被当地公安强行扣押。10月27号是王平珍的忌日，江国兴在警察的监管下，前往武进区横林殡仪馆看望被妻子的遗体。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_寻找高智晟.html" target="_blank">
  寻找高智晟
 </a>
 活动发起人李发旺病重入院
 <br/>
 <br/>
 参与发起〝寻找高智晟万里行〞活动的山西民主人士李发旺，日前病重入院。他去年为营救高智晟而入狱，在狱中被迫停止服药致使糖尿病加重，导致多种疾病并发。
 <br/>
 <br/>
 4. 声援杨前案 广西多人遭警告
 <br/>
 <br/>
 广西拆迁受害者杨前，因维权被以〝寻釁滋事罪〞关押在合浦县看守所，前不久传出他遭电击酷刑、虐待性侵的消息，10月25号，律师和家属得以会见他，但随后都被噤声，几位关注事件的当地维权人士也被警告不得声援杨前。
</div>
