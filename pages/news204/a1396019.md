### 〝顶回〞中共已成全球趋势 美媒：这只是开始
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 开打后，美中关系急转而下。目前，美国和其盟友正在对中共进行全面反制。分析人士指出，一个全球〝顶回〞中共影响的趋势正在形成，而这一切只是刚刚开始。
 <br/>
 <br/>
 <h4>
  全面〝封堵〞中共
 </h4>
 <br/>
 美国副总统彭斯近日发表的白宫对华政策演说，全方位抨击中共，被指是美国对中共下达的檄文。美国政府正在从经济、政治、军事、外交等层面，全面反制中共这个〝战略竞争对手〞。外界普遍认为，美中已进入〝新冷战〞时期。
 <br/>
 <br/>
 在经济上，美国有〝封堵〞中共的战略考量，在美加墨三国新协议中的〝毒丸〞条款，阻止签约国与非市场经济体（中共）达成贸易协议，被指是用来孤立中共。
 <br/>
 <br/>
 美国商务部长罗斯表示，〝毒丸〞条款还将在未来美国与其他国家达成的贸易协定中被复制。如此一来，中共将被排除在以美国为主导的国际贸易体系之外。
 <br/>
 <br/>
 不仅如此，美国与其他盟友国家正建立〝统一战线〞，从情报分享到经济领域全面〝封堵〞中共。美国之音报导，由美国、英国、澳大利亚、加拿大和纽西兰的情报机构组成〝五眼联盟〞，正在加强与德国和日本的合作，交换有关中共在全球扩张的情报。
 <br/>
 <br/>
 〝顶回〞中共的影响差不多已经成为全球的一个趋势。美国国际战略谘询与商业发展公司克伦普顿集团（Crumpton Group）的中国主管、政治分析人士裘德．布兰切特（Jude Blanchette）表示。
 <br/>
 <br/>
 美国智库詹姆斯敦基金会（Jamestown Foundation）最近举行了中共军力和影响力的研讨会，布兰切特在研讨会表示，世界各国都在讨论如何制定法律来监管外国，尤其是中共的投资。
 <br/>
 <br/>
 他说：澳大利亚制定了新法律来监管来自中共的投资，虽然他们没有明确的说出来。在很多发达国家，对中共掠夺性〝国家资本主义〞的担心几乎是普遍的。
 <br/>
 <br/>
 目前，欧洲三大经济体德国、法国和英国都在采取措施限制中共的投资。今年夏天，德国政府阻止一家中国公司对德国西北部艾伦镇的机床制造企业--莱菲尔德金属旋压机制造公司（Leifeld Metal Spinning）发起的收购，理由是交易可能会〝危及德国的公共秩序与安全〞。
 <br/>
 <br/>
 后来，德国政府出于安全政策考虑，又叫停了国有复兴信贷银行（KfW），收购能源公司〝50赫兹〞公司20%的股份的计划。
 <br/>
 <br/>
 今年7月，法国将圣纳泽尔（Saint-Nazaire）STX造船厂暂时收归国有的决定，与对中共的担忧不无关系。
 <br/>
 <br/>
 英国政府也计划赋予政府更大权力，阻止或撤销外资对英国企业的收购交易。以防止外资在国防和技术等领域的投资，危害英国国家安全。中国公司2016年参与收购欣克利角C核电站项目曾引发争议。
 <br/>
 <br/>
 近日，美日欧盟正联手推动对世界贸易组织（WTO）的改革，针对中共国有企业补贴，盗窃知识产权以及强制转让技术等行为，采取行动。
 <br/>
 <br/>
 乔治城大学美中全球议题对话项目研究员韦德宁（Dennis Wilder）说：在经济领域，欧洲和日本等发达经济体对中共与美国有着类似的担忧。很多美国提出的担忧，欧洲和其他亚洲国家都提出过。
 <br/>
 <br/>
 <h4>
  〝
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
   一带一路
  </a>
  〞频频遭〝退货〞
 </h4>
 <br/>
 另外，〝一带一路〞国家也在抵制中共影响力，北京5年前极力推进的〝一带一路〞项目，让沿途国家陷入债务陷阱，正频频遭到〝退货〞。
 <br/>
 <br/>
 马来西亚首相马哈蒂尔8月取消前任政府与中共达成的铁路、能源管道两个〝一带一路〞项目。10月，西非国家塞拉利昂宣布，取消一项约4亿美元中共资助的机场项目。塞拉利昂政府官员认为，这个项目〝不划算〞。
 <br/>
 <br/>
 有分析指，〝一带一路〞沿途国家日渐意识到地缘政治安全性，他们不愿意让中共在未来控制自己的国家利益。目前，美国正开始大规模扩大对外援助，为非洲、亚洲和美洲的基础设施项目提供资金，被指是抗衡中共〝一带一路〞。
 <br/>
 <br/>
 <h4>
  海上军事联盟
 </h4>
 <br/>
 在军事上，一些国家也正在形成针对中共的海上联盟。10月2日，澳大利亚联合英国、纽西兰、马来西亚、新加坡四国，在敏感的南中国海海域举行〝国际安全演习〞。而且，在美国的影响下，越来越多的国家加入了自由航行计划，包括远在万里之外的英法两国。有分析人士称，一个全球〝顶回〞中共影响的趋势正在形成，而这一切只是刚刚开始。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉）
