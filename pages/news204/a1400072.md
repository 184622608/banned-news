### 中共士兵成越战炮灰？军报罕提三个教训
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】1979年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中越战争.html" target="_blank">
  中越战争
 </a>
 ，中共官方一直对此讳莫如深。日前，中共军报罕见发表文章称，中共军队在中越战争中得出三个教训。此前的一些解密资料中已曝光，中共军队从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战术素养.html" target="_blank">
  战术素养
 </a>
 到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事装备.html" target="_blank">
  军事装备
 </a>
 ，远远落后越南，士兵被当成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炮灰.html" target="_blank">
  炮灰
 </a>
 ，伤亡惨重。
 <br/>
 <br/>
 北京时间11月19日，中共军方喉舌军报发表文章，谈到1979年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中越战争.html" target="_blank">
  中越战争
 </a>
 （中共称自卫反击战）的教训。文章称，中共军队有三大教训。
 <br/>
 <br/>
 首先是在对越南作战之前，中共军队缺乏针对性训练，这在战场中是要用鲜血和生命来偿还的。其次是指挥官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事素质.html" target="_blank">
  军事素质
 </a>
 落后，不会指挥合成作战。
 <br/>
 <br/>
 再者，当时中共军队中不少部队只有理论知识，没有实战经验。参加中越战争的部队，除了少数部队之前在西藏边境打过仗外，其余部队近30年未曾打过仗。
 <br/>
 <br/>
 公开资料显示，中越战争始于1979年2月17日，3月16日结束。中共当时集结20万军人，在云南至广西772英里的边防战线作战。
 <br/>
 <br/>
 2016年，网络上有文章称，中越战争是中共对越南发动的战争，由邓小平力主并迫使当时的领导人华国锋同意。战争的军事目的是援助柬埔寨红色高棉政权，迫使越南从柬埔寨撤军；政治目的则是教训一下投靠苏联的越南政府。但是这场战争，共军打得极为惨烈，用血流成河来描述决不夸张。
 <br/>
 <br/>
 同年11月，作者佚名也在网上曝光了鲜为人知的中越战争内幕。文章称，由于文革期间中共部队光突出政治，很少军事训练，共军军事素质差。战士不会打仗，军官不会指挥，且缺乏山地、丛林战与夜间战斗的训练，装甲部队和步兵不能协同作战，为此付出了巨大代价。
 <br/>
 <br/>
 作者披露，战争刚刚打响两天，中共军队伤亡就达4000多人。进入实战初期，中共军队死亡率之高令人震惊，个别连队伤亡率甚至到达了90％。加之医护人员缺乏，大量伤员得不到及时救治死亡。
 <br/>
 <br/>
 西方媒体此前在报导中越战争时指出，中共军队对进行一场现代战争毫无准备，在重武器装备和用兵理论方面同越南人相比大为逊色。而且，中共军队高级参谋人员年龄老化，仍然用传统的〝人海战术〞冲击敌人的阵地，在越南付出了惨痛代价。
 <br/>
 <br/>
 文章还披露，文化大革命时期，中共部队受到冲击，兵工厂生产质量下降，武器低劣，在战场上造成了严重的后果。中共士兵的手榴弹扔过去不爆炸，冲锋枪开两下就卡壳，甚至炮弹在炮膛里就爆炸的事情非常多，共有500多名中共士兵没有死在敌人的炮火下，而是死在了自己的劣质武器手中。
 <br/>
 <br/>
 相反，当时的越南刚刚结束和美军的战争，越军作战经验丰富，使用的是美军装备、苏联援助的大量军火。越南士兵普遍装备AK冲锋枪，而中共士兵还在使用56式半自动步枪，士兵甚至连钢盔都没有装备。
 <br/>
 <br/>
 共军一个四0火箭筒阵地遭越军炮击，弹片就从班长李建国右侧耳朵上缘将半个头顶削去，脑浆崩了一地，当场死亡。
 <br/>
 <br/>
 作者说，从中共军队暴露的问题来看，1979年时的作战方法还跟50年代差不多，而且其惨烈程度直逼朝鲜战争。作者举例，老山守卫战役中的一连官兵，除一人炸瞎了双眼外，其余全部丧生。
 <br/>
 <br/>
 曾有文章称，中越战争是中共建政以来，打得最窝囊、最没水平的战争。战争到底死了多少人，双方都没有公布数字，战争结束后叶剑英、李先念等有个内部讲话，中方死亡4万8000人，越南死亡也是4万8000人，但中方全是年富力强的小伙子（包括支前抬担架的民兵），越南则包括所谓的民兵（包括妇女、老人等）。言外之意得不偿失。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
