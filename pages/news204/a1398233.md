### 王岐山因一称呼大惊失色  罕见脱稿澄清
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】在新加坡举行的首届彭博新经济论坛上，美国企业界大亨布隆伯格，称赞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 是中国与世界上〝最有影响力的政治人物〞，王岐山听闻大惊失色， 罕见的脱稿提及要保持〝冷静〞、〝清醒〞，因为面对捧杀，经常有人被捧晕。
 <br/>
 <br/>
 香港《南华早报》11月6日报导说，美国企业界大亨布隆伯格（Michael Bloomberg）在新加坡圣淘沙嘉佩乐酒店，出席首届彭博新经济论坛（Bloomberg New Economy Forum）时，大力称赞中国国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 ，是中国与世界上〝最有影响力的政治人物〞。
 <br/>
 <br/>
 王岐山听闻似乎大惊失色，罕见的发表了即席脱稿澄清。王岐山说，纽约前市长、彭博社创办人布隆伯格，在介绍他的时候，有很多溢美之词，自己听到好话的时候，担心是捧杀。听到坏话的时候，那叫棒杀。
 <br/>
 <br/>
 王岐山表示，保持〝冷静〞、〝清醒〞很重要，他忆述，他在美国曾说过：〝有名的人、有钱的人，或者是帅哥靓女都会面对着捧杀和棒杀〞〝所以做人啊，要保持这份冷静头脑是不容易的，因为经常有人被捧晕。〞
 <br/>
 <br/>
 王岐山接着又岔到其它话题上调侃说，早前在介绍布隆伯格时，说他是纽约的第108任市长，〝108位的市长，我说我是多少任的北京市长啊？〞〝我说中国人怎么就没把这个排一排，看看王某人担任北京市长的时候是第几百任。我想肯定是几百任。〞
 <br/>
 <br/>
 最后，王岐山表示：〝放松一下气氛……下面我将宣读一份我的稿子。〞
 <br/>
 <br/>
 据悉，王岐山于11月5日至7日应邀赴新加坡出席2018年创新经济论坛并对新加坡进行访问。
 <br/>
 <br/>
 〝创新经济论坛〞旨在促进中美关系发展，由中共国家发改委主管的经济智库〝中国国际经济交流中心〞和彭博社合办。论坛的发起人是美国富商兼纽约前市长彭博、美国前国务卿基辛格（Henry Kissinger）、前财长保尔森（Hank Paulson）等人。
 <br/>
 <br/>
 据报导，出席〝创新经济论坛〞的美国政商界人士，包括：美国前国务卿基辛格（Henry Kissinger）、美国前财长保尔森（Hank Paulson）、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 前战略顾问班农（Steve Bannon）等人与王岐山都保持着长期的交往。
 <br/>
 <br/>
 因此，对于王岐山而言，出席此次〝创新经济论坛〞更像是一场与美国〝老友〞的聚会。
 <br/>
 <br/>
 王岐山〝创新经济论坛〞上正式发言时说，中美两国对扩大经贸合作都有良好的愿望。中方愿与美方就双方关切问题展开磋商，推动经贸问题达成一个双方都能接受的方案。
 <br/>
 <br/>
 王岐山还强调，当前世界面临的重大问题，都需要中美紧密合作，中美合则两利，斗则两伤，中方愿与美方就双方关切问题展开磋商，推动经贸问题达成一个双方都能接受的方案。
 <br/>
 <br/>
 习近平与川普（特朗普）11月1日进行了贸易战以来的首次通话。习近平也表示，两国要推动中美经贸问题达成一个双方都能接受的方案。
 <br/>
 <br/>
 次日，川普在接受美媒采访时也表示，中方〝非常希望达成协议〞，〝我认为我们将和中国达成协议〞，自从发生贸易冲突后，中国经济下行。
 <br/>
 <br/>
 而王岐山的发言似乎也再次显示，北京有意让步，双方有望在阿根廷G20峰会会面时，达成某种贸易协议。外界期待颇高，很多人认为这或将成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 的转折点。
 <br/>
 <br/>
 不过，王岐山在演讲中也说，国家应相互尊重彼此的发展道路。被认为，这是对美国川普政府发出的信号，即不要试图改变中国由中共领导及国有经济占主导地位的发展模式。
 <br/>
 <br/>
 中美贸易战爆发至今，不少观点认为，双方的紧张关系，源于比贸易争端更深层次的原因。美国副总统彭斯10月4日发表的措辞强硬的演讲，全面批评中共内政、外交、经济、军事政策后，明确表示，要〝调整美国对中共经济和战略关系〞。
 <br/>
 <br/>
 如果王岐山上述那句话意味着北京当局无意进行制度改革，那么又该如何解决美国提出的窃取知识产权、市场准入、不公平的政府补贴等问题呢？中美贸易战何时熄火，也必将成为一个疑问。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉)
