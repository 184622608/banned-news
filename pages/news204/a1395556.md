### 陆中产阶级大量移民 不惜受骗认栽
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】港媒近日报导，目前大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中产阶级.html" target="_blank">
  中产阶级
 </a>
 出于对中共局势的担忧与害怕，忙于把资金和子女转移到国外，许多人因此
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受骗.html" target="_blank">
  受骗
 </a>
 。但不少人却表示，受骗上当损失的钱财，会打官司。无论最终能否拿回财产，都会继续寻求海外投资
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 的契机。
 <br/>
 <br/>
 一直以来，愈来愈多大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中产阶级.html" target="_blank">
  中产阶级
 </a>
 急于把资金与子女移转到国外，这些人藉由投资所在国指定的建设项目，换取长期居留签证，被称为投资
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 ，以此为手中的财产留一条后路，因为他们知道，共产党向来是不喜欢私有财产这个概念。这种担忧与害怕从未消除过。
 <br/>
 <br/>
 另外，大陆每况愈下的空气品质、食品安全、疫苗丑闻、不公正的教育体制以及法制与人权的缺失，也逼使大陆人背井离乡，向海外寻找落脚地。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 是许多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资移民.html" target="_blank">
  投资移民
 </a>
 的首选，良好的空气、优质的生活环境与宽松的教育体系等等因素，吸引着大陆人。
 <br/>
 <br/>
 但另一方面，不少人在办理移民的过程中，遭到诈骗，财产尽失。
 <br/>
 <br/>
 〝澳信〞(Ausin China)房地产经纪公司瞄准商机，以免费旅游为噱头，召集有意投资移民的大陆人，收缴订金20万元人民币。〝澳信〞承诺，在顾客返回大陆后35天归还订金，但这些投资人迄今未拿回订金。
 <br/>
 <br/>
 〝澳信〞的另一项建筑投资，也令投资者血本无归。来自大陆的3名投资人，原本素不相识，在澳洲加入一个建筑投资的微信社群才认识彼此，但现在成了同病相怜的受害者：由于负责接洽他们的〝澳信〞毫无预警地倒闭了，约200名成员的近7000万澳元的投资金额下落不明。
 <br/>
 <br/>
 部分受害者计划提告澳信与澳信集团(Ausin Group)。受害者之一张先生表示，由于忧虑大陆每况愈下的经济环境，此番官司即使争不回财产，都会继续寻求海外投资移民契机。
 <br/>
 <br/>
 投资移民大增引起北京当局注意，大陆外汇管理局(SAFE)几度收紧政策，控制资金流动。同时，因为大陆人大量购房投资，使居屋价格飙升，当地居民难以负担房价，澳洲、纽西兰、加拿大与马来西亚等国相继取消投资移民政策，此番种种，令计划移民的大陆家庭进退维谷，更添不安。
 <br/>
 <br/>
 (记者李蒨蒨报导/责任编辑：李泉）
