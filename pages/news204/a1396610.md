### 刘强东性侵惹祸？落榜官方〝百名民企〞名单
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】2018年是中共的改革开放40周年，24日官方推出〝百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杰出.html" target="_blank">
  杰出
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民营企业家.html" target="_blank">
  民营企业家
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_名单.html" target="_blank">
  名单
 </a>
 〞，此前备受关注的中共电商巨头京东集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 ，与万达集团董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王健林.html" target="_blank">
  王健林
 </a>
 均未上榜。
 <br/>
 <br/>
 10月24日，官媒称，在中共改革开放40周年之际，为宣传民营经济发展的成就，鼓励广大非公有制经济人士不忘创业初心，接力改革，中央统战部、全国工商联决定推荐宣传百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杰出.html" target="_blank">
  杰出
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民营企业家.html" target="_blank">
  民营企业家
 </a>
 。
 <br/>
 <br/>
 当天，全国工商联在北京举行新闻发布会，发布了《改革开放40年百名杰出民营企业家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_名单.html" target="_blank">
  名单
 </a>
 》。
 <br/>
 <br/>
 但这份名单中，并没有出现此前备受关注的万达集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王健林.html" target="_blank">
  王健林
 </a>
 ，和电商巨头京东集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 。
 <br/>
 <br/>
 王健林此前因涉嫌在海外违规并购而招致北京重罚。
 <br/>
 <br/>
 而刘强东所创立的京东集团公司，自1998年发展至今，市值约500亿美元，跻身全球500强企业。但刘强东因8月底在美国涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 被捕丑闻震惊舆论。
 <br/>
 <br/>
 据美国警方9月4日公布的报告，刘强东涉嫌强奸发生在8月31日凌晨1点左右，当晚11时32分被捕入狱，16个小时后获释，0保释金。
 <br/>
 <br/>
 警方对刘强东一案的描述是〝一级性犯罪：强奸既遂〞（CSCR-1,CSC-RAPE-COMPLETE）。
 <br/>
 <br/>
 根据明尼苏达州的法律，一级强暴罪只要最后被定罪，刑期至少12年起。
 <br/>
 <br/>
 有华裔刑案律师认为，刘强东遭起诉的可能性超过90%，案件一旦开始会涉及重新逮捕等问题。
 <br/>
 <br/>
 刘强东强奸事件发生之初，中共外交部就曾罕见介入，发言人华春莹声称，中方正与美国有关部门，了解核实刘强东强奸一案。
 <br/>
 <br/>
 有分析人士认为，中共外交部介入中国人在外国涉刑事的案件，做法颇为罕见。甚至香港前高官何志平，为了帮助中共在非洲发展能源战略，涉嫌在联合国行贿，被美国司法部控告，也未见中共外交部为他出声。
 <br/>
 <br/>
 而刘强东不同，他不仅是中国商界亿万富豪，同时还有中共全国政协委员身份，并且兼任中国民间商会副会长。过去一年，刘强东多次对外发表关于〝共产主义〞的言论，还曾与腾讯首席执行官马化腾一起身穿红军服装，头戴八角帽，拜访中共红色基地延安，号称接受中共红色洗礼。
 <br/>
 <br/>
 如此注重政治名气的中共红顶商人，此次却意外在美国遭遇性侵犯罪指控，无疑是中共一大丑闻。有网友扒出他6月去延安的图片，认为或因他向中共表忠而运程逆转。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：赵云）
