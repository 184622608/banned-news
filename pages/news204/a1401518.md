### 美国国会：川习会应提出法轮功受迫害问题
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】美国国会及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政当局中国委员会.html" target="_blank">
  行政当局中国委员会
 </a>
 （CECC）呼吁，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）在和中国国家主席习近平在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 会晤期间，提出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 受严重迫害等议题。
 <br/>
 <br/>
 〝我敦促
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统向习近平提出宗教自由议题，希望他将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 受严重迫害及其它宗教议题，作为与安全、经济等问题一并提出讨论。〞CECC的共同主席、联邦众议员史密斯（Christ Smith）11月29日接受大纪元记者采访时表示。
 <br/>
 <br/>
 据北美明慧网报导，2018年11月9日，黑龙江哈尔滨和大庆119名法轮功学员被中共警方非法绑架；目前84人依然被关押，35人获释。
 <br/>
 <br/>
 史密斯议员谴责中共19年来持续迫害法轮功，他说，〝中共持续消灭法轮功的做法，是中国近代史上的黑色烙印。〞
 <br/>
 <br/>
 〝中共甚至需要控制中国人的良知（以维持其统治），这导致了可怕的侵犯人权的行为，如：酷刑、任意拘禁和强摘器官。〞

 1999年7月20日，中共前党魁江泽民以法轮功〝真、善、忍〞原则与共产党意识形态不同等为理由，下令镇压。据不完全统计，19年来，至少4,258名法轮功学员被迫害致死，难以计数的法轮功学员被非法绑架、被判刑、遭受酷刑，甚至被活摘器官。
 <br/>
 <br/>
 史密斯议员表示，中共对宗教自由的打压引发诸多社会问题。〝现在有足够的证据显示，那些保护宗教自由的国家，政治上更稳定，经济上更繁荣，恐怖主义和本土暴力事件更少。〞
