### 刘卫国受伤 宣布退出王全璋案代理
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709案.html" target="_blank">
  709案
 </a>
 被抓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 王全璋的妻子李文足21号在推特披露，收到来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋案.html" target="_blank">
  王全璋案
 </a>
 代理律师刘卫国的信息，对方称，因自己意外受伤不再代理王全璋案件。
 <br/>
 <br/>
 10月21号，王全璋律师的妻子李文足，在个人推特上转发了刘卫国律师，有关退出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋案.html" target="_blank">
  王全璋案
 </a>
 件代理的声明。
 <br/>
 <br/>
 在声明中，刘卫国说，他8月20号晨练期间意外受伤，摔断了包括门牙在内3颗牙齿。经山东大学口腔医院多次会诊，认为他已经无力履行辩护职责。
 <br/>
 <br/>
 声明中还表示，他目前已经将自己的病例及身体状况，书面告知审判法院及王全璋家属，并向社会各界声明，正式退出王全璋案代理工作。
 <br/>
 <br/>
 对此，有不少网民认为，〝估计刘律师被打了吧！〞〝估计王全璋出了啥事，中共不可也不敢告人，只能暗害帮忙应诉的律师，让王的案情不了了之⋯⋯〞
 <br/>
 <br/>
 网民的质疑似乎也不是空穴来风，过去3年多来，王全璋案聘请了多名律师，都在中共当局的施压下退出了代理。
 <br/>
 <br/>
 余文生律师妻子许艳表示，李文足说刘卫国确实受伤了，至于背后的原因不太清楚。
 <br/>
 <br/>
 余文生律师妻子许艳：〝我十几分钟前刚跟她（李文足）在一块，现在分开了，她说的也就是和那个发的贴子上的内容差不多。〞
 <br/>
 <br/>
 北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 野靖环说，刘卫国代理王全璋案件过程中，做过不少努力，但法院对他的限制太多了。
 <br/>
 <br/>
 北京维权人士野靖环：〝法院不允许家属聘请的律师，刘卫国是王全璋聘请的律师，王全璋最希望程海（律师）给他代理，法院不同意，法院只同意刘卫国代理，刘卫国也会见过王全璋，在这个过程中间，也给法院互相提了一些条件。〞
 <br/>
 <br/>
 野靖环认为，现在刘卫国之所以退出，可能是他觉得案件太复杂，而且又不能公平公正为王全璋辩护，正好借这次身体受伤退出了代理。
 <br/>
 <br/>
 现年42岁的王全璋，于2015年的709大抓捕中被带走，至今仍音讯全无。过去3年来，李文足四处奔走呼吁，包括多次前往最高法院递交控告申诉材料等，都被以各种理由拒绝。
 <br/>
 <br/>
 今年7月13号，李文足说，刘卫国律师成功与王全璋见面，之后，她又表示，王全璋见到刘卫国时〝非常害怕〞，很多话只敢用口型表达。她怀疑王全璋遭到酷刑折磨。
 <br/>
 <br/>
 新唐人记者熊斌、舒灿采访报导
