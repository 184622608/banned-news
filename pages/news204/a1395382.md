### 【禁闻】10月14日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 贵大前教授杨绍政又被开除农工党党籍
 <br/>
 <br/>
 因在网上发表批评中共〝公款养党〞等言论被开除的贵州大学教授杨绍政，日前又被其所在的中国农工党贵州省委员会开除党籍。杨绍政说，他最近受到来自各方的围剿，包括关闭博客。
 <br/>
 <br/>
 2. 山西矿业员工子女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 不合理安置
 <br/>
 <br/>
 山西汾西矿业集团数百名矿业员工的子女，因不满被不合理安置到外省，10月8号开始，连续三天在集团门前打横幅抗议，10号，遭到特警暴力镇压，多人被打伤。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四天网.html" target="_blank">
  六四天网
 </a>
 创办人黄琦被加〝故意泄密罪〞
 <br/>
 <br/>
 大陆维权网站〝六四天网〞创办人黄琦，两年前被中共指控〝为境外非法提供国家机密罪〞关押至今，10月13号，他的代表律师刘正清接到当局通知书，说黄琦又被加控一项〝故意泄露国家秘密罪〞。
 <br/>
 <br/>
 4. 重庆市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_异议人士.html" target="_blank">
  异议人士
 </a>
 被关精神病3月
 <br/>
 <br/>
 重庆市荣昌区异议人士刘刚，今年6月初，因准备在〝六四〞期间举办纪念活动，被重庆荣昌区警方抓捕，之后被区综治维稳办送到精神病院关押3个月，获释后刘刚的手机被扣押不还。日前，刘刚重新购买了手机才与外界取得联系。
</div>
