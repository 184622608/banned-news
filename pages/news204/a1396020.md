### 湖南公民为被押维权人士朱承志庆生
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】10月18号，一批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南公民.html" target="_blank">
  湖南公民
 </a>
 在湘潭为被关押的知名维权人士
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱承志.html" target="_blank">
  朱承志
 </a>
 举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生日宴会.html" target="_blank">
  生日宴会
 </a>
 ，祝他生日快乐，并要求中共还他自由。
 <br/>
 <br/>
 湖南知名维权人士
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱承志.html" target="_blank">
  朱承志
 </a>
 ，被中共当局关押已经半年多，10月18号是他68岁生日，湖南一些人士为他举办
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_庆生.html" target="_blank">
  庆生
 </a>
 活动，但是受到警方干扰。
 <br/>
 <br/>
 成功参加庆生宴的湖南民众胡剑鸣19号告诉新唐人，宴会在湖南湘潭举行。
 <br/>
 <br/>
 湖南民众胡剑鸣：〝昨天晚上我们在湘潭为他聚了一餐，我们是老朱的朋友呢，早几天我们在召集的时候，我们本来要召集2、30个人的，但是中间出现了一些变故，受到当局的干预，所以昨天那个规模就很小了，只有十几个人。〞
 <br/>
 <br/>
 朱承志多年来一直冲在民主及维权行动前面，受到当地维权界的尊敬。
 <br/>
 <br/>
 湖南民众袁小华：〝朱承志先生是在践行公民价值方面的一个德高望重的老先生，去祭奠林昭啊每年的4月29，福建三网民案的围观啊、全国各地的一些良心犯的声援啊，都有他的身影。〞
 <br/>
 <br/>
 今年4月29号朱承志因到苏州祭奠在文革中遭中共虐杀的北大右派女学生林昭，被苏州警方以〝寻釁滋事〞罪名指定居所监视居住，家属和律师多次要求会见，都遭到拒绝。
 <br/>
 <br/>
 参加活动的人士在宴会上举牌，祝贺朱承志生日快乐，并要求当局释放他，让他自由。
 <br/>
 <br/>
 新唐人记者常春、李沛灵采访报导
</div>
