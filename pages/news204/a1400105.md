### 陕西前人大副主任魏民洲被判无期
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】中共官媒周二通报，陕西省前人大副主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魏民洲.html" target="_blank">
  魏民洲
 </a>
 受贿案宣判，魏民洲受贿超过一亿元人民币，被判
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无期徒刑.html" target="_blank">
  无期徒刑
 </a>
 。
 <br/>
 <br/>
 判决称，1996年到2017年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魏民洲.html" target="_blank">
  魏民洲
 </a>
 担任中共陕西省委常委、商洛市委书记，中共陕西省委常委、秘书长，任陕西省委常委、西安市委书记，陕西省人大常委会副主任等职务期间，非法收受财物共计折合人民币1亿零978万元，构成受贿罪。案发前，魏民洲与他人串供，指使他人销毁部分受贿所得。
 <br/>
 <br/>
 据报，魏民洲被抓时瘫倒在地。
 <br/>
 <br/>
 魏民洲落马后，媒体披露了魏民洲攀附时任中办主任令计划的细节。
 <br/>
 <br/>
 据报导，西安市已有67人因涉及魏民洲等问题线索被处理。
</div>
