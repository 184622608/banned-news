### 美专家：结构性改变是关键  否则90天后贸战重燃
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】在阿根廷结束的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞结束后，白宫发布声明称，中方在四大方面做出了承诺，美方暂停
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加征关税.html" target="_blank">
  加征关税
 </a>
 ，给习近平90天缓冲谈判时间。对此，有美国专家说，如果双方不能在涉及美中关系的结构性问题上达成一致，贸易战还会重燃。
 <br/>
 <br/>
 美国总统川普（特朗普）与中国国家主席习近平于12月1日，在阿根廷布宜诺斯艾利斯举行晚宴后，白宫发表声明称，双方都表示会谈〝高度成功〞。
 <br/>
 <br/>
 声明表示，川普总统同意，在2019年1月1日，不会将2,000亿中国商品10%惩罚性关税提高到25%，而中共同意增加采购美方农业、能源、工业和其它产品，以减少两国之间的贸易不平衡。声明说，中共已同意立即从美国购买农产品。
 <br/>
 <br/>
 另外，中方同意将芬太尼指定为管制物质，这意味着向美国出售芬太尼的人，将受到中国法律规定的最高惩罚。
 <br/>
 <br/>
 声明还说，美中双方同意立即就结构性改变问题，展开为期90天的谈判，这些问题包括强制技术转让、知识产权保护、非关税壁垒、网路入侵和网路盗窃。如果双方无法在这个期限内达成一致，美方将继续把关税税率提高至25%。
 <br/>
 <br/>
 美中双方还一致认为，在朝鲜问题上取得了很大的进展，川普和习近平一起，与金正恩共同努力，以完成朝鲜半岛无核化。
 <br/>
 <br/>
 川普总统在声明中说：〝这是一场精彩的、富有成果的会谈，对美国和中国具有无限可能性。与习主席合作，是我的荣幸。〞
 <br/>
 <br/>
 不过，有美国专家对美中贸易战的前景不乐观。美国智库战略与国际研究中心资深研究员葛来仪（Bonnie Glaser）对美国之音分析说：此次中美的贸易协议是一个占位符（placeholder），本可以在数月前达成。
 <br/>
 <br/>
 葛来仪说，除非美国和中国的谈判人员，就美中经济关系中的结构性的问题达成协议，否则〝 两国可能会在90天后重返贸易战〞，而这些结构性的问题，也是最初促使川普总统实施关税措施的起因。
 <br/>
 <br/>
 华盛顿智库威尔逊中心基辛格美中关系研究所主任戴博(Robert Daly)认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 达成的协议让贸易战暂时停火，给出谈判时间，但是，中共做出川普政府要求的所有结构性改变的可能性很低，在90天的谈判期内要维持积极的氛围也不容易。
 <br/>
 <br/>
 时政评论周晓辉评论说，从双方达成的协议内容看，北京在贸易问题上已不再像以往那样狂妄称〝奉陪到底〞，而是向美国示好。
 <br/>
 <br/>
 他认为这表明在美国强力施压下，北京担心经济进一步下滑，市场信心进一步降低，且国内危机重重，自身已无太多办法应对，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加征关税.html" target="_blank">
  加征关税
 </a>
 后所产生的严重后果，而不得不进行的选择。
 <br/>
 <br/>
 同时也意味着未来贸易战的主导权仍在美国手中。美国此次基于善意，仍选择再次相信北京，愿意就涉及结构性改变的议题谈判，但给出了3个月的时间表。一旦谈判无法达成一致，美国将再次启动加征关税。
 <br/>
 <br/>
 而这无疑给北京上了一个〝紧箍咒〞，这也是美国代表团中强硬派如纳瓦罗，亦赞同的原因。也就是说，头上顶着这个紧箍咒的北京政权，未来也只有两条路可选：
 <br/>
 <br/>
 一是信守承诺，在贸易问题上做出结构性改变；二是将本次达成的协议当成缓兵之计，在与美国谈判的同时，力图在3个月内寻求到更好的办法应对国内外的危机。
 <br/>
 <br/>
 由于早前习近平在与川普会晤时提出的〝百日计划〞并未实现，也同样有理由对未来谈判结果抱持怀疑。而川普政府或许也难以对未来达成协议乐观，因为，北京高层在出访前的几次讲话中，都提到要坚持中共的领导，走中国特色的社会主义道路。
 <br/>
 <br/>
 如果在贸易问题上进行真正的结构性上的改变，那就意味着无法不触及政治体制上的改变，由于中共的底线就是维持统治，影响到其政权和统治的事情，北京高层愿意吗？
 <br/>
 <br/>
 分析认为，不管其是否愿意，是否是真心回应美国的需求，美国给出的3个月限期，可能是其最后的机会了。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
