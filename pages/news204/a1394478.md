### 曾传朝鲜重大车祸罹难  毛新宇携全家现身河南
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十一长假.html" target="_blank">
  十一长假
 </a>
 期间，中共前党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毛泽东.html" target="_blank">
  毛泽东
 </a>
 之孙
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毛新宇.html" target="_blank">
  毛新宇
 </a>
 全家集体现身河南引发关注，早前传闻称，毛新宇在朝鲜重大车祸
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罹难.html" target="_blank">
  罹难
 </a>
 。
 <br/>
 <br/>
 据媒体10月7日报导，中国十一黄金周期间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毛泽东.html" target="_blank">
  毛泽东
 </a>
 孙子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毛新宇.html" target="_blank">
  毛新宇
 </a>
 于3日携妻子刘滨，儿子毛东东和女儿毛甜甜，集体现身老子故里旅游区（河南周口鹿邑），游览了明道宫、太清宫等景区。
 <br/>
 <br/>
 该消息立即引来网民关注，毛新宇此前被传在朝鲜重大车祸中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罹难.html" target="_blank">
  罹难
 </a>
 。
 <br/>
 <br/>
 4月22日晚，中共毛左网站〝乌有之乡〞开设的非法旅游公司，到朝鲜进行〝红色旅游〞期间，遭遇重大车祸，造成36人死亡，2人重伤，死者中有32人为中国公民，4人为朝鲜工作人员。
 <br/>
 <br/>
 事故发生后，有北美中文媒体刊发署名文章说，遭遇车祸的旅行团是〝纪念抗美援朝战争胜利65周年访朝文化交流参观团〞，除了多个毛粉团体和代表人物遇难外，据说毛新宇也在死伤者中。
 <br/>
 <br/>
 事故发生后，朝鲜领导人金正恩不仅到中共驻朝鲜使馆对事故遇难者表示哀悼，还专程到医院看望受伤人员，并在向习近平发送的电文中，使用了〝赎罪〞这样的表述。
 <br/>
 <br/>
 4月26日凌晨，金正恩还动用朝鲜专列，运送34名中国死伤者回国，并率领朝鲜政府众官员，前往平壤火车站送行。足见这次车祸有多么敏感。
 <br/>
 <br/>
 然而对于该事故的详情及死亡者名单，中方至今仍守口如瓶，朝鲜方面也三缄其口。
 <br/>
 <br/>
 在中共外交部举行的一次例行记者会上，有记者追问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜车祸.html" target="_blank">
  朝鲜车祸
 </a>
 的原因，以及遇难者的身份，外交部发言人陆慷都未有正面回应，仅表示如果有进一步消息会及时公布。
 <br/>
 <br/>
 毛家人在事故发生后，曾经通过港媒辟谣说毛新宇没有去朝鲜。5月4日，毛新宇还曾以照片方式露面中共军工机构似有意辟谣。
 <br/>
 <br/>
 9月9日，毛泽东死亡42周年纪念日当天，毛的后代齐聚北京。有大陆自媒体发布的现场图片显示，毛新宇与毛的大女儿李敏都曾现身毛纪念堂。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：祝馨睿）
