### 贸战引发连锁反应 俄专家：中共将面临多国施压
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】G20峰会上川习会谈后，中方作出了大幅度让步并承诺进行更进一步的改革。中美贸易战暂时休战。有俄媒援引专家观点称，中美回复正常的贸易关系，首先对双方的农业都十分有利，但中美将来如果真的达成协议，将引发连锁反应，会有更多国家向中共政府提出同样的贸易优惠要求。
 <br/>
 <br/>
 俄罗斯卫星通讯社4日的一篇报导讨论了中美贸易战的发展动向。该报导援引俄罗斯国立研究型大学〝高等经济大学〞世界经济与世界政治系东方学部主任马斯洛夫（Alexei Maslov）的观点表示，中美暂停贸易战，恢复正常的贸易关系，首先在农业上对美中双方都是有利的。
 <br/>
 <br/>
 这名专家分析指出，中国是世界上最大的粮食进口国，过去长期向美国购买大量粮食。中美贸易战开打以后，为报复美国对中国产品提高关税，中共政府也对进口的美国农产品提高关税，这样做虽然打击了美国农产品生产商，但中国市场上的进口农产品价格也上涨了15%到25%。
 <br/>
 <br/>
 马斯洛夫称，尽管中国开始调整其农产品进口贸易关系，并降低对东南亚国家和印度的关税壁垒，但不可能立即取代美国的进口，因为〝这些国家甚至是加在一起都无法一次性地满足中国市场的食品供应〞，所以关税战对中国自身造成的副作用也很严重。他说：〝因此，恢复正常贸易关系对中国也有利，因为有大量农产品需供应，对美国生产者也有利。〞
 <br/>
 <br/>
 不过，报导接着声称，主要问题在于，如果90天后两国真的达成某种贸易协定或一系列贸易协定，中方在作出进一步的让步后，不排除紧随美国之后，其他国家也可能会要求在中国市场上的贸易优惠。
 <br/>
 <br/>
 俄罗斯媒体的上述说法，引起了中国网友各不相同的反应。
 <br/>
 <br/>
 有中国网友表示，这种说法有一定的依据，因为按照WTO的 most favored nations 原则，中共政府对美国政府的让步应该也适用于所有其它成员国。
 <br/>
 <br/>
 但也有网友对俄媒的这种说法提出反驳，质疑：〝俄罗斯人这是挑拨吗，美国施压是因为500b顺差。其他国家对中国，大都是中国逆差。中国不反压就不错了。〞
 <br/>
 <br/>
 还有中国网友分析称：〝这次Trump的贸易战给两党上了很好的一课，如果打，美国有机会得到更大利益，不打美国就会一点点被中国（中共）蚕食掉。所以未来几年，无论哪个政党执政，与中国的贸易关系都不会太平顺。对于跨国公司根本不可能将宝全部压在中国，寻找中国的替代国家，将是各大公司的重中之重。〞
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：戴明）
