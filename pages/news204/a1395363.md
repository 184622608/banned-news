### 美毒丸条款令北京紧张 王毅接连致电向加墨施压
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】10月1日达成的《美墨加贸易协定》中，有禁止成员国和〝非市场国家〞签署自贸协议的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒丸条款.html" target="_blank">
  毒丸条款
 </a>
 〞，被指川普政府对中共完成贸易合围的前奏。日前，中共外长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 接连致电加拿大和墨西哥外长，对两国施压。
 <br/>
 <br/>
 中共外交部周日（10月14日）消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 周六晚间就《美墨加协定》与墨西哥外长比德加赖通话时强调，任何自贸协定都〝不应针对第三方〞。
 <br/>
 <br/>
 据《美国之音》报导，墨西哥外交部也在声明中说，比德加赖在跟王毅通话时表示，《美墨加协定》并没有阻碍墨西哥与任何国家发展双边关系。
 <br/>
 <br/>
 在此之前，王毅周三（10日）还曾跟加拿大外交部长方慧兰通电话，希望加方以与中方一道〝维护全球自由贸易体系〞，推进中加自贸区建设。
 <br/>
 <br/>
 据中共外交部官网，方慧兰也表示，〝美墨加协定不应损害其他国家的正当权益，加方将根据自身决定推进与其他国家的自贸协定谈判。〞
 <br/>
 <br/>
 《美国之音》报导说，10月1日达成的《美墨加贸易协定》还需要得到三国国会的批准才能生效。中共正在加紧对加墨两国施压。
 <br/>
 <br/>
 《美墨加协定》规定，如果三个成员国中有一方与非市场化国家达成自贸协定，那么其他两方就可以退出该协定，另行建立双边贸易协定。
 <br/>
 <br/>
 《协定》没有指明〝非市场化国家〞就是中共，但是美国商务部长罗斯对路透社说，将该〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒丸条款.html" target="_blank">
  毒丸条款
 </a>
 〞纳入《协定》就是为了对中共施压，以推动中国实现市场经济。罗斯并称，美国可以在跟欧盟和日本等贸易伙伴签订协议时，也纳入类似条款。
 <br/>
 <br/>
 外界分析，如果美、欧、日和加拿大等发达国家联手，中共将陷入空前的经济孤立。近期以来，欧盟和日本等美国盟友的表现，也显示了配合美国、敦促北京遵守贸易规则的意向。北京则重提〝自力更生〞，凸显自身的孤立处境。
 <br/>
 <br/>
 据外媒消息，川普此前警告，如果10月1日零时之前，美加不能达成贸易协议，美方就将抛开加拿大，和墨西哥签署双边协议。为此，加拿大总理特鲁多9月30日晚连夜召开紧急会议，双方赶在最后时刻之前达成了共识。
 <br/>
 <br/>
 路透引述分析指，美方加入的〝毒丸条款〞，令中共与加拿大和墨西哥达成自由贸易协定的希望落空。对墨西哥和加拿大而言，不大可能冒着失去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨加贸易协议.html" target="_blank">
  美墨加贸易协议
 </a>
 的风险，去和中共签署贸易协议。
 <br/>
 <br/>
 中国是加拿大的第二大出口市场，也是墨西哥的第四大出口市场。不过，加拿大和墨西哥的最大出口市场都是美国。
 <br/>
 <br/>
 加拿大前总理哈珀表示，理念相同的国家签订有关条款，可能帮助推动中方开放市场。
 <br/>
 <br/>
 此前，中共〝联欧抗美〞遭遇失败，近日〝联日抗美〞也在降调。港媒曾分析，无论西方民主国家之间如何闹矛盾，中共的共产政治体制对他们来说都是〝外人〞。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：云涛）
