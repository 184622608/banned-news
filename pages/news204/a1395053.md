### 孟宏伟背景复杂 外媒：习近平清理政法系已到微调阶段
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】原国际刑警主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 落马一事，引发国际关注。有消息指，孟曾长期在中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政法委.html" target="_blank">
  政法委
 </a>
 任职，背景复杂。有外媒爆料，孟拥有中共〝极其重要人士〞违法证据，并保存在海外。孟被拿下，显示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清理政法系.html" target="_blank">
  清理政法系
 </a>
 已到微调阶段。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 落马事件，因孟妻在海外曝光演变成国际事件。中共公安部对外通报称，孟宏伟因收受贿赂、涉嫌违法被查，但未说明细节。孟宏伟究竟因何而落马，引发外界猜测。
 <br/>
 <br/>
 据香港的人权组织〝中国人权民运信息中心〞爆料指，孟宏伟拥有中共〝极其重要人士〞违法证据，并保存在海外。这或许是孟被突然拿下的原因之一。
 <br/>
 <br/>
 孟宏伟曾长期在中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政法委.html" target="_blank">
  政法委
 </a>
 任职。自由亚洲10月11日报导，前中共公安部学者、现为美国执业律师的高光俊透露，孟宏伟在来公安部之前，曾担任原政法委书记乔石的秘书，历经几届派系争斗，背景复杂。
 <br/>
 <br/>
 不过，梳理孟宏伟的仕途，发现他是在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 接替乔石后，获得提拔。众所周知，乔石与江泽民派系是死对头，显然，孟宏伟后期应是改换了门庭，重新站队，获得了升迁。
 <br/>
 <br/>
 然而，孟宏伟此次落马，显然站错了队，以致遭到当局清洗。目前在中共公安部网站领导信息一栏，已找不到孟宏伟的资料。公安部纪委书记邓卫平，日前已接任中共公安部副部长一职，坐上第三把交椅。
 <br/>
 <br/>
 邓卫平一直在福建从事政法工作，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 福建主政时的旧部。除了邓卫平，公安部部长赵克志及分管日常工作的副部长王小洪都是习近平的亲信。
 <br/>
 <br/>
 高光俊表示，公安系统大换血，显示习近平要把刀把子完全掌握在手，全部换上自己信得过的人。公安部12名党委委员，除了两名业务专家外，其余全部被换掉。
 <br/>
 <br/>
 时事评论人士张刚分析认为，孟宏伟被拿下，显示习近平
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清理政法系.html" target="_blank">
  清理政法系
 </a>
 统，已经到了〝微调〞的阶段。
 <br/>
 <br/>
 他表示，习近平执政六年，先埋头军队、后牢握军队枪杆子、现在又对政法系统、武警、公安，已有总体控制。从总的来讲，习近平已全部掌握，但是实际来讲，他并不确定。所以，习近平还在不断调整，框架完了之后的块块条条的调整。
 <br/>
 <br/>
 孟宏伟落马事件曝光后，他被指是周永康〝余毒〞。梳理孟宏伟的简历可以发现，他于2004年4月任公安部副部长时，当时的公安部部长是周永康。早在2003年6月，孟宏伟还曾作为部长助理陪同周永康去往浙江考察。
 <br/>
 <br/>
 2004年7月，公安部举行副总警监警衔授衔仪式，周永康向4个人颁发了警衔命令证书，孟宏伟便是其中之一。外界认为，中共政法系统镇压法轮功、维权人士甚至活摘器官等罪行，孟宏伟应该都有份参与。在继蒋洁敏、王永春、李春城、李东生、张越等人落马之后，孟宏伟成为周永康又一落马下属。
 <br/>
 <br/>
 另外，也有专家分析，孟宏伟出事前担任国际刑警主席职务，北京当局不惜损害自身国际形象抓捕孟宏伟，说明中南海一定是遇到十万火急的事情，不得不采取这种手段。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：戴明)
