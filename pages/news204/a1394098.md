### 乙肝疫苗也出问题？深圳幼儿接种后死亡官方急否认
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】近日，中国深圳一名婴儿接种
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乙肝疫苗.html" target="_blank">
  乙肝疫苗
 </a>
 3天后死亡，消息在网络上曝光后引起舆论的极大关注。周四，当地官方发布通报辩称，婴儿接种的批次疫苗不存在质量问题，但仍招来大量质疑。目前婴儿的死亡原因需等待尸体解剖后的鉴定结论。
 <br/>
 <br/>
 综合陆媒和网帖的消息，今年9月27日，龙华区大浪街道一名2个月大的婴儿，在深圳市龙华区人民医院大浪同胜社区健康服务中心接种了大连汉信的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乙肝疫苗.html" target="_blank">
  乙肝疫苗
 </a>
 。在留院观察的半小时里，孩子曾出现吐奶现象。
 <br/>
 <br/>
 第二天（28日）上午，婴儿的父母发现孩子出现了吐奶、手脚冰凉、心跳快等情况，病状危急，于是送往市儿童医院ICU病房抢救。其间，孩子出现了肝肾功能衰竭、心率下降等症状。经过一天多的抢救，孩子最终于9月30日凌晨1:21分死亡。
 <br/>
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-5/p9065351a679300144.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-5/p9065351a679300144-ss.jpg"/>
   <br/>
   <font size="-1">
    中国深圳一名两个月大的婴儿接种乙肝疫苗3天后死亡，当地官方发布通报称涉事的批次疫苗不存在质量问题。图为婴儿生前和死亡时的对比照片。（网络图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 <br/>
 孩子的家长把上述情况在网络上公开后，引发社会舆论的极大关注。10月4日，龙华区卫生和计划生育局就此事件发布了一份通报。
 <br/>
 <br/>
 当地官方的这份通报确认了这名婴儿接种的乙肝疫苗是大连汉信生物制药有限公司生产的产品，批号为201709096，批签发号为LRA20174335，但同时又强调有关监督机构检查后判定该批次疫苗〝不存在质量问题〞，并声称此次婴儿死亡事件仅属个案。
 <br/>
 <br/>
 但民众对官方这份通报仍然抱有强烈质疑。有家长到涉事的健康服务中心门口拉起大幅标语讨公道，也有许多网友留言提出种种担忧和讨论。
 <br/>
 <br/>
 目前，由当地政府组织的调查组对外宣称，由多个医院专家对患儿病情进行讨论后得出的初步意见是：患者存在休克、先天性心脏病、多器官功能障碍等症状，至于患儿病情与疫苗的相关性和因果性，目前还无法确认，需进一步调查。
 <br/>
 <br/>
 据陆媒报导，南方医科大学司法鉴定中心的法医，10月2日已对接种乙肝疫苗3天后死亡的婴儿进行了尸体解剖，尚未得出最终的死因鉴定结论。
 <br/>
 <br/>
 近十余年间，中国大陆频频爆出婴幼儿接种
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_问题疫苗.html" target="_blank">
  问题疫苗
 </a>
 后死亡或致病、致残的案例，引发社会舆论的强烈反响。就在今年10月1日下午，来自中国宁夏的妇女王金凤，因自己的儿子接种疫苗后致病的问题长期不得解决，在北京天安门广场欲头撞国旗杆寻死，被中共警方带走。
 <br/>
 <br/>
 据《自由亚洲电台》报导，王金凤的儿子于2010年在宁夏银川市出生。因为地方政府强制接种，王金凤的儿子于该年底在银川市西夏区医院接种了乙肝疫苗和防止肺结核的卡介苗。接种后，王的儿子出现了经常抽搐等不良反应。其后王金凤带儿子到北京诊治，经北京儿童医院诊断而得知自己的儿子存在认知障碍，而这些症状可能与接种疫苗有关。
 <br/>
 <br/>
 此后王金凤长年往返于银川市医院和政府相关部门讨公道，但各部门之间互相推诿，问题一直未得解决。无路可走的王金凤便决定到北京上访，却多次被地方政府派人强行带回银川，期间还遭受截访人员殴打。
 <br/>
 <br/>
 据了解，2013年11月，中国南方多地曾相继发生多起婴儿注射乙肝疫苗后致伤致死事件。2016年3月，山东非法疫苗案爆发，有价值5.7亿元未经严格冷链储存的疫苗流入24个省市。在最近的14年，中国大陆至少爆发了9起造成严重后果的问题疫苗案件。
 <br/>
 <br/>
 （记者李明综合报导/责任编辑：秦锋）
