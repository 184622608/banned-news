### 中国亿万富豪人数骤减 祸起两大因素
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡润.html" target="_blank">
  胡润
 </a>
 研究院日前公布2018年中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富豪榜.html" target="_blank">
  富豪榜
 </a>
 ，当中提及身家达20亿元（人民币．下同）以上的中国亿万富翁，减少了237人，这是6年来首度出现减少。究其原因，有分析认为，除了今年初中国股市大跌以外，中美贸易战也是其中重要原因之一。
 <br/>
 <br/>
 2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡润.html" target="_blank">
  胡润
 </a>
 百富榜10月10日发布，马云家族以2700亿元财富再次问鼎中国首富。许家印今年卫冕首富失败位列第二。马化腾2400亿元排第三。
 <br/>
 <br/>
 除了马云、马化腾，前5名中剩下3位均来自地产行业，均经历了财富缩水。恒大许家印、碧桂园董事局主席杨国强之女杨惠妍、万达王健林的财富分别为2500亿元、1500亿元和1400亿元，名列第2、4、5名，财富分别缩水400亿、100亿和150亿元。
 <br/>
 <br/>
 据〝公司深读〞统计，364位从事地产相关行业的富豪中，113人的财富较去年同期有所减少，总额高达3127亿元。
 <br/>
 <br/>
 除此之外，根据胡润百富榜数据，身家达20亿元以上的中国亿万富翁，6年来首度出现减少。
 <br/>
 <br/>
 2017年中国有2130人拥有20亿元以上资产。但在今年初中国股市大跌后，这些亿万富翁的财富大量〝缩水〞，今年中国亿万富翁数目减少了237人，只有1893人，减幅约13%。
 <br/>
 <br/>
 有法国传媒分析说，中国亿万富翁减少受多种因素影响，除了今年初中国股市大跌以外，中美贸易战中，美国对中国商品征收惩罚性关税，导致来自工业领域的亿万富翁比例跌至26.1%，较去年减少约1.8%。
 <br/>
 <br/>
 报导还提到，中共的反腐败斗争，使得这些亿万富翁即使有庞大的财富，也无法保证他们能逃离监狱，目前，9名最富有的中国人正在接受调查。
 <br/>
 <br/>
 近一年多来，中国不少
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融大鳄.html" target="_blank">
  金融大鳄
 </a>
 ，因充当中共权贵的〝白手套〞，并卷入中共帮派权力内斗，甚至涉及金融政变等，也纷纷锒铛入狱。如〝明天系〞掌门人肖建华，安邦集团总裁、邓小平孙女婿吴小晖等人，都先后被查。
 <br/>
 <br/>
 港媒评论称，中国富豪真正的危机，也就是他们人人自危的真正原因，恰恰是他们暴富的原因，在中共权贵资本主义的制度环境下，与政治权力的深度关联与勾兑，靠卷入政治来发财的富豪们，因此成为了政治变动的牺牲品。
 <br/>
 <br/>
 评论认为，即使与政治权力没有深度关联与勾兑，但中国的私营企业家一旦暴富，都难免成为中共公有制砧板上待宰的肥肉。即使见好就收的马云能够明哲保身，侥幸躲过一劫，也难以改变中国〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富豪榜.html" target="_blank">
  富豪榜
 </a>
 〞沦为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杀猪榜.html" target="_blank">
  杀猪榜
 </a>
 〞的噩运和魔咒！
 <br/>
