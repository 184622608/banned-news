### 【禁闻】房地产泡沫破裂? 深圳现〝断供潮〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】中国大陆一线城市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳楼市.html" target="_blank">
  深圳楼市
 </a>
 ，近来出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_业主.html" target="_blank">
  业主
 </a>
 因无力偿还银行贷款的断供现象，导致房屋被银行拍卖。
 <br/>
 <br/>
 今年下半年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳楼市.html" target="_blank">
  深圳楼市
 </a>
 出现实质性的下跌，有房产中介表示，有些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_业主.html" target="_blank">
  业主
 </a>
 因为生意经营不下去，还有的业主因为加槓杆炒房，无力还贷，房产被法院拍卖，而出现了〝断供〞的情况。
 <br/>
 <br/>
 所谓〝断供〞，是指业主不愿继续承担每个月的房屋贷款按揭，而选择违约，将房子还给银行。
 <br/>
 <br/>
 对比11月28号，深圳万科星城楼盘，网路上购房选房，半小时卖出1,900套，同一区域的华强城，当天也售出1,000套左右。一时之间，很多人以为深圳楼市回温了。
 <br/>
 <br/>
 旅美投资分析师王天娇 ：〝这其实是体现了房地产开发商在年末有一个回笼资金所做的手段，但它并不代表整个楼市有什么回暖的现象。如果看二手楼盘的成交量和成交价格，会发现其实全年都有下降的趋势，二手市场的价格，几乎是比较反映真实房市的情况。〞
 <br/>
 <br/>
 房地产短暂的〝热闹〞交易背后，究竟隐藏什么真相？旅美投资分析师王天娇分析指出，民众不要因为个别现象而忽略整体，房地产市场和一个国家的经济能不能健康发展是息息相关。
 <br/>
 <br/>
 王天娇：〝因为目前有一些增加关税的措施在进行，如果按照现在美国的立场，明年年初还会继续增加关税，肯定会对外贸行业有很大的冲击，那么现在已经看到一些进出口企业在面临很多的挑战，甚至有一些正面临倒闭或破产的风险。〞
 <br/>
 <br/>
 王天娇表示，深圳非常依赖进出口贸易和制造业，所以当宏观经济有了变化，肯定会有连锁反应发生在房地产，会引起波动。
 <br/>
 <br/>
 王天娇：〝这些实业才是整个经济的根本，肯定相应的也会波及到房地产市场，到时候（经济）下行的压力会更加大一些，可能宏观的调控措施未必能够达到预期的效果。〞
 <br/>
 <br/>
 大陆房产频道网站引述消息说，深圳市中级人民法院的数据显示，近半年来，司法拍卖的房源增加了50%左右，成交率达到91.99%。深圳法拍房公司〝广安居业〞的统计数据显示，去年深圳法拍房共2,100套，今年已经达到2,800套左右，增幅为33%。特别是近半年以来，深圳法拍房增加了五成。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田认为，中国的房地产市场完全违背了经济的基本面，是公认的巨大泡沫。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝这个泡沫的引爆点，可能跟外贸有关、跟外汇储备有关，跟人民币汇率有关，中国其它的产能过剩造成的失业有关。这个情况还会大量出现，因为人们看到房价下跌的时候，更会收紧钱包不敢购买，这样实际上会加速
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房地产泡沫.html" target="_blank">
  房地产泡沫
 </a>
 的破灭。〞
 <br/>
 <br/>
 曾经在深圳经商的台湾企业家高为邦认为，如果连深圳市这种一级城市都出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_断供潮.html" target="_blank">
  断供潮
 </a>
 的话，那问题是相当严重了。
 <br/>
 <br/>
 台湾企业家高为邦：〝因为中国的房价跟人民的收入，差距是非常大。中国一旦发生（楼市泡沫破灭）的话，就会恐慌性的卖压，可能这个时间也许明年就会发生，如果是明年就会发生的话，那人民币也会大跌，那中共这个麻烦就大了，是不是还能够继续执政可能都是问题。〞
 <br/>
 <br/>
 2008年，深圳房价下跌，引发大规模的集体〝断供潮〞，对当时的深圳楼市和银行造成巨大冲击。
 <br/>
 <br/>
 在中国买房，30%是心理防线，这是指房屋成交的首付款多数占30%。如果一套100万的房子，首付款30万，贷款70万。那么，当总体经济持续下行，房地产也持续下跌，房价跌破70万以下的时候，民众还会有偿还贷款的意愿吗？
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/周天
