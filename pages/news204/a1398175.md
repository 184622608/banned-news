### 【禁闻】美中达成贸易协议？ 川习会结果难测
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 正式开打至今将近4个月，日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 电话通话后表示，双方将达成一个很好的协议，不过，有中国学者分析，如果美方提出〝城下之盟〞，中共很难接受。也有评论指出，没有公平、互惠，就没有贸易协议。
 <br/>
 <br/>
 11月1号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 通了电话，川普发推文说，〝我们谈了很多议题，重点在贸易方面。那些讨论与预计在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷G20峰会.html" target="_blank">
  阿根廷G20峰会
 </a>
 举行的峰会正在朝好的方向前进。〞
 <br/>
 <br/>
 2号，川普向媒体表示，他在与习近平的通话中感受到，中方非常希望达成协议，相信可以和中方达成一个很好的协议。
 <br/>
 <br/>
 据中共统计部门日前公布的数据，10月出炉的GDP增长率跌至6.5%，创10年来新低；制造业采购经理指数 (PMI) 也降至两年来最低值。10月31号，习近平主持召开了政治局会议，会议指出，〝当前经济运行稳中有变，经济下行压力有所加大，部分企业经营困难较多，长期积累的风险隐患有所暴露。〞
 <br/>
 <br/>
 这是最近几年来，北京首次承认经济形势出现了严峻情况。上海华东理工大学经济系宏观经济教研室主任沈凌向《德国之声》表示，明年走势完全取决于年底的中美谈判，如果中美达成妥协，明年走势会好一点，不然非常不乐观。
 <br/>
 <br/>
 那么，预计11月底登场的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞能否让中美贸易战有所转机呢？
 <br/>
 <br/>
 旅美时事评论员蓝述认为，中美两国这次达成妥协的可能性相当高。
 <br/>
 <br/>
 旅美时事评论员蓝述：〝川普总统他是希望能够达成一个协议，中共它也希望达成一个协议。中国的经济受到了巨大的损失，北京估计是不会再愿意这种情况继续拖延下去，北京会做出巨大的让步，川普总统有可能也会给北京一个比较体面的下台。〞
 <br/>
 <br/>
 蓝述表示，美国很可能通过一些不同的方式去解决贸易问题。
 <br/>
 <br/>
 蓝述：〝不是说作出让步，而是通过具体的条件上的一些转换，既让中共有体面的下台，同时也把川普总统需要达到的中美贸易平衡，减少贸易逆差这个目的，能够达到最后的目标。〞
 <br/>
 <br/>
 中国人民大学美国研究中心主任时殷弘3号接受《中央社》访问时提出了川习会三种可能的结果，最好的结果是双方在互征高关税上休战。次好结果是，贸易战继续，但激烈程度有所缓解。最坏的情况是美方提出〝城下之盟〞，如要求大量增加进口美国商品，大幅拓宽市场准入，并要求改变中国金融体制和经济政策，特别是改变由国家控制、国家发动的技术升级的努力，那中共一定无法接受，贸易战将加剧。
 <br/>
 <br/>
 时事评论员田园博士指出，时殷弘所谓的三种结果完全是站在中共的角度来看待这个问题。他表示，自己对这个会议的预期相当悲观。
 <br/>
 <br/>
 时事评论员田园博士：〝我不认为它能够达成任何实质性的协议。川普总统已经执政两年，他是非常诚挚地想去达成协议的，他是这样一个人，他是愿意采取实质的手段来解决实际的问题。但是中共有一些(东西)，它是无论如何不愿意放弃的。〞
 <br/>
 <br/>
 4号，白宫首席经济顾问库德洛在接受福克斯独家采访时说，〝川普总统告诉我，他将与中国达成协议，但这必须是一项强大且符合美国利益的协议，这是他的关键点。我们会解决这个问题。〞
 <br/>
 <br/>
 田园博士说，符合美国利益的协议要具备公平和互惠两个因素。
 <br/>
 <br/>
 田园博士：〝川普政府已经多次强调，第一它必须公平，第二必须得互惠，就是大家在对等的基础上，都从这个贸易协定里受益。如果做不到，川普总统是绝对不会和中共达成任何真正的实质上的贸易协定。〞
 <br/>
 <br/>
 田园博士表示，川普上台以来就明确指出，中共种种不公平、不互惠的贸易，对美国造成巨大的伤害。中共政府对川普打贸易战的决心远远的低估了。
 <br/>
 <br/>
 采访/常春 编辑/陈洁 后制/葛雷
