### APEC峰会美中交锋 彭斯当面呛声习近平
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 蒙上阴影。17日，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）和习近平在峰会上激烈交锋。习批评美国打
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的做法。彭斯则反驳习，在中共改变行为方式前，美国不会改变征收关税的政策。
 <br/>
 <br/>
 2018年的亚太经济合作组织（APEC）领导人峰会，目前正在巴布亚纽几内亚举行，中国国家主席习近平和美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence），都出席了此次峰会，而且都发表了演讲。
 <br/>
 <br/>
 习近平发表主旨演讲时说，如果走上对抗的道路，无论是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冷战.html" target="_blank">
  冷战
 </a>
 、热战还是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，都不会有真正的赢家。
 <br/>
 <br/>
 习近平还说，征收关税，切断经济联系是〝短视的做法，注定会失败〞。保护主义和单边主义会阻碍了全球经济增长，树立经济壁垒的政策注定会失败。
 <br/>
 <br/>
 针对〝一带一路〞的倡议，习近平说，共建〝一带一路〞是开放的合作平台，不针对谁、也不排除谁，不会关起门来搞小圈子。
 <br/>
 <center>
  <br/>

    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
     APEC峰会

    峰会而且发表了演讲。(FAZRY ISMAIL/AFP/Getty Images)
   </font>

 </center>
 <br/>
 随后，彭斯也在APEC峰会发表了演讲，他再次对中共进行了猛烈的抨击。
 <br/>
 <br/>
 美国彭博社17日报导说，彭斯抨击，中共的〝一带一路〞为合作国家设立债务陷阱。
 <br/>
 <br/>
 彭斯称，美国是基于平等、透明、共创繁荣为原则，并以创造当地就业机会为特征的发展援助项目，并再次警惕发展中国家，不要掉进某些国家设置的债务陷阱，不要接受会损害主权和独立的外债，讽刺〝一带一路〞是〝约束他国的不归路〞。
 <br/>
 <br/>
 他还说：〝我们不会让我们的搭档淹死在债务的海洋之中。我们不会恐吓，不腐败，也不会损害你们的独立。美国公开公正地与他国打交道，我们不会提供一个约束性的带，或一个单向的路。〞
 <br/>
 <br/>
 此外，彭斯直接点名批评中共的贸易政策，称中共设置的贸易壁垒史无前例，包括关税、盗用知识产权、强逼技术转让和不公平补贴，美国已经对中国的贸易政策采取决定性措施，对2,500亿美元的中国产品征收关税，并可能将征税规模扩大一倍。
 <br/>
 <br/>
 彭斯说，美国不会改变这一应对方式，除非中共改变其行为方式。
 <br/>
 <br/>
 彭斯在访问亚洲期间，于13日在新加坡接受《华盛顿邮报》专访时，也曾警告中共必须全面改变其行为作法，才能避免与美国发生新冷战。
 <br/>
 <br/>
 彭斯强调，11月底的二十国集团（G20）峰会期间，〝习川会〞上，除了讨论中美两国贸易纠纷问题外，中方也必须在保护知识产权、禁止强迫技术转让、取消限制市场准入、遵守国际规则、及保障国际水域航行的自由等问题上让步。
 <br/>
 <br/>
 他还警告，如果中方的提议未能令美方满意，美国已经准备好在经济、外交及政治等，多个层面上对中共施加更大压力。
 <br/>
 <br/>
 他说，美国经济强劲到可以抵御这种升级，而中共经济则不那么持久。
 <br/>
 <br/>
 在出访亚洲之前，彭斯于9日也曾刊文批评，有国家（中共）在国内打压人民，对外侵犯邻国主权，美国不会容许印太地区出现威权及侵略行为。
 <br/>
 <br/>
 目前中美这两个世界最大经济体在贸易、安全和人权等众多问题上存在冲突。
 <br/>
 <br/>
 中国知名金融学家贺江兵在自媒体上说，美方的要求都是中方曾经答应过的，签署过协议的，也是国际相关条约要求过的，对中国普通百姓有利的。他强调：〝中方需要作出实际的改革开放行动，而不是口头承诺了。〞
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田说，要中共解除对贸易、对经济的控制、放弃外汇操控⋯⋯就等于让中共退出中国的经济生活。相当于让中共退出政治舞台、历史舞台，这实际上就是要解体中共。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
