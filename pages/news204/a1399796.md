### 专家警告：吃了病猪肉 人就带病毒
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】17日，中共农业农村部新闻办公室发布通报称，上海市金山区爆发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情。目前，非洲猪瘟已蔓延至大陆19个省市。有专家警告，只要吃了病猪肉，人就会带有病毒，建议民众暂时不要吃猪肉。
 <br/>
 <br/>
 11月17日16时，上海市金山区一处生猪养殖户爆发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情。该养殖户存栏生猪314头，发病50头，死亡11头。
 <br/>
 <br/>
 与此同时，江西省上饶市鄱阳县一处养殖场、云南省昆明市呈贡区一间屠宰场、四川省成都市新津县一处养殖场〝排查〞出非洲猪瘟疫情。
 <br/>
 <br/>
 通报称，江西鄱阳县养殖场存栏生猪150头，发病10头，死亡10头。四川新津县养殖场存栏生猪110头，发病27头，死亡13头。云南呈贡区屠宰场共有待宰生猪348头。
 <br/>
 <br/>
 新增疫情导致大陆非洲猪瘟扩散地，增至19个省市自治区。目前，整个长江流域均爆发疫情，长江以南仅剩广东省、广西壮族自治区与海南省，尚未传出猪瘟疫情。
 <br/>
 <br/>
 非洲猪瘟病毒生命力顽强，可在肉类产品、动物饲料和馊水中存活数月。目前没有疫苗及治疗方法，感染病毒的猪几乎是必死无疑。
 <br/>
 <br/>
 联合国粮农组织首席兽医官卢布罗斯（Juan Lubroth）表示：中国的猪瘟疫情极有可能通过受感染的猪肉产品向邻国传播，且猪肉产品在亚洲国家广受欢迎。
 <br/>
 <br/>
 目前，韩国、日本、台湾都先后检测到中国的猪肉制品中携带猪瘟病毒，各国正加强防控措施，防止猪瘟疫情入侵。
 <br/>
 <br/>
 值得一提的是，台湾在金门入境处，从大陆游客丢弃的〝双汇香脆肠〞中发现猪瘟病毒，这家公司是大陆最大的肉品加工厂。8月初，沈阳爆发第一例非洲猪瘟，就是从双汇公司30头待宰的猪中检测出的。
 <br/>
 <br/>
 台湾农委会副主委黄金城10月31日在记者会上表示，这表示病毒可能已经普遍存在于双汇的肉品中。他认为，中国的所有省份可能都已经全部沦陷。
 <br/>
 <br/>
 猪瘟病毒是否对人体有影响？中国健康教育研究所前所长陈秉中对《自由亚洲》表示，只要吃过病猪肉，人就带有病毒。
 <br/>
 <br/>
 他说：〝他（人）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吃病猪肉.html" target="_blank">
  吃病猪肉
 </a>
 之后，他会消化然后排泄，排泄物可能有猪瘟病毒，只要他吃过病毒，身上就带有病毒。〞即使是不传染（人），但你能带着病毒到别的地方去，造成病毒扩散。建议民众最好暂时不要食用猪肉。
 <br/>
 <br/>
 近日，一位负责防控猪瘟疫情的防疫人员也向《自由亚洲》透露，目前疫情的扩散情况难以想像，当局封锁消息。她担忧猪瘟病毒很可能发生变异，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人体被感染.html" target="_blank">
  人体被感染
 </a>
 ，那后果会不堪设想。
 <br/>
 <br/>
