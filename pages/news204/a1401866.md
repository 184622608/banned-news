### 传朱鎔基批评当局 法媒：北京已被套上紧箍咒
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】在日前结束的川习会上，北京做出重大让步，换取了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_90天缓刑期.html" target="_blank">
  90天缓刑期
 </a>
 。有评论认为，北京此举相当于是被美国套上了〝紧箍咒〞。有消息人士透露，日前，前中共国务院总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 ，在一个公开场合批评北京当局没有尽早
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遵守世贸协议.html" target="_blank">
  遵守世贸协议
 </a>
 ，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 爆发，令北京陷入被国际孤立的困境。
 <br/>
 <br/>
 在阿根廷首都布宜诺斯艾利斯举行的〝川习会〞12月1日晚落幕，美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 进入一个〝休战期〞，但期限只有90天。
 <br/>
 <br/>
 在这期间，中方同意购买美方农产品和能源，这等于是中方把之前两轮对美国的贸易报复都取消了，而美方对中方的两轮贸易措施仍原封不动。
 <br/>
 <br/>
 不仅如此，北京还首次对美国承诺，同意就〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性改革.html" target="_blank">
  结构性改革
 </a>
 〞进行谈判，内容包括强迫技术转移、知识产权保护、非关税壁垒、网路入侵和网路盗窃、服务和农业等方面。
 <br/>
 <br/>
 如果谈判不成功，美国将把目前针对中国2000亿美元的关税从10%提升到25%。
 <br/>
 <br/>
 事实上，外界发现，这些〝结构性改革〞的承诺，都是中国加入世界贸易组织时就已经承诺过的，只是16年来迟迟没有兑现。
 <br/>
 <br/>
 据香港中文大学客座教授林和立引述北京消息人士透露，不久前，前中共国务院总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 在一个聚会场合，批评北京当局没有尽早履行入世承诺，以致引发贸易战，导致北京在世界上被孤立。
 <br/>
 <br/>
 北京消息人士称，朱鎔基认为，遵守世贸规定对北京的改革开放有好处。
 <br/>
 <br/>
 朱鎔基旧部龙永图，近日也曾公开批评北京处理贸易问题的举措失当。11月18日，在陆媒财新网举行的第九届财新峰会上，曾任中方入世（加入世贸组织）谈判首席代表的龙永图表示，贸易谈判要就事论事，谈关税就谈关税，如果北京一直将政治与贸易谈判掺和在一起，那中美永远都无法达成协议。
 <br/>
 <br/>
 他还直言，北京对美国大豆加征关税〝太草率〞。
 <br/>
 <br/>
 龙永图说，用美国大豆反击川普政府的关税措施是不明智的。因为农产品在贸易中是非常敏感的，打击农产品本应该是最后才考虑的办法。但是北京从贸易战一开始就选择了农产品。
 <br/>
 <br/>
 从11月初的美国中期选举结果显示，中方打击美国农产品，并没有让川普和共和党失去美国农业州的支持。但是北京方面，却因此造成大豆巨大缺口，使中国市场陷入困境。
 <br/>
 <br/>
 此外，北京打〝朝鲜牌〞和〝联欧抗美〞，也被指是激化贸易战的政治手段。当时有分析人士指，北京下出这些〝臭棋〞，是主动将贸易问题〝政治化〞，〝逼迫〞美国政府将中美对抗升级。
 <br/>
 <br/>
 另一方面，贸易战也加剧了中共内部的矛盾，就如何应对贸易战引发激烈争论。川习会结束后，对于北京承诺的〝结构性调整〞，党内的反对声此起彼伏。因为〝结构性调整〞牵涉到各方面的利益集团，习近平要动利益集团很难不招致反弹。
 <br/>
 <br/>
 此外，对于美方并未升级炮火、增加压力，国际问题专家唐浩评论说，川普暂不升级炮火，除了给习近平这位〝老朋友〞更大的善意与空间来应对贸易战外，或许也是给习近平更多时间来冷静思考，要不要解体中共这部毁弃传统文化、背离普世价值的极权机器。
 <br/>
 <br/>
 唐浩此前也表示，贸易战实际上不仅是美国与中国的冲突，更是自由社会与中共体制的冲突，是普世价值与变异价值的冲突。只有把变异价值彻底消解了，才能彻底化解贸易战冲突；而要彻底消解变异价值，就得彻底解体中国共产党。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：范铭）
