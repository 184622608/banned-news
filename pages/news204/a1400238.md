### 遭官方闭门 疫苗受害家长借媒体表诉求
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】中共药监部门目前正在起草《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗管理法.html" target="_blank">
  疫苗管理法
 </a>
 》并征询公众意见，但疫苗受害儿童家长前去提出建议却被拒之门外。11月21号，一位河南疫苗受害儿童家长将自己的很多建议告诉了我们，包括设立〝疫苗风险
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_补偿基金.html" target="_blank">
  补偿基金
 </a>
 〞，对受害家庭提供生活保障等。
 <br/>
 <br/>
 中共国家药监局公布的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗管理法.html" target="_blank">
  疫苗管理法
 </a>
 （征求意见稿）》目前正向公众征求意见和建议，反馈截止日到11月25号。但20号去药监局递交修改意见的疫苗受害者家长，却吃了闭门羹。
 <br/>
 <br/>
 21号，河南疫苗受害儿童家长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_何方美.html" target="_blank">
  何方美
 </a>
 将自己的建议告诉了《新唐人》。何方美的女儿接种疫苗后，患上灰质脊椎炎，却被说成是偶合症。
 <br/>
 <br/>
 河南疫苗受害儿童家长何方美：〝我就想让它取消偶合症，取消疾控中心的调查诊断和医学会鉴定，然后成立一个体检的机构，从小孩生下来体检，每个月接种之前也要体检，体检如果没问题，可以打针，那就打，打了要是有问题，那就是疫苗导致的。〞
 <br/>
 <br/>
 何方美表示，无论接种疫苗的孩子的病是什么原因导致的，都希望政府给予前期救助。
 <br/>
 <br/>
 何方美：〝不能等到孩子在医院没钱花，然后病情恶化了呀，对不对？希望在他调查疫苗的情况下，不影响给孩子前期救助，等孩子救了以后再打官司。〞
 <br/>
 <br/>
 何方美还希望政府成立一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_补偿基金.html" target="_blank">
  补偿基金
 </a>
 。
 <br/>
 <br/>
 何方美：〝我希望成立一个疫苗风险补偿基金，由疫苗厂家从每一支疫苗里边拿出一部份，投到基金会，另一部份再由政府来出。〞
 <br/>
 <br/>
 针对政府给的反馈时间，何方美认为太短，她还透露，有部份家长想当面递交意见，结果被维稳。
 <br/>
 <br/>
 何方美：〝按照规则应该是不少于30天，而它只给了我们13天时间，提了，发邮件都没有用，没有回应，昨天也有部份家长去了国药监局，被拦下来了，他们不接受材料意见稿，被维稳了，有部份家长被带回老家。〞
 <br/>
 <br/>
 何方美还表示，问题疫苗厂商吉林省长春市长生科技公司被当局罚款91亿元。她提议，政府应该用这91亿元，设立〝疫苗风险补偿基金〞，对受害者提供前期治疗和后期生活保障。
 <br/>
 <br/>
 新唐人记者陈汉、陈洁、施宜君采访报导
</div>
