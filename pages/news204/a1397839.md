### 假新闻？库德洛：川普没要求起草中美贸易协议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】中美首脑周四（11月1日）通话后，有美媒报导
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）下令起草协议草案，准备在月底的G20峰会期间与北京和解。但该消息引发多方质疑。白宫经济顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 随后亲自出面辟谣。
 <br/>
 <br/>
 美东时间周五（2日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 接受CNBC访问时表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统并没有要求白宫准备和北京达成协议。〝没有针对中方的大动作，我们（只是）就贸易向中方提出了要求。〞
 <br/>
 <br/>
 库德洛说，现在白宫只是在为月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 做一些常规的准备工作，〝我们还没到达成交易的临界点〞。
 <br/>
 <br/>
 虽然川普对周四与北京通话的评价是〝进展良好〞，但库德洛强调，美中两国在正式贸易谈判中仍然分歧很大。
 <br/>
 <br/>
 库德洛说，中美首脑在G20期间将会面并讨论贸易问题，中方只需要展示对美方贸易要求的具体回应。〝如果总统选择（对中方继续加征）关税，我们还有更多工作要做。〞
 <br/>
 <br/>
 周五早些时候，彭博新闻社援引四位知情人士的消息称，川普希望在G20峰会上与北京达成协议，而且已经指示美国核心官员起草美方可能提出的协议条款。
 <br/>
 <br/>
 随后，美国消费者新闻与商业频道（CNBC）记者埃蒙．杰弗斯提出质疑。杰弗斯引述一名白宫高官的消息称，跟北京的谈判〝还有很长的路要走〞。
 <br/>
 <br/>
 《华盛顿时报》也刊出前一天对川普总统的专访内容称，川普明确表示，预计中美不会在本月底的G20峰会期间达成贸易协议。
 <br/>
 <br/>
 遏制中共已是美国朝野的共识，向北京妥协让步的消息或能提振美国股市，但不利于川普。
 <br/>
 <br/>
 目前，美国媒体的报导呈现出越来越混乱的局面。8月时，美国《华尔街日报》也曾报导称，中美正在准备在11月的川习会上结束贸易战。但川普次日即出面驳斥该说法。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
