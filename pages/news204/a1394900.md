### 美围剿中共效应初显 孟宏伟案凸显中南海矛盾激化
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，遭中共秘密抓捕引发全球关注。有港媒10日说，在遭到以美国为首的国际性政治、经济、军事围剿之下，北京不顾及顔面抓人，这表明中共高层的矛盾当前激化到不可想像的程度。
 <br/>
 <br/>
 香港苹果日报10日报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 妻子4日傍晚向法国警方报案其丈夫失踪，并于7日召开记者会，向世界公布其丈夫回到中国境内与家人失联。立即引发全世界的关注，使得中共不顾及国际法和颜面秘密关押孟宏伟的计划暴露。
 <br/>
 <br/>
 报导说，这表明中南海高层当前的矛盾激化到何等程度，在遭到美国为首的世界性政治、经济、军事围剿之下，北京才不顾一切进行党内大清洗。像孟宏伟这样人，当了一辈子共产党刀把子的酷吏，掌握中共重大机密的高官，当然要格外小心。
 <br/>
 <br/>
 孟宏伟的妻子格蕾丝・孟（Grace Meng）7日在法国里昂的一家酒店举行记者会时，展示其丈夫9月29日使用WhatsApp，发来最后一次信息中有一张刀的图案，说明他们已经防范到何等程度，如果使用国产手机和微信软件根本发不出来。
 <br/>
 <br/>
 中纪委国家监委官网同日通报，孟宏伟涉嫌违法被查。几小时后，国际刑警组织发公告，已收到孟宏伟的辞呈。公安部长赵克志次日凌晨紧急召开会议，宣布成立工作专班查处孟的同党。说明孟之后，公安部还会有更多高官落马。
 <br/>
 <br/>
 在官方公布孟宏伟下台前，法国警方透露，孟宏伟于9月29日只身从国际刑警组织的里昂总部，返回中国后失踪，孟妻于法国时间10月4日傍晚，向警方报案。
 <br/>
 <br/>
 5日港媒爆料，孟宏伟抵达中国后，立即被中纪委带走，其因涉及经济问题及其他贪腐问题被抓。但有分析认为，中共因孟贪腐而大动干戈，对其扣押惊动全世界，逻辑上不成立。
 <br/>
 <br/>
 孟宏伟是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 的亲信，曾在中共安全机关担任过多项职务，在周落马后，孟却能全身而退，甚至于2016年当选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 。但在中共十九大，赵克志担任中共公安部部长后，多次要求继续清洗周永康的〝余毒〞。
 <br/>
 <br/>
 中共官方信息显示，孟宏伟是作为周的〝余毒〞被拿下。
 <br/>
 <br/>
 日本《产经新闻》则认为，孟被逮捕主要是因其涉及中共重要机密。
 <br/>
 <br/>
 一个是，孟宏伟可能与中国海航集团董事长王健7月在法国坠亡事件有关；二是，孟宏伟可能曾计划逃往美国，打算向美方提供中共领导人亲属在海外的资产清单。
 <br/>
 <br/>
 报导说，孟宏伟案目前依然真相不明，不过，对于中共高层来说，此案是一件不容忽视的大案。
 <br/>
 <br/>
 港媒苹果也分析，孟宏伟可能涉嫌中共政治忌讳，涉嫌北戴河政变案。因为案件重大而紧急，中共才不顾国际影响而紧急对孟宏伟问话。
 <br/>
 <br/>
 分析认为，中共高官孟宏伟受查后，孟妻高调向国际社会求救，并召开新闻发布会。这等于向中共摊牌，能做出这种决定，除非到了生死攸关的时刻。
 <br/>
 <br/>
 这也说明孟妻尚未向外界披露孟宏伟究竟干了什么，惹得中共不顾国际脸面而抓人，很可能孟妻手握中共重大国家机密，如果孟回国是有安全预案的，他自然会留有后手，以便与中共博弈或玉石俱焚。
 <br/>
 <br/>
 分析认为，一旦孟妻握有的机密文件曝光，对中共的打击将是沉重的，震撼力将远超当年王立军事件。目前，法国警方已经对孟的妻儿进行了安全保护。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：赵云）
