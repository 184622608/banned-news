### 刘强东性侵案  德媒：美警方一做法不合常理
------------------------

 【新唐人北京时间2018年11月24日讯】

 </a>
 集团CEO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 在美国涉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵.html" target="_blank">
  性侵
 </a>
 事件，23日又有更多细节被披曝光，有德媒称，美国警方一做法存在非议，如今事发已超过两个月，然而，针对刘强东是否会面临刑事指控仍然没有任何消息。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东.html" target="_blank">
  京东
 </a>
 集团CEO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 被捕事件发生后，公司股价已下跌逾三分之一，随着警方调查，被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵.html" target="_blank">
  性侵
 </a>
 女大学生所提供的案件细节也一一曝光。德媒认为，美国警方的一做法不合常理。
 <br/>
 <br/>
 据德国之声11月23日称，如今事发已经超过两个月，针对刘强东是否会面临刑事指控仍然没有任何消息。
 <br/>
 <br/>
 至于美检方为何一直不起诉？该州亨内平县(Hennepin County)检察官办公室对美媒表示：〝每个案件都不同，审查案件需要花费的时间......我们还有其他需要花费更长的时间的案件，有些花的时间就比较少。〞
 <br/>
 <br/>
 检察官还称，刘强东是富豪与否与他们对案件的审议无关。
 <br/>
 <br/>
 律师马什．哈尔伯格（Marsh Halberg）分析称，检察官可能在等待化验结果；也有可能是检察官在与辩护律师对话；而在个别情况下，检察官也可能会要求办理该案的警察局重新采访证人或收集更多的证据。
 <br/>
 <br/>
 哈尔伯格表示：如果刘强东不被起诉，人们就会怀疑他是否受到了特殊对待，如果检方要提出指控，刘强东也必定会动用大量资源进行辩护，因此检方必须要有充足的理由才能提起诉讼。
 <br/>
 <br/>
 同一天，英媒《路透社》披露了刘强东性侵案的更多细节。
 <br/>
 <br/>
 报导说，事发当晚，刘强东在一间日本餐厅举行晚宴，21岁的中国女大学生是参与者之一，晚宴接近尾声时，女生说喝醉了，要回家，她和刘强东还有一位姓张的私人女助理(Alice Zhang)一起上了一辆黑色轿车。
 <br/>
 <br/>
 根据警方问讯录音，女生说这名张姓助理起初试图坐在她和刘强东之间，但刘强东对助理说:〝别碍事〞，并开始在后座上抚摸她。
 <br/>
 <br/>
 女生的律师弗罗林和一位知情人士说，后来坐在前面的张姓助理调高了收音机音量，并折起了汽车的后视镜，然后就发生强脱女生衣服的举动。
 <br/>
 <br/>
 女生称，当时她试图阻止刘强东脱她的衣服。〝我乞求他，‘别那样，你有老婆和小孩，’〞她还对警方说:〝他不听我的。〞
 <br/>
 <br/>
 女生还对警方说，抵达她的住处后，刘强东告诉司机和助理，很快会回去。但是他进入公寓后，试图将女生拉进浴室，未果后自己洗澡，然后裸身躺在床上。
 <br/>
 <br/>
 女大生表示自己已竭尽全力和刘强东对抗，但最终还是被性侵得逞。女生还告诉警方，她保留了床单上的精液作为证据。
 <br/>
 <br/>
 女生与警方的谈话录音显示，事发后，刘强东的私人助理，曾请深圳卓越置业集团有限公司董事局长李华出面调解。李华跟刘强东一样是中国高管工商管理博士（DBA）课程学生。该课程学生包括腾讯CEO马化腾。
 <br/>
 <br/>
 根据美国警方9月4日公布的报告，刘强东性侵案发生在8月31日凌晨1时，当天刘强东被捕入狱，16个小时后获释，0保释金。
 <br/>
 <br/>
 警方对刘强东一案的描述是〝一级性犯罪：强奸既遂〞（CSCR-1,CSC-RAPE-COMPLETE）。据警察局的拘留记录，刘强东的状态为〝释放有待正式控告〞。 根据明尼苏达州的法律，一级强暴罪只要最后被定罪，刑期至少12年起。
 <br/>
 <br/>
 明尼苏达亨内平县警方9月21日宣布，警方已完成对刘强东案件的初步调查，接下来将由检方接手，审查案件全部证据，并最终决定是否起诉。
 <br/>
 <br/>
 有美媒分析说，如今唯一能够帮刘强东脱罪的，就是女方承认当晚的行为是自愿的。否则，刘强东被判强奸罪成立的概率很大。
 <br/>
 <br/>
 有华裔刑案律师认为，刘强东遭起诉的可能性超过90%，案件一旦开始会涉及重新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逮捕.html" target="_blank">
  逮捕
 </a>
 等问题。
 <br/>
 <br/>
