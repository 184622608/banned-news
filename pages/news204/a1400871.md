### 政治局会议依旧未提 中共四中全会或拖到明年
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】11月中中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治局会议.html" target="_blank">
  政治局会议
 </a>
 依然没有提及本届四中全会召开事宜。台媒分析，年底中共还要召开经济会议，年底前再开聚焦经济问题的四中全会已无必要。
 <br/>
 <br/>
 北京最高领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 11月27号出访的前一天，主持了中共中央
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治局会议.html" target="_blank">
  政治局会议
 </a>
 ，官方的会议通稿中没有提及何时召开四中全会。
 <br/>
 <br/>
 中央社报导分析称，依中共惯例，中共举行中央委员全体会议前，通常要在前一个月的政治局会议上公布具体会期。而政治局例行会议每月召开一次，若11月的会议未决定四中全会何时召开，最快要等到12月下旬的政治局会议才会决定何时召开。
 <br/>
 <br/>
 报导据此分析，四中全会今年底很有可能无法召开。
 <br/>
 <br/>
 另外，习近平27日起将前往西班牙、阿根廷、巴拿马、葡萄牙，整个行程将于12月5日结束。之后，12月18日习近平还要在北京主持召开〝改革开放40周年纪念大会〞。
 <br/>
 <br/>
 而且，一年一度的中共中央经济工作会议，依例将于年底前举行。预计今年的经济会议可能将讨论应对美中贸易战、中国经济成长趋缓以及潜藏的金融危机，并对2019年的经济提出规划。
 <br/>
 <br/>
 综合上述各个情况，报导分析，中共年底前已没有时间召开四中全会。而且，四中全会可能涉及的经济问题，可能在经济工作会议上就会讨论，因此四中全会议题或不再聚焦经济，不一定非在今年底前召开。
 <br/>
 <br/>
 按照此前惯例，今年下半年会召开中共十九届三中全会，讨论经济议题。不过，因为今年上半年加开了一次讨论修宪的二中全会，原本讨论人事议题的二中全会改成了三中全会。因此，外界一度猜测下半年将举行四中全会，讨论贸易战对策。
 <br/>
 <br/>
 不过，中共官方从未透出有关四中全会的信息。外界分析，四中全会被拖延，或因为北京政局不稳，或因为当局根本没有因应贸易战的有效对策。
 <br/>
 <br/>
 自由亚洲电台11月2日引述熟悉中共高层动向的人士透露，北京正在等待11月的美国中期选举结果，然后再制定应对方案。而方案制定需要数个月时间，因此今年举行四中全会的可能性越来越小。
 <br/>
 <br/>
 也有分析指，中共一年内连开三次中央委员全体会议的情况罕见，因此四中全会或明年再开。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
