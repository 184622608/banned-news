### 非洲猪瘟攻陷中国16省市 政府难辞其咎
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】
 <b>
  新闻周刊
 </b>
 （653）中国大陆的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 ，不断蔓延，中共官方通报，有15个省市沦陷。外界日前评估，中国境内的疫情早已全面失控，而中共当局对当前的形势难辞其咎。
 <br/>
 <br/>
 8月初，中国大陆报道首例
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 案例。开始在一些城市设立检疫点，官方通报称都〝已得到有效处置〞，还不断强调非洲猪瘟不感染人或其它动物。
 <br/>
 <br/>
 11月8日，中共农业部称，福建莆田市、湖南省涟源市、吉林省龙井市和江西省万年县发生非洲猪瘟案例，共计已有53例。
 <br/>
 <br/>
 也就是说，3个月来，非洲猪瘟从北向南蔓延，包括吉林、辽宁、黑龙江、河南、内蒙古、山东、天津、重庆、江苏、安徽、浙江及福建等16个省、直辖市和自治区都出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 ，范围覆盖大半个中国。
 <br/>
 <br/>
 官方日前通报终于出现了〝形势严峻〞字眼，似乎已默认疫情失控。
 <br/>
 <br/>
 日前，在大陆网路上流传的一段视频显示，有民众开车行驶在路上，沿路都是病死猪的尸体，每隔几米就有数只病死猪，有些死猪嘴边还流出大摊血渍，十分恐怖。视频中未见有卫生防疫人员在现场处理，而且当地居民也在死猪旁边随意行走。影片没有标示拍摄的时间和地点。
 <br/>
 <br/>
 近日，一位参与防控工作的非洲猪瘟专家徐女士对自由亚洲说，疫情在大陆蔓延的情况比想像的要严重。当局对最能掌握疫情变化的实验室和疫控人员，已进行严管，不允许将疫情情况向外界透露，否则就是泄密。
 <br/>
 <br/>
 徐女士认为，无论是管理能力上，还是生物技术方面，官方都已无力阻止非洲猪瘟疫情的扩散。
 <br/>
 <br/>
 尽管目前没有通报有人感染猪瘟的案例，但徐女士表示，即便是业内人士，也不清楚疫情到底会发展到何种程度。她担忧，猪瘟病毒很可能和禽流感病毒一样，有变异的可能，传染给人类。如果真的发生这种情况，后果将不可设想。
 <br/>
 <br/>
 官方至今没有正式公布猪瘟病毒的源头及传染途径。此前虽然发文禁止各地使用餐厨剩余物喂猪，但对疫情控制几乎没有影响。
 <br/>
 <br/>
 有专家指，非洲猪瘟病毒可通过蚊虫传播，甚至在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪肉.html" target="_blank">
  猪肉
 </a>
 制品中长期保持活性。另外，被病毒污染的饲料、水源、器具，甚至是农场工作人员和服装，以及污染农场附近的空气都是潜在的传染源。
 <br/>
 <br/>
 有评论认为，非洲猪瘟在大陆失控，特别是在部分地区一爆再爆，与当局不够重视有关，地方政府防控不力更是难辞其咎。
 <br/>
 <br/>
 中国是世界上最大的生猪养殖国家，生猪存栏量超过五亿头，占世界一半以上。养猪业更关系到五千万人口就业问题。非洲猪瘟不仅有可能对中国的生猪养殖业造成灾难性的打击，也将影响社会民生，包括供给减少，导致猪肉价格上涨等。
 <br/>
 <br/>
 疫情爆发至今，中国猪肉价格，已经涨价两成五。香港这两个月猪肉价钱上升10%左右。
 <br/>
 <br/>
 台湾行政院为了防堵病毒进入台湾，全面禁止肉品入境，违规者罚金上限最高30万新台币，相当于一万美元。
 <br/>
 <br/>
 非洲猪瘟是由非洲猪瘟病毒引起的猪急性、热性、高度接触性
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传染病.html" target="_blank">
  传染病
 </a>
 ，病猪死亡率高达100%。
 <br/>
 <br/>
 撰稿：梁东
