### 一文看懂 中国被列入汇率操纵国有何后果
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】彭博社消息，美国财政部在白宫要求下，正考虑在下周的报告中，把中国列为汇率（货币）操纵国。那么，什么是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率操纵国.html" target="_blank">
  汇率操纵国
 </a>
 ，中国被列入后有何影响呢？
 <br/>
 <br/>
 <h4>
  什么是
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率操纵国.html" target="_blank">
   汇率操纵国
  </a>
  ？
 </h4>
 <br/>
 根据中外合资投资银行——中国国际金融股份有限公司（中金公司）去年2月发布的研究报告，从2016年开始，美国财政部开始采用量化指标，来判断贸易伙伴是否操纵汇率：
 <br/>
 <br/>
 1）巨额对美贸易顺差（超过200亿美元）；
 <br/>
 2）大规模经常帐户盈余（超过GDP的3%）；
 <br/>
 3）持续、单向的外汇干预（净买入外汇超过GDP的2%）。
 <br/>
 <br/>
 如果一个国家（地区）符合上述3个指标，就可被认定为滙率操纵。如果满足两项标准，或者在美国的总体贸易逆差中占比过大，则被列入滙率操纵观察名单。
 <br/>
 <br/>
 中金公司报告指，美国目前评估〝汇率操纵国〞的法律依据，是《2015年贸易便捷与贸易促进法》。但是具体执行时，财政部对相关技术标准、界定阀值拥有裁量权，总统也可指示财政部进行调整。
 <br/>
 <br/>
 此外，美国财政部每年4月及10月向国会提交《国际经济和滙率政策报告》，审视其他主要经济体的滙率政策有否对美国不公。因此财政部每年有两次机会更新汇率操纵国名单或观察名单。
 <br/>
 <br/>
 <h4>
  中国是否符合标准？
 </h4>
 <br/>
 美中间存在巨额贸易逆差，中方显然符合有关对美贸易差额的指标。
 <br/>
 <br/>
 根据中共官方数据，中国2007年经常帐户盈余占比超过GDP的10%以上，2015年降至3.5%，2017年已降至1.38%。但是，中共官方数据经常为迎合需要而变动。
 <br/>
 <br/>
 在汇率干预方面，人民币兑美元汇率近期以来跌跌不休，外界普遍认为中共有意贬值人民币，以抵消贸易战导致的出口下降。美国已多次对此发出警告。
 <br/>
 <br/>
 美国总统和财政部有权调整判定汇率操控的标准，因此如果白宫下定决心，财政部有很大概率会将中国列入汇率操纵国。
 <br/>
 <br/>
 中国在1992至1994年间，曾被美国五次认定为滙率操纵国。今年4月的美国财政部报告中，中国、韩国、日本、德国、瑞士和印度被列在观察名单中。
 <br/>
 <br/>
 <h4>
  有何影响？
 </h4>
 <br/>
 根据法律，如果中国被列入汇率操纵国，美国财政部会与中方启动谈判，敦促北京改变做法。如果一年后中共继续操纵滙率，美国总统会采取以下一项或多项行动：
 <br/>
 <br/>
 一、禁止中国任何项目获取美国海外私人投资公司融资；
 <br/>
 二、将中国排除在在美国政府采购供应地之外；
 <br/>
 三、呼吁国际货币基金组织（IMF）加强对中方的监督；
 <br/>
 四、指示美国贸易代表评估是否与中方签订贸易协定，或启动、参与贸易协定谈判时须考虑中方操控滙率的行为。
 <br/>
 <br/>
 外界分析指，美国总统可对相关规定进行调整。如果将中方列为汇率操控，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）政府或很快推出制裁措施，除了限制美国公司对中国的投资外，还可能以此为依据，推出包括惩罚性关税在内的更多举措，将中美贸易战再次升级。
 <br/>
 <br/>
 此外，多名专家曾指出，美国一旦将中方列为汇率操纵，很可能意味着中美间将开启货币战，美国或利用在国际金融领域的优势，对人民币展开全面阻击。
 <br/>
 <br/>
 韩媒10月9日文章称，如果中美贸易战延伸到金融、资本市场的制裁甚至汇率战，很可能会出现中方资产被冻结和特定商品的世界贸易被禁止等状况。
 <br/>
 <br/>
 文章指，美国拥有可掌控贸易支付和兑税交易的两大金融系统——全球金融控制系统和金融支付结算系统，美国财政部可通过海外资产控制室实施金融制裁。如果美国将中方列入制裁范围，中国对外经济会陷入困境。
 <br/>
 <br/>
 自美国今年初警告要将中共列入汇率操控以来，海内外亲共媒体一直在宣传中国〝不符合汇率操控的标准〞，列入汇率操控对中方〝影响不大〞等等。
 <br/>
 <br/>
 日前，在川普准备将中国列入汇率操纵国的消息传出后，中共外交部第一时间回应，再次强调中方〝不会将贬值人民币作为贸易战工具〞。
 <br/>
 <br/>
 分析人士指，这些现象从侧面说明，中共对川普此项举措非常忌惮。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
