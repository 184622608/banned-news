### 习近平称支持民企 党媒批〝国进民退〞蹊跷被删
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】在〝民营经济退场论〞持续发酵之际，党媒刊登北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 给民营企业家的回信，再次强调〝毫不动摇〞地支持民企。而党媒另一篇批评〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 〞的旧文，却因为遭网民聚焦而被迅速删除。
 <br/>
 <br/>
 新华社10月21日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 给民营企业家回信时，肯定了民营企业投身脱贫，鼓励民营企业家〝坚定发展信心〞。习近平表示，〝任何否定、弱化民营经济的言论和做法都是错误的〞，北京支持民营企业发展这一点〝丝毫不会动摇〞。
 <br/>
 <br/>
 与此同时，耐人寻味的是,人民网一篇批评〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 〞的旧文被删。
 <br/>
 <br/>
 日前，中国互联网上流传一篇文章《人民日报集团突发雄文：清末以来中国三次〝国进民退〞的历史教训》。
 <br/>
 <br/>
 文章总结称，清末以来，中国发生过三场〝国进民退〞。第一场是在清朝末年，结果引发辛亥革命；第二场在抗战胜利后，结果拖垮了国民党政权；第三场发生在中共建政后，结果将经济推到崩溃边缘。
 <br/>
 <br/>
 文章随后批评说，2008年金融危机后，北京的经济政策开始偏向国企，新一轮〝国进民退〞大有潮涌之势，这不利于未来经济发展。
 <br/>
 <br/>
 经查询发现，这篇文章是《人民日报》旗下《文史参考》杂志2010年第3期刊登的，原题为《清末以来中国三次〝国进民退〞的历史教训》。当年8月，该文曾经发表在人民网，但现在已被删除。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-22/p9090441a791767960.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-22/p9090441a791767960-ss.jpg"/>
   <br/>
   <font size="-1">
    谷歌快照显示，文章在10月14日还没有被删。（网页截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 不过，在谷歌上还能查到该文今年10月14日在人民网的快照。这说明文章很可能是因为引起网民关注，于10月14日后才被人民网删除。文章被删原因，被猜测是因为〝政治不正确〞。
 <br/>
 <br/>
 近年来中国经济放缓，又加上中美贸易战高压，中国民企陷入经营困境。而北京近期加强对民企的接管和控制，社会上担忧北京会以〝国有化〞应对危机。
 <br/>
 <br/>
 近几个月来，地方国有资产不断并购和入股民企。日前，股市暴跌令大量民企股价逼近平仓线，面临被国有银行收回质押股权的危险。这令市场再次陷入恐慌，北京高层经济官员接连喊话，稳定股市。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
