### 【禁闻】10月16日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 河南邢望力案开庭 法院戒备森严
 <br/>
 <br/>
 河南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 邢望力被控〝寻釁滋事罪〞及〝敲诈勒索罪〞一案，10月12号在息县法院开庭审理，当局出动一百多名警察和便衣公安，将法院围得水泄不通，不让任何人接近法院。邢望力的辩护律师常伯阳事后对媒体表示，邢望力没有犯罪，他在庭上做了无罪辩护。
 <br/>
 <br/>
 2. 黄琦母北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 西方外交官关注
 <br/>
 <br/>
 被羁押一年多的〝六四天网〞创办人黄琦，最近被加控罪名，他85岁的母亲日前抵达北京，向中共最高检察院和最高法院递交信访材料，但遭到拒绝。10月12号，在北京的英国、德国及瑞典外交官探望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦母亲.html" target="_blank">
  黄琦母亲
 </a>
 ，并了解案情进展，表示会持续关注黄琦案。
 <br/>
 <br/>
 3. 辽宁丹东维权人士姜家文被带走失联
 <br/>
 <br/>
 辽宁丹东维权人士姜家文，10月15号在北京遭警方强行带走，与外界失联，目前下落不明。姜家文是近年〝访民春晚〞的策划人，因常年维权曾遭当局多次劳教、刑拘。
 <br/>
 <br/>
 4. 忧政府迫害 北京葛志慧搭帐篷露宿
 <br/>
 <br/>
 北京丰台区维权人士葛志慧，10月8号发现自己的残疾人车油箱底边被捆绑类似爆炸物的物体，而警方不予立案，她担心遭到政府的暗中迫害，10月15号在朝阳区莱太花卉市场对面的街头，搭帐篷露宿。
</div>
