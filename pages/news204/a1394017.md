### 刘晓庆为何比范冰冰〝冤〞？传因得罪江泽民
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 逃税案终于落地。官方3日通报，范冰冰缴纳8.8亿人民币罚款可免予刑责。对此，网民纷纷为大陆名星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘晓庆.html" target="_blank">
  刘晓庆
 </a>
 打抱不平，认为她当年坐牢坐的好冤。据悉，当年刘晓庆之所以入狱，是因为遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陷害.html" target="_blank">
  陷害
 </a>
 。
 <br/>
 <br/>
 10月3日，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_税务机构.html" target="_blank">
  税务机构
 </a>
 公布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 只要缴纳8.8亿人民币罚款，将不予追究刑事责任。
 <br/>
 <br/>
 对于范冰冰因巨额逃税而免除刑责，网民纷纷与知名演员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘晓庆.html" target="_blank">
  刘晓庆
 </a>
 做对比，直呼〝刘晓庆坐牢坐的好冤〞。因为按官方公开的逃税金额来看，范冰冰的逃税金额达刘晓庆的18倍。最终结果却是，范冰冰认罚了事，而刘晓庆却为此坐牢400多天。
 <br/>
 <br/>
 当年税务机关调查报告称，刘晓庆和她的公司，采取不列、少列收入，多列支出，虚假申报，通知申报而拒不申报等手段，偷逃税1458.3万人民币。
 <br/>
 <br/>
 2002年7月，刘晓庆被逮捕，2003年8月，被取保候审。刘晓庆在监狱中共呆了422天。
 <br/>
 <br/>
 对比两人的遭遇，中国律师直言，范冰冰免除
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牢狱之灾.html" target="_blank">
  牢狱之灾
 </a>
 得益于新税法。
 <br/>
 <br/>
 北京资深律师刘巧玉发文说，中共在2009年对刑法201条偷税罪进行重大修改，逃税罪取代偷税罪，加之范冰冰属于初犯，只需缴税款及滞纳金，将〝不追究刑事责任〞。
 <br/>
 <br/>
 但是外界分析，刘晓庆与范冰冰不同，除了中共旧的税法之外，更重要的一点，她得罪了时任中共党魁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 。
 <br/>
 <br/>
 根据早期报导，1993年国税局就开始掘地三尺查刘晓庆的账，但没查出毛病，贴身跟踪她的人问：〝你到底得罪了哪个大人？〞
 <br/>
 <br/>
 新华网1996年报导，刘晓庆出现偷漏税问题。当时《北京青年报》刊文质疑，这不是笑话吗，1993年查时没偷税，1996年却敢于让账目出了毛病？
 <br/>
 <br/>
 知情人士说，在刘晓庆一案中，关键人物是其公司的王建中，以及与之配合的还有一位李姓女子，二人联手为司法机关提供了最〝致命证据〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陷害.html" target="_blank">
  陷害
 </a>
 刘晓庆〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_偷税漏税.html" target="_blank">
  偷税漏税
 </a>
 〞。这就是为什么1993年没问题，1996年账面有了问题的原因。
 <br/>
 <br/>
 事实上，王建中是江泽民安插进刘晓庆公司的，目的就是报复陷害刘晓庆。
 <br/>
 <br/>
 2002年底，刘晓庆案还没有结论，江泽民已让人把她的所有房产都变卖了，使得她无栖身之地。江还让办案的人直截了当地告诉刘晓庆，只给她留一条命，条件还是让她闭嘴。
 <br/>
 <br/>
 这段恩怨要从邓小平掌权时说起。据报，当时刘晓庆是中国最红演员，也是邓小平家中的座上宾。江得知后百般讨好，满口〝晓庆妹妹〞，刘晓庆则以〝江叔叔〞回应，让江碰了一鼻子灰。
 <br/>
 <br/>
 刘晓庆曾在邓小平家中挤兑江泽民说：〝您一天兵都没当过，晓庆我还拿过枪呢。〞〝您还拿枪啊，您应该闪远点儿，您别走了火把自己的大肚囊子给崩了！〞在众人的哄笑声中，江泽民抱着肚子笑说：〝晓庆妹妹真幽默！〞不料却怀恨在心，得势后，马上开始着手报复刘晓庆。
 <br/>
 <br/>
 刘晓庆曾不止一次表白：〝我不是一个喜欢炒作的人，到今天我可以自豪地说：我的一生中没有过不仁不义，没有任何潜规则。〞
 <br/>
 <br/>
 外界认为，不肯做江泽民的情妇，是刘晓庆蒙冤入狱的最大原因。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
