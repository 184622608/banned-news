### 江西13岁男孩探出轿车天窗 撞限高横槓身亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】江西日前发生一起交通意外事故，一名13岁的男孩在轿车行驶过程中，将身体伸出车顶天窗外，之后即撞到道路上设置的车辆限高横槓，当场身亡。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_儿童乘车安全问题.html" target="_blank">
  儿童乘车安全问题
 </a>
 ，再度引发舆论热议。
 <br/>
 <br/>
 据陆媒10月29日报导，28日下午5点左右，一辆私家车在江西新余市渝水区经开大道上，由南向北（良山镇往新余市区）方向行驶时，车内13岁男孩钟某将身体伸出车顶天窗外，不久钟某的身体与限高横槓发生碰撞，当场死亡。
 <br/>
 <center>
  <br/>

 </center>
 <br/>
 从网传视频中可以看到，一辆银灰色轿车行驶在一条单向两车道的公路上。此时，车内有个小男孩探出车顶天窗，将上半身伸出车外。
 <br/>
 <br/>
 当轿车行驶致一处弯道时，突然遇到了车辆限高横槓，但轿车并未减速，继续前行。身体探出车顶天窗的男孩也没有做出任何应急反应，随即与横槓正面相撞，仰面倒在了车顶。
 <br/>
 <br/>
 据报导，涉事私家车驾驶员袁某和钟某没有亲属关系，钟某的监护人当时不在他旁边，而是坐在后面的车辆中。目前，该事件正在处理中。
 <br/>
 <br/>
 对于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_儿童乘车安全问题.html" target="_blank">
  儿童乘车安全问题
 </a>
 ，再度引发舆论热议。早在今年5月，大陆网路也疯传一段视频，在湖南张家界的高速公路上，一辆飞驰的私家车打开天窗，任由3名不到10岁的小孩坐上车顶〝吹风〞，场面十分惊险。
 <br/>
 <br/>
 据目击网友说，当时，那辆私家车车速相当快，快驶近收费站时，私家车才慢慢减速，3名小孩这才返回车内。有网友说，〝太危险，摔下来才会让家长长记性〞。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
