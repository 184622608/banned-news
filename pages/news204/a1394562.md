### 【禁闻】老兵聚平度讨说法 与警方爆激烈冲突
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】10月4号，山东平度的38名老兵在进京途中遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力截访.html" target="_blank">
  暴力截访
 </a>
 、殴打，多人受伤。这引发山东及全国各地的老兵纷纷前往平度，向官方讨要说法。警方多次尝试强行清场，双方一度发生激烈冲突。请看报导。
 <br/>
 <br/>
 从4号晚间开始，就陆续有各地老兵赶往山东平度市，声援被截访殴打的老兵。到5号下午，平度市政府附近的文化馆广场上已经聚集了近500名老兵。他们打出大型横幅：〝雇用黑社会 打压退役老兵 揪出幕后黑手 除恶打黑〞，向官方讨要说法。
 <br/>
 <br/>
 值得注意的是，与以往不同，这次老兵们为了应对警方的暴力镇压，特意准备了木棍、灭火器等用来防身，有的老兵还头戴钢盔。
 <br/>
 <br/>
 5号晚间，平度警方试图清场，他们将周边围观群众驱散，并让数十名黑衣人冲进老兵阵营，想要把老兵们冲散、驱离现场。双方发生了小规模冲突，最后黑衣人被手持木棍的老兵们赶跑。
 <br/>
 <br/>
 辽宁访民姜家文：〝这场事件应该引发一次大的维权行动，直接惊动中共上层，触动他们的要害，他们知道疼了，才知道人民的力量。现在他们不怕人民。〞
 <br/>
 <br/>
 6号，不断有全国各地的老兵赶来声援，平度官方则调集大批警力对老兵所在地进行围堵，现场气氛越发紧张。老兵们要求平度市政府向被打老兵道歉，赔偿老兵损失，但不被接受。
 <br/>
 <br/>
 当天下午，更多警力调入平度，将周边道路封堵，拦截全国各地赶来增援的老兵。现场附近大街上则到处是警察，他们拿着盾牌、警棍围向老兵。
 <br/>
 <br/>
 现场老兵：〝现场警方力量达到将近3000人，我们老兵在场内的不到500人，外围的进不去。他们把我们给隔离了。〞
 <br/>
 <br/>
 过程中，双方发生激烈冲突。一些特警不敌老兵手中的木棍，被打得逃上警方的中巴车顶，显得异常狼狈。
 <br/>
 <br/>
 姜家文：〝对共产党的统治、专政独裁的暴政统治，大家以前是敢怒不敢言，现在真正站出来敢发声、勇敢地直面抗争、敢拿起武器那天，就是共产党要灭亡的时候。〞
 <br/>
 <br/>
 警方还动用催泪瓦斯等袭击老兵，强行驱赶老兵。虽然老兵们使用木棍、灭火器进行反击，但最终无奈在大批警力的攻击下，被迫撤离现场。
 <br/>
 <br/>
 本次山东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平度老兵.html" target="_blank">
  平度老兵
 </a>
 维权事件，起因于当地官方对进京老兵们的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力截访.html" target="_blank">
  暴力截访
 </a>
 。4号上午，平度市转业军人、家属到平度市李元广场举行活动，之后其中38人决定前往北京，因买不到火车票，只好选择自驾。
 <br/>
 <br/>
 当他们行车至东营市高速路利津服务区的加油站时，遭到当地警方雇用的数十名黑社会人员的突然袭击。老兵们被暴力殴打，6人受伤，其中2人伤势较重。
 <br/>
 <br/>
 这一导火索直接导致了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全国老兵.html" target="_blank">
  全国老兵
 </a>
 集结平度，讨要说法。据现场老兵介绍，在6号现场老兵们与警方发生冲突、撤离后，后续还有大量全国各地的老兵聚集到广场声援。
 <br/>
 <br/>
 现场老兵：〝现在现场已经超过了昨天老兵的数量了。（老兵们）一直在往这边赶，全国各地。〞
 <br/>
 <br/>
 官方则再次调集大约上千名特警携带盾牌、警棍赶到广场，现场局势再次紧绷，非常紧张。
 <br/>
 <br/>
 近年来，中国大陆的退伍老兵们因为待遇不公等问题得不到解决，被迫进行维权、抗议。仅最近几个月，老兵们就已经进行了多次大型维权行动。
 <br/>
 <br/>
 〝十一〞期间，河北、四川、河南等多个地方都有大批老兵聚集。
 <br/>
 <br/>
 9月中下旬，全国老兵到中共退役军人事务部维权，后又转至中央军委政治工作部信访大院上访维权。
 <br/>
 <br/>
 稍早前，在6月中下旬，更有全国数千老兵聚集到镇江市政府前的广场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权抗议.html" target="_blank">
  维权抗议
 </a>
 。当局如临大敌，软硬兼施，极力阻止老兵进入镇江。6月23号凌晨，当局动用大批警力暴力清场，许多老兵被打，酿成流血事件。
 <br/>
 <br/>
 采访/陈汉 编辑/李谦 后制/钟元
