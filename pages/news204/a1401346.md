### 看着是你听着是你但真不是你 中国AI诈骗频发
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】中国大陆近日频频发生骗子利用人工智能（AI）技术模仿他人的脸孔和声音进行诈骗的事件。由于声音和画面的拟真程度很高，受害者难辨真伪，防不胜防。
 <br/>
 <br/>
 中国大陆微信公众号〝极客视界〞日前发表一篇文章提醒公众，已经开始有人利用AI技术和社交媒体〝微信语音〞的平台设计骗局诈骗钱财。
 <br/>
 <br/>
 据称，中国一名赵姓女子近日收到〝父亲〞的语音留言，称自己没钱买菜了，要赵女转帐200人民币救急。赵女一听确实是父亲的口音就转了钱，结果等她回到家中才发现自己被骗了。
 <br/>
 <br/>
 另外，苍洲一名王姓女子也因接到〝同学〞的语音留言要借钱而信以为真，结果被骗走500元人民币。
 <br/>
 <br/>
 这篇文章指出，骗子首先盗取了诈骗目标的微信帐号，然后透过提取语音文件或者安装〝增强版微信〞，以达到转发语音的效果，再编造一个理由就可轻易骗取对方的信任。
 <br/>
 <br/>
 文章称，骗子不但可以透过AI技术逼真地模仿出他人的声音，而且还可以惟妙惟肖地模仿他人的语气情绪。因此就算接到语音留言的人打电话询问对方。对方也能够模仿出足以以假乱真的回答。
 <br/>
 <br/>
 更为严重的是，就算人们想通过视讯方式来分辨真假，骗子也可以透过AI伪造他人的脸孔来进行诈骗。
 <br/>
 <br/>
 文章提到，有一名网络名称为〝deepfakes〞的工程师，就曾经利用透过人工智能技术的换脸〝FakeApp〞程式，把好莱坞女星的脸孔，移花接木到一名成人片的女星身上。
 <br/>
 <br/>
 换句话来说，通过使用AI技术。完全可以做到〝看上去是你，听起来也是你，但那真的不是你。〞
 <br/>
 <br/>
 文章称，这项技术甚至被人应用于制造假新闻，引发了不小的争议。
 <br/>
 <br/>
 据介绍，声音合成技术Lyrebird是谷歌2016年在加拿大蒙特娄大学建立的人工智能研究实验室（MILA）开发的。这项技术透过人工神经网络（Neural Network）和机器学习（Machine Learning），可以基于音色、音调、音节、停顿等多种特征来定义某个人的声音，然后产生十分拟真的声音。Lyrebird创始人曾表示，他们希望通过这项技术去帮助那些发声困难的人。
 <br/>
 <br/>
