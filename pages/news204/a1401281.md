### 李登辉跌倒撞伤额头 加护病房住院三天
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾前总统.html" target="_blank">
  台湾前总统
 </a>
 、95岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李登辉.html" target="_blank">
  李登辉
 </a>
 ，11月29日上午在家不慎跌倒，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_撞伤.html" target="_blank">
  撞伤
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_额头.html" target="_blank">
  额头
 </a>
 ，在家紧急处理后，由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_救护车.html" target="_blank">
  救护车
 </a>
 送往荣总治疗。接下来将会住院3天，接受检查。
 <br/>
 <br/>
 11月29日上午11时，现年95岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾前总统.html" target="_blank">
  台湾前总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李登辉.html" target="_blank">
  李登辉
 </a>
 ，在家看完电视想休息，从客厅走回卧室时，疑因没扶好而跌倒，撞到头部受伤流血，由家中护理师紧急处理后，再通知
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_救护车.html" target="_blank">
  救护车
 </a>
 ，送往台北荣民总医院救治。
 <br/>
 <br/>
 李登辉的主治医师陈云亮表示，目前李登辉病情虽稳定，但他年事已高，风险仍相当高，未来3天住进神经外科加护病房治疗，将密切监控脑部及肺部状况。
 <br/>
 <br/>
 29日下午1时30分，李登辉基金会秘书长王燕军受访时表示，送医时，李登辉的意识还很清楚，没有骨折等明显外伤，只有鼻腔流血，血已止住，晚上6时将会安排更完整的检查，包括计算机断层和血管出血性检查等。
 <br/>
 <br/>
 他说，目前外伤是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_额头.html" target="_blank">
  额头
 </a>
 撞击伤，因为李登辉是站起来瞬间正面趴下去的，造成额头碰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_撞伤.html" target="_blank">
  撞伤
 </a>
 和流鼻血，接下来将会住院3天，接受检查。而蔡英文总统也有致电关心。
 <br/>
 <br/>
 李登辉曾因心脏血管阻塞，多次进行心导管支架置放手术，并曾在2015年11月轻微中风，这之后，有关李登辉死亡的传言便接连不断的出现。
 <br/>
 <br/>
 9月份，多家台湾媒体称李登辉数次进出医院，还有传言称，李登辉因心脏病死于家中。不过王燕军在受访时辟谣说，〝不知道为什么会有这种消息传出〞。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：赵云）
