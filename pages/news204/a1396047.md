### 曾上春晚唱〝血染的风采〞 军人〝偶像〞徐良成访民
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】中国各地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退伍军人维权.html" target="_blank">
  退伍军人维权
 </a>
 浪潮风起云涌，此起彼伏。日前，曾因央视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_春晚.html" target="_blank">
  春晚
 </a>
 上一曲《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_血染的风采.html" target="_blank">
  血染的风采
 </a>
 》而家喻户晓的伤残军人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐良.html" target="_blank">
  徐良
 </a>
 ，也成为其中一员，引起外界关注。
 <br/>
 <br/>
 10月18日，海外推特流出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐良.html" target="_blank">
  徐良
 </a>
 当天现身北京退役军人事务部的一段视频。爆料网友指，北京治安总队一名副队长身穿便衣，跟随在徐良的轮椅后方，在现场负责〝维稳〞。

 徐良1980年代在中越战争中受重伤，失去一条腿，后被中共封为〝一级战斗英雄〞，还曾被安排在央视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_春晚.html" target="_blank">
  春晚
 </a>
 上和歌手王虹合唱《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_血染的风采.html" target="_blank">
  血染的风采
 </a>
 》，一夜间成为全国家喻户晓的人物。
 <br/>
 <br/>
 有网友指，徐良在六四事件中多次〝慰问〞北京戒严部队，显示和当局〝保持了高度一致〞。

 <br/>
 19日，徐良一名不愿透露姓名的退伍军人好友，向自由亚洲电台证实了徐良18日到访退伍军人事务部的消息。
 <br/>
 <br/>
 这名退伍军人还说，当时是他推徐良去的，一起去的战友大概有不到10个人。他和徐良获得当局高规格待遇，退伍军人事务部有专人出来接待。他们填了登记表格递上去了。
 <br/>
 <br/>
 徐良现居陕西西安。这名好友不愿透露徐良到访北京的具体诉求，只说徐良希望体现出〝退伍军人的尊严〞。但他也表示，徐良不是心血来潮才上京，〝包括退役之后遇到的各种困难，反正有这方面问题的人，他都会想到要到退伍军人事务部〞。
 <br/>
 <br/>
 本身是退伍军人的珠海维权人士陈凤强表示：〝在这种体制之下，什么事情都有可能发生。以前的老红军老八路，抗美援朝也很多人上访维权，说明当局要你保家卫国的时候，就把你捧上天。现在和平年代，贪官就不理你死活了。〞
 <br/>
 <br/>
 他说，徐良与很多退役老兵一样，都是中共专政的受害者。
 <br/>
