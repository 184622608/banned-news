### 10月8日全球看中国
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】下面是今天的全球看中国
 <br/>
 <br/>
 1. 自由亚洲电台
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵维权.html" target="_blank">
  老兵维权
 </a>
 再次失败 被官方刻意贬称〝兵痞〞
 <br/>
 <br/>
 10月5日，38名平度老兵欲进京上访，遭当地警察拦截殴打。周日，数以千计来自全国各地的退役军人到平度市中心抗议，引来大量武警以武力清场。官方并刻意贬低老兵，将他们定性为〝兵痞〞，并严禁民众给予他们帮助。
 <br/>
 <br/>
 2. 大纪元时报
 <br/>
 中国抄袭34部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国综艺节目.html" target="_blank">
  韩国综艺节目
 </a>
 其它仿冒案例比比皆是
 <br/>
 <br/>
 韩国议员金圣洙表示，中国抄袭了34部韩国综艺节目。例如《我家的熊孩子》，被抄袭为《我家那小子》；《PRODUCE 101》第一季被抄袭成《加油！美少女》等。另外，韩国统计2017年遭侵权的物品共有7,263件。其中，中国占了94.5%。
 <br/>
 <br/>
 3. 希望之声
 <br/>
 威胁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家安全.html" target="_blank">
  国家安全
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肯尼亚.html" target="_blank">
  肯尼亚
 </a>
 再逮捕5中国人
 <br/>
 <br/>
 10月5号，肯尼亚以威胁国家安全为由在首都内罗毕逮捕了5名中国人。当地警方从他们的住所发现了金属探测器，对讲机，催泪瓦斯等。这5人声称计划在当地开一家安保公司，但还没有工作许可。
