### 北京出击欧盟〝软肋〞遇挫 西班牙拒入一带一路
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】中美贸易战正在紧张对阵，北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 已到访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 ，稍后将走访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_葡萄牙.html" target="_blank">
  葡萄牙
 </a>
 。专家分析，这两个欧盟国家对中共态度相对较温和，北京正试图寻找欧洲的软肋。但西班牙官员在习到访前就已放话，不会加入中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在当地时间周二（11月27日）抵达马德里，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 展开为期3天的访问。这是最近12年来中国国家主席首次出访西班牙。12月初，习近平在阿根廷出席G20峰会之后，还将到访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_葡萄牙.html" target="_blank">
  葡萄牙
 </a>
 。
 <br/>
 <br/>
 法广引述西班牙地缘政治问题专家卡伦萨分析指，北京面临美国对其两千亿产品追加关税的沉重压力，欧盟委员会也将于下个月通过加强监控外国投资的指令，预计习近平这次出访西班牙和葡萄牙，是为寻求与欧洲的合作，并将在欧洲发表演讲安抚舆论。
 <br/>
 <br/>
 法国智库亚洲研究中心主席迪茂柳先生对法新社评论称，因为担忧中国企业并购欧洲高科技产业，法国，德国以及意大利一直在呼吁欧盟加强对外资的监控，而西班牙和葡萄牙在这方面立场不那么强硬。而且，在欧盟国家中，中国资本在西班牙与葡萄牙国内投资中所占比例位居前列。
 <br/>
 <br/>
 因此，迪茂柳认为，习近平到访西班牙与葡萄牙，是为寻找中国资本渗透欧洲的软肋，并试图以此为切入口改善与欧盟的关系。
 <br/>
 <br/>
 不过，西班牙在与北京的交往中依然刻意保持着距离。
 <br/>
 <br/>
 周二，习近平在会见西班牙国王费利佩六世时表示，中方愿同西班牙继续相互支持，双方要在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 〞框架内加强合作等。习近平还希望西班牙为推动中欧关系发展〝发挥积极作用〞。
 <br/>
 <br/>
 不过，费利佩六世回应称，西班牙愿意和中方保持交往，提升务实合作水平，并会帮助推动欧中关系。但没有提及〝一带一路〞。
 <br/>
 <br/>
 其实，早在习近平到访前，西班牙首相府一名官员就公开表示，西班牙不会加入〝一带一路〞倡议，并称欧洲国家应在现行框架内作业。
 <br/>
 <br/>
 外媒指，西班牙官员口中的〝现行框架〞，是指联盟（EU）9月提出的新战略，即通过支援亚洲地区基础建设来加强欧亚之间的联系。该计划被指欧盟版〝一带一路〞，被指明显旨在抗衡中共〝一带一路〞。
 <br/>
 <br/>
 西班牙官员预料，习近平此行将与西班牙签署近20项具体的商业和合作协议，包括中国进口西班牙猪肉产品的协议，进口葡萄协议，以及避免双重征税，还有港口合作文件等。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
