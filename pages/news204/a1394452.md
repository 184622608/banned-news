### 【禁闻】彭斯〝伐共宣言〞：美国决定性回击
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】日前，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 发表重磅讲话，重整
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中政策.html" target="_blank">
  美中政策
 </a>
 。专家认为，彭斯的讲话矛头直指
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 ，同时表达了对中国人民的关切。
 <br/>
 <br/>
 日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 在华盛顿哈德逊研究所，就美国政府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_对华政策.html" target="_blank">
  对华政策
 </a>
 的演讲，引发全球关注。
 <br/>
 <br/>
 彭斯开篇历数美国过去几十年对中国的巨大支持，例如美国向中国投资、帮助培养人才、打开加入世贸大门等等，彭斯重申
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的话説：是〝美国重建了中国〞，如今
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 却对美国进行种种挑衅。
 <br/>
 <br/>
 彭斯説，中共企图利用从美国获得的技术，在陆、海、空，乃至外太空抗衡美国军力，并试图阻止美国援助盟友，将美国挤出西太平洋。彭斯还说，北京正在使用一种全政府的手段，利用政治、经济、军事工具以及宣传等方式，在美国推进其影响和利益，干预美国民主选举。
 <br/>
 <br/>
 同时，彭斯例举了中共在世界和中国的种种霸凌行经，包括通过所谓的〝债务外交〞扩大其影响力、支持国际上腐败和欺压人民的专制政权、在南中国海军事化、打压拥有民主的台湾、对国内民众进行监控等等。
 <br/>
 <br/>
 彭斯説，美国在川普总统的指导下，对中共开启了决定性的回击。
 <br/>
 <br/>
 卡内基国际和平基金会副总裁包道格（Douglas Paal）在接受《美国之音》采访时表示，川普政府认识到美国是在与中共进行存亡之争，彭斯的讲话是一个进行战斗的号声，并认定美国会赢得这场战争。
 <br/>
 <br/>
 美国南加州大学公共政策博士叶科：〝彭斯的讲话实质是自由世界和正义力量反击邪恶的一个重要宣言，自由世界几十年来在自由和人权的价值已经沉默的太久了，今天等于美国真正的挺起了胸膛。我觉得这将重建一个新的世界次序，会带来一个新的世界体系和规则。〞
 <br/>
 <br/>
 中华经济研究院特约研究员吴惠林认为，川普等美国精英，早已认识到此前美国对中共是养虎为患，因此川普上台后态度非常坚定，采取了一系列动作，包括解决朝鲜核武问题，要求退出各种世界组织，阻止世贸组织那些所谓大法官连任等等行动，都是川普重建世界次序的组成部分。
 <br/>
 <br/>
 中华经济研究院特约研究员吴惠林：〝川普早就计划好了，所以有的人看到怎么那么快，而且又那么准，打到要害。因为这些国际组织都已经变了，所以他一上来就退出这些组织，另外一方面又单独的同各国进行谈判，所以这些国家现在都靠拢了，也就是説他在重整国际的各种次序，把中共排除在外。〞
 <br/>
 <br/>
 彭斯在演讲中还表达了对中国人民的关切，彭斯指出，近年来中共朝着控制和压迫本国人民的方向急转弯，对民众进行监控，迫害宗教信仰，中国人民自由的希望仍没有实现。
 <br/>
 <br/>
 彭斯向北京最高层领导人呼吁：回归几十年前两国关系开始时的改革开放精神，彭斯说，美国人民别无所求，但是中国人民理应得到更多。
 <br/>
 <br/>
 叶科：〝他的目标是非常明确的，他实质上是真正指向了中共政权，他区分了中国这个政府和中国民众，非常清楚的把中共做的一切恶行和中国老百姓区分开来，他甚至把北京主政者和这个政权都做了区分，他真正反对的是这个邪恶的政权采用这些与历史背道而行的政策，他并没有説反对中国，甚至对习近平的抨击他都还没有。〞
 <br/>
 <br/>
 吴惠林：〝他也要重塑中国，也就是把中共拿下去，让他们变成民主自由的世界，所以他是为中国人着想，为全世界人民在着想，并不只是美国人民，可是开始的时候，美国一定要强大，你不强大，人家都怕中共。现在美国也上来了，人民的信心也有了，现在全世界也逐渐的归队。〞
 <br/>
 <br/>
 美国南加州大学公共政策博士叶科説，中共绑架13亿中国人，在全世界进行邪恶扩张，同时通过收买美国的政客、商人、媒体、游说公司，再加上特务组织、大外宣等手段，已经对自由世界的领头羊－－美国，渗透的非常厉害，川普政府能够在紧急关头清醒，做出勇敢而明智的决定，对人类历史而言是万幸。
 <br/>
 <br/>
 采访编辑/刘惠 后制/陈建铭
