### 范冰冰复出无望？党媒罕见再发严厉批评
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 因卷入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃税案.html" target="_blank">
  逃税案
 </a>
 遭当局罚款8.8亿元人民币后，有关她将复出的消息不断传出。然而党媒16日再度发文，呼吁加强文艺界法治建设，强调不再让有法律和道德污点的艺人出名获利。有观点认为，这意味着范冰冰复出之路被堵死。
 <br/>
 <br/>
 11月16日，中共党媒人民日报刊文再度批评〝严重失德失信行为的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演艺人员.html" target="_blank">
  演艺人员
 </a>
 〞，声称要让这些人接受行业制定的、与其言行严重程度相应的、有足够约束力的处罚。
 <br/>
 <br/>
 文章还强调不能再让有法律和严重道德污点的艺人出名获利。并批评一些演艺人员忽视自身思想道德建设，导致品位低下、格调低俗、责任缺失，缺乏必要的法律、道德修养和深厚的文化涵养。
 <br/>
 <br/>
 文章还引用习近平的讲话，警告中国娱乐圈的演员处理好〝德与艺〞〝义与利〞的关系，抵制低俗庸俗媚俗，努力成为时代风气的先倡者。
 <br/>
 <br/>
 文章虽然没点名，但舆论将其女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 联系到一起，范冰冰因偷税漏税10月3日被官方罚超8.8亿元人民币，在传出范冰冰缴清罚款后，网路上有关范冰冰可能复出的消息不断。
 <br/>
 <br/>
 10月19日，范冰冰与母亲张传美现身在北京三里屯的工作室，因张传美早期曾担任过范冰冰的经纪人，当时媒体就曾纷纷猜测，范冰冰可能即将复出。再加上最近一些广告商重新上架范冰冰代言的产品等，有关范冰冰即将复出的说法似乎真实不虚。
 <br/>
 <br/>
 然而，如今人民日报再度刊文批有法律和道德污点的艺人。有观点认为，这或意味着范冰冰复出之路彻底被堵死，其演艺生涯可能会终止，范冰冰今后只能在幕后进行工作了。
 <br/>
 <br/>
 再加上范冰冰诸多代言被取消，好莱坞电影《355》准备解雇她，连她客串的《大轰炸》在大陆也取消上映，而且她手里至少有6个电影项目面临〝上映困难〞。即使她再度回到公众视野，演艺生涯恐怕也不再如以前顺利了。
 <br/>
 <br/>
 近期有关范冰冰与未婚夫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李晨.html" target="_blank">
  李晨
 </a>
 将要移民美国的消息在网上流传。很多中国网友和范粉纷纷表示：远离这个是非之地，离开了就别再回来了。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
