### 上海再爆特大丑闻 原公安局长涉杀二奶和人大代表
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】原上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金山区.html" target="_blank">
  金山区
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安局长.html" target="_blank">
  公安局长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马淮海.html" target="_blank">
  马淮海
 </a>
 ，5年前被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_情妇.html" target="_blank">
  情妇
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_举报.html" target="_blank">
  举报
 </a>
 贪腐20多亿，并
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谋杀.html" target="_blank">
  谋杀
 </a>
 市人大代表和二奶。这一特大丑闻被曝光后，直到2015年马淮海才被免职。值得关注的是，马淮海据传是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侄子.html" target="_blank">
  侄子
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴志明.html" target="_blank">
  吴志明
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马仔.html" target="_blank">
  马仔
 </a>
 。该事件日前再被翻炒，引发猜测。
 <br/>
 <br/>
 近日，网络媒体再次翻炒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马淮海.html" target="_blank">
  马淮海
 </a>
 被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_情妇.html" target="_blank">
  情妇
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_举报.html" target="_blank">
  举报
 </a>
 的特大丑闻。据网上流传的长微博显示，上海一家著名媒体的记者爆料，2013年8月5日上午9时30分左右，大多报社记者都还没有上班，一名颇有姿色的少妇来到报社，自称是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金山区.html" target="_blank">
  金山区
 </a>
 副区长，区公安局局长马淮海的情人陈萍（化名）。
 <br/>
 <br/>
 陈萍向报社提供材料和线索显示，马淮海贪污受贿20多亿，拥有房产60多处。陈萍还提供了其中31套房产证的复印件。
 <br/>
 <br/>
 马淮海，1955年2月出生，汉族，山东诸城人。马淮海从2010年5月至2015年9月，一直担任金山区政府副区长、党组成员、区委政法委副书记、公安金山分局党委书记和局长。
 <br/>
 <br/>
 据陈萍介绍，马淮海在公安系统工作20多年，多年来以受贿、收钱〝捞人〞和〝减刑〞等手段为自己攫取巨额财富。
 <br/>
 <br/>
 2013年5月，马淮海还与人合伙
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谋杀.html" target="_blank">
  谋杀
 </a>
 了一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海市.html" target="_blank">
  上海市
 </a>
 人大代表、身价数亿的企业家。
 <br/>
 <br/>
 陈萍称，马淮海与这位亿万富翁的下属合谋，派人将在KTV包厢中已喝得半醉的富翁叫出打死，并丢到废弃的电梯井内，伪装成失足摔死。马在发现尸体后，马上下令定性为意外死亡，为此他与人瓜分了亿万富翁的数亿财产。
 <br/>
 <br/>
 据中共官媒同年报导，2013年5月9日晚间，上海金山区朱泾镇亭枫公路皇家会所KTV内，一男子从电梯井坠下身亡。次日，上海媒体记者证实，坠亡男子是上海进念室内设计装饰有限公司董事长、上海市室内装饰行业协会会长黄振。
 <br/>
 <br/>
 当时的报导称，上海警方宣布已经排除他杀。黄振是上海市人大代表、致公党上海市常委。
 <br/>
 <br/>
 此外，陈萍还举报称，马淮海包养了十几名年轻貌美的情妇，送这些情妇每人一套高档住房和一辆名车。
 <br/>
 <br/>
 2013年4月，一名情妇惠儿为马淮海生了一个儿子，提出结婚未果，一怒之下发帖举报马，结果被马活活打死。
 <br/>
 <br/>
 该件事令陈萍感到非常害怕。因为陈萍这两年受到冷落，跟马淮海手下的一名年轻的警察好上了，害怕被马发现丢了性命，于是决定冒险到报社求助。
 <br/>
 <br/>
 据称陈萍面临马淮海的追杀，当时逃离上海躲避。
 <br/>
 <br/>
 但陈萍的检举当时并没有被媒体曝光，只是在网络上流传，原始帖也很快遭到删除。上海官方也没有对此事进行回应。至于陈萍的命运，也无从知晓。
 <br/>
 <br/>
 直到2015年9月25日，上海市金山区政府网站消息称，中共上海市公安局金山分局局长马淮海被免职。
 <br/>
 <br/>
 但马淮海被免职似乎与上述举报无关。据说是因2015年6月22日起，上海市金山区曾有上万民众每天持续上街维权反对PX项目，抗议活动持续了一周，上海市官员迟迟不露面。
 <br/>
 <br/>
 当时消息人士透露，相关抗议事件是江派上海代理人谋划激发群体性事件，目的是引发开枪滋事，要挟习近平。
 <br/>
 <br/>
 上海是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 的大本营，江氏家族在上海建有庞大的政治经济利益网络。其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_侄子.html" target="_blank">
  侄子
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴志明.html" target="_blank">
  吴志明
 </a>
 曾担任上海市政协主席，曾经掌控上海公安、政法系统十几年。马淮海是吴的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马仔.html" target="_blank">
  马仔
 </a>
 。2018年两会吴志明下台。而马淮海的丑闻如今再被翻炒，是否会引起官方注意，甚至籍机查出一连串大案要案，值得关注。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：范铭）
