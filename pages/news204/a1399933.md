### APEC宣言流产内幕曝光 各成员国围堵中共
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】2018年APEC亚太经合组织领导人峰会闭幕后，没有按照惯例发表领导人宣言。对此，与会的知情官员称，APEC
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首脑宣言.html" target="_blank">
  首脑宣言
 </a>
 流产的原因不仅仅是中美分歧，而是中共与其他各个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_成员国.html" target="_blank">
  成员国
 </a>
 都有分歧，在围堵中共方面，西方国家已达成了共识。
 <br/>
 <br/>
 澳大利亚媒体《澳大利亚人报》19日评论称，由于美中之间的紧张局势加剧，在巴布亚新几内亚召开的亚太经合会议（APEC）峰会，在混乱中结束，而双方〝戏剧性的〞分歧，导致该峰会近29年来首次没能发表联合公报。
 <br/>
 <br/>
 文章称，在美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）指责中共企图〝将我们的伙伴淹没在债务海洋〞后，中美两国未能在联合公报中三个有争议的部分做出让步，此举导致各国领导人在峰会结束后未能达成共识。
 <br/>
 <br/>
 彭斯和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 17日在峰会上激烈交锋。习近平谈了中美贸易战和北京所倡议的〝一带一路〞等。彭斯则点名批评中共的贸易政策，及中共的〝一带一路〞是给合作国家设立的债务陷阱。
 <br/>
 <br/>
 峰会18日结束后，主办国巴布亚新几内亚高层指责中共官员施加威胁。据《华尔街日报》报导，东道国两位高级官员称，要求与巴布亚新几内亚外交部长会面的中共官员，17日闯进了他的办公室，在一场对峙后被警方带了出去。但中共否认发生此事，当地警方未回复置评请求。
 <br/>
 <br/>
 据一位听取了情况通报的官员称，这些中共官员要求会面，是为了表达对公告草案措辞的不满。官员们在结束为期两天的会议后，没有按照惯例发表领导人宣言。这是自亚太经合组织首脑会议自1993年首度召开以来，第一次未发表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首脑宣言.html" target="_blank">
  首脑宣言
 </a>
 。
 <br/>
 <br/>
 美国政府一位高级官员称，领导人宣言流产并非因为中共和美国的分歧，而是中共与所有其他APEC
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_成员国.html" target="_blank">
  成员国
 </a>
 的分歧。而分歧主要是一句拟议的内容，即〝我们反对贸易保护主义，包括所有不公平贸易行为。〞
 <br/>
 <br/>
 这位美国官员表示，中共不同意这句话，认为这相当于单独挑出中共的贸易行为，但所有其他20个APEC国家都同意最终在公报中加入这个内容。
 <br/>
 <br/>
 此外，西方国家在峰会期间，就如何合力阻止中共在太平洋地区的扩张方面，也取得了共识。在峰会闭幕日，澳大利亚、美国、日本以及新西兰，达成了一项关于为巴布新几内亚提供电力和光纤的投资计划，旨在对抗中方〝一带一路〞倡议下的投资。
 <br/>
 <br/>
 17日峰会期间，彭斯也曾抨击中共的〝一带一路〞，是为合作国家设立债务陷阱。彭斯说，美国是基于平等、透明、共创繁荣为原则，并以创造当地就业机会为特征的发展援助项目，并再次警惕发展中国家，不要掉进某些国家设置的债务陷阱，不要接受会损害主权和独立的外债，讽刺〝一带一路〞是〝约束他国的不归路〞。
 <br/>
 <br/>
 他还说：〝我们不会让我们的搭档淹死在债务的海洋之中。我们不会恐吓，不腐败，也不会损害你们的独立。美国公开公正地与他国打交道，我们不会提供一个约束性的带，或一个单向的路。〞
 <br/>
 <br/>
 彭斯13日接受《华盛顿邮报》专访时，也警告中共必须全面改变其行为作法，才能避免与美国发生新冷战。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
