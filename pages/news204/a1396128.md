### 揭秘谷开来海外偷情 遭3杀手跨洋追杀 薄瓜瓜也在场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】7年前重庆发生的一件谋杀案引发的中共官场大地震余波至今未了。英媒曾披露，重庆前市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 之妻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷开来.html" target="_blank">
  谷开来
 </a>
 ，与被其毒杀的英国商人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海伍德.html" target="_blank">
  海伍德
 </a>
 有染，海伍德因此曾遭3名来自中国的杀手追杀，当时薄瓜瓜也在场。
 <br/>
 <br/>
 据英国《每日镜报》报导，尼尔‧
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海伍德.html" target="_blank">
  海伍德
 </a>
 （Neil Heywood）贴身保镖赖特披露，海伍德早于90年代末已与薄
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷开来.html" target="_blank">
  谷开来
 </a>
 结识，2001年圣诞节前夕，两人在英国伯恩茅斯同居时，曾有一中国男子伪装成厨师，混入两人爱巢调查〝奸情〞。
 <br/>
 <br/>
 尔后，3名来自中国大陆会武术、身材魁梧的男子，企图冲进他们的住处，但与赖特等保镖狭路相逢。
 <br/>
 <br/>
 赖特称，这3人应是奉命追杀海伍德，打斗中有保镖受伤，而对方也负伤乘汽车逃走。
 <br/>
 <br/>
 薄谷开来及海伍德当时目睹打斗皆十分惊恐，薄谷开来更吓得浑身颤栗。这之后薄谷开来及海伍德关系更为密切，海伍德也支付赖特等人更多酬劳，希望确保生命安全。
 <br/>
 <br/>
 2014年底，英媒又爆出猛料，赖特披露，这3名中国杀手是中共高层对谷开来与海伍德婚外情极为不满，才派人赴英暗杀海伍德，中国杀手冲进谷开来和海伍德的公寓时，薄瓜瓜也在场。
 <br/>
 <br/>
 赖特还说，薄谷开来在此公寓藏巨款，除房门额外上锁，连窗户都要遮光、上栅栏。
 <br/>
 <br/>
 谷开来在2002年返回中国，海伍德则在中国与另一女子王露露结婚。
 <br/>
 <br/>
 <h4>
  薄谷开来的淫乱生活
 </h4>
 <br/>
 除海伍德外，薄谷开来与法国人德维莱尔的关系〝非同一般〞。据《纽约时报》报导，早在2000年，谷开来在英国设立公司，寻找欧洲建筑师参与大连市政建设，当时就认识了德维莱尔，俩人在英国东南海岸城市租用一套公寓，对外都用其地址。
 <br/>
 <br/>
 薄谷开来与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 也有不正当男女关系。2013年8月，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 案件庭审期间，自揭与王立军和薄谷开来的〝三角关系〞。他称，谷开来与王立军〝如胶似漆〞，这才是他真正叛逃的原因，让各界大跌眼镜。
 <br/>
 <br/>
 英媒披露，薄谷开来当初就是在床上向王立军建议，以涉毒案拘捕为名，来击毙海伍德，而当时王立军除顾虑风险太大拒绝后，还暗地里将两人的对话进行录音。
 <br/>
 <br/>
 报导说，2011年11月14日，海伍德被发现死在重庆南山丽景度假酒店，时任重庆公安局长的王立军组织调查，迅速定论海伍德之死是因〝饮酒过量〞，并将尸体未经检验火化。
 <br/>
 <br/>
 两个多月后，此案出现戏剧性转变。2012年2月6日，王立军出逃美国领馆，揭露海伍德是被其上司重庆市委书记薄熙来的妻子谷开来主导毒杀。
 <br/>
 <br/>
 被抓后在庭审中，王立军供认，因海伍德案与薄熙来翻脸，并被薄痛打一拳，因担心自己也被杀才出逃美领馆。
 <br/>
 <br/>
 <h4>
  海伍德卷入活摘罪恶遭灭口
 </h4>
 <br/>
 大纪元、新唐人等中文媒体也曾报导，被杀死的海伍德，因卷入薄熙来夫妇在国际贩卖器官、尸体等事件遭灭口。
 <br/>
 <br/>
 当年薄熙来、薄谷开来、王立军为攀上中共权力顶峰、满足巨大贪欲，都曾紧跟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 迫害法轮功，并活摘〝法轮功学员器官〞。
 <br/>
 <br/>
 薄谷开来罪行还涉及与薄熙来在中国大连从事器官贩卖、活体摘除器官和非法贩卖尸体等。
 <br/>
 <br/>
 据追查国际披露的调查证据显示，海伍德是谷开来盗卖法轮功学员器官、贩卖尸体的海外联系人。他们与哈根斯工厂合作，活摘器官后，先卖器官再卖尸体。这个项目薄家挣了很多钱，海伍德向谷开来索要自己那一份提成，而谷开来担心海伍德泄露了这个惊天黑幕，于是动了杀机。
 <br/>
 <br/>
 海伍德被杀害后，王立军被中纪委多次秘密约谈。薄熙来夫妇开始对王立军戒备，王恐惧像海伍德一样遭薄熙来夫妇灭口，投奔美国领事馆，导致三人先后身陷牢狱。
 <br/>
 <br/>
 2012年8月20日，谷开来被以〝故意杀人罪〞判处死刑缓期执行；9月24日，王立军被以〝徇私枉法、叛逃、滥用职权、受贿罪〞判有期徒刑15年；2013年9月22日，薄熙来被以〝受贿罪、贪污罪、滥用职权罪〞，判处无期徒刑。
 <br/>
 <br/>
 此案犹如多米诺骨牌，引发中共官场大地震一波又一波，令全球目光聚焦中共政局。
 <br/>
 <br/>
