### 护照被废的小因素 不可不知
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】出国的行囊里，比钱包还重要的还有一样东西，就是护照。钱用完了，随处可从提款机上取，也可以随时用手机支付。一旦护照出了问题，行程立即受阻。老话说，在家千日好，出门一时难。所以，了解各国的护照规定，正确地使用护照，能使您的行程愉悦放心。
 <br/>
 <br/>
 <h4>
  护照重要部分不能污损
 </h4>
 <br/>
 善待自己的护照，同时也注意工作人员在查验后是否善待了您的护照。在拿回护照时，检查护照是否无污损。
 <br/>
 <br/>
 曾经有一名大陆游客在日本药妆店买了三只防晒霜，然而，药妆店的职员却在她的护照上订了15个订书针，按了22个免税章。由于签证页被订书针钉住打不开，无法出境。
 <br/>
 <br/>
 同时，护照上有很多敏感的元件，比如，有个人信息那一页，是绝对不能污损的。下面如果有磁条，千万保护好，因为哪怕只裂了一个小口，你的护照也瞬间作废了。
 <br/>
 <br/>
 至于英国和欧盟的电子护照呢，最重要的部分是护照封面最下方的小标识。你的个人信息储存在里面，机场、海关都可以通过扫这个标识来放行。
 <br/>
 <br/>
 除了这些敏感元件，护照装订线也是专用装订线！如果被替换，就涉嫌伪造护照！曾经有一名从南宁搭飞机去越南的乘客，因为护照装订线散了，自己缝了一下，结果被禁止出境。
 <br/>
 <br/>
 <h4>
  代替孩子签名整本
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_护照作废.html" target="_blank">
   护照作废
  </a>
 </h4>
 <br/>
 加拿大护照必须签名，没有签名的护照等同于无效。可是，未成年人的护照，该由谁来签名呢？加拿大蒙特利尔市的一对夫妇，就因为替孩子在护照上签了名，导致护照作废，最后多花了冤枉钱才得以成行。
 <br/>
 <br/>
 蒙特利尔的弗加诺夫妇（Muriel Frenois and Gilbert Delambre），总想找一个机会带着两个养子全家旅游，但是苦于预算有限，总是不能成行。直到他们碰到了去古巴的特价机票，这才梦想成真，皆大欢喜。
 <br/>
 <br/>
 谁知到了出发那天，在登机的时候，航空公司的工作人员告诉他们：你们两个儿子的护照无效，不能登机。工作人员解释，两个儿子护照上的手写签名都是由夫妻两人代签的，这不合法，所以直接导致护照无效。
 <br/>
 <br/>
 原来，在加拿大儿童护照申请表上，写明了11岁及以上的儿童必须自己签名。除了儿童本人，任何情况下都不可由他人代替儿童在签名栏内签名。11岁以下的儿童可以不用签名，等到他们满11岁之后，再自己签上去。
 <br/>
 <br/>
 两个儿子的护照申请表，是弗加诺夫妇很早以前填写的，可是当时并没有注意到这一条。他们只是想着孩子小，随手就替他们签了。
 <br/>
 <br/>
 结果一家四口被困登机口，叫天天不应，叫地地不灵。航空公司的职员虽然可怜他们，也无法放行。不过天无绝人之路，最后还是在航空公司职员的指点下，拨打了政府热线，花钱申请了两份紧急护照。弗加诺夫妇不忍心扫孩子们的兴，又重新定了去古巴的机票，不过这次可没有廉价机票可以买啦。
 <br/>
 <br/>
 大陆的未成年人的护照签名也是有类似规定的，只不过年龄比加拿大小了一岁。大陆规定，未满10周岁的申请人护照不要求签字，如签名，务必由申请人本人签名。10周岁以上(包括10周岁)的申请人护照必须由本人签名；任何情况下都不允许未成年申请者的父母/法定监护人代替其在护照签名页签名。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
