### 国内捧习不够？胡鞍钢哈佛名校线上开课〝习思想〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】近期〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_捧习运动.html" target="_blank">
  捧习运动
 </a>
 〞再度恢复热度，这股吹捧风甚至还走出国门。近日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清华大学.html" target="_blank">
  清华大学
 </a>
 国情研究中心主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡鞍钢.html" target="_blank">
  胡鞍钢
 </a>
 ，在美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈佛.html" target="_blank">
  哈佛
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麻省理工.html" target="_blank">
  麻省理工
 </a>
 合办的线上平台开课教授〝习思想〞课程，对海外人士进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗脑.html" target="_blank">
  洗脑
 </a>
 。
 <br/>
 <br/>
 在美中贸易战开打之初，〝梁家河热〞、〝习思想〞等〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_捧习运动.html" target="_blank">
  捧习运动
 </a>
 〞一度降温，但随着贸易战日渐激烈，〝捧习运动〞再度升温。近期，中共当局强力宣扬
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平思想.html" target="_blank">
  习近平思想
 </a>
 ，不但侵入中国社会各个角落，甚至还走出中国国门。
 <br/>
 <br/>
 日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清华大学.html" target="_blank">
  清华大学
 </a>
 国情研究中心主任、公共管理学院教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡鞍钢.html" target="_blank">
  胡鞍钢
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_edX平台.html" target="_blank">
  edX平台
 </a>
 上开课教授〝习近平思想〞，课程语言为中文，但配备英文字幕、讲解及习题。
 <br/>
 <br/>
 据悉，edX是美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈佛.html" target="_blank">
  哈佛
 </a>
 大学和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麻省理工.html" target="_blank">
  麻省理工
 </a>
 学院合办的线上课程平台。
 <br/>
 <br/>
 美国《外交政策》杂志记者18日刊文讲述自己学习〝习近平思想〞这门课程的经过，他说自己以满分100分的成绩通过了考试，并在支付49美金证书费后，获颁具有胡鞍钢签名的电子证书。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-21/p9089751a313062245.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-21/p9089751a313062245-ss.jpg"/>
   <br/>
   <font size="-1">
    该记者称，胡鞍钢在edX线上平台教授〝习思想〞课程，考试合格后，支付49美金，可获颁具有胡鞍钢签名的电子证书。（网络截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 课程资料显示，这门课程故意误导学生，用习思想给学生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗脑.html" target="_blank">
  洗脑
 </a>
 ，让学生系统且充分地认识到中共所谓的以〝人民〞为核心，并深刻理解中共特色社会主义道路。课程共8周，学生获得60分即合格，85分则为优异。
 <br/>
 <br/>
 该名记者说，课程存在英文翻译字幕不完整或不通顺的问题。课程习题在考试日期两周前仍未发布。他向平台反应后，平台发布了包含34道多项选择题的课程作业，记者以满分作答通过课程。
 <br/>
 <br/>
 胡鞍钢是清华大学国情研究院院长，曾在一个报告中吹嘘说，中国经济实力在2013年已是美国的1.15倍，中国的科技实力在2015年已是美国的1.31倍，中国综合国力在2012年已是美国的1.36倍。
 <br/>
 <br/>
 但在中美贸易战开打后，中国经济受到严重影响，股市跌至4年来新低，各种危机显现，中共陷入严重的外患内忧困境，胡鞍钢的言论，因此遭到中国学界的抨击。
 <br/>
 <br/>
 8月上旬，一批清华校友联署上书清华大学校长，批评胡的超美论〝上误国家决策、下误黎民百姓，远引他国戒心〞，要求校方将其解聘。
 <br/>
 <br/>
 专家认为，胡鞍钢是中共体制造就和孕育的产物，其人以宣传洗脑为己任，以维护中共统治，但随着国民了解真相的渠道越来越多，中共的传统洗脑术已越来越没有市场。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：祝馨睿）
