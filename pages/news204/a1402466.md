### 加拿大外长：逮捕孟晚舟时就已通报中共外交部
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月09日讯】美中已达成共识，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 CFO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被捕事件不会影响美中谈判，显示北京已准备抛弃孟晚舟之意。最新消息更印证了北京态度。加拿大官方透露，12月1日逮捕孟时已通知中方。但直到6日消息曝光前，中方从未就此发声。
 <br/>
 <br/>
 加拿大媒体12月7日编译CTV最新报导指，加拿大外交部长方慧兰（Chrystia Freeland）当天表示，加方应美方要求在温哥华逮捕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 时，就已通过加拿大驻北京大使麦家廉(John McCallum)将情况通报给中共外交部。
 <br/>
 <br/>
 报导称，身在德国柏林的外长方慧兰在电话会议中表示，抓捕孟晚舟不涉及政治，加方已经向中方保证，将严格遵守法律程序处理此事，并遵循恰当的外交程序，为当事人提供领事通道。
 <br/>
 <br/>
 上述消息从侧面印证了此前《金融时报》的一则相关报导。这家英媒指，12月1晚的美中首脑会晤前，北京最高领导人已经获悉孟晚舟被抓的消息，但是在首脑会晤晚宴上没有提，原因是北京〝要集中精力处理美中贸易争端〞。
 <br/>
 <br/>
 多方评论指，美中贸易战令中共陷入困境，急于和美方〝求和〞的北京，根本无暇顾及孟晚舟。
 <br/>
 <br/>
 事实上，孟晚舟被抓后的5天内，中共官方从未公开此事。直到12月6日加方公布孟晚舟保释听证会将举行的消息后，中方才象征性的发出几声〝抗议〞。
 <br/>
 <br/>
 虽然海内外党媒不断炒作中共将针对美加企业家〝展开报复〞，但中共官方至今没有实施、甚至没有提及任何反击行动。
 <br/>
 <br/>
 相反，美国官方公布消息指，美中已达成共识，孟晚舟事件不会影响双方贸易谈判。
 <br/>
 <br/>
 网友评论称，为中共效力的人，必须随时做好〝为中共党国利益做出牺牲〞的准备。
 <br/>
 <br/>
 根据美国法律，如果孟晚舟被引渡到美国受审，将面临最高30年刑期。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 被指中共情报机构。网友披露，孟晚舟拥有多国护照，符合中共情报人员的特征。
 <br/>
 <br/>
 华为具有中共军方背景，多年来帮助中共在海外从事网络间谍活动，为美国制裁的〝流氓国家〞提供技术支持，同时也是中共对外输出网络垄断的龙头企业。
 <br/>
 <br/>
 美国共和党联邦参议员克鲁兹（Ted Cruz）6日在推特（Twitter）上表示，〝华为是蒙上一层电信公司薄薄面纱的中共间谍机构，它的监视网络跨越全球，它的客户是诸如伊朗、叙利亚、朝鲜和古巴这样的流氓政权。〞
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：）
