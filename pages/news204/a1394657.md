### 山东老兵维权遭暴力清场 官方贬称〝兵痞〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】中国上千名退伍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 连续3天聚集在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东平度.html" target="_blank">
  山东平度
 </a>
 展开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 抗争，7日遭当地政府出动数千名警察暴力清场，5、600名老兵被抓后遣返回家，不少老兵被打伤。当地官方贬低老兵为〝兵痞〞，严禁民众给予他们帮助。
 <br/>
 <br/>
 有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东平度.html" target="_blank">
  山东平度
 </a>
 上千名退伍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 展开的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 抗争，中共媒体没有任何报导，10月8日，自由亚洲说，平度市民政局复退军人服务中心主任曲林洲的微信圈显示，山东方面已将前往平度的老兵们定性为兵痞。
 <br/>
 <br/>
 山东方面要求当地民众不得为这些老兵提供吃住行的方便，并称他们是不受欢迎的人，要他们限期离开。
 <br/>
 <br/>
 据媒体群消息显示，平度市委书记的秘书，也在其微信朋友圈发布了内容相同的帖子，当地媒体人透露，该定性是山东高层的态度。
 <br/>
 <br/>
 参与维权的老兵赵先生对自由亚洲说，尽管官方将老兵们称为兵痞，并试图挑拨市民和老兵们的关系，但当地市民还是很同情老兵们，为他们送水和食物。
 <br/>
 <br/>
 赵先生认为，今年多次的老兵集体维权，看似都失败了，但是他们能承受得起所有失败，只要有一次胜利就够了，但官方只要有一次失败，他们就会彻底完了。
 <br/>
 <br/>
 此次老兵维权事件的导火索是10月4日，山东青岛平度市38名退伍军人，打算乘车进京上访，还没有走出山东省，就被当地政府人员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拦截.html" target="_blank">
  拦截
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_殴打.html" target="_blank">
  殴打
 </a>
 。
 <br/>
 <br/>
 5日早上，数百名来自山东其他地方及江苏、四川等省的退伍军人赶来平度声援。当局派出大批特警，双方一度推撞。当晚，各地老兵不断赶往当地，至6日凌晨已超过千人。
 <br/>
 <br/>
 6日下午，当地警方暴力清场，双方激战。7日，老兵第2次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集结.html" target="_blank">
  集结
 </a>
 在农业技术市场的广场。
 <br/>
 <br/>
 青岛维权老兵李先生说，7日早上他与其他战友再次来到广场集合时，当局也调动了大批警察，在现场围堵他们。上午，老兵们决定在市区游行。当地政府增派大批防暴警察、特警到现场将老兵包围起来，警察人数达到2、3000。冲突过程中又有7、8名老兵受伤。
 <br/>
 <br/>
 河南老兵陆先生说，特警、防暴警察拿盾牌、胶棍围堵他们，后来警察又拿辣椒水喷他们。当时场面比较混乱，最后，老兵们寡不敌众，5、600名老兵被送到西关小学校关押起来。
 <br/>
 <br/>
 8日早上，在山东平度市农业市场及医院外的两个维权现场，已经没有老兵聚集，但大批的员警仍在现场及周边戒备。平度周边的高速公路出口依然被封锁，整个县戒严了。
 <br/>
 <br/>
 当地市民李先生透露，员警还在继续搜查各地赶来的老兵。
 <br/>
 <br/>
 梁先生说，7日下午清场后，现场的最新消息已中断，当晚，很多老兵都已被强制遣返。因为缺乏后援及欠缺完善部署，这次老兵平度维权彻底已经失败。
 <br/>
 <br/>
 但老兵们表示，他们将继续讨说法，不给一个交代，维权不止。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：赵云）
