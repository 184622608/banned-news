### 港商刘希泳遭酷刑逼供致死 9检察官获刑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】中共央视女主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘芳菲.html" target="_blank">
  刘芳菲
 </a>
 的丈夫、港商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘希泳.html" target="_blank">
  刘希泳
 </a>
 被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酷刑逼供致死.html" target="_blank">
  酷刑逼供致死
 </a>
 案日前宣判，审讯小组9名中共检察官被判刑。据网上流传的一份法庭文件显示，刘希泳在去年3月被审讯时，被剥夺睡眠和遭受酷刑80小时后死亡。
 <br/>
 <br/>
 《南华早报》报导，天津市第一中级法院10月23日作出判决，被告首席审讯人许学哲被判入狱15年，被告周庆军被判入狱11年，被告调查组组长赵伯忠因渎职罪被判处有期徒刑4年，还有6人被判处15个月至3年有期徒刑。
 <br/>
 <br/>
 英国《金融时报》的报导有一点不同，报导称许学哲是以故意伤害罪被判入狱13年。一名代理人表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘希泳.html" target="_blank">
  刘希泳
 </a>
 家属反对法院的判决、希望以谋杀罪定罪。而中共官媒未对此案进行报导。
 <br/>
 <br/>
 此前有中国媒体人透露，中共有关当局指令对刘希泳致死案不报导、不转载、不评论。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘芳菲.html" target="_blank">
  刘芳菲
 </a>
 本人9月7日晚在微博发布的有关庭审的微博，也遭新浪微博删除。
 <br/>
 <br/>
 据悉，9名被告中共检察官全部是来自吉林延边，在刘希泳死后不久，这9人被拘留，之后遭到起诉。根据网上流传的起诉书显示，刘希泳在被剥夺睡眠和遭受酷刑80小时后，于2017年3月死亡。
 <br/>
 <br/>
 起诉书中包括刘希泳遭受酷刑的一些骇人听闻的细节，当局称，2017年3月，在靠近中朝边境的一处审讯室内，刘希泳被蒙上双眼，审讯小组将他绑在一把椅子上，并将其双腿捆绑，架在前方椅背上。
 <br/>
 <br/>
 起诉书称，审讯小组对刘希泳采用蒙眼、胶带封口、马桶塞捅口鼻、锁匙刺脚心等酷刑。审讯到第五天的时候，即2017年3月19日午夜，审讯者多次将刘希泳的上身〝折叠〞到他的腿上，直到他失去知觉，驻地医护人员未能救活刘希泳。
 <br/>
 <br/>
 香港《星岛日报》此前报导称，中共官方在对刘希泳进行尸检时发现，刘希泳有七个地方骨折，包括胸骨和肋骨，属窒息死亡。
 <br/>
 <br/>
 而起诉书表示，验尸报告说，刘希泳全身有多处骨折，因口鼻被封等多种因素导致机械性窒息死亡。9个中共检察官事后企图毁坏审讯室内的录音设备。此案引起了外界对中共警方在审讯中所使用的残酷手段的关注。
 <br/>
 <br/>
 刘希泳是香港尖沙咀君怡酒店老板，2016年9月，刘希泳从香港飞往北京，在机场失踪，他的家人四处寻找。2017年3月，中共当局对他的家人通报了死讯。今年9月7日，天津市第一中级法院开庭审理这起案件。
 <br/>
 <br/>
 9月9日，法广引述知情人士的观点称：刘希泳有三重保护层都未能够保护他免予被刑讯致死。第一，他是香港公民；第二，他是富豪；第三，妻子刘芳菲是著名中共央视花旦。
 <br/>
 <br/>
 刘希泳是70年代末期中国第一批留美学生，哈佛大学毕业后到香港经商，一度成为拥有几十亿资产的富豪。1990年代末，刘希泳因涉原光大集团董事长朱小华案而被中纪委〝双规〞，并被关押过一段时间。朱小华因受贿罪于2002年10月被判处15年有期徒刑。
 <br/>
 <br/>
 刘希泳在重获自由后，便与中共央视美女主持人刘芳菲相识并结婚，刘芳菲曾被称作中共央视内部〝台花〞。
 <br/>
 <br/>
 2016年11月刘希泳被指涉嫌诈骗工银亚洲2亿多港币贷款，因未到庭应讯，遭香港法庭通缉令。随后刘希泳一直失踪，直至去年4月传出离奇死亡消息。
 <br/>
 <br/>
 法广曾援引中共高层知情人的话披露，刘希泳被查的主要原因是涉及替高层洗钱，即俗称的〝白手套〞。刘希泳在审讯期死亡非常蹊跷，外界盛传他的死亡与中共政法系统的腐败有关，更被称为是〝海伍德死亡案〞的翻版。
 <br/>
 <br/>
 另据港媒披露，刘希泳的背后，涉及中共前政治局常委李岚清、刘云山；前最高检察院检察长贾春旺等江派大佬，以及最高检察长等一大批政法系统的高官。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
