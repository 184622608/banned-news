### 【禁闻】多维网刊文批习 再紧急补镬
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】
 <b>
  11月份 31名
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
   法轮功学员
  </a>
  被
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法判刑.html" target="_blank">
   非法判刑
  </a>
 </b>
 <br/>
 <br/>
 中共对法轮功的迫害依然严酷，据〝法轮大法明慧网〞报导，在刚刚过去的11月份，中共至少又对31名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法判刑.html" target="_blank">
  非法判刑
 </a>
 、对48名法轮功学员非法庭审，其中，被非法判刑年龄最大的学员80岁，被非法庭审学员年龄最大的89岁。
 <br/>
 <br/>
 非法判刑、庭审涉及18个省、自治区、直辖市。其中判刑最严重地区为：吉林省，共9人。庭审最严重地区是河北，共12人。
 <br/>
 <br/>
 报导说，这一个月内，中共法庭和警察还敲诈勒索了法轮功学员184000元现金。
 <br/>
 <br/>
 〝明慧网〞说，为法轮功学员辩护的律师们强调，信仰无罪，法庭应该立即无罪释放这些学员。
 <br/>
 <br/>
 <b>
  多维网刊文批习 再紧急补镬
 </b>
 <br/>
 <br/>
 总部设在北京的所谓海外华文媒体〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_多维新闻.html" target="_blank">
  多维新闻
 </a>
 〞，12月3号在旗下的《多维月刊》最新一期中刊载一篇题为《极左撕裂中国，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 应负起责任》，指称习近平作为党核心，〝必须进行自我检讨〞。
 <br/>
 <br/>
 由于〝多维新闻〞被认为是中共国安派系的外宣媒体，因此文章引发中共内部发起倒习运动的测猜。
 <br/>
 <br/>
 但不久文章又被从网上撤下，重新上载之后，内容已经被修改，大量对习近平的批评质疑句子全部消失。标题也变成了〝防止极左撕裂中国 习近平强势拨乱反正〞。
 <br/>
 <br/>
 12月4号，多维新闻又专门发表一篇文章解释前文，并批评海外媒体与民主人士对文章作了恶意曲解，说华人世界总有那么一群党〝叫不醒〞的人。
 <br/>
 <br/>
 中共大外宣媒体的这番表现，在推特华人推友中引发议论。
 <br/>
 <br/>
 旅美学者何清琏说：〝看来，多维北京总部一定被有关方面约谈了。〞她呼吁大家将有关消息存档，以观后续。
 <br/>
 <br/>
 编辑/周玉林
