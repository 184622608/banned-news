### 不只是大豆 路透：中方被迫重新进口美国天然气
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】为了报复美国政府对中国商品加征关税，中共政府一个月前宣布对美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_液化天然气.html" target="_blank">
  液化天然气
 </a>
 征税10%，此后近一个月内没有中国企业购买美国液化天然气。然而近日有消息指中企又被迫重新进口美国天然气。舆论认为可能是冬季来临，中国国内的天然气供应不足所致。有中国网友感叹：中共搬起的石头怎么经常砸了自己的脚？
 <br/>
 <br/>
 据路透社24日报导，有美国公司近日向中国运出了一批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_液化天然气.html" target="_blank">
  液化天然气
 </a>
 。这是今年9月底以来，美国企业向中国大陆输送的首批液化天然气。
 <br/>
 <br/>
 该报导援引Refinitiv Eikon数据称，这批天然气来自Cheniere Energy 的Sabine Pass 码头，预计将于11月11日运抵中国。
 <br/>
 <br/>
 有分析称，当前全球对液化天然气的需求非常高，而现在冬季来临，中国北方需要大量的天然气来供暖，国内天然气供应不足，可能是迫使中国企业不得不重新向美国购买天然气的主要原因。
 <br/>
 <br/>
 就在不久前，美国能源部副部长布鲁耶特（Dan Brouillette）出席在日本举办的国际液化天然气会议时对媒体表示，现在印度对液态天然气的需求正在上涨，欧洲也开始寻求更多元的能源来源，再加上日本等其他亚洲国家的经济增长，目前全球对液化天然气的需求非常高。所以，中国企业因为关税问题而减少进口美国液化天然气〝不足为虑〞。
 <br/>
 <br/>
 今年9月18日,为报复美国对2,000亿美元中国商品加征关税的政策，中共政府宣布将从9月24日起对600亿美元美国商品加征报复性关税，其中就包括对美国液化天然气征税10%。
 <br/>
 <br/>
 而早在于美国政府的第一轮关税战开打之初，中共政府为了打击支持美国总统川普的农业大州，曾于7月初宣布对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 征收25%关税，但这项举措产生的实际效应并不如中共政府所期望的那么好。
 <br/>
 <br/>
 就在10月中旬，由于巴西大豆供应不足，已有中国企业又不得不开始进口美国大豆以满足中国国内庞大的市场需求。
 <br/>
 <br/>
 据彭博社报导，今年10月11日前后，有两艘货船在美国装满了大豆后驶向中国。其中，〝劳拉星号〞（Star Laura）货轮装载着大约6.5万吨美国大豆驶离墨西哥湾的港口运；而〝金色皇后号〞货轮转载了大约6.9万吨美国大豆驶离西北太平洋的港口。
 <br/>
 <br/>
 当时就有中国大陆网友在微博留言发帖议论说，〝这样不得不买的话，关税就落在自己人的头上了。〞
 <br/>
 <br/>
 还有中国网友感慨说：〝难道我们的石头都没了？怎么美帝没有砸脚啊？〞
 <br/>
 <br/>
 有分析认为，由于贸易战的原因，中国2018年的大豆缺口将达1,000万吨左右，而中国大豆企业从巴西购买大豆，不但价格比美国大豆更高，而且运输距离更远，企业的成本将因此加大，这可能是中国进口商重新进口美国大豆的原因之一。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：明轩）
