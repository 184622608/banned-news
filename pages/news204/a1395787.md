### 【禁闻】中宣部见港媒高层  中联办令删改报导
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】
 <b>
  中宣部见
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港媒.html" target="_blank">
   港媒
  </a>
  高层
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
   中联办
  </a>
  令删改报导
 </b>
 <br/>
 <br/>
 约20名香
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港媒.html" target="_blank">
  港媒
 </a>
 体高层10月16号访问北京，与中共中宣部长黄坤明会面。访京团团长、星岛集团行政总裁萧世和在会后对媒体说，黄坤明在会上说〝希望香港媒体不要成为干扰内地政治的基地〞。
 <br/>
 <br/>
 消息报导出来不久，又被香港媒体集体删除，引发了外界对香港新闻界自我审查的隐忧。香港记者协会对事件表示关注，认为〝极不寻常〞。
 <br/>
 <br/>
 随后萧世和向《南华早报》否认黄坤明有相关说法。
 <br/>
 <br/>
 但〝自由亚洲电台〞报导说，一位在北京参与会面的传媒高层透露说，黄坤明在会上确实说过那句话，不过，事后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 官员告知，这段发言只属内部讲话，不能报导，因此多名随团的港媒高层下达指令，要求重新处理或删除相关内容。
 <br/>
 <br/>
 <b>
  大陆防疫专家：
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
   非洲猪瘟
  </a>
  疫情难以遏止
 </b>
 <br/>
 <br/>
 非洲猪瘟疫情正在大陆全范围蔓延，中共卫生部门10月12号启动一级应急预案之后，辽宁省的鞍山、锦州、铁岭，继续出现案例。
 <br/>
 <br/>
 据报导，目前大陆非洲猪瘟疫情已达29起，而且还有进一步失控的迹象。
 <br/>
 <br/>
 一位参与防控工作的大陆专家接受〝自由亚洲电台〞采访时表示，无论是管理能力还是生物技术的掌握，中国都无力阻止非洲猪瘟疫情扩散。
 <br/>
 <br/>
 中共官方此前多次发布消息，声称猪肉可以吃，说只要经过80度高温，30分钟就可以杀死病毒。但这名徐姓专家表示，虽然目前还没有病毒传染到人的报告，但这并不代表猪肉就很安全，和禽流感病毒一样，非洲猪瘟的病毒也有变异的可能。
 <br/>
 <br/>
 <b>
  美使馆
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监测空污.html" target="_blank">
   监测空污
  </a>
  10周年 网民齐道谢
 </b>
 <br/>
 <br/>
 10月16号是美国驻华大使馆空气监测项目十周年纪念日，美使馆官方微博当天发文说，驻华大使布兰斯塔德（Terry Branstad）当天在馆内主持新空气质量监测器启用仪式，并发表讲话。
 <br/>
 <br/>
 贴文一发出，大陆网民蜂拥而至，纷纷跟帖表示感谢。
 <br/>
 <br/>
 有人说：〝感谢美帝让屁民知道还有pm2.5这个东东。〞
 <br/>
 <br/>
 也有人讽刺中共说：〝记得刚公开数据时被渲染成〝干涉别国内政〞〞、〝天天骂的人却救了无数人的生命，天天夸的人却处心积虑致无数人死命！〞
 <br/>
 <br/>
 还有人请求美国说：〝千万别放弃我们。〞
