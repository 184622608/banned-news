### 华为遭德国封杀？德5G招标拒绝中企
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】中国电信设备制造商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 因具有中共军方背景，在海外屡屡被封杀。日前，英媒报导，德国高级官员正在说服政府，在修建5G通讯基础设施时，将华为等中国企业排除在外。
 <br/>
 <br/>
 路透社11月14日报导，德国预计将在2019年初开始5G（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第五代移动通信.html" target="_blank">
  第五代移动通信
 </a>
 系统）招标，德国官员希望将中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 等电信公司排除在招标范围之外。因为华为具有中共军方背景，被指存在情报安全隐患问题，他们担心可能威胁到德国国家安全。
 <br/>
 <br/>
 有报导指，华为创始人任正非（Ren Zhengfei）曾是中共军队的工程师，1978年加入共产党，十年后开始在华为工作。近年来，中共积极投资于5G的技术研发，中兴、华为是中共发展5G押注的对象。外界担忧，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共情报部门.html" target="_blank">
  中共情报部门
 </a>
 可以直接介入这些公司的网路运作。
 <br/>
 <br/>
 要求匿名的德国官员表示，目前还不清楚上述建议能否取得成功。但这已显示出，德国政府部门对中共在德国建设
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5G移动网路.html" target="_blank">
  5G移动网路
 </a>
 中所扮演的角色，感到担忧。
 <br/>
 <br/>
 〝如果由我来决定，我们会做澳洲方面正在做的事情，〞一位在柏林参与5G内部讨论的德国高官表示。
 <br/>
 <br/>
 美国和澳洲已经先后禁止华为参与5G网络建设，因为担心华为、中兴等中国公司的产品被中共利用来进行监控，导致机密情报泄露和遭到网路攻击。
 <br/>
 <br/>
 负责5G建设的德国外交部和内政部官员表示，他们已经与美国和澳洲相关部门的官员进行了会谈，他们也对使用华为等中国供应商的风险感到担忧。
 <br/>
 <br/>
 这位高官表示，他们正在敦促在招标程序开始前，就5G进行更加严肃的讨论。这可能导致此事被延后。
 <br/>
 <br/>
 〝我们需要能够审查个别情况，以确保我们的关键基础设施得到保护。这可能导致中国公司被排除在我们的5G基建建设之外。〞德国绿党议员Katharina Droege表示。
 <br/>
 <br/>
 美国是第一个禁止华为等中国供应商参与5G建设的国家。美国数位民主党和共和党参议员提出了立法草案，促使政府禁止购买中兴通讯和华为的产品和服务。
 <br/>
 <br/>
 〝华为和中兴对美国的国家安全构成严重威胁，这些公司与中共政府和共产党有直接联系，〞共和党参议员卢比奥说。
 <br/>
 <br/>
 他说：〝它们的产品和服务被用于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍活动.html" target="_blank">
  间谍活动
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权盗窃.html" target="_blank">
  知识产权盗窃
 </a>
 ，并且它们一直把美国人民和经济置于风险之中。〞
 <br/>
 <br/>
 早在2011年10月的一份公开调查报告中，美国中央情报局（CIA）就曾披露，在过去三年里，华为从中共政府那里拿到近2.5亿美金的资助，为中共提供情报服务。
 <br/>
 <br/>
 报告中说，中国最大的电信公司华为，是类似于前苏联克格勃式的情报部门。
 <br/>
 <br/>
 有专家呼吁，应对5G与物联网时代到来，美国必须从国家安全的角度制定统一战略。
 <br/>
 <br/>
 美国情报部门也警告澳大利亚，如果由华为负责筹建澳洲5G行动网路，可能使华为得以掌控澳洲下一代行动网路，并可利用它来从事间谍活动，引起国家安全隐忧。澳大利亚于今年8月份宣布，禁止华为参与5G网络建设。
 <br/>
 <br/>
 继美国和澳洲封杀华为之后，日本、俄罗斯、韩国、印度等国先后表示，将把华为等中国电信公司排除在5G建设之外。
 <br/>
 <br/>
 日本《产经新闻》8月26日报导，据不具名消息人士透露，日本政府决定在引进政府层面的情报系统时，排除中国华为或中兴的电信设备。
 <br/>
 <br/>
 俄罗斯《生意人报》也报导称，俄罗斯多家电子设备生产商和行业协会，已经向政府提议，从严规定对华为和中兴电信设备的进口，俄罗斯也正在考虑这条建议。
 <br/>
 <br/>
 9月，《印度经济时报》报导称，印度电信部已经排除中国的华为和中兴两家公司参与印度的5G网络试验与合作。印度电信部长阿鲁纳．桑德拉加表示，这是出于安全的考虑。
 <br/>
 <br/>
 《韩国时报》也报导称，韩国第一大行动电信商SK Telecom9月15日宣布，已选择三星电子（Samsung）、爱立信（Ericsson）和诺基亚（Nokia）作为5G设备的首选投标企业，华为被排除在外。
 <br/>
 <br/>
 11月，《华尔街日报》报导，英国政府部门正在审查该国的电信供应链的安全性，中国5G供应商华为公司或将受到英政府的严厉审查。
 <br/>
 <br/>
 路透社分析认为，各国对华为的封杀，不仅对华为是一个重大打击，并且还削弱了中共的〝中国制造2025〞议程。
 <br/>
 <br/>
 通信业界有关人士称：〝大多数国家或通信企业碍于中国政府的面子，无法像美国一样公开表示‘排除华为’的立场，但随着安全问题不断凸显，引进华为设备势必将更加慎重〞，〝5G时代来临在即，通信设备业界发生地壳变动的可能性极大。〞
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：文慧)
