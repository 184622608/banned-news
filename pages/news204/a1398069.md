### 美严打中共间谍 华男走私潜艇设备或被囚70年
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】美国正在加大对中共间谍的打击力度。日前，美国司法部对一名涉嫌向中共军队
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法走私潜艇设备.html" target="_blank">
  非法走私潜艇设备
 </a>
 和技术的华裔男子，追加起诉4项罪名。该男子今年6月已被警方逮捕，若其数项罪名成立，将面临最高70年刑期和250万美元罚金（约人民币1700多万元）。
 <br/>
 <br/>
 当地时间11月2日，美国司法部在官网发布新闻稿，中国籍男子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦树仁.html" target="_blank">
  秦树仁
 </a>
 （Shuren Qin，音译）在10月30日被起诉，涉嫌串谋欺诈美国，走私，洗钱，以及向政府官员做虚假陈述等罪名。这是对他的第二轮指控。
 <br/>
 <br/>
 秦树仁被抓前住在马萨诸塞州韦尔斯利。今年6月，他被指控提供欺诈性文件，将美国海上设备出售给中共军方，被美国警方逮捕并拘留。
 <br/>
 <br/>
 美国麻州检察官办公室表示，若秦树仁数项罪名成立，最高可判70年有期徒刑，连带处罚250万美元罚金。
 <br/>
 <br/>
 秦树仁于2014年成为美国合法永久居民，他在中国大陆经营海运业务。秦树仁是〝海洋链技术公司〞（LinkOcean Technologies）的总裁，该公司专门从事〝低价进口和海运相关的商品和技术〞，并与中共军方有直接往来。
 <br/>
 <br/>
 秦树仁还接受中共军方的指派，对西北工业大学军事研究所（NWPU）输出美国反潜战军备，该大学为中共军用研究机构，2001年就被美国商务部列入拒绝往来清单。
 <br/>
 <br/>
 起诉书写道，2016年12月至2017年7月，秦树仁在未获得商务部颁发的出口许可证情况下，向NWPU秘密出口用于探测、监测水下声音的〝水听器〞，数量至少60个。
 <br/>
 <br/>
 秦树仁对美国水听器制造商隐瞒NWPU是最终用户，并向美国政府提交虚假的最终用户信息，以便顺利出口。此外，他还从中国银行帐户转帐超过10万美元（约人民币69万元）到美国银行洗钱，让非法出口计划得以有资金进行。
 <br/>
 <br/>
 起诉书说，秦树仁于2015年11月向中共军队（PLA）出口了美国制造的远程操作侧扫声纳系统。在2017年11月与海关和边境保护局（CBP）官员的访谈中，秦谎称，他只出口附在浮标上的仪器。但他实际出口的产品，包括远程操作侧扫声纳系统、无人水下航行器、无人水面舰艇、机器人船和水听器等。
 <br/>
 <br/>
 起诉书还提到，2014年，秦树仁涉嫌透过诈欺的方式骗取美国居留资格。他在申请绿卡时谎称自己从未〝从事出口管制违规或其他非法活动〞。但事实上，他于2012年就有此违法行为，并在数次访谈中对政府官员隐瞒此讯息，绝口不提出口行为与中共军方有关。
 <br/>
 <br/>
 <h4>
  美国加紧严惩中共间谍
 </h4>
 <br/>
 值得一提的是，秦树仁案并不是个案，最近一段时间，美国政府正在加紧惩办外国通过间谍、代理人及黑客入侵窃取和走私商业、军事机密或受管制产品的行为。
 <br/>
 <br/>
 11月1日，美国司法部长塞申斯（Jeff Sessions）宣布成立专案小组，打击中共经济间谍活动，以保护美国公司知识产权。塞申斯表示，美国将运用所有司法资源，以利〝有效且快速地起诉中共经济间谍〞。
 <br/>
 <br/>
 塞申斯称，自从川普（特朗普）总统去年1月上任以来，联邦政府积极应对来自国外政府的精心设计且有系统的经济间谍行动，特别是〝中国共产党所做的恶名昭彰的窃盗知识产权行为〞。
 <br/>
 <br/>
 在此之前，美国司法部10月30日已起诉中共情报及黑客团队十人，包括两名江苏省国安厅情报人员。他们被指控攻击美国和欧洲的国防和航空航天承包商，以窃取商业秘密，开发中国的商用飞机涡扇发动机。
 <br/>
 <br/>
 9月下旬，美方起诉一名在美国陆军预备役服役的中国公民纪超群（音译Ji Chaoqun），原因是他为江苏国安厅情报人员招募工程师和科学家。
 <br/>
 <br/>
 8月，美国起诉56岁华裔男子郑孝清（Xiaoqing Zheng），指控他涉嫌盗取通用电气公司涡轮技术的机密电子档案，泄露给中国公司。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：祝馨睿）
