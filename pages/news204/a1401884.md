### 李克强谈话太敏感？ 党媒报导推迟数日
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】日前，中共国务院总理李克强在南京召开经济形势座谈会。李克强在会上表示，中国经济下滑压力加大，北京当局正面临国内外各种挑战。值得一提的是，该会谈报导被推迟数天，直到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 召开之后才发表，引发外界诸多联想。
 <br/>
 <br/>
 11月30日，李克强赴江苏南京召开部分省（区）政府主要负责人经济形势座谈会，就当前经济形势和明年发展听取意见建议。
 <br/>
 <br/>
 在会谈中，江苏省委书记娄勤俭、省长吴政隆、黑龙江省长王文涛、河南省长陈润儿、湖北省长王晓东、广东省长马兴瑞、广西壮族自治区政府主席陈武、四川省长尹力等，就本省经济发展情况，下一步工作打算，及对国家政策的建议发了言。
 <br/>
 <br/>
 此次经济会谈时间是在G20〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞前夕，故中共官媒对会谈的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_报导推迟.html" target="_blank">
  报导推迟
 </a>
 了3天，直到12月3日才见报。
 <br/>
 <br/>
 香港《明报》12月4日报导称，这个座谈会召开时，川习会虽未举行，但中美双方的谈判进展，李克强不可能不知道。而且，他对于川习会的可能结果，应该也有所掌握。李克强的谈话内容或涉及川习会敏感议题，因此，中共官媒才推迟报导。
 <br/>
 <br/>
 但也有分析认为，或许是李克强的谈话内容太过敏感，报导才被推迟。
 <br/>
 <br/>
 据报导，李克强在会谈中说，当前国内外形势错综复杂，中国经济下滑压力加大，地区走势分化，各级政府〝对困难和矛盾要有充分估计〞，必须把形势和问题分析透。
 <br/>
 <br/>
 他还强调，各级政府要把〝就业优先〞放在突出位置，稳定社会发展。李克强此言暗示
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 已使北京早前提出的六稳之一〝稳就业〞，呈现不稳态势。
 <br/>
 <br/>
 据中共官媒报导，近期有700多万农民工〝返乡创业〞。但外界认为，这其实是经济下滑导致的大量农民工失业，被迫返乡。
 <br/>
 <br/>
 近日，大陆网路上几篇有关〝裁员潮〞、〝202万条招聘广告消失〞、〝就业进入寒冬〞等相关报导也被当局封杀，官方此举从侧面佐证了大陆目前的就业形势相当严峻。
 <br/>
 <br/>
 李克强还表示，今年以来，国际、国内各种挑战和不确定因素明显增多，经济在应对风险挑战中要〝保持平稳发展〞。
 <br/>
 <br/>
 他强调，各地政府要聚焦支持制造业、服务业，尤其是小微企业、民营企业等实体经济，广泛听取各方面意见，实施更大力度减税降费。
 <br/>
 <br/>
 民营经济占据中国经济的半壁江山，曾帮助化解减员增效的国企改革面对的重大挑战。
 <br/>
 <br/>
 据数据显示，截止2017年底，民营企业贡献了50%以上的税收、60%以上的GDP、70%以上的技术创新、80%以上的城镇就业、90%以上的新增就业。中国民营企业的数量已超过2700万家，个体工商户超过了6500万户，注册资本超过165万亿元。
 <br/>
 <br/>
 然而，由于中共一直以来的〝国进民退〞机制，以及对民营企业施行各种盘剥、打压政策，加之贸易战对中国经济的冲击，导致近期大量民营企业陷入倒闭潮。
 <br/>
 <br/>
 尤其是大陆近期出现〝私营经济离场论〞、〝民企帮助国企改革的历史使命已经完成〞等说法，被认为是北京意图拿私有企业开刀先〝试水温〞，更使民营企业的经营者倍感焦虑。
 <br/>
 <br/>
 此后，北京当局虽然多次宣称支持、保护民营企业，但未能提升民营企业对中国经济前景的信心。
 <br/>
 <br/>
 在中共经济遭遇重大危机的背景下，日前举行的川习会，北京以重大让步换取了90天的休战期。但北京需在此期间做出重大结构性改变，这将会动摇中共体制的根本，并非一件易事。因此外界对于贸易战的前景仍不乐观。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：范铭）
