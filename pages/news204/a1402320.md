### 韩媒揭孟晚舟被捕原因 北京90天〝缓刑〞日子难熬
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创始人任正非长女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被捕与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 发生在同一天，令外界对此产生诸多联想。韩媒称，美国逮捕孟晚舟是为了在谈判中对华施压。但有白宫官员表示，川普（特朗普）总统在会谈时对此事并不知情。还有分析认为，孟晚舟被捕事件显示华为深陷美国关注的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性改革.html" target="_blank">
  结构性改革
 </a>
 〞问题，北京未来90天的〝缓刑〞期将会很难熬。
 <br/>
 <br/>
 12月1日，倍受瞩目的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 在阿根廷首都布宜诺斯艾利斯举行。与此同时，中国电信巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司副董事长、首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在加拿大温哥华被捕。不过该消息在4天后，才被加拿大媒体曝光。
 <br/>
 <br/>
 据加拿大官方公布的信息，加拿大政府是因美国当局的要求逮捕孟晚舟的。孟晚舟被指涉嫌违反美国对伊朗的制裁，美国要求将她引渡，美加之间签署了引渡条约。
 <br/>
 <br/>
 韩国news1新闻12月7日报导称，有专家认为，美国逮捕孟晚舟很可能是一次缜密的计划，美国此举是为了在美中协商中占据有利地位。
 <br/>
 <br/>
 不过，白宫一名官员12月6日表示，川普在与习近平会晤之前，并不知道美国要求从加拿大引渡华为首席财务官。
 <br/>
 <br/>
 白宫国家安全顾问博尔顿6日在接受国家公共广播电台（NPR）采访时表示，他从司法部事先知道华为一名高管被捕的消息。但不确定川普总统是否事先知道此事。
 <br/>
 <br/>
 博尔顿说，这类案件发生的频率很高，不会每件事都告诉总统。
 <br/>
 <br/>
 另据《金融时报》引述一位知情人士称，中方在川习晚宴举行前，已知晓孟晚舟被捕情况，但决定不在会谈中提出这一问题，因为习近平希望集中精力解决贸易紧张局势。
 <br/>
 <br/>
 有分析认为，不论孟晚舟被捕事件是否与川习会有关联，北京未来90天的〝缓刑〞日子都会很难熬。因为华为公司所涉及的问题，与美国要求中共进行的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性改革.html" target="_blank">
  结构性改革
 </a>
 〞紧密相关。
 <br/>
 <br/>
 美国共和党参议员克鲁兹、鲁比奥和Ben Sasse就对逮捕行动表示欢迎，他们说，华为构成了安全威胁。克鲁兹在推特上写道：〝华为是一个伪装成一家电信公司共产党
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍.html" target="_blank">
  间谍
 </a>
 机构。它的监视网路跨越全球，它的客户是诸如伊朗、叙利亚、朝鲜和古巴这样的流氓政权。〞
 <br/>
 <br/>
 作为全球第二大智能手机品牌，华为是全球最大的蜂窝塔，也是最大的网际网路和电信基础设施生产商。同时，华为还是中共和军方电信类业务的最大承包商，众所周知，华为是中共科技领域一个最有实力的打手。
 <br/>
 <br/>
 华为不仅协助中共推出了监控中国民众的〝天眼〞系统，为新疆百万人〝再教育营〞提供了管理系统等等。还对海外输出中共这一套网路监控的模式及设施。
 <br/>
 <br/>
 华为手机被质疑安装监控后门，会收集用户的谘询反馈给中共。美国国防部今年4月25日发布命令，基于华为和中兴手机可能构成的安全威胁，要求在全球美军基地的商店，下架所有华为和中兴生产的手机等装置。
 <br/>
 <br/>
 近年来，华为在美国国会也一直备受怀疑，被认为是对美国国家安全的一个潜在威胁。美国中央情报局（CIA）在2011年10月的一份公开调查报告中披露，中国最大的电信公司华为类似于前苏联克格勃式的情报部门。
 <br/>
 <br/>
 美国众议院情报委员会2012年的一份报告警告说，华为为中共军队的精英网路战机构提供特别网路服务，华为的设备可能被用来监视美国人，或让美国的电信网路不稳定。
 <br/>
 <br/>
 对于华为的这些指控，与美国对北京提出的停止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网路侵入.html" target="_blank">
  网路侵入
 </a>
 、停止知识产权盗窃等要求，一脉相承。
 <br/>
 <br/>
 有报导称，孟晚舟早在2年前已被美国政府盯上。自2016年以来，美国一直在调查华为是否违反了美国对伊朗的制裁令。而这一调查源自伊朗提供的线索。
 <br/>
 <br/>
 2016年3月，《超讯》引述美方人士消息指，美伊当时关系正走向正常化，美国要求伊朗提供在美国制裁期间，与伊朗违规交易的中企名单。伊朗供出华为、中兴。
 <br/>
 <br/>
 当年，美商务部公布中兴罪状时，揭示另一家代号F7的中企也在逃避美国的出口管制，而F7被曝是中兴内部对竞争对手华为的称呼。
 <br/>
 <br/>
 有分析认为，中共企图利用科技霸凌世界，助纣为虐的华为是中共的一个打手，美国对华为的打击，也打痛了中共。不排除是对中共的又一次警告。
 <br/>
 <br/>
 在川习会上，中共同意就〝结构性改革〞进行谈判，换取90天的缓刑期。不过，有分析指，这个谈判其实非常困难，因为中共是〝政经一体〞，进行结构性改革，会触及到中共体制问题。北京这段缓刑期的日子会很难熬。
 <br/>
 <br/>
 外界还担心，中共又使出〝拖〞的老招数，苟延残喘。但川普显然对此已有所警觉，他在12月5日发推文警告称，〝如果行，我们就达成协议；如果不行，记住，我是个打关税牌的人。〞
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
