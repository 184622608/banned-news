### 遭粤警脱衣羞辱后 女律师再面临处罚
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】中国广州女律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙世华.html" target="_blank">
  孙世华
 </a>
 遭警察〝脱衣羞辱〞事件再起波澜，案件的3名证人此前遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州警方.html" target="_blank">
  广州警方
 </a>
 非法抓捕审讯后，受害律师孙世华目前面临
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政处罚.html" target="_blank">
  行政处罚
 </a>
 。有律师表示，这是警方为了掩盖自己的罪行而采取的打压行动。
 <br/>
 <br/>
 11月15号上午，广州女律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙世华.html" target="_blank">
  孙世华
 </a>
 接到通知，广州市公安局荔湾分局将向她送达处罚告知书。当天下午，孙世华等人在广州市司法局见到了荔湾区分局警察。
 <br/>
 <br/>
 警方声称，孙世华伙同张五洲、梁颂基等人，9月20号在荔湾区华林派出所〝扰乱派出所办公秩序〞，纠缠办案警察，因此打算对她进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政处罚.html" target="_blank">
  行政处罚
 </a>
 ，并让她提出申辩。
 <br/>
 <br/>
 11月16号，孙世华的丈夫、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 隋牧青对《新唐人》说，警方这样〝自任裁判〞是非法的。
 <br/>
 <br/>
 孙世华丈夫、维权律师隋牧青：〝他们要以处罚来掩盖他们的罪行，这样来说明他们没错，是孙世华有错，就这样的一个做法。〞
 <br/>
 <br/>
 孙世华在9月20号和委托人、访民李小贞到华林街派出所，与警官沟通李小贞丈夫周建斌取保一事，遭到警方拒绝。当孙世华询问警官的姓名和警号时，对方将工作证甩向她，声称她〝袭警〞。
 <br/>
 <br/>
 接着，孙世华被一群警察掐住脖子施暴，后来又被带到派出所办案区内，几名警察强制她脱光衣服，作出羞辱性检查。期间孙世华被拍照、打指模，以及强制验尿，并接受了5个小时的讯问。
 <br/>
 <br/>
 此事引发海内外关注。就在孙世华展开法律行动时，10月15号，此案的3名证人张五洲、梁颂基和李小贞遭警方抓捕拘留。
 <br/>
 <br/>
 维权律师蔺其磊说，警方除了拘留3名证人，还连续发布两个自相矛盾的声明，也一直不敢公布案发视频，这足以说明警方在无理打压。
 <br/>
 <br/>
 维权律师蔺其磊：〝当时的事发现场，广州警察他是有违法行为的，整个事情孙世华没有任何的过错，完全是被他们‘碰瓷’施暴，然后进行侮辱的一个行为。通过以上几种现象，我认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州警方.html" target="_blank">
  广州警方
 </a>
 是利用公权力，要把他本来违法的事情来压制，然而对这些知道案情事实的人进行打压。〞
 <br/>
 <br/>
 孙世华律师在11月15号下午已经向警方申辩，事发前她不认识张五洲和梁颂基，更没有滋扰派出所和纠缠警察的行为。她表示将继续刑事控告，同时还继续呼吁警方公布全部监控视频。
 <br/>
 <br/>
 新唐人记者常春、李韵、施宜君采访报导
</div>
