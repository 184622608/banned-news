### 官二代秦岭别墅细节曝光：仅狗舍就78平米
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】中国西安秦岭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建别墅.html" target="_blank">
  违建别墅
 </a>
 的问题，近几年引起了海内外舆论越来越多的关注。日前陆媒曝光了秦岭最大违建〝陈路别墅〞的部分内幕细节。该别墅圈占农田14.11亩，其中两个鱼塘面积逾千平方米，仅狗舍面积就达78平方米，园内各处还分布着各种文物逾2百件，其奢华阔气令观者咂舌。
 <br/>
 <br/>
 综合多家陆媒的报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈路超大违建别墅.html" target="_blank">
  陈路超大违建别墅
 </a>
 建位于西安市鄠邑区石井镇蔡家坡村，最初在西安本地媒体及官方信息发布平台上被称为〝支亮超大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建别墅.html" target="_blank">
  违建别墅
 </a>
 〞，基本讯息显示该别墅系以陕西秦悦贸易有限公司法定代表人支亮名义所建。
 <br/>
 <br/>
 不过该别墅被曝光系违法建筑后，有陆媒经调查发现，涉事人支亮出生于1980年，系陕西省太白县梅湾村人。他当年与石井镇蔡家坡村签订土地租赁合同时年仅25岁，大学刚毕业不久，因此怀疑这些违建别墅的真正主人另有其人。
 <br/>
 <br/>
 据陆媒披露，2005年8月，陈路、支亮以盆景栽植、园林绿化名义，与石井镇蔡家坡村第三村民小组签订土地租赁合同，面积约15亩，期限70年，并于2005年底开始圈地建设。
 <br/>
 <br/>
 至2008年8月，该园区建成了砖混建筑1栋1层，其他建筑3处，另有2个大鱼塘和1个超大狗舍，实质是一处具有园林特色的中式仿古别墅。
 <br/>
 <br/>
 陆媒探究发现，这个违建的园区内，在道路、草坪、水池边更遍布着一些石刻、石雕、石碑、拴马桩、石鼓等疑似文物的物件超过2百件，其阔气奢华令人观者咂舌。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-18/p9084552a543581020.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-18/p9084552a543581020-ss.jpg"/>
   <br/>
   <font size="-1">
    中国秦岭最大违建〝陈路别墅〞圈占农田14.11亩，其中两个鱼塘面积逾千平方米，仅狗舍面积就达78平方米，园内分布着各种文物共211件。（网络图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 该处违建别墅被媒体曝光后，迅速引发社会舆论的广泛关注，有网友惊叹：〝除去公摊面积，我家还没他的狗舍大。〞一时间，网络上呼吁追查这个超级违建别墅的呼声高涨。在巨大社会舆论压力下，西安市纪委8月8日成立专案组开始进行调查。
 <br/>
 <br/>
 后经查实，这些别墅的真正业主是一个名为陈路的人，陆媒《财经》杂志曾爆料称，陈路的父亲曾在西安市党政系统担任要职，但并未披露陈路其人和他父亲更多详细的信息。
 <br/>
 <br/>
 今年10月15日晚，在陕西多家媒体及官方信息发布平台上报导有关违建别墅的消息时，已统一将这处别墅园区称为〝陈路超大违建别墅〞。
 <br/>
 <br/>
 10月16日，西安电视台《每日聚焦》栏目披露了该别墅更多细节：总圈墙面积9280.53平方米；建成砖混建筑1栋1层，面积517.29平方米；鱼塘两处，面积1098.68平方米；狗舍一处，面积78.03平方米；其他建筑3处，面积113.58平方米；通往各处道路及道路硬化1417.72平方米。
 <br/>
 <br/>
 违建别墅院内布置的石刻、石雕、石碑、拴马桩、石鼓等共237件物件，经省文物鉴定机构鉴定后确认，其中现代工艺品26件，文物211件，可流通文物30件，不可流通文物181件，其中更有3件属于三级文物，而一般文物为178件。
 <br/>
 <br/>
 目前，上述〝不可流通文物〞 已经全部被当地官方文管所没收，而陈路超大违建别墅从今年9月29日开始被拆除，于10月15日恢复了绿地原貌。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：秦锋）
