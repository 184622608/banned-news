### 电影《大轰炸》北美重生 范冰冰戏份仅剩9秒
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 接连举报，电影《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大轰炸.html" target="_blank">
  大轰炸
 </a>
 》10月中已取消在大陆上映，但10月26日在北美重生，比之前计划在中国公映的《大轰炸》版本缩减了35分钟，原属女主角的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 角色不变，但只有5个镜头，戏份仅剩9秒。
 <br/>
 <br/>
 香港苹果日报27日称，中国女影星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 因逃税被罚近9亿人民币，其主演的电影《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大轰炸.html" target="_blank">
  大轰炸
 </a>
 》因卷入这场风波，而在中国上映无期。不过，该片10月26日在北美地区〝重生〞。
 <br/>
 <br/>
 根据流出的北美版本显示，《大轰炸》全片时长97分钟，与之前大陆计划公映的132分钟版缩减了35分钟，片中所有演员都是英语台词，没有讲英文的演员都有英语配音。
 <br/>
 <br/>
 原属女主角的范冰冰虽然戏份不变，但只有5个镜头，且都集中在同场戏中，她饰演幼稚园老师叶佩璇，带着学生躲避日军飞机的狂轰乱炸时，说了3句台词：〝no，to the basement（不，去地下室）〞、〝everybody，hurry，come now（所有人，快点）〞、〝come on，dear （没事，亲爱的）〞。
 <br/>
 <br/>
 就这样，范冰冰出场时间只有9秒钟，而且在最后的演员名单中还被放到第15位。
 <br/>
 <br/>
 据悉，该片自范冰冰逃漏税丑闻后，在大陆曾经历多次调动上映档期，但至今仍未能公映。
 <br/>
 <br/>
 电影《大轰炸》是由中国电影股份公司、原画（北京）影业、上影南国影业出品的历史战争片。由施建祥旗下的上海快鹿集团投资，获中共军方鼎力支持拍摄，更一度被官方列为重点电影而被外界视为前景美好。
 <br/>
 <br/>
 但在2015年11月电影杀青后，传出主要投资人施建祥出事外逃，上映一直被搁置，直到今年当局才宣布8月17日上映。但在8月7日，《大轰炸》官方微博宣布，《大轰炸》改档至10月26日，同时，范冰冰的名字也从海报和宣传稿中〝消失〞。
 <br/>
 <br/>
 范冰冰由于卷入逃税案而〝消失〞4个月后，中共税务部门10月3日称，范冰冰在《大轰炸》拍摄中，获片酬3,000万元涉嫌偷税漏税，缴纳8.8亿元人民币的罚金可免予刑责。
 <br/>
 <br/>
 随着范冰冰逃税案落幕及《大轰炸》上映在即，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 又接连在微博举报称：电影《大轰炸》牵扯使用不明的资金至少30亿人民币以上，仅在拍摄中就使用各种肮脏手段弄走资金17亿。他还呼吁大众抵制《大轰炸》上映。
 <br/>
 <br/>
 10月17日，导演萧锋在微博宣布电影《大轰炸》取消上映。他说，〝再多的事实都于事无补，再多的清白都难挡抹黑，是到放下的时候了，只是可怜了养育八年的孩子《大轰炸》！向仍对电影有期待的观众道歉！〞
 <br/>
 <br/>
 新浪娱乐向电影宣传方求证后，确认《大轰炸》确实是取消了在院线公映的计划。
 <br/>
 <br/>
 此前一度传出《大轰炸》已将范冰冰的戏份全部删除，电影海报也将范冰冰除名，希望可以因此而〝逃出生天〞，在中国拿到批文上映，但最后事与愿违，经过多次大动作的修改和处理的《大轰炸》，最后还是被取消了在院线公映的计划。
 <br/>
 <br/>
 美国媒体《Deadline》10月18日报导说，电影公司Lionsgate打算，10月26日在美国本土上映电影《大轰炸》；北美地区的另一家发行商Grindstone也预计，会让该片在美国院线小规模上映。
 <br/>
 <br/>
 当时外媒推测，北美上映的版本，将会保留范冰冰的戏份。如今靴子终于落地。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
