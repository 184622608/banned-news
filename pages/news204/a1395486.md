### 【禁闻】扬州户主驾车撞强拆人员 致10人死伤
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
   王全璋
  </a>
  被关超三年 妻最高法控告受阻
 </b>
 <br/>
 <br/>
 〝709案〞被抓律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 ，被关押已经超过三年，至今仍然不审、不判、不让见家属和律师。10月15号，他的妻子李文足前往中共最高法院递交《督促函》，要求纠正〝严重超出审理期限〞的违法行为，但被警察挡在法院外，不得进入。
 <br/>
 <br/>
 李文足在推特上发消息说，当天在最高法大门口，各路控告人，被最高法的法警忙不迭地连声催促着：赶快进！但到了她，最高法的各个入口，就都被法警用身体死死挡着。无论她怎么讲理，那些法警都犹如木偶，没有反应。
 <br/>
 <br/>
 王全璋是大陆知名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 律师，长期为弱势群体、尤其是法轮功学员维权。他在2015年7月中共针对维权律师的大抓捕行动中被捕，杳无音讯三年之后，今年7月有律师通过官方渠道，会见了他两次，但之后又杳无音讯。
 <br/>
 <br/>
 <b>
  扬州户主驾车撞
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
   强拆
  </a>
  人员 致10人死伤
 </b>
 <br/>
 <br/>
 中共政府的强拆行为再次引发血案，10月15号，江苏省扬州市杭集镇曙光路的一户居民，因抵制强拆，驾车撞向强拆人员。当地官方通报说，事件造成1死9伤。
 <br/>
 <br/>
 当地市民向〝自由亚洲电台〞介绍说，镇政府雇佣一帮社会临时工到这户人家去强拆，可能伤及了户主的老母亲，于是他开辆车冲向了人群。
 <br/>
 <br/>
 大陆《新京报》则报导说，房屋是在没有告知户主的情况下突然强拆的，户主夫妇赶到现场交涉，遭到2、30人殴打。
 <br/>
 <br/>
 目前，涉事人员已经被警方拘捕，当地官方也开始维稳并阻止消息传播。
 <br/>
 <br/>
 <b>
  孙世华遭警羞辱案 三目击证人遭刑拘
 </b>
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州女律师.html" target="_blank">
  广州女律师
 </a>
 孙世华遭警察侮辱一案继续发酵，三名目击证人梁颂基、张五洲、李小贞，10月14号相继被广州警方传唤，其中梁颂基更是在北京遭跨省拘捕。15号，三人全部被刑事拘留。
 <br/>
 <br/>
 目前，网络上要求警方公布现场监控视频、还原事件真相的呼声仍然很多，也有人质疑警方将3名目击证人治罪，是要封口或灭口，掩盖羞辱孙世华的真相。
 <br/>
 <br/>
 女律师孙世华9月20号到广州荔湾区华林派出所办案，遭到该所警察殴打、脱光衣服羞辱。警方事后否认事实，又不公布现场录像，引发舆论广泛质疑。
 <br/>
 <br/>
