### 10月12日全球看中国
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】下面是今天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球看中国.html" target="_blank">
  全球看中国
 </a>
 <br/>
 <br/>
 华尔街日报
 <br/>
 10月11日，美国出台新规，限制面向中国的核产品及技术出口，以防止中共非法获取核技术来提高自身军力。新规立即生效，其中包括禁止出口核产品及技术给中国最大的核电公司中国广核集团。这项政策被视为川普政府试图挫败中国采购美国关键技术的最新举措。
 <br/>
 <br/>
 大纪元时报
 <br/>
 美国联邦参议院近日发起决议案，谴责中共打压法轮功、新疆、基督教、天主教、藏传佛教徒等团体，要求中共停止迫害， 重申美国对促进全球宗教自由和宽容的承诺。决议案由共和党参议员大卫‧珀杜提出，已获17位参议员联署。
 <br/>
 <br/>
 德国之声
 <br/>
 近年，美、英、加拿大、澳大利亚、新西兰组成的情报共享网络〝五眼联盟〞加强合作，主要针对中共在海外的动向。今年起，该联盟还将对中共的情报共享范围扩大到德国、日本和法国。路透社认为，这说明一个针对中共扩张的跨国联合阵线正在形成。
