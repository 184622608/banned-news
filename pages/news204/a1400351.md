### 刘强东性侵再爆内情 警告助理：别妨碍我
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东性侵案.html" target="_blank">
  刘强东性侵案
 </a>
 又有更多细节曝光。英媒披露，在刘强东实施性侵前，他的女助理曾试图坐在他与受害女生之间。但刘强东警告她：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_别妨碍我.html" target="_blank">
  别妨碍我
 </a>
 。
 <br/>
 <br/>
 据路透社11月21日独家报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东性侵案.html" target="_blank">
  刘强东性侵案
 </a>
 发生在一家日本餐馆举办的晚宴上，当时该案受害女学生坐在刘的旁边。
 <br/>
 <br/>
 据警方记录，宴会结束时，受害女生说她喝醉了，需要帮助才能回家。这时，一位名叫爱丽丝．张（Alice Zhang）的女子自称是刘强东的私人助理，她帮助女生上了刘强东的车。
 <br/>
 <br/>
 受害女生称，这名女助理最初试图坐在她和刘强东之间，但被刘强东警告〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_别妨碍我.html" target="_blank">
  别妨碍我
 </a>
 〞，于是坐到了前排座位上。
 <br/>
 <br/>
 随后，刘强东开始在后排座位上对受害女生动手动脚。而坐到前排的那名女助理打开了收音机，并折起了后视镜。
 <br/>
 <br/>
 根据受害女生对警方的陈述，她当时很害怕，喝醉了，不知道自己在哪里。她以为可以〝坐下来和刘说话〞，就说服刘送她回家。
 <br/>
 <br/>
 报导称，当他们到达女生住的公寓后，刘强东告诉司机和张姓助理，他会很快回来。
 <br/>
 <br/>
 路透社记者拨通了张姓助理的手机，但接电话的女士否认自己是张某，并在听到有关报导的描述后挂断了电话。而刘强东的律师对此没有回应置评。
 <br/>
 <br/>
 报导还引述一位知情人士透露，刘在受害女生的公寓里呆了大约四个小时，并在那洗澡。受害女生说，她一再拒绝刘，并试图让他离开。
 <br/>
 <br/>
 根据路透社听到的一段警方录音，受害女生对警方表示：〝我请求他，‘别那样做。你有妻子和孩子，’但他不听我的。〞
 <br/>
 <br/>
 她还表示：〝他试图把我拉进去洗澡。我竭尽全力对付他。〞
 <br/>
 <br/>
 最终事情还是发生了，受害女生保留了当时的床单作为证据。
 <br/>
 <br/>
 事件发生后的第二天（8月31日），刘强东的另一位女助理，薇薇安．杨（Vivian Yang）联系了受害女生。
 <br/>
 <br/>
 路透社看到了受害女生与杨姓助理的微信聊天信息，受害女生的律师证实，这些微信是真实的。受害女生也向警方提供了相同的微信信息，她对警方说，本以为刘强东可以向她道歉。
 <br/>
 <br/>
 据微信信息显示，受害女生对杨姓助理表示，她想谈谈〝昨天发生的事情〞。她希望〝公正〞，并正在考虑找律师。
 <br/>
 <br/>
 杨姓助理则建议受害女生当晚与卓越集团董事长李华见面，〝你说你的，我们说我们的。让李老板先和你说，他是我们双方都信任的人，他可以先和你沟通一下。〞
 <br/>
 <br/>
 根据美国明尼苏达大学的资料，李华和刘强东一同参加了该大学8月底举办的工商管理博士项目。
 <br/>
 <br/>
 报导称，无法确定，杨姓助理为何提出由李华做中间人，也无法确定李华是否同意这样做，或者他是否知道这个计划。
 <br/>
 <br/>
 受害女生拒绝了这个提议，她表示只想和刘强东谈。
 <br/>
 <br/>
 微信信息还显示，杨姓助理最终同意在8月31日晚上10点半左右，带刘强东去明尼苏达大学卡尔森管理学院见她。
 <br/>
 <br/>
 报导称，当晚，刘强东到达那里时，受害女生正在向警方作陈述。警方在晚上11点左右逮捕了刘强东。
 <br/>
 <br/>
 第二天下午4点左右，刘强东被释放，几个小时后，他飞回了中国。根据美国警方9月4日发布的报告显示，刘强东被指控〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一级强奸.html" target="_blank">
  一级强奸
 </a>
 〞罪名。
 <br/>
 <br/>
 此前，有华裔刑案律师表示，刘强东遭起诉的可能性超过90%，案件一旦开始会涉及重新逮捕等问题。目前，负责刘强东性侵案的美国检察官，正在权衡当事人陈述以外的证据，以求打破此案的僵局。
 <br/>
 <br/>
 （记者罗婷婷报导）
