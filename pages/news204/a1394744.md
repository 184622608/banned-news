### 中共网安新规 挑战美国贸易诉求 白宫怒批
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部.html" target="_blank">
  公安部
 </a>
 日前发布新规，授权警察可以查阅和复制企业的网络安全资料。在华外企担心敏感资料外洩，白宫批评，这与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 的公平贸易原则背道而驰。
 <br/>
 <br/>
 中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部.html" target="_blank">
  公安部
 </a>
 新《网安监督检查规定》，宣称从11月1日起，所有县级以上公安局的网安部门，可派警察到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_互联网.html" target="_blank">
  互联网
 </a>
 服务提供商和互联网使用单位，采取现场检查，或远程检测的方式；来查阅、甚至复制企业的网络安全相关资讯。
 <br/>
 <br/>
 例如《规定》第11条第2款，公安部门将监督检查〝互联网数据中心服务商〞，是否纪录了各类用户信息。
 <br/>
 <br/>
 任何云服务相关的，都可以算在这个范围内。其实2017年的时候，百度、阿里（云服务）已经都实名制了，可是一些小的企业还没有要求实名制，但是一旦出台这个，就等于所有你跟云服务有关系的，一定要实名制，而且一定要帮政府提供所有用户资讯。
 <br/>
 <br/>
 《规定》第11条第4款要求，公安部门要检查互联网信息服务商，是否对用户发布信息进行了〝管理〞，是否对用户已发布的所谓〝违禁信息〞进行了处理和保存。
 <br/>
 <br/>
 就像是WeChat微信，大家都知道它会针对每条信息进行过滤筛选。这个法规一出来，对于想要在大陆做生意的所有信息服务商，它们也要做相似的配合，配合政府来做信息的管控、信息的过滤、信息的筛选，而且要提供用户的资料。它们背后势必都要有一个后门，能够让政府来监管。影响很大。
 <br/>
 <br/>
 新规宣称，如果企业允许敏感资讯在网上传播，要负法律责任。
 <br/>
 <br/>
 它这个写的这么广泛，我觉得看起来就是要完完全全的，让所有人在使用网际网路上面，对政府来说要全面透明。那对消费者或对使用者来说，就是毫无隐私可言。
 <br/>
 <br/>
 《华尔街日报》说，外国企业也担忧，中共还可能要求外企交出公开源代码和其他机密，来证明其服务是否安全。
 <br/>
 <br/>
 它里面有一条写说〝对于上网服务的，监督检查，需要符合国家标准的网络与信息安全保护技术措施。〞如果它提出这个要求，那对于软体或者服务商来说，有时候必须要把储存方法、加解密的方式的技术细节，白纸黑字的提供给人家，才能够验证他们符合中共所要求的安全措施管理办法。
 <br/>
 <br/>
 《华尔街日报》说，外企担心，这些机密信息，可能被洩露给中国本土竞争业者。
 <br/>
 <br/>
 白宫首席经济顾问库德洛，也对中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_严控网络.html" target="_blank">
  严控网络
 </a>
 ，表达强烈批评。
 <br/>
 <br/>
 库德洛：这规定允许中共官员，检查
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 和所有其他外国公司的信息技术，并有权接触已经具备所有权的资讯。这是走入错误的方向，这是强制技术转移，这是知识产权盗窃，这都是我们警告过他们的，我们会试着跟中共沟通这些问题。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
