### 网文：天将大变 中国将爆发〝雪崩式〞返贫！
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的爆发将对中国人的利益产生什么影响？是大多数中国人关心的话题之一。在贸易战火升高的敏感时刻，微博上2017年的文章再次被翻炒。文章预言，在未来3年，中国将爆发前所未有的雪崩式的〝返贫〞，不论富人还是普通民众，将无一幸免。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房市泡沫.html" target="_blank">
  房市泡沫
 </a>
 将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_呑噬.html" target="_blank">
  呑噬
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富人财富.html" target="_blank">
  富人财富
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泡沫产业.html" target="_blank">
  泡沫产业
 </a>
 效应迫使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外商.html" target="_blank">
  外商
 </a>
 被洗出大陆等，整个社会的财富都将面临重组。
 <br/>
 <br/>
 台湾财信传媒董事长谢金河在脸书说，微博这篇文章言简意赅，暗藏玄机，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 升温之际，作者似乎看到中国经济下一个调整方向，值得参考。
 <br/>
 <br/>
 微博这篇题为〝天将大变，中国将爆发雪崩式返贫，你知道？〞的文章，转载如下，供读者参考。
 <br/>
 <br/>
 有句话是这样说的：出来混，早晚都会还的。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 的30多年以来，中国一种处于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_造富运动.html" target="_blank">
  造富运动
 </a>
 阶段，无论从人数上、还从财富总量上，造富结果均令世界刮目相看：
 <br/>
 <br/>
 80年代发财靠投资倒把。当时暴发户主要是〝倒爷〞，靠特权批出内部低价商品，转手到市场高价卖出，小到肥皂，中到电视，大到钢铁汽车。其巅峰是1989年，牟其中从俄罗斯倒来一架图154飞机，转手卖给了四川航空，这让他成为中国最大的〝倒爷〞。
 <br/>
 <br/>
 90年代发财靠国企改制。当时一大批国企员工下岗，而也有一批人因此富起来了。尤其是当年颇具争议的民企入股和MBO模式，非常容易出现暗箱操作、导致国资贱卖。
 <br/>
 <br/>
 2000年以后发财靠房地产。当时中国进行住房改革，万科就是这时才开始了房地产业务，这时无论是造房子的，还是炒房子的都大发横财。
 <br/>
 <br/>
 2010年以后的发财靠融资。这时企业赚钱或者不赚钱都不是最关键的，关键看你会不会讲故事，最典型的代表就是京东，一年亏损几十亿，老板刘强东一年分红却可以几十亿。
 <br/>
 <br/>
 这个大盛筵一直持续到如今，我们忽然发现中国传统产业的生命期已经到了，这是一个生命周期的问题，而不是结构调整的问题。
 <br/>
 <br/>
 未来3年，中国将爆发前所未有的雪崩式的〝返贫〞。
 <br/>
 <br/>
 富人如何〝返贫〞？
 <br/>
 <br/>
 首先我们先看看富人是如何〝返贫〞的。
 <br/>
 <br/>
 可以毫不客气的说，中国人的发家方式大多是不光彩的，是见不得光的。那些财富的拥有者很少敢出来晒太阳。于是这些年做移民中介的生意非常好，然而能溜走的总是少数。
 <br/>
 <br/>
 <h4>
  一，楼市泡沫吞噬
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富人财富.html" target="_blank">
   富人财富
  </a>
 </h4>
 <br/>
 如今中国亿万富豪中近一半是房产商，但楼市泡沫总有一天会破灭，到那一天，富人圈会爱红边沿。就在当下，楼市下行，已经让很多房地产商资金链断裂，甚至跑路。
 <br/>
 <br/>
 <h4>
  二，富人理财出现大面积违约
 </h4>
 <br/>
 以信托、私人银行为代表的富人理财方式已经出现了大面积违约的兆头。标普统计中国企业借款14万亿美元全球第一，影子银行融资占三分之一，这将是一个不定时的炸药包。
 <br/>
 <br/>
 <h4>
  三，〝世界工厂〞风光不再
 </h4>
 <br/>
 在上一波造富潮中，〝世界工厂〞成为一大推手。很多民营制造业老板抓住这个机遇，由此身家十倍增长，成为亿万富豪。但如今，中国廉价劳动力优势不再，大面积的低端制造产业无法升级转型，等待他们的，只有破产和财富大量缩水。
 <br/>
 <br/>
 <h4>
  四，重污染、重能耗企业被清除
 </h4>
 <br/>
 中国富人返贫，〝首当其冲的是矿产、房地产和钢铁等重污染领域。本质而言，是他们精神太过贫乏，物质太过嚣张，当社会游戏规则改变时，他们缺乏足够智慧改变自己，因而守不住财富积累。〞曾以7000万元嫁女而闻名的山西煤老板xxx已经被警方带走，破产崩盘。
 <br/>
 <br/>
 <h4>
  五，人民币升值泡沫破灭
 </h4>
 <br/>
 以前美元贬值热钱涌入曾对中国造富锦上添花，而今美元复兴热钱外流，难免对中国落井下石。有点评称，人民币贬值可能导致此前借大宗商品价格上涨暴富的富豪资产大幅缩水。
 <br/>
 <br/>
 <h4>
  六，
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泡沫产业.html" target="_blank">
   泡沫产业
  </a>
  挤出效应
 </h4>
 <br/>
 一些在中国经济高速发展时被吹胀的产业最终会现出原型，譬如以施正荣、彭小峰等为代表的光伏富豪，如今只能承受潮水退去后的〝裸泳〞。而这样的产业，还有多少呢？
 <br/>
 <br/>
 <h4>
  七，富二代败家
 </h4>
 <br/>
 富二代接班已经成为中国第一代民营企业最大的心头痛。譬如近日媒体报导的xx集团董事长李xx，其接班的十年，也是家族实业盛极而衰的十年。富二代如果没有力挽狂澜的能力，很难在未来的变局中守业。
 <br/>
 <br/>
 <h4>
  普通百姓财富如何〝蒸发〞？
 </h4>
 <br/>
 接下来，我们再看看普通百姓的财富是如何蒸发的。
 <br/>
 <br/>
 主要有8中情形：
 <br/>
 <br/>
 1. 2015年到2016年的两次大股灾，直到现在熊市还在继续；
 <br/>
 <br/>
 2. P2P跑路的，财富管理公司崩盘的。从泛亚到e租宝再到如今的中晋；
 <br/>
 <br/>
 3. 民间借贷崩盘带来的负面效应还一直在；
 <br/>
 <br/>
 4. 实体店的亏损还在继续，即使房租在下降，仍然无济于事；
 <br/>
 <br/>
 5. 现在天猫、淘宝开网店的基本上都已经赚不到钱了，都在那耗着；
 <br/>
 <br/>
 6. 创业者这好不容易拿到投资，却开始A轮死、B轮死、C轮死；
 <br/>
 <br/>
 7. 大量工厂的利润率越来越低，开始资不抵债，工人的工资得不到保障；
 <br/>
 <br/>
 8. 由于经济萎靡，服务业也开始萧条，大量白领的岗位岌岌可危。
 <br/>

 <h4>
  点评
 </h4>
 <br/>
 这一次变革将彻底淘汰那些暴富者、投机主义者、特权主义者，2017年中国80%的富人将返贫，包括贪官污吏、矿产、房地产等传统暴富领域。究其本质而言，是中国传统的产业、传统的企业、甚至传统的人已经陷入僵化的思维不可自拔，当中国迎来真正的好时代，游戏规则发生改变时，他们将因缺乏足够的文化、智慧、创新、责任去改变自己，从而陷入被革命的危险绝境！
 <br/>
 <br/>
 整个社会的财富也将面临一场重组！所以，对于那些真正有理想、有抱负、有想法的年轻人来说，真正的机会来了！不要再为你之前的成就骄傲了，更不要为你当下的窘迫而失望。未来，你只要有理想、有文化、理解变革，并携带正能量，就能在中国下一个时代大展拳脚！
