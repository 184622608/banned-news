### 唐代诗人杜牧墓遗址成菜地 学者感慨
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】〝清明时节雨纷纷，路上行人欲断魂；借问酒家何处有，牧童遥指杏花村。〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_唐代诗人.html" target="_blank">
  唐代诗人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜牧.html" target="_blank">
  杜牧
 </a>
 的〝清明〞是许多人熟悉的名句。但有学者日前发现，杜牧在西安的坟墓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遗址.html" target="_blank">
  遗址
 </a>
 已沦为一片
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菜地.html" target="_blank">
  菜地
 </a>
 ，一代诗人的墓落得如此下场，学者为之感慨。
 <br/>
 <br/>
 史书记载，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杜牧.html" target="_blank">
  杜牧
 </a>
 是唐代著名诗人，京兆万年人，也就是今天的西安人。名作有〝秋夕〞，内容是：银烛秋光冷画屏，轻罗小扇扑流萤；天阶夜色凉如水，卧看牵牛织女星。
 <br/>
 <br/>
 他还有〝遣怀〞诗作，原文是：落魄江南载酒行，楚腰纤细掌中轻；十年一觉扬州梦，赢得青楼薄幸名。
 <br/>
 <br/>
 2012年，西安公布一份文物名录通知，认定杜氏家族墓位于长安区大兆街道，也就是今天的大兆街道司马村。
 <br/>
 <br/>
 据长安县志转述〝咸宁县志．陵墓志〞记载，〝杜牧墓，葬少陵原司马村先茔（墓地），自为墓志。〞其墓在司马村西南，高约7公尺，面积约1亩，墓顶有一大树，1960年代遭平毁。
 <br/>
 <br/>
 不过，中国唐史学会专家学者和陆媒记者11月25日到杜牧墓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遗址.html" target="_blank">
  遗址
 </a>
 探访时，却发现这里已沦为一片低洼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菜地.html" target="_blank">
  菜地
 </a>
 。学者为此发出感慨。
 <br/>

 </center>
 <br/>
 报导引述司马村68岁的关山回忆说，杜牧墓遭毁坏发生在1967至1968的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文化大革命.html" target="_blank">
  文化大革命
 </a>
 时期，当时，为了修筑中共生产队饲养室，官方指使民众挖掉了墓土。
 <br/>
 <br/>
 随后，当时村民不时挖土盖房，短短一两年时间内，7公尺高的墓土便被挖尽。如今，冢的中心只遗留下一个方坑。
 <br/>
 <br/>
 西北大学文化遗产学院教授徐卫民看到现场感慨地说，希望能按照史书记载将杜牧墓封土恢复起来，并建一个纪念馆，在清明节能够开展祭祀杜牧的活动，既是对杜牧的纪念，也是对传统文化的保护。
 <br/>
 <br/>
 中共发动的文化大革命对中华传统文化是一场浩劫，中共打着破四旧的口号，将中华历史文物古迹几乎被摧毁殆尽，多少圣君帝王、历史先贤、文人墨客的坟墓被毁。
 <br/>
 <br/>
 如，文革期间，炎帝陵主殿及其附属建筑遭严重破坏，并刨挖陵墓内存物抢夺一空，最后全部夷为平地；造字者仓颉的墓园被毁，改造成了〝烈士陵园〞；山西舜帝陵被毁；浙江绍兴会稽山的大禹庙被拆毁，高大的大禹塑像被砸烂，头颅齐颈部截断，放在平板车上游街示众。
 <br/>
 <br/>
 世界佛教第一至宝，佛祖释尊在世时亲自开光的三圣像之一八岁等身像被捣毁。
 <br/>
 <br/>
 曲阜孔庙遭受毁灭性的打击，孔子墓被铲平挖掘，其内孔子遗物与骨骇示众后焚毁，〝大成至圣先师文宣王〞大碑被毁。庙碑被毁，孔庙的塑像被毁。
 <br/>
 <br/>
 最早进行的破坏活动由北京师范大学的200余名师生组织，被摧毁的之前登记在册的文物就有6618件，其中画929幅，书籍2700余本，石碑1000余块，墓葬2000余。孔子的七十六代孙孔令贻的坟墓被掘开。
 <br/>
 <br/>
 杭州革命青年砸了岳庙，连岳飞的坟也刨了个底朝天。岳武穆被焚骨扬灰。河南汤阴县中学生将岳飞等人的塑像、铜像，秦桧等〝五奸党〞的铁跪像，连同历代传下的碑刻〝横扫〞殆尽。
 <br/>
 <br/>
 医圣张仲景的塑像被捣毁，墓亭、石碑被砸烂，〝张仲景纪念馆〞的展览品也被洗劫一空。〝医圣祠〞已不复存在。
 <br/>
 <br/>
 河南南阳诸葛亮的〝诸葛草庐〞(又名武侯祠)。的〝千古人龙〞、〝汉昭烈皇帝三顾处〞、〝文韬武略〞三道石坊及人物塑像、祠存明成化年间塑造的十八尊琉璃罗汉全部捣毁，殿宇饰物砸掉，珍藏的清康熙《龙岗志》、《忠武志》木刻文版焚烧。
 <br/>
 <br/>
 书圣王羲之的陵墓及占地二十亩的金庭观几乎全部平毁，祗剩下右军祠前几株千年古柏陪伴书圣失去了居所的亡魂。
 <br/>
 <br/>
 朱元璋巨大的皇陵石碑被拉倒；石人石马被炸药炸得缺胳膊少腿；皇城也拆得一干二净。
 <br/>
 <br/>
 红卫兵掘开蒲松龄的坟，教书匠蒲松龄真穷，墓里除了手中一管旱烟筒、头下一迭书外，只有四枚私章。他们对蒲氏私章不屑一顾，弃之于野。尸体被捣毁。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
