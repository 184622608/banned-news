### 高智晟似人间蒸发　近百律师吁中共放人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权律师.html" target="_blank">
  人权律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高智晟.html" target="_blank">
  高智晟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 逾一年多，各地公民先后发起多起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_寻找行动.html" target="_blank">
  寻找行动
 </a>
 ，但是至今高律师仍音讯全无，生死不明。近日，全国各地公民及上百名维权律师再次发起关注高智晟行动，要求当局立即释放高智晟。
 <br/>
 <br/>
 参与联署的人士表示，直到现在全国各地公民，仍然在到各地法院、看守所、监狱等场所查问，希望能够打听到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高智晟.html" target="_blank">
  高智晟
 </a>
 的下落。
 <br/>
 <br/>
 参与联署的辽宁维权人士王素娥4日对自由亚洲表示，她与多名民众自从关注高智晟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 后，一直都受到当局监控，电话也一直被干扰。
 <br/>
 <br/>
 但她表示，不怕当局打压，会继续参与行动，她相信事件在愈来愈多人关注下，当局迟早会公开高智晟的情况，她强调绝对不会放弃寻找高智晟行动，直到高律师出现为止。
 <br/>
 <br/>
 参与联署的覃永沛律师认为，即使有一万人联署关注高智晟的情况，估计作用也不会大，因为共产党都不会在乎的。但是他也强调，作为一个公民，应该要为被打压的人士发声，免得更多人再受害。
 <br/>
 <br/>
 覃永沛律师认为，现时只有高智晟的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_耿和.html" target="_blank">
  耿和
 </a>
 可以帮到其丈夫，他希望耿和可以在美国循法律途径，控告中共侵犯人权，迫使中共交待高智晟的下落。
 <br/>
 <br/>
 上述
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联署信.html" target="_blank">
  联署信
 </a>
 是继9月20日，来自中国各地的众多维权人士发起〝寻找高智晟律师万里行〞之后，又一次进行的联署行动。
 <br/>
 <br/>
 参与联署的中国维权律师包括：刘跃、汪丽、李发旺、邵重国、王素娥、王一、刁继军、崔斌、赵鹏飞、李勇生、张五洲等近百名律师。
 <br/>
 <br/>
 维权律师们在联署信中表示，释放高智晟律师是人心所向，高智晟律师是正义的象征，他维护弱势苦难群体的合法利益，维护法律的正当实施。他的行动不但没有违法，而且于国有功。
 <br/>
 <br/>
 因此，律师们强烈要求中共当局立即无罪释放高智晟、恢复他的自由，让高智晟律师与家人团聚。
 <br/>
 <br/>
 高智晟的妻子耿和说，中共用恶毒的手段，对高智晟迫害和折腾了十多年了。高智晟已经完成了对他的判刑，也完成了缓刑，现在是不是应该让他回家了。但中共不但一直对高智晟加以迫害，把他强迫失踪。
 <br/>
 <br/>
 耿和继续说，中共〝不但迫害高智晟，还殃及他和我的亲属们，我在新疆亲属们的身份证已被当局没收了，他们没法离开当地。他们每月还必须到公安局报到和签字，当局的理由是，我和孩子是通缉犯。为了不连累亲属们，我现在尽量不联系他们。〞
 <br/>
 <br/>
 耿和表示，她很感激那勇敢的维权律师们发起行动和写联署信为高智晟说话，但同时她也担心，这些律师们因此会遭到中共的打压和迫害。
 <br/>
 <br/>
 耿和希望国际社会、海外人权组织和媒体，也多多关注这些为高智晟说话的律师们的命运。
 <br/>
 <br/>
 目前在美国纽约任律师的李进进说，没有任何法律支持中共把人强迫失踪的做法，就连中共自己的法律也没有这样的条文。中共不仅对异议人士，而且对它认为有问题的所有人，都进行这种强迫失踪的惩罚。这完全是违法的，更违反了国际人权法。
 <br/>
 <br/>
 现年54岁的高智晟曾被中共司法部评为〝中国十佳律师〞。从2004年12月开始，他多次上书中共高层，要求停止对法轮功惨绝人寰的迫害。2006年8月，他被吊销律师执业证，随后被长期绑架，遭受骇人听闻的酷刑，最后，他被关进新疆偏远的沙雅监狱。
 <br/>
 <br/>
 2014年8月7日，高智晟从沙雅监狱释放后，回到陕西省北部老家，3年多来，被大批国安一天24小时监控。
 <br/>
 <br/>
 2017年8月7日，高智晟刑满出狱3周年，当天他在接受新唐人采访时表示，希望能改变中共邪恶的制度，让中国人回到正常的生活中去，回归正常的人类文明。
 <br/>
 <br/>
 高律师说，尽管当局对他严密监控打压，但2015年至今，他陆陆续续在这样的恶劣的环境下做了一些事情，尤其是让他感到欣慰的是：他的著作《2017年，起来中国》，《2016年中国人权报告》和《2016年中华联邦共和国宪法》的出版。
 <br/>
 <br/>
 他说，自己完全是出于个人对国家、对民主、对未来的责任和考量或者担负的心态下做成的，〝所以我认为这是公事，我希望更多的人去关注它、传播它。〞
 <br/>
 <br/>
 为了摆脱中共国安24小时昼夜监控，2017年8月13日，高智晟在山西维权人士李发旺、邵重国等人帮助下，逃离到山西躲藏。但于6天后的19日，高智晟再次遭警方搜捕带走，至今生死不明。
 <br/>
 <br/>
 2018年8月底，高智晟获得了人权机构〝第一步论坛〞授予的〝沙赫巴兹-巴蒂自由奖〞，该奖每年颁发给一人，以表彰获奖人在人权、自由、宗教方面的杰出贡献和重要影响。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明）
