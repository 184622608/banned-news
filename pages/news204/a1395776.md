### 10月17日全球看中国
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】下面是今天的全球看中国
 <br/>
 <br/>
 1. 大纪元时报
 <br/>
 近日，深圳市政府提交了一份文件给市发改委。针对目前中美贸易战新局势提出多项建议，包括采取更精准、隐蔽的策略获取美国等发达国家高科技
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 。
 <br/>
 <br/>
 2. 华尔街日报
 <br/>
 中共国家统计局周二公布，受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_食品和燃油.html" target="_blank">
  食品和燃油
 </a>
 价格上涨的影响，9月份消费者价格指数(CPI)涨幅连续第四个月走高，同比上涨2.5%，触及七个月高点，通胀率高于8月份的2.3%。经济学家认为，数据没能反映生活成本的实际涨幅。
 <br/>
 <br/>
 3. 美国之音
 <br/>
 经济合作与发展组织（OECD）对全球100多个投资移民项目进行分析后，认定21个国家和地区的36个项目对该组织推行的防止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海外避税.html" target="_blank">
  海外避税
 </a>
 的税务申报系统构成高风险，颇受中国富豪青睐的欧洲国家马耳他上榜。
 <br/>
 <br/>
 4. 法国国际广播电台
 <br/>
 美国驻中国大使馆10月17日启用新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空气质量监测器.html" target="_blank">
  空气质量监测器
 </a>
 。这是其空气监测项目十周年的纪念日。美驻华大使布兰斯塔德表示，希望两国专家一起分享最佳办法，改善空气质量。
