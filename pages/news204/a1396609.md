### 范冰冰之后 党媒又点名狠批一当红女星
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】中国女星范冰冰逃税事件风波未平，中国娱乐圈又掀批评女星演技的热潮。近日，官媒又点名狠批另一名当红女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨幂.html" target="_blank">
  杨幂
 </a>
 ，起因是杨幂主演的文艺片《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宝贝儿.html" target="_blank">
  宝贝儿
 </a>
 》票房不如预期。
 <br/>
 <br/>
 10月22日，北京市委机关报《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京日报.html" target="_blank">
  北京日报
 </a>
 》发布《形象一旦透支，别怪观众不买账》的评论文章，主要涉及的是由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨幂.html" target="_blank">
  杨幂
 </a>
 主演的电影《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宝贝儿.html" target="_blank">
  宝贝儿
 </a>
 》。
 <br/>
 <br/>
 文章称，〝顶级流量明星〞杨幂被寄予厚望，可最后〝票房口碑双输〞。
 <br/>
 <br/>
 据悉，电影《宝贝儿》10月19日在中国上映，影片是由侯孝贤监制、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘杰.html" target="_blank">
  刘杰
 </a>
 执导，杨幂、郭京飞、李鸿其主演的一部纪实风格文艺电影。
 <br/>
 <br/>
 影片讲述一个因为严重先天缺陷而被父母抛弃的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弃儿江萌.html" target="_blank">
  弃儿江萌
 </a>
 （杨幂 饰），拯救另一个被父母宣判了〝死刑〞的缺陷婴儿的故事。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-24/p9094691a711317273.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-24/p9094691a711317273-ss.jpg"/>
   <br/>
   <font size="-1">
    杨幂主演的文艺片《宝贝儿》票房不如预期，被《北京日报》署名评论文章炮轰。（北京日报截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 影片中，杨幂不仅素面朝天的大胆扮丑，而且还用南京方言讲台词，用手语与其他演员对戏。因此，该片被视为杨幂的转型之作，不论是她本人还是其粉丝都寄予厚望。
 <br/>
 <br/>
 但北京日报的文章称，杨幂此举不仅没能打动观众，也没有任何突破，演技一如既往的〝烂〞，该电影在豆瓣网上的评分仅为5.6分，超1万人评价。
 <br/>
 <br/>
 文章最后还称，这一现象给流量明星敲响了警钟：〝观众不可能等到你赚了大钱，现在你想求名，就会买你的帐。〞
 <br/>
 <br/>
 《宝贝儿》上映前，该报也曾发表《艺术片不是镀金片》的评论文章，称〝杨幂因为在（中国）开粉丝电影先河，成为国内最具商业价值的演员之一，但与她越来越高的知名度相对应的，还有一个尴尬的局面，那就是她一直饱受诟病的演技。〞
 <br/>
 <br/>
 文中还称，杨幂想通过文艺片，扭转自己演技的缺失。同时还引用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_导演.html" target="_blank">
  导演
 </a>
 李少红的话称，〝杨幂最大的问题是从小待在剧组，对演戏太习以为常，都下意识去程序化表演，快乐就是哈哈哈，痛苦就是哇哇哇，不过脑子。〞
 <br/>
 <br/>
 值得关注的是，中共党报人民网也转载了《北京日报》的上述两篇文章。
 <br/>

 对此，《宝贝儿》导演刘杰23日发文驳斥党媒的报导，并点名文章的作者说，〝你们是官媒，党报，请你们关注一下影片中表现的你们的人民。〞
 <br/>
 <br/>
 在此之前，与杨幂类似的Angelababy，因电视剧《创业时代》也被批演技全无；而范冰冰因逃税案暂无作品上映，在被罚款8.8亿人民币之前，也曾被官媒炮轰演技。
 <br/>
 <br/>
 由于杨幂与范冰冰都属于高收入的知名女星，外界分析认为，杨幂遭中共官媒点名狠批，可能是范冰冰事件的后效应。
 <br/>
 <br/>
