### 【禁闻】世界互联网大会 矽谷高管为何集体缺席
------------------------

<div class="wysiwyg">
 【新唐人2018年11月10日讯】中共在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_浙江.html" target="_blank">
  浙江
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌镇.html" target="_blank">
  乌镇
 </a>
 举办的第五届〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界互联网大会.html" target="_blank">
  世界互联网大会
 </a>
 〞，跟往年相比，今年出席的重量级人物屈指可数，其中美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_矽谷科技公司.html" target="_blank">
  矽谷科技公司
 </a>
 的高管一律缺席。
 <br/>
 <br/>
 中国第五届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界互联网大会.html" target="_blank">
  世界互联网大会
 </a>
 ，11月9号在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_浙江.html" target="_blank">
  浙江
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乌镇.html" target="_blank">
  乌镇
 </a>
 闭幕。
 <br/>
 <br/>
 英国《金融时报》报导，对比往年中共大举炒作苹果和谷歌公司等矽谷领袖的出席，今年有明显的不同。
 <br/>
 <br/>
 今年的互联网大会，除了在会议前才公布与会者名单之外，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_矽谷科技公司.html" target="_blank">
  矽谷科技公司
 </a>
 高管一律缺席。
 <br/>
 <br/>
 总部位于北京的科技市场研究公司〝迈博瑞谘询〞（Marbridge Consulting）董事及总经理马克・纳特金（Mark Natkin）指出，〝外国科技公司可能觉得，在目前的政治环境下，被看成与中国（中共）关系过于密切，并不是一个理想的形象。〞
 <br/>
 <br/>
 旅美中国问题研究人士张健：〝包括这次的进博会，跟这次高科技的会议，都能看到这是一个风向标，我相信美国很多高科技产业，已经考虑和中国进行切割。〞
 <br/>
 <br/>
 这届互联网大会主题，为所谓的〝创造互信共治的数字世界，携手共建网络空间的命运共同体〞。
 <br/>
 <br/>
 旅美中国问题研究人士张健认为，中共所做的一切完全违背互联网精神。
 <br/>
 <br/>
 张健：〝它喊的这种口号是非常荒唐可笑的。中共它不破坏、不窃取、不山寨、不盗取人家的技术，西方的这些国家和企业就已经感到万幸了。没有人再相信中共还会在互联网世界当中，它能起到安定的因素。〞
 <br/>
 <br/>
 张健表示，西方国家和大型科技产业缺席，就是一个态度的彰显，他们应该非常清楚中共的目地。
 <br/>
 <br/>
 张健：〝这么多的证据揭示了中共用各种方式，无论是军方的、民间的、政治的影响，以及对于美国渗透、颠覆，以及用‘千人计划’，让华裔的科学家变成自己的内线，种种行为让所有企业深深的恐惧担忧。〞
 <br/>
 <br/>
 《金融时报》指出，中共官方媒体被告知要〝淡化〞《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 》产业政策。政府官员也不再强调〝千人计划〞——那个引进全球顶级工程与科学专家的举措。
 <br/>
 <br/>
 中国是世界上最大的互联网市场，目前拥有8亿网民。不过，中共当局对网民实施各种网络审查和监管措施，严格程度全球数一数二。
 <br/>
 <br/>
 时事评论员凌晓辉：〝中共是一个人权纪录最恶劣的大国，这个国家利用一切现代科技，对人类的文明和普世价值进行颠覆。由中国这样一个严密监控和打压互联网信息的国家，来举办什么世界互联网大会，这是对人类的一个污辱和讽刺。〞
 <br/>
 <br/>
 中共对互联网的管理政策，直接影响到西方国家在中国市场的发展。时事评论员凌晓辉认为，互联网是中共监控民众的工具，而现在又是处于美中贸易战阶段，所以谁都不愿意为中共〝站台〞。
 <br/>
 <br/>
 凌晓辉：〝外国科技公司可能觉得，在现在的国际环境下，被视为跟中共关系密切，不是一个理想的状态和形象。所以，中共的科技发达对百姓是件坏事，与中共合作就是变相和中共镇压百姓，当人们认识到这一点，自然就会远离它。〞
 <br/>
 <br/>
 总部位于华盛顿的非政府组织〝自由之家〞，上个月底发表〝网络自由度〞报告。报告指出，在全世界范围内，网路正在失去越来越多的自由，人民个人资料遭浮滥搜集，民主也因此而出现倒退。
 <br/>
 <br/>
 报告说，中国是2018年侵犯网路自由最严重的国家。中共政府正在把网络审查和监控技术出口到其它国家。同时，中共正在试图巩固它的互联网领导地位。
 <br/>
 <br/>
 《美国之音》引述〝自由之家主席〞阿布拉莫瓦茨的话报导说，〝民主正在数位时代中挣扎，中国正在海内外强化其审查、监视，以控制讯息的模式。〞而这种〝数位独裁主义〞将危及全球民主。
 <br/>
 <br/>
 采访/常春 编辑/黄亿美 后制/周天
