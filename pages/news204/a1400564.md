### 【禁闻】贸易战下 中共传达文件〝做最坏打算〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 最近传达内部文件，指示要做好〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最坏打算.html" target="_blank">
  最坏打算
 </a>
 〞面对中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 可能引发的问题。专家认为，这显示中共没有信心在二十国集团（G20）峰会上能够和美国达成协议。但中、美贸易纠纷的癥结正是由于〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党的领导.html" target="_blank">
  党的领导
 </a>
 〞，实际上中国民众的利益和美国是一致的。
 <br/>
 <br/>
 《自由亚洲电台》11月22号引述多个信息来源透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 中央最近向全国厅级以上的机关传达文件，指示要做好〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_最坏打算.html" target="_blank">
  最坏打算
 </a>
 〞来面对中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 下可能出现的经济、社会问题。文件中说，预计会有一批企业倒闭，股市、楼市会有波动，并引发社会矛盾。
 <br/>
 <br/>
 中国经济目前正处于下行通道，随着中美贸易战的延烧，中国财政部资料显示，今年前9个月的全国财政收入增幅持续走低、股市持续低迷、房市持续探底。仅上半年就有504万家企业倒闭。同时，中共在国际上也接连受挫，在刚刚结束的亚太经合组织（APEC）峰会上受到孤立，毫无斩获。
 <br/>
 <br/>
 美国总统川普预计11月底赴阿根廷出席二十国集团（G20）峰会，届时将与中国国家主席习近平共进晚餐，讨论两国间的贸易冲突等议题。美国南卡罗莱纳大学艾肯商学院教授谢田分析，目前中共下发这样的文件，说明中共没有信心达成协议。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝主要是中共对G20的谈判达成协议显然不太有信心。因为实际上川普的要求非常的高，非常直接。对中共来说，它接受了，那就意味着它就要放弃对中国经济的控制，放弃中共的政治地位。它不接受的话，这个关税全部上来以后，中国经济也要崩溃 。〞
 <br/>
 <br/>
 时事评论员田园博士认为，贸易战爆发以来中共一直采取对抗，党媒也高调重复有能力、有信心打贸易战。但这份文件显示，当局终于承认了自己的处境。
 <br/>
 <br/>
 时事评论员田园博士：〝经济现实非常清楚。今年这个冬天非常寒冷，但是中共所要经历的经济寒冬要比这个不知道要超过多少倍。所以目前可以看到中共已经陷入了一种比较惊慌的状态，因为它终于认识到了中国的经济在美国以及美国盟友对它的围堵的状况下，根本就坚持不了多久。〞
 <br/>
 <br/>
 《自由亚洲电台》说，这份文件主要强调了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自力更生.html" target="_blank">
  自力更生
 </a>
 ，要全党〝在党中央的领导下，战胜困难〞。
 <br/>
 <br/>
 田园博士：〝目前中国的困局正是因为所谓的‘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党的领导.html" target="_blank">
  党的领导
 </a>
 ’，正是因为中共独裁，正是因为中共在所谓的成为第二大经济体之后，耀武扬威、炫耀自己的武力、炫耀自己的实力造成的。〞
 <br/>
 <br/>
 谢田分析，恰恰是当中国没有了中共对国企、外汇、贸易等方面的控制之后，中、美才有可能达成公平贸易。但操控中国经济的中共利益集团显然不会放手。
 <br/>
 <br/>
 谢田：〝对经济的控制，这是他们的生财之道。横竖都只能死的话，干脆就拉着全中国人民跟它一起陪葬。让中国社会倒退到闭关锁国的年代。这是中共预计可能会发生的，所以它才会要中国老百姓跟它一起吃苦，要自力更生。 〞
 <br/>
 <br/>
 如果持续对抗，当局所谓的〝最坏打算〞，会坏到什么程度？
 <br/>
 <br/>
 田园博士：〝如果没有这个资金的净流入，没有像美国的贸易顺差，还有从欧盟来的贸易顺差，中共它的经济马上会发生巨大的动荡，不仅仅楼市它会支撑不起，如果对外出口崩溃，那么国内消费也崩溃，直接带动的就是经济断崖式崩溃。另外一方面，如果出现巨型的失业潮，它还要面临社会的动荡。那么这是中共最害怕的。〞
 <br/>
 <br/>
 田园博士分析，其实中国民众的利益和美国是一致的，都希望公平贸易、自由经济。《路透社》专栏作家考克斯（Rob Cox）21号的文章也透露，中国民营企业的主管暗中支持川普对中共施加贸易压力 ，期待这能成为刺激大陆经济改革的外部驱动力，以改善中国民营企业经营的环境。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/钟元
