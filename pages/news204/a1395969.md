### 胡锦涛等7大佬接连现身 江泽民传出〝噩耗〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】中美贸易战令北京陷入外忧内患的困境，值此局势敏感之际，中共政坛更显诡异。近来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李瑞环.html" target="_blank">
  李瑞环
 </a>
 等7名中共退休高层接连露面，引发关注。而此时，一向爱出风头的江泽民，却传出病危或更糟的〝噩耗〞。
 <br/>
 <br/>
 近期，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡启立.html" target="_blank">
  胡启立
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李瑞环.html" target="_blank">
  李瑞环
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴邦国.html" target="_blank">
  吴邦国
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴官正.html" target="_blank">
  吴官正
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 等中共退休常委接连以不同方式密集〝露面〞，引发舆论关注。
 <br/>
 <br/>
 10月18日，中共前总书记胡锦涛与现任高层习近平、王岐山等，现身在中共维族高官、人大前副委员长司马义・艾买提遗体送别仪式上。
 <br/>
 <br/>
 同一天，中共前政治局常委胡启立，现身上海最大的集成电路产业投资项目——华力二期12英寸先进生产线投片大会。
 <br/>
 <br/>
 10月15日，中国网媒刊登中共前常委李瑞环的文章，谈自己从政经历的〝看法与说法〞，如为官者〝要少讲空话，多干实事〞等。
 <br/>
 <br/>
 10月11日，中共前国务院总理朱鎔基，现身清华大学经济管理学院顾问委员会会议。照片显示，90岁的朱鎔基身穿西装，脸露笑容，精神不错。
 <br/>
 <br/>
 10月7日，中共海外党媒称，中共前常委吴邦国于〝十一〞期间，在祖籍合肥市肥东县露面。
 <br/>
 <br/>
 再往前推，9月30日，中共最高检察院成立的〝中国反腐败司法研究中心〞公号，刊发两篇中纪委前书记吴官正的文章，大谈他从常委之位退下来的心态。
 <br/>
 <br/>
 9月27日，中共十九大卸任后，一直销声匿迹的前中共政治局常委刘云山，突然出现在河南嵩山少林寺。此前丑闻缠身的少林寺方丈释永信接待了他。
 <br/>
 <br/>
 舆论分析，中共中央卸任领导人因身份特殊，鲜少出现在公众视野，他们的露面，往往被外界揣测是否有政治意涵。由于目前时值中美贸易战白热化，国内经济遭遇严重危机，各种不明朗因素浮现，中共习派、江派退休常委突然接连现身，或许与这波政坛波动有关。
 <br/>
 <br/>
 就在上述中共退休高层纷纷露面之际，一向爱出风头的中共前党魁江泽民却销声匿迹，10月16日，更传江家人紧急集合的消息，可能是因江泽民病危或更糟。
 <br/>
 <br/>
 10月上旬，已有多方消息披露，江泽民身体不断反覆，似乎并不乐观。
 <br/>
 <br/>
 9月30日，中共国务院在北京大会堂举行的十一招待会上，没有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共元老.html" target="_blank">
  中共元老
 </a>
 的镜头，美媒爆料，江泽民、李鹏等人〝集体缺席〞。由于十一招待会早已成为观察中共政坛动向的窗口。有分析认为，元老集体缺席存在多种原因，其中不排除掩盖某元老病危的消息。
 <br/>
 <br/>
 按理说江泽民已经92岁了，无论出现何种情况都是正常，但外界之所以如此关注他的生死，分析认为，主要与江身上背负太多血债且是习近平的头号政敌有关。江一人的生死，将牵扯到整个江家乃至包括曾庆红、刘云山等人在内的庞大江氏派系的存亡。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：戴明）
