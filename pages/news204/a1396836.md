### 新疆181座集中营 批量采购手铐电棍
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】随着中共在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 修建大型〝法外拘留中心〞的证据不断曝光，中共近期改口，承认拘留中心存在，但宣称是用于职业培训的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育营.html" target="_blank">
  再教育营
 </a>
 〞。国际人权界强烈批评中共掩盖真相。
 <br/>
 <br/>
 随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集中营.html" target="_blank">
  集中营
 </a>
 的卫星照片曝光，加上多名证人出面举证，中共最近终于改口，承认新疆存在所谓〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育营.html" target="_blank">
  再教育营
 </a>
 〞。但官方宣称，这是〝职业培训中心〞。
 <br/>
 <br/>
 美国维吾尔人协会主席 伊利夏提：〝我们知道有名有姓的（被关押者），有新疆大学的校长、副校长、十几个教授副教授、讲师。这些人需要职业培训吗？80多岁的人需要职业培训吗？这不是培训中心，这是监狱。〞
 <br/>
 <br/>
 哈萨克人贝利卡，曾被关入新疆的改造中心8个月。他获释后披露，他们被强迫违反宗教信仰吃猪肉、喝酒，还被迫每天唱红歌。
 <br/>
 <br/>
 纽约大学亚美法中心访问学者 滕彪：〝中共一贯相信强制洗脑的作用。中共的目的就是要清洗维吾尔人包括哈萨克人，他们的民族身分，宗教身分。强制他们认同共产党。总的来说，就是一个种族清洗，或者说是一个文化清洗。〞
 <br/>
 <br/>
 法新社查阅了1500份新疆政府公开文件，分析出新疆至少有181座再教育营。
 <br/>
 <br/>
 美国联邦参议员鲁比奥，以新疆和田区的采购清单举例说：〝怎样的‘职业培训中心’，会购买2,768支警棍，550根电枪，1,367幅
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手铐.html" target="_blank">
  手铐
 </a>
 和2,792罐胡椒喷雾？〞
 <br/>
 <br/>
 美国维吾尔人协会主席 伊利夏提：〝首先，被抓进去的人不知道什么时候能放出来；其次，集中营里的人是被强制的；再一个，这些人都没有经过任何司法程序，只是根据共产党官员的鉴定，这些人思想〝可能〞有问题。就是莫须有的罪名。〞
 <br/>
 <br/>
 纽约大学亚美法中心访问学者 滕彪：〝几乎所有都是普通的平民，只是中共给他们贴上‘受到极权思想影响’的标签。中共的藉口是‘反对和清除极端主义、恐怖主义和分裂主义’。但是它的做法，恰恰是极端主义、国家恐怖主义，在客观上导致分裂主义。〞
 <br/>
 <br/>
 日前，《环球时报》主编胡锡进，前往一间集中营拍摄视频。营区内的人正在进行文体活动，看起来气氛轻松。但有证人指出，中共时常用这种方式，掩盖残酷的人权迫害。
 <br/>
 <br/>
 法轮功学员 前中国政法大学人文学院哲学系教授 游兆和：〝我们当时就是因为维护自己的信仰，被他们抓捕，然后就把我送（北京）团河劳教所进行迫害。劳教所固定有记者、家属来参观，（劳教所）就表面上组织点文艺活动，表面上看来歌舞升平的。但是受过酷刑或者被严管的学员，它会给隐蔽起来，外界根本看不到。现在我看新疆集中营的性质是一样的。〞
 <br/>
 <br/>
 联合国专家认为，新疆集中营里关押了约100万人。至今已传出多起死亡案例。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
