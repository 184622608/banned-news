### 湖南12岁男孩吸菸被骂 持菜刀砍死母亲
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南省沅江市.html" target="_blank">
  湖南省沅江市
 </a>
 日前发生一起儿弑母的人伦惨剧，一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_12岁男孩.html" target="_blank">
  12岁男孩
 </a>
 疑似因吸菸被母亲训斥，情绪失控，持菜刀将母亲砍死在卧室。因男孩疑似一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_留守儿童.html" target="_blank">
  留守儿童
 </a>
 ，且案情血腥恶劣，中共官方害怕在社会上造成不良影响，已实施全面舆论管控，禁止民众谈论该案件。
 <br/>
 <br/>
 据自由亚洲报导，12月2日晚上9点多钟，在湖南沅江市泗湖山镇西南村，就读小学6年级的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_12岁男孩.html" target="_blank">
  12岁男孩
 </a>
 吴X康，因在家吸菸被34岁的母亲陈艳发现，母子发生争执，吴X康愤怒中持菜刀对母亲狂砍20多刀致死。
 <br/>
 <br/>
 案发后，吴X康和2岁的弟弟呆在家中，他还装作没事一样，用母亲的手机和亲友联络，还用母亲的手机向老师请假，不去上学。直到翌日，邻居发现有异常，叫来吴X康的外公开门，才发现惨案。
 <br/>
 <br/>
 据了解此案的媒体负责人程先生透露，他的记者了解到，疑凶是一位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_留守儿童.html" target="_blank">
  留守儿童
 </a>
 ，从小到大父母很少管他，都是爷爷奶奶带大，性格有点固执，而且和社会上的人一起又抽烟、又打牌又乱搞。他母亲看到儿子小小年纪抽烟就上前制止，没想到他很反感，就跑到厨房拿菜刀行凶。
 <br/>
 <br/>
 程先生说，现在官方对该案已进行管控，怕再次发生这种事情，怕在社会上造成不良影响。他还透露，因为是留守儿童作案，加上案情太血腥恶劣，以至于留守儿童的问题在当地也成了敏感话题。
 <br/>
 <br/>
 有分析认为，留守儿童问题根源是体制问题，尤其是中共户籍制度的产物，使得农民在农村没法谋生，不得不出门打工，导致留守儿童的产生。
 <br/>
 <br/>
 一位要求匿名的律师称，导致留守儿童出现严重的问题，是中共漠视农民工以及他们家庭基本权利的结果。中共实际上是在用最小的成本榨取民众最大的价值，并且拒绝承担政府应该承担的责任。
 <br/>
 <br/>
 他说，大的是政府的这个政策问题，各地政府都不愿意农民工带他们的孩子来，也不允许农民工的孩子来读书，因为这样他们就要办学校，就要花很多成本，他们不想花这个钱。只要你从20岁干到50岁，干完以后就把你扔了，这样成本最低。这是一种比奴隶制度还坏的制度。
 <br/>
 <br/>
 沅江市政府回应称，该案件还在调查中，并称当地妇联有保护留守儿童权益的工作。
 <br/>
 但记者致电沅江市妇联，对方回复：听说这个事情，但是具体是什么样子的我们也不知道。你可以去找我们当地的宣传部门啦。
 <br/>
 <br/>
 沅江市妇联：那留守儿童这么多，我们也没有去针对每一个留守儿童做一个目前的境况的调查啊，没有统计。那个关心下一代委员会是关工委，不是属于我们的，那是独立的部门，知道吧？
 <br/>
 <br/>
 据官方今年8月发布的消息，目前中国有697万留守儿童。据陆媒报导，很多留守儿童因缺失父母的教育与关爱，或沦为边缘人，或对社会失去信任感而仇视社会。近年频发儿童自杀或杀人事件，显示这个群体陷入了黑暗的心理状态。据报导的统计数据显示，在某些地方，留守儿童已占到未成年人犯罪的70%。
 <br/>
 <br/>
 此外，对于这起儿弑母的伦理惨剧，大陆媒体有另一个版本。根据沅江市教育局的案情介绍，指嫌疑人的父亲常年在市外打工，母亲则在家照顾2个儿子生活。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
