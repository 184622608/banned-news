### 5国家队基金清盘引恐慌 证监会〝辟谣〞语焉不详
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 连续下跌之际，2015年股灾时成立的5支〝国家队〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_救市基金.html" target="_blank">
  救市基金
 </a>
 宣布清盘撤出，再次引起市场恐慌情绪。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共证监会.html" target="_blank">
  中共证监会
 </a>
 称相关报导〝存在曲解〞，但没有指出哪里存在〝曲解〞。
 <br/>
 <br/>
 北京时间周一（10月29日）午后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共证监会.html" target="_blank">
  中共证监会
 </a>
 宣称，有关5支〝国家队基金〞已清盘的报导〝存在曲解〞，并指〝相关机构〞持有的股票并没有减持，反而有所增加。
 <br/>
 <br/>
 但证监会并未做出详细解释，也没有明确所谓〝相关机构〞的具体身份。
 <br/>
 <br/>
 据路透社26日报导，2015年A股股灾时为救市组建的5支〝国家队〞基金，即南方基金旗下南方消费活力基金、华夏基金旗下华夏新经济基金、嘉实基金旗下嘉实新机遇、易方达基金旗下易方达瑞惠和招商基金旗下招商丰庆A，当天全部刊登出召开基金持有人大会的公告，拟结束基金运作。
 <br/>
 <br/>
 公告指，未来一个月将以通讯方式表决终止基金合同的议案，基金合同终止后将进行财产清算。
 <br/>
 <br/>
 路透报导引述这些基金的季报指，当年规模各400亿人民币的这5大〝国家队〞基金，今年一季度均已被赎回70%份额，截至9月底更都已不足两亿元，已处于空仓状态。
 <br/>
 <br/>
 报导引述业内人士分析，随着国有资金撤出，这些基金的净值规模和持有人数量已不能满足相关法规要求，清盘是很正常的事。
 <br/>
 <br/>
 而陆媒报导称，这5支基金〝成功躲过了股市暴跌〞。国有资金的统一退出，显示〝国家队基金〞成立三年，〝已经完成历史使命〞。
 <br/>
 <br/>
 不过，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 跌跌不休之际，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_救市基金.html" target="_blank">
  救市基金
 </a>
 〝完成使命〞的时机显然不合时宜。
 <br/>
 <br/>
 中美贸易战爆发以来，中国股市一直弥漫恐慌情绪。日前许多上市公司股价已跌近平仓线。虽然中共官方接连喊话救市，但很难推出有效措施，股市短暂上扬之后又开始继续跌势。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
