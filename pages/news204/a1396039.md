### 川普点评中国股市：类似1929年美股大崩盘
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】中美贸易战开打以来，中国股市接连暴跌，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 经常用它来说明中共经济陷入危局。日前，川普再次提及这个话题，称自己并不想让中国衰退，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市暴跌.html" target="_blank">
  中国股市暴跌
 </a>
 很象1929年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市大崩盘.html" target="_blank">
  美国股市大崩盘
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 日前在接受哥伦比亚电视台《60分钟》节目访谈时，谈到了中美贸易战。他说，他已经对2,500亿美元中国商品加征了关税，并在考虑对更多的中国商品征税。
 <br/>
 <br/>
 川普表示，他〝同中国的习主席有很好的化学反应〞，北京也想再谈判，但美国〝不能继续让中国每年通过贸易和其他形式从美国拿走5,000亿美元〞。
 <br/>
 <br/>
 当被问到是否在试图将中国推向衰退时，川普说：〝不，不是，虽然他们的股市4个月下跌了32%，犹如1929年（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国股市大崩盘.html" target="_blank">
  美国股市大崩盘
 </a>
 ）。〞
 <br/>
 <br/>
 川普接着说：〝我不想这样。我想要他们同我们谈判一个公平的协议。我想要他们开放市场，象我们的市场一样的开放。〞
 <br/>
 <br/>
 川普还说，美国肯定将在中美这场〝小冲突〞中获胜。
 <br/>
 <br/>
 川普提到的1929年美国股市大崩盘，是指纽约证券交易所的崩盘事件。这次事件沉重打击了投资者的信心，对于美国经济的影响持续了数十年之久。
 <br/>
 <br/>
 而中国股市连续数月的暴跌，也正在令中国投资者们陷入极度悲观。日前，中共经济高官接连喊话，表示将出手救市，才稍微提振一点信心。
 <br/>
 <br/>
 日前，中共证监会主席刘士余与股民座谈，声称〝春天已经不远了〞。而有海外媒体形容，台下听众们的表情都已是〝生无可恋〞。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
