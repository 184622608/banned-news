### 河北化工厂附近发生大爆炸 数十死伤
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】当地时间周三（11月28日）零点41分，河北省张家口市的一间化工厂附近发生爆炸，造成至少22人死亡，22人受伤。
 <br/>
 <br/>
 11月28日零点41分，河北张家口市桥东区〝河北盛华化工有限公司〞附近发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸事故.html" target="_blank">
  爆炸事故
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大爆炸.html" target="_blank">
  大爆炸
 </a>
 导致50多辆汽车起火燃烧，其中还包括数十辆卡车，爆炸引发的滚滚黑烟升入夜空当中。到目前为止，爆炸事故至少造成22人死亡，22人受伤。
 <br/>
 <br/>
 据一位不愿透露姓名的官员表示，一辆等在化工厂外运送化学品的卡车发生爆炸，并且引发了一系列连锁爆炸反应，当地民众对爆炸是否会造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_有毒化学物质.html" target="_blank">
  有毒化学物质
 </a>
 泄漏表示担忧。
 <br/>
 <br/>
 新唐人记者李博文综合报导
</div>
