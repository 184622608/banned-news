### 范冰冰案蝴蝶效应：问责无锡税务 影视板块暴跌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】范冰冰涉嫌逃税案经过3个多月的调查，于10月3日公布结案，日前中共有关当局进一步对涉及此案的无锡市税务局一系列人员进行了追责处理。同时，外界注意到，自范冰冰被查以来，中国股市上影视板块的股价普遍暴跌，损失最惨重的影视公司跌幅逾60 %，中共官方近日更公布了勒令中国娱乐圈〝自首〞的时间表。
 <br/>
 <br/>
 据中共官媒新华社10月8日的报导，中国女星范冰冰逃税案件结束后，在中共国家税务总局的〝责成〞下，江苏省税务局对在范冰冰偷逃税案件中管理不力、负有领导责任的相关单位和人员进行了问责，并责令无锡市税务局做出检查并整改。
 <br/>
 <br/>
 问责的结果是：给予现无锡市税务局局长丁源行政警告处分；现无锡市税务局总会计师李青行政记过处分；现无锡市滨湖区税务局副局长钟小新行政记过处分，并免去其现任领导职务；现无锡市梁溪区税务局副局长沈治国行政记过处分；现无锡市税务局干部许崇金诫勉谈话处理。还有其他一些有关连的责任人也受到了相应处理。
 <br/>
 <br/>
 就在10月8日这天，中国大陆股市上的影视板块一早就大幅低开，之后更普遍下跌。截至午间收盘，28只影视动漫股中，只有华策影视一度逆市上涨，但至下午3时亦止升下跌1.14%。陆媒认为，这是女星范冰冰逃税被重罚而引起的〝蝴蝶效应〞，并有媒体在报导中称，影视圈多年来出现的泡沫，在范冰冰被调查的这3个月中已被〝完全挤掉〞。
 <br/>
 <br/>
 据《每日经济新闻》引述Choice数据报导，在过去的3个月期间，除了仍在停牌的万达电影外，其他所有和影视相关公司的股价全部缩水。其中其股价与3个月前相比下跌了60.62%；而中南文化、文投控股、唐德影视、奥飞娱乐、慈文传媒的股价跌幅均超过了50%的大关；受影响较轻的北京文化、华策影视，跌幅也分别为12.38%和12.43%。
 <br/>
 <br/>
 有陆媒评论称，影视板块曾是A股市场上的〝明星板块〞，深受投资者的热捧，各路影视娱乐公司曾纷纷筹划IPO（首次公开发行股票，Initial Public Offerings）上市或登陆新三板挂牌，一些非影视公司也跨界并购中小型影视娱乐公司后将其装入上市公司以提升公司的估值，进行资本套利游戏。但如今华视娱乐、新丽传媒、开心麻花、和力辰光等影视公司都中止了IPO，嘉行传媒和乐华娱乐也从新三板摘牌。
 <br/>
 <br/>
 今年6月，已感受到〝风雨欲来〞的光线传媒董事长、总裁王长田在上海国际电影节上就曾哀叹说：〝资本在撤离市场，整个社会的不信任，导致影视公司普遍的融资困难。影视行业的好日子快结束了。〞
 <br/>
 就在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰案.html" target="_blank">
  范冰冰案
 </a>
 的结果正式出炉的前一天（10月2日），中共国家税务总局曾发布一份通知，要求各地税务机关通知本地区影视制作公司、经纪公司、演艺公司、明星工作室等影视行业企业和高收入影视从业人员，从2018年10月10日起对2016年以来的申报纳税情况进行 〝自查自纠〞。
 <br/>
 <br/>
 据了解，这个针对中国娱乐圈高收入从业人员订出的〝自首〞时间表共分为三个阶段：
 <br/>
 <br/>
 其一，对于那些在2018年12月底前〝自查自纠、主动补缴税款的〞，可免予行政处罚，不予罚款。
 <br/>
 <br/>
 其二，从2019年1月至2月底，税务机关根据纳税人自查自纠等情况，有针对性地提醒相关纳税人〝进一步自我纠正〞。对经提醒自我纠正的纳税人，可从轻、减轻行政处罚；对违法情节轻微的，可免予行政处罚。
 <br/>
 <br/>
 其三，从2019年3月至6月底，税务机关需结合自查自纠、督促纠正等情况，对拒不纠正的影视行业企业及从业人员开展重点检查并〝严肃处理〞。
 <br/>
 <br/>
 此外，该通知还声称，在此过程中如果发现税务机关和税务人员存在违法违纪问题，或者出现大范围偷逃税行为而且税务机关和税务人员未履职的，也要查处。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：秦锋）
