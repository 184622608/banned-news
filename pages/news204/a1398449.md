### 【禁闻】 德媒：美国下一个制裁目标或是中国
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】日前，德国媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 可能是继
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 之后, 美国将要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 的目标。这种可能性有多大呢?  又会采取什么方式制裁中共？请看专家的分析。
 <br/>
 <br/>
 据〝德国之声〞11月2号报导，世贸组织前总干事拉米(Pascal Lamy)日前在巴黎召开的一次国际投资会议上指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 方面必须为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 进一步升级做好准备。
 <br/>
 <br/>
 拉米说，面对美国即将恢复的对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 ，许多欧洲企业在权衡利弊之后，不得不撤出伊朗市场；相似的美国策略也有可能日后针对中国，欧盟各成员国政府必须要做好思想准备。〝如果美国决定更进一步，不仅仅用贸易手段来打击中国，而是像现在针对伊朗那样动用制裁措施，欧洲就会遇到大麻烦。〞
 <br/>
 <br/>
 法国圣日尔曼．昂莱政治学院教授 张伦：〝我的看法暂时不会，但是中国的个别企业和伊朗发生商贸关系的话，可能会导致美国对这些相关企业的制裁，但是对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 本身整体的像对伊朗那样的制裁，我觉得暂时是不会的。〞
 <br/>
 <br/>
 美国作为国际警察，能够用来制裁的措施很多，除了军事、外交、金融等方式外，国际宗教自由法、全球人权法案和国际紧急权力法都是制裁独裁国家和人权恶棍的工具。
 <br/>
 <br/>
 为了阻止朝鲜的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核讹诈.html" target="_blank">
  核讹诈
 </a>
 ，美国当时制裁了帮朝鲜
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗钱.html" target="_blank">
  洗钱
 </a>
 的澳洲银行, 之后，所有的银行包括中共的银行不得不和朝鲜进行切割，最后迫使朝鲜与国际金融的联系中断，金正恩不得不坐下来和美国谈判。
 <br/>
 <br/>
 旅美时政评论员唐靖远：〝中共在贸易问题上继续要坚持过去那种靠偷抢致富发家的途径来和美国对抗的话，美国对中共实施制裁是迟早的事情，但是目前这个阶段暂时还没有具备条件。〞
 <br/>
 <br/>
 实际上，美国对中共零星的制裁一直在进行，一些为朝鲜洗钱的公司、和俄罗斯勾结的中共军官都已经受到了制裁。
 <br/>
 <br/>
 最近，美国要求中共媒体注册为外国代理人，也被认为是制裁中共的方式之一。
 <br/>
 <br/>
 9月份，中共《中国日报》花老百姓的血汗钱在川普的票仓爱荷华州购买《德梅因纪事报》四个版面做广告，攻击川普。也被外界解读为给美国制裁中共提供口实，因为美国制裁俄罗斯的理由就是俄罗斯干预美国选举。
 <br/>
 <br/>
 唐靖远：〝更深刻的这个层面上讲，中共对宗教信仰的迫害，包括对新疆维族人这种大规模的灭绝式的迫害，其实这个才是更值得对中共实施制裁的理由。〞
 <br/>
 <br/>
 〝德国之声〞认为，美、加、墨自贸协定中规定的，任何一个协定签字国如果与〝非市场经济国家〞签署贸易条约，其他签字国就可以在六个月内退出协定、另立新约，是在针对中共。因为外界认为中国已经被公认为〝非市场经济国家〞。
 <br/>
 <br/>
 文章说，这一条款可能被复制到正在谈判中的美欧自贸协定，从而阻止欧盟与北京的贸易合作。
 <br/>
 <br/>
 美国哥伦比亚大学政治学博士李天笑认为，川普选择用经济来对中共下手是最好的选择。
 <br/>
 <br/>
 美国哥伦比亚大学政治学博士李天笑：〝经济这东西川普是最拿手的，从关税不断的极限施压，过程中步步升高，这个策略是很见效的，如果说明年1月对全部中国出口美国的商品进行关税制裁，整个的中国经济要走到崩溃的边缘了。这个对中共政权是一个非常大的打击和压力。〞
 <br/>
 <br/>
 旅美时政评论员唐靖远认为，因为关税制裁达到的效果比较滞后，川普采取对中共军方高官李尚福制裁、对晋华这类技术公司制裁等，更加立竿见影，更容易起到施加压力的作用。
 <br/>
 <br/>
