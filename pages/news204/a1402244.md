### 【禁闻】华为CFO加拿大被抓 面临美国引渡
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】中国华为公司副董事长兼首席财务官（CFO）孟晚舟（Wanzhou Meng）传出被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 逮捕的消息。她因涉嫌违反美国对伊朗的贸易制裁，有可能被引渡到美国。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 《环球邮报》12月5号报导，应美国执法部门的要求，加拿大逮捕了华为技术有限公司（Huawei Technologies Co.）首席财务长孟晚舟，她涉嫌违反美国对伊朗的贸易制裁。
 <br/>
 <br/>
 孟晚舟是华为创始人任正非的女儿，她同时还兼任华为的副董事长。
 <br/>
 <br/>
 加拿大司法部的一名发言人表示，孟晚舟12月1号在温哥华被捕，美国政府正寻求引渡她。她的保释听证会初步定在7号举行。因孟晚舟申请的〝禁止报导令〞有效，当局目前无法提供更多详情。
 <br/>
 <br/>
 消息传出后，华为一度表示这是不实报导。但12月6号华为官方微博发声明说，孟晚舟的确被抓，并且面临纽约东区未指明的指控，因为公司得到的具体指控信息非常少，所以声明表示：相信〝加拿大和美国的法律体系，最终将得出公正的结论。〞
 <br/>
 <br/>
 美国商务部和财政部外国资产管制办公室分别在2016年6月和12月，向华为发出行政传票，调查华为是否违反美国贸易禁令向叙利亚、伊朗、朝鲜等流氓国家提供美国技术。今年4月25号，《华尔街日报》援引知情者的消息报导说，美国司法部也在调查，华为是否违反美国对伊朗的贸易制裁。
 <br/>
 <br/>
 旅美中国问题观察人士张健：〝像这样的一个公司在美国受到各个职能部门的调查，这种现象首先就比较少。这足以证明了，华为在美国引发了重大的安全问题。美国政府有强大的理由和证据，来证明华为涉嫌了向伊朗这样的国家，提供了它不应该提供的东西。〞
 <br/>
 <br/>
 旅美中国问题观察人士张健认为，美国要求加拿大协助抓捕孟晚舟，应该是在证据上有突破。
 <br/>
 <br/>
 张健：〝孟晚舟她在华为是CFO，是负责财务的。美国之所以要抓CFO，就是明确的告诉华为无论是从它的营运面上，还是从它的资金链，还有可能涉嫌到其它的问题上，（美国的调查）它都有一个极大的证据上的突破。否则的话它一定不会先抓CFO。〞
 <br/>
 <br/>
 这一消息在微博上也引发关注，出现各种猜测的声音。
 <br/>
 <br/>
 加拿大华裔作家盛雪：〝又是‘美国的阴谋’了，又是‘川普在给这个中国90天的宽限相反的一手’等等。拿各种各样的阴谋论来解读这件事情是毫无意义的。因为不管是美国还是加拿大都是法治国家。这样的逮捕行动如果没有确凿的证据，在这样的法治国家是不可能实施这种行动的。〞
 <br/>
 <br/>
 中共驻加拿大使馆12月6号发表措辞强硬的声明，声称〝逮捕一个没有违反任何美、加法律的中国公民〞，是严重侵犯人权。并要求美国和加拿大所谓〝立即纠正错误做法，恢复孟晚舟女士的人身自由。〞
 <br/>
 <br/>
 盛雪：〝那么这种喊话是完全没有意义的，而且也正好证明了，中共驻外的这些使节，他们是对法律一窍不通。因为既然是逮捕行动，那么一定会最终走法律的程序。〞
 <br/>
 <br/>
 在此之前，中国另一家电信设备巨头〝中兴通讯〞公司也因违反美国向伊朗的禁售令，而被美国商务部制裁。但张健认为，现在涉及到有军工背景的华为，当局可能采取和中兴事件上不同的态度。
 <br/>
 <br/>
 张健：〝在中兴的问题上，中共采用了非常低姿态的身段。但是对于华为来说，首先（驻加使馆）推出的这个版本我们看到是比较高姿态的。华为拥有中共官方的背景，尤其是军方的背景。涉嫌到了亚太、欧美、非洲很多地方的军事和经济利益。所以中共在这次上可能和面对中兴是不一样。〞
 <br/>
 <br/>
 路透社报导，孟晚舟被捕消息传出的当天，英国电信集团（BT.L）表示，正在将华为的设备从现有的3G和4G移动网路的核心中移除，而且下一代网络的核心部分也不会使用华为产品。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/李沛灵
