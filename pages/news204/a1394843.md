### 贸战致大豆紧缺 中共〝奇招〞强令4亿头猪节食
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】中美贸易战中，中共对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 加征关税，为打击川普〝票仓〞不惜代价，结果令自身损失惨重。中国大豆价格飙升，大豆加工业也受到致命打击。为弥补大豆巨量缺口，北京日前祭出〝奇招〞，拟降低
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪饲料.html" target="_blank">
  猪饲料
 </a>
 中粗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蛋白质含量.html" target="_blank">
  蛋白质含量
 </a>
 的标准。
 <br/>
 <br/>
 香港《苹果日报》10月10日报导，中国饲料工业协会日前公开〝征求业界意见〞，打算降低
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪饲料.html" target="_blank">
  猪饲料
 </a>
 中粗蛋白质（crude protein）的最低含量，同时加设上限。猪只越重，所需粗蛋白的下限越低。此外，蛋鸡与肉鸡饲料的所需粗蛋白，可能也会降低。
 <br/>
 <br/>
 中国是全世界最大猪肉生产国与消费国，进口黄豆是加工猪饲料的主要来源。中共新华社8月曾指出，通过降低动物饲料中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蛋白质含量.html" target="_blank">
  蛋白质含量
 </a>
 等方法，中国今年可减少进口黄豆超过1,000万吨。
 <br/>
 <br/>
 中国饲料工业协会〝征求意见〞的声明，也印证了官方的意图。声明的引言称：〝中国是饲料资源短缺的国家，蛋白质饲料长期依赖进口，成为制约饲料工业和养殖业发展的瓶颈。〞
 <br/>
 <br/>
 声明中称，〝科学研究〞证明，〝饲料中粗蛋白和磷的水准可显著降低〞，蛋白质的不足可〝通过添加氨基酸和酶制剂来弥补〞。
 <br/>
 <br/>
 报导略带调侃意味指，中美贸易战越打越烈，中国4亿头猪成为〝受害者〞，或将被迫改变饮食习惯。
 <br/>
 <br/>
 大陆网民担心，猪只缺少蛋白质很难长肉，猪农可能被迫暗中喂食其他药物，甚至向猪只灌水，以后市场上的大陆猪肉可能会有更多脂肪。
 <br/>
 <br/>
 也有网友表示，北京又开始宣传〝自力更生〞，强制猪〝节衣缩食〞这一套，很快就会用到人身上。
 <br/>
 <br/>
 彭博社报导指，中共对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 加征关税，而美国是中国最大的大豆供应国之一。虽然中国已转向巴西、阿根廷和印度购买更多大豆，但来不及应付今年第4季度的需求，届时中国大陆的豆油加工厂恐怕会〝断粮〞。因为南北半球季节轮替的原因，美国在第4季度会成为主要卖家。
 <br/>
 <br/>
 在预期短缺的心理作用下，中国黄豆今年价格暴涨26%，黄豆期货周二（9日）上涨1.1%，达每吨3487元人民币，创下2014年7月以来新高。
 <br/>
 <br/>
 中共从7月6日开始对美国农产品加征关税，已导致大陆多家大豆加工厂减产或停产，并有不止一家粮食公司倒闭，其中包括中国最大的民营大豆进口商山东晨曦集团。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
