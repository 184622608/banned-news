### 江泽民老家再震荡 江苏省副省长缪瑞林落马
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】江泽民老家所在地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江苏官场.html" target="_blank">
  江苏官场
 </a>
 再震荡。11月15日，中纪委国家监委官网发布消息，江苏省副省长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_缪瑞林.html" target="_blank">
  缪瑞林
 </a>
 涉嫌严重违纪违法，接受纪律审查和监察调查。
 <br/>
 <br/>
 11月15日，陆媒称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_缪瑞林.html" target="_blank">
  缪瑞林
 </a>
 最后一次露面是10月30日，赴江苏镇江视察生态环境工作。
 <br/>
 <br/>
 现年54岁的缪瑞林是江苏如东人，仕途一直未离开江苏省内。他早年从江苏农学院农学专业毕业后，进入农业系统工作20年，曾任江苏省农林厅农业局副局长、省农业资源开发局局长等职。
 <br/>
 <br/>
 2004年，缪瑞林调任地方，主政宿迁8年，历任常务副市长、代市长、市长、市委书记。2013年1月，他跻身省部级出任江苏副省长。
 <br/>
 <br/>
 2013年10月，时任南京市长季建业落马，一个多月后，缪瑞林补缺南京市长职务。
 <br/>
 <br/>
 2015年1月，与缪瑞林党政搭档了一年时间的江苏省委常委、南京市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨卫泽.html" target="_blank">
  杨卫泽
 </a>
 落马。
 <br/>
 <br/>
 同年3月，已调任云南担任省委副书记的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_仇和.html" target="_blank">
  仇和
 </a>
 被中纪委宣布落马。2002年至2006年，缪瑞林任职宿迁市委副书记与仇和正担任宿迁市委书记、宿迁市人大常委会主任。
 <br/>
 <br/>
 对于杨卫泽与仇和落马，缪瑞林不但没有受影响，反倒于2018年初，再度转任江苏副省长职务。
 <br/>
 <br/>
 而此次缪瑞林落马因何原因，官方暂未公布详情。
 <br/>
 <br/>
 江苏是江泽民老家所在地，江派重要势力范围，也是周永康的老家、江派利益集团的大窝点。
 <br/>
 <br/>
 近年来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江苏官场.html" target="_blank">
  江苏官场
 </a>
 持续震荡。江苏原南京市市长季建业、南京市委书记杨卫泽、江苏省委前常委兼秘书长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵少麟.html" target="_blank">
  赵少麟
 </a>
 、江苏常务副省长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李云峰.html" target="_blank">
  李云峰
 </a>
 等，多名省部级官员先后落马。
 <br/>
 <br/>
 2014年10月11日，已经退居二线八年的赵少麟落马。
 <br/>
 <br/>
 2013年10月，江苏原南京市市长、前扬州市委书记季季建业被调查。2015年4月7日，建业被判15年；其曾主政扬州八年，被指是江泽民的心腹、江老家的〝大管家〞。
 <br/>
 <br/>
 被指是周永康马仔的江苏省委前常委、南京市委前书记杨卫泽是2015年落马的〝首虎〞。杨卫泽被抓的同时，其秘书、南京市委办公厅副主任张志炎和杨卫泽的妻子亦被带走调查。
 <br/>
 <br/>
 杨卫泽落马半年后，张志炎因涉嫌受贿罪亦被监视居住。2015年7月底，其被刑事拘留，8月中旬被执行逮捕；2016年12月14日，杨卫泽犯〝受贿罪〞一审被判处有期徒刑12年半。
 <br/>
 <br/>
 据报导称，江派人物、江苏一把手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗志军.html" target="_blank">
  罗志军
 </a>
 与季建业及蒋宏坤均关系密切。罗志军因卷入周永康案被调查，他和南京市委书记杨卫泽给周永康的兄弟和妹妹以及儿子周斌输送了大量利益，目的是在薄熙来夺权后，罗做公安部长，周永康对此做了承诺。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：李泉）
