 <br/>
 <br/>
 2. 杭州P2P金融难友省府

 <br/>

 10月25号，杭州十多个P2P平台的投资受害者到浙江省政府维权，全部被警察拉上大巴车,  押送到黄龙体育中心，然后分平台遣散。

 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">

 </a>

 <br/>
 <br/>
 〝六四天网〞创办人黄琦的代理律师刘正清，10月23号再次到四川省绵阳看守所会见，黄琦披露，看守所人员联同医生，伪造血压数据，掩盖他病危的状况。办案单位还以〝不讲政治〞为由否决救治方案。
