### 朝鲜3架货机同赴俄罗斯 疑为金正恩访俄运〝粮草〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】俄媒8日报导，从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平壤.html" target="_blank">
  平壤
 </a>
 起飞的三架朝鲜高丽航空公司的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞机.html" target="_blank">
  飞机
 </a>
 ，7日降落在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 符拉迪沃斯托克，向该地区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_运送货物.html" target="_blank">
  运送货物
 </a>
 。分析认为，此举或为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 访俄做准备。韩国总统文在寅日前透露，金正恩预计很快会访俄。
 <br/>
 <br/>
 10月8日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 卫星通讯社援引消息人士的话披露，三架朝鲜高丽航空公司的伊尔-76运输机从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平壤.html" target="_blank">
  平壤
 </a>
 起飞后，于7日降落在俄罗斯符拉迪沃斯托克，向该地区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_运送货物.html" target="_blank">
  运送货物
 </a>
 ，这些货物将从那里被转运到俄罗斯各地。
 <br/>
 <br/>
 根据符拉迪沃斯托克机场消息，高丽航空的三个航班7日从平壤抵达该机场，三架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飞机.html" target="_blank">
  飞机
 </a>
 都是伊尔-76。
 <br/>
 <br/>
 符拉迪沃斯托克运输机构的消息人士说：〝飞机抵达符拉迪沃斯托克，但是货物是经那里中转。〞
 <br/>
 <br/>
 另一名接近朝鲜驻符拉迪沃斯托克领事馆的消息人士说，中转运输的货物已经离开那里前往莫斯科。但消息人士都没有透露，飞机上搭载的是何种货物。
 <br/>
 <br/>
 值得注意的是，伊尔-76运输机曾在今年6月美朝首脑会晤之前，将朝方后勤物资运抵新加坡并返回平壤。
 <br/>
 <br/>
 而此次三架朝鲜高丽航空公司的飞机，前往俄罗斯运送货物，也被认为或是为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 访问俄罗斯运送货物。
 <br/>
 <br/>
 《联合早报》10月8日报导，韩国总统文在寅在当日上午出席内阁会议时透露，朝鲜领袖金正恩预计很快就会访问俄罗斯，中国国家主席习近平也计划前往朝鲜访问。
 <br/>
 <br/>
 根据韩联社当日报导，朝鲜外务省副相崔善姬8日在莫斯科，与俄罗斯负责亚太事务的副外长莫古洛夫举行闭门会谈。报导分析，双方除了谈论朝核问题，可能还讨论了金正恩访俄的相关议题。
 <br/>
 <br/>
 俄媒9月8日就曾报导，正在朝鲜访问的俄罗斯联邦委员会主席马特维延科（Valentina Ivanovna Matvienko）证实，金正恩将访问俄罗斯，访问时间及地点将经外交渠道协商。
 <br/>
 <br/>
 马特维延科当时还称，由于金正恩日程繁忙，所以不出席在俄罗斯符拉迪沃斯托克（海参崴）举行的东方经济论坛。
 <br/>
 <br/>
 今年以来，金正恩已经三次访华与习近平会面，6月份还曾前往新加坡与美国总统川普（特朗普）会晤。美国国务卿蓬佩奥日前访问东亚四国。美国国务院表示，蓬佩奥访朝时与金正恩讨论了两国领导人第二次会晤可能的地点和日期。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：赵云）
