### 【禁闻】川习会在即 川普：不会推迟提升关税
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】白宫11月27号表示，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 将在阿根廷布宜诺斯艾利斯举行的二十国集团（G20）会议上，与7位世界领导人进行双边会晤。其中，与中国国家主席习近平的会面安排在12月1号晚。这次双方是否能达成贸易协定，外界都在关注。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 与中国国家主席习近平，12月1号晚，将在阿根廷举行的G20峰会期间会晤，预计双方将就中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的解决方案摊牌。
 <br/>
 <br/>
 《华尔街日报》11月28号报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 官员表示，当务之急是说服美方暂停在明年1月1号将2,000亿美元中国进口商品的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 税率从10%上调到25%的计画。中共还希望美国能放弃其它针对中国产品的关税行动，并取消对中国钢铝产品征收的惩罚性关税。
 <br/>
 <br/>
 而川普则对《华尔街日报》表示，他预计会持续推进这个提高关税税率的计画。川普还说他〝极不可能〞接受北京的要求，推迟提高关税。因为在美、中之间还没有达成协议之前，如果美国政府就停止提高关税税率等政策，中共将会继续敲诈美国，就像过去那样继续任意妄为。
 <br/>
 <br/>
 旅美时事评论员唐靖远分析，这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 ，双方达成协议的可能性非常低。
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝第一个原因是中共的体制问题。如果说中共它真的全盘答应了川普的要求，它必然会触及到中共经济体制的结构性改变。而中共经济体制结构性改变，按照他们自己的高层智库研究的一个结果就是，最多两年左右的时间，就很可能引发中共在政治体制方面的这种变化。〞
 <br/>
 <br/>
 唐靖远表示，其他的原因还包括，美国在最新公布的301调查报告更新版中，用实例指出中共不单没有改善不公平贸易行为，反而变本加厉，客观上显示出：中共对和美国达成贸易协议，缺乏基本诚意。另外，中共在刚结束的APEC会议上，也显示出强硬的态度。
 <br/>
 <br/>
 唐靖远：〝要想指望在这么短的一个时间之内，让中共能够发生一个非常巨大的转变，去符合美国提出的要求，我觉得基本是不太可能的事情。〞
 <br/>
 <br/>
 旅美时事评论员郑浩昌表示，中方应对贸易战的说法一直是：没有使用不公平贸易手段，而且责怪川普有意刁难。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝北京这种态度的潜台词就是说，我没错。在这种态度下面，很难相信习近平会作出实质性的让步，最终的结果可能也就是一个形式意义上的妥协。现在的情况，要北京真正转变态度相当难，但是这个却正是贸易战解套的关键。〞
 <br/>
 <br/>
 川普还对《华尔街日报》表示，如果这次他和习近平不能达成协议，将会对另外价值2,670亿美元的中国商品加征关税，税率将会是10%或25%。
 <br/>
 <br/>
 川普还提到，下一轮关税也可能涵盖从中国出口到美国的〝苹果〞手机和笔记本电脑。这一消息造成苹果公司股价在市后交易时段下跌近1.6%。
 <br/>
 <br/>
 唐靖远：〝如果中美之间不能够达成协议，我觉得最直接的一个结果，就是很有可能中美之间会真正的演变成为一种全面冷战。美国方面不但可能继续升级关税战，而且很有可能针对中共发动类似于技术封锁战，金融制裁战，反商业间谍战等等。〞
 <br/>
 <br/>
 白宫首席经济顾问库德洛星期一在白宫记者会上表示，川普总统对和中国达成协议持开放态度。不过需要满足公平、互惠的某些条件，例如，知识产权盗窃问题、强制技术转让问题、重大的关税与非关税壁垒问题，以及所有制的问题都必须要解决。
 <br/>
 <br/>
 但对于能不能和美方达成协议，中共似乎也信心不足。《自由亚洲电台》此前曝光，中共中央已经向全国厅级以上的机关传达文件，要求做好〝最坏打算〞来面对中美贸易战下可能出现的经济、社会问题，并预计会有一批企业倒闭，股市、楼市会有波动，并引发社会矛盾等。
 <br/>
 <br/>
 采访/常春 编辑/尚燕
