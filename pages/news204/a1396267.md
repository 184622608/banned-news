### 【禁闻】新西兰政治献金丑闻 中共渗透引关注
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】10月16号，新西兰前国会议员罗斯召开记者会，揭发国家党党魁布里奇斯接受华人社区领袖10万纽币的政治献金，并试图通过特别操作隐瞒真相。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丑闻.html" target="_blank">
  丑闻
 </a>
 爆光后，再次引发国际社会对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_渗透.html" target="_blank">
  渗透
 </a>
 外国政坛的广泛关注。
 <br/>
 <br/>
 被新西兰在野党〝国家党〞高层逐出国会的议员罗斯（Jami-Lee Ross），在10月16号举行的新闻发布会上披露，国家党党魁布里奇斯（Simon Bridges）今年5月参加华人领袖张乙坤家中的晚宴和其后筹款活动后，接受了张乙坤10万纽币的政治献金。
 <br/>
 <br/>
 布里奇斯还指使张乙坤将这笔资金分成多份较小的金额，以规避1500元的申报门槛，并隐藏捐赠来源。
 <br/>
 <br/>
 为佐证自己的话，罗斯还公布了一段他秘密录下的与布里奇斯的通话录音，并在新闻发布会的第二天将录音交给警方。
 <br/>
 <br/>
 时事评论员凌晓辉：〝这是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 大外宣计划的一个部分，中共大外宣计划包括用金钱去贿赂和腐蚀其他国家的政府特别是民主国家的立国之本。它的目的是要赤化所有其他西方国家，用共产主义意识形态来控制和操控其他的国家。〞
 <br/>
 <br/>
 录音中，布里奇斯除了提到十万元的捐款，还谈及吸收更多华人作为选举候选人的可能性,  准备将张乙坤推荐的华裔人选Colin Zheng列入候选名单。
 <br/>
 <br/>
 滕彪：〝首先在新西兰已经披露的这些情况来看，它对新西兰的外交尤其是对中国的政策，影响是非常大的，一些跟中共关系非常密切的人，他们成为议员，或者是在政府里边有非常重要的位置，因为他们对中国更熟悉更了解往往成为一些新西兰的官员访华的一些桥梁，然后也会对于新西兰的外交政策，对中国的政策施加非常重要的影响。〞
 <br/>
 <br/>
 这场政治献金
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_丑闻.html" target="_blank">
  丑闻
 </a>
 ，引发国际媒体关注，也让新西兰成为了最新一个爆出华商政治献金丑闻的国家。
 <br/>
 <br/>
 凌晓辉：〝有一点我想提醒，海外众多华人仍然生活在中共意识形态的影响和控制之下，许许多多的华人，由于长期的意识形态被中共控制，而恐惧中共的淫威，而选择继续拥抱中共，但这部分人，其实是被中共欺骗和迫害且至于危险境地而不自知。〞
 <br/>
 <br/>
 新西兰亚裔作家、种族和民族问题专家莫志明在《新西兰先驱报》发表评论文章说，实际上与中共政府过分亲近的人，都不认为华商向新西兰政客献上巨额捐款有什么不对，因为多年来国家党和工党高层人士都接受了与中共有关的献金。
 <br/>
 <br/>
 在这方面，中国问题专家布莱迪（Anne-Marie Brady）教授一直在大声疾呼，为此布莱迪曾遭受恐吓和盗贼袭击。
 <br/>
 <br/>
 滕彪：〝布莱迪教授她的报告以及一系列的文章，从去年开始影响很大，所以中共对这个人是非常恼火以至于派特务去恐吓，到她的家和她的办公室偷偷的去侵入，而且拿走了她的工作的电脑等等。〞
 <br/>
 <br/>
 莫志明表示，寒蝉效应伤害的是华人，中共越是试图在更广泛的新西兰华人社区插入自己的计划，同时又声称代表所有海外华人发声，这让不是在大陆出生的华人对大陆华人的敌意越大。
 <br/>
 <br/>
 莫志明还表示，可悲的是，只有美国和澳洲等公开对抗中共的西方国家找到了应对中共影响活动的政治资本。新西兰需要成为能够抵制中共影响力的独角兽，以此来维护中国人民的政治独立权利。
 <br/>
 <br/>
 采访编辑／常春 后制/陈建铭
 <br/>
