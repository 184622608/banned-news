### 【禁闻】澳智库: 中共军方派人偷学外国军事技术
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】澳大利亚一家智库日前发表研究报吿指出，多年来，中共派遣了数千名军方
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技人员.html" target="_blank">
  科技人员
 </a>
 ，前往美国等西方大学学习军事技术，他们隐瞒自己的军方身份，逐步建立起一个促进中共军事科技发展的研究合作网络。评论认为，这份报告将会引起美国重视。
 <br/>
 <br/>
 《华尔街日报》10月29号报导说，澳大利亚国防部资助的智库〝战略政策研究所〞发表的最新研究报告指出，过去十年，中共军队附属大学的大约2,500名研究人员，被派往美国和英国等西方国家的大学，研习军事技术，其中许多人隐瞒了他们与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共军方.html" target="_blank">
  中共军方
 </a>
 的关系。
 <br/>
 <br/>
 许多人在向西方大学提出申请、或以英文发表报告时，常常不提他们的军方背景，自称是来自看起来像民间学术机构的成员。
 <br/>
 <br/>
 期间，中共军方
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技人员.html" target="_blank">
  科技人员
 </a>
 与国外学者共同发表论文的数量大约增长了8倍，他们逐步建立起了一个可大大促进中国军事科技发展的研究合作网络。
 <br/>
 <br/>
 一位不愿透露姓名的中共高级军官告诉〝新唐人〞，中共军队附属大学的教职员和研究员本身就是军人，他们是文职，并且都是共产党员，一旦透露他们的军方身份，像美国等西方国家就不可能给这些人签证，因而不得不隐瞒自己的真实身份。
 <br/>
 <br/>
 台湾国立政治大学国际关系研究中心主任严震生：〝你希望去进修，你是来自于军方或者是国安单位的，对方可能会有些疑虑，所以只好用智库研究的身份或者以这种贫民的身份出去。〞
 <br/>
 <br/>
 全球自由信息运动网站创始人张新宇：〝偷别人东西当然要伪装起来才偷嘛。坑蒙拐骗偷它都要伪装起来，他自己都知道是做坏事。〞
 <br/>
 <br/>
 早年在中共军队教授电脑技术的全球自由信息运动网站创始人张新宇认为，西方国家不知道中国的一切都是党控制的，才被蒙骗。
 <br/>
 <br/>
 张新宇：〝其实军队大学这些教授出国之前都要经过培训，特别是那些公派的所谓访问学者，都是经过共产党的安全部布置任务。〞
 <br/>
 <br/>
 报告作者乔斯科（Alex Joske）在研究了中国和外国科学家自2006年后的合作计画、以及中共派往海外研究人员的统计资料后发现，中共的兴趣主要集中在导航科技、电脑科学和人工智慧等学科，尤其是新兴军民两用科技领域。
 <br/>
 <br/>
 报告説，通常中共军方派出的学者不会和东道国的军方官员直接接触，而是专心收集能促进中共军事科技进步的知识。
 <br/>
 <br/>
 而这些中共军事研究人员选择海外研习的国家，主要是〝五眼联盟〞（Five Eyes）国家集团成员，也就是美国、英国、澳大利亚、新西兰和加拿大。
 <br/>
 <br/>
 报告指出，过去10年来，美国大学就接待了大约多达500位来自中共军方所属学校的访问学者。
 <br/>
 <br/>
 不过，台湾国立政治大学国际关系研究中心主任严震生认为，中共的学者很难取得关键技术。
 <br/>
 <br/>
 一位曾任因特尔公司高级工程师的人士也向〝新唐人〞透露，为防止中共窃取技术，因特尔绝不让中国人进入核心部门。
 <br/>
 <br/>
 国立政治大学国际关系研究中心主任严震生：〝我觉得美国，它认为敏感的议题，比如説它认为会有技术转移，敏感的军事技术，它不会让你来干，。主动权在美国那里，它要决定哪些东西它愿意跟你分享，它愿意给你看到，就不会去追究你的身份，如果它认为有点比较敏感，涉及国安，当然它就不会让你知道。〞
 <br/>
 <br/>
 随着美中关系日益紧张，贸易战越演越烈，川普政府正在竭力制止美国尖端科技流往中国。
 <br/>
 <br/>
 张新宇认为，澳大利亚的这份报告一定会引起美国政府的重视，美国随后就会采取行动。
 <br/>
 <br/>
 采访编辑/刘惠 后制/周天
