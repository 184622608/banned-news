### 非洲猪瘟肆虐中国 云南再被〝攻陷〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国非洲猪瘟疫情.html" target="_blank">
  中国非洲猪瘟疫情
 </a>
 加剧恶化。21日，云南通报发现首例猪瘟疫情。至此，非洲猪瘟已在中国大陆11个省市区蔓延。近日，有大陆防疫专家披露，北京当局已无力阻止非洲猪瘟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情扩散.html" target="_blank">
  疫情扩散
 </a>
 ，并禁止掌握疫情变化的实验室和疫控人员洩密。
 <br/>
 <br/>
 中共农业农村部10月21日通报，云南省昭通市镇雄县一养殖场，及母享镇一养殖专业合作社发现非洲猪瘟疫情。镇雄县养殖场共有生猪804头、发病298头、死亡298头；母享镇养殖专业合作社共有生猪353头、发病247头、死亡247头。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国非洲猪瘟疫情.html" target="_blank">
  中国非洲猪瘟疫情
 </a>
 8月初爆发，最先出现在辽宁省沈阳市，随后由北向南扩散。根据中共官方通报，目前疫情已蔓延至11个省市区，包括辽宁、黑龙江、河南、江苏、安徽、浙江、内蒙古、吉林、天津、山西和云南，并有进一步失控的迹象。
 <br/>
 <br/>
 本月12日，中共卫生部启动应对非洲猪瘟疫情的一级应急预案。不过，参与防控工作的非洲猪瘟专家徐女士日前向《自由亚洲》透露，非洲猪瘟疫情资讯已受到官方管控，最能掌握疫情变化的实验室和疫控人员都被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京当局严管.html" target="_blank">
  北京当局严管
 </a>
 ，禁止洩密。
 <br/>
 <br/>
 目前，非洲猪瘟疫情加剧恶化，正在向全中国蔓延，而官方却多次发布消息称，猪肉可以放心吃。对此，徐女士指出，猪肉并不是安全的，非洲猪瘟病毒是一类病毒，致病致死率甚至高达100%。而且它和禽流感病毒一样，也有变异的可能，很可能令人感染猪瘟病毒。
 <br/>
 <br/>
 对于非洲猪瘟疫情在大陆失控蔓延的情势，联合国粮农组织非常担忧，已发布警告称，非洲猪瘟疫情不可避免将扩散至亚洲其他国家，呼吁各国对从中国进口的猪肉进行严查。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：祝馨睿）
