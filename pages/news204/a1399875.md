### 【禁闻】11月18日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 母亲发声明 吁国际关注
 <br/>
 <br/>
 大陆民间维权网站〝六四天网〞创始人黄琦，被中共关押接近两年，身患重病，他85岁高龄的母亲蒲文清一直为他奔走呼吁，要求当局立即放他出狱治病，但都毫无结果。11月18号蒲文清发表声明，呼吁国际社会关注黄琦在看守所的生命权、医疗权和其他基本人权。
 <br/>
 <br/>
 2. 丁德元案将二审 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 吁放人
 <br/>
 <br/>
 上海维权人士丁德元被控〝妨碍公务罪〞一案，二审将于12月3号开庭，他的律师纪中久11月16号收到出庭通知书，17号，一批上海维权人士上街举牌，呼吁当局〝停止迫害  立即释放丁德元〞。
 <br/>
 <br/>
 3. 揭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊利集团.html" target="_blank">
  伊利集团
 </a>
 高层贪腐 两记者获刑
 <br/>
 <br/>
 大陆财经记者刘成昆和邹光祥，因在网上披露乳业巨头伊利集团高层涉嫌贪腐外逃，被当局指控造谣，10月24号，被内蒙呼和浩特市法院被以〝寻釁滋事罪〞判刑。11月15号，记者无国界组织发表声明，批评中共以〝恐怖手法〞打压独立新闻，敦促当局尽快放人。
 <br/>
 <br/>
 4. 律师要求会见吉林维权人士田野遭拒
 <br/>
 <br/>
 吉林维权人士田野，因参与发起〝中国公民维权同盟〞，被以涉嫌〝寻釁滋事罪〞逮捕，11月15号，他的律师马纲权前往看守所要求会见，不仅遭到拒绝，而且还遭警方跟踪调查。
</div>
