### 股市极度恐慌 刘鹤罕见带一行两会喊话救市
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 面临股票质押平仓压力，许多私企陷入生存危机，投资者退场抛售意愿强烈。北京经济掌舵人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 带领
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一行两会.html" target="_blank">
  一行两会
 </a>
 接连喊话稳定市场，极度恐慌的情绪稍有缓解，但业内人士指效果难以持久。
 <br/>
 <br/>
 周五（10月19日），中共国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 在新华社专访中表示，最近央行、银保监会、证监会及其他部门均在研究并出台各种政策，包括稳定市场、鼓励市场长期资金来源、促进国企改革和民企发展及扩大开放等。
 <br/>
 <br/>
 他说，〝改革开放方针〞已定，现在是〝一个行动胜过一打纲领〞的时刻，各方要快速行动，切实推出具体政策，推动股市发展。
 <br/>
 <br/>
 对于市场近期国资入股并购私企的担忧，刘鹤再次宣称，这只是国有银行或国有企业在〝帮助民营企业度过难关〞，并〝不存在‘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 ’的问题〞。
 <br/>
 <br/>
 当日稍早时候，中共央行行长易纲、银保监会主席郭树清和证监会主席刘士余接连发声，向A股喊话。
 <br/>
 <br/>
 港媒分析称，三者讲话显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一行两会.html" target="_blank">
  一行两会
 </a>
 预计可能会给A股提供一定资金和政策支持，帮助企业缓解危机，并要求银行暂缓或采取变通方式处理平仓，以及进一步放宽私企融资环境等。
 <br/>
 <br/>
 路透社引述私募基金少数派投资董事总经理周良的分析指，高层官员的讲话对股市还是有很大影响的，重要的是改变了市场极度悲观的预期。
 <br/>
 <br/>
 他说，此前投资人对股市已失去信心，大多在退场观望，场内抛盘多接盘少。现在一行两会齐出手，至少能够提振信心，减少投资者抛售意愿。
 <br/>
 <br/>
 周五一行两会的盘前喊话之后，沪综指.SSEC低开并惯性下滑后，终于止跌。中午刘鹤讲话后，沪综指开始上涨，最终收升2.58%。
 <br/>
 <br/>
 但上海一家私募基金总经理指出，虽然高官喊话对缓解市场情绪上有所帮助，但这种效应只是暂时性的。
 <br/>
 <br/>
 华西证券分析师毛胜分析，股市疲弱不是受股权质押困局一个因素影响，还包括经济下行压力加大、外盘持续回调、人民币汇率疲软等多方面原因。因此，短期内股指可能会反弹，但难有大级别上涨。必须等当局一直强调的改革举措，例如减税减费等切实落地，才可能真正提升上市公司业绩。
 <br/>
 <br/>
 不过外界分析，中共改革已进入僵局，经济陷入结构性困境，再加上贸易战影响，当局再想给企业减负，推行实质性改革，已是难上加难。
 <br/>
 <br/>
 目前，许多公司用作抵押获取贷款的法人股都已面临银行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平仓线.html" target="_blank">
  平仓线
 </a>
 。这意味着，贷款公司的股价跌至一定水平之后，国有银行有权收回其质押股权。有体制内金融人士表示，A股正在上演一场〝国有化大洗劫〞。这是导致这一轮市场恐慌的主要原因。
 <br/>
 <br/>
 中共央行今年已四次降准，释放巨量资金，声称用于支持微小私企。但许多中小企业融资依然困难，陷入经营危机。今年以来，各地国有资金纷纷出手接盘私企股权，被指趁火打劫。这已引发外界对中共抢劫私企、回归计划经济的担忧。
 <br/>
 <br/>
 中共央行行长易纲日前谈及中国经济、贸易战时称，北京已〝为最坏情况做好准备〞。
 <br/>
 <br/>
 美国高地智库（High Point Institute）研究员秦伟平对自由亚洲解读，即使没有中美贸易战，中国经济也会出大问题，所以中共很早就做了〝最坏的打算〞，就是重走计划经济道路，闭关锁国，以保证中共在恶劣的经济社会形势下仍可继续执政。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
