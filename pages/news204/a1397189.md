### 重庆幼园血案 传凶嫌上访多年 当局噤声
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】震惊社会的重庆巴南区幼儿园血案，中共政法委书记郭声琨批示要求实施舆论管控，禁止炒作。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凶嫌.html" target="_blank">
  凶嫌
 </a>
 的身分和犯案动机也一直未披露，日前一份官方内部通报被曝光，其中显示，砍伤14名儿童的女嫌犯曾因交通事故受伤，多次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 。
 <br/>
 <br/>
 10月28号晚间，重庆市教委和公安局内部会议情况通报被网络披露，教委的通报显示， 巴南区幼儿园案件的女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_凶嫌.html" target="_blank">
  凶嫌
 </a>
 姓刘，40岁，未婚，1991年交通事故受伤，多次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 ，2016年6月签署补偿协议后，因个人诉求不满，采用极端方式报复，持刀砍伤14名儿童。
 <br/>
 <br/>
 这是案发后，官方首次披露凶嫌的身分，但是大陆新闻媒体对此只字不提。有市民表示，官方媒体只报导对它们有利的消息，对它们不利的负面消息都封锁。
 <br/>
 <br/>
 重庆市民刘先生:〝网络上面它什么版本都有，然后报复社会，很多种说法，这边政府只肯定是传播正面的嘛，负面的话降到最小嘛。〞
 <br/>
 <br/>
 据〝自由亚洲电台〞报导，女凶嫌全名刘小玲，大约在1991年，在当地的红山铸造厂附近发生车祸，导致右手四只手指需要切去，此后她长期上访。到2010年，重庆官方牵头，一次性给她25万元的车祸补偿，并签订停止上访保证书，但她近年再度上访。
 <br/>
 <br/>
 维权人士认为，刘小玲长期受到打压和维稳 ，可能是导致极端行为的主要原因 。但舆论一致认为，〝无论有何理由，都不应将刀砍向一群无辜的孩子。〞
 <br/>
 <br/>
 重庆维权人士胡先生:〝这个内部通报是真的话，这个以后的矛盾还会有很多的，极端的事情会越来越多，政府应该释出来解决问题的，这个是法律赋予政府的权力嘛，但是他们不但不解决问题，反而是高压的那种维稳，就是激化矛盾嘛。〞
 <br/>
 <br/>
 重庆访民向新唐人记者表示，访民是中共统治下最悲惨的群体。
 <br/>
 <br/>
 重庆访民郭女士:〝重庆市有很多访民的冤屈没解决，甚至访民被抓被关，这是事实，北京开会，哪里开会，怕这些访民去反映问题，就会维稳呀，关押呀。我现在就在公安部信访办，人可多了，还没排上队。马上就下班了 ，国家信访局也是很多很多的人的。〞
 <br/>
 <br/>
 10月26号，巴南区幼儿园发生持刀砍伤14名幼童血案，凶嫌第二天被以涉嫌故意杀人罪批捕。〝自由亚洲 电台〞得到的重庆市公安局情况通报录音显示，案发后，中共总理李克强和中央政法委书记郭声琨都作出批示。其中郭声琨的批示要求实施舆论管控，禁止炒作。此前已经有三名网友因传播有多名孩子死亡的不实消息而被拘留。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
