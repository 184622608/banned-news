### 刘卫国发王全璋亲笔信 宣布续代理案件
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】首先来关注中国的消息。〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709案.html" target="_blank">
  709案
 </a>
 〞被捕律师王全璋一案，日前传出新消息，早前宣布因身体受伤而退出案件的官方指定的律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘卫国.html" target="_blank">
  刘卫国
 </a>
 ，10月31号突然又在网上声明说，他将继续代理王全璋案。
 <br/>
 <br/>
 10月31号，已经宣布退出代理王全璋案的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘卫国.html" target="_blank">
  刘卫国
 </a>
 律师，突然在微信朋友圈展示了一封据称是王全璋亲笔写的信件，和一张天津第一看守所的〝被监管人员亲友送款凭证〞，并加上了〝当仁不让，继续代理〞的简短文字。
 <br/>
 <br/>
 信件内容说，现在出现了一些新的情况，希望刘卫国近期能前往看守所进行沟通。并希望刘卫国帮忙处理他在山东济南房子，以及帮他配一副便宜的眼镜。
 <br/>
 <br/>
 〝被监管人员亲友送款凭证〞则显示，刘卫国10月31号以律师的身份，向天津市第一看守所为王全璋存了1000元现金。
 <br/>
 <br/>
 大陆律师刘晓原当天在推特转发刘卫国的微博截图，并评论说，不知羁押在天津市第一看守所的王全璋律师出现了什么新情况。
 <br/>
 <br/>
 新唐人记者致电刘卫国，想询问一下详情，但他一直不接电话，后来干脆直接挂断。
 <br/>
 <br/>
 王全璋的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 表示，刘卫国至今也没有和她通话。
 <br/>
 <br/>
 王全璋妻子李文足：〝在王全璋被抓的时候，我就聘请律师了，我聘请的律师已经有7位了，一旦有律师介入之后，官方就开始施压，威胁他们退出，所以就不断的更换律师，所以王全璋我前后给他聘请了7位。〞
 <br/>
 <br/>
 王全璋被中共羁押已经超过3年，案件移送至天津市中院也超过20个月，但家属聘请的律师从未获准与他会面。
 <br/>
 <br/>
 7月份，刘卫国突然声称，王全璋委托他为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代理律师.html" target="_blank">
  代理律师
 </a>
 。但8月底，他又发声明说，因自己意外受伤，需要退出案件。现在他突然又宣布继续代理案件，令外界疑惑不解。
 <br/>
 <br/>
 李文足：我已经聘请了律师，我的律师已经很早介入这个案件了。在我聘请了律师的情况下，其它律师再介入的话从程序上是违法的，但是现在刘卫国律师他说有王全璋写的一个亲笔信。〞
 <br/>
 <br/>
 李文足表示，由于一直无法与刘卫国联络上，关于这封信究竟是不是真的，她也没有办法确认。她说，如果她得到王全璋的消息，会第一时间公布。
 <br/>
 <br/>
 新唐人记者常春、施宜君采访报导
</div>
