### 【禁闻】12月5日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年12月05日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 辽宁数百教师省教育厅集体上访
 <br/>
 <br/>
 辽宁省沈阳、大连、鞍山等14个市的500多名民办、代课、幼儿教师，12月3号集体到省教育厅上访，要求政府兑现20年前出台的政策，解决他们的养老等问题。这是他们今年第6次数百人集体上访，但与前几次一样，没有任何结果。
 <br/>
 <br/>
 2. 中共法制日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 北京寄信被抓
 <br/>
 <br/>
 12月4号是中共的所谓〝法制日〞，一些在京访民到北京市力学胡同邮局邮寄举报信，但都遭到阻拦。多人因此被抓，其中有人被遣返原籍，有人则下落不明。
 <br/>
 <br/>
 3. 北京市政路桥公司工人拉横幅
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_讨薪.html" target="_blank">
  讨薪
 </a>
 <br/>
 <br/>
 北京市政路桥股份有限公司，承建北京新机场高速公路项目，但是拖欠工人工资，12月3号一批工人集体到公司门口拉横幅讨薪。
 <br/>
 <br/>
 4. 荆州福娃集团欠薪 员工镇政府维权
 <br/>
 <br/>
 湖北省荆州市监利县新沟镇的福娃集团，因资金链断裂导致工厂停工，拖欠近千名员工近一年的工资，12月3号，员工集体到镇政府维权，要求发工资。
</div>
