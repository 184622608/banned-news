### 央视主持刘芳菲丈夫死亡案 幕后黑手落马
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】中共吉林省检察院副检察长吴长智13日被通报受查，官方未说明他具体涉及什么问题，但多家境外媒体报导显示，他与央视女主播
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘芳菲.html" target="_blank">
  刘芳菲
 </a>
 的丈夫、富商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘希泳.html" target="_blank">
  刘希泳
 </a>
 非正常死亡案有关。
 <br/>
 <br/>
 11月13日，吉林省纪委省监委网站消息称，吉林省检察院党组成员、副检察长吴长智涉嫌严重违纪违法，目前正接受纪律审查和监察调查。
 <br/>
 <br/>
 公开资料显示，吴长智生于1962年，吉林磐石人，历任任吉林桦甸市委副书记、任桦甸市委副书记、政法委书记、任吉林市委组织部副部长、吉林松原市人民检察院检察长。
 <br/>
 <br/>
 2016年11月，任吉林检察院党组成员、副检察长，吉林检察院检察委员会委员。
 <br/>
 <br/>
 中共官方没有说明他具体涉及什么问题，不过根据多家境外媒体报导显示，他与央视女主播
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘芳菲.html" target="_blank">
  刘芳菲
 </a>
 的丈夫、富商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘希泳.html" target="_blank">
  刘希泳
 </a>
 非正常死亡案有关。此前多名涉案检察官已被判刑。
 <br/>
 <br/>
 中共官媒报导，9月7日，吉林延边9名检察官以涉嫌刑讯逼供、故意伤害、玩忽职守等在天津受审。10月23日，9名检察官被判刑入狱。
 <br/>
 <br/>
 其中，审讯小组负责人许学哲，以故意伤害罪被判刑15年，周庆军被判刑11年，调查组组长赵伯忠，因渎职罪被判刑4年。其他6人被判处15个月至3年徒刑不等。
 <br/>
 <br/>
 英国《金融时报》引述一名代理人表示，刘希泳家属反对法院的判决，希望以谋杀罪定罪。报导说，根据对这9人的起诉书，刘希泳在被剥夺睡眠和遭受酷刑80小时后，于2017年3月死亡。
 <br/>
 <br/>
 2017年3月15日至19日期间，中共延边朝鲜族自治州检察院，对刘希泳涉嫌行贿一案成立调查组，采取监视居住强制措施。之后，刘希泳在看守所突然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离奇死亡.html" target="_blank">
  离奇死亡
 </a>
 。
 <br/>
 <br/>
 网上流传的起诉书显示，刘希泳曾遭到检方刑讯逼供。在审问过程中，刘希泳先是被蒙住眼睛并绑在审讯椅上，他的双腿被高高抬起，为了加快让他认罪，以许学哲为组长的9名检察官，对他采取胶带缠口、钥匙捅脚心、通渠泵捅口鼻、双腿绑在前方椅背上的方法逼取口供。
 <br/>
 <br/>
 在4天内，只让刘希泳到居住室休息两个小时，其余时间一直约束在审讯椅上，且在夜间进行审问。
 <br/>
 <br/>
 根据验尸报告显示，刘希泳胸骨、肋骨有七处骨折，因口鼻受压等多种因素机械窒息死亡。
 <br/>
 <br/>
 消息称刘希泳在被抓后，拒绝交代许多事情。警察遂将沾有芥末的毛巾捂在其口鼻，导致其死亡。而他被查的主要原因是涉及替高层洗钱。
 <br/>
 <br/>
 刘希泳是70年代末期中国第一批留美学生，哈佛大学毕业后到香港经商，成为JC Penny服装供应商、香港君怡酒店老板、香港新立基（国际）集团有限公司董事长，一度成为拥有几十亿资产的〝金融大鳄〞。
 <br/>
 <br/>
 上世纪80年代，刘希泳认识了中共国安部六局局长刘植桢，而刘的上司是当时中共国安部部长贾春旺，当时认识刘希泳的商人，把他标签为〝国安部外围的人〞。
 <br/>
 <br/>
 1990年代末，刘希泳曾卷入原光大集团董事长朱小华案，被中纪委〝双规〞，并被关押过一段时间。后获释，与央视主持人刘芳菲相识并结婚。
 <br/>
 <br/>
 2016年11月刘希泳因涉嫌诈骗工行2亿多港币贷款遭到通缉，随后被中共检查系统扣查，原本关押在北京，但在去年3月，他被秘密转押至吉林省延边州看守所，随后传出已死亡。
 <br/>
 <br/>
 消息称，刘希泳的死涉及刘云山、贾春旺、李岚清等江派大佬，以及最高检察长等一大批政法系统的高官。刘为何遭〝灭口〞，相关报导并没有提供任何具体的信息。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
