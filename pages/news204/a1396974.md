### 北京要拿红十字会开刀？中纪委高官接任一把手
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红十字会.html" target="_blank">
  红十字会
 </a>
 官网26日消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐科.html" target="_blank">
  徐科
 </a>
 辞去中共红十字会党组书记及中共红十字会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_常务副会长.html" target="_blank">
  常务副会长
 </a>
 职务，由第十九届中纪委委员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梁惠玲.html" target="_blank">
  梁惠玲
 </a>
 担任。外界分析，近年来，中共红十字会贪腐黑幕大量曝光，如今中纪委高官直接掌管该机构，不排除当局想对红十字会强力反腐。
 <br/>
 <br/>
 10月26日，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红十字会.html" target="_blank">
  红十字会
 </a>
 官网发布消息称，由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梁惠玲.html" target="_blank">
  梁惠玲
 </a>
 取代
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐科.html" target="_blank">
  徐科
 </a>
 ，担任中共红十字会党组书记，同时梁惠玲被提名为中国红十字会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_常务副会长.html" target="_blank">
  常务副会长
 </a>
 ，而原常务副会长徐科也不再担任此职务。
 <br/>
 <br/>
 公开资料显示，梁惠玲，1962年8月出生，湖北宜城人，历任湖北省孝感市委常委，孝感市副市长、副书记、代市长、市长，湖北省妇联主席、党组书记，湖北省鄂州市委书记，湖北省副省长，常委、统战部部长、省总工会主席。
 <br/>
 <br/>
 2016年12月，梁惠玲任河北省委常委、省纪委书记。2018年1月，任河北省委常委、省纪委书记、省监察委主任，同时她还是第十九届中纪委委员。
 <br/>
 <br/>
 刚刚辞职的徐科，1956年7月出生，于2014年9月2日接替赵白鸽担任红十字会党组书记、常务副会长职务。
 <br/>
 <br/>
 当时舆论一度关注中共红十字会调整高层之举能否从丑闻中脱身出来。而这次人事调整由中纪委高官直接掌管该机构，引发猜测。外界分析，当局这种安排，不排除是想对红十字会内部的腐败进行查处。
 <br/>
 <br/>
 多年来，大陆红十字会丑闻不断，让慈善事业的公信力一再遭到质疑。
 <br/>
 <br/>
 如，2008年汶川地震后，各界人士向灾区捐献大量钱款，据中共民政局资料显示，仅地震后半年内，全国为汶川地震灾区捐款652亿元人民币，另募集物资价值约110亿元。但陆媒综合数据发现近八成、达501亿元款项至今去向不明。
 <br/>
 <br/>
 2016年8月，四川省红十字会原党组书记、常务副会长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文家碧.html" target="_blank">
  文家碧
 </a>
 ，因〝受贿罪、贪污罪〞二审被判16年。文家碧被踢爆曾利用汶川大地震敛财。
 <br/>
 <br/>
 2013年〝4.21〞雅安地震，该会在震后第一时间在微博上呼吁捐款，不到两天，该会收到高达14多万个〝滚〞字的回帖。
 <br/>
 <br/>
 2013年12月，陕西汉中红会办公室原副主任张琪，被指〝涉嫌挪用1,000万元公款〞自杀身亡。
 <br/>
 <br/>
 2011年一年之内，先后曝出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭美美事件.html" target="_blank">
  郭美美事件
 </a>
 〞、审计发现多笔资金存在问题、中共红十字会总会配车案、官员贪污案等。
 <br/>
 <br/>
 〝郭美美事件〞是指一名微博认证身份是〝中共红十字会商业总经理〞的年轻女子，在微博上炫富，之后被曝她是一名与红十字会有关联的商人〝干爹〞的情妇。
 <br/>
 <br/>
 同年8月9日，《博客天下》杂志副总编吴晨光曝出，中共红十字会总会的车库内停着若干辆豪华公车，红十字会司局级以上领导每人配车两辆。
 <br/>
 <br/>
 公开资料显示，中共红十字会还参与中共器官移植项目。从2006年开始，中共前卫生部副部长黄洁夫与红十字会共同推行器官捐赠试点、OPO、COTRS系统。
 <br/>
 <br/>
 2006年正是中共活摘法轮功学员器官黑幕被国际广泛曝光的一年。一位大陆参与活摘罪行的医生妻子及记者在美国揭露中共这一罪行，加拿大人权律师经过调查证实中共这一罪行是存在的。
 <br/>
 <br/>
 而黄洁夫是中国肝胆外科专家，从2001年11月担任中共卫生部副部长12年期间，任人体器官捐献与移植委员会主任，包揽全国器官移植方面所有职务。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
