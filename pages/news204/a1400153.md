### 【禁闻】11月20日退党精选
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】由于中小企业的倒闭潮，大批去城市打工的外来人口失业。中共农业农村部日前声称，目前有740万人返乡创业。但一名山东青年在退党声明中，说出了他们的实情。来看今天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退党精选.html" target="_blank">
  退党精选
 </a>
 。
 <br/>
 <br/>
 郎保福说：我是山东菏泽的一个青年，在家靠经营几分土地不能养家糊口，出来外地打工深受其苦，日出而起，日落不归，一天十几个小时劳作，所挣工钱除去自己吃饭所剩无几。
 <br/>
 <br/>
 时下生活艰难，电视上新闻联播天天都是好消息，哪知道我们农村人的真实情况。
 <br/>
 <br/>
 我从小加入过中共少先队，过去梦想要当共产主义接班人。看来，这都是骗人的把戏，现在城市农村腐败成风，贪官遍地，豪宅、高档轿车都是权贵阶层拥有，多数百姓生活并不幸福。
 <br/>
 <br/>
 所以我看透了共产党的丑恶面目，声明退出少先队，期待天灭中共，恩泽百姓。
</div>
