### 民企亏损关停潮来袭 大面积欠薪 农民工返乡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】日前，中国广东某家具店的工人以威胁要〝集体跳楼〞的方式向雇主讨要被拖欠的工资。因大批中国民企亏损关停，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欠薪.html" target="_blank">
  欠薪
 </a>
 现象在中国大陆大面积发生，同时更有数百万农民工因失业已被迫返乡。舆论认为，中国真正的制造业崩溃或将于2019年的上半年发生。
 <br/>
 <br/>
 中国广东东莞〝名将家具〞店因亏损而发不出工资，于是拖欠农民工的工资，结果引发工人罢工。当地时间11月14日，该家具店的多名工人为讨薪，跑到公司楼上以要〝集体跳楼〞施压雇主，发誓要讨回自己被拖欠的工钱。
 <br/>
 <br/>
 针对这起讨薪事件，东莞居民郑先生15日接受《自由亚洲电台》采访时表示，当地的经济环境很差，不少企业倒闭，民营企业已到了生死存亡的关键时刻。
 <br/>
 <br/>
 他坦言：〝中美贸易战、人工成本、政府的苛捐杂税，高油价导致材料成本和运输成本等等高居不下。我所有的民营企业家朋友都异口同声的说，企业已经很难再经营下去了。没有倒闭的也计划明年关闭企业。特别是出口制造业，已经对美国明年全面对中国增加关税，作好了心理准备。〞
 <br/>
 <br/>
 郑先生透露，自己朋友所开办的企业都已经在进行关闭前的程序，打算到今年年底就提前给工人放假，然后关闭工厂，观望一下国际形势的发展再做定夺。
 <br/>
 <br/>
 他推测，明年上半年，中国大陆制造业可能发生倒闭潮。他说：〝真正的制造业崩溃应该在明年初，大规模发生。〞
 <br/>
 <br/>
 中国农业农村部11月8日公布的数据显示，目前，全国返乡下乡的人员已达740万人。一贯把坏事当作好事来报导的中共官媒，把这种
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_返乡潮.html" target="_blank">
  返乡潮
 </a>
 宣传为〝大量社会资本、技术、人才‘上山下乡’〞，声称这数百万人返乡是为了〝创业〞。
 <br/>
 <br/>
 不过，中共官方的这种说辞在互联网上遭到众多网民的质疑。财经博主〝颜值财经程凌虚〞就曾发帖评论道，〝明明是农民工大规模失业，却偏偏要说成是返乡创业！从城市失业回农村，摇身一变成为返乡创业了？你当农民工是傻子呀，他们的老板都干不下去了，工厂都倒闭了，只好返乡。〞
 <br/>
 <br/>
 不久前，中国大陆门户网站网易曾发表了一篇题为《今年上半年国内504万家企业倒闭，失业人数超过200万》的文章，指2018年上半年，中国大陆有504万家企业倒闭，是全国企业总数（3,100万家）的六分之一。
 <br/>
 <br/>
 事实上早在2017年10月，阿里巴巴董事局主席马云在预测2018年时就说过，中国恐将迎来〝30年来最恐怖的失业潮和倒闭潮〞。
 <br/>
 <br/>
 马云这次可谓一语成谶。2018年以来，中国民营企业〝倒闭〞、〝关停〞、〝破产〞、〝拖欠工资〞、〝业绩下滑〞、〝亏损〞的消息不断曝出。
 <br/>
 <br/>
 陆媒〝易懂财经〞8月25日发表的《民营企业关停潮来袭，我们真的要过冬了吗？》一文引述业内人士指出，高昂的成本压力、激烈的市场竞争、较长回报周期等问题，让众多中国民企步履维艰。〝因为疯狂上涨的成本，叠加空前的监管风暴，将会造成多米诺骨牌效应的倒闭潮和失业潮，民营企业等实体经济可能扛不住了。〞
 <br/>
 <br/>
 文章表示，有业内人士指出，当前中国民企中的主力军〝已纷纷缴械投降〞，就连一梯队生存都很艰难，更何况那些中小民营企业，〝很多都活不过今年，谁能扛到明年才是关键。〞
 <br/>
 <br/>
 民企之所以出现纷纷倒闭的现象，与贸易战带来的巨大压力有着莫大的关系，而中国民营企业500强〞之一的山东晨曦集团在中美贸易战正式爆发后迅速走向破产，就是一个典型的例子。
 <br/>
 <br/>
 文章援引东方艾格分析师马文峰的观点指出，目前中国整个贸易顺差在4600亿美金，而其中来自于美国的贸易顺差就达到了3700亿美金。也就是说，中国企业从国外挣来的钱中，绝大部分来自于美国市场。而中国企业的平均税后利润率大概在16%-20%之间（含出口退税），只要中美贸易战开打，利润率将会被挤掉50%，由此将导致中国制造业出现大规模的倒闭潮和失业潮
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：戴明）
