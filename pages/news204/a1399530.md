### 6成VPN归中国公司所有  使用恐隐私洩光光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】一份新的研究报告显示，中国境内大约60%的免费
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_VPN.html" target="_blank">
  VPN
 </a>
 都在中国本地开发，或者归中国公司所有，如果用这类VPN上网，用户的隐私和资料可能有风险。然而，〝自由门〞〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无界浏览.html" target="_blank">
  无界浏览
 </a>
 〞等免费
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻墙.html" target="_blank">
  翻墙
 </a>
 软件，可以帮助网民安全地突破封锁。
 <br/>
 <br/>
 美国之音15日报导说，运营
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_VPN.html" target="_blank">
  VPN
 </a>
 端口的Metric Labs公司研究部主管西蒙・米格利亚诺说：〝调查显示，超过半数的免费VPN应用，归中国公司所有或干脆就在中国境内。〞
 <br/>
 <br/>
 这位研究员说，中国过去1年大量取缔VPN服务，而且严密的监管境内的网络，他们分析了Google、Apple商店中，共30个免费VPN应用程式后，发现有86％的隐私政策〝令人无法接受〞。
 <br/>
 <br/>
 他说，这些免费的VPN，多数都没有给用户提供正式的隐私保护措施。一些应用还在政策中，承认他们与第三方分享数据，特别是与中共第三方公司分享数据及追踪用户。
 <br/>
 <br/>
 中国大约有2亿民众，经常使用VPN〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻墙.html" target="_blank">
  翻墙
 </a>
 〞绕开中共的网络审查，获得境外资讯，尤其喜欢登入facebook、twitter、gmail和youtube，这些中共屏蔽的社交平台。
 <br/>
 <br/>
 然而中共一直对VPN虎视眈眈，中共为禁止国民浏览境外资讯，建立长城防火墙，把近8亿的中国网友封锁在中共建立的区域网内。
 <br/>
 <br/>
 自2017年1月22日起，中共已经收紧了对VPN的监管，大幅取缔VPN，要求提供VPN的公司向中共当局注册，以获取能够在中国运营的营业执照。
 <br/>
 <br/>
 媒体披露，这种向中共官方注册的VPN不仅使用月费昂贵，而且通过采用添加自己的侧录程式、监控程式，翻墙者的身份也会被曝光且被中共记录下来。
 <br/>
 <br/>
 中共打压VPN不仅影响到2亿中国网民，外国企业也受到影响，甚至有网络工程师因提供VPN服务被中共当局判刑。
 <br/>
 <br/>
 免费VPN应用受到影响，但由动态网公司开发的〝自由门〞、〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无界浏览.html" target="_blank">
  无界浏览
 </a>
 〞免费翻墙软件，却一直在帮助中国网民突破封锁，了解真实讯息。
 <br/>
 <br/>
 据动态网公司总裁比尔．夏（Bill Xia）介绍，专门针对中共网路封锁而开发的自由门和无界浏览，一直在不断升级，使用业界最强的加密方式，可以让软件永远不被封锁，且能保护用户隐私。
 <br/>
 <br/>
 动态网公司是2002年由海外一批专门关注中共网络封锁的技术人才组建，他们主要是法轮功学员，通过自费开发自由门、无界浏览软件，帮助中国民众了解被中共封锁的真相。
 <br/>
 <br/>
 他们开发的自由门、无界浏览软件被越来越多的中国人使用，还帮助了中国之外的国家，例如伊朗、越南、叙利亚等国的民众，翻越本国的互联网审查。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
