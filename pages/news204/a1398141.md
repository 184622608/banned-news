### G7领袖全数缺席进博会 北京允加大开放难获世界信任
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】当地时间11月5日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首届中国进口博览会.html" target="_blank">
  首届中国进口博览会
 </a>
 在上海开幕。中共官方藉此机会大力宣传中国将更积极主动向世界开放市场。但外界发现，以美国为首的G7峰会领袖都没有出席这个博览会。舆论称，中共过去有太多承诺却极少兑现，所以现在北京当局要想获得世界的信任很难。
 <br/>
 <br/>
 首届中国国际进口博览会（简称〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 〞）开幕式11月5日上午在上海国家会展中心举行，中国国家主席习近平出席开幕式并发表主旨演讲。
 <br/>
 <br/>
 习近平这次的演讲，旨在回应外界对中国开放市场、保护知识产权等领域的疑虑，对外传递一个明确的信号——北京当局将致力于推动新一轮所谓〝高水平对外开放〞。
 <br/>
 <br/>
 习近平提出了中国将进一步改革的5大方面：1，进一步降低关税，提升通关便利化水准等；2，持续放宽市场准入，包括放宽外资股比限制；3，引入侵犯知识产权行为惩罚性赔偿制度；4，加快探索建设自由贸易港进程；5，推动多边和双边合作深入发展，推动达成区域经济伙伴关系协定。
 <br/>
 <br/>
 然而，对于这场中共官方相当高调宣传的博览会，欧美各国的反应却显得相当冷淡。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-6/p9112541a732041331.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-6/p9112541a732041331-ss.jpg"/>
   <br/>
   <font size="-1">
    2018年11月5日，首届中国国际进口博览会（CIIE）在中国上海举行。图为一名保安站在博览会的会展中心入口前。(Lintao Zhang/Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 外界发现，美国、加拿大、英国、法国、德国、意大利及日本的国家元首或政府首脑，都未出席上海进博会；一向与中共走得比较近的俄罗斯也仅派出总理梅德韦杰夫（Dmitry Medvedev）出席，此外就只有巴基斯坦、越南这样的经济欠发达的小国的政府首脑和领导人来给这次的博览会捧场。
 <br/>
 <br/>
 此前，曾有媒体引述中国的某〝专家〞观点称，美中贸易战火下，北京当局举办〝世界上第一个以进口为主题的国家级展会〞，除了可藉此表明中方对外贸易开放的关切，亦可藉此机会绕过美国来加深中国与其他国家的贸易联系，减轻贸易战的负面影响，增加与美国的谈判筹码。
 <br/>
 <br/>
 然而，《华尔街日报》日前对约30个政府、10多间跨国公司及几个行业协会进行调查后发现，尽管许多企业希望能够扩大在华销售，却没有几个外国政府或跨国公司打算派出最高级别的代表亲赴这场〝进博会〞以示支持。
 <br/>
 <br/>
 报导引述一些外交官表示，正当中美贸易战打得火热、充满变数之际，中方举办这次〝进博会〞的政治寓意存在不确定性，因此许多国家的领袖选择了继续观望。
 <br/>
 <br/>
 海外有舆论认为，西方大国领导人不出席上海〝进博会〞有其现实的考量，因为中美之间的贸易战还在进行中，美国政府固然不愿在此时参与进博会而给外界造成〝放低身段〞的印象，美国的盟友们也都尽量避免〝选边站〞，所以都决定不出席进博会。
 <br/>
 <br/>
 亲北京的《香港经济日报》则引述未具名的〝观察人士〞分析称，〝中美贸易战仍然充满变数，主要国家领袖大概还在观望未来发展，以不变应万变。〞
 <br/>
 <br/>
 该文同时表示，〝包括美国在内，不少经常与中国打交道的国家或许仍在考虑，应否为中国推进改革开放投下信任票。〞
 <br/>
 <br/>
 中共当年为了加入世界贸易组织（WTO），曾对国际社会做出过有关改革经济体制、开放中国市场、转型为市场经济等一系列承诺。但时至今日，中共十几年前就作出的那些承诺，绝大多数都未能兑现，中共反而乘机利用其WTO会员的资格大占他国的便宜，至今仍在实行给外资进入设限，通过政府补贴对外倾销，强制外企交出关键技术等贸易政策，引发国际社会持续指控。
 <br/>
 <br/>
 分析人士认为，多年备受损失的现实，令欧美各国对中共任何新承诺都报有严重的质疑。所以，尽管北京当局投入大钱举办〝进博会〞，并大张旗鼓地宣传其将要开放中国市场，但仍然无法摆脱其被国际社会主流国家集体孤立的现状。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
