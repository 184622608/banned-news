### 习近平南下目的引猜测 中山大学排定接待〝剧本〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】港媒接到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办.html" target="_blank">
  中联办
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_采访通知.html" target="_blank">
  采访通知
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 23日在广东珠海举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开通仪式.html" target="_blank">
  开通仪式
 </a>
 。多家港媒披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 有可能20日抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 ，先视察
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中山大学.html" target="_blank">
  中山大学
 </a>
 ，然后出席港珠澳大桥开通仪式。有美媒说，中山大学已经排好了接待习的〝剧本〞。
 <br/>
 <br/>
 自由亚洲电台报导说，中共驻香港联络办公室17日向港媒发出〝特急〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_采访通知.html" target="_blank">
  采访通知
 </a>
 ，各报纸、电台及电视台每家可派一名记者（电视台两名），采访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开通仪式.html" target="_blank">
  开通仪式
 </a>
 。
 <br/>
 <br/>
 采访地点在珠海，主办方还要求记者自费每晚600元人民币，入住指定的一家酒店。不过，对于记者关注的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 是否参加开通仪式，该通知中未提及。
 <br/>
 <br/>
 多家港媒17日报导，中国国家主席习近平将于19日至22日南下广东视察，20日到访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中山大学.html" target="_blank">
  中山大学
 </a>
 ，并在23日担任港珠澳大桥通车仪式的主礼嘉宾。
 <br/>
 <br/>
 中山大学消息人士透露，习近平的确切行踪尚不清楚，该校目前确实正为习到访做相关准备工作。
 <br/>
 <br/>
 一位不愿具名的消息人士18日对自由亚洲说，他也听说习近平将到广州和出席港珠澳大桥通车仪式，20日到广州。因中山大学有几个校区，珠海好像也有两个，现在还不知道他去哪一个校区。
 <br/>
 <br/>
 这位消息人士还透露，中山大学相关人士说，习近平将在校园与学生有一个见面，就象以前到其它大学参观一样，习一来，学生会围上去。
 <br/>
 <br/>
 另有消息人士披露，港珠澳大桥9月底试运后，10月底〝具备通车条件〞，何时正式通车还未确定。
 <br/>
 <br/>
 耗资人民币1200多亿元的港珠澳大桥，全长约40公里，连接香港、珠海和澳门，该项工程历时近9年。北京高层希望港珠澳大桥通车能带动广东西部地区的经济发展。但是，随着外资企业近几年陆续撤离珠江三角洲，其经济价值大不如前。
 <br/>
 <br/>
 有分析认为，如果习近平此次出席大桥通车仪式的话，除了表现中央政府对香港及广东经济发展的高度重视，也在纪念中国改革开放40周年。
 <br/>
 <br/>
 香港01报导说，近日，中共军方亦密锣紧鼓进行筹备，经陆空运送大批军备抵港。有网民18日，目睹有两架运输机突然抵达香港赤鱲角国际机场，并卸下多辆军车。陆路方面，当日早晨亦有10多辆军车，在香港警方护送下，经三号干线浩浩荡荡驶出九龙。
 <br/>
 <br/>
 有军事专家认为，军方多以〝粮草先行〞的策略行事，估计是为习南下作先行准备。
 <br/>
 <br/>
 但港府消息称，中共高层曾经研究习近平为港珠澳大桥主持通车仪式，但暂时未确定习是否会出席。
 <br/>
 <br/>
 9月20日，也曾有多家港媒报导，习近平南下广州、深圳、东莞、中山、惠州多地企业视察，并且出席在深圳举办的改革开放40周年纪念大会。当时传出深圳安保进入一级戒备。但相关消息至今都未获证实
 <br/>
 <br/>
 而此次传出习南下广州视察的同时，多家港台媒体报导，目前，中共正在秘密筹备第十九届四中全会，最快将于10月20日召开。报导还引述消息人士透露，日本首相安倍晋三将于10月25日访问中国，考虑到国内外的日程，预计四中全会，会在安倍晋三访华前举行。
 <br/>
 <br/>
 但有分析认为，如果习南下的消息属实的话，中共这一时间段举行四中全会的消息似乎并不靠谱。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：戴明）
