### 政审过的？上海大妈进博会哄抢食物惊煞老外（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】一段在上海国际进口博览会上拍摄的视频近日在网络上热传。画面中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国大妈.html" target="_blank">
  中国大妈
 </a>
 大叔们哄抢试吃食品的场面，再次令中国网友倍感蒙羞。有消息指，这些〝群众〞都是经过中共地方基层组织政审过的〝代表〞，现场更有外企工作人员感叹称，哪怕是在非洲，都从没看到过如此不文明的现象。
 <br/>
 <br/>
 据互联网流传的视频片段，在中国上海举行的国际进口博览会上的食品参展摊点处，一大群中年以上的中国男女彼此前胸贴后背地挤压在展示食物的台桌前。当工作人员一端出供参观者试吃的食物时，这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国大妈.html" target="_blank">
  中国大妈
 </a>
 大叔们立即一通哄抢，不但没有人排队，而且人们大都拚命把更多的食物往自己手中的盒子或袋子里扒拉，完全不像是〝试吃〞而是〝抢食〞。
 <br/>
 <br/>
 中国社交媒体微博用户@袁炫华日前发博文称，上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 上开放了街道居委会的团体参观，不料这帮大妈、大爷、大婶、大叔们一进展馆哪里闻香就往哪里冲。〝只要看到有食物试吃的，像饿狼般地蜂拥而上，嘴叼手抓，甚至还拿出自带购物塑料袋装，把每一个有试食的摊位给一扫而光。〞
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-20/p9133611a450734237.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-20/p9133611a450734237-ss.jpg"/>
   <br/>
   <font size="-1">
    据称经过中共基层组织政审过的居委会代表们，在上海国际进口博览会上嘴叼手抓地哄抢试吃食品的场面引发舆论关注。（视频截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 据称，进博会现场的那些〝老外〞们，看到此种场面一个个都在摇头。其中，还有一位欧洲来的参展工作人员表示，他已去过全世界各大洲的近百个国家参展，但从没看到过如此不讲文明的参观者，〝即使在非洲也没见过〞。
 <br/>
 <br/>
 该网友并披露，〝事后知道，进博会组委会预料到对公众全面开放会出现这样的局面，所以对一般观众不开放，而这些疯抢者都是街道、居委会挑选和预审过的代表。〞
 <br/>
 <br/>
 中国网友们在转发这段视频片段的同时，纷纷留言感叹〝丢人现眼〞、〝尴尬至极〞、〝令国人蒙羞〞、〝太丢脸啦〞。
 <br/>
 <br/>
 中国名嘴崔永元在转发这篇微博时也发表评论称，〝谁能告诉我，用什么办法培养了这么一大批‘人才’？全世界都没见过。 中国老年人在各种场合哄抢的事件屡见不鲜，此前曝出‘中国大妈’吃垮了一艘豪华邮轮〞。
 <br/>
 <br/>
 据分析，崔永元在微博中所说的〝吃垮〞豪华邮轮一事，可能是指早前网传美国迈阿密的诺唯真〝喜悦号〞邮轮宣布将退出中国一事。
 <br/>
 <br/>
 有消息指，这艘进入中国市场仅一年多的豪华游轮之所以决定将于明年4月退出中国市场，据称主要是因为很多中国游客在船上只吃那些不另外收费的食物，却不愿购买其他消费品，令邮轮盈利低微难以为继。
