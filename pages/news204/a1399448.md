### 上千陕西原民办、代课教师再次上访省府
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】陕西省原民办和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代课教师.html" target="_blank">
  代课教师
 </a>
 1000多人，11月13号再次集体到省信访局
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 维权，要求省教育厅、省政府落实中央文件，解决他们的身份和待遇问题。参与维权的老师表示，他们已经上访十几年，当地政府依然是推托。
 <br/>
 <br/>
 11月13号，陕西省原民办和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代课教师.html" target="_blank">
  代课教师
 </a>
 1000多人集体到省信访局
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 ，这些老师是各县市派出的维权代表，不少老师们胸前挂有一个统一特制吊牌，上面写着〝还我身份、还我待遇、还我尊严〞落款是陕西民师。
 <br/>
 <br/>
 参与维权的老师表示，陕西省各县市只派出数名维权代表，因为人数太多会被政府镇压。当天谈判结果是，政府对教师们上访诉求，会在下月13号前答复，依然跟以前一样推托。
 <br/>
 <br/>
 陕西民办教师张女士：〝我们上无关系，下无关系，就把你开除了嘛，有的工作二十多年，有的是十几年，一个月80块钱，医疗保险都自己交嘛，有主转正的机会，名额都被人家有关系的人占光了嘛，都是生活没办法才上访，政府都一个推一个，一个推一个。〞
 <br/>
 <br/>
 老师们表示，当年他们在农村师资短缺的情况下，投身教育事业，忍受着跟民办教师同工不同酬的不公平待遇，奉献了一生的青春年华，却被迫一刀切下岗。
 <br/>
 <br/>
 陕西民办教师冦女士：〝下岗十年了，一无所有，亏待我们，白白给国家干了，待遇问题嘛，跑了十几年了，上访都没有解决嘛，我们现在都老了，我们靠什么嘛，现在一个月就是80块钱，解决不了我们的生活问题嘛，我们都有家庭的，就是这样对待我们，太不公平了嘛。〞
 <br/>
 <br/>
 据《维权网》报导，不少乡
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_镇政府.html" target="_blank">
  镇政府
 </a>
 的工作人员在13号前，曾给不少教师〝做工作〞，希望他们不要参与集体上访活动。13号当天，省信访大厅内外还有一些〝劝访〞的工作人员跟一些特警。
 <br/>
 <br/>
 新唐人记者熊斌、陈杰采访报导
