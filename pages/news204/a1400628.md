### 美专家：G20是拐点 中美关系或全面恶化
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】在亚太经合（APEC）峰会上，美中双方针锋相对，不欢而散。一周后，美中将迎来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 这个贸战关键点。美专家称，美中双方存在重大分歧，如果在G20上仍僵持不下，两国关系或将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全面恶化.html" target="_blank">
  全面恶化
 </a>
 。
 <br/>
 <br/>
 11月18日，在巴布亚新几内亚莫尔斯比港举行的亚太经合组织峰会，29年来首次未能发表联合公报。峰会期间，中国国家主席习近平和美国副总统迈克．彭斯(Mike Pence)，在贸易和安全问题上隔空叫阵。外界认为，中美分歧是导致联合公报流产的主要原因。
 <br/>
 <br/>
 23日，俄罗斯卫星通讯社报导，美国三一学院经济系教授文贯中在接受采访时表示，美中两国对涉及世界走向的一系列基本概念和原则都有重大分歧。APEC峰会只是开了一个头，今后各种国际经济、政治的场合，都会成为中美激烈角力的场所。
 <br/>
 <br/>
 文贯中表示，即将举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 是一个拐点，如果中美继续针锋相对，两国关系或将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全面恶化.html" target="_blank">
  全面恶化
 </a>
 。
 <br/>
 <br/>
 11月13日，美国副总统彭斯接受《华盛顿邮报》专访时，首次公开提到〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 〞的说法。他指出，中方要想避免与美国及其盟友陷入新冷战，必须〝从根本上〞改变其行为。
 <br/>
 <br/>
 文贯中认为，从彭斯连续几次公开对华喊话来看，他对中共的批评比川普（特朗普）总统还要强硬和全面。但和美国国防部安全报告，以及川普最近在联合国大会上，抨击中共和共产主义的发言精神，是一脉相承的。
 <br/>
 <br/>
 目前，美国两党已达成共识，将中共视为最大战略竞争对手，将从经济、政治、外交、军事层面全方位遏制中共。美国现有的对华政策，直接针对中共以政府为主导的贸易、经济体制。
 <br/>
 <br/>
 事实上，早在半年前，美国、欧盟和日本已形成共识，认为中共现行经济体制不是市场经济体制，和美国一手缔造的基于市场经济原则的现行国际秩序无法相容。
 <br/>
 <br/>
 为此，美国多次强调，中共现有的经济体制是造成美中贸易巨额逆差的根源，中共改变经济结构才是解决
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的根本。
 <br/>
 <br/>
 不过，中共似乎并未有改变的意向。11月20日，美国贸易代表办公室发布301调查报告更新版本。当天，美国贸易代表莱特希泽表示，中共根本没有改变与技术转让、知识产权和创新相关的行为、政策和做法，这类不合理行为仍在继续。
 <br/>
 <br/>
 相反，美方最新报告显示，最近几个月，中共针对美国的网路盗窃行为的频率和复杂性都有所增加。
 <br/>
 <br/>
 川普总统日前表示，中方向美国提交的一份清单漏了4、5个关键内容，其中包括知识产权相关内容，他还不能接受。川普认为，中方应该把这些内容加上。
 <br/>
 <br/>
 中方对此尚未回应，不过在日前召开的WTO会议上，针对美国对中共盗窃知识产权的指控，中共官员予以否认。
 <br/>
 <br/>
 近日，北京当局发布最新文件，指国内或将出现一波企业倒闭潮，股市和房市也会有所波动，进而引发社会矛盾，要求全国厅级以上政府机关最好〝自力更生〞的准备。这似乎透露出，中方对此次G20川习会达成协议，并不乐观。北京并不愿意做出根本性的改变和退让，贸易战前途未卜。
 <br/>
 <br/>
 文贯中则认为，即使美中两国对一系列的经济、贸易分歧达成妥协，休战也只是暂时的。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
