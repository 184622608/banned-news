### 中方对孟宏伟下落保持缄默 传孟在某方面对抗当局
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共公安部副部长孟宏伟近日从法国返回中国大陆后即影讯全无。在法国警方展开调查的同时，中共外交部和公安部对此却没有任何回应。有港媒引述知情人爆料称，孟在某方面对抗当局因而被扣查。此事件引发中国网络舆论的强烈反响。
 <br/>
 <br/>
 据法国内政部发布的消息，孟宏伟今年9月29日从法国里昂乘飞机返回中国大陆后即下落不明。他的妻子10月5日向法国警方报案后，收到了来自网络社交媒体的恐吓短讯以及电话恐吓，法国警方已经为她提供保护。
 <br/>
 <br/>
 目前，法国警方能够确认的是，孟宏伟并不是在法国境内失踪的，法国警方与中共官方保持着沟通，但该案目前没有新的进展。
 <br/>
 <br/>
 现年64岁的孟宏伟2016年11月当选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 ，此后一直与妻儿住在国际刑警组织总部所在地里昂。
 <br/>
 <br/>
 国际刑警组织周五发表的声明称，他们已注意到关于主席失踪的报导，此事与法国和中国当局有关。声明强调，国际刑警组织每日事务是由秘书长负责，而非主席孟宏伟。国际刑警秘书处并表明，该机构将不会再对孟宏伟失踪事件发布消息与回应。
 <br/>
 <br/>
 中共外交部发言人华春莹在例行记者会上被问及孟宏伟的下落时，表示〝无可奉告〞；中共公安部对此则一直保持缄默，未发表任何公开回应。
 <br/>
 <br/>
 香港《苹果日报》日前引述接近调查的人士爆料称，孟宏伟可能在某方面与中国当局对抗，因而一回国就被拘留。
 <br/>
 <br/>
 香港《南华早报》10月5日也引述消息人士透露说，孟宏伟正在中国接受调查。据称，孟宏伟9月29日乘机返回中国大陆，〝一落地〞就被中共纪检部门〝带走〞。目前尚不知道孟宏伟被调查的原因以及他究竟被关在何处。
 <br/>
 <br/>
 上述消息一经公开，中国网络社交媒体上立即〝炸了锅〞，网络评论铺天盖地，其中一些〝涉及敏感议题的言论遭网管删除，但海外中文媒体相关报导下的留言和海外社交媒体上的帖文则得以保存。
 <br/>
 <br/>
 有网友发帖嘲讽：〝中国驻法公民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人间蒸发.html" target="_blank">
  人间蒸发
 </a>
 ，怎么不见近日一天到晚对瑞典、英国厉声责骂的中国外交部出来严正交涉？〞
 <br/>
 <br/>
 有网友留言称：〝血腥多米诺骨牌开始了。〞
 <br/>
 <br/>
 网友aChineseBostonian嘲笑道：〝可以让国际刑警组织孟主席补签一张通缉他本人红色通缉令，事情就可以圆满解决！〞
 <br/>
 <br/>
 网友cui2b8 则评论道：〝派出来的，是贪官；招回去的，也是贪官；失踪的，还是贪官；发红通，是抓贪官；但发红通的，也是贪官；没有被发红通，还是贪官。〞
 <br/>
 <br/>
 据了解，孟宏伟从2004年以来长期担任中共公安部副部长，被视为落马的中共前政治局常委周永康的嫡系。他去年12月被免除了国家海洋局副局长与海警局局长职务，今年4月又失去了公安部党委委员职务，这一连串削职事件令外界猜测他仕途不妙。
 <br/>
 <br/>
 孟宏伟于2016年在印度尼西亚巴厘岛国际刑警大会时被选为主席，当时这项任命曾引发强烈争议，遭到国际人权组织多方持续批评。如今身为国际刑警主席并兼任中共公安部副部长的孟宏伟竟然在回国后失踪，让国际刑警组织的处境十分尴尬。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
