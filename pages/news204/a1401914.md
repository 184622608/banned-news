### 〝乐映影业〞APP关闭  投资人报警不立案
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】一款名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乐映影业.html" target="_blank">
  乐映影业
 </a>
 〞的投资理财APP此前在中共中央电视台等媒体大做广告，自称有70万投资人，总投资额几十亿元。但11月26号起〝乐映影业〞却突然关闭平台，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受害投资人.html" target="_blank">
  受害投资人
 </a>
 报警，警方不予立案，目前已有七、八百人开始维权。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乐映影业.html" target="_blank">
  乐映影业
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受害投资人.html" target="_blank">
  受害投资人
 </a>
 12月4号告诉《新唐人》，〝乐映影业〞在中央电视台、腾讯视频，和一些微信公众号投了大量的广告，并以高利润吸引投资人加入。平台突然关闭后，目前加入维权群的人数已有七、八百人。
 <br/>
 <br/>
 安徽受害投资人吴先生：〝报案的话，上海警方说这个公司地址已经变更到广州那边去了，上海警方不予立案，他说必须到公司所在地去。也有人到广州那边去，他也是（不予立案）。反正两边都是相互推诿嘛，他那个APP平台是已经停掉了，已经没有在营业了。〞
 <br/>
 <br/>
 据了解，〝乐映影业〞在11月24号12点关闭系统提现功能，声称只要投资人交清投资总额20％的个人所得税，马上就可以提现，但投资人发现，在交税后仍然不能提现，于是询问客服。客服声称到26号就会恢复正常，但到了26号，〝乐映影业〞的系统和QQ客服全部都关闭了。
 <br/>
 <br/>
 河南受害投资人张先生:〝光我们河南几百个（投资受害人），我们打钱到那个中国建设银行，前期都能收到钱，后期这个在第六天，说要20%的税，我交了。〞
 <br/>
 <br/>
 河南受害投资人匿名:〝受害人好多啊，最少五千块钱，多的不等，出钱嘛有个利息，钱放着也是放着，然后就投了，他把那个APP给关掉，电话呢打不通。〞
 <br/>
 <br/>
 〝乐映影业〞的受害投资人警告说，类似的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_P2P平台.html" target="_blank">
  P2P平台
 </a>
 还有很多，诈骗的模式都一模一样，这些平台迟早都会关闭，希望其他投资人及早退出，以免蒙受损失。
 <br/>
 <br/>
 公开资料显示，乐映影业投资控股（上海）股份有限公司，于2014年08月1号注册成立，资本为10000万人民币，主要经营影业投资，文化产业投资，实业投资等。公司总裁为电影制片人刘剑华。
 <br/>
 <br/>
 新唐人记者熊斌、周天采访报导
</div>
