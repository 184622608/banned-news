 1. 大纪元时报
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 前夕，中共官员频繁藉助海外媒体散播经济〝威胁论〞，宣称如中美〝全面冲突〞、后果将〝不可想像〞，白宫首席经济顾问库德洛回应说，中方做好该做的，双方就能达成协议，对世界才是好事。

 <br/>
 2. 美国之音
 <br/>
 美国国家卫生研究院(NIH)周三发表声明，对中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生物学家.html" target="_blank">
  生物学家
 </a>
 贺建奎搞出基因编辑婴儿表示〝深度忧虑〞，认为在基因编辑伦理和规范尚未达成共识前修改人类基因是蔑视国际伦理准则的行为。
 <br/>
 <br/>
 3. 德国之声
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电动汽车.html" target="_blank">
  电动汽车
 </a>
 已成为中共政府追踪和监控中国人的手段之一，包括特斯拉、大众、宝马等超过200家汽车制造商被要求把汽车的位置资讯及数十个其它数据发送给政府支持的监控中心。
 <br/>
 <br/>
 4. 明慧网
 <br/>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
