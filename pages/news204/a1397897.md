### 港珠澳大桥造假案 两人被判入狱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 工程报告造假案中，土木工程拓展署承办商的两名前工地实验室技术员和助理，本周五在香港区域法院承认一项串谋诈骗罪，二人因此案分别被判监23个月和21个月。
 <br/>
 <br/>
 据港媒报导，这两名涉案的前实验室人员分别是27岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵志兴.html" target="_blank">
  赵志兴
 </a>
 和26岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潘家伟.html" target="_blank">
  潘家伟
 </a>
 。他们二人被指控在2013年1月1日至2016年7月26日期间，串谋他人更改试验所内与压力测试机连接的电脑资料，包括更改测试的日期、时间，以及利用曾通过测试的混凝土样本或强力混凝土砖，代替要送交测试的样本去做压力测试，企图欺骗署方两名首席技术员及其他人员，隐瞒他们没照规定进行混凝土压力测试的事实。
 <br/>
 <br/>
 辩方律师在为赵志兴和潘家伟陈情时，强调两名被告出身劳工家庭，而他们两人的父亲已相继身亡，他们现在是家庭的支柱。
 <br/>
 <br/>
 律师辩称，此案中他们两人并不是主谋，事发后也坦白承认犯错，而且其所作的虚假测试并不影响大桥的结构，没有对大桥安全产生实质性的影响。
 <br/>
 <br/>
 法官则在宣判时表示，造假〝一次都嫌多〞，因为造假会令市民对公共工程的检测和认证系统失去信心。
 <br/>
 <br/>
 据报导，为解除公众的疑虑，工程报告造假案被踢爆后，政府额外花费了5800万元来进行补救。
 <br/>
 <br/>
 据公开的资讯，2017年年中，土木工程拓展署的监管人员察觉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 的一份检测报告有〝不寻常的迹象〞，经内部调查后决定转交香港廉署跟进。廉署接获土木工程拓展署转交的投诉后，开始对该案展开深入的调查。
 <br/>
 <br/>
 其后，香港廉政公署逮捕了21人，指控他们涉嫌在港珠澳大桥工程的混凝土检测中造假，其中就包括赵志兴和潘家伟。
 <br/>
 <br/>
 该事件令舆论质疑，港珠澳大桥会否因监管不力而演变成〝豆腐渣工程〞。
 <br/>
 <br/>
 （记者欣然综合报导/责任编辑：李泉）
</div>
