### 太极拳师魏雷首次复出 惨败后泣不成声 （视频）
------------------------

 【新唐人北京时间2018年11月03日讯】日前，在峨眉山
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统武术.html" target="_blank">
  传统武术
 </a>
 大赛上，曾被〝格斗教练〞徐晓东KO的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雷公太极.html" target="_blank">
  雷公太极
 </a>
 拳师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魏雷.html" target="_blank">
  魏雷
 </a>
 （又称雷雷）首次复出，对上〝心意拳传人〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王知亮.html" target="_blank">
  王知亮
 </a>
 ，但魏雷仍以惨败告终，赛后泣不成声。
 <br/>
 <br/>
 第六届峨眉山国际武术节10月29日在峨眉山开幕，得胜者可获得100万奖金。31日，销声匿迹近一年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_魏雷.html" target="_blank">
  魏雷
 </a>
 参加比赛，首轮即和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王知亮.html" target="_blank">
  王知亮
 </a>
 对决。
 <br/>
 <br/>
 从网传比赛视频可见，魏雷此次虽然没有被王知亮KO，勉强打完三局，但却打得相当吃力，几乎无还手之力。整场比赛，魏雷都是采用防守招数，只能不断用擒抱等方式躲避危机，并未看到他使出太极的技巧，最后以王知亮单方面的痛殴结束比赛。
 <br/>
 <br/>
 赛后，魏雷接受媒体访问时表示，他在3个回合内，吃了王知亮上百拳。更语带哽咽的说〝太极拳是只适合全民健身的一种运动方式了，它再不是为了打斗而生存，但不代表我们的心是软弱的〞。随后，他长叹一口气后泣不成声，结束了访问。
 <br/>

 <br/>
 去年4月27日，〝杨氏太极拳传承人〞魏雷与格斗教练
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐晓冬.html" target="_blank">
  徐晓冬
 </a>
 比武，魏雷不到10秒就被徐晓冬一招击倒KO。赛后，他的太极拳师傅罗师傅表示，〝雷雷的功夫没学到〞。
 <br/>
 <br/>
 这场比赛也引发各界对中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传统武术.html" target="_blank">
  传统武术
 </a>
 实战价值的探讨，同时也折射了中国大陆传统武术文化丢失后的乱象。
 <br/>
 <br/>
 有人认为，中国功夫被武侠小说神化，到了该回归现实的时候了。也有人认为，传统武术在中共建政后就已经失传了。还有人认为传统武术还是有真功夫的人在，只是太多人为名利诱惑，学了一点皮毛也敢自称大师，影响了传统武术的声誉。
 <br/>
 <br/>
 <h4>
  中共破坏传统武术几近失传
 </h4>
 <br/>
 武术属于中华传统文化的一部分，在中国已经流传了几千年。武术一词，术，在过去一直是包含道术、术类的意思。武术想炼到高深境界对习练者做人、品行都有很高的要求。一个练武者真的达到了高深境界，绝不会轻易出手伤人或是为求名利来表演的。所以〝武〞字拆开是〝止〞和〝戈〞，可能并非巧合。
 <br/>
 <br/>
 传统的中国武术已经失传，主要原因在于1949年中共执政后，开始系统地铲除中国传统文化，武术也是其中之一，一大批武术家被扣上反动会道门的帽子而被镇压枪毙。
 <br/>
 <br/>
 在经过中共改造之后的中国武术，几乎完全失去了传统武术的内涵，传统武术讲究修心炼身加实用，中共改造的比赛套路追求的是花哨好看，这等于是把传统武术给消灭了。
 <br/>
 <br/>
 在新唐人电视台主办的〝全世界华人武术大赛〞任裁判长的李有甫先生曾表示，传统武术在大陆，已经面目皆非了。内家已经不像内家，动作变异。外家也不像外家，就像体操和杂技一样的新武术。
 <br/>
 <br/>
 李有甫表示，那么传统武术的传统究竟体现在哪里呢？以太极拳为例，它源于中国《易经》和道家理论，由明代真人张三丰所创，蕴含深沉的修炼因素。但它的心法已经失传，武德这些精华的东西在传承过程当中已经失去。后来的人为了名、利而乱改，把太极拳改得面目皆非了。
 <br/>
 <br/>
 李有甫还表示，传统武术朴实无华，它有一种自然的美，动作非常的协调，它不过分追求难度和表面的美。它的劲力、力量也很到位。它有内涵的东西在里边。
 <br/>
 <br/>
 据了解，李有甫曾在文革期间，在山西大学的牛棚里找到中国第一位武术教授陈盛甫，并拜陈为师，十年苦练，不但精练了长拳、底功拳、八卦、太极、刀、枪、剑、棍等功夫，还继承和研习了陈盛甫独特的功夫——山西鞭杆。1982年李有甫凭着鞭杆获得全国比赛的冠军。
 <br/>
 <br/>
 练武之余，陈盛甫老师也教他静坐气功、站桩、八段锦、易筋经、五禽戏等气功养生法。后来陈盛甫又把他介绍给了自己的结拜兄弟——山东武术馆馆长陈济生。陈济生教给他〝静功太极108式〞，后来还传了〝活步太极拳〞，〝游身八卦掌〞、〝迷魂掌〞、〝闪剑〞等秘不外传的高深功夫。
 <br/>
 <br/>
 李有甫回忆称，陈老师常对他说，慢练就是快练，当真的慢下来达到入静状态时，在别人眼里你就快得不行。后来他明白这是走了另外空间，两个时空概念不一样。
 <br/>
 <br/>
 李有甫表示，武术是分层次的，最低层武术重点是攻击，高一层的就是防守，再高一层就是不攻也不防，随心所欲，出神入化。
 <br/>
 <br/>
