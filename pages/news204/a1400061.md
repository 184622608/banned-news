### 刘强东性侵案后再爆猛料 传被踢出管理层
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】11月20日，坊间传闻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_踢出京东管理层.html" target="_blank">
  踢出京东管理层
 </a>
 。此前，刘强东因在美国涉嫌性侵被关入监狱的丑闻曝光后，曾有业内人士预测，刘强东很可能会因此失去京东的管理权。
 <br/>
 <br/>
 11月19日晚，京东发布第三季度财报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 等京东部分高管出席财报电话会。据悉，京东新的管理团队已建成，刘强东的角色将会发生转变。
 <br/>
 <br/>
 次日，坊间有传言，刘强东被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_踢出京东管理层.html" target="_blank">
  踢出京东管理层
 </a>
 。有网友提问，这是刘强东将因强奸案被明尼苏达当局起诉的信号？还是仿效马云〝退居二线〞？京东尚未对此回应。
 <br/>
 <br/>
 刘强东性侵事件一直备受舆论关注。8月31日晚间，刘强东在美国明尼苏达涉嫌强奸一名中国女留学生，被当地警方以〝性犯罪指控〞拘捕，关入监狱。9月1日下午，刘强东被保释。
 <br/>
 <br/>
 当日，刘强东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵丑闻.html" target="_blank">
  性侵丑闻
 </a>
 在网路曝光，瞬时掀起舆论狂潮。刘强东被美国警方指控涉嫌强奸一级重罪，若罪名成立将面临最高可判30年监禁。外界猜测，刘强东很可能因此失去京东的管理权。
 <br/>
 <br/>
 〝如果你的公司的头号人物被关在明尼苏达监狱里，那么这可能意味着公司失去决策权，〞安可公关顾问公司驻上海董事总经理詹姆斯．罗宾逊(James Robinson)此前表示。
 <br/>
 <br/>
 有分析指出，刘强东掌握京东79.5%的投票权，权力的过分集中意味着高风险，刘强东的性侵丑闻极有可能令公司运转失灵，失去方向。
 <br/>
 <br/>
 如今刘强东仍面临被诉讼，逮捕甚至判刑的风险。一旦这些情况发生，京东的损失将无可弥补。
 <br/>
 <br/>
 此外，具有中共政协委员头衔的刘强东爆出性侵丑闻，这无疑也让中共当局颜面尽失。网路有文章称，曾经出入中南海的红顶商人刘强东，可能再也难以触摸到中共政商那层天花板了。
 <br/>
 <br/>
 刘强东此后一落千丈，不仅频频缺席重大会议和场合，京东股价也遭受重创，市值跌掉一半。此后，关于刘强东性侵案件细节陆续在网路曝光。11月2日，明尼苏达州《星坛报》曝光了刘强东案女大学生的供词，并根据多方收集的消息，还原了事发当晚的全过程。对于此报导，刘强东的律师表示不方便谈论。迄今为止，刘强东本人一直未公开回应性侵事件。
 <br/>
 <br/>
 刘强东性侵丑闻也频频遭到大陆商界大佬嘲讽，马云曾公开嘲讽刘强东性侵一事，称自己不会出那类事。房地产大老任志强也批评说，刘强东影响了企业的社会形象。
 <br/>
 <br/>
 而网易创始人丁磊则第一时间在网路发表评价，称刘强东是灌酒性侵惯犯，在一次多人参与的宴席上，刘强东把一名女子灌醉，当场在宴席包间的厕所内，对其施暴。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
