### 【禁闻】中共加强网攻 盗窃澳洲公司商业机密
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】
 <h4>
  中共加强网攻 盗窃澳洲公司商业机密
 </h4>
 <br/>
 澳大利亚媒体11月20号报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共黑客.html" target="_blank">
  中共黑客
 </a>
 最近大幅加强对澳洲公司的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络攻击.html" target="_blank">
  网络攻击
 </a>
 ，窃取他们的知识产权,  而这些攻击行动都是在中共国家安全部主导下进行的。
 <br/>
 <br/>
 报导说，虽然中澳去年达成了网络安全协议，承诺互不盗窃知识产权或商业秘密，但澳洲政府高层官员披露说，今年上半年，中共对澳洲的网络攻击行为明显增加，攻击范围涵盖各行各业。
 <br/>
 报导说，其中一个最为活跃的黑客组织是〝APT10〞，这个组织对目标公司发动〝云端跳跃〞（Cloud Hopper）网络攻击，侵入防卫较弱的IT服务提供商。
 <br/>
 <br/>
 多家媒体揭露，〝APT10〞是一个与中共国安部天津分局有关的黑客组织。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙文广.html" target="_blank">
   孙文广
  </a>
  被囚禁在家 警察严密监控
 </h4>
 <br/>
 山东大学84岁的退休教授孙文广，因接受外媒电话直播采访遭中共打压，与外界失联将近100天。
 <br/>
 〝美国之音〞11月20号报导说，孙文广一直被软禁在家，遭到严密监控。
 <br/>
 <br/>
 孙文广的一位支持者说，她和朋友不久前到孙文广家探访时，楼下有人把守，楼上也有人上岗。孙文广的夫人给她们开门，但因看守人员干涉，没能进门。探访者估计，孙文广当时在家。
 <br/>
 <br/>
 孙文广的另一位朋友则披露，孙文广两个月前曾在警察看押下到济南齐鲁医院就医。这位人士表示，国保警察抄走了孙文广所有的手机、电脑等通讯设备，断绝了他与外界的联络。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗.html" target="_blank">
   疫苗
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受害儿童家长.html" target="_blank">
   受害儿童家长
  </a>
  促设基金 保障治疗
 </h4>
 <br/>
 中共当局正在就《疫苗管理法》公开征求意见，但疫苗受害儿童家长到中共国家药品监督管理局，想要递交修改意见，却无人理睬。
 <br/>
 <br/>
 有疫苗受害儿童家长向海外媒体表示，问题疫苗厂商吉林省长春市长生科技公司被政府罚款91亿元。政府应该用这91亿元，设立〝疫苗风险补偿基金〞，对受害者提供前期治疗和后期生活保障。
 <br/>
 <br/>
 河南疫苗受害儿童家长何方美还表示，除了成立补偿基金外，还应该成立专门针对初生婴儿身体检查的部门，判断他们接种疫苗的风险，这样便于确定儿童出现问题的原因。
 <br/>
 <br/>
 编辑/周玉林
