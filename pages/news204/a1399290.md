### 〝大sir今去世〞中港台悲讯不断 3月内10余名人过世
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】人称香港〝大sir〞的前行政局首席非官守议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_钟士元.html" target="_blank">
  钟士元
 </a>
 ，14日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_过世.html" target="_blank">
  过世
 </a>
 ，终年101岁。据不完全统计，过去3个月内，中港台三地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政坛.html" target="_blank">
  政坛
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文艺界.html" target="_blank">
  文艺界
 </a>
 至少10多位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_名人.html" target="_blank">
  名人
 </a>
 过世。
 <br/>
 <br/>
 11月14日，人称〝大sir〞的香港前行政局首席非官守议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_钟士元.html" target="_blank">
  钟士元
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_过世.html" target="_blank">
  过世
 </a>
 。钟士元于1917年于香港出生，1950年获英国雪菲尔大学博士，翌年回港投身工业界，创办崇佳实业有限公司，崇佳后来更一度成为全球最大的电筒厂。
 <br/>
 <br/>
 1958年钟士元参加工业总会筹委会而步入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政坛.html" target="_blank">
  政坛
 </a>
 ，历任立法局首席非官守议员、行政局首席非官守议员，成为当时香港政坛地位最高的华人。
 <br/>
 <br/>
 除钟士元外，梳理过去数月媒体公开报导，中港台三地悲讯不断，一位又一位政坛、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文艺界.html" target="_blank">
  文艺界
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_名人.html" target="_blank">
  名人
 </a>
 过世，仅过去的100天内，至少已有10多名知名人士离世。社群媒体被离世、悼念等贴子刷屏。
 <br/>
 <br/>
 11月13日，台湾著名女作家李维菁，因癌症去世，终年49岁。
 <br/>
 <br/>
 李维菁2010年出版首本小说集〝我是许凉凉〞，曾被选为台北书展文学大奖，随后于2012年出版〝老派约会之必要〞。2015年推出的作品〝生活是甜蜜〞，被香港作家钟晓阳形容是可以创造一个世界的人。
 <br/>
 <br/>
 11月11日，台湾国学大师、中国式管理之父曾仕强，在三军总医院辞世，享年84岁。
 <br/>
 <br/>
 11月3日，有〝靓绝五台山〞之称的香港女演员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓝洁瑛.html" target="_blank">
  蓝洁瑛
 </a>
 被发现倒毙家中，终年55岁。
 <br/>
 <br/>
 蓝洁瑛曾演出周星驰大话西游被称为最美的蜘蛛精，在这部电影前，她就曾在港剧最红的时代叱吒港台，和刘德华、温兆伦、周海媚都是同期，之后，因情绪不稳离开娱乐圈，还传出曾被性侵，加上双亲离世饱受精神打击，不但经济状况不佳，据传甚至靠救济金过活，还被拍到流连街头翻垃圾的场景，令人嘘唏不已。
 <br/>
 <br/>
 10月30日，香港电影教父邹文怀离世，享年91岁。
 <br/>
 <br/>
 10月30日，香港武侠小说泰斗查良镛（笔名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金庸.html" target="_blank">
  金庸
 </a>
 ）去世，终年94岁。
 <br/>
 <br/>
 金庸的代表作包括：《射雕英雄传》、《神雕侠侣》、《倚天屠龙记》、《天龙八部》、《笑傲江湖》、《鹿鼎记》等，作品也多次被改编成电视、电影。
 <br/>
 <br/>
 10月29日，中共前领导人刘少奇的大舅哥、有〝红色资本家〞之称的中共前全国人大副委员长、香港光大集团创办人王光英去世，终年100岁。
 <br/>
 <br/>
 10月25日，央视主持人李咏因患癌在美国治疗期间去世，终年50岁。
 <br/>
 <br/>
 10月20日，中共驻澳门特别行政区联络办公室主任郑晓松跳楼身亡，终年59岁。
 <br/>
 <br/>
 10月19日，曾卷入绑架事件及争产风波的新鸿基地产前主席郭炳湘病逝，终年68岁。
 <br/>
 <br/>
 10月20日，香港资深演员岳华逝世，终年76岁。
 <br/>
 <br/>
 9月23日，2009年诺贝尔物理学奖得主、〝光纤之父〞高锟教授因肺炎病逝，终年84岁。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
