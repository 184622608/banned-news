### 非洲猪瘟疫情又〝攻陷〞三省 中共〝有效处置〞失灵
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟疫情.html" target="_blank">
  非洲猪瘟疫情
 </a>
 疑似在大陆失控，中共农业农村部30日发布疫情通报称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山西.html" target="_blank">
  山西
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南.html" target="_blank">
  湖南
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_云南.html" target="_blank">
  云南
 </a>
 三省同时新增非洲猪瘟疫情。外界发现，中共官方每通报一起疫情，都声称〝疫情已得到有效处置〞，但讽刺的是，连日来受波及的城市不断增加。
 <br/>
 <br/>
 北京时间10月30日，中共农业农村部新闻办公室发布疫情通报称：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山西.html" target="_blank">
  山西
 </a>
 省阳曲县、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南.html" target="_blank">
  湖南
 </a>
 省元陵县和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_云南.html" target="_blank">
  云南
 </a>
 省普洱市思茅区，各发生一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟疫情.html" target="_blank">
  非洲猪瘟疫情
 </a>
 。
 <br/>
 <br/>
 通报称，截至目前，山西省太原市阳曲县养殖户存栏生猪210头，发病75头，死亡47头；湖南省怀化市元陵县养殖户存栏生猪144头，发病25头，死亡22头；云南省普洱市思茅区养殖户存栏生猪36头，发病5头，死亡1头。
 <br/>
 <br/>
 非洲猪瘟疫情自8月在大陆爆发以来，中共官方通报每起疫情时都说〝该起疫情已得到有效处置〞，但讽刺的是，连日来被非洲猪瘟疫情攻陷的城市不断增加。
 <br/>
 <br/>
 有分析认为，实际情况很可能是，中共官方对非洲猪瘟疫情已束手无策，只能采取低调报导降温的手段，收紧言论。
 <br/>
 <br/>
 近日，一位参与防控工作的非洲猪瘟专家徐女士对自由亚洲说，疫情在大陆蔓延的情况比想像的要严重。当局对最能掌握疫情变化的实验室和疫控人员，已进行严管，不允许将疫情情况向外界洩露，否则就是泄密。
 <br/>
 <br/>
 目前，非洲猪瘟已经肆虐了大半个中国，但徐女士表示，即便是业内人士，也不清楚疫情到底会发展到何种程度。对此很是担忧。
 <br/>
 <br/>
 另一方面，中共官媒一直对民众宣传，吃猪肉没有风险，可以放心吃。而对一些仅仅在社交媒体上提醒亲友少吃猪肉的网民，却进行抓捕、管控。
 <br/>
 <br/>
 不过，仍有许多猪瘟疫情爆发地的网友表示，现在已经不敢吃猪肉了。
 <br/>
 <br/>
 对此，徐女士指出，猪肉并不是安全的，非洲猪瘟病毒是一类病毒，致病致死率甚至高达100%。而且它和禽流感病毒一样，也有变异的可能，很可能令人感染猪瘟病毒。
 <br/>
 <br/>
 根据中共官方通报，迄今疫情已蔓延至辽宁、黑龙江、河南、江苏、安徽、浙江、内蒙古、吉林、天津、山西、云南、湖南、贵州，并有进一步失控的迹象。
 <br/>
 <br/>
 非洲猪瘟疫情不仅肆虐中国大陆，还对周边邻国造成威胁。目前，韩国、日本均已发现源自大陆的猪肉制品中，携带有非洲猪瘟病毒。联合国粮农组织对此非常担忧，早已发布警告称，非洲猪瘟疫情不可避免将扩散至亚洲其他国家。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：戴明)
