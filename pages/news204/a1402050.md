### 重庆一公安副局长死在办公室 官方紧急晒〝遗书〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共官员.html" target="_blank">
  中共官员
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀.html" target="_blank">
  自杀
 </a>
 事件频现。陆媒消息称，重庆市荣昌区公安局常务副局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴修远.html" target="_blank">
  吴修远
 </a>
 4日上午，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_办公室.html" target="_blank">
  办公室
 </a>
 自杀身亡。当地官方紧急通报称，吴修远
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遗书.html" target="_blank">
  遗书
 </a>
 自证
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_精神抑郁.html" target="_blank">
  精神抑郁
 </a>
 。
 <br/>
 <br/>
 12月4日，重庆市荣昌区政府通报称，重庆市荣昌区公安局常务副局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴修远.html" target="_blank">
  吴修远
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遗书.html" target="_blank">
  遗书
 </a>
 反映他
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_精神抑郁.html" target="_blank">
  精神抑郁
 </a>
 ，还因有亲属病重，深感压力巨大。
 <br/>
 <br/>
 吴修远最近一次公开活动，是在11月15日出席座谈会，与到访的大足区公安局展开经验交流活动。
 <br/>
 <br/>
 事实上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共官员.html" target="_blank">
  中共官员
 </a>
 因〝抑郁症〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀.html" target="_blank">
  自杀
 </a>
 早已司空见惯。
 <br/>
 <br/>
 据中共公开报导，今年以来，至少有10多个中共官员自杀，多因〝抑郁症〞，方式以跳楼居多。
 <br/>
 <br/>
 最具爆炸性是10月20日晚，澳门中联办主任郑晓松在住所坠楼身亡，终年59岁。事发后，澳门司警局强调案件仍在调查，但港澳办迅速定调称郑晓松因患抑郁症坠楼。
 <br/>
 <br/>
 11月1日凌晨，内蒙古公安厅副厅长、呼和浩特市公安局局长李志斌，在该市公安局休息室内缢亡。官媒报导中特别强调，李志斌留在现场的〝遗书〞中称，自己〝一直被抑郁症困扰，每天都靠高效安眠药才能入睡〞。
 <br/>
 <br/>
 此前，中共官方宣布官员自杀原因时，多引用死者身边人或医生的说法来引证自杀官员患有〝抑郁症〞。因饱受民间调侃，如今中共官方开始让死者用〝遗书〞自证〝抑郁症〞。
 <br/>
 <br/>
 有消息透露，去年中国有2,000名官员自杀身亡，不少自杀者都被定调为〝抑郁症〞。网民嘲讽：在中国，普通人都是患了抑郁症才自杀，而中共官员都是自杀了才患抑郁症。
 <br/>
 <br/>
 有评论人士说，中共高官早已成为一个高危行业。在中共这种邪恶的体制下，没有任何一个人是安全的，随着中共政权危机的加剧，预计未来将有中共更多的官员出现所谓自杀和被自杀的情况。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
