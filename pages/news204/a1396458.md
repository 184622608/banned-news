### 习近平南巡罕见低调 港媒曝4大〝诡异〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 23日举行通车仪式，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 主持了这个被称为〝世纪工程〞的大桥揭幕礼。不过，习此次南下的安排极为低调，揭幕礼后更是行踪成谜，对此，港媒归纳其中的4大〝诡异〞之处。
 <br/>
 <br/>
 香港苹果日报23日报导说，中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 趁南下视察期间，主持了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 的揭幕礼。不过，习近平此次南下及主持大桥揭幕礼的安排都是极为低调，与过去历届的领导人高调做法大相迳庭。主要表现为以下4大要点：
 <br/>
 <br/>
 1，极度低调。过去多日，习近平将南下视察的消息盛传，但官方对习的行程一直保持神秘，直至22日下午习抵达珠海，才有人在上午〝吹风〞透露行踪。
 <br/>
 <br/>
 2，罕见迟到。23日大桥揭幕礼数分钟前，大会仍拒绝公布习近平是否会出席，在大会首排座位上，留有一张无名牌的凳子，也没有习的名字。更罕见的是，习近平一行迟到半小时才现身。
 <br/>
 <br/>
 3，来去匆匆。港珠澳大桥揭幕礼前，香港政务司司长说，相信习近平会在仪式上发言，但习上台只说出12个字，〝我宣布，港珠澳大穚正式通车〞，便走下台，始终不见习的笑容和开心表情。
 <br/>
 <br/>
 4，安检严密。19日起珠海便加强了安保措施，大批警察和便衣人员在主要干道巡逻把守；深圳全面禁飞无人机；出席仪式的嘉宾23日要求接受严密安检，甚至连手机也不可带入场。揭幕仪式后，习近平匆匆离开现场，行踪成谜。
 <br/>
 <br/>
 不过，有香港媒体记者看到习近平的车队上了港珠澳大桥。英媒BBC消息称，习近平将赴深圳前海经济区。
 <br/>
 <br/>
 报导引述一名接近深圳政府的匿名知情人士透露，习近平此行，还将前往深圳前海经济区。今年是中国改革开放40周年，广东一直被认为是改革开放的前沿。
 <br/>
 <br/>
 分析认为，习近平在中美贸易战火升高之时考察广东，是为了提振国内外对于中国经济和改革开放的信心。
 <br/>
 <br/>
 港珠澳大桥的通车仪式前，官媒22晚报导说，习近平
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南下调研.html" target="_blank">
  南下调研
 </a>
 ，当天考察了珠海横琴新区粤澳合作中医药科技产业园及格力电器厂。
 <br/>
 <br/>
 在考察期间，习近平说，要从大国发展到强国，〝实体经济发展至关重要〞，他更称〝制造业的核心就是创新〞，要求企业〝加快增强自主创新能力和实力〞。
 <br/>
 <br/>
 22日下午，习近平到珠海横琴新区，考察首个落地项目中医药科技产业园。他说，这是10年来第4次到横琴，要让横琴充满创新发展活力，促进澳门经济适度多元化发展。
 <br/>
 <br/>
 在考察格力电器，参观精密模具车间和重点实验室期间。习近平说，从大国到强国，实体经济发展至关重要，制造业的核心就是创新，就是掌握关键核心技术，必须靠自力更生奋斗，靠自主创新争取，希望所有企业都朝着这个方向去奋斗。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
