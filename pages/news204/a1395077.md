### 崔永元举报上海警方 政法委：深查下去
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】大陆主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 日前在微博举报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海警方.html" target="_blank">
  上海警方
 </a>
 参与娱乐圈〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大欺诈.html" target="_blank">
  大欺诈
 </a>
 〞，将这把娱乐圈的火烧到了上海警方身上。对此，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共政法委.html" target="_blank">
  中共政法委
 </a>
 机关报罕见发声，表态应该对崔永元的举报深查。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 因举报电影《大轰炸》与演员签订〝阴阳合同〞搅动大陆娱乐圈。日前，大陆女艺人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 涉税案刚刚尘埃落定，崔永元又开始炮轰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海警方.html" target="_blank">
  上海警方
 </a>
 涉嫌违法违纪。
 <br/>
 <br/>
 10月7日，崔永元在其微博发布《一声长叹一声雷》的文章，暗指上海警方参与娱乐圈〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大欺诈.html" target="_blank">
  大欺诈
 </a>
 〞。
 <br/>
 <br/>
 他在文中指，上海公安局经侦大队对所有他参与的公司彻底侦察，对他以前的助理不断询查，彻夜询查。原因在于〝大轰炸〞，参与这次大欺诈的既有演艺界大腕，也有上海经侦的警察。
 <br/>
 <br/>
 崔永元披露，这些上海警察〝曾经当着我的面，他们喝两万一瓶的酒，抽一千一条的烟，几十万的现金用个书包就提走。〞
 <br/>
 <br/>
 对此，上海警方发通报称，多次尝试联系崔永元但未果，希望其能主动配合调查。崔永元则怒怼上海警方，警方已把他的公司查个底朝天，居然联系不上？随后他更指名道姓地提到，〝你们（上海警察）内部问题，可以先查原宁经侦副支队长，他儿子叫彭明达。〞
 <br/>
 <br/>
 崔永元与上海警方的一连串互动，将这把娱乐圈的火烧到了上海警方。对此，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共政法委.html" target="_blank">
  中共政法委
 </a>
 机关报罕见发声。北京时间10月11日，中共《法制日报》刊文称，既然崔永元捅破这层纸，就该深查下去。
 <br/>
 <br/>
 报导称，崔永元的回应让上海警方尴尬，但比尴尬更要紧的是，如何对待崔永元的实名公开举报。
 <br/>
 <br/>
 报导提到，崔永元此前举报范冰冰涉阴阳合同案，不仅把矛头指向了整个娱乐圈，更指向了税务部门，导致范冰冰被重罚近9亿人民币，涉事的税务官员亦被查。不但艺人们要倒霉，还会有一批体制内的人要丢官。因此上海警方不必打嘴仗，应尽快拿出调查结论来说话。
 <br/>
 <br/>
 报导还称，要保护好崔永元。他以一己之力对抗整个娱乐圈，已经够遭恨，而举报上海警方，更是一步险棋，麻烦更大。
 <br/>
 <br/>
 报导最后称，崔永元为很多局外人揭开了娱乐圈天价片酬的内幕。如果演员天价片酬、偷税漏税只是个人行为是不可能走到今天的，其背后的利益链之深之长，难以想像。崔永元无意中触动一些深层次的问题，很多部门很多人都可能会卷入其中。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
