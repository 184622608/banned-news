### 传刘鹤即将访美 手中握有几张便条
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】海外亲共传媒日前引述消息人士爆料称，中共国务院副总理刘鹤即将访美进行贸易谈判，手中握有几张便条，引起网友好奇。另有美媒亦引述消息人士透露，中方已经交给美方一份方案，为本月底的中美两国首脑在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上举行双边会谈开路。
 <br/>
 <br/>
 据编辑部设在北京的〝海外〞中文媒体多维网15日引述消息人士披露，刘鹤即将前往美国就一份清单进行谈判。据称，刘的手中〝握有几张便条〞。
 <br/>
 <br/>
 上述传闻随即引发中国网友的好奇，纷纷留言拿刘鹤手中的那几张〝便条〞开涮。有人调侃：〝难道是习总密旨？〞有人猜测：〝怕谈判时有疏漏，提前写几条谈判要则？〞还有人调侃说：〝或许是‘高人’秘授的锦囊妙计？估计谈不下去时拿出来看看。〞
 <br/>
 <br/>
 此前，中共商务部在北京时间15日举行记者会时，有记者提问：能否证实外界有关刘鹤已经与美国财长努钦（Steven Terner Mnuchin）通过电话，并将在近期访美的消息？
 <br/>
 <br/>
 当时中共商务部发言人高峰回答称：〝11月1日中美两国元首通话后，双方经贸领域高层接触已经恢复，同时，工作团队正在保持接触，以落实中美两国元首通话达成的共识。〞
 <br/>
 <br/>
 就在15日这一天，美国有线电视新闻网（CNN）也刊文爆料称，据两位〝得到对话简报〞的消息人士透露，本月初习近平和川普（特朗普）通话后，中方已经交给美国政府一份方案，想要在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上双方首脑会面前，推动已停滞的双边贸易谈判。
 <br/>
 <br/>
 不过，其中一位消息人士表示，这次北京给出的方案并未包含白宫反覆提出的几项必须包含在内的内容，比如强迫技术转让和保护智慧产权等问题，所以这个方案仍然只是属于〝新瓶装旧酒〞的性质，只不过中方这次〝改了配方〞，例如：方案里包含了放宽对外资的管制，在一些行业领域里取消合资限制条件等。中方只是为了给双方形成协议框架打下个基础而加了一点〝赌注〞。
 <br/>
 <br/>
 此前，《华尔街日报》11月12日的一篇报导曾披露，本月9日努钦与刘鹤已电话沟通，美方希望中方先提出一个明确方案，才好开启贸易协议的谈判，但北京希望先谈，再给出确实的方案。
 <br/>
 <br/>
 该报导称，9日的电话沟通并未化解中美双方的僵局，但〝仍被视作达成共识的重要一步〞。
 <br/>
 <br/>
 （记者欣然综合报导/责任编辑：明轩）
