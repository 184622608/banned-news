### IMF分析：若贸易战全面打响 中国经济损失惨重
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】国际货币基金组织（IMF）10月9日公布经济分析报告，称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 若全面打响，将对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 造成沉重打击，会损失惨重。
 <br/>
 <br/>
 据香港《南华早报》10月9日报导，由于中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 战的很大一部分影响要到明年才会感受到，IMF将美国2019年经济增长率预估从此前的2.7%调降至2.5%。
 <br/>
 <br/>
 IMF首席经济学家奥伯斯法尔德(Maurice Obstfeld)在报告中称，尽管美国目前需求状况良好，IMF仍将调降2019年美国增长预估，这是因为美国近期针对大规模中国商品
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加征关税.html" target="_blank">
  加征关税
 </a>
 的措施，以及中共的反制举措。
 <br/>
 <br/>
 IMF还预测，若美国对所有中国进口商品加征关税，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 全面打响，将对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 造成沉重打击。中国经济增速将下降1.6个百分点。从2017年的6.9%下降到2018年的6.6%，2019年会继续下降到6.2%。
 <br/>
 <br/>
 目前，中共正在为即将面临的金融危机做最坏的打算。10月7日，中共央行宣布将从本月15日起，再次下调人民币存款准备金率1个百分点，这也是今年以来第四次降准。估计可释出7500亿人民币的资金，但此举将导致人民币进一步贬值。
 <br/>
 <br/>
 就在中共央行公布降准消息的次日，中国股汇市双双大跌，人民币汇率再次跌近破7，显示贬值预期明显升温，中共管理层再次面临挑战。
 <br/>
 <br/>
 而大陆A股沪指、深成指、创业板三大股指齐跌，均超过3%。据华尔街见闻消息显示，截至收盘，沪股通净流出73.08亿元，深股通净流出22.96亿元。总计流出资金96.04亿元，创下年内资金流出第二高。
 <br/>
 <br/>
 独立金融分析师豪伊（Fraser Howie）认为，中共目前正面临2008年全球金融危机以来最糟的时刻。
 <br/>
 <br/>
 新加坡银行固定收益主管尼克森（Gareth Nicholson）则分析称，虽然中共多次宣称〝不怕打〞贸易战，但从降准的决定来看，中共是真的有些紧张了！
 <br/>
 <br/>
 尼克森还指出，若美国经济持续表现强劲，中美贸易战的不断升级将导致越来越多投资者将资金撤出中国。如果发生这样的情况，将使正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经历增速.html" target="_blank">
  经历增速
 </a>
 减慢的中国经济雪上加霜。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
