### 港珠澳大桥不是人人都能上 法媒：必须符合2条件
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】历时9年，耗资1200亿人民币的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 ，10月23日由习近平亲自宣布开通。但法媒透露，并不是所有人都可以驾驶私家车上这座桥的，只有那些富有且拥有政治关系的人，才能使用港珠澳大桥。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 总长达55公里，水下隧道的长度为6.7公里，是世界上最长的海桥和隧洞系统，大桥将香港、澳门、珠海三地连接起来。中国工程师们称，该桥大大缩短了中国大陆与香港和澳门之间的旅行时间。
 <br/>
 <br/>
 不过，法新社10月23日报导称，并不是所有人都可以驾驶私家车上这座桥的，得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_符合2个条件.html" target="_blank">
  符合2个条件
 </a>
 。只有那些富有，并且拥有政治关系的人，才能使用港珠澳大桥。
 <br/>
 <br/>
 报导称，北京当局将对一万辆拥有大陆-香港双板车牌的私家车，发放可以上桥的牌照，澳门只能获得300个类似牌照。
 <br/>
 <br/>
 申请这种牌照的人必须符合严格的标准，例如必须属于中共某个政府部门，必须在大陆上缴了税款，或者对中共慈善机构捐献过5百万元人民币。
 <br/>
 <br/>
 外界质疑，如此严苛的要求，何时才能让这座耗资千亿的大桥收回成本。
 <br/>
 <br/>
 对此，自由亚洲22日刊登评论文章指，港珠澳大桥是为政治理由而建，用政治压倒经济，必然会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蚀大本.html" target="_blank">
  蚀大本
 </a>
 。
 <br/>
 <br/>
 文章称，按规定，无论是澳门人开车到香港，或者香港人开车到澳门，都必须在管制区（口岸）转乘，香港政府只建650个车位，比澳门的3000个还少。质疑这过千亿的工程如何回本？
 <br/>
 <br/>
 据香港政府公布的数据显示，港府向立法会申请了超过1200亿港元建桥拨款，且大桥严重超支逾100亿，香港市民对此表示强烈不满。
 <br/>
 <br/>
 更荒谬的是，要驾车上港珠澳大桥居然还要〝过五关〞。首先要向香港、澳门与中国大陆同时申请或报备许可证，随后要买〝三地保险〞，出发前还要在网上预定停车泊位。如此一来，驾车由香港去澳门，只停留12小时不过夜，预计最少要花费1500港元。
 <br/>
 <br/>
 另外，港珠澳大桥要求香港、澳门车辆，都必须按照大陆的〝右上左落〞的方向行驶，而且大桥只收香港与澳门都不使用的人民币。文章指，这种完全不以方便为本，不以利民为本，只为〝政治正确〞去一国压倒两制的做法，其结果是政治压倒经济，大桥必然会蚀大本收场。
 <br/>
 <br/>
 此外，港珠澳大桥于24日仓促通车运营，但大桥配套基建却尚未完成。亲共的香港建制派议员、民建联立法会议员周浩鼎，对此〝忧心忡忡〞。
 <br/>
 <br/>
 他于21日在香港电台节目中表示，大桥香港口岸仅连接青屿干线，相关屯门赤鱲角连接线未能同步启用，一旦出现突发事故，那儿就变成了一个孤岛，是〝死路一条〞。
 <br/>
 <br/>
 另外，环保人士也警告说，该大桥对珠江三角洲，已经濒临灭绝的中华白海豚的生态环境，带来致命的负面影响。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
