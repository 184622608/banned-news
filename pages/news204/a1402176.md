### 任正非女儿孟晚舟被捕 牵出华为家族神秘身世
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】中国通讯设备巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的女儿、华为副董事兼首席财务官孟晚舟，在加拿大被捕，引爆舆论。同时牵出孟晚舟家族神秘身世。孟晚舟随母亲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟军.html" target="_blank">
  孟军
 </a>
 姓，网传孟军的父亲曾任副省长，任正非靠孟军才飞黄腾达。
 <br/>
 <br/>
 加拿大《环球邮报》5日报导称，加拿大警方，以中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 涉嫌违反美国对伊朗的出口禁令，而逮捕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 长女，即华为副董事长、全球首席财务官孟晚舟。
 <br/>
 <br/>
 加拿大司法部发言人说，孟晚舟于12月1日在温哥华被捕，定于7日举行保释听证。
 <br/>
 <br/>
 孟晚舟是华为创办人任正非与第一任妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟军.html" target="_blank">
  孟军
 </a>
 所生的长女，毕业于华中理工大学，拥有硕士学位，1993年进入华为工作。
 <br/>
 <br/>
 1997年，孟晚舟赴华中理工大学读硕士，一年半后重新回到华为，在华为内部做过财务负责人。2005年起，孟晚舟主导在全球建立了五个共享中心。2011年4月起担任华为常务董事兼CFO。现任公司CFO、副董事长。担任华为系旗下共11间公司的董事。
 <br/>
 <br/>
 孟晚舟在华为蛰伏了20年，直到2013年1月21日，她才以华为首席财务官（CFO）身份首度亮相于公众面前。当时透社报导说，孟晚舟曾是香港天通科技有限公司的董事会成员，该公司后来试图向伊朗最大的行动电话运营商出售禁运的Hewitt Packard（HP）计算机设备。
 <br/>
 <br/>
 天通科技的销售提案至少有13页被标记为〝华为机密〞，并有着华为公司的标识。至此，孟晚舟的名字才真正和华为创始人任正非女儿的身份联系起来。
 <br/>
 <br/>
 人们好奇，孟晚舟为什么姓孟不姓任呢？任正非曾经就此说明，孟晚舟之所以姓孟，是因为任正非曾经从孟晚舟的姥爷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟东波.html" target="_blank">
  孟东波
 </a>
 身上学到了广阔的视野和不屈不挠的精神，所以让女儿随了姥爷的姓。
 <br/>
 <br/>
 而孟晚舟却直言，名字是自己在16岁那年自己改的，改随母姓。
 <br/>
 <br/>
 而背后的原因，网路猜测众多。任正非共有3任妻子，分别是孟军、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚凌.html" target="_blank">
  姚凌
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏薇.html" target="_blank">
  苏薇
 </a>
 。孟晚舟是任正非与孟军所生。
 <br/>
 <br/>
 网传孟军的父亲孟东波曾经担任过副省长，孟东波的领导
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨超.html" target="_blank">
  杨超
 </a>
 则担任过中共国家领导人的秘书，任正非正是靠孟军才飞黄腾达。
 <br/>
 <br/>
 当年任正非和孟军结婚后，经常去看望岳父孟东波和杨超，并从他们身上学到了不少东西。1982年，任正非随孟军来到深圳，孟军做了南油集团的高管，任正非做了下属一家电子厂的领导。在公开场合，任正非称，两人是因理念不合而离婚。
 <br/>
 <br/>
 但坊间传闻，任正非擅长利用职权，玩弄女性。任正非与孟军的离婚是否与此有关，不得而知。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
