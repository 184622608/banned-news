### 【禁闻】3名中国人在中非被打死 中共低调
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】
 <b>
  维权网：贵阳72岁
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
   法轮功学员
  </a>
  被非法
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_判刑.html" target="_blank">
   判刑
  </a>
 </b>
 <br/>
 <br/>
 据中国〝维权网信息中心〞10月6号报导，贵阳市72岁高龄的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 陈贤忠，9月26号被中共贵阳市南明区法院非法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_判刑.html" target="_blank">
  判刑
 </a>
 4年。
 <br/>
 <br/>
 陈贤忠是在去年1月8号与另外9名法轮功学员一起在家里被绑架的，关押在贵阳市南明区看守所一年多之后，再被非法判刑。
 <br/>
 <br/>
 据报导，陈贤忠在法庭上说：自己只是告诉人们做好人，没有违法，要求无罪释放。
 <br/>
 <br/>
 〝维权网〞评论说，中共当局对法轮功信仰者的残酷迫害和镇压从未停止，每个月都不断有法轮功人士被抓捕和判刑，法轮功仍是当下遭受镇压最严厉的群体之一。
 <br/>
 <br/>
 <b>
  3名
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
   中国人
  </a>
  在
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中非.html" target="_blank">
   中非
  </a>
  被打死 中共低调
 </b>
 <br/>
 <br/>
 10月4号，三名中国公民在中非共和国被村民打死，村民表示：他们要为当地一名青少年的死亡负责。
 <br/>
 <br/>
 据当地媒体报导，一个大约由10名中国男子组成的团队招募这名青少年做向导，带他们前往一处金矿。小船在途中翻船，造成这名当地人死亡。
 <br/>
 <br/>
 当这些中国人回到村警察所做陈述时，愤怒的村民用棍棒殴打他们，三名中国男子被打死,  另外三人重伤。
 <br/>
 <br/>
 事发两天后，中共驻中非大使馆才在官网刊发消息，声称已向中非官方提出严正交涉，但对西方媒体的询问不做回复。
 <br/>
 <br/>
 而在国内仅仅发布了简讯。
 <br/>
 <br/>
 对比中共官方在中国游客瑞典旅馆撒泼、央视记者孔琳琳大闹英国保守党年会事件上的激烈反应，大陆有网友讽刺说：〝被打死的不如被赶出旅馆的〞。也有网友质问：〝这不是比瑞典所谓的辱华事件更严重吗？中共外交部为什么不抗议？〞
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五星旗.html" target="_blank">
   五星旗
  </a>
  变垃圾袋 引网友议论
 </b>
 <br/>
 <br/>
 10月5号，有网友在微博爆料说，在河北高速公路定兴收费站，发现有一袋用旧五星旗包裹的垃圾，放在服务区南侧。
 <br/>
 <br/>
 中共定兴县委宣传部当晚发通报说，是两名清洁工的无心行为，已要求河北省高速公路的主管单位对她们〝严肃批评并更换〞。
 <br/>
 <br/>
 消息在网络引发广泛议论。
 <br/>
 <br/>
