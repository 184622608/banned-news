### 北京将落实〝个人诚信分〞制度 严控百姓
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】首先来关注中国的消息。中共持续加强监控百姓，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 将在2020年底前建成覆盖全部常住人口的〝个人诚信分〞工程，这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_积分制度.html" target="_blank">
  积分制度
 </a>
 将伴随民众终身。
 <br/>
 <br/>
 11月19号，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 市委发布《北京市进一步优化营商环境行动计划》，正式确定在2020年底前完成北京市社会信用条例立法，信用积分高的个人或企业法人未来将在市场准入、公共服务、旅游出行、创业求职等通道常保绿灯，信用积分低者则处处受限、各方面寸步难行，这个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_积分制度.html" target="_blank">
  积分制度
 </a>
 将伴随民众终身。
 <br/>
 <br/>
 中共国务院此前公布，预计在2020年前建立〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_社会信用体系.html" target="_blank">
  社会信用体系
 </a>
 〞，透过科技记录金融信用、个人行为和企业不当行为等，藉此给予公民和企业奖励或惩罚。但原有计划中并未提及分数制，而北京市19号公布的新计划却试图藉〝个人诚信分〞来控制全市的商业环境。
 <br/>
 <br/>
 山东大学张教授：〝有了现代的大数据，有各种各样这种东西，它首先想到的是如何来控制这个社会。民运、维权、知识圈非常担心评分制成为控制异议人士的一种工具。这个都是侵犯人权的，违背普世价值的。〞
 <br/>
 <br/>
 北京推出这项政策，让人联想到欧威尔（George Orwell）笔下的反乌托邦小说《1984》，这部小说描述了极权国家政府对民众无所不在的监控和操控。
 <br/>
 <br/>
 湖南维权人士何家维：〝这种邪恶政权怎么样过年杀猪，对我们老百姓割韭菜，老百姓在我手里面，我手里有枪，我手里有权，我说怎么办就怎么办。〞
 <br/>
 <br/>
 张教授：〝现在中国政府面临最大的问题还是怎么能够维持社会的政治稳定、经济稳定，其实它就是强调控制社会的这么一种努力。〞
 <br/>
 <br/>
 根据计划，未来将使用数据、行为与措施等3项清单，为北京市民和企业建构〝个人诚信分〞试验系统。虽未透露如何计算分数，却明确指出有关资讯可能影响个人未来做生意、担任公职、旅游、求职、创业等方面。
 <br/>
 <br/>
 新唐人记者常春、陈洁、施宜君采访报导
</div>
