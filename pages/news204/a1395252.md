### 湘男骗保假装车祸身亡 妻带一双儿女殉情
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南新化县.html" target="_blank">
  湖南新化县
 </a>
 日前发生一起殉情悲剧，一名男子为骗取上百万保险金，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假装车祸身亡.html" target="_blank">
  假装车祸身亡
 </a>
 。妻子悲痛不已，遂携带一双年幼儿女殉情。男子获悉后情绪崩溃，赴警局自首。
 <br/>
 <br/>
 10月11日上午10时许，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南新化县.html" target="_blank">
  湖南新化县
 </a>
 琅塘镇的戴某花及两名孩子的尸体，在该镇一水塘被打捞上来，经公安部门现场勘察，三人均是溺水身亡。
 <br/>
 <br/>
 据陆媒报导，戴某花现年31岁，从小父母双亡，与34岁的何某结婚后，育有一子(4岁）一女(3岁）。9月7日，何某驾驶着一辆借来的车辆离奇失踪。9月19日，失踪车辆在资江河中被发现，但何某生死不明。
 <br/>
 <br/>
 据悉，何某在出事前欠下10余万巨额债务。何某失踪后，戴某花非常痛苦，她相信丈夫已经身亡，遂萌生自杀的念头。
 <br/>
 <br/>
 10月10日中午11时30分，戴某花带着女儿在幼儿园接走了儿子。中午12时27分，她在微信朋友圈留下了遗书。
 <br/>
 <br/>
 遗书中说，〝说好一起慢慢变老，一起离开，怎么能舍得你单独离去呢，所以宝贝，老婆来陪你了，我只想一家四口在一起，此时做这样的选择我也好害怕，害怕没成功死去会痛苦，如果还有一辈子，我仍然会选择嫁给你，不知你是否还愿意娶我，现在虽不知你是否还活着，但每天这样思念你，已让我没有活下去的念想，我更没有勇气承担外界的压力言论而活着。〞
 <br/>
 <br/>
 〝我本想自己独自离开，可儿子女儿没有爸爸妈妈陪伴，他们会很痛苦，他们也会像我一样受人欺负，所以只能带他们一起离开，长痛不如短痛。〞
 <br/>
 <br/>
 然而，就在人们为戴某花母子三人殉情感到悲痛时，意想不到的事情发生了，原来何某并没有死，他一直躲藏在外地。当得知妻儿自杀身亡后，何某情绪崩溃，于10月12日到新化县公安局自首。
 <br/>
 <br/>
 经查，何某为偿还10余万的网路贷款，欲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假装车祸身亡.html" target="_blank">
  假装车祸身亡
 </a>
 以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_骗取保费.html" target="_blank">
  骗取保费
 </a>
 。9月7日，他瞒着妻子在某保险公司购买了一份100万元的人身意外保险。
 <br/>
 <br/>
 9月19日，何某驾驶借来的车辆，在新化曹家镇城坪村资江河段伪造坠河现场，制造车毁人亡假象，企图骗取保险金。而受益人是妻子戴某花。
 <br/>
 <br/>
 何某称，之所以瞒着妻子，是想让她顺利拿到赔偿金。不过，当他知道妻子和一双儿女死亡后，已经崩溃。目前，何某因涉嫌故意损坏财物罪和保险诈骗罪被刑拘。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
