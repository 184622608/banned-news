### 范冰冰寒蝉效应？中共限酬令艺人难回击
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 被指控逃税，遭重罚超过8亿人民币，现在中国娱乐圈似乎掀起寒蝉效应，浙江横店影视基地格外冷清；现在，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 当局还对准
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演艺圈.html" target="_blank">
  演艺圈
 </a>
 下达〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_限酬令.html" target="_blank">
  限酬令
 </a>
 〞，包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艺人.html" target="_blank">
  艺人
 </a>
 赵薇、舒淇都成为目标。
 <br/>
 <br/>
 陆媒早前报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 广电总局对综艺节目下达〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_限酬令.html" target="_blank">
  限酬令
 </a>
 〞，赵薇、舒淇传出〝被打两折〞，真人秀谈定价格，本来是每一季每人领5千万人民币，竟被要求缴回4千万。
 <br/>
 <br/>
 娱乐圈资深媒体人 陈玲玲：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 事件之后，中共当局就决定说杀鸡儆猴，然后开始要求这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_艺人.html" target="_blank">
  艺人
 </a>
 片酬，包括舒淇、很多艺人，杨幂在里面，通通都是称为限酬名单里面的一员。〞
 <br/>
 <br/>
 限酬令规定，明星参加节目，每集片酬不能超过80万元人民币；如果担任常驻嘉宾，则每季片酬的上限是1,000万人民币。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演艺圈.html" target="_blank">
  演艺圈
 </a>
 资深媒体人 陈玲玲：〝中国大陆的艺人片酬实在太高了，有时候几乎到达整个戏的制作费的一半以上[CUT，变成说请得起大明星，就顾不了戏的品质]。〞
 <br/>
 <br/>
 学者曾建元说，一般艺人会重视长期市场利益，但大陆演艺界风险高，艺人能拿就拿，片酬就高。
 <br/>
 <br/>
 中华大学行政管理学系副教授 曾建元：〝他着重的是每次演出的短期利益，能够拿的就尽量拿。这行业有它的风险。〞
 <br/>
 <br/>
 范冰冰被举报〝阴阳合约〞消失几个月，新华社3日称范要补缴税8亿8千万人民币。曾建元说，很多大陆明星，想办法把资产用各种方式移出海外，还取得外国籍，根本原因，是共产党统治下，大陆法律对个人安全与财产保障不足，限酬令作法也很粗暴。
 <br/>
 <br/>
 中华大学行政管理学系副教授 曾建元：〝中国经济开始出现危机、下滑，另方面，国家(中共)对个人资讯掌握能力增强的情况下，个人自我(姿色与)利益保护的能力，事实上是非常脆弱的，(易成为)不肖官员、或国家(中共当局)剥削的对象。〞
 <br/>
 <br/>
 中国演艺圈近半年吹寒风，大批循范冰冰模式、在第三地设公司或个人工作室的明星，注销法人机构。
 <br/>
 <br/>
 新唐人亚太电视张东旭台湾台北报导
</div>
