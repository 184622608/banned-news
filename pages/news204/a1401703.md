### 中共乱局：新华社〝鼓励〞官场消极怠工
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扶贫.html" target="_blank">
  扶贫
 </a>
 一直是北京关注的重点，但中共扶贫中的欺瞒和贪腐泛滥，北京不断加大督查力度。而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新华社.html" target="_blank">
  新华社
 </a>
 日前文章却指责扶贫中的问责机制不当，导致〝不干
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不作为.html" target="_blank">
  不作为
 </a>
 ，少干慢作为，多干乱作为〞，不满之情跃然纸上。
 <br/>
 <br/>
 11月30日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新华社.html" target="_blank">
  新华社
 </a>
 旗下报纸《新华每日电讯》刊文&lt;问责〝脱靶〞又〝跑偏〞，基层
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扶贫.html" target="_blank">
  扶贫
 </a>
 怎么干&gt;。
 <br/>
 <br/>
 文中指责，督查组和上级干部在考察扶贫工作时，对基层干部〝一拍脑袋就问责，一不高兴就给处分〞，让问责泛滥成压在扶贫干部头上的一座〝大山〞。
 <br/>
 <br/>
 文章举例说，今年上半年，西部某省一个贫困县在扶贫检查中，发现一份扶贫手册中有两处标点符号错误。结果填写资料的驻村干部不仅被全县通报，还被扣了一年的绩效奖励。
 <br/>
 <br/>
 文章还提到，今年下半年，西南省份一位市领导到村里调研脱贫时，两次发现驻村干部低头看手机，顿时大发雷霆。最后，县里根据这位市领导的〝建议〞，处分了该驻村干部。
 <br/>
 <br/>
 文章指，扶贫问责〝跑偏〞〝脱靶〞的背后，首先是上级干部的能力不足和滥用权力，其次是扶贫评价考核机制本身就存在问题。
 <br/>
 <br/>
 文章举例，中部某省组织各县成立检查组，交叉检查扶贫成效，结果每个县都怕在省里排名垫底，所以想尽办法查找别县的问题。
 <br/>
 <br/>
 文章引述专家说法称，上层对扶贫干部只有〝严管〞缺乏〝厚爱〞，令基层感叹〝诚惶诚恐、又累又怕〞，有官员戏称〝不干
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不作为.html" target="_blank">
  不作为
 </a>
 ，少干慢作为，多干乱作为〞。
 <br/>
 <br/>
 中共的所谓扶贫工作，被指欺上瞒下的政绩工程，其中充斥大量贪腐问题。而据上文揭露，当局的所谓扶贫督查工作，又被中共官僚体制〝执行〞成了抓不到重点的形式主义和权力的任性发挥，导致中共官场〝官不聊生〞。
 <br/>
 <br/>
 除此之外，官媒这种替中共官员〝抱怨扶贫压力〞的行为，也被质疑并非只是帮助〝纠偏〞那么简单。
 <br/>
 <br/>
 北京的反腐打虎触动了全党上下的贪腐体制，不作为、乱作为等消极抵抗行为早已是中共官场常态。
 <br/>
 <br/>
 另外，中共宣传系统曾被江派掌控多年，一直是给习近平搅局的主力。虽然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中宣系统.html" target="_blank">
  中宣系统
 </a>
 几经整肃，但至今还不能和北京完全〝同调〞。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
