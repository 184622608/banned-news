### 高层分歧摆平？对习近平个人宣传出现新变化
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】随着美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 火的日渐升高，一度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降温.html" target="_blank">
  降温
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_捧习运动.html" target="_blank">
  捧习运动
 </a>
 〞再次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_恢复热度.html" target="_blank">
  恢复热度
 </a>
 。日前，有中共党媒高层说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强化领袖意识.html" target="_blank">
  强化领袖意识
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维护习核心.html" target="_blank">
  维护习核心
 </a>
 ，是特殊时期保持政治稳定的必要手段。分析认为，这或意味着外患内忧困境下，中共高层为保政权暂时摆平了党内分歧。
 <br/>
 <br/>
 10月12日，中共海外党媒多维引述人民日报有关负责人的话说，最近对习近平热度报导的模式和形式以及文风，和之前的〝造神运动〞出现了区别，浮夸风出现了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降温.html" target="_blank">
  降温
 </a>
 ，聚焦了更加务实的领域。
 <br/>
 <br/>
 报导回顾在北戴河会议后，8月21日至22日，习近平中共全国宣传思想工作会议上要求，要继续〝坚持党管宣传〞，〝坚决维护中央权威〞。
 <br/>
 <br/>
 被认为这是习近平在〝自我肯定〞的同时，给〝向左转〞的中共宣传机器撑腰打气，似乎已在提前预告捧〝习运动〞将强势回归。
 <br/>
 <br/>
 一个月后的9月26日，央视成立60周年，习近平发贺信表示，希望包括央视在内的中央广播电视总台及全国电视工作者，深入学习贯彻〝习思想〞，及壮大主流舆论等。
 <br/>
 <br/>
 当天，与习有关的议题随即重新〝复出〞，中共党校刊物《学习时报》重新刊出了在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 之初，一度传出被喊停的〝梁家河热〞。9月26日同一天，中国网也刊出梁家河报导，宣称习近平是〝留学人员的榜样〞。
 <br/>
 <br/>
 湖南卫视则在十一长假晚间的黄金时段，在问答节目〝新时代学习大会〞中，要求〝参赛者〞快速回答〝习思想〞的要义、习最喜欢的书等问题。
 <br/>
 <br/>
 长假结束后的10月8日起，央视又推出一档《平〝语〞近人——习近平总书记用典》新节目，从习引用的古代典籍和名句入手，来阐释〝习思想〞，节目还找来习近平文革在梁家河避难时的村民〝回忆当年〞。
 <br/>
 <br/>
 上述人民日报负责人分析说，现在的党媒推出的有关习近平的议题，是维护〝以习为核心的中央〞的权威的政治操作手段。在中美冲突对抗激化的背景下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强化领袖意识.html" target="_blank">
  强化领袖意识
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维护习核心.html" target="_blank">
  维护习核心
 </a>
 ，对保持政治稳定非常有必要，毕竟非常时期需要有非常策略。
 <br/>
 <br/>
 对比北戴河会议之前，也就是中美贸易战正式开打之初，中南海突然风云突变，中共官方对习近平的吹捧热突然变调。包括各地急撤习近平画像、梁家河研究项目也被叫停、华国锋道歉旧闻重炒、《人民日报》头版的标题连续几日不见习近平的名字等等。一度引发外界猜测〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_捧习运动.html" target="_blank">
  捧习运动
 </a>
 〞降温的背后，释放出的信号非同寻常。
 <br/>
 <br/>
 当时有说法认为，随着中美贸易战的升温，过度强硬的民族主义在中共内部权斗暗流，贸易战导致中南海高层出现严重裂痕，才导致了诡异局面。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：戴明）
