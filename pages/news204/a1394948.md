### 崔永元继续爆雷 举报两大政府部门
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】继举报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 偷税引爆娱乐圈强震后，大陆主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 日前又在微博爆雷，举报中共两大政府部门——中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_税务总局.html" target="_blank">
  税务总局
 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海公安局.html" target="_blank">
  上海公安局
 </a>
 。崔永元称，日前遭税务总局不法传唤，且态度粗暴；又披露上海警方对其进行〝抄家式〞报复调查。
 <br/>
 <br/>
 北京时间10月7日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 在其微博发布《一声长叹一声雷》的文章，文中称，因实名举报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 ，获10万人民币奖金，但没心情领，〝钱多少先放一边，实在看不了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_税务总局.html" target="_blank">
  税务总局
 </a>
 的一张张脸。〞
 <br/>
 <br/>
 紧接着，崔永元披露，自己去了税务总局三次，第三次是被突然叫去的，多了两张陌生面孔，没有自我介绍，没有出示证件，也没有看他提供的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 证据，而是拐弯抹角地询问证据（大轰炸合同）的来源。
 <br/>
 <br/>
 文章写道，〝一高一矮两位越问越急，终于原形毕露，这是俩蛮横的警察。〞
 <br/>
 <br/>
 〝在他们的世界里，有罪推定，人人都是罪犯。在他们的世界观里，他们就是法律，没必要对谁客气。〞崔永元表示。
 <br/>
 <br/>
 他在文中称，〝这两个身份不明的人不出示证据随意传唤审问公民是违法的，要求国税总局予以合理解释，国税总局称要请示领导，至今无下文，说话像放屁一样。〞
 <br/>
 <br/>
 让崔永元感到愤怒的是，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海公安局.html" target="_blank">
  上海公安局
 </a>
 经侦大队对所有他参与的公司彻底侦察，对他以前的助理不断询查，彻夜询查。
 <br/>
 <br/>
 他表示，原因在于〝大轰炸〞，参与这次大欺诈的既有演艺界大腕，也有上海经侦的警察。此前，崔永元曾举报范冰冰出演
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电影大轰炸.html" target="_blank">
  电影大轰炸
 </a>
 ，涉阴阳合同偷逃税，掀起娱乐圈风暴。
 <br/>
 <br/>
 崔永元此次又披露，这些上海警察〝曾经当着我的面，他们喝两万一瓶的酒，抽一千一条的烟，几十万的现金用个书包就提走。〞
 <br/>
 <br/>
 崔永元此文一经发出，上海警方10月10日即发布通报回应，称多次联系崔永元未果，希望他主动联系警方配合调查。但沪警〝联系不上〞的说法，立即遭到崔永元的反击和嘲笑。
 <br/>
 <br/>
 崔永元随后在微博发文表示，听说你们多次联系我未果，是写信联系的？你们都把我查的底朝天了，还联系不到我？
 <br/>
 <br/>
 网友也纷纷表示，的确，警察要想找一个人很容易，尤其是崔永元这样的大V，说联系不上有些勉强。
 <br/>
 <br/>
 不仅如此，崔永元还直指，“你们（上海警察）内部问题，可以先查原宁经侦副支队长，他儿子叫彭明达。”
 <br/>
 <br/>
 有分析认为，既然崔永元给出了被举报者姓名职务，上海警方必须要给出交代，如今，上海警方与崔永元之间的〝互动〞，已经让警方有点骑虎难下了。而大陆围观网友则对此事后续发展，拭目以待。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
</div>
