### 改革开放本身也需改革？习近平和邓朴方说法针锋相对
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】白宫对北京抛弃邓小平〝改革开放路线〞的批评，引发中共内部进一步分裂。日前港媒披露邓小平之子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓朴方.html" target="_blank">
  邓朴方
 </a>
 的讲话称，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平路线.html" target="_blank">
  邓小平路线
 </a>
 一百年不变〞，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 今年〝南巡〞时有关推进〝更高层次改革开放〞的讲话，则被中共喉舌重点解读。
 <br/>
 <br/>
 10月下旬，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 考察广东期间，网上传出中共残联名誉主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓朴方.html" target="_blank">
  邓朴方
 </a>
 在9月16日残联第七次代表大会上的发言稿，其中力挺习当局绝口不提的〝邓小平理论〞。
 <br/>
 <br/>
 10月30日，香港《南华早报》又跟进报导了该报获得的邓朴方完整发言稿。
 <br/>
 <br/>
 报导称，邓朴方在发言中重提邓小平的〝韬光养晦〞策略，宣称〝目前最重要的问题是适时着眼处理中国自己的事〞，并不点名地提醒北京当局〝头脑保持清醒，清楚所在的位置〞。
 <br/>
 <br/>
 邓朴方还表示，邓小平倡导的〝改革〞已经〝不可逆转〞，〝应该持续走这条路〞，〝一百年不变〞。
 <br/>
 <br/>
 就在同一天，中共喉舌《人民日报》发表评论员文章&lt;以改革开放的眼光看待改革开放&gt;，解读习近平10月〝南巡〞时的一段讲话，明确要对〝改革开放〞本身进行〝改革〞的态度。
 <br/>
 <br/>
 文章引述习近平在广东考察时的讲话称，要用〝辩证唯物主义和历史唯物主义〞的方法论，〝以改革开放的眼光看待改革开放〞，认识〝新形势下改革开放的时代性〞，〝在更高起点、更高层次、更高目标上推进改革开放〞。
 <br/>
 <br/>
 文章解读，〝改革开放〞就是要不断革除原有的体制〝弊端〞，适应〝新时代〞要求，应对〝新挑战新问题〞，要〝变革创新〞，不要〝僵化停滞〞。
 <br/>
 <br/>
 文章还特别提到，〝改革〞目前进入〝攻坚期〞，要向〝顽瘴痼疾〞开刀，突破〝利益固化藩篱〞等。
 <br/>
 <br/>
 习近平上台后，通过反腐扫荡江泽民派系，削平政治山头。为解决经济危局，习当局又用铁腕手段清扫中共权贵掌控的各大利益地盘。不过，经济改革举步维艰，权力和利益之争却让中共继续分化，就连红二代内部也开始分裂。
 <br/>
 <br/>
 习近平和邓朴方的这种〝公开论战〞，不只是代表中共体制内应对贸易战的策略之辩，背后还涉及中共残酷的内斗。中共历史上，〝路线之争〞往往伴随着你死我活的权力斗争。
 <br/>
 <br/>
 自中美贸易战爆发以来，中共内部批评习当局过早抛弃〝韬光养晦〞的声音开始抬头，由此传出各种版本的所谓〝政变〞流言。而日前美国白宫对北京抛弃〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平路线.html" target="_blank">
  邓小平路线
 </a>
 〞的批评，更加深了中共内部分裂。其中包括习、邓两家族的矛盾公开化。
 <br/>
 <br/>
 白宫经济顾问库德洛（Larry Kudlow）10月17日对美国福克斯新闻表示，北京基本上抛弃了邓小平路线，正在把中国变成一个更加封闭和不自由的经济体。
 <br/>
 <br/>
 10月4日，美国副总统彭斯（Mike Pence）在其重磅演讲中，也公开呼吁北京改变路线，回归中美关系开始时的〝改革开放精神〞。
 <br/>
 <br/>
 不过，邓小平的所谓〝改革开放精神〞，与美国当前要求的中美贸易公平相去甚远。例如，饱受美国谴责的强制技术转移，就是邓小平〝用中国市场换外国技术〞政策的产物。
 <br/>
 <br/>
 鉴于外国政府对中共密室政治的普遍不了解，有观察家认为，白宫对邓小平式〝改革开放〞的肯定，或许只是一种外交策略。
 <br/>
 <br/>
 港媒曾分析，彭斯演讲中对中国文化引经据典，显然不是一个西方文化背景的人所能做到的，其讲稿背后必有华人专家智囊，甚至是来自中共体制内人士的参与。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
