### 孟宏伟失踪原因曝光 港媒：已被纪检部门调查
------------------------

 【新唐人北京时间2018年10月06日讯】

  国际刑警组织
 </a>
 主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 在中国〝被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 〞后，至今下落不明。港媒日前引述消息人士的话透露，孟宏伟已被中共纪检部门带走，正在接受调查。
 <br/>
 <br/>
 香港《南华早报》周五（5日）引述消息人士的话称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 正在中国接受调查。消息称，孟宏伟上周（9月29日）乘机返回中国时，〝一落地〞即被中共纪检部门〝带走〞。但目前尚不知道孟宏伟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被调查.html" target="_blank">
  被调查
 </a>
 的原因，以及他被关在何处。

 声明还指出，国际刑警组织秘书长卓尔金・斯多克（Jurgen Stock）是负责刑警组织日常工作的秘书长，所以孟宏伟的失踪并不会影响到该组织的日常运作。
 <br/>
 <br/>
 除上述声明外，国际刑警组织没有进一步对孟宏伟的失踪作出评论。而至目前，中共公安部和外交部也没有对事件作出任何置评。
 <br/>
 <br/>
 <h4>
  失踪前被两度削权
 </h4>
 <br/>
 根据公开资料，现年64岁的孟宏伟目前担任中共公安部副部长。2016年11月，他当选国际刑警组织主席，任期四年。他也是国际刑警组织自成立以来首位来自中共的主席。
 <br/>
 <br/>
 孟的当选曾引发舆论对国际刑警组织公正性的质疑和担忧。一些组织担忧孟宏伟作为广泛侵害人权的中共政府高官，会令国际刑警组织成为在全球通缉和抓捕中国异见人士和政治难民的工具。
 <br/>
 <br/>
 今年4月，中共官媒曾报导，公安部副部长孟宏伟已不再担任公安部党委委员。这是孟宏伟再次被削权。
 <br/>
 <br/>
 去年12月，他被免去中共国家海洋局副局长、中共海警局局长职务。当时的分析认为，孟宏伟仕途可能不妙。
 <br/>
 <br/>
 孟宏伟自2004年开始长期任公安部副部长，与江派大员周永康有不少交集。周永康在2002年升任政治局委员，2003年3月兼任公安部部长、国务委员后，孟宏伟在第二年被提拔为公安部副部长、党委委员，成为周的副手之一。
 <br/>
 <br/>
 时政评论员周晓辉曾表示，如果孟宏伟作为周永康的〝余毒〞被拿下，应当不会令人意外。
 <br/>
 <br/>
