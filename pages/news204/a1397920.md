### 谷开来和王立军暧昧照曝光   惊现半截手臂
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】前重庆市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 案件庭审期间，自揭薄妻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷开来.html" target="_blank">
  谷开来
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 〝如胶似漆〞，让各界大跌眼镜。网上流传一张谷开来给王立军送水时，眼神暧昧的照片似乎证实了这一点。而暧昧照中，王立军肩膀上有一条半截手臂吓坏网友。
 <br/>
 <br/>
 2012年2月6日，时任重庆公安局长的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出逃.html" target="_blank">
  出逃
 </a>
 美国领馆寻求庇护，引发中共官场大地震，令全球目光聚焦中共政局。
 <br/>
 <br/>
 同年12月17日出版的《南都周刊》，以封面文章的形式刊发了〝起底王立军〞系列文章，全面回顾了前重庆市公安局长王立军发迹、治警、打黑乃至涉入薄
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷开来.html" target="_blank">
  谷开来
 </a>
 杀人案的过程，并通过对〝王立军模式〞的还原，揭示了这宗惊天大案的由来与深层根源。
 <br/>
 <br/>
 其中《王立军东窗事发》一文中，披露了王谷二人的交往细节。文中披露，薄谷开来、王立军一度关系非常密切。2010年1月12日 ，身着红风衣、黑裙、黑色长靴的薄谷开来携其母范承秀，牵着宠物狗，到了重庆市局。王立军作陪。
 <br/>
 <br/>
 当天，薄谷开来不仅参观了打黑展，还在王立军的办公室里把玩了微缩骷髅模型，亲手为王立军制作桌上装饰品。
 <br/>
 <br/>
 同年8月20日，谷开来到重庆街头探望执勤的王立军，并送水慰问。
 <br/>
 <br/>
 王立军落马后，谷开来给正在街头执勤的王立军送水时，眼神暧昧的照片，在网路上广泛流传。网友们发现，照片中，王立军肩膀上吊着一条半截手臂，吓坏网友。
 <br/>
 <br/>
 有人问 〝这是谁的手臂？是人是鬼？〞也有网友说：〝看了两个小时，最终还是没看出来是谁的？〞还有网友调侃，〝肯定是王立军杀了黑社会的人，吊在膀上吓薄嫂的。〞
 <br/>
 <br/>
 但有媒体人说，那不过是一个正在另一个机位拍摄的随行人员罢了。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
   薄熙来
  </a>
  自揭与谷王〝三角恋〞
 </h4>
 <br/>
 2013年8月，薄熙来案件在庭审期间，也曾自揭与王立军和薄谷开来的〝三角关系〞。
 <br/>
 <br/>
 他说，王立军逃跑的真正理由是王立军暗恋谷开来，情感纠结不能自拔，王立军曾向谷开来做过表白。薄熙来还说，谷开来和王立军如胶似漆，王表白自打八个耳光称，我现在正常了。
 <br/>
 <br/>
 薄熙来还称：谷开来对王立军言听计从。他曾撞见王立军向谷开来表白，谷开来还曾将王立军皮鞋拿到家中，他对两人的关系烦透了。
 <br/>
 <br/>
 薄熙来自曝家丑，自揭与王立军和谷开来之间的三角爱恨情仇，沦为大众笑柄。
 <br/>
 <br/>
 而薄熙来生活糜烂与众多女人有染的传闻也一直在网路流传甚广。
 <br/>
 <br/>
 据港媒爆料，当年薄熙来缺军中人脉，薄妻谷开来为其开路，以自身勾引权贵铺路，而薄的政治打手重庆前公安局长王立军，在帮薄谷夫妻遮掩之余不忘秘录制淫乱影像。
 <br/>
 <br/>
 爆料称，薄熙来当年的〝白手套〞大连实德总裁徐明被捕后供认，他负责安排输送了100个女人供薄熙来淫乱，其中有28人为公众人物。最爆炸性的是，徐明说，曾是薄左右手的王立军已经将薄熙来的淫乱行为秘密录了像。
 <br/>
 <br/>
 徐明还交代说，谷开来曾经为了薄熙来的政治前途，多次与一些政坛人士百般周旋，甚至不惜出卖肉体。薄熙来早年对谷开来充满敬意，很多问题上，谷开来实际上是当家人。
 <br/>
 <br/>
 但也有消息说，谷开来后来经常被薄熙来暴打和性虐待。
 <br/>
 <br/>
 谷开来的父亲谷景生曾任中共军队总政治部副主任，对军方人事脉络了如指掌，谷开来作为北大高材生，在父亲身边多少学了点东西。而部队子弟是另一个圈子，薄熙来也切不进去，但谷开来一去，大门为她而开。
 <br/>
 <br/>
 王立军对谷开来也曾经忠心耿耿，因为王立军妻子在北京工作、女儿在北京求学读书，都是谷一手安排的。有一次谷开来在重庆一家五星级酒店宴客，餐后突然腹泻，王立军立即下令将五星级酒店封了，停业检查，甚至亲自看着化验餐厅的食物。
 <br/>
 <br/>
 当时重庆人看在眼里，说王立军天不怕地不怕，就怕谷开来。
 <br/>
 <br/>
 <h4>
  海伍德卷入活摘罪恶遭灭口
 </h4>
 <br/>
 除王立军外，薄谷开来与被其毒杀的英国商人海伍德，以及法国人德维莱尔的关系〝非同一般〞。
 <br/>
 <br/>
 英媒披露，薄谷开来当初就是在床上向王立军建议，以涉毒案拘捕为名，来击毙海伍德，而当时王立军除顾虑风险太大拒绝后，还暗地里将两人的对话进行录音。
 <br/>
 <br/>
 报导说，2011年11月14日，海伍德被发现死在重庆南山丽景度假酒店，时任重庆公安局长的王立军组织调查，迅速定论海伍德之死是因〝饮酒过量〞，并将尸体未经检验火化。
 <br/>
 <br/>
 两个多月后，此案出现戏剧性转变。2012年2月6日，王立军出逃美国领馆，揭露海伍德是被其上司重庆市委书记薄熙来的妻子谷开来主导毒杀。
 <br/>
 <br/>
 被抓后在庭审中，王立军供认，因海伍德案与薄熙来翻脸，并被薄痛打一拳，因担心自己也被杀才出逃美领馆。
 <br/>
 <br/>
 大纪元、新唐人等中文媒体也曾报导，被杀死的海伍德，因卷入薄熙来夫妇在国际贩卖器官、尸体等事件遭灭口。
 <br/>
 <br/>
 据追查国际披露的调查证据显示，海伍德是谷开来盗卖法轮功学员器官、贩卖尸体的海外联系人。他们与哈根斯工厂合作，活摘器官后，先卖器官再卖尸体。
 <br/>
 <br/>
 这个项目薄家挣了很多钱，海伍德向谷开来索要自己那一份提成，而谷开来担心海伍德泄露了这个惊天黑幕，于是动了杀机。
 <br/>
 <br/>
 海伍德被杀害后，王立军被中纪委多次秘密约谈。薄熙来夫妇开始对王立军戒备，王恐惧像海伍德一样遭薄熙来夫妇灭口，投奔美国领事馆，导致三人先后身陷牢狱。
 <br/>
