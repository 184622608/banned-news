### 平度老兵被打 各地老兵声援进入第5天
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】中国各地老兵连日赶到山东平度声援被打的老兵，10月6号和7号遭到警方清场，8号仍有外地老兵绕过封锁线赶来声援，部分无法进入平度的老兵，决定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进京.html" target="_blank">
  进京
 </a>
 讨说法，目前北京也在拦截和抓捕老兵。另外，佳士工人声援团也发出声援信。
 <br/>
 <br/>
 平度市民透露，目前平度市仍是全面封锁状态，但10月8号持续有河南和辽宁等地老兵绕过封锁线，进入平度市，市民们送来食水，支持老兵的维权行动。
 <br/>
 <br/>
 平度市民李先生:〝全国各地老兵连日到平度，特警、武警、公安出动很多，封锁各个路口，老兵要躲开各个公路口，从村里穿过，进入平度市，广场不让他们在那儿呆，用高压水枪、辣椒水驱散、不离开就强制带走，所以说他们转移到马路边。〞
 <br/>
 <br/>
 部分外地声援老兵无法进入平度的，决定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进京.html" target="_blank">
  进京
 </a>
 讨说法。在北京的老兵透露，目前北京气氛紧张，也在拦截和抓捕老兵。
 <br/>
 <br/>
 贵州退伍老兵刘先生:〝平度他还在抓，抓了不少。地方的截访人员，信访窗口旁边，马路上抓，现在南站挑着抓，来没用啊，他们拦回去，基本上就上岗。〞
 <br/>
 <br/>
 佳士工人声援团8号发出声援信，信中说：佳士工人为了组建工会，维护工友的合法权益，被黑警察殴打，又被抓进监狱。我们声援佳士工人，也声援为工人阶级保家卫国的老兵同志！并要求严惩打人凶手，保障老兵权益！
 <br/>
 <br/>
 佳士工人声援团成员赵先生:〝声援老兵嘛，都是基于一个同样的道理，都是权利被剥夺了，都是处在一个被压迫的阶级，老兵几十年来都没有得到公平的对待，他们比较恼火，还动用了武力欺负老兵。〞
 <br/>
 <br/>
 10月4号，38名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平度老兵.html" target="_blank">
  平度老兵
 </a>
 自驾进京途中被当地公安暴力截访，6人住院，其中两人重伤，引发全国老兵声援，上千老兵赶到平度讨说法，6号和7号遭到暴力清场。另外，8号，福建有一百多名退伍老兵聚集省政府维权，高喊〝我们要吃饭、我们要生存、还我身分、还我编制、反对打压、反对腐败〞等口号。
 <br/>
 <br/>
 新唐人记者熊斌、周天采访报导
</div>
