### 陆股集体暴跌再现千股跌停 跌穿熔断低点恐慌蔓延
------------------------

<div class="wysiwyg">
 本周四，亚太股市遭遇重挫，A股开盘跌逾3%，中国上证综合指数收市暴跌5.22%，已失守2600点大关，全天逾千股票跌停，成交放大。同日，沪深300指数及创业板指数的跌幅也都超过3%。有陆媒引述业内人士分析称，此后在短期内或将出现技术性反弹，但之后还会继续震荡探底，这与中美关系恶化，中方在贸易战中的处境不利有关。
 <br/>
 <br/>
 综合陆媒消息，当地时间10月11日，中国股市上证、深证、沪深300三大指数集体暴跌。其中，上证综合指数低开低走，跌破2016年的融断机制低点2,638点关口后，市场上恐慌情绪开始蔓延，盘中一度下跌逾6%，最后报收2583.46点，跌幅为5.22%，创2014年11月以来的新低。深证指数则收报7524.09点，下跌6.09%。创业板指数收报1261.88点，下跌6.33%。
 <br/>
 <br/>
 港股也受拖累，恒生指数一度下跌3.7%，创2月份以来最大跌幅，股指跌至2017年7月份以来低点。
 <br/>
 <br/>
 对恒指拖累最大的个股包括：腾讯控股下跌5.6%，恒隆地下跌1.9%，长江基建集团下跌3.2%，恒安国际下跌2.1%，中国神华下跌2.7%，工商业在恒指四大行业分类指数中表现最差，下跌3.8%。

 11日当天，中国股市的行业板块全线飘绿，全日超过1000只股票跌停，其中电讯、通讯、电子信息、仪器仪表、券商、软件、交运设备、文化传媒、电子元件、航天航空、有色金属等板块，均暴跌逾8%，银行、煤炭、保险等板块的跌幅虽然较小，但也被重挫逾5%。
 <br/>
 <br/>
 同时，沪深两市11日的成交额，与上一日比较扩大逾五成。其中，沪市成交额扩大至1701亿元（人民币，下同），深市为1886亿元，合计成交额为3587亿元，业界认为这是市场恐慌导致资金出逃的反映。
 <br/>
 <br/>
 由于恐慌情绪的蔓延，使得亚太股市也遭受重创。
 <br/>
 <br/>
 有媒体引述分析师指出，11日大跌后，中国股市短期内可能会出现技术性反弹，但考虑到中美贸易战无缓和迹象，还有可能升级至货币战、金融战，加上中国经济下行的压力明显加大，因此短期技术性反弹后，股市还会再次回落探底。
 <br/>
 <br/>
