### 贸易战影响 北京警察禁市民美使馆听课
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】10月10号，几位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京市民.html" target="_blank">
  北京市民
 </a>
 前往美国驻华大使馆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_听课.html" target="_blank">
  听课
 </a>
 时，在大使馆门口遭到中共警察阻截，并带往麦子店派出所关押至深夜。
 <br/>
 <br/>
 美国驻华大使馆当晚举办的是一场面向公众开放的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_教育知识讲座.html" target="_blank">
  教育知识讲座
 </a>
 ，题目是〝孟禄与近代中国教育转型〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京市民.html" target="_blank">
  北京市民
 </a>
 陈洪旺10月12号对新唐人表示，他的孩子正在上高二，所以他特别关注教育问题，因此在美国驻华大使馆的公共平台上报名参加。但在使馆门口排队的时候，被中共警察拦截。
 <br/>
 <br/>
 北京市民陈洪旺：〝他说呀，现在在打这个中美贸易战，不许跟美国人接触。报出身份证，然后就等着，陆续的又来了很多的中国的安保人员，把我们就强制给䠬上车了。〞
 <br/>
 <br/>
 陈洪旺说，其他前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_听课.html" target="_blank">
  听课
 </a>
 的人士也跟他一样被受到阻拦，其中有3人跟他一样被带到了麦子店派出所，并被关押到半夜才由住地派出所警察接回。
 <br/>
 <br/>
 另一名被拦截的人士葛志慧也告诉新唐人，她是正式申请，获得邀请按时前往的，但到达以后，就被警察查身份证，然后扣下，不让去听。
 <br/>
 <br/>
 市民：〝为什么不让我们进，你说出理由来，我们就回去。〞
 <br/>
 警察：〝得拿出行动来〞
 <br/>
 市民：〝我这不是有邀请吗？这不我报名成功了吗？〞
 <br/>
 警察：〝就不行〞
 <br/>
 <br/>
 葛志慧说，在麦子店派出所关押期间，警察不许他们吃饭、喝水，还逼他们坐冷板凳。
 <br/>
 <br/>
 陈洪旺认为，警察随意抓人的行为是违法的，中国需要加强法制建设。
 <br/>
 <br/>
 新唐人记者常春、李沛灵采访报导
</div>
