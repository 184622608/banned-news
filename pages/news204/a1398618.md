### 【禁闻】11月8日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 湖南尘肺工人继续在深圳维权
 <br/>
 <br/>
 在深圳维权的三百多名湖南尘肺病矿工和家属，11月7号被深圳警方用催泪弹、辣椒水和警棍驱赶后，8号继续前往深圳市政府要求赔偿，官方安排4人与工人代表谈判，但没有结果。
 <br/>
 <br/>
 2. 贵州近两百下岗教师省信访局
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 <br/>
 <br/>
 贵州省遵义等五个市的近两百名下岗教师，11月5号集体到省信访局上访，要求解决下岗补偿和养老等问题，省教育厅派出一名接访官员与5名教师代表谈判，承诺解决问题，但教师们担心能否真正兑现。
 <br/>
 <br/>
 3. 西安鱼化寨村遭强拆 村民网上求助
 <br/>
 <br/>
 陕西西安鱼化寨村11月2号遭到上千警察、特勤和打手强拆后，最近一周数百〝协勤〞人员持续在村子路口把守、巡逻，并放话说：〝鱼化寨拆迁有死人名额〞，导致村民人心惶惶。8号，村民在网上发文要求中共总理李克强评理。
 <br/>
 <br/>
 4. 辽宁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马三家.html" target="_blank">
  马三家
 </a>
 劳教所受害者继续遭迫害
 <br/>
 <br/>
 辽宁铁岭访民杨秀梅因父亲工伤死亡上访多年，曾三次被劳教，先后在马三家女子劳教所被虐待7年，现在不仅上访诉求没有得到答覆，她自己还被政府用断水断电、半夜砸锁敲门、持枪威胁等手段继续迫害。
</div>
