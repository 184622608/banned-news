### 【禁闻】《纽时》：中共在全球难找贸易盟友
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 正在上海举办中国国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进口博览会.html" target="_blank">
  进口博览会
 </a>
 。《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约时报.html" target="_blank">
  纽约时报
 </a>
 》说，这是中共在全球寻找
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易盟友.html" target="_blank">
  贸易盟友
 </a>
 ，但要让世界相信中共宣传的真的，将是一个挑战。
 <br/>
 <br/>
 中国大陆举办首届以进口为目标的博览会，号称是一场最重要的外交贸易展示会。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 当局把这次博览会营造出所谓〝万商云集〞、〝丰硕成果〞的荣景。
 <br/>
 <br/>
 《北京之春》主编胡平认为，北京当局藉由所谓的〝国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进口博览会.html" target="_blank">
  进口博览会
 </a>
 〞，企图把自己塑造成促进自由贸易的领头羊。
 <br/>
 <br/>
 《北京之春》主编胡平：〝当然国际是会对这些说法持保留态度，因为类似的话过去也说过，但是并没有实际的改进，所以一般的态度就是走着瞧。〞
 <br/>
 <br/>
 外界认为，首届国际进口博览会上，习近平的一番话，透露了中国在当前内、外经济困境交逼下，经济转型的策略，希望用中国庞大的市场来促进进口，藉以推动国内内需市场的经济转型升级，对外则争取盟友、平衡贸易顺差、引进新产品与新技术等。
 <br/>
 <br/>
 而面对美国加征越来越多的关税可能会进一步放缓中国经济的情势，《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约时报.html" target="_blank">
  纽约时报
 </a>
 》中文网分析，中共领导人习近平正在努力让世界其他国家相信，中国能够成为全球贸易的积极力量。不过，报导还说，要让世界相信他说的是真的，将是一个挑战。
 <br/>
 <br/>
 胡平表示，中共精心准备这么久的首届进口博览会，世界经济大国却全部缺席，这本身就已经说明问题了。
 <br/>
 <br/>
 胡平：〝去的这些国家，它们加起来的经济体量，在全世界连百分之一的比例都不到。你本来是一个搞经济贸易的会，最重要的你就要吸引那些经济大国来参加，这方面它是非常失败，所以这个效果应该是非常清楚的。〞
 <br/>
 <br/>
 外界对中共当局所谓的加大市场开放，放松投资限制的措施，反应并不热烈，甚至认为是没有实际行动的老调重弹。
 <br/>
 <br/>
 包括欠中国很多钱的肯尼亚，都发出不平的声音。肯尼亚总统肯雅塔在大会发言说，〝肯尼亚与中国的贸易在过去10年增加了8倍，但是双方的贸易严重地倾向于对中国有利，因此必须要纠正贸易不平衡，确保更公平的贸易分配。〞
 <br/>
 <br/>
 旅美时事评论员蓝述：〝有些小国家确实国内的经济遇到了许多难以解决的问题，走投无路之下，把中共这种所谓的促销活动，当作救命稻草一样的。〞
 <br/>
 <br/>
 《纽约时报》的报导还说，就在寻求与美国达成贸易协议的同时，中共政府也在试图打动世界其他国家，一方面是为了寻求其他国家的政治支持，另一方面是为了确保中国可以有更多的地方出售自己的产品。不过，报导认为：中共要找到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易盟友.html" target="_blank">
  贸易盟友
 </a>
 并不容易。
 <br/>
 <br/>
 蓝述：〝因为它产能过剩，它产能过剩以后，就造成大规模的国内这些工人下岗，那怎么办？它当然是在全球寻找贸易伙伴，这不是一个新的策略，但是问题是中共这种发展模式，以及中共和西方国家的经济交往的模式，让人家只能对它敬而远之，人家没有办法跟你做生意。〞
 <br/>
 <br/>
 另一方面，大陆媒体高调晒出博览会出大单，包括大众汽车和德国大众签备忘录，到2019年进口上百亿的汽车部件和奥迪整车；阿里集团号称抛出今后5年采购2千亿美元的进口产品；京东集团号称也准备上千亿元的采购大单；德阳市政府与美国赛仕软件的签约、中国检验认证有限公司与比利时泽布鲁日港务局项目的签约等。
 <br/>
 <br/>
 据了解，这些大单已经洽谈很长时间，只不过是把签约仪式挪到进博会的会场，成就了进博会的所谓〝丰硕成果〞。
 <br/>
 <br/>
 采访/常春 编辑/黄亿美 后制/周天
