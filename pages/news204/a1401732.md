### 中共新疆再教育营：强迫被押者与海外配偶离婚
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 持续加强对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 的高压监控，被曝强制关押、拘留数百万维族人。近期，当局大规模逼迫遭关押者向境外配偶
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离婚.html" target="_blank">
  离婚
 </a>
 ，以切断海外人士的声援。
 <br/>
 <br/>
 据《自由亚洲》电台报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 持续加强对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 高压控制，其臭名昭着的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育营.html" target="_blank">
  再教育营
 </a>
 强制拘押300万民众。最近新疆当局再下狠招，逼迫被羁押者与身在海外的配偶、亲属
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离婚.html" target="_blank">
  离婚
 </a>
 或脱离关系，以阻挠海外团体的救援行动。
 <br/>
 <br/>
 报导称，越来越多的海外维吾尔族或哈萨克族人士向该报披露，新疆当局正加紧逼迫遭关押者与海外配偶离婚。受访人士表示，一旦离婚，海外配偶便失去了与遭关押者间的法律关系，因而无法起诉政府或提出相关诉求。这是中共对境内少数民族进行的切断后路的狠招。
 <br/>
 <br/>
 该报导引述伊犁作家海拉提的话说，新疆当局试图通过逼迫离婚，以切断他们与海外亲友的联系，阻止海外亲友为身陷囹圄的穆斯林发出求助。
 <br/>
 <br/>
 《自由亚洲》电台11月30日引述哈萨克妇人卧沃热阿丽汗表示，她的丈夫被拘留在新疆再教育营，最近受到当局逼迫，口头向她提出离婚。
 <br/>
 <br/>
 卧热阿丽汗表示，2017年9月上旬，其丈夫返回新疆后，护照被没收，人被送入所谓的政治学习营。
 <br/>
 <br/>
 阿丽汗说，她丈夫最近通过微信提出离婚：〝不知道他在哪里，现在他不回我们电话，他现在活着还是死了，都不知道。〞
 <br/>
 <br/>
 现居日本去年获释的哈萨克人卧弥尔别克告诉《自由亚洲》电台：〝我是在2017年3月26日被抓，被关进他们的集中营，民族大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_屠杀.html" target="_blank">
  屠杀
 </a>
 的集中营。2017年11月24日释放出来。〞他说〝现在集中营内恶劣的情况，・・・ ・・现在中共在邪恶的政策下，执行民族大屠杀〞。
 <br/>
 <br/>
 （记者李倩倩报导/责任编辑：祝馨睿）
</div>
