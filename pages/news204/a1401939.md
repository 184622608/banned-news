### 停火协议墨迹未干 中共官员扬言：核心议题不妥协
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】对于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 后的90天
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易谈判.html" target="_blank">
  中美贸易谈判
 </a>
 ，外界本就不抱多大希望。而美方将谈判代表换成强硬派人物，更令达成协议的可能性大大降低。北京方面，有外贸官员也直言，中方已〝无路可走〞，不可能再做出更多妥协。
 <br/>
 <br/>
 自由亚洲电台12月3日报导，中共对外贸易部门一位匿名官员宣称，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 上，中方已经对美国做出〝巨大让步〞，三个月后不大可能做出更大妥协。
 <br/>
 <br/>
 他声称，美方〝需要调整谈判内容〞，别指望中共做出〝影响社会制度〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性改革.html" target="_blank">
  结构性改革
 </a>
 。他说，中共在贸易领域已经〝无路可走〞，美方不能把北京逼入〝死胡同〞。
 <br/>
 <br/>
 海外评论人士普遍认为，如果北京按照美方要求进行经济体制改革，实现真正的公平贸易，必将危及中共政治体制的安全，中共已经退无可退。而美国总统川普和白宫高级官员一再强调，达成协议的前提必须是实现贸易公平。因此，美中贸易战将长期化。
 <br/>
 <br/>
 北京媒体人张女士对自由亚洲电台表示，相信在未来90天内，中美谈判代表将进行激烈交锋。对中方而言，扩大进口美国商品是最容易做到的，但在结构性改革方面，有关互联网封锁及国企补贴等问题上，还会有非常激烈的谈判。
 <br/>
 <br/>
 她说，如果取消国企补贴，中共的整个经济体就会完蛋。
 <br/>
 <br/>
 北京一民间网站创办人李新德也认为，开放互联网和取消国企补贴等都属于中共所谓的〝原则问题〞，预料中共不会妥协。
 <br/>
 <br/>
 12月1日晚宴上，中美首脑达成了暂缓贸易战升级的临时协议。根据白宫声明，北京承诺从美国购买大量农业、能源、工业和其它产品，管控此前大举输送美国的毒品，从新考虑高通收购案等；作为交换，美方同意将2019年1月1日提升中国产品关税税率的计划搁置90天。
 <br/>
 <br/>
 不过在这90天内，中方必须与美方开始就强制技术转让、知识产权保护、非关税壁垒、网络入侵和网络盗窃等问题展开谈判。如果届时不能达成协议，美方将重启关税升级计划。
 <br/>
 <br/>
 美媒消息，中共副总理刘鹤12月中旬将率团赴美谈判。而华盛顿已经知会北京，美方首席谈判代表将由财政部长梅努钦更换为贸易代表莱特希泽。
 <br/>
 <br/>
 莱特希泽主张对中共强硬，而梅努钦则倾向于与北京和解。5月时，梅努钦曾经和刘鹤达成暂时搁置贸易战的协议，遭到美国政府内部指责。数日后，白宫重启了贸易战。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
