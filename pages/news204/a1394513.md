### 韩媒：孟宏伟被查一事反常 恐涉嫌严重犯罪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 突然被查一事，引发国际舆论关注。韩媒称，北京当局在孟宏伟被查一事上行动反常，不惜损害其国际形象，透露出孟可能涉嫌非常严重的犯罪。
 <br/>
 <br/>
 当地时间10月4日傍晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 妻子在其失联一周后向法国警方报案，致使孟宏伟被北京当局调查一事演变成国际事件。
 <br/>
 <br/>
 10月7日，位于法国里昂的国际刑警组织在推特（Twitter）上发表声明，称收到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 孟宏伟的辞呈，〝立即生效〞。
 <br/>
 <br/>
 韩国《中央日报》10月8日报导，国际刑警组织在北京方面公布孟宏伟接受调查的消息后，对外公布了他辞职的消息，但并未提及其辞职的具体原因。而北京当局在孟宏伟被调查一事上的行动反常，透露出孟可能涉嫌非常严重的犯罪。
 <br/>
 <br/>
 韩国《文化日报》也持相同观点，认为孟宏伟的一个身份是国际刑警组织主席，北京当局对他采取这种超越法律之上的调查方式，必然会损害北京的形象。但北京当局仍不惜采用这种方法，有可能孟宏伟涉嫌的犯罪行为，非常严重。
 <br/>
 <br/>
 迄今为止，中共官方并没有给出孟宏伟接受调查的具体原因，但外界各种传言满天飞。有港媒称，孟宏伟是因为经济问题和在香港购置产业等而落马。
 <br/>
 <br/>
 但外界普遍认为，北京当局以涉嫌贪腐的名义将孟紧急拿下，这种理由难以服众，其中必然涉及更严重的政治类犯罪。
 <br/>
 <br/>
 有港媒指，孟宏伟案涉中南海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政变.html" target="_blank">
  政变
 </a>
 阴谋，事态紧急。还有说法认为，孟宏伟担任国际刑警主席后，在国际上有不利北京的举动，涉嫌替
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 反扑。
 <br/>
 <br/>
 今年64岁的孟宏伟，2016年11月4日就任国际刑警组织主席，他被指是时任中共公安部部长周永康擢升提拔的亲信之一。近年来，当局不断清洗周永康的〝余毒〞，周在政法系统的多名心腹被抓，而孟宏伟作为周的〝余毒〞被拿下，其实并不令人意外。
 <br/>
 <br/>
 不过，孟宏伟作为国际刑警组织主席，要拿下他并不容易。有海外中文媒体刊文称，稍有不慎，孟宏伟很可能效仿当年王立军投奔美国大使馆，在法国寻求政治庇护，让北京颜面尽失。
 <br/>
 <br/>
 文章称，对孟宏伟的调查，当局本拟秘密操作，但因孟妻报警而洩露，引发国际关注。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
</div>
