### 连云港猪瘟二次爆发 官方否认非洲猪瘟拒送检
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】中国各地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情大爆发，中共官方被质疑掩盖真相，弹压舆论。江苏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_连云港.html" target="_blank">
  连云港
 </a>
 继上次非洲疫情爆发后，日前再次出现猪只大规模死亡，但当地政府这次坚称只是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_季节猪瘟.html" target="_blank">
  季节猪瘟
 </a>
 〞，拒绝进行检疫和上报。
 <br/>
 <br/>
 据港媒报导，江苏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_连云港.html" target="_blank">
  连云港
 </a>
 海州区8月曾爆发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情，但当地官方宣布本月4日已〝处理完毕〞。
 <br/>
 <br/>
 但是，近日当地再次传出东海县青湖镇及石梁河镇多个猪场，出现数百只养猪死亡，又掀起一轮非洲猪瘟恐慌。不过，这次官方宣称只是〝季节性猪瘟〞，拒绝对死亡猪只进行检疫。
 <br/>
 <br/>
 一名在青湖镇的孙姓养殖户表示，猪场内的110只猪在半个多月前出现高烧、口鼻流血及身体发红发紫等症状，之后陆续死亡。孙某说，自己养猪20多年，还是首次出现如此严重的情况。
 <br/>
 <br/>
 另一名马姓养殖户亦指，7月底开始，一个月内有约200只猪死亡。该养殖户说，〝摄氏42度高烧，退烧针打完又成低温，有的猪腹部出血，有的浑身发红发紫，有的呕吐，然后死掉〞。
 <br/>
 <br/>
 同在青湖镇养猪的王姓养殖户表示，〝政府说不是非洲猪瘟，让我们自行处理〞。
 <br/>
 <br/>
 青湖镇一名副镇长表示，这只是寻常的季节性猪瘟，每年秋季都有很多猪只死亡，〝基本症状也是高热发烧〞，所以没有必要将死猪送到国家农业部进行检测。石梁河镇的兽医站站长也否认出现非洲猪瘟疫情。
 <br/>
 <br/>
 连云港市农业委员会一名王姓主任则表示，目前未收到东海县有疑似非洲猪瘟的病例上报，并说当地政府不会向上瞒报。
 <br/>
 <br/>
 不过有养殖户质疑官方说法，〝（过去）高热病，好治，打个退烧针就好，一般100头里就死3、5头，但这次死的数量太多〞，怀疑地方政府隐瞒上报疫情。
 <br/>
 <br/>
 中国非洲猪瘟疫情最早出现在辽宁省沈阳市，后由北向南部省市扩散。仅官方报导的疫情就有近30起，涉及辽宁、黑龙江、河南、江苏、安徽、浙江、内蒙古、吉林和天津9个省市。
 <br/>
 <br/>
 不过，外界普遍质疑中共官方有意掩盖疫情。多家外媒披露，多数中国市民至今仍不了解非洲猪瘟的危害，包括很多疫情爆发地区。
 <br/>
 <br/>
 此前有消息指，此轮非洲猪瘟爆发，原因是中共在贸易战中向廉价的美国猪肉加征关税，改从非洲猪瘟多发的俄罗斯进口高价猪肉。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：云涛）
