### 中共耗巨资操控APEC失败 外媒:拿走糖衣退回炮弹
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】今年刚刚结束的亚太经合组织（APEC）峰会罕见未能达成联合公报，主因是中共政府拒绝接受公报中提及杜绝不公平贸易的说法。有外媒引述评论人士表示，中共花重金欲操控这届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 却未能如愿，太平洋岛国大多拿了中共提供的资金援助却并未站在中共一边，可以说是拿走了〝糖衣〞退回了〝炮弹〞。
 <br/>
 <br/>
 今年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 11月18日在巴布亚新几内亚落幕，与会的各国首脑最终未能达成联合公报，这在APEC历史上是首次出现。《华尔街日报》报导称，原因是各国都同意在联合公报中写入〝反对保护主义，包括不公平贸易行为〞这句话，但中共政府认为此语针对中国（中共）而坚决反对。
 <br/>
 <br/>
 与此同时，还有多家外媒爆料称，联合公报宣言流产的背后还有另外一个事件发生。
 <br/>
 <br/>
 据称，在11月17日这天，中国代表团的4名成员曾要求进入东道国巴新外长帕托（Rimbink Pato）的办公室去讨论正在草拟的APEC宣言的措辞，遭到拒绝后还想强行闯入，最终被驰援的警察和保安阻止并〝护送离开〞。
 <br/>
 <br/>
 澳大利亚广播公司称，帕托办公室的一名发言人证实，中方曾有会议请求，但身为这次APEC会议主席的帕托想保持公正立场而拒绝了中方的要求，但这名发言人拒绝证实当时是否发生了冲突。
 <br/>
 <br/>
 帕托本人则在一份声明中表示，该〝事件〞对东道国来说根本不是问题，APEC峰会迄今为止取得了巨大的成功。他在声明中并否认了中国代表曾企图〝强闯〞其办公室的说法。
 <br/>
 <br/>
 针对上述事件，现居于美国的一名北京历史学者对《自由亚洲电台》表示，这次在巴布亚新几内亚举行的APEC峰会的会议花费大多都是由中共政府承担的，北京想要通过花钱来操控这次会议，但最终其目的却落了空。
 <br/>
 <br/>
 该学者说：〝中美对抗还没结束，影响到这样的一些国际会议，有些岛国就是两边得好处，最终要说站队还得看谁强大，‘糖衣’拿来，‘炮弹’退回了。〞
 <br/>
 <br/>
 香港新世纪出版社创办人鲍朴也指出，中国藉〝一带一路〞试图主导国际秩序，但其扩张模式建立在〝撒币〞和压榨国内百姓的价值上，具有很大的风险性。因此，不少小国家接收了中共撒出的钱后却〝自己干自己的事情〞。
 <br/>
 <br/>
 他表示，中共是靠极权政策摄取了财力和影响力，它还试图向全世界去推行这种〝中国模式〞，那是注定要失败的。
 <br/>
 <br/>
 报导称，近年中共加强了对斐济、萨摩亚、巴布亚新几内亚等太平洋国家的资金援助，但这次巴布亚新几内亚当局在APEC峰会上并未有配合中共，这也让外界对接下来的G20峰会〝充满想法〞。
 <br/>
 <br/>
 （记者黎明报导/责任编辑：明轩）
