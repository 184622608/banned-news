### 港媒：四中全会破例非寻常 上次相同情况是1958年
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】中共十九届四中全会预计10月底举行，今年上半年已召开过二中和三中全会。港媒指，中共一年开三次全会不寻常，此前只在1958年出现过，说明事态紧急。正是在1958年，中共开始了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大跃进.html" target="_blank">
  大跃进
 </a>
 〞。
 <br/>
 <br/>
 香港《明报》10月16日文章分析了北京本月的重要外交活动，认为四中全会将在28日至下月1日之间举行。
 <br/>
 <br/>
 文章说，中共每两次全国党代会之间的5年间，一般都会召开7次中央委员全体会议，简称〝×中全会〞。今年因为加开了一次讨论修宪的二中全会，原本决定换届人选的二中全会改成了三中全会，再加上即将召开的四中全会，一年举行三次全会，这在中共历史上，只有1958年出现过。这次再次破例，显示相关议题刻不容缓。
 <br/>
 <br/>
 目前，因为中共各种经济问题积重难返，再加上利益集团顽抗，北京的改革已成僵局，经济陷入困境。再加上贸易战高压，中共经济更是危机重重。
 <br/>
 <br/>
 在这种背景下，外界普遍认为，此次四中全会将聚焦经济政策。由于中共经济左转，北京再提〝自力更生〞、〝共克时艰〞，人们担忧中共会抢劫私产，牺牲民众利益来对抗贸易战。
 <br/>
 <br/>
 事实上，在上次破例召开三次全会的1958年，中共和苏联闹翻，在国际上陷入孤立，因此启动了〝赶英超美〞的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大跃进.html" target="_blank">
  大跃进
 </a>
 〞，从而制造了史无前例的三年大饥荒。
 <br/>
 <br/>
 那一年，中共连续开了八届四中、五中、六中全会，在5月一个月内就开了两次中央全会。5月16日，中共八大四中全会后，〝大跃进〞在全国展开，直接导致了1958-1962年的全国性大饥荒，期间饿死数千万人。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
