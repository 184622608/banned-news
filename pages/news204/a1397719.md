### 【禁闻】11月1日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 北京数百投资受害者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 遭警察驱赶
 <br/>
 <br/>
 北京华龙、华信、中润等网贷平台的数百投资受害者，10月31号，前往北京市政府上访，遭到大批警察驱赶，一些人被抓上大巴车运走。
 <br/>
 <br/>
 2. 湖南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 谢福林北京上访遭遣返
 <br/>
 <br/>
 湖南维权人士谢福林，10月30号，到北京上访反映自己权利遭受侵害情况，被北京警方抓捕并遣返。据报导，他这次前往北京，特别制作了写有〝习近平先生我的问题找谁解决〞的T恤。
 <br/>
 <br/>
 3. 律师要求会见朱承志 再遭警方推诿
 <br/>
 <br/>
 湖南维权人士朱承志被监视居住半年期满后，日前又遭刑事拘留，关押到苏州市第一看守所。11月1号，他的代理律师黄志强前往苏州要求会见，但遭到苏州警方的推诿
 <br/>
 <br/>
 4. 武汉访民胡正秋北京街头行走遭刑拘
 <br/>
 <br/>
 湖北武汉访民胡正秋，10月24号，在北京西单正常行走时，遭警察查验身份证，警方得知他是访民后，将其送往久敬庄关押，之后，武汉警方将他从北京带回，并以〝寻釁滋事罪〞刑拘。
</div>
