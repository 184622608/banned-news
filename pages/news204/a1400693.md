### G20前传美游说盟友防堵华为 重创中国股市
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】有美国媒体披露，美国政府官员正试图说服德国、意大利和日本等盟友，提防并谨慎对待其国内广泛使用的华为设备。该消息见诸报端后，令外界对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中首脑会谈.html" target="_blank">
  美中首脑会谈
 </a>
 可能达成协议的期望受到打击，中国股市随即应声下跌。
 <br/>
 <br/>
 近日有美国媒体爆料称，美国政府正敦促其盟国避免使用中国华为公司的电信设备。该消息令外界担忧即将到来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中首脑会谈.html" target="_blank">
  美中首脑会谈
 </a>
 恐怕难以达成任何协议。加上美国商务部正就一项更严格的技术出口管制规定征求公众意见，使得外界对中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 前景的预估更加悲观。
 <br/>
 <br/>
 上述相关消息被媒体广泛报导后，中国上证综合指数立刻下跌了2.5％，深证综合指数下跌3.3％。
 <br/>
 <br/>
 《美国之音》援引投资专家分析称，中国市场日前的大跌，说明投资者担心美中两国领导人可能不会达成协议，担心这次峰会后，两国之间互相加征关税的贸易战还会继续。
 <br/>

 <br/>
 此前，美国《华尔街日报》11月22日曾援引知情人士爆料称，美国在一些盟国拥有军事基地，例如德国、意大利和日本。而这些国家却在广泛使用中国的电信设备，这令美国政府十分担心。出于国家安全的考虑，美国官员已敦促这些广泛使用了中国华为公司提供的电信设备的盟友应慎重审视这个问题。
 <br/>
 <br/>
 据称，美国官员已向上述盟国的同级官员和电信高管进行了简要说明，告知他们美国方面所担心的网络风险。美国政府甚至还考虑了为那些避开使用中国设备的国家增加电信开发方面的财务援助等问题。
 <br/>
 <br/>
 知情人士称，美国国防部拥有自己的卫星和电信网络，用于特别敏感的通信，但很多军事设施里的大部分通信都通过商用网络，而美国在这些盟国中建有军事基地，因此这些国家广泛使用中国华为的电信设备，也会给美军基地带来资讯洩密的风险，这是美国政府十分担心的一个重要问题。
 <br/>
 <br/>
 据公开的资讯，早在今年10月中旬，两位美国参议院议员Mark Warner 与 Marco Rubio就已经致信加拿大总理 Justin Trudeau，希望他考虑禁用华为公司的 5G 设备。
 <br/>
 <br/>
 当时两位议员表示，有许多证据显示，中国没有一家大型企业可以完全躲过中共政府的掌控，而美国政府尤其对华为公司与中共政府的关系十分担忧。如果加拿大继续采用华为设备，不仅会给本国带来国安方面的疑虑，还可能使五眼联盟（Five Eyes）的友邦（美国、英国、澳洲与新西兰）被渗透。
 <br/>
 <br/>
 此外，澳洲近期已经下令禁止华为、中兴等中国通讯设备厂参与该国建置 5G 网络的项目，理由是澳洲政府认为这两家公司可能接受中共政府的指挥，从而给澳洲带来资讯安全方面的风险，并进而危害其国家安全。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：明轩）
