### 先捧后贬 党媒为何对基因编辑婴儿态度急转？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】中国副教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 宣布〝产出〞一对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 ，党媒第一时间宣称这一〝全球首例的创举〞。但在海内外舆论谴责声中，官方迅速集体转向，贺成为被牺牲的对象。网友指，中共企图冲击〝世界第一〞，此举旨在试探社会各界的底线。
 <br/>
 <br/>
 11月27日，中共喉舌新华社发表时评&lt;技术应用当在伦理红线前止步&gt;，称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 编辑婴儿基因〝违背学术伦理和学术规范〞，〝有关部门〞需要〝加强监管〞。此文疑似为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 〞事件定性。
 <br/>
 <br/>
 而一天前的26日上午，当南方科技大学副教授贺建奎在香港宣布〝先天免疫艾滋病的基因编辑婴儿〞在其团队手中诞生时，另一家党媒喉舌人民网在当天午前即推出报导&lt;世界首例免疫艾滋病的基因编辑婴儿在中国诞生&gt;，宣传〝中国（中共）医学界〞的这一〝历史性突破〞。
 <br/>
 <br/>
 报导甚至引述贺建奎的言论称，该项技术体现了〝5项伦理原则〞。
 <br/>
 <br/>
 人民网这篇报导，迅速在多家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共党媒.html" target="_blank">
  中共党媒
 </a>
 上获转载扩散。上海党媒澎湃新闻还刊登贺建奎的受访视频称，基因手术虽然会有争议，但〝有家庭需要该技术〞。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-28/p9145861a541428005.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-28/p9145861a541428005-ss.jpg"/>
   <br/>
   <font size="-1">
    中共喉舌起初吹捧这项〝历史性突破〞，还有陆媒报导使用了〝吓尿体〞标题。（网页截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 贺建奎在26日，即第二届国际人类基因组编辑峰会开幕的前一天，特别接受美国最大通讯社美联社的专访，公布了中国基因编辑婴儿的诞生，显然意在制造轰动效应。
 <br/>
 <br/>
 不过，出乎贺建奎意料的是，该消息带来的轰动效应，是全球媒体以及海内外同行的舆论围剿。就在当天下午，全球122名华人科学家联署发表声明，强烈谴责其〝疯狂〞行为。
 <br/>
 <br/>
 德国伦理委员会主席称，这是科学界的〝超级大危机〞，足见此事给全世界带来的震动。
 <br/>
 <br/>
 据多家媒体报导，西方医学界并非无力争夺这个〝技术第一〞，而是因为无法确认基因编辑是否会给人类带来不可逆的灾难，多数科学家不敢越过雷池一步，还有很多国家明确立法禁止对人类实施基因编辑。
 <br/>
 <br/>
 在强大的舆论压力面前，中共官媒的态度迅速逆转，纷纷和贺建奎〝撇清关系〞，贺的团队很快遭遇官方的〝抛弃〞。
 <br/>
 <br/>
 根据中国临床试验注册中心的一份《深圳和美妇儿科医院医学伦理委员会审查申请书》，该医院通过了对贺建奎基因编辑项目的伦理审查。陆媒揭露，深圳和美妇儿科医院是一家〝莆田系医院〞。
 <br/>
 <br/>
 而针对这份《医学伦理审查申请书》，中共央视26日却给出了不同说法。央视报导称，批准这份申请书的深圳和美妇儿科医院医学伦理委员会，没有按照规定在官方备案，当局将展开〝调查〞。
 <br/>
 <br/>
 据陆媒报导，27日晚间，贺建奎在南方科技大学的办公室已经被查封。
 <br/>
 <br/>
 中共的技术竞争向来以不遵守规则、无视道德伦理著称。因为中共只有一个形式上的〝医学伦理审查〞，并不立法约束，因此许多在海外被明令禁止的医学手术得以在中国实施。
 <br/>
 <br/>
 有网友分析，中共官方在基因编辑婴儿事件中表演的这场闹剧，可能是有意试探海内外和中国社会各界的反应，为开展和垄断基因编辑这一前沿技术铺路。雄心勃勃准备冲刺诺贝尔医学奖的贺建奎成了牺牲品。
 <br/>
 <br/>
 根据上述《医学伦理审查申请书》，开展此项基因编辑项目的目标是：建立基因手术治疗行业的〝质量控制标准〞，占领整个基因编辑相关治疗技术门槛的〝制高点〞，在国际竞争中〝脱颖而出〞。
 <br/>
 <br/>
 这与中共一贯争夺〝科技第一〞的目标相符。北京近年来推出的所谓〝中国制造2025〞和〝中国标准2035〞计划，被指企图在全世界夺取尖端科技领域的主导权，并由中共来制定行业标准。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
