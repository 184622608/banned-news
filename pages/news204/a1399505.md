### 11月15日全球看中国
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】下面是全球看中国
 <br/>
 <br/>
 1. 美国之音
 <br/>
 尽管重庆、福建等地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高考政审.html" target="_blank">
  高考政审
 </a>
 消息引发巨大争议，但浙江省教育考试院15号公布的《2019年浙江省普通高校招生考试报名办法》，仍规定公安、警官、北京电子科技学院等高校都需〝政审〞。
 <br/>
 <br/>
 2. 自由亚洲电台
 <br/>
 中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民营企业.html" target="_blank">
  民营企业
 </a>
 盈利下降，甚至亏损，欠薪问题时有发生。据中共农业农村部的数据，目前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_返乡农民工.html" target="_blank">
  返乡农民工
 </a>
 高达740万人，外界质疑这实际上是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业潮.html" target="_blank">
  失业潮
 </a>
 。
 <br/>
 <br/>
 3. 德国之声
 <br/>
 联合国八月表示，上百万维吾尔人被关押在新疆所谓〝职业培训中心〞。十五个国家在中国的大使因此集体致信新疆党委书记陈全国，要求与他当面讨论当地人权状况。
 <br/>
 <br/>
 4. 大纪元时报
 <br/>
 美中经济与安全审查委员会周三发表2018年度报告，其中提到，建议司法部启用〝阴谋对抗〞相关法律起诉恐吓美国居民的中共相关组织。
