### 刘强东何时被起诉？ 美国检方最新回应引猜测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】京东CEO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 性侵案曝光迄今已近3个月，美国检方日前就该案的起诉问题作出了新的回应。检方称，虽然刘强东是一位富商，但这与他们是否提起诉讼无关。美国活动人士霍诺德则认为，因为刘强东的富商身份，他被另眼相待。〝在我看来，刘强东并未受到大多数刑事被告遭到的同样待遇。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 被美国警方指控涉嫌〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸一级重罪.html" target="_blank">
  强奸一级重罪
 </a>
 〞，于8月31日晚间在美国明尼苏达州被逮捕。9月1日下午，刘强东被保释，9月3日回国。
 <br/>
 <br/>
 刘强东是首位大陆富商在海外被以强奸罪名抓捕，此事件掀起国际舆论轩然大波。如今，刘强东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵丑闻.html" target="_blank">
  性侵丑闻
 </a>
 曝光已近3个月，外界对他何时被起诉十分关注。
 <br/>
 <br/>
 目前，警方已经结束对该案的调查，并将案件转交给亨内平县检察官办公室。该办公室此前在一份电子邮件声明中称，对于何时对刘强东做出起诉决定，目前还没有具体的时间表。
 <br/>
 <br/>
 〝审查一起案件需要花费一定的时间。有时需要的时间很长，有时很短。刘强东是一位富商，但这与我们审查该案件无关。〞声明中称。
 <br/>
 <br/>
 不过，美国明尼阿波利斯市活动人士(曾遭遇性侵)霍诺德(Abby Honold)认为，因为刘强东的社会地位，当局对其另眼相待。
 <br/>
 <br/>
 刘强东被指是一名〝红顶商人〞，除了商人的身份外，他还有一个中共全国政协委员的头衔。他的性侵丑闻对北京当局也造成恶劣影响。
 <br/>
 <br/>
 霍诺德说：〝在我看来，刘强东并未受到大多数刑事被告遭到的同样待遇。〞的确，刘强东被关进监狱后，在短时间内即迅速获得保释，这非常罕见。
 <br/>
 <br/>
 据《星岛日报》报导，中共外交部在第一时间就介入该事件，且约谈受害女学生、施压女学生父母，企图逼迫该女学生返回中国。
 <br/>
 <br/>
 时事评论员文昭曾在自媒体节目中表示，中共外交部发言人华春莹在例行记者会上曾公开说：已请中共驻芝加总领馆密切关注相关消息，并对消息进行核实，等于承认了政府已经介入。
 <br/>
 <br/>
 领馆的所谓核实消息是很容易的，只要确定了这个女生，马上就能找到她国内的家人，当然以党和政府无所不在的社会控制能力，就能够影响案件的进程，文昭表示。
 <br/>
 <br/>
 与此案无关的律师哈尔伯格（Marsh Halberg）近日对《德国之声》表示，刘强东至今仍未被起诉可能有两种情况，一种是检察官可能在等待化验结果。当然，检察官也可能在与辩护律师对话。
 <br/>
 <br/>
 据报导，刘强东案涉事女子向警方提供了事发时的床单作为证据。
 <br/>
 <br/>
 另外哈尔伯格称，在个别情况下，检察官也可能回到警察局，要求他们重新采访证人或收集更多证据。
 <br/>
 <br/>
 近日英媒披露，在刘强东实施性侵前，他的女助理曾试图坐在他与受害女生之间。但刘强东警告她：别妨碍我。检方正在对更多证据进行调查。
 <br/>
 <br/>
 哈尔伯格还表示，如果刘强东不被起诉，人们会怀疑他是否得到优惠待遇。但另一方面，如果美国检方提出指控，刘强东必定会动用大量资源进行辩护。为此，检方必须要有充足的理由才能提起诉讼。
 <br/>
 <br/>
 刘强东并非首次爆出涉入性侵丑闻。2015年12月26日，刘强东在悉尼豪华公寓为地产商人徐龙威举办奢华的生日晚宴。宴会后，徐龙威将出席宴会的一名澳洲女模，带至悉尼香格里拉酒店并实施了强奸。
 <br/>
 <br/>
 该女模事后报警，将徐龙威告上法庭。当时，作为涉事人员的刘强东，曾要求澳洲检方隐匿其个人信息，但遭到拒绝，媒体对此事件进行了曝光。
 <br/>
 <br/>
 今年10月12日，徐龙威被澳大利亚新南威尔士州地方法院判处4年有期徒刑，2年半不得保释。
 <br/>
 <br/>
 刘强东此次又爆出性侵丑闻，网易创始人丁磊曾在第一时间对事件进行点评，他表示，刘强东是性侵惯犯，曾经在一次多人出席的宴会上，将一名女子灌醉，并当场在宴会包间的厕所内对女子施暴。
 <br/>
 <br/>
 自性侵丑闻曝光后，刘强东身价一落千丈，各大重要活动及会议均不见其身影，疑似被当局打入冷宫。近日，坊间又传出消息，称因性侵丑闻导致京东股票大跌，刘强东被踢出京东管理层。目前，消息尚未得到证实。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：祝馨睿)
