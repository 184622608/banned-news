### 【禁闻】千人计划专家堕亡 公司涉盗美技术
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】
 <b>
  山东庆云36位
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
   法轮功学员
  </a>
  被非法抓捕
 </b>
 <br/>
 <br/>
 中共今年藉口〝打黑除恶〞继续迫害法轮功，在大陆各地大规模绑架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 。据〝法轮大法明慧网〞报导，11月18号，山东德州市公安局与庆云县公安局的警察，以〝打黑〞之名，一天之内，在庆云县非法抓捕了36位法轮功学员。
 <br/>
 <br/>
 据报导，这些学员多数是在家里被抓的，其中最大年龄的80多岁。据说这些学员目前大部分被非法关押在山东德州看守所和拘留所，不允许家属会见，详细情况不得而知。
 <br/>
 <br/>
 〝明慧网〞说，法轮功学员都是按照〝真善忍〞原则做好人的普通百姓，中共以〝打黑〞名义抓捕他们，完全颠倒了黑白，而这个做法恰恰证明中共自己才是黑恶集团。
 <br/>
 <br/>
 <b>
  华为首席财务官被捕 重燃
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
   贸易战
  </a>
  忧虑
 </b>
 <br/>
 <br/>
 中国电信巨头华为的全球首席财务官(CFO)孟晚舟，日前在加拿大被捕，目前正面临引渡到美国受审的局面。西方舆论认为，〝川习会〞后稍有缓和的美中贸易关系可能面临再度升温。
 <br/>
 <br/>
 〝美国之音〞报导说，这位华为高管的被捕，被认为是川普政府对违规的中国最大电讯器材商惩罚的开端。
 <br/>
 <br/>
 〝路透社〞也报导说，孟晚舟被捕，引起了美中贸易战重新点燃的忧虑。受事件的影响，亚太地区股汇市12月6号再受重创。而由于投资者避险情绪迅速回升，中国股市沪综指当天也加大了跌幅。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
   千人计划
  </a>
  专家堕亡 公司涉盗美技术
 </b>
 <br/>
 <br/>
 美国华裔物理学家、斯坦福大学教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张首晟.html" target="_blank">
  张首晟
 </a>
 ，12月1号去世，终年55岁。据称他是罹患抑郁症堕楼身亡。他创立的风险投资公司〝丹华资本〞，上个月刚被美国政府指控帮助中共获得美国的尖端技术和相关知识产权。
 <br/>
 <br/>
 据报导，张首晟在理论物理学领域成绩斐然。2009年入选中共引进海外高科技人才的〝千人计划〞，被清华大学特聘为教授，开始在中美两边发展。
 <br/>
 <br/>
 有消息说，在他的公司被指控后，他本人作为中共〝千人计划〞的华裔科学家，可能也已经受到美国联邦调查局（FBI）的关注甚至调查。
 <br/>
 <br/>
