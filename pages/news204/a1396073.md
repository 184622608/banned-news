### 【禁闻】忧拉美依赖 美国会报告揭中共四大目标
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】10月17号，美国国会下属的〝美中经济与安全审查委员会〞发布报告，披露
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 近年来，如何积极强化在拉丁美洲与加勒比海地区的多方面参与，并揭露其主要目标。
 <br/>
 <br/>
 这份长达65页的〝中国与拉丁美洲暨加勒比海地区往来〞报告，阐述过去十年来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 深化在该地区的经济、外交与军事参与，成为那些国家的最大债权国，以及仅次于美国的第二大贸易伙伴。
 <br/>
 <br/>
 报告列举中共的四大主要目标：一，确保取得该地区丰富的天然资源和消费市场；二，获得该地区的外交政策支持；三，塑造拉美地区对中共的印象和看法；四，增加中共在该地区的地缘政治影响力。
 <br/>
 <br/>
 时事评论员田园博士举例，由于中共支持世界第一大储油国，也就是南美的委内瑞拉执政者，大搞共产主义经济，造成委内瑞拉面临经济的即将崩溃。
 <br/>
 <br/>
 时事评论员田园博士：〝委内瑞拉只是南美地区受到中共投资，最后债台高筑，经济要崩溃，国民基本上是民不聊生，大举逃到哥伦比亚，它只是一个典型例子，其它例子还非常非常多。〞
 <br/>
 <br/>
 在基础建设方面，报告指出，〝华为〞和〝中兴〞公司负责或参与建造了拉美24个国家的电讯网络系统。这让美国政府担忧，将带来潜在安全风险。
 <br/>
 <br/>
 此外，报告提到，2016年，中国占拉美地区出口的9%，进口比例也达到18%。
 <br/>
 <br/>
 2005年以来，中共国家政策性银行对拉美国家贷款总额高达1,504亿美元，超过世界银行、美洲开发银行（Inter-American Development Bank）与拉丁美洲开发银行（CAF Development Bank of LatinAmerica）的贷款总和。
 <br/>
 <br/>
 这让拉美国家对中国的经济依赖加深，也降低美国在该区域的经济影响力。
 <br/>
 <br/>
 田园博士指出，中共大肆撒钱，是它在国际上的长期策略。由于受共产主义思潮影响强烈，现在拉美很多国家强烈反美，比如委内瑞拉、古巴、秘鲁、阿根廷，给中共造成可趁之机。
 <br/>
 <br/>
 田园：〝把所谓的拉美和非洲的小兄弟，用金钱、外交、军事的手段，把它拢在周围。目的是在联合国里边取得和美国抗衡的能力。〞
 <br/>
 <br/>
 根据报告，2000年至2018年间，中国企业在拉美地区参与多达91个大型基础建设项目，目前已有42个完工。中共当局还加强双边关系，目前有9个拉美国家支持中共的〝一带一路〞倡议。
 <br/>
 <br/>
 田园博士：〝这些国家要想兴建这些基础设施，必须向中共大肆举债。就像在东南亚，中共一带一路造成很多南亚国家债台高筑，基本上就无法偿债，巴基斯坦最近已经向国际货币基金组织提出援救的请求。所以中共其实是在加勒比海和南美洲也在重复这样的战略。意图非常明确，我给你投资，我让你债台高筑，最后你走投无路就得来找我中共。〞
 <br/>
 <br/>
 对于台湾问题，报告特别提到，中共休兵8年后，再次重启外交孤立策略。台湾17个邦交国中，拉美国家占了9个。但一年之内，巴拿马、多明尼加、萨尔瓦多相继与台湾断交，转而与北京建交。
 <br/>
 <br/>
 中共除了利用政治、媒体与教育交流，增加与民间往来，还扩大军事交流、武器销售与非传统军事行动，获得该地区的战略地位。
 <br/>
 <br/>
 报告提醒美国政府必须继续强化与该地区的合作。
 <br/>
 <br/>
 中国宪政学者陈永苗：〝中国（中共）不仅仅渗透到美国的本土，而且渗透到了它的大后方。原来苏联跟美国之间很著名的一个导弹危机就是在古巴产生的，这个有类似于古巴导弹危机的效果吧！〞
 <br/>
 <br/>
 田园：〝古巴的导弹危机，就是因为古巴愿意安装苏联的核导弹。如今中国（中共）采取了更加隐蔽，试图在软实力方面在这个地区取得突破，几乎可以说是重演当年的古巴导弹危机，只不过这一次的危机不是由导弹引发的，而是由中共的金援和大棒引发的。〞
 <br/>
 <br/>
 田园博士认为，美国国会选择此时发布报告，是因为中共在拉美地区的影响力已经大到让人无法忽视。如果让中共在该地区取代美国的地位，后果不堪设想。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/陈建铭
