### 上海进博会 习近平再度承诺开放市场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 的阴影下，首届中国国际进口博览会（简称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 ）5日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海.html" target="_blank">
  上海
 </a>
 开幕，中国国家主席习近平出席仪式并发表讲话，习再度承诺会向世界开放市场，与世界〝共享〞经济发展的成果。
 <br/>
 <br/>
 11月5日，在首届中国进口博览会(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 )开幕式上，习近平做了半个小时的主旨演讲，习称，进博会是中共推进新一轮对外开放的重大决策，是中共主动向世界开放市场的重大举措。
 <br/>
 <br/>
 习还作出了一系列承诺，包括降低关税、给予外国企业国民待遇，保护他们的合法权益，保护外国企业的知识产权，严惩侵犯知识产权的行为。
 <br/>
 <br/>
 美国之音报导说，贸易行为在国际上广受批评的中共，欲藉此次博览会打造中共欢迎外商进入市场、在全球发挥积极力量的正面形象。
 <br/>
 <br/>
 然而长期以来，中共一直在作出这些承诺，但外国企业在中共治下的市场准入、知识产权、法律法规等诸多方面依然面临艰难挑战。
 <br/>
 <br/>
 很多外国公司认为，中共的监管机构仍然试图把他们挤出有前景的产业，外国公司仍面临被强制转移技术的压力。
 <br/>
 <br/>
 美欧外交官还怀疑中共是否有诚意履行降低贸易壁垒的承诺，同时对中共在美中贸易冲突升级之际举办进博会表示困惑。
 <br/>
 <br/>
 尽管北京当局释出善意，但美、欧政府却冷静对待，因为他们除了要中共开放市场，还要求中共改变体制，进行深层次的结构性改革，取消对国企的补贴，因在政策倾斜下，国企集中力量办大事便较容易，令外企难以竞争。因此美欧政府都希望中共在商业环境多做具体而合乎西方标准的措施，用行动来证明深化改革开放。
 <br/>
 <br/>
 美国驻华大使馆一名发言人说：〝我们鼓励中国（共）为美国的商品和服务提供公平的竞争环境，中国（共）应进行必要的改革，以终止其不公平的贸易行为。〞
 <br/>
 <br/>
 一位欧洲外交官告诉《金融时报》，中共可以借进博会发表有关实质开放和改革的声明，而不是看起来像是在回应美国的压力，多国家一直在推动北京履行开放承诺。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海.html" target="_blank">
  上海
 </a>
 的美国商会主席肯尼斯・杰瑞特在一封电邮中说：除非这场盛大的活动，在中共的贸易做法中，能相应做到有实质意义和可衡量的改变，否则对美国及其他外国的公司来说得不到什么成果。
 <br/>
 <br/>
 根据举办方的统计，来自130个国家的3000多家公司参加了此次博览会，但美国、加拿大、英国、法国、德国、意大利及日本的政要，无一出席，韩国、澳大利亚等中共主要贸易国家的政要也拒绝出席。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉)
</div>
