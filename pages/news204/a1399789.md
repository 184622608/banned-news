### 川普说中方清单漏掉4-5项 法媒：或是这几条
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】中共日前向美国提交一份关于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易纠纷.html" target="_blank">
  贸易纠纷
 </a>
 的让步清单，向美国求和。但川普（特朗普）总统表示，这份清单他还不能接受，因为漏掉了4-5项重要内容。法媒推测，中共在清单中漏掉的内容可能与〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞计划、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海.html" target="_blank">
  南海
 </a>
 以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 有关。
 <br/>
 <br/>
 川普11月16日在白宫向媒体表示，中共提出的这个清单挺全面，涵盖142项内容，但仍有4-5个重要内容没有被放进去，还没有达到双方达成协议的条件，因此他还不能接受这份清单。
 <br/>
 <br/>
 川普表示，〝我认为可能也要将它们加上〞。他还强调称，未来美国与中共要达成协议，必须是公平互惠的。
 <br/>
 <br/>
 据《路透社》引述消息说，中共提出的让步清单，主要分成3大类：中共愿意协商的问题、正在解决的问题、以及不愿协商的内容。川普所指的漏掉的4-5个重要内容，可能是属于中共不愿协商的内容。
 <br/>
 <br/>
 《法广》报导称，之前曾有中国学者透露，美方要求北京按照市场经济标准，停止政府补贴企业开发高技术的2025计划，或2035计划，以及停止盗窃知识产权问题。。。，应该是触到了北京的底线。
 <br/>
 <br/>
 有分析指，北京目前的经济政策，包括〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞计划等，已经和最高当局确立的政治路线捆绑在一起，如果做出实质改变，最高层将面临重大政治风险。
 <br/>
 <br/>
 另外，白宫经济顾问库德罗日前表示，中美双方的谈判不限于贸易。美国副总统彭斯日前也强调，将在11月底举行的G20川习会，除了讨论中美两国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易纠纷.html" target="_blank">
  贸易纠纷
 </a>
 问题外，中方也必须在保护知识产权、禁止强迫技术转让、取消限制市场准入、遵守国际规则、及保障国际水域航行的自由等问题上让步。
 <br/>
 <br/>
 据此猜测，中方清单中遗漏的问题，或许跟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海.html" target="_blank">
  南海
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 等议题也有关联。
 <br/>
 <br/>
 目前，中美双方正在为即将举行的G20川习会做准备。中共在这个关键时刻向美国提交这份清单，被认为是中共急于摆脱困局，向美国妥协、求和。
 <br/>
 <br/>
 在此之前，中方一直拒绝在正式谈判前，首先向美方提交书面的让步清单，担心因此失去谈判筹码。
 <br/>
 <br/>
 不过，中美双方后续谈判的过程显然不会那么顺利，也不是短期内能够完成的。许多分析人士并不看好北京的〝诚意〞，而且白宫鹰派日前也公开表达了对中共的不信任。
 <br/>
 <br/>
 白宫贸易高级顾问纳瓦罗日前表示，中共一贯言而无信，所以即使美中达成协议，也无法相信中共能够真正执行。
 <br/>
 <br/>
 美国商务部长罗斯16日对《彭博社》表示，美方将按原计划，在明年1月把税率从10%提升至25%，因为中美在1月前不可能达成一个全面的正式协议。
 <br/>
 <br/>
 他表示，目前所做的只是为川习会做准备。如果进展顺利，这次会晤最多能就进一步谈判，达成一个框架性协议。
 <br/>
 <br/>
 彭斯于13日在新加坡接受《华盛顿邮报》专访时警告中共，必须全面改变其行为作法，才能避免与美国发生新冷战。
 <br/>
 <br/>
 他还警告，如果中方的提议未能令美方满意，美国已经准备好在经济、外交及政治等多个层面上对中共施加更大压力。
 <br/>
 <br/>
 川普在10月底曾表示，美国有可能会与中共达成一项重大交易，但倘若在阿根廷谈不拢，美国会在12月初再对中国2670亿美元的输美商品征收关税。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：祝馨睿）
