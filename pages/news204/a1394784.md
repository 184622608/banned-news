### 孟宏伟突然被抓  国际刑警组织主席换人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】目前，国际刑警组织已换〝掌门〞，据披露，来自韩国的国际刑警组织高级副主席金正阳，将接替
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 担任国际刑警代理主席一职。此前，孟宏伟突然被抓，而且辞去了国际刑警组织主席一职。
 <br/>
 <br/>
 10月10日，陆媒报导，中纪委国家监委通报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，接受国家监委监察调查消息后，国际刑警组织将换〝掌门〞。
 <br/>
 <br/>
 国际刑警组织5日就孟宏伟〝失踪〞一事作出回应，称负责国际刑警组织机构日常工作的是秘书长而非主席。
 <br/>
 <br/>
 中纪委国家监委官网10月7日通报，孟宏伟涉嫌违法被查。同日，国际刑警组织发布公告，已收到孟宏伟的辞呈，来自韩国的国际刑警组织高级副主席金正阳，将接替孟宏伟担任该组织代理主席一职。
 <br/>
 <br/>
 国际刑警组织成立于1914年，总部设在法国里昂，现有192个成员国，是唯一的全球性警务合作组织，也是仅次于联合国的第二大国际组织。
 <br/>
 <br/>
 国际刑警组织主席由全体大会选举产生，任期4年，4年选举一次，不得连选。作为该组织最高负责人，负有确保该组织各项活动遵循会议的各项决议、指导和监督秘书长工作等职责。
 <br/>
 <br/>
 64岁的孟宏伟，2016年11月当选国际刑警组织主席，是首个当选国际刑警组织主席的中国人，任期至2020年。
 <br/>
 <br/>
 孟宏伟上任之初，就有人权组织担心，致力于破坏人权的中共，或会利用组织作为通缉海外异见人士的平台。
 <br/>
 <br/>
 因此，孟宏伟当选主席，被外界认为是中共利用贿赂渗透了国际刑警组织，以及中共与法国幕后交易的结果，同时也意味着后者已经堕落为中共的外围组织，从而在国际社会已经丧失了公信力，正在逐渐淡出人们的视野。
 <br/>
 <br/>
 如今，号称除联合国之外第二大国际机构的堂堂国际刑警组织主席孟宏伟，被中共秘密抓捕一个礼拜之后，其妻只能依据他归国后最后给他妻子发的一把刀子的表情，认定他已经落入魔掌。在国际舆论压力下，中共直至7日才承认孟宏伟被抓捕。
 <br/>
 <br/>
 评论人士张林认为，这个事实已经雄辩地证明了，这个国际刑警组织已经处在荒缪状态，连主席失踪都不明就里，都无可奈何，谈何调查其它犯罪？谈何保护人权？
 <br/>
 <br/>
 各界在高度关注孟宏伟失踪的同时，舆论也没有忘记，孟宏伟这个当了一辈子共产党刀把子的酷吏，自己终于也将死于共产党屠刀之下。
 <br/>
 <br/>
 许多网友认为，中国法国交易竟然推出一个专制独裁政权的刽子手，做所谓国际刑警组织的头目本身就涉嫌许多贿赂犯罪。本届法国政府与联合国应该合作，趁机调查国际刑警组织的系列非法活动。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：戴明）
