### 揭秘：胡锦涛险被自己授予的上将暗杀内幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月01日讯】中共前总书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 在任期内，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 集团曾策划3次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暗杀.html" target="_blank">
  暗杀
 </a>
 他。关于暗杀胡锦涛的内幕早在海外媒体曝光。大陆媒体也罕见曝光了相关细节。而刺杀胡锦涛的凶嫌，正是胡亲自授予上将军衔的原海军司令员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张定发.html" target="_blank">
  张定发
 </a>
 。
 <br/>
 <br/>
 中共军方授衔仪式一般在中共建军节〝八一〞前夕，但也有例外。2004年9月25日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 曾向两名军委委员，海军司令员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张定发.html" target="_blank">
  张定发
 </a>
 和第二炮兵司令员靖志远，授予了上将军衔。
 <br/>
 <br/>
 同年9月19日的中共十六届四中全会上，张定发、靖志远都已晋升为军委委员；同时胡锦涛出任军委主席，接替下台的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 。
 <br/>
 <br/>
 据报，张升民、魏凤和、张定发和靖志远都是以中将军衔晋升中央军委委员之后不久，即晋升上将军衔。按惯例，中共中央军委委员一般是由上将来担任。如果中将晋升军委委员，当局一般会很快授予其相应的上将军衔。
 <br/>
 <br/>
 虽然张定发是胡锦涛授予的上将，然而张却是江泽民的心腹。从2002年十六大到2004年的四中全会期间，江泽民把张定发提升为海军司令员，靖志远提升为二炮司令员，并把他们全部塞进军委会，使中央军委由8人扩展为11人。
 <br/>
 <br/>
 陆媒军事栏目曾罕见报导，2006年，时任军委主席胡锦涛在黄海检阅军队时，险遭张定发等人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暗杀.html" target="_blank">
  暗杀
 </a>
 。
 <br/>
 <br/>
 文章说，胡锦涛2006年5月初，乘坐中共最先进的一艘导弹驱逐舰，到黄海视察北海舰队，突然两艘军舰同时向胡乘坐的导弹驱逐舰开火，驱逐舰上5名海军被打死。惊慌失措之下，驱逐舰立即调转头，疯了似地驶离舰队演习海域，直到安全海域。
 <br/>
 <br/>
 为了避免再次遭遇暗杀，胡换乘舰上的直升飞机飞回青岛基地。未敢作停留，也未回北京，直飞云南。一个星期后，胡把一切安排妥当，才回北京露面。
 <br/>
 <br/>
 文章直接点明，张定发是江泽民的死党，外界怀疑他与黄海暗杀有关。
 <br/>
 <br/>
 然而有关胡锦涛黄海险遭暗杀事件，最初是由港媒率先曝光。
 <br/>
 <br/>
 北海舰队是张定发的老巢，他从1985年开始任北海舰队参谋长助理后，先后任北海舰队参谋长、北海舰队副司令员，北海舰队司令员兼济南军区副司令员，2000年任海军副司令员，2003年6月任海军司令员。
 <br/>
 <br/>
 港媒披露，暗杀事件发生后，被拘捕的舰艇官员供认，命令是江泽民下达的，江的军中心腹张定发指挥手下人干的。为确保万无一失，张定发要求攻击胡锦涛的时候，要两舰夹击，并许愿给他们大幅度晋级。
 <br/>
 <br/>
 黄海刺杀事件7个月后，即2006年12月14日，63岁的张定发死在北京三军总院301医院。
 <br/>
 <br/>
 张定发死后没有吊唁，没有悼词，中共官媒也没有发布讣告。只有海军军报刊出个简讯，其中只有一个简单的履历，连个黑白照片都没有。
 <br/>
 <br/>
