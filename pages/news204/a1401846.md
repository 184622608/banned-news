### 【禁闻】〝川习会〞中共承诺管控〝芬太尼〞输美
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】曾被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 总统川普抨击危害美国人健康的止痛剂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芬太尼.html" target="_blank">
  芬太尼
 </a>
 (Fentanyl)，是一种致命的合成鸦片类药物，它导致每年有几万美国人死亡。12月1号的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞，中方承诺管制芬太尼出口美国。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 外交部在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞结束后发表声明，决定〝对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芬太尼.html" target="_blank">
  芬太尼
 </a>
 类物质进行整类列管，并启动相关法律规范的调整程序〞。美方声明则强调，这意味着把芬太尼贩售到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 的人，将会受到中国法律最严厉的惩罚。
 <br/>
 <br/>
 所谓对芬太尼类物质进行〝整类列管〞，就是将结构和芬太尼类似的化合物全部列入管制范围，也就是说，不再像以前一样，发现一种类似物才管制一种。
 <br/>
 <br/>
 旅美时事评论员唐靖远表示，芬太尼是属于强效麻醉性镇痛药，是经过人工合成就可以完成，它不像海洛因需要经过人工种植才能取得，是一种实验室毒品。
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝它拥有几个特点，第一，它是制造成本低。第二，销售渠道非常的广泛，非常的方便快捷。第三，这个药物本身它的毒性大，成瘾性也大，所以就导致这个毒品的扩散速度非常快，而且它自己还可以被产生出多达数十种的这种延伸品，这个导致相关的监管机构在对这个药物进行管制，去进行监别，去进行检查的时候，就面临着非常大的技术上的困难。〞
 <br/>
 <br/>
 唐靖远表示，中共对芬太尼这种药物一贯采取放任态度，没有实行严格监管的制度。所以导致中国大陆很多药厂可以合法生产制造，然后通过网络直接配送出去。
 <br/>
 <br/>
 唐靖远：〝因为这个毒品它是没有什么味道的，所以美国它在邮政系统上很难去把它给查出来，一个小小的信封它就可以装下上千次服用量的这样的芬太尼。〞
 <br/>
 <br/>
 据了解，芬太尼在美国仅被批准用于癌症等引起的严重疼痛，但由于这种合成药物制造成本低廉，又可以在互联网上销售，直接邮寄到美国，走私商几乎可以轻易取得。媒体指出，2016年，美国有将近四万三千人因鸦片类药物丧命。
 <br/>
 <br/>
 唐靖远：〝这种毒品它还有一个非常突出的特点，就是它被人体吸收的途径非常的可怕，它直接就可以通过人体的皮肤。从另一个方面就是说明芬太尼的这种药品，它进入人体的这个途径和渠道也是非常的快捷方便。〞
 <br/>
 <br/>
 广东中山大学药学院教授黎星术接受本台记者采访时表示，芬太尼药效迅速而且作用时间非常短。
 <br/>
 <br/>
 广东中山大学药学院教授黎星术：〝芬太尼主要是镇痛用的，它的药效很强的，是吗啡的80倍，它本身是作为鸦片受体的激动剂。如果静脉注射的话，一分钟以后就起效，长期服用了以后，它能够上瘾。它的副作用就是可能出现恶心，呕吐，还会出现识别模糊。〞
 <br/>
 <br/>
 今年1月，美国联邦参议院公布报告指出，中国经销商在2年的时间内，运用网路匿名和电子商务，将价值近8亿美元的芬太尼透过美国邮政系统非法销入美国。
 <br/>
 <br/>
 唐靖远：〝现在我们可以看到由芬太尼所引起的这种对美国社会的破坏和冲击，这种性质跟那个鸦片战争其实没有什么两样，只不过这一次这个毒品战争的这个对象倒转过来，就是中共成为了这个芬太尼的最重要的输出国，而美国成为这个芬太尼的最重要的输入国。〞
 <br/>
 <br/>
 去年10月，美国总统川普便指出，鸦片类药物危机是全国性公共卫生紧急事件，美国必须阻止〝廉价又致命的芬太尼〞从中国涌入。
 <br/>
 <br/>
 唐靖远：〝从客观上证明了，中共多年以来其实一直就是一种放任的态度，放任芬太尼这种药物对美国社会进行扩散和渗透。也就是说，中共其实它是一种有意识的在使用一种毒品战争、化学战这种方式，来对美国社会进行直接摧毁和祸害。〞
 <br/>
 <br/>
 唐靖远认为，这次中、美双方达成协议，至少遏止了中共这个企图。至于中共要管制芬太尼，要堵住芬太尼对美国的输出，这种承诺能够做到多少，还需要进一步的观察。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/周天
