### 【禁闻】12月6日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 四川乐山数千人上街 抗议泊车新规
 <br/>
 <br/>
 12月3号，四川省乐山市数千民众自发走上街头，在各自小区附近举行示威活动，抗议政府日前出台的〝差别化停车收费〞新规定，满城到处是〝抗议〞、〝抗议〞的喊声。
 <br/>
 <br/>
 2. 重庆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 冉崇碧被判刑三年半
 <br/>
 <br/>
 重庆维权人士冉崇碧，因在北京上访，2017年2月被截访遣返后拘留，一年后，一审被以寻釁滋事罪判刑3年6个月，冉崇碧不服上诉，今年12月6号，重庆市第二中级法院二审宣布维持原判。
 <br/>
 <br/>
 3. 遭村官打伤 江苏村民公安局讨说法
 <br/>
 <br/>
 江苏省张家港市金港镇学田村村民刘网生，2017年5月遭到村干部殴打致鼻梁骨骨折，但打人者至今逍遥法外。今年12月5号，刘网生向张家港市公安局申请信息公开，要求公开这起故意伤害案件的处理结果。
 <br/>
 <br/>
 4. 武汉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 北京住店 被以寻釁滋事罪拘留
 <br/>
 <br/>
 武汉市硚口区访民金桂兰，9月2号在北京正常住店时被警察带走，遣返回武汉后，被以寻釁滋事的罪名关进武汉市第一看守所，90多天不被允许会见家属。12月5号，当地维权人士高新等4人，前往看守所，为她存钱存物。
</div>
