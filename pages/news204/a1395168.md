### 【禁闻】10月12日维权动态
------------------------

 【新唐人2018年10月12日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1、南京公民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_私家车贴标语.html" target="_blank">
  私家车贴标语
 </a>
 遭正式逮捕
 <br/>
 <br/>
 南京公民秦沪辉因在自家的私家车上粘贴宪政、新闻自由、司法独立、官员财产公示等标语，9月4号遭南京警方以寻釁滋事罪名刑事拘留，10月10号再被正式逮捕。
 <br/>
 <br/>

 <br/>
 <br/>
 安徽省合肥市瑶海区访民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘宁.html" target="_blank">
  刘宁
 </a>
 被控〝寻釁滋事、妨碍公务〞一案，10月10号在瑶海区法院开庭审理，当地数十名维权人士到场声援。刘宁坚称自己无罪，法官未当庭宣判。
