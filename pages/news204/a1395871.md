### 孟宏伟妻子在中国职务曝光 还是政协委员
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】国际刑警组织主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 落马事件引发国际关注。他的妻子Grace Meng（中文名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高歌.html" target="_blank">
  高歌
 </a>
 ）更多个人信息也被曝光。据山东省政协官网显示，高歌是中国民主建国会（民建）青岛市委员会副主委，还是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东省政协委员.html" target="_blank">
  山东省政协委员
 </a>
 。
 <br/>
 <br/>
 香港《星岛日报》报导称，青岛市民建办公室工作人员表示：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高歌.html" target="_blank">
  高歌
 </a>
 现在还是副主委，其他情况不太清楚。〞
 <br/>
 <br/>
 10月7日，国际刑警组织主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 妻子在法国里昂举行记者会，曝光孟宏伟被失踪一事。当晚，中共中纪委网站发文回应，孟宏伟涉嫌违纪被调查。
 <br/>
 <br/>
 由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟妻子.html" target="_blank">
  孟宏伟妻子
 </a>
 Grace Meng接受外媒采访时，刻意背对镜头，不愿暴露真容，也不愿对外披露其中文姓名等更多个人信息，引起外界猜测。
 <br/>
 <br/>
 随后，孟宏伟妻子的身份和个人信息，被多家媒体起底。有消息称，孟宏伟妻子中文名叫高歌，现年46岁，是北京大学经济学博士研究生毕业，拥经济学博士，法律硕士，曾任中国青联委员、中国青年企业家协会常务理事等职务。
 <br/>
 <br/>
 近日，据称是高歌2015年参加官方活动的照片，也在社交媒体上流传。高歌以〝中国民主建国会青岛市委员会副主委〞的身份参加会议。照片的来源为〝中国民主建国会山东省委员会〞及官方相关报导的视频截图。
 <br/>
 <br/>
 不过，推特（Twitter）账号〝棱镜新闻中文站〞发消息说，高歌涉嫌学历造假、坐领高薪、生活奢靡。
 <br/>
 <br/>
 此外，有坊间传言，高歌已经离开里昂，正在巴黎某处受到法国、美国情报部门的保护。中国民运人士张健日前在Twitter发文称，高歌最近继续联系法国和欧美媒体，准备召开新的发布会。
 <br/>
 <br/>
 外界猜测，高歌或手握中共高层某些内部机密，以此作为与北京谈判的筹码。
 <br/>
 <br/>
 张健还表示，北京当局已把孟宏伟和其妻子涉嫌收受贿赂、循私舞弊等违法犯罪部分资料，传给法国警方和外交部门，要求引渡高歌。不过这一消息未得到中共官方证实。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
