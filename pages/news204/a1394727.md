### 【禁闻】10月9日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 老兵平度维权数日 政府干扰通讯
 <br/>
 山东平度老兵维权事件已经持续多天，10月7号，数千警察暴力清场后，五六百名老兵被抓遣返回家。8号，平度市被警察全面封锁，外地声援老兵无法进入。由于政府干扰当地的网络通讯，至10月9号中午，现场已经很少有消息传出。
 <br/>
 <br/>
 2. 北京访民残疾人车被装不明物体
 <br/>
 10月8号，北京市丰台区维权人士葛志慧，驾驶残疾人三轮车到北京市政府上访，发现车下的油箱被人捆绑安装了黑色不明物体，葛志慧怀疑是爆炸物，立即报警。警察到场后，轻松拆下不明物体，并拒绝告诉葛志慧该物体是什么。〝维权网〞观察图片后认为，可能是维稳用的电子跟踪器。
 <br/>
 <br/>
 3. 西安村民省府上访 要求安置
 <br/>
 10月8号，西安市长安区郭杜街道办事处岔道口村的村民，前往陕西省政府上访，举报长安区和郭杜街道办事处在对该村进行〝城中村改造〞中的弄虚作假、违法违规行为，要求政府按照当初承诺，给全体村民建安置房，并查处在征地、拆迁安置中的违法行为。
 <br/>
 <br/>
 4. 内蒙古数百退伍老兵区政府维权
 <br/>
 10月9号，内蒙古数百名退伍老兵聚集在自治区政府大楼前维权，要求政府兑现承诺，解决安置问题。
</div>
