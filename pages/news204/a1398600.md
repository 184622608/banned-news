### 【禁闻】新疆再教育营引众怒 多国要求前往调查
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】关于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育营.html" target="_blank">
  再教育营
 </a>
 〞羁押
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维吾尔人.html" target="_blank">
  维吾尔人
 </a>
 ，侵犯人权的问题，引发国际关切。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 多个成员国要求联合国前往新疆调查。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
  联合国
 </a>
 人权理事会规定5年一度定期审议所有联合国成员国的人权纪录。
 <br/>
 <br/>
 6号，联合国人权理事会审议中国的人权状况时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育营.html" target="_blank">
  再教育营
 </a>
 〞问题再次成为各国代表关切的主要焦点。
 <br/>
 <br/>
 美国、加拿大、英国、法国、德国、瑞士、澳洲和其他国家的代表呼吁中共停止拘禁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维吾尔人.html" target="_blank">
  维吾尔人
 </a>
 和其他少数民族成员，并敦促中共尊重宗教、言论和结社自由。
 <br/>
 <br/>
 华府希望北京如实说明〝过去5年来，共有多少人被迫关押在新疆所有拘留设施。〞
 <br/>
 <br/>
 中共当局投入新疆的维稳经费日趋庞大。根据美国智库詹姆斯顿基金会11月5号发表报告，指出，去年中共在新疆，职业培训的支出下降了7%，而用于保安的支出则上升了近200亿元人民币，增幅高达213%。
 <br/>
 <br/>
 流亡美国原新疆干部阿尔曼接受记者采访时表示，新疆现在的局势，就像当年二战纳粹消灭犹太人一样。他说，新疆当地的维吾尔人只要一出门，晚上能平安回到家，就算幸运！
 <br/>
 <br/>
 流亡美国原新疆干部阿尔曼：〝新疆现在也没有生产，白天晚上就是抓人，查你的电话号码，查你的手机，查你的身分证，你不要说出国，连这个城市到另一个城市都去不了。你去医院、去学校、商场，所有的地方都是监控、安检。〞
 <br/>
 <br/>
 北京维权人士胡佳表示，新疆是中共严控最厉害的地方，也是中共维稳系统的一个实验场。中共的政法机关被称为刀把子，它和枪杆子是维护中共专制统治的两把利器。现在对新疆而言，就是属于枪杆子和刀把子共管的方式。
 <br/>
 <br/>
 北京维权人士胡佳：〝维稳支出其实就是打内战，就是对内维稳镇压，上万亿的支出，听起来增加两、三百亿在新疆的话，好像不算多，但是从新疆人口来讲，上千万的维吾尔民族，大部分居住在新疆。如果说，这个钱都是用在他们头上的话，那么，这个个人承受被维稳的支出，应该是在全中国(维稳)系统中最高的。〞
 <br/>
 <br/>
 总部设于德国的流亡维吾尔人组织〝世界维吾尔大会〞，今年8月向联合国一个人权委员会提交报告，指控中共政府大约将100万名维族人非法拘禁在〝再教育中心〞，进行思想改造。
 <br/>
 <br/>
 胡佳：〝所谓的思想改造从来就是邪恶的，尤其是来自于共产党，它的那种思想改造无异于纳粹党思想改造，如果真的有上百万人在所谓的学习班里面，那我只能说，那里有上百万人集中营的受难者。〞
 <br/>
 <br/>
 胡佳表示，到底是谁把这股无名邪火点燃，又是谁制造像魔鬼一样的行为。
 <br/>
 <br/>
 胡佳：〝这种升级式的手段，究竟是什么造成的？这套东西也不是共产党一党发明的，整个共产主义的赤色法西斯群体，历来是采取这样的手段。〞
 <br/>
 <br/>
 关于新疆再教育营羁押信奉伊斯兰教的维吾尔人，中共代表团团长、外交部副部长乐玉成在联合国辩称，这是一种预防措施，是为了制止新疆民众转向恐怖主义。
 <br/>
 <br/>
 美国和德国要求联合国前往新疆和西藏，调查中国大规模拘押少数民族和限制宗教自由的指控。英国也提出书面要求。
 <br/>
 <br/>
 《法国国际广播电台》报导，法国呼吁中国关闭新疆再教育营，停止在新疆地区的大规模拘押，并且邀请联合国人权问题高级专员以及其他人权专员到新疆就地视察。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/周天
