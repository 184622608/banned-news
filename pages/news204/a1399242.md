### 黑龙江原网信办主任自缢 官方紧急排除他杀
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑龙江工程学院.html" target="_blank">
  黑龙江工程学院
 </a>
 党委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李耀东.html" target="_blank">
  李耀东
 </a>
 日前在办公室自缢身亡，官媒称，公安机关现场勘查取证，排除他杀。
 <br/>
 <br/>
 11月13日，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑龙江工程学院.html" target="_blank">
  黑龙江工程学院
 </a>
 官网发布公告称，12日下午，该学院党委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李耀东.html" target="_blank">
  李耀东
 </a>
 被发现在其办公室自缢身亡，经公安机关现场勘查取证，排除他杀。
 <br/>
 <br/>
 目前，在黑龙江工程学院的官网上，李耀东的名字已经从现任领导一栏撤下。李耀东今年2月才从中共黑龙江省委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网信办主任.html" target="_blank">
  网信办主任
 </a>
 一职调任黑龙江工程学院党委书记。
 <br/>
 <br/>
 公开报道显示，李耀东1963年8月出生，黑龙江哈尔滨人。历任中共黑龙江省广播电视局副局长、党组成员、副总编辑，中共七台河市委常委、宣传部长，行署常务副专员，中共黑龙江省地矿局局长、党组书记，中共黑龙江省委宣传部副部长、省委网信办主任等职。2018年2月起任中共黑龙江工程学院党委书记。
 <br/>
 <br/>
 中共十九大后大陆频频发生官员〝不正常死亡〞现象。其中，中共前军委委员、上将张阳去年11月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀.html" target="_blank">
  自杀
 </a>
 ，是40年来自杀官员中级别最高的，事件震惊朝野。
 <br/>
 <br/>
 中共澳门中联办主任郑晓松于10月20日晚在其澳门住所堕楼身亡，消息更是震惊海内外。
 <br/>
 <br/>
 综合媒体报导，2009年至2016年不完全统计，有243名中共官员自杀，〝十九大〞后未有退潮迹象，至今有数十人自杀。当局通常对死因轻描淡写，调查亦讳莫如深，结论草率武断，不少人被指患抑郁症。
 <br/>
 <br/>
 有评论认为，中共官员们自杀成风有多种原因：有犯案被查时畏罪自杀保家人；有卷入中共内斗相互廝杀中被强行〝自杀〞；也有因贪欲膨胀或迫害民众遭到天理报应。因此他们既是中共流氓政权下的施害者，也是中共体制下的牺牲品。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：戴明）
