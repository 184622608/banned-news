### 川习会前党媒翻炒义和团 谁是当代慈禧？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】11月底的中美首脑会晤在即，北京不断释放愿与美国和解的信号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 经济智囊
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 将再次访美协商。当此敏感时刻，党媒喉舌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新华网.html" target="_blank">
  新华网
 </a>
 突然发文纪念〝扶清灭洋〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_义和团.html" target="_blank">
  义和团
 </a>
 运动，显得别有意味。
 <br/>
 <br/>
 11月13日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新华网.html" target="_blank">
  新华网
 </a>
 发表题为&lt;民族意识觉醒——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_义和团.html" target="_blank">
  义和团
 </a>
 反帝爱国运动&gt;的文章，高调赞扬义和团当年抗击八国联军时表现出的〝抵御外侮的决心和勇气〞。
 <br/>
 <br/>
 此文发表的时机微妙。日前，北京邀请〝老朋友〞、美国前国务卿基辛格访华，被指有意让基辛格从中斡旋，将对美方做出某种让步。连一贯强硬的《环球时报》都罕见发声，强烈建议放弃〝中国制造2025〞。
 <br/>
 <br/>
 与此同时，中美谈判中方首席代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 正在积极和美方沟通，希望达成和解协议。日前美国商务部长罗斯证实，刘鹤即将赴美协商。
 <br/>
 <br/>
 新华网此时〝纪念义和团〞，显然和北京的调子不大合拍。
 <br/>
 <br/>
 今年5月刘鹤赴美谈判时，共青团中央的媒体也曾刊登图片，将中美对话现场照片和当年签署辛丑条约的旧照做对比。此举被指试图用〝民族情绪〞绑架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 当局，阻止其对与美国签署〝卖国〞协议。
 <br/>
 <br/>
 不过，如今时过境迁，中共体制内对经济危局已有共识，此时再炒作〝民族情绪〞，似乎政治倾向更加明确。
 <br/>
 <br/>
 时政评论员唐靖远分析，此文炒作义和团，除了暗示习近平是当代
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_慈禧.html" target="_blank">
  慈禧
 </a>
 之外，另一个主要目的是用〝丧权辱国〞的政治后果捆绑习近平，用舆论给川习谈判制造难题，逼迫他继续和美国对抗，这可以说是一种借刀杀人的路数，令习近平处在当初刘鹤一样进退两难的境地。
 <br/>
 <br/>
 和炒作义和团〝异曲同工〞的是，中国人民大学国际关系学院教授尹继武11月9日举行专题讲座，从〝学术角度〞剖析美国总统川普的〝人格障碍〞。这似乎与北京不许对川普进行人身攻击的禁令相左。
 <br/>
 <br/>
 目前中共政局混乱，阻止中美和解的各种声音，背后也可能涉及不同派系的不同利益。
 <br/>
 <br/>
 除了党内反习势力给习近平挖坑外，也有体制内强硬派出于保党目的，不愿对美妥协。不过，现在贸易战已经令中共陷入真正的危机，〝宁死不妥协〞的论调，多数已被〝假妥协换生存〞的建议所取代。
 <br/>
 <br/>
 另外，不可否认的是，还有相当一部分中国人希望中美对抗继续升级，加速中共政权解体。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
