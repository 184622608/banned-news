### 〝宁见阎王不见习〞？传澳门中联办主任被灭口
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办主任.html" target="_blank">
  中联办主任
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 近日突然坠楼身亡的消息，震惊各界。据悉，在习近平23日主持的港珠澳大桥通车仪式上，郑晓松已被安排出席。在此敏感时刻郑晓松却〝宁见阎王，不见习〞，疑点重重。甚至有海外爆料称，郑是被人灭口，接下来中共官场会有更多人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀.html" target="_blank">
  自杀
 </a>
 和被杀，特别是公检法、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 。
 <br/>
 <br/>
 中共国务院港澳办官网21日早上发声明称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中联办主任.html" target="_blank">
  中联办主任
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 因抑郁症病发，于20日晚在其澳门住所堕楼身亡。
 <br/>
 <br/>
 澳门中联办网页显示，郑晓松在坠楼身亡的前一天，即19日，还在澳门办公室内会见澳门民联智库负责人一行；郑晓松还在澳门美高梅出席了另一场活动，与会高官包括澳门特首崔世安、中共政协副主席何厚铧等。
 <br/>
 <br/>
 从官方或媒体公布的郑晓松澳门活动照片看，这位澳门中联办主任并未显露出任何异样。
 <br/>
 <br/>
 澳门立法会主席贺一诚向传媒披露，20日晚与澳门中联办人员吃饭时，中联办人员突然离席，未有透露原因。
 <br/>
 <br/>
 澳门司警局局长薛仲明21日下午被传媒追问时重申，案件已转交调查科跟进，未有提及官媒声称的〝抑郁症〞。
 <br/>
 <br/>
 消息人士引述澳门中联办某部长级官员的话称，对郑晓松突然死亡感到很惊讶，因为最近才见过他，讲话思路清晰，发言也很精神，看不出有毛病，更不要说抑郁症了。
 <br/>
 <br/>
 该消息人士还说，对于郑晓松死亡，目前澳门中联办〝非常紧张，消息全部封锁〞。因为习近平23日可能会为港珠澳大桥剪彩，现在爆出这么一单事，实在很难看，故上下乱作一团。
 <br/>
 <br/>
 <h4>
  郑晓松死因有诸多疑点
 </h4>
 <br/>
 法学博士、时事评论员桑普对《大纪元》分析认为，郑晓松死因有诸多疑点。一是，官媒和澳门警方说法相互矛盾，澳门警方还没公布案情，港澳办却急以〝抑郁症〞为郑之死定调。
 <br/>
 <br/>
 二是，港珠澳大桥即将举行通车仪式，外传习近平出席珠海通车仪式，郑晓松已被安排出席，〝这么重大的事情，这么敏感的时候跳楼，是不是一种巧合？〞
 <br/>
 <br/>
 桑普分析认为，从近期国际刑警主席孟宏伟突然失踪，到郑晓松突然跳楼，加上从中共近期很多官员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自杀.html" target="_blank">
  自杀
 </a>
 、或被杀的现象来看，说明中共已步入末期阶段，气数已尽，不得不用强力方式镇压自己的官员。
 <br/>
 <br/>
 时评家黎则奋表示，中共接二连三有官员自杀或被自杀，肯定是有些事情摆不平才这样做。
 <br/>
 <br/>
 <h4>
  郑晓松〝实际上是被干掉的〞？
 </h4>
 <br/>
 21日总部设在香港的中国人权民运信息中心引述消息称，郑晓松近日被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 官员问话，这可能是郑晓松死亡的主因之一。
 <br/>
 <br/>
 自习近平上台反腐〝打虎〞以来，中共官员非正常死亡的案例不断增加，5月份，更是达到〝高峰〞，一个月内有7名官员死亡。
 <br/>
 <br/>
 10月21日，有海外爆料者称，郑晓松〝实际上是被干掉的〞，〝郑晓松最好的哥们就是孟宏伟！〞
 <br/>
 <br/>
 爆料者预言，〝香港的中联办姓王的哥们也快了！大家往下看吧！〞似乎是暗示现任香港中联办主任王志民亦将出事，〝接下来很多人会跳楼、很多人自杀和被杀，特别是公检法、中纪委〞。
 <br/>
 <br/>
 资料显示，郑晓松1959年出生，河北石家庄人，曾在中共外交部、新华社香港分社等单位工作。2016年6月郑晓松任中共对外联络部副部长，去年9月任澳门中联办主任。
 <br/>
 <br/>
 江派二号人物曾庆红曾主管港澳事务多年，驻港机构历来潜藏曾的势力较多，香港澳门历来也是江派大老虎们敛财及洗黑钱要地。曾庆红是江泽民的〝大内总管〞，曾利用海外掌控的特务系统，不断制造事端。而郑晓松上任一年突然死亡，是否与曾庆红有某种关联，引发外界联想。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：祝馨睿）
