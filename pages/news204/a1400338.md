### 恶人先告状？  中共欲以〝垄断〞为由重罚美光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】日前，中共官方以〝垄断〞为由，调查美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美光科技.html" target="_blank">
  美光科技
 </a>
 等3家公司，或将予以重罚。北京此举被认为是报复美国。本月初，美国司法部起诉中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 公司窃取美光商业机密。更糟糕的是，福建晋华因威胁美国国家安全，步中兴通讯后尘，被美国商务部禁售。
 <br/>
 <br/>
 中共反垄断局官员日前表示，市场监管总局对三星（Samsung）、海力士（SK Hynix Inc.）和美光（Micron）3家公司的反〝垄断〞调查取得进展。美光等3家企业控制着动态随机存取记忆体（DRAM）晶片95%的全球市场。2017年，中国占美光半导体产品销售额的51%，占三星销售额的40%，占海力士销售额的33%。
 <br/>
 <br/>
 有分析说，美光等企业若被认定操纵价格，最高可能面临25亿美元以上的罚款。
 <br/>
 <br/>
 北京此举被指是报复美国。美国司法部11月1日正式起诉中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 及台湾联电，指控两家公司共谋窃取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美光科技.html" target="_blank">
  美光科技
 </a>
 的商业机密等三项罪名。
 <br/>
 <br/>
 在此之前，10月29日，美国商务部以保护国家安全为由，将福建晋华列入出口管制实体清单，自10月30日起限制对其出口。福建晋华恐面临中兴通讯此前的局面，迅速瘫痪。
 <br/>
 <br/>
 时政评论员唐靖远分析说，美方制裁晋华，象征美国反击中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_盗窃知识产权.html" target="_blank">
  盗窃知识产权
 </a>
 的行动进一步升级，与美中贸易战相关。
 <br/>
 <br/>
 事实上，美光与晋华的知识产权纠纷已持续多时。早在去年12月，美光就已在加州起诉福建晋华和台湾联电。
 <br/>
 <br/>
 而福建晋华和台湾联电则在今年1月，反以侵犯专利为由，将美光告上中国法庭。随后，美光部分产品在中国境内被禁售。
 <br/>
 <br/>
 唐靖远称：中共把行政和政治渗入到经济行为、经济活动里面来，甚至动用司法的力量。然后跟西方公司形成了一种完全不对等的竞争关系，这个就是不公平竞争，也就是美国指责的中共不公平贸易行为。
 <br/>
 <br/>
 以晋华为例，中国企业窃取美国企业的技术，然后利用权力、手段把美国企业挤出中国市场。这种行为对全球的贸易和正常的经济行为，毫无疑问是一种破坏。
 <br/>
 <br/>
 福建晋华是〝中国制造2025〞计划企业，中共对其投入370亿发展存储晶元，其晶元产品是该计划的核心内容。很显然，中共一定会支持晋华，排挤美光。
 <br/>
 <br/>
 因此，外界认为，中共官方日前以反〝垄断〞为由欲重罚美光，有恶人先告状之嫌。
 <br/>
 <br/>
 事实上，不仅仅是美光与福建晋华这一个案例，中共多年来无所不用其极，从美国社会的各方面入手，在大众无法察觉的情况下，用盗窃美国知识产权等手段，掌握美国的经济命脉。这是川普政府之所以对中共开启贸易战的关键原因之一。
 <br/>
 <br/>
 唐靖远分析，美国对晋华的制裁可以视为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技制裁战.html" target="_blank">
  科技制裁战
 </a>
 〞，特点是立竿见影。一制裁马上就瘫痪，效果非常直接。而且可以定点打击、对象精确。中共这些大型国企业背后都是中共权贵阶层的集团、家族在掌控。〝所以从这个角度上讲，这种科技制裁战对中共震慑的力度其实更大。〞
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
