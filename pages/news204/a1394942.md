### 孟宏伟被抓搅动政法委？孟妻：我被两组人马锁定
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】截止11日，国际刑警组织主席、中共前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被调查已4天，中共政法委至今仍没有表态。孟妻则在国际是高调替孟发声，她更称，自己遭〝两组人马〞锁定。
 <br/>
 <br/>
 中共官方7日深夜通报：中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 、国际刑警组织主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 涉嫌违法，目前正接受中共监委监察调查。
 <br/>
 <br/>
 这是在孟妻就丈夫只身返回中国失联，4日向法国里昂警方报警之后，中共官方的首次回应。
 <br/>
 <br/>
 中共公安部官网8日即于头条位置刊文称，公安部部长赵克志当日凌晨，紧急召开公安部党委会议，就孟宏伟被调查一事进行通报，并宣布成立工作组，查处孟的同党。
 <br/>
 <br/>
 与公安部形成鲜明对比的是，至今中共政法委仍未就此事发声。据称，现任中共政法委书记郭声琨，与孟宏伟关系密切。两人即是湖南中南工业大学校友，同时又都曾担任公安部正副部长。郭声琨辖下的政法委一直失声，令外界颇为奇怪。
 <br/>
 <br/>
 对此，有港媒分析说，从中共公安部连夜召开会议，通报孟宏伟的案情，以及政法委书记郭声琨失声来看，此案是由中纪委直接操作，公安部和政法委事前很可能都蒙在鼓里。
 <br/>
 <br/>
 目前中共政法委〝一把手〞郭声琨，以及委员周强、傅政华等人，都属
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 的亲信。孟宏伟也曾是周的亲信，在公安部担任副部长长达14年，此次孟被查，赵克志宣布查处孟的同党。分析认为，接下来，公安系统甚至整个政法系统或将大震荡，背后很可能涉及生死博弈。
 <br/>
 <br/>
 值得关注的是，孟宏伟被调查后，孟妻葛瑞丝‧孟（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Grace Meng.html" target="_blank">
  Grace Meng
 </a>
 ）不但高调召开记者会为孟发声，她还接连接受美联社、CNN等媒体采访。
 <br/>
 <br/>
 孟妻说，孟宏伟失踪后，她接到陌生男子的威胁电话。该男子说，他们有〝两组人马〞已经过来，目标就是她。
 <br/>
 <br/>
 孟妻认为，她丈夫被中共拘押是因为政治原因。 她还透露，她和孟宏伟有两个7岁的双胞胎儿子，她现在和媒体联系是因为担心一家人的安全，她需要得到帮助等。
 <br/>
 <br/>
 就在她接受采访期间，她接到3个来自中共驻法国领事馆的电话，其中一个电话是副总领事打来的，但她拒绝单独与他们见面。
 <br/>
 <br/>
 目前，法国警方已经对Grace Meng母子进行了安全保护，但Grace Meng不愿意公开自己的相貌，外界亦不知晓她的中文姓名。
 <br/>
 <br/>
 消息称，孟妻持香港护照，无业，2年来在里昂最chic的街区之一租住一幢实用面积270平方米的豪宅，无〝驻外人员家属〞身份。
 <br/>
 <br/>
 对于孟妻求助西方社会声援保护，很多网民认为孟宏伟被抓是咎由自取、作恶多端的报应，做了10多年中共的酷吏刀把子，他曾经让很多人无故被失踪，现在终于轮到他自己了。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
