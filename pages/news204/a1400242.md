### 美媒曝中共骇客瞄准一国际盛事 目的让人意想不到
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】美媒曝光中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网路攻击.html" target="_blank">
  网路攻击
 </a>
 正在将目标瞄准2020年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东京奥运会.html" target="_blank">
  东京奥运会
 </a>
 ，攻击主要分为五种，采取的手段极具隐蔽性，网路专家称，中共骇客的最终目的让人意想不到，竟然只是为了拉低外界对日本的评价。
 <br/>
 <br/>
 11月21日，美国新闻周刊网站日语版刊文称，住在日本及美国的17.4万名日本人9月份同时收到一份特殊的电子邮件。
 <br/>
 <br/>
 该邮件名为〝2020年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东京奥运会.html" target="_blank">
  东京奥运会
 </a>
 免费门票及礼品〞，内容为：致东京奥运会的关注者们，点击以下链接，可以为家人及朋友提供免费奥运门票……还有机会获取600美元。
 <br/>
 <br/>
 文章称，这封邮件是〝钓鱼邮件〞针对的对像是对体育感兴趣的相关人员、媒体及体育企业，诱使受骗人打开链接，从而获取密码及个人信息。而这一系列行动是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共黑客.html" target="_blank">
  中共黑客
 </a>
 ，发起的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网路攻击.html" target="_blank">
  网路攻击
 </a>
 。
 <br/>
 <br/>
 中国网路攻击代号主要分为五种，分别是：七月到七月、我的傲、20京、面具日本和色2020，而且这五种代号，针对的目标和主要目的也各不相同。如〝我的傲〞瞄准的是个人；〝面具日本〞瞄准的是电信企业及网路接入商等通讯相关企业。
 <br/>
 <br/>
 网路信息专家称，这些黑客的最终目的不是钱，也不是破坏基础设施，而是意在拉低外界对日本的评价。因为这些黑客不希望2020年东京奥运会的评价，超过2008年北京奥运会。
 <br/>
 <br/>
 同时，这些黑客还希望通过对企业施加网路攻击，让跨国公司对投资日本产生恐惧，从而有利于中共。文章说，由于中共黑客行动呈现组织化，东京奥运会可能受到大规模网路攻击影响。
 <br/>
 <br/>
 美国贸易代表办公室（USTR）11月20日发布报告也称，中共黑客攻击主要目标，不仅有美国，日本也是重点。为此，日本政府特意在通讯器材采购招标中，拒绝中共的两个大型通讯企业参与投标，显示出极强的戒备心态。
 <br/>
 <br/>
 针对中共对他国发动的网路攻击。11月8日，印媒Zee News报导中引述一名不愿具名的印度情报官员警告，中共秘密网军部队正在拦截和破译全球数字通讯。
 <br/>
 <br/>
 这名印度情报官员说，他们发现，中共负责网路战的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_61398部队.html" target="_blank">
  61398部队
 </a>
 旗下的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_骇客团体.html" target="_blank">
  骇客团体
 </a>
 ，在中共情报官员的协助下，透过对电脑系统，长期入侵的〝高级长期威胁〞（APT）系统，从事网路间谍活动，定位情报搜集、追踪网路资讯来源，且拦截和破译全球数字通讯，同时使用木马恶意软体透过网路窃取资料。
 <br/>
 <br/>
 7月份也有报导称，中共军方的61398网路部队，6月底已完成史上最大规模的〝扩编〞，除了继续加大各类网路和社交平台的监控外，对全球特定目标进行分散式阻断服务攻击（DDoS），将是日常〝训练〞的主要任务。
 <br/>
 <br/>
 报导引述中共军方高层知情者的话称，此次扩编方案，早已报请中南海最高层核准，整个行动历时长达半年，主要包括从各大军事院校和科研机构，以及战区选拔各类专业人才。
 <br/>
 <br/>
 而且还对相关人员的整个家族和好友圈，进行了周密细致和深入的背景审查，经层层把关后，分别抽调上海和北京。一名技术主管透露，网路部队可以综合运用DDoS攻击的多种方式，协调作战并已进行数以万计的远程测试。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉)
