### 王全璋妻致信主审法官：你们故意违法
------------------------

<div class="wysiwyg">
 【新唐人2018年11月06日讯】首先来关注中国的消息。709
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 ，失踪至今超过1200天。周一，他的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 给案件的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_主审法官.html" target="_blank">
  主审法官
 </a>
 写了一封公开信，痛批他们故意违法。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 在公开信中控诉，作为家属，她有权为丈夫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 聘请律师，但自己聘请的律师程海、余文生、蔺其磊、谢阳，多次到天津二中院，要求跟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_主审法官.html" target="_blank">
  主审法官
 </a>
 林昆、周虹见面沟通案情，都被以〝法官不在〞的理由回绝。而当律师们每次到天津第一看守所递交代理手续，要求会见王全璋时，也屡遭〝需法院批准〞的理由拒之门外。
 <br/>
 <br/>
 李文足表示，主审法官这种故意违法的行为，必将在中国法治上留下耻辱一笔。
 <br/>
 <br/>
 王全璋妻子李文足：〝我聘请的律师去，他们就是以（法官）不在，每次去找不到人，逃避了一年多了。但是他们却不远千里跑到山东或者北京找其他的律师，所以王全璋这个案件他们是故意违法了，所以我现在要求他们为自己违法的这个事情，需要他们一个交代或者承担相应的责任吧！〞
 <br/>
 <br/>
 王全璋常年为社会弱势群体提供法律援助，包括为法轮功学员辩护。自2015年7月10号失踪至今，已经三年四个月。他是〝709案〞中唯一仍被羁押的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 。去年2月天津市检察院以涉嫌〝颠覆国家政权罪〞正式起诉王全璋，但案件至今仍未开审。李文足表示，这严重超出规定的案件办理期限。
 <br/>
 <br/>
 今年10月21号，李文足发文表示，王全璋案代理律师刘卫国，因伤不再代理该案。刘卫国成为第8个退出王全璋案的律师。
 <br/>
 <br/>
 不过，刘卫国最近又突然在社交网站发布一封据说是王全璋的亲笔信，并附上他当天为王全璋存款的收据。刘卫国还表示自己会当仁不让，继续代理案件。 外界推测，背后很可能是当局一手安排。
 <br/>
 <br/>
 李文足则表示对此事不知情。她在信中说，刘卫国等律师进出看守所一路绿灯。虽然刘卫国是王全璋点名聘请的律师，但被点名的律师可不止一人。当局不批准其他律师辩护，实际是剥夺了王全璋的合法权益。
 <br/>
 <br/>
 新唐人记者熊斌、王子琦、陈杰采访报导
</div>
