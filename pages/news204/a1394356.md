### 山东老兵进京遭暴力截访 各地老兵声援
------------------------

<div class="wysiwyg">
 【新唐人2018年10月06日讯】10月4号，山东青岛平度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 进京被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力截访.html" target="_blank">
  暴力截访
 </a>
 ，导致6人住院。5号，数百名青岛退伍老兵集结山东平度市文化馆前的广场抗议，晚上警方清场失败，陆续有各地老兵，赶来声援，凌晨，现场老兵人数，估计已经过千。
 <br/>
 <br/>
 10月4号晚上，38名山东平度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 进京途中，在利津服务区遭当地公安副局长亲自带领数十名截访人员暴力拦截，6名老兵被打伤住院。
 <br/>
 <br/>
 5号上午，数百名青岛退伍老兵列队集结平度文化馆前的广场，为无辜被打的平度老兵讨说法。
 <br/>
 <br/>
 现场视频：〝我们都吃包子，大家看一下，我们为了共同一个目的，我们的战友被打，我们要来声援，要一个说法，公安局副局长带队打人，他就自己用公职权力打压老兵。〞
 <br/>
 <br/>
 入夜以后，更多老兵聚集，吸取了镇江事件被警察夜袭的教训，老兵们事先准备了木棍、钢盔。晚上9点左右，警方试图清场，老兵们奋力抵抗，清场失败。
 <br/>
 <br/>
 现场视频:〝晚上9点，山东青岛平度，战友们还在这里为被打的老兵讨说法。开始清人了，警察开始清人，警察开始清人。已经把老百姓清走了，他们想干嘛，还想成为镇江第二吗?〞
 <br/>
 <br/>
 全国各地老兵不断赶来声援，晚上10点，辽宁、即墨、日照、江苏等地约500名老兵陆续到达现场，凌晨老兵人数估计已经过千。
 <br/>
 <br/>
 现场视频:〝2018年10月5号晚上22点，辽宁省退役军人赶到山东青岛平度；即墨的战友，从即墨带来了衣服、被子，晚上12点了，即墨的战友又从即墨赶到平度，送来了衣服；现在是江苏战友跟全国各地、山东本地战友会合。〞
 <br/>
 <br/>
 此前在6月19号，江苏镇江多名维权老兵被打，各地老兵连日前往声援，22号深夜，遭上万警察暴力清场，数千名老兵遭到驱赶或拘捕，很多人被打得头破血流。
 <br/>
 <br/>
 新唐人记者熊斌、元真采访报导
 <br/>
