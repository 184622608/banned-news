### 【禁闻】11月27日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 购牛被骗 内蒙农牧民牧业厅抗议
 <br/>
 <br/>
 内蒙古3个盟市11个旗县的牧民，早前因购牛被骗数百万资金，多名牧民代表从11月19号开始，连续5天在呼和浩特维权
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 ，他们在自治区牧业厅等多个政府部门门口拉横幅抗议，要求尽快解决他们资金被骗的问题。
 <br/>
 <br/>
 2. 杨佳去世十周年 大陆民众墓地祭奠
 <br/>
 <br/>
 11月26号，是北京青年杨佳被执行死刑十周年，大陆各地的一些民众当天专程前往北京福田公墓祭奠他。杨佳当年因无辜被上海警察暴力对待又讨不到说法而冲进闸北区公安分局，杀死6名警察，他的名言是：〝你不给我一个说法，我就给你一个说法〞。前往祭奠的人士表示，杨佳是一位勇敢向当局讨公道的英雄。
 <br/>
 <br/>
 3. 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 吴士豪看守所遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酷刑.html" target="_blank">
  酷刑
 </a>
 <br/>
 <br/>
 75岁高龄的上海维权人士吴士豪，因今年7月1号前往北京上访，被关押到虹口区看守所，至今已经近5个月，日前家属聘请的律师到看守所会见他，得知他在里面遭受酷刑，被手铐脚镣铐在床上，嘴里还插着留置胃管。
 <br/>
 <br/>
 4. 山西介休村民北京控告村官后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失联.html" target="_blank">
  失联
 </a>
 <br/>
 <br/>
 山西省介休市城关乡梁吉村村民贾银，11月18号进京上访，先后到多个部门控告村官贪污和破坏集体土地等问题，25号还在中纪委门口自录视频揭露官员给他封口费。但他随后与外界失联，目前下落不明。
</div>
