### 家藏3吨现金被控3大罪  赖小民朋友圈牵出两女星
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】中国华融资产管理股份有限公司前董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 日前被双开。中共官方通报其三宗罪。陆媒称，赖小民家中藏有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赃款.html" target="_blank">
  赃款
 </a>
 2.7亿元人民币相当于3吨重，另外赖小民的朋友圈还牵出两名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女演员.html" target="_blank">
  女演员
 </a>
 。
 <br/>
 <br/>
 10月15日，中共官方通报，中国华融前党委书记、董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 被立案审查。
 <br/>
 <br/>
 通报指控，赖小民违反政治纪律和政治规矩，搞政治投机，违背中央金融工作方针，任人唯圈，与多名女性搞权色交易，收受巨额财物涉嫌受贿犯罪，非法占有公共财物涉嫌贪污犯罪等。
 <br/>
 <br/>
 赖小民被开除中共党籍及公职，其涉嫌犯罪问题移送检察机关审查起诉。
 <br/>
 <br/>
 中国华融是一家强大的中共国有借贷机构。领导华融近十年的赖小民4月起被调查。财新网8月10日报导说，华融集团8月3、4日召开会议，通报了赖小民的部分案情。
 <br/>
 <br/>
 通报称，办案人员在赖小民的几处房产里，搜出的外币现金以人民币计算，共计2.7亿元，相当于约3吨重，放在一起，超过3个立方米，但这还不是其腐败案的全部涉案财产。
 <br/>
 <br/>
 根据中纪委的通报，赖小民〝政治问题与经济问题相互交织〞，包括搞政治投机，搞美化宣传个人，捞取政治资本；任人唯权、任人唯利、任人唯圈。
 <br/>
 <br/>
 谈到赖小民的〝朋友圈〞，最引人关注的是台湾知名女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴佩慈.html" target="_blank">
  吴佩慈
 </a>
 的男友纪晓波，以及大陆知名女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵薇.html" target="_blank">
  赵薇
 </a>
 的丈夫黄有龙。
 <br/>
 <br/>
 2013年4月，华融首家境外平台子公司——中国华融国际控股有限公司在香港成立。赖小民早年频繁出没香港，与一些大陆在港的私企老板们关系甚密，且被称为香港的〝财神爷〞。
 <br/>
 <br/>
 2015年3月，〝华融系〞旗下华融金控正式借壳天行国际。天行国际此前从事大陆客户放贷、典当贷款和融资租赁等业务，纪晓波曾是这一上市公司的执行董事及行政总裁。
 <br/>
 <br/>
 但〝华融系〞入主不久，纪晓波持股公司便退出天行国际股东层面，纪的另一个身份是台湾女星吴佩慈的男友。据悉，纪和赖小民相熟，纪为东北人是〝香港富商〞，有媒体称其是神秘富豪〝车峰〞马仔。
 <br/>
 <br/>
 此外，赵薇的丈夫黄有龙也曾被指与赖小民关系密切。陆媒《稜镜》曾报导，顺龙控股的大股东黄有龙去年4月24日将其持有的全部67.5%股份，向华融投资做了股票质押融资。
 <br/>
 <br/>
 去年4月1日，黄有龙和赵薇夫妇就因空壳收购万家文化被立案调查。赖小民及华融却仍在同年4月24日继续为黄有龙提供融资服务。
 <br/>
 <br/>
 此外，赖小民还被指与多家企业存在着利益输送关系。
 <br/>
 <br/>
 陆媒已曝光宁夏天元锰业集团董事长、宁夏首富贾天将也卷入赖小民案。赖小民与明天系掌门人肖建华、华信董事会主席叶简明，以及重庆前市委书记孙政才的关系也浮出水面。
 <br/>
 <br/>
 现年56岁的赖小民曾长期在中共央行工作，曾任央行银行监管二司副司长、银监会办公厅主任，2009年起担任华融总裁、董事长。
 <br/>
 <br/>
 业界评价赖小民〝胆子大得惊人〞〝好大喜功〞，自他掌管华融后，将企业从谨慎变得激进。
 <br/>
 <br/>
 据披露，赖小民已经暗中离婚，其前妻原在光大银行总行任信贷管理部总经理，现已退休，而其在香港有一对双胞胎幼子。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
