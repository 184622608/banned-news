### 冯小刚为李咏发悼文 现身上海脸色〝惨白〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】受娱乐圈丑闻牵扯、许久未露面的中国知名导演
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冯小刚.html" target="_blank">
  冯小刚
 </a>
 ，最近发文悼念前央视主持人李咏。又现身上海温哥华电影学院，与高校师生互动。官微发布的照片显示，冯小刚脸色〝惨白〞。
 <br/>
 <br/>
 11月1日，实名认证的〝中国电影导演协会〞官微发布消息称，中国知名导演
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冯小刚.html" target="_blank">
  冯小刚
 </a>
 ，突然现身上海温哥华电影学院，他以〝从《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集结号.html" target="_blank">
  集结号
 </a>
 》出发〞为题，为学生们上了一堂导演课。
 <br/>
 <br/>
 冯小刚表示他在拍摄《天下无贼》时期，创作上已经有所变化，而正是因为这部电影打消了转型的疑虑，让他找回拍电影的初心才有了之后的《集结号》。
 <br/>
 <br/>
 当被问及导演如何给演员信任感时，冯小刚表示：〝最重要的是自尊心，当人一自信就会有不一样的光彩。他自己从来不对演员发脾气。〞
 <br/>
 <br/>
 从官微发布的照片可见，冯小刚脸色被灯光照得〝惨白〞，脸上不见一丝笑容。
 <br/>
 <br/>
 在此之前，许久未露面的冯小刚罕见发贴，对前央视主持人李咏去世表达痛心。冯小刚还安慰李咏的太太说，哈文辛苦了，送走咏哥好好歇歇吧！
 <br/>
 <br/>
 近半年来，冯小刚的日子似乎并不好过，因拍摄电影《手机》一事得罪了央视前主持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 ，受崔永元爆料影响，冯小刚被指连夜补缴百万管理费。
 <br/>
 <br/>
 7月中旬，冯小刚在微博发布〝十问〞文章，反击崔永元的爆料，但始终未露面。
 <br/>
 <br/>
 中国知名女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 事件似乎也牵扯到冯小刚。9月9日，一段据称是中共司法部主管的京衡律师事务所内部交流群的微信截图显示：范冰冰关押在上海某看守所。冯小刚逃亡美国，也在抓捕过程中。
 <br/>
 <br/>
 当晚19时44分，冯小刚在微博怒斥：〝所有处心积虑给我造谣，无中生有，颠倒黑白，混淆是非，恶意中伤的公号和平台，你们已经不仅是丧心病狂了，你们已经到了彻底藐视法律保护公民名誉权的程度。〞
 <br/>
 <br/>
 冯小刚还警告说：〝不要煽动和绑架民间的戾气，拿影视行业当出气筒，这很卑鄙。我没有所谓的阴阳合同，更没有偷逃税。我所在的公司美拉传媒每年纳税几千万人民币。〞
 <br/>
 <br/>
 11月2日，崔永元又在〝今日头条〞发文称，范冰冰没事了，疑似证实范已经缴纳巨额罚款，后续就等在什么时候覆出。
 <br/>
 <br/>
 崔永元还称〝冯小刚回来了〞，〝我们很弱小，我们只有证据，一个一个抛，就够了。〞疑似证实范冰冰和冯小刚还有麻烦。
 <br/>
 <br/>
 范冰冰由于卷入崔永元爆料娱乐圈逃税丑闻〝消失〞4个月后，10月3日中共税务部门称，范冰冰涉嫌偷税漏税，缴纳8.8亿元人民币的罚金可免予刑责。
 <br/>
 <br/>
 当时就评论人士分析说，一个范冰冰就收回近9亿元，数百个范冰冰（包括富商等人）又如何？这形同勒索。评论人士认为，本次由崔永元担纲的这场娱乐大戏的主角，其前台幕后应该都是中共高层政军之间的角力。
 <br/>
 <br/>
 再加之近期正值贸易战升级、中共经济陷入困境、内忧外患之际，中共突拿油水丰厚的娱乐圈〝宰肥羊〞、〝割韭菜〞，大搞建政后的〝第二次公私合营〞，是将民营财产充公的讯号。预料中国未来震荡不断。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：赵云）
