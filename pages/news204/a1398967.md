### 刘强东性侵案后 〝最担心的事〞再添一桩
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】近日，深陷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵.html" target="_blank">
  性侵
 </a>
 丑闻的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东集团.html" target="_blank">
  京东集团
 </a>
 创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 ，最担心的事再添一桩。陆媒称，刘去年曾宣布的百万
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_便利店.html" target="_blank">
  便利店
 </a>
 计划，如今因其丑闻缠身，该计划并未达成且又迎来重量级对手。目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴.html" target="_blank">
  阿里巴巴
 </a>
 已正式杀入连锁便利店领域。
 <br/>
 <br/>
 陆媒近日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_互联网.html" target="_blank">
  互联网
 </a>
 巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴.html" target="_blank">
  阿里巴巴
 </a>
 出重金收购
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_便利店.html" target="_blank">
  便利店
 </a>
 品牌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_喜士多.html" target="_blank">
  喜士多
 </a>
 20%至25%的股权。 对于上述传闻，阿里巴巴方面11日回应说，对市场传言不做评论。
 <br/>
 <br/>
 喜士多是台湾润泰集团在内地成立的连锁便利店，直营及加盟店总计接近1,000间，而润泰集团是超市大润发的母公司。
 <br/>
 <br/>
 阿里巴巴回应陆媒查询时表示，不评论市场传闻。报导说，阿里巴巴继拿下超市大润发、联发超市等后，已经正式杀入连锁便利店领域。
 <br/>
 <br/>
 多个自媒体分析认为，马云此举成为电商平台
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东集团.html" target="_blank">
  京东集团
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 〝最担心的事〞。2017年刘强东曾宣布百万便利店计划。但如今因深陷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵.html" target="_blank">
  性侵
 </a>
 丑闻，该计划被搁置且又迎来重量级对手。
 <br/>
 <br/>
 11月7日，第五届世界网际网路大会在浙江乌镇开幕，刘强东再一次缺席如此重要场合，而且还遭到参会大佬调侃，说主办方为了保护与会的美女引导员。
 <br/>
 <br/>
 去年的世界互联网大会上，刘强东不仅露面，还公开炮轰马云〝为富不仁〞。如今刘强东因在美国涉性侵案，在中国舆论场上失声60多天。海外舆论分析，刘强东因在美涉性侵案丑闻发酵，被北京〝抛弃〞。
 <br/>
 <br/>
 根据美国警方9月4日公布的报告，刘强东在美性侵案属于〝一级性犯罪：强奸既遂〞（CSCR-1,CSC-RAPE-COMPLETE）。明尼苏达亨内平县警方9月21日宣布，已完成对案件的初步调查，接下来将由检方接手，审查案件全部证据，并最终决定是否起诉。
 <br/>
 <br/>
 根据当地法律，刘强东所涉的一级强暴罪被定罪后，刑期至少12年起。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：祝馨睿）
