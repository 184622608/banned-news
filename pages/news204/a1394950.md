### 羊年生人灭中共？ 传江泽民秘访高僧惊闻天机
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】尽管中共宣传无神论，但中共前党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 自知罪孽深重，被曝到处拜各路高人给他出招儿。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九华山高僧.html" target="_blank">
   九华山高僧
  </a>
  ：
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共亡.html" target="_blank">
   中共亡
  </a>
  于
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_羊年生人.html" target="_blank">
   羊年生人
  </a>
 </h4>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 是踩着〝六四〞学生的鲜血上台的，上台后又犯下了残酷迫害修炼〝真善忍〞法轮功学员的罪行，香港杂志曾引述北京消息人士透露，江自知罪孽深重，深恐难逃千层地狱的惩罚，退下来之后赶着在家抄《地藏经》，并到九华山去拜地藏菩萨。但是，所有这些都无法得到神的佑护，挽回他被清算和失势的命运。
 <br/>
 <br/>
 前大陆贵州记者曾节明2017年5月15日在一篇回忆文章中谈到∶〝1998年3月，他以桂林电视台新闻记者的身份参加一次官员的考察，在酒宴上、乘着酒兴，一位省级高官的秘书竟洩露‘国家机密’，大谈江泽民信佛教、到处求神拜佛的事。
 <br/>
 <br/>
 他说，江泽民几乎每年都要去名山古刹拜佛，去年（1997年）邓小平死，老贼终于多年媳妇熬成婆，大权独揽成真，高兴得涕泪横流，快感之余，专程秘访九华山，让某高僧测一测自己的帝运，及中共的命运，看看自己能滋润多久。
 <br/>
 <br/>
 结果高僧只说破一点∶中共亡于羊年生人之手。
 <br/>
 <br/>
 文章说，老贼闻言大惊，认为〝羊年生人〞即为1919年生人（土羊）赵紫阳，而深忌之，从此大大强化了对赵紫阳的软禁，严防赵紫阳东山再起。
 <br/>
 <br/>
 作者说：〝邓小平活着时，还允许软禁中的赵紫阳，在专人陪同下，到全国各地旅游散心、打打高尔夫球，并允许地方领导接待和陪同，只是严禁报导。但江泽民从九华山回来后，即下令不准赵紫阳离开北京，要打高尔夫球只能在北京打；且老贼还下令在赵紫阳的住宅——北京富强衚衕6号安装了铁丝电网，加派了武警防守。〞
 <br/>
 <br/>
 文章说，有道是〝人算不如天算〞，显然，灭亡中共国的人羊年人并非耄耋之年的赵紫阳，而是另有其人，江泽民是枉费心机。
 <br/>
 <br/>
 奇特的是，从历史来看，共产党政权的危亡，的确与羊年人有不解之缘：比如，羊年生人赵紫阳，在1989年差一点就亡了中共；而苏共及几乎整个东欧共产党阵营，实亡于戈尔巴乔夫之手，戈尔巴乔夫和叶利钦都是羊年生人（1931年出生，金羊）。
 <br/>
 <br/>
 <h4>
  现任中共高官中属羊的
 </h4>
 <br/>
 在中共国家高层官员中属羊的有李克强（1955年7月）、王沪宁（1955年10月）、汪洋（1955年3月），以及中共现任检察长曹建民、北京市委书记蔡奇也是1955年的羊。
 <br/>
 <br/>
 另外，中共发改委主任何立峰、工业和信息化部长苗圩、商务部长钟山、文化部长雒树刚、审计署审计长胡泽君俱为1955年羊年生人。而在中共军队中东部战区政委郑卫平、西部战区司令员赵宗岐、北部战区政委范骁骏、海军政委苗华、火箭军政委王家胜、武警司令王宁也是1955年羊年人。
 <br/>
 <br/>
 再看、中共国家省委书记中羊年生人有吉林省委书记巴音朝鲁、浙江省委书记车俊、湖南省委书记杜家毫、海南省委书记刘锡贵、新疆书记陈全国、他们都生于1955年羊年。此外还有两名省长，黑龙江省长陆昊、福建省长于伟国生在1955年羊年。
 <br/>
 <br/>
 <h4>
  天要灭中共谁都挡不住
 </h4>
 <br/>
 事实上，不一定是这些人推翻中共，再往后推，属羊的人很多。中共作恶太多，几乎每个有良知的中国人都想中共早日解体。早在15年前，结束中共暴政的许多预言一直流传民间。
 <br/>
 <br/>
 2002年6月，贵州省平塘县掌布乡发现一块距今2亿7000万年的〝藏字石〞。500年前崩裂的巨石断面内惊现六个排列整齐的大字〝中国共产党亡〞。
 <br/>
 <br/>
 2003年12月5日至8日，中科院专家组对〝藏字石〞进行了实地考察，结论是：〝藏字石〞上未发现人工雕凿及其他人为加工痕迹，堪称世界奇观。
 <br/>
 <br/>
 从2004年12月3日起，在《大纪元》退党网站上，自愿公开声明退出中共党、团、队的人数，截至2017年12月1日已有2亿9080万4326人。贵州〝藏字石〞的出现，体现了天意和民心，预示着中共即将覆灭。
 <br/>
 <br/>
 中国共产党将亡，然而中国不会亡，神佑炎黄子孙，神佑大中华，届时神州大地将迎来崭新的一页。
 <br/>
 <br/>
 ——转自《新纪元》
