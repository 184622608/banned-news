### 【禁闻】珠海航展曝抄袭丑闻 中共为何一抄再抄
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】两年一届的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海航展.html" target="_blank">
  珠海航展
 </a>
 目前正在举行。俄罗斯媒体刊文，揭露中方展示的中俄合资CR929客机，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抄袭.html" target="_blank">
  抄袭
 </a>
 了一家俄罗斯设计局的方案。
 <br/>
 <br/>
 第12届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海航展.html" target="_blank">
  珠海航展
 </a>
 于11月6号至11号举行。不少飞机与地面装备集中亮相。其中包括俄罗斯联合航空公司（PJSC UAC）和中国商飞公司(COMAC) ，联合研制的民用CR929远程宽体客机的全尺寸模型。
 <br/>
 <br/>
 这是CR929客机首次对外展出，因此备受关注。此前这一合作项目一直被高调宣传，希望有朝一日能打破欧洲〝空中客车〞和美国波音公司双垄断格局。
 <br/>
 <br/>
 不过，俄罗斯《观点报》7号刊文，指CR929客机涉嫌剽窃。
 <br/>
 <br/>
 文章说，俄罗斯一家名为〝Atom〞的设计局发表声明说，CR929客机剽窃了他们的设计方案。他们曾于2015年，在上海，向招标CR929客机的单位，提供了自己的设计方案，后来没有被采用，但想不到，设计方案被中方偷窃、使用。
 <br/>
 <br/>
 〝Atom〞设计局的总经理兼总工程师库津（Maxim Kuzin）激动表示：〝对这种无耻的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抄袭.html" target="_blank">
  抄袭
 </a>
 行为感到非常震惊。〞不过他表示，不会为此采取法律措施，因为没有用。
 <br/>
 <br/>
 俄罗斯媒体也说，追究剽窃设计方案的法律责任非常困难，只要改变一些斜率、边角，就可以说是自己的新设计。
 <br/>
 <br/>
 时事评论员田园博士表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 从建政开始，很多重工业产品都是从苏联抄袭过来，进行逆向工程，然后在中国生产出来。
 <br/>
 <br/>
 时事评论员田园博士：〝这就是中共发家时候的秘密之一。它为什么能够变成所谓的重工业。俄国在几个方面仍然是远远超越于中国。比如说在火箭发动机、飞机发动机，还有航空电子部件上面。所以中共对俄国的抄袭是不会停止的。〞
 <br/>
 <br/>
 至于中方一再抄袭的原因，田园认为，首先是中共非常急功近利，它根本不愿投资，也不愿潜下心来去做基础研究。
 <br/>
 <br/>
 田园：〝基础研究需要花费非常长时间，像航空的发动机，火箭发动机，还有航空电子部件，这都需要有非常强的基础研究，和对这些部件的基本理解，才能够支撑这样一个比较庞大的产业，像发动机它是由几千个部件构成的，这种东西它不是一朝一夕能学得来的。〞
 <br/>
 <br/>
 田园表示，第二个原因，中共搞的是举国科技体制，如果要搞什么项目攻关，它会疯狂投入，完全不顾人民死活。就像当年搞原子弹和氢弹等。而这种举国体制，就把私有行业，和个人的创造力完全扼杀了。
 <br/>
 <br/>
 田园：〝大家知道在美国搞航空发动机、火箭发动机最厉害的是谁？是私营公司。比如说像波音、雷神、洛克希德马丁。这三个例子全部都是私有公司，没有一个说是美国国营的机构研发出来的。〞
 <br/>
 <br/>
 无独有偶，日本《产经新闻》6号报导，珠海航展上展示的中国最新款隐身无人机〝彩虹-7〞模型，外观也涉嫌剽窃美国的〝X-47B〞试验机。
 <br/>
 <br/>
 而在上个月，江苏国安厅副处长徐彦钧（Yanjun Xu，音译）被引渡到美国受审，以及美国司法部群体起诉的10个中国人，都因涉嫌窃取美国的航空技术。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝飞机这种高科技产品，试错的成本是很高的，人力、财力、时间的投入都非常惊人。就算当局真的是希望能自主研发，在中共的制度环境下，如果不能按期望的进度、在有限的经费投入下完成研发，那主事者可能就会有麻烦。换个角度讲，一个形成了山寨文化的国度，你要它老老实实的从头研发，那是很难的。〞
 <br/>
 <br/>
 这次航展，中方展示的是CR929客机的前段机身全尺寸样品。而该客机的发动机仍未确定。
 <br/>
 <br/>
 《经济学人》此前报导，中共国产飞机翔凤ARJ21和C919项目相继受挫，便企图窃取外国航空航天业的商业机密。而中共的下一个目标是制造自己的飞机部件，而不只是机身。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/李沛灵
