### 广东伤残军人陈风强遭跨区绑架
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】家住广州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伤残军人.html" target="_blank">
  伤残军人
 </a>
 陈风强，10月20号清晨，遭到至少7、8名不明身份的人，闯入家中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绑架.html" target="_blank">
  绑架
 </a>
 。家属在当地派出所得知是珠海金湾区金海岸派出所跨区绑架，但遭到该派出所的否认。 陈风强失踪已超过两天，家属呼吁外界关注。
 <br/>
 <br/>
 20号清晨，租住在广州市花都区赤坭镇乌石村的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伤残军人.html" target="_blank">
  伤残军人
 </a>
 陈风强遭到暴力
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绑架.html" target="_blank">
  绑架
 </a>
 ，他的太太22号向《新唐人》描述当时的情景。
 <br/>
 <br/>
 陈风强的太太郭凤珍 :〝20号早上天没亮，5点10分至20分左右，不知道是哪里的人敲门，我老公陈风强去开门，7、8个人以上跑上来2楼，楼下的门就搞坏了，楼下有两台面包车。什么话都没说，就把他拉走了，把我反锁在里面。〞
 <br/>
 <br/>
 郭凤珍得知是珠海金海岸派出所跨区绑架陈风强。陈风强的二哥陈凤明，21号上午到金海岸派出所找人，但该所否认绑架陈风强。陈凤明打电话给广州市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安.html" target="_blank">
  公安
 </a>
 ，确认该所绑人。下午又到该所找人，同样遭到否认。
 <br/>
 <br/>
 郭凤珍:〝我去当地的赤坭派出所，说是珠海金海岸派出所带走了。亲哥哥在珠海嘛，我是叫他们去金海岸派出所报案找人嘛 ，金海岸派出所的人说他们没参与，但是这边赤坭派出所登记协助他们上来抓人。〞
 <br/>
 <br/>
 陈风强在金海岸的八层楼房，2006年被珠海市金湾区城管以违法为由强行拆除了5层半而上访控告。当局两次将陈风强构陷入狱，每逢所谓的敏感时期，都要被限制人身自由。陈风强的哥哥陈凤明表示，因为维权陈风强多次遭到绑架。
 <br/>
 <br/>
 陈风强的哥哥陈凤明 :〝(珠海金海岸派出所)他说没有啊(抓人)，以前一样一样有(绑架)。〞
 <br/>
 <br/>
 据《维权网》报导，陈风强被绑架估计跟珠港澳大桥24号的剪彩活动维稳有关。 18号晚上，陈风强户籍所在地的珠海市金湾区三竈镇综治办来电话，要陈风强在23号协商非法强拆的赔偿事宜，这是当地政府敏感时期常用的维稳招术，陈风强没有理会，想不到隔天就被绑架。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
