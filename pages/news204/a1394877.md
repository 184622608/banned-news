### 【禁闻】美报告：中共黑客复出 网攻超越俄罗斯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】根据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络安全.html" target="_blank">
  网络安全
 </a>
 公司公布的最新报告，中共开始加大力度偷窃美国商业机密。报告指出中国现在已经取代
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 ，成为对西方国家进行网路攻击的最大国家。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络安全.html" target="_blank">
  网络安全
 </a>
 公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警告.html" target="_blank">
  警告
 </a>
 ，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共黑客.html" target="_blank">
  中共黑客
 </a>
 又回来了〞。
 <br/>
 <br/>
 美国网络安全公司CrowdStrike Inc 10月9号公布最新报告，过去半年，中共加大偷窃美国商业机密，报告并指出中国现在已经取代
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 ，成为对西方国家进行网路攻击的最大国家。
 <br/>
 <br/>
 报告指出，中共黑客主要针对美国和西欧国家的生物科技、国防、采矿业、医药、职业服务和运输企业等方面。
 <br/>
 <br/>
 旅美时事评论员蓝述：〝中共可能意识到，以后面对美国越来越严厉的制裁和网络安全的封锁，所以它现在很可能窃取美国高科技产业技术的最后一次好机会，以后的情况会越来越困难，所以他们在中美贸易战打的正热络的时候，他们就开始进行大量的技术窃取。〞
 <br/>
 <br/>
 CrowdStrike公司因为揭露俄罗斯黑客入侵美国民主党全国委员会网站，而声名大噪。
 <br/>
 <br/>
 CrowdStrike也曾经揭露中共利用一只代号为61398的上海部队，窃取美国公司的商业机密，最后美国司法部在2014年，宣布起诉被公布的五名中国公民，罪名为美国经济间谍罪。
 <br/>
 <br/>
 如今该公司又再次提出警告，公司技术长阿尔佩洛维奇在华府一场安全会议上表示，中共黑客回来了。他说：〝我们过去一年半看到黑客活动大幅增加。目前，他们是我们所看到最主要的威胁方，威胁美国和西欧各个机构。〞
 <br/>
 <br/>
 根据这份报告，在中共对军队进行重组，将很多黑客行动移交给合同公司后，中国成为更大的威胁，因为这些实际上是将行动私营化。报告说，这些合同公司摆脱了先前政府的官僚管理，由与黑客论坛和组织有着相当联系的电脑科学专家在管理。
 <br/>
 <br/>
 蓝述：〝哪有私营化？那些黑客原来是军方的那些人，跑到外面，把军装脱下来换上便衣，然后去注册一个公司，其它的组织、财务、方方面面的东西都不会变的，但是万一被抓住了以后，它就说这些跟中共的政权没有关系。〞
 <br/>
 <br/>
 《美国之音》指出，中共窃取美国商业机密在奥巴马执政末期有稍微停止，不过，从今年以来又开始加大了窃取机密的力度。
 <br/>
 <br/>
 旅美时事评论员邢天行：〝习近平作为政府执政人来讲，它是对于美国政府是采取了一种友好的态度，在奥巴马的后期，既给了奥巴马的面子，同时，他对于新上任的川普政府，或者那个时候他还没有预期到川普要上台，也就他对于新的美国政府，他是采取了一个审慎的，表示友好的这样一个态度，所以这段时间，他就给了他相当的一个面子，就是黑客攻击这些事情，就暂时的减缓下来。〞
 <br/>
 <br/>
 报导指出，随着川普政府对它所认为的中国不公平贸易手段的打击越来越强硬，与中共政府有联系的黑客组织对美国企业知识产权的盗窃又在上升。
 <br/>
 <br/>
 旅美时事评论员邢天行认为，中美贸易战中，中方一边采用拖延战术，另一方面又恢复本性，开始肆无忌惮的进行黑客盗窃攻击行为。
 <br/>
 <br/>
 邢天行：〝这种网络战争对于它(中共)来讲，它认为是一种爱国的行为，这就是中共党文化洗脑之后，他们对于这个世界价值观颠倒的认识。所以美国这边，无论是如何谴责它盗窃美国知识产权，盗窃美国大量的信息，这都是可耻的行为，但是对中共这些黑客来讲，这些批评和指责是不起作用的。〞
 <br/>
 <br/>
 10月4号《路透社》报导，一个叫做〝云端跳跃〞的黑客组织，涉嫌从事网络间谍和知识产权盗窃活动。该组织被西方网络安全公司认为与中共政府有联系。
 <br/>
 <br/>
 采访/常春 编辑/黄亿美 后制/陈建铭
