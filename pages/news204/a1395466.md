### 家藏3吨现金 华融前董事长赖小民刷新贪官记录
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】中国华融资产管理股份有限公司（以下简称〝华融〞）前董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 被〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双开.html" target="_blank">
  双开
 </a>
 〞，案件已移送检察机关审查起诉。官方查处赖小民家藏巨额现金2.7亿元（人民币，下同），重达3吨，再破中共贪官藏现钞记录。
 <br/>
 <br/>
 据中纪委网站10月15日最新通报，华融前党委书记、董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 被立案审查。
 <br/>
 <br/>
 通报指，赖小民违反政治纪律和政治规矩，盲目扩张、无序经营导致公司严重偏离主责主业，搞政治投机，为个人职务升迁拉关系，搞美化宣传个人，捞取政治资本，对抗审查。
 <br/>
 <br/>
 通报指控赖小民挥霍浪费国家财产、组织公款宴请、安排或接受下属单位公款接待亲属旅游，收受礼品、礼金，利用职权或职务影响为亲友经营活动谋利，为他人谋取利益并收受巨额财物涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受贿.html" target="_blank">
  受贿
 </a>
 犯罪；利用职务上的便利，非法占有公共财物涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贪污.html" target="_blank">
  贪污
 </a>
 犯罪；此外，赖小民与多名女性搞权色交易。
 <br/>
 <br/>
 综上，赖小民被开除中共党籍及公职；其涉嫌犯罪问题已移送检察机关审查起诉。
 <br/>
 <br/>
 今年4月17日赖小民落马。该案被指是中纪委和国家监察委合署办公后宣布的第一个金融大案。前几年，有关赖小民被查的消息就不断传出，今年才被坐实。
 <br/>
 <br/>
 办案人员在赖小民的几处房产里，搜出多种外币现金等，折合人民币总值约2.7亿元。这些现钞堆积在一起后，体积超过3立方米，总重量则高达约3吨重。而这些只是赖小民腐败案全部涉案财产的冰山一角。
 <br/>
 <br/>
 赖小民私藏现金2.7亿元，刷新了2014年4月前能源局煤炭司副司长魏鹏远被查后，在家被发现2.3亿元现金、当场烧坏4台点钞机的纪录。
 <br/>
 <br/>
 赖小民曾长期在中共央行工作，曾任央行银行监管二司副司长、银监会办公厅主任，2009年起担任华融总裁、董事长。
 <br/>
 <br/>
 业界评价赖小民〝胆子大得惊人〞、〝好大喜功〞，其被指与多家企业存在着利益输送关系。陆媒已曝光宁夏天元锰业集团董事长、宁夏首富贾天将，也卷入赖小民案。
 <br/>
 <br/>
 赖小民与贾天将是同学。天元锰业正是华融近年来大力扶持的一家民企。
 <br/>
 <br/>
 财新网此前报导，赖小民在任职董事长期间，不顾公司内外质疑，连年向天元锰业大量输血，金额高达数百亿元，将其扶植为华融的〝影子公司〞，也实现赖小民所需要的利益输送和安排。
 <br/>
 <br/>
 此外，赖小民与明天系掌门人肖建华、华信董事会主席叶简明、以及已落马的重庆前市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙政才.html" target="_blank">
  孙政才
 </a>
 的关系也浮出水面。赖小民与孙政才的合照也被曝光。
