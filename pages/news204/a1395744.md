### 延安整风再现贵州？全省教师人人交代〝不当言论〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】中共当年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_延安整风.html" target="_blank">
  延安整风
 </a>
 ，要求人人自我揭发，开启了恐怖的整人模式，被称为文革预演。日前，网传贵州省当局文件，要求全省教职工报告2012年以来的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不当言论.html" target="_blank">
  不当言论
 </a>
 〞，被指整风运动再现。
 <br/>
 <br/>
 10月14日，有推特网友曝光一张〝贵州省教职员工言论行为自查报告表〞。报告表显示，该省教师和学校职工被要求上报2012年以来在网络上、著作或演讲中曾经发表过的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_不当言论.html" target="_blank">
  不当言论
 </a>
 〞。
 <br/>
 <br/>
 虽然从报告表来看，这种上报是采取〝自觉自愿〞的方式，但在经历过历次政治运动的中国，势必引发人们心中深埋的恐惧。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-17/p9084421a889570745.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-17/p9084421a889570745-ss.jpg"/>
   <br/>
   <font size="-1">
    网传贵州省教职工自查表。（推特图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 爆料的推友评论称，这张报告表令人联想起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_延安整风.html" target="_blank">
  延安整风
 </a>
 时，窑洞里每天深夜传出的鬼哭狼嚎，让人毛骨悚然。
 <br/>
 <br/>
 推文还指，贵州当局如此倒行逆施，证明中共已是秋后的蚂蚱。
 <br/>
 <br/>
 有推友跟帖评论，〝时过境迁，物极必反！〞
 <br/>
 <br/>
 目前尚未看到有媒体报导证实此事，暂未确定这只是贵州省当局所为，还是全国性的行动。
 <br/>
 <br/>
 但有推友留言称，〝看到这条推的时候身边刚好坐了个教师朋友，证实是真的，各学校老师都被要求填写这个表格〞。
 <br/>
 <br/>
 目前中共经济陷入困境，贸易战更是雪上加霜。中共政权内忧外患，风雨飘摇。中共正在加紧收紧言论，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_箝制思想.html" target="_blank">
  箝制思想
 </a>
 ，防止意识形态崩溃。
 <br/>
 <br/>
 其中，负责塑造下一代思想灵魂的教师和教育系统职员，成为中共压制的首要目标。
 <br/>
 <br/>
 日前外媒报导，中共开始严管全国中小学教师的护照，限制其出国。外界分析，中共害怕教师出国后，带回人类普世价值观念，影响中国下一代。
 <br/>
 <br/>
 另外，近期多有大学教授因为发表与北京向左的言论遭到打压或解职。其中非常典型的是浙江传媒学院文学院副院长赵思运，因为在2018级新生开学典礼致辞遭严重警告处分。
 <br/>
 <br/>
 外媒研究其致辞，诧异地发现，全文没有涉及任何政治内容，只是要求学生独立思考，拥有独立人格，具备自律、尊重与尊严的素养，要有道义担当。
 <br/>
 <br/>
 外界认为，赵思运要求学生具备的这些品质，正是中共维护暴政统治必须要打压的对象。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
