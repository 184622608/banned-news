### 贸易战中渔翁得利 巴西还向WTO投诉中共
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】世界贸易组织（WTO）22日公布的文件显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 已经就北京的食糖进口限制向WTO提出申诉。由于中共向美国农产品加征关税，巴西成为大豆交易的最大受益者。
 <br/>
 <br/>
 路透社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西.html" target="_blank">
  巴西
 </a>
 在申诉文件中，对中共向进口食糖采取的限制措施、关税配额管理，以及配额以外的食糖〝自动进口许可〞提出异议。
 <br/>
 <br/>
 中共去年对进口食糖额外加征45%的关税，导致巴西食糖出口大幅下降。加征税率在今年5月已降至40%，明年5月将降到35%。
 <br/>
 <br/>
 世界贸易组织的协议允许成员国加征临时性关税，以应对进口突然意外飙升对本国生产商的损害，但实施这种措施需要满足一些条件。巴西的申诉文件称，中方这种额外关税并不符合WTO的协议。
 <br/>
 <br/>
 巴西指称，中共违反了WTO的12项关于保障措施的规则，5项关于配额措施的规则，以及13项关于许可制度的规则。
 <br/>
 <br/>
 因为中美贸易战，中共向美国大豆加征关税，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴西大豆.html" target="_blank">
  巴西大豆
 </a>
 出口大增，价格暴涨，巴西官方和农场主都在计划扩大种植规模。
 <br/>
 <br/>
 巴西是中国最大的大豆进口国，贸易战前美国位居第二。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
