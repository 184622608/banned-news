### 女子高歌庆离婚 儿子忧郁怼白眼（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】近日，一条女子在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离婚.html" target="_blank">
  离婚
 </a>
 庆典上高歌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热舞.html" target="_blank">
  热舞
 </a>
 的视频在网路流传。与大人不同的是，视频里一个男童神情忧郁，并且时不时翻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白眼.html" target="_blank">
  白眼
 </a>
 。不少网友表示父母离婚，〝可怜了孩子〞。
 <br/>
 <br/>
 10月22日，微博网友〝杀马特强子〞发布视频，视频中，一长发年轻女子左手拿着话筒，右手拿着酒瓶，一身V领黑衣，赤脚站在一张桌子上，随着音乐强劲的节奏，扭腰摆臀。女子边舞边喊：〝今天是我
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离婚.html" target="_blank">
  离婚
 </a>
 的第一天，我希望大家吃好喝好玩好，好不好？干杯！〞在座的男士们立刻举酒瓶回应：〝好!好！〞
 <br/>
 <br/>
 从视频可见，女子身后挂着一副对联，内容是〝离婚快乐〞、〝苦尽甘来〞，横幅是〝热烈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_庆祝.html" target="_blank">
  庆祝
 </a>
 离婚〞。
 <br/>
 <br/>
 不过，更吸引网友注意的是，在女子的右后方，一名男童呆坐，似是黑衣女子的儿子，男童在母亲高歌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热舞.html" target="_blank">
  热舞
 </a>
 之际，一直表情阴郁，还不时斜眼观看女子热舞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白眼.html" target="_blank">
  白眼
 </a>
 多，黑眼少。
 <br/>
 <br/>
 据网民爆料，该名女子多年来一直离婚不果，直至近日办完手续，且争取到儿子的抚养权。所以该女子在办完离婚手续后，立即联系朋友一同庆祝。
 <br/>
 <br/>
 视频发布后，有150多万网民观看。网友纷纷表示〝可怜的孩子〞。也有网民指，现代人家庭观念淡漠，婚姻缺乏责任感，过于放纵个人感情，才会发生这种庆祝离婚的怪事。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
