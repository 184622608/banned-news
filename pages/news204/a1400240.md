### 江泽民中南海丑态视频热传 曾因此遭港媒〝群殴〞（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】近日，中南海一段内部视频在海外网络热传。18年前，中南海发生一件事，导致香港媒体前所未有的〝群殴〞当时的中共党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 ，怒斥江〝心虚与傲慢〞。江的名言〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_图样图森破.html" target="_blank">
  图样图森破
 </a>
 〞和〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_闷声发大财.html" target="_blank">
  闷声发大财
 </a>
 〞也由此广为人知。
 <br/>
 <br/>
 2000年10月，正在北京访问的董建华，在中南海会见
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 ，双方在正式谈话前接受了在场的香港记者采访。
 <br/>
 <br/>
 当时，一名香港女记者连续追问〝中央钦定特首〞的敏感问题，江泽民大怒，现场大骂香港记者水平不够、太天真。〝Too Young，too simple，sometimes naïve。〞这一幕在香港新闻界造成巨大震撼。
 <br/>
 <br/>
 次日，香港媒体前所未有的齐声谴责江泽民，指责江〝心虚与傲慢〞。几乎所有报章都刊登了江泽民的丑态照片，并全文刊录江泽民和记者的问答。
 <br/>
 <br/>
 有报导称，在国际上，很少有国家领导人被媒体有机会记录下如此真实的情绪表达，这段经典问答，配上江泽民当时咆哮的面部表情，与张牙舞爪的手势，江的低级无聊再次让世界震惊。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 具有讽刺意味的是，江泽民当时曾对香港记者提及的美国大记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华莱士.html" target="_blank">
  华莱士
 </a>
 ，就曾当面直戳江的痛处。华莱士与江泽民在2000年有过一次专访，当时华莱士谈及的问题都特别敏感，像六四和法轮功，他都坦率直问。
 <br/>
 <br/>
 华莱士：〝你，对我而言，是一个独裁者，一个权力主义者。〞
 <br/>
 <br/>
 江泽民：〝我坦白说，我不同意你的观点。〞
 <br/>
 <br/>
 华莱士嘲讽地说：〝我知道你不同意。但美国俗语说，如果你走起来像只鸭子，叫起来像只鸭子，那你就是只鸭子。一个独裁者就是强行压制者，无论对象是新闻自由、宗教自由，或私人企业自由。你现在是有点接近了。〞
 <br/>
 <br/>
 华莱士被美国新闻界推崇为〝新闻拓荒者〞，他是《60分钟》节目的前王牌主持人，他的节目长期排名美国十大收视节目。华莱士于2012年4月7日去世，终年93岁。
 <br/>
 <br/>
 <h4>
  全球
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_起诉江泽民.html" target="_blank">
   起诉江泽民
  </a>
 </h4>
 <br/>
 虽然彼时的江泽民一副不可一世，嚣张跋扈的嘴脸，不过，他于1999年发起对法轮功修炼团体的迫害，导致他在全球范围内被起诉，从此胆战心惊，惶惶不可终日。
 <br/>
 <br/>
 《江泽民其人》第廿一章：〝全球起诉风云激荡法网恢恢无处可藏〞章节中记述，2002年10月，法轮功学员在美国芝加哥对江泽民提起诉讼案，令江非常恐惧。
 <br/>
 <br/>
 当时，江泽民正在访问芝加哥的前夕，这期间，法轮功学员成功地完成了递送传票的程序，把江泽民告上了美国法庭。自此，全世界都知道了江泽民对法轮功发动的群体灭绝和酷刑迫害。
 <br/>
 <br/>
 这起诉江案掀起了全球起诉江泽民的连锁反应，控告江泽民的诉讼案在世界各地此起彼伏。世界各国30多名律师一同起诉江泽民，这在世界的历史上、在法律史上还没有过先例。
 <br/>
 <br/>
 诉江案也给中南海带来巨大震动。中共利用外交途径向美国和其他国家透露〝江愿意付一切代价阻止此案成立〞，并要求美国动用〝元首豁免〞条款中止此案。
 <br/>
 <br/>
 据一位中共大使馆的官员私下透露，国际有关起诉江的报导和法庭开庭的资料，大使馆被要求立即报送北京，中共政治局常委们需要马上阅读。
 <br/>
 <br/>
 江泽民被起诉后，还曾通过美国的亲信试探法轮功口风，提出可以像文革一样，枪毙一些打死法轮功学员的警察来偿命，换取法轮功不起诉。江还说可以比文革处理得更严厉些，可以死多少法轮功学员就枪毙多少警察。
 <br/>
 <br/>
 （注：文革结束后，1976年打倒〝四人帮〞后发生过一场追查。那些迫害过中共高干及其子弟的军人受到内部清理，一批人被押解到云南秘密处决，对被清理的这些人的家属只是宣布〝因公殉职〞。当时的北京公安局局长刘传新则在追查开始之前就畏罪自杀了。）
 <br/>
 <br/>
 据悉，法轮功方面并没有答应江泽民的要求。他们认为，这场对法轮功的残酷迫害就是江泽民一意孤行发起的，那些劳教所的警察打人时，常说的一句话就是〝江泽民叫我干的〞。因此，一直坚持要将迫害法轮功的元凶江泽民绳之以法。
 <br/>
 <br/>
