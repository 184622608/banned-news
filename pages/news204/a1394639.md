### 范冰冰事件不简单 德媒揭中共另有目的
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 失踪4个月后，缴纳8.8亿元人民币罚款，免除牢狱之灾。德媒称，范冰冰事件绝非偷漏税这样简单，而是中共想通过此事发出警示信号，那就是在中国大陆成功成名的人士，一切荣誉和地位都是〝党〞暂时借的，随时都可能收回。
 <br/>
 <br/>
 德国《新苏黎世报》9月7日发表题为《借来的荣誉》一文称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 为了逃过牢狱之灾，不得不在年底前缴纳8.8亿元人民币的罚款，尽管金额巨大，但想必范冰冰是有能力缴清的。
 <br/>
 <br/>
 范冰冰自2004年开始，已连续14年打入福布斯中国名人收入榜，2013年起更是蝉联5年名单榜首。据福布斯的数据，范冰冰从2013年至2017年的总收入达9亿元。
 <br/>
 <br/>
 而这些款项只不过是范冰冰台面上看得到的进帐。范冰冰曾说：〝我从来没想过要嫁入豪门，因为我觉得我就是豪门。〞
 <br/>
 <br/>
 知名博客〝岩松随笔〞10月6日在微博爆料称，传言范冰冰已经凑够9亿元现金，并已把罚款交给了税务局。但该消息还未得到中共官方证实。
 <br/>
 <br/>
 德媒文章说，表面上看起来，范冰冰事件似乎已经尘埃落定，但事件并不简单。
 <br/>
 <br/>
 评论认为，共产党想通过范冰冰事件，发出一个警示信号，以便让所有人都明白，一切荣誉和地位都是党暂时借给你的，随时都可能收回。同时，中共领导层也想要通过此事件确立在艺术界的最高领导权。
 <br/>
 <br/>
 《德国之声》报导说，不管怎么说，范冰冰事件总算〝靴子〞落地了，也似乎证实了外界的分析，就是中共政府没钱了，要在娱乐圈这块〝韭菜地〞里〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_割韭菜.html" target="_blank">
  割韭菜
 </a>
 〞，在艺人身上〝薅羊毛〞。
 <br/>
 <br/>
 报导说，在中共的眼中，知名度很高的范冰冰就是一只〝大肥羊〞，拿她开刀〝祭旗〞，〝寒蝉效应〞会更有效，对整个娱乐圈都会形成一种震摄。至于中共说什么〝端正社会风气〞之说，那只不过是它的一块遮羞布而已。
 <br/>
 <br/>
 《苹果日报》评论文章说，中国影视界是习近平势力范围之外的领地，过去20多年，中国影视界被曾庆红家族操控，其弟曾庆淮是娱乐圈最重要的〝幕后大佬〞。可以说，整治娱乐圈只是时间问题。
 <br/>
 <br/>
 文章说，崔永元公开爆料范冰冰偷漏税这一事件，习当然不会错过，藉此整肃江派势力。令曾家等人交出控制权，已达到习的政治目的，暂时毋须将这个行业一棍打死。
 <br/>
 <br/>
 此外，范冰冰并非曾家圈子的核心人物，又是当红女星，她失踪期间就受到不少国际传媒关注，对她下手就是对娱乐圈起到足够的警告作用。
 <br/>
 <br/>
 文章最后称，一个范冰冰就收回近9亿元，数百个范冰冰（包括富商等人）又如何？这形同勒索。早年中共就是强行向地主富商〝募款〞、〝罚款〞、〝绑票〞等起家的！
 <br/>
 <br/>
 分析认为，由于近期正值贸易战升级、中共经济陷入困境、内忧外患之际，中共突拿油水丰厚的娱乐圈〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宰肥羊.html" target="_blank">
  宰肥羊
 </a>
 〞、〝割韭菜〞，大搞建政后的〝第二次公私合营〞，是将民营财产充公的讯号。预料中国未来震荡不断。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：赵云）
