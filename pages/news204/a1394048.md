### 中共严打讽刺性图片 〝秋语图说〞作者被刑拘
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】上海网民唐燕涛因制作传播具讽刺性时政图片，日前遭上海警方刑事拘留，而此前宝庆图文和尖刀图说作者也遭到打压。有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 指出，这是因为他们的作品触动了当权者的神经。
 <br/>
 <br/>
 这一张张利用时政性网络图片和文字编辑而成的〝秋雨图说〞，是上海网民唐燕涛的作品，由于简洁明了、讽刺幽默，很受网友欢迎。据维权网报导，唐燕涛网名唐唐，于9月29号被上海市公安局宝山分局刑事拘留，目前关押在宝山看守所。
 <br/>
 <br/>
 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 徐女士表示，除了唐燕涛外，其他稍有影响力的图说作者也遭到整肃，因为这些图片触动了当权者的神经。
 <br/>
 <br/>
 上海维权人士徐女士：〝她是因为做图片给关起来了，刑拘了，还有一个宝庆也抓起来了，也拘留了，我觉得是不应该的，这个图片也属于言论自由这一类的嘛，凭什么不能做啊，还有一个叫尖刀的，尖刀好像放出来了，他们触动了当官的、当权者的神经了，他们肯定觉得不舒服了嘛！〞
 <br/>
 <br/>
 不少维权人士对唐燕涛被刑拘一事表示愤慨。
 <br/>
 <br/>
 上海维权人士王先生：〝这不对的，言论自由，怎么可以打压她，公安是任意抓捕，百姓遭殃，司法现在腐败到极顶，我对这个表示同情，被抓。〞
 <br/>
 <br/>
 上海维权人士孙洪琴：〝中国是一个没有法的国度，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 抓人、关人、拘留、判刑，它不根据法律依据，它就根据政权的需要，凡是对政权有威胁，对制度不满、对官员违法犯罪行为不服，这样的人，它随时随地会抓的。〞
 <br/>
 <br/>
 〝尖刀图说〞作者梁勤辉向《自由亚洲》证实，唐燕涛被抓的同一天，他也遭到抄家和传唤，问询时间长达6个多小时。他指出，官方这次针对性的对影响力相对较大的图说作者下手，目的是想让所有人噤声。
 <br/>
 <br/>
 新唐人记者常春、陈洁、元真采访报导
 <br/>
 <br/>
</div>
