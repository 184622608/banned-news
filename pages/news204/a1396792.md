### 非洲猪瘟继续蔓延  贵州现疫情
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】中国大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情目前已从东北
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔓延.html" target="_blank">
  蔓延
 </a>
 至西南地区。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 官方周四（25日）通告，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贵州.html" target="_blank">
  贵州
 </a>
 省毕节市发现首例疫情。至此，非洲猪瘟已蔓延至中国13个省、直辖市，发病案例超过40起。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 农业农村部新闻办公室10月25日发布通知称，确认来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贵州.html" target="_blank">
  贵州
 </a>
 毕节市赫章县一养殖户送检的样品呈阳性，确诊为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 。该养殖户存栏生猪10头，其中发病8头，死亡8头。
 <br/>
 <br/>
 自8月初辽宁沈阳市发生第一起非洲猪瘟至今，中国已经爆发超过40起病例，疫情
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔓延.html" target="_blank">
  蔓延
 </a>
 到12个省份及直辖市天津。这些遭感染区域饲养猪只数量，几乎占据全国总数量的一半以上。
 <br/>
 <br/>
 目前仅知非洲猪瘟病毒可通过受污染的饲料或接触受感染的动物传播，染病猪只死亡率几乎100%，且没有疫苗。
 <br/>
 <br/>
 <h4>
  美国州农业局长：中国猪瘟疫情可能更严重
 </h4>
 <br/>
 中共农业部周三表示，将禁止给猪喂食餐厨剩余物，并声称在最先爆发的21次猪瘟中，有62%与厨余喂食有关。但北京网站上发表的一份声明显示，中共当局尚未说明猪瘟是如何爆发和蔓延的。
 <br/>
 <br/>
 根据美国农业部的报告，猪瘟病毒传播的真实规模和风险仍然未知。中国养猪行业由小型生产者和农场主导，生物安全措施不严格，更加大了蔓延的可能性。
 <br/>
 <br/>
 该疫情还可能影响全球的中国猪肉产品。曾担任美国内布拉斯加州农业局局长的伊巴赫（Gregory Ibach）认为，中国猪瘟的传播范围，可能比报导出来的要大得多。他对此表示十分担忧。
 <br/>
 <br/>
 〝我们一直在重新评估出货来自何处，特别关注来自从中国购买产品的国家，〞伊巴赫说，〝与此同时，我们需要各州合作伙伴的帮助。〞
 <br/>
 <br/>
 <h4>
  非洲猪瘟持续蔓延  中共极力
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_掩盖疫情.html" target="_blank">
   掩盖疫情
  </a>
 </h4>
 <br/>
 中国非洲猪瘟疫情自爆发后一路南下，近期继续向南、西方向蔓延。贵州省是第12个受到侵袭的省份。
 <br/>
 <br/>
 在中共官方通报之前，已有大陆网民向〝大纪元〞留言披露：〝山东省、河北省、贵州省已经发生非洲猪瘟疫情。〞
 <br/>
 <br/>
 若情况属实，到目前，在大陆22个省份中，辽宁、河南、黑龙江、内蒙古、吉林、山西、山东、河北、安徽、江苏、浙江、云南、湖南、贵州14个省份和天津直辖市，均出现非洲猪瘟疫情。
 <br/>
 <br/>
 仅剩，陕西、甘肃、青海、福建、湖北、广东、海南、四川，以及北京、上海、重庆三个直辖市，目前未被曝出有疫情。
 <br/>
 <br/>
 周二（23日），湖南省爆发疫情后，广东省禁止运输生猪。
 <br/>
 <br/>
 同一天，日本发现首宗非洲猪瘟病例。一名从北京飞抵北海道新千岁机场的旅客，其携带重约1.5公斤、真空包装的香肠，被检出含非洲猪瘟病毒。这种病毒可以在冷冻猪肉制品中存活至少三年。
 <br/>
 <br/>
 面对持续扩大的非洲猪瘟疫情，中共官方在通报每起疫情时，都称〝该起疫情已得到有效处置〞。
 <br/>
 <br/>
 有网民判断说：〝也许大陆所有的省市都出现了猪瘟，疫情严重不得不报的有十几个省。没有办法相信中共的新闻。〞
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
