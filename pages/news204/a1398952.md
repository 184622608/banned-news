### 【禁闻】上海进博会 中共雇外国人签单演戏
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】
 <b>
  上海
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
   进博会
  </a>
  中共雇
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外国人签单.html" target="_blank">
   外国人签单
  </a>
  演戏
 </b>
 <br/>
 <br/>
 中共在上海高调举办的〝中国国际进口博览会〞11月10号落幕，〝路透社〞报导说，许多在展览上公布的签单，都是演戏和旧单。
 <br/>
 <br/>
 报导说，在一次签约后，一位外国人士告诉〝路透社〞，他与他所代表的公司没有任何关系。
 <br/>
 <br/>
 这位不愿透露姓名的人士说：〝你在签约仪式上看到的一切都是演戏或闹剧。我本质上是一名演员，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 期间受雇，为了给人留下国际合作的印象。〞
 <br/>
 <br/>
 <b>
  网文抨击
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高考政审.html" target="_blank">
   高考政审
  </a>
  当局秒删 网络飞传
 </b>
 <br/>
 <br/>
 继重庆宣布2019年高考将采取政审制度之后，福建省11月9号也宣布，要对高考考生的政治态度和道德品德作出全面鉴定，不合格者不得参加高考。
 <br/>
 <br/>
 与此同时，网友〝红拂女〞写的一篇猛烈抨击高考政审的文章也在网上热传。
 <br/>
 <br/>
 这篇题为〝政审你大爷〞的文章说，〝这国很多政策荒谬绝伦〞，〝高考报名要政审？大学是你家开的吗？这国绝大多数大学都是公立大学，公立的意思是花纳税人的钱办教育。〞〝你们收税时从来不觉得我们道德品质恶劣，不要我们的肮脏钱，凭啥到了我们的孩子考大学的时候，就嫌弃我们的孩子道德品质恶劣，不让报名考大学了呢？〞
 <br/>
 <br/>
 文章质问：〝道德品质恶劣是怎么界定的？谁有资格来界定？〞〝你们自己的道德品质过关了吗？〞
 <br/>
 <br/>
 文章还指出：〝老以为老百姓的一切都是你们恩赐的，忘了你们的一切都是纳税人恩赐的。〞
 <br/>
 <br/>
 这篇文章一出来就立刻遭到当局秒删，不过仍然以各种形式在各个网络渠道飞传。
 <br/>
 <br/>
 <b>
  外媒曝光：
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
   孟宏伟
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职信.html" target="_blank">
   辞职信
  </a>
  为打印件无签名
 </b>
 <br/>
 <br/>
 国际刑警组织主席、中共公安部副部长孟宏伟，被中共政府失踪和调查后，曾向国际刑警组织递交过一封辞职信，国际刑警组织根据这封信同意了他的辞职。
 <br/>
 <br/>
 但是法国《世界报》11月9号报导说，这封辞职信是打印的，只有两行字，说他涉嫌违反法律，自愿辞去国际刑警组织主席的职务。不过辞职信上并没有孟宏伟的亲笔签名，他的名字是打印上去的。
 <br/>
 <br/>
 报导质疑信件是否出自孟宏伟之手。并表示，即使是，也无法证实他是不是自愿辞职。
 <br/>
 <br/>
 编辑/周玉林
