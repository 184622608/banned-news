### 中国知名摄影师卢广在新疆被带走
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】旅美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国知名摄影师.html" target="_blank">
  中国知名摄影师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢广.html" target="_blank">
  卢广
 </a>
 的妻子11月26号在推特上披露，卢广3号晚上与家人失去联系，随后得知被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆国保.html" target="_blank">
  新疆国保
 </a>
 带走，但她至今没有接到拘捕文件。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卢广.html" target="_blank">
  卢广
 </a>
 的妻子徐小莉11月26号开通推特帐号，她在推文上透露，10月23号，卢广受朋友之邀飞往新疆乌鲁木齐进行摄影指导和交流。10月31号，卢广到喀什旅行，11月3号回到乌鲁木齐，期间和家人一直有联系，但3号晚上开始失联。而卢广原定5号要与一位朋友前往四川大凉山做慈善活动，也没有出现。
 <br/>
 <br/>
 徐小莉说，她辗转得知，卢广与邀请他去新疆的朋友都被当地国保带到喀什，她向卢广户籍地浙江永康有关部门询问，有关部门也证实卢广被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆国保.html" target="_blank">
  新疆国保
 </a>
 带走。她说，卢广失联至今20多天，没有接到任何拘捕文件或通知，多次拨打新疆警方电话，也都没有接通。
 <br/>
 <br/>
 新唐人记者28号联系到徐小莉，但她没有接受采访。
 <br/>
 <br/>
 现年57岁的卢广，目前与家人定居美国纽约，他从1993年起从事纪实摄影，他的作品记录了包括中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生态污染.html" target="_blank">
  生态污染
 </a>
 、吸毒、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱滋病.html" target="_blank">
  爱滋病
 </a>
 等社会问题，曾获得〝尤金．史密斯人道主义纪实摄影奖〞等多个国际奖项。
 <br/>
 <br/>
 中国知名维权人士胡佳：〝我们做爱滋病方面的社会问题的揭露，也经常使用卢广拍摄的照片，因为有震撼力，跟他是朋友，我们做的事情都是充满风险的，因为他们往往不愿意我们去揭示社会的黑暗面，社会的疮疤。〞
 <br/>
 <br/>
 胡佳表示，卢广有可能去新疆拍摄他认为不可掩盖的社会问题而被抓。
 <br/>
 <br/>
 胡佳：〝新疆，我们现在都知道，焦点是法治培训班，其实就是集中营。当然那些地方你很难进去，但是即使拍摄的是路上三步一岗、五步一哨，走一个小时要停车查5次身分证这种类型的事情，那也是当局不可示人的。〞
 <br/>
 <br/>
 旅美中国艺术家吴玉仁是卢广的朋友，他向美国之音表示，因为拍摄爱滋病题材，卢广曾与当年的河南政法干部、当今的新疆一把手陈全国结下梁子，他很担心卢广的安危。他还说，徐小莉一度想回国营救丈夫，但在他的劝说下，打消了这个念头。
 <br/>
 <br/>
 新唐人记者陈汉、陈洁、施宜君采访报导
</div>
