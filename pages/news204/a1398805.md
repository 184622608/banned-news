### 川普顾问警告华尔街：不要插手美中贸谈
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】美国中期大选已降下帷幕，川普（特朗普）总统与习近平的G20会面，会使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 走向何方引发外界关注。日前，川普经济顾问彼得．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 在华府猛烈抨击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华尔街.html" target="_blank">
  华尔街
 </a>
 高管，称他们是与中共合作，想要左右川普的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_没有登记的外国代理人.html" target="_blank">
  没有登记的外国代理人
 </a>
 〞。纳瓦罗警告华尔街，不要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_插手美中贸谈.html" target="_blank">
  插手美中贸谈
 </a>
 ，这不会有任何好处。
 <br/>
 <br/>
 当地时间11月9日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 在华盛顿智库战略与国际研究中心(CSIS)发言时，严厉抨击
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华尔街.html" target="_blank">
  华尔街
 </a>
 高管与中共进行〝穿梭外交〞（Shuttle Diplomacy），试图向川普施压，要求他在G20川习会晤时，与北京达成协议。
 <br/>
 <br/>
 纳瓦罗表示，〝华尔街银行家和对冲基金经理人〞参与中共当局〝影响力行动〞，在G20峰会之前，他们向白宫展开〝全面施压〞。
 <br/>
 <br/>
 他将这些督促美中结束贸易纠纷的华尔街高管，称做〝未注册的外国代理人〞。指他们的行为只会削弱川普总统和他的谈判立场。不会有任何好处。
 <br/>
 <br/>
 纳瓦罗警告华尔街不要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_插手美中贸谈.html" target="_blank">
  插手美中贸谈
 </a>
 ，称这种交易都将带有〝高盛铜臭味〞。他还表示，〝如果有协议，以及当达成协议时，它将依据川普总统的条件，而不是华尔街的条件。〞
 <br/>
 <br/>
 纳瓦罗说，川普总统有勇气和智慧对抗世界上那些实施不公平贸易、非对等贸易、高关税、高非关税壁垒的国家。
 <br/>
 <br/>
 他表示，〝川普在解决这些问题上做得非常出色，他不需要华尔街的帮助。他不需要高盛的帮助。〞
 <br/>
 <br/>
 随着美中争端升级，多位华尔街高管与美中官员会面，敦促双方达成解决方案。周三，美国前财长、原高盛集团主席保尔森在新加坡发表讲话时警告，美中之间正在竖起新的〝经济铁幕〞，破坏全球化的益处。现任财政部长姆努钦也曾担任高盛高管多年，被认为是对华温和派。
 <br/>
 <br/>
 纳瓦罗发表这番讲话之际，外界正期待川普与习近平于本月底在G20峰会期间的会晤。这次会晤为两国缓和贸易紧张局势提供了机会。
 <br/>
 <br/>
 不过，纳瓦罗对川习会并不看好，他说，〝如何能和根本不知道你关心什么的人达成交易？〞
 <br/>
 <br/>
 10月23日，纳瓦罗在接受CNN采访时也曾表示，中共并未处理其对美国的不公平贸易行为，而且他相信中共现在的一些做法反而更糟糕，〝他们正在朝相反的方向迈进〞。
 <br/>
 <br/>
 他表示，美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 何时消停没有时间表，〝球在中国（中共）的手上〞，不过〝中共走在错误方向〞。
 <br/>
 <br/>
 在美中贸易问题上，纳瓦罗被认为持鹰派经济观点。他一直强调，中共必须做出结构性的改变，而非仅仅多购买一些美国货。
 <br/>
 <br/>
 不过，川普最近几天表示，他相信能与习近平达成协议。11月1日，川普与习近平进行长时间良好的通话，两人重点谈了贸易问题。有分析认为，北京有可能向美国做出了某些承诺，以便为G20川习会晤营造气氛。
 <br/>
 <br/>
 目前，白宫已经对价值2500亿美元的中国商品加征了关税，并警告说，如果未能达成协议，将对全部中国进口商品加税。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
