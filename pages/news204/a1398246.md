### 200余北京黑衣保安  深夜打砸店铺
------------------------

<div class="wysiwyg">
 【新唐人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 时间2018年11月06日讯】北京市最近发生了两起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_打砸.html" target="_blank">
  打砸
 </a>
 店铺的事件，其中海淀区，白石桥南懒龙店一条街，7家店铺的门面于深夜被打砸，房主与商户称，是200余名身穿保安制服的不明身份人员的黑恶势力所为。
 <br/>
 <br/>
 位于海淀区白石桥地铁站附近，懒龙店一条街，于10月31日凌晨2时30分许闯进七台推土车、铲车以及200余名身穿保安制服的不明身份人员。
 <br/>
 <br/>
 当事者一商家负责人马先生对大纪元记者说：〝保安拿着防暴盾牌和棍子，把人清走后，用挖掘机砸玻璃，打伤好几个人。把我从店门口一直打到路对面，6、7个人围着我拳打脚踢。〞据了解，商户中有7人受伤。
 <br/>
 <br/>
 店铺里面的所有人被强行清退后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 人员用挖掘机与铲车将7家店门的玻璃等砸碎，警方到现场时，对方已扬长而去。
 <br/>
 <br/>
 现场图片显示，一条街的数家商户门前一片狼藉，到处是碎玻璃。由于马先生与店员的反抗，强拆者害怕事情扩大化，最终未对其店铺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_打砸.html" target="_blank">
  打砸
 </a>
 。
 <br/>
 <br/>
 据马先生透露，强拆人员全部都是黑保安，幕后指使者具体是哪一方不清楚，但是房东称，他们是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 汽车集团有限公司派人过来打砸的，他们只做生意都是受害者。
 <br/>
 <br/>
 房东胡先生未正面证实是北汽所为，但他承认〝应该是企业行为〞。胡先生透露，强拆人员全部是东北人，一帮黑恶势力，这些人都是有犯罪前科者。前后来了三次进行骚扰、打砸，10月31日最为严重，把所有的店铺都给砸了。
 <br/>
 <br/>
 胡先生原是北京轻型汽车有限公司员工（简称轻汽），2009年，轻汽频临破产，他与其他员工10余人承包了目前的一条街与一空地，总占地面积3,000多平方米，以此来解决失业后的生存问题。
 <br/>
 <br/>
 酒店经营近10年，2011年轻汽破产，之后没有与任何方面再签订合同，而且与轻汽的合同期限至2029年。胡先生说：〝我们轻汽破产以前是北汽公司所领导的，虽然没有正式接触，谁都知道这个轻汽上面是北汽集团。听说他们是要建养老院。〞
 <br/>
 <br/>
 据商户统计，此次打砸共损失达200多万元。胡先生表示，他们将通过法律手段维护自己的权益。
 <br/>
 <br/>
 在此之前，北京市朝阳区南磨房农工南路甲1号西门烤翅也发生类似砸店事件。
 <br/>
 <br/>
 10月25日中午11时许，50～60名手持盾牌的黑保安闯进店中，不由分说往外面拖人。店主陈先生说：一堆保安拿着大铁鎚、钳子、灭火器什么的就过来了，过来抡起大铁鎚就砸，我叔叔受伤，现在是没水没电，不管不问。
 <br/>
 <br/>
 陈先生在此经营该店已11年，去年刚刚与房东（二房东）续签合同。房东介绍，他与该乡直属的东方盛泽商贸集团碧水云天餐饮有限公司签合同，租赁他们的房子，租10年合同到期后，又续签6年，刚履行不到一年，他们要强行收回，说是要建养老院。
 <br/>
 <br/>
 房东说，黑保安利用调虎离山计，打砸一楼的店铺西门几次，同时将其二楼的酒店门锁换掉，阻止其营业。〝现在把我们这片都用铁皮圈上了，我们上区政府和市政府告状也没有结果。〞
 <br/>
 <br/>
 据了解，房东与碧水云天餐饮公司因拆迁赔偿问题商谈多次，但是对方执意无赔偿，强行腾退，最后是利用黑势力
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力.html" target="_blank">
  暴力
 </a>
 打砸。
 <br/>
 <br/>
