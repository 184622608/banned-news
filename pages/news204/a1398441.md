### 资本外流加剧？中国外储10月创两年来最大降幅
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】中共央行最新公布的数据显示，外汇储备连续第三个月下降，10月份创下2016年12月以来的最大单月降幅。中共官方将其归因于金融市场波动，而《华尔街日报》则认为是人民币持续贬值引发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_资本外流.html" target="_blank">
  资本外流
 </a>
 。
 <br/>
 <br/>
 中共央行11月7日公布，10月份外汇储备比九月份减少339.27亿美元，降至3.053万亿美元，达到2017年4月份来最低点。
 <br/>
 <br/>
 另外，10月份外储也低于预期。此前，七位经济学家向《华尔街日报》提供的预测中值为，10月份降至3.057万亿美元。
 <br/>
 <br/>
 对此，中共外汇局的解释是，目前跨境资金流动〝总体稳定〞，外储下降主要原因是美元指数和一些国家的资产价格波动，以及汇率折算变动等，导致资产重估。
 <br/>
 <br/>
 但《华尔街日报》报导称，中国外储急降，显示人民币持续贬值可能已引发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_资本外流.html" target="_blank">
  资本外流
 </a>
 。
 <br/>
 <br/>
 中美贸易战开打以来，人民币兑美元汇率连续7个月下跌。WIND图表显示，10月份在岸人民币累计下跌920点，跌幅1.34%。
 <br/>
 <br/>
 这7个月来，中共央行疑似多次出手，阻止人民币跌势。美国政府也多次喊话北京，不能将人民币汇率作为贸易战工具。
 <br/>
 <br/>
 外界分析，美国不希望中共贬值人民币，以抵消美国关税造成的中国出口下降。而中方也不希望人民币过快贬值，带动资本加速外流。
 <br/>
 <br/>
 多有猜测认为，人民币兑美元汇率破7或是中共央行设置的心理防线。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
