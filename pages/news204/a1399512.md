### 【禁闻】经济疲软 刺激失效 陆房市泡沫或破裂
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共统计局.html" target="_blank">
  中共统计局
 </a>
 最新数据显示，中国10月份的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房地产.html" target="_blank">
  房地产
 </a>
 销售增速创半年来以来最低。有专家指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济危机.html" target="_blank">
  经济危机
 </a>
 已经在悄然发生，明年房地产泡沫会破灭。
 <br/>
 <br/>
 11月14号，中共国家统计局公布的10月份经济数据显示，中国10月份社会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_消费品零售总额.html" target="_blank">
  消费品零售总额
 </a>
 为人民币3.55兆元，月增率仅8.6%，低于9月的9.2%，创5月以来新低，显示民间消费持续疲软。
 <br/>
 <br/>
 而驱动中国大宗商品需求、建筑活动和消费支出的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房地产.html" target="_blank">
  房地产
 </a>
 市场，也成为经济逆风的推手，中国房地产市场同样在持续降温，商品房销售面积连续第2个月收缩。今年9月份较8月下滑3.6%，10月则比9月份降低了3.1%。
 <br/>
 <br/>
 大陆金融分析师任中道认为，10份的金融数据更能说明问题。
 <br/>
 <br/>
 大陆金融分析师任中道：〝10月份的金融数据让很多人都非常吃惊，人民币贷款和社会融资规模大幅的缩水，甚至达到环比腰斩的程度，这意味着整个经济在萎缩当中，企业不想去贷款去投入资金作发展，这就是很大的一个问题。〞
 <br/>
 <br/>
 实际上，今年中共已经作出多次刺激政策来拉动经济增长。7、8月份就提出大力加快基础建设；随后财政部又发行地方政府的专项债券刺激基建投资。
 <br/>
 <br/>
 统计局的数据还显示，今年1月-10月，道路、建筑和其他固定资产投资同比攀升5.7%，高于1-9月5.4%的增幅。
 <br/>
 <br/>
 任中道：〝那就是说，无论从资金还是项目上面都是要在基建上面去下功夫努力，但是都没有产生出他们期望的效果，经济并没有因为基建而恢复了增长。〞
 <br/>
 <br/>
 任中道说，之所以出现基建再也不能拉动GDP的原因，是经济的萎缩来自与需求端，不论老百姓的消费和企业的发展都在收缩。
 <br/>
 <br/>
 北京师范大学MBA导师段绍译认为，目前中国的GDP下跌、出口减少、老百姓的生活水平下跌，都和中共打压民营企业息息相关，而最近所谓的扶持民营企业和小企业也只是口号。
 <br/>
 <br/>
 北京师范大学MBA导师 段绍译：〝其实民营企业根本不需要扶持，只要不打压就可以了，到目前来讲，民营企业都是二等公民，很多方面和国有企业是无法比的，很多优惠政策资源它都是倾向于国有企业的民营企业从来贷款都很难，因为国有银行只贷给那些效率低下的国有企业。〞
 <br/>
 <br/>
 湖南大学电脑城业主吴先生告诉新唐人，民营企业确实生存艰难，他们现在的生意和前几年无法相比，以前需要夫妻两经营的店面，现在他一个人看管还閒得不行，要靠妻子外出打工才能勉强维持生计。
 <br/>
 <br/>
 段绍译说，实际上，中国早就应该发生的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济危机.html" target="_blank">
  经济危机
 </a>
 现在已经悄然开始。
 <br/>
 <br/>
 段绍译：〝其实已经在向经济危机迈进，主要表现为：第一股市长期低迷、第二出口减少、第三工厂开工率下跌、第四失业率增加、第五消费下降，这些都是经济危机的表现。〞
 <br/>
 <br/>
 另外，用来支撑中共经济的房地产萎缩，也成为外界关注焦点。
 <br/>
 <br/>
 段绍译：〝我认为2019年中国的房地产泡沫会破灭，其实早就应该破灭的。〞
 <br/>
 <br/>
 而明年中国房企3850亿元的人民币在岸债务到期，更加引发债务违约的担忧。
 <br/>
 <br/>
 任中道：〝比较难的一方，就是经济下行就使业主恐怕还不上贷款，就会被法院进行拍卖，而且最关键的一点和美国不同的，美国还不起贷款房子拍卖了，他的债务就完事了，但是在中国是无限责任的，在中国哪怕走到了这个地步他还是需要去还债的。〞
 <br/>
 <br/>
 任中道说，当民众被逼得走投无路时可能会出现大面积的自杀，甚至大面积的群体事件，或报复社会都有可能发生。
 <br/>
 <br/>
