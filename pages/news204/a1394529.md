### 央行放水无功 中国长假后首日现股汇双杀
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】尽管在〝十一〞长假最后一天，中共央行宣布降准释放巨额资金，但未能阻止次日股市一开市就开始大跌。不过，央行放水为当日人民币兑美元汇率跌近破7关口增加了助力。分析师指，中国市场信心脆弱。
 <br/>
 <br/>
 据路透社周一（10月8日）报导，长假后首个交易日，中国金融市场普遍走弱。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 沪综指周一大盘低开低走，收盘重挫3.7%，创逾三个半月最大单日跌幅。沪综指.SSEC收报2,716.51点，跌104.84点或3.72%。A股市场仍有二次探底的可能。
 <br/>
 <br/>
 同日，人民币兑美元即期CNY=CFXS收盘大跌321点，破6.91关口，跌至近一年半来新低，中间价则达到近17个月新低。
 <br/>
 <br/>
 中共央行周日宣布，自10月15日起，下调大型商业银行、股份制商业银行、城市商业银行、非县域农村商业银行、外资银行人民币存款准备金率1个百分点。此次降准将释放7,500亿元人民币的增量资金。
 <br/>
 <br/>
 报导引述大陆分析师指，中国股市下跌，是受长假期间海外新兴市场走弱影响。尽管央行宣布降准，但释放的资金对市场的利好支撑程度有限。
 <br/>
 <br/>
 报导亦指，人民币汇率大跌的原因是，长假期间美元指数大幅上涨，以及中共央行再次降准放水。汇率再次接近破7关口，显示贬值预期明显升温，中共管理层再次面临挑战。
 <br/>
 <br/>
 还有交易员认为，除了美指和降准，中国A股大跌亦损伤市场对汇率的信心，〝最大的空头就是A股〞。
 <br/>
 <br/>
 中美贸易冲突爆发以来，中国金融市场持续恐慌。其中，中国股市暴跌，已将全球第二大股市的头衔归还给日本股市。
 <br/>
 <br/>
 长假期间，美国副总统彭斯在10月4日发表重要演讲，对中共进行了不留情面的全方位谴责。海外学者惊呼〝新冷战宣言〞，中国网络上也开始互相提醒，〝新冷战开始，你准备好了吗？〞
 <br/>
 <br/>
 美媒分析，中共央行周日宣布降准，说明中美贸易战持续升级，已经让北京非常紧张。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
