### 范冰冰案引持续震荡 传17名艺人被约谈
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】大陆女艺人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 逃税事件，引发娱乐圈持续震荡。近日，圈内社交平台流传一则消息称，在最近3天内，浙江已有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_17名艺人被约谈.html" target="_blank">
  17名艺人被约谈
 </a>
 ，需在限定时间内
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_补交税金.html" target="_blank">
  补交税金
 </a>
 。
 <br/>
 <br/>
 近期引起轰动的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞事件，最终以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 补缴近9亿人民币税金罚款告终。不过，〝阴阳合同〞引发的风暴，仍在席卷大陆娱乐圈。中共官方日前已正式开始对娱乐圈〝剪羊毛〞。
 <br/>
 <br/>
 11月29日，大陆影视圈从业人员的微信朋友圈传出一则消息，部分影视工作室已陆续接到中共税务部门的通知，补税工作正式实施，共分为四个阶段：自查自纠、约谈补税、税务上门辅导、检查以及重点检查。
 <br/>
 <br/>
 消息称，在3天内，浙江已有17位艺人被约谈，补税工作需要在12月15日前完成。据浙江税务局的通知显示，影视工作室需补缴费用和滞纳金的日期截至12月30日。
 <br/>
 <br/>
 按照通知，影视工作室需要按2016至2018年三年总收入的70%，来计算个人劳务税款。
 <br/>
 <br/>
 举例说，若总收入为100万，则以70万为个人补税依据，再按照最高40%劳务税率计算，同时扣除之前已经缴纳过的税金，最终补缴税款为192,500元人民币。也就是说，100万即要补缴约20万税款。
 <br/>
 <br/>
 日前，已有多位大陆影视从业人员向陆媒《证券时报》证实，确有补缴税款一事，不少人对此表现出悲观情绪。另有消息称，大陆编剧行业需按原规定，缴纳16%税款，未足16%补足即可。
 <br/>
 <br/>
 有影视从业者表示，不同影视工作室经营情况不同，有收入好的，也有大量收入情况不乐观的，如果按新规交税，经营压力很大。
 <br/>
 <br/>
 还有不愿透露姓名的导演建议称，可以对补税主体做一定区分，对一些小微型工作室，还应该适当减免税，把影视行业的鱼养大，不能一刀切，否则〝大工作室伤筋，小工作室伤命〞。
 <br/>
 <br/>
 据此前报导，自范冰冰〝阴阳合同〞事件曝光后，8月份，横店各影视工作室就陆续收到当局实行〝新税制〞的口头通知。原来定期定额的征税方式改为查帐征收，艺人缴交税率从6.7%猛增到42%，并需补缴今年前6个月税款。
 <br/>
 <br/>
 〝新税制〞不仅牵连所有大陆艺人、编剧和制作公司，还包括多名港台艺人。台媒此前引述知情人士消息称，第一波被中共当局锁定调查的、片酬超过1,000万元以上的近200名影视明星中，包括多名台湾演员，如彭于晏、霍建华、赵又廷、阮经天等人，传需补缴税金暴涨14倍。
 <br/>
 <br/>
 有分析指，在贸易战的袭击下，中共现在已国库空虚，当局将目标对准油水丰厚的娱乐圈，开始〝宰肥羊〞、〝割韭菜〞。
 <br/>
 <br/>
 据陆媒披露，在范冰冰〝被失踪〞的四个月里，为避免遭当局打压，大陆诸多知名艺人工作室与影视公司纷纷关闭。还有不少艺人开启休假模式。业内人士表示，大陆影视圈的寒冬已经提前来临。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
