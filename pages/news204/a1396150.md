### 中美贸战缩影 中国女首富身价暴跌超6成
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 火下，中国亿万富翁数目骤减，身价暴跌。一度成为中国女首富的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓝思科技.html" target="_blank">
  蓝思科技
 </a>
 董事长周群飞，是所有中国亿万富豪中，身价暴跌最惨烈的企业家之一，身价缩水超6成。
 <br/>
 <br/>
 根据彭博亿万富豪指数，身为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓝思科技.html" target="_blank">
  蓝思科技
 </a>
 大股东的周群飞，2018年以来，财富大幅缩水66亿美元，身价暴跌66%。
 <br/>
 <br/>
 蓝思科技为全球最大触控萤幕制造商，一直为苹果和特斯拉生产触控萤幕，将公司董事长周群飞，一度推上中国女首富的宝座。但是由于受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 火的不断冲击，今年以来中国股市一路下行、跌跌不休，蓝思科技
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股价.html" target="_blank">
  股价
 </a>
 更遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重挫.html" target="_blank">
  重挫
 </a>
 62%。
 <br/>
 <br/>
 彭博分析师汤普森和贾乐禾说：〝陆股这几个月更容易受贸易战的负面新闻影响，恢复速度比美股慢，而且受贸易战的正面新闻激励的幅度远比任何地方都小。〞
 <br/>
 <br/>
 不仅仅蓝思科技董事长周群飞身价暴跌，根据胡润研究院日前公布的2018年中国富豪榜，身家达20亿元以上的中国亿万富翁减少了237人，这是6年来首度出现减少。
 <br/>
 <br/>
 有法媒分析说，中国亿万富翁减少受多种因素影响，除了今年中国股市大跌以外，中美贸易战中，美国对中国商品征收惩罚性
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，导致来自工业领域的亿万富翁比例跌至26.1%，较去年减少约1.8%。
 <br/>
 <br/>
 针对美中贸易战，摩根大通银行10月上旬下调了中国股票评级，从增持下调到中性，并预测贸易战升级将影响明年的中国经济。
 <br/>
 <br/>
 摩根大通报告说：〝根据我们的分析，中国能源、IT和工业等行业将受到最大影响，而房地产、保险、多元化金融、电信和水电不会从美国获得任何收入。〞
 <br/>
 <br/>
 报告说，随着美国对中国商品加大关税，世界两大经济体的冲突将只会升级，美元将继续走强，而人民币将继续走弱。
 <br/>
 <br/>
 彭博社说，在摩根大通下调中国股市并预测贸易战将全面爆发后，追踪中国股市的最大交易所交易基金（ETF）大跌。价值49亿美元的iShares中国大型股ETF跌至两周低位。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
