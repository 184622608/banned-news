### 川普要跟习近平谈人权？彭斯透露川习会五大议题
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】亚太经合组织峰会（APEC）闭幕后，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）在即将回国时透露，11月底的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞上，川普（特朗普）和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将讨论五大议题，包括：中共的不正当贸易行为、强制转移技术、人权问题等。
 <br/>
 <br/>
 美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，17日在亚太经济合作组织（APEC）峰会上激烈交锋。习近平谈了中美贸易战和北京所倡议的〝一带一路〞等问题。
 <br/>
 <br/>
 彭斯直接点名批评中共设置的贸易壁垒史无前例，并且抨击中共的〝一带一路〞是给合作国家设立的债务陷阱。在中共改变行为方式前，美国不会改变征收关税的政策。
 <br/>
 <br/>
 不过，在17日的晚宴会等场合，习近平与彭斯有过两次交谈。
 <br/>
 <br/>
 彭斯表示，他和习近平有过两次的坦诚对话。他对习近平说，美国总统与人民愿意与中方建立更好的关系，但中方需要先作出改变。
 <br/>
 <br/>
 彭斯表示，中美两国目前的分歧始于贸易问题、关税及配额，以至强制技术转移、窃取知识产权、航行自由及人权问题，而习近平回应说，中美对话十分重要。
 <br/>
 <br/>
 彭斯还说，美国总统川普希望11月底在阿根廷举行的二十国集团（G20）峰会和习近平会面。
 <br/>
 <br/>
 彭斯披露，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞将讨论的5大议题，包括：中共的不正当贸易行为的关税壁垒、强制要求的技术转移、侵犯知识产权、航行的自由和人权问题。
 <br/>
 <br/>
 彭斯在新加坡接受专访时还警告中共，必须全面改变其行为作法，才能避免与美国发生新冷战。
 <br/>
 <br/>
 另据美国之音和大纪元此前披露，川普政府将改变美国以往在中国宗教自由、人权、法治问题上，抱持的软弱态度。美国国家安全顾问博尔顿也表示，中共对人权和宗教自由的迫害，将是他的一个重要议题。
 <br/>
 <br/>
 彭斯10月初在华盛顿就川普政府对中共的政策发表长篇演说时，从政治、经济、军事、外交等方面，全方位指控中共。
 <br/>
 <br/>
 他强调，中共从未变好，无论是对外或对本国人民。美国在过去25年帮助〝重建〞了中国，相反，中共选择了经济侵略，也没有像美国希望的那样，让自己的人民迈向更大的自由，而是朝着控制和压迫中国民众的方向急转弯。
 <br/>
 <br/>
 彭斯说，现在美国真的觉醒了，〝历史已经证明，那些压迫本国人民的国家，很少就此住手。北京还试图将其势力扩展到全世界各地。〞
 <br/>
 <br/>
 因此，川普总统在经济、政治、外交、军事等领域展现出的全新姿态，绝非中共诋毁的〝单边保护主义〞，而是反击邪恶共产主义的进取之姿。在这场正邪对抗赛中，全世界的人都已经看清，美国的对手不是中国或中国人民，而是中共。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
