### 鄱阳湖水枯 千年〝落星墩〞露真容
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】位于江西九江庐山市的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鄱阳湖.html" target="_blank">
  鄱阳湖
 </a>
 ，随着枯水期的到来，水位迅速下降，这座拥有千年历史的石岛〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落星墩.html" target="_blank">
  落星墩
 </a>
 〞终于露出了真面目。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落星墩.html" target="_blank">
  落星墩
 </a>
 总面积大约1800平方米，传说是星星坠落湖中，因而得名。自宋代开始，便有人在上面修建亭院，明代又加建亭台楼阁。然而落星墩大多时候被淹在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鄱阳湖.html" target="_blank">
  鄱阳湖
 </a>
 里。
 <br/>
 <br/>
 连日来，随着水位迅速下降，鄱阳湖已提前进入枯水期，而在水下的千年石岛〝落星墩〞也因此得以重现人前，露出其庐山真面目。
 <br/>
 <br/>
 据报，截至9月26日，中共水文部门监测到鄱阳湖星子站水位是10.79米，与历史同期相比偏低5.07米。
 <br/>
 <br/>
 该部门解释说，今年鄱阳湖水位迅速下降的主要原因，是长江三峡蓄水，今年汛期水位较往年偏低，加上汛期江西境内降水明显少于往年，导致鄱阳湖水位消退速度加快。
 <br/>
 <br/>
 传说落星墩在五代时已经出现，当时被称为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宝石山.html" target="_blank">
  宝石山
 </a>
 〞，宋朝初年，就有人在石岛上建亭院，历代都加以维修。
 <br/>
 <br/>
 明代又加建亭台楼阁，如浮玉楼、玉京轩、岗漪轩、清晖阁等等，但都已经历尽沧桑，残存无已。落星墩远看像是漂浮在水面，宋人蒋之奇曾经用〝今日湖中石，当年天上星〞来形容它。
 <br/>
 <br/>
 而鄱阳湖则是中国最大的淡水湖，每年有季节性洪流，水域面积在丰水期，最大可达4400平方公里，它的流域占江西省流域面积的97%、长江的9%，流量更超过黄河、淮河、海河水量的总和。
 <br/>
 <br/>
 自古以来，鄱阳湖流域就是带动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 的摇篮，但由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三峡大坝.html" target="_blank">
  三峡大坝
 </a>
 的兴建、人为的采砂、气候变迁等因素影响，湖域面积开始减少。
 <br/>
 <br/>
 目前鄱阳湖水位仍在下降，预计未来水更会退去，周边变成草洲，民众也可以用步行的方式走到〝落星墩〞。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
