### 政治局会议首认经济下行 未提四中全会惹猜测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】10月最后一天，北京召开了本月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治局会议.html" target="_blank">
  政治局会议
 </a>
 ，首次承认经济下行压力加大，风险隐患暴露，但应对之策仍然强调此前提出的〝六稳〞。对于外界关注的中共十九届四中全会，官方通稿只字不提。四中全会一再推迟，引起各种猜测。
 <br/>
 <br/>
 据新华社报导，北京时间10月31日，北京领导人习近平主持召开了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治局会议.html" target="_blank">
  政治局会议
 </a>
 ，〝研究当前经济形势，部署当前经济工作〞。
 <br/>
 <br/>
 会议指，当前经济下行压力加大，企业经营困难，风险隐患暴露，并称造成这种形势的原因包括内部因素，也包括外部环境发生的〝深刻变化〞。
 <br/>
 <br/>
 这是中共核心领导层首次公开承认经济下行。同时，官方表述中也承认了中美贸易战升级的影响。三个月前，政治局只称外部环境发生了〝明显变化〞。
 <br/>
 <br/>
 谈及应对之策时，本月政治局会议再次重复了三个月前的加大〝改革开放〞，以及〝稳就业、稳金融、稳外贸、稳外资、稳投资、稳预期〞的调子。
 <br/>
 <br/>
 不过，为了回应海内外对〝国进民退〞的担忧，这次通稿加上了〝促进多种所有制经济共同发展〞的内容。
 <br/>
 <br/>
 对于外界关注的四中全会召开时间，会议通稿没有提及。
 <br/>
 <br/>
 按照此前惯例，中共每届三中全会都在召开当年的7、8月间公布大致开幕时间，然后在正式召开前夕的政治局会议上再次敲定准确日期和细节。今年因为加开了一次确定修宪的二中全会，外界猜测本届四中全会将沿用往届三中全会的旧例，最迟在10月召开。
 <br/>
 <br/>
 不过，中共官方至今没有公布有关四中全会的任何信息。有猜测指，会议或推迟到11月下旬召开，甚至可能要延后到明年。
 <br/>
 <br/>
 按照惯例，此次全会的中心议题将是经济改革。外界认为，四中全会一再推迟，和中共高层尚无统一的经济决策有关。面对积重难返的国内经济问题以及中美贸易战的压力，北京很难再推出有效的应对之策。
 <br/>
 <br/>
 还有分析指，四中全会会期无法确定，或说明中共内斗激烈，习近平当前无力掌控局势。
 <br/>
 <br/>
 也有猜测称，北京或在等待美国11月初的中期选举结果，然后再确定对策。不过，外界普遍认为这是北京误判，因为遏制中共已是美国两党共识，中期选举对贸易战影响不大。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
