### 【禁闻】湖北一垃圾处理专案在抗议中停建
------------------------

<div class="wysiwyg">
 【新唐人2018年11月11日讯】自11月6号开始，湖北黄冈市浠水县
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴河镇村民.html" target="_blank">
  巴河镇村民
 </a>
 不断上街拉横幅、游行、到当地政府请愿，甚至是堵高速路等行动，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 在当地建设垃圾焚烧发电厂。浠水县政府最后不得不宣布停建这个专案。不过，在抗议过程中，有村民被抓、被打伤等。请看报导。
 <br/>
 <br/>
 11月10号，湖北省浠水县当局发布停止〝山水环保发电专案〞的通告。
 <br/>
 <br/>
 通告说，山水环保发电项目由于没有获得周边环境范围内的群众多数人同意，决定正式停止建设山水环保发电专案。
 <br/>
 <br/>
 湖北省浠水县
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴河镇村民.html" target="_blank">
  巴河镇村民
 </a>
 刘先生表示，浠水县政府发布的通告，是在众多村民持续的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 下做出的，但不排除是缓兵之计。因为抗议村民被抓走20多人，现在只放出来10多人，还有10多村民仍被关押。
 <br/>
 <br/>
 湖北省浠水县巴河镇村民刘先生：〝主要是昨天闹起来了。我们市政府要在我们一个14万人口，密集的地方小镇要建一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_垃圾焚烧厂.html" target="_blank">
  垃圾焚烧厂
 </a>
 、焚烧发电厂。在没有经过村民通过的情况下，竟然说是全票通过的，我们所有的村民在维权。〞
 <br/>
 <br/>
 刘先生介绍说，因为当地的年轻人大多在外地打工，所以抗议的村民大部分是老人、妇女。但当局为了打压抗议村民对当地断电，导致网路、手机无法使用，什么消息都发不出去。
 <br/>
 <br/>
 刘先生表示，在没办法的情况下，有近千名当地民众10号将大庆至广州高速上下路口，以及通往黄冈市的国道堵死一、两个小时。但当局派出上千名警察、武警，对所有抗议村民大打出手，其中一孕妇被打流产，还有一些人被打伤后需要到当地医院救治。直到现在，武警还在控制各个路口。
 <br/>
 <br/>
 其实，巴河镇6号就有村民拉起〝蓝天不是垃圾场〞的标语，冒雨在镇内游行，反对在当地建立垃圾焚烧发电站。
 <br/>
 <br/>
 当天，浠水县政府也发布公告，指称，在未得到广大群众认可前，将暂缓这个项目但当地村民认为那是政府在敷衍，并没有停止抗议。8号，约200村民到浠水县政府门前聚集。9号的大规模抗议游行遭到镇压。
 <br/>
 <br/>
 无奈之下，10号，村民们实施堵高速、国道的策略，希望引起外界关注。
 <br/>
 <br/>
 据了解，浠水县计划兴建的垃圾焚烧发电厂紧靠长江，项目投资6.3亿元，占地面积120亩，预计每天处理垃圾量约1,200吨。
 <br/>
 <br/>
 前陕西电视台记者马晓明：〝垃圾要焚烧的话，主要就是会产生毒性很高的二恶英，对人的健康造成很大的损害。但是世界上有些国家掌握那种技术，能够控制焚烧的温度，可以减少垃圾焚烧的危害。〞
 <br/>
 <br/>
 前陕西电视台记者马晓明表示，他知道台湾对垃圾的分类回收、分类处理做得很好，台湾很多官员、志愿者也一直不厌其烦的到民众家讲解垃圾分类的重要性、示范如何分类。大陆在这方面的文明程度大大落后。如果大陆把宣传垃圾分类像宣传领袖如何伟大、党如何正确那样卖力去宣传，情况早就不一样了。
 <br/>
 <br/>
 马晓明认为，垃圾的产生与每个人都有关系，每个人都要重视这一问题。
 <br/>
 <br/>
 他表示，主要问题不是要不要建、能不能建垃圾焚烧厂，关键在于当地政府如何遵循有关法规、就如何处理当地的垃圾向当地民众进行比较好的公示。让当地民众知道无论选择焚烧、掩埋等措施，都会产生副作用。如何处理民众需要做出选择。
 <br/>
 <br/>
 马晓明：〝政府很可能在公示、向周边相关的这些民众讲明并通过对话、交流得到大家的谅解、认可，在这些方面事情上做得不够。〞
 <br/>
 <br/>
 马晓明指出，在没有取得当地民众认可的情况下强行施工，民众就可能爆发一些大规模的请愿、堵路等行动。如果对这些行动再采取打压、抓人等措施，那是更错误的侵犯公民权利的做法。
 <br/>
 <br/>
 采访/陈汉 编辑/宋风 后制/钟元
