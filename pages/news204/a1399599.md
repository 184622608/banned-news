### 北大成立两新机构 加强监控师生言行
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】最近，北京大学成立两个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监控新机构.html" target="_blank">
  监控新机构
 </a>
 的文件在网上被曝光。 北京维权人士认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北大.html" target="_blank">
  北大
 </a>
 一些学生参与工人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权行动.html" target="_blank">
  维权行动
 </a>
 ，让当局感到忌讳，担心六四学运历史重演，所以要严控。
 <br/>
 <br/>
 最近一份北京大学委员会的通知文件在网络曝光，并引发热议。通知称，学校决定成立中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北大.html" target="_blank">
  北大
 </a>
 委员会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巡查办.html" target="_blank">
  巡查办
 </a>
 公室（巡查办）、北大内部控制管理办公室（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_控管办.html" target="_blank">
  控管办
 </a>
 ）。〝巡查办〞设在北大纪检委员会，是〝学校巡查工作领导小组〞的日常办事机构，与〝控管办〞合署办公。
 <br/>
 <br/>
 网友认为，这标志着中共对大学生、教师等知识分子进行全面监控。
 <br/>
 <br/>
 北大匿名校友：〝这些机构是要介入比如说老师的教学，或学生的一些思想管理的话，那我觉得可能跟北大自由的校风不符，也跟整个世界的趋势，就是大学自主管理或者是自由开放的思潮都是相违背。不光是北大，我觉得在全中国的校园都不应该对学生的思想啊、行为做过多的箝制。〞
 <br/>
 <br/>
 北京维权人士认为，北京大学曾是学术自由的典范，89年六四学运，北大是重要的阵地，因此北大学生是当局率先要严控的对象。
 <br/>
 <br/>
 北京维权人士马先生：〝北大前一段时间，一些学生参与了广东的工人维权事件当中，使当局对北大学生更加忌讳一些，所以说要严控。党国本身是邪恶嘛，它主要是为了邪恶的利益，不管你左派、右派，只要你站在正义的角度上，那一切都是他的敌人。未来所有中国的大学、学术机构，言论空间会越来越小，而且打压也会越来越严重。〞
 <br/>
 <br/>
 今年夏天以来，一些左派的高校学生加入声援深圳佳士工人组建工会的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权行动.html" target="_blank">
  维权行动
 </a>
 ，至今已经有20多名声援人士失联，10几名左派青年被绑架。其中北大毕业生顾佳悦、岳昕、张圣业、孙敏陆续被抓捕。而张圣业被同学曝光，是11月9号晚上， 在北大校园遭到秘密绑架。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
</div>
