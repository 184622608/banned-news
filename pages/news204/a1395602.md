### 中共罕见重提制造2025 中美科技冷战迅速升温
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】白宫官员指，11月川习会的前提是北京要有〝重大让步〞。但中共似乎释放了态度转向强硬的信号。中共主要喉舌日前批评贸易战〝必败论〞，罕有地重提此前已被禁止报导的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞计划。
 <br/>
 <br/>
 《人民日报》10月15日刊登署名为〝钟轩理〞的文章，批评中国国内〝恐美论〞、〝必输论〞等悲观论调，并引述一些所谓海外〝知名学者〞、〝知名机构〞力挺中共的说法，为贸易战树立〝信心〞。
 <br/>
 <br/>
 〝钟轩理〞是〝中宣部理论局〞的谐音，《人民日报》近日多次刊出〝钟轩理〞的批美长文。
 <br/>
 <br/>
 15日的文章，在有关知识创新的部分，罕有地提及〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞，直言承认美国启动贸易战瞄准的就是〝中国制造2025〞，并称美国无法阻挡中方〝高新技术产业前进步伐〞，因为中方的创新基础现在已与美国〝大致相当〞。
 <br/>
 <br/>
 当然，文章没有提到美国一纸禁售令就让中兴陷入瘫痪，北京不得不向川普请求放生。
 <br/>
 <br/>
 今年上半年，中美贸易冲突紧张之际，中方曾下令媒体严禁再提〝中国制造2025〞。虽然中共一再对美喊话，声称绝不放弃这一〝核心利益〞，但是〝中国制造2025〞的名词本身已在中共党媒消失多日。
 <br/>
 <br/>
 如今中共喉舌再提〝中国制造2025〞，并罕见地向国内承认这是中美贸易纷争的核心议题，被指为中共官方态度趋于强硬的信号。
 <br/>
 <br/>
 时政评论员唐靖远认为，美方最近一系列行动，直指〝中国制造2025〞的窃密行为，而中共亦开始展示强硬姿态，双方势必在科技领域爆发更大冲突。
 <br/>
 <br/>
 日前，因为盗窃美国航空发动机商业机密被美国FBI诱捕的中共情报官员徐燕军（又译徐彦钧），从比利时被引渡到美国。美方高调对其提出控诉，被指杀鸡儆猴。
 <br/>
 <br/>
 此外，美媒近期接连爆料指，美国多家厂商的电脑主板，在中国生产环节被植入恶意芯片，并指这是中共情报部门主导下的系统行为。消息引发中国科技版股价大跌。
 <br/>
 <br/>
 另据美媒消息，美国认定北京以不正当方式取得美国核技术，以强化军力并暗中侵蚀美国产业，因此美方将对出口核技术至中国实施新的限制。
 <br/>
 <br/>
 此外，今年以来，以获取外国高科技为主要目的的中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞，被FBI盯紧调查。消息指，一些美国研究机构已开除参与〝千人计划〞的华裔学者。另有多名华裔学者因卷入〝千人计划〞，被FBI以间谍罪逮捕。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
