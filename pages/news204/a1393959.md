### 【禁闻】又一法官离职 世贸组织仲裁机构或停摆
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】由于美国拒绝了目前仅剩的四名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世贸组织.html" target="_blank">
  世贸组织
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官.html" target="_blank">
  大法官
 </a>
 中的一名连任，世贸组织
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_仲裁机构.html" target="_blank">
  仲裁机构
 </a>
 面临因法官人数不足而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停摆.html" target="_blank">
  停摆
 </a>
 的危机。评论指出，世贸组织成了中共不诚信贸易的避风港。
 <br/>
 <br/>
 9月26号，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世贸组织.html" target="_blank">
  世贸组织
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大法官.html" target="_blank">
  大法官
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯旺森.html" target="_blank">
  斯旺森
 </a>
 连任的问题上，美国投下了反对票。
 <br/>
 <br/>
 近70个国家多次要求美国不要阻挠这项任命。但一名美国官员在世贸组织总部拒绝了这一要求。
 <br/>
 <br/>
 这不仅意味着这位毛里求斯籍的法官将在9月30号任期届满后，要离开世界贸易的最高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_仲裁机构.html" target="_blank">
  仲裁机构
 </a>
 ，也意味着这个机构面临
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停摆.html" target="_blank">
  停摆
 </a>
 危机。
 <br/>
 <br/>
 美国总统川普反对任命这些对于贸易争端有最终仲裁权的大法官。
 <br/>
 <br/>
 据《福布斯》杂志报导，美国驻世贸组织大使谢伊今年5月曾在一次演讲中表示，世贸上诉机构已经成为一个流氓组织，经常违反其运作规则。
 <br/>
 <br/>
 时事评论员田园博士：〝这个机构它负责的就是，根据世界贸易组织的贸易规则，来裁定各个国家的贸易纠纷。但是实际上，这个机构比橡皮图章还不如。它的仲裁决议是没有任何约束力的。（比如）如果这个仲裁机构判断中共败诉，中共仍然能够继续倾销钢铁而不用负任何实际的责任。〞
 <br/>
 <br/>
 世贸组织上诉机构的法官常设7人，此前因美国杯葛新法官遴选，只剩下4人。如今斯旺森离职，只剩下3人。
 <br/>
 <br/>
 另两位美国籍的格莱姆和印度籍的首席大法官巴提亚，将在明年12月结束任期。中国籍法官赵宏将在2020年11月结束任期。
 <br/>
 <br/>
 因为世贸组织的每项裁决都需要3名法官，而目前它应对的多项争议，很多是由美中贸易战所引发。如果剩下的3名法官在某一案件上需要避嫌，整个仲裁机制就会陷入停摆。
 <br/>
 <br/>
 旅美时事评论员蓝述：〝过去这一段时间以来美国一直采取的一个政策，就是世贸的法官退休以后，就阻止新的法官上任。道理很简单，就像川普总统讲的，全世界都在占美国的便宜。你世贸没什么用嘛，还要它去裁决？而且你真要让世贸去判的话，它就给中共很多的机会。〞
 <br/>
 <br/>
 田园：〝中共在加入世界贸易组织之后，可以说是被各个国家告的最多的国家之一。几乎每一个主要的贸易国，包括美国、加拿大、印度和日本，都和中国有各种各样的贸易争端。〞
 <br/>
 <br/>
 时事评论员田园博士表示，由于以往审理一个案子旷日持久，又没有惩罚机制，上诉机构存在与否，对目前已处于半瘫痪的世贸仲裁机制都没有影响。
 <br/>
 <br/>
 〝LIFETIME视界〞发推文说：美国就是想〝强拆〞世贸，世贸成了中共不诚信贸易的避风港。
 <br/>
 <br/>
 田园：〝盗版在中国依然横行，强迫外国公司转移技术、公开源代码，还有用非关税的壁垒，把外国的互联网公司挡在（中共）防火墙之外，这个事情一直到今天都没有解决。这就是为什么川普总统上台以来明确指出，你中共就是流氓，我现在要撇开WTO这个仲裁机构，我要直接对你动手了。这就是中美贸易战最根本的原因。〞
 <br/>
 <br/>
 此前，中共为了美国关税问题告上世贸，希望找世贸救命，来解决贸易战。
 <br/>
 <br/>
 蓝述：〝现在川普总统采取强硬的政策，你就去找世贸它也是解决不了的。你这个中共不解决所有美方提出来这些不公平、不守规矩的现象，他直接就给你加税。你告也没用。世贸停不停摆，它对中美之间（贸易）问题的处理没有太大的影响。〞
 <br/>
 <br/>
 有网友说：中共以WTO平台赚取了足够多的非法利润，如果WTO就此能改变规则，迫使中共国改弦更张，那么还有一点用。否则废除WTO，彻底孤立中共，势在必行！
 <br/>
 <br/>
 周日晚，美国、加拿大就《北美自由贸易协定》谈判达成协议。外界认为这是川普在重塑全球贸易方面取得重大胜利。
 <br/>
 <br/>
 旅美时事评论员蓝述表示，川普现在采取的办法，就是和自由世界这些国家一个个去谈判，这种谈判方式也是做给中共看的。即使再不守规则，将来中共也得过来谈判。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/钟元
