### 【禁闻】蔡英文双十节讲话 吁北京不做冲突来源
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】中华民国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 10月10号发表国庆谈话，她表示，中共单方面的文攻武吓和外交打压伤害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_两岸关系.html" target="_blank">
  两岸关系
 </a>
 ，挑战区域和平现状。她呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 当局不要充当冲突的来源，同时强调要有全新的经贸战略，让台湾变得无法取代。
 <br/>
 <br/>
 中华民国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 10月10号发表国庆谈话，她表示，国际政经局势正在面临剧烈的变化，近期中共单方面文攻武吓和外交打压，严重挑战了台海和平稳定的现状。
 <br/>
 <br/>
 蔡英文表示，她坚持维持台海和平跟区域稳定。不会贸然升高对抗，也不会屈从退让，并呼吁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 当局，应该扮演良性的角色，而不是冲突的来源。
 <br/>
 <br/>
 台商受害联盟理事长、台湾企业家与社会运动人士高为邦：〝两岸形势当然中共用不同手段压迫台湾，这个事情太多了，（友邦）断交的事情，还有要求航空公司要把台湾置于中国之下，这些都是发生的事情，也是造成两岸不稳定的情况。〞
 <br/>
 <br/>
 台湾大学国家发展研究所法学博士曾建元：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_两岸关系.html" target="_blank">
  两岸关系
 </a>
 呢，它置身于这样一个环境中，它不能被独立出来看，它必须放在全球秩序这个角度来看。不仅台湾是民主国家的前线，台湾在自由经济上面，特别是在科技产业、电子产业上面，它扮演重要的地位。所以呢，对两岸现状的一种破坏它实际上是对于全球的人类文明这个进展的一个重大挑战。〞
 <br/>
 <br/>
 蔡英文谈到，捍卫台湾的最佳方案，就是要让台湾在世界上变得不可或缺，也不可取代。
 <br/>
 <br/>
 中华民国总统蔡英文：〝过去两年在中国（中共）的压力之下，我们坚持价值理念，展现高度的韧性，让越来越多理念相近国家，表达对中华民国台湾的支持。〞
 <br/>
 <br/>
 为了强化这种外交上的价值连结，建构台湾不可取代的战略重要性，蔡英文说，他们战略选择相当清楚，就是要坚守自由民主以及市场经济。
 <br/>
 <br/>
 曾建元：〝台湾在做这样一个价值选择。也就是说，它从市场经济出发，希望国家在维持一个公平交易的市场秩序当中呢，扮演一个中立性的角色，公正的一个角色，而不是像党国权贵资本主义，那个党国本身是市场当中最重要的一个行动者，这个就是今天中国大陆的问题。〞
 <br/>
 <br/>
 另外蔡英文还谈到提升国防战力；阻止外来势力渗透破坏，确保民主制度及社会经济正常运作；以及全球经贸战略的重整及全新布局的面向。
 <br/>
 <br/>
 蔡英文在演讲中也呼应美国副总统彭斯此前的演讲。彭斯日前在哈德逊研究所的演讲中，赞许台湾的民主。
 <br/>
 <br/>
 中华民国总统蔡英文：〝台湾就是一个灯塔。我们的民主转型，不仅照亮了自己曾经走过的黑暗，也为所有追求民主的人们，提供了暗夜中的光芒。所以，在香港、在中国大陆，以及在世界上各个角落，追求民主的朋友，请往台湾的方向看过来。台湾的民主，会照亮世界！〞
 <br/>
 <br/>
 包括CNN、彭博等多家国际媒体都报导了蔡英文的演讲内容。中央社说，美国国务院周三以发言人背景说明的方式引述彭斯日前的演说表示，将持续遵守美中三公报和台湾关系法下的一中政策，相信台湾对民主的拥抱为所有华人展示一条更好道路。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/李沛灵
