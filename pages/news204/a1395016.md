### 【禁闻】中共国安官员窃密 首次引渡美国受审
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】大陆江苏省国家安全厅一名副处长，因涉嫌窃取多家美国航空航天公司商业机密，今年4月在比利时被逮捕，并于本周被带到美国接受间谍指控。据信，这是第一次有中共政府间谍被引渡到美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受审.html" target="_blank">
  受审
 </a>
 。
 <br/>
 <br/>
 周三，美国司法部以涉嫌经济间谍罪名，起诉江苏省国安厅六局的副处长徐彦钧（Yanjun Xu，音译），他被指以多个化名进行涉及美国军方的经济间谍活动。
 <br/>
 <br/>
 徐彦钧4月1号在比利时被逮捕，他以为能获取有关通用电气航空的情报，结果被诱入圈套。
 <br/>
 <br/>
 10月9号，徐彦钧被引渡抵达美国。
 <br/>
 <br/>
 美国司法部官员表示，他们一直等到徐彦钧被引渡后，才公布对他罪行的指控，包括对美国进行经济间谍活动，以及试图窃取美国航空企业的商业机密。
 <br/>
 <br/>
 10号，徐彦钧现身俄亥俄州辛辛那提联邦法院。他被控四项罪名，包括串谋和企图从事间谍活动以及窃取商业秘密。
 <br/>
 <br/>
 联邦调查局反恐处的助理主任比尔（Bill Priestap）说：对一名中共情报人员前所未有的引渡，曝光了中共政府对美国经济间谍活动的直接督管。
 <br/>
 <br/>
 《华盛顿邮报》引述美国官员表示，这是首次有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 被引渡至美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受审.html" target="_blank">
  受审
 </a>
 。
 <br/>
 <br/>
 旅美时事评论员郑浩昌表示，这次事件对中共情报系统冲击很大。2014年虽然美国起诉了5名中共军方黑客，但那5人都在中国境内。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝美方的起诉如果不能落实到具体的人身上，中共是不会觉得痛的。所以这次抓到一个具体的人，杀伤力要大得多了，跟上次美方制裁中共军委装备发展部部长李尚福一样，都有很大的政治意义。〞
 <br/>
 <br/>
 对徐彦钧的指控，正值川普政府对中共采取更强硬立场。
 <br/>
 <br/>
 10月4号，美国副总统彭斯对中共进行了猛烈抨击，指控中共策划全面窃取美国的科技技术。
 <br/>
 <br/>
 美国华府人权律师叶宁说，川普总统进驻白宫之后，美国政府大幅度修正对华政策。特别是彭斯副总统宣告新的对华政策，中美之间将在政治、经济、意识形态、文化等所有领域全面开战。而徐彦钧被引渡，正是发生在这样的大背景下。
 <br/>
 <br/>
 美国华府人权律师叶宁：〝美国政府已经宣布，美国的脑力、火力绝不允许中共的间谍机关再来偷盗。说明美国确实是感觉到了中共无处不在、咄咄逼人的挑战威胁。原来都是中共用主权这个字来给自己所有的恶行做挡箭牌，现在美国已经到了处处强调要维护自己国家政策受到威胁的主权，那威胁美国主权的实际上只有一个，就是中共嘛！其它国家对美国都不会造成伤筋动骨的影响的。〞
 <br/>
 <br/>
 叶宁表示，过去美国政府对中共不够警觉，让美国吃了比较多的亏。
 <br/>
 <br/>
 叶宁：〝据说美国也是在奥巴马总统执政期间，中国（中共）政府破译了美国联邦政府机关和州政府机关所有人员的秘密档案，据说当时中共的头头脑脑们都高兴的快要晕过去了。所以美国在和中共这种不对称的全面的战争当中，美国是比较吃亏的。这是因为美国长期以来被利益集团掐住了脖子，美国的对华政策是被人劫持的，这种情况已经维持几十年了。如果没有川普总统出现的话，美国这种情况再维持2、30年，美国也就没有了。0254现在美国终于觉醒了。〞
 <br/>
 <br/>
 美国司法部9月宣布逮捕一名居住在芝加哥的中国公民季超群（译音，Ji Chaoqun），他被控非法充当中共政府代理人，也与江苏省国安厅有关联。
 <br/>
 <br/>
 郑浩昌：〝中国的东部沿海省份，上海的国安是最多的，其次是浙江，然后才到江苏，从这次抓的人来看，这三块地方很可能都是面向美国的。以后美国如果用点心，估计会抓到分别来自上海和浙江国安局的特务。〞
 <br/>
 <br/>
 美国司法部助理部长约翰・戴默斯（John C. Demers）表示，徐彦钧案不是孤立事件。〝这是以美国为代价发展中国整体经济政策的一部分。我们不能容忍一个国家窃取我们的弹药和智慧成果。我们不能容忍一个不劳而获的国家。〞
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/钟元
