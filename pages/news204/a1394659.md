### 大陆多地开发商降价求售 引爆业主怒火
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】十一期间，大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房地产.html" target="_blank">
  房地产
 </a>
 市场极不平静。由于多个省市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开发商.html" target="_blank">
  开发商
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降价.html" target="_blank">
  降价
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_求售.html" target="_blank">
  求售
 </a>
 ，引来原有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_业主.html" target="_blank">
  业主
 </a>
 不满，各地接连爆发抗议活动，甚至要求退房。
 <br/>
 <br/>
 10月2号，景德镇某楼盘数十名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_业主.html" target="_blank">
  业主
 </a>
 围堵售楼部，高喊要退房，原因是房价下跌，且一天一个价。
 <br/>
 <br/>
 3号，上海浦东惠南碧桂园南郡项目，突然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降价.html" target="_blank">
  降价
 </a>
 20%多，房价一夜之间从均价3万5一平米暴跌到2万6到2万8万一平米，引起业主极大不满。
 <br/>
 <br/>
 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房地产.html" target="_blank">
  房地产
 </a>
 仲介王女士：〝是一平米3万5，然后有特价房，特价房便宜一些。现在上海这边不是限价吗？以前凡是碧桂园的房子都是带精装修出售的，现在它就按正常的备案价来卖。所以现在它们就毛坯出售的。其实说句良心话，上海房产市场的泡沫太多，因为它从15年年底到16年，一下子涨了2、3万上来。〞
 <br/>
 <br/>
 江西上饶碧桂园因把1万元一平米的精装修房，降到7000元一平米的毛坯，引起大量业主维权，打砸售楼处。有业主称，当初买房时，碧桂园称不会推出毛坯房，如今却食言。
 <br/>
 <br/>
 江西上饶房地产仲介王女士：〝碧桂园是降了很多。之前他买了精装房1万多，现在一下子跌了几千，心里不平衡嘛！〞
 <br/>
 <br/>
 河南平顶山碧桂园应国府，也因在十一期间降价8折出售，被业主围攻售楼部。
 <br/>
 <br/>
 促销、出货，似乎成了目前各地楼市的主旋律。深圳、北京多个楼盘都下调价格。上海不仅有降价的，还有送车的。二线城市，如合肥文一豪门金地、合肥院子、杭州滨江新希望新城未来海岸部分二期房源等，都在降价
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_求售.html" target="_blank">
  求售
 </a>
 。一些三、四线城市的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开发商.html" target="_blank">
  开发商
 </a>
 也在降价促销。
 <br/>
 <br/>
 合肥房地产仲介张女士：〝（合肥文一豪门金地）业主维权是因为一期跟二期价格差的比较多。一期的价格应该是在2万2、2万3的样子，二期大概就在1万9。〞
 <br/>
 <br/>
 据中信建投证券数据，今年一手房成交量创下至少2014年以来的最低水平。
 <br/>
 <br/>
 有大陆房产分析师向媒体表示，近期楼市价格下滑，除了中美贸易摩擦升级，更重要的是当局在压抑楼价。
 <br/>
 <br/>
 新唐人记者熊斌、王子琦、施宜君采访报导
</div>
