### 冯小刚回应被罚20亿：太少了，怎么也得罚100亿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 因逃税案被罚8.8亿人民币后，网曝导演
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冯小刚.html" target="_blank">
  冯小刚
 </a>
 也被罚20亿人民币，近期公布细节。对此，冯小刚宣传总监称，该消息为假新闻。冯小刚也称这是大V的造谣行为，〝20亿太少了，怎么也得罚100亿〞。
 <br/>
 <br/>
 陆媒19日报导，知名财经大V、微博认证北京银泰安投资顾问有限公司董事长的〝老c队长〞在微博爆料，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冯小刚.html" target="_blank">
  冯小刚
 </a>
 被罚款20亿元人民币，近期国税总局也将公布出来，可以转发500次。
 <br/>
 <br/>
 因中共规定网帖转发数量超过500条，如果被认定是谣言，就会构成诽谤罪。截图显示，上述消息转发数早已超过500次，很多媒体也跟进了报导。
 <br/>
 <br/>
 19日当天，《华夏日报》记者向〝老C队长〞发信息求证。〝老C队长〞回复说，该消息来源于周末时与朋友聚会，他的朋友来自于国税总局和财政部，称冯小刚被罚款20亿元人民币，近期国税总局也将公布出来。
 <br/>
 <br/>
 〝老C队长〞的说辞令人感觉消息来源似乎非常可靠。
 <br/>
 <br/>
 但一位认证为某媒体总编辑的大V转发上述消息并辟谣：20亿数字绝对是假的！
 <br/>
 <br/>
 这位名叫凯雷的粉丝数50多万，认证信息为香港大公文汇传媒集团北京新闻中心副总编辑，头条文章作者，简介是香港文汇媒体人。这一消息来源似乎也很可靠。
 <br/>
 <br/>
 当天冯小刚的宣传总监也回应称，冯小刚被罚款20亿元人民币为假新闻。冯小刚本人则反讽说，20亿太少了，怎么也得罚100亿。你说这些大号儿，他们真敢顶风作案啊，正收拾他们呢，就敢胡说八道造这谣。
 <br/>
 <br/>
 尽管如此，但仍有不少网友表示，之前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 在面对税收问题时，其工作室也是当即澄清为虚假消息，还要追究造谣者的刑事责任。之后的结果大家也都看到了。
 <br/>
 <br/>
 所以，这次冯小刚方面的澄清，许多网友依然不信。在官方没有公布之前，所有的传闻都有可能是谣言，也有可能是遥遥领先的预言。
 <br/>
 <br/>
 今年5月，冯小刚导演的《手机2》开拍。前央视主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 认为，该片影射他本人有作风问题，因而怒怼娱乐圈。致使在《手机2》中饰演武月的范冰冰最先被调查。10月3日，国税总局公布，范冰冰因逃税罚款8.84亿人民币。
 <br/>
 <br/>
 然而崔永元针对的并非范冰冰一个人。他爆料称，利用〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞偷税的演员并非是一个人，而是〝一个团伙〞。冯小刚因此也陷入舆论旋涡。
 <br/>
 <br/>
 10月中旬，中国多个网媒报导，官方调查组进驻北京电视艺术中心（北视）后，疑似冯小刚因受〝惊吓〞，连夜补缴百万人民币管理费。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：范铭）
