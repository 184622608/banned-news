### 中国女客泰国PP岛跳快艇 惨遭螺旋桨砍腿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】近日，一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国女游客.html" target="_blank">
  中国女游客
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_泰国.html" target="_blank">
  泰国
 </a>
 游览时，从快艇船尾跳下，被螺旋桨打伤腿部而送往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_PP岛.html" target="_blank">
  PP岛
 </a>
 医院救治。
 <br/>
 <br/>
 泰媒报导说，12月1日，一名39岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国女游客.html" target="_blank">
  中国女游客
 </a>
 ，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_PP岛.html" target="_blank">
  PP岛
 </a>
 附近的Pilae海湾被快艇螺旋桨打伤腿部，被送往PP岛医院救治，后因伤口面积大且失血过多，被紧急转至普吉哇栖拉医院治疗。
 <br/>
 <br/>
 快艇的主人描述，事发前该快艇从布吉出发，接载20多人的旅行团到Pilae海湾游览。当船隻正在后退准备停靠让游客下船游览之际，该女游客就从船尾跳下，不料被螺旋桨打伤腿部，于是立即将伤者送到医院救治。
 <br/>
 <br/>
 事发后，警方将船上2名工作人员，以及该旅游团导游带返警局作进一步调查。
 <br/>
 <br/>
 （责任编辑：文馨）
