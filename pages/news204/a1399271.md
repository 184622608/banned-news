### 近万自媒体账户被删  大陆网民不满
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 进一步收紧网路审查。近期突然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关闭.html" target="_blank">
  关闭
 </a>
 9800多个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自媒体.html" target="_blank">
  自媒体
 </a>
 帐号，并严重警告腾讯微信和新浪微博等自媒体平台。此举引发大陆网民强烈不满。
 <br/>
 <br/>
 11月12日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 网信办在一份声明中称，近期对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自媒体.html" target="_blank">
  自媒体
 </a>
 账号开展了集中清理整治专项行动。目前已经全网处理9800余个账号，约谈腾讯微信和新浪微博。
 <br/>
 <br/>
 上述行动从10月20日起，已全网处置〝唐纳德说〞、〝傅首尔〞〝、紫竹张先生〞、〝有束光〞、〝万能福利吧〞、〝野史秘闻〞、〝深夜视频〞等9800多个自媒体账号。这是中共自今年6月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关闭.html" target="_blank">
  关闭
 </a>
 微博、微信、网易和百度上的十几个娱乐新闻账号以来，针对社交媒体的又一次整肃。
 <br/>
 <br/>
 网信办称，在清理的账户中，包括〝传播政治类有害信息、恶意篡改党史国史、诋毁英雄人物和抹黑国家形象〞。网信办官员还表示，下一步将继续加大依法管网和治网的力度，对〝一些屡教不改和继续从事危害社会、扰乱正常秩序的自媒体违规行为坚决从严查处，决不姑息。〞
 <br/>
 <br/>
 大陆〝自媒体〞账户近年来激增，其中包括专业性很强的调查新闻平台，也有的专门制作有关名人花边新闻。一些自媒体因给网民提供有别于官方媒体的内容或新奇的信息，而颇受网民欢迎。
 <br/>
 <br/>
 网际网路平台〝NGO发展交流网〞专门致力于观察社会问题，它的三个微信账号也被删除。网站的一份声明表示：目前是一个网路账户被删除的时代，他们处于无处生存的状态。但他们将继续设法表达自己的立场。
 <br/>
 <br/>
 新浪微博本月发布四则声明，抨击一些自媒体账户，散发有害的政治信息或粗俗的黄色内容；腾讯微信也发表声明表示，它正在加强自我审查，打击有害政治信息、黄色低俗内容。
 <br/>
 <br/>
 网路评论人士指出，中共的网路监控一直存在，这次的清肃行动中对一些目标账户的处理过于严苛。
 <br/>
 <br/>
 在美国华人节目主持人观山对自由亚洲电台表示，当局不应该删除专业性自媒体，许多多年来从事自媒体新闻业务的人一刹那失去了职业，而且这也触及到了许多在背后支持这些自媒体的利益。
 <br/>
 <br/>
 他说：自媒体中存在虚假信息、低俗内容和侵犯知识产权信息等的说法在一定程度上正确的。但这需要通过执法和有关技术来处理，而不是大规模封杀近一万个自媒体账户。这种做法不但显示了中共做事的作风，也揭示了社会矛盾的激化。这种做法只能进一步激化矛盾，因为在许多自媒体账号的背后都存在一定的利益。自媒体这些年来已经变成了产业，你每封杀一个账户，就埋下了矛盾和仇恨的种子。〞
 <br/>
 <br/>
 目前在美国的大陆作家王清营对自由亚洲电台表示，他在国内时的自媒体帐号也遭删除，〝共产党就是做这种事，他为了维持自己的政权，不断地制造恐怖。
 <br/>
 <br/>
 大陆网民对当局在三周内关闭近一万个社交网站自媒体帐号的打压举措表示强烈不满。一些网民认为，被封杀的自媒体账户是被冤枉了；另一些网民则忧虑此后自媒体的前景。
 <br/>
 <br/>
 网民表示，这些自媒体平台的运营者是在做有意义的事，关闭这些自媒体账户并不能阻止人们的思考，或阻止他们想方设法获取信息的努力，而只能促使他们设法从别的渠道获得信息。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
