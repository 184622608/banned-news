### 孟宏伟与一家公司关系密切 国际刑警〝特别关注〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】日前，中共官方宣布神秘失踪多日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警.html" target="_blank">
  国际刑警
 </a>
 组织（Interpol）主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 涉嫌违法，正接受国家监委监察调查。有外媒称，孟宏伟一直与一家专门从事网路安全的中国公司保持密切的联系，引发国际刑警组织的担忧和特别关注。
 <br/>
 <br/>
 前中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，在2016年11月被任命为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警.html" target="_blank">
  国际刑警
 </a>
 组织主席。据欧洲第一电台称，孟宏伟与一家中国专门从事网路安全的大型公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美亚柏科.html" target="_blank">
  美亚柏科
 </a>
 （Meiya Pico）有着密切联系，一直致力于加强国际刑警组织与美亚柏科的战略合作。在美亚柏科的网站上，可以看到孟宏伟与美亚柏科老板的合影，但从10月8日早上开始，这个页面已经完全消失。
 <br/>
 <br/>
 报导称，美亚柏科与国际刑警组织全球创新综合机构（IGCI）形成直接竞争，让这样一家中国公司渗透国际刑警组织，会对其构成威胁。这引起了国际刑警组织的担忧，孟宏伟也一直是国际刑警组织内部特别关注的对象。
 <br/>
 <br/>
 报导说，IGCI的日本总局长表示，如果美亚柏科公司成为国际刑警组织创新机构的主要参与者，那么他会直接提出辞职。
 <br/>
 <br/>
 据美亚柏科官网介绍，该公司成立于1999年，总部位于厦门软体园二期，主要服务于国内各级司法机关以及行政执法部门，目前在中国刑侦器材市场的占有率为45%。
 <br/>
 <br/>
 此前有媒体报导，今年8月份美亚柏科曾向俄罗斯销售被称为〝魔方〞的智能手机和移动设备数据提取系统。据称该系统能够在几分钟之内轻松破解脸书、推特、Telegram等各种社交媒体账号内的信息，解码Skype等各种即时通讯软体中的信息存档，也能够获取几乎所有已知智能手机和各种硬碟中的内容。
 <br/>
 <br/>
 今年5月15日在北京举办的国际警备警用装备博览会上，美亚柏科也曾展示其生产的XDH-CF-5600手机扫描仪。这一手机扫描仪只需几秒钟就可以破解智能手机密码，提取用户数据，令不少海外华人担心回国时，被海关查出曾发表过批评中共的言论而遭到报复。
 <br/>
 <br/>
 孟宏伟自9月底在法国登上一架飞往中国的班机后，便消失超过10天。10月7日晚间，中纪委在网上公布孟宏伟正在接受调查，数小时后，国际刑警组织宣布收到孟宏伟主席的辞职信，并〝立即生效〞。
 <br/>
 <br/>
 中共的声明中表示孟宏伟〝收受贿赂、涉嫌违法〞，但并未透露更多细节。声明也没有说明这些指控是指向他的公安部副部长职能，还是他在国际刑警组织的主席职能。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：文慧）
</div>
