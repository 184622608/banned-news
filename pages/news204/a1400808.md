### 女儿被〝富二代〞男友骗8万 父亲设计〝骗〞回
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】四川眉山市青神县女子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小琪.html" target="_blank">
  小琪
 </a>
 ，在网上认识了自称〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富二代.html" target="_blank">
  富二代
 </a>
 〞的男子黄峰后，被骗走8万元（人民币，下同）。小琪的父亲李涛心生一计，以千万业务为诱惑，成功将已跑路的黄峰和8万元骗了回来。
 <br/>
 <br/>
 11月25日，成都传媒从眉山市青神县警局了解到，目前，犯罪嫌疑人黄峰，因涉嫌诈骗已被刑拘，案件正在进一步审理之中。
 <br/>
 <br/>
 该
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诈骗案.html" target="_blank">
  诈骗案
 </a>
 发生于11月1日，在一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_社交平台.html" target="_blank">
  社交平台
 </a>
 做
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_主播.html" target="_blank">
  主播
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小琪.html" target="_blank">
  小琪
 </a>
 ，认识了自称香港〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富二代.html" target="_blank">
  富二代
 </a>
 〞的男子黄峰，网路上，黄峰俊朗帅气，出手阔绰，在经过简短的聊天后，两人确定了情侣关系。
 <br/>
 <br/>
 次日，黄峰便来到眉山市青神县与小琪见面，期间，黄峰以要给小琪买房而自己公司账户冻结为由，先后让小琪偷偷从父亲李涛的对公司账户里，转走了4万元。当收到银行简讯时，李涛才知道女儿借出去了钱。
 <br/>
 <br/>
 李涛说，他在青神经营着一家公司，平时忙不过来时会让女儿帮忙转账。因当时李涛对黄峰的印象不错，再加上黄峰说第二天就还钱，李涛并没有多加怀疑。
 <br/>
 <br/>
 不料想，借走4万元之后，第二天早上黄峰便以需要去接父母为由，带着小琪去了成都。临走时黄峰还怂恿小琪带上了李涛公司的银行卡账户及转账所需要用到的K宝。
 <br/>
 <br/>
 在成都期间，黄峰以钱没有到账为由，让小琪又转了李涛公司的4万元资金。当发现公司的对公账户没钱了，再加上黄峰迟迟未还钱，李涛意识到不对，但又不能确定黄峰是骗子，便要求他回青神，以防跑路。
 <br/>
 <br/>
 为了稳住黄峰，李涛说，自己最近有一个流水上千万的大项目，需要二人帮忙拿货。为了取得信任，他还把签的单子给黄峰看。见有利可图，黄峰爽快的答应回到青神。
 <br/>
 <br/>
 回去之后，又等了几天之后，看到黄峰所谓的父母一直没到时，李涛便报了警。
 <br/>
 <br/>
 11月13日，黄峰被捕，刚开始他不承认自己是诈骗。之后，警方观察黄峰提供给小琪的证明自己是富二代的图片时，发现一张豪车自拍图片角落里有个很小的二维码。
 <br/>
 <br/>
 一扫进去一看是个提供各种假图片的公众号，其中一张和豪车一模一样的图片。最后，黄峰终于承认了自己诈骗，以及来自江苏是一个无业游民的事实。
 <br/>
 <br/>
 黄峰还供认在外地多次冒充〝富二代〞行骗，约诈骗11万元，大多打赏给了女主播。
 <br/>
 <br/>
 近年来，中国各类诈骗案泛滥成灾。根据360研究报告显示，粗略估计，仅2015年网路诈骗产业链上至少就有160万从业者，〝年产值〞超过1152亿元。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
