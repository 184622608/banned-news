### 汉黄陂强拆农贸市场 爆发警民冲突
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖北武汉.html" target="_blank">
  湖北武汉
 </a>
 黄陂区泡桐街
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农贸市场.html" target="_blank">
  农贸市场
 </a>
 ，11月22号，遭到当地政府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 ，商户和警察爆发激烈冲突，多人被警方打伤，现场围观群众估计有上千人，成功阻止了强拆。
 <br/>
 <br/>
 22号早上8时左右，大约二、三百名身穿黑色制服、头戴钢盔、手持盾牌的特警、城管、交警、防暴警察等，还有四、五辆防暴警车以及两台挖掘机到达位于泡桐大道的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农贸市场.html" target="_blank">
  农贸市场
 </a>
 进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 。
 <br/>
 <br/>
 网传视频显示，警方与商户爆发激烈冲突，围观群众向警察投掷石块、灰包，并用棍棒与防暴警察对抗。
 <br/>
 <br/>
 泡桐街居民王先生：〝就是无协调，政府就要把老菜场拆了，到新菜场去，老菜场的人就不愿意，那天有政府的县里、镇里，强行的开挖机来要把这个菜市场拆掉，当地人就不愿意，发生冲突，(围观)上千的人有，打伤9个，有男的、有女的，那天就没拆。〞
 <br/>
 <br/>
 当地居民还透露，农贸市场拆迁涉及到很多人的利益，首先是当地江姓和陈姓两大家族的利益没摆平，官方处置又不公平，引发百姓以及两大家族对官方不满。
 <br/>
 <br/>
 泡桐街居民蔡先生：〝城市化了嘛，乡下人都搬到城里来住，镇上就越来越发达，农贸市场也就不够用了，农贸市场要拆迁，盖一个更大的，从甲地搬到乙地，对官方不满嘛，矛盾激化到这种程度，肯定是不敢轻易的强拆了，如果强制压一方问题就多了，还是要和和平平的坐下来谈，对不对。〞
 <br/>
 <br/>
 据了解，泡桐农贸市场建立于上世纪八十年代中期，当时政府以〝致富一条街〞的名义征收土地，未有任何赔偿，当地村民改以做生意维生，此地成了繁华的商业街，现在政府要搬迁市场到偏僻的地方，村民担心生意受到影响，因此拒绝搬迁。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
