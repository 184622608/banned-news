### 【禁闻】公民状告外交部遭拒 中国版图竟涉密？
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】中国退伍军人殷敏鸿，因查问中共是否刻意隐瞒唐努乌梁海遭俄罗斯占领一事，向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交部.html" target="_blank">
  外交部
 </a>
 申请政府讯息公开遭拒绝，随后他向法院状告外交部，却屡诉屡败。来看他的遭遇。
 <br/>
 <br/>
 去年8月，中国退伍军人、独立研究人员殷敏鸿向中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交部.html" target="_blank">
  外交部
 </a>
 申请政府信息公开，查问1949年以后，中共政府有没有签署过放弃唐努乌梁海地区的协议？
 <br/>
 <br/>
 9月，中共外交部声称，殷敏鸿申请的信息涉密，不属于政府信息公开范畴，拒绝答覆殷敏鸿。回函还称，如果不服，可申请行政覆议或向法院提起行政诉讼。
 <br/>
 <br/>
 今年3月7号，殷敏鸿向法院提起诉讼。北京市第三中级法院裁定不予立案，殷敏鸿再向北京市高级法院上诉。而北京高院在9月10号裁定，驳回殷敏鸿上诉，维持一审裁定，不予立案。
 <br/>
 <br/>
 中国的〝律酷法律〞服务有限公司对此进行解读：〝外交部答覆：中国领土版图涉密，中国人没有权利知道；北京高院回答：外交部答覆正确，我们不管。〞
 <br/>
 <br/>
 此事最近引起多家媒体关注。
 <br/>
 <br/>
 安徽居民萧先生：〝（殷敏鸿）觉得自己姓‘赵’，是中华人民共和国的一员，实际上不是的，他是中华人民共和国的奴隶们的一员。你奴隶要知道奴隶主有多少钱，庄园有多宽，这个没意思的。〞
 <br/>
 <br/>
 安徽居民萧先生表示，《宪法》规定，中国是工人阶级领导、以工农联盟为基础。那么请问现在国民到底是主人还是奴隶呢？
 <br/>
 <br/>
 大陆律师谢燕益表示，根据《宪法》第一条和第二条，中国的一切权力属于人民。因此主权在民，人民有知情权等。另外，《政府信息公开条例》也明确规定，一般政务，凡是没有所谓涉密的，公开，是一般原则，不公开，是作为特例。
 <br/>
 <br/>
 大陆律师谢燕益：〝所以外交部是以所谓涉密为由，不予答覆的话，它必须向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公民.html" target="_blank">
  公民
 </a>
 提供涉密所谓的认定的文件，而且作为公民还有权利，对这个是否涉密按照《国家保密法》进行行政覆议，或者行政诉讼。因为有些政府部门不排除它会渎职、滥用权力。法院不受理或者最后进行裁决，或者不会起诉，它也是要有一个根据的。〞
 <br/>
 <br/>
 谢燕益强调，按照《行政诉讼法》，举证责任倒置。因此在该案中，要拿出法律依据的是公权部门。
 <br/>
 <br/>
 谢燕益：〝政府没有法律依据则禁止作为。公民的权利是天赋人权，所以这个不是说需要我们还要拿出什么法律依据来，现在是公权部门要向我们拿出法律依据来。这不仅是在《宪法》上有规定的，而且这是《自然法》的逻辑。不要搞颠倒了。〞
 <br/>
 <br/>
 根据中国《刑法》，非法获取国家秘密者，可处3年以下有期徒刑，情节严重的，处3年以上7年以下有期徒刑。
 <br/>
 <br/>
 〝律酷法律〞反讽提醒：某些人小心了，如果你知道中国实际版图，是否应该被追究刑事责任呢？
 <br/>
 <br/>
 而对于中共大肆宣传的中国领土〝一点都不能少〞，网友嘲讽：怎么？现在是不能说吗？
 <br/>
 <br/>
 据了解，唐努乌梁海在历史上属于中国领土。中共夺权之前，曾被沙俄和苏联非法侵占，但中国并未在法律上放弃其领土主权。
 <br/>
 <br/>
 1999年，时任中共党魁江泽民和俄罗斯签定《议定书》，将被俄国侵占的100多万平方公里领土，永久划给俄罗斯。据说其中包括唐努乌梁海。
 <br/>
 <br/>
 外界认为，江泽民卖国，正是中共外交极力掩盖的公开秘密。
 <br/>
 <br/>
 萧先生：〝他们卖国肯定是偷偷的卖啦！49年以后所签的那些协议没有一个是拿得出来给大家看的。〞
 <br/>
 <br/>
 2004年，香港资深记者程翔曾发文，指出江泽民和俄罗斯签订的议定书，构成今后中、俄边界的法律文件。这桩大事一直被江隐瞒，直到俄方公布条约，中国人才得知江悄悄出卖了大片国土。
 <br/>
 <br/>
 2006年，程翔被中共以间谍罪逮捕，判刑5年。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/钟元
