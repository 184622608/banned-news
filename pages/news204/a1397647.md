### 湖南奇葩承诺书：杜绝外出吃饭叫美女坐陪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】湖南省衡阳市某政府部门人员集体签署的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奇葩承诺书.html" target="_blank">
  奇葩承诺书
 </a>
 ，日前在大陆网路爆红。承诺书中有一条〝杜绝外出吃饭叫美女坐陪问题〞，被网友嘲讽，这明显是〝此地无银三百两〞。
 <br/>
 <br/>
 近日，大陆网路传出一份湖南省衡阳市某政府部门人员集体签署的承诺书。承诺书显示，为迎接省委巡视组的检查，科室全体工作人员特作出以下承诺：
 <br/>
 <br/>
 〝一、坚决杜绝开发商请客吃饭与办证速度挂钩问题，杜绝一切有业务往来的饭局。二、杜绝交100元钱才能办证的问题。三、杜绝外出吃饭叫美女坐陪问题。四、上下班坚决不迟到早退，若临时请假必须事前写好请假条。〞
 <br/>
 <br/>
 网传承诺书上盖有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_衡阳市不动产登记中心.html" target="_blank">
  衡阳市不动产登记中心
 </a>
 权籍调查科〞公章，承诺时间是10月29日，共有10人在承诺书上签字。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-1/p9106721a273045665.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-1/p9106721a273045665-ss.jpg"/>
   <br/>
   <font size="-1">
    衡阳市不动产登记中心权籍调查科人员集体签署的
    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奇葩承诺书.html" target="_blank">
     奇葩承诺书
    </a>
    。（网络截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 这份奇葩承诺书一经传出，立即引发网友热议。知名媒体人褚朝新在其微信公号上发文称：〝这个承诺书的内容，让人有点哭笑不得，不知道是该赞他们自查得深入了灵魂，还是该批他们平日里任性无度。〞
 <br/>
 <br/>
 也有网民说，〝这是工作人员该遵守的一般常识，用得着专门用白纸黑字签名承诺吗？〞。还有网民嘲讽，这明显是〝此地无银三百两〞。
 <br/>
 <br/>
 据陆媒报导，衡阳市不动产登记中心11月1日上午对此事件进行通报。通报称，该中心于10月18日召开科室主要负责人会议，通报了全国不动产登记窗口作风问题的5个典型案例。并要求各科室自查。然而，权籍调查科会后交差应付、敷衍了事，从而导致网传奇葩承诺书的出现。
 <br/>
 <br/>
 该中心负责人还批评权籍调查科科长唐某〝觉悟不高〞，给中心带来极大的负面影响，对其予以诫勉谈话处理。并表示纪检组已成立专人组调查核实，如发现有违规违纪行为，坚决处理。
 <br/>
 <br/>
 不过，有网民质疑，纪检调查恐怕上查不到〝美女〞层级。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
