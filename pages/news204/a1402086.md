### 疑报复社会 四川男巴士引爆炸弹酿17伤（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】中国四川乐山市夹江县日前发生公交车被炸事件，至少17人在事件中受伤，其中有人伤势严重，被炸巴士的闭路监控视频片段已在网络上曝光。点燃爆炸物的男子据传曾因土地问题被警方拘押，此举动机疑似因心怀愤怒而报复社会。
 <br/>
 <br/>
 四川夹江县3路公交车被炸事件发生于当地时间12月5日下午5点31分。从网络上流传出的巴士闭路电视画面可看到，排队的一些乘客陆续登上公交车后，一名身穿蓝白色外套的男子，手中抱持着一个疑似装有土制炸弹的包裹，也从巴士前门登车。他一上车立即在车门口用一只打火机点燃了手中的爆炸物并扔入车厢之中，紧接着就发生了爆炸，整辆公交车被一片白光和白色烟雾所笼罩。

 <br/>
 据陆媒报导，公交车被炸后，车头玻璃龟裂，两侧的车窗玻璃粉碎并喷射出车外。巴士内的杂物和铁枝则被炸得飞到数米以外，还有一辆靠近公交车的私家车受到玻璃碎片的波及而受损。现场一片狼藉，附近有飞溅的血迹，多名女伤者因惊吓和伤痛而尖叫或嚎哭。
 <br/>
 <br/>
 这起爆炸事件至少造成17人受伤，其中有一名男乘客背部被炸至血肉模糊，内脏亦流出体外。在受伤的乘客中，有一名女童手部被炸得血肉模糊，鲜血直流。还有消息指一名女童在这次爆炸中失去听觉。
 <br/>
 <br/>
 这起爆炸事件在网络上公开后，有自称知情的网民在社交媒体上透露，肇事的男子因土地纠纷曾被警方拘留多天。不少网民怀疑这起事件的动机是报复社会，也有网友认为这个事件是恐怖袭击。
 <br/>
 <br/>
 当地警方事后仅在通报中承认一辆3路公交车发生了爆炸，但对于有人引爆炸弹的说法却避而不提，只声称事件的起因仍在调查之中。
