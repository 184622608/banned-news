### 刘强东案牵出另一富商 受刘指使骗受害女赴宴
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】中国电商巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 涉性侵案，更多细节流出。近日陆媒称，中国知名富商姚其涌涉刘强东案。事发当晚，刘要求姚确保涉事女大学生参加晚宴之后，姚骗女主参加晚宴，并将其特意安排在刘强东身边。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 3个月前在美国涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 被捕入狱，16个小时后获释。根据美国警方公布的报告，刘强东在美性侵案属于〝一级性犯罪：强奸既遂〞。
 <br/>
 <br/>
 美国检方表示，明尼阿波利斯警察局已经完成调查，并已将案件移交给Hennepin县检察官办公室。检方目前正在考虑是否提起刑事指控。
 <br/>
 <br/>
 刘强东性侵案发当晚，在一间日本餐厅举行晚宴，21岁的涉事中国女大学生是参与者之一。
 <br/>
 <br/>
 大陆门户网站新浪网12月1日称，涉事女主最初以为，当晚的晚宴是为了向她这样的志愿者致敬，但实际上，她却发现自己被十几名中国企业高管包围，他们都是男性，来到这里是为了参加一个博士管理项目。这位女主曾为他们做过志愿服务。
 <br/>
 <br/>
 据涉事女主与朋友的聊天记录及其他女学生的口供，当晚，参加晚宴的有中国知名商人姚其涌，他曾邀请涉事女主参加晚宴。姚其涌让她坐在刘强东的身旁，并称刘是中国最富有的人之一。
 <br/>
 <br/>
 据美国Florin Roebig律师事务所律师透露，刘强东曾经要求姚其涌确保这名女学生参加晚宴。姚其涌通过电子邮件回应。
 <br/>
 <br/>
 公开资料显示，姚其涌是深圳宏兆企业集团董事局主席，该企业集团是一家股权投资和基金管理集团。
 <br/>
 <br/>
 涉事女主口供中说，她那周早些时候担任慢跑活动的志愿者协调员，并因此遇到姚其涌。姚其涌在几次谈话中赢得了她的信任，并为她提供了一个毕业后，到他公司工作的机会。
 <br/>
 <br/>
 随后，姚其涌邀请她参加晚宴。但涉事女主表示，她是被骗去晚宴的。
 <br/>
 <br/>
 涉事女主透露，晚宴上，她被〝强迫喝酒〞，刘强东和其他男人，在餐桌上反覆使用〝干杯〞这样的短语向她敬酒。
 <br/>
 <br/>
 而受邀赴宴的男大学生都在另一张桌子上，跟中国高管们的助理们坐在一起。
 <br/>
 <br/>
 据路透社披露，晚宴接近尾声时，女主说喝醉了要回家，她和刘强东还有一位姓张的私人女助理(Alice Zhang)一起上了一辆黑色轿车。
 <br/>
 <br/>
 根据警方问讯录音，女主说这名张姓助理起初试图坐在她和刘强东之间，但刘强东对助理说:〝别碍事〞。
 <br/>
 <br/>
 之后刘强东开始在后座上抚摸她，还发生强脱她衣服的举动。抵达她的住处后，刘强东试图将她拉进浴室，未果后自己洗澡，然后裸身躺在床上。而她已竭尽全力和刘强东对抗，但最终还是被性侵得逞。
 <br/>
 <br/>
 女主与警方的谈话录音显示，事发后，刘强东的私人助理，曾请深圳卓越置业集团有限公司董事局长李华出面调解。
 <br/>
 <br/>
 李华跟刘强东一样是中国高管工商管理博士（DBA）课程学生。该课程学生包括腾讯CEO马化腾。
 <br/>
 <br/>
 今年45岁的刘强东是中国商界
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亿万富豪.html" target="_blank">
  亿万富豪
 </a>
 ，也是中共全国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政协委员.html" target="_blank">
  政协委员
 </a>
 ，并兼任中国民间商会副会长。自8月31日在美国涉嫌性侵案发后，刘强东在连续缺席多项重要场合后，坊间还传出他已退出京东管理层。舆论纷纷猜测，刘强东遭北京〝抛弃〞了。
 <br/>
 <br/>
 (记者李芸报导/责任编辑：李泉)
