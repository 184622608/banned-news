### 中国GDP增速创近10年新低 下行压力持续
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】中共官方宣称，中国第
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三季度.html" target="_blank">
  三季度
 </a>
 经济增长为6.5%，是2009年金融危机以来的最低增幅。
 <br/>
 <br/>
 中共官方周五宣称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三季度.html" target="_blank">
  三季度
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_GDP.html" target="_blank">
  GDP
 </a>
 增长6.5%，低于市场预期的6.6%。业界质疑其中仍有水分，不过这已是2009年一季度以来的最低点。
 <br/>
 <br/>
 资产管理公司WH Ireland首席市场策略师Mike Ingram：〝最重要的原因，是工业生产放缓。从最近的PMI数据能看出，特别是小企业出现了大幅放缓，这是整体拖累数据的原因。〞
 <br/>
 <br/>
 大陆经济〝三驾马车〞中的〝出口〞项，工业出口交货值9月增长11.7%，比上季度略高0.2%。但业界认为，这是制造商赶在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 继续升级之前，抢先把假日订单发货。等于透支了下季度的增长。
 <br/>
 <br/>
 谢田：〝（贸易战）真正造成深刻的影响，是要一两个季度之后。有的分析认为，贸易战的影响至少会让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 GDP减少1.5%。接下来一两季到半年形势会越来越严峻。〞
 <br/>
 <br/>
 三驾马车中的〝消费〞，前三季度对GDP帐面贡献率达到78%，成为经济增长最大动力。但民生银行研究院认为，随着居民收入增速回落，还有房屋支出挤压消费，决定了未来消费难以持续拉动GDP。
 <br/>
 <br/>
 谢田：〝内需不足实际上是收入不足。增加内需，首要做法是增加老百姓的收入。那么应该做的是，废除中共的官僚体制。减轻财政支出以后，减轻税负，让老百姓手中有钱。〞
 <br/>
 <br/>
 《路透社》采访的几名大陆专家也认为，北京应该减税降费，刺激民间经济活力，拉动经济良性增长。
 <br/>
 <br/>
 不过中共最新公布的数据显示，今年前三季度，全国财政收入同比增长8.7%，增速超过GDP。
 <br/>
 <br/>
 谢田：〝不管中共怎么去说，经济在下滑，财政收入在大幅增长，一定是政府和民间争利。这实际上就是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 ，国富民穷的一个表现。〞
 <br/>
 <br/>
 外界担忧，随着经济下行压力加大，中共会放松对债务规模的控制，启动宽松的货币政策，来拉动三驾马车中的〝投资〞增长。外界高度担忧，这会让高危的地方债和企业债，规模继续扩大。
 <br/>
 <br/>
 谢田：〝这是真正的饮鸩止渴。过度借贷造成中国各级地方政府债务迅速的增加，产能过剩，我们都已经看到了。以后的问题越来越大。但现在他们可能顾不了那么多了。〞
 <br/>
 <br/>
 《路透社》预计，中国四季度经济增速会持续放缓，明年的下行压力会更加明显。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
