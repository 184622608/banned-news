### 大陆25岁女在日本被杀 同胞室友〝用菜刀砍人〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】近日，一大陆赴日女研修生（技能实习生）被室友持刀砍死在宿舍内。目前，陆籍女凶手已遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 警方逮捕。
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 媒体报导，当地时间10月27日上午，日本群马县伊势崎市一公寓浴室内，附近居民发现一名25岁的中国女研修生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李雪.html" target="_blank">
  李雪
 </a>
 被30岁的女室友
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘秋颖.html" target="_blank">
  刘秋颖
 </a>
 杀死。28日凌晨，日本警方以涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杀人.html" target="_blank">
  杀人
 </a>
 逮捕刘秋颖。刘承认因生活琐事而〝用菜刀砍了人〞。
 <br/>
 <br/>
 据报导，27日上午11点时许，在日本群马县伊势崎市曲泽町一公寓，附近居民报称二楼的一个房间里〝有女性流血倒地〞。经查住在该房间的大陆技能实习生李雪头部流血倒在浴室内，送医后死亡。
 <br/>
 <br/>
 当地警方称，27日接到了附近居民的报警。附近居民因听到事发房间里传出女性的哭泣声而前去查看，发现李雪穿着衣服倒在浴室地上且头部流血，遂拨打119向当地警方报警。
 <br/>
 <br/>
 当地警方在28日凌晨以涉嫌杀人逮捕了李雪的室友刘秋颖。警方称，命案现场没有外部侵入痕迹，从尸体情况来看，很可能是杀人事件，凶手刘秋颖一度行踪不明。根据另一位女室友的陈述，警方对刘秋颖展开了搜查，警方在伊势崎市内别处找到她。
 <br/>
 <br/>
 刘秋颖对警方承认杀人，杀人动机为〝因生活的事被警告，无法忍耐〞。刘秋颖供述，因为死者李雪在日常生活琐事上对她不满，让她注意，她内心的愤怒忍无可忍。所以案发当天，刘秋颖用一把尖角菜刀砍了李雪头部。刘秋颖承认是〝用菜刀砍了人〞。警方在现场发现了凶器，认为极可能是从厨房顺手拿的菜刀。
 <br/>
 <br/>
 30岁的刘秋颖与李雪同是研修生，今年8月到日本。刘秋颖来自吉林省，李雪来自江苏省，两人一起在伊势崎市内的一家动物用品工厂里工作。
 <br/>
 <br/>
 据悉，现场位于JR两毛线国定站以北约3公里处，附近散布着住宅与田地。
 <br/>
 <br/>
 1980年代初，日本设立〝外国人研修制度〞，大陆一直是日本研修生的最大来源国。1993年，日政府在研修生接收制度的基础上，设立了技能实习生制度，并随后进行了多次修改。
 <br/>
 <br/>
 根据现行的制度，前往日本的务工人员在经过一个月的研修生过渡后，便转为技能实习生，大部分是三年期合同，而且合同期间原则上不能中途回国。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
