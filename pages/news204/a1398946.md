### 郭树清宣布强制增加民企贷款 银行股价应声齐跌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】中共的内外政策导致经济困境，北京再次祭出政治手段挽救危局。民企被定性为〝自己人〞后，各部门争相表态。其中银保监会主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭树清.html" target="_blank">
  郭树清
 </a>
 日前下令银行大幅增加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民企贷款.html" target="_blank">
  民企贷款
 </a>
 ，银行股票随即集体大跌。学者称其为〝瞎折腾〞。
 <br/>
 <br/>
 11月7日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭树清.html" target="_blank">
  郭树清
 </a>
 接受中国财经类党媒采访时，提出增加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民企贷款.html" target="_blank">
  民企贷款
 </a>
 的所谓〝一二五〞目标，即大型银行的新增公司类贷款中，对民企的贷款不低于三分之一；对中小型银行，这一比例是不低于三分之二；争取三年以后整个银行业对民企的贷款占新增公司类贷款的比率不低于50%。
 <br/>
 <br/>
 郭树清在采访中提到，2016年行业新增民企贷款占新增企业贷款17%，存量民企贷款占贷款总额22%。截至今年第三季度末，存量民企贷款占贷款总额25%。
 <br/>
 <br/>
 由此可见，当局在宣布提升民企贷款措施时，〝魄力〞不可谓不大。
 <br/>
 <br/>
 该党媒11月8日报导此消息时称，企业界和金融界都很〝振奋〞。
 <br/>
 <br/>
 不过，9日中国股市一开盘，A股三大股指均低开，银行股全线下跌。截至午盘，招商银行跌近5%，平安银行、工商银行、建设银行、农业银行均跌超3%。保险股中国平安、中国太保也均跌超2%。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-12/p9121811a948771915.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-12/p9121811a948771915-ss.jpg"/>
   <br/>
   <font size="-1">
    郭树清讲话公布的次日，中国大陆的银行股价全线下跌。（网络图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 网友评论，这显示民企并非〝赵家的自己人〞，而是待割的〝韭菜〞，银行都不愿意投资高风险低回报的私企，只愿帮国企〝滚雪球〞。
 <br/>
 <br/>
 有财经媒体报导称，金融机构该项计划将信用资源转移给民企，短期内有助于修复市场情绪。但长期来看，打破国企隐性刚兑难以快速实现，此举会降低投资效率，反而损伤市场信心。
 <br/>
 <br/>
 11日，海外中文网刊出署名文章《别再瞎折腾股市了！》，对当局这个〝拍脑袋〞决定进行了通俗的解读。
 <br/>
 <br/>
 文章说，给民企贷款大幅增加了，给国企的贷款就会减少，那些靠借新还旧过日子的国企和官方融资平台就会爆雷。如果那时候再转头救国企，这边民企又会爆雷。而这种〝踩踏式爆雷〞之后，只能由放贷的银行去扛雷。这就是银行股大跌的原因。
 <br/>
 <br/>
 文章说，现在中国经济亟需的是去槓杆，这时候逼着银行去放贷加槓杆，结果只能是企业和银行一起死。而实体民企真正缺的不是钱，而是利润，现在不对症下药，而是要银行加大放债，等于让银行去买地雷。
 <br/>
 <br/>
 文章感叹，官字里面两个口，一个用来媚上，另一个用来欺下。当局这种〝运动式〞的监管，不去考虑实际后果和风险，而是一味地保持〝政治正确〞。其结果只能是〝半年一变脸〞的〝瞎折腾〞。
 <br/>
 <br/>
 近期以来，自从北京宣布民企是〝自己人〞之后，不只是国务院相关经济部门和地方政府，连司法部、最高检察院、最高法院等部门都争相表态，并出台支持民企的〝20条〞、〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一二五目标.html" target="_blank">
  一二五目标
 </a>
 〞、〝三个100亿〞等等各种花样繁多的〝政策〞。
 <br/>
 <br/>
 而在此前，私营经济一直是官方压制的对象。各部门最新出台的各种政策，包括放贷、减税、谨慎定性〝非法集资〞案件、不把经济犯罪升级为刑事案件等等，反而洩露了此前中共政府打击民企的各种手段。
 <br/>
 <br/>
 在中美贸易战中，中共官方一度宣传称，相对于处处被掣肘的美国民主政府，北京极权政府的〝政策执行力〞是中方的一大〝优势〞。
 <br/>
 <br/>
 不过，很多经济学者认为，共产主义在全世界的实践证明，共产党官僚体制这种保持〝政治正确〞的〝运动式〞执政，完全违背经济规律，共产党的〝计划经济〞最终只能以失败告终。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
