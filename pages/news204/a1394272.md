### 【 禁闻】10月5日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】下面我们来关注一下中国大陆各地民众的维权情况。
 <br/>
 <br/>
 1.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗受害家长.html" target="_blank">
  疫苗受害家长
 </a>
 北京西单募捐再被带离
 <br/>
 <br/>
 大陆一些走投无路的疫苗受害家长，10月3号在北京王府井为孩子募捐医疗费遭警方带走关押后，一些被释放的家长，10月5号又来到北京西单商业街募捐，但再次遭到警方干涉，警方粗暴对待募捐者，抢走募捐牌子，然后把她们全部带走。
 <br/>
 <br/>
 2. 曾狱中受虐 甘肃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_藏人.html" target="_blank">
  藏人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雪努班丹.html" target="_blank">
  雪努班丹
 </a>
 去世
 <br/>
 <br/>
 甘肃玛曲县阿万仓乡藏人牧民雪努班丹，因早前被中共当局指控涉嫌〝带头煽动数起示威抗议活动〞入狱，在狱中遭受严重虐待，心脏部位因多次被狱警用脚猛踢和用重物击打而严重受损，获释后一直卧床不起，9月30号不幸去世。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无锡维权人士.html" target="_blank">
  无锡维权人士
 </a>
 拘留所前拉横幅抗议
 <br/>
 <br/>
 江苏无锡抗强拆维权人士
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李梅芳.html" target="_blank">
  李梅芳
 </a>
 ，10月2号前往无锡市拘留所要求会见因进京上访而被关押的丈夫倪满良，但被拘留所以〝十一长假〞为由拒绝，李梅芳于是在拘留所门口打开写有〝房子被中共抢劫，倪满良被陷害拘留〞的横幅抗议。
 <br/>
 <br/>
 4. 北京新机场回迁房变豆腐渣 村民拟抗议
 <br/>
 <br/>
 北京大兴区榆垡镇各村为新机场建设拆迁三年之后，村民日前拿到了回迁房的钥匙，但发现新房是豆腐渣工程。他们对媒体表示，十一长假结束后，10月8号将到政府部门讨说法。
