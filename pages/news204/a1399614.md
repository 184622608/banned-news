### 美高官:川习会不可能达成全面协议 贸战明年续升级
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】为争取中美〝贸易和解〞，北京日前向华盛顿提交了书面让步清单，但被指老调重弹，依然回避核心问题。美国商务部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗斯.html" target="_blank">
  罗斯
 </a>
 直言，11月底的中美首脑会晤不可能达成协议，中美贸易战明年1月将继续升级。
 <br/>
 <br/>
 路透社11月16日引述川普政府一名官员的消息称，中共提交给美国的书面清单包括142项，分为3类：中方正在解决的问题；愿意继续讨论的问题；拒绝谈判的问题。
 <br/>
 <br/>
 这位官员说，美方不接受中方排除那些〝不可谈判〞的项目，而且对其它项目清单也持怀疑态度，因为中共此前做出的经济和贸易改革承诺，都没有兑现。
 <br/>
 <br/>
 这名官员称，美方仍在研究这份清单，但离G20峰会只有两周左右，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 取得重大突破的期望大大降低。
 <br/>
 <br/>
 而据彭博社同日报导，美国商务部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗斯.html" target="_blank">
  罗斯
 </a>
 直接否定了川习会取得成果的可能性。
 <br/>
 <br/>
 他确认，美方将按原计划，在2019年1月1日把对2,000亿中国商品加征的关税税率从10%提升至25%。原因是，〝在1月前，我们当然无法达成一个全面的正式协议。不可能。〞
 <br/>
 <br/>
 罗斯表示，G20期间的川习会，所能达成的〝最好结果〞是商定一个进一步谈判的〝框架〞。
 <br/>
 <br/>
 此外，英国《金融时报》15日称，美方已暂缓向额外2670亿美元中国商品加征关税的计划。但是，美国贸易代表办公室次日即指该消息不实，强调原计划将维持不变。
 <br/>
 <br/>
 此前有消息指，美方将于2019年初推出针对2670亿美元中国商品的关税计划。
 <br/>
 <br/>
 美国摩根大通中国首席经济学家朱海斌13日表示，中美均无做出实质让步的迹象，预计未来3至6个月中美贸易战将加剧，明年中国经济将面临更多风险。
 <br/>
 <br/>
 美国副总统彭斯13日警告，中方如果不能在保护知识产权、强迫技术转移、限制外企进入中国市场、尊重国际规则及在国际水域航行自由等问题上做出显著的、实质的让步，美方准备在经济、外交和政治上对北京施加更大压力，新冷战就将开启。
 <br/>
 <br/>
 15日，英美媒体披露，中方已回应美方要求，提交了具体的书面让步清单。但文件大多是重申此前已经做出的承诺，仍然回避政府补贴、强制技术转移、盗窃知识产权以及〝中国制造2025〞等美国核心关切。文件对南海问题也只字不提。
 <br/>
 <br/>
 外界普遍认为，中方积极寻求和美国和解，只是一种拖延战术。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：）
