### 两天筹到9亿？网传范冰冰缴纳罚金现场视频
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】10月3日，新华社公布对中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 处以近9亿人民币罚金，免于刑责。10月5日就有网络消息称，范冰冰已将罚金缴纳完毕，还传出据称是范缴完罚金后离开税务局的视频。但消息无法证实，也引发质疑之声。
 <br/>
 <br/>
 10月5日，有认证为〝深圳龙凤盛世文化传媒有限公司董事长〞的新浪微博〝@龙凤魔术师〞声称，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 9亿罚款已经缴纳完毕〞，并附上一段长约6秒的视频。
 <br/>
 <br/>
 视频显示，范冰冰在安保人员的护卫下，由疑似税务机关的地方乘车离开。博文暗示，这是范冰冰缴完9亿税款、滞纳金和罚款后离开的现场影像。
 <br/>
 <br/>
 据大陆社交媒体消息，〝@龙凤魔术师〞真名叫郑龙凤，和范冰冰、李晨过从甚密，经常发布一些他们的消息，也曾屡次发声支持范冰冰。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-7/p9069391a625170531.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-7/p9069391a625170531-ss.jpg"/>
   <br/>
   <font size="-1">
    有微博放出据称是范冰冰缴纳罚金后离开税务局的视频。（网页截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 10月6日，上述视频开始在大陆社交媒体流传，还有陆媒曝光据称是范冰冰之弟范丞丞现身北京机场的一组照片，称其〝心情大好〞。
 <br/>
 <br/>
 对此，有人质疑这些是替范冰冰减压的放风之作，认为税务局和银行对公业务部门10月5日还在休假，范冰冰无法缴纳
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_9亿罚金.html" target="_blank">
  9亿罚金
 </a>
 。
 <br/>
 <br/>
 但也有人认为，不排除〝特事特办〞的可能。新华社公布对范冰冰的处罚，也是在〝十一〞休假期间。
 <br/>
 <br/>
 新华社10月3日报导，范冰冰因〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞行为涉及逃漏税，被处罚补缴税款及滞纳金、罚款共8亿8394.6万元，若在限期内缴清则可免于追究刑责。
 <br/>
 <br/>
 报导没有透露范冰冰缴纳罚金的最后期限。但有网络消息指，最后日期是12月31日。
 <br/>
 <br/>
 对于范冰冰能否在短期内筹足近9亿罚金，有陆媒盘点其近几年的收入，认为其〝勉强可以承受〞。但也有消息指，范冰冰身价数十亿，9亿罚金不成问题。
 <br/>
 <br/>
 网上传出各种消息指，范冰冰从8月起就已经在抛售房产，筹措罚金。
 <br/>
 <br/>
 港媒指，当局借范冰冰案整肃娱乐圈，收缴被曾庆红之弟曾庆淮家族把持的利益地盘。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
