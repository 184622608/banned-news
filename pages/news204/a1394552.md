### 美媒:拿下孟宏伟是北京大胆举措 但令渗透国际组织遇挫
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】日前，中共公安部副部长兼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警主席.html" target="_blank">
  国际刑警主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 落马，引发一场国际风波。《纽约时报》10月8日报导称，拿下国际组织的首脑，是北京的大胆举措，但是此举会不利于北京继续寻求渗透控制国际机构。
 <br/>
 <br/>
 报导称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 于2016年11月成为国际刑警组织的首位中国主席，许多人认为，这是北京在国际组织中寻求合法性、〝发挥领导作用〞的重要一步。
 <br/>
 <br/>
 在这种情况下，即使是以北京专职体制为标准，拿下孟宏伟对当局来说也是一个大胆行动。但是，这种举措破坏了中共继续寻求国际影响力的努力。
 <br/>
 <br/>
 报导引述霍夫斯特拉大学莫里斯．A．迪安法学院(Hofstra University’s Maurice A. Deane School of Law)教授古举伦(Julian Ku)的分析称，孟宏伟的任命是北京〝国际地位和影响力日益增强的标志〞。但是，北京在没有通知国际刑警组织的情况下，就让孟宏伟〝失踪〞，这将破坏北京在全球扩展影响力的努力，因为其他国际组织不会再放心让一个中国公民担任负责人。
 <br/>
 <br/>
 古举伦认为，此案〝表明国内考量超过了国际〞，虽然北京一贯如此，但以前〝可能不像这次那么明显〞。
 <br/>
 <br/>
 目前北京的对外渗透，因为美国的强势反击而遇挫。但外媒报导指，北京还在继续努力操控联合国等国际组织。此时曝出孟宏伟事件，对北京也是一种打击。
 <br/>
 <br/>
 对于北京为何不顾国际影响，突然拿下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警主席.html" target="_blank">
  国际刑警主席
 </a>
 ，外界传出多种说法。有港媒指，孟宏伟案涉中南海政变阴谋，事态紧急。还有说法认为，孟宏伟担任国际刑警主席后，在国际上有不利北京的举动，涉嫌替
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 反扑。
 <br/>
 <br/>
 有外媒分析，中共公安部近期再次强调肃清〝周永康余毒〞，孟宏伟应该已有所警觉。孟妻在其失踪一周后向法国警方报案，使其成为国际事件，这应该是孟宏伟早就安排好的安全预案。
 <br/>
 <br/>
 还有猜测认为，北京突然拿下孟宏伟，是担心他效仿王立军，在法国寻求政治庇护，洩露政治秘密。
 <br/>
 <br/>
 日前中共公安部声明，当局处理孟宏伟非常〝及时〞，并强调〝绝不允许跟组织讲条件、讲‘价钱’〞。这种表态或间接证明，孟宏伟之前和北京当局之间曾经有过某种较量。
 <br/>
 <br/>
 10月4日，孟宏伟妻子在国际刑警总部法国里昂向法国警方报案，称孟在返回中国后失联。10月7日孟妻又召开记者会，曝光孟宏伟发回的最后一条短讯息：一把尖刀图案。同日，中纪委通报孟宏伟落马。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：云涛）
</div>
