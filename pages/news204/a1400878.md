### 【禁闻】哈尔滨大庆一天绑架119名法轮功学员
------------------------

 【新唐人2018年11月26日讯】
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈尔滨.html" target="_blank">
   哈尔滨
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大庆.html" target="_blank">
   大庆
  </a>
  一天绑架119名
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
   法轮功
  </a>
  学员
 </h4>
 <br/>
 中共对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 的迫害加剧，最近在多地大规模
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绑架法轮功学员.html" target="_blank">
  绑架法轮功学员
 </a>
 。
 <br/>
 <br/>
 据〝法轮大法明慧网〞报导，11月9号，中共在一天之内，就在黑龙江省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈尔滨.html" target="_blank">
  哈尔滨
 </a>
 市各区县和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大庆.html" target="_blank">
  大庆
 </a>
 市， 绑架了119名法轮功学员，还骚扰了10几名学员。
 <br/>
 <br/>
 报导说，这次大规模绑架是黑龙江省公检法司部门蓄谋已久的阴谋，警察是拿著名单到法轮功学员家抓人的。
 <br/>
 <br/>
 <h4>
  造
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因婴儿.html" target="_blank">
   转基因婴儿
  </a>
  违法 46律师发声明促查
 </h4>
 <br/>
 11月26号，一位名叫贺建奎的中国研究人员声称，一对经过他进行〝基因编辑〞、具有爱滋病免疫力的双胞胎女婴这个月在大陆出生。这是首宗人类基因改造案例。外媒报导说，如果消息属实，那将是对人类伦理的极大挑战。
 <br/>
 <br/>
 大陆46名律师当天联名发布声明说，贺建奎涉嫌违法进行人体胚胎活动，警方应立即立案侦查，并对他采取强制措施。 同时还应全面调查事件的各方责任人，追究他们的行政和刑事责任。
 <br/>
