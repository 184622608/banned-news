### 大陆手机巨头回应赌博传闻：只输10多亿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】近日，有关大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手机巨头.html" target="_blank">
  手机巨头
 </a>
 、深圳
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金立集团.html" target="_blank">
  金立集团
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_董事长.html" target="_blank">
  董事长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘立荣.html" target="_blank">
  刘立荣
 </a>
 ，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赌博.html" target="_blank">
  赌博
 </a>
 上输了超过人民币100亿的消息广传。有股东则推测刘立荣挪用公款数额或在60亿元左右。刘立荣回应说自己只输了〝十几个亿。〞
 <br/>
 <br/>
 界面新闻24日援引接近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金立集团.html" target="_blank">
  金立集团
 </a>
 股东消息人士的话称，金立集团
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_董事长.html" target="_blank">
  董事长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘立荣.html" target="_blank">
  刘立荣
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赌博.html" target="_blank">
  赌博
 </a>
 上输了超过100亿（人民币，下同），股东们还推测，刘立荣挪用公款的数目可能在60亿左右。但赌博地点不是在香港、澳门，而是在塞班。
 <br/>
 <br/>
 报导还披露了刘立荣在塞班赌博的细节。上述知情人士称，刘立荣去过两次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塞班岛.html" target="_blank">
  塞班岛
 </a>
 。第一次就输了20个亿。第二次，〝最后一把牌，就一把牌，一次性输了7亿美金。〞
 <br/>
 <br/>
 针对此事，金立智能手机官方微博回应称，相关说法是捏造，严重侵害公司声誉。并要求界面立即删除相关消息，否则将追究法律责任。
 <br/>
 <br/>
 11月24日，有记者在香港采访了风波中的金立集团董事长刘立荣。刘本人坦承参与了赌博，但他否认挪用60亿公款的说法。
 <br/>
 <br/>
 刘立荣表示，他创办金立16年了，我个人没有其他收入，难免在生活上有些公私不分，有借用公司资金的行为。他没有说出从公司挪用资金的准确数字。当被问到他究竟输了多少钱时，他思考了一会，用很轻的声音说：〝十几个亿吧。〞
 <br/>
 <br/>
 他感叹，〝赌这个东西真的不能沾，一失足成千古恨。不光是涉及多少钱，它还会对你的品行定性，让一个人人格破产。〞他还提到，在塞班岛输钱发生在2017年的1月。
 <br/>
 <br/>
 据悉，在2017年金立曝资金危机后，刘立荣就一直呆在香港。当时就有消息称，刘立荣因赌博输掉好几个亿。但金立官方否认，后来刘立荣接受媒体采访时说，金立的资金链危机主要是2016年和2017年的营销和投资费用超限。
 <br/>
 <br/>
 今年7月份开始，金立分别被深圳市福田区法院和深圳中院列为失信人。8月12日，深圳中院在法院公告中明确表示刘立荣作为被告〝下落不明〞。
 <br/>
 <br/>
 据陆媒报导，今年以来，陷入危机的金立不断被供应商告上法庭。
 <br/>
 <br/>
 1月16日，刘立荣拥有的41.4%的股权被法院冻结两年，时间为2018年1月10日到2020年1月9日。当时网易科技引述业内人士的话透露，金立对外拖欠的债务达到百亿元。
 <br/>
 <br/>
 3月，金立公司主要人员名下共计价值156520000元的财产已被查封、扣押或冻结；8月15日，诠鼎科技股份有限公司已经向深圳中院申请对金立进行破产清算。
 <br/>
 <br/>
 8月至10月广东华兴银行股份有限公司深圳分公司，两次向深圳中院申请对金立进行破产清算；11月20日，近20家金立供应商，向深圳中院提交对金立进行破产重整的申请。
 <br/>
 <br/>
 一位参与申请破产重整的供应商说，他们已给金立足够的时间，但金立并没有拿出任何有效挽救措施，他们认为碰到了无赖，最终决定让法院进行裁决。
 <br/>
 <br/>
 金立是中国移动设备研制与软体开发企业，成立于2002年，旗下的手机品牌曾一度占据中国市场约5%的份额，也曾是印度手机市场份额最大的中国品牌。
 <br/>
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
