### 党媒紧跟习近平  谈改革只字不提邓小平
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】2018年是中共改革40周年，党媒10月30日刊发评论称，要以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 的眼光看待改革开放，但评论中罕见的只字未提推动改革开放的中共前领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 。而日前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 南下视察被称为改革前沿的广东，同样只字未提邓小平，引起揣测。
 <br/>
 <br/>
 人民日报10月30日刊发的这篇题为〝以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 的眼光看待改革开放〞的评论文章称，〝如何看待改革开放，怎样推进改革开放〞，是必须回答的基本问题。
 <br/>
 <br/>
 但文章中却只字未提推动中共改革开放的前领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 ，引发舆论关注。
 <br/>
 <br/>
 不过，评论中的那句〝摸着石头过河〞，是邓小平式改革的招牌，这个比喻被认为是八十年代打开中共改革局面的主要策略，至今，仍然是中共官方论述中经常引用的所谓〝邓小平理论〞的主要内容。
 <br/>
 <br/>
 而评论中的这句〝一系列重大改革开放举措推出，解决了许多长期想解决而没有解决的难题，办成了许多过去想办而没有办成的大事〞，则是去年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在一次会议上的讲话。
 <br/>
 <br/>
 今年是中共改革开放40周年，10月22日至24日，习近平南下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳.html" target="_blank">
  深圳
 </a>
 等多地视察，广东被认为是改革开放的前沿。习此次在中美贸易战火升高之时考察广东，被认为是为了提振国内外对于中国经济和改革开放的信心。
 <br/>
 <br/>
 但习近平此次南下相当低调，且只字没有提及邓小平。对此，有观察人士认为，习近平不想与邓小平
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南巡.html" target="_blank">
  南巡
 </a>
 做比较，因在中美贸易战背景下，及国内经济停滞下，习近平的任何说话都会被外界演绎，或会得出反效果。
 <br/>
 <br/>
 不过，就在习近平〝南巡〞的同时，中国网络上热传邓小平长子邓朴方盛赞邓理论的一篇演讲稿。
 <br/>
 <br/>
 该演讲除了热捧邓小平〝改革开放的伟大革命〞之外，还宣称〝我们要坚持长期处于社会主义初级阶段这个基本判断〞，并不点名地提醒，〝我们一定要有这种实事求是的态度，保持清醒的头脑，知道自己的份量〞。
 <br/>
 <br/>
 邓朴方突然公开讲话力挺邓小平，被指不太寻常。旅美异议人士魏京生认为，习近平虽然看似大权在握，但实际上中共内部反对声音正在集聚，邓朴方敢于公开这么讲话，说明党内已经形成批评现当局的气候。
 <br/>
 <br/>
 近一段时间以来，对习近平来说，可谓是诸事不顺，外交上有中美贸易战；国内，有对经济方面的反对声音，再加上澳门中联办主任突然堕亡、港珠澳大桥被声讨是〝大白象工程〞，这都认为不是他发表讲话的好时机。
 <br/>
 <br/>
 有港媒分析，中共四中全会即将于11月召开，估计会议的主题是深化改革，落实三中全会提出的改革目标，同时对改革开放40年做一个总结。因此，习近平今次南下考察的结果，也将会在四中全会中呈现。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
</div>
