### 央视前主持李咏死因曝光  曾被指与朱军大打出手
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】央视前主持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李咏.html" target="_blank">
  李咏
 </a>
 突然去世的消息引发关注。他的死因及早年离开央视的原因也被曝光。网上消息称，李咏当年曾因一件事与央视主持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军.html" target="_blank">
  朱军
 </a>
 打起来了。如今李咏逝世，朱军也因涉嫌性骚扰实习生事件惹上官司。
 <br/>
 <br/>
 10月29日，央视主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李咏.html" target="_blank">
  李咏
 </a>
 妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈文.html" target="_blank">
  哈文
 </a>
 发微博说：〝在美国，经过17个月的抗癌治疗，2018年10月25日凌晨5点20分，永失我爱。〞据悉，李咏在美国因患癌去世。
 <br/>
 <br/>
 李咏曾主持多届央视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_春晚.html" target="_blank">
  春晚
 </a>
 ，早年主持央视《非常6+1》、《幸运52》、《梦想中国》、《咏乐汇》等热门综艺。
 <br/>
 <br/>
 2013年李咏辞职，从央视转入中国传媒大学，之后，他在传媒大学任教。
 <br/>
 <br/>
 2017年2月5日，李咏与孙坚、罗希合作，主持美食传情真人秀《熟悉的味道第二季》。同年12月2日，他与谢楠、雅琴一同主持〝2018爱奇艺尖叫之夜〞，这是他最后一次公开活动。
 <br/>
 <br/>
 李咏与妻子哈文早年是央视的同事，俩人育有一女。
 <br/>
 <br/>
 据多家媒体报导，哈文在担任2012年、2013年、2015年央视春晚总导演期间，让春晚逐渐走出赵本山时代，赵被指曾卷入薄熙来、周永康政变计划。
 <br/>
 <br/>
 在央视沦为中共高官后宫、主播纷纷辞职的背景下，李咏离开央视，哈文也于2015年辞去了央视职务。
 <br/>
 <br/>
 网上另有一说法，李咏曾因〝春晚
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑色三分钟.html" target="_blank">
  黑色三分钟
 </a>
 〞，离开了央视。在2007年春晚进行到零点倒计时阶段，6位主持人却多次忘词、抢词、口误，发生多次冷场尴尬。事件发生后，广大网友称之为〝春晚黑色三分钟〞。
 <br/>
 <br/>
 对于这〝黑色三分钟〞，网上一直争议不断，甚至传出李咏和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军.html" target="_blank">
  朱军
 </a>
 因为此事而大打出手。据悉，当时两人并没有对此事件做出回应，于是事情愈演愈烈。
 <br/>
 <br/>
 事后，朱军曾在节目中回应称，当时6个主持人，都试图想用自己的努力来弥补空档，因人多嘴杂就给茬到一起了。至于网上传言因这事朱军和李咏打起来了，朱军否认。
 <br/>
 <br/>
 该事件已经过去多年，如今物是人非，李咏因病过世，而朱军也因涉嫌性骚扰女实习生丑闻，如今官司缠身。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
