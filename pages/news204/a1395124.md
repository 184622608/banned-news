### 中共常委资产曝光与孟案巧合？陆媒起底孟妻惹猜测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】前中共公安部副部长兼国际刑警主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 案发酵的同时，港媒开始报导中共高层的海外资产，法媒指两者或有关联。日前，陆媒也开始起底孟妻小三上位、贪腐挥霍的内幕。
 <br/>
 <br/>
 10月12日，法广引述分析称，中共家族海外资产曝光，或与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 知情有关。
 <br/>
 <br/>
 日前，港媒开始陆续报导中共多名前任和现任政治局常委在香港和海外的豪宅和资产。此事发生在孟宏伟妻子在法国报警，孟宏伟案曝光之后。
 <br/>
 <br/>
 10月4日，孟妻向法国警方报警称，孟宏伟返回中国后失踪。7日，孟妻接受采访，曝光孟9月25日失联前发回的尖刀短信。同日中纪委通报孟宏伟落马。
 <br/>
 <br/>
 外媒分析，孟宏伟应该早有安全预案，他发送给妻子的尖刀图案，或是某种行动代号。孟妻手中应该有孟宏伟掌握的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共高层海外资产.html" target="_blank">
  中共高层海外资产
 </a>
 情况，以此作为和北京谈判的筹码。
 <br/>
 <br/>
 在港媒报导中共高层海外资产之后，陆媒《稜镜》10月12日开始起底孟妻，称其原名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高歌.html" target="_blank">
  高歌
 </a>
 ，与孟宏伟通奸后双双抛弃配偶，两人于2005年结婚。之后高歌利用孟的权势，担任大陆、香港乃至海外多家企业的高管，但只领取丰厚〝薪水〞不上班。
 <br/>
 <br/>
 报导还曝光，孟宏伟夫妇在北京拥有两处豪宅，高歌每年的开支不低于300万元人民币。
 <br/>
 <br/>
 《明报》日前也披露，孟宏伟利用分管出入境的便利，早已帮助妻儿取得外籍身份，妻儿在海外生活多年。
 <br/>
 <br/>
 而推特消息指，孟妻在法国里昂高档社区租住豪宅，月租5万多人民币，每年花销也有数百万。
 <br/>
 <br/>
 虽然孟妻对外媒声称孟宏伟〝没有能力贪腐〞、〝一生追求法治〞，但许多网友质疑，孟妻是孟宏伟贪腐的共犯。
 <br/>
 <br/>
 有分析认为，陆媒起底孟妻不大光彩的历史，意在打击其形象及可信度，另一方面也可能是对她发出某种警告。孟宏伟妻未来会否如当年令完成等归于沉寂，孟案走向会否受到双方博弈影响，仍需观察。
 <br/>
 <br/>
 孟宏伟被指周永康在中共政法系统的余党。对于孟宏伟回国后失联，大陆网友称，以前他让中国人失踪，这回轮到他自己被失踪了。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
