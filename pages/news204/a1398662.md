### 崔永元揭杂交水稻内幕：先转基因再杂交
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】一直抵制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 9日转发微博，揭露
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国杂交水稻.html" target="_blank">
  中国杂交水稻
 </a>
 一个公开的秘密，炮火瞄准
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_袁隆平.html" target="_blank">
  袁隆平
 </a>
 ，文中指中国专家将转基因或基因编辑称之为〝杂交〞。袁隆平此前曾表示，转基因有没有毒，争议挺大，试吃转基因大米，恐面临绝育风险。
 <br/>
 <br/>
 11月9日零时26分，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 转发了微博名为〝@米阑-忧伤〞的微博，文中写道：〝中国的杂交水稻现状：先
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 或基因编辑，然后再杂交。中国的挺转分子就称其是杂交，赤裸裸的自欺欺人的行为！〞
 <br/>
 <br/>
 文中称〝这已经是公开的秘密〞。
 <br/>
 <br/>
 崔永元随后又附上微博名为〝往事919〞的一段文字：〝杂交已成为转基因与基因编辑推广的渠道。中国的现状是：很多干了转基因的人都不承认自己开发的新品种是转基因，不论是转基因还是基因编辑，要投入到运用，都要回到田间地头，最后一道工序毫无例外的都要用到传统杂交。〞
 <br/>
 <br/>
 〝只要最后一道工序是传统杂交，他们都敢宣称自己开发的新品种是非转基因。李家洋、张启发、夏新界、邓兴旺、唐晓燕、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_袁隆平.html" target="_blank">
  袁隆平
 </a>
 都是如此。〞
 <br/>
 <br/>
 文中提到的杂家水稻专家袁隆平很多人都熟知，对于转基因食品他此前曾表示：〝要说这个转基因能不能吃，到底有没有毒，其实还是争议挺大的。〞
 <br/>
 <br/>
 袁隆平还说：〝这个转基因产品还得看基因，比如含有bt毒蛋白基因的时候，虫子吃了会死掉，但是小白鼠不会，但这也不能证明用在人身上就不会有事，所以还得看基因。〞
 <br/>
 <br/>
 针对转基因大米的安全问题，去年8月24日，在大陆某网站一段视频中袁隆平说，转基因人吃了会怎么样呢？大家心里很担心，这个可以理解的。可又不能拿人来做实验，它只能拿小白鼠做实验。
 <br/>
 <br/>
 袁隆平表示，自己作为志愿者，已经吃了转基因大米，但他已无生育能力，无法观察对下一代的影响。〝所以要号召有生育能力的年轻人去吃。他吃了之后，他生的儿子也没有问题，就没有问题了！〞
 <br/>
 <br/>
 他同时也表示，试吃转基因大米，恐会面临绝育的风险，因此不能强迫。
 <br/>
 <br/>
 正是袁隆平这些话，让崔永元感到这个转基因不会那么简单，他觉得这是一个大问题。而且袁隆平的话也让他感觉到，转基因食品还会有很大发展趋势，这正是他一直担心的事情。
 <br/>
 <br/>
 有报导称，对于转基因的争议，有点类似于〝进化论〞和〝上帝造人论〞之争。这个争议如今在美国仍旧是一个很大的议题。反对者认为，大自然自身就是一个非常好的基因工程师。而转基因是把不同物种的基因进行转移和重组，这本身就是不安全的。
 <br/>
 <br/>
 据悉，除了农业，转基因技术还应用于其他众多领域，包括医药、洗衣液、发光鱼等。
 <br/>
 <br/>
 (记者罗婷婷综合报导/责任编辑：李泉)
