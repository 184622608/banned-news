### 重庆坠江巴士已打捞出水 13死2失踪现场鸣笛致哀
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】中国重庆近日发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公交车翻车.html" target="_blank">
  公交车翻车
 </a>
 坠江事故，经过多日搜救，已找到13名罹难者遗体，尚有2人失踪。公交车已于本周三打捞出水，当时现场参与搜救的船隻都鸣笛致哀。
 <br/>
 <br/>
 这起意外事故发生于10月28日早间，重庆一辆22路公交车在行经万州区的长江二桥时，不知何故突然失控，冲到对向车道上并撞上一辆小轿车，然后直接坠落长江中。据陆媒报导，当时车中包括司机在内共有15人。但中共官方说法受到当地民众广泛质疑。
 <br/>
 <br/>
 当地消防人员到达事发现场后，巴士已沉入约71公尺深的水底，而车上的司机和所有乘客均已失踪。
 <br/>
 <br/>
 当时，中共警方一度错把在桥上与公交车相撞的小轿车邝姓女驾驶员对此事故负有责任，遂将邝女拘捕。但经过进一步调查后，证实事发时是公交车先逆向冲到对面的车道上才酿成撞车、翻车的惨剧。邝女10月30日才被释放。当地警方的处理，也引起舆论的谴责。
 <br/>
 <br/>
 经过几天的打捞，截至目前共找到13具罹难者尸体。失事巴士的黑盒子已在30日找到，但未向外界公布细情。10月31日晚间11点28分落水的巴士被打捞起来，现场搜救船隻鸣笛致哀。
 <br/>
 <br/>
 不过，这次导致这次事故发生的真正肇事原因至今尚未厘清。桥梁护栏设置是否质量达标，也是民众质询要点。
 <br/>
 <br/>
 （记者李明综合报导/责任编辑：戴明）
