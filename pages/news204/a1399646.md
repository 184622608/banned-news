### 【禁闻】《战狼2》被党媒批评〝不符合规范〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家电影局.html" target="_blank">
  国家电影局
 </a>
 近期下发了规定，将从12月起对国产电影的复映进行管控。原被称为〝主旋律〞的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战狼2.html" target="_blank">
  战狼2
 </a>
 》复映片，被党媒特别点名说，〝不符合规范〞。来看报导。
 <br/>
 <br/>
 中共宣传部下辖的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家电影局.html" target="_blank">
  国家电影局
 </a>
 11月8号下发了《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国产电影复映暂行规定.html" target="_blank">
  国产电影复映暂行规定
 </a>
 》，从12月1号开始执行。《暂行规定》中说，所谓的〝复映影片〞是取得了《电影公映许可证》或公映许可，距首次公映结束超过两年再次进行发行放映的国产电影。
 <br/>
 <br/>
 根据这个《规定》，《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战狼2.html" target="_blank">
  战狼2
 </a>
 》不符合规范。为什么呢？大陆《新京报》11月15号的报导说，因为影片在2018年9月19号复映，距离首次公映结束没有超过两年。
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战狼.html" target="_blank">
  战狼
 </a>
 2》是2017年上映的军事题材电影，宣扬所谓的爱国和民族主义。影片两个月前复映时，曾引发一波网络热议。当时网友纷纷调侃：〝最近党又犯了什么错误需要洗白吗？〞〝洗脑吗，我的国？〞等。
 <br/>
 <br/>
 严格来说，有关复映的新规定从12月1号才开始执行，但中共北京市委宣传部主办的《新京报》却在之前就特别点名《战狼2》复映属于违规，引发猜测。
 <br/>
 <br/>
 原大陆影视从业者朱先生告诉《新唐人》，如果没有军方背景，一般人在大陆根本不能碰《战狼2》这样的军事题材。影片当时肯定获得官方撑腰，现在却处境尴尬。
 <br/>
 <br/>
 原大陆影视从业者朱先生：〝作为电影来说的话，可能就是吴京我领着自由团队把这个片子做出来。实际上他后面有可能解放军的总装部在给你撑着。但现在风向一转，国际上看完这个片子以后，觉得中国是军国主义、霸权主义。最后就是拍电影的人要背这个锅了。〞
 <br/>
 <br/>
 今年6月，香港《苹果日报》曾引述消息指称，《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红海行动.html" target="_blank">
  红海行动
 </a>
 》和《战狼2》被中共官方内部批评为〝鼓吹军事主义〞，最终让国家树敌。这两部影片的续集拍摄，已经很难拿到批文。官方也已经下达〝禁军令〞，具有军方背景的电影公司及影片一律被限制。
 <br/>
 <br/>
 大陆传媒业者、电影制作人石宇歌认为，《战狼》挨批并不是由于宣扬军事主义，因为党媒《人民日报》11月16号头版还在高调宣称要〝开创强军事业〞。
 <br/>
 <br/>
 大陆传媒业者、电影制作人石宇歌：〝它根本上就不是说，这个文艺作品本身在中国真正是触犯什么政治性错误。实际上就是现在国际形势上紧张了，（中共）它要应对目前来自各方面的责难，不得不说它的宣传是做错了。那么它得找个替罪羊，而这个替罪羊是谁呢？就是当时被捧的非常红的《战狼》、《红海行动》这个剧组了。〞
 <br/>
 <br/>
 今年还有另一部〝主旋律〞影片陷入同样的窘境。官媒制作的纪录片《厉害了，我的国》，在中兴通讯公司遭受美国制裁陷入休克后，被紧急下架。
 <br/>
 <br/>
 石宇歌：〝现在中共的宣传工作出了一个很大的问题，已经是处于一种不知道该如何发声了。它一直都是处于一种官员逆淘汰。现在搞宣传的这批官员，他们是没有什么远见，国际上的变化太快，他们所鼓吹的东西，可能今天还是鼓吹的，明天就得把它关起来。然后又得找个替罪羊去搧耳光。〞
 <br/>
 <br/>
 石宇歌分析，中共宣传工作陷入混乱的原因，包括国际形势变化实在太快，也包括中共内部的分裂互斗。目前中共对文艺管控越来越严，有可能借用文革样板戏的方式，把电影内容局限在几种类别里面。但是他质疑，以中共现在的水平，能否拍出统一人民思想的样板戏？万一这个样板戏又出了问题，那中共该如何收场。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/陈建铭
