### 【禁闻】法轮功纽约游行  四百多人三退
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】
 <b>
  法轮功
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约游行.html" target="_blank">
   纽约游行
  </a>
  四百多人三退
 </b>
 <br/>
 <br/>
 10月21号，美国纽约地区的部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 ，在纽约市布鲁克林的华人社区举行游行，展现法轮大法的美好、揭露中共对法轮功学员的迫害，同时也劝告中国人退出中共的党、团、队组织，三退保平安。
 <br/>
 <br/>
 当天下午，由天国乐团、舞龙队、彩旗队、小花船、横幅队等组成的游行队伍，沿布鲁克林第八大道行进， 吸引了众多市民观看。
 <br/>
 <br/>
 据〝法轮大法明慧网〞报导，现场民众反应热烈，当场就有400多人表示要退出中共的党、团、队组织。
 <br/>
 <br/>
 <b>
  抵制红色歌剧 澳
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人团体.html" target="_blank">
   华人团体
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联署.html" target="_blank">
   联署
  </a>
  请愿
 </b>
 <br/>
 <br/>
 澳大利亚华人团体〝澳洲价值守护联盟〞日前在网络上发起联署活动，征集公众签名，抵制中共红色歌剧在当地上演。
 <br/>
 <br/>
 据报导，11月4号，中共毛时代的歌剧《洪湖赤卫队》，将在当地著名的悉尼歌剧院上演，歌剧院在官方网站上介绍说，这部歌剧讲述的是洪湖畔一个有关自由和希望的故事。
 <br/>
 <br/>
 但〝澳洲价值守护联盟〞在联署信中说，〝这部红色歌剧宣传暴力、美化红军，传达了一种关于中共的虚假形象〞。其广告宣传假借〝为自由和希望而战〞掩盖中共大外宣的真实用意，向澳洲输出中共意识形态。
 <br/>
 <br/>
 据了解，联署行动目前已经征集到了1000多个签名。
 <br/>
 <br/>
 <b>
  《福布斯》刊文 披露中共强摘器官
 </b>
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共活摘.html" target="_blank">
  中共活摘
 </a>
 人体器官的罪行，正在受到越来越多的西方主流媒体关注，继英国广播公司10月8号播出一则长达12分钟的独家报导，揭露中共活摘法轮功学员器官的内幕之后，美国商业杂志《福布斯》10月16号也刊登文章，揭露中共的这项罪行。
 <br/>
 <br/>
 这篇题为〝中国摘取器官和诸多待解疑问〞的文章，是法学研究者欧恰布所写。
 <br/>
 <br/>
 文章说，大量的证据表明，中共政府现在正在杀害包括法轮功学员在内的良心犯，摘取他们的器官，以供应器官移植产业。
 <br/>
 <br/>
 编辑/周玉林
