### 官媒过滤习近平〝自力更生〞讲话 洩露敏感信息
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】英媒报导，北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 9月在黑龙江发表〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自力更生.html" target="_blank">
  自力更生
 </a>
 〞的讲话，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
  央视
 </a>
 报导中删去了其中〝外国技术越来越难获得〞的内容。分析认为，这段讲话等于变相承认盗窃外国先进技术。
 <br/>
 <br/>
 英国《金融时报》11月12日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 今年9月在黑龙江省发表讲话时，号召国企制造出现在多数需要进口的先进零部件。
 <br/>
 <br/>
 习近平还表示，在国际上，先进技术与关键技术越来越难获得，迫使北京要走〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自力更生.html" target="_blank">
  自力更生
 </a>
 〞的道路。
 <br/>
 <br/>
 报导指，当时中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
  央视
 </a>
 《新闻联播》在编辑习近平讲话录像时，将〝技术越来越难获得〞的部分删除，只保留了〝自力更生〞的发言。
 <br/>
 <br/>
 中共党媒《人民日报》海外版也使用了同样的手法。
 <br/>
 <br/>
 观察人士解读，这反映出中共宣传机构担心，这段讲话会被视作承认中方利用强迫与偷窃等手段获取外国知识产权，而这正是白宫与西方社会一直谴责的做法。
 <br/>
 <br/>
 虽然中共官方在公开场合一直拒绝承认盗窃外国技术，但其很多相关的表态变相证实了这一做法。
 <br/>
 <br/>
 自从贸易战爆发以来，北京一直在强调〝保护知识产权〞、〝保护外企在华合法利益〞，以回应美方对中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_盗窃知识产权.html" target="_blank">
  盗窃知识产权
 </a>
 、强制转移技术的谴责。
 <br/>
 <br/>
 不过外媒指，北京自加入世贸组织以来，就一直在〝承诺〞保护知识产权，但从不兑现。
 <br/>
 <br/>
 日前，北京高层在北京会见美国前国务卿基辛格时强调，希望美国〝尊重中国选择的发展道路〞。
 <br/>
 <br/>
 而中共官方此前把〝用中国市场换技术〞、〝引进西方先进技术〞等手段宣传成〝弯道超车〞，认为这是中共应有的〝发展权利〞。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
