### 项俊波罪状被展出 自称来日无多
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】近日，中共举办的〝改革开放40周年大型展览〞上，展出了中共原保监会主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_项俊波.html" target="_blank">
  项俊波
 </a>
 的被处分决定及其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_签字.html" target="_blank">
  签字
 </a>
 ，项俊波写道，自己来日无多，但愿意认真改造，重新做人。项俊波被指涉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融政变.html" target="_blank">
  金融政变
 </a>
 且好色成性，与123名有夫之妇勾搭成奸。
 <br/>
 <br/>
 日前，中共举办的〝庆祝改革开放40周年大型展览〞在国家博物馆开幕，展板资料显示，中共十八大以来，中共中央批准立案审查的省军级以上党员干部及其他中管干部共计440人，包括十八届中共中央委员、候补委员43人，中央纪委委员9人。
 <br/>
 <br/>
 在以反腐成果为主题的展区一角，展出了多名落马高官的忏悔书手稿或处分决定。在盖有〝中共中央纪委〞红印章的A4纸上，展出中共原保监会主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_项俊波.html" target="_blank">
  项俊波
 </a>
 的被处分决定，及其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_签字.html" target="_blank">
  签字
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红手印.html" target="_blank">
  红手印
 </a>
 ，以及中共甘肃省委原书记王三运、中宣部原副部长鲁炜的忏悔书手稿。
 <br/>
 <br/>
 处分决定文稿显示，在中纪委印发的《关于给予项俊波开除党籍处分的决定》的通知下面，项俊波于2017年9月21日签字时写道：〝完全同意组织对我的处理意见。我所犯错误的性质是非常严重的。〞
 <br/>
 <br/>
 项俊波还写道，自己作为一名中共中央委员、正部级干部……，崇尚金钱，贪恋财物，利用职权大搞权钱交易……。虽然自己来日无多，以后也不在党组织了……，愿认真改造，重新做人。
 <br/>
 <br/>
 项俊波去年4月份被指涉嫌严重违纪，被中纪委调查，成为北京当局清查金融界贪腐案，首名被查的〝第一虎〞。
 <br/>
 <br/>
 去年9月23日，中纪委通告，项俊波被开除党籍、公职，通告中列举了项俊波的5条罪名。包括：项俊波严重违反中共中央八项规定精神，违规接受宴请，搞权钱交易、权色交易。利用职务上的便利为他人谋取利益并收受巨额财物，涉嫌受贿犯罪。
 <br/>
 <br/>
 2018年6月，项俊波受贿案公开审理，其被控受贿1,942万元人民币。
 <br/>
 <br/>
 公开资料显示，项俊波现年60岁曾在审计系统任职多年，后转至金融系统，曾任央行副行长、中国农业银行行长、党委书记、保监会主席等职。
 <br/>
 <br/>
 此前消息透露，项俊波好色成性，共有21个固定情人，经常在办公室和下属淫乱，还有6个私生子，经鉴定只有1个是他亲生，其余5个均给他戴了绿帽子。
 <br/>
 <br/>
 中南海的消息人士此前透露，项俊波是明天系掌门人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肖建华.html" target="_blank">
  肖建华
 </a>
 咬出的金融〝第一虎〞。2015年中国股灾，项因发动〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融政变.html" target="_blank">
  金融政变
 </a>
 〞，对抗习近平当局，遭调查。
 <br/>
 <br/>
 肖建华被指是曾庆红的白手套，为江派利益集团在海外走资、洗钱。2017年大年三十已被北京当局抓捕。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
