### 愈千老兵与鲁警对峙  当局调重兵放催泪弹驱散(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东平度.html" target="_blank">
  山东平度
 </a>
 市接连3天发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 集体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 ，中国各地上千名老兵赶赴平度声援，当地政府出动大批特警，释放催泪弹驱散，老兵则用灭火器还击，混乱中双方均有人受伤。直至6日晚，警方全面清场，带头维权的10多名老兵被带走。
 <br/>
 <br/>
 据多家香港媒体10月6日报导，山东青岛平度市38名退伍军人，10月4日打算乘车进京上访，但还没有走出山东省，就被当地政府人员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拦截.html" target="_blank">
  拦截
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_殴打.html" target="_blank">
  殴打
 </a>
 。
 <br/>
 <br/>
 5日早上，数百名来自山东其他地方及江苏、四川等省的退伍军人赶来平度声援。他们身穿迷彩服、高举党旗，有人手持木棍称要为平度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵.html" target="_blank">
  老兵
 </a>
 被殴打事件〝讨说法〞。
 <br/>
 <br/>
 当局派出大批特警，双方一度推撞。当晚，中国各地老兵不断赶往当地，聚集人数不断增加，至6日凌晨已超过千人，当地政府出动大批特警清场。
 <br/>
 <br/>
 6日，平度居民刘女士对《自由亚洲》表示，现在当地的各个路口都封了，不准进，各个高速路口都封了，还有老兵被殴打重伤住院了，具体几个人不清楚。现场穿警服的有好几百人吧。
 <br/>
 <br/>
 <br/>
 <br/>
 刘女士透露，6日上午时已有青岛的大批警察赶到当地，因此外面的人无法进去与老兵会合，情况继续恶化。尽管老兵提出，要惩办殴打老兵的公安局副局长，但该名副局长并没有被处罚，相反警察还在围困老兵。
 <br/>
 <br/>
 现场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 的老兵王先生称，在接到平度老兵的求援讯息后，各地老兵都准备前往平度，初步估计应该有3000人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集结.html" target="_blank">
  集结
 </a>
 。但全国的老兵群体都受到官方的限制，而且有老兵通报，山东方面已从附近的潍坊市调集20多辆车的警察增援。
 <br/>
 <br/>
 王先生说：全国老兵都准备往那里集结，估计6日要到3000人左右吧，但全国的各个老兵群都被封了，全国的网警都在监视了。潍坊的警方也出动了，增援青岛去了。
 <br/>
 <br/>
 <br/>
 <br/>
 从网民上传的视频可见，6日，现场老兵们遭数百名警察包围，警方一度施放催泪弹，现场老兵有的四散走避，也有不少人手持棍棒、烟雾喷射器等与警方冲突，有老兵受伤或被警察制伏在地。
 <br/>
 <br/>
 警方大规模清场后，公安拘捕了带头者，聚集人群大多逃走散去。至当晚9时许，坚守现场的十几名老兵被带走，部分老兵被强行带离现场。
 <br/>
 <br/>
 近年来，中国各地持续爆发大规模的老兵维权，单在今年6月份，就先后发生河南漯河、江苏镇江老兵聚集维权，并与警方发生激烈冲突。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：祝馨睿）
