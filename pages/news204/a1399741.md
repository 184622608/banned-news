### 中共让步清单内容曝光 川普仍不满
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】为了给〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞铺路，中共日前向美国政府提交了一份贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_让步清单.html" target="_blank">
  让步清单
 </a>
 ，给出继续尝试解决贸易战的充分诚意。但是美国官员仍不满。川普（特朗普）表示，还有4、5件事被漏掉了。
 <br/>
 <br/>
 美国之音11月17日报导说，美国总统川普表示，中共递交了一份清单回应美国对贸易改革的要求，中共想要达成协议。他们送了一份清单，列出他们愿意做的事情，那是一份很大的清单。
 <br/>
 <br/>
 路透社引述一名美国官员透露，中共向美国递交的清单包括142个项目，主要分为3类：包括：中共愿意进一步谈判采取行动的问题、已经着手解决的问题，以及被视为不会谈判的禁区问题。
 <br/>
 <br/>
 对此，美国商务部长罗斯（Wilbur Ross）在接受彭博社采访时表示，川普总统和习近平主席在阿根廷G20的一对一会谈之前，所有内容都只是准备工作。会晤将决定是否会形成一个真正的框架。
 <br/>
 <br/>
 报导引述一名消息人士透露，中共的让步方案上的条款，很多都是重新列出北京以前做出的调整，例如在特定行业提高外资比例，而这些文件并未涵盖改变工业政策例如〝中国制造2025〞计划上的承诺。
 <br/>
 <br/>
 CNN也报导称，中共开出的条件并没有包括白宫的核心要求，例如解决强迫技术转移和偷窃知识产权问题。 一名消息人士透露，中方给的东西并不是什么新的，双方仍处在僵局之中。通道现在再次打开了，但是仍有很多工作要做。
 <br/>
 <br/>
 对于中共的这份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_让步清单.html" target="_blank">
  让步清单
 </a>
 ，川普说：这是好事，〝但是，我仍然认为不可接受。我认为，我们在中国方面做得非常好。我非常尊重习近平主席。我非常尊重中国。但是中国占了美国的便宜很多年了。〞
 <br/>
 <br/>
 川普还说：〝那是一个非常完整的清单，有很多我们要求的东西，但是还有4、5件事被漏掉了。我们或许也能得到这几个事。但是你知道，这是一个非常完整的清单。〞
 <br/>
 <br/>
 川普表示，中美贸易不公平，中方对美国征收了巨额的关税，而美国没有对中方征收。〝现在一切都将改变。我们已经向中国价值2,500亿美元的商品征收了关税，如果我们想的话，我们还可以征收另外2,670亿美元。我们或许不必这么做，中国想要达成协议。〞
 <br/>
 <br/>
 他还说：〝我们的国家发展的非常好，中国发展的不好。他们的股市最近下跌了30%、32%，大幅下跌了。〞〝中国从来没有处在这种境地。我不想让他们处在糟糕的境地，但是我们必须拥有互惠互利的贸易。我们不能接受那些蠢人才会要的贸易。这就是他们占我们便宜的方式。我们不会再要了，他们也明白了。〞
 <br/>
 <br/>
 川普说：〝我认为，会达成一份协议。我们很快就知道了。〞
 <br/>
 <br/>
 美国政府要求中国改善美国企业的市场准入和知识产权保护、削减产业补贴，并削减3,750亿美元的贸易逆差。
 <br/>
 <br/>
 11月17日，在2018年的亚太经济合作组织（APEC）领导人峰会上，美国副总统彭斯称，中共设置的贸易壁垒史无前例，包括关税、盗用知识产权、强逼技术转让和不公平补贴，美国已对中共的贸易政策采取决定性措施，对2500亿美元的中国产品征收关税，并可能将征税规模扩大一倍。
 <br/>
 <br/>
 彭斯说，美国不会改变这一应对方式，除非中共改变其行为方式。彭斯13日接受专访时，也曾警告中共，必须全面改变其行为作法，才能避免与美国发生新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冷战.html" target="_blank">
  冷战
 </a>
 。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
