### 敢在〝龙脉〞动土？秦岭别墅神秘主人曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】被历代帝王尊为华夏文明龙脉的西安秦岭，因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建别墅.html" target="_blank">
  违建别墅
 </a>
 多次惊动中共高层，并引发陕西官场持续震荡。近期，秦岭违建别墅案代表性的〝陈路别墅〞神秘主人也遭媒体曝光，有关人员已经遭当局留置。
 <br/>
 <br/>
 秦岭是中国南北气候的分界线和重要的生态安全屏障，被历代帝王尊为华夏文明的龙脉，地位不言而喻。
 <br/>
 <br/>
 早前官媒连续披露，西安秦岭北麓存在大量违法建筑，分布在秦岭北麓5000多平方公里的范围内，对植被和河流等生态环境造成严重破坏。官媒调查显示这些别墅多达300多栋，不少已建成。业主非富即贵，还有党政要员。
 <br/>
 <br/>
 在被官媒披露之后，秦岭北麓违法建筑仍屹立不倒，甚至绵延成景、满山开花。一些别墅项目还将售楼广告竖立在进山的必经之路上。
 <br/>
 <br/>
 今年7月30日，中纪委副书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐令义.html" target="_blank">
  徐令义
 </a>
 亲赴西安，针对秦岭北麓违规建别墅召开专项会议。
 <br/>
 <br/>
 大会召开之后，自8月中旬起，当地政府出动大量人力和大批设备，开始对秦岭违规建别墅群进行大规模拆除。
 <br/>
 <br/>
 《中国新闻周刊》报导，秦岭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建别墅.html" target="_blank">
  违建别墅
 </a>
 案代表性的〝陈路别墅〞，从2005年兴建到2018年拆除，存世超过10年之久。
 <br/>
 <br/>
 这座园林特色的中仿式别墅占地9,280.53平米，建筑物占地517.29平米，鱼塘占地1,098.68平米，狗舍占地78.03平米。别墅内还有各种文物，经鉴定现代工艺品达26件，文物211件。
 <br/>
 <br/>
 在别墅建设过程中，栽种了多种名贵树木，包括一棵500年树龄的国槐，还有价值超过20万元人民币的皂角树。，院内有大量磨盘、石鼓、拴马桩、古代石狮等，正面悬挂〝望重成均〞红匾。
 <br/>
 <br/>
 别墅内还有一两米多高的黄花梨达摩像，大厅挂着范增的巨幅八骏图。前几年别墅主人还从西藏空运来5只藏獒幼崽，由于太小，同时从西藏空运羊肉喂藏獒。
 <br/>
 <br/>
 报导说，这座别墅的主人真名叫陈路，是西安市一位高官的公子。陈路的父亲在西安党政系统担任要职。一位村官曾见过这名党政要员3次，并称别墅酒窖内全是茅台酒和五粮液酒，〝50年茅台当水喝，一次一箱，喝不了让我带走。〞
 <br/>
 <br/>
 陈路父亲特别看重风水，在进行别墅选址时，他还专门从西安请来风水大师。多名施工工人表示见过陈路的父亲。
 <br/>
 <br/>
 2014年来，习近平先后6次对秦岭违规建别墅案下令彻查，直到今年7月底，中纪委副书记徐令义和中共自然资源部部长、国家自然资源总督察陆昊先后赴陕西进行调查。
 <br/>
 <br/>
 8月起，西安市纪委才决定成立调查组彻查，陈路得到消息，想在政府拆除之前自行拆除别墅，当时已经开始转移别墅内的物品，但被阻止。目前，西安市纪委调查组已对〝陈路别墅〞涉及滥用职权、徇私舞弊的有关人员采取留置措施。
 <br/>
 <br/>
 对于陈路的父亲，报导中并未公开点名，有消息称，陈路的父亲就是西安市人大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_常委.html" target="_blank">
  常委
 </a>
 会原主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈宝根.html" target="_blank">
  陈宝根
 </a>
 。
 <br/>
 <br/>
 公开资料显示，现年65岁的陈宝根是甘肃人，曾在陕西、四川官场深耕23年，四川广元市长，陕西西安副书记、市长、市人大常委会主任等。2017年2月，陈宝根辞去西安市人大常委会主任职务。
 <br/>
 <br/>
 近期陕西官场大震荡，陆媒和自媒体披露多个西安官场变动消息，其中包括西安前市长陈宝根降为科员，继续接受调查；原陕西省委常委钱引安
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 ；西安市长上官吉庆；西安市政协原主席程群力被处分等。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
