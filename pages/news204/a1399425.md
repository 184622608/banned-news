### 阴霾遮蔽北京 司机等半天难辨红绿灯
------------------------

<div class="wysiwyg">
 【新唐人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 时间2018年11月15日讯】冬季刚到，大陆北方的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴霾.html" target="_blank">
  阴霾
 </a>
 卷土重来。北京自14日起阴霾蔽日，街头的红绿灯一度无法辨识，严重影响出行，气象部门发布黄色大雾警报，民众饱受阴霾污染之苦。
 <br/>
 <br/>
 据外媒报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 、河北中南部和东北部等地，均出现特强浓雾。部分地区能见度甚至不足50公尺。14日上午起，北京已封闭13条高速公路。
 <br/>
 <br/>
 网路流传的一段视频中，街头一人向若隐若现的交警喊话：〝警察叔叔，现在红灯绿灯啊？等半天了，看不见！〞。
 <br/>
 <br/>
 据大陆环境监测总站实况平台的消息，14日下午1时，北京市空气品质指数为301，达到"六级严重污染"，这是大陆空气品质等级中最严重级别。15日北京的污染程度进一步加剧。
 <br/>
 <br/>
 《彭博》新闻社指出，与野火灾情惨重的美国加州相比，北京的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_PM2.5.html" target="_blank">
  PM2.5
 </a>
 指数更差。北京市35个监测站点中17个站点达到重度污染，13个站点显示严重污染。部分地区空气中PM2.5浓度，每立方米超过300微克。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
</div>
