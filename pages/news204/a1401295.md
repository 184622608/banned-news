### 广州女律师受辱案 目击证人遭批捕
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】首先来关注中国的消息。广州女律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙世华.html" target="_blank">
  孙世华
 </a>
 两个月前被派出所警察殴打羞辱，两名目击证人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张五洲.html" target="_blank">
  张五洲
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梁颂基.html" target="_blank">
  梁颂基
 </a>
 被当局以涉嫌〝寻釁滋事〞的罪名逮捕。11月28号，张五洲的家属收到逮捕通知书。梁颂基家属透露，当局主要目的是要打压孙世华律师为自己维权。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张五洲.html" target="_blank">
  张五洲
 </a>
 是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙世华.html" target="_blank">
  孙世华
 </a>
 律师被派出所警察羞辱案的目击证人。她在上个月15号被警方拘留。11月28号，家属收到逮捕通知书，罪名是涉嫌所谓的〝寻釁滋事〞。
 <br/>
 <br/>
 张五洲的妹妹张唯楚29号告诉《新唐人》，张五洲被抓后遭受酷刑，又不让会见。看守所管教一直不同意家人送东西过去，所以她很担心姊姊的状况。
 <br/>
 <br/>
 张五洲妹妹张唯楚:〝逮捕通知书昨天才拿到，所以律师还没有来得及去申请会见。（姐姐）平时的身体都还好，但是她这次进去之后说被酷刑，身体状况不太好，我之前去探视的时候，狱警说了，就是她在里面听管教的话，才会允许家里人送衣服、送东西，管教不同意，不可以送东西过去的。〞
 <br/>
 <br/>
 该案的另一名目击证人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梁颂基.html" target="_blank">
  梁颂基
 </a>
 ，上个月14号在北京被广州警方跨省拘捕，罪名也是涉嫌所谓的〝寻釁滋事〞。
 <br/>
 <br/>
 梁颂基的妹妹梁一鸣透露，因为当局施压，为哥哥找律师遇到困难。
 <br/>
 <br/>
 梁颂基妹妹梁一鸣:〝律师还没有办法见到，对他的妻子施压了，解雇了他的律师。广东来说很难找到这方面的律师，敢接这一类案的律师很难找到，呼吁一下全国能不能有律师愿意帮我们。〞
 <br/>
 <br/>
 梁一鸣还表示，根据这段时间警方传唤她的问话，她了解到抓目击证人，强行删除手机上拍到的证据，目的是要打压孙世华律师为自己维权。
 <br/>
 <br/>
 梁一鸣:〝主要的是打压孙世华律师，要逼孙世华律师低头，就是道歉，承认自己有过激的行为，如果梁颂基跟张五洲的罪名成立被判啊，那孙世华律师也必然是有罪的啦，所以梁颂基这个事情，广州这边没有律师敢介入。〞
 <br/>
 <br/>
 孙世华律师是年初被吊照的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 隋牧青的妻子，她9月20号到广州荔湾区华林派出所办案，遭到警察殴打、脱光衣服羞辱。警方事后否认事实，也不公布现场录像。
 <br/>
 <br/>
 新唐人记者熊斌、陈杰采访报导
</div>
