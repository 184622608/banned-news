### 陆2名异议人士逃到台湾 声援退党抵制中共
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】两名原泰国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国难民.html" target="_blank">
  联合国难民
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘兴联.html" target="_blank">
  刘兴联
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颜伯均.html" target="_blank">
  颜伯均
 </a>
 ，上月27日从泰国转机台湾时，称担心遭中共跨国迫害，向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾政府.html" target="_blank">
  台湾政府
 </a>
 请求政治庇护，并获准不被遣返。目前他们一直滞留在台湾机场，期间接受本台电话专访，披露遭中共迫害详情，并支持民众退党。
 <br/>
 <br/>
 滞留台湾机场22天，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆异议人.html" target="_blank">
  大陆异议人
 </a>
 士
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颜伯均.html" target="_blank">
  颜伯均
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘兴联.html" target="_blank">
  刘兴联
 </a>
 等待第三国救援，坦言心情忐忑不安。
 <br/>
 <br/>
 新公民运动成员 颜伯均：〝在这里的22天，我们的心情跌宕起伏，非常复杂。一方面我们希望得到国际社会、人权组织的关注，另一方面我们也受到各个方面的堵截迫害，网络上的诬陷诽谤，再一次请求台湾在第三国接受我们的日子里，让我们能够暂留台湾。〞
 <br/>
 <br/>
 他们称，因为怀疑遭中共公安跟踪，在泰国期间感到不安全，被迫逃到台湾。曾多次遭中共抓捕的刘兴联说，害怕被遣返，因为曾被中共迫害到病危。
 <br/>
 <br/>
 中国人权观察秘书长 刘兴联：〝2015年5月起，我被海口警方秘密抓捕到武汉，在武汉被秘密单独关押了九个月，最后身体发生了严重病业，已经奄奄一息的情况下，才被交保释放。〞
 <br/>
 <br/>
 颜伯钧是北京工商大学副教授，因支持香港占中，参加新公民运动等而入狱。他呼吁全球抵制中共邪恶体制，并表示已经退出中共。
 <br/>
 <br/>
 声音来源：新公民运动成员 颜伯均：〝当时我是在沈阳当老师，东北有很多法轮功学员，要我们退党。他们留了一个邮箱，我就把他发过去了，他们就给我退了。(党团队吗？)退党，退党，退党。马克思在共产党宣言第一句话就说了，是一个幽灵。既然是一个幽灵，我们就呼吁全世界的人抵制幽灵、远离幽灵。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾政府.html" target="_blank">
  台湾政府
 </a>
 表示，不会遣返他们。
 <br/>
 <br/>
 新唐人记者梁珍香港报导
 <br/>
 <br/>
