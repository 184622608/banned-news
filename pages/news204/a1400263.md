### 斥中共行为不端 川普顾问：应该逐出WTO
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】中美贸易战爆发后，美国开始推动针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共贸易不公.html" target="_blank">
  中共贸易不公
 </a>
 问题的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界贸易组织.html" target="_blank">
  世界贸易组织
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 ）改革。日前白宫一位主要经济顾问表示，中共作为WTO一员行为不端，并再度提议将中共逐出WTO。
 <br/>
 <br/>
 周三(11月21日)，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白宫经济顾问.html" target="_blank">
  白宫经济顾问
 </a>
 委员会主席哈塞特(Kevin Hassett)在接受BBC采访时表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 在许多方面令美国失望，特别是在北京问题上。
 <br/>
 <br/>
 他说，美国从没有预料到，一个国家在加入WTO之后，会像中共那样行为不端。对WTO而言，拥有中共这样不守规则的成员，是一个从没遇到过的挑战。
 <br/>
 <br/>
 哈塞特表示，美国通常都能够赢得提交给WTO的案件，但这需要5到6年时间，那时候损失早已造成。由于WTO的处罚措施太轻，〝有些国家〞无所顾忌地违反规则，WTO需要更好地惩治〝这些国家〞。
 <br/>
 <br/>
 哈塞特表示，美国正在考虑能否通过双边谈判，或者改革WTO，甚至将中共逐出WTO来解决这个问题。
 <br/>
 <br/>
 不过，哈塞特解释说，从WTO剔除中共，目前还不是美国拟定的官方政策，但这是美国正在考虑的选项之一。
 <br/>
 <br/>
 哈塞特表示，川普总统希望看到世界上所有国家都能无壁垒地进行贸易。美国对中共施加关税的目的，是在给美国国内造成最小伤害的情况下，迫使中共改变其不公平的贸易行为。
 <br/>
 <br/>
 这并非哈塞特首度提出把中共踢出WTO。今年8月份他就曾公开质问，如果中共不像其他成员国一样遵守规则，为什么还要把它留在WTO？
 <br/>
 <br/>
 而美国贸易代表莱特希泽（Robert Lighthizer）则直言，美国2001年让中共在没有开放市场的情况下就加入WTO是个错误。
 <br/>
 <br/>
 日前，美国已联合日本、欧盟提出WTO改革方案。其中许多条款被指直接针对中共，包括惩罚给本国企业提供经济补贴的成员国等。
 <br/>
 <br/>
 近日，白宫首席经济顾问库德洛（Larry Kudlow）重申了美国对中共的贸易要求：〝零关税、零非关税壁垒、零补贴，停止窃取知识产权、停止强制技术转让，允许美国企业(在中国)拥有自己的公司。〞
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
