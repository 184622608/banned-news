### 为何不顾面子拿下孟宏伟？瑞媒：中共领导层受威胁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席职务的中共公安部副部长孟宏伟，回国后被北京当局紧急抓捕，令际社会一片哗然。有瑞士媒体日前发文分析这起事件指出，北京明知抓捕正担任着国际组织高层职务的官员必然损坏其国际形象还是这么做了，显示中共领导阶层已〝备感威胁〞。
 <br/>
 <br/>
 近日，孟宏伟案在国际社会产生的轰动效应持续发酵。瑞士德文版〝新苏黎世报〞（NZZ）10月9日发表了一篇署名评论文章，分析孟宏伟案必然产生的后效应。文章指出，当时还担任着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席职务的孟宏伟被中共秘密抓捕，让世人看清了一个事实，那就是〝没有人在北京长手臂前是安全的〞。
 <br/>
 <br/>
 该文评论说，习近平上台后发起的反贪腐运动，已〝大大改变中国政治生态〞，不但多名曾大权在握的高层官员中箭落马，而且这些高官在被正式起诉前，接受的调查动辄长达数个月或数年之久。
 <br/>
 <br/>
 而对于这次的孟宏伟案，北京领导人相当清楚，对身份敏感的孟宏伟采取秘密抓捕行动，必然会受到国际社会的密切关注与严厉批评，对于北京想让更多中国籍人士担任国际组织高层职位的努力是一种破坏，但经过〝精心盘算〞后北京当局还是决定逮捕孟宏伟，说明他们判断这样做的效益〝远大于破坏国际形象的高额代价〞。
 <br/>
 <br/>
 文章并进一步分析称，北京方面不愿意等孟的〝主席〞任期结束后再处理他，而是不计代价以贪污罪名紧急把孟宏伟拿下，再次说明中国领导高层已感受到威胁，毕竟对中共来说，〝维持共产党的统治超乎一切〞。
 <br/>

 <br/>
 孟宏伟9月底从法国里昂返回中国后即与在法国的妻子失去了联系。孟妻于10月5日向法国警方报案后，此事件引发国际社会的高度关注。迫于国际舆论的压力，中共官方才向外界承认孟宏伟因涉贪腐案而正在国内接受调查。
 <br/>
 <br/>
 10月7日，总部位于法国里昂的国际刑警组织对外宣布，该组织已收到孟宏伟辞去主席职务的辞呈，立即生效。新任主席将于今年11月在杜拜召开的组织大会中选出。
 <br/>
 <br/>
 其后中共公安部于10月8日发布了一份正式的通报，指孟宏伟涉嫌〝收受贿赂〞正接受中共国家监察委的调查。该通报还特意指称，孟宏伟被查完全是其〝一意孤行、咎由自取〞的结果。
 <br/>
 <br/>
 北大前教授、现旅居美国的夏业良在接受《美国之音》采访时表示，孟宏伟事件说明肯定发生了涉及中共高层的〝大事情〞。
 <br/>
 <br/>
 夏业良分析称，正常情况下，中共一般会使用常规手段来控制官员及其家属，但是本次显然等不及孟宏伟正常回国休假以便将其妻儿一并控制。估计是发生了跟习近平高层有关的、意料之外的重大事件，官方才必须〝紧急出手〞。或者，孟的某些举动让中共高层认为存在可能脱离其控制的迹象，如果不立即办理恐他直接投靠西方。
 <br/>
 <br/>
