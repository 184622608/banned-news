### 《新史记．范冰冰列传》网络爆红
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】日前中共国税总局向知名女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 收取8.8亿元人民币的罚款，对其免予刑责。至此，喧嚣4个月的范冰冰〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞案落幕。有网友7日撰写一篇《新史记．范冰冰列传》在网络爆红。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 因卷入〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞逃税案，从公众视线中消失4个月后，国税总局3日公开对其调查和裁罚结果。税务部门称，已查清范冰冰及其工作室以欺骗手段逃漏税，将向她收取逾8.8亿元人民币，的罚款和未缴税款，缴清税款，将不追究其刑事责任。
 <br/>
 <br/>
 至此，一场〝范冰冰在哪里〞的悬案终于落幕。
 <br/>
 <br/>
 10月7日，有大陆网友撰写《新史记．范冰冰列传》，回顾范冰冰的从影生涯。全文如下：
 <br/>
 <br/>
 范氏冰冰者，莱州人，伶人世家女也，共和三十三年（1981）生。父范涛，业乐坊，善吟咏。母张氏，为舞姬，尝学琵琶于善才，年长色衰，委身为范涛妇。
 <br/>
 <br/>
 及笄，冰冰游学魔都申城，师谢晋。识者或曰：冰冰杏眼桃腮，有沉鱼落雁之姿。其将倾国倾城者，莫非厉害国之冰冰耶欤？ 时魔都多商贾，重利而寡义，冰冰恶之。学既成，念伶人多集于帝都，冰冰遂亦北漂京师矣。
 <br/>
 <br/>
 初入帝都，冰冰携漂友共僦一室。斯室也，与鼠为邻，密不渗风，俗言〝地下室〞者。日食泡面曰〝康师傅〞，艰甚。双亲怜之，欲载之归闾里，冰冰泣曰：〝时艰易克，妾业未成，何颜见莱州父老哉！〞
 <br/>
 <br/>
 戊寅（1998）中，有大导曰孙树培，计划《还珠格格》者。孙索伶人，已得名媛赵薇（小燕子）、林心如（紫薇），唯侍婢金锁者，付之阙如。
 <br/>
 <br/>
 期貌美而性内敛者，方将堪之。 榜立月余，应者若过江之鲫，然皆不和洽。既不得觅，孙导急，斥爪牙曰：〝吾非求良驹，吾所求者，一婢女耳，何难耶？！〞遂亲莅京郊，至怀柔，于千人中见冰冰，颇合其意，与语，大悦。遂擢而叙用之。
 <br/>
 <br/>
 时国人多愚戆，小燕子、紫薇一时燥暴于天下，金锁附骥尾，冰冰竟亦名渐起。时冰冰年方十七耳。
 <br/>
 <br/>
 共和五十六年（2004），冯小刚导《手机》，冰冰携葛优出。冰冰因武月名，艳容乃得为天下人识。时人皆曰：〝冰冰乃狐媚也、艳妾也，胜小燕子、紫薇者，远矣！〞
 <br/>
 <br/>
 其媚容娇形，天下男子悉为之痴狂。 或有闺蜜好事者揆其秘籍，冰冰始言：〝小燕子、紫薇虽美，然不媚也。秋波送娇，瓠犀微露，冁然一笑，皆天然未雕琢也，复以内力助，乃方谐。〞
 <br/>
 <br/>
 使闺蜜效之，凡数十作，闺蜜始略得其彷佛。 冰冰既名，犹自奋不已。十数年间，得名奖如探囊，或东瀛摘冠，或金马折桂，或戛纳压场……凡出秀，皆力压群芳，惊艳寰宇。
 <br/>
 <br/>
 冰冰自是得美誉若〝范爷〞、〝国际范〞者。王者气象，终成矣。
 <br/>
 <br/>
 冰冰王业既成，然犹久未成佳配，盖无意中郎也。
 <br/>
 <br/>
 甲午（2014）春，忽有李公子晨者，宣言曰：吾与冰冰结琴瑟之好矣。李晨亦伶人，登徒子之后也。既闻，范粉皆惊。 戊戌（2018）夏，有崔生永元者，忽示契书于众，言阴阳之契约事也。
 <br/>
 <br/>
 尤言冰冰常作阴阳之契，以避什一之税。崔生怨冰冰日进千万金，赋税甚微。 契书事既出，天下始知。一时也，海内鼎沸，举世大哗。
 <br/>
 <br/>
 有司寻案之。三月余，流言满布，蜚语涨天。或言冰冰已偕李晨遁逸米利坚，或言冰冰被执，或言冰冰已畏罪自引决……中外汹汹不安，人情震骇。
 <br/>
 <br/>
 唯范粉不服，咸曰：避税者，乃厉害国National Sport也，岂独伶人也哉? 是年秋，时十一长假也，邸报忽宣言：冰冰案已厘清，俟输钱八亿八千万入衙廨者，可免囹圄之灾。
 <br/>
 <br/>
 冰冰旋以《悔书》出，其言曰： 妾蒙国恩，伶业有小成，然私心蒙尘，竟隐税数亿金，妾诚知罪孽滔天，死有余辜。今赖王法宽囿，幸免刀斧之诛。
 <br/>
 <br/>
 妾名望虽素出群媛，当复自振，上不负观众恩宠，中不负父母惠育，下不负范粉期冀……
 <br/>
 <br/>
 书既出，天下牙牙，唯李晨哑然。
 <br/>
 <br/>
 东吴野人沈阿瑟曰： 范氏冰冰，起于寒微，以一己之力，临华夏伶界十余载，直以〝范爷〞自王，此其智，必有过人者。
 <br/>
 <br/>
 然营百业者，什一而税，乃今古之成例、中外之通则也，遂为律令。冰冰岁入钱数亿，十数年间，致赀累巨万，几与王者埒富，竟私作阴阳契书以避徭赋，其亦大昏聩矣！
 <br/>
 <br/>
 编户之民，尚知输其家半以助国用，丰饶若冰冰者，失大志，竟匿财自喜，余不知其何益也！
 <br/>
 <br/>
 冰冰方会要冲，适逢时艰，其所遇者，宜矣。俗以崔生永元为起祸者，谬矣。赖今河清海晏，世道朗朗，律法宽仁，冰冰惟输币，犹囿其牢狱之灾，幸矣。
 <br/>
 <br/>
 后之来者，当三思也。
 <br/>
 <br/>
