### 华融前董事长赖小民被逮捕 家中现金三吨
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】中共最高检察院周三通报，前〝中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融.html" target="_blank">
  华融
 </a>
 资产管理公司〞董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 被逮捕。据报，办案人员曾在赖小民住处搜出3吨现金，创下大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融贪腐.html" target="_blank">
  金融贪腐
 </a>
 新纪录。
 <br/>
 <br/>
 指控称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 为个人职务升迁拉关系，美化宣传个人，捞取政治资本、挥霍浪费国家财产，与多名女性搞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_权色交易.html" target="_blank">
  权色交易
 </a>
 等。
 <br/>
 <br/>
 据报，赖小民因贪腐被调查后，办案人员在他的几处房产中搜出3吨重的现金，相当于人民币2.7亿元，放在一起超过3立方米。
 <br/>
 <br/>
 但这还不是赖小民贪腐的全部金额。报导披露，单单赖小民的母亲逢年过节收到的礼金，就多达3亿元人民币。
 <br/>
 <br/>
 据报，此案可能是中共建政以来第一大金融腐败案。
 <br/>
 <br/>
 赖小民2012年9月起担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融.html" target="_blank">
  华融
 </a>
 公司党委书记、董事长。今年4月，因涉嫌严重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违纪违法.html" target="_blank">
  违纪违法
 </a>
 被调查。
</div>
