### 网友吃蛋炒饭庆贺毛岸英被炸死 吁定为中国感恩节
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】周日（11月25日）是中共前党魁毛泽东长子毛岸英的忌日，中国网民吃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蛋炒饭.html" target="_blank">
  蛋炒饭
 </a>
 庆祝，并建议将这一天定为中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 ，因为这一天联合国军队帮助中国摆脱了毛家世袭权力。
 <br/>
 <br/>
 据自由亚洲电台报导，中国大陆微信圈出现大量
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蛋炒饭.html" target="_blank">
  蛋炒饭
 </a>
 的图片，并声称共同庆祝11月25日的〝中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_感恩节.html" target="_blank">
  感恩节
 </a>
 〞。
 <br/>
 <br/>
 有网民别出心裁，将毛泽东、毛岸英和炸死毛岸英的南非空军飞行员的照片，以及蛋炒饭放在一起，以示对联合国军队的感谢。
 <br/>
 <center>
  <br/>

    </a>

   </font>

  <br/>
 </center>
 <br/>
 1950年11月25日，毛岸英在中共军队驻朝鲜的司令部，无视军方规定，擅自起火做当时高级将领都难吃到的蛋炒饭，结果炊烟引来联合国军的飞机，遭轰炸身亡。
 <br/>
 <br/>
 而毛泽东派毛岸英赴朝的目的，被指是积累军功，为将来接班做准备。
 <br/>
 <br/>
 网民呼吁将这一天定为中国感恩节的原因，是因为美国感恩节定于每年11月第四个星期四，和毛岸英忌日很接近。2010年，这两个日子完全重合。
 <br/>
 <br/>
 自由亚洲引述中国一名自由派人士称，这种倡议已流传多年，最开始应该是来自民主人士圈。但官方并没有彻底封杀，这说明针对批评毛泽东的言论管控可能并不是那么严厉。
 <br/>
 <br/>
 另一名观察人士指，网民用含蓄和婉转的方式表达不满，使官方的监控和打压变得很困难。只要有一定常识的人，都会明白民间庆祝毛岸英忌日的真正意图。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
