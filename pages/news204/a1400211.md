### 〝六四〞脱敏？温家宝探望阎明复引猜测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】近日，大陆网路上突然传出一张中共前总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_温家宝.html" target="_blank">
  温家宝
 </a>
 的敏感照片。照片显示，温家宝在医院探望〝六四〞中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被撤职.html" target="_blank">
  被撤职
 </a>
 的前中共统战部部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阎明复.html" target="_blank">
  阎明复
 </a>
 。外界猜测，此举并不是网控漏网，或许是显示当局对〝六四〞有脱敏迹象。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_温家宝.html" target="_blank">
  温家宝
 </a>
 在医院看望前中共统战部部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阎明复.html" target="_blank">
  阎明复
 </a>
 ，为他贺寿的照片，日前在中国网路上悄悄流传。阎明复本月度过87岁生日，他是中共领导人赵紫阳旧部，曾任中共中央统战部长、中共中央书记处书记。他因在〝六四〞期间同情学生曾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被撤职.html" target="_blank">
  被撤职
 </a>
 。
 <br/>
 <br/>
 照片显示，在一个房间内，墙上有一个大大的〝寿〞字，87岁的阎明复坐在轮椅上，温家宝则坐在他旁边的椅子上。照片中还有一名戴口罩的医护人员，一位背对镜头的女士似乎正在说着什么，其他人都静静地听着。
 <br/>
 <br/>
 正值中共〝庆祝改革开放40周年〞之际，网路突然传出这张温家宝探望〝六四〞敏感人物的照片，引发外界诸多猜测。
 <br/>
 <br/>
 1989年〝六四〞事件中，阎明复公开表明同情学生，一直担任调停学生与政府矛盾的角色，多次出面和学生对话。当年5月16日下午，阎明复亲自到广场劝学生返校，但未能扭转局势。
 <br/>
 <br/>
 6月，在中共第十三届四中全会上，阎明复与胡启立、芮杏文一同被免去中央书记处书记的职务。1990年，阎明复又被免去中共中央统战部部长、全国政协副主席以及全国政协党组副书记等职务。
 <br/>
 <br/>
 次年，阎明复复出，担任中共民政部副部长。参与创办中华慈善总会，并于1997年至2002年担任会长一职。
 <br/>
 <br/>
 温家宝在〝六四〞期间担任中央办公厅主任，当时，他也曾陪同赵紫阳前往广场看望学生，劝说学生。
 <br/>
 <br/>
 外界注意到，就在温家宝探望阎明复的照片传出之前，11月18日，湖南省浏阳市举行了胡耀邦塑像的揭幕仪式。胡耀邦长子胡德平，湖南省委副书记乌兰等人为塑像揭幕。这些信息显示，当局似乎对〝六四〞有脱敏迹象。
 <br/>
 <br/>
 当年〝六四〞事件就是由北京的大学生们悼念胡耀邦而引发的，此后，胡耀邦与〝六四〞一起成为中共禁忌话题，他的生辰死忌也被中共视为敏感日。
 <br/>
 <br/>
 据报导，胡耀邦去世后，温家宝每年都去胡耀邦家中探望，直到2017年3月11日胡耀邦妻子李昭去世。
 <br/>
 <br/>
 此外，温家宝在担任中共政治局常委期间，还曾探望中共政协前委员吴祖光。吴祖光当年公开呼吁为〝六四〞平反，并撤掉天安门上悬挂的毛泽东画像，毛的死尸也〝应该烧掉〞。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
