### 中共炫耀贸战货币政策工具 陆学者解读：就是抢劫
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】日前，中共央行行长易纲在G30国际银行研讨会上发言时声称，中共在应对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 方面可利用的货币政策工具还有〝相当的空间〞。这番谈话见诸媒体报导后，有中国大陆金融学者在美国社交媒体上发帖解读称，中共的主要政策工具通俗来说，就是变着方式不断加大〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_放水.html" target="_blank">
  放水
 </a>
 〞力度，总结起来就是〝抢劫〞。
 <br/>
 <br/>
 2018年G30国际银行业研讨会日前在印尼巴厘岛举行。中共央行行长易纲14日在这场研讨会上发言时表示，中美贸易冲突使中国经济下行的风险增加，中国经济正面临〝极大不确定性〞。不过，他同时又声称，中国的物价当前仍然基本平稳，中国经济今年仍然可以实现年初设定的6.5%增长目标。如果需要，中国在利率政策和准备金率仍然有足够的空间调节。
 <br/>
 <br/>
 随后，有人向易纲提问：中国现在下调了存款准备金率，中共政府怎么才能避免资金流入房地产等他们不希望流入的领域？
 <br/>
 <br/>
 易纲当时在回答这次提问时，对〝如何避免资金流入房地产等领域〞的问题避而不谈，却强调说：〝我们在货币政策工具方面还有相当的空间，包括利率、准备金率以及货币条件等……我们的上述工具足以应对不确定性。〞
 <br/>
 <br/>
 上星期，为了稳定中国市场以应对美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，中共政府宣布降准，释放了一兆多人民币来支持社会融资规模的增长，试图以此来降低贸易冲突的负面影响。
 <br/>
 <br/>
 针对易纲的上述谈话和中共政府近期为应对贸易战而采取应对措施，中国大陆金融学者贺江兵10月14日在海外社交网站推特上发文解读称：〝（中共）主要政策工具有：再贷款、中长期借贷便利、短期借贷便利、再贴现、降准…如果看不懂我翻译一下：向银行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_放水.html" target="_blank">
  放水
 </a>
 、短期放水、中期放水、放大水、大放水、实在不行放洪水。鸟的工具啊，总结一句话：抢劫。〞
 <br/>
 <br/>
 ‏当天，他还在推特上发帖感叹称，〝无锚印钞，最直接抢劫的是强国人（中国），其次才是外国人。〞
 <br/>
 <br/>
 贺江兵的另一条帖文又分析称，〝虽然美国可以无锚印钞，然而美国并没有这么做，即便2008年美国出现史上最大的金融危机之第二次，依然没有狂印钞票。倒是中国（中共）疯狂开始印钞始于次年和2010年，美国财政部与美联储联手购买债券和拯救大型金融机构而已。中国（中共）今年公开提出以债为锚印钞，生怕死的慢。〞
 <br/>
 <br/>
 到目前为止，美国对价值约2,500亿美元的中国商品实施惩罚性关税，同时美国总统川普（特朗普）还表示，未来不排除对另外的2,670亿美元中国产品加征关税，这意味着美国政府将对几乎所有来自中国大陆的商品都征收惩罚性关税。中共则到目前为止一共宣布了对大约1,100亿美元的美国产品征收报复性关税。
 <br/>
 <br/>
 今年以来，随着中美贸易冲突日益加剧，外界普遍判断高度依赖出口的中国经济会因此而遭受重创，而中国股市在过去的10个月内下跌幅度已经超过20%。
 <br/>
 <br/>
 （记者何雅婷综合报导）
