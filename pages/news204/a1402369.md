### 孟晚舟国籍又添疑点 丈夫名下两处温哥华豪宅
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】华为太子女、CFO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在温哥华被捕，牵出其是否拥有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大国籍.html" target="_blank">
  加拿大国籍
 </a>
 的疑云。当地媒体披露，有人在温哥华一处住宅区见过孟晚舟，其丈夫名下有两处房产，其中一处价值千万。
 <br/>
 <br/>
 《温哥华太阳报》日前报导，在温哥华一个住宅区，有邻居确认曾在一处住宅见过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 。该报记者到该住宅探询，但无人应门。
 <br/>
 <br/>
 该报通过物业查册发现，2009年及2016年，与孟晚舟丈夫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘晓棕.html" target="_blank">
  刘晓棕
 </a>
 同名的一名男子（Xiaozong Liu）在温哥华分别以270万加元、1500万加元购入两处物业。
 <br/>
 <br/>
 香港《苹果日报》则披露，孟晚舟和父亲一样，拥有香港身份证。并且，孟晚舟和丈夫刘晓棕在香港维港湾共同持有一个复式单位。
 <br/>
 <br/>
 12月6日，加拿大官方公布孟晚舟被捕的消息后，中共使馆指责加拿大拘捕一名未违反本国法律的外国公民是〝侵犯人权〞。但专家指，根据美加签订的司法双边协定，加拿大有权暂时扣留违反美国法律的第三国公民，并引渡当事人至美国。
 <br/>
 <br/>
 不过，中共官方这项指控，引发了网络上对孟晚舟国籍身份的大起底。多名网友指，孟晚舟同时拥有中国国籍和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大国籍.html" target="_blank">
  加拿大国籍
 </a>
 。
 <br/>
 <br/>
 还有记者就孟晚舟被抓时是否持加拿大护照入境追问中共外交部，但中方发言人拒绝回应，只称孟晚舟是中国公民。
 <br/>
 <br/>
 不过网友指，虽然中共最新规定获得外国国籍等同放弃中国国籍，但中共官商界同时拥有两个以上国籍的情况仍然常见。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
</div>
