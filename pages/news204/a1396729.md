### 举报范冰冰及公安后 崔永元要让三部门官员提心吊胆
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】在举报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 涉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演艺圈.html" target="_blank">
  演艺圈
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 之后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 又举报了上海警方和北京警方的〝蛮横〞和腐败。25日，崔永元自爆现在改打法了，下一步的目标，把经侦的、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政府.html" target="_blank">
  政府
 </a>
 的、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_证监会.html" target="_blank">
  证监会
 </a>
 的……〝一个一个扔出来，让它们天天提心吊胆〞。
 <br/>
 <br/>
 10月25日，央视主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 在个人微博〝小崔读书汇〞爆料称，有朋友问他本人是否后悔，崔永元表示后悔，之所以后悔是〝早知今日，还不如把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演艺圈.html" target="_blank">
  演艺圈
 </a>
 的、经侦的、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政府.html" target="_blank">
  政府
 </a>
 的、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_证监会.html" target="_blank">
  证监会
 </a>
 的……全它妈扔出来，心还是太软了〞。
 <br/>
 <br/>
 朋友称，现在也可以一起扔出来，崔永元说，现在改打法了，〝一个一个扔，让它们天天提心吊胆〞。也就是说崔永元下一步瞄准的目标将会是经侦系统，以后还可能是某些政府官员及中共证监会系统官员。
 <br/>
 <br/>
 崔永元5月底在举报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 涉嫌逃税后，10月3日中共税务部门称，范冰冰涉嫌偷税漏税，缴清8.8亿人民币罚款可免予刑责。
 <br/>
 <br/>
 10月7日，崔永元在〝小崔读书汇〞发表题为《一声长叹一声雷》的长文中，自爆因实名举报范冰冰，获得10万人民币奖金，但没心情领取。
 <br/>
 <br/>
 他还自曝受税务总局刁难，警察三次传唤他，追问他手中〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞的来源，他称还遭到警方抄家式报复侦查。
 <br/>
 <br/>
 同时崔还检举经侦警察有份参与某部电影的欺诈，他说，目击上海经侦警察〝当着我的面，喝两万一瓶的酒，抽一千一条的烟，几十万现金用个书包提走。
 <br/>
 <br/>
 10月9日，《检察日报》以《举报人遭威胁，法律如何保护》为题做出回应。呼吁公安机关为举报人提供适当的保护。
 <br/>
 <br/>
 上海警方10日回应说，成立了调查小组，多次联系崔永元未果，希望他主动联系警方配合调查。
 <br/>
 <br/>
 对此，崔永元迅速在今日头条的微头条发文反击说，〝你们把我的原助手、与我合作的公司、与我合作公司合作过的公司都查个底儿掉，居然还联系不上我？〞
 <br/>
 <br/>
 除举报上海警方外，10日晚间，崔永元的微博再发文〝举报公安〞，对北京朝阳分局新源里派出所进行举报与抗议。
 <br/>
 <br/>
 崔还透露自己向公安部门举报遭〝死亡威胁〞，得到的回复永远是〝正在讨论，等待批准〞。
 <br/>
 <br/>
 有分析，崔永元爆料的背后，似乎一直有支持，有高层的角力。〝这是要查上海公安的节奏？这火力，说背后没人谁信？〞
 <br/>
 <br/>
 北京独立学者对美国之音分析说，本次由崔永元担纲的这场娱乐大戏的主角，其前台幕后应该都是中共高层政军之间的角力。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
