### 扬州屋主开车撞强拆队 致1死9伤
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】10月15号上午，江苏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扬州.html" target="_blank">
  扬州
 </a>
 杭集镇发生一起暴力强拆事件，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_屋主.html" target="_blank">
  屋主
 </a>
 拒绝拆迁，遭到强拆人员围殴，挣脱后开车撞向强拆队人员，至少造成1死9伤，目前屋主已
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被警方拘捕.html" target="_blank">
  被警方拘捕
 </a>
 。
 <br/>
 <br/>
 10月15号上午，江苏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扬州.html" target="_blank">
  扬州
 </a>
 杭集镇中心广场附近的一家店面遭到强拆，韦姓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_屋主.html" target="_blank">
  屋主
 </a>
 闻讯赶到现场，因为没有合法手续，屋主拒绝拆迁，结果遭到强拆人员围殴，屋主挣脱后开车，又遭到砸车，车子就冲进强拆队人员聚集的人群里，撞人后，强拆人员又将屋主拖出车外进行殴打。下午官方通报，事件造成1死9伤。
 <br/>
 <br/>
 事发地点附近店家杨先生：〝赔偿未谈好嘛，强行拆迁，是那个主人家开车撞的，人家为什么开车撞？强行拆迁，打人家了嘛，那么多的人，打人家打得没办法了，我们这边拆迁太多了，政府的嘴巴大，老百姓的嘴巴小，你根本说不过他的。〞
 <br/>
 <br/>
 官方通报称，当地一家拆迁公司，受杭集镇防汛防旱指挥部委托对侵占小运河河道的违章建筑物进行拆除。但附近店家透露，官方以治理河道的名义，大搞拆迁，拆的房子没有违建。
 <br/>
 <br/>
 事发地点附近店家施先生：〝屋主行事为什么这么激动啊，听说屋主是被围殴了嘛。不是说的违建，那边就是治理河道，然后做绿化才拆迁的，那要拆很多房子啊，他那个拆迁，每个人都有自己的心理价位对吧，肯定是一定要双方满意对吧。〞
 <br/>
 <br/>
 据《新京报》报导，屋主的妻子表示，被强制拆迁的房子是2000多年建成。今年以来，由于需要疏通小运河，他们和相关部门陆陆续续就拆迁一事在谈，随后拆迁在没有通知的情况下突然发生。
 <br/>
 <br/>
 新唐人记者熊斌、周天采访报导
