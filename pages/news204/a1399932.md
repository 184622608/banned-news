### 黄琦狱中病重 母发声明吁国际关注
------------------------

<div class="wysiwyg">
 【新唐人2018年11月19日讯】首先来关注中国的消息。大陆《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四天网.html" target="_blank">
  六四天网
 </a>
 》创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 的母亲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蒲文清.html" target="_blank">
  蒲文清
 </a>
 ，11月18号发表声明，呼吁国际社会关注黄琦的生命安全，她说被中共关押近两年的黄琦，身患多种严重疾病，在看守所生命堪忧，她担忧儿子被整死，不断替儿子发声，同时她本人的生命也受到威胁。
 <br/>
 <br/>
 现年85岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蒲文清.html" target="_blank">
  蒲文清
 </a>
 ，因担心自己
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_身患绝症.html" target="_blank">
  身患绝症
 </a>
 的儿子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 死在看守所，近两年来，她抱病为儿子四处奔走呼吁，但至今仍无果。
 <br/>
 <br/>
 蒲文清11月18号发表声明说，自黄琦2016年11月被构陷抓捕后，她已经走上了艰难的维权之路，呼吁当局无罪释放黄琦，让患绝症的黄琦回家治病。
 <br/>
 <br/>
 11月19号，蒲文清对新唐人说，中共当局在她上访期间投入了大量警力，随时随地监控她的行踪，最近她家附近出现更多不明身份的人。
 <br/>
 <br/>
 蒲文清：〝我的亲戚朋友家人全部被当局打招呼，不准接近我，所以两年以来，我的家人亲戚以及儿子不敢跟我见面，不敢跟我通话，在这种情况下，没有朋友敢来我这个地方，我在无助之中，我才发出这个声明。〞
 <br/>
 <br/>
 蒲文清哭诉说，目前她的生命受到了严重威胁，但她绝不会放弃为儿子维权，如果她失踪、致伤致残或死亡，责任全在当局。
 <br/>
 <br/>
 蒲文清：〝我从地方到中央一直反映黄琦被构陷的冤案，所有问题都无果，如泥牛入海无回音，加上我目前这种情况，我只有救助于国际社会，我求求大家多给我呼吁，还我儿子，我儿子是无罪的，是他们构陷的。〞
 <br/>
 <br/>
 蒲文清说，尽管两年来她四处奔走，至今都没有见上儿子一面。她希望在自己有生之年，能够见儿子一面。
 <br/>
 <br/>
 现年55岁的黄琦于1999年创立〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四天网.html" target="_blank">
  六四天网
 </a>
 〞，为访民、异议人士、法轮功学员等弱势群体发声，并因此被中共多次判刑入狱。2000年被判处5年徒刑；2008年6月被判刑3年；2016年11月，黄琦再次被抓捕关押至今。
 <br/>
 <br/>
 新唐人记者熊斌、元真采访报导
