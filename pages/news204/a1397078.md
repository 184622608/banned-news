### 日本再现〝江歌案〞？ 25岁中国女研修生浴室被杀（组图）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月28日讯】日本10月27日发生一起凶杀案，一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国女研修生.html" target="_blank">
  中国女研修生
 </a>
 在居住的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公寓浴室被杀.html" target="_blank">
  公寓浴室被杀
 </a>
 害，另有两名中国女性与死者一同居住，日本警方正在对该案进行调查。外界猜测此案或许是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江歌案.html" target="_blank">
  江歌案
 </a>
 〞翻版。
 <br/>
 <br/>
 据日本《共同社》报导，当地时间10月27日上午11点，日本群马县伊势崎市警方接到报警电话，当地居民报警称，发现一名中国女性倒在浴室内。
 <br/>
 <br/>
 警方随即赶往案发地点，在一处公寓2楼浴室发现，25岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国女研修生.html" target="_blank">
  中国女研修生
 </a>
 李雪，头部流血，倒在地上。李雪被送医后死亡，日本警方认为，这是一起杀人案件。
 <br/>
 <br/>
 根据警方通报，李雪和另外2名中国女研修生一起居住。目前，警方正在对这两名中国女性了解情况。
 <br/>
 <br/>
 外界猜测，此案或许是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江歌案.html" target="_blank">
  江歌案
 </a>
 的再现。2016年11月3日凌晨，日本东京都中野区发生一起凶杀案。当日，就读于法政大学的中国女研究生江歌，被中国男子陈世峰在寓所门前用刀刺死，同在案发现场的还有陈世峰的前女友刘鑫。江歌和刘鑫是好友，案发时刘鑫住在江歌寓所。

 近年来，在日本学习的中国研修生事故频发，留学生的安全问题引发关注。1981年，日本设立〝外国人研修制度〞，发展中国家的人才可以在最长3年时间内，在日本边劳动边学习技术。中国是日本研修生的最大来源国。据日本法务省统计，2016年共有8.5万名中国研修生在日本工作。
