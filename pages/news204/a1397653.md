### 湖南摩的司机当街刺死运管所副所长
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南.html" target="_blank">
  湖南
 </a>
 怀化市一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摩的司机.html" target="_blank">
  摩的司机
 </a>
 10月31号刺死
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_运管所副所长.html" target="_blank">
  运管所副所长
 </a>
 。网上消息称，这名摩的司机曾多次遭到对方罚款，才铤而走险。
 <br/>
 <br/>
 10月31号上午，网络流传一段视频，显示一名身穿运管制服的中年男子趴在地上，一动不动。
 <br/>
 <br/>
 网上消息称，事发在怀化市洪江区新民路，洪江区道路
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_运管所副所长.html" target="_blank">
  运管所副所长
 </a>
 张仲凯被罗姓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摩的司机.html" target="_blank">
  摩的司机
 </a>
 连捅数刀死亡 。据说这名摩的司机近期被运管频繁罚款，才铤而走险。 目前嫌疑人已经自首。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南.html" target="_blank">
  湖南
 </a>
 维权人士袁先生：〝(运管所)掌握这方面的一个执法权，就利用这个执法权一直罚款，这种形况很普遍。据说是罚了三次，然后每次1000块，肯定也就是因为这样的原因，才让他恼羞成怒，然后就做出这种举动。〞
 <br/>
 <br/>
 怀化市民黎先生表示，怀化市摩的司机绝大多数来自农村，因为入行的门槛很低，人数不少。而当地政府虽然搞〝禁摩〞行动，但取缔并不严格，可能是为了罚款收入。
 <br/>
 <br/>
 湖南怀化市民黎先生：〝怀化市里面，我们那个的士是六块钱起步，那个摩的是五块起步，大概一趟也就几块钱嘛人民币，基本上就是勉强餬口嘛。交警啊、运管听说很随意性的，都可能会查。政府经常禁摩，以罚款为主，睁一眼闭一只眼，禁不绝嘛。〞
 <br/>
 <br/>
 这段视频在网上引发议论，有网友说：〝求生计，都不行，也是被罚的没法了吧。〞还有网友说：〝冤有头债有主......比那些报复无辜老百姓的强多了。〞
 <br/>
 <br/>
 袁先生：〝最近我们也是接二连三看到很多这样报复的事情，有的是报复社会、有的是报复执法部门、有的是报复其他的相关人员或者部门。可见现在官民之间的矛盾、恨社会、反社会的这种情绪是一种普遍现象。〞
 <br/>
 <br/>
 中共交警及运管部门肆意罚款，曾多次导致冲突事件。例如：今年5月，陕西富县一名交管所中队长查车时与司机发生冲突，双双住院。交管所更纠集黑社会人员对司机进行殴打。今年6月，中国卡车司机大罢工，要求停止对大卡车的随意罚款行为 。
 <br/>
 <br/>
 新唐人记者熊斌、陈杰采访报导
</div>
