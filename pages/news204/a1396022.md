### 数百P2P难友北京市府维权遭强制清场
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】10月17号，来自大陆各地的数百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_P2P难友.html" target="_blank">
  P2P难友
 </a>
 到北京市政府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 ，当局调动大批警力连同数家外媒记者一起清场。同一天还有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_聚财猫.html" target="_blank">
  聚财猫
 </a>
 〞平台的200余名难友在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海市银监会.html" target="_blank">
  上海市银监会
 </a>
 维权。
 <br/>
 <br/>
 据了解，17号上午，银豆网、金联储、联璧金融、抓钱猫、银客、爱钱包等多家平台的数百多名难友联合进京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 。但警方早已掌握信息，在活动前一晚抓捕了5名维权代表。当天，北京市政府停满警车和几十辆大巴，现场难友和数家外媒记者遭到大批特警清场驱散，所拍摄的画面全部被删除。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_P2P难友.html" target="_blank">
  P2P难友
 </a>
 郝英俊：〝去的人数可能有几百吧，各地去的那些平台他们都有，被强制性清场，都遣返了，有一部分可能还没有放，目前的话就说是带头的可能抓了十几个吧，基本上是刑拘了。〞
 <br/>
 <br/>
 当天下午2点多，约有150名银豆网难友聚集市政府附近的东单公园，手举着〝王珺（案件负责人）不作为，蔡奇烤红薯〞标语，高呼〝还我血汗钱〞等口号，又有10人被抓。
 <br/>
 <br/>
 P2P难友质疑警方包庇犯罪，因为各地经侦不作为，迟迟不抓涉案人员，询问案件进展始终一句话：案件正在调查中。
 <br/>
 <br/>
 郝英俊：〝这一次大家又去北京维权也是不得已，我们把这一辈子辛辛苦苦，省吃俭用挣来的血汗钱，投入当时所谓宣传的合法合规的P2P网贷平台，结果爆雷之后，一下子成了非法集资，当地政府也不采取任何措施，而是把责任推卸给企业，这样是典型的不负责任。〞
 <br/>
 <br/>
 此次行动是继8月6号、10月10号，难友们发起的又一次大型维权活动，三次维权失败让难友们感到绝望。
 <br/>
 <br/>
 P2P难友江女士：〝反正已经打压了，我已经死心了，上海那边只发了一个通知，安抚了一下，说延迟了，等呦，就等等等，我不乐观，政府办事不是这样的吗?〞
 <br/>
 <br/>
 另外，17号还有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_聚财猫.html" target="_blank">
  聚财猫
 </a>
 〞平台的200余名难友在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海市银监会.html" target="_blank">
  上海市银监会
 </a>
 维权，被大批警察包围，带离现场，派代表跟官员见面后，不但不解决问题还被威胁。
 <br/>
 <br/>
 新唐人记者熊斌、施宜君采访报导
</div>
