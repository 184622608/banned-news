### 北京敏感时刻为胡耀邦立像 外媒曝幕后原因
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】前中共总书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡耀邦.html" target="_blank">
  胡耀邦
 </a>
 诞辰103周年的前夕，其大型铜像在家乡湖南浏阳揭幕。胡耀邦多年来是中共的禁忌，如今当局允许为胡立像引发舆论关注。有海外媒体分析中共在改革开放40周年之际，批准为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革派.html" target="_blank">
  改革派
 </a>
 大将胡耀邦立像的幕后原因。
 <br/>
 <br/>
 湖南传媒报导，经中共中央、国务院批准，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡耀邦.html" target="_blank">
  胡耀邦
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_塑像.html" target="_blank">
  塑像
 </a>
 落成揭幕仪式，11月18日在他的故乡湖南省浏阳市举行。湖南省委副书记乌兰、长沙市委书记胡衡华，以及胡耀邦之子胡德平、亲属、胡耀邦生前身边工作人员代表等，出席揭幕仪式。
 <br/>
 <br/>
 据报，除了湖南浏阳之外，在江西共青城、浙江大陈岛，都建有胡耀邦的塑像、雕像。
 <br/>
 <br/>
 20日《联合早报》微信公众号引述分析人士的观点称，近些年，中共高层很少同意为领导人建立塑像，在改革开放40周年到来之际，高层批准为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革派.html" target="_blank">
  改革派
 </a>
 大将胡耀邦立像，或为向外界展示继续推进改革开放的决心。
 <br/>
 <br/>
 胡耀邦生于1915年11月20日，1981年6月被邓小平等元老推选为中共主席，1982年改任总书记，但是由于胡属于开明派，引发党内保守派元老不满，1987年被赶下台。1989年4月15日，胡耀邦去世，直接触发后来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_八九民运.html" target="_blank">
  八九民运
 </a>
 和〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四事件.html" target="_blank">
  六四事件
 </a>
 〞。
 <br/>
 <br/>
 当时，时任中共总书记的赵紫阳，也因〝支持爱国学生运动〞而被罢免和软禁十余年。
 <br/>
 <br/>
 胡耀邦离世后，没有葬在八宝山官方公墓，骨灰按照他的遗愿葬在江西九江市共青城。
 <br/>
 <br/>
 多年来胡耀邦和赵紫阳一直是中共的禁忌，他们忌日也成为中共的敏感日，官方不曾有过任何纪念活动，也没有对他们做过任何公开的正面评价。他们的名字也曾在中国大陆销声匿迹多年。
 <br/>
 <br/>
 分析人士认为，如今当局批准为胡耀邦立像本身，说明胡仍被官方视为正面遗产，胡纪念馆、故居以及胡塑像也被定为〝爱国主义教育基地〞。在公众眼里，胡最重要的遗产是推行改革开放，这也是官方在改革开放40周年，为他立像的原因。同时也被认为标志胡政治〝脱敏〞的延续。
 <br/>
 <br/>
 港媒曾报导，胡锦涛和温家宝都曾力主纪念胡耀邦。在是否举行纪念胡耀邦活动的问题上，中共最高层有过激烈争论，当时江派人马黄菊、李长春、罗干表示反对，是胡锦涛力排众议，一锤定音。
 <br/>
 <br/>
 外界认为，胡耀邦、赵紫阳等人，是中共体制内为数不多的具有人性的官员。但他的命运说明，有血有肉的人只要加入中共这部绞肉机，最后都会被绞碎。能够避免被绞的，恰恰是那些对中华民族欠下血债的人，像毛泽东、邓小平、江泽民那样，他们发动的文革、六四屠杀、镇压法轮功等血腥运动，中共至今不肯认错，不肯道歉。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：赵云）
