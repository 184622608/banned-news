### 海外代购爆黑幕 加沃尔玛奶粉惊现中文名
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】近期，海外华人代购爆出黑幕。有网友爆料，在加拿大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沃尔玛超市.html" target="_blank">
  沃尔玛超市
 </a>
 出售的奶粉，罐底部惊现用黑色记号笔手写的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中文名.html" target="_blank">
  中文名
 </a>
 字，而且不只一罐。据信，这是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海外代购.html" target="_blank">
  海外代购
 </a>
 做直播时写给中国买家看的，但奶粉被留在了超市，中国买家收到的奶粉来源可疑。
 <br/>
 <br/>
 近日，网上传出一段视频，一位加拿大网友在一家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沃尔玛超市.html" target="_blank">
  沃尔玛超市
 </a>
 发现有奶粉正在促销，打算买两罐，不料却发现惊天秘密。
 <br/>
 <br/>
 该网友在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奶粉罐.html" target="_blank">
  奶粉罐
 </a>
 底部发现有用黑色记号笔手写的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中文名.html" target="_blank">
  中文名
 </a>
 字，而且一连拿了5罐，全都被写了中文名。
 <br/>
 <br/>
 沃尔玛奶粉惊现中文名的视频曝光时，起初网友们认为这些写了名字的奶粉，应该是代购们买回去之后没卖出去退回来的。
 <br/>
 <br/>
 但据了解，加拿大的超市，不管是costco还是沃尔玛，为保证公众食品安全，食品类的退货绝对不会重新上架。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-9/p9071632a459771444.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-9/p9071632a459771444-ss.jpg"/>
   <br/>
   <font size="-1">
    一些无良代购并没有把写了名字的真奶粉寄给中国卖家。（网络截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 因此，有网友分析，这些写了名字的奶粉应该是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海外代购.html" target="_blank">
  海外代购
 </a>
 在超市做现场直播时，帮中国国内买家在奶粉罐上写下的，好让买家放心。不过，一些无良代购并没有把写了名字的真奶粉寄给中国卖家，而是直接又放回了卖场售货架上。那么那些中国买家收到的写著名字的奶粉，来源就很可疑了。
 <br/>
 <br/>
 自从中国毒奶粉事件曝光后，中国妈妈们以及众多消费者，为了食品安全，不惜花费时间和高昂费用，到海外购买奶粉，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奶粉代购.html" target="_blank">
  奶粉代购
 </a>
 者也因此越来越多。其中一些无良代购为赚黑心钱，使出的骗术也越来越多，直播也被一些代购利用来骗取买家信任。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-9/p9071633a647684700.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-9/p9071633a647684700-ss.jpg"/>
   <br/>
   <font size="-1">
    有网友爆料，就连内裤上也都有手写的中文名，看来也应该是代购惹的祸。（网络截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 事实上，不仅仅是奶粉，有网友爆料，就连内裤上也会出现手写的中文名，看来也应该是代购惹的祸。
 <br/>
 <br/>
 奶粉惊现中文名的视频在网路曝光后，引发网友一片谴责声，纷纷指责这些不良代购的恶劣行径，让海外华人的颜面尽失。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
