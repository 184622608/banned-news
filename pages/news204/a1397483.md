### 中国医师抱怨〝战狼〞不存在 遭约谈
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】电影《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战狼.html" target="_blank">
  战狼
 </a>
 》一度激起不少中国人的〝爱国热情〞。但日前一名中国医生在网上吐槽，〝战狼〞在现实生活中并不存在，自己被困国外时，没有得到大使馆救援。结果他所任职医院声明要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_约谈.html" target="_blank">
  约谈
 </a>
 他，并做出相应处理。
 <br/>
 <br/>
 超强颱风玉兔日前袭击塞班岛，10月26号，中国医学科学院阜外医院医生孙宏涛在微博发文披露，包括他在内有上千名中国游客滞留塞班，但给中国驻洛杉矶大使馆打电话求助，大使馆基本不予理会，所谓〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战狼.html" target="_blank">
  战狼
 </a>
 〞这样的事情现实中是不存在的，大家想都别想。
 <br/>
 <br/>
 孙宏涛的这番话引发热议，有网友认同，有网友声讨，不过很多评论已被删除。
 <br/>
 <br/>
 安徽民众萧先生：〝这个医生说得非常的对，他只是把我们要说的心里话说出来了。那些骂他的人，都是一些五毛、一些机器人，一些言不由衷的人，或者是用心不良的人。〞
 <br/>
 <br/>
 旅加华裔作家盛雪：〝他只是披露了现在他遇到的这么一个现实情况。但是中共用大量的经济利益，用一批人上网去制造舆论。《战狼》其实是它舆论宣导的一个例子而已。时间长了之后，人们真的是会被这些东西所蒙蔽，就失去判断力。〞
 <br/>
 <br/>
 10月29号，孙宏涛任职的中国医学科学院阜外医院向外声明，院内多个部门将联合
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_约谈.html" target="_blank">
  约谈
 </a>
 他，会做出相应处理。
 <br/>
 <br/>
 萧先生：〝各级‘战狼’不是在找他谈话了吗？只是这个‘战狼’不是用来拯救受苦受难的国民的，他是用来对付满腹牢骚的老百姓，和有这种反骨的人的。〞
 <br/>
 <br/>
 盛雪：〝就是一个目的，封住你的嘴，甚至我让你道歉、我让你认错，如果不行的话，那当然他们还会有很多其他的手段。〞
 <br/>
 <br/>
 有网友说，孙宏涛对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_领事馆.html" target="_blank">
  领事馆
 </a>
 等部门发几句牢骚、埋怨，这是任何人都可能出现的情绪。《战狼》不过是一部电影而已，即便是平时，也可以质疑，犯的着上纲上线吗？
 <br/>
 <br/>
 新唐人记者陈汉、陈洁、柏妮采访报导
</div>
