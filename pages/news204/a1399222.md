### 【禁闻】11月13日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 不满基督教堂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_挂五星旗.html" target="_blank">
  挂五星旗
 </a>
 中大博士遭打压
 <br/>
 <br/>
 广东中山大学中国文学博士李保阳，今年8月在微信上发了一条消息，表达对大陆基督教堂悬挂中共五星旗的不满。之后，他被中大校方约谈，并遭警方入室搜查、带走传唤，他的中大研究员职务也被剥夺。日前，李保阳携家人逃亡美国。
 <br/>
 <br/>
 2. 江西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵维权.html" target="_blank">
  老兵维权
 </a>
 遭殴打 送医急救
 <br/>
 <br/>
 11月12号，一些江西老兵到退役军人事务厅维权，等了两个小时无人理睬，老兵们于是前往南昌火车站，但在那里与警方爆发冲突，一名老兵疑似遭警察殴打倒地，被送医抢救。
 <br/>
 <br/>
 3. 秦永敏转监到湖北潜江广华监狱服刑
 <br/>
 <br/>
 湖北武汉的知名政治异见人士秦永敏，今年7月遭中共以〝颠覆国家政权罪〞判处13年有期徒刑后，一直被羁押在武汉市第二看守所。11月13号，维权网发布消息说，他目前已经被转至湖北潜江市广华监狱第七监区服刑。
 <br/>
 <br/>
 4. 江苏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 被关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黑监狱.html" target="_blank">
  黑监狱
 </a>
 卫生纸写信求救
 <br/>
 <br/>
 江苏江阴市访民毛黎惠被当地政府长期关黑监狱，10月30号逃出后，到上级公安局报警，但警方又将他交回当地维稳部门继续拘禁。11月8号，毛黎惠用卫生纸写出求救信，希望外界解救他。
</div>
