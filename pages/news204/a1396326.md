### 孟宏伟落马惊动郭声琨？政法委紧急喊话意味深
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】原国际刑警组织主席、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被捕的消息公布后，中共政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭声琨.html" target="_blank">
  郭声琨
 </a>
 的命运也引发关注。郭日前在一项推进会紧急喊话，中共政法机关要敢于〝刀刃向内〞。分析认为，郭的表述信息量巨大，或意味着孟宏伟被捕后，政法系统也面临大整肃。
 <br/>
 <br/>
 10月16日至17日，在湖北省武汉市召开的一项推进会上，中共中央政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭声琨.html" target="_blank">
  郭声琨
 </a>
 特别强调，要完善中共政法机关的〝内部清理机制〞，〝各级政法机关要敢于‘刀刃向内’〞，不仅要清除内部充当黑恶势力的〝害群之马〞，对〝不担当、不作为〞的中共政法官员也要追究责任。
 <br/>
 <br/>
 有分析认为，郭的上述表述信息量巨大，意味着北京当局将着手彻底清理中共政法系统的积弊，在数轮针对各领域各层面的整顿后，北京终于开始向〝刀把子〞动刀。
 <br/>
 <br/>
 新唐人此前分析，原国际刑警组织主席、原中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，被诱捕回国遭中纪委调查，很可能就是习近平着手清理政法系统的一个切入点。
 <br/>
 <br/>
 中共官方10月7日深夜通报：孟宏伟涉嫌违法，目前正接受中共监委监察调查。这是孟妻就丈夫9月底只身返回中国失联，向法国里昂警方报警之后，中共官方的首次回应。
 <br/>
 <br/>
 中共公安部官网10月8日即于头条位置刊文称，公安部部长赵克志当日凌晨，紧急召开公安部党委会议，就孟宏伟被调查一事进行通报，并宣布成立工作组，查处孟的同党。
 <br/>
 <br/>
 与公安部形成鲜明对比的是，中共政法委并未在第一时间就此事作出反应。
 <br/>
 <br/>
 据报，中共政法委书记郭声琨与孟宏伟关系密切，两人即是湖南中南工业大学校友，同时又都曾担任公安部正副部长。
 <br/>
 <br/>
 港媒评论称，中共公安部虽对孟案表态，但中央政法委未表态，箇中玄机，值得留意。评论还称，按官方通报暗示，孟宏伟案似乎涉及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 等政法系统贪官旧案，但会否牵连他近年的上司孟建柱、郭声琨仍需观察。
 <br/>
 <br/>
 而郭声琨此次强调要〝刀刃向内〞，究竟是真心协助习近平清理政法系统，还是做表面文章，甚至是有意搅混水，外界仍在观望。
 <br/>
 <br/>
 中共公安部过去20年来，一直被江派人马把持，历任部长分别为江派的要员贾春旺、周永康、孟建柱、郭声琨。
 <br/>
 <br/>
 孟宏伟曾任中共公安部副部长长达14年，期间担任部长的周永康，是江泽民利益集团的核心人物，〝政法系〞被称为周永康的〝独立王国〞。
 <br/>
 <br/>
 江泽民派系不仅通过中共政法系统打压民众迫害人权人士，而且将〝刀把子〞用于中共内部派系斗争。
 <br/>
 <br/>
 周永康落马后，习近平一直未能彻底清理中共政法系统，郭声琨与委员周强、傅政华等都有浓厚的江泽民派系色彩。而孟建柱和郭声琨更是江派二号人物曾庆红的嫡系。
 <br/>
 <br/>
 有海外媒体分析，孟宏伟的落马或将牵动中共公安系统，甚至整个政法系统的又一轮反腐风暴，目前已经浮出水面的郭声琨和孟建柱等人否受到牵连，还有待观察。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
