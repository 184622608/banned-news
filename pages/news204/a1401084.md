### 川习会想达成协议？ 库德洛向北京提四个〝必须〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞在即，美中双方已为此进行了会前沟通。白宫首席经济顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 （Larry Kudlow）日前在记者会上表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府对北京的回复不太满意，北京的立场没有变化，他对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 停火的前景表示怀疑。库德洛称，如果北京想要在〝川习会〞上达成协议，必须做到四点。
 <br/>
 <br/>
 二十国领导人（G20）峰会将于11月30日至12月1日在阿根廷举行。12月1日，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）将与中国国家主席习近平，在阿根廷首都布宜诺斯艾利斯举行晚宴会谈。
 <br/>
 <br/>
 此次会谈关系到美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的走向，双方对此十分慎重。在会谈前，美中政府重启了各级部门之间的讨论。
 <br/>
 <br/>
 当地时间11月27日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 在白宫记者会上表示，要想结束贸易战，北京需要做得更多。即将召开的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞为打破贸易僵局提供了一个机会。
 <br/>
 <br/>
 库德洛称，川普总统认为此次美中峰会，双方〝很有可能达成协议〞，他对此持开放态度。但他同时强调，北京要想达成协议必须符合公平、互惠的原则。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-28/p9146581a397539909.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-28/p9146581a397539909-ss.jpg"/>
   <br/>
   <font size="-1">
    白宫首席经济顾问库德洛（Larry Kudlow）日前在记者会上表示，川普政府对北京的回复不太满意，如果北京想要在〝川习会〞上达成协议，必须做到四点。（ NICHOLAS KAMM/AFP/Getty Images）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 库德洛指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京必须做到四点.html" target="_blank">
  北京必须做到四点
 </a>
 :〝知识产权盗窃的问题必须要解决，强制技术转让的问题必须要解决，重大的关税与非关税壁垒的问题必须要解决，所有制的问题必须要解决。〞
 <br/>
 <br/>
 〝川普总统可能会重申他的观点，那就是我们想要世界在理想情况下实现零关税、零非关税壁垒和零补贴。〞库德洛称。
 <br/>
 <br/>
 不过，他表示自己对贸易战停火的前景并不乐观。在这次峰会之前举行的谈判没有取得进展，北京的立场没有变化，新一轮关税很可能按计划出台。
 <br/>
 <br/>
 〝我们现在与中方各级政府进行了很多沟通〞，库德洛说，彼此都向对方介绍了双方的问题和历史，并提出谈话要点。
 <br/>
 <br/>
 但他表示看不出北京的姿态有太大变化，〝他们如此抵制变化，只会说我们必须保护我们的国家，仅此而已。〞
 <br/>
 <br/>
 库德洛说，〝我认为总统展现出硬脊梁骨……打破这些中国城墙是完全正确的。〞〝我们必须保护我们的技术，我们的创造力和创新。〞
 <br/>
 <br/>
 〝我非常失望，我知道总统非常失望，因此周六晚上的晚餐……是一个翻新页面的机会，一个突破的机会。〞库德洛说，〝习主席可以站出来，为我们提出一些新的想法。〞
 <br/>
 <br/>
 在此之前，川普总统已提前向北京给出了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谈判底线.html" target="_blank">
  谈判底线
 </a>
 。川普26日接受《华尔街日报》采访时说，达成协议只有一个机会，那就是〝北京必须向美国开放，和美国公平竞争。〞
 <br/>
 <br/>
 同时，川普表示，华盛顿方面很可能按计划把针对2000亿美元中国输美商品的关税税率从当前的10%提高到25%。川普说，他〝极不可能〞接受北京方面有关推迟提高关税的要求。
 <br/>
 <br/>
 另外，川普还强调，如果和习近平未能达成协议，他很可能会对其余的2670亿美元中国输美商品加征关税。
 <br/>
 <br/>
 北京方面对此回应称，已开始对外国投资开放。27日，一直与美国进行贸易谈判的中共国务院副总理刘鹤在汉堡发表讲话时表示，中共领导人正在加快电信、教育、医疗服务和文化领域的开放。中国经济的未来对外开放的大门只会越开越大。
 <br/>
 <br/>
 但是，库德洛表示，看不出中方在贸易谈判中有什么开放态度。
 <br/>
 <br/>
 到目前为止，白宫没有对川习晚宴提供细节，也尚未透露谁将参加晚宴。库德洛表示，两位首脑的夫人都可能参与晚宴，同时他提醒贸易不会是两位领导人之间唯一的讨论话题。
 <br/>
 <br/>
 川普总统内阁的几位成员将出席G20峰会，其中包括财政部长姆钦、贸易代表莱特希泽，以及国务卿蓬佩奥和国家安全顾问博尔顿。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑:戴明）
