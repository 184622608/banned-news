### 川普习近平同意暂定G20前一天场边会谈
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】香港《南华早报》周五（19日）引述消息人士的话说，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）和中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 已初步同意，将于11月在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 举行的G20集团峰会上会面。
 <br/>
 <br/>
 2018年G20集团峰会将于11月30日至12月1日，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 首都布宜诺斯艾利斯（Buenos Aires，当地华人常略称为〝布宜诺市〞或〝布市〞）举行。
 <br/>
 <br/>
 《南华早报》引述一位不愿透露姓名的中共官员的话说，拟议的会谈时间对两位领导人来说都可行，因为到11月底，他们已解决各自的国内议程。
 <br/>
 <br/>
 报导称，双方会面日期初定在11月29日，即
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 正式开始的前一天；地点暂定于峰会会场边进行。
 <br/>
 <br/>
 与此同时，中共国务院副总理刘鹤周五（19日）也向官媒透露，北京和华盛顿〝目前正在接触〞。但刘鹤并未提及更多细节。
 <br/>
 <br/>
 正值
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易.html" target="_blank">
  中美贸易
 </a>
 冲突不断升级之际，川习的第二次会面被外界解读为，双方有意降低贸易冲突导致的紧张关系。
 <br/>
 <br/>
 有分析认为，这或许表明中国（中共）已放弃对美贸易议程的阻挠。如果双方确认会面，这将是一年来两国领导人之间第一次面对面会谈，并表明华盛顿和北京都准备好缓和贸易紧张局势。
 <br/>
 <br/>
 华盛顿派智库美国企业研究所（American Enterprise Institute）中国问题研究员史剑道（Derek Scissors）表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 〝要会谈，除非有什么事情发生〞。
 <br/>
 <br/>
 <h4>
  处处留余地 川普：不想中国经济下挫
 </h4>
 <br/>
 美国财政部10月17日公布众所瞩目的《美国主要贸易伙伴外汇政策报告》，中国并未被列入〝汇率操纵国〞，但中国与日本、韩国、印度、德国、瑞士继续留在〝观察名单〞。
 <br/>
 <br/>
 不过，中美贸易冲突自7月初升级以来，中国各项经济数据随即开始下滑，包括股市的长期不景气和人民币汇率持续走低等，中共官方更甚喊出〝自力更生〞的口号，折射出北京已陷入困境。
 <br/>
 <br/>
 川普近期也频频透露，中方受到巨大压力，想要恢复谈判，达成协议。
 <br/>
 <br/>
 国际时事评论员唐浩分析认为，美方此次未将中国划入〝汇率操纵国〞，从中或可看出，美方并不想〝趁人之危、趁火打劫〞，也无意将中共〝置之死地〞。
 <br/>
 <br/>
 美国总统川普8月8日与企业家们餐叙时，就曾表示，〝我们现在与中国发生一场小小的争斗，我们（经济）持续上扬，但我也不想要他们（中国）下挫。〞
 <br/>
 <br/>
 此外，川普上周日（14日）接受哥伦比亚广播公司（CBS）专访时，用〝小冲突（skirmish）〞一词来指称当前的中美贸易对抗，而非〝贸易战〞，并再度表示，他仍对习近平保有尊敬。
 <br/>
 <br/>
 〝我跟中国习主席有着很棒的默契，但我不知道是否有必要持续下去。〞川普说。
 <br/>
 <br/>
 唐浩分析指出，川普对中方处处留下余地，无疑也是为11月底G20峰会上，川习二人将再次会面，提前释出善意。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
