### 【禁闻】10月7日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 江西上饶楼盘降价 业主打砸售楼部
 <br/>
 <br/>
 江西上饶知名楼盘〝碧桂园信州府〞，因价格从10000元降到7000，引发业主不满，10月4号，大量业主打砸售楼部。
 <br/>
 <br/>
 2. 湖北
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 夫妇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 被打死打伤
 <br/>
 <br/>
 湖北省黄石市阳新县白沙镇村民柯爱群、罗克欢夫妇，2014年9月17号因工伤赔偿问题到镇信访办上访，遭到工作人员暴力殴打，之后罗克欢不治死亡，柯爱群重伤，事后，当地政府以将柯爱群和子女伤害致残、致死相威胁，逼迫他们签订〝息访协议〞书，而打人凶手则至今逍遥法外。
 <br/>
 <br/>
 3. 北京幼儿被抢未遂 嫌犯仅被拘五天
 <br/>
 <br/>
 10月2号，三名妇女在北京丰台区大红门银泰百货内抢夺一名未满周岁的孩子，被周围民众抓获，但丰台警方只以〝扰乱公共场所秩序罪〞对他们拘留5天。受害家长认为这是一起拐卖儿童未遂案，向警方提出复议。
 <br/>
 <br/>
 4. 被关精神病院两年 重庆访民讲述经历
 <br/>
 <br/>
 重庆合川区访民张芬，两年前被地方维稳人员从北京带回原籍后，先后送到当地两家精神病院关押，期间因不堪折磨吞玻璃自杀，被剖腹抢救。今年8月30号获释后，仍然被限制自由，10月4号，她才得以脱身找到朋友，讲述自己的遭遇。
</div>
