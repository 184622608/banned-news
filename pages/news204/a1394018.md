### 防局势失控？北京邀蓬佩奥访华重启安全对话
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】中共外交部日前公布，美国国务卿蓬佩奥将于近期访华，中美双方将就〝双边、区域和全球议题〞进行商讨。有分析认为，北京邀
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥访华.html" target="_blank">
  蓬佩奥访华
 </a>
 重启对话，应是为防止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美局势.html" target="_blank">
  中美局势
 </a>
 失控之举。此前，美媒消息称，中方取消了原定的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美外交.html" target="_blank">
  中美外交
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安全对话.html" target="_blank">
  安全对话
 </a>
 。
 <br/>
 <br/>
 中共外交部10月3日宣布：经中美双方商定，美国国务卿蓬佩奥（Mike Pompeo）将于10月8日访问北京。中美双方将就中美关系及共同关心的国际和地区问题交换意见。
 <br/>
 <br/>
 蓬佩奥在访问北京之前，将先对朝鲜、日本、韩国进行访问。美国国务院2日宣布，蓬佩奥此次东亚之行为期3天(6日~8日)，将访问东亚4国，分别与日本首相安倍晋三、外相河野太郎、韩国总统文在寅和外长康京和会面，并在7日与朝鲜领导人金正恩在平壤会面。北京是蓬佩奥东亚之行的最后一站。
 <br/>
 <br/>
 蓬佩奥此次访华，适逢美国防长马蒂斯刚刚取消原定于10月中旬在北京举行的第二轮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美外交.html" target="_blank">
  中美外交
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安全对话.html" target="_blank">
  安全对话
 </a>
 。美媒此前预计，蓬佩奥将与马蒂斯一同出席。
 <br/>
 <br/>
 美国国防部10月1日宣布，马蒂斯10月访华取消。美媒称，是中方以〝没有时间〞为由，取消了对话。
 <br/>
 <br/>
 有分析认为，北京此次邀请
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥访华.html" target="_blank">
  蓬佩奥访华
 </a>
 ，重启安全对话，为防止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美局势.html" target="_blank">
  中美局势
 </a>
 失控的意味浓厚。
 <br/>
 <br/>
 近日，美中对抗进入新阶段。川普政府将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易.html" target="_blank">
  中美贸易
 </a>
 战火升级，制裁中共军头，批准对台军售，派B-52轰炸机和导弹驱逐舰直闯南海，以及在联大谴责北京干预美国中期选举等等举措，都令中共惶恐不安。
 <br/>
 <br/>
 10月2日，中共外交部发言人华春莹表示，中美对话只是〝推迟〞，并指是美方提出希望推迟。声称中美双方将继续就举行对话保持沟通。
 <br/>
 <br/>
 外界认为，北京将叫停外交安全对话的〝主动权〞让给美方，比较罕见。在数月的中美贸易冲突中，每当美方推出惩罚措施时，中方虽无反制之力，但都以〝拒绝谈判〞的腔调展示〝底气〞。
 <br/>
 <br/>
 对于此次北京重启对话，美国国务院发言人希瑟．奈特(Heather Nauert)表示：在任何关系之中都会遇到低点，但是关系会随时间高低起伏，所以美国肯定有很多东西跟北京谈。
 <br/>
 <br/>
 奈特还表示，蓬佩奥此行最主要的目的是针对朝鲜弃核问题。国务卿在一年之内第四次访朝，显示谈判有进展和充满动力。〝当然，我们还有一段路要走，但我们期待这次会谈能踏出下一步。〞
 <br/>
 <br/>
 此前，川普（特朗普）总统取消了蓬佩奥8月的访朝行程，因为朝鲜在弃核问题上出现反覆，川普决定先解决与中共的贸易纠纷。他暗指北京在朝鲜问题上起到不良作用。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
