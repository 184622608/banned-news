### 辽宁数万人抗议建垃圾焚烧厂 老人孕妇遭警殴
------------------------

<div class="wysiwyg">
 【新唐人2018年12月06日讯】首先来关注中国的消息。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁鞍山市.html" target="_blank">
  辽宁鞍山市
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_腾鳌镇.html" target="_blank">
  腾鳌镇
 </a>
 政府未经居民同意，兴建大型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_垃圾焚烧厂.html" target="_blank">
  垃圾焚烧厂
 </a>
 ，引发抗议。12月5号，上万民众与警方爆发激烈冲突，包括老人、孩子和孕妇等多人被打。6号上午，双方仍持续对峙。
 <br/>
 <br/>
 当地居民李先生6号早上告诉《新唐人》，他居住的地方就有六、七千人走上街头抗议兴建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_垃圾焚烧厂.html" target="_blank">
  垃圾焚烧厂
 </a>
 ，当局从沈阳调来了武警和公安，双方对峙。
 <br/>
 <br/>
 当地居民李先生：〝门口正在聚，现在门口，人就一直在走，得有六、七千人，来很多特警，来好几百，都是从沈阳调来的特警。警察也有，特警也有，他们手握盾牌、棍子。〞
 <br/>
 <br/>
 当地居民张女士也透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_腾鳌镇.html" target="_blank">
  腾鳌镇
 </a>
 有三四个地方的居民出来抗议，当局也从大连调来了特警。
 <br/>
 <br/>
 当地居民张女士：〝我听说大连那边都调来特警了，地段不同、三、四个地方，一个地方得有五、六千人，这三个地方就是说下高速或者说鞍山那边过来的主要要道。〞
 <br/>
 <br/>
 李先生透露，5号，共有两万居民上街抗议，包括老人、孩子、孕妇在内的多位民众被打。
 <br/>
 <br/>
 李先生：〝统计吧，因为各个地方都是，昨天两万人应该得有，给老百姓狠打，孕妇、老人、孩子，什么他们都打，他们不管你是谁。〞
 <br/>
 <br/>
 张女士表示，他们日前才得知腾鳌镇政府偷偷兴建大型垃圾焚烧厂，李先生则表达了愤慨。
 <br/>
 <br/>
 张女士：〝百姓因为污染都不同意，因为这个污染环境，对人体伤害太大了。〞
 <br/>
 <br/>
 李先生：〝它离我们正常居民住的地方两公里都不到，人家老百姓能干吗，一天就处理垃圾四、五百吨，四、五百吨垃圾烧完之后，它不可能一点烟一点味没有吧。〞
 <br/>
 <br/>
 李先生说，这个大型垃圾焚烧厂一旦建成，居民花五、六十万买的房子，立刻就不值钱。一名当地私营业主表示，腾鳌镇有很多化工厂和钢厂，政府建的是大型垃圾焚烧厂，医疗垃圾都在这里处理，产生的有毒气体将影响镇上12万人的生活。
 <br/>
 <br/>
 新唐人记者常春、陈洁、元真采访报导
 <br/>
 <br/>
