### 外媒卧底视频 揭露新疆〝再教育营〞真容
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】中共在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育营.html" target="_blank">
  再教育营
 </a>
 〞大规模拘押维族人的行为受到国际社会的强烈谴责。意大利杂志《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_寒冬.html" target="_blank">
  寒冬
 </a>
 》的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卧底记者.html" target="_blank">
  卧底记者
 </a>
 ，揭露了〝再教育营〞的内幕。
 <br/>
 <br/>
 这是一座位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 伊宁市英也尔乡的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育营.html" target="_blank">
  再教育营
 </a>
 〞，被中共称作〝伊宁市英也尔职业技能教育培训中心〞，其实是中共对新疆维吾尔人进行洗脑转化的地方。
 <br/>
 <br/>
 此地原本是一块空地，也是民兵训练的打靶场。
 <br/>
 <br/>
 总部位于意大利的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_寒冬.html" target="_blank">
  寒冬
 </a>
 》杂志的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卧底记者.html" target="_blank">
  卧底记者
 </a>
 在8月潜入这里，带回了宝贵的一手资料。
 <br/>
 <br/>
 这所〝转化营〞拥有4座建筑楼群，每个楼群都有4层，建筑面积约11万平方米。
 <br/>
 <br/>
 据内部人士透露，这里的每个房间可住15个人，每层有28个这样的房间，整个〝转化营〞可关押6000多人。
 <br/>
 <br/>
 9月份，该转化营的一名工作人员向《寒冬》杂志记者透露，此地已关押数千人。
 <br/>
 <br/>
 大楼的设置就和监狱一样，所有窗户都装了严密的铁栏杆或者用铁丝网封堵。
 <br/>
 <br/>
 楼道出口处安装三个摄像头，进行全方位监控。
 <br/>
 <br/>
 大楼里的房间和监狱牢房一样，每个房间都设有双层铁门，外面加防护栏杆，并装有电子锁。
 <br/>
 <br/>
 监控摄像头遍布转化营的每一个角落，包括教室，宿舍，走廊，甚至是厕所。被关押者24小时都处于严密监控之下，完全没有自由。
 <br/>
 <br/>
 新唐人记者王羽音综合报导
