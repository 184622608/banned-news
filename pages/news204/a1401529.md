### 【禁闻】中国首例〝基因编辑婴儿〞背后推手是谁？
------------------------

<div class="wysiwyg">
 【新唐人2018年12月01日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 〞事件持续发酵，中共有关部门11月29号称，该事件〝性质恶劣〞，将查处相关人员，不过，有评论指出，〝基因编辑婴儿〞并非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 个人行为，背后另有推手。
 <br/>
 <br/>
 在中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南方科技大学.html" target="_blank">
  南方科技大学
 </a>
 副教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 因首创〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 〞，而遭舆论声讨的同时，中共国家卫健委、科技部、科协等三部门负责人29号表示，这起事件性质极其恶劣，已要求有关单位暂停相关人员的科研活动，并对违法违规行为予以查处。
 <br/>
 <br/>
 据报导，贺建奎至少是7家公司的股东，这7家公司的总注册资本为1.51亿元人民币，其中注册资本最高的是深圳市南科生命科技公司，南方科技大学旗下深圳市南科大资产经营管理公司持有该公司24.5%的股份。
 <br/>
 <br/>
 报导还说，参与试验志愿者的知情同意书列明，南科大承担每对夫妇28万元人民币的试验费用。
 <br/>
 <br/>
 中国问题专家横河表示，〝基因编辑婴儿〞绝不是贺建奎个人行为，因为〝基因编辑婴儿〞需要两大系统，一个是体外受精系统，另一个是单细胞里面的基因编辑系统，这要有相当规模、相当级别的医院才能做这件事，而且整个过程牵涉庞大的资金。
 <br/>
 <br/>
 中国问题专家横河：〝从立项到批准，到资金，到医疗的设备和医疗的病房等等这一系列的问题，绝对不是哪一个人，甚至都不是哪一个公司或者是哪一个小医院能够完成的。所以他的背后肯定有超出深圳这个医院和超出他所在的学校的背景在。〞
 <br/>
 <br/>
 化名〝草祭〞的原上海某大学理学教授在推特上指出，〝基因编辑婴儿〞实验是由中共上层推动，南方科技大学负责实施，贺建奎具体执行的一项秘密计划。从4个迹象不难看出中共是幕后的操盘手。上亿以上研究经费不是一个副教授能独立争取到的，背后一定有中共科技部支持。贺建奎是中共通过〝千人计划〞从海外引进的人才，长期在南科大留职停薪做研究，没有上层支持，不可能办到。
 <br/>
 <br/>
 事实上，基因编辑被中共列入2016年制定的《〝十三五〞国家战略性新兴产业发展规划》，中共声称要推进基因编辑技术研发与应用，促进基于基因编辑研究的临床转化和产业发展。
 <br/>
 <br/>
 微信公众号〝偶尔治愈〞日前发表文章说，中国的公众和官方都十分期待本土科学家能够在基因领域取得突破性的进展。在基因研究上，科学已经扛起为民族、国家争夺荣耀的大旗。
 <br/>
 <br/>
 横河表示，中共推进基因编辑研究，除了展现中国崛起外，还符合了特权阶层的利益，这些人对延年益寿之类的东西特别感兴趣。
 <br/>
 <br/>
 横河：〝为什么中国的移植行业得到非常高的支持，就是因为这些高官年纪大了以后，哪个器官不行了就换掉，像这种基因的话也是一样的，某些特权阶层的人可以从当中得到好处的这些项目，中共会大力支持。〞
 <br/>
 <br/>
 文章还说，中国盛产〝科学狂人〞，除了贺建奎，还有痴迷于〝换头术〞的哈尔滨医大教授任晓平，以及用电击治疗孩子网瘾，发表近百篇论文的精神科医生杨永信。第一例基因编辑婴儿诞生在中国，可能是一种必然。
 <br/>
 <br/>
 美国西达赛奈医学中心博士后科学家陈力：〝中国的体制不健全，所以就会有这种喜欢名利的人不择手段来做这些事情。〞
 <br/>
 <br/>
 横河：〝在无神论的宣传下培养了几代无神论者，导致了他们可以拿人随便做实验，没有一个道德障碍。再加上中共当局的鼓励，这个事情所以为什么会在中国发生，而没有在世界其它的国家发生。〞
 <br/>
 <br/>
 横河表示，换头术实际上是任晓平的意大利合作伙伴卡纳维罗想出来的，但他在意大利做不了，只好到中国做，因为在中国，既没有法律约束力，也没有道德约束。
 <br/>
 <br/>
 采访/常春 编辑/陈洁 后制/葛雷
