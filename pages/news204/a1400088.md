### 西安货车车主抗议新规  集体大罢工
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陕西西安.html" target="_blank">
  陕西西安
 </a>
 市数百名车主已经持续半个月集体到政府部门维权。他们拥有排气标准为国III、国Ⅳ的柴油货车，都反对当局实施限时进入三环等区域的新规定。11月19号，车主
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集体大罢工.html" target="_blank">
  集体大罢工
 </a>
 ，其中四百多人聚集到市交警支队及市政府抗议，遭到警方驱散。
 <br/>
 <br/>
 西安市从11月16号开始，对国III、国Ⅳ的柴油货车实施限时进入三环、绕城高速部分区域的规定 ，大批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货车车主.html" target="_blank">
  货车车主
 </a>
 因此被断了生路。从7号起，几百名货车车主开始维权。
 <br/>
 <br/>
 西安货车车主透露，参与维权的大都是个体车主，少部分是物流公司。他们的维权行动都受到打压。
 <br/>
 <br/>
 西安货车车主高先生:〝这规定基本上已经不可逆了，因为政府既然已经决定做这个事了，那些应急措施啊什么的，对于你来上访，肯定是不会起到很大的一个作用。〞
 <br/>
 <br/>
 西安货车车主还表示，政府的规定改变太快，引发不满。国Ⅳ的车跑了4年，国V的车去年才上市的，现在又让买国VI的，一部车几十万。车主都是贷款买车，损失都得自己买单。
 <br/>
 <br/>
 高先生:〝国III的车再维权可能是没用，国Ⅳ的车稍微能过渡一下，但是能缓冲多长时间就不好说。尤其是物流公司这块，自有车辆都是油车这种，影响很大，已经有很多物流公司在开始换新。〞
 <br/>
 <br/>
 车主上访维权至今已半个月，但大陆媒体只字未报导。西安车行经理葛先生希望媒体能关注，他认为政府的新规定不合理，要先让车主赚钱才有能力换车。
 <br/>
 <br/>
 西安车行经理葛先生:〝他是要禁止，我觉得这个事情不应该禁止，我建议你们报导一下，然后让不要禁止，国III呀、国Ⅳ都能跑的话，那车主也能挣到钱，能挣到钱的话，买车的人多了，我们生意也就好了嘛。〞
 <br/>
 <br/>
 此前在11月7号、8号，几百名西安货车车主集体到交警支队及市政府、信访办抗议，政府派出特警抓了5人。19号，包括周边县市区的货车车主
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集体大罢工.html" target="_blank">
  集体大罢工
 </a>
 ，其中四百多人聚集市交警支队及市政府抗议，遭到警方驱散。
 <br/>
 <br/>
 新唐人记者熊斌、周天采访报导
 <br/>
