### 美国土安全部长：选举在即 中共正发动空前行动施影响
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国土安全部.html" target="_blank">
  美国国土安全部
 </a>
 长日前表示，中共正在发动空前行动，试图干扰美国选举。近来，白宫官员接连谴责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共干预美国选举.html" target="_blank">
  中共干预美国选举
 </a>
 ，美国网络安全公司也透露，中共黑客正加紧攻击美国政治团体。
 <br/>
 <br/>
 台媒中央社报导，当地时间10月10日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国土安全部.html" target="_blank">
  美国国土安全部
 </a>
 长尼尔森对参议院国土安全及政府事务委员会表示，美国选举目前正面临外国势力的两大威胁——黑客攻击以及对美国选举基础设施的攻击，后者包含选民造册、投票机器、影响竞选活动等等。
 <br/>
 <br/>
 她说，北京绝对正在进行一项〝空前…或说是展开前所未见的努力〞，以影响美国人的看法，但目前还没有发现中共做出企图损害美国选举基础设施的举动。
 <br/>
 <br/>
 9日，美国资安公司CrowdStrike Inc刚刚公布最新报告指，中国已经取代俄罗斯，成为对西方国家进行网路攻击的最大国家。
 <br/>
 <br/>
 上月，美国总统川普（特朗普）在联合国大会公开谴责中共干预
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 。而中共官媒此前在美国农业州爱荷华的最大报纸上刊登广告，宣传川普打贸易战〝伤害美国豆农〞，也给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共干预美国选举.html" target="_blank">
  中共干预美国选举
 </a>
 提供了明证。
 <br/>
 <br/>
 美国情报总监科茨也在一次活动中说，中共正在网络空间实行一项〝规模前所未有的国家战略〞。他说：〝中国利用我们透明而开放的社会，也把目标对准美国州县政府和官员。它试图利用联邦政策和地方政策的任何差异。〞
 <br/>
 <br/>
 美国联邦调查局副局长伯蒂奇则直言，中共已成为美国反间谍工作面临的头号威胁，他说：〝中国（中共）在扩大行动，我们各地都有人被接触。〞
 <br/>
 <br/>
 9月底，美国网络安全公司〝火眼〞对《美国之音》透露，他们观察到的证据显示，中共可能正在效仿俄罗斯干预美国2016年总统选举的做法，通过发送钓鱼邮件来攻击美国的一些政治行动委员会和智库。
 <br/>
 <br/>
 10月4日，美国副总统彭斯在华盛顿发表演讲，对中共展开全方位谴责。其中提到，北京正在干涉美国中期选举，伤害美国民主制度，并指中共的目标是〝想给美国换一个总统〞。
 <br/>
 <br/>
 外界分析，中共或正在〝豪赌〞美国中期选举，期望选举结果能为中美贸易战带来转机。但多位评论人士认为，遏制中共已是美国朝野、包括两党的共识，中期选举对贸易战走向的影响不大。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
