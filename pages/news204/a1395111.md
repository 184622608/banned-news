### 北京金融难友连日集会要求见市长
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】首先来关注中国的消息。北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_P2P.html" target="_blank">
  P2P
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融难友.html" target="_blank">
  金融难友
 </a>
 和各地访民等群体，连日在北京市政府门前维权，10月10号，人数达到上千人。11号，现场金融难友高喊〝要求见市长〞等口号，遭到公安局长警告要抓人。有金融难友表示，市政府和信访局每天都挤满人，但都不解决问题。
 <br/>
 <br/>
 北京市政府门前连日来聚集了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融难友.html" target="_blank">
  金融难友
 </a>
 等维权群体，当局出动了大批便衣警察。
 <br/>
 <br/>
 北京金融难友表示，市政府和信访局每天都挤满人，但是都不解决问题。
 <br/>
 <br/>
 北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_P2P.html" target="_blank">
  P2P
 </a>
 金融难友江女士:〝每天都有人啊，很多人，信访局都去，没什么说法，在那边，都很那种官场上的官场话，要不然怎么这么久也没也一个具体的解决措施啊。〞
 <br/>
 <br/>
 11号，现场P2P金融难友高喊口号，公安局长出面警告：谁带头拘留谁，谁带头喊口号处理谁，只要有个别人不遵守就拘留。
 <br/>
 <br/>
 现场视频:〝要求见市长!要求见市长!给老百姓做主!为老百姓的血汗钱!为老百姓的血汗钱! 要求见市长! 要求见市长!还我们血汗钱! 还我们血汗钱! 喊口号的打。这里头不是光你们这个群体，很多群体在这里，你看他都在那儿不许堵门、堵路、不许喊口号、不许砸锅子、打横幅。我的便衣全在这儿。〞
 <br/>
 <br/>
 10号，来自全国各地的访民在北京市政府门前维权，要求依法解决问题，并高喊〝打倒腐败、依法治国〞的口号。
 <br/>
 <br/>
 现场视频:〝大家好，今天是2018年10月10日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全国维权日.html" target="_blank">
  全国维权日
 </a>
 ，我们是来自全国各地的访民，来到北京市市政府，这一路请求习主席三到位一处理，有理合理依法解决，无理随你们处置。〞
 <br/>
 <br/>
 10月1号至7号是大陆〝十一〞长假，各地金融难民发起维权行动，由于北京警方管控严密，难友无法聚集，改为10月10号在市政府维权，打出的诉求有，摘掉非吸参与者的帽子；加大办案透明度、不能只办案不追赃；加大追缴力度；严惩罪犯；理财公司案与P2P应同等对待。
 <br/>
 <br/>
 新唐人记者熊斌、施宜君采访报导
