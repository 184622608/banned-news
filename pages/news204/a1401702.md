### 【禁闻】不到一年 中共制造70多起集体绑架案
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】今年1月至11月，中国大陆很多省分的公安部门假借〝扫黑〞之名，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员制造五人以上的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集体绑架案.html" target="_blank">
  集体绑架案
 </a>
 至少70起。机密文件显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_610.html" target="_blank">
  610
 </a>
 〞办公室还在部署进一步的迫害。
 <br/>
 <br/>
 11月9号一天之内，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在黑龙江省哈尔滨市的各个区县和大庆市，非法绑架了119名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员，还骚扰了10几名学员。据〝法轮大法明慧网〞的消息，这次绑架行动是黑龙江省公、检、法、司部门合谋的，警察拿著名单挨家绑架。
 <br/>
 <br/>
 10月12号，长春市也有20多名法轮功学员，包括不修炼的家属突然遭到绑架。《法轮大法明慧网》的消息说，两天前，吉林省刚以省、市、区联署发了个所谓〝打黑除恶〞的公告。
 <br/>
 <br/>
 8月22到23号，辽宁省十个城市的警察非法抓捕了42名法轮功学员和4名家属。据了解，这次绑架是辽宁省公安厅督导的〝专案组〞联合行动，简称〝8.22案〞。
 <br/>
 <br/>
 类似的案件不仅局限在东北。中共在今年1月发出〝扫黑除恶专项斗争〞通知，但全国很多省分的公安部门假借〝扫黑〞之名，大规模绑架、关押、骚扰法轮功学员。根据《明慧网》的消息，据不完全统计，从2018年1月份至11月29号，中共对5名以上法轮功学员的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集体绑架案.html" target="_blank">
  集体绑架案
 </a>
 件高达72起，9人以上的集体绑架案有25起。
 <br/>
 <br/>
 另外，仅上半年中共就绑架法轮功学员2,392人次。至少244名65岁以上老年法轮功学员被绑架、骚扰，年龄最大的89岁。上半年430人被非法判刑，非法庭审380场，律师为法轮功学员出庭做无罪辩护186场。
 <br/>
 <br/>
 时事评论员任百鸣：〝法轮功学员实际上是社会上最善良的一群好人。那么（中共）它现在打黑除恶，当然应该是打击那些真正的坏人。可它却借用这个来打击最善良的人。而且绑架过程当中，事实上也是遭到了很多民众的谴责，尤其很多律师也越来越了解到真相。所以它这种绑架只会更加让人看清中共的非法性。〞
 <br/>
 <br/>
 中共从1999年开始镇压法轮功，当时的党魁江泽民曾指示要在〝三个月内消灭法轮功〞，对法轮功学员实行〝名誉上搞臭、肉体上消灭、经济上截断〞。
 <br/>
 <br/>
 时事评论员任百鸣表示，中共至今仍在大面积的绑架法轮功学员，说明了几个事实。
 <br/>
 <br/>
 任百鸣：〝第一个就是，尽管面对19年残酷迫害，仍然有数量非常巨大的法轮功学员在坚持着他们的信仰。第二个就是说明，（中共）它这种镇压实质上是失败的。因为虽然它已经极尽了所有这种迫害方式，但是不得民心。真正修炼法轮功的学员们，他们是知道法轮功有多好。〞
 <br/>
 <br/>
 最近，海外曝光一份中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_610.html" target="_blank">
  610
 </a>
 办公室〞下发的机密文件，目标任务是从今年10月到12月〝严厉打击法轮功〞，深入推进对法轮功学员的〝敲门行动〞，严打法轮功学员讲真相活动等。
 <br/>
 <br/>
 评论人士表示，中共迫害法轮功不但违反中国宪法和国际人权公约，还对中国司法体系造成了巨大的伤害。
 <br/>
 <br/>
 时事评论员凌晓辉：〝中共在各地打压法轮功，它构成了大量的警察刑事犯罪，很多把学员就这么活活打死、打残。以至于发生由中共国家体制所实行的，活体摘除法轮功学员的器官，这样一个人类从没有过的邪恶。这就是共产党打压法轮功的结果。〞
 <br/>
 <br/>
 11月27号，美国国际宗教自由委员会（USCIRF）的研究与政策副主任Tina Mufford表示，中共一直在部署对法轮功学员的恐吓和骚扰，还在继续。他说：〝要让中共知道美国正在看着，美国对此关注。〞——中共对法轮功的迫害必须停止。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/陈建铭
