### 同意书曝南科大资助基因剪辑 非人道条款令人发指
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】饱受谴责的中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 事件，再次曝光中共体制不计后果争夺〝科技第一〞的疯狂。相关项目团队官网的一份知情同意书，曝光其经费来自此前极力撇清关系的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南方科技大学.html" target="_blank">
  南方科技大学
 </a>
 ，该项目针对志愿者的条款，更被指拿人当
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小白鼠.html" target="_blank">
  小白鼠
 </a>
 。
 <br/>
 <br/>
 11月27日，NGO发展交流网的官方微博，公布了来自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南方科技大学.html" target="_blank">
  南方科技大学
 </a>
 副教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 研究室官网的一份基因项目知情同意书。根据内容判断，同意书所指项目即备受谴责的〝全球首例免疫艾滋病的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 〞项目。
 <br/>
 <br/>
 知情同意书显示，该项目的经费来源是南方科技大学，项目目标是〝产出〞（原文用词）具有免疫艾滋病毒能力的婴儿。
 <br/>
 <br/>
 26日上午，贺建奎对美联社公布，一对能〝天然免疫艾滋病〞的基因编辑双胞胎婴儿在其团队手中诞生，该消息很快引来全球媒体舆论围剿。下午，一百多名海内外华人科学家联名谴责其〝疯狂〞行为。
 <br/>
 <br/>
 在舆论压力下，贺建奎所在南方科技大学第一时间发出声明，声称贺建奎的基因编辑项目是〝在校外开展〞，〝没有告知学校〞，校方〝完全不知情〞。
 <br/>
 <br/>
 而这份知情同意书表明，贺建奎开展该项目，得到了校方的大力支持。
 <br/>
 <br/>
 另据党媒澎湃新闻26日晚的报导，该项研究已经在中国临床试验注册中心获得注册号为：ChiCTR1800019378，项目的经费或物资来源还包括深圳市科技创新委员会。而项目的试验主办单位（即项目批准或申办者）为南方科技大学和深圳和美妇儿科医院。
 <br/>
 <br/>
 此外，上述知情同意书中提到，项目团队不承担此项实验中婴儿基因编辑〝脱靶〞（指基因编辑失败）的风险，因为这是超出了现有医学科学和技术的风险后果。
 <br/>
 <br/>
 并且，如果在人工授精手术期间，母亲或婴儿感染上艾滋病毒，项目团队也不对此负责，相关风险由志愿者（指实验对象）自行承担。如果新生儿畸形，先天性缺陷，患有常见的遗传性疾病，项目组也概不负责。
 <br/>
 <br/>
 此外，如果志愿者在研究中受伤害，项目组将承担相应的医疗费用，但最高限额为5万元人民币。项目组为每对夫妇提供28万元人民币实验费用，但超过这个限额的任何费用由志愿者负责。
 <br/>
 <br/>
 同意书中还有条款规定，如果志愿者在中途退出研究，需要补偿项目组已经为其付出的实验费用，如果志愿者超期未支付费用，需要另交10万元人民币罚款。
 <br/>
 <br/>
 项目补充协议中称，针对出生的婴儿，如果因为基因编辑引起异常，产生的治疗费用由保险公司及团队承担，且无上限。但有网民指出，这是一个欺骗条款，因为对于这种保险费用，保险公司根本不会承认。
 <br/>
 <br/>
 网友用一句话总结这些条款称，母亲和婴儿感染艾滋病病毒不负责，婴儿畸形缺陷不负责，中途退出实验要退还费用并罚款10万，研究伤害封顶赔偿5万。
 <br/>
 <br/>
 事实上，在实际操作中，每例基因编辑都存在不可控的〝脱靶〞风险，成功率不足五成。但贺建奎团队无视该风险，仍然直接针对人类进行了基因编辑手术。
 <br/>
 <br/>
 据外媒披露，贺建奎宣称〝产出〞的一对基因编辑双胞胎姐妹中，一人基因编辑成功，另一人事实上已经〝脱靶〞。专家指，对后者来说，不但没能成功免疫艾滋病，还将承担因为基因编辑失败带来的巨大健康风险。
 <br/>
 <br/>
 贺建奎团队目前没有透露，此前如何招募志愿者，以及在〝成功制造〞这对双胞胎之前，该团队是否曾有、或者有多少〝失败〞的案例。
 <br/>
 <br/>
 专家指，基因编辑目前还处在试验阶段，医学界无法确认该项技术可能带来的危害，而且经过基因编辑的婴儿，将来还会把基因遗传给后代，这可能给人类带来无法逆转的灾难。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
