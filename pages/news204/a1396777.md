### 上海前检察长陈旭被判无期徒刑
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】中共官媒周四通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海前检察长.html" target="_blank">
  上海前检察长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈旭.html" target="_blank">
  陈旭
 </a>
 被判
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无期徒刑.html" target="_blank">
  无期徒刑
 </a>
 。陈旭被指是〝上海帮〞中的重要成员之一。
 <br/>
 <br/>
 判决称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈旭.html" target="_blank">
  陈旭
 </a>
 在2000年到2015年间，直接或通过其亲属，非法收受他人财物，共计折合人民币7423多万元。
 <br/>
 <br/>
 陈旭2003年到2014年，曾担任上海市委政法委员会副书记、上海市委副秘书长及上海检察院检察长等职务。2017年3月1日落马。
 <br/>
 <br/>
 据报，陈旭是〝上海帮〞重要成员之一，被指涉多年前〝四证人离奇死亡〞案，以及上海社保案、陈良宇案、周正毅案等多个大案。因涉陈旭案被调查的上海政法系统已超过百人。
 <br/>
 <br/>
 另外，陈旭任上海市高级法院副院长、上海政法委副书记期间，追随江泽民迫害法轮功，被〝追查迫害法轮功国际组织〞列入追查名单。
</div>
