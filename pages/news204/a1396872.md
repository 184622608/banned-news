### 崔永元指转基因推广人张启发：罪该万死
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日电】近日，大陆名嘴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 食物问题再次发声，并且点名陆中科院院士
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张启发.html" target="_blank">
  张启发
 </a>
 ，指其推广
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因大米.html" target="_blank">
  转基因大米
 </a>
 贻害无穷，罪该万死。
 <br/>
 <br/>
 10月25日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 转发崔永元后援会的微博，并配文：1996年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张启发.html" target="_blank">
  张启发
 </a>
 就在湖北偷偷种下了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 水稻，污染了久负盛誉的米粮仓，开始伤害湖北百姓的身体。真是罪该万死！
 <br/>
 <br/>
 崔永元后援会的微博的原文为：〝为什么湖北人支持崔老师的最多？因为那里民众深受转基因作物的毒害，是无德商人与转基因研究专家试验场，是转基因农作物的重灾区〞。
 <br/>
 <br/>
 据公开资料，张启发，1953年12月出生于湖北，大陆科学院院士、华中农业大学生命科学技术学院教授、博士生导师。张启发致力于水稻基因组研究，有〝中国转基因水稻之父〞之称。
 <br/>
 <br/>
 2014年8月，博客文章《重大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_食品安全.html" target="_blank">
  食品安全
 </a>
 事件：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因大米.html" target="_blank">
  转基因大米
 </a>
 毒死10多名学生，应依法严惩张启发们和华中农大》引起关注。文章说：〝华中农大学生白血病超高发这个极不正常现象最初应该是校内议论，然后有学生冒险于2013年左右在校内论坛和贴吧上发出疑问和讨论帖子，接着，引起了崔永元等反转人士的高度关注，并把初步的网路调查结果公之于众，结论是华中农大和张启发非法让学生食用转(毒)基因大米导致该校学生的白血病比正常情况高出3倍！〞
 <br/>
 <br/>
 有华中农业大学的学生在讨论相关话题时发帖说：我作为本专业的人，多多少少也了解一些。华农的白血病发病率2万分之3…..真是异常的高，据我在校三年的情况，学校至少有8例因血癌而休学或退学。我们学校人吃的,大家觉得是不是转基因呢？当我们作为新生入学的时候，学校就义正言辞的公然要求，我们每一位学生都有责任和义务推广转基因。
 <br/>
 <br/>
 此前，因转基因食品问题，崔永元也曾发长文批评《科技日报》对转基因食品的报导不实。在文中，崔永元点名该报总编辑马爱平：〝马爱平不能再这么做下去了，会遭报应的。专家在转基因食品安全问题上，忽悠百姓、蒙骗百姓，死了会下地狱的。〞
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：戴明）
