### 〝朱日和〞首任司令谢勇病逝  中共神秘军事基地曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱日和.html" target="_blank">
  朱日和
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_训练基地.html" target="_blank">
  训练基地
 </a>
 原
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司令员.html" target="_blank">
  司令员
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢勇.html" target="_blank">
  谢勇
 </a>
 日前病亡。谢勇为〝朱日和〞基地首任司令员。该基地曾是中共最神秘的战术训练基地，在过去很长时间里，这里对外隐匿，甚至连名字都没有。
 <br/>
 <br/>
 11月28日，陆媒报导称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢勇.html" target="_blank">
  谢勇
 </a>
 因病医治无效，于25日在北京病逝，享年67岁，送别仪式于27日在北京八宝山菊厅举行。
 <br/>
 <br/>
 公开资料显示，谢勇1951年8月出生，入伍后长期在原北京军区工作。上世纪90年代末，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱日和.html" target="_blank">
  朱日和
 </a>
 合同战术
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_训练基地.html" target="_blank">
  训练基地
 </a>
 组建后，谢勇受命出任基地首任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司令员.html" target="_blank">
  司令员
 </a>
 ，2007年卸任。
 <br/>
 <br/>
 谢勇病亡也令中共〝朱日和〞神秘军事基地再度曝光。
 <br/>
 <br/>
 <h4>
  朱日和基地曾是神秘地方
 </h4>
 <br/>
 〝朱日和〞基地位于内蒙古自治区乌兰察布市四子王旗和锡林郭勒盟苏尼特右旗境内，〝朱日和〞是蒙古语的译音，意为〝心脏〞，是亚洲最大，也是中共最大的陆空军军事训练基地。
 <br/>
 <br/>
 〝朱日和〞原是古战场占地1066平方公里，地形由丘陵、沙漠、草原、山地、沟壑等组成，具有排兵布阵等演练优势。
 <br/>
 <br/>
 在过去很长时间里，朱日和基地曾是一个神秘的地方，这里对外隐匿，甚至连名字都没有。仅以〝华北某训练场〞代替，外军也只能在侦察卫星图片上探究。
 <br/>

 </center>
 <br/>
 该基地的前身是1957年组建的加强
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坦克师.html" target="_blank">
  坦克师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_战术演习场.html" target="_blank">
  战术演习场
 </a>
 ，中共于1997年决定将这里扩编为中共军队规模最大、科技含量最高的合同战术训练基地，并在2003年对外开放。
