### 张首晟讣告发出后张家人称不知其患有抑郁症
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】近年来声名鹊起的美国华裔物理学家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张首晟.html" target="_blank">
  张首晟
 </a>
 日前突然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_跳楼自杀.html" target="_blank">
  跳楼自杀
 </a>
 ，令外界倍感意外。多家中共媒体周四发表了张首晟过世的讣告，称张首晟与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抑郁症.html" target="_blank">
  抑郁症
 </a>
 〝顽强对抗〞后意外离世。随后，张家发言人对外界解释说，张并不是在斯坦福大学离世的，家人之前也不知他患有抑郁症。
 <br/>
 <br/>
 12月6日，斯坦福大学教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张首晟.html" target="_blank">
  张首晟
 </a>
 的家人通过媒体发布了张首晟去世的讣告。这份讣文中称，现年55岁的张首晟〝在与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抑郁症.html" target="_blank">
  抑郁症
 </a>
 斗争后〞于12月1日〝意外离世〞。因在拓扑绝缘体方面的贡献，张首晟一度被誉为〝最接近诺贝尔物理学奖的华人科学家〞。
 <br/>
 <br/>
 张首晟去世的讣告发出后的第二天（7日），张首晟家庭发言人Sean McCormack在接受陆媒采访时针对外界有关张的一些消息作出了澄清。
 <br/>
 <br/>
 张首晟家庭发言人首先声明，张首晟教授过世的地点并不是外界传言所说的斯坦福大学，而是死于旧金山，那里距离位于帕罗奥多市的斯坦福大学还有大约1个小时的车程。
 <br/>
 <br/>
 其次，这位发言人又解释说，因为讣告中提到张首晟〝与抑郁症斗争后〞意外离世，不少媒体便理解为张首晟教授死于抑郁症，但实际上〝张首晟教授的家人从来不知道张教授有过这样（抑郁症）的诊断。然而，回想起来，他们开始相信他有抑郁症的症状。〞
 <br/>
 （His family was not aware of any such diagnosis． However， in retrospect， they have come to believe he had symptoms of depression．）
 <br/>
 <br/>
 张首晟家庭发言人接着表示，事情发生后，张的家人才意识到张生前的挣扎比家中每个人经历的日常生活压力都更严重。现在希望媒体能尊重张教授家人的隐私和他们的悲伤。
 <br/>
 <br/>
 张首晟近年来声名鹊起，可谓名利双收。因此他
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_跳楼自杀.html" target="_blank">
  跳楼自杀
 </a>
 的死讯突然传出后，外界十分震惊，舆论界对他的真实死因颇有怀疑。但当地警方很快就认定，张首晟确实是死于自杀，案件已经结案。
 <br/>
 <br/>
 资料显示：张首晟祖籍在中国江苏高邮，他本人1963年生于上海。他于1983年取得德国柏林自由大学物理学学士学位，后赴纽约州立大学修读物理学博士，1987年至1989年在加州大学圣塔芭芭拉分校担任物理研究所博士后研究员，1989-1993年任IBM阿尔玛登研究中心高级研究员。1993年担任斯坦福大学物理学教授，2013年成为中国科学院外籍院士。
 <br/>
 <br/>
 2007年，张首晟发现的〝量子自旋霍尔效应〞被《科学》杂志评为当年的〝全球十大重要科学突破〞之一。他因此获得物理界多项重量级奖项。
 <br/>
 <br/>
 由于张首晟跳楼自杀的时间和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 副董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在加拿大被捕的时间都是12月1日，而这两个消息又都是在12月6日同时传出，因此外界怀疑这两件事互有关联。
 <br/>
 <br/>
 日前有美国媒体指出，张首晟从2009年接受中共 〝千人计划〞的招揽后曾受聘于清华大学，此后他与北京的联系日益密切，更于2013年成为了中共官办的最高科研学术机构〝中科院〞的外籍院士。
 <br/>
 <br/>
 而恰恰就是在2013年，张首晟创办了风险投资公司丹华资本，专门投资人工智能、虚拟与增强现实、大数据、区块链等新兴技术领域。在此过程中，张首晟与华为公司有所接触，网络上曾有消息称，张首晟可能有意与华为合作开发新型芯片。
 <br/>
 <br/>
 另据美国贸易代表办公室（USTR）今年11月更新的《301调查报告》，其中在提及中共利用风险投资获取美国的尖端技术和相关知识产权的部分，曾点名丹华资本。
 <br/>
 <br/>
 虽然迄今为止，并没有确凿的消息显示张首晟自杀是否与丹华资本被调查有关，但他自杀的时间点与华为总裁的女儿孟晚舟被逮捕的时间高度重合，不能不令外界生出诸多揣测与联想。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：戴明）
 <br/>
