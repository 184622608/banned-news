### 大陆顶级探月专家猝死 做报告时突发心脏病
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】大陆粒子天体物理和空间探测领域专家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王焕玉.html" target="_blank">
  王焕玉
 </a>
 ，日前在做学术报告过程中，突发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心脏病.html" target="_blank">
  心脏病
 </a>
 去世。王焕玉是大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_探月工程.html" target="_blank">
  探月工程
 </a>
 的顶级专家，曾担任中共科学院高能物理研究所党委书记兼副所长。
 <br/>
 <br/>
 中共科学院高能物理研究所官网11月5日发布讣告称，安徽合肥市4日召开〝第二届射线成像新技术及应用研讨会〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王焕玉.html" target="_blank">
  王焕玉
 </a>
 在会上做学术报告过程中，突发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心脏病.html" target="_blank">
  心脏病
 </a>
 ，经抢救无效，与当日下午17时12分去世，享年64岁。
 <br/>
 <br/>
 公开资料显示，王焕玉1954年12月生于河北省文安县，毕业于中国科技大学近代物理系。他是大陆粒子天体物理和空间探测领域的知名专家。
 <br/>
 <br/>
 王焕玉曾担任中共空间学会常务理事、空间探测专业委员会副主任，中共科学院高能物理研究所（以下简称〝中科院高能所〞）原党委书记兼副所长、研究员，硬X射线调制望远镜卫星地面应用系统总指挥、卫星系统副总指挥。
 <br/>
 <br/>
 据中科院高能所5日发布的王焕玉生平介绍披露，大学毕业后，王焕玉即进入中科院高能所工作。1998年3月至2001年4月，王焕玉担任粒子天体物理中心（原宇宙线与天体物理研究室）副主任，2001年4月至2003年3月担任高能所党委副书记（主持工作）、副所长、纪委书记，2003年3月至2014年10月担任高能所党委书记、副所长。
 <br/>
 <br/>
 此外，王焕玉还担任中共科学院大学岗位教授，兼任中共科技大学教授、博士生导师，中共核仪器协会常务理事，中共空间科学学会常务理事、空间探测专业委员会副主任，中共电子学会核电子学与探测技术分会副秘书长等职务。
 <br/>
 <br/>
 王焕玉长期从事高能物理、粒子天体物理和宇宙射线探测研究。
 <br/>
 <br/>
 1992年至2001年，王焕玉参加了中共载人航天神舟-2飞船空间天文分系统项目〝太阳和宇宙天体高能辐射监测仪〞，任分系统副主任设计师。
 <br/>
 <br/>
 自2003年起，他一直领导中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_探月工程.html" target="_blank">
  探月工程
 </a>
 。据中科院高能所官网介绍，王焕玉领导开展的月球X射线荧光探测是一项开创性科学工作，在大陆尚属首次。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
