### 孟宏伟继任者浮出  履历有一大特别之处
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共公安部.html" target="_blank">
  中共公安部
 </a>
 前副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 后，公安部官网显示，其排名位置已由中纪委驻公安部纪检组组长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓卫平.html" target="_blank">
  邓卫平
 </a>
 填补。根据公开资料，邓卫平曾是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在福建任职时的老部下。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共公安部.html" target="_blank">
  中共公安部
 </a>
 长赵克志在8日凌晨主持的党委会议上通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 收受贿赂、涉嫌违法，正接受调查。而与此同时，中共公安部官网上，有关孟宏伟的档案资料已消失。
 <br/>
 <br/>
 孟宏伟曾是中共公安部第三把手，地位仅次于常务副部长王小洪。但目前，该位置已由兼任公安部党委委员的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓卫平.html" target="_blank">
  邓卫平
 </a>
 填补。这意味邓有可能接替孟的副部长职务。
 <br/>
 <br/>
 根据公开履历显示，邓卫平1990年11月起，任福州市鼓楼区委副书记；1993年8月，任福州市郊区区委副书记、政法委书记；1995年11月，任福州市晋安区委书记。
 <br/>
 <br/>
 值得注意的是，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在1990年4月至1996年4月担任福州市委书记。由此可知，邓卫平是习近平在福建任职时的老部下。
 <br/>
 <br/>
 直到2015年3月，邓卫平赴京任公安部纪委书记；4月兼任公安部督察长；2016年1月改任中纪委驻公安部纪检组组长。
 <br/>
 <br/>
 至目前，中共公安部部长赵克志，分管常务的副部长王小洪，中纪委驻公安部纪检组组长兼公安部党委委员邓卫平，以及副部长孟庆丰（经济犯罪，海外追赃）、许甘露（出入境管理，国家移民管理局局长），均是习近平的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旧部.html" target="_blank">
  旧部
 </a>
 和亲信。
 <br/>
 <br/>
 <h4>
  公安部通报孟宏伟罪名蹊跷  隐藏玄机
 </h4>
 <br/>
 中纪委国家监委网站10月7日消息，公安部副部长孟宏伟涉嫌违法，目前正接受国家监委监察调查。通报中没有交代孟宏伟涉及的具体罪名。
 <br/>
 <br/>
 微信公众号〝政知圈〞8日的报导指，孟宏伟的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 通报除了比其他〝大老虎〞的通报内容少之外，还有一些不同之处：
 <br/>
 <br/>
 其一，孟宏伟是首个只接受〝监委监察调查〞的〝老虎〞。自今年3月中共国家监察委员会成立以来，部级〝老虎〞落马通报的表述都是〝接受中央纪委国家监委纪律审查和监察调查〞。孟宏伟是第一个通报中只写了〝接受国家监委监察调查〞。
 <br/>
 <br/>
 其二，在通报中提及孟宏伟涉嫌〝违法〞，而不是〝违纪违法〞。
 <br/>
 <br/>
 报导还说，从目前通报表述上的不同，可以感受到孟宏伟的落马原因与以往落马〝老虎〞相比有其特殊之处，但具体有哪些不同，还需要有官方更多的披露。
 <br/>
 <br/>
 另外，8日凌晨，在公安部长赵克志主持召开的公安部党委会议上，除通报孟宏伟因收受贿赂、涉嫌违法接受调查外，会议还强调，要〝坚定不移反对腐败，坚决彻底肃清周永康流毒影响〞、〝依法查处涉嫌与孟宏伟共同收受贿赂人员〞、〝绝不允许跟组织讲条件、讲‘价钱’〞、〝严密防范敌对势力的各种捣乱破坏颠覆活动〞等。
 <br/>
 <br/>
 外界认为，孟宏伟落马案不寻常，背后可能涉及相当程度的政治势力角斗。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
