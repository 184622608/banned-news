### 改革开放纪念缩水 美媒：邓小平成掣肘习近平工具
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】北京今年纪念〝改革开放〞40周年规模明显缩水。美媒指，原因是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 理论已经成为中共党内牵制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的武器。
 <br/>
 <br/>
 2008年，北京曾经大张旗鼓地庆祝〝改革开放〞30周年。但今年官方的纪念计划大幅缩减，不但北京没有召开相关会议，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 〝南巡〞也无〝重要讲话〞，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 更是只字不提。官方宣传中，〝去邓抬习〞的意味明显。
 <br/>
 <br/>
 《纽约时报》引述分析指，在邓小平启动的〝改革开放〞40周年之际，北京却在忙着提升习近平的地位，大有取代邓小平之势。
 <br/>
 <br/>
 报导引述一名中共官媒资深记者的说法指，北京限制今年庆祝活动的原因是，〝一旦你开了这个口子，就很容易失去控制，引发新的争论〞。
 <br/>
 <br/>
 中美爆发贸易战以来，习近平因为改变邓小平〝韬光养晦〞的策略，已成为中共党内批评的目标。
 <br/>
 <br/>
 报导指，在此之前，习近平似乎也没有试图将自己定位为邓小平的继承人，他虽然没有完全抹杀邓小平，但已经在有意降低邓小平的地位。
 <br/>
 <br/>
 美国哈佛大学研究学者葛维茨（Julian Gewirtz）表示，江泽民、胡锦涛政治地位的党内〝合法性〞都来自邓小平，而习近平上位并非来自邓小平的拔擢。
 <br/>
 <br/>
 澳洲中国研究学者白杰明（Geremie Barme）说，习近平将邓小平及后继党魁定义为一个〝经济转型的时代〞，而他自己将领导一个〝强大的新时代〞。
 <br/>
 <br/>
 报导提到，对习近平来说，邓小平的遗产是潜在的限制，也是党内其他人用来掣肘习近平的〝传统源泉〞。
 <br/>
 <br/>
 事实上，习近平不愿肯定邓小平，还因为邓确立的路线和政经体制，也成为习近平反腐和经济改革的阻力。
 <br/>
 <br/>
 当年邓小平启动〝改革开放〞的同时，也确立了〝腐败治国〞的方针。邓留下的政治遗产，自然成为党内用〝反腐亡党〞来遏制习近平打虎的有力工具。
 <br/>
 <br/>
 此外，习近平为了挽救经济危局，强推经济改革，其主要阻碍之一也是〝腐败治国〞造成的政商关系，以及中共权贵瓜分掌控经济命脉的局面。
 <br/>
 <br/>
 除了施政方向的差异，习邓两家还牵扯上一辈的私人恩怨。邓小平当年曾经参与整肃习仲勋，两家族向来不睦。随着邓家女婿吴小辉入狱和邓朴方好友樊立勤贴大字报批习，两家矛盾已经公开化。
 <br/>
 <br/>
 日前，邓朴方公开演讲力挺邓小平理论，喊话〝邓小平路线百年不变〞。随后，刊登邓讲话全文的残联官网陷入〝休克〞状态。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
