### 齐齐哈尔官场大震荡 政法委副书记被查
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】齐齐哈尔市持续震荡，黑龙江省纪监委11月29日通报，齐齐哈尔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政法委副书记.html" target="_blank">
  政法委副书记
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维稳办主任.html" target="_blank">
  维稳办主任
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_夏继明.html" target="_blank">
  夏继明
 </a>
 被查。
 <br/>
 <br/>
 法制晚报微信公号〝观海解局〞11月29日报导，齐齐哈尔市委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政法委副书记.html" target="_blank">
  政法委副书记
 </a>
 、市委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维稳办主任.html" target="_blank">
  维稳办主任
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_夏继明.html" target="_blank">
  夏继明
 </a>
 ，涉嫌严重违纪违法，目前正接受纪律审查和监察调查。
 <br/>
 <br/>
 公开资料显示，现年57岁的夏继明，从事新闻工作多年，曾是《光明日报》驻齐齐哈尔记者站站长，兼任市新闻中心主任；2008年到2012年，夏继明任齐齐哈尔市委宣传部副部长，之后成为市委政法委副书记、市委维稳办主任。
 <br/>
 <br/>
 在夏继明受查前10天，齐齐哈尔市宣传部前部长高虹，于11月19日因涉嫌严重违纪违法受查；另外，该市前市委书记韩冬炎、副市长黄宇，以及市组织部长胡福绵，相继在今年及去年底被〝双开〞及批捕。
 <br/>
 <br/>
 另外，日前齐齐哈尔铁锋区曾召开的整改会，通报该市多名因受贿、贪污及非法持枪等罪被判囚的官员的典型案例。
 <br/>
 <br/>
 而该省最近还发生了严重迫害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 事件。据明慧网报导称，11月9日同一天，黑龙江哈尔滨和大庆119名法轮功学员，于被中共警方非法绑架。报导说，这次的绑架是黑龙江省公检法司部门蓄谋已久的阴谋，警察是拿著名单到法轮功学员家绑架的。
 <br/>
 <br/>
 中共江泽民集团发起迫害法轮功近20年来，作为操控协调这场迫害的主要部门政法委、610是这场迫害的罪魁祸首。中共公安部、最高检、最高法、司法部、国安部等是主要帮凶。
 <br/>
 <br/>
 10月底中共曾公开通报了130多名中共公安正、副局长被撤职、查办。这些中共公安正、副局长们，都是参与迫害法轮功学员的指挥者，他们参与迫害的事实已在明慧网上曝光。
 <br/>
 <br/>
 明慧网报导说，他们的〝落马〞并不是中共的大义〝灭亲〞，而是中共的〝卸磨杀驴〞，也正应了中国的古训〝善恶有报是天理〞。
 <br/>
 <br/>
 而此次落马的齐齐哈尔市委政法委副书记，也被视为现世报。据明慧网11月报导，中共政法委在迫害法轮功中罪恶累累，遭受的恶报也非常多。中共网站公开通报的就有60多个中共政法委正、副书记相继落马，其中包括政治局常委级的、省部级的、厅局级的官员相继落马。
 <br/>
 <br/>
 在明慧网梳理的逾万个因迫害法轮功而遭恶报的案例中，政法委、610均为死亡职位。因此明慧网奉劝那些仍在此职位的中共官员，认清形势，停止迫害，悔过自新，为自己及家人留一条后路。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
