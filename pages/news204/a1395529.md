### 瑞银：若川普再征税 人民币汇率明年将达7.5
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 正酣。国际知名投行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瑞银.html" target="_blank">
  瑞银
 </a>
 （UBS）发布报告称，随着贸易战影响逐步显现，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 贬值压力继续加剧。如果川普（特朗普）对中国所有出口商品加征25%的关税，2019年人民币兑美元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率.html" target="_blank">
  汇率
 </a>
 恐将逼近7.5。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瑞银.html" target="_blank">
  瑞银
 </a>
 在最新报告〝中国经常帐户顺差或消失、影响几何？〞指出，中国大陆经常帐顺差的时代接近尾声，虽然今年全年经常帐仍保持顺差，但2019年可能转为小额逆差。若是如此，这将是年度经常帐户在24年来首次出现逆差。
 <br/>
 <br/>
 另外，中共为对冲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 影响而出台的宽松政策力度越强，经常帐户逆差可能也会更大。经常帐户顺差转逆差不仅会降低其对资本外流的缓冲作用，还有可能打击市场信心、加速资本外流，从而给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币.html" target="_blank">
  人民币
 </a>
 带来更大贬值压力。
 <br/>
 <br/>
 报导引述瑞银大中华区首席经济学家汪涛称，2018年底人民币对美元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率.html" target="_blank">
  汇率
 </a>
 为7.0左右，2019年底将为7.3。如果贸易战进一步升级、美国对中国所有出口商品加征25%的关税，则2019年人民币对美元汇率可能逼近7.5。
 <br/>
 <br/>
 另一国际投行美银美林在本月初也调整了对人民币的预估。之前美银美林预估明年一季度1美元兑换6.9人民币，现在调至7.05人民币，明年第二季度也从1美元兑换6.85人民币调至现在预估的7.1人民币。
 <br/>
 <br/>
 德意志银行则认为，人民币汇率会在明年一举跌至7.4。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：范铭）
