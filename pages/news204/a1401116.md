### 张家口爆炸惨状：大楼烧焦货车成空壳 村民紧急转移
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】11月28日凌晨，河北省张家口市一化工厂附近发生爆炸，截至早8时，已造成22人死亡22人受伤。现场画面惨烈，整栋大楼被烧焦，一排货车烧成了空壳。距爆炸事故地点约1公里的村民说，被通知尽快转移，以防有害气体危害健康。
 <br/>
 <br/>
 河北省张家口官方消息称：11月28日0时41分，在张家口市桥东区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_河北盛华化工有限公司.html" target="_blank">
  河北盛华化工有限公司
 </a>
 附近发生的爆炸起火事故，经公安消防现场搜救确认，目前有22人死亡，22名伤者已分别送往河北北方学院第一医院和251等医院救治。截至当日下午，死亡人数升至23人。
 <br/>
 <br/>
 网上影片显示，爆炸事故发生时现场火光熊熊，浓烟冲天，医院内有多名伤者等候救治，部份人手、脚、脸部被熏黑。另有现场画面显示，整栋大楼被烧焦，一排货车烧成了空壳。
 <br/>
 <br/>
 距事发地2.6公里处的一商铺老板说：〝事发时爆炸声巨大，当时睡觉被惊醒，受此影响店铺目前已关门。
 <br/>
 <br/>
 张家口市安全生产监督管理局办公室9时许称，该事故系一辆运输危险化学品的车辆等待进工厂过程中发生爆炸，并引燃了周围车辆。事故中过火大货车38辆、小型车12辆。
 <br/>

 <br/>
 11时许，张家口市宣化县大仓盖镇北甘庄村一名村民说，该村距此次爆炸事故地点约1公里。事故发生后，村大队通过广播通知村民尽快转移，以防有害气体危害健康。目前已有部分村民离开村子。
 <br/>
 <br/>
 该村民还称，此前每天都看到有许多输送危险化学品的车辆经过，几年前，就有村民抗议化工原料污染村里水源，但有关部门检测水源称合格。
 <br/>
 <br/>
 河北盛华化工有限公司成立于1970年，于2000年5月经股份制改造而成。位于张家口市盛华东大街10号，是中国化工500强企业之一，河北省百强重点民营企业，同时也是张家口市民营经济支柱企业。
 <br/>
