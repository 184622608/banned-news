### 看守所伪造血压数据 黄琦控法西斯式迫害
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四天网.html" target="_blank">
  六四天网
 </a>
 〞创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 ，日前向探望他的代理律师透露，看守所
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伪造血压数据.html" target="_blank">
  伪造血压数据
 </a>
 ，以掩盖他的病危状况。办案单位还以〝不讲政治〞为由否决救治方案，黄琦控诉遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法西斯式迫害.html" target="_blank">
  法西斯式迫害
 </a>
 ，并表示会抗争到底。
 <br/>
 <br/>
 10月23号下午，广州维权律师刘正清再次到四川
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绵阳看守所.html" target="_blank">
  绵阳看守所
 </a>
 会见
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦.html" target="_blank">
  黄琦
 </a>
 。黄琦透露，18和19号两天，同监室人员使用电子仪器测量他的血压，高低压数值分别超过220和140，远超过早前看守所医生给出的血压数值。
 <br/>
 <br/>
 刘正清表示，此后黄琦要求看守所使用电子仪器测量，但遭到拒绝。22和23号上午，看守所通过手动测血压方式，伪造虚假血压数据，以掩盖黄琦病危状况。
 <br/>
 <br/>
 黄琦还披露，8月份看守所组织会诊后，曾考虑提供救治方案，但被办案单位以〝不讲政治〞为由否决。黄琦表示，自己病情恶化是当局采用法西斯暴行的结果，他会抗争到底。
 <br/>
 <br/>
 黄琦的母亲蒲文清表示，当局不给黄琦治疗，是想把他折磨致死。
 <br/>
 <br/>
 黄琦母亲蒲文清：〝黄琦血压很高，如不能及时治疗，会猝死、中风或昏迷而死亡。绵阳市办案单位其目的就是不给黄琦治疗高血压，加重黄琦病情，致黄琦而死。当局藉此机会逃脱他们的罪行，是地地道道的反人类的罪行。〞
 <br/>
 <br/>
 现年85岁的蒲文清最近在网上透过视频哭诉，黄琦在狱中备受虐待，每天多次被毒打，现在全身浮肿、肾衰竭、尿毒症。蒲文清求大家救救黄琦，让她死前能见见儿子。
 <br/>
 <br/>
 成都维权人士干兴艳：〝现在政府官员很坏的，对访民很恶劣，对黄琦就更恶劣了，这些都是他们惯用手段，打击报复，因为黄琦是在帮助这些弱势群体，他就恨黄琦嘛。〞
 <br/>
 <br/>
 黄琦现年55岁，1999年创立
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六四天网.html" target="_blank">
  六四天网
 </a>
 ，因持续披露公共事件、人权讯息，两次被捕判刑。2016年4月，黄琦在六四天网发表前警察访民陈天茂提供的一份信访报告，并接受〝自由亚洲〞采访，当年11月28号再遭抓捕，后被控所谓的〝非法为境外提供国家秘密〞，日前又被追加〝洩露国家秘密〞罪名。黄琦始终拒绝认罪。
 <br/>
 <br/>
 新唐人记者常春、陈洁、元真采访报导
</div>
