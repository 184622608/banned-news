### 福建7吨毒碳九泄漏 网友：不要让40万人无声中消逝
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】〝油已经飘过来了，鱼开始死了，鱼死了很多，早上刚修好的鱼排，又沉下去了〞。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碳九泄漏.html" target="_blank">
  碳九泄漏
 </a>
 物已波及到距离泉港50公里外的惠安，有市民表示，吃的海鲜都有一股油漆味。
 <br/>
 <br/>
 6天前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建泉港.html" target="_blank">
  福建泉港
 </a>
 发生7吨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碳九泄漏.html" target="_blank">
  碳九泄漏
 </a>
 事故，造成严重生态灾难。署名〝北风刘先生〞的网友日前发文称，〝7吨碳九泄漏，不要让40万泉港人在无声中消逝……〞
 <br/>
 <br/>
 文中称，碳九的成份包括异丙苯、正丙苯、乙基甲苯、均三甲苯、茚等，7吨碳九泄漏相当于7吨油漆，直接倒入这个小城市，而且是还没分解的油漆，比毒药还毒。
 <br/>
 <br/>
 碳九及其易燃，其蒸气与空气混合，能形成爆炸性混合物。它的侵入方式是通过皮肤接触，吸入，食入。高浓度的碳九对中枢神经系统有麻醉作用，会引起急性中毒。若从口中食入中毒，会使胃、肺、肝、肾、脾等脏器受损。
 <br/>
 <br/>
 此外，碳九对水体、土壤和大气造成污染的更为严重。
 <br/>
 <br/>
 文章描述，泉港曾经是〝长寿之乡〞，能源之都，常住人口39.27万，现在却成为〝癌症之乡〞，从十几年前炼油厂化工厂（东港石油化工石业有限公司）搬进去泉港开始，环境就一天天变差。
 <br/>
 <br/>
 海域、土壤、大气污染，甚至连正常呼吸都觉得困难，而这些在泉港习以为常，在福建省肿瘤医院里，60%以上的癌症患者都是来自知泉港。
 <br/>
 <br/>
 如今这个城市，因为7吨碳九意外泄漏，成为一个〝死城〞。
 <br/>
 <br/>
 在事件发生当天，政府给的解释，一直是空气质量合格，事故已经解决，他们唯一做的就是用吸油毯把泄漏物吸起来，然后，等风来。
 <br/>
 <br/>
 据了解，此次碳九泄漏事件导致的污染范围广，周边受到影响的还有肖厝、峰尾、郭厝、南浦等村镇。村民还反映，在泉州、石狮、惠安、莆田等地也能闻到空气中的刺鼻气味。

 <h4>
  中共掩盖造假
 </h4>
 <br/>
 碳九泄漏故发生在11月4日凌晨，官方通报称，东港石油化工石业有限公司在执行工业用裂解碳九装船过程中，发生泄漏，共达6.97吨。
 <br/>
 <br/>
 根据网民在微信号发布的文章，4日凌晨3时起，泉州市泉港区几个镇的空气就出现刺鼻异味，民众在睡梦中被恶臭熏醒，有不少人报警。
 <br/>
 <br/>
 但当地政府部门互相推诿，警方推给海事局，海事局又推给环保局，环保局再踢给农林水局，直到当日早上9点，才有政府人员来查看，证实是碳九泄漏。
 <br/>
 <br/>
 但官方未及时通报碳九对人体的危害。当地民众对此完全没有防范，不少人因徒手打捞污染物中毒入院，甚至有人出现生命危险。
 <br/>
 <br/>
 5日，泉港区环保局发布通告称，至4日晚18时，碳九泄漏海域清理工作已基本完成，一切正常。
 <br/>
 <br/>
 有网民嘲讽，近7吨化工原料泄漏到海面上，只用了一天就恢复正常，甚至说〝油污已全部清除完〞，这种速度世界无敌。
 <br/>
 <br/>
 当地民众王先生也向大纪元透露，4日晚，泉港区环保局用喷水车向环保大楼喷水，村民怀疑环保局为了降低空气污染浓度，让仪器数值能够〝达标〞，而进行的〝喷水作业〞。
 <br/>
 <br/>
 当地网友YQ-萌之初发文称，5日还有恶臭味，味道和4日的时候一样，只是没了空气中没有了那种恐怖的灰蓝色烟。而海里的鱼，正在大量的死去。
 <br/>
 <br/>
 〝11月6日，白天依旧会有一阵阵恶臭，海水依旧没有清理干净，而那些非专业人士的普通人代替专业人士在没有任何防护的情况下，手受伤了。他们什么也不懂，只是拿着一点点可笑得钱，自己去处理了。〞
 <br/>
 <br/>
 但6日，泉港区广播电视台却发布消息称，〝空气指标正常，海水符合养殖水质要求〞。据悉，泉州电视台记者在现场进行报导时，工作人员一边戴着两层口罩，一边报导说空气质量正常，遭到现场村民斥责。
 <br/>
 <br/>
 自6日中午以后，碳九泄漏相关消息在传媒和社交媒体上扩散，舆情开始升温。
 <br/>
 <br/>
 8日凌晨，泉港区环保局再次发布消息，称泉港城区空气自动监测站各项大气指标已恢复正常。而且连日来，环保局没有发出任何有毒物质警报。
 <br/>
 <br/>
 不过，同日，大陆财新网却报导称，呛鼻气味仍在泉港区范围内弥漫，肖厝村多位村民出现不适，入住泉港医院急救。肖厝村支撑水产养殖的塑胶鱼排已被碳九腐蚀，海面油污也仍未打捞干净。
 <br/>
 <br/>
 碳九泄漏这一重大安全事故爆发后，当局一直刻意掩盖。直到消息压不住，才被广泛曝光。然而中共官方仍在淡化事故影响，称事故导致入院的52名当地村民〝多数已好转〞，并引述〝专家〞说法称碳九〝有毒但非剧毒〞，〝不太会吸入中毒〞等。
 <br/>
 <br/>
 网民对此表示愤怒，怒斥〝为什么政府没公信力，这不是很明显吗〞。
 <br/>
 <br/>
 此次事故令当地村民损失惨重，不少人家几十年积累的鱼排血本无归，损失数百万，还有人表示，损失大到已无法估量。
 <br/>
 <br/>
 当地村民们为讨说法，纷纷将受污染死亡的烂鱼、烂虾摆在区政府和东港石化公司门前。〝每天都有二三百人去抗议，要把这些烂鱼虾卖给区长。〞。
 <br/>
 <br/>
 还有村民在东港石化公司的大门上挂起〝无良东港毒害百姓毁我家园！！！〞横幅。并在公司门前搭起帐篷，坐在那里抗议。
 <br/>
 <br/>
 但直到8日，当地政府与石化公司方面都没有任何官员出现，给村民们一个说法。
 <br/>
 <br/>
 有评论指，在危及公共安全事件上，中共政府和官员会习惯性地隐瞒事故严重程度，同时又缺乏处理问题的专业性和及时性，更一味造假掩盖，欺骗民众称问题〝完美解决〞，却更加引发民众质疑，而想要中共实事求是的〝公开进度〞，更是绝无可能。
 <br/>
 <br/>
 (记者罗婷婷综合报导/责任编辑：文慧)
