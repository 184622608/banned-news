### 王岐山罕见独占党媒头版  陈破空:习王唱黑白脸与美谈判
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】周三，中共喉舌媒体《人民日报》在头版版面配图的安排上，罕见整版只配发了一张图片，却不是中国国家主席习近平或国务院总理李克强的图片，而是国家副主席王岐山的图片，这个现象引起舆论的关注。有外媒引述观点称，近日习近平和王岐山在公开场合的讲话显示，他们正为对美贸易谈判而分别扮〝黑脸〞和〝白脸〞。
 <br/>
 <br/>
 据了解，7日的《人民日报》头版的上半部分虽然安排的是习近平和李克强的报导，但并未配发任何图片。在头版右下角刊登的有关王岐山在新加坡出席彭博创新经济论坛的报导，却配发了王发表讲话时的单人照片。
 <br/>
 <br/>
 这种安排与过去中共官媒的头版图片以优先安排习、李二人照片的惯例不同，引起外界的注意，有舆论戏称王岐山〝独占〞党媒头版。
 <br/>

 </center>
 <br/>
 《自由亚洲电台》7日在一篇报导中提及了这个特殊的现象。该报导并引述美国纽约的政治评论人士陈破空的观点，讨论了当前北京当局急于为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 中已陷入困境的中国经济解套的心态。
 <br/>
 <br/>
 陈破空表示，中共政权一贯是欺软怕硬恃强凌弱的，而现在美国很强大，所以中共高层当下的心态，还是不敢得罪美国政府，哪怕是〝委曲求全〞 也要跟美国尽量搞好关系。
 <br/>
 <br/>
 陈破空对比分析了近日习近平在上海的讲话和王岐山在新加坡的讲话指出，习、王二人的讲话〝调子〞有所不同。习近平的语气更强硬一些，而王岐山的语气就明显比较软，总的来说他们二人是为了与美国政府的贸易谈判而分别在演〝黑白脸〞的角色。
 <br/>
 <br/>
 据公开的资讯，习近平11月5日在上海出席中国首届国际进口博览会开幕式时讲话称，中国将降低关税，减少投资壁垒，并加快开放国内市场。但他同时声称，那些批评中国贸易政策的人，〝在指责中国前应先解决自身存在的问题〞。
 <br/>
 <br/>
 习还声称，〝中国经济是一片大海，而不是一个小池塘。狂风骤雨可以掀翻小池塘，但不能掀翻大海。经历了无数次风狂骤雨，大海依旧在那儿。〞
 <br/>
 <br/>
 王岐山则于当地时间11月6日在新加坡出席〝彭博创新经济论坛〞时发言称，〝贸易和经济合作仍然是中美关系稳定健康的支柱和推动力。〞
 <br/>
 <br/>
 王岐山并声称，〝当前世界面临的重大问题都需要中美紧密合作〞，而中方愿意与美方就双方关切的问题展开磋商，〝推动经贸问题达成一个双方都能接受的方案〞云云。
 <br/>
 <br/>
 陈破空对此分析称，习近平在上海的进博会上唱高调，一方面是考虑到要顾一点〝面子〞，另一方面也是为月底可能出现的变数预先准备一个台阶下；而王岐山赶在美国中期选举结果出炉前到新加坡发表一番对美国〝示好〞的言论，是想向川普政府表明一个态度，就是中方没有指望等民主党赢了再来对付美国，以此换取川普和共和党在贸易问题上〝放他一马〞。
 <br/>
 <br/>
 陈破空表示：中共一直期望美国两党的政治博弈可以给中共面临的被动局面解套。在川普强硬放话指责中共介入美国国内选举后，北京方面赶快补救，由王岐山出面主动示好，也为自己先找个下台之阶。
 <br/>
 <br/>
 （记者闻轩综合报导/责任编辑：明轩）
