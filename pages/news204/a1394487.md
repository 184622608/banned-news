### 陆客海外观光新流行：冻卵考照买房
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】大陆7天长假结束，许多大陆人在假期出境
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旅游.html" target="_blank">
  旅游
 </a>
 的同时，在境外考驾照、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冻卵.html" target="_blank">
  冻卵
 </a>
 子、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_买房.html" target="_blank">
  买房
 </a>
 子等，而不是传统意义上单纯的旅游观光。
 <br/>
 <br/>
 据英国广播公司（BBC）中文网的报导，很多大陆游客在海外游时，不是简单地前往热门景点，而是有一些意想不到的目的。大陆游客〝边玩边办事〞，如考取国际驾照，冷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冻卵.html" target="_blank">
  冻卵
 </a>
 子，买卖房子等等。
 <br/>
 <br/>
 据报导，韩国南部的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_济州岛.html" target="_blank">
  济州岛
 </a>
 成为不少急求驾照而不得的中国人心中的驾考天堂。根据韩国济州岛驾考中心的数据，到目前为止，2018年已有2172名中国公民在当地获得驾照，平均每周有近60人。
 <br/>
 <br/>
 大陆游客在济州岛的考试速度很快，成功率很高。笔试还提供了中文版本。通常，一个下午可以完成考试的多个项目。
 <br/>
 <br/>
 由于大陆没有加入〝联合国陆路交通国际条约〞，大陆人无法取得国际驾照。很多经常出国的人，便专门前来考韩国驾照。
 <br/>
 <br/>
 大陆法律允许其他国家有效驾照的持有者，可换领大陆驾驶证。因此，一些没有驾照且完全不会开车的人，也来韩国取得驾照，再回去大陆更换。
 <br/>
 <br/>
 越来越多的大陆人前往台湾进行〝冻卵
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旅游.html" target="_blank">
  旅游
 </a>
 〞。由于大陆禁止单身女性冻卵，台湾因价格低廉且距离不远受到了大陆女性的青睐。台湾新竹的送子鸟生殖中心的统计数据显示：其冻卵手术外来客户大部分来自大陆，且年增长率都将近50%。
 <br/>
 <br/>
 大陆购房者前往柬埔寨，兴起〝看房游〞。数据显示，2017年，大陆首次超过越南，成为柬埔寨第一大游客来源国。
 <br/>
 <br/>
 在这些来柬埔寨的大陆游客中，一部分人的目标并非吴哥窟，而是一些建成没多久的公寓楼。
 <br/>
 <br/>
 据一名从事柬埔寨地产生意的商人介绍：〝中国游客最看重的是地段。〞〝今年每天都有3组左右。外来人口和交通压力也增加不少〞。
 <br/>
 <br/>
 金边的这番景象并非特例。越南旅游城市芽庄也曾掀起了〝看房游〞，很多大陆人藉着在芽庄旅游的机会，边看景边看房，有的人当场买下。
 <br/>
 <br/>
 (记者李蒨蒨报导/责任编辑：文慧)
