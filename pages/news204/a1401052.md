### 【禁闻】债务与贸易战叠加 威胁中国经济
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 面临危机是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中外专家.html" target="_blank">
  中外专家
 </a>
 的共识，美国专家说最大的问题是中国的债务而不是美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，而中国专家却认为，贸易战会影响中国一些大省的收入，加剧整体债务。两相叠加会更加威胁中国的经济，乃至社会稳定。
 <br/>
 <br/>
 11月24号，《福布斯》专栏作家、纽约长岛大学和哥伦比亚大学经济学教授潘诺斯．莫道寇塔斯（Panos Mourdoukoutas）发表文章称，中国最大的问题是债务而非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 。
 <br/>
 <br/>
 中国金融智库研究员巩胜利：〝国内的债务和贸易战其实是比肩的，因为以前的40年都是靠贸易来拉动（经济）的，这个贸易战一打，出口创汇就没有了，所以外来资本也好，出口创汇也好，没有了。再加上国内地方债务的问题，那可能就是内外交困叠加。〞
 <br/>
 <br/>
 不过，莫道寇塔斯在文章中也指出，美中贸易战本身虽然不会对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 构成重大威胁，但将会加剧中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_债务危机.html" target="_blank">
  债务危机
 </a>
 。
 <br/>
 <br/>
 文章中还说，虽然中共官方的数字称国债占GDP的比重仅为47.6%，但非官方的数字并不如此，如国际金融协会上星期说，中国国债占GDP的比重高达300%。
 <br/>
 <br/>
 中国金融智库研究员巩胜利告诉新唐人，根据他看到的数据，2017年中国除了北京、上海、广州和深圳盈利外，其他省市都要依靠财政补贴来过日子，说明中国的地方债务相当严重，而这种严重程度还是有以前40年的发展兜底，一旦贸易战效果显现，广州和深圳这两个主要依靠出口的城市收入大幅萎缩，中国的经济困境就可想而知。
 <br/>
 <br/>
 巩胜利：〝深圳和广州都是两个盈利的城市，它们的GDP超过了2万亿，两个城市超过4万亿，4万亿相当于中国31个省市自治区当中，除了上海和北京之外，把其他的财政亏损就打平掉。〞
 <br/>
 <br/>
 根据中共财政部的官方数据，截至2018年6月末，中国地方政府债务余额167997亿元。财政部称，这个规模控制在中共全国人大批准的限额之内。
 <br/>
 <br/>
 不过，中共人大财经委副主任委员贺铿在今年5月的一次会议上透露，中国的企业债务太高，已经超过50%的危险线，尤其央企资产负债率91.8%是大问题；而地方债比企业债更加可怕，大概是40万亿，但是政府信用比企业信用还差，地方政府就没有一个想还债的，因为地方政府连工资都发不出去。
 <br/>
 <br/>
 在美中贸易战冲击之下，中共在今年8月决定鼓励银行增加放贷，并再次允许负债累累的地方政府建大型项目，以应对日益放慢的经济增长。同时允许银行几乎不受限制地持有地方政府债券。
 <br/>
 <br/>
 大陆金融分析师任中道：〝其实进行了一个转移，现在坏帐银行已经纷纷开始采取行动了，就是购买所谓的这些资产包，就是不好的一些资产啊债务啊都打到一起，然后再拆分，以各种所谓的资产证券化，甚至拿到二级市场上进行交易，其实最后还是转嫁到老百姓的身上去。〞
 <br/>
 <br/>
 大陆金融分析师任中道表示，一旦把这些来自地方政府的不良债务几经周转证券化，在证券市场被民众购买后，民众就会受到损失，这等于是继今年P2P暴雷，和房市限售等方式割韭菜后，再次对民众财富的巧取。
 <br/>
 <br/>
 据中国人民大学11月24号发布的报告，中国消费者在房地产去库存中债务率大幅度上升，消费基础受到严重削弱。目前居民财富基本上被房地产掏空，广大中产阶级和中下收入阶层被房地产套牢。
 <br/>
 <br/>
 任中道再次强调，一旦贸易战导致这些被套牢的民众收入下降，还不起债务的民众就可能起来反抗，导致中共最恐惧的社会动乱。
 <br/>
 <br/>
