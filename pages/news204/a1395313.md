### 3〝T〞令中美关系跌底 外媒：中共官员对北京立场沮丧
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】自贸易战开打后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 快速恶化，美媒称，有3个〝T〞让美中关系跌入谷底，分别是台湾（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Taiwan.html" target="_blank">
  Taiwan
 </a>
 ）、贸易（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Trade.html" target="_blank">
  Trade
 </a>
 ）与美国总统川普（特朗普
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Trump.html" target="_blank">
  Trump
 </a>
 ）。还有中共官员私下透露，对于北京在贸易、南海争议上的决策感到沮丧，认为北京的立场激怒美国。
 <br/>
 <br/>
 目前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 已陷入前所未有的低潮，《CNN》报导称，虽然有消息指习近平将与川普在G20峰会见面，但美中双方专家都担忧恐为时已晚。
 <br/>
 <br/>
 多年以来，美中两国因为意识形态上的差异，在很多问题上存有争议。在20世纪90年代到21世纪，美中之间最有争议的是台湾（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Taiwan.html" target="_blank">
  Taiwan
 </a>
 ），西藏（Tibet）与天安门（Tiananmen）这3〝T〞。
 <br/>
 <br/>
 但现在，除了台湾之外，另外两个〝T〞已经换成了贸易（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Trade.html" target="_blank">
  Trade
 </a>
 ）与川普（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Trump.html" target="_blank">
  Trump
 </a>
 ）。新3〝T〞主导了美中关系，也令其因贸易战开打跌入谷底。
 <br/>
 <br/>
 目前，面对川普政府的强力攻势，北京已经出现软化迹象，中共驻华盛顿大使表示，如果美国愿意表现出诚意，则北京愿意〝让步〞。还有一些中共官员则私下表示，对北京在贸易和南海争议上的决策感到沮丧，认为北京的立场激怒美国。
 <br/>
 <br/>
 外界普遍认为，北京当局对于川普政府有诸多误判，对于川普总统提出的公平贸易、停止盗窃知识产权等要求，北京不以为然，错误的以为沿用之前的〝大订单〞等老套路，就能搞定川普。
 <br/>
 <br/>
 孰料川普迟迟未见到北京做出实质的改变，果断开启关税制裁，以逼迫北京走到公平贸易的道路上来。今年7月6日，中美贸易战正式开打，中美关系由此急转而下。
 <br/>
 <br/>
 亚洲协会美中关系中心主任谢尔（Orville Schell）说，川普不害怕制造混乱，而这是过往美国总统不愿意做出的举动。
 <br/>
 <br/>
 美国国家安全顾问博尔顿（John Bolton）则表示，川普对中共政府的强硬态度让对方〝感到困惑〞。
 <br/>
 <br/>
 他说：〝他们（中共）从未见过有美国总统如此强硬。我认为，他们的行为需要在贸易领域、国际、军事和政治领域以及一系列领域进行调整。〞
 <br/>
 <br/>
 事实上，美国现在的对华政策已经发生根本转变。2017年，川普政府出台新的国家安全战略，已经将中共与俄罗斯列为美国的〝战略竞争者〞。凸显美国对让中共走向自由民主已彻底失去信心。
 <br/>
 <br/>
 一些美国学者表示，美中交手不仅仅局限于贸易领域，双边关系已经从1970年代的〝全面接触〞，转向今天的全方位〝对抗性竞争〞（Rivalry）。
 <br/>
 <br/>
 这些学者还说，即便是北京继续〝韬光养晦〞，也改变不了美中关系最终走向〝对抗性竞争〞的轨迹。
 <br/>
 <br/>
 日本《外交官》杂志4月的一篇文章也指出，川普总统的对华强硬态度，体现了华盛顿正在形成的两党共识，那就是，〝早该对北京强硬了〞。
 <br/>
 <br/>
 美国8月出台的《2019年国防授权法案》，首次明确将中共和俄罗斯作为战略和竞争对手，美国将从政治、经济、军事等层面对中共进行全方位严厉反制。
 <br/>
 <br/>
 威尔逊学者中心的戴博认为，即便不是川普，也没有习近平，美中关系也会走向越来越激烈的竞争之路。他解释说，西方国家对中国的担心，其实是对共产党政权的担忧。
 <br/>
 <br/>
 美国总统川普9月25日在联合国大会上的演讲中，痛批社会主义，称社会主义和共产主义给人类带来了苦难和腐败，所有国家都应抵制社会主义和它给所有人带来的痛苦。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
