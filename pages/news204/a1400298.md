### 【禁闻】11月21日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月21日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 律师李金星因网络言论遭司法局调查
 <br/>
 <br/>
 山东律师李金星11月20号收到济南市司法局的书面通知书，说他因在网络上发表不当言论遭立案调查，要求他11月23号前往司法局说明情况、并接受调查。
 <br/>
 <br/>
 2. 无锡
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 进京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 一个月被拘留二次
 <br/>
 <br/>
 无锡抗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 维权人士倪满良11月初因进京上访，遭拘留10天，17号获释后，18号再次进京，并再次遭截访遣返拘留10天。倪满良在一个月内，被连续拘留两次。
 <br/>
 <br/>
 3. 在京访民举牌 要求释放河南陈文慧
 <br/>
 <br/>
 河南访民陈文慧因被中共党员干部抄家放火、贩卖妹妹及妹妹的三个孩子而长期上访，10月15号在家中被抓，一直被关押在信阳市看守所。11月19号，多名在京访民上街举牌，要求当局立即放人。
 <br/>
 <br/>
 4. 重庆市民遭强拆 被殴打拘留
 <br/>
 <br/>
 重庆九龙坡市民刘富祥家的房屋，11月9号遭当地政府暴力强拆，警方当时对刘富祥等4人暴力殴打后带走，并以涉嫌妨碍公务罪关进九龙坡区看守所，21号家属委托的律师前往会见，但当局只允许刘富祥一人与律师见面。据了解，刘富祥被强拆人员打断腿骨，11月19号才得以就医。其他几人的伤势不明。
