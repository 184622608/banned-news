### 断尾求生？路透：海航狂抛逾百亿美元房产
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】面对巨额债务，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海航集团.html" target="_blank">
  海航集团
 </a>
 正在悄悄变卖巨额资产还债。路透日前报导，相关文件显示，海航集团已挂牌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出售.html" target="_blank">
  出售
 </a>
 或打算出售逾80处
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房地产.html" target="_blank">
  房地产
 </a>
 资产，总价值逾110亿美元。
 <br/>
 <br/>
 《路透》9日报导，消息人士称，相关文件是在8月份发给潜在投资者的。文件显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海航集团.html" target="_blank">
  海航集团
 </a>
 已挂牌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出售.html" target="_blank">
  出售
 </a>
 或打算出售逾80处
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房地产.html" target="_blank">
  房地产
 </a>
 资产，包括酒店、商业地产和住宅，大部分位于海航集团总部所在地海南岛。
 <br/>
 <br/>
 除此之外，自今年1月以来，海航已出售或准备出售的资产超过了200亿美元，其中包括纽约、雪梨、香港的房产。
 <br/>
 <br/>
 2月9日，海航集团出售了其在德意志银行0.7%的股份，套现超过16亿人民币；2月16日（大年初一），海航把位于纽约曼哈顿第六大道1180号的物业出售给了Northwood Investors，交易金额超过3亿美元。
 <br/>
 <br/>
 4月中旬，海航集团出售
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希尔顿酒店.html" target="_blank">
  希尔顿酒店
 </a>
 旗下三家上市公司股份，共计套现约83.45亿美元；8月8日，海航集团拟转让其下属子公司Avolon30%股权，交易对价22.12亿美元。
 <br/>
 <br/>
 截止9月，海航已卖出了超过了170亿美元的资产。
 <br/>
 <br/>
 据法广9月中旬称，海航集团正在洽谈出售德意志银行股权、渤海金控债券、凯撒旅游股份等更多资产以期偿债。
 <br/>
 <br/>
 熟悉海航集团规划的公司内部消息人士说，他们正在战略性地退出一些地区，但这不是在甩卖。
 <br/>
 <br/>
 据悉，截止6月底，海航集团债务总计高达5416亿元人民币，其下属的一家附属公司此前也因未能按时偿还一笔3亿元人民币的信托贷款，造成违约。
 <br/>
 <br/>
 海航、万达、安邦、复星4家民企，去年，被中共点名涉嫌以内保外贷名义向境外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转移资产.html" target="_blank">
  转移资产
 </a>
 后，万达甩卖了超过80%的资产以还债，安邦被中共接管，董事长吴小晖被捕判刑。
 <br/>
 <br/>
 然而海航却受到了中共当局的救助，当局指令各大银行继续为其提供贷款，同时指令海航变卖部分资产还债，海航的不同待遇引发外界关注与疑问。
 <br/>
 <br/>
 有分析认为，中共担心海航一旦崩溃，会连带许多银行垮台，造成难以想像的更严重的后果。也有评论表示，海航是中共多家权贵的白手套，所以要把海航保下来。
 <br/>
 <br/>
 7月初，57岁的海航集团董事长王健在法国离奇死亡事件，也曾一度震惊海内外舆论。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
