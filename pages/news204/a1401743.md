### 贸战停火埋隐患？美政界要人：美中关系已变质
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】日前结束的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 火暂缓，但有专家发现，北京隐瞒了本次协议中两个最关键细节，为将来埋下隐患。另有美政界要人称，美中关系已从根本上〝变质〞，即使美中双方达成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_休战协定.html" target="_blank">
  休战协定
 </a>
 ，但这个结果也可能很难实现。
 <br/>
 <br/>
 12月1日在阿根廷首都举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 ，再度引发国际舆论热议。美中双方先后发布声明，都称此次会谈是〝成功〞的。不过，有一些专家仔细分析中美双方的报导和声明，发现〝魔鬼〞藏在细节里。北京隐瞒了本次协议中两个最关键细节，为将来埋下隐患。
 <br/>
 <br/>
 在中共官媒的报导中称，双方决定停止升级关税等贸易限制措施，包括〝不再提高现有针对对方的关税税率，及不对其他商品出台新的加征关税措施。〞中方似乎暗示，美国不会再对其余2670亿美元中国输美商品征税。
 <br/>
 <br/>
 报导还称，中方将进一步〝扩大开放〞，在此过程中，美方关注的一些经贸问题会得到解决。然而，报导中并没有公布，本次休战协议有一个要命的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_90天期限.html" target="_blank">
  90天期限
 </a>
 ，以及更要命的是，中共必须要在此限期内进行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性改革.html" target="_blank">
  结构性改革
 </a>
 〞。
 <br/>
 <br/>
 外界发现，中共外长王毅在会谈结束后的记者会上，也声称川普和习近平已同意2019年不会有额外的征税措施，并有意识忽略了90天的期限，以及〝结构性改革〞的内容。
 <br/>
 <br/>
 中方的报导与官方表态，显然与白宫的声明有天壤之别。白宫在发布的声明中明确表示，〝在贸易问题上，川普总统同意在2019年1月1日，他将把对价值2千亿美元的产品的关税税率保持在10%，而不是在那一天提高到25%。〞川普并没有表示取消对2670亿美元中国商品征税的计划。
 <br/>
 <br/>
 而且，更重要的是，白宫的声明中称，〝川普总统和习近平主席同意立即开始有关结构性改变的谈判。这些结构性改变涉及强制技术转让、知识产权保护、非关税壁垒、网路入侵和网路盗窃、服务业和农业。〞
 <br/>
 <br/>
 声明还指出，〝双方同意，他们会努力在未来90天内完成谈判。如果双方在这个期限内未能达成协议，10%的关税将会被提高至25%。〞
 <br/>
 <br/>
 毫无疑问，这份协议对美中双方来说，只是将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 暂缓升级而已，之前的关税仍要征收，更谈不上什么〝停火〞之说。大陆经济学人巩胜利表示，这次川习会可以用四个字来说，就是〝维持原战〞。
 <br/>
 <br/>
 不仅如此，美国对这个暂缓贸易战升级的动作，还划定了一个界线。90天后，如果北京没有做出实质性改变，美国已经征税的2000亿美元中国商品〝10%的关税将会被提高至25%〞。
 <br/>
 <br/>
 法广分析称，川习晚宴结束时，会议厅里的确响起一片掌声。川普的高兴可以理解，美方寸步不让，已经实施的10%的关税率继续实施，只是把原先预告提高税率至25%延缓90天。而北京是被动的，被套上了〝紧箍咒〞，而且不愿向自己的人民如实禀告。
 <br/>
 <br/>
 报导称，总之一句话，贸易战没有结束。说轻点，北京获得喘一口气的机会；说重点，北京被判处90天缓刑。
 <br/>
 <br/>
 此外，对于这场川习会能否缓和美中对抗态势，曾担任美国助理国防部长的傅立民提出了自己的看法。
 <br/>
 <br/>
 《日本经济新闻》报导称，傅立民表示，中美从不损害相互利益的〝竞争（rivalry）〞关系，已转向虽不至于发展成战争、但是损害、削弱对方能力的〝敌对对立（adversarial　antagonism）〞关系。就像冷战时期的美苏关系。
 <br/>
 <br/>
 他认为，美中首脑会谈即使达成暂时阻止贸易战恶化的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_休战协定.html" target="_blank">
  休战协定
 </a>
 ，但这个结果也可能很难实现。因为中美关系已从根本上变质，前景不容乐观。
 <br/>
 <br/>
 外界还有不少声音称，本次川习会谈判没有意义，中共又是用〝拖字诀〞。
 <br/>
 <br/>
 国际问题专家唐浩曾表示，美中贸易战的冲突，不是美国与中国的冲突，而是自由社会与中共体制的冲突，是普世价值与变异价值的冲突。只有把变异价值彻底消解了，才能彻底化解贸易战冲突；而要彻底消解变异价值，就得彻底解体中国共产党。接下来，就看习自己能否真正认清中共，做出理性睿智、利国利民的选择了。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
