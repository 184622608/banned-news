### 暗藏政治秘密？澳门中联办主任讣告语带玄机
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月28日讯】中共驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 中联办主任郑晓松日前猝死，官方第一时间宣称〝因抑郁症堕楼〞，外界普遍猜测是涉案自杀或遭他杀。而官方最新公布的讣告称其〝清正廉洁〞，并表示中共高层出面〝哀悼慰问〞，或刻意否定涉贪被查的传闻。
 <br/>
 <br/>
 10月28日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 中联办官网公布郑晓松的讣告，其中几处内容引发关注。
 <br/>
 <br/>
 其一，讣告称郑晓松〝忠于职守、敢于担当〞，〝坚持原则、清正廉洁〞。外媒分析，这可能是有意回应有关郑晓松〝死前遭中纪委问话〞的传闻。
 <br/>
 <br/>
 其二，讣告提到，郑晓松〝身患疾病依然忘我工作〞，回避了此前官方通告中的〝抑郁症〞。
 <br/>
 <br/>
 其三，讣告指，中共高层和中组部、国务院和澳门相关部门都出面表示〝哀悼〞和〝慰问〞，郑晓松骨灰将被安放到北京八宝山。
 <br/>
 <br/>
 中共官方20日宣布了郑晓松的死讯。而23日北京最高领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 出席了珠港澳大桥通车仪式，而郑晓松本已被安排出席仪式。
 <br/>
 <br/>
 港媒分析，中共官方启动应急程序，20日紧急宣布郑晓松因〝忧郁症〞于19日晚〝跳楼自杀〞，旨在〝从速善后〞，迅速淡化影响。
 <br/>
 <br/>
 对于郑晓松在和习近平见面前突然死亡，外界普遍认为内有黑幕。有人认为是郑怕见习近平，也有猜测指有人怕他见到习近平。
 <br/>
 <br/>
 《美国之音》引述香港浸会大学高级讲师吕秉松的分析指，郑晓松猝死背后肯定涉及〝更大的机密〞。
 <br/>
 <br/>
 郑晓松2013年得到重用，空降习近平曾经主政的福建省任副省长，去年9月接替习近平旧部王志民，出任澳门中联办主任。有分析指，郑晓松或属于习系人马。
 <br/>
 <br/>
 还有海外媒体称，港澳历来是江派
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾庆红.html" target="_blank">
  曾庆红
 </a>
 的势力地盘，郑晓松在习近平抵达前突然死亡，是曾庆红给了习近平〝当头一棒〞。
 <br/>
 <br/>
 据港媒报导，习近平22日开始的这次〝南巡〞，行程非常隐秘低调。在大桥通车仪式开始时，习近平迟到达半个小时，只用一句话宣布大桥通车即匆匆离场。而台下给他事先安排的座位，也罕见地没有放置名牌。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：云涛）
