### 分析：川普一招直击要害 北京若不让步恐无路可走
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】即将举行的G20川习会晤，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 火能否平息的关键。有中共体制内学者分析，在对华贸易政策上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府联合各国采用〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_封锁战术.html" target="_blank">
  封锁战术
 </a>
 〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_直击北京要害.html" target="_blank">
  直击北京要害
 </a>
 ，中共经济内外交困，北京若不做出让步，恐陷入无路可走的境地。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）发起的对华
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，对中共经济与世界经贸格局都造成了深远的影响。11月12日，中国人民大学国际关系学院教授时殷弘，对北京将如何应对贸易战做出如下分析。
 <br/>
 <br/>
 在美国关税打击下，近期中国股市低迷、汇率下跌，经济增速放缓。同时，中共经济长期突出的问题，去槓杆与稳增长之间的矛盾也显露出来。近日，中共召开政治局会议罕见承认，中共经济正遭遇下行压力，凸显中共经济的确已陷入危险境地。
 <br/>
 <br/>
 时殷弘认为，与国内经济危局相比，更危险的是，北京可能会被美日欧在国际经贸市场上孤立，这种局面将让北京无路可走。
 <br/>
 <br/>
 他表示，在对华贸易政策上，川普总统表现出了很强的战略思维。美国在不断提升对华关税的同时，整合与欧洲、日本等国家的经贸联系，采用技术封锁的方式向北京施压，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_直击北京要害.html" target="_blank">
  直击北京要害
 </a>
 。
 <br/>
 <br/>
 10月份，美国与墨西哥、加拿大签订美墨加协定，其中有一条抵制中共等非市场行为体的〝毒丸条款〞。时殷弘称，一旦该条款应用到美国与日本、欧洲签订的自贸协定中，将意味着北京被杜绝于国际经贸高端市场，因为世界上主要的先进科技，都来源于美日欧三方。
 <br/>
 <br/>
 他警告，一旦这种局面形成，对中共经济的打击将会十分严重。
 <br/>
 <br/>
 中共经济风雨飘摇之际，11月1日，川普与习近平进行长时间通话。这是自贸易战开打后，5个月以来两国元首的首次通话。随后，双方宣布将在11月末阿根廷G20峰会期间，举行川习会晤。
 <br/>
 <br/>
 外界传闻双方可能会达成协议。日前传出消息，刘鹤即将再度访问美国，继续两国的贸易谈判。刘鹤此行被认为是在给川习会铺路。
 <br/>
 <br/>
 对此，时殷弘预测，面对中国国内经济下行的压力，北京不排除对美国做出空前让步的可能，以降低中共的经济损失。
 <br/>
 <br/>
 不过，中美双方存在重大分歧，此前的多次谈判均未果。美国副总统彭斯日前对《华盛顿邮报》表示，此次川习会是处理两国分歧的一次机会。但前提是，北京必须在经济、军事和政治行为上做出彻底的改变。
 <br/>
 <br/>
 〝中国（共）需要做出让步的领域包括对保护知识产权、禁止技术转移、取消限制进入中国市场、尊重国际规则及规范，以及保障在国际水域航行自由和中共对西方政治的干预〞。彭斯说，这是中共避免与美国发生冷战的最好机会。
 <br/>
 <br/>
 他同时警告，若中方不能做到这些，美国将会在经济、外交、政治及军事上，对中共施加更大压力。美国会向更多的中国进口商品加征关税。
 <br/>
 <br/>
 彭斯说，〝对华关税现已达到2500亿美元，我们可以承受比这一数字两倍还多〞。美国的经济有足够的实力，相比较而言，中共的经济〝耐性较差〞。
 <br/>
 <br/>
 他最后强调，美方看重的并不仅是来自中方的承诺，更为重要的是最终的结果，〝我们期待中方改变姿态〞。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
