### 朋友圈暗示11月20日复出？ 范冰冰动向引人关注
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】近日，曾有中国网友晒出一张疑似中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 朋友圈的截图，暗示范将于11月20日露面，外界纷纷猜测范将很快
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_复出.html" target="_blank">
  复出
 </a>
 。不过随后中共党媒又发文狠批触犯法律的艺人，令事态再次峰回路转。如今20日已至，外界都在等待范冰冰是否会有新的动向。
 <br/>
 <br/>
 当地时间11月14日，有网友晒出一张疑似
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 朋友圈的截图。截图中有一张范冰冰的照片，上书〝11月20日见〞，一度有许多人猜测范冰冰可能很快就会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_复出.html" target="_blank">
  复出
 </a>
 。
 <br/>
 <br/>
 范冰冰因涉嫌签署〝阴阳合同案〞卷入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃税.html" target="_blank">
  逃税
 </a>
 风波，今年6月以后，从公众视野中消失了三个多月。直到今年10月3日，中共当局才公布了对范冰冰的处罚决定，范以缴清税金、滞纳金和罚金共计逾8.8亿元人民币而逃过了牢狱之灾。随后，她在微博发表〝致歉信〞表示认错认罚。
 <br/>
 <br/>
 之后，范冰冰于10月15日现身北京机场被人拍到照片，当时的她一身全黑打扮，非常低调，全程由保镳帮撑着黑伞，试图挡住媒体拍照。
 <br/>
 <br/>
 10月19日，陆媒新浪娱乐爆料称，范冰冰与母亲张传美当天一同出现在其位于北京三里屯的工作室。在里面停留近5个小时后，于当晚9点左右，母女二人才走出大楼。
 <br/>
 <br/>
 网络上曝光的照片显示，范冰冰当时身穿白色上衣，头戴蓝色贝雷帽和黑色墨镜，背着一个白色包包。虽然衣着朴素，但妆容精致。
 <br/>
 <br/>
 与此同时，范冰冰的粉丝们在网络社交媒体上发帖为范冰冰〝复出〞营造声势的势头逐渐开始抬头。
 <br/>
 <br/>
 不料，中共党媒11月16日突然发文狠批艺人，强调〝不能再让有法律和严重道德污点的艺人出名获利〞。虽然此文并未指名道姓，但大陆网友普遍认为那些说辞很可能是针对范冰冰的。舆论分析，这或意味着范冰冰的复出之路遭中共官方封堵，范的演艺生涯可能会就此终止。
 <br/>
 <br/>
 近日，台湾金马奖获奖感言事件后，在中共官媒的一片声讨声中，范冰冰也连夜转发中共共青团中央的微博，并留言称〝中国，一点都不能少！〞 她的未婚夫李晨则转发了中共党报的微博，与范冰冰一同站队，以表明其〝爱国立场〞。
 <br/>
 <br/>
 范、李二人此举被认为是向中共官方表态站队。舆论认为，他们可能是指望通过这些表态来替范冰冰的复出换回一点政治筹码。
 <br/>
 <br/>
 近期，网络还一度流传范冰冰与李晨将要移民美国的消息。有许多网友和范冰冰的粉丝在网上留言：离开了就别再回来了，远离这个是非之地。
 <br/>
 <br/>
 （记者欣然综合报导/责任编辑：范铭）
