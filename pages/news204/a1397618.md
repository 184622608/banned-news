### 法媒：中共〝外包〞肮脏工作给流氓恶棍
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 地方政府与流氓和恶棍沆瀣一气，对付人民的内幕，日前被一家调查机构曝光。地方政府雇用私人保安或社会闲散人员充当政府的打手，拦截上访人士；黑帮势力与地方政府相互依赖勾结，代替警察和政府执行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肮脏.html" target="_blank">
  肮脏
 </a>
 任务〞。
 <br/>
 <br/>
 法广10月31日报导，专门研究中国问题的网站〝中国风〞（Chinoiresie）一篇文章指出，不论是在2014年的香港雨伞运动，还是2012年山东临沂市东师古村陈光诚被软禁在家事件，地方政权都有雇用流氓和恶棍充当政府的打手。雇用地方黑社会人员拦截上访人士也是世人皆知的事实。也有的地方，政府利用专业人士，诱骗人民放弃维权。
 <br/>
 <br/>
 文章称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 为了避开被外界批评使用非合法性的暴力，因此很多时候会将这些暴力工作外包给第三者。即雇用流氓采取非合法性的暴力手段对付人民，而中共在事后推卸责任。除了军队、警察以及政府工作人员，这种恶棍式的执法，是其控制人民的补充手段。
 <br/>
 <br/>
 当中共地方政府明知执行手段是非法的，或推行不得人心的政策时，会雇用一些流氓。这些〝第三者〞就代替警察和政府执行这种〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肮脏.html" target="_blank">
  肮脏
 </a>
 任务〞。
 <br/>
 <br/>
 例如，大约从30年前开始，中共执行臭名昭着的计划生育政策时，逼迫妇女打胎，有时还会被这些流氓性骚扰。
 <br/>
 <br/>
 后来，流氓通常被雇用来驱赶居民离开他们的农地和家园，或者威吓示威者和异见份子。
 <br/>
 <br/>
 由于上述这些行为极其不得人心，往往会遇到反抗；有时还涉及非法。所以地方政府不愿意派遣正式的官员出面执行，而是雇用流氓。
 <br/>
 <br/>
 据知情人透露，被雇用的流氓可被视为合约工人，一般是按件付款，可以随时结束雇佣关系。不像列入编制以内的官员，除了工资，还需福利开支。
 <br/>
 <br/>
 近年来随着地方政府藉着土地开发发财，在房屋拆迁过程中，遇到了上级限时的命令，又担忧民怨爆发造成群体抗议事件，于是处于中介地位的〝黄牛党〞应运而生。这些黄牛党往往透过台底交易解决问题。
 <br/>
 <br/>
 当流氓执法时，往往发生使用过度暴力，从而出现伤亡。被媒体曝光的上访者被殴打的事例，比比皆是。
 <br/>
 <br/>
 此外，受雇的流氓从这种关系中得益，客观上得到了政府的资助，开始壮大。例如经营赌博和卖淫集团。有些地方政府亦逐渐过度依赖地方黑帮，开始与黑帮勾结。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：李泉）
