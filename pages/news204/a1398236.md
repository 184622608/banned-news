### 王全璋失踪1216天 妻痛批法官〝故意违法〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709维权律师.html" target="_blank">
  709维权律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 已失踪1216天，他的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 11月5日给案件主审法官写了一封公开信，痛批法官〝故意违法〞，长期变相拖延案件办理期限。
 <br/>
 <br/>
 42岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 在709律师案案发前，常年为社会弱势群体提供法律援助，也是最早为法轮功学员辩护的中国律师之一。他在2015年7月10日失踪，并在去年2月被天津市检察院以涉嫌〝颠覆国家政权罪〞正式起诉，但案件迄今仍未开庭。
 <br/>
 <br/>
 据自由亚洲报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 在公开信中写道，她作为王全璋律师的家属，从去年年初开始，到受理王全璋案的天津二中院寻找林昆、周虹两位主审法官不下几十次，但两人一直以不在法院的理由，避而不见。
 <br/>
 <br/>
 李文足聘请的律师程海、余文生、蔺其磊、谢阳也先后数次到法院，要求和主审法官沟通，但都被以〝法官不在〞的理由回绝。
 <br/>
 <br/>
 并且，数名律师到天津第一看守所递交代理手续要求会见王全璋时，也都被以〝需要法院批准〞的理由拒之门外。
 <br/>
 <br/>
 王全璋是200多名涉案律师和相关活动人士中，唯一至今仍杳无音信的律师。
 <br/>
 <br/>
 李文足在信中强调，王全璋已经失踪40个月，严重超出了案件办理期限的规定，林昆、周虹两位主审法官拖延案件的行为是违法行为。
 <br/>
 <br/>
 她控诉：〝我作为家属，有权聘请律师，但是律师会见王全璋却被你们法官设置了无法逾越的障碍！〞
 <br/>
 <br/>
 另外，曾代理本案的刘卫国律师，近日在微信朋友圈里发布了一封疑似王全璋的亲笔信，并附上他当天为王全璋在看守所存款的收据。但李文足表示，她对此事并不知情。
 <br/>
 <br/>
 而在2个多月前，刘卫国律师称在晨练期间意外受伤，摔断了几颗牙齿，所以无法履行辩护职责，退出该案。近日，刘卫国又在网上留言说自己会继续代理案件，这让李文足倍感困惑。
 <br/>
 <br/>
 李文足聘请的代理律师程海表示，刘卫国目前续任代理律师并不合法。从法律层面，他和蔺其磊是王全璋的亲友辩护人，法律规定只允许两名辩护人，所以刘卫国进行代理是违法的.
 <br/>
 <br/>
 他分析，除非王全璋本人解除他们其中一人的职务，否则刘卫国是不能作为第三名律师继续代理本案的。
 <br/>
 <br/>
 北京维权律师谢燕益表示，王全璋案已经过去了好几年，李文足始终无法见到丈夫，非常缺乏安全感。他呼吁当局尽快做出无罪化处理，尽快释放王全璋。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
</div>
