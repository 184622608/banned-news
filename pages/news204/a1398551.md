### 湖南尘肺病工人深圳维权 遭镇压
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】近三百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南尘肺病工人.html" target="_blank">
  湖南尘肺病工人
 </a>
 及遗属从11月5号起到深圳，连日进行维权。11月7号晚上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权民众.html" target="_blank">
  维权民众
 </a>
 遭到镇压。武警暴力殴打民众，并扔烟雾弹、喷辣椒水，11人受伤送医治疗。
 <br/>
 <br/>
 11月7号晚上，〝劳工研究〞在推特发出紧急声援，称湖南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尘肺工友.html" target="_blank">
  尘肺工友
 </a>
 及已故工友家属到深圳市政府门口，要求解决工伤工亡赔偿问题。市政府秘书长原本答应九点钟出来和工友谈，然而不到半个小时，警察就暴力殴打
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权民众.html" target="_blank">
  维权民众
 </a>
 并扔烟雾弹。
 <br/>
 <br/>
 之后，武警撤进深圳市政府大楼里，持枪严阵以待。
 <br/>
 <br/>
 现场视频显示，一名维权工友大拇指被打伤流血。
 <br/>
 <br/>
 维权工友:〝你看这就是深圳警察打的，你看，现在都没人管，都是警察在打人啊。〞
 <br/>
 <br/>
 湖南尘肺工友徐先生:〝11个送到医院，打伤几个。（警察）他不是放了烟雾弹嘛、催泪弹，有好几个伤到眼睛，呼吸也很困难。不给他们打针，受伤的就缝了针，就给他们走了，眼睛就给他们洗了，昨天晚上就出来了。〞
 <br/>
 <br/>
 11月5号开始，来自湖南耒阳、桑植和汨罗的尘肺工友陆续来到深圳人才园，这是他们今年第九次到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳维权.html" target="_blank">
  深圳维权
 </a>
 。7号上午，近三百名工友从人才园出发，散步到市委，高喊〝我们要吃饭，我们要看病〞的口号。工友们没有钱住宾馆，晚上都在人才园大厅睡觉。
 <br/>
 <br/>
 维权工友表示，他们对于深圳市政府之前出台的赔付方案不满意，要求谈判。
 <br/>
 <br/>
 徐先生：〝还在现场，总共现在有两、三百个人，深圳已经出来一个方案，但是大家都不满意，金额太少，它是按照十几年前的工资的基数来算的。〞
 <br/>
 <br/>
 据了解，这些湖南尘肺工友十多年前在深圳打工时得病。他们表示，自己为深圳发展付出健康，却得不到医疗保障和赔偿，当中70多人已病故。
 <br/>
 <br/>
 新唐人记者熊斌、陈杰采访报导
</div>
