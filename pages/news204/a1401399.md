### 习近平罕见穿中山装亮相 释何信号引关注
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】近日，在西班牙为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 举行的欢迎晚宴上，包括中共外长王毅在内的所有中方官员，都身着中山服出席活动，引发舆论关注。由于中共领导人着装往往有讲究，此次中共高层罕见的着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中山装.html" target="_blank">
  中山装
 </a>
 亮相，释放何种信号引各界猜测。
 <br/>
 <br/>
 在参加阿根廷召开的G20峰会之前，中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 一行首先抵达西班牙，进行以商业为主轴的国事访问。
 <br/>
 <br/>
 当地时间11月28日晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙国王.html" target="_blank">
  西班牙国王
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_费利佩.html" target="_blank">
  费利佩
 </a>
 六世，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马德里.html" target="_blank">
  马德里
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王宫.html" target="_blank">
  王宫
 </a>
 为习近平一行举行了欢迎
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宴会.html" target="_blank">
  宴会
 </a>
 。
 <br/>
 <br/>
 鉴于中共领导人着装往往有讲究，此次中共高层罕见的集体着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中山装.html" target="_blank">
  中山装
 </a>
 亮相，而非西装，外界认为其中释放的信号非同一般。
 <br/>
 <br/>
 有分析认为，习近平此举或是向美国总统川普释放信号，在G20〝川习会〞期间，习近平会固守中共即定的策略，不会向川普做出实质的让步。
 <br/>
 <br/>
 自习近平上台以后，除了有时穿西装以外，在不少场合都穿带拉链的深蓝色防风夹克，媒体将其称为〝习夹克〞。
 <br/>
 <br/>
 而中山装则是中共建政初期，中共第一代党魁毛泽东等中共领导人的着装风格。因此，这种服装被西方称为〝毛服装〞，也成了〝极左〞和〝守旧〞的象征。
 <br/>
 <br/>
 中共10年文革浩劫结束后，中共领导人在公开活动中，改变了清一色中山装的风格，开始穿夹克和西装。而西装成了〝西化〞和〝开放〞的象征。
 <br/>
 <br/>
 2014年习近平访问欧洲四国期间，也曾一改人们习惯的西装领带的形象，穿了一套类似中山装的礼服，参加荷兰王室举办的国宴。
 <br/>
 <br/>
 当时有分析说，中共没有对应英美国家黑领结晚礼服的正式礼服，因此，有时候中山装会被当成了救场的好东西，成了有中国特色的礼服。
 <br/>
 <br/>
 但是由于此次正值G20〝川习会〞前，中共领导人罕见的集体着〝极左〞和〝守旧〞象征的中山装，出席重大晚宴，难免会引起舆论猜测。
 <br/>
 <br/>
 11月30日至12月1日，G20峰会将于阿根廷首都布宜诺斯艾利斯召开。期间，习近平将与美国总统川普共进晚餐，讨论中美贸易问题及其它议题。
 <br/>
 <br/>
 川普26日在接受《华尔街日报》的采访时表示，他〝极不可能〞接受北京推迟提高关税的要求，预计会推进对2000亿美元的中国商品加征关税。
 <br/>
 <br/>
 川普强调唯一可能的协议是，在贸易上，中方必须公平对待美国。中共必须向来自美国的竞争者开放市场。如果和习近平不能达成协议，将会对另外价值2670亿美元的中国商品加征关税。
 <br/>
 <br/>
 在〝川习会〞前夕川普说这番话，外界认为这是向北京施压，告诉北京，〝公平对待美国〞是美国的〝底线〞，如果不能满足，下一步就将加征关税。
 <br/>
 <br/>
 同一天，中共驻美大使崔天凯也接受了《华尔街日报》的采访，他表示期待美中首脑会晤推动双方在经贸等领域的进展。其预期的会谈成果是达成指导性的框架协议，但在美方关切的知识产权问题，包括强制技术转让方面中方将坚持一概否认的立场。
 <br/>
 <br/>
 分析人士认为崔天凯的这番话，透露了中共的贸易战〝底牌〞。中共急切希望〝川习会〞达成〝停火〞协议，不再升级贸易冲突、不再继续推动贸易议程，这显然不现实。
 <br/>
 <br/>
 而美国尽管没有直接提出中共需要在体制上，或是在政治结构上做出改变，但这种经济上的要求，实际上直指中共的体制，但中共的底线就是维持统治，影响到其政权和统治的事情，它绝不会干。因此对于〝川习会〞是否达成协议外界并不看好。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
