### 【禁闻】陆吸引外资最多？专家：假外资回国套利
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 打得正酣，海外媒体纷纷报导
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外资.html" target="_blank">
  外资
 </a>
 撤离中国的消息，而大陆媒体则反而报导说，2018年上半年，中国成了吸引外资的第一大国。消息是真是假？其中有何猫腻？我们来看看专家的说法。
 <br/>
 <br/>
 10月17号，大陆第一财经网站报导称，根据联合国的一份报告，今年上半年，全球外国直接投资下降了41%，连美国也下降了73%，而中国吸收外国直接投资则增长6%，达700亿美元，成了全球吸引
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外资.html" target="_blank">
  外资
 </a>
 的第一大国。
 <br/>
 <br/>
 但文章没有说明这些外资来自哪些国家和地区。
 <br/>
 <br/>
 有学者在网上分析説，联合国的统计，只是统计资金流向，根本不问这些钱的具体持有者。中国外资当中，港资一直占70%左右，另有6-8%来自开曼等避税天堂，美日欧等国数据才是真料。
 <br/>
 <br/>
 大陆金融分析师任中道：〝很多中共的权贵阶层把资金转移到海外，它绕了一圈变成外资又进入到中国去赚钱获利。其实没有什么外资，都是假外资。〞
 <br/>
 <br/>
 有网友也指出:〝这些港资开资大都是中资。〞
 <br/>
 <br/>
 还有人质疑说，不是那些在外边被逼，卖了资产回流的钱吧？万达和海航在国外企业汇回的钱，按照中共的统计也都应该算在内的，就这两家汇回的钱也几百亿了。
 <br/>
 <br/>
 任中道：〝都是中国人民的钱，被他们转移出去，利用开曼群岛、维京群岛，或者所谓的正规的公司在欧洲，在哪儿注册之后，摇身一变变成外资了，然后再回到中国，再利用他们所掌握的信贷资源、政策资源再去获利，这是最隐蔽的一个中共对国家财富、百姓财富掠夺的一个方式。〞
 <br/>
 <br/>
 大陆金融分析师任中道认为，今年中国的外资突然多起来，是因为这些假外资看中了今年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共经济.html" target="_blank">
  中共经济
 </a>
 特别混乱的局面，回国混水摸鱼。
 <br/>
 <br/>
 任中道：〝这个股市在不断的下跌，有些公司的市值都已经腰斩了，有一些上市公司确实是有优质资产，有发展前途的，现在不是説要内外资同权嘛，现在已经有这个迹象了，这些假外资就进来掠夺这些优质的股权。
 <br/>
 <br/>
 今年美国电车公司特斯拉(Tesla Motors)在上海的投资是为数不多的外商投资之一。但中国金融智库研究员巩胜利披露，特斯拉在上海投资，地方政府有参与，资金来自大陆银行的贷款。
 <br/>
 <br/>
 中国金融智库研究员巩胜利：〝它是上海市政府的特批，这是一方面，它是美国控股的，中国政府享受了银行贷款而已，也就是説，如果特拉斯工厂造了一半，不想搞了要走人，中国政府要分担这个风险。〞
 <br/>
 <br/>
 中共商务部的统计数据显示，从2016年开始，香港就是中国最大的外资来源。
 <br/>
 <br/>
 巩胜利：〝香港的资本主要是中国的国有企业在香港收购，控股了以后再回到内地，基本上是这个情况。〞
 <br/>
 <br/>
 中国外资投资的实际情况是，近年来，随着国内人工成本增加，以及包括外资企业也要设立中共党支部等不利因素，已经有很多外企撤离，特别是今年美国开始对中国产品加征关税后，外企出口成本升高，外资撤离更多也更快。
 <br/>
 <br/>
 美国商会调查显示，美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 开打后，至少有143家在中国经商的美企，已经迁出或考虑迁出生产线。
 <br/>
 <br/>
 据〝彭博社〞报导，这些从中国迁出的外资，大多迁往了东南亚。目前东南亚国家的外国投资正盛。
 <br/>
 <br/>
 马来亚银行（Maybank）Kim Eng Research Pte数据显示，韩国制造业的外资流入量，在2018年前9个月增长约18％。
 <br/>
 <br/>
 泰国央行的数据也显示，泰国1至7月的净外国直接投资额，同比增长53％；菲律宾制造业的净外国直接投资，也在今年1至7月从1.44亿美元，飙升至8.61亿美元。
 <br/>
 <br/>
 专家们表示，这些数据显示，东南亚国家的外资流入增长率，远高于中国大陆。所以，中共自称外资流入第一大国，从这方面来说，也站不住脚。
 <br/>
 <br/>
