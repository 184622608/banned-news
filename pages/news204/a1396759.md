### 上海政法〝首虎〞陈旭被判无期 涉4人连环命案
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】上海检察院原检察长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈旭.html" target="_blank">
  陈旭
 </a>
 ，10月25日被以受贿罪判处无期徒刑。陈旭是上海政法系统的〝首虎〞，此前一名港商曾实名举报陈旭与一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_4人连环命案.html" target="_blank">
  4人连环命案
 </a>
 有关。
 <br/>
 <br/>
 根据广西南宁法院判决书显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈旭.html" target="_blank">
  陈旭
 </a>
 在2003年至2014年共11年间，任职上海政法委、上海市委及上海检察院期间，收受他人财物，共计折合人民币7423万元。
 <br/>
 <br/>
 法院称，陈旭被捕后主动交代办案机构未掌握的受贿事实，认罪悔罪、积极退赃，获从轻处罚，故作出上述判决。
 <br/>
 <br/>
 现年65岁的陈旭是上海人，仕途一直在上海政法系统，历任上海市高院副院长、上海市一中院院长、上海市政法委副书记、上海市检察院检察长等职务。
 <br/>
 <br/>
 2017年3月1日，中纪委宣布陈旭落马。而早在2016年4月，上海裕通房地产公司老板任骏良，就在网路上实名举报陈旭利用上海司法权力，为恶势力巧取豪夺，充当保护伞。
 <br/>
 <br/>
 陈旭被指涉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_4人连环命案.html" target="_blank">
  4人连环命案
 </a>
 :上海两名法官范培俊、潘玉鸣在接受北京反贪机构问话后，接受私人宴请时被害；上海商人王鑫明夫妇在接受调查后，也在家中被杀。
 <br/>
 <br/>
 2017年5月25日，陈旭被双开。中纪委称，陈旭违规干预和插手司法活动，以案谋私，长期搞迷信活动。今年7月5日，陈旭案开庭审理，但官方未提及陈旭涉连环命案事件。
 <br/>
 <br/>
 陈旭被指是江派〝上海帮〞的重要成员。上海维权律师郑恩宠此前向大纪元揭露，陈旭靠处理与江泽民之子江绵恒有关的周正毅案和整治郑恩宠，被提拔为市政法委副书记，成为江泽民外甥吴志明的副手。
 <br/>
 <br/>
 其后，陈旭又被时任上海市委书记陈良宇提拔为市委副秘书长，做上海第三号人物刘云耕的秘书。
 <br/>
 <br/>
 郑恩宠说，官场内检举陈旭的人非常多，上海中院一位潘姓副院长和前上海仲裁委员会主任汪康武，都检举过陈旭。
 <br/>
 <br/>
 有分析称，近期落马的上海官员几乎无一例外的〝拔出萝卜带出泥〞，背后都有江派重量级人物。种种信号表明上海〝政法王〞吴志明处境不妙。据多个消息源证实，陈旭被查后，上海政法系统内牵涉此案而被调查者已超过百人。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：赵云）
