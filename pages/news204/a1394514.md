### 十一长假 大陆多地金融难友维权被打压
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】大陆〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十一.html" target="_blank">
  十一
 </a>
 〞长假，上海、合肥、杭州、武汉等地出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融难友.html" target="_blank">
  金融难友
 </a>
 集体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 活动，但都遭到打压。难友们表示，很多人把钱都投入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_P2P网贷.html" target="_blank">
  P2P网贷
 </a>
 平台，生活发生困难，政府不作为，维权还被打、被抓，一些人因而想不开自杀身亡。
 <br/>
 <br/>
 目前大陆金融难民数量，内部统计有二千多万，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十一.html" target="_blank">
  十一
 </a>
 〞长假的第一天，10月1号，上海南京路有数百
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融难友.html" target="_blank">
  金融难友
 </a>
 集体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 ；合肥淮河路步行街五百多人集会游行；3号杭州数十名银豆网难友维权； 武汉市政府前每天都有人维权，5号江汉路步行街约三百多人维权。
 <br/>
 <br/>
 难友们表示，他们维权都遭到打压，很多人遭到警方控制，维权代表被抓，有人被警察殴打，之前的维权行动，有难友被关了一个多月，至今未释放。
 <br/>
 <br/>
 北京P2P难友江女士:〝当初建立平台，政府允许了之后才能营业的啊，连锁反映一样，一家出问题就很多家连着出问题，很多人都说不作为嘛，也报了警了呀，交了材料的，反正都没消息，然后维权的人也要被抓，到处都有被抓的，导致很多受难者想不开。〞
 <br/>
 <br/>
 难友们还说，很多人把所有的积蓄都投入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_P2P网贷.html" target="_blank">
  P2P网贷
 </a>
 平台，现在血本无归，生活发生困难，维权还被打、被抓，有人因而想不开，自杀身亡。
 <br/>
 <br/>
 温州P2P难友杨女士:〝政府也不为我们主持公道，去维权还欺压百姓，抓人，去了也没有用，你看政府也不让你维权，自杀的很多，我知道的有七、八个了，我们所有生活费都在上面，血汗钱都在上面。〞
 <br/>
 <br/>
 武汉市民透露，市政府门前每天都有人去维权，除了金融难友外，还有不少拆迁户。
 <br/>
 <br/>
 武汉市民彭女士:〝每天都有维权的，市政府门口都有啊，哪里都有啊，北京上访的话，关那个黑监狱，现在那个黑监狱真的是很厉害知道吧，打人不得了。〞
 <br/>
 <br/>
 今年7月，大陆P2P网贷平台出现倒闭潮，〝十一〞放假前有人在网上号召〝各路访民和P2P受害者大联动〞，在7天假期里，每天下午到全国10多个省会城市示威。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
</div>
