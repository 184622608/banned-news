### 美记者揭露 中共在新疆滥用监控科技
------------------------

<div class="wysiwyg">
 【新唐人2018年11月10日讯】国际人权盛会〝奥斯陆自由论坛〞周六(10日)在台北登场，美国网路新闻记者李香梅，因揭露
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 侵犯人权等〝敏感议题〞，被中共拒发签证被迫离境，李香梅出席论坛表示，中共滥用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监控.html" target="_blank">
  监控
 </a>
 科技，以脸部辨识指纹辨识等技术，监控新疆人民。
 <br/>
 <br/>
 《BuzzFeed》美国驻中国记者李香梅：〝这是非常清楚的，没有强大可靠的个人隐私保护，这已成定局，这些科技一定会被滥用。〞
 <br/>
 <br/>
 出席奥斯陆自由论坛，揭露出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 的维安打压，维吾尔人被拘留在教育营。[半透图卡]她是美国记者李香梅，派驻北京长达六年的时间，今年中共拒发签证导致她被迫离境，其实李香梅今年才获得人权新闻奖肯定，她的报导揭露中共在新疆以脸部、指纹识别系统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监控.html" target="_blank">
  监控
 </a>
 当地人民。
 <br/>
 <br/>
 《BuzzFeed》美国驻中国记者李香梅：〝当我们过去常常谈论监控技术滥用的情况时，它在今天的中国，始终是一个假设，但是在新疆所发生的一切，展示了一个反乌托邦的未来，就在我们的身边。〞
 <br/>
 <br/>
 而另一位讲者池成镐，是一名脱北者，成长于北韩90年代艰困的饥荒时代，他说上中学的时候，就有很多同学因饥饿而死亡。
 <br/>
 <br/>
 脱北者池成镐：〝在1995年，4月12号我的奶奶也是因为饥饿而死亡，但我什么事情都不能做，只能看着她离开。过去70年，金氏家族不断欺骗北韩人民，学校老师对我们说谎，说北韩是世界主义，是全世界最好，但我的朋友不断的死掉。〞
 <br/>
 <br/>
 因为长期营养不良，池成镐有一次昏迷掉落到铁轨，被火车辗过失去了左手左脚，不愿继续待在这个国家，他与父亲相约分头逃离北韩再会合，拄着拐杖走了一万公里，才逃离到南韩，但父亲却不幸去世了。
 <br/>
 <br/>
 脱北者 池成镐：〝我爸爸在中国的边界也是要逃离北韩，被秘密警察抓到，不断被施以酷刑，后来因为伤势过重而去世，我没能见到爸爸最后一面。〞
 <br/>
 <br/>
 池成镐说，拐杖象征追求自由的历程，这个拐杖也是他的父亲亲手做给他，是最后的一件遗物，尽管历经艰辛，但未来他会尽一切可能，追求北韩的自由。
 <br/>
 <br/>
 新唐人亚太电视林嘉韦、张媛婷台湾台北采访报导
</div>
