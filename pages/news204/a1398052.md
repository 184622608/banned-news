### 马列教内讧 南大马克思学会围堵校党委遇暴力清场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】内忧外患之下，中共内部斗争也在不断白热化和复杂化。因响应北京号召兴起的高校马列学会，转眼间又成为打击的对象。日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南京大学.html" target="_blank">
  南京大学
 </a>
 马克思研究会学生因无法完成注册围堵校党委书记，被便衣人员暴力清场。
 <br/>
 <br/>
 据自由亚洲电台报导，11月1日，有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南京大学.html" target="_blank">
  南京大学
 </a>
 学生在网上发布消息说，由学生自己组建的〝马克思主义阅读研究会〞早在50天之前就申请注册，但一直受到该校哲学系和团委的推诿。同时，研究会的学生还遭到便衣和其他学生的随身监视。
 <br/>
 <br/>
 从10月29日起，南大马会的学生发起抗议，在学校行政楼前等待，要求面见学校党委书记胡金波。
 <br/>
 <br/>
 但11月1日，参加抗议的几个学生，在行政楼前遭到不明身份的人暴力拦阻和殴打，横幅和传单被撕毁，一名学生被拖走，后又被警察扭送南京仙林派出所。
 <br/>
 <br/>
 今年的深圳佳士工人维权运动，得到中国左派团体的声援和支持，其中包括中国大学里的左翼学生。中共官媒将声援佳士的团体定义为〝海外反华势力〞，中共官方随后开始打压。
 <br/>
 <br/>
 早在今年9月，北京大学学生社团组织〝马列主义学会〞也已注册失败。
 <br/>
 <br/>
 中共为维护自己的统治，一直打击人间正义，否定普世价值。但是中国社会矛盾空前激化，一些关心民间疾苦的青年，开始在所谓〝原教旨的马克思主义〞中寻找出路，希望再次通过〝革命〞实现〝社会公平〞。
 <br/>
 <br/>
 不过，所谓〝马克思主义〞本身就不是〝解药〞，而是一剂烈性毒药。和当初被中共欺骗的〝革命青年〞一样，今天这些年轻学生，也可能再次成为一些中共政客们玩弄政治阴谋的工具。
 <br/>
 <br/>
 据自由亚洲此前引述知情人透露，佳士工人运动本就是毛派人士混入佳士工厂挑动起来的。
 <br/>
 <br/>
 当年薄熙来在重庆〝唱红打黑〞，曾得到中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毛左.html" target="_blank">
  毛左
 </a>
 们的拥护。薄熙来倒台后，仍有毛左为其喊冤。薄熙来是江泽民派系当年选定替换习近平的人选。
 <br/>
 <br/>
 中国高校的左派社团，本在校团委领导之下。而社团组织包括共青团，都划归中共〝群团工作〞范畴，此前归属江派前常委刘云山领导。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
