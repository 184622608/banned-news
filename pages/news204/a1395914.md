 【新唐人2018年10月18日讯】1. 美宣布退出万国邮联 中国电商将受影响
 <br/>
 <br/>
 美国白宫10月17号宣布，即日起启动退出万国邮政联盟（Universal Postal Union）的程序。白宫表示，万国邮政联盟的现行费率制度让中国和其他国家受益，对美国不公平。
 <br/>
 <br/>
 在现行的国际邮政体系下，从中国寄送小包裹到美国，比从美国寄送到中国，运费便宜很多。这削弱了美国商家的优势，同时让廉价的中国产品充斥美国市场。
 <br/>
 <br/>
 根据万国邮联的规定，退出程序为期一年。白宫表示，接下来的一年时间，若美国对国际邮联的邮费谈判不成功，美国将从2020年1月1号实施自己的终端收费标准。
 <br/>
 <br/>
 外界认为，这是川普政府再度出招制衡中共，惩罚其不公平贸易政策。而随着寄美包裹价格上涨，中国的电商将受到直接影响。
