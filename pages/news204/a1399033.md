### 否认放弃 中共官媒披露制造2025经费分配
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】中共官媒日前对外公开了中共政府今年制定的国家重点研发计划的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经费分配方案.html" target="_blank">
  经费分配方案
 </a>
 。该方案显示，中国大陆今年有429家单位共获人民币127亿元（人民币，下同）国拨经费资助，其中高科技领域约占40亿元。舆论认为，这表明中共仍在以政府大规模提供资金援助的方式来落实〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞计划。
 <br/>
 <br/>
 <h4>
  中共重点研发计划的
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经费分配方案.html" target="_blank">
   经费分配方案
  </a>
  曝光
 </h4>
 <br/>
 中共官媒新华社旗下的《经济参考报》11日报导称，据中国科技部提供的资讯，截至目前，中国今年国家重点研发计划共计立项640项，总计127亿元国拨经费的资助范围，涉及社会发展、高新技术、农林科技以及基础研究4大板块的38个专项。
 <br/>
 <br/>
 其中，社会发展和高新技术两大板块，分别投入50亿元和40亿元，占年度总经费的40%和30%，占总经费的7成；而应对公共突发事件的〝公共安全风险防控与应急技术装备专项〞，本年度共获资助55个项目、经费12亿8660万元，是目前已公布的所有专项中的〝最大赢家〞；此外，新能源汽车项目亦受到当局的〝青睐〞，获得10亿元经费资助，主要投入电池和电机研究。
 <br/>
 <br/>
 该报导披露，上述127亿元国拨经费的分配对象主要是科研院所、高等院校和科技公司，其各自所占的比重约为4：4：2。其中科研机构和高等院校以社会发展和基础研究为主，而接受资助的科技公司则偏重于重大高新技术的研发。
 <br/>
 <br/>
 外界注意到，上述获得中共政府资金援助的项目，明显集中于政府直接领导的科研机构或学府、国企，而民营企业所占份额就十分有限。
 <br/>
 <br/>
 例如，中国科学院系统共获380余项、90亿元经费；北京清华大学是大学中的翘楚，3年共获80余项、18亿元经费；从事轨道交通研究的中国中车股份有限公司3年共获11项、15亿元的〝大单〞。
 <br/>
 <br/>
 中国国家重点研发计划是由原来的中国国家重点基础研究发展计划（973计划）、国家高技术研究发展计划（863计划）和国家科技计划等整合而成，从2015年开始实施。
 <br/>
 <br/>
 中共政府制定的上述重点研发计划的经费分配方案公开后，观察家认为，尽管北京当局近期淡化放低了对〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞计划的宣传，但前述科技研发经费的分配方案显示，中共并未放松对〝中国制造2025〞的落实。
 <br/>
 <br/>
 <h4>
  〝中国制造2025〞因何成为发达国家的众矢之的？
 </h4>
 <br/>
 据公开的资讯，〝中国制造2025〞是习近平执政后，由中共国务院总理于2015年针对中国经济十年发展战略提出，即计划用十年时间，让中国摆脱对外国高科技依赖，成为〝引领世界高科技产业〞的强国。
 <br/>
 <br/>
 通常情况下，一个国家要升级自己的产业，他国政府并不会加以干涉。但中共实施这一经济发展战略的方式，却主要是依靠国家资本主义的手段，以政府提供大规模补贴的手段让国企在与外国企业的竞争中获得不公平的所谓〝优势〞；同时中共政府又以中国市场为诱饵，配合不公正的国家政策，强迫在中国经营的外资企业转移科学技术，侵犯了以美国为首的发达国家的智慧财产权，因而〝中国制造2025〞成为了发达国家的〝众矢之的〞，同时也成为美中两国政府贸易谈判中最棘手的障碍。
 <br/>
 <br/>
 稍早前，美国政府在宣布对中国进口产品加征关税时，就曾直言不讳地声明，被加征关税的很多中国产品都与〝中国制造2025〞有关。
 <br/>
 <br/>
 <h4>
  舆论：中共不可能真正放弃〝中国制造2025〞
 </h4>
 <br/>
 经过几个月的关税大战的较量后，明显落于下风的中共政府开始有意在宣传方面淡化〝中国制造2025〞的影响力。在与中国国家主席习近平通过电话后，美国总统川普（特朗普）曾经于本月7日在一次记者会上比较乐观地表示，
 <br/>
 <br/>
 《日经亚洲评论》（Nikkei Asian Review）报导，川普7日在记者会上表示，北京当局或将舍弃〝中国制造2025〞计划。
 <br/>
 <br/>
 〝我发现它非常具威胁性〞川普说：〝这代表他们（中国） 将于2025年在全球经济称霸。我说，这不会发生〞。
 <br/>
 <br/>
 不过，外界舆论却认为北京不可能真正放弃〝中国制造2025〞计划，更可能是做出某些政策调整〝忽悠〞川普政府，而中共官媒刚刚披露的中共政府重点研发计划经费分配方案，则被认为从侧面证实了北京仍在致力于推行其〝中国制造2025〞计划的问题。
 <br/>
 <br/>
 《自由亚洲电台》日前曾引述美国智库外交关系协会（CFR）高级研究员、亚洲研究部主任易明（Elizabeth C.Economy）表示，实际上中国在开放市场和保护智慧财产权方面没什么重大进展。基于这些原因，美国商人、官员和分析人士得出的结论是，他们不再相信中国朝着更加改革开放的方向前进。
 <br/>
 <br/>
 美国华盛顿智库战略暨国际研究中心（CSIS）高级研究员芮恩希（William Reinsch）也明确表示，目前并没有证据表明，中共已经放弃了中国制造2025计划，相信他们在未来也不会放弃。
 <br/>
 <br/>
 芮恩希表示，虽然中共政府要求媒体不再提〝中国制造2025〞这个词，但这与放弃该政策截然不同。习近平在很多讲话中仍在强调自力更生、自主创新，显示〝中国制造2025仍是他的重要政策〞。
 <br/>
 <br/>
 他说，中国希望到2025年成为全球行业领导者并不是不可以，但如果该目标是透过大规模政府补贴等不公正方式来完成，就违反了世界贸易组织（WTO）规则，其他国家就有权对此进行报复。
 <br/>
 <br/>
 海外时政分析人士唐靖远表示，川普所说的北京〝放弃〞，可能是指习近平允诺不再以偷盗方式来发展中国科技，而与此同时习近平也在公开的讲话中强调〝自力更生〞，二者信息是一致的。问题的关键并不是世界不允许中国发展科技，而是反对中共以政府补贴、强制技术转让，甚至动用经济间谍盗窃发达国家的关键科技等不当手段来图谋成为世界科技的〝霸主〞。
 <br/>
 <br/>
 （记者唐迪综合报导/责任编辑：明轩）
