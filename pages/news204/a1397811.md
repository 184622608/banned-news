### 公交车坠入长江 只因乘客与司机打架
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】大陆官媒周五报导，日前重庆一辆公交车从桥上坠入长江，原因是司机在行驶期间和一名乘客打架，导致车辆失控。包括司机和这名乘客在内的15人无一生还。
 <br/>
 <br/>
 周日，重庆万州区一辆公交车，从60多米高的长江二桥坠入江中，沉入80米深的江底，车上15人全部遇难。
 <br/>
 <br/>
 中共官媒周五公布公交车内部监控视频，当时一名女乘客与司机发生激烈争执。女乘客多次攻击司机，司机右手离开方向盘反击，导致车辆向左偏离车道。司机想矫正方向盘已来不及。车撞破护栏，冲进长江。
 <br/>
 <br/>
 附近车辆摄像头拍下的视频显示，公交车进入对面车道后，撞上一辆正常行驶的小轿车。该车司机受轻伤。
 <br/>
 <br/>
 周三，公交车被打捞出水。
 <br/>
 <br/>
 有知情大陆网友解释，该乘客因错过下车地点和司机争执，而其他乘客没有制止，冲突升级，最终他们也都陪葬了。
 <br/>
 <br/>
 新唐人记者王羽音、张琪综合报导
