### 【禁闻】德媒：中共间谍疑瞄准德国支柱产业
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 政府单位曾多次警告：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 正积极渗透社交网络和政府部门等。近期，又有德国媒体指出，中共间谍疑似染指德国的支柱产业。
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国.html" target="_blank">
  德国
 </a>
 之声》11月4号报导，德国经济最发达地区之一－－西南部的巴符州，近年来越来越频繁的成为中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济间谍.html" target="_blank">
  经济间谍
 </a>
 的目标，当地的经济支柱－－
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汽车产业.html" target="_blank">
  汽车产业
 </a>
 ，更是重灾区。
 <br/>
 <br/>
 巴符州内政部在回复州议会质询时指出，2012年至今，州刑警局总共发现8起经济间谍案件，其中至少6起不能排除中国方面参与作案的可能性。内政部表示，还有很多企业发现网络安全漏洞时不一定会报案。
 <br/>
 <br/>
 而受到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络间谍.html" target="_blank">
  网络间谍
 </a>
 案波及的，不仅是当地的中型企业，一家坐落于斯图加特的汽车制造商，也受到了境外网络攻击。
 <br/>
 <br/>
 报导指出，奔驰、保时捷的总厂，都位于斯图加特近郊。
 <br/>
 <br/>
 澳大利亚华裔张小刚博士表示，中共长期以来所谓的〝国策〞之一，就是以政府力量盗取先进国家的商业、经济以及技术机密。
 <br/>
 <br/>
 澳大利亚华裔张小刚博士：〝德国在政府层面开始意识到这个问题，开始防范，这属于亡羊补牢吧！一些血淋淋的教训，才让西方国家意识到中共长期盗用技术，而它不会关心你本地老百姓的需要。它完全是为了它们来进行所谓的‘厉害了我的国’（的宣传）。〞
 <br/>
 <br/>
 根据德国官方说法，网络经济间谍行为每年在巴符州造成70亿欧元的潜在经济损失。全德国每年因经济间谍行为蒙受的直接经济损失，高达500亿欧元。
 <br/>
 <br/>
 大陆IT技术员陈达威：〝（中共）间谍始终都有的。中国制造那么多商品，你物美价廉怎么来啊？完全靠专利权全部买下来，根本就没办法做。再加上现在这种独裁的环境之下，国内就培养不出人才。所以你说发明的话，基本上就没戏。〞
 <br/>
 <br/>
 大陆IT技术员陈达威举例，目前中国很多手机企业，如果不是靠窃取知识产权，根本难以生存。
 <br/>
 <br/>
 陈达威：〝比如三星的手机可能卖3000（元），我（窃取技术后）就1000（元）左右的成本生产出来，卖1500（元）我就能赚钱了。但是你把专利权（费用）全部付掉，你可能成本都跟三星一样，甚至更高，因为你手中没有核心技术，全部都要买专利权嘛！（中国）间谍这个土壤，好像没有解药啊，现在。〞
 <br/>
 <br/>
 在此之前，德国针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 渗透，曾多次发出警报。
 <br/>
 <br/>
 去年12月，德国联邦宪法保护局警告，中共间谍利用社交网站广招线人，上万名德国公民成为目标。
 <br/>
 <br/>
 虽然北京坚称德方指控〝毫无根据〞，但未能消除德国各界担忧。
 <br/>
 <br/>
 《南德日报》（Sueddeutsche Zeitung）今年7月披露，中共间谍渗透德国国会，一位德国议员差点落入圈套。中共国安部在社交网站〝领英（LinkedIn）〞，至少有500个假帐号，是专门用来接触德国各界人士的。
 <br/>
 <br/>
 德国联邦情报局（BND）估计，中共的情报人员超过100万人。
 <br/>
 <br/>
 今年2月，德国两大研究所发布联合调研报告，警告中共在欧洲媒体攻势火力大增，一些德国知名报刊也无法独善其身。
 <br/>
 <br/>
 随后，《南德意志报》宣布停止刊发中共官媒提供的《中国观察》增页。
 <br/>
 <br/>
 4月初，德国联邦宪法保护局局长马森（Hans-Georg Maaßen）对新的经济间谍现象提出警告说：中资收购德企，可能会危害国家安全。
 <br/>
 <br/>
 中资家用电器制造商〝美的集团〞（Midea Group）2016年以46亿欧元收购德国机器人公司〝库卡〞，被视为典型例子。
 <br/>
 <br/>
 张小刚：〝在（大陆）国内这一两年看到一些报导非常得意洋洋的说，他们已经拿到德国的技术，开始赶上和超过德国。包括汽车行业、机器人行业、机械制造行业，德国人做的非常严谨、先进，一直属于它们的支柱产业，现在中共这种盗取技术方式已经开始掏空德国的这些支柱产业。〞
 <br/>
 <br/>
 不仅如此，德国对来自中国的设备也提高警惕。
 <br/>
 <br/>
 德国《经济周刊》今年3月报导，〝华为〞等中国电信设备制造商在德国市场具有绝对优势地位，一旦它们的硬件设施中留有〝后门〞，中共间谍就可以很方便地窃取德企数据。针对该报导，巴符州议会自民党党团已向州政府发起质询。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/陈建铭
