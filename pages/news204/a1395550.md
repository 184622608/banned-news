### 港媒：中共四中全会月内召开  将习近平定于一尊
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】香港媒体称，中共的很多重要会议都选在10月份召开，目前，中共正在秘密
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_筹备.html" target="_blank">
  筹备
 </a>
 第十九届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 ，最快将于20日召开。今次四中全会的主题将聚焦进一步的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 ，以及将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_定于一尊.html" target="_blank">
  定于一尊
 </a>
 。
 <br/>
 <br/>
 按中共惯例，历届中央委员会在任期内都会召开七次全体会议，每次会议履行的职能均不同，第一次全会进行党的换届；第二次全会进行政府换届；第三次全会布局未来五年的工作。第四次全会的主题历届并不尽相同，一般是根据当前阶段相对重要的问题而确定会议主题。
 <br/>
 <br/>
 中共第十八届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 是2014年10月20日到10月23日召开。
 <br/>
 <br/>
 香港明报16日报导说，第十九届四中全会将于10月份在北京召开。由于今年是中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 40周年，又恰逢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 处于胶着状态，故今次四中全会的主题将聚焦进一步的改革开放。
 <br/>
 <br/>
 《韩联社》15日引述北京消息人士透露，此次四中全会将聚焦有关中共整体运营方向的基本设计，确定中共的经济政策，以及讨论应对中美贸易摩擦的计划。
 <br/>
 <br/>
 该消息人士还表示，中央全会一般会在开始前的一个月或者两个月公布，但并不一定全都遵循这样的规则，全会的开会日期是在中央政治局扩大会议上决定的，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 可以随时召开扩大会议。
 <br/>
 <br/>
 另一消息人士透露，日本首相安倍晋三，将于10月25日访问中国，考虑到国内外的日程，预计四中全会，会在安倍晋三访华前举行。
 <br/>
 <br/>
 《苹果日报》引述报导说，四中全会最快于10月20日召开，目前中共高层正在秘密
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_筹备.html" target="_blank">
  筹备
 </a>
 中。
 <br/>
 <br/>
 报导说，中共四中全会将涉及中美贸易战、政治、经济和人事等议题。自去年中共十九大结束后，一年来，中共政治和经济〝左倾〞明显，但对于习近平的个人崇拜遭遇挫折，此次会议也将是对习的效忠和歌颂的大会，将习
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_定于一尊.html" target="_blank">
  定于一尊
 </a>
 。
 <br/>
 <br/>
 〝定于一尊〞出自《史记.秦始皇本纪》，是指思想、学术、道德等以一个最有权威的人做唯一的标准。北戴河会议前，〝捧习运动〞降温，舆论热炒习近平失势之时，栗战书等习家军掌控的要害部门纷纷站队表态，要求确保习中央〝一锤定音、定于一尊〞的权威。
 <br/>
 <br/>
 近期，随着美中贸易战火的日渐升高，一度降温〝捧习运动〞再次恢复热度，有中共党媒高层说，强化领袖意识维护习核心，是特殊时期保持政治稳定的必要手段。
 <br/>
 <br/>
 另外，由于中共反腐败，拿下大大小小官员，中共官员普遍消极怠工。表面〝支持、拥护〞中央政策，实则不作为看〝笑话〞，目前，整个中共官僚集团暮气沉沉，四中全会将对怠政官员进行严厉警告。
 <br/>
 <br/>
 消息人士还称，中共十九大前，习近平完成了最高层的换届。十九大后，省部级的调整仍在继续。四中全会除了对已经〝出事〞省部级官员的处理公示外，近期，中组部还将对更多省部级官员做出职务调整。
 <br/>
 <br/>
 报导称，外交上，近期美国总统和副总统猛烈抨击中共，中美迈入新冷战时代。军事上，中美舰艇在南海几乎擦枪走火，台海形势趋于紧张。
 <br/>
 <br/>
 经济上，货币超发、地方债务居高不下、P2P等影子银行暴雷频繁、股市、汇市下挫，外资纷纷撤离，特别是〝国进民退〞民营企业离场的传闻更使民营企业忧心忡忡。
 <br/>
 <br/>
 因上述种种原因，习近平将在四中全会上，藉改革开放40周年的时机，就中美贸易战、民营企业市场定位及中共的经济、外交、军事形势进行阐述，强调改革开放，以鼓舞和稳定民心将是四中全会的重心。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：范铭）
