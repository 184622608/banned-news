### 政法委未就孟宏伟案表态 港媒:或涉习曾两大系权争
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 前公安部副部长、国际刑警组织前主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 突然落马，目前事件仍在发酵中。虽然国际媒体不断追踪报导，但中共政法委却至今未有表态。
 <br/>
 <br/>
 自10月5日法国电视台《Europe 1》率先爆出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 在中国失踪后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 中纪委和国家监察委7日被迫发表通报承认，失踪的孟宏伟在中国因〝涉嫌违法〞，正在接受监委监察调查。
 <br/>
 <br/>
 8日凌晨，中共公安部召开党组紧急会议，通报孟宏伟因收受贿赂、涉嫌违法接受国家监委监察调查情况，并表态称当局对孟宏伟的调查〝非常及时、完全正确，十分英明〞。
 <br/>
 <br/>
 不过，事件被踢爆至今一周时间，中共政法委却始终沉默。外界关注孟的落马，是否会引发中共政法系高官面临被再次清洗。
 <br/>
 <br/>
 香港媒体《明报》分析指出，从中共官方对孟宏伟的通报中提到，〝坚决彻底肃清
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 流毒影响〞，可能在暗示他涉及了周永康的政法系统旧案。
 <br/>
 <br/>
 根据公开资讯显示，2004年孟宏伟升任公安部副部长时，正是周永康任公安部长期间。如果孟宏伟是作为周永康遗毒被抓，《明报》指出，那么他近些年的上司孟建柱和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭声琨.html" target="_blank">
  郭声琨
 </a>
 ，会不会也将受到牵连呢？
 <br/>
 <br/>
 报导说，之前的中共公安部一直被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江派.html" target="_blank">
  江派
 </a>
 人马把持，并不〝姓习〞。公安部名义上属国务院，但实际归中共政法委管辖，因此公安部长一般都是政法委书记的继任人，周永康、孟建柱、郭声琨都是按此路线升迁。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 拿下周永康后，虽在政法系统清理了一批周的余党，但对孟建柱、郭声琨的人仍基本未动。而孟、郭亦被指是江派二号人物曾庆红的嫡系。据称，郭声琨还是曾庆红的表外甥女婿，郭声琨妻子的祖母是曾庆红的母亲邓六金的亲妹妹。
 <br/>
 <br/>
 另有消息透露，孟宏伟的仕途升迁，与现任中共政法委书记郭声琨也有着千丝万缕的联系——两人算是湖南中南工业大学校友，孟宏伟2013年以中共公安部副部长身份兼任中共海警局首任局长，得以跻身正部级，当时的中共公安部长正是郭声琨。
 <br/>
 <br/>
 因此，郭声琨为脱干系也应快速对孟宏伟被调查一事做出反应。可事实上，他辖下的中共中央政法委一直失声，令外界颇为奇怪。
 <br/>
 <br/>
 此外，中共司法部10日召开党组会议时，由党组书记袁曙宏主持，而身为司法部长的傅政华并未出席，而傅也是具有江派背景的官员。
 <br/>
 <br/>
 根据此前的报导，傅政华原是在北京公安局搞刑侦出身的，2010年2月官至北京公安局局长、党委书记。北京一直是江派的势力范围，傅政华也隶属江派。2012年重庆原公安局局长王立军事件后，周永康发动的〝3・19〞政变未遂，与傅政华倒戈投向胡锦涛、习近平阵营有关。
 <br/>
 <br/>
 随后，傅政华被提拔为公安部副部长、常务副部长，但在2015年，由习近平在福建时的旧部王小洪接任其北京公安局局长之职。京畿重地职务被夺，说明曾经背靠江派的傅政华并未获得习近平的真正信任。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：）
