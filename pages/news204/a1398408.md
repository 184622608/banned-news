### 黄琦病危 14家国际人权组织呼吁释放
------------------------

<div class="wysiwyg">
 【新唐人2018年11月07日讯】14家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际人权组织.html" target="_blank">
  国际人权组织
 </a>
 最近发表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合声明.html" target="_blank">
  联合声明
 </a>
 ，指身陷囹圄的〝六四天网〞创办人黄琦健康状况严重恶化，已直接危及生命，要求中共政府立即
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无条件释放.html" target="_blank">
  无条件释放
 </a>
 黄琦。黄琦两年前第三次被捕，案件至今还没有开审。
 <br/>
 <br/>
 11月5号，包括记者无国界、国际特赦、自由之家在内的14家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际人权组织.html" target="_blank">
  国际人权组织
 </a>
 发表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合声明.html" target="_blank">
  联合声明
 </a>
 。声明说，据黄琦的辩护律师10月23号探视后的消息显示，黄琦在被羁押期间没有得到充分的医疗救治，其健康状况严重恶化，已经直接危及生命。
 <br/>
 <br/>
 声明指出，黄琦身患肾脏疾病、脑积水、心脏病和其他疾病，且在被羁押期间受到各种形式的虐待和其他非人道的对待，而身患重病的良心犯不能获得救治也是一种酷刑折磨。近年来由于缺乏及时医疗，包括曹顺利等多位人权捍卫者在拘留期间死亡，加剧了国际社会对黄琦可能遭受同样命运的担忧。
 <br/>
 <br/>
 声明强调，中共当局必须立即
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无条件释放.html" target="_blank">
  无条件释放
 </a>
 因为和平行使言论自由而被拘留的黄琦，并结束拒绝为良心犯进行及时治疗的政策。
 <br/>
 <br/>
 黄琦85岁的母亲蒲文清表示，黄琦病情非常严重，她多次要求四川当局给予治疗，但都没有结果，在这种情况下，只能向国际人权组织求助，希望死前能见黄琦一面。
 <br/>
 <br/>
 黄琦母亲蒲文清：〝肾功能衰竭相当严重，黄琦已经进入尿毒症，当局不治疗，黄琦很可能会死在狱中，我要求国际人权组织敦促中国（中共）当局从人道出发，释放无罪的、患绝症的黄琦出来治病，希望我能见他一面，我死也瞑目。〞
 <br/>
 <br/>
 国际特赦中国研究员潘嘉伟：〝我们跟其他组织一起呼吁中国（中共）政府给他足够的治疗，而且要释放他，因为以所谓洩露国家秘密，还有向其他人提供国家秘密这样的一些控罪，从来都是口袋罪，根本都没有实质的证据。〞
 <br/>
 <br/>
 国际特赦中国研究员潘嘉伟表示，昨天是联合国对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人权.html" target="_blank">
  中国人权
 </a>
 状况的普遍定期审议，他很高兴美国有提到黄琦的个案，他希望有更多的国际关注，帮助改善黄琦和中国其他维权人士的情况。
 <br/>
 <br/>
 新唐人记者陈汉、陈洁、柏妮采访报导
</div>
