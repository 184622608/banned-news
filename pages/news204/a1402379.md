### 孟晚舟听证会洩其罪证：亲手在伊朗开设过桥公司
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 CFO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 12月1日在加拿大被捕，今日（7日）举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保释听证会.html" target="_blank">
  保释听证会
 </a>
 。官方在听证会上透露，孟被抓原因是当年任华为香港公司总经理时，亲自主导开设〝过桥〞公司与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 违规交易，欺骗美国。
 <br/>
 <br/>
 美东时间7日上午10点14分，推特帐号〝LIFETIME视界〞发布消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_保释听证会.html" target="_blank">
  保释听证会
 </a>
 已经开始，吸引了大批记者。
 <br/>
 <br/>
 听证会上，官方透露了孟晚舟被抓的直接原因，是当年直接主导与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 的违规交易。
 <br/>
 <br/>
 2009年至2014年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 涉嫌利用一家名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Skycom.html" target="_blank">
  Skycom
 </a>
 〞的公司在伊朗开展业务，违反美欧对伊制裁。SkyCom虽然设在伊朗，但总部位于香港。
 <br/>
 <br/>
 美司法部查证华为就是Skycom，以此控告华为欺诈。
 <br/>
 <br/>
 孟晚舟当时担任华为香港公司的总经理，是这起欺诈业务的主要执行者。孟被指担任Skycom的财务长，涉嫌误导美国银行从事违反对伊制裁的业务。
 <br/>
 <br/>
 在听证会上，美国司法部律师指证，Skycom其实就是华为开设的〝过桥〞公司，用来掩盖华为与伊朗的违规交易。
 <br/>
 <br/>
 路透社2012年曾经详细报导了这起事件。报导指，在伊朗Skycom公司的员工，自称是〝华为-Skycom联合体〞员工。
 <br/>
 <br/>
 听证会透露的信息显示，孟晚舟主导犯罪证据确凿，并非只是象此前媒体猜测的，只是华为与伊朗违规交易的财务系统参与者。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
