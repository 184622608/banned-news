### 云南80后官员头发全白形如老翁 疑年龄造假
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】中共云南楚雄州委组织部近日拟提名为大姚县政协副主席候选人的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李忠凯.html" target="_blank">
  李忠凯
 </a>
 ，生于1980年，才38岁却一头白发，形如老人而火爆网路，令不少网民质疑他为了晋升而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_年龄造假.html" target="_blank">
  年龄造假
 </a>
 。
 <br/>
 <br/>
 陆媒16日报导，中共云南省楚雄州委组织部，15日发布一批州管官员任期公示引发社会关注。其中，包括出生于1980年8月、现任大姚县湾碧乡党委书记的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李忠凯.html" target="_blank">
  李忠凯
 </a>
 ，拟任提名为中共大姚县政协副主席候选人。
 <br/>
 <br/>
 公示称，李忠凯大学学历，1999年10月参加工作。历任中共大姚县湾碧乡人大主席等职。2018年10月，被中共楚雄州委评为楚雄州优秀基层官员。
 <br/>
 <br/>
 但任前公示配发的照片显示，李忠凯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_头发已白.html" target="_blank">
  头发已白
 </a>
 ，看起来形如老人，众多网友对公示年龄质疑。
 <br/>
 <br/>
 中共楚雄州委组织部一名工作人员证实，照片上的确是李忠凯，〝照片是他本人，年龄也对得上〞。〝我们与大姚县委组织部做了沟通，他们再次确认，李忠凯的简历信息与照片都是准确的。〞
 <br/>
 <br/>
 照片和年龄长相为什么差距很大？该工作人员解释，大姚县委组织部对其表示，李忠凯所在的乡镇工作量很大，特别是移民搬迁工作，涉及到移民搬迁等工作，是任务最重的一个乡镇。〝所以工作几年后，他就大变样了，人的长相就判若两人。〞
 <br/>
 <br/>
 这名工作人员还说，今年10月，楚雄州表彰了10名担当作为的官员，李忠凯是其中之一。
 <br/>
 <br/>
 尽管相关人员做出解释，但网民仍提出诸多质疑：〝19岁大学毕业参加工作？〞〝工作量很大，涉及移民搬迁，天天下基层，这位80后官员还能吃的白白胖胖的？〞〝严重质疑这个白发苍苍的80后干部背后有靠山，为晋升而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_年龄造假.html" target="_blank">
  年龄造假
 </a>
 。〞
 <br/>
 <br/>
 而网民的质疑似乎也并非空穴来风，因在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共官场.html" target="_blank">
  中共官场
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_造假骗官.html" target="_blank">
  造假骗官
 </a>
 者比比皆是。中共官员从年龄到学历，乃至工作经验，无不作假，有的人甚至除性别之外，全是造假。
 <br/>
 <br/>
 如原共青团石家庄市委副书记王亚丽，人称〝骗官书记〞，除了性别是真的，其身份、年龄、履历、档案均为造假。之所以造假得手，档案被发现问题能不了了之，是因其有靠山，王亚丽被指〝从床上培养起来的女干部〞。
 <br/>
 <br/>
 再如，中共司法部〝首虎〞、中共前司法部政治部主任卢恩光，被发现入党材料造假、年龄造假、学历造假、工作经历造假、家庭情况造假等。其一边造假一边行贿，由一名私营企业主竟然官至副部级。
 <br/>
 <br/>
 中共自2002年规定任党政领导职务的干部，要求大学专科以上学历，厅局级以上的要求大学本科以上学历，之后，不少官员不再满足于〝修改年龄〞、〝工作经验注水〞，而是致力于打造学历、职称造假。
 <br/>
 <br/>
 如中共深圳市前副市长许宗衡，接受调查后被发现其美国国际东西方大学工商管理硕士，是个〝野鸡大学〞文凭。
 <br/>
 <br/>
 中共广东中山市原市长李启红落马后。媒体曝光，只有小学文化的她，却拥有中共广东省委党校的研究生文凭。
 <br/>
 <br/>
 人民日报自曝，在12轮中央巡视中，抽查2.5万多份中共干部档案，499名存在档案造假问题的官员被处理。截至2017年，仅河北省衡水一地，就有97人不符合任职资格；69人〝三龄两历一身份〞存疑、档案涂改涉嫌造假；33人被举报暂缓提拔。
 <br/>
 <br/>
 中共官场伪造档案最知名的案例，当数中共前党魁江泽民伪造〝两假〞档案、掩盖〝两奸〞历史。
 <br/>
 <br/>
 中国著名学者吕加平曾刊文揭露江泽民和他的亲生父亲都是日伪汉奸，但江冒充1949年前，加入中共党员，并冒充是中共所谓〝烈士〞江上青养子的〝烈士〞子弟。
 <br/>
 <br/>
 学者表示，中共严控档案，它可以〝以假乱真〞，也可以〝以真乱假〞。中共所做的一切都是为了极权政权。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明）
