### 台联电停止合作 晋华或步中兴后尘瘫痪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】日前，中国福建科企晋华集团遭美国签发禁售令，被指〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴第二.html" target="_blank">
  中兴第二
 </a>
 〞。随后，晋华的主要合作商台湾联华电子宣布暂停合作。中共高科技企业严重依赖外界组件和技术。晋华遭遇此番打击，可能很难象中兴一样〝起死回生〞。
 <br/>
 <br/>
 据《美国之音》报导，台湾芯片制造商联华电子公司周三（10月31日）发布声明称，将暂停跟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 的研发合作，直到有关当局获准恢复合作。
 <br/>
 <br/>
 联华电子官员对路透社表示，美国下达了制裁令，联华也会遵守美国政府的规定。
 <br/>
 <br/>
 两天前，美国对福建晋华颁布禁售令，切断了美国企业对福建晋华的产品和技术供应。该公司有中共政府背景，并涉嫌偷盗美国半导体公司美光公司的知识产权。
 <br/>
 <br/>
 福建晋华主要制造在电脑和手机中应用的DRAM存储芯片，其一期芯片工程已在今年9月正式投产。
 <br/>
 <br/>
 美光公司指控，福建晋华和台湾联华电子窃取其芯片设计。不过，福建晋华和台湾联华电子随后在中国反诉美光并胜诉，导致美光一些产品在中国遭禁售。
 <br/>
 <br/>
 在福建晋华之前，中国中兴通讯公司曾遭到美国禁售令处罚，很快陷入瘫痪。北京随后向川普总统求情。在支付巨额罚金后，中兴获美国商务部〝放生〞。
 <br/>
 <br/>
 目前中美对抗升级，已经走向新冷战。如果福建晋华再陷入困境，美国网开一面的可能性大减。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
