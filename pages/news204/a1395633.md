### 【禁闻】调查中共活摘器官法庭在英国成立
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】
 <b>
  调查中共活摘器官
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法庭.html" target="_blank">
   法庭
  </a>
  在英国成立
 </b>
 <br/>
 <br/>
 10月16号，一个专门负责调查中共强制摘除良心犯器官的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法庭.html" target="_blank">
  法庭
 </a>
 ，在英国伦敦宣布成立。
 <br/>
 <br/>
 据〝大纪元新闻网〞报导，这个法庭名为〝独立人民法庭〞，调查目标是：任何涉嫌中共或国家批准的机构、组织在强行摘取器官方面的刑事犯罪。
 <br/>
 <br/>
 法庭包括来自英美的人权律师、医学专家等八名独立成员，由英国御用大律师尼斯爵士主持。
 <br/>
 <br/>
 法庭将于12月在伦敦举行第一次公开听证会，预计在三天的听证会期间，会收到30名证人和专家提供的证据。
 <br/>
 <br/>
 <b>
  多名中共
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退休常委.html" target="_blank">
   退休常委
  </a>
  齐现身 引发关注
 </b>
 <br/>
 <br/>
 中共退休高官朱鎔基、李瑞环、吴官正等，最近分别以不同方式出现在公众视野，引发议论。
 <br/>
 <br/>
 据报导，最先现身的是中共前常委吴邦国，他在10月7号，带着浩浩荡荡的车队出现在安徽肥东县。
 <br/>
 <br/>
 然后是10月11号，中共前总理朱鎔基现身北京清华大学经济管理学院顾问委员会会议。
 <br/>
 <br/>
 紧接着是10月15号，大陆网络媒体刊登中共前常委李瑞环的文章，聚焦他的为官之道和对中共政坛的看法。
 <br/>
 <br/>
 海外媒体报导说，在当下中共面临贸易战沉重打击、经济下行、外资撤退、股市暴跌、内交外困之际，这些中共前高官密集露面，不知是时间巧合，还是别有用意。
 <br/>
 <br/>
 <b>
  明慧网：〝法轮大法好〞在铁岭广泛传播
 </b>
 <br/>
 <br/>
 中共对法轮功的迫害还在持续，中国大陆的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 不屈不饶的反迫害活动也没有停止。
 <br/>
 <br/>
 据〝法轮大法明慧网〞报导，在中国东北辽宁省的铁岭市，人们上街就有机会遇到法轮功学员告诉他们中共迫害修炼人的真相，抬头就能看到〝法轮大法好〞、〝三退保平安〞等真相标语。
 <br/>
 <br/>
 当地法轮功学员发出的图片显示，这些写着〝法轮大法好〞、〝真善忍好〞的标语，被张贴在公共汽车站、闹市中心、居民小区等场所。
 <br/>
 <br/>
 报导说，当地很多市民都对法轮功非常了解，一提起法轮功就会说，〝法轮功可不得了，听说美国人都知道〞、〝三退保平安，可灵了！〞
 <br/>
 <br/>
 编辑/周玉林
