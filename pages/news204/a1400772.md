### 人造月亮后又有天河工程 中国科技界再现浮夸风？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】日前清华大学联合启动的〝腾云调雨〞的所谓〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天河工程.html" target="_blank">
  天河工程
 </a>
 〞，被指〝荒谬幻想工程〞。此前，中共官方宣布将发射
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人造月亮.html" target="_blank">
  人造月亮
 </a>
 ，也备受质疑。在北京重提〝自力更生〞的背景下，中国科技界被指又开始流行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_浮夸风.html" target="_blank">
  浮夸风
 </a>
 〞。
 <br/>
 <br/>
 近日，陆媒一篇《气象学家实名批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天河工程.html" target="_blank">
  天河工程
 </a>
 不顾质疑仓促上马》的文章引起广泛关注。文章引述多名气象学家对该工程的质疑，称其为〝既没有科学基础也没有技术可行性的荒诞幻想项目〞。
 <br/>
 <br/>
 据报导，天河工程于2015年8月由清华大学牵头发起，青海大学、青海科技厅参与，获青海省政府每年资助600万元（人民币，下同）经费。2016年申请中共科技部〝科技创新国际化环境〞类项目。去年11月启动，获经费1,002万元。今年3月与中国航天集团签战略合作协议，先后在北京、深圳等地召开研讨会。
 <br/>
 <br/>
 工程目前已耗资超过6,000万元。官媒宣称，正在进行相关卫星和火箭的研制，预计在2020年和2022年分批发射升空。
 <br/>
 <br/>
 该项目目标是将位于青藏高原上空的水汽，以人工方式截留并调运到中国西北干旱地区，然后形成降雨。陆媒引述中科院院士分析指，该工程主要由水利专家主导，并没有听取气象专家和相关科学家的意见就仓促上马。
 <br/>
 <br/>
 项目相关单位面对媒体的询问，大多推诿回避。有参与的专家称，〝天河工程有自己的考量。〞
 <br/>
 <br/>
 外界质疑，官员们所谓的〝考量〞其实就是〝政治效应〞，以及工程背后的巨大利益。
 <br/>
 <br/>
 中共所谓〝南水北调〞工程，被指始于中共前党魁毛泽东1952年视察黄河时的讲话。工程分为东线、中线和西线三条调水线路，东线和中线已动工，西线尚在规划阶段。而天河工程，则被官宣传成〝空中版南水北调〞。
 <br/>
 <br/>
 上个月陆媒还披露，四川成都天府系统科学研究会正牵头研究，计划在2020年发射三颗国产卫星以制造〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人造月亮.html" target="_blank">
  人造月亮
 </a>
 〞，以取代城市的路灯。
 <br/>
 <br/>
 当时陆媒并没有质疑项目可行性，而是藉此宣传中共的〝科技强大〞。但是，海内外的华人网友纷纷吐槽，怀疑中共能否完成这个〝政治工程〞，以及〝人造月亮〞会带来的光污染，以及对动植物作息造成的负面影响，并质疑可能给人类带来生态灾难。
 <br/>
 <br/>
 在此之前，三峡工程被认为是近年来中共〝政治工程〞的典型代表，背后还涉及巨大的腐败问题。工程上马时遭遇中共体制内水利专家的广泛质疑。三峡大坝建成后，其造成的各种生态灾难不断被披露。有专家指，该工程的最后结局是被炸掉。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
