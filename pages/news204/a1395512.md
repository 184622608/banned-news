### 深圳女〝腰缠万贯〞闯海关 行走怪异被拦下
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】近日，深圳一名女子〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_腰缠万贯.html" target="_blank">
  腰缠万贯
 </a>
 〞闯海关，不料因走路姿势僵硬被拦截。经查，该女子在腰部、裆部绑藏数万欧元，甚至连鞋底也藏有1千欧元，共计3.6万欧元，超过海关规定携带的现币外钞数额。
 <br/>
 <br/>
 据陆媒报导，日前，深圳
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙头角海关.html" target="_blank">
  沙头角海关
 </a>
 查获一宗人身
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绑藏欧元.html" target="_blank">
  绑藏欧元
 </a>
 出境案件。从网传监控视频可见，一名身穿黑色外套的女子背着个斜挎包，低着头，小步快走，但走姿有些怪异，显得有些僵硬，像是脚里踩着异物。
 <br/>
 <br/>
 海关工作人员介绍当时的情形说，〝她进来的时候全程低着头，偶尔还会瞄一下我们关员，我们关员就注意到她了，看到她走路姿势有点奇怪，所以就将她拦下了。〞
 <br/>
 <br/>
 就在工作人员示意黑衣女子停下接受检查时，她不但没有停下，反而加快了脚步，想要逃离监管区域。工作人员随即追上去，把女子拦下来。
 <br/>
 <br/>
 经检查发现，这名女子的两只鞋子各藏有1千元欧元现钞。不仅如此，女子腰部、裆部都分别捆绑有欧元现钞，共计3.6万欧元，折合约29万元人民币。
 <br/>
 <br/>
 据海关相关规定，出境人员携带现币外钞额度是2万元人民币，或者是5千美元以内的（等值）外钞，超过部分必须登记，而且必须等值带出等值带回。
 <br/>
 <br/>
 目前，这名藏带现钞的黑衣女子已移交海关缉私部门处理。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：范铭)
