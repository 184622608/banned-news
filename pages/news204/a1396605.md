### 大桥开通〝冲喜〞失败 陆股重挫跌破2600大关
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 23日在习近平主礼下开通，24日早9时正式通车。但大桥开通并未给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陷入阴霾.html" target="_blank">
  陷入阴霾
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共经济.html" target="_blank">
  中共经济
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冲喜.html" target="_blank">
  冲喜
 </a>
 〞，中港两地股市当日跟随外围大跌，陆股更遭受重挫，跌破2600大关。
 <br/>
 <br/>
 10月23日上午10点左右，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 开通仪式在珠海口岸旅检大楼出境大厅举行，较原定时间迟了30分钟。广东省委书记李希、香港行政长官林郑月娥、澳门行政长官崔世安和国务院副总理韩正先后致辞。最后，习近平上台宣布，〝港珠澳大桥正式开通。〞
 <br/>
 <br/>
 全长55公里、耗资千亿人民币的港珠澳大桥，是世界上最长跨海大桥及隧道项目。中共很希望借港珠澳大桥，增加其国际影响力。但有消息人士指，大桥只是一个〝形象工程〞，实际作用不大。
 <br/>
 <br/>
 〝毫无价值，毫无意义的大桥，从太子去澳门都需140分钟，车费不便宜，100多港币，还没坐船舒服和快。但花了这么多钱去起这样一个桥，打肿脸充胖子而已。〞
 <br/>
 <br/>
 据中共官媒报导，开通仪式结束后，习近平等人乘车巡览港珠澳大桥。习在大桥东人工岛上会见大桥管理和施工等方面的代表，称港珠澳大桥是〝国家工程、国之重器〞，是〝自信桥、复兴桥〞。
 <br/>
 <br/>
 大桥的规划建造，是中共推进粤港澳大湾区建设的重要构件，在当前中美贸易战不断燃烧、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共经济.html" target="_blank">
  中共经济
 </a>
 前景又添阴霾时，大桥的开通被认为是给萎靡不振的中共经济〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冲喜.html" target="_blank">
  冲喜
 </a>
 〞，提振民众信心。
 <br/>
 <br/>
 然而，开通仪式当天的股市，却给了中共当头一盆冷水。港股23日收市报25346点，跌806点，跌幅达3.08%，回吐前两日升幅；陆股更跌破2600大关，沪综指跌2.26%，收报2594点。
 <br/>
 <br/>
 有专家称，股市持续下泄，不仅反映了投资者对股市没有信心，更反映了整个中国社会对未来的经济，正在失去信心。
 <br/>
 <br/>
 另外，中共视为崛起象征的港珠澳大桥，却被指破坏了香港〝风水〞。有台媒称，港珠澳大桥的兴建，是中共安排的风水阵，以破坏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港风水.html" target="_blank">
  香港风水
 </a>
 格局。香港和澳门对接部分，如同两条蛇对冲，意味着有口舌之争。
 <br/>
 <br/>
 此外，自港珠澳大桥主体建好之后，〝天鸽〞颱风就来了，之后吹袭香港的颱风不断。
 <br/>
 <br/>
 不仅如此，港珠澳大桥施工九年以来，爆出多个丑闻，包括香港口岸人工岛〝飘移〞，填海用钢圆筒急剧沉降，香港接线填海区海堤崩塌，以及石屎检测报告造假等。大桥施工期间共发生9次致命工业意外，至少18人丧生。工伤的数字超过600宗，被外界谑称为〝奈何桥〞。民间传说中，奈何桥是通向地狱的入口。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
