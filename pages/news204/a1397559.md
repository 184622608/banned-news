### 【禁闻】美国制裁福建晋华 评：震摄范围超过中兴
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】美国商务部10月29号以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_盗窃知识产权.html" target="_blank">
  盗窃知识产权
 </a>
 〞，涉及违反美国国家安全利益为由，将福建〝晋华集成电路有限公司〞加入《出口管理条例》的名单中。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 ，成为继
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴.html" target="_blank">
  中兴
 </a>
 通讯之后，第二家遭美国禁售制裁的大陆科技企业。有评论指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晋华案.html" target="_blank">
  晋华案
 </a>
 的震摄范围要超过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴案.html" target="_blank">
  中兴案
 </a>
 。
 <br/>
 <br/>
 2016年7月，中共党媒《人民网》高调宣告了福建省晋华存储器集成电路生产线项目即将开动。〝晋华集成电路〞与台湾的〝联华电子〞技术合作，主要生产DRAM（动态随机存储器集成电路）芯片。晋华项目是当局〝十三五规划〞和〝中国制造2025〞战略中的重点高科技企业。一期投资370亿元，其中国家专项建设基金支援30亿元。
 <br/>
 <br/>
 但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 在去年陷入知识产权纠纷后，今年10月29号受到了美国商务部的制裁。
 <br/>
 <br/>
 美国商务部的声明表示，福建晋华大量生产DRAM的能力即将完备。晋华能力的扩大，很可能是得到了美国的原创技术，而威胁到美国军事系统基本供应商的长期经济生存能力。
 <br/>
 <br/>
 商务部长罗斯在声明中说：〝当外国企业的活动抵触美国国家安全利益时，我们会采取强硬行动维护国家安全。〞
 <br/>
 <br/>
 这是继〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴.html" target="_blank">
  中兴
 </a>
 通讯〞之后，美国商务部第二次对大陆科技企业祭出禁售制裁。美国企业向福建晋华出口软件及技术产品将受到限制。
 <br/>
 <br/>
 旅美时事评论员唐靖远表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晋华案.html" target="_blank">
  晋华案
 </a>
 表面看和中兴类似，但其实性质不同。
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝中兴当初就是把一些敏感技术和敏感的设备，违反了美国的规定，卖给了伊朗、朝鲜等这些跟美国敌对的国家。这种行为它其实执行的是中共政权的职能，所以它只是针对的一些少数的企业。但是晋华这个案子有所不同，因为晋华受到处罚最关键的原因，是因为它盗窃了美方公司的知识产权。〞
 <br/>
 <br/>
 唐靖远表示，中共治下，大陆很多公司都存在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_盗窃知识产权.html" target="_blank">
  盗窃知识产权
 </a>
 的行为，因此晋华案的震摄范围更大。
 <br/>
 <br/>
 唐靖远：〝晋华这个公司的受罚，对整个中共的政治界和经济界所带来的冲击和震摄的意义，是要超过中兴那个案子。因为盗窃知识产权而受到处罚，这个的意义它就具有非常大的普遍性。〞
 <br/>
 <br/>
 和晋华有知识产权纠纷的企业，是美国存储芯片巨头〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美光科技.html" target="_blank">
  美光科技
 </a>
 〞。
 <br/>
 <br/>
 去年12月，美光在美国加州起诉联电与晋华，指控联电通过招揽美光台湾地区员工，窃取其知识产权和关键技术，并交给晋华。台湾当局也表示，这些工程师窃取了美光超过900多份技术文件。目前这一诉讼还在审理中。
 <br/>
 <br/>
 然而今年1月，晋华和联电反在中国福建起诉美光侵犯其知识产权，导致美光的26种芯片产品在中国遭到临时禁售。
 <br/>
 <br/>
 美光已经对此提出上诉。中国是美光科技的最大市场，约占美光销售额的半数以上。
 <br/>
 <br/>
 台湾企业家与社会运动人士高为邦：〝（美光）提出来的证据当然是罪证确凿的。但是你在中国是不可能得到公平的审判。对它们不利的，它们就是两个办法，一个就是拖时间。同时它还吿你，告你的它都赢。这个我们太清楚了，我们台商受害案件能赢已经不简单了，赢了也不给你执行。〞
 <br/>
 <br/>
 唐靖远表示，晋华案非常典型，阐释了川普为什么要打贸易战。
 <br/>
 <br/>
 唐靖远：〝它明明是先去窃取了别人的知识产权，它却是倒打一耙。晋华公司的这种行为事实上是利用政府权力，来夺取了美国公司的市场。这个其实就是川普在多次强烈的谴责中共政府的说辞里面提到的，就是典型的国家资本主义，也是一种非常典型的非关税贸易壁垒。〞
 <br/>
 <br/>
 唐靖远表示，晋华案和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中兴案.html" target="_blank">
  中兴案
 </a>
 最大的共同点是，揭露了中共所谓经济崛起的底牌，而晋华案是美国对中共不公平贸易手段的一次强烈反击，他相信还会有其它类似的案子出现。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/陈建铭
