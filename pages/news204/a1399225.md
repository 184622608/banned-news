### 改革开放40年展 6常委现身 王沪宁开幕发言不提邓小平
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】中共庆祝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 40周年大型展览13日举行开幕式，中共政治局常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王沪宁.html" target="_blank">
  王沪宁
 </a>
 致辞时，只字未提推动改革开放的中共前领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 。当天，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 等6名中共政治局常委，与中国国家副主席王岐山现身会展，习同样只字未提邓小平。
 <br/>
 <br/>
 11月14日，党媒报导说，〝庆祝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 40周年〞的大型展览，13日在北京天安门广场旁的中国国家博物馆举行开幕式，中共中央政治局常委、中央书记处书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王沪宁.html" target="_blank">
  王沪宁
 </a>
 出席并致辞。
 <br/>
 <br/>
 王沪宁在讲话中说，自中共十八大以来，以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_核心.html" target="_blank">
  核心
 </a>
 的中央推进全面深化改革，为中共的改革开放翻开了新的篇章。
 <br/>
 <br/>
 王沪宁还说，要坚持以习近平思想为指导，增强〝四个意识〞，坚定〝四个自信〞，把改革开放不断推向深入。
 <br/>
 <br/>
 展览中展示了40年来，特别是2012年中共18大以来，中国人民的生产生活发生的变迁。
 <br/>
 <br/>
 报导说，当天除出访新加坡的李克强外，习近平等六名中共政治局常委与中国国家副主席王岐山等高层，现身北京国家博物馆参观〝改革开放40周年大型展览〞。
 <br/>
 <br/>
 习近平在讲话中称，全面深化改革、扩大对外开放要〝坚定不移〞。
 <br/>
 <br/>
 但根据中共党媒报导内容，习近平和王沪宁均只字未提推动中共40年来改革开放的前领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 。
 <br/>
 <br/>
 习近平10月下旬，南下视察被称为改革前沿的广东期间，整个行程中，同样只字未提到邓小平；而人民日报10月30日，针对改革开放发表评论文章，也罕见的只字未提邓小平。
 <br/>
 <br/>
 对于习近平当局不提邓小平的做法，有观点认为，这是当局提升总书记习近平党史地位的行动，且在相当程度上要超越邓小平。
 <br/>
 <br/>
 美国哈佛大学研究学者葛维茨（Julian Gewirtz）表示，习近平〝肯定不满足于在邓小平的阴影之下运作〞，且希望建立一个特殊的政治体系，〝由他自己居于中心〞。
 <br/>
 <br/>
 而在习近平〝南巡〞期间，邓小平长子邓朴方，盛赞邓理论的一篇演讲稿，在中国网路上热传。
 <br/>
 <br/>
 邓朴方在演讲中，除了热捧邓小平〝改革开放的伟大革命〞外，还宣称〝我们要坚持长期处于社会主义初级阶段这个基本判断〞，并不点名地提醒，〝我们一定要有这种实事求是的态度，保持清醒的头脑，知道自己的份量〞。
 <br/>
 <br/>
 有传媒分析说，在外界质疑习近平是否还继续坚持邓小平提出的改革开放，以及是否重新评价邓的改革开放〝功绩〞之际，邓朴方的讲话，显示中共内部对坚持〝改革开放〞立场的分歧。
 <br/>
 <br/>
 时政评论夏小强说，对于中共高层和内部来讲，分歧一直在路上，分歧的实质其实也就是争夺权力的斗争，而中共内部的所有的理论、路线、方针、政策，都是用来攻击政敌的工具和手段。
 <br/>
 <br/>
 而习近平这些年来，改变邓小平所谓〝韬光养晦〞政策的做法，及采取的一些经济改制措施，也未必就是要回到文革的闭关自守时代，可能也包括针对中共内部清洗的因素。
 <br/>
 <br/>
 但在中共体制下，特别是在中美贸易战冲击下的危局下，习近平接下的是江泽民时代造成的从经济到政治及道德层面，全面危机爆发前的炸药包，在中共面临内忧外患走向绝路的局面下，无论习左转右转，都难以阻止中共这艘沉船的急速下沉。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明)
 <br/>
