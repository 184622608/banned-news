### 中共政经界激烈交锋 红二代直批北京政策
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】中国的经济危局加剧了中共内部分裂。近日一次经济研讨活动中，多名政治经济界头面人物公开批评北京的经济政策，其中包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红二代.html" target="_blank">
  红二代
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦晓.html" target="_blank">
  秦晓
 </a>
 。
 <br/>
 <br/>
 北京时间11月13日至14日在北京举办的《财经》年会，聚焦2019的经济战略。一些金融、学界人士和中共官员与会，其中包括中科院党组前副书记秦力生之子、中国招商银行前董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦晓.html" target="_blank">
  秦晓
 </a>
 。
 <br/>
 <br/>
 秦晓在会上坦承国企与民企之争的癥结。他认为，国企存在的本质就是政府直接介入市场竞争，而不是一个中性的调控者或者监管者。在这种情况下，重要政策的决定很难〝完全到位〞，包括对国企和民企无法〝完全一视同仁〞。
 <br/>
 <br/>
 他还表示，刘董事长（与会的新希望集团有限公司董事长刘永好）做得风生水起，但在民企中只是凤毛麟角。民企在经营中的交易费用高，融资的溢价成本高，在与国企的竞争中处于劣势。
 <br/>
 <br/>
 近期以来，北京最高层表态要〝做大国企〞，同时又宣称民企是〝自己人〞，对国企和民企〝一视同仁〞。秦晓的言论显然和北京当局不同调。
 <br/>
 <br/>
 另外，北京大学中国经济研究中心主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姚洋.html" target="_blank">
  姚洋
 </a>
 指出了北京此前〝去槓杆〞中的两个问题，一个是〝降槓杆太猛〞，等于把病人强行按到床上动手术，但不给输血；另一个是〝用宏观经济政策来代替改革〞，所谓〝改革〞是空喊口号无实质行动。
 <br/>
 <br/>
 姚洋表示，国企资金和技术最雄厚，但经营机制不善，应该融入民资控股，改善管理机制，把企业的管理交给社会来做，不再纠缠〝姓资姓社〞的问题。
 <br/>
 <br/>
 而目前北京当局正在企业内部设立党委，以及推进〝党领导下的民主管理〞，以加强对国企和民企的控制权。同时，地方国资则在不断收购民企股权。
 <br/>
 <br/>
 另外，姚洋还批评了中共银保监主席郭树清日前要求银行对民企贷款不低于1/3的新政策：没有银行愿意贷款给风险很高的中小型民企，如果强迫银行以低利率贷款给民企，不足以覆盖民企的风险，银行就会想别的办法再把钱要回来。与其这样，还不如让银行有风险定价的权力。
 <br/>
 <br/>
 当前，在中国经济下行压力下，中美贸易战更加重了经济危局，由此引发中共执政危机。生存的压力引发中共内部公开的激烈争论。这其中还掺杂着复杂的权力斗争因素。
 <br/>
 <br/>
 在今年9月北京钓鱼台国宾馆举行的〝50人论坛学术研讨会〞上，与会专家们大胆犀利地集体炮轰北京的经济政策。
 <br/>
 <br/>
 据外媒报导，论坛的发起人、中共副总理刘鹤没有发言，在会议开始不久就藉故离场。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
