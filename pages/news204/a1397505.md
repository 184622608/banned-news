### 美媒：金庸去世登微博热搜榜首 川普也成当日明星
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武侠小说.html" target="_blank">
  武侠小说
 </a>
 家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金庸.html" target="_blank">
  金庸
 </a>
 10月30日在香港养和医院去世的消息，当日登上了大陆微博
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_热搜榜.html" target="_blank">
  热搜榜
 </a>
 榜首，据《福布斯》报导称，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）同样是当日中国社交媒体的〝最大外国明星〞。
 <br/>
 <br/>
 10月30日，美国《福布斯》刊发题为《惊讶：很多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
  中国人
 </a>
 其实
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_喜欢.html" target="_blank">
  喜欢
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：并希望他的〝良好协定〞》报导说，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 是中国社交媒体新浪微博的最大外国明星。
 <br/>
 <br/>
 报导称，中国人理解川普，理解他的民族主义烙印。中国人喜欢川普在媒体上对中共直白的评论。就连中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武侠小说.html" target="_blank">
  武侠小说
 </a>
 家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金庸.html" target="_blank">
  金庸
 </a>
 去世当日（10月30日），川普仍是当日第15大热门话题。
 <br/>
 <br/>
 而川普在贸易对中共的抨击和戏谑，也让他成了中国这个庞大的社交网络上分享次数最多的非中国故事。
 <br/>
 <br/>
 早在川普宣布竞选总统之时，CNBC财经网站就曾报导，川普不按牌理出牌的言论，引起中国13亿人口的注目，让平时不关心美国大选的中国网民，透过追踪川普的新闻而了解美国总统选举的民主程序及其价值。
 <br/>
 <br/>
 报导说，尽管川普在竞选活动中总是批评中国的不公平贸易行为，包括不合理的关税及操控汇率，威胁要对自中国进口的产品课征45%的关税等，但这些指责似乎没有引起中国人的愤怒，反而被其政治不正确的言论所吸引，有些人还对川普甚为着迷，为他在网络上架起川普粉丝团，追随人数越来越多。
 <br/>
 <br/>
 特别是进入2018年以来，川普政府将中共公开列为竞争对手，致力于改善美中两国的长期不公平贸易上，双方展开激烈交锋；川普指责朝鲜、伊朗等流氓政权的威胁，其深层本质又都与中共密切有关，促使川普将攻击火力与炮口，全面转向，瞄准中共。
 <br/>
 <br/>
 今年初川普的首度《国情咨文》便已明确宣告，中共是美国的挑战对手。
 <br/>
 <br/>
 他说，〝在全世界，我们面对着流氓政权、恐怖主义团体以及像中共、俄罗斯这样的对手，挑战着我们的利益、我们的经济与我们的价值观。〞
 <br/>
 <br/>
 而在方兴未艾的美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，川普频频挥出重拳，从绥靖退让转向强硬对抗，把中共当局敲得七荤八素，让全球社会瞩目震惊，中国众多网民却为之兴奋，期盼川普再出更重的重拳打垮中共，重建一个民主与法制的中国。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：戴明)
