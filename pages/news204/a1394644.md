### 〝小三〞染艾滋 怒曝平安集团高管身份
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】近日，网传大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平安集团.html" target="_blank">
  平安集团
 </a>
 高管巨款包养情妇并传染爱滋病毒给情妇，随后网上流传平安集团要求员工进行体检的消息。但事件当事人以及平安集团均对相关传闻予以否认。
 <br/>
 <br/>
 10月4日，微博用户〝爱橙子的sabrina〞，微博实名举报大陆平安证券副总经理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱益勇.html" target="_blank">
  朱益勇
 </a>
 ，恶意传播爱滋病给自己，事后拒不承担责任。随后网上流传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平安集团.html" target="_blank">
  平安集团
 </a>
 要求员工进行包括爱滋病项目的体检。
 <br/>

 <br/>
 苏玲娜在微博中称，今年5月起，自己持续发烧，7月通过HIV试纸检测呈阳性，疑似感染HIV病毒；同时苏玲娜还称朱益勇HIV也呈阳性，并控诉是朱益勇传染给自己HIV病毒。
 <br/>
 <br/>
 苏玲娜微博发布的其与〝朱益勇〞的微信聊天记录显示，朱益勇自知自己的病无药可愈，在得知两人感染HIV之后，言称苏玲娜称可以服用进口药；间接承认自己患爱滋病。
 <br/>
 <br/>
 上述的微博还显示，苏玲娜疑似向朱益勇索要200万元不果，表示要让朱益勇的〝老婆孩子和同事知道，去公司闹〞。
 <br/>
 <br/>
 平安集团回应员工体检表示，集团9月份就启动了整个集团的员工体检，并没有特别提及某项疾病。同时，当事人朱益勇在微信朋友圈回应事件表示，此系网路谣言，自己已报案。
 <br/>
 <br/>
 据公开资料显示，朱益勇是平安证券总经理助理。另外，在职业社交网站LinkedIn上显示，朱益勇自1999年起一直在平安集团从事人力资源方面的工作，现在还担任平安证券人力资源部经理和总监的职务。
 <br/>
