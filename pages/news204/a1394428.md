### 浙江宁波男持刀砍人 劫车乱撞2死16伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】中国浙江省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宁波.html" target="_blank">
  宁波
 </a>
 市一名42岁男子疑因个人纠纷，6日晚间突然当街拿出预藏刀械，刺伤路过女子后，劫持一辆轿车逃逸，沿途见人就撞，其中1名路人当场被撞死，1人送医抢救不治身亡，加上被刺女子另有16人受伤。
 <br/>
 <br/>
 警方于事发后晚间6时28分接获报案称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宁波.html" target="_blank">
  宁波
 </a>
 市北仑区街道发生一起持刀伤人案件。
 <br/>
 <br/>
 据目击者说，嫌犯在西子山路持刀捅伤一名女子后，拦下一辆路过的轿车，逃离现场，沿途见人就撞，撞伤数人后逃离现场。后来轿车漏油起火，又抢了旁边停下来的教练车，一路撞下去，到了海口村……
 <br/>
 <br/>
 也有当地网友反映，6点30就有人报警，但警方没下达通报，过了30分钟后，网友才从微博知道，23点半抓到人，〝如果采取措施不可能伤那么多人〞。〝据说在咸祥（镇）被抓了，在那儿也砍了五个人……〞。
 <br/>
 <br/>
 警方晚间6点28分接获报案，循线于晚间11点15分逮捕王姓嫌犯。
 <br/>
 <br/>
 据大陆媒体报导，犯罪嫌疑人王某（男，42岁，北仑人）于晚间11时15分被逮捕。警方在逮捕时，王嫌曾一度与警方发生肢体冲突，双方都有挂彩。
 <br/>
 <br/>
