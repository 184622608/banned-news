### 三代总理齐〝露面〞 独缺李鹏引猜测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】近日，武侠小说家金庸（原名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_查良镛.html" target="_blank">
  查良镛
 </a>
 ）去世，中共三代总理包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_温家宝.html" target="_blank">
  温家宝
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 全部〝露面〞哀悼金庸，独缺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李鹏.html" target="_blank">
  李鹏
 </a>
 。引外界猜测。之前，有消息称，李鹏与江泽民的健康状况频亮 〝红灯〞，甚至遭遇多次病危险情。
 <br/>
 <br/>
 11月2日晚，北京中央驻香港联络办公室发布公布称，中联办主任王志民，前往武侠小说家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_查良镛.html" target="_blank">
  查良镛
 </a>
 家中，向其夫人林乐怡转达了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 等中共领导人，对查良镛去世的哀悼和对亲属的慰问。
 <br/>
 <br/>
 中联办还称，查良镛去世后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 、韩正、孙春兰、黄坤明、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_温家宝.html" target="_blank">
  温家宝
 </a>
 、张德江、李岚清、刘延东、李源潮、王汉斌、廖晖等也以不同方式表示哀悼，并向其亲属表示慰问。
 <br/>
 <br/>
 10月31日晚，由查良镛生前创办的明河社出版有限公司，代表查良镛家人发布讣告：查良镛于2018年10月30日下午在家人陪伴之下，于香港养和医院安详逝世。
 <br/>
 <br/>
 讣告还说，根据查良镛生前意愿，丧礼以私人形式举行。兹定于11月12日至30日在香港文化博物馆的〝金庸馆〞设置吊唁册，以供公众向查良镛作最后致意。
 <br/>
 <br/>
 查良镛出生于1924年3月10日，浙江海宁人。1940年代后期移居香港，后以笔名〝金庸〞书写多部脍炙人口的武侠小说。1959年查良镛创办了《明报》，与中共针锋相对，亲自撰写社评批评中共当局。
 <br/>
 <br/>
 但在文化大革命结束，邓小平重新上台后，查良镛由亲台倒向亲中共。之后，开始参政，查良镛于1985年出任香港基本法起草委员会成员，1999年3月，被浙江大学聘为教授，兼任浙大人文学院院长。2009年，又获任中共作协名誉副主席。
 <br/>
 <br/>
 虽然查良镛早年与中共大打笔战，但立场反覆在政治道路上多次进进出出，晚年一直跟中共保持非常良好关系，直到离世。
 <br/>
 <br/>
 此次中共领导人纷纷悼念查良镛，可见中共对查良镛的重视。不过外界也发现，中共三代总理朱鎔基、温家宝、李克强全部露面悼念金庸，但独缺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李鹏.html" target="_blank">
  李鹏
 </a>
 引外界猜测。
 <br/>
 <br/>
 之前，有消息称，李鹏与江泽民的健康状况频亮 〝红灯〞，甚至遭遇多次病危险情。
 <br/>
 <br/>
 9月30日，中共国务院在北京大会堂举行十一招待会。有美媒爆料，此次招待会上，没有中共元老的镜头，江泽民、李鹏、朱鎔基、胡锦涛、温家宝等人〝集体缺席〞。
 <br/>
 <br/>
 中南海的十一招待会，已经成为观察中共政坛动向的窗口。
 <br/>
 <br/>
 有中南海知情者对美媒爆料说：由于江泽民和李鹏年岁已高，患有老年人常见的疾病，健康频频亮起〝红灯〞，甚至遭遇多次病危险情。
 <br/>
 <br/>
 中南海知情者表示：中共元老们在卸任后一旦突然出现，都会被外界解读为释放出某种政治信号；因为高度敏感，元老们的健康状况，宛如国家最高机密，就更是焦点中的焦点，令人讳莫如深。
 <br/>
 <br/>
 据港媒报导，就中共前总理李鹏的健康问题，早在2008年，海外媒体曾传出两个版本：一是说李鹏于2008年2月5日中风，送医院抢救已脱离生命危险，但留下了后遗症。
 <br/>
 <br/>
 另一说法是李鹏在2008年新年脑血栓发作，经抢救病情已基本稳定。但对于这两种版本，李鹏之女李小琳一概否认。
 <br/>
 <br/>
 李鹏生于1928年10月，现今90岁。李鹏之子李小鹏于8月30日辞去山西省长，随后出任交通运输部部长、党组副书记。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：赵云）
