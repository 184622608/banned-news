### 北京频发诱人信号  纳瓦罗：谁敢相信中共？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】即将举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20川习会.html" target="_blank">
  G20川习会
 </a>
 晤，成为国际舆论关注的焦点。美媒称，北京希望借会晤之机终止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，近来频频发出诱人信号，美中之间可能会达成某种协议。但白宫贸易顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 直言，问题在于：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谁敢相信中共.html" target="_blank">
  谁敢相信中共
 </a>
 可能遵守协议？
 <br/>
 <br/>
 《华尔街日报》11月12日刊文称，在美中两国元首会晤前，北京一直在向美国发出诱人的信号，习近平日前在上海进博会上表示，预计未来15年，中国进口的服务将超过10万亿美元。如果得以实现，美企很有机会获得其中大量份额。
 <br/>
 <br/>
 与此同时，中共总理李克强与六家多边机构负责人举行会面，中方承诺将支持自由贸易。
 <br/>
 <br/>
 外界盛传，北京期盼终止美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，可能会在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20川习会.html" target="_blank">
  G20川习会
 </a>
 谈中，做出重大妥协与让步，美中双方很可能达成某种协议。但许多川普政府官员对中共以往兑现此类承诺的情况深感怀疑，这其中包括莱特希泽、白宫贸易顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 (Peter Navarro)以及防务部门官员。
 <br/>
 <br/>
 纳瓦罗日前警告说：〝即使美国与中国（共）在贸易方面达成协议，也无法保证中国（共）能够真正执行。〞
 <br/>
 <br/>
 他指出：〝我们同加拿大和墨西哥重新谈判（达成新的协议），我们很信任他们会遵守协议。我们正在同日本和欧洲谈判。我们同世界上我们能信任的国家谈判。但是中国（共）却与众不同。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谁敢相信中共.html" target="_blank">
  谁敢相信中共
 </a>
 可能遵守协议？纳瓦罗还举例说：〝中国（共）加入世贸组织时有诸多承诺，但16年过渡期之后，（这些承诺）却都没有兑现。〞
 <br/>
 <br/>
 纳瓦罗的担心和警告绝非多虑，自由亚洲的评论文章指，言而无信、失信背信是中共臭名昭着的本性。早在2015年，中共就曾向美国做出两项承诺：不会在南海搞军事化；不会对美国公司搞网路窃密活动，双方还签署了协议。
 <br/>
 <br/>
 然而，三年过去了，中共在南海大搞军事化，肆无忌惮；而中共大量盗窃美国公司机密，也愈益猖獗。
 <br/>
 <br/>
 文章指，中共之所以能够长期占美国便宜，就在于它毫无底线的厚黑学与超限战。随时毁约失信而后百般诡辩，只是其中之一。
 <br/>
 <br/>
 事实上，中共并非只是对外毁约失信，对内亦然。早期中共高喊〝抗日〞口号，背地里却与日本军方勾结，破坏抗日，共同瓦解中华民国。
 <br/>
 <br/>
 国共内战时，中共高喊〝民主〞、〝多党〞口号，夺取政权后却厉行一党专政，疯狂独裁；建政后，中共高喊〝为人民服务〞，实际上却屠杀人民、压迫人民、剥夺人民基本权利，官僚阶层贪污腐败、巧取豪夺、大肆敛财……
 <br/>
 <br/>
 文章最后指，苏共党魁列宁曾说过〝外交是内政的延伸。〞中共践行了这句话。中共对内背信弃义，必然带来对外背信弃义，世界需要认清中共的本质。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
