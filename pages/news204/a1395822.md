### 陆专家：非洲猪瘟肆虐大半中国 当局严禁洩密
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情现已肆虐大半个中国，并有进一步失控的迹象。有大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陆防疫专家.html" target="_blank">
  陆防疫专家
 </a>
 披露，情况比想像的严重！北京当局已无力阻止非洲猪瘟疫情扩散，而掌握疫情变化的实验室和疫控人员，已被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_当局严管.html" target="_blank">
  当局严管
 </a>
 ，禁止洩密。
 <br/>
 <br/>
 10月17日，中共农业农村部新闻办公室通报，山西省大同市首次爆发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 ，涉事的养猪场存栏生猪15只，当中发病7只、死亡4只。
 <br/>
 <br/>
 自8月1日爆发首宗非洲猪瘟疫情，至今已有2个半月。然而，疫情仍持续在全国范围蔓延。迄今为止，官方通报的疫情已达29起，并有进一步失控的迹象。
 <br/>
 <br/>
 12日，中共卫生部启动应对非洲猪瘟疫情的一级应急预案。不过，令人费解的是，唯一被官方允许发布非洲猪瘟疫情的中共农业农村部官网，近日突然将疫情讯息撤出了首页。
 <br/>
 <br/>
 官方解释说，正在对网站内容和功能进行调整。但外界质疑，非洲猪瘟疫情资讯是否已受到官方管控。
 <br/>
 <br/>
 参与防控工作的非洲猪瘟专家徐女士向《自由亚洲》证实，最能掌握非洲猪瘟疫情变化的实验室和疫控人员，已被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_当局严管.html" target="_blank">
  当局严管
 </a>
 。
 <br/>
 <br/>
 徐女士介绍，按照当局的规定，任何涉疫情的资讯发布，直接归农业农村部，其他机构和实验室都不能对外说话，否则就是泄密。
 <br/>
 <br/>
 她还透露，目前，无论是管理能力或是生物技术掌握，北京当局都无力阻止非洲猪瘟疫情扩散。徐女士坦言，情况比想像的严重，即便是业内人士，也不清楚疫情会发展到何种程度。
 <br/>
 <br/>
 早在一个月前，已有业内防控人士发出警告称，从非洲猪瘟疫情爆发日开始，到其后的一个多月时间里，是疫情防控的关键时期，如果不能成功切断传染链条，后果不堪设想。
 <br/>
 <br/>
 此外，徐女士认为，目前想控制疫情很困难。因为实验室的能力有限，加之政府担心试验过程中散毒，也禁止他们进行疫苗有关的研究。
 <br/>
 <br/>
 目前，非洲猪瘟疫情正在向全国蔓延。而官方却多次发布消息称，猪肉可以放心吃，只要经过80度高温，30分钟就可以杀死病毒。
 <br/>
 <br/>
 对此，徐女士表示，虽然目前还没有病毒传染到人的报告，但这并不代表吃猪肉是安全的，和禽流感病毒一样，非洲猪瘟的病毒也有变异的可能。〝任何病毒、微生物，它都有变异的可能。〞而且非洲猪瘟病毒是一类病毒，致病致死率甚至高达100%。
 <br/>
 <br/>
 此前已有消息指，大陆爆发的非洲猪瘟病毒是从俄罗斯传入的。据中共官媒报导，自贸易战开打后，中国停止从美国进口猪肉，转而从俄罗斯进口了24万吨冻猪肉，其中部分已运抵中国境内。但非洲猪瘟疫情爆发后，该消息迅速被官方遮罩。
 <br/>
 <br/>
 不过，经国家动物卫生流行病学中心，对中国境内发现的非洲猪瘟病毒毒株进行DNA测序发现，该毒株和此前俄罗斯西伯利亚非洲猪瘟的毒株DNA序列一致。表明这场疫情的爆发源是俄罗斯的可能性很高。
 <br/>
 <br/>
 徐女士认为，俄罗斯是疫区，将疫区的冻猪肉向外销售，本身就是不道德的行为。因为病毒在冻肉中也能存活1千天，病毒只要是在存活期，就会导致这个猪瘟疫病的流传。
 <br/>
 <br/>
 另外，中国红十字总会原高管任瑞红表示，非洲猪瘟即便是全面失控，对中共高级官员不会有什么影响。当年SARS疫情爆发时，中共官方进口的针对非典唯一有效的药物，都是为高官备用，就连大量一线冒着生命危险抗击非典的医护人员都不给使用。导致大量医护人员出现严重后遗症。
 <br/>
 <br/>
 任瑞红还指出，最近官方又宣扬要民众〝吃草也要打贸易战〞的说法，也显示中共高官们对民生并不在意。
 <br/>
 <br/>
 对于中国爆发的非洲猪瘟疫情，周边邻国和联合国都表示担忧。上月7日，联合国粮农组织已发布警告，称中共官方曝光的非洲猪瘟疫情只是冰山一角，担忧疫情将扩散至亚洲其他国家。据悉，韩国已发现从中国大陆传入的非洲猪瘟病毒，并加强了防疫措施。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：范铭)
