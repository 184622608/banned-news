### 郑晓松坠楼引各方质疑 知情人：死前一天声音有异
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】中共官方向外界宣称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门中联办主任.html" target="_blank">
  澳门中联办主任
 </a>
 郑晓松因患
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抑郁症.html" target="_blank">
  抑郁症
 </a>
 跳楼自杀。对此说法，社会各界反应不同。有知情人士透露，在郑晓松死前一天，曾与他会面，感觉他声音异常，表情也不丰富。
 <br/>
 <br/>
 对于郑晓松突然堕亡，香港、澳门一些官员说法不一。据《自由亚洲》10月21日报导，澳门中联办网站显示，郑晓松最后一次公开露面是19日，他在澳门中联办办公楼，会见民联智库成员。
 <br/>
 <br/>
 参与会面的民联智库副主席骆伟建透露，郑晓松当日会面时，说话声音与平常有异。
 <br/>
 <br/>
 骆伟建说：我只是感觉到他（郑晓松）在讲话的时候，他的声音不似平时，比较轻一点，我只是感觉到这个有变化，一般如果精神非常好的人，可能会动作、表情丰富一点，他相对起来表情没有这么丰富。
 <br/>
 <br/>
 澳门立法会主席贺一诚也透露，20日晚，他与澳门中联办人员吃饭时，他们突然离席，但没有透露原因。
 <br/>
 <br/>
 澳门立法会议员梁安琪则表示，不觉得郑晓松近期有异样（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抑郁症.html" target="_blank">
  抑郁症
 </a>
 ），对他的死觉得很突然。
 <br/>
 <br/>
 另一名议员吴国昌称，他与郑晓松没有个人接触。但他指出，假设郑晓松真的有抑郁症，官方为什么在这个时间公布，这是很奇怪的事。
 <br/>
 <br/>
 香港理工大学应用社会科学系助理教授钟剑华也质疑，如果郑晓松是有抑郁症，那为什么他一直在工作，似乎没有因为病而接受治疗。
 <br/>
 <br/>
 澳门时事评论员谭志强则表示，官方的说法有不少疑团。他说：过去一星期、甚至几个月，见过郑晓松的人都不觉得他有异常。因此，官方指郑晓松抑郁症跳楼，大家便觉得此事很不寻常。
 <br/>
 <br/>
 他还表示，中央挑选的驻港中联办或驻澳中联办主任，事先一定有政治审查、健康检查之类，没有理由会派一个身心出问题的人来澳门。
 <br/>
 <br/>
 谭志强还指出，虽然澳门政治环境没有香港复杂，但中央驻澳官员仍面对不少挑战，例如能否忍受或拒绝来自各个利益团体的诱惑。
 <br/>
 <br/>
 另据总部设在香港的中国人权民运信息中心透露消息，指郑晓松近日被中纪委官员问话。而在郑晓松坠楼之前，去年8月，在港澳任职长达13年、曾任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门中联办主任.html" target="_blank">
  澳门中联办主任
 </a>
 的国务院侨务办公室前副主任李刚落马。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：祝馨睿）
