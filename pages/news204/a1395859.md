### 扬州强拆血案 数百人围派出所要求放人
------------------------

<div class="wysiwyg">
 【新唐人2018年10月18日讯】江苏扬州市杭集镇日前发生抗强拆血案，官方通报，已造成两死八伤。10月17号，数百人围堵
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_派出所.html" target="_blank">
  派出所
 </a>
 、镇政府要求释放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_屋主.html" target="_blank">
  屋主
 </a>
 。屋主是一名特种兵
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退伍军人.html" target="_blank">
  退伍军人
 </a>
 ，当天原部队战友赶到镇上，今天持续跟政府官员谈判。
 <br/>
 <br/>
 10月15号，江苏扬州市杭集镇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韦刚.html" target="_blank">
  韦刚
 </a>
 家遭到强拆，韦刚驾车撞向拆迁队，造成两死八伤。17号早上，韦刚的妻子王女士到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_派出所.html" target="_blank">
  派出所
 </a>
 给丈夫送衣服，被扣押在派出所做笔录。韦刚是一名特种兵
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退伍军人.html" target="_blank">
  退伍军人
 </a>
 ，当天原部队的上司带着特种兵战友还有当地民众到派出所要求放人。
 <br/>
 <br/>
 当地民众赵先生：〝他老婆进去给他送衣服的，他老婆被他们给扣了，我昨天看退伍军人到那儿，两百号有，政府把人放出。〞
 <br/>
 <br/>
 当天下午，民众又聚集在镇政府前，在战友们要求下，受伤的韦刚从派出所被接到医院治疗，今天继续跟政府官员谈判。
 <br/>
 <br/>
 当地民众王女士：〝上千人没有，三百人差不多吧，对，（战友）这个是昨天就到了，今天在乡政府谈了。〞
 <br/>
 <br/>
 据悉，一百多名强拆人员在强拆时，围殴韦刚，往死里打，韦刚挣脱后开着轿车冲撞他们。
 <br/>
 <br/>
 王女士：〝撞死的，公布的是两个，确定的就是当场一个嘛，然后抢救无效一个，其余的八个住院，现在好像是脱离危险了吧。应该是拆迁人雇的吧，反正穿着那个黄马甲嘛。〞
 <br/>
 <br/>
 当地民众表示，政府很腐败，强拆很普遍，当地前几年也发生过强拆酿成的命案，因为政府封锁消息，很多人都敢怒不敢言。
 <br/>
 <br/>
 当地民众杨先生：〝我跟你说了，把我的身分曝露，马上政府来找我麻烦了，我有点害怕，不保密我不给你说。〞
 <br/>
 <br/>
 当地民众还表示，当地政府以整治河道为由，拆迁河道边所有的房子，目前只剩几户还没拆，由于补偿价格过低，引发民怨。
 <br/>
 <br/>
 新唐人记者熊斌、陈杰采访报导
</div>
