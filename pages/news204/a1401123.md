### 日媒：贸易战逼中共出狠招 靠杀猪缓解大豆危机
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的焦点，中共曾将其作为对付美国的所谓〝杀手鐧〞。不过这个〝杀手鐧〞对中国的伤害似乎更大，目前，大陆与大豆相关的产业均遭受重创。日媒称，受影响严重的中国养猪户称，只能靠
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杀猪.html" target="_blank">
  杀猪
 </a>
 来缓解大豆危机。
 <br/>
 <br/>
 将于11月30日召开的二十国集团（G20）峰会，最受各国关注的是美中之间的没有争端。外界认为，暂时还看不到能够打开局面的因素，预计双方会依然保持激烈的对抗态势。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 在本次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 中倍受关注，为了打击美国总统川普（特朗普）的共和党票仓，今年4月4日，中共将美国大豆作为美中贸易战的所谓〝杀手鐧〞，加征报复性的25%关税。
 <br/>
 <br/>
 当时，大陆媒体还宣传，中国并不担心〝这会对国内相关领域产生重大影响〞。不过，短短几个月，有消息显示，大豆〝杀手鐧〞的反作用已经让中共担心。
 <br/>
 <br/>
 日媒《朝日新闻》11月28日报导称，河南新乡的农民们表示，〝无法买到喂猪的饲料，没办法，只有靠杀掉来减少数量。〞
 <br/>
 <br/>
 自贸易战开打后，美国大豆价格上升，中国国内大豆也上涨了10%，中国很多饲料生产产家面临成本过高和大豆供应短缺的危机。
 <br/>
 <br/>
 此前已经有很多经济学者、分析人士警告，对美国大豆加征关税会影响到中国的养殖业，提醒中共慎打大豆牌。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田表示，美国对中国出口的大豆是很大的一项。中国有四亿头猪需要养，因为猪肉是中国百姓最主要的需求之一。所以这四亿头猪的圈养的话，需要大量的大豆，而中国已经不能自己提供了，消减美国大豆必然会引起强烈反弹。
 <br/>
 <br/>
 此外，美国是最大的大豆供应国，中国不购买美国大豆，也很难找到别的替代方。近期，中国改从巴西进口大豆，但巴西大豆价格比美国还高，而且巴西的大豆库存也比去年少了很多，预计将在2019年3月彻底枯竭，届时价格将达到最高。
 <br/>
 <br/>
 自由亚洲10月的报导称，广州市一名猪农表示，喂猪的饲料成份以大豆为主，所以近日购买大豆的成本增加，他亦感到很无奈。〝（成本）贵了，有影响的，（饲料）贵了几百元一吨，没有得选择这件事，成本就是高了。〞
 <br/>
 <br/>
 据悉，豆粕含有44%的粗蛋白，不仅容易获得，而且价格适中，因此成为全世界最常见的猪、家禽和奶牛配合饲料的蛋白源。
 <br/>
 <br/>
 广州中山大学经济系教授林江表示，由于家禽的饲料多以大豆成份为主，再加上美国大豆的质量比其他国家优良，中国无可奈何亦要向美国购买昂贵的大豆。
 <br/>
 <br/>
 彭博社报导，两艘满载美国大豆的货船〝劳拉星〞号及〝金色皇后〞，近日装船完毕，运往中国的大豆约6.9万吨，预计12月抵达中国。
 <br/>
 <br/>
 除了养殖业报，中国庞大的大豆压榨业也出现多年来最大规模的减产现象。已至少有20家大豆压榨厂减产或停产。
 <br/>
 <br/>
 7月中旬，中国民营企业家、被称为〝大豆王〞的山东晨曦集团创始人邵仲毅，向当地法院申请破产。几年前，这家公司曾是大陆最大的大豆进口民营企业之一，2012年晨曦集团共进口大豆551万吨，约占全国当年进口总量的十分之一。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
