### 王岐山遭四人暗算 习近平将计就计接连干掉三对手
------------------------

<div class="wysiwyg">
 <br/>
 【新唐人北京时间2018年10月17日讯】中共两会上，十九大裸退的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 强势回归，以普通人大代表的身份〝获选〞中国国家副主席，成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 之后的〝2号人物〞，引发外界高度关注。有港媒揭，王岐山曾遭到4人〝暗算〞，导致十九大上退出权力中心。而习近平则将计就计，利用王岐山的裸退，挤掉3名江派常委。
 <br/>
 <br/>
 王岐山是习近平的政治盟友，中共十九大上，王岐山最终未能留任中共政治局常委、中央委员及中纪委委员。当时，不少观点认为王岐山彻底退出了权力中心。
 <br/>
 <br/>
 但在两会前，王岐山又罕见的当选湖南省全国人大代表。3月4日中共人大预备会议上，王岐山被选为主席团成员，同时与六名政治局常委同坐一排，被誉为〝第八常委〞，将成为习近平之后的〝二号人物〞。
 <br/>
 <br/>
 那么习近平为什么让王岐山在中共十九大卸任，两会上又大费周折的让王岐山重返政坛呢？
 <br/>
 <br/>
 红二代罗宇认为，这很可能是王岐山和习近平俩人用的〝小计谋〞，十九大上王卸任是要〝陪绑〞，目的是让江派
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张德江.html" target="_blank">
  张德江
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张高丽.html" target="_blank">
  张高丽
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 三政治局常委同时卸任。
 <br/>
 <br/>
 香港前哨杂志曾报导，中共十九大上，习近平并没有完全掌握人事，新一届常委名单系各派妥协的产物。其中博弈的焦点王岐山，更遭各方拚死抵制，因而未能如愿留任。
 <br/>
 <br/>
 报导讲，起初习近平打算让王岐山留任，王本人也愿意。十九大的9月之前，二人的部署都是围绕这个目标运作的。但在中共党内征求意见时，王岐山留任遭到异常激烈抵抗。
 <br/>
 <br/>
 第一大阻力来自党内大大小小的利益集团。王岐山反腐与整个官僚集团为敌，如果王留任，他们将寝食难安。第二大阻力来自中共退休大佬，特别是当年因〝七上八下〞出局的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾庆红.html" target="_blank">
  曾庆红
 </a>
 。
 <br/>
 <br/>
 报导说，王岐山曾经多次约谈曾庆红，训令他交代本人和家族的贪腐问题。如果王不退，曾庆红很可能在秦城监狱度过残生。因此，曾庆红使出〝浑身解数〞，动用所有关系网，拚死阻止。
 <br/>
 <br/>
 第三大阻力来自和像王岐山一样年满68岁的刘云山、张德江和张高丽等江派常委。他们对王岐山留任不服，或明或暗反对。刘云山甚至一度提议常委人数扩展到11人，以便他也能留任。
 <br/>
 <br/>
 在强大阻力下，王岐山表态因健康原因不再留任。王还私下讲，想多活几年，退出常委会。
 <br/>
 <br/>
 最后博弈的结果是，习近平架空了集体领导制，废除了隔代指定接班人，但没能打破〝七上八下〞年龄潜规则。
 <br/>
 <br/>
 但在3月的中共人大会议开幕日，当王岐山紧接现任政治局七常委一起入场后，有外媒形容，过去5年令中共贪官闻之色变、视为〝阎王〞的前中纪委书记王岐山，又杀了个回马枪，上演了一幕政坛版的〝王者归来〞。
 <br/>
 <br/>
 对王岐山重返政坛，红二代罗宇分析说，这样的安排没破中共规矩，因为中共前领导人邓小平，也曾以中共党员的身份任职中央军委主席，因此，王岐山任国家副主席令党内无话可说。
 <br/>
 <br/>
 不过，罗宇也表示，虽然王重返政坛没破规矩，但需要关注的是，习究竟有无在党内有什么妥协，是否能掌控全部权力和得民心。他说，若习在得到权力后继续走专政之路，肯定〝站不住脚〞，中共很快会垮台。他不管做什么，中共注定要垮台。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：赵云）
