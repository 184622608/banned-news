### 严寒将至？中共救市幅度超金融海啸 外汇近红线
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】中美贸易冲突爆发以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 和汇市一直跌跌不休，日前汇率再次接近破7红线。港媒指，中共央行今年放水救市的幅度远超2008年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融海啸.html" target="_blank">
  金融海啸
 </a>
 ，显示中共经济依然危机四伏并有加重趋势。
 <br/>
 <br/>
 周一（10月8日），〝十一〞长假后的首个交易日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 继续大跌。此前一日，中共央行宣布第四次降准〝救火〞，放水近7,500亿元人民币，力度达到今年之最。
 <br/>
 <br/>
 香港《苹果日报》指，今年前三次降准，累计释放1.85万亿人民币，比2008年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融海啸.html" target="_blank">
  金融海啸
 </a>
 救市规模更庞大。
 <br/>
 <br/>
 此前有陆媒总结，仅仅今年上半年，央行通过降准和其它各种名目，已经释放4万亿资金。
 <br/>
 <br/>
 港媒引述分析指，央行第四次降准明显与贸易战升温有关。而上月制造业采购经理指数（PMI）低过预期，以及上周港股跌势急速，令本拟第四季中开始的降准，被迫提前实施。
 <br/>
 <br/>
 据中共官方数据，9月份制造业采购经理指数（PMI）按月跌0.5个百分点至50.8%，出口定单指数及进口指数分别只得48%和48.5%，按月跌1.4个百分点和0.6个百分点，更是全年新低。
 <br/>
 <br/>
 《彭博周刊》日前爆出晶片窃密风波，令一众香港科技股下跌，恒指上周泻近1215点或4.38%，带动上证指数周一开市跟跌。港媒指，有爆发金融危机的风险。
 <br/>
 <br/>
 另外，港媒文章分析，美国关税措施直插中共经济心脏地带，多个财金指标都出现问题，除了A股走弱、制造业数据转差之外，人民币也贬值不休，甚至连楼市都出现疲态，中共经济面临〝十面埋伏〞，迫使央行不得不出招救市。
 <br/>
 <br/>
 文章说，中美股市的表现，是判断贸易战胜负的明显指标。今年以来美股三大指数升幅显著，道指升近7%，纳指则升近12.8%。而上海及深圳指数分别跌近14.7%及24%，后者表现更是全球最差。
 <br/>
 <br/>
 此外，虽然中共官方一直强调实施〝稳健中性〞货币政策，但外界普遍认为，贸易战会让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币贬值.html" target="_blank">
  人民币贬值
 </a>
 压力越来越大。
 <br/>
 <br/>
 港媒引述经济师分析指，贸易战及国内信贷压力越来越大，明年央行可能续降存款准备金率，甚至会减息。这会导致资金流向息率更高的美元，引发人民币沽压，今年人民币兑美元汇率或将破7。
 <br/>
 <br/>
 9月，中共外汇储备跌约227亿美元，至30,870亿美元，再次逼近3万亿美元的心理关口。港媒分析，下跌的227亿美元，或被消耗来维持滙价稳定。
 <br/>
 <br/>
 海外经济学者曾分析，人民币目前被严重高估，而中共要保汇率，最终还是要靠外汇储备支撑。这会导致外汇储备很快被耗尽。
 <br/>
 <br/>
 中共经济全面走弱，就连长期支撑贷款增长的楼市，近日亦出现减价的现象。龙头万科主席郁亮更明言以〝活下去〞为最终目标。受按揭贷款放慢影响，部份房企开始减价促销，包括万科、恒大、碧桂园在厦门、杭州、南京等多个城市的楼盘。
 <br/>
 <br/>
 对于央行第四次降准，一次性放水7,500亿，美媒指北京真的对贸易战感到紧张了。
 <br/>
 <br/>
 而在央行9月份在6天内净投放5,950亿元人民币时，美国时事评论家章家敦在推特评论称，〝以这个速度印钞表明中国（中共）很绝望，局势危急。〞
 <br/>
 <br/>
 中国大陆金融学者贺江兵此前断言，贸易战会轻易引爆中共内债炸弹，刺破金融和楼市泡沫，中共经济很快将开始进入崩盘模式。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
