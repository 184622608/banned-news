### 【禁闻】新西兰禁止5G移动网络使用华为设备
------------------------

<div class="wysiwyg">
 【新唐人2018年11月29日讯】出于对国家安全的担忧，新西兰禁止国内主要电信运营商使用中国华为的移动通信系统（5G）设备。这是继澳洲之后，另一个国家限制使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为产品.html" target="_blank">
  华为产品
 </a>
 ，这也显示西方国家对中共渗透越来越警觉。
 <br/>
 <br/>
 新西兰最大电信公司Spark 11月28号表示，他们已经接到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新西兰通讯安全局.html" target="_blank">
  新西兰通讯安全局
 </a>
 的通知，禁止公司使用中国电信龙头〝华为〞的设备，来建设第五代移动通信系统（5G）移动网络，理由是会〝带来重大的国家安全风险〞。
 <br/>
 <br/>
 Spark的5G移动网络预计在2020年7月前完工。根据新西兰2013年国家电信法，电信公司在更新通讯网络时，必须向政府申报使用的技术。Spark公司此前提议使用华为的设备。新西兰通讯安全局局长汉普顿 (Andrew Hampton) 在简短声明中表示，通讯安全局在审查后告诉Spark，它们的计画中发现了潜在的国安风险。
 <br/>
 <br/>
 华为成为新西兰国内第一家被拒的电信商。但此前，华为在多个国家都引发了国安忧虑。
 <br/>
 <br/>
 今年8月，澳洲政府出于国家安保理由，禁止澳洲手机运营商使用华为和中兴提供的5G技术。英国政府7月公布报告，指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为产品.html" target="_blank">
  华为产品
 </a>
 的技术和供应链问题，使得英国通讯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络安全.html" target="_blank">
  网络安全
 </a>
 性面临新的风险。而德国预计明年初开始招标5G设备，德国官员11月透露，希望将中国华为等电信公司排除在招标范围之外。而美国出于〝国家安全〞顾虑，已多次拒绝华为并购美国企业，或限制华为产品进入美国。
 <br/>
 <br/>
 网络自由观察人士古河：〝众所周知，华为它是一个军工控制的企业。它的所有产品中毫无疑问的都埋有后门。如果西方社会不加防范，大批量的使用它的产品以后，那么在适当的时候，它就可以进行大规模的网络控制。〞
 <br/>
 <br/>
 根据公开资料，华为的创办人任正非曾是中共军队团职干部，刚刚卸任的董事长孙亚芳在加入华为前，曾长期在中共国安部任职。
 <br/>
 <br/>
 而美国2011年3月4号阻止华为收购3Leaf公司时曾给出理由，包括：华为创业早期合约全部为军方控制的中资驻港企业，政府背景明显；中共军方长期无偿向华为提供关键技术，双方签署多项长期合作项目，军方背景明显；华为长期策划及从事与盗取知识产权相关的商业犯罪，商业道德败坏；还有，华为有向包括萨达姆‧侯赛因、伊朗、塔利班提供服务的长期历史......等。
 <br/>
 <br/>
 旅美时事评论员蓝述：〝在这种情况下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为生产.html" target="_blank">
  华为生产
 </a>
 的所有的通讯设备，当然有可能影响到西方国家的政府，还有一些最重要的经济和国防相关的、大型的、能源通讯基础设施的安全。西方的这些民主国家他当然要采取相应的应对措施，去维护自由世界的安全。〞
 <br/>
 <br/>
 对华为在新西兰遭禁一事，中共外交部发言人耿爽在28号的例行记者会上做出回应，说，会〝严重关切〞，并说，希望新西兰为中国企业在当地运营提供公平竞争环境。
 <br/>
 <br/>
 旅美时事评论员蓝述认为，中共官方这种说法故意混淆了概念。
 <br/>
 <br/>
 蓝述：〝市场是市场，安全是安全，必须分得很清楚。中共故意把它摆在一起，是因为中共没有向西方企业提供平等的竞争机会。但是，西方国家现在对华为的制裁主要是出于安全上的考量。华为既然有军方和情报部门的背景，那华为它就是个间谍企业。这根本就不存在公平竞争的问题。〞
 <br/>
 <br/>
 目前美国还在说服盟友避免使用华为设备，《华尔街日报》上周四援引知情人士的消息说，对于德国、意大利和日本等广泛使用华为设备的盟友，美方已告知这些国家的官员和电信高管，有关他们认为存在的网络安全风险。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/李沛灵
