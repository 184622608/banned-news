### 又一中共官员〝因病〞自杀 这次不是抑郁症
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】中共官方多以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抑郁症.html" target="_blank">
  抑郁症
 </a>
 〞搪塞官员自杀死因，饱受民间调侃。因此，官方也不断对此说法〝推陈出新〞。继有地方用官员〝遗书〞自证抑郁症之后，四川又开创出官员自杀的新死因：胃病。
 <br/>
 <br/>
 四川官方日前宣布，北京时间11月15日13时20分左右，什邡市财政局局长黄卓被发现缢亡于办公室。
 <br/>
 <br/>
 通报说，什邡市公安局已排除他杀。黄卓〝患有严重胃病，长期受病痛折磨〞。
 <br/>
 <br/>
 中共官员因为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抑郁症.html" target="_blank">
  抑郁症
 </a>
 〞自杀已经司空见惯。因为〝胃病〞自杀，似乎还是头一个。
 <br/>
 <br/>
 此前，内蒙古官媒引述当地公安厅的说法称，北京时间11月1日凌晨，内蒙古公安厅副厅长、呼和浩特市公安局局长李志斌，在该市公安局休息室内缢亡。
 <br/>
 <br/>
 报导特别强调，李志斌留在现场的〝遗书〞中称，自己〝一直被抑郁症困扰，每天都靠高效安眠药才能入睡〞。
 <br/>
 <br/>
 此前，中共官方宣布官员自杀原因时，多借用〝医生〞或〝亲友〞、〝同事〞的说法来引证该官员患有〝抑郁症〞。官员用〝遗书〞自证〝抑郁症〞，也是头一遭。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
