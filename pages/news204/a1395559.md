### 崔永元与〝李肇星侄女〞开战：就喜欢打有来头的
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 最近频频在微博开炮，被称为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李肇星.html" target="_blank">
  李肇星
 </a>
 侄女、香港卫视综合台副台长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦枫.html" target="_blank">
  秦枫
 </a>
 成为最新目标。崔永元在微博中对其怒爆粗口，称秦枫以为家里是高干就随手欺负人，并说〝我就喜欢打有来头有背景的X子！〞
 <br/>
 <br/>
 10月12日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 发布微博称：〝我记得很清楚，这个X子是第二次骂我了，无缘无故。〞并附上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦枫.html" target="_blank">
  秦枫
 </a>
 当天发布的微博信息截图。秦枫在微博中嘲讽崔永元〝恶心、太能装〞。
 <br/>
 <br/>
 早在2014年10月，崔永元和秦枫就有过对骂，当时崔永元与方舟子正在大战，秦枫在微博形容崔永元是〝狡猾的战队者〞。
 <br/>
 <br/>
 时隔4年，秦崔再起骂战。崔永元13日又连发三条微博，对秦枫怒爆粗口，称〝受再多的教育，也想叫她X子。〞
 <br/>
 <br/>
 次日，崔永元又转发一条网友评价秦枫的微博，并说：〝这种X子以为家里是高干就随手欺负人为乐，不巧，碰上我了。我就喜欢打有来头有背景的X子！〞
 <br/>
 <center>
  <br/>

   <br/>
   <font size="-1">
    崔永元在微博转发媒体旧报导，曝光2012年中共〝两会〞期间，秦枫两手抓住
    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李肇星.html" target="_blank">

 </center>
 <br/>
 同时，崔永元还转发了其他网友贴出的媒体旧报导，曝光2012年中共〝两会〞期间，秦枫两手抓住李肇星西装前襟提问的旧照，指秦枫是李肇星的侄女。因是自己侄女提问，李肇星当时哭笑不得。
 <br/>
 <br/>
 报导称，这一场景引起外界注意，遂曝光秦李的亲戚关系。公开信息显示，秦枫的祖父秦力真，外祖父杨琪良，前外交部长李肇星是她的姑父。网民还查出，李肇星在2012年1月15日会见香港卫视新闻局主席高洪星，2月秦枫即离开凤凰卫视，并出任香港卫视新闻采访总监。
 <br/>
 <br/>
 李肇星被指有浓厚的江泽民派系色彩，他在任美国大使期间言语粗鲁，美国媒体嘲讽他为〝红卫兵大使〞。
 <br/>
 <br/>
 前美国副总统切尼在回忆录中曾详细描写，2002年，时任中共总书记胡锦涛首次访美期间，切尼尝试与胡进行一对一会谈，却被随行的李肇星强行闯入，并坐在两人之间，导致计划落空，切尼也因此见识到江、胡斗的激烈程度。
 <br/>
 <br/>
 李肇星曾在公开场合表示：〝中国人权比美国好5倍〞。但据港媒披露，李肇星的子女全部在美国发展。2007年4月27日，李肇星被免去外交部长职务。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
