### 余文生案审查起诉再延期 仍禁会见
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】首先来关注中国的消息。被控〝煽动颠覆国家政权罪〞的北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">
  余文生
 </a>
 ，今年1月被当局控制至今。11月7号，他的妻子与律师再次到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐州看守所.html" target="_blank">
  徐州看守所
 </a>
 和检察院要求会见和阅卷，都遭到拒绝，而且被告知案件继续被退回检察院审查起诉。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">
  余文生
 </a>
 律师的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_许艳.html" target="_blank">
  许艳
 </a>
 8号向〝新唐人〞透露，7号早晨，她和律师到达看守所，要求会见余文生，但被拒绝了。
 <br/>
 <br/>
 余文生律师的妻子许艳：〝常伯阳律师、谢阳律师、卢廷阁律师还有我，大概是早晨10点前就到达徐州市看守所，三位律师要求会见余文生，可是辩护手续一交，工作人员就说不让会见，律师也是争取，然后又让在那等，等了很长时间，最后的结果还是不让会见。〞
 <br/>
 <br/>
 当天下午，他们一行人又到徐州检察院要求阅卷，结果被告知案件起诉又延期了。
 <br/>
 <br/>
 许艳：〝下午，律师又带着我去一趟徐州市检察院，查询余文生这案件的现在情况，徐州市检察院的工作人员回复说，他的案件又被延期了，延期到11月20号，让我到那时候再问情况是什么样。〞
 <br/>
 <br/>
 余文生律师，现年51岁，曾代理多起法轮功学员辩护案、〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709事件.html" target="_blank">
  709事件
 </a>
 〞被抓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 案，今年1月15号被注销律师证，17号，他在推特发表〝关于修宪的公民建议〞，19号被抓，20号被控涉嫌〝妨害公务罪〞，4月19号遭正式逮捕，罪名是〝煽动颠覆国家政权罪〞。
 <br/>
 <br/>
 同样因为代理维权案件而被吊照的律师祝圣武，11月3号在网上发文呼吁大家声援余文生，他说，11月11号是余文生的生日，希望关心他的朋友在那天拍摄照片或视频并分享到网络。
 <br/>
 <br/>
 许艳：〝对各界的关注，是一种感谢。我从妻子角度，非常信任他，他是一个很好的律师，现在遭受这些，其实对律师和对我家庭来说是很不公平，也是很残酷的一件事情。〞
 <br/>
 <br/>
 许艳表示，希望公平、正义、法治能早日体现在维权律师案件上，希望所有被关的维权律师都能早日回家。
 <br/>
 <br/>
 新唐人记者常春、陈洁、施宜君采访报导
