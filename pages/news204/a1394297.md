### 法媒：中共密召高官子女回国恐遭抵制
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】贸易战令美中关系处于深度冻结之际，有消息称，中共密令在美国的高干子女年内
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_回国.html" target="_blank">
  回国
 </a>
 ，同时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁止.html" target="_blank">
  禁止
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高官子女.html" target="_blank">
  高官子女
 </a>
 赴美留学。5日，有法媒说，近期多次传说中共官方下令官员子女回国，但相关命令遭到中共高官抵制。
 <br/>
 <br/>
 香港经济日报10月2日援引消息说，美国总统川普（特朗普）发出〝来美国中国留学生都是间谍〞的言论后，中共官方随即开始了有关行动，发出内部文件。
 <br/>
 <br/>
 商业期刊日语版1日报导，中共外交界的消息人士披露，近来一份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁止.html" target="_blank">
  禁止
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高官子女.html" target="_blank">
  高官子女
 </a>
 去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国留学.html" target="_blank">
  美国留学
 </a>
 ，让已经出去的子女年内
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_回国.html" target="_blank">
  回国
 </a>
 的内部文件出炉。中共高层认为，贸易摩擦短时间内难以解决，因此中共官员的孩子被要求回国，也暂时禁止去美国留学。
 <br/>
 <br/>
 尽管上述消息尚待证实，但有关说法引起舆论广泛关注。
 <br/>
 <br/>
 近年来，无论是中共官员，还是富豪，纷纷让亲属子女移民，将财产转移到海外，在美中贸易战令美中关系处于深度冻结之际，传出中共下令召回留美的高官子女的消息，其背后原因引猜测。
 <br/>
 <br/>
 10月5日，法广引述消息人士分析称，中共密召在美高官子女回国，主要还是为切断官员二心的后退之路。还有分析认为，近期多次传说中共官方下令官员后代回国，但命令没有被有效执行。
 <br/>
 <br/>
 新唐人分析认为，中共密召在美高官子女回国有两大原因，一是，可能与川普政府要抓中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_留学生间谍.html" target="_blank">
  留学生间谍
 </a>
 有关。目前正值中美贸易战扩大，中美关系恶化，川普暗示中国留学生几乎每个都是间谍，中共担心川普政府可能对中国留学生间谍进行追究、驱逐等。
 <br/>
 <br/>
 二是，与川普制裁高官有关，尤其是刚刚制裁了中共军委装备发展部部长李尚福。北京担心美国或制裁更多的高官，届时不仅中国间谍留学生被驱逐出境，而且中共高官的海外资产也可能被没收，同时面临法律制裁，因此中共先把高官子女召回国。
 <br/>
 <br/>
 时政评论员周晓辉则认为，北京此举有更深层用意，那就是为了避免中共高官们〝身在曹营心在汉〞，避免中共高官们因顾忌自己的子女、财产在美国而对中共政权〝阳奉阴违〞，甚至在川普极限施压后在内部发难，是以迫使他们将子女召回。
 <br/>
 <br/>
 北京独立社会学者胡星斗对美国之音说，中共出台要求在美高官子女回国的文件不太可能，但也不排除中共可能就某些重点高干子女采取特殊政策，提出特别要求。
 <br/>
 <br/>
 他说：中共对外开放的政策，鼓励留学的政策，包括到美国留学的政策不会改变。但是某些高官的子女，他们可能涉及到种种原因，例如，国家的一些敏感的问题，裸官的问题等等，可能会要求部分高干子女回国，这是可以想像得到的。
 <br/>
 <br/>
 海外中文网《人民报》早年曾在〝中国高干子女在美国大揭秘〞一文中说，曾有人开玩笑说，一旦中美交恶，中共最大的人质就在美国。因为美国是全世界中共高干子女最集中的国家，而江泽民是最〝亲美〞的中共领导人。
 <br/>
 <br/>
 报导说，许多中共高层领导人，包括邓小平、杨尚昆、刘少奇、万里、江泽民和钱其琛等人的子女都曾留学美国，有的已在美安居乐业。纽约是中共高干子女的大本营，旧金山和西雅图次之。
 <br/>
 <br/>
 中共〝太子党〞上世纪八十年代，捷足先登赶上〝出国潮〞后，大多学成回国。不过，他们中，不少人后来却千方百计将子女再送往美国。
 <br/>
 <br/>
 报导认为，在美的高干子女，其实对政治大多没有兴趣，他们关注的只是股票、房地产、国际贸易，或者女人、男人，成为中共的〝新贵族〞，而且他们用脚投了票，公开选择在西方制度下生活。
 <br/>
 <br/>
 美国之音分析说，这些与中共政权离心离德的高干子女，是否能听从指挥，年底前返回中国似乎是个问号。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
