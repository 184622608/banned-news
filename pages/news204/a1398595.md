### 中共招募高中生研发杀手机械人 专家忧惧
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】中共利用北京理工大学公开招募了数十名〝最聪明〞的高中生，这些学生将受训成为专注于研发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_智能机器杀手.html" target="_blank">
  智能机器杀手
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_AI武器.html" target="_blank">
  AI武器
 </a>
 科学家。在全球数千AI专家联署呼吁各国政府承诺永不发展致命性AI武器系统的当下，中共这项野心勃勃的计划令国际科学政策专家深感忧惧。
 <br/>
 <br/>
 <h4>
  中共招募青少年精英培养
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_AI武器.html" target="_blank">
   AI武器
  </a>
  科学家计划曝光
 </h4>
 <br/>
 据香港《南华早报》报导，北京理工大学公开招募了31名〝爱国青少年〞，打算通过一项为期4年的〝智慧武器系统实验计划〞，把他们培养成〝世界最年轻的AI武器科学家〞。
 <br/>
 <br/>
 据报导，上述31名青少年是从5千多名候选人中挑选出来的精英人才，有27名少年和4名少女，年龄都在18岁以下。
 <br/>
 <br/>
 报导援引一名参与遴选的北理大教授透露，这些被选中的孩子都〝异常聪明〞，负责这项挑选工作的人还同时兼顾了这些孩子的创意思考能力、勇于斗争和能够迎难而上等特质，而且还要求这些青少年〝必须爱国〞。
 <br/>
 <br/>
 根据相关的培训计划，这些学生入学后都会接受两名资深武器科学家的辅导，其中一人拥有学术背景，另一人则来自国防产业领域。
 <br/>
 <br/>
 首学期后，这些学生将根据自己所长选择自己的研修科目，比如机械工程、电子或武器设计等。之后，他们将会被分配到相关的国防研究室，通过参与实际的研发工作来发展自己的技术。
 <br/>
 <br/>
 在完成4年课程后，这些学生将会继续博士课程，最终目标是成为中国人工智能武器工程的下一代领袖。
 <br/>
 <br/>
 报导表示，中国（中共政府）正积极推动发展致命性人工智能(AI)的应用，范围从核子潜舰到自我学习晶片，再到可以进入人类血管的微小机械人。而北理大是中国众多顶尖武器研究中心之一，该校对人工智能在军事方面的应用非常重视。
 <br/>
 <br/>
 报导引述纽约联合国大学政策研究中心学者鲍威尔斯（Eleonore Pauwels）表示，她对上述计划深感忧心，因为这是全球首家大学展开这样的项目，极具野心和战略地培养下一代思考、设计如何把人工智能运用于军事用途。
 <br/>
 <br/>
 国际科学政策专家也表示，中共政府的这些课程计划〝极度强大且令人不安〞。
 <br/>
 <br/>
 据香港《亚洲时报》9月12日的一篇报导，中共政府宣称，到2030年将要把中国建成为〝世界主要人工智能创新中心〞，预计届时中国人工智能（AI）产业的价值将达到1,500亿美元。
 <br/>
 <br/>
 美国智库新美国安全中心2017年11月发表的一份报告指出，引入高端人工智能的中国军队正致力于在陆海空实现世界领先的〝无人化〞。
 <br/>
 <br/>
 <h4>
  世界顶尖AI科学家联署呼吁〝永不发展致命的AI武器系统〞
 </h4>
 <br/>
 事实上，全世界有许多科学家对于发展〝有思考能力的杀人机器〞（即AI武器）可能给世界带来毁灭性灾难深感忧惧。就在今年7月中旬，有2000多名世界知名的AI专家联署了一份特殊的宣言，其核心内容就是承诺〝永不发展致命的AI武器系统〞。
 <br/>
 <br/>
 这次联署行动由MIT著名物理学家Max Tegmark发起，宣言的文本则是由他任职主席的〝生命未来研究所〞（Future of Life Institute，FLI）起草的。
 <br/>
 <br/>
 包括企业领域的马斯克；DeepMind联合创始人哈萨比斯、穆斯塔法、Shane Legg；Skype创始人Jaan Tallinn；全球顶尖AI学者Yoshua Bengio、Stuart Russell、Jürgen Schmidhuber等，共2400名遍布全球90个国家160多家AI企业或机构的AI科学家郑重签下了大名。
 <br/>
 <br/>
 这份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_《致命性自主武器宣言》.html" target="_blank">
  《致命性自主武器宣言》
 </a>
 （lethal autonomous weapons pledge）指出，掌控人类个体生死的决定权，决不能交给没有感情的机器。这不仅是出于道德层面的考虑，也是因为致命性的AI武器拥有自主选择攻击目标的能力，其可归责性与人类掌控的难度都前所未有，尤其是当AI武器技术与监控、数据系统相结合时，它们将成为暴力活动中的强大工具。
 <br/>
 <br/>
 宣言强调：致命性自主武器的特征不同于核武器、化学武器及生物武器，研发团队可以单方开启军备竞赛，而国际社会还缺乏实行管制的技术手段。因此，呼吁各国政府及首脑为了地球的未来，为了维护全球安全，应该谴责并尽力避免发展AI武器的军备竞赛。
 <br/>
 <br/>
 （记者唐迪综合报导/责任编辑：秦锋）
