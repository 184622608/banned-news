### 河北一直升机坠毁起火 机上人员无一生还（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】10月26日，在河北省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦皇岛.html" target="_blank">
  秦皇岛
 </a>
 市一村落的空地上，一架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_直升机.html" target="_blank">
  直升机
 </a>
 在飞行时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠落.html" target="_blank">
  坠落
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_起火.html" target="_blank">
  起火
 </a>
 ，机上人员无一生还。截止26日晚，官方还没公布飞机坠落原因及人员伤亡情况。
 <br/>
 <br/>
 有微博网友发帖称，10月26日中午12时许，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦皇岛.html" target="_blank">
  秦皇岛
 </a>
 市海港区东港镇崔家庄村附近的空地上，有一架小型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_直升机.html" target="_blank">
  直升机
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠落.html" target="_blank">
  坠落
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_起火.html" target="_blank">
  起火
 </a>
 ，现场浓烟滚滚。附近村民介绍，当时这架小型直升机在飞行时失控，随后坠落起火。
 <br/>
 <br/>
 现场的目击者描述，坠毁的是一架蓝色的小型直升机，村民在现场并未看到有直升机的驾驶人员或者乘客，怀疑飞机上的人在发生坠机后无一生还，飞机所起火焰在12时20分许被扑灭。
 <br/>
 <br/>
 一位目击者告诉陆媒记者，事发时他在院子中，有人看到飞机掉落下来。他赶到现场后，发现飞机卡在坑里，消防队员已经把火扑灭，能看到飞机尾翼为蓝色。据他了解，飞机在空中〝一转圈就掉下来了〞。
 <br/>
 <br/>
 截止26日晚，官方还没有公布飞机坠落原因、人员伤亡情况。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
