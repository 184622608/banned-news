### 辽宁2重刑犯越狱 警察追捕不成搭上两命
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】辽宁日前发生越狱事件，2名男重刑犯现已逃至河北境内，但仍未抓获。中共官方至今未公布2名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃犯越狱.html" target="_blank">
  逃犯越狱
 </a>
 的原因及过程。河北政府5日公布一条消息，河北警察在追捕2名逃犯的过程中，突发车祸，造成2人死亡，2人受伤。
 <br/>
 <br/>
 据陆媒报导，10月4日凌晨，2名男重刑犯从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁凌源第三监狱.html" target="_blank">
  辽宁凌源第三监狱
 </a>
 越狱。逃犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王磊.html" target="_blank">
  王磊
 </a>
 ，辽宁人，1985年3月出生，初中文化。曾于2013年12月参与绑架杀害一名11岁儿童，被以绑架罪判处死缓，2017年12月28日减为无期徒刑。
 <br/>
 <br/>
 逃犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张贵林.html" target="_blank">
  张贵林
 </a>
 ，安徽临泉县人，1979年9月出生，文盲。2001年7月11日，张贵林因抢劫罪被判刑7年，2007年9月18日刑满释放。
 <br/>
 <br/>
 2009年1月19日，张贵林又以抢劫罪被判无期。据悉，在此次越狱前，张贵林已有过两次从监狱脱逃经历。2011年6月和2012年7月，张贵林两次因脱逃罪被判刑1年6个月和2年。与前罪并刑决定执行无期徒刑。
 <br/>
 <br/>
 据知情人士透露消息称，王磊与张贵林逃走时偷走了狱警的衣服，利用衣服中所夹带的门禁卡逃出监狱。但官方至今没有公布2名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逃犯越狱.html" target="_blank">
  逃犯越狱
 </a>
 的原因和经过。
 <br/>
 <br/>
 有网友对这起越狱事件发出质疑，称〝跑了三次？这是监狱还是自由市场。〞
 <br/>
 <br/>
 还有网友表示：如果按照一般监狱的配置和出监察关卡，最外层都是武警，没有狱警的内应基本是绝对出不去的……都是重刑犯，出去了还有钱，就是有人接应了……。
 <br/>
 <br/>
 据陆媒公布的监控画面显示，10月4日上午11时许，两名疑似逃犯出现在距离监狱约20公里外的凌源市宋杖子镇，一人进入一家小卖部购买火腿肠、香菸、面包。
 <br/>
 <br/>
 随后，网上传出消息称，4日晚6点多，2名逃犯在河北省平泉市榆树林子镇房申村二级小凤商店购买东西，随后逃往房申南山方向，之后再无消息。辽宁凌源到河北平泉大约有60多公里。
 <br/>
 <br/>
 10月5日，河北省平泉市官方发布一份报告称，当日上午9时许，河北平泉4名辅警追捕2名逃犯时，行至平泉市台头山镇三家村陈台子路段，过凹洼路时，车辆颠起失控侧翻，与公路右侧的路树相撞，导致2人死亡，2人受伤。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
</div>
