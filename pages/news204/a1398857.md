### 福建碳九污染现恐怖〝生化鱼尸〞记者戴口罩报空气达标
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月11日讯】福建泉州碳九泄漏事件发生数日后，当地水产养殖户惊恐发现，在事发地附近海域打捞上来的很多鱼都没有眼珠，其惨状十分恐怖。而当地政府为了粉饰太平，出动官媒人员前往事发地报导〝空气已达标〞，但现场的记者却戴着口罩，令当地民众大为愤怒。
 <br/>
 <br/>
 福建泉州泉港区一周前有近7吨的有毒化工原料碳九泄漏，导致该地区附近的水域和空气都遭受严重污染，不少当地居民因接触到水面的油污或吸入了弥漫着异味的空气而中毒病倒送医。在泄漏事故发生数日后，更严重的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环境污染.html" target="_blank">
  环境污染
 </a>
 后果日益突显出来。
 <br/>
 <br/>
 据香港《苹果日报》报导，因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碳九泄漏事故.html" target="_blank">
  碳九泄漏事故
 </a>
 而遭受损失最惨重的是当地的水产养殖户。因承托鱼排浮起的发泡胶被泄漏的碳九腐蚀，导致鱼排受损下沉，养殖户们设置的鱼排内的鱼不是漏网逃掉，就是因污染而死掉。
 <br/>
 <br/>
 近日，当地养殖户更发现从受污染的海域鱼排打捞上来的鱼，很多都没有眼珠，其状十分恐怖，怀疑这些鱼的眼球是因为遭受油污腐蚀而掉落。
 <br/>

 <br/>
 <br/>
 与此同时，碳九泄漏还导致当地的空气弥漫着浓烈的柴油味，迫使不少当地居民逃离避祸，因此有不少学生跟随家长逃离学校，即使坚持去学校上课的学生，最初几天也都是戴着口罩去上学。
 <br/>
 <br/>
 但地方当局为了淡化该事故的严重性，蓄意对外界夸大安全信息，甚至为了显示当地〝安全〞，竟强行要求逃学避难的学生返学上课，也不让学生戴口罩，结果导致许多学生感到身体不适，不但是呼吸道感觉受刺激，更有孩子因恶心而呕吐。
 <br/>
 <br/>
 当地一名姓肖的女士接受《苹果》采访时披露，她的弟弟与同学共11人，因吸入油污味后不适求医，接连引发许多学生请病假躲避毒气。校方9日特别召见数名学生家长，要求他们让孩子正常返学，〝免耽误学习进度〞。
 <br/>
 <br/>
 为了让家长放心，校方声称〝碳九没毒，容易蒸发。〞又告诉家长现在吹的是东北风，可吹散污染，空气中不会有异味。
 <br/>
 <br/>
 不过也家长质疑，〝要是风向改变什么办？〞许多家长感到十分无奈，难以抉择——〝不上学怕耽误学业，去了是拿生命当赌注。〞还有家长怀疑校方应是收到教育局指示，在拿孩子的安全作赌注来证明无污染。
 <br/>
 <br/>
 当地政府在事件发生后曾多次发通报称〝油污已全部清除完〞。但陆媒派出记者到事发当地去采访时却发现，直到11月9日受污染的海面仍有油污，被派遣去清理污染物的人员仍在海面放置吸油毡进行清理工作。而且事发地附近的空气仍刺鼻，令人感到喉咙难受、胸闷。
 <br/>
 <br/>
 与此同时，当地政府却只顾着派人用高压水枪在空气质量监测点附近大肆冲洗，令到现场采访的媒体记者质疑地方当局只是在忙着〝捂盖子〞 〝掩耳盗铃〞。
 <br/>
 <br/>
 当地官媒泉港广播电视台曾连日转发官方通报，声称泉港区空气及水质已〝达标〞，但该电视台的记者在场采访时却都戴着口罩，遭当地民众怒呛，〝既然达标，你们为何还要戴着口罩采访？〞
 <br/>
 <br/>
 资料显示，〝碳九〞的全称应为碳九芳烃（C9A），是石油提炼时获得的碳氢化合物，其成份主要包括三甲苯、异丙苯、正丙苯、乙基甲苯等。而碳九可分为裂解碳九和重整碳九两种，前者毒性相对较低，但后者的毒性就较高。
 <br/>
 <br/>
 泄漏事件发生后，曾有专家在接受陆媒采访时指出，碳九虽然不是剧毒，但也有一定毒性，
 <br/>
 食用被碳九污染过的动植物海产品，有中毒、致癌等风险，而中毒者会出现眼鼻喉和肺刺激，头痛、头晕等中枢神经和上呼吸道刺激症状，长期接触可致皮肤脱脂。此外，碳九属易燃危险品，可污染水、土壤和大气。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
