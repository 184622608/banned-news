### 广西公民砖场遭强拆 反被抓捕虐待
------------------------

<div class="wysiwyg">
 【新唐人2018年10月23日讯】广西北海市公民杨前，因家里
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_砖场.html" target="_blank">
  砖场
 </a>
 遭合浦县常乐镇政府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 ，和家人一同讨说法，却反被当局以寻畔滋事罪抓捕，并被殴打虐待，期间还遭到性侵。
 <br/>
 <br/>
 据杨前家属和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_律师.html" target="_blank">
  律师
 </a>
 10月23号告诉新唐人，今年5月，常乐镇政府土地征收办公室在未经签字同意情况下，将杨家有股份的砖窑厂用推土机铲平。事发后，杨前与母亲及弟弟相继去常乐镇政府讨要说法。
 <br/>
 <br/>
 杨前妹妹杨丽莎：〝5月中去的政府（讨说法），7月2号被抓的。政府就说他寻釁滋事。他们来抓人的时候连证件、拘留的手续这些都没有，而且他们是穿着便衣，来的两辆车还没有车牌的。〞
 <br/>
 <br/>
 代理律师杨在新第一次会见时，见到杨前被打惨况，脸部尤其眼睛、背部、腰部都有明显伤痕，手被手铐铐进很深的肉。杨前表示，在进看守所之前自己被公安机关反覆殴打。但是看守所的验伤记录却称杨前没有任何伤痕。此外，杨前还被强迫从事有污染的劳动，但这些警方和看守所都拒不承认。
 <br/>
 <br/>
 杨前代理律师杨在新：〝第三个就是被鸡奸，就是最严重的问题。鸡奸都两三次了，杨前都说出了什么时间段，我又及时的向合浦县看守所、合浦县检察院住所监察所投诉，要求他们调查取证，但是他们的回复就是说没有鸡奸。那没有鸡奸你拿证据说话，给那个时间段的视频给我们看，他又不给。〞
 <br/>
 <br/>
 律师最后一次会见杨前是在8月10号。
 <br/>
 <br/>
 杨在新律师表示，看守所以各种理由拒绝律师会见。
 <br/>
 <br/>
 杨丽莎：〝律师进去看他的时候说他已经拉了好多天的血，现在我们律师都没有得进去看他啊，也不知道现在人是生是死。〞
 <br/>
 <br/>
 杨在新：〝常乐派出所在拘留室的时候，就强迫他要骂共产党，要骂习近平，才给他吃饭、喝水、松手铐，因为手铐勒得很紧。那这种做法就是严重违法了。他就以此为由说杨前反共产党、反习近平。〞
 <br/>
 <br/>
 对于司法机关的种种违法行为，杨在新附上证据，向有关部门投诉并请求调查，至今没有任何回复。而且合浦县司法部门还向杨在新所在单位和律师协会施压、投诉。
 <br/>
 <br/>
 新唐人记者熊斌、王子琦、施宜君采访报导
</div>
