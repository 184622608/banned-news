### 突发：重庆红衣女闯幼稚园 乱刀砍伤14幼童
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】中国大陆再次爆出砍杀幼儿血案。四川一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_持菜刀.html" target="_blank">
  持菜刀
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红衣妇人.html" target="_blank">
  红衣妇人
 </a>
 26日早上闯到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 一间
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幼儿园.html" target="_blank">
  幼儿园
 </a>
 门口大肆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乱砍.html" target="_blank">
  乱砍
 </a>
 ，将14名3至4岁的幼童斩伤。
 <br/>
 <br/>
 中共警方通报，10月26日早上9时30分许，一名39岁的刘姓女子，闯到巴南区鱼洞巴县大道一间
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幼儿园.html" target="_blank">
  幼儿园
 </a>
 门口，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_持菜刀.html" target="_blank">
  持菜刀
 </a>
 大肆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乱砍.html" target="_blank">
  乱砍
 </a>
 ，由于学生们刚做完早操，正返回教室的14名学童被砍。
 <br/>
 <br/>
 根据网络流传的照片和影片可见，案发现场情况一片混乱，多名幼童整脸是血，大半部脸面遭到砍伤；还有幼童脖子遭砍伤，鲜血染满外套领口，还有学童被吓到大声哭喊，地面上血迹斑斑。
 <br/>
 <br/>
 另一段影片显示，一名穿红衣的妇女被警方抓捕时，仍不停反抗。多名幼童遭到突如其来的攻击后，头上缠上绷带，还有孩童躺在担架上被抬出来送医。
