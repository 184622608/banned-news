### 一个铜板惹祸？济州岛〝辱华事件〞掀风波
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_济州岛.html" target="_blank">
  济州岛
 </a>
 一家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_便利店.html" target="_blank">
  便利店
 </a>
 贴出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
  中国人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出入.html" target="_blank">
  出入
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁止.html" target="_blank">
  禁止
 </a>
 〞的告示，中国网民把其称之为〝辱华事件〞。目前告示已撤掉，便利店回应说，中国人过去常光顾这家便利店，期间有人将铜板丢在柜台上，店主难以接受才贴出告示。
 <br/>
 <br/>
 11月28日，有中国网友在中共四大门户网站之一的新浪网，上传了一段在韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_济州岛.html" target="_blank">
  济州岛
 </a>
 一家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_便利店.html" target="_blank">
  便利店
 </a>
 门口拍到的视频。视频中这家便利店入口贴着〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
  中国人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出入.html" target="_blank">
  出入
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁止.html" target="_blank">
  禁止
 </a>
 〞的汉字，这一消息迅速受到中国网民们的广泛关注，点击量一天内超过2.39亿次。
 <br/>
 <br/>
 上传视频的中国网友表示，〝店主侮辱中国人形象〞，〝我们是去买东西的却遭到这种待遇。特别是中国人如果不在济州岛消费，济州岛整体将赚不到任何收益〞。
 <br/>
 <br/>
 韩媒报导说，如果是便利店总部下达的命令，那么所有分店都应该贴这一告示，但只有这家店贴了相关告示，这说明是这家店主对中国人有反感情绪。 事发后，该便利店接到中国人的抗议电话撤掉了相关告示。
 <br/>
 <br/>
 据自由亚洲电台报导，涉事的7-11便利店是全球最大的一家连锁便利店，原本由美国人创建，2005年后成为日本公司，目前在韩国共有9200多个分店，是韩国的第二大连锁便利店。
 <br/>
 <br/>
 对于贴出〝中国人出入禁止〞的原因，便利店韩国总部负责人解释说，一些可能是居住当地的中国人，也可能是中国游客，在过去一段时间常光顾这家便利店，在购物过程中，顾客和店主发生了一些摩擦，因此店主贴出了〝中国人禁止出入〞的告示。
 <br/>
 <br/>
 该负责人说：在购物过程中，中国顾客将铜板丢在柜台上，这让店主难以接受，〝我们现在还无法确定孰是孰非，只是的确发生过不愉快的事情。若是原则性的错误，我们会改善，对于错误，我们会尽力纠正。〞
 <br/>
 <br/>
 济州岛是一个对中国游客免签证的旅游地。2017年韩国政府允许驻韩美军部署萨德反导系统之前，平均每月有30万左右的中国游客前来旅游。韩国部署萨德反导系统后，中国方面停止了到济州岛的团体旅游，目前抵岛旅游的人数每月约7万多。
 <br/>
 <br/>
 报导说，暂时还不很清楚贴出〝中国人禁止出入〞的原因，若根据韩国总部的说明，有些说不过去。虽然目前告示已撤下，但没有说明原因，也没有道歉。
 <br/>
 <br/>
 相关事件也让舆论联想到中国游客，在海外因各种不文明行为遭抵制事件，质疑7-11便利店贴出相关告示或因一些人还有其它一些不文明行为引店主反感。
 <br/>
 <br/>
 此前网上曾疯传一段3名中国游客在韩国乐天免税店互殴的视频。
 <br/>
 <br/>
 视频中，两女一男在化妆品柜台前为抢货大打出手。一名白衣女子和另一名黑衣女子扭打在一起，黑衣女子被掀翻在地，而另一名男子，则用脚猛跺倒地的黑衣女子的头。
 <br/>
 <br/>
 中国男子的野蛮行为引发韩国当地围观者的不满，旁边的两名男子强行将打人男子推开。
 <br/>
 <br/>
 许多大陆网民也看不下去自己同胞在海外的行径，希望中国游客，〝给中国人留点脸面吧〞。
 <br/>
 <br/>
 在日本也曾发生中国女游客因不文明行为，遭店主轰走事件。据日本富士电视台早间节目6月报导，两名中国女游客在日本大阪一家自助烤肉店用餐，用餐期间和店员发生争执，她们称，遭日本店员歧视，〝被骂了还被赶走〞。
 <br/>
 <br/>
 该店店员则表示，之所以说出上述一番话，是因为两名女游客不仅用餐超时，还把虾壳乱扔在地上，最后，店方没有收费，而是让两人赶紧离开，并直呼〝受不了了！〞
 <br/>
 <br/>
 相关视频在网上广泛传播，中国网友纷纷指责两名女子做法失礼，丢人丢到了国外，还〝恶人先告状〞。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
