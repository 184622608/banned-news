### 前甘肃书记王三运受审 牵出金融大鰐叶简明
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】河南地方法院10月11日公开审理甘肃省委原书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王三运.html" target="_blank">
  王三运
 </a>
 受贿一案，庭审画面显示，该案牵出已被查的中国华信掌门人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叶简明.html" target="_blank">
  叶简明
 </a>
 贿赂的证据。
 <br/>
 <br/>
 10月11日，中共全国人大教育科学文化卫生委员会前副主任委员、甘肃前书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王三运.html" target="_blank">
  王三运
 </a>
 案在河南郑州地方法院受审。
 <br/>
 <br/>
 王三运被控在1993年至2017年间，利用担任贵州省六盘水市委书记、贵州省委副书记、安徽省长及甘肃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_省委书记.html" target="_blank">
  省委书记
 </a>
 等职时，为相关单位和个人在入股银行、工程承揽和职务晋升等事项上提供帮助，非法收受财物共计6685万余元（人民币，下同）。
 <br/>
 <br/>
 在庭上，王三运当庭表示〝认罪悔罪〞。该案将择期宣判。
 <br/>
 <br/>
 央视公布的当天庭审画面显示，王三运案中第一起为收受被查的中国华信掌门人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叶简明.html" target="_blank">
  叶简明
 </a>
 贿赂的证据，其中第一项为王三运通过时任交通银行董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡怀邦.html" target="_blank">
  胡怀邦
 </a>
 ，为上海华信入股海南银行提供帮助的证据。
 <br/>
 <br/>
 第二项是提及时任国家开发银行（国开行）董事长胡怀邦。
 <br/>
 <br/>
 胡怀邦在9月27日，已被中组部宣布免去国开行党委书记、董事长职务。
 <br/>
 <br/>
 财新网3月曾刊文披露，叶简明被调查，其被查与王三运落马有关，但没有说详情。目前叶简明仍处于〝失联〞状态。
 <br/>
 <br/>
 有陆媒10月12日报导了有关叶简明行贿王三运细节。据报，银行是许多热衷于做资本运作的企业家最希望拿到的金融牌照，叶简明一直在寻找涉足银行的契机。
 <br/>
 <br/>
 对于叶来说，海南是最好的机会。自1998年海南唯一一家本地商业银行海南发展银行因为挤兑，被央行下令关闭清算，并停止一切经营活动后，很长的一段时间内，海南成为全国唯一一个没有法人银行的省份。
 <br/>
 <br/>
 这之后，海南一直试图努力向国家争取。为了搭上最后这班车，叶简明找到王三运成为了一个通道。时任甘肃省委书记的王三运，为了让叶简明掌控的华信拿到入股海南银行的通行证，他找到了时任交通银行董事长的胡怀邦帮忙。胡怀邦在海南银行组建过程中具有重要的话语权。
 <br/>
 <br/>
 最后，王三运通过胡怀邦为海南华信公司获得国家开发银行48亿美元授信额度提供了帮助。
 <br/>
 <br/>
 叶简明被指中共军方〝白手套〞，游走在商界、政界、文化界，背后的靠山包括曾庆红，张高丽等江派大员。
 <br/>
 <br/>
 王三运2011年12月任甘肃省委书记，主政甘肃长达5年，被指位高权重、〝欺下瞒上〞，与地方势力合作发财，在甘肃建立庞大政商关系网。2017年7月11日王三运落马。
 <br/>
 <br/>
 王三运是中共中办前主任令计划的铁杆。据称，王三运通过关系搭上令计划，在自己曾担任的领域最终取得令的充分信任，双方结为同盟。
 <br/>
 <br/>
 此外，王三运还因积极跟随中共江泽民集团迫害法轮功，被海外〝追查迫害法轮功国际组织〞及海外明慧网列为参与迫害法轮功学员的主要责任人。
 <br/>
 <br/>
 据报，王三运自2001年起先后在四川省、福建省、安徽省、甘肃4省担任要职。这期间，他主导且直接参与迫害法轮功。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
