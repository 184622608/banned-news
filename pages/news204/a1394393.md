### 人间蒸发事出有因？港媒惊传孟宏伟涉北戴河政变
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】国际刑警组织主席、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共公安部.html" target="_blank">
  中共公安部
 </a>
 副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 日前回国后，被中共纪检部门带走调查。港媒最新爆料称，孟宏伟可能涉嫌参与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北戴河政变.html" target="_blank">
  北戴河政变
 </a>
 。
 <br/>
 <br/>
 现有消息说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 涉及经济问题及其它贪腐问题，因而被中纪委调查。还有消息透露，与孟一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被调查.html" target="_blank">
  被调查
 </a>
 的，还有公安部一名部级高官。
 <br/>
 <br/>
 香港《苹果日报》惊爆孟宏伟可能涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北戴河政变.html" target="_blank">
  北戴河政变
 </a>
 。因为案件重大而紧急，北京当局才不顾影响而紧急对孟宏伟问话。该篇报导并无细节透露。
 <br/>
 <br/>
 截至目前，中共官方并没有对孟宏伟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 案做出证实或澄清，也没有给予国际刑警任何答覆。各大媒体的消息均无法得到证实。
 <br/>
 <br/>
 身为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共公安部.html" target="_blank">
  中共公安部
 </a>
 副部长兼国际刑警组织主席的孟宏伟，突然间〝人间蒸发〞，引发网友议论。有网友留言称，〝厉害了我的国〞，〝连国际刑警主席都可以绑架〞。还有人嘲讽，中国公民居然人间蒸发，怎么不见近日一天到晚对瑞典、英国厉声责骂的中共外交部出来〝严正交涉〞？
 <br/>
 <br/>
 <h4>
  事件回顾：
 </h4>
 <br/>
 10月5日，率先披露孟宏伟失踪的法国电视台《Europe 1》报导，孟宏伟是在9月29日留下妻小后，只身从国际刑警组织的里昂总部返回中国，之后音讯全无。
 <br/>
 <br/>
 法国警方透露，10月4日傍晚，孟宏伟的妻子向里昂巿警方报案。法国检警确认〝孟宏伟不是在法国境内失踪〞、〝全案不是单纯的刑事案件〞。
 <br/>
 <br/>
 香港《星岛日报》10月6日引述来自北京的消息说，孟宏伟在刚一抵达北京后，即被中纪委人员带走。据知他涉及经济问题及其它贪腐问题，其中包括孟宏伟违规在香港购置房产。但也有消息指，北京因孟宏伟贪腐违规购房而大动干戈扣押，惊动世界，逻辑上不成立。
 <br/>
 <br/>
 《南华早报》5日也表示，孟宏伟正在中国接受调查。但不知道孟宏伟被调查的原因及他人被关在何处。
 <br/>
 <br/>
 另据海外中文媒体明镜火拍透露，3日晚上，孟宏伟和另一名在公安部的副部级高官被中纪委和国家监委宣布，对他们进行调查。不过，中共当局还没有正式公布此消息。而另一名被调查的副部级高官的名字仍在核查。
 <br/>
 <br/>
 目前，孟宏伟的妻、儿仍留在法国里昂，由于孟妻早前曾收到社交网络及电话恐吓，法国警方已对其予以司法安全保护。
 <br/>
 <br/>
 <h4>
  孟宏伟被调查 亦在意料之中
 </h4>
 <br/>
 港媒报导说，如今，孟宏伟作为周永康的〝余毒〞被拿下，不令人意外。
 <br/>
 <br/>
 孟宏伟长期在中共公安部任职，被指与前中共政法委书记周永康关系密切。他2004年升任副部长时，时任公安部长就是周永康，是周的主要助手之一。
 <br/>
 <br/>
 中共公安部过去20年来，一直被江派人马把持，历任部长分别为江派要员贾春旺、周永康、孟建柱、郭声琨。
 <br/>
 <br/>
 习近平〝十八大〞上台后发动〝打虎〞运动，江派要员数十人应声
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 ，孟宏伟在近一年中也曾被两度削权。
 <br/>
 <br/>
 去年12月，孟宏伟被免去中共国家海洋局副局长、中共海警局局长职务。今年4月，孟宏伟又被免去中共公安部党委委员职务。孟宏伟接连被削权，中共官场已传其仕途〝很不妙〞。
 <br/>
 <br/>
 此外，中共公安部自1999年以来，跟随江泽民集团不断地造谣、诬陷、攻击法轮功。作为中共公安部副部长的孟宏伟，已被海外明慧网列入迫害法轮功学员的涉案责任人之一。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
