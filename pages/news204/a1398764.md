### 【禁闻】11月9日退党精选
------------------------

 【新唐人2018年11月09日讯】北方等三人发表了500多字的声明退出中共组织，来看他们的声明节选。
 <br/>
 <br/>
 北方等三人说：中共执政的几十年，生活在底层的中国人民饱受着中共的各种邪恶洗脑。没有公正的法律保障、没有公平。冤假错案越来越多，得不到解决。
 <br/>
 <br/>
 《宪法》第35条：〝中华人民共和国公民有言论、出版、集会、结社、游行、示威的自由。〞
 <br/>
 <br/>
 《宪法》第36条规定：〝中华人民共和国公民有宗教信仰的自由。〞
 <br/>
 <br/>
 这里特别要提到的是：修炼法轮功的人个个善良，对社会、对国家有百利而无一害。是法轮功救了亿万民众的生命。
 <br/>
 <br/>
 1999年7月，江泽民一意孤行的发动了对法轮功学员的迫害。并绑架全国各级政府部门及公检法司等工作人员参与犯罪。至今还在继续犯罪。我们退出邪党及其一切组织，做中华儿女、神的子民。
