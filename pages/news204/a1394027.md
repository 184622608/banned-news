### 美国副总统怒批：北京想要另一个总统
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】美国副总统彭斯发表演说，再度指控中方干预美国内政、企图削弱
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）的管治，更直言北京〝想要另一位美国总统〞。
 <br/>
 <br/>
 美东时间10月4日上午11时，美国副总统彭斯（Mike Pence）将在华盛顿智库哈德逊研究所发表演说。根据彭斯办公室公开的演说摘录，彭斯将从政治、外交、经贸及军事多方面批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的政策。
 <br/>
 <br/>
 政治方面，彭斯说，美国情报部门已确定州及地方的政府与官员，成为中共的目标，企图在政策上，导致联邦与地方出现分裂。他还指控中共，利用挑衅及高压方式，干预美国国内政策及政治，直言北京〝想要另一位美国总统〞。
 <br/>
 <br/>
 南海问题上，彭斯将重提美军驱逐舰，日前在南沙群岛航行期间，遭到中共军舰艇逼近一事。他重申：不会被中共的卤莽骚扰威吓，美国海军将继续在国际法的容许下，以及美国国家安全利益的要求下，在任何地方飞行、航行及行动。
 <br/>
 <br/>
 中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 及中共的〝一带一路〞战略计划，也是彭斯的演说重点，他批评中共以〝债务外交〞扩大在全球的影响力，中共向亚洲、非洲、欧洲及拉丁美洲的国家，借出数千亿美元的基建资金，这些国家只会倒向中方。
 <br/>
 <br/>
 彭斯还指控中共，迫使3个拉丁美洲国家与台湾断交。同时他也将不点名提到一间美国企业，近期被中共威胁，要该企业公开反对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的贸易政策，否则拒绝向其发出商业许可证。
 <br/>
 <br/>
 稍早前的9月27日，川普在联合国安理会上指控，中共企图干预美国11月将举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 。川普说，〝很遗憾，我们发现中国（共）一直试图干预，即将到来的2018年选举，将在11月举行的选举，以对抗我的政府。〞
 <br/>
 <br/>
 川普接着说，〝因为我是第一个敢在贸易上挑战中国（共）的美国总统〞，而且美国正在贸易方面取胜，所以中方不希望他或共和党赢得中期选举。
 <br/>
 <br/>
 当天稍后，川普又发推文说：事实上中共在《德梅因纪事报》（Des Moines Register）和其他报纸刊登宣传广告，看起来像是新闻报导。那是因为美国在贸易、开放巿场上打击了他们。这事结束的时候，农民们将会发财。
 <br/>
 <br/>
 川普9月17日在一份声明中宣布，自9月24日起对中国增课2000亿美元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，年底前设定为10%的水平，2019年关税将上升至25%。如果中共采取报复行动，美国将对约2670亿美元的中国商品征收关税。
 <br/>
 <br/>
 9月23日，中共官媒《中国日报》在爱荷华州第一大报《得梅因纪事报》的增刊中，刊登了四个版面的广告文章。
 <br/>
 <br/>
 其中一篇文章以《争端：总统做蠢事的后果》为题，指中美贸易摩擦，给该州大豆农民带来的影响是〝总统愚蠢行为的后果〞，致使该州大豆农民损失惨重。
 <br/>
 <br/>
 还有一篇文章被冠以《北京可以被世界引为典范》这样高调的标题。同时，中共官媒也有意软硬结合，特别用部分版面来介绍一本书。该书描写了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 1985年及2012年，到爱荷华州访问时发生的一些〝有趣的〞故事。
 <br/>
 <br/>
 对此，美国国家安全委员会前发言人维特尔（Tommy Vietor）评论称，中共玩弄相当狡猾的政治游戏。
 <br/>
 <br/>
 《美国之音》评论说，中共高层达成一种共识，即在11月的中期选举后，才同川普政府展开实质性的谈判，这是中共花钱在爱荷华州第一大报，刊登四版广告的主因。即试图影响在2016年美国大选中，支持川普的当地选民，在中期选举中的投票。
 <br/>
 <br/>
 而中共此次买下4页整版广告高调批评川普，正好送上〝攻击美国〞、〝干扰选举〞，甚至〝辱骂美国元首〞、〝煽动颠覆美国〞的最招眼证据。
 <br/>
 <br/>
 据美媒Axios披露，白宫将在数周内，执行一个大规模且全面制裁中共的措施。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
