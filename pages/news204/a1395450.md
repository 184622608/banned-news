### 美中迈向新冷战 华日：这正是川普想要的
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】美国政府近期的系列举措，正将中美贸易战扩展到涵盖多个领域的全面对抗。副总统彭斯（Mike Pence）痛批中共的演讲，更被解读为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 宣言〞。《华尔街日报》日前报导，中美正逐渐迈向新冷战，而这正是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普） 政府的真实战略意图。
 <br/>
 <br/>
 《华尔街日报》10月15日报导，针对中共多年来肆无忌惮的经济侵略行为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府正在从军事、政治、经济上从容不迫地展开反击，并暗示美中关系可能进入更为冷淡的新时代。
 <br/>
 <br/>
 报导指出，川普上台后的前18个月中，美中关系主要围绕朝核问题和贸易谈判展开。白宫这些高调动作掩饰了准备对中共采取更强硬立场所做的筹备工作。现在，随着中共在朝鲜问题上的作用减弱，以及美中贸易谈判陷入僵局，美国的真实战略开始浮出水面。
 <br/>
 <br/>
 报导还说，对美国白宫和其他政府机构高级官员的采访明确了一点：美国最近看似
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 行动的连环出击，并没有违背川普的对中政策宗旨，这恰恰正是川普政府想要看到的。
 <br/>
 <br/>
 川普和白宫官员近日的公开表态，似乎也印证了《华尔街日报》的上述判断。
 <br/>
 <br/>
 川普日前多次对媒体表示，北京很想和华盛顿谈判，但他认为北京还没有〝做好达成协议的准备〞。该表态暗示，华盛顿并不急于和北京达成和解。
 <br/>
 <br/>
 10月14日，白宫经济顾问库德洛(Larry Kudlow)接受《福克斯星期天新闻》主播华莱士的专访时，也没有刻意否定〝新冷战〞说法。
 <br/>
 <br/>
 库德洛表示，他觉得应该让后世的历史学家们来决定是否采用〝冷战〞一词描绘美中关系。不过，他明确表示，美中必须在所有领域建立起对等关系，包括贸易、安全和军事问题，〝这才是问题关键，正如总统多次说过的那样，我们必须捍卫美国的利益。〞
 <br/>
 <br/>
 13日，美国共和党参议员卢比奥（Marco Rubio）也在推特表示，〝冷战〞是一个过时的术语，但是美国必须展开与中共的对抗，否则将是〝灾难性的错误〞。卢比奥身兼〝美国国会及行政当局中国委员会〞（CECC）主席，在对美中关系上也是一个重磅角色。
 <br/>
 <br/>
 他说，〝我们（过去）允许中国（中共）进行欺骗和偷窃，因为希望他们可以改变，然而事实相反，他们利用所获得的力量来企图超越我们，再不与之对抗将会是灾难性的错误，并且会将21世纪定义为美国和民主的衰落时代。〞
 <br/>
 <br/>
 中共一直将美国的遏制行动解释为〝阻止中国崛起〞。但海外舆论认为，近些年的实际情况是中共在崛起而非中国，如果放任共产暴政扩张，让意图称霸全球的中共赶超美国，这对西方自由社会甚至整个世界都将是毁灭性的灾难。
 <br/>
 <br/>
 或许正是这个原因，多名美国高级官员的表态，都显示美国政府针对的是中共政权，而非中国人和中华民族。
 <br/>
 <br/>
 10日，卢比奥在CECC发布2018年度人权报告时曾表示，〝我们今天或任何时候对中国提出的批评，都不是针对中国人民。这个批评，针对的是中国共产党和中国政府，我们很敬重中国人民，中国文化长期以来，对人类做出了重大贡献。〞
 <br/>
 <br/>
 4日，彭斯在谴责中共的重磅演讲中，也刻意对〝中共〞和〝中国〞进行了区别，每指出中方一种恶劣行径时，都点名一次〝中国共产党〞，而非依惯例笼统地称呼〝中国〞或〝中国政府〞。
 <br/>
 <br/>
 彭斯在演讲的最后还表示，相信美国人民和中国人民之间的持久友谊，希望共同促进中美两国的繁荣，并称〝（中共治下的）中国人民理应得到更多〞。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
