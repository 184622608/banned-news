### 伊利举报直指前国级领导 内蒙一虎应声落马
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】内蒙乳业巨头伊利集团24日实名举报前董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑俊怀.html" target="_blank">
  郑俊怀
 </a>
 ，并指有前国家级和多名省部级高官充当其保护伞。25日中纪委宣布内蒙前人大副主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邢云.html" target="_blank">
  邢云
 </a>
 被查。未知两者仅仅是时间点巧合，还是确有关联。
 <br/>
 <br/>
 10月25日，中共中纪委网站通报，中共内蒙古自治区人大常委会原副主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邢云.html" target="_blank">
  邢云
 </a>
 涉嫌〝严重违纪违法〞，正接受中纪委审查和监察调查。
 <br/>
 <br/>
 公开资料显示，邢云出生于1952年，是内蒙古土生土长的官员，历任伊克昭盟盟长、鄂尔多斯市市委书记、包头市市委书记、内蒙古政法委员会书记、人大常委会副主任等职务，2016年卸任。
 <br/>
 <br/>
 就在中纪委宣布邢云落马的前一天，内蒙古伊利集团在官网刊出万字长文，实名举报该集团前董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑俊怀.html" target="_blank">
  郑俊怀
 </a>
 ，并表示已经正式向正在内蒙古的中共中央巡视组递交了相关举报材料。
 <br/>
 <br/>
 伊利在公开举报信中称，郑俊怀有〝原国家级领导、多位省部级、厅局级领导〞充当其保护伞，这些人合谋将伊利的国有资产非法转移，但郑俊怀在这些资产还没有落入他们名下前东窗事发，被判刑入狱。
 <br/>
 <br/>
 郑俊怀出狱后，搬出多名高官向伊利现领导层施压，要求配合将已转移的资产落到郑俊怀及其保护伞的名下。
 <br/>
 <br/>
 举报信中提到，2012年，时任内蒙古自治区检察院检察长邢宝玉曾找到伊利董事长潘刚，说：〝最高检一位领导要求伊利配合，把郑俊怀当年转移出去的资产帮助落实到他们名下〞。2015年，时任内蒙古检察院检察长马永胜也向潘刚提到，最高检某原副检察长（即上述那位〝最高检领导〞）也向他提了相同的要求。
 <br/>
 <br/>
 资料显示，2006年12月至2012年2月间，邢云任中共内蒙古自治区政法委书记，正是内蒙古检察院的直接上级。2012年2月后，已满副部级退休年龄的邢云退居二线，调任内蒙古人大常委会副主任，直到2016年卸任。
 <br/>
 <br/>
 伊利官网的举报信中没有透露更多郑俊怀保护伞的具体身份和姓名。不过，其递交给中纪委巡查组的材料相信应该有详细说明。
 <br/>
 <br/>
 至于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊利举报信.html" target="_blank">
  伊利举报信
 </a>
 中提到的〝原国家级领导〞，外界普遍猜测此人应该有内蒙古任职经历。而在内蒙古出生、仕途也在内蒙古起步的中共前政治局常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 被认为有重大嫌疑。
 <br/>
 <br/>
 2013年6月，网上曾流传一篇据称是中办官员撰写的文章《刘云山死都不知道是怎么死的》。其中提到，刘云山20多年在内蒙官场混迹打下的关系网，为其贪腐提供了便利条件，刘氏家族始终以内蒙为依托，大肆窃取国家财富，满足其无限贪欲。
 <br/>
 <br/>
 文章披露，〝2004年前，刘氏家族已经暗中实际掌控了大象投资公司，并且操作了对内蒙伊利股份法人股的操控，股改后，其掌握的伊利法人股时值超过数亿元。同时还控股了另一家内蒙上市公司——金宇集团的大部分法人股。而且，在内蒙掌控了相当多的矿产资源的所有权，包括煤矿、钼矿等等。〞
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
