### APEC公报难产 4中共官员强闯巴新外交部被赶走
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 出席第26届亚太经济合作会议（APEC）峰会，不但美俄首脑缺席，中方还被美国副总统当面抨击，相当尴尬。另外，APEC联合公报拟批北京，中方试图强闯主办国外长办公室施压，遭警察驱赶。最后联合公报未能公布。
 <br/>
 <br/>
 18日，为期两天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 结束，本应当天下午宣布的联合声明未能出台。主办国巴布亚新几内亚总理欧尼尔(Peter O'Neill)表示，巴新会过几天公布闭幕声明。
 <br/>
 <br/>
 中方外交官员则表示，由于中美贸易分歧，这次峰会没有就联合公报达成协议，因此依照惯例，将把问题交由主办国，由该国代表与会21国发表主席声明。
 <br/>
 <br/>
 据法新社18日引述三名知情人士报导，中共代表团4名官员17日一度强行闯入位于莫尔兹比港的巴新外长办公室，试图在最后紧要关头对峰会公报草案施加影响。
 <br/>
 <br/>
 消息说，但是巴新外长帕托（Rimbink Pato）态度坚决，一直拒绝接见，最后还叫来了警察坐镇，中共官员无功而返。
 <br/>
 <br/>
 帕托当时表示，〝外长单独与中国人谈判是不合时宜的。中方谈判官员明白这一点。〞
 <br/>
 <br/>
 不过，帕托在事后努力保持低调，对法新社否认曾发生此事。
 <br/>
 <br/>
 报导说，这不是中共代表团在峰会期间首次对东道主采取〝霸道〞行径。
 <br/>
 <br/>
 据路透社此前报导，今年10月的欧亚经济论坛期间，中方曾在10月18日晚间跟与会的欧盟和亚洲国家连夜进行谈判，最后迫使联合公报中删除了〝取消政府实施的不公平的扭曲市场的措施〞等针对北京的措辞。
 <br/>
 <br/>
 不过，这次APEC峰会上，中方施压遭遇了挫折。
 <br/>
 <br/>
 在这次峰会上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 和美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）的演讲针锋相对，中美激烈交锋的气氛笼罩会议。
 <br/>
 <br/>
 习近平在峰会演讲中，不点名批评贸易〝保护主义〞，并宣称〝无论热战、冷战和贸易战都没有赢家〞。
 <br/>
 <br/>
 而与会的彭斯则反击称，北京拒绝开放市场，才是真正的贸易保护主义。彭斯还辛辣地抨击北京的〝一带一路〞对沿线国家来说是一条〝不归路〞，并宣布美方将支援印太地区基础建设，但不会推出带有约束力的〝带〞，和单方向的〝路〞。
 <br/>
 <br/>
 美国及其盟友强势介入亚太地缘政治竞争，给该地区国家摆脱中共影响提供了机会。这些国家将面临重新选择。
 <br/>
 <br/>
 以峰会主办国巴布亚新几内亚为例，该国此前一直接受中共经济支援。但据外媒消息，美国、日本、澳大利亚和新西兰在峰会期间，已和该国签署协议，帮助其改善供电。此外，美国宣布将与澳大利亚一道，联合开发巴布亚新几内亚的一个海军基地。
 <br/>
 <br/>
 巴布亚新畿内亚总理欧尼尔在峰会闭幕式上表示，与会各国领袖了解目前彼此贸易关系紧张。
 <br/>
 <br/>
 菲律宾总统杜特尔特(Rodrigo Duterte)18日没有出席峰会闭幕晚宴。他表示，出席晚宴没什么显著的成效，甚至有点浪费时间。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：云涛）
