### 外媒：中共隐形地方债务40万亿 泰坦尼克将撞冰山
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】一项新的统计数据显示，中共各级
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地方政府.html" target="_blank">
  地方政府
 </a>
 的资产负债表外债务，可能高达40万亿元人民币，甚至更高。美媒形容这笔巨债相当于〝一座充满巨大信贷风险的债务冰山〞。
 <br/>
 <br/>
 综合路透社和彭博社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_标准普尔.html" target="_blank">
  标准普尔
 </a>
 全球评级公司（S&amp;P Global Ratings）周二（16日）在一份报告中表示，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地方政府.html" target="_blank">
  地方政府
 </a>
 可能累计了超过40万亿元人民币（合约5.8万亿美元）的资产负债表外债务，〝这是一座充满巨大信贷风险的债务冰山。〞
 <br/>
 <br/>
 标准普尔指出，如果包含各地方政府的〝隐藏〞债务，那么，政府债务与国内生产总值（GDP）的比率可能在2017年就已经达到60%的〝惊人〞高点。截至2017年底，中共在资产负债表上的未偿还政府债务总额，达到29.95万亿元。
 <br/>
 <br/>
 标准普尔表示，随着国民经济放缓，以及北京当局设定的地方政府债券发行配额，不足以支持那些能够促进地方经济增长的基础设施项目时，各地政府纷纷采取〝地方融资平台公司〞（Local Government Financing Vehicles, LGFV）来筹集资金。这些融资平台因此而积累了大量的〝隐藏〞债务，虽然当局一再声称采取措施减少此类债务，但目前进展极为有限。
 <br/>
 <br/>
 标准普尔分析师表示，这种做法使得地方政府融资平台在〝去槓杆化〞和〝将其业务转变为更典型的国有企业〞之间〝走钢丝〞。
 <br/>
 <br/>
 此外，在中美贸易冲突不断加剧的情况下，中国经济进一步疲软，外界对中共债务水平的担忧，尤其是对地方政府债务情况的担忧，日趋上升。
 <br/>
 <br/>
 花旗集团（Citigroup Inc.）驻香港首席中国经济学家刘立刚表示，〝我们认为，市场对中国债务的可持续性和金融风险的增加，感到更加担忧。〞他还指出，人民币正面临新的压力。
 <br/>
 <br/>
 随着中国各地的地方政府融资平台不断爆雷、地方政府加速违约的情况下，全球评级机构目前已将与中共有关的发行人的信用评级降低，以避免违约现象的增加。
 <br/>
 <br/>
 上个月，标准普尔将隶属于重庆、天津、江苏和湖南省地方政府的7家融资平台的长期发行人信用评级下调一级。
 <br/>
 <br/>
 与此同时，穆迪投资者服务公司（Moody's Investors Service）也降低了天津、江苏、湖南和湖北省各地方政府拥有的5家非金融企业和基础设施发行人的评级。
 <br/>
 <br/>
 而第三大国际评级机构——惠誉国际（Fitch）则早在去年就向中共提出警告，中国出现地方债违约可能性将会升高。
 <br/>
 <br/>
 标准普尔在这份报告中还指出，中共可能需要十年或者更长时间，来解决其隐藏的地方政府债务问题。
 <br/>
 <br/>
 <h4>
  大陆专家：中国地方政府债务 远高于40万亿
 </h4>
 <br/>
 中国的地方政府债务泡沫，被外界视为中国经济危机的爆破点，不过到底有多大，目前尚无定论。
 <br/>
 <br/>
 中共人大财经委委员贺铿5月19日称：〝国外机构估计，我们的地方债大概是40万亿，这应该是合理的，不高。但是地方政府就没有一个想还债的。〞
 <br/>
 <br/>
 贺铿还提到，〝政府信用是很差的，比企业可能更差！〞中国的〝金融秩序很乱，影子银行、互联网金融野蛮生长，隐藏风险很大，还有资产泡沫，房地产问题……当今中国金融泡沫明显。〞
 <br/>
 <br/>
 中共财政部7月18日的数据显示，截至2018年6月末，全国地方政府债务余额16.8万亿元，比国外估算低了一倍多，这其中的猫腻，就在于地方债务的〝隐形化〞。
 <br/>
 <br/>
 地方政府除了发行财政部许可额度内的政府债券之外，通过城市建设投资公司等各类平台融资的其它部分，则被视为地方政府隐性债务。
 <br/>
 <br/>
 据长江产业经济研究院陈柳的研究，对隐性债务中最主要的地方政府融资平台债务进行不完全估算，2016年底各省平台合计总负债为30.27万亿。
 <br/>
 <br/>
 清华大学财税研究所所长白重恩团队的调研结果为，截至2017年6月底，地方政府隐性债务约为47万亿元。
 <br/>
 <br/>
 中国地方政府的真实债务，如果按照白重恩团队的研究来估算，应该约为63.8万亿元；即使按陈柳团队的不完全估算，也超过47万亿。
 <br/>
 <br/>
 今年7月起，中国各地陆续出现的P2P网贷爆雷潮、房地产企业跑路、能源企业违约等，外界普遍解读为是债务风险失控，危机爆发的前兆。而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_债务危机.html" target="_blank">
  债务危机
 </a>
 中最大的〝地雷〞则是地方政府负债，而中共当局对此的态度却是首鼠两端，模棱两可。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
