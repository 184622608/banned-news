### 【禁闻】兰德报告 中共最想接触的11个国家
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】美国知名智库兰德公司最近发布长篇调查报告，披露了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东南亚.html" target="_blank">
  东南亚
 </a>
 、大洋洲、非洲等七大区域的战略，并找出了在这些区域中，中共最重视的11个国家。
 <br/>
 <br/>
 这份长达300多页的调查报告（《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
  一带一路
 </a>
 的黎明时分：中国在发展中国家》），披露了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 在全球七大区域的战略，并研究中共在这些发展中地区的政治、经济、外交和军事参与等，重点关注〝一带一路倡议〞。
 <br/>
 <br/>
 报告找出了每个区域中，引起中共重视的〝关键国家〞（pivotal state）或潜在的〝关键国家〞。
 <br/>
 <br/>
 其中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东南亚.html" target="_blank">
  东南亚
 </a>
 是中共在经济和政治利益上的最重要地域。它是中国最大的进口源和出口目的地，也是中共〝一带一路〞的重点区域。
 <br/>
 <br/>
 调查人员发现，东南亚的印尼（政治）、马来西亚（经济）、泰国（值得信赖）和越南（地缘战略风险）对中共极其重要。印尼被认为是中共〝海上丝绸之路〞项目的关键枢纽国家。
 <br/>
 <br/>
 而〝东南亚国家联盟〞的10个国家，都是中共主导的亚投行的成员国，亚投行被认为是给一带一路倡议提供资金的工具。
 <br/>
 <br/>
 从2003年到2014年，中共政治领袖对东南亚地区进行了94次访问，以及66次高级别的军事访问。
 <br/>
 <br/>
 中共还在该地区建立了25个〝孔子学院〞。
 <br/>
 <br/>
 位于欧亚大陆的俄罗斯，也是这些〝关键国家〞之一。中、俄除了寻求军事关系和能源方面的重要合作，在分裂主义和人权方面也分享共同利益。
 <br/>
 <br/>
 巴基斯坦，是中共在南亚利益的重要组成部分。中、巴签署的〝中巴经济走廊〞被认为是中共一带一路倡议的旗舰项目。报告指，中共一直利用巴基斯坦来牵制印度。同时，巴基斯坦还帮助中共维稳，尤其是对中国西部的维吾尔人。
 <br/>
 <br/>
 时事评论员田园博士表示，被中共当作〝关键国家〞，可能不是好事。
 <br/>
 <br/>
 时事评论员田园博士：〝中共最近在巴基斯坦动作连连，包括签订‘中巴经济走廊’协议，然后又向巴基斯坦大举提供贷款，但最后在经济危机的极大压力下，巴基斯坦最近公开向国际货币基金组织提出，给我提供救援，能把我从中共巨额贷款造成的经济危机里边挽救出来。〞
 <br/>
 <br/>
 报告指出，中共和伊朗在政治、经济和军事层面上进行合作。
 <br/>
 <br/>
 而哈萨克斯坦是中亚五国中经济实力最强的国家。
 <br/>
 <br/>
 田园表示，中亚是很复杂的区域，中共除了觊觎其丰富的资源，还想在〝新丝绸之路〞上，培植反美势力。
 <br/>
 <br/>
 田园：〝在哈萨克斯坦的西边就是伊朗。伊朗和美国对抗了很多年，中共想不但通过用撒钱、以开发的名义打通哈萨克斯坦，还能加强和伊朗之间的合作。即使在美国强力的制裁情况下，中共都一直向伊朗不停的购买石油和天然气。〞
 <br/>
 <br/>
 而南非强大的金融业和基础设施，使得南非一直是中国企业投资非洲的首选。北京特别指出，南非从20世纪90年代就开始与中共建立长期关系。
 <br/>
 <br/>
 拉丁美洲的委内瑞拉，由于有大量的石油储备，也吸引中共迅速对其大量投资，并提供军用物资和贷款。
 <br/>
 <br/>
 中共在大洋洲优先考虑澳大利亚和新西兰。这两个国家，是大洋洲地区与中共签署〝战略伙伴关系〞仅有的国家。
 <br/>
 <br/>
 田园认为，兰德公司的报告，把中共在全球布局，搞〝一带一路〞的战略方针脉络，清楚梳理了一遍。
 <br/>
 <br/>
 不过，旅美时事评论员郑浩昌提醒，兰德公司点名的关键国家中，没有北美和欧洲国家。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝这不是说欧洲就没有软柿子，只是因为欧盟一体化的特点，中共要影响某个欧洲国家，会牵一发而动全身，中共在这方面推进的难度比较大。而在北美，因为美、加的联系很紧密，所以只要美国的警惕性足够高，中共要动摇加拿大也是很难的。〞
 <br/>
 <br/>
 郑浩昌表示，中共对外扩张不仅有野心且很有手段，它不是把网全面撒开，而是有重点的推进，西方社会如果对此掉以轻心，很容易中招。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/钟元
