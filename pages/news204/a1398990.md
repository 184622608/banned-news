### 大陆双11掀购物潮 山寨名牌网上泛滥
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】大陆双11网上购物节反应热烈，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山寨.html" target="_blank">
  山寨
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_仿冒品.html" target="_blank">
  仿冒品
 </a>
 充斥其间，淘宝网声称全力打击仿冒牌货，但消费者上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴.html" target="_blank">
  阿里巴巴
 </a>
 （Alibaba）的淘宝App，便可找到山寨品。使中共保护
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 成为一句空话。
 <br/>
 <br/>
 大陆窃取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 遭欧美企业长期诟病，同时这也是美中贸易的核心问题之一。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴.html" target="_blank">
  阿里巴巴
 </a>
 因被指控属下网购平台兜售假货，而被美国制裁。
 <br/>
 <br/>
 据法新社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双十一.html" target="_blank">
  双十一
 </a>
 的促销期间，各家网购平台肆无忌惮推销名牌抄袭翻版品，打脸中共信誓旦旦打击侵犯知识产权的承诺。
 <br/>
 <br/>
 在双11销售的产品中，几乎世界名牌产品都被另类搬上双11的购物狂欢节平台。这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山寨.html" target="_blank">
  山寨
 </a>
 品或是直接抄袭，或是改头换面，或是稍微改动一个字，一个发音，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_仿冒品.html" target="_blank">
  仿冒品
 </a>
 涉及各个领域。从高级时装，到名贵手表、鞋包，再到各种名酒，法国葡萄酒成为酒类山寨造假的第一入选者。
 <br/>
 <br/>
 在网购平台上可以轻易找到外观与知名品牌阿迪达斯（Adidas）相似的鞋款，商品名叫〝阿巴比斯〞（Ababis）。〝阿巴比斯〞（Ababis）运动鞋，以4条线取代阿迪达斯的三条线设计，一双只要人民币39元。
 <br/>
 <br/>
 还有著名的时装品牌巴黎世家（Balenciagas），神似的山寨版叫Balenciaca。假的路易威登（LV）名牌包一个只要17美元。美国服饰品牌凯文克莱（Calvin Klein），其仿冒品叫做〝凯文卡尼〞（Caiwen Kani），此外，还有盗版的乐高（Lego）玩具等等。
 <br/>
 <br/>
 法新社的报导指，美国总统特朗普严厉指责中国贸易问题，其中一个聚焦点就是批评中国侵犯知识产权。许多欧美中小企业产品被剽窃山寨的次数频率与范围有增无减，而这些企业因为没有财力聘用国际司法追究维权，而只能眼巴巴看着被山寨被改头换面抛售。面对规模宏大的被山寨的产品国际大展，无疑是再打中共保护知识产权不力的脸。
 <br/>
 <br/>
 普通民众分不清真假，许多人往往买了假货而不自知。但也有人因假货与真品没有多少区别而沾沾自喜。在北京街头，有些受访民众也分不清记者手上哪个是真品。37岁的王姓受访者面对盗版乐高玩具时表示：〝我会买。我想所有商品都有点相互抄袭。只要适合孩子，我可以接受（山寨品）。但前提是商品品质要好。〞
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
</div>
