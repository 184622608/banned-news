### 广西人贩子入室抢男童 母亲被殴打绑在厕所（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】广西日前发生一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人贩子.html" target="_blank">
  人贩子
 </a>
 入室抢走儿童事件，引爆网络舆论。据网传视频显示，孩子的母亲向警方哭诉，一名蒙面人闯入她的家中，对她殴打并绑在屋内卫生间。等她挣脱出来后，发现3岁半的儿子已被绑匪掳走。所幸，几天后疑犯即被抓捕，男童也安全获救。
 <br/>
 <br/>
 11月2日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广西河池市.html" target="_blank">
  广西河池市
 </a>
 安阳镇惊传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人贩子.html" target="_blank">
  人贩子
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入室绑架.html" target="_blank">
  入室绑架
 </a>
 儿童事件，导致当地民众人心惶惶。被抢走孩子的是该镇镇北社区建材小区住户彭女士家。
 <br/>
 <br/>
 据网传视频显示，彭女士赤脚站在家门口，正在向警方哭诉孩子被抢走的经过。据彭女士说，当日上午8点多钟，一名蒙面男子闯入她的家中，殴打她，并将她捆绑，关在卫生间，等她逃出来后，却发现3岁半的儿子已被绑匪掳走。彭女士连忙报警。
 <br/>
 <br/>
 在视频中，彭女士一边述说一边大哭，哀伤地呼喊〝我的儿子呀！〞
 <br/>
 <br/>
 据彭女士的女儿介绍说，自己的弟弟平时很乖巧可爱，非常讨人喜欢。掳走弟弟的蒙面人，还取走了家里安装的监控主机，怀疑是熟人作案。
 <br/>
 <br/>
 据陆媒报导，当地警方称，疑犯于11月4日晚上被抓捕，被绑架的男童也已安全获救。但警方未在通报中交代该案详情。
 <br/>
 <br/>
 有当地网民透露，当时该蒙面男子入屋后使用喷药将男童掳走，〝妈妈被殴打绑起来，真的很吓人〞。
 <br/>
 <br/>
 人贩子拐卖儿童事件在大陆频频发生。人贩子甚至猖獗到大白天入室抢儿童，或当街抢儿童，大陆民众对此忧心忡忡。
 <br/>
 <br/>
 早在2009年11月5日，中共央视曾播出一档关于拐卖儿童的节目。节目中，在东莞寮步镇，有一个9个月大的孩子叫叶锐聪，他在自己家门口被人从家人的怀里给抢走了。
 <br/>
 <br/>
 叶锐聪的母亲邓惠东说，当时一辆面包车倒过来，我女儿就把她弟弟抱在怀里了，但是只有几秒钟，面包车打开车门，人贩子在她眼皮子底下，从她11岁的女儿怀里抢走了9个月大的儿子。她虽然拚命追赶，但人贩子仍然带着他的儿子逃之夭夭。
 <br/>
 <br/>
 据30多位东莞失踪儿童的家长说，无论是放学路上、超市、自己的家门口，人贩子几乎无孔不入，他们偷走或抢走的孩子，最小的刚生下来几天，最大的也只有9岁，绝大多数集中在3到7岁之间。
 <br/>
 <br/>
 据网络热传的一段视频显示，一位爷爷带着孙子在路边散步，一名女人贩子走上来就去抱他的孙子，爷爷连忙阻拦，这时旁边数名同伙，一起涌上来，把爷爷按倒在地上，3名女子抱着男童离去。
 <br/>
