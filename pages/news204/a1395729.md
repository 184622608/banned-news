### 传媒系统腐败重灾区 湖南广电又一副台长被查
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】湖南省纪委监委10月17日发布消息，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南广播电视台.html" target="_blank">
  湖南广播电视台
 </a>
 副台长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄伟.html" target="_blank">
  黄伟
 </a>
 涉嫌严重违纪违法，接受调查。早前，湖南广电另一位副台长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗毅.html" target="_blank">
  罗毅
 </a>
 ，因贪腐等罪名被逮捕。湖南被指是江泽民派系一大窝点，也是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传媒系统腐败.html" target="_blank">
  传媒系统腐败
 </a>
 重灾区。
 <br/>
 <br/>
 现年51岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄伟.html" target="_blank">
  黄伟
 </a>
 ，是湖南本土人，曾在湖南经济电视台、湖南广播影视集团任职，2011年出任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南广播电视台.html" target="_blank">
  湖南广播电视台
 </a>
 副台长。他是继湖南广播电视台女副台长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗毅.html" target="_blank">
  罗毅
 </a>
 被抓后，又一名落马副台长。
 <br/>
 <br/>
 2016年10月，罗毅因〝贪污罪、受贿罪、非法经营同类营业罪〞被逮捕。2018年6月1日，罗毅被指贪污受贿200万元人民币，获刑五年六个月。
 <br/>
 <br/>
 罗毅曾历任中共湖南电视台信息频道办公室主任、图文编播部主任，湖南广播电台交通频道总监，湖南广播电台副台长、台长等职，2010年3月起任湖南广播电视台党委委员、副台长。
 <br/>
 <br/>
 从履历上看，黄伟与罗毅同属于一个〝领导班子〞，共事5年。外界推测，黄伟此番落马很可能与罗毅有相当的关联。
 <br/>
 <br/>
 另外，中共湖南省委巡视组曾在2017年2月，对湖南广播电视台展开〝回头看〞巡视，称该电视台存在〝党的领导〞不强，导致节目事故频发，重收视、轻导向，舆论导向存在偏差。
 <br/>
 <br/>
 湖南被指是江派的一大窝点，也是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_传媒系统腐败.html" target="_blank">
  传媒系统腐败
 </a>
 重灾区。在罗毅之前，湖南日报社系列腐败案件中，包括原党组成员、社务委员薛伯清，原社长覃晓光，原总经理皮林，原社务委员刘树林等10人被查。《湖南日报》报业集团原副总经理徐勇已被判刑5年6个月。
 <br/>
 <br/>
 2018年1月，湖南电广传媒董事长龙秋云因受贿罪被逮捕，其管理层被指存在贪腐窝案，多人或将入狱。
 <br/>
 <br/>
 近年来，中共宣传系统掀起反腐风暴，多名高官涉嫌贪腐落马。据中共官媒此前报导，电视剧购销、工程建设和广告投放，是广电系统三大腐败重灾区。以电视剧购销为例，之所以此处能成为腐败重灾区，与买卖双方不对等、电视剧市场的不透明以及流程监管缺失脱不了关系。作为全球电视剧第一播出大国，中国的电视剧播出集数赶不上产量。
 <br/>
 <br/>
 2015年，安徽广播电视台原台长张苏洲贪腐案曝光时，有网友曾愤怒表示，张苏洲的贪污，导致了安徽卫视的没落。2016年9月19日，张苏洲因受贿、贪污上千万人民币被判14年徒刑。同月，原副台长赵红梅也因贪贿750万被判刑12年。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
