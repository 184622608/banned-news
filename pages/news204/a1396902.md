### 网贷平台难友杭州维权 却被当局押走
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_P2P网贷平台.html" target="_blank">
  P2P网贷平台
 </a>
 在今年7月份起出现倒闭潮，杭州为重灾区之一，10月25号，杭州的多个网贷平台的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难友.html" target="_blank">
  难友
 </a>
 到浙江省政府维权，现场大批警力戒备，将难友们全部用大巴押送黄龙体育中心维稳，让难友们大失所望。
 <br/>
 <br/>
 杭州众多爆雷的网贷平台，包括草根投资、投融家、多多理财、萌小薪、齐天优贷、人人爱家、可溯金融、财富中国等十几家平台的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难友.html" target="_blank">
  难友
 </a>
 ，25号聚集在位于杭州市西湖区环城西路的省政府前维权，现场大批警力戒备，警察比难友人数还多。
 <br/>
 <br/>
 涉及176万人，总金额236亿元的〝人人爱家〞难友表示，〝人人爱家〞被查出，投假标，借款人的信息都是假的，涉嫌诈骗，他们曾多次到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杭州维权.html" target="_blank">
  杭州维权
 </a>
 都没有结果。
 <br/>
 <br/>
 〝人人爱家〞难友李女士:〝最前面他的市政府，他们也不监管，我们都是把全部的积蓄都投进去，几十万啊，我们的一生就存了那么多钱，我们现在都已经六、七十岁了，结果一下给他们黑了，已经三个多月了，具体的资金流向，也没有说，我们去了杭州合理合法的维权，也没有说怎样还钱。〞
 <br/>
 <br/>
 现场视频显示，警察力劝难友们上车，用大巴先后拉走三车人 到黄龙体育中心，各平台的难友由不同的人接待，但名为接待，实际是维稳，让难友们大失所望。
 <br/>
 <br/>
 〝投融家〞难友匿名:〝这平台杭州政府支持的嘛，我们一块的难友他去杭州维权不顺利的，说实话这点钱我已经不打算要了，就当是一个教训吧!你怎么办呢?这个你找谁说理啊!〞
 <br/>
 <br/>
 涉及13万人，97亿资金的〝草根投资〞，7月31号爆雷 ， 10月19号以〝非法吸存〞立案 ，难友们不服，发出〝出借人联名追责诉求书〞，声明他们是借钱给草根平台，不属于非法集资。并指控该平台涉嫌假标，合同诈骗、金融诈骗等行为。
 <br/>
 <br/>
 〝草根投资〞难友夏先生:〝当初宣传的比较好，上市公司的背景，融资也特别多，还有政府官员去他的公司考察，国资背景啊，那些都有，宣传的也比较大，所以都相信他了，好多人都把钱放进去。之前我们难友已经在报案了，他们的经侦就说给公司3个月时间。〞
 <br/>
 <br/>
 22号，草根平台难友去省信访办维权，有5人被抓，其中一人以〝寻釁滋事〞的罪名刑拘7天。 网上视频显示，这次来杭州维权的外地草根难友，因为没钱，夜宿马路 。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
</div>
