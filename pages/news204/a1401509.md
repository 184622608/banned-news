 2. 上海维权人士喊冤被打断肋骨 控告无果
 <br/>
 <br/>
 上海维权人士
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万文英.html" target="_blank">
  万文英
 </a>
 今年6月27号在上海市公安局门口,  只因喊了一声〝冤枉〞，就被警察暴力殴打，导致4根肋骨骨折。万文英事后多次向当地检察院控告, 都无结果。11月27号，她再次用特快专递向中共公安部、最高检察院等机构邮寄刑事控告状。
 <br/>
 <br/>
 3. 武汉访民进京上访 被批捕
 <br/>
 <br/>

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡正秋.html" target="_blank">
  胡正秋
 </a>
 10月24号在北京上访时，被警察查验身份证扣押拦截，交给地方截访人员押回武汉之后，又被以〝寻釁滋事罪〞刑拘。日前，警方通知家属，他已经被以同样罪名批准逮捕。
