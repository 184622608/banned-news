### 习近平视察大桥人工岛 干扰车紧随防遥控炸弹
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月25日讯】中国国家主席习近平日前出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥开通仪式.html" target="_blank">
  港珠澳大桥开通仪式
 </a>
 后，曾坐车到港珠澳大桥视察。在此过程中，除了有飞虎队快艇直升机的严密巡逻外，香港方面还出动了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全频无线电干扰车.html" target="_blank">
  全频无线电干扰车
 </a>
 以防发生遥控爆炸。
 <br/>
 <br/>
 当地时间10月23日上午9时28分，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥开通仪式.html" target="_blank">
  港珠澳大桥开通仪式
 </a>
 在珠海旅检大楼举行。习近平出席仪式并宣布大桥正式开通。其后，习近平在港澳工作协调小组组长副总理韩正、行政长官林郑月娥、澳门行政长官崔世等人的安陪同下，坐小巴到港珠澳大桥视察，途中经过大桥的隧道段。习一度登上人工岛西侧平台，遥望大桥全貌和香港机场。
 <br/>
 <br/>
 据香港苹果新闻报导，为了防止这次开通仪式上发生意外，香港警方从当天清晨7时就已出动警车在港珠澳大桥、人工岛及三地沿岸巡逻，为通车仪式作最后准备。香港东涌沿岸所有码头都有警员驻守，东涌码头对开海面亦有水警小艇戒备，其间更有5艘疑似飞虎队快艇在海面上高速驶过，同时还有直升机在桥面上空盘旋监视。
 <br/>
 <br/>
 在习近平出席开通仪式及巡视大桥的过程中，香港警方为防止意外发生，除了上严密的海陆空巡逻警戒外，还出动了一辆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全频无线电干扰车.html" target="_blank">
  全频无线电干扰车
 </a>
 ，赶在习近平车队到达东人工岛参观前，驶至香港口岸的桥上护驾，防范遥控炸弹及遥控飞机。
 <br/>
 <br/>
 据港媒的报导，所有前往现场进行报导的媒体人员都会被警员截查，通过登记证件及检查随身物品后，才被准许在远处拍摄。
 <br/>
 <br/>
 据了解，早在2017年6月29日，习近平前往香港出席“香港回归20周年”庆典活动时，香港警方就已经出动过无线电干扰车沿途〝护驾〞。
 <br/>
 <br/>
 据港媒的介绍，这辆灰黑色的无线电干扰车，车顶装有10多条天线，可干扰及侦测多种无线电波，如遥控炸弹及遥控飞机等，以防突发事故。同时为免警方执勤受阻，该车还可设定部分指定的无线电频谱不受干扰。
