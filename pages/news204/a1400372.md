### 潜江访民向省府提出人身安全保护申请
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】11月19号，湖北潜江市十多名访民集体到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖北省信访局.html" target="_blank">
  湖北省信访局
 </a>
 ，递交〝个人安全保护申请书〞，他们还向省政府快递了同样的申请书。据了解，这些访民在维权路上遭到当局打压，很多人被非法绑架、酷刑折磨、殴打致残、24小时监控和软禁等。
 <br/>
 <br/>
 湖北
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_潜江访民.html" target="_blank">
  潜江访民
 </a>
 11月22号表示，三个月前，他们向市政府提出个人安全保护申请，法律规定的两个月期限到了，市政府没有答覆，他们到当地法院控告，也不立案，于是在19号前往省政府递交申请书。
 <br/>
 <br/>
 湖北潜江访民张玉还:〝因为我们这些年在维权路上遭到关押啊、绑架呀，有很多人都打残疾，担心自己的人身安全，就向市政府提出申请，没有给我们任何答覆，我们就向中院提起诉讼，中院也没有给任何答覆，就向省政府提起申请。〞
 <br/>
 <br/>
 湖北潜江访民彭峰:〝因为我举报关于腐败（案件），过程中遭到打击，选举方面也遭到打击，限制我们的自由啊，非法绑架我们，买了火车票都不让我们上啊，我的房子也被他们砸毁啊。我们大概有十来个人，提出人身财产安全保护 ，按照程序他没有给我们做出答覆，我们现在向上一级湖北省政府人民政府提出了。〞
 <br/>
 <br/>
 据《维权网》报导，在长期维权过程中潜江访民大部分遭到非法绑架，非法关押私设黑监狱，酷刑折磨关押。有些被非法送到精神病医院关押一年多的访民，至今问题没有得到解决。
 <br/>
 <br/>
 湖北维权人士伍立娟:〝我们长期被软禁、打压，失去自由，我家门口有好几个人24小时监控，所有访民都是这样的。曾祥军关押50多天，打了50多天，一跪跪十几个小时、20几个小时，叫他写信访条款，70来岁的人，眼睛都看不见，他们还叫人家背，背不会就打，打得眼睛鼻子都有血。 〞
 <br/>
 <br/>
 潜江访民在申请书中，请求潜江市政府，市公安局依法对申请人的人身安全予以保护，防止腐败官员动用黑警阻挡申请人的举报控告，禁止官警联手，雇佣涉黑人员参与拆迁、占地、截访，或授意社会流氓、黑恶人员绑架、殴打、威胁、辱骂、跟踪申请人等一切行为。
 <br/>
 <br/>
 新唐人记者熊斌、陈杰采访报导
</div>
