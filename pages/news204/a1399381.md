### 鲁炜忏悔内容首曝光 称自己生活作风让妻绝望
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】中共原中宣部副部长、网信办主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲁炜.html" target="_blank">
  鲁炜
 </a>
 被公审后，又爆出新消息。近日，鲁炜的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_忏悔书.html" target="_blank">
  忏悔书
 </a>
 首次被官方曝光，他在忏悔书中提及，其妻因其生活作风问题，对他完全绝望。
 <br/>
 <br/>
 中共官方举办的〝改革开放40周年大型展览〞目前正在北京展出。展板资料显示，中共十八届中央批准立案审查的省军级以上党员干部，及其他中管干部440人，包括十八届中央委员、候补委员43人，中央纪委委员9人。
 <br/>
 <br/>
 在所谓的反腐成果展区的一角，展出了多名落马高官的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_忏悔书.html" target="_blank">
  忏悔书
 </a>
 手稿或处分决定。在盖有〝中共中央纪委〞红印章的A4纸上，展出中共原保监会主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_项俊波.html" target="_blank">
  项俊波
 </a>
 的处分决定，及其签字红手印；还有甘肃省委原书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王三运.html" target="_blank">
  王三运
 </a>
 、中宣部原副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲁炜.html" target="_blank">
  鲁炜
 </a>
 的等人的忏悔书手稿。
 <br/>
 <br/>
 这是鲁炜的忏悔书《我的忏悔》首次被官方曝光。
 <br/>
 <br/>
 鲁炜在忏悔书中写道：〝我在政治上、经济上、工作上、生活上，都犯下了严重的不可饶恕的错误……我严重违反六大纪律，‘七个有之’条条都犯。〞
 <br/>
 <br/>
 鲁炜还在忏悔书中提及其妻子和儿子。他写道：〝我的生活作风问题使她受到了很大伤害，我们经常为此吵架，她对我完全绝望，……。’而今，正是一语成谶！我的儿子，刚刚度过30岁生日。在他人生的路上，我没有做好人生的榜样，也没有尽到一个父亲的责任。〞
 <br/>
 <br/>
 10月19日，浙江宁波法院公审鲁炜受贿一案。法院指控，鲁炜在过去15年中，在宣传系统内利用职务之便，收受财物摺合人民币3,200多万元。鲁炜案择日宣判。
 <br/>
 <br/>
 鲁炜被视为大陆〝网际网路大总管〞，曾在新华社工作长达20年，官至副社长，2011年起，调任北京市委常委兼宣传部长，北京市副市长。2013年改任网信办主任，后兼任中宣部副部长。2016年被免去其它职务，只保留中宣部副部长一职。
 <br/>
 <br/>
 2017年11月21日深夜，中纪委网站发布消息，鲁炜正在接受审查。
 <br/>
 <br/>
 2018年2月13日，中纪委官网通报，鲁炜因严重违纪被双开。并怒批鲁炜频繁出入私人会所，大搞特权，作风粗暴、专横跋扈、以权谋色、毫无廉耻。
 <br/>
 <br/>
 有美媒援引消息称，鲁炜在北京任职时，长期在北京日报大厦设有吃喝使用的专门包间，在其落马前，曾有一批官员向中纪委投书，指鲁炜在北京任职时，与其发生性关系和被其强行猥亵与强奸的女人就有60多人，且有名有姓。
 <br/>
 <br/>
 而鲁炜最大的丑闻是，他曾参加〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人奶宴.html" target="_blank">
  人奶宴
 </a>
 〞。新华社记者周方5年前曾实名举报，中宣部正部级官员曾受一名大老板宴请，席间，人奶是作为一道菜上来的，在每位食客旁都来了一位全裸的美艳少妇，做为宴席上最贵的一道〝菜〞，每位标价5,000元。
 <br/>
 <br/>
 举报信还说，中共高官非首次出席同类宴会，而请客的大老板目前还在狱中。
 <br/>
 <br/>
 同年8月，新华社内部消息称，周方已被控制，指使打压周方的是新任的网信办主任鲁炜，而鲁炜也正是涉〝人奶宴〞的〝正部级宣传部门高官〞。
 <br/>
 <br/>
 另外，当时有参与〝人奶宴〞的宣传口高官，还指向长期掌控中宣部的前江派常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 。
 <br/>
 <br/>
 网上署名庄丰的《挺铁流！吁习近平尽快处理〝淫棍〞刘云山》一文披露，有人举报刘云山，曾在北京高级会所的富商宴会当众吃美女的人奶。
 <br/>
 <br/>
 据说以刘云山做后台的前央视主持人芮成钢交代，这种事情的确发生过多次。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：李泉）
