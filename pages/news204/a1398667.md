### 朱军猥亵案再生事端 女当事人突遭网路暴力
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军.html" target="_blank">
  朱军
 </a>
 猥亵案一波三折，受害女子近日遭遇网路暴力，收到大量针对其外貌的辱骂，疑似朱军方面的网络水军。
 <br/>
 <br/>
 11月8日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军.html" target="_blank">
  朱军
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵.html" target="_blank">
  性侵
 </a>
 案女当事人微博发文称，自从在法院露面后，收到大量的辱骂，最近网路又出现许多类似水军通稿的内容，拿自己的外貌替朱军洗白。
 <br/>
 <br/>
 认证为编剧的当事人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弦子.html" target="_blank">
  弦子
 </a>
 （化名），其微博名为〝弦子与她的朋友们〞，粉丝11万，弦子在8日的博文中说：〝从开庭那天开始，就收到足够多关于外貌的辱骂了，说不困扰是不可能的，只能安慰自己每个人判断标准不一样，选择站出来就要有心理准备。不过最近这种类似水军通稿的内容也出现了，麻烦大家看到类似的推送把链接发给我，我承受得住〞。
 <br/>
 <br/>
 博文还说，在收到的数千评论中，多数网友支持〝弦子与她的朋友们〞。但是，最近有类似水军通稿出现，多数攻击女当事人的外貌，疑似以此替朱军解围。
 <br/>
 <br/>
 8日的微博中附两张截图，截图内容显示，有两家公众号的文章标题分别为：〝朱军性侵案开庭，女主亮相长这样？网友：我相信朱军〞〝朱军性侵案开庭，女主角第一次亮相，网友见后：这样子可能吗？〞
 <br/>
 <br/>
 11月6日，弦子在微博中说：〝即使官司打赢，受害者也不是赢家，灯光落下后，留给她们的，是他人即使伸出援手也无法解决的，千疮百孔的生活——〝希望赢吗？/我们更希望它从未发生。〞
 <br/>
 <br/>
 上月25日，朱军性侵案在北京海淀法院开庭，女当事人现身，朱军未露面。当时，女当事人向法院提交了证据和录音。
 <br/>
 <br/>
 朱军性侵案源自今年7月的一份举报，微博用户〝麦烧同学〞称，4年前朱军在化妆室对一名女生做出猥亵行为。8月15日，朱军委托发布律师声明称，猥亵女生的信息不实。9月19日，朱军正式起诉微博、女当事人和〝麦烧同学〞，其代理律师提出四项诉讼请求，其中求偿65万元（人民币，下同）。随后，女当事人起诉朱军，要求其赔偿6万元，其中精神抚慰金5万元。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
