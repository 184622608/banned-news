### 朱军性骚扰案开庭在即 对方申请朱军出庭
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】近日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军.html" target="_blank">
  朱军
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性骚扰.html" target="_blank">
  性骚扰
 </a>
 案爆料人〝麦烧同学〞说，9月已经收到朱军以及代理律师的起诉消息。10月25日，将在法庭进行证据交换，目前已通过律师向法院申请朱军本人出席。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军.html" target="_blank">
  朱军
 </a>
 案爆料人〝麦烧同学〞9月25日发文称，已经收到朱军以及代理律师的起诉消息。10月15日，他再发微博表示，10天后（10月25日）进行证据交换，目前已经通过律师向法院申请朱军本人出席。
 <br/>
 <br/>
 〝麦烧同学〞还表示，另外他要求北京市公安局，公开2014年案件发生时的全部案件材料与证据，他在微博直言，这两个申请难度都不小，只能等待法院的判断
 <br/>
 <br/>
 该案当事人弦子透露，朱军在9月19日开始以微博、弦子、〝麦烧同学〞三方对象发起了诉讼，朱与代理律师提出4项诉讼请求，分别是删除弦子、〝麦烧同学〞的微博账号，并披露相关博文数据，爆料人与当事人在报纸与微博道歉，赔偿朱军65万元人民币，并承担本次诉讼费用。
 <br/>
 <br/>
 随后，弦子也起诉朱军，并向朱军索赔人民币6万元，其中精神抚慰金人民币5万元。当事人弦子说，这是她与律师共同商议的结果。她说10月25日主要是交换证据，朱军本人出现可能性极小，但会一直努力。
 <br/>
 <br/>
 该事件源于7月26日，〝麦烧同学〞转发举报长文爆料，玄子2014年在央视做实习生时，朱军在化妆室里隔着衣服对她猥亵、强吻等。
 <br/>
 <br/>
 玄子曾报警，警察也进行了取证。但警方考虑朱军的〝对社会巨大正面影响力〞，规劝她放弃起诉。先是学校相关老师找该女生谈话，接着取证的警察也被更换。最后，此事不了了之。
 <br/>
 <br/>
 财新网向参与处置的老师、同学、律师求证，证实朱军
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性骚扰.html" target="_blank">
  性骚扰
 </a>
 一事。受害女生还强调朱军是个惯犯。有警察曾对她说：〝这事情发生不止一件，希望你们坚持下去。〞最后，该案由于换了负责警察而不了了之。
 <br/>
 <br/>
 今年7月，〝弦子〞深受〝Me Too〞运动的鼓舞，时隔4年又发长文控诉朱军曾对自己发生过性侵行为。
 <br/>
 <br/>
 朱军在8月15日发布律师声明，称有关〝朱军性骚扰实习生〞的消息不实，他已起诉原匿名举报者及微博实名转载用户，且北京海淀区法院受理了此案件。
 <br/>
 <br/>
 8月29日，〝麦烧同学〞说，陆续收到消息说朱军已被央视内部处理但又不了了之。外界质疑，官方迟迟不回应朱军事件，而且不给公众一个公开的处理结果，说明朱军的背景不一般。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：李泉）
