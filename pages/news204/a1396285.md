### 疑吃春药惹祸 湘官猝死在妇联主席家中
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】近日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南.html" target="_blank">
  湖南
 </a>
 常德桃源县一小区退休官员敬某，猝死于前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_妇联主席.html" target="_blank">
  妇联主席
 </a>
 伍某家中，因原因不明，官方说法也比较奇怪，引发舆论热议。有爆料称，敬某因服用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_春药.html" target="_blank">
  春药
 </a>
 导致猝死。目前该微博已被删除。
 <br/>
 <br/>
 近日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南.html" target="_blank">
  湖南
 </a>
 桃源一小区一名63岁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_原建设局长.html" target="_blank">
  原建设局长
 </a>
 ，死于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_妇联主席.html" target="_blank">
  妇联主席
 </a>
 家中的视频、图片通过微信朋友圈大量转发。
 <br/>
 <br/>
 陆媒10月20日报导说，经了解得知，死者敬某，男，系原桃源县住建局一领导干部，已退休5～6年，事发地位于沅水江旁的盛世桃源小区，原妇联主席伍某的家安居其中。
 <br/>
 <br/>
 同住该小区的刘女士介绍：事情发生在10月15日上午，当小区内有人死亡的消息传开后，围观群众越来越多，当地公安接警后也赶到了现场。因当时死者家属情绪失控，之后，有县里领导出面调解，尸体才于16日下午3点被运至殡仪馆。
 <br/>
 <br/>
 另据该小区保安透露，事件发生时，伍某的丈夫及家人均外出旅游，得知消息后及时赶回家中。
 <br/>
 <br/>
 10月19日上午9点，现任桃源县妇女联合会主席周某在介绍事发情况时说：伍某是我们妇联的正科级领导干部，是前任主席，还有一年就要退休了。敬某前往伍主席家是为了谈事，因为他们之前一起工作过，是同事关系，敬某后在谈事中，因心脏病突发不幸死亡。
 <br/>
 <br/>
 就敬某为何去伍某家，期间到底发生了什么？陆媒前往常德市桃源县公安局进行核实，一涂姓工作人员称：〝具体死亡原因，不便告知。〞
 <br/>
 <br/>
 新浪微博@〝报料吧〞披露，敬某因服用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_春药.html" target="_blank">
  春药
 </a>
 导致猝死。目前该微博已被删除。
 <br/>
 <br/>
 网民热议：〝妇联和团委这种闲差，去查查，一般都是二奶或情妇，专门给那些学历不高又想靠关系吃官饷的女人准备的。〞〝这早已经是共产党的传统了，只有当官才能随便搞！〞〝共党高官服用春药导致猝死不是什么新闻了。〞
 <br/>
 <br/>
 早在2013年，网络就曾曝光中共军队内部特供〝首长老干部专用〞春药
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伟哥.html" target="_blank">
  伟哥
 </a>
 。
 <br/>
 <br/>
 当时网上被疯转的照片是一只白色的包装盒，中间最大的三个字是〝万艾可〞，下面括号内标注〝内部专用〞，左上方印着中共军队后勤总部，并特别说明〝推荐首长、老干部专用〞，右下角写着〝大连瑞辉集团出品〞。
 <br/>
 <br/>
 万艾可(英文名：viagra，又名伟哥)是由美国辉瑞研制开发的一种口服治疗ED的壮阳葯，2000年7月在大陆上市。
 <br/>
 <br/>
 此照片令众网民惊呆了，〝原来这东西也有特供，中共内部的贪腐淫乱可见一斑〞。
 <br/>
 <br/>
 中共官场最严重的贪腐淫乱源于江泽民当政时期。江以〝色情腐败治国〞，中共官员大肆淫乱，二奶、小三、公共情妇数不胜数，淫乱视频、日记、公开的卖淫嫖娼层出不穷，江泽民、周永康等众多高官色情丑闻不断，中共军队更是前所未有地大搞黄色产业。
 <br/>
 <br/>
 《江泽民其人》一书写道，在江泽民的领导下，总参、总后、总政色情泛滥，沉溺于声色犬马之中，而且在江泽民故意助长的这种淫乱风气中，其心腹铁杆个个是〝淫棍〞。江淫乱治国遗祸在中共官场越演越烈，近年来曝出的相关丑闻举世震惊。
 <br/>
 <br/>
