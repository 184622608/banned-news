### 北京接受日本新〝三原则〞 安倍访华现不愉快一幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】日本首相安倍晋三10月25日至27日访华。贸易战高压下的北京为拉拢日本，一反多年的反日腔调，大举宣传〝中日共存共荣〞。法媒指，北京接受了安倍提出的中日关系新的〝三原则〞，但对日方有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_钓鱼岛.html" target="_blank">
  钓鱼岛
 </a>
 的要求未予回应。
 <br/>
 <br/>
 安倍晋三26日傍晚在北京的钓鱼台国宾馆与北京最高领导人习近平举行了首脑会谈。当日晚间，安倍在推特上透露，在这次会谈中，双方确认了日中关系新〝三原则〞。
 <br/>
 <br/>
 法广报导称，安倍提出了〝化竞争为协调〞、〝成为合作伙伴而非威胁〞、〝推进自由公平的贸易〞的新〝三原则〞，得到中方首肯。
 <br/>
 <br/>
 在会谈中，双方确认中日经济合作将围绕在第三国的经济合作、金融、技术革新和高龄化对策这四个方面展开。但安倍并没有明确表示参与中方的〝一带一路〞，而是称之为〝共同开拓第三方市场〞。
 <br/>
 <br/>
 在会谈中，安倍邀请习近平明年访日，习近平回应称将认真考虑。
 <br/>
 <br/>
 安倍还向习近平提起以〝从事间谍活动〞为由等被中方关押的日本人，习近平表示将按照中方法律适当处理。
 <br/>
 <br/>
 在与中共国务院总理李克强的会谈中，安倍提到新疆维吾尔族的境遇，并指中国的人权状况受到包括日本在内的国际社会的关注。
 <br/>
 <br/>
 对于日本在朝鲜无核化和绑架日本人问题上的立场，习近平表示〝理解和支持〞，并表示将维持对朝经济制裁。
 <br/>
 <br/>
 港媒指，几年前北京还对日本不屑一顾，现在迫于中美贸易战压力，不得不接受日本，以免被进一步孤立。而安倍则藉机向中方索取好处。
 <br/>
 <br/>
 报导称，虽然中方拉拢日本的动机尽人皆知，但北京〝吃相实在难看〞。此前，中共一贯指责日本用所谓〝大东亚共荣圈〞奴役中国，如今却露骨宣传〝中日共存共荣〞，一副欣然接受的〝奴才相〞。
 <br/>
 <br/>
 就在几年前，《环球时报》还在鼓吹跟日本〝断交〞、限制安倍进入中国，如今又大肆宣传安倍是〝好朋友〞。
 <br/>
 <br/>
 网传大陆居委会通知，提醒这次上街游行是宣传〝中日友好〞，不是反日，别闹出笑话。而中共将日本国旗悬挂到天安门的〝精日〞举动，外媒指让一贯反日的中共五毛们〝无所适从〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍访华.html" target="_blank">
  安倍访华
 </a>
 前高调阅兵、内阁成员参拜靖国神社、叫停对华援助，中共官方都选择了〝视而不见〞。
 <br/>
 <br/>
 不过，据日媒消息，此次安倍访华期间，也曾出现不太愉快的一幕。
 <br/>
 <br/>
 共同社28日报导说，日本外相河野太郎26日在北京与中共外长王毅会谈时，要求中方撤除在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_钓鱼岛.html" target="_blank">
  钓鱼岛
 </a>
 周边海域设置的浮标。但王毅没有直接回应，只强调将〝妥善处理意见的分歧对立〞。
 <br/>
 <br/>
 钓鱼岛问题多年来都是中共煽动反日的有力工具，北京如果突然示弱，显然很难对中国民众交代。不过有消息称，中方已将钓鱼岛问题暂时搁置。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
