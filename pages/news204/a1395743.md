### 云南红河州政法委书记和建落马
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】中共官网周二通报，云南省红河州委政法委前书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_和建.html" target="_blank">
  和建
 </a>
 ，因〝涉嫌严重违纪违法〞，正在接受纪律审查和监察调查。
 <br/>
 <br/>
 通报没有说明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_和建.html" target="_blank">
  和建
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 的具体原因。
 <br/>
 <br/>
 和建2001年12月起，任云南省红河州外贸局局长、2003年1月任云南省弥勒县委书记。2006年7月起，任红河州常委、政法委书记。2018年3月退休。
 <br/>
 <br/>
 另据《明慧网》报导，和建在担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红河州政法委书记.html" target="_blank">
  红河州政法委书记
 </a>
 期间，积极指挥、参与并胁迫下属迫害法轮功学员，被列入参与迫害法轮功学员的不法官员名单。
 <br/>
 <br/>
 另外，和建因参与迫害法轮功学员，也被海外〝追查迫害法轮功国际组织〞追查。
</div>
