### 任正非女儿被捕祸不单行 全球启动〝封杀华为〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的女儿，公司副董事长兼首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 ，因涉嫌违反美国对伊朗的贸易制裁，近日被加拿大警方逮捕。与此同时，全球正开启〝封杀华为〞的模式。
 <br/>
 <br/>
 据加拿大司法部发言人称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 于12月1日在温哥华被捕，美国要求将其引渡，保释听证会将于12月7日举行。 由于孟晚舟根据相关规定申请〝报导禁止令〞（即不能对外公开），目前无法提供任何进一步的细节。
 <br/>
 <br/>
 加拿大《环球邮报》称，加拿大司法部门是以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 涉嫌违反美国对伊朗的出口禁令而逮捕孟晚舟的。孟是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的长女，目前担任华为副董事长和全球首席财务官。
 <br/>
 <br/>
 自2016年以来，华为一直受到美国司法当局的暗中调查，认为其违反了美国的出口和制裁法律。 2018年初，中国另一家通讯设备公司中兴，就因违规向伊朗出口产品而遭到美国制裁。随后，美国司法部也展开调查华为是否违反了美国的制裁禁令，和伊朗进行贸易。
 <br/>
 <br/>
 另据英国《金融时报》12月5日报导，作为英国最大电信营运商的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国电信.html" target="_blank">
  英国电信
 </a>
 ，已将华为排除出核心5G网路设备供应合同的投标厂商名单外。另外，英国电信还将在2年之内将华为设备从其核心的4G网路中剥离出来，以使其手机业务符合内部政策，将华为设备用在电信基础设施的边缘。
 <br/>
 <br/>
 英国成为继德国之后，第二个质疑华为涉及国安隐患的欧洲大国。今年11月中旬，数名德国外交部、内政部官员曾匿名向路透社表示，该国预计在明年初启动5G基础建设招标，官员们正尽力说服政府对华为提高警觉。
 <br/>
 <br/>
 今年8月，澳大利亚政府以国家安全为由，禁止华为与中兴向当地提供5G行动网路设备。上个月，纽西兰政府也拒绝批准当地电信运营商采用华为公司的设备来建设5G网路。此外，美国也禁止政府部门采购中兴以及华为的产品，并一直尝试说服其盟国的电信运营商不要使用华为的设备。
 <br/>
 <br/>
 华为公司于1987年在中国深圳成立，自2012年以来一直是全球最大的电信设备制造商之一。不过外界一直质疑华为设备的安全性，认为它受制于中共政府，存在间谍和安全隐患。
 <br/>
 <br/>
 据悉，华为的创办人任正非曾是中共军队团职干部，而刚刚卸任的董事长孙亚芳，在加入华为之前也曾长期在中共国安部任职。
 <br/>
 <br/>
 外界注意到，美国总统川普就职后加强了对中共盗窃知识产权的打击，〝封杀华为〞的行动也正从美国向世界各个发达国家扩散开去。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：李泉）
