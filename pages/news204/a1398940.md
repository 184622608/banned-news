### 重庆之后 又传出福建高考政审文件
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆高考政审.html" target="_blank">
  重庆高考政审
 </a>
 的舆论风波未平，网上又传出福建省高考政审文件。不过新唐人发现，高考政审是中共统一规定，已实施多年。舆论翻炒此事，除了曝光中共极权本质外，也可能凸显中共内斗的激烈。重庆和福建都与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 关系密切。
 <br/>
 <br/>
 11月11日，多家海外中文网络报导了11月5日公布的《福建省高等学校招生委员会福建省教育厅关于做好2019年普通高等学校招生考试报名工作的通知》。
 <br/>
 <br/>
 文件中规定了对考生进行〝思想政治品德考核〞的实施办法和标准。其中两类考生被禁止报名：有反对宪法所确定的〝基本原则〞的言行或参加〝邪教组织〞的；触犯刑法、治安管理处罚法，受到刑事处罚或治安管理处罚且情节严重的。
 <br/>
 <br/>
 此前11月7日，《重庆日报》曾报导重庆考试院有关考生政审不合格的三大标准：反对〝四项基本原则〞；道德品质恶劣；有违法犯罪行为的。该报导随即在海内外被广泛聚焦，外界纷纷谴责中共侵犯人权、箝制舆论等等。同时，因为重庆是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 亲信
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈敏尔.html" target="_blank">
  陈敏尔
 </a>
 执政，舆论指责的矛头也同时指向北京当局〝复辟文革〞。
 <br/>
 <br/>
 不过，新唐人记者查询安徽、河南、湖南、黑龙江等多个省市今年出台的2018年高考招生和2019年高考报名文件，其中涉及〝思想政治品德考核〞的部分，都与福建省文件的表述基本相同。这显示高考政审并非重庆和福建独有，而是中共教育部统一规定。
 <br/>
 <br/>
 其中，河北省今年11月公布的文件中，虽然回避了考核的具体细节和标准，但文中多次提及〝思想政治品德考核〞，并强调考核必须〝合格〞。
 <br/>
 <br/>
 而重庆官方11月9日回应〝政审〞事件时表示，2019年该市高考招生有关〝思想政治品德考核〞的内容及方式〝均无变化〞，或说明重庆只是在延续此前的统一规定。
 <br/>
 <br/>
 而《重庆日报》的表述中，略去了有关〝邪教组织〞的部分，多出了〝道德品质恶劣〞一项。
 <br/>
 <br/>
 其实，虽然文革后邓小平取消了对高考考生〝家庭成份〞的考察，但从来没有放弃对考生本人的〝政治考核〞。
 <br/>
 <br/>
 1999年后，随着中共对法轮功的残酷迫害，考生本人甚至直系家属是否坚持修炼法轮功，也成了高考〝政审〞是否合格的一个严酷标准。上述官方文件中所谓〝邪教组织〞的内容，就是这一标准的隐晦表达。
 <br/>
 <br/>
 根据官方规定，普通高考（以及研究生入学考试）的〝思想政治品德考核合格〞，由考生所在学校或工作单位（无工作单位的由乡镇、街道办事处）负责。考生报名时必须提交由这些单位提供的相关材料，一旦其中有涉及〝考核不合格〞的内容，考生将被禁止报名。
 <br/>
 <br/>
 中共窃取政权以来，一贯通过各种政治审查剥夺人民的社会地位、经济来源甚至生存权利，以此来逼迫民众放弃自由思想，接受其暴政统治。高考政审只是其中一部分。
 <br/>
 <br/>
 不过，观察人士认为，海内外舆论这次先是热传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆高考政审.html" target="_blank">
  重庆高考政审
 </a>
 ，接着跟进福建高考政审，背后原因不只是民众表达对中共的不满，其中显然还涉及中共内部博弈的因素。中共江派一度掌控宣传系统和海外特务，海内外某些〝舆论阵地〞对习近平的攻击从未停止。
 <br/>
 <br/>
 在重庆，陈敏尔还在奉命对薄熙来和孙政才的余党进行扫荡。而福建是习近平的政治根据地。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：云涛）
