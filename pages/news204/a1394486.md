### 各地老兵持续集结 平度维权发酵或成全国事件
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东平度.html" target="_blank">
  山东平度
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵维权.html" target="_blank">
  老兵维权
 </a>
 事件继续发酵，正在升级为全中国范围的维权事件。10月7日，各地老兵继续向平度集结，重新占领日前被当局暴力清场的平度文化馆广场，声援平度老兵遭殴打事件。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东平度.html" target="_blank">
  山东平度
 </a>
 市爆发千名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵维权.html" target="_blank">
  老兵维权
 </a>
 事件，已持续4天。上周六（6日），当地政府出动大批特警，释放
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_催泪弹.html" target="_blank">
  催泪弹
 </a>
 ，暴力清场。带头维权的10多名老兵也被带走。不过，各地老兵仍在向平度集结，退伍老兵和中共当局的暴力冲突仍在继续。
 <br/>
 <br/>
 据海外维权网10月7日的消息，当天早晨8点多钟，广东、河南等地声援老兵赶到，重新占领被当局清场的山东青岛平度市文化馆广场。另外，来自河北、江苏、四川、辽宁、安徽等地数百名声援老兵正在赶来的路上，本地老兵也赶来汇合。
 <br/>
 <br/>
 在此前一天，已有上千维权老兵在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平度医院.html" target="_blank">
  平度医院
 </a>
 集结抗议，当局出动大批特警，释放催泪弹驱散，老兵则用灭火器还击。
 <br/>
 <br/>
 据网传视频显示，现场老兵有的四散走避，也有不少人手持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_棍棒.html" target="_blank">
  棍棒
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_烟雾喷射器.html" target="_blank">
  烟雾喷射器
 </a>
 等与警方冲突，有老兵受伤或被警察制伏在地。混乱中双方均有人受伤。
 <br/>
 <br/>
 6日晚，当地警方全面清场，并将现场的十几名老兵强行带走。
 <br/>
 <br/>
 平度居民刘女士对《自由亚洲》表示，现在当地的各个路口都封了，不准进，各个高速路口都封了，还有老兵被殴打重伤住院了，具体几个人不清楚。现场穿警服的有好几百人吧。
 <br/>
 <br/>
 十一期间，北京当局为防止各地老兵进京维权，各地严加管控。据港媒报导，10月4日，平度市38名退伍老兵准备进京上访，刚刚出发就遭到当地政府人员拦截、殴打。
 <br/>
 <br/>
 自5日早上起，山东、江苏、四川等省的退伍老兵，闻讯赶来平度声援被殴打老兵，有人手持木棍称要为被打老兵〝讨说法〞。
 <br/>
 <br/>
 （记者罗婷婷报导//责任编辑：李泉）
