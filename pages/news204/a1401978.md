### 【禁闻】12月4日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】下面我们来关注中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 长春百余教师不满职称评聘集体上访
 <br/>
 <br/>
 11月28号，吉林省长春市九台、农安、榆树、德会四个区县的一百多位教师，集体到长春市政府大门前上访，要求见市长，解决职称评定、晋级等问题。
 <br/>
 <br/>
 2. 利巢理财百余储户潜江市府示威
 <br/>
 <br/>
 12月3号开始，湖北省潜江市的100多名〝利巢理财〞公司的储户，因取不回本金及利息，连续两天到潜江市政府门前拉横幅、喊口号示威，但两天都被警方镇压。
 <br/>
 <br/>
 3. 纪念六四遭关押 广东李小玲取保获释
 <br/>
 <br/>
 因发起〝六四光明行〞活动而被中共关押一年多的广东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 李小玲，12月3号终于取保获释。李小林患有严重眼疾，但中共此前一再驳回她的取保候审申请。据说最后，她不得不向当局妥协，解聘了代表律师。
 <br/>
 <br/>
 4. 中共宪法日 维权人士上街宣传遇阻
 <br/>
 <br/>
 12月4号是中共所谓的〝宪法日〞，据北京维权人士李美青披露，当天她们20多人上街宣传，呼吁中共政府遵守宪法，切实保障公民的合法财产权及言论、集会、游行的自由权利。但在抵达天安门广场附近时，遭到警察的非法阻拦，一部分人还被抓捕关押。
</div>
