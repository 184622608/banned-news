### 网传刘强东妹妹去世 奶茶妹妹现身葬礼
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月11日讯】深陷性侵丑闻的京东集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 真是祸不单行。网传其年仅40多岁的亲妹妹
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强茹.html" target="_blank">
  刘强茹
 </a>
 突然离世，遗体10日已经在宿迁殡仪馆出殡，刘强东与妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奶茶妹妹.html" target="_blank">
  奶茶妹妹
 </a>
 出席葬礼。
 <br/>
 <br/>
 11月10日，多维网消息称，京东集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 的亲妹妹
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强茹.html" target="_blank">
  刘强茹
 </a>
 ，因羊水栓塞已于11月9日在老家宿迁第一医院去世。另有消息透漏，刘强茹遗体10日已在宿迁殡仪馆出殡，刘强东与妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奶茶妹妹.html" target="_blank">
  奶茶妹妹
 </a>
 到场。
 <br/>
 <br/>
 也有网友称，刘强东唯一的亲妹妹刘强茹是因生育难产，在分娩过程中过世，年仅40岁。
 <br/>
 <br/>
 不过上述消息仅在社交网路上流传，尚未得到京东方面及刘强东家人的证实。至于刘强茹的实际年龄，也有不同的说法。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-10/p9119511a219272981.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-10/p9119511a219272981-ss.jpg"/>
   <br/>
   <font size="-1">
    从曝光的近照看来，刘强茹十分低调朴实，穿衣打扮也很简朴。（视频截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 据财经头条报导，刘强东的妹妹刘强茹今年42岁，比刘强东小2岁，在综艺节目中有过露脸。从刘强茹生前的近照看，她十分低调朴实，穿衣打扮也很简朴。
 <br/>
 <br/>
 消息称，刘强茹曾长期在宿迁的一所中学任教，是一名英语老师。消息称，刘强茹一直当老师到2013年。后来刘强东出名了，她决定辞职，现在在宿迁京东客服中心开超市。
 <br/>
 <br/>
 不过，从2010年开始，〝刘强茹〞的名字，就已出现在另外一家与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东系.html" target="_blank">
  京东系
 </a>
 密切关联的公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股东.html" target="_blank">
  股东
 </a>
 名单中。2017年1月，刘强茹与刘强东在宿迁成立泰合四方投资公司，对外从事股权投资等业务。
 <br/>
 <br/>
 泰合四方投资注册资本1亿元，在两名股东信息中，除了刘强东，另一位名为刘强茹，不过，该公司并未披露股东的出资信息。
 <br/>
 <br/>
 今年8月，刘强东在美国涉嫌强奸而遭当地警方逮捕。虽然刘强东随后被释放，但美国警方9月4日公布报告称，刘强东一案属于〝一级性犯罪：强奸既遂〞（CSCR-1, CSC-RAPE-COMPLETE）。
 <br/>
 <br/>
 明尼苏达亨内平县警方9月21日宣布，已完成对案件的初步调查，接下来将由检方接手，审查案件全部证据，并最终决定是否起诉。
 <br/>
 <br/>
 根据当地法律，刘强东所涉的一级强暴罪被定罪后，刑期至少12年起。有华裔刑案律师认为，刘强东遭起诉的可能性超过90%，若起诉的话会涉及重新逮捕等问题。
 <br/>
 <br/>
 不仅如此，因刘强东涉嫌强奸丑闻，京东集团股价以及刘形象均受到严重打击。性侵案后，刘强东也鲜有官方露面，缺席众多重要场合。海外舆论分析，刘强东遭北京〝抛弃〞了。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
