### 陆金融人士:中国A股正上演〝国有化大洗劫〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】中共经济政策左转，中国私企本就生存困难。贸易战压力下，外界更担心中共会以重返
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_计划经济.html" target="_blank">
  计划经济
 </a>
 应对危机。近期以来，中共国资接连入股或收购私企。中共金融人士日前更透露，中国A股上市公司正上演〝国有化大洗劫〞运动。
 <br/>
 <br/>
 《德国之声》10月17日报导，中国A股不断下跌，许多公司用作抵押获取贷款的法人股都已面临银行的平仓线，这意味着银行很快就将有权收回贷款方的质押股权。
 <br/>
 <br/>
 一位匿名的中国金融业内人士表示，许多股价暴跌的上市公司，其股权都面临被国有银行收回的风险，中国A股正在上演一场〝国有化大洗劫〞。
 <br/>
 <br/>
 中共央行今年已四次降准，释放巨量资金，声称用于支持微小私企。但《德国之声》报导指，中小企业融资依然困难，因为经济萎缩，大型国有银行不愿给中小企业贷款。
 <br/>
 <br/>
 在缺乏政府支持的情况下，许多私企陷入经营困境。国有资金纷纷出手接盘私企股权，被指趁火打劫。
 <br/>
 <br/>
 据陆媒日前统计，今年以来已有37家私营上市公司被国资入主，另外还有多家公司接受国资作为战略投资者。
 <br/>
 <br/>
 其中，山东、福建、四川、河南等17个省市的国资均已出手收购民营上市公司。深圳、山东、北京、河南、湖南五地国资成为接盘主力，接手19家公司股权。深圳国资出手最快最阔绰，一个月时间内拿下怡亚通、英唐智控、梦网集团控股权，并成为英飞拓和科陆电子的重要股东。
 <br/>
 <br/>
 据此前陆媒报导，截至9月12日，中国沪深两市已有73个上市公司发布控制权转让公告，其中29个被国有资产收购控股。
 <br/>
 <br/>
 另外，有更多私企陷入经验危机，或遭当局打压。贸易战以来，很多私企接连破产，其中不乏民企巨头。
 <br/>
 <br/>
 截至10月10日，A股上市公司今年有360多位董事长、8000多名高管接连离职、跑路或遭警方调查。
 <br/>
 <br/>
 除了收购民企之外，中共还强迫上市公司设立党委，推行中共领导下的〝民主管理模式〞，被指抢夺私企控制权。
 <br/>
 <br/>
 中共央行行长易纲日前谈及中国经济、贸易战时称，北京已〝为最坏情况做好准备〞。
 <br/>
 <br/>
 美国高地智库（High Point Institute）研究员秦伟平对自由亚洲解读，即使没有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 ，中国经济也会出大问题，所以中共很早就做了〝最坏的打算〞，就是所谓的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_计划经济.html" target="_blank">
  计划经济
 </a>
 2.0时代〞，亦即重走计划经济道路，然后闭关锁国，这样可以挽救中共政治危机，保证中共在恶劣的经济社会形势下仍可继续执政。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
