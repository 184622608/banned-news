### 宋祖德曝李咏患癌有隐情 不便启齿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
  央视
 </a>
 主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李咏.html" target="_blank">
  李咏
 </a>
 日前因患癌在美国去世，很多人打听李咏患的是什么癌？相关媒体称是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_喉癌.html" target="_blank">
  喉癌
 </a>
 。中国娱乐圈昵称大嘴的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宋祖德.html" target="_blank">
  宋祖德
 </a>
 认为李咏喉癌之说，背后有隐情，而李咏家人没公布实情是有不便启齿原因。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
  央视
 </a>
 主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李咏.html" target="_blank">
  李咏
 </a>
 妻子、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视导演.html" target="_blank">
  央视导演
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈文.html" target="_blank">
  哈文
 </a>
 ，日前发文称，丈夫李咏因病去世，她写道：〝在美国，经过17个月的抗癌治疗，2018年10月25日凌晨5点20分，永失我爱。〞
 <br/>
 <br/>
 李咏去世的消息引发舆论哗然，当天微博热搜前20个话题中，与李咏相关的占了一半。
 <br/>
 <br/>
 不少人觉得李咏去世的很突然：此前并没有任何关于李咏患癌的消息发布，而且很多人也知道，李咏一家去了美国以后，还被质疑移民等，李咏方面也没有主动澄清。
 <br/>
 <br/>
 现年50岁的李咏可以说还是正当壮年，怎么忽然就去世了呢，而且是癌症！很多人表示惋惜的同时，都在打听：李咏患的是什么癌？但哈文在公布消息时没有说明。
 <br/>
 <br/>
 陆媒透露，李咏患的是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_喉癌.html" target="_blank">
  喉癌
 </a>
 ，或许是因病情严重，李咏在美国治疗了17个月后离开了人世。
 <br/>
 <br/>
 不少人叹息人在疾病面前的脆弱的同时，提出质疑问：喉癌发现及时是可以治疗的，为何李咏没及时发现？这个喉癌的消息，是否准确？
 <br/>
 <br/>
 10月30日，中国娱乐圈昵称大嘴的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宋祖德.html" target="_blank">
  宋祖德
 </a>
 发文，对李咏患喉癌一事进行了分析，并且提出疑问：〝李咏老婆为什么不坦诚公布，既然人都死了，坦诚公布一下得了什么癌有什么关系？网上质疑李咏老婆遮遮掩掩，或其中大有隐情。〞
 <br/>
 <br/>
 最后宋祖德推测认为，可能是肾癌或肝癌让李老婆难以启齿而不公布！
 <br/>
 <br/>
 不过，不少网友不认同，〝质疑肾癌和肝癌有什么丢人的？瞎猜测〞。也有网友说：〝现在中国大陆一些知名人士什么事都神秘不公布，所以人家才会瞎猜〞、有人表示〝人家的隐私。为什么要让你知道呢？这种打听人家隐私的行为，真令人不能接受。〞
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-1/p9106371a734589261.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-1/p9106371a734589261-ss.jpg"/>
   <br/>
   <font size="-1">
    宋祖德质疑李咏喉癌之说，背后有隐情，没向外公布是因为有不便启齿原因。（宋祖德微博截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 公开资料显示，李咏1968年5月生于乌鲁木齐，曾主持节目《非常6+1》、《幸运52》、《梦想中国》、《咏乐汇》等，2011年到2013年连续三年担任央视春晚的节目主持人。
 <br/>
 <br/>
 李咏2013年辞职离开了工作22年的央视，转入中国传媒大学，同年5月成立〝李咏工作室〞。
 <br/>
 <br/>
 李咏曾对媒体说，〝我现在最大的自由就是，我不想干什么就不干什么〞。
 <br/>
 <br/>
 李咏病亡，也让他曾任职的央视再次成为话题。
 <br/>
 <br/>
 央视做为中共喉舌，不仅替中共歌功颂德，粉饰太平，掩盖真相，还牵扯出许多丑闻。
 <br/>
 <br/>
 2013年底央视前副台长李东生落马后，随即爆出他将央视变成〝淫窝〞，给高官提供女性，淫乱程度惊人。周永康的现任妻子贾晓烨也是李东生拉皮条促成的。此后，李瑞英、沈冰、叶迎春、欧阳智薇等的丑闻接连被曝光。
 <br/>
 <br/>
 除了央视的女主持人外，男主持人芮成钢涉嫌卷入令计划案亦轰动一时，据港媒消息称，芮成钢不仅与令计划的妻子谷丽萍有染，更是20多名中共高官夫人的〝公共情人〞。
 <br/>
 <br/>
 这些年还频频传出罹患癌症而亡的央视高层、主持人及编导。包括前央视《新闻联播》主持人罗京、央视社会专题部副主任、前新闻评论部副主任陈虻、央视编导、主持人王欢、央视军事节目主持人方静、央视制片人、主持人肖晓琳等。
 <br/>
 <br/>
 其中罗京被指是死于艾滋病复发。1999年7月，中共镇压法轮功后，罗京、陈虻、王欢、方静、肖晓琳等人也都参与央视播报假新闻，给修炼真善忍的法轮功群体造谣、栽赃。
 <br/>
 <br/>
