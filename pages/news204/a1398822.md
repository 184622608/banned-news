### 非洲猪瘟防不胜防 亚洲多国发现中国食品带病毒
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情在中国大陆已失控，当局束手无策，任由疫情蔓延。从目前趋势来看，猪瘟疫情对亚洲其他国家也造成严重威胁。近日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 再次发现来自中国的水饺含有猪瘟病毒；
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 也从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国香肠.html" target="_blank">
  中国香肠
 </a>
 中检测出猪瘟病毒，联合国对此高度警惕。
 <br/>
 <br/>
 11月9日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 农林水产省公布消息，一名来自中国上海的旅客，10月14日从东京羽田机场入境时，所携带的自制水饺被检测出含
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 病毒，这是日本发现的第二例猪瘟病毒。
 <br/>
 <br/>
 10月1日，一名中国旅客从北京飞往北海道，从新千岁机场入境时，被查出携带的香肠含有非洲猪瘟病毒。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-10/p9120191a270466580.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-10/p9120191a270466580-ss.jpg"/>
   <br/>
   <font size="-1">
   </font>
  </a>
  <br/>
 </center>
 <br/>
 报通称，非洲猪瘟致死率相当高，若病毒入侵会对日本产生巨大影响。因此，日本当局将在入境环节采取彻底的防疫措施。
 <br/>
 <br/>
 临近大陆的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾.html" target="_blank">
  台湾
 </a>
 对猪瘟疫情也高度警惕。10月31日，台湾农委会在金门码头大陆旅客弃置的猪肉香肠制品中，也检测出有非洲猪瘟病毒。
 <br/>
 <br/>
 11月8日下午4点，中共农业农村部发布消息，福建省莆田市城厢区一养殖场爆发非洲猪瘟疫情。该养殖场存栏生猪4521头，发病85头，死亡85头。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-10/p9120192a882473491.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-10/p9120192a882473491-ss.jpg"/>
   <br/>
   <font size="-1">
    台湾从
    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国香肠.html" target="_blank">
     中国香肠
    </a>
    中检测出猪瘟病毒。示意图（PIXABAY.COM）
   </font>
  </a>
 </center>
 <br/>
 由于福建与台湾金门、马祖距离较近，猪瘟疫情由此侵入台湾的风险升高。因此，台湾陆委会主委陈明通将于11日，赴马祖考察小三通业务，同时视察非洲猪瘟防检疫措施。
 <br/>
 <br/>
 陆委会表示，检疫单位防疫如同作战，应确实落实相关工作。同时，陆委会还呼吁往来两岸旅客，不要去中国大陆畜牧相关场所参访。并警告两岸旅客切勿非法走私或挟带动物及其产品入境，以共同防范疫病入侵。
 <br/>
 <br/>
 陆委会表示，行政院会8日已通过〝动物传染病防治条例〞修正草案，针对违规旅客最高处罚额度，由新台币1万5千元提高至30万元（约67000元人民币）。
 <br/>
 <br/>
 除了日本、台湾以外，早在8月27日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国.html" target="_blank">
  韩国
 </a>
 最先发现源自中国大陆的非洲猪瘟病毒。韩国游客从中国大陆带回的加工猪肉制品，包括米肠、饺子，被检测出了猪瘟病毒。韩国方面已加强边境疫检措施，以防止非洲猪瘟疫情侵入韩国。
 <br/>
 <br/>
 针对非洲猪瘟疫情在大陆失控，且可能蔓延至其他国家的情况，联合国粮农组织8月29日已发出警告：中国曝光的非洲猪瘟疫情只是冰山一角，猪瘟疫情将摧毁中国的养猪业，并蔓延至亚洲他国。
 <br/>
 <br/>
 粮农组织首席兽医官卢布罗兹表示，猪瘟病毒通过含有感染猪肉的产品扩散到其他国家，几乎是必然的。〝所以问题不是会不会发生，而是何时会发生，以及我们应该如何开展合作，来预防并减小损失。〞
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
