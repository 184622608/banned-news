### 【禁闻】中共法官在港成立党支部 引发质疑
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】一些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共法官.html" target="_blank">
  中共法官
 </a>
 前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港城市大学.html" target="_blank">
  香港城市大学
 </a>
 进修，但立即就成立了临时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党支部.html" target="_blank">
  党支部
 </a>
 ，并公开活动，引发关注。评论质疑，中共组织未经注册在香港活动是否合法，并认为这种红色渗透将进一步侵蚀〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一国两制.html" target="_blank">
  一国两制
 </a>
 〞的底线。
 <br/>
 <br/>
 11月13、14号，多家海外中文媒体报导，三批前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港城市大学.html" target="_blank">
  香港城市大学
 </a>
 攻读法学硕士、博士班的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共法官.html" target="_blank">
  中共法官
 </a>
 ，不仅在香港设立了中共临时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党支部.html" target="_blank">
  党支部
 </a>
 ，10月20号还在校内召开支部会议，中共国家法官学院党委书记黄文俊前去参加，并宣讲所谓〝党课〞。
 <br/>
 <br/>
 香港城市大学法学院官网指称，这三批大陆法官参加的是第五届及第六届高级法官法学博士班，和第九届法官法学硕士班。开班典礼是在前一天的10月19号晚上。
 <br/>
 <br/>
 这一消息，原本发布在10月25号中共国家法官学院官网上，但现在已经被删除。
 <br/>
 <br/>
 原消息中还显示，各临时党支部39名党员参加了这次支部会议，还有11名非党员学员也应邀参加。而黄文俊在会议上也明确对他们提出了政治和意识形态的要求。
 <br/>
 <br/>
 香港城市大学政治学退休教授郑宇硕披露，事实上，之前不少来香港进修的大陆学者、学生都是中共党员或共青团员，都会私下进行类似的活动。
 <br/>
 <br/>
 香港城市大学政治学退休教授郑宇硕：〝他们通常开会呢，就当然要分配工作，其中也包括了统战的工作。意思就是说，在学校里头哪些老师经常批评中共的领导层。也有提出哪一些老师、哪一些学生领袖是可以争取的。〞
 <br/>
 <br/>
 郑宇硕表示，这次临时党支部引发关注，是因为最近两三年中共特别强调党的作用，因此他们在香港的活动也越来越浮出水面。
 <br/>
 <br/>
 郑宇硕：〝现在这一两年，党的角色越来越明显。举个例子，中共十八大以后就派它的党史研究室的负责人来香港，对香港自行政长官以下的所有重要的官员、议员谈十八大的所谓精神。这样的做法以前是没有的。〞
 <br/>
 <br/>
 香港作家、自由撰稿人张成觉表示，中共一向把基层党支部看得非常重要。而香港回归21年以来，除了中资公司，新华社等党媒里面有不公开的党支部，中共始终没有在香港高调搞过党支部。
 <br/>
 <br/>
 香港作家、自由撰稿人张成觉：〝现在它从幕后走到前台，就是城市大学这些大陆留学生这样子搞，这个姿态很明显的是北京授意的，要把共产党的影响扩大。香港的学术界，要有所警惕，不要对共产党掉以轻心，它是要全面的占领香港社会。〞
 <br/>
 <br/>
 至今中国共产党在香港并没有正式注册。《自由亚洲电台》引述香港浸会大学新闻系高级讲师吕秉权的观点说，如果要以非法组织取缔其他政治团体，其实对中国共产党也应以非法组织名义去调查一下。
 <br/>
 <br/>
 张成觉：〝这个合法性的问题，共产党它是根本不管的。因为共产党的成立，从一开始它就是非法。它实际上是俄国人共产国际豢养的走狗。并没有一个法律通过，说从此中国共产党在中国是合法社团了，从来没有。〞
 <br/>
 <br/>
 郑宇硕表示，香港的政党一般以公司的名义注册，中共虽然没有注册，但中联办事实上也是中共驻香港的党委，拥有很大资源和人力，但运作不透明。
 <br/>
 <br/>
 郑宇硕：〝这个党委也在香港运作了，而且它掌握了很多的资源，很多的人手。现在大家担心的不单是中共对香港的控制了，就是大家最基本的生活方式，大家的核心价值也受到侵蚀了。〞
 <br/>
 <br/>
 至今，港府和城市大学都没有对此事件作出评论。张成觉表示，由于中共的渗透，香港一些大学校长的任命，都需要中共的默认或支持，他们自然不会和当局唱反调。香港〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一国两制.html" target="_blank">
  一国两制
 </a>
 〞的底线，在一点点的逐渐被侵蚀，冲破。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/陈建铭
