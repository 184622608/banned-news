### 〝习思想〞多写3个字 大陆编辑被罚款
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陕西日报.html" target="_blank">
  陕西日报
 </a>
 》一名编辑把〝总书记〞3字加进原文16个字的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习思想.html" target="_blank">
  习思想
 </a>
 〞专有名词中，被指犯了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治类差错.html" target="_blank">
  政治类差错
 </a>
 〞，该编辑和值班主管，分别被罚款5000元和一万元人民币。
 <br/>
 <br/>
 22号，香港《南华早报》刊登了《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陕西日报.html" target="_blank">
  陕西日报
 </a>
 》的内部惩处文件。文件指出，15号，由新闻部编辑刘汇和值班主任王戈华发的稿件，文中将〝习近平新时代中国特色社会主义思想〞错写为〝习近平总书记新时代中国特色社会主义思想〞，多了〝总书记〞3个字，属
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治类差错.html" target="_blank">
  政治类差错
 </a>
 。
 <br/>
 <br/>
 湖南维权人士周先生:〝党媒管控是很严格的，不允许在意识形态上，或者党啊国家领导人的称呼上，或者各种言论上，就不能出任何差错嘛，一些记者啊或者编辑，甚至被降级的、开除的都有，判刑的就更加不说，也有。有些人就不想做了，因为压力太大了，这个压力不是来自工作的压力，很多来自于政治的压力。〞
 <br/>
 <br/>
 山西维权人士李先生:〝这是很荒唐的事情，这也只有独裁国家才有这样子，不但是媒体，对普通百姓言论管的也是非常严的。〞
 <br/>
 <br/>
 江苏民主人士邵先生表示，中共原是苏共扶植的非法政权，靠谎言和暴力维持统治，与所谓的主旋律不同的言论都会入罪。
 <br/>
 <br/>
 江苏民主人士邵先生:〝现在非常恐惧，共产党就是恐怖组织，就是极端纳粹组织，它也是不合法的，它不是按照现在的政党选举的政体。〞
 <br/>
 <br/>
 去年10月，中共19大将〝习近平新时代中国特色社会主义思想〞列入党章，今年3月又被写入中共〝宪法〞。中共官媒的编辑被教导，新闻报导中提到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习思想.html" target="_blank">
  习思想
 </a>
 时，必须完整提这16个字。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
 <br/>
