### 进博会罕见一幕 肯尼亚总统批中共贸易不公
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】贸易战背景下，北京正在国际上陷入孤立。上海进口博览会没有迎来西方主流国家领导人，就连接受北京多年援助的肯尼亚，也上门声讨中肯双边贸易不公。
 <br/>
 <br/>
 《纽约时报》报导称，北京正在贸易战中寻找盟友，但这并不容易。虽然中方几个月来都在努力邀请他国领导人出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海进博会.html" target="_blank">
  上海进博会
 </a>
 ，但最终只有10多个国家的总统或者总理出席了11月5日一早的开幕式，而这些领导人大多来自欠了北京很多钱的国家。
 <br/>
 <br/>
 就连北京的主要贸易伙伴，包括德国、英国、韩国和日本的领导人都缺席了进博会。
 <br/>
 <br/>
 另外，在出席博览会的10几个领导人中，还有一些人发出了不和谐的声音。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肯尼亚总统.html" target="_blank">
  肯尼亚总统
 </a>
 （Uhuru Kenyatta）在5日早上的演讲中说，中肯贸易在过去10年里翻了8翻，〝但是，贸易严重地向中国（中共）有利的一面倾斜。〞
 <br/>
 <br/>
 肯尼亚也是中方〝无偿贷款〞喂养多年的非洲国家。但是北京在非洲的〝殖民〞政策越来越引起当地民众的不满，肯尼亚当局对北京也越来越不〝客气〞。
 <br/>
 <br/>
 今年9月5日，正在北京召开中非论坛、宣布继续向非洲〝大撒币〞之际，肯尼亚报纸The Standard在头版刊登大幅漫画，把中国画成一头给非洲领导人们喂奶的母猪。同日，肯尼亚警方持枪突袭搜查了中共官媒环球电视网(CGTN)的非洲总部，并扣留多名记者。有评论指，一些非洲国家对待北京的态度是〝左手拿钱右手打脸〞。
 <br/>
 <br/>
 此外，北京领导人习近平也在进博会开幕式上发表了演讲。
 <br/>
 <br/>
 在演讲中，习近平用〝手电筒只照别人〞讽刺美国，并发表了中国经济〝大海论〞，与此前〝自力更生〞的发言一脉相承，被指暗示在贸易战中顽抗到底的立场。
 <br/>
 <br/>
 不过，与此同时，习近平也宣布将〝一如既往〞地继续推进〝开放市场〞。除了宣布未来15年将进口40万亿商品外，还承诺将加大开放、降低关税、保护外国企业的合法权益和知识产权等。
 <br/>
 <br/>
 不过，外媒引述商界人士指，北京宣布的进口额度不过稍高于此前的预期。而且中方只字不提备受外界谴责的政府补贴政策，其它的承诺也只停留在口头上，并没有给出具体措施和时间表，这并不代表北京从此真正的〝开放〞。
 <br/>
 <br/>
 鉴于北京多年来不遵守国际规则，拒不履行加入世贸组织的义务，国际社会对北京的空头承诺越来越失去信任。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
