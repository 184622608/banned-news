### 王岐山外访新职务首曝光 罕见分担刘鹤职责
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】中共官方日前宣布，中共国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 将出访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 ，其最新职务——中以创新合作联合委员会中方主席首次曝光。该职务前任是中共国务院前副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘延东.html" target="_blank">
  刘延东
 </a>
 ，理应由现任副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 接任。
 <br/>
 <br/>
 中共外交部10月18日宣布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 将于10月22日至30日访问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_以色列.html" target="_blank">
  以色列
 </a>
 、巴勒斯坦、埃及和阿联酋，并在访以期间与以色列总理内塔尼亚胡共同主持中以创新合作联合委员会第四次会议。
 <br/>
 <br/>
 该消息显示，王岐山已接任中以创新合作联合委员会中方主席。
 <br/>
 <br/>
 据公开资料，该委员会是中以政府间合作论坛组织，于2015年1月成立，首任中方主席为时任国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘延东.html" target="_blank">
  刘延东
 </a>
 ，当时刘延东在国务院内部分管教科文卫。今年中共〝两会〞上，国务院领导层换届，新任副总理孙春兰分管教育卫生，主管科技的则是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 。
 <br/>
 <br/>
 中以创新合作联合委员会主要事务涉及科技创新，按理应该由刘鹤接替刘延东的中方主席职务。9月，刘鹤已出任国务院下属的国家科技体制改革和创新体系建设领导小组组长，该小组首任组长也是刘延东。
 <br/>
 <br/>
 中共外宣分析，中方主席从副总理变更为有〝第八常委〞之称的王岐山，表明中方有意提升该委员会的规格，也凸显了贸易战背景下，北京对科技创新的重视。
 <br/>
 <br/>
 以色列虽然国土狭小，但被世界公认为仅次于矽谷的全球第二大科技创新基地。
 <br/>
 <br/>
 科技创新事关被北京视为〝核心利益〞的〝中国制造2025〞计划。王岐山分担刘鹤的相关职责，显示王的角色已经不限于国家副主席这个荣誉职务。
 <br/>
 <br/>
 此前，王岐山也曾多次会见美方政商界人士，为贸易战发声传话。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
