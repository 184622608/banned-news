### 中共海外渗透猖獗 清查推特网友 报复反共学者
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月18日讯】中共近期对海外的渗透活动十分猖獗，近日已针对海外社交媒体展开了〝清网〞行动，大批拥有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推特.html" target="_blank">
  推特
 </a>
 帐号的中国人权活动人士遭威胁，并被强迫删除推特上发布的内容，甚至有人因此被抓捕。与此同时，海外反共学者遭到疑似中共特务的报复行动，人身安全受到严重威胁。
 <br/>
 <br/>
 <h4>
  中共开始对
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推特.html" target="_blank">
   推特
  </a>
  等海外社交媒体采取
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清网行动.html" target="_blank">
   清网行动
  </a>
 </h4>
 <br/>
 据维权网报导，中共近日开始对海外社交媒体推特的中国用户帐号采取清网行动。北京、重庆、山东、湖北、福建和广东等地，一大批拥有推特帐户的中国大陆人权活动人士相继遭到中共公安部门的恐吓威胁，被迫删除推特上已发布的内容或撤销推特征号。其中，重庆的刘继春、福建厦门市的潘细佃等人士，因为拒绝配合中共的要求删除推特内容而被公安部门逮捕或扣押。
 <br/>
 <br/>
 该报导表示，估计在接下来的一段时间内，受到中共清网行动冲击的人数还将继续增加。
 <br/>
 <br/>
 旅美中国民主人士，民间组织〝公民力量〞的发起人杨建利在接受《自由亚洲电台》采访时表示，中共政府现在大力对微信、微博和推特的进行〝清网〞，表明这个政权害怕异议、担心社会不稳。
 <br/>
 <br/>
 他说：〝中国当局对言论的钳制和限制信息流通这也不是第一次，但这次措施的力度明显在加大。当局最近关闭诸多自媒体帐户、封锁推特网、清理推特帐户的内容等。这些都说明，这个政权处于明显的不自信中。〞
 <br/>
 <br/>
 <h4>
  中共特工在海外对批评中共的外国学者实施恐怖攻击
 </h4>
 <br/>
 事实上，中共对海外舆论的控制，不仅限于对拥有海外社交媒体帐号的中国民众进行言论箝制，还包括嚣张地针对批评中共的外国学者实施恐怖行动。
 <br/>
 <br/>
 屡屡批评中共的新西兰中国问题专家安妮．玛丽．布雷迪（Anne-Marie Brady），本月15日在社交网上发帖披露，日前有人闯入她的车库，对汽车两个轮胎做了手脚，如果不是及时发现问题并送到维修公司进行修检，那种人为的破坏将会导致汽车在急刹车时失控，从而危及生命安全。
 <br/>
 <br/>
 新西兰媒体《先驱报》随后就布雷迪发布的上述消息向维修公司了解详细情况，机械师告诉该媒体，布雷迪汽车的两个轮胎失去了阀盖，这样加速会磨损并带来严重安全问题，而这种情况只可能是人为破坏造成的。
 <br/>
 <br/>
 据公开的资讯，布雷迪在最近的一年半里，持续发文揭露中共在新西兰等国家扩张渗透的方式方法。然后在今年2月，她的住所遭匪徒闯入，被盗走了三部手提电脑、两部手机和一张加密记忆卡。而在此之前，她的办公室也曾被人偷窃，并收到了恐吓信。有一些证据显示，这一系列针对她的攻击行动很可能来自中共特工。
 <br/>
 <br/>
 目前，这些案件已交由新西兰警方负责国家安全的部门来处理，警方发言人称，警方已到汽车检修公司取证，对相关线索的调查正在进行中。
 <br/>
 <br/>
 澳大利亚战略政策研究所中国问题研究者乔斯克（Alex Joske）对《自由亚洲电台》表示，他认为破坏汽车与入室偷窃事件有关连，威胁行为应该都是来自中共。
 <br/>
 <br/>
 乔斯克说：〝中共这种手段已在香港、台湾出现了好几年，它们在海外用暴力和恐怖的手段已有先例，这种暴力和恐怖的活动会愈来愈多。我们首先要揭露这种恐怖和威胁，普通的人民就会了解中国（中共）政府是用什么方法来影响和干涉海外民主国家。〞
 <br/>
 <br/>
 乔斯克并指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共海外渗透.html" target="_blank">
  中共海外渗透
 </a>
 手段多样，除恐怖威胁外，还有很多并不〝违法〞的渗透方式。希望布雷迪所遭遇的〝升级版〞威胁事件，能够引起西方国家对中共渗透行为足够的重视，并积极采用立法的方式来阻截中共在海外的渗透。
 <br/>
 <br/>
 近年新西兰有华人国会议员被曝与中共军方关系密切，而华商领袖向新西兰两大政党提供巨额政治献金的事件也引发了该国舆论的高度关注。
 <br/>
 <br/>
 （记者唐迪综合报导/责任编辑：明轩）
