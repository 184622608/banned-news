### 被指吹响倒习号角 多维紧急改文降调
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】川习会结果凸显了北京败相。编辑部设在北京的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_多维网.html" target="_blank">
  多维网
 </a>
 随后刊出评论文章，要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 为中共乱局负责，做出全面检讨。海外学者指，中共外宣疑似吹响了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_倒习号角.html" target="_blank">
  倒习号角
 </a>
 。次日多维即修改了相关文章，调子从批习改成了变相捧习。
 <br/>
 <br/>
 12月3日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_多维网.html" target="_blank">
  多维网
 </a>
 刊载旗下《多维月刊》的文章&lt;极左撕裂中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 应负责任&gt;。文章指，作为中共〝核心〞，习近平要为中共左转〝妖风〞负责，必须进行自我检讨。
 <br/>
 <br/>
 文章对习近平的个人集权、意识形态左转、个人崇拜以及其执政理论、经济政策进行了全面的抨击，称这是导致社会左右撕裂的根源。文章并指责，习近平上台后，一些社会问题反而更加激化，质疑习是否已经〝失去了判断〞等。

 在推特评论此文称，〝国安派系外宣媒体疑似吹响
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_倒习号角.html" target="_blank">
  倒习号角
 </a>
 〞。她还分析说，〝该刊总部在北京，这样做，只有两个可能：一是准备鱼死网破，二是认为己方有胜算。〞

 多维文章公开要求习近平自我检讨，被指有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逼宫.html" target="_blank">
  逼宫
 </a>
 〞之嫌。在中共体制中，最高领导人进行自我检讨具有特定政治含义，往往意味着必须让权甚至退位。
 <br/>
 <br/>
 12月4日，多维网发文辩解称，部分海外媒体与民主人士〝恶意曲解〞该文原意，并否认文章涉及中共派系斗争。文章还言之凿凿表示，这篇要求习近平全面检讨的文章，意在〝支持习近平正在推动的全面深化改革和国家治理现代化〞。
 <br/>
 <br/>
 此外，多维网上&lt;极左撕裂中国 习近平应负责任&gt;的原文也被大幅修改，目前不但标题变成了&lt;防止极左撕裂中国 习近平强势拨乱反正&gt;，还删改了许多内容，所有直接指责习近平的语句都消失无踪。

 据自由亚洲电台12月4日报导，这篇文章在修改成上述版本前，一度被多维网撤回，原有链接变成对川习会的分析。后来又恢复成类似文章，但标题改成了&lt;防止极左撕裂中国，习近平应扛起时代责任&gt;，其中要求习近平检讨的文字改为〝要求庞大的官僚系统进行反思〞，直接批评习近平的句子被删除。
 <br/>
 <br/>
 据此看来，&lt;防止极左撕裂中国 习近平强势拨乱反正&gt;这个版本，至少已经过了两次修改，调子从最初的批习，最终定调在捧习上。
 <br/>
 <br/>
 对于此文的改头换面，何清涟在推特评论称，〝看来，多维北京总部一定被有关方面约谈了。〞
 <br/>
 <br/>
 相关资料显示，多维网于1999年创立，原来总部设在纽约。但其最初创立者就被指背景可疑，与中共江系暧昧不明。2009年，多维网股权易手。
 <br/>
 <br/>
 加拿大媒体人姜维平2010年在自由亚洲发文引述媒体界消息指，多维2009年被一名香港商人收购后，编辑部大本营就迁到了北京，其背后资金来源也令人生疑。
 <br/>
 <br/>
 而自由亚洲电台引述知情人士透露，多维目前分别在北京和香港两地设有编辑部。香港编辑部人员多为各地新闻组记者，北京方面则是大陆新闻业者。
 <br/>
 <br/>
 多维网报导方向和风格，被指多与中共党媒有相似之处，十几年来其政治立场也被认为几经变化。
 <br/>
 <br/>
 外界观察人士注意到，习近平拿下薄熙来和周永康后，多维热捧习近平的文章一度得到中国大陆党媒原文转载。但近期以来其态度又有所变化，开始密集刊登肯定江泽民的文章，而疑似对习近平批评、贬低的报导亦逐渐增多。
 <br/>
 <br/>
