### 《疫苗管理法》征求意见 民众反馈遭中共维稳
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】首先来关注中国的消息。中共有关当局日前就《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗管理法.html" target="_blank">
  疫苗管理法
 </a>
 》公开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_征求意见.html" target="_blank">
  征求意见
 </a>
 。不过，有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗受害者.html" target="_blank">
  疫苗受害者
 </a>
 家长到中共国家药品监督管理局，想要递交修改意见，却被警察维稳。
 <br/>
 <br/>
 11月11号，中共国家市场监管总局就《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗管理法.html" target="_blank">
  疫苗管理法
 </a>
 (
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_征求意见.html" target="_blank">
  征求意见
 </a>
 稿)》，公开征求意见。意见反馈截止到 11月25号。
 <br/>
 <br/>
 资深非政府组织人士杨占青表示，这份草案基本是废话和谎言，比如〝应当自律〞、〝有权举报〞、〝对违法行为进行舆论监督〞等，这些都是基本的社会规则，也是公民的权利，没必要大篇幅论述。
 <br/>
 <br/>
 资深非政府组织人士杨占青：〝另外，制定草案的人就是平时打压对疫苗领域的违法问题举报（的人）。他们也是禁止、封杀媒体报导的人。〞
 <br/>
 <br/>
 20号，有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗受害者.html" target="_blank">
  疫苗受害者
 </a>
 家长来到中共国家药品监督管理局，想要递交修改意见，却吃了闭门羹。
 <br/>
 <br/>
 疫苗受害者家长：〝现在不是在征求大家的意见吗？我们是疫苗受害者家长。现在在食药监门口，呆了一个多小时了，手上拿着材料，基本上进都进不了啊，没人理会我们。〞
 <br/>
 <br/>
 杨占青：〝一个江苏的受害者家长，姓杨的，被他们当地的徐州开发区的派出所带回家。这就很讽刺啊，这个草案口口声声说，欢迎去监督、投诉、举报、提意见，但是真正带着材料去提意见（的民众），不仅不接待，反而被维稳。〞
 <br/>
 <br/>
 杨占青指出，按照《行政法规制定程序条例》，向公众征求意见，期限不应少于30天。而这份草案公开征集意见比法定规定少了半个多月，并且没有任何理由。
 <br/>
 <br/>
 杨占青：〝它是说把人民的健康放在第一位，要重视疫苗问题，但是搞的这么草率，并且很多内容让我担心啊，以后公众评论、监督疫苗的风险更大。〞
 <br/>
 <br/>
 这次草案把疫苗提到〝国家安全〞的高度，又要求〝有关疫苗的宣传报导应当科学、客观、公正〞，但杨占青认为，这些标准都是政府说了算。
 <br/>
 <br/>
 杨占青：〝一旦说你不公正，一旦扣一个危害国家安全的帽子，那国内谁还敢去举报、监督？媒体更是不敢报导。〞
 <br/>
 <br/>
 过去20多年来，中国反覆发生疫苗丑闻。尤其今年7月长春长生的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_假疫苗事件.html" target="_blank">
  假疫苗事件
 </a>
 ，惹得民怨沸腾。
 <br/>
 <br/>
 新唐人记者陈汉、王子琦、施宜君采访报导
</div>
