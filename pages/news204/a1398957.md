### 范冰冰李晨将移民美国？替身邵小珊曝内幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】近期有关女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 与未婚夫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李晨.html" target="_blank">
  李晨
 </a>
 将要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民美国.html" target="_blank">
  移民美国
 </a>
 的消息在网上流传，女星章子怡的裸替
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邵小珊.html" target="_blank">
  邵小珊
 </a>
 在微博中称，自己已经查看过相关记录并确认范冰冰和李晨将移民美国。很多网友纷纷调侃道：离开了就别再回来了。
 <br/>
 <br/>
 最近，中国知名女星章子怡的裸替
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邵小珊.html" target="_blank">
  邵小珊
 </a>
 ，在一段短片中，提及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李晨.html" target="_blank">
  李晨
 </a>
 改变国籍的书面报告，并称，在最近的一段时间里面，她一直在关注着范冰冰和李晨国籍的问题，直到最近才给出了明确的消息。她称，范冰冰与李晨确实将要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民美国.html" target="_blank">
  移民美国
 </a>
 。
 <br/>
 <br/>
 很多网友在看到这条消息后，纷纷留言道，离开了就别再回来了。
 <br/>
 <br/>
 事实上，早在央视前主播
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 因电影《手机2》影射到自己，而爆料中国娱乐圈签〝阴阳合同〞逃税初期，陆媒就曾报导，范冰冰与男友李晨准备移民美国。
 <br/>
 <br/>
 据〝搜狐娱乐〞6月报导，微博热传范冰冰与男友李晨同时注销中国籍和户籍，将67.8亿元人民币财产转移到美国，准备移民美国。
 <br/>
 <br/>
 报导称，有专业律师通过合法途径进行核查，发现范冰冰一家及李晨名下均无任何美国房产。
 <br/>
 <br/>
 该律师还对传闻进行分析说，范冰冰已经移民美国的可能性几乎为零。如果要加入美国国籍，首先必须成为美国永久居民，即获得〝绿卡〞。而未婚者必须在拥有绿卡的5年内在美国住满30个月，方能申请加入美国国籍。
 <br/>
 <br/>
 该律师表示，范冰冰作为曝光率高的大明星，显然无法满足5年内在美国住满30个月的要求，因此不可能已经移民美国。
 <br/>
 <br/>
 39岁的李晨与范冰冰因戏结缘，于3年前公开恋情，去年9月16日，李晨在范冰冰36岁生日时成功求婚。之后粉丝纷纷催婚，希望看到两人在2018年完成人生大事。
 <br/>
 <br/>
 然而，范冰冰因卷入逃税案，自6月起从公众视野消失。直到10月份才有了消息。10月3日，中共税务机关宣布，范冰冰涉嫌偷税漏税，缴纳8.8亿元人民币罚金，可免除牢狱之灾。
 <br/>
 <br/>
 当天范冰冰在微博发表〝致歉信〞说，最近经历了〝从未有过的痛苦和煎熬〞，将〝尽全力克服一切困难，筹措资金、补缴税款、缴纳罚款〞。
 <br/>
 <br/>
 就在外界估算范冰冰的身家、关注她如何缴纳罚款时，消息称，为了筹款，范冰冰8月份即在房屋中介打包抛售41套房产变现，李晨也挂牌变卖北京二环的四合院帮凑钱。不过，有房屋中介否认了部分消息。
 <br/>
 <br/>
 有网友说，近9亿元对连续5年蝉联〝福布斯中国名人榜〞榜首的范冰冰，可能是小菜一碟，但也有人认为，〝流动资金和资产是两回事，就算卖房子，也不是今天挂出来，明天就有买家付钱的。〞
 <br/>
 <br/>
 因此，网友推测：范冰冰筹措近9亿元的现款谈何容易？
 <br/>
 <br/>
 再加之范冰冰事件引发系列寒蝉效应，不仅诸多代言被取消，好莱坞电影《355》准备解雇她，连她客串的《大轰炸》在大陆也取消上映，另外她手里至少有6个电影项目面临〝上映困难〞。即使她再度回到公众视野，演艺生涯恐怕也不再如以前顺利了。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：范铭）
