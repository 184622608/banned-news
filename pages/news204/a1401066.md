### 突发：河北发生特大爆炸 已酿22死22伤（视频/组图）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张家口.html" target="_blank">
  张家口
 </a>
 市委宣传部官方微博最新消息称，11月28日凌晨，在张家口市桥东区河北盛华化工有限公司附近发生一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸.html" target="_blank">
  爆炸
 </a>
 起火事故。截至早上8点，事故已经造成22人死亡22人受伤。
 <br/>
 <br/>
 河北〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张家口.html" target="_blank">
  张家口
 </a>
 发布〞微信公众号消息称，11月28日凌晨零点41分，张家口市桥东区大仓盖镇盛华化工有限公司附近，发生一起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸.html" target="_blank">
  爆炸
 </a>
 事故。经现场搜救确认，截至早上8点，事故已经造成22人死亡、22人受伤。伤者已分别送往河北北方学院第一医院和251等医院救治。
 <br/>
 <br/>
 网上影片显示，现场火光熊熊，浓烟冲天，医院内有多名伤者等候救治，部份人手、脚、脸部被熏黑。
 <br/>
 <br/>
 经初步调查，事故中共有50辆车被烧毁，包括大货车38辆、小型车12辆。现场明火已扑灭，搜救工作和事故原因调查仍在紧张进行中。事故原因也在进一步调查核实中。
 <br/>
 <br/>
 河北盛华化工有限公司成立于1970年，于2000年5月经股份制改造而成。位于张家口市盛华东大街10号，是中国化工500强企业之一，河北省百强重点民营企业，同时也是张家口市民营经济支柱企业。
 <br/>
 <br/>
 有人猜测，此次事故是车辆运输化工品所致。
 <br/>
 <br/>
 针对这起爆炸事故，新唐人将继续跟进报导。
