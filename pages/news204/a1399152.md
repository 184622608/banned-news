### 惊！甘肃女行走中被街道〝吞没〞 掉3米深坑（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_甘肃兰州市.html" target="_blank">
  甘肃兰州市
 </a>
 发生人行道〝吃人〞事件，一名年轻女子在行走过程中，道路突然塌陷，该女子躲避不及，掉入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_3米深坑.html" target="_blank">
  3米深坑
 </a>
 ，导致肋骨骨折。
 <br/>
 <br/>
 据网上热传的一段视频显示，11月11日晚6时许，在兰州市城关区中医药大学附近，一名年轻女子正行走在人行道上，道路突然发生塌陷，该女子躲避不及，面部撞击地面后，掉入3米的深坑，有地砖不断剥落砸入坑中。
 <br/>
 <br/>
 据陆媒报导，该女子在坑内不断呼救，随后被送院治疗，经诊断有肋骨骨折，目前已无大碍。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_道路塌陷.html" target="_blank">
  道路塌陷
 </a>
 原因正在进一步调查中。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 兰州官方称，今年8月以来，当地已发生约20多起类似路陷事故。发生在9月28日的路陷事故中，有3人掉入深坑，其中1人抢救无效死亡。
 <br/>
 <br/>
 在四川也发生过道路塌陷事故，数人掉入深坑，有多人遇难。目前，官方未解释为何大陆频频发生此类事故，但大陆网民们认为，这与大陆道路工程偷工减料，〝豆腐渣〞工程频频出现有关。
 <br/>
