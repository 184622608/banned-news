### 平度老兵维权第7天 持续戒严拦老兵
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】山东
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平度老兵.html" target="_blank">
  平度老兵
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 事件，10月11号进入第7天，目前平度依然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戒严.html" target="_blank">
  戒严
 </a>
 ，被关押的老兵陆续被遣返，但仍不断有各地老兵赶来声援遭到拦截。当地老兵都被软禁家中，他们表示，如果没有公正的解决，将维权到底。
 <br/>
 <br/>
 平度市民10月11号上午向《新唐人》透露，被关押的老兵陆续被遣返。之前到老兵聚集的广场围观声援的，或给老兵送水送饭的市民，据他所知，至少有五、六人被拘留，当局还从6号开始看管有上访纪录的市民，直到10号才撤岗。
 <br/>
 <br/>
 平度市民张先生:〝警察也很多，各个路口都有，老兵看不到了，但是可能还有在这里，进去就抓走，但是从全国各地还有像平度赶来，还有继续来的，都进不来了，都封锁了。关在西关小学、奥体中心的，今天都回去了。〞
 <br/>
 <br/>
 10月10号上午，多名广东云浮老兵千里迢迢来到平度，被拦截在市区外的高速路口。守在那里的上百名的特警、公安将老兵团团围住。
 <br/>
 <br/>
 现场视频:〝今天是2018年10月10号上午10点半钟，我们在这个平度，在这个平度站下高速，现在很多警察，在这里检查我们这个车。现在你看很多警察把我们拦住，不让我们前进，刚才抢我们的手机。〞
 <br/>
 <br/>
 青岛退伍老兵透露，他们都被软禁在家中，18号才能解禁。上面来领导跟老兵谈判，结果还没有对外公布，大家都在等消息，如果没有公正的解决，他们将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 到底。
 <br/>
 <br/>
 青岛退伍老兵孙先生:〝中央领导和我们退役军人事务部的领导都来了，现在老兵有一部分都走了，还有一部分不让走，在那等着各地的领导来接嘛。我们都在家里，5号以后就不让我们出去，每天都有人盯着我们，现在外面都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戒严.html" target="_blank">
  戒严
 </a>
 了，当兵的不允许聚集，聚集了可能就要清理。〞
 <br/>
 <br/>
 10月4号，38名山东青岛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_平度老兵.html" target="_blank">
  平度老兵
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进京.html" target="_blank">
  进京
 </a>
 途中被当地公安暴力
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_截访.html" target="_blank">
  截访
 </a>
 ，6人住院，各地老兵连日赶到平度讨说法，遭到暴力清场。
 <br/>
 <br/>
 新唐人记者熊斌、陈杰采访报导
