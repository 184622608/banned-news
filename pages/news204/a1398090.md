### 港媒：进博会一石二鸟 时间点巧妙
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】首届中国国际进口博览会11月5日到10日在上海举行。有港媒分析称，中共在这一时间点召开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 ，是想在中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 持续升级的情况下，收到一石二鸟之效。但这种做法可能事与愿违。
 <br/>
 <br/>
 首届〝中国国际进口博览会〞（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 ）5日早上在上海开幕。中国国家主席习近平前往出席，并做了半个小时的主旨演讲。
 <br/>
 <br/>
 习在演讲中再次定位称，举办中国国际进口博览会，是中共着眼于推动新一轮对外开放作出的重大决策，是中共主动向世界开放市场的重大举措。
 <br/>
 <br/>
 习在演讲中还抛出一个颇为诱人的〝饵〞：〝预计未来15年，中国进口商品和服务将分别超过30万亿美元和10万亿美元。〞
 <br/>
 <br/>
 中共商务部此前在宣传〝进博会〞时，也提到一个数字——未来5年中国将进口超过10万亿美元的商品和服务。
 <br/>
 <br/>
 分析认为，北京当局正用巨额经贸数字呼唤〝小伙伴〞们站队。此次20多个世界发展中国家参加〝进博会〞，如巴拿马、多米尼加、萨尔瓦多等国甚至出动了总统本人赴会；而美国、加拿大、英国、法国、德国、意大利及日本政要无一出席，韩国、澳大利亚等中国主要贸易国家的政要也拒绝出席。
 <br/>
 <br/>
 这本身就凸显出中共及其〝小伙伴〞们与西方渐趋对立的贸易格局。
 <br/>
 <br/>
 香港《经济日报》说，习近平倡议主办进博会，本来是希望藉此吸引各国领袖来华参与盛事，结果是只有发展中国家的领导人应允参加，欧美日等发达国家的领导人均缺席。
 <br/>
 <br/>
 不过，报导认为在中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 持续升级的情况下，中共欲藉此次进博会收到一石二鸟之效。但这种做法很可能事与愿违。
 <br/>
 <br/>
 一是，中美爆发贸易战后，北京仍如期举办进博会，就是趁改革开放40周年之便，对外释放善意，表明中共将进一步开放市场。
 <br/>
 <br/>
 二是，中共办进博会，一改以往作为出口大国赚取外汇的形象，现在加大进口力度，变身为全球最大进口市场，未来5年将进口超过10万亿美元的产品和服务，给人一种中共并非威胁，而是机遇的感觉。
 <br/>
 <br/>
 报导说，尽管中共释出善意，但美欧政府却冷待，因为他们除了要中共开放市场，还要求中共改变体制，进行深层次的结构性改革，取消对国企的补贴，因在政策倾斜下，国企集中力量办大事较容易，令外企难以竞争。
 <br/>
 <br/>
 因此，美欧政府都希望，中共在商业环境多做具体而合乎西方标准的措施，用行动来证明深化改革开放。不过，尽管美欧企业对中共经贸措施不满，但仍积极参与进博会，配合中共，尽量保持低调，因为他们都不想错失在中国这个庞大市场。
 <br/>
 <br/>
 报导认为，中共办进博会摆出开放姿态，以便在中美博弈中争取更多国家支持，中共踏出这重要一步是好事，但不等于一步到位成功，因背后牵扯到中美角力的政治暗战，困难重重。
 <br/>
 <br/>
 截止目前，美中贸易战交火已近4个月，经过四轮谈判无果。贸易战持续升温的过程中，川普对中共使用了孤立政策。重新签订的〝美国－墨西哥－加拿大协定〞（USMCA ）就是一个最鲜明的例子。
 <br/>
 <br/>
 其中有一个〝毒丸〞条款规定：协定中任何成员国若与一个〝非市场经济国家〞签署贸易协定，其他成员国有权在6个月内自由退出协定。该条款若被复制到美国与其他国家的贸易协定中，将使得希望和美国有友好贸易关系的国家断绝与中共的贸易来往。
 <br/>
 <br/>
 此外在9月联合国大会期间，美国、日本与欧盟曾发布联合公报，宣示一起抵制倾销、迫使外国企业转让技术、盗窃商业机密等不公平贸易行为。
 <br/>
 <br/>
 白宫经济顾问库德洛（Larry Kudlow）在采访时说，这是一项非常重要的三方协定，中共正在贸易战中被孤立。
 <br/>
 <br/>
 库德洛还表示，习近平与川普将在阿根廷二十国集团（G20）峰会期间，举行一次相当正式的双边会谈，贸易战及川普的孤立政策，给中共经济造成相当显著的影响，北京或正考虑与华盛顿达成某些协议。
 <br/>
 <br/>
 在贸易战不断升温之下，中国各项经济数据不断下滑，中共已经感受到日益沉重、甚至足以窒息的经济压力。
 <br/>
 <br/>
 10月底，中共政治局会议聚焦讨论经济，罕见的强调经济下行压力增大，要高度重视，增强预见性，及时采取对策。；11月1日，习会见企业家时也罕见承认，目前中国经济发展的不确定性明显上升，下行压力加大，企业经营困难增多。
 <br/>
 <br/>
 加之贸易战令中共内斗加剧，中共四中全会一直悬空未定等等因素交织，分析认为北京不得不找上川普、重修旧好，甚至共商如何应对中共内部与国际政局，因此有理由推断〝川习会〞将就贸易战达成共识或协议。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉)
