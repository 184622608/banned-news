### 法媒：孟宏伟事件时机敏感 中南海雪上加霜
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】国际刑警组织原主席、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 自法国返回中国后被抓一事持续发酵。有法媒引述分析说，北京当局未待孟宏伟任满，即召他回国秘密逮捕，料想可能事态紧急，而孟案发生的时机和影响，对北京当局更是雪上加霜。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 妻子葛瑞丝‧孟（Grace Meng）10月初向法国警方报案，其丈夫9月底回中国后失踪。中共官方10月7日深夜通报：孟宏伟涉嫌违法，目前正接受中共监委监察调查。
 <br/>
 <br/>
 自由亚洲电台引述知情人士的话披露，孟宏伟目前被软禁在北京郊区一处，拘禁手法与过往软禁其他高官相似，由专人看守，他多个部下亦受牵连，但具体情况不明。该事件显得非常神秘，无人得到真相。
 <br/>
 <br/>
 日本《产经新闻》分析说，孟宏伟被逮捕主要是因其涉及中共重要机密。
 <br/>
 <br/>
 报导分析说，孟宏伟可能与中国海航集团董事长王健7月在法国坠亡事件有关；另外，孟宏伟可能计划逃往美国，打算向美方提供中共领导人亲属在海外的资产清单，因此被召回国逮捕。
 <br/>
 <br/>
 目前孟宏伟案依然真相不明，不过，对于中共高层来说，此案是一件不容忽视的大案。
 <br/>
 <br/>
 法广引述一篇社论称：〝孟宏伟失踪多日后，证实被中共中纪委以涉嫌违法而留置调查，孟妻在法国寻求国际协助，成了国际大新闻。这是王立军叛逃进美国驻成都领事馆、令计划哥哥令完成逃来美国后，中共高官或亲属再次向外国求救。〞
 <br/>
 <br/>
 社论认为，孟案是送给美国总统川普打击中共的子弹；也有评论说，孟案重伤中共参与国际事务形象，对今后高官外逃和中共政治控制都将发生负面影响。〞
 <br/>
 <br/>
 孟宏伟被拿下的原因很多传闻：一、政治背叛、派系权力斗争或重大失职。二，孟落马可能与其担
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 没完成北京交待的任务有关。三，贪腐和经济问题。四，孟宏伟被传曾参与2015年北戴河〝政变〞。但都无法证实。
 <br/>
 <br/>
 社论认为：〝北京当局未待孟宏伟任满，即召他回国并秘密逮捕，不顾欧美重视的‘正当法律程序’和透明度，对参与国际执法机构实为一大讽刺。料想可能事态紧急，北京已无法顾及其负面效应。此事也给中共官场带来另一次震慑效应。〞
 <br/>
 <br/>
 社论还称，孟案可能让所有中共官员陷入惶恐，更失安全感。为了逃脱整肃，为自己准备后路，或表忠，或留存可要挟的筹码，导致官心不稳，官僚阶层更失向心力，多少动摇政权稳定的基石，而孟案发生的时机和影响，对北京当局更是雪上加霜。
 <br/>
 <br/>
 中国民主活动家王军涛15日对法广表示，孟宏伟是十九大以后的权利斗争中逐步下来的，共产党内部的两股力量，正在围绕孟宏伟案件在较劲。
 <br/>
 <br/>
 他说，《南华早报》最先爆出孟宏伟出事的消息，这说明该报对这事有一定的了解。该报还说孟案不牵扯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 案，这点出了案件的更多背景，由该报澄清与周案无关，就是和公安部长赵克志对着干了。赵表明孟是周的余党，因为赵克志代表的是习近平的意见，而南早到底代表谁呢？
 <br/>
 <br/>
 政治与经济学者程晓农先生在接受美国之音访谈时表示，阿里创始人马云虽然是中共购买《南华早报》的出资人，但该报的经营权和控制权，却在中共港澳工委手里，实际控制人则是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾庆红.html" target="_blank">
  曾庆红
 </a>
 。
 <br/>
 <br/>
 王军涛分析说，如果孟宏伟事件是一个政治事故的话，按照共产党的规矩，孟的举荐人，这事的负责人会被问责。跟孟一派的人也会受牵连，孟妻在海外高调发声上做过安排的人也会被问责。
 <br/>
 <br/>
 共产党对高官的家人都有边控，他们出去时家人留在国内做人质，但是很显然孟妻不在这个安排之列，这就意味着有人可能要对此承担责任。
 <br/>
 <br/>
 王军涛认为，习近平可能要进行危机管理，进行控制，但孟宏伟一派的可能就要极力扩大这件事，藉助这事的影响问责习近平。
 <br/>
 <br/>
 王军涛强调，孟宏伟是在2012到2018年初这段时间，作为一个正部级的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 ，他应该知道很多内幕，他手里的资料可能是前高官不知道的，孟妻手中一定也掌握着一些情况，如果他们认可的底线被突破，孟妻会不会将材料拿出来是关注的焦点。
 <br/>
 <br/>
 时事评论员文昭在自媒体节目中说，孟妻接棒爆料的话，再爆中共高官的贪腐冲击力已经不行了，就跟多撒几次农药，害虫有了抗药性一样。而曝光中共人权问题才会引起西方社会的共鸣。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：赵云）
