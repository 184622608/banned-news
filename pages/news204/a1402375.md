### 【禁闻】美波音卫星技术 辗转落入中资公司
------------------------

<div class="wysiwyg">
 【新唐人2018年12月07日讯】中共意图取得被美军视为高度机密的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波音.html" target="_blank">
  波音
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卫星技术.html" target="_blank">
  卫星技术
 </a>
 ，而触动美国敏感神经。美国媒体报导，北京当局暗中资助一家在美国洛杉矶的新创公司，目地是想获取卫星的机密技术。
 <br/>
 <br/>
 12月5号，美国《华尔街日报》披露，中共政府通过金援控制一家美国初创公司，意图取得
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_波音.html" target="_blank">
  波音
 </a>
 公司的卫星绝密技术。
 <br/>
 <br/>
 世界最大的航天航空器制造商美国〝波音〞公司，同时是美国航空航天局（NASA）最大的承包商。
 <br/>
 <br/>
 美国洛杉矶这一家初创公司，从〝波音〞订购了一颗卫星。公司创始人表示，他们得到了中国资金的支持，现在公司由中国控制。
 <br/>
 <br/>
 但是，美国的相关技术法规禁止中国获取这一类的技术。
 <br/>
 <br/>
 中国企业是如何获取美国技术？
 <br/>
 <br/>
 今年年初，中共号称有超过170颗卫星在太空运行。中共党媒《新华社》说，中共十九大报告明确提出建设航天强国的战略目标，可是，不到一年的时间，却被爆出偷窃海外
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卫星技术.html" target="_blank">
  卫星技术
 </a>
 。
 <br/>
 <br/>
 澳大利亚华裔博士张小刚接受本台记者采访时表示，中共从来没有停止窃取西方国家先进技术。
 <br/>
 <br/>
 澳大利亚华裔博士张小刚：〝它们（中共）从掌握政权开始，其实也就没停止过往外面派各种人去窃取技术，那么我们这一代人出来海外留学的时候，很多人都在国内受到当局的要挟，叫他们在海外接触到西方的关键技术的时候，要把它送回中国去。〞
 <br/>
 <br/>
 《华尔街日报》的报导认为，中共正试图获得可以和美国平起平坐的超级大国地位，而这些技术能帮助中国填补一项空白，有助于实现中共主宰高科技产业的目标。
 <br/>
 <br/>
 张小刚：〝那么现在中共因为这些年发展以后，它们有非常明确的目标，要很快跟美国不但是平起平坐，而且要求能够超过美国，压制美国，在军事上和各个方面，当然也就包括航天领域。〞
 <br/>
 <br/>
 美国华府人权律师叶宁表示，中共和波音公司之间的合作，早已经超过经济和技术层面的合作。
 <br/>
 <br/>
 美国华府人权律师叶宁：〝这个应该不是一个偶然事件。我觉得这种技术的转让，背后是不是具有违反美国的出口管制法，这个事情美国政府需要高度密切注意这样的发展动态。〞
 <br/>
 <br/>
 据了解，这家名为Global IP的新创公司，2008年前后提出以卫星改善非洲网路服务的构想，但需要一笔庞大的资金，当初，他们对外募资的款项收入并不理想。2015年7月，中国〝中国东方公司〞突然主动联系他们，表达投资意愿。
 <br/>
 <br/>
 中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东方资产管理公司.html" target="_blank">
  东方资产管理公司
 </a>
 ，是中共四大国有金融资产管理公司之一，根据官网资料显示，东方公司的上级机构为中共财政部，资产注册资本为人民币100亿元，全部由财政部全额拨入。而公司高层几乎都是中共资深干部。
 <br/>
 <br/>
 叶宁：〝如果美国现在的领导团队、政府精英，如果不对中共这种长期的、潜在的全面威胁，有一种清醒的认识，以此作为制定政策的引导的话，如果美国不从防微杜渐的角度，进行全面的预警措施，那么将来有可能会输得干干净净。〞
 <br/>
 <br/>
 虽然，中、美两国在〝川习会〞后达成了双方〝休兵90天〞的共识，但是关于〝中国制造2025〞技术转移方面的谈判，还是没有明朗化。
 <br/>
 <br/>
 叶宁：〝我觉得中共这样窃取技术的事件，如果它有一些扩张的影响的话，有可能对这个三个月缓冲期以后的政策走向，美国对华政策走向产生一些影响。〞
 <br/>
 <br/>
 张小刚：〝中共它也会利用这个缓冲期的作法，希望骗过西方国家，因为它们以前也是一路搞这种骗术。那么90天以后会怎么样？这也说不清楚，但是我相信它们（中共）绝对不会停止对西方的军事机密或者各种机密的窃取。〞
 <br/>
 <br/>
 美国方面担心，这颗卫星一旦升空，中共政府有可能会变更用途，或以逆向工程取得技术。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/钟元
