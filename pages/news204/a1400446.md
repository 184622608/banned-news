### 【禁闻】中国证券基金经济学家 被〝维稳〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】由于受到经济增长放缓、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 疲软的困扰，有消息说，中共正在对证券基金行业施压，要求确保他们的首席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济学家.html" target="_blank">
  经济学家
 </a>
 们不要做出所谓〝对党和国家不利〞的分析。
 <br/>
 <br/>
 美国《彭博新闻网》11月21号刊登《中共对市场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济学家.html" target="_blank">
  经济学家
 </a>
 的警告：遵循党的路线》文章。
 <br/>
 <br/>
 知情人士透露，中共证券监督管理委员会主席刘士余，月初在北京会见30多家证券经纪公司和基金公司代表。他要求这些经济学家在出版研究时，要以所谓〝更高角度思考，考虑党和国家的利益〞，才不会给市场参与者所谓的〝误导〞。
 <br/>
 <br/>
 大陆媒体稍早之前也曾报导，这场座谈会在11月2号召开，共有30多位证券基金行业的首席经济学家与会。刘士余要求他们要〝引导市场的预期〞。
 <br/>
 <br/>
 中国证券业协会上周五公告，证券基金行业首席经济学家们签署了一份〝自律倡议书〞。内容同样要求经济学家要审慎发表研究观点。
 <br/>
 <br/>
 还有消息人士透露，安信证券首席经济学家高善文，5月和7月曾发表演说，表示中美关系正处于1972年以来最困难的时刻，当局以所谓〝无麻醉手术〞降低了中国的债务水平。在这之后，中共开始更关注经济学家的研究。
 <br/>
 <br/>
 《彭博》认为，证监会的行动表明，虽然中共声称将扩大向外国公司开放证券市场，但仍然监管外界对中国经济的看法。而且由于美中贸易战加剧，中共领导人对于〝看空市场〞的研究，他的容忍度在未来几个月可能会受到考验。
 <br/>
 <br/>
 台湾企业家高为邦：〝你以为你共产主义万能，把经济衰退也来扭曲为经济变好。这个太荒谬了！第一个你做不到，第二个即使你做到了，你会受伤更厉害，因为你扭曲了这个市场。其实你的目的是达不到的。不让这个市场反映实际情况，那你现在不反应，将来可能就会更惨，泡沫就会更厉害。〞
 <br/>
 <br/>
 旅美政经分析人士秦鹏表示，中共对证券行业首席经济学家们的〝维稳〞，并非新动作，这只是中共所谓的稳信心和舆论管控的表现。在中共眼里，国计民生和人民福祉，都没有自己的统治重要。它的政治理念，与它对外鼓吹的〝立党为公，执政为民〞完全相反。它遇到问题，不是解决问题本身，而是先解决提出问题的人。
 <br/>
 <br/>
 旅美政经分析人士秦鹏：〝今年年初以来，随着经济持续下滑，特别是3月份贸易战开启之后，问题越来越严重。中共第一时间就想到了两点，第一是继续加大掠夺，所以10月份之前，越来越多的对企业政策方面有坏消息；第二方面是控制和封锁信息。也就是中共六稳里面的稳信心。〞
 <br/>
 <br/>
 另一方面，尽管中、美贸易战情势加剧，中共官员、媒体和企业表面上扬言要反击。但实际情况可能是另一番景象。
 <br/>
 <br/>
 《路透社》专栏作家罗伯・克斯（Rob Cox）21号撰文指出，中国企业家普遍认为，中共当局对经济议题没有太大兴趣，反而更着重于中共政权本身，也不愿意花费政治资本来推动改革。所以不少中国民营企业家纷纷将川普的制裁，视作推动中共当局改革的动力源头，暗自支持川普向中共施压。
 <br/>
 <br/>
 秦鹏：〝天下苦赵（中共）久矣，面对着大洋彼岸，像川普推出的减税降费、降低政府管制等等对企业有利的环境，和中共越来越严苛和变态的作风，中国的民营企业家嘴上不敢说，但是心里边对中共越来越厌恶。前一阵国内朋友来告诉我，说现在大家都不敢说话，也没有心思经营，有门路的都跑了，大家都在道路以目。他们都感觉中共这种情况下维持不了多久了。〞
 <br/>
 <br/>
 秦鹏说，中国的民营企业家，在早期政策红利、人口红利耗光后，越来越被中共涸泽而渔——不留余地的掠夺伤害。他们背负着全世界最高的税负，还要面对地方官员的敲诈勒索和中共对民企的种种歧视，诉苦无门。尤其今年贸易战开打之后，中共加大掠夺来应对财政危机，更伤害了大批的民营企业家。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/周天
