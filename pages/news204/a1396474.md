### 大陆高校人事洗牌 3天5所高校换人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】拥有120年历史的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京大学.html" target="_blank">
  北京大学
 </a>
 ，23日出现人事变动。一度把〝鸿鹄之志〞读成〝鸿浩之志〞的校长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林建华.html" target="_blank">
  林建华
 </a>
 卸任。北大前党委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郝平.html" target="_blank">
  郝平
 </a>
 接任校长职务。至此，3天内已有5所部属高校换帅。至于大调整的原因目前尚未明确。
 <br/>
 <br/>
 10月23日下午，教育部网站发布消息称，中共中央组织部，当天在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京大学.html" target="_blank">
  北京大学
 </a>
 宣布了中央、国务院的任免决定，邱水平任北京大学党委书记（副部长级），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郝平.html" target="_blank">
  郝平
 </a>
 由北京大学党委书记转任校长；
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林建华.html" target="_blank">
  林建华
 </a>
 不再担任北京大学校长职务。
 <br/>
 <br/>
 今年56岁的邱水平，为北大法律系硕士。曾任北京市委副秘书长等职，2017年1月，任山西省高院院长、党组书记；今年59岁的郝平，北大历史系毕业，曾任教育部党组成员、副部长等职，2016年12月起，担任北京大学党委书记。
 <br/>
 <br/>
 至于63岁卸任的北大校长林建华，曾一度把〝鸿鹄之志〞读成〝鸿浩之志〞引起热议。
 <br/>
 <br/>
 林建华担任校长的3年多时间里，力主推动综合改革。为此还将启动〝30+6+2〞的学科建设布局，并将建立通识教育与专业教育相融合的多样化人才培养体系，力促学生在学部内自由转专业、在全校自由选课。
 <br/>
 <br/>
 当天早些时候，教育部网站发布消息称，中央组织部在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南京大学.html" target="_blank">
  南京大学
 </a>
 宣布了任免决定，胡金波任南京大学党委书记（副部长级）；张异宾不再担任南京大学党委书记职务。
 <br/>
 <br/>
 22日，中共教育部人事司在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国药科大学.html" target="_blank">
  中国药科大学
 </a>
 ，也宣布了教育部党组的任免决定，徐慧不再担任中国药科大学党委书记；由金能明接替。由于金能明是宁夏大学党委书记，其调动也给宁夏唯一的这所〝双一流〞高校留下了职务空缺。
 <br/>
 <br/>
 同日，中共教育部人事司在江南大学宣布教育部党组的任免决定，朱庆葆任江南大学党委书记，朱拓不再担任江南大学党委书记。
 <br/>
 <br/>
 中共教育部人事司21日也在华中农业大学宣布了教育部党组的任免决定，李召虎任华中农业大学校长、党委副书记，邓秀新不再担任华中农业大学校长。
 <br/>
 <br/>
 至此，3天内已有5所部属高校换帅。至于大调整的原因目前尚未明确。近几年来，中共官场已经完成了密集大调整。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
</div>
