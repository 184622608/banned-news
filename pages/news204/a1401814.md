### 中共封杀结构改革90天期限 美使馆官微也不放过
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】美中两国首脑在布宜诺斯艾利斯达成贸易战暂停协议后，中共官媒在报导中都对有关中方需在今后的90天内作出结构性改变的前提条件只字不提。中国网友在网络上提及这条限制的言论也都遭删除，连美国驻中国大使馆发表的白宫的声明都遭分享限制。有网友感叹中共明目张胆地〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自欺欺人.html" target="_blank">
  自欺欺人
 </a>
 〞。
 <br/>
 <br/>
 美中两国首脑日前在布宜诺斯艾利斯达成暂停贸易战升级的协议。其中美方同样暂不调高对中国2000亿美元产品的关税，但中方需在今后的90天内通过与美谈判，解决强迫技术转移等美方关切问题，并作出结构性改变。如果在90天后，中方在结构改革方面仍然没有进展，美国目前加征的10%关税税率将会上调到25%，贸易战将继续升级。
 <br/>

 不过中共官方随后在对外发布相关消息时，却没有提到这90天的期限，中共所有官媒的相关报导对此也只字不提，只是一味地强调新一轮谈判后有望完全取消关税。
 <br/>
 <br/>
 外媒发现，就连中国网友在中共官媒的相关报导下方所留的评论，凡是提到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_90天期限.html" target="_blank">
  90天期限
 </a>
 等敏感内容的也全都遭删除。
 <br/>
 <br/>
 另据美国CNBC电视台报导，就连一向被网民们视为中国大陆难得的〝言论自由高地〞的美国驻中大使馆的官方微信上，日前发表的有关这次〝川习会〞协议的白宫声明，也疑似受到中共监管机构的阻挠，仅供浏览，不得转发或分享。其他的消息和文章则不受影响。
 <br/>
 <br/>
 有网友质疑：中共在中国国内封杀有关90天期限的资讯，是因为觉得有点〝丢脸〞呢，还是因为中共官方压根就没打算在这个期限内作出实质性的改变？也有网友感慨称，〝这简直就是明目张胆地搞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自欺欺人.html" target="_blank">
  自欺欺人
 </a>
 的把戏嘛！〞
 <br/>
 <br/>
 有外媒认为，中美双方在首脑会谈后对外发布消息存在的差异，反映出双方仍有重大歧见，未来的谈判将困难重重。
 <br/>
 <br/>
