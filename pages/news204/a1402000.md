### 阿里巴巴大文娱总裁杨伟东被查 涉案金额逾亿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴大文娱.html" target="_blank">
  阿里巴巴大文娱
 </a>
 集团12月4日宣布，前阿里大文娱轮值总裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨伟东.html" target="_blank">
  杨伟东
 </a>
 因为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济问题.html" target="_blank">
  经济问题
 </a>
 〞被调查。有消息称，早在一个多月前，阿里内部已收到举报，开始调查杨伟东，其涉案金额可能超过上亿元人民币。
 <br/>
 <br/>
 根据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴大文娱.html" target="_blank">
  阿里巴巴大文娱
 </a>
 集团发布的通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨伟东.html" target="_blank">
  杨伟东
 </a>
 是被人举报的。他最后一次露面时间是11月29日，当天，他出席第六届中国网路视听大会，以阿里文娱大优酷总裁、阿里音乐CEO的身份发表了主题演讲。
 <br/>
 <br/>
 不过，在此之前（11月26日），阿里巴巴执行长（CEO）张勇已发布全员公开信，宣布由阿里影业董事长兼执行长、大麦网执行长樊路远接棒杨伟东，出任阿里大文娱事业群新一届轮值总裁。
 <br/>
 <br/>
 阿里大文娱自2016年成立以来，已换过三任掌门人，包括俞永福、杨伟东和樊路远。
 <br/>
 <br/>
 现年44岁的杨伟东，2009年即出任诺基亚（Nokia）大中国区市场营销总监。2013年3月加入优酷土豆，先后担任副总裁、总裁。优酷土豆被阿里收购后，杨伟东继续受到重用。
 <br/>
 <br/>
 2016年11月1日，阿里巴巴文化娱乐集团成立。杨伟东被任命为大优酷事业群总裁；2017年11月15日，阿里大文娱实行轮值总裁制，杨伟东接替俞永福任第一任轮值总裁。2018年5月11日，他又兼任阿里音乐执行长。
 <br/>
 <br/>
 据多家陆媒报导，早在一个多月前，阿里内部已开始调查杨伟东，他涉嫌贪腐的项目集中在优酷于2018年推出的〝这就是〞系列综艺节目，金额可能超过上亿元人民币。
 <br/>
 <br/>
 阿里大文娱旗下有8个相关业务部门，包括阿里影业、合一集团（优酷土豆）、阿里音乐、阿里体育、UC（浏览器）、阿里游戏、阿里文学和阿里数字娱乐事业部，都是阿里巴巴透过收购整合而成，构成情况复杂。
 <br/>
 <br/>
 而阿里大文娱成立至今，仍处于巨额亏损状态。根据今年第3季报显示，阿里在文娱板块的亏损达人民币48.05亿元，为单季最大亏损。今年前两季，也分别亏损35.41亿元、38.02亿元。
 <br/>
 <br/>
 此外，杨伟东不仅在阿里任高职，还在外部企业拥有股权。据〝天眼查〞数据显示，杨伟东旗下企业共有45家公司，其中他担任法定代表人的有16家，担任股东的有2家，担任高管的有44家。
 <br/>
 <br/>
 在杨伟东出事前，2015年，从腾讯视频跳槽到阿里出任副总裁的刘春宁，也被举报涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济问题.html" target="_blank">
  经济问题
 </a>
 ，最终以职务犯罪被判刑9年。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
</div>
