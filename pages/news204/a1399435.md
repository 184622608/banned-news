### 藏头诗泄秦岭别墅案神秘主人 中央震怒难奏效
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦岭.html" target="_blank">
  秦岭
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建.html" target="_blank">
  违建
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_别墅.html" target="_blank">
  别墅
 </a>
 案惊动中南海，尽管习近平先后下达6道指令，但秦岭别墅违规违建现象并未得到有效扼制。实际上在此之前，网上就流传一首藏头诗，暗示〝陈路别墅〞背后的神秘人物是一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安.html" target="_blank">
  西安
 </a>
 高官。
 <br/>
 <br/>
 在习近平多次要求彻查下，中办近日以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违反政治纪律.html" target="_blank">
  违反政治纪律
 </a>
 ，定性
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陕西.html" target="_blank">
  陕西
 </a>
 官场涉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦岭.html" target="_blank">
  秦岭
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建.html" target="_blank">
  违建
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_别墅.html" target="_blank">
  别墅
 </a>
 案。随后，中共多个省部级官场，甚至基层都开始被要求学习中办下达的通报。
 <br/>
 <br/>
 11月15日，《联合早报》援引北京分析人士的消息称，从中办通报以及各省级党委高调学习贯彻通报的行动看，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高层.html" target="_blank">
  高层
 </a>
 已将陕西和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安.html" target="_blank">
  西安
 </a>
 制止秦岭违建别墅不力问题，上升到违反政治纪律的高度。
 <br/>
 <br/>
 报导说，在中共话语体系中，严重违反政治纪律是一条〝重罪〞，表明高层对陕西和西安在秦岭违建别墅问题上，处置不力已经相当不满，两地或将有更多官员难保〝乌纱帽〞。
 <br/>
 <br/>
 近几个月来，陕西官场因秦岭别墅案发生大震荡，从省级到局级不断有人落马。
 <br/>
 <br/>
 8月初，履新不到6个月的西安市长安区委书记王强被替换。紧接着，曾掌管长安土地审批的卫旭峰被查，十几天前，西安市鱼化寨街道办党工委书记史凡被调查。
 <br/>
 <br/>
 11月1日，陕西省委常委、秘书长钱引安落马。11月5日，西安市市长上官吉庆，与退休两年多的西安市政协原主席程群力同时被处分。
 <br/>
 <br/>
 这些落马官员不仅相互之间有勾连，落马原因都指向秦岭违建别墅案。但别墅案背后的神秘主人却始终没有被公开点名。
 <br/>
 <br/>
 陆媒近日曝光了秦岭违建别墅的奢靡和神秘的内幕。报导说，秦岭违建别墅案代表性的〝陈路别墅〞，存世超过10年之久。这座园林特色的中仿式别墅占地9,280.53平米，仅狗舍就占地78.03平米。别墅主人从西藏空运来5只藏獒幼崽，曾还空运羊肉喂藏獒。
 <br/>
 <br/>
 别墅内还有各种文物，以及一两米多高的黄花梨达摩像，大厅挂着范增的巨幅八骏图。
 <br/>
 <br/>
 报导说，这座别墅的主人真名叫陈路，是西安市一位高官的公子。对于陈路的父亲，报导中没点名。消息称，陈路的父亲就是西安市人大常委会原主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈宝根.html" target="_blank">
  陈宝根
 </a>
 。
 <br/>
 <br/>
 事实上，10月份，就有知情者透露，陈路的父亲是已退休的高官陈宝根。当时，有网友发布了一首藏头诗：〝陈年旧人今莫提，宝气珠光观者迷，根深树大惹风起，家道中落猢狲离〞，暗示陈路别墅的神秘主人是陈宝根。
 <br/>
 <br/>
 公开资料显示，现年65岁的陈宝根是甘肃人，曾在陕西、四川官场深耕23年，历任四川广元市长，陕西西安副书记、市长等。2017年2月，辞去西安人大常委会主任职务。
 <br/>
 <br/>
 消息称，近期，陕西官场大震荡中，陈宝根已被降为科员，目前仍在接受调查。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
