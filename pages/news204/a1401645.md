### 想买电动车？中共将随时锁定你位置
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】目前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 已开始对大陆110万辆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电动车.html" target="_blank">
  电动车
 </a>
 进行即时监控，且监控规模可能继续扩大，以达到监控中国电动车用户以及与外国企业竞争占优势的目的。
 <br/>
 <br/>
 《美联社》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 当局要求所有电动汽车的厂商，必须向中共设置的监控平台提供每辆车的61项数据。并要求这些电动汽车厂商通过对出厂的汽车设定自动发送信息功能，向中共提供这些数据，从而使中共随时知道这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_电动车.html" target="_blank">
  电动车
 </a>
 所在位置以及其它多项信息。但大多数中国驾驶人并不知道此事。
 <br/>
 <br/>
 这些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汽车制造商.html" target="_blank">
  汽车制造商
 </a>
 包括特斯拉、大众、宝马、戴姆勒、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福特.html" target="_blank">
  福特
 </a>
 、通用、日产和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三菱.html" target="_blank">
  三菱
 </a>
 在内的200多家公司。
 <br/>
 <br/>
 报导称，外国汽车厂商说，他们仅仅是遵从中国规定。中共官员说，中国通过对这些信息进行分析，促进公共安全、产业发展以及基础设施规划。
 <br/>
 <br/>
 不过，美国、日本和欧洲国家也是电动汽车的主要市场，而这些国家并未要求厂商实时提供这类信息，令人们对汽车制造商及中共的说辞感到怀疑。
 <br/>
 <br/>
 有批评人士认为，中共官方上述行为很可能用于社会监视及用来侵害外国车厂的竞争优势。
 <br/>
 <br/>
 据《美国之音》报导，中共对民众的监控超过绝大多数国家。中共央视在一个节目中称，中国全国有2000多万个监控摄像头，称之为〝中国天眼〞，可以识别出路上行人的年龄和性别。
 <br/>
 <br/>
 中共最近还开始实行了社会信用系统，从个人的购物等日常生活到工作中的行为以及在社交媒体是否发表批评中共的言论都是打分标准。评分低的人被禁止乘坐飞机和高铁旅行，找工作也非常困难。
 <br/>
 <br/>
 《美联社》报导称，美联社走访了上海市新能源汽车公共数据采集与检测研究中心，发现在一个墙壁大小的屏幕上，显示了很多圆点，每一个点代表着上海街道上正在行进中的车辆，这构成了一个随时移动的地图，可以很清楚地看到人们在哪里居住、逛街、工作和朝拜。
 <br/>
 <br/>
 上海市新能源汽车公共数据采集与检测研究中心官网展示的数据显示，其分析系统中包含来自一百多家汽车制造商出产的22万辆新能源汽车。（网站截图）
 <br/>
 <br/>
 随机点开一个圆点，就会弹出一个窗口，显示出标识每辆车的编号、汽车的品牌和型号、里程和电池电量等。该屏幕能显示上海超过22万2000辆汽车的数据。
 <br/>
 <br/>
 该报导还引述一份2016年中共政府的文件指出，电动车制造商会将从车上反应器收集到的车辆资讯，传给像上海这家一样的各地的数据采集和监控中心，这包含了定位资讯、电池、引擎功能等至少61项资料。
 <br/>
 <br/>
 不仅如此，这些新能源车的相关资料还会流向由北京理工大学运行的国家级监控中心。根据新能源汽车国家大数据联盟的数据，该监控中心从中国境内超过110万辆汽车提取资讯，而且数量还在不断地增加。
 <br/>
 <br/>
 中共对本国公民加强监控的同时，还不断帮助外国独裁者加强监控本国公民的设置。
 <br/>
 <br/>
 《路透社》11月14日报导，中国的中兴通讯正协助委内瑞拉建立一个〝通过身份证监控公民行为〞的系统。委内瑞拉政府正推出一种名为〝祖国卡〞的新型智能身份证，中兴在有关项目扮演核心角色。
 <br/>
 <br/>
 台媒《自由时报》30日报导，中兴通讯在委内瑞拉建立的资料库，可以透过委国的身份证追踪民众的医疗史、财务状况等，甚至包含政治倾向、投票记录都可追踪。有美国参议员克里斯・范霍伦（Chris Van Hollen）和马可・鲁比奥（Marco Rubio）质疑，中兴协助委国政府打压人民，同时也质疑中兴违反与美国的和解条件，因而建议启动调查。
 <br/>
 <br/>
 （记者欧阳静报导/责任编辑：祝馨睿）
