### 未提〝违纪〞不寻常  官方通报孟宏伟引各种揣测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】中共官方周日深夜发布了一份通报，指中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 因涉嫌违法已被调查。外界注意到，这份通报没有出现中共高官被查时都会使用的〝违纪〞一词，似暗藏玄机；同时有英媒披露，孟失踪前最后给妻子发送的短讯是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一把刀.html" target="_blank">
  一把刀
 </a>
 〞的图案。这两个反常的敏感现象引发外界的各种分析与解读。
 <br/>
 <br/>
 当地时间10月7日深夜，中共国家监察委官网发布了一则26字的短讯，通报称，〝公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 涉嫌违法，目前正接受国家监委监察调查。〞
 <br/>
 <br/>
 外界注意到，这份通报与其它中共落马官员的通报说辞有所不同，通报中只提及孟宏伟〝涉嫌违法〞，并没有出现中共官方通报惯用的〝违纪〞二字，这个细节引发外界的分析解读。
 <br/>
 <br/>
 比较普遍的一个看法是，孟宏伟不仅是中共公安部的高官，他同时还拥有国际刑警组织主席身分，他这次返回中国失踪多日后才因其妻子向法国警方报案而促使他被抓的时间曝光。因此，中共官方在对外发布的通报时，为了淡化此案与〝中纪委执行党纪〞的色彩，突出孟某某是因涉嫌〝违法〞才被查，所以才会蓄意隐去了〝违纪〞字眼。
 <br/>
 <br/>
 就在中共官方发布孟宏伟被拘留接受调查的通报前不久，孟的妻子向英国广播公司（BBC）透露，孟宏伟失联当天传来的最后讯息是一把尖刀图案，并要她〝等他电话〞。BBC的报导认为，那把刀的图案，是孟暗示自己正身历险境。
 <br/>
 <br/>
 孟宏伟在中国大陆〝失踪〞的消息被法国警方踢爆后，香港《苹果日报》曾援引〝接近调查的人士〞爆料称，孟宏伟可能在某方面〞对抗〝北京当局而被拘留。
 <br/>
 <br/>
 针对上述消息，有海外观察人士分析称，中共官方的通报只提〝违法〞不提〝违纪〞，主要用意是要突出孟所涉案情属于〝犯罪〞性质，从而间接降低外界对该案的政治性揣测所带来的负面效应，同时也在暗示该案的案情相当严重。而孟宏伟最后在危机关头发给妻子的是一副〝尖刀〞图案，或意味着孟心里很明白自己所涉案情十分严重，一旦东窗事发后果也十分严重，恐怕性命难保。
 <br/>
 <br/>
 法国《世界报》日前也评论称，虽然孟宏伟并不掌管国际刑警组织的运作实权，但作为重大跨国治安组织的主席却在回国后凭空消失，这种做法很难不让国际社会对中共当局投以质疑的眼光。但北京当局宁可冒着在国际上丢丑的政治风险也要逮人，反过来也说明孟宏伟所涉及的问题必定〝极为敏感或严重〞。
 <br/>
 <br/>
 中国网友对的孟宏伟案的分析评论则更为大胆犀利。
 <br/>
 <br/>
 有网友在海外中文媒体的相关报导下留言分析称，〝大批驻外使领馆官员屁股都不干净，恐惧同样下场，因此在任期届满前，准备策划出逃，中共高层确实乱套了，进退失据，船可能突然下沉。〞
 <br/>
 <br/>
 还有网友在留言中爆料称，〝中国（中共）大使馆里有几个神秘人物，专门监视其他人。如果有一天崔天凯大使被失踪，大家不要感到奇怪。〞
 <br/>
 <br/>
 孟宏伟的履历显示，他过去长期在中共公安系统工作，是前政治局常委、曾任公安部长及政法委书记的周永康老下属。2004年即担任公安部副部长至今，也是中共公安部党委成员，还曾兼任国家反恐办主任，
 <br/>
 <br/>
 2017年12月孟宏伟突然被北京当局免去了从2012年3月开始兼任的中共国家海洋局副局长、海警局局长的职务；今年4月又被免去公安部党委委员的头。孟宏伟接连被削权后，外界已经开始猜测他的仕途可能〝不妙〞，结果不到一年孟果然被北京当局〝拿下〞。
 <br/>
 <br/>
 （记者唐迪综合报导）
