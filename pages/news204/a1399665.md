### 【禁闻】11月16日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月17日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 退休法官黄云敏遭秘判十年
 <br/>
 <br/>
 新疆退休法官黄云敏因替当地百姓维权，去年3月被抓，之后被秘密判刑10年，11月15号，他的妻子前去监狱探视，黄云敏戴着手铐脚镣来会见。因黄向妻子表示他不服判决，双方会见仅一分钟，就被狱警中断，狱警当黄妻的面卡住黄的脖子，将他拖离座位进行殴打，还警告黄妻，不准告诉外界，黄云敏遭殴打的情况。
 <br/>
 <br/>
 2. 哈尔滨残疾人遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_逼迁.html" target="_blank">
  逼迁
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 探视被撞伤
 <br/>
 <br/>
 黑龙江哈尔滨市松北区残疾人张铁华一家，遭遇当地拆迁部门倒土石封门逼迁，当地访民孙东生10月30号前往探视，被倒土车故意撞伤，警察到场按交通事故处理，11月15号才将事故责任认定书送到。
 <br/>
 <br/>
 3. 福建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碳九泄漏.html" target="_blank">
  碳九泄漏
 </a>
 受害村民堵路抗议
 <br/>
 <br/>
 福建泉州市泉港区的碳九泄漏事件已过去10多天，受害的村民仍然未从政府部门得到任何说法，最近几天，受害的下朱尾村等村民，在村路口堵路抗议。
 <br/>
 <br/>
 4. 广西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_业主.html" target="_blank">
  业主
 </a>
 上街 抗议
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开发商.html" target="_blank">
  开发商
 </a>
 未兑现承诺
 <br/>
 <br/>
 广西南宁大型楼盘〝恒大绿洲〞的150多名业主，11月11号上街维权，抗议开发商恒大地产集团未兑现绿化建设的承诺，他们打出大型条幅、高喊口号，要求开发商整改。
