### 美中期选举结果出炉 中国股市应声下跌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】11月7日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 大局已定，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 所在共和党失去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 主导权，但保住了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 。欧美股市随后上涨，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 应声下跌。多有分析称，美国对中共的贸易战将继续推进。
 <br/>
 <br/>
 美东时间11月7日凌晨，中期选举投票结束，开始统计选举结果。截至7日6点20分，共和党确定拿下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 100个席位中的51席（42席不改选），笃定成为多数党。民主党拿下45席（23席不改选）。还有3席待定。
 <br/>
 <br/>
 与此同时，民主党确定已拿下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 435席中的222席，成为多数党。共和党拿下199席。还有14席待定。
 <br/>
 <br/>
 实际上，早在7日零时左右，两院选举就大局已定。零时55分，共和党的参议院席位再添一席，确定成为多数党。零时50分，众议院少数党领袖南希.佩洛西（Nancy Pelosi）即宣布民主党重夺众议院控制权。
 <br/>
 <br/>
 据陆媒新浪财经11月7日报导，截至发稿，美国道指期货上涨184点，涨幅为0.7%；标普500期指涨0.9%，纳指期货涨1.2%。欧洲的Stoxx 600指数上涨1.1%。
 <br/>
 <br/>
 而据日本广播协会（NHK）11月7日报导，中国沪市当日上午综合指数一度超过6日收盘指数0.6％。但是，从午后（即美东时间凌晨）开始，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 传来战况，沪市股指在下午出现震荡。最终，沪市股指当天收盘指数比6日低0.7％。
 <br/>
 <br/>
 陆媒券商中国当日亦报导，截至发稿，深成指下跌0.45%，创业板指下跌0.06%。港股市场午后也出现下跌，恒指盘中一度跌逾0.4%，
 <br/>
 <br/>
 此外，日经指数当日日小幅高开后盘初小幅跳水，再度走高逾后呈震荡下行趋势。截至收盘，日经225指数收跌0.28%；东证指数收跌0.4%。韩国首尔综指高开高走，午后亦持续下行，跌幅达0.73%。
 <br/>
 <br/>
 日本《朝日新闻》11月6日报导指，失去众议院将让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 面临更多国内压力，为了两年后的大选做准备，向支持者展现自己的力量，川普可能会推出更加强硬的政策。很多人认为，中期选举过后，美国对其它国家的压力会减小，但伊朗、日本、中共面对的局面将更严峻。
 <br/>
 <br/>
 美国财经新闻频道（CNBC）11月5日援引班诺克本全球外汇（Bannockburn Global Forex）首席市场策略师钱德勒（Marc Chandler）的分析指，如果民主党夺回众议院，中美贸易战将由国会两党一起进行，因为中共不公贸易行为伤害美国已经是两党共识。
 <br/>
 <br/>
 彼得森研究所（Peterson Institute）高级研究员胡夫鲍尔（Gary Hufbauer）表示，基于上周二（10月30日）的一项民调结果，预计中美贸易战将持续至川普的整个总统任期。
 <br/>
 <br/>
 此外，英国《金融时报》亦引述观点称，如果民主党上台，对中共的政策可能比川普更加强硬，因为这是美国整个政治经济风向的一次大转变，跟党派没有关系。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
