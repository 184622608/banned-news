### 【禁闻】11月23日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月24日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 安徽扬子壁纸公司工人拉横幅讨薪
 <br/>
 <br/>
 安徽省滁州市扬子壁纸公司从2017年起拖欠工人工资，11月21号，工人集体在公司门口拉横幅讨薪。
 <br/>
 <br/>
 2. 安徽巢湖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出租车罢工.html" target="_blank">
  出租车罢工
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗议.html" target="_blank">
  抗议
 </a>
 气价上涨
 <br/>
 <br/>
 安徽省巢湖市天然气价格上涨，导致出租车运营成本上升，11月21号，全市数百辆出租车全部停运抗议，造成交通紧张，最后迫使政府部门不得不下调气价。
 <br/>
 <br/>
 3. 辽宁省朝阳建筑工人爬塔吊讨薪
 <br/>
 <br/>
 辽宁省朝阳市（音:chaoyang）建平县一个建筑工地的7名工人，因被拖欠工资，11月20号爬上塔吊讨薪，所幸最后被救援下来，没有发生悲剧。
 <br/>
 <br/>
 4. 陕西百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货车司机.html" target="_blank">
  货车司机
 </a>
 抗议被禁通行
 <br/>
 <br/>
 陕西省西安市政府日前发出通知，禁止一些型号的柴油车在市内路段通行，11月19号，上百名受到影响的货车司机，拉横幅抗议。
</div>
