### 【禁闻】中共央行再降准  印钞拉动经济不再灵
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共央行.html" target="_blank">
  中共央行
 </a>
 进行了今年内第四次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降准.html" target="_blank">
  降准
 </a>
 ，不过，并没有引起市场的积极反映，股市和汇市反而大跌。专家指出，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印钞.html" target="_blank">
  印钞
 </a>
 已经上瘾，但中美贸易战几乎断绝其从美国捞钱的出路后，这种唯一依靠开动印钞机拉动经济的方式，带来的只会是更大的麻烦，却不能再营造假繁荣的假象。
 <br/>
 <br/>
 10月7号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共央行.html" target="_blank">
  中共央行
 </a>
 宣布，从10月15号起，下调金融机构存款准备金率1个百分点。在置换到期的中期借贷便利操作后，预期可额外释放约7500亿元人民币。
 <br/>
 <br/>
 大陆金融分析师任中道：〝按照货币乘数5来计算，有3、4万亿的货币派生出来，意思是说，上一次金融危机4万亿，到现在都变成了小儿科，所以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 就是靠不断释放货币来进行拉动。〞
 <br/>
 <br/>
 2008年全球金融危机时，中共通过投放4万亿拉动基建，虽然造成了中共经济不受世界金融危机影响的假象，但是造出的鬼城和多余的交通设施等过剩产能，已经成为中国经济的严重拖累，那次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降准.html" target="_blank">
  降准
 </a>
 同样是1%。
 <br/>
 <br/>
 在过去十年，除2015年股灾之后，降准达到过1%以外，一般降准幅度只是大约0.25个百分点。
 <br/>
 <br/>
 这次再降准1%，是中共今年第四次降准，其投放的货币量超过前面三次降准总和的一半。
 <br/>
 <br/>
 中国金融智库研究员巩胜利：〝通常是0.25，稍微严重一点的是0.5，如果再严重是0.75，一般超过1都是很重大事件，也就是説流通领域货币短缺可能比想像的更严重一点。〞
 <br/>
 <br/>
 中国金融智库研究员巩胜利估计，这次中共央行宣布降准，是为了应对上周美国公布强劲经济数据推动美国国债收益率飙升，造成全球股市下跌，大部分国家货币贬值的场面。
 <br/>
 <br/>
 不过，人民币并没有因为降准而占稳，〝十一〞长假后的第一天开盘，跌去了300点，跌破6.91元，创下8月中旬以来的新低。
 <br/>
 <br/>
 大陆金融分析师任中道认为，中共通过释放流动性拉动经济，已经上瘾却又左右为难。
 <br/>
 <br/>
 任中道：〝中共央行左右为难的一个意思，就是它不可能全都保住，不可能在货币宽松的情况下还能保汇率，这几天人民币对美元又开始要破7的状态，目前的迹象来说，它很可能要放弃汇率了，为了国内经济只能不断的采用货币工具去释放流动性。〞
 <br/>
 <br/>
 任中道透露，加上其他货币工具，实际上，这次已经是中共今年第8次投放货币。
 <br/>
 <br/>
 巩胜利认为，中共这种继续开动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印钞.html" target="_blank">
  印钞
 </a>
 机提振经济的方式，带来的只会是麻烦。
 <br/>
 <br/>
 巩胜利：〝现在很麻烦啦，全球的各国货币都进入加息的通道，就是收回货币的通道，而现在人民币是在放水，不放就缺钱了嘛，这是一个难题，那美元一直加下去，人民是不是一直放下去？放水多了以后一个是人民币面临贬值，那么贬值以后，这些钱是进中国股市还是房地产，这是一个难题。如果美元走入减息的周期了以后，那中国再怎么办，再加息？按道理来讲全球货币都是跟着美元的。〞
 <br/>
 <br/>
 巩胜利説，由于美国的经济太强盛，美元一定会继续加息，中共走一条背道而驰的路，随着中美贸易战的持续会越来越麻烦。
 <br/>
 <br/>
 巩胜利：〝如果贸易战打中期或长期的话，那人民币会放更多的水，为什么，因为贸易战打的主题就是减少美元的中国顺差，接着放水很麻烦，因为货币发行量太大了，放出来的钱太多，如果不放，闹钱荒了怎么办？放出来以后，人民币贬值就是不可能阻挡。〞
 <br/>
 <br/>
 实际上，这次降准也没有达到救市营造繁荣假象的目的，长假后的第一天股市开盘与汇市一样跌得很惨，上证综指跌去3.72%，报收2716.51点，并且下跌范围广泛，连受青睐的银行股也没能逃脱。
 <br/>
 <br/>
 《华尔街日报》的报导説，这次下跌进一步印证了经济增长放缓情况下中国投资者异常悲观的情绪。
 <br/>
 <br/>
 采访编辑/刘惠  后制/周天
