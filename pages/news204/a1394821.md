### 税务总局新公文出台 崔永元3000多份〝合同〞废了？
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】央视前主持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 曾告诉媒体，他通过不同渠道得到3000多〝天价
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞的相关材料，准备提交给税务部门。然而随着税务总局最新公文出台，崔永元手中的〝阴阳合同〞似乎将变成了一堆废纸。
 <br/>
 <br/>
 10月9日，中共官媒报导，中共国家税务总局日前印发《关于进一步规范影视行业税收秩序有关工作的通知》(以下简称〝通知〞)，明确针对近期税务机关，查处的影视行业高收入从业人员偷逃税等问题，根据《税收征管法》相关规定，进一步规范影视行业税收征管秩序。
 <br/>
 <br/>
 通知称，从2018年10月10日起，各地税务机关通知本地区的影视制作公司、经纪公司、演艺公司、明星工作室等企业及影视行业高收入从业人员，对2016年以来的申报纳税情况进行自查自纠。
 <br/>
 <br/>
 凡在2018年12月底前，认真自查自纠、主动补缴税款的影视企业及从业人员，免予行政处罚，不予罚款。
 <br/>
 <br/>
 从2019年1月至2月底，税务机关根据纳税人自查自纠等情况，有针对性地督促提醒相关纳税人进一步自我纠正，对经税务机关提醒后，自我纠正的纳税人，可从轻或减轻行政处罚；对违法情节轻微的，可免予行政处罚。
 <br/>
 <br/>
 从2019年3月至6月底，税务机关结合自查自纠、督促纠正等情况，对个别拒不纠正的影视行业企业及从业人员开展重点检查，并严肃处理。
 <br/>
 <br/>
 也就是说，按照上述《通知》规定，凡是今年底以前主动补缴税款的，不但可免于行政处罚，而且不予罚款。即使此次自查自纠中自己没发现问题，而被税务机关查获的，同样也可从轻或减轻行政处罚。
 <br/>
 <br/>
 这也就等于即使有人举报或被税务机关查获，仍然没有坐牢的风险。因此有观点认为，这等于废了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 手中的3000多份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 ，白费了崔冒着生命危险弄来的一抽屉阴阳合同。
 <br/>
 <br/>
 崔永元5月底在微博称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 采用〝阴阳合同〞，只在片场演出4天，就拿走了6000万元人民币的片酬，涉嫌偷税。引来中共税务总局的介入调查。
 <br/>
 <br/>
 之后，崔永元在接受陆媒采访时放话：自己〝还有一抽屉这样的合同〞，〝随便拿出来一个当事人都得进去，不光范冰冰，牵扯到的全是大腕儿！〞
 <br/>
 <br/>
 9月24日，崔永元又对《明报》称，4个月以来，他通过不同渠道得到3000多份〝阴阳合同〞的相关材料，仅华谊兄弟影业的就有1000多份。
 <br/>
 <br/>
 他还称，自己也接到利益集团或范冰冰粉丝的死亡威胁，其间他报警近10次，均未立案。他身边的工作人员也频频被约谈或传唤，使其仍存担忧。
 <br/>
 <br/>
 他还说，已经将部分重要资料备份，藏于多处，若遇危险会将其公之于世。
 <br/>
 <br/>
 10月3日，官媒称，范冰冰涉〝阴阳合同〞逃税案，被罚8.8亿人民币免予刑责。
 <br/>
 <br/>
 事后，崔永元说，自己实名举报范冰冰，获得10万人民币奖金，没心情领取。他还称，自己遭到警方抄家式的报复侦查。
 <br/>
 <br/>
 如今不少人指望掀起一场更大的痛打影视圈大腕的好戏估计也要收场了。
 <br/>
 <br/>
 10月10日，一段关于崔永元的视频传出，崔颓废地坐在一间屋子的角落说，自己很忙，有很多事情要做。目前这段视频已被删除。
 <br/>
 <br/>
