### 歼20飞行员神秘死亡 疑似反常升空画面曝光（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】中共空军
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_歼-20战机.html" target="_blank">
  歼-20战机
 </a>
 飞行员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_廖勇.html" target="_blank">
  廖勇
 </a>
 25日被证实在执行飞行任务时死亡，官方尚未公布其死亡原因。有消息人士称，廖勇可能是在本月
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海航展.html" target="_blank">
  珠海航展
 </a>
 期间发生意外，网路上也传出一段当时战机反常升空的视频。
 <br/>
 <br/>
 编辑部设在北京的多维网引述知情人士称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_廖勇.html" target="_blank">
  廖勇
 </a>
 死亡的具体日期是今年11月9日，正值当局举办2018
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠海航展.html" target="_blank">
  珠海航展
 </a>
 期间。当天3架歼20现身珠海航展上空，进行编队飞行表演。有分析认为，廖勇可能死于歼-20事故。报导还称，如果属实，那会是歼-20首开〝坠毁先例〞。
 <br/>
 <br/>
 11月27日，多维网又引述消息人士称，廖勇出事或由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_歼-10B.html" target="_blank">
  歼-10B
 </a>
 。9日下午，歼-10B突然飞走了，当时在场外迫降未成功。报导还附上一段歼-10B验证机疑似正从珠海机场飞离的视频。
 <br/>
 <br/>
 报导称，11月9日，有大陆网友拍到这段视频，画面显示这架歼-10B并没有采用中规中矩的起飞方式，而是如同表演一般，离地后平飞加速，之后一口气拉起机头垂直爬升，还在大迎角姿态下做了滚转动作。尚不清楚该视频是否与廖勇死亡有关。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 不过，外界注意到，廖勇死亡的消息被当局封锁了两周后才曝光。11月26日，有官方背景的〝新湖南〞网媒公布照片显示，廖勇的骨灰已被运送至家乡。但中共军方至今没有公布廖勇死亡的具体原因。
 <br/>
 <br/>
 目前对于廖勇死亡的原因众说纷纭，另有消息说，珠海航展期间，原定的歼-16、歼-10C战机没有进行表演。廖勇当天可能是驾驶歼-10C坠机而殁。
 <br/>
 <br/>
 不过，外界发现，中共军方对于歼20的事故消息似乎尤为保密。2016年下半年，中共空军第九师师长王列虎死于飞行事故，但因官方密而不宣，直到2017年3月王列虎的名字被刻上空军英烈墙后，外界才确定其死讯。
 <br/>
 <br/>
 中国人权民运讯息中心引述消息人士指，2016年下半年，中共空军开始进行〝红军〞与〝蓝军〞的对战，使用的战机为歼10和歼20。王列虎及其他一些空军高级军官亲自驾机测试，随后发生意外。因涉及歼20敏感讯息，当局对王列虎丧生的消息三缄其口。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：文慧)
