### 华日：中美关系剑拔弩张 伊朗石油将成最新引爆点
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】美国对伊朗的新制裁措施即将生效，目标是全面封杀伊朗的石油出口。中共作为伊朗最大石油进口国，首当其冲遭受影响。有分析指，目前美中关系剑拔弩张，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗石油.html" target="_blank">
  伊朗石油
 </a>
 或成为新的引爆点，一旦〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_石油战.html" target="_blank">
  石油战
 </a>
 〞开打，中共会更痛更难过。
 <br/>
 <br/>
 11月4日，美国对伊朗的新制裁措施将全面生效，美国已经讲明，其目标是让伊朗的石油出口量从每天170万桶下降至零。
 <br/>
 <br/>
 在此之前，美国总统川普（特朗普）5月8日宣布华盛顿退出伊朗核协议，并称将恢复所有此前在该协议框架内暂停的对伊朗制裁。
 <br/>
 <br/>
 美国重启制裁伊朗分两个阶段进行，自8月7日起，对伊朗实施首波经济制裁，以金融交易为主，并预计于11月4日，执行第2波制裁，主要目标为石油、石化产业。
 <br/>
 <br/>
 美国财政部长史蒂文．努钦7月曾发出警告，美国会对任何继续购买
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗石油.html" target="_blank">
  伊朗石油
 </a>
 的国家进行制裁，包括中共、俄罗斯。川普政府随后表示，已有50多家国际企业宣布停止与伊朗的业务往来。
 <br/>
 <br/>
 而在伊朗石油买家中，中共近年已成为伊朗石油的最大进口国，伊朗出口的石油有大约三分之一卖给中共。
 <br/>
 <br/>
 华尔街日报报导，国际主要石油贸易公司PVM Oil Associates石油分析师Stephen Brennock称：〝考虑到两国关系日益恶化，中国（共）不会甘心向美国屈服。〞这意味着又一场贸易争斗一触即发。
 <br/>
 <br/>
 早在7月，美国发出要求全球买家11月前停购伊朗石油的声明。中共外交部立即回应，称中方会与伊朗保持经贸与能源合作。并强调，中共与伊朗的贸易关系〝开放、透明而且合法〞。
 <br/>
 <br/>
 对于中共的回应，被任命为美国国务院伊朗行动小组主管和特别代表的胡克（Brian Hook）表示，〝我们准备对那些继续跟伊朗贸易的政府实施二级制裁。〞
 <br/>
 <br/>
 这不禁让人联想到中国通讯制造商〝中兴通讯〞被制裁一事。〝中兴通讯〞因违反美国对伊朗的制裁对其出售产品，被禁止购买美国公司的芯片及软体，最终以缴纳逾10亿罚金代替禁售令制裁。
 <br/>
 <br/>
 有分析人士称，如果真如中共所言不违法，中兴通讯为何会无异议的缴纳天价罚款。
 <br/>
 <br/>
 美国对伊朗恢复制裁，旨在施压德黑兰（Tehran）当局停止支持国际恐怖主义、撤回发展核武的计划。迄今很多公开资讯直指，今天伊朗的核问题如同朝鲜问题一样，都与中共核扩散、核支持有很大的关系。
 <br/>
 <br/>
 美国国务卿蓬佩奥曾呼吁美国盟友，联手对伊朗施加经济压力，他还在推特上贴文说，要切断伊朗用于资助恐怖主义和代理战争的所有资金。
 <br/>
 <br/>
 外界认为，一旦美国落实制裁伊朗石油，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_石油战.html" target="_blank">
  石油战
 </a>
 〞或继
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 之后被点燃，届时中共会更痛更难过。
 <br/>
 <br/>
 Eurasia Group中国项目负责人Michael Hirson对华尔街日报表示：〝按照我们预测的基本情境，双方将避免就伊朗制裁爆发严重冲突，但这仍是一个潜在冲突点。〞
 <br/>
 <br/>
 报导称，伊朗的决策层现在保持克制，希望将来能有立场更温和的人接替川普，看看2020年如何发展。中共决策层似乎也做出了类似的决定。不过，没有几个投资者愿意担惊受怕熬过两年。围绕伊朗制裁的对决，仍是不可忽视的风险因素。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
