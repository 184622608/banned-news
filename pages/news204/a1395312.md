### 范冰冰遭重罚  李晨〝憔悴暴瘦〞近照曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】因卷入〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞逃税案，女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 失踪4个月后终于现身，被罚款8亿人民币免予刑责。她的男友
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李晨.html" target="_blank">
  李晨
 </a>
 发文送暖，〝甭管多难，咱们一起度过去〞。但日前李晨近照曝光，双颊凹陷暴瘦。
 <br/>
 <br/>
 10月13日，中国女星杨采钰在微博PO出新戏《七日生》杀青照，意外泄漏了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 的未婚夫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李晨.html" target="_blank">
  李晨
 </a>
 最新状况，画面中的李晨露出浅笑、双手抱胸。对比过去，可以明显发现他双颊凹陷，瘦到下巴尖了不少，有粉丝直呼心疼。
 <br/>
 <br/>
 〝怎么觉得李晨憔悴了很多〞、〝看来美国拍戏真的很辛苦〞、〝八个亿不见了，换我也憔悴〞、〝看来没少为女朋友范冰冰操心〞。
 <br/>

   </font>
  </a>

 </center>
 <br/>
 范冰冰在公众视野里消失4个月后，中共官媒10月3日报导，范冰冰因涉〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞案，被罚需缴8.8亿元人民币才能免除牢狱之灾。
 <br/>
 <br/>
 当天范冰冰在微博发表致歉信称，深感羞愧，经历从未有过的痛苦，自己将全力筹措资金缴清罚款。并向社会、向爱护关心她的朋友，以及大众道歉。
 <br/>
 <br/>
 尽管不少网友认为，近9亿元对范冰冰这样明星的大腕来说不算什么，因为范冰冰连续5年蝉联〝福布斯中国名人榜〞第一名，但有媒体爆料，范冰冰欲抛41间房套现，李晨已在中介挂牌出售自家在北京二环的四合院帮女友凑款。
 <br/>
 <br/>
 据微博大V〝留几手〞3日公开爆料，李晨家在二环有个大四合院，房屋的详细信息已挂在中介的网上。该爆料者，还针对有网友在范冰冰出事后，对李晨的指责评论道：〝一个男人，说什么不重要，重要的是看他干什么了。〞
 <br/>
 <br/>
 文中还大赞李晨〝是有情有义的好男人〞，明显表明李晨在为范冰冰卖房筹款了。
 <br/>
 <br/>
 李晨早前也曾在综艺节目中透露，自己小时候经常去北京故宫玩，从自己家到故宫骑车也就半小时，节目还录到他带客人去四合院的家里玩，还有他母亲出来迎接客人的画面。
 <br/>
 <br/>
 范冰冰没有露面之前的9月16日，是她37岁生日，因李晨当天没在微博祝福，一度招致范冰冰的许多粉丝讨伐。事后，范冰冰的御用造型师卜柯文公开在微博发声说，李晨和范冰冰是想低调过生日，不发微博是不想听到乱七八糟的声音。
 <br/>
 <br/>
 微博中还公布了李晨在朋友圈生日祝福的截图，显示送祝福的时间是凌晨5点20分，谐音是〝520（我爱你）〞，力证两人感情如初。
 <br/>
 <br/>
