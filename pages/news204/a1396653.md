### 【李天笑快评】四中全会有重大决策出台吗？
------------------------

<div class="wysiwyg">
 <center>
  <br/>
  <br/>
 </center>
 <br/>
 <br/>
 【新唐人北京时间2018年10月25日电】迄今为止，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 召开的日期还没有宣布，这主要与对11月6日美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 的评估有关。其实中期选举结果对贸易战影响不大。这是因为：一、决定贸易战力度和时间表的是川普行政当局，而不是国会；二、国会两党对纠正中美贸易不公正状态的看法是一致的，这是自上世纪40年代以来首次发生。三、由于川普政绩惊人，民众支持率超高，共和党选民人气高旺，投票率会很高，这次很可能导致类似2016年川普大选时的黑马效应，即共和党议员大获全胜的意外结果。
 <br/>
 <br/>
 现在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 面临什么情况呢？如果习川不在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 上达成一个使贸易战缓和下来的口头协议或默契，那么川普5000亿和先前预定的25%的关税惩罚就会接踵而来，不管中方采取什么稳定措施，都难避免经济下滑加剧和社会不稳甚至动荡，江派残余势力可能会乘机蠢蠢欲动。
 <br/>
 <br/>
 在这种形势下，四中全会有重大决策出台吗？我觉得这种可能性是存在的。从某种意义上说，支持民营企业、说一些坚持改革开放、或“稳就业、稳金融、稳外贸、稳外资、稳投资、稳预期”笼统的老话，都只能算既定政策。重大决策就两点：1、从实质上满足美方要求的结构和体制改革；2、从抓捕和审判江泽民入手，彻底清除江派残余势力和江派留下的经贸体制。习近平能不能使贸易战发生转机，这取决于习如何判断目前局势。对此我有几点观察：
 <br/>
 <br/>
 1、今年是所谓改革开放40周年，实际是个启动转机的绝佳机会，习可以把美方要求的所有结构性、体制性改革纳入到深化改革的大框架中去，作为深化改革开放的措施提出来。这么做对内对外都不会失面子。习目前的二度南巡、抓捕孟宏伟、严厉清算房峰辉张扬政治罪行等举动应该是坚持改革和继续打江的信号。习会不会用改革的名义启动美国要求的结构性改革，这是四中全会的主要观察点。
 <br/>
 <br/>
 2、习有没有这个智慧，会不会这么做，取决于习如何看待中美贸易战的原因。中美贸易磨擦并非是应对问题，也不是多买一点美国货，根本原因在要废除江执政时期和江架空胡时期形成的掠夺性经贸结构和体制，中美成为平等、公平的贸易伙伴。所以贸易磨擦的罪责事实上应归咎于江泽民，习不必为江扛罪顶罪。从这点上看，贸易战为继续清除和清算江派注入了新的动力，体现出必要性和紧迫性；同时这也是解决中美贸易磨擦的根本出路。
 <br/>
 <br/>
 3、抓捕江泽民是平息各种社会危机的捷径。19大以来，由于江泽民等首犯逍遥法外，江派残余心存侥幸，一直在捣乱搅水。江派的策略是：利用各种社会矛盾制造事端，迫使习加强管制手段，引起社会反弹后，向习发难，迫使习下台。所以习近平不抓捕江泽民，不除恶务尽，就危机不断，失去民心，自身难保。
 <br/>
 <br/>
 4、江的罪行与中共是分不开的：江的贪腐罪、政变罪、卖国罪、迫害法轮功活摘器官的反人类罪等，都是依靠中共的体制、资源、组织原则、党文化等实现的。中共的罪行已大到必须解体，而清算江泽民必然导致清算和解体中共。国际上都把中共视为邪恶的洪水猛兽川。普选择贸易为突破口是对准中共的，而不是针对中国和中国人民的。这个突破口选的非常好（如选择军事位突破口，就可能演变为战争）。从经贸改革发展到政治改革，顺理成章。所以，习在利用最高权力做完该做的事后，必须转换权力基础，抛弃中共这只沈船，把民意作为权力基础，避免随船而沈。
 <br/>
 <br/>
 5、江泽民不但罪大恶极，而且奄奄一息，随时一命呜呼。如果习不及时抓捕审判江，江对国家和人民犯下的罪行，很可能就要由习来承担。这是一件迫在眉睫、又是非常现实的事。
 <br/>
 <br/>
 总而言之，习在四中全会上确实有启动转机的选择，这不但对解决
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易.html" target="_blank">
  美中贸易
 </a>
 战，对国家民族的前途，对自身的安危都有重大影响。这是四中全会最主要的看点。习真的应该把握机遇，顺应天意民意而行，不错失良机。
 <br/>
 <br/>
