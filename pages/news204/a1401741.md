### 中国手机巨头负债280亿 豪赌前变卖房产藏内情
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赌博.html" target="_blank">
  赌博
 </a>
 输掉人民币十几亿之后，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手机巨头.html" target="_blank">
  手机巨头
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金立集团.html" target="_blank">
  金立集团
 </a>
 创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘立荣.html" target="_blank">
  刘立荣
 </a>
 因欠下巨款，已被执行人列入失信被执行人名单。陆媒3日引述消息称，刘立荣赌博前就卖掉了深圳车公庙附近的房子，涉嫌有预谋性的洗钱出逃。
 <br/>
 <br/>
 12月3日，腾讯网报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘立荣.html" target="_blank">
  刘立荣
 </a>
 总债务高达280亿元（人民币，下同），牵连400多家大大小小的企业欠款，10余家上市公司。
 <br/>
 <br/>
 11月28日，一场名为债务8,000万元以上的经营性债权人会议，在金立总部召开。与会人士称，会议现场绝大多数人都同意债务重组，认为这样总比破产清算好一些。
 <br/>
 <br/>
 该与会人士表示，会议上并未透露有具体投资方愿意接手，没有实质性内容，意向书也没有，仅仅是听取经营性债权人对重组债务的意见。目前也未得到金立详细的财务数据，仅仅看到总资产202亿元，总负债约为280亿元。
 <br/>
 <br/>
 〝这个数据我们也不太相信，因为是去年的，今年最新的，除了金立本身，没人清楚〞，该与会人士说。
 <br/>
 <br/>
 11月24日，界面新闻援引接近金立股东消息人士的话称，刘立荣在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赌博.html" target="_blank">
  赌博
 </a>
 上输了超过100亿。股东们推测，刘立荣挪用公款的数目可能在60亿左右。但赌博地点不是在香港、澳门，而是在塞班。
 <br/>
 <br/>
 针对此事，处于风波中的刘立荣，当天在香港接受采访时坦承参与了赌博，但否认挪用60亿公款。他说自己创办金立16年了，个人没有其他收入，难免有借用公司资金的行为。至于输了多少钱，他说：〝十几个亿吧。〞
 <br/>
 <br/>
 对此，腾讯网引述知情人士表示，〝刘立荣的赌博，明显是涉嫌洗钱行为。公司正常怎么可能有这么大的窟窿，太可怕了〞。他说，刘立荣早在2016年就卖掉了深圳车公庙附近的房子，这属于有预谋性的洗钱、出逃。
 <br/>
 <br/>
 金立手机在2006年到2008年两年的时间里，成为中国国产手机的第一大品牌。但在2017年底，金立的净负债就已高达80亿，资不抵债。
 <br/>
 <br/>
 金立曝资金危机后，刘立荣就一直呆在香港。刘立荣曾对媒体说，金立的资金链危机主要是2016年和2017年的营销和投资费用超限。
 <br/>
 <br/>
 今年7月份开始，金立分别被深圳市福田区法院和深圳中院列为失信人。8月4日深圳市中法查封了金立通讯名下以及刘立荣名下的5辆车、26套房产等。
 <br/>
 <br/>
 8月12日，深圳中院公告中明确表示，刘立荣作为被告〝下落不明〞。
 <br/>
 <br/>
 网上曝光的一份落款日期为11月14日的合同纠纷执行裁定书显示，因金立无法偿还债务，兴业银行请求法院强制其偿付高达2.06亿元的债务及利息。
 <br/>
 <br/>
 11月20日，近20家金立供应商，向深圳中院提交对金立进行破产重整的申请。
 <br/>
 <br/>
 一位参与申请破产重整的供应商说，他们已给金立足够的时间，但金立并没有拿出任何有效挽救措施，他们认为碰到了无赖，最终决定让法院进行裁决。
 <br/>
 <br/>
 金立是中国移动设备研制与软体开发企业，成立于2002年，旗下的手机品牌曾一度占据中国市场约5%的份额，也曾是印度手机市场份额最大的中国品牌。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
