### 川习会悬而未决  曝习近平提条件不见纳瓦罗
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】有关〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞将于阿根廷G20峰会期间举行的消息，日前再传新的变故。知情人爆料称，中南海御用团队已经为〝川习会〞制定了多种预案，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 也为〝川习会〞提出了一大〝条件〞，不希望见到川普总统的助理、美国国家贸易委员会主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 （Peter Navarro）。
 <br/>
 <br/>
 11月30日至12月1日，20国集团会议将在阿根廷布宜诺斯艾利斯举行。《华尔街日报》引述消息人士的话爆料，美国总统川普（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，计划在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_20国峰会.html" target="_blank">
  20国峰会
 </a>
 期间举行会晤。中方希望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晤，对中美贸易关系产生积极的影响。
 <br/>
 <br/>
 10月14日，有美媒引述中共驻美大使馆知情者的话表示，中南海御用团队已经为〝川习会〞制定了多种预案，中方原本希望巴布亚新几内亚APEC峰会举行〝川习会〞，但是被川普拒绝了，而改由副总统彭斯取而代之。
 <br/>
 <br/>
 知情者爆料：〝中南海核心智囊和御用团队正在积极筹备〝川习会〞，但是却绕不开一个颇令习近平头疼的〝关键人物〞，〝习近平为‘川习会’明确提出了一大条件——非常不希望见到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 ！〞
 <br/>
 <br/>
 纳瓦罗（Peter Navarro）是川普总统的贸易顾问、白宫贸易和制造业政策办公室主任，一名强硬的中共批评者。
 <br/>
 <br/>
 纳瓦罗也是《致命中国：中共赤龙对人类社会的危害》的作者，书中强烈抨击中共滥用贸易，〝世界上人口最多的国家正迅速变成最大的经济体，快速变成全球最厉害的刺客，不道德的中共企业家用致命的产品淹没世界市场，中共采取非法保护主义，掠夺美国的产业和工作机会。〞
 <br/>
 <br/>
 纳瓦罗表示，中美间的贸易不正常，而半数以上的中国优势却来自八项不公平的贸易手段，任何一项在自由贸易正常规则下都被明确禁止。他还指责中共进行货币操控等，强烈要求中共降低美国贸易逆差。
 <br/>
 <br/>
 6月份，纳瓦罗还曾细数有关中共对美国经济与国家安全造成损害的6大策略及50多种行为，从繁琐的行政审批与许可程序，到通过《反垄断法》进行敲诈，再从网络间谍活动、货币操纵与贬值，以及向发展中国家设置债务陷阱等等，无所不包。
 <br/>
 <br/>
 在纳瓦罗对中共强烈抨击不久，同样是6月，白宫发布了一份长达65页的《中共经济侵略如何威胁美国和全球科技及知识产权》，强烈指责中共发动系统性的〝经济侵略〞。
 <br/>
 <br/>
 《华尔街日报》披露，这份报告的主要执笔人，正是纳瓦罗。因此中共领导人不愿见纳瓦罗以免场面尴尬也不足为怪。
 <br/>
 <br/>
 美国国家安全顾问博尔顿（John Bolton）10月12日在接受美国电台访问中也指责中共在贸易、国际事务、军事和政治领域的行为需要调整，他说，两国领袖11月底会面期间，将讨论这些问题。
 <br/>
 <br/>
 博尔顿还批评中共在南海的行为危险，强调美国会捍卫国际海上通道的开放。9月底，中共驱逐舰在南海南薰礁附近驶近美国海军驱逐舰狄卡特号，几乎相撞。
 <br/>
 <br/>
 对于中共经济及军事的扩张，博尔顿说，中共是违反国际贸易和商业规范得来的。如果他们不被允许窃取美国的技术，他们的军事能力将会大大降低。川普总统相信中共长期从国际秩序中占便宜，过往少有美国人出来阻止，现在是时候要做了。
 <br/>
 <br/>
 川普在接受福克斯新闻采访时表示，针对中国贸易，美国还有更多招数。他说，中共依靠占美国的便宜过的好日子太久了，〝我估计他们认为美国人是傻瓜，美国人不是傻瓜。〞
 <br/>
 <br/>
 不过，对于中美首脑是否在11月底的20国峰会期间会晤，美国财长姆努钦（Steven Mnuchin），向美媒CNBC透露，目前尚无定论。
 <br/>
 <br/>
 姆努钦表示，要确保11月份的川习会能够成行，仍有很多工作要做。但是否与中国国家主席习近平会晤，将由美国总统川普自己决定。
 <br/>
 <br/>
 美国首席中国问题权威白邦瑞(Mike Pillsbury)表示，川普同意与习近平会晤的主要原因，是中方有望在贸易问题上做出一些重大让步。
 <br/>
 <br/>
 白邦瑞说：〝中方正在研究一套方案，他们试图了解川普总统的底线在哪儿，但显然还不确定。〞而〝川习会〞是否能取得突破，这将取决于中方是否做出重大让步。
 <br/>
 <br/>
 他还表示，川普总统的理想目标是在互惠的原则下，实现美中贸易的平衡，如果中方能够提出在两年内，连续大幅削减美中贸易赤字的方案，并有助于美国增加就业，川普总统才可能会给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 暂时划上句号。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉)
