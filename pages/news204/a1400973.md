### 四川老兵维权遭暴力清场 再传流血事件
------------------------

<div class="wysiwyg">
 【新唐人2018年11月27日讯】首先来关注中国的消息。最近两天，大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵维权.html" target="_blank">
  老兵维权
 </a>
 被当局镇压的消息频传。就在昨天，一批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四川.html" target="_blank">
  四川
 </a>
 老兵在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退役军人.html" target="_blank">
  退役军人
 </a>
 事务厅〞维权时，遭当局
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力清场.html" target="_blank">
  暴力清场
 </a>
 ，有老兵被打到头破血流。
 <br/>
 <br/>
 11月26号，大批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四川.html" target="_blank">
  四川
 </a>
 退伍军转志愿兵来到〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_退役军人.html" target="_blank">
  退役军人
 </a>
 事务厅〞门口维权，没有任何官员出来接待，大批警察却迅速包围老兵，据说双方爆发激烈冲突。警察殴打老兵，并快速
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力清场.html" target="_blank">
  暴力清场
 </a>
 。
 <br/>
 <br/>
 一名老兵头部被打得鲜血直流。
 <br/>
 <br/>
 现场视频：〝习主席，请您看一下，这是四川的警察暴力清场，打压我们四川的转业老兵，请您为我作主。〞
 <br/>
 <br/>
 除了老兵，现场还有一批军嫂抗议警方打压、抓人。
 <br/>
 <br/>
 现场视频：〝你们简直是无法无天了！你们说法就是法，是不是？你们说法就是法，老百姓就是不是法了，是不是？〞
 <br/>
 <br/>
 四川老兵张先生表示，这一次当局让转业老兵在11月25号以前去登记，自己也去了。但相关部门在登记过程中耍滑头，欺骗老兵。
 <br/>
 <br/>
 四川老兵张先生：〝其中有一项，有何诉求和要求？这一项啊，那个公务员啊都没有问，我们也不知道，我们当然没有回答。这本来是一项非常重要的信息。他把这一项隐瞒了。〞
 <br/>
 <br/>
 另据《大纪元新闻网》报导，25号，约有一、两千名来自东北、山东的老兵赶赴北京，聚集在中共中央军委政治部信访处大院。个别老兵骨干被警察抓走。不到5小时所有老兵全部被清场，其中部分老兵被带去久敬庄〝黑监狱〞。
 <br/>
 <br/>
 四川老兵孙先生：〝有两个人被打了，抓了。抓了具体几个我还真不知道。〞
 <br/>
 <br/>
 四川老兵孙先生表示，目前包括自己在内的很多老兵被当局〝稳控〞，所以无法出门维权。
 <br/>
 <br/>
 近年来，中国大陆各地频频爆发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老兵维权.html" target="_blank">
  老兵维权
 </a>
 事件。老兵们的主要诉求是退伍军人的基本生活保障，提高工资待遇等。
 <br/>
 <br/>
 因应老兵们的不满情绪，今年3月中共宣布成立退役军人事务部。这个部门成立以来，每天都有大批老兵前来上访，地方维稳人员则在周边驻扎。
 <br/>
 <br/>
 9月10号，大批退役老兵再次在北京和四川成都等地维权上访。不过当局却重兵围堵，打压这些请愿老兵。
 <br/>
 <br/>
 新唐人记者熊斌、王子琦、施宜君采访报导
</div>
