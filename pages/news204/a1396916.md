### 学运领袖周勇军因藏有法轮功资料被逮捕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】中国〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_八九学运.html" target="_blank">
  八九学运
 </a>
 〞的领袖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周勇军.html" target="_blank">
  周勇军
 </a>
 今年8月中旬在在广西旅游期间，因被发现包中藏有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 资料而被警方扣留。日前有消息证实，他被关押一个多月后已被广西警方正式逮捕。周勇军是当年在北京大会堂前向北京当局递交请愿书的3名学生代表之一。
 <br/>
 <br/>
 据维权网报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周勇军.html" target="_blank">
  周勇军
 </a>
 已于9月26日被广西东兴警方正式逮捕，罪名是涉嫌利用X教组织破坏法律实施罪，现被羁押在广西东兴市看守所
 <br/>
 <br/>
 周勇军今年8月19日与友人同往东兴游玩时，在过入市边检是被警员从其背包中藏有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 资料而遭扣留，随即于8月20日被广西东兴市警方刑事拘留。
 <br/>
 <br/>
 资料显示，周勇军1967年9月16日出生于四川省蓬溪县，1989年中国发生民主运动时，正在中国政法大学85级政治系读书的周勇军是北京高校学生对话代表团的团长。
 <br/>
 <br/>
 1989年4月21日，周勇军与郭海峰、张智勇作为学生代表曾跪于北京大会堂门前，试图向北京当局递交请愿书；4月28日，周被推举为北京〝北京高校学生自治联合会〞主席；5月，他担任北京高校学生对话代表团团长，后又与韩东方、赵品潞等人组织工人参与由大学生发起的民主运动。〝六四屠城〞后，周勇军被北京警方逮捕。；
 <br/>
 <br/>
 1993年已获释的周勇军被迫流亡美国；1998年12月，因返回中国大陆又再次被中共当局以〝偷越国境罪〞抓捕并被以判处劳教3年。
 <br/>
 <br/>
 2008年9月，周勇军试图从香港罗湖进入中国大陆时再次被捕，其后被中共以〝诈骗罪〞判刑9年，在狱中曾遭到酷刑殴打。出狱后，周勇军通过自考本科毕业，并通过了司法资格考试，但始终无法获得各项合法证件。
 <br/>
 <br/>
 （记者李明报导/责任编辑：秦锋）
