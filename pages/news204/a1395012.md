### 【禁闻】不理中共要求 马来西亚释放11名维族人
------------------------

 【新唐人2018年10月11日讯】
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
   法轮功
  </a>
  成阿根廷省教育厅大型教育项目
 </b>
 <br/>
 <br/>
 在中国大陆遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 长期打压的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 ，在世界上却受到越来越广泛的欢迎。最近在南美的阿根廷，教授法轮功功法已经成为布宜诺斯艾利斯省教育厅的大型教育项目。
 <br/>
 <br/>
 〝法轮大法明慧网〞报导说，今年8月在当地学校开始的法轮功功法教授，已经收到了很好的效果。老师们反映，学习法轮功之后，一些有行为和注意力问题的学生，情况得到了明显改善。学生们也表示：炼功之后，感到放松和平静。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
   一带一路
  </a>
  再遇挫 塞拉利昂叫停机场项目
 </b>
 <br/>
 <br/>
 中共在海外推行的〝一带一路〞再次遇挫，10月10号，西非国家塞拉利昂宣布取消由中国投资建设的新机场项目，理由是该项目没有任何必要性，而且近4亿美元的费用也过于昂贵。
 <br/>
 <br/>
 据报导，这个项目由前总统科罗马与中方签署，当时世界银行和国际货币基金组织就警告，该项目会给塞拉利昂造成沉重的债务负担。今年3月，科罗马竞选失败下台，新任总统比奥重新评估了前任的合作项目后，决定取消。
 <br/>
 <br/>
 目前，塞拉利昂交通和航空部已经同时叫停了该项目旗下的所有合同。
 <br/>
 <br/>
 <b>
  中共令外国领馆不参加台湾国庆活动
 </b>
 <br/>
 <br/>
 10月10号是中华民国107周年国庆。香港自由新闻11号独家报导说，中共外交部驻港特派员公署10月5号发出外交备忘录，要求在港的外国领事馆不要出席台湾国庆日活动。
 <br/>
 <br/>
 报导引述消息人士的话披露，这份外交备忘录先送到身兼香港领事团团长的阿根廷驻港总领事那里，然后再转交到其他各国驻港领事馆。
 <br/>
 <br/>
 英文台湾日报11号报导说，中共对外国驻港机构下达行为指导令，违反了外交规则。这可能导致不满，因此有人向媒体揭露了这份备忘录。
 <br/>
 <br/>
 <b>
  不理中共要求
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马来西亚.html" target="_blank">
   马来西亚
  </a>
  释放11名
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维族人.html" target="_blank">
   维族人
  </a>
 </b>
 <br/>
 <br/>
 马来西亚当局不理会中共的要求，本周释放了来自中国新疆的11名维吾尔人。这些维族人是去年从泰国监狱越狱，逃到马来西亚的。中共当局一直要求马来西亚把他们遣送回中国。
 <br/>
 <br/>
 路透社10月11号报导说，马来西亚的检察官倾听了维族人的呼吁，基于人道的理由释放了他们。这些人已经在本周二乘飞机抵达土耳其。
 <br/>
 <br/>
 报导说，中共近年来在新疆严酷迫害维族人，导致数百、甚至数千人通过东南亚转逃到土耳其避难。
 <br/>
 <br/>
