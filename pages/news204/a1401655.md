### 贸易战暂停90天 白宫称川习会〝高度成功〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】12月1日举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 在掌声中结束。当晚，白宫发表声明称，这是一次〝高度成功的会谈〞，并宣布明年暂停对2000亿美元中国进口商品增加关税。双方将就中共经济〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性改变.html" target="_blank">
  结构性改变
 </a>
 〞的问题继续谈判，未来90天内将完成谈判。如果谈判失败，美国将把关税从10%提高到25%。
 <br/>
 <br/>
 12月1日晚，美国总统川普（特朗普）与中国国家主席习近平在布宜诺斯艾利斯共进工作晚宴，就贸易问题进行协商。双方会谈气氛良好，晚宴上传出掌声，还临时插进了合照环节，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 以〝积极的调子结束〞。
 <br/>
 <br/>
 当晚白宫发表声明称，这是一次〝高度成功的会谈〞，美国将暂停对中国商品增加关税。最重要的是，双方已同意就中共经济〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性改变.html" target="_blank">
  结构性改变
 </a>
 〞的问题继续谈判。
 <br/>
 <br/>
 《美国之音》发表的白宫声明全文翻译如下：
 <br/>
 <br/>
 美国总统唐纳德・川普和中国国家习近平，完成了两人及双方最高级代表在阿根廷布宜诺斯艾利斯举行的双方所说的〝高度成功的会谈〞。
 <br/>
 <br/>
 非常重要的是，习主席做出了不起的人道主义姿态，同意把芬太尼列为〝管控物质〞。这意味着向美国出售芬太尼的人将依法受到中国最重的处罚。
 <br/>
 <br/>
 在贸易问题上，川普总统同意在2019年1月1日，他将把对价值2千亿美元的产品的关税税率保持在10%，而不是在那一天提高到25%。中国将同意从美国购买仍未商定但数量非常大的农业、能源、工业和其它产品，以减少我们两国之间的贸易不平衡。中国已同意立即从我们的农民那里购买农产品。
 <br/>
 <br/>
 川普总统和习近平主席同意，立即开始有关结构性改变的谈判。这些结构性改变涉及强制技术转让、知识产权保护、非关税壁垒、网路入侵和网路盗窃、服务业和农业。双方同意，他们会努力在未来90天内完成谈判。如果双方在这个期限内未能达成协议，10%的关税将会被提高至25%。
 <br/>
 <br/>
 双方还一致认为，朝鲜问题取得了巨大进展；川普总统将和习主席一起，并与金正恩委员长一道，努力实现一个无核的朝鲜半岛。川普总统表达了他对金委员长的友谊和尊敬。
 <br/>
 <br/>
 习主席也表示，如果先前未获批准的高通-恩智浦协议再次呈交于他，他对批准协议持开放态度。
 <br/>
 <br/>
 川普总统表明：〝这是一场精彩的、富有成果的会谈，对美国和中国具有无限可能性。与习主席合作，是我的荣幸。〞
