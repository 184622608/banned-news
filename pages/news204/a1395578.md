### 女律师孙世华受辱案 三名目击证人被治罪
------------------------

<div class="wysiwyg">
 【新唐人2018年10月16日讯】首先来关注中国的消息。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州女律师.html" target="_blank">
  广州女律师
 </a>
 孙世华遭警察侮辱一案继续发酵，目前三名现场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_目击证人.html" target="_blank">
  目击证人
 </a>
 都被抓，其中两人遭拘留，一人被监视居住。
 <br/>
 <br/>
 孙世华律师被广州华林派出所警察羞辱案的三名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_目击证人.html" target="_blank">
  目击证人
 </a>
 梁颂基、张五洲、李小贞，10月14号相继被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州警方.html" target="_blank">
  广州警方
 </a>
 传唤，15号，梁颂基、张五洲遭拘留，李小贞被监视居住。
 <br/>
 <br/>
 张五洲的妹妹张唯楚16号告诉新唐人，当地公安通知她，她姐姐已经被拘留。
 <br/>
 <br/>
 张五洲的妹妹张唯楚：〝她那个传唤证上面写的是扰乱单位秩序，是华林派出所那个事情，她是目击证人，她就是被拘留了，但是是行政还是刑事的话呢，他们现在还没有给我家属通知单。〞
 <br/>
 <br/>
 网上有消息说，与张五洲一起在华林派出所目睹孙世华被侮辱的维权人士梁颂基，是被警察跨省从北京抓回广州刑事拘留的。
 <br/>
 <br/>
 湖南维权人士周先生：〝梁颂基是涉嫌寻釁滋事罪，刑拘通知书是有的啊，因为他们两个是一起的嘛，一起当天在派出所出现的。〞
 <br/>
 <br/>
 另一名目击证人，也是就是孙世华办案的委托人李小贞，目前被监视居住。
 <br/>
 <br/>
 张唯楚：〝那个李小贞的话，她原来本来也准备抓她的，因为她的小孩才一岁9个月，没有家属能带，所以又把她放出来了。〞
 <br/>
 <br/>
 孙世华9月20号到广州荔湾区华林派出所办案，遭到该所警察殴打、脱光衣服羞辱。警方事后否认事实，但不公布现场录像。
 <br/>
 <br/>
 三名目击证人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被治罪.html" target="_blank">
  被治罪
 </a>
 后，网上有人质疑，警方可能是要封口或者灭口。
 <br/>
 <br/>
 新唐人记者熊斌、施宜君采访报导
</div>
