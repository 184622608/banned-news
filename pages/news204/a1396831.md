  维权

 <br/>
 <br/>
 2.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_顺德居民游行.html" target="_blank">

 </a>
 抗议建垃圾处理中心
 <br/>
 <br/>
 广东佛山市顺德区杏坛镇数以千计的居民，10月23号、24号连续两天在当地主干道游行，抗议政府建高危物品垃圾处理中心。
 <br/>
 <br/>
 3. 湖北潜江
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>

 <br/>
 <br/>
 湖北潜江的5名在京访民，最近集体到中共的国家信访局、中纪委、最高法、中组部、公安部、工总行等多个信访部门上访，但因访民人数太多，登记困难，维权无果。
 <br/>
 <br/>
 4. 安徽界首市村民
