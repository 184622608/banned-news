### 福建泉州7吨毒〝碳九〞泄漏 现场视频曝光（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建泉州.html" target="_blank">
  福建泉州
 </a>
 市一艘石油化工公司的船隻将近7吨化工原料〝碳九〞泄漏，导致当地海域被严重污染，空前中弥漫着刺鼻气味。泄漏事件令当地水产养殖户损失惨重，有的损失高达数百万，几十年辛苦瞬时血本无归。更有多人因捞污水，导致身体不适入院，有的甚至出现生命危险。
 <br/>
 <br/>
 据陆媒报导，11月4日凌晨，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建泉州.html" target="_blank">
  福建泉州
 </a>
 泉港区出现呛鼻气味，海面飘来大片黄褐色油污状污染物。经查，是福建东港石油化工公司的船隻，在泉州市泉港区码头进行装卸操作时，因软管垫片老化、破损，导致近7吨化工原料〝碳九〞泄漏，爆发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_严重生态灾难.html" target="_blank">
  严重生态灾难
 </a>
 。
 <br/>
 <br/>
 据了解，〝碳九〞是石油经过催化重整以及裂解后的副产品，主要包含异丙苯、正丙苯、乙基甲苯、三甲苯等。〝碳九〞无法分解，对水、土壤和大气都可造成污染，且容易吸入人体，比毒药还毒。
 <br/>
 <br/>
 事件发生后，当地政府部门互相推诿。有网友贴文说，4日凌晨3点有人就开始报警，警察推给海事局，海事局又推给环保局，环保局再踢给农林水局，直到早上9点，才有政府人员来查看，证实是碳九泄漏。
 <br/>
 <br/>
 泄漏的污染物令养殖的水产品大量死亡，更腐蚀鱼排的泡沫板，导致鱼排沉入海中，令当地水产养殖户损失惨重。因官方未及时通报是有毒化学品泄漏，又导致当地多人被污水导致身体不适，甚至有的出现生命危险。
 <br/>
 <br/>
 据肖厝村渔民肖先生说，事发当日，肖厝村老一辈人几乎全都扑在渔排上想拯救渔排，没有做任何防护措施，有些人直接用手去捞〝油污〞。他65岁的父亲用渔排养鱼已23年，事发当日整天都在渔排上清理那些油污状物质。
 <br/>
 <br/>
 当晚，他父亲就开始出现胸闷、头晕、喉咙痒、呕吐的症状，还出现抽搐，他连忙将父亲送到泉港区医院。结果在医院4楼，遇到了同村7至8名村民，都因身体不适到医院求诊。
 <br/>
 <br/>
 肖厝村另一位渔民肖先生，更因不慎掉入被污染的渔排中，出现生命危险，目前正在泉港区医院重症监护室抢救。
 <br/>
 <br/>
 肖先生的妻子说，她丈夫等家人4日在渔排上，用棍子捞类似油污的污染物，但〝渔排上味道太臭了〞，她丈夫突然呼吸困难，掉进渔排的污水中。家人将肖先生捞起来后，他出现头晕呕吐等症状，被立即送到医院。
 <br/>
 <br/>
 此次〝碳九〞泄漏事件导致当地渔民损失严重，住在肖厝码头附近的肖女士说她家渔排并不多，只是养了一些经济鱼，但依然受损不小。她大哥则表示，他有上百格子渔排，损失好几百万元人民币。
 <br/>
 <br/>
 71岁的肖如春和67岁的肖珠英夫妇从事渔排养殖已经有26年，从最初的18个格子一直扩大到138个，这次事件令他们损失惨重。〝在这次事故中，渔排沉没了，鱼也跑了，就算剩下来的鱼也没有人敢买、敢吃，原本还盼着这些收入拿去翻修房子，现在落得血本无归。〞
