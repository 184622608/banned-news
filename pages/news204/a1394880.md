### 【禁闻】双十国庆 蔡英文吁北京别做冲突来源
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_双十国庆.html" target="_blank">
   双十国庆
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
   蔡英文
  </a>
  吁北京别做冲突来源
 </b>
 <br/>
 <br/>
 10月10号，在台北的总统府府前广场举行中华民国107年国庆庆典。
 <br/>
 <br/>
 总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英文.html" target="_blank">
  蔡英文
 </a>
 在演说中指出:  台湾目前面临的国际艰困局势，很大部分是来自〝北京当局的施压〞。
 <br/>
 她说：〝这段时间以来，中国单方面的文攻武吓和外交打压，不但伤害了两岸关系，更严重挑战了台海和平稳定的现状〞。
 <br/>
 <br/>
 蔡英文在演讲中呼吁北京当局在区域及全球扮演良性的角色，而不是冲突的来源。
 <br/>
 <br/>
 <b>
  中共政治局常委家族在港拥巨额物业
 </b>
 <br/>
 <br/>
 香港《苹果日报》10月10号披露，中共多名现任和前任政治局常委的家族成员，在香港拥有巨额
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_资产.html" target="_blank">
  资产
 </a>
 。其中，前副总理张高丽家族拥有资产最多，在香港元朗加州花园和加州豪园有20栋洋房，连同其他物业总资产，价值近8.6亿元。
 <br/>
 <br/>
 而中共前政协主席贾庆林家族则对港楼最情有独钟，其妻子、女婿及外孙女都多次在香港扫货。其中，出手最豪阔的是外孙女李紫丹，她在2015年以3.87亿元购入港岛一处豪宅自用，全部以现金结清。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国会.html" target="_blank">
   美国会
  </a>
  报告：中共加强对人民的控制
 </b>
 <br/>
 <br/>
 美国国会及行政当局中国委员会，10月10号公布2018年年度中国人权报告，对中国大陆的人权及香港的自主权等情况进行分析。
 <br/>
 <br/>
 这份长达405页的报告指出，中共在过去一年中，在新疆实施有系统的人权侵犯，对政府、社会和商业的控制加强，利用高新技术监控人民的行动加剧。
 <br/>
 <br/>
 报告还说，香港基于《基本法》〝一国两制〞所保障的自主权，受到来自北京的侵蚀，香港的自由继续受到北京打压。
 <br/>
 <br/>
 编辑/周玉林
