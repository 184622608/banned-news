### 港媒:孟宏伟裸官多年还想升 被职务安排诱回国
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】中共公安部副部长兼国际刑警组织主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 回国被抓，其妻出面发声。网传其妻持香港护照，在法国租住豪宅。港媒最新消息指，孟的妻儿已获外籍身份多年，孟身为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裸官.html" target="_blank">
  裸官
 </a>
 〞仍为仕途奔波，这次是被以职位安排为由诱骗回国。
 <br/>
 <br/>
 香港《明报》10月11日引述北京消息透露，得益于分管出入境，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 多年前已帮妻子格蕾丝(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Grace Meng.html" target="_blank">
  Grace Meng
 </a>
 ，音译)取得外籍身份，孟妻与孩子已在境外生活了很长时间。
 <br/>
 <br/>
 因此，孟宏伟在2016年被派驻法国里昂，出任国际刑警组织主席，其家属并没有以驻外人员家属的身份同行。
 <br/>
 <br/>
 北京当局近年来公开批评党内〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裸官.html" target="_blank">
  裸官
 </a>
 〞，部分〝裸官〞被降级或免职。中纪委2014年更明文规定禁止提拔〝裸官〞。按照官方规定，所谓〝裸官〞，是指〝配偶已移居国（境）外，或者没有配偶但子女均已移居国（境）外〞。
 <br/>
 <br/>
 不过《明报》消息指，孟宏伟已经当〝裸官〞多年，也已届退休年龄，但他在中共十九大后还在为仕途奔波。这次他回京被抓，也是被北京以职位安排及参加会议为由诱骗回国。
 <br/>
 <br/>
 据公开资料，孟宏伟现年65岁，已到正部级退休年龄。如果本届任期期满前能够晋升公安部长等副国级官员，则可继续留任。即便是到龄退居二线，仍有机会在全国人大或政协担任副委员长和副主席，获得副国级待遇。不过，随着孟宏伟落马，这都已成泡影。
 <br/>
 <br/>
 港媒消息还称，孟宏伟涉嫌受贿长达十余年，其妻等家人都是行贿目标。
 <br/>
 <br/>
 10月4日，孟妻格蕾丝向法国警方报警，指孟宏伟回中国后失踪。7日，格蕾丝接受CNN采访，曝光孟宏伟9月25日失联前用手机给她发回的尖刀图案。同日，中纪委宣布孟宏伟落马。
 <br/>
 <br/>
 推特消息称，格蕾丝持香港护照，无业，两年来在法国里昂最豪华别致的街区之一租住一幢豪宅，月租5万多人民币，但其奢侈生活并非由国际刑警组织或中共公安部出资。
 <br/>
 <br/>
 许多网友质疑，孟宏伟妻子是其贪腐的共犯。
 <br/>
 <br/>
 孟宏伟被指周永康余党。外界认为，孟宏伟应对中共政法系统镇压法轮功、维权人士甚至活摘器官等罪行负有相应责任。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
