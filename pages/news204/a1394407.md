### 美国防部跟进彭斯 报告批中共军事扩张威胁国防
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】继美国副总统彭斯（Mike Pence）发表演讲，痛批中共的各项国内和国际政策后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国防部.html" target="_blank">
  美国国防部
 </a>
 也发布评估报告跟进，点名批评中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事扩张.html" target="_blank">
  军事扩张
 </a>
 ，对美国的经济和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家安全.html" target="_blank">
  国家安全
 </a>
 构成另一层
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威胁.html" target="_blank">
  威胁
 </a>
 。
 <br/>
 <br/>
 10月7日，美媒报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国防部.html" target="_blank">
  美国国防部
 </a>
 5日公布一份报告，针对美国制造业与国防工业基础与供应链弹性评估，指出逾280项可能影响美军重要军品与零部件供应的漏洞，并提出了改进措施。
 <br/>
 <br/>
 报告列举了6个面向竞争国家产业的政策，其中直接点名中共，对中共的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事扩张.html" target="_blank">
  军事扩张
 </a>
 、经济侵略战略、软实力战略及军事研发等进行说明。报告指，中共被视为对美国国防安全具有战略性的关键材料与技术构成重大风险，且与日俱增。
 <br/>
 <br/>
 报告还指称，虽然多个国家推行的经济政策有损于美国制造业，但没有一个国家像中共那样成功锁定美国工业基础。中共是许多美军武器与飞弹所需关键原材料的主要供应来源。中共对其他国家，特别是非洲与拉丁美洲国家，以投资换取当地资源和市场准入，也增加了对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国家安全.html" target="_blank">
  国家安全
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威胁.html" target="_blank">
  威胁
 </a>
 。
 <br/>
 <br/>
 报告称，中共加入世界贸易组织（WTO）并进入美国市场之后，美中贸易逆差持续扩大，中共对经济竞争环境的非市场扭曲必须结束，否则美国将失去技术优势与军事优势，并面临工业能力风险。而中共正试图提升自我军事能力以达到美国的水准。
 <br/>
 <br/>
 7月份，英国广播公司（BBC)也曾报导说，中共军委于2月发布的内部文件显示，北京希望增加境外军事影响力，使军队有能力〝管控危机、遏制战争、打赢战争〞，军力超越美国。
 <br/>
 <br/>
 文件还预计，如果军改继续，中共同美国的关系将会更加紧张，同包括日本在内的邻国的关系在东海和南海也会更加紧张。并强调〝强大的军事实力对一个大国转变为强国至关重要〞，还宣称美国、俄罗斯和日本的发展都证明这点。
 <br/>
 <br/>
 事实上，中共在军事上〝超越美国〞的目标，同被中共最近淡化宣传的经济领域的〝中国制造2025〞异曲同工。一方面叫嚷大力赶超，一方面通过胁迫、行窃等各种非法手段施行，引发国际社会的反弹。
 <br/>
 <br/>
 美国官员透露，中共政府黑客侵入了罗德岛海军水下作战中心承包商的计算机，窃取海量水下战作战（潜艇战）高度敏感数据，其中包括2020年前开发用于美国潜艇的超音速反舰导弹秘密计划。
 <br/>
 <br/>
 美国先进战机的机密一直是中共设法盗窃的对象，而中共的J-20、J-31都被指是在盗窃美国的技术上仿制出来的。
 <br/>
 <br/>
 除了非法获取技术，中共也不断增强对周边区域的军事布署，引发国际社会争端及担忧。
 <br/>
 <br/>
 9月底中美战舰在南沙群岛附近海域，几乎造成碰撞，美舰不得不采取紧急行动避免碰撞。近期，美国派遣B-52重型远程轰炸机，在三天内两度飞越南海。另一架B-52轰炸机与日本自卫队的战机，还在有争议的东海举行了联合演习。
 <br/>
 <br/>
 据CNN报导称，美军正在研讨11月在全球范围展开联合军演震慑中共。
 <br/>
 <br/>
 10月4日，美国副总统彭斯发表演讲，对中共进行了全方位的抨击，涉及内容包括贸易不公、南海军事化、侵犯宗教自由、干预美国政治、破坏美国自由等等。
 <br/>
 <br/>
 彭斯要求北京用实际行动〝重新尊重美国〞，并警告在美中关系达到〝公平、对等和尊重我们主权〞之前，美方绝不会让步。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：祝馨睿）
</div>
