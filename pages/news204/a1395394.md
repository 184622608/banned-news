### WTO被取代成定局？陆学者承认〝底线准备〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 持续升温，外界担忧两国的〝新冷战〞铁幕已经拉下。有大陆学者认为，随着中美关系急剧降温，世界
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易.html" target="_blank">
  贸易
 </a>
 组织（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 ）将会被取代，北京要有〝底线准备〞。
 <br/>
 <br/>
 〝香港经济日报〞引述中国人民大学国际关系学院教授时殷弘称，在川普政府将中国（中共）视为中长期头号对手的战略定位下，世界正被撕裂成两块，以美国为核心的发达国家形成集团，中共的主要经济活动很可能被迫集中在一部分发展中国家，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
  WTO
 </a>
 也一定会被替代。
 <br/>
 <br/>
 华盛顿在近期已经多次责难WTO的机制。美国驻世贸大使谢伊说，中共治下的中国经济体制与WTO的规范不相容，并呼吁其他成员国在此问题上发声。谢伊还说，美欧日正在合作，希望就遏制中共的产业补贴、强迫技术转让等问题达成新规。
 <br/>
 <br/>
 同时，美国还在与各个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易.html" target="_blank">
  贸易
 </a>
 伙伴不断达成新协议。《美墨加协定》（USMCA）已经达成，美韩的自贸协定在9月底也已签订。时殷弘预测，下一步与日本及欧盟的协议也会很快完成，未来可能会形成集团经贸的新规则新格局。
 <br/>
 <br/>
 时殷弘认为，如果世界形成这样分野态势，中美之间可能出现如美国和苏联的冷战状态，这对中国将形成非常大伤害，〝若形成此局面，届时大量低回报甚至无回报资本将外流，对中国经济产生深远影响，一方面我们得不到最想要的东西--高技术，另一方面与第三世界国家的轇轕和矛盾也可能增加。〞
 <br/>
 <br/>
 时殷弘还表示，川普政府已经明确将中国（中共）作为中长期头号对手，〝已超越俄罗斯，未来中美下行趋势还会进一步加剧，甚至要做好底线思维的心理准备。〞
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：赵云）
