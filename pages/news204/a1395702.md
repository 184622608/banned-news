### 中南海〝国难当头〞 外媒聚焦四中全会四大议题
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】各种迹象显示中共十九届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 月内将召开。国际媒体分析，在美中贸易战令中共陷入空前困境，可谓〝国难当头〞之下，中共的四中全会或将聚焦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 、政治、经济和人事四大议题。
 <br/>
 <br/>
 10月16日，法广报导说，各种迹象及讯息显示，中共将于10月20日前后召开第十九届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 。在美中贸易战持续升温之际，经济议题可能是会议讨论及政策宣示核心。
 <br/>
 <br/>
 明报报导说，由于今年是中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 40周年，又恰逢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 处于胶着状态，故今次四中全会的主题将聚焦进一步的改革开放。
 <br/>
 <br/>
 中央社则认为，今次全会讨论的核心议题，可望集中在经济方面，包括如何因应美中贸易战、更大幅度但损失最少地开放国内市场，以及解决产业发展上与美国的差距等亟待解决的切身问题。
 <br/>
 <br/>
 此外，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 日前对贸易战提出的〝自力更生〞口号，也可能成为全会讨论话题。
 <br/>
 <br/>
 除此之外，由于中共陷入金融危机的阴霾至今未解，若再与美中贸易战挂鈎，恐将酿成巨变，危及中共政权稳定，今次全会也可望讨论金融危机的因应及化解之道。
 <br/>
 <br/>
 苹果日报引述报导称，今次四中全会将涉及中美贸易战、政治、经济和人事等议题。自去年十九大结束后，中共政治和经济〝左倾〞明显，但对于习近平的个人崇拜遭遇挫折，今次全会也将是对习的效忠和歌颂的大会，将习
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_定于一尊.html" target="_blank">
  定于一尊
 </a>
 。
 <br/>
 <br/>
 此外，由于中共反腐败，习执政前5年拿下140多万党政官员，中共官员普遍消极怠工，目前，整个中共官僚集团暮气沉沉，四中全会将对怠政官员进行严厉警告。
 <br/>
 <br/>
 消息人士称，中共十九大前，习近平完成了最高层的换届。十九大后，省部级的调整仍在继续。此次全会除了对已经〝出事〞省部级官员的处理公示外，中组部还将调整更多省部级官员。
 <br/>
 <br/>
 外交上美国在贸易战持续半年后不但不见收手，反而由副总统彭斯出面严词抨击中共，中美迈入新冷战时代。同时，中共因强制转让技术及盗窃知识产权等问题，还招致欧盟发达国家的围堵。
 <br/>
 <br/>
 分析认为，在经济停滞、外交孤立的空前困境之下，习近平在今次全会上，将就中美贸易战、民营企业市场定位及中共经济、外交、军事形势进行阐述，强调改革开放，鼓舞和稳定民心将是四中全会的重心。
 <br/>
 <br/>
 对于十九届四中全会日程，先前已传出在10月举行。由于日本首相安倍晋三抵达北京访问的时间是10月25日。外界推测，四中全会最迟将在23、24日结束，因此20日前后或将召开全会。
 <br/>
 <br/>
 台湾中央社称，习近平年初推动修宪取消国家主席任期限制，一度在党内受阻，因此罕见地在1至2月接连召开二中和三中全会，因此，此次全会，日程上较往届的四中全会要提早许多。
 <br/>
 <br/>
 按中共惯例，历届中央委员会在任期内都会召开七次全体会议。其中第四次全会的主题，一般是根据当前阶段相对重要的问题而确定，会议召开时间一般都是在10月份。中共第十八届四中全会于2014年10月20日到10月23日召开。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
