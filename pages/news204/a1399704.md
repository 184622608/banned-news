### 刘强东保险界又遇阻 〝京东互保〞上线一天就下架
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】深陷
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵丑闻.html" target="_blank">
  性侵丑闻
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 在保险业界又受阻。京东金融的保险版块11月13日推出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东互保.html" target="_blank">
  京东互保
 </a>
 ．重大疾病相互保险计划〞，仅上线一天，就被紧急
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下架.html" target="_blank">
  下架
 </a>
 。
 <br/>
 <br/>
 在大陆网路巨头阿里巴巴推出互保业务之后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 统领的京东也急于在保险业界分一杯羹。
 <br/>
 <br/>
 所谓相互保险，就是具有同质风险保障需求的单位或个人，通过订立合同成为会员，并缴纳保费形成互助基金，由该基金对合同约定的事故发生所造成的损失承担赔偿责任，或者当被保险人死亡、伤残、疾病或者达到合同约定的年龄、期限等条件时承担给付保险金责任的保险活动。
 <br/>
 <br/>
 11月13日早上6:00，京东金融的保险版块上线了〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京东互保.html" target="_blank">
  京东互保
 </a>
 ．重大疾病相互保险计划〞。不过，刚推出内测仅1天，就被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下架.html" target="_blank">
  下架
 </a>
 。
 <br/>
 <br/>
 14日，在〝京东金融—保险〞的首页上，已经找不到〝京东互保〞的入口了。网页提示页面正在升级。
 <br/>
 <br/>
 据悉，〝京东互保〞遭下架主要是因为触动了传统保险公司的利益，受到各方投诉。目前，〝京东互保〞的合作方〝众惠财产相互保险社〞正在试图与监管部门沟通。
 <br/>
 <br/>
 15日，〝众惠相互〞回应称，将和京东配合，对〝京东互保〞进行用户体验升级后再择期推出。
 <br/>
 <br/>
 在此之前，10月14日，阿里系旗下的支付宝推出了〝相互保〞新产品。据统计，〝相互保〞推出一个月，参会会员已超1800万。
 <br/>
 <br/>
 据保险专业人士对比〝京东互保〞和〝相互宝〞后表示，与京东金融合作的众惠相互是财险公司，相比人寿保险公司，可靠性差了那么一点。
 <br/>
 <br/>
 另一家网络公司腾讯旗下的〝微保〞，目前没有推出类似的相互保产品。
 <br/>
 <br/>
 2016年6月，中共保监会首批放行三家相互保险机构，〝京东互保〞和〝相互保〞的合作方众惠相互和信美相互是其中的两家。
 <br/>
 <br/>
 但根据三家相互保险社2017年年报显示，3家相互保险社当年均未实现盈利，共亏损2.7亿元。
 <br/>
 <br/>
 目前看来，刘强东此次进军相互保业务并不顺利。自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵丑闻.html" target="_blank">
  性侵丑闻
 </a>
 曝光以来，刘强东这名曾经出入中南海的红顶商人，一落千丈，似乎再也触摸不到那层政商交汇的天花板。
 <br/>
 <br/>
 近期，刘强东不仅频频缺席一系列重大会议和公开场合，京东股价也遭受重创，市值已跌掉一半，刘强东自身资产也减少3成，缩水至62亿美元，跌出中国富豪榜前20名。
 <br/>
 <br/>
 刘强东性侵丑闻也频频遭到大陆商界大佬嘲讽，9月，马云在阿里巴巴投资人大会上，公开嘲讽刘强东性侵一事，称自己不会出那类事。
 <br/>
 <br/>
 10月，在天津举办的第十届智慧山观峰讲坛上，房地产大老任志强批评说，不管事情是真是假，刘强东影响了企业的社会形象。
 <br/>
 <br/>
 而刘强东在性侵丑闻被爆光不久，网易创始人丁磊就在网络评价说，刘强东是灌酒性侵惯犯，在一次多人参与的宴席上，刘强东把一名女子灌醉，当场在宴席包间的厕所内，对其施暴。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
