### 四川〝严书记〞被双开 曾惊动中南海
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】11月13日，中共官方发布消息称，四川广安市委原副书记严春风被双开。严春风落马源于此前的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_严书记女儿.html" target="_blank">
  严书记女儿
 </a>
 〞幼儿园打人事件。该事件引发舆论轰动，更惊动中南海，高层快速介入，严春风随即被〝秒杀〞。
 <br/>
 <br/>
 中共四川省纪委监委13日发布消息称，广安市委原副书记严春风对抗组织审查，违反组织纪律，不如实报告个人婚姻状况、房产、持有股票等有关事项；违规收受礼品、礼金，利用职务影响，为亲友经营活动谋取利益；利用职务上的便利为他人谋取利益，索取或收受他人所送财物，涉嫌受贿犯罪。省纪委决定给予严春风开除党籍处分；省监委给予其开除公职处分。
 <br/>
 <br/>
 严春风落马源于〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_严书记女儿.html" target="_blank">
  严书记女儿
 </a>
 〞事件的发酵。今年5月11日，四川一名严姓女童，因为在幼儿园经常动手打同学，带班陈老师在微信上和其他老师抱怨说：〝以后严某某放学那会单独坐，或者周围的人给她清空，她单独坐一边。真的是够了。〞
 <br/>
 <br/>
 不料消息错发到了家长群内，随后，家长群署名为：〝严某某妈妈〞的人，立即威胁说：〝陈老师，你马上在全班当着所有师生给严某某道歉，否则，我通知你们集团领导来给我解释，你对‘严书记女儿’说这话什么意思！〞〝我明天等你们学校的处理意见！〞
 <br/>
 <br/>
 之后，她又在群里称：〝学校处理结果已出：对陈老师开除处理！〞
 <br/>
 <br/>
 该事件被认为是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_我爸是李刚.html" target="_blank">
  我爸是李刚
 </a>
 〞的又一翻版，迅速引爆舆论热议。网民人肉搜索发现，〝严某某妈妈〞所提到的严书记即严春风，事发地点位是成都市金苹果国际幼儿园。
 <br/>
 <br/>
 身陷舆论风暴的严春风为自证〝清白〞，主动承认自己戴了5年〝绿帽子〞。随后，网路上流传一份严春风的自白信，他向省委组织部澄清，严姓女童母亲是自己的第二任妻子，但自己在藏区工作时，妻子因长期分居出轨，已经离婚5年。
 <br/>
 <br/>
 但据严书记邻居披露，严春风夫妻在社区一直同进出。还有幼儿园家长说，事发前，严春风还到班上给娃娃们讲了鸡蛋孵小鸡的原理。此外，严春风自称已离婚5年，但严姓女童却还有一个3岁的弟弟。
 <br/>
 <br/>
 更有网友披露，大陆官场〝2014年开始配偶申报财产，严书记2013年12月离婚，前妻带小孩住高档小区上高档幼儿园……〞坐拥豪宅的严书记〝政治嗅觉敏感〞。
 <br/>
 <br/>
 〝严书记女儿〞不仅掀起舆论热潮，也惊动中南海高层。四川省纪委监委网站5月14日发布消息称，官方近日已关注到网友反映〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_严书记事件.html" target="_blank">
  严书记事件
 </a>
 〞相关情况，已介入调查核实。
 <br/>
 <br/>
 4天之后，四川省纪委监委即发布消息：广安市委副书记严春风涉嫌严重违纪违法，正接受纪律审查和监察调查。〝严书记〞正式落马。
 <br/>
 <br/>
 有观察人士认为，严春风被秒杀的背后，可能与赵乐际在四川召开巡视会议有关。就在官方宣布介入的当天，中纪委在四川成都召开有关巡视工作的会议。中共政治局常委、中纪委书记、中央巡视工作领导小组组长赵乐际现身。
 <br/>
 <br/>
 对于严春风案件，官方强调，与过往打虎不同的是，严春风落马堪称民间反腐的标本案例，因为〝严书记事件〞的每一步进展都在舆论的参与下进行的。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
