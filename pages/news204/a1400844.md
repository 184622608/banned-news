### 歼20首开坠机先例？飞行员确认珠海航展期间死亡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】中共国产歼-X系列战斗机因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发动机.html" target="_blank">
  发动机
 </a>
 缺陷，频发坠机事故。日前中共〝最新型隐身战机〞歼-20的飞行员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_廖勇.html" target="_blank">
  廖勇
 </a>
 确认近日死亡，但官方未透露具体死因。有媒体透露廖勇死于珠海航展期间，外界质疑是否有歼-20因飞行表演出事。
 <br/>
 <br/>
 北京时间11月26日，有官方背景的〝新湖南〞网媒公布照片显示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_廖勇.html" target="_blank">
  廖勇
 </a>
 的骨灰已被运送至家乡。照片中，有女军人陪同廖勇家人迎接骨灰，当地村政府制作的黑色横幅上写〝迎接廖勇烈士回家〞。
 <br/>
 <br/>
 中共军媒今年10月曾刊文称，廖勇是中共空军通飞歼-20、 歼-16、歼-10C三款战机的最年轻飞行员之一。今年廖勇在空军基地和妻子参加集体婚礼时，〝伴郎〞就是一架歼-20战机。
 <br/>
 <br/>
 编辑部设在北京的多维网引述知情人士称，廖勇死亡的具体日期是今年11月9日。彼时正在举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_2018珠海航展.html" target="_blank">
  2018珠海航展
 </a>
 。11月9日当天，歼-20现身珠海进行了飞行表演。
 <br/>
 <br/>
 消息说，中共军方没有公布廖勇死亡的具体原因，因此有分析认为，廖勇可能死于歼-20事故。报导还称，如果属实，那会是中共隐身战机首开〝坠毁先例〞。
 <br/>
 <br/>
 但报导也说，廖勇也可能死于歼-16、歼-10C战机坠毁，或者其它原因。
 <br/>
 <br/>
 根据此前官方报导，11月6日至11日举行2018珠海航展期间，歼-20在11月6日、9日、10日和11日，先后四次举行飞行表演。6日和9日，都是三架歼-20进行特技表演，10日减少为两架，11日四架歼-20现身，编队飞行秀机载导弹。
 <br/>
 <br/>
 〝心脏病〞一直是中共国产战机的顽疾，此前歼-X系列频频曝出坠机事件，但中共军方多秘而不宣，并隐瞒事故原因。
 <br/>
 <br/>
 据《南华早报》11月8日披露，中共专为歼-20开发的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发动机.html" target="_blank">
  发动机
 </a>
 尚未通过可靠性测试，6日表演的三架歼-20都是安装的俄罗斯AL31发动机。
 <br/>
 <br/>
 今年7月9日，中共《国防报》曾披露，歼-20在研制过程中，机载武器系统首次靶试失败。
 <br/>
 <br/>
 2015年7月28日，中共《航空报》也曾披露，歼-20原型机在试飞时，曾遭遇非常惊险的情况，所幸飞机未坠毁或报废。
 <br/>
 <br/>
 歼-20由中航工业集团研制，中共称其是继美国F-22之后第二型进入现役的重型双发隐身战斗机。但军事专家分析认为，歼-20存在致命缺陷，无法和美国F-22相提并论。
 <br/>
 <br/>
 今年2月16日，《国家利益》（National Interest）引述香港《南华早报》的调查报告指出，歼-20最初装备W-15发动机，但在2015年测试时，W-15发生爆炸，问题至今未得到解决。此后，歼-20换装WS-10B发动机，但WS-10B无法让歼-20达到超音速。
 <br/>
 <br/>
 2016年11月2日，俄罗斯军事评论网站《红星》称，歼-20只是一头用美俄技术拼凑的〝大母牛〞，无法实现真正隐身的功能。其中，歼-20气动布局中的一对鸭翼，是采用早前苏联的技术，但该设计会使雷达反射截面大大增加，目前国际上先进的第五代隐形战机没有任何一家使用。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
