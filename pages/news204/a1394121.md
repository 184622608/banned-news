### 【禁闻】10月4日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_异议人士.html" target="_blank">
  异议人士
 </a>
 逃亡巴林
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遭关押.html" target="_blank">
  遭关押
 </a>
 <br/>
 <br/>
 北京异议人士张文和，因参与维权行动，多次被中共当局关精神病医院迫害，他最近前往巴林王国避难，9月26号，他的妻子王素娥接到来自巴林的电话称，张文和在24号遭到巴林警方带走，但因言语不通，无法交流。王素娥担心丈夫安危，10月4号向外界求救。
 <br/>
 <br/>
 3.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗受害家长.html" target="_blank">
  疫苗受害家长
 </a>
 王府井募捐 遭关押
 <br/>
 <br/>
 10月3号，大陆一批疫苗受害家长在北京王府井商业街募捐，筹集孩子的医疗费，但遭警察强制带到派出所，之后转送久敬庄黑监狱关押。
 <br/>
 <br/>
 3. 上海绿地集团
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拖欠工资.html" target="_blank">
  拖欠工资
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工人讨薪.html" target="_blank">
  工人讨薪
 </a>
 <br/>
 <br/>
 在山东省济南市施工的上海绿地集团拖欠工资，10月3号，工人集体讨薪。
 <br/>
 <br/>
 4. 广西柳州20多名工人拉横幅讨薪
 <br/>
 <br/>
 中微博大广西分公司拖欠工资，9月30号，20多名工人在柳州市万达华城门前拉横幅讨薪。工人声称他们承建了万达的部分工程项目，但公司欠他们数万元工钱未付。
