### 北京对贸易谈判无信心？易纲：做好最坏打算
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 迄今仍无缓解迹象。近日有消息称，中美双方正在策划
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晤，以解决贸易争端。对此，有专家建议，北京的谈判策略可以〝更大胆一些〞。但中共央行行长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_易纲.html" target="_blank">
  易纲
 </a>
 回应称，已为最坏的情况做好准备。暗示北京对谈判缺乏信心。
 <br/>
 <br/>
 10月14日，中共央行官网公布消息，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_易纲.html" target="_blank">
  易纲
 </a>
 在G30国际银行业研讨会上，对G30成员、墨西哥前总统查迪约（Ernesto Zedillo）的谈判建议时，作出上述表示。易纲所说的最坏情况，即美国再对2670亿美元中国商品征税，贸易战全面开打。
 <br/>
 <br/>
 目前，中美双方寄希望于下个月在阿根廷举行的G20元首会议，川普（特朗普）总统和北京领导人习近平举行会面，并解决贸易问题。《华尔街日报》12日报导称，中美双方都有专门团队在策划
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 面。
 <br/>
 <br/>
 11日，白宫首席经济顾问库德洛(Larry Kudlow)证实，对于安排川普和习近平会晤，〝有一些动作正在进行〞。不过，他补充说，会谈本身或领导人将要讨论的话题并没有确定。
 <br/>
 <br/>
 美国财长姆钦12日对CNBC表示，如果有正确的方向，川普可能计划与习近平会面，但北京必须先提出具体的〝让步清单〞。重新启动贸易谈判，将要求中方承诺对其经济进行结构性改革。
 <br/>
 <br/>
 有分析认为，彻底改变经贸体制，会直接摧毁中共政治权力的基础。虽然川普政府只是要求中美贸易公平，但已触及中共体制安全。
 <br/>
 <br/>
 此前，美媒已经透露中方对美方140条贸易要求的态度：三分之一可以立即实施，三分之一可以讨论，而另外关键的三分之一则根本不予考虑。
 <br/>
 <br/>
 近日库德洛又重申了美方的核心要求：零关税、零非关税壁垒、零补贴，停止盗窃知识产权、停止强迫转让技术，允许美国人在华拥有自己的公司。而这些都是在北京不予考虑的范畴之中。
 <br/>
 <br/>
 而易纲日前的表态，也从侧面显示北京对于和美国的贸易谈判并无太大信心。
 <br/>
 <br/>
 此外，货币问题正成为川普政府关注的问题。美国财长姆钦在出席G30国际银行业研讨会期间，与易纲进行了会面，双方广泛讨论了货币问题。姆钦对人民币的疲软表示担忧，他说，作为贸易谈判的一部分，货币必须成为讨论议题之一。
 <br/>
 <br/>
 自贸易战开打后，人民币跌跌不休。10月7日，央行宣布自15日起降准，9日，人民币兑美元汇率再度暴跌，失守6.9关卡，创2017年5月11日以来新低。
 <br/>
 <br/>
 美国财政部目前正在考虑将中共正式列入〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汇率操纵国.html" target="_blank">
  汇率操纵国
 </a>
 〞，预计在本月18日公布的，美国主要贸易伙伴外汇政策报告中，将做出定论。
 <br/>
 <br/>
 有分析认为，一旦中共被列入〝汇率操纵国〞，贸易战演变为金融战，中共将面临更为严厉的制裁，中共对内对外经济将陷入困境。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
