### 山东维权老兵被清场 送校舍监视
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】大陆老兵日前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东青岛.html" target="_blank">
  山东青岛
 </a>
 集体维权，遭警方强制清场，多人被带走。部分老兵被分批送进学校校舍，受当局监视和看管。
 <br/>
 <br/>
 10月6日至8日，大陆数千名退役老兵，集结至山东举行抗议，要求当局为之前平度市的一起老兵上访被打事件道歉，并改善老兵待遇。示威者一度与警方发生冲突。
 <br/>
 <br/>
 据美国之音报导，现场维权老兵发消息说，当局出动大批警力，还从青岛、潍坊、烟台等周边城市调遣数千公安特警，动用催泪弹清场；老兵们用喷雾灭火器还击，有人被打伤，头破血流。
 <br/>
 <br/>
 视频显示有武警参与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_镇压.html" target="_blank">
  镇压
 </a>
 ，老兵则手持木棍抵抗，突围后有围观民众拍手叫好。维权老兵被围困期间，当地民众赠送食品饮水等补给。在抗议现场，有老兵向当局派出的维稳人员喊话：我们的今天就是你们的明天。
 <br/>
 <br/>
 一位到平度声援的老兵表示，抗议老兵被强力驱散，分散监控在一些学校的空房内，等待遣返或由原籍地官员接回，伤者送入医院。
 <br/>
 <br/>
 该老兵介绍，平度一些地方手机信号被屏蔽，被关押的老兵无法与外界联系、发出消息。
 <br/>

 <br/>
 他说，他和一些人得以逃出警察的包围，而外地赶去增援的老兵被围堵在高速路口。
 <br/>
 <br/>
 青岛伤残军人李青向英国BBC中文介绍说，抗议活动始于10月6日，老兵们占据一个主要的广场进行示威。目前，大多数抗议老兵被驱散后已经回家或撤离至周边地区，还有多名老兵被捕或受伤送医。
 <br/>
 <br/>
 当被问及抗议原因时，他表示，事件直接的导火索是平度当地的多名老兵，在前往北京上访的过程中，遭遇当地公安的截访，部分人员〝被打受伤〞；因此他们前来〝声援〞。另一名来自辽宁阚春雷称：〝我们都是自发前来声援的，有东北三省的、河南、浙江、福建的。〞
 <br/>
 <br/>
 今年6月，同样是因老兵被打，百余名各地老兵集结江苏镇江抗议。中共官方媒体未进行报导，但相关视频在微博、微信等中国社交媒体上流传。
 <br/>
 <br/>
 一名微博网友写道：〝听党的话又‘光荣转业’，结果(共产)党卸磨杀驴。〞
 <br/>
 <br/>
