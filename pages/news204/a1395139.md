### 双方底线南辕北辙 川习再会难解贸战死结
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】中美正走向全面对抗，双方首脑在11月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上将难免〝遭遇〞。虽然美媒报导称双方正在策划会晤，商讨贸易问题，但外界普遍认为，双方都各守底线不让步，会谈难有突破，甚至是否有正式会晤都是未知数。
 <br/>
 <br/>
 11日，白宫首席经济顾问库德洛(Larry Kudlow)证实，对于安排川普总统和北京领导人习近平下个月在阿根廷G20会议期间的会面，〝有一些动作正在进行〞。
 <br/>
 <br/>
 《华尔街日报》12日报导称，中美双方都有专门团队在策划
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晤。美国方面力促会面的是库德洛和财政部长梅努钦（Steven Mnuchin），两人希望尽快结束贸易战。
 <br/>
 <br/>
 不过，《华尔街日报》报道相当审慎，只说白宫将〝继续推进〞此次会晤的计划，并没有肯定此次川习会最终能成功实现。
 <br/>
 <br/>
 多家外媒评论，中美双方都无意做出实质让步，此次会谈很难有突破。
 <br/>
 <br/>
 日前已有美媒消息指，川普政府已经开出川习会的条件，就是中方必须先提出让步清单。而中方近期对美方施压的回应，是让美方〝放弃幻想〞。
 <br/>
 <br/>
 事实上，此前美媒已经透露中方对美方140条贸易要求的态度：三分之一可以立即实施，三分之一可以讨论，而另外关键的三分之一则根本不予考虑。
 <br/>
 <br/>
 近日库德洛重申了美方的核心要求：零关税、零非关税壁垒、零补贴，停止盗窃知识产权、停止强迫转让技术，允许美国人在华拥有自己的公司。
 <br/>
 <br/>
 外界分析，中方要实现这六大要求，除非彻底改变经贸体制，但这也会直接摧毁中共政治权力的基础。因此，虽然川普政府只是要求中美贸易公平，但已触及中共体制安全，贸易战已经演变为政治体制之争。这也是中美正在走向全面对抗的原因。
 <br/>
 <br/>
 上月，川普在联合国大会上痛批社会主义（共产主义）给全世界带来的灾难。本月初，副总统彭斯发表重要演讲，对中共政权进行全方位的谴责。外界普遍认为，新冷战时代正在来临。
 <br/>
 <br/>
 在这种情况下，中美内部的温和派虽然有意结束贸易战，但已很难发挥影响力。
 <br/>
 <br/>
 《华尔街日报》上述报导还引述知情人士指，白宫温和派的计划是安排川普与习近平11月会晤，取得一场小胜利，然后就宣布贸易战结束。但该知情人士对谈判持怀疑态度。
 <br/>
 <br/>
 这是《华尔街日报》近期第二次提及11月的川习会〝结束贸易战〞。
 <br/>
 <br/>
 早在8月中共商务部副部长王受文准备赴美谈判时，《华尔街日报》就报导双方正在制定〝谈判路线图〞，准备在11月的川习会上结束贸易战，王受文赴美是在打前站。
 <br/>
 <br/>
 不过，该报导很快遭到川普反驳。川普随后对路透表示，他对王受文来访并没抱多大希望，并称自己并没有什么〝贸易战时间表〞，但是有一个长期战略。
 <br/>
 <br/>
 随后，白宫宣布由副总统彭斯代替川普，出席11月的亚太经济合作会议（APEC）。外界认为这是川普有意避免和习近平碰面。
 <br/>
 <br/>
 不过，11月底的G20首脑峰会上，二人的尴尬碰面将难以避免。不过，是否会有正式会晤，目前还是未知数。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
