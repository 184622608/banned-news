### 【禁闻】美海关：近90%进口仿冒品来自中国
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】
 <b>
  捷克参院
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_听证会.html" target="_blank">
   听证会
  </a>
  吁中共停止
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迫害法轮功.html" target="_blank">
   迫害法轮功
  </a>
 </b>
 <br/>
 <br/>
 11月19号，捷克国会参议院在瓦尔茨特荫宫馆（Valdštejnsky palác）举行公众
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_听证会.html" target="_blank">
  听证会
 </a>
 ，呼吁中共停止对法轮功学员种族灭绝性的迫害。
 <br/>
 <br/>
 据〝法轮大法明慧网〞报导，这次听证会由参议员哈露派克（Chaloupek）提议召开。在此之前，捷克参议院收到了当地法轮功学会转交的近四万人签名的请愿书。
 <br/>
 <br/>
 这次听证会是建议捷克共和国国会参议院起草决议案文本的初级程序。
 <br/>
 <br/>
 报导说，参加听证会的除了当地媒体、从事调查盗窃人体器官的工作人员外，还有捷克赫尔辛基委员会成员、外交部代表等，美国调查记者伊森．葛特曼（Ethan Gutmann）也参加了会议，并在会上呈交了他有关中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迫害法轮功.html" target="_blank">
  迫害法轮功
 </a>
 学员的调研报告，以及对中共强摘器官问题的调研结果。
 <br/>
 <br/>
 <b>
  美海关：近90%进口
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_仿冒品.html" target="_blank">
   仿冒品
  </a>
  来自中国
 </b>
 <br/>
 <br/>
 美国海关与边境保护局(CBP)和国土安全调查处(HSI)，在圣诞节购物季来临之前，严格检查从各国进口的圣诞商品，结果发现大批仿冒品,  而这些仿冒品有近90%来自中国和香港。
 <br/>
 <br/>
 美国海关主管卢梭(Frank Russo)日前在纽约肯尼迪机场举行的记者会上说，执法部门在近来的一系列行动中，共没收超过4400批次仿冒品，总值12亿美元。
 <br/>
 <br/>
 海关在记者会上展出的中国仿冒品，有三星、苹果等智能手机及配件，耐克服饰和球鞋，还有其他名牌手袋、香水等。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆编辑.html" target="_blank">
   大陆编辑
  </a>
  多写〝
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_总书记.html" target="_blank">
   总书记
  </a>
  〞三字 遭罚万元
 </b>
 <br/>
 <br/>
 香港《南华早报》11月22号报导说，大陆《陕西日报》的两名编辑，因在〝习近平思想〞提法外加了〝总书记〞三个字，遭罚款人民币1万元和5千元。
 <br/>
 <br/>
 《南华早报》得到的一份《陕西日报》内部文件显示，该报的科教文编辑刘汇和值班主任王戈华两人签发了一份陕西省美术家协会召开会员大会的新闻稿，文中〝习近平新时代中国特色社会主义思想〞被写成〝习近平总书记新时代中国特色社会主义思想〞，多了〝总书记〞三个字。
 <br/>
 <br/>
 《陕西日报》将之定性为〝重大政治类差错〞，并对编辑重罚。
 <br/>
 <br/>
 消息披露后，海外媒体纷纷加以报导，并嘲讽中共官媒小题大做、〝一字千金〞。
 <br/>
 <br/>
 编辑/周玉林
