### 见习近平前夕突然堕楼 澳门中联办主任死因惹猜疑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】21日，官方通报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 中联办主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 堕楼身亡。郑死前接连公开活动，未见异常。外界热传北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 23日将主持港珠澳大桥通车仪式，郑晓松也会出席。有评论指，或是郑本人怕见习近平，或是有人怕他见到习。
 <br/>
 <br/>
 周日（21日）一早，中共国务院港澳事务办公室发布消息称，中共驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳门.html" target="_blank">
  澳门
 </a>
 联络办公室主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑晓松.html" target="_blank">
  郑晓松
 </a>
 于周六晚在其澳门住所坠楼身亡，死因是〝抑郁症〞。
 <br/>
 <br/>
 在中共官方话语中，〝抑郁症〞已被视为官员自杀的代名词。不过郑晓松这次突发〝抑郁症〞的时间点敏感，外界猜疑或不止〝自杀〞这么简单。
 <br/>
 <br/>
 郑晓松突然坠亡，发生在港珠澳大桥开通前夕。多家媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将主持23日港珠澳大桥通车仪式。而郑晓松身为中共驻澳门机构的首脑，也在出席之列。
 <br/>
 <br/>
 加拿大作家兼时评人士盛雪在其推特表示：〝中共官员是越来越严重的高危人群，基本人生轨迹是：害人…被害…各种死法。郑晓松宁肯跳楼也不见习近平……看来见习近平将生不如死。如果他是被跳楼……说明他见了习近平有人会生不如死。〞
 <br/>
 <center>
  <br/>

   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-22/p9090481a692370509-ss.jpg"/>
   <br/>
   <font size="-1">
    盛雪评论，也可能是有人害怕郑晓松见到习近平。（推特截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 香港和澳门回归后，很快落入江派大佬
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾庆红.html" target="_blank">
  曾庆红
 </a>
 的掌控，曾的势力至今仍遍及港澳。此外，港澳也是中共权贵们对外洗钱的基地，众多江派高官家族在这两地拥有盘根错节的利益关系。
 <br/>
 <br/>
 翻查此前报导，郑晓松死前一天（19日）还与民联智库负责人会晤，祝贺该智库成立，并要求〝高举爱国爱澳旗帜〞，为澳门〝作出积极贡献〞。18日，郑晓松还在澳门美高梅出席澳门仁协之友联谊会的庆祝活动。
 <br/>
 <br/>
 据公开资料，郑晓松2013年7月从财政部空降福建省出任副省长，其后还担任过中共福建省委常委、秘书长等职务。福建是习近平曾经主政之地，外界认为郑受到习近平重用。
 <br/>
 <br/>
 2016年，郑晓松短暂调回北京，任中共中央对外联络部副部长，当时正值习近平整肃中联部之际。2017年9月，中共十九大前一个月，郑晓松被任命为澳门中联办主任，晋升为正部级。中共十九大上，他入列中央委员。
 <br/>
 <br/>
 郑晓松空降澳门，是接替调任香港中联办主任的王志民。而王志民被指习近平的福建旧部。王和郑的任命，被认为是习近平准备接管和整肃曾庆红地盘的动作。
 <br/>
 <br/>
 郑晓松
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离奇堕楼.html" target="_blank">
  离奇堕楼
 </a>
 ，距离调入澳门刚刚过去一年。
 <br/>
 <br/>
 不过，港媒日前披露，郑晓松在仕途初期曾参与香港事务，或暗示其另有其它背景。
 <br/>
 <br/>
 据报，1990年代，郑晓松曾担任时任中共外交部副部长姜恩柱的秘书。姜恩柱在香港回归后，曾出任新华社香港分社社长，并在2000年成为香港中联办首任主任。而郑晓松当时也跟随姜恩柱，出任新华社香港分社办公厅副巡视员、中共香港工委办公厅秘书、助理巡视员等职。
 <br/>
 <br/>
 1997年香港回归，是由时任中共党魁江泽民主持。2003年，曾庆红出任首任中共中央港澳工作协调小组组长，港澳成为曾庆红的地盘。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
