### 会晤前夕川普再谈习近平：习是朋友是好人，但…
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 正在布宜诺斯艾利斯举行。出席这次峰会的美国总统川普（特朗普）在会议间歇接受美国媒体的采访时，再次谈到了习近平。他表示，习是〝朋友〞，是〝好人〞，但他们之间也有一点〝纠纷〞，因为中共占美国便宜太久了。
 <br/>
 <br/>
 当地时间11月30日，川普在出席G-20峰会的间歇接受了《美国之音》(VOA)特约撰稿人的专访，他谈论了美中贸易冲突，以及他即将与习近平进行的双边会谈。
 <br/>
 <br/>
 据《美国之音》的报导，川普在这次的专访中谈到了美国所面临的危险，他说：〝我们有很多威胁，但美国被这么多的国家坑了。我说‘中国’的时候，并不是在告诉你什么秘密，中国害了我们这么多年、25年，我这话已经说了很久了。〞
 <br/>
 <br/>
 谈及他即将于周六与习近平举行的双边会谈时，川普表示，自己这次会与习近平谈到〝很多问题〞，而大家最想听的〝大话题〞 是贸易。
 <br/>
 <br/>
 然后川普评价说：〝我认为他是…他是我的朋友，是好人。但是你也知道，我们有那么一点纠纷。还是那句话，多少年来我们国家一直在被占便宜，这种情况不会继续下去了。〞
 <br/>
 <br/>
 针对川普的上述谈话，有观察人士分析称，川普的这一番话表明，他不会在贸易问题上让步，也不会轻易停止征收关税的行动。但至少他对习近平本人还是存有好感，也释放出了某种善意。
 <br/>
 <br/>
 按中美两国政府事先沟通定下的计划，川普与习近平将于12月1日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 的相关议程结束后，留在布宜诺斯艾利斯举行一次双边的〝工作晚宴〞，并藉此机会针对贸易等双方共同关心的问题进行交涉与磋商。外界都高度关注这次的双边会谈，能否成为美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的新拐点。
 <br/>
 <br/>
 不过，从近日海外舆论界释放出的信号来看，外界大多对这次的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞不报太高的期望。不少学者专家都认为，美中两国磨擦已久，贸易冲突的原因也很复杂而且〝根深蒂固〞，因此对这次两国首脑的会谈成果不表乐观。
 <br/>
 <br/>
 不过，中共官方媒体却纷纷发文为这次的〝川习会〞 吹出〝和风〞，对两国贸易战有机会缓和，表达了比较乐观的态度。
 <br/>
 <br/>
 中国问题专家、学者胡星斗在接受自由亚洲电台的访问时引述消息透露说，〝中国方面已提出二百多项可以答应美方要求的清单，但美方或许觉得仍不够，会要求中国在更多领域进一步开放。
 <br/>
 <br/>
 胡星斗表示，中国其实有多方面可以去做，包括降低关税、取消补贴、扶持民营企业等措施，如果真的去做，都对中国本身有利。
 <br/>
 <br/>
 他进一步分析称：〝欧盟、日本肯定都会步美国的后尘，那中国最后是几乎成为孤家寡人，除了跟非洲等国家能够来往之外，你（北京当局）与先进国家几乎就不能来往了。〞
 <br/>
 <br/>
 他表示，中国接下来是要继续深化改革开放，还是逐步的关上国门，这关系到中国何去何从的问题。
 <br/>
 <br/>
 不过，也有学者对川习会面的结果并不乐观。
 <br/>
 <br/>
 北京大学国际关系学院教授梁云祥就对《自由亚洲电台》表示，中美磨擦已久，两国矛盾是意识形态上的，很难通过一次首脑会面就化解。
 <br/>
 <br/>
 梁云祥说： 〝因为我觉得两国的这种矛盾已经是带有战略性的竞争，不完全是贸易问题。所以即使妥协，就是短时间的可能会妥协局部的，但是从根本上来说，可能比较困难。〞
 <br/>
 <br/>
 梁云祥表示，中美两国的根本体制不一样，双方的诉求差距实在很大，因此总体来说不乐观。至于中共官媒最近这两天开始吹点暖风，表达一下乐观的姿态，是因为中方一直想要缓解与美国之间的贸易摩擦，〝否则的话对中国的经济影响实在是太大了〞。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：明轩）
