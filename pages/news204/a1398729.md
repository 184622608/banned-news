### 川习会前开〝门缝〞 美运通卡有望拔下中国市场头筹
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】中国人民银行日前对外宣布，美国运通（American Express）的在华合资公司〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_连通公司.html" target="_blank">
  连通公司
 </a>
 〞在中国大陆筹建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_银行卡清算机构.html" target="_blank">
  银行卡清算机构
 </a>
 的筹备申请已获通过。舆论认为，这应该是北京当局为了给即将于本月底登场的川习会营造比较好的气氛，而有意释出的一小点〝善意〞。
 <br/>
 <br/>
 中国人民银行通报称，该行日前会同中银保监会审查通过了〝连通(杭州)技术服务有限公司〞提交的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_银行卡清算机构.html" target="_blank">
  银行卡清算机构
 </a>
 筹备申请。这意味着运通有望成为首家进入中国市场提供信用卡网络服务的美国公司。
 <br/>
 <br/>
 人行的通报强调称，这次批准
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_连通公司.html" target="_blank">
  连通公司
 </a>
 银行卡清算机构筹备申请，是响应北京当局有关〝稳步扩大金融业双向开放政策〞的要求而做出的决定，是中国开放银行卡市场，培育〝公平竞争环境〞的一个步骤，人行接下来将继续推进外资银行卡清算机构的准入工作。
 <br/>
 <br/>
 〝连通公司〞是美国运通在中国境内发起设立的合资公司，这次作为市场主体申请筹备银行卡清算机构、运营美国运通品牌，被外界视为美国企业尝试打开中国金融市场的一个代表性案例。
 <br/>
 <br/>
 按照监管机构规定，〝连通公司〞将在一年内完成筹备工作，然后再向人民银行申请正式开业。
 <br/>
 <br/>
 事实上，早在今年4月份中共监管层已口头受理运通合资公司的申请，但并未及时给予审查批覆。之后中美贸易战正式开打并步步升级，运通的合资公司筹备申请的审批进度便一直处于停滞状态。
 <br/>
 <br/>
 有港媒引述观察人士分析称，现在中方批准了运通合资公司的银行卡清算机构筹备申请，是中方有意向美方释放〝善意〞，展示北京当局愿意开放中国市场的姿态，但实际上能否达到预期目标，仍有待观察。
 <br/>
 <br/>
 适逢川习会本月底即将登场，中方以同意美国运通的合资公司进入中国金融市场的方式表现其开放金融业的姿态，也被认为是有意为习近平与川普的这次会谈营造比较良好的气氛。
 <br/>
 <br/>
 不过，外界注意到，美国运通今次是以与中方企业在华组建合资公司方式来开拓中国市场的，而合资公司这种形式，过去经常被中共政府用作套取美国企业先进技术的工具，因此美国政府对此仍抱有某种警惕。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
