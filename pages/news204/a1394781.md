### 与北京谈判〝还早〞 川普警告再对2670亿商品课税
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）日前在白宫表示，北京现在很想就贸易问题进行谈判，但目前〝为时过早〞，因为北京还没有准备好。川普再度警告，若中共继续对美国的关税制裁采取报复行动，他将再对2670亿美元的中国商品加征关税，届时
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 会全面开打。
 <br/>
 <br/>
 当地时间10月9日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在白宫椭圆形办公室发表讲话，他表示北京当局〝真的很想〞与美国进行贸易谈判。
 <br/>
 <br/>
 他说，北京现在已经很想谈，〝真的很想谈〞，但现在谈判〝为时过早〞，因为北京还没有准备好，两国间近期有几次会谈被取消。
 <br/>
 <br/>
 川普说，中共剥削美国的时间太长了，所以没办法这么快进行谈判。〝他们多年来每年都从我们国家拿走5000亿美元。我们不能让这种情况发生。〞
 <br/>
 <br/>
 川普还强调，〝在政治上，如果人们太快被强迫，你就不会为我们的劳工、我们的国家做出正确的决定！〞川普政府近日与加拿大和墨西哥达成新的贸易协定，是对川普此言有利的佐证。
 <br/>
 <br/>
 川普在讲话中再次警告北京，若继续对美国采取报复行动，他将再对价值2670亿美元的中国商品加征进口关税。这意味着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 或将全面开打。
 <br/>
 <br/>
 一直以来，川普都是说到做到。贸易战从最初的500亿美元，到增加2000亿，川普都事先警告北京，不要采取报复手段，否则制裁会加码。但北京仍然实施报复，导致贸易战升级。
 <br/>
 <br/>
 贸易战使中美关系进入空前紧张僵持阶段，外界已发出〝冷战〞开启的讨论。美国国务卿蓬佩奥日前访华，蓬佩奥与中共外交部长王毅，在公开的外交场合，不同寻常直白的批评对方。
 <br/>
 <br/>
 蓬佩奥直言〝我们有根本分歧。〞透露中美双方没有达成任何一致性的协议。
 <br/>
 <br/>
 在此之前，美国副总统彭斯在白宫发表讲话，严厉指责中共军事侵略、商业盗窃、侵犯人权以及试图干涉即将举行的美国中期选举。对中共进行全方位的抨击。令外界震惊，也让中共惶恐不安。
 <br/>
 <br/>
 中共官媒将彭斯演说，称之为美国对华发表的檄文，在大陆网络严厉封杀。
 <br/>
 <br/>
 另据彭博社报导，10月8日，美国财政部一名官员透露，白宫密切关注人民币近期的贬值情况，正在向财长姆钦施压，要求将中共正式列入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货币操控国.html" target="_blank">
  货币操控国
 </a>
 。财政部下周将发表报告作出定案。
 <br/>
 <br/>
 专家指出，美国一旦将中共列为汇率操纵国，意味着贸易战将演变为货币战。美国一旦对中共实施金融制裁，中共对外经济将陷入困境。
 <br/>
 <br/>
 此消息一经传出，中共外交部第一时间回应，再次强调中方〝不会将贬值人民币作为贸易战工具〞，凸显北京当局对此非常忌惮。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：戴明)
</div>
