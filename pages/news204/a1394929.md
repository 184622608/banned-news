### 乘客在滴滴专车内喝到尿  还有温度(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】中国大陆滴滴专车再曝丑闻。近日，江苏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏州.html" target="_blank">
  苏州
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙先生.html" target="_blank">
  孙先生
 </a>
 反映，他在上海乘坐滴滴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_礼橙专车.html" target="_blank">
  礼橙专车
 </a>
 喝到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尿液.html" target="_blank">
  尿液
 </a>
 ，而且还有温度。孙先生向滴滴方面投诉，得到的回应是，司机因内急找不到厕所，使用矿泉水瓶方便，导致乘客误饮。
 <br/>
 <br/>
 据陆媒报导，10月9日上午，到上海出差的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙先生.html" target="_blank">
  孙先生
 </a>
 与同事叫了一辆滴滴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_礼橙专车.html" target="_blank">
  礼橙专车
 </a>
 。上车后孙先生感到有点口渴，正好专车有专门为乘客提供的饮用水，他看到副驾驶放着一满瓶的水，拿起来拧了下，发现瓶盖是松动的。
 <br/>
 <br/>
 他说，他向专车司机确认，司机说可能是上一个乘客拧了没动过。孙先生随后喝了这瓶水，一入口感觉不对，立马吐了，〝确认应该是尿。〞
 <br/>
 <br/>
 孙先生反覆问司机〝尿〞是不是他的，司机坚决否认，同时表示，不清楚这个尿是谁的。孙先生在视频中说，〝这瓶水还有温度〞。
 <br/>
 <br/>
 孙先生称，当自己准备打电话投诉，司机情绪开始激动，说自己会因此失业，并不停自责，打自己，向孙先生道歉，提议用钱私了。
 <br/>
 <br/>
 最终，车辆停靠路边，孙先生拨打了滴滴客服电话，并报警。
 <br/>
 <br/>
 10月10日，滴滴方面表示，经过核实，当事司机因内急找不到厕所，使用矿泉水瓶方便，并将瓶子放置在驾驶室中控台，导致乘客误饮。
 <br/>
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：李泉）
