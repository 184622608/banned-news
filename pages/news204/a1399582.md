### 江西一幼儿园煤气泄漏 5女幼师身亡（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】11月16日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江西省南昌市.html" target="_blank">
  江西省南昌市
 </a>
 某
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幼儿园.html" target="_blank">
  幼儿园
 </a>
 发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_煤气泄漏.html" target="_blank">
  煤气泄漏
 </a>
 事故，5名女教师疑似煤气中毒身亡。目前，事故原因正在调查中。
 <br/>
 <br/>
 据陆媒报导，事发地点是南昌市进贤县丰和名城幼稚园。当地警方接到报案称，在一出租房内有5人死亡。
 <br/>
 <br/>
 从网传视频可见，有多名女子倒在床上和地上，现场疑是女教师们的宿舍。
 <br/>
 <br/>
 当地一名医生表示，16日上午8点左右，"我们到达现场时，5名幼师已经没有了生命体征。"
 <br/>
 <br/>
 当地官方通报称，女教师遗体均无外伤，疑是因洗澡时屋内空气不流通，导致一氧化碳中毒。具体原因还需进一步调查。
 <br/>
