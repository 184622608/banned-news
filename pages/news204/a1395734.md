### 长春长生遭罚 赔偿方案引不满
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】10月16号，中共药监部门对吉林
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长春长生.html" target="_blank">
  长春长生
 </a>
 公司违法违规生产狂犬病疫苗做出行政处罚，吊销其药品生产许可证，并没收违法所得和罚款，共计91亿元人民币。此外，保险监督管理委员会网站发布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赔偿方案.html" target="_blank">
  赔偿方案
 </a>
 ，不过，民众对行政处罚和赔偿方案均表示不满。
 <br/>
 <br/>
 10月16号，国家药监局撤销了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长春长生.html" target="_blank">
  长春长生
 </a>
 公司狂犬病疫苗药品批准证明文件以及涉案产品生物制品批签发合格证，吉林省食药监局则吊销其《药品生产许可证》，并没收违法所得18.9亿元和处违法生产、销售货值金额三倍罚款72.1亿元，共计91亿元；此外，案件中涉嫌犯罪的，由司法机关追究刑事责任。
 <br/>
 <br/>
 山东济南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫苗受害家长.html" target="_blank">
  疫苗受害家长
 </a>
 王世霞：〝对受害者来说，肯定是不满意的，说惩罚，只是一个数字，没有真正落实到这些受害的孩子还有这些家庭上，一个字也没有说啊，判刑了吗？罚了91个亿，对我们这些孩子不公平，对以后受害的孩子都没有什么保障。〞
 <br/>
 <br/>
 根据发布的〝赔偿实施方案〞，因接种长春长生公司狂犬病问题疫苗造成一般残疾的，一次性赔偿20万元；造成重度残疾或瘫痪的，一次性赔偿50万元；导致死亡的，一次性赔偿65万元。有疫苗受害者家长表示，看到这些数字很气愤。
 <br/>
 <br/>
 湖北黄冈疫苗受害家长：〝看到这个数字我很震惊，也很气愤，因为疫苗一旦造成损伤，轻的话就是终生残疾，重的话命都没了，那只有一个20万、50万，死亡也才65万赔偿。91个亿啊，这些钱全部收到国库去了，而我们这些受害者就区区的这几十万，怎么够孩子一辈子康复。出现这么大的问题，难道你监管没有责任吗？还得这种不义之财，我们觉得很不合理。〞
 <br/>
 <br/>
 有网友说，任何一件灾难，都能割一把韭菜。雁过拔毛的精神，世界第一，无党能及。也有人问，中共对长春长生罚款91亿，问题疫苗更严重的国企武汉生物为何仅仅批评教育了事？
 <br/>
 <br/>
 对此，旅居加拿大作家盛雪表示，这恰恰说明了中共不是以法律为标准，而是以他们的利益为核心来画圈的，所以毫无公平正义可言。
 <br/>
 <br/>
 新唐人记者陈汉、陈洁、施宜君采访报导
</div>
