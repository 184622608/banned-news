### 连夜补缴百万管理费 红二代曝冯小刚被查内幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】范冰冰被中共央视前名嘴崔永元踢爆〝阴阳合同〞而险遭牢狱之灾，其案近日以追缴8.84亿人民币罚金尘埃落定。同样被崔永元死咬不放的知名导演
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冯小刚.html" target="_blank">
  冯小刚
 </a>
 不久前则安然现身美国。近日，有红二代在网络上爆料冯小刚被查内幕，指其为避祸曾连夜向其挂名的北京电视艺术中心补缴逾百万人民币的管理费。
 <br/>
 <br/>
 中共红二代蔡小心近期在微博发帖爆料称，中共官方今年9月24日曾派调查组进驻北京电视艺术中心（北视），共清查了20多位知名导演的资料，发现这些人的工作关系仍在北视，其中就包括〝某裤子〞。
 <br/>
 <br/>
 由于崔永元之前曾在微博上以〝冯裤子〞指代
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冯小刚.html" target="_blank">
  冯小刚
 </a>
 ，因此外界认为蔡小心所说的〝某裤子〞应是指冯小刚。
 <br/>
 <br/>
 蔡小心还进一步披露，调查组的人员曾追问北视，〝为何他们（20多名导演）不在北视上班？为何不向北视缴管理费？〞 冯小刚在得知上述情况后，几乎被〝吓傻〞，于是〝十万火急地连夜给北视办公室补缴一百万管理费。〞
 <br/>
 <br/>
 第二天（9月25日），调查组的人员再次到北视进行调查，得知冯小刚补缴百万管理费后，便问〝这与工作组有啥关系？〞据称，调查组针对冯小刚等长期未在北视上班的情况还追问〝有请假批条证明吗？〞。
 <br/>
 <br/>
 蔡小心表示，按照规定，冯小刚不管拍过多少电影，其所赚的钱必须向北视上缴，然后再由后者按规定给冯小刚发〝劳务费〞。
 <br/>

 </center>
 <br/>
 今年9月16日，冯小刚与中国娱乐集团华谊兄弟实控人之一的王中军现身美国洛杉矶，出席了《复仇者联盟：无限战争3》的导演乔． 罗素在洛杉矶新开的Simone餐厅的开业派对，一度引起中国网民的调侃与热议。
 <br/>
 <br/>
 当时有网友在海外中文媒体的相关报导下留言称：〝这两位是虎口脱险，胜利大逃亡。〞还有网友调侃称，〝党中央说人可以走，钱留下。〞
 <br/>
 <br/>
 对于蔡小心近日的爆料，冯小刚本人至今未作出任何回应。
 <br/>
 <br/>
 外界注意到，自从被崔永元点名爆料后，虽然冯小刚曾几次公开发帖辟谣，坚称自己从未逃税漏税，还大骂崔永元〝卑鄙〞，但在范冰冰被调查期间，有知情人士透露说，冯小刚在今年9月份频繁注销自己名下公司，连另一位知名女影星赵薇也在注销公司，看来娱乐圈将〝迎来新一轮大地震〞。
 <br/>
 <br/>
 早前范冰冰〝阴阳合同〞丑闻被踢爆后，中共官媒新华社曾发文称，中共国家税务总局即将针对影视行业开展所谓〝专项行动〞，对2018年12月31日前〝自查自纠并到主管税务机关补缴税款的影视企业及相关从业人员〞，可免予行政处罚，不予罚款，否则将〝严肃处理〞云云。
 <br/>
 <br/>
 当时舆论界普遍认为，这是北京当局向影视圈和娱乐圈下的〝最后通牒〞。因此也有分析认为，冯小刚注销自己名下公司，有可能是在为今后〝跑路〞做准备。
 <br/>
 <br/>
 （记者竺颖综合报导）
