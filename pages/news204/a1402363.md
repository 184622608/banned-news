### 外媒披露华为任正非秘闻：为何子女三人三个姓
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月08日讯】中国电信设备巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 集团的创始人任正非的长女孟晚舟在加拿大被捕后，外界对他们父女二人为何姓氏不同产生了好奇。随着外媒陆续爆料，任正非和他的三任妻子以及三个子女的关系也逐一被公诸于众。
 <br/>
 <br/>
 综合海外多家媒体的爆料，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 总裁任正非有过3段婚姻，有1个儿子2个女儿。近日在加拿大被逮捕的孟晚舟和儿子任平（曾名孟平）是他与原配夫人所生，第二个女儿姚安娜则是他与第二任妻子所生。
 <br/>
 <br/>
 任正非的第一任妻子孟军出身高干家庭，可以说任正非的事业就是得到妻子家庭的支持才得以起步。因此外界都称任正非当年是入赘孟家，而他与孟军所生的儿子最初名为孟平，长大后才改从父姓，名为任平。长女早年名为任晚舟，16岁以后才自行选择随母姓孟。
 <br/>
 <br/>
 据称，当初孟军的父亲不但帮助任正非的父母〝平反〞，还帮助他于1978年以军代表的身份参加了全国科学大会，对任正非的事业发展给予了很大帮助。
 <br/>
 <br/>
 1982年孟军南下深圳，做了南油集团的高管后，任正非也来到深圳成为南油集团下属一家电子厂的领导。
 <br/>
 <br/>
 不过，最终任正非还是以〝理念不合〞为由与原配夫人离婚。不过有传闻称，他们婚姻破裂的真实原因是出身高干家庭的孟军〝无法忍受被红颜包围的丈夫〞而断然提出离婚。
 <br/>
 <center>

 </center>
 <br/>
 任正非的第二任妻子姚凌（姚玲育）原本是任的秘书，比任小十几岁。任正非54岁时与第二任妻子生育下小女儿姚安娜（英文名Annable Yao），也是随母姓。
 <br/>
 <br/>
 姚安娜现年20岁，从5岁起学习跳芭蕾舞，目前在美国就读哈佛大学电脑工程与统计科系。
 <br/>
 <br/>
 据爆料，姚凌1998年是在昆明生下了女儿姚安娜，直到女儿8个月大的时候，母女二人才回到深圳与任正非一同生活。早年间外界几乎都不知道任正非还有第二个女儿，直到
 <br/>
 <br/>
 而这段婚姻最终还是破裂的结局，至于究竟是谁因为什么理由主动提出离婚，外界已不得而知。直到今年11月她突然高调出现在巴黎名媛舞会上，并接受媒体专访而一度成为媒体新宠。
 <br/>
 <br/>
 《巴黎竞赛画报》在舞会开幕前夕，曾用长达6版的篇幅报道了 Annabel Yao的家庭背景、成长经历，并刊出了一张全家福。
 <br/>
 <br/>
 任正非的第3任妻子苏薇据传出生于1980年后，也是任的秘书，年龄比任正非的长子、长女都还要小。具体结婚的时间不明确，但至少从2007年年底起，华为内部员工之间已在盛传任总离婚又结婚了。
 <br/>
 <br/>
 苏薇与任正非结婚后就在深圳负责照顾任正非的生活起居，两人并未育有子女。
 <br/>
 <br/>
