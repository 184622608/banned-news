### 沙尘暴严重 新疆多地降下〝黄雪〞
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 多地日前出现严重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙尘暴.html" target="_blank">
  沙尘暴
 </a>
 ，其中乌鲁木齐等地还降下〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄雪.html" target="_blank">
  黄雪
 </a>
 〞。堆积在地面的雪，表面掺杂着浅咖啡色的沙尘，民众苦中作乐戏称为〝提拉米苏蛋糕〞。
 <br/>
 <br/>
 受强冷空气影响，11月30日起，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆.html" target="_blank">
  新疆
 </a>
 多地出现7级以上大风，克拉玛依一带出现
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙尘暴.html" target="_blank">
  沙尘暴
 </a>
 天气。大风带着黄沙，进入乌鲁木齐等地，形成降雪降尘。
 <br/>
 <br/>
 从12月1日拍摄的画面看到，城市街道和汽车被雪和沙尘覆盖着，被网友戏称提拉米苏蛋糕。
 <br/>
 <br/>
 专家分析，降雪之后又落下黄沙，出现了如今的分层效果。
 <br/>
 <br/>
 乌鲁木齐和附近城市石河子、昌吉降下的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄雪.html" target="_blank">
  黄雪
 </a>
 有10到20公分。
 <br/>
 <br/>
 新唐人电视综合报导
