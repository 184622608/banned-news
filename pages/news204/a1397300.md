### 重庆公交车坠江正面视频曝光 目击者称满载乘客
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】日前，网上传出重庆公交车正面撞击私家车后坠江的视频，整个过程不到5秒钟。据官方通报公交车上有15人遇难，但有目击者称，车上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_满载乘客.html" target="_blank">
  满载乘客
 </a>
 ，死亡人数远超官方声称的数字。
 <br/>
 <br/>
 10月28日，一辆公交客车在重庆万州区长江二桥撞上一辆私家车后坠入江中。日前，大陆网路传出公交车正面撞击私家车坠江的视频。
 <br/>
 <br/>
 视频显示，公交车失控后，一共横跨3条行车线，与对面一辆红色轿车相撞后，公交车撞断桥边护栏，坠入长江。整个过程不到5秒时间。事故发生的瞬间，被红色轿车后面的一辆汽车的行车记录仪拍下。
 <br/>
 <br/>
 事故发生后，官方通报公交车上有15人失联，但迄今只打捞出2名遇难者遗体。有现场市民对《自由亚洲》表示，两人只是一直漂浮在水面很长时间，才被打捞。还有目击者称，车上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_满载乘客.html" target="_blank">
  满载乘客
 </a>
 ，死亡人数远超官方披露数字。
 <br/>
 <br/>
 据陆媒报导，事故发生后，第一时间曾有网民爆料称，出事公交车车牌为F27085，22路车，驾驶员叫冉勇，车上可能有29人。
 <br/>
 <br/>
 更令外界质疑的是，公车上都安装与公司联网的随车监控系统，但迄今为止，官方仍未公开随车监控系统的任何资讯。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 迄今，公交车坠入长江已3天，但救援工作进展迟缓，官方仍未将公交车打捞上来。据中共官媒29日中午援引万州方面的消息称，在长江二桥公车坠江现场有70多艘船隻进行搜救，待车辆位置确定后，将根据打捞方案派潜水员水下定位起吊。
 <br/>
 <br/>
 不过，当地人说，迄今为止，官方仍无法找到坠江公交车的准确位置。官媒只是对救援没有进展，换了一个说法。
 <br/>
 <br/>
 据熟悉该水域的当地人张先生透露，〝应该是还没找到，现在长江三峡库区蓄水已经达到最高水位了，水比较深。长江二桥肯定超过70米了，有江水流动的话，这个车掉下去的话，一定会漂移，应该在江水的两公里范围内，往下。三峡库区由于那个流速放缓，水底的淤泥很多的，我想声纳也不是很顶用。〞
 <br/>
 <br/>
 〝今天万州一中在它门口拉起了警戒线，所有的行人都不准往那个二桥方向走了。而且我听人们的传言说，捞起来的两具尸体，实际上是一家三口走在桥上，被公交车给撞下去了两个。实际上车里的人一个都没起来。〞张先生称。
 <br/>
 <br/>
 张先生还表示，车祸刚出来，我最先想到的是那个桥面的路拦，实际上是不规范的。那才是真正的要命的原因。桥面上的护栏是按照城区公路标准施工的，而不是按照桥梁方面的那种安全防护，这车辆很容易就冲出去。而且7月份那个桥出现重大的品质事故，万州还把二桥给禁止通行了，维修那个桥。
 <br/>
 <br/>
 有护栏设计人员也明确表示，该桥在桥面与人行道的高差存在较大缺陷，这也可能是导致悲剧的主要原因。
 <br/>
 <br/>
 有消息称，28日凌晨5时24分时，出事公交车司机冉勇，曾在全民K歌APP演唱歌曲《再回首》，此时，距离事发只有4个半小时。外界猜测，冉勇或因疲劳驾驶，酿成悲剧。
 <br/>
 <br/>
 另据香港《苹果日报》报导，29日晚9点左右，有网民发布视频，指坠江公交车已被吊起。但该消息尚未得到官方证实。有当地旅行社也指，公交车现在还未被打捞上来。
 <br/>
 <br/>
 不过，网民对此消息已议论纷纷，有人甚至质疑当局有心隐瞒真相:〝晚上看不清到底死多少人，你懂的。〞
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：范铭）
