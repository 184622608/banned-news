### 北京女抢公交车方向盘：要死一起死(视频)
------------------------

<div class="wysiwyg">
 【新唐人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 时间2018年11月16日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆公交车.html" target="_blank">
  重庆公交车
 </a>
 坠江惨剧尚未从人们心中淡去。11月15日，北京再发生类似事件。一名女子在公交车上抢夺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司机.html" target="_blank">
  司机
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_方向盘.html" target="_blank">
  方向盘
 </a>
 ，还高喊〝要死一起死〞，险些造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_交通事故.html" target="_blank">
  交通事故
 </a>
 。
 <br/>
 <br/>
 据陆媒报导，11月15日上午10点30分，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 市顺义区945公交车上，一名女子先是拿起后车门的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灭火器.html" target="_blank">
  灭火器
 </a>
 ，想要喷射，失败以后，把灭火器扔在了地上，开始乱砸东西大闹。
 <br/>
 <br/>
 接着又冲往前车想要抢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司机.html" target="_blank">
  司机
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_方向盘.html" target="_blank">
  方向盘
 </a>
 ，还高喊〝要死一起死〞，之后被随车保安拦住。目前女子已被北京顺义警方带走。
 <br/>
 <br/>
 <br/>
 <br/>
 近年来，类似事件在中国大陆各地不断出现。10月28日，重庆一女子与公交车司机互殴，造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公交车坠江.html" target="_blank">
  公交车坠江
 </a>
 全车人遇难。
 <br/>
 <br/>
 10月29日，北京一辆678路公交车驶出洋桥西公交站时，有女乘客因坐过站，要求司机停车不果，将手提的整箱盒装牛奶砸向司机手部，结果司机闪躲时，巴士偏离了正常行驶方向，与旁边一辆私家车发生碰撞。女乘客事后被刑事拘留。
 <br/>
 <br/>
 11月14日，中国人民大学国发院公共政策研讨会上，一名教授说，类似万州公交车坠江事件，并不鲜见，人们不仅没有从这些事件里吸取一点教训，反而变本加厉。
 <br/>
 <br/>
 他举例说，一位老人怒拽司机，倒致9车相撞，有关部门也只是对老人进行了批评教育，他说，〝如果抢了方向盘，最多只是警告，或者只是缓刑，制度根本没有为制止犯罪提供动力。〞
 <br/>
 <br/>
 署名袁斌的作者认为，类似重庆公车坠江惨剧也好，形形色色的类似事件也好，无一不是当下中国社会道德普遍沦丧的恶果，也无一不是这种沦丧的折射。
 <br/>
 <br/>
 袁斌说，中共从起家，一直不遗余力的破坏老祖宗传给我们的传统文化，热衷于你死我活的阶级斗争，其结果使得今天的中国人全面丧失了道德底线。
 <br/>
 <br/>
 再加上在中共的暴政下，中国人尤其是底层民众，长期生活在一个没有爱，没有笑容的环境中，对生活资源的争夺又极其残酷，造成人与人之间缺乏起码的尊重和理解，动辄因为一点微不足道的小事，争来斗去彼此互掐。
 <br/>
 <br/>
 袁斌说，在当下中国，这种戾气和冷漠，已经演变成了一种弥漫于整个社会，而且还在不断扩散的普遍现象.如果不能及早结束中共的暴政，复兴老祖宗留给中国人的传统文化，实现全民族的道德重生，试想等待中国人的又将是什么？
 <br/>
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
