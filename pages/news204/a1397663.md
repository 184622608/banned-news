### 中共山东省前副省长季缃绮受审
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】中共官媒周四通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东省前副省长.html" target="_blank">
  山东省前副省长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_季缃绮.html" target="_blank">
  季缃绮
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受审.html" target="_blank">
  受审
 </a>
 ，他被指控受贿人民币3795多万元。
 <br/>
 <br/>
 法庭指，2003年到2017年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_季缃绮.html" target="_blank">
  季缃绮
 </a>
 担任山东省商业集团总公司董事长、山东省商业集团有限公司董事长、山东银座美术馆法定代表人、山东副省长等职务期间，为相关单位在商业合作、承揽工程等事项上谋取利益，非法收受财物，共计2571万余元，骗取山东银座美术馆馆藏书画作品等据为己有，共计折合人民币1224万余元。
 <br/>
 <br/>
 据报，季缃绮2009年主掌山东国企鲁商集团时，曾被举报涉嫌侵吞国有资产。
 <br/>
 <br/>
 山东曾是江派的势力范围，江派前常委吴官正与张高丽曾相继主政山东，而当年鲁能事件牵涉到曾庆红家族。
 <br/>
 <br/>
 另据《明慧网》报导，山东省商业集团下属单位的法轮功学员李建强遭迫害，季缃绮是相关责任人。
</div>
