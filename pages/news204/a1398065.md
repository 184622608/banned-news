### 3女孩搅动陕西千亿国企 西安财政高官被拉下马
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】陕西〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安高新控股.html" target="_blank">
  西安高新控股
 </a>
 〞是个总资产超过千亿人民币的大型
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国企.html" target="_blank">
  国企
 </a>
 。近日，有陆媒曝光，两个月前其企业管理高层换帅，新任用了3名年轻女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高管.html" target="_blank">
  高管
 </a>
 ，引发舆论质疑。10月4日，官方宣布这3人已被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_停职调查.html" target="_blank">
  停职调查
 </a>
 。5日凌晨，西安官方再次通报，西安高新区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财政局.html" target="_blank">
  财政局
 </a>
 长被免职。
 <br/>
 <br/>
 针对〝千亿资产
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国企.html" target="_blank">
  国企
 </a>
 由80后任董事长，95后任董事〞事件，11月5日凌晨，西安高新区管委会官方微信公众号再次发声明，不仅曝光了3位年轻女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高管.html" target="_blank">
  高管
 </a>
 的工资，还公布了西安高新区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财政局.html" target="_blank">
  财政局
 </a>
 长被免职的消息。
 <br/>
 <br/>
 通报称，高新区财政局违反管委会国有企业管理人员任用的相关规定，擅自变更企业法人代表及董事，经研究决定免去王进杰西安高新区财政局局长职务。
 <br/>
 <br/>
 对于舆论关注的李甜等3名高管，通报称，她们并没有特殊的家庭背景。目前，李甜月工资4,351元人民币；朱玥月工资3,600元人民币；赵雪莹月工资4,200元人民币，在任职前后薪资无变化。
 <br/>
 <br/>
 其中李甜在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安高新控股.html" target="_blank">
  西安高新控股
 </a>
 有限公司早前的公示中，为该公司法人、董事长兼总经理。赵雪莹和朱玥为该公司董事。
 <br/>
 <br/>
 上述事件缘于西安高新控股9月份发布人事调整公告称，1984年8月出生的李甜，出任公司的法人代表、董事长兼总经理。
 <br/>
 <br/>
 而新任的董事分别是赵雪莹和朱玥，其中赵雪莹1993年生、2016年毕业于西安建筑科技大学、曾任职银行；朱玥1995年生，2017年毕业于陕西师范大学。
 <br/>
 <br/>
 相关消息在10月2日被大陆媒体挖出，李甜等3名年轻高管掌控总资产超过千亿人民币的国企，引起舆论质疑。
 <br/>
 <br/>
 10月3日晚，西安高新区财政局首度发布声明回应称，经调查，没有发现影响公司正常运营的社会关系，同时也承认在选用企业管理人员的方法上存在问题。
 <br/>
 <br/>
 11月4日，西安高新区财政局再次发声明称，西安高新区纪工委已经介入，正在调查3名担任高新控股董事长、董事的年龄、学历、资格和社会背景是否合规。目前已决定对3人停职，并启动相关法律程序。在接下来，将对所属国有企业全面展开调查。
 <br/>
 <br/>
 但网友们对迟来的停职和调查及官方的回应仍感不满，〝毕业就成董事，现在的国有企业俨然就是私有企业，搞世袭吗？〞〝只能说这是中国的现状，普遍的现状〞。
 <br/>
 <br/>
 公开资料显示，西安高新控股公司于2003年10月成立，注册资本11.3亿元，由高新区财政局负责管理。截至今年6月底，高新控股公司总资产1270亿元，净资产403亿元。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿)
