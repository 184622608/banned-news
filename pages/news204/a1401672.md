### 陆媒首次披露 项俊波情妇与北京豪宅
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】中共保监会前主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_项俊波.html" target="_blank">
  项俊波
 </a>
 案更多细节被曝光。近日，陆媒首次披露，项俊波有一名孙姓女友和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_豪宅.html" target="_blank">
  豪宅
 </a>
 ，当女友向他索要这套房产时，他迅速将豪宅出售。
 <br/>
 <br/>
 据〝中国金融报〞日前披露，中共保监会前主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_项俊波.html" target="_blank">
  项俊波
 </a>
 ，曾在位于北京金融街核心区购置一套面积为263平方米的大户型住宅，住宅小区对面是北京市第八中学，该房是项俊波和妻子杨某给女儿购置的房产。
 <br/>
 <br/>
 报导说，项俊波与杨某离婚后，与一孙姓女子交往，2009年孙某提出想要这套房，项俊波没有答应，二人还因此吵架，于是，项俊波决定赶紧处理掉这套〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_豪宅.html" target="_blank">
  豪宅
 </a>
 〞。最后，该房以约为1,100万元的价格成交，而这个价格超出当时市场正常价格三四百万元。
 <br/>
 <br/>
 这是陆媒首次公开项俊波的其中一名〝女友〞，和他在北京繁华地段拥有〝豪宅〞。
 <br/>
 <br/>
 项俊波长期在金融体系工作，曾任审计署副审计长，中国农业银行行长、董事长。2011年10月起任保监会主席。去年4月9日，项俊波被宣布接受审查。成为北京当局清查金融界贪腐案，首名被查的〝第一虎〞。
 <br/>
 <br/>
 去年9月23日，中纪委通告，项俊波被开除党籍、公职。今年6月，项俊波受贿案公开审理，其被控受贿1,942万元人民币。检方指控，项俊波直接或通过特定关系人（妻子）杨光受贿约数千万元人民币。
 <br/>
 <br/>
 杨光是项俊波的妻子，长期在美国生活，二人结婚、离婚，后又复婚，他们所受贿赂中，最大一笔达到700余万元。杨光现已被另案处理。
 <br/>
 <br/>
 公开资料显示，项俊波现年60岁，曾在审计系统任职多年，后转至金融系统，曾任央行副行长、中国农业银行行长、党委书记、保监会主席等职。
 <br/>
 <br/>
 此前消息透露，项俊波
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_好色成性.html" target="_blank">
  好色成性
 </a>
 ，共有21个固定情人，经常在办公室和下属淫乱，还有6个私生子，经鉴定只有1个是他亲生。
 <br/>
 <br/>
 中南海的消息人士此前透露，项俊波是明天系掌门人肖建华咬出的金融〝第一虎〞。2015年中国股灾，项因发动〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融政变.html" target="_blank">
  金融政变
 </a>
 〞，对抗习近平当局遭调查。
 <br/>
 <br/>
 肖建华被指是曾庆红的白手套，为江派利益集团在海外走资、洗钱。2017年大年三十已被北京当局抓捕。北京评估其经手的资产多达2万亿元人民币。其中很大一部分是保险资金，与项俊波密切相关。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
