### 闪避逆向轿车 重庆大巴冲破护栏坠入长江(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月28日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万州区.html" target="_blank">
  万州区
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长江二桥.html" target="_blank">
  长江二桥
 </a>
 上28日上午发生一起严重交通事故，一辆大巴士为阬闪避逆向行驶的红色轿车，冲破护栏掉入长江，目前不清楚巴士上乘客数量，但已打捞出1名遇难者尸体。
 <br/>
 <br/>
 综合媒体报导，事发在上午10时左右，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万州区.html" target="_blank">
  万州区
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长江二桥.html" target="_blank">
  长江二桥
 </a>
 上，一名驾驶红色轿车的女司机逆向行驶，一辆当地的22路公车疑为闪避红色轿车而冲出护栏，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠入长江.html" target="_blank">
  坠入长江
 </a>
 。
 <br/>
 <br/>
 目前无法确定公车上的乘客数量，而现场已经拉起封锁线，并已打捞出1名遇难者尸体，相关部门正在搜救中。
