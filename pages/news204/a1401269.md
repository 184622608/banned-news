### 四川男吞3鱼胆〝清火〞 肝肾衰竭险送命
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】近日，四川一名男子因上火眼睛红痛，于是生吞3颗鱼胆〝清热〞，不料却出现恶心呕吐、腹泻等中毒症状。家人随后将他送院救治，被诊断出现急性
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肝肾衰竭.html" target="_blank">
  肝肾衰竭
 </a>
 症状，所幸经抢救已脱离生命危险。
 <br/>
 <br/>
 据中医医书《证治准绳》记载，鱼胆有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清热解毒.html" target="_blank">
  清热解毒
 </a>
 、清肝明目、止痛的功效，可治目赤肿痛。但因其有毒性，所以用药稍有不当就会使人中毒。
 <br/>
 <br/>
 近日，四川一名30岁的男子，因眼睛发红，且感觉眼睛肿胀有异物感，他母亲认为是上火，热气重引起的，于是建议他吃鱼胆清火。
 <br/>
 <br/>
 该男子于是买了几条大鱼，并取出3个如鸽蛋般大小的鱼胆生吞。不料，1个小时后男子就出现恶心呕吐、腹泻等症状。他没有去医院就诊，而是自己服用了一些药物缓解。
 <br/>
 <br/>
 3天后，该男子病情加重被送往医院救治。医生检查后发现，他当时已出现肝肾功能损害或衰竭的症状，肝脏肿大、肝脏部位痛、皮肤黄、浮肿、少尿、血尿、头晕、嗜睡、出血及溶血等情况。
 <br/>
 <br/>
 经过13天的抢救，该男子现已脱离生命危险。
 <br/>
 <br/>
 医生提醒道，一旦鱼胆中毒，没有特殊的解毒药，必须及早治疗，并指鱼胆愈大，所含毒素愈多。吞食鱼重2.5公斤的鱼胆2个或鱼重5公斤鱼胆1个，便可导致死亡。所以不要盲目自行服用鱼胆，以免出现生命危险。
 <br/>
 <br/>
 （记者罗婷报导/责任编辑：文慧）
