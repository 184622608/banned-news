### 中共强化控制少数民族 宁夏欲移植〝新疆经验〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】中共宁夏回族自治区政府的一众官员近日前往新疆〝考察学习〞，并与新疆维吾尔自治区政府签订了所谓反恐合作协议，拟移植新疆当局管控维族人的手段，打压和控制宁夏回族等少数民族群体，引发海外舆论界的关注。
 <br/>
 <br/>
 据中国大陆媒体报导，中共宁夏回族自治区党委常委、政法委书记张韵声等官员，近日前往新疆乌鲁木齐
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_再教育营.html" target="_blank">
  再教育营
 </a>
 （〝职业教育培训中心〞）、看守所等大规模关押维族人进行强制洗脑的地方进行〝考察〞。
 <br/>
 <br/>
 张韵声高调声称，新疆当局在〝反恐维稳〞和民族宗教的管控等方面创造的经验做法，值得宁夏〝学习借鉴〞，并表示要在这些方面加强两地的〝深层合作〞。
 <br/>
 <br/>
 在这次考察期间，宁夏新疆两地的公安和教育机构还签订了所谓〝反恐维稳〞等方面的合作协议。
 <br/>
 <br/>
 上述消息引起海外舆论界的关注与担忧。外界担忧宁夏等少数民族聚居的地区，恐将移植新疆再教育营大规模非法拘押少数民族民众的做法，跨省份强化对少数民族的管控，从而让中国的人权状况更趋恶劣。
 <br/>
 <br/>
 英国《金融时报》日前发表一篇文章分析称，上述现象显示，中共打压少数民族已从新疆扩展至宁夏，而且有意将管控新疆维族人的手段输出到宁夏，说明中共对其境内少数民族的管控更趋严厉。
 <br/>
 <br/>
 世界维吾尔大会发言人迪里夏提在接受《自由亚洲电台》的采访时表示，中共过去曾因在宁夏强制拆迁宗教场所，引起回族民众的反抗与动荡，〝军警民爆发激烈冲突，让北京当局感到非常惊恐〞，所以现在中共政府想把在新疆的强硬肃清政策移植到宁夏去推行。
 <br/>
 <br/>
 香港珠海书院专家则评论称，中共加强打压回族等少数民族，说明其过往〝以经济解决民族问题〞的政策已宣告失败。而在中国大陆，除宁夏外，青海、陕西及甘肃都有大量回族人口，如果中共肆意在这些地区扩大推行其强硬管控的手段，势必激起这些地区民众的反抗意识。
 <br/>
 <br/>
 另据《自由亚洲电台》报导，新疆当局最近再出狠招，逼迫那些被羁押在〝再教育营〞中接受洗脑的维族人与其身在海外的配偶离婚或与亲属脱离关系。这样做将使得海外的配偶或亲属失去与遭关押者之间的法律关系，让海外团体的救援行动无法正常展开。
 <br/>
 <br/>
 （记者李明综合报导/责任编辑：秦锋）
