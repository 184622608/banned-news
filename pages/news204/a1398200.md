### 兰州特大连环车祸是〝人祸〞？ 收费路口14年死57人（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】甘肃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_兰州南高速收费路口.html" target="_blank">
  兰州南高速收费路口
 </a>
 日前发生31辆车连环被撞的特大交通事故。据当地媒体和网友揭露，该收费站设在17公里的下坡路上，开通14年来，发生200多起车辆失控相撞事故，迄今已有57人死亡。网民认为这是当地政府官员不作为导致的人祸。
 <br/>
 <br/>
 11月3日傍晚19时21分，辽宁省籍驾驶人李某驾驶一辆重型半挂载重牵引车，经17公里长下坡路段，行驶至距兰州南收费站50米处，与在排队等候缴费的31辆车连环相撞。
 <br/>
 <br/>
 经初步调查，驾驶人李某是第一次在该路段行驶，因频繁采取制动（指踩刹车），导致车辆刹车失灵，引发特大车祸。
 <br/>
 <br/>
 据官方通报，截至11月4日凌晨5时，事故已导致15人死亡，44人受伤。当地媒体及网民都认为这又是一起人祸。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 据悉，该收费站设在17公里的下坡路上，当地媒体称此路段为〝危险路段〞。早在2013年11月，《兰州日报》就曾报导称，兰临高速17公里长下坡路段本身就容易发生货车失控引发的重特大道路交通事故，如果有失控的大货车冲了下来，正好又碰上了停在这里等待进城的大货车，那么发生群死群伤的交通事故就在所难免了。
 <br/>
 <br/>
 据大陆官方统计数据显示，该收费站自2004年12月开通至今，至少已发生241辆车辆失控事故，共导致57人死亡，99人受伤。
