### 吉林东丰县机械厂大爆炸  多人死伤
------------------------

<div class="wysiwyg">
 【新唐人2018年11月24日讯】当地时间周五晚，吉林省东丰县，一处机械厂房发生爆炸，并引发大火。附近居民震感强烈，官方微博称，已有2人死亡，57人受伤，但实际死伤人数，还有待核实。
 <br/>
 <br/>
 周五晚间11点40分左右，位于吉林省东丰县三合乡永安村的江城建设集团机械厂，突然发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大爆炸.html" target="_blank">
  大爆炸
 </a>
 。附近梅河口市居民都听到巨响并有强烈震感。
 <br/>
 <br/>
 爆炸引发大火，直到周六清晨，现场仍然浓烟滚滚。
 <br/>
 <br/>
 有民众透露，事发工厂距离村民居住地很近。爆炸造成大量民房受损。
 <br/>
 <br/>
 根据东丰县官方微博，事件已造成2人死亡，57人受伤，370户房屋受损。不过，有网民留言说实际伤亡数字不止于此。
 <br/>
 <br/>
 新唐人记者林宇综合报导
</div>
