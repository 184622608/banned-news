### 央视名嘴6人患癌死 罗京最惨疑染艾滋
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】央视前主持人李咏日前因患喉癌病逝，成为各媒体头条话题。据媒体盘点，过去9年间，已有至少6个央视知名主持人罹患癌症死亡。其中最惨的是央视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新闻联播.html" target="_blank">
  新闻联播
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_播音员.html" target="_blank">
  播音员
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗京.html" target="_blank">
  罗京
 </a>
 ，但也有消息称，罗京死于艾滋病复发。
 <br/>
 <br/>
 10月29日，李咏的妻子、前央视导演哈文发微博称，李咏因患癌症医治无效，在美国去世。哈文还披露，李咏临终最后一句话：〝没有遗憾，只有不舍。〞
 <br/>
 <br/>
 李咏生前的最后一条微博停留在去年的感恩节。他感谢了妻子、女儿以及所有人。两年前他接受采访时，他曾描述过离开央视后的生活状态：〝其乐融融，挣钱养家，准备学业。〞
 <br/>
 <br/>
 据透露，李咏患的是喉癌，或许是因病情严重，李咏在美国治疗了17个月后离开了人世，终年50岁。
 <br/>
 <br/>
 近年来，已有不少
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视主持人.html" target="_blank">
  央视主持人
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_播音员.html" target="_blank">
  播音员
 </a>
 、制片人因患癌症死亡。据陆媒盘点，过去9年间，仅央视名主持人就有6人患癌症死亡。
 <br/>
 <br/>
 除李咏外，还有央视前主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肖晓琳.html" target="_blank">
  肖晓琳
 </a>
 、央视
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新闻联播.html" target="_blank">
  新闻联播
 </a>
 播音员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罗京.html" target="_blank">
  罗京
 </a>
 、央视主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_方静.html" target="_blank">
  方静
 </a>
 、央视主持人王欢等。
 <br/>
 <br/>
 其中，央视前主持人肖晓琳于去年7月因患直肠癌，在美国治病期间去世，时年55岁。肖晓琳长期担任央视《今日说法》栏目的制片人、主持人，该栏目曾多次推出污衊法轮功的节目。
 <br/>
 <br/>
 如在2001年3月22日的访谈节目中，提及同年1月23日发生在天安门广场的所谓〝自焚事件〞，时任央视主持人张绍刚和两名受访者，对法轮功进行了歪曲和攻击。
 <br/>
 <br/>
 〝自焚事件〞发生一星期后，央视《焦点访谈》推出了一个30分钟的相关节目，但节目破绽百出。
 <br/>
 <br/>
 同年8月，国际教育发展组织明确指，自焚事件〝是由（中共）政府一手导演的〞。北美中文电视台〝新唐人〞制作的分析自焚真相的纪录片《伪火》（False Fire），于2003年11月8日获得第51届哥伦布国际电影电视节荣誉奖。
 <br/>
 <br/>
 在纪录片《伪火》揭穿〝天安门自焚〞真相后，央视主持人方静仍在2005年1月23日央视的《焦点访谈》节目中，重复播报〝天安门自焚〞谎言，并宣称去河南做了所谓的〝追踪采访〞。
 <br/>
 <br/>
 2015年11月，方静因为癌症医治无效，在台湾死亡，年仅44岁。
 <br/>
 <br/>
 <h4>
  罗京死状最惨
 </h4>
 <br/>
 央视名主持人中死状最惨的是罗京。2009年正值事业与生活最颠峰的罗京忽然死亡。
 <br/>
 <br/>
 官方称，罗京因淋巴癌不治身亡，后来又爆出罗京的真正死因是艾滋病。当时央视高层让所有与罗京有过密切接触者全部接受体检，甚至隔离，对外严密封锁消息，时任台长的赵化勇也因牵涉其中，被调离原有岗位。
 <br/>
 <br/>
 据爆料，外表不苟言笑、〝正襟危坐〞的罗京，是北京富豪出没的后海、三里屯等娱乐场所的常客。以罗京、赵忠祥等为首的所谓央视系，喜欢出没北京灯红酒绿、充斥着繁华与显贵的〝后海〞。
 <br/>
 <br/>
 据说，罗京为人十分风流，给小费也非常丰厚。罗京发病当晚仍在夜总会鬼混，突然罗京晕厥，赶紧送医院治疗。虽然过几天出院，但后来几天罗京症状越来越严重。
 <br/>
 <br/>
 后来，罗京的全身不断泛出红斑点，随后全身淋巴结肿大，血样报告显示，阳性，艾滋病！2009年6月5日，罗京停止了呼吸。时年48岁。
 <br/>
 <br/>
 罗京死后，时任中共中央政治局常委的李长春向央视发出唁电，称罗京〝政治坚定〞。
 <br/>
 <br/>
 所谓〝政治坚定〞具体体现在罗京在重大问题上，与中共保持高度一致。比如，1989年〝六四〞学生运动时，罗京声称，天安门没死一个人，〝六四〞运动是暴乱等。
 <br/>
 <br/>
 1999年7月，中共镇压法轮功后，罗京成为中共的传声筒，播报给法轮功造谣、栽赃的假新闻。
 <br/>
 <br/>
 据悉，罗京死前非常痛苦。护士邢桂芝透露：罗京〝喝一口水，疼得把眉毛都纠结在一起，我们就给他配了麻药，漱完口之后再吃药、吃饭。〞
 <br/>
 <br/>
 明慧网记述的类似案例在迫害法轮功修炼者的恶人身上数不胜数，许多追随江泽民集团迫害、污衊法轮功的人噩运随身，痛苦死亡，这或是善恶有报天理的展现。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：赵云）
