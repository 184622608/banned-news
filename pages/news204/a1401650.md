### 川习会气氛良好传出掌声 晚宴菜单曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月02日讯】美国总统川普（特朗普）与中国国家主席习近平在G20峰会期间的工作晚宴，全球瞩目。据报导，当天
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 的气氛良好，晚宴结束时，会议室传来掌声。白宫已将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晚宴菜单.html" target="_blank">
  晚宴菜单
 </a>
 对外公布。
 <br/>
 <br/>
 当地时间下午5点30分左右，美中两国元首和双方会谈代表，在阿根廷首都布宜诺斯艾利斯共进工作晚宴。
 <br/>
 <br/>
 这场会晤倍受世界瞩目，关系到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 的走向，对世界经济格局是一个关键转折点。
 <br/>
 <br/>
 英国《路透社》报导称，在会晤开始时，川普总统说，他同习近平的关系非常特殊，〝我认为，我同习主席的关系，是我们最终可能会做一些对中国和美国都有利的事情的首要原因。〞
 <br/>
 <br/>
 习近平则表示，他和川普有很好的个人友谊，很高兴和川普共同规划下一阶段的中美关系。
 <br/>
 <br/>
 此次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 时间长达两个半小时，比计划延长半小时。中共党媒人民日报海外版微博帐号〝侠客岛〞表示，会谈结束后会议室传出掌声，并临时增加合照环节。
 <br/>
 <br/>
 香港《南华早报》报导称，会谈时间超出预期，显示两国都有意缓和中美紧张局势。〝川习会在阿根廷布宜诺斯艾利斯以积极的调子结束。〞
 <br/>
 <br/>
 据中共央视报导称，中美双方同意明年起不再额外加征关税。
 <br/>
 <br/>
 晚宴结束后，美中双方没有发表联合声明。截至目前，川习会的会谈内容仍未对外公布。据中共官方报导，中方对会谈比较乐观。不过，美国官方尚未对会谈做出表态。白宫仅于12月2日，例行公事地将本次川习会工作晚宴的菜单公布如下：
 <br/>
 <br/>
 前菜：
 <br/>
 <br/>
 阿根廷时令蔬菜沙律（Seasonal Vegetable Salad）
 <br/>
 <br/>
 罗勒蛋黄酱（Basil Mayonnaise Dressing）
 <br/>
 <br/>
 帕尔马芝士酱（Parmesan Emulsion）
 <br/>
 <br/>
 2009年卡帝那‧沙巴达阿德里安娜白石霞多丽白葡萄酒（Catena Zapata Adrianna White Stones Chardonnay 2009）
 <br/>
 <br/>
 主菜：
 <br/>
 <br/>
 烤西冷牛排配洋葱、羊奶乳酪和椰枣（Grilled Sirloin with red onions,goat ricotta,and dates）
 <br/>
 <br/>
 甜点：
 <br/>
 <br/>
 焦糖薄饼卷（Caramel Rolled Pancakes）
 <br/>
 <br/>
 朱古力酥片（Crispy Chocolate）
 <br/>
 <br/>
 鲜奶油（Fresh Cream）
 <br/>
 <br/>
 2014年卡帝那‧沙巴达尼古拉斯马贝克葡萄酒（Catena Zapata NicolÁs Malbec 2014）
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：祝馨睿）
