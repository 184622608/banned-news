### 彭斯讲话引发大陆共鸣 中美专家解读惊人一致
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】10月4日，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 发表有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）政府对华政策的讲话，对中共的内外政策提出严厉批评。对于彭斯的讲话，中美专家的解读惊人相似，一致认为，这是川普政府对中共发出的最后通牒。
 <br/>
 <br/>
 <h4>
  美智库：美中关系正处在十字路口
 </h4>
 <br/>
 华盛顿智库国际与战略研究中心亚洲事务高级顾问兼中国实力项目主任葛来仪（Bonnie Glaser）在接受《美国之音》采访时表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府正试图在诸多议题上对中共进行反击。
 <br/>
 <br/>
 她说：〝这个讲话的确是川普政府关切中国的综合阐述。川普政府里的人已经谈到了美国在与中国（中共）打交道时，有必要加强竞争性。对我来说，这看起来像是他们也正在试图加强对抗性。〞
 <br/>
 <br/>
 葛来仪认为，美中关系很可能处于一个关键的节点。〝这个讲话与川普政府一直采取的立场是一致的，但在美中关系上，我们很可能处于一个十字路口。如果我们不找到消除双边关系中的摩擦的途径，我们可能正在走向一个更为负面的方向，进入更多的战略竞争和对抗的阶段。人们开始谈论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 ，这不是完全不可能的。〞
 <br/>
 <br/>
 <h4>
  美官员：对北京采取
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强硬立场.html" target="_blank">
   强硬立场
  </a>
 </h4>
 <br/>
 研究美国外交政策的专家傅克斯（Michael Fuchs）也对美中关系的前景感到担忧。他认为，这个讲话表明，川普的对华政策将更为强硬。
 <br/>
 <br/>
 他说：〝副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 的演讲清楚地表明，川普政府的对华政策正在走向对中共更为强硬的阶段，这从今年年初的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 就开始了。〞
 <br/>
 <br/>
 傅克斯认为，在今后，人们可能会看到美国在关税和贸易领域，对中共采取更多的行动；而在安全领域，尤其是在南中国海和台湾问题上，美国也可能会对中共采取更为强硬的行动。
 <br/>
 <br/>
 曾担任美国驻北京大使馆外交官的何瑞恩（Ryan Hass）也表示，川普总统基本上支持对中共采取更为强硬的立场，同时，彭斯副总统在讲话中也向美国公司（如Google）发出警告，敦促他们不要屈从北京的压制。
 <br/>
 <br/>
 卡内基国际和平基金会的副总裁包道格（Douglas Paal）说：〝这是一个进行战斗的号声〞，〝并认定美国会赢得这场斗争〞。
 <br/>
 <br/>
 <h4>
  大陆学者：川普仍给
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
   习近平
  </a>
  留有余地
 </h4>
 <br/>
 彭斯的讲话也在引起大陆民众的共鸣。
 <br/>
 <br/>
 有大陆网友表示，〝（彭斯讲话）把中共的性质向全世界说明白了〞，〝很多中国人，中国底层的人，想讲但是讲不出来，因为中国没有这种言论自由，老百姓的心里话到哪去讲？没有地方讲。他们从彭斯的讲话中获得共鸣。〞
 <br/>
 <br/>
 也有舆论认为，彭斯的讲话是美中冷战开打的〝檄文〞和〝号角〞。
 <br/>
 <br/>
 退休的上海政法学院国际问题学者教授倪乐雄，对《美国之音》说：〝最大的感受就是，美国彻底向我们摊牌了。但是，美国摊牌也不想就此全面决裂，产生全面的冲突。〞
 <br/>
 <br/>
 倪乐雄认为，彭斯副总统的讲话，对中国表示了某种期待。否则，川普为什么不出来（直接）讲这些话？说明他还考虑同中国最高领导人（习近平）的关系，所以才让副总统出来发声，以便留有回旋余地。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：祝馨睿）
</div>
