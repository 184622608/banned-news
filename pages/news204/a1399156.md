### 中共〝灭推〞行动  推友斥：邪恶
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 严控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络言论.html" target="_blank">
  网络言论
 </a>
 的行动，近期再次加码升级。据多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推特.html" target="_blank">
  推特
 </a>
 网友爆料，最近中共公安部统一部署，大面积约谈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆推友.html" target="_blank">
  大陆推友
 </a>
 ，一些人被行政拘留或逮捕，更多人被要求删推。一些推友批评中共这种让人民消音的行径〝邪恶〞，并发起抗议行动。
 <br/>
 <br/>
 近期，多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推特.html" target="_blank">
  推特
 </a>
 网友向《新唐人》反映，最近
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 公安部大面积约谈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆推友.html" target="_blank">
  大陆推友
 </a>
 ，警告他们不准在推特上发表所谓的〝不当言论〞，甚至有些被要求写〝保证书〞不再上推特。
 <br/>
 <br/>
 网络自由观察人士古河：〝中共公安部门正在进行一项‘灭推’行动，所有要求民主的人士，在推特上比较活跃的人士，都被警方约谈，然后要求删除推特所有的文章，不论这个文章是否反共、是否和政治有关。〞
 <br/>
 <br/>
 网络自由观察人士古河表示，自己最近因此事被警方找了两次。
 <br/>
 <br/>
 《新唐人》11月12号收到的爆料显示，除了多名推友被警告或约谈，也有推友表示自己〝被抓，被冻了一夜。〞
 <br/>
 <br/>
 大陆推友陈先生说，自己近期也被当地警方找去所谓的接受调查。警方明确要胁他，不允许在推特上发表任何政治言论。
 <br/>
 <br/>
 大陆推友陈先生：〝不允许转发政治言论，不允许转推，也不要去发表任何的政治观点，而且非常明确，没有第二次，这次回去接受调查就已经是最后一次了，如果再不听的话，那个寻釁滋事罪名直接套在你头上，直接开审，想判几年是随意的。〞
 <br/>
 <br/>
 推友〝向莉〞表示，受此次事件影响的，还包括何德普、高瑜、王亚军等知名人士。她感慨，大陆的言论环境可以用〝道路以目〞来形容。一些中文推友也于近日来到美国斯坦福大学，抗议中共让人民消音的〝邪恶行径〞。
 <br/>
 <br/>
 陈先生：〝现在在中国没办法说言论，在中国（我们）描述的真的是一个事实，我们没有刻意去抹黑，事实就是这样的。〞
 <br/>
 <br/>
 推友〝刘强本〞质问发短信〝警告〞自己的警察：你所谓〝不当言论〞的标准是什么？你知道自己在知法犯法，以发恐吓短信的方式侵犯公民的言论自由吗？
 <br/>
 <br/>
 新唐人记者陈汉、王子琦、周天采访报导
</div>
