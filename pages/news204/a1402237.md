### 华为掌门之女被捕 中美爆科技冷战 专家忧中共报复
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】中国电讯网络设备制造巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司的首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在加拿大被逮捕后，中共外交部向加、美两国政府提出了抗议，并要求立即释放孟晚舟。中共官媒则制造舆论，指该事件显示中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技冷战.html" target="_blank">
  科技冷战
 </a>
 已公开化。有美国专家分析称，中共恐将以扣押美国公司高管的方式进行报复。
 <br/>
 <br/>
 应美国执法部门的引渡要求，加拿大警方近日逮捕了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 副董事长兼首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 ，中共官方对此十分恼怒。
 <br/>
 <br/>
 当地时间12月6日，中共外交部发言人在记者会上宣布，中方已就孟晚舟在加拿大被逮捕事件分别向加、美两国政府表明了立场，〝要求对方立即对拘押的理由作出澄清，立即释放被拘押的人员。〞
 <br/>
 <br/>
 事实上，孟晚舟12月1日晚在加拿大被捕后，中共驻渥太华大使馆已经表达了〝强烈抗议〞，并指此举伤害孟晚舟〝人权〞，要求加拿大立即放人。
 <br/>
 <br/>
 加拿大司法部5日则发表声明称，逮捕孟晚舟是应美国执法部门的引渡要求而采取的行动，原因是孟晚舟涉嫌违反美国的制裁禁令，向伊朗和其他被制裁的国家提供了源自美国的产品。
 <br/>
 <br/>
 但中共官方媒体却把孟晚舟被捕事件描述为美国政府欲打压中方在互联网通讯技术领域的发展而采取的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科技冷战.html" target="_blank">
  科技冷战
 </a>
 〞行动。
 <br/>
 <br/>
 亲北京的《香港经济日报》日前发文宣称，〝哪个国家掌握最新的5Ｇ通讯技术，将在今后一个时代里稳握世界科技、经济、甚至军事巨人的权杖。〞因此各国政府都在争夺5G科技主导权，而华为是〝中国队〞的主力。
 <br/>
 <br/>
 文章援引未具名的〝观察人士〞的观点称，华为目前已是全球第二大通讯装备生产商，但受美国遏抑的力度却逐年增加，而美方打击华为的动机就是想〝阻截中国获得最高端通讯科技〞云云。
 <br/>
 <br/>
 该文把近年来AT&amp;T、Verizon 和Best Buy等美国电讯商放弃销售华为手机或停止从华为采购最新款智能手机的决策，以及美国政府制定法案禁止美国核武基础设施采用华为和中兴设备的措施，都视为美中之间〝科技冷战〞正在开打的明证。
 <br/>
 <br/>
 据悉，孟晚舟是华为创始人任正非的长女，在华为公司的地位与发挥的作用可谓举足轻重，因此被外界戏称为〝华为太子女〞。而这次中共官方对孟晚舟被捕表现出的激烈反应，引起了美国舆论界的猜测与担忧。
 <br/>
 <br/>
 美国战略与国际问题研究中心技术政策主管James Lewis日前向美国媒体Axios表示，华为是最受中共政府喜爱的企业之一，孟晚舟被逮捕可能会促使中美紧张关系升级。Lewis提醒称，〝他们（中共官方）将报复，中国（中共）将扣押人质〞。
 <br/>
 <br/>
 鉴于中共官方这次对孟晚舟被捕一事作出的异乎寻常的激烈反应，英国媒体BBC评论称，〝任正非之女被捕激怒了北京〞。
 <br/>
 <br/>
 据公开的资讯，华为虽然名义上并不是中共治下的〝国企〞，但该公司的创始人和前董事长却有着中共军方或国安系统的背景，因此西方国家十分担心华为公司会被中共安全机构利用来搜集情报。
 <br/>
 <br/>
 2012年美国国会的一份报告曾明确指出，华为和中兴通讯都可能成为中共政府支持的间谍或蓄意破坏行为的工具。为此，美国国会参众两院的一些议员们还发起了一项禁止美国政府、以及其承包商使用华为和中兴设备的法案。
 <br/>
 <br/>
 与此同时，美国政府也在敦促其盟友国家停用华为高端装备。今年 11 月，美国代表团访问欧洲是就曾明确表示，出于安全的考虑，欧洲作为美国的盟友必须在5G排除华为。而澳洲亦已把华为5G拒诸门外。
 <br/>
 <br/>
 前不久，澳大利亚和新西兰政府已先后作出抉择，明确表态拒絶本国移动网络使用华为制造的5G技术设备。
 <br/>
 <br/>
 就在12月4日，英国情报部门军情六处的负责人扬格刚刚对英国政府发出警告说，中共的新技术主宰将对未来构成更大的安全威胁。他表示，中共参与西方的通讯基础设施建设造成了安全风险，可能令西方国家陷入受制于中共的被动境地，或遭受中共通过网络后门的监视。
 <br/>
 <br/>
 扬格还专门提到了华为公司，他表示，不久当局将会对华为能否参加英国的5G移动网建设作决定。
 <br/>
 <br/>
 12月5日，英国电信公司（BT）也发表声明称，他们将在两年内把华为技术的设备从其现有的3G和4G移动业务核心中删除，并且不会将中国公司产品用于下一代5G网络的核心部分。
 <br/>
 <br/>
 （记者唐迪综合报导）
