### 中共四中全会罕见推迟  中南海气氛诡异
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】最新消息称，中共十九届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 罕见延期至11月召开。这将成为了中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 以来，召开时间最晚的四中全会。对于会议延期的原因，分析认为与中南海气氛诡异，及贸易战使得中国经济急剧恶化有关。
 <br/>
 <br/>
 依照中共惯例，十九届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 应在10月下旬召开，但此时正值
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 南下考察广东，10月25日安倍访华，显然没有时间召开四中全会。
 <br/>
 <br/>
 香港《明报》10月30日报导说，中共商务部在吹风会上说，习近平11月5日将出席进博会的开幕式，并致开幕辞。进博会从5日开到10日，参会的俄罗斯、巴基斯坦、萨尔瓦多、多米尼加、古巴等5国领导人，还将顺道访问中国，日期从10月31日起到11月8日。所以，四中全会最快也要11月10日才能召开。
 <br/>
 <br/>
 由于11月14日和17日，李克强和习近平将分别参加菲律宾的东亚系列峰会，和巴布亚新几内亚的APEC年度峰会，日程相当紧凑。
 <br/>
 <br/>
 这样来看，十九届四中全会肯定是中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 以来，开会时间最晚的一次，除了1989年开会时间定在6月，其他几届大都是在9月中下旬召开，只有2012年前相对定在10月下旬。
 <br/>
 <br/>
 有美媒引述消息说，北京自10月中旬已经进入会议安保模式，持续至今，但十九届四中全会已被推迟到11月10日前后。
 <br/>
 <br/>
 至于会议推迟的原因，报导说，北京政界有两个传言，一是因为有政变风险，传武警有一批人策划〝起事〞。不过，自习近平掌管权力后，各种政变传闻一直不断，习的安保也在不断加强。
 <br/>
 <br/>
 另外一个传言是，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 使得中国经济急剧恶化，习近平希望在四中全会上，能拿出一个有说服力的计划。习要求主管经济、建设的部门，以及负责常委重新制定计划。习在相关会议上，对央行和证监会爆粗口，气氛非常紧张。
 <br/>
 <br/>
 据《自由亚洲》分析报导，认为中共权力斗争陷入胶着状态，是十九届四中全会推迟的原因之一。
 <br/>
 <br/>
 原清华大学政治系讲师吴强表示，习近平南下期间连一篇完整的讲话都没有，也没有进一步阐述改革开放，加上四中全会具体日期迟迟不宣布，令外界有理由怀疑，北京的权力斗争是否陷于胶着。
 <br/>
 <br/>
 吴强说，这显示习近平也好，或者中共目前也好，对未来的长期的改革开放缺乏明确规划，公众也好，党内的干部也好，现在把一切责任全部都归于习近平一个人。中共的干部现在是面临两个选择，向中共表忠诚或者向习近平个人表忠诚的问题。
 <br/>
 <br/>
 《自由亚洲》预测，四中全会在美国中期选举有了结果之后（也是10号前后）。北京期盼民主党重新掌控美国国会参、众两院，到时会针对贸易战最新形势，确定后续方案。
 <br/>
 <br/>
 原香港城市大学政治学教授郑宇硕认为，无论美国中期选举结果如何，对中共施压是美国民主共和两党的共识。美国的中期选举也不应是四中全会延期的原因。
 <br/>
 <br/>
 他相信四中全会延期真正原因，是中共领导层意见不一，就如何回应美国的要求仍然举棋不定。
 <br/>
 <br/>
 自中美贸易战开打以来，中共在内政、外交及军事方面均遭遇前所未有的挑战，政治上，多个信号显示习近平在中共内部遭遇挑战。
 <br/>
 <br/>
 经济上，货币超发、地方债务居高不下、P2P等影子银行暴雷频繁、股市、汇市下挫，外资纷纷撤离，特别是〝国进民退〞民营企业离场的传闻更使民营企业忧心忡忡。
 <br/>
 <br/>
 外交上，美国川普总统和彭斯副总统对中共进行了猛烈抨击，中美似已进入新冷战。军事上，美国发展印太战略中，美舰艇在南海游戈，几乎出现擦枪走火，台海形势趋于紧张。
 <br/>
 <br/>
 分析认为，在此种背景之下，习近平将在四中全会上，藉改革开放40周年的时机，就中美贸易战、民营企业市场定位及中共的经济、外交、军事形势进行阐述，并强调改革开放，以鼓舞和稳定民心以及政治上更强调
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集权.html" target="_blank">
  集权
 </a>
 将是四中全会的重心。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：范铭）
