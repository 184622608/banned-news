### 【禁闻】谷歌CEO承认蜻蜓计划 说辞遭外界批评
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】谷歌首席执行官周一在美国举行的一个会议上，承认公司正在进行〝蜻蜓〞计划，准备重新进入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国市场.html" target="_blank">
  中国市场
 </a>
 。他对此项计划的辩护説词遭到舆论质疑。
 <br/>
 <br/>
 据美国《The Verge》网站报导，10月15号，谷歌首席执行官桑达尔・皮查伊（Sundar Pichai），在《连线》杂志25周年峰会上，除了承认〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蜻蜓计划.html" target="_blank">
  蜻蜓计划
 </a>
 〞（Dragonfly）的存在，还为该项目作出辩护。
 <br/>
 <br/>
 皮查伊称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国市场.html" target="_blank">
  中国市场
 </a>
 对谷歌来说很重要，因为它规模巨大，而且很有可能成为全球最大、最赚钱的互联网用户群体。虽然还不知道谷歌最终能不能落户中国，但他声称，〝蜻蜓计划〞可以为99%的查询提供搜索结果。
 <br/>
 <br/>
 言下之意，即使经过中共审查，皮查伊认为受影响的搜寻结果将少于1%。
 <br/>
 <br/>
 不过，纽约大学访问学者滕彪批评，这个藉口不成立。
 <br/>
 <br/>
 纽约大学访问学者滕彪：〝这个藉口是不成立的。因为共产党不想让中国民众看到所有的东西，谷歌也不可能都让中国民众看到，它要配合中共的审查。至于（谷歌对于）在中国境内民众信息的传播，思想意识的提高，包括对民众通过网络进行连结，都没有任何帮助。〞
 <br/>
 <br/>
 有网友说：谷歌技术先进，体量巨大，一旦作恶，后果非常严重，受害的不止是美国人、中国人，全世界都会跟着受害！
 <br/>
 <br/>
 滕彪表示，中共利用经济优势向全球渗透，吸引西方的学术机构、大企业等趋之若鶩。谷歌在2010年离开中国时，也曾说过〝不作恶〞，不配合中共的网络审查。如今8年过去，谷歌为了回到中国市场，不惜一切配合中共开发审查版的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_搜索引擎.html" target="_blank">
  搜索引擎
 </a>
 ，非常令人遗憾。
 <br/>
 <br/>
 〝蜻蜓计划〞今年8月上旬被美国《拦截》网站首度踢爆，此前大部分谷歌员工并不知情。该计划引发外界及谷歌内部强烈反弹和谴责。
 <br/>
 <br/>
 超过1400名的谷歌员工为此联署抗议。另有多名员工提出辞呈，其中包括此前在谷歌公司机器智能部工作的高级科学家普尔森（Jack Poulson）。
 <br/>
 <br/>
 美国国会和白宫也表达了反对意见。
 <br/>
 <br/>
 16名美国国会议员曾向谷歌发信，质询公司是否为了重返中国市场，而准备配合中共政府信息过滤和监测用户的要求。
 <br/>
 <br/>
 美国副总统彭斯则公开要求谷歌，立即停止〝蜻蜓〞应用软件的开发，认为这将加强共产党的审查，并损害中国消费者隐私。
 <br/>
 <br/>
 大陆异议人士钱进：〝共产党它很坏，用防火墙屏蔽掉所有它们不想让老百姓知道的东西，如果谷歌现在进来就等于是共产党的帮凶。〞
 <br/>
 <br/>
 网友说：〝中共的贪婪和野心世人皆知，中共会仅仅满足审查版搜索吗？不会。中共，只要控制住14亿人，不愁金山银山。〞〝中共绑架十四亿人更加嚣张跋扈，肆无忌惮。〞
 <br/>
 <br/>
 滕彪：〝的确是这样，中共它控制或者说绑架了这么大的一个市场。虽然这些向中共低头、献媚的这些人、机构还是公司都是应该受到谴责，但是如果全世界都这么做的话，中共就会越发的强大、越发的蛮横，所以希望有一个真正公平的评价体系，要把中国（中共）定为极权、独裁，倡导自由、民主、宪政的国家必须要认识这一点。〞
 <br/>
 <br/>
 钱进：〝共产党它本来就是匪。中国老百姓因为都是（被）洗脑，尤其是网络，不让我们知道一些真实的东西。我们很欢迎谷歌能来到中国，但是一定要有它们的自由精神嘛！我们在网络上都说一句嘛：不自由毋宁死！〞
 <br/>
 <br/>
 另据《美国之音》报导，一名谷歌前工程师博亚帕蒂本周一透露，2010年谷歌中止在中国的搜索业务前，中共要求，谷歌中国版搜索引擎必须在当局下令的15分钟内删除一些新闻内容。他担心，如今谷歌为了重返中国，面临的审查要求可能会更加严格。
 <br/>
 <br/>
 采访/常春 编辑/王子琦 后制/钟元
