### 孟妻曝孟宏伟发出最后信息：一张刀子图
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】中共公安部副部长兼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警主席.html" target="_blank">
  国际刑警主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 返国后失踪，其妻在法国报警，但中共官方当时并未回应。日前其妻召开记者会，曝光孟失踪时发回的示警暗语，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 随后公布了孟落马的消息。孟宏伟落马，被指是北京肃清〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康余毒.html" target="_blank">
  周永康余毒
 </a>
 〞的继续。
 <br/>
 <br/>
 10月7日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 的妻子格蕾丝．孟在法国里昂一家酒店举行了新闻发布会。里昂是国际刑警组织的总部所在地，也是格蕾丝．孟的居住地。
 <br/>
 <br/>
 孟宏伟之妻的中文姓名不详。她举行记者会时，有两名负责保护她的法国警察陪同。
 <br/>
 <br/>
 据美联社报导，孟宏伟的妻子提到，孟宏伟在9月25日用国际刑警组织的手机，给她发短信〝等我电〞，4分钟后又发来一把利刀的照片，然后就失去联系。她表示，这是孟宏伟的示警信号。
 <br/>
 <center>
  <br/>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-8/p9069841a858904299.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-8/p9069841a858904299-ss.jpg"/>
   <br/>
   <font size="-1">
   </font>
  </a>
  <br/>
  <br/>
 </center>
 <br/>
 孟宏伟的妻子不让记者拍摄她的正面照，说是担心她和孩子的安全。
 <br/>
 <br/>
 她表示，召开记者会的目的是〝追求真相、正义和历史责任〞，〝为所有的妻子的丈夫和孩子的父亲不再消失〞。
 <br/>
 <br/>
 但据多家外媒报导，孟宏伟是中共前政法委书记周永康的旧部和亲信。中共政法系统一直是中共强力维稳、残酷镇压中国民众的工具。特别是在周永康时期，政法系统的一众江派马仔，手上沾满法轮功学员、维权异见人士和西藏新疆少数民族的鲜血，成为中共〝血债帮〞的代表。
 <br/>
 <br/>
 十几年来，在中共政法系统的手下，数以百万计法轮功学员的〝妻子的丈夫和孩子的父亲〞，因为〝追求真相、正义和历史责任〞而〝消失〞。而孟宏伟正是其中主要的一员。
 <br/>
 <br/>
 据海外中文媒体分析，中共公安部近期再次强调肃清〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康余毒.html" target="_blank">
  周永康余毒
 </a>
 〞，孟宏伟应该已有所警觉。孟妻在其失踪一周后向法国警方报案，使其成为国际事件，这应该是孟宏伟早就安排好的安全预案。
 <br/>
 <br/>
 其妻曝光其失踪时发回利刃短信息，恰好佐证了上述判断。
 <br/>
 <br/>
 10月7日，就在孟妻记者会发言后不久，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 网站上就发表一条〝一句话新闻〞称：〝公安部副部长孟宏伟涉嫌违法，目前正接受国家监委监察调查。〞
 <br/>
 <br/>
 通报中没有提到孟宏伟的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警主席.html" target="_blank">
  国际刑警主席
 </a>
 职务。
 <br/>
 <br/>
 有分析指，为防止孟宏伟效仿王立军寻求政治庇护，北京当局本拟低调处理孟宏伟，尽量降低影响。但孟妻将事件公开，可能让孟宏伟落马成为〝第二个王立军事件〞。
 <br/>
 <br/>
 2012年，活摘法轮功学员器官和重庆〝唱红打黑〞的打手、前重庆市公安局长王立军，因为害怕被薄熙来灭口，只身闯入美国驻成都大使馆寻求庇护，揭开了薄熙来、周永康团伙覆灭的序幕。外媒消息，王立军向美国使馆提供了薄周等人活摘器官、以及受江泽民指使计划政变的证据。
 <br/>
 <br/>
 对于北京为何不顾国际影响，突然拿下国际刑警主席孟宏伟，外界传出多种说法。
 <br/>
 <br/>
 有港媒引述消息称，孟宏伟落马是因为经济问题和在香港购置产业等。但该说法遭到普遍质疑。
 <br/>
 <br/>
 也有港媒指，孟宏伟案涉中南海政变阴谋，事态紧急。还有说法认为，孟宏伟担任国际刑警主席后，在国际上有不利北京的举动，涉嫌替周永康反扑。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
