### 北京长租公寓巨头爆仓 租客天天维权
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】去年以来，十家大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长租公寓.html" target="_blank">
  长租公寓
 </a>
 接连
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆仓.html" target="_blank">
  爆仓
 </a>
 。日前，北京长租公寓巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_昊园恒业.html" target="_blank">
  昊园恒业
 </a>
 房地产经纪有限公司（下称昊园恒业）也传出资金链断裂，至少数千名租客及房东受到波及，现在每天都有受害人到公司总部维权。
 <br/>
 <br/>
 受害租客透露，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_昊园恒业.html" target="_blank">
  昊园恒业
 </a>
 〞总部每天都有人去维权，追讨押金、租金或房租，租客一般被拖欠数千元到数万元不等，公司给的答覆是〝没钱还〞。
 <br/>
 <br/>
 受害租客安女士：〝我们交了钱，房东也收不到钱，房东就把我们很多租客赶出来了，很多人就特别激动，因为还得另租房子，就相当于现在交两份房租，报警也不管啊，我们每天都去，周六、周日去的人就特别多，三、四百人嘛，晚上也不回去，一直待那儿要钱，但是也要不回来，他们那个中介特别横，有的时候还会打人呢。〞
 <br/>
 <br/>
 据受害租客投诉，有人缴了押金及几个月的租金后被赶出房子，该公司还用霸王条款诱骗租客用网贷平台〝元宝e家〞提供的租金贷。
 <br/>
 <br/>
 安女士：〝而且好多人贷款，叫〝元宝e家〞，一贷就贷一年，有的刚签合同，没住两天就给赶出来了，这个贷款还必须得还，不还的话就会影响征信，我们觉得这就是金融诈骗，各个部门都认为这是债务问题、经济问题，没人去管。〞
 <br/>
 <br/>
 〝昊园恒业〞于2016年6月起正式运营，两年来扩张迅速，公开资料显示，该公司曾在2017年并购了近20家行业品牌，房屋管理数量增加一倍，突破10万间。今年8月13号，中共国家信息中心主办的〝信用中国〞发布的失信黑名单中，该公司赫然在列。8月22号，北京市住建委公布了23家违法违规问题中介，该公司名列榜首。
 <br/>
 <br/>
 受害租客胡先生：〝公司资金链也断裂了，还不上钱了，最近很多中介都出现这种情况。很多他们公司的业务员都已经离职了，好像说拖欠了他们很多个月的工资没有给，房东也说中介已经很久没给他们房租了，房东也要赶房子，上个月我被房东赶了，贷款软件中途退租的时候那软件还在绑。〞
 <br/>
 <br/>
 去年以来，大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长租公寓.html" target="_blank">
  长租公寓
 </a>
 已有十家先后
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆仓.html" target="_blank">
  爆仓
 </a>
 。中国房产公司〝我爱我家〞前副总裁胡景晖曾曝料说，〝长租公寓爆仓，一定比P2P爆雷更厉害。〞
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
 <br/>
