### 朱鎔基频繁亮相 官媒罕见报导
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】现年90岁的中共前总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 ，近一个月来，非常罕见地多次在官媒亮相，引发舆论关注。除朱鎔基之外，近来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李瑞环.html" target="_blank">
  李瑞环
 </a>
 等多名中共退休高层接连露面，外界揣测或有政治意涵。
 <br/>
 <br/>
 11月2日，香港中联办官网消息，中国国家主席习近平以及诸多现任或退休中共高层对查良镛（金庸）的去世表示哀悼。包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱鎔基.html" target="_blank">
  朱鎔基
 </a>
 、温家宝等人也对金庸去世表示哀悼。
 <br/>
 <br/>
 近一个月来，朱鎔基非常罕见地多次亮相官媒，引发舆论关注。
 <br/>
 <br/>
 10月23日是朱鎔基90岁生日，《人民日报》主管主办的环球人物杂志于10月19日发表题这《致敬朱鎔基》一文向其祝贺，〝健康长寿，乐享天伦。〞
 <br/>
 <br/>
 10月18日，朱鎔基拜访其老领导、中共人民大学前校长袁宝华。
 <br/>
 <br/>
 10月11日，清华大学经济管理学院顾问委员会于北京时间召开2018年会议，朱鎔基做为清大经管学院创院院长亮相，与历任院长合影留念。
 <br/>
 <br/>
 10月12日，朱鎔基及夫人劳安，会见参加清华大学经济管理学院顾问委员会的顾问委员。中共国家副主席王岐山、政治局委员、国务院副总理孙春兰，国务院原副总理刘延东、马凯等人一同参与了会见。
 <br/>
 <br/>
 中共十九大后，朱鎔基还曾高调露面，在海内外知名人士面前公开力挺习近平，同时也是十九大后，首名公开力挺习近平的中共退休常委。
 <br/>
 <br/>
 在习、江斗的过程中，朱鎔基多次在关键时期露面，力挺习近平的反腐〝打虎〞运动及〝习核心〞的地位。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 的对头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 、温家宝等中共高层，也多次在习、江斗的过程中露面，力挺习近平。
 <br/>
 <br/>
 而在近期，除了朱鎔基之外，胡锦涛、胡启立、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李瑞环.html" target="_blank">
  李瑞环
 </a>
 、吴邦国、吴官正、刘云山等中共退休常委接连以不同方式密集〝露面〞，引发舆论关注。
 <br/>
 <br/>
 外界分析，中共卸任领导人因身份特殊，鲜少出现在公众视野，对于他们的非官方公开活动，官媒普遍秉持不予报导的原则，因此他们露面通常藉助非官方渠道曝光。然而，这种有意安排的行踪，往往被外界认为另有政治意涵。
 <br/>
 <br/>
 由于目前时值
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 白热化，国内经济遭遇严重危机，各种不明朗因素浮现，中共习派、江派退休常委突然接连现身，或许与这波政坛波动有关。
 <br/>
 <br/>
 中共十九大前，习近平以反腐之名，拿下了包括徐才厚、令计划、周永康等众多江派要员，基本收拢了党政军大权。十九大曾被外界视为是习近平经过5年打〝虎〞铺排、彻底拿下江泽民等政敌的好机会。
 <br/>
 <br/>
 但在中共十九大后，北京当局打虎势头明显放缓。此后，中南海政坛鬼影幢幢，8月的北戴河会议前，甚至一度传出，中共老常委们要求开会解决中央主要领导问题。
 <br/>
 <br/>
 北戴河会议后，北京当局内忧未解，外患又至，中美贸易战使得中共政治、经济乱象频生，同时还召来国际社会的围堵。
 <br/>
 <br/>
 在外忧内患困境之下，习近平留下的政敌是否会藉机添乱？而习派、江派退休常委突然接连现身，是否与这波政坛波动有关？都是外界聚焦所在。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉)
