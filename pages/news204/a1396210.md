### 鲁炜和刘云山共享〝人奶宴〞曝光 习近平震怒
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月21日讯】原中宣部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲁炜.html" target="_blank">
  鲁炜
 </a>
 涉嫌受贿案近期在浙江
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公审.html" target="_blank">
  公审
 </a>
 。鲁炜与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 当年享受〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人奶宴.html" target="_blank">
  人奶宴
 </a>
 〞等丑闻再次被聚焦。据披露，除上述丑闻，鲁炜的负面新闻非常多，早已让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_震怒.html" target="_blank">
  震怒
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 曾怒批鲁炜〝以权谋色，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毫无廉耻.html" target="_blank">
  毫无廉耻
 </a>
 〞。
 <br/>
 <br/>
 10月19日，浙江宁波法院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公审.html" target="_blank">
  公审
 </a>
 原中宣部副部长、网信办主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲁炜.html" target="_blank">
  鲁炜
 </a>
 受贿一案。法院指控，鲁炜在过去15年中，在宣传系统内利用职务之便，收受财物摺合人民币3,200多万元。
 <br/>
 <br/>
 中共最高检7月30日发布消息称，原中宣部副部长、网信办主任鲁炜，涉嫌受贿一案由浙江监察委调查终结，并移送宁波检察院审查起诉。
 <br/>
 <br/>
 鲁炜被视为大陆〝互联网大总管〞，曾在新华社工作长达20年，官至副社长，2011年起，调任北京市委常委兼宣传部长，北京市副市长。2013年改任网信办主任，后兼任中宣部副部长。2016年被免去其它职务，只保留中宣部副部长一职。
 <br/>
 <br/>
 2017年11月21日深夜，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 网站发布消息，鲁炜正在接受审查。
 <br/>
 <br/>
 2018年2月13日，中纪委官网通报，鲁炜因严重违纪被双开。其腐败问题归结为政治腐败、作风腐败、经济腐败、道德腐败。怒批鲁炜频繁出入私人会所，大搞特权，作风粗暴、专横跋扈、以权谋色、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毫无廉耻.html" target="_blank">
  毫无廉耻
 </a>
 。
 <br/>
 <br/>
 2月14日，有美媒援引消息称，鲁炜在北京任职时，长期在北京日报大厦设有吃喝使用的专门包间，在其落马前，曾有一批官员向中纪委投书，指鲁炜在北京任职时，与其发生性关系和被其强行猥亵与强奸的女人就有60多人且有名有姓。
 <br/>
 <br/>
 据披露，除了上述丑闻，鲁炜的负面新闻非常多，早已让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_震怒.html" target="_blank">
  震怒
 </a>
 。其中包括在新华社负责经营时涉嫌贪腐，乌镇互联网大会造假，压制习近平亲信徐麟等等。
 <br/>
 <br/>
 而鲁炜最大的丑闻是，他曾参加〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人奶宴.html" target="_blank">
  人奶宴
 </a>
 〞，以及〝反宪政〞、支持贪官搞谋杀等。
 <br/>
 <br/>
 2013年7月，新华社记者周方实名举报，称中宣部正部级官员曾受一名大老板宴请。席间，人奶是作为一道菜上来的，在每位食客旁都来了一位全裸的美艳少妇。请客者说：大家请随意，有些人按耐不住，带着自己的〝奶妈〞到里间〝潇洒〞去了。
 <br/>
 <br/>
 〝人奶〞是当天宴席上最贵的一道〝菜〞，每位标价5,000元。举报信还说，中共高官非首次出席同类宴会，而请客的大老板目前还在狱中。
 <br/>
 <br/>
 另外，当时有份参与〝人奶宴〞的宣传口高官，还指向长期掌控中宣部的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 。
 <br/>
 <br/>
 2014年9月15日，网上署名庄丰的《挺铁流！吁习近平尽快处理〝淫棍〞刘云山》一文，披露有人举报刘云山，曾在北京高级会所的富商宴会当众吃美女的人奶。
 <br/>
 <br/>
 据说以刘云山做后台的前央视主持人芮成钢已经交代，这种事情的确发生过多次。
 <br/>
 <br/>
 有港媒引述消息说，根据央视要员以及芮成钢和其他主持人的招供，央视早已经成为周永康所主管的政法高官和宣传部主管的后宫，尤其是主管宣传的刘某（刘云山）淫乐享受的淫窟，这些年来，刘某所玩弄的女性超过380余人。
 <br/>
 <br/>
