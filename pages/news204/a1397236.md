### 【禁闻】美参院司法委员会主席 支持川普对华强硬
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】美国国会参议院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法委员会.html" target="_blank">
  司法委员会
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_主席.html" target="_blank">
  主席
 </a>
 查克．格拉斯利（Sen.Chuck Grassley），日前撰文，支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府对中共所采取的强硬立场。值得注意的是，格拉斯利当年曾投票赞成中国加入世界贸易组织（WTO），如今他表示，中共长期不遵守入世承诺。美国必须对中共的谎言、偷窃等行为，进行强力回击。
 <br/>
 <br/>
 格拉斯利10月28号在美国《福克斯新闻网》发表文章，题目是《美国必须对华采取更强硬立场》。他说，美、中关系是21世纪最重要、最复杂的双边关系。但多年来，中共一直从事不公平的贸易行为。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统及其政府在美中贸易问题上采取了强硬立场，努力扭转不平等局面。对于这种努力，格拉斯利表示支持。
 <br/>
 <br/>
 值得留意的是，格拉斯利提到，他曾投票赞成中国加入世贸组织。但中共长期以来不遵守世贸规则。他说，中共开展并支持针对美国公司的网络入侵和其它形式的经济间谍活动。美国知识产权委员会估计，美国在软件和商业机密方面的损失，每年高达6千亿美元。
 <br/>
 <br/>
 格拉斯利认为，中共觊觎美国技术，不容小觑。今年早前他随同美国国会代表团访华，发现中国在第五代移动通信系统（5G）的竞争中已经领先美国。而且将近一半的中国大学生主修工科，美国的这一比例却只有6%。
 <br/>
 <br/>
 因此，格拉斯利特别强调，监于中共这个专制独裁政权，无时无刻都想要影响和控制其它国家，美国对此不能掉以轻心。
 <br/>
 <br/>
 旅美时事评论员郑浩昌认为，格拉斯利从当年投票赞成中国加入世贸组织，到如今态度转变，这实际体现了美国国会整体上对中共态度的转变。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝格拉斯利前几天还呼吁美国司法部，对指控（美国最高法院大法官）卡瓦诺性侵的其中一位妇女和她的律师启动刑事调查。从这个来看，他是川普阵营中比较主动出击的一位，加上资格非常老，37年的老牌参议员，估计他的频繁动作，对推进美国对北京持续保持强硬姿态会起到相当的促进作用。〞
 <br/>
 <br/>
 旅美时事评论员邢天行表示，格拉斯利当年投票支持中国入世，是因为期待中国能走上与共产主义极权国家不一样的道路，但事实却让他大失所望。
 <br/>
 <br/>
 旅美时事评论员邢天行：〝从他（格拉斯利）的讲话当中，可以看出来，美国以他为代表的共和党，对于中共的这些手法和本质是认识的非常清楚的。因为他也谈到过，现在中美之间的战争，不是简单的一个贸易战争，为宗教自由而战应该是美中关系的核心部分。我觉得这个发言是点到了根本问题。〞
 <br/>
 <br/>
 格拉斯利提到，川普总统和其他许多人正采取积极措施，纠正中共对美国所犯的错误，使美国重回公平的竞争环境；美国国会也正寻求通过立法和改革，使中共更难从事不公平和非法行为。这些都是向前迈出的重要步骤，但关键问题是美国不能放松。他认为，美国必须继续展示一个强而有力的统一战线，回击中共用令人震惊的谎言、偷窃和欺骗等手段，成为世界经济领袖的企图。
 <br/>
 <br/>
 邢天行表示，中共作为一个完全反神的专制极权政权，跟整个人类的普世价值是对抗的。而美国所坚持的，是以维护人类公平、正义的基点来作为出发点。两者在信仰上，有着本质不同。
 <br/>
 <br/>
 邢天行：〝（美国）它要维护一种，比方说，我们在经济贸易中的公正原则，还有，你要真诚，你不要欺骗，你不要盗窃，等等这些东西，而它要维护这个普世价值观，恰恰跟中共本性上的谎言、欺骗、掠夺，完全是对立的。〞
 <br/>
 <br/>
 邢天行说，正因为这一点，目前美国共和党和民主党的很多有识之士，都站在了同一阵线。他们都认识到，中共的存在，就是世界的一个问题。中共做事，是站在邪恶的基础上，而美国目前通过贸易战打击中共的不法行为，实质是站在正义的角度上去打击邪恶。
 <br/>
 <br/>
 采访/陈汉 编辑/王子琦 后制/钟元
