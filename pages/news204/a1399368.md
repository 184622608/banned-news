### 【禁闻】11月14日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月14日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 陕西1000多民办和代课教师省政府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上访.html" target="_blank">
  上访
 </a>
 <br/>
 <br/>
 11月13号，陕西省1000多名原民办和代课教师，再次集体到省信访局上访维权，要求落实中共的三个文件，解决他们的身份、待遇等问题。他们在现场唱歌、喊口号抗议。
 <br/>
 <br/>
 2. 广州伤残军人遭非法拘禁22天
 <br/>
 <br/>
 因为港珠澳大桥通车，习近平到场剪彩，珠海市金湾区三灶镇副书记10月20号带人到广州市花都区，暴力绑架伤残军人陈风强，并将他裸体关押在黑监狱长达22天，直到11月12号，才将他扔在雨中的公路上。
 <br/>
 <br/>
 3. 福建碳九泄露 传官方控制治疗人数
 <br/>
 <br/>
 福建泉港11月4号发生碳九泄漏，当地政府11月8号公布，只有52人受害就医，其中9人住院观察，到11号都没有新增住院人员。当地民众披露，就医人数没有增加是因为政府控制治疗人数，口头指示医院，不得任意批准病患住院。
 <br/>
 <br/>
 4. 上海
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 举牌：〝中国法治在哪里〞
 <br/>
 <br/>
 11月12号，一批维权人士上街举牌，抗议中共国务院法制办拖延1年多, 不给上海57位申请信息公开的人士作出最终裁决书面答覆。他们责问：〝中国法治在哪里？〞
</div>
