### 毛泽东秘书李锐病情恶化 揭中共丑闻著作被封杀
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】毛泽东的前秘书
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李锐.html" target="_blank">
  李锐
 </a>
 女儿李南央，日前向朋友表示，父亲病情恶化。李锐早前出版过多本揭露中共高层内幕的著作，他曾直言〝毛泽东发动的文革运动是邪教〞，但他的书被中共列为禁书，遭到封杀。
 <br/>
 <br/>
 据多家港媒报导，李南央向朋友表示，父亲近日病情恶化，可能活不到自己（李南央）案件开庭的一天，但这并不重要，〝正是毫无希望，才充满希望〞。就像父亲看不到奋斗一生的宪政在大陆开展一样，看不到不等于宪政不会到来。
 <br/>
 <br/>
 2013年10月29日，李南央一家人从香港至北京机场入境，她先生和女婿（美国人）的行李在入关后被海关人员要求开箱检查，包括53本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李锐.html" target="_blank">
  李锐
 </a>
 在香港出版的《李锐口述往事》和一些刊物等被作为禁书扣留。经沟通无果后，李南央遂委托律师状告北京机场海关侵犯出版和言论自由，但法院至今未开庭处理。
 <br/>
 <br/>
 《李锐口述往事》一书记述了李锐的一生经历，其中，他在延安整风运动被关押时期的经历，在书中占了很多篇幅。
 <br/>
 <br/>
 李锐，祖籍湖南平江，他于1959年被下放到北大荒农场，1967至1975年，他在北京秦城监狱关押8年，直到1979年1月恢复工作。
 <br/>
 <br/>
 他曾任中共中央委员，中共组织部副部长，水电部副部长，毛泽东兼职秘书。他被称为开明派元老，晚年以敢言著称，长年呼吁民主宪政。
 <br/>
 <br/>
 李锐还是大陆〝三峡工程〞主要的反对派代表人物，曾多次上书中共高层，将来三峡工程完工，库尾洪水位剧升，必然要为重庆〝准备后事〞。
 <br/>
 <br/>
 2006年，李锐曾致函时任中共总书记胡锦涛，希望看到中共停止〝以党代法〞，以重建信誉，他说自己的临终愿望是言论自由。
 <br/>
 <br/>
 李锐在出席中共十八大时，在书面发言中猛烈抨击中共前党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 ，称江在上海的别墅非常奢侈，并要求〝结束中共老人干政〞。
 <br/>
 <br/>
 李锐出版了多本著作，揭露许多中共高层黑幕。他曾直言〝毛泽东发动的文革运动是邪教〞，并揭露毛泽东强奸杨开慧堂妹的丑闻。
 <br/>
 <br/>
 李锐的书被列为禁书后，他曾公开表示，〝那些不喜欢我的上面的人可以封杀我的书籍，但是不能封杀我的头脑。只要我活一天，就会继续思考一天。〞
 <br/>
 <br/>
 近期，已101岁高龄的李锐，多次传出病情恶化的消息。11月22日，原新华社记者高瑜在推特援引李南央的消息称，李锐日前被一位患感冒的探视者感染，持续发烧，病情恶化，经抢救脱离危险，但现在已经离不开呼吸机的帮助。
 <br/>
 <br/>
 此前，李锐曾对家人叮嘱，〝死后不开追悼会，不覆盖党旗，不进八宝山。〞
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
