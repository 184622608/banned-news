### 【微视频】常委频调研 中共要求经济学跟党走
------------------------

<div class="wysiwyg">
 【新唐人2018年11月26日讯】2018年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的经济形势不好，中共七常委频繁各地调研。习近平9、10月分别前往东北和广东调研，李克强6月湖南、7月西藏考察，韩正5月广东、11月北京调研，栗战书7月底在福建调研，汪洋8月在西藏调研。不是常委的副主席王岐山也没有闲着，9月前往江苏、上海，11月中旬前往湖北。
 <br/>
 <br/>
 换句话说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 常委们除了管纪检的赵乐际和管意识形态的王沪宁，剩下的常委和要员全都开始研究经济问题了，除了常委开会的时候全部下去研究了。这种现象说明了两个问题，第一是经济下滑严重了，下面的消息也信不过了；第二是政治局常委们在面对美国贸易制裁的情况下分裂严重，大家都想找支持自己观点的证据，甚至是四中全会也迟迟没有召开。
 <br/>
 <br/>
 咱们首先说经济下滑严重，这个都不用下去调研，官方数据也都说明问题了。新浪微博上已经又针对10月份经济数据的分析了，手机、汽车、二手楼、集成电路进口等国民经济中最重量级的行业普遍出现了双位数的下滑。同时，中共的全国财政收入在10月份首次出现同比下降，地方政府已经还不上债务了，收入再减少，这就等于宣布中共政权可以破产了。
 <br/>
 <br/>
 这些常委们调研了半天，也没有什么好招数。这个原因很简单，经济就是那么一块大饼，共产党和政府吃的太多，老百姓就得饿着，咱们老百姓手里没钱肯定不敢消费，经济自然不好。真正的好招数就是共产党下台，财政负担就减轻了，政府就有减税和增加福利的钱了，咱们老百姓就敢消费了，经济自然就好了，人家美国总统川普就是这么做的。共产党不下台，这么大个包袱，咱们中国人背不起，不要说中国人了，人家委内瑞拉人有石油都养不起社会主义。
 <br/>
 <br/>
 中共没有好招数但是损招是不少的，这就是维稳了。P2P泡沫破了之后，中共不是追查跑路的老板们，反而是控制P2P访民，这就是维稳政策。中共下一个维稳对象是谁呢？那就是各地的房主们。杭州萧山区已经成立房地产维稳小组，据说是要把房地产危机扼杀在基层，中共人民大学自己的报告都说，中国居民财富已经被房地产掏空了。中共下一步的房地产维稳就是怕房主们找中共算账。
 <br/>
 <br/>
 除了维稳之外，中共也加强了骗的手段。11月初，中共证监会召集国内学者和分析师开会，要求他们做研究也要跟〝党〞走，证券基金行业首席经济学家必须签署自律书，控制研究内容，这是明着要求投资分析报告造假呀。不过这里也透露了商机呀，很多海外敢于讲真话的媒体可以收集数据建立智囊库为海外投资人提供真实的投资报告，这就是赚钱的机会。当然这对媒体的专业、科技要求都是高的，希望有骨气的媒体能拓展新的商业领域。
</div>
