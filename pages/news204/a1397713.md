### 【禁闻】经济下滑 中共重提〝六稳〞于事无补
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 当局开会，罕见承认经济下行压力增大，并重申〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_六稳.html" target="_blank">
  六稳
 </a>
 〞，企图安抚民心。外界认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 急速下滑与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 有直接关系，如果明年1月1号美国将2000亿关税税率提高到25%，情况还会更糟。
 <br/>
 <br/>
 10月31号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 国家统计局公布的数据显示，10月份，中国制造业指数PMI和非制造业指数分别为50.2和53.9，环比双双下降。作为国家宏观经济先行指标的PMI，已经连续27个月高于临界点，创下2016年7月以来新低。
 <br/>
 <br/>
 10月份的数据显示国内生产和需求双双放缓。
 <br/>
 <br/>
 同时，反映外部需求的出口订单也在萎缩，10月新的出口订单指数为46.9%，低于上月1.1个百分点。
 <br/>
 <br/>
 此外，最近公布的中国三季度GDP同比增速降至6.5%，创下2009年一季度以来的最低。
 <br/>
 <br/>
 大陆时政评论人士田奇庄：〝国际国内的压力同时加大使中共没法承受了，国际上以美国为首的要求中共遵守世贸规则，同时加大了制裁的力度，从国内来看，由于中共没有一个明确的对美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的总体规划，大伙对前景不知道作何判断，很多商人不敢投资了，很多企业不敢往下进行了，导致很多后续的事情难以为继。〞
 <br/>
 <br/>
 10月31号，中共中央政治局开会罕见承认，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 下行压力有所加大，部分企业经营困难较多，长期积累的风险隐患有所暴露。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝这个时候也导致了中国社会各阶层的不满，包括中共高层的不满和内斗加剧，这个时候对他们来说，政治经济的压力同时在聚集，实际是一种内外交困的现象。〞
 <br/>
 <br/>
 会议称，当前经济形势是长期和短期、内部和外部等因素共同作用的结果。
 <br/>
 <br/>
 田奇庄：〝因为中国在此之前其实这个经济存在很多问题，一个是房价过高，另外一个就是国内消费不足，光靠出口外贸维持，第三个就是城乡差距进一步拉大，最重要的就是中国需要政治体制改革，需要压缩政府的人员、机构，不但没有压缩而且膨胀，维稳经费开支越来越大，自己本身积累的矛盾解决不了，再加上外部突然又来这么一下子，所以导致许多矛盾在这个集点上就爆发出来了。〞
 <br/>
 <br/>
 田奇庄表示，中共走入困境，关键在于当局不愿意放弃党说了算，不让司法独立，而是在幻想出现恐怖事件改变局势，同时期待美国的中期选举把川普选下去，但是美国两党在解决中共不公平贸易上已经高度一致，等到明年1月2000亿关税的税率提高到25%，中国经济还会更糟。
 <br/>
 <br/>
 政治局称，将继续专注于所谓解决民营企业、中小企业发展中遇到的困难；同时重申稳就业、稳金融、稳外贸等6稳。
 <br/>
 <br/>
 田奇庄认为，事实比口号更能说明问题，日前中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国企.html" target="_blank">
  国企
 </a>
 一汽一次性得到一万亿贷款授信，而他的一个民营企业家朋友，公司资产负债率才20%，可是银行就是不给它贷款。
 <br/>
 <br/>
 实际上，近来大陆打压民营企业，国进民退是不争的事实。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝其实中共从来就没有改变它的流氓本性，在出现经济危机的时候，一定会向民营企业家开刀，实际中共已经在做了，但是这个在当今的中国社会也引起了巨大的反弹，看到风声不妙，反抗的声音太强烈，又开始装模作样。〞
 <br/>
 <br/>
 至于6稳中的所谓坚持有效利用外资，维护在华外企的合法权益。外界认为也没有可能，因为中美贸易战已经导致大批外资撤离。
 <br/>
 <br/>
 采访编辑/刘惠  后制/周天
