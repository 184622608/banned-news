### 四川高颜值美女嫌犯自首 〝酒托〞诈骗内幕曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】四川省绵阳市警方日前公开7名在逃的犯罪嫌疑人照片，其中一名叫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卿晨璟靓.html" target="_blank">
  卿晨璟靓
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女嫌犯.html" target="_blank">
  女嫌犯
 </a>
 引发话题。陆媒称，该女嫌犯目前已经向警方自首。有网友说，卿晨璟靓因长相甜美读书时被评为校花，没想到变成通缉犯。
 <br/>
 <br/>
 日前，四川省绵阳市警方在网路上公开了一个〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒托.html" target="_blank">
  酒托
 </a>
 〞犯罪集团，2男5女在逃犯罪嫌疑人的照片，希望民众提供线索，将对举报人奖励1000元至3000元人民币，并将对报案公民个人信息进行保密。
 <br/>
 <br/>
 通告一出，没想到其中一名叫做〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卿晨璟靓.html" target="_blank">
  卿晨璟靓
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女嫌犯.html" target="_blank">
  女嫌犯
 </a>
 ，因长相相当清秀，引起网友关注。不少网友翻出她的私照，发现她化妆后更是亮丽动人，纷纷留言称〝怎么会做贼呢，可惜了〞！
 <br/>
 <br/>
 据陆媒28日报导，卿晨璟靓目前已向警方自首。
 <br/>
 <br/>
 网友爆料说，卿晨璟靓初中就读四川广元市大东英才学校，因为长相甜美曾被评为班花，很受男生欢迎，但因学习成绩差，被学校劝退后，转入绵阳市一所中学。高中阶段辍学，进入社会工作，没想到变成通缉犯。
 <br/>
 <br/>
 《新京报》曾爆料，〝卿晨璟靓〞被现男友带去当〝酒托〞，遭通缉后一度失去联系。
 <br/>
 <br/>
 与其一起在某上市公司共事过的一位同事说，〝卿晨璟靓〞是一个性格合群，外向、爱美的小姑娘，〝共事的时间不多，就个把月，她1999年出生，在我们几个年龄稍长的同事面前，感觉她就是一个不懂事的小孩子。〞
 <br/>
 <br/>
 说起〝卿晨璟靓〞做〝酒托〞的事，该同事也认为，和她男友有关，〝大概在6、7月份，辞职前给我们说，要跟着男友去外地，这才几个月时间，人生就发生了这样的变化。〞
 <br/>
 <br/>
 绵阳警方介绍，这个〝酒托〞团伙从今年4月以来，先后在绵阳多个酒吧茶楼作案。目前，到案的犯罪嫌疑人已被检察院机关批准逮捕，正全力追捕在逃人员。
 <br/>
 <br/>
 据称，酒托集团作案手法是，从交友程式中找网友下手，邀请他们到餐厅或酒吧约会，点选价钱不合理的酒菜，再要求他们结账，藉此诈取金钱。
 <br/>
 <br/>
 有网友为一顿晚饭的费用支付了数万元人民币。部份酒托集团更会使用暴力，去逼迫网友为高昂的账单埋单。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
