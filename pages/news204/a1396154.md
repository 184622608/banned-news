### 继抢棺事件后 江西强拆老子像再掀风波
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】江西省继抢棺事件引发舆论抨击后，近日，江西吉安市安福县武功山境内的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_羊狮慕景区.html" target="_blank">
  羊狮慕景区
 </a>
 ，一座巨型老子石刻雕像，被要求拆除整改，再掀风波。
 <br/>
 <br/>
 位于江西省吉安市安福县武功山境内的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_羊狮慕景区.html" target="_blank">
  羊狮慕景区
 </a>
 内，在天然花岗岩上，雕刻了一尊老子石像，高23米，宽16米，与山体浑然一体。
 <br/>
 <br/>
 根据微博上流传一份《安福县民族宗教事务局责令改正通知书》的文件，称这尊老子像违反了《宗教事务条例》第30条，〝宗教团体、寺观教堂以外的组织以及个人不得修建大型露天宗教造像〞的规定，限期在10月16日前拆除。
 <br/>
 <br/>
 据了解，羊狮慕景区的微信公众号也公布了这一消息，但随后就将该文章删除。
 <br/>
 <br/>
 江西省民族宗教委员会的工作人员回应称，安福县民族宗教事务局确实下发了整改通知，网传通知属实，目前仍在对该事件进行调查。
 <br/>
 <br/>
 江西省羊狮慕景区在10月18日表示，雕像目前暂未拆除。
 <br/>
 <br/>
 但事件仍引发中国网友抨击：〝拆老子像，那是不是敦煌莫高窟、龙门石窟、云岗石窟也得拆，孔庙也得拆？这怎么愈整愈像塔利班了〞、〝江西怎么尽干伤天害理的事，抢棺事件刚刚平息，现在又要拆老子像？明天是不是要拆黄帝像？〞
 <br/>
 <br/>
 据羊狮慕景区官网介绍，羊狮慕景区与江西省境内第一高峰武功山金顶一脉相承。因景区常年云雾蒸腾，常现〝羊〞、〝狮〞追逐嬉戏于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山间.html" target="_blank">
  山间
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_神奇.html" target="_blank">
  神奇
 </a>
 气象
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_景观.html" target="_blank">
  景观
 </a>
 ，令人思慕，而得名〝羊狮慕〞。景区以花岗岩峰林地貌闻名。
 <br/>
 <br/>
 羊狮慕景区还以〝老子〞哲学为主旨，为纪念老子，2015年邀请福建省雕刻艺术家在景区的天然花岗岩上雕刻了这尊
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_老子雕像.html" target="_blank">
  老子雕像
 </a>
 ，并无宗教意义。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
</div>
