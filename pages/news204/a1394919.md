### 河南再现小官巨贪 女科员坐拥53套房2亿现金
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】近年来，大陆频曝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小官巨贪.html" target="_blank">
  小官巨贪
 </a>
 案。河南省39岁的基层女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科员.html" target="_blank">
  科员
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李娟.html" target="_blank">
  李娟
 </a>
 ，因为坐拥超过2.3亿元（人民币，下同）现金与53户房产，但无法说明来源，日前被河南安阳市检方提起公诉。
 <br/>
 <br/>
 陆媒9日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李娟.html" target="_blank">
  李娟
 </a>
 落马前，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新乡市.html" target="_blank">
  新乡市
 </a>
 委组织部电教中心主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_科员.html" target="_blank">
  科员
 </a>
 ，属于中共基层的正科级官员。2011年6月，她曾出任辉县市纪委第三纪检监察室主任。
 <br/>
 <br/>
 根据检察院案件信息公开网9月下旬发布的一份〝起诉书〞显示，李娟自1996年10月担任公务员至2015年8月，总计薪资才36万余元。
 <br/>
 <br/>
 检察机关查明，李娟名下资产，包括：人民币约2.014亿元、港币54万、美元29.98万、韩元425万、房产53套（价值约3290万元）、车辆5辆（其中1辆调取不到票据，剩余4辆价值216.84万元）。
 <br/>
 <br/>
 能够认定上述事实的证据，包括李娟在32个银行的交易单、房产手续、车辆手续，证人崔某某等人的证言，李娟的供述和辩解等。
 <br/>
 <br/>
 检察机关认为，李娟身为国家工作人员，其财产明显超过合法收入，差额特别巨大，且不能说明来源，应当以巨额财产来源不明罪追究其刑事责任。
 <br/>
 <br/>
 根据报导，李娟于2015年7月9日被捕后，已于2016年7月被起诉。
 <br/>
 <br/>
 近年来，李娟工作过的河南辉县市，也有多名官员被查。其中包括早年担任过辉县市委书记的崔学勇，崔于2015年1月在新乡副市长任内落马后，网络传出他不但贪污上亿元，李娟也是他的情妇之一。
 <br/>
 <br/>
 中共官场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小官巨贪.html" target="_blank">
  小官巨贪
 </a>
 的案例，近年被频频曝光。如：
 <br/>
 <br/>
 北京昌平区一〝村官〞涉案金额达8亿多元人民币；西安市一居委会主任涉案总金额达1.2亿元，单笔受贿就达5,000万元人民币；安徽省淮北市烈山区烈山镇烈山社区原党委书记刘大伟，涉案金额逾1.5亿元。
 <br/>
 <br/>
 河北秦皇岛北戴河区供水总公司经理马超群家中，搜出1.2亿现金、68套房产、37公斤黄金；黑龙江龙煤集团物资供应分公司前副总经理于铁义，受贿超3亿；中共国家能源局煤炭司副司长魏鹏远家中，搜出2亿现金烧坏4台点钞机。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
