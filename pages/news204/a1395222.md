### 郭文贵公司遭罚600亿 案涉前国安部高官
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】逃亡美国的中国商人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭文贵.html" target="_blank">
  郭文贵
 </a>
 旗下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京政泉控股有限公司.html" target="_blank">
  北京政泉控股有限公司
 </a>
 ，被中共大连法院以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强逼交易.html" target="_blank">
  强逼交易
 </a>
 、挪用资金罪，判
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_罚600亿.html" target="_blank">
  罚600亿
 </a>
 人民币。
 <br/>
 <br/>
 10月12日，中共辽宁省大连市中级法院官网发布相关判决书公告。公告称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭文贵.html" target="_blank">
  郭文贵
 </a>
 在2008年至2014年间，为进入金融证券领域，以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京政泉控股有限公司.html" target="_blank">
  北京政泉控股有限公司
 </a>
 的名义，收购中国民族证券有限责任公司的股权，并实现控股。
 <br/>
 <br/>
 公告中还称，在收购过程中，郭文贵指使政泉公司前投资顾问郭汉桥、民族证券前董事长赵大建具体负责收购事宜。郭又与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马建.html" target="_blank">
  马建
 </a>
 共谋，由马建以中共国家安全部发函或派员的方式进行干预，向有关单位和个人施压，威胁、排挤竞争对手，最终使政泉公司实现控股民族证券的目的。
 <br/>
 <br/>
 2015年1月，马建涉嫌犯罪问题落马。2016年12月，马建被双开，移送司法机关处理。
 <br/>
 <br/>
 马建是江西人，与曾庆红是〝老乡〞，受曾庆红提拔、推荐，于2006年8月成为国安部副部长。马建曾帮助周永康建立针对北京高层的〝秘密资料库〞，在令计划与周永康结盟后，马建又涉周永康、令计划的阴谋，该资料库被周、令双方共同充实、利用，为江派打击政敌。
 <br/>
 <br/>
 马建被抓后，中共官媒《环球人物》杂志于2015年1月16日曾发文称，马建落马或与北大方正集团前总裁李友被查有关；与令计划有关；与周永康有关；参与窃听领导人谈话等非法活动。
 <br/>
 <br/>
 同年3月，大陆多家媒体报导，马建因与郭文贵有利益勾结而犯事。郭文贵在短短20年间获得了庞大资产，2014年的胡润百富中国富豪榜中他以155亿资产，排名第74位。有传闻称，郭文贵所以能快速发迹，是因靠上了中共政坛中一些重要人物。郭文贵被指曾为中共情报部门收集情报。在马健落马前，郭文贵于2013年12月23日出逃境外。
 <br/>
 <br/>
 不过，外界观察发现，本次大连中级法院的判决书中，虽多次提及郭文贵之名和其罪行，却并未对他个人追究刑事责任；相反他手下的顾问与高层，分别遭到3年以下有期徒刑，以及人民币十万元以上不等的罚金。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：戴明)
