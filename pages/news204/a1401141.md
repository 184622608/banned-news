### 709案家属李文足获瑞典颁发人权奖
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709案.html" target="_blank">
  709案
 </a>
 当事人、维权律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 至今仍被关押，他的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 三年多来，一直四处奔走，呼吁中共当局释放丈夫。11月27号，李文足获得〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瑞典人权奖.html" target="_blank">
  瑞典人权奖
 </a>
 〞 (Edelstam Prize)。评审委员会表示，奖项是为了表扬她捍卫人权、以及对抗当局打压展现出的无比勇气。
 <br/>
 <br/>
 11月27号，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_瑞典人权奖.html" target="_blank">
  瑞典人权奖
 </a>
 〞颁奖礼于在瑞典首都斯德哥尔摩举行，由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 被限制出境，流亡美国的维权律师陈光诚的妻子袁伟静代她到场领奖。
 <br/>
 <br/>
 28号，李文足通过新唐人发表她的得奖感言。
 <br/>
 <br/>
 维权律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 妻子李文足：〝其实当我在得知我奖的时候，我心里是很不安的，因为我觉得在中国，包括在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709案.html" target="_blank">
  709案
 </a>
 这个案件上，所付出还有努力的这些人，很多很多，他们所出的这成果，或者做的成就，远远超过我，比我做的好的人要多的多，那我觉得这个奖对我来说，我受之有愧的。〞
 <br/>
 <br/>
 评审委员会主席Caroline Edelstam 在公告中指出：中国依然是一党专制的极权国家，它在各方面系统地限制基本人权，包括言论，结社和集会自由，以及使用强迫失踪的手段。李文足及其他709案律师的妻子，不仅致力拯救自己的丈夫，同时用自己的方式反抗当局的暴政，包括通过网络呼吁，多次尝试探监，以及走访检察机关及法院，展现出非凡的勇气。
 <br/>
 <br/>
 李文足：〝但同时呢，我非常感谢大家对我的认可和肯定，把这个奖颁给我，其实我觉得这个奖，不仅仅是我一个人的，我觉得对我们整个家属都是个鼓励。〞
 <br/>
 <br/>
 李文足传给获奖现场的视频也在网络流传。
 <br/>
 <br/>
 李文足：〝我也藉此机会，盼望大家能够持续关注王全璋的情况，唯愿王全璋和所有被关押的人，早日获得自由，跟家人和朋友团聚。〞
 <br/>
 <br/>
 王全璋曾经代理法轮功等敏感案件，在2015年的〝709大抓捕〞期间被抓，2017年2月被当局以涉嫌所谓〝颠覆国家政权罪〞起诉，关押在天津市第一看守所，家属先后聘请的7名律师都遭当局否决，家属和律师一直没能见到王全璋，案件至今没有审理。
 <br/>
 <br/>
 新唐人记者熊斌、柏妮采访报导
</div>
