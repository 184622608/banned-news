### 大陆股市狂跌 股民求美国使馆〝救灾〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】美国考虑将中共正式列为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货币操纵国.html" target="_blank">
  货币操纵国
 </a>
 〞的消息，令大陆股市反应激烈，狂跌不止。许多大陆股民涌入美国驻北京大使馆官方微博，请求美国〝救灾〞。网友称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美使馆微博.html" target="_blank">
  美使馆微博
 </a>
 已经成为大陆的一个言论飞地。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美使馆微博.html" target="_blank">
   美使馆微博
  </a>
  言论飞地
 </h4>
 <br/>
 10月11日，上证综合指数失守2600点大关，一日暴跌5.22%，创近4年新低；深证成份指数跌6.07%，沪深300指数跌4.8%。香港恒指亦跌3.5%，是8个月以来最大单日跌幅。
 <br/>
 <br/>
 当日收盘后，许多大陆股民涌入美国驻北京使馆官方微博，在最新一篇关于援助印尼地震和海啸的微博下，大陆网民纷纷留言说，中国股市也需要〝救灾〞。
 <br/>
 <br/>
 有网民问说，〝A股灾民呢？没人援助？〞〝中国股市比十次海啸加地震都惨，你们能给中国提供点人道主义援助不？〞
 <br/>
 <br/>
 近年来由于中共严控网路言论，中国社会发生重大事件，或者是热点事件后，大陆网民开始涌入美大使馆微博留言。美使馆微博已经成为中国大陆的一个言论飞地。
 <br/>
 <br/>
 假疫苗事件爆发后，不少大陆网友跑到美使馆微博下留言，〝救救中国的孩子吧。〞
 <br/>
 也有不少帖子呼吁美国把中共消灭：〝川普先生，请把这个世界上最大的组织清除吧！〞
 <br/>
 <br/>
 还有网友调侃，〝大使馆成了开封府〞，〝真正的言论自由的地方，美国领土〞，〝在这里体验一下自由〞。
 <br/>
 <br/>
 <h4>
  美国考虑定中共为〝汇率操纵国〞
 </h4>
 <br/>
 11日股市暴跌后，大陆网友纷纷涌入美大使馆微博留言，请求美国〝救灾〞。有分析认为，此次股市暴跌应是受美国考虑宣布中共为〝汇率操纵国〞困扰。
 <br/>
 <br/>
 《香港经济日报》报导，美国考虑下周宣布中共为〝汇率操纵国〞。正在印尼峇里岛出席国际会议的美国财长努钦（Steven Mnuchin），11日会晤了中共央行行长易纲。
 <br/>
 <br/>
 努钦会后在推特发文称，〝我们讨论了重要的经济问题。〞但未透露详情。
 <br/>
 <br/>
 易纲稍后又会见了纽约联储银行总裁威廉斯，就中美经济金融形势、货币政策等议题交换意见。
 <br/>
 <br/>
 <h4>
  美联储加息 中共央行降准
 </h4>
 <br/>
 另有分析认为，大陆股市暴跌与美联储加息也有关联，美联储今年已加息3次。受加息影响，10月10日，美国股市也大跌。
 <br/>
 <br/>
 对此，川普（特朗普）总统批评说，美联储持续加息做法有错，政策过紧，形容美联储已经疯了。
 <br/>
 <br/>
 美联储持续加息使美国国债收益率上升，也使得股票对投资者的吸引力下降，这是股市下跌的主要催化剂。
 <br/>
 <br/>
 《华尔街日报》引述法国巴黎资产管理（BNP Paribas Asset Management）高级经济学家佛利民（Steven Friedman）称，投资者越来越担心，美联储的加息路径将阻碍美国经济发展。
 <br/>
 <br/>
 值得一提的是，在美联储不断加息之际，10月7日，中共央行宣布年内第四次降准，下调部分金融机构人民币存款准备金率1个百分点。
 <br/>
 <br/>
 业界人士认为，降准会加大人民币贬值压力。8日，人民币在岸和离岸兑美元双双跌破6.93关口。近6个月以来，人民币兑美元已重挫9%，成为亚洲表现最差货币之一。
 <br/>
 <br/>
 一方面，美联储加息使美元升值；另一方面，中共央行降准导致人民币贬值。业界人士分析，这一增一减，会加剧冲减美国对华关税制裁的成效。
 <br/>
 <br/>
 一直以来，川普都指控中共操控人民币汇率，以逃避美国的关税制裁。他表示，人民币汇率〝像石头一样下坠〞，不能让他们这么做下去。
 <br/>
 <br/>
 日前，针对人民币持续贬值，美国考虑将中共正式列入〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_货币操纵国.html" target="_blank">
  货币操纵国
 </a>
 〞。这令中共非常紧张，中共外交部急急回应，称中方〝不会将贬值人民币作为贸易战工具〞。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
