### 港媒：元老给改革40周年泼冷水 习南下计划被打乱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在中美贸易战火升高之时罕见低调南巡，23日主持了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 通车仪式。港媒披露，习此次南下遭一班中共元老泼冷水，说40年的改革开放成果被败光了。而习原本9月南巡的计划也因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳.html" target="_blank">
  深圳
 </a>
 无〝钱进中央〞而被打乱了。
 <br/>
 <br/>
 香港苹果日报报导说，中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 趁南下视察期间，23日主持了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 的揭幕礼。不过，习近平此次南下及主持大桥揭幕礼的安排都是极为低调，与2012年中共十八大后，习首次以总书记的身份南巡，表现高调亲民大相迳庭。
 <br/>
 <br/>
 原因是习近平这次于中共十九大后的再次南下〝波折重重〞。
 <br/>
 <br/>
 今年是中国改革开放40周年，但对习近平来说却是诸事不顺的一年。外交上，美国对华的贸易战持续半年后仍不见收手，10月初副总统彭斯更出面严词抨击中共，中美迈入新冷战时代。此外因中共强制转让技术及盗窃知识产权等问题，还招致了欧盟国家的围堵。
 <br/>
 <br/>
 在国内，自贸易战开打以来，中共节节败退，并导致股汇市双跌，外资撤退，国内资金也在悄悄外逃避险，企业违约逼近空前高点，意识形态也在向左转。
 <br/>
 <br/>
 苹果报导说，在经济停滞、外交孤立的空前困境之下，习近平原本计划在下半年高规格纪念改革开放40周年，但遭一班中共元老泼冷水，〝有啥好搞，改革开放的成果都给习自己败光了！〞
 <br/>
 <br/>
 9月中多家媒体亦报导，习近平南下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳.html" target="_blank">
  深圳
 </a>
 等多地企业视察，并出席在深圳举办的改革开放40周年纪念大会，当时传出深圳安保已进入一级戒备。但根据官方报导显示，习近平9月25日到28日北上，罕见考察东北三省，并发表重要讲话，重提闭关锁国时期的〝自力更生〞。
 <br/>
 <br/>
 苹果引述消息说，习9月份南下深圳的消息，并非空穴来风，深圳当局当时已经做好迎接的准备，但因深圳无钱进贡中央，打乱习南下的计划。最终，习匆忙选在10月22日南下，让港珠澳大桥提前通车。
 <br/>
 <br/>
 港珠澳大桥23日举行通车仪式，习近平主持了这个被称为〝世纪工程〞的大桥揭幕礼之后，匆匆离开现场，英媒BBC消息称，习近平去了深圳前海经济区。
 <br/>
 <br/>
 苹果爆料说，习近平24日到深圳龙华区龙悦居参观，当地政府从外地调派数万警力，挨家挨户敲门登记，通知居民24日上午只许出不许入，留在家中就会有警察入屋〝陪聊天〞，防止有人闹事，整个屋苑等同戒严。
 <br/>
 <br/>
 消息人士透露，日本首相安倍晋三，将于10月25日访问中国，考虑到国内外的日程，预计25日之前，习近平将结束南下行程返回北京。
 <br/>
 <br/>
 广东一直被认为是改革开放的前沿。习近平此次在中美贸易战火升高之时考察广东，被认为是为了提振国内外对于中国经济和改革开放的信心。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：戴明）
