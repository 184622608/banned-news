### 吉林机械公司爆炸 烈火冲天致59死伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉林.html" target="_blank">
  吉林
 </a>
 省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东丰县.html" target="_blank">
  东丰县
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江城机械公司.html" target="_blank">
  江城机械公司
 </a>
 23日深夜发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸.html" target="_blank">
  爆炸
 </a>
 ，现场烈火冲天。当地官方微博发布通报称，截止到24日中午12时40分，爆炸事故共造成2人死亡，57人受伤。
 <br/>
 <br/>
 陆媒24日早上报导说，位于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吉林.html" target="_blank">
  吉林
 </a>
 省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东丰县.html" target="_blank">
  东丰县
 </a>
 三合乡永安村三组的吉林省
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江城机械公司.html" target="_blank">
  江城机械公司
 </a>
 ，23日23点40分左右，发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸.html" target="_blank">
  爆炸
 </a>
 并引起火灾，现场烈火冲天。截至24日早上4时40分，事故已造成2人死亡，24人受伤，受损房屋41户，具体情况在核实中，救援工作仍在进行中。
 <br/>
 <br/>
 随后，吉林辽源市东丰县在官方微博发布通报称，截止到24日中午12时40分，爆炸事故共造成2人死亡，57人受伤（3人转至长春接受治疗，其中1人危重）。排查出受损房屋370户，其中轻微受损282户，鉴定需大修73户，彻底倒塌15户。
 <br/>
 <br/>
 通报说，经过排查，此次事件排除暴恐行为。​​​​初步调查相信，爆炸是由易燃易爆物品引起，波及附近民居，具体肇事原因仍在调查。
 <br/>
 <br/>
