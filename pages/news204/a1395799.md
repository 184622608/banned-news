### 【禁闻】10月17日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年10月17日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 内蒙购牛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诈骗案.html" target="_blank">
  诈骗案
 </a>
 开庭 四十牧民旁听
 <br/>
 <br/>
 内蒙一家企业在政府支持下涉嫌诈骗250户农牧民购牛款一案，10月15号在呼和浩特市中级法院开庭。乌兰察布、巴彦淖尔等7个盟市的近40位农牧民，前往法院旁听。牧民们担心即使胜诉，也追不回2700万元的投资款。
 <br/>
 <br/>
 2. 甘肃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 瞿明学去世
 <br/>
 <br/>
 甘肃维权人士瞿明学10月16号因病去世，大陆维权界哀悼。瞿明学两年前因参与基层选举，帮助独立候选人而被甘肃省永靖县公安局以〝破坏选举罪〞刑事拘留。事件当年被列为中国十大人权案件之一。
 <br/>
 <br/>
 3. 刘正清遭当局调查 疑逼其退出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄琦案.html" target="_blank">
  黄琦案
 </a>
 <br/>
 <br/>
 大陆〝六四天网〞创办人黄琦的代理律师刘正清，10月17号收到广州市律师协会发出的立案调查通知，声称要调查他在7月底会见黄琦时传递香菸的事，黄琦母亲认为当局此举意在压迫刘正清退出黄琦案，然后再安排官派律师将黄琦定罪。
 <br/>
 <br/>
 4. 老政治犯为生存权抗争 诉状遭驳回
 <br/>
 <br/>
 大陆政治犯因被中共判刑导致工龄归零，到了退休年龄无法办理退休和医保，生存面临危机，最近一大批老政治犯，包括何德普、牟传珩、姜福祯等，集体起诉中共人力资源和社会保障部，为生存权抗争，但是诉状被北京高级法院驳回。
</div>
