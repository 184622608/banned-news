### 辽宁900斤大猪无人敢买 只因〝生不逢时〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月19日讯】近日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁丹东.html" target="_blank">
  辽宁丹东
 </a>
 一头900斤的大猪在社交网路爆红。然而，受正在中国境内蔓延的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟疫情.html" target="_blank">
  非洲猪瘟疫情
 </a>
 影响，这头猪中〝巨无霸〞至今无人敢买。养猪人为此担忧，〝卖不出去愁人啊！〞
 <br/>
 <br/>
 10月16日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁丹东.html" target="_blank">
  辽宁丹东
 </a>
 养猪人张先生喂养的一头大母猪，刷爆朋友圈。这头大猪重900斤，体长2.25米、腰围1.75米，站立高度1.10米，堪称猪中〝巨无霸〞。村里一位83岁的老人说，活了一辈子，第一次见到这么大的猪。
 <br/>
 <br/>
 可惜的是，这头大猪〝生不逢时〞，近期中国大陆正爆发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟疫情.html" target="_blank">
  非洲猪瘟疫情
 </a>
 ，辽宁多地也发现猪瘟疫情，导致这头大猪至今无人问津。养猪人张先生担忧的说，〝卖不出去愁人啊！〞
 <br/>
 <br/>
 自今年8月1日爆发首宗非洲猪瘟疫情以来，疫情已在全中国范围蔓延。迄今为止，官方通报的疫情已达29起，并有进一步失控的迹象。
 <br/>
 <br/>
 有大陆防疫专家日前披露，情况比想像的严重！北京当局已无力阻止非洲猪瘟疫情扩散，而掌握疫情变化的实验室和疫控人员，已被当局严管，禁止洩密。
 <br/>
 <br/>
 尽管中共官方一再宣称，〝猪肉可以放心吃。〞但大陆民众对此表示质疑，辽宁沈阳市的郭女士对自由亚洲透露，〝听到这消息很多人不敢吃猪肉了。现在猪瘟为什么发生这么强大，没有跟百姓公布原因。〞
 <br/>
 <br/>
 根据联合国粮农组织（FAO）与世界动物卫生组织（OIE）介绍，非洲猪瘟具有传染性，死亡率高达100%，没有商品化疫苗可用于预防，也没有有效的治疗方法。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
</div>
