### 【禁闻】西方15国驻华大使致信新疆官员问人权
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】
 <b>
  10月份 34名
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
   法轮功学员
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遭冤判.html" target="_blank">
   遭冤判
  </a>
 </b>
 <br/>
 <br/>
 中共对法轮功的迫害仍在进行，据〝法轮大法明慧网〞报导，今年10月份，又有34名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 被非法判刑，还有另外33名学员被非法庭审。
 <br/>
 <br/>
 报导说，非法判刑、庭审涉及16个省、自治区、直辖市的28个城市，其中非法判刑最严重的地区是天津，一个月内6人被判刑，其次是辽宁。
 <br/>
 <br/>
 〝明慧网〞说，法轮功学员都是修心向善的好人，中共表面声称〝依法治国〞、〝信仰自由〞，实际上却不断的制造冤案，迫害善良民众。
 <br/>
 <br/>
 <b>
  15国
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻华大使.html" target="_blank">
   驻华大使
  </a>
  致信新疆官员问人权
 </b>
 <br/>
 <br/>
 据路透社11月15号报导，15个西方国家的驻华大使日前联名起草了一封信函，要求中共新疆党委书记陈全国，对维吾尔族群的人权状况做出解释。
 <br/>
 <br/>
 在这封致陈全国的信中，大使们表达了对新疆再教育营的关切，希望与陈全国见面，了解当地情况，并讨论所关切的问题。
 <br/>
 <br/>
 报导说，这封信由加拿大驻华大使馆牵头，目前已经有英国、法国、德国、澳大利亚等15个国家的驻华大使参与联署。
 <br/>
 <br/>
 据说信件还会传给更多国家，希望更多的人签名支持。
 <br/>
 <br/>
 报导分析说，这次针对人权问题的罕见大规模联合行动显示，中共在新疆地区的作为引起了越来越多的反对。
 <br/>
 <br/>
 <b>
  北大成立两新机构 监控高知人群
 </b>
 <br/>
 <br/>
 一份北京大学成立两大监控机构的文件，日前在网上流传。
 <br/>
 <br/>
 网上图片显示，被曝光的文件为中共北京大学委员会签发。文件通知说，学校决定成立中共北大委员会巡查办公室和北大内部控制管理办公室。
 <br/>
 <br/>
 网上舆论认为，这是中共对大学生、教师等高知人群进行全面监控的标志。
 <br/>
 <br/>
 有北大学生则表示，新机构应该是针对校园内学生运动的。
 <br/>
 <br/>
 11月9号，北大校园刚刚发生了暴力绑架学生事件，校保卫部不加阻拦。因为这些学生都参与了声援深圳佳士工人的维权活动。
 <br/>
 <br/>
 编辑/周玉林
