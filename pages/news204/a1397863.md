### 【禁闻】川普预计美中不会在G20宣布达成贸易协议
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
   川普
  </a>
  预计美中不会在G20宣布达成协议
 </h4>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 11月1号与习近平通话之后，外界纷纷预测
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 可能缓解，双方可能在本月底的阿根廷G20峰会上达成协议。
 <br/>
 <br/>
 不过，美国《华盛顿时报》11月1号独家报导说，川普总统在前一天接受该报专访时说，尽管两国贸易问题上有进展，但他并不认为两国会在G20峰会期间宣布达成贸易协议。
 <br/>
 <br/>
 在此之前，彭博新闻社援引四位知情人士提供的消息报导说，川普已经指示美国主要官员开始起草美方可能提出的条款。
 <br/>
 <br/>
 而美国消费者新闻和商业频道记者杰弗斯随后则在其推特上说，白宫高级官员表示，跟中国的谈判〝还有很长的路要走〞。
 <br/>
 <br/>
 <h4>
  陆人权报告提交
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国.html" target="_blank">
   联合国
  </a>
  民：都是空话
 </h4>
 <br/>
 联合国人权理事会11月6号至9号将对中国的人权状况进行审查，中共声称高度重视此次审查工作，已向联合国提交了《国家人权报告》。但中国民间人权受害者却批评说，中国的人权和法律都是空话。
 <br/>
 <br/>
 据〝美国之音〞报导。〝709案〞受害律师王全璋的妻子李文足说，她家的经历就足以证明中国的人权状况。
 <br/>
 <br/>
 她说，她丈夫被抓，失踪了三年，生死不明。不让律师会见，完全剥夺了律师的会见权、辩护权。而她的家庭也因此遭受株连，孩子不能上学，她们在北京没有稳定的住所，还被警察监控、威胁、恐吓。
 <br/>
 <br/>
 她质问: 〝这有人权吗？〞
 <br/>
 <br/>
 李文足的丈夫王全璋，是中国知名维权律师，2015年7月在中共针对维权律师的大规模抓捕行动中被捕。
 <br/>
 <br/>
 李文足说，她从自己的亲身经历体会到，中国的人权和法律都是空喊口号。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_涿州公安局.html" target="_blank">
   涿州公安局
  </a>
  副局长恶行上街头海报
 </h4>
 <br/>
 河北省涿州市的街头10月底出现一些曝光涿州公安局副局长杨玉刚恶行的海报，引来市民观看。
 <br/>
 据〝法轮大法明慧网〞报导，杨玉刚是当地迫害法轮功的主要负责人，多年来对当地法轮功学员进行绑架、抄家、毒打、酷刑折磨、勒索钱财、非法关押、构陷等迫害。
 <br/>
 <br/>
 报导说，根据不完全统计，涿州地区的法轮功学员至少有6人被迫害致死，3人被关精神病院，16人次被判刑，103人次被劳教，98人次被关洗脑班，被勒索钱财超过160万元；而被骚扰绑架、毒打酷刑、被迫辞去工作、被迫流离失所的法轮功学员则无法统计。 而每迫害一个法轮功学员，杨玉刚都会得相应奖金。
 <br/>
 <br/>
 编辑/周玉林
