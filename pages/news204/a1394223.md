### 范冰冰因何得放生？港媒：习近平借力收缴曾庆红地盘
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】失踪近四个月的中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 日前〝露面〞，只被判处巨额罚金了事。北京对此案高举轻放，引起外界猜疑。港媒指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 不过借机收缴江派大佬
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾庆红.html" target="_blank">
  曾庆红
 </a>
 家族掌控的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_娱乐圈.html" target="_blank">
  娱乐圈
 </a>
 ，没必要置范于死地。
 <br/>
 <br/>
 香港《苹果日报》10月5日评论文章指，当年刘晓庆被判监是因为得罪江泽民，汤灿和芮成钢入狱是分别牵涉周永康和令计划案，都涉嫌在政治上有问题。这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 出事，也有些许这方面的因素。
 <br/>
 <br/>
 文章说，过去20多年，中国文艺、影视娱乐业基本掌握在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾庆红.html" target="_blank">
  曾庆红
 </a>
 家族手中，其弟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾庆淮.html" target="_blank">
  曾庆淮
 </a>
 是影视圈只手遮天的人物。很多官二代、军二代借曾家势力染指电视电影业，近年走红或经常亮相央视春晚的明星，多跟曾庆淮或宋祖英有关系。这个圈子里的明星，宋祖英、赵本山等人已经沉寂，而冯小刚、成龙等还很活跃。
 <br/>
 <br/>
 文章认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 要整治
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_娱乐圈.html" target="_blank">
  娱乐圈
 </a>
 ，收缴曾家控制权，同时打击对习不满的官二代、军二代，这只是时间问题。而范冰冰被曝出〝阴阳合同〞事件，正是一个很好的时机。
 <br/>
 <br/>
 不过，范冰冰和习近平本无私怨，又非圈子中核心人物，没必要一棍子打死。范本身是当红女星，失踪期间得到国际媒体关注，因此只是将她高高举起，就能起到足够的警告作用，令整个娱乐圈人心惶惶。如果一定要重判她，会对整个行业造成毁灭性打击，对经济不利。
 <br/>
 <br/>
 文章说，整治娱乐圈还有一个重要原因。中国经济每况愈下，地方政府债台高筑。当局缺钱〝发穷恶〞，逃税避税的富豪、企业家及影视红星，就成了开刀的对象。更何况近年来大量热钱涌入影视业，这里成了洗钱、避税天堂。
 <br/>
 <br/>
 实际上，早在范冰冰案开始发酵时，就有港媒消息指，北京当局对军方操控娱乐圈不满，或藉机进行整顿。崔永元爆料是趁机公报私仇。
 <br/>
 <br/>
 对于范冰冰免于刑责，有媒体指其符合中共法律，认为范冰冰的〝免死金牌〞，是2009年对《刑法》201条〝逃避缴纳税款罪〞的修改。不过，也有媒体引用空姐代购逃税8万被判三年一案指出，其背后关键原因不在于法律条款。
 <br/>
 <br/>
 新华社日前报导，范冰冰因〝阴阳合同〞行为涉及逃漏税，被处罚补缴税款及滞纳金、罚款共8亿8394.6万元，若在限期内缴清则可免于追究刑责。范冰冰随后也第一时间发出道歉声明，表示会努力补缴罚款。至此，范冰冰案基本尘埃落定。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
