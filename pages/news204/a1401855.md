### 【禁闻】党媒避提川习会重大让步 评：中共为自保
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】在阿根廷举行的美中首脑会谈〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞12月1号落幕，中方做出重大让步，但是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_党媒.html" target="_blank">
  党媒
 </a>
 对此却只字不提。有评论指出，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 形势非常严峻，中共决定让步，但它需要多一些时间进行面子工程，以免危及政权。
 <br/>
 <br/>
 12月1号晚间，美国总统川普和中国国家主席习近平，带着各自团队在晚宴桌上进行了贸易谈判。彭博社表示，美中双方都称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 非常成功，但白宫发言人和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 外交部长王毅却发布了内容上没有多少交集的声明。
 <br/>
 <br/>
 中共喉舌〝新华社〞称，两位领导人达成共识，双方都同意〝停止〞课征新
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，但是通稿闭口不提美方暂缓提高关税的前提条件，没有任何细节，也不提中共做出的一些重大让步。
 <br/>
 <br/>
 白宫则声明，双方同意立即开始有关结构性改革的谈判，这些结构性改革涉及强制技术转让、知识产权保护、非关税壁垒、网络入侵和网络盗窃、服务业和农业等。如果在未来90天内，双方不能在这些方面达成协议，10%的关税将会被提高至25%。
 <br/>
 <br/>
 旅美时事评论员蓝述表示，美方原本明年1月1号要对价值2000亿美元的中国商品加征25%关税，现在给90天期限，那就是延至3月1号，等于多给了中共60天的时间去考虑问题。
 <br/>
 <br/>
 旅美时事评论员蓝述：〝还是要加25%的税，如果中方不完全达到美方最后的要求的话。我们看到川普总统的决心是非常之大的，中共只不过是为自己争取了一点时间。〞
 <br/>
 <br/>
 声明还说，中共同意从美国购买数量庞大的农业、能源、工业和其它产品，此外，中共还同意将芬太尼列为〝管控物质〞，这意味着向美国出售芬太尼的人将受到中共的严厉处罚。
 <br/>
 <br/>
 白宫还说，习近平表示，如果达成协议，他将重新考虑高通公司和恩智浦半导体公司的合并问题。
 <br/>
 <br/>
 对于美中各自表述，蓝述表示，这说明双方并没有达成共识，而中共官媒故意美化这次会谈结果，也有两种可能。
 <br/>
 <br/>
 蓝述：〝北京决定争取一点时间，然后全面
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 开打，新华社明目张胆的撒谎，为北京多拖60天的时间，还有一种方式就是，北京已经做出让步的决定，只不过需要多两个月的时间去做一些面子工程。〞
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝经历了前几个回合的交锋，中共应该是已经非常清楚，川普是不好惹的，如果再玩花招，肯定没有好果子吃。所以它不会像以前那样无所顾忌，很可能还是会做出一定程度的让步。〞
 <br/>
 <br/>
 法国广播电台说，从声明看，白宫主人显然掌握着主动权，而且，川普半年来坚持要求中共做出〝结构性〞改变的内容没有丝毫改变，总之一句话，贸易战没有结束，说轻点，北京获得喘一口气的机会，说重点，北京被判处3个月缓刑，这是中共官媒不愿向自己人民如实汇报的。
 <br/>
 <br/>
 郑浩昌：〝眼下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 形势非常严峻，光是今年上半年就倒了500万家企业，北京是不想低头也得低头。但是光靠3个月的缓冲期，很难想像会出现真正突破性的进展。〞
 <br/>
 <br/>
 蓝述表示，即使双方未来达成共识，签订协议，美方也要谨慎，因为中共从来不信守承诺。
 <br/>
 <br/>
 蓝述：〝它很可能采取的一种办法就是，从技术上去钻空子，然后，在每一件具体的事情上死打烂缠。所以美方在3月份之前如果真的能够跟北京达成正式的协议，这个协议必须签的非常的细致，很多的细节要做出非常明确的规定。〞
 <br/>
 <br/>
 香港时事评论员刘锐绍则向《苹果日报》指出，美国已视中共为〝真敌人〞，而非假想敌，因此贸易战一定会持久，美国会居主导地位，而中共只能〝且战且退〞。
 <br/>
 <br/>
 采访/常春 编辑/陈洁 后制/葛雷
