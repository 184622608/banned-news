### 美媒分析：孟宏伟和王立军犯中共大忌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】国际刑警组织主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 落马事件，引爆海内外舆论关注。有美媒将孟宏伟案和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 案做比较，认为这两名中共〝老公安〞都有一个共同点：向西方国家求助，犯了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共大忌.html" target="_blank">
  中共大忌
 </a>
 。同时，这两案又有不同之处：王全家在中国境内，而孟一家已到海外，而且孟身兼国际组织的高级职务，北京要想处理他会更加投鼠忌器，分寸拿捏不易。
 <br/>
 <br/>
 <h4>
  〝老公安〞向外国求助
 </h4>
 <br/>
 美国之音10月11日刊文称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 这两个〝老公安〞都是在警察界工作多年，并进入高层后突然坠马的。
 <br/>
 <br/>
 孟宏伟是9月下旬奉召回国失踪的。其妻格瑞丝．孟（Grace Meng）10月5日在国际刑警组织总部法国里昂报警将事件曝光，引发国际舆论高度关注。
 <br/>
 <br/>
 6日，国际刑警组织要求北京说明情况。7日，格瑞丝举行记者会，数小时后，中纪委（北京时间7日晚）宣布孟宏伟落马，罪名是：涉嫌违法。国际刑警组织同时收到了孟的〝辞呈〞。
 <br/>
 <br/>
 而前重庆市公安局长王立军，于2012年2月6日突然进入了美国驻成都总领馆，要求政治庇护，酿成轰动一时的重大国际事件。王立军此举可谓创了记录，北京派公安部高官到成都美领馆要人，将王带回北京。
 <br/>
 <br/>
 旅美中国学者夏业良认为，王立军和孟宏伟都是出事后求助于西方。不过，王立军并没做足够准备，而孟宏伟则有准备。他还认为，孟宏伟妻子格瑞丝．孟的决定，应是事前同其丈夫商量好的。
 <br/>
 <br/>
 夏业良说，格瑞丝如此高调，就是等于〝给中共一个警告：如果你要迫害我的丈夫、置他于死地的话，我手中有重要的材料，可能会通过国际媒体曝光。〞
 <br/>
 <br/>
 他表示，当年令计划的弟弟令完成跑到美国，至今也没有爆料。所以现在的情况是，有真料的人不曝，爆的人往往没什么真料。〝我们现在倒是希望，如果中方不买她的账继续迫害其丈夫，她接下来有可能曝出猛料。〞
 <br/>
 <br/>
 文章称，中共高官向外国求助，特别是到外国驻华使领馆申请政治庇护，是犯了大忌，党必诛之。2012年9月，王立军被判处15年徒刑，罪名是徇私枉法、滥用职权、叛逃和受贿。
 <br/>
 <br/>
 10月8日凌晨，中共公安部部长赵克志宣布，孟宏伟涉嫌收受贿赂和违法以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 案被拿下。在中国大陆，被双规是一条通向被捕判刑、仕途到顶、政治名誉身败名裂的单行道，绝少有人能全身而退。
 <br/>
 <br/>
 <h4>
  孟宏伟与王立军的异同
 </h4>
 <br/>
 孟宏伟案在海外社交媒体推特上也成为热点，网友们纷纷将孟与王立军做比较。推友Lifetime视界说：孟没有王走得远，明知有麻烦还是返回中国，而王已经走进了美国领事馆，并递交相关材料。
 <br/>
 <br/>
 他还表示，孟虽然回国，却让妻子向国际舆论曝光，似乎又走得比王立军远。王立军得到轻判，除因揭发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 之外，中共对美国有承诺。而孟目前并无国际保护。
 <br/>
 <br/>
 也有网友说，两案最关键的一点就是，王与孟所处的地方不同：王全家在中国境内，孟一家已到海外，而且孟身兼国际组织的高级职务。北京要想处理孟会更加投鼠忌器，分寸拿捏不易。
 <br/>
 <br/>
 还有网友说，两案都有破釜沉舟的意图和现象，但还是有重大不同。孟宏伟本来可以留在法国申请庇护，他在中国的仕途已是穷途末路，回去只有死路一条。但尽管如此，他还是〝奉召〞回去了。而王立军则走进了美领馆。
 <br/>
 <br/>
 另有推特网友（anneroloy）曝料说：格瑞丝．孟在法国里昂住的高档住宅，面积270平米，每月房租法郎相当5万人民币。质疑孟宏伟在公安部门内为官数十载，肯定干了不少亏心事，贪了不少不义之财。而在六年多前，王立军出事时，也有诸多网友发出同样的声音。
 <br/>
 <br/>
 王立军事件引发中共高层大地震。因王立军揭发，时任中共政治局委员、原重庆市委书记薄熙来、时任政法委书记周永康随后相继落马。
 <br/>
 <br/>
 梳理孟宏伟的履历发现，孟宏伟在公安部一路得到提拔和重用的时候，正是周永康掌管公安部的时期。孟宏伟被提到了副部长的位置上，成为周永康副手。孟宏伟因此被指是周永康余党，外界认为，孟宏伟应对中共政法系统镇压法轮功、维权人士甚至活摘器官等罪行负有相应责任。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
