### 湖北军官贫病去世   91岁遗孀维权受阻
------------------------

<div class="wysiwyg">
 【新唐人2018年10月30日讯】湖北潜江市91岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡永兰.html" target="_blank">
  蔡永兰
 </a>
 ，经常在潜江市政府门口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 ，她的丈夫、儿子和孙子，一家三代五人都是军人，退役后都没有得到国家照顾。蔡永兰的丈夫最近在贫病交迫中去世。她在当地维权没人理会，想去北京上访，在火车站又遭到拦截。
 <br/>
 <br/>
 91岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡永兰.html" target="_blank">
  蔡永兰
 </a>
 是潜江市熊口镇莲市村一组的村民。她的丈夫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张鼎海.html" target="_blank">
  张鼎海
 </a>
 原来是上尉军官，参加过无数次大小战役，多次因立功而受奖。但因炮震导致双耳失聪，1959年因伤残转业回乡。张鼎海在文化大革命期间被关押，房子和农地陆续被充公，只能靠捡破烂和每月几十块钱的残疾军人抚恤金，养活妻子及6个孩子。张鼎海的儿子张振峰和三个孙子也先后参军。
 <br/>
 <br/>
 10月中旬，蔡永兰又到市政府门口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 ， 整整一星期都没人理会。
 <br/>
 <br/>
 现场视频:〝今天是2018年10月18号，开国军官残疾老兵张鼎海，这是***91岁的家属蔡永兰，在潜江市政府门口表达诉求，一个军人世家，一家三代五人参军，平均是18年，政府没有做相应的安置，目前的状况是没有农民身份、没有工人身份，也没有任何相应的待遇，请全国的有志之士关注这样一个军人之家的悲惨遭遇。〞
 <br/>
 <br/>
 据了解，张鼎海生前因为没钱治病，曾要求地方当局给予辅助，但遭到拒绝。潜江维权人士伍立娟表示，蔡永兰的儿子和孙子因为安置问题也一直在上访。
 <br/>
 <br/>
 潜江维权人士伍立娟:〝这老太太我认识，她老公啊、她儿子啊、她孙子都是当兵的，上访好多年了，十几年了嘛，经常到潜江市政府门口，摆的牌子啊、挂的牌子啊，告状的那些材料，上访维权啊。〞
 <br/>
 <br/>
 因潜江市政府没人理会蔡永兰的诉求，蔡永兰和儿子张振峰于是进京上访，但在武汉火车站遭到地方维稳人员阻拦。现场视频显示，蔡永兰气愤地跟维稳人员交涉，说地方当局忘恩负义，遗弃为共产党流血、流汗的老兵，骂他们忘本。从24号到26号3天，母子俩都被拦截无法上车，最后不得不把火车票退掉。
 <br/>
 <br/>
 新唐人记者熊斌、周天采访报导
</div>
