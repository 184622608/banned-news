### 中国男星蒋劲夫施暴日本女友 恐怖细节曝光（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】27岁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国男星.html" target="_blank">
  中国男星
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蒋劲夫.html" target="_blank">
  蒋劲夫
 </a>
 ，施暴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本女友.html" target="_blank">
  日本女友
 </a>
 事件，被日本多家电视台追踪报导。近日，其女友中浦悠花现身，讲述了受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_家暴.html" target="_blank">
  家暴
 </a>
 的惊人细节。目前日本警方已对中蒲悠花的伤势展开取证，并对蒋劲夫发布逮捕令。有日媒分析，一旦逮捕罪名成立，蒋劲夫可能面临4-5年刑期。
 <br/>
 <br/>
 27日，中浦悠花以〝A小姐〞为名出现在日本TBS电视台晨间新闻节目中。
 <br/>
 <br/>
 面对镜头，中蒲悠花详细讲述了自己与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蒋劲夫.html" target="_blank">
  蒋劲夫
 </a>
 交往的几个月里，所遭遇的严重暴力行为。细节堪称触目惊心。
 <br/>
 <br/>
 中蒲悠花说，蒋劲夫的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_家暴.html" target="_blank">
  家暴
 </a>
 行为非常频繁且不断升级，甚至当众对她施暴，哪怕做梦，梦见女友做了不好的事情也要诉诸暴力，有时甚至使用鎚子。
 <br/>
 <br/>
 她说，蒋劲夫还控制她手机不许她和亲友联系，严重暴力伤害后，女方报警，花了一个半月才养好伤。
 <br/>
 <br/>
 最严重的那次发生在10月，失控的蒋劲夫从晚上12点一直打到第二天早上，打到〝家里全是血〞。
 <br/>
 <br/>
 在一段视频中，中浦悠花说：〝我被他殴打到两颗牙齿都断了，还勒着我的脖子往上吊，严重的时候还会用锥子打。〞
 <br/>
 <br/>
 蒋劲夫家暴事件于11月20日被曝光，当日中浦悠花在Instagram贴出自己鼻青脸肿的照片。
 <br/>
 <br/>
 她说，最近这段时间，让关心我的大家担心了，〝我，中浦悠花还健在。蒋劲夫行踪不明这件事的确与我有关，应律师和搜查相关人员的要求，恕我不便透露过多〞，似乎暗示蒋劲夫对自己家暴并因此遭到搜捕，而蒋劲夫的微博更新停于9月。
 <br/>
 <br/>
 隐身1个多月后，蒋劲夫11月20日9时48分发微博承认施暴。他说，这一个月我一直在忏悔和悔恨中度过。对不起悠花，我冲动的行为伤害了你和你的家人，我会为自己的行为负起责任，接受惩罚。在这里，诚恳地向大家道歉。
 <br/>
 <br/>
 日媒在节目透露，目前日本警方已对中蒲悠花的伤势展开取证，并对蒋劲夫发布逮捕令。另有日媒分析，一旦逮捕罪名成立，蒋劲夫可能面临4-5年刑期。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉)
