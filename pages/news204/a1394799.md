### 孟宏伟妻子被起底 并无〝驻外人员家属〞身份
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】因在国际社会曝光
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 〝被失踪〞一事，国际刑警组织主席、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟妻子.html" target="_blank">
  孟宏伟妻子
 </a>
 成为舆论关注的焦点。日前网上传出消息称，孟妻其实并无〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻外人员家属.html" target="_blank">
  驻外人员家属
 </a>
 〞身份，她在法国的生活并非由国际刑警组织或中共公安部出资。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 于9月25日返回中国后失踪，其妻子于当地时间10月7日在法国里昂召开记者招待会，对媒体发声、向法国警方报案。在新闻发布会召开不到一小时的时间，中共中纪委国家监委网站发布了一条消息：〝公安部副部长孟宏伟涉嫌违法，目前正接受国家监委监察调查。〞
 <br/>
 <br/>
 日前，坊间传言，孟妻其实并无〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_驻外人员家属.html" target="_blank">
  驻外人员家属
 </a>
 〞身份，她在法国的生活并非由国际刑警组织或公安部出资。网上传出的消息称，孟妻中文姓名不详，持香港护照，无业，2年来在里昂租住一幢实用面积270平方米（带4000平方米花园）的豪宅。
 <br/>
 <br/>
 据称，孟妻向法国警方报案丈夫孟宏伟失踪，同时指自己受到不明生命安全威胁。法国警方据此已经立案调查，并给予其司法安全保护。孟妻在法国里昂举行记者会时，两名负责保护她的法国警察全程陪同。
 <br/>
 <br/>
 孟妻暂时不愿将正面示人，说是担心她和孩子的安全。她在记者会上向外界展示了孟宏伟给她发的简讯和一把刀的照片，然后两人就失去联系。她认为这是孟宏伟的示警信号，表示遇到危险。
 <br/>
 <br/>
 截至目前，中共官方尚未通报孟宏伟所涉具体罪行，有人猜测，孟宏伟或涉及三大罪行，其一，周永康案牵出的政治问题；其二，受贿方面的经济问题；其三，红色通缉背后的国际问题。
 <br/>
 <br/>
 有分析称，孟宏伟与周永康案有关但不会是主因。若二者关系密切，孟不可能在周获刑后，还长期担任公安部副部长，更不会担任国际刑警组织主席一职。
 <br/>
 <br/>
 此外，由于海航董事长王健今年夏天在法国坠亡，网上传言，在法国担任国际刑警组织主席的孟宏伟，与该案件有关。
 <br/>
 <br/>
 旅美时评人、经济学者夏业良认为，孟宏伟事件说明中南海肯定发生了大事情。正常情况下，中共一般会使用常规手段来控制官员及其家属，但是本次显然等不及孟宏伟正常回国休假，以便将其妻儿一并控制。
 <br/>
 <br/>
 〝肯定发生了跟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 高层有关的、意料之外的重大事件，官方才必须紧急出手。或者，孟的举动被官方看出存在脱离控制的迹象，如果不立即办理，恐他直接投靠西方。〞夏业良称。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
</div>
