### 西安鱼化寨暴力强拆 爆发警民冲突
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安.html" target="_blank">
  西安
 </a>
 市最大的城中村
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鱼化寨.html" target="_blank">
  鱼化寨
 </a>
 ，从10月29号起遭到暴力
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 ，几天内已强拆了两条商业街，百家商户，因为没有安置方案和拆迁补偿，遭到当地民众反抗，爆发激烈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警民冲突.html" target="_blank">
  警民冲突
 </a>
 。
 <br/>
 <br/>
 网上消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鱼化寨.html" target="_blank">
  鱼化寨
 </a>
 百余家商户在10月29号早上被通知拆〝违建〞，下午就遭官府雇佣的黑社会几千人清场，几天内拆了两条商业街，百家商户、十多年的菜市场，2间幼儿园，老百姓一丝的缓冲余地都没有。
 <br/>
 <br/>
 现场视频显示，当地民众和身穿黑色制服的特警爆发激烈冲突，有民众遭到殴打和抓捕。
 <br/>
 <br/>
 当地民众透露，安置方案和拆迁补偿都没谈就来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强拆.html" target="_blank">
  强拆
 </a>
 ，因此老百姓反抗激烈。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安.html" target="_blank">
  西安
 </a>
 鱼化寨李先生：〝那拆迁肯定政府非法拆的，不满意你也没招啊，没有赔多少钱，钱不给到位，人家肯定不同意，不给钱肯定不会让他们拆的，给人家强拆了。鱼化寨整个要拆了，现在好多都在这儿找房子，现在房子很不好找，房租已经涨价了。〞
 <br/>
 <br/>
 网传视频显示，鱼化寨暴力强拆造成四死多伤，视频中有三人倒在地上，其中一人血流满地，当地民众未能确认此消息，只知道官方通报一则11月2号的宝鸡持刀伤人事件，遭成一死两伤。
 <br/>
 <br/>
 西安鱼化寨匿名民众：〝是啊四个老百姓，我发视频给你看吧。〞
 <br/>
 <br/>
 当地民众表示，鱼化寨总人口有35万，因为房租低，物价低，居住了很多外来打工者，周围一共有八个村子要拆，估计有上万户，当地政府从10月31号到12月底进村动迁。11月4号传出，一把手亲自出席鱼化寨拆迁动员大会，开会开到深夜1点多，103位村民代表全部签字同意拆迁补偿方案，签字率达到100%。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
</div>
