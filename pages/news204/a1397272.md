### 央视名嘴李咏病逝美国内幕曝光  9年前已拟好遗言
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
  央视
 </a>
 前主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李咏.html" target="_blank">
  李咏
 </a>
 近日在美国去世的消息引发舆论关注。9年前李咏就在自传中提到，自己已经想好将来要在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_告别仪式.html" target="_blank">
  告别仪式
 </a>
 上放的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遗言.html" target="_blank">
  遗言
 </a>
 ：〝欢迎大家光临我的告别仪式……都走到这一程了，还在这儿说话。没吓着你们吧？……〞
 <br/>
 <br/>
 10月29日上午，主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李咏.html" target="_blank">
  李咏
 </a>
 的妻子、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
  央视
 </a>
 导演
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈文.html" target="_blank">
  哈文
 </a>
 发微博称：〝在美国，经过17个月的抗癌治疗，2018年10月25日凌晨5点20分，永失我爱。〞
 <br/>
 <br/>
 据悉，李咏因患癌症医治无效，在美国去世，终年50岁。
 <br/>
 <br/>
 陆媒报导，李咏在美国时间10月25日去世，28日早上10时（北京时间28日晚上10时），在位于纽约的麦迪逊大道1076号的弗兰克林坎贝尔殡仪馆举行了葬礼。
 <br/>
 <br/>
 李咏去世的消息引发舆论哗然，微博热搜前20个话题中，与李咏相关的占了一半。
 <br/>
 <br/>
 公开资料显示，李咏1968年5月生于乌鲁木齐，曾主持节目《非常6+1》、《幸运52》、《梦想中国》、《咏乐汇》等，2011年到2013年连续三年担任央视春晚的节目主持人。
 <br/>
 <br/>
 2013年李咏辞职，离开了工作22年的中共央视，转入中国传媒大学，同年5月成立〝李咏工作室〞。
 <br/>
 <br/>
 李咏曾对媒体说，〝我现在最大的自由就是，我不想干什么就不干什么〞。
 <br/>
 <br/>
 李咏生前曾在一档节目中演讲《生命中的最后一天》——〝我会找一个安静的地方，静静地待着，我不会有道歉，也不会有离别，更不会有抱怨，我只会有感谢……所有电视机前的观众朋友们，感谢你们给我留下了美好的回忆。〞
 <br/>
 <br/>
 2009年长江文艺出版社出版李咏的自传《咏远有李》一文中提到，自己已经想好将来要在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_告别仪式.html" target="_blank">
  告别仪式
 </a>
 上放的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遗言.html" target="_blank">
  遗言
 </a>
 ：〝欢迎大家光临我的告别仪式，劳累各位了，你们也都挺忙。今天来的都是我的亲朋好友，既然不是外人，我也没跟你们客气。〞
 <br/>
 <br/>
 〝走之前，都说好了，今儿来送我，就别送花了，给我送话筒吧。我希望我身边摆满了话筒。人生几十年，一晃就过，我李咏这辈子就好说个话，所以临了临了，都走到这一程了，还在这儿说话。没吓着你们吧？〞
 <br/>
 <br/>
 〝在特定的舒缓音乐中，旁白仍在继续……闭目在话筒丛中，我肯定特安详。我这个自封的终身成就奖挺有创意吧？您觉得怎么样？〞
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：祝馨睿）
