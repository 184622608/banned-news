### 英媒：习近平事先知道孟晚舟被捕 但川习会上没提
------------------------

<div class="wysiwyg">
 【新唐人2018年12月08日讯】中国华为CFO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被抓和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 同日发生，引起各种猜测。白宫官员透露，美国总统川普当晚出席会晤时，对孟晚舟事件并不知情。而英媒引述消息指，北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 却事先得到相关报告，但在川习会晚宴上没有提。
 <br/>
 <br/>
 英国《金融时报》12月6日报导，12月1日，中方在会晤前已经知道了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被捕之事，但是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晚宴上并没有提及此事，原因是〝想集中精力解决中美贸易矛盾〞。
 <br/>
 <br/>
 对此，时政评论员唐靖远认为，目前该消息无法获得官方证实。但若是实情，也有其合理性。因为美方抓捕孟晚舟，说明已掌握华为违法的确凿证据。这种情况下，北京如果向美方提及此事，缺乏合适的立场。如果当场提出抗议，不但于事无补，还会破坏这次向美方〝求和〞的机会。
 <br/>
 <br/>
 有线新闻也引述分析认为，估计事发当晚中方已经知道孟晚舟被抓消息，不过北京有意低调处理而已。
 <br/>
 <br/>
 中国问题专家林和立则表示，孟晚舟被捕事件反映中美已进入新冷战时代，和美苏冷战不相伯仲。不过，由于习近平无法解决中美危机，在党内承受巨大压力，相信即使习近平即使知道孟晚舟被抓，仍然会优先处理中美贸易战，事后因为孟晚舟被捕打〝民族主义牌〞的机会也不大。
 <br/>
 <br/>
 事实上，孟晚舟事件12月6日曝光后，中方的反应也确实相当〝克制〞。
 <br/>
 <br/>
 虽然外界舆论一再强调此事会〝激怒北京〞，给美中贸易谈判〝蒙上阴影〞，但是中方除了提出〝抗议〞、要求放人之外，尚无任何报复措施。
 <br/>
 <br/>
 相反，中共商务部在12月6日当天再次在新闻发布会上强调川习会的〝成功〞，并承诺切实推进与美国间达成的关于农业、能源、汽车等领域的协议，还宣称中方〝有信心〞能在90天内与美国就贸易问题达成一致。不过，中方对90天谈判的重点——结构性改革依然只字不提。
 <br/>
 <br/>
 值得注意的是，与12月5日相比，中共商务部不但再次强调〝信心〞，还首次提及汽车领域，等于变相承认了川普此前透露的〝中方承诺削减和取消美国汽车进口关税〞的消息。
 <br/>
 <br/>
 事后，川普在推特转发了中共官方这次表态，并表示〝我同意〞。如此看来，川普〝相当赞赏〞中方就孟晚舟事件所表现出的〝克制〞。
 <br/>
 <br/>
 此次美方抓捕孟晚舟，罪名是华为违反了美国对伊朗的制裁令。但实际上，具有中共军方背景的华为，多年来都在海外从事网络间谍活动，替中共窃取外国情报和尖端技术，已被西方普遍视为对国家安全的重大威胁。
 <br/>
 <br/>
 美国共和党联邦参议员克鲁兹（Ted Cruz）6日在推特（Twitter）上表示，〝华为是蒙上一层电信公司薄薄面纱的中共间谍机构，它的监视网络跨越全球，它的客户是诸如伊朗、叙利亚、朝鲜和古巴这样的流氓政权。〞
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
