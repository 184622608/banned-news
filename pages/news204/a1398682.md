### 进博会订单曝光 韩媒：中共表演的政治独唱
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】中国首届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海进博会.html" target="_blank">
  上海进博会
 </a>
 将于11月10日结束，有消息指，中共对央企下达了大规模采购〝政治任务〞。但韩媒称，中共的订单〝政治气味〞浓厚，整个氛围更像是中共的〝独唱会〞。还有知情人士透露，中共这些〝进口订单〞可能并没有什么实际意义。
 <br/>
 <br/>
 习近平在11月5日的进博会开幕式上曾发表演讲称，〝这次博览会不是中国的独唱，而是与其他国家的一次合唱〞。中国主动扩大进口，不是权宜之计，而是面向世界、面向未来、促进共同发展的长远考量。
 <br/>
 <br/>
 本次进博会在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 的大背景下举办，中共有意释放继续对外开放的信号，以改善目前中共被国际社会孤立的现状。有消息指，中共已向央企、国企下达〝政治任务〞，必须完成大规模进口订单。
 <br/>
 <br/>
 路透社11月8日报导称，近百家中共央企及逾一千家下属单位，成为中共的采购主力军。一汽大众将在明年进口价值约90亿美元的汽车及零部件，东航集团与罗尔斯罗伊斯签订发动机采购协议，中国机械工业集团旗下企业签下价值84亿美元的订单，而中核集团也抛出总额达120亿美元的五年国际采购需求。
 <br/>
 <br/>
 不过，外界普遍认为，中共央企在进博会的订单，与其说是实实在在的订单合同，不如说是重复原有订单合同，或一些没有实际约束力的备忘录之类的单子。
 <br/>
 <br/>
 据某位参加国的知情人士透露，〝中国省市都以‘交易团’的名义大规模动员企业参加，这到底是不是真正意义上的‘进口订单’还不得而知〞。
 <br/>
 <br/>
 路透社称，这次进博会的风格更像是过气很久的传统宣传秀，外国企业的作用，倒像是用来彰显中共开放成果的〝调味剂〞。
 <br/>
 <br/>
 韩国企业对此次进博会的评价也大打折扣。韩媒《东亚日报》11月9日报导称，进博会表面上声称是面向世界的一次〝开放〞，而实际上更像是〝合唱团簇拥下的独唱〞。
 <br/>
 <br/>
 一位韩国企业相关人士透露，〝参加此次进博会的企业不是主角，〞此次进博会更像是中共的成果展示会。韩国企业在此次进博会上的宣传影像，也无端遭到中方阻拦。
 <br/>
 <br/>
 报导称，12家韩国动画及游戏等文化企业，从5日开始将在进博会向中国买家们公开放映相关的文化展演影像，但却遭到中共的阻拦。中方的理由是，外企只能播放有关企业介绍的影像，不可以进行商品销售为目的的文化宣传。
 <br/>
 <br/>
 对此，韩国相关人士称，中共仍然未撤销〝限韩令〞。
 <br/>
 <br/>
 另一方面，执行采购任务的一些中国企业表示，用钱并不能买来核心技术。〝现在，有卖的技术不想买，但想买的技术又买不到。〞中国某大型化工企业领导表示，未来中国企业引进、消化、吸收、再创新的传统技术发展道路会变得更加艰难。
 <br/>
 <br/>
 据报导，本次进博会明显遭到世界大经济体七国集团（G7）冷遇。美国、加拿大、英国、法国、德国、意大利及日本都没有国家领导人或政府首脑出席。韩国、澳大利亚等中国贸易伙伴国，也没有派出重量级人物参加。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
