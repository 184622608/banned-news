### 秦岭违建别墅拆除后续 陕西官场或降大地震
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】自中南海关注陕西秦岭违建别墅案后，地方官场多人因涉案相继落马。近期官媒披露，陕西官场或再有大变动。
 <br/>
 <br/>
 11月25日，中共官方背景的《了望》周刊称，秦岭违建别墅问题，不仅给生态环保带来阵痛，还将有一批领导干部被问责，一批干部因涉嫌贪腐而受到惩处。
 <br/>
 <br/>
 报导称，截至11月初，历时3个月的秦岭违建别墅整治接近尾声，共清查违建别墅1,194栋，其中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拆除.html" target="_blank">
  拆除
 </a>
 1,185栋，没收9栋。
 <br/>
 <br/>
 位于陕西秦岭北麓的别墅区，自2014年5月起，被习近平多次点名要求整改，但陕西不为所动，甚至〝失管失控〞。直到2018年8月中旬，秦岭违规建别墅群开始拆除。随着拆除，中共官员的违建、官场的贪腐以及政商勾结的黑幕陆续曝光。
 <br/>
 <br/>
 在拆除中备受关注的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈路.html" target="_blank">
  陈路
 </a>
 超大别墅，牵出前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安.html" target="_blank">
  西安
 </a>
 市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈宝根.html" target="_blank">
  陈宝根
 </a>
 。有媒体直指陈路是陈宝根的儿子。
 <br/>
 <br/>
 陈路超大别墅位于陕西鄠邑区石井镇蔡家坡村，2005年开始兴建，占地超过14亩，其中两个鱼塘面积逾千平米，狗舍面积78平米、文物211件。别墅内栽种了多种名贵树木，包括价值超过20万元人民币的皂角树。别墅酒窖内全是茅台酒和五粮液酒。期间，别墅主人还从西藏空运来5只藏獒幼崽，同时从西藏空运来羊肉喂藏獒。
 <br/>
 <br/>
 知情人称，就在8月西安市纪委成立调查组时，陈路得到消息后开始转移别墅内的物品。
 <br/>
 <br/>
 陈宝根曾在西安党政系统担任要职。资料显示，现年65岁的陈宝根是甘肃人，曾在陕西、四川官场23年，历任陕西宝鸡副市长、市长，四川广元市长，陕西西安副书记、市长、市人大常委会主任等。
 <br/>
 <br/>
 外界分析，周永康是陈宝根的后台。履历显示，陈宝根1995年至2017年的官场生涯中，唯一不在陕西的是2000年至2002年，他在四川担任广元市委副书记、市长，任期与由国土部部长转任四川省委书记的周永康完全贴合。
 <br/>
 <br/>
 2008年4月，时任中共政法委书记的周永康、罗干来到陕西，出席西安阎良航空高技术产业基地三年展，当时的西安市市长陈宝根作陪。
 <br/>
 <br/>
 此前，因秦岭违建别墅案，陕西官场已经历一轮大地震。今年11月，西安有多名官员涉及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦岭别墅.html" target="_blank">
  秦岭别墅
 </a>
 案。陕西省委常委、秘书长钱引安被调查，西安前市长上官吉庆、西安市前政协主席程群力被降职处分，西安国土资源局长安分局前局长卫旭峰被逮捕等。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
