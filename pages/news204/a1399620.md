### 彭斯李克强激烈交锋 南海问题高度对峙
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】中美第二轮外交安全对话之后，双方在南海的角力继续。美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 和中共国务院总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 日前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东亚峰会.html" target="_blank">
  东亚峰会
 </a>
 上发表针锋相对的言论。李克强指责第三国介入南海，而彭斯则抨击南海的军事化行动违法。
 <br/>
 <br/>
 路透社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 周五（16日）在新加坡表示，南海不属于任何一个国家，美国将在国际法允许的范围内继续航行和飞行。
 <br/>
 <br/>
 而据共同社报导，在15日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东亚峰会.html" target="_blank">
  东亚峰会
 </a>
 上，彭斯和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 在南海问题上发生激烈交锋。
 <br/>
 <br/>
 李克强表示，南海争端应由当事国之间解决，〝第三国〞不应介入，要求〝区域外国家〞尊重并支持该地区的努力。
 <br/>
 <br/>
 彭斯则针锋相对地反驳称，美国是印度太平洋地区的一部分，因此也是当事国之一，并指责南海军事化行动是违法的。
 <br/>
 <br/>
 9日，中美在华盛顿举行的第二轮外交安全对话中，就曾在南海问题上发生激烈争论。随后美国国务院发表声明，要求中共拆除在南沙群岛部署的导弹。
 <br/>
 <br/>
 中共2015年曾承诺不在南海建设军事设施，但其南海军事化行动从未停止。而美国也一直在该海域执行〝自由航行〞，保障国际航线畅通。此外，除了中方，汶莱、马来西亚、菲律宾、越南和台湾均宣称拥有南海部分主权。
 <br/>
 <br/>
 15日，彭斯还向东盟领导人表明，美国将会在各方面支援印太地区，让〝帝国主义和侵略行为〞在该地区没有立足之地，不点名地谴责中共的霸权行径和殖民政策。
 <br/>
 <br/>
 另外，彭斯此次访问亚洲，他的专机在飞往新加坡过程中穿越了南海上空，距离备受争议的南沙群岛仅80公里。他对《华盛顿邮报》表示，这趟飞行是美国体现自由航行的方式之一，强调〝我们不会屈服〞。
 <br/>
 <br/>
 美国《华尔街日报》称，彭斯已逐渐成为川普（特朗普）政府中对中共态度最强硬的人物。
 <br/>
 <br/>
 据《南华早报》报导，在东亚峰会开始前，与会领导人拍完合照后，彭斯和李克强曾留在台上短暂地握手交谈。但报导没有提到是谁主动发起交流。
 <br/>
 <br/>
 而彭斯在新加坡表示，此行并没有计划和中方官员举行正式会面。外界认为，这是美方有意怠慢或拒绝中方。
 <br/>
 <br/>
 根据美国白宫11月8日声明，彭斯于11月11日至18日访问亚太地区，并代表川普总统出席在新加坡举行的亚细安—美国峰会、东亚峰会等。在此期间，彭斯将会晤日本首相安倍晋三、印度总理莫迪、澳大利亚总理莫里森等多国领导人。但声明中没有提及和李克强会晤。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
