### 中共进口核污染区大米引恐慌 陆网民：让领导先吃
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】近日大陆宣布，解禁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 核污染地区食品进口，允许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新潟米.html" target="_blank">
  新潟米
 </a>
 进口。此次为日本
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福岛.html" target="_blank">
  福岛
 </a>
 核泄漏限制后的首度解禁。大陆网民批评中共罔顾民众身体健康。
 <br/>
 <br/>
 大陆海关总署11月28日公告，当日起允许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新潟米.html" target="_blank">
  新潟米
 </a>
 进口。此次开放日本新潟米进口，是2011年禁令后首次解禁。
 <br/>
 <br/>
 2011年3月11日，日本发生强震和海啸，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福岛.html" target="_blank">
  福岛
 </a>
 核电厂辐射外洩事故；2011年4月8日，大陆限制日本多个地区进口食品、食用农产品及饲料，新潟县为其中之一。
 <br/>
 <br/>
 消息一出，民众强烈不满，表示〝绝不会买。〞有人质疑，中共的特权阶层都是吃特供的，所以问：〝农业部的领导会带头吃吗？海关总署的领导吃不吃？〞，有网民诙谐回应：〝火灾的时候让领导先走，这次也让领导先吃〞。
 <br/>
 <br/>
 大陆宣布开始解禁核食的时机，引起民众注意。有分析表示，开放核食是因为贸易战。今年大陆同日本关系明显进展，大背景是中美贸易摩擦，大陆需要跟日本加强贸易上的往来。
 <br/>
 <br/>
 事实上，10月日本首相安倍晋三访问中国，他与大陆国务院总理李克强会面后就曾表示，获得北京允诺考虑，放宽因福岛核事故禁止的日本农产品。
 <br/>
 <br/>
 因此部分网民指责：为了拉拢日本，〝命都不要了〞?还有网民评论:〝谁同意解禁的必成千古罪人〞。
 <br/>
 <br/>
 还有网民留言说：〝前几天台湾公投刚否决掉吧？这边转身就进口啦？〞台湾11月24日举行九合一并公投选举，其中反核食公投过关。目前，台湾仍禁止日本福岛等5县的农产品进口。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
