### 国际狙击手大赛  暴露中共狙击步枪隐秘
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】由中共武警部队举办的为期5天的〝锋刃-2018〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际狙击手.html" target="_blank">
  国际狙击手
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_射击.html" target="_blank">
  射击
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竞赛.html" target="_blank">
  竞赛
 </a>
 ，日前在北京某
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武警训练基地.html" target="_blank">
  武警训练基地
 </a>
 举行。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共狙击手.html" target="_blank">
  中共狙击手
 </a>
 在大赛中，使用进口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 高精度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狙击步枪.html" target="_blank">
  狙击步枪
 </a>
 ，暴露了中共自制狙击步枪的隐秘。
 <br/>
 <br/>
 参考消息网报导，根据中共年度军事工作安排，中共武警部队于10月18日至22日，在燕山脚下的某训练基地隆重举办〝锋刃-2018〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际狙击手.html" target="_blank">
  国际狙击手
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_射击.html" target="_blank">
  射击
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竞赛.html" target="_blank">
  竞赛
 </a>
 。
 <br/>
 <br/>
 中国、白
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 、中非共和国、匈牙利、以色列和巴基斯坦等，21个国家的100多名军警宪同类部队的顶级狙击手参加了比赛。此次狙击手国际大赛进行了侦察判定狙击、潜伏狙击、解救人质狙击、对抗狙击等多个科目的竞赛。
 <br/>
 <br/>
 中共党媒称，来自中共海陆空特种部队的狙击手，参加了比赛，经过激烈角逐，中共武警猎鹰突击队包揽个人全能总冠军和小组团体总冠军。
 <br/>
 <br/>
 作为东道主，中方提供给一些国家选手，使用国产的CS/LR4外贸型高精度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_狙击步枪.html" target="_blank">
  狙击步枪
 </a>
 。但是中共自己的选手，却使用了一款过去从来没有对外曝光过的狙击步枪。
 <br/>
 <br/>
 原来，这是一款俄罗斯产奥尔西T5000型.338 Lapua狙击步枪，该狙击步枪，是狙击武器的王者之一。
 <br/>
 <br/>
 中国和俄罗斯官方都未对外透露，采购俄罗斯的狙击步枪的消息，但是此次中共官方媒体发表的狙击大赛上的中国选手射击图片，把这个秘密暴露出来。
 <br/>
 <br/>
 T-5000高精度手动狙击步枪，被业界称为是曾获得40多次国际射击赛冠军的〝狙击明星〞，由位于俄罗斯首都莫斯科市的枪械公司奥尔西集团公司2011年研发生产。
 <br/>
 <br/>
 对此，生产T5000高精度狙击步枪的俄罗斯〝工业技术〞公司总经理阿列克谢．索罗金说：〝我们使用了独特的新生产技术工艺，在俄罗斯和欧洲都没有类似技术，唯一有此类技术的是美国。〞
 <br/>
 <br/>
 与俄罗斯狙击步枪相比，目前最先进的中共国产CS/LR4狙击步枪，使用7.62毫米口径子弹。该狙击枪的全系统公务采购单价据传为26万元人民币，军迷们将其之称为〝26万〞或〝26式大狙〞。
 <br/>
 <br/>
 尽管昂贵价格，但中共狙击步枪的水平一直落后，在2009年世界狙击手大赛上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共狙击手.html" target="_blank">
  中共狙击手
 </a>
 使用从苏联时期的SVD德拉古诺夫狙击步枪仿制的85式狙击步枪参赛，结果因精度不够首轮便遭淘汰。
 <br/>
 <br/>
 经过多年研发，如今中共终于有了最新式的军用CS/LR4狙击步枪。然而，中共军事专家也承认，目前国产狙击步枪，仍只是解决了有无的问题，同世界先进的狙击步枪相比差距巨大。
 <br/>
 <br/>
 新浪军事8月8日刊文分析说，中共狙击步枪与欧美狙击步枪的差距都比较巨大，原因是多方面的，既有设计方面的原因，也有材料、机械加工精度的原因，更有不重视专用子弹的原因。
 <br/>
 <br/>
 文章说，设计方面，长久以来中共军队并没有使用狙击步枪的传统。中共的模仿对象，苏联SVD狙击步枪，它的精度相对西方国家成熟的狙击步枪，处于中下水平，这样也造成了中共在狙击步枪设计这一环节的薄弱。
 <br/>
 <br/>
 另外，中共有〝重枪不重弹〞的思维传统。相对于枪械的研究，对于配套子弹的研究总是滞后的、落后的。但枪械是靠激发弹药来杀伤目标。弹药不过关、打不中目标，枪械再好，也不过是一根烧火棍。
 <br/>
 <br/>
 文章称，在设计、材料与机械加工、配套子弹等方面，全面落后的情况下，中共国产狙击步枪的精度，自然是要落后于国外的高精度狙击步枪。
 <br/>
 <br/>
 相比中共的狙击步枪，俄罗斯产产T5000高精度狙击步枪不但性能高，而且价格比较低，成本只有1.65万卢布（约1750元人民币），该狙击步枪不但被俄军和内务部队采用，并出口多国。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：戴明）
