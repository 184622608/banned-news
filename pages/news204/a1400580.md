### 【禁闻】辽宁610文件曝光 迫害法轮功更加严厉
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 江泽民集团迫害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 19年来，作为操控、协调这场迫害的主要部门：政法委、610，成为这场迫害的最前线。最近在海外曝光的一份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁省.html" target="_blank">
  辽宁省
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_610办公室.html" target="_blank">
  610办公室
 </a>
 〞文件，内容显示，从10月开始，中共系统性的加大力度迫害法轮功。
 <br/>
 <br/>
 2017年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的〝敲门行动〞，在全国范围内挨家挨户地骚扰、绑架
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员。
 <br/>
 <br/>
 最近，海外曝光一份中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_610办公室.html" target="_blank">
  610办公室
 </a>
 〞下发的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_机密文件.html" target="_blank">
  机密文件
 </a>
 ，文件下达各个有关部门，包括文件下达到辽宁各地的〝610办公室〞、法院、检察院、公安局、司法部等，目标任务就是〝严厉打击打压法轮功〞。从10月开始，连续三个月。
 <br/>
 <br/>
 这份文件由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁省.html" target="_blank">
  辽宁省
 </a>
 传出，内容详细传达如何镇压法轮功。旅美时政评论人士横河表示，这类型应该是属于地方性文件，就是那些上不了台面或者怕别人知道的内部文件。
 <br/>
 <br/>
 旅美时政评论人士横河：〝因为这个就是指挥系统，指挥链，就是中央610办公室的文件，它也不能够从中央610直接发到各省省委，它只能发到各级610，所以它必须是由610把这个报告打到中央办公厅，中共中央办公厅以中央文件下达，才能让各级党委去接受这个命令。〞
 <br/>
 <br/>
 横河认为，文件内容提到很多具体的措施，明显违反中国宪法和中国法律，由此可见，中国根本不存在所谓的依法治国，而是〝文件治国〞。
 <br/>
 <br/>
 横河：〝尤其是对于打压宗教、信仰迫害，打压人权，它完全是没有按照法律或者哪怕是公开的规定，而只能用秘密的方式来进行，因为如果要把法律公开了，它就不需要任何文件了，但是它也就没有办法打击宗教信仰，所以它只能用这种方式。〞
 <br/>
 <br/>
 2018年3月，中共当局在所谓的机构改革中，把所谓〝处理法轮功问题领导小组〞，和江泽民时期专职迫害法轮功的机构〝610〞办公室，合并到中央政法委员会。而根据最近下发的文件内容显示，这个所谓〝防范打击法轮功的专项行动方案〞，是由公安局具体负责，〝互联网信息办公室〞（网信办）配合，行动时间是10月到12月。
 <br/>
 <br/>
 横河：〝那么这个部署是2018年10月份到12份的行动，因此这个文件应该是在9月到10月下达的，而这个体制改革是在3、4月份就已经完成了，所以这个可以证明，无论是防范和处理领导小组，还是610办公室，仍然在以它们原来的机构在活动。〞
 <br/>
 <br/>
 横河认为，〝处理法轮功问题领导小组〞和〝610办公室〞还没有消失。
 <br/>
 <br/>
 横河：〝从现在看来，这两个机构并没有取消，而是说它的功能甚至连名称，就是一起并到中央政法委，而不作为一个独立机构存在了，但是这样，它在中央政法委里面，仍然是独立机构，就是迫害法轮功，仍然是由专职机构来处理的。〞
 <br/>
 <br/>
 以江泽民为首的中共江氏集团，从迫害法轮功的初期就提出〝名誉上搞臭、经济上截断、肉体上消灭〞、〝打死算白死，打死算自杀〞和〝不查身源，直接火化〞等灭绝政策。十九年来，法轮功学员被污衊、长期被监控者近亿，数百万人被绑架、关押、劳教、判刑及关洗脑班，甚至被酷刑致死或活摘器官致死，数百万家庭被迫害得支离破碎、妻离子散、家破人亡。
 <br/>
 <br/>
 《法轮大法明慧网》指出，中共各级政法委、〝文件内容提到很多具体的措施，专家表示，这表明，臭名昭着的中共610〞办公室、公安部门，也假借〝扫黑〞的名义，一再迫害法轮功学员。今年8月22号，辽宁省各市出动警察绑架法轮功学员，仅仅两天，就有34名法轮功学员及家属被绑架。
 <br/>
 <br/>
 而11月9号这一天，黑龙江省超过百位法轮功学员被中共非法绑架，抓捕的理由是所谓的〝扫黑除恶〞。但那关进黑牢的却是坚定修炼佛法的善良中国人民。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/周天
