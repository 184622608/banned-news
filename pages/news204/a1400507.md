### 美中舌战延伸世贸会议 美代表斥中共扭曲全球市场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】川习会在即，美中双方角力不断。日前，双方在世贸组织（WTO）会议上唇枪舌剑，火药味十足。美代表指责中共利用WTO推动〝非市场〞政策，扭曲全球市场。与此同时，还有美国官员表示，可以考虑把〝行为不端〞的中共逐出WTO。
 <br/>
 <br/>
 11月21日，WTO在位于瑞士日内瓦的总部举行会议，美国贸易副代表习达难（Dennis Shea）在会上表示，中共正在利用WTO推动〝非市场〞政策，扭曲全球市场，导致大量产能过剩，尤其是在钢铁和铝方面。
 <br/>
 <br/>
 习达难还表示，中共对国际贸易体系构成威胁，并且正试图利用WTO争端解决机制，阻止任何成员国采取任何行动，解决其不公平、扭曲贸易的政策。
 <br/>
 <br/>
 在会上，WTO同意对美国的指控进行调查，将任命一个由三名专家组成的小组，以确定中共政策是否违反WTO条款。明年可以做出裁定。
 <br/>
 <br/>
 而中共官员对美国的指控，则一如既往地予以否认。在回应美国的知识产权申诉时，中方代表还辩称，中共没有违反知识产权协议。
 <br/>
 <br/>
 而就在此前一天(20日)，美国贸易代表办公室发布了〝301调查报告〞更新版，例举各类实例，曝光中共各种不公平贸易行为，包括知识产权盗窃。
 <br/>
 <br/>
 英媒BBC11月21日报导称，美国总统经济顾问委员会主席哈塞特（Kevin Hassett）表示，中共作为WTO成员〝行为不端〞。
 <br/>
 <br/>
 哈塞特表示：〝我们从未真正设想过，一个国家加入世贸组织后会像中共那样行事。对世贸组织来说，有这么一个行为不端的成员，真是头一遭。〞
 <br/>
 <br/>
 他指出WTO规则让美国受到伤害，提出解决WTO问题的三个方案是，通过双边谈判、世贸组织改革，甚至将中共逐出世贸组织，也可视为一个解决方案。
 <br/>
 <br/>
 这是哈塞特第二次在公开场合提出〝把中共赶出WTO〞的构想，今年8月份他在受访时表示，若中共不像其它成员国一样遵守规则，为什么还要将它留在WTO里？
 <br/>
 <br/>
 美国贸易代表莱特希泽（Robert Lighthizer）也曾表示，当年美国允许中共在没有做出这种改变的情况下就加入WTO，是个错误的决定。
 <br/>
 <br/>
 目前，WTO的改革议题受到全球重视。11月1日，美国及欧日等其它会员国向WTO共同提出〝强化WTO透明化程序及通知要求〞文件，依其内容，未来不履行通知义务的WTO会员（如中共），将受到严厉惩罚。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
