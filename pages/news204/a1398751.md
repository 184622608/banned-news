### 【 禁闻】彭博：中国空屋率22% 逾5000万套
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】
 <h4>
  彭博：中国
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空屋率.html" target="_blank">
   空屋率
  </a>
  22% 逾5000万套
 </h4>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国房市.html" target="_blank">
  中国房市
 </a>
 一直备受外界关注，11月8号,  〝彭博社〞报导，中国目前有五分之一的房屋是空屋，数量超过5000万套。
 <br/>
 <br/>
 报导说，根据一项由成都西南财经大学教授甘犁，去年所主导的调查显示，中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空屋率.html" target="_blank">
  空屋率
 </a>
 高达22%。一旦房市泡沫开始破裂，这些空屋将被火速抛售，引发房价螺旋式下跌。这对中共执政者来说,  将是可怕的噩梦。
 <br/>
 <br/>
 甘犁说，世界上没有一个国家的空屋率如此之高，如果房市出现问题，被抛售的房屋将会像洪水一样袭击中国。
 <br/>
 <br/>
 <h4>
  流亡作家
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马建.html" target="_blank">
   马建
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
   香港
  </a>
  讲座 一波三折
 </h4>
 <br/>
 中国流亡异见作家马建原定11月10号在香港大馆举行两场文学讲座，推介他的新书《中国梦》，但11月8号，香港大馆以〝不愿见到大馆成为任何个别人士，促进其政治利益的平台〞为由，拒绝提供场地。导致活动取消，引发舆论哗然。
 <br/>
 <br/>
 9号，事件又出现逆转，香港大馆再发表声明说，由于马建已公开表示只是以作家的身份参与文学讨论，因此愿意再提供场地，讲座将按原定计划进行。
 <br/>
 <br/>
 马建当天从英国抵达香港，并对媒体表示，对于大馆曾用政治理由取消活动，他感到十分愤怒。他认为〝大馆〞背后一定有一个更大的黑手，这个〝黑手〞可能是〝在赛马会之上〞、是〝可以控制香港舆论和未来〞的团体。
 <br/>
 <br/>
 <h4>
  俄罗斯威胁停止向中国出口木材
 </h4>
 <br/>
 俄罗斯高级官员日前警告，如果中国不出钱帮俄罗斯恢复西伯利亚的森林植被资源，俄罗斯将全面禁止向中国出口木材。
 <br/>
 <br/>
 据〝美国之音〞报导，西伯利亚和远东的森林被大规模砍伐出口中国，已经日益引起俄罗斯社会的关注甚至愤怒。
 <br/>
 <br/>
 11月7号，俄罗斯自然资源和生态部部长科贝尔金，在上议院联邦委员会表示，他在会晤中共高官时警告他们说，中国现在需要木材，那就应该帮助俄罗斯重新恢复森林植被资源，中国应该出钱在俄罗斯境内为俄罗斯兴建10-20个树苗繁育设施，否则，俄罗斯将停止向中国出口木材。
 <br/>
 <br/>
 编辑/周玉林
