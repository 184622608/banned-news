### 阴霾围城 陆82城市发布重污染预警
------------------------

<div class="wysiwyg">
 【新唐人2018年12月04日讯】首先来关注中国的消息。最近几天，中国多个地区出现持续性大雾天气，多达82个城市发布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重污染预警.html" target="_blank">
  重污染预警
 </a>
 ，当地民众苦不堪言。
 <br/>
 <br/>
 11月30号至12月3号，中国华北、华东、华中地区出现持续性大雾天气，同时受沙尘滞留影响，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_京津冀.html" target="_blank">
  京津冀
 </a>
 及周边地区、长三角区域、湖南、湖北部分地区，正经历大范围重污染过程。
 <br/>
 <br/>
 河北中南部、河南、山东中西部、安徽和江苏北部、湖南和湖北部分地区，共62个城市日均浓度达到重度及以上污染水准。
 <br/>
 <br/>
 京津冀及周边地区41个城市，细悬浮微粒（PM2.5）日均浓度，达到重度及以上污染水平。
 <br/>
 <br/>
 西安郑先生：〝严重啊，现在污染很严重啊！灰蒙蒙的一片。〞
 <br/>
 <br/>
 河南郑州李先生：〝（空气品质）相当不好。 四、五天都是这样。雾霾已经很严重了。往天上看，完全是乌霾霾的一片，根本看不见太阳！〞
 <br/>
 <br/>
 河北邯郸田先生表示，即使戴上口罩作用也不大，现在当地民众是能不出门尽量不出门。
 <br/>
 <br/>
 河北邯郸田先生：〝现在我这个地方能见度大概也就是200来米。已经持续一个礼拜了吧。〞
 <br/>
 <br/>
 中国中央气象台从11月24号到12月3号，共发布大雾黄色预警8次，橙色预警9次。
 <br/>
 <br/>
 据大陆媒体报导，这次是今年以来最严重的一次污染过程。
 <br/>
 <br/>
 田先生估计，这和国内经济形势不好有关。
 <br/>
 <br/>
 田先生：〝现在为了增加就业，发展经济，对雾霾的控制比以前有所放松了。原来基础就没有控制好。你像咱们国内石油这个标号都是比较低的，达不到国际上那个排放标准。〞
 <br/>
 <br/>
 另外，田先生表示，中共的治理模式，是把问题压到官员头上。为了摆脱空气质量在全国排名倒数，去年邯郸罢免了一些环保部门官员，因此当地空气质量迅速提升。可是运动式的治理过后，问题还是没有解决。整个体系都是向上负责，而不是由民众来监督。
 <br/>
 <br/>
 李先生：〝政府加大力度（治理）是骗人的，那是假的！
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工业污染.html" target="_blank">
  工业污染
 </a>
 是最主要的。彻底（治理）共产党就完了，它不会彻底（治理）的。〞
 <br/>
 <br/>
 至于为什么政府做不到彻底治理污染问题，西安郑先生认为，这牵扯到政治制度和民众权利的问题。
 <br/>
 <br/>
 新唐人记者熊斌、王子琦、施宜君采访报导
