### 救火贸易战？ 王岐山将出席中美新经济论坛
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 或出现转机，有消息称，中国国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 将于下周访问新加坡，出席首届〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美新经济论坛.html" target="_blank">
  中美新经济论坛
 </a>
 〞，并就中美经济发表主题演讲。王历来有〝救火队长〞之称，但此前并未就贸易战公开发声。因此，他此次突然〝出山〞，引发外界广泛关注。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美新经济论坛.html" target="_blank">
  中美新经济论坛
 </a>
 是由中共国家发改委主管的经济智库〝中国国际经济交流中心〞和彭博社合办，旨在促进中美关系发展。论坛发起人是前纽约市长布隆伯格（Michael Bloomberg），美国前国务卿基辛格（Henry Kissinger）及美国前财长保尔森（Hank Paulson）等多位全球商界精英参与规划。
 <br/>
 <br/>
 论坛的中方成员包括有中共国务院前副总理曾培炎、全国政协副主席董建华、中国银行董事长陈四清，中共前驻澳大利亚和英国大使傅莹等。
 <br/>
 <br/>
 综合媒体报导，中美新经济论坛将于11月6日至7日在新加坡举行，议题是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 将出席并发表演说，承诺北京会继续开放经济，以安抚国际社会，避免北京被进一步孤立。
 <br/>
 <br/>
 外界认为，这可能是王岐山首次在国际平台公开谈及中美贸易战话题。还有评论指，王岐山此行，或负有为中美贸易战〝救火〞的使命。
 <br/>
 <br/>
 王岐山被指是中共〝救火队长〞，中美在贸易领域发生争端后，国际舆论认为，与美国政府打过交道并有丰富经验的王岐山，会成为处理中美关系的中方核心人物。
 <br/>
 <br/>
 不过，在贸易战开打前后期，中美双方有过多次谈判，但王岐山并未参与。期间，他至多也就是会晤美方一些商界人士，谈谈中方对在华美企的立场。
 <br/>
 <br/>
 对此，有评论认为，王岐山之所以没有就中美贸易战发声，与他是中共政坛〝二号人物〞的身份不无关联。王岐山是习近平的得力干将，中共政坛甚至有〝习王体制〞一说。如果王岐山在应对美方时出错，外界也会将此视作是习近平的个人失误。
 <br/>
 <br/>
 加之，北京当局在贸易战中进退失据，一直拿不出好的应对策略。因此，王岐山也不知道该如何应对美方。
 <br/>
 <br/>
 有分析称，王岐山此次出席〝中美新经济论坛〞，并就贸易战首度发声，意味着北京或已作出一些重要决定。不排除王岐山或奉命〝出山〞，为贸易战〝救火〞。
 <br/>
 <br/>
 在此之前，10月22日，王岐山还率团参加在以色列举办的创新峰会。外界认为，王岐山是在为中共在贸易战的施压下，寻找别的出路。
 <br/>
 <br/>
 目前，中美贸易战处于恶化的边缘，川普总统已多次警告，或将对另外2670亿美元的中国商品加征关税。另一方面，贸易战已导致中共经济急速下行，中国股市、汇市一蹶不振。大批民营企业宣布注销、清算，北京当局对此忧心忡忡，习近平近日多次发声，欲提振民营企业信心。
 <br/>
 <br/>
 11月1日，川普总统意外在推特发文称，他与习近平进行了电话长谈，重点讨论了贸易问题，并称〝进展良好〞。他们还将在G20峰会上继续讨论这些议题。
 <br/>
 <br/>
 诸多迹象显示，中美贸易战近期很可能会发生一些变化，但究竟会有多大进展？走向如何？将是国际舆论关注的话题。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：戴明)
