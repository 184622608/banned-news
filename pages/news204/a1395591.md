### 借共克时艰之名趁火打劫？地方国资频出手被质疑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】贸易战背景下，中国经济陷入困境。今年以来，上市公司高管接连离职、出逃，股权交易频繁，国企趁机大举收购民企。外界质疑，中共借〝共克时艰〞之名，抢劫民间资产。
 <br/>
 <br/>
 据多家陆媒报导，截至今年9月12日，中国沪深两市已有73个上市公司发布控制权转让公告，其中29个被国有资产收购控股，不少收购方是各地国资运营平台或地方国资委。
 <br/>
 <br/>
 另外，截至10月10日，A股上市公司今年有360多位董事长、8000多名高管接连离职或遭警方调查。一张多家上市公司董事长离职的截图在社交网络上热传。

 近期以来，中共体制内〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二次国有化.html" target="_blank">
  二次国有化
 </a>
 〞、〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 〞的舆论抬头。上述消息更加重了社会的恐慌。
 <br/>
 <br/>
 中国经济学者胡星斗教授周一（15日）对自由亚洲电台表示，中国的政治经济体制已被严重扭曲，上市公司本身就是一个欺诈场所。国有企业最初上市的动机，就是诈骗民众的钱财，给国企脱困。
 <br/>
 <br/>
 同一天，中共国务院国有资产监督管理委员会（国资委）召开新闻发布会，回应民间有关〝国企藉机大规模并购民企是国进民退〞的说法，称这是〝正常市场化行为〞，是〝国企和民企互惠共赢的市场选择〞。
 <br/>
 <br/>
 还有海外党媒宣称，中国股市日前再次上演〝千股跌停〞，国资出手是为〝稳定市场〞。
 <br/>
 <br/>
 近年来，中国经济已经开始放缓，贸易战更令市场陷入困境，原本已经营困难的民企更加面临生存危机。中共党媒号召民间〝共克时艰〞，外界担心中共会藉机洗劫民企资产，补偿贸易战。
 <br/>
 <br/>
 除了收购民企之外，中共还强迫上市公司设立党委，抢夺企业控制权。另外，当局也在借〝推行民主管理〞加强共产党对民企的控制。日前消息，山东青岛市工会开始派驻干部挂职民企第一主席，在民企建立和强化工会领导。
 <br/>
 <br/>
 虽然北京高层日前接连表态〝一如既往〞地支持民营企业，但同时也强调〝做大做强国企〞，无法完全打消外界的担忧。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
