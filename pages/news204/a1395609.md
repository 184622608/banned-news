### 私家车贴吁官员公开财产 南京市民被控危害国家安全
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】中国江苏省南京市公民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦沪辉.html" target="_blank">
  秦沪辉
 </a>
 因在自己的私家车上贴标语，呼吁官员财产公开、司法独立，竟然被中共警方以寻釁滋事罪刑事拘留，其后更指称秦〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_危害国家安全.html" target="_blank">
  危害国家安全
 </a>
 〞，不准许律师与当事人见面。
 <br/>
 <br/>
 现年68岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦沪辉.html" target="_blank">
  秦沪辉
 </a>
 女士是江苏南京市民，她平日喜欢在自己的私家车贴上〝一人一票〞、〝宪法〞及〝司法独立〞等字句，呼吁争取公民权益，因此长期受到当地国保的警告与监视。从大约一年前，秦的寓所外被警方安装了监控摄像头。今年9月4日，南京警方以寻釁滋事的罪名把秦沪辉刑事拘留，10月10日又正式逮捕。
 <br/>
 <br/>
 秦沪辉案的代理律师王磊本月15日接受《自由亚洲电台》采访时证实，他近日到南京市看守所要求会见此案的当事人时，看守所的接待窗口称〝办案单位有指令，不准许会见。〞而且接待窗口的值班警察还声称此案在侦查过程中〝发现有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_危害国家安全.html" target="_blank">
  危害国家安全
 </a>
 的事实行为〞。经过律师的一再要求后，该警员表示可以与办案单位〝沟通〞一下，一周内给回复。
 <br/>
 <br/>
 王磊介绍案情称，秦沪辉在自己的私家车贴上了〝官员财产公开〞、〝司法独立〞及〝新闻自由〞等标语，完全是一个公民行使言论自由的宪法权利，是完全正当的行为，与寻釁滋事毫无关系。但中共就怕民众揭露政府丑陋的一面，因而对敢言的民众进行打压甚至构陷。

 <br/>
 秦沪辉的一名朋友在接受《自由亚洲电台》采访时，因为惧怕遭中共打压报复，特别要求电台改变他受访的声音。
 <br/>
 <br/>
 这位朋友表示，这次秦沪辉女士被抓可能与近期〝中国政治气氛紧张〞有关，中国民众批评政府的每一件事都会成为敏感事件，〝一有人表达诉求，就容易惹上官非〞。
 <br/>
 <br/>
 该朋友表示，他估计如果秦沪辉肯屈服认罪的话，也许可以获得轻判，但秦女士是一个性格很刚强的人，她是不会屈服认罪的，所以还是希望海外媒体能够多多关注报导此事件，给当局施加压力，可能结果会好一点。
