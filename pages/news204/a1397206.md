### 马云登〝套现企业〞榜首 胡润称〝洞悉先机〞阿里急否
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】贸易战当头，中国经济陷入危机，很多人在寻求自保，但避免成为中共打击目标。胡润研究院日前公布最新《胡润套现企业家30强》，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马云.html" target="_blank">
  马云
 </a>
 荣登榜首。胡润高层称赞马云等有〝先见之明〞，阿里随即发声明，指责胡润〝挑动矛盾〞。
 <br/>
 <br/>
 根据上周五（26日）公布的《胡润套现企业家30强》排行榜，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马云.html" target="_blank">
  马云
 </a>
 家族去年7月1日至今年9月30日，通过减持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴.html" target="_blank">
  阿里巴巴
 </a>
 的美国上市股份，套现110亿元人民币，较去年增加近1.5倍，遂成〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_2018套现大王.html" target="_blank">
  2018套现大王
 </a>
 〞。
 <br/>
 <br/>
 胡润百富董事长兼首席调研员胡润称赞，马云等大陆套现富豪有先见之明，在中美贸易战扩大、大陆经济放缓之前成功套现。
 <br/>
 <br/>
 不过，阿里方面并不接受胡润的赞扬，周日（28日）发表题为〈中国不需要富豪榜，中国需要企业家精神〉的声明，言辞激烈地斥责胡润套现榜〝价值扭曲〞、〝浮夸误导〞、〝误导公众〞、〝挑动矛盾〞，并质疑胡润制作榜单的〝动机〞。
 <br/>
 <br/>
 声明还指责胡润研究院制订排行榜时没有和阿里核实，因此质疑数据由来，声称〝怒难接受〞。
 <br/>
 <br/>
 阿里强调，马云和阿里董事局副主席蔡崇信是〝按计划减持股票〞，套现是用来扶持〝公益事业〞。
 <br/>
 <br/>
 同一天，胡润研究院就回应阿里，公布了马云套现的计算方法：根据阿里上市公司年报，在过去一年里，马云名下各个持股平台共减持阿里1111万股股份，扣除用于慈善的持股平台减持的200万股，剩余部份约为911万股，按照这个区间每股175美元的均价计算，套现金额约16亿美元，即110亿元人民币。
 <br/>
 <br/>
 胡润指，榜单制作过程公平公正，也会尽量联系涉及企业，但若是上市公司就直接从公开资讯中计算。
 <br/>
 <br/>
 今年9月，马云突然宣布一年后退休的计划，外界普遍认为是急流勇退的保命之举。但马云则公开否认，称退休是为了〝更好地专注教育事业〞。
 <br/>
 <br/>
 英国会计师胡润（Rupert Hoogewerf）在1999年创立的中国富豪榜《胡润百富榜》，如今已成为中国大陆最有影响力的富豪榜之一。《胡润套现企业家30强》是《胡润百富榜》的姐妹榜。
 <br/>
 <br/>
 去年10月公布的《2017胡润百富榜》中，有43名阿里系股东进入百富榜。阿里当时就曾公开质疑该排行榜，时任蚂蚁金服董事长彭蕾甚至称之为〝抽风榜〞。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
