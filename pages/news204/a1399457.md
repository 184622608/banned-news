### 被注销律师证 覃永沛索赔89亿元
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】今年5月遭注销律师执业证的广西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_覃永沛.html" target="_blank">
  覃永沛
 </a>
 ，11月13号起诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广西司法厅.html" target="_blank">
  广西司法厅
 </a>
 ，指其注销律师证违法，要求赔偿89亿5000万元人民币，并点名司法厅厅长林金文，要求对方出庭对质。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_覃永沛.html" target="_blank">
  覃永沛
 </a>
 在起诉书中写道，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广西司法厅.html" target="_blank">
  广西司法厅
 </a>
 涉嫌强奸法律的尊严，强迫、威胁解散他合伙经营的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_百举鸣律师事务所.html" target="_blank">
  百举鸣律师事务所
 </a>
 ，并违法注销他的律师执业证，给他造成了巨大的精神伤害和经济损失。
 <br/>
 <br/>
 广西
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 覃永沛：〝它没有法律依据嘛，共产党有8千9百多共产党员，我以为，一个赔10块钱是没问题的，由于它依法治国是假货，就是假一赔十嘛，我们中国有法律嘛，我就根据这个理念，索赔89.564亿。〞
 <br/>
 <br/>
 覃永沛在诉状中还点名广西司法厅厅长林金文，要求他出庭对决，并警告掌控公权力却视法律为儿戏的林金文，将会成为〝全民公敌〞。
 <br/>
 <br/>
 不过，覃永沛告诉《新唐人》，他的诉状遭到法院拒收，他计划联合其他几位被除牌的律师，一起控告最高法院院长周强。
 <br/>
 <br/>
 覃永沛：〝它拒绝签收，法院，后来我了解到，是中央政法委针对我们这帮人权律师的重点清除对象，他不讲法律，就是说，法院我就不受理，你爱告就告去，那我就去告最高法院院长周强，我认为上梁不正下梁歪。〞
 <br/>
 <br/>
 目前旅居加拿大的祝圣武也曾被吊销律师证，他很认同覃永沛的做法。
 <br/>
 <br/>
 旅加原维权律师祝圣武：〝我对这个事情是非常支持的，因为从去年开始新一轮的对律师的迫害，存在很多非法的迫害人权律师的问题。共产党对这个国家治理，它负有责任，所有的党员干部对这个国家的混乱都负有责任，所以我完全支持他的这种请求。〞
 <br/>
 <br/>
 祝圣武对覃永沛
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_起诉司法厅.html" target="_blank">
  起诉司法厅
 </a>
 的行为表达了敬佩之意。
 <br/>
 <br/>
 祝圣武：〝他提起这个诉讼，我们很敬佩他的战斗的精神，和他这种甘愿去冒风险，去揭露这个法治的虚伪的这种勇气。〞
 <br/>
 <br/>
 覃永沛是广西百举鸣律师事务所主任，曾实名举报中共司法部部长傅政华，代理过徐纯合、秦永敏、709江天勇案及多起法轮功案件。目前覃永沛与其他数位〝被除牌〞律师成立了〝中国律师后俱乐部〞，再遭警方登门威胁。
 <br/>
 <br/>
 新唐人记者常春、陈洁、元真采访报导
</div>
