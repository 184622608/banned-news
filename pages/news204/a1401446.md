### 中共经济内忧外患 陆820万大学生恐〝 毕业就失业〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】受美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 影响，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共经济.html" target="_blank">
  中共经济
 </a>
 内忧外患，企业面临危机，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大裁员.html" target="_blank">
  大裁员
 </a>
 〞的消息满天飞。对中国大陆820多万应届大学毕业生来说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业.html" target="_blank">
  就业
 </a>
 成为一个巨大的挑战，不少人可能会遭遇〝 毕业就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业.html" target="_blank">
  失业
 </a>
 〞的尴尬局面。
 <br/>
 <br/>
 大陆财新网日前的一条微博说，今年4月到9月间，中国大型求职网站前程无忧的招聘广告数量，从285万条锐减至83万条，减少了202万条，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_就业.html" target="_blank">
  就业
 </a>
 形式不容乐观。
 <br/>
 <br/>
 与此同时，美国表示，可能在明年年初，将2000亿美元中国商品的税率，按计划从10%提高到25%。美国总统川普（特朗普）日前还表示，如果12月1日的川习会谈判失败，美国可能会对其余的2670亿美元中国商品征税。这无疑对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共经济.html" target="_blank">
  中共经济
 </a>
 来说，将是一个噩耗。
 <br/>
 <br/>
 自7月美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 爆发以来，短短5个月，中共经济已遭受重创。经济增速下滑至十年来最低点，制造业也面临危机，大批国际制造商已经或正在准备撤离中国，中国沿海地区不少工厂已出现了严重裁员或停工状况。
 <br/>
 <br/>
 多家大型厂商爆出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大裁员.html" target="_blank">
  大裁员
 </a>
 〞的消息，其中，代工之王富士康被传将裁员34万。玻璃镜片供应商伯恩光学惠州厂区订单骤减，不得已裁员5千人，由此引发了大规模劳务纠纷。
 <br/>
 <br/>
 今年9月，大陆房地产巨头万科喊出〝活下去〞的惊人口号，如今，〝活下去〞已成为大陆现在流行的热词。网上有文章称，饭碗才是硬道理，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失业.html" target="_blank">
  失业
 </a>
 〞已成为笼罩在大陆的又一场阴霾。
 <br/>
 <br/>
 2019年，这场〝失业〞阴霾可能会更加严重。中共官媒11月29的报导，透露出就业形势下行的风向。报导称，明年中国高校毕业生预计会达到834万人，就业形势更加复杂严峻。
 <br/>
 <br/>
 对此，中共教育部副部长林蕙青提出，中国社会应该广泛运用〝网际网路+就业〞新模式。
 <br/>
 <br/>
 不过，此前有消息传出，阿里巴巴、京东、百度、腾讯等网路巨头，正全面停招或减招应届毕业生。虽然一些公司否认这一消息，但这一趋势还是有迹可循的。
 <br/>
 <br/>
 阿里巴巴创始人马云近日对媒体说，近年来公司对应届生的招聘人数骤减，每年此类入职员工不超过300人。
 <br/>
 <br/>
 另据智联招聘平台大数据显示，今年第三季度，IT/网际网路行业的招聘职位数同比减少51%。
 <br/>
 <br/>
 上述财新网的微博引述天风证券收集的数据，数据覆盖大中小型城市和60个行业。数据显示，202万条招聘广告消失，其中二线城市的比例最高，500人以下的中小微企业和民营企业是减招主力。财新网的这条微博随后被删除。
 <br/>
 <br/>
 在此之前，10月22日，大陆媒体《网易》也刊出一篇报导，称〝今年上半年国内504万家企业倒闭，失业人数超过200万〞。报导还指出，上半年还有453家上市公司董事长辞职或被炒。不过，文章也很快就被删除。
 <br/>
 <br/>
 这些报导显然和官方通报的数据存在明显差异。中共官媒《经济日报》8月报导称，中国上半年城镇新增就业达到752万人，而城镇登记失业率第二季度末不到4%，也是近10年来的新低。
 <br/>
 <br/>
 深圳当代社会观察研究所所长刘开明，长期关注中国劳工市场的动向。在他看来，中共官方发布的就业数据并不可靠，尤其是就失业率而言。
 <br/>
 <br/>
 〝失业率是根据登记数字决定的。现在中国城镇就业的主力是农民工，而农民工通常不做这样的登记。〞
 <br/>
 <br/>
 中国金融学者贺江兵也认为，就业机会大面积蒸发，最直接的受害者仍然是社会的传统弱势群体。〝还是农民工受害最深，特别是对制造业和服务业的影响最大。这些人只好回老家接着种地，他们无计可施，现在被迫回家务农的人太多了。〞
 <br/>
 <br/>
 中共农业农村部日前宣称，目前有740万人〝返乡创业〞。但外界质疑，这波所谓的返乡创业潮，只不过是〝失业〞的另一种说法罢了。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
</div>
