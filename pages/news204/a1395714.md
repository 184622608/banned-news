### 港媒：习近平将南下视察 到访广州一所大学
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】港媒报导，中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将于19日至22日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南下.html" target="_blank">
  南下
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广东.html" target="_blank">
  广东
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视察.html" target="_blank">
  视察
 </a>
 ，20日到访广州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中山大学.html" target="_blank">
  中山大学
 </a>
 ，目前该校正为习近平的到访作准备。但相关消息未得到官方证实。
 <br/>
 <br/>
 10月17日，港媒明报引述消息说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广东.html" target="_blank">
  广东
 </a>
 的活动，包括到访广州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中山大学.html" target="_blank">
  中山大学
 </a>
 ，该校正为习近平的到访作准备。
 <br/>
 <br/>
 中山大学消息人士透露，习近平可能在10月20日到中山大学，但确切行踪尚不清楚，该校目前确实正为习近平到访该校做相关准备工作。
 <br/>
 <br/>
 报导称，中共领导人外出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视察.html" target="_blank">
  视察
 </a>
 时，通常会有多套方案，不到最后一刻很难知道其确切行踪。
 <br/>
 <br/>
 报导还引述消息人士披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 9月底试运后，10月底〝具备通车条件〞。据港府消息，中共高层曾研究习近平为港珠澳大桥主持通车仪式。但暂时未确定习是否会出席通车仪式。
 <br/>
 <br/>
 香港行政长官林郑月娥17日早接受媒体访问时说，目前还不知道大桥何时通车。
 <br/>
 <br/>
 港珠澳大桥连接香港大屿山、澳门半岛和广东省珠海市，由桥、岛、隧道和粤港澳连接线组成。
 <br/>
 <br/>
 9月20日，就有报导称，习近平
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南下.html" target="_blank">
  南下
 </a>
 广东视察，深圳安保进入一级戒备。有腾讯内部员工则披露，习近平出席在深圳举办的改革开放40周年纪念大会，以及考察广州、深圳、东莞、中山、惠州多地企业。同时见证深圳升格为直辖市。
 <br/>
 <br/>
 但相关消息至今都未获证实，根据官方报导显示，习近平9月25日到28日去了东北，并重提闭关锁国时期的〝自力更生〞。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
