### 藏2.7亿现金 华融前董事长赖小民被审查
------------------------

<div class="wysiwyg">
 【新唐人2018年10月15日讯】中共官方周一通报，中国华融资产管理股份有限公司前党委书记、董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 ，被立案审查。赖小民案被指是中纪委和国家监察委合署办公后，第一个金融大案。
 <br/>
 <br/>
 通报指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 违反政治纪律和政治规矩，盲目扩张、无序经营，搞政治投机，为个人职务升迁拉关系，美化宣传个人，捞取政治资本，参加迷信活动，对抗审查；与多名女性搞权色交易；收受巨额财物涉嫌受贿犯罪；非法占有公共财物涉嫌贪污犯罪。
 <br/>
 <br/>
 据报，办案人员在赖小民的几处房产里，搜出的外币现金相当于人民币2.7亿元，约3吨重，这还只是其涉案财产的一小部分。
 <br/>
 <br/>
 赖小民曾长期在中共央行工作，曾任央行银行监管二司副司长、银监会办公厅主任，2009年起担任华融总裁、董事长。
 <br/>
 <br/>
 据报，赖小民与明天系掌门人肖建华、华信董事会主席叶简明及已落马的重庆前市委书记孙政才关系密切。
</div>
