### 【禁闻】11月15日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月16日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 江苏宿迁教师抗议克扣福利 遭驱散
 <br/>
 <br/>
 江苏省宿迁市宿豫中学的数百名教师，因不满校方改制后，拖欠和克扣福利待遇，11月12号起连续3天到宿豫区委上访维权，14号遭警方驱散。
 <br/>
 <br/>
 2. 南京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权人士.html" target="_blank">
  维权人士
 </a>
 史竞遭公司解聘
 <br/>
 <br/>
 南京维权人士史庭福的儿子史竞，因经常与父亲一起参与维权活动，11月初在南京机场准备登机前往日本旅游时，被警方拦截，并以〝寻釁滋事〞罪名拘留，11月15号获释后，遭雇主解聘。
 <br/>
 <br/>
 3. 杀截访警察
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访民.html" target="_blank">
  访民
 </a>
 许有臣被执行死刑
 <br/>
 <br/>
 河南焦作访民许有臣，2014年因在被截访过程中杀死警察，2016年被判死刑，许有臣不服，上诉，今年1月被二审裁定驳回，维持一审死刑判决。据维权网报导，11月15号他已经被执行死刑。
 <br/>
 <br/>
 4. 朱承志被以网上寻釁滋事罪名批捕
 <br/>
 <br/>
 湖南维权人士朱承志，今年4月赴苏州祭奠文革中被迫害致死的北大女生林昭，被警方抓捕，监居半年期后，再以〝煽动颠覆国家政权罪〞刑事拘留，11月13号，他的家属接到苏州国保电话通知说，朱承志的罪名又有变化，已经被以涉嫌〝网络寻釁滋事罪〞逮捕。
</div>
