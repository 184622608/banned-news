### 孟宏伟重创中共形象  舆论关注中共在国际机构任职情况
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】国际刑警组织前主席、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，日前因涉嫌违法被调查。孟的突然落马引发国际社会对中共官员在国际组织中担当
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_要职.html" target="_blank">
  要职
 </a>
 的关注和担忧。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 9月底回国后，随即被中共羁押。中共公安部部长赵克志8日凌晨主持召开党委会议，通报了孟宏伟收受贿赂、涉嫌违法接受国监委监察调查的情况。而作为世界上规模第二大的国际刑警组织（INTERPOL），在事件发生之前却全不知情。
 <br/>
 <br/>
 美国哈佛大学政治学博士、独立学者顾为群，通过种种迹象分析认为，孟宏伟回国后失踪应该是遭到了当局的伏击，并不是受命自首。
 <br/>
 <br/>
 他表示，身为公安部副部长的孟宏伟，其安全警惕性、政治敏感度应该比一般人灵敏得多，没有受过专业训练的人看不出来的情况，警察们甚至能很快看出来。所以，他指出，中共政府之所以没有提前告知国际刑警组织，也没有要求孟辞去他的主席职务，是为了避免打草惊蛇，让他没有机会窜逃。
 <br/>
 <br/>
 美国国际战略研究中心亚洲事务高级顾问兼〝中国实力项目〞主任葛来仪（Bonnie Glaser）认为，经历了这场风波，其它国际组织会更加慎重地考虑任用中共官员履行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_要职.html" target="_blank">
  要职
 </a>
 。但她推断，此事发生并不是政府的草率决定，这背后一定经过了利弊权衡。
 <br/>
 <br/>
 她说：〝最终无论如何，中共做决定的时候首先考虑的，是一个事件会怎样影响国内时局；国际关系和中共的名声固然重要，但占据的只是次要位置。〞
 <br/>
 <br/>
 时事评论员文昭表示，近一、两年间，中共对其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际形象.html" target="_blank">
  国际形象
 </a>
 表现得越发不在意，但这种〝不在意〞并不一定是真的不在意，而是有另外让它〝更在意〞的事情，也就是对中共党内不稳定性的焦虑。
 <br/>
 <br/>
 文昭认为，孟宏伟事件也同时释放出一种信号，就是在警告党内官员，不要以为〝国际身份〞可以充当某种保护伞，当中共有必要的时候，是会牺牲国际形象也要维护所谓〝内部忠诚〞的。
 <br/>
 <br/>
 目前，由中共推荐出任国际组织高层的官员，还有联合国（U.N.）副秘书长刘振民、国际货币基金组织（IMF）副总裁张涛、联合国工业发展组织（UNIDO）总干事李勇、国际民航组织（ICAO）秘书长柳芳、世界贸易组织（WTO）副总干事易小准、国际电信联盟（ITU）秘书长赵厚麟，以及上周落马的前国际刑警组织主席孟宏伟。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
</div>
