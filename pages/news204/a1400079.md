### 秦岭违建别墅再曝内幕  先交50万才能进院看房
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】被历代帝王尊为华夏文明龙脉的西安秦岭，因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建别墅.html" target="_blank">
  违建别墅
 </a>
 多次惊动中共高层，并引发陕西官场持续震荡。近日陆媒再曝新消息，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦岭别墅.html" target="_blank">
  秦岭别墅
 </a>
 群中，一处叫西安院子违建别墅项目，院落内部奢靡，需交50万元才允许进入院落看房。
 <br/>
 <br/>
 秦岭北麓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建别墅.html" target="_blank">
  违建别墅
 </a>
 背后细节触目惊心。陆媒近日曝光了秦岭违建别墅的奢靡和神秘的内幕。中纪委曾数次专门派员到陕西开展〝专项整治〞，习近平6次批示仍无法清拆，近期中纪委副书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐令义.html" target="_blank">
  徐令义
 </a>
 两次赴陕西〝指导工作〞，高层接二连三的出手，显示陕西的处理方式令中央非常震怒。
 <br/>
 <br/>
 《中国经济周刊》20日再曝新消息说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦岭别墅.html" target="_blank">
  秦岭别墅
 </a>
 群位于西安市鄠邑区草堂镇，属秦岭适度开发区，建有别墅241栋，占地面积14.96万平方米。
 <br/>
 <br/>
 其中，西安院子项目是典型的破坏秦岭环境的违建别墅项目，此项目是秦岭别墅群中，一个有山有水、浑然天成的别墅项目，所建成的院落内部奢靡。2005年正式启动，2010年左右设计出来，期间主人不断调整策划。
 <br/>
 <br/>
 报导介绍，这处院落在设计中兼顾文旅和住宅。从文旅层面讲，西安院子是有私密性的、酒店式的，但住在这里，让人有家的感觉，还可以在屋里做饭。
 <br/>
 <br/>
 从住宅层面讲，出了房间，到了院子里可以下棋、喝茶，一家人还可以烧烤，每个院子里面还有一个用植被环抱的露天温泉汤泡池，让人有在自家院子里泡澡的感觉。
 <br/>
 <br/>
 在设计风格上，采用古建筑和现代民居融合的方法，主人称，完全按传统古建做的话，人住着并不舒服。
 <br/>
 <br/>
 在建筑材料上，房子用的都是精品，突出质感，墙体厚度达到37公分，中间是红砖墙，在外面又加了一层青砖，叫手工水磨砖。门全是用老榆木打造的老门板，工艺很好。
 <br/>
 <br/>
 看房者，首先要交50万元人民币才允许进院子，不买房可以退。
 <br/>
 <br/>
 除了资产，学历也要摆过来。在住户方面，主人要求购房者学历不能太低，得有文化，只有文化人才能够品味院子的古典情怀。
 <br/>
 <br/>
 据设计者介绍，西安院子项目一期，更像文旅项目，手续确实拿得出来，所以不在违建拆除之列。但二期是一个住宅项目，〝二期闭着眼睛就拆了，不用质疑。〞
 <br/>
 <br/>
 相关人士还介绍了规模庞大的秦岭违建别墅的建成内幕。
 <br/>
 <br/>
 西安一位涉足房地产业的老板称，在秦岭山麓建别墅，最初是一种个人行为，1997、1998年就有了。一些城里人到农村买地盖别墅，其实也是不合法的，但有人看到别人能盖，那我也能盖，慢慢人就多了。
 <br/>
 <br/>
 另一位大型房地产集团陕西负责人刘劲说，在7、8年前，改变土地性质的事情经常发生，甚至到了疯狂的地步。
 <br/>
 <br/>
 明明是一块工业用地，80万元一亩拿的，但如果拿地的开发商是关系户，交一点土地出让金，就把土地性质变成了住宅用地，一亩地利润就是几百万元，转变土地性质后，文旅项目摇身一变就成了别墅项目。
 <br/>
 <br/>
 而〝土地变性通常涉及到利益输送。过去给领导送现金、黄金，后来一般就送房子了。外头一平方米卖两万，他卖给领导只有3,000，还装修好了〞，刘劲说。
 <br/>
 <br/>
 西安市委原书记魏民洲，因从开发商人那里要来了一套别墅，于2017年5月22日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 ，他是秦岭别墅问题落马官员中最高官之一。目前陕西官场大震荡仍在持续中。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
