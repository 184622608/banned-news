### 江泽民一张照片震惊国际 中方为此大为尴尬
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 任中共党魁时，在外交场合戏子般的表演和卖弄，丑态百出。其中江泽民2002年访问冰岛在国宴上突然站起来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高歌.html" target="_blank">
  高歌
 </a>
 一曲，其老婆王冶平当时尴尬表情难看，整个情景被外媒以大幅彩色照片详细报导，国际哗然。
 <br/>
 <br/>
 2002年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 去冰岛访问时，在冰岛总统举办的国宴上，江泽民吃着半截饭，突然站起来
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高歌.html" target="_blank">
  高歌
 </a>
 一曲，令在场宾主都错愕不已。江泽民老婆王冶坪当时哭丧着脸，面部表情十分尴尬。一旁的中方人员对江投去不屑的目光。冰岛总统则目不斜视，表情尴尬。
 <br/>
 <br/>
 整个情景被冰岛最大的日报以大幅彩色照片详细报导，江泽民让国际媒体好一顿挖苦。
 <br/>
 <br/>
 江泽民这种不顾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外交礼仪.html" target="_blank">
  外交礼仪
 </a>
 和场合、随时高歌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_跳舞.html" target="_blank">
  跳舞
 </a>
 卖弄的丑闻时有发生。
 <br/>
 <br/>
 1996年江泽民出访菲律宾，晚上菲律宾总统拉莫斯在游艇上宴请江泽民，江泽民突然拿起麦克风高歌了一曲猫王的《温柔地爱我》（Lovemetender）。
 <br/>
 <br/>
 同年6月下旬，江泽民去西班牙访问。西班牙国王卡洛斯请江泽民一起检阅三军仪仗队。令卡洛斯吃惊的是，江泽民居然在这个时候，突然拿出一把梳子，在国王面前梳理头发，搔首弄姿，令西班牙国王惊讶地张着合不上的嘴在旁观看。
 <br/>
 <br/>
 晚间在西班牙欢迎国宴上，江泽民坐在王后右侧，再次在摄像机面前梳头。事后，西班牙第一大报《国家日报》和其它许多报纸以头版头条刊出新闻图片:《卡洛斯国王看江泽民梳头》。
 <br/>
 <br/>
 很快，全球多家报纸进行了转载。许多海外华侨看后顿觉炎黄子孙颜面尽失。
 <br/>

 <br/>
 1999年3月30日，江泽民在奥地利总统克莱斯蒂尔的陪同下参观萨尔斯堡莫扎特故居。故居内最有价值的收藏品是莫扎特1785年在维也纳购买的一架钢琴，据说除他本人，世界上还没有第二个人弹过。钢琴前挂着〝禁止触摸〞的德英文告示。
 <br/>
 <br/>
 谁知，在奥地利总统介绍完这架钢琴后，江泽民竟一屁股坐下来，开始弹奏〝洪湖水，浪打浪〞。江的突发举动让在场熟知奥地利文化背景的中国大使馆陪同人员大为吃惊和羞愧，而奥地利总统有心阻止江泽民，但又顾及外交礼节，表情十分尴尬。
 <br/>
 <br/>
 在奥地利人心目中，莫扎特就像圣人一样，对莫扎特文物的亵渎，就是对整个奥地利的羞辱。第二天，奥地利媒体铺天盖地头条用大标题〝丑闻〞报导此事，《多瑙日报》更刊登了一幅漫画：江拿着锤子和镰刀弹钢琴。
 <br/>
 <br/>
 同年10月24日，江泽民在法国参观一座博物馆时，拉起总统希拉克夫人贝娜黛特的手就跳起华尔兹舞来，这件事让希拉克非常不悦，认为是给自己难堪。而法国民众更义愤填膺，认为是对整个法兰西民族的侮辱。
 <br/>
 <br/>
 2002年2月21日，江泽民在北京大会堂设宴欢迎美国总统布什。江泽民当着在场百余名嘉宾高歌一曲〝我的太阳〞，美国总统布什马上鼓掌，并接着半开玩笑地请国务卿鲍尔唱一首小夜曲，鲍尔礼貌地微笑拒绝。
 <br/>
 <br/>
 晚宴中，江泽民又拉着美国第一夫人劳拉跳舞，跳完之后又先后拉着美国国家安全顾问莱斯及驻北京大使夫人莎拉共舞。
 <br/>
 <br/>
 在国际媒体公开报导中，江泽民不分时间场合地吟诗赋词、引吭高歌、卖弄英语，在大庭广众之下抠鼻孔、死盯着女人不放等出人意料的丑态已是常态。江泽民以中国国家元首身份出访时的种种丑闻，也令国家和民族在世界舞台上丧失了尊严。
 <br/>
 <br/>
