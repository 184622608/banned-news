### 重庆〝误报〞高考政审 牵出教育部违宪规定
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 媒体6日报导，重庆的高考录取，将把〝政审〞作为必备材料引发强烈反响。专家解释，中共〝政审政策〞从未真正停止。来看报导。
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 日报》6号报导，重庆2019年高考录取，将把〝政审材料〞作为必备材料，政审不合格者不能参加普通高校的录取。
 <br/>
 <br/>
 唐靖远：从中共建政起，政审就具有特定含义，特指用祖辈父辈的所谓阶级成分和政治立场，来株连后辈。文革前很多人因为政审不过关而上不了大学。
 <br/>
 <br/>
 报导引发轩然大波，重庆教育考试院随后紧急出面回应，称记者用词错误，不是恢复〝政审〞，而是对考生本人施行〝思想政治考核〞。
 <br/>
 <br/>
 唐靖远：文革后，中共面临社会瓦解的危机，邓小平不得已放宽了政审条件，不强调家庭出身情况了，但仍然保留了对考生本人政治立场的审查。重庆现在事实上就在延续这种做法，考察考生本人。
 <br/>
 <br/>
 夏业良：这套做法是让人深恶痛绝的。都是共产党那一套控制人的办法。它要控制人，要成为党的驯服工具。
 <br/>
 <br/>
 重庆教育考试院声称，重庆市按照中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_教育部.html" target="_blank">
  教育部
 </a>
 的文件，对考生施行思想政治考核，并要求高等学校〝不录取思想政治考核不合格的考生〞，多年来没有变化。
 <br/>
 <br/>
 本台记者查询发现，中共教育部从2000年开始，在《关于做好普通高等学校招生工作的通知》中，就明文要求对考生进行〝思想政治品德考核〞，直到2018年从未间断。
 <br/>
 <br/>
 滕彪：这种做法显然侵犯了《宪法》关于中国公民受教育权的规定。
 <br/>
 <br/>
 横河：中国的《高等教育法》是1999年1月1日施行的，里面关于学生的入学权利只有一条，〝公民依法享有接受高等教育的权利〞。也就是说，接受高等教育的唯一条件，就是是公民就可以了。教育部不是立法机构，它没有权力修改或增加任何条件。
 <br/>
 <br/>
 中共教育部还列出几种情况，算作〝思想政治不合格〞。包括考生〝反对四项基本原则或参加邪教组织；扰乱社会治安〞等。
 <br/>
 <br/>
 横河：违反了宪法保障的言论自由和信仰自由。违宪的是教育部的规定。最可能侵犯的群体，排第一位的就是宗教信仰团体，在过去20年当中,主要是法轮功群体。还有异议人士、民主人士，甚至连访民都很难逃出这样的黑手，因为访民被治安处罚的太多了。
 <br/>
 <br/>
 专家指出，中共虽然字面宣称只考察考生个人情况，但在现实中，株连式〝政审〞仍在进行。
 <br/>
 <br/>
 横河：几年前安徽民主人士张林的女儿安妮，受到她父亲牵连，被剥夺受教育的权利。过去19年当中，有无数的法轮功学员的子女被剥夺了受教育的权利。
 <br/>
 <br/>
 《重庆日报》〝误报〞风波，意外牵出中共教育部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违宪规定.html" target="_blank">
  违宪规定
 </a>
 ，横河表示，这一违宪规定属于思想罪，当局可能根据需要，随时改变标准，最终每个人都是潜在受害者。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
