### 贪官自杀新动向 公安局长〝留遗书自证抑郁症〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】每有中共官员自杀时，官方多以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抑郁症.html" target="_blank">
  抑郁症
 </a>
 〞搪塞死因，饱受民间调侃。官方或许自己也觉得太过单调乏味，日前已有所变通，让自杀官员通过〝遗书〞，自己宣布患有〝抑郁症〞。
 <br/>
 <br/>
 内蒙古官媒引述当地公安厅的说法称，北京时间11月1日凌晨，内蒙古公安厅副厅长、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_呼和浩特.html" target="_blank">
  呼和浩特
 </a>
 市公安局局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李志斌.html" target="_blank">
  李志斌
 </a>
 ，在该市公安局休息室内缢亡，已〝初步排除他杀〞。
 <br/>
 <br/>
 报导特别强调，李志斌留在现场的〝遗书〞中称，自己〝一直被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抑郁症.html" target="_blank">
  抑郁症
 </a>
 困扰，每天都靠高效安眠药才能入睡〞。
 <br/>
 <br/>
 有海外网友评论称，〝能不能找个专家谘询一下，换个病〞，〝这个传染病只传染高官〞。更有网友指，〝这是2018年最佳内涵段子。〞
 <br/>
 <br/>
 此前，中共官方宣布官员自杀原因时，多借用〝医生〞或〝亲友〞、〝同事〞的说法来引证该官员患有〝抑郁症〞。官员用〝遗书〞自证〝抑郁症〞，似乎还是头一遭。
 <br/>
 <br/>
 因此，有网民猜测，李志斌是〝被缢亡，也被现场留遗书〞，〝牺牲一个人却保全了整个党组织〞。
 <br/>
 <br/>
 外界对中共官方所谓〝临时工〞、〝报复社会〞等说法，以及中共监狱看守所的各种〝离奇死法〞已经司空见惯。而公安系统借用死者所谓〝遗书〞自证案情，也是常见手法。
 <br/>
 <br/>
 今年9月，正在洛阳市第十七中学绝食抗议的数学老师王宏召突然死亡。官方事后称他是跳楼自杀，并指在其手机内发现还没发出的短信〝遗言〞。而所谓〝遗言〞的第一句话就是〝我是自杀〞。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：秦锋）
