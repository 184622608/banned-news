### 福建政府称〝毒碳九〞清理干净 村民愤怒曝真相（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】福建泉港日前发生毒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碳九泄漏.html" target="_blank">
  碳九泄漏
 </a>
 ，引发严重生态事故。中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_当局掩盖真相.html" target="_blank">
  当局掩盖真相
 </a>
 ，不仅未及时通报泄漏的是有毒化学品，提醒民众防护，更谎称污染物已清理干净，一切正常。当地民众对此非常愤怒，发布视频曝光谎言，更有大批民众到中共地方当局门前抗议。
 <br/>
 <br/>
 据中共官方发布的消息，11月4日凌晨，福建东港石油化工实业有限公司进行油品装卸作业，发生油品泄漏事故，共造成6.97吨有毒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_碳九泄漏.html" target="_blank">
  碳九泄漏
 </a>
 。
 <br/>
 <br/>
 5日，泉港区环保局发布通告称，至4日晚18时，碳九泄漏海域清理工作已基本完成，一切正常。
 <br/>
 <br/>
 但直到8日，当地村民们表示，事发地附近的空气仍有刺鼻气味，海面上的污染物也没有彻底清除，更有不少村民陆续出现呕吐等身体不适现象。
 <br/>
 <br/>
 事故发生地肖厝村的肖厝小学，有老师公布缺勤表，从5日开始至7日，已经有11名儿童因头晕、恶心、呕吐等症状请假缺课。由于气味薰人，当地学校的学生都不得不戴着口罩上学。
 <br/>
 <br/>
 据官方通报，当地至少已有52人身体不适入院，其中一人因污染物导致肺感染，正在重症室抢救。有专家担心，若此次泄漏的是毒性大的〝重整碳九〞，会有中毒、致癌等风险。
 <br/>
 <br/>
 距离事发地四五公里的肖厝村民李女士说，她家〝邻居的女儿（7日）出现口吐白沫、翻白眼，家长带孩子去医院检查花了六百多元，问是不是中毒，医生都不敢说，在泉港医院里面很多村民都是去看呕吐的，症状都一样〞。

 <br/>
 据了解，此次泄漏事件导致的污染范围广，周边受到影响的至少有肖厝、峰尾、郭厝、南浦等村镇。村民还反映，在泉州、石狮、惠安、莆田等地都可以闻到空气中的刺鼻气味。
 <br/>
 <br/>
 碳九泄漏事故发生后，中共地方当局封锁消息，中共官媒也未将真实情况报导出来。据悉，事发后中共当地泉州电视台的记者在现场进行报导时，工作人员一边戴着两层口罩，一边报导说空气质量正常，遭到现场村民斥责。
 <br/>
 <br/>
 肖厝村的村民王先生向大纪元表示，事发当晚，在鱼排上睡觉的渔民被刺鼻气味熏醒，发现鱼排在往下沉，水产品逃走、死亡。一看发现鱼排下面的泡沫被腐蚀，最初村民根本不知道是什么东西，只发现海水变得像胶水一样，粘粘的。
 <br/>
 <br/>
 许多渔民为抢救鱼排，徒手去捞油污，什么防护措施都没有，有多人身体不适，出现头晕、呕吐、手抽筋等中毒症状，被送往医院就诊。
 <br/>
 <br/>
 〝最后（5日）才知道是碳九，刚开始都不知道，政府有发文，刚开始说是油品泄漏，后来才改成碳九泄漏。〞王先生说。
 <br/>
 <br/>
 王先生还透露，他知道有一名村民因到水里捞东西，造成手被（泄漏的碳九）腐蚀、烧伤。还有参与清理的军人也出现了身体不适症状，被送入医院。后来军人全部都戴上防毒面具。
 <br/>
 <br/>
 当地村民对此非常气愤，他们表示，官方一是没有真实报导，二是没有告知村民相关的防护措施。
 <br/>
 <br/>
 〝毒到什么程度，多少村民需要撤离或者是隔离，还是怎么样，都没有（讲）。我们老百姓都是在无知的状态下，像无头苍蝇一样，不知道怎么弄。反正总的来说，是损失越来越严重。〞王先生说。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 碳九泄漏这一重大安全事故一直被中共当局刻意掩盖。直到消息压不住，8日才得以广泛曝光。然而中共官方的声明仍在淡化影响，称52人就诊〝多数已好转〞，还引述〝专家〞说法称碳九〝有毒但非剧毒〞，〝不太会吸入中毒〞等。
 <br/>
 <br/>
 这些言论在微博上招来一片骂声：〝丧尽天良的官员！〞〝这帮灭绝人性的东西！〞〝这种新闻你们也敢压？视人命如草芥的畜生是要下地狱的！〞
 <br/>
 <br/>
 王先生还透露，4日晚中共泉港区环保局为了数据造假，利用喷水车向环保大楼喷水，视频被附近村民拍下来了。村民怀疑是因为环保局内有检测空气质量的仪器，为了降低空气污染浓度，让仪器数值能够〝达标〞，而进行的〝喷水作业〞。
 <br/>
 <br/>
 中共环保局5日即通报称，空气、水质等环保数值正常。连日来的持续监测，环保局也没有发出任何有毒物质警报。
 <br/>
 <br/>
 但大陆财新网报导，直到8日，呛鼻气味仍在泉港区范围内弥漫，肖厝村多位村民出现不适，入住泉港医院急救。肖厝村支撑水产养殖的塑胶鱼排已被碳九腐蚀，海面油污也仍未打捞干净。
 <br/>
 <br/>
 碳九泄漏事件不仅对当地居民生活环境造成严重污染，更令渔民损失惨重。当地刘女士说：〝渔民大概一户要损失500万元，鱼排鱼苗都打水漂了。我自己亲戚也有做海的，他们收的紫菜、海蛎都不能卖，买了的人要退货。〞
 <br/>
 <br/>
 王先生也说：〝我们长达二三十年积累下来的一千多个鱼排，顷刻之间全部都没有了，损失无法估量。〞
 <br/>
 <br/>
 此次泄漏事件中共地方当局欺上瞒下，引发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_村民愤怒.html" target="_blank">
  村民愤怒
 </a>
 ，纷纷将受污染死亡的烂鱼、烂虾摆在区政府和东港石化公司门前。〝每天都有二三百人去抗议，要把这些烂鱼虾卖给区长。〞村民们说。
 <br/>
 <br/>
 通过视频显示，村民在东港石化公司门前搭起帐篷，坐在那里表示抗议，还在石化公司的大门上挂起〝无良东港毒害百姓毁我家园！！！〞横幅。
 <br/>
 <br/>
 村民们说，直到8日，中共当地政府与石化公司方面都没有一名官员出现，给村民们一个说法。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
