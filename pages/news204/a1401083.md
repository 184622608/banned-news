### 苏轼名画《木石图》拍出4亿天价 真伪却遭存疑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】近日，中国北宋著名文学家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏轼.html" target="_blank">
  苏轼
 </a>
 的传世名画《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_木石图.html" target="_blank">
  木石图
 </a>
 》，以4亿多元人民币在香港
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佳士得.html" target="_blank">
  佳士得
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拍卖.html" target="_blank">
  拍卖
 </a>
 专场成交，创下中国古画拍卖的最高纪录。不过，监于目前没有确定属于苏轼的画作流传下来，有专家质疑这幅画的真伪。
 <br/>
 <br/>
 11月26日晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佳士得.html" target="_blank">
  佳士得
 </a>
 香港秋季
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拍卖.html" target="_blank">
  拍卖
 </a>
 中，经过激烈的竞争，一幅被认为由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏轼.html" target="_blank">
  苏轼
 </a>
 绘制的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_木石图.html" target="_blank">
  木石图
 </a>
 》以4.1亿港元落槌，加上后续买家需要支付的傭金，总价高达4.636亿港元，约合4.117亿元人民币。
 <br/>
 <br/>
 佳士得拍卖公司称，这个价格刷新了4年前另一件拍品〝明永乐御制红阎摩敌刺绣唐卡〞创下的3.484亿港元竞拍纪录。据悉，这幅《木石图》也是迄今为止中国古代绘画拍出的第一高价。
 <br/>
 <br/>
 《木石图》又称《枯木怪石图》，有近千年的历史，是苏轼任徐州太守时前往萧县圣泉寺时所创作的一幅水纸本手卷，全卷长185.5公分，画的是一株枯木状如鹿角，一具怪石形如蜗牛，怪石后伸出星点矮竹。手卷上有四位文人刘良佐、米芾、俞希鲁和郭淐的题字，以及41名收藏家的印章。
 <br/>
 <br/>
 苏轼传世画作极少。据古书记载，苏轼传世画作只有两幅，一副是中国美术馆的镇馆之宝《潇湘竹石图》，另一幅就是这次拍卖的主角《木石图》。
 <br/>
 <br/>
 佳士得官方表示，此画于1937年流传至北京书画商及古董商白坚夫手中，其妻为日本人，此后该画被带往日本，直至被佳士得公司从日本寻回。尽管佳士得认可这幅画为苏轼真迹，但仍有不少学者认为这幅画的真伪需要进一步鉴定。
 <br/>
 <br/>
 有大陆媒体引述业内专家称，鉴定这幅画的一大难点在于目前没有确定属于苏轼的画作流传下来，虽然国内也有博物馆藏有传为苏轼的作品，但都没有明确能认定是苏轼作品的证据，争议还是存在的，真伪还需要今后学界进一步研究鉴定。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：戴明）
