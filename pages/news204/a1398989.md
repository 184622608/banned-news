### 继范冰冰被罚8亿后 赵薇退回4千万
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月12日讯】中国影视娱乐演员逃税问题发酵后，女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 遭中共重罚8.8亿元（人民币，下同）税款。有媒体称，除范冰冰外，范的好友
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵薇.html" target="_blank">
  赵薇
 </a>
 ，以及女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舒淇.html" target="_blank">
  舒淇
 </a>
 ，因参与综艺节目《中餐厅》薪酬超标，也被勒令退回4000万元。
 <br/>
 <br/>
 自央视前主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 曝光娱乐圈以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞逃税事件后，大陆影视娱乐演员及其公司的偷税漏税问题逐渐浮出水面。之后，中共国家广播电视总局（简称〝广电总局〞）对影视圈的整治力度不断加强。
 <br/>
 <br/>
 广电总局9日在网站公布《关于进一步加强广播电视和网路视听文艺节目管理的通知》，《通知》要求遏制影视圈追星炒星的不良倾向，严格控制综艺节目嘉宾片酬，加大网路剧治理力度。
 <br/>
 <br/>
 《通知》还提出惩罚措施，如果制作机构无正当理由或隐瞒不报嘉宾薪酬情况，一经查实，由所属协会上报广电总局，视情况采取暂停直至永久取消剧目播出、制作资质等处罚措施。
 <br/>
 <br/>
 至此，中国演员〝限薪令〞落地。早些时候市场传出〝限薪令〞后，就有媒体报导，因参与综艺节目《中餐厅》薪酬超标，女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵薇.html" target="_blank">
  赵薇
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舒淇.html" target="_blank">
  舒淇
 </a>
 退回4000万元。
 <br/>
 <br/>
 2017年4月，赵薇和黄有龙夫妇就曾因为空壳收购万家文化，遭中共证监会重罚，禁止两人5年内投入股市，并各处以30万人民币的罚款。
 <br/>
 <br/>
 消息曝光后，同月10日赵薇夫妇持有的多家股票股价大跌，其中投资31亿港币的阿里影业跌了1.56%，投资3.26亿港币的顺龙控股一度大跌6.25%，光这两支股票帐面上损失就高达约5858万人民币。
 <br/>
 <br/>
 赵薇早年因出演《还珠格格》走红，近年影视作品减少，聚焦在投资上，除了入股阿里巴巴影业，也曾花费约4404万人民币买下法国葡萄园酒庄，同时还经营影视、汽车销售服务等贸易。
 <br/>
 <br/>
 赵薇和黄有龙夫妇名下公司达33家，赵薇也因此被外界冠上〝女版巴菲特〞封号。
 <br/>
 <br/>
 赵薇的好友、同样因出演《还珠格格》而走红的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 ，因卷入娱乐圈〝阴阳合同〞案，消失于公众视线4个多月后，10月3日，中共税务机构公布案情，宣布范冰冰涉逃税案，缴纳约8.8亿元人民币税款，可免于牢狱之灾，至此范案终于落幕。
 <br/>
 <br/>
 然而范冰冰事件引发的巨震仍未消停，舆论场的各种声音仍是此起彼伏。大批中国影视圈人士深受范冰冰事件影响，行业税务地震仍暗潮汹涌。
 <br/>
 <br/>
 早在范冰冰失踪期间，中国经营报9月份报导说，一位不愿具名的娱乐圈老板透露，受范冰冰〝阴阳合同〞事件影响，娱乐圈发生大地震，除了公布的演员工作室税收税率从6%提高到42%外，政府口头传达要求各路演员10月份必须要补缴完税收。
 <br/>
 <br/>
 这位娱乐圈老板说，政府的要求是先去补缴，到10月底再抽查，这个阶段，娱乐圈一众演员，都在为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_补税.html" target="_blank">
  补税
 </a>
 忙得团团转，现在影视圈一见面，画面是这样的：〝还有票吗？我现在缺3千万。〞
 <br/>
 <br/>
 他说，除了被曝光的演员外，大部分艺人都有偷税漏税的问题。歌手更严重，前200名艺人和前50间影视公司都会中招。
 <br/>
 <br/>
 另一位娱乐圈人士透露说，所有艺人补的税加起来会是一大笔，演员工作室是用来避税的，大部分演员都有偷税漏税，演员就是为了逃税才开工作室。
 <br/>
 <br/>
 一位圈内人士分析，税不可怕，可怕的是蝴蝶效应，不但在影视娱乐圈爆发，更会席卷所有广告主。税务+限薪令会来一场风暴。
 <br/>
 <br/>
 在业内人士看来，这次税务局是下了决心，要整治影视产业链，因为新税制实行，已经有影视宣传公司倒闭了。
 <br/>
 <br/>
 曾经在中国一线娱乐节目担任导演的姜先生对德媒说，娱乐界新税制实行，是中共政府到娱乐圈〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_割韭菜.html" target="_blank">
  割韭菜
 </a>
 〞，总的来说是国库缺钱。
 <br/>
 <br/>
 他表示，当局处于中美贸易战，地方财政状况堪忧，崔永元的爆料，无疑给当局增加财政收入提供了绝佳的机会。之所以针对演员，是因演员们收入高，民怨也高，确实也存在逃税漏税的问题。
 <br/>
 <br/>
 分析认为，中共从演员大腕中割了最大的一大把〝韭菜〞，也令范冰冰消了大财免了大灾。有网民调侃，范冰冰成了〝共克时艰〞的〝功臣〞。
 <br/>
 <br/>
 美国之音分析认为，本次由崔永元担纲的这场娱乐大戏的主角，其前台幕后应该都是中共高层政军之间的角力。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：赵云）
