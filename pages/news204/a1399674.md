### 反腐幽默剧：芮成钢陪睡20高官夫人 因党内奸细轻判
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】中共吉林省副检察长吴长智日前落马。据信吴与前央视名人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芮成钢.html" target="_blank">
  芮成钢
 </a>
 以贪腐结案或轻判有关。此前消息称，芮曾与至少20多名中共高官夫人上床，芮还在审讯时〝态度跋扈〞。而吴负责调查芮案，涉嫌销毁芮成钢犯罪证据，使其罪名大大缩水。
 <br/>
 <br/>
 吉林省纪委监委13日通报，该省副检察长吴长智涉嫌严重违纪违法，目前正接受纪律审查和监察调查。
 <br/>
 <br/>
 56岁的吴长智是吉林磐石人，历任中共吉林桦甸市委副书记、政法委书记、吉林市委组织部副部长等。2016年11月，任中共吉林检察院副检察长、吉林检察委员会委员。
 <br/>
 <br/>
 吴长智是中共江派前中央书记处书记杜青林的外甥。最为人熟悉是他2016年与央视前主播
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_芮成钢.html" target="_blank">
  芮成钢
 </a>
 涉贪案件扯上关系。
 <br/>
 <br/>
 芮成钢因涉行贿受贿于2014年被检方带走。2016年4月，陆媒称，包括芮成钢案在内的央视腐败窝案将进入审理程序。
 <br/>
 <br/>
 根据当年起诉书，芮成钢被指向央视财经频道副总监李勇行贿40万元人民币，而芮成钢个人受贿20万元人民币。另外，芮成钢被指还有洩密问题。
 <br/>
 <br/>
 而消息源透露，芮成钢的个人资产约有3亿人民币，而且相当部分已经转移到了海外。
 <br/>
 <br/>
 2016年港媒称，芮成钢案9月14日秘密开审，因为芮案涉泄密问题。而且案件证据出现大漏洞，使高层震怒并已下令追查。
 <br/>
 <br/>
 究竟是谁在为芮成钢通风报信和为芮成钢求请？
 <br/>
 <br/>
 央视官员透露，中共政法部门和纪委随后开始秘密调查，调查的一个最为严重的问题是，一些涉及芮成钢犯罪的卷宗材料哪里去了？由于重要卷宗材料神秘失踪，最后导致检方对芮成钢的指控和法院量刑都非常轻。
 <br/>
 <br/>
 消息称，当时负责芮成钢案件调查的正是刚落马的吴长智。由于吉林反贪局提供的芮成钢涉嫌犯罪的证据有瑕疵，检方无法用这些证据来指控芮成钢，使芮案定性和刑罚大缩水，最终芮仅被判囚6年。
 <br/>
 <br/>
 此前就有海外媒体披露，芮成钢案的调查和审理，一直受到某种〝神秘力量〞持续干扰，芮成钢在看守所享有中共副部级被审查人员的待遇，芮本人也因此有恃无恐。
 <br/>
 <br/>
 2017年2月，芮成钢被中共吉林省蛟河市法院一审判处有期徒刑6年。
 <br/>
 <br/>
 港媒曾引述消息来源称，与芮成钢有染的女性人数众多，至少20多名中共副部级以上高官的夫人曾和他有染。他被捕后，不少中共高官夫人动员各种力量想〝捞人〞。
 <br/>
 <br/>
 据称芮遭审讯时态度跋扈，威胁审讯人员说，手上握有影片。
 <br/>
 <br/>
 另有报导，芮成钢卷入中共前中办主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_令计划.html" target="_blank">
  令计划
 </a>
 案，亦与令计划之妻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷丽萍.html" target="_blank">
  谷丽萍
 </a>
 有染，多次乘飞机与令计划妻子谷丽萍幽会。
 <br/>
 <br/>
 另外值得一提的是，除谷丽萍在央视有情人外，其丈夫令计划也在央视有情人。
 <br/>
 <br/>
 一位晨练的大爷说：〝令计划这两口子，够绝的。一个央屎找情妇，一个央屎寻面首。这CCTV就是一个窑子。〞
 <br/>
 <br/>
