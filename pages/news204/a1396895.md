### 淫僧释学诚〝成魔之路〞曝光：明目张胆违戒收女尼
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】中共佛教协会会长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_释学诚.html" target="_blank">
  释学诚
 </a>
 早前因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诱奸.html" target="_blank">
  诱奸
 </a>
 多名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼姑.html" target="_blank">
  尼姑
 </a>
 下台，近日传出他在福州、莆田的地位，仍非常稳固，操控多家庙宇。港媒爆料，淫僧释学诚明目张胆违戒收女弟子等内情，释学诚的〝成魔之路〞，再一次被震撼曝光。
 <br/>
 <br/>
 香港《苹果日报》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_释学诚.html" target="_blank">
  释学诚
 </a>
 虽然已经辞职下台，但他在根据地福州、莆田的地位仍相当稳固，仍然操控多间庙宇。
 <br/>
 <br/>
 报导说，针对释学诚的95页举报信中称，释学诚长期以简讯挑逗女弟子，藉〝修法〞谈〝淫欲〞，想让出家女弟子以释学诚为信仰和情感上的唯一依靠，让女弟子主动献身，再以性深化精神控制，来达到其自创的〝男女双修〞核心精神。
 <br/>
 <br/>
 有举报人称，释学诚不仅多次发不雅简讯骚扰女弟子，还强行性侵高学历女尼。举报信形容，释学诚根本是〝淫僧成魔〞。被精神控制的女弟子，甚至甘愿带其他人上师父的床，来换取出国深造的机会。
 <br/>
 <br/>
 举报信中还详述了释学诚是如何一步步培养出不达标的女弟子。
 <br/>
 <br/>
 2013年，释学诚先后把7名在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京龙泉寺.html" target="_blank">
  北京龙泉寺
 </a>
 的专职女义工，调配到他的家乡福建仙游极乐寺，谎报她们已满足出家女尼〝式叉摩那六法〞中，学戒二年的规定，让她们成为〝冒牌比丘尼〞。
 <br/>
 <br/>
 这明显是违背戒律的，但女尼却认为〝那是师父对自己的无限慈悲〞。
 <br/>
 <br/>
 2014年约有30名女弟子受戒，成为释学诚收下的第一批女弟子；2015年共有47人，成为释学诚收下的第二批女弟子；2016年有78人；2017年有100人，〝龙泉寺〞的女尼队伍日益壮大。
 <br/>
 <br/>
 举报信中指控，这些快速出家的女尼，约有一半被陆续派到国内外近20个分院，出家女尼因此成了释学诚〝弘法〞的主力。举报信中提到6名受性骚扰的女尼，都在国外弘法，都是释学诚专门挑选出来的。
 <br/>
 <br/>
 现年52岁的释学诚是中共培养的最高级别政治和尚，拥有一大堆头衔。
 <br/>
 <br/>
 8月初，两位出身清华大学的工程博士，前龙泉寺都监释贤佳（刘新佳）、释贤启（杜启新），联手撰写了长达95页的检举文件。
 <br/>
 <br/>
 文件中举报释学诚利用密宗的〝男女双修〞等说法迷惑、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诱奸.html" target="_blank">
  诱奸
 </a>
 修学不深的女弟子，给出家女弟子发送言词不堪入目简讯。导致一名女尼自杀，至少有3位被外派的女尼精神失常。
 <br/>
 <br/>
 另外，释学诚主持北京龙泉寺期间，未经许可的5处房屋违章建筑总面积达约1.37万平方米，估计使用人民币数亿元，资金动向不明。
 <br/>
 <br/>
 世界各大媒体8月初广泛报导相关新闻，美国CNN更是在头条位置，斥责释学诚是〝披着袈裟的恶魔〞。迫于舆论压力，中共不得不做出回应。
 <br/>
 <br/>
 8月中旬起，释学诚陆续被辞去了佛教协会会长、常务理事、理事、龙泉寺住持(
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_方丈.html" target="_blank">
  方丈
 </a>
 )等职务。
 <br/>
 <br/>
 当时有佛教界知情人士透露，释学诚8月中旬已经被带返故乡福建福州，在一个小院〝闭门思过〞，禁止出国。
 <br/>
 <br/>
 而如今传出淫僧释学诚在福州、莆田的地位，仍非常稳固，操控多家庙宇。因此，释学诚的〝成魔之路〞再一次被媒体震撼曝光。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
