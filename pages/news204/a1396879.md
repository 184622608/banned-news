### 习近平低调南巡  专家析两大原因
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】中美贸易战火升高之际，外界期望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 趁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南巡.html" target="_blank">
  南巡
 </a>
 发表讲话，但习除了简单宣布大桥启用外，没有发表任何公开演讲。习近平此次南下为何如此低调，专家分析有两个原因。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 趁南下视察期间，23日主持了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 的开通仪式，中港澳及西方媒体均到场。外界起初预计，习近平会在仪式发表有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 40年的讲话，讲述未来发展的方向。
 <br/>
 <br/>
 但习近平只说了〝我宣布港珠澳大桥正式开通〞这12个字，仪式主要是粤港澳三地官员及副总理韩正的致辞，但韩正演说也没有提到〝改革开放〞相关的内容。
 <br/>
 <br/>
 22日习还考察了横琴新区粤澳合作中医药科技产业园以及格力电器公司。
 <br/>
 <br/>
 习近平强调，从大国到强国，实体经济发展至关重要，任何时候都不能脱实向虚。制造业是实体经济的一个关键，制造业的核心就是创新，就是掌握关键核心技术，必须靠自力更生奋斗，靠自主创新争取。
 <br/>
 <br/>
 习近平24日还前往
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深圳.html" target="_blank">
  深圳
 </a>
 前海经济区考察。不过，习此次南下及主持大桥揭幕礼的安排都是极为低调。6年前，2012年中共十八大后，习近平当选中共总书记不久，也曾南下广东，在深圳、珠海、顺德、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 、惠州等地视察。
 <br/>
 <br/>
 当时习近平表示：〝之所以到广东来，就是要到中国改革开放中得风气之先的地方，现场回顾中国改革开放的历史进程，将改革开放继续推向前进。〞
 <br/>
 <br/>
 此次习近平南下，正值中美贸易战令中共陷入内外交困之际。外界期望习在改革开放40年之际南下，或会在大桥开通仪式发表讲话，但最后他除了简单宣布大桥启用外，并没有发表任何公开演讲。
 <br/>
 <br/>
 有观察人士认为，习近平不这样做有两方面的原因，一是，他想保持一定的〝低调〞；二是，他不想与中共前领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南巡.html" target="_blank">
  南巡
 </a>
 做比较，因在中美贸易战背景下，及国内经济停滞下，习近平的任何说话都会被外界演绎，或会得出反效果。
 <br/>
 <br/>
 今年是中共改革开放40周年，但对习近平来说却是诸事不顺的一年，外交上有中美贸易战；国内有对经济方面的反对声音，再加上澳门中联办主任突然堕楼身亡、港珠澳大桥被香港反对派声讨是〝大白象工程〞，这都不是他发表讲话的好时机。
 <br/>
 <br/>
 时事评论员刘锐绍对BBC中文分析说，由于受到中美贸易战困扰，习近平不能着墨太多，因为不知外界又会如何解读，如果习在中美贸易摩擦上发表强硬讲话，或会被美国反击，但说得太委婉，又会失去国内士气。
 <br/>
 <br/>
 《香港经济日报》25日分析认为，中共四中全会即将召开，估计会议的主题是深化改革，落实三中全会提出的改革目标，同时对改革开放40年做一个总结。因此，今次南下考察的结果，也将会在四中全会中呈现。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
