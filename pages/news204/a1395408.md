### 只因一句把家业送给党 大陆网民被逮捕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】中共对网络言论的管制手段日趋紧缩。消息称，一名大陆网民因在网贴说：〝把家业送给共产党〞，就被当局处以行政拘留14日。还有人因私家车贴有〝宪政民主、新闻自由〞等遭逮捕。
 <br/>
 <br/>
 维权网报导，江苏南京68岁女子秦沪辉，因在自己私家车贴上〝宪政民主、新闻自由、司法独立、官员财产公示〞等标语，9月4日被南京警方以寻釁滋事罪刑拘，10月10日遭正式逮捕。
 <br/>
 <br/>
 此外，25岁河南省商县男子吴邦用10月12日在微信群中发文说：〝应该把自己的家业送给共产党〞，否则就是〝政治觉悟不高，背离国家和人民〞，〝党要对你进行教育，把你送到新疆再教育集中营进行马克思主义教育，谁不服试试〞。
 <br/>
 <br/>
 结果，吴邦用遭公安行政拘留。网上流传的一份当地公安局行政处罚决定书显示，吴邦用被商城县公安局〝以寻釁滋事罪行政拘留14日〞。
 <br/>
 <br/>
 而上海网民唐燕涛女士，也因在网上发布民主、人权，宣传揭露社会黑暗的图片，9月29日被上海市公安局宝山分局刑事拘留。
 <br/>
 <br/>
 中共当局严控和打压社会和网络言论的行动，已升级至近乎荒唐的趋势。
 <br/>
 <br/>
 据报导，浙江传媒学院副院长赵思运教授，在开学典礼的致辞中寄语大学生：要体现独立人格，具备自律、尊重与尊严的文化素养，具有批判精神和道义担当，肩负起国家和民族未来的重任。
 <br/>
 <br/>
 赵思运讲话中还提到，只有对国家民族满怀深沉挚爱的人，才会批评社会的阴暗面；只有怀揣光明的人，才会去发现和揭露生活中的龌龊，因为他们知道，这是一种责任，一种现代公民义不容辞的责任……。
 <br/>
 <br/>
 因上述言论，赵思运于10月12日被传媒学院党委给以党内严重警告。
 <br/>
 <br/>
 自由亚洲电台12日引述评论人士的话说，自由、宪政，甚至是社会担当都不能讲，根本上说，就是反自由，全面反对普世价值。由原来内部监控，到现在公开用行政手段、法律手段，中共已经走得非常远，这是一个纳粹化的过程，非常危险。
 <br/>
 <br/>
 有分析人士抨击，中共匪帮治国，严控舆论和言论，说明了政体内的官员内心极度恐惧不安，惶惶不可终日。如果一个政府没有把自己放在法律框架之内，党大于法，那么毫无疑问，这就是一个流氓政府，这个党就是一个打砸抢无恶不作的流氓团伙。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
