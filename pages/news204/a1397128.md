### 【禁闻】美提重启谈判条件 中共满足与否内斗定
------------------------

<div class="wysiwyg">
 【新唐人2018年10月28日讯】美国媒体日前披露，美国拒绝重启贸易谈判，除非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 拿出具体方案。专家认为，对中共而言，中美贸易战实质是政治战，能否答应美国的要求，完全取决于中共派别之间廝杀的结果。
 <br/>
 <br/>
 据《华尔街日报》10月26号报导，中美两国官员表示，如果北京不拿出具体方案，来解决华盛顿对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 强制技术转让和其他经济问题的抱怨，美国拒绝与中共重启贸易谈判。
 <br/>
 <br/>
 清华大学副校长、经济学家杨斌：〝我认为它(中共)可能答应一部分条件，因为现在国内这个经济状况它压力也很大，我觉得应该缓和，如果是明智的话。〞
 <br/>
 <br/>
 不过，美国海军学院历史系教授、东亚问题专家余茂春认为，逼迫在中国的外国公司转让技术，已经是中共的发家之道，它是不会让步的，但是中共这种公然违背基本贸易准则的行为，反过来会使中共越来越孤立。
 <br/>
 <br/>
 美国海军学院历史系教授、东亚问题专家余茂春：〝但是它不让步的话对美国也很有利，因为这种最基本的国际贸易准则中国不能遵守的话，世界上其他的国家都会转到美国这边来，现在这个事态已经在这样了。〞
 <br/>
 <br/>
 报导说，在美国9月中旬宣布对2,000亿美元中国进口商品加征关税之后，中方取消了访问华盛顿的计划，随后谈判一直处于搁置状态。自那时起，中方一直在寻求重新接触的机会，包括提议美国财政部副部长David Malpass重启谈判。美国官员称，在白宫贸易团队的支持下，Malpass拒绝了上述提议。
 <br/>
 <br/>
 一些听取了中方简报的人士称，对中共来说，提出正式方案存在几个风险。首先，这可能会暴露中共的谈判立场。其次，中共担心川普发推文或声明披露方案内容，藉此堵住中共的退路。
 <br/>
 <br/>
 报导称，中共的担心源于以往的教训。1999年中共进行加入世界贸易组织的谈判时，时任中共总理朱鎔基提出了一个包括重大让步和重组中国经济的方案，克林顿政府将朱鎔基提出的方案公布于众，以防止中共故态复萌。这件事让朱鎔基在国内遭到强硬派抨击。
 <br/>
 <br/>
 杨斌：〝党内已经是两派，关键是谁说了算，党内内部斗争，假如是左派说了算，它就会让得少一点，如果改革派占了上风，会让得多一点。, 所以中美谈判最大的问题就是党内的问题，不是让不让步的问题。很多稍微懂点儿经济常识，都知道应该让步，让步对中国更有利嘛。〞
 <br/>
 <br/>
 余茂春认为，其实，中美贸易战对中共来説，是政治战。
 <br/>
 <br/>
 余茂春：〝它（中共）让步了以后这个政权就有危机，你比如说美国脸书到中国去，中国（如果）不能控制这些外国的公司，那么这些社交媒体一旦在中国起飞了之后，老百姓就有自由言论的空间，就形成巨大的威力，所以说中国跟美国打的不是贸易战，而是政治战。〞
 <br/>
 <br/>
 杨斌认为，对于中国经济发展、国计民生来说，业界更多的人都希望北京当局能与美国妥协。
 <br/>
 <br/>
 杨斌：〝而且这个妥协，人家提出的条件也是对国家民族有好处的，只是对共产党的政治不利，触犯了它的一些政治规条，我们首先是爱国的，而不是爱党，所以西方是希望对国家有利的事就妥协，达成妥协对国家的经济有利，从而对人民生活有利，对国家融入世界体系有利，从而对国家最终走向民主自由有利。〞
 <br/>
 <br/>
 《华尔街日报》的报导还说，能否答应美国的要求，甚至可能会影响到川普和习近平在阿根廷G20峰会的双边会谈。
 <br/>
 <br/>
