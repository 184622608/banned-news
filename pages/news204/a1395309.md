### 传九华山老僧透玄机 江泽民闻言大惊
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】尽管中共宣传无神论，但中共前党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 自知罪孽深重，被曝到处走寺瞻观，拜各路高人给他出招儿。知情人透露，江曾秘访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九华山.html" target="_blank">
  九华山
 </a>
 ，有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高僧.html" target="_blank">
  高僧
 </a>
 对江点破
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天机.html" target="_blank">
  天机
 </a>
 ：中共亡于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_羊年生人.html" target="_blank">
  羊年生人
 </a>
 之手。
 <br/>
 <br/>
 翻查网上资料，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 是中共领导人中烧香拜佛最多的一个。中国佛教四大圣地，包括山西五台山、安徽
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九华山.html" target="_blank">
  九华山
 </a>
 、四川峨嵋山、浙江普陀山，都有江泽民参观后留下的字迹。
 <br/>
 <br/>
 江泽民是踩着〝六四〞学生的鲜血上台的，上台后又犯下了残酷迫害修炼〝真善忍〞法轮功学员的罪行，香港杂志曾引述北京消息人士透露，江自知罪孽深重，深恐难逃千层地狱的惩罚，退下来之后赶着在家抄《地藏经》，并到九华山去拜地藏菩萨。
 <br/>
 <br/>
 但是，所有这些都无法得到神的佑护，挽回他被清算和失势的命运。
 <br/>
 <br/>
 早前，原广西桂林电视台记者曾节明在网络上发表题为：〝十九年前在广州的异闻：中共国亡于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_羊年生人.html" target="_blank">
  羊年生人
 </a>
 之手〞一文说：1998年3月，他以桂林电视台新闻记者的身份，被指派随桂林市委、市政府考察团赴广东考察，带队的是市委副书记高雄。
 <br/>
 <br/>
 在广州，高雄一行受到广东省政府宴请，席间乘着酒兴，一位省级高官的秘书竟洩露〝国家机密〞，大谈江泽民到处求神拜佛的事。
 <br/>
 <br/>
 他说，江泽民几乎每年都要去名山古刹拜佛，1997年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 死后，老贼终于多年媳妇熬成婆，大权独揽成真，高兴得涕泪横流，快感之余，专程秘访九华山，让某
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高僧.html" target="_blank">
  高僧
 </a>
 测一测自己的帝运，及中共气数，看看自己能滋润多久。
 <br/>
 <br/>
 结果高僧只说破一点：中共亡于羊年生人之手。
 <br/>
 <br/>
 文章说，老贼闻言大惊，认为〝羊年生人〞即为1919年生人（土羊）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵紫阳.html" target="_blank">
  赵紫阳
 </a>
 ，而深忌之，从此大大强化了对赵紫阳的软禁，严防赵紫阳东山再起。
 <br/>
 <br/>
 作者说，邓小平活着时，还允许软禁中的赵紫阳，在专人陪同下，到全国各地旅游散心、打打高尔夫球，并允许地方领导接待和陪同，只是严禁报导。
 <br/>
 <br/>
 但江泽民从九华山回来后，即下令不准赵紫阳离开北京，要打高尔夫球只能在北京打；且老贼还下令在赵紫阳的住宅——北京富强衚衕六号安装了铁丝电网，加派了武警防守。
 <br/>
 <br/>
 文章说，有道是〝人算不如天算〞，显然，灭亡中共国的人羊年人并非耄耋之年的赵紫阳，而是另有其人，江泽民是枉费心机。
 <br/>
 <br/>
 奇特的是，从历史来看，共产党政权的危亡，的确与羊年人有不解之缘：
 <br/>
 <br/>
 羊年生人赵紫阳，在1989年差一点就亡了中共；而苏共及几乎整个东欧共产党阵营，实亡于戈尔巴乔夫之手，戈尔巴乔夫和叶利钦都是羊年生人（1931年出生，金羊）。
 <br/>
 <br/>
 羊年生人，就如共产党政权的克星一样！
 <br/>
 <br/>
 事实上，有关中共解体、结束的许多预言一直流传民间。
 <br/>
 <br/>
 2002年6月，贵州省平塘县掌布乡发现一块距今2.7亿年的〝藏字石〞。500年前崩裂的巨石断面内惊现六个排列整齐的大字〝中国共产党亡〞。
 <br/>
 <br/>
 2003年12月5日至8日，中科院专家组对〝藏字石〞进行了实地考察，结论是：〝藏字石〞上未发现人工雕凿及其他人为加工痕迹，堪称世界奇观。
 <br/>
 <br/>
 贵州省平塘县掌布乡发现的〝藏字石〞，预示着中国共产党即将覆灭，警示中国人退出中共党团队保平安。中国共产党亡，而中国不会亡，神佑炎黄子孙，神佑大中华，届时神州大地将迎来万年未有之大变局。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：赵云）
