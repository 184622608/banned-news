### 陆学者：美国不会松绑 北京只能调整衰退节奏
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】中共政治局公开承认经济下行压力，似乎令体制内学者发言尺度也有所放大。日前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国社科院.html" target="_blank">
  中国社科院
 </a>
 学者公开表示，希望
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 出问题或者给中方松绑，这都是幻想；中国经济衰退无可避免，当局能调整的只是〝收缩节奏〞。
 <br/>
 <br/>
 发出该言论的是中国社会科学院金融实验室主任、博士生导师、天风证券首席经济学家刘煜辉教授。
 <br/>
 <br/>
 11月12日，他在接受编辑部设在北京的多维网的专访时，重点谈及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民币汇率.html" target="_blank">
  人民币汇率
 </a>
 的问题。他说，不是北京政府要〝保7〞，而是因为受困于全球化产业结构，人民币汇率已经没有了弹性空间。
 <br/>
 <br/>
 刘煜辉解释称，由于北京在国际上推行〝一篮子货币〞对抗美元，许多背后支持北京的国家，其生产和消费都因此已经和人民币高度相关。所以，人民币在替这些国家抵挡着美元升值的压力。如果人民币这时候贬值逃跑，这些国家就都会翻脸，〝群殴〞和孤立北京。
 <br/>
 <br/>
 因此在北京推动全球化，而遭遇美国抵制对抗的关键时刻，人民币根本就不敢动。
 <br/>
 <br/>
 他表示，人民币这个名义汇率不能再动，但中国国内经济状况不断滑坡，导致人民币的实际汇率（即人民币的实际价值）正处在一个不断贬值的通道里。名义汇率和实际汇率的差值越来越大，中国经济需要承受的痛苦就越来越大。
 <br/>
 <br/>
 最终这个差值会以通货膨胀，即人民币购买力对内贬值的方式呈现出来。表现最突出的就是中国房价的一路飙升。因此大家都不愿投资实体经济，经济也就开始下滑。
 <br/>
 <br/>
 刘煜辉表示，如果〝可能的话〞，北京政府也想通过〝挤出泡沫〞、〝去槓杆〞的政策，去提高人民币实际汇率，但是经济必须要接受一个衰退的过程。而实际上中国经济已经迈入衰退下行，汇率和利率压力会逐步释放，各种经济指标目前都处在一个收缩的状态。
 <br/>
 <br/>
 他说，目前的经济大趋势已经没有办法回头了。当然在整个衰退的过程中，当北京政府在经济风险释放集中的时候，政府财政受不了的时候，可能会推出一些改革或减税措施，调整一下〝节奏〞。
 <br/>
 <br/>
 为了说明中共政府保持人民币名义汇率〝稳定〞的大环境，刘煜辉还特别提到，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济.html" target="_blank">
  美国经济
 </a>
 不会出现突发的危机。
 <br/>
 <br/>
 他说，目前是美国右翼执政，摒弃极左政策，强调个体效率，对资本比较友好。这使得美国经济已经走上一条高效率的道路，这个过程中一般不会出现什么意外。
 <br/>
 <br/>
 刘煜辉指出，指望美国经济出问题，或者是希望美国主动妥协，给中国经济松绑，这都是应该尽早摒弃的幻想。中美两国目前正处在一个根本对立的历史关口。
 <br/>
 <br/>
 他说，北京政府应该着手国内，经历〝痛苦〞和〝代价〞，取得〝改革的成功〞。这个表态，迎合了北京〝贸易战不妥协〞、〝推进改革〞的立场。
 <br/>
 <br/>
 不过，一些海外学者认为，在内忧外患之下，中国经济的这次大衰退，或成为中共经济崩溃、政权解体的先兆。
 <br/>
 <br/>
 此前，中国学者贺江兵就曾断言，贸易战会轻易引爆中国内债炸弹，刺破楼市和金融泡沫，中国经济数月后就将进入〝崩盘模式〞。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
