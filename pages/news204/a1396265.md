### 【禁闻】中共党媒：不屑对美以牙还牙 评：已无牙
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】近日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 媒体突然发文宣称，不屑于对美以牙还牙、直接报复中美贸易争端。外界评论认为，这一充满阿Q精神的文章，充分显示了中共当前穷途末路的处境。
 <br/>
 <br/>
 19号，中国大陆官媒《人民日报》发表一篇署名文章，宣称对美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易制裁.html" target="_blank">
  贸易制裁
 </a>
 ，中方已经不屑于实施〝以牙还牙〞的直接报复，而是努力释放自身潜力，渡过难关。
 <br/>
 <br/>
 在此之前，《人民日报》有关中、美贸易冲突的文章，出现的标题是《打，奉陪到底！》、《这场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，中国有底气打赢！》、《如果美方任性妄为，中方必将依然亮剑》、《坚决有力，打痛美方》等。
 <br/>
 <br/>
 但最近这篇文章，除了不忘继续为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 的经济模式进行辩护之外，还号称对美国已不屑于〝以牙还牙〞。
 <br/>
 <br/>
 网上有评论认为，中共对美国的制裁已经〝无牙可还〞，现在开始为缴械服输找下台阶，同时又像是对所谓〝自力更生〞的宣言。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝中共高层想‘自力更生’，其实是它死不退让、但又面对重重危机、无路可走之下，无奈的选择，根本就不是文章写的那种无所谓的心态。对中共的宣传要反向理解，这种表面上的‘云淡风轻’，其实是一种阿Q心态，吃不到葡萄就假装不屑于吃。〞
 <br/>
 <br/>
 据了解，中国国家主席习近平和美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 暂定11月29日，于20国集团（G20）峰会期间会晤。
 <br/>
 <br/>
 川普10月11日接受美国《福斯新闻》访问时表示，关税政策已对中国经济造成很大的冲击，他说：〝我不想这么做，前提是他们必须来谈判桌。〞
 <br/>
 <br/>
 华裔加拿大籍作家盛雪分析指出，多年来，中共藉助于所谓的经济强势发展，在国际社会得到发言权，西方国家很多政府、政治人物出于发展经济、取得政绩的考虑，都陷入了中共的这个经济圈套当中。但实际上，中国的经济发展是不可持续的，是以付出了巨大的人文、道德、信仰、环境等一系列灾难性后果为代价的。
 <br/>
 <br/>
 华裔加籍作家盛雪：〝就是中共现在遇到了问题。中共的经济真正遇到了问题，而且中共现在应该说已经是一个黔驴技穷的处境。〞
 <br/>
 <br/>
 盛雪谈到，这些年来，中共疯狂的、不顾一切的开发掠夺，已经远远超出一个社会所能够承担的底线，这必然会引起社会的强烈反弹。因此，中共现在已经八面受敌。而《人民日报》的这篇文章在强词夺理的同时，也透露出了非常明显的虚弱。
 <br/>
 <br/>
 盛雪：〝那么这种虚弱是因为什么原因呢？是因为他们非常清楚，目前的这个困境是他们几乎是无法去回避的。〞
 <br/>
 <br/>
 《人民日报》的文章除了对中共政府进行鼓吹之外，还宣称：中国的问题需要亿万中国人一起去面对解决。
 <br/>
 <br/>
 盛雪：〝它们（中共）以为，它们可以继续从中国老百姓身上无休止的去掠夺，来对应美国总统川普对中共实行的这样一个经济打击。在这一点上，实际上由于中国老百姓的铺天盖地的、无时无刻的反抗行动，应该已经让它们感到了真正的穷途末路。〞
 <br/>
 <br/>
 近来，川普政府相继与加拿大、墨西哥、欧盟、韩国等签署、商讨自由贸易协议，逐渐形成了孤立中共的新的国际态势。
 <br/>
 <br/>
 10月4号，美国副总统彭斯更在华盛顿发表类似讨共宣言的划时代演讲。外界普遍认为，这宣告了美国和西方30多年来对中共绥靖政策的终结，预示川普政府将在政治、经济、外交、文化和军事等全方位，对中共政权展开打击。
 <br/>
 <br/>
 采访/陈汉 编辑/李谦 后制/周天
