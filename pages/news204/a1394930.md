### 大陆房市拐点到来？万科6折卖房一天售罄
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】秋意渐浓的10月，楼市的寒意也已袭来。大陆地产界龙头万科，近日在厦门的楼盘降至6折促销，200多套房一天售罄。业界分析，这一波降价潮预示大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房市拐点.html" target="_blank">
  房市拐点
 </a>
 已经到来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_楼市泡沫.html" target="_blank">
  楼市泡沫
 </a>
 即将刺破。
 <br/>
 <br/>
 万科对市场的反应向来敏感，9月28日至29日，万科在深圳大梅沙总部召开2018秋季例会。网上流传的现场图片显示，会议投影片（PPT）凸显三个大字〝活下去〞。
 <br/>
 <br/>
 早前，万科集团董事长郁亮已在内部会议上表示，万科未来以〝活下去〞为目标。暗示大陆房市即将进入寒冬。
 <br/>
 <br/>
 万科随即开启大幅降价促销模式。在〝十一〞长假前夕，位于厦门的万科白鹭郡楼盘以6折价促销，原价约人民币500万元的套房，降到278万到298万元。10月1日，该楼盘正式开盘，206套房一天全部售罄。
 <br/>
 <br/>
 厦门其他楼盘也纷纷效仿，降价促销，以便在〝冰凉〞的厦门房市中分得一杯羹。
 <br/>
 <br/>
 不仅仅是万科，包括恒大、碧桂园、泰禾、中粮等大型地产商都在加紧降价出货。9月，品牌房企恒大进行全国性的促销打折，住宅8.9折、商铺6折起；泰禾某地项目若购房人全款购房，房价给予7折优惠。
 <br/>
 <br/>
 目前大陆多个一线及热点城市的新房市场，均已出现降价潮。最近，环北京的燕郊楼市暴跌，从去年的每平米3万元人民币，跌至现在的1万5千元左右。
 <br/>
 <br/>
 安徽合肥一楼盘近日打响了合肥楼市降价促销第一枪，原价20000元/平米的房子，直降5000元/平米，甚至更多。引发此前购房业主的不满，前往售楼处维权。
 <br/>
 <br/>
 目前，上海、深圳、杭州、南京、合肥等城市都已经出现房价下行的苗头。中原地产首席分析师张大伟说，〝房价调整正从点到区域扩散。热点城市楼市逐渐降温，当前的购买力很难再支撑市场继续冲高。〞房市降温已成现实。
 <br/>
 <br/>
 广州一位大型开发商人士认为，现在只是一二线市场开始下行，接着三四线城市也会降价，大家现在都在想怎么〝过冬〞。
 <br/>
 <br/>
 此外，开发商对后市预期普遍不乐观。多数房企对于投资的态度更趋谨慎。近期，大陆各地土地流拍事件频现。2017年，北京卖地收入高达2800个亿。土地疯抢现象浓厚，但2018年拍地伊始，短短半年内就有七宗地块惨遭流拍，一贯以来寸土寸金的北京土地市场，被市场直接打脸！
 <br/>
 <br/>
 有业内人士分析指，流拍节奏加快，一二线城市〝地冷价跌〞现象明显，这些都证明楼市拐点已经到来。
 <br/>
 <br/>
 对此，万科董事会主席郁亮表示，过去20年盖的房子太多了，旺盛需求的时代、住房全面短缺的时代、房价快速上涨的时代已经结束。如果只靠房地产的上涨继续获利，以前怎么赚的钱，现在怎么还回去。
 <br/>
 <br/>
 融创中国的孙宏斌也对后市产生悲观态度。他在2018年中期业绩会上表示，房企的好日子已经到尽头。现在各房企都在掉头转行。
 <br/>
 <br/>
 有分析认为，导致近年来房价上涨的罪魁祸首是货币政策的放松，2015年、2016年是房地产狂飙发展的两年，正是国家货币最宽松、货币严重超发的时期：首套房贷款利率历史最低、首付比例下调至2成、棚户区改造货币化补偿投入了数万元……正是这些货币政策的支持加剧了楼市〝全民狂买〞的节奏，也把房价推向了最高点，房地产泡沫吹大到了一触即破的边缘。
 <br/>
 <br/>
 大陆民间智库智谷趋势认为，相对于P2P和股市，中共当局最担心的就是房地产崩盘，因为房地产扣住虚拟和实体的命门，上下牵扯太多，一旦出现非理性，就是一场山崩海啸。
 <br/>
 <br/>
 而历史经验表明，十次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融危机.html" target="_blank">
  金融危机
 </a>
 九次是房地产引起的。不论是发展中国家还是发达国家，最后都是由房地产崩盘引发的金融危机。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
