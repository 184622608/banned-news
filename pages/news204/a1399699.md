### 新疆法官黄云敏被判十年 妻子探监才得知
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】新疆原法官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄云敏.html" target="_blank">
  黄云敏
 </a>
 因为帮助访民维权被判刑十年，他的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_妻子.html" target="_blank">
  妻子
 </a>
 最近前往探监的时候才知道。黄云敏案被视为中国最离奇最难以置信的假案之一。黄云敏的原代理律师冉彤也与黄云敏解除了代理合约，原因不详。
 <br/>
 <br/>
 11月16日，自由亚洲电台引述消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄云敏.html" target="_blank">
  黄云敏
 </a>
 于2017年因为帮助访民被逮捕。11月15日，黄云敏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_妻子.html" target="_blank">
  妻子
 </a>
 前往新疆兵团监狱探望丈夫，黄云敏隔着玻璃与她会面时，戴着手铐脚镣，表示自己因向新疆百姓普及法律知识而被判刑十年，因此不服判决，并强烈抗议。之后，监狱的看守卡住黄云敏的脖子将他拉走，同时狱方还警告他妻子不得向外透露见到的情况。
 <br/>
 <br/>
 报导称，黄云敏的哥哥
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄敏青.html" target="_blank">
  黄敏青
 </a>
 向自由亚洲电台证实了黄云敏的妻子探望他的消息。黄敏青称，〝允许家属去见面，见了一分半钟，说到案情的时候，就说要上诉，（监狱方）就说网路不好就卡断了。〞
 <br/>
 <br/>
 黄云敏退休前是新疆生产建设兵团农三师图木舒克市的法官。2018年3月，他被当局以〝煽动民族仇恨、民族歧视罪〞刑事拘留，案件被检察院退回一次，后于9月19日在喀什农垦法院进行审判，当时法院没有进行宣判。
 <br/>
 <br/>
 黄敏青表示，那一次审判后，黄云敏家人至今没有获得任何判决通知。黄敏青认为，黄云敏案完全是新疆有关政府官员的一个报复行动。因为他给弟弟四川维权人士
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄晓敏.html" target="_blank">
  黄晓敏
 </a>
 提供了一些官员贪污的材料，后来黄晓敏告一个就下台一个。所以他们就是报复他。
 <br/>
 <br/>
 黄云敏的原代理律师冉彤，今年五月份被黄云敏解除了代理合约，原因不明。冉彤对自由亚洲电台表示，黄云敏的案件是中国最荒唐的冤案。
 <br/>
 <br/>
 〝一个法官，而且是一个处理过暴恐案件的法官，最后被戴上暴恐的帽子。一个好人，一个好法官，我只能这么说，不能为他继续代理案件，我非常遗憾，也非常痛苦。〞
 <br/>
 <br/>
 众所周知，中共不讲法律，制造了很多冤假错案，据海外媒体报导，江泽民和中共对法轮功的迫害完全是非法的。而江泽民集团及中共迫害法轮功的政策至今还在延续，而律师在法庭上为法轮功学员作无罪辩护，也受到中共的阻挠，如果任由律师在法庭上为法轮功学员作无罪辩护，则不仅会使原本审判法轮功学员的法庭由此变成揭露江泽民和中共非法迫害法轮功的正义讲坛，越来越多的旁听者因为明白了真相而觉醒，越来越多的当事法官受到良知的拷问，更重要的还在于它将使对法轮功学员的非法审判难以顺利进行和完成。
 <br/>
 <br/>
 而中共非法审判法轮功学员也往往是秘密进行，不通知家属，事后非法判刑，家属也全然不知。
 <br/>
 <br/>
 （记者欧阳静报导/责任编辑：李泉）
