### 杭州网贷平台〝牛板金〞投资人投告无门
------------------------

<div class="wysiwyg">
 【新唐人2018年12月03日讯】杭州为大陆P2P
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网贷平台.html" target="_blank">
  网贷平台
 </a>
 爆雷重灾区，被戏称为〝雷都〞，其中涉及86万人，总金额43亿元的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牛板金.html" target="_blank">
  牛板金
 </a>
 〞的受害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资人.html" target="_blank">
  投资人
 </a>
 ，12月2号，在网上投诉，杭州金融办监管缺失，平台爆雷5个多月，江干经侦不作为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受害人.html" target="_blank">
  受害人
 </a>
 投告无门，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 遭镇压。
 <br/>
 <br/>
 12月2号，杭州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网贷平台.html" target="_blank">
  网贷平台
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牛板金.html" target="_blank">
  牛板金
 </a>
 〞受害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投资人.html" target="_blank">
  投资人
 </a>
 在网上投诉，牛板金平台通过虚构标的自融，诈骗80多万出借人资金43亿元，至今无法收回。
 <br/>
 <br/>
 受害投资人3号告诉《新唐人》，〝牛板金〞在今年7月3号爆雷，立案4个多月，经侦至今没进展。
 <br/>
 <br/>
 牛板金受害投资人张先生:〝86万人投入，全国各地的都有，总共是43个亿，一点都没追回来，老板据说还两个在国外嘛，还没抓回来。很多的群里面都是绝望的喔，有的投了七、八十万的多嘛，我投了十几万，据说是有一个上吊自杀了。〞
 <br/>
 <br/>
 受害投资人多次去浙江省信访办、浙江省公安厅申诉，都被当地派特警镇压，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 至今都投告无门。受害投资人质疑，腐败官员包庇金融诈骗分子。
 <br/>
 <br/>
 牛板金受害投资人金先生：〝就是网上宣传嘛，说有比较正规化嘛，他们可能网上弄的虚假讯息。利息稍微比别的平台高一点嘛，百分七、百分八、百分之十，他没有超过百分之二十四嘛。去北京、杭州都有的，还有被关的啊。杭州市雷都，还是说骗子城市嘛，而且是个腐败城市啊。〞
 <br/>
 <br/>
 受害投资人还披露，牛板金平台曾获得国家公安部监制信息系统安全等级三级证书、增值电信业务经营许可证，有北京银行存管，上市公司佐力集团和风投春晓天泽、融数集团入股。更有杭州副市长姚峰、中国互联网金融协会会长为其站台。然而，杭州金融办监管缺失，平台暴雷之后又懒政怠政。
 <br/>
 <br/>
 新唐人记者熊斌、李沛灵采访报导
</div>
