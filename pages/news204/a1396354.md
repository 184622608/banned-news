### 坐等美国中期选举结果？路透：四中全会11月举行
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月23日讯】路透社日前消息指，中共十九届四中全会可能要在外部不确定因素〝稍有明确结果〞后，于11月上旬举行，这意味着可能会在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 结果出炉后再召开四中全会，讨论经济政策。
 <br/>
 <br/>
 <h4>
  四中全会或11月上旬举行
 </h4>
 <br/>
 路透社引述北京消息人士透露，中共一些地方要员已被要求要把11月份的时间留出来，预计四中全会将在11月上旬召开。
 <br/>
 <br/>
 消息又指，四中全会的主题就是〝落实三中全会提出的改革目标〞，但目前中美贸易谈判结果未知，北京希望在〝外部大的不确定因素稍有明确结果〞后，再确定后续的〝改革方案〞。
 <br/>
 <br/>
 报导说，这可能意味着，四中全会或在11月6日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 后召开。
 <br/>
 <br/>
 此前，港媒《明报》曾分析，日本首相安倍10月27日结束访华，11月5日习近平将赴上海出席首届进口博览会开幕式，而四中全会的会期一般是4天左右，因此四中全会的开幕日可能介于10月28日和11月1日之间。
 <br/>
 <br/>
 据中共官媒报导，今年11月5日-10日在上海举行首届中国国际进口博览会，是习近平去年5月在〝一带一路〞国际合作高峰论坛上宣布的。中共副总理胡春华强调，该会展由习近平〝亲自谋划、亲自提出、亲自部署推动〞。因此，习近平出现开幕式的可能性很大。
 <br/>
 <br/>
 因此，如果四中全会确定11月上旬召开，那么开幕日期或者是11月1日，或者就要等到11月6日之后。
 <br/>
 <br/>
 路透消息验证了此前外界的判断，即北京在〝豪赌〞美国中期选举，企盼民主党获胜，会扭转贸易战局势。但外界普遍认为，这是北京的误判，因为遏制中共已是美国两党的共识，中期选举不会对川普的贸易政策有大的影响。
 <br/>
 <br/>
 日前，外媒披露中共深圳政府官员到美国实地考察后出台的一份报告，其中更指共和党赢得中期选举和川普连任总统已基本没有悬念。
 <br/>
 <br/>
 <h4>
  或继续推行〝
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政左经右.html" target="_blank">
   政左经右
  </a>
  〞
 </h4>
 <br/>
 路透社报导还引述消息人士指，虽然四中全会要讨论的具体经济政策未定，但大方向仍然是〝经济上向市场放权，政治上更强调集权〞，因为〝深化改革〞已触及体制内利益集团，难度愈来愈大，只有通过〝经济上放权，政治上集权〞的方式才有可能推进〝后续的改革〞。
 <br/>
 <br/>
 另一位匿名官员称，现在的〝改革〞不能再着眼于某个点或某个领域的改革，而是要全盘考量，因为很多的改革就是受困于某一个点，从而影响全局。
 <br/>
 <br/>
 党媒近期多次喊话称改革已进入〝深水区〞，只剩难啃的〝硬骨头〞。前经济官员甚至公开承认，当局所谓的〝经济改革〞，目前只是习近平团队亲自关注的金融领域在唱〝独角戏〞，其它部门都在空喊口号。
 <br/>
 <br/>
 海外经济学者分析，中共畸形经济积累数十年的问题已经积重难返，再加上党内各大利益集团的极力抵制，北京所谓的〝经济改革〞其实已无出路。
 <br/>
 <br/>
 北京推行的〝政左经右〞，被指明显互相矛盾。中共的官僚体制决定了，政治上加强控制必然遏制市场主动性，因此改革成功的可能性很低。
 <br/>
 <br/>
 近期以来，中共国有资金接连并购或入股私企，当局通过强制在私企设置党委和推行〝民主管理〞来加强控制，令民间产生〝中共重回计划经济〞的恐慌。虽然当局一再强调〝继续支持私企〞，但同时也喊话〝做大做强国企〞，更无法扫除民间资本的担忧。
 <br/>
 <br/>
 《德国之声》日前引述一名中国金融业人士透露，虽然当局近期一再强调加快向中小企业发放贷款，但私企已经陷入经营困境，看不到前景，因此很少有人申请贷款。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
