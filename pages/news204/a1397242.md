### 【禁闻】重庆大巴坠江 随车监控信息仍未公开
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】观众朋友们大家好 欢迎收看中国禁闻
 <br/>
 <br/>
 1.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆大巴坠江.html" target="_blank">
  重庆大巴坠江
 </a>
 随车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监控信息.html" target="_blank">
  监控信息
 </a>
 仍未公开
 <br/>
 <br/>
 10月28号，重庆万州一辆公交车在长江二桥撞车后坠落长江，中共官媒报导说，有2名遇难者被打捞上来，还有15人失联。
 <br/>
 <br/>
 但据〝自由亚洲电台〞报导，有当地民众透露，已经打捞上来的两具尸体，是被撞的桥上行人，车里的人实际上一个都没有打捞上来。
 <br/>
 <br/>
 这位人士还说，官方公布的失踪人数，只是公交卡刷卡的记录，用现金上车没有刷卡的应该还有很多。车上乘客人数可能远超官方披露的数字。
 <br/>
 <br/>
 而且，公交车上都安装有与公司联网的随车监控系统，但直到目前，官方仍然没有公开随车监控系统的任何信息。外界质疑，官方在刻意隐瞒事故真相。
 <br/>
 <br/>
 目前，救援工作进展仍然缓慢，中共官方媒体10月29号声称，已确定坠江大巴位于大桥上游约28米、水深约71米处，但是打捞难度很大。
 <br/>
 <br/>
 2. 在华美企考虑推延投资或移出工厂
 <br/>
 <br/>
 中国南部美国商会10月29号公布的一份商业调查显示，因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 侵蚀商业利润，在中国南部运营的美国企业中，有超过70%的企业考虑推迟在当地进一步投资，并将部分或全部的制造活动迁移至其他国家。
 <br/>
 <br/>
 据〝路透社〞报导，这项调查共访问了219家公司，其中有三分之一是制造业的。64%的公司称，他们正考虑将生产线移至中国之外。
 <br/>
 <br/>
