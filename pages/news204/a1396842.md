### 奇闻！北京仿〝无印良品〞反告日商竟胜诉 获赔百万
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】北京一间毛巾工厂旗下的品牌店，店铺几乎完全翻版
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无印良品.html" target="_blank">
  无印良品
 </a>
 〞。中日厂商因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商标.html" target="_blank">
  商标
 </a>
 问题闹上法庭，不料想，日本正版〝无印良品〞店竟然败诉了，还须赔偿北京山寨版〝无印良品〞102万人民币。
 <br/>
 <br/>
 据香港媒体报导，中国的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无印良品.html" target="_blank">
  无印良品
 </a>
 Natural Mill〞和〝无印工坊〞，自2004年后，已变成北京棉田纺织品公司旗下的品牌，其原公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_注册.html" target="_blank">
  注册
 </a>
 的无印
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商标.html" target="_blank">
  商标
 </a>
 ，也自动转让给
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京棉田公司.html" target="_blank">
  北京棉田公司
 </a>
 ，有效期限至2021年。
 <br/>
 <br/>
 转让给北京棉田公司的山寨版〝无印良品Natural Mill〞店，其商标名称、字型、用色及内部装潢，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本.html" target="_blank">
  日本
 </a>
 的正版〝无印良品〞店，一点不差，几乎就是翻版〝无印良品〞，只是改成简体字。
 <br/>
 <br/>
 日本的〝无印良品MUJI〞曾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_控告.html" target="_blank">
  控告
 </a>
 中国山寨的〝无印良品〞侵权，但山寨的〝无印良品〞声称〝无印工坊 Natural Mill〞店名没有抄袭，〝无印良品〞只是该公司旗下一条生产线。
 <br/>
 <br/>
 在这之后，北京棉田公司反过来控告日本的〝无印良品〞侵犯注册商标，经历多年的诉讼，去年12月，中共法院裁定北京棉田公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胜诉.html" target="_blank">
  胜诉
 </a>
 ，日方须赔偿102万人民币，并在网店及门市发表声明。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-26/p9097302a524171949.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-26/p9097302a524171949-ss.jpg"/>
   <br/>
   <font size="-1">
    中国山寨版的无印良品。（微博图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 日本正版〝无印良品〞为什么败诉呢？让许多人感到极为吃惊。
 <br/>
 <br/>
 有网友细读判决书发现，中共法官认定，中国版的无印商标〝长期宣传和使用已具备极高的知名度〞，商标能继续使用，因此，法庭不认为山寨版无印良品，在中国境内侵权，所以判决日本无印良品败诉。
 <br/>
 <br/>
 而根据日本〝无印良品〞第二季年报，无印良品在中国市场上半年的收入，下滑了0.2%，截至今年8月底，日本〝无印良品〞在中国，总共开了235间门市。
 <br/>
 <br/>
 有分析指出，无印良品的生意，受到大量山寨店的威胁。据报近期中国黑龙江牡丹江，以及山东青岛的万达广场百货，又陆续新开了几间中国自创的〝无印良品Natural Mill〞。
 <br/>
 <br/>
 许多中国网友在参观完中国新开幕的无印良品后，纷纷批评〝产品漏洞百出〞、〝这么大的店，结果还是山寨的〞，还有网友嘲讽〝无印良品vs无良膺品〞、〝太厉害了山寨店还告得赢正牌店〞。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：戴明）
