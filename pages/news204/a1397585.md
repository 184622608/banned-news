### 军队二轮整肃已打响？东部战区司令员被点名
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】随着中共前军委委员张阳和房峰辉被开除中共党籍及褫夺上将军衔，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军队.html" target="_blank">
  军队
 </a>
 正面临新一轮整肃。最新消息称，目前最可能〝出事〞的是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东部战区.html" target="_blank">
  东部战区
 </a>
 司令员刘粤军，据称他已受到某种控制。不过相关消息没得到证实。
 <br/>
 <br/>
 10月31日，法广引述消息称，随着中共中央军委联合参谋部原参谋长房峰辉移送审判程序，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将开始的第二轮
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军队.html" target="_blank">
  军队
 </a>
 大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_清洗.html" target="_blank">
  清洗
 </a>
 中，首当其冲的是东部和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北部战区.html" target="_blank">
  北部战区
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_领导层.html" target="_blank">
  领导层
 </a>
 ，虽然未必是战区司令，但级别一定不低。
 <br/>
 <br/>
 而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东部战区.html" target="_blank">
  东部战区
 </a>
 司令员刘粤军，被指最可能成为下一个〝出事〞的上将。报导引述知情人士消息说，刘粤军的〝贪腐〞证据早已备齐，巡视组已接到指令安排〝查处〞。
 <br/>
 <br/>
 东部战区地域涵盖台湾以及与日本存在争议的钓鱼岛，司令员一职非常重要，现任司令员刘粤军是中共第十六届、十七届候补中央委员，第十八届、十九届中央委员。
 <br/>
 <br/>
 2015年7月，刘粤军晋升上将军衔。曾任中共驻澳门部队的首任司令员、兰州军区司令员。自2016年任中共东部战区司令员。
 <br/>
 <br/>
 报导说，刘粤军曾参加过1979年对越自卫反击战，是现任将军中具有丰富作战经验的，但都是较早之前的战功。刘粤军1989年升任师级参谋长，之后的提拔都是江胡时期完成。
 <br/>
 <br/>
 报导还透露，中共十八大以来，军队反腐力度超乎想像，已有2名中央军委副主席、8名上将、160名中将少将和18000名校级被指贪腐军官落马。但这可能只是冰山一角。
 <br/>
 <br/>
 报导中未提及刘粤军可能涉什么罪名〝出事〞，但提到不久前，军纪委给军队高级干部每人发一张表，要他们填写〝在提拔过程中是否行贿〞和〝是否为他人谋取利益而受贿〞等内容，并挨个谈话。
 <br/>
 <br/>
 若自曝罪行并退贜，不予追究刑责，只做纪律处分，若隐瞒不交代，一旦被查出将加重处罚。
 <br/>
 <br/>
 谈话不久，组织上即命他们回家接受监视居住，不得乱走乱动，听候组织处理。处理就是〝免职降职然后退下〞。报导估计全军受此处分的高级干部将超过被抓〝军中老虎〞，数量非常惊人。
 <br/>
 <br/>
 不过相关消息并没得到中共官方证实。
 <br/>
 <br/>
 10月24日法广就曾引述消息称，〝八一〞前原本有4人要升上将，结果取消，当时因为有120名将领正在接受反腐调查，军中已出现极大内乱。
 <br/>
 <br/>
 这一消息也未获官方证实。
 <br/>
 <br/>
 今年的〝八一〞中共建军节期间，习近平破例不升上将，一度引起不少猜测。
 <br/>
 <br/>
 8月19日晚，央视播出的中共中央军委党建会议上，习近平强调，部队要对中央绝对忠诚，全军要坚决维护中央权威和统一领导，反腐不会〝变风转向〞等。
 <br/>
 <br/>
 上述会议之后，习近平打虎风声趋强。
 <br/>
 <br/>
 港媒称，在清除了郭伯雄、徐才厚以及郭徐〝余毒〞后，又拿下了张阳、房峰辉两只〝军虎〞，接下来，军中还要继续清除张房〝余毒〞。
 <br/>
 <br/>
 8月下旬起，陆续传出，原总参情报部部长的东部战区副司令兼参谋长杨晖中将落马，以及西部战区副司令刘小午被传已拘查。还有传言称，南部战区政委魏亮和原广州军区司令徐粉林也被降级退役等。
 <br/>
 <br/>
 而上述军中高层与郭伯雄、徐才厚、张阳、房峰辉等人有某种关联。
 <br/>
 <br/>
 有分析认为，习近平8月份又一次发出反腐不会〝变风转向〞信号之时，就意味着军队新一轮的大整肃已拉开帷幕。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明）
