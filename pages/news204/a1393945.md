### 央视画面天安门城楼无毛像   节目未改照常播引发关注
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国殇日.html" target="_blank">
  国殇日
 </a>
 （10月1日）晚间，在中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
  央视
 </a>
 当晚的节目中，有网民意外发现舞台背景天安门城楼上的毛泽东画像〝没了〞，而该档节目不仅正常播出，怒骂央视〝亵渎毛〞的帖子亦被删除，引发外界关注。
 <br/>
 <br/>
 北京时间9月27日，中国官方按照惯例对北京天安门城楼上的中共前领导人毛泽东画像进行“换新”工作。
 <br/>
 <br/>
 10月1日晚间，在中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视.html" target="_blank">
  央视
 </a>
 第11频道播出的所谓纪念节目中，有网友惊奇的发现，节目中的背景，即天安门城楼中的毛泽东画像〝消失了〞。
 <br/>
 <br/>
 尽管一些听惯了中共〝一言堂〞宣讲的网友大呼央视的举动是〝明目张胆〞地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亵渎毛泽东.html" target="_blank">
  亵渎毛泽东
 </a>
 ，〝是一次政治错误〞，但类似的微博帖文很快竟被删除，相关节目也正常播完。
 <br/>
 <br/>
 中共此举引发外界关注和网民热议。有网民留言说：〝毛醉了，罪了，坠了；还是睡了，祟了，碎了？〞〝毛粪渣的像被移除，一帮走狗就跟死了亲爹似的！〞
 <br/>
 <br/>
 还有网民讥讽道：〝静候央视怎么说，央视记者怎么说〞、〝如此严重的政治错误，足以使央视台长、书记、总监撤职查办！〞
 <br/>
 <br/>
 更有网民一语道破：那些被中共欺骗的中国人〝多崇拜独裁的暴民（指毛泽东）！〞
 <br/>
 <br/>
 也有网民坦言：〝真要是宣布永久摘下天安门毛XX像，大部分中国人还是欢呼雀跃的。〞
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
