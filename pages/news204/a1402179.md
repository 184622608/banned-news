### 传任正非原定本月退休 女儿被捕打乱华为接班计划
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 长女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被捕的消息，登上全球各大媒体头条。有传闻称，现年73岁的任正非原拟定本月底正式退休，由孟晚舟接替担任华为集团CEO一职。不过，孟晚舟日前意外被捕，打乱了华为的接班计划。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
   孟晚舟
  </a>
  背景非同一般
 </h4>
 <br/>
 孟晚舟现任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 副董事长、全球首席财务官，她是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 与第一任妻子孟军的长女。任正非共有三任妻子，分别是孟军、姚凌、苏薇。
 <br/>
 <br/>
 外界传言，任正非是靠孟军才飞黄腾达的。孟军的父亲是原四川省副省长孟东波，与孟东波过往甚密的原四川省委书记杨超，曾担任过中共领导人周恩来的政治秘书。
 <br/>
 <br/>
 据网路传闻称，任正非将于今年12月31日正式退休，不再参与公司管理。与此同时，华为将废除目前的轮值CEO制度，自2019年1月1日起，由任正非女儿孟晚舟接替集团CEO一职。
 <br/>
 <br/>
 孟军与任正非还育有一子叫孟平。据传，孟平近日也将进入华为EMT，接替余承东担任消费者BG CEO。此前网上就有传言，任正非对于2016年度的利润水平不甚满意，痛批了余承东。
 <br/>
 <br/>
 根据华为公司的规章，EMT是华为日常经营的最高责任机构，受董事会委托执行华为的日常管理。EMT成员不多，属于华为最为核心的高层，一般都是在华为工作十年以上的元老。
 <br/>
 <br/>
 对于华为此项人事变动的真实性目前不得而知，不过业界人士普遍认为，任正非年事已高，并且身体健康状况每况愈下，退休是迟早的事情。不过，孟晚舟12月1日在加拿大意外被捕，恐打乱华为的接班计划。
 <br/>
 <br/>
 孟晚舟被捕源于华为被指控涉嫌向伊朗违规销售美国公司设备。路透社在2013年曾经报导，香港公司Skycom Tech涉嫌向伊朗最大的运营商，违规销售美国惠普公司的设备——惠普称它禁止向伊朗出售任何其产品，而孟晚舟担任Skycom的董事职位。
 <br/>
 <br/>
 根据报导，至少有13页Skycom的商业文件打着〝华为机密〞的字样并带有华为标志。但华为随后进行了否认。
 <br/>
 <br/>
 <h4>
  华为有中共军方背景
 </h4>
 <br/>
 华为被指有中共军方背景，创始人任正非于1974年入伍，后成为辽阳化纤总厂工程建设的工程师、副所长（技术副团级）。他曾在深圳南油集团下属一家通讯研究所担任副所长。1983年复员转业，1987年集资21,000元人民币创立华为公司，1988年任华为公司总裁。
 <br/>
 <br/>
 华为最初专门代理生产多家中共军队驻香港企业的小型交换机，后接获大量国企订单，发展为大陆制造能力最强的电信设备制造商。
 <br/>
 <br/>
 美国政论家伍凡曾表示，华为创始人任正非是军官，跟军方总参谋部关系密切。〝他的角色有点类似于香港凤凰电视台，也有军方的背景，分属舆论媒体、通讯网路两方面。任正非最高的目标为中共服务。所以他讲一句话：‘我拥护共产党等等’〞。
 <br/>
 <br/>
 2013年7月，美国中央情报局（CIA）前局长海登（Michael Hayden）曾表示，华为是中共当局的间谍，这是〝不争的事实〞。
 <br/>
 <br/>
 他告诉澳洲金融评论报，中共毫无节制对西方国家发动间谍行动，并且他深信华为与中共国家情报机构分享情资。
 <br/>
 <br/>
 2015年，华为成功跻身世界500强，成外与美国信息产业支柱微软、谷歌、苹果，以及韩国支柱产业三星抢占全球高新技术市场的中国企业。
 <br/>
 <br/>
 不过，在全球开始进入5G移动网路的关键时期，基于对国家安全的考量，华为的移动通信系统（5G）设备在美国、澳洲、新西兰、德国等多个国家被禁用。
 <br/>
 <br/>
 英国电信集团（BT）12月5日也表示，将在两年内从其核心4G网路中彻底移除华为设备，同时将华为排除出核心5G网路设备供应合同的投标厂商名单外。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
