### 央视女主播〝百亿家产〞案 关键视频证据曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】知名地产商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李贵斌.html" target="_blank">
  李贵斌
 </a>
 去世后，相关股权被其弟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李贵杰.html" target="_blank">
  李贵杰
 </a>
 全部转走，导致其妻、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_央视女主播.html" target="_blank">
  央视女主播
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐珺.html" target="_blank">
  徐珺
 </a>
 携子维权一事有了新进展。近日，一份总时长30多分钟的录像被作为关键证据提交法庭。
 <br/>
 <br/>
 《中国经营报》15日报导，光耀东方系企业创始人、知名地产商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李贵斌.html" target="_blank">
  李贵斌
 </a>
 ，于2017年1月25日病重住院，28日医院下病重通知，2月3日医院下病危通知，2月13日李贵斌病逝。
 <br/>
 <br/>
 根据其官网信息梳理，光耀东方系企业资产，其位于北京、天津、山东、河北等地的22处大型物业，保守估算也在100亿元之上。
 <br/>
 <br/>
 在李贵斌死后，其名下实名或隐名（他人代持）多家企业股权，被陆续转至其弟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李贵杰.html" target="_blank">
  李贵杰
 </a>
 名下。
 <br/>
 <br/>
 据知情人透露，上述股权变化过程中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐珺.html" target="_blank">
  徐珺
 </a>
 显然毫不知情，当时沉浸在丧夫之痛中的徐珺是经人提醒，才得知股权已经发生了变更。且事后李贵杰表态强硬，未向徐珺母子三人支付分文。
 <br/>
 <br/>
 为此，徐珺在山东、北京两地发起诸多诉讼，要求确认多家企业股权变更无效。
 <br/>
 <br/>
 近日，随着一系列案件的开庭，李贵斌生前最后一段视频被提交司法鉴定。
 <br/>
 <br/>
 10月13日，徐珺在微博贴出司法鉴定结果，在该视频录制前后时间段内（2017年2月3日、4日），李贵斌〝患有器质性精神障碍，根据《司法精神病学法律能力鉴定指导标准》属于限制民事行为能力。
 <br/>
 <br/>
 报导说，李贵斌在多个企业的相关股权，均在上述时间，被集中转移至其弟弟李贵杰名下，这导致徐珺与两个孩子〝零继承〞，随后，徐珺发起多个诉讼，要求确认上述转移无效。
 <br/>
 <br/>
 由于涉及多家公司，相关案件分别在山东、北京两地开庭，开庭时，一份总时长30多分钟的录像被作为关键证据提交法庭，以证明李贵杰受让李贵斌名下股权是〝合法的〞。
 <br/>
 <br/>
 该录像摄于2017年2月3日，画面中，李贵斌半卧病床，目光呆滞，注意力不集中，反应迟钝，不知道自己的姓名是什么、更无法正确签字。
 <br/>
 <br/>
 据知情人透露，该视频由李贵杰与律师等人用手机摄制而成，主要内容则是让已经被医院下达病危通知的李贵斌签署股权转让文件，将名下所有股权转至李贵杰名下。
 <br/>
 <br/>
 但作为李贵斌妻子、法定监护人的徐珺，对此并不知情，直到在法院起诉后交换证据时才第一次见到这份录像。
 <br/>
 <br/>
 然而对方作为证据提交的这份录像，却清楚地显示，李贵斌似乎并不记得自己的名字，反覆提醒，仍未能写对。在后来提交给工商部门的变更文件中，李贵斌2017年2月3日一天之内，〝出现〞在8家光耀东方系企业的股东会上，会场分别在北京、聊城、冠县三地。
 <br/>
 <br/>
 这些会议文件的核心内容，则是将李贵斌名下股权，以超低对价转给弟弟李贵杰。
 <br/>
 <br/>
 据律师介绍，所谓的〝股权转让款〞在李贵斌的个人银行卡账户上仅仅待了十几天，便于李贵斌去世7天之后，被人瞒着徐珺偷偷转走，至今经徐珺多次索要也未归还。
 <br/>
 <br/>
 徐珺母子三人作为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_第一顺序继承人.html" target="_blank">
  第一顺序继承人
 </a>
 ，实则变成了近乎清零的出局。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
