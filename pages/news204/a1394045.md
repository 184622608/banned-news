### 日媒曝北京一家公司崛起内幕 替中共监控14亿人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】中国北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旷视科技.html" target="_blank">
  旷视科技
 </a>
 称，其技术已逮捕超过万名的通缉犯，并且炫耀：AI（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人工智能.html" target="_blank">
  人工智能
 </a>
 ）发展最重要的元素就是资料。日媒说，该公司使用庞大的14亿人口的个人资料快速崛起，由最初3人创业，7年内员工已增至1600人。
 <br/>
 <br/>
 日媒《产经新闻》报导，日本经济团体联合会于9月11日访问中国，前往被称为〝中国的矽谷〞的北京中关村视察
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_旷视科技.html" target="_blank">
  旷视科技
 </a>
 。该公司是由3名清华大学毕业、当时才23岁上下的年轻人独创，7年来员工已成长至1600人。
 <br/>
 <br/>
 报导说，旷视科技被誉为〝中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人工智能.html" target="_blank">
  人工智能
 </a>
 独角兽〞，并拥有世界上最大脸部辨识开放平台〝Face++〞，该公司将脸部辨识系统提供给各地的公安部门，以及用于金融业的个人认证技术。
 <br/>
 <br/>
 一名女主管骄傲地炫耀：〝我们的技术已经解决了5千起以上的犯罪案件，逮捕了超过1万名通缉犯〞，她还说：〝AI（人工智能）发展最重要的元素就是资料，而这正是中国的强项〞。
 <br/>
 <br/>
 报导表示，如果能获得中共的承诺，就能无视隐私问题，直接使用庞大14亿中国人的个人资料。政府全力支援，会下金鸡蛋的企业家，而企业家也必须要充分展现对中共的忠诚，这名女主管正是中共商务部出身。
 <br/>
 <br/>
 事实上，这种
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人脸识别技术.html" target="_blank">
  人脸识别技术
 </a>
 ，在2016年G20（二十国集团）峰会期间，已经由旷视科技和其它几家公司与杭州警方合作，在一个多月里有超过60人被拘留。
 <br/>
 <br/>
 中共运用新技术管控民众不是为了行使政府的管理职能，而是在严密
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监控.html" target="_blank">
  监控
 </a>
 老百姓的一举一动，以达到自己的一党专制统治。
 <br/>
 <br/>
 安徽省检察院前检查官沈良庆表示，中共采用大数据、人脸识别技术等窃取百姓隐私，让人活在玻璃屋中，是很可怕的，就像奥威尔的小说《1984》描述的那样——〝老大哥正在监视着你〞。
 <br/>
 <br/>
 沈良庆说：〝在中国，真正碰到治安问题，尤其涉及中共统治问题时，就让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摄像头.html" target="_blank">
  摄像头
 </a>
 坏了，比如雷洋事件。为了维护其政权，这些摄像头不是针对中共自己的，而是针对老百姓的。其最主要目的是完成镇压职能。〞
 <br/>
 <br/>
 中共公安部于2004年启动〝天网工程〞，在全中国的主要城市和交通枢纽安装大量监控摄像头。〝天网〞采用人工智能（AI），其中一项开发中的技术，可让中共随时随地识别14亿中国人。
 <br/>
 <br/>
 央视的节目中承认遍及各大城市的摄像头，构成了〝中国天网〞，并美其名曰是〝守护百姓的眼睛〞。
 <br/>
 <br/>
 那么中共的这个〝天网〞到底有多厉害呢？
 <br/>
 <br/>
 中共自称这个监控系统可以读取车牌，识别人脸、步态，判定年龄、性别、身高、族裔，甚至可以知道你的亲属是谁，你经常跟什么人在一起。说白了，只要你被当局确定为目标，不管你到哪儿，你都在警察，或者说当局的视线范围之内。
 <br/>
 <br/>
 大陆网络媒体〝AI前线〞报导，〝中国天网〞去年已部署1.7亿台摄像头，政府计划到2020年，在全国安装另外4亿台摄像头。
 <br/>
 <br/>
 该系统可让镜头随时随地识别捕捉到的任何一个人，并且被部署在新疆与西北的阿富汗接壤处，以及北京的天安门广场，有些可以辨识15公里以外的人物，甚至可以在夜间、雪地，穿过烟雾发现目标。
 <br/>
 <br/>
 不过，报导说，天网工程目前正面临巨大的挑战。这个全球最大，部署在中国的视频监控系统关键零部件供应，严重依赖西方国家。
 <br/>
 <br/>
 参与天网项目的科学家和工程师们透露，由于中美贸易和其他国家收紧贸易制裁，美国和部分欧洲国家，已停止向中国出口重要部件。中共号称世界最大规模的视频监控系统〝天网〞工程也面临瘫痪。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
