### 抗日雷剧再上新高度 竹筒迫击炮发射〝木陀螺〞炮弹
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 为了给国人洗脑，近年不断翻拍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗日雷剧.html" target="_blank">
  抗日雷剧
 </a>
 ，但剧情及所谓的〝兵器〞却愈发令人感觉荒诞和夸张。不少网友叹息〝难怪抗日打了那么久〞、中共现在〝还这么蠢〞。
 <br/>
 <br/>
 这些所谓的〝秘密武器〞多半被认为是毫无历史根据和科学来源的〝幻想武器〞，由于其过于〝落后〞而被网民指，就是在侮辱人民的智商。
 <br/>
 <br/>
 比如〝竹筒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迫击炮.html" target="_blank">
  迫击炮
 </a>
 〞，在剧中号称是〝不亚于小鬼子迫击炮的国产武器〞。网友调侃称，该〝武器〞就连炮弹都是那么乖巧可爱，你确认这不是小朋友们玩的木陀螺吗？
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-7/p9068562a880658108.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-7/p9068562a880658108-ss.jpg"/>
   <br/>
   <font size="-1">
    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
     中共
    </a>
    〝
    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗日雷剧.html" target="_blank">
     抗日雷剧
    </a>
    〞中的〝
    <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_竹筒迫击炮.html" target="_blank">
     竹筒迫击炮
    </a>
    〞，其炮弹确认不是小朋友们玩的木陀螺吗？（网络图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 根据公开资料，迫击炮（Mortar）是一种炮身短、射角大（一般为45°～85°）、弹道弧线高、以座钣承受后座力、发射带尾翼弹的曲射火炮。由于它可以〝迫近射击〞，并因此而得名。
 <br/>
 <br/>
 迫击炮虽然结构简单，却经久不衰，用途广泛，且无死角打击。1904年日俄战争的旅顺攻城战中，日军官兵离俄军战壕很近，普通火炮和机枪无法发挥威力，俄国军官便发明了这种迫击炮；至1927年有炮身和炮架刚性连接的版本之后，迫击炮就没有再更进过外观。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-7/p9068571a488142536.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-7/p9068571a488142536-ss.jpg"/>
   <br/>
   <font size="-1">
    〝迫击炮〞结构简单，且打击无死角，因此战争中被广泛应用。（网络图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 为什么迫击炮的外观没有随着时代变化而改变呢？这主要涉及到它的工作原理。传统的迫击炮和其它火炮不一样，它没有膛线，采用的是古老的从前装弹方式，利用炮弹的自身重量下落激发炮底的击发针，然后点燃炮弹的飞行火药，使炮弹飞出去。
 <br/>
 <br/>
 所以，迫击炮炮弹一般都很特殊，都有尾翼，而且不象其它炮弹一样只有〝底火〞，迫击炮弹的〝底火〞是包裹着炮弹的，只有这样才能保证飞得又远又笔直。
 <br/>
 <br/>
 正是因为迫击炮的结构简单，便于携带（轻型迫击炮一般在5kg至20kg之间，只需要两个人操作即可），所以一直都活跃在世界各地战场；除了一般军队外，连一些游击队都可以装备。
 <br/>
 <br/>
 简言之，〝竹筒迫击炮〞的工作原理实际上就是一个弹弓，将火药包射出去。而中共抗日雷剧中的所谓〝竹筒迫击炮〞，其式样无论如何都不可能存在。
 <br/>
 <br/>
 因此，有网民戏称其为〝爆竹〞，还有网友说：〝怪不得小日本打下了大半个中国，直到现在中国人还这么蠢。〞
 <br/>
 <br/>
 近年来，中共翻拍的抗日雷剧多种多样，不少戏剧性画面，如〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_手撕鬼子.html" target="_blank">
  手撕鬼子
 </a>
 〞、〝九阴白骨爪〞、〝化骨绵掌〞、〝轻功〞等，更令网友无言以对。
 <br/>
 <br/>
 有网友表示，〝如果鬼子如此不堪一击，怎么大半个中国都给鬼子占领？真是无知……〞
 <br/>
 <br/>
