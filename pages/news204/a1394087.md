### 媒体拆解范冰冰案：涉党内权斗 为贸战补镬
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】喧嚣多日的中国影星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 偷税漏税案，以近9亿元人民币的巨额罚款结案，再度引起舆论关注。港媒分析，北京或借整顿娱乐圈敲打背后的利益集团，要他们与最高当局〝共克时艰〞，同时也为贸易战下节节下滑的经济补镬。
 <br/>
 <br/>
 香港《苹果日报》10月4日文章指，对于偷漏税严重的中国大陆娱乐圈，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 案具有典型的震慑意义。而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中南海权斗.html" target="_blank">
  中南海权斗
 </a>
 传闻甚嚣尘上，该事件也很容易被猜测或涉中共权力斗争。
 <br/>
 <br/>
 文章称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国娱乐圈.html" target="_blank">
  中国娱乐圈
 </a>
 背后多涉及中共军方前高层家族，例如华谊兄弟老板被传是军二代，保利华亿传媒总裁也有军方背景，特别是涉及中共前常委曾庆红胞弟、〝文化大亨〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾庆淮.html" target="_blank">
  曾庆淮
 </a>
 的利益。
 <br/>
 <br/>
 因此，外界很容易联想，北京当局整顿娱乐界，明显在敲打这些利益集团，要他们在当前的经济危局中〝不要添乱〞，而是与北京〝共患难〞。
 <br/>
 <br/>
 另外，当局借重罚范冰冰威吓娱乐圈补缴税款，同时追究税务机关相关人员的责任，加上国税地税体制改革，显示税务部门要立威，迫使全国税务系统下狠手查税，为受中美贸易战影响而下滑的经济补镬。
 <br/>
 <br/>
 新华社日前报导，范冰冰因〝阴阳合同〞行为涉及逃漏税，被处罚补缴税款及滞纳金、罚款共8亿8394.6万元，若在限期内缴清则可免于追究刑责。范冰冰随后也第一时间发出道歉声明，表示会努力补缴罚款。至此，范冰冰案基本尘埃落定。
 <br/>
 <br/>
 对于范冰冰免于刑责，舆论界有很多争议。有媒体指出，不追究刑事责任符合中共法律，范冰冰的〝免死金牌〞，就是2009年对《刑法》201条〝逃避缴纳税款罪〞的修改。
 <br/>
 <br/>
 根据修改后的条款，对于逃税金额巨大的嫌犯，如果只是〝初犯〞，则只要补缴〝应纳税款〞，就可以〝不予追究刑事责任〞。
 <br/>
 <br/>
 2009年时虽然是胡锦涛当政，但江泽民还在〝垂帘听政〞，〝闷声发大财〞仍在大行其道。此时政法系统修改刑法，为逃税者网开一面，也给江家旗下贪官、包括江的铁杆曾庆红家族捞取利益提供了方便。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
