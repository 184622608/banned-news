### 【禁闻】〝川习会〞落幕 党媒过滤重要信息
------------------------

<div class="wysiwyg">
 【新唐人2018年12月02日讯】
 <b>
  〝
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
   川习会
  </a>
  〞落幕 党媒过滤重要信息
 </b>
 <br/>
 <br/>
 全球瞩目的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞12月1号落幕，中共喉舌〝新华社〞当天抢先发布消息，但过滤重要信息。报导只说中美达成共识，停止课征新关税，却只字不提暂停升级关税的前提条件和中方让步的具体内容。
 <br/>
 <br/>
 根据美国白宫稍后发出的声明，美国只是同意暂时不增加新的关税，并设定了90天的谈判期。如果双方在未来90天内未能就强制技术转让、知识产权保护、非关税壁垒、网络入侵和网络盗窃、服务业和农业等达成协议，美国仍将提高关税。
 <br/>
 <br/>
 另外，中共还同意从美国购买数量庞大的农业、能源、工业和其它产品，并将每年导致数千美国人死亡的〝芬太尼〞列为〝管控物质〞。这意味着往美国销售芬太尼的人将面临中共的严格处罚。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
   G20峰会
  </a>
  中共收买警察对
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
   法轮功学员
  </a>
  施暴
 </b>
 <br/>
 <br/>
 世界各国领导人在阿根廷首都布宜诺斯艾利斯举行二十国集团（G20）峰会期间，中共使领馆渗透西方社会、破坏民主自由的行为再次被曝光。
 <br/>
 <br/>
 11月29号，在中国国家主席习近平下榻的酒店附近，阿根廷法轮功学员在栅栏上挂出两条写有〝停止迫害法轮功〞、〝法办江泽民〞的横幅，进行和平请愿。
 <br/>
 <br/>
 但有目击者表示，一名中共大使馆官员和一名便衣警察交谈后，突然就出现几名警察，上前强行取下法轮功学员的横幅。并谎称法轮功学员的背包里有炸弹，以暴力强行非法拘捕法轮功学员，直到第二天下午才释放。
 <br/>
 <br/>
 <b>
  G20峰会领袖公报 首次同意改革
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_WTO.html" target="_blank">
   WTO
  </a>
 </b>
 <br/>
 <br/>
 12月1号，G20峰会发布领袖公报，内容大部分支持美国主张，包括删去保护主义的措辞、改变全球贸易体系等。
 <br/>
 <br/>
 据报导，各国首脑同意于2019年6月在日本大阪举行的峰会上继续讨论世界贸易组织（WTO）改革。
 <br/>
 一名美国高级官员告诉〝路透社〞说，这是G20首脑首次承认WTO目前无法达成它的设立目标，需要改革。
 <br/>
 <br/>
 编辑/周玉林
