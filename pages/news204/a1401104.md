### 赖小民情人过百 舒淇许晴紧急澄清
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融.html" target="_blank">
  华融
 </a>
 资产管理股份有限公司原中共党委书记、董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 被〝双开〞后，贪腐细节随之曝光，除了现金房产，还有女人无数。最近再有两名女影星被牵扯进去。
 <br/>
 <br/>
 11月27日，大陆财经媒体人曹山石在推特（Twitter）发布一条消息，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖小民.html" target="_blank">
  赖小民
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舒淇.html" target="_blank">
  舒淇
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_许晴.html" target="_blank">
  许晴
 </a>
 〞；该帖文发布后，被网友大量转载和揣测。
 <br/>
 <br/>
 舒淇与许晴是分别是台湾和大陆的女演员。有网友表示，其内容似乎是影射赖小民与舒淇和许晴有染。
 <br/>
 <br/>
 对此，舒淇工作室发布声明回应传闻称，〝舒淇小姐不认识赖小民。〞并〝警告〞发布者、转载者，以及相应网路服务平台，立即删除已发布的上述贴文及相关链接，停止中伤揣测舒淇的行为，表示〝将向侵权者采取法律行动并保留追究法律责任的权利。〞
 <br/>

 <br/>
 舒淇本人则在微博上称，〝清者清，不落浊。这世上除了生离死别，其他的都不值得难过、执着、气恼。〞
 <br/>
 <br/>
 许晴也发布声明称，〝一切针对许晴女士的不实信息请发布者立刻删除，并采取一切有效措施防止不实信息进一步扩散。〞表示将会采取一切法律措施，对恶意诽谤、侮辱的行为坚决追究到底，绝不姑息。〞
 <br/>
 <br/>
 10月15日，中共中纪委宣布对赖小民开除党籍、开除公职的处分决定。此次中纪委通报遣词严厉，称赖小民〝擅权妄为、腐化堕落、道德败坏、生活奢靡，甘于被‘围猎’，严重违反党的纪律〞。还用了三个〝特别〞，即〝群众反映特别强烈、腐败问题特别严重、性质特别恶劣。〞
 <br/>
 <br/>
 据陆媒披露，赖小民母亲账户有3亿元（人民币，下同）的存款，全是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华融.html" target="_blank">
  华融
 </a>
 内外人士以赖小民〝孝子〞为由，逢年过节以各种方式〝孝敬〞其母的贿款。另外，赖小民被调查期间，在其北京的多处房产中，共搜出现金2.7亿元，拥有〝100多套房、100多个关系人、100多位情人。其情人包括华融内外的员工，也涉及影视女明星。
 <br/>
 <br/>
 据海外媒体披露，赖小民的原配曾任大陆光大银行总行信贷管理部任总经理；赖小民在香港还有一位情妇，两人育有一对双胞胎。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
