### 中共政府被指雇傭流氓恶棍干非法肮脏的事
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】近日，有陆媒发文讨论了中国普遍存在的地方政府雇傭流氓恶棍充当〝打手〞的现象。文章指出，不少地方政府为了〝强拆〞、〝截访〞以及恐吓异见人士等，往往暗中收买地方黑恶势力，动用暴力手段来干政府工作人员不方便出面的〝脏活〞。
 <br/>
 <br/>
 据法广10月31日的报导，专门研究中国问题的网站〝中国风〞（Chinoiresie）近日发表的一篇文章指出，不论是2014年的香港雨伞运动过程中，还是2012年山东临沂市东师古村陈光诚被软禁在家时，都有媒体爆料地方政府雇用流氓和恶棍来充当政府的打手的情况。
 <br/>
 <br/>
 文章表示，中共的地方政府或执法机构很多时候会雇用一些流氓来执行一些〝非法或厌恶性的政策〞，例如：严刑逼供，强行驱赶老百姓离开家园，或者采取暴行以威吓示威者和异见分子，或暴力拦截上访人员、控制民众。
 <br/>
 <br/>
 由于上述这些特殊的〝工作〞如果由中共官方正式的人员来干，会有损警察以及政府的形象，使政府的合法性受到质疑，所以地方当局就花钱雇用流氓来干这些〝脏活〞。
 <br/>
 <br/>
 文章表示，地方政府采取这种非正常的方式来〝执法〞的理由有很多：
 <br/>
 <br/>
 首先，为了不被外界批评使用非合法性的暴力，地方政府将那些难以见光的〝暴力工作〞外包给予第三方来进行，并让这些流氓与政府保持一定的距离。一旦〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暴力执法.html" target="_blank">
  暴力执法
 </a>
 〞造成了死人、伤人的恶劣后果，官方可以很容易就推卸责任。
 <br/>
 <br/>
 其次，被雇用的流氓可被视为合约工人，随时可以结束工作关系，而不必像对待列入编制以内的正式职员那样支付大笔的福利，这对于财政上往往捉襟见肘的地方政府来说，是节省打压老百姓的〝维稳成本〞的一条捷径。
 <br/>
 <br/>
 第三，那些流氓恶棍通过非正常的手段去〝办事〞，可以快速达到政府人员以正常方式去办事难以达到的目的。
 <br/>
 <br/>
 文章同时指出，地方当局〝外包暴力工作〞的做法，往往会因为难以管控居中转介的公司而出现纰漏和问题。例如：当流氓〝执法〞时，往往发生过度使用暴力而导致出现伤亡的情况。
 <br/>
 <br/>
 由于地方政府太过依赖这些地方黑恶势力来〝执法〞，使得受雇的流氓〝渐成气候〞，形成帮派并经营赌博和卖淫之类的非法活动。
 <br/>
 <br/>
 此外，有些地方政府在强征土地或强拆房屋的过程中，为了完成〝上头〞下达的限时命令，又害怕触发社会群众事件，于是转而依靠处于中介地位的〝黄牛党〞来帮助解决问题。这些〝黄牛党〞会安排政府人员与房子面临被拆命运的房主进行非正式的谈判，透过〝台底交易〞来解决问题。
 <br/>
 <br/>
 文章最后表示，目前难以估计这样问题有多严重，但有一点可以肯定的就是，地方政府与地方黑帮勾结的情况，已严重到迫使习近平当局不得不发起大规模扫黑运动来应对。
 <br/>
 <br/>
 （记者李明报导）
