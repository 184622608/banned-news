### 美制裁党官曾震动北京 习近平仓促开会3常委缺席
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】直接针对中共官员实施制裁，被认为是美国对付北京的致命武器。美媒日前一篇报导佐证了这个观点。消息指，白宫9月宣布制裁中共中将李尚福时，北京紧急召开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治局会议.html" target="_blank">
  政治局会议
 </a>
 部署反击，因为太过仓促，3名政治局常委来不及参加会议。
 <br/>
 <br/>
 《华尔街日报》11月28日报导称，今年9月20日，正当北京还不知道如何应对美方针对2,000亿美元中国商品加征的关税时，美国又宣布对中共军委装备发展部及其部长李尚福中将进行制裁，这令北京措手不及。
 <br/>
 <br/>
 第二天（9月21日），北京最高领导
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 就召集中共政治局委员紧急开会，因为安排会议太过匆忙，7名中共政治局常委中有3名因外出无法与会。
 <br/>
 <br/>
 报导说，最后政治局得出一致结论，必须进行强硬反击。随后，中方取消了即将在华盛顿举行的中美贸易谈判，叫停了与美国军方官员的会晤，并召见了美国驻华大使提出抗议。
 <br/>
 <br/>
 报导引述一名中共高级官员的回忆称，〝大环境如此恶劣，对话沟通是没有意义的。〞
 <br/>
 <br/>
 根据此前报导，9月20日，美国宣布了对李尚福的制裁，原因是中共军方购买了美国明确禁止的俄罗斯装备。9月21日，北京即召开了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治局会议.html" target="_blank">
  政治局会议
 </a>
 。
 <br/>
 <br/>
 但在党媒通稿中，这次会议只审议了中共党支部条例和中共干部培训规划，并没有提及贸易战相关内容。不过，这次会议召开的时间点确实有些反常。按惯例，如无特殊情况，中共每月的政治局例会都会在月底召开。
 <br/>
 <br/>
 外界普遍认为，众多中共高官的巨额资产和直系亲属都已转移的欧美国家，美国如果直接对中共官员实施制裁，将迅速摧毁中共体制内抵抗贸易战的信心，是美方对付中共的杀手鐧。
 <br/>
 <br/>
 针对李尚福启动的制裁，是今年中美贸易战以来，川普直接向中共高官动刀的首例。
 <br/>
 <br/>
 观察人士指，北京高层和中共官员的反应如此激烈，正说明美方这一击对中共的杀伤力惊人。北京9月下旬对美方的一系列强硬反击，颇有鱼死网破之意。
 <br/>
 <br/>
 据此前报导，当时北京不但取消了贸易谈判，还召回了正在华盛顿访问的海军司令沈金龙，取消了本拟10月举行的第二轮中美外交安全对话，拒绝了美国军舰10月访问香港的申请，在南海和美国军舰危险〝互动〞等等。
 <br/>
 <br/>
 但是，经过这一轮反击后，北京在10月底的政治局会议上罕见承认了经济下行压力，随后开始极力缓和中美关系，不断释放让步的信号。
 <br/>
 <br/>
 不过，北京依然回避美方的关键贸易要求。有评论指，如果遵从美国要求，彻底改革经济政策，对中共政权来说同样是一条死路。
 <br/>
 <br/>
 目前，北京拒不让步，中美新冷战一触即发，包括制裁中共官员等一系列致命武器，依然随时可能落到中共体制的头上。
 <br/>
 <br/>
 近期以来，新疆大规模的集中营成为全球关注的焦点，美国政府要求制裁中共新疆自治区区委书记陈全国的呼声越来越高，川普可能因此随时再次祭出制裁中共官员的杀手鐧。
 <br/>
 <br/>
 这也引起中共体制的新一轮恐慌。路透消息，中共驻美大使崔天凯11月28日威胁称，如果美国对新疆官员和科技公司实施制裁，中方〝必须报复〞，而且报复举措将〝按比例进行〞。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
