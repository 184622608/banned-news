### 传周永康郭伯雄秦城监狱交〝心得〞 供出江泽民刘云山
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】自中共十九大至今，官方多次提及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 、薄熙来、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 等六只江派大老虎，搞政治阴谋活动，几乎是公开了江派篡党夺权罪行。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中南海.html" target="_blank">
  中南海
 </a>
 更向港媒放风，郭伯雄和周永康在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦城监狱.html" target="_blank">
  秦城监狱
 </a>
 每周上交〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_学习改造.html" target="_blank">
  学习改造
 </a>
 〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心得.html" target="_blank">
  心得
 </a>
 ，供出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 等人。
 <br/>
 <br/>
 香港争鸣杂志去年初报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 被关入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦城监狱.html" target="_blank">
  秦城监狱
 </a>
 后，监狱要求他每周一至周五都要〝学习反思〞5小时，而且每周要上交〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_学习改造.html" target="_blank">
  学习改造
 </a>
 〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心得.html" target="_blank">
  心得
 </a>
 。
 <br/>
 <br/>
 而周在上交〝学习改造〞的心得中，检举了原政治局内部、中央政法委内部和四川省委内部不少官员的严重违法情况，其中包括其秘书、曾担任中共政法委秘书长的周本顺的问题、中共常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 ，以及中宣部部长刘奇葆等人的贪腐等问题。
 <br/>
 <br/>
 而习近平当局已经把这些检举材料列作防扩散材料。
 <br/>
 <br/>
 报导同期披露，前中共军委副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 在每周上交的〝学习心得〞中，不但自曝接受徐才厚转交贿款的详情，而且还咬出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 。
 <br/>
 <br/>
 报导说，郭伯雄把军队变质问题归咎于江泽民对军队的领导，特别是在军队领导层政治思想以及组织构建方面，并承认自己是国家的罪人，是〝军队败类〞。
 <br/>
 <br/>
 郭伯雄并自曝〝在2003年十一前夕收到徐才厚转交一张金额150万元现金支票，一直放在制服内口袋里长达22天，才叫家属存入银行〞。
 <br/>
 <br/>
 2015年6月11日，习当局通报周永康案已在天津进行了不公开审理，并于当天宣判其无期徒刑，并没收个人财产。周永康被指控的罪名是受贿罪、滥用职权罪和故意泄露国家秘密罪。报导还称周永康当庭表示服从判决、不上诉，同时表示认罪悔罪。
 <br/>
 <br/>
 2016年7月25日，前中共中央军委副主席郭伯雄受贿案被一审宣判。郭伯雄被判处无期徒刑。官方通报称，庭审中，郭伯雄表示认罪悔罪，并表示不上诉。
 <br/>
 <br/>
 当时外界分析认为，周永康与郭伯雄的〝认罪悔罪〞，表明二人完全配合习近平当局的调查，已经全盘交待其罪行和供出其后台老板江泽民。
 <br/>
 <br/>
 胡、温当政期间，郭伯雄、徐才厚、周永康分别把控军队与政法委武警力量，听令于江泽民架空时任军委主席的胡锦涛。郭、徐、周等人均深度参与江泽民集团的政变。
 <br/>
 <br/>
 2012年2月重庆事件爆发，曝光了江泽民集团政变的阴谋后，习近平、王岐山以反腐名义陆续拿下薄熙来、周永康等众多江泽民集团涉政变的高官。
 <br/>
 <br/>
 分析指出，中共十九大至今，官方多次提及周永康、薄熙来、郭伯雄等江派大老虎，搞政治阴谋活动，几乎是公开了江派篡党夺权罪行，甚至点明了江泽民是〝新四人帮〞政变集团的总后台。
 <br/>
 <br/>
