### 网曝刘强东欲花5000万美元与受害女主私了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】中国电商平台京东创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 在美国涉性侵事件，如今事发已超过3个月，但针对刘强东是否会面临刑事指控，仍然没有任何消息。近日，有消息称，刘强东欲花5000万美元与被性侵女子私了。
 <br/>
 <br/>
 12月3日，有中共海外党媒多维引述消息称，京东创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 欲砸5,000万美元重金与被性侵女子和解。消息还称，涉事女子21岁，比中国名人邓文迪高多了，该女子全家下半辈子不用愁了。此人要么很单纯，要么很有心计。不过该消息并未得到证实。
 <br/>
 <br/>
 今年45岁的刘强东是中共的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红顶商人.html" target="_blank">
  红顶商人
 </a>
 〞，头戴中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政协委员.html" target="_blank">
  政协委员
 </a>
 的大帽，且并兼任中国民间商会副会长。
 <br/>
 <br/>
 根据美国警方9月4日公布的报告，刘强东性侵案发生在8月31日凌晨1时，当天刘强东被捕入狱，16个小时后获释，0保释金。
 <br/>
 <br/>
 警方对刘强东一案的描述是〝一级性犯罪：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 既遂〞（CSCR-1,CSC-RAPE-COMPLETE）。据警察局的拘留记录，刘强东的状态为〝释放有待正式控告〞。
 <br/>
 <br/>
 根据明尼苏达州的法律，一级强暴罪只要最后被定罪，刑期至少12年起。
 <br/>
 <br/>
 明尼苏达亨内平县警方9月21日宣布，警方已完成对刘强东案件的初步调查，接下来将由检方接手，审查案件全部证据，并最终决定是否起诉。
 <br/>
 <br/>
 刘强东性侵被捕之初，中共外交部曾罕见第一时间介入，而刘强东在被指控涉嫌强暴的一级重罪后，仍能迅速获保释，返回中国，与中共外交部的介入不无关联。
 <br/>
 <br/>
 9月上旬，就有美媒引述知情人士透露，因中共官方的介入，刘强东性侵案情或变的复杂。当时就有传闻称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受害女生.html" target="_blank">
  受害女生
 </a>
 正接受当局谈话，或被逼回中国。
 <br/>
 <br/>
 如今事发已超过3个月，针对刘强东是否会面临刑事指控，仍然没有任何消息。
 <br/>
 <br/>
 至于美检方为何一直不起诉？明尼苏达亨内平县(Hennepin County)检察官办公室对美媒表示：〝每个案件都不同，审查案件需要花费的时间......我们还有其他需要花费更长的时间的案件，有些花的时间就比较少。〞
 <br/>
 <br/>
 检察官还称，刘强东是富豪与否与他们对案件的审议无关。
 <br/>
 <br/>
 律师马什．哈尔伯格（Marsh Halberg）分析称，检察官可能在等待化验结果，在个别情况下，检察官也可能会要求办理该案的警察局重新采访证人或收集更多的证据。
 <br/>
 <br/>
 他说：如果刘强东不被起诉，人们就会怀疑他是否受到了特殊对待，如果检方要提出指控，刘强东也必定会动用大量资源进行辩护，因此检方必须要有充足的理由才能提起诉讼。
 <br/>
 <br/>
 对此，有美媒分析说，如今唯一能够帮刘强东脱罪的，就是女方承认当晚的行为是自愿的。否则，刘强东被判强奸罪成立的概率很大。
 <br/>
 <br/>
 有华裔刑案律师认为，刘强东遭起诉的可能性超过90%，案件一旦开始会涉及重新逮捕等问题。
 <br/>
 <br/>
 随着事件的发酵，刘强东霉运接踵而至，近3个月来，刘在连续缺席多项重要场合后，坊间又传出他已退出京东管理层。11月底再有陆媒刊文提出，刘强东应〝切割〞相关角色，包括辞去中共全国政协委员职务等。
 <br/>
 <br/>
 报导说，刘强东此时主动辞去全国政协委员职务〝是明智之选〞，与〝全国政协委员〞这个角色切割后，刘强东可以静下心来打官司，同时安排好京东集团的各种〝后手〞。
 <br/>
 <br/>
 报导还称，希望刘强东认错改错，重新做人，即使最终入狱服刑，也应该通过合适的方式维护好京东的运营，比如换一个董事局主席和首席执行官。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：范铭）
