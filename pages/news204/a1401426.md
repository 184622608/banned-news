### 港媒：张柏芝诞下第三胎 男友为新加坡6旬富豪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】11月29日，有港媒爆料，香港女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张柏芝.html" target="_blank">
  张柏芝
 </a>
 已于11月25日在香港养和医院生下第三胎，孩子的爸爸是65岁的新加坡连锁饮食集团股东，有一间上市企业，是身家雄厚的富豪。但仍没获证实。
 <br/>
 <br/>
 香港媒体11月19日报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张柏芝.html" target="_blank">
  张柏芝
 </a>
 于11月25日在香港养和医院诞下第三胎，和张柏芝早前生女愿望不同，这次又是一个男孩，报导配图中还调侃：〝说好的妹妹呢？〞
 <br/>
 <br/>
 当天港媒还拍到张妈妈前往医院，被问及是否去探望生产的女儿时，张妈否认，她说自己只是去医院看病。另有媒体向张柏芝经纪人，对方表示〝没有话说，sorry〞便挂了电话。
 <br/>
 <br/>
 现年38岁的张柏芝是香港影视女演员、歌手，2006年她和香港男星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢霆锋.html" target="_blank">
  谢霆锋
 </a>
 结婚，婚后育有两子，2012年5月，二人离婚。
 <br/>
 <br/>
 今年2月底，张柏芝在社交网上载一张拥着小女孩的相片，留言说：〝2018年之前，必定要生出BB来，this is my goal！〞当时网友就揣测她应该已经有固定男友了。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-30/p9150532a543807102.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-30/p9150532a543807102-ss.jpg"/>
   <br/>
   <font size="-1">
    11月29日，有港媒爆料，香港女星张柏芝已于11月25日在香港养和医院生下第三胎。( Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 29日媒体爆料三胎宝宝的生父是一位65岁的新加坡富豪，让粉丝们觉得不可思议，报导还称，张柏芝和孩子的爸爸是3月认识的，张柏芝今年经常出入新加坡就是为了与男友约会。
 <br/>
 <br/>
 张柏芝妈妈几个月前亮相某节目时，也曾提到女儿新恋情，是一位65岁的新加坡富豪，虽然年龄大张柏芝27岁，但很会照顾人。
 <br/>
 <br/>
 11月29日深夜，又有网友称，三胎宝宝的生父并非新加坡富豪，而是一位香港歌手！他说，张柏芝在养和医院生下男宝，产房号是2733，孩子父亲叫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梁钊峰.html" target="_blank">
  梁钊峰
 </a>
 。
 <br/>
 <br/>
 梁钊峰出生于1986年，是香港某组合的主唱。曾于2013年与张柏芝传过绯闻，而且有媒体拍到他去柏芝家里做客。事后，梁钊峰解释说，自己去柏芝家里只是因为工作。
 <br/>
 <br/>
 陆媒表示，惹说梁钊峰是张柏芝三胎宝宝的生父，这压根就不成立。因为梁钊峰25日的微博非常活跃，丝毫没有任何升级当爹的迹象。他的微博相册显示，22日他还在丹麦的哥本哈根。
 <br/>
 <br/>
 张柏芝的前夫谢霆锋11月29日在香港尖沙嘴出席庆功宴，被问及张栢芝产下第三胎一事，霆锋没有黑脸礼貌地回应：〝不说了！〞随后一行人由商场走火通道离开。
 <br/>
 <br/>
