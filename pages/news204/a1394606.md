### 孟宏伟妻子报案有预谋？港媒：她已踏上不归路
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 受查震惊国际舆论。该消息出炉前，孟妻在法国报警说孟已失踪多日。9日有港媒说，孟突然出事，孟妻在法国报警，应是踏上不归路。此前报导称，孟妻报案是孟宏伟夫妇早已设计好被调查后的应对计划，目的是想让孟的妻儿能获得法国的庇护身份。
 <br/>
 <br/>
 10月8日凌晨，公安部党委书记、部长赵克志主持公安部党委会议，通报了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 收受贿赂、涉嫌违法接受国监委监察调查的情况。
 <br/>
 <br/>
 赵克志要求公安部官员要认识到孟宏伟案件的严重危害，彻底肃清
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 流毒。据报，公安部已成立专门的班子查处与孟宏伟共同收受贿赂官员。
 <br/>
 <br/>
 10月7日晚，中央纪委国监委网站通报孟宏伟涉嫌违法，正接受国监委监察调查。
 <br/>
 <br/>
 同日，国际刑警组织（Interpol）在其官网上说，国际刑警组织总秘书处接到孟宏伟辞去
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 的职务，并立即生效。
 <br/>
 <br/>
 孟洪伟的妻子格蕾丝・孟（Grace Meng）在法国里昂的一家酒店举行了记者会，有两名负责保护她的法国警察陪同。但她不愿意公开自己的相貌，外界亦不知晓她的中文姓名。
 <br/>
 <br/>
 孟宏伟受查的消息曝出前，法国警方透露孟宏伟9月29日告别妻小，只身从国际刑警组织的里昂总部返回中国后失踪，孟妻于10月4日傍晚向法国警方报案。
 <br/>
 <br/>
 对于孟宏伟此次被召回北京受查后，孟妻罕见地向法国警方报案。有报导认为，这是孟宏伟夫妇早已设计好，被调查后的应对计划，目的是想让孟宏伟的妻儿能获得法国的庇护身份。
 <br/>
 <br/>
 孟妻向法国警方报案丈夫失踪的同时，称自己受到不明生命安全威胁。法媒称，法国警方已经给予孟宏伟在法国的妻子予以司法安全保护。
 <br/>
 <br/>
 10月9日，香港明报报导说，孟宏伟在当选国际刑警组织主席后，已卸任海警局长，又退出公安部党委，3月还当选了中共政协外事委员会委员，本来两年后以67岁之龄应退休。现在突然出事，妻子在法国报警，应是踏上不归路。
 <br/>
 <br/>
 按公安部会议通报，孟宏伟应该是涉及周永康等政法系统贪官旧案被拿下。
 <br/>
 <br/>
 孟宏伟是黑龙江哈尔滨人，曾在中共安全机关担任过多项职务。2016年当选国际刑警组织主席，任期将于2020年届满。
 <br/>
 <br/>
 孟宏伟在公安部一路高升期间，公安部长正好是权倾一时的周永康。当周落马后，与周有牵连的孟却能全身而退，还被中共推上国际刑警组织主席的要职。
 <br/>
 <br/>
 但孟宏伟上任一年多后，中共政局剧变，孟被免去海警局长一职后，又退出了公安部党委。加之近几年，孟宏伟接连缺席中共公安部高层的内部会议。中共官场早已传出孟宏伟仕途〝很不妙〞的消息。
 <br/>
 <br/>
 对于此次孟宏伟被查。法国《世界报》评论说，孟宏伟所涉及的问题，必定极为敏感或严重，才会让北京宁可冒着对国际出丑也要逮人的政治风险。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：祝馨睿）
