### 大陆留守儿童:〝感觉自己就像一只流浪狗〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_留守儿童.html" target="_blank">
  留守儿童
 </a>
 问题，已成为大陆社会一个严重的社会问题。一方面，留守儿童因疏于保护而遭伤害；另一方面，留守儿童缺乏关爱造成的报复心理，也在制造新的社会问题。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_留守儿童.html" target="_blank">
  留守儿童
 </a>
 ，指的是为父母双方或一方外出打工而留守在家乡，无法与父母正常共同生活的，不满16周岁的农村户籍未成年人。在过去30多年间，近3亿的大陆农民背井离乡，他们的孩子，或者与祖辈生活在一起，或者自生自灭。
 <br/>
 <br/>
 据大陆民政部今年8月底发布的数据，2018年大陆共有农村留守儿童697万余人，96%的农村留守儿童由祖父母或者外祖父母照顾。
 <br/>
 <br/>
 2017年冬天，云南头顶霜花而一夜成名的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冰花男孩.html" target="_blank">
  冰花男孩
 </a>
 〞，就是一位留守儿童。有留守儿童在接受陆媒采访时表示:〝我有时候感觉自己就像一只流浪狗〞。
 <br/>
 <br/>
 10月16日，大陆公益促进中心〝上学路上〞公布了《2018年中国留守儿童心灵状况白皮书》。《白皮书》调查发现，留守儿童家庭往往一年到头无法团聚几次，长期的分离会对儿童与青少年造成心理伤害。在核心自我评价方面，留守儿童的得分显著低于非留守儿童和半留守儿童；在情绪状态方面，留守儿童的愉悦状态得分显著低于非留守儿童。
 <br/>
 <br/>
 据统计，大陆每年有近5万名儿童死于意外伤害，其中大部分是留守儿童；根据大陆农村
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_留守人口.html" target="_blank">
  留守人口
 </a>
 问题专家叶敬忠的调查，溺水和交通事故成为留守儿童非正常死亡的两大隐患。
 <br/>
 <br/>
 威胁留守儿童的另一个危险是〝被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性侵.html" target="_blank">
  性侵
 </a>
 〞。有资料显示，过去三年里，广东省就有超过2500名女童被性侵，近半数在14岁以下。而广东正是留守儿童高度集中的省份之一，占大陆留守儿童总数的7.18%。宁夏灵武市一个村庄内，过去一年间，有12名幼女被教师性侵，其中11人为留守儿童。
 <br/>
 <br/>
 一篇题为《二十多年来留守儿童的伤痛，已累积为重大社会问题》的文章说，很多被分离的儿童因缺失父母的教育与关爱，或沦为边缘人，或对社会失去信任感而仇视社会。近年频发儿童自杀或杀人事件，显示这个群体陷入了黑暗的心理状态。
 <br/>
 <br/>
 有分析认为，留守儿童问题根源是体制问题，尤其是中共户籍制度的产物，使得农民在农村没法谋生，不得不出门打工，导致留守儿童的产生。
 <br/>
 <br/>
 大陆农村留守人口问题专家叶敬忠分析，留守人口现象根源于城市偏向的发展模式，以农村家庭幸福为代价来汲取乡村资源，对农村和农民的生存空间产生挤压，导致乡村的衰败、凋敝与乡村共同体的瓦解，加深城乡差距。
 <br/>
 <br/>
 城乡分割的义务教育体制、二元户籍制度，学费太高等等问题，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农民工.html" target="_blank">
  农民工
 </a>
 所无法解决的，各地政府的帮扶和疏导责无旁贷。
 <br/>
 <br/>
 2016年，大陆民政部表示计划首度彻查留守儿童的确切数量。正如陆媒的调查分析所说，〝收集数据和强调执行并不能从根本上解决问题。有统计显示，在某些地方，留守儿童已占到未成年人犯罪的70%。〞
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：李泉）
