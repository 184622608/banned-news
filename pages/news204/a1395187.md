### 【禁闻】工业蛋白替代豆粕  中共〝坑猪〞害人
------------------------

<div class="wysiwyg">
 【新唐人2018年10月12日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 成为中共的打击目标，为减少对美国大豆的依赖，中共大幅降低猪饲料中的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_豆粕.html" target="_blank">
  豆粕
 </a>
 及粗蛋白质含量，并以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工业蛋白.html" target="_blank">
  工业蛋白
 </a>
 替代豆粕。有评论指出，这种做法会导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_食品污染.html" target="_blank">
  食品污染
 </a>
 大面积爆发，将对人民健康造成极大的损害。
 <br/>
 <br/>
 大陆国营的中国饲料工业协会日前发布声明称，针对减少猪饲料中的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_豆粕.html" target="_blank">
  豆粕
 </a>
 及粗蛋白质含量，公开征求业者意见。不过，〝新浪财经〞10月10号的报导说，据〝新浪期货〞获得的消息，有迹象显示，饲料业协会会员已经开始执行新的豆粕标准。
 <br/>
 <br/>
 据〝中国饲料工业协会团体标准技术委员会〞发布的最新饲料标准，猪饲料中的粗蛋白质和总磷，在常规只设定下限值的基础上增设了上限值。此外，猪饲料中的豆粕含量，也从原来的不低于14%，降低到11%至13%。
 <br/>
 <br/>
 一位资深的产业界人士对〝新浪期货〞表示，此举可能会使中国的大豆进口量因此减少1000万吨，他还透露，官方说，减少的豆粕不会用玉米替代，而是用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工业蛋白.html" target="_blank">
  工业蛋白
 </a>
 来替代。
 <br/>
 <br/>
 这则消息引发广泛关注，有不少网友在问，工业蛋白是什么？有人回答，就是臭名昭着的三聚氰胺。有人惊呼:这是坑猪啊! 也有人感叹说，这就是他们口中喊的〝代价〞，小民命贱不值钱！
 <br/>
 <br/>
 在美国一家生物科技公司担任副总的怀海鹰博士表示，工业蛋白有可能是比三聚氰胺还不如的工业废料或医疗废料，对猪只和人体健康有害，中共这种做法很不负责任。
 <br/>
 <br/>
 美国生物科技公司副总怀海鹰博士：〝工业蛋白，有可能加了东西，连三聚氰胺都不如，搞不好。有可能是工业废料，含些氮，医学一些废料，含些氮，使这个东西含氮里面的元素，就转基因那些元素，用这个来替代了，对猪的健康肯定是不利的，猪如果不健康，人也是，猪的结构和人比较近，那绝对是有问题，这很不负责任的。〞
 <br/>
 <br/>
 怀海鹰博士指出，饲料中添加工业蛋白，说可以提高含氮量，那是骗人的。
 <br/>
 <br/>
 怀海鹰博士：〝表面上提高含氮量，那是糊弄的，糊弄一时是可以的，为了弥补这个缺口嘛，它如果不向美国进口大豆的话，的确是不够用，它就搞这个来糊弄，中国现在缺好几千万吨啊一年，大豆。〞
 <br/>
 <br/>
 在美中贸易战中，中共为了打击美国总统川普的票仓，瞄准美国的豆农，故意提高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国大豆.html" target="_blank">
  美国大豆
 </a>
 进口关税，造成中国大豆短缺。
 <br/>
 <br/>
 旅美中国问题研究人士张健表示，用工业蛋白替代豆粕，是严重的违法行为。
 <br/>
 <br/>
 旅美中国问题研究人士张健：〝豆粕这个事情最近闹得是沸沸扬扬，就是因为中国人担心三聚氰胺。它作为一个毒奶品，残害了我们很多的儿童，事实上，中国的食品安全，这就是又一大隐患。当
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 ，中国的猪因为饲料问题得不到解决的时候，中共体制下的这些官僚机构就能突发奇想，用工业蛋白来取代豆粕优质饲料的蛋白。中国有食品安全法，这个就是严重的违法行为。〞
 <br/>
 <br/>
 张健指出，以国家标准的形式推行这个做法，危害性极大。
 <br/>
 <br/>
 张健：〝如果工业蛋白由国家的这些部委规定，强行地进入食品链条之中之后，那么，所有各地的不法商贩更肆无忌惮、明目张胆地使用各种工业蛋白，包括工业添加剂，这种更大的污染就会大面积的爆发，会给人民生活、健康造成极大的损害。这和民间这些不法商人做的假食品、毒食品，本质上不同的。〞
 <br/>
 <br/>
 资深产业界人士还向〝新浪期货〞表示，当前美国豆子确实很便宜，即便加上关税价格，进口美豆也是有利可图的。但是这不是价格的事儿，中国人是有骨气的。
 <br/>
 <br/>
 对此，有网友说，为了面子，打折自己的腿，突然想到一个成语：削足适履！咱们国家在解决问题的思路上总是如此独树一帜脑洞清奇。
 <br/>
 <br/>
 也有一些网友在问，领导吃不吃呢？谁提议给谁吃吧。
 <br/>
 <br/>
