### 安倍访华一组照片被聚焦 北京尴尬
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】日本首相安倍晋三25日前往北京访问，临行前，他发推特说：〝羽田机场晴空万里。〞但一到北京时，整个天空雾茫茫一片。当天日本驻华大使馆微博，发布一组日本与北京对比照片，网民认为这让北京尴尬。
 <br/>
 <br/>
 日本首相安倍晋三，于10月25日至27日，对中国进行为期3天的访问。
 <br/>
 <br/>
 25日下午12时44分，安倍乘专机启程赴中国访问前，他发了一条推特说：〝羽田机场晴空万里。已经有7年没正式访问中国了。〞
 <br/>
 <br/>
 但安倍发送这条推特之前，北京当天的空气质量已达到了中度污染水平，部分地区则是重度污染。
 <br/>
 <br/>
 根据日本国驻华大使馆微博发布的多张安倍访问北京的影像。其中一张是安倍准备从羽田机场搭机访问中国的照片，可见天空一片湛蓝；但在安倍抵达北京机场时，当地正好受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_雾霾.html" target="_blank">
  雾霾
 </a>
 影响，整个天空雾蒙蒙一片，形成强大对比。

 <br/>
 网民纷纷在日本驻华大使馆微博留言戏谑说，〝吸到这么醇厚的雾霾，不知道安倍是什么感受〞、〝这天气实在太不争气了！让北京高层尴尬了〞。
 <br/>
 <br/>
 根据陆媒报导，25日当天，北京全市达四级中度污染，部分站点已达五级重度污染。空气质量监测平台显示，当天下午2点，北京全市空气质量指数为170，达四级中度污染。其中，京东南、京南、通州北苑监测点PM2.5小时浓度超过200，达五级重度污染。
 <br/>
 <br/>
 值得关注的是，2015年12月3日，日本政府曾向〝日中绿化交流基金会〞提供100亿日元（约合8,300万美元）的资金援助，用来帮助中国改善日益严重的雾霾。
 <br/>
 <br/>
 原因是，同年日本环境省的《越境大气污染报告》显示，从日本西部九州到东部的东京，空气中PM2.5含量，61%和39%是从中国飘洋过海贡献的。
 <br/>
 <br/>
 日本目前实行的PM2.5浓度国家标准为75微克/立方米，日本全年空气PM2.5浓度约为10~20微克/立方米。
 <br/>
 <br/>
 而北京全年平均PM2.5浓度约为85.9微克/立方米。2015年，北京局部地区PM2.5浓度一度高达到945微克/立方米。
 <br/>
 <br/>
 2017年5月7日，日本气象厅还观测到了沙尘暴来袭，范围覆盖从日本西部到北部的广大地区，还包括关东地区的部分地区。日本气象厅的报告称，从中国吹来的大风携带着黄沙。
 <br/>
 <br/>
 近年来，日本和韩国的民众对中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空气污染.html" target="_blank">
  空气污染
 </a>
 扩散至自己国家多次表达强烈不满。
 <br/>
