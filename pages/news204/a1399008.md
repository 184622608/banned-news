### 北京天则所谘询公司被吊销执照 拟上诉
------------------------

<div class="wysiwyg">
 【新唐人2018年11月12日讯】大陆民间智库〝北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天则经济研究所.html" target="_blank">
  天则经济研究所
 </a>
 〞属下的谘询公司日前遭到北京市政府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吊销执照.html" target="_blank">
  吊销执照
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天则研究所.html" target="_blank">
  天则研究所
 </a>
 11月9号发表声明说，这个决定违反国家主席习近平强调要保护
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民营企业.html" target="_blank">
  民营企业
 </a>
 的说法，并表示将针对此事提出上诉。
 <br/>
 <br/>
 11月12号，曾任〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天则研究所.html" target="_blank">
  天则研究所
 </a>
 〞所长助理的经济学者、专栏作家段绍译，在接受新唐人电话采访时说，北京天则所谘询有限公司，于10月22号，收到北京市工商行政管理局海淀分局的通知，该公司的营业执照被吊销。
 <br/>
 <br/>
 原〝天则研究所〞所长助理段绍译：〝官方的理由是，说它违犯了规则经营，没有许可办了培训班，说是超范围经营了，但是实际北京天则研究所的人认为，他们是说天则研究所说真话太多。〞
 <br/>
 <br/>
 段绍译说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天则经济研究所.html" target="_blank">
  天则经济研究所
 </a>
 所长盛洪和经济学者蒋豪，日前也遭到边控被禁出境，理由是他们有危害国家安全之嫌。
 <br/>
 <br/>
 天则研究所经济学者蒋豪告诉新唐人，天则所11月9号发了一份声明，指出当局吊销天则所咨询公司执照是〝出于错误的目的，采信错误的证据，实施于错误的时间和地点〞，并不是出于中国宪法和法律内涵中〝维护市场秩序〞及〝保护企业经济自由〞的善意。
 <br/>
 <br/>
 所谓错误的时间，就是在习近平和刘鹤最近一再强调要保护
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民营企业.html" target="_blank">
  民营企业
 </a>
 的大背景下；所谓错误的地点，就是在有着全国示范影响的北京。
 <br/>
 <br/>
 〝天则经济研究所〞所长盛洪也对媒体表示，官方
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吊销执照.html" target="_blank">
  吊销执照
 </a>
 的决定是一个〝反市场〞、〝反民企〞的行政处罚，公司不会接受处罚决定，将按照正常法律程序，提出上诉。
 <br/>
 <br/>
 天则经济研究所以自由派立场著称，1993年，由中国经济学家茅于轼、盛洪、樊纲、唐寿宁等人创立。自2003年起，研究所不断受到不同程度的打压。今年7月10号北京办公室大门曾被人焊死和锁上，并遭逼迁。
 <br/>
 <br/>
 新唐人记者熊斌、元真采访报导
</div>
