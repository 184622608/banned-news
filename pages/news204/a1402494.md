### 德国总统在四川大学演讲 痛批马克思主义
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月09日讯】应邀访华的德国总统施泰因迈尔（Frank-Walter Steinmeier）日前在中国成都的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四川大学.html" target="_blank">
  四川大学
 </a>
 发表演讲，痛批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克思主义.html" target="_blank">
  马克思主义
 </a>
 曾给东德和东欧带来浩劫，并强调人权问题。陆媒报导中对此只字不提。
 <br/>
 <br/>
 北京时间12月7日15时，施泰因迈尔在座无虚席的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四川大学.html" target="_blank">
  四川大学
 </a>
 望江校区西五教演播厅发表演讲。路透社从德国大使馆获得了一份演讲稿。
 <br/>
 <br/>
 据路透报导，施泰因迈尔在演讲中说，德国人和中国人不但对一些现实问题有不同观点，对历史和知识分子思想的看法也存在分歧。
 <br/>
 <br/>
 他说，马克思虽然自称〝人道主义者〞，要求过〝新闻自由〞和〝环境保护〞等，但他的思想并〝没有停留在理论上〞。〝我们德国人在想到马克思的时候，不能不想到在他的名字之下的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克思主义.html" target="_blank">
  马克思主义
 </a>
 ，为东德和东欧所带来的浩劫，和铁幕之下那段压抑的历史。〞
 <br/>
 <br/>
 相比之下，北京今年为纪念马克思诞辰举行了一系列大型活动，大举宣传〝马克思主义是完全正确的〞，〝能够永葆青春〞等等。
 <br/>
 <br/>
 施泰因迈尔在演讲中没有在具体问题上对中共提出批评，但在很多方面含蓄地表达了对中共的谴责。有外媒指，他表达的这些〝不同观点〞应该对台下的学生有所启发。
 <br/>
 <br/>
 在演讲中，施泰因迈尔强调了捍卫人权以及以联合国宪章作为国际秩序基础的重要性，并称七十年前通过的人权宣言是人类历史上的一大幸事。
 <br/>
 <br/>
 他还指出，德国也曾经历过〝没有自由、充满压迫〞的历史，因此，〝我们对那些持非主流观点、属于少数民族或实践个人信仰的人，所受到的待遇非常敏感和关注。无论哪里出现个人权利受到限制的问题，德国都会感到担忧和不安。〞
 <br/>
 <br/>
 近年以来，国际社会对中共镇压新疆少数民族、打压家庭教会和残酷迫害法轮功学员给予持续的关注。
 <br/>
 <br/>
 值得一提的是，施泰因迈尔在演讲中称赞邓小平时，特别引述了后者的一句话：〝我是维族姑娘，辫子多〞。
 <br/>
 <br/>
 据《德国之声》报导，在演讲结束后的问答环节，一名来自德国的交流生特别就〝维族姑娘〞发问，问施泰因迈尔是否会在此行中向中方提及维吾尔族遭打压及新疆〝再教育营〞问题。
 <br/>
 <br/>
 施泰因迈尔回答说，〝当然会谈到民众自由和人权。〞
 <br/>
 <br/>
 在中共官方有关此次演讲的报导中，没有看到上述内容，只简单提到施泰因迈尔〝肯定〞四川变化和中德交流合作等。相比之下，施泰因迈尔演讲后到成都街头〝閒逛〞，反倒成了官媒报导重点。
 <br/>
 <br/>
 施泰因迈尔获中共政府邀请，自12月5日起对中国进行为期5天的国事访问。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
