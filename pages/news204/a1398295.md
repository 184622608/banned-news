### 官媒罕见引述美专家：民调模型有误 川普将拿下两院
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】11月6日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 结果，因为关系中美关系走向，获得中共官方空前关注。中共党媒日前引述详尽的数据分析称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 将拿下参众两院，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 翻盘的可能性已微乎其微。
 <br/>
 <br/>
 北京时间11月6日，上海党媒观察者网刊出美学者专栏文章，预测
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 结果。文章结论指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 在参议院的优势将继续扩大，会从51席增至最少54席。而两党看似在众议院争夺激烈，其实通过分析美国民调数据可以得出确切结论：共和党将稳操胜券。
 <br/>
 <br/>
 文章指，中方早在一个月以前，就开始追踪美国各种民调资源以及提前投票数据。
 <br/>
 <br/>
 其中，参议院的预测相当简单，主流民调结果就可以显示结论：共和党将会守住现有的所有议席。其中，亚利桑那、田纳西、德克萨斯、内华达这些本来被认为可能变蓝的席位都是安全的。
 <br/>
 <br/>
 而且，共和党基本上将稳获如下
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 占据的议席：北达克他、密苏里、印第安纳、佛罗里达和蒙大拿州。这样共和党在参议院的领先优势很可能将扩大为56:44。
 <br/>
 <br/>
 文章说，美国主流民调总结的众议院选举形势图显示，共和党将在众议院面临一场硬仗。但实际上，美国这些主流民调都带有政治倾向，抽去民调中的水分后就会发现，近一个月来共和党优势不断扩大，民主党已经翻盘无望。
 <br/>
 <br/>
 文章分析，民主党要想赢得众议院多数，需要超过共和党8%的支持率。这是因为很多民主党选区在人口稠密的城市，而共和党的支持者位于广大的中产阶级社区和广袤的农村，因此共和党可以在支持率上落后却占据更多席位。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-7/p9114142a451825653.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-7/p9114142a451825653-ss.jpg"/>
   <br/>
   <font size="-1">
    美国主流民调中的两党支持率。（网络图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 从目前民调数据上看，现在民主党支持率正好在临界点上。但是仔细分析这些民调，就会发现两大问题。
 <br/>
 <br/>
 第一，这些民调普遍假设在65岁以上公民中，民主党领先超过18个百分点，但在2016年总统大选中，该年龄组的选民一边倒地支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 。
 <br/>
 <br/>
 第二，主流民调中假设白人选民中，共和党支持率只领先1%，而实际上2016大选中，白人选民支持共和党和民主党的比例是59:37。
 <br/>
 <br/>
 川普的支持者班农说，民调中根本没有采访那些川普的基本盘，他们还在用过时的选民组成模型来预测。
 <br/>
 <br/>
 文章认为，主流媒体民调的原始数据是准确的，得出的却是荒谬的结论。因此，民主党绝对不可能达到8%的支持率，想赢得众议院的可能性微乎其微。
 <br/>
 <br/>
 另外，美国27个州的提前投票数据也支持这个结论。共和党过去在提前投票中基本上都落后于民主党。可是今年的提前投票数据显示，共和党选民在关键州的表现远超预期，在许多州反而领先民主党。
 <br/>
 <center>
  <br/>

   <br/>
   <font size="-1">
    提前投票的统计数据。（网络图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 文章分析，在过去一个月里，选情也发生了巨大变化，在朝着有利共和党的方向发展。
 <br/>
 <br/>
 直到九月底十月初，共和党的支持率都徘徊不前，可是卡瓦诺法官提名战和洪都拉斯非法移民硬闯边界，彻底唤醒了共和党选民的投票热情，美国日益巩固的经济也激发了共和党支持者的信心。因此，在过去的一个月里，共和党候选人的支持率普遍有了大幅增长。
 <br/>
 <br/>
 文章引述前民主党总统候选人桑德斯和民主党全国委员会主席佩雷斯的说法称，不会再有所谓的〝蓝潮〞了。
 <br/>
 <br/>
 法媒此前披露，中共今年高度关注美国中期选举的选情，命令对美机构全方位搜集选情，情报直达北京。
 <br/>
 <br/>
 在中期选举前的11月1日，北京就开始与白宫通话，对美国全方位释放〝友好〞信号。有分析认为，这可能说明北京已经预知选举结果，对民主党翻盘不再抱希望。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
