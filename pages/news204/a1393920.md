### 传江泽民李鹏健康〝亮红灯〞 习近平下令中办负责会诊
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月03日讯】日前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京大会堂.html" target="_blank">
  北京大会堂
 </a>
 举行的十一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_招待会.html" target="_blank">
  招待会
 </a>
 上，据传中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_元老.html" target="_blank">
  元老
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集体缺席.html" target="_blank">
  集体缺席
 </a>
 ，外界纷纷猜测其中原因。有消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 和李鹏的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健康状况.html" target="_blank">
  健康状况
 </a>
 频亮 〝红灯〞，习近平严令中南海医疗团队，争取让他们〝挺过〞中共建政70周年，但该说法未获证实。
 <br/>
 <br/>
 9月30日，中共国务院在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京大会堂.html" target="_blank">
  北京大会堂
 </a>
 举行十一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_招待会.html" target="_blank">
  招待会
 </a>
 。有美媒爆料，今年的招待会上，没有中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_元老.html" target="_blank">
  元老
 </a>
 的镜头，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 、李鹏、朱鎔基、胡锦涛、温家宝等人〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集体缺席.html" target="_blank">
  集体缺席
 </a>
 〞。
 <br/>
 <br/>
 中南海的十一招待会，已经成为观察中共政坛动向的窗口。外界分析，如果元老集体缺席的消息属实，可能存在几种原因。其中包括元老们的身体出现状况。
 <br/>
 <br/>
 博闻社引述中南海知情者爆料说：由于江泽民和李鹏年岁已高，患有老年人常见的疾病，健康频频亮起〝红灯〞，甚至遭遇多次病危险情。在今年的十一招待会之前，习近平让中办主任丁薛祥紧急召集中南海医疗团队，对中共元老们的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_健康状况.html" target="_blank">
  健康状况
 </a>
 进行会诊。
 <br/>
 <br/>
 爆料还指，习近平指示医疗团队，争取让高层元老挺过中共建政70周年。但有观察人士质疑该说法，因为现代医学虽然已有长足进步，但要精确掌握人的生死寿命，其实仍有较大差距。
 <br/>
 <br/>
 也有分析指出，习近平上任以来利用反腐从江系手中夺权成功，习江之间早已势同水火，习本人也多次曝出遭遇暗杀的传闻，习在某些场合的场面话，或难代表习内心真实想法。
 <br/>
 <br/>
 不过，江泽民能否再挺过明年还不好说。早在去年十九大召开前，就有美媒引述知情人爆料说，习近平特别为出席十九大开幕式的江泽民准备了一套〝死亡预案〞，一旦江泽民突然在会场咽气，当局该如何应对。
 <br/>
 <br/>
 当时的直播镜头中也可以看到，江泽民出席十九大时，几乎是被架进会场落座。
 <br/>
 <br/>
 近年来，有关江泽民〝死去活来〞的消息不断传出。不过，江每次传出死讯不久，便会以〝露面〞、〝露名〞等方式间接澄清。
 <br/>
 <br/>
 而每次传出江的死讯，网上民意都是一面倒的欢呼庆祝，并藉此把江的汉奸身世、出卖国土、镇压〝六四〞学生、迫害法轮功、垂帘听政、党政军大老虎总后台、江家父子贪腐等等罪行抖露一番。
 <br/>
 <br/>
 坊间流传江泽民是癞蛤蟆转世，江当政时在外国元首面前照镜子、梳头、弹琴唱歌，以及神情恍惚，甚至半死不活等怪异表现，据称另有原因。
 <br/>
 <br/>
 《江泽民其人》一书中介绍，安徽黄山一位高僧透露：江泽民元神本是个蛤蟆。由于迫害法轮功，造下无边罪业，导致其元神在2000年9月间，已经彻底被打入无间地狱了，现在世间上活动的，不过是江的人皮，以及操纵它的烂鬼而已。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：赵云）
