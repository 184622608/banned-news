### 港媒：习近平已抵达珠海 下榻酒店安保细节曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 23日举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开通仪式.html" target="_blank">
  开通仪式
 </a>
 ，外传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 会出席通车典礼。香港电台22日消息称，习近平已抵达珠海，入住于曾经多次接待国家领导人的〝珠海度假村酒店〞，该酒店员工说，25日前酒店已经无空房。
 <br/>
 <br/>
 香港电台22日消息称，曾经多次接待中国领导人的珠海度假村酒店外围已架设了围栏，并有安保人员驻守，不准外来人士和车辆进入。
 <br/>
 <br/>
 报导引述当地的士司机的话说，19日起，珠海加强安保措施，大批警察和便衣人员在主要干道巡逻把守，胜过以往领导人来访视察的规模。
 <br/>
 <br/>
 有当地市民表示，在珠海格力电器集团工作的朋友称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 20日视察了集团的厂房，视察期间，整个厂房的窗户都要关闭。
 <br/>
 <br/>
 报导还称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 23日举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开通仪式.html" target="_blank">
  开通仪式
 </a>
 ，预料习近平将会出席。
 <br/>
 <br/>
 《苹果日报》记者22日中午来到〝珠海度假村酒店〞，见门外保安不算多，但酒店内明显已经〝清空〞，不见有任何游客在酒店。
 <br/>
 <br/>
 该报记者以客人身份致电珠海度假村酒店要求订房，但酒店职员说，该酒店于10月25日前已经无空房，记者表示，在网上订房平台显示有空房，但职员仍强调〝没房、没房〞，未有透露原因。
 <br/>
 <br/>
 公开资料显示，珠海度假村酒店位于吉大石花东路，距离港珠澳大桥珠海口岸只有约5分钟车程。
 <br/>
 <br/>
 耗资过千亿港元的港珠澳大桥，将于10月23日在珠海举行开通仪式，24日正式通车，连接香港口岸人工岛及北大屿山公路往来市区的连接路也同步开放。
 <br/>
 <br/>
 中共驻香港联络办公室于17日向香港媒体发出〝特急〞采访通知，采访港珠澳大桥开通仪式。采访地点在珠海。主办方还要求记者自费每晚600元人民币，入住指定的一家酒店。
 <br/>
 <br/>
 不过，对于记者关注的习近平是否参加开通仪式，该通知则未提及。
 <br/>
 <br/>
 多家港媒此前报导，习近平于19日南下广东视察，20日到访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广州.html" target="_blank">
  广州
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中山大学.html" target="_blank">
  中山大学
 </a>
 ，并在23日担任港珠澳大桥通车仪式的主礼嘉宾。
 <br/>
 <br/>
 根据中国大型空拍机生产商DJI（大疆）的网页GEO系统显示，整个深圳近日也突然全面禁止空拍，加之近日珠海各区保安明显加强。均被视为为习近平南下的保安工作作准备。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
