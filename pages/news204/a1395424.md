### 任志强炮轰北京政策：川普关税只是防御
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】日前，中国知名地产商人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任志强.html" target="_blank">
  任志强
 </a>
 炮轰中共政策的一段视频在网上热传。针对中美贸易问题，任志强以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长城.html" target="_blank">
  长城
 </a>
 做隐喻，认为北京当局错误的战略体现在〝加大自己的长城〞，而川普的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加征关税.html" target="_blank">
  加征关税
 </a>
 是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自我防御.html" target="_blank">
  自我防御
 </a>
 ，而非
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进攻.html" target="_blank">
  进攻
 </a>
 。
 <br/>
 <br/>
 10月11日，素有〝任大炮〞称号的北京华远地产前董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任志强.html" target="_blank">
  任志强
 </a>
 ，在第十届〝观峰讲坛〞上就美中贸易问题发表讲话。他用中国古
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长城.html" target="_blank">
  长城
 </a>
 比喻中共当局处理贸易纠纷的手法。
 <br/>
 <br/>
 他说，首先要明白关税是长城。长城是干什么的？大家知道长城是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自我防御.html" target="_blank">
  自我防御
 </a>
 。但是修长城一定会有两个结果，一个是你抵御了外敌，让敌人侵略越来越困难；同时也会伤害边境的人民进行境外贸易。不能自由交流了。
 <br/>
 <br/>
 任志强表示，从这个意义理解，如果单纯讲关税，关税就是长城。你不来我这儿，我能收你的关税吗？只有你进入我的长城才涉及到关税。所以它一定不是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进攻.html" target="_blank">
  进攻
 </a>
 ，是防御。
 <br/>
 <br/>
 他说，〝如果所有的媒体都把关税认为是恶意进攻，首先站位就是错了。认识都错了，战略自然也错了。〞任志强认为，北京当局的错误战略体现在〝加大自己的长城〞，即针对美国加征报复性关税，〝现在把门关上就不对了。〞
 <br/>
 <br/>
 任志强表示，中美出现对立的原因出在中国。美国总统川普（特朗普）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加征关税.html" target="_blank">
  加征关税
 </a>
 是自我防御，而不是进攻，因为他对所有国家都一样。〝别人就能谈得通，因为别人开放；中国没有谈通，就是中国不肯把大门打开。〞〝因为你要把自己的大门关起来的，而不是别人挑起来的。别搞错了。〞
 <br/>
 <br/>
 他还隐晦批评北京加征关税以牙还牙的政策，导致了中美几乎走向全面的对抗。他说，做企业家有个最简单的东西：买卖不成仁义在。但现在买卖不成了，仁义也不在了。
 <br/>
 <br/>
 任志强还批评当局未延续邓小平韬光养晦的外交政策。他说，目前，政府正在加大自己的长城，跟中共已故领导人邓小平提倡改革开放背道而驰。如果你把后两个字弄没了，就不叫改革了，中国的改革是来自于你把大门打开了，现在你非把要门关上，怎么能对呢？
 <br/>
 <br/>
 任志强表示，他不懂贸易战，但起码认识到关税是自我防御的长城，不是为了进攻而设。如果不拆长城，长城以外都不是中国的地方。也就是说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 结束的唯一前提就是北京拆掉长城。
 <br/>
 <br/>
 在阐述完自己的观点后，任志强最后说了一句〝能不能听懂是你们的事了〞。
 <br/>
 <br/>
 自中美贸易战爆发后，中国经济疲软，股市暴跌，人民币不断贬值。外资加速撤资，中国各地大小企业更是哀鸿遍野，一时间怨声载道，恐慌情绪蔓延。
 <br/>
 <br/>
 分析认为，任志强直接点出贸易战并非是美国挑起，而是北京当局自我认知的错误导致了战略上的错误。这显然代表着相当一部分企业家、知识分子的看法。至于其是否代表某些高官的看法就不得而知了。
 <br/>
 <br/>
 当前，在北京日趋紧缩舆论的情况下，大胆直言的任志强是否会再度被封杀也同样引发关注。
 <br/>
 <br/>
 2年前，任志强曾因质疑中共〝官媒必须姓党〞，引发官媒文革式〝围攻〞，甚至被扣上〝反党〞的大帽子，还暗示任志强有后台。该事件背后，被指涉及与任志强关系密切的时任中纪委书记王岐山，和时任江派宣传总管刘云山之间的角力。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：范铭）
