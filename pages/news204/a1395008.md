### 【禁闻】中共监督网络新规 恐加深外企担忧
------------------------

<div class="wysiwyg">
 【新唐人2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 公安部日前发布《互联网安全
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监督.html" target="_blank">
  监督
 </a>
 检查规定》，宣称从11月1号起，公安机关有权对互联网服务提供者和联网使用单位，开展监督检查。分析指出，这项规定可能加深外国企业对在华业务的担忧。
 <br/>
 <br/>
 10月4号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 党媒新华社称，根据公安部日前发布的规定，公安机关可进入营业场所、机房、工作场所、要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_监督.html" target="_blank">
  监督
 </a>
 检查对象的负责人或者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网络.html" target="_blank">
  网络
 </a>
 安全管理人员对相关事项作出说明、查阅、复制与互联网安全监督检查事项相关的信息、查看网络与信息安全保护技术措施运行情况。
 <br/>
 <br/>
 《华尔街日报》报导指出，在这一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新规.html" target="_blank">
  新规
 </a>
 下，中共警方将有权对企业进行实体检查，并远程存取企业网络。警方还将被授以复制信息的权力，并检查所谓可能危及国家安全、公共安全和社会秩序的记录。
 <br/>
 <br/>
 旅美中国问题研究人士张健：〝这个所谓的安全监督检查规定，它本身就不是一个法律，用不是法律的方式对个人、企业，可以说包含了所有在中国的，无论是外资还是内资或者是个人，0137只要涉及到所谓的公众安全、社会秩序，用这样的一个藉口，都可以明目张胆的潜入到你的隐私里面去。中共无非是想靠这个所谓的网路安全监督检查规定来威胁、阻吓、要胁迫很多个人和企业，让他们在中共的箝制之下不敢发声。〞
 <br/>
 <br/>
 报导指出，新的网络安全规定将赋予中共政府全方位权力，对企业的信息技术进行检查，并获取专有信息，这可能会加深外国企业对在华业务的担忧。
 <br/>
 <br/>
 国际特赦组织分析师威廉表示，中共去年出台的网络安全法涉及面甚广，新规又充实该法赋予监管部门的权力。他说，新规加强了中共有关部门的检查权，并要求互联网服务提供商和使用网络的公司完全服从中共政府的网络安全特权。
 <br/>
 <br/>
 前香港立法会资讯科技界议员单仲偕：〝其实这个举措是多年前已经开始，中国通过国家安全网络法，这个法规之后，根据它的时间表，一步一步的扩检，对所有网络上的监控，包括所有的在内地的外国公司也要遵守。〞
 <br/>
 <br/>
 报导说，这个新网络安全法要求，供应给中共政府和银行、电信等关键行业的科技产品，都必须接受安全检查，企业也要将数据储存在中国。
 <br/>
 <br/>
 外国企业对这项新规深表不满，认为中共政府可能利用该法迫使外国企业公开源代码以及其他机密，来证明其设备是安全的，进而有可能把相关信息洩露给这些企业的中国竞争对手。
 <br/>
 <br/>
 单仲偕：〝透过这个网络安全法，中共可以看到在互联网所有的东西，方便它能够监管，什么人看什么东西，它们完全可以看到，这对于一些外国的企业来说，当然是造成非常重大的威胁，外国公司在内地要小心。〞
 <br/>
 <br/>
 报导说，为继续进入中国市场，很多外国企业设法满足中共的网络安全要求。微软已在北京启用〝透明技术中心〞，让中共官员查验其产品的安全性。苹果也在贵州建立数据中心，以便遵守中国客户的云数据必须储存在中国的规定。
 <br/>
 <br/>
 中国美国商会主席蔡瑞德说，新规允许中共获取与网络安全相关的任何信息，这种信息的定义实在太广，几乎包括了所有内容。这样一来，企业似乎别无选择，只能就范。
 <br/>
 <br/>
 有推特网友发推文说，这就是强迫技术转让！这就是盗窃知识产权！这正是我们警告中共的，在我看来，他们正在朝我们所期望的背道而驰！
 <br/>
 <br/>
 张健：〝中共从来没有想放弃它对互联网的占有，所以现在推出这样一个非常邪恶的规定。〞
 <br/>
 <br/>
 旅美中国问题研究人士张健表示，从美国副总统彭斯日前的讲话，可以看出美国对于中共盗窃知识产权，已经到了无法容忍的程度。相信在不久的将来，美国会有和过去不一样的做法。
 <br/>
 <br/>
 采访/陈汉 编辑/陈洁 后制/陈建铭
