### 华人携87万欧元回中国 被德国机场没收
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】近日，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_德国杜塞尔多夫机场.html" target="_blank">
  德国杜塞尔多夫机场
 </a>
 ，一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人乘客.html" target="_blank">
  华人乘客
 </a>
 携带
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_87万欧元.html" target="_blank">
  87万欧元
 </a>
 （约677万人民币），并将钱
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伪装成中文书.html" target="_blank">
  伪装成中文书
 </a>
 籍放在行李箱中，安检时被海关发现。由于该华人未对现金进行任何申报手续，这些现金被海关当场没收。
 <br/>
 <br/>
 据德国海关当局发布的消息，11月10日，一名26岁的法籍华人随身携带的登机行李箱，被安检工作人员发现异常，安检透视中显示箱内内有10本书，而且每本书都厚的不太正常。
 <br/>
 <br/>
 于是，工作人员要求这名华人打开他的箱子，发现原来每本书里都装着用塑料袋牢牢包住的巨额现金，一共有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_87万欧元.html" target="_blank">
  87万欧元
 </a>
 （约677万人民币）。由于该华人未对此现金进行任何申报手续，该笔现金被海关当场没收。
 <br/>
 <br/>
 根据欧盟第〝1889/2005号条例〞以及〝海关管理条例〞，超过10000欧元的现金必须要经过口头申报，或者必要情况下根据要求申报。
 <br/>
 <br/>
 据悉，这名华人目前居住在法国，当天，他计划乘坐从德国飞往中国、途经俄罗斯转机的航班。这名华人随后被转交给海关工作人员，经过询问，他交代已经通过邮寄寄出一个箱子，里面也有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伪装成中文书.html" target="_blank">
  伪装成中文书
 </a>
 的巨额现金，一共约22.4万欧元（约174万人民币）。
 <br/>
 <br/>
 由于该华人无法解释清楚这笔巨额现金的来源及携带缘由，海关当场将现金没收并成立相关小组对此案进行调查。不过，虽然钱被没收，但该名华人仍然可以完成既定行程并返回中国。
 <br/>
 <br/>
 其实这样的案例屡见不鲜，2015年11月4日，一名华人市民在罗马达芬奇机场出境回国，在行李箱中携带的37.2万欧元（约290万人民币）被发现，超过10000欧元的部分的一半，也就是18.1万欧元（约140万人民币）被没收。
 <br/>
 <br/>
 同年5月11日,在西班牙首都马德里巴拉哈斯机场，安检人员发现到一名中国籍人士携89万欧元（约692万人民币）现金欲乘机回中国,因其无法提供税务部门开出的资金证明而构成违法行为,有关部门依法将其逮捕。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：文慧)
</div>
