### 陆客扫空澳洲奶粉 当地家长哭求：留点给我们孩子
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】大陆毒
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奶粉.html" target="_blank">
  奶粉
 </a>
 贻害无穷，海外婴儿奶粉大受欢迎。大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代购.html" target="_blank">
  代购
 </a>
 趁机囤购，导致澳洲当地奶粉短缺。近日一位澳洲父亲写信请求：〝拜托你留点奶粉给我们孩子吧！〞
 <br/>
 <br/>
 据澳洲9NEWS近日报导，由于大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代购.html" target="_blank">
  代购
 </a>
 大量囤购
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奶粉.html" target="_blank">
  奶粉
 </a>
 ，导致澳洲当地民众在超市买不到奶粉。10月8日一位澳洲父亲，直接写信向澳洲最大连锁超级市场Woolworths：〝拜托你留点奶粉给我们的孩子吧！〞
 <br/>
 <br/>
 今年5月，当地一对夫妻跑了很多商店和超市都发现奶粉货架空空如也。同时，他们发现在一个仓库堆满了即将运往大陆的奶粉。
 <br/>
 <br/>
 因大陆代购在澳洲的大量购买，将婴儿配方奶粉品牌抢购一空，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲奶粉.html" target="_blank">
  澳洲奶粉
 </a>
 供不应求，让许多当地澳洲父母感到不便和焦虑。
 <br/>
 <br/>
 对于澳洲人来说，奶粉就像其它的日常用品一样，喝完了再买。但是，自从大陆代购大量抢购、囤积奶粉后，澳洲人常常得面对空空的奶粉货架。本来垂手可得的奶粉，现在要跑很多商店，在很偏僻的地方才能找到，家中的婴儿嗷嗷待哺，家长自然焦急万分。
 <br/>

 <br/>
 澳洲民众表示：〝这是怎么回事？我们生活在澳洲当地买不到，却能在超市仓库里看到一堆要送往大陆的奶粉！〞〝我们沦为二等公民了！〞
 <br/>
 <br/>
 这位写信的父亲表示，自从今年8月，Woolworths将每人购买奶粉的数量从2罐改为8罐以后，他一直很难买到奶粉。他希望将限购的数量从8罐降至2罐。
 <br/>
 <br/>
 对此，Woolworths表示，先前规定〝每人限购2罐〞的政策会继续执行，另一间〝Coles〞超市也表示，初生婴儿的配方奶粉只会在服务柜台出售，防止代购商不断折返采购，确保有需要的民众可以买到这些商品。
 <br/>
 <br/>
 2008年，大陆曝光三聚氰胺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒奶粉.html" target="_blank">
  毒奶粉
 </a>
 事件。该事件导致至少6名婴儿死亡，几十万婴儿受害。令大陆人对国产奶粉缺乏信心，不得已购买高价外国奶粉。不仅澳洲，多个国家、地区受到影响，很多国家因此实施限购令。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：祝馨睿）
