### 如何看北京结构改革诚意？美专家：12月18日是关键
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】在阿根廷的美中首脑会谈上，北京当局以进行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性改革.html" target="_blank">
  结构性改革
 </a>
 〞为条件，换取了美方在90天内的临时休战，但外界对北京的结构改革究竟有几分诚意并不乐观。日前，有美国智库专家称，12月18日前后，北京当局会否发表有关改革的重大公告，将是观察其是否有改革诚意的一个关键指标。
 <br/>
 <br/>
 当地时间12月3日，美国智库国际战略研究中心中国研究副主任肯尼迪（Scott Kennedy）在参与美国财经电视台CNBC的《财经论坛》（Squawk Box）节目时分析称，12月18日是中共政府设定的〝改革开放纪念日〞，而今年恰恰是其〝改革开放〞40周年，因此这个日子可作为外界观察北京当局有没有诚意进行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_结构性改革.html" target="_blank">
  结构性改革
 </a>
 的一个关键点。
 <br/>
 <br/>
 他说：〝如果那一天或那一周（北京）没有新的重大公告，那么我们就知道中方不可能以此为契机改变其经济和产业政策方向。〞 他接着又表示，如果是这样，中美之间的关税战会将在90天结束后再次升级。
 <br/>
 <br/>
 肯尼迪表示，除了观察中方在12月18日前后的动作外，华盛顿方面的事态发展也可能会影响美中关系的立场。中期选举后控制了众议院多数席位的民主党，将如何与川普政府合作也很重要。
 <br/>
 <br/>
 此前，美国总统川普与中国国家主席习近平12月1日在阿根廷会晤后，双方宣布暂停相互加征新关税。美国政府原定从明年1月1日起把对中国2,000亿美元产品征收的关税从10%提高至25%，现在则决定在明年1月1日之后继续维持10%的关税。
 <br/>
 <br/>
 作为交换条件，北京当局在接下来的90天内，需要通过谈判与美国政府就中方进行结构性改革达成协议。如果做不到这一点，那么在90天的期限结束后，美中之间的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 将继续打下去，而且还会升级。
 <br/>
 <br/>
 12月4日，川普发推文宣布，〝（美国）与中国的谈判已经开始。〞
 <br/>
 <br/>
 这篇推文还强调，这次美中之间的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易谈判.html" target="_blank">
  贸易谈判
 </a>
 ，将在阿根廷晚宴之后的90天内结束，除非延期。而美国方面领衔谈判的莱特希泽（Bob Lighthizer）将与努钦（Steve Mnuchin）、库德洛（Larry Kudlow）、罗斯（Wilbur Ross）和纳瓦罗（Peter Navarro）等人密切合作，来完成这项谈判任务。
 <br/>
 <br/>
 此前，英国《金融时报》曾援引3位知情人士透露，习近平和川普达成休战协议后，北京方面将由中共国务院副总理刘鹤为首席贸易谈判代表，率领由30名中国官员组成的代表团前往美国首都华盛顿执行谈判任务。首轮谈判的时间则暂定于12月12日至15日。
 <br/>
 <br/>
 日前有媒体在中共外交部的例行记者会上问及上述消息是否属实，中共外交部发言人耿爽回答称：〝（中美）双方工作团队会按照两国元首达成的共识加紧接触，争取早日达成互利共赢的协议。至于双方接触的具体安排，建议你向主管部门询问。〞
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
