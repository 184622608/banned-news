### 朱军涉性侵被正式起诉 当事人要求朱本人出庭
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】本周二，中共央视主持人朱军性骚扰案的当事人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弦子.html" target="_blank">
  弦子
 </a>
 正式向法庭申请起诉朱军并获得立案。据陆媒称，此案将在10月25日开庭。
 <br/>
 <br/>
 当地时间10月23日，朱军性骚扰案的当事人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弦子.html" target="_blank">
  弦子
 </a>
 在其代理律师的陪同下，前往北京市海淀区复兴路法庭，就人格权纠纷申请立案起诉朱军，并且要求朱军本人出庭。
 <br/>
 <br/>
 据中国大陆媒体新浪娱乐报导，弦子的代理律师王律师介绍说，这个案件控方申请朱军到庭的理由十分充分。首先，案件发生在密闭空间，需要双方解释，其次，朱军本人也没有不能到场的客观障碍。若朱军不到庭，那他可能需要承担一些不利后果。
 <br/>
 <br/>
 弦子在接受陆媒采访时也表示，在法庭上自己会尽量冷静克制，以正常态度面对朱军，希望法庭最终能还自己一个公道。
 <br/>
 <br/>
 陆媒报导称，此案将在10月25日开庭。
 <br/>

 <br/>
 据公开的资讯，今年7月26日，微博用户〝麦烧同学〞转发了弦子举报自己4年前遭中共央视主持人朱军性侵猥亵的长文，率先曝光了2014年6月朱军在央视的化妆室里，对化名〝弦子〞的女生实习生隔着衣服抚摸猥亵并强吻的丑闻。
 <br/>
 <br/>
 据长文爆料，弦子当年遭性侵后曾在老师和朋友的陪同下报警，警方也曾立案并进行了取证。但最终警方还是以朱军具有〝对社会巨大正面影响力〞为由，规劝弦子放弃起诉。先是学校相关老师找该女生谈话劝其放弃起诉，接着取证的警察也被更换，然后此事不了了之。
 <br/>
 <br/>
 朱军涉嫌性侵的丑闻再次被弦子和她的朋友在网络上曝光后，随即引发广泛关注与社会舆论的热议。
 <br/>
 <br/>
 今年8月15日，北京星权律师事务所受朱军委托发布律师声明，称近网络上出现的有关朱军性骚扰实习生的消息为〝不实信息〞，该律师事务所现已针对上述情况取证调查，将通过法律途径追责谣言散布者。
 <br/>
 <br/>
 9月26日，朱军案当事人朋友在微博晒出一张截图，公开了朱军方起诉状的诉讼请求部分，内容显示，朱军要求〝麦烧同学〞与〝弦子与她的朋友们〞删除针对原告发布的侵权内容，并公开赔礼道歉，并要求两人赔偿65.5万元整。
 <br/>
 <br/>
 10月22日，与弦子一起被朱军起诉的爆料人〝麦烧同学〞在微博发帖称，朱军案10月25日的交换证据改在北京海淀法院本院进行。
 <br/>
 <br/>
 10月23日，弦子对外界公开了自己已正式起诉朱军的消息。
 <br/>
 <br/>
