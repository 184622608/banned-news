### 辽宁再现〝秦岭别墅案〞 党媒指顶风作案
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安.html" target="_blank">
  西安
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦岭别墅案.html" target="_blank">
  秦岭别墅案
 </a>
 〞引发陕西官场震荡风波未平，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁.html" target="_blank">
  辽宁
 </a>
 再爆类似违法工程。近日，中共党媒刊文痛批当地政府顶风作案，召开专题会议加快推进违法项目建设。
 <br/>
 <br/>
 党媒22日刊文称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁.html" target="_blank">
  辽宁
 </a>
 下辖的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_东戴河新区.html" target="_blank">
  东戴河新区
 </a>
 存在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建项目.html" target="_blank">
  违建项目
 </a>
 ，为了应付上级检查伪造文件。文章说，当地官方一边编造假文件，上报省市政府谎称违法项目已停止建设；一边顶风作案，召开专题会议加快违法施工进度……。
 <br/>
 <br/>
 一个县级违建项目，本应是在上级省市的监管调查范畴之内，但党媒此时发声，让外界将〝东戴河违建〞与震惊中央的〝秦岭别墅〞事件产生联系。
 <br/>
 <br/>
 被历代帝王尊为华夏文明龙脉的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西安.html" target="_blank">
  西安
 </a>
 秦岭，因违建别墅问题，中纪委曾数次专门派员开展〝专项整治〞，习近平6次批示仍无法清拆。近期中纪委副书记徐令义，两次赴陕西指导拆除工作，同时引发陕西官场大地震，目前震荡仍在持续中。
 <br/>
 <br/>
 在中共高层接二连三向〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦岭别墅案.html" target="_blank">
  秦岭别墅案
 </a>
 〞问责之际，辽宁官场被党媒怒批顶风作案，引发人们对中共高层和地方之间的博弈困境的关注。
 <br/>
 <br/>
 陆媒称，中央生态环保督察组近日在辽宁〝回头看〞督察发现，葫芦岛市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绥中.html" target="_blank">
  绥中
 </a>
 县政府、东戴河新区管委会在违法围填海问题整改中，阳奉阴违问题严重。
 <br/>
 <br/>
 据生态环保部19日发布一则案例通报中的东戴河，是辽宁省葫芦岛市绥中县的一个风景区，根据媒体报导说，此前那里没有商业开发，而是一个距离北京298公里处的近海盐硷地，也不叫〝东戴河新区〞而是叫绥中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_滨海经济区.html" target="_blank">
  滨海经济区
 </a>
 。
 <br/>
 <br/>
 但在当地政府活跃政策引进开发商后，开发商为了营销需要，便利用其与北戴河同处渤海边，且距离相近的地理优势，借势北戴河，生造出了一个东戴河的概念。
 <br/>
 <br/>
 报导称，提出这个概念的是参与〝东戴河违建〞的企业之一佳兆业公司。
 <br/>
 <br/>
 有关〝东戴河违建〞的问题，中共中央环保督察去年7月在辽宁期间，就曾反馈，东戴河新区管理委员会违法，将39.6公顷沿海滩涂，转让给佳兆业公司等3家企业，用于房地产开发，并由绥中县国土资源部门为其办理了土地使用证，并要求其进行整改。
 <br/>
 <br/>
 对此，辽宁省在去年8月出具了整改方案，明确表示停止相关项目建设。
 <br/>
 <br/>
 2018年6月，辽宁省上报整改落实情况时声称，涉事违建的项目已按照要求停止建设。然而在7月份，中共环境部抽查时，发现辽宁的违建项目不仅没有停止建设，反而在加快施工进度。
 <br/>
 <br/>
 而绥中县的这个项目是违法填海项目，去年5月，第一轮中央环保督察时，才刚刚开始土建施工，如今，商业街项目已经基本建成，酒店会议中心项目1至8号楼主体也已封顶。
 <br/>
 <br/>
 中央生态环保督察组在辽宁〝回头看〞督察之后，这些违法建筑已被实施拆除中。目前，包括绥中县时任县长在内的23人被问责。与〝秦岭别墅案〞 类似，目前当地官场仍在持续震荡中。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
