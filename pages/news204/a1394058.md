### 中共驻美大使改口：北京愿让步 有准备达成协议
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】北京拒不纠正不公平的贸易政策，美国政府继续加大施压。北京虽然在暗中干预美国中期选举，但在公开立场上避免和美国直接对抗。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共驻美大使.html" target="_blank">
  中共驻美大使
 </a>
 日前强调中美合作，并称愿意和美国达成协议，但不过老调重弹，并无实质让步。
 <br/>
 <br/>
 当地时间10月3日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共驻美大使.html" target="_blank">
  中共驻美大使
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔天凯.html" target="_blank">
  崔天凯
 </a>
 在美国国家公共电台（NPR）总部接受采访时说，北京〝准备好达成一份协议〞，结束与美国的贸易战。中共也愿意做出一些让步，包括采取措施削减美中贸易逆差，承诺改革中国经济。
 <br/>
 <br/>
 不过，崔天凯再次强调，达成协议的前提是，美国〝必须阐明到底想要什么〞，华盛顿要派出一名能让中方〝信任〞的谈判官员。
 <br/>
 <br/>
 观察人士认为，从崔天凯的表态来看，不过是重复中共之前的立场，并没有表现出更多诚意。
 <br/>
 <br/>
 据此前报导，直到最近第四轮贸易谈判，美方向中方提出的贸易要求，也不过是重复5月首轮谈判的要求。白宫也不断强调美中贸易逆差、中共政府补贴、贸易壁垒和盗窃知识产权等问题，立场一直很明确。但中共官方的说法却一直是〝不知道
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 到底要什么〞。
 <br/>
 <br/>
 在以往的贸易谈判中，中方也不断重复表态，称愿意通过增加采购美国商品来降低贸易逆差，并承诺有限地开放市场，但是一直拒绝在政府补贴、知识产权等关键问题上做出实质让步，导致中美对话一再谈崩。
 <br/>
 <br/>
 日前，美国总统川普公开表示，中方现在渴望和美方达成协议，但他认为〝还不到时候〞，暗示将继续施压直到北京真正妥协。
 <br/>
 <br/>
 因此，有分析指，崔天凯的上述言论，其真正目的并非面向白宫表态，而是意欲面向国际搏取同情，占据舆论制高点，扭转当前的被动局面。
 <br/>
 <br/>
 9月29日，崔天凯也在中共驻美大使馆的〝十一〞庆典上表示，中美关系又到了需要从新选择的〝历史性时刻〞，中共选择了〝合作〞、〝不对抗〞，要看美国如何选择。
 <br/>
 <br/>
 而出席庆典的白宫国安会亚洲事务资深主任马特．波廷杰当场表示，中美战略竞争已是川普总统最重要的国家安全战略，美国已选择了竞争。
 <br/>
 <br/>
 9月以来，川普政府在军事、政治、外交领域接连推出反制中共的举措，中美贸易战开始向全面对抗扩散，新一轮冷战格局正在形成。
 <br/>
 <br/>
 早在8月出台的美国新国防法案中，就已明确与中共〝长期战略竞争〞是美国根本性的优先战略，美国将在军事、经济、外交和法治领域和中共长期对抗。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
