### 【禁闻】封自媒体灭推 中共严控网络言论
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 互联网信息办公室（简称-网信办）近期查封了9,800多个自媒体帐号，并对微信、微博等自媒体平台提出严重警告。同时有网友披露，公安部门正在进行一项〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灭推.html" target="_blank">
  灭推
 </a>
 〞行动，警告大陆〝推特〞用户，不准发表所谓〝不当〞的言论。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 互联网信息办公室11月12号通过官网公布，从10月20日起，已经全网处置了9,800多个自媒体帐号。其中包括：〝唐纳德说〞、〝傅首尔〞、〝紫竹张先生〞、〝有束光〞、〝万能福利吧〞、〝野史秘闻〞、〝深夜视频〞等知名帐号。
 <br/>
 <br/>
 另外，〝网信办〞还约谈了腾讯微信、新浪微博等自媒体平台，严重警告他们所谓〝疏于管理〞。而相关平台负责人却也反应，将会〝自查自纠〞，严格管理。
 <br/>
 <br/>
 旅美学者吴祚来表示，目前的迹象显示，中共设置的这些网络审查机构，正在扩大和纵深打击网络言论。
 <br/>
 <br/>
 旅美学者吴祚来：〝这些自媒体当然情况都是不同。但是比如说所谓的‘丑化中共的历史’，或者是‘政治上有害’，这个都不成为（处置）理由 。中共它伪造了很多历史，现在很多的人通过微博、微信就是在恢复历史真相。还有这个‘批评政府’，这个在全世界文明国家都是一个常态。〞
 <br/>
 <br/>
 网络自由观察人士古河表示，当局通常以所谓的〝造谣〞、〝传谣〞为由，封杀网络帐号，但当局的封锁，才是产生所谓谣言的原因。
 <br/>
 <br/>
 网络自由观察人士古河：〝因为封锁，他们（网民）的来源就比较少，所以就会产生一些误判。把这些误判当作胡说八道其实是不对的。正常的社会应该是言论自由，当所有的言论都自由的时候，那些胡说八道的自然就会被人们识破。〞
 <br/>
 <br/>
 目前，当局不仅消灭国内社交平台上的不同声音，甚至公安部门开始针对海外〝推特〞用户进行〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_灭推.html" target="_blank">
  灭推
 </a>
 〞行动。有用户透露，在推特上比较活跃的人士，都被警方约谈，他们受到警告，有些人因此被抓。
 <br/>
 <br/>
 大陆推友陈先生：〝在被抓之前就已经在全国大摸底了。在放出来之前就已经全部交代清楚了：不允许转发政治言论，不要去发表任何的政治观点。而且非常明确，没有第二次。如果再不听的话，那个寻釁滋事罪名直接套在你头上，想判几年是随意的。〞
 <br/>
 <br/>
 古河也因此被警方找了两次。他透露，当局还使用经济手段压制。
 <br/>
 <br/>
 古河：〝某个省出现一个情况，国保对当事人说，你上吧，因为我没有办法每时每刻都盯着你。你上一个小时，我罚你一千块。每天上一个小时，一个月就是三十个小时。你算算看有多少钱？谁能受得了呢？所以他们是政治上、经济上双管齐下，封杀言论自由。〞
 <br/>
 <br/>
 古河认为，当局可能以大陆推特用户作为突破口，第二步范围将扩大到脸书或其它海外社交、通讯软件用户，第三步则将全面禁止〝虚拟专用网路〞（VPN），阻断大陆网民在海外发声。
 <br/>
 <br/>
 陈先生：〝现在在中国没办法说言论。大家就像一种潜规则那样。比如说，倒楣没有落在你头上的时候，反正就不要发声。不要说话你就能保住命，大家就这样一个心态。如果你敢于发声，甚至你还有一点点反抗的意识，那牢狱之灾基本上是铁定的。〞
 <br/>
 <br/>
 吴祚来表示，这是由于中共很恐惧。它目前正在进行覆盖性的欺骗宣传，只要有人从推特上搬来真相信息，通过网络传播，对他们就是威胁。所以当局才耗费巨资管控网路，试图阻挡网民与世界的交流。
 <br/>
 <br/>
 采访/陈汉 编辑/尚燕 后制/李沛灵
