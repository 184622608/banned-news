### 中国甘肃沙尘暴掀起百米高〝沙墙〞 场面震撼（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】中国甘肃地区近日再次发生严重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙尘暴.html" target="_blank">
  沙尘暴
 </a>
 。当地网友拍摄的视频显示，狂风掀起的沙尘俨然如一堵高达百米的移动〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙墙.html" target="_blank">
  沙墙
 </a>
 〞，从天边席卷而来，转眼间张掖市就被铺天盖地的沙墙吞噬，宛如灾难大片中惊心动魄的场景。
 <br/>
 <br/>
 据陆媒报导，中国长达近千公里的甘肃河西走廊自西向东遭遇大风沙尘天气袭击，多地出现犹如世界末日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙尘暴.html" target="_blank">
  沙尘暴
 </a>
 天气。
 <br/>
 <br/>
 当地时间11月25日下午约4点左右，大风挟裹着沙尘形成了一堵近百米高的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙墙.html" target="_blank">
  沙墙
 </a>
 〞，向位于河西走廊中段的张掖市席卷而来。行驶在公路上的一辆汽车的行车纪录器，拍下了沙尘暴袭来的画面。
 <br/>
 <br/>
 视频画面中，大风卷着黄沙从远处滚滚袭来，县城逐步被沙尘吞噬。一时间，天地相连〝一片昏黄〞，户外民众〝四处奔逃〞躲避。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-27/p9144631a715535629.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-27/p9144631a715535629-ss.jpg"/>
   <br/>
   <font size="-1">
    中国甘肃地区近日再次发生严重沙尘暴，狂风掀起的沙尘俨然一堵高达百米的移动〝沙墙〞，从天边席卷而来，场面震撼。（视频截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 据全国城市空气质量实时发布平台的消息，25日19时的监测数据显示：甘肃河西地区的嘉峪关、酒泉、张掖、金昌、武威空气指数已达到〝爆表〞的程度，属于严重污染。
 <br/>
 <br/>
 26日11时，包头、鄂尔多斯、兰州、白银、武威、张掖、平凉、酒泉、定西、西宁、银川、吴忠、中卫、固原等城市空气质量指数均达到500，其中武威、包头、兰州、白银、西宁、海东地区、吴忠等地PM10小时浓度已破千。
 <br/>
 <br/>
 其中，武威市PM10小时浓度为每立方米1708微克，西宁PM10小时浓度为每立方米1279微克，金昌市PM10小时浓度为每立方米1212微克，兰州PM10小时浓度为每立方米1008微克。
 <br/>
 <br/>
 由于空气检测中，空气质量指数设置的最高数值为500，所以一旦相关数值超过500就被称为〝爆表〞。
 <br/>
 <br/>
 至当地时间26日傍晚，酒泉、嘉峪关、张掖、武威等市已相继发布沙尘暴黄色预警。
 <br/>
 <br/>
 据中国环境监测总站则预测，从26日下午至27日，沙尘暴的影响将不断东移，扩散到京津冀及周边区域，并使该地区空气质量进一步恶化。
