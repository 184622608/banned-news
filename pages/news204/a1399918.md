### 大陆酒店卫生乱象没解决 曝料人反被〝解决〞了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】近日，微博知名网友@
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花总.html" target="_blank">
  花总
 </a>
 丢了金箍棒（简称：花总）在网路发布的一段关于中国五星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒店卫生.html" target="_blank">
  酒店卫生
 </a>
 乱象的视频，捅翻了马蜂窝，掀起舆论热潮。事发后，酒店卫生乱象的问题尚未解决，花总倒先被人〝解决〞了，他的个人信息被酒店泄漏。
 <br/>
 <br/>
 11月14日晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花总.html" target="_blank">
  花总
 </a>
 发布一段《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杯子的秘密.html" target="_blank">
  杯子的秘密
 </a>
 》视频，视频中显示，14家大陆五星酒店的清洁工，都上演了用同一块脏抹布、或顾客用过的脏浴巾，擦拭杯子、洗手台、镜面、座便器等情节。
 <br/>
 <br/>
 令人震惊的是，这14家顶级酒店每晚的房价从1千多元到5千元左右，被视为高端消费场所。包括北京康莱德酒店、北京柏悦酒店、北京王府半岛酒店、北京颐和安缦酒店、贵阳贵航喜来登酒店、南昌喜来登酒店、福州香格里拉酒店、上海宝格丽酒店、上海四季酒店、上海浦东文华东方酒店、上海世茂皇家艾美酒店、上海璞丽酒店、上海浦东丽思卡尔顿酒店、上海外滩华尔道夫酒店。
 <br/>
 <br/>
 〝现实比视频展现出来的更加糟糕。〞花总说。
 <br/>
 <br/>
 他对陆媒透露，一共暗访了30多家五星酒店，曝光的是比较有代表性的14家，事实上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒店卫生.html" target="_blank">
  酒店卫生
 </a>
 乱象的波及面将近100%。
 <br/>
 <br/>
 出门在外住酒店，泡茶、喝咖啡，晚上洗漱、刷牙，再平常不过。然而，与你嘴唇亲密接触的杯子，却是拿用过的浴巾或擦过马桶的抹布擦拭透亮，这令人作呕的一幕被曝光后，立即掀起舆论轩然大波。
 <br/>
 <br/>
 有网民发文称，这次事件让人们的高端信任感崩塌了。视频中清洁员的作业，显露出这些〝高端〞酒店的〝低端〞本质。
 <br/>
 <br/>
 花总本人表示，以前也了解酒店行业的乱象，但没想到自己住的五星级酒店也是这样。他说，〝去年在江苏一间酒店，吃完午饭回到客房，正好撞见做卫生的大姐在用我洗澡用过的浴巾擦杯子，当时我就有些不舒服，因为浴巾也擦过脚什么的。〞
 <br/>
 <br/>
 随后，花总花了3个月的时间，秘密拍摄了〝30家多家〞酒店，除了两家宾馆外，〝全体沦陷〞。
 <br/>
 <br/>
 一位在丽江、大理以及重庆等地从事多年酒店管理的人士向BBC中文表示，酒店行业〝一块抹布擦所有〞非常普遍，一些中低档酒店甚至会培训清洁人员这么做。这次引起这么大反响，是因为曝光酒店数量多，而且都是顶级酒店。
 <br/>
 <br/>
 事后，14家酒店中的13家发布声明，承认存在问题，并表示整改。但花总说，〝以后在搞这种监督已经不可能了。事后就已经有酒店把我的个人信息暴露出来。〞
 <br/>
 <br/>
 实际上，在中国大陆，有关酒店卫生标准和要求都有明文规定，但显然执行成了一个问题。有网友评论说，出了问题后，卫生监督人员拿着设备上门取样和约谈就像〝紧急作秀〞。如果监管只是一阵风，那么这些酒店也只会〝虚心接受，屡教不改〞。
 <br/>
 <br/>
 还有网民讽刺说：〝等这风头过去了，依旧原先怎样擦就怎样擦，难不成擦的时候旁边站上个监督的？〞
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
</div>
