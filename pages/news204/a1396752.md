### 广东万人抗议建高危物品垃圾处理中心
------------------------

<div class="wysiwyg">
 【新唐人2018年10月25日讯】首先来关注中国的消息。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广东.html" target="_blank">
  广东
 </a>
 佛山市顺德区杏坛镇两万多人，10月23号和24号，连续两个晚上上街游行，抗议在麦村建
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高危物品.html" target="_blank">
  高危物品
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_垃圾处理中心.html" target="_blank">
  垃圾处理中心
 </a>
 。
 <br/>
 <br/>
 当地政府10月16号在麦村公示，要建一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高危物品.html" target="_blank">
  高危物品
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_垃圾处理中心.html" target="_blank">
  垃圾处理中心
 </a>
 ，引发村民不满。村民在村里游行抗议后，有五、六名村民被警方抓走。
 <br/>
 <br/>
 10月23号起，杏坛镇村民聚集在镇中心的文化广场，在繁华的主干道上游行示威，他们手持反对的标语，高喊口号抗议。
 <br/>
 <br/>
 麦村村民麦先生估计，游行人数有两万多人。
 <br/>
 <br/>
 麦村村民麦先生：〝这两天都是啊，两万多(人游行)。〞
 <br/>
 <br/>
 麦村另一位麦先生表示，连续两个晚上警察没有和村民发生冲突，24号当地政府调动了更多的警力。
 <br/>
 <br/>
 麦村村民麦先生：〝昨天晚上几万人去游行，刚开始的时候，出动了很多很多警察，后来不知道什么原因，走了一部分，我们抗议了一两个小时，从八点多的时候开始到九点多，快十点，带领的人，好像昨天晚上被抓，不知道去什么地方。〞
 <br/>
 <br/>
 麦村村民还表示，当地已经饱受很多工厂的污染，值得关注的是杏坛镇是重要的水源地，该项目一旦建成，珠江三角地区居民的饮用水安全将受到威胁。
 <br/>
 <br/>
 麦先生：〝政府方面也没什么说法。是很近啊！很近啊！（离居民区）只有两、三百米，又是河道旁边，还是吃用水的旁边，取水点很近啊。〞
 <br/>
 <br/>
 根据政府公告，该项目总投资为8亿元，设计处理规模为每天417吨垃圾。据悉，目前已有7万人联名反对该项目入驻杏坛镇。麦村还出现身穿红色马甲的〝工作人员〞，以红包利诱村民在一张空白纸上签字画押。
 <br/>
 <br/>
 新唐人记者熊斌、陈杰采访报导
</div>
