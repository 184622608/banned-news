### 【禁闻】〝国进民退〞 50上市公司改姓〝国〞
------------------------

<div class="wysiwyg">
 【新唐人2018年11月02日讯】从今年初开始，中国已经有50几家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上市公司.html" target="_blank">
  上市公司
 </a>
 由〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民企.html" target="_blank">
  民企
 </a>
 〞转为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国企.html" target="_blank">
  国企
 </a>
 〞，多数股权转由国有资本掌握。不过，近来，中共领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 四度喊出保护民营经济，试图是在为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 论〞灭火。
 <br/>
 <br/>
 据了解，从今年1月至10月期间，中共地方政府下属的资产管理公司，已向50多家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上市公司.html" target="_blank">
  上市公司
 </a>
 投资300亿元人民币。大约有一半的上市公司将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国企.html" target="_blank">
  国企
 </a>
 列为最大股东。
 <br/>
 <br/>
 外界质疑，这些地方政府并不是在危急情况下拯救私营企业，而是以极低价格攫取私人企业。
 <br/>
 <br/>
 流亡美国之中国民营企业家肖运军：〝就是明着把私企变成国企，就是这样的现象。只不过是做的冠冕堂皇了一点，稍微冠冕堂皇一点，没有直接抢了。〞
 <br/>
 <br/>
 对于这种新形式的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 ，中共当局否认这一点。11月1号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 主持召开民营企业座谈会，他要所有民营企业和民营企业家〝吃下定心丸、安心谋发展〞。
 <br/>
 <br/>
 习近平强调，非公有制经济在中国经济社会发展中的地位和作用没有变。他还在会上昵称民营企业和民营企业家是〝我们自己人〞。
 <br/>
 <br/>
 而针对民营企业融资难、融资贵这个〝老大难〞的问题，他也提出了关注。尤其对有股权质押平仓风险的民营企业，习近平强调，金融机构应避免发生企业所有权的转移，也就是说，是〝财务救助〞，不是抢夺。
 <br/>
 <br/>
 肖运军：〝不可能
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民企.html" target="_blank">
  民企
 </a>
 跟国企是一样的，它每次就是说得很好听，就是流动很多资金到市场因应。它现在走的就是国有化的道路，它不可能让私企跟国企有竞争力。然后国企它就可以亏本经营，一直把私企搞垮为止，然后它就夺取，国家垄断经营。〞
 <br/>
 <br/>
 中共官媒报导说，习近平在座谈会表示，公有制为主体、多种所有制经济共同发展的基本经济制度，是中国特色社会主义制度的重要组成部分，也是完善社会主义市场经济体制的必然要求。
 <br/>
 <br/>
 台湾企业家高为邦：〝民营企业是为自己工作，每个人非常努力，所以这个经济活起来，经济成长就往上飙升。如果全部国进民退，大部分都变成国有企业的话，它们没有竞争力，管理的效率不高，那就是更惨了，这已经证明共产主义是失败的主义。〞
 <br/>
 <br/>
 微信公众号〝侠客岛〞发表文章说，北京的这场会议对眼前的经济形势有个基本判断，也就是：当前的经济运行稳中有变，经济下行压力有所加大，部分企业经营困难较多，长期积累的风险隐患有所暴露。
 <br/>
 <br/>
 高为邦：〝因为国营企业之所以能生存，它是靠特权，它在银行贷款方面，它有优势，民营企业往往没有优势，付出的代价高。现在又再回去走共产主义，所以我觉得整个来讲，中国的前景实在是不看好啦。〞
 <br/>
 <br/>
 旅美政经分析人士秦鹏认为，在中共〝党管一切〞，〝国有资本是党的执政基础〞面前，民营企业只是配角。
 <br/>
 <br/>
 旅美政经分析人士秦鹏：〝陪太子读书的角色。而且，要从中共马克思原教旨主义说的共产主义的意义上来说，就是中共对民营企业的这种态度，实际上是一脉相承，因为共产主义就是要‘消灭私有制〞，虽然看起来血淋淋，但是现实它是活生生。〞
 <br/>
 <br/>
 今年是中共改革开放40周年，回顾一个多月以来，习近平已经四度喊出民营经济，试图为〝国进民退〞论灭火。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/周天
</div>
