### 彭斯区分中共与中国 川普留薄面 双边关系达〝危险〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国副总统.html" target="_blank">
  美国副总统
 </a>
 彭斯周四演讲痛批中共，引爆舆论。外界观察，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯演讲.html" target="_blank">
  彭斯演讲
 </a>
 中特意对〝中共〞和〝中国〞做了区分，显示针对的是中共政权。港媒指，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 没有亲自上阵，给北京保留一些情面。而彭斯演讲的背景是中美关系已升至〝危险层级〞。
 <br/>
 <br/>
 彭斯周四（10月4日）演讲，阐述了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府最新、最完整的对中政策，细数中共的种种罪状，对中共进行了全方位的抨击，涉及内容包括贸易不公、南海军事化、侵犯宗教自由、干预美国政治、破坏美国自由等等。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯演讲.html" target="_blank">
   彭斯演讲
  </a>
  区分〝中共〞和〝中国〞
 </h4>
 <br/>
 彭斯演讲中刻意对〝中共〞和〝中国〞进行了区别，每指出中方一种恶劣行径时，都点名一次〝中国共产党〞，而非依惯例笼统地称呼〝中国〞或〝中国政府〞。
 <br/>
 <br/>
 据港媒《经济日报》报导，彭斯提到中共花费巨资在美国与其它国家进行宣传时，点名在美国许多主要城市播出的中国环球电视网（CGTN），是依共产党大佬的命令行事。
 <br/>
 <br/>
 虽然演讲中一直痛批中共的恶行，但彭斯在最后表示，相信美国人民和中国人民之间的持久友谊，希望共同促进中美两国的繁荣，并称〝（中共治下的）中国人民理应得到更多〞。
 <br/>
 <br/>
 <h4>
  港媒：给北京保留情面
 </h4>
 <br/>
 《经济日报》10月5日文章指，显然中美冲突已从贸易战和南海冲突，延伸至意识形态领域，正在走向全面开战。
 <br/>
 <br/>
 不过，文章认为，川普还是保留了情面与谈判空间，并没有亲自上场与北京撕破脸，而是派彭斯扮演黑脸，目的是把北京逼上谈判桌，在〝贸易方面〞让步。
 <br/>
 <br/>
 事实上，彭斯在演讲结尾，也为中美关系保留了余地。他说，〝（中美）竞争并不总是意味着敌意〞，川普总统希望共同促进双方的繁荣与安全，而不是分离。北京领导人现在仍可以改变路线，回归当初的〝改革开放〞精神。
 <br/>
 <br/>
 最后，彭斯还呼吁，要相信川普总统〝与中国国家主席建立起的关系〞等。
 <br/>
 <br/>
 不过，从其演讲内容看，川普政府关注的已不再仅仅是〝贸易方面〞。
 <br/>
 <br/>
 彭斯列举的中共恶行，还包括台海军事行动、干预美国选举、试图让美国换总统、对美国进行政治渗透和网络攻击等等。
 <br/>
 <br/>
 彭斯要求北京用实际行动〝重新尊重美国〞，并警告称，在美中关系达到〝公平、对等和尊重我们主权〞之前，美方绝不会让步。
 <br/>
 <br/>
 <h4>
  中美关系已升至〝危险层级〞
 </h4>
 <br/>
 进入9月以来，川普政府在军事、外交、经济、政治领域接连推出反制中共的措施，中美对抗明显升级。
 <br/>
 <br/>
 川普公开谴责中共干预美国中期选举，并警告北京要么公平交易，要么断绝生意来往。川普还对媒体公开表示，和北京领导人习近平〝可能不再是朋友〞。
 <br/>
 <br/>
 与此同时，中美军事对抗持续升温，更是令外界直呼〝危险〞。
 <br/>
 <br/>
 9月30日，中美战舰在南沙群岛附近海域发生一次〝不安全〞互动。
 <br/>
 <br/>
 原定10月中旬举行的中美第二轮外交安全对话已经搁浅。
 <br/>
 <br/>
 美国日前对中共红二代军头、军委装备发展部部长李尚福实施经济制裁，并提出3.3亿美元的对台军售。而中方则召回访美的海军司令沈金龙，并拒绝美国海军军舰10月访问香港的申请。
 <br/>
 <br/>
 此外，美国派遣美国空军B-52重型远程轰炸机，在三天内两度飞越南海。另一架B-52轰炸机与日本自卫队的战机，还在有争议的东海举行了联合演习。
 <br/>
 <br/>
 CNN报导称，美国军方正在研讨11月在全球范围展开联合军演，震慑中共。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
