### 中美〝口水战〞搅黄APEC峰会？史上首次未发表宣言
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】由于美中两国在贸易和投资上的分歧，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在2018年亚太经济合作组织（APEC）峰会期间进行了激烈交锋，致使本届峰会未按惯例发表领导人宣言。
 <br/>
 <br/>
 11月18日，2018年APEC亚太经合组织领导人峰会在巴布亚新几内亚闭幕，会议没有按照惯例发表领导人宣言。这是自亚太经合组织首脑会议自1993年首度召开以来，第一次未发表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_首脑宣言.html" target="_blank">
  首脑宣言
 </a>
 。
 <br/>
 <br/>
 《俄罗斯新闻社》18日引述与会消息人士表示，峰会最终通过了一份不完整的领导人宣言，内容不包括有争议性的问题，例如涉及《茂物宣言》中确立的贸易和投资自由化目标，关于世贸组织（WTO），贸易惯例等有争议的问题。
 <br/>
 <br/>
 在回答记者询问是哪些APEC
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_成员国.html" target="_blank">
  成员国
 </a>
 不同意首脑宣言内容时，主办国巴布亚新几内亚总理欧尼尔表示是〝众所周知的两大经济体〞。
 <br/>
 <br/>
 欧尼尔接着说明了难以达到共识的原因，他说是因世界贸易组织和相关改革问题。但〝这不在APEC的职权范围内〞。他还表示，美中关系紧张令全世界担心。
 <br/>
 <br/>
 有观察人士表示，由于中美在贸易和发展模式上激烈交锋，导致拥有21个成员的亚太经合组织，首次不能在峰会闭幕时发表共同领袖宣言。
 <br/>
 <br/>
 根据《美联社》获得的美国版宣言草稿，美国指控中共采取不公平贸易行为，希望以强烈措辞让中共改变其行为方式。
 <br/>
 <br/>
 美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike Pence）和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 17日在峰会上激烈交锋。习近平谈了中美贸易战和北京所倡议的〝一带一路〞等问题。
 <br/>
 <br/>
 彭斯直接点名批评中共的贸易政策，称中共设置的贸易壁垒史无前例，在中共改变行为方式前，美国不会改变征收关税的政策。并且抨击中共的〝一带一路〞是给合作国家设立的债务陷阱。在离开巴布亚新几内亚时，他还列举了美中之间存在的分歧。
 <br/>
 <br/>
 彭斯在峰会期间在新加坡接受专访时，也曾警告中共，必须全面改变其行为作法，才能避免与美国发生新冷战。
 <br/>
 <br/>
 彭斯强调，11月底的二十国集团（G20）峰会期间，〝习川会〞上，除了讨论中美两国贸易纠纷问题外，中方也必须在保护知识产权、禁止强迫技术转让、取消限制市场准入、遵守国际规则、及保障国际水域航行的自由等问题上让步。
 <br/>
 <br/>
 他还警告，如果中方的提议不能令美方满意，美国已经准备好在经济、外交及政治等多个层面对中共施加更大压力。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
 <br/>
