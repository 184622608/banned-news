### 老红军脑洞大开  称长征用猪油发电机一夜爆红
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】近日，在中国大陆网络社交媒体上突然热传一段视频，当中一个老红军在中共央视讲当年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红军长征.html" target="_blank">
  红军长征
 </a>
 的故事，声称当时物质条件艰苦，没有汽油就用柴油或
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪油发电.html" target="_blank">
  猪油发电
 </a>
 。这个说法立即让众网友炸了锅，众网友对此说法纷纷表示〝不可置信〞、〝相当胡扯〞、〝想像力爆棚〞。
 <br/>
 <br/>
 据网络上流传的视频片段是中共央视2016年9月3日播出的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开学第一课.html" target="_blank">
  开学第一课
 </a>
 》。这个节目中，一名姓胡的〝老红军〞作为嘉宾讲述了中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红军长征.html" target="_blank">
  红军长征
 </a>
 途中通信兵的故事。
 <br/>
 <br/>
 视频中，据称现年已99岁的老红军胡正先，身穿文革时期〝红卫兵〞常穿的那种绿军服，在镜头前讲到自己当年为了保证给电台供电，要想方设法维持发电机运转。他说：〝充电机要用汽油，没有汽油了可以用柴油，没有柴油，那个时候我们在长征路上，用的是猪油〞。
 <br/>
 <br/>
 胡正先的这段说辞立即引发众多中国网友的激烈大讨论。
 <br/>
 <br/>
 有网友留言称：〝其他不说，我只知道，长征路上没东西吃，吃皮带，吃树皮，草根。竟然说猪油不用来拌饭拿去当燃油？？？太假了。我反正是不信的。〞还有网友笑称：〝等著文宣部门紧张的编理由出来辟谣。〞
 <br/>
 <br/>
 @重生的沧老诗发帖称：〝这图是在CCTV中截图的。里面这位九十多岁的老红军，说到长征时携带的发电机既可烧汽油、又可烧柴油，更可以烧猪油时，我当即被震惊了，幸好下巴够结实，没掉到地下，但也费好大回功夫才推回原位！你大爷呀！真有种哈！〞
 <br/>
 <br/>
 @fxqf留言道：〝一本正经的胡说八道。〞
 <br/>
 <br/>
 @newsmg1972则调侃道：〝这说明我国上世纪30年代就已经取得汽油柴油猪肉三型混合动力发动机的研发技术。涛锅以为，绝对可以申请国际专利，原来我们的技术在几十年前都达到了如此高度，内燃机可以同时随时使用任何一种油，其中还包括了食用型油…〞
 <br/>
 <br/>
 @yankexin07嘲笑道：〝难怪墙那儿地沟油提取技术世界一流，原来三十年代老红军就掌握了核心技术。〞
 <br/>
 <br/>
 @元素_原子_源头则讽刺道：〝 特殊材料做的，什么不可以？ 图片上人的年龄似乎不支持。〞
 <br/>
 <br/>
 @文藻泽檩评论称 ：〝 社会主义的免费猪油吃多了，蒙了心了！什么假话都敢说！〞
 <br/>
 <br/>
 也有网友一本正经地从科学的角度进行分析称：〝首先汽油和柴油的性质完全不一样，一个是高挥发的易燃气液体，一个是低挥发的重油……所以汽油机和柴油机的原理完全不一样，汽油机直接喷汽油气体就可以了，而柴油机是压燃的，需要高温高压才能点燃柴油，所以理论上，柴油机可以适用于其他低挥发的重油，甚至植物油，柴油机在发明之初，就是想要使用植物油代替石油的。所以汽油机用柴油，完全是胡扯；柴油机用汽油，那是作死；但是柴油机用猪油，还是完全有可能的。〞
