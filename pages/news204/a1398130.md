### 华为等数十企业被美锁定 制造2025或遭团灭重创
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 公司因盗窃美国芯片技术，遭美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁售令.html" target="_blank">
  禁售令
 </a>
 处罚，生产已经停摆。此前，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 、联想等多家企业都被美国指控存在间谍行为。因为中共核心技术大多涉嫌盗窃，许多位列〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞计划的中国芯片公司都陷入恐慌。
 <br/>
 <br/>
 日经新闻报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_福建晋华.html" target="_blank">
  福建晋华
 </a>
 位列〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞计划。随着晋华遭美国重罚，数十家中国芯片厂商都面临美国商务部制裁的危险，包括受中共官方资助的国企长江存储、合肥长鑫、中芯国际等。
 <br/>
 <br/>
 有军方背景的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 、PC大厂联想、面板业者京东方，也同样处在美国出口管制的威胁之下。
 <br/>
 <br/>
 日前，美国司法部门高调起诉福建晋华和台湾联华电子，指其合谋盗窃美国美光公司的DRAM（动态随机存取记忆体）技术，并对晋华下达出口
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_禁售令.html" target="_blank">
  禁售令
 </a>
 。与晋华合作的美国公司随即全部撤出，晋华原本预计年底试产的DRAM生产线立即全部瘫痪。
 <br/>
 <br/>
 美方起诉书指，中共本不掌握DRAM技术。该技术是晋华和台湾联华电子从DRAM巨头美光公司盗窃而来，随后被中共国务院列入〝国家经济重点〞。
 <br/>
 <br/>
 官方对福建晋华的DRAM生产线投资56.5亿美元，也是〝中国制造2025〞计划中的核心项目之一。〝中国制造2025〞旨在成为半导体、生技、航太等敏感科技的〝世界领导者〞，尤其积极扶植芯片业。
 <br/>
 <br/>
 除了福建晋华之外，长鑫正在合肥兴建价值80亿美元的DRAM生产线、长江存储拟在武汉投资240亿美元生产NAND flash（储存型快闪记忆体）等，也都位列〝中国制造2025〞计划。
 <br/>
 <br/>
 报导引述业内人士分析称，全世界芯片制造商都无法摆脱对美国公司技术和零组件的依赖，中国的任何芯片计划如果遭遇美国禁售令，都意味着末日来临。
 <br/>
 <br/>
 此外，美国国会下属美中经济暨安全检讨委员会今年4月的报告指控，中兴通讯、华为、联想都存在网络间谍行为，而歌尔声学、力神动力电池系统也和中共军方、核能和间谍活动有关联。这十几家中企，包括伺服器大厂浪潮（Inspur），都可能被列为美国出口管制对象。
 <br/>
 <br/>
 而华为最新高阶旗舰手机Mate20也无法避免使用美国美光、Skyworks、德州仪器、英特尔等元件和谷歌Android软件；京东方依赖美国康宁公司的玻璃基质及美国应材的设备；联想和浪潮则依赖英特尔和超微的核心处理器与微软的Windows操作系统。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
