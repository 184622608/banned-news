### 范冰冰身边最重要的人提离婚 被疑转移资产
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 〝失踪〞4个月后，缴纳近9亿罚金〝虎口脱险〞，但她的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经纪人.html" target="_blank">
  经纪人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆晓光.html" target="_blank">
  穆晓光
 </a>
 已被捕。穆是把范打造成一线大牌明星的主要推手，因此被称为范身边最重要的人。值得关注的是，穆晓光提出的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离婚.html" target="_blank">
  离婚
 </a>
 案日前获法院判准。港媒质疑此举或为转移个人财产。
 <br/>
 <br/>
 10月17日，台媒《中央社》报导称，中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 的台籍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经纪人.html" target="_blank">
  经纪人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆晓光.html" target="_blank">
  穆晓光
 </a>
 （本名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_牟恩广.html" target="_blank">
  牟恩广
 </a>
 ）要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离婚.html" target="_blank">
  离婚
 </a>
 案，台北地方法院审理后近日判准离婚。他离婚的理由是，婚后从未与妻子同居、夫妻之情荡然无存，希望好离好散。
 <br/>
 <br/>
 判决显示，牟恩广与妻子在民国102年（2013年）结婚，但婚后并未共同生活，牟恩广回台湾期间，都是住在台北的晶华酒店，从未与妻同居，夫妻之情荡然无存，已经没有维持婚姻关系的主观意愿和客观事宜。
 <br/>
 <br/>
 而牟恩广的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄南茜.html" target="_blank">
  黄南茜
 </a>
 说，因丈夫事业心重，婚后虽然丈夫一直在中国，聚少离多但绝非丈夫所说双方从没有共同生活。
 <br/>
 <br/>
 台湾《苹果日报》称，穆妻不愿意离婚，要求法官驳回穆晓光的离婚请求。
 <br/>
 <br/>
 10月18日，香港《明报》称，身家过亿的穆晓光卷入范冰冰逃税案，目前已被警方采取强制措施，有网民揣测，不排除穆晓光以离婚方式转移个人财产。
 <br/>
 <br/>
 穆晓光是将范冰冰打造成一线大牌明星的主要推手。
 <br/>
 <br/>
 范冰冰因卷入娱乐圈逃税案，在公众视野里消失4个月后，党媒10月3日报导，税务机构查实，范冰冰及其旗下企业，累计偷税漏税2.55亿处罚5.96亿，滞纳金0.33亿，共需补缴8.83946亿，只要缴清费用，不予追究刑事责任。
 <br/>
 <br/>
 但范冰冰的经纪人牟某广，则因涉嫌犯罪，已经被刑拘。
 <br/>
 <br/>
 党媒称，6月份，税务机构调查范冰冰旗下公司时，牟某广被控指示员工隐匿、故意销毁涉案企业会计凭证、会计账簿，阻挠调查涉嫌犯罪。
 <br/>
 <br/>
 目前，牟恩广等人已被警方采取强制措施，案件仍在调查中。
 <br/>
 <br/>
 陆媒披露，穆晓光从小自台湾长大，2007年前往大陆发展，他不仅是范冰冰的经纪人，还是上海著名的151会所老板，而且和台湾的演艺圈幕后互动也相当活络，更帮助范冰冰脱离华谊公司，创立范冰冰工作室，背景不容小觑。
 <br/>
 <br/>
 报导说，穆晓光是一个非常厉害的人物，范冰冰能有今天的成就，与穆晓光是分不开的。穆晓光是一个很好的娱乐圈幕后推手，在他的精心打造下，范冰冰一步一步走到了一线大牌的位置。
 <br/>
 <br/>
 在此期间，范冰冰遭受诸多负面新闻影响，穆晓光均以经纪人身份护航，力保范冰冰的声誉。2007年的6月，范冰冰创立了〝范冰冰工作室〞，此工作室全力制作电视剧。有传闻称，穆晓光其实才是工作室的老板。
 <br/>
 <br/>
 报导说，穆晓光还曾投资制作过多部电影、电视剧，传闻他的身价高达5亿。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：范铭）
