### 华为任正非之女孟晚舟加拿大被捕  美要求引渡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】近日，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的女儿、副董事长兼CFO孟晚舟（Wanzhou Meng），在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 温哥华国际机场被捕，她因涉嫌违反美国对伊朗的贸易制裁，将被引渡到美国。
 <br/>
 <br/>
 12月6日，香港01报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 副董事长兼首席财务官（CFO）孟晚舟，已于当地时间12月1日在温哥华国际机场被捕。
 <br/>
 <br/>
 卑诗省最高法院（Supreme Court of British Columbia），在当地时间12月3日，因应孟晚舟的案件，发出〝禁止报道令〞。
 <br/>
 <br/>
 禁令称，孟晚舟以Cathy或Sabrina为别名。孟晚舟是华为创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 ，与第一任妻子孟军所生的长女，随母姓。1972年出生，现年46岁。
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 环球邮报》（the Globe and Mail）引述加国司法部的声明报导说，加拿大是应美国当局要求，在温哥华逮捕了华为首席财务官孟晚舟，她因涉嫌违反美国对伊朗的贸易制裁，而将被引渡到美国，12月7日举行保释听证会。
 <br/>
 <br/>
 司法部声明还说，〝受限于禁止报导令，我们目前无法进一步提供任何细节。这道禁止令是由孟女士申请所提出的。〞
 <br/>
 <br/>
 路透社4月份曾引述了解情况的消息人士称，纽约联邦检察官至少从2017年开始，就一直在调查华为，检察官就原产于美国的产品运往伊朗和其它国家、违反美国出口和制裁法令的指控进行调查。
 <br/>
 <br/>
 美国调查华为主要有3个原因：1，华为有中共军方背景，可能被中共利用在海外从事技术间谍活动；2，华为违规向伊朗和朝鲜等被制裁国家出口电讯设备和提供技术支持；3，华为号称将〝引领5G时代〞，威胁全世界资讯安全。
 <br/>
 <br/>
 公开资料显示，孟晚舟毕业于华中理工大学，拥有硕士学位，于1993年加入华为，2011年4月起担任华为常务董事兼CFO，现任公司CFO、副董事长，主导在全球建立了五个共享中心，担任华为系旗下共11间公司的董事。
 <br/>
 <br/>
 根据香港公司注册处资料，孟晚舟持有香港身份证和中国护照，报住大角嘴豪宅维港湾一个由华为持有的高层单位。她目前担任3间本港注册的公司董事，包括〝华为技术投资有限公司〞、〝香港华为国际有限公司〞及〝华盈管理有限公司〞。
 <br/>
 <br/>
 华为1987年在深圳成立，公司多年来一直没有上市，2012年开始成为全球最大的电讯设备制造商。
 <br/>
 <br/>
 多年以来西方情报机构一直质疑华为设备的安全性，认为它受制于中共政府，存在间谍和安全隐患，美国此前已经采取了一系列行动，阻止或减少华为和中兴通讯公司进入美国市场，因为这两家公司被指控，利用其技术在美国进行间谍活动。
 <br/>
 <br/>
 11月22日，华尔街日报报导说，美国政府正努力劝说德国、意大利和日本等盟国官员及电信高管，弃用华为的电讯设备。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
