### 范冰冰被抓细节曝光 算命时被戴上头套押走
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月04日讯】中共官方3日通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 缴纳8.8亿人民币罚款，可免于刑责。同日，范冰冰也在微博发致歉信说，将全力筹措资金缴清罚款。随着案件的尘埃落定，范冰冰被抓的诸多细节也随之曝光。
 <br/>
 <br/>
 中共官媒3日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_税务机构.html" target="_blank">
  税务机构
 </a>
 查实，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 及其旗下企业，累计
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_偷税漏税.html" target="_blank">
  偷税漏税
 </a>
 2.55亿（人民币，下同），处罚5.96亿，滞纳金0.33亿，共需补缴8.83946亿。9月30日，税务部门已经向范冰冰发出了巨额罚单。
 <br/>
 <br/>
 报导称，范冰冰受到的是税务机关的行政处罚，并没有构成犯罪，只要在规定期限内缴清罚款，将不予追究刑事责任。
 <br/>
 <br/>
 但范冰冰的经纪人牟某广，则因销毁会计凭证、会计账簿，涉嫌犯罪被公安机关采取刑事强制措施。
 <br/>
 <br/>
 同日，范冰冰在个人微博发布道歉信说，深感羞愧，经历从未有过的痛苦，自己将全力筹措资金缴清罚款。
 <br/>
 <br/>
 这是范冰冰自卷入中国演艺圈逃税避税风波以来的首次露面。范冰冰失踪4个月里全世界都在找她。网上的传闻也很多。如今随着范冰冰案件的尘埃落定，她被抓的诸多细节也随之曝光。
 <br/>
 <br/>
 10月4日，知名体育博主〝辣笔小球〞在个人微博上爆料称，范冰冰是在江苏南京德基20楼普慈堂被抓的。
 <br/>
 <br/>
 爆料说，范冰冰为知晓将来祸福，在德基老板吴铁军的安排下，花费200万元找绰号叫〝小神仙〞的沈爱栩算命。就在算命当天，中共警方将范冰冰抓获。
 <br/>
 <br/>
 抓捕范冰冰的行动是北京警察主导，南京警察协助，警察从20楼给范冰冰戴上头套后，直接坐电梯去停车场，德基监控已被中共警方删除，为的是防止有人将视频上网，因为行动在当时是保密的。
 <br/>
 <br/>
 爆料还说，一同被捕的还有算命人沈爱栩和德基老板吴铁军。在抓获范冰冰后，南京警察直接开车将其押到北京，途中范冰冰要上厕所，警察提前在休息区对卫生间进行清场后，才让她去。直到补税金额达成一致前，范冰冰一直被羁押在北京。
 <br/>
 <center>
  <br/>

   </font>

  <br/>
 </center>
 <br/>
 在范冰冰露面之前，也有网友在9月3日微博中说，范冰冰曾找过沈爱栩算命，之后在南京被抓。目前沈爱栩也失踪，另有一位神秘富豪也牵扯其中。该网友还称，曾找到那位神秘富豪求证，他承认范冰冰找他帮了点忙。
 <br/>
 <br/>
 当时也有网贴补充说，范冰冰找沈爱栩算命花费了200万人民币。
 <br/>
 <br/>
 沈爱栩1997年出生在江苏盐城，3岁时被称为〝盐城小神童〞，有风水专家和易学专家等身份。媒体曾报导，沈爱栩参与诈骗，涉案金额无数。
 <br/>
 <br/>
 网上还有帖子说，沈爱栩是其小学同学，真名叫沈爱旭，特别笨，汉语拼音都认不全。
 <br/>
 <br/>
 5月底，中国演艺圈逃税避税黑幕被踢爆。中共税务部门6月初已介入调查范冰冰案件。如今范冰冰案终于有了最后定论。
 <br/>
 <br/>
 但中共官方在10月3日的通报中称，除范冰冰逃税案之外，税务部门已开展〝影视行业自查自纠〞，年底前，主动补缴税款的免于行政处罚，拒不纠正的将严肃处理。
 <br/>
 <br/>
 有台湾时事财经专家之前爆料，范冰冰还不是重点，当局最终是要打击中共掌控娱乐圈的最大势力，中共江派二号人物曾庆红家族。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：范铭）
