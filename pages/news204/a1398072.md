### 台媒：川普要求一点没变 中共难签〝城下之盟〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）和习近平11月初通话后，又计划与习在G20会面共进晚餐，引发舆论对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 露曙光的乐观猜想。但台媒分析说，川普对中共的要求一点没变，而针对美方提出的〝城下之盟〞，中方也很难接受。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 与习近平11月初进行了贸易战以来首次通话。
 <br/>
 <br/>
 习近平表示，很高兴再次与川普通电话，重视与川普的良好关系，愿在出席阿根廷20国集团领导人峰会期间再次会晤，就中美关系及其他重大问题深入交换意见。
 <br/>
 <br/>
 川普也在推特上说：与习近平进行了长时间和非常良好的对话。两人讨论了很多话题，当中重点是贸易问题，这些讨论进行顺利，而且已安排在G20峰会期间会面，就朝鲜问题也有良好讨论！
 <br/>
 <br/>
 在川习两人电话通话之后，川普表示将会与中共达成一个很好的协议，他与习近平还将在G20峰会会面时共进晚餐。
 <br/>
 <br/>
 《香港经济日报》在社评中说，中美元首通电话后，令贸易战露曙光，但双方分歧严重，要达成协议言之尚早。
 <br/>
 <br/>
 社评认为，重大障碍可能在于川普要求北京停止〝中国制造2025〞产业和国企补贴，以及停止强迫美国企业技术转移，这已牵涉到中共结构性改变，中方未必能够让步。
 <br/>
 <br/>
 香港《星岛日报》认为，这次〝川习会〞能否达成协议，要看美国中期选举结果，中共期望川普的共和党落败，这样他的政策可能发生变化。
 <br/>
 <br/>
 但观察家普遍认为，北京这一期待注定会落空，因为对抗中共是美国民主党和共和党共识。即使民主党掌握了众议院，也很难朝着对中共有利的方向发展。
 <br/>
 <br/>
 有了解中共政局内情的中国学者分析，如果美方提出〝城下之盟〞，中共高层很难接受。
 <br/>
 <br/>
 中国学者、美国研究中心主任时殷弘对台媒《中央社》分析认为，〝川习会谈〞可能有三种结果：
 <br/>
 <br/>
 第一种结果：对于美国启动的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关税.html" target="_blank">
  关税
 </a>
 ，中方被动反击，而且由于美中贸易美方有巨额赤字的缘故，中方已远远不能进行等额报复，互征高关税的说法有点勉强。
 <br/>
 <br/>
 第二种结果：贸易战继续升温，但激烈程度有所减缓，美方不再将目前加征的10%关税提升至25%；川普也不会再下令对另外2670亿美元
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国商品.html" target="_blank">
  中国商品
 </a>
 加征关税。
 <br/>
 <br/>
 第三种是最坏的结果，就是美方若提出〝城下之盟〞，中共无法接受。在这种情况下，不但达不成任何协议，贸易战还将加剧。
 <br/>
 <br/>
 何为〝城下之盟〞？即美方不仅要求中共大量增加进口美国商品，大幅拓宽市场准入，还要求改变中共金融体制和经济政策，特别是改变由中共控制、发动的技术升级的努力。
 <br/>
 <br/>
 在川普与习近平进行通话的同一天，美国司法部长塞申斯（Jeff Sessions）在记者会上指责中共持续进行商业间谍活动，他表示已启动一项新的执法行动计划，专门打击中共经济间谍活动。
 <br/>
 <br/>
 英媒认为，华盛顿透过白宫和司法部同一天内对北京软硬兼施，一方面保持高层对话，另一方面毫不客气地批评提诉，目的更有利于美方接下来的谈判以及迫使中共改变。
 <br/>
 <br/>
 截止目前，美中贸易战交火已近4个月，经过四轮谈判无果。在贸易战不断升温之下，中国各项经济数据不断下滑，中共已经感受到日益沉重、甚至足以窒息的经济压力。
 <br/>
 <br/>
 10月底，中共政治局会议聚焦讨论经济，罕见的强调〝经济下行压力增大〞，〝要高度重视，增强预见性，及时采取对策。〞
 <br/>
 <br/>
 11月1日，习近平会见企业家时也罕见承认，目前中国经济发展的不确定性明显上升，下行压力加大，企业经营困难增多。
 <br/>
 <br/>
 加之贸易战令中共内斗加剧，中共四中全会一直悬空未定等等因素交织，分析认为北京不得不找上川普、重修旧好，甚至共商如何应对中共内部与国际政局。因此有理由推断〝川习会〞将就贸易战达成共识或协议。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿)
