### 敏感时刻 习近平〝刺刀监会〞 胡锦涛下水道藏兵
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】中共政局自重庆事件后就发生了巨变。中共十八大前后围绕薄熙来案件，中共高层
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡习阵营.html" target="_blank">
  胡习阵营
 </a>
 与江系展开激烈搏击。港媒称，当时中共局势异常紧张，为了防止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江派阵营.html" target="_blank">
  江派阵营
 </a>
 异动，胡习阵营曾在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地下水道.html" target="_blank">
  地下水道
 </a>
 内藏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军队.html" target="_blank">
  军队
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 则以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刺刀监会.html" target="_blank">
  刺刀监会
 </a>
 〞。
 <br/>
 <br/>
 据香港媒体报导，2012年11月中共十八大期间，中共局势异常紧张，一触即发，当时，胡锦涛、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 为确保十八大顺利换届，曾秘密布局，在北京市大会堂附近的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地下水道.html" target="_blank">
  地下水道
 </a>
 ，秘密布满
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军队.html" target="_blank">
  军队
 </a>
 ，只要江派有武装异动，胡习则立即会采取军事行动。
 <br/>
 <br/>
 重庆市委书记薄熙来在2012年3月15日被撤职控制后，3月19日深夜，北京城突然传出枪声，有北京网民表示，当日夜间有军车进城。随后，传出周在京城发动了政变，但在当夜就被胡、温控制住，而后周在北京的权力全部被移交。
 <br/>
 <br/>
 消息称，薄熙来在落马前的几年中就在准备武装力量，他与时任中共政法委书记周永康，先后在北京、成都和重庆5次见面，密谋政变之事。
 <br/>
 <br/>
 习近平十八大上台之后将自己的贴身警卫进行了〝大换班〞，还一度移居西山军事指挥中心，以防不测。
 <br/>
 <br/>
 2013年8月，中共高层在北戴河会议上，一致决定对周永康本人展开调查。接近中南海的消息，周认为末日来临，于是孤注一掷、策划
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暗杀.html" target="_blank">
  暗杀
 </a>
 。第一次是在会议室中放置定时炸弹，另一次是趁习近平在北京301医院做体检时打毒针。
 <br/>
 <br/>
 同年11月，中共三中全会召开期间，为防不测，习近平以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刺刀监会.html" target="_blank">
  刺刀监会
 </a>
 〞，选择隶属于中共军队总参谋部的北京京西宾馆召开，同时把334名省部级高官，变相〝软禁〞在北京郊外党校，轮训学习习近平的系列讲话精神，不准串门、人人表态过关。
 <br/>
 <br/>
 2013年12月，周永康被抓的消息传出之时，又有201名省部级高官，参加党校轮训第二期研讨班，由中共中纪委、中组部、中宣部等官员做了专题报告，让所有人在周永康事件上表态。
 <br/>
 <br/>
 据美媒披露，2012年2月，重庆前公安局长王立军逃亡成都美领馆寻求庇护，曝光了薄熙来、中共政法委书记周永康等人联手政变、活摘等关键罪行。
 <br/>
 <br/>
 同年3月两会期间，薄熙来被抓捕。2013年9月被判处无期徒刑。目前在秦城监狱里服刑。
 <br/>
 <br/>
 薄熙来落马后，薄的政治盟友周永康也被查处。
 <br/>
 <br/>
 2013年12月1日，周永康被中纪委核查；2014年7月29日，中纪委宣布，对周立案审查，12月5日，周被移送司法处理；2015年6月，周被判处无期徒刑，目前在秦城监狱里服刑。
 <br/>
 <br/>
 周、薄等人虽然被判处无期徒刑，但官方对他们所涉及的政变、暗杀，以及外媒热炒的有关周薄活摘人体器官、贩卖尸体等反人类罪，至今一项都没有提及。
 <br/>
 <br/>
