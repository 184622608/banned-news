### 【禁闻】11月22日维权动态
------------------------

<div class="wysiwyg">
 【新唐人2018年11月23日讯】下面我们来关注一下中国大陆各地民众维权的情况。
 <br/>
 <br/>
 1. 被强拆 河南87岁老人法院上访遭冷遇
 <br/>
 <br/>
 11月22号，河南许昌市魏都区半截河宋庄87岁的退役军人宋西章，坐着轮椅到许昌中级法院上访，要求受理强制执行申请，但没有结果。据了解，宋西章的房屋早前被违法强拆，许昌中级法院判决宋西章胜诉，要求强拆方天宝路街道办事处补偿，但一直不能执行。
 <br/>
 <br/>
 2. 荆门
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主人士.html" target="_blank">
  民主人士
 </a>
 被传唤 遭殴打送医急救
 <br/>
 <br/>
 湖北荆门民主人士刘艳丽11月21号被警方传唤，家人第二天得知她被打伤送往医院急救后，赶往荆门第二医院，但在门口被多名公安围住，不准进医院探视。
 <br/>
 <br/>
 3. 安徽
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土地承包权案.html" target="_blank">
  土地承包权案
 </a>
 开庭 近百人旁听声援
 <br/>
 <br/>
 11月20号，安徽合肥中级法院开庭审理访民王可翠起诉政府土地承包证违法一案，近百民众到场旁听，法庭只审理十几分钟就宣布休庭。旁听民众于是与王可翠和律师在庭外合影，表示声援。
 <br/>
 <br/>
 4. 被关押律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_余文生.html" target="_blank">
  余文生
 </a>
 获人权法治奖
 <br/>
 <br/>
 大陆被关押的人权律师余文生，11月21号获得〝德法人权法治奖〞，德法两国外长当天宣布了他获奖的消息。据了解，这个奖项旨在表彰那些在本国做出突出贡献、或是在国际层面推进和保护人权、推动法治规则的人权人士。
