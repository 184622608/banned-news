### 内部人士曝光新千人计划  北京陷入被动
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】日前外媒曝光中共深圳政府内部文件，其中提及应对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 的策略，建议吸取〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞教训，抓紧以各种低调隐蔽的方式引进尖端人才。有分析指，文件为美国指证中共盗窃技术提供了新证据，或是内部人士流出。
 <br/>
 <br/>
 日前，中共深圳地方政府一份题为&lt;〝不宣而战〞，世界格局的改变将比我们预期来得更快更猛烈&gt;的内部报告，被海外中文媒体曝光。
 <br/>
 <br/>
 据报导，这是深圳政府近期派员到美国考察后撰写的报告，由深圳市政府政策研究室主任吴思康执笔，中共深圳市委书记王伟中和副市长艾学锋都做了相关批示。
 <br/>
 <br/>
 网络上公布的文件截图显示，报告明确认为贸易战已经拉开了中美长期冷战的序幕，美国对中共的政策已从〝接触战略〞转向〝全方位围堵〞。报告验证了此前的外界评论，即遏制中共已是美国朝野的共识。
 <br/>
 <br/>
 报告还提及了多数外媒拒绝提及的内容，即美国共和党赢得中期选举和川普（特朗普）获得连任都已没有悬念，美国对中共的强硬政策将持续六年。即使将来川普离任，已经形成的全面遏制中共的战略也不会有大的调整。
 <br/>
 <center>
  <br/>

 </center>
 <br/>
 报告也提醒，美国可能在半年至一年内，在高科技层面对中共实施全面封锁。报告建议中共政府要以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 思路〞在各方面制定〝战事策略〞，包括必须在美国全面封锁前的空窗期做好准备，要〝不惜一起代价〞、以各种〝超常规手段〞加速引进科研仪器设备。
 <br/>
 <center>
  <br/>

    网传中共深圳政府内部报告。（网络截图）

  <br/>
 </center>
 <br/>
 报告还特别提出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人才战.html" target="_blank">
  人才战
 </a>
 〞的概念，提议要在窗口期内，采取更加〝精准、隐蔽的特殊策略〞，以优厚条件加速引进〝战略性科技人才〞，特别是非华裔人才。并实施类似承认〝多重国籍〞的人才政策，鼓励华人学者〝加入其他国籍〞，参与国际科技活动。
 <br/>
 <br/>
 报告特别提到，要吸取〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞被美国FBI盯上的教训，不宜大张旗鼓宣传。
 <br/>
 <br/>
 据外媒报导，这份报告于9月29日以正式文件的方式呈送深圳市委书记王伟中，王伟中随后批示让深圳市有关官员传阅，最后转交深圳市发展与改革委员会（发改委）研究讨论报告所提建议。
 <br/>
 <br/>
 有熟悉中共内部事务的人士分析认为，这个流程意味着报告获得了认可，已交由发改委具体落实。
 <br/>
 <br/>
 另有报导引述北京体制内学者披露，北京学术界圈子也已传阅了这份报告。这似乎说明中共内部对该报告的讨论至少已经提升到中央智库的层面。
 <br/>
 <br/>
 有分析指，报告涉及中美科技竞争敏感内容，以及盗取外国尖端科技的隐蔽操作细节，可能是由中共内部人士流出，目的是通过曝光这些机密文件让北京陷入被动。
 <br/>
 <br/>
 早在数月前，美国FBI就已开始调查卷入中共〝千人计划〞的华裔学者。日前，美方又高调引渡并起诉盗取美国航空技术的中共情报官员，并接连出台政策限制中共获取高科技的渠道。美国全面封堵中共盗窃科技产权的行动正在展开。
 <br/>
 <br/>
 上述中共内部文件，可能给美国推行封堵行动提供新证据。
 <br/>
 <br/>
 目前中国政界形势复杂。在中共党内，有反对势力在不惜代价地给北京当局拆台，也有开明人士在助推中共倒台。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：云涛）
