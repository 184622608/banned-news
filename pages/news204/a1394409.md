### 被击中痛点？中共罕见一日二回应彭斯讲演
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】日前，美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 发表40分钟讲演，对中共强烈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抨击.html" target="_blank">
  抨击
 </a>
 。中共官媒形容彭斯的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演讲.html" target="_blank">
  演讲
 </a>
 为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_檄文.html" target="_blank">
  檄文
 </a>
 〞，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共外交部.html" target="_blank">
  中共外交部
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发言人.html" target="_blank">
  发言人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华春莹.html" target="_blank">
  华春莹
 </a>
 更是罕见的一日之内做出两次回应，显示彭斯的演说，已对中共产生重大杀伤力。
 <br/>
 <br/>
 美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 10月4日在美国华府智库哈德森研究所（Hudson Institute）发表40分钟的长篇演说，内容涉及中共干涉内政、窃取机密、威胁台湾、南海扩张、贸易不公、侵犯宗教自由、对第三世界进行债务外交等领域。
 <br/>
 <br/>
 彭斯对中共火力全开，细数中共种种罪状的演说，被认为是美中建交近40年来，美国官方对中共最强烈的批评。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共外交部.html" target="_blank">
  中共外交部
 </a>
 10月5日罕见地对此一天内二次回应。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华春莹.html" target="_blank">
  华春莹
 </a>
 在当日上午以答记者问的名义，否认美方的指控。
 <br/>
 <br/>
 华春莹还声称，〝我们的朋友遍天下〞，美方说中方干涉美国内政和选举是不实。
 <br/>
 <br/>
 5日下午，华春莹再度以答记者问的形式声称，有关台湾、南海、人权和宗教等问题，美方没资格说三道四。
 <br/>
 <br/>
 她还重复中共外交部长王毅的话称，〝中国的人权状况怎么样，中国人民最有发言权〞，美方不要利用人权和宗教问题干涉中共内政。
 <br/>
 <br/>
 华春莹的论调被外界认为老调重弹。面对国际指责，中共一向是睁眼说瞎话进行强辩。
 <br/>
 <br/>
 尽管如此，中共外交部在同一天对同一起事件二度回应，被认为彭斯的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_演讲.html" target="_blank">
  演讲
 </a>
 已对中共产生重大杀伤力。
 <br/>
 <br/>
 《环球时报》发表社论称，彭斯的演讲像篇〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_檄文.html" target="_blank">
  檄文
 </a>
 〞。而〝檄文〞是古代用于晓谕、征召、声讨、揭发罪行等的文告。
 <br/>
 <br/>
 有网友表示，彭斯的演说真的触及到中共的痛点，希望美国加大打击力度。
 <br/>
 <br/>
 也有网友批评华春莹有关〝我们的朋友遍天下〞言论，应是〝我们丐帮遍布天下〞，意指中共所谓的友好国家，不是经济崩溃就是在崩溃的边缘，不是动乱就是正在走向动乱的路上，全靠中共多年来大撒币维持着政权。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：祝馨睿）
</div>
