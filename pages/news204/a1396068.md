### 广东多地工厂倒闭商铺关门 企业贷款无人申请
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】中国经济下行，而中共执意开打贸易战，更令国内经济陷入危局。美媒消息，在中国经济龙头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广东.html" target="_blank">
  广东
 </a>
 省，很多工厂倒闭，商铺关门，物价飞涨，官方下指标向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中小企业贷款.html" target="_blank">
  中小企业贷款
 </a>
 ，但基本没有需求。
 <br/>
 <br/>
 自由亚洲电台10月19日引述一名不愿透露姓名的国有银行职员透露，过去一周内，上级不断下达硬指令，要求尽快向中小企业发放贷款，但问题是现在严格限制把资金投入房地产，又没有多少人投资实业，所以贷款基本上没有需求。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广东.html" target="_blank">
  广东
 </a>
 当地居民对自由亚洲说，东莞、江门等地，部分店铺已人去楼空，外省人就业困难。
 <br/>
 <br/>
 广东中山的李女士表示，当地很多大中型企业关闭，有的破产倒闭，有的搬出中国，外省打工的人纷纷返乡，周边的商场都空了，超市也空了。她说，很多工厂经营困难，成本太高，产品又卖不出去。
 <br/>
 <br/>
 江门一名厨师陈锦强说，当地居民节衣缩食，酒楼食客减少，再加煤气汽油涨价，他的老板正在苦苦经营。
 <br/>
 <br/>
 在东莞从事贸易的李先生说：〝许多商品都在涨价，外企在里面的生存空间显得很艰难，确实有往外搬迁的情况。这不仅仅是东莞，深圳、广州都有这样的情况。〞
 <br/>
 <br/>
 顺德居民周志明表示，〝将来大量工厂倒闭、工人失业、经济崩溃、物价飙升，老百姓将重回几十年前的苦日子，社会也会更加不稳定。〞他说，中共与美国为首的西方阵营抗衡，〝是一条祸国殃民的死路〞。
 <br/>
 <br/>
 此前，有港商向自由亚洲表示，许多港商在广东投资生产的产品主要销往美国，如果中美贸易战继续，广东至少有一半工厂倒闭，还有很多港商将被迫将工厂搬走。
 <br/>
 <br/>
 日前，中共商务部官员也罕见地公开承认，贸易战令〝个别企业面临着停产转产、工人失业等风险〞。
 <br/>
 <br/>
 中共国家统计局19日公布，今年第三季度国内生产总值(GDP)较上年同期增长6.5%，达到1992年以来第二个最低位。而9月份消费品零售总额在扣除通胀因素后，实际增幅只有6.4%，比8月份的6.6%更低。
 <br/>
 <br/>
 另外，中共央行17日公布9月份的社会融资规模，实际只有14700亿。而融资规模总存量是197万亿，比去年同期增长10%，增幅是历史新低。
 <br/>
 <br/>
 银行业人士称，这显示实体经济对银行贷款需求不大，投入生产的人越来越少。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
