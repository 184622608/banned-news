 </a>
 遭警察〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱衣羞辱.html" target="_blank">
  脱衣羞辱
 </a>
 〞事件发酵后，引发海内外广泛关注。就在孙世华展开法律行动之际，案件的3名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_证人.html" target="_blank">
  证人
 </a>
 15日遭警方
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_拘留.html" target="_blank">
  拘留
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙世华.html" target="_blank">
  孙世华
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_律师.html" target="_blank">
  律师
 </a>
 被广州市荔湾区华林街派出所警察殴打、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱衣羞辱.html" target="_blank">
  脱衣羞辱
 </a>
 以及非法拘禁事件中，3位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_证人.html" target="_blank">
  证人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张五洲.html" target="_blank">
  张五洲
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李小贞.html" target="_blank">
  李小贞
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梁颂基.html" target="_blank">
  梁颂基
 </a>
 15日遭到刑拘。
 <br/>
 <br/>
 《自由亚洲》电台报导称，3位证人因被中共当局怀疑会对孙世华冤案提供有力的反证，14日起相继遭到警方的传唤，如今已经证实遭到刑拘，但目前罪名仍然有待核实。
 <br/>
 <br/>
 孙世华的丈夫、维权律师隋牧青15日指出，当局要把3位目击者定罪，试图掩盖警察对孙世华律师的羞辱行为。他批评当局的行为犹如继续制造冤案。
 <br/>
 <br/>
 隋牧青说∶〝因为我们是要采取法律行动，要求要拿到事发时的视频，还有可能追究相关人士的刑事责任。我觉得广州市公安局荔湾分局是想破釜沉舟，就开始抓证人。〞
 <br/>
 <br/>
 被刑拘的三名证人之一，梁颂基是在北京游玩期间遭到6、7名广东警方跨省拘捕。另一名被刑拘的证人李小贞，从11日开始，3日之内被警方传唤了2次。
 <br/>
 <br/>
 被刑拘的另一名维权人士张五洲，是在梁颂基被警方拘捕的同一天傍晚，在居住在清远的妹妹张唯楚家被以涉嫌〝扰乱单位秩序罪〞传唤。
 <br/>
 <br/>
 张唯楚说，相信姐姐是因为目击孙世华律师被警方涉嫌殴打和羞辱时拍下了照片和视频有关。
 <br/>
 <br/>
 今年48岁广州女律师孙世华，长期代理商业案件。事发时，她第一次接手办理刑事案件。她在网帖中，将自己在广州市公安局荔湾分局华林街派出所的经历称为〝近20年律师生涯中最黑暗、恐惧、屈辱的一天〞。
 <br/>
 <br/>
 孙世华说，9月20日，她与其委托人、访民李小贞到华林街派出所，与警官沟通李小贞丈夫周建斌取保的事情。当天下午，孙世华与警方未能讨论其委托人的案件，当她询问经办的陈姓警官姓名警号时，对方将工作证甩向她，反咬她〝袭警〞。
 <br/>
 <br/>
 接着，孙世华被一群警察掐脖施暴，双脚也几乎离地，令她瞬间失去知觉，并被带到派出所办案区内，强制她脱光衣服作出羞辱性检查，脱衣、裸身过程达20分钟，随后她被拍照、打指模及强制验尿，并且接受了5个小时的讯问。
 <br/>
 <br/>
 10月11日，孙世华律师就事件向广州市公安局荔湾区分局递交4份信息公开申请，要求官方还自己一个公道。
 <br/>
 <br/>
 广州公安局13日发布消息予以否认。广州市律师协会竟声称，警方行为有失风范、态度不当、语言有失文明，但未发现孙世华被殴打羞辱情况。
 <br/>
 <br/>
 对此，孙世华两度回应说，声明与真相距离太远，市公安局有意包庇犯事警员，她敦促广州警方公开案发时的录音录像证据，并要求省纪检和人大介入彻查。同时，孙世华还呼吁广东司法行政机关和律协，顶住压力，依法履行法定职责，不为虎作伥。
 <br/>
 <br/>
 北京人权律师梁小军以余文生律师袭警视频事件与这次的案件作出对比。他说，〝余文生律师打警察的事情传到满世界都是，大家都在喊‘律师打警察了’。现在孙世华律师被警察侮辱、施以暴力，明明有视频却不发布。之所以不发布，我认为是因为可以客观证实孙世华律师的陈述。这是警察权力扩张，对律师权益的压制。〞
 <br/>
 <br/>
 梁小军补充说，中国律师的职业环境非常恶劣。律师权益受到侵犯以后，很难得到救济和保护。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：范铭）
