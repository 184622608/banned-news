### 哈尔滨再传暴力逼迁 打伤残疾人
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈尔滨.html" target="_blank">
  哈尔滨
 </a>
 松北区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_残疾人.html" target="_blank">
  残疾人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张铁华.html" target="_blank">
  张铁华
 </a>
 家正面临强拆，因为拆迁补偿不合理，张铁华拒绝搬迁，但却在日前遭到封门堵路，逼迁，诬告等一系列骚扰。警察反而将张家人强行带到派出所做笔录，当地残疾人闻讯纷纷赶去声援。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张铁华.html" target="_blank">
  张铁华
 </a>
 10月31号告诉《新唐人》，当地政府和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈尔滨.html" target="_blank">
  哈尔滨
 </a>
 水务投资公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强行征地.html" target="_blank">
  强行征地
 </a>
 ，因为拆迁补偿不合理，双方没有达成协议，对方就强行施工，晚上偷偷倒土封门堵路。
 <br/>
 <br/>
 哈尔滨松北区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_残疾人.html" target="_blank">
  残疾人
 </a>
 张铁华：〝政府和水务集团互相勾结之后，在我家门口修一栈道，他没有审批手续，在我这个门口强行施工，已经十天了，现在我家就面临着强拆，他白天不来，尽晚上来，弃土，晚上推。〞
 <br/>
 <br/>
 哈尔滨访民孙东生介绍，张铁华家有六口人，两个是学生，四个大人中，有三个是残疾人，家里被封门堵路后更加出行不便。29号，张铁华全家阻止一、二十名工人施工，遭到殴打。但打人者反过来说自己被殴打，警察反而要带走张家人。在当地残疾人声援下，最后只带走儿媳王海英和她孙子到派出所作笔录 。
 <br/>
 <br/>
 哈尔滨访民孙东生：〝前天阻止他们，铁华大姊和她儿媳妇被打了，派出所要把铁华大姊家五人全部但带到派出所去，很多残疾人就到了现场来声援了。我昨天在现场，我说你们这行为太卑鄙了，他就拿车撞我，给我撞了大跟头，摔得我浑身就跟散架子似的。〞
 <br/>
 <br/>
 据了解，此前当地强拆曾发生过两起命案：2015年7月万宝镇大亮子村两死三重伤；同年8月，松浦镇松花江村苏永祥家一人死亡。
 <br/>
 <br/>
 哈尔滨松北区残疾人张铁华：〝以投资集团的名义，强征老百姓的地，完了他们转手倒卖。当初15年施工，他也有没达成协议，就给人强拆了，有一家叫苏永祥的，强拆他家，把他弟弟用勾机给砸死了。〞
 <br/>
 <br/>
 据《民生观察》报导，哈尔滨水务投资集团有限公司，从2010年开始以排旱排涝为由，在没有任何审批手续的情况下，在松北区大面积征收农村耕地和农民宅基地，但农民根本得不到合理补偿。如今征收上来的耕地，大部分都没有投入所谓的排旱排涝工程，致使百姓的耕地荒弃了八年左右，严重的损害了国家的经济利益。
 <br/>
 <br/>
 新唐人记者熊斌、施宜君采访报导
</div>
