### 重庆重返文革？政审不合格不能高考引发轩然大波
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】近日，重庆市教育考试院发布消息称，政审材料是参加高考录取的必备材料和重要参考依据，政审不合格者不能参加高考。由于〝政审〞一词带有浓厚的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文革.html" target="_blank">
  文革
 </a>
 色彩，立即引起舆论哗然，批评和质疑的声浪迅速高涨。随后，重庆官方对外解释称，重庆市并未违背教育部的文件精神，〝政审〞之说是媒体记者对相关政策进行了〝片面误读〞。
 <br/>
 <br/>
 今年11月7日至16日是重庆市高中生参加2019年普通高考的报名时间。在此之前，中共重庆机关报《重庆日报》于11月6日发表了一篇报导，专门介绍这次高考报名制度。
 <br/>
 <br/>
 该报导称，据重庆市教育考试院发布的消息，政审材料是参加高考录取的必备材料，将反映在考生综合素质评价中，是高考录取时的重要参考依据，政审不合格者不能参加普通高校的录取。
 <br/>
 <br/>
 这篇专题报导还进一步阐释说，〝政审就是对考生的政治思想品德及现实表现进行审核。政审的结论分合格与不合格两类，政审不合格者不能参加普通高校的录取。〞
 <br/>
 <br/>
 《重庆日报》还列出了政审不合格的主要事项，包括：反对四项基本原则；道德品质恶劣；有违法犯高考罪行为。此外，报考军警、公安以及有特殊要求的院校，公安部门和院校会对考生进行再政审。
 <br/>
 <br/>
 据称，这项工作主要由考生所在的学校或单位对考生作出鉴定。
 <br/>
 <br/>
 其中，普通高中毕业生的政审，反映在普通高中学生综合素质评价中，以《重庆市普通高中毕业生综合素质评价报告》呈现，由重庆市普通高中学生综合素质评价系统生成和打印，分为电子和纸质两种形式，电子档案将在录取时提供给高校；纸质档案装入学生档案，交给录取高校存档。
 <br/>
 <br/>
 非普通高中毕业考生的政审，则使用《重庆市普通高考学生综合信息表》。对受过法律处罚或违纪处分的考生，单位要提供所犯错误的事实、处理意见和本人对错误的认识及改正错误的表现等材料，如有隐瞒，〝要承担相应的法律责任〞。
 <br/>

 <br/>
 一石激起千层浪，上述报导刊发后，很快就引发了舆论的强烈反弹，网络上质疑与批评的声浪迅速高涨。
 <br/>
 <br/>
 有网民质疑：写鉴定的人未必就公正客观，把一个不一定公正客观的结论，写进年仅十几岁尚不成熟的中学生档案里影响一生，这种事情谁能接受？
 <br/>
 <br/>
 也有网民惊呼：〝天啊，这么是要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文革.html" target="_blank">
  文革
 </a>
 吗？政审，是党说了算？〞
 <br/>
 <br/>
 还有网友评论称：〝如今的政审就是蒙人的。钞票开路，没有通不过的政审。其实这就是大学领导给自己新开了个生财的门道。〞
 <br/>
 <br/>
 有网友评论道：〝十几岁的孩子被高考压得几乎都成书呆子了，哪里还有什么政治思想？政审说白了就是查背景出身，富二代官二代拼爹怕是要被洗牌了，有海外关系的也早做准备吧。〞
 <br/>
 <br/>
 还有不少中国网民翻墙在海外中文媒体的相关报导下留言评论。
 <br/>
 <br/>
 有网友评论称：〝先把中共自己审审吧，财产公开喊了快30年还不敢实施，共产？共谁的产？公开都不敢还共产。一个混蛋政权只能培养一批满口假话大话胡话的下一代。看看这个政权还能挣扎多少年。〞
 <br/>
 <br/>
 网友jigengdui56则评论道：〝政审，像法西斯那样邪恶反动，中华文明五千年，世界文明史，没有哪国哪个朝代是以政审取士，谁也没有权利去剥夺一个年轻人获取知识的权力，按联合国宪章，牢房里的犯人都有权力在牢房中上大学。毛腊肉把人分三六九等，扼杀了我中华多少精英，现在袁二又借尸还魂，逆历史潮流而动，愚昧至极，不得人心，绝无好下场！如果共产党坚持这样搞，那就是全球最反动最无人性的政党！〞
 <br/>
 <br/>
 面对网络上几乎一边倒的质疑与抨击，重庆市教育考试院办公室主任罗胜奇出面接受陆媒《中国新闻周刊》的采访辩解说：〝关于政审是媒体记者写错了。我们叫思想政治的考核和现实表现的审核。记者把这个理解错了，就出了一个‘政审’。〞
 <br/>
 <br/>
 罗某并一再强调，重庆市并没有另搞一套，而是〝严格按照教育部的精神〞办事，〝文件是一脉相承的，没有做变化。只是记者的片面误读。〞
 <br/>
 <br/>
 罗胜奇声称，重庆市往年也都会按照教育部的文件要求，对考生进行〝思想政治的考核和现实表现的审核〞。
 <br/>
 <br/>
 据了解，在中国大陆，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高考政审.html" target="_blank">
  高考政审
 </a>
 最严苛时期是1957年的反右派运动之后。从1958年起，高考前对学生进行的政审，主要是依照家庭出身和社会关系把学生分为四类：1、可录取机密专业；2、可录取一般专业；3、降格录取；4、不宜录取（有的地方用不予录取）。
 <br/>
 <br/>
 当时，凡是出身地主富农资本家家庭的，家长被划为右派的，有海外关系的、社会关系复杂的学生基本上都是不宜录取和降格录取者。而任何学生一旦高考政审表上被盖上〝不宜录取〞或〝不予录取〞的印章，不管他的高考成绩多么优秀，都没有机会跨入大学的校门。
 <br/>
 <br/>
 文革期间，中国的高考被取消，大学一度停止招生。1970年以后，全国的大学开始按推荐制招生，而想要获得推荐成为〝工农兵大学生〞或〝工农兵学员〞，〝政审〞这一关就更加重要。不仅前述〝出身不好〞的学生过不了关，文革中被政治对象的子女也受歧视，不能被推荐上大学。
 <br/>
 <br/>
 直到〝文革〞结束后，1977年恢复高考时，在邓小平的主导下，〝政审〞的尺度才开始放松，之后〝高考政审〞的说法逐渐淡出公众视线，但实质性的〝政审〞其实一直都存在，只不过评判是否〝合格〞的具体标准，随着时代的变化有所调整。
 <br/>
 <br/>
 1999年7月20日中共在全国范围内发起对法轮功的政治迫害后，在高考招生时，中共仍然在政审环节规定了，凡是不放弃修炼法轮功的人都不能参加高考，甚至不能得到中学毕业证书，表明中共的〝高考政审〞从来都没有真正撤消，只不过变换了更具欺骗性的名词而已。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：明轩）
