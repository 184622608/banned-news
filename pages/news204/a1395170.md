 </h4>
 <br/>
 大陆河北省秦皇岛海港区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马桂兰.html" target="_blank">
  马桂兰
 </a>
 ，上个月被秦皇岛看守所迫害致死，消息10月12号才被披露出来。
 <br/>
 <br/>
 据〝法轮大法明慧网〞报导，马桂兰是一名60多岁的女法轮功学员，今年7月4号被秦皇岛市开发区珠江道派出所绑架，第二天被关进秦皇岛市拘留所非法拘留15天，7月18号，再改为刑事拘留。9月17号，被迫害致死。
 <br/>
 <br/>
 <h4>
  对付中共
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_五眼联盟.html" target="_blank">
   五眼联盟
  </a>
  与德日分享机密情报
 </h4>
 <br/>
 由美国、英国、加拿大、澳大利亚和新西兰组成的情报共享网络〝五眼联盟〞，从今年年初开始与德国和日本等国家加强合作，分享中共对外活动的机密情报。
 <br/>
 <br/>
 路透社报导说，联盟中4个国家的7名匿名官员表示，这是各国扩大战线对抗中共影响力与投资的迹象之一。多国经常以双边形式私下讨论，法国也参与其中。
 <br/>
 <br/>
 路透社评论说，〝五眼联盟〞加强合作显示，尽管美国总统川普表示准备单独对抗中共，但他的政府官员仍在幕后努力组成一个非正式联盟，让中共意图拉拢欧洲国家的希望落空。
 <br/>
 <br/>
 <h4>
  美加强管制对
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国民用核能技术.html" target="_blank">
   中国民用核能技术
  </a>
  出口
 </h4>
 <br/>
 10月11号，美国能源部表示，将加强对中国民用核能技术出口的管制，防止中共将其转用于军事或其他非授权目的。
 <br/>
 <br/>
 美国能源部长佩里在一个声明中表示，美国不能无视中共在美中民用核合作过程之外获取核技术,  所带来的国家安全影响。
 <br/>
 <br/>
 路透社报导，新的规定立即生效，并为未来审查向中国技术转让提供指引。
 <br/>
 <br/>
 <h4>
  迎新致辞提独立人格 浙江学者遭处分
 </h4>
 <br/>
 浙江传媒学院文学院副院长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵思运.html" target="_blank">
  赵思运
 </a>
 ，10月12号被校方处分。当天在大陆社交媒体流传的文件显示，赵思运遭到党内严重警告，原因是9月30号他在新生开学典礼上的致辞有〝个别不当用语〞。
 <br/>
 <br/>
 根据浙江《钱塘晚报》转载的赵思运迎新致辞全文显示，他在演讲中提到了大学生要体现独立人格，具备自律、尊重与尊严的文化素养，具有批判精神和道义担当，肩负起国家和民族未来的重任等。
 <br/>
 <br/>
 这些言论，看起来似乎没有触碰到中共的红线，外界因此对校方的处理感到不解。
 <br/>
 <br/>
