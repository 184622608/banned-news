### 【禁闻】微电影《心愿》小故事大启发 心转路也转
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】当今影视界以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员做为艺术创作背景的影视作品并不多见，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_微电影.html" target="_blank">
  微电影
 </a>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心愿.html" target="_blank">
  心愿
 </a>
 》入围了今年美国佛罗里达墨尔本独立电影人节，主办单位表示，电影传达善的信息值得推广。
 <br/>
 <br/>
 故事是从一位小偷的遭遇开始说起。
 <br/>
 <br/>
 光天化日下，一位年轻人闯入民宅，在翻箱倒柜的寻找钱财时，正巧被女主人发现了。
 <br/>
 <br/>
 〝我知道你不是我哥，我刚刚是因为害怕才叫你哥。〞
 <br/>
 <br/>
 〝我必须得拿，这是救命钱，我爸在医院躺着呢，我没办法呀！〞
 <br/>
 <br/>
 这名小偷因为父亲病危，在医院等着开刀，为了筹凑医药费，走投无路的情况下，他选择了偷窃。女主角了解实情后，决定把这笔钱给小偷，尽管这是她治疗眼睛的费用。就在小偷准备离开时，女主角好像想到什么事情。
 <br/>
 <br/>
 〝等一下，这个是我妈妈被抓之前留下来的，她告诉我，一定要把这本书送给有缘的人。我今天把他送给你。〞
 <br/>
 <br/>
 女主角的母亲因为修炼
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 被中共当局非法抓走。在中国，有无数的法轮功修炼者，像女主角的母亲一样，被非法关押在看守所、监狱。他们因为坚持不肯放弃真善忍的信仰，被中共莫名的打压迫害。
 <br/>
 <br/>
 而女主角送给小偷的书《转法轮》，在全球已发行多种语言的译本，这本书指导法轮功学员如何修炼，如何做一个更好的人。
 <br/>
 <br/>
 男主角在阅读这本书之后，为自己的所作所为感到羞耻难过，他反省自己的人生，短短数十年，应该要活得有意义。
 <br/>
 <br/>
 他发愤图强，餐厅洗碗、搬运货物，日以继夜打工赚钱，就是要把那笔钱还给女主角。
 <br/>
 <br/>
 〝小妹妹，当妳能看到这封信的时候，我的心情和你一样激动。我就是当年偷走了妳治眼睛钱的那个哥哥。〞
 <br/>
 <br/>
 原来女主角治疗眼睛的费用，是当初那位小偷帮忙支付的。他写信告诉女主角，是她妈妈留下的那本《转法轮》，改变了他。
 <br/>
 <br/>
 〝小妹妹，当妳重见光明的时候，我也要把这本书送给我的亲朋好友，我的同事以及更多的人，要让他们也能够选择光明的人生。〞
 <br/>
 <br/>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心愿.html" target="_blank">
  心愿
 </a>
 》入围美国佛罗里达墨尔本独立电影人节(Melbourne Independent Filmmakers Festival，简称MIFF)。主办方表示，每年都会从大量参赛作品中，选择少数入围影片展映，《心愿》就是今年的展映影片之一。
 <br/>
 <br/>
 影展创办人 Terry Cronin：〝显然影片传递出非常强烈的信息。一个好的消息，讲述人如何成为一个好人。像救赎一样。我认为人们真的很喜欢看。〞
 <br/>
 <br/>
 《心愿》中饰演男主角的演员姜光宇表示，因为法轮功学员作为当今最大的修炼团体，当今影视界以他们做为艺术创作题材的影视作品并不多见。
 <br/>
 <br/>
 《心愿》男主角姜光宇：〝这个电影节组委会能够选择心愿作为入围电影，我觉得从一方面，也表达了他们对于法轮功题材的认可。同时，也表达了他们对‘停止对法轮功学员的迫害’，以及维护法轮功人权，他们表达了一个非常积极正面的态度。〞
 <br/>
 <br/>
 有观众说，〝《心愿》启迪人心，坏事可能会变成好事的开头，传达的这个讯息非常好。〞
 <br/>
 <br/>
 还有许多观众在看完电影后，纷纷询问哪里可以寻找到影片中那本神奇的书──《转法轮》。
 <br/>
 <br/>
 编辑/黄亿美 后制/李沛灵
