### 【禁闻】中共没交让步清单 G20川习会蒙阴影
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 下个月将在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 召开，外界一直在揣测，美国总统川普和中国国家主席习近平，是否可能在会议期间见面，给贸易战降温。不过，川普高级经济顾问库德洛最新的讲话，似乎降低了这种可能性。来看报导。
 <br/>
 <br/>
 《美国之音》10月19号报导，美国国家经济委员会主任库德洛，周四，在底特律经济俱乐部，指责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 与美国贸易谈判中寸步不让。库德洛说：〝中国（中共）没有对我们的要求做出任何积极地回应。〞
 <br/>
 <br/>
 美国已经对价值约2,500亿美元的中国进口商品加征了关税，希望迫使中共改变不公平的贸易行为，包括知识产权、市场准入和政府补贴等领域，来平衡美、中的巨额贸易逆差。但中共也用提高关税的手段进行了报复。
 <br/>
 <br/>
 英国《金融时报》10月9号引述两位了解美中谈判情况的人士的话说，美国官员警告中共，如果不能拿出清单详细说明如何让步，川普总统将不会在11月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 上，与习近平进行贸易磋商。
 <br/>
 <br/>
 川普和习近平都会参加11月30号在布宜诺斯艾利斯举行的G20峰会。有关两人是否会面的揣测，已经流传了多天。库德洛14号在接受美国《福克斯新闻》（Fox News）的采访时刚透露〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞可能举行的讯息，随后，他的最新讲话似乎又降低了外界的预期。
 <br/>
 <br/>
 旅美时事评论员蓝述：〝北京并没有提出来切实可行，又能够让美方满意的一揽子方案吧！所以说到目前为止，川习会会不会进行都要打一个问号。〞
 <br/>
 <br/>
 旅美时事评论员唐靖远：〝其实这个是中共的体制惯性所促成的。中共如果它真的按照美国的这些要求去做到，那么中共这个体制就会受到冲击。这个是中共最担心，也是最不愿意去面对的。〞
 <br/>
 <br/>
 不过中方官员曾表示，他们有这样一份清单，但要等到〝华盛顿方面政治氛围稳定〞时，才会提交这份清单。
 <br/>
 <br/>
 蓝述：〝中共它实际上是打这个投机牌了。它并不想大幅度的让步，是因为他们对（美国）中期选举之后华府的政治气候是否有变化，抱着一丝侥幸的心理。它认为如果民主党取得多数的话，就有可能采取一些手段，让川普总统在许许多多的政策的执行方面受到阻挠。〞
 <br/>
 <br/>
 唐靖远：〝这个是他们自己打的如意算盘，并不一定能够成立。为什么呢？因为第一呢，中期选举至少从目前选前的这些情况综合来看的话，共和党其实一点不差。第二，现在川普对待中共这种强硬的立场，早已经在两党之间达成了这种高度的共识。〞
 <br/>
 <br/>
 库德洛当天还暗示美国将继续利用关税向中国施压，他说川普不会放弃批评中共〝必须做出改变〞。
 <br/>
 <br/>
 库德洛表示，美国经济强于中国，他指出，人民币贬值就是投资者对中国经济失去信心的一个迹象。今年以来，人民币兑美元已下跌近10%。
 <br/>
 <br/>
 另外，中国股市从今年2月开始，A股已经经历了9个月的绝望周期，截至10月17号收盘，沪指，年内下跌超过25%，深成指，跌超35%，创业板跌超32%。
 <br/>
 <br/>
 蓝述：〝这种经济下滑的同时，再去面对来自于美方的经济制裁的话，这就是雪上加霜。让近期内中共的体制产生大规模经济危机的可能性大大增加。〞
 <br/>
 <br/>
 唐靖远：〝如果这个僵局持续下去的话，那么中共它采取的手段就是通过牺牲老百姓的利益，把民间的财富转移到国有大型企业的手中。就是所谓的国进民退。严格意义上说它并不是国有命脉，而是中共这个特权利益阶层他们的命脉，它是要保住这个东西。〞
 <br/>
 <br/>
 包括《华尔街日报》和《彭博社》在内的美国媒体此前报导说，川普决定推进在G20峰会上与习近平会晤的计划，并委派白宫内持温和派立场的官员予以落实。不过美国财政部长姆努钦表示，川习会的前提是中共必须承诺进行重大的结构性改革。
 <br/>
 <br/>
 采访/常春 编辑/尚燕 后制/钟元
