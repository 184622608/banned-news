### 官媒首次公开马云党员身份 终结外界多年猜测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】对于中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴.html" target="_blank">
  阿里巴巴
 </a>
 创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马云.html" target="_blank">
  马云
 </a>
 是否
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共党员.html" target="_blank">
  中共党员
 </a>
 ，外界猜测争论多年。日前中共喉舌公布百名所谓〝改革开放功臣〞名单，首次洩露马云的党员身份。
 <br/>
 <br/>
 11月26日，中共机关报《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人民日报.html" target="_blank">
  人民日报
 </a>
 》刊文称，为庆祝〝改革开放40周年〞，拟表彰100名各领域的〝杰出贡献人物〞。
 <br/>
 <br/>
 党媒公布的百人名单中，占最多比例的是企业家，包含中国三大科技巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿里巴巴.html" target="_blank">
  阿里巴巴
 </a>
 董事局主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马云.html" target="_blank">
  马云
 </a>
 、腾讯CEO马化腾、百度CEO李彦宏等人。
 <br/>
 <br/>
 其中，官方的个人简介中标明马云是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共党员.html" target="_blank">
  中共党员
 </a>
 ，引起外界关注。
 <br/>
 <br/>
 在此之前，有关马云是否中共党员，外界并无定论。马云多次公开表明对中共组织和中共政策的拥护，但媒体对他是否中共体制内人士鲜有提及。在中国社交网络上，网友们为此已公开争论了多年。
 <br/>
 <br/>
 有网友提及，2010年6月，为纪念中共建党89周年，马云亲自为阿里巴巴全体党员上了一次党课。
 <br/>
 <br/>
 2010年11月，马云曾参加中共中央组织部的报告会，介绍阿里巴巴发展经验。马云在报告中多次提及〝我们党〞。
 <br/>
 <br/>
 2015年，马云出任浙商总会第一届会长时，官网的个人简介中就曾公开披露马云是中共党员，但当时并没有引起太多关注。
 <br/>
 <br/>
 外界猜测，马云近期宣布明年退休，以及将阿里巴巴旗下支付宝拱手送给官方银联组织，是否与其党员身份有关。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
