### 11月30日全球看中国
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】下面是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_全球看中国.html" target="_blank">
  全球看中国
 </a>
 <br/>
 <br/>
 1. 香港
 <br/>
 1995年，吉林永吉县发生一起杀人案，警方证据不足，却认定金哲宏为凶手，并判处死缓，多次重审无果。直到今年11月30日，金哲宏最终被判无罪。他入狱后母亲郁郁而终，妻子带着孩子改嫁。23年后，50岁的他已拄起拐杖，他说出狱后的最大心愿是去祭奠双亲。另有媒体称，大陆类似
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冤案.html" target="_blank">
  冤案
 </a>
 屡见不鲜。
 <br/>
 <br/>
 2. 大纪元时报
 <br/>
 2018年11月9日，黑龙江哈尔滨和大庆119名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 被中共警方非法绑架。美国国务院周四电邮回复大纪元记者表示：要求中共停止对法轮功的禁令，允许法轮功学员自由地修炼、实践信仰。
 <br/>
 <br/>
 3. 德国之声
 <br/>
 2017年12月，中共在新疆举办了一个〝结对认亲周〞活动，将110万政府人员配对到维吾尔人家中，一同进食甚至就寝。此后，该活动成为常态。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆统战部.html" target="_blank">
  新疆统战部
 </a>
 今年2月宣布，政府人员每两个月得与配对到的维吾尔家庭一起生活五天。
 <br/>
 <br/>
 4. 自由亚洲
 <br/>
 近期，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 增速达到十年来最低，使今年下半年中国就业市场大幅紧缩。如今年4月到9月，一大型招工网站广告条数从285万锐减至83万。这给今年820多万大学毕业生构成巨大挑战。
