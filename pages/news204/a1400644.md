------------------------

 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_绵阳公安.html" target="_blank">
  绵阳公安
 </a>
 近日在微博曝光了7名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌疑人.html" target="_blank">
  嫌疑人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姓名.html" target="_blank">
  姓名
 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_照片.html" target="_blank">
  照片
 </a>
 ，希望民众提供线索，举报在逃的犯罪嫌疑人。然而，一名叫〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卿晨璟靓.html" target="_blank">
  卿晨璟靓
 </a>
 〞的美女犯罪嫌疑人，因为高颜值意外走红。
 <br/>
 <br/>
 陆媒上游新闻11月24日消息称，〝平安绵阳〞在微博20日发布检举〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒托.html" target="_blank">
  酒托
 </a>
 〞诈骗犯罪团伙在逃人员的通告，共曝光了7名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫌疑人.html" target="_blank">
  嫌疑人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姓名.html" target="_blank">
  姓名
 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_照片.html" target="_blank">
  照片
 </a>
 ，有包括〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卿晨璟靓.html" target="_blank">
  卿晨璟靓
 </a>
 〞在内的5名女性，以及向礼貌和卢兴波两名男性。
 <br/>
 <br/>
 通告中称，凡有人提供线索，公安机关查证属实并成功抓获嫌犯后，将依据有关规定，对举报人奖励1000元至3000元人民币。并将对报案公民个人信息进行保密。
 <br/>

 然而，上述通告中，网友们没有关注什么举报奖励之类的重点，而是把关注的重点放在了一名叫〝卿晨璟靓〞的美女犯罪嫌疑人的外貌和名字上。
 <br/>
 <br/>
 有网友表示，第六个女性犯罪嫌疑人〝清新脱俗〞，怎么会做贼呢，可惜了。也有网友称，这位女嫌犯的名字也好奇怪，原来卿晨璟靓是真名？！还有网友发出〝卿本佳人，奈何做贼〞的叹息。有人则提醒说，不是贼，是〝酒托〞。
 <br/>
 <br/>
 据警方介绍，这个〝酒托〞团伙从4月以来，先后在绵阳多个酒吧茶楼作案。目前，到案的犯罪嫌疑人已被检察院机关批准逮捕，正全力追捕在逃人员，希望广大民众积极提供线索。
