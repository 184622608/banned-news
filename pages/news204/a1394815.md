### 举报范冰冰获10万奖金 崔永元自爆没心情领
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 逃税案尘埃落定。爆料娱乐圈〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞黑幕的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 说，自己实名举报范冰冰，获得10万人民币奖金，没心情领取。他还称，他遭到警方抄家式报复侦查。
 <br/>
 <br/>
 10月10日，又有一段关于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 痛斥遭报复的视频流出，视频中，崔永元颓废地坐在一间屋子的角落。
 <br/>
 <br/>
 他说，自己很忙，有很多事情要做，但〝因为我动了他们的奶酪，照死了折腾我！天天恨不得我死！〞目前这段视频已被删除。
 <br/>
 <br/>
 10月7日，崔永元也在微博小号〝小崔读书汇〞发表题为《一声长叹一声雷》的长文中，自爆因实名举报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 ，获得10万人民币奖金，但没心情领取，〝钱多少先放一边，实在看不了税务总局的一张张脸。〞
 <br/>
 <br/>
 文中披露，他第一次去税务总局，对方给看了证件倒了热茶，相对仔细地看了证据，和电影《大轰炸》有关的范冰冰的证据就是这次提供的。
 <br/>
 <br/>
 第二次去，他提供的全是华谊兄弟的证据，相当一部分税务总局看不懂。
 <br/>
 <br/>
 第三次，他是突然被叫去的，多了两张陌生的面孔，没有自我介绍也没有出示证件。对其补充提供的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 证据看都不看。拐弯抹角地询问他是从哪里弄到的《大轰炸》的合同。
 <br/>
 <br/>
 他说，这〝一高一矮两位越问越急，终于原形毕露，这是俩蛮横的警察。在他们的世界里，有罪推定，人人都是罪犯。在他们的世界观里，他们就是法律，没必要对谁客气。〞
 <br/>
 <br/>
 他表示，〝这两个身份不明的人不出示证据随意传唤审问公民是违法的，要求国税总局予以合理解释，国税总局称要请示领导，至今无下文。〞
 <br/>
 <br/>
 崔永元还说，之后上海公安局经侦大队，对他参与的所有公司彻底侦察，对他以前的助理不断询查，彻夜询查。
 <br/>
 <br/>
 他说，上海警察〝曾经当着我的面，他们喝两万一瓶的酒，抽一千一条的烟，几十万的现金用个书包就提走。就这样，我的危险来自四面八方。〞
 <br/>
 <br/>
 9月24日，崔永元对《明报》称，自己爆料〝天价阴阳合同〞就是在报私仇，4个月以来，他通过不同渠道得到3000多份材料。期间他接到利益集团或范冰冰粉丝的死亡威胁，其间他报警近10次，均未立案。他身边的工作人员也频频被约谈或传唤，使其仍存担忧。
 <br/>
 <br/>
 他说，已将部分重要资料备份，藏于多处，若遇危险会将其公之于世。
 <br/>
 <br/>
 5月底，崔永元开始曝光中国娱乐圈天价〝阴阳合同〞黑幕，直指范冰冰拍戏4天收入6000万人民币片酬。引来中共税务部门介入调查。
 <br/>
 <br/>
 范冰冰因此在公众视野里消失4个月，10月3日，中共官媒报导，税务机构查实，范冰冰及其旗下企业，累计偷税漏税8.83946亿。缴清罚款可免予牢狱之灾。
 <br/>
 <br/>
 同日，范冰冰在微博发致歉信说，接受税务机构作出的处罚决定，将〝尽全力克服一切困难，筹措资金、补缴税款、缴纳罚款〞。
 <br/>
 <br/>
 曾经在中国一线娱乐节目担任导演的姜先生，向德国之声说，眼下各路娱乐界明星及影视制作公司纷纷〝踊跃〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_补税.html" target="_blank">
  补税
 </a>
 。他认为一系列因素导致政府到娱乐圈〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_割韭菜.html" target="_blank">
  割韭菜
 </a>
 〞，总的来说是国库缺钱。
 <br/>
 <br/>
 他表示，当局处于中美贸易战，地方财政状况堪忧，崔永元的爆料，无疑给当局增加财政收入提供了绝佳的机会。之所以针对明星，是因明星们收入高民怨也高，确实也存在逃税漏税的问题。
 <br/>
 <br/>
 分析认为，中共从明星大腕中割了最大的一大把〝韭菜〞，也令范冰冰消了大财免了大灾。有推特网民调侃，范冰冰成了〝共克时艰〞的〝功臣〞。
 <br/>
 <br/>
 《美国之音》则分析认为，本次由崔永元担纲的这场娱乐大戏的主角，其前台幕后应该都是中共高层政军之间的角力。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：赵云）
