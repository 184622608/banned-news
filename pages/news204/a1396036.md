### 中共前网信办主任鲁炜出庭受审
------------------------

<div class="wysiwyg">
 【新唐人2018年10月19日讯】中国官媒周五消息，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 中宣部前副部长、前互联网信息办公室主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲁炜.html" target="_blank">
  鲁炜
 </a>
 在浙江宁波受审，他被控受贿3200多万元人民币。
 <br/>
 <br/>
 指控称，2002年7月到2017年下半年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲁炜.html" target="_blank">
  鲁炜
 </a>
 担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 新华社秘书长、副社长，北京市委常委、宣传部部长、北京市副市长，国家互联网信息办公室主任，中央网络安全和信息化领导小组办公室主任，中央宣传部副部长等职务期间，非法受贿共计3200多万元。
 <br/>
 <br/>
 鲁炜表示〝认罪、悔罪〞。
 <br/>
 <br/>
 中共前中宣部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网信办主任.html" target="_blank">
  网信办主任
 </a>
 鲁炜：〝对检察机关指控我的犯罪事实和罪名，我没有任何异议，不论如何判决，我都认罪，服判，不上诉。〞
 <br/>
 <br/>
 鲁炜去年11月落马，今年2月被〝双开〞。中共中纪委通报措辞严厉，指其〝欺骗中央、干扰中央巡视、以权谋色、拉帮结派、搞小圈子、专横跋扈、收钱敛财〞等。
 <br/>
 <br/>
 据报，鲁炜曾长期在江派前常委刘云山等掌控的中共文宣领域工作，其后台是刘云山。
 <br/>
 <br/>
 另据《明慧网》报导，鲁炜在任期间，新华社和新华网大规模编造文章，诋毁和诽谤法轮功，不少法轮功学员因为在网络发言而被迫害。对此鲁炜负有不可推卸的责任。
</div>
