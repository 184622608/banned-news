### 中国歌手陈羽凡涉毒被抓 又一新告密组织意外曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国歌手.html" target="_blank">
  中国歌手
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈羽凡.html" target="_blank">
  陈羽凡
 </a>
 ，因涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吸毒.html" target="_blank">
  吸毒
 </a>
 落网，目前已被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政拘留.html" target="_blank">
  行政拘留
 </a>
 。陆媒披露，为案件提供线索而立下大功的是北京〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_石景山老街坊.html" target="_blank">
  石景山老街坊
 </a>
 〞，这一消息令又一新生的草根版〝准特务告密组织〞意外曝光。
 <br/>
 <br/>
 北京市公安局官方微博〝平安北京〞11月28日通报，石景山公安分局根据群众举报，在某小区抓获2名涉毒违法人员陈某（男，43岁，歌手）和何某某（女，25岁，无业），现场起获冰毒7.96克、大麻2.14克。
 <br/>
 <br/>
 经尿检，陈某呈冰毒类和大麻类阳性，何某某呈大麻类阳性。目前，陈某因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吸毒.html" target="_blank">
  吸毒
 </a>
 、非法持有毒品，何某某因吸毒均被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政拘留.html" target="_blank">
  行政拘留
 </a>
 。此案正在进一步工作中。
 <br/>
 <br/>
 〝平安北京〞的微博随后转发了〝平安石景山〞的微博称，〝毒品，让‘最美’凋零〞。 因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈羽凡.html" target="_blank">
  陈羽凡
 </a>
 所在的组合，最初因单曲《最美》走红，通报所指的陈某被认为是陈羽凡。
 <br/>
 <br/>
 之后〝羽．泉〞的经纪公司〝巨匠娱乐〞证实陈羽凡涉案，并发出致歉函。而陈羽凡原定12月25日在北京登场的〝羽．泉20周年巡回演唱会〞也宣布取消，目前已展开退票手续。
 <br/>
 <br/>
 陆媒11月29日报导称，为陈羽凡吸毒案件提供线索而立下大功的是北京一个基层社会治理团体〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_石景山老街坊.html" target="_blank">
  石景山老街坊
 </a>
 〞。而这一新的草根版〝准特务告密组织〞意外曝光引发关注。
 <br/>
 <br/>
 据公众号〝长安街知事〞介绍，〝石景山老街坊〞是在基层社会治理中，逐渐发展壮大的新生群众力量。此前北京已有〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝阳群众.html" target="_blank">
  朝阳群众
 </a>
 〞、〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西城大妈.html" target="_blank">
  西城大妈
 </a>
 〞、〝海淀网友〞、〝丰台劝导队〞、〝石景山老街坊〞等以告密换取酬劳的一系列特殊组织。
 <br/>
 <br/>
 这一系列告密组织均由中共官方或警方扶持起来，被中国网友戏称为与美国的中情局、苏联的克格勃、以色列情报组织摩萨德、英国秘密情报局军情六处，并列的世界第五大王牌情报组织。
 <br/>
 <br/>
 中共官方声称，这些组织为警方提供违法犯罪信息，弥补了警力的不足。但中国民间和海外舆论界称其为〝告密组织〞，或草根版的〝准特务组织〞。
 <br/>
 <br/>
 根据北京官方公布的统计数据，2012年北京实名注册的上述组织成员已突破85万人，在全市覆盖率超过80%，共部署6.3万个巡逻点，全年累计向警方提供各类线索41万多条。网友讽刺北京已成〝告密者之都〞。
 <br/>
 <br/>
 自由撰稿人朱欣欣曾对新唐人表示，中共可以说集古今中外权术厚黑学集大成者，可以说是历史上最大的一个邪恶的组织，它们搞这种特务治国，而不是用人人平等的统一靠法律、靠公平正义来管理社会，那么这个社会肯定越发沦丧，越发危险。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
