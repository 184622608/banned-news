### 【禁闻】民企恐慌难平 习近平发信安抚
------------------------

<div class="wysiwyg">
 【新唐人2018年10月22日讯】最新数据显示，大陆民企经营恶化，国资大举收购、控股民企上市公司。中共党媒在上周末报导了中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 支持民营企业的言论，被认为是试图挽救民营企业家信心。
 <br/>
 <br/>
 中共党媒《新华社》10月21号刊登
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 给一些受表彰民营企业家的回信，其中肯定了民营企业的重要作用，并表示〝任何否定、弱化民营经济的言论和做法都是错误的〞，北京支持民营企业发展的一贯方针〝丝毫不会动摇〞。
 <br/>
 <br/>
 就在上个月，自称〝资深金融人士〞的吴小平在大陆社交平台上发文称：〝私营企业已经完成历史使命，应该退场。〞文章论点在社交媒体引发轩然大波。
 <br/>
 <br/>
 而不久前，阿里巴巴主要创始人马云也宣布将卸任董事局主席，有人解读，这是中国商业环境恶化下马云的明智离场。一时间，惶恐的情绪在中国民营企业中蔓延。
 <br/>
 <br/>
 旅居纽约的政经分析人士秦鹏认为，习近平对民企的这番话，是试图挽救民营企业家信心。
 <br/>
 <br/>
 旅居纽约政经分析人士秦鹏：〝今年以来经济加速下行、民营经济经营更加困难，企业家本来就人心惶惶，然而还是不断的有像国进民退、消灭私有经济等这种言论，马云的退休也被视为是明哲保身，所以整个的民营企业家的信心受到沉重打击。〞
 <br/>
 <br/>
 中国经济正处于下行期，中美贸易战近期又持续升温，中国〝长江商学院〞上月公布的一份研究报告显示，中国民营企业8月份的经营状况指数从7月的55.6下滑到49.8，是4年来最差的表现。
 <br/>
 <br/>
 秦鹏：〝大环境来看，经济继续下行。中美贸易战又开启，7月份以后的关税进一步造成出口压力，国内成本上升。 其次，从政策层面看，中共政府去槓杆、供给侧改革、环保法执行更加严苛等等，都明显有利于国企，对民营经济造成更大的冲击。〞
 <br/>
 <br/>
 台湾企业家与社会运动人士高为邦：〝中美贸易战目前在表面上的影响力还不是那么大，可是预期的影响力非常大，所以造成了很多的民营企业面临破产。因为这个民营企业大部分是靠外销，外销的主要对象是靠美国。〞
 <br/>
 <br/>
 经营恶化加上股市一跌再跌，民企市值不断蒸发，国资也大举收购、控股民企上市公司。《上海证券报》统计显示，今年以来，已有46家民营Ａ股上市公司获国有资本入股，当中26家属于控股权转让。台湾《中央社》说，在北京、上海、四川、湖南、福建等10多个省，今年都有当地政府资金买进上市私营企业的股权。
 <br/>
 <br/>
 高为邦：〝历史都证明了，共产主义的国家计划经济、国有产业，到最后都变成最穷的国家。今天因为这个贸易战反而搞国进民退，这个解决不了它的问题。它走回头路，那这个对整个国家的前途是更糟糕的。〞
 <br/>
 <br/>
 秦鹏：〝这个趋势还会进一步的加剧。对经济的管控也会只是越来越紧。〞
 <br/>
 <br/>
 怎么应付民企的危机？台湾企业家与社会运动人士高为邦表示，中共必须从根本上改变。
 <br/>
 <br/>
 高为邦：〝美国的要求是你基本要改变这个制度，不是说你要多买一点美国的产品。怎么解决这个问题？改变你的作风！对你的产品不补贴，你不去偷窃别人的智慧财产权，大家就变成一个公平的竞争。但是中国（中共）现在不是这个方法，它不从根本上改变。〞
 <br/>
 <br/>
 《中央社》引述招银国际首席经济学家丁安华（音译）的话表示，经济改革40年来，中国的私营企业，正陷在一个前所未见的艰难环境中。他说，政府与市场、国有和私有两者在本质上是互斥的，而〝我们不可能两者兼得〞。
 <br/>
 <br/>
 采访/陈汉 编辑/尚燕 后制/陈建铭
