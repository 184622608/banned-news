### 北京十一国宴出意外？传元老集体缺席 习感冒输液
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月03日讯】依中共惯例，每年十月一日之前，会在北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大会堂.html" target="_blank">
  大会堂
 </a>
 举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_招待会.html" target="_blank">
  招待会
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中南海.html" target="_blank">
  中南海
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_领导人.html" target="_blank">
  领导人
 </a>
 、高官集体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_聚餐.html" target="_blank">
  聚餐
 </a>
 。但有消息称，今年招待会没有中共元老们的镜头，因而怀疑元老集体缺席。另外还传出习近平患有严重感冒，招待会后加剧，在输液治疗中。
 <br/>
 <br/>
 9月30日，中共国务院在北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大会堂.html" target="_blank">
  大会堂
 </a>
 举行十一〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_招待会.html" target="_blank">
  招待会
 </a>
 〞，习近平、李克强、栗战书、汪洋、王沪宁、赵乐际、韩正、王岐山等中共高层,与1200多中外人士出席。
 <br/>
 <br/>
 10月3日，有外媒爆料，今年的十一招待会上，没有中共元老的镜头，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 、李鹏、朱鎔基、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_温家宝.html" target="_blank">
  温家宝
 </a>
 等〝集体缺席〞。不过，消息是否属实，有待进一步查证。
 <br/>
 <br/>
 依中共过去〝惯例〞，元老们几乎每年都会出席中共十一〝招待会〞，但如今曝光的镜头中没有这些人，使得关注中共政治派系及权争的人士感到意外。
 <br/>
 <br/>
 至于中共元老疑似集体缺席的原因，有一种说法是，中共元老们对现任执政者不满，才决定〝集体缺席〞。
 <br/>
 <br/>
 但上述说法有待进一步查证。因为按中共高层惯例，退休的中共元老仍然会被严格控制，不可能随便相互往来，更不可能串连起来，达成一致〝集体缺席〞。如果是真的集体缺席，不排除是北京当局的有意安排。
 <br/>
 <br/>
 另一种分析认为，可能是因为北京当局的惯例改变了，不再安排这些元老出席这场招待会，自从习近平执政以来，早已经释放出反对元老干政的信号。而且之前已经有先例。
 <br/>
 <br/>
 2016年，中共在〝七一〞到来前，于6月29日晚，在北京举行一场音乐会，于7月1日当天，又举行一个〝庆祝大会〞，然而这两场大型〝庆祝活动〞中，已退休的中共元老和前任高层，都没有出席相关活动。
 <br/>
 <br/>
 这种安排打破了以往中共党建纪念日活动，广邀退休中共元老出席的惯例。
 <br/>
 <br/>
 当时，有美媒爆料，庆祝建党活动筹备小组原先策划的预案中，这两个活动中，都有邀请中共一众元老出席。由于同年4月新疆发生〝倒习事件〞以及5月份的纪念文革50周年的〝地上和地下〞活动，被当局视为〝明目张胆的挑战行为〞，高层怀疑有〝某些中共元老〞参与，因此拒绝邀请元老们出席相关活动。
 <br/>
 <br/>
 海外有观察人士分析，此次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国殇日.html" target="_blank">
  国殇日
 </a>
 招待会，不邀请中共元老们出席，与当局两年前不邀请他们参加〝党建〞纪念活动如出一辙。根据以往的经验来看，不排除是为掩盖已退休的某中共高层人物〝出事〞内情，而采取的应急措施。
 <br/>
 <br/>
 不过，央视画面显示，此次招待会上，曾一度传言被查的前中共海军司令吴胜利露面，以及中共前国家副主席李源潮和军委副主席范长龙等人均有出席。
 <br/>
 <br/>
 尽管他们露面，但分析认为，因他们都被视为〝问题人物〞，也都曾一度传出受到调查的消息，在中共官场上，〝问题人物〞露面，并不能证实他们是安全的，近年落马的大老虎中，很多人在宣布被调查之前，也是经常露面。
 <br/>
 <br/>
 如中共前军委副主席徐才厚，在垮台前也曾多次露面，甚至还随习近平等人一起出席，在北京举行的军委文艺演出，3个月后被调查。
 <br/>
 <br/>
 另外，美媒还引述消息人士爆料，习近平从东北视察回京后患有感冒，十一招待会之后加剧，在输液治疗。
 <br/>
 <br/>
 去年平安夜，也曾传出习近平因〝腹痛〞，入住北京301医院的消息。当时美媒爆料，习近平很可能是因连续遭到数次暗杀，而导致精神紧张，引发腹痛住院。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑:戴明）
