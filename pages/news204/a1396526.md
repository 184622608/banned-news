### 【禁闻】10月23日维权动态
------------------------

 【新唐人2018年10月23日讯】下面我们来关注一下中国大陆各地民众
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 的情况。
 <br/>
 <br/>
 1.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江西千人抗议.html" target="_blank">
  江西千人抗议
 </a>
 建垃圾焚烧厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遭镇压.html" target="_blank">
  遭镇压
 </a>
 <br/>
 <br/>
 江西省鄱阳县游城乡北塘村和土塘村的上千村民，连续多天拉横幅到县政府请愿，抗议在当地兴建垃圾焚烧发电厂，大批警察到现场维稳，10月23号，更入村抓人，有妇女被打伤。
 <br/>
 <br/>
 2.
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_女律师被辱案.html" target="_blank">
  女律师被辱案
 </a>
 发酵 丈夫被消声
 <br/>
 <br/>
 广州女律师孙世华遭警方殴打、〝裸体〞羞辱事件持续发酵，继3名现场目击证人遭警方拘押后，10月20号，一直在网上为妻子发声的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权.html" target="_blank">
  维权
 </a>
 律师隋牧青的微信公众号也被永久封杀。
 <br/>
 <br/>
 3. 安徽女村民詹传云上访 多次遭暴打
 <br/>
 <br/>
 安徽省长丰县岗集镇张小圩村村民詹传云，因房屋被拆、土地被征收没有得到合理补偿而长期上访，期间多次遭警察暴力殴打和判刑。10月22号，她向媒体投诉，两个月前她到合肥市找中央巡视组申诉冤情，又被行政拘留五天。
 <br/>
 <br/>
 4. 在京维权访民拉横幅诉说冤情
 <br/>
 <br/>
 一批在京维权的访民听说中共全国人大委员长会议10月22号在北京召开，于是集体拉横幅诉说各自的冤情，并要求〝践行宪法保障人权〞。
