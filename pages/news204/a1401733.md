### 北京天安门前多人自杀 黑压压的警察围住现场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】近来，中国金融领域频频爆发诈骗大案，导致许多投资者血本无归。日前有5名受害者前往北京上访讨公道，却频频吃闭门羹，在走投无路之下，5人在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天安门.html" target="_blank">
  天安门
 </a>
 广场喝农药自杀。事发后，大批警察及救护车黑压压的来到现场。
 <br/>
 <br/>
 据《大纪元》报导，知情人士透露，11月30中午12时30分左右，5名投资受害者骑着脚踏车来到正对着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天安门.html" target="_blank">
  天安门
 </a>
 的广场前，一下车就喝农药自尽。由于天安门广场前三五米就有警察，他们的举动立刻被警察发现，随后大批黑压压的警察及救护车来到现场。
 <br/>
 <br/>
 有知情人士透露，这些自杀者目前生死不明，都是天津现货交易所金融诈骗受害者，5人中有，2名男子来自山东，1名男子来自湖北，另外2名女子来自新疆和湖南。他们被诈骗资金最多的达到250万人民币，最少的也超过40万人民币。其中1人曾在天津喝过农药，且讨回20%的资金。
 <br/>
 <br/>
 由于事发地处于高度敏感的天安门广场附近，北京警方没有公布此次事件来龙去脉。
 <br/>
 <br/>
 宁夏银川的天津矿产资源交易所受害者杨先生对大纪元说，他被骗75万余元，现在被家人赶出来，陪伴他的只有一辆破旧的车子，他历经3年的司法程序，公安局、检察院跑了至少有200趟，法院进不去，最近案子处于搁浅状态。
 <br/>
 <br/>
 杨先生说：〝3年中，我见证了两次跳楼，五次喝葯，老百姓命太贱了，我们在维权的路上没有尊严 。〞
 <br/>
 <br/>
 在此之前，天津现货交易金融诈骗受害者通过自杀讨血汗钱已发生多次。2016年8月25日，2人在北京地铁喝农药自杀。同年12月26日，有8人在北京天安门西站的电车上喝农药。他们因债台高筑、妻离子散、家破人亡，走投无路，最后选择自杀的方式进行维权。
 <br/>
 <br/>
 天津现货交易所当初对外宣传为，国内唯一采用矿产资源产品等现货协议转让制度的现货电子交易，号称门槛低、交易自由、保值避险、以小博大、资金回笼快、涨跌易判断等。
 <br/>
 <br/>
 但投资受害人说：交易所盘上的数据只是一个数字，数字没了投资人的钱也就没了，就是一个〝买空卖空〞的对赌式交易骗局。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
