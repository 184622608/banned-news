### 内忧外患 中共严控信息 或适得其反
------------------------

<div class="wysiwyg">
 【新唐人2018年11月15日讯】随着中美贸易争端不断升级、中国经济数据不断走低，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 近期进一步加强了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舆论管控.html" target="_blank">
  舆论管控
 </a>
 。但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆自媒体.html" target="_blank">
  大陆自媒体
 </a>
 人认为，中共这样做反而会促使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网民.html" target="_blank">
  网民
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻墙.html" target="_blank">
  翻墙
 </a>
 找真相。
 <br/>
 <br/>
 大陆经济不断下走，企业倒闭潮和民众失业潮引发
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网民.html" target="_blank">
  网民
 </a>
 热议；不少中产家庭沦为金融难民后，维权艰难，也开始关注
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 迫害人权的历史和现状。美国等西方国家全面遏制中共的消息，也被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻墙.html" target="_blank">
  翻墙
 </a>
 网民搬回墙内。
 <br/>
 <br/>
 目前中共内外交困，我相信中共统治者他们认为对他们来说是一个坎儿，中共它越到危机的时候，它越要管控的严厉，为了度过这个危机。
 <br/>
 <br/>
 中共网信办11月12号宣布，10月20日至今集中关闭了9800多个自媒体帐号，并警告腾讯微信和新浪微博，要求它们加大力度，审查平台上的言论。
 <br/>
 <br/>
 自媒体业者 王先生：〝对言论的控制，对自媒体的管控比以前严格的多。讲到人权、自由、历史真相的公众号或者微博，特别容易被封号。有不少很好的文章以前能发出来，但是现在就发不出来。还没发送，在后台就已经被强行删除，这种现象以前是没有的。〞
 <br/>
 <br/>
 时政评论员 唐靖远：〝当前中共仍然试图把它面临的经济和金融危机转嫁到民间去，如果民众掌握真相，大规模的自救或恐慌都有可能摧垮中共对社会的管控状态。中共的如意算盘是希望民众在无知中保持自我麻醉状态，成为它一茬茬反复收割的韭菜。〞
 <br/>
 <br/>
 广西桂林电子科技大学日前发文件，要求清查全体师生的手机电脑、移动硬盘等，查看是否有所谓反动视频等内容。引起大陆网民和海外媒体强烈批评，校方滥权违法，侵犯师生隐私权。校方随后改口，可能缩小清查范围，但承认这项行动是〝上面有要求〞。
 <br/>
 <br/>
 时政评论员 唐靖远：〝中共最近的做法有几个目的，一是定点打击已经觉醒的人，通过剥夺他们的发言权，来阻止他们进行社会启蒙；二是恐吓那些正在觉醒的人，通过制造恐怖氛围，想迫使这些人自我审查不再寻找真相，退回到愚蒙状态。〞
 <br/>
 <br/>
 做（公众号）的人觉得没有什么可发的，想看文章的人看的都是千篇一律的。一个事情发生了，没有深度分析，也没有真实报导。有网友讲，看来以后国内的网络没有什么可用的价值了，不如翻墙到墙外去，用墙外的网络才能看到一些真相。反而促使人翻墙。
 <br/>
 <br/>
 有网络名人质疑，中国人〝还能谈啥〞？〝医学谈深了，就涉及法律；历史谈深了，就涉及政治；经济谈深了，就涉及政策；艺术谈深了，就涉及价值取向；人文谈深了，就涉及思想道德〞，质疑中共试图让百姓完全噤声。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
