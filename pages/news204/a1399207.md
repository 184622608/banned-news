### 【禁闻】半个中国笼罩猪瘟 疫情失控 防治仍无解
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 持续在中国蔓延，目前已知有17个省市爆发疫情。不仅疫情无法控制，现在又传出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪饲料.html" target="_blank">
  猪饲料
 </a>
 检测出非洲猪瘟病毒。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 疫情在中国不断扩大。
 <br/>
 <br/>
 今年8月中国出现首例非洲猪瘟疫情后，短短三个月，疫情扩散快速。到目前为止，有17省市出现疫情，包括辽宁、黑龙江、河南、江苏、安徽、浙江、内蒙古、吉林、天津、山西、云南、湖南、贵州、重庆、湖北、福建、江西。
 <br/>
 <br/>
 11月7号，湖北省罗田县的一处养殖场查出非洲猪瘟，这个养殖场实际生猪数量821头，发病22头，死亡4头，当时中共农业农村部声称，疫情已得到有效处置。然而，11月12号晚间，湖北省武穴市又传出两个养殖户发生非洲猪瘟疫情。短短不到一星期的时间，同一省分陆续有三个养殖户发生猪瘟疫情，虽然当局都说已经有效处置，但还是令人担忧。
 <br/>
 <br/>
 有民众指控，政府不积极采样、甚至封锁相关消息，导致疫情如野火般扩散。本台记者电话采访上海兽医研究所，询问有关猪瘟疫情现况，不过，接电话人员马上把问题推给另外一个单位。
 <br/>
 <br/>
 上海兽医研究所：〝我们这里是上海兽医研究所，请问有什么事情？〞
 <br/>
 记者：〝我想问一下猪瘟，我看现在很多…〞
 <br/>
 上海兽医研究所：〝这个是另外的一个部门，你记一下电话号码34293773。〞
 <br/>
 <br/>
 记者接着拨打这支电话，已没人接听。
 <br/>
 <br/>
 中共前卫生部官员陈秉中：〝就是隐瞒疫情，监管严重失职，那么这也是造成疫区扩大的原因。还有老百姓不知道猪瘟的危害，还吃这个死猪肉。〞
 <br/>
 <br/>
 中共前卫生部官员陈秉中表示，各地发生猪瘟疫情，中共当局不能在疫情发生后才说已经得到有效处置，应该早点对外公布，让民众做好准备以免买到病死猪肉。
 <br/>
 <br/>
 陈秉中：〝发生的病猪应该火化，不要吃，这就切断了传染源，也不会发生。可是疫区发生之后，仍然继续销售，病猪还往外卖，这样就造成传染。〞
 <br/>
 <br/>
 陈秉中认为，中共中央应该负起责任，但目前没看到有人出面作检讨。
 <br/>
 <br/>
 陈秉中：〝这么多的猪瘟，到现在中央没有追究一个部门领导的责任，现在就是一笔糊涂帐了，已经失控了，那只能任其泛滥，受损失的是养猪的老百姓。〞
 <br/>
 <br/>
 据了解，非洲猪瘟，是一种经病毒传播的猪只传染病。感染了剧毒性病毒的猪会出现发烧、淋巴腺和内脏出血等，病程持续1-2周后暴毙，猪只死亡率几乎达到100%。
 <br/>
 <br/>
 非洲猪瘟传染力超强，可以在冷冻肉内存活三年，猪只被感染后致死率达100%，没有疫苗可以接种。
 <br/>
 <br/>
 中国过去从来没有应对过非洲猪瘟疫情，中共当局至今也没有办法有效控制疫情蔓延。大陆《财新网》日前以〝幽灵〞来形容非洲猪瘟，指它〝无声无息、踪迹不明〞，在中国的传播途径仍然是个谜。
 <br/>
 <br/>
 不仅猪瘟难以控制，现在又传出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪饲料.html" target="_blank">
  猪饲料
 </a>
 也检测出非洲猪瘟病毒，这无疑是雪上加霜。
 <br/>
 <br/>
 11月9号，安徽省青阳县一家养殖场送检的饲料样品中，检出疑似非洲猪瘟病毒核酸阳性。送检的饲料样品是〝唐人神〞集团旗下公司江苏溧阳比利美英伟营养饲料有限公司所生产。据了解，唐人神集团是首批农业产业化国家重点龙头企业，主要业务围绕生猪全产业链开展经营，包括饲料、养殖、肉品加工共3大业务。
 <br/>
 <br/>
 这也是中国爆发非洲猪瘟以来的首宗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_饲料污染.html" target="_blank">
  饲料污染
 </a>
 案例。
 <br/>
 <br/>
 10月24号，中共农业农村部官网刊文说，境内发生的前21起非洲猪瘟疫情中，62%与喂食餐厨剩食有关，但盘锦市的多位疫区生猪养殖户和防疫员表示，当地早已不用馊水喂养生猪，但会使用来自沈阳和营口的饲料。而这两个地方都已爆发非洲猪瘟疫情。
 <br/>
 <br/>
 采访/常春 编辑/黄亿美 后制/陈建铭
