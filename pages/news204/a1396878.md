### 朱军性骚扰案首次开庭 弦子：证人很多
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】中共央视主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军性.html" target="_blank">
  朱军性
 </a>
 骚扰案10月25日首次开庭，当事人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弦子.html" target="_blank">
  弦子
 </a>
 （化名）现身，但朱军并未出现。庭审结束后，弦子和好友〝麦烧〞一同接受了媒体采访。麦烧表示，愿意帮助弦子出庭作证的人很多。
 <br/>
 <br/>
 朱军起诉
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_弦子.html" target="_blank">
  弦子
 </a>
 和〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麦烧同学.html" target="_blank">
  麦烧同学
 </a>
 〞（微博名）的名誉权纠纷案，25日在北京市海淀区法院首次开庭。据悉，当日主要进行证据交换流程，全程历经两小时。
 <br/>
 <br/>
 庭审结束后，弦子向等候在外的媒体表示，已向法院提交了对朱军的〝反诉〞，至于第一次正式开庭时间是何时尚未确定，也有可能会根据本次证据提交情况添加一个庭前会议。
 <br/>
 <br/>
 〝麦烧同学〞说，愿意帮助弦子出庭作证的人很多，包括弦子出事后，第一时间倾诉的同学，和求助的老师，以及这位老师的律师朋友。
 <br/>
 <br/>
 弦子表示，自己有一定压力，但是必须要克服，她的父母也在背后支持。现在，她正努力协调工作和生活，希望能赢得此案，告诉大家女性在这个社会的应该要勇敢。
 <br/>
 <br/>
 从网传视频可见，庭审结束后，朱军代表律师离开法院时被大批媒体记者围堵与追问，但其拒绝回答任何提问。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朱军性.html" target="_blank">
  朱军性
 </a>
 骚扰案于8月在网路曝光，女当事人弦子发文称，2014年6月自己在央视实习，在化妆室单独采访朱军时，被朱军〝隔着衣服试图猥亵〞。弦子当即表示抗绝，但朱军不顾她的阻拦，直至有嘉宾进入化妆间，弦子才得以脱身。
 <br/>
 <br/>
 事后，朱军委托律师否认性骚扰事件，并起诉弦子和〝麦烧同学〞名誉侵权；弦子一方表示应诉，并起诉朱军。案件正在进行中。
