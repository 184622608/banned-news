### 【禁闻】〝进博会〞气氛异常 中共热烈 西方杯葛
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对外宣称，11月5号在上海开幕的〝中国国际进口博览会〞（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 ）是新一轮对外开放政策的宣示。不过，这场自去年就开始宣传的首届进博会，世界主要经济体与中国重要经贸伙伴，却都没有派出重要人物出席。请看报导。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 党媒宣称，国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 会在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 〞的开幕式上讲话，他还会在前一天出席外国领导人和政府官员的宴会。
 <br/>
 <br/>
 不过，以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 为首的七国集团等重要人物都不出席进博会，中国的主要交易伙伴国韩国、澳大利亚也没派出有影响力的人物参会。
 <br/>
 <br/>
 旅美时事评论员郑浩昌：〝这次进博会比2015年的9・3阅兵更冷清，那次有30个外国领导人到场，至少还有个韩国总统朴槿惠撑场。这次除了‘老朋友’俄罗斯，实在拿不出什么有分量的人物了。〞
 <br/>
 <br/>
 美国大使馆发言人10月24号表示，美国不派政府高官参加进博会，以此敦促北京结束有害及不公的贸易行为。
 <br/>
 <br/>
 近日，德国和法国的驻华大使，也罕见地在中共媒体上联名发表文章，呼吁中共以实际行动证明它的改革开放政策将继续深化。
 <br/>
 <br/>
 欧、美政商界认为，中共应提出取消贸易壁垒的具体行动，而不是举办华而不实的博览会。
 <br/>
 <br/>
 中共商务部副部长王炳南，11月3号，在进博会的新闻发布会上说，这是迄今为止世界上第一个以〝进口〞为主题的国家级博览会。他宣称，中国开放的大门只会越开越大，而且会以更大力度和更高水准对外开放。
 <br/>
 <br/>
 王炳南还表示，为促进成交，中国组织了39个交易团，近600个交易分团，有10万多家骨干企业，超过15万以上采购商将到进博会洽谈采购。
 <br/>
 <br/>
 习近平早在去年5月声称，进博会是宣示中国主动开放市场的重大政策和行动。
 <br/>
 <br/>
 旅美时事评论员郑浩昌表示，中共在主场国际活动中，历来都只是重面子。但由于中共在贸易壁垒、盗窃知识产权等方面表现恶劣，西方大国不愿与其交好而前来捧场。
 <br/>
 <br/>
 时事评论员田园则介绍说，最近欧盟商会代表1,600家欧洲企业发布了一个报告，指出欧盟企业在中国受到不公平的竞争、受到强迫技术转让等问题。欧盟商会还抱怨他们的企业在与中共国企竞争中受到歧视，甚至没有一家企业能在一带一路的专案中获得订单。
 <br/>
 <br/>
 时事评论员田园：〝欧盟对中共的态度一直是比较低调，一直处于绥靖的状态，基本上欧盟接受了中共制定的贸易规则。它就准备在中共制定的贸易框架里面，和中共老老实实做生意。〞
 <br/>
 <br/>
 田园表示，在川普总统的带领下，美国对中共发起了要公平竞争、公平交易、互利互惠的这样一种贸易战。在此背景下，这让以前对六四、对西藏等问题上一向软弱的欧盟也有了勇气，敢于对中共提出了批评。
 <br/>
 <br/>
 田园：〝如果欧盟、美国还有日本，真正能够齐心协力，真正能够达成一致说，如果你中共不在技术转让、不在智慧财产权、不在市场准入、不在非法补贴这个问题上去符合国家贸易规则，去符合国际贸易框架，那么我们就要对你采取真正的行动。这才是最终西方对中共在贸易上取胜的唯一办法。〞
 <br/>
 <br/>
 今年以来，外资持续流出中国资本市场，进而影响了中国股市。中国已经约有500多万家企业倒闭，A股市值也蒸发近20万亿人民币。
 <br/>
 <br/>
 田园认为，如果只有美国一个国家对中共采取贸易制裁行动，想让中共屈服的时间就会拖长，过程也艰难。如果欧盟、日本、加拿大、澳洲、台湾等被偷窃知识产权的国家都加入，制裁的力量才够大。
 <br/>
 <br/>
 采访/陈汉 编辑/宋风 后制/钟元
