### 【禁闻】非洲猪瘟疫情扩及中国12省市 恐失控
------------------------

<div class="wysiwyg">
 【新唐人2018年10月24日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 蔓延到湖南省。中国已有12个省市区受灾，疫情仍然无法有效控制。有民众质疑，当局对非洲猪瘟的蔓延采取了一贯性的新闻管制，掩盖其严重程度。
 <br/>
 <br/>
 中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非洲猪瘟.html" target="_blank">
  非洲猪瘟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疫情.html" target="_blank">
  疫情
 </a>
 扩大，现在连湖南省也出现疫情。
 <br/>
 <br/>
 中共农业农村部23号通报表示，湖南省的益阳市桃江县和常德市桃源县，查出猪只感染非洲猪瘟的疫情，在两个养殖场饲养的814只生猪，有252只病发，48只死亡。
 <br/>
 <br/>
 这是继浙江台州和云南发现非洲猪瘟后，又一起新的疫情。
 <br/>
 <br/>
 农业农村部分别在21号与22号公布云南省发生首起非洲猪瘟疫情，以及浙江省的1起疫情，现在又公布湖南省的疫情，外界担心，非洲猪瘟可能已经扩散到中国南方省分。
 <br/>
 <br/>
 然而，从农业农村部的官网可以看到，当局很快解除了疫区的封锁。《中国食品科技网》报导，疫区陆续解除封锁，猪价有望回升。至于灾情如何，从网站上看不到相关讯息，其它媒体也没有报导。
 <br/>
 <br/>
 湖南居民张先生：〝这段时间以来，我们从官方的报导上面就没有这个讯息，我们这边老百姓，就没有关注这个问题，所以老百姓吃猪肉还是在照常的吃，跟过去没什么区别。这个东西如果是真实的话，那就表明政府就跟过去一样，它仍然是采取封闭、封锁的手段，不告知老百姓真相。〞
 <br/>
 <br/>
 之前，中共政府对于〝非典型肺炎〞（SARS）和禽流感等蔓延迅速的疫情，一向采取封锁消息的手段。对比非洲猪瘟在俄罗斯爆发的来势汹汹，一些舆论质疑，当局对非洲猪瘟也采取了新闻管制。
 <br/>
 <br/>
 虽然中共当局相关单位表示，会严防严控染上非洲猪瘟的产品流入市场，流向餐桌，不过，近期却有其它国家海关检查出中国旅客携带有问题的猪肉制品，试图闯关。
 <br/>
 <br/>
 10月初，日本海关首度查获，北京旅客私带香肠过境，这些香肠被验出非洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪瘟病毒.html" target="_blank">
  猪瘟病毒
 </a>
 。之前也传出中国游客违规携带猪肉制品，闯韩国海关。
 <br/>
 <br/>
 中国不愿具名农业研究人员：〝目前国内的状况是这样的，上面喊得很厉害，但是下面阳奉阴违也很厉害。所以非洲猪瘟这个时候在中国，我个人的看法，完蛋了，没什么招，一定就会蔓延，就不知道气候会不会有影响。〞
 <br/>
 <br/>
 这位不愿具名的大陆农业研究人员接受记者采访时表示，如果今年冬季特别冷，病毒比较不活跃，或许可以稍微抑制疫情。
 <br/>
 <br/>
 中国农业研究人员：〝从现在这种情况看，政府的重视程度也不够，刚开始的时候，听到了一些风声，各地都在抢救什么的，但是现在就听不到了，不知道会怎么办。总之这个情况，我们就是无法乐观。〞
 <br/>
 <br/>
 自今年8月起，中国各地爆发非洲猪瘟疫情并持续扩散，由于没有有效的办法控制，猪只的死亡率几乎达到100%，
 <br/>
 <br/>
 据统计，目前中国大陆有12个省、市、自治区发生非洲猪瘟疫情，包括：黑龙江、江苏、浙江、安徽、河南、吉林、内蒙古、辽宁、天津、山西、云南与湖南。
 <br/>
 <br/>
 由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_猪肉产品.html" target="_blank">
  猪肉产品
 </a>
 是跨境传播非洲猪瘟病毒的主要载体，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_联合国粮农组织.html" target="_blank">
  联合国粮农组织
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FAO.html" target="_blank">
  FAO
 </a>
 ）在日前也发布声明警告，非洲猪瘟有可能扩散至亚洲其它国家，并且〝具有迅速在全球蔓延的可能性〞。
 <br/>
 <br/>
 为防堵中国大陆非洲猪瘟疫情入侵，台湾农委会防检局表示，旅客如果从非洲猪瘟或口蹄疫疫区携带家畜肉品等入境，将被处以重罚。
 <br/>
 <br/>
 采访/陈汉 编辑/黄亿美 后制/周天
