### 重庆坠江公交打捞出水 官方未通报确切死亡人数（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】重庆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠江公交车.html" target="_blank">
  坠江公交车
 </a>
 被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_打捞出水.html" target="_blank">
  打捞出水
 </a>
 ，距事发时已过去80多个小时。中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_官方未通报.html" target="_blank">
  官方未通报
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_确切死亡人数.html" target="_blank">
  确切死亡人数
 </a>
 ，但目击者透露，事发时，公交车上满载乘客，死亡人数远超官方此前通报的15人。
 <br/>
 <br/>
 北京时间10月31日晚23时30分左右，重庆万州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠江公交车.html" target="_blank">
  坠江公交车
 </a>
 被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_打捞出水.html" target="_blank">
  打捞出水
 </a>
 。从网传视频可见，该车车身明显变形，车窗玻璃全部破裂。车体全部露出水面后，在打捞船上的工作人员拿出手电筒照射车厢内部。23时41分，整辆车全部离开水面。此时距事故发生已过去80多个小时。
 <br/>
 <br/>
 中共官方并未通报事故
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_确切死亡人数.html" target="_blank">
  确切死亡人数
 </a>
 ，但据此前报导，官方初步核实公交车上有15人（含公交车司机1人），目前已有9人遗体被打捞上来，另有6人仍然失踪。
 <br/>
 <br/>
 不过，有事发时的目击者此前透露，公交车坠江时车上满载乘客，死亡数字远超官方通报的数字。还有网民质疑，当局选在深夜打捞或许是有心隐瞒真相:〝晚上看不清到底死多少人，你懂的。〞
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 另外，在事故发生后第一时间就有网民爆料称，出事公交车车牌为F27085，22路车，驾驶员叫冉勇，车上可能有29人。
 <br/>
 <br/>
 随后，又有消息称，冉勇在出事4个半小时前的凌晨时分，曾在全民K歌APP演唱歌曲《再回首》。外界猜测，冉勇或因疲劳驾驶，酿成悲剧。
 <br/>
 <br/>
 最令外界质疑的是，公车上都安装与公司联网的随车监控系统，但迄今为止，中共官方仍未公开随车监控系统的任何资讯。这也令外界对其通报的事故死亡人数，产生质疑。
 <br/>
 <br/>
 这起事件发生在10月28日10时8分，重庆市万州区一辆公交车行驶至长江二桥时，突然失控横跨3条行车线，与对面一辆红色轿车相撞后，公交车撞断桥边护栏，坠入长江。整个过程不到5秒时间。事故发生的瞬间，红色轿车后面的一辆汽车的行车记录仪拍下全过程。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
