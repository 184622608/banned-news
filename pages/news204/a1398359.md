### 遭北京〝抛弃〞  刘强东再缺席互联网大会
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】11月7日，中国第五届乌镇世界
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_互联网.html" target="_blank">
  互联网
 </a>
 大会开幕。多位中国互联网巨头大老现身乌镇。 但未见电商巨头京东集团创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 。刘强东在美涉性侵案以来，已〝失声〞两个月，中共海外党媒称其已经遭北京〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抛弃.html" target="_blank">
  抛弃
 </a>
 〞。
 <br/>
 <br/>
 中国第五届乌镇世界
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_互联网.html" target="_blank">
  互联网
 </a>
 大会开幕的前一天晚，中国互联网巨头阿里巴巴创始人马云、腾讯集团马化腾、小米集团雷军等陆续抵达乌镇，但未见性侵丑闻缠身的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 。
 <br/>
 <br/>
 中共海外党媒5日报导称，刘强东已经〝失声〞60天遭北京〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抛弃.html" target="_blank">
  抛弃
 </a>
 〞了。
 <br/>
 <br/>
 11月4日，陆媒刊文关注在过去60多天里，刘强东的行踪和京东集团的发展情况。文章称，刘强东除了案发初期现身京东集团〝正常开展工作〞，以及10月中旬亮相英国王室婚礼外，几乎所有重要场合都缺席。
 <br/>
 <br/>
 9月17日，2018世界人工智慧大会在上海开幕，马云、马化腾、李彦宏、雷军等众互联网大佬云集，刘强东缺席那次重磅大会。
 <br/>
 <br/>
 9月18日至20日，天津2018夏季达沃斯论坛上，依然没有刘强东的身影，而就在年初的2018冬季达沃斯论坛，刘强东携章泽天在达沃斯开专场会议、秀英文、宴宾客。
 <br/>
 <br/>
 中共官方10月份发布《改革开放40年百名杰出民营企业家名单》，鼓励民企老板〝不忘创业初心〞。名单中也没有刘强东。
 <br/>
 <br/>
 11月1日，习近平主持召开民营企业座谈会，安抚所有民营企业和民营企业家都是自己人。刘强东同样缺席会议。
 <br/>
 <br/>
 另外，随着刘强东性侵丑闻发酵，京东集团的股价也急转直下，持股机构数量也暴跌。2018年第二季度的持股机构达585家，而第三季度仅有100家。
 <br/>
 <br/>
 刘强东8月底在美国涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 被捕丑闻震惊舆论。据美国警方9月4日公布的报告，刘强东涉嫌强奸发生在8月31日凌晨1点左右，当晚被捕入狱，16个小时后获释。
 <br/>
 <br/>
 警方对刘强东一案的描述是〝一级性犯罪：强奸既遂〞。
 <br/>
 <br/>
 9月21日，美国警方称，已初步完成调查案件移交检方。美国检方表示，待审查所有证据后决定是否指控刘强东。美国检方称这是一个〝棘手的案子〞案件仍在审查中，有华裔刑案律师认为，刘强东遭起诉的可能性超过90%，涉及重新逮捕等问题。
 <br/>
 <br/>
 今年45岁的刘强东不仅是中国商界亿万富豪，同时还是中共全国政协委员，并兼任中国民间商会副会长。作为中共的红顶商人，在美国遭遇强奸犯罪指控，无疑也是中共一大丑闻。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：戴明）
