### 【禁闻】欧盟研究：中国正为贸易关税付更高代价
------------------------

<div class="wysiwyg">
 【新唐人2018年11月22日讯】美中贸易战对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 影响加剧，美国已经对2,500亿美元中国商品征收关税。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 （EconPol Europe）新发表的论文指出，因为川普政府在选择征税商品上深思熟虑，使得中共在关税上付出更多，也承担贸易战的大部分成本。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 的〝欧洲经济和财政政策研究网络〞（EconPol Europe）发表新论文谈到，美国总统川普正成功迫使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国.html" target="_blank">
  中国
 </a>
 承担贸易战的大部分成本。
 <br/>
 <br/>
 欧盟研究员强调，在美国对2,500亿美元的中国商品征收25%关税之后，美国企业和消费者只承担其中的4.5%，另外20.5%则由中国制造企业承担。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田认为，过去中美贸易之间极度的不平衡，是造成中共有利可图的主要原因。
 <br/>
 <br/>
 美国南卡罗莱纳大学艾肯商学院教授谢田：〝中共可以享受很大的贸易顺差，很多的外汇。对美国来说，直接贸易的逆差损失之外，还有美国的公司迁出美国进入中国，造成美国国内失业。从川普这种贸易制裁，除了把那些迁出的企业迁回美国，降低美国的失业率，增加美国政府的税收。所以说，我认为至少90%的成本是在中国大陆。〞
 <br/>
 <br/>
 《彭博社》报导，美中贸易纠纷的影响，并没有因为川普将和中共领导人习近平在阿根廷见面而退烧。美方加征关税的做法正如川普所愿，将减少三分之一以上的中国商品进口量，并将双边贸易逆差降低17%。
 <br/>
 <br/>
 欧盟的论文还指出，川普政府选择的加征关税产品，具有最高的〝价格弹性〞或替代性，受关税影响的中国产品，大多可被其它地区的商品取代，这就迫使出口商必须降低价格以吸引买家。而川普政府加征关税的做法，将为美国带来184亿美元的净收益。
 <br/>
 <br/>
 美国UCLA安德森管理学院经济学家俞伟雄：〝美国从中国进口的产品，消费者其实有其它的选择，当这个弹性比较大的时候，通常生产者必须要来负担这个所谓关税的部分。那他讲的没错，很大部分由中国出口商来承担。〞
 <br/>
 <br/>
 美国贸易代表办公室（USTR）去年8月依总统川普的指示，对中共展开301调查。今年3月提出调查报告，确认中共在四个方面存在贸易不公平手段，包括强迫美国公司交出商业机密以换取进入中国大陆市场的机会、迫使美国企业以不合理的条件在大陆授权其技术，和利用国家资本购买美国技术，以及直接采取盗窃手段。
 <br/>
 <br/>
 《路透社》消息指出，北京已经就美方对贸易问题所提的要求及其它议题，向华府提出书面回应。
 <br/>
 <br/>
 俞伟雄：〝（北京）给美国一个所谓改善的清单，有一百多项，但是少了五项最重要的东西，中国就是面临一个选择，我觉得如果是有智慧的话，应该全面配合美国各项的要求，这样就可以避免关税的增加。因为川普之前就已经讲了，他愿意、他很希望跟中国有一个很好的结果。〞
 <br/>
 <br/>
 中共在美国关税上付出高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_代价.html" target="_blank">
  代价
 </a>
 ，另一方面，中国企业在海外大肆并购也受到欧盟关注。美国《华尔街日报》报导，在中国海外并购狂潮之后，欧盟将加强外国投资审查。
 <br/>
 <br/>
 谢田：〝现在欧盟也从中美贸易中看到了，中共这些卑劣的做法，开始严加防范，那就可能让中共未来的计划可能也要受到挫折。〞
 <br/>
 <br/>
 美国UCLA安德森管理学院经济学家俞伟雄认为，中共想要透过这种购并直接取得先进的技术，这种强烈的企图心，让欧美先进国家很不舒服。
 <br/>
 <br/>
 俞伟雄：〝就是中国有很多不符合世界自由贸易标准的一个架构，欧美觉得过去这种尊敬的态度没有办法让中国做改变，那现在开始采取强硬的态度，我觉得这也是不得不的一个作法。〞
 <br/>
 <br/>
 欧盟委员会周二暂时达成协议，将成立一个系统，来审核进入欧洲的投资。议员们在磋商中提到〝国有企业〞对欧洲的投资，以及〝技术窃取〞等问题。明显剑指中共。
 <br/>
 <br/>
 采访/常春 编辑/黄亿美 后制/钟元
