### 范冰冰逃税爆内幕  6名税务局高层涉案受查
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】中国女演员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 因逃税被罚款8.8亿元人民币，再次成为人们关注的焦点。有媒体10月8日爆料，一共有5名税务局高层和一名会计师涉范冰冰案，如今已经被惩处。
 <br/>
 <br/>
 陆媒报导，对于女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 涉〝阴阳合同〞案，10月8日，中共国家税务总局以〝管理不力〞为由，问责了江苏省税务局6名高层。
 <br/>
 <br/>
 六名被处罚的江苏省税务局官员包括：无锡市税务局局长丁源和总会计师李清被给予行政警告处分，干部许崇金被诫勉谈话处理，滨湖区税务局副局长钟小新被给予行政记过处分，并免去其现任领导职务，梁溪区税务局副局长沈治国则以行政记过处分。
 <br/>
 <br/>
 这六人全都是被中共国家税务总局按问责条例、行政公务员处分条例和税收违法违纪处分条例进行的惩处。
 <br/>
 <br/>
 对涉范冰冰案惩处违规税务局官员，网民称〝大快人心！〞并喊话高层，继续深挖涉案人员，千万不要把大鱼放跑了。
 <br/>
 <br/>
 范冰冰因卷入〝阴阳合同〞逃税案，从公众视线中消失4个月后，10月3日，国税总局公开对其调查和裁罚结果。
 <br/>
 <br/>
 税务部门称，已查清范冰冰及其工作室以欺骗手段逃漏税，将向她收取逾8.8亿元（人民币，下同）的罚款和未缴税款，如果税款缴清，将不追究其刑事责任。
 <br/>
 <br/>
 至此，一场〝范冰冰在哪里〞的悬案终于落幕。
 <br/>
 <br/>
 然而舆论又开始质疑，范冰冰偷漏税这么多，为什么现在才要求补税，当地税务机关哪去了？
 <br/>
 <br/>
 如今虽然中共官方追责6名税务官员，但分析认为，这只是揭开了逃税案的冰山一角，范冰冰免予刑责实为保更多人。
 <br/>
 <br/>
 《美国之音》引述旅美资深媒体人凌沧洲的分析，范冰冰逃税案4个多月来出了这么个结果，估计是经过精心琢磨出来的，这个结果不仅保护了范冰冰，可能还保护了很多人。
 <br/>
 <br/>
 《证券日报》此前曾公布范冰冰三桩罪名，一个是逃税，一个是银行违规贷款，一个是腐败。而现在来看后面两项罪名消失得无影无踪。
 <br/>
 <br/>
 凌沧洲认为，正是后面这两项罪名可能卷入了很多人。所以从这个意义上讲，官方一方面要处理范冰冰这个〝烫手山芋〞，毕竟她有很高的知名度，国内外都很关注这个事件；另一方面，这里边的水很深，不是普通民众可以想像到的。
 <br/>
 <br/>
 凌沧洲说，范冰冰这个事，以现在这个结果告一段落，官方应该是想尽快平息这件事，不想让事情继续炒成热点。范冰冰把8.8亿交上去后，肯定是官府收益。这是个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_割韭菜.html" target="_blank">
  割韭菜
 </a>
 的时代，现在这把韭菜很肥沃，政府收到8.8个亿，你也不知道它会用到哪里去了。
 <br/>
 <br/>
 范冰冰逃税案落幕之际，国税总局要求涉逃税的影视企业及相关从业人员年底前自首，可免除刑责。中共政法委也扬言逃税一事，并非针对范冰冰一个人，而是要惩治背后乱象〝见光必死〞。
 <br/>
 <br/>
 据悉，近期中共向明星开刀行动明显升级。广电总局早前颁布〝限酬令〞，规定明星参加节目，每集的片酬不能超过80万元；担任常驻嘉宾每季片酬的上限为1,000万。近期还传出，早前参加综艺节目的赵薇和舒淇被迫分别退回4000万人民币给电视台。
 <br/>
 <br/>
 另外，国税总局从8月起把明星收入所得税税率从6.7%大幅上调至42%，且今年1月起的收入，都将以42%税率追讨。有娱乐圈老板爆料说，受范冰冰事件影响，影视圈一众演员，为补税忙得团团转，大家一见面，画面是这样的：〝还有票吗？我现在缺3千万。〞
 <br/>
 <br/>
 有分析认为，正值贸易战升级、中共经济陷入困境、内忧外患之际，中共突拿油水丰厚的娱乐圈开刀，是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宰肥羊.html" target="_blank">
  宰肥羊
 </a>
 〞、〝割韭菜〞，大搞建政后的〝第二次公私合营〞，是将民营财产充公的讯号。预料中国未来震荡不断。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿）
