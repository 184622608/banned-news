### 美副总统警告中共：不许对内打压对外侵略
------------------------

<div class="wysiwyg">
 【新唐人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 时间2018年11月13日讯】美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 （Mike  Pence）即将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出访.html" target="_blank">
  出访
 </a>
 亚洲，他是否与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在APEC上互动引关注。出访前，彭斯罕见刊文，批中共〝对内打压人民，对外侵犯邻国主权〞，同时警告中共，美国不会允许这种威权及侵略行为。
 <br/>
 <br/>
 中共官方12日发布消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将于11月17日至18日，出席第26次亚太经合峰会（APEC），这是2018年巴布亚新几内亚首次主办APEC峰会。
 <br/>
 <br/>
 而美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 11月12日起开启了为期一周的亚洲之行。在访问日本、新加坡、澳大利亚后，彭斯将在17日出席APEC峰会，就美国的〝印度-太平洋〞战略发表重要演讲。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出访.html" target="_blank">
  出访
 </a>
 前，彭斯于11月9日罕见的在《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华盛顿.html" target="_blank">
  华盛顿
 </a>
 邮报》发表文章说，压制自己国家人民的政府，也往往会侵犯邻国的主权。
 <br/>
 <br/>
 他说，〝亚太地区不能容忍专制和侵略〞，美国将和志同道合的国家，与印度和太平洋岛国一起〝共同对抗任何威胁我们利益和价值观的人〞。
 <br/>
 <br/>
 彭斯强调，美国的印太战略支柱分别是繁荣、安全及法治，美国有意达成自由、公平及互惠的双边贸易协议，并提供600亿美元的资金，协助区内国家建造世界级的港口、机场、道路、铁路、油管及网路线。
 <br/>
 <br/>
 在安全保障上，美国会继续与区内盟友合作，保护航行及飞行自由。
 <br/>
 <br/>
 彭斯也警告区内的国家尊重法治、保护人权及宗教自由。同时他还批评，有国家（中共）在国内打压人民，对外侵犯邻国主权，美国不会容许印太地区出现威权及侵略行为。
 <br/>
 <br/>
 在此之前，彭斯于10月4日在华盛顿，就川普政府的中共政策发表长篇演说，从政治、经济、军事、外交等方面，对中共进行全方位指控，被认为是〝讨伐中共檄文〞。
 <br/>
 <br/>
 彭斯在演讲中道出一个事实：中共从未变好，无论是对外或对本国人民。
 <br/>
 <br/>
 彭斯强调，美国在过去25年帮助〝重建〞了中国，相反，中共选择了经济侵略，也没有像美国希望的那样，让自己的人民迈向更大的自由，而是朝着控制和压迫中国人民的方向急转弯。
 <br/>
 <br/>
 彭斯说，现在美国真的觉醒了，〝历史已经证明，那些压迫本国人民的国家，很少就此住手。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 还试图将其势力扩展到全世界各地。〞
 <br/>
 <br/>
 因此，美国川普总统在经济、政治、外交、军事等领域展现出的全新姿态，绝非中共诋毁的〝单边保护主义〞，而是反击邪恶共产主义的进取之姿。
 <br/>
 <br/>
 美中之争，从贸易起步，向政治、外交、军事等全方位升级。在这场正邪对抗赛中，全世界的人都已经看清，美国的对手不是中国或中国人民，而是中共。
 <br/>
 <br/>
 外界预计，即将召开的APCE峰会上，美中双方将再次激烈交锋。而此次彭斯与习近平的互动将成为11月底G20峰会〝川习会〞的风向标。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
