### 孟宏伟落马事态紧急 公安部连夜宣布查处同党
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 受查一事，外界发现孟的落马时间诡异。北京当局7日深夜宣布孟落马的消息，紧接着，公安部长赵克志8日凌晨紧急召开会议，宣布公安部已成立工作专班，查处孟的同党。另据报导，公安部还有一名副部级高官同期被查。
 <br/>
 <br/>
 10月8日凌晨，身兼国务委员的公安部党委书记、部长赵克志，主持公安部党委会议，通报了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 收受贿赂、涉嫌违法接受国家监委监察调查的情况。
 <br/>
 <br/>
 会议说，中共公安部党委一致，对孟宏伟涉嫌违法受查，表示坚决拥护和支持。会议同时要求公安部官员，要认识到孟宏伟案件的严重危害，彻底肃清
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 流毒，要过好家庭关、亲情关等。
 <br/>
 <br/>
 会议还透露，公安部部已成立工作专班，查处与孟宏伟共同收受贿赂官员。
 <br/>
 <br/>
 中央纪委国家监委官网于7日深夜发布通告称，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 孟宏伟涉嫌违法，目前正接受国家监委监察调查。
 <br/>
 <br/>
 据陆媒报导，孟宏伟是中共国家监委自今年3月组建后，首位仅接受〝国家监委监察调查〞的官员。
 <br/>
 <br/>
 分析认为，孟宏伟深夜落马时间点上比较诡异，而赵克志凌晨主持公安部党委会议，说明孟宏伟事件非同一般，加之公安部宣布成立专班查处孟的同党，说明孟之后，公安部还会有更多高官落马。
 <br/>
 <br/>
 据海外中文媒体明镜火拍透露，在官方公布孟宏伟下台之前的3日晚，孟宏伟和另一名在公安部的副部级高官，被中纪委和国家监委宣布调查。不过，中共并没有正式公布另一名被调查的副部级高官的消息，该官员的名字目前仍在核查中。
 <br/>
 <br/>
 报导称，随着孟宏伟被查，将引发中共公安系统的大地震，或又会出现一系列的大清洗，或还有更高级的官员被清查。
 <br/>
 <br/>
 在官方公布孟宏伟下台消息的一星期之前，赵克志曾于9月30日召开公安部党委会议，特别强调〝从严治党〞，对发现的问题严肃查处，对屡屡发生问题的单位严肃问责。
 <br/>
 <br/>
 据法国警方透露，孟宏伟于9月29日只身从国际刑警组织的里昂总部，返回中国后失踪，孟妻于法国时间10月4日傍晚，向警方报案。
 <br/>
 <br/>
 5日，港媒爆料，孟宏伟抵达中国后，立即被中纪委带走，其因涉及经济问题及其他贪腐问题被抓。但有分析认为，中共因孟贪腐而大动干戈，对其扣押惊动全世界，逻辑上不成立。
 <br/>
 <br/>
 港媒分析，孟宏伟可能涉嫌中共政治忌讳，涉嫌北戴河政变案。因为案件重大而紧急，中共才不顾国际影响而紧急对孟宏伟问话。
 <br/>
 <br/>
 据悉，关于外界盛传北戴河政变的其中一个版本是，2015年的北戴河会议前夕，江泽民和江派二号人物曾庆红，联合周永康的前秘书、时任河北省委书记周本顺等人，炮制了一份《河北政情通报》，试图在北戴河会议期间发难，逼习近平下台。但在北戴河会议前夕，周本顺突然被中纪委带走调查。
 <br/>
 <br/>
 今年64岁的孟宏伟也是周永康的亲信，曾在中共安全机关担任过多项职务，在周永康落马并判以无期徒刑后，孟却能全身而退，甚至于2016年当选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 。
 <br/>
 <br/>
 中共十九大，赵克志担任中共公安部部长后，多次要求继续清洗周永康的〝余毒〞。2017年底，孟宏伟被免除中共海洋局副局长、海警局局长职务，今年4月，又被免去公安部党委委员一职。如今孟作为周的〝余毒〞被拿下似乎也不令人意外。
 <br/>
 <br/>
 随着孟宏传的落马，接下来公安系统势必引起一场大清洗，相信周永康的〝余毒〞或将再次被清剿。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：赵云）
