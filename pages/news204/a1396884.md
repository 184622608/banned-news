### 重庆女持刀砍杀14幼童  被抓获时遭围殴（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月26日讯】10月26日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 巴南一妇女持刀砍伤新世纪
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幼儿园.html" target="_blank">
  幼儿园
 </a>
 14名幼童，被抓获时遭众人围殴。
 <br/>
 <br/>
 中共警方通报，10月26日上午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 巴南区鱼洞新世纪
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_幼儿园.html" target="_blank">
  幼儿园
 </a>
 发生砍人案。通报称，师生系在园外做操返程时遇袭，保安与教师控制住该妇女。伤者随即被送医。
 <br/>
 <br/>
 网络流传的照片和影片可见，案发现场情况一片混乱，多名幼童整脸是血，大半部脸面遭到砍伤；还有幼童脖子遭砍伤，鲜血染满外套领口，还有学童被吓到大声哭喊，地面上血迹斑斑。还有孩童躺在担架上被抬出来送医。
 <br/>
 <br/>
 据报导，至少14名学生被斩伤。事发后，大批民众赶到现场，见凶手被警方控制时，在场民众群情汹涌，不少人上前围殴凶手洩愤，警方在场控制场面。
 <br/>
 <br/>
 网上无数网民声讨凶手，有人形容行为丧心病狂要严惩；也有人为受伤的学生心痛，提出冤有头债有主，请不要把愤怒发泄在无辜幼儿身上。
 <br/>
 <br/>
 据了解，犯罪嫌疑人已被警方控制，案情正在进一步调查当中。其中4名幼童因伤势较重转入重庆医科大附属儿童医院。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
