### 风险很大  美财长笑言不怕中共抛售美债
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】美国财政部长姆努钦（Steven Mnuchin）日前表示，他完全不担心中共因为贸易战而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抛售.html" target="_blank">
  抛售
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国债.html" target="_blank">
  美国国债
 </a>
 ，因为如果中共不要，还有一堆国家想要。他还笑言，如果中共抛售美债，反而会付出非常高昂的代价。
 <br/>
 <br/>
 正在印尼峇里岛出席国际货币基金组织（IMF）及世界银行年会的姆努钦，10月12日，被记者问到有经济学家警告，中共可能会
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抛售.html" target="_blank">
  抛售
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国债.html" target="_blank">
  美国国债
 </a>
 ，来报复美国征收关税时，他回答说，征收关税是非常重要的谈判工具。
 <br/>
 <br/>
 姆努钦更直言，如果中共决定不持有美债，会有其他买家等着要，他完全不担心，不会因此失眠。他还笑言，中共如果抛售，将会付出非常高昂的代价。
 <br/>
 <br/>
 中共近年来一直是美国国债第一或第二持有者，持有量超过一万亿美元，中共每年从美国债券收取了数十亿美元的利息。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 的背景下，业界或舆论讨论中共反制美国贸易战最后筹码时，讲得最多的便是中共会抛售美国国债，企图由此触发金融市场动荡，用以制造潜在的资产损失。
 <br/>
 <br/>
 近期北京确实减持了美债。据美国财政部9月18日公布的数据，7月份中国所持美债规模已降至1.171万亿美元，为年内第二低，仅次于1月的1.168万亿美元；而且当月持仓按月减少77亿美元，减持规模为1月以来最大。
 <br/>
 <br/>
 对于中共抛售美国国债，批评人士表示，中国经济太过脆弱，不能冒风险做出任何会引发不稳定的事情。同时，川普政府也正把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国经济强劲.html" target="_blank">
  美国经济强劲
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济下行.html" target="_blank">
  中国经济下行
 </a>
 压力大，视为美国手握的谈判筹码。
 <br/>
 <br/>
 对于美国来说，美国国债若被抛售，美国很容易应付。
 <br/>
 <br/>
 经济学家Brad Setser说：美联储（Fed, 即美国央行）完全可以自行购入和出售国债。他说〝美联储是世界上唯一有能力买入比中国卖出量更多国债的一方。〞
 <br/>
 <br/>
 亚洲经济专家陶冬也强调，将抛售国债作为武器的前提是美联储不干预。如果美联储采取宽松货币因应，释出流动性资金，再大的抛售也可以被吸收。
 <br/>
 <br/>
 《纽约时报》报导称，北京抛售美债的〝核选项〞没有试验场所，没有可以预测的计算结果，也没有比例模型。如果中共采用〝核选项〞，而市场不作出反应，北京将在众目睽睽之下失去影响力，对中国经济本身也将带来伤害。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明)
