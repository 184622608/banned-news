### 【禁闻】610文件命加剧打压法轮功 美官员谴责
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_610文件.html" target="_blank">
   610文件
  </a>
  命加剧打压
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
   法轮功
  </a>
  美官员谴责
 </b>
 <br/>
 <br/>
 日前，一份辽宁省610办公室命令升级打压
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 的文件在海外被曝光。文件内容显示打压行动被列为专项行动，具体由公安局负责实施，行动时间是2018年10月到12月。文件还详细列出了这次针对法轮功迫害的具体措施，包括〝主动进攻〞、〝高压恐吓〞、〝建立专案〞等。
 <br/>
 <br/>
 美国国际宗教自由委员会（USCIRF）研究与政策副主任Tina Mufford，11月27号表示：〝国际社会应对此特别警惕。因为这一行动计划来自臭名昭着的610办公室，这是中共实施迫害的非法工具。〞
 <br/>
 <br/>
 她说：〝这非常令人不安，这表明可能会有更多的事情发生。〞
 <br/>
 <br/>
 Tina Mufford表示：〝十几年来，中共一直在部署对法轮功学员的恐吓和骚扰。迫害时间太长了，必须停止。〞
 <br/>
 <br/>
 <b>
  国际特赦启紧急救援 促中共释放
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李明哲.html" target="_blank">
   李明哲
  </a>
 </b>
 <br/>
 <br/>
 台湾社工李明哲在中国大陆被判刑已经一周年，11月28号，台湾多个团体举行记者会，谴责中共任意将李明哲转移监狱，并拒绝家属探望。
 <br/>
 <br/>
 国际特赦组织台湾分会副理事长刘璐娜在会上表示，李明哲最近遭狱方多次移监、失联，他的妻子申请探监又遭狱方拒绝。
 <br/>
 <br/>
 她说，国际特赦组织27号已经启动全球紧急救援，要求中共政府立刻释放李明哲，并保证在释放前，李明哲免受酷刑及其他虐待。
 <br/>
 <br/>
 <b>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马云.html" target="_blank">
   马云
  </a>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共党员.html" target="_blank">
   中共党员
  </a>
  身份曝光 外界热议
 </b>
 <br/>
 <br/>
 中共喉舌媒体《人民日报》11月26号公布了一份对中共所谓〝改革开放〞有贡献的人员名单，其中包括大陆网商巨头、阿里巴巴创始人马云。
 <br/>
 <br/>
 《人民日报》为他撰写的简介揭示，马云是中共党员。
 <br/>
 <br/>
 由于马云此前在公开表态中刻意拉开与中共的距离。他党员身份的暴露，引发了外界议论。
 <br/>
 <br/>
 〝自由亚洲电台〞引述评论人士的话说，党媒披露马云党员身份，说明中共想要马云承担更多的政治任务。
 <br/>
 <br/>
 但有网友则认为，《人民日报》揭露马云的党员身份，一是为了警告他听话，二是为了把今后中共处置他的事当做党内事务，以堵外媒之口。
 <br/>
 <br/>
 编辑/周玉林
