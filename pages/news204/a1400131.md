### 【禁闻】1至10月中共非法抓捕数千法轮功学员
------------------------

<div class="wysiwyg">
 【新唐人2018年11月20日讯】中共迫害
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 加剧，今年初，中共政法委、610、公安部假借〝扫黑除恶〞之名，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信仰.html" target="_blank">
  信仰
 </a>
 〝真善忍〞做好人的法轮功学员进行全国范围绑架、抄家，截至10月31号，共有3944名学员被非法抓捕，3783名学员被骚扰，其中，10月份有300人被抓。
 <br/>
 <br/>
 据《法轮大法明慧网》报导，今年1至10月，中共绑架了3944名学员，以5月份最多，达693人。10月，共有71个城市的警察参与绑架，导致300人被抓，其中，长春市就非法抓捕了48名学员。
 <br/>
 <br/>
 法轮大法信息中心发言人张而平：〝今年每一个月都有几十人、上百人被抓捕的报导，这个数据其实还不是完整的，大家知道，在国内收集这些资料是很困难的，由于信息封锁，实际的情况比报导的情况更严重，更多的人被抓捕。〞
 <br/>
 <br/>
 除了绑架，10月份有95名学员被骚扰、21人被非法批捕，12人被强制送入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗脑班.html" target="_blank">
  洗脑班
 </a>
 。
 <br/>
 <br/>
 广州市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员余萍因包里有几张法轮功资料，10月21号在北京遭警察绑架，22号被劫持到广州市一个洗脑班，进行迫害。11月9号，家属和律师一起到洗脑班了解情况，洗脑班不但拿不出非法拘禁的依据，还威胁要殴打律师。
 <br/>
 <br/>
 张而平：〝它以各种所谓打黑或者其他形式对法轮功进行严酷的迫害，在各个省市都有大量的法轮功学员被抓捕，他们甚至到法轮功学员的家里去搜电脑，去查那个资料，甚至还要罚款、非法判刑，还有的是遭到拘留。〞
 <br/>
 <br/>
 中共对老年法轮功学员的迫害也是严重的，10月份就有20名老年学员被绑架或骚扰。
 <br/>
 <br/>
 山东济南75岁法轮功学员刘嗣堂与妻子张慧清9月6号外出讲真相时被绑架。刘嗣堂因年龄大放回家，但12天后又在家中被绑架，并与妻子一起被劫持到济南看守所。10月25号，刘嗣堂因之前遭迫害被非法判刑5年半的经历，再次遭检察院非法批捕。
 <br/>
 <br/>
 中国问题研究学者凌晓辉表示，由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_信仰.html" target="_blank">
  信仰
 </a>
 的对立，中共必然会迫害法轮功。
 <br/>
 <br/>
 中国问题研究学者凌晓辉：〝法轮功相信的是真善忍，它相信的是假恶斗。中共它一直是靠恐怖、杀人来维持政权的，法轮功讲究的是平和，在这方面是对立的，所以它一定要迫害法轮功。〞
 <br/>
 <br/>
 长春市73岁法轮功学员杨培侠，10月12号在家中被警察绑架和抄家，当天被送到看守所非法关押，近况不明。杨培侠曾多次被关押，期间遭警察搧嘴巴，用电棍电、用皮鞋踢、用拳头打等。
 <br/>
 <br/>
 凌晓辉：〝它对法轮功学员的迫害，完完全全是非法的，连它自己制定的宪法，都是违背的。因为公民有信仰自由、言论自由。〞
 <br/>
 <br/>
 进入11月，黑龙江省成为重灾区，仅11月9号一天，就有超过110名法轮功学员被绑架。
 <br/>
 <br/>
 在这次行动之前，也就是10月25号，黑龙江省公安厅召开党委会议，介绍全国〝扫黑除恶〞工作进展，声称此项专项行动推进至〝关键阶段〞。据《明慧网》报导，消息来源称，黑龙江省正在预谋对法轮功学员进行大规模的抓捕迫害行动，预计抓捕规模全省在1000人以上。
 <br/>
 <br/>
 凌晓辉：〝整整19年对法轮功学员的迫害从来就没有停止和放松过，甚至于迫害得更加严厉，包括对法轮功学员器官活体摘除，这个事情是一直到现在还在进行的。〞
 <br/>
 <br/>
 凌晓辉指出善恶有报的天理。
 <br/>
 <br/>
 凌晓辉：〝人间有一个天理，就是说善恶有报。前不久我一个朋友从广东过来，他是处级干部，他就讲，他们当时的那个处长迫害法轮功，迫害没几年，他就全身瘫痪。〞
 <br/>
 <br/>
 据明慧网报导，吉林榆树政府官员、政法系统，这些年在中共对法轮功的迫害中充当打手，目前，至少有50名官员和警察遭厄运，有的被判刑，有的被〝双规〞调查，有的上吊、跳楼自杀，有的患病死亡或车祸伤亡，也有的殃及家人死亡等。
 <br/>
 <br/>
 采访/常春 编辑/陈洁 后制/葛雷
