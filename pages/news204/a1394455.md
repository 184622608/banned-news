### 【禁闻】国际刑警主席在中国失踪 法国警方介入
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】日前，海外媒体非常关注，曾频繁颁发〝红通令〞来抓捕中国逃亡海外人士的国际刑警组织主席孟宏伟，自己也在中国被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 中纪委10月7号深夜发出消息称，孟宏伟涉嫌违法，正在接受国家监委监察调查。
 <br/>
 <br/>
 身居国际刑警组织总部法国里昂的孟宏伟妻子，10月4号晚向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国警方.html" target="_blank">
  法国警方
 </a>
 报案，称孟宏伟自9月29号回中国后，自己至今没有任何有关丈夫的消息。
 <br/>
 <br/>
 法国警方随后表示，他们已就此案展开调查。
 <br/>
 <br/>
 孟宏伟的妻子报案后，她本人不断遭到来自社交网络和电话等方式的威胁。
 <br/>
 <br/>
 孟宏伟的另一个身份是现任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 公安部副部长。2016年11月，他当选国际刑警组织主席，任期到2020年。他也是国际刑警组织自成立以来首位来自中国的主席。
 <br/>
 <br/>
 国际刑警组织北京分部就此事询问中共当局，但对方〝没有提供任何细节〞。法国内政部则表示，〝法方与中国的沟通一直在继续。〞
 <br/>
 <br/>
 美国华府人权律师叶宁表示，中共把派往国际组织担任领袖的这么一个人，用绑架的方式给他玩
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 ，这件事情贻笑天下。
 <br/>
 <br/>
 美国华府人权律师叶宁：〝做的是非常非常的毛躁愚蠢，而且恐怕发生了一些非常紧急的事情，才会用这种硬着陆的这么粗暴、这么不光彩、不体面的方式来做这样的事情。这会在国际社会造成笑话，整个就是一个丑闻。〞
 <br/>
 <br/>
 叶宁认为，孟宏伟的夫人向法国警方报警，等于是向国际社会通报。这会让中共非常尴尬：中共是怎么执行、怎么面对国际社会政策的？
 <br/>
 <br/>
 国际刑警组织主席被失踪事件疯传全世界，但中共至今对此保持沉默。
 <br/>
 <br/>
 根据香港《南华早报》引述消息称，孟宏伟上周搭乘的航班抵达中国后，立即被纪检部门带走，目前正在接受调查。但报导没有透露他为何被调查。
 <br/>
 <br/>
 旅居澳洲的前台湾总统府国策顾问邱垂亮对《新唐人》表示，孟宏伟这个国际刑警组织主席也是中共推荐的，但他在国内以前有过什么问题，是否犯过法，外界根本不知道。
 <br/>
 <br/>
 旅居澳洲的前台湾总统府国策顾问邱垂亮：〝但是我们外面得到的一个印象是说，中国人不管全世界哪里，是没有什么人权、安全的，他要抓就抓。现在唯一能解决这问题是国际刑警组织在法国。法国跟中国之间怎么样进行讨论，处理这个问题？〞
 <br/>
 <br/>
 邱垂亮指出，只要是中国人，不管是大陆人还是台湾人，中共现在是想抓就抓、想关就关。中共就根据它的专制独裁法律抓人，也不向世界解释，它秘密审判，秘密投入监狱。中国现在是完全没有人权的一个国家。
 <br/>
 <br/>
 孟宏伟于2016年底当选国际刑警组织主席，他的当选当时就引起很多争议，外界质疑和担忧该组织的公正性。
 <br/>
 <br/>
 目前，孟宏伟的失踪也给国际刑警组织的运作带来挑战。主席失踪，将由谁来签发〝红色通缉令〞？又由谁来决定谁该列入红通名录？
 <br/>
 <br/>
 《德国之声》对此报导说，〝国际刑警组织主席失踪：中国开国际玩笑？〞
 <br/>
 <br/>
 叶宁：〝极权社会、极权主义横行霸道发展到最后的阶段，崩溃阶段它就是这样一种社会形式，它整个就乱了，已经是荒腔走板、没有底线了。国际刑警组织的主席不是不可以抓，但问题是，抓你得有一个程式、有一个手续啊！哪有这么个抓法的？〞
 <br/>
 <br/>
 叶宁指出，这事中共做的极不光彩、极不体面，在国际社会上还会进一步陷入被动局面。而孟宏伟作为中共正部级的高官都随时被抓、成为权斗牺牲品，这对中共高层是个震撼，人人自危。
 <br/>
 <br/>
 采访/陈汉 编辑/宋风 后制/周天
