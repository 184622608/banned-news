### 习近平乐观讲话效果不彰 亚洲和陆股继续下跌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】中国首届国际进口博览会未能释放利好信号。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 开幕式上的〝乐观〞讲话效果不佳，当天亚洲和中国股市继续下跌。港媒称，习已多次承诺加大开放市场，但北京的实际行动却不一样，世界政商领袖对此已不感兴趣。
 <br/>
 <br/>
 为期一周的中国国际进口博览会11月5日在上海开幕，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在开幕式主旨演讲中承诺，中国将降低关税并加快开放国内市场。
 <br/>
 <br/>
 不过，在中美陷入激烈贸易战之际，习近平的〝乐观〞言论并未产生积极影响。开幕式当天，亚洲股市继续下滑，香港恒生指数收盘下跌2.1%，中国沪深300指数也下跌0.8%。
 <br/>
 <br/>
 香港《亚洲时报》11月6日报导称，习近平的话并不能如实地反映北京的行为。在过去两年里，他多次做出同样的呼吁与承诺，但北京的实际行动却和习近平宣扬倡导的并不一样。
 <br/>
 <br/>
 报导称，习近平不应再做出更多同样的承诺，而是要践行承诺的内容。否则，世界政商领袖将不再相信或对他的讲话感兴趣。
 <br/>
 <br/>
 上海欧盟商会主席安德烈（Carlo Diego D’Andrea）表示，习近平的讲话虽然做出了多项承诺，但缺乏具体措施与改革的明确时间表。他们不能在改革只是〝有望到来〞的基础上，就让在中国的欧洲公司首席执行官（CEO）们开展商贸活动。
 <br/>
 <br/>
 而且，外界认为，习近平所说的扩大开放这些政策，中共早在16年前加入世界贸易组织时，就已经向世界承诺了，但都没有兑现。
 <br/>
 <br/>
 从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国入世.html" target="_blank">
  中国入世
 </a>
 之后，2002年，美国贸易代表办公室（USTR）每年都会撰写中国履行WTO承诺情况的报告，递交给国会。每年的内容都不同，但这17份报告中都无一例外地点明，中共至今仍然未能完全遵循WTO的许多基本原则。
 <br/>
 <br/>
 美国今年1月公布的中国2017年度报告，详细列举了中共在执行世贸组织政策上存在的多项问题。如，中共政府限制商品和服务进口，限制外国制造商和服务提供商的市场准入；中共施压外国公司技术转让；利用产业政策保护国企，同时对外国同类公司设置障碍等等。
 <br/>
 <br/>
 中共入世11方面承诺未兑现。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-7/p9114831a653014592.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-7/p9114831a653014592-ss.jpg"/>
   <br/>
   <font size="-1">
   </font>
  </a>
  <br/>
 </center>
 <br/>
 今年4月，习近平在海南省的博鳌亚洲论坛开幕式上，宣布将进一步扩大开放的消息。扩大开放的内容包括放宽市场准入，放宽外资金融机构设限，放宽外资股比限制，降低汽车进口关税，保护知识产权等。但专家表示，这些多是加入世贸组织作出的承诺，却一直没有兑现。
 <br/>
 <br/>
 中国经济学家夏业良教授表示，〝中国加入WTO已经超过16年，这些承诺仍未兑现。现在宣传说的怎么怎么样好，但都是应该做而没有做的。〞
 <br/>
 <br/>
 他强调，〝美国去年就宣布中国不是市场经济的国家，欧盟、日本也说不是市场经济，其它国家也会越来越多抵制中国，现在它再说什么也很难让人相信了。现在中共虽然说得很好听，但是到落实过程中，会有很多走样的。〞
 <br/>
 <br/>
 （记者罗婷婷/责任编辑：戴明）
